     <p>
      2019 年 9 月，坐标微软上海。
     </p>
     <p>
      2012 年暑假在微软实习，之后成功转正，在 2013 年正式加入微软。
     </p>
     <p>
      其实当年拿到转正 offer 时，还在微软和直研之间纠结，思考再三，还是选择了来微软。时至今日，一直很庆幸当初做了正确的决定！
     </p>
     <p>
      入职以来，见证了微软市值重回第一，股价从 30 美元一直涨到现在的 130 多美元，自己的技术和软实力也有很大的提升。
     </p>
     <p>
      工作方面，身边的同事都很牛，整个团队的技术氛围很好，公司有很丰富的技术资源，微软也越来越拥抱开源。六年多的时间下来，公司对我成长的帮助是非常巨大的。
     </p>
     <p>
      在 Engineering 方面，微软在业界一直做的很好的：严格执行的 Code Review，严谨的 Coding Styles 规范，完善的 CI/CD 流程。
     </p>
     <p>
      在开源方面，从十多年前的“为什么要开源？”，变成了如今的“为什么不开源”。
      <a href="http://mp.weixin.qq.com/s?__biz=MzU1NjgwNTExNQ==&amp;mid=2247483756&amp;idx=1&amp;sn=d4faca9d8f69b67e6e1833431531c831&amp;chksm=fc3e3ff8cb49b6eebe63083a2080d9d304d38604fbb608544249f5dda3311760f7efa6367c35&amp;scene=21#wechat_redirect" rel="nofollow">
       VS Code 就是微软开源之典范
      </a>
      。
     </p>
     <p>
      微软对技术的多元化以及包容性也是极大地提升了我的技术能力：
     </p>
     <ul class="list-paddingleft-2">
      <li>
       <p>
        你也许不会想到，我在微软除了写过一年多的 C#，还写过 PHP，Java，Python 以及我的最爱 —— TypeScript。真是因为编程语言的多元化，也使得我有机会在业余时间开发出了超过 900 万下载量的
        <a href="http://mp.weixin.qq.com/s?__biz=MzU1NjgwNTExNQ==&amp;mid=2247483744&amp;idx=1&amp;sn=0fb72fd7366fccf03d747fccedbfd514&amp;chksm=fc3e3ff4cb49b6e2df257e4a3e5755c99c4128695bd2d9bb52baa4f6c6597c9a570fae28db8d&amp;scene=21#wechat_redirect" rel="nofollow">
         Code Runner for VS Code
        </a>
        。
       </p>
      </li>
      <li>
       <p>
        对于 CI/CD，我们在不同的项目用过 TeamCity、Jenkins、Travis CI、AppVeyor 以及 Azure Pipelines。
        <a href="http://mp.weixin.qq.com/s?__biz=MzIwODE4Nzg2NQ==&amp;mid=2650548529&amp;idx=1&amp;sn=1be09ba83a2277f802cc864055ad143a&amp;chksm=8f0e7187b879f8914ccfe4e03808c8d7b123ba170eeee5864bf0b02e42fe574b45fdcf219b36&amp;scene=21#wechat_redirect" rel="nofollow">
         不同的 CI/CD 工具适合于不同的场景，我们总是能选出最合适的 CI/CD 工具
        </a>
        。
       </p>
      </li>
     </ul>
     <p>
      随着技术的提升，我也有了很多机会，把我的一些经验分享给更多的朋友们：
     </p>
     <ul class="list-paddingleft-2">
      <li>
       <p>
        <a href="http://mp.weixin.qq.com/s?__biz=MzU1NjgwNTExNQ==&amp;mid=2247483710&amp;idx=1&amp;sn=8d1caa1e80262f29c3a1f21dcc1df008&amp;chksm=fc3e3faacb49b6bc84977ac53076ff242e2aef6354b4f38a3cebd166670655872cedfcae8d73&amp;scene=21#wechat_redirect" rel="nofollow">
         2018 微软技术暨生态大会 —— 《从零开始开发一款属于你的 Visual Studio Code 插件》
        </a>
       </p>
      </li>
      <li>
       <p>
        <a href="http://mp.weixin.qq.com/s?__biz=MzU1NjgwNTExNQ==&amp;mid=2247483832&amp;idx=1&amp;sn=94c56843e47e53fd5a0ec251dcea8bf7&amp;chksm=fc3e3f2ccb49b63a56a7cbb7152285fd2f2fe10b224dba2b2f248ee217640525b98c497768c6&amp;scene=21#wechat_redirect" rel="nofollow">
         Google Developer Group —— 《VS Code插件开发，从零基础到精通》
        </a>
       </p>
      </li>
      <li>
       <p>
        <a href="http://mp.weixin.qq.com/s?__biz=MzIwODE4Nzg2NQ==&amp;mid=2650548684&amp;idx=1&amp;sn=fbf562c3b4030b1ccb65d14b147d3cb3&amp;chksm=8f0e717ab879f86cf1261502ad15308ff64a4c11750937b22158208bdadba41b50c6999de101&amp;scene=21#wechat_redirect" rel="nofollow">
         微软 Reactor 上海 —— 《开源利器 —— VS Code 应用开发面面观》
        </a>
       </p>
      </li>
      <li>
       <p>
        <a href="http://mp.weixin.qq.com/s?__biz=MzIwODE4Nzg2NQ==&amp;mid=2650548682&amp;idx=1&amp;sn=290f527eea5feaceb80658e25f2b9e36&amp;chksm=8f0e717cb879f86ad4e5985c62896937d402bb7d3f47e950ab30282f74280dc2ea888a44f570&amp;scene=21#wechat_redirect" rel="nofollow">
         工欲善其事，必先利其器，《VS Code 从入门到进阶》知乎 Live
        </a>
       </p>
      </li>
     </ul>
     <p>
      另外还有个很有意思的地方。微软对于离开微软的人，依旧具有很大的吸引力。六年来，我看到很多人离开了微软，而过了一段时间，又回来了。今年我开始帮公司做内推，
      <a href="http://mp.weixin.qq.com/s?__biz=MzIwODE4Nzg2NQ==&amp;mid=2650548696&amp;idx=1&amp;sn=c80a8f60ca27c87d6e30f1f10c4ef81b&amp;chksm=8f0e716eb879f8781174b107ae2d3b56a104cda451a048931a94538ed6325b1572425034b24e&amp;scene=21#wechat_redirect" rel="nofollow">
       一共成功内推 13 个人拿到 offer
      </a>
      ，而其中有 3 个人曾经是微软的员工。
     </p>
     <p>
      工作之外的方面，公司会在各个方面把你照顾地很好。:-)
     </p>
     <ul class="list-paddingleft-2">
      <li>
       <p>
        15天全薪年假，15天全薪病假，6周的全薪陪产假 （根据工作年限，年假最高可提升到20天）
       </p>
      </li>
      <li>
       <p>
        两台 2K 高清显示器 + 高性能台式机必须是标配。根据工作内容，还可以选择配 MacBook 或者 Surface Book。（嗯，我们组还配了 Filco 机械键盘。撸代码效率大大提高。）
       </p>
      </li>
      <li>
       <p>
        每天都有新鲜的水果，如果是柚子之类的水果，茶水间阿姨会帮你一个个剥好。:P
       </p>
      </li>
      <li>
       <p>
        无限的零食供应 （肥宅快乐水，健康椰子水，果汁，咖啡——用的是星爸爸豆子哟，养生枸杞茶应有尽有）
       </p>
      </li>
      <li>
       <p>
        多方位的休闲环境 （园区内 GYM，八块腹肌不是梦，瑜伽舞蹈 A4 腰；广阔的大草地可以踢球⚽️，足球/篮球场/网球场上赛一把，战斗友谊更紧密；Game room 街机厅，让你放松我们是认真的；K歌房、跳舞机谁还不是个宝宝；台球、乒乓球、foosball、影音厅，带娃上班 so easy ）
       </p>
      </li>
      <li>
       <p>
        丰富的节日活动 （ Family Day，Ice Cream Day...）（前两年的 Family Day 是让员工带着家属去迪士尼嗨哦~）
       </p>
      </li>
      <li>
       <p>
        多达几十班的遍布全上海的班车，上下班通勤也绝不是什么问题
       </p>
      </li>
     </ul>
