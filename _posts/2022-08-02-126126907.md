    <h3>
     <a id="_JDK_5">
     </a>
     卸载 JDK
    </h3>
    <ol>
     <li>
      <p>
       打开 “我的电脑-属性-高级系统设置-环境变量-系统变量-JAVA_HOME”
      </p>
     </li>
     <li>
      <p>
       如下图所示，复制变量值，在文件夹的地址栏输入变量值 “D:\Program Files\Java” 找到 “jdk1.8.0_181”，把 “jdk1.8.0_181” 直接删掉：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/046f978e11533d2d6a7b16b7f0e205a4.png">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/812c722f9ae3f6a80142c3e97ea8abe6.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       删除 JAVA_HOME。删除了 jdk1.8.0_181 的文件夹之后，再把上图中的 JAVA_HOME 变量给删除。
      </p>
     </li>
     <li>
      <p>
       删除 path 下关于 Java 的目录，如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/be2cc1587c4bf9cd684838442c7bb9dd.png"/>
      </p>
     </li>
     <li>
      <p>
       java -version。在 cmd 中输入 java -version 命令，如下图所示，则卸载成功：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/26368d651a1bf3fa70856aa82ad1391c.png"/>
      </p>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_JDK_18">
     </a>
     安装 JDK
    </h3>
    <ol>
     <li>
      <a href="https://www.oracle.com/java/technologies/downloads/#java8" rel="nofollow">
       去官网下载 JDK8
      </a>
      ，根据自己电脑位数安装对应的版本，我的是 Windows 64 位，则如下图所示，点击右侧蓝色链接进行下载：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a052378052be44d90f6525f10bf3452.png"/>
     </li>
     <li>
      记住安装的路径。正常安装，并记住环境路径，我的路径是：D:\Program Files\Java\jdk1.8.0_341
     </li>
     <li>
      配置环境变量。
      <ol>
       <li>
        打开 “我的电脑-属性-高级系统设置-环境变量-系统变量” ，并点击 “新建” 来新建系统变量如下图：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dcf1e55b0ad30118f0f02c6457521c5b.png"/>
       </li>
       <li>
        如下图所示，变量名设置为 “JAVA_HOME”，变量值就是刚刚记录的安装路径 “D:\Program Files\Java\jdk1.8.0_341”，点击确定：
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4c5c3b917466d70c8f16d277119474e1.png"/>
       </li>
      </ol>
     </li>
     <li>
      配置 path 变量。
      <ul>
       <li>
        同上边操作一样，找到系统变量，双击打开 path 操作如下：
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/27b9eef8d6a2c0c8add2e3e820cb9bed.png"/>
       </li>
       <li>
        点击 “新建” 如下图所示：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1061406487bfe983eab0ce16292677c7.png"/>
       </li>
       <li>
        添加两个路径
        <code>
         %JAVA_HOME%\bin
        </code>
        和
        <code>
         %JAVA_HOME%\jre\bin
        </code>
        ，其中两个百分号表示引用，此处 JAVA_HOME 就是我们刚刚配置的系统变量路径 “D:\Program Files\Java\jdk1.8.0_341”，相当于指向
        <code>
         D:\Program Files\Java\jdk1.8.0_341\bin
        </code>
        和
        <code>
         D:\Program Files\Java\jdk1.8.0_341\jre\bin
        </code>
        。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a2cb6a7e2cdd1ed7c854109f2f5c0f5f.png"/>
       </li>
      </ul>
     </li>
     <li>
      测试 JDK 是否安装成功。在 cmd 中输入：
      <pre><code>java -version
</code></pre>
      出现下图红框中内容则表示成功：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6df814539fb3d91e8ace84b9a0db78e3.png"/>
      <br/>
      注意：用系统带的cmd进行查看，若用 anaconda prompt 等查看则可能不成功
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8df9e3fe15577f426dd4499d89e1b59b.png"/>
     </li>
    </ol>
    <h3>
     <a id="JDK__40">
     </a>
     JDK 中的文件简单介绍
    </h3>
    <h4>
     <a id="bin__41">
     </a>
     bin 目录（重点）
    </h4>
    <p>
     bin 目录：放一些可执行文件
    </p>
    <h4>
     <a id="include__43">
     </a>
     include 目录
    </h4>
    <p>
     include 目录：由于 JDK 是用 C 和 C++ 编写的，所以启动时要引入一些 C 和 C++ 的头文件，这里边就存放这些头文件。
    </p>
    <h4>
     <a id="jre__45">
     </a>
     jre 目录（重点）
    </h4>
    <p>
     jre 里点开是 Java 运行环境，如果只跑 Java，不开发，有这个 jre 就够了，在 path 里面配置也是因为要用到 jre，如果没配置该路径，Java 程序则无法运行。
    </p>
    <h4>
     <a id="lib__47">
     </a>
     lib 目录
    </h4>
    <p>
     lib 里放了 Java 相关的类和库文件
    </p>
    <h4>
     <a id="srczip_49">
     </a>
     src.zip
    </h4>
    <p>
     src 是资源文件，里面放了很多 Java 类
    </p>
