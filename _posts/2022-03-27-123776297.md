    <h2>
     <a id="Python_4">
     </a>
     一.Python安装前必须知道的三个概念
    </h2>
    <h3>
     <a id="11_python_5">
     </a>
     1.1 python解释器（翻译）
    </h3>
    <p>
     <img alt="python解释器" src="https://i-blog.csdnimg.cn/blog_migrate/8f9d4568c412ba88722858fa3773434e.png">
      <br/>
      Python解释器是将
      <strong>
       python源码高级语言
      </strong>
      解析为
      <strong>
       二进制机器语言的工具
      </strong>
      ！
      <mark>
       通常说安装python就是指安装python解释器。
      </mark>
     </img>
    </p>
    <p>
     目前最新的python解释器版本是python3.8.x；另外，python2.x和python3.x不兼容！
    </p>
    <h3>
     <a id="12__11">
     </a>
     1.2 编辑器（编写代码）
    </h3>
    <p>
     编辑器这个概念依然非常好理解，比如你要写一个文档，你可能会使用WPS,Word等编辑软件；做演示文稿你需要用PPT，修图用PS一样，编写代码也需要有一个编译器，
     <strong>
      这个用来编写代码的工具就叫做编辑器
     </strong>
     ！
     <br/>
     <img alt="pycharm" src="https://i-blog.csdnimg.cn/blog_migrate/a1986a2e4946052088d6d486d3cd648d.png"/>
    </p>
    <p>
     Python的编辑器有很多，有Python解释器自带的
     <strong>
      IDLE
     </strong>
     ，基于iPython的
     <strong>
      Jupyter Notebook
     </strong>
     ,也有比如
     <strong>
      Pycharm
     </strong>
     这种主要针对python代码编写的编辑器，功能强大；对于别的语言还有很多编辑器，比如VSCode,IDEA等适合各种语言的编辑器。
    </p>
    <h3>
     <a id="13__18">
     </a>
     1.3 包管理工具
    </h3>
    <p>
     Python最大的优点之一在于其有非常多丰富的
     <strong>
      库
     </strong>
     ，也可称之为
     <strong>
      包
     </strong>
     。
     <mark>
      pip(package installer for Python)就是库管理工具
     </mark>
     。通过pip就可以安装、卸载、更新众多的库，比如Numpy,Pandas,Tensorflow等。
     <br/>
     <img alt="Numpy" src="https://i-blog.csdnimg.cn/blog_migrate/d366e503526ec32b731b4fd9c0a829f8.png"/>
    </p>
    <p>
     这一块的知识博主一开始也不是很明白，为此还专门研究了一段时间，写了一篇通俗易懂文章，分享给大家。
     <br/>
     链接:
     <a href="https://blog.csdn.net/m0_51675604/article/details/121357841">
      关于库与Python的相关问题及镜像的解释【通俗易懂版】
     </a>
     .
    </p>
    <ul>
     <li>
      <strong>
       优势及用处
      </strong>
     </li>
    </ul>
    <p>
     这样我们就可以基于别人写好的代码之上，做开发，开发效率就会更高！
     <br/>
     <br/>
    </p>
