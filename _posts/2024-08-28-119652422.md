    <p>
     1、几个原则必须要遵守
     <br/>
     （1）华为喜欢那种喜欢加班的人，所以必须要能吃苦耐劳不计较报酬
     <br/>
     （2）华为喜欢中庸，不要表现自己的特性，有自己性格一定会被刷掉
     <br/>
     （3）华为喜欢稳定的，稳定超过一切包括技术、包括是否能做事
     <br/>
     （4）性格不能过激，比如那种会跳楼第一时间会被刷掉
     <br/>
     （5）团队合作很重要，不要自己来往
    </p>
    <p>
     2、性格测试的主要测试点就是：
     <br/>
     （1）地雷，比如你选择了不愿意加班，你自己觉得你英明神武等，直接淘汰
     <br/>
     （2）自相矛盾，比如你刚开始选择你广泛听取别人意见，后面又选择你一个人做决定等
     <br/>
     （3）就是不同维度的比较让你选择最合适和最不合适，这是连环题，会有3到5道题形成一个连环，先后不要矛盾。比如
     <br/>
     题37：我做事广泛听取别人的意见、我喜欢创新、我坚持按时完成任务
     <br/>
     题45：我喜欢创新、我做事不喜欢半途而废、我坚持按时完成任务
     <br/>
     题59：我坚持按时完成任务、我做事不喜欢半途而废、我做事广泛听取别人的意见
    </p>
    <p>
     技巧：
     <br/>
     （1）遇到表现团队合作的，想都不用想，直接拉到最强烈同意那个。
     <br/>
     （2）守时，承担高压力工作，大事面前不慌张等表现优良工作品质的全部最高等级同意
     <br/>
     （3）遇到善于领导他人，健谈等表现你的领导力的，最好选择不同意，具体程度自己斟酌
     <br/>
     （4）是否有创造力等问题，这个不好说，我自己选择的轻微同意或者轻微不同意，怕表现太抢眼容易挂。
     <br/>
     （5）主要分为考察你的优良品质。比如乐观积极、喜欢压力、团队协作、善于倾听和参考意见、能够按时完成任务、循规蹈矩、守时等，华为不希望性格测试里有很多不安定的因素比如领导力、叛逆心、遇事会紧张、健谈等
     <br/>
     （6）要可靠，诚实可信，吃苦耐劳，艰苦奋斗，乐于分享，乐于讨论，能够循规蹈矩，不排斥枯燥的重复性工作，适当规划未来，对未来有信心。踏实肯干，事实为依据，数据为依据，强调团队协作
     <br/>
     （7）不能太能言善辩，不能雄心勃勃，不能太广泛交友，工作不要独来独往，不要过于自信。不能盲目乐观，或者热衷规划，空谈理论，无法实践。不欢迎个人主义，张扬，浮夸，激进。
     <br/>
     （8）不要有太多不确定和轻微xxx，这样表示没有认清自己，HR说的积极向上、乐观就行。
    </p>
    <p>
     4、所以我列出了一些会出现的性格，其他新出现的大部分都是同义词，背熟下面的顺序，按照下面的顺序选择最合适和最不合适
    </p>
    <p>
     <strong>
      A、最合适的性格
     </strong>
     <br/>
     我信守自己作出的承诺
     <br/>
     我觉得人们信守承诺相当重要
     <br/>
     我坚持按时完成任务
     <br/>
     我做事广泛听取别人的意见
     <br/>
     做事前我会征询大家的意见
     <br/>
     我愿意花时间去帮助他人
     <br/>
     我做事不喜欢半途而废
     <br/>
     我必须了解底层原理才能更有效地学习
     <br/>
     我倾向于根据客观事实做决定
     <br/>
     我必须找到解决问题的办法
     <br/>
     我需要了解论点背后的逻辑
     <br/>
     我喜欢创新
     <br/>
     做事时我喜欢有新方法，新点子
     <br/>
     我能想出很多主意
     <br/>
     我喜欢提出很多主意
     <br/>
     我喜欢提出独到的见解
    </p>
    <p>
     <strong>
      B、适中的性格
     </strong>
     <br/>
     我善于和别人建立融洽关系
     <br/>
     我喜欢有压力的环境下工作
     <br/>
     我喜欢忙碌的生活
     <br/>
     我需要有明确的远景计划
     <br/>
     我喜欢思考未来
     <br/>
     我给自己设定了长远目标
     <br/>
     我经常展望未来
     <br/>
     我对未来比较乐观
     <br/>
     我相信未来的事情会是好的
     <br/>
     我做事很有远见
     <br/>
     做事的时候我需要有章可循
     <br/>
     我做事井井有条
     <br/>
     我非常注重细节
     <br/>
     我善于同时处理多项任务
     <br/>
     我循规蹈矩地去做事情
     <br/>
     我喜欢对数据进行统计与分析
     <br/>
     我喜欢讨论抽象性的问题
     <br/>
     我善于处理数字资料
     <br/>
     我不容易被别人激怒
     <br/>
     我信任他人
     <br/>
     我在与陌生朋友交流时会轻松自在
     <br/>
     我大部分时间是快乐的
     <br/>
     我有强烈的好奇心
     <br/>
     我喜欢有竞争的工作
     <br/>
     我是一个竞争心很强的人
    </p>
    <p>
     <strong>
      C、不适合的性格
     </strong>
     <br/>
     我掌控着自己的未来
     <br/>
     我在重大事情发生前会紧张
     <br/>
     我想有人陪在我身边
     <br/>
     我坚持自己的做事方式
     <br/>
     我善于结交朋友
     <br/>
     我喜欢结识陌生人
     <br/>
     我善于谈判
     <br/>
     我善于推销
     <br/>
     我是个很健谈的人
     <br/>
     我关注别人的做事目的与做事动机
     <br/>
     我善于理解人们背后的动机
     <br/>
     我善于体会他人的感受
     <br/>
     我要超越别人
     <br/>
     我有较大的野心
     <br/>
     我想成为团队中的领导
     <br/>
     我更愿意领导别人
     <br/>
     我喜欢挑战别人的想法
     <br/>
     我享受独自工作
    </p>
