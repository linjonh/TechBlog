    <p>
     在 Ubuntu 22.04 上安装 Java 8 环境可以通过以下步骤完成。请注意，Java 8 已经到达了维护期末，建议升级到更新的 Java 版本以获得更好的安全性和性能。
    </p>
    <h3>
     <a id="java_8_1">
     </a>
     java 8
    </h3>
    <h4>
     <a id="_OpenJDK_8_3">
     </a>
     方法一：使用 OpenJDK 8
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        更新包管理器
       </strong>
       ：
      </p>
      <p>
       在终端中执行以下命令以更新包管理器：
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        安装 OpenJDK 8
       </strong>
       ：
      </p>
      <p>
       在终端中执行以下命令以安装 OpenJDK 8：
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> openjdk-8-jdk
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        验证安装
       </strong>
       ：
      </p>
      <p>
       安装完成后，可以通过以下命令验证 Java 版本：
      </p>
      <pre><code class="prism language-bash"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="_Oracle_JDK_8_29">
     </a>
     方法二：使用 Oracle JDK 8
    </h4>
    <p>
     如果你希望使用 Oracle JDK 8，可以按照以下步骤操作：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        下载 Oracle JDK 8
       </strong>
       ：
      </p>
      <p>
       访问
       <a href="https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html" rel="nofollow">
        Oracle 官方网站
       </a>
       下载适用于 Ubuntu 的 JDK 8 版本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        安装 Oracle JDK 8
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         将下载的 JDK 8 压缩包解压到一个目录中，例如
         <code>
          /opt
         </code>
         ：
        </p>
        <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">tar</span> <span class="token parameter variable">-zxvf</span> jdk-8uXXX-linux-x64.tar.gz <span class="token parameter variable">-C</span> /opt/
</code></pre>
       </li>
       <li>
        <p>
         配置 Java 环境变量，编辑
         <code>
          /etc/profile
         </code>
         文件：
        </p>
        <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">nano</span> /etc/profile
</code></pre>
        <p>
         在文件末尾添加以下内容：
        </p>
        <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span>/opt/jdk1.8.0_XXX
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token environment constant">$PATH</span><span class="token builtin class-name">:</span><span class="token variable">$JAVA_HOME</span>/bin
</code></pre>
        <p>
         请将
         <code>
          jdk1.8.0_XXX
         </code>
         替换为你解压的 JDK 版本。
        </p>
       </li>
       <li>
        <p>
         使配置生效：
        </p>
        <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> /etc/profile
</code></pre>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        验证安装
       </strong>
       ：
      </p>
      <p>
       执行以下命令验证 Java 安装是否成功：
      </p>
      <pre><code class="prism language-bash"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
     </li>
    </ol>
    <p>
     以上是在 Ubuntu 22.04 上安装 Java 8 的两种方法。选择适合你需求的方法进行安装。如果需要进一步帮助，请随时告诉我！
    </p>
