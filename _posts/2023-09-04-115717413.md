    <h4>
     <a id="Axiosoptions__0">
     </a>
     使用Axios时，会发送options 请求
    </h4>
    <blockquote>
     <p>
      http分为复杂请求和简单请求
     </p>
    </blockquote>
    <p>
     当浏览器认为是简单请求时则不会发送options请求，反之会先发送一个options预请求，然后才会发送复杂请求。
    </p>
    <blockquote>
     <p>
      满足以下条件的请求是简单请求：
     </p>
    </blockquote>
    <ul>
     <li>
      请求方式：GET、POST、HEAD
     </li>
     <li>
      请求的header里不能超出以下几种字段：
      <ul>
       <li>
        Accept
       </li>
       <li>
        Accept-Language
       </li>
       <li>
        Content-Language
       </li>
       <li>
        Last-Event-ID
       </li>
       <li>
        Content-Type
       </li>
      </ul>
     </li>
     <li>
      Content-Type 的值 只限于以下三种：
      <ul>
       <li>
        application/x-www-form-urlencoded
       </li>
       <li>
        multipart/form-data
       </li>
       <li>
        text/plain
       </li>
      </ul>
     </li>
    </ul>
    <blockquote>
     <p>
      Vue默认的Content-Type:application/json
     </p>
    </blockquote>
    <ul>
     <li>
      为复杂请求
     </li>
    </ul>
    <h4>
     <a id="_20">
     </a>
     解决办法
    </h4>
    <blockquote>
     <p>
      后端
     </p>
    </blockquote>
    <ul>
     <li>
      使用拦截器过滤options请求
     </li>
    </ul>
