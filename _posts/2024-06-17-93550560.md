     5G 时代，微软又走对了一步棋！
    </h1>
   </div>
  </div>

      <img alt="640?wx_fmt=gif" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/0aeae888bc82277a992c06d20bc45a35.gif" style="text-align:center;">
       <br/>
      </img>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <img alt="640?wx_fmt=jpeg" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/b1b0946b738add3b999d5b84aec04ac9.jpeg"/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       2019年4月，CSDN采访微软（中国）首席技术官韦青，期间谈到5G。他认为，5G绝对是一个划时代的革命性突破，但是这个突破不止于现在所说的“5G”通讯技术，它为未来以“万物互联”为基础的智能社会开创了全新的局面。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而这一任务，需要通过未来几代通讯技术的发展来实现，是一个循序渐进的过程，是一批“下一代通讯技术”的逐次实现。
      </span>
     </p>
     <p style="text-align:justify;line-height:1.75em;letter-spacing:1px;margin-left:8px;">
      <span style="font-size:15px;">
       <strong>
        <br/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/f939ea3ff91c35b7d714894a8b0ae23c.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        大众对5G初期布署期望偏高
       </strong>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       目前的舆论宣传，把社会大众对5G的期望值拉的偏高。如果大家把诸如自动驾驶所需要的无线通讯技术全都加到5G这一代技术进步之上，容易让市场因过高的期望反而产生不必要的失望，这对5G这么重要的技术发展不利。
      </span>
     </p>
     <p style="margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       回顾通讯技术的发展历史，从1G到4G，基本满足了覆盖和传输速度的需求。人类社会继续发展下去，绝不仅仅只是要求网速快，还得要求反应快，传输信号的延时也要发展到毫秒
      </span>
      <span style="font-size:15px;">
       级。
      </span>
     </p>
     <p style="margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而以5G通讯技术为代表的下一代通讯技术，就是在覆盖和速度的基础上，进一步满足寻址容量大、设备功耗低、信号延迟小和服务虚拟化的强劲需求。
      </span>
     </p>
     <p style="margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       但我们所处的物理世界是有其物理瓶颈的，当香农定理和冯·诺依曼计算机结构没有本质性突破，当sub-6G和毫米波无线信号的传输特性无法改变的时候，人类在通讯与计算技术上的进步节奏会逐渐到达一个瓶颈，此时更需要大家具备严谨与脚踏实地工作作风。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       一方面，让社会大众切实感受到技术进步所带来的好处，另一方面，下大力气在基础理论的研究上，争取能够突破目前技术发展的理论瓶颈。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       韦青提到，人类终将从互联网时代，进入到万物互联的时代，这对通讯技术的快速迭代，提出了非常高的要求。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       这种高要求，不是一代通讯技术的进步所能达到的。比对互联网的发展，互联网技术的变革始终处于一个进行时，也就不需要、也很难用某个具体的数字去描述某一代的进步。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       无线通讯技术尽管复杂程度高，模拟技术含量大，前进的步伐不会像数字技术那样快，会给人以“代差”的感觉。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       但随着软件定义通讯技术（SDN/SDR）的日趋成熟，无线通讯技术也会将像其它数字化技术一样，进入快速迭代的进程。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       在这种前提下，一旦将自己的进步贴上标签，就会有竞争对手拿一个更炫酷的名词来比对，比如5G刚刚火热起来，就会有人拿6G的话题抢风头。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       在这样的竞争环境下，过于局限于一种固定数字的定义，反而可能成为竞争取胜的包袱，因为未来技术的发展，会进入一个急速变化的小步快跑时代。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <strong>
        <br/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;line-height:1.75em;letter-spacing:1px;margin-left:8px;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/77f9597d084ecba6230e7a63cc8c0f58.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;line-height:1.75em;letter-spacing:1px;margin-left:8px;">
      <span style="font-size:18px;">
       <strong>
        微软早已布局5G
       </strong>
      </span>
     </p>
     <p style="margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       事实上，微软早已布局5G。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       2019年6月9日，在E3展会前夕，微软公司宣布将于10月开始在北美试上线云游戏，此举意味着微软将与11月发布云游戏的谷歌展开直接竞争。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       微软游戏业务的执行副总裁菲尔·斯宾塞（Phil Spencer），在被问及与谷歌的竞争时表示：“（我们）从数十年前开始就涉足游戏产业，与设计者和消费者保持着紧密关系”。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       他还表示，在致力于云游戏的同时，微软还将投放Xbox的新机型，双管齐下攻占游戏市场。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       有分析师表示，微软因为云端基础建设较好、有自己厂牌的游戏内容，相比谷歌等其他对手，更能运用云游戏潮流获利。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       据荷兰调查公司Newzoo的数据，游戏的全球规模在2021年达到1801亿美元，较2018年增长约30%。此前在游戏机主导的业界版图主要被微软、索尼、任天堂瓜分。
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而云技术带动的云游戏生态，预计将成为科技巨头们动摇固有版图的切口。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       日本经济新闻社文章指出，这次竞争将不仅限于游戏市场的框架之内。对于全球电脑的基础、越来越不可或缺的云服务主导权之争来说，游戏业的竞争很有可能产生深远影响。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       与视频和影像的单方向供应相比，游戏需要更高的技术。如果一家公司在游戏领域的云技术，领先于其他公司，那么它就会在随着5G普及，而扩大在云技术领域确立优势地位。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       不过，虽然微软在发力云游戏，但是未来乘风5G，必将爆发出更大的能量。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而且，不仅是在云游戏方面，在AI方面，微软扎根更深。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p style="text-align:center;letter-spacing:1px;margin-left:16px;line-height:1.75em;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/1d00b42e12d7dee761dc67eee8f7f380.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;letter-spacing:1px;margin-left:16px;line-height:1.75em;">
      <span style="font-size:18px;">
       <strong>
        微软在AI方面的专利也有利于自家5G发展
       </strong>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       据中国科技新闻援引外媒消息，目前，微软人工智能的专利已经超过18000项。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       这些专利，必将极大助理微软在5G方面的发展。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       因为5G如果和AI结合起来，就像鸟类的两只翅膀一样，会让业务非得更高。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       5G是网络，人工智能则像是接收网络的一个终端，如果人工智能没有网络，依靠已经设定好的程序，依旧能运行。但是失去了5G网络，人工智能将会失去从云端获取数据的能力。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       总而言之，微软公司仍将深度参与人类生活......
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       5G真的来了，于个人来说，你该如何抓住趋势？
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       6月29日，
       <strong>
        微软（中国）首席技术官韦青，
       </strong>
       即将在CSDN技术沙龙上和大家畅谈5G。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       此外，
      </span>
      <strong style="font-size:15px;">
       北京邮电大学信息与通信工程学院多媒体技术教研中心主任/博士生导师孙松林、金山云AIoT事业部高级研发总监肖江、爱立信中国研发部多天线高级专家朱怀松、爱立信中国研发部主任系统工程师刘阳
      </strong>
      <span style="font-size:15px;">
       等行业内顶尖的领军者、资深的技术专家们共聚一堂，共同探讨5G在物联网中的巨大潜能。
      </span>
     </p>
