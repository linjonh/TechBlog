    <h2>
     第一种方法：
    </h2>
    <blockquote>
     <p>
      <strong>
       前端传数组：
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-javascript">//定义一个数组
var majorList = [1,2,3,4];

//可以用表单或者ajax等方法将数组传给后端
JSON.stringify(majorList); //将该数组封装成JSNO</code></pre>
    <blockquote>
     <p>
      <strong>
       后端来接收：
      </strong>
     </p>
    </blockquote>
    <pre><code>//后端用用String类型 接收
String majorList;

JSONArray array = JSONArray.fromObject(majorList);//转换成Json
   for (int i = 0; i &lt; array.size(); i++) { //遍历输出
         System.out.println(array.get(i));
    }</code></pre>
    <blockquote>
     <p>
      <strong>
       结果：["1","2","3","4"]，然后可以转换成json遍历输出
      </strong>
     </p>
    </blockquote>

    <h2>
     第二种方法：
    </h2>
    <blockquote>
     <p>
      <strong>
       前端传数组
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-javascript">//定义一个数组
var majorList = [1,2,3,4];

//可以用表单或者ajax等方法将数组传给后端
majorList.toString(); //将该数组z转化为字符串</code></pre>
    <blockquote>
     <p>
      <strong>
       后端来接收：
      </strong>
     </p>
    </blockquote>
    <pre><code>//后端用用String类型 接收
String majorList;</code></pre>
    <p>
    </p>
    <blockquote>
     <p>
      <strong>
       结果：1,2,3,4，然后可以转换成json遍历输出
      </strong>
     </p>
    </blockquote>
    <div>
    </div>
    <p>
     如果这些是 id的话  可以拼接sql语句进行查询
    </p>
    <pre><code>//后端用用String类型 接收
String majorList;
String sql = "select * from user where id in ( " + majorList + " );";
System.out.println(sql);</code></pre>
