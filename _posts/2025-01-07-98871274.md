      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       前言
       <br/>
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       作为开发者我们都知道，开发后台语言可选择的方向会很多，比如，Java，go,Python,C/C++,PHP，NodeJs….等很多，这么多语言都有什么样的优势？如果让你学习一门后端语言，又该如何选择呢？
      </p>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       Java
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       编译语言，速度适中（2.67s），目前的大型网站都是拿java写的，比如淘宝、京东等。主要特点是稳定，开源性好，具有自己的一套编写规范，开发效率适中，目前最主流的语言。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       作为编程语言中的大腕。具有最大的知名度和用户群。无论风起云涌，我自巍然不动。他强任他强，清风拂山岗；他横由他横，明月照大江。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        Java可以做什么：
       </strong>
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       安卓和IOS的应用开发、视频游戏开发、桌面GUI、软件开发等等；
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        Java的优点：
       </strong>
      </p>
      <ul class="list-paddingleft-2" style="list-style-type:circle;margin-left:5px;">
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          Java开发人员需求量大：
         </span>
         <span style="font-size:15px;">
          这个是根据统计得出的。
         </span>
         <span style="font-size:15px;">
          JAVA在很多语言当中，是需求量最大的；
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          进化语言：
         </span>
         <span style="font-size:15px;">
          首先C++是基于C语言优化的，Java是被优化过来的。
         </span>
         <span style="font-size:15px;">
          而且在这人平台是增加了很多的功能，lambda等功能
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          安卓应用开发：
         </span>
         <span style="font-size:15px;">
          谷歌的安卓移动平台是世界第一的移动平台，编写安卓应用开发者使用的主要语言是Java；
         </span>
        </p>
       </li>
      </ul>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        Java的缺点：
       </strong>
      </p>
      <ul class="list-paddingleft-2" style="list-style-type:circle;margin-left:5px;">
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          使用大量的内存：
         </span>
         <span style="font-size:15px;">
          Java和C++相比使用更多的内存所以占用的内存就更大
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          学习曲线：
         </span>
         <span style="font-size:15px;">
          这边指的是Java虽然不是最简单的入门语言，但是也不是最难- -||
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          启动时间慢：
         </span>
         <span style="font-size:15px;">
          用java写过安卓的应用的人应该都知道。
         </span>
         <span style="font-size:15px;">
          同样的代码在模拟器中启动是非常缓慢的事情。
         </span>
         <span style="font-size:15px;">
         </span>
        </p>
       </li>
      </ul>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       c#
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       执行速度快（4.28），学习难度适中，开发速度适中。但是由于c#存在很多缺点，京东、携程等大型网站前身都是用c#开发的，但是现在都迁移到了java上。
      </p>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       C/C++
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       现存编程语言中的老祖，其他语言皆由此而生。执行速度最快无人能及。但是写起来最为复杂，开发难度大。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       C++目前是被看做编写大型程序应用最好的面向对象编程语言。C++是C语言的升级版本；
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        C++用来做什么：
       </strong>
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       搜索引擎、软件开发、操作系统、视频游戏等
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        C++的优点：
       </strong>
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       在熟练掌握了C++之后，处在各个位置当中都能成为佼佼者。使用C/C++的话，帮助我们更理解其他编程语言。比如说是Java或者是Python是如何进行内存管理以及如何处理缺陷；能调整性能：C++能允许调整你的应用性能以及影响所有计算机性能，它写起来的话比Java来说就并不是很友善，但是我们也可以利用C++做很多事情；
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        C++的缺点：
       </strong>
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       学习困难：C++相对来说学习难度很大，但是如果我们可以流畅使用的话，会圆圆超过其他的程序员；尺寸很大：C++可以称为巨大，它的大部分功能互相影响是机器复杂的方式。没有开发者是可以使用全部提供的组成部件。但是会发挥大量的时间来挑选你的程序的子部件的。
      </p>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       Javascript
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       编程语言中特立独行的傲娇美女。前端处理能力是其它语言无法比拟。发展中的js后端处理能力也是卓越不凡。前后端通吃，舍我其谁？
      </p>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       Python
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       脚本语言，速度最慢（258s），代码简洁、学习进度短，开发速度快。豆瓣就是拿python写的。Python著名的服务器框架有django，flask。但是python在大型项目上不太稳定，因此有些用python的企业后来迁移到了java上。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/a5e2e1c1e1080c101b3193615f222792.jpeg"/>
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       学完之后可以干什么：web开发、应用开发、大数据、数据挖掘、科学计算、机器学习、人工智能、运维、自然语言处理等等等。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        Python的优点：
       </strong>
      </p>
      <ul class="list-paddingleft-2" style="list-style-type:circle;margin-left:5px;">
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          易于学习：
         </span>
         <span style="font-size:15px;">
          就像是楼主说的，逛了下论坛发现做了几年Java或者是C++的，几天就可以写Python了。
         </span>
         <span style="font-size:15px;">
          但是这个不是什么坏事，入门来说，从简入难，或者从难入简，都是很好的选择。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          库：
         </span>
         <span style="font-size:15px;">
          库都是免费的，并且有很多库和函数把编程变得相对容易很多。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          物联网：
         </span>
         <span style="font-size:15px;">
          Python也许是会成为物联网当中最受欢迎的语言，我们都知道Instagram的后端的确是用Python开发的，用的还是Django
         </span>
        </p>
       </li>
      </ul>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        Python的缺点：
       </strong>
      </p>
      <ul class="list-paddingleft-2" style="list-style-type:circle;margin-left:5px;">
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          速度：
         </span>
         <span style="font-size:15px;">
          开发速度是快，比如java100行代码python20行就搞定了。
         </span>
         <span style="font-size:15px;">
          但是作为解释型的语言来说，比编译型语言的速度慢很多。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          移动端：
         </span>
         <span style="font-size:15px;">
          Python在移动计算方面是弱的，很少有智能机的应用是Python开发的
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          设计：
         </span>
         <span style="font-size:15px;">
          python是动态型的语言，需要更多的测试以及错误仅仅是在运行的时候展示的。
         </span>
        </p>
       </li>
      </ul>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       scala
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       编译语言，比python快十倍，和java差不多，但是学习进度慢，而且在实际编程中，如果对语言不够精通，很容易造成性能严重下降。，后来比如Yammer就从scala迁移到了java上。微服务框架有lagom等。
      </p>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       Go
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       编程界的小鲜肉。高并发能力无人能及。即具有像Python一样的简洁代码、开发速度，又具有C语言一样的执行效率，优势突出。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       go语言的难度，相对来说，GO语言不难的。可是GO语言的学习资料相对其他的语言来说很少，所以学习起来没有其他的语言那么便利；
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       很多人说GO语言在国内更火，按照数据来说，是的。因为中国人多，基数大。其实GO在国外更火。对于现在来说，GO实际上也已经站稳脚跟了。不管是Google自带光环也好，实际应用也好。go算是找到了属于自己的空间。
      </p>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <strong style="color:rgb(191,54,12);">
        GO语言的优点：
       </strong>
      </p>
      <ul class="list-paddingleft-2" style="list-style-type:circle;margin-left:5px;">
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          编译时间快：
         </span>
         <span style="font-size:15px;">
          GO语言编写最大的微服务的时间大概需要6秒，相对Java和C++呆滞的编译速度来说，GO语言快速编译是主要的效率优势。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          并发性和通道：
         </span>
         <span style="font-size:15px;">
          GO语言的logo大家可以了解一下，它就是致力于事情简单化，也就是快。
         </span>
         <span style="font-size:15px;">
          其实并没有引入很多的新的概念。
         </span>
         <span style="font-size:15px;">
          就是打造一门简单的语言，使用起来很快。
         </span>
         <span style="font-size:15px;">
          在goroutine上运行一个函数最小的样板代码，我们只需要使用关键词go添加函数调用
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          生态系统也是很强大的：
         </span>
         <span style="font-size:15px;">
          面向Redis、RabbitMQ、Template等等很多稳定的库。
         </span>
         <span style="font-size:15px;">
          有很强大的工具支持。
         </span>
        </p>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          <strong style="color:rgb(191,54,12);">
           GO语言的缺点：
          </strong>
          <strong style="color:rgb(191,54,12);">
          </strong>
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          缺少框架：
         </span>
         <span style="font-size:15px;">
          GO是没有一个主要的框架。
         </span>
         <span style="font-size:15px;">
          但是很多人认为不应该从框架的使用开始。
         </span>
         <span style="font-size:15px;">
          也可以从社区的讨论了解一下这个问题。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          错误处理：
         </span>
         <span style="font-size:15px;">
          在错误处理方式，很容易丢失错误发生范围，所以在编程过程中很难向用户提供出有意义的错误信息。
         </span>
        </p>
       </li>
       <li>
        <p style="line-height:2em;">
         <span style="font-size:15px;">
          软件包管理：
         </span>
         <span style="font-size:15px;">
          在默认的情况下，没有办法制定特定版本的依赖库。
         </span>
         <span style="font-size:15px;">
          、也没有办法创建可以复写的builds。
         </span>
        </p>
       </li>
      </ul>
      <h4 style="font-size:18px;letter-spacing:1.5px;line-height:2em;color:rgb(0,150,136);text-align:left;">
       最后
      </h4>
      <p style="font-size:15px;letter-spacing:1.5px;line-height:2em;">
       <span style="font-size:15px;letter-spacing:1.5px;">
        上面列举了几种常见的语言，其实还有很多。
       </span>
       最后注意的是：没必要纠结语言，它只是工具，而且不同的语言针对的场景也不同。如果是创业公司，从快速开发来说，应该选择PHP或Python；同样如果是体量较大的成熟应用，对于不同的业务层，会有多个语言，如Java，Go等。
