    <p>
     xcode是一款运行在Mac OS X上的集成开发工具，具有统一的用户界面设计，且编码、测试、调试都在一个简单的窗口内完成，还能够与Swift编程语言相结合，使开发应用程序比以往更轻松，更有趣，便有许多用户也想要进行了解和尝试一下，然而苹果电脑价格昂贵，并不是每一个人都可以承受的，为此小编带来了
     <strong>
      <a href="http://www.ddooo.com/softdown/181799.htm" rel="nofollow">
       xcode windows版
      </a>
     </strong>
     ，通过虚拟机软件VMWare可正常使用该软件。同时，xcode提供了项目编辑、搜索和浏览，文件编辑，项目构建和调试设备等功能，并兼容CodeWarrior风格在线编码汇编，用户可以把CodeWarrior项目文件快速方便地转移到软件中，不仅如此，还支持开发人员使用 C、 C++、Objective C、 AppleScript 和 Java，可用来辅助开发应用程序、工具、架构、数据库、嵌入包、核心扩展和设备驱动程序，功能非常强大。
     <br/>
     除此之外，xcode甚至能够和Mac OS X里众多其它的工具协作，例如编译器如 gcc、javac 和jikes，及调试工具gdb，而且可以用AppleScript Studio组增加一个Aqua界面到系统和应用程序脚本、命令行工具、以及网络应用程序中，无论如何都能在任何平台组成最强大的脚本环境。但因安装过程过于繁琐，下文奉上了详细的安装图文教程，新手用户只要严格地按照本教程一步步操作，即使不了解苹果电脑，也能顺利地安装并运行Mac OS X，亲测有效，欢迎有需要的小伙伴们前面下载体验。
     <br/>
     <img alt="xcode windows版" height="418" src="https://i-blog.csdnimg.cn/blog_migrate/e088ac074a45c643a2227818812cf211.jpeg" width="600"/>
    </p>
    <p>
     xcode windows版安装教程：
    </p>
    <p>
     一、准备工作
     <br/>
     1、在本站下载解压，得到VmWare、Unlocker补丁工具，及Mac OS X懒人版光盘映像；
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2347bf885033d678fc2aeca88c3f6c88.jpeg">
      <br/>
      2、首先需要一个Apple ID，如果没有的话，请前来此网址申请一个：https://support.apple.com/zh-cn/apple-id、
      <br/>
      二、安装VMWare并打补丁
      <br/>
      3、接下来安装VMWare，用户可直接在本站进行下载：http://www.ddooo.com/softdown/165956.htm；
      <br/>
      提示：有详细的安装激活教程
      <br/>
      4、安装好之后，需要给VMWare打补丁，因为VMware本身不支持Mac OS X的安装，需要打个补丁才能安装Mac OS X；
      <br/>
      5、将软件包中的Unlocker进行解压，以管理员的身份运行"win-install.cmd"，会弹出一个命令行窗口，待补丁打好以后，命令行窗口会自动关闭；
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/75f2e9021f2d1ffed22ff525bd2b038a.jpeg">
       <br/>
       三、为Mac OS X创建虚拟机
       <br/>
       6、双击桌面上的VMWare Pro图标，启动VMWare，单击file→New Virtual...新建虚拟机；
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/77417535f8d780250afc863ba7b74158.jpeg">
        <br/>
        7、选择虚拟机的类型，推荐“典型”（Typical (recommended)）；
        <br/>
        <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d7a190674c145721c714bc5c66acf5c8.jpeg">
         <br/>
         8、要求你指定操作系统的安装光盘，选择“安装程序光盘映像文件（iso）”（Installer disc image file (iso)）；
         <br/>
         提示：点击Browse，选择软件包中的"OS X 10.11.1(15B42).cdr"
         <br/>
         <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/08001e307c7d3e1dad3c8fa11f14059a.jpeg">
          <br/>
          9、接下来选择虚拟机的操作系统。选择“Apple Mac OS X”，再将“版本”（Version）设为“OS X 10.11”；
          <br/>
          <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1a9f451a4939ff0cbd66227e7a0a097a.jpeg">
           <br/>
           10、指定虚拟机的位置，但是，你要把虚拟机的位置（Location）记录下来，后面会用到；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/dbbffccc016582b5bdf23979b9adf918.jpeg"/>
           <br/>
           11、指定虚拟磁盘，选中Split virtual disk into multiple files；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/48b68561e030851b31612c923009b924.jpeg"/>
           <br/>
           12、确认虚拟机的硬件配置，若需要调整，即可点击Customize Hardware；
           <br/>
           注意：根据小编的经验，要运行Mac OS X，虚拟机至少要有4 GB内存，为了流畅运行虚拟机，你的电脑（宿主机）至少要有6GB内存。笔者的电脑有12GB内存。
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5e3880a83a4bdb061f92a91527768739.jpeg"/>
           <br/>
           13、此时VMWare会进入虚拟机页面，点击Power on this virtual machine开启此虚拟机；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a1a0ec1f421e6c5b5ff090486cce8a89.jpeg"/>
           <br/>
           14、会提示错误，需要手动修改虚拟机的隐藏参数设置错误；
           <br/>
           说明：用资源管理器打开第十步骤中虚拟机的位置，把OS X 10.11.vmx文件拖进记事本窗口，找到smc.present = "TRUE"这一行，在它后面添加一行，内容为smc.version = 0，保存退出
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6130931ae32bc1cab3d1ac3c65414297.jpeg"/>
           <br/>
           四、安装Mac OS X
           <br/>
           15、再回到VMWare开启此虚拟机，就能正常启动了；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f5561ea7c45e720b235730bcca5849b6.jpeg"/>
           <br/>
           16、虚拟机屏幕上会显示欢迎界面，选择“以简体中文作为主要语言”，点击对话框底部圆圈里的右箭头；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a80250615117b5000c002b01998cb52c.jpeg"/>
           <br/>
           17、因为我们还没抹掉虚拟机的硬盘，需要点击屏幕顶部a菜单栏上的“实用工具"→“磁盘工具”，启动“磁盘工具”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/996d215021e9c11f31b0934c6b3cf5b1.jpeg"/>
           <br/>
           18、先在左侧栏里选中“Vmware Virtual SA……”，然后点击工具栏里的“抹掉”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e1542a8ca119dcffd4a6c9ffc4867dc6.jpeg"/>
           <br/>
           19、此时，系统会弹出一个对话框，为虚拟硬盘指定名称、格式、方案；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ea192b0ba87e5714e3733d8bdcc20541.jpeg"/>
           <br/>
           20、静待进度条走完，然后点击“完成“，点击屏幕顶部菜单栏上的“磁盘工具”→“退出”，回到步骤17所示的界面；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/421bef96372a23eaa2f65ad18e7e3112.jpeg"/>
           <br/>
           21、同意许可协议，选择把Mac OS X安装到哪里；
           <br/>
           提示：假如我们没有抹掉虚拟硬盘，就会卡在这一步
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/26d0755987f2ba590918059f34836d12.jpeg"/>
           <br/>
           22、就静待漫长的安装过程，等待进度条走完；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c2f7df9cffe9f421dc05aef0e07e7904.jpeg"/>
           <br/>
           23、进度条走完后，虚拟机会重启，会让你选择区域；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a82df20609af165f3e3b73460ba73c07.jpeg"/>
           <br/>
           24、要求你选择键盘，将键盘布局设为“简体中文”，输入法选中“简体拼音”，然后点击“继续”即可；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b38cee4b5140761cd77ee98b3b6d84d0.jpeg"/>
           <br/>
           25、“传输信息到当前Mac”类似Windows的“数据迁移助手”，选择“现在不传输任何信息”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e0be52e64794db0db1f12874ec72fb5f.jpeg"/>
           <br/>
           26、是否启用定位服务，准备好的Apple ID填入；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/cec44ab0ef90003f94eccb08ca63bd8e.jpeg"/>
           <br/>
           27、要求你同意安装条款，并创建一个用于登录的账户，选择“使用我的iCloud账户登录”和“基于当前位置设定时区”，点击“继续”，然后等待系统创建账户；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2f94cb44b9ee302cb0fd009f14d134f0.jpeg"/>
           <br/>
           28、弹出弹框询问你是否开启“查找我的Mac”，点击“允许”，是否设置iCloud钥匙串，选择“稍后设置”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b92d9511d65062ba1796aed13e73a834.jpeg"/>
           <br/>
           29、在这里，选择“升级到iCloud Drive”，最后，询问你是否将使用数据发送给苹果，可选可不选，静待系统设置你的Mac；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5981780e92fe935e4d4022efb0df963d.jpeg"/>
           <br/>
           五、安装VMWare Tools、更新系统到最新版本
           <br/>
           30、将右击桌面上的光盘图标，选择“推出OS X Base System”；然后，点击虚拟机窗口菜单栏上的“虚拟机”→“安装VMWare Tools”（VM → Install VMWare Tools）；
           <br/>
           提示：它可以改善虚拟机的显示，并且有助于宿主机和虚拟机之间交换数据
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/05e0a23b10fa8a476033c418500cd6cc.jpeg"/>
           <br/>
           31、虚拟机桌面上弹出一个“VMWare Tools”窗口，双击“安装VMWare Tools”，弹出“安装VMWare Tools”向导，系统要求你输入密码。输入密码（这里的密码是你的Apple ID密码）以后，点击“安装软件”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/fee606f8aeadc0fbe2868f817ce97b30.jpeg"/>
           <br/>
           32、点击“继续安装”，最后点击“重新启动”，静待虚拟机重启即可；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/fb0742c88213bf6a07aebc55c7be7833.jpeg"/>
           <br/>
           33、虚拟机重启后会进入这个登录界面，选择你的Apple ID对应的邮箱，输入密码，进行登录，点击屏幕左上角的苹果图标，选择“关于本机”——“软件更新”；
           <br/>
           <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/752972296fcd9a3bd00292161149ddc9.jpeg"/>
           <br/>
           34、然后，系统会进入AppStore，并会检查更新，找到更新包以后就会显示出来，你只需要点击“更新”按钮，就会进入更新过程，更新结束后会重启虚拟机。
           <br/>
           35、安装好环境之后，用户可自行下载安装xcode啦。
