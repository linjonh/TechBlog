    <h3>
     <a id="_2">
     </a>
     个人情况
    </h3>
    <p>
     西安某211的2018级本科生，本科专业是网络空间安全，当时是被调剂的这个专业。到大三保研的时候才觉得网安专业保研还比较轻松，相对于计算机科学啥的轻松多了，专业也不是很卷。总结起来就是：
     <strong>
      网安真香
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       排名
      </strong>
      ：2/56
     </li>
     <li>
      <strong>
       四六级
      </strong>
      ：四级507，六级474
     </li>
     <li>
      <strong>
       竞赛获奖
      </strong>
      ：数模省一、数学竞赛二等奖、数模校赛一等
     </li>
     <li>
      <strong>
       项目经历
      </strong>
      ：无人机通信的项目（信安作品赛做的，混个经历哈哈）
     </li>
     <li>
      <strong>
       夏令营情况
      </strong>
      ：申请了
      <strong>
       中科大、信工所3室、北邮、武大、南开
      </strong>
      ，入营了
      <strong>
       中科大、信工所3室、北邮、武大
      </strong>
      ，拿到offer的有
      <strong>
       中科大、北邮、武大
      </strong>
      ，等差数列可还行。
     </li>
    </ol>
    <h3>
     <a id="_10">
     </a>
     武汉大学网安报名
    </h3>
    <ul>
     <li>
      报名情况
     </li>
    </ul>
    <p>
     武大是我当时第二想去的学校，所以夏令营一出来我就赶快报名了，当时截止时间是
     <strong>
      2021年7月9日
     </strong>
     。当时夏令营之前联系了两个老师，他们都表示夏令营过了再找我。夏令营入营结果出来之后，入了大概有205个人，但是由于武大夏令营是线下，而且与上交夏令营撞了，所以好多大佬去了上交，实际参加武大夏令营的也就是150人左右。
    </p>
    <ul>
     <li>
      需要交的表
     </li>
    </ul>
    <p>
     武大需要交的表并不是很多，需要夏令营开营之前当面上交，需要以下表格：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b1d30df252906ba52d0b40d526008f9.png"/>
    </p>
    <ul>
     <li>
      报销与住宿情况
     </li>
    </ul>
    <p>
     报销的话，只报销铁路
     <strong>
      硬卧
     </strong>
     级别，已经算比较好的了（上交啥也不报，可能是因为给的offer少校方觉得亏，
     <strong>
      武大优营率还是比较高，能达到60-70%
     </strong>
     ）。住宿条件的话，学校在外面包了俩宾馆（丽枫酒店和城市便捷），两人一间，总体上来说还是比较不错，距离学校不算近，但是学校又包了几辆大巴接送。来的第一天晚上，我看了一下分宿舍的情况，我是和一个北邮的兄弟（那兄弟和最近进监狱的那个明星名字贼像，笑死我），结果最后不知咋的当天成我一人一间了，贼离谱，而且浴室灯关不了，还一闪一闪的，总体房间呈现一种很吓人的气氛，没咋睡好觉。。。。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a8bc37f3dd12ea948baa99aa0cb91c3b.jpeg">
      <br/>
      更离谱的是，住进去第一天晚上，我门口就有人往门缝里塞小广告，具体是啥广告我就不说了。。我以为现在治安条件这么好，这种情节只会出现在电影里，没想到啊。我和其他三个同学一块来的，晚上去楼下饭店吃了个饭，那饭感觉也不咋样，米饭感觉都有点馊，总之来武汉的第一天感觉不太好。
     </img>
    </p>
    <h3>
     <a id="_24">
     </a>
     夏令营过程
    </h3>
    <p>
     先给大家瞅瞅网安的日程安排。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/29a76bc49217dcfd0ff670ea9e06b398.png"/>
    </p>
    <ul>
     <li>
      <strong>
       Day1-----7月19日
      </strong>
      <br/>
      我是7月18号就住了武大安排的酒店，晚上吃了个饭也没出来玩，然后明儿一早（7月19日）就坐了大巴去了武大。首先就被拉到一个教室交材料和火车票（方便报销），武大网安学院教室都不戳，都是那种扇形的，我以前也没见过这个。然后中午我们四个加上一个同学的舍友（一个福州大学的兄弟）一起去食堂吃了饭，也没回宾馆，吃完饭直接去了武大网安图书馆逛了逛。他这个图书馆是真的好，听其他武大兄弟说
      <strong>
       期末都会有很多位置，根本坐不满
      </strong>
      ，而且武大真的有钱，投影电脑、各种大屏电脑随便用，真幸福啊！哭了。。。
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5f1ff760245b776f285229a83d64b380.jpeg">
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/431e05790ebd544830c82f4942e66bb2.jpeg">
        <br/>
        <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ca0b78a18917e829dad1fc23f76275ba.jpeg">
         <br/>
         之后就去了图书馆报告厅，发现武大老师正在准备会场（因为下午宣讲与考试都在这里），然后他们建议我可以先去武大网安宿舍逛逛，贼好。武大网安宿舍我以前在网上看过视频，当时很震撼，本科生三四年级、硕士生是两人一间、博士生是一人一间，我们之后就混进了武大宿舍，确实和网上没啥区别。但是当时忘了拍照了。。。。。
         <strong>
          六个房间公用一套厨房、洗衣机和冰箱
         </strong>
         ，
         <strong>
          然后每个房间都有一个阳台，关键是他们的家具都是那种木制的（感觉像），所以看上去很高级，然后进门的时候也是用指纹（但是门常开，感觉没啥用），上下楼也是电梯
         </strong>
         。估摸着他们一层楼也就住
         <strong>
          48个人
         </strong>
         ，真离谱！（武大还是太有钱了）
         <br/>
         关于他们的食堂呢，感觉就稍微差了点，
         <strong>
          食堂的位置其实挺挤，付账的方式可以微信或支付宝，他们的食物种类比较少，菜卖的也相对较贵，而且由于校区比较偏，所以不能点外卖
         </strong>
         （可能能点，我也不是很清楚，反正这两天没看到外卖员）。但是下午宣讲老师说武大网安学院
         <strong>
          每个学生，每个月都有800的饭食补助！
         </strong>
         。。。。这就算食堂差了点也值了。
         <br/>
         <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dc1b3357f74bf7592cb342c2b8e716b6.jpeg">
          <br/>
          <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/375c7a990243451b809f4755a93b670e.jpeg">
           <br/>
           到了下午，首先作了两场
           <strong>
            学术报告
           </strong>
           ，一个女老师一个男老师（欧长海，很年轻的教授），分别讲了啥我也没仔细听，男老师好像讲的密码芯片啥的，我当时太困了，听着听着就想睡觉。
           <br/>
           之后，就开始了
           <strong>
            笔试
           </strong>
           。笔试主要考
           <strong>
            计算机网络、密码学、还有手写代码，题目并不是很难
           </strong>
           。我记得代码部分主要有
           <strong>
            找代码错误
           </strong>
           的，计算机网络里面考了三四个简答题，具体考的我也记不清了，反正挺简单的。
           <br/>
           回到宾馆就大概看了看自己的简历，然后稍微背背自己准备的英语问题（我英语不是很好，所以提前准备的），准备明儿的面试。
          </img>
         </img>
        </img>
       </img>
      </img>
     </li>
     <li>
      <strong>
       Day2-----7月20日
      </strong>
      <br/>
      <strong>
       明天一天都是面试，面试分为7个组，我是分在了第二组，不同的组没啥区别。我进去之后，里面有6个老师，我首先把自己的简历给了老师们一人一份，然后老师让我英语自我介绍，还让我英语介绍自己的项目（我就说了一些套话），他们也没问啥项目问题，主要问了我的数学建模（关于新冠病毒传播），然后最后问我又报了哪些学校，面试很快就完事儿了，感觉也就9分钟左右
      </strong>
      。
      <br/>
      然后就是晚上去了武汉市内玩，江边是真的凉快（
      <strong>
       强烈建议坐一下轮渡码头，虽然人多但是10块钱也值了
      </strong>
      ），江汉街也很不错，很多建筑都有民国的感觉哈哈哈·。
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b612a1b43097ce14cc64518efd129c17.jpeg"/>
     </li>
    </ul>
    <h3>
     <a id="_43">
     </a>
     最终结果
    </h3>
    <p>
     那几天郑州发水灾，好多列车都停运了，我回西安的火车也不例外。所以又在武汉多呆了几天，那几天主要是准备中科大线上的面试，最终终于辗转回到了学校，在回西安途中优营名单就公布了（
     <strong>
      在官网上公布，还是很快的，具体是面试分40%，笔试20%，背景分40%，背景分差别都不是很大，笔试虽然只占20%但也是很重要的
     </strong>
     ），我也很幸运成为优营，但是最终还是没有联系老师，因为科大才是我最终的目标。
    </p>
