    <h2>
     <a id="_1">
     </a>
     登陆终端
    </h2>
    <p>
     输入 java -version 查看java的版本
     <br/>
     输入 java -verbose 查看java的安装环境路径
    </p>
    <p>
     没有的话，自己去搜一下怎么安装java的jdk，记得选1.18版本
    </p>
    <p>
     安装路径在这里
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7402fe38ac9edab440a97498faaf65d4.png"/>
    </p>
    <p>
     安装完jdk后，输入
     <br/>
     /usr/libexec/java_home -V
     <br/>
     查看jdk版本
    </p>
    <h2>
     <a id="JDK_16">
     </a>
     复习什么是JDK
    </h2>
    <p>
     Java Development Kit
     <br/>
     开发工具包
     <br/>
     包含 Java开发工具和JRE
     <br/>
     java开发工具： 编译工具javac.exe， jar.exe打包工具等。
    </p>
    <p>
     JRE= JVM+java SE 标准类库
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c87b69996d083991b4561852d6fe50ad.png"/>
    </p>
    <h2>
     <a id="_24">
     </a>
     整理安装步骤
    </h2>
    <ol>
     <li>
      下载VSCODE
      <br/>
      https://code.visualstudio.com/
     </li>
     <li>
      下载jdk 8.0版本
      <br/>
      https://www.oracle.com/java/technologies/downloads/#java8-mac
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/37573a73151900941198f3e89f5dfe79.png"/>
     </li>
     <li>
      配置环境变量
      <br/>
      touch .bash_profile
      <br/>
      open -e .bash_profile
      <br/>
      在里面添加
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span>/Library/Java/JavaVirtualMachines/jdk1.8.0_172.jdk/Contents/Home
