     <h3>
      了解 Uni-App
     </h3>
     <p>
      <code>
       uni-app
      </code>
      是一个使用
      <code>
       Vue.js
      </code>
      开发所有前端应用的框架，开发者编写一套代码，可发布到iOS、Android、H5、以及各种小程序（微信/支付宝/百度/头条/QQ/钉钉）等多个平台。
     </p>
     <p>
      即使不跨端，
      <code>
       uni-app
      </code>
      同时也是更好的小程序开发框架。
     </p>
     <p>
      <code>
       uni-app
      </code>
      是由
      <code>
       DCloud
      </code>
      公司打造的，
      <code>
       DCloud
      </code>
      公司拥有340万开发者用户，旗下
      <code>
       uni-app
      </code>
      有5万+案例、600+插件、50+微信/qq群、更高的百度指数，可以放心选择。
     </p>
     <p>
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/uni0124.png"/>
     </p>
     <h3>
      为什么要用 Uni-App
     </h3>
     <p>
      uni-app在开发者数量、案例、跨端抹平度、扩展灵活性、性能体验、周边生态、学习成本、开发成本等8大关键指标上拥有更强的优势。
     </p>
     <p>
      1、开发者/案例数量更多： 5万+案例、600+插件、50+微信/qq群、更高的百度指数，跨端完善度更高，真正落地的提高生产力
     </p>
     <p>
      2、平台能力不受限： 在跨端的同时，通过条件编译+平台特有API调用，可以优雅的为某平台写个性化代码，调用专有能力而不影响其他平台。支持原生代码混写和原生sdk集成。
     </p>
     <p>
      3、性能体验优秀：体验更好的Hybrid框架，加载新页面速度更快。App端支持weex原生渲染，可支撑更流畅的用户体验。小程序端的性能优于市场其他框架。
     </p>
     <p>
      4、周边生态丰富：丰富的插件市场，各种模板拿来即用。支持NPM、支持小程序组件和SDK、兼容mpvue组件和项目、兼容weex组件。微信生态的各种sdk可直接用于跨平台App。
     </p>
     <p>
      5、学习成本低： 基于通用的前端技术栈，采用vue语法+微信小程序api，无额外学习成本。
     </p>
     <p>
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/run1x7.png"/>
     </p>
     <h3>
      快速开始
     </h3>
     <h4>
      一、HBuilderX 下载安装
     </h4>
     <p>
      可视化的方式比较简单，HBuilderX内置相关环境，开箱即用，无需配置nodejs。
     </p>
     <p>
      开始之前，开发者需先下载安装如下工具：
     </p>
     <p>
      HBuilderX：
      <a href="http://www.dcloud.io/hbuilderx.html" rel="nofollow">
       官方IDE下载地址
      </a>
     </p>
     <p>
      HBuilderX是通用的前端开发工具，但为
      <code>
       uni-app
      </code>
      做了特别强化。
     </p>
     <h4>
      二、创建项目
     </h4>
     <p>
      在点击工具栏里的文件 -&gt; 新建 -&gt; 项目：
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/create1.png"/>
     </p>
     <p>
      选择
      <code>
       uni-app
      </code>
      ，输入工程名，并且你可以从模板里的 你喜欢的模板 即可体验官方示例。最后点击创建，即可成功创建
      <code>
       uni-app
      </code>
      项目。
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816152753.jpg"/>
     </p>
     <p>
      官方模板地址：
      <a href="http://ext.dcloud.net.cn/?cat1=3&amp;cat2=33" rel="nofollow">
       DCloud 插件市场
      </a>
     </p>
     <p>
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816153205.jpg"/>
     </p>
     <p>
      我这里选择“电商模板min-amll”创建项目
     </p>
     <h4>
      三、运行uni-app项目
     </h4>
     <p>
      1、浏览器运行：进入项目，点击工具栏的运行 -&gt; 运行到浏览器 -&gt; 选择浏览器，即可在浏览器里面体验
      <code>
       uni-app
      </code>
      的 H5 版。
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/run-chrome.png"/>
     </p>
     <p>
      预览图：
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816153543.jpg"/>
     </p>
     <p>
      2、真机运行：连接手机，开启USB调试，进入项目，点击工具栏的运行 -&gt; 真机运行 -&gt; 选择运行的设备，即可在该设备里面体验uni-app。
     </p>
     <p>
      （1）、android手机，USB连接后，打开开发者模式
     </p>
     <p>
      （2）、ios 手机，电脑先下载
      <code>
       iTunes
      </code>
      ，然后USB连接电脑，添加信任设备即可。
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816153936.jpg"/>
     </p>
     <p>
      这里我们以ios为例。
     </p>
     <p>
      3、手机上面已经安装了一个名叫“HBuilder”的APP了，这时候点击是打不开的，需要我们ios手机 -&gt; 设置 -&gt; 通用 -&gt; 设备管理 -&gt; 添加信任。为什么要这样？因为ios必须要证书，正在发布的时候，就需要去apple申请证书，上线后，就不需要这样操作了。
     </p>
     <p>
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816154838.png"/>
     </p>
     <p>
      4、我们可以打开手机上面的APP了
      <img alt="Vue.js 一套代码，发布到iOS、Android、H5、以及各种小程序" src="http://cdn.javanx.cn/wp-content/themes/lensnews2.2/images/201909/20190816154841.png"/>
      这就是手机上面打开的样子，是不是感觉非常nice了？
     </p>
     <h3>
      体验原生功能
     </h3>
     <h4>
      扫码
     </h4>
     <p>
      通过上图，我们看到左上角有一个扫描，但是模板只写了事件，并没有实现，所以我们添加功能进去。 pages/index/index.vue
     </p>
     <pre><code class="language-javascript">//点击导航栏 buttons 时触发
onNavigationBarButtonTap(e) {
	const index = e.index;
	if (index === 0) {
		this.$api.msg('点击了扫描');
	} else if (index === 1) {
		// #ifdef APP-PLUS
		const pages = getCurrentPages();
		const page = pages[pages.length - 1];
		const currentWebview = page.$getAppWebview();
		currentWebview.hideTitleNViewButtonRedDot({
			index
		});
		// #endif
		uni.navigateTo({
			url: '/pages/notice/notice'
		})
	}
}
</code></pre>
     <p>
      阅读：
      <a href="https://uniapp.dcloud.io/api/README" rel="nofollow">
       uni-app api
      </a>
      ，设备 - 扫码 文档。操作很简单
     </p>
     <pre><code class="language-javascript">uni.scanCode(OBJECT)
// 调起客户端扫码界面，扫码成功后返回对应的结果。
</code></pre>
     <p>
      OBJECT:
     </p>
     <p>
      (1)、onlyFromCamera Boolean 否 是否只能从相机扫码，不允许从相册选择图片 头条小程序不支持
     </p>
     <p>
      (2)、scanType Array 否 扫码类型，参数类型是数组，二维码是'qrCode'，一维码是'barCode'，DataMatrix是‘datamatrix’，pdf417是‘pdf417’。 头条小程序不支持
     </p>
     <p>
      (3)、success Function 否 接口调用成功的回调，返回内容详见返回参数说明。
     </p>
     <p>
      (4)、fail Function 否 接口调用失败的回调函数（识别失败、用户取消等情况下触发）
     </p>
     <p>
      (5)、complete Function 否 接口调用结束的回调函数（调用成功、失败都会执行）
     </p>
     <pre><code class="language-javascript">onNavigationBarButtonTap(e) {
	const index = e.index;
	if (index === 0) {
		this.$api.msg('点击了扫描');
		uni.scanCode({
			success: (result) =&gt; {
				this.$api.msg('扫描成功' + result);
			},
			fail: (res) =&gt; {
				this.$api.msg('扫描成功' + res);
			}
		})
	} else if (index === 1) {
		...
	}
}
</code></pre>
     <p>
      这样就能调起扫描功能了。
     </p>
