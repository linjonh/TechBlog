    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">
       一、
      </span>
      <span style="color:#191919;">
       Jupyter
      </span>
      <span style="color:#191919;">
       介绍
      </span>
     </strong>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Jupyter Notebook
     </span>
     <span style="color:#191919;">
      是以
     </span>
     <span style="color:#191919;">
      web
     </span>
     <span style="color:#191919;">
      交互式的编程接口，是
     </span>
     <span style="color:#191919;">
      IPython notebook
     </span>
     <span style="color:#191919;">
      的升级版本。主要是针对
     </span>
     <span style="color:#191919;">
      python
     </span>
     <span style="color:#191919;">
      ，另外支持运行
     </span>
     <span style="color:#191919;">
      40
     </span>
     <span style="color:#191919;">
      多种编程语言。
     </span>
     <span style="color:#191919;">
      Jupyter
     </span>
     <span style="color:#191919;">
      可以在个人机器开发，也可以连接到集群中使用分布式计算引擎
     </span>
     <span style="color:#191919;">
      spark
     </span>
     <span style="color:#191919;">
      等以及数据库（
     </span>
     <span style="color:#191919;">
      mysql/hive/hdfs
     </span>
     <span style="color:#191919;">
      ）。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Jupyter
     </span>
     <span style="color:#191919;">
      相对于其他
     </span>
     <span style="color:#191919;">
      python
     </span>
     <span style="color:#191919;">
      编程工具来说，除了通常的新建、删除、更改、下载编程文件外，还支持在线编程运算可帮助持续开发，特别在企业中有些项目需要持续很长时间的开发，每天下班后关闭
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      ，只要服务器的
     </span>
     <span style="color:#191919;">
      kernel
     </span>
     <span style="color:#191919;">
      不关闭，
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      会保存好开发时的数据缓存和可视化结果，第二天继续开发，而不用又从头再运行，再配合支持
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      的功能，使得
     </span>
     <span style="color:#191919;">
      jupyternotebook
     </span>
     <span style="color:#191919;">
      方便记录开发过程和教学。下面对特征工程的教程就是利用了在
     </span>
     <span style="color:#191919;">
      jupyternotebook
     </span>
     <span style="color:#191919;">
      中使用
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      生成的，并且支持对文章的内容字体高亮。
     </span>
    </p>
    <br/>
    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">

      </span>
      <span style="color:#191919;">

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      anaconda
     </span>
     <span style="color:#191919;">
      ，里面有自带的
     </span>
     <span style="color:#191919;">
      jupyter notebook
     </span>
     <span style="color:#191919;">
      。
     </span>

    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      第二种是使用
     </span>
     <span style="color:#191919;">
      pip
     </span>
     <span style="color:#191919;">

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">

     <span style="color:#191919;">
     </span>

     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#000000;">
      如果网络不可达，可以多试几次
     </span>
    </p>
    <span style="color:#000000;">
     执行
     <span style="color:rgb(0,0,0);background-color:rgb(255,255,255);">

     </span>
     jupyter notebook， jupyter notebook
    </span>
    <span style="color:#000000;">
     的编辑器就启动了
    </span>
    <br/>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      python
     </span>

     </span>
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">

     <span style="color:#191919;">

     <span style="color:#191919;">

      下已经安装了
     </span>
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      ，可以使用
     </span>
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      创建特制
     </span>
     <span style="color:#191919;">
      python
     </span>
     <span style="color:#191919;">
      环境，

     <span style="color:#191919;">
      。
     </span>

    <p align="left" style="background:#FFFFFF;">

     </span>
     <span style="color:#191919;">
      启动

     <span style="color:#191919;">

     <span style="color:#191919;">

    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">

    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      进入
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      界面如下：
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      红色部分
     </span>
     <span style="color:#191919;">
      Python2
     </span>
     <span style="color:#191919;">
      就是新建一个
     </span>
     <span style="color:#191919;">
      notebook
     </span>
     <span style="color:#191919;">
      ，另外
     </span>
     <span style="color:#191919;">
      Terminals Unavailable
     </span>
     <span style="color:#191919;">
      是启动终端，这里
     </span>
     <span style="color:#191919;">
      linux
     </span>
     <span style="color:#191919;">
      和
     </span>
     <span style="color:#191919;">
      windows
     </span>
     <span style="color:#191919;">
      系统略有不同，由于
     </span>
     <span style="color:#191919;">
      windows
     </span>
     <span style="color:#191919;">
      下不支持类似
     </span>
     <span style="color:#191919;">
      linux
     </span>
     <span style="color:#191919;">
      的
     </span>
     <span style="color:#191919;">
      tty(
     </span>
     <span style="color:#191919;">
      控制终端
     </span>
     <span style="color:#191919;">
      )
     </span>
     <span style="color:#191919;">
      ，因此无法使用，
     </span>
     <span style="color:#191919;">
      Linux
     </span>
     <span style="color:#191919;">
      下启动效果如下：
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Jupyter
     </span>
     <span style="color:#191919;">
      能够下载文件，上传文件，但是无法在两个文件夹之间进行复制和移动，一种方式是把文件先下载下来，然后在另外文件夹中上传，比较麻烦，如果配合
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">

     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">
       3.2jupyter
      </span>
      <span style="color:#191919;">
       的使用功能
      </span>
     </strong>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      有三种类型的
     </span>
     <span style="color:#191919;">
      cells
     </span>
     <span style="color:#191919;">
      ：
     </span>
     <span style="color:#191919;">
      code cells,markdown cells
     </span>
     <span style="color:#191919;">
      ，
     </span>
     <span style="color:#191919;">
      raw cells
     </span>
     <span style="color:#191919;">
      ，常用的是
     </span>
     <span style="color:#191919;">
      code cells
     </span>
     <span style="color:#191919;">
      和
     </span>
     <span style="color:#191919;">
      markdown cells
     </span>
     <span style="color:#191919;">
      类型。
     </span>

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      ESC
     </span>
     <span style="color:#191919;">

     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">

    </p>

     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      转入
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      状态
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">

     </span>
     <span style="color:#191919;">
      ：转入
     </span>
     <span style="color:#191919;">
      raw
     </span>
     <span style="color:#191919;">
      状态
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      a:
     </span>
     <span style="color:#191919;">
      上方插入新单元
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">

     </span>
    </p>

     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">

      c:
     </span>
     <span style="color:#191919;">

     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">

     <strong>
      <span style="color:#191919;">
       3.2.1Markdown
      </span>
     </strong>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Jupyter
     </span>
     <span style="color:#191919;">
      提供的
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      是一个非常实用的功能，有了
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      可以将写代码和内容编辑非常好的结合，特别代码内容说明，
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      是一种纯文本标记语言，满足标题，公式，字体加粗，颜色甚至流程图和表格的内容编辑；例如，下图中是使用
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      编写的，能够生成我们想要的公式和字体颜色。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      一些简单的用法如下，编辑好之后使用
     </span>
     <span style="color:#191919;">
      ctrl+enter
     </span>
     <span style="color:#191919;">
      运行结果：
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      更多内容可以了解
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      相关教程。（公众号回复
     </span>
     <span style="color:#191919;">
      markdown
     </span>
     <span style="color:#191919;">
      或者进
     </span>
     <span style="color:#191919;">
      QQ
     </span>
     <span style="color:#191919;">
      群：
     </span>
     <span style="color:#191919;">
      319571120
     </span>
     <span style="color:#191919;">
      即可免费下载）
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">
       3.2.2 CodeCells
      </span>
     </strong>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Code
     </span>
     <span style="color:#191919;">
      模式下，除了一般的
     </span>
     <span style="color:#191919;">
      python
     </span>
     <span style="color:#191919;">
      代码编辑外，
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      还支持一些其他的编程语言，如
     </span>
     <span style="color:#191919;">
      R
     </span>
     <span style="color:#191919;">
      ，
     </span>
     <span style="color:#191919;">
      Julia
     </span>
     <span style="color:#191919;">
      ，
     </span>
     <span style="color:#191919;">
      bash
     </span>
     <span style="color:#191919;">
      等。不过需要使用
     </span>
     <span style="color:#191919;">
      magic
     </span>
     <span style="color:#191919;">
      单元。
     </span>
     <span style="color:#191919;">
      Magic
     </span>
     <span style="color:#191919;">
      单元分为两种，一种是
     </span>
     <span style="color:#191919;">
      line magics
     </span>
     <span style="color:#191919;">
      ，另外一种
     </span>
     <span style="color:#191919;">
      cell magics
     </span>
     <span style="color:#191919;">
      。
     </span>
     <span style="color:#191919;">
      Line magic
     </span>
     <span style="color:#191919;">
      是通过在前面加
     </span>
     <span style="color:#191919;">
      %
     </span>
     <span style="color:#191919;">
      ，表示
     </span>
     <span style="color:#191919;">
      magic
     </span>
     <span style="color:#191919;">
      只在本行有效。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      Cell magics
     </span>
     <span style="color:#191919;">
      ：通过在前面加
     </span>
     <span style="color:#191919;">
      %%
     </span>
     <span style="color:#191919;">
      ，表示在整个
     </span>
     <span style="color:#191919;">
      cell
     </span>
     <span style="color:#191919;">
      单元有效。比如下图中使用
     </span>
     <span style="color:#191919;">
      %%bash
     </span>
     <span style="color:#191919;">
      ，产生了
     </span>
     <span style="color:#191919;">
      linux
     </span>
     <span style="color:#191919;">
      下的
     </span>
     <span style="color:#191919;">
      shell
     </span>
     <span style="color:#191919;">
      环境（
     </span>
     <span style="color:#191919;">
      window
     </span>
     <span style="color:#191919;">
      下不支持，不过可以使用
     </span>
     <span style="color:#191919;">
      %%cmd
     </span>
     <span style="color:#191919;">
      ），这样就可以运行
     </span>
     <span style="color:#191919;">
      pwd
     </span>
     <span style="color:#191919;">
      和
     </span>
     <span style="color:#191919;">
      ls
     </span>
     <span style="color:#191919;">
      命令了。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      其中一些比较常用的
     </span>
     <span style="color:#191919;">
      magic
     </span>
     <span style="color:#191919;">
      ：
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      %matplotlineinline
     </span>
     <span style="color:#191919;">
      在
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      内打印图片；
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      %load
     </span>
     <span style="color:#191919;">
      ：将本地
     </span>
     <span style="color:#191919;">
      py
     </span>
     <span style="color:#191919;">
      文件代码导入进来，例如
     </span>
     <span style="color:#191919;">
      %load test.py
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      %run:
     </span>
     <span style="color:#191919;">
      运行本地代码，利用这个
     </span>
     <span style="color:#191919;">
      magic
     </span>
     <span style="color:#191919;">
      ，我们可以把一些头文件，基本设置，共同函数写在不同的
     </span>
     <span style="color:#191919;">
      notebook
     </span>
     <span style="color:#191919;">
      内，用的时候运行一下就可以了。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      例如下面，将公共的函数写在
     </span>
     <span style="color:#191919;">
      common_import.ipynb
     </span>
     <span style="color:#191919;">
      中，一些导入函数的配置文件存在
     </span>
     <span style="color:#191919;">
      utils.ipynb
     </span>
     <span style="color:#191919;">
      中，需要的时候使用
     </span>
     <span style="color:#191919;">
      %run
     </span>
     <span style="color:#191919;">
      直接运行一下，就可以把公共函数和环境配置好了。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      这样可以将代码写成不同的模块，而不是全部写进一个
     </span>
     <span style="color:#191919;">
      notebook
     </span>
     <span style="color:#191919;">
      。
     </span>
     <span style="color:#191919;">
      %time
     </span>
     <span style="color:#191919;">
      ：统计代码运行的时间，当然还有更多有趣的一些
     </span>
     <span style="color:#191919;">
      magic
     </span>
     <span style="color:#191919;">
      属性来方便我们的工作。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <strong>
      <span style="color:#191919;">
       四、拓展功能
      </span>
     </strong>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      上面已经将
     </span>
     <span style="color:#191919;">
      jupyter notebook
     </span>
     <span style="color:#191919;">
      的基本功能和用法都涉及，但是直接工作的话感觉还是缺一点东西，类似下图这样能够生成
     </span>
     <span style="color:#191919;">
      notebook
     </span>
     <span style="color:#191919;">
      的内容目录和点击快速定位的功能。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      当在
     </span>
     <span style="color:#191919;">
      notebook
     </span>
     <span style="color:#191919;">
      中编写了大量代码的时候，如果想要快速的定位到代码块或者给编辑内容的各级标题生成标题号，可以使用
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      中的延伸功能模块，
     </span>
     <span style="color:#191919;">
      Jupyter Notebook extensions
     </span>
     <span style="color:#191919;">
      来完成这部分工作，
     </span>
     <span style="color:#191919;">
      Jupyter Notebook extensions
     </span>
     <span style="color:#191919;">
      可以通过
     </span>
     <span style="color:#191919;">
      conda
     </span>
     <span style="color:#191919;">
      安装，调用
     </span>
     <span style="color:#191919;">
      conda install -c conda-forgejupyter_contrib_nbextensions
     </span>
     <span style="color:#191919;">
      ，如果
     </span>
     <span style="color:#191919;">
      conda
     </span>
     <span style="color:#191919;">
      安装不成功的话，可以尝试使用
     </span>
     <span style="color:#191919;">
      pip
     </span>
     <span style="color:#191919;">
      安装，不过需要两步工作：
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      1. pip installjupyter_contrib_nbextension
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      2. jupytercontrib nbextension install --user
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:#191919;">
      安装完成后重新打开
     </span>
     <span style="color:#191919;">
      jupyter
     </span>
     <span style="color:#191919;">
      会发现多了
     </span>
     <span style="color:#191919;">
      Nbextensions
     </span>
     <span style="color:#191919;">
      项
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:rgb(25,25,25);">
      通过勾选
     </span>
     <span style="color:rgb(25,25,25);">
      Table of Contents(2)
     </span>
     <span style="color:rgb(25,25,25);">
      ，能够生成
     </span>
     <span style="color:rgb(25,25,25);">
      notebook
     </span>
     <span style="color:rgb(25,25,25);">
      的浏览目录，如果希望在顶部生成目录的话，还需要点击
     </span>
     <span style="color:rgb(25,25,25);">
      Table ofContents
     </span>
     <span style="color:rgb(25,25,25);">
      下勾选下图中的项。
     </span>
    </p>
    <p align="left" style="background:#FFFFFF;">
     <span style="color:rgb(25,25,25);">
      至此
     </span>
     <span style="color:rgb(25,25,25);">
      jupyter
     </span>
     <span style="color:rgb(25,25,25);">
      的安装，常用用法和技巧就基本讲完了，当然
     </span>
     <span style="color:rgb(25,25,25);">
      jupyter
     </span>
     <span style="color:rgb(25,25,25);">
      的功能要不止这些，如果有兴趣可以继续拓展，当时对于日常工作使用的话，这些基本就够了。
     </span>
