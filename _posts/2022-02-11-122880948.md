    <h2 id="article-title">
     创建应用
    </h2>
    <h4 id="l1AZ3">
     概述
    </h4>
    <p id="uda6c25a4">
     在企业微信中，企业应用分为基础应用、第三方应用及自建应用
    </p>
    <ol>
     <li id="ufca0d8ac">
      基础应用：企业微信本身自带的应用
     </li>
     <li id="ud9538856">
      第三方应用：由第三方服务商提供的办公应用套件，管理员可根据企业办公需要选择安装
     </li>
     <li id="u4aa858d6">
      自建应用：根据需求进行定制化，是目前所采用的方式
     </li>
    </ol>
    <h4 id="f0kR1">
     创建应用
    </h4>
    <ol>
     <li id="uc172f316">
      进入企业微信管理后台，【应用管理】&gt;【自建】&gt;【+ 创建应用】
     </li>
    </ol>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2beb7f11cf2e1e389d61559c34acc44e.png"/>
    </p>
    <ol>
     <li id="u60d57094">
      完成应用logo/应用名称/应用介绍/可见范围等基本设置，点击“创建应用”按钮，完成应用创建
     </li>
    </ol>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6a04b9aa8162d15db3314004d15aa95e.png"/>
    </p>
    <h2>
     配置应用
    </h2>
    <h4 id="nXpux">
     1. 企业微信应用配置
    </h4>
    <p id="u9d140235">
     创建应用后，就直接进入该应用的配置页面。
    </p>
    <p id="u23e74bef">
     如果是后期进行应用配置调整，则进入企业微信管理后台，【应用管理】&gt;【自建】&gt; 【选择应用】
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/69734eb6e622b7b0c5af680f3fefd297.png"/>
    </p>
    <p id="u4b209020">
     找到自己项目的应用
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/09217608c33ece4e67247e93d3b74e86.png"/>
    </p>
    <p id="u227ff61f">
     点击应用，进入应用配置页
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c92c4e957ef0bbbb81ead61cd8b8e3bd.png"/>
    </p>
    <p id="u0ecf1f41">
    </p>
    <p>
     1）设置可见范围
    </p>
    <p id="uf49e7fbd">
     在创建应用时，进行可见范围的初始设置；当后期需要调整时，选择“编辑”按钮
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/96fa25ae75a253c5fa2ea1f158fe108e.png"/>
    </p>
    <p id="u816c51bc">
     可添加/删除 部门、成员或标签，设置完成后，点击“完成”按钮。
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ecfa7f7ef0ed30db24b35b9d35cdbecb.png"/>
    </p>
    <p id="ub014cd40">
     调整过后，如下显示
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/68743158e4977d953bbb16c06579a546.png"/>
    </p>
    <p id="u639d9073">
    </p>
    <p>
     2）设置应用主页
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1bf46ddfe4164d49d1421e57bbc1a6a4.png"/>
    </p>
    <p id="u0c70c7f9">
     以SFA为例，地址结构如下：
    </p>
    <pre id="HTYlA">http://dev.recloud.com.cn:8099/api/weixinqy/authorize?redirectUrl=/main</pre>
    <p id="u4aa91469">
     解释如下：
    </p>
    <ul>
     <li id="ubdde12ca">
      <code>
       http://dev.recloud.com.cn:8099
      </code>
      ：站点的协议、域名加端口号
     </li>
     <li id="ud2172e10">
      <code>
       /api/weixinqy/authorize
      </code>
      ：授权接口地址
     </li>
     <li id="u40d0fc23">
      <code>
       ?redirectUrl=/main
      </code>
      ：验证通过后，应用的首页路由
     </li>
    </ul>
    <p id="u019cb93f">
     备注：其中授权接口地址可能因为产品不同而有差异，例如ServiceOne产品的地址结构如下：
    </p>
    <pre id="VRHSF">http://dev.recloud.com.cn:7203/api/weixinsso/authorize?homeurl=/home
</pre>
    <p>
     3）设置网页授权及JS-SDK
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/db42be1bff6ed5e3b82fc4f91853ea50.png"/>
    </p>
    <p id="ud9b3bd9d">
     <strong>
      JS-SDK概述
     </strong>
    </p>
    <p id="ub2e95f68">
     企业微信JS-SDK是企业微信面向网页开发者提供的基于企业微信内的网页开发工具包。
    </p>
    <p id="u8a5c79c1">
     通过使用企业微信JS-SDK，网页开发者可借助企业微信高效地使用拍照、选图、语音、位置等手机系统的能力，同时可以直接使用企业微信分享、扫一扫等企业微信特有的能力，为企业微信用户提供更优质的网页体验。
    </p>
    <p id="ud8c043d6">
     所有的JS接口只能在企业微信应用的可信域名下调用（包括子域名），且可信域名必须有ICP备案且在管理端验证域名归属
    </p>
    <p id="u513e97a3">
     <strong>
      配置说明
     </strong>
    </p>
    <ul>
     <li id="522f18fe05e0d7387421f8bddf17aa88_li_3">
      设置可信域名：设置OAuth2.0 网页授权功能的回调域名、调用JS-SDK的可信域名
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/882add75e8f6875d59570293818150ca.png"/>
    </p>
    <ul>
     <li id="6cbb64d0cbaf87d47d5fe75388f4144c_li_4">
      申请校验域名：下载 JS 校验文件并拷贝到网站的根目录，切保文件可以正常访问
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5753bad9e817e4642e66c7fdc49b25a4.png"/>
    </p>
    <h4 id="R3oot">
     2. 网站及账号配置
    </h4>
    <p id="u55ef6fec">
     <strong>
      1）调整企业微信网站的配置文件
     </strong>
    </p>
    <p id="u55ffa4ad">
     修改站点 config 目录下RekTec.XStudio.WeChatQy.WeChatQyConfig.xml.config配置文件，设置企业 corpId、 应用 AgentId 和 Secret，具体配置如下：
    </p>
    <pre id="elQNH">&lt;?xml version="1.0" encoding="utf-8"?&gt;
&lt;WeChatQyConfig&gt;
  &lt;CorpId&gt;wx445e317f35ba1d01&lt;/CorpId&gt;
  &lt;CorpSecret&gt;q-IWcep6clApDFBIcttZyH2tQd6Ws2Mmx1aiXoNIiOD&lt;/CorpSecret&gt;
  &lt;AgentId&gt;1000068&lt;/AgentId&gt;  
&lt;/WeChatQyConfig&gt;</pre>
    <p id="u4b9ffe70">
     备注
    </p>
    <ul>
     <li id="ub1e27662">
      CorpId：进入企业微信管理后台，【我的企业】 &gt; 【企业信息】：企业ID
     </li>
     <li id="u04bc8815">
      AgentId、CorpSecret：进入企业微信管理后台，【应用管理】&gt;【自建】&gt; 【选择应用】，查看AgentId、CorpSecret
     </li>
    </ul>
    <p id="uf057e46e">
     <strong>
      2）互联用户配置
     </strong>
    </p>
    <p id="u2f3f5d96">
     互联用户配置参考文档
     <a href="https://rektec.yuque.com/all-tc/rektec-crm-dev/gebglw#erfau" rel="nofollow" title="互联用户">
      互联用户
     </a>
     进行配置
    </p>
    <h4 id="htLkj">
     3. 访问测试
    </h4>
    <ul>
     <li id="ue31ab6d2">
      在企业微信里访问应用，正常情况下就会跳转至应用主页，如有异常，根据提示信息及调试工具进行分析
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d11da6c2a936f794d88de38d8d69d111.png"/>

    <h2>
     开发
    </h2>
    <ul>
     <li>
      <strong>
       前端框架
      </strong>
      ：前端框架可以选择vue、react、angular等，目前所选用的vue框架。
     </li>
     <li>
      <strong>
       后端框架
      </strong>
      ：.NET的话基于ASP.NET的MVC，其他语言基于各自的框架即可
     </li>
    </ul>
    <h2>
     调试
    </h2>
    <h4 id="Rn9Hm">
     1. 微信开发者工具
    </h4>
    <p>
     概述:
    </p>
    <p>
     为帮助开发者更方便、更安全地开发和调试基于微信的网页，微信推出了Web开发者工具。它是一个桌面应用，通过模拟微信客户端，使开发者可以使用这个工具方便地在PC上进行开发和调试工作。
    </p>
    <p id="u6b46424c">
     立即
     <a href="https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html" rel="nofollow" title="下载体验">
      下载体验
     </a>
    </p>
    <p id="u3258b163">
     微信开发者工具有以下功能：
    </p>
    <ol>
     <li id="u8688161c">
      使用自己的微信号来调试微信网页授权
     </li>
     <li id="u88e16c8e">
      调试、检验页面的 JS-SDK 相关功能与权限，模拟大部分 SDK 的输入和输出
     </li>
     <li id="u078450ca">
      利用集成的 Chrome DevTools 协助开发
     </li>
    </ol>
    <p id="u234070e5">
     工具界面主要由几大部分组成，如下图所示：
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e2b6a801fa4f597b29fe29254dc6dae7.png"/>
    </p>
    <p>
     1）准备工作
    </p>
    <ul>
     <li id="ud886b375">
      <strong>
       下载并安装
      </strong>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html" rel="nofollow" title="微信开发者工具">
       微信开发者工具
      </a>
      ：根据操作系统下载稳定版 Stable Build 的安装包，若已安装，则忽略。
     </li>
     <li id="u13bd2680">
      <strong>
       绑定企业号开发者
      </strong>
      ：使用企业微信扫码登录管理后台，选择【我的企业】&gt;【微信插件】
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="uce657421">
        使用微信扫“邀请关注”旁边的二维码进行关注
       </li>
       <li id="u6a5f5c85">
        勾选开发者工具的复选框
       </li>
      </ul>
     </li>
    </ul>
    <p id="udddf2dfa">
     <strong>
      注意
     </strong>
     ：账号必须是企业管理员（超级管理员 / 分级管理员），否则就会给出提示“当前账号不是企业管理员（请使用企业管理员帐号扫码）
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1247918dfa56b0e12c681ad6733c7a45.png"/>
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c3332b64fe56369043f5c6fad71adb89.png"/>
    </p>
    <p>
     2）使用微信开发者工具调试
    </p>
    <ul>
     <li id="u6e57ab55">
      打开微信开发者工具，使用微信扫码登录
     </li>
    </ul>
    <ul>
     <li>
      注意：扫码登录Web开发者工具与企业微信里绑定的微信需是同一个微信
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/bb4688ec75935f7969a27b812fd2cc14.png"/>
    </p>
    <ul>
     <li id="u1c0db705">
      选择公众号网页
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/985e8e7301552ab0609efbb0d88f56c6.png"/>
    </p>
    <ul>
     <li id="ud3c3b19b">
      在Web开发者工具URL地址栏输入网址（在企业微信管理后台，自建应用的工作台“应用主页”所填写的URL），接下来就可以在微信开发者工具右侧的Chrome DevTools进行调试。
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4a7972a236875ece599faea04a078c76.png"/>
    </p>
    <p id="ucb4557be">
     若访问应用主页配置的URL时，报“未绑定企业号开发者”的错误，则需要先绑定 企业号开发者。
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c362608595a08f0fce4c760d05d7ca9c.png"/>
    </p>
    <h2>
     发布
    </h2>
    <h4 id="w2kSM">
     前端代码更新
    </h4>
    <p id="u5a24eaec">
     按照以下步骤进行操作：
    </p>
    <p>
     1. 打包H5
    </p>
    <ul>
     <li id="ub626c295">
      在VS Code中，通过
      <code>
       yarn build
      </code>
      打包H5，最终会在文件夹目录下生成一个dist文件夹，里面包含static文件夹和index.html文件。
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4cf32758fe0f20fa648d8d1a1f50e0f1.png"/>
    </p>
    <p>
     2. 更新H5文件
    </p>
    <ul>
     <li id="u6cad512b">
      将生成的文件拷贝到网站目录的weixinqy文件夹下
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/578b873ebf8eac76f3f2da3506be0e1c.png"/>
    </p>
    <h4 id="gLntd">
     后端代码更新
    </h4>
    <p>
     1. 生成DLL
    </p>
    <ul>
     <li id="u88f69903">
      打开VS，选中有调整的项目，【右击】&gt;【重新生成选定内容】，在生成目录下就可以看到生成成功的文件
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/26aa634e52c1bb53de000625ba98db31.png"/>
    </p>
    <p>
     2. 更新DLL文件
    </p>
    <ul>
     <li id="u9c06feda">
      将bin文件夹生成的文件拷贝到网站目录的bin文件夹下
     </li>
    </ul>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a23aa554c674cd923638c88908dc156b.png"/>
    </p>
