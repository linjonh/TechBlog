    <h2 style="margin-left:.1em;">
     <span style="color:#007af4;">
      选择安装方式
     </span>
    </h2>
    <div id="body12286261" style="margin-left:0;">
     <p id="ZH-CN_TOPIC_0000001135912433__p1361971" style="margin-left:.5em;">
      <span style="color:#333333;">
       <span style="color:#666666;">
        由于
        <a href="http://www.ict18.com/xfusion-servers" rel="nofollow" title="服务器">
         服务器
        </a>
        安装的OS类型、匹配的板卡不同，用户可以选择的安装方式以及相应的板卡驱动程序也不同。可通过兼容性查询助手查询目标OS对应的安装方式。
       </span>
      </span>
     </p>
     <p id="ZH-CN_TOPIC_0000001135912433__p12257747" style="margin-left:.5em;">
      <span style="color:#333333;">
       <span style="color:#666666;">
        <a href="http://www.ict18.com/xfusion-servers" rel="nofollow" title="服务器">
         服务器
        </a>
        的操作系统可通过如表1所示的四种方式进行安装，四种安装方式特点如下。
       </span>
      </span>
     </p>
     <div style="margin-left:0;">
      <table cellspacing="0" id="ZH-CN_TOPIC_0000001135912433__table53353470">
       <caption style="text-align:left;">

          <p id="ZH-CN_TOPIC_0000001135912433__p11050762" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             安装方式
            </span>
           </span>
          </p>
         </th>
         <th id="mcps1.3.3.2.3.1.2" style="text-align:left;vertical-align:baseline;">
          <p id="ZH-CN_TOPIC_0000001135912433__p22696536" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             特点
            </span>
           </span>
          </p>
         </th>
        </tr>
       </thead>
       <tbody>
        <tr id="ZH-CN_TOPIC_0000001135912433__row8554181220204">
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p153636132110" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             Smart Provisioning方式安装
            </span>
           </span>
          </p>
         </td>
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p85582041162018" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             Smart Provisioning是一个操作系统的安装引导工具，简化了工程师安装操作系统和驱动的过程。
            </span>
           </span>
          </p>
          <p id="ZH-CN_TOPIC_0000001135912433__p1755964112014" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             此方式要求iBMC 2.64以上版本以及BIOS 0.37以上版本。
            </span>
           </span>
          </p>
          <p id="ZH-CN_TOPIC_0000001135912433__p19559144115203" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             Smart Provisioning具有以下特点：
            </span>
           </span>
          </p>
          <ul id="ZH-CN_TOPIC_0000001135912433__ul10560124112209">
           <li id="ZH-CN_TOPIC_0000001135912433__li13560194162020">
            <span style="color:#333333;">
             帮助用户安装操作系统，并根据检测出的硬件提供设备驱动程序。
            </span>
           </li>
           <li id="ZH-CN_TOPIC_0000001135912433__li13560241142017">
            <span style="color:#333333;">
             提供一致性的安装过程，通过简单的用户界面对话引导用户完成整个安装。
            </span>
           </li>
           <li id="ZH-CN_TOPIC_0000001135912433__li15562164132013">
            <span style="color:#333333;">
             降低了用户在安装过程中所面临的复杂性，确保用户能够在
             <a href="http://www.ict18.com/xfusion-servers" rel="nofollow" title="服务器">
              服务器
             </a>
             上快速部署操作系统。
            </span>
           </li>
          </ul>
         </td>
        </tr>
        <tr id="ZH-CN_TOPIC_0000001135912433__row25017441">
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p13146863" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             直接安装
            </span>
           </span>
          </p>
         </td>
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p54604565" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             通过操作系统光盘或镜像文件安装操作系统是一种最常见的方式。
            </span>
           </span>
          </p>
          <p id="ZH-CN_TOPIC_0000001135912433__p21679043" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             此方式可以利用物理光驱读取光盘内容，也可以使用虚拟光驱加载镜像文件。
            </span>
           </span>
          </p>
         </td>
        </tr>
        <tr id="ZH-CN_TOPIC_0000001135912433__row60893663">
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p33439661" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             加载驱动安装
            </span>
           </span>
          </p>
         </td>
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p16995625" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             当
             <a href="http://www.ict18.com/xfusion-servers" rel="nofollow" title="服务器">
              服务器
             </a>
             配置RAID控制卡时，可以采用加载驱动方式进行操作系统和RAID控制卡驱动的一次性安装。
            </span>
           </span>
          </p>
          <p id="ZH-CN_TOPIC_0000001135912433__p18742903" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             加载驱动方式有通过虚拟光驱加载ISO格式的和通过虚拟软驱加载IMG格式的驱动：
            </span>
           </span>
          </p>
          <ul id="ZH-CN_TOPIC_0000001135912433__ul34468402">
           <li id="ZH-CN_TOPIC_0000001135912433__li41780163">
            <span style="color:#333333;">
             加载ISO格式的驱动在安装系统时通过虚拟光驱对操作系统和RAID控制卡驱动的镜像文件进行切换。
            </span>
           </li>
           <li id="ZH-CN_TOPIC_0000001135912433__li40477147">
            <span style="color:#333333;">
             加载IMG格式的驱动在安装系统时可直接加载驱动。
            </span>
           </li>
          </ul>
         </td>
        </tr>
        <tr id="ZH-CN_TOPIC_0000001135912433__row28750003">
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p47048879" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             制作安装源安装
            </span>
           </span>
          </p>
         </td>
         <td style="text-align:left;vertical-align:top;">
          <p id="ZH-CN_TOPIC_0000001135912433__p6003880" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             此安装方式当前仅适用于VMware
             <br/>
             ESXi操作系统。将安装操作系统所需的驱动打包到ISO文件中，使用打包后的ISO镜像进行操作系统的安装。
            </span>
           </span>
          </p>
          <p id="ZH-CN_TOPIC_0000001135912433__p54034926" style="margin-left:.2em;">
           <span style="color:#333333;">
            <span style="color:#666666;">
             此方式需要使用虚拟光驱加载镜像文件。
            </span>
           </span>
          </p>
         </td>
        </tr>

      </table>
