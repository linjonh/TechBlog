    <blockquote>
     <h2>
      <a id="DDNS_0">
      </a>
      本文主要详记免费二级域名注册和免费DDNS解析
     </h2>
    </blockquote>
    <h3>
     <a id="_1">
     </a>
     写在前头
    </h3>
    <blockquote>
     <p>
      近几年国内注册域名需要备案，域名解析服务商都陆续需要收费了，对于我这种想简单玩玩的人来说实在是太麻烦了(主要是费钱），花了点时间找到了免费域名注册和DDNS解析
     </p>
    </blockquote>
    <blockquote>
     <p>
      本文基于
      <a href="https://blog.csdn.net/idiot_qi/article/details/124610801">
       家用宽带搭建个人服务器（一）
      </a>
      配置的动态公网ip和DMZ主机
     </p>
    </blockquote>
    <h3>
     <a id="1__6">
     </a>
     1. 免费域名注册
    </h3>
    <blockquote>
     <p>
      网站：
      <a href="https://nic.eu.org/arf/en/login/" rel="nofollow">
       https://nic.eu.org/arf/en/login/
      </a>
      <br/>
      这个网站申请的是二级域名，不过这都没什么影响，就是申请之后等待的时间有点长，大概需要一周左右，最近我申请的几个域名，快的4天慢的10来天，申请成功会给你发个邮件告诉你，具体注册步骤放到后面
     </p>
    </blockquote>
    <h3>
     <a id="2_DNS_9">
     </a>
     2. 免费DNS解析
    </h3>
    <blockquote>
     <p>
      网站：
      <a href="https://dns.he.net/" rel="nofollow">
       https://dns.he.net/
      </a>
      <br/>
      这是个国外的免费dns解析，还提供了ddns方案，使用起来也很简单方便，解析的速度也很快，没啥延迟的感觉
     </p>
    </blockquote>
    <h3>
     <a id="3__12">
     </a>
     3. 域名注册详细步骤
    </h3>
    <blockquote>
     <p>
      3.1. 先注册一个账号，翻译一下填好账号信息就行，他会根据你填的名字，生成两个字母三个数字的账户名（用来登录），然后会给你的邮箱发一个激活账号的链接，你要去邮箱里验证一下
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4e39085dfb62df66d830c5f582e7daf3.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/57de801e3d2823245db4f359ee6e8fb5.png"/>
     </img>
    </p>
    <blockquote>
     <p>
      3.2. 创建新域名，点击
      <strong>
       New Domain
      </strong>
      ，然后填写域名名称（
      <strong>
       domain name
      </strong>
      ），list链接里可以看能使用的域名后缀，我觉得eu.org就可以，也可以用cn.eu.org，就是有点长；在最后
      <strong>
       Name servers
      </strong>
      中选择第一项，然后填写
      <strong>
       域名解析运营商
      </strong>
      的提供的DNS解析地址，我用的是本文免费的DNS解析地址：ns1.he.net；ns2.he.net；ns3.he.net；ns4.he.net等等；然后submit就可以等待了
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1df4403dd18d6dc607f0d1a18b4abc79.png"/>
    </p>
    <hr/>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9bb4b49ca75f9e2f5557436c9d0a77b5.png"/>
    </p>
    <hr/>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d84287e6e96948231a782b7a2fe5f85d.png"/>
    </p>
    <blockquote>
     <p>
      3.3. 提交后显示
      <strong>
       Done
      </strong>
      就可以了
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/edcb6554e875c59d0c7b80ceb660a56e.png"/>
    </p>
    <h3>
     <a id="4__31">
     </a>
     4. 域名解析详细步骤
    </h3>
    <blockquote>
     <p>
      4.1. 先注册账号登录之后，点击页面右侧
      <strong>
       Quick Links
      </strong>
      栏中的
      <strong>
       Free DNS
      </strong>
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4f00902cd739b9b6699397e7b353d463.png"/>
    </p>
    <blockquote>
     <p>
      4.2. 然后点击左侧的添加新域名，填写你要解析的域名
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/79372d8b385e6af026667c59912250f2.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a6c6403d544a800d0760644c49e2fd0e.png"/>
     </img>
    </p>
    <blockquote>
     <p>
      4.3. 添加成功，我这个由于域名是刚申请的域名，还不能解析，提示一段红字
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2d0770f8d86ddf48e3441b3e727d7d13.png"/>
    </p>
    <blockquote>
     <p>
      4.4. 点击列表中域名前面的编辑图标，页面中一片红色的提示，和上面大致意思差不多，等域名正常了就不会提示了
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5a14a09ada3fcaf872cb78698225191b.png"/>
    </p>
    <hr/>
    <blockquote>
     <p>
      过了几天域名注册成功后，就没有红色提示了
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f64d0dc6ce6f6f7fe135e301e35ca4c0.png"/>
    </p>
    <blockquote>
     <p>
      4.5. 添加A记录，把
      <em>
       <strong>
        www加域名
       </strong>
      </em>
      填上在name，ipv4地址填上自己家当前的公网ip，最下面那个动态dns解析勾上，然后提交；如果都是域名和公网ip都是可以使用的，这一步完成之后就可以通过域名和端口正常访问家里的DMZ主机服务了
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eccdf2fd7e08b365dc7204c15b1675cb.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d568f5eae1f796cc6f3aee342264acc7.png"/>
    </p>
    <blockquote>
     <p>
      4.6. 接下来就是配置DDNS，也很简单；点击A记录右侧的刷新按钮，生产一个key，自己复制保存一下然后提交，下一步需要用到
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/aa7fa69edcd6450d67ea4b24d815bb13.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/55c8e49b256424fa1c770d9a48b3372b.png"/>
    </p>
    <blockquote>
     <p>
      4.7. 这一步是把自己的公网ip发送给域名解析服务器，首先按格式拼一下请求链接，然后在DMZ主机中
      <strong>
       命令行
      </strong>
      访问，如果有公网IP会自动获取本机的公网IP发送给域名解析服务器，最好设置一个五分钟的定时任务，间隔五分钟就请求一次：
      <br/>
      链接格式：http://[your domain name]:[your password]@dyn.dns.he.net/nic/update?hostname=[your domain name]
      <br/>
      IPV4 例如:
      <br/>
      % curl -4 “http://www.qq2xx.eu.org:z1kfFLlWCuNvsYGs@dyn.dns.he.net/nic/update?hostname=www.qq2xx.eu.org”
     </p>
    </blockquote>
    <blockquote>
     <p>
      在官网首页中也操作步骤，可以自己去详细看一下
      <br/>
      <a href="https://dns.he.net/" rel="nofollow">
       dns.he.net
      </a>
     </p>
    </blockquote>
