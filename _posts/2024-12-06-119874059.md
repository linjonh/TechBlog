    <h3>
     <a id="mysql_0">
     </a>
     本地mysql连接远程服务器数据库
    </h3>
    <p>
     1.确保本地已安装mysql,并配置好了环境变量
     <br/>
     2.用win+R 回车 打开命令行
     <br/>
     3.输入：mysql -u 用户名 -p密码 -h 服务器IP地址 -P 服务器端MySQL端口号(一般为3306) -D 数据库名
     <br/>
     注意：必须严格按照上述的格式，特别是空格的位置，不能少。
    </p>
    <p>
     如果出现
     <em>
      Lost connection to MySQL server at ‘reading initial communication packet’
     </em>
     <br/>
     这个就是ip地址或者端口号没写对多注意一下
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0264271ba840dbff1be5de81007cdef1.png"/>
    </p>
    <p>
     如果出现这个问题
     <em>
      ERROR 1049 (42000): Unknown database
     </em>
     <br/>
     <img alt="mysql免安装error1049_mysql ERROR 1049 (42000): Unknown database '******' ”错误处理办法" src="https://i-blog.csdnimg.cn/blog_migrate/6e0663a3e3b83ff5b3597929bbf9a1c3.png">
      <br/>
      -p和密码是连在一起的，不能加空格
      <br/>
      这样就能连上了
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5d3468b5a36eafb3a54d30eb9f1ac538.png"/>
     </img>
    </p>
