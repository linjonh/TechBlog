    <div class="article_content tracking-ad" id="article_content" style='margin:35px 0px; font-size:15px; color:rgb(85,85,85); line-height:35px; font-family:"microsoft yahei"'>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       jsp和html相比，最显著的优势就在于，jsp里面可以写java代码，但是html中不能。
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       那么如何在jsp中写java代码呢！
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       jsp中写java代码有如下三种方式：
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       &lt;%！ %&gt;，这里面可以申明变量或方法，注意：这里面申明的变量是全局的
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       &lt;% %&gt;，与上面的方法相比，这个方法的局部的
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       &lt;%= %&gt;，用于输出表达式到浏览器，注意：这里面的表达式不能跟分号
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       另外，我想谈一谈jsp指令的使用。
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       在jsp2.0中定义了三个指令：page指令，include指令，taglib指令。
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       1.关于page指令，我们平时见得最多的就是这样的：
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        &lt;%@page
       </span>
       <span style="font-size:16pt; font-family:宋体">
        contentType
       </span>
       <span style="font-size:16pt; font-family:宋体">
        ="text/
       </span>
       <span style="font-size:16pt; font-family:宋体">
        html;charset
       </span>
       <span style="font-size:16pt; font-family:宋体">
        =gb2312" import="
       </span>
       <span style="font-size:16pt; font-family:宋体">
        java.util.Date
       </span>
       <span style="font-size:16pt; font-family:宋体">
        "%&gt;
       </span>
       <span style="font-size:24pt; font-family:宋体">
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        主要用于：设置编码，导入jar包
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        <br/>
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        2.关于include指令，他用于引入其他jsp页面，jsp引擎会将两个jsp翻译成一个servlet，所以也称为静态引入
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        <br/>
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-size:16pt; font-family:宋体">
        3.关于taglib指令，是定义一个标签库以及其自定义标签的前缀。比如我们常用的c:foreach的使用。先导入jstl的包，然后在jsp中加入这个标签：&lt;%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %&gt;就可以使用c:foreach了
       </span>
      </span>
     </p>
