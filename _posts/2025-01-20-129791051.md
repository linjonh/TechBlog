        常用的python遍历数组代码
       </span>

     <li>
      <p>
       1. 使用for循环遍历数组
       <br/>
       <br/>
       ```python
       <br/>
       arr = [1, 2, 3, 4, 5]
       <br/>
       for i in arr:
       <br/>
       print(i)
       <br/>
       ```
       <br/>
       <br/>
       2. 使用while循环遍历数组
       <br/>
       <br/>
       ```python
       <br/>
       arr = [1, 2, 3, 4, 5]
       <br/>
       i = 0
       <br/>
       while i &lt; len(arr):
       <br/>
       print(arr[i])
       <br/>
       i += 1
       <br/>
       ```
       <br/>
       <br/>
       3. 使用enumerate()函数遍历数组
       <br/>
       <br/>
       ```python
       <br/>
       arr = [1, 2, 3, 4, 5]
       <br/>
       for i, val in enumerate(arr):
       <br/>
       print(i, val)
       <br/>
       ```
       <br/>
       <br/>
       4. 使用zip()函数遍历多个数组
       <br/>
       <br/>
       ```python
       <br/>
       arr1 = [1, 2, 3, 4, 5]
       <br/>
       arr2 = ['a', 'b', 'c', 'd', 'e']
       <br/>
       for a, b in zip(arr1, arr2):
       <br/>
       print(a, b)
       <br/>
       ```
       <br/>
       <br/>
       5. 使用列表推导式遍历数组
       <br/>
       <br/>
       ```python
       <br/>
       arr = [1, 2, 3, 4, 5]
       <br/>
       new_arr = [i*2 for i in arr]
       <br/>
       print(new_arr)
       <br/>
       ```
      </p>
