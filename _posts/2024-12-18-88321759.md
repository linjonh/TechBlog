    <h3>
     <a id="1_0">
     </a>
     1.直接下载
    </h3>
    <p>
     前提是环境变量中设置的python默认路径为你安装的python，而不是其他工具，例如Anaconda等。
     <br/>
     在cmd窗口中，输入命令行
     <br/>
     python -m pip install python包
    </p>
    <h3>
     <a id="2_6">
     </a>
     2.在特定路径下下载
    </h3>
    <p>
     在cmd窗口中，切换路径到你python的安装目录Scripts文件夹下，例如
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7454b4b583580f4be427e0de6498ce28.png">
      <br/>
      切换到路径后，输入命令行：
      <br/>
      pip install python包
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/419d90193c573767c15bd34a65facab1.png">
       <br/>
       等待下载安装即可。
      </img>
     </img>
    </p>
    <h3>
     <a id="3_14">
     </a>
     3.下载超时、连接不上等问题解决
    </h3>
    <p>
     下载时经常会出现超时、速度慢、连接不上等问题，是因为pip安装是是默认连接的国外的服务器地址进行下载，因此会出现上述问题。解决时有两种方法。
     <br/>
     1)去国外资源站或其他地方找到要安装包的资源，下载后就cd到你下载包的文件夹中即可。
     <br/>
     2）切换默认的下载地址到国内资源站上。
     <br/>
     打开该文件夹
     <strong>
      C:\Users\DELL\AppData\Roaming
     </strong>
     或是
     <strong>
      C:\Users\Administrator\AppData\Roaming
     </strong>
     ，AppData文件夹如果找不到就是被隐藏了，在文件管理中显示被隐藏的文件夹即可。
     <br/>
     在
     <strong>
      Roaming
     </strong>
     文件夹下新建文件夹
     <strong>
      pip
     </strong>
     ,然后在此文件夹下新建txt文档，并在其中输入以下内容：
    </p>
    <p>
     [global]
     <br/>
     timeout = 60000
     <br/>
     index-url =
     <a href="https://pypi.tuna.tsinghua.edu.cn/simple" rel="nofollow">
      https://pypi.tuna.tsinghua.edu.cn/simple
     </a>
     <br/>
     [install]
     <br/>
     use-mirrors = true
     <br/>
     mirrors =
     <a href="https://pypi.tuna.tsinghua.edu.cn" rel="nofollow">
      https://pypi.tuna.tsinghua.edu.cn
     </a>
    </p>
    <p>
     保存并命名为“
     <strong>
      pip.ini
     </strong>
     ”
    </p>
    <p>
     然后再去cmd上下载时，速度问题便会得到解决，亲测有效！
    </p>
