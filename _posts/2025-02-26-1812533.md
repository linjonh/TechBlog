     <font size="5">
      支持北京大学陆峰同学起诉微软公司
     </font>
     <br/>
    </div>
    <font size="3">
     <br/>
     本人与陆峰君素昧平生，但我今日在此大声呼吁“支持北京大学陆峰同学起诉微软公司”，其原因无他，只因为陆峰君以中国知识分子之责任挑起民族的脊梁。在当今物欲横流，俗文化遍地的时代，陆峰君敢于单骑挑战具有国际影响的公司，通过一个案例启蒙国民在信息社会应有的觉悟，实乃难能可贵。其明知不可为而为之，存玉碎之心而不留瓦全之念的精神，足以惊天泣鬼。此何等之品行，无人喝彩，无人支持、无人呐喊，岂不怪哉！
     <br/>
     <br/>
     <span style="font-weight: bold;">
      事件经过：
     </span>
     <br/>
     据法制日报2007年9月11日报道：8月28日北京大学学生陆峰起诉美国微软公司、微软(中国)有限公司一案，北京市第一中级人民法院已立案受理。
     <br/>
     陆峰是北京大学的一名法律专业硕士生，最近成为国内第一起针对微软“Windows正版增值计划”程序KB905474的起诉者。这起法律诉讼的起因是陆峰购买了一台预装WINDOWS   XP操作系统的笔记本电脑，在今年4月份的时候他所在的群里面有人发贴说：“盗版的用户要小心，微软推出这个正版增值计划(WGA)是为美国收集盗版用户的数据”。陆峰认为“Windows正版增值计划通知”的程序 KB905474，没有清楚告知用户该程序安装后厂商可以反复验证用户电脑里的信息，并成为永久性安装。这严重的威胁了消费者信息安全，他希望通过这个诉讼，能对维护国家和广大微软系统用户的利益，做出些微小的贡献。"  陆峰介绍自己是预装操作系统的正版用户，而且通过WGA验证为正版了。他的诉讼是针对WGA安装程序没有充分提示用户要获取其数据的情况下，通过该程序收集用户数据，威胁用户信息安全和隐私。这个问题造成即使是正版用户的权利也同样受到侵害的。侵害了消费者知情权、选择权和财产权益。但中国没有类似美国法律中的《个人信息安全法》，所以，陆峰决定选择适用《消费者权益保护法》，主要打知情权的问题。
     <br/>
     <span style="font-weight: bold;">
      <br/>
      同时陆峰请求法院：
     </span>
     <br/>
     一、撤消被告与原告订立的本案诉争程序的安装许可协议；
     <br/>
     二、判令被告删除获取的原告计算机信息和个人信息；
     <br/>
     三、判令被告向原告公开其获取原告的信息；
     <br/>
     四、禁止被告在任何场合下以任何目的、任何方式使用其获取的上述信息；
     <br/>
     五、判令被告提供卸载工具，卸载其安装在原告计算机上的诉争程序。
     <br/>
     六、判令被告在全国发行的报纸上公开向原告赔礼道歉；
     <br/>
     七、撤消被告技术支持服务中显失公平的收费条款；
     <br/>
     八、责令被告以公平合理方式向原告提供技术支持服务；
     <br/>
     九、赔偿原告1350元费用等。
     <br/>
     陆峰解释这个诉讼属于个人行为，并且是公益诉讼。为此，他和同学成立了一个诉讼团队，并表示做好了打持久战的准备。他们希望通过这起诉讼，能起到抛砖引玉的作用，对维护国家和广大微软系统用户的利益，做出些微小的贡献。"
     <br/>
     <br/>
     <span style="font-weight: bold;">
      微软对“Windows正版增值计划”的解释：
     </span>
     <br/>
     “Windows正版增值计划”是一个全球性的计划,并不只针对任何一个市场.这个计划自2004年9月试行实施以来,500多万用户参加了第一阶段的计划.从微软2005年7月在全球正式推出“Windows正版增值计划”,全球已有5.12亿用户参与了此项计划.此次“正版增值计划”是针对全球Windows   XP使用者,通过对用户所使用系统的验证和提示,让正版和盗版的“区别”更加泾渭分明,而非收集用户个人信息.
     <br/>
     <br/>
     <span style="font-weight: bold;">
      事件难点：
     </span>
     <br/>
     法庭要依据本案原告提供的证据和法律依据审理此案，而在这两方面陆峰存在劣势。
     <br/>
     1、法庭要依据谁诉讼谁举证的原则，即使法庭能够认定微软WGA程序存在给用户带来安全隐患的问题，但又怎样能取得微软获得用户私有信息的证据呢？微软的“Windows正版增值计划”验证服务器应该远在美国，法院何不能让微软把“Windows正版增值计划”验证服务的数据和微软获得的用户信息当庭呈送给法庭吧！你想微软也不会同意这样做吧！那陆峰同学就只有告微软的理由，没有告微软的证据了。这是本案的关键所在，如果考虑不周容易受人以口舌。
     <br/>
     2、2005年微软在美国推出
    </font>
    <font size="3">
     “Windows正版增值计划”
    </font>
    <font size="3">
     不久就遭到两起集体诉讼，起诉微软的WGA涉嫌侵犯用户隐私。因为美国法律是有专门的《个人信息安全法》，他们可以直接依据该法状告微软，而中国在此领域还存在立法空白。这就带来一个问题，陆峰诉讼“Windows正版增值计划”的WGA程序核心问题是威胁用户信息安全和隐私，但在我国目前情况下只能选择适用《消费者权益保护法》主打知情权。这就好比因为只有抢劫杀人犯的抢劫法律审判依据，没有杀人审判依据，只能当庭判处罪犯抢劫罪，而令其杀人罪行逃避法律制裁。这显然是对微软“Windows正版增值计划”的WGA程序核心问题的从轻发落，能不能对微软起到惩前毖后、敲山震虎的作用，可能要大打折扣。
     <br/>
     <br style="font-weight: bold;"/>
     <span style="font-weight: bold;">
      事件点评：
     </span>
     <br/>
     陆峰同学这种以天下人为念，为天下人谋永福的精神是值得社会赞扬和当代青年人学习的，但在整个事件的战略谋划上尚显瑕疵。
     <br/>
     1、有能力发现微软“Windows正版增值计划”WGA程序危害用户信息安全的实质，没有办法对症下药
    </font>
    <font size="3">
     根本
    </font>
    <font size="3">
     解决问题，但一定程度起到了警告微软、警惕国人的作用。
     <br/>
     2、陆峰同学在接受新浪网采访时表示做好了打持久战的准备，但忽视了在国内建立诉讼微软统一战线的作用，况且在美国微软也面临的是集体诉讼，毕竟人多力量大。
     <br/>
     3、建立诉讼团队，进行分工是必要的，但诉讼团队的专业性、凝聚力、战斗力和长期稳定是基本因素。毕竟是北大学子做事，应该考虑吸收有计算机专业背景的同学也加入团队，北大应该不乏既有社会责任感又懂IT专业知识的人才。
     <br/>
     4、不愿意牵扯老师和校方是同学真诚之举，但要面向大众呼吁社会舆论和媒体给予适当道义支持；孤军奋战，勇气可嘉，但毕竟对手是一个世界级的商业帝国。
     <br/>
     <br/>
     <span style="font-weight: bold;">
      几点建议：
     </span>
     <br/>
     1、以诉讼必将失败之心，重新规划本案诉讼策略。本人不是不希望陆峰同学诉讼成功，本人是料定在目前的主、客观条件下，即使毕其功于一役也难有胜算，微软毕竟是微软。以适用《消费者权益保护法》主打知情权来惊世国人信息安全意识实在是勉而为之。但只有客观的预测失败才能合理地利用失败带给我们的其他无形财富。以一个诉讼的胜利者的姿态去惊世国人不见得比以哀兵姿态惊世国人效果更好。
     <br/>
     2、本案诉讼重要的不是结果，而是过程，以及给我们带来的经验和启示。因此，建议陆峰同学应当在社会上广泛建立诉讼微软统一战线（可以实事求是的告诉陆峰同学：当今中国受操作系统等垄断的危害的企业大有人在）；争取周围同学、院系老师、学校方面的支持与理解；通过本案正确引导社会舆论；迅速扩大诉讼团队，吸收多方面人才，完善团队专业化建设。
     <br/>
     3、对本案只做最坏思想准备，这起诉讼会给当今中国留下一笔宝贵的精神遗产，明知不可为而为之是先知先觉者的宿命。
     <br/>
     4、不管你、你们和你们的团队多么被别人误解、辱骂都要坚持下去，必要时还要敢于面向公众，历史的发展会启示真理的价值。
     <br/>
     <br/>
     <span style="font-weight: bold;">
      事件启示：
     </span>
     <br/>
     这个案件充分显露了商业软件打着为消费者服务的旗号同时又剥夺消费者知情权和选择权的实质，甚至给用户带来信息和数据的损失。被商业软件剥夺的用户权力恰恰是自由软件（Free Software）倡导的精神主旨，这不是巧合，而是事物的矛盾性。软件被人类发明创造出来的那一天就是要解决人类社会发展道路上的实际问题，就是为社会生产力的提高而服务的。在软件发明的初期软件以开放的程序形式，也就是源代码开放的状态示人。但随着技术和产业的发展，软件出现了私有的版权形式，私有的软件形式是软件发展的必然阶段，同时私有软件在特定时期促进了相关产业的发展，但我们必须要看到私有软件的弊端，充分利用事物的矛盾性，把握事物发展的客观规律，在尊重知识产权的游戏规则外，合理利用既能提高我们生产力和竞争力，又能不在知识产权方面束缚我们发展的公共知识体系，如GPL协议下的技术，在新的历史时期下，为我们民族和平崛起于世界，为我们国家和谐建设，为社会健康发展寻求可能。
     <br/>
     <br/>
     最后，祝愿陆峰君和他们的团队心想事成！
     <br/>
     <br/>
     王开源
