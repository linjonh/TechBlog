     Linux下查看和停止所有java进程
    </h2>
    <h3>
     <a id="1ps_ef__grep_java_1">
     </a>
     <a href="http://1.ps" rel="nofollow">
      1.ps
     </a>
     -ef | grep java
    </h3>
    <p>
     在Linux下查看所有java进程命令：ps -ef | grep java
    </p>
    <p>
     停止特定java进程命令：kill -9 java进程序号
    </p>
    <p>
     <em>
      停止所有java进程命令：pkill - 9 java
     </em>
    </p>
    <h2>
     <a id="2jps_9">

     2.jps
    </h2>
    <p>
     jps -l -v
    </p>
    <p>
     比较常用的参数：
     <br/>
     -q 只显示pid，不显示class名称,jar文件名和传递给main 方法的参数

      -m 输出传递给main 方法的参数，在嵌入式jvm上可能是null
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf3f1c2ed94d9d139b46ed2cfedfed97.png">
       <br/>
       -l 输出应用程序main class的完整package名 或者 应用程序的jar文件完整路径名
