    <h5>
     <a id="_3">
     </a>
     安装
    </h5>
    <p>
     开始在 Java 中使用 Redis 前， 我们需要确保已经安装了 redis 服务及 Java redis 驱动，且你的机器上能正常使用 Java。 Java的安装配置可以参考我们的
     <a href="https://www.runoob.com/redis/java-environment-setup.html" rel="nofollow">
      Java开发环境配置
     </a>
     接下来让我们安装 Java redis 驱动：
     <br/>
     首先你需要下载驱动包 下载 jedis.jar，确保下载最新驱动包。
     <br/>
     在你的 classpath 中包含该驱动包。
     <br/>
     本站提供了 2.9.0 jar 版本下载：
     <a href="https://static.runoob.com/download/jedis-2.9.0.jar" rel="nofollow">
      jedis-2.9.0.jar
     </a>
    </p>
