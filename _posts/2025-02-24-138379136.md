    <h2>
     <a id="_5">
     </a>
     前言
    </h2>
    <p>
     当下，随着人工智能的普及，对编程学习的重要性越来越大。所以，现在大部分孩子都在学习编程。但是，编程在我眼中看来，也是在内行人看来，适龄性是极为重要的
     <br/>
     所以，这期博客让我来告诉大家适龄的编程语言学习
    </p>
    <hr/>
    <h2>
     <a id="6_14">
     </a>
     一、6岁以下
    </h2>
    <h3>
     <a id="1_15">
     </a>
     1.推荐
    </h3>
    <p>
     对于一般的娃来说，这个年龄段是不适宜编程的，因为在这个年龄段，大部分的孩子的逻辑思维以及模块化思维尚未培育完善。
     <br/>
     在这个期间，娃娃们可以先培养一下逻辑思维，这样对以后的编程是非常有利的
    </p>
    <h3>
     <a id="2_18">
     </a>
     2.软件
    </h3>
    <p>
     如果非要学的话，大家可以先尝试一下简单的编程模块
     <br/>
     大家可以下载“探月校园版”，或“探月少儿编程网页版”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/30c80b3a1b95fe521d3f2679b89b6ed9.png">
      <br/>
      打开之后，我们下载App即可
      <br/>
      家长们可以先帮孩子下载教程，使用Nemo图形化编程即可
      <br/>
      Nemo对于6岁以下孩子说实话还是很‘难’的，所以只能适当接触，家长应当给予帮助
      <br/>
      下载链接：https://tanyue.codemao.cn
     </img>
    </p>
    <h2>
     <a id="610_26">
     </a>
     二、6至10岁
    </h2>
    <h3>
     <a id="1_27">
     </a>
     1.推荐
    </h3>
    <p>
     此时孩子的学习段应当是大班（或中班）到小学三四年级，认知应该也相对成熟一些了，这个年龄段，即是编程发展的关键期
    </p>
    <h3>
     <a id="2_30">
     </a>
     2.软件
    </h3>
    <p>
     这里我先写一下具体框架
    </p>
    <h4>
     <a id="168NemoScratch_32">
     </a>
     （1）6-8:Nemo编程——Scratch图形化编程
    </h4>
    <p>
     这个过度非常关键，因为nemo与scratch的唯一区别就是
     <strong>
      难度与逻辑
     </strong>
     <br/>
     scratch相较于nemo，逻辑提升极为重要
     <br/>
     所以，如果在8岁时觉得scratch难的，不要害怕，可以慢慢摸，知道摸熟悉之后，scratch马上就会变得很
     <strong>
      幼稚
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d837e1598b3cd15fa375354ee4e6ef23.png">
      <br/>
      这是scratch的图标，下载很简单，只要在应用商城中下载“scratch3”或者“TurboWarp”，后者是一种更加高级的软件，和scratch差不多，功能相较于前者更加功能全面
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8538419ebd921d72ea3ce80affe098ac.png"/>
     </img>
    </p>
    <h4>
     <a id="2810ScratchPython_39">
     </a>
     （2）8-10岁：Scratch编程——Python编程
    </h4>
    <p>
     此时，我们就需要从scratch转到代码编程
     <br/>
     这个过渡可能大家在前期无法忍受，因为可能是大家研究了3年的启蒙图形化编程的缘故。但是，大家不用慌，因为前期的Python编程时极为简单的，只要大家稍稍学习即可，不用深入研究
     <br/>
     Python的环境搭建在这里不方便细说，大家可以看一下我的文章
     <a href="https://editor.csdn.net/md/?articleId=132636510" rel="nofollow">
      https://editor.csdn.net/md/?articleId=132636510
     </a>
     <br/>
     这是关于Python环境搭建的文章，大家可仔细端详
    </p>
    <h2>
     <a id="10_44">
     </a>
     三、10岁以后
    </h2>
    <h3>
     <a id="1_45">
     </a>
     1.推荐
    </h3>
    <p>
     这是代码编程的重点培养期，这期间应当适当学习代码编程
    </p>
    <h3>
     <a id="2_47">
     </a>
     2.软件
    </h3>
    <h4>
     <a id="1Python_48">
     </a>
     （1）Python
    </h4>
    <p>
     12岁以前，我们可以学习Python
     <br/>
     我有Python的低级教程，大家可以关注我去阅读
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3ee1ef5c81ccaa924faff6d8de330e83.png">
      <br/>
      Python不论是算法还是逻辑结构，相较于其他语言都是比较简单的
      <br/>
      大家学习完后，可以参加比赛来检验学习结果
     </img>
    </p>
    <h4>
     <a id="2C_55">
     </a>
     （2）C++
    </h4>
    <p>
     C++是与Python外耳熟能详的编程语言，当下的大部分语言中，C++与Python的使用率还是极高的
     <br/>
     所以，我们对C++的学习还是很重要
     <br/>
     当然，C++是非常困难的
     <br/>
     现在网络上流行一种说法：Python只要1个月就可以摸爬虫了，但是C++一个月还在研究for循环？！
     <br/>
     这个说法虽然略有夸张，但是还是比较准确的。
     <br/>
     但是，大家不要怕，因为学C++难免会碰壁，只要坚持下去就好了
     <br/>
     软件我推荐两个，一个是VS（visual studio），另一个是DEV-C++
     <br/>
     这两个软件对于初学者来说还是大有帮助的，大家可以多多学习
    </p>
