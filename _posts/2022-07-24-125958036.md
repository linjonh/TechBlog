    <h2>
     <a id="_0">
     </a>
     需求
    </h2>
    <p>
     ● 需要一个保存数据的仓库
     <br/>
     ● 仓库可在不同地方上传/同步数据
     <br/>
     ● 仓库安全性高些
     <br/>
     ● 仓库费用低些
     <br/>
     ● 仓库随处可用
     <br/>
     ● 仓库迁移容易些
     <br/>
     ● 更新时, 步骤方便
     <br/>
     iPhone免越狱Git服务器, 能完美解决上述需求. 人与Git服务器同在, 随时可更新笔记/项目代码, 无需收费, 需要时开启服务器即可, 更换iPhone手机时, 手机自带迁移数据也能把Git服务器数据一起迁移.
    </p>

    <h3>
     <a id="1_iSH_ssh_19">
     </a>
     1. iSH 安装及ssh功能启用
    </h3>
    <h4>
     <a id="11_iSH_20">
     </a>
     1.1 iSH安装
    </h4>
    <blockquote>
     <p>
      apk add neofetch openssh neovim git
     </p>
    </blockquote>
    <h4>
     <a id="12_SSH_root__22">
     </a>
     1.2 设置SSH root 密码
    </h4>
    <blockquote>
     <p>
      ssh-keygen -A
      <br/>
      passwd
      <br/>
      此处输入root密码(123456)
     </p>
    </blockquote>
    <h4>
     <a id="13_rootssh_26">
     </a>
     1.3 打开root用户登录ssh权限
    </h4>
    <blockquote>
     <p>
      echo ‘PermitRootLogin yes’ &gt;&gt; /etc/ssh/sshd_config
     </p>
    </blockquote>
    <h4>
     <a id="14__sshd_28">
     </a>
     1.4 启用 sshd
    </h4>
    <blockquote>
     <p>
      /usr/sbin/sshd
     </p>
    </blockquote>
    <h3>
     <a id="2_CMD_30">
     </a>
     2. CMD命令窗口
    </h3>
    <h4>
     <a id="21_SSHiSH_31">
     </a>
     2.1 远程SSH连接iSH
    </h4>
    <blockquote>
     <p>
      ssh root@手机ip(如: ssh root@192.168.1.129) // 电脑ip需要同一个网段, 如192.168.1.x
      <br/>
      输入root密码(123456)
     </p>
    </blockquote>
    <h4>
     <a id="22_projectgit_cmd_34">
     </a>
     2.2 创建project.git仓库, 电脑cmd终端:
    </h4>
    <pre><code>    iPhone:~# pwd                               		// 查看路径
    /root 
    iPhone:~# 
    iPhone:~# mkdir project.git                    		// 创建文件夹
    iPhone:~# ls
    iPhone:~# cd project.git/                      		// 进入文件夹
    iPhone:~# git --bare init                      		// 初始化仓库
    // 路径 /private/var/root/project.git/
    Initialized empty Git repository in /private/var/root/project.git/      
</code></pre>
    <h3>
     <a id="3_Git__44">

     3. Git 终端

     <a id="31_clone_projectgit_45">

[remote "iphone"]
url = ssh://root@192.168.1.184/root/projects/notes/notes.git
fetch = +refs/heads/*:refs/remotes/iphone/*
</code></pre>

    <h3>
     <a id="5__62">
     </a>
     5. 免密配置
    </h3>
    <h4>
     <a id="51_PC_63">
     </a>
     5.1 PC生成公钥秘钥
    </h4>
    <pre><code>命令提示符窗口输入:
</code></pre>
    <blockquote>
     <p>
      ssh-keygen -t rsa // 一直回车
      <br/>
      C:\Users\Administrator.ssh\ 目录将生成两个文件:
      <br/>
      C:\Users\Administrator.ssh\id_rsa // 私钥
      <br/>
      C:\Users\Administrator.ssh\id_rsa.pub // 公钥
     </p>
    </blockquote>
    <h4>
     <a id="52_iPhone_69">
     </a>
     5.2 iPhone文件权限设置
    </h4>
    <pre><code>命令提示符窗口输入:
</code></pre>
    <pre><code class="prism language-bat">ssh root@手机ip(如: ssh root@192.168.1.129)
输入root密码(123456)
// cd到手机 .ssh目录, 如目录不存在, 则新建 .ssh目录
iPhone:~# cd ~/.ssh
-ash: cd: can't cd to /root/.ssh: No such file or directory            
iPhone:~# mkdir .ssh
// 进入 .ssh 目录
iPhone:~# cd .ssh
// 给目录及文件授权
iPhone:~/.ssh# touch authorized_keys
iPhone:~/.ssh# ls
authorized_keys
iPhone:~/.ssh# chmod 700 ~/.ssh
iPhone:~/.ssh# chmod 600 ~/.ssh/authorized_keys

iPhone:~/.ssh# ls
authorized_keys  id_rsa.pub
iPhone:~/.ssh# cat id_rsa.pub &gt;&gt; ~/.ssh/authorized_keys
</code></pre>
    <h4>
     <a id="53__iphone_91">
     </a>
     5.3 复制公钥文件到 iphone:
    </h4>
    <pre><code>window → cmd → scp C:\Users\Administrator\.ssh\id_rsa.pub root@192.168.1.49:/root/.ssh
password:
</code></pre>
    <h4>
     <a id="54_iPhone__94">
     </a>
     5.4 iPhone 添加公钥:

    <pre><code>window → cmd → ssh root@192.168.1.129

iPhone: cat id_rsa.pub &gt;&gt; ~/.ssh/authorized_keys
</code></pre>

    <h3>
     <a id="_Permission_denied_please_try_again_101">
     </a>
     问题: Permission denied, please try again.
    </h3>
    <pre><code>● 原因: 
iSH app 复制黏贴执行的的 echo 'PermitRootLogin yes' &gt;&gt; /etc/ssh/sshd_config 有时候会写入失败.
● 解决:
方式1. iSH app 手动输入:
	echo 'PermitRootLogin yes' &gt;&gt; /etc/ssh/sshd_config
	重启App.
方式2. iSH app 手动cd到目录ssh, 然后手动输入:
	echo 'PermitRootLogin yes' &gt;&gt; sshd_config   
	重启App.
</code></pre>

    <h3>
     <a id="__git_pull_iphone____fatal_unable_to_access_http1921681129rootprojectsnotesnotesgit_118">
     </a>
     问题: git pull iphone 时, 报错 fatal: unable to access ‘http://192.168.1.129/root/projects/notes/notes.git/’:
    </h3>
    <pre><code>git bash:
    fatal: unable to access 'http://192.168.1.129/root/projects/notes/notes.git/': 
    Failed to connect to 192.168.1.129 port 80: Connection refused
● 原因:
     ssh 的链接填错导致
● 解决:
    .git/config → 修改 iphone 的url 为:
        [remote "iphone"]
            url = ssh://root@192.168.1.129/root/projects/notes/notes.git
            fetch = +refs/heads/*:refs/remotes/iphone/* 
</code></pre>
