      <p>
       <span style="font-family:arial;font-size:medium;">
        1.HTTP Live Streaming
       </span>
      </p>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         HTTP Live Streaming 是由 Apple 提出的基于 HTTP 的流媒体传输协议。 它将一整个音视频流切割成可由 HTTP 下载的一个个小的音视频流，并生成一个播放列表（M3U8），客户端只需要获取资源的 M3U8 播放列表即可播放音视频。可实现直播和点播。
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          HLS是以点播的技术方式来实现直播，
         </span>
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          它的延迟一般总是会高于普通的流媒体直播协议。
         </span>
        </span>
       </span>
      </p>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
         </span>
        </span>
       </span>
      </p>
      <p style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
       需要研究并实现以下技术关键点
      </p>
      <ol class="list-paddingleft-2">
       <li>
        <p>
         采集视频源和音频源的数据
        </p>
       </li>
       <li>
        <p>
         对原始数据进行H264编码和AAC编码
        </p>
       </li>
       <li>
        <p>
         视频和音频数据封装为MPEG-TS包
        </p>
       </li>
       <li>
        <p>
         HLS分段生成策略及m3u8索引文件
        </p>
       </li>
       <li>
        <p>
         HTTP传输协议
        </p>
       </li>
      </ol>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          <br/>
         </span>
        </span>
       </span>
      </p>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          支持HTML5的浏览器
          <br/>
         </span>
        </span>
       </span>
       <br/>
      </p>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          2.RTMP（
          <span style="font-size:12px;line-height:20px;">
           Real Time Messaging Protocol（实时消息传送协议）
          </span>
          ）
         </span>
        </span>
       </span>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         基于TCP/IP协议，支持Flash产品的浏览器，支持HTML5的浏览器（文件播放良好，实况直播支持较差）
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         <br/>
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         3.RTSP/TRCP/RTP
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         基于UDP协议 支持播放器
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         视频数据由RTP协议传输
         <br/>
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         视频质量由RTCP控制
        </span>
       </font>
      </p>
      <p>
       <font color="#393939" face="black verdana, arial, helvetica, sans-serif">
        <span style="font-size:14px;line-height:21px;">
         视频控制由RTSP提供（例如，前进，后退，暂停等）
        </span>
       </font>
      </p>
      <p>
       <span style="font-family:arial;font-size:medium;">
        <span style="color:#393939;font-family:helvetica, arial, freesans, clean, sans-serif;font-size:13px;line-height:18px;">
         <span style="font-family:'black verdana', arial, helvetica, sans-serif;font-size:14px;line-height:21px;">
          <br/>
         </span>
        </span>
       </span>
      </p>
