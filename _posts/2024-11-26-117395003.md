    <h2>
     <a id="_0">
     </a>
     微信小程序开发工具能正常请求后台数据，手机预览请求失败
    </h2>
    <h2>
     <a id="_3">
     </a>
     问题描述
    </h2>
    <p>
     微信开发者工具调试完想用手机预览效果，结果发现凡是后台请求都失败。
    </p>
    <h2>
     <a id="_7">
     </a>
     原因分析
    </h2>
    <p>
     可能的原因有三个：
    </p>
    <ol>
     <li>
      微信开发者工具详情里的本地设置，没有勾选上不校验合法域名等选项
     </li>
     <li>
      手机的网络和电脑的网络不在同一网段
     </li>
     <li>
      开发者工具用后台 ip 写成了 localhost
     </li>
    </ol>
    <h2>
     <a id="_15">
     </a>
     解决办法
    </h2>
    <p>
     针对上面的解决办法如下：
    </p>
    <ol>
     <li>
      微信开发者工具-右侧详情-本地设置-勾选上不校验合法域名
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d45c7083f222f04a9bf7b20e5dd75b3f.png#pic_center"/>
    </p>
    <ol start="2">
     <li>
      <p>
       是手机连接的网络和电脑一致，也就是连同一个 wifi，或者手机打开热点，电脑连接上这个热点
      </p>
     </li>
     <li>
      <p>
       把后端请求链接的 ip 设置成电脑本机 ip，不要写成 localhost
      </p>
     </li>
    </ol>
