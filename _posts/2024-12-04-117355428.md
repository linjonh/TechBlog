     magic4.0跟harmonyos,支持升级Harmony 2.0 Magic UI 4.0 9月中旬招募公测
    </h1>
   </div>
  </div>
 </div>

  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <div style="font-size:16px;">
     <p>
      Magic UI 4.0系统将于9月中旬开始招募公测，适配荣耀30系列以及荣耀V30系列产品，后续同样支持升级为HarmonyOS 2.0系统。
     </p>
     <p>
      【PChome手机频道资讯报道】9月10日，华为开发者大会(HDC 2020)正式召开，正式推出HarmonyOS 2.0与EMUI 11操作系统。与此同时，荣耀在微博官宣，Magic UI 4.0系统也将于9月中旬开始招募公测，
     </p>
     <p align="center">
      <img alt="1be6703b21eae4e3ffc58655934a9f90.png" src="https://i-blog.csdnimg.cn/blog_migrate/2822349abf28cc4c83507bf95e14f959.jpeg"/>
     </p>
     <p>
      Magic UI 4.0广泛应用分布式技术，或具有与EMUI 11相同的UX设计与隐私安全保护功能。与此同时，Magic UI 4.0在AI智慧等方面也重大升级，将派生出多种全新玩法。
     </p>
     <p align="center">
      <img alt="96c69203afeddb68c488e72b0e6f0ebd.png" src="https://i-blog.csdnimg.cn/blog_migrate/a7d18358f44a98262a9212a55784495a.png"/>
     </p>
     <p>
      Magic UI 4.0也将采用一镜到底的动效设计理念，有效凸显核心信息，适合进行灵活多变的设备间信息共享。分布式技术的加入，带来丰富的设备间交互功能，无论是文件还是正在进行的操作选项，都可突破设备本身限制，大大提升工作与生活效率。手机、Pad、PC与IoT设备间的协作等级也会得到明显加强。
     </p>
     <p align="center">
      <img alt="f9fc3ebe048240f584bd36a6359b0395.png" src="https://i-blog.csdnimg.cn/blog_migrate/456a2059fab1ee11a9f612b0d7854994.png"/>
     </p>
     <p>
      Magic UI 4.0或同样具有视频AI字幕功能，能够直接识别视频语音内容，支持中英、中日、中韩全场景语言互译，甚至能够识别、翻译图片上的内容，打破形式对翻译的限制。Magic UI 4.0具有极高的的数据信息安全等级和全方位的隐私保护措施，并获得多项权威认证，其安全隐私表现也不会差。无论是图片还是备忘录上的敏感数据，都能够得到有效的保护。
     </p>
     <p>
      Magic UI 4.0系统将于9月中旬开始招募公测，适配荣耀30系列以及荣耀V30系列产品，后续同样支持升级为HarmonyOS 2.0系统。此次公测能够大大完善Magic UI 4.0的系统功能，并对正式版设备支持与部分功能选项的优化扩展提供参考。届时，还请各位“花粉”踊跃参与本次公测。
     </p>
