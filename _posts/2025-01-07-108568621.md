     EntityFrameworkCore生成数据库方法
    </h3>
    <p>
     <strong>
      使用EFCore生成数据库表有下面的三种方式：
     </strong>
    </p>
    <ol>
     <li>
      代码生成
     </li>
     <li>
      程序包管理器控制台迁移
     </li>
     <li>
      命令行迁移
     </li>
    </ol>
    <p>
     下面分别介绍上面的三种方法。
    </p>
    <p>
     <strong>
      一、代码生成
     </strong>
     <br/>
     在程序里面执行下面的代码可以在运行时生成数据库：
    </p>
    <pre><code class="prism language-csharp">dbContext<span class="token punctuation">.</span>Database<span class="token punctuation">.</span><span class="token function">EnsureCreated</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     其中dbContext是数据上下文的实例对象。看下面的代码：
    </p>
    <pre><code class="prism language-csharp"><span class="token keyword">using</span> EFCore<span class="token punctuation">.</span>Data<span class="token punctuation">;</span>
<span class="token keyword">using</span> System<span class="token punctuation">;</span>

<span class="token keyword">namespace</span> EFCore<span class="token punctuation">.</span>Con
<span class="token punctuation">{<!-- --></span>
    <span class="token keyword">class</span> <span class="token class-name">Program</span>
    <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">Main</span><span class="token punctuation">(</span><span class="token keyword">string</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span>
        <span class="token punctuation">{<!-- --></span>
            Console<span class="token punctuation">.</span><span class="token function">WriteLine</span><span class="token punctuation">(</span><span class="token string">"Hello World!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
            <span class="token comment">// 实例化数据上下文对象</span>
            <span class="token class-name">EFDbContext</span> dbContext <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">EFDbContext</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
            <span class="token comment">// 生成数据库</span>
            <span class="token keyword">bool</span> tfTrue <span class="token operator">=</span> dbContext<span class="token punctuation">.</span>Database<span class="token punctuation">.</span><span class="token function">EnsureCreated</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
            <span class="token comment">//也可以先通过下面这句先删除数据库再创建新数据库</span>
            <span class="token comment">//dbContext.Database.EnsureDeleted();</span>
            <span class="token keyword">if</span> <span class="token punctuation">(</span>tfTrue<span class="token punctuation">)</span>
            <span class="token punctuation">{<!-- --></span>
                Console<span class="token punctuation">.</span><span class="token function">WriteLine</span><span class="token punctuation">(</span><span class="token string">"数据库创建成功!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
            <span class="token punctuation">}</span>
            <span class="token keyword">else</span>
            <span class="token punctuation">{<!-- --></span>
                Console<span class="token punctuation">.</span><span class="token function">WriteLine</span><span class="token punctuation">(</span><span class="token string">"数据库创建失败!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
            <span class="token punctuation">}</span>

            Console<span class="token punctuation">.</span><span class="token function">ReadKey</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     我们去数据库查看，发现会自动创建数据库，并且添加了表。
    </p>
    <p>
     <strong>
      二、程序包管理器控制台迁移
     </strong>
     <br/>
     使用程序包管理器控制台迁移的方式分为三个步骤。
     <br/>
     1、安装Microsoft.EntityFrameworkCore.Tools包
     <br/>
     由于迁移需要使用到Microsoft.EntityFrameworkCore.Tools包，所以先要在EFCore.Data类库项目上安装这个包，直接在NuGet里面安装即可。
     <br/>
     <em>
      这里是把EFCore安装在一个单独的类库项目里，所以作为启动项的控制台程序也要安装这个包。
     </em>
    </p>
    <p>
     2、添加迁移命令
     <br/>
     使用下面的命令可以生成一个迁移：
    </p>
    <pre><code class="prism language-csharp">Add<span class="token operator">-</span><span class="token class-name">Migration</span> Init_First
</code></pre>
    <p>
     其中Init_First是为这次迁移起的一个名称，如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b850cda0bf3538b6399a2241b09bd7d4.png#pic_center">
      <br/>
      执行完迁移命令以后，会自动生成几个文件：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b48d0b2f62282b61d26f5a2628e4de3.png#pic_center">
       <br/>
       20200224163036_Init_First.cs：是针对本次迁移生成的文件。每次迁移都会生成一个对应的迁移文件。里面有一个Up方法和一个Down方法，Up方法表示要执行的操作，Down方法相当于回滚，如果需要还原到上一个状态，就会执行Down方法。
       <br/>
       EFDbContextModelSnapshot.cs：生成的配置文件。只会生成一次。
      </img>
     </img>
    </p>
    <p>
     3、更新数据库
     <br/>
     执行完上面的迁移命令，执行下面的命令就可以更新到数据库：
    </p>
    <pre><code class="prism language-csharp">Update<span class="token operator">-</span>Database
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e3ca396ff731965b97879bdf94c65257.png#pic_center">
      <br/>
      这样就会自动生成数据库。
     </img>
    </p>
    <p>
     <strong>
      三、命令行迁移
     </strong>
     <br/>
     我们可以在PowerShell或者CMD命令行窗口里面进行迁移，使用命令行迁移分为下面的两个步骤。
    </p>
    <p>
     1、添加迁移
     <br/>
     首先我们要进入EFCore.Data目录下面，然后执行下面的命令添加迁移：
    </p>
    <pre><code class="prism language-csharp">dotnet ef migrations <span class="token keyword">add</span> Initial
</code></pre>
    <p>
     如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1f90a3af5fbdbae349bc72afc9bdd206.png#pic_center">
      <br/>
      迁移之后同样会生成上面所讲的文件。
     </img>
    </p>
    <p>
     2、更新数据库
     <br/>
     执行下面的命令会将配置更新到数据库：
    </p>
    <pre><code class="prism language-csharp">dotnet ef database update
</code></pre>
    <p>
     如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/364b1f398dbe8783acd4028950ebd391.png#pic_center">
      <br/>
      执行成功以后就会生成数据库。
     </img>
    </p>
    <p>
     <strong>
      总结：
     </strong>
     <br/>
     上面介绍了三种生成数据库表的方式，代码生成的方式适合于控制台程序，这种方式不太常用。尤其是在Web项目里面，经常使用第二种和第三种方式。
    </p>
