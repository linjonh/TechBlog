    <p>
     实验室的开发环境在Linux操作系统下，时不时就需要打开Linux环境去操作，而且需要本地编译或者远程SSH。这时候window和Linux切换很不方便。本科的做法就是window+虚拟机的Linux，那时候装上了，但是不知道怎么用，又慢又卡的。
     <br/>
     到了研究生17年末18年出，跟舍友交流过程中发现win10电脑有内嵌的（bash/shell） Linux系统。之后回来电脑开启了新世界的大门。18年之后win10应用商店对Linux版本进行了更新，win10应用商店提供多种Linux版本，然后就安装了，没想到之后有很大的用处。获取的方法很简单，下面简单说明一下。
    </p>
    <h2>
     <a id="_4">
     </a>
     获取
    </h2>
    <p>
     1.首先我的电脑是win10专业版1809版本，64位，处理器i5-4210M，内存12G。14年的电脑了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f61a2682dccd3848102cd46318a0fb1e.png">
      <br/>
      2.打开设置&gt;更新与安全&gt;开发者选项，选择开发人员模式。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ab28415eeea1a44a7636d55192f5077c.png">
       <br/>
       3.在任务栏的搜索窗口搜索“控制面板”，然后选择程序，到下面的界面。选择“启用或关闭window功能”，在window功能窗口勾选“适用于Linux的window子系统”，确定。需要重启系统就重启。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b4e0df59209cf25df84e143eb9a38e0.png">
        <br/>
        4.前往应用商店，在右上方的搜索栏搜索Ubuntu，在搜索结果处选择Ubuntu，然后获取，它就自动安装上了。
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/95de80d49f0a36c6ed15083c4ef3c940.png"/>
    </p>
    <p>
     5.安装成功，点击启动。提示创建帐户名和密码了，自己创建即可，之后可以正常使用了。我这里没有设置用户名密码直接在root下使用了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/987df5ad9ee3973e8bc29c43447fac73.png"/>
    </p>
    <h2>
     <a id="_18">
     </a>
     启动
    </h2>
    <p>
     启动方式大概有三种。
     <br/>
     1.在任务栏搜索bash，点击bash on ubuntu on windows应用，打开程序。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/faecdf6b92d16aefdf514579130cf21b.png">
      2.在应用商店里面安装的应用里找到安装的 Ubuntu ，点击启动，打开程序。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f8b2727a3acf2381c35fa0417c62e96c.png">
       <br/>
       3.在window菜单栏找到 Ubuntu ，然后右击&gt;更多&gt;固定到任务栏。到时候就可以直接在任务栏打开了，很方便。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d06cc803963a6ad1b9562e71c6c6f711.png"/>
      </img>
     </img>
    </p>
    <h2>
     <a id="_25">
     </a>
     预览及使用
    </h2>
    <h4>
     <a id="1Ubuntu_26">
     </a>
     1.查看Ubuntu版本，位数，存储情况，网络状况等。
    </h4>
    <p>
     常用命令如下：
    </p>
    <blockquote>
     <p>
      uname -a
      <br/>
      lsb_release -a
      <br/>
      df -h
      <br/>
      ifconfig
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d90f41efddc084498fa211a5bd728205.png"/>
     <br/>
     ①可以发现内置的Linux系统内显示的window系统硬盘的路径都在/mnt 目录下，cd进到相应的目录下就可以访问电脑上的文件。
     <br/>
     ②可以发现内置的Linux系统使用的IP地址与window系统的IP是共享的。
    </p>
    <h4>
     <a id="2_Linux_37">
     </a>
     2. 安装Linux常用软件
    </h4>
    <p>
     首先更新一下软件源，这里就先不设置别的镜像库了。有别的教程会教你如何更改镜像库。
    </p>
    <blockquote>
     <p>
      sudo apt-get update
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ef4c823fc18e6cd9e0eb4bba7c1af990.png"/>
     <br/>
     等着更新好了就可以开始apt-get install软件了。
     <br/>
     常用的Linux下的应用有：ssh，java/javac，gcc/g++，python，go，git，vim/vi等等。输入名称一般都会有安装提示，按照提示安装就好，网上也有很多教程。
    </p>
    <h2>
     <a id="_45">
     </a>
     结语
    </h2>
    <p>
     接下来就可以开始愉快的在window系统下方便的使用Linux了。
    </p>
