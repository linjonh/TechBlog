    <p>
     在读研期间的研究方向与图像处理有一点相关，后来由于工作可能会有需要，所以晚上回去也闭门研究了OpenCV和图像处理相关的知识。也准备写一个OpenCV系列的专栏作为自己的学习笔记，也分享给大家，如果有写错的地方也欢迎大家及时指出。
    </p>
    <p>
     好了这篇文章作为OpenCV系列的第一篇，我们先介绍一下OpenCV的开发环境。在学习OpenCV的过程中也搜集了网上的很多资料，发现多为C++、Python开发，很少使用Java，所以我打算使用Java来学习OpenCV。
    </p>

      话不多说，我们开始。

    <h3>
     <a id="1OpenCV_6">
     </a>
     1.下载和安装OpenCV
    </h3>
    <p>
     IDEA我们不说了，Java开发的基本上多使用这个开发工具。安装OpenCV相对比较简单，就是傻瓜式的下载，解压。
     <br/>
     <a href="https://opencv.org/releases/" rel="nofollow">
      OpenCV官网
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0b4931eb405ce952b1563b9d52a549cb.png">
      <br/>
      本次我们是在Windows下学习开发，所以就直接下载Windows的版本。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d4ac994f63fe39eb05f57591ddac8c23.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6afeb3d528ab89857f754b199d5f4e0d.png">
        <br/>
        等过一段时间后就会将OpenCV的文件解压到指定的文件夹中，我们找到build文件夹下，打开java文件夹，会发现x64、x86文件夹，由于我的电脑是64位的所以使用x64文件夹下的
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/51a16fa2b86942895bb9d0fadf6fdd97.png">
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa9f2d1d606d071f15cd2843d7e2c06c.png">
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/33271bed234ece3bddcf8970500e5e01.png">
           <br/>
           我们找到dll文件，
           <strong>
            注意
           </strong>
           dll文件是我们使用Java学习OpenCV的关键，我们需要将此文件放到JDK的bin文件夹下
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/28dcf1c614083cbf56b3f4c9e7bc6329.png">
            <br/>
            此外还有一个opencv-java460的jar包就需要我们在创建Java项目时，将其放到项目里。
            <br/>
            由于本人比较喜欢使用maven项目，所以本次就通过IDEA创建一个maven项目。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b3860736ba94e2050a11dc7f2f9699ba.png">
             <br/>
             找到Dependencies，点击右边的**+**号，将上述的jar包添加进去。
