    <p>
     <br/>
    </p>
    <p>
     <span style="font-size:24px">
      当需要多个场景时，就用到了场景转换，
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      <img alt="" src="https://img-blog.csdn.net/20150707143451207?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center"/>
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      这里呢，我有两个场景，一个move（开场镜头），另一个second（游戏场景）；
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      要想运行完开长镜头后接着运行游戏场景，需要用到一个函数；；；
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      Application.LoadLevel("second");引号里变是需要运行的场景，即要转换的场景；
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      当然这是Csharp脚本；
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      这里呢，我用到了GUI的按钮来启动第二个场景，
     </span>
    </p>
    <p>
     <span style="font-size:24px">
      开长镜头结束后，我们需要点一下按钮，比如Play，我们就进入游戏了；
     </span>
    </p>
    <span style="font-size:24px">
    </span>
    <pre><code class="language-csharp">using UnityEngine;
using System.Collections;
public class cameramoveplay : MonoBehaviour {
    void Start () {
       
	}
    void Updata(){

    }
    void OnGUI()
    {
        if(GUILayout.Button("PLAY"))
            {
                Application.LoadLevel("second");
             }
    }
}</code></pre>
    <p>
     <br/>
    </p>
    <p>
     <br/>
    </p>
    <span style="font-size:24px">
    </span>
    <p>
     忘记了一个问题，就是两个场景还需要在File-&gt;Building Setting中设置添加场景，否则也不能实现切换成功。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20150707144919250?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center"/>
    </p>
    <p>
     需要的场景加进来，这样才会成功
    </p>
    <p>
     <br/>
    </p>
    <p>
     <br/>
    </p>
    <p>
     将此段代码挂载给主摄像机，当我们点击PLAY按钮时，我们就会进入下一个场景了；
    </p>
    <p>
     是不是很容易啊，欢迎同学互相交流学习；；
