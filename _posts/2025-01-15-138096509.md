    <h3>
     <a id="_NebulaGraph_33">
     </a>
     二.部署 NebulaGraph
    </h3>
    <h4>
     <a id="1_35">
     </a>
     1.拉取代码
    </h4>
    <pre><code class="prism language-apl">git clone -b release-3.6 https://github.com/vesoft-inc/nebula-docker-compose.git
</code></pre>
    <h4>
     <a id="2_41">
     </a>
     2.启动容器
    </h4>
    <pre><code class="prism language-apl"># 工作目录
cd  /data/nebula-docker-compose/

# 启动
docker-compose up -d
</code></pre>
    <h3>
     <a id="_NebulaGraph_51">
     </a>
     三.连接 NebulaGraph
    </h3>
    <h4>
     <a id="1_53">
     </a>
     1.查看容器
    </h4>
    <pre><code class="prism language-apl">docker-compose ps
</code></pre>
    <h4>
     <a id="2_59">
     </a>
     2.进入容器
    </h4>
    <pre><code class="prism language-apl">docker exec -it nebula-docker-compose-console-1 /bin/sh
</code></pre>
    <h4>
     <a id="3_65">
     </a>
     3.连接服务端
    </h4>
    <pre><code class="prism language-apl">./usr/local/bin/nebula-console -u root -p nebula --address=graphd --port=9669
</code></pre>
    <blockquote>
     <p>
      默认情况下，身份认证功能是关闭的，只能使用已存在的用户名（默认为
      <code>
       root
      </code>
      ）和任意密码登录。如果想使用身份认证，请参见
      <a href="https://docs.nebula-graph.com.cn/master/7.data-security/1.authentication/1.authentication/" rel="nofollow">
       身份认证
      </a>
      。
     </p>
    </blockquote>
    <h4>
     <a id="4_73">
     </a>
     4.查看集群状态
    </h4>
    <p>
     <img alt="image-20240327180108901" src="https://i-blog.csdnimg.cn/blog_migrate/b920157191d5c1f6713814c19782e095.png"/>
    </p>
    <h3>
     <a id="_SQL_77">
     </a>
     四.基础 SQL
    </h3>
    <h4>
     <a id="1_hosts_79">
     </a>
     1.显示 hosts
    </h4>
    <pre><code class="prism language-apl">show hosts;
</code></pre>
    <h4>
     <a id="2_SPACE_85">
     </a>
     2.创建 SPACE
    </h4>
    <pre><code class="prism language-apl">CREATE SPACE IF NOT EXISTS course(vid_type=FIXED_STRING(256), partition_num=1, replica_factor=1);
</code></pre>
    <h4>
     <a id="3_TAG_91">
     </a>
     3.创建 TAG
    </h4>
    <pre><code class="prism language-apl">USE course;
</code></pre>
    <pre><code class="prism language-apl">CREATE TAG IF NOT EXISTS entity(name string);
CREATE EDGE IF NOT EXISTS relationship(relationship string);
CREATE TAG INDEX IF NOT EXISTS entity_index ON entity(name(256));
</code></pre>
    <h4>
     <a id="4_103">
     </a>
     4.插入知识图谱元数据
    </h4>
    <pre><code class="prism language-apl">insert vertex entity values "鲁迅":("鲁迅");
insert vertex entity values "作家":("作家");
insert edge relationship values "鲁迅"-&gt;"作家":("是");
</code></pre>
    <h4>
     <a id="5_111">
     </a>
     5.退出
    </h4>
    <pre><code class="prism language-apl"># 退出数据库
exit

#退出容器
exit
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5ba46fad724cd75cf4e3d1c9229048e0.png"/>
    </p>
    <h3>
     <a id="_122">
     </a>
     五.常见查询
    </h3>
    <h4>
     <a id="1_124">
     </a>
     1.显示顶点
    </h4>
    <pre><code class="prism language-apl"># 显示顶点字段
show tags;

# 显示所有实体
match (e) return e;

# 显示名称
match (e) return e.`entity`.`name`;
</code></pre>
    <h4>
     <a id="2_137">
     </a>
     2.显示边
    </h4>
    <pre><code class="prism language-apl"># 返回会去重

# 显示边字段
show edges;

# 边的解释信息,关系

MATCH ()-[e]-&gt;() RETURN e
</code></pre>
    <h4>
     <a id="3_150">
     </a>
     3.所有节点的关联关系
    </h4>
    <pre><code class="prism language-apl"># 不会去重
MATCH ()-[e:relationship]-() RETURN e
MATCH ()-[e]-() RETURN e
</code></pre>
    <h4>
     <a id="4_158">
     </a>
     4.指定名词的关系查询
    </h4>
    <pre><code class="prism language-apl">MATCH (p:`entity`)-[e:relationship]-&gt;(m:`entity`)\
  WHERE p.`entity`.`name` == '鲁迅'\
RETURN p.`entity`.`name`, e.relationship, m.`entity`.`name`;
</code></pre>
    <h4>
     <a id="5_166">
     </a>
     5.删除顶点
    </h4>
    <pre><code class="prism language-apl">DELETE VERTEX "鲁迅";
</code></pre>
    <h4>
     <a id="6_space_172">
     </a>
     6.删除 space
    </h4>
    <pre><code class="prism language-apl"># 删除space
DROP SPACE course;
</code></pre>
