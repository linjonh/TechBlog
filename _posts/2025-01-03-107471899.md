     <p>
      Python doesn’t have any specific data type as an array. We can use List that has all the characteristics of an array.
     </p>
     <p>
      Python没有任何特定的数据类型作为数组。 我们可以使用具有数组所有特征的List。

     <p>
      Python array module can be used to create an array of integers and floating-point numbers.
     </p>
     <p>
      Python数组模块可用于创建整数和浮点数的数组。

     <h3>
      1. Python添加到数组
      <span style="font-weight: bold;">
       (
      </span>
      <span id="1-python-add-to-array" style="font-weight: bold;">
       1. Python add to Array
      </span>
      <span style="font-weight: bold;">
       )
      </span>
     </h3>
     <ul>
      <li>
       If you are using
       <a href="https://www.journaldev.com/14353/python-list" rel="nofollow">
        List
       </a>
       as an array, you can use its append(), insert(), and extend() functions. You can read more about it at
       <a href="https://www.journaldev.com/33182/python-add-to-list" rel="noopener noreferrer noopener noreferrer" target="_blank">
        Python add to List
       </a>
       .
       <p class="nodelete">
       </p>
       如果将
       <a href="https://www.journaldev.com/14353/python-list" rel="nofollow">
        List
       </a>
       用作数组，则可以使用其append（），insert（）和extend（）函数。 您可以在
       <a href="https://www.journaldev.com/33182/python-add-to-list" rel="noopener noreferrer noopener noreferrer" target="_blank">
        Python add to List中
       </a>
       阅读有关它的更多信息。
      </li>
      <li>
       If you are using array module, you can use the concatenation using the + operator, append(), insert(), and extend() functions to add elements to the array.
       <p class="nodelete">
       </p>
       如果使用的是数组模块，则可以使用+运算符，append（），insert（）和extend（）函数进行串联，以将元素添加到数组中。
      </li>
      <li>
       If you are using NumPy arrays, use the append() and insert() function.
       <p class="nodelete">
       </p>
       如果您使用的是NumPy数组，请使用append（）和insert（）函数。
      </li>
     </ul>
     <h3>
      2.使用数组模块将元素添加到数组
      <span style="font-weight: bold;">
       (
      </span>
      <span id="2-adding-elements-to-an-array-using-array-module" style="font-weight: bold;">
       2. Adding elements to an Array using array module
      </span>
      <span style="font-weight: bold;">
       )
      </span>
     </h3>
     <ul>
      <li>
       Using + operator: a new array is returned with the elements from both the arrays.
       <p class="nodelete">
       </p>
       使用+运算符：返回一个新数组，其中包含两个数组中的元素。
      </li>
      <li>
       append(): adds the element to the end of the array.
       <p class="nodelete">
       </p>
       append（）：将元素添加到数组的末尾。
      </li>
      <li>
       insert(): inserts the element before the given index of the array.
       <p class="nodelete">
       </p>
       insert（）：将元素插入到数组的给定索引之前。
      </li>
      <li>
       extend(): used to append the given array elements to this array.
       <p class="nodelete">
       </p>
       extend（）：用于将给定的数组元素附加到此数组。
      </li>
     </ul>
     <pre class="has"><code>import array

arr1 = array.array('i', [1, 2, 3])
arr2 = array.array('i', [4, 5, 6])

print(arr1)  # array('i', [1, 2, 3])
print(arr2)  # array('i', [4, 5, 6])

arr3 = arr1 + arr2
print(arr3)  # array('i', [1, 2, 3, 4, 5, 6])

arr1.append(4)
print(arr1)  # array('i', [1, 2, 3, 4])

arr1.insert(0, 10)
print(arr1)  # array('i', [10, 1, 2, 3, 4])

arr1.extend(arr2)
print(arr1)  # array('i', [10, 1, 2, 3, 4, 4, 5, 6])</code></pre>
     <h3>
      3.向NumPy数组添加元素
      <span style="font-weight: bold;">
       (
      </span>
      <span id="3-adding-elements-to-the-numpy-array" style="font-weight: bold;">
       3. Adding elements to the NumPy Array
      </span>
      <span style="font-weight: bold;">
       )
      </span>
     </h3>
     <ul>
      <li>
       append(): the given values are added to the end of the array. If the axis is not provided, then the arrays are flattened before appending.
       <p class="nodelete">
       </p>
       append（）：将给定值添加到数组的末尾。 如果未提供轴，则在附加之前将阵列弄平。
      </li>
      <li>
       insert(): used to insert values at the given index. We can insert elements based on the axis, otherwise, the elements will be flattened before the insert operation.
       <p class="nodelete">
       </p>
       insert（）：用于在给定索引处插入值。 我们可以基于轴插入元素，否则，将在插入操作之前将元素展平。
      </li>
     </ul>
     <pre class="has"><code>&gt;&gt;&gt; import numpy as np
&gt;&gt;&gt; np_arr1 = np.array([[1, 2], [3, 4]])
&gt;&gt;&gt; np_arr2 = np.array([[10, 20], [30, 40]])
&gt;&gt;&gt; 
&gt;&gt;&gt; np.append(np_arr1, np_arr2)
array([ 1,  2,  3,  4, 10, 20, 30, 40])
&gt;&gt;&gt;
&gt;&gt;&gt; np.append(np_arr1, np_arr2, axis=0)
array([[ 1,  2],
       [ 3,  4],
       [10, 20],
       [30, 40]])
&gt;&gt;&gt;
&gt;&gt;&gt; np.append(np_arr1, np_arr2, axis=1)
array([[ 1,  2, 10, 20],
       [ 3,  4, 30, 40]])
&gt;&gt;&gt; 
&gt;&gt;&gt; np.insert(np_arr1, 1, np_arr2, axis=0)
array([[ 1,  2],
       [10, 20],
       [30, 40],
       [ 3,  4]])
&gt;&gt;&gt; 
&gt;&gt;&gt; np.insert(np_arr1, 1, np_arr2, axis=1)
array([[ 1, 10, 30,  2],
       [ 3, 20, 40,  4]])
&gt;&gt;&gt;</code></pre>
