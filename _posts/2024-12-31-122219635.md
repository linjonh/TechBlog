    <h2>
     <a id="_79">
     </a>
     二、控制节点配置网络选项—私有网络
    </h2>
    <blockquote>
     <p>
      前一章我们在控制节点配置了公共网络，接下来，我们在controller节点上安装并配置私有网络组件。
     </p>
    </blockquote>
    <p>
     所需的安装组件已经装过了，接下来配置服务组件，编辑
     <code>
      /etc/neutron/neutron.conf
     </code>
     文件
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a2c378067785f3559fcbee496adf501e.png"/>
     <br/>
     在
     <code>
      [DEFAULT]
     </code>
     部分，启用路由服务和重叠的IP地址
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5846d7d69a26351a6b0d82075b9eded.png"/>
     <br/>
     接下来配置 Modular Layer 2 (ML2) 插件，ML2插件使用Linuxbridge机制来为实例创建layer－2虚拟网络基础设施。
     <br/>
     编辑
     <code>
      /etc/neutron/plugins/ml2/ml2_conf.ini
     </code>
     文件
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/78cb22e5d18915c496f4ddd0b990e3ca.png"/>
     <br/>
     在
     <code>
      [ml2]
     </code>
     部分，启用flat，VLAN以及VXLAN网络；
     <br/>
     启用VXLAN私有网络；
     <br/>
     启用Linuxbridge和layer－2机制
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ff813ece7a7987f473ebaba43a34c8e8.png"/>
     <br/>
     在
     <code>
      [ml2_type_vxlan]
     </code>
     部分，为私有网络配置VXLAN网络识别的网络范围
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8b39b429388357005fb9abf7b2893b14.png"/>
     <br/>
     对文件内容进行过滤查看
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2c1e1448663a25074054e3efdb41cdf4.png"/>
     <br/>
     接下来配置Linuxbridge代理，Linuxbridge代理为实例建立layer－2虚拟网络并且处理安全组规则。
     <br/>
     编辑
     <code>
      /etc/neutron/plugins/ml2/linuxbridge_agent.ini
     </code>
     文件
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7ec3638ac6919432cc33df82445e2475.png"/>
     <br/>
     在
     <code>
      [vxlan]
     </code>
     部分，启用VXLAN覆盖网络，配置覆盖网络的物理网络接口的IP地址，启用layer－2 population
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b262a9237d8411eacb8ff1fab5e68a5.png"/>
