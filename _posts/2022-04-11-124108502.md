 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     备份mysql数据库，可通过以下命令的方式：
    </p>
    <p>
     <strong>
      1、备份数据库表结构及数据
     </strong>
    </p>
    <p>
     导出的脚本中，只包含表结构和表中数据，不包含创建数据库语句，即不包含：CREATE DATABASE /*!32312 IF NOT EXISTS*/ `test` /*!40100 DEFAULT CHARACTER SET utf8 */;
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 test &gt; ./test.sql</code></pre>
    <p>
     <span style="color:#fe2c24;">
      其中
     </span>
     ，-h指定主机地址，-u指定数据库用户名，-p指定数据库密码，test为数据库。
    </p>
    <p>
     <strong>
      2、备份一张表student
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 test student &gt; ./student.sql</code></pre>
    <p>
     <strong>
      3、备份多张表，如student、grade
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 test student grade &gt; ./test.sql</code></pre>
    <p>
     <strong>
      4、备份整个数据库，包含创建数据库脚本
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 -B test &gt; ./test2.sql</code></pre>
    <p>
     <span style="color:#fe2c24;">
      通过指定参数
     </span>
     ：-B，备份整个数据。
    </p>
    <p>
     <strong>
      5、备份所有数据库
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 --all-databases &gt; back.sql</code></pre>
    <p>
     <strong>
      6、只备份表结构，不包含数据
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 -d test &gt; ./test3.sql</code></pre>
    <p>
     <span style="color:#fe2c24;">
      通过指定参数
     </span>
     ：-d
    </p>
    <p>
     <strong>
      7、只备份数据，不包含表结构
     </strong>
    </p>
    <pre><code>mysqldump -hlocalhost -uroot -p123456 -t test &gt; ./test4.sql</code></pre>
    <p>
     <span style="color:#fe2c24;">
      通过指定参数
     </span>
     ：-t
    </p>
    <p>
     <strong>
      8、忽略某个表，比如忽略student、grade表
     </strong>
    </p>
    <pre><code class="hljs">mysqldump --ignore-table=test.student --ignore-table=test.grade -hlocalhost -uroot -p123456 -d test &gt; ./test3.sql</code></pre>
    <p>
     注意：必须是test.student，即：
     <span style="color:#fe2c24;">
      数据库名称.表名称
     </span>
    </p>
