     数据结构在操作系统中的应用
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
      <span style="font-size: xx-small;">
       在计算机科学中是一门综合的专业基础课，研究非数值计算的程序设计问题中计算机的操作对象以及它们之间的关系和操作等等。
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        它和计算机的硬件，特别是计算机软件的研究有着密切的关系，是介于数学
       </span>
       <span lang="EN-US">
        ,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        计算机硬件和软件三者之间的一门核心课程。无论是汇编程序还是操作系统
       </span>
       <span lang="EN-US">
        ,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        都涉及到是数据元素在存储器中的分配问题。数据结构不仅涉及到图论，表，树的理论，现在还扩充到网络，集合代数论，格，关系等方面。它对操作系统的发展起推波助澜的作用，是操作系统的重要基础。
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
      <span style="font-size: xx-small;">
       操作系统是配置在计算机硬件上的第一层软件，是一组程序的集合，其他所有的软件如汇编程序等等，都将依赖操作系统的支持，取得它的服务。
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
      <span style="font-size: xx-small;">
       同时，操作系统也离不开一定软件的支持，它的数据如何分配，软件所采取的数据结构，算法，都将大大影响到本系统的能。无论是汇编程序还是操作系统，都涉及到是数据元素在存储器中的分配问题，所以不能把数据结构和操作系统视为两个独立无关的学科，而是应该把它们挂钩起来。
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
      <span style="font-size: xx-small;">
       下面就操作系统中几个具体问题谈谈数据结构在其中的作用。
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        首先
       </span>
       <span lang="EN-US">
        ,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        进程在共享临界资源发生的死锁问题。
       </span>
       <span lang="EN-US">
        Os
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的目标是提高系统的资源利用率和增加系统的吞吐量。如何使并发执行的的诸进程之间能有效地共享资源和相互合作，从而让程序的执行具有可再现，避免进程在争用临界资源时给系统造成混乱，是进程同步的主要任务。但多个进程之间并不知道其它进程的存在，死锁现象可能会出现。
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        这就需要我们在
       </span>
       <span lang="EN-US">
        0s
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        中采取适当的软件，保证诸进程能互斥的访问临界资源
       </span>
       <span lang="EN-US">
        .
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        最有代表的避免死锁的方法，是
       </span>
       <span lang="EN-US">
        Dijkstra
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的银行家算法，其中设置了若干数据结构
       </span>
       <span lang="EN-US">
        .
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        现将有关算法描叙如下：
       </span>
       <span lang="EN-US">
        <span style="mso-spacerun: yes;">
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">
        <span style="mso-spacerun: yes;">
        </span>
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        一，银行家算法中的数据结构
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">
        1
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ．可利用资源向量
       </span>
       <span lang="EN-US">
        Available
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        它是一个含有
       </span>
       <span lang="EN-US">
        m
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        个元素的数组，其中的每一个元素代表一类可利用的资源数目，其初始值是系统中所配置的该类全部可用资源数目，其数值随该类资源的的分配和回收而动态地改变。如果
       </span>
       <span lang="EN-US">
        Available[j]=k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        表示系统中现有
       </span>
       <span lang="EN-US">
        Rj
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类资源
       </span>
       <span lang="EN-US">
        k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        个。
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">
        2
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ．最大需求距阵
       </span>
       <span lang="EN-US">
        Max
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        这是一个
       </span>
       <span lang="EN-US">
        n*m
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的矩阵，它定义了系统中
       </span>
       <span lang="EN-US">
        n
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        个进程中的每一个对
       </span>
       <span lang="EN-US">
        m
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类资源的最大需求。如果
       </span>
       <span lang="EN-US">
        Max[i,j]=k,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        表示进程
       </span>
       <span lang="EN-US">
        i
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        需要
       </span>
       <span lang="EN-US">
        Rj
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类资源的最大数目为
       </span>
       <span lang="EN-US">
        k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        。
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">
        3
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ．分配矩阵
       </span>
       <span lang="EN-US">
        Allocatian
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        这是一个
       </span>
       <span lang="EN-US">
        n*m
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的矩阵，它定义了系统中每一类资源当前已分配的资源数。如果
       </span>
       <span lang="EN-US">
        Allocation[i,j]=k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ，表示进程
       </span>
       <span lang="EN-US">
        i
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        当前已分得
       </span>
       <span lang="EN-US">
        Rj
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类资源数目为
       </span>
       <span lang="EN-US">
        k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        。
       </span>
       <span lang="EN-US">
        4
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ．需求矩阵
       </span>
       <span lang="EN-US">
        Need
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        它是一个
       </span>
       <span lang="EN-US">
        n*m
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的矩阵，用以表示每一个进程尚需的各类资源数，如果
       </span>
       <span lang="EN-US">
        Need[i,j]=k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        表示进程
       </span>
       <span lang="EN-US">
        i
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        还需要
       </span>
       <span lang="EN-US">
        Rj
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类资源
       </span>
       <span lang="EN-US">
        k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        个
       </span>
       <span lang="EN-US">
        ,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        方能完成任务；
       </span>
       <span lang="EN-US">
        Need[i,j]=Max[i,j]-Allocatian[i,j]
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        二
       </span>
       <span lang="EN-US">
        ,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        银行家算法设
       </span>
       <span lang="EN-US">
        Request i
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        是进程
       </span>
       <span lang="EN-US">
        Pi
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        的请求向量。如果
       </span>
       <span lang="EN-US">
        Request[j]=k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ，表示进程
       </span>
       <span lang="EN-US">
        Pi
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        需求
       </span>
       <span lang="EN-US">
        k
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        个
       </span>
       <span lang="EN-US">
        Rj
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        类型的资源。当
       </span>
       <span lang="EN-US">
        Pi
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        发出资源请求后，系统按下述步骤进行检查：
       </span>
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">
        (1)
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        如果
       </span>
       <span lang="EN-US">
        Request(2)
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        如果
       </span>
       <span lang="EN-US">
        Request(3)
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        系统试探把要求的资源分配给进程
       </span>
       <span lang="EN-US">
        Pi,
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        并修改下面数据结构中的数值；
       </span>
       <span lang="EN-US">
        Available:=Available-Request I
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ；
       </span>
       <span lang="EN-US">
        Allocatian:=Allocatian i Request I
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ；
       </span>
       <span lang="EN-US">

      </span>
     </span>

    <p class="MsoNormal" style="margin: 7.8pt 0cm; text-indent: 21pt; line-height: 150%; mso-para-margin-top: .5gd; mso-para-margin-right: 0cm; mso-para-margin-bottom: .5gd; mso-para-margin-left: 0cm; mso-char-indent-count: 2.0;">
     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">

       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        ．首次适应算法
       </span>
       <span lang="EN-US">
        FF
       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
        。在此算法中，空闲区链按起始地址递增顺序排列，在进行内存分配时，从链首开始顺序查找，直到找到一个能满足其大小要求的空闲区为止。然后，再按照作业的大小，从该分区划出一块内存给请求者，余下的空闲分区仍留在空闲链中。本算法的特点是底地址端造留许多碎片，高地址端集中了大空闲区。这样有利于给后到的大作业分配大的内存空间。
       </span>
      </span>
     </span>
    </p>

     <span style="font-size: small;">
      <span style="font-size: xx-small;">
       <span lang="EN-US">

       </span>
       <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">

       </span>
      </span>
     </span>
    </p>

       <span lang="EN-US">

       </span>

      </span>

       </span>

       </span>
