    <p>
     原想通过python请求禅道登录接口，获取到session后再创建bug，试了很多次发现登录需要的rand随机数一直没有获取到（可能是我方法不对），遂放弃，查看禅道官方公布的
     <a href="https://www.zentao.net/book/apidoc-v1/664.html#1" rel="nofollow">
      API接口文档
     </a>
     后，可使用token进行后续的登录操作
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/79cd7607a04e3a620c52dde8df0e045b.png"/>
    </p>
    <h2>
     <a id="token_5">
     </a>
     一、获取token
    </h2>
    <p>
     import requests
    </p>
    <p>
     path = ‘http://XXXX/zentao/api.php/v1’
     <br/>
     body = {“account”: “admin”, “password”: “123456”}
     <br/>
     url = path + “/tokens”
     <br/>
     res = requests.post(url=url, json=body)#参数传递需用json格式，否则返回{‘error’: ‘登录失败，请检查您的用户名或密码是否填写正确。’}
     <br/>
     token = (res.json())[‘token’]
     <br/>
     print(token)
    </p>
    <h2>
     <a id="_16">
     </a>
     二、获取用户基本信息
    </h2>
    <p>
     userUrl = path + “/user”
     <br/>
     headers = {
     <!-- -->
     <br/>
     “Token”: token
     <br/>
     }
     <br/>
     user = requests.get(userUrl, headers=headers)
    </p>
    <h2>
     <a id="BUG_22">
     </a>
     三、获取BUG详细
    </h2>
    <p>
     bugDeUrl = path + “/bugs/id”#需替换为实际的bugid值
     <br/>
     bugDe = requests.get(bugDeUrl,headers=headers)
     <br/>
     print(bugDe.json())
    </p>
