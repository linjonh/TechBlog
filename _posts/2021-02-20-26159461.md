      1. 首先安装window下Python环境:

       </a>
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; font-family:Arial; font-size:14px; line-height:26px">

      我安装的是: python-3.4.0.msi

     <span style="font-family:'Comic Sans MS'">
      <span style="line-height:24px">
       1.1  点击下载好的文件按提示一步一步安装
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-family:'Comic Sans MS'">
      <span style="line-height:24px">
       1.2 检测 Python 是否安装成功:
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-family:'Comic Sans MS'">
      <span style="line-height:24px">
       1.2.1 "开始" -&gt; "运行" -&gt; "cmd" 输入 python, 如果出现一大段的说明文字, 那恭喜您, windows下的 Python 安装好了
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-family:'Comic Sans MS'">
      <span style="line-height:24px">
       1.2.2 如果提示不是内部或外部命令, 那说明安装没有自动把 python 的文件路径自动添加到环境变量中去.
      </span>
     </span>
    </p>

     </span>

     </span>
