    <p>
     Python3 可在Windows，Mac OS和大多数Linux操作系统运行。本教程主要介绍在Windows环境下安装Python3。之所以用python3系列是因为Python2.7再过几年官方就不会再支持了，而且Python3相比较Python2会 支持更多的功能。
     <br/>
     下面介绍一下Python3在Windows10中的安装方法。
     <br/>
     1. 首先打开Python下载官网：
     <a href="https://www.python.org/downloads/" rel="nofollow">
      https://www.python.org/downloads/
     </a>
     ，如下图所示，下载最新版本的Python3.6.4，单击下载得到安装包。
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/adb3097ad6167f84c8e6af39e7e9206d.png" title="">
      <br/>
      2. 下载完成后双击python-3.6.4.exe文件，然后进入安装界面。
      <br/>
      <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ee3dfb87fe0e6b385b91966cb1ee6665.png" title="">
       <br/>
       3. 选中“add python3.6 to PATH”，点击“Customize Installation” 。
       <br/>
       <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f2d81deee4864d92c1158a3d19c89bbb.png" title="">
        <br/>
        4. 后面全部默认，全部勾选，一路点击“next”。
        <br/>
        <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7a2cf37a6a9515dc1b8564c5c7d54f81.png" title="">
         <br/>
         5. 如下图所示选项，然后修改安装路径改为 C:\python36，我们直接安装到C盘下面，避免出现其他不必要的问题。然后点击“install”开始安装 。
         <br/>
         <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3613d6b7f9225fa06231e018fb7e7a17.png" title="">
          <br/>
          6. 大概等待喝一杯茶的时间，安装完成。
          <br/>
          <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7f9b94db4f27abc4f7a47b88f3851b14.png" title="">
           <br/>
           7. 在windows的运行里输入cmd，然后输入python，会提示python的版本，说明安装成功。
           <br/>
           <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7794b91049c4936da12145b2632ecd15.png" title="">
            <br/>
            注：python3的安装过程中省去了环境变量的设置，所以安装过程很简单。
