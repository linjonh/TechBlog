    <h2>
     <span style="color:#0d0016;">
      一、认识枚举
     </span>
    </h2>
    <p>
     <span style="color:#0d0016;">
      枚举是在JDK1.5后引入的，主要作用是将一组常量组织起来进行统一管理。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      应用场景：错误状态码、消息类型、颜色划分……
     </span>
    </p>
    <p>

      自定义的枚举类都会默认继承于java.lang.Enum，因此自定义的枚举类就可以使用Enum类中的方法：

    <h2>
     <span style="color:#0d0016;">
      二、枚举的使用
     </span>
    </h2>
    <p>
     <span style="color:#0d0016;">
      我们先自定义一个简单的枚举类：
     </span>
    </p>
    <pre><code class="language-java">public enum TestEnum {
    RED,GREEN,BLUE,YELLOW;
}</code></pre>
    <h3>
     <span style="color:#0d0016;">
      1、switch语句的判定
     </span>
    </h3>
    <p>
     <span style="color:#0d0016;">
      枚举对象可以用于switch语句的判定条件：
     </span>
    </p>
    <pre><code class="language-java">public static void main(String[] args) {
    TestEnum test = TestEnum.RED;
    switch (test){
        case RED:
            System.out.println("red");
            break;
        case GREEN:
            System.out.println("green");
            break;
        case BLUE:
            System.out.println("blue");
            break;
        case YELLOW:
            System.out.println("yellow");
            break;
        default:
            break;
    }
}</code></pre>
    <p>

    <h3>
     <span style="color:#0d0016;">
      2、values()方法和ordinal()方法的使用
     </span>
    </h3>
    <pre><code class="language-java">    public static void main(String[] args) {
        //将自定义枚举类型中的所有对象以数组的形式返回
        TestEnum[] testEnums = TestEnum.values();

        for (int i = 0; i &lt; testEnums.length; i++) {
            //输出每个对象及其对应的序号
            System.out.println(testEnums[i]+"序号："+testEnums[i].ordinal());
        }
    }</code></pre>

    System.out.println(TestEnum.valueOf("GREEN"));

    <h3>
     <span style="color:#0d0016;">
      5、枚举类的构造方法
     </span>
    </h3>
    <p>
     <span style="color:#0d0016;">
      枚举类中的构造方法默认是一个无参的构造方法，我们也可以自定义构造方法：
     </span>
    </p>
    <pre><code class="language-java">public enum TestEnum {
    RED("RED",1),GREEN("GREEN",2),
    BLUE("BLUE",3),YELLOW("YELLOW",4);

    public String color;
    public int ordinal;

    private TestEnum(String color,int ordinal){
        this.color = color;
        this.ordinal = ordinal;
    }
}</code></pre>
    <p>
     <span style="color:#0d0016;">
      枚举类型的构造方法默认是被
     </span>
     <span style="color:#4da8ee;">
      <strong>
       private
      </strong>
     </span>
     <span style="color:#0d0016;">
      修饰的，如果改为其它的访问权限会报错。
     </span>
    </p>

     <img alt="" height="318" src="https://i-blog.csdnimg.cn/blog_migrate/4974b1cc319ec9fb3083051d08e4c718.png" width="705"/>

    <h2>
     <span style="color:#0d0016;">
      三、枚举和反射
     </span>
    </h2>
    <p>
     <span style="color:#0d0016;">
      既然枚举类型的构造方法默认是被
     </span>
     <span style="color:#4da8ee;">
      <strong>
       private
      </strong>
     </span>
     <span style="color:#0d0016;">
      修饰的，那么能否通过反射的方式来实例化一个枚举对象呢？
     </span>
    </p>
    <pre><code class="language-java">public static void main(String[] args) {

    //获取TestEnum的Class对象
    Class&lt;?&gt; c = TestEnum.class;

    try {
        //还需要给父类Enum的构造方法进行传参
        Constructor&lt;?&gt; constructor = 
        c.getDeclaredConstructor(String.class,int.class,String.class,int.class);

        //设置为true后可以修改访问权限
        constructor.setAccessible(true);

        //父类参数不给也可以，默认为空
        TestEnum testEnum = 
        (TestEnum) constructor.newInstance("WHITE",5,"父类的参数",999);

        System.out.println(testEnum);
    } catch (NoSuchMethodException e) {
        e.printStackTrace();
    } catch (InvocationTargetException e) {
        e.printStackTrace();
    } catch (InstantiationException e) {
        e.printStackTrace();
    } catch (IllegalAccessException e) {
        e.printStackTrace();
    }
}</code></pre>
    <p>
     <span style="color:#0d0016;">
      <span style="background-color:#ed7976;">
       注意：
      </span>
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      自定义的枚举类型默认继承于标准库里的Enum类，Enum中只有一个构造方法：
     </span>
    </p>
    <p>
     <img alt="" height="128" src="https://i-blog.csdnimg.cn/blog_migrate/416808be3d4b68c64c0b80c398d09212.png" width="563"/>
    </p>
    <p>
     <span style="color:#0d0016;">
      我们在构造子类对象之前需要先帮助父类构造，所以调用
     </span>
     <em>
      <strong>
       <span style="color:#4da8ee;">
        getDeclaredConstructor
       </span>
      </strong>
     </em>
     <span style="color:#0d0016;">
      方法时需要给4个参数，2个是自定义枚举类TestEnum构造方法的参数类型，2个是父类Enum构造方法的参数类型。
     </span>
    </p>

       IllegalArgumentException
