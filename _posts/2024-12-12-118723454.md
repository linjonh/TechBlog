    <p style="margin-left:.0001pt;">
     <strong>
      java是用来做什么的？
     </strong>
     随着电子产业的迅猛发展，Java技术也得到越来越广泛的应用，Java随之也成为受欢迎的IT方向。那很多想进入Java的人就产生了疑惑，虽然Java乍一听起来似乎跟生活不沾边，但实际上生活中处处都离不开Java语言，今天小编就来给大家介绍一下什么是Java，
     <strong>
      java是用来做什么的？
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1c8daa77e8b7de13cf0400f3dcf856d5.jpeg"/>
    </p>
    <p style="text-align:center;">
    </p>
    <h2 style="margin-left:.0001pt;text-align:justify;">
     <strong>
      一、Java的含义
     </strong>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     Java语言其实就是一种给计算机使用的编程语言，我们需要让计算机帮助我们处理一些问题，但是直接说人类语言计算机无法识别，就只能通过计算机语言也就是Java语言来告诉计算机，这样计算机就能够按照你的要求去一步步的执行任务，这就是Java语言了。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     当然计算机的语言不止一种，常见的C语言、C++、Python等等都是其中的一种，只不过因为Java语言的优点比较多，所以才有这么多的开发者选择这门语言，并且按照目前的趋势来看，在短期内还没有哪门语言能够取代Java语言在编程界的地位，所以如果你想要进入编程行业，那么Java语言绝对是你的首选。
    </p>
    <h2 style="margin-left:.0001pt;text-align:justify;">
     <strong>
      二、Java是一种软件开发平台
     </strong>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     所谓软件开发简单地理解为：编程的结果是软件或者程序，而编程的过程就是软件开发。 那什么是开发平台？ 在软件开发的过程中，我们需要很多的工具来辅助我们的工作。我们把编程的环境和相应的辅助工具统称为开发环境，开发平台就是用来提供这个开发环境的。而Java作为一种开发平台，不单纯是一个编程的语言，它自身提供了一系列开发 Java 所需要的环境和工具，来进行编译、解释、文档生成、打包等，比如：javac.exe、javadoc.exe 等等，所以我们讲 Java 是一个开发平台。
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/60a90be3ec6adcd192574ade0df5fb34.jpeg"/>
    </p>
    <p style="text-align:center;">
    </p>
    <h2 style="margin-left:.0001pt;text-align:justify;">
     <strong>
      三、Java 是一种软件运行平台
     </strong>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     Java 本身提供 Java软件所需要的运行环境，Java应用可运行在安装了 JRE(Java Runtime Environment)的机器上，所以我们说 Java 是一个运行平台。JRE：Java Runtime Environment，Java 运行环境。
    </p>
    <h2 style="margin-left:.0001pt;text-align:justify;">
     <strong>
      四、Java也是一种软件部署环境
     </strong>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     简单地讲，部署就是安装，就是把软件放置到相应的地方，并且进行相应的配置(一般称作部署描述)让软件能够正常运行起来。Java 本身是一个开发的平台，开发后的Java程序也是运行在Java平台上的。也就是说，开发后的Java程序也是部署在Java平台上的，这个尤其在后面学习JEE(Java的企业版)的时候，体现更为明显。
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9cf13e73d3d8b0c621d95d1065e8a52e.jpeg"/>
    </p>
    <p style="text-align:center;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      java
     </strong>
     <strong>
      是用来做什么的？
     </strong>
     以上就是小编对Java的工作进行的一个简单总结，如果您掌握Java技术，则使用范围也非常广泛，无论是计算机软件和硬件开发还是手机软件，Java都是必不可少的技术，只要您想从事IT工作，就必须精通Java。
    </p>
