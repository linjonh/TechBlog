    <p>
     ArrayList 类是一个可以动态修改的数组，与普通数组的区别就是它是没有固定大小的限制，我们可以添加或删除元素。
    </p>
    <p>
     ArrayList 继承了 AbstractList ，并实现了 List 接口。
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/49b1d520e4ced4ecedeb907242a8f949.png"/>
    </p>
    <h3 id="vs-array" style="text-align:justify;">
     Java 数组 与 ArrayList
    </h3>
    <p style="text-align:justify;">
     在Java中，我们需要先声明数组的大小，然后才能使用它。一旦声明了数组的大小，就很难更改它。
    </p>
    <p style="text-align:justify;">
     要解决此问题，我们可以使用ArrayList类。  java.util包中存在的ArrayList类允许我们创建可调整大小的数组。
    </p>
    <p>
     使用前需要引入它，语法格式如下：
    </p>
    <pre><code class="language-java">import java.util.ArrayList; // 引入 ArrayList 类</code></pre>
    <p style="text-align:justify;">
     与数组不同，当我们向数组列表添加或删除元素时，数组列表（ArrayList类的对象）可以自动调整其容量。 因此，数组列表也称为动态数组。
    </p>
    <h3 id="create" style="text-align:justify;">
     创建ArrayList
    </h3>
    <p style="text-align:justify;">
     这是我们可以在Java中创建数组列表的方法：
    </p>
    <pre><code class="language-java">ArrayList&lt;Type&gt; arrayList= new ArrayList&lt;&gt;();</code></pre>
    <p style="text-align:justify;">
     此处，Type指示数组列表的类型。例如，
    </p>
    <pre><code class="language-java">//创建整数类型arraylist
ArrayList&lt;Integer&gt; arrayList = new ArrayList&lt;&gt;();

//创建字符串类型arraylist
ArrayList&lt;String&gt; arrayList = new ArrayList&lt;&gt;();</code></pre>
    <p style="text-align:justify;">
     在上面的程序中，我们使用了Integer和String。 在这里，Integer是int类型的相应包装类。
    </p>
    <p style="text-align:justify;">
     包装类是包装原始数据类型的类。例如，Integer类包装了int类型，Float类包装了Float类型，等等。
    </p>
    <p style="text-align:justify;">
     <strong>
      注意：
     </strong>
     我们不能创建原始数据类型（如int，float，char等）的数组列表。相反，我们必须使用它们对应的包装器类。
    </p>
    <p style="text-align:justify;">
     对于字符串，String是一个类，没有包装类。因此，我们按原样使用String。
    </p>
    <p style="text-align:justify;">
     我们还可以使用List接口创建ArrayList。这是因为ArrayList类实现了List接口。
    </p>
    <pre style="text-align:justify;">List&lt;String&gt; list = new ArrayList&lt;&gt;();</pre>
    <h3 id="ArrayList%E7%9A%84%E6%96%B9%E6%B3%95" style="text-align:justify;">
     ArrayList的方法
    </h3>
    <p style="text-align:justify;">
     ArrayList 提供了允许我们执行数组列表操作的各种方法。
    </p>
    <h3 id="add">
     将元素添加到ArrayList
    </h3>
    <p>
     <strong>
      1.使用 add() 方法
     </strong>
    </p>
    <p>
     要将单个元素添加到数组列表，我们使用add()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args){
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        //添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Cat, Horse]</code></pre>
    <p>
     <strong>
      2.使用索引号
     </strong>
    </p>
    <p>
     我们还可以使用索引将元素添加到数组列表中。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args){
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        //添加元素
        animals.add(0,"Dog");
        animals.add(1,"Cat");
        animals.add(2,"Horse");
        System.out.println("ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Cat, Horse]</code></pre>
    <p>
     <strong>
      3.将一个数组列表的元素添加到另一个数组列表
     </strong>
    </p>
    <p>
     要将数组列表的所有元素添加到新的数组列表，请使用addAll()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args){
        ArrayList&lt;String&gt; mammals = new ArrayList&lt;&gt;();
        mammals.add("Dog");
        mammals.add("Cat");
        mammals.add("Horse");
        System.out.println("Mammals: " + mammals);

        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();
        animals.add("Crocodile");

        // 添加mammals所有元素到animals
        animals.addAll(mammals);
        System.out.println("Animals: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">Mammals: [Dog, Cat, Horse]
Animals: [Crocodile, Dog, Cat, Horse]</code></pre>
    <h3 id="initialize">
     使用asList()初始化ArrayList
    </h3>
    <p>
     与数组不同，我们不能直接初始化数组列表。 但是，我们可以使用Arrays类的asList()方法来达到相同的效果。
    </p>
    <p>
     为了使用asList()方法，我们必须先导入 java.util.Arrays 包。
    </p>
    <p>
     例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        //创建数组列表
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;(Arrays.asList("Cat", "Cow", "Dog"));
        System.out.println("ArrayList: " + animals);

        //访问数组列表中的元素
        String element = animals.get(1);
        System.out.println("访问元素: " + element);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Cat, Cow, Dog]
访问元素: Cow</code></pre>
    <p>
     在上面的示例中，请注意以下表达式：
    </p>
    <pre><code class="language-java">new ArrayList&lt;&gt;(Arrays.asList(("Cat", "Cow", "Dog"));</code></pre>
    <p>
     在这里，我们首先在3个元素的数组："Cat"，"Cow"，和"Dog"。然后，该asList()方法用于将数组转换为数组列表。
    </p>
    <h3 id="access">
     访问ArrayList的元素
    </h3>
    <p>
     <strong>
      1.使用get()方法
     </strong>
    </p>
    <p>
     要随机访问数组列表的元素，我们使用get()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals= new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Horse");
        animals.add("Cat");
        System.out.println("ArrayList: " + animals);

        //从数组列表中获取元素
        String str = animals.get(0);
        System.out.print("索引0处的元素: " + str);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Horse, Cat]
索引0处的元素: Dog</code></pre>
    <p>
     <strong>
      2.使用 iterator() 方法
     </strong>
    </p>
    <p>
     要按顺序访问数组列表的元素，我们使用iterator()方法。我们必须导入 java.util.Iterator 包才能使用此方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.Iterator;

class Main {
    public static void main(String[] args){
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        animals.add("Zebra");

        //创建一个Iterator对象
        Iterator&lt;String&gt; iterate = animals.iterator();
        System.out.print("ArrayList: ");

        //使用Iterator的方法访问元素
        while(iterate.hasNext()){
            System.out.print(iterate.next());
            System.out.print(", ");
        }
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: Dog, Cat, Horse, Zebra,</code></pre>
    <p>
     <strong>
      注意事项
     </strong>
     ：
    </p>
    <blockquote>
     <ul>
      <li>
       <p>
        hasNext() - 如果数组列表中有下一个元素，则返回true。
       </p>
      </li>
      <li>
       <p>
        next()  -  返回数组列表中的下一个元素
       </p>
      </li>
     </ul>
    </blockquote>
    <h3 id="change">
     更改ArrayList元素
    </h3>
    <p>
     要更改数组列表的元素，我们可以使用set()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals= new ArrayList&lt;&gt;();
        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("ArrayList: " + animals);

        //更改数组列表的元素
        animals.set(2, "Zebra");
        System.out.println("修改后的ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Cat, Horse]
修改后的ArrayList: [Dog, Cat, Zebra]</code></pre>
    <h3 id="remove">
     删除ArrayList元素
    </h3>
    <p>
     <strong>
      1.使用 remove() 方法
     </strong>
    </p>
    <p>
     要从数组列表中删除元素，我们可以使用remove()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("初始ArrayList: " + animals);

        //从索引2中删除元素
        String str = animals.remove(2);
        System.out.println("最终ArrayList: " + animals);
        System. out.println("删除元素: " + str);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">初始ArrayList: [Dog, Cat, Horse]
最终ArrayList: [Dog, Cat]
删除元素: Horse</code></pre>
    <p>
     <strong>
      2.使用 removeAll() 方法
     </strong>
    </p>
    <p>
     要从数组列表中删除所有元素，我们使用removeAll()方法。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        // 在ArrayList中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("初始ArrayList: " + animals);

        // 删除所有元素
        animals.removeAll(animals);
        System.out.println("最终ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">初始ArrayList: [Dog, Cat, Horse]
最终ArrayList: []</code></pre>
    <p>
     <strong>
      3.使用 clear() 方法
     </strong>
    </p>
    <p>
     我们还可以使用clear()方法从数组列表中删除所有元素。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals= new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("Initial ArrayList: " + animals);

        //删除所有元素
        animals.clear();
        System.out.println("Final ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">初始ArrayList: [Dog, Cat, Horse]
最终ArrayList: []</code></pre>
    <p>
     <strong>
      注意：
     </strong>
     clear()方法比removeAll()方法更高效。
    </p>
    <h3 id="loop">
     遍历 ArrayList
    </h3>
    <p>
     <strong>
      1.使用 for 循环
     </strong>
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {

        for(int i = 0; i &lt; animals.size(); i++) {
            System.out.print(animals.get(i));

        }
    }

    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Cow, Cat, Dog]
访问所有元素:
Cow, Cat, Dog,</code></pre>
    <p>
     <strong>
      2.使用 forEach 循环
     </strong>

    public static void main(String[] args) {

        }

    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Cow, Cat, Dog]
访问所有元素:
Cow, Cat, Dog,</code></pre>

    <h3 id="%E5%AF%B9ArrayList%E7%9A%84%E5%85%83%E7%B4%A0%E8%BF%9B%E8%A1%8C%E6%8E%92%E5%BA%8F">
     对ArrayList的元素进行排序
    </h3>
    <p>
     要对数组列表的元素进行排序，我们使用Collections类的sort()方法。为了使用它，我们必须先导入java.util.Collections包。
    </p>
    <p>
     默认情况下，排序以字母或数字升序进行。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.Collections;

class Main {
    public static void main(String[] args){
        ArrayList&lt;String&gt; animals= new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Horse");
        animals.add("Zebra");
        animals.add("Dog");
        animals.add("Cat");

        System.out.println("未排序的ArrayList: " + animals);

        //对数组列表进行排序
        Collections.sort(animals);
        System.out.println("排序后的ArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">未排序的ArrayList: [Horse, Zebra, Dog, Cat]
排序后的ArrayList: [Cat, Dog, Horse, Zebra]</code></pre>
    <p>
     要了解有关对数组列表进行排序的更多信息，请访问Java ArrayList sort。
    </p>
    <h3 id="to-array">
     Java ArrayList转换为数组
    </h3>
    <p>
     在Java中，我们可以使用toArray()方法将数组列表转换为数组。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals= new ArrayList&lt;&gt;();

        //在数组列表中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("ArrayList: " + animals);

        //创建一个新的String类型数组
        String[] arr = new String[animals.size()];

        //将ArrayList转换成数组
        animals.toArray(arr);
        System.out.print("Array: ");
        for(String item:arr) {
            System.out.print(item+", ");
        }
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Cat, Horse]
Array: Dog, Cat, Horse,</code></pre>
    <h3 id="from-array">
     Java Array转换为ArrayList
    </h3>
    <p>
     我们还可以将数组转换为数组列表。为此，我们可以使用Arrays类的asList()方法。
    </p>
    <p>
     要使用asList()，我们必须先导入java.util.Arrays包。例如，
    </p>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        //创建一个字符串类型的数组
        String[] arr = {"Dog", "Cat", "Horse"};
        System.out.print("Array: ");

        //打印数组
        for(String str: arr) {
            System.out.print(str);
            System.out.print(" ");
        }

        //从数组创建ArrayList
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;(Arrays.asList(arr));
        System.out.println("\nArrayList: " + animals);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">Array: Dog, Cat, Horse
ArrayList: [Dog, Cat, Horse]</code></pre>
    <p>
     在上面的程序中，我们首先创建了String类型的数组arr。
    </p>
    <p>
     然后，我们使用asList()方法将数组转换为数组列表。
    </p>
    <h3 id="to-string">
     Java ArrayList转换为字符串
    </h3>
    <p>
     要将数组列表转换为字符串，可以使用toString()方法。例如
    </p>
    <pre><code class="language-java">import java.util.ArrayList;

class Main {
    public static void main(String[] args) {
        ArrayList&lt;String&gt; animals = new ArrayList&lt;&gt;();

        //在ArrayList中添加元素
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Horse");
        System.out.println("ArrayList: " + animals);

        //将ArrayList转换为字符串
        String str = animals.toString();
        System.out.println("String: " + str);
    }
}</code></pre>
    <p>
     <strong>
      输出结果
     </strong>
    </p>
    <pre><code class="language-java">ArrayList: [Dog, Cat, Horse]
String: [Dog, Cat, Horse]</code></pre>
    <p>
     <strong>
      注意：
     </strong>
     toString()将整个数组列表转换为单个String。
    </p>
    <h3 id="%E5%85%B6%E4%BB%96%E7%9A%84%E5%BC%95%E7%94%A8%E7%B1%BB%E5%9E%8B">
     其他的引用类型
    </h3>
    <p>
     ArrayList 中的元素实际上是对象，在以上实例中，数组列表元素都是字符串 String 类型。
    </p>
    <p>
