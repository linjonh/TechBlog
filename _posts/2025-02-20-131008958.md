    <h3>
     <a id="6_66">
     </a>
     6.注意点
    </h3>
    <ol>
     <li>
      方法要在 main 函数上面定义,脚本语言和 js 一样的,按顺序执行
     </li>
     <li>
      tab 和空格的混用,缩进比较敏感
     </li>
     <li>
      python 和 go 一样,不需要括号,简洁
     </li>
     <li>
      不需要尾部的分号
     </li>
     <li>
      java 中的大括号在 python 中是冒号:,在 go 中是大括号不变{}
     </li>
     <li>
      python 中的注释是用#,不是//,注意区别
     </li>
     <li>
      bool 类型为 True 和 False
     </li>
     <li>
      python 中没有 a++操作但是可以 a+=1
     </li>
    </ol>
