     Java 计算字符串表达式(字符串代码)
    </h2>
    <p>
     Java 执行字符串代码的方案有很多中，一般情况下我们计算字符串表达式的场景有：
    </p>
    <ul>
     <li>
      计算逻辑判断式，并返回判断结果(true,false)
     </li>
     <li>
      计算表达式值,一般返回结果为数值
     </li>
     <li>
      根据条件简单拼接字符串,返回结果为拼接文本
     </li>
    </ul>
    <p>
     我们可以使用
     <code>
      Java
     </code>
     内部自带的
     <code>
      JavaScript
     </code>
     引擎实现上述效果。
    </p>
    <pre><code class="prism language-java"><span class="token keyword">import</span> javax<span class="token punctuation">.</span>script<span class="token punctuation">.</span>ScriptEngine<span class="token punctuation">;</span>

<span class="token keyword">class</span> <span class="token class-name">Scratch</span> <span class="token punctuation">{<!-- --></span>

        ScriptEngineManager manager <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">ScriptEngineManager</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <h4>
     <a id="_Hutool_27">
     </a>
     工具: Hutool
    </h4>
    <p>
     当然，在通常情况下，我们不希望管
     <code>
      ScriptEngineManager
     </code>
     或者
     <code>
      ScriptEngine
     </code>
     什么鬼，只是希望能够计算一个字符串表达式的结果，这里强烈推荐在项目中引入工具包：
     <a href="https://hutool.cn/" rel="nofollow">
      Hutool
     </a>
     。通过这个工具包，我们可以不用管上述两个东西是什么，只需要关注我们的字符串表达式，并且确保表达式的语法符合 JavaScript 语法即可,Hutool
     <a href="https://hutool.cn/docs/#/script/Script%E5%B7%A5%E5%85%B7-ScriptUtil" rel="nofollow">
      文档
     </a>
     。
    </p>
    <p>
     Maven 引入工具包坐标：
    </p>
    <pre><code class="prism language-xml"><span class="token comment">&lt;!-- https://mvnrepository.com/artifact/cn.hutool/hutool-all --&gt;</span>
