     1. Doxygen, what？
    </h3>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     Doxgen就是大名鼎鼎的文档生成工具，而且是免费开源的，它使用非常方便，能提取C++，
     <a class="replace_word" href="http://lib.csdn.net/base/17" rel="nofollow noopener noreferrer" style="color:rgb(223,52,52); text-decoration:none; font-weight:bold" target="_blank" title="Java EE知识库">
      Java
     </a>
     ，Objective-C，
     <a class="replace_word" href="http://lib.csdn.net/base/11" rel="nofollow noopener noreferrer" style="color:rgb(223,52,52); text-decoration:none; font-weight:bold" target="_blank" title="Python知识库">
      Python
     </a>
     ，IDL，PHP，C#等语言的注释，从而生成文档。
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     你可以访问其官方网站，下载安装包，它的官网上有详细的使用手册。
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">

     3. 配置Doxygen
    </h3>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     配置doxgen是最核心的，你可以设置你要提取注释的源文件，生成的文档格式，工程名称，文档的Logo等信息，这些配置是可以存储起来的，当你的源代码更新后，重新再运行这个配置文件，就可以重新生成一个新的文档。
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     在安装后，进入到其安装目录下的
     <strong>
      bin
     </strong>
     文件夹，它里面有两个文件：doxygen.exe和doxywizard.exe，我们先运行doxywizard.exe来进行配置，从而生成配置文件（如果是第一次运行）。
