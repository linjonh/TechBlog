    <h3>
     <a id="11__1">
     </a>
     1.1. 基本概念
    </h3>
    <p>
     HarmonyOS应用通过数字证书（.cer文件）和Profile文件（.p7b文件）来保证应用的完整性，数字证书和Profile文件可通过申请发布证书和Profile文件获取。
     <br/>
     申请数字证书和Profile文件前，首先需要通过DevEco Studio来生成密钥文件（.p12文件）和证书请求文件（.csr文件）。同时，也可以使用命令行工具的方式来生成密钥文件和证书请求文件。
     <br/>
     •
     <strong>
      密钥
     </strong>
     ：包含非对称加密中使用的公钥和私钥，存储在密钥库文件中，格式为.p12，公钥和私钥对用于数字签名和验证。
     <br/>
     •
     <strong>
      证书请求文件
     </strong>
     ：格式为.csr，全称为Certificate Signing Request，包含密钥对中的公钥和公共名称、组织名称、组织单位等信息，用于向AppGallery Connect申请数字证书。
     <br/>
     •
     <strong>
      数字证书
     </strong>
     ：格式为.cer，由华为AppGallery Connect颁发。
     <br/>
     •
     <strong>
      Profile文件
     </strong>
     ：格式为.p7b，包含HarmonyOS应用的包名、数字证书信息、描述应用允许申请的证书权限列表，以及允许应用调试的设备列表（如果应用类型为Release类型，则设备列表为空）等内容，每个应用包中均必须包含一个Profile文件。
    </p>

     1.2.2.1. 应用发布证书
    </h4>
    <p>
     一个账号最多可申请1个发布证书，2个调试证书。
     <br/>
     （1）登录
     <a href="https://developer.huawei.com/consumer/cn/service/josp/agc/index.html#/" rel="nofollow">
      AppGallery Connect网站
     </a>
     ，在网站首页中点击“用户与访问”。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ae5cc199c6ab88a59150aab79080a14f.png"/>
     <br/>
     （2）在左侧导航栏点击“证书管理”，进入“证书管理”页面。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b6dc165dd8cd8e4dbfb777a1d62f38a0.png"/>
     <br/>
     （3）点击“新增证书”，弹出“新增证书”窗口。
     <br/>
     （4）在“新增证书”窗口中填写要申请的证书信息，点击“提交”。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fc527a94ed64b8bcfab4aeac747f27f9.png"/>
     <br/>
     证书申请成功后，“证书管理”页面展示“证书名称”、“证书类型”,“失效日期”和“操作”列信息。
     <br/>
     在“操作”列点击“下载”，可下载证书。
     <br/>
     在“操作”列点击“废除”，且在弹出的“提示”窗口中点击“确认”，可废除证书。
     <br/>
     说明：如证书已过期，“失效日期”列展示“已于YYYY-MM-DD过期”。您可以下载或废除过期证书。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4f28c7ffaa3f421d6bec78cd73716215.png"/>

     1.2.3. Profile文件
    </h4>
    <p>
     一个账号最多可申请100个Profile文件。
     <br/>
     （1）登录AppGallery Connect网站，在网站首页中点击“我的项目”。
     <br/>
     （2）找到您的项目，点击您创建的HarmonyOS应用。
     <br/>
     （3）在左侧导航栏选择“HarmonyOS应用 &gt; HAP Provision Profile管理”，进入“管理HAP Provision Profile”页面。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ca7c3cebc0bc24b190ffb40ca909205e.png"/>
     <br/>
     （4）点击右上角“添加”，弹出“HarmonyAppProvision信息”窗口。
     <br/>
     （5）在弹出的“HarmonyAppProvision信息”窗口中添加Profile。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eeebaceaa6ff68c92341ff0e7f5dc673.png"/>
     <br/>
     （6）发布Profile申请成功后，“管理HAP Provision Profile”页面展示“名称”、“类型”、“证书”、“失效日期”和“操作”列信息。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d0ec491f146cfd0b5ab315a3fd560010.png"/>
     <br/>
     （7）在“操作”列点击“下载”，可下载Profile文件。
     <br/>
     （8）在“操作”列点击“删除”，且在弹出的“提示”窗口中点击“确认”，可删除Profile文件。
    </p>
