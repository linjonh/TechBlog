   <div class="htmledit_views" id="content_views">
    <p align="justify">
     <span style="font-family:宋体">
      安装
     </span>
     TestWriter
     <span style="font-family:宋体">
      工具时，突然弹出如下提示框，怎么办？
     </span>
    </p>
    <p align="center">
     <img alt="" src="https://img-blog.csdn.net/20170317095413642?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzAyNjEzMDE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
     </img>
    </p>
    <p align="center">
     <span style="font-family:宋体">
      错误：
     </span>
     <span style="font-family:宋体">
      连接失败。填写的数据库信息不正确或未开启数据库服务，请检查！
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="font-family:宋体">
      遇到这样的情况，首先确认数据库是否正确连接，有两个注意事项：
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="font-family:宋体">
      一、
     </span>
     SQL
     <span style="font-family:宋体">
      主服务保证开启
     </span>
    </p>
    <p>
     <span style="font-family:宋体">
      打开
     </span>
     SQL Server
     <span style="font-family:宋体">
      配置管理器，确认
     </span>
     <a href="http://www.51testing.com/html/91/javascript:;" rel="nofollow noopener noreferrer" target="_blank">
      SQL
     </a>
     Sever 2008
     <span style="font-family:宋体">
      服务开启以及
     </span>
     TCP/IP
     <span style="font-family:宋体">
      协议开启；
     </span>
    </p>
    <p>
     <span style="font-family:宋体">
      打开
     </span>
     SQL Server Management Studio
     <span style="font-family:宋体">
      ，确认数据库是否可以直接连接。
     </span>
    </p>
    <p>
     <span style="font-family:宋体">
      可以参考：
     </span>
     <a href="http://bbs.51testing.com/thread-1090124-1-12.html" rel="nofollow noopener noreferrer" target="_blank">
      <u>
       <span style="color:#00ff">
        http://bbs.51testing.com/thread-1090124-1-12.html
       </span>
      </u>
     </a>
    </p>
    <p>
    </p>
    <p>
     <span style="font-family:宋体">
      二、服务名写对了
     </span>
    </p>
    <p>
     1
     <span style="font-family:宋体">
      、如果安装数据库时实例配置选择的是
     </span>
     <span style="font-family:Calibri">
      MMSQLServer
     </span>
    </p>
    <p>
     <span style="font-family:宋体">
      如果你安装的时候没有改实例名，那么你的服务名就是
     </span>
     localhost
     <span style="font-family:宋体">
      、
     </span>
     <a href="https://www.baidu.com/s?wd=127.0.0.1&amp;tn=44039180_cpr&amp;fenlei=mv6quAkxTZn0IZRqIHckPjm4nH00T1Y3nHc4nAN-n1T1nWK-PjDv0ZwV5Hcvrjm3rH6sPfKWUMw85HfYnjn4nH6sgvPsT6KdThsqpZwYTjCEQLGCpyw9Uz4Bmy-bIi4WUvYETgN-TLwGUv3EnHDLn1fYnHb3rjTYnWRsnWTLrf" rel="nofollow noopener noreferrer" target="_blank">
      127.0.0.1
     </a>
     <span style="font-family:宋体">
      、计算机名或直接写个
     </span>
     .(
     <span style="font-family:宋体">
      英文状态下
     </span>
     <span style="font-family:Calibri">
      )
     </span>
     <span style="font-family:宋体">
      都可以。
     </span>
     <br/>
     <span style="font-family:宋体">
      如果改了实例名，那么就是：计算机名
     </span>
     \
     <span style="font-family:宋体">
      服务名。
     </span>
    </p>
    <p>
    </p>
    <p>
     2
     <span style="font-family:宋体">
      、如果安装数据库时实例配置选择的是
     </span>
     SQLExpress
    </p>
    <p>
     a.
     <span style="font-family:宋体">
      默认是没有开启
     </span>
     TCP/IP
     <span style="font-family:宋体">
      服务的（不支持远程访问），数据库服务器地址请填写
     </span>
     .
     <span style="font-family:宋体">
      而不要用
     </span>
     IP
     <span style="font-family:宋体">
      地址（
     </span>
     127.0.0.1
     <span style="font-family:宋体">
      ）或
     </span>
     localhost
     <span style="font-family:宋体">
      ；
     </span>
     <br/>
     b.TestWriter
     <span style="font-family:宋体">
      配置界面上的账户密码是否是你安装数据库配置的帐号密码
     </span>
    </p>
    <p>
    </p>
