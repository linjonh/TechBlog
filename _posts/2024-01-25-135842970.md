    <blockquote>
     <h2>
      <span style="color:#0d0016;">
       <strong>
        服务器
       </strong>
      </span>
     </h2>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      服务器定义
      <br/>
      <strong>
       广义:
      </strong>
      专门给其他机器提供服务的计算机。
      <br/>
      <strong>
       狭义:
      </strong>
      一台高性能的计算机，通过网络提供外部计算机一些业务服务
      <br/>
      个人PC内存大概8G，服务器内存128G起步
     </span>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        服务器是什么
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      服务器指的是
      <strong>
       网络中能对其他机器提供某些服务的计算机系统
      </strong>
      ，相对普通PC，服务器指的是高性能计算机稳定性、安全性要求更高。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      服务器的
      <strong>
       高性能
      </strong>
      体现在
      <strong>
       高速的运转能力，长时间的可靠运行，强大的数据吞吐能力。
      </strong>
     </span>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        服务器特点
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      <strong>
       可用性 可靠性、持续性。
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      服务器一般需要724H不间断工作(如Web网站)。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      可利用性 服务器要为多用户提供服务，因此需要较高的连接和运算性能(性能要高)。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      可扩展性服务器需要具备可扩展空间和冗余件(可以增加内存、硬盘等)。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      可管理性保证在设备不停机的情况下修复服务器故障(一般服务器有两个电源，坏了一个还有一个)
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      服务器还需要具备如双机备份、系统备份、在线诊断、故障预警等功能。
     </span>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        台式机可以当服务器用吗
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      仅仅从能够在网络中提供服务的角度来看，是可以的，但是个人台式机的性能、稳定性、安全性都
      <strong>
       达不到服务器的标准
      </strong>
      ，因此更适合个人工作使用。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      且个人笔记本、台式机一般都安装
      <strong>
       windows系统
      </strong>
      、易用，美观，处理日常工作
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      而服务器一般安装
      <strong>
       linux系统
      </strong>
      ，提供更强大、稳定的计算服务。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      <strong>
       硬件实体服务器可以分为四大类:塔式服务器、刀片服务器、机式服务器、柜式服务器。
      </strong>
     </span>
    </p>
    <h2>
     <span style="color:#0d0016;">
      <strong>
       四种服务器类型
      </strong>
     </span>
    </h2>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        刀片式服务器
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      就像家里摆放菜刀组合的架子一样，每一块刀片，就是每一块系统主板，也是每一个独立的系统，每一个独立的服务器，
      <strong>
       明显很节省空间，属于低成本服务器平台。
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      <img alt="" height="332" src="https://i-blog.csdnimg.cn/blog_migrate/6dc7119e2026a8bfc21374fa7b1103fb.png" width="500"/>
     </span>
    </p>
    <p>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        柜式服务器
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      一体化柜式机房无需太多空间、资金、人员，只需要提供一路或两路市电电源，就可以快速地构建可靠运行的机房。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      主要组成是：
     </span>
    </p>
    <ol>
     <li>
      <strong>
       机柜模块
      </strong>
      ：这是柜式服务器的基础结构，通常采用19英寸标准尺寸的服务器机柜，用于容纳服务器硬件和其他设备。机柜的设计通常考虑到空间效率、散热能力和易于维护等因素。
     </li>
     <li>
      <strong>
       配电模块
      </strong>
      ：配电模块负责将输入的电力分配给服务器内的各个组件，确保每个部件都能得到稳定的电力供应。这个模块还包括电源管理和监控，以保证电力使用的安全性和效率。
     </li>
     <li>
      <strong>
       监控模块
      </strong>
      ：监控模块用于实时监测服务器的运行状态，包括硬件健康状况、温度、湿度等环境参数。这些信息对于及时发现并解决问题至关重要，有助于保持服务器的稳定运行。
     </li>
     <li>
      <strong>
       制冷模块
      </strong>
      ：由于服务器在运行时会产生大量热量，制冷模块的作用就是通过风扇、空调等设备来散发热量，维持服务器内部的温度在安全范围内，防止过热导致性能下降或损坏硬件。
     </li>
     <li>
      <strong>
       UPS不间断电源
      </strong>
      ：UPS（Uninterruptible Power Supply）不间断电源是为了防止突然断电对服务器造成影响。它通过蓄电池提供临时电力，保障服务器在电源中断时能够继续运行，直到电力恢复或者安全地关闭系统。
     </li>
     <li>
      <strong>
       消防模块
      </strong>
      ：消防模块是数据中心安全的重要组成部分，包括烟雾探测器、自动灭火系统等，用于在发生火灾时及时响应，保护服务器免受火灾损害。
     </li>
    </ol>
    <p>
     <span style="color:#0d0016;">
      一体化柜式机房建设时长短，各子模块之间不易发生故障，极大地方便了管理与维护，这就意味着系统的兼容性很高，机房的整体利用率很充足，用“智能灵活，高性价比”来形容再合适不过了。
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      <img alt="" height="500" src="https://i-blog.csdnimg.cn/blog_migrate/c8b06a39383e35c741fb44668f4c8046.png" width="666"/>
     </span>
    </p>
    <p>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        机架式服务器
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      这个就是我们常说的那种1U,2U等这样的服务器，是按照统一标准外观设计的服务器，可以理解为是那种放倒的PC主机，能够一台一台的放倒服务器机架上。机架式服务器的优点是在于管理方面更加灵活与方便，但是这个需要部界在特定的机房环境下
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      <img alt="" height="362" src="https://i-blog.csdnimg.cn/blog_migrate/9f4b7c03dc523c1255dedf9989db2329.png" width="351">
       <img alt="" height="355" src="https://i-blog.csdnimg.cn/blog_migrate/888beb00f6e537c30c165cfd7fae1b46.png" width="349"/>
      </img>
     </span>
    </p>
    <p>
    </p>
    <blockquote>
     <h3>
      <span style="color:#0d0016;">
       <strong>
        塔式服务器
       </strong>
      </span>
     </h3>
    </blockquote>
    <p>
     <span style="color:#0d0016;">
      塔式服务器 (Tower Server)应该是见得最多也最容易理解的一种服务器结构类型，因为它的外形以及结构都跟立式PC差不多
      <br/>
      当然，由于服务器的主板扩展性较强、插槽也多出一堆，所以个头比普通主板大一些，因此塔式服务器的主机机箱也比普通机箱要大，一般都会预留足够的内部空间以便日后进行硬盘和电源的冗余扩展。
     </span>
    </p>
