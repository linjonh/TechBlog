    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      <span style="font-size: medium;">
       今年元月
      </span>
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: large;">
       <span style="font-size: medium;">
        22
       </span>
      </span>
     </span>
     <span style="font-size: medium;">
      <span style="font-size: medium;">
       日，微软首次宣布裁员（规模不大不小），像是要死人似的，快过年了，让人笑不起来。这到底是什么原因？
      </span>
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      元月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       22
      </span>
     </span>
     <span style="font-size: medium;">
      日当天，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ballmer
      </span>
     </span>
     <span style="font-size: medium;">
      发给全体微软员工的一份电邮，开头第一句话就是，“为了应对日渐恶化的经济现实，我们正准备采取步骤，重整微软的业务。”
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ballmer
      </span>
     </span>
     <span style="font-size: medium;">
      还说，今年第二财季仅比去年同期增长了
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       2
      </span>
     </span>
     <span style="font-size: medium;">
      个百分点，比预期低了
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       9
      </span>
     </span>
     <span style="font-size: medium;">
      亿美元。但是，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ballmer
      </span>
     </span>
     <span style="font-size: medium;">
      话锋一转，他说：在这两代人期间经济最为衰退的（历史）时期里面，我们取得这样的业绩反映了我们强大的业务基础，也是你们努力工作的实际证明。全球性的经济衰退影响到
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       PC
      </span>
     </span>
     <span style="font-size: medium;">
      的销售和
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IT
      </span>
     </span>
     <span style="font-size: medium;">
      的消费，我们不能幸免这种实际影响。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ballmer
      </span>
     </span>
     <span style="font-size: medium;">
      说，面对全球经济恶化的局势，我们迅速采取行动，降低了业务操作成本
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       6
      </span>
     </span>
     <span style="font-size: medium;">
      亿美元（指该财季），但是，这还不够。他说，考虑到现在和未来的实际收入情况，我们还要进一步提高工作效率，为此，在未来
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       18
      </span>
     </span>
     <span style="font-size: medium;">
      个月里面，我们还要减少（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       eliminate
      </span>
     </span>
     <span style="font-size: medium;">
      ，有淘汰的意思）
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       5,000
      </span>
     </span>
     <span style="font-size: medium;">
      个工作岗位，今天就要裁减
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       1,400
      </span>
     </span>
     <span style="font-size: medium;">
      个（工位）。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ballmer
      </span>
     </span>
     <span style="font-size: medium;">
      还说，对这次裁员影响到员工，微软将提供帮助，成立一个专门的服务中心，帮助他们找到新的工作，对最终需要离开微软的员工，要发放解雇费和其他的有关福利。由此可见，微软对自己的员工还是（很）负责任的，不像我们国内某些老板，对被裁员工“怒目相视”，扫地出门了事。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      全球经济衰退是微软裁员的唯一原因吗？显然不是。微软衰弱的（第一）直接原因是
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      闹的。近年来，尤其是“迷你本”的兴起，使得
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      和
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows
      </span>
     </span>
     <span style="font-size: medium;">
      面对面（“头碰头”）相撞。微软
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Vista
      </span>
     </span>
     <span style="font-size: medium;">
      不顶事，就拿
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       XP
      </span>
     </span>
     <span style="font-size: medium;">
      硬顶着，但是，不赚钱（简直是白送
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       OEM
      </span>
     </span>
     <span style="font-size: medium;">
      厂商）。近日，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ubuntu
      </span>
     </span>
     <span style="font-size: medium;">
      的领头人
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mark Shuttleworth
      </span>
     </span>
     <span style="font-size: medium;">
      出面表示，他希望微软尽快推出
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows 7
      </span>
     </span>
     <span style="font-size: medium;">
      ，在“迷你本”市场上与
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      展开“面对面”的竞争，及早结束这种尴尬（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       awkward
      </span>
     </span>
     <span style="font-size: medium;">
      ）局面（见：
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ubuntu's Shuttleworth praises Windows 7
      </span>
     </span>
     <span style="font-size: medium;">
      ，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       welcomes fight
      </span>
     </span>
     <span style="font-size: medium;">
      ，副标题是
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       End of an 'awkward situation'
      </span>
     </span>
     <span style="font-size: medium;">
      ）。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      面对未来，微软的命运如何？
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      不怕经济衰退，“赤脚不怕穿鞋的”。在将来，等到经济好转的那一天，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      必将更为壮大（比如，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ubuntu 9.10
      </span>
     </span>
     <span style="font-size: medium;">
      ），那时，微软的对手将更加不好惹。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       1
      </span>
     </span>
     <span style="font-size: medium;">
      月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       22
      </span>
     </span>
     <span style="font-size: medium;">
      日，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Gregory T. Huang
      </span>
     </span>
     <span style="font-size: medium;">
      发表文章，质疑微软这次裁员的规模是否足够？是否会进一步裁员？微软能否留得住（或吸引）技术天才？毫无疑问，这些问题都是非常尖锐的。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       1
      </span>
     </span>
     <span style="font-size: medium;">
      月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       22
      </span>
     </span>
     <span style="font-size: medium;">
      日，《
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      基金会》主席
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Jim Zemlin
      </span>
     </span>
     <span style="font-size: medium;">
      在回答
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       LinuxDevices
      </span>
     </span>
     <span style="font-size: medium;">
      记者时说，微软总是设想硬件成本不断下降，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       PC
      </span>
     </span>
     <span style="font-size: medium;">
      威力越来越大，需要更多的内存和更强大的
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       CPU
      </span>
     </span>
     <span style="font-size: medium;">
      处理能力，但是，客观现实却不是这样（比如，“迷你本”和
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       MID
      </span>
     </span>
     <span style="font-size: medium;">
      的兴起）。微软能够清醒过来吗？谁能够知道呢？也许，微软加快
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows 7
      </span>
     </span>
     <span style="font-size: medium;">
      的开发就是微软的“大彻大悟”了。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ubuntu 9.10
      </span>
     </span>
     <span style="font-size: medium;">
      在等着你（微软）呢！（说明：
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       EXT4
      </span>
     </span>
     <span style="font-size: medium;">
      是
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Ubuntu 9.10
      </span>
     </span>
     <span style="font-size: medium;">
      版本缺省的安全文件系统，微软有吗？）我真替微软担心。
