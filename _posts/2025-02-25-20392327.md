    <div style="word-wrap:break-word; font-size:13px">
     <h4 style="background-color:rgb(47,88,156); color:rgb(255,255,255); font-size:17px">
      <span style="font-family:'Microsoft Yahei'">
       Weinre 的安装和运行
      </span>
     </h4>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       1、安装weinre
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       cmd执行以下命令
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="color:rgb(255,0,0); font-family:'Microsoft Yahei'">
       npm -g install weinre
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       <br/>
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       2、运行weinre调试服务器
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       进入目录C:\Users\xxx\node_modules\.bin，这个node_modules可能不同电脑的路径不一样
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       建立一个空文本，输入一下内容，重命名为"运行weinre.bat"
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="color:rgb(255,0,0); font-family:'Microsoft Yahei'">
       weinre.cmd --boundHost -all-
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       注意：只运行weinre.cmd的话，默认启动的是localhost:8080，
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       boundHost参数可以指定调试服务器IP，-all-的意思是绑定到所有可用的ip，如localhost和内网IP，这样其他局域网内的机器就可以访问你的调试服务器了
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       httpPort参数可以指定端口
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="color:rgb(255,0,0); font-family:'Microsoft Yahei'">
       weinre.cmd --boundHost -all- --httpPort 8081
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       <br/>
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       3、webapp的启动页面加上下面的js引用
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       &lt;script type="text/javascript"&gt;http://localhost:8081/target/target-script-min.js#
       <span style="color:rgb(255,0,0)">
        {app标识}
       </span>
       &lt;/script&gt;
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       其中{app标识}是webapp的唯一标识，任意字符串
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       <br/>
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       4、手机的网络必须和安装调试服务器的电脑在一个局域网，或者调试服务器是公开的服务器，手机可以访问到
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       <br/>
      </span>
     </div>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <h4 style="background-color:rgb(47,88,156); color:rgb(255,255,255); font-size:17px">
      <span style="font-family:'Microsoft Yahei'">
       Weinre 的使用
      </span>
     </h4>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       成功启动 Weinre 后就可以使用绑定的 IP 或者域名加上端口访问 Weinre 服务器了（我们这里以 http://localhost:8081 为例）。在 Webkit 核心的浏览器（例如 Chrome、Safari 等）中打开 Weinre 服务器主页：
      </span>
     </div>
     <div style="word-wrap:break-word">
     </div>
     <div style="word-wrap:break-word">
      <img alt="" src="https://img-blog.csdn.net/20140303192730937?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbG92ZWx5ZWxmcG9w/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
      </img>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       在服务器主页有两项内容很重要：
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       A 链接到调试客户端页面，打开后默认到远程面板，如下图所示。
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       B 目标代码，这段代码要加入到需要调试的页面中，也可以用书签的方式动态加入。
      </span>
     </div>
     <div style="word-wrap:break-word">
      <span style="font-family:'Microsoft Yahei'">
       <br/>
      </span>
     </div>
     <div style="word-wrap:break-word">
     </div>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <img alt="" src="https://img-blog.csdn.net/20140303192740265?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbG92ZWx5ZWxmcG9w/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      <br/>
     </img>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      远程面板总共有四部分：
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      A 面板切换，用过 Chrome 或者 Safari 开发者工具的对这个界面肯定很熟悉。
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      B 连接到调试服务器的页面，即可以调试的页面。
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      C 连接到调试服务地的客户端，当前只有一个。
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      D 调试服务器属性，绑定的端口和调试服务器能够响应式的 IP 地址列表。
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      <br/>
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      远程面板以外，还有元素面板、资源面板、网络面板、时间线面板和控制台，后面这几个 Web 开发人员都很熟悉的，就不一一介绍了。
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">

     <h4 style="background-color:rgb(47,88,156); color:rgb(255,255,255); font-size:17px">
      <span style="font-family:'Microsoft Yahei'">
       Weinre 支持的平台
      </span>
     </h4>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      支持的调试客户端（运行调试界面的浏览器）：
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      Google Chrome
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      Apple Safari
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      其它基于 WebKit 的浏览器
     </span>
    </div>

     <span style="font-family:'Microsoft Yahei'">
      <br/>
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      支持的调试目标（需要调试的网站或应用的界面）：
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      Android 浏览器应用
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      iOS Mobile Safari 应用
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      PhoneGap/Cordova other

    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      不支持的调试目标：
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      iOS 3.1.3 及更早版本
     </span>

    <div style="word-wrap:break-word; font-size:13px">
     <h4 style="background-color:rgb(47,88,156); color:rgb(255,255,255); font-size:17px">
      <span style="font-family:'Microsoft Yahei'">
       Weinre 缺点
      </span>
     </h4>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <span style="font-family:'Microsoft Yahei'">
      暂时不支持脚本的断点调试，如果要调试脚本，可以在js代码中多写一些console.log，检查输出的内容，从而达到调试的目的
