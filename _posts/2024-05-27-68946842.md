    <p>
     测试类型：
    </p>
    <p>
    </p>
    <p align="left">
     <span style="color:#333333">
      黑盒测试
     </span>
     <span style="color:#333333">
      ：
     </span>
     <span style="color:#333333">
      把测试对象当成一个黑盒子
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      测试人员完全不考虑逻辑结构和内部特性
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      只依据程 式的需求说明书来检查程式的功能是否满足它的功能说明。
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      白盒测试
     </span>
     <span style="color:#333333">
      ：
     </span>
     <span style="color:#333333">
      把测试对象当成一个透明的盒子
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      允许测试人员利用程序内部逻辑结构及相关信 息
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      设计或选择测试用例
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      对程式所有逻辑路径进行测试。
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      单元测试
     </span>
     <span style="color:#333333">
      ：
     </span>
     <span style="color:#333333">
      白盒测试的一种
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      对软件设计中的单元模块进行测试。
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      集成测试
     </span>
     <span style="color:#333333">
      ：
     </span>
     <span style="color:#333333">
      在单元测试的基础上
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      对单元模块之间的连接和组装进行测试。
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      系统测试
     </span>
     <span style="color:#333333">
      ：
     </span>
     <span style="color:#333333">
      在所有都考虑的情况下
     </span>
     <span style="color:#333333">
      ，
     </span>
     <span style="color:#333333">
      对系统进行测试。
     </span>
    </p>
    <span style="color:#333333">
     验收测试
    </span>
    <span style="color:#333333">
     ：
    </span>
    <span style="color:#333333">
     第三方进行的确认软件满足需求的测试。
    </span>
    <br/>
    <p>
     <span style="color:#333333">
      <br/>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      bug生命周期：
      <br/>
     </span>
    </p>
    <p>
     <span style="color:#333333">
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      1
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
      发现
     </span>
     <span style="color:#333333">
      BUG---new(
     </span>
     <span style="color:#333333">
      新的
     </span>
     <span style="color:#333333">
      )
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      2
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
      提交指派
     </span>
     <span style="color:#333333">
      BUG---Assigned
     </span>
     <span style="color:#333333">
      （
     </span>
     <span style="color:#333333">
      已指派的
     </span>
     <span style="color:#333333">
      ）
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      3
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
      打开、解决
     </span>
     <span style="color:#333333">
      BUG---Open
     </span>
     <span style="color:#333333">
      （
     </span>
     <span style="color:#333333">
      打开的
     </span>
     <span style="color:#333333">
      ），
     </span>
     <span style="color:#333333">
      Fixed
     </span>
     <span style="color:#333333">
      （
     </span>
     <span style="color:#333333">
      已修复的
     </span>
     <span style="color:#333333">
      ）
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      4
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
      回归
     </span>
     <span style="color:#333333">
      BUG---Reset(
     </span>
     <span style="color:#333333">
      再测试
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      5
     </span>
     <span style="color:#333333">
      ）
     </span>
     <span style="color:#333333">
      关闭
     </span>
     <span style="color:#333333">
      BUG---Closed
     </span>
     <span style="color:#333333">
      （
     </span>
     <span style="color:#333333">
      已关闭的
     </span>
     <span style="color:#333333">
      ）
     </span>
    </p>
