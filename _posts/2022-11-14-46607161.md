    <div style="font-family:微软雅黑; font-size:14px; line-height:21px">
     要说计算机现在什么最火，无异于网络。看看各大计算机网站，满眼尽是网络工程师，什么IP啊，路由啊，虚拟机啊，
     <span style="background-color:inherit; line-height:1.5">
      总之，操作性的计算机技能牢牢占据了半壁江山。这些技巧当然重要，但是也不能没有软件开发性技能吧，咋们亲情奉上C/C++工程师成长道路，为立志成为软件开发工程师的同学们提供指导。
     </span>
    </div>

     </span>
    </div>
    <h3 style="font-family:微软雅黑; font-size:14px; line-height:21px">
     <span style="background-color:inherit; line-height:1.5">
      <strong>
       第一、成为C/C++工程师，需要学习哪些内容？
      </strong>
     </span>
    </h3>
    <div style="font-family:微软雅黑; font-size:14px; line-height:21px">
     <span style="background-color:inherit; line-height:1.5">
      <br style="background-color:inherit"/>
     </span>
    </div>
    <div style="font-family:微软雅黑; font-size:14px; line-height:21px">
     <span style="font-family:宋体; background-color:inherit">
      从速成的角度，成为C/C++工程师的
     </span>
     <span style="background-color:inherit; line-height:1.5; font-family:宋体">
      成长图如下：
     </span>
    </div>
    <div style="font-family:微软雅黑; font-size:14px; line-height:21px">
     <div style="background-color:inherit">
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <img alt="" src="https://img-blog.csdn.net/20150623162654230?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemh1eXVueGlhbmdodQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
        <br/>
       </img>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="font-family:宋体; background-color:inherit">
        上面已经列举了很多课程，但是我仍然认为是“速成”，因为我认为：
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="font-family:宋体; background-color:inherit">
        1、学习计算机编程，真的不是一朝一夕就可以做到的，需要学习很多的基础和专业课程，而这些课程在上图中并没有显示。比如《计算机组成原理》、《汇编语言》、《编译原理》、《运筹学》、《计算机图形学》、《密码学》等等，还有很多，这里不一一列举了。
       </span>
       <span style="background-color:inherit; font-family:宋体; line-height:1.5">
        这些课程培养的是素质，是一种气场，
       </span>
       <span style="background-color:inherit; font-family:宋体; line-height:1.5">
        你要想在百家争鸣的计算机界站稳脚跟，就必须有这么一种气场，有这么一种自信，无论计算机发展多么的快，你都能抓住他们的核心。
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="font-family:宋体; background-color:inherit">
        <br style="background-color:inherit"/>
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="font-family:宋体; background-color:inherit">
        2、学习计算机编程，真的不是只学会计算机专业知识就可以的，它最需要跨界，需要学习专业的知识。软件编程，只需要计算机知识。
       </span>
       <span style="background-color:inherit; font-family:宋体; line-height:1.5">
        编好软件，就必须需要其他知识。举国外的例子（前一段时间我带着小孩去美国大学游，感触很大），美国的常青藤大学，一般都需要学习四个专业才能毕业。比如斯坦福大学计算机专业就必须要学习人类学专业才能毕业；耶鲁大学接待我们的女生是物理系的，她还学习了计算机、金融和艺术专业，即将去华尔街摩根投行工作。当然中国的国情不同，但是大家多几门兴趣爱好，总是好的。（比如鄙人，除了计算机，还懂金融，所以成为了湖南大学金统学院的研究生导师）。
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        <br style="background-color:inherit"/>
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        3、其实众多语言软件开发工程师的学习成长路程相差无几，上面很多课程，Java工程师照样需要学习。
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        <br style="background-color:inherit"/>
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        4、很多大学一般会开设上图中圆形部分的课程，而其他课程则需要学生自学。
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        <br style="background-color:inherit"/>
       </span>
      </p>
      <h3 style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="background-color:inherit; font-family:宋体">
        第二、
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        C/C++
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        语言的从业方向
       </span>
      </h3>
      <p style="margin-top:5px; margin-bottom:5px; background-color:inherit">
       <span style="font-family:宋体; background-color:inherit">
        要说以前那真的啥都能行，现在缩水很多了。最常见的是：
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        Unix/Linux
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        服务器程序设计（
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        C
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        语言）、嵌入式程序开发（
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        C
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        语言）和
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        Windows
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        桌面程序设计（
       </span>
       <span lang="EN-US" style="background-color:inherit; line-height:1.5">
        C++
       </span>
       <span style="background-color:inherit; line-height:1.5; font-family:宋体">
        ），当然如果你特别牛皮的话，也可以进入微软、INTEL、NBA等大型公司的研发部门。
