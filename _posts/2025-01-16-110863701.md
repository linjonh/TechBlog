    <p>
     <strong>
      协议
     </strong>
     ： 约定，就比如我们的普通话。
    </p>
    <p>
     <strong>
      网络通信协议
     </strong>
     ： 速率，传输码率，代码结构，传输控制…
    </p>
    <p>
     问题大事化小 ：分层
    </p>
    <p>
     <strong>
      TCP/IP协议簇
     </strong>
     <br/>
     <strong>
      重要
     </strong>
     ：
    </p>
    <ul>
     <li>
      TCP ：用户传输协议
     </li>
     <li>
      UDP ：用户数据报协议
     </li>
    </ul>
    <p>
     <strong>
      知名协议
     </strong>
    </p>
    <ul>
     <li>
      TCP
     </li>
     <li>
      IP ： 网络互联协议
      <br/>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/6317f429940afe6234a91763e6e323d6.png"/>
     </li>
    </ul>
    <p>
     <strong>
      TCP / UDP
     </strong>
     对比
     <br/>
     TCP ：打电话
    </p>
    <ul>
     <li>
      <p>
       连接，稳定
      </p>
     </li>
     <li>
      <p>
       <strong>
        三次握手，四次挥手
       </strong>
      </p>
      <pre><code>A : 你瞅啥？
B ：瞅你咋地？
A ：来！干一场
 

A ：分手吧
B ：你真的要分手吗？
B : 你真的决定要分手吗？
A : 嗯，真的要分手了
</code></pre>
     </li>
     <li>
      <p>
       客户端，服务端
      </p>
     </li>
     <li>
      <p>
       传输完成，释放连接，效率低
      </p>
     </li>
    </ul>
    <p>
     UDP ： 发短信
    </p>
    <ul>
     <li>
      不连接，不稳定
     </li>
     <li>
      客户端，服务端
      <strong>
       :
      </strong>
      没有明确的界限
     </li>
     <li>
      不管有没有和你连接，都可以发给你
     </li>
     <li>
      导弹
     </li>
     <li>
      DDOS ： 洪水攻击 (饱和攻击)
     </li>
    </ul>
