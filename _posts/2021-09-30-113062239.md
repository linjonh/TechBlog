     <a id="dosMySQL_0">

     使用dos命令行操作MySQL数据库
    </h2>
    <p>
     <strong>
      登录mysql数据库
     </strong>
    </p>
    <p>
     1.cd Mysql/bin的路径
     <br/>
     2.mysql -u用户名 -p,然后回车输入密码
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8c626f236eacc24d616b0fc765f79767.png#pic_center">
      <br/>
      <strong>
       若出现如下图的错误则为密码出错
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/505ed91ba0d57915396d29c25213d62d.png">
       <br/>
       <strong>
        出错原因一：
       </strong>
       <br/>
       密码错误。
       <br/>
       <strong>
        出错原因二：
       </strong>
       <br/>
       如果密码确认无误但一直报错，则有可能你的密码对应的并不是正在运行的服务。
       <br/>
       win10系统下，win+q搜索-&gt;输入“服务”-&gt;在右侧查找MySQL相关服务。注意系统中正在运行的MySQL服务名称，有的系统中会存在两个MySQL相关服务，一个MySQL服务，一个MySQL57服务。有可能你的root密码对应的是MySQL57服务的root密码但是现在运行的却是MySQL服务，这种情况则需要右键关闭MySQ服务，运行MySQL57服务。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/02a7b3b6f5b1e073f987ced9480d75f0.png">

     <strong>
      MySQL数据库命名规则（标识符规则）：
     </strong>
    </p>
    <ol>
     <li>
      不能和已存在的命名重名；
     </li>
     <li>
      由大小写字母、数据、下划线、@、# 和 $ 符号组成；
     </li>
     <li>
      首字母不能是数字和$符。
     </li>
     <li>
      不允许有空格和特殊字符。
     </li>
