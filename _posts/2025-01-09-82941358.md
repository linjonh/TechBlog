    <p>
     二叉搜索树的性质：
    </p>
    <p>
     1）若它的左子树不为空，则左子树上所有节点都小于根节点的值。
    </p>
    <p>
     2）若它的右子树不为空，则右子树上所有节点都大于根节点的值。
    </p>
    <p>
     3）它的左右子树也分别为二叉搜索树。
    </p>
    <p>
     二叉搜索树如下图：
    </p>
    <p>
     <img alt="" class="has" height="282" src="https://i-blog.csdnimg.cn/blog_migrate/72c8ee0319c3a7b206a8bef33971f4e6.png" width="463"/>
    </p>
    <p>
     二叉搜索树的创建过程：
    </p>
    <p>
     先找到需要插入值得父节点，比较父节点与插入值得大小，确定插入的位置。
    </p>
    <p>
     （找父节点，需要通过根节点与插入值得大小进行比较，确定根节点的左右孩子位置，继续循环，直到找到父节点的单个孩子节点为叶子结点。）
    </p>
    <p>
     头文件：二叉搜索树.h
    </p>

    </p>

     <br/>

     <!-- -->

     <br/>

     <br/>

     <!-- -->

     //如果root为空，表示此二叉树为空，需要建立根节点。
     <br/>

     <br/>
     <br/>

     <br/>

     <br/>

     <br/>

     <!-- -->

     <br/>

     <br/>
     }
     <br/>
     parent = root; //双亲节点指向根节点。
    </p>
    <p>
     //根节点的值大于所要插入的值，要插入的值在根节点的左边。
     <br/>
     if (root-&gt;key &gt; aa)
     <br/>
     {
     <!-- -->
     <br/>
     root = root-&gt;left;
     <br/>
     }
     <br/>
     else
     <br/>

     <br/>
     }
     <br/>
     }
     <br/>
     }
     <br/>
     <br/>
     BST*pp = (BST*)malloc(sizeof(BST));
     <br/>
     pp-&gt;key = aa;
     <br/>
     pp-&gt;left = pp-&gt;right = NULL;
     <br/>
     if (parent == NULL)
     <br/>
     {
     <!-- -->

     if (aa &lt; parent-&gt;key)
     <br/>
     parent-&gt;left = pp;
     <br/>
     else
     <br/>
     parent-&gt;right = pp;
     <br/>
     return 0;
