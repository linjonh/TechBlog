    </h4>
    <ul>
     <li>
      <p>
       负责人
      </p>
      <ul>
       <li>
        <p>

        <p>

        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       测试计划
      </p>
      <ul>
       <li>
        <p>
         概念：是指描述了要进行的测试活动的范围、方法、资源和进度的文档。
        </p>
       </li>
       <li>
        <p>
         核心内容：
        </p>
        <ul>
         <li>
          <p>
           范围与目标
          </p>
         </li>
         <li>
          <p>
           角色与职责
          </p>
         </li>
         <li>
          <p>
           进度与资源
          </p>
         </li>
         <li>
          <p>
           风险与应对
          </p>
         </li>
         <li>
          <p>
           准入准出标准
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       测试方案
      </p>
      <ul>
       <li>
        <p>
         概念：是从测试的技术角度去分析需求，在方向上明确要怎么测，分析结果重点在于测试策略与技术实现。
        </p>
       </li>
       <li>
        <p>
         核心内容
        </p>
        <ul>
         <li>
          <p>
           方法
          </p>
         </li>
         <li>
          <p>
           环境
          </p>
         </li>
         <li>
          <p>
           工具
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       面试题：测试计划与测试方案的区别?
      </p>
      <ul>
       <li>
        <p>
         测试计划是【管理型】文档，测试方案是【技术性】文档
        </p>
       </li>
       <li>
        <p>
         测试计划主要解决【做什么？】【谁来做？】，测试方案主要解决【怎么做？】
        </p>
       </li>
       <li>
        <p>
         主要内容存在差异：
        </p>
        <ul>
         <li>
          <p>
           测试计划主要内容如下：
          </p>
          <ul>
           <li>
            目标与范围
           </li>
           <li>
            角色与职责
           </li>
           <li>
            资源与进度
           </li>
           <li>
            风险与应对
           </li>
           <li>
            准入与准出
           </li>
          </ul>
         </li>
         <li>
          <p>
           测试方案主要内容如下：
          </p>
          <ul>
           <li>
            <p>
             策略与方法
            </p>
           </li>
           <li>
            <p>
             环境
            </p>
           </li>
           <li>
            <p>
             工具的选择
            </p>
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
