    <p>
     一、
     <strong>
      <strong>
       申请微信公众账号测试号：
      </strong>
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       <strong>
        因本地开发自己公众号无法认证，所以在本地开发时需要用微信公众账号测试号申请系统；申请步骤如下：
       </strong>
      </strong>
     </li>
     <li>
      <strong>
       <strong>
        登录注册自己的微信公众平台，选择左侧菜单栏中开发-开发者工具，出现的页面点击开发者文档-进入后点击开始开发-选择接口测试号申请-点击进入微信公众帐号测试号申请系统-自己的微信扫码登录即可开通自己的测试号；具体如下几图所示；
       </strong>
      </strong>
     </li>
     <li>
      <img alt="" class="has" height="449" src="https://i-blog.csdnimg.cn/blog_migrate/27f5a0743d27b29f9303c7182d19bc6c.png" width="226"/>
     </li>
     <li>
      <img alt="" class="has" height="271" src="https://i-blog.csdnimg.cn/blog_migrate/a63f15c3af39d48dc53c1d64a6d91a31.png" width="619"/>
     </li>
     <li>
      <img alt="" class="has" height="493" src="https://i-blog.csdnimg.cn/blog_migrate/e24bd9e824f4215022d0067f2043518a.png" width="626"/>
     </li>
     <li>
      <img alt="" class="has" height="286" src="https://i-blog.csdnimg.cn/blog_migrate/0c95d912ed5783dd5dab32bf3c28bc48.png" width="568"/>
     </li>
    </ol>
    <p>
     <strong>
      <strong>
       二、
      </strong>
     </strong>
     上图中接口配置信息中的URL地址为Ngrok开内网映射后的地址，实际生产中需替换成项目部署服务器地址；Ngrok使用可见我的另一篇博客：https://mp.csdn.net/postedit/103903333
    </p>
    <p>
     <strong>
      <strong>
       三、
      </strong>
     </strong>
     <strong>
      <strong>
       添加本地服务器
      </strong>
     </strong>
     <strong>
      <strong>
       URL
      </strong>
     </strong>
     <strong>
      <strong>
       到公众号测试号上。
      </strong>
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       <strong>
        测试账号在”测试号管理-接口配置信息”中点击修改，添加自己服务器URL地址指向的全局服务的servlet，我的本工程是com.wx.servlet.WxServlet 调用为/wx,所以URL=http://zfr.free.idcfengye.com/wx。Token的值跟程序中设置的值对应，如下图。
       </strong>
      </strong>
     </li>
     <li>
      <img alt="" class="has" height="279" src="https://i-blog.csdnimg.cn/blog_migrate/c01bdf7e4fd8c7f6a5120db432374138.png" width="604"/>
     </li>
     <li>
      <img alt="" class="has" height="331" src="https://i-blog.csdnimg.cn/blog_migrate/060fb31080bc154b470224334b7affad.png" width="620"/>
     </li>
     <li>
      注意：如果接口配置信息修改一直提示"配置失败"，并且错误信息为{"errcode":-106,"errmsg":"token check fail"}，可以考虑下是网络问题或者是内网穿透工具的问题(我的就是这个问题引起的，我用的免费的，用了同事买的就可以了，欲哭无泪啊！！！)
     </li>
    </ol>
    <p>
     <strong>
      <strong>
       四、
      </strong>
     </strong>
     <strong>
      <strong>
       公众号JSSDK使用注意事项:
      </strong>
     </strong>
    </p>
    <p>
     <strong>
      <strong>
       在测试账号在”测试号管理-JS接口安全域名”中点击修改，加入自己的域名(注意格式)；此域名跟网页授权配置的域名一致(zfr.free.idcfengye.com)
      </strong>
     </strong>
    </p>
    <p>
     <img alt="" class="has" height="344" src="https://i-blog.csdnimg.cn/blog_migrate/76b9b905cbf198c9b9d5899c37ee258f.png" width="574"/>
    </p>
    <p>
     <strong>
      <strong>
       五、
      </strong>
     </strong>
     <strong>
      <strong>
       微信公众号网页授权配置:
      </strong>
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       <strong>
        需要在”接口权限-网页服务-网页账号-网页授权获取用户基本信息”中点击后面的修改，增加自己服务的域名；如下图；
       </strong>
      </strong>
     </li>
     <li>
      <img alt="" class="has" height="91" src="https://i-blog.csdnimg.cn/blog_migrate/c2917c598967a2e9aaeca27bd3e6414a.png" width="932"/>
