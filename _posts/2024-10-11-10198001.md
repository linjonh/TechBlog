    <p>
     <strong>
      <span style="font-size:14px">
       一、背景
      </span>
     </strong>
    </p>
    <p>
     <span style="font-size:14px">
      公司要求应届毕业生必须参加软考，我觉得挺好的。看了下资格项目，我对 嵌入式系统设计师[中级资格项目] 比较感兴趣。并希望来年能考 系统架构设计师[高级资格项目]。不管怎么样，考了对以后肯定有帮助。
      <span style="color:#FF0000">
       自己一定要好好复习！必须一次通过，并且两个都需要过！
      </span>
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       二、软考介绍
      </span>
     </strong>
    </p>
    <ul>
     <li>
      <span style="font-size:14px">
       高级资格：
       <span style="color:#000000">
        <a href="http://baike.baidu.com/view/1278542.htm" rel="nofollow noopener noreferrer" target="_blank">
         网络规划设计师
        </a>
       </span>
       、
       <a href="http://baike.baidu.com/view/1278545.htm" rel="nofollow noopener noreferrer" target="_blank">
        系统规划与管理师
       </a>
       、
       <a href="http://baike.baidu.com/view/1983635.htm" rel="nofollow noopener noreferrer" target="_blank">
        系统架构设计师
       </a>
       、
       <a href="http://baike.baidu.com/view/1760754.htm" rel="nofollow noopener noreferrer" target="_blank">
        信息系统项目管理师
       </a>
       、
       <a href="http://baike.baidu.com/view/335304.htm" rel="nofollow noopener noreferrer" target="_blank">
        系统分析师
       </a>
       ；
      </span>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px">
      </span>
      <div class="para">
       <span style="font-size:14px">
        中级资格：计算机硬件工程师、信息安全工程师、软件过程能力评估师、
        <a href="http://baike.baidu.com/view/1278553.htm" rel="nofollow noopener noreferrer" target="_blank">
         系统集成项目管理工程师
        </a>
        、软件评测师、软件设计师、网络工程师、多媒体应用设计师、
        <span style="color:#FF0000">
         <strong>
          嵌入式系统设计师
         </strong>
        </span>
        、电子商务设计师、信息系统监理师、数据库系统工程师、信息系统管理工程师；
       </span>
      </div>
     </li>
    </ul>
    <ul>
     <li>
      <div class="para">
       <span style="font-size:14px">
        初级资格：网页制作员、信息处理技术员、信息系统运行管理员、电子商务技术员、网络管理员、程序员。
       </span>
      </div>
     </li>
    </ul>
    <p>
     <span style="font-size:14px">
      从2004年开始，
      <span style="color:#FF0000">
       每年举行两次考试，上、下半年各举行一次
      </span>
      ，上半年和下半年考试的级别不尽相同，以全国软考办公布的考试计划为准。每次公布计划的时间大约在考试前后一周。
     </span>
     <strong>
      <br/>
      <span style="font-size:14px">
       <span style="background-color:rgb(255,255,255)">
       </span>
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
      </span>
     </strong>
     <span style="font-size:14px; color:#FF0000">
      自2004年至2012年历年的软考情况，所有各科考试合格标准为45分。
     </span>
    </p>
    <p style="text-align:center">
     <span style="font-size:14px; color:#FF0000">

       <br/>
      </img>
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       三、网友经历
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
      </span>
     </strong>
     <span style="font-size:14px">
      计算机软考本人考过两次，科目选的是系统集成项目管理工程师（含金量比较高），在此和你交流一下，希望对你有帮助。
     </span>
     <span style="font-size:14px">
      <br/>
     </span>
    </p>
    <ul>
     <li>
      <span style="font-size:14px">
       关于报名：到你所在地区软考考试报名网站网上报名即可，用网银交费用78元，全国各省软考报名网站你可以到
       <span style="color:#FF0000">
        中国计算机职业技术资格网
       </span>
       去查询
      </span>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px; color:#FF0000">
       报名时间：上半年大概在三月下旬到四月上旬，下半年大概在九月下旬到十月上旬。
      </span>
      <span style="color:#FF0000">
       <br/>
      </span>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px; color:#FF0000">
       考试时间：上半年 大概在五月下旬的一个周六，下半年 大概在十一月的一个周六。
      </span>
      <br/>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px">
       报名方法：登陆报名网站，依次填写报名信息，网银支付费用，成功。
      </span>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px">
       打印准考证：考前20天开始密切关注考试报名网站发布的信息，一般考前1--20天可打印准考证，不同省份时间不一样。
      </span>
     </li>
    </ul>
    <ul>
     <li>
      <span style="font-size:14px">
       考试：带身份证和准考证，2B铅笔，答题笔。
      </span>
     </li>
    </ul>
    <p>
     <span style="font-size:14px">
      大概就是这样，祝你好运啦！
     </span>
     <strong>
      <span style="font-size:14px">
       <br/>
      </span>
     </strong>
    </p>
