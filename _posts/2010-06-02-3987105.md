     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      vs2008(MFC)
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      vs2008
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      (10.5)
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      <br/>
      <br/>
      <span style="mso-spacerun: yes;">
      </span>
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      我尝试了添加水晶报表，步骤是新建一个对话框项目
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      (
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      默认设置
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      )
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      Active X controls(Crystal ActiveX Report Viewer Control 10.5),

     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      之后添加变量与这个空间关联
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      ,
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      但是这是出现一个错误
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      The Extended Provider failed to return an Extender for this object
     </span>
     <span style="font-size: 9pt; color: blue; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      。如果我忽略这个继续，在对话框的变量类型里是空的，我添加变量无法成功，所以请教各位我是否需要配置什么或者安装什么？
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: blue; font-family: Verdana;">
      <br style="mso-special-character: line-break;"/>
      <br style="mso-special-character: line-break;"/>
     </span>
    </p>

     </span>

       </span>
      </span>
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      要先添加类，不是在可视化编辑界面下，而是在类视图模式下添加
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      ActiveX
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      控件中的
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      MFC
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      类
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      -&gt;Crystal Report Viewer Control 9
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      cpp
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">

     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      h
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      文件。
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      <br/>

     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      然后再从可视化编辑界面下添加
     </span>
     <span lang="EN-US" style="font-size: 9pt; color: red; font-family: Verdana;">
      ActiveX
     </span>
     <span style="font-size: 9pt; color: red; font-family: 宋体; mso-ascii-font-family: Verdana; mso-hansi-font-family: Verdana;">
      控件。
     </span>
    </p>
    <p class="MsoNormal" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">

       <span style="font-family: Times New Roman;">

      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman';">
