    <h2>
     业务需求：
    </h2>
    <p>
     近期小程序要做一个分享海报，带小程序码携带用户分享的参数。
    </p>
    <h2>
     接口文档：
    </h2>
    <h4>
     使用小程序云函数，先去看下文档吧
    </h4>
    <p>
     地址：
     <a href="https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/qr-code.html" rel="nofollow" title="获取小程序码 | 微信开放文档">
      获取小程序码 | 微信开放文档
     </a>
    </p>
    <p>
     为满足不同需求和场景，这里提供了两个接口，开发者可挑选适合自己的接口。
    </p>
    <ul>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api-backend/open-api/qr-code/wxacode.get.html" rel="nofollow" title="接口 A: 适用于需要的码数量较少的业务场景">
       接口 A: 适用于需要的码数量较少的业务场景
      </a>
      <ul>
       <li>
        生成小程序码，可接受 path 参数较长，生成个数受限，数量限制见
        <a href="https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/qr-code.html#%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9" rel="nofollow" title="注意事项">
         注意事项
        </a>
        ，请谨慎使用。
       </li>
      </ul>
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api-backend/open-api/qr-code/wxacode.getUnlimited.html" rel="nofollow" title="接口 B：适用于需要的码数量极多的业务场景">
       接口 B：适用于需要的码数量极多的业务场景
      </a>
      <ul>
       <li>
        生成小程序码，可接受页面参数较短，生成个数不受限。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     我们要求是能带参数，又生成比较多所以用B接口
    </p>
    <p>
     <a href="https://developers.weixin.qq.com/miniprogram/dev/api-backend/open-api/qr-code/wxacode.getUnlimited.html" rel="nofollow" title="wxacode.getUnlimited | 微信开放文档">
      wxacode.getUnlimited | 微信开放文档
     </a>
    </p>

    <h2>
     代码实现
    </h2>
    <h4>
     建立云函数
    </h4>
    <p>
     <img alt="" height="652" src="https://i-blog.csdnimg.cn/blog_migrate/c107b41785dd56015bb032fcfc59e36a.png" width="484"/>
    </p>
    <p>
     我们建一个qrcode云函数，然后在index.js里面写我们的业务逻辑，具体代码如下 ：
    </p>
    <pre><code class="language-javascript">// 云函数入口文件
const cloud = require('wx-server-sdk')
const rp = require('request-promise')

cloud.init()

// 云函数入口函数
exports.main = async (event, context) =&gt; {

  const page = event.page
  const scene = event.scene

    //appid和秘钥
    const appid = '小程序的appid',
      secret = '小程序的secret 在小程序后台设置里获取下';

    const AccessToken_options = {
      method: 'GET',
      url: 'https://api.weixin.qq.com/cgi-bin/token',
      qs: {
        appid,
        secret,
        grant_type: 'client_credential'
      },
      json: true

    };

    //获取AccessToken
    const resultValue = await rp(AccessToken_options);
    const token = resultValue.access_token;

    //获取小程序码配置
    const code_options = {
      method: 'POST',
      url: 'https://api.weixin.qq.com/wxa/getwxacodeunlimit?access_token=' + token,
      body: {
        'page': page,
        'width': 430,
        'scene': scene
      },
      json: true,
      encoding: null
    };

    //获取二进制图片
    const buffer = await rp(code_options);

    const upload = await cloud.uploadFile({
      cloudPath: 'qr/' + new Date().getTime() + '.png',
      fileContent: buffer,
    })
    return {
      wxacodefileID: upload.fileID
    }

}
</code></pre>
    <h4>
     前端调用
    </h4>
    <p>
     需要生成小程序码的页面写以下代码
    </p>
    <pre><code class="language-javascript">qrcode:function(){
    // 调用云函数
    wx.cloud.callFunction({
      // 需调用的云函数名
      name: 'qrcode',
      // 传给云函数的参数
      data: {
        page: 'pages/file/file',
        scene:'id=504&amp;uid=123',
      },
      // 成功回调
      complete: console.log
    })
  },</code></pre>

     在需要生成的按钮上添加你写的动作this.qrcode();，page就是你小程序的页面，scene就是你要携带的参数，注意这个参数长度是有限制的尽量不要写太长。运行看到如下信息：

    <h2>
     识别SCENE
    </h2>
    <p>
     生成的二维码你扫描后发现所带参数是无法直接识别的，我们稍加处理即可获得生成时传递的参数
    </p>
    <pre><code class="language-javascript">console.log(this.parseScene(decodeURIComponent(options.scene)))</code></pre>
    <p>
     参数的解析函数
    </p>
    <pre><code class="language-javascript">parseScene(scene) {      
    var params = new Object(); // var params = {};也行        
    var scene = decodeURIComponent(scene).split('&amp;');       
    for (var i = 0; i &lt; scene.length; i++) {          
      params[scene[i].split('=')[0]] = scene[i].split('=')[1];    
    }         
    let args = JSON.stringify(params); 
    return args;    
    // wx.navigateTo({url: `/shop/pages/detail/detail?args=${args}`});
  }</code></pre>
