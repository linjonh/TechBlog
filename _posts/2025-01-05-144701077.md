    <h2>
     开发技术
    </h2>
    <ol>
     <li>
      开发环境：JDK1.8 + Tomcat8+
     </li>
     <li>
      开发语言：Java
     </li>
     <li>
      开发框架：springboot
     </li>
     <li>
      模板引擎：Thymeleaf
     </li>
     <li>
      开发工具：Idea2022
     </li>
     <li>
      数据库：mysql8

     <li>

    <h2>
     功能清单
    </h2>
    <p>
     <img alt="" height="428" src="https://i-blog.csdnimg.cn/direct/ad675b66ec544cc0b1eb620d4a134e3f.png" width="744"/>
    </p>
    <p>
     【后台管理员功能】
     <br/>
     广告管理：设置小程序首页轮播图广告和链接
     <br/>
     留言列表：所有用户留言信息列表，支持删除
     <br/>
     会员列表：查看所有注册会员信息，支持删除
     <br/>
     资讯分类：录入、修改、查看、删除资讯分类
     <br/>
     录入资讯：录入资讯标题、内容等信息
     <br/>
     管理资讯：查看已录入资讯列表，支持删除和修改
     <br/>
     资讯评论列表：所有用户的评论信息列表
     <br/>
     资讯评论管理：支持对评论信息审核，删除；审核后的信息用户才可见
    </p>
    <p>
     <br/>
     【网站功能】
     <br/>
     用户注册：填写手机账号和密码，注册新用户
     <br/>
     登录功能：注册普通账号登录；登录后可以修改用户的基本信息，也可以退出。
     <br/>
     关于我们：关于我们、联系我们、加入我们、法律声明
     <br/>
     轮播广告：后台设置首页轮播广告图，可以连接到广告页面。
     <br/>
     留言反馈：用户填写李哭咽的主题、联系人、电话、邮箱、留言内容；后台管理可以查看留言列表，可以删除留言。
    </p>
    <p>
     资讯查询：支持查询整个新闻系统的内容，输入关键字点击搜索即可。
     <br/>
     资讯阅读：游客和用户都可以进行资讯的阅读。
     <br/>
     资讯收藏：阅读到有兴趣的资讯，注册登录的用户可以进行收藏操作。
     <br/>
     资讯评论：阅读资讯的时候，登录的用户，可以发表评论；后台审核后显示。
    </p>
    <p>
     浏览记录：记录用户所有浏览过的资讯信息
     <br/>
     我的收藏：收藏的感兴趣的资讯文章，可以直接点击去阅读，支持删除。
     <br/>
     我参与评论的文章：列出参与评论的文章、评论内容、审核状态。
    </p>
    <p>
     用户信息：姓名、联系方式、邮箱、头像、简介、介绍等，支持随时修改；用户注册的信息后台管理员可见；后台管理员可以删除。
     <br/>
     密码修改：修改注册的密码。
     <br/>
     退出登录：清除登录的cookie，返回到首页。
    </p>
