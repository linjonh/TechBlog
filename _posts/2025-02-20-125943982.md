     COPU副主席刘澎：中国开源在局部领域已接近或达到世界先进水平
    </h1>
   </div>
  </div>

    <p>
     <span style="color:#494949;">
      整理 |
      <a class="link-info" href="https://blog.csdn.net/qq_35529258?type=blog" title="西狩xs">
       西狩xs
      </a>
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      自UNIX引入中国，中国开源发展已历经三十余年。在这三十年间，开源以“开放、共享、协同＋商业模式”的新型生产方式，通过“自由”传播，成为信息技术发展的强大推动力。在全球开源浪潮中，中国开源飞速发展，在开源世界中所扮演的角色已经从受益者转变为了极为重要的贡献者。
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      2022年7月21日，由中国开源软件推进联盟主办，赛迪传媒、《软件和集成电路》杂志社联合承办，CSDN独家直播的“第十七届开源中国开源世界高峰论坛”上，中国开源软件推进联盟副主席刘澎发布了《2022中国开源发展蓝皮书》，对目前的中国开源发展进行了全面阐述。
     </span>
    </p>
    <p>
    </p>
    <p style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b2375d69cde970575b52c27a60e012d1.jpeg"/>
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      <strong>
       以下为
      </strong>
      <strong>
       发布
      </strong>
      <strong>
       实录：
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      《2022中国开源发展蓝皮书》由中国开源软件推进联盟（COPU）牵头，联合中国电子信息产业发展研究院、中国开发者网络（CSDN）、北京开源创新委员会、开放原子开源基金会、中科院软件研究所等85家企业及行业机构、120多位开源专家和志愿者共同协作编撰完成。
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      《2022中国开源发展蓝皮书》旨在展现中国开源发展现状的真实图谱，对中国开源发展从开源基本概念、开源开发者现状、开源项目、开源社区、开源组织及开源生态、开源教育、开源商业、开源发展机遇与挑战、开源贡献的维度进行总结，为国家政府、企业机构、开源从业者提供理论和数据基础。
     </span>
    </p>
    <h2 style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/3ccc3fd284fcf8f7601cce3249455371.png"/>
     </span>
    </h2>
    <p style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a75fba3852f45d405b3de4e5c7e0fdbe.png"/>
     </span>
    </p>
    <h3 style="text-align:center;">
     <strong>
      <span style="color:#494949;">
       发展历程
      </span>
     </strong>
    </h3>
    <p>
     <span style="color:#494949;">
      中国开源的发展速度为全球最快，在局部领域已接近或达到世界先进水平，但仍然存在发展不平衡的特点。中国开源在社区、基金会、孵化器、代码托管平台，以及教育、开源合规、知识产权保护、风险投资等开源基础设施和支撑体系建设方面，已取得较大进展，但部分环节仍然薄弱。
     </span>
    </p>
    <p>
     <span style="color:#494949;">
      目前，中国开源的发展水平可观察到三个层面。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       COPU在2004年成立后，创建并汇聚了一支世界级开源团队的开源高地，在推动中国、东北亚及全球开源运动中产生深刻影响。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       早期，以红旗为代表的一批开源企业，在推动中国开源运动中发挥了先导作用。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       国内一批大型IT企业积极拥抱开源，推动中国开源运动的蓬勃发展。
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      随着2021年开源被列入国家顶层规划《中华人民共和国国民经济和社会发展第十四个五年规划和2035年远景目标纲要》，中国开源在开源基础设施和支撑体系建设方面取得了较大进展。同时，中国开源产业仍需对各个方面加强和完善，持续打造开源高地、创新高地、科技高地、人才高地。
     </span>
    </p>
    <h2 style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/71e9613224521c42fde4e6173ce48975.png"/>
     </span>
    </h2>
    <h3 style="text-align:center;">
     <strong>
      <span style="color:#494949;">
       发展现状
      </span>
     </strong>
    </h3>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        开源的最大活力来源于数千万的开发者
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      据统计，CSDN的中国开发者用户注册数超过3500万，2021年用户增长超700万，超94%正在使用开源，42%参与过开源项目。2021年，Gitee新增用户数超180万，累计超800万。同年，GitLink支撑了国家9个软件园区创新平台，服务2500余家中小型软件企业研发，GitHub中国开发者数量增长103万，总数达到755万，增幅48%，排名全球第二。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        开源项目的高速发展，技术与产业协同共进。
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      操作系统迅速发展，形成了以手机、嵌入式设备及自动驾驶方向为核心的产业热点。在数据库、人工智能领域，部分中国开源项目已跻身世界前列。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        开源社区不断崛起，但运营和治理还处在初级阶段。
       </strong>
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       <strong>
        开源组织及生态高速发展，越来越多的开源组织在生态建设过程中发挥着积极作用。
       </strong>
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       <strong>
        开源教育平台
       </strong>
       <strong>
        、
       </strong>
       <strong>
        开源教育体系不断落地，取得成绩同时，也存在待攻克的难题。
       </strong>
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       <strong>
        开源项目和开源企业已成为资本市场追捧的投资热点领域，Open Core
       </strong>
       <strong>
        +
       </strong>
       <strong>
        云原生已经成为主流。
       </strong>
      </span>
     </li>
    </ul>
    <h2 style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/bbdf5f928f17ff5d05699386d5028a0a.png"/>
     </span>
    </h2>
    <h3 style="text-align:center;">
     <strong>
      <span style="color:#494949;">
       机遇与挑战
      </span>
     </strong>
    </h3>
    <p>
     <span style="color:#494949;">
      <strong>
       中国开源发展的机遇主要表现在
      </strong>
      <strong>
       以下几个方面。
      </strong>
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       中国的产业链优势为开源发展提供了历史机遇，极大的市场容量、众多的开源企业、完善的电子工业产业链使得开源技术在中国市场陆续落地。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       传统行业也在积极拥抱开源，企业数字化转型带来了新的增长机遇，开源的开放性协作使开发流程更加敏捷，开源社区打通了企业与外部经营的合作，提升了企业竞争力，带动了开源技术创新。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       开源与各技术领域深度融合，开拓了开源应用的广阔市场。
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      <strong>
       中国开源发展过程中，机遇与挑战是同时存在的。挑战表现在
      </strong>
      <strong>
       以下几个方面。
      </strong>
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       技术风险长期存在。开源软件的漏洞数量保持高位，且影响范围巨大，同时需要探索上游组件快速修复方式，并大规模、全覆盖地推送到下游依赖环节。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       法律风险日趋突出。体现在两个方面：一是如何正确使用开源许可证；二是开源代码与其他代码相互渗透，由此，引发著作权、专利权和商业秘密判断等问题。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       开源软件供应链日趋庞杂。体现在两个方面：一是准入、管理、退出，是如何确保开源软件长期维护的行业行政任务。二是国际局势动荡，凸显出开源组织（如基金会）、开源代码托管平台（属于商业公司所有）受到了属地出口管制政策的制约。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       人才问题比较突出。拥有参加重大开源项目的经历的人才，在市场中有着很强的竞争力。这侧面也反映出开源高端人才的匮乏现状，以及企业对开源人才的培养成本投入偏少，开源人才留存困难等问题。
      </span>
     </li>
    </ul>
    <h2 style="text-align:center;">
     <span style="color:#494949;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9e096c640cd72e6125686207c1b036bc.png"/>
     </span>
    </h2>
    <h3 style="text-align:center;">
     <strong>
      <span style="color:#494949;">
       发展建议
      </span>
     </strong>
    </h3>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        加强开源人才培养，推动开源教育和价值引导。
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      企业和社会要加强开发者人才培养，加强国际顶级开源人才的交流，推动中国开源人才高地建设。推动开源成果在高校的课程和认证体系的设计、师资队伍建设和培养计划，培养开源创新人才。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        推动开源开发者以及社区建设，
       </strong>
       <strong>
        强大
       </strong>
       <strong>
        开源基础动力。
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      鼓励开发者和开发者社区普及开源文化。倡导开源精神，积极参加开源项目，贡献开源，建设好开发者社区，开源代码平台，壮大开源基金会。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        大力扶持优秀开源项目，推动商业化运作以及生态发展。
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      针对重点开源项目如操作系统、数据库、中间件等基础软件要进一步加大探索的力度，建立开源生态。在热门融资赛道，如云原生、人工智能、大数据、物联网、元宇宙等领域，与传统行业及数字化应用相结合，打造稳定的商业化路径。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#494949;">
       <strong>
        加强行业标准以及规范研究，解决安全隐患以及供应链风险。
       </strong>
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#494949;">
      培养开源软件供应链安全人才，建设开源供应链安全运维团队。建立开源供应链安全实验室，构建开源供应链安全评估体系。
     </span>
    </p>
