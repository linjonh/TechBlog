    <h3 style="margin-left:.0001pt;text-align:center;">
     编程语言中的语法符号和标识符
    </h3>
    <p style="margin-left:.0001pt;text-align:justify;">
     语法符号是编程语言中用于结构化和组织代码的符号，而标识符是用于标识和命名代码中的元素，使其具有唯一性和可识别性。它们在编程语言中发挥不同的作用，但共同构成了编程语言的语法体系和代码的组织方式。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      语法符号
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     在编程，语法符号（也称为标点符号或特殊字符）是用来定义和分隔代码结构的特殊字符。它们帮助编译器或解释器理解程序的结构和意图。以下是一些常见的语法符号及其用途：
    </p>
    <ul>
     <li style="text-align:justify;">
      <strong>
       ;
      </strong>
      （分号）：用于语句的结束。
     </li>
     <li style="text-align:justify;">
      <strong>
       ()
      </strong>
      （括号）：用于函数调用、参数列表或表达式的分组。
     </li>
     <li style="text-align:justify;">
      <strong>
       {}
      </strong>
      （大括号）：用于代码块的定义，如函数体、循环体和条件语句的范围。
     </li>
     <li style="text-align:justify;">
      <strong>
       []
      </strong>
      （方括号）：用于数组或列表的定义和访问。
     </li>
     <li style="text-align:justify;">
      <strong>
       ""
      </strong>
      或
      <strong>
       ''
      </strong>
      （引号）：用于字符串的定义。
     </li>
     <li style="text-align:justify;">
      <strong>
       :
      </strong>
      （冒号）：用于定义键值对（如字典）或在某些语言中用于控制流（如Python中的循环和条件语句）。
     </li>
     <li style="text-align:justify;">
      <strong>
       ,
      </strong>
      （逗号）：用于分隔函数参数、数组元素或变量。
     </li>
     <li style="text-align:justify;">
      <strong>
       .
      </strong>
      （点）：用于访问对象的属性或方法。
     </li>
     <li style="text-align:justify;">
      <strong>
       #
      </strong>
      （井号）：用于注释。
     </li>
     <li style="text-align:justify;">
      <strong>
       //
      </strong>
      或
      <strong>
       /* */
      </strong>
      ：单行或多行注释。
     </li>
     <li style="text-align:justify;">
      <strong>
       +
      </strong>
      、
      <strong>
       -
      </strong>
      、
      <strong>
       *
      </strong>
      、
      <strong>
       /
      </strong>
      （算术运算符）：用于数学运算。
     </li>
     <li style="text-align:justify;">
      <strong>
       =
      </strong>
      、
      <strong>
       ==
      </strong>
      、
      <strong>
       !=
      </strong>
      、
      <strong>
       &lt;
      </strong>
      、
      <strong>
       &gt;
      </strong>
      （比较运算符）：用于赋值和比较操作。
     </li>
    </ul>
    <p style="margin-left:.0001pt;text-align:justify;">
     这些符号
     <span style="color:#fe2c24;">
      必须在英文状态下输入
     </span>
     ，因为大多数编程语言都是基于英文字符集设计的。使用中文状态的符号会导致语法错误，因为编译器或解释器无法识别这些符号。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     不同编程语言可能会有一些特定的语法符号，但以上列出的符号是大多数语言中常见的。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      标识符
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     标识符是由程序员定义的名称，用于标识变量、函数、类、模块等编程元素。标识符通常由字母、数字和下划线组成，但不能以数字开头。标识符的具体规则可能因编程语言而异。示例如下：
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     变量名：counter, totalSum, userName
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     函数名：calculateTotal(), fetchData()
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     类名：Customer, Employee
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     标识符用于在程序中引用数据和功能，它们是代码的核心组成部分，可以由开发者自由命名，但必须遵循语言的命名规则和惯例。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     标识符的命名则受到编程语言的命名规则和约定的限制，例如首字符不能是数字，不能使用关键字作为标识符等。大多数编程语言如C++、Python、JavaScript、Java等标识符的命名区分大小写（称为大小写敏感），如 myVariable、myvariable、MyVariable和MYVARIABLE被视为是不同的标识符。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      现代编程语言支持 Unicode，因此支持中文字符，但大多数情况下，建议使用英文标识符以提高代码的可读性和维护性。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      语法符号和标识符的区别
     </strong>
    </p>
    <ol>
     <li style="text-align:justify;">
      <strong>
       功能和用途
      </strong>
      ：语法符号用于结构化和组织代码，指示编程语言的结构，例如括号、分号、操作符等。标识符用于标识和命名代码中的变量、函数、类等元素，使其具有唯一性和可识别性。
     </li>
     <li style="text-align:justify;">
      <strong>
       表示方式
      </strong>
      ：语法符号通常是特定的字符或符号，例如括号、点、操作符等，用于表示编程语言中的语法结构。标识符是由字母、数字和下划线组成的字符串，用于命名代码元素。
     </li>
     <li style="text-align:justify;">
      <strong>
       规则和约束
      </strong>
      ：语法符号在编程语言中有特定的语法规则和用法，例如操作符的优先级、括号的配对等。标识符的命名则受到编程语言的命名规则和约定的限制，例如首字符不能是数字，不能使用关键字作为标识符等。
     </li>
    </ol>
