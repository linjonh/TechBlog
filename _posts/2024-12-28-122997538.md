    <h2>
     <a id="_0">
     </a>
     树查找
    </h2>
    <p>
     对于层次结构的树，需要遍历其节点，根据遍历方式不同，可分为广度优先和深度优先，对于如下树结构
    </p>
    <pre><code>class TreeNode&lt;T&gt; {

    T value;
    TreeNode&lt;T&gt; leftChild;
    TreeNode&lt;T&gt; rightChild;

    TreeNode(T value) {
        this.value = value;
    }
}
</code></pre>
    <h3>
     <a id="_14">
     </a>
     广度优先搜索
    </h3>
    <p>
     从根节点开始判断，若元素不相等，则将其左右节点存进队列，利用队列先进先出的性质从上往下从左往右依次遍历节点
    </p>
    <pre><code>public &lt;T&gt; boolean BreadthFirstSearch(TreeNode&lt;T&gt; root, T value) {
    Queue&lt;TreeNode&lt;T&gt;&gt; queue = new LinkedList&lt;&gt;();
    StringBuilder stringBuilder = new StringBuilder();
    queue.offer(root);
    while (!queue.isEmpty()) {
        TreeNode&lt;T&gt; node = queue.poll();
        stringBuilder.append(node.value + " ");
        if (node.value == value) {
            System.out.println(stringBuilder.toString() + "found");
            return true;
        }
        if (node.leftChild != null) {
            queue.offer(node.leftChild);
        }
        if (node.rightChild != null) {
            queue.offer(node.rightChild);
        }
    }
    System.out.println(stringBuilder.toString() + "not found");
    return false;
}
</code></pre>
    <p>
     调用过程
    </p>
    <pre><code>TreeNode&lt;Integer&gt; node1 = new TreeNode&lt;&gt;(1);
TreeNode&lt;Integer&gt; node2 = new TreeNode&lt;&gt;(2);
TreeNode&lt;Integer&gt; node3 = new TreeNode&lt;&gt;(3);
TreeNode&lt;Integer&gt; node4 = new TreeNode&lt;&gt;(4);
TreeNode&lt;Integer&gt; node5 = new TreeNode&lt;&gt;(5);
node1.leftChild = node2;
node1.rightChild = node3;
node2.leftChild = node4;
node2.rightChild = node5;

BreadthFirstSearch(node1, 4);
</code></pre>
    <h3>
     <a id="_53">
     </a>
     深度优先搜索
    </h3>
    <p>
     从根节点开始判断，若元素不相等，则将其左右节点存进栈，利用栈先进后出的性质从上往下依次遍历节点
    </p>
    <pre><code>public &lt;T&gt; boolean DepthFirstSearch(TreeNode&lt;T&gt; root, T value) {
    Stack&lt;TreeNode&lt;T&gt;&gt; stack = new Stack&lt;&gt;();
    StringBuilder stringBuilder = new StringBuilder();
    stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode&lt;T&gt; node = stack.pop();
        stringBuilder.append(node.value + " ");
        if (node.value == value) {
            System.out.println(stringBuilder.toString() + "found");
            return true;
        }
        if (node.rightChild != null) {
            stack.push(node.rightChild);
        }
        if (node.leftChild != null) {
            stack.push(node.leftChild);
        }
    }
    System.out.println(stringBuilder.toString() + "not found");
    return false;
}
</code></pre>
    <ul>
     <li>
      若右节点先入栈，则先遍历左子树
     </li>
     <li>
      若左节点先入栈，则先遍历右子树
     </li>
    </ul>
    <p>
     调用过程
    </p>
    <pre><code>TreeNode&lt;Integer&gt; node1 = new TreeNode&lt;&gt;(1);
TreeNode&lt;Integer&gt; node2 = new TreeNode&lt;&gt;(2);
TreeNode&lt;Integer&gt; node3 = new TreeNode&lt;&gt;(3);
TreeNode&lt;Integer&gt; node4 = new TreeNode&lt;&gt;(4);
TreeNode&lt;Integer&gt; node5 = new TreeNode&lt;&gt;(5);
TreeNode&lt;Integer&gt; node6 = new TreeNode&lt;&gt;(6);
node1.leftChild = node2;
node1.rightChild = node3;
node2.leftChild = node4;
node2.rightChild = node5;
node3.leftChild = node6;

DepthFirstSearch(node1, 6);
</code></pre>
