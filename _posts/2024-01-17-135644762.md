    <h2>
     <a id="1__1">
     </a>
     1. 知识储备
    </h2>
    <p>
     <code>
      HarmonyOS
     </code>
     提供了一套
     <code>
      UI
     </code>
     开发框架，即方舟开发框架（
     <code>
      ArkUI
     </code>
     框架）。方舟开发框架可为开发者提供应用
     <code>
      UI
     </code>
     开发所必需的能力。
    </p>
    <p>
     <code>
      ArkTS
     </code>
     是
     <code>
      HarmonyOS
     </code>
     优选的主力应用开发语言。ArkTS围绕应用开发在
     <code>
      TypeScript
     </code>
     生态基础上做了进一步扩展，继承了
     <code>
      TypeScript
     </code>
     的所有特性，是
     <code>
      TypeScript
     </code>
     的超集。
    </p>
    <p>
     <code>
      TypeScript
     </code>
     是通过在
     <code>
      JavaScript
     </code>
     的基础上添加静态类型定义构建而成。
    </p>
    <p>
     因此，在进行
     <code>
      HarmonyOS
     </code>
     应用开发之前，需要掌握以下三种语言：
    </p>
    <ul>
     <li>
      JavaScript
     </li>
     <li>
      TypeScript
     </li>
     <li>
      ArkTS
     </li>
    </ul>
    <p>
     <strong>
      <font color="red">
       不具备以上基础的开发同学，也不用担心。可以持续关注下我的博客，接下来我会写一系列关于HarmonyOS应用开发的博文，从ArkTS语言基础开始。共同学习、共同探讨、共同进步，熟练的掌握HarmonyOS应用开发的技能。
      </font>
     </strong>
    </p>
    <h2>
     <a id="2__14">

     2. 环境搭建
    </h2>
    <h3>
     <a id="21_nodejs_15">
     </a>
     2.1 安装node.js
    </h3>
    <p>
     node.js 安装包下载地址：
     <a href="https://nodejs.org/en" rel="nofollow">
      node.js官网
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cd2e095f358bf2897514d75ed8766dbc.png#pic_center">
      <br/>
      <font color="red">
       ps：下载LTS版本即可，当前最新版本可能不太稳定。
      </font>
     </img>
    </p>
    <p>
     <strong>
      注意事项：DevEco Studio 支持的node.js版本范围v14.19.1 to v16
     </strong>
    </p>
    <h3>
     <a id="22_nodejs_22">
     </a>
     2.2 配置node.js
    </h3>
    <ul>
     <li>
      安装完
      <code>
       node.js
      </code>
      ，可以查看
      <code>
       node.js
      </code>
      和
      <code>
       npm
      </code>
      的版本。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token comment"># 查询node.js版本号</span>
<span class="token function">node</span> <span class="token parameter variable">-v</span>

<span class="token comment"># 查询npm的版本号</span>
<span class="token function">npm</span> <span class="token parameter variable">-v</span>
</code></pre>
    <ul>
     <li>
      配置npm仓库的地址信息。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">npm</span> config <span class="token builtin class-name">set</span> registry https://repo.huaweicloud.com/repository/npm/
</code></pre>
    <h3>
     <a id="23__38">
     </a>
     2.3 安装命令行工具
    </h3>
    <p>
     该命令行工具集合了
     <code>
      HarmonyOS
     </code>
     应用开发所用到的系列工具，包括
     <code>
      SDK
     </code>
     管理
     <code>
      sdkmgr
     </code>
     、代码检查
     <code>
      codelinter
     </code>
     、三方库的包管理
     <code>
      ohpm
     </code>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a8b18fa03ff37b0d191c24380ed82983.png#pic_center"/>
    </p>
    <ol>
     <li>
      命令行工具安装包下载，下载地址：
      <a href="https://developer.harmonyos.com/cn/develop/deveco-studio#download_cli" rel="nofollow">
       命令行工具下载
      </a>
     </li>
     <li>
      解压文件，进入“ohpm/bin”目录，执行如下指令初始化ohpm。
      <ul>
       <li>
        Windows 系统执行
        <code>
         init.bat
        </code>
       </li>
       <li>
        macOS 系统执行
        <code>
         ./init.sh
        </code>
       </li>
      </ul>
     </li>
     <li>
      ohpm配置到环境变量中。
      <ul>
       <li>
        Windows 环境变量配置：电脑 &gt; 属性 &gt; 高级系统设置 &gt; 高级 &gt; 环境变量中，将ohpm命令行工具的bin目录路径添加到PATH变量中。
       </li>
       <li>
        macOS 环境变量配置：
        <br/>
        <code>
         export OHPM_HOME=ohpm的安装路径
        </code>
        <br/>
        <code>
         export PATH=${OHPM_HOME}/bin:${PATH}
        </code>
       </li>
      </ul>
     </li>
     <li>
      安装完成之后，执行如下命令：
      <code>
       ohpm -v
      </code>
     </li>
    </ol>
    <h3>
     <a id="24_DevEco_Studio_52">
     </a>
     2.4 安装DevEco Studio
    </h3>
    <p>
     HUAWEI DevEco Studio 是
     <code>
      HarmonyOS
     </code>
     应用开发的官方指定工具。
    </p>
    <p>
     HUAWEI DevEco Studio 安装包下载地址：
     <a href="https://developer.harmonyos.com/cn/develop/deveco-studio/#download" rel="nofollow">
      DevEco Studio 下载
     </a>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ad9a592cac11329c84331d08ad706df6.png#pic_center"/>
    </p>
    <h3>
     <a id="25_DevEco_Studio_60">
     </a>
     2.5 配置DevEco Studio
    </h3>
    <ol>
     <li>
      运行已安装的DevEco Studio，选择Do not import settings，单击OK。
     </li>
     <li>
      安装Node.js与ohpm。可以指定本地已安装的Node.js或ohpm（选择Local）。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/589504d7d8da7047f3bc7434efb07b46.png#pic_center">
      3. 在SDK Setup界面，单击按钮，设置HarmonyOS SDK存储路径，单击Next进入下一步。
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/aa38e4d4a9cd9737f7517e08c3369f34.png#pic_center"/>
    </p>
    <ol start="4">
     <li>
      在弹出的SDK下载信息页面，单击Next，并在弹出的License Agreement窗口，阅读License协议，需同意License协议后，单击Next。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a6c458550e0020881b0692f524603a3e.png#pic_center"/>
    </p>
    <ol start="5">
     <li>
      确认设置项的信息，点击Next开始安装。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6dc863a9204b4fe2d1570301f8fa9bc5.png#pic_center"/>
    </p>
