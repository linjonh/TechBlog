    <p>
     背景：各种音视频测试
     <br/>
     影响音视频质量的因素有：
     <br/>
     源视频的采集质量（硬件决定）；
     <br/>
     音视频 SDK 的服务质量（SDK 服务商决定）；
     <br/>
     网络情况；
     <br/>
     实时传输视频质量专项有：（不同网络环境下）
     <br/>
     性能、码率、抗性、时延、音画同步（由 SDK 服务保障并提供技术指标）
     <br/>
     卡顿（流畅度）：手机终端测试
     <br/>
     视频质量（人工）：通过开源算法进行评估
     <br/>
     图片质量和视频质量不是一个概念，评估时不能以图像质量代替视频质量进行评估。
