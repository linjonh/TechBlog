    <h5>
     <a id="PHP___37">
     </a>
     PHP 版本
    </h5>
    <blockquote>
     <p>
      用到的主要技术框架 ：
     </p>
    </blockquote>
    <ul>
     <li>
      uni-app+ thinkphp6
     </li>
    </ul>
    <blockquote>
     <p>
      框架特点 ：
     </p>
    </blockquote>
    <ul>
     <li>
      系统采用前后端分离技术，基于 TP6 + Uni-app框架开发；客户移动端采用uni-app开发，管理后台前端使用iviewUI开发。系统支持微信公众号端、微信小程序端、H5端、PC端多端账号同步
     </li>
    </ul>
    <h5>
     <a id="Java__48">
     </a>
     Java 版本
    </h5>
    <blockquote>
     <p>
      用到的主要技术框架 ：
     </p>
    </blockquote>
    <ul>
     <li>
      SpringBoot + Maven + Swagger + Mybatis Plus + Redis + Uniapp
      <br/>
      +Vue+elementUI
     </li>
    </ul>
    <blockquote>
     <p>
      包含内容如下 ：
     </p>
    </blockquote>
    <ul>
     <li>
      包含移动端、小程序、PC后台、Api接口；有产品、用户、购物车、订单、积分、优惠券、营销、余额、权限、角色、系统设置、组合数据、可拖拉拽的form表单等模块，
      <strong>
       很容易二次开发
      </strong>
     </li>
    </ul>
