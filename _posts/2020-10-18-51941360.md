    <h3 id="环境">
     环境
    </h3>
    <p>
     操作系统：ubuntu 16.04
    </p>
    <h3 id="操作步骤">
     操作步骤
    </h3>
    <ol>
     <li>
      安装pip工具
      <br/>
      终端输入以下命令：
     </li>
    </ol>
    <pre class="prettyprint"><code class="hljs lasso">sudo apt<span class="hljs-attribute">-get</span> install python<span class="hljs-attribute">-pip</span></code></pre>
    <ol>
     <li>
      安装ipython
      <br/>
      终端输入以下命令：
     </li>
    </ol>
    <pre class="prettyprint"><code class="hljs bash"><span class="hljs-built_in">sudo</span> apt-get install ipython</code></pre>
    <ol>
     <li>
      安装ipython[notebook]
      <br/>
      终端输入以下命令：
     </li>
    </ol>
    <pre class="prettyprint"><code class="hljs lasso">sudo apt<span class="hljs-attribute">-get</span> install ipython<span class="hljs-attribute">-notebook</span></code></pre>
    <ol>
     <li>
      启动ipython[notebook]
      <br/>
      终端输入以下命令：
     </li>
    </ol>
    <pre class="prettyprint"><code class="hljs">ipython notebook</code></pre>
    <h3 id="出错">
     出错
    </h3>
    <blockquote>
     <p>
      No module named notebook.notebookapp
     </p>
    </blockquote>
    <p>
     解决办法
    </p>
    <pre class="prettyprint"><code class="hljs cmake">pip <span class="hljs-keyword">install</span> jupyter</code></pre>
