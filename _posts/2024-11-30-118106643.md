    <h2>
     <a id="_0">
     </a>
     摘要
    </h2>
    <p>
     在使用搜索引擎搜索关键词时，搜索首页经常会推荐一些相关或不相关的广告，例如有些人想要了解“职称”，百度一下，发现首页基本上全是广告，如图所示：
     <br/>
     <img alt="百度搜索：职称" src="https://i-blog.csdnimg.cn/blog_migrate/66764acd7c273fa7075f9a580082cfcf.png"/>
    </p>
    <p>
     下面教大家一个方法，该方法可以精简搜索结果，尽可能的去除广告推荐，提高搜索效率。
    </p>
    <h2>
     <a id="_7">
     </a>
     去广告搜索
    </h2>
    <p>
     <strong>
      intitle:关键词
     </strong>
     <br/>
     例如在搜索引擎对话框中填入
     <strong>
      intitle:职称
     </strong>
     ，我们的搜索结果如下图所示：
     <br/>
     <img alt="去除广告搜索" src="https://i-blog.csdnimg.cn/blog_migrate/d68b0bfc935a5c82a70719ef2f8bc322.png">
      <br/>
      虽然也有个别广告推荐，但是搜索结果已经去除掉了大部分的广告。
     </img>
    </p>
    <h2>
     <a id="_13">
     </a>
     限定文件类型
    </h2>
    <p>
     如果我们想下载指定文件格式的文档怎么办？
     <br/>
     <strong>
      关键词 filetype:文件类型
     </strong>
     <br/>
     例如：
     <strong>
      数值分析 filetype:pdf
     </strong>
     ，注意中间加个空格，搜索结果如下图所示：
     <br/>
     <img alt="限定文件类型" src="https://i-blog.csdnimg.cn/blog_migrate/c5378ad0be4b1e5715e1ba34d4c2695f.png"/>
    </p>
    <p>
     常见的文件类型有：
     <strong>
      pdf、xls、ppt、doc、txt
     </strong>
     等。
    </p>
    <h2>
     <a id="_22">
     </a>
     关键词包含在正文中
    </h2>
    <p>
     <strong>
      intext:关键词
     </strong>
     <br/>
     这个关键词和intitle很像，但是区别是intitle的索引在标题中，intext的索引在正文中。
     <br/>
     <strong>
      intext:数值分析
     </strong>
     <br/>
     有兴趣的小伙伴可以实际搜索一下看看二者实际的区别。
     <br/>
     <img alt="关键词包含在正文中" src="https://i-blog.csdnimg.cn/blog_migrate/ca89ba3d5cec5617042880cc47800436.png"/>
    </p>
    <h2>
     <a id="_31">
     </a>
     限定搜索网站
    </h2>
    <p>
     <strong>
      关键词inurl:网站类型
     </strong>
     <br/>
     例如：
     <strong>
      东北大学 inurl:.edu
     </strong>
     ，搜索结果会显示所有的东北大学的官方网站，搜索结果如下：
     <br/>
     <img alt="限定搜索网站" src="https://i-blog.csdnimg.cn/blog_migrate/7c39928261e5eb03eaa36a633ba3b8d8.png">
      <br/>
      除此之外，常见的网站类型还有：
      <br/>
      <strong>
       .com (商业组织和公司)
       <br/>
       .net（网络服务商）
       <br/>
       .gov（政府部门）
       <br/>
       .org（非营利性组织）
       <br/>
       .int（国际组织）
       <br/>
       .edu（教育部门）
      </strong>
     </img>
    </p>
    <h2>
     <a id="_44">
     </a>
     限定搜索时间
    </h2>
    <p>
     <strong>
      关键词 开始时间…结束时间
     </strong>
     <br/>
     例如我想搜索2017-2018年关于职称的相关信息
     <br/>
     在对话框中输入：
     <strong>
      职称 2017…2018
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/35051958cbaa5fc406f13348b64a424e.png"/>
    </p>
    <h2>
     <a id="_51">
     </a>
     混合使用
    </h2>
    <p>
     不同的限定词可以同时混合使用，例如：
     <br/>
     <strong>
      intitle:北京 inurl:.edu
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1177b284c859e5eff6e4e05b0f97f230.png"/>
    </p>
    <h2>
     <a id="_56">
     </a>
     结语
    </h2>
    <p>
     以上方法仅在百度搜索引擎上验证过，并没有尝试在其他搜索引擎上验证过，一般来说不同的引擎会显示不同的搜索结果，所以有兴趣的小伙伴可以多多尝试。
    </p>
