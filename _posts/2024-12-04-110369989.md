     作为一个不新的新手，这里给大家推荐一下我是
     <strong>
      如何入门MATLAB
     </strong>
     的。
    </p>
    <p>
     <strong>
      根据我的个人经验，看项目代码学MATLAB是最有效的学习方式，同样，这也适合所有编程语言。
     </strong>
    </p>
    <h3>
     <a id="_13">
     </a>
     步骤：
    </h3>
    <h6>
     <a id="1MATLAB_15">
     </a>
     1.对于零基础的小白，首先必须了解MATLAB的基本语法及基础知识。
    </h6>
    <p>
     这里重点推荐慕课上刘卫国老师的科学计算与MATLAB语言，课程链接：
     <a href="https://www.icourse163.org/course/CSU-1002475002" rel="nofollow">
      https://www.icourse163.org/course/CSU-1002475002
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69a0e5f6e02f281c694d3a56cece833a.png"/>
    </p>
    <p>
     <strong>
      【注意】我这里说三点：
     </strong>
    </p>
    <ul>
     <li>
      课后习题必须完成，哪怕多花点时间查阅资料或者问其他精通MATLAB的人，这真的有所帮助
     </li>
     <li>
      尽可能又快又准的看完，时间最好不超过半个月，当然也不要低于一个星期，切记不要走马观花
     </li>
     <li>
      关于书籍，这个就因人而异了。如果你觉得不看书不踏实，刘老师推荐的那本书也挺不错，如果你觉得不需要看书，当然不买也是可以的
     </li>
    </ul>
    <h6>
     <a id="2MATLABMATLAB_25">
     </a>
     2.了解了MATLAB基础之后，下面才是精通MATLAB的关键——实战。
    </h6>
    <p>
     <strong>
      修改完成一个项目，这才是学习MATLAB最快且最好的方式
     </strong>
     。对于新手而言，做一个项目或许来说是很难的，那就从看懂项目入手吧。逐行看懂项目的每一行代码的含义，把不懂的语句查明白弄清楚，并进行进一步增删补改，当弄明白一个大项目时，你离精通就已经不远了。
    </p>
    <p>
     以我本人为例，我从MATLAB小白到现在
     <strong>
      略微精通
     </strong>
     ，之间只有一个毕业设计“
     <strong>
      基于MATLAB的手写数字识别系统的设计
     </strong>
     ”，大四上学期，对MATLAB基本一无所知的我，开始了看代码修改代码之行。一行一行看，一行一行查MATLAB官方文档，花费两个多月之后，当我看完并修改完善了所有代码，我感觉MATLAB离我已经不远了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/303d9dd3885627e0bdcf01b66fd247cf.png"/>
    </p>
    <h6>
     <a id="3_31">
     </a>
     3.关于笔记
    </h6>
    <p>
     关于笔记，我的建议是整理，而且必须要整理。不过不一定是记在本子上。我发现onenote也是一件笔记利器，尤其是对于编程语言。
     <br/>
     我将所有用到过且学会的函数都放在一个笔记本里，包括用法，实例等等等等。方便以后查阅，这样可以充分节省时间。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eefcb644f13ff17d46948d3c4de03626.png"/>
    </p>
    <h6>
     <a id="5_35">
     </a>
     5.关于资料
    </h6>
    <p>
     我并不推荐MATLAB书籍，尤其是动辄五六百页的《MATLAB R2018a完全自学一本通》，跟视频一样，一方面你你根本看不完，另一方面，用到的不到十分之一。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9816e5e50d4191341487d1807d62bb39.png">
      <br/>
      我只推荐两个网站：
     </img>
    </p>
    <ol>
     <li>
      <strong>
       MATLAB官方文档
      </strong>
      地址：
      <a href="https://ww2.mathworks.cn/help/pdf_doc/matlab/matlab_refbook_zh_CN.pdf" rel="nofollow">
       https://ww2.mathworks.cn/help/pdf_doc/matlab/matlab_refbook_zh_CN.pdf

      这里，你可以查阅需要的函数功能及实例

     <li>
      <strong>
       MATLAB中文论坛

       https://www.ilovematlab.cn/

    </ol>
    <p>
     除此之外，如果遇到问题，欢迎关注我的博客主页：
     <a href="https://blog.csdn.net/didi_ya/article/details/108247536">
      https://blog.csdn.net/didi_ya/article/details/108247536

     <br/>
     这里可能有MATLAB疑难杂症的解决方案。
    </p>
    <hr/>
    <p>
     ok，当做完这几步之后，你就发现你已经对MATLAB几乎了如指掌了。
    </p>

     <strong>
      【总结】
