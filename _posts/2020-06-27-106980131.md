     二叉搜索树中的搜索
    </h2>
    <hr/>
    <h5>
     <a id="_3">
     </a>
     题目
    </h5>
    <p>
     二叉搜索树中的搜索（力扣：700）
    </p>
    <p>
     给定二叉搜索树（BST）的根节点和一个值。 你需要在BST中找到节点值等于给定值的节点。 返回以该节点为根的子树。 如果节点不存在，则返回 NULL。
    </p>
    <h5>
     <a id="_9">
     </a>
     分析
    </h5>
    <p>
     题目中给出的二叉树是二叉搜索树，所以，我们不用全局遍历二叉树，即可实现搜索。
    </p>
    <p>
     递归法：如果当前节点等于空或等于要找的值，则直接返回；否则，如果当前节点值大于目标值，则从左子树查找，否则从右子树查找。
     <br/>
     遍历法：当前节点等于跟节点，如果当前节点大于目标值，将左子树赋值给当前节点，否则，将右子树赋值给当前节点，如果遍历完成，没找到，则返回null。
    </p>
    <h5>
     <a id="_16">
     </a>
     代码实现：递归
    </h5>
    <pre><code>    /**
     * 700. 二叉搜索树中的搜索
     */
    public TreeNode searchBST(TreeNode root, int val) {
        if(root == null || root.val == val){
            return root;
        }
        return root.val &gt; val ? searchBST(root.left, val) : searchBST(root.right, val);
    }
</code></pre>
    <h5>
     <a id="_30">
     </a>
     代码实现：遍历
    </h5>
    <pre><code>    /**
     * 700. 二叉搜索树中的搜索
     */
    public TreeNode searchBST2(TreeNode root, int val) {
        if(root == null){
            return root;
        }
        while(root != null){
            if(root.val == val){
                return root;
            }else if(root.val &gt;val){
                root = root.left;
            }else{
                root = root.right;
            }
        }
        return null;
    }
</code></pre>
