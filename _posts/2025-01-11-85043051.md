      <span style="font-size:15px;letter-spacing:1px;">
       不知道你们还记得那个背乘法口诀背到崩溃的小女孩吗？隔着屏幕都感觉到了她的绝望。

     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       真的是我们的孩子对数字或者数学没有天赋吗？数学难道只是枯燥的数数和计算吗？数学真的很难学吗？非也。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       事实上，数学本身就源于我们的真实生活，从小培养孩子的数学思维，帮助孩子认识数学概念、养成数学思维，轻轻松松地透过游戏学习数学，逐步建立那些看似“高深”的数学思维与方法，才是王道！
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/dc9082bc659ea820829e9ed613d29873.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       这一套面向学龄前儿童的原创趣味数学绘本——
       <strong>
        《玩儿出来的数学》，共七册。
       </strong>
       涵盖了按
       <strong>
        顺序数数字、数字的加减、简单对比、平面图形、等分概念、学会简单测量和数的简单分解
       </strong>
       这七个知识点。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/9eb0b80fba73b57bb266451bc2059ab0.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       全套书以动物为主角，讲述一个个的小故事，巧妙地设计任务，并运用数学知识成功解决问题。故事最后的数学主题游戏和练习，更是可以帮助孩子巩固知识。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>

      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       以最简单、最快乐、最有趣的方式让孩子走进数学、爱上数学，还能培养他们的逻辑思维能力，教会他们从不同的角度看待问题和解决问题。
      </span>
     </p>

     <p style="clear:both;text-align:center;line-height:1.5em;">
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(171,25,66);">
        （一）
       </span>
      </strong>
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(171,25,66);">
        与我国幼儿心理和语言成长特点契合
       </span>
      </strong>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       这是本套书最大的特点。
      </span>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       面对市面上诸多数学绘本，着实让家长们挑花了眼。加之大部分为引进的绘本，通常较为符合引进国家或是地区的儿童，我们的宝宝在阅读上难免出现“水土不服”的情况。
      </span>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       而本书的创作团队——小白爸爸团队，由晓晔老师、壮子、庞国涛组成，他们均是国内学前教育的专家，深谙儿童语言和思维特点。
      </span>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       无论是故事的语言、深度还是知识点，乃至是每本书后小游戏，都是我国学前阶段的孩子能理解和接受的。
      </span>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/513685025f68724d237a0fe4ef6896d3.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       以《旺旺的晚餐》这本书后的小游戏以及旺旺的一周食谱为例~

     </p>

      <span style="font-size:15px;letter-spacing:1px;">
       无论是语言还是生活，都跟孩子们平时的状态接近；游戏里出现的碟子、筷子这些在生活中常见的餐具，更有助家长在生活中培养孩子的数学思维
      </span>

     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>

     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(171,25,66);">
        （二）
       </span>
      </strong>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(171,25,66);">
        以兴趣带学习，以学习促思维
       </span>
      </strong>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       一提起数学，很容易想到那些让人头痛的数字和公式，仿佛“数学”的脸蛋儿上天生就挂着个“难”字一样，孩子们更觉得数学就是个“大坏蛋”，难免心生畏惧，时间久了就会产生厌烦之感。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/1acda2f51305a59c9ed2881e61fc2a74.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       本书的宗旨是：
      </span>
      <span style="font-size:15px;letter-spacing:1px;color:rgb(171,25,66);">
       通过一个个故事和游戏引起孩子对数学极大的兴趣
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       ，克服畏惧感，再一步步地传达数学知识，在运用知识完成任务的过程中掌握数学思维。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/4e3807485a76d4bdede9fa609da7e18f.jpeg" style="color:rgb(51,51,51);"/>

     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       比如在《三角形的叶子在哪儿》中，蚁后妈妈给能干的小蚂蚁丁丁布置了一个任务——去找各种形状的叶子
      </span>
      <span style="color:rgb(34,34,34);font-family:'微软雅黑', sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;background-color:rgb(255,255,255);">
       。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;min-height:1em;color:rgb(51,51,51);">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/78ee082d2946d4cdc895bb601cfaa441.jpeg"/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       丁丁找到了杨树叶、松树叶、柳树叶……还找到了车前草、马齿苋和蒲公英的叶子……可是，这都不是蚁后妈妈想要的叶子。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       丁丁要到哪里去寻找圆形和三角形的叶子呢？聪明的丁丁又做了什么让蚁后就一下做出来这么多图形呢？
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">

        间接的启发了孩子对数字、形状的认知，

      </span>

     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       小朋友们的生活中总会出现和家人、小伙伴们分享食品、玩具等等的情景，如果有12个面包圈要分给孩子和他的3个小伙伴，改如何去分呢？
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/956fa40468d71420f531bc9e657c765e.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       在《丢了一个面包圈》书中，4只小松鼠们做了12个面包圈，不小心丢了一个，这时候该怎么办呢？
       <strong>
        他们还能平均的分享美食吗？
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;min-height:1em;color:rgb(51,51,51);">
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/6b9d444a3ee7762a33185fe4d7b27871.jpeg"/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       当然可以啦~家长不要提前说，让小朋友们自己尝试去解决这样的问题，切忌家长包办哦~
       <strong>
        只有让孩子自己尝试，才能建立起数学思维的“高楼大厦”。
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <span style="color:rgb(171,25,66);">
       <strong>
        <span style="font-size:15px;letter-spacing:1px;">
         （三）
        </span>
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <span style="color:rgb(171,25,66);">
       <strong>
        <span style="font-size:15px;letter-spacing:1px;">
         走进数学，掌握奥秘
        </span>
       </strong>
      </span>
     </p>
     <p style="clear:both;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       数学就在我们身边，与我们的生活息息相关，而孩子成长的过程更是离不开数学。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       无论是简单的数数、识数、加减乘除、认知几何图形等等都是每个孩子成长中必学的。数学，不知不觉已成为我们的刚需。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       当孩子围着你一直问“我长高了吗？”你会如何作答呢？
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/aef5487bc3e1c879dd84763578d28bde.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       小熊球球是个好奇宝宝，他也总是喜欢问“我长高了吗？我长长了吗？”，熊爸爸送给他一个格尺，球球高兴地量量手、量量脚，小熊球球成为了测量高手。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/d37f00b5baac2919e18dacf9a5f43e0f.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       但是面对测量100米的跑道、乌鸦的家有多高，湖中心的水深、鼹鼠家的地洞长度，球球可该怎么办呢？
      </span>
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/07bfc9e9a6487e0ff6737568a7362347.jpeg" style="color:rgb(51,51,51);"/>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       球球最后真的测量出来了，但真的是那把格尺测量出来的吗？
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       在球球的故事里，
       <strong>
        我们可以引导孩子发现那些隐藏在生活现象背后的数学，
       </strong>
       也展现了生活中常见的与测量有关的几个问题。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       书中的故事
       <strong>
        紧紧贴合我们日常生活中会遇见的问题，很容易引起小朋友的共鸣，潜移默化地影响孩子日后学会用数学作为“工具”来处理问题。
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <span style="color:rgb(171,25,66);">
       <strong>
        <span style="font-size:15px;letter-spacing:1px;">
         （四）
        </span>
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <span style="color:rgb(171,25,66);">
       <strong>
        <span style="font-size:15px;letter-spacing:1px;">
         孩子爱上数学的好帮手
        </span>
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:center;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       英国自然科学、哲学家培根曾说：“数学是科学大门的钥匙，忽视数学必将伤害所有的知识，因为忽视数学的人是无法了解任何其他科学乃至世界上任何其他事物的。更为严重的是，忽视数学的人不能理解他自己这一疏忽，最终将导致无法寻求任何补救的措施。”
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       儿童发展心理学研究认为：所谓数学启蒙，首先是让孩子对数字感兴趣，然后再培养孩子的思维能力和逻辑理解能力。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       其实孩子很小就会对各种数字感兴趣，无论是家里的电话拨号键，还是小区的门牌号，或者是你衣服图案的数量。这些平时很容易被忽略的小细节，
       <strong>
        都是孩子们学习数学的好帮手。
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/8035891a715c71c528102937187e104a.jpeg" style="color:rgb(51,51,51);"/>
       </strong>
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       书中无论是故事导读，还是每篇故事后的游戏还是每本书的音频课堂，
       <strong>
        都是学前教育专家为家长制作的“教育手册”，
       </strong>
       既能让家长带小朋友更深入地掌握本书内容，还能在日常生活中自然、积极地锻炼孩子运用数学思维解决问题。
      </span>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <br/>
     </p>
     <p style="clear:both;text-align:justify;line-height:1.5em;">
      <span style="font-size:15px;letter-spacing:1px;">
       现在就让我们和孩子一起通过趣味的童话故事、生活化的小游戏，边学玩儿，探索丰富的数学知识吧！
