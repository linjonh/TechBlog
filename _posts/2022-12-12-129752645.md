     Python出现No module named “Crypto” 解决方案
    </h1>
   </div>
  </div>
 </div>

  <div class="article_content clearfix" id="article_content">

    <div id="js_content">
     <p>
      之前看过这问题，没引起注意，今天自己用到了，翻车了。整活了半天，记录一下。
     </p>
     <hr/>
     <p>
      有时候需要用到Crypto库，但当用
     </p>
     <pre class="has"><code class="language-nginx">pip install Crypto</code></pre>
     <p>
      安装后仍提示：No module named ‘Crypto’。
     </p>
     <p>
      在PyCharm安装pycryptodome库也依旧不行（pycryptodome是crypto的延伸版本，用法和crypto是一模一样的，可以完全替代crypto）。
     </p>
     <p>
      尝试卸载再安装：
     </p>
     <pre class="has"><code class="language-properties">pip uninstall crypto pycryptodome
pip install pycryptodome</code></pre>
     <p>
      还是一样报错，尝试修改文件夹名称，因为文件夹名是crypto，而报错的是Crypto，首字母大写。
     </p>
     <p>
      找到python 目录下面的\Lib\site-packages，手动将crypto改为Crypto。果然，还是不行。
     </p>
     <p>
      经过一番网上冲浪，找到解决方案如下：
     </p>
     <p>
      1. Win+R打开cmd，按以下顺序输入安装命令，
      <strong>
       无视类似下图的报错！
      </strong>
     </p>
     <pre class="has"><code class="language-properties">pip install pycryptodome
pip install crypto
pip install pycrypto</code></pre>
     <p style="text-align:center;">
      <img alt="564ec0c2da231dc506da8d3199df7f31.png" src="https://i-blog.csdnimg.cn/blog_migrate/c425c3ea2b3c51252bade9625c998ab4.png"/>
     </p>
     <p style="text-align:center;">
      <img alt="89b3e1a2e84643de9eadba01a4b291e9.png" src="https://i-blog.csdnimg.cn/blog_migrate/d0252b8dfb2563618f33903f08ac9ba7.png"/>
     </p>
     <p style="text-align:center;">
      <img alt="c877df5e68acb85ea093397c397ff85f.png" src="https://i-blog.csdnimg.cn/blog_migrate/79024ec4c565cfa814b163abbdc9678d.png"/>
     </p>
     <p>
      2. 找到python 目录下面的\Lib\site-packages，手动将crypto改为Crypto
     </p>
     <p style="text-align:justify;">
      3. 重新运行程序，No module named 'Crypto'报错消失，程序嘎嘎乱跑了。
     </p>
     <p style="text-align:justify;">
      机子配置、py版本等因素很多，什么win7和win10，py3.6和3.9啥的，多少都有点差异，解决方法多样，以上仅供参考
     </p>
     <p style="text-align:justify;">
      反正都是瞎搞，只要代码跑得起来就不要动了
     </p>
