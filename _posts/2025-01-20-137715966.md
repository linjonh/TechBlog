    <h3>
     <a id="2_0">

     2、设置
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a56e2ac9e6be74468385f50edf98099f.png">
      <br/>
      在vscode左下角点击设置按钮，选择“设置”，然后输入“fileheader”,
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac4f2b76b7e705f75d9e726124354fbb.png">
       <br/>
       文件头部注释：Fileheader:custom Made
       <br/>
       函数注释：Fileheader:cursor Mode
       <br/>
       随便点击哪个"在setting.json"中编辑，输入以下设置后保存，然后重启vscode更新设置：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/32759466a85cd9b9bbb1a783f9efa356.png"/>
      </img>
     </img>
    </p>
    <p>
     // 文件头部注释
     <br/>
     “fileheader.customMade”: {
     <!-- -->
     <br/>
     “Descripttion”:“”,
     <br/>
     “version”:“”,
     <br/>
     “Author”:“sueRimn”,
     <br/>
     “Date”:“Do not edit”,
     <br/>

     <br/>

     “msg”:“”,
