     <p>
      <strong>

         10 大顶级开源 ERP 系统
        </a>
       </span>
      </strong>
     </p>
     <div id="news_content">
      企业资源规划（ERP）和客户关系管理（CRM）系统现在已经成为各种组织和企业的必需品，通过它们，可以轻松实现企业的信息数据标准化、系统运行集成化、业务流程合理化、绩效监控动态化、管理改善持续化。
      <br/>
      <br/>
      本文将为你介绍10种顶级的开源ERP软件，它们都可以在网上免费下载到。
      <br/>
      <br/>
      <strong>
       1.  OpenERP
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0615/284340d5-76c2-3214-a2e1-cbb0f94a6810.jpg">
       </img>
      </div>
      <br/>
      <br/>
      当提到开源ERP软件时，OpenERP是无可争议的重量级冠军。OpenERP开发的初衷是为了提供SAP、Microsoft Dynamics等商业ERP软件的开源替代产品。
      <br/>
      <br/>
      OpenERP功能涵盖财务管理、采购/销售管理、库存管理、MRP、CRM、人力资源管理、销售点管理、项目管理等众多方面。
      <br/>
      <br/>
      OpenERP使用Python开发，数据库采用开源的PostgreSQL，它的核心和所有模块都是开放源代码的，采用GNU GPL开源协议。你可以自由使用、修改和发布，只要你也保证开源即可。
      <br/>
      <br/>
      项目地址：
      <a href="https://www.openerp.com/" rel="nofollow">
       https://www.openerp.com/
      </a>
      <br/>
      <br/>
      <strong>
       2.  Openbravo
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0617/815bdc64-44c6-3f80-ac45-6f944a90f2dc.jpg">
       </img>
      </div>
      <br/>
      <br/>
      Openbravo ERP是一套适合于中小企业并且基于web可扩展的ERP系统。这个ERP系统所包括的功能可实现生产管理、仓库管理、销售管理、财务管理。同时内置CRM（客户关系管理）和BI（商业智能）。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.openbravo.com/" rel="nofollow">
       http://www.openbravo.com/
      </a>
      <br/>
      <br/>
      <strong>
       3.  Apache OFBiz
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0619/38ad8cdd-98bb-3309-b6b0-18e917b4abaa.jpg">
       </img>
      </div>
      <br/>
      <br/>
      OFBiz是Apache的顶级开源项目，提供了创建基于最新JavaEE/XML规范和技术标准，构建大中型企业级、跨平台、跨数据库、跨应用服务器的多层、分布式电子商务类Web应用系统的框架。
      <br/>
      <br/>
      OFBiz不仅是一个产品及订单管理系统，它还提供了一整套功能，涵盖企业所需的方方面面。除了管理产品及其相关内容（如电子商店）外，Apache OFBiz还能履行许多其它重要角色，包括客户关系管理、项目进度、计费管理、人力资源管理以及订单管理。
      <br/>
      <br/>
      项目地址：
      <a href="http://ofbiz.apache.org/" rel="nofollow">
       http://ofbiz.apache.org/
      </a>
      <br/>
      <br/>
      <strong>
       4.  Compiere
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0621/94da335a-bd48-3026-9f1d-a1898cbfee6a.jpg">
       </img>
      </div>
      <br/>
      <br/>
      Compiere是全球著名的开源 ERP（集成CRM）企业应用解决方案，适用于全球范围的市场。
      <br/>
      <br/>
      通过Compiere，你只需短短几小时就可以使用申购-采购-发票-付款、报价-订单-发票-收款、产品与定价、资产管理、客户关系、供应商关系、员工关系、财务管理、经营业绩分析等强大功能了。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.compiere.com/" rel="nofollow">
       http://www.compiere.com/
      </a>
      <br/>
      <br/>
      <strong>
       5.  webERP
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0623/9b9c96e0-5257-39b4-b25e-12db1150b860.jpg">
       </img>
      </div>
      <br/>
      <br/>
      WebERP是一套ERP与财务管理软件，它支持多公司、多币种、多税种、多库存管理；权限角色管理便于员工、客户和供应商使用；订单管理支持发票的跟踪与管理；销售费用管理及强大的销售分析功能，为管理者提供决策支持；提供全面的财务特性包括总帐、应收/应付帐目等，且拥有简单易于操作的Web界面。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.weberp.org/" rel="nofollow">
       http://www.weberp.org/
      </a>
      <br/>
      <br/>
      <strong>
       6.  ERP5
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0625/5bca49c9-11d1-3f86-95da-a8c3c58307f6.jpg">
       </img>
      </div>
      <br/>
      <br/>
      ERP5是一个全功能的CRM系统，其中包括客户关系管理、生产管理、供应链管理、产品设计管理、财务管理、人力资源管理、电子商务等多个模块。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.erp5.com/" rel="nofollow">
       http://www.erp5.com/
      </a>
      <br/>
      <br/>
      <strong>
       7.  opentaps
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0627/36552277-0428-3a00-a24e-e433ba994025.jpg">
       </img>
      </div>
      <br/>
      <br/>
      OpenTaps是在ofBiz基础上开发的开源的ERP及CRM企业级应用平台，其包含一个完整的应用程序套件，可与移动、商业集成。它支持客户关系管理、订单管理、存活和仓库、购买和支持链自动化、在线和卖点存储、帐户和财务管理等。
      <br/>
      <br/>
      OpenTaps支持大部分主流关系型数据，包括MySQL、PostgreSQL、Oracle、Sybase与Microsoft SQL Server等。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.opentaps.org/" rel="nofollow">
       http://www.opentaps.org/
      </a>
      <br/>
      <br/>
      <strong>
       8.  Dolibarr
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0629/398ddad8-b20e-3e1f-b308-f53a09eecb57.jpg"/>
      </div>
      <br/>
      <br/>
      Dolibarr 是一个针对中小型企业、组织或自由职业者的开源ERP/CRM软件。它包括了企业资源计划（ERP）和客户关系管理（CRM）以及可应用于其它不同活动的功能。
      <br/>
      <br/>
      项目地址：
      <a href="http://www.dolibarr.org/" rel="nofollow">
       http://www.dolibarr.org/
      </a>
      <br/>
      <br/>
      <strong>
       9.  ADempiere
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0631/5558449f-08af-3958-88d8-dcdd43e6ddf7.jpg"/>
      </div>
      <br/>
      <br/>
      ADempiere是一个由开源社区所领导的ERP 项目。该项目始于2006年，是由Compiere ERP &amp; CRM分支出来的。由于Compiere是由公司为主导，虽然开放源码，但是在语言本地化以及文件数据都需要透过商业服务付费取得。而 ADempiere 的整个项目（包括源码、说明文件）都以开放的方式发布。
      <br/>
      <br/>
      项目地址：
      <a href="http://adempiere.org/" rel="nofollow">
       http://adempiere.org/
      </a>
      <br/>
      <br/>
      <strong>
       10.  PostBooks
      </strong>
      <br/>
      <br/>
      <div style="text-align:center;">
       <img alt="" src="http://dl.iteye.com/upload/attachment/0083/0633/ea1df5fd-9f15-3f2d-9a82-1469089e2972.jpg"/>
      </div>
      <br/>
      <br/>
      PostBooks是xTuple公司推出的一套开源ERP软件，适用于中小型企业，可视化的客户端可运行于Linux、MAC和Windows上（基于Qt组件库），使用的是PostgreSQL数据库，支持国际化。
      <br/>
      <br/>
      PostBooks包含了会计、销售、CRM、购买、产品定义、清单、OpenRPT等模块。
      <br/>
      <br/>
      项目地址：
      <a href="https://www.xtuple.com/postbooks" rel="nofollow">
       https://www.xtuple.com/postbooks
      </a>
