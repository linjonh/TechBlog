     <h3>
      一、Python
     </h3>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/8dc64593860df5f37da629d3afd2783d.png"/>
     </p>
     <p>
      <code>
       Python
      </code>
      这个插件必装，别问为啥！因为它是微软
      <code>
       VSCode
      </code>
      开发团队自己开发的，亲儿子的级别。虽然 VSCode 不安装任何插件也能高亮Python代码，但该插件提供的功能远不止如此，还有很多强大的功能，比如：
      <br/>
     </p>
     <ul>
      <li>
       <p>
        智能补全：通过自动补全，代码导航，语法检查等来编辑代码
       </p>
      </li>
      <li>
       <p>
        代码检查：使用
        <code>
         Pylint
        </code>
        、
        <code>
         Flake8
        </code>
        获取额外的代码分析，检查编程错误
       </p>
      </li>
      <li>
       <p>
        代码格式：用
        <code>
         black
        </code>
        ，
        <code>
         autopep
        </code>
        或
        <code>
         YAPF
        </code>
        调整代码格式
       </p>
      </li>
      <li>
       <p>
        调试：调试Python脚本，Web应用程序以及远程或多线程进程
       </p>
      </li>
      <li>
       <p>
        测试：通过测试Explorer中
        <code>
         unittest
        </code>
        ，
        <code>
         pytest
        </code>
        或
        <code>
         nose
        </code>
        运行和调试测试
       </p>
      </li>
      <li>
       <p>
        Jupyter Notebook：创建和编辑Jupyter Notebook，添加和运行代码单元、渲染图、通过变量资源管理器可视化变量、使用数据查看器可视化数据框等等
       </p>
      </li>
      <li>
       <p>
        环境：自动激活并在
        <code>
         virtualenv
        </code>
        、
        <code>
         venv
        </code>
        、
        <code>
         pipenv
        </code>
        、
        <code>
         conda
        </code>
        和
        <code>
         pyenv
        </code>
        环境之间进行切换
       </p>
      </li>
      <li>
       <p>
        重构：通过变量提取、方法提取和导入排序来重构Python代码
       </p>
      </li>
     </ul>
     <p>
      如下所示：
     </p>

     </p>
     <p>
      文档链接：https://marketplace.visualstudio.com/items?itemName=ms-python.python

     <h3>
      二、Python Snippets
     </h3>
     <img src="https://i-blog.csdnimg.cn/blog_migrate/de37cb2670fcad24a294de4844c25e35.png">
      <p>
       <code>
        Python Snippets
       </code>
       插件可以让我们的 Python 编程更加高效。它包含了大量的内置方法，以及
       <code>
        string
       </code>
       、
       <code>
        list
       </code>
       、
       <code>
        sets
       </code>
       、
       <code>
        tuple
       </code>
       、
       <code>
        dictionary
       </code>
       、
       <code>
        class
       </code>
       代码片段，并且还为每个代码段提供至少一个示例。
      </p>
      <ul>
       <li>
        <p>
         所有python内置方法代码片段
        </p>
       </li>
       <li>
        <p>
         所有python string片段
        </p>
       </li>
       <li>
        <p>
         所有python list片段
        </p>
       </li>
       <li>
        <p>
         所有python set片段
        </p>
       </li>
       <li>
        <p>
         所有python tuple片段
        </p>
       </li>
       <li>
        <p>
         所有python dictionary 字典片段
        </p>
       </li>
       <li>
        <p>
         并包含许多其他代码段（例如if/else、for、while、while/else、try/catch，文件处理和类片段和oop类示例（多态性、封装、继承.i.g）
        </p>
       </li>
      </ul>
      <p>
       如下所示：
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/e3e693ce5df4b05827373f42338249ec.gif"/>
      </p>
      <p>
       文档链接：https://marketplace.visualstudio.com/items?itemName=frhtylcn.pythonsnippets
      </p>
      <h3>
       三、Python Docstring Generator
      </h3>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/3c8ec026470421647a27dcecf5635b12.png"/>
      </p>
      <p>
       <code>
        Python Docstring Generator
       </code>
       可以自动创建docstring，这真的为开发人员减少了的很大工作量。并且，生成的文档字符串遵循所有标准格式，包括
       <code>
        Google
       </code>
       、
       <code>
        docBlockr
       </code>
       、
       <code>
        Numpy
       </code>
       、
       <code>
        Sphinx
       </code>
       和
       <code>
        PEP0257
       </code>
       。它的主要功能包括：
       <br/>
      </p>
      <ul>
       <li>
        <p>
         快速生成一个可通过Tab键浏览的docstring代码段
        </p>
       </li>
       <li>
        <p>
         在几种不同类型的文档字符串格式之间进行选择
        </p>
       </li>
       <li>
        <p>
         通过pep484类型提示、默认值和var名称推断参数类型
        </p>
       </li>
       <li>
        <p>
         支持args，kwargs，修饰符，错误和参数类型
        </p>
       </li>
      </ul>
      <p>
       下面是实例：
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/3a650b9d453ab7b99ad29016befd12d0.gif"/>
      </p>
      <p>
       具体使用方法、扩展配置、自定义模板可参考文档：
      </p>
      <p>
       文档链接：https://marketplace.visualstudio.com/items?itemName=njpwerner.autodocstring
      </p>
      <h3>
       四、Python Test Explorer for Visual Studio Code
      </h3>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/703ac8888ab3d05952f1a1e3556232f4.png"/>
      </p>
      <p>
       这个插件通过使用
       <code>
        Test Explorer UI
       </code>
       运行
       <code>
        Python Unittest
       </code>
       或
       <code>
        Pytest
       </code>
       测试，对于功能测试非常方便。具体功能如下：
       <br/>
      </p>
      <ul>
       <li>
        <p>
         在VS Code的侧栏中的“测试”视图中显示“测试资源管理器”，其中包含所有检测到的测试、套件及其状态
        </p>
       </li>
       <li>
        <p>
         在测试发现期间方便的错误报告
        </p>
       </li>
       <li>
        <p>
         单元测试和Pytest调试
        </p>
       </li>
       <li>
        <p>
         在资源管理器中选择测试后，显示失败的测试日志
        </p>
       </li>
       <li>
        <p>
         支持多根工作区
        </p>
       </li>
       <li>
        <p>
         支持Unittest和Pytest测试框架及其插件
        </p>
       </li>
      </ul>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/461fd979cf5c4ed107f6af5f89e6088d.png"/>
      </p>
      <p>
       文档链接：https://marketplace.visualstudio.com/items?itemName=LittleFoxTeam.vscode-python-test-adapter
      </p>
      <h3>
       五、Python Preview
      </h3>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/2f96874d3c45161ff9d189d186f67d91.png"/>
      </p>
      <p>
       <code>
        Python Preview
       </code>
       可展现可视化调试的过程，并添加到我们的Python代码中。它将调试代码转换为带有动画和图形元素的交互式会话，以表示应用程序状态，对于代码调试非常直观。
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/e03b9bbcb7a12a6e30f2cd3403278b69.gif"/>
      </p>
      <p>
       文档链接：https://marketplace.visualstudio.com/items?itemName=dongli.python-preview
      </p>
      <h3>
       六、Python Type Hint
      </h3>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/017ab73ae303685b9749a2155879e091.png"/>
      </p>
      <p>
       <code>
        Python Type Hint
       </code>
       提供针对Python的类型提示自动完成功能，以及针对内置类型、类和键入模块的完成项。
       <br/>
      </p>
      <ul>
       <li>
        <p>
         提供内置类型、估计类型和键入模块的类型提示完成项
        </p>
       </li>
       <li>
        <p>
         估计正确的类型，提供完成项
        </p>
       </li>
       <li>
        <p>
         可以在工作区中搜索Python文件以进行类型估计
        </p>
       </li>
      </ul>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/4663cbd1eb606892bff47ec876368074.gif"/>
      </p>
      <p>
       文档链接：https://marketplace.visualstudio.com/items?itemName=njqdev.vscode-python-typehint
       <br/>
      </p>
      <h3>
       七、Jupyter
      </h3>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/e49cd675292536e42291c58957820174.png"/>
      </p>
      <p>
       <code>
        Jupyter
       </code>
       可以说是我最喜欢的
       <code>
        VS Code
       </code>
       插件之一，可以让我们在VS Code中完美使用Jupyter Notebooks。使用方法如下：
       <br/>
      </p>
      <p>
       1、创建新笔记本，打开命令面板（
       <code>
        Windows：Ctrl + Shift + P
       </code>
       ；
       <code>
        iOS：Command + Shift + P
       </code>
       ），然后选择命令
       <code>
        Jupyter: Create New Blank Jupyter Notebook
       </code>
       。
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/7b058bff8db309c142f4b697114fe033.png"/>
      </p>
      <p>
       2、通过单击状态栏右下方的内核选择器或调用
       <code>
        Notebook: Select Notebook Kernel
       </code>
       命令来选择内核。
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/fc62ef65298e63f1092c09c9b27c73b3.gif"/>
      </p>
      <p>
       3、通过单击语言选择器或调用
       <code>
        Notebook: Change Cell Language
       </code>
       命令来更改单元格语言。
      </p>
      <p style="text-align: center">
       <img src="https://i-blog.csdnimg.cn/blog_migrate/5fdd45fb00b6be22df4b1a89f5cd4457.gif"/>
      </p>
      <p>
       另外，
       <code>
        Jupyter
       </code>
       插件还支持VS Code Insiders中的其他语言，例如
       <code>
        Julia
       </code>
       和
       <code>
        R
       </code>
       。
      </p>
      <p>
       文档：https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter
      </p>
