    <h4>
     1、什么是数据库？
    </h4>
    <p style="text-indent:50px;">
     数据库简称 DB，是按照数据结构来组织、存储和管理数据的仓库，用户可以对文件的数据进行增加、删除、修改、查找等操作。
    </p>
    <h4 style="text-indent:0px;">
     2、常见的数据库产品
    </h4>
    <p style="text-indent:50px;">
     ①
     <strong>
      Oracle
     </strong>
     ：oracle公司推出的数据库系统。
     <strong>
      优势：移植性好、使用方便、功能性强、适用于大、中、小微机环境。
     </strong>
    </p>
    <p style="text-indent:50px;">
     ②
     <strong>
      SQL Server
     </strong>
     ： Microsoft 公司推出的数据库系统，广泛应用于电子商务、银行、保险、电力等行业。
     <strong>
      缺点：只能在 Windows 平台上运行。
     </strong>
    </p>
    <p style="text-indent:50px;">
     ③
     <strong>
      MySQL
     </strong>
     ：开放源码数据库。
     <strong>
      优点：开源、快捷、跨平台性、方便、免费等特点。
     </strong>
    </p>
    <h4 style="text-indent:0px;">
     <strong>
      ３、ＤB存储结构
     </strong>
    </h4>
    <p style="text-indent:50px;">
     <strong>
      <img alt="" class="has" height="358" src="https://i-blog.csdnimg.cn/blog_migrate/627f110e54d185ca2509e4e6f49c76b3.png" width="420"/>
     </strong>
    </p>
    <h4 style="text-indent:0px;">
     4、表结构
    </h4>
    <p style="text-indent:50px;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/f39bce2888bf4252bc2340f8bc3a494b.png"/>
    </p>
    <p style="text-indent:50px;">
     数据表的横向被称为“行”，纵向被称为“列”，行列交叉处的数据被称为“值”。数据表中的的每一行内容被称为“记录”，每一列的列名称被称为“字段”。
    </p>
    <h4 style="text-indent:0px;">
     5、MySQL数据库
    </h4>
    <ul>
     <li style="text-indent:50px;">
      开源的关系型数据库管理系统
     </li>
     <li style="text-indent:50px;">
      由瑞典 MySQL AB公司开发，先后被Sun 和 Oracle 公司收购
     </li>
     <li style="text-indent:50px;">
      在web开发领域，MySQL依然占据着举足轻重的地位
     </li>
    </ul>
    <h4 style="text-indent:0px;">
     6、MySQL数据库优点：
    </h4>
    <p style="text-indent:50px;">
     ①
     <strong>
      低成本
     </strong>
    </p>
    <p style="text-indent:50px;">
     MySQL是开源免费的，开发人员自由修改，降低了开发成本。
    </p>
    <p style="text-indent:50px;">
     ②  跨平台
    </p>
    <p style="text-indent:50px;">
     不仅可在 Windows 平台上使用，还可以在 Linux、Macos 等14个平台使用。
    </p>
    <p style="text-indent:50px;">
     ③  高性能
    </p>
    <p style="text-indent:50px;">
     多线程以及sql算法的设计，使其可以充分使用 CPU 资源。
    </p>
    <p style="text-indent:50px;">
     ④上手快
    </p>
    <p style="text-indent:50px;">
     MySQL 使用标准的SQL数据语言形式，方便用户操作。
    </p>
    <p style="text-indent:50px;">
     ⑤API 接口
    </p>
    <p style="text-indent:50px;">
     提供多种编程语言的 API ，方便操作数据库。例如 Java、C、C++、php等。
    </p>
