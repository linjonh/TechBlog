    Exception in thread "main" java.lang.UnsupportedClassVersionError:
    <br/>
    com/desktop/DesktopServerLauncher : Unsupported major.minor version 51.0
    <br/>
    出现该问题，说明JAVA库的编译版本和当前的运行环境版本不一致！运行环境的版本太低了！
    <p>
     JAVA官方下载地址：
     <a href="http://www.oracle.com/technetwork/java/javase/downloads/index.html" rel="nofollow noopener noreferrer" target="_blank">
      http://www.oracle.com/technetwork/java/javase/downloads/index.html
     </a>
     <br/>
     下载并安装最新版本即可！下载时，可能需要用户名和密码（
     <span style="color:rgb(54,46,43); line-height:20px">
      gaojs#foxmail.com；Gao#shan）
     </span>
    </p>
    <p>
     <span style="color:rgb(54,46,43); line-height:20px">
      <br/>
     </span>
    </p>
    <p>
     <span style="color:rgb(54,46,43); line-height:20px">
      编译时，JAVA版本不一致（我用JDK7按JavaSE-1.6
      <span style="color:rgb(54,46,43); line-height:20px">
       编译，见下图
      </span>
      ），Eclipse可能会报如下的警告：
      <br/>
      Build path specifies execution environment JavaSE-1.6.
      <br/>
      There are no JREs installed in the workspace that are strictly compatible with this environment.
      <br/>
      想去除警告，只需安装一个JDK1.6的环境即可；下载地址，可以从上述地址页面中选择“
     </span>
     <a href="http://www.oracle.com/technetwork/java/javase/archive-139210.html" rel="nofollow noopener noreferrer" style="" target="_blank" title="Previous Releases">
      Previous Releases
     </a>
     <span style="color:rgb(54,46,43); line-height:20px">
      ”
      <br/>
      <img alt="" src="https://img-blog.csdn.net/20140227095956125?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvZ2Fvamluc2hhbg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
       如上面在工程的属性页面中，选【Java Build Path】-【Library】-【JRE System Library】-【Edit】；
       <br/>
       然后，选择【Install JREs...】安装JDK1.6后，再选中【Alternate JRE】，即可使用该版本的系统库。
      </img>
     </span>
    </p>
