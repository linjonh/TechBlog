     </span>
     <span style="color:#000000;">
      ,
     </span>
     <span style="color:#000000;">
      使用它来验证
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">

    <p style="margin-left:0pt;">
     <em>
      <span style="color:#365f91;">
       <em>
        2.1.

     </em>
     <em>
      <span style="color:#365f91;">

     </em>
     <em>
      <span style="color:#365f91;">
       <em>
        cts
       </em>
      </span>
     </em>
     <em>
      <span style="color:#365f91;">
       <em>
        测试
       </em>
      </span>
     </em>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      连接至少一台设备，打开
     </span>
     <span style="color:#000000;">
      cts
     </span>
     <span style="color:#000000;">
      控制台，在解压好的
     </span>
     <span style="color:#000000;">
      cts
     </span>
     <span style="color:#000000;">
      目录中，执行如下命令
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#006600;">
      $ ./android-cts/tools/cts-tradefed
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      开始执行
     </span>
     <span style="color:#000000;">
      cts
     </span>
     <span style="color:#000000;">
      默认测试项
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#006600;">
      run cts --plan CTS
     </span>
    </p>

     <span style="color:#000000;">
      对于
     </span>
     <span style="color:#000000;">
      android6.0
     </span>
     <span style="color:#000000;">
      或者更早的版本，使用的是
     </span>
     <span style="color:#000000;">
      cts v1 console.
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      对于
     </span>
     <span style="color:#000000;">
      android7.0
     </span>
     <span style="color:#000000;">
      或者最新的版本，使用的是
     </span>
     <span style="color:#000000;">
      cts v2 console.
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      cts v1
     </span>
     <span style="color:#000000;">
      与
     </span>
     <span style="color:#000000;">
      cts v2
     </span>
     <span style="color:#000000;">
      在执行
     </span>
     <span style="color:#000000;">
      cts
     </span>
     <span style="color:#000000;">
      测试时所使用的命令以及执行内容有所不同
     </span>
    </p>

      <span style="color:#365f91;">

      跑一次完整
     </span>
     <span style="color:#000000;">
      CTS

     <span style="color:#000000;">
      ，所有测试项都需要兼容
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      可以通过执行
     </span>
     <span style="color:#000000;">
      l/list p/plans
     </span>
     <span style="color:#000000;">
      列出当前所有可执行的
     </span>
     <span style="color:#000000;">
      plan
     </span>
    </p>

     <span style="color:#000000;">

     </span>

     <span style="color:#000000;">
      可以通过执行
     </span>
     <span style="color:#000000;">
      l/list packages
     </span>
     <span style="color:#000000;">
      列出当前所有可执行的
     </span>
     <span style="color:#000000;">
      packages
     </span>
    </p>

     <span style="color:#000000;">
