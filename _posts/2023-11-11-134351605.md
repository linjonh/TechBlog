     前言
    </h4>
    <p>
     大家好，我是bigsai，很早就在写博客，我将csdn的文章整理成了一个pdf，并且开源到github上！
    </p>
    <p>
     自己写东西断断续续也不少时间了，也写了不少东西(虽然是偏向小白)，这个其实花费的时间还是比较多的，这次的话主要将数据结构与算法中一些文章整理出来，初步整理成一版pdf，先分享给大家。
    </p>
    <p>
     因为在整理pdf方面没啥经验，目前还是md直接导出的pdf的，看了下有些部分代码太长太占页面，有些部分图片太长也很占地方，有部分文章还尝试不同风格显得不那么条理，有些地方页面突然分割也很影响视觉体验……
    </p>
    <p>
     虽然有很多缺点和问题，这些地方后面也在想办法考虑一点点优化，也欢迎大伙提出宝贵的意见！当然这个仓库也是开源的，有兴趣一起维护的可以维护。
    </p>
    <p>
     更新会同步到Github仓库中，也会告知大家。近期会将以前写的其他数据结构算法部分(未优化)优化更新上去。

       </li>
       <li>
        <strong>
         写给小白硬核递归
        </strong>
        ：通过解决经典问题，深入理解递归，掌握记忆化递归的妙用。

      <p>

       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         图解线性表
        </strong>
        ：手写顺序表和链表，理解带头结点和不带头结点链表的实现区别。
       </li>
       <li>
        <strong>
         图解双链表
        </strong>
        ：手写双链表，考虑好节点直接联系，特别是考研重点。

      </ul>
     </li>
     <li>
      <p>
       <strong>
        栈与队列的熟练使用
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         栈
        </strong>
        ：掌握后进先出规则，实现上用顺序表和链表考虑区别。
       </li>
       <li>
        <strong>
         队列
        </strong>
        ：理解先进先出规则，实现上使用循环数组和链表分别实现。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        更高级的线性结构
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         跳表
        </strong>
        ：了解跳表的增删改查原理和流程，拓宽对链表的认识。
       </li>
       <li>
        <strong>
         约瑟夫环问题
        </strong>
        ：通过链表法、List模拟、公式法逐步深入这个问题。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        树的深入学习
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         二叉树层序遍历
        </strong>
        ：学会使用队列进行层序遍历，解决二叉树之字形遍历。
       </li>
       <li>
        <strong>
         二叉树前中后遍历
        </strong>
        ：熟练掌握非递归方式遍历，建立对树结构的全面理解。
       </li>
       <li>
        <strong>
         二叉搜索(查找)树
        </strong>
        ：学会查找方式和删除节点逻辑。
       </li>
       <li>
        <strong>
         二叉平衡(AVL)树
        </strong>
        ：了解平衡二叉树的旋转方式，以及不同平衡方式的应用。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        其他重要数据结构
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         哈夫曼树
        </strong>
        ：理解哈夫曼树的weight计算方式和哈夫曼编码。
       </li>
       <li>
        <strong>
         字典(Trie)树
        </strong>
        ：手写字典树，熟悉其应用场景和优势。
       </li>
       <li>
        <strong>
         优先队列
        </strong>
        ：了解借助堆的运行原理。
       </li>
       <li>
        <strong>
         并查集(不相交集合)
        </strong>
        ：手写并查集，理解路径压缩的优势。

      </ul>
     </li>
     <li>
      <p>
       <strong>
        问题解决方法的学习
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         回溯算法
        </strong>
        ：通过解经典问题如八皇后问题，深刻理解回溯思想。
       </li>
       <li>
        <strong>
         分治算法
        </strong>
        ：学会先分后合并，解决经典问题如二分搜索、快排、归并排序、最近点对等。
       </li>
       <li>
        <strong>
         搜索算法
        </strong>
        ：深度优先搜索、广度优先搜索，解决问题如蓝桥杯，提高解决实际问题的能力。
       </li>
       <li>
        <strong>
         拓扑排序
        </strong>
        ：理解图论算法，得到一个前后的顺序序列。
       </li>
       <li>
        <strong>
         Dijkstra算法
        </strong>
        ：了解图论单源最短路径，队列+贪心实现。
       </li>
       <li>
        <strong>
         Floyd算法
        </strong>
        ：了解图论多源最短路径，代码简短但需要深刻理解。
       </li>
       <li>
        <strong>
         最小生成树算法
        </strong>
        ：Prim和Kruskal，了解不同贪心策略的应用。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        位运算与数论算法
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         位运算
        </strong>
        ：介绍几种位运算，解决大部分经典位运算问题。
       </li>
       <li>
        <strong>
         求素数
        </strong>
        ：掌握素数筛和欧拉筛的应用。
       </li>
       <li>
        <strong>
         快速幂
        </strong>
        ：数论算法，掌握非递归、递归写法，了解矩阵快速幂。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e09681d81d56f3213e862d2ccc680a45.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        大数运算
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         大数加减乘除
        </strong>
        ：手写大数运算，了解大数除法的思想。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        十大排序算法
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         冒泡排序、快速排序
        </strong>
        ：了解交换类排序的原理和应用。
       </li>
       <li>
        <strong>
         插入排序、希尔排序
        </strong>
        ：理解插入类排序的方法和优势。
       </li>
       <li>
        <strong>
         归并排序
        </strong>
        ：学会归并类排序的实现。
       </li>
       <li>
        <strong>

        </strong>
        ：掌握选择类排序的原理。

       <li>
        <strong>
         桶排序、计数排序、基数排序

        ：了解桶类排序的应用场景和实现。

        <strong>
         双轴快排
