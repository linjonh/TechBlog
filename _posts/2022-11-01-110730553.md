    <p>
     第一步，到要执行的python程序根目录
     <br/>
     <img alt="Linux上演示代码" src="https://i-blog.csdnimg.cn/blog_migrate/30c8c3c9d94a65227191e163acba225c.png">
      <br/>
      第二步，后台执行python程序，并将结果重定向到某个结果文件中，格式如下：
     </img>
    </p>
    <pre><code class="prism language-shell"><span class="token function">nohup</span> python -u XXX.py <span class="token operator">&gt;&gt;</span> result.txt 2<span class="token operator">&gt;</span><span class="token operator">&amp;</span>1 <span class="token operator">&amp;</span>
</code></pre>
    <p>
     解释
    </p>
    <table>
     <thead>
      <tr>
       <th>
        code
       </th>
       <th>
        description
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         &gt;&gt;
        </code>
       </td>
       <td>
        为重定向追加输出log到现有result.txt文件；如果写&gt;重定向的话，则是清空原文件result.txt，再输入
       </td>
      </tr>
      <tr>
       <td>
        <code>
         python -u
        </code>
       </td>
       <td>
        这个
        <strong>
         非常重要
        </strong>
        , 由于python有缓冲机制，print不一定会立刻输出到文件！！！
        <br/>
        加了-u可以让stdout等强制无缓冲, 立刻输出到文件！
       </td>
      </tr>
      <tr>
       <td>
        <code>
         2&gt;&amp;1
        </code>
       </td>
       <td>
        将错误重定向到标准输出中(意思是将不同等级的输出都显示在一个文件里)
       </td>
      </tr>
      <tr>
       <td>
        最后一个
        <code>
         &amp;
        </code>
       </td>
       <td>
        表明此python在后台运行
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     对上面的
     <code>
      数字表示的含义
     </code>
     补充解释
    </p>
    <ul>
     <li>
      0：代表 – stdin (standard input)标准输入
     </li>
     <li>
      1： 代表– stdout (standard output)标准输出
     </li>
     <li>
      2：代表– stderr (standard error) 标准错误
     </li>
     <li>
      2&gt;&amp;1: 是将标准错误（2）重定向到标准输出（&amp;1），标准输出（&amp;1）再被重定向输入到result.txt文件中。
     </li>
    </ul>
    <p>
     演示如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a347ed40b1a61d8f329a54eabb6cdc7d.png">
      <br/>
      其中
      <code>
       [1] 2294
      </code>
      表示当前后台程序的pid号。
     </img>
    </p>
    <h5>
     <a id="_34">
     </a>
     其他
    </h5>
    <ol>
     <li>
      <p>
       想要关闭后台执行的这个程序，如下：
      </p>
      <pre><code class="prism language-shell"><span class="token function">kill</span> 2294    <span class="token comment"># kill + pid 即可</span>
</code></pre>
     </li>
     <li>
      <p>
       想要查看输出的结果，在同一目录下：
      </p>
      <pre><code class="prism language-shell"><span class="token function">cat</span> result.txt
</code></pre>
     </li>
    </ol>
