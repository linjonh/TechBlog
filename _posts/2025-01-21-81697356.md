    <h3 id="前言">
     前言
    </h3>
    <p>
     Python 中的文件数据存储和读取可以说是非常方便了，这里记录一下 JSon 数据的存储和读取，需要用到的模块就是 json，该模块能够将简单的 Python 数据结构转储到文件中，并在程序再次运行时加载该文件中的数据，还可以使用 json 在 Python 程序之间分享数据，更重要的是，JSon 数据格式并非 Python 专用的，能够将 JSon 格式存储的数据与其他编程语言共享。
    </p>
    <h3 id="正文">
     正文
    </h3>
    <h4 id="使用jsondump写入数据">
     使用json.dump()写入数据
    </h4>
    <p>
     函数 dump()接受两个参数，要储存的数据以及可用于存储数据的文件对象。
     <br/>
     下面演示如何使用json.dump()来存储数字列表：
     <br/>
     新建一个 number_writer.py
    </p>
    <pre class="prettyprint"><code class="hljs python"><span class="hljs-keyword">import</span> json
numbers = [<span class="hljs-number">1</span>,<span class="hljs-number">2</span>,<span class="hljs-number">3</span>,<span class="hljs-number">4</span>,<span class="hljs-number">5</span>,<span class="hljs-number">6</span>]
filename = <span class="hljs-string">'number.json'</span>
<span class="hljs-keyword">with</span> open(filename,<span class="hljs-string">'w'</span>) <span class="hljs-keyword">as</span> f_obj:
    json.dump(numbers,f_obj)</code></pre>
    <p>
     运行，在本地会生成一个number.json文件，打开后可以看到其内容，数据的存储格式与 Python 中一样：
     <br/>
     [1,2,3,4,5,6]
    </p>
    <h4 id="使用jsonload-读取数据">
     使用json.load() 读取数据
    </h4>
    <p>
     下面再写一个程序，使用 json.load()将这个列表读取到内存中：
     <br/>
     新建一个 number_read.py文件
    </p>
    <pre class="prettyprint"><code class="hljs python"><span class="hljs-keyword">import</span> json

filename=<span class="hljs-string">'number.json'</span>
<span class="hljs-keyword">with</span> open(filename) <span class="hljs-keyword">as</span> f_obj:
    numbers = json.load(f_obj)

print(<span class="hljs-string">"numbers = "</span>,numbers)</code></pre>
    <p>
     运行输出：
    </p>
    <pre class="prettyprint"><code class="hljs fix"><span class="hljs-attribute">numbers </span>=<span class="hljs-string">  [1, 2, 3, 4, 5, 6]</span></code></pre>
    <p>
     这是一种在程序之间共享数据的简单方式。
    </p>
