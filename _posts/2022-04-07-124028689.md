    <h3>
     <a id="Anaconda_9">
     </a>
     一、Anaconda下载（官网和清华源）
    </h3>
    <h4>
     <a id="11Anaconda_10">
     </a>
     1.1、Anaconda官网首页地址
    </h4>
    <p>
     <a href="https://www.anaconda.com/" rel="nofollow">
      Anaconda官网
     </a>
    </p>
    <p>
     进入官网后如下图所示，点击
     <strong>
      Download
     </strong>
     即可开始下载（若无法下载，请转至清华源下载）
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5bc38d93c7b1b71be62aef804e38b796.png"/>
    </p>
    <hr/>
    <h4>
     <a id="12Anaconda_17">
     </a>
     1.2、清华源Anaconda地址
    </h4>
    <p>
     <a href="https://repo.anaconda.com/archive/" rel="nofollow">
      清华源
     </a>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ec5975bd11b11d4207b8889067a2be5c.png"/>
    </p>
    <hr/>
    <p>
     下载完成后将会得到exe文件，双击即可开始安装（一般下载完成后会自动打开安装界面无需点击exe文件，若没有自动打开安装页面再点击此exe文件）
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/90cc903790acaa4911e0182281fa808f.png"/>
    </p>
    <hr/>
    <h3>
     <a id="Anaconda_32">
     </a>
     二、Anaconda安装
    </h3>
    <p>
     点击
     <strong>
      Next
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/378df61cd0562719264fb809df84abb1.png"/>
    </p>
    <hr/>
    <p>
     点击
     <strong>
      I Agree
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e7c36fb3822f49337fe76c6bcc6e5b35.png"/>
    </p>
    <hr/>
    <blockquote>
     <p>
      根据评论区众多大佬的指点与教导
     </p>
    </blockquote>
    <p>
     在此选择
     <strong>
      Just Me(recommended)
     </strong>
     ----&gt;点击
     <strong>
      Next
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bfeb0026ace8c448f7e207c265ee8e24.png"/>
    </p>
    <hr/>
    <p>
     在此笔者将Anaconda装入D盘（
     <mark>
      注意安装路径不要包含汉字并且不要包含空格
     </mark>
     ）尽量不要装入系统盘，系统盘的空间资源太宝贵了
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b57d5979104838d002d349a95c73ef45.png"/>
    </p>
    <hr/>
    <p>
     这里根据下图
     <strong>
      打勾
     </strong>
     之后点击
     <strong>
      Install
     </strong>
     <br/>
     <mark>
      打勾即为使用自动配置环境变量，使用自动配置的环境变量之后在cmd中就可直接运行conda命令，python命令等。
     </mark>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7b80f67e1ee7aeed3098c459adb2db2.png"/>
    </p>
    <hr/>
    <p>
     这里等待稍许便可
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b9ca0c3c928814c1e91a29057440ac53.png"/>
    </p>
    <hr/>
    <p>
     点击
     <strong>
      Next
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a13793a395cb4ab7b2768f6d149068c2.png"/>
    </p>
    <hr/>
    <p>
     此处为pycharm的推广，如果有需要的可以点击如图所示链接，当然pycharm也可自行下载，在此笔者直接点击
     <strong>
      Next
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d7acda665aea61cb09a69ea493b2b64e.png"/>
    </p>
    <hr/>
    <p>
     点击
     <strong>
      Finish
     </strong>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f142bf0e5a246763c44dc9f68be43d15.png"/>
    </p>
    <hr/>
    <h3>
     <a id="Anaconda_89">
     </a>
     三、测试Anaconda是否安装配置成功
    </h3>
    <p>
     <strong>
      WIN+R键
     </strong>
     调出运行窗口，输入
     <strong>
      cmd
     </strong>
     回车
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a8fab3029bcc3faad1d8e5f0a06a0710.png"/>
    </p>
    <hr/>
    <p>
     输入conda命令查看是否安装成功
    </p>
    <pre><code class="prism language-xml">conda
</code></pre>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/64aab818ace4c1ba912a560685fbcc84.png"/>
    </p>
    <hr/>
    <p>
     输入conda -V命令（
     <mark>
      注意conda与-V之间有一个空格，评论区有好多童鞋反馈文章空格难以看到
     </mark>
     ）可查看当前Anaconda版本
    </p>
    <pre><code class="prism language-xml">conda -V
</code></pre>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b1867d1a8f336a013ce30958063e649b.png"/>
    </p>
    <hr/>
    <p>
     输入python命令查看是否可以使用python
    </p>
    <pre><code class="prism language-java">python
</code></pre>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/65a7734b44565db00abf36c1c0656110.png"/>
    </p>
    <hr/>
    <p>
     OK！安装完成！
    </p>
