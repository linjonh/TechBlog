    <p>
     <strong>
      ubuntu 16.04操作系统中用管理员权限安装以下软件
     </strong>
    </p>
    <p style="text-indent:33px;">
     安装xrdp：
    </p>
    <pre><code>sudo apt-get install xrdp</code></pre>
    <p style="text-indent:33px;">
     安装vnc4server：
    </p>
    <pre><code>sudo apt-get install vnc4server</code></pre>
    <p style="text-indent:33px;">
     安装xubuntu-desktop或ubuntu-desktop：
    </p>
    <p style="text-indent:33px;">
    </p>
    <pre><code>sudo apt-get install xubuntu-desktop</code></pre>
    <pre><code>sudo apt-get install ubuntu-desktop</code></pre>
    <p style="text-indent:33px;">
     xsession中写入xfce4-session,需要远程的用户都要单独执行这条命令：
    </p>
    <pre><code>echo "xfce4-session" &gt;~/.xsession</code></pre>
    <p style="text-indent:33px;">
     重启xrdp服务：
    </p>
    <pre><code>/etc/init.d/xrdp restart</code></pre>
