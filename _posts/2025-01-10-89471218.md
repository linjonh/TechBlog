    <h6>
     <a id="JDKJava_Development_Kit_1">
     </a>
     一、安装JDK(Java Development Kit开发工具包)
    </h6>
    <p>
     下载地址:
    </p>
    <blockquote>
     <p>
      <a href="http://www.oracle.com/technetwork/java/javase/downloads/index.html" rel="nofollow">
       http://www.oracle.com/technetwork/java/javase/downloads/index.html
      </a>
     </p>
    </blockquote>
    <p>
     选择
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/377c16d8c82ead253885fd4eab4f3892.png">
      <br/>
      然后
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1f7e59b1a88d832df27754e82a4e18e8.png">
       <br/>
       选择
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e6a7fd81efc263f5a5bb3f0705614b6d.png">
        <br/>
        然后等待下载安装。
        <br/>
        下载完成之后开始点击安装（安装的路径尽量不要有中文出现，否则后期可能出现问题 ）
        <br/>
        一般安装路径默认就好。
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6b9eba33f4efe07b7eeef6d3924ad96a.png">
      <br/>
      然后一直下一步，直到显示安装成功。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/843e366ef3e64998a6e95c52c1e350b9.png"/>
     </img>
    </p>
    <p>
     安装完成之后。开始我们的下一步。配置环境变量。
    </p>
    <h6>
     <a id="_19">
     </a>
     二、配置环境变量
    </h6>
    <p>
     1.鼠标右键 “计算机” 、“我的电脑”—&gt;选择属性
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f533297033e45ada3961d7c1ca11d886.png">
      <br/>
      2.选择 “高级环境变量”—&gt;然后选择 “环境变量”
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/51d76f618a1eea313652751f9933ecd0.png">
       <br/>
       3.开始配置
       <br/>
       3.1.在环境变量中，要配置三个属性（要注意区分大小写）:JAVA_HOME、Path、CLASSPATH（可也不设置）
       <br/>
       变量属性参数：
      </img>
     </img>
    </p>
    <ul>
     <li>
      JAVA_HOME
     </li>
     <li>
      C:\Program Files\Java\jdk1.8.0_192
     </li>
     <li>
      Path
     </li>
     <li>
      %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
     </li>
     <li>
      CLASSPATH
     </li>
     <li>
      .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; （注意最前面有点）
     </li>
    </ul>
    <p>
     配置完成效果图
     <br/>
     JAVA_HOME
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/261d36c2a653e9afca8433f4aa89c641.png">
      <br/>
      Path（因为win10是打开这样的，不影响，也可以点击“编辑文本”变成正常的输入框格式）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6c2225c5faaffe3d4738939cbe0aa3aa.png"/>
     </img>
    </p>
    <p>
     4.配置完成之后，测试是否配置成功
     <br/>
     win+r快捷键或者 选择“开始”–&gt;“运行” 输入cmd 回车 在弹出的 DOS 窗口内
     <br/>
     输入：java -version
     <br/>
     输入：javac
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a062eb7211940537308491942b707e94.png"/>
    </p>

     <a id="java_48">
     </a>
     到这里java的安转和配置环境变量就完成啦！！！
