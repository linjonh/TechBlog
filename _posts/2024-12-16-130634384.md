    <h3>
     <a id="_0">
     </a>
     三方库的获取
    </h3>
    <ol>
     <li>
      通过访问
      <a href="https://gitee.com/openharmony-tpc/tpc_resource" rel="nofollow">
       Gitee网站
      </a>
      获取
     </li>
     <li>
      通过
      <a href="https://repo.harmonyos.com/#/cn/application/atomService" rel="nofollow">
       HarmonyOS开发者
      </a>
      官网提供的资源中心获取
     </li>
    </ol>
    <h4>
     <a id="_5">
     </a>
     安装
    </h4>
    <ul>
     <li>
      方式一：在Terminal窗口中，执行如下命令安装三方包，DevEco Studio会自动在工程的oh-package.json5中自动添加三方包依赖。
     </li>
    </ul>
    <pre><code>ohpm install @ohos/lottie
</code></pre>
    <ul>
     <li>
      方式二：在工程的oh-package.json5中设置三方包依赖，配置示例如下：
     </li>
    </ul>
    <pre><code>"dependencies": { "@ohos/lottie": "^2.0.0"}
</code></pre>
