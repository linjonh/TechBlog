    <p>
     最近找到几个小游戏，在这里推荐一下。它们都是Puzzle类小游戏，关卡都不多，不会耽误你太多时间，却又让你感觉意犹未尽。
     <br/>
     <a href="http://armorgames.com/play/751/shift" rel="nofollow noopener noreferrer" target="_blank">
      Shift
     </a>
     ：按一下Shift键，整个世界黑白颠倒……
     <br/>
     <a href="http://www.kokoromi.org/gamma256games/mr_heart_loves_you_very_much.zip" rel="nofollow noopener noreferrer" target="_blank">
      Mr. Heart Loves You Very Much
     </a>
     ：绝对有创意。zip下载解压运行，文件不大。
     <br/>
     <a href="http://mathsnet.net/geometry/solid/houses.html" rel="nofollow noopener noreferrer" target="_blank">
      Building Houses 2
     </a>
     ：用指定数量的方块搭建出符合三种视图的3D图形。并不简单。
     <br/>
     <a href="http://www.skt-products.com/contents/bomb_sw_side2.html" rel="nofollow noopener noreferrer" target="_blank">
      3D版扫雷
     </a>
     ：按空格键开始标雷，再按一下空格键后鼠标回到排雷状态。
     <br/>
     <br/>
     然后推荐几个Quiz类游戏：
     <br/>
     <a href="http://proveyourlogic.com/" rel="nofollow noopener noreferrer" target="_blank">
      Prove Your Logic
     </a>
     ：新建成的一个网站。比较弱的智力趣题，个别题目有点意思。
     <br/>
     <a href="http://splapp-me-do.deviantart.com/art/The-Impossible-Quiz-Lite-50433260" rel="nofollow noopener noreferrer" target="_blank">
      The Impossible Quiz
     </a>
     ：我最喜欢的Quiz游戏。
     <strong>
      非常强大，强烈推荐！
     </strong>
     <br/>
     <a href="http://splapp-me-do.deviantart.com/art/The-Impossible-Quiz-2-63631070" rel="nofollow noopener noreferrer" target="_blank">
      The Impossible Quiz 2
     </a>
     ：偶然发现上面这玩意儿出了续集。依然很强大。
