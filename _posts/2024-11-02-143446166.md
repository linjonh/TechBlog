     教你如何在Visual Studio Code (VS Code) 中配置Python 3开发环境
    </h1>

  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <p>
     废话不说，直接上操作步骤，教你如何在Visual Studio Code (VS Code) 中配置Python 3开发环境：
    </p>
    <h4>
     第一步：安装Python 3
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        下载Python
       </strong>
       ：
      </p>
      <ul>
       <li>
        访问Python官方网站
        <a href="https://www.python.org/downloads/" rel="nofollow" title="Download Python | Python.org">
         Download Python | Python.org
        </a>
        。
       </li>
       <li>
        选择适合你操作系统的Python 3版本进行下载。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装Python
       </strong>
       ：
      </p>
      <ul>
       <li>
        在Windows上，运行下载的安装程序，确保勾选“Add Python to PATH”选项，以便在命令行中直接使用Python。
       </li>
       <li>
        在macOS和Linux上，通过终端安装Python，可以使用包管理器如
        <code>
         brew
        </code>
        （macOS）或
        <code>
         apt
        </code>
        （Ubuntu）。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第二步：验证Python安装
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        打开命令行
       </strong>
       ：
      </p>
      <ul>
       <li>
        Windows上打开“命令提示符”或“PowerShell”。
       </li>
       <li>
        macOS和Linux上打开“终端”。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        检查Python版本
       </strong>
       ：
      </p>
      <ul>
       <li>
        输入
        <code>
         python --version
        </code>
        或
        <code>
         python3 --version
        </code>
        并回车，确保显示的是Python 3.x.x版本。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第三步：安装Visual Studio Code
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        下载VS Code
       </strong>
       ：
      </p>
      <ul>
       <li>
        访问VS Code官网
        <a href="https://code.visualstudio.com/" rel="nofollow" title="Visual Studio Code - Code Editing. Redefined">
         Visual Studio Code - Code Editing. Redefined
        </a>
        。
       </li>
       <li>
        下载适合你操作系统的VS Code版本。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装VS Code
       </strong>
       ：
      </p>
      <ul>
       <li>
        运行下载的安装程序并按照提示完成安装。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第四步：在VS Code中安装Python扩展
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        打开VS Code
       </strong>
       ：
      </p>
      <ul>
       <li>
        启动VS Code。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        访问扩展市场
       </strong>
       ：
      </p>
      <ul>
       <li>
        点击左侧的扩展图标，或按下
        <code>
         Ctrl+Shift+X
        </code>
        （Windows/Linux）/
        <code>
         Cmd+Shift+X
        </code>
        （macOS）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        搜索Python扩展
       </strong>
       ：
      </p>
      <ul>
       <li>
        在搜索框中输入“Python”，找到由Microsoft提供的Python扩展。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装Python扩展
       </strong>
       ：
      </p>
      <ul>
       <li>
        点击“安装”按钮安装Python扩展。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第五步：配置Python解释器
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        打开Python文件
       </strong>
       ：
      </p>
      <ul>
       <li>
        在VS Code中打开一个
        <code>
         .py
        </code>
        文件，或者创建一个新的Python文件。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择Python解释器
       </strong>
       ：
      </p>
      <ul>
       <li>
        按下
        <code>
         Ctrl+Shift+P
        </code>
        （Windows/Linux）/
        <code>
         Cmd+Shift+P
        </code>
        （macOS）打开命令面板。
       </li>
       <li>
        输入“Python: Select Interpreter”并选择它。
       </li>
       <li>
        从列表中选择你安装的Python 3解释器。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第六步：配置Python环境
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        创建虚拟环境
       </strong>
       （可选）：
      </p>
      <ul>
       <li>
        在终端中，导航到你的项目目录。
       </li>
       <li>
        输入
        <code>
         python -m venv venv
        </code>
        创建虚拟环境。
       </li>
       <li>
        激活虚拟环境：
        <ul>
         <li>
          Windows：
          <code>
           .\venv\Scripts\activate
          </code>
         </li>
         <li>
          macOS/Linux：
          <code>
           source venv/bin/activate
          </code>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        在VS Code中使用虚拟环境
       </strong>
       ：
      </p>
      <ul>
       <li>
        确保虚拟环境被激活。
       </li>
       <li>
        重新运行“Python: Select Interpreter”，选择虚拟环境中的Python解释器。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第七步：安装Python依赖
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        使用pip安装依赖
       </strong>
       ：
      </p>
      <ul>
       <li>
        在终端中，使用
        <code>
         pip install package_name
        </code>
        安装所需的Python包。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        在VS Code中查看和管理依赖
       </strong>
       ：
      </p>
      <ul>
       <li>
        打开
        <code>
         requirements.txt
        </code>
        文件，VS Code将提供安装和卸载包的功能。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     第八步：编写和运行Python代码
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        编写代码
       </strong>
       ：
      </p>
      <ul>
       <li>
        在VS Code中编写Python代码。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        运行代码
       </strong>
       ：
      </p>
      <ul>
       <li>
        按下
        <code>
         F5
        </code>
        或点击左侧的“运行”按钮开始调试。
       </li>
       <li>
        你也可以直接在终端中运行
        <code>
         python script.py
        </code>
        来执行Python脚本。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     按照这些步骤，你应该能够在VS Code中成功配置Python 3环境，并开始编写和运行Python代码。如果在安装或配置过程中遇到任何问题，可以查看VS Code的官方文档或搜索相关错误信息以获得帮助。
    </p>
    <blockquote>
     <p>
      <strong>
       <span style="color:#fe2c24;">
        小贴士
       </span>
      </strong>
     </p>
     <p>
      windows系统注意：如果VS Code中默认的终端是Power Shell，那么在激活虚拟环境时可能会出现：
      <strong>
       .\venv\Scripts\activate : 无法加载文件 E:\workspace\python\venv\Scripts\Activate.ps1，因为在此系统上禁止运行脚本。
      </strong>
     </p>
     <p>
      这个错误表明你正在尝试执行一个PowerShell脚本（Activate.ps1），但是你的系统配置不允许执行该脚本。这通常是因为系统的执行策略限制了脚本的执行。
     </p>
     <p>
      解决方法：
     </p>
     <ol>
      <li>
       <p>
        使用管理员权限打开PowerShell。
       </p>
      </li>
      <li>
       <p>
        执行以下命令来为当前用户更改策略：Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass。
       </p>
      </li>
      <li>
       <p>
        如果VS Code需要重新打开策略才能生效。
       </p>
      </li>
     </ol>
    </blockquote>
