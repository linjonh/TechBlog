    <p>
     本篇文章是以前端vue框架演示
    </p>
    <h3>
     <a id="1_1">
     </a>
     1、项目打包
    </h3>
    <p>
     vue项目我们一般使用的编辑器为vscode，打包vue项目在vscode中打开项目文件夹，按ctrl + ` 打开vscode控制台，运行以下命令
    </p>
    <pre><code>npm run build
</code></pre>
    <p>
     等待控制台运行打包命令完毕之后，我们的项目文件夹中会生成一个dist文件夹。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/06f181842c04e482d8ca22cb7b7fb8bb.png"/>
    </p>
    <h3>
     <a id="2_10">
     </a>
     2、将打包文件上传的服务器
    </h3>
    <p>
     进入宝塔页面的文件选项中，并进入目录www/wwwroot
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac8080fa12b264dc20a08f0e52c43ca8.png">
      <br/>
      新建一个文件夹，用于存放我们的vue项目文件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/02f873adb46d749e826c8025456550f0.png">
       <br/>
       将刚才打包生成的dist文件夹整个上传到该文件夹中
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/095b38312b5e642ff98a2af353911460.png"/>
      </img>
     </img>
    </p>
    <h3>
     <a id="3_18">
     </a>
     3、添加站点
    </h3>
    <p>
     进入网站选项卡中
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/08c4656b0483ed8d658140f2da673efd.png">
      <br/>
      点击添加站点
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a684a277cea60252712fe9bc33e58dfc.png">
       <br/>
       域名：任意字符串:端口 例如 test:81 ，81即为我们想要在云服务器上展示该前端项目的端口。
       <br/>
       根目录：选择刚才上传至www/wwwroot中的dist文件夹
       <br/>
       数据库和FTP都为不创建
       <br/>
       PHP版本选择纯静态
       <br/>
       网站分类为默认分类即可
       <br/>
       如图所示
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9831a5fc8062bf471ec61ae7270e81bf.png">
        <br/>
        点击提交即可
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="4_32">
     </a>
     4、云服务器开放端口
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0162259815e74eedad1b61c63904cff8.png">
      <br/>
      在云服务器中点击添加规则，然后再在端口中输入刚才域名处输入的端口，例如我刚才的是test:81，则就在这里的端口地方填81，然后点击确认即可。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a86050bda15937007eed9cca999d2ad8.png">
       <br/>
       5、访问网站
       <br/>
       如没有备案域名则在网址输入 云服务器地址:端口号即可
       <br/>
       有备案域名且已绑定该云服务器的话 也可以 域名:端口
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4cb44f49cb1bb80a2127d6ea231648ff.png">
        <br/>
        我这里访问成功。
       </img>
      </img>
     </img>
    </p>
