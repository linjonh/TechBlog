    <h3>
     一、前言
    </h3>
    <blockquote>
     <ul>
      <li>
       伴随着云计算的第三次浪潮的滚滚而来，云原生 (CloudNative) 的概念应运而生。
      </li>
      <li>
       我们又如何选择适合自己的开发平台？不同的开发平台有什么样的特点？这篇文章为你做一个大概的了解。
      </li>
     </ul>
    </blockquote>
    <h3>
     二、阿里云
    </h3>
    <p style="text-align:center;">
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/435d28656843f70a3636cff7bac6ca21.png"/>
    </p>
    <blockquote>
     <ul>
      <li>
       阿里云目前拥有国内更加多样化的的云原生产品系列，应用实践案例以及广泛的用户基础，对于企业或者个人开发者在国内是一个不错的选择。
      </li>
      <li>
       <strong>
        特点
       </strong>
       ：上手简单，支持多种语言以及windows、linux兼容。适合业务大规模、分发性能要求高的互联网平台、金融教育、以及零售市场。
      </li>
      <li>
       <strong>
        优势
       </strong>
       ：AWS、Google不支持HelmChart等云原生产品管理，不支持独享实例形态。
      </li>
     </ul>
    </blockquote>
    <h3>
     三、AWS
    </h3>
    <p style="text-align:center;">
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/794df48703e58e5c6a485993bab4d756.png"/>
    </p>
    <p>
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/2ea14ffcee483179e78078a06c18d08f.png"/>
    </p>
    <blockquote>
     <ul>
      <li>
       亚马逊云科技连续
       <strong>
       </strong>
       11年 被 信息研究技术公司Gartner 评为“全球云计算领导者”， 是一个世界型云基础设施与平台服务的领导者。
      </li>
      <li>
       <strong>
        特点
       </strong>
       ：亚马逊云科技对于上云计算、存储和数据库等基础设施技术，到机器学习、人工智能、数据湖和分析以及物联网提供丰富完整的服务及功能。
      </li>
      <li>
       <strong>
        优势
       </strong>
       ：对于扩展的可见性与控制力、高隐私化与数据安全，对于一些小型的云计算产品优惠力度较大。
      </li>
     </ul>
    </blockquote>
    <h3>
     四、Azure
    </h3>
    <p style="text-align:center;">
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/e6fd5cc9ecde8d3379fe92f5051f52d1.png"/>
    </p>
    <p>
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/d0f9dbec0e873d20a651dd0f71a4c74b.png"/>
    </p>
    <blockquote>
     <ul>
      <li>
       Microsoft Azure简称Azure，是微软基于
       <a href="https://baike.baidu.com/item/%E4%BA%91%E8%AE%A1%E7%AE%97/9969353" rel="nofollow" title="云计算">
        云计算
       </a>
       的操作系统，企业能够在多个数据中心快速开发、部署、管理应用程序 。
      </li>
      <li>
       一种灵活和支持兼容操作的平台，帮助开发可运行在云服务器、数据中心、Web与PC上开发部署，对网站、虚拟机、云服务、移动应用服务、大数据支持以及媒体功能的支持。
      </li>
      <li>
       <strong>
        优势
       </strong>
       ：云存储技术性能、扩展性和稳定性三项功能
      </li>
     </ul>
    </blockquote>
    <h3>
     五、Google
    </h3>
    <p style="text-align:center;">
     <img alt="watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBA55-l5b-D5a6d6LSd,size_20,color_FFFFFF,t_70,g_se,x_16" src="https://i-blog.csdnimg.cn/blog_migrate/ecbe477a763a01b3ad905b9bd5a929c2.png"/>
    </p>
    <blockquote>
     <ul>
      <li>
       Google Cloud 提供了安全、强大、经济高效且持续更新的基础架构。
      </li>
      <li>
       <strong>
        优势：
       </strong>
       团队超强的协同工作、 Google Cloud Platform 基础架构、BigQuery 等数据仓库平台以及 Data Studio 和 Looker 等可视化工具
      </li>
     </ul>
    </blockquote>
    <h3>
     六、总结
    </h3>
    <blockquote>
     <ul>
      <li>
       <strong>
        个人开发者
       </strong>
       ：不建议同时使用多个大厂的服务，如果是第一次使用云管理平台的话。可以试着从国内的阿里云平台开始开发，后面可以选择AWS或者微软的Azure的平台，google技术虽然很强，但对于初学者来说并不是十分友好。
      </li>
      <li>
       <strong>
        团队公司开发
       </strong>
       ：如果有需求可以同时使用多个大厂的服务，但要注意兼容性问题。
      </li>
     </ul>
    </blockquote>
