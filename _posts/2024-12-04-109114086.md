    <p>
     <strong>
      TMC
     </strong>
     <br/>
     TMC是一个开源智慧消防云平台，该平台融合了无线烟感监测、可燃气器监测、电气火灾监测、防火门监测、消防用水监测、消防主机联网、消防电源监测、消防巡检、视频智能识别九大子系统，是城市级消防联网的全套解决方案。 随着智慧消防发展到5G时代的，越来越多的传统消防企业亟待转型升级，中科图灵顺势而为，希望通过拥抱开源，降低智慧消防行业准入门槛，帮助更多的传统企业轻松转型智慧消防，推动行业标准建设。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/454f4fc881b887e1f333c4aec604bc73.png#pic_center">
      <br/>
      <strong>
       系统架构
      </strong>
      <br/>
      智慧消防设备满足物联网的三个层次(应用层、传输层、感知层)， 运用传感器等技术，使信息传感设备实时感知需要的信息，按照约定 的协议，通过可能的网络(如基于 LoRa 的无线局域网)接入方式，把物品与互联网相连接，进行信息交换和通信，实现物与物、物与人的 泛在链接，实现对消防设备的智慧化识别、跟踪、监控和管理。
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14bcbce6e0e4d3e38460e55b5585edde.png#pic_center">
      <br/>
      <strong>
       主要功能
      </strong>
      <br/>
      1.无线烟感远程监测预警
      <br/>
      支持接入无线烟感、无线温感报警器，远程查看室内烟雾度、温度、传感器电池电量
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69d826c06e3e935ae0c0bde412425841.png#pic_center">
       <br/>
       2.电气火灾远程监测预警
       <br/>
       支持接入组合式电气火灾报警器，远程查看电气线路的漏电值、电线温度、电流值、电压值
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1aacc726cc69acbcee89678267fbb08f.png#pic_center">
      <br/>
      3.可燃气体远程监测预警
      <br/>
      支持接入无线可燃气体报警器，远程查看室内可燃气体浓度
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/70b51d99cc3e2a1c0b2925395d2cbf64.png#pic_center">
       <br/>
       4.消防用水监测预警
       <br/>
       支持接入无线水压、水位传感器，远程查看消火栓、喷淋管、消防水池的水压值、水位值
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1988fa72b9a4d455eb581f805db3e2fe.png#pic_center">
        <br/>
        5.消防主机远程联网
        <br/>
        支持接入主流传统消防企业的消防主机数据，远程查看消防主机工作状态、实时报警数据
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a6d65515e9decb68d0ee46876be17eb.png#pic_center">
         <br/>
         6.消防电源远程监测预警
         <br/>
         支持接入无线消防电源监测报警器，远程查看消防电源电压值、电流值
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/57cdb9378d1c1e3ba64652556422ae1a.png#pic_center"/>
         <br/>
         7.防火门远程监控预警
         <br/>
         支持接入无线门磁报警器，远程查看防火门开启、闭合状态
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8690d2b2825f922de3fc2f7fad894822.png#pic_center"/>
         <br/>
         8.消防电子巡检及故障上报
         <br/>
         支持在消防设施上粘贴二维码或RFDI标签，使用移动终端对消防设备、设施进行巡检和故障上报
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69ba2e7bfe5cbbb1c39554de1e63b51c.png#pic_center"/>
         <br/>
         9.无线物联传感器监测
         <br/>
         支持各类物联网传感器的接入，自定义传感器测点，远程查看传感器实时监测数值
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/43688dd93c0d961691b91809f13002ec.png#pic_center"/>
        </img>
       </img>
      </img>
     </img>

     <strong>
      演示案例
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de4d30aded9f3b51d3dc7e21fe5ac076.png#pic_center"/>

     <strong>
      智慧消防云平台部署说明
     </strong>
     <br/>
     <strong>
      数据库环境准备
     </strong>
     <br/>
     mysql 5.7
     <br/>
     redis 5.0.0
     <br/>
     elasticsearch 6.5.4
     <br/>
     rabbitmq 3.6.9
     <br/>
     mysql数据库资源创建好后，执行如下sql创建初始化数据库
     <br/>
     CREATE DATABASE
     <code>
      turing-admin
     </code>
     CHARACTER SET ‘utf8’ COLLATE ‘utf8_general_ci’;CREATE DATABASE
     <code>
      turing-auth
     </code>
     CHARACTER SET ‘utf8’ COLLATE ‘utf8_general_ci’;CREATE DATABASE
     <code>
      turing_datahandler
     </code>
     CHARACTER SET ‘utf8’ COLLATE ‘utf8_general_ci’;CREATE DATABASE
     <code>
      turing_device
     </code>
     CHARACTER SET ‘utf8’ COLLATE ‘utf8_general_ci’;
     <br/>
     并导入sql目录下的各个sql文件来创建初始化表和初始化数据
     <br/>
     <strong>
      基础配置准备
     </strong>
     <br/>
     SpringCloud使用gitee来作为配置中心 示例配置文件地址
     <br/>
     修改gitee上各个微服务配置下的数据库链接信息，并对应修改微服务代码配置yml里面的gitee配置信息
     <br/>
     <strong>
      代码编译打包
     </strong>
     <br/>
     1.编译各个微服务代码，生成对应jar包
     <br/>
     2.执行各个服务文件夹下Dockerfile文件，打包成docker镜像
     <br/>
     <strong>
      启动服务
     </strong>
     <br/>
     对应修改docker-compose.yml配置文件，执行docker compose up -d 即可启动服务
     <br/>
     <strong>
      演示地址
     </strong>
     <br/>
     TMC智慧消防云平台官网： https://tmc.zkturing.com/
     <br/>
     TMC智慧消防云平台演示教程： https://tmc.zkturing.com/list-60-1.html
