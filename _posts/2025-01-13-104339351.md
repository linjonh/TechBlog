    <h4>
     <a id="SQL_Server_0">
     </a>
     删除SQL Server数据库
    </h4>
    <p>
     如果用户不再需要某一数据库时，只要满足一定的条件即可将其删除，删除之后，相应的数据库文件及其数据都会被删除，并且不可恢复。
    </p>
    <p>
     1．以界面方式删除数据库
     <br/>
     下面介绍如何删除数据库“MingRi”。具体操作步骤如下：
     <br/>
     （1）启动SQL Server Management Studio，并连接到SQL Server 2012中的数据库。在“对象资源管理器”中展开“数据库”节点。
     <br/>
     （2）鼠标右键单击要删除的数据库“MingRi”选项，在弹出的快捷菜单中选择“删除”命令。
    </p>
    <p>
     （3）在弹出的“删除对象”对话框中单击“确定”按钮即可删除数据库。
    </p>
    <p>
     注意：
     <br/>
     系统数据库（msdb、model、master、tempdb）无法删除。删除数据库后应立即备份master数据库，因为删除数据库将更新master数据库中的信息。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/41a91d385773da4286207aa316f77960.png">
      <br/>
      2．使用DROP DATABASE语句删除数据库
      <br/>
      语法格式如下：
      <br/>
      DROP DATABASE database_name [ ,…n ] --如果有多个要删除的数据库，用逗号隔开
      <br/>
      其中database_name是要删除的数据库名称，中括号内为多个数据库的情况。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f0dbb3ea6678b48ca390eda344476f54.png"/>
     </img>
    </p>

    <p>
     <strong>
      –删除数据库
     </strong>
    </p>
    <pre><code class="prism language-csharp"><span class="token class-name">DROP</span> DATABASE <span class="token punctuation">[</span><span class="token class-name">DB</span><span class="token operator">-</span>date02<span class="token punctuation">]</span>
</code></pre>
    <p>
     <strong>
      注意：
     </strong>
     <br/>
     使用DROP DATABASE命令删除数据库时，系统中必须存在所要删除的数据库，否则系统将会出现错误。
     <br/>
     另外，如果删除正在使用的数据库，系统将会出现错误。
    </p>
