       【程序人生 编者按】作为科技巨头之一——微软，是很多人梦寐以求的工作企业之一。但是本文作者却有着与之
       <span style="color:rgb(136,136,136);font-size:15px;letter-spacing:1px;">
        完全
       </span>
       相反的观点，其认为是时候和微软说再见了，而这究竟是怎么一回事？
      </span>
      <br/>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">

     <p style="margin-left:8px;min-height:1em;letter-spacing:1px;line-height:normal;text-align:right;">

       <br/>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;letter-spacing:1px;line-height:normal;text-align:right;">
      <span style="font-size:14px;color:rgb(136,136,136);">
       译者 | 王艳妮，责编 | 屠敏
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;letter-spacing:1px;line-height:normal;text-align:right;">
      <span style="font-size:14px;color:rgb(136,136,136);">
       出品 | CSDN（ID：CSDNnews）
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       很多人会问——要怎样才能进入微软？面试是什么样的？在微软工作是一种什么体验？对我而言，在曾经的三五年里，微软意味着一切。现在，在离开18个月后，我终于准备好分享我的故事，并真正地说一句再见了，微软！
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       我当时正在一个营销机构实习，并在网上浏览工作机会，在此之前我在创业公司有过工作经验。我那时23岁，大学还没毕业，脑子里充满了各种想法和对未来的希望。就这样我申请了一个MACH项目——微软校园招聘学院。招聘的过程很漫长，分几个阶段，最后以评估中心告终。有一件有趣的事：我被问到一个“刁钻”的问题：为什么下水道井盖是圆的？还有一个，没那么有趣，但是最为重要，即
       <strong>
        我在开发者体验和传播部门
       </strong>
       （现在叫CSE？或者其他名字，这个部门一直在演变......）成为一个位于波兰华沙的受众市场经理。我的角色是让人们，尤其是小型IT公司的人们尝试使用Microsoft Azure（负责广度市场）。那时5年前，云服务还是一个新奇事物，媒体评价并不是那么好。我收到过许多封把微软拼写成“M $”的企业家发来的正经电子邮件。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       我注意到的第一件事就是
       <strong>
        几
       </strong>
       <strong>
        乎所有为微软工作的人都拥有这样一个完美的生活——配偶、孩子、漂亮的房子、好车。
       </strong>
       这些我也想要。幻想着如果我留在这家公司，那么我的生活就会像他们的一样。（这是一个一下子就把我迷住了的幻象，但稍后我会看到破碎的现实......）那时候，我正处于一段认真的感情中，并决定把这个微软提供的专业的机会最大化地利用起来，专注于自身的成长。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       微软给了我太多了。
       <strong>
        我在入职第一天就获得了我的第一个职责和目标，但我也得到了很多支持和培训。
       </strong>
       作为一名新入职的员工，我在亚特兰大参加MGX（现在好像是叫Inspire？），在马德里和阿姆斯特丹接受过培训。那里我遇到了那么多有趣的人，他们自信满满，当然我也遇到过尖酸刻薄的人。什么样的人都有。我每天都和来自世界各地的人通电话，而且我得到了很高的报酬。这是一份梦想一般的工作。我当时的团队有4个人，跟我年龄相仿，我和他们相处得很好，所以我开始花相当长的时间待在办公室里工作。作为一名为DX工作的人，我被鼓励去任何我想去的会议，研讨会，编程马拉松，聚会等等，这些我也真的都去了。有一段时间我几乎不着家。我当时很兴奋。每天都会遇到新的，鼓舞人心的人，并且每天能学到很多东西！与此同时，头等舱旅行，出租车，不错的酒店和高档餐厅都快把我宠坏了。除了这些，还有来自他人的赞赏，在他人眼中我已经是成功人士了。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       <img alt="640?wx_fmt=jpeg" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/7660a237f9239064a1ec48937617091e.jpeg" style="text-align:center;"/>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       我担任共同组织人的会议，Satya Nadella作为主要发言人
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       另一方面，它也并不总是那么迷人。很多员工都不尊重我，对他们来说，我是一个“小明星”，这个“年轻的傻女孩，牙缝还大”（是的，办公室的某个人就是这么说我的），有些人对公司雇用了一个23岁的女孩而不是一个“专业男性”感到生气。但我觉得没什么，我知道我必须首先证明自己。但我不喜欢他们对于我外表的评论。所有这些看起来都不是什么重大的事情，并且可以接受——毕竟我在微软的工作给了我这么多的回报。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       回到家里，我的伴侣却并不跟我一样如此热忱。他起初觉得我们会一起开始组建我们的小家庭，但我却几乎不怎么在他身边。二十多岁的年纪，那时我正在最大限度地享受生命，或者仅仅是我认为的最大限度吧——不停地工作，而且经常喝很多酒。我记得曾经在凌晨2点的俱乐部写电子邮件！即使现在这对我来说似乎并不那么疯狂了。我不懂得照顾自己不仅仅是在喝酒方面——我在工作的第一年胖了10公斤（22磅）。一旦你被录用，体重的增加是很常见的事情。我们笑着谈论这件事，这简直是个标准流程。每个人都经历过。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       时间渐渐过去。队员们要么换团队要么被解雇。这很痛苦，实际上，有那么一次，当我告诉我的一个团队成员，他被解雇了，而且没有任何真正解释的时候，我没忍住哭了。他不是全职员工，所以他没有得到任何补偿（对于那些不知道的人——你可以作为全职员工被微软雇佣——获得所有福利、旅行、培训等等——或者通过劳务机构被雇佣——无法享受任何全职员工的福利），但似乎没有别人关心他被解雇这件事。到目前为止，我在这项工作上已经投入了太多精力了。没有别人理会我的抗争。我尝试过改变一些事情，但我到头来只是在不断为此付出代价——用我的健康和我的爱情。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       后来我去参加了火人节的活动，搬出了我曾和男朋友同住的公寓。曾经我对“完美”生活的整个幻象都消失了。很多人都像我一样，不停地工作，不能为他们的亲近之人腾出时间，但他们能够找到对此不以为意的伴侣罢了。我觉得从这一刻起，我不再相信在微软工作有那么棒了，尽管周围的每个人似乎仍然对工作充满激情。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       气氛不复存在，事情正悄然发生变化，同事们不再那么有趣了，但我却多了更多有趣的责任——与技术社区的leader会面及合作。后来我从2年的MACH课程毕业了，并被评为该部门的最佳员工。我收到了一份很不错的奖金，以及一个去雷德蒙德工作几个月的offer，以更好地进行员工交流。我还与部门主管进行了一对一的会谈，他告诉我，我不能再那么叛逆了，并且有点威胁我的意思，暗示我应该更加谨慎一些，因为微软为我提供了一个“金笼子”，如果有一天从笼子里出去，我将不得不面对严酷的现实。是的，这就是这样的讯息：你年轻而充满活力，这很酷，但现在你给我冷静一下，永远也别妄想离开微软了，因为外面的世界会将你生吞活剥，明白了吗？
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       与此同时，我与男朋友也分手了，我很难受，但不知怎的，我的职业生涯似乎却在蓬勃发展。我有机会在雷德蒙德工作并有机会在那里找到一个自己的位置，所以我可以成为那些“总部”员工之一！多么惊人！这是这一行大多数人梦寐以求的事情，对吧？
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       <img alt="640?wx_fmt=png" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/3111136ce83fa0ec78df0cc5fef74b26.jpeg" style="text-align:center;"/>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       我在西雅图贝尔镇的住所，窗外的景色
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       我在西雅图市中心选择了一处公司的住房公寓，这是个很漂亮的地方（这可能是我生命中会住的最好的公寓）。位于豪华建筑的最后一层，带一个海景阳台。简直华丽。然后我开始工作了。巨大的园区，精致的自助餐厅，树林，池塘，令人应接不暇。一切看起来都很棒，但是......是的，不幸的是有一个“但是”...一切都很好，除了人。我团队中的人思想非常封闭。他们中的大多数人在公司工作多年，所以更加封闭，他们只关心经理的批准。这种环境下我简直要被摧毁了。他们是我到今天为止遇到过的最虚伪的人。我甚至无法解释，我必须帮助他们，以及对他们表现出关心并假装支持他们正在做的事情这让我觉得有多么恶心。你可能在想——为什么你不跟他们说实话呢？我试过了。与我的经理，我的上级进行一对一的平等对话，但根本没人关心，我的角色被最小化了，以至于最后我仅仅在做一个Excel文件。我百分百肯定，如果我遵守他们的规则，并像他们那样，专注于办公室政治，我会在那里做得很出色，但我拒绝。我真的很关心我的角色，我关心的是我们员工间的交流和纽带而不仅仅是我自己的事业，似乎没有人理解这一点。我觉得在这段时间里我慢慢开始变得很沮丧。我不再出现在办公室，基本上只是在倒数回家的日子。我在那里待了3个月。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       当我回来的时候，前几天我真的很开心，但那种感觉开始快速消退。我也不想来华沙的办公室了。
       <strong>
        我每天都越来越多地待在家里，甚至从床上起床，只是浏览电子邮件并在Skype上签到
       </strong>
       ，假装我正在工作。我希望我能说这仅仅持续了一两天，但这实际上持续了几周。从雷德蒙德回来以后，我一直在考虑辞职，但我仍然喜欢与技术社区leader一起工作，这也是我那时坚持工作的动力。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       <img alt="640?wx_fmt=jpeg" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/573ecde78b520c46e93fee5b673cbe27.jpeg" style="text-align:center;"/>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;text-align:center;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       CEE MVP峰会，贝尔格莱德，塞尔维亚，2017年3月
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       微软在全球范围内发生了结构性变化。我告诉我的经理那年我不想要任何奖金，因为我当年在雷德蒙德，而对华沙当地没有做出很大的贡献，他应该奖励其他人。我也不配得到奖金，我再也无法忍受我的工作了，但仍然没有胆量辞职。这确实是一个“梦想的工作”，它给了我很多，但它也从我这里拿走了很多，我需要休息一下。在局势动荡起伏的时候，我却要求3个月的无薪休假。其他人警告我，这不是一个好主意，现在不是休假的时候，但我真的再也忍不了了。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       最终结果是我得到了5个星期的休假，然后前往葡萄牙参加了一次健身活动。这次活动改变了我，也改变了我对生活的看法。它给了我一种简约生活方式的全新视角，让我觉得我不需要微软的所有福利和高薪来获得快乐。在我度假期间，我收到了一些我可能会被解雇的信号。我当时的职称是技术布道者，这有点骗局的意味。我在技术上并不是很棒，但是官方的蓝图说团队需要更多的技术布道者而我又已经学了编程，所以这个职称给了我。这是许多“小小谎言”中的一个......公司总部试图尽量减少这些地方级别的恶作剧似的职称，我尊重这一点，但这意味着我回到华沙将接到解雇文件。有一个新的职位提案在等待更多的进展，但我当时在波兰的职位已经是对我而言最好的了，而回到雷德蒙德对我将是最糟糕的噩梦。所以我签了字，哭了一个多小时。从年度最好员工到被解雇。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       最可悲的是，告诉了我这件事的人并不是我的经理。
       <strong>
        到那时我已经与我的经理并肩工作了三五年了，但他在整个过程中甚至连句话都没跟我说。
       </strong>
       我所认为拥有的大多数朋友，我花了这么多时间陪伴的所有这些人，立刻变成了陌生人。我没有一个欢送会，再也没有被邀请去参加派对。在牺牲爱情和牺牲自我的同时，我一直将自己奉献给这一份工作，但整个微软里只有两个人关心（其中一个是因为我们约会过）。我听到的客气话来自公司外面的人——技术社区leader们，他们都非常能欣赏和支持他人，对此我永远心怀感激。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       我所处的并不是“金笼子”，
       <strong>
        而是一个装满幻象的泡泡，在我签署解雇文件的瞬间就幻灭了
       </strong>
       <strong>
        。一个关于重要性，使命感，友谊和奢侈生活的幻象。
       </strong>
       后来我很容易地找到一份薪水相当的工作，而且也不需要一天24小时，一周7天都随时待命。唯一真正“变差”了的是，我不再总用Uber打车了，而且也不会再花很多钱住高档的酒店了。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
