    <p>
     Go是由Google创建的现代开源编程语言。 Go中编写了许多流行的应用程序，包括Kubernetes，Docker和Grafana。
    </p>
    <h5>
     <a id="1_go_1">
     </a>
     1. 下载go并解压缩
    </h5>
    <ul>
     <li>
      到Go官网下载Go安包，下载地址
      <a href="https://golang.org/dl/" rel="nofollow">
       https://golang.org/dl/
      </a>
      。我直接下载的是最新的
      <code>
       go-1.12.1
      </code>
      。选择的是linux 64位版本：
      <a href="https://dl.google.com/go/go1.12.1.linux-amd64.tar.gz" rel="nofollow">
       go1.12.1.linux-amd64.tar.gz
      </a>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a1ee80bd3d315c5d2b90fbec427c2bd9.png"/>
    </p>
    <ul>
     <li>
      <p>
       国内比较快的下载链接，通过golang中文社区进行下载：
       <a href="https://studygolang.com/dl" rel="nofollow">
        Go 安装包下载
       </a>
      </p>
     </li>
     <li>
      <p>
       解压缩
       <code>
        go1.12.1.linux-amd64.tar.gz
       </code>
       。
      </p>
     </li>
    </ul>
    <pre><code class="prism language-python">$ tar <span class="token operator">-</span>zxvf go1<span class="token punctuation">.</span><span class="token number">12.1</span><span class="token punctuation">.</span>linux<span class="token operator">-</span>amd64<span class="token punctuation">.</span>tar<span class="token punctuation">.</span>gz
</code></pre>
    <ul>
     <li>
      将解压后的文件夹，放到自己平常的安装包位置，并设置
      <code>
       0777
      </code>
      权限
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo cp <span class="token operator">-</span>rf go <span class="token operator">/</span>usr<span class="token operator">/</span>local
$ cd <span class="token operator">/</span>usr<span class="token operator">/</span>local
$ sudo chmod <span class="token operator">-</span>R <span class="token number">0777</span> go
</code></pre>
    <h5>
     <a id="2_go_20">
     </a>
     2. 配置go环境变量
    </h5>
    <p>
     go语言的环境变量配置，需要两个值：
    </p>
    <ol>
     <li>
      <code>
       GOROOT
      </code>
      是系统上安装Go软件包的位置
     </li>
     <li>
      <code>
       GOPATH
      </code>
      是工作目录的位置
     </li>
    </ol>
    <ul>
     <li>
      创建工作目录，我选择创建在
      <code>
       /home/hadoop
      </code>
      目录下.
     </li>
    </ul>
    <pre><code class="prism language-python">$ mkdir GOPATH
$ sudo chmod <span class="token operator">-</span>R <span class="token number">0777</span> GOPATH
</code></pre>
    <ul>
     <li>
      使用
      <code>
       sudo gedit ~/.bashrc
      </code>
      命令修改环境变量，在弹出的记事本中添加以下内容。同时也在
      <code>
       GOPATH
      </code>
      目录下创建src和bin目录。
     </li>
    </ul>
    <pre><code class="prism language-python">export GOROOT<span class="token operator">=</span><span class="token operator">/</span>usr<span class="token operator">/</span>local<span class="token operator">/</span>go <span class="token comment">#GOROOT是系统上安装Go软件包的位置。</span>
export GOPATH<span class="token operator">=</span><span class="token operator">/</span>home<span class="token operator">/</span>hadoop<span class="token operator">/</span>GOPATH <span class="token comment">#GOPATH是工作目录的位置。</span>
export PATH<span class="token operator">=</span>$GOPATH<span class="token operator">/</span><span class="token builtin">bin</span><span class="token punctuation">:</span>$GOROOT<span class="token operator">/</span><span class="token builtin">bin</span><span class="token punctuation">:</span>$PATH
</code></pre>
    <ul>
     <li>
      使用一下命令使环境变量生效，并可以查看环境变量
     </li>
    </ul>
    <pre><code class="prism language-go">$ source ~<span class="token operator">/</span><span class="token punctuation">.</span>bashrc
$ export
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d9e99a7072c9dde6692ff220c138615f.png"/>
    </p>
    <ul>
     <li>
      使用同样的方法，修改
      <code>
       /etc/profile
      </code>
      文件，为其添加go环境变量（这是我的个人习惯，喜欢两个地方都作更改）。
     </li>
    </ul>
    <h5>
     <a id="3_go_51">
     </a>
     3. 验证go安装是否成功
    </h5>
    <ul>
     <li>
      输入以下命令查看go版本，顺便验证go是否安装成功。
     </li>
    </ul>
    <pre><code class="prism language-python">$ go version
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b67a64c6bf1e37f5867baa4114a4aa3b.png"/>
    </p>
    <ul>
     <li>
      编写hello程序，验证go安装是否成功。
      <br/>
      ① 在
      <code>
       GOPATH
      </code>
      目录下创建
      <code>
       hello
      </code>
      目录，用于存放go的第一个程序。
      <br/>
      ② 在
      <code>
       hello
      </code>
      目录下，创建
      <code>
       hello.go
      </code>
      文件，内容如下：
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">package</span> hello

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"Hello, World\n"</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     ③ 切换到
     <code>
      hello.go
     </code>
     的目录，并运行
     <code>
      hello.go
     </code>
     文件。
    </p>
    <pre><code class="prism language-java">$ cd <span class="token operator">~</span><span class="token operator">/</span>GOPATH<span class="token operator">/</span>hello  <span class="token comment">// 切换到`hello.go`的目录</span>
$ go run hello<span class="token punctuation">.</span>go    <span class="token comment">// 运行`hello.go`文件</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/94e50d8fd8ffe2967878bcea6716b929.png">
      <br/>
      ④ 也可以使用
      <code>
       go build
      </code>
      编译go文件的方式执行程序。
     </img>
    </p>
    <pre><code class="prism language-python">$ cd <span class="token operator">~</span><span class="token operator">/</span>GOPATH<span class="token operator">/</span>hello <span class="token comment"># 切换到`hello.go`的目录</span>
$ go build    <span class="token comment"># 编译`hello.go`文件，得到一个名为hello的可执行文件</span>
$ <span class="token punctuation">.</span><span class="token operator">/</span>hello     <span class="token comment"># 执行名为hello的可执行文件</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bde85423ebbe62ebf9f7b5819e7ca382.png">
      <br/>
      至此，go语言已经成功安装！
     </img>
    </p>
    <h5>
     <a id="4__85">
     </a>
     4. 使用过程中的发现
    </h5>
    <ul>
     <li>
      最开始，自己一直不明白为什么需要创建
      <code>
       GOPATH
      </code>
      目录，后来在编译别人的go项目时，别人的比编译命令如下：
     </li>
    </ul>
    <pre><code class="prism language-go">echo <span class="token string">"building orderer..."</span>     # 终端显示的内容
<span class="token keyword">go</span> install madledger<span class="token operator">/</span>orderer   # 安装某个模块
</code></pre>
    <p>
     这时会报错,说不能在
     <code>
      ${GOPATH}/src
     </code>
     和
     <code>
      ${GOROOT}/src
     </code>
     下面找到对应的包。
    </p>
    <pre><code class="prism language-go">building orderer<span class="token operator">...</span>
can't load <span class="token keyword">package</span><span class="token punctuation">:</span> <span class="token keyword">package</span> madledger<span class="token operator">/</span>orderer<span class="token punctuation">:</span> cannot find <span class="token keyword">package</span> <span class="token string">"madledger/orderer"</span> in any of<span class="token punctuation">:</span>
	<span class="token operator">/</span>usr<span class="token operator">/</span>local<span class="token operator">/</span><span class="token keyword">go</span><span class="token operator">/</span>src<span class="token operator">/</span>madledger<span class="token operator">/</span>orderer <span class="token punctuation">(</span>from $GOROOT<span class="token punctuation">)</span>
	<span class="token operator">/</span>home<span class="token operator">/</span>hadoop<span class="token operator">/</span>GOPATH<span class="token operator">/</span>src<span class="token operator">/</span>madledger<span class="token operator">/</span>orderer <span class="token punctuation">(</span>from $GOPATH<span class="token punctuation">)</span>
</code></pre>
    <ul>
     <li>
      <p>
       自己恍然大悟，原来go命令会自动到
       <code>
        ${GOPATH}/src
       </code>
       或
       <code>
        ${GOROOT}/src
       </code>
       下面去寻找对应的包。所以自己将
       <code>
        madledger
       </code>
       拷贝到
       <code>
        ${GOPATH}/src
       </code>
       目录下，再运行编译名命令。执行结果如下：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/57b91bb7a68bfd888c8cef1ed7309521.png"/>
      </p>
     </li>
     <li>
      <p>
       而且如果是install命令，还会在
       <code>
        ${
        <!-- -->
       </code>
       GOPATH
       <code>
        }/bin
       </code>
       或
       <code>
        ${
        <!-- -->
       </code>
       GOROOT
       <code>
        }/bin
       </code>
       下面安装对应的go可执行文件。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7be28142458e8347cdbca62d25971293.png"/>
      </p>
     </li>
     <li>
      <p>
       所以，如果想直接在任何目录编译go文件，可以将go文件放到src目录下，他会自动去寻找相应的包。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/234d285a57d5ee8d44bc10105fe8f658.png"/>
      </p>
     </li>
     <li>
      <p>
       <code>
        go build
       </code>
       命令会在当前目录生成对应的可执行文件。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c1f803008309f299a1b5f9e6b9000643.png"/>
      </p>
     </li>
    </ul>
