    <h3>
     <a id="_1">
     </a>
     前言
    </h3>
    <p>
     由于最近项目做到了需求前端判别用户使用的不同语言并且进行自动切换语言，所以想着记录一下。
    </p>
    <h4>
     <a id="_3">
     </a>
     一、环境准备
    </h4>
    <p>
     我们需要准备开发环境，我这里使用vue-cli官方脚手架搭建了一个vue的开发环境。
    </p>
    <h4>
     <a id="_vuei18n_5">
     </a>
     二、安装 vue-i18n
    </h4>
    <p>
     我这里使用npm安装依赖
    </p>
    <pre><code>$ npm install vue-i18n
</code></pre>
    <h4>
     <a id="_vuei18n_10">
     </a>
     三、引入 vue-i18n
    </h4>
    <p>
     这里我是在main.js中引入vue-i18n
    </p>
    <pre><code class="prism language-javascript"><span class="token comment">//多语言切换</span>
<span class="token keyword">import</span> VueI18n <span class="token keyword">from</span> <span class="token string">'vue-i18n'</span>

  <span class="token comment">// locale: '', // 语言标识</span>

    <p>
     这时我们可以全局调用这个i18n的实例对象，通过改变this.$i18n.locale的值来改变语言，但这时我们还没有语言切换的条件以及语言包
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/11b7b961b3d2d59e886a79f190d1fe86.png">

      此时我们再创建一个lang语言文件夹,在lang中创建我们需要的语言包
     </img>
    </p>

     <a id="_30">
