    消息人士透露，微软与雅虎打破之前的僵局，正在进行非正式合并谈判。
    <br/>
    <br/>
    自微软2月1日以每股31美元的价格提出洽购要约以来，该事件已经发生了很多变化。
    <br/>
    <br/>
    消息人士称，雅虎已经表现出了与微软进行对话和谈判的意愿。他强调，微软总部已经得出结论，雅虎可能永远不会正式拒绝微软。但这些非正式谈判能否达成交易尚未可知。
    <br/>
    <br/>
    最近一系列的事件令微软CEO鲍尔默数周前在德国汉诺威信息技术及通信展览会(CeBIT)上发表的言论更加明朗化。鲍尔默曾于该会议上声称，微软和雅虎会就被推迟的收购展开一系列对话。
    <br/>
    <br/>
    但是，另一接近该交易的人士警告称，如果双方之间的谈判已经到了实质性阶段，那么微软应该会予以公开披露。
    <br/>
    <br/>
    消息人士称，在洽购雅虎的进程上，微软有不少挫折。不过，这是不可避免的，毕竟这是考验耐力的长跑而不是短跑比赛。
