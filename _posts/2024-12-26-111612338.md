    <p>
     每一个版本的Eclipse开发平台软件需要的JAVA  JDK的版本不同，需要根据IBM公司公布的对应关系配对，才能开发Android项目。下面是IBM公司公布的对应关系
    </p>
    <p>
     链接为：
     <a href="http://wiki.eclipse.org/Eclipse/Installation" rel="nofollow">
      http://wiki.eclipse.org/Eclipse/Installation
     </a>
    </p>
    <p>
    </p>
    <p>
     Installing Eclipse is relatively easy, but does involve a few steps and software from at least two different sources.
     <strong>
      Eclipse is a Java-based application
     </strong>
     and, as such,
     <em>
      <strong>
       requires a Java Runtime Environment or Java Development Kit (JRE or JDK)
      </strong>
     </em>
     in order to run.
    </p>
    <p>
     Note that on recent versions of Mac,
     <strong>
      a full JDK needs to be installed
     </strong>
     , not just a JRE; see
     <a href="http://wiki.eclipse.org/Eclipse/Installation#Install_a_JVM" rel="nofollow">
      instructions
     </a>
     below.
    </p>
    <pre><code class="language-bash">1 Install a JVM
2 Eclipse 4.18 (2020-12)
3 Eclipse 4.17 (2020-09)
4 Eclipse 4.16 (2020-06)
5 Eclipse 4.15 (2020-03)
6 Eclipse 4.14 (2019-12)
7 Eclipse 4.13 (2019-09)
8 Eclipse 4.12 (2019-06)
9 Eclipse 4.11 (2019-03)
10 Eclipse 4.10 (2018-12)
11 Eclipse 4.9 (2018-09)
12 Eclipse 4.8 (Photon)
13 Eclipse 4.7 (Oxygen)
14 Eclipse 4.6 (Neon)
15 Eclipse 4.5 (Mars)
16 Eclipse 4.4 (Luna)
17 Eclipse 4.3 (Kepler)
18 JRE/JDK Sources
19 Download Eclipse
20 Configure Eclipse to use the JVM
21 Extending Eclipse
22 Troubleshooting
23 More information</code></pre>
    <h3>
     Eclipse 4.18 (2020-12)
    </h3>
    <p>
     <a href="https://projects.eclipse.org/projects/eclipse/releases/4.18.0" rel="nofollow">
      Eclipse 4.18
     </a>
     (
     <a href="https://wiki.eclipse.org/Category:SimRel-2020-12" rel="nofollow">
      2020-12
     </a>
     ) was released on December 16, 2020. It is the supported release.
    </p>
    <p>
     A
     <a href="http://wiki.eclipse.org/Eclipse/Installation#Install_a_JDK" rel="nofollow">
      <strong>
       Java 11
      </strong>
      or newer JRE/JDK is required, LTS release are preferred
     </a>
     to run all Eclipse 2020-12 packages based on Eclipse 4.18, with certain packages choosing to provide one by default. The Installer now includes a JRE. Consider using the Installer. Please see
     <a href="https://www.eclipse.org/downloads/eclipse-packages/?show_instructions=TRUE" rel="nofollow">
      5 Steps to Install Eclipse
     </a>
     .
    </p>
