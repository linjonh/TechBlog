     刚才从
     <a href="http://www.ams.org/news/home-news.html#ito" rel="nofollow">
      AMS网站
     </a>
     看到，概率论先驱，最伟大的日本数学家之一，沃尔夫奖、高斯奖和京都奖得主，日本学士院院士
     <span class="Apple-style-span" style="font-weight: bold; font-size: 12pt;">
      伊藤清
     </span>
     先生已于2008年11月10日去世，享年93岁。
     <div>
      <br/>
     </div>
     <div>
      <br/>
     </div>
     <div>
      伊藤清在就读于东京帝国大学期间，迷上了概率论。当时世界上研究概率的数学家非常少，在前苏联有大师柯尔莫哥洛夫，在法国有莱维，在美国有杜布、费勒等。1938年毕业，在内阁统计局工作。当时正处于二战期间，他得以有大量时间专心研究。在维纳（控制论之父）对随机过程研究的基础上，伊藤清创立了随机分析理论。但当时学术交流基本中断，他的成果几乎完全是独立完成的。
     </div>
     <div>
      <br/>
     </div>
     <div>
      1943年他成为名古屋帝国大学助教授。1945年被授予博士学位。战后他继续发表了一系列重要论文。1950年成为京都大学教授，1979年退休。
     </div>
     <div>
      <br/>
     </div>
     <div>
      他的研究对其他学科尤其是金融数学产生了深远影响。有个逸闻说，因创立金融衍生工具基础模型而获诺贝尔奖的斯科尔斯有一次见到伊藤，像个追星族一样跑上前去握手致意。这样说来，伊藤和次贷危机还有些联系呢。
      <br/>
      <br/>
      他的重要著作包括《概率论》和《随机过程》都即将由人民邮电出版社图灵公司引进翻译出版。
