    <p>
     <strong>
      HarmonyOS（鸿蒙操作系统）
     </strong>
     作为一种面向全场景多设备的智能操作系统，正逐渐在市场上崭露头角。为了进入
     <strong>
      HarmonyOS
     </strong>
     生态，开发者需要搭建一个高效的开发环境。本文将详细介绍如何搭建
     <strong>
      HarmonyOS
     </strong>
     开发环境，特别是如何安装和配置
     <strong>
      DevEco Studio
     </strong>
     。
    </p>
    <h2>
     前置条件
    </h2>
    <p>
     在开始搭建开发环境之前，请确保你的计算机满足以下要求：
    </p>
    <blockquote>
     <ul>
      <li>
       <strong>
        操作系统
       </strong>
       ：
       <ul>
        <li>
         Windows 10/11 64位
        </li>
        <li>
         macOS (X86) 10.15及以上版本，macOS (ARM) 11及以上版本
        </li>
       </ul>
      </li>
      <li>
       <strong>
        内存
       </strong>
       ：至少8GB，推荐16GB
      </li>
      <li>
       <strong>
        硬盘
       </strong>
       ：至少100GB可用空间
      </li>
      <li>
       <strong>
        分辨率
       </strong>
       ：1280*800像素及以上
      </li>
     </ul>
    </blockquote>
    <h2>
     1.下载与安装DevEco Studio
    </h2>
    <p>
     <strong>
      DevEco Studio
     </strong>
     是基于
     <strong>
      IntelliJ IDEA Community
     </strong>
     开源版本打造，为
     <strong>
      HarmonyOS
     </strong>
     和
     <strong>
      OpenHarmony
     </strong>
     系统上的应用和服务提供一站式的开发平台。
    </p>
    <h3>
     1.1.下载
     <strong>
      DevEco Studio
     </strong>
    </h3>
    <ul>
     <li>
      访问华为开发者联盟官网：
      <a href="https://developer.harmonyos.com/" rel="nofollow" title="https://developer.harmonyos.com">
       https://developer.harmonyos.com
      </a>
     </li>
    </ul>
    <p>
     <img alt="" height="954" src="https://i-blog.csdnimg.cn/direct/e5765bf4fa024eed86981d934b9bf5f6.png" width="1200"/>
    </p>
    <ul>
     <li>
      进入“开发”板块，选择“下载DevEco Studio”
     </li>
    </ul>
    <p>
     <img alt="" height="954" src="https://i-blog.csdnimg.cn/direct/03a7c900561c4e8abaa7a1b35fe0c3a0.png" width="1200"/>
    </p>
    <ul>
     <li>
      这里根据你自己的操作系统选择对应的版本 ( 博主这里是Windows 11 64位 操作系统 )
     </li>
    </ul>
    <p>
     <img alt="" height="954" src="https://i-blog.csdnimg.cn/direct/24cf7df88b054bdfac4ea32fc5a2bd0e.png" width="1200"/>
    </p>
    <h3>
     1.2.
     <strong>
      安装DevEco Studio
     </strong>
    </h3>
    <blockquote>
     <h3>
     </h3>
     <ul>
      <li>
       <strong>
        Windows系统
       </strong>
       ：
       <ol>
        <li>
         解压缩包
         <strong>
          devecostudio-windows-5.0.5.200.zip
         </strong>
        </li>
        <li>
         双击下载的安装包运行（或右键以管理员身份运行）
        </li>
        <li>
         选择安装路径（建议安装在非系统盘）
        </li>
        <li>
         勾选需要安装的组件，点击“安装”直到完成
        </li>
       </ol>
      </li>
      <li>
       <strong>
        macOS系统
       </strong>
       ：
       <ol>
        <li>
         双击下载的“deveco-studio-xxxx.dmg”软件包
        </li>
        <li>
         在安装界面中，将“DevEco-Studio.app”拖拽到“Applications”中，等待安装完成
        </li>
       </ol>
      </li>
     </ul>
    </blockquote>

    <h2>
     2.配置开发环境
    </h2>
    <h3>
     2.1.
     <strong>
      运行DevEco Studio
     </strong>
    </h3>
    <p>
     <img alt="" height="291" src="https://i-blog.csdnimg.cn/direct/572b37ea5ba0436ebb089e51b2a86010.png" width="717"/>
    </p>
    <blockquote>
     <p>
      首次运行时，我们选择
      <strong>
       “Do not import settings”
      </strong>
      ，点击
      <strong>
       “OK”
      </strong>
     </p>
    </blockquote>
    <p>
     <img alt="" height="493" src="https://i-blog.csdnimg.cn/direct/ae72e69937bf4f399f2ff5cfb87faad3.png" width="755"/>

    <blockquote>
     <p>
      好，这里接受

       “防沉迷”

      条款 ，接受了你就可以开始使用
      <strong>
       DevEco Studio
      </strong>
      来开发你的第一款
      <strong>
       HarmonyOS
      </strong>
      应用了
     </p>
    </blockquote>
    <h2>
     3.创建第一个项目
    </h2>
    <p>
     安装完成后，可以开始创建第一个
     <strong>
      HarmonyOS
     </strong>
     项目：
    </p>
    <h3>
     3.1.启动
     <strong>
      DevEco Studio
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       如果是第一次启动DevEco Studio，点击 created project 创建项目
      </strong>
     </li>
     <li>
      <strong>
       如果已经进入到DevEco Studio的开发界面，在右上角选择“File”&gt;“New”&gt;“New Project”
      </strong>
     </li>
    </ul>
    <p>
     <img alt="" height="545" src="https://i-blog.csdnimg.cn/direct/b18f87aca24f4b229bd671b35df056ce.png" width="696"/>
    </p>
    <h3>
     <img alt="" height="663" src="https://i-blog.csdnimg.cn/direct/bbb702fec78c474cbe8d773be55f3f81.png" width="996"/>
     3.2.选择模板
    </h3>
    <p>
     选择
     <strong>
      Application
     </strong>
     应用开发，选择
     <strong>
      “Empty Ability”
     </strong>
     ,点击
     <strong>
      Next
     </strong>
     进行下一步配置。如果你需要需要进行
     <strong>
      Native
     </strong>
     相关工程的开发，请选择“
     <strong>
      Native C++
     </strong>
     ”模板，更多的模板详情和介绍，请参考
     <a class="link-info" href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V13/ide-template-V13" rel="nofollow" title="华为Harmony OS开发指南 &gt; 工程模板介绍">
      华为Harmony OS开发指南 &gt; 工程模板介绍
     </a>
    </p>
    <h3>
     3.3.配置工程
    </h3>
    <h3>
     <img alt="" height="663" src="https://i-blog.csdnimg.cn/direct/5d8858cef46147b1bc6b71526a155973.png" width="996"/>
    </h3>
    <blockquote>
     <ul>
      <li>
       <strong>
        Compatible SDK
       </strong>
       选择“
       <strong>
        5.0.1(13)
       </strong>
       ”，其他参数保持默认即可
      </li>
     </ul>
     <p>
      <strong>
       【注意】：Bundle name
      </strong>
      是由
      <strong>
       “.”
      </strong>
      分割的
      <strong>
       3个部分
      </strong>
      共同组成
     </p>
    </blockquote>
    <p>
     点击
     <strong>
      Finish
     </strong>
     ，工具会自动生成示例代码和相关资源，等待工程创建完成。
    </p>
    <h3>
     3.4.项目结构 [ ArkTS工程目录结构（Stage模型）]
    </h3>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="567" src="https://i-blog.csdnimg.cn/direct/78a94c98d7ac488cb4c512df3bc24c53.png" width="356"/>
    </p>
    <blockquote>
     <p>
     </p>
     <ul>
      <li>
       <strong>
        AppScope &gt; app.json5
       </strong>
       ：应用的全局配置信息，详见
       <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/app-configuration-file-V5" rel="nofollow" title="app.json5配置文件">
        app.json5配置文件
       </a>
       。
      </li>
      <li>
       <strong>
        entry
       </strong>
       ：HarmonyOS工程模块，编译构建生成一个HAP包。
       <ul>
        <li>
         <strong>
          src &gt; main &gt; ets
         </strong>
         ：用于存放ArkTS源码。
        </li>
        <li>
         <strong>
          src &gt; main &gt; ets &gt; entryability
         </strong>
         ：应用/服务的入口。
        </li>
        <li>
         <strong>
          src &gt; main &gt; ets &gt; entrybackupability
         </strong>
         ：应用提供扩展的备份恢复能力。
        </li>
        <li>
         <strong>
          src &gt; main &gt; ets &gt; pages
         </strong>
         ：应用/服务包含的页面。
        </li>
        <li>
         <strong>
          src &gt; main &gt; resources
         </strong>
         ：用于存放应用/服务所用到的资源文件，如图形、多媒体、字符串、布局文件等。关于资源文件，详见
         <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/resource-categories-and-access-V5" rel="nofollow" title="资源分类与访问">
          资源分类与访问
         </a>
         。
        </li>
        <li>
         <strong>
          src &gt; main &gt; module.json5
         </strong>
         ：模块配置文件。主要包含HAP包的配置信息、应用/服务在具体设备上的配置信息以及应用/服务的全局配置信息。具体的配置文件说明，详见
         <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/module-configuration-file-V5" rel="nofollow" title="module.json5配置文件">
          module.json5配置文件
         </a>
         。
        </li>
        <li>
         <strong>
          build-profile.json5
         </strong>
         ：当前的模块信息 、编译信息配置项，包括buildOption、targets配置等。
        </li>
        <li>
         <strong>
          hvigorfile.ts
         </strong>
         ：模块级编译构建任务脚本。
        </li>
        <li>
         <strong>
          obfuscation-rules.txt
         </strong>
         ：混淆规则文件。混淆开启后，在使用Release模式进行编译时，会对代码进行编译、混淆及压缩处理，保护代码资产。详见
         <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/ide-build-obfuscation-0000001731754312-V5" rel="nofollow" title="开启代码混淆">
          开启代码混淆
         </a>
         。
        </li>
        <li>
         <strong>
          oh-package.json5
         </strong>
         ：用来描述包名、版本、入口文件（类型声明文件）和依赖项等信息。
        </li>
       </ul>
      </li>
      <li>
       <strong>
        oh_modules
       </strong>
       ：用于存放三方库依赖信息。
      </li>
      <li>
       <p>
        <strong>
         build-profile.json5
        </strong>
        ：工程级配置信息，包括签名signingConfigs、产品配置products等。其中products中可配置当前运行环境，默认为HarmonyOS。
       </p>
      </li>
      <li>
       <p>
        <strong>
         hvigorfile.ts
        </strong>
        ：工程级编译构建任务脚本。
       </p>
      </li>
      <li>
       <strong>
        oh-package.json5
       </strong>
       ：主要用来描述全局配置，如：依赖覆盖（overrides）、依赖关系重写（overrideDependencyMap）和参数化配置（parameterFile）等。
      </li>
     </ul>
    </blockquote>
