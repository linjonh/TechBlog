       定义
       <br/>
       可以用两种方式给予解释：
       <br/>
       （1）离散概率
       <br/>
       试验中每次可能的结果乘以其结果概率的总和。
       <br/>
       （2）连续分布函数
       <br/>
       如果X是在概率空间（Ω,P）中的随机变量，那么它的期望值E[X]的定义是：
       <br/>
       <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4864ec4c99b7c083ed823c00e204b3a8.png" title="">
        <br/>
        F-分布函数 并不是每一个随机变量都有期望值的，因为有的时候这个积分不存在。
        <br/>
        （3）抽样
        <br/>
        随机试验在同样的机会下重复多次，所有那些可能状态平均的结果。
       </img>
      </p>
     </li>
     <li>
      <p>
       别名
       <br/>
       数学中：数学期望值、数学期望、均值、期望。
       <br/>
       物理学中：期待值。
      </p>
     </li>
     <li>
      <p>
       实际使用
       <br/>
       （1）统计学
       <br/>
       重复测量此变量的值，然后用所得数据的平均值来作为此变量的期望值的估计。
       <br/>
       （2）概率分布
       <br/>
