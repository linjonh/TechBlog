     大致查阅了现阶段使用较多的几种方案,，大概有以下几种方式：
    </p>
    <p>
     <span style="color:#a2e043;">
      一、原生window.print()方法导出pdf
      <br/>
      二、
      <a class="link-info" href="https://rawgit.com/MrRio/jsPDF/master/docs/jsPDF.html" rel="nofollow" title="jspdf">
       jspdf
      </a>
      <br/>
      三、jspdf +
      <a class="link-info" href="https://sg-info.cn/article/show/6705" rel="nofollow" title="html2canvas">
       html2canvas
      </a>
      <br/>
      四、pdfmake
     </span>
    </p>

       <td>

         1、兼容性最好
         <br/>
         2、可以将任意内容导出成 pdf 文档, 甚至是非改页面上的内容
        </span>
       </td>
       <td>
        <span style="color:#0d0016;">
         1、调用方法时部分条件下导出pdf需要用户手动选择
        </span>
        <br/>
       </td>
      </tr>
      <tr>
       <td>
        <span style="color:#0d0016;">
         jspdf
        </span>
       </td>
       <td>
        <p>
         <span style="color:#0d0016;">
          1. 调用方法创建 pdf 文件,对于简单的内容可直接使用，
          <br/>
          2. 也可以将 dom 节点转换为 pdf
         </span>
        </p>
        <p>
         <span style="color:#0d0016;">
          3、生成内容为文本可复制

         <br/>
         3、如果想要导出的pdf文件中包含图片，视觉效果略微模糊
         <br/>
         4、pdf分页不好处理
        </span>
        <br/>
       </td>

       <td>

        <span style="color:#0d0016;">
         1、在jspdf上将生成效果不佳的部分可以转成图片，适用于对样式有要求的场景
         <br/>
         2、将乱码部分转为了图片，解决了中文乱码问题
         <br/>
         3、没有预览点击即可保存
        </span>
       </td>
       <td>
        <p>
         <span style="color:#0d0016;">

          <br/>
          2、生成的pdf实际为图片，不支持复制
          <br/>
          3、不同浏览器生成可能会有略微差异（页面周边留白部分差异）
          <br/>
          4、由于整体效果为图片，导致pdf文件较大（两页2.5MB左右）
         </span>
        </p>

         <span style="color:#0d0016;">
          5、pdf分页不好处理

     方案二：

    <ul>
     <li>
      <p>
       <strong>
        sPDF：
       </strong>
       用于在浏览器中生成PDF文件。你可以在Vue项目中使用它，通过引入jsPDF库并编写相应的代码来生成PDF。
      </p>
     </li>
     <li>
      <pre><code>npm install jspdf</code></pre>
      <pre><code class="language-javascript">import jsPDF from 'jspdf';

// 创建一个 PDF 实例
const pdf = new jsPDF();

// 添加内容
pdf.text('Hello world!', 10, 10);

// 保存为 PDF 文件
pdf.save('example.pdf');
</code></pre>
<ul>
<li>
<p>
<strong>
html2pdf：
</strong>
这个库可以将 HTML 内容转换为 PDF。它支持更复杂的页面结构和样式。
</p>
<pre><code class="language-javascript">npm install html2pdf.js</code></pre>
<pre><code class="language-javascript">import html2pdf from 'html2pdf.js';

// 获取要转换为 PDF 的 DOM 元素
const element = document.getElementById('content');

// 转换为 PDF
html2pdf(element);
</code></pre>
<p>
在 vue 中基础使用
</p>
<pre><code class="language-javascript">&lt;template&gt;
&lt;div&gt;
&lt;button @click="generatePDF"&gt;Generate PDF&lt;/button&gt;
&lt;/div&gt;
&lt;/template&gt;

&lt;script&gt;
import jsPDF from 'jspdf';

export default {
methods: {
generatePDF() {
// 创建一个 PDF 实例
const pdf = new jsPDF();

      // 添加内容
      pdf.text('Hello world!', 10, 10);

       // 添加图像
       pdf.addImage('path/to/image.jpg', 'JPEG', 10, 10, 50, 50);

        // 添加矩形
        pdf.rect(10, 10, 50, 30);

        // 添加圆形
        pdf.circle(80, 25, 20);

        // 添加文本和链接
        pdf.textWithLink('Click me', 10, 10, { url: 'https://example.com' });


      // 保存为PDF文件
      pdf.save('example.pdf');
    },

},
};
&lt;/script&gt;</code></pre>
<p>
还可以配合添加表格

使用
<code>
autoTable

插件可以方便地添加表格。这个插件需要额外安装
</p>
<pre><code class="language-javascript">npm install jspdf-autotable

<pre><code class="language-javascript">import 'jspdf-autotable';

// 添加表格

</code></pre>
<p>
还可以配合 echart 添加图表，本质是把 echart 图表转了图片
<br/>
</p>
<pre><code class="language-javascript">&lt;template&gt;
&lt;div&gt;
&lt;canvas ref="chartCanvas"&gt;&lt;/canvas&gt;
&lt;button @click="generatePDF"&gt;Generate PDF with Chart&lt;/button&gt;
&lt;/div&gt;
&lt;/template&gt;

&lt;script&gt;
import Chart from 'chart.js';
import jsPDF from 'jspdf';

export default {
data() {
return {
chartData: {
labels: ['Label 1', 'Label 2', 'Label 3'],
datasets: [
{
label: 'Sample Data',
data: [10, 20, 15],
backgroundColor: ['red', 'blue', 'green'],
},
],
},
};
},
mounted() {
// 使用 Chart.js 创建柱状图
const ctx = this.$refs.chartCanvas.getContext('2d');
    new Chart(ctx, {
      type: 'bar',
      data: this.chartData,
    });
  },
  methods: {
    generatePDF() {
      // 获取图表的Base64编码
      const chartImage = this.$refs.chartCanvas.toDataURL('image/png');

      // 创建一个PDF实例
      const pdf = new jsPDF();

      // 添加图像
      pdf.addImage(chartImage, 'PNG', 10, 10, 80, 50);

      // 保存为PDF文件
      pdf.save('example_with_chart.pdf');
    },

},
};
&lt;/script&gt;
</code></pre>

<li>
<p>
<span style="color:#0d0016;">
<code>
pdfmake
</code>
是一个用于在浏览器中生成 PDF 文档的 JavaScript 库。它允许你使用简单的 JavaScript 对象来定义文档的结构和内容，而不是直接使用 HTML 或其他标记语言。这样可以更轻松地生成复杂的、自定义的 PDF 文档。
</span>
</p>
<p>
<span style="color:#0d0016;">
以下是
<code>
pdfmake
</code>
的一些关键特点和概念：
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
声明式文档定义：
</strong>
<code>
pdfmake
</code>
使用声明式的方式定义 PDF 文档的结构和内容。你通过 JavaScript 对象来描述文档的各个部分，如页眉、页脚、段落、表格等。
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
支持丰富的样式：
</strong>
你可以为文本、段落、表格等元素指定各种样式，包括字体、颜色、大小、对齐方式等。这使得你能够创建具有良好格式和外观的 PDF 文档。
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
支持表格：
</strong>
<code>
pdfmake
</code>
提供了强大的表格支持，允许你创建具有复杂结构和样式的表格。你可以设置单元格的边框、背景颜色、合并单元格等。
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
支持图片：
</strong>
你可以将图片添加到 PDF 文档中，支持从 URL 加载图片或使用 Base64 编码的图片数据。
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
支持页面布局：
</strong>
<code>
pdfmake
</code>
允许你设置页面的大小、方向和边距，以满足不同的打印和布局需求。
</span>
</p>
</li>
<li>
<p>
<span style="color:#0d0016;">
<strong>
支持中文和多语言：
</strong>
<code>
pdfmake
</code>
对于国际化支持相当友好，包括对各种语言和字体的支持。
</span>
<br/>
</p>
<pre><code class="language-javascript">// 引入 pdfmake 库
import pdfMake from 'pdfmake/build/pdfmake';
import pdfFonts from 'pdfmake/build/vfs_fonts';

// 注册字体
pdfMake.vfs = pdfFonts.pdfMake.vfs;

// 创建文档定义
const documentDefinition = {
content: [
{ text: 'Hello, PDFMake!', fontSize: 16, bold: true, alignment: 'center', margin: [0, 0, 0, 10] },
{
table: {
body: [
['Name', 'Age', 'Country'],
['John Doe', 25, 'USA'],
['Jane Doe', 30, 'Canada'],
],
},
},
{ image: 'data:image/jpeg;base64,...', width: 100, height: 100 },
],
};

// 创建 PDF 文档
const pdfDocGenerator = pdfMake.createPdf(documentDefinition);

// 下载 PDF 文件
pdfDocGenerator.download('example.pdf');
</code></pre>
