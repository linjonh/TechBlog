    <font color="#0000ff">
     Windows Mobile讲座笔记
    </font>
    <div class="bvMsg" id="msgcns!a62bf721067c2f56!202">
     <div>
      Windows Mobile校园开放日是由信息产业部CSIP携手微软（中国）有限公司联合微软高校关系部联合推出的普及Windows Mobile技术的大型校园活动，希望推广Windows Mobile技术，发现Windows Mobile人才。4月29日下午在理综楼303拉开帷幕。
     </div>
     <div>
      那天下午去晚了，结果得到了坐在两排座位中间小马扎的荣幸，看来微软还是有蛮强的召唤力的。由于抄写了任彬同学的答卷，本人得到了一个Windows Mobile的包和帽子，在此表示感谢zZ~~
     </div>
     <div>
      首先是微软中国技术中心（CTC）硬件创新实验室的Group Manager（微软公司数十年工作经验，从微软台湾到微软北京，微软资深管理人员） 赵靖宇发言，他概念性的给我们分类了
      <font color="#0000ff">
       创意style,共有4类：
      </font>
     </div>
     <ol>
      <li>
       Modifying(improve what we have one step at a time)
      </li>
      <li>
       Experimenting(combining ideas)
      </li>
      <li>
       Exploring(questioning assumption)
      </li>
      <li>
       Visioning(focus on ideal solutio,let goal be the guide)
      </li>
     </ol>
     <div>
      然后就是MII-MS（信息产业部-微软）嵌入式软件实验室工程师（信息产业部软件与集成电路促进中心，嵌入式与Windows Mobile资深工程师）李静 给我们从大方面介绍了Windows Mobile：
     </div>
     <ul>
      <li>
       Windows Mobile官方网站：
       <a href="http://www.winbile.net/" rel="nofollow">
        <font color="#8df381">
         http://www.winbile.net
        </font>
       </a>
      </li>
      <li>
       嵌入式系统开发（交叉编译）和PC开发
      </li>
      <li>
       主要的开发平台有：
      </li>
     </ul>
     <ol>
      <li>
       <font color="#0000ff">
        Windows Mobile Pocket PC
       </font>
      </li>
      <li>
       <font color="#0000ff">
        Windows Mobile Smartphone
       </font>
      </li>
      <li>
       <font color="#0000ff">
        Wondows Mobile Center
       </font>
      </li>
     </ol>
     <p>
      以上三种都支持嵌入式数据库,其中smart phone和pocket pc的区别就是smart phone用的是键盘（keyboard），而pocket pc用的是触摸屏（touching screen）,有数据处理软件（office mobile等等）
     </p>
     <ul>
      <li>
       Windows Mobile最新版本：Windows Mobile 5.0
      </li>
      <li>
       开发工具：Visual Studio 2005
      </li>
      <li>
       集成开发环境：
      </li>
     </ul>
     <ol>
      <li>
       VS 2005(支持所有应用程序开发)
      </li>
      <li>
       VS .NET 2003,Embedded Visual Tools 3
      </li>
      <li>
       Embedded VC++ 4.0.
      </li>
     </ol>
     <ul>
      <li>
       基于Windows Mobile设备应用程序的代码类型有：
      </li>
     </ul>
     <ol>
      <li>
       <font color="#0000ff">
        本地代码(C,C++等)（直接执行）
       </font>
      </li>
      <li>
       <font color="#0000ff">
        托管代码(C#,VB.NET,.NE CF等)（跨平台）
       </font>
      </li>
      <li>
       <font color="#0000ff">
        服务器端代码(JS等)
       </font>
      </li>
     </ol>
     <ul>
      <li>
       VS 2005全新特点：
      </li>
     </ul>
     <ol>
      <li>
       真正的ARM Emulator
      </li>
      <li>
       共享目录(Folder Sharing)
      </li>
      <li>
       保存目录(Save State)
      </li>
      <li>
       同步(ActiveSync)
      </li>
      <li>
       屏幕旋转(Screen Rotation)
      </li>
      <li>
       串口映射(Serial Port Mappig)
      </li>
     </ol>
     <ul>
      <li>
       如何搭建Windows Mobile开发环境：
      </li>
     </ul>
     <ol>
      <li>
       Visual Studio 2005
      </li>
      <li>
       Windows Mobile 5.0 SDK
      </li>
      <li>
       ActiveSync
      </li>
     </ol>
     <ul>
      <li>
       C#'s shortage:编写组件必须用C++去实现，而C#只是用于封装组件。
      </li>
      <li>
       要使用C++，C#或Visual Basic.NET为Smartphone 2003开发应用程序，必须安装Smartphone 5.0 SDK,VS 2005.
      </li>
      <li>
       Smartphone 2003支持.NET Compact Framework.
      </li>
      <li>
       本地代码和托管代码的区别是：本地代码要考虑硬件兼容问题，而托管代码不需要。Web Service Server要用托管代码来实现。
      </li>
      <li>
       Windows Mobile GDI比GAPI写屏效率高。
      </li>
      <li>
       Cell phone的程序：
      </li>
     </ul>
     <ol>
      <li>
       <font color="#00ff00">
        Windows Mobile:Smartphone
       </font>
      </li>
      <li>
       <font color="#00ff00">
        Linux:MOTO
       </font>
      </li>
      <li>
       <font color="#00ff00">
        Symbian:NOKIA
       </font>
      </li>
      <li>
       <font color="#00ff00">
        BREW:CDMA
       </font>
      </li>
     </ol>
     <p>
      <font color="#00ff00">
       最后由微软中国技
      </font>
      术中心（CTC）硬件创新实验室工程师（Windows Mobile技术专家、C++技术专家）张汝颇给我们更为深入地介绍了Windows Mobile的编程问题。
     </p>
     <p>
      最后的最后就是合影啦，感觉好好喔
     </p>
