     软件工程 之 概述
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        软件工程是一门研究用工程化方法构建和维护有效的、实用的和高质量的软件的学科。它涉及到程序设计语言
       </span>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        、数据库
       </span>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        、软件开发工具
       </span>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        、系统平台、标准、设计模式
       </span>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        等方面。在现代社会中，软件应用于多个方面。典型的软件有电子邮件、嵌入式系统、人机界面、办公套件、操作系统、编译器、数据库、游戏等。同时，各个行业几乎都有计算机软件的应用，如工业、农业、银行、航空、政府部门等。这些应用促进了经济和社会的发展，也提高了工作和生活效率。
       </span>
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        <br/>
       </span>
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       <span style="color:rgb(51,51,51); line-height:24px; text-indent:28px">
        软件工程概述是软件工程这个阶段学习的第一课，按照习惯应该是这门课最提纲挈领的开篇。下面就让我们来看看软件工程概述的内容
       </span>
      </strong>
     </span>
    </p>
    <p>
     <br/>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-size:14px; line-height:24px; text-indent:28px">
      <span style="white-space:pre">
       <span style="font-size:18px; line-height:26px">
        <span style="font-family:KaiTi_GB2312">
         <img alt="" src="https://img-blog.csdn.net/20131208202851640?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcnk1MTM3MDU2MTg=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
          <br/>
         </img>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-size:14px; line-height:24px; text-indent:28px">
      <span style="white-space:pre">
       <span style="font-size:18px; line-height:26px">
        <span style="font-family:KaiTi_GB2312">
         <br/>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       下面就重点给大家介绍一下软件的生存期：
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       1.计划（Planning）
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ①确定要开发软件系统的总目标
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ②给出功能、性能、可靠性以及接口等方面的要求
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ③完成该软件任务的可行性研究
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <br/>
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       2.需求分析（RequirementAnalysis）
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ①对待开发软件提出的需求进行分析并给出详细的定义
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ②编写软件需求说明书
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <br/>
      </strong>
     </span>
    </p>
    <p>
     <span style="text-indent:28px">
      <span style="font-family:KaiTi_GB2312; font-size:24px">
       <strong>
        3.软件设计（SoftwareDesign）
       </strong>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="text-indent:28px">
       </span>
       <span style="white-space:pre">
       </span>
       ①概要设计：把各项需求转换成软件的体系结构。结构中每一组成部分都是意义明确的模块
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ②详细设计：对每个模块要完成的工作进行具体的描述
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <br/>
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       4.程序编写（Coding，Programming）
      </strong>
     </span>
    </p>
    <p>
     <br/>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      <strong>
       5.测试（Testing）
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ①单元测试：查找各模块在功能和结构上存在的问题并加以纠正
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span style="white-space:pre">
       </span>
       ②
       <span lang="zh-CN">
        组装测试：将已测试过的模块按一定顺序组装起来
       </span>
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <strong>
       <span lang="zh-CN">
        <br/>
       </span>
      </strong>
     </span>
    </p>
    <p>
     <span lang="zh-CN">
      <span style="text-indent:28px">
       <span style="font-family:KaiTi_GB2312; font-size:24px">
        <strong>
         6.运行和维护（Runand Maintenance）
        </strong>
       </span>
      </span>
      <span style="color:rgb(51,51,51); line-height:26px; white-space:pre; font-size:18px">
       <strong>
        <br/>
       </strong>
      </span>
     </span>
    </p>
    <p style="text-indent:28px">
     <span style="font-size:18px">
      <span style="font-weight:bold">
       <span style="line-height:26px; white-space:pre; color:rgb(51,51,51)">
        <span lang="zh-CN">
         <span style="color:rgb(51,51,51); font-family:KaiTi_GB2312; font-size:24px; line-height:26px; text-indent:28px; white-space:pre">
          <strong>
           <br/>
          </strong>
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span lang="zh-CN">
      <strong>
       <span style="font-family:KaiTi_GB2312; font-size:24px">
        软件工程的理念使得整个软件开发过程规范化、标准化、甚至模型化，在大型的软件系统开发中更能够体现这些理念的重要性。它们保证了软件开发过程的严谨和一丝不苟，从而保证了软件有好的质量。不仅仅更好的为用户服务，更能够为开发人员在后期的维护和完善工作中有迹可循，减轻工作的负担。
       </span>
      </strong>
     </span>
    </p>
    <p style="text-indent:28px">
