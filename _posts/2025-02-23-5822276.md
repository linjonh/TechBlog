    <p>
     著名开发技术媒体
     <span lang="EN-US">
      DDJ
     </span>
     于近日发布了
     <span lang="EN-US">
      <a href="http://www.drdobbs.com/joltawards/226500145" rel="nofollow noopener noreferrer" target="_blank">
       Jolt
       <span lang="EN-US">
        <span lang="EN-US">
         大奖图书获奖名单
        </span>
       </span>
      </a>
     </span>
     ，《编程之魂：与
     <span lang="EN-US">
      27
     </span>
     位编程语言创始人对话》（
     <span lang="EN-US">
      Masterminds of Programming: Conversations with the Creators of Major Programming Languages
     </span>
     ）获得卓越图书大奖。
    </p>
    <p>
     <strong>
      <span lang="EN-US">
       Jolt
      </span>
     </strong>
     <strong>
      卓越大奖：
     </strong>
     <strong>
      <span lang="EN-US">
       Masterminds of Programming: Conversations with the Creators of Major Programming
      </span>
     </strong>
     <strong>
      <span lang="EN-US">
      </span>
     </strong>
     <strong>
      <span lang="EN-US">
       Languages
      </span>
     </strong>
    </p>
    <p>
     <strong>
      获奖理由
     </strong>
     ：每个程序员都曾有过这样的问题
     <span lang="EN-US">
      “
     </span>
     那些编程语言创始人是如何设计出语言的？
     <span lang="EN-US">
      ”
     </span>
     这本书可以回答你的问题。《编程之魂：与
     <span lang="EN-US">
      27
     </span>
     位编程语言创始人对话》由
     <span lang="EN-US">
      Federico Biancuzzi
     </span>
     及
     <span lang="EN-US">
      Shane Warden
     </span>
     合著，将你带回这些众多的编程语言创作之初，力图揭示背后的设计思想。
     <span lang="EN-US">
      Federico Biancuzzi
     </span>
     和
     <span lang="EN-US">
      Shane Warden
     </span>
     向这些设计者问出了很好的问题，事实上，这些访谈更像是个讨论而不是问答。你对语言的任何困惑都可以从本书中得到营养，也适合对某一编程语言有兴趣的人阅读。
    </p>
    <p align="center">
     <span lang="EN-US">
     </span>
    </p>
    <p>
     <strong>
      出版机构
     </strong>
     ：
     <span lang="EN-US">
      O'Reilly Media
      <strong>
      </strong>
     </span>
     <strong>
      作者
     </strong>
     ：
     <span lang="EN-US">
      Federico Biancuzzi
     </span>
     和
     <span lang="EN-US">
      Shane Warden
     </span>
     。
    </p>
    <p>
     <strong>
      作者简介：
     </strong>
     <span lang="EN-US">
      FedericocBiancuzzi
     </span>
     是位自由职业采访者
     <span lang="EN-US">
      ,
     </span>
     他的采访在
     <span lang="EN-US">
      ONLamp
     </span>
     、
     <span lang="EN-US">
      News Forge
     </span>
     、
     <span lang="EN-US">
      TheRegister
     </span>
     、
     <span lang="EN-US">
      ArsTechnica
     </span>
     等很多网站上在线出版。
    </p>
    <p>
     <span lang="EN-US">
      Shane Warden
     </span>
     是
     <span lang="EN-US">
      O'Reilly
     </span>
     网络（
     <span lang="EN-US">
      O’Reilly Network
     </span>
     ）的技术编辑，专攻程序设计、
     <span lang="EN-US">
      Linux
     </span>
     和开源软件开发。他负责了
     <span lang="EN-US">
      O’Reilly
     </span>
     多本书籍的出版，是《
     <span lang="EN-US">
      Extreme Programming Pocket Guide
     </span>
     》（极限编辑程袖珍手册）一书的作者，该书将极限编程浓缩成简单明了的解释和参考。许多读者说他们为所有客户都买了这本书，以便于向客户解释他们如何工作。
     <span lang="EN-US">
      Ward Cunningham
     </span>
     （极限编程的协同创始人）认为该书是对极限编程实践的最佳阐释。
    </p>
    <p>
     <strong>
      内容简介
     </strong>
     ：本书是
     <span lang="EN-US">
      27
     </span>
     位杰出的设计师与你分享他们的智慧和经验。书中以问答方式告诉为什么要创建某种编程语言、它在技术上如何开发、如何教授和学习，以及它如何顺应时代发展等。你会发现构建成功编程语言所需的思想和步骤，它广受欢迎的原因，以及如何处理程序员常见的问题。因此，如果你想深入学习设计成功编程语言的思想，本书会对你大有帮助。
    </p>
