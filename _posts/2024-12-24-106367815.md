   <div class="htmledit_views" id="content_views">
    <p style="margin-left:0px;">
     <span style="color:#231f20;">
      下面是一个利用Delphi受欢迎的开源项目的示例。
     </span>
    </p>
    <ul style="margin-left:0px;">
     <li>
      <a href="https://github.com/HeidiSQL/HeidiSQL">
       HeidiSQL
      </a>
      ：在Delphi中开发的一种非常流行的用于管理MariaDB，MySQL，SQL Server，PostgreSQL和SQLite的客户端（当然，Delphi使数据库操作非常容易）。
     </li>
     <li>
      <a href="https://github.com/pyscripter/pyscripter.">
       Pyscripter
      </a>
      ：一个非常流行的功能丰富且轻量级的Python IDE 。
     </li>
     <li>
      <a href="https://github.com/Virtual-TreeView/Virtual-TreeView">
       Virtual-TreeView
      </a>
      ：Delphi树视图控件。
     </li>
     <li>
      <a href="https://github.com/danieleteti/delphimvcframework">
       DelphiMVCFramework
      </a>
      ：Delphi Web解决方案的流行且功能强大的框架，支持RESTful和JSON-RPC API开发。
     </li>
     <li>
      概要
      <a href="https://github.com/synopse/mORMot">
       mORMot
      </a>
      ORM / SOA / MVC框架。
     </li>
     <li>
      <a href="https://github.com/Zeus64/alcinoe">
       用于Delphi的Alcinoe组件库：用于Delphi的
      </a>
      完整OpenGL视频播放器，WebRTC Delphi包装，本机iOS / Android TEdit，改进的FireMonkey控件，Firebase云消息传递，Android / iOS Facebook SDK登录，Json / Bson Parser，ImageMagick包装，MongoDB。这是供客户和其他人使用的组件库。
     </li>
     <li>
      <a href="https://github.com/MojtabaTajik/Robber">
       强盗
      </a>
      ：一种工具，可检测容易受到DLL劫持的可执行文件。
     </li>
     <li>
      <a href="https://sourceforge.net/projects/portableapps/" rel="nofollow">
       PortableApps
      </a>
      ：提供可从云，本地和便携式USB驱动器运行的便携式应用程序。
     </li>
     <li>
      <a href="https://sourceforge.net/projects/orwelldevcpp/" rel="nofollow">
       Dev-C ++
      </a>
      ：免费，可移植，快速且简单的C / C ++ IDE 。
     </li>
     <li>
      <a href="https://sourceforge.net/projects/peazip/" rel="nofollow">
       PeaZip
      </a>
      ：免费的Zip和Rar提取程序。
     </li>
     <li>
      <a href="https://github.com/cnpack/">
       不用说
      </a>
      ，标准工具和库，例如
      <a href="https://github.com/cnpack/">
       CnPack
      </a>
      ，
      <a href="http://www.gexperts.org/" rel="nofollow">
       GExperts
      </a>
      ，
      <a href="https://github.com/TurboPack">
       TurboPack
      </a>
      和
      <a href="https://github.com/project-jedi">
       Project JEDI
      </a>
      。
     </li>
    </ul>
