    <h3>
     <a id="1msys2_2">
     </a>
     1.msys2简介
    </h3>
    <p>
     MSYS2（Minimal SYStem 2）是一个集成了大量的GNU工具链、工具和库的开源软件包集合。它提供了一个类似于Linux的shell环境，可以在Windows系统中编译和运行许多Linux应用程序和工具。
    </p>
    <p>
     MSYS2基于MinGW-w64平台，提供了一个完整的开发环境，包括GCC编译器、GDB调试器、Make、Git版本控制系统和许多其他开发工具。除了常用的开发库和工具之外，MSYS2还提供了许多专门针对Windows平台的库和工具，方便开发人员进行跨平台开发和移植工作。
    </p>
    <p>
     使用MSYS2可以轻松地在Windows系统中搭建一个类似于Linux的软件开发环境，方便开发人员进行开发和调试工作。
    </p>
    <p>
     https://www.msys2.org/
    </p>

    <h3>
     <a id="3_98">
     </a>
     3.使用
    </h3>
    <p>
     MSYS2 软件发行版使用pacman管理（安装、删除和更新）二进制包。
    </p>
    <h4>
     <a id="_102">
     </a>
     启动:
    </h4>
    <p>
     进入安装目录D:\msys64,双击
    </p>
    <pre><code class="prism language-shell">mingw64.exe
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14529b045f810b2d38bb2ebd1dfb4af3.png#pic_center"/>
    </p>
    <h4>
     <a id="_113">
     </a>
     帮助:
    </h4>
    <pre><code>$ pacman -h
usage:  pacman &lt;operation&gt; [...]
operations:
    pacman {-h --help}
    pacman {-V --version}
    pacman {-D --database} &lt;options&gt; &lt;package(s)&gt;
    pacman {-F --files}    [options] [file(s)]
    pacman {-Q --query}    [options] [package(s)]
    pacman {-R --remove}   [options] &lt;package(s)&gt;
    pacman {-S --sync}     [options] [package(s)]
    pacman {-T --deptest}  [options] [package(s)]
    pacman {-U --upgrade}  [options] &lt;file(s)&gt;

use 'pacman {-h --help}' with an operation for available options
</code></pre>
    <h4>
     <a id="_132">
     </a>
     更新:
    </h4>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Suy</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/675783c1417703f920df6ce09571020d.png#pic_center"/>
    </p>
    <p>
     输入Y后会关闭窗口。
    </p>
    <h4>
     <a id="_144">
     </a>
     查找:
    </h4>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Ss</span> <span class="token operator">&lt;</span>name or part of the name of the package<span class="token operator">&gt;</span>
</code></pre>
    <p>
     例如:
    </p>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Ss</span> openjp
</code></pre>
    <p>
     如果只想在已安装的软件包中搜索**，**请使用以下命令：
    </p>
    <pre><code>pacman -Qs &lt;name or part of the name of the package&gt;
</code></pre>
    <p>
     例如:
    </p>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Qs</span> openjp
<span class="token comment">#列出所有安装的包</span>
pacman <span class="token parameter variable">-Q</span>
</code></pre>
    <h4>
     <a id="_170">
     </a>
     列出包的内容
    </h4>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Ql</span> <span class="token operator">&lt;</span>name of the package<span class="token operator">&gt;</span>
</code></pre>
    <p>
     例如:
    </p>
    <pre><code class="prism language-shell">pacman <span class="token parameter variable">-Ql</span> xz
</code></pre>
