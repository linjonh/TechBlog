      <span style="font-family:微软雅黑,sans-serif">
       有多少高管认为人工智能可以帮他们的公司开辟新业务呢？75%。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       看好AI，认为这项技术能给自己公司带来竞争优势的就更多了，85%的高管都这么想。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       但是，只有20%的公司在产品、服务、流程里用了一些人工智能，全面部署AI的，只有5%。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       这些数据，来自《麻省理工学院斯隆管理评论》和波士顿咨询公司近日联合发布的一份研究报告。他们对来自112个国家、21个行业的3000多名企业高管、管理人员和分析师发进行了调研，还深入采访了30多位技术专家。
      </span>
      <strong style="color:rgb(47,47,47)">
       <span style="font-family:微软雅黑,sans-serif">
        这就是AI在全球企业中应用状况的缩影：一边热情地看好这项技术的潜力，一边迟迟不动手，甚至连制定了AI战略的公司都不是很多。
       </span>
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       报告显示，不到40%的公司制定了人工智能战略，就连员工规模超10万人的大公司，也只有一半这么做，要知道，这类公司是最有可能制定人工智能战略的。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       “在多数公司，野心与执行之间都有很大的落差。”波士顿咨询公司高级合伙人兼该报告联合作者Philipp Gerbert这样评价这种现象。
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       当然，这样一份报告并不是完全为了嘲讽全球公司眼高手低，下面就把这份报告的干货完整奉上。
      </span>
     </p>
     <p style="line-height:27px">

     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      <strong>
       期望
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       企业对人工智能的看法通常是这样的：现在它的影响还有限，但在5年内会发展成一个重量级要素。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       从报告的数据来看，只有约15%的受访者认为人工智能目前对其组织的产品和流程产生了重大影响，但约有60%左右的受访者预计这些影响将在5年内发生。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/3f99096056c691a03c2e4e17f01231a4.jpeg" style="width:640px!important; height:552px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <span style="color:rgb(136,136,136); font-family:微软雅黑,sans-serif; font-size:14px">
       技术/传媒/通讯业、消费性行业、金融服务业、专业服务业、健康、工业、能源、公共部门等行业，都有很多人认为，5年内AI会对他们的产品有很大影响
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       具体到产品方面，认为人工智能对自己公司有巨大影响的有14%，预计这些影响在5年内将会发生的占63%。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         在各行业之中，技术/传媒/通讯从业者认为AI在5年内将对自己公司产品有很大影响的人最多。
        </span>
       </strong>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>

      </span>

     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       各行业都有很多人认为，5年内AI会对他们的流程有很大影响
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       在流程方面，认为人工智能对自己公司有巨大影响的有15%，预计这些影响在5年内将会发生的占59%。消费型行业，是其中最积极的一个。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/0e17eaed291a53ef70317a2fcff75131.jpeg" style="width:640px!important; height:667px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-family:微软雅黑,sans-serif; font-size:14px; color:rgb(136,136,136)">
       人工智能将在哪些方面对公司产生最大影响？各行业的回答不尽相同
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       多数组织都预计，人工智能将对IT、运营/制造、供应链管理和面向客户的活动（例如销售、市场营销、客服等）产生可观的影响。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/9ffb3047ea51bf11d2cd1a99bd073a63.jpeg" style="width:640px!important; height:486px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       企业对人工智能的态度
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       超过80%的公司把人工智能视作一项战略机会，约有40%认为人工智能存在战略风险。只有很少（13%）认为人工智能既不是机会，也不是风险。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       大部分公司对人工智能的地位如此看重，那么，他们究竟认为AI可以给自己带来什么呢？
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/ff5f737fd1bb501dbe135b71c1778389.jpeg" style="width:640px!important; height:407px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       企业期望AI给他们带来竞争优势、新业务；同时也担心新公司和现有的竞争对手会运用AI超越自己
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       84%的公司认为，AI能带来竞争优势，或者帮他们维持现有的优势；75%的公司认为AI能帮他们开拓新业务。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       但同时，也有75%的公司认为会有运用AI技术的新公司进入他们所在的市场；69%的公司认为他们的现有竞争对手会运用AI技术。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         是什么因素驱动着企业去用AI呢？
        </span>
       </strong>
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       63%的公司认为是出自削减成本的压力，61%的公司认为会有供应商提供AI驱动的产品和服务，59%认为客户会要求他们提供AI驱动的产品。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      2
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      <strong>
       现状
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       但是，人工智能技术目前在企业中的应用情况和高企的期望值并不匹配，有超过半数的企业，都完全没有用上AI。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/28b1313734e683fa0174917b3f581d6b.jpeg" style="width:640px!important; height:450px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-family:微软雅黑,sans-serif; font-size:14px; color:rgb(136,136,136)">
       人工智能在企业中的应用情况
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       报告显示，有22%的企业还完全没有应用AI，也不打算去用；32%的企业目前没有应用AI但计划将来用；23%的企业有一个或者多个AI试点项目；18%的企业在部分产品和流程中应用了AI；5%企业的产品和流程中大量应用了AI。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       企业对人工智能的理解也存在巨大差异。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       例如，16%的受访者强烈同意他们的组织明白开发人工智能产品和服务的成本，同时，完全不了解这些花费的企业，几乎也占了同样的比例（17%）。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       类似地，19%的受访者强烈同意自己的组织理解训练人工智能算法所需的数据，同时，完全不了解这些知识的企业也有16%。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       根据对人工智能的理解和部署情况，这份报告把企业分成四类：
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       先锋（19%）：这些组织理解人工智能，而且已经部署。他们在把人工智能整合到自家产品和内部流程的过程中处于领先地位。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       研究者（32%）：这些组织理解人工智能，但除了试点阶段外尚未部署这项技术。他们希望先观察再行动。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       实验者（13%）：这些组织正在试点或采纳人工智能，但却并未深入理解。他们边做边学。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       消极者（36%）：这些组织没有采用人工智能，对这项技术的理解也很少。
      </span>
     </p>
     <p style="line-height:27px">
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       对于AI在技术、商业、人力、产业层面的意义，这四类企业的理解情况也不尽相同。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/9a198cca145c330228026dc16f583e76.jpeg" style="width:640px!important; height:441px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       企业对人工智能在各层面意义的理解情况
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         总的来说，企业对AI改变商业价值创造方式这一点理解得最好，对AI算法训练的流程、开发基于AI的产品或服务所需的成本最不了解。
        </span>
       </strong>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         <br/>
        </span>
       </strong>
      </span>
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      3
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      <strong>
       制约因素
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       文章一开头我们就提到，公司纷纷看好人工智能技术潜力的同时，真正应用在自己业务中的并不算多。“野心和执行”之间为何会出现如此巨大的落差？
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       这份报告得出的结论是，这在很大程度上是一个商业问题，而不是由于技术限制，
      </span>
      <span style="font-family:微软雅黑,sans-serif; color:rgb(47,47,47)">
       <strong>
        多种新趋势争夺投资优先级、缺乏清晰的商业案例是AI在企业中应用受阻的重要因素。
       </strong>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/7d5c624b58f61f9a572c1a168d4ea136.jpeg" style="width:640px!important; height:375px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       人工智能技术应用的主要制约因素
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       在调查中，MIT和波士顿咨询公司让受访者选出将AI应用于自己公司时遇到的三个最大制约因素，上图显示了7种制约因素对先锋、研究者、实验者、消极者这4类角色的影响情况。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       这7种因素从左到右分别是：
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       1.通过吸引、收购、内部培养等方式找到合适的人工智能人才。受这一因素影响最大的，是先锋们，而消极者可能还根本没有考虑这个问题。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       2.公司内部投资优先级的竞争。各类公司受这一因素的影响都很大。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       3.担心AI的应用可能带来安全问题。除了先锋们之外，所有公司对这一因素考虑得都很少。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       4.文化上的阻力。各类公司受这一因素的影响差不多，都在30%左右。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       5.在通用技术（比如分析、数据、IT）上能力有限。近50%的消极者、40%左右的研究者和实验者、以及20%多的先锋表示他们收到这一因素的制约。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       6.缺乏领导层的支持。55%左右的消极者说他们受到这一因素的影响。从调查结果来看，这个因素的影响力和人工智能在企业中的应用负相关。
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       7.AI应用没有明确的商业案例。近70%的消极者说他们受到这一因素的影响。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       企业在对新技术进行投资时，通常会面临这样一个选择：自己招人开发，还是购买服务？
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/1bbdb3090788558581c57a67e4cf8e5c.jpeg" style="width:640px!important; height:552px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       △ 企业想怎样实现人工智能需求
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <strong>
       <span style="font-family:微软雅黑,sans-serif; color:rgb(47,47,47)">
        在人工智能这项技术上，对AI的理解和应用都比较好的先锋企业们，会更倾向于在工作培养内部的开发人才，或者从其他机构、高校招募人才。对AI理解和经验有限的公司，会更倾向于通过外包来解决。
       </span>
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       “每一类企业在进一步采用人工智能方面都面临自己的挑战。”《麻省理工斯隆管理评论》执行主编兼该报告联合作者David Kiron说，“先锋已经克服了与理解有关的问题，他们最大的障碍是掌握各种实践机会来开发和获取必备人才，并解决投资中相互矛盾的重点。消极者还不知道人工智能可以为自己做什么。他们还没有找到可靠的商业案例。领导力或许还没有就位。很多都没有意识到获取和部署人工智能人才所面临的困难。”
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       这份报告还包含了对AI的理解和应用程度各不相同的企业，在管理、领导力、策略、保护用户隐私方面表现如何。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       报告作者Gerbert说，先锋公司和落后者之间在各方面都有着巨大的差距，：“我们还发现今天的领导者——已经理解且采用人工智能的公司——与落后者之间的巨大差距。领导者不仅比落后者更加深入地理解生产人工智能所需的要素，而且更有可能为人工智能项目提供配套的高级领导力支持，同时拥有一个成熟的商业案例。”
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/09f7fbd27f8cd2f99d79b88638189222.jpeg" style="width:640px!important; height:552px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="font-size:14px; font-family:微软雅黑,sans-serif; color:rgb(136,136,136)">
       企业想怎样实现人工智能需求
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       报告显示，对AI的理解和应用程度最好的公司，在其他方面也同样更好。特别是在数据隐私方面，73%的先锋企业认为自己有很好的数据管理措施，而只有34%的实验者、30%的消极者认为自己在这方面做得很好。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      4
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      <strong>
       管理层面临的挑战
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       要在企业中用好AI，除了技术、人才培养上的问题外，管理层也需要提高自身水平。这份报告指出，管理层需要：
      </span>
     </p>
     <span style="font-family:微软雅黑,sans-serif; font-size:medium; line-height:27px; text-decoration:inherit">
      对AI建立基本的理解；
     </span>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       加深自己对如何围绕AI来管理公司业务的认识；
      </span>
     </p>
     <p style="font-family:'Helvetica Neue',Helvetica,'Hiragino Sans GB','Microsoft YaHei',Arial,sans-serif; font-size:medium; white-space:normal; line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       对自己公司所处行业的竞争格局建立更广阔的认识。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         管理层了解AI、建立战略的情况，也和企业了解、应用AI的情况一样：野心很大，执行很差。
        </span>
       </strong>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/d60b968dcaf0d75eb6e45dcb674234d8.jpeg" style="width:640px!important; height:282px!important"/>
     </p>
     <p style="line-height:27px; text-align:center">
      <span style="color:rgb(136,136,136); font-family:微软雅黑,sans-serif; font-size:14px">
       大部分企业都认为他们急需建立AI战略，但真正着手于此的只有一半
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       如上图显示，只有先锋企业已经迈过了那个想要建立AI战略却不动手的阶段，79%认为自己的组织急需AI战略，88%已经建立。其他几类企业，都是认为急需AI战略的多，真正建立了的少。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       另外，该调查还发现，尽管媒体普遍担心人工智能导致失业，但只有不到一半的受访者预计自己公司的员工人数会在未来5年减少。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       <br/>
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      5
     </p>
     <p style="max-width:100%; word-wrap:break-word!important">
      <strong>
       前路：对未来的影响
      </strong>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       随着人工智能越来越多的应用于知识领域，越来越多的西方中产阶级的工作会受到影响。不过与常见的悲观情绪不同，这份调查显示出谨慎的乐观。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       例如，这份调查还发现，尽管媒体普遍担心人工智能导致失业，大多数受访者并不认为AI能在未来五年内减少其所在组织的就业岗位。
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       近70%的人表示并不担心AI能自动完成他们的工作。实际上，受访者希望AI能接管目前一些无聊和恼人的工作。当然为了应对AI到来，他们也认为需要继续学习提高自己。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="text-align:center">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/451c85ec617e93fec85fb7f5bf62362e.jpeg" style="width:640px!important; height:418px!important"/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
      <br/>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       麻省理工斯隆管理学院教授Erik Brynjolfsson表示，“AI不会很快取代大多数工作岗位。但几乎在每个行业，使用AI的人类将开始取代不用AI的人类，而且这个趋势只会加速”。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="color:rgb(47,47,47)">
       <strong>
        <span style="font-family:微软雅黑,sans-serif">
         AI能创造价值，但其影响力很可能并不均衡。
        </span>
       </strong>
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       在一个行业内，到底哪些组织能受益于此，哪些反而会受到影响？目前还难以得出结论。不过也并非不能着手准备。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       “当下随着大数据的大量涌现，这些数据绝对是宝贵的”，慕尼黑再保险集团的Marcus Winter说。换句话说，数据和AI算法相结合可以创造出更有效的解决方案。而管理层也在期待AI能带来产品和流程和显著提高。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       许多公司都致力于此。但实际上，只是改善和提高并不能为公司带来可持续的竞争优势，因为随着所有企业的能力都提升到差不多，会推进基线的改变。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       为了让AI成为未来战略中的一个重要因素，企业必须清楚人类和计算机如何协作，互相取长补短。
      </span>
      <span style="font-family:微软雅黑,sans-serif">
       这并不容易：企业需要特殊的数据来源，大多数公司都不具备这种优势；企业需要学会让人类和机器更有效率地协作，目前只有少数先锋公司具备这种能力；企业还需要有灵活的组织架构，也就是说，公司和员工都面临着文化上的改变。
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
      </span>
     </p>
     <p style="line-height:27px">
      <span style="font-family:微软雅黑,sans-serif">
       可能任何一家公司，现在都需要一个关于人工智能的计划，但大多数都没有。目前落后的公司，需要立刻加快脚步追赶，如果持续落后下去，最终可能会发现摆在眼前的路越来越难走。
      </span>
     </p>
