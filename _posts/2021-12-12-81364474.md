    <p>
     蜂窝模型：
     <br/>
     （1）有用性。面对的用户需求是真实的。
     <br/>
     （2）可用性。功能可以很好地满足用户需求。
     <br/>
     （3）满意度。涉及情感设计的方面，比如图形、品牌、形象。
     <br/>
     （4）可找到。用户能够轻易地找到他们需求的东西。
     <br/>
     （5）可获得。用户能够方便地完成操作，达到目的。
     <br/>
     （6）可靠性。让用户产生信任。
     <br/>
     （7）价值。产品要为投资人产生价值。
    </p>
    <p>
     5E原则：
     <br/>
     （1）有用性(effevtive).
     <br/>
     （2）效率(efficient).
     <br/>
     （3）易学习(easy to learn).
     <br/>
     （4）容错(error tolerant).
     <br/>
     （5）吸引力(engaging).
    </p>
