    <h2 id="FluentEmail">
     <strong>
      FluentEmail
     </strong>
    </h2>
    <p>
     FluentEmail 是一个开源的 .NET 组件库，100%由C# 开发，它是 .NET 最流行的邮件发送库。您可以在几分钟内，配置完整的邮件发送功到您的 .NET 项目中。
    </p>
    <p>
     FluentEmail 的 API 语法非常简洁，开箱即用且支持依赖注入。
    </p>
    <p>
    </p>
    <h2 id="%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8%3F">
     <strong>
      如何使用?
     </strong>
    </h2>
    <p>
     您可以使用下面的命令在 Nuget 中安装 FluentEmail。
    </p>
    <pre><code>dotnet add package FluentEmail.Core
</code></pre>
    <p>
     下面是一个最简单的使用示例，使用了 Fluent API
    </p>
    <pre><code class="language-cs">var email = await Email
    .From("john@email.com")
    .To("bob@email.com", "bob")
    .Subject("邮件标题")
    .Body("邮件主题")
    .SendAsync();
</code></pre>
    <p>
     FluentEmail 提供了开箱即用的依赖注入支持.
    </p>
    <pre><code class="language-cs">public void ConfigureServices(IServiceCollection services)
{
    services
        .AddFluentEmail("fromemail@test.test")
        .AddRazorRenderer()
        .AddSmtpSender("localhost", 25);
}
</code></pre>
    <p>
     使用 IfluentEmail 接口发送邮件
    </p>
    <pre><code class="language-cs">public class EmailService {

   private IFluentEmail _fluentEmail;

   public EmailService(IFluentEmail fluentEmail) {
     _fluentEmail = fluentEmail;
   }

   public async Task Send() {
     await _fluentEmail.To("hellO@gmail.com")
     .Body("The body").SendAsync();
   }
}

</code></pre>
    <p>
     对于邮件内容，FluentEmail 还支持 Razor 和 Liquid 两种模式渲染模式。
    </p>
