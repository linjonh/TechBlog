    <h2>
     <a id="_26">
     </a>
     开发前准备
    </h2>
    <h3>
     <a id="_27">
     </a>
     任务说明
    </h3>
    <p>
     通过构建一个简单的具有页面跳转功能的应用（如下图预览器运行效果所示），熟悉HarmonyOS应用开发流程。
    </p>
    <p>
     为确保运行效果，本文以使用DevEco Studio 3.0 Beta2版本为例。
    </p>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/b8a46c11db53736db194865760ce6963.png"/>
    </p>
    <p>
     我们需要知道，HarmonyOS提供了两种FA（Feature Ability）的UI开发框架：Java UI框架和方舟开发框架。
    </p>
    <p>
     没有对比就没有伤害：
    </p>
    <ul>
     <li>
      Java UI框架提供了细粒度的UI编程接口，UI元素更丰富，使应用开发更加灵活。
     </li>
     <li>
      方舟开发框架提供了相对高层的UI描述，使应用开发更加简单。
     </li>
    </ul>
    <p>
     对于DevEco Studio V2.2 Beta1及更高版本，在使用JS语言开发时，支持选择使用低代码方式。低代码开发方式遵循HarmonyOS JS开发规范，具有丰富的UI界面编辑功能，通过可视化界面开发方式快速构建布局，可有效降低用户的上手成本并提升用户构建UI界面的效率。
    </p>
    <h3>
     <a id="_42">
     </a>
     开发小天使
    </h3>
    <p>
     开始前请参考下载与安装软件、配置开发环境，完成DevEco Studio的安装和开发环境配置。
    </p>
    <h4>
     <a id="_44">
     </a>
     下载与安装软件
    </h4>
    <p>
     DevEco Studio支持Windows和macOS系统，下面将针对两种操作系统的软件安装方式进行介绍。
     <br/>
     DevEco Studio的编译构建依赖JDK，DevEco Studio预置了Open JDK，版本为1.8，安装过程中会自动安装JDK。
    </p>
    <ol>
     <li>
      进入HUAWEI DevEco Studio产品页，点击下载列表后的按钮，下载DevEco Studio。
     </li>
     <li>
      下载完成后，双击下载的“deveco-studio-xxxx.exe”，进入DevEco Studio安装向导，在如下安装选项界面勾选DevEco Studio后，点击Next，直至安装完成。
     </li>
    </ol>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/e8a3f495a02427488272d9edff033abc.png"/>
    </p>
    <ol start="3">
     <li>
      安装完成后，点击Finish完成安装。
      <br/>
      <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/d3c22a14db29ff9a9c0f061c4cf3d855.png"/>
     </li>
    </ol>
    <p>
     当然，如果你是mac用户，相信你一定也是个小机灵鬼儿！
    </p>
    <h4>
     <a id="_58">
     </a>
     配置开发环境
    </h4>
    <h5>
     <a id="HarmonyOS_SDK_59">
     </a>
     下载&amp;更新HarmonyOS SDK
    </h5>
    <p>
     DevEco Studio提供SDK Manager统一管理SDK及工具链，下载各种编程语言的SDK包时，SDK Manager会自动下载该SDK包依赖的工具链。
    </p>
    <p>
     第一次使用DevEco Studio，需要下载HarmonyOS SDK及对应工具链。如果是非首次安装DevEco Studio，SDK下载和更新方法请参考更新HarmonyOS SDK。
    </p>
    <ol>
     <li>
      运行已安装的DevEco Studio，首次使用，请选择Do not import settings，点击OK。
     </li>
     <li>
      进入配置向导页面，设置npm registry，DevEco Studio已预置对应的仓，直接点击Start using DevEco Studio进入下一步。
     </li>
    </ol>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/dc91471cae1cc97372c1f7cff9926fb6.png"/>
    </p>
    <ol start="3">
     <li>
      <p>
       DevEco Studio向导指引开发者下载SDK，默认下载OpenHarmony SDK。SDK下载到user目录下，也可以指定对应的存储路径，SDK存储路径不支持中文字符，然后点击Next。
       <br/>
       <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/c8683efde015b5c867537d283877f586.png"/>
      </p>
     </li>
     <li>
      <p>
       在弹出的SDK下载信息页面，点击Next，并在弹出的License Agreement窗口，点击Accept开始下载SDK。
      </p>
     </li>
    </ol>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/39c48b6051dae39003cf2cdabb51acea.png"/>
    </p>
    <ol start="5">
     <li>
      <p>
       等待OpenHarmony SDK及工具下载完成，点击Finish，界面会进入到DevEco Studio欢迎页。
       <br/>
       <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/7c10c4cfc87ebea71e5dc1af398278ba.png"/>
      </p>
     </li>
     <li>
      <p>
       点击欢迎页中的Configure（或图标） &gt; Settings &gt; SDK Manager &gt; HarmonyOS Legacy SDK，然后点击Edit设置HarmonyOS SDK存储路径。
       <br/>
       <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/50772e633aecf13e7e1239241690fef1.png"/>
      </p>
     </li>
     <li>
      <p>
       设置HarmonyOS Legacy SDK Location存储路径，然后点击Next。请注意，该路径不能与OpenHarmony SDK存储路径相同，否则会导致OpenHarmony SDK的文件被删除。
       <br/>
       <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/26835894a54c3682002baf336d8346df.png"/>
      </p>
     </li>
     <li>
      <p>
       在弹出的SDK下载信息页面，点击Next，并在弹出的License Agreement窗口，点击Accept开始下载SDK。
      </p>
     </li>
     <li>
      <p>
       下载完成后，点击Finish。自此，HarmonyOS SDK就安装完成了。
       <br/>
       <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/6d3e8571519201afc5fea4e68e5391e0.png"/>
      </p>
     </li>
    </ol>
    <p>
     SDK默认只会下载最新版本的Java SDK、JS SDK、Previewer和Toolchains，点击上图欢迎页中的Configure（或图标） &gt; Settings &gt; SDK Manager &gt; HarmonyOS Legacy SDK，进入HarmonyOS Legacy SDK页面，可以下载其它组件，只需要勾选对应的组件包，然后点击Apply即可。
    </p>
