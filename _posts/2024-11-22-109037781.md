    <h3>
     <a id="_0">
     </a>
     软考嵌入式系统设计师
    </h3>
    <p>
     最近在准备中国计算机技术职业资格（软考）------
     <strong>
      嵌入式系统设计师
     </strong>
     方向的考试，开此专栏记录复习备考过程，总结复习要点以及考试考点，同时还可以在手机端随时查看笔记，方便利用碎片时间随时随地复习
    </p>
    <h5>
     <a id="_4">
     </a>
     嵌入式系统设计师考试说明
    </h5>
    <h6>
     <a id="_6">
     </a>
     一、考试科目设置
    </h6>
    <ul>
     <li>
      嵌入式系统基础知识，考试时间为150分钟，笔试，选择题
     </li>
     <li>
      嵌入式系统设计，考试时间为150分钟，笔试，问答题
     </li>
    </ul>
    <h6>
     <a id="_10">
     </a>
     二、考试要求
    </h6>
    <ol>
     <li>
      掌握计算机系统的基础知识
     </li>
     <li>
      掌握嵌入式系统的硬件、软件基础知识
     </li>
     <li>
      掌握嵌入式系统需求分析方法
     </li>
     <li>
      掌握嵌入式系统设计与开发的方法及步骤
     </li>
     <li>
      掌握嵌入式系统实施、运行、维护知识
     </li>
     <li>
      熟悉软件过程改进和软件开发项目管理等软件工程基础知识
     </li>
     <li>
      了解信息化基础知识、计算机控制、网络通信等计算机应用基础知识
     </li>
     <li>
      了解系统的安全性、可靠性、信息技术标准以及有关法律法规的基本知识
     </li>
     <li>
      了解嵌入式技术发展趋势
     </li>
     <li>
      正确阅读和理解计算机及掌握嵌入式系统领域的英文资料
     </li>
    </ol>
    <h6>
     <a id="_22">
     </a>
     三、指定教材
    </h6>
