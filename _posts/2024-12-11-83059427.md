    <p>
     创建库（分4步走）：
     <br/>
     1.单击工具栏按钮（
     <strong>
      第四个
     </strong>
     ，即鼠标放在 上面显示
     <strong>
      Create a new schema in the connected server
     </strong>
     ），创建数据库。
     <br/>
     2.修改数据库
     <strong>
      名字
     </strong>
     ，collation保持默认，单击
     <strong>
      确定
     </strong>
     。
     <br/>
     3.其他选项
     <strong>
      保持默认
     </strong>
     ，单击
     <strong>
      Apply
     </strong>
     。
     <br/>
     4.其他选项
     <strong>
      保持默认
     </strong>
     ，单击
     <strong>
      Finish
     </strong>
     。
    </p>
    <p>
     创建表：
     <br/>
     1.首先要 “
     <strong>
      双击
     </strong>
     ” 一下刚刚建立好的数据库mydatabase，然后再创建表，不然会出错，右键点击
     <strong>
      Tables
     </strong>
     然后点击
     <strong>
      Create new tables
     </strong>
     ,填写表名，以及表列的信息，之后点击
     <strong>
      apply
     </strong>
     。
    </p>
    <p>
     <strong>
      PK
     </strong>
     : primary key (column is part of a pk)
     <strong>
      主键
     </strong>
    </p>
    <p>
     <strong>
      NN
     </strong>
     : not null (column is nullable)
     <strong>
      是否为空
     </strong>
    </p>
    <p>
     <strong>
      -UQ
     </strong>
     : unique (column is part of a unique key)
     <strong>
      外键
     </strong>
    </p>
    <p>
     <strong>
      AI
     </strong>
     : auto increment (the column is auto incremented when rows are inserted)
     <strong>
      自动增加
     </strong>
     <br/>
     <strong>
      UN
     </strong>
     : unsigned (for integer types, see docs: “10.2. Numeric Types”)
     <br/>
     <strong>
      ZF
     </strong>
     : zero fill (rather a display related flag, see docs: “10.2. Numeric Types”)
    </p>
