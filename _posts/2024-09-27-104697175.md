    <h4>
     <a id="2__4">
     </a>
     2. 题目分析
    </h4>
    <p>
     这道题我们需要关注的点有一下几个：
    </p>
    <ul>
     <li>
      数组整体是
      <strong>
       升序
      </strong>
      的，并且数组中
      <strong>
       不存在重复的元素
      </strong>
     </li>
     <li>
      数组在某一点进行了
      <strong>
       旋转
      </strong>
     </li>
     <li>
      返回值是相同元素的
      <strong>
       下标
      </strong>
      ，没有则返回-1
     </li>
     <li>
      算法时间复杂度必须是
      <strong>
       O(logn)
      </strong>
     </li>
    </ul>
    <p>
     从上面几点分析，要想时间复杂度达到
     <strong>
      O(logn)
     </strong>
     ，要使用二分法进行搜索。但是怎么处理旋转之后的数组呢？对于旋转之后的数组我们可以用下面这个柱形图图来表示，柱形图的高相当于元素的值。以[4, 5, 6, 7, 0, 1, 2]为例：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d77516d4b21da023a461cf426ce1fb99.png#pic_center" width="450">
      <br/>
      从这张图可以看出：
     </img>
    </p>
    <ul>
     <li>
      最左边的元素
      <strong>
       一定大于
      </strong>
      最右的元素
     </li>
     <li>
      如果
      <strong>
       一个元素大于最右的元素
      </strong>
      ，那么这个元素
      <strong>
       一定
      </strong>
      在区间[1, 4]之间，否则该元素
      <strong>
       一定
      </strong>
      在[5,7]之间
     </li>
    </ul>
    <p>
     我们可以根据这两点来进行二分法的实现。
    </p>
