       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        从此，我开始了更为狂热的计算机技术学习之路，开始整天逃课躲在办公室学习，每天晚上
       </span>
       <span lang="EN-US">
        10
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        点后才从办公室离开，早上很早就来到办公室。那时候，由于
       </span>
       <span lang="EN-US">
        Mars Studio
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        需要开发和维护的网站并不仅仅局限于用
       </span>
       <span lang="EN-US">
        Java
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        编写，还包括
       </span>
       <span lang="EN-US">
        ASP
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        和
       </span>
       <span lang="EN-US">
        C#
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        。刚进入
       </span>
       <span lang="EN-US">
        Mars Studio
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        后，交给我的第一个项目就是和“笨小孩”一起，给“工大在线”增加一个类似于
       </span>
       <span lang="EN-US">
        ChinaRen
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        校友录功能，用
       </span>
       <span lang="EN-US">
        ASP + MS SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        实现。可当时我从来没有用过
       </span>
       <span lang="EN-US">
        ASP
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        ，也没有用过
       </span>
       <span lang="EN-US">
        SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        数据库。
       </span>
       <span lang="EN-US">
       </span>
      </font>
     </font>
    </p>
    <p class="IT" style="margin: 3pt 0cm 1pt; text-indent: 20pt">
     <font size="2">
      <font face="Times New Roman">
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        “笨小孩”对我说：你先学习一个星期的
       </span>
       <span lang="EN-US">
        ASP
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        和
       </span>
       <span lang="EN-US">
        SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        的基本使用，然后我们再开始做。“笨小孩”这句轻描淡写的话，给了我极大的压力，心想一个星期怎么可能学会一门编程语言并用来开发项目？可当时由于自己刚刚进入
       </span>
       <span lang="EN-US">
        Mars Studio
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        ，也不敢有任何抱怨，只能硬着头皮接受。接下来的一个星期，我开始没日没夜地学习
       </span>
       <span lang="EN-US">
        ASP
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        和
       </span>
       <span lang="EN-US">
        SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        基本知识。而当时的“笨小孩”可以说在
       </span>
       <span lang="EN-US">
        ASP
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        方面已经有非常多的经验，开发过很多成功的项目，并且对
       </span>
       <span lang="EN-US">
        SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        数据库也有较为深入的研究和理解。我只要一遇到任何疑问，就会缠着他问，他也会尽他所能，耐心给我讲解，直到我完全弄明白为止。在这一个星期中，我的进步真的是非常之快，连我自己都非常吃惊。一个星期之后，我们就开始做这个小项目，两个人边研究
       </span>
       <span lang="EN-US">
        ChinaRen
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        的功能，边分析我们自己的特点，边进行功能整理，边分析设计数据库表结构。又经过一个星期的时间，我们基本完成了这个小项目的设计工作。虽然比较粗糙，但是对于我们两个当事人来说，也算比较清晰了，后面就是正式的编码阶段了。
       </span>
       <span lang="EN-US">
       </span>
      </font>
     </font>
    </p>
    <p class="IT" style="margin: 3pt 0cm 1pt; text-indent: 20pt">
     <font size="2">
      <font face="Times New Roman">
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        编码阶段可以说是我最有成就感的过程了，因为随着一行一行代码的完成，一个一个的小功能开始被实现，心里感到特别兴奋。我想，这种成就感应该每一个开始学编程的人都有过吧。在编码进行到快两个星期的时候，由于一个性能问题，我的自信心第一次受到了打击。一个留言列表分页显示的功能，在我自己的电脑（开发环境）上可以“很快”地运行，可发布到测试环境之后，性能非常差，每次点击之后，都要好几秒才能显示出来。为此，我花了一整个晚上去追查问题，可任凭我如何跟踪，都找不到问题出在哪里，整个人都快崩溃了。第二天，“笨小孩”来到办公室之后，我向他描述了问题，他什么也没有说，打开
       </span>
       <span lang="EN-US">
        SQL Server
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        的管理界面，快速敲了一条命令并执行后，让我再测试一下。我重新发布了代码，测试了一下之前响应非常之慢的留言列表分页显示功能，响应变得非常之快。看到如此神奇的现象，我追问原因。“笨小孩”告诉我，之前运行很慢，是因为存放留言信息的表上面没有合适的索引，所以每次显示留言都需要读取整个留言表，由于之前他做测试对留言表生成了很多的数据，所以很慢，我自己电脑上运行较快只是因为里面的测试数据很少。我第一次知道了数据库中原来还有如此神奇的功能
       </span>
       <span lang="EN-US">
        ——
       </span>
       <span style="font-family: 楷体_GB2312; mso-ascii-font-family: 'Times New Roman'">
        索引，可以让查询效率变得如此高。也正是从这时候开始，我对数据库产生了极其浓厚的兴趣。可以说，这个项目及“笨小孩”对我以后的从业方向有非常深远的影响。
       </span>
       <span lang="EN-US">
       </span>
       <span lang="EN-US" style='font-size: 9pt; color: black; font-family: "Arial","sans-serif"; mso-bidi-font-size: 10.0pt'>
       </span>
      </font>
     </font>
    </p>
    <p class="IT1" style="margin: 4pt 0cm 2pt; text-indent: 19pt; mso-pagination: none">
     <font size="2">
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       从此，我开始加强对数据库的关注，开始了数据库相关知识的学习。不过，这时自己的学习重点仍然是编程语言，一来是因为平时在
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Mars Studio
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       中有不少工作内容需要通过
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Java
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       和
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        ASP
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       来做，二来是
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Mars Studio
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       里的其他朋友并没有对数据库非常热衷，大都是专注于编程语言的学习。
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
       </font>
      </span>
     </font>
    </p>
    <p class="IT1" style="margin: 4pt 0cm 2pt; text-indent: 19pt; mso-pagination: none">
     <font size="2">
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       接下来的一年多时间中，我开始和
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Sailing
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       一起重点学习
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Java
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       编程，因为我们希望改版当时的“玄武雅阁”
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        BBS
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       论坛，在
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Jive Forum
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       的基础上进行二次开发，增加一些定制化的功能，如站内留言消息、投票、帖子评分等。而且，我们还同时迷上了
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        JavaRobocode
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       。还非常清楚地记得，为了学习
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Robocode
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       中实现各种瞄准以及运动算法中使用非常频繁的三角函数算法，我们俩在非典期间逃出学校的封锁，跑到“军人俱乐部”（南京图书市场）去买初中的数学课本，最后在学校围墙外等到天黑才翻墙回到校内。
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
       </font>
      </span>
     </font>
    </p>
    <p class="IT1" style="margin: 4pt 0cm 2pt; text-indent: 19pt; mso-pagination: none">
     <font size="2">
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       和
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Sailing
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       一起学了一年多的
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Java
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       编程，也完成了“玄武雅阁”的二次开发，手边的项目基本都已经处于稳定的维护阶段。接下来的一年多时间里，我们开始在校内校外承接一些小项目进行开发，一来锻炼大家的编程能力，二来还可以带来一定的经济收入，通过自己的双手赚取自己的生活费。在这段时间里，
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Mars Studio
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       整个
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Team
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       做了大约
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        20
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       多个大大小小的项目，通过实际项目的锻炼，每个人都进步得非常快，也非常有成就感，不仅通过自己的双手解决了个人经济问题，还为
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Mars Studio
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       办公室增添了一台空调，改善了大家的学习和工作环境。当然，在这个过程中，大家也非常辛苦，每天都起早摸黑，还经常通宵在办公室中学习工作。但是，每个人都保持着高度的激情，每天早上都是非常兴奋地进入办公室，晚上也是非常满足地离开办公室，一直保持着交流分享的氛围。
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
       </font>
      </span>
     </font>
    </p>
    <p class="IT1" style="margin: 4pt 0cm 2pt; text-indent: 19pt; mso-pagination: none">
     <font size="2">
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       同时，我也一直没有放下数据库的学习，在做“玄武雅阁”的过程中，我学到了很多
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        MySQL
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       数据库的知识，还从网络中心崔北亮老师那里学到了很多在
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Windows
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       下搭建和维护
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Web
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       服务器的知识，从尹晨姐那里学到了很多
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Solaris
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       的维护管理知识。偶尔还跟着
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        Chagel
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       和曾军学点
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">

       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
       知识，
      </span>
      <span lang="EN-US">
       <font face="Times New Roman">
        PS
       </font>
      </span>
      <span style="font-family: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
