      <strong>
       1.基础：
      </strong>
     </span>
     <span style="font-family:'Microsoft YaHei UI','Microsoft YaHei',SimSun,'Segoe UI',Tahoma,Helvetica,sans-serif,'Microsoft YaHei',Georgia,Helvetica,Arial,sans-serif,宋体,PMingLiU,serif; line-height:21px">
     </span>

    <div style="font-family:'Microsoft YaHei UI','Microsoft YaHei',SimSun,'Segoe UI',Tahoma,Helvetica,sans-serif,'Microsoft YaHei',Georgia,Helvetica,Arial,sans-serif,宋体,PMingLiU,serif; line-height:21px">
     <span style="font-size:18px">
      <span style="text-indent:28px">
       <span style="white-space:pre">
       </span>
       LINQ，语言集成查询（Language Integrated Query）是一组用于
      </span>
      <a href="http://baike.baidu.com/view/6590.htm" rel="nofollow noopener noreferrer" style="text-indent:28px" target="_blank">
       c#
      </a>
      <span style="text-indent:28px">
       和
      </span>
      <a href="http://baike.baidu.com/view/14260.htm" rel="nofollow noopener noreferrer" style="text-indent:28px" target="_blank">
       Visual Basic
      </a>
      <span style="text-indent:28px">
       语言的扩展。它允许编写C#或者Visual Basic代码以查询
      </span>
      <a href="http://baike.baidu.com/view/1088.htm" rel="nofollow noopener noreferrer" style="text-indent:28px" target="_blank">
       数据库
      </a>
      <span style="text-indent:28px">
       相同的方式操作内存数据。
      </span>
      它是集成在.NET编程语言中的一种特性。已成为编程语言的一个组成部分，在编写程序时可以得到很好的编译时语法检查，丰富的元数据，智能感知、静态类型等强类型语言的好处。并且它同时还使得查询可以方便地对内存中的信息进行查询而不仅仅只是外部数据源。
     </span>
     <div>
      <span style="font-size:18px">
      </span>
     </div>
     <div>
      <span style="font-size:18px">
       <span style="white-space:pre">
       </span>
       LINQ包括五个部分：LINQ to Objects、LINQ to DataSets、LINQ to SQL、LINQ to Entities、LINQ to XML：
       <span style="text-indent:28px">
        <br/>
       </span>
      </span>
     </div>
     <div>
      <span style="font-size:18px">
       Linq to Objects：为任意类型的C#内存对象提供查询，如数组、列表和其它集合类型。
       <br/>
      </span>
     </div>
     <p style="margin-top:5px; margin-bottom:5px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       LINQ to SQL：全称基于关系数据的.NET语言集成查询，用于以对象形式管理关系数据，并提供了丰富的查询功能。其建立于公共语言类型系统中的基于SQL的模式定义的集成之上，当保持关系型模型表达能力和对底层存储的直接查询评测的性能时，这个集成在关系型数据之上提供强类型。
      </span>
     </p>
     <p style="margin-top:5px; margin-bottom:5px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       LINQ to XML：在System.Xml.LINQ命名空间下实现对XML的操作。采用高效、易用、内存中的XML工具在宿主编程语言中提供XPath/XQuery功能等。
      </span>
     </p>
     <p style="margin-top:5px; margin-bottom:5px; padding-top:0px; padding-bottom:0px">
      <strong>
       <span style="font-size:18px">
        2.用法：
       </span>
      </strong>
     </p>
     <p style="margin-top:5px; margin-bottom:5px; padding-top:0px; padding-bottom:0px">
      <span style="font-size:18px">
       <span style="font-family:Microsoft YaHei UI">
       </span>
       <span style="font-family:'Microsoft YaHei UI'; line-height:1.5">
       </span>
       <span style="line-height:1.5">
        查询表达式必须以
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        <span lang="EN-US" style="word-wrap:normal; word-break:normal">
         from
        </span>
       </span>
       <span style="line-height:1.5">
        子句开头，并且必须以
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        select
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        或
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        group
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        子句结尾。在第一个
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        from
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        子句和最后一个
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        select
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        或
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        group
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        子句之间，查询表达式可以包含一个或多个下列可选子句：
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        where
       </span>
       <span style="line-height:1.5">
        、
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        orderby
       </span>
       <span style="line-height:1.5">
        、
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        join
       </span>
       <span style="line-height:1.5">
        、
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        let
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        甚至附加的
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        from
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        子句。还可以使用
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        into
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        关键字使
       </span>
       <span style="line-height:1.5">
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        join
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        或
       </span>
       <span lang="EN-US" style="line-height:1.5; word-wrap:normal; word-break:normal">
        group
       </span>
       <span style="line-height:1.5">
       </span>
       <span style="line-height:1.5">
        子句的结果能够充当同一查询表达式中附加查询子句的源。
       </span>
      </span>
     </p>
