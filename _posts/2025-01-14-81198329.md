    <p>
     云服务器为了保证安全，对外不提供外网访问的端口，往往需要加上跳板机或者堡垒机，要不然就是搭建VPN，这里只讲一下跳板机的搭建。
    </p>
    <p>
     需要的资源：
     <br/>
     1、公司一台可以提供外网服务的服务器
     <br/>
     2、阿里云等云服务器
    </p>
    <p>
     配置的步骤：
     <br/>
     1、使用xshell登录到跳板机，为了秘钥文件的安全，将高级下面的这几项直接勾上
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/21df7d54a5fa6cf04149293b79fad2cf.png" title="">
      <br/>
      2、修改~/.ssh/config 文件，修改的命令为：
     </img>
    </p>
    <pre class="prettyprint"><code class="hljs ruby">vi ~<span class="hljs-regexp">/.ssh/config</span></code></pre>
    <p>
     文件的内容
    </p>
    <pre class="prettyprint"><code class="hljs perl">Host tiaobanji
HostName XXX
Port XXX
User XXX


Host web_master
HostName XXXX
port XXXX
User XXXX
ProxyCommand ssh root<span class="hljs-variable">@tiaobanji</span> -W <span class="hljs-variable">%h</span>:<span class="hljs-variable">%p</span></code></pre>
    <p>
     其中：
     <br/>
     Host 主机别名
     <br/>
     HostName 主机的外网Ip
     <br/>
     Port 主机登录的端口号
     <br/>
     User 主机登录的用户名
     <br/>
     ProxyCommand ：代理执行的命令
    </p>
    <p>
     3、使用 ssh web_master 命令就可以登录到云服务上，因为Xshell勾选了高级选项，所以需要双方密码。
    </p>
    <pre class="prettyprint"><code class="hljs">ssh web_master</code></pre>
    <p>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/79d4e4ffb5819e387c665b9a9c7b88b9.png" title=""/>
    </p>
    <p>
     4、当然如果觉得上诉方法太麻烦，也可以执行执行命令
    </p>
    <pre class="prettyprint"><code class="hljs lasso">ssh username@目标机器ip <span class="hljs-attribute">-p</span> <span class="hljs-number">22</span> <span class="hljs-attribute">-o</span> ProxyCommand<span class="hljs-subst">=</span><span class="hljs-string">'ssh -p 22 username@跳板机ip -W %h:%p'</span></code></pre>
