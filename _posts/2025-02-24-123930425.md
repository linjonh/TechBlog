    <h2>
     <a id="_0">
     </a>
     一、自我介绍
    </h2>
    <p>
     我是一个刚开始学习编程语言的小萌新，带着满脑子的问题开始了我的编程之旅。为了能够加深对编程知识的理解，决定在这分享我的学习经历，以及一些我所学习的知识，希望能够和各位小伙伴一起学习，一起进步。
    </p>
    <hr/>
    <h2>
     <a id="_5">
     </a>
     二、编程目标
    </h2>
    <p>
     我希望能够通过不断的学习，掌握以下的相关知识：
    </p>
    <ol>
     <li>
      能够搭建 c/c++ 开发环境
     </li>
     <li>
      掌握 c/c++ 基本语法
     </li>
     <li>
      掌握算法和数据结构
     </li>
    </ol>
    <hr/>
    <h2>
     <a id="_15">
     </a>
     三、如何去学习：
    </h2>
    <p>
     1、每个问题都要了解透彻
     <br/>
     2、不断的学习一些编程书籍，保持学习的不间断，保证一种良好的学习状态和习惯
    </p>
    <hr/>
    <h2>
     <a id="_21">
     </a>
     四、学习计划
    </h2>
    <p>
     1、每天都要至少做3道编程题
     <br/>
     2、每天都要有敲代码，逐渐熟悉手感
     <br/>
     3、每天至少学习2小时
    </p>
    <hr/>
    <h2>
     <a id="_29">
     </a>
     五、工作目标
    </h2>
    <p>
     想进入腾讯工作
    </p>
    <hr/>
    <h2>
     <a id="_35">
     </a>
     结语
    </h2>
    <p>
     希望在未来的一天，我能够达到我的目标，并为自己的努力而自豪！
    </p>
