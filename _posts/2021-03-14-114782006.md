    <p>
     idea报错 j
     <strong>
      ava: Compilation failed: internal java compiler error
     </strong>
    </p>
    <p>
     <strong>
      原因
     </strong>
     ：主要是因为jdk版本不一致问题
    </p>
    <ul>
     <li>
      <ol>
       <li>
        <strong>
         编译版本
        </strong>
        (
        <mark>
         compile
        </mark>
        )不匹配
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        当前项目jdk版本不支持
       </li>
      </ol>
     </li>
    </ul>
    <p>
     <strong>
      解决
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        查看设置项目的jdk
       </strong>
       <br/>
       File -&gt;Project Structure-&gt;Project Settings -&gt;Project或使用快捷键Ctrl+Alt+shift+S打开项目的jdk配置
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1e157a0bba907ca0b7e2a0ad3c03b730.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看设置工程的jdk
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/79f60854069a1abbf237f0521907d519.png">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e36d9d05f042a09b989a721d8cfb2f01.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看设置java编译器版本
       </strong>
       <br/>
       File–&gt;Setting…–&gt;Build,Execution,Deployment–&gt;
       <code>
        Compiler
       </code>
       –&gt;
       <code>
        Java Compiler
       </code>
       设置相应Module的target bytecode version的合适版本如：jdk1.8
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5ed2700780d46a5d953319244b2b10a6.png"/>
      </p>
     </li>
    </ul>
