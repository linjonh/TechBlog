    <p>
     <strong>
      什么是 Answer ？
     </strong>
    </p>
    <blockquote>
     <p>
      <code>
       Answer
      </code>
      是一个开源的知识型社区软件。您可以使用它快速建立您的问答社区，用于产品技术支持、客户支持、用户交流等。
     </p>
    </blockquote>
    <p>
     <code>
      Answer
     </code>
     是国内
     <code>
      SegmentFault
     </code>
     思否团队开发的技术问答社区，
     <code>
      Answer
     </code>
     不仅拥有搭建问答平台（
     <code>
      Q&amp;A Platform
     </code>
     ）的基础功能，还加入了标签内容管理、评分与声望值体系等社区功能。
    </p>
    <h2>
     <a id="_10">
     </a>
     安装
    </h2>
    <h3>
     <a id="_12">
     </a>
     建数据库
    </h3>
    <p>
     数据库直接用的群晖的
     <code>
      MariaDB 10
     </code>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f8d0d0f828148887840206622f4ec792.png#pic_center"/>
    </p>
    <p>
     在
     <code>
      phpMyAdmin
     </code>
     中创建了名为
     <code>
      answer
     </code>
     的数据库
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8c73e54261d52fc8c03c7215cac6d051.png#pic_center"/>
    </p>
    <blockquote>
     <p>
      为了便于说明，我们假设数据库密码为
      <code>
       123456
      </code>
     </p>
    </blockquote>
    <h3>
     <a id="_Answer_26">
     </a>
     安装 Answer
    </h3>
    <p>
     在群晖上以 Docker 方式安装。
    </p>
    <p>
     在注册表中搜索
     <code>
      answer
     </code>
     ，选择第一个
     <code>
      answerdev/answer
     </code>
     ，版本选择
     <code>
      latest
     </code>
     。
    </p>
    <blockquote>
     <p>
      本文写作时，
      <code>
       latest
      </code>
      版本对应为
      <code>
       1.0.2
      </code>
      ；
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8329fb3c13c17d0dcc4f0ffe182bcaa0.png#pic_center"/>
    </p>

    <h2>
     <a id="_64">
     </a>
     命令行安装
    </h2>
    <p>
     如果你熟悉命令行，可能用
     <code>
      docker cli
     </code>
     更快捷
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 新建文件夹 answer 和 子目录</span>
<span class="token function">mkdir</span> -p /volume2/docker/answer/data

<span class="token comment"># 进入 answer 目录</span>
<span class="token builtin class-name">cd</span> /volume2/docker/answer

<span class="token comment"># 运行容器</span>
<span class="token function">docker</span> run -d <span class="token punctuation">\</span>
--restart unless-stopped <span class="token punctuation">\</span>
--name answer <span class="token punctuation">\</span>
-p <span class="token number">9080</span>:80 <span class="token punctuation">\</span>
-v <span class="token variable"><span class="token variable">$(</span><span class="token builtin class-name">pwd</span><span class="token variable">)</span></span>/data:/data <span class="token punctuation">\</span>
answerdev/answer:latest
</code></pre>
<p>
也可以用
<code>
docker-compose
</code>
安装，将下面的内容保存为
<code>
docker-compose.yml
</code>
文件
</p>
<pre><code class="prism language-yaml"><span class="token key atrule">version</span><span class="token punctuation">:</span> <span class="token string">"3"</span>

<span class="token key atrule">services</span><span class="token punctuation">:</span>
<span class="token key atrule">answer</span><span class="token punctuation">:</span>
<span class="token key atrule">image</span><span class="token punctuation">:</span> answerdev/answer<span class="token punctuation">:</span>1.0.2
<span class="token key atrule">container_name</span><span class="token punctuation">:</span> answer
<span class="token key atrule">restart</span><span class="token punctuation">:</span> on<span class="token punctuation">-</span>failure
<span class="token key atrule">ports</span><span class="token punctuation">:</span>
<span class="token punctuation">-</span> <span class="token string">'9080:80'</span>
<span class="token key atrule">volumes</span><span class="token punctuation">:</span>
<span class="token punctuation">-</span> ./data<span class="token punctuation">:</span>/data
</code></pre>
<p>
然后执行下面的命令
</p>
<pre><code class="prism language-bash"><span class="token comment"># 新建文件夹 answer 和 子目录</span>
<span class="token function">mkdir</span> -p /volume2/docker/answer/data

<span class="token builtin class-name">cd</span> /volume2/docker/answer

<span class="token comment"># 将 docker-compose.yml 放入当前目录</span>

<span class="token comment"># 一键启动</span>
<span class="token function">docker-compose</span> up -d
</code></pre>
