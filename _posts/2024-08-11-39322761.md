    <h5>
     <span style="color:#ff0000">
      window下的环境变量配置
     </span>
    </h5>
    <p>
     默认情况下，在windows下安装python之后，系统并不会自动添加相应的环境变量。此时不能在命令行直接使用python命令。
     <br/>
    </p>
    <p>
     ①首先需要在系统中注册python环境变量：假设python的安装路径为d:\IT\python27，则修改我的电脑-&gt;属性-&gt;高级-&gt;环境变量-&gt;系统变量中的PATH为:
    </p>
    <p>
     ;D:\IT\python27;
    </p>
    <p>
     (为了在命令行模式下运行Python命令，需要将python.exe所在的目录附加到PATH这个环境变量中。)
     <br/>
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140916202733368?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      <br/>
     </img>
    </p>
    <p>
     上述环境变量设置成功之后，就可以在命令行直接使用python命令。或执行"python *.py"运行python脚本了。
     <br/>
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140916202817765?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      <br/>
     </img>
    </p>
    <p>
     ②此时，还是只能通过"python *.py"运行python脚本，若希望直接运行*.py，只需再修改另一个环境变量PATHEXT:
     <br/>
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140916203433040">
      <br/>
     </img>
    </p>
    <p>
     在环境变量PATHEXT中添加   ;.PY;.PYM
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140916203932123?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      <br/>
     </img>
    </p>
    <p>
     上述环境变量设置成功之后，就可以在命令行直接使用“ *.py"运行python脚本了。
     <br/>
    </p>
    <p>
     ③如何使Python解释器能直接import默认安装路径以外的第三方模块？
    </p>
    <p>
     为了能import默认安装路径以外的第三方的模块（如自己写的模块），需要新建PYTHONPATH环境变量，值为这个模块所在的目录。
    </p>
    <p>
     <br/>
    </p>
    <p>
     <br/>
    </p>
    <h5>
     <span style="color:#ff0000">
      notepad++运行python
     </span>
    </h5>
    <p>
     ①打开notepad++，运行F5或者点击运行；
     <br/>
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140916204721984?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      <br/>
     </img>
    </p>
    <p>
     ②
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      这一步是最关键的，将下列语句拷贝粘贴至输入框中：
      <br style="margin:0px; padding:0px"/>
      <a style="margin:0px; padding:0px" target="_blank">
       <span style="color:#ff0000">
        cmd /k python "$(FULL_CURRENT_PATH)" &amp; PAUSE &amp; EXIT
       </span>
      </a>
     </span>
     <span style="font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; font-size:14px; line-height:24px">
      <span style="color:#ff0000">
      </span>
     </span>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      <br style="margin:0px; padding:0px"/>
      单击保存
     </span>
    </p>
    <p>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      解释运行命令：
     </span>
    </p>
    <p>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      cmd /k的含义是执行后面的命令，并且执行完毕后保留窗口. (也就是说，cmd表示打开Command Prompt窗口，且运行跟在/k后边的命令—python)
      <br/>
      cmd /k python == 开始 –&gt; 运行 –&gt; 输入cmd –&gt; 输入python
      <br/>
      $(FULL_CURRENT_PATH)的含义是当前文件的完整路径，这是 Notepad++ 的宏定义
      <br/>
      &amp;是连接多条命令
      <br/>
      PAUSE表示运行结束后暂停，等待一个任意按键
      <br/>
      EXIT表示关闭命令行窗口 （如果使用 cmd /c 就可以省掉 EXIT 了。）
      <br/>
     </span>
    </p>
    <p>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      <img alt="" src="https://img-blog.csdn.net/20140916205035374?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
      </img>
     </span>
    </p>
    <p>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">
      在Shortcut窗口的设置：
      <br/>
      Shortcut窗口只是为了设置运行此命令的快捷键
      <br/>
      其中，Name可以随便输入（例如：Run Python），
      <br/>
      快捷键也可以选择，唯一的规则就是，不要跟已经设置的冲突，否则将不会起效，而且此快捷键可以修改，所以无须担心（例如Ctrl+F5）。
      <br/>
      设置完以后，单击OK保存此命令。
      <br/>
     </span>
    </p>
    <p>
     <span style="margin:0px; padding:0px; font-family:微软雅黑,宋体,Arial,Verdana,arial,serif; line-height:24px; font-size:12px">

     ① 在工具栏点击Preferences，打开Browse Packages。在打开的文件夹中找到Python，并打开这个文件夹。找到文件Python.sublime-build，并打开。
     <br/>
     ②修改以下内容：
     <br/>
     <img alt="" src="https://img-blog.csdn.net/20140916211807557?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvU3VubnlZb29uYQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center"/>
