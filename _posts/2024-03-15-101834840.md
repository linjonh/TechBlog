     <p>
      为了把现实中的数学公式利用到计算机里面，所以在库中已经实现了很多数学函数的代码，我们直接调用就可以了！
      <br/>
      用的时候需要加入头文件
     </p>
     <p>
      #include &lt;math.h&gt;
      <br/>
      #include &lt;math&gt;
      <br/>
      using namespace std;
      <br/>
      常用函数：
     </p>
     <p>
      abs绝对值函数
      <br/>
      acos反余弦函数
      <br/>
      asin反正弦函数
      <br/>
      atan反正切函数
      <br/>
      exp e的x次方
      <br/>
      cos余弦函数
      <br/>
      sin正弦函数
      <br/>
      tan正切函数
      <br/>
      ceil求不小于x的最小整数
      <br/>
      cosh求x的双曲余弦值
      <br/>
      fabs求浮点数x的绝对值
      <br/>
      fmod计算x/y的余数
      <br/>
      frexp把浮点数x分解成尾数和指数
      <br/>
      hypot对于给定的直角三角形的两个直角边，求其斜边的长度。
      <br/>
      ldexp装载浮点数
      <br/>
      log,e为底对数
      <br/>
      log10，10为底对数
      <br/>
      modf，把数分为指数和尾数
      <br/>
      pow，计算x的y次幂
      <br/>
      sinh，求x的双曲正弦值
      <br/>
      sqrt，开方
      <br/>
      tanh求x的双曲正切值
     </p>
