      <a class="postTitle2" href="https://www.cnblogs.com/versionk/p/10618033.html" id="cb_post_title_url" rel="nofollow">
       Python 【导入外部模块的三种方法】
      </a>
     </h2>
     <div class="clear">
     </div>
     <div class="postBody">
      <div class="blogpost-body" id="cnblogs_post_body">
       <p align="left">
       </p>
       <p align="left">
        <span style="font-size:18px;">
         在同一个Python Project内，想要在一个模块内引入另一个模块，只需要import即可。
        </span>
        <br/>
        <span style="font-size:18px;">
         但若有时候想使用位于计算机中的外部py文件内的方法、函数等等，python在当前的Project内找不到py文件，就会到（1）
         <strong>
          PYTHONPATH
         </strong>
         环境变量 （2）
         <em>
          venv —&gt; lib —&gt; set-packages
         </em>
         文件夹的
         <strong>
          PTH
         </strong>
         类型的文件中寻找py文件。
        </span>
        <br/>
        <span style="font-size:18px;">
         有以下三种方法解决该问题。
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         python程序导入外部模块的三种方法
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         【1】 新增PYTHONPATH环境变量，值为所加模块的所在路径
        </span>
        <br/>
        <span style="font-size:18px;">
         如：
        </span>
        <br/>
        <span style="font-size:18px;">
         <img alt="" height="98" src="https://i-blog.csdnimg.cn/blog_migrate/c9012e6216affd6fc1fcd092734bc0f3.png" width="262"/>
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         <img alt="" height="104" src="https://i-blog.csdnimg.cn/blog_migrate/af8d5e753ac19057e46e7aad50768c14.png" width="402"/>
        </span>
        <br/>
       </p>
       <hr/>
       <p align="left">
       </p>
       <p align="left">
        <span style="font-size:18px;">
         【2】
        </span>
        <br/>
        <span style="font-size:18px;">
         <em>
          第一步：打开当前
         </em>
         <em>
          PythonProject
         </em>
         <em>
          的
         </em>
         <em>
          venv—&gt; Lib —&gt; site-packages
         </em>
         <em>
          文件夹第一步：打开当前
         </em>
         <em>
          JavaProject
         </em>
         <em>
          的
         </em>
         <em>
          venv—&gt; Lib —&gt; site-packages
         </em>
         文件夹
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         <img alt="" height="372" src="https://i-blog.csdnimg.cn/blog_migrate/7079c0e5b2eb770fbc543fe66366a729.png" width="323"/>
        </span>
       </p>
       <p align="left">
        <br/>
        <span style="font-size:18px;">
         *第二步：*在文件夹内新建一个PTH类型的文件，写入所加模块的所在路径，保存。
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         <img alt="" height="128" src="https://i-blog.csdnimg.cn/blog_migrate/170fdefc25cf8be158b84f6a47c07f76.png" width="247"/>
        </span>
        <br/>
       </p>
       <hr/>
       <p align="left">
       </p>
       <p>
        <span style="font-size:18px;">
         【3】直接编写代码：
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         import sys
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         sys.path.append(" 所加模块的所在路径")
        </span>
       </p>
       <p align="left">
        <span style="font-size:18px;">
         end
        </span>
       </p>
       <p>
        <span style="font-size:18px;">
