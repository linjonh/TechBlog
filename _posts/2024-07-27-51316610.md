      java
     </span>
     <span style="font-family:宋体; word-wrap:normal; word-break:normal; line-height:21px">
      中，当两个整数相除时，由于小数点以后的数字会被截断，运算结果将为整数，此时若希望得到运算结果为浮点数，必须将两整数其一或是两者都强制转换为浮点数。
     </span>
     <span style="word-wrap:normal; word-break:normal; line-height:21px">
      <br/>
     </span>
     <span style="font-family:宋体; word-wrap:normal; word-break:normal; line-height:21px">
      例如：
     </span>
     <span style="word-wrap:normal; word-break:normal; line-height:21px">
      <br/>
      (float)a/b //
     </span>
     <span style="font-family:宋体; word-wrap:normal; word-break:normal; line-height:21px">
      将整数其中一个强制转换为浮点数，再与另一个整数相除
     </span>
     <span style="word-wrap:normal; word-break:normal; line-height:21px">
      <br/>
      a/(float)b
      <br/>
      (float)a/(float)b //
     </span>
     <span style="font-family:宋体; word-wrap:normal; word-break:normal; line-height:21px">
      将两个整数同时强制转换为浮点数后再相除
     </span>
    </p>
    <p style="">
     <span style="word-wrap:normal; word-break:normal; line-height:21px">
      Java
     </span>
     <span style="font-family:宋体; word-wrap:normal; word-break:normal; line-height:21px">
      代码
     </span>
     <span style="word-wrap:normal; word-break:normal; line-height:21px">
      <wbr/>
      <wbr/>
     </span>
    </p>

      <wbr/>
