      b，移动窗体.快速或慢速移动窗体，背景及窗体本身刷新必须正确;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      c，缩放窗体，窗体上的控件应随窗体的大小变化而变化;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      d，显示分辨率.必须在不同的分辨率的情况下测试程序的显示是否正常;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      进行测试时还要注意状态栏是否显示正确;工具栏的图标执行操作是否有效，是否与菜单懒中图标显示一致;错误信息内容是否正确，无错别字，且明确等等;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      <strong>
       2)控件
      </strong>
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      测试方法:
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      a，窗体或控件的字体和大小要一致;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      b，注意全角，半角混合
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      c，无中英文混合.
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      <strong>
       3)菜单
      </strong>
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      进行测试时要注意
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      a，选择菜单是否可以正常工作，并与实际执行内容一致;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      b，是否有错别字:
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      c，快捷键是否重复;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      d，热键是否重复;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      e，快捷键与热键操作是否有效
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      f，是否存在中英文混合
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      g，菜单要与语境相关，如，不同权限的用户登陆一个应用程序，不同级别的用户可以看到不同级别的菜单并使用不同级别的功能;
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      h，鼠标右键快捷菜单
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      <strong>
       4)特殊属性
      </strong>
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      1，安装界面应有公司介绍或产品介绍，有公司的图标
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      2，主界面及大多数界面最好有公司图标
     </span>
    </p>
    <p style="padding-bottom:0px; line-height:28px; background-color:#ffffff; margin-top:0px; padding-left:0px; padding-right:0px; font-family:simsun; margin-bottom:0px; color:#333333; font-size:14px; padding-top:0px">
     <span style="font-size:18px">
      3，选择"帮助"-&gt;"关于"命令，应 看见相关版权和产品信息
     </span>
    </p>
