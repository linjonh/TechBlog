    <h6>
     <a id="_0">
     </a>
     一、问题说明
    </h6>
    <p>
     微信小程序在使用
     <code>
      qs
     </code>
     库时，出现下面的错误信息：
    </p>
    <pre><code class="prism language-shell">module <span class="token string">"miniprogram_npm/side-channel/call-bind/callBound"</span> is not defined<span class="token punctuation">;</span>
</code></pre>
    <hr/>
    <h6>
     <a id="_10">
     </a>
     二、解决方案
    </h6>
    <p>
     原因是 qs 版本太高，无法使用，把
     <code>
      "qs": "^6.0.0"
     </code>
     换成
     <code>
      "qs": "^5.2.1"
     </code>
     。
    </p>
    <pre><code class="prism language-js"><span class="token comment">// 1、package.json 文件 降低qs版本</span>
<span class="token string">"dependencies"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">"qs"</span><span class="token operator">:</span> <span class="token string">"^5.2.1"</span>
 <span class="token punctuation">}</span>

<span class="token comment">// 2、重新安装 qs</span>
npm install

<span class="token comment">// 即可解决</span>
</code></pre>
