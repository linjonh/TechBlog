     之前的RubyChina社区上发起了一个讨论：最近有看中什么新的编程语言么？不少网友都列出了自己喜欢的编程语言，在这里，我们搜集了一些大家耳熟能详的小众编程语言，或许在你看来它们根本不是什么小众语言。

    <div class="con news_content" style="margin:0px 0px 30px; padding:0px; list-style:none">
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">

       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/09/536c3646e8e4e.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      估计大家比较熟悉的小众语言都有Google的Dart、Facebook的Hack和WhatsApp的Erlang等等，但成名之后估计它们已经是大众语言了。除此之外，我想还有一些语言你可能并不了解。下面我们就一一列出这些害羞的语言与你共分享！
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       1.
       <a href="http://golang.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Go
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      是基于Inferno操作系统所开发的。Go语言于2009年11月正式宣布推出，成为开放源代码项目，并在Linux及Mac OS X平台上进行了实现，后追加Windows系统下的实现。Go是Google开发的一种编译型、并发型编程语言，并具有垃圾回收功能。
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b22c62b36f.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b22c62b36f.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       2.
       <a href="http://elixir-lang.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Elixir
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Elixir 是一种函数式编程语言，建立在Erlang虚拟机之上。它是一种动态语言,灵活的语法与宏支持,利用Erlang的能力来构建并发、分布式、容错应用程序与热代码升级。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b246280953.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b246280953_middle.jpg?_=5096" style="vertical-align:middle; border:none; float:none; margin:0px"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       3.
       <a href="http://www.rust-lang.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Rust
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Rust是一种系统编程语言，运行速度快，几乎不存在死机现象，并消除了数据争用。Rust由Web语言的领军人物Brendan Eich（JavaScript之父），Dave Herman以及Mozilla公司的Graydon Hoare 合力开发。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b1814d3c82.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b1814d3c82.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       4.
       <a href="http://www.scala-lang.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Scala
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Scala编程语言近来抓住了很多开发者的眼球。作为一种纯粹的
      <a href="http://baike.baidu.com/view/125370.htm" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       面向对象
      </a>
      编程语言，Scala天衣无缝地结合了命令式和函数式的编程风格。有人说，Scala可能是下下一代Java。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b1a691e4bc.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b1a691e4bc_middle.jpg?_=44517" style="vertical-align:middle; border:none; float:none; margin:0px"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       5.
       <a href="http://www.erlang.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Erlang
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      我想大家对Erlang并不陌生，就在前段时间Facebook收购了移动IM（即时通讯）公司WhatsApp的早期架构就是使用Erlang来做优化的。Erlang是一种通用的面向并发的编程语言，它由
      <a href="http://baike.baidu.com/view/14445.htm" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       瑞典
      </a>
      电信设备制造商爱立信所辖的CS-Lab开发，目的是创造一种可以应对大规模并发活动的编程语言和运行环境。
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b1d5a99a47.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b1d5a99a47_middle.jpg?_=50506" style="vertical-align:middle; border:none; float:none; margin:0px"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       6.
       <a href="http://mth.github.io/yeti/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Yeti
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      是一种
      <a href="http://en.wikipedia.org/wiki/ML_%28programming_language%29" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       ML
      </a>
      形式的函数式编程语言，运行在
      <a href="http://java.sun.com/docs/books/jvms/second_edition/html/VMSpecTOC.doc.html" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       JVM
      </a>
      上。
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b1f2836afb.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b1f2836afb.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       7.
       <a href="http://www.pyret.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Pyret
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Pyret是一种主要是为编程教育为主要用途的杰出的编程语言，你可以使用Pyret探索脚本和函数式编程的融合。使用Pyret进行开发的项目比较活跃，而且供免费使用或修改。
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b2604b588f.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b2604b588f.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       8.
       <a href="http://nimrod-code.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Nimrod
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Nimrod是一个新型的静态类型、命令式编程语言，支持过程式、函数式、面向对象和泛型编程风格而保持简单和高效。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b270fc3a03.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b270fc3a03.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       9.
       <a href="http://chapel.cray.com/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Chapel
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Chapel语言的语法有很多源头，除了常见的如C，C++，Java外，它还借鉴了一些科学研究性语言（比如Fortran和Matalb）里的概念。这种语言专门为超级计算机和集群设计的，它是
      <a href="http://en.wikipedia.org/wiki/Cray" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       Cray
      </a>
      的Cascade研究课题的一部分，由美国国防部高级研究计划局(DARPA)参与启动，有一个宏大的高性能计算设想。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b2905a6e73.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
       <img alt="" border="0" src="http://cms.csdnimg.cn/article/201405/08/536b2905a6e73.jpg" style="vertical-align:middle; border:none"/>
      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       10.
       <a href="http://whiley.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">
        Whiley
       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Whiley是一个轻量级面向对象和函数式编程语言，目标代码在 Java 虚拟机上执行。Whiley在编译时就可以排除很多错误，例如除0、数组越界和空引用等。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
      <a href="http://cms.csdnimg.cn/article/201405/08/536b2bb4beb81.jpg" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">

      </a>
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      <strong>
       11.
       <a href="http://clojure.org/" rel="nofollow noopener noreferrer" style="color:rgb(0,102,204); text-decoration:none" target="_blank">

       </a>
      </strong>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px">
      Clojure是一个在JVM平台运行的动态函数式编程语言，其语法接近于LISP语言，在JVM平台运行的时候，会被编译为JVM的字节码进行运算。
      <br/>
     </p>
     <p style="margin-top:0px; margin-bottom:1.5em; padding-top:0px; padding-bottom:0px; list-style:none; color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; font-size:14px; line-height:24px; text-align:center">
