    <p>
     HarmonyOS为我们提供了基础的定时器能力——Timer (定时器)
    </p>
    <p>
     这里介绍setInterval和clearInterval的使用
    </p>
    <p>
     这里我们以累加为例，
    </p>
    <p>
     点击【开始累加】按钮进行累加，实现每一秒加1的效果
    </p>
    <p>
     <img alt="" height="760" src="https://i-blog.csdnimg.cn/blog_migrate/347b17803336d945a1b02e75fd91793c.png" width="1200"/>
    </p>
    <p>
     代码：
    </p>
    <pre><code class="hljs">@Entry
@Component
struct Index {
  @State num: number = 0
  @State x: number = 1

build() {
Column() {
Row() {
}.height(200)

      Text(`${this.num}`)
      Button('开始累加').onClick(() =&gt; {
        // 点击后每1秒加1
        setInterval(() =&gt; {
          this.num++
        }, 1000)
      })
    }
    .height('100%')
    .width('100%')

}
}</code></pre>
<p>
在上述的基础上进行操作，进行点击【开始累加】后累加，以及点【停止】后停止累加，再次点击后【开始累加】后实现继续累加的效果
</p>
<p>
<img alt="" height="904" src="https://i-blog.csdnimg.cn/blog_migrate/9a8a73060ddeb40a3790e6c303c126e2.png" width="1200"/>
</p>
<p>
代码：
</p>
<pre><code class="hljs">@Entry
@Component
struct Index {
@State num: number = 0
@State x: number = 1

build() {
Column() {
Row() {
}.height(200)

      Text(`${this.num}`)
      Button('开始累加').onClick(() =&gt; {
        // 点击后每1秒加1
        this.x = setInterval(() =&gt; {
          this.num++
        }, 1000)
      })
      Button('停止').onClick(() =&gt; {
        // 点击停止累加，再次点击【开始累加】按钮继续累加
        clearInterval(this.x)
      })
    }
    .height('100%')
    .width('100%')

}
}</code></pre>
