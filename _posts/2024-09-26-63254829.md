     <h2>
      CCS安装过程
     </h2>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       下载安装包之后，我们一起来看下安装过程：
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       1.解压我们下载的安装包，双击解压之后的文件夹里面的
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       ccs_setup_6.1.0.00104.exe
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       开始我们的安装过程，双击打开之后显示截图如下：
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       <img alt="" src="https://img-blog.csdn.net/20170318152413149?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenpmZW5nbGlu/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
        <br/>
       </img>
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
     </p>
     <p align="justify">
      <span style="font-size:18px">
       点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       是(Y)
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，然后显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20170318152455822?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenpmZW5nbGlu/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
       <br/>
      </img>
     </span>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
     </p>
     <p align="justify">
      <span style="font-size:18px">
       继续点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       是(Y)
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，弹出安装界面，显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20170318152524644">
       <br/>
      </img>
     </span>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       2.勾选
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       I accept ......
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，然后点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       Next
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，操作执行之后显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20170318155211351?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenpmZW5nbGlu/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
      </img>
     </span>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       安装路径设置界面，默认是
      </span>
      <span style="font-size:18px; font-family:宋体; line-height:26px">
       “
      </span>
      <span style="font-size:18px">
       c:\ti
      </span>
      <span style="font-size:18px; font-family:宋体; line-height:26px; text-align:justify">
       ”
      </span>
      <span style="font-size:18px">
       ，可以点路径后面的
      </span>
      <span style="font-size:18px; font-family:宋体; line-height:26px; text-align:justify">
       “
      </span>
      <span style="font-size:18px">
       Browse
      </span>
      <span style="font-size:18px; font-family:宋体; line-height:26px; text-align:justify">
       ”
      </span>
      <span style="font-size:18px">
       按钮更换路径，根据自己的情况来修改即可。建议新手用默认路径。
      </span>
     </p>
     <p>
      <span style="font-size:18px">
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       3.路径确定之后，继续点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       Next
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，操作执行之后显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20170318152643776?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenpmZW5nbGlu/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
       <br/>
      </img>
     </span>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
     </p>
     <p align="justify">
      <span style="font-size:18px">
       按照截图中的配置，点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       SimpleLink Wireless MCUs
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，然后在下拉选项中勾选
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       CC26xx Device Support
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       和
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       TI ARM Complier
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       。
      </span>
     </p>
     <p align="justify">
      <span style="font-size:18px">
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       4.选择好支持的平台和编译器之后，继续点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       Next
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，操作执行之后显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20170318152721839?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenpmZW5nbGlu/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
       <br/>
      </img>
     </span>
     <p>
      <span style="font-size:18px">
       <br/>
      </span>
     </p>
     <p>
     </p>
     <p align="justify">
      <span style="font-size:18px">
       用默认选择的
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       TI XDS Debug Probe Support
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，选择调试调试仿真器。
      </span>
     </p>
     <p align="justify">
      <span style="font-size:18px">
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       5.继续点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       Next
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        ”
       </span>
       ，操作执行之后显示截图如下：
      </span>
     </p>
     <span style="font-size:18px">

       <br/>
      </img>
     </span>

       <br/>

     <p align="justify">
      <span style="font-size:18px">
       暂时先不配置，当然，也可以根据自己的需要选择，此处不选择也可以。
      </span>
     </p>
     <p align="justify">
      <span style="font-size:18px">
      </span>
     </p>
     <p>
      <span style="font-size:18px">
       6.直接点击
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
        “
       </span>
       Finish
       <span style="font-family:宋体; font-size:18px; line-height:26px; text-align:justify">
