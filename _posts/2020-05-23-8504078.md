    <h2 style="text-align: center;">
     Web设计师的五个排版工具
    </h2>
    <br/>
    <strong>
     作者：chszs，转载需注明。博客主页：http://blog.csdn.net/chszs
    </strong>
    <br/>
    排版是Web设计的一个重要方面，用户界面和用户体验设计是开发网站的挑战性工作之一。
    <br/>
    <br/>
    本文探讨一些排版工具。
    <br/>
    <br/>
    <h3>
     1. TypeTester
    </h3>
    TypeTester可以在屏幕上比较不同的字体和样式。
    <br/>
    地址：http://www.typetester.org/
    <br/>
    <br/>
    <img alt="" src="https://img-my.csdn.net/uploads/201301/15/1358212609_3233.jpg">
     <br/>
     <br/>
     <h3>
      2. Fontifier
     </h3>
     Fontifier允许Web设计师使用手写的方式写文本。
     <br/>
     地址：http://www.fontifier.com/
     <br/>
     <br/>
     <img alt="" src="https://img-my.csdn.net/uploads/201301/15/1358212618_7429.jpg">
      <br/>
      <br/>
      <h3>
       3. WhatTheFont
      </h3>
      WhatTheFont：通过上传图片或者是指定图片的URL，就可以知道使用的是什么字体。
      <br/>
      地址：http://www.myfonts.com/WhatTheFont/
      <br/>
      <br/>
      <img alt="" src="https://img-my.csdn.net/uploads/201301/15/1358212627_6458.jpg">
       <br/>
       <br/>
       <h3>
        4. CSS Type Set
       </h3>
       CSS Type Set：让网页设计师可以尝试各种样式和属性的组合。
       <br/>
       地址：http://csstypeset.com/
       <br/>
       <br/>
       <img alt="" src="https://img-my.csdn.net/uploads/201301/15/1358212637_6302.jpg">
        <br/>
        <br/>
        <h3>
         5. FontStruct
        </h3>
        FontStruct：让网页设计师可以快速、简单地创建字体。
        <br/>
        <p>
         地址：http://fontstruct.com/
        </p>
