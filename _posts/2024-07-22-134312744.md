    <h3>
     <a id="_36">
     </a>
     二.根据关键词搜索
    </h3>
    <p>
     <strong>
      关键词
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        关键词
       </th>
       <th>
        含义
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        name
       </td>
       <td>
        项目名称
       </td>
      </tr>
      <tr>
       <td>
        description
       </td>
       <td>
        项目描述
       </td>
      </tr>
      <tr>
       <td>
        readme
       </td>
       <td>
        项目帮助文档
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      语法
     </strong>
    </p>
    <pre><code class="prism language-apl">需要检索的内容（后有空格） in:name或description或readme
单独使用
1)eg:检索项目名称包含sensor的项目
     sensor in:name
2)eg:检索项目描述包含sensor的项目
     sensor in:description
3)eg:检索readme包含sensor的项目
     sensor in:readme
组合使用语法：xxx关键字  stars:数字1..数字2 fork:数字1..数字2
Eg:查找arm stars数从500到2000 fork数从200到2000的项目
arm stars:500..2000 fork:200..2000
</code></pre>
    <pre><code class="prism language-apl">常用:
eg:检索readme包含sensor的项目
     sensor in:readme
</code></pre>

    <h3>
     <a id="_stars_72">
     </a>
     三.根据 stars
    </h3>
    <p>
     活 fork 关键字查找
    </p>
    <p>
     <strong>
      语法
     </strong>
    </p>
    <pre><code class="prism language-apl">单独使用语法
1)数量范围： xxx关键词 stars:&gt;=或者&lt;=
eg:查找stars数量大于等于2000的sensor项目
     sensor stars:&gt;=2000
2)区间范围：xxx关键词 fork:数字1..数字2
eg:查找fork数量500到2000的sensor项目
     sensor fork:500..2000
</code></pre>
