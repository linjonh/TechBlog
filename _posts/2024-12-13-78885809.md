    <h4 id="notepad配置java开发环境">
     notepad++配置java开发环境
    </h4>
    <ul>
     <li>
      <p>
       下载nppexec插件
      </p>
      <ul>
       <li>
        如果你安装的notepad++有Plugin Manager
       </li>
      </ul>
      <pre class="prettyprint"><code class="hljs haskell">菜单栏<span class="hljs-type">Plugins</span>-&gt;<span class="hljs-type">Plugin</span> <span class="hljs-type">Manager</span>-&gt;show <span class="hljs-type">Plugin</span> <span class="hljs-type">Manager</span>-&gt;available
找到nppexec插件安装即可</code></pre>
      <ul>
       <li>
        针对64位和部分32位notepad++无Plugin Manager
       </li>
      </ul>
      <pre class="prettyprint"><code class="hljs avrasm">下载PluginManager<span class="hljs-preprocessor">.dll</span>复制到你安装notepad++的plugins目录,重启notepad++
之后跟上面的步骤一样
下载地址 http://download<span class="hljs-preprocessor">.csdn</span><span class="hljs-preprocessor">.net</span>/download/zycxnanwang/<span class="hljs-number">10171315</span></code></pre>
     </li>
     <li>
      <p>
       配置java命令
      </p>
      <pre class="prettyprint"><code class="hljs bash">npp_save  
javac <span class="hljs-string">"<span class="hljs-variable">$(FILE_NAME)</span>"</span>
java <span class="hljs-string">"<span class="hljs-variable">$(NAME_PART)</span>"</span></code></pre>
      <ul>
       <li>
        <strong>
         解释命令
        </strong>
       </li>
      </ul>
      <p>
       npp_save 保存当前文件
      </p>
      <p>
       javac “$(FILE_NAME)” 编译当前java文件
      </p>
      <p>
       java “$(NAME_PART)” 运行当前java文件
      </p>
      <ul>
       <li>
        按F6,出现nppexec的命令弹窗，把命令复制粘贴进去，并命名为java,保存
       </li>
      </ul>
      <p>
       <img alt="这里写图片描述" src="https://img-blog.csdn.net/20171224165149005?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenljeG5hbndhbmc=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
      </p>
     </li>
     <li>
      <p>
       配置宏命令
      </p>
      <pre class="prettyprint"><code class="hljs applescript">NppExec-&gt;Adanced Options
<span class="hljs-number">1.</span>把刚才的java命令脚本添加进去
<span class="hljs-number">2.</span><span class="hljs-property">item</span> <span class="hljs-property">name</span> 命名为JAVA
<span class="hljs-number">3.</span>勾选 Place <span class="hljs-keyword">to</span> Macros submenus的确认框</code></pre>
      <p>
       <img alt="这里写图片描述" src="https://img-blog.csdn.net/20171224165218801?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvenljeG5hbndhbmc=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
      </p>
     </li>
     <li>
      <p>
       配置快捷键
      </p>
      <pre class="prettyprint"><code class="hljs haskell"><span class="hljs-title">settings</span>-&gt;<span class="hljs-type">ShortCut</span> <span class="hljs-type">Mapper</span>-&gt;<span class="hljs-type">Plugins</span> <span class="hljs-type">Command</span> 找到刚才保存的<span class="hljs-type">JAVA</span></code></pre>
     </li>
    </ul>
