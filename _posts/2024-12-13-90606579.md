     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">

      </span>
     </p>
     <p style="color:rgb(34,38,53);font-size:19px;font-family:Arial, Helvetica, sans-serif;">
      <span style="font-size:16px;">
       人工智能
      </span>
      <span style="font-size:16px;vertical-align:inherit;">
       已经存在了很长时间。然而，由于各种库和框架的发展，该领域发生了巨大改进。
      </span>
      <br/>
     </p>
     <p style="color:rgb(34,38,53);font-size:19px;font-family:Arial, Helvetica, sans-serif;">
      <span style="vertical-align:inherit;font-size:16px;">
       本文，我们将研究人工智能的高质量库的优缺点，以及它们特点。让我们潜入并探索这些AI库的世界！
      </span>
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      TensorFlow
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="font-size:19px;">
      <br/>
     </p>
     <p style="font-size:19px;">
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/2be79e0c10ec9e20af9297e7606cf783.jpeg">
       <br/>
      </img>
     </p>
     <p style="font-size:19px;">
      <br/>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        语言：C++或Python
       </span>
      </strong>
      <br/>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       当你进入AI时，你会听到的第一个框架之一就是Google的TensorFlow。TensorFlow是一个使用数据流图进行数值计算的开源框架。这个框架被称为允许在任何CPU或GPU上进行计算的框架，无论是台式机，服务器还是移动设备，另外这个框架在Python编程语言中是可用的，这也是Python大火的原因。
      </span>
     </p>
     <p style="font-size:19px;">
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/9414f3dd2e06207e4b7987c2be7ef2c6.jpeg"/>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        优点：
       </span>
      </strong>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       使用易于学习的语言（Python）；
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       使用计算图抽象；
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       可以使用可视化的TensorBoard；
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        缺点：
       </span>
      </strong>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       很慢，因为Python不是编程语言中最快的；
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       缺乏许多预先训练的模型；
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       不完全开源；
      </span>
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      CNTK
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="font-size:19px;">
      <br/>
     </p>
     <p style="font-size:19px;">
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/1d4122022ac3e312f52d4efc4d327783.jpeg"/>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        语言：C++
       </span>
      </strong>
      <br/>
     </p>
     <p style="font-size:19px;">
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       我们可以称之为它是微软对Google的TensorFlow的回应。
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       微软的CNTK是一个增强分离计算网络模块化和维护的库，它提供了学习算法和模型描述。在需要大量服务器进行操作的情况下，CNTK可以同时利用多台服务器。
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p style="font-size:19px;">
      <span style="font-size:16px;">
       据说它的功能与Google的TensorFlow相近，但是，它更快，在这里了解更多。
      </span>
     </p>
     <p style="font-size:19px;">
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/95767c40530b482a98ece2105b4642c3.jpeg"/>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        优点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       非常灵活；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       允许分布式培训；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       支持C++，C#，Java和Python；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        缺点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       它以一种新的语言——Network Description Language（NDL）来实现；
      </span>
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      Theano
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <p>
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/2ee392779acb53e695daa40d109fd43a.png"/>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        语言：Python
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       Theano是TensorFlow的强有力竞争者，它是一个功能强大的Python库，允许以高效率的方式进行多维数组的数值操作。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       该库使用GPU来执行数据密集型计算而不是CPU，因此操作效率很高。出于这个原因，Theano已经被用于为大规模的计算密集型操作长达10年的时间。然而，于2017年9月，Theano的1.0版本停止。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       但这并不意味着它不再是一个强大的库，仍然可以随时进行深入的学习研究。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/8335d39e9c556731dd00fb05cb0cc3ab.jpeg"/>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        优点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       优化CPU和GPU；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       有效的计算任务；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        缺点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       与其他库相比，原生Theano有点低级；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       需要与其他库一起使用，以获得高度的抽象；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       AWS使用它有点bug；
      </span>
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      Caffe
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p style="text-align:center;">
      <strong>
       “快速，开放的深度学习框架”
      </strong>
     </p>
     <br/>
     <p>
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        语言：C++
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       Caffe是一个强大的深度学习框架，像这个清单上的其他框架一样，深度学习的研究速度非常快。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       借助Caffe，可以非常轻松地构建用于图像分类的卷积神经网络。Caffe在GPU上运行的也很不错，这有助于在运行期间提高速度。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       Caffe主类：
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/fa55579575db65d85d49ec416e6f0e85.jpeg"/>
     </p>
     <p>
      <strong>
       优点：
      </strong>
     </p>
     <p>
      Python和MATLAB都可用；
     </p>
     <p>
      表现的很好；
     </p>
     <p>
      无需编写代码即可进行模型的训练；
     </p>
     <p>
      <br/>
     </p>
     <p>
      <strong>
       缺点：
      </strong>
     </p>
     <p>
      对于RNN网络不太友好；
     </p>
     <p>
      对于新体系结构不太友好；
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      Keras
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <br/>
     <p style="text-align:center;">
      <strong>
       “为人类普及深度学习”
      </strong>
     </p>
     <br/>
     <p>
      <strong>
       <br/>
      </strong>
     </p>
     <p>
      <strong>
       语言：Python
      </strong>
     </p>
     <p>
      <strong>
       <br/>
      </strong>
     </p>
     <p>
      Keras是一个用Python编写的开源的神经网络。与TensorFlow，CNTK和Theano不同，Keras并不意味着是一个端到端的机器学习框架。
     </p>
     <p>
      <br/>
     </p>
     <p>
      相反，它作为一个接口，提供一个高层次的抽象，这使得神经网络的配置变得简单，无论它在哪个框架上。
     </p>
     <p>
      <br/>
     </p>
     <p>
      Google的TensorFlow目前支持Keras作为后端，而微软的CNTK也会在很短的时间内做到这一点。
     </p>
     <p>
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/7988fa5226a3dc9be13c66164cb88b0d.jpeg"/>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        优点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       对用户非常友好；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       很容易扩展；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       在CPU和GPU上无缝运行；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       与Theano和TensorFlow无缝工作；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        缺点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       不能有效的用作独立的框架；
      </span>
     </p>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <p style="min-width:1em;color:rgb(28,95,195);font-size:16px;font-weight:bold;">
      Torch
     </p>
     <span style="width:8px;border-width:1px;border-style:solid;border-color:rgb(28,95,195);vertical-align:middle;">
     </span>
     <span style="border-bottom:1px solid rgb(28,95,195);vertical-align:middle;">
     </span>
     <p>
      <br/>
     </p>
     <p>
      <span style="font-size:16px;">
      </span>
     </p>
     <p style="text-align:center;">
      <strong>
       “一个开源的机器学习库”
      </strong>
     </p>
     <br/>
     <p>
      <br/>
      <span style="font-size:16px;">
      </span>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        语言：C
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        <br/>
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       Torch是一个用于科学和数字操作的开源机器学习库。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       这是一个基于Lua编程语言的库而不是Python。
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       它通过提供大量的算法，使得深度学习研究更容易，并且提高了效率和速度。它有一个强大的N维数组，这有助于切片和索引等操作。它还提供了线性代数程序和神经网络模型。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p>
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/3a99bacafed7dc7ebbd50631176ebae0.jpeg"/>
     </p>
     <p>
      <strong>
       <span style="font-size:16px;">
        优点：
       </span>
      </strong>
     </p>
     <p>
      <span style="font-size:16px;">
       非常灵活；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       高水平的速度和效率；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       大量的预训练模型可用；
      </span>
     </p>
     <p>
      <span style="font-size:16px;">
       <br/>
      </span>
     </p>
     <p>
      <strong>

      </span>
