     2.这里没有
     <code>
      grid组件
     </code>
     ，可能是vant版本的问题，如果是版本问题，建议删除老版本，安装新版本，我在这篇中有安装指定版本的vant
    </h6>
    <h2>
     <a id="15npmvant_4">
     </a>
     小程序学习（15）：使用npm，以vant为例
    </h2>
    <h4>
     <a id="1packagelockjson_5">
     </a>
     1.点击操作，创建package-lock.json
    </h4>
    <p>
     点击详情=》构建npm，
     <br/>
     然后会自动创建package-lock.json文件
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/466459c2eca0d68cbefd1a0ea9ad3064.png"/>
    </p>
    <h4>
     <a id="2packagejson_9">
     </a>
     2.点击操作及命令行，创建package.json
    </h4>
    <p>
     2.1 点击版本管理=&gt;初始化git库=&gt;在终端打开
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b1c8add268624ab7e5695ceb918648cc.png">
      <br/>
      2.2 小黑框命令行创建package.json
      <br/>
      可以
     </img>
    </p>
    <pre><code class="prism language-c">npm init <span class="token operator">-</span>y
</code></pre>
    <p>
     或者，然后一路回车，（其中可以设置，但是不懂的就别盲目设置了）
    </p>
    <pre><code class="prism language-c">npm init
</code></pre>
    <h4>
     <a id="3vant_22">
     </a>
     3.下载vant
    </h4>
    <p>
     还是在终端小黑框
     <br/>
     3.1初始化
    </p>
    <pre><code class="prism language-cpp">npm init
</code></pre>
    <h6>
     <a id="32_vant_28">
     </a>
     3.2 安装vant（重点）
    </h6>
    <pre><code class="prism language-c">npm install <span class="token operator">--</span>save vant<span class="token operator">-</span>weapp@<span class="token number">1.0</span><span class="token number">.0</span><span class="token operator">-</span>beta<span class="token punctuation">.</span><span class="token number">0</span>
</code></pre>
    <p>
     <strong>
      注：
     </strong>
     这个是安装指定版本，@符号后面是版本号，
     <br/>
     因为现在网站上的是
     <a href="https://youzan.github.io/vant-weapp/#/quickstart" rel="nofollow">
      vant小程序https://youzan.github.io/vant-weapp/#/quickstart
     </a>
    </p>
    <p>
     如果这样安装
    </p>
    <pre><code class="prism language-c">npm i vant<span class="token operator">-</span>weapp <span class="token operator">-</span>S <span class="token operator">--</span>production
</code></pre>
    <p>
     会默认安装0.5的版本，我在使用中，0.5的版本少很多东西，比如这个组件就没有
     <br/>
     <strong>
      2020年4月16日 修改：已经不会默认安装老版本，可使用官方方法直接安装
     </strong>
    </p>
    <pre><code class="prism language-json"><span class="token string">"van-grid"</span><span class="token punctuation">:</span> <span class="token string">"vant-weapp/grid/index"</span><span class="token punctuation">,</span>
<span class="token string">"van-grid-item"</span><span class="token punctuation">:</span> <span class="token string">"vant-weapp/grid-item/index"</span>
</code></pre>
    <p>
     所以最好安装最新版的，和网址上面的一样，
    </p>
    <h4>
     <a id="4npm_49">
     </a>
     4.构建npm
    </h4>
    <p>
     这个没啥好说的，工具=&gt;构建npm
    </p>
    <h4>
     <a id="vant_53">
     </a>
     看vant版本
    </h4>
    <p>
     打开package.json
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/832d92cbbef89bbbebd3d6857adfcfd0.png">

      或打开package-lock.json

    </p>
