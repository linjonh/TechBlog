     一、快速生成HTML5的头部信息的步骤：
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     1.Ctrl + N，新建一个文档；
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     2.Ctrl + Shift + P，打开命令模式，再输入 sshtml 进行模糊匹配，将语法切换到html模式；切换语法
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     3.输入 !，再按下 Tab键或者 Ctrl + E ，就能快速打开HTML5的整体结构。
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     二、十分常用的快捷键：
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     Ctrl + P，打开搜索框。具体操作为：1、输入当前项目中的文件名，快速搜索文件，2、输入@和关键字，查找文件中函数名，3、输入：和数字，跳转到文件中该行代码，4、输入#和关键字，查找变量名。
     <br style=""/>
     command + F 搜索当前文件函数、方法
     <br style=""/>
     Ctrl + 回车，在当前行的下一行添加一行空行。
     <br style=""/>
     Ctrl + Shift + 回车，在当前行的上一行添加一行空行。
     <br style=""/>
     Ctrl + Z，撤销。
     <br style=""/>
     Ctrl + Y，取消撤销。
     <br style=""/>
     Ctrl + ]，或者Tab键，向右缩进。
     <br style=""/>
     Ctrl + [，或者Shift +Tab，向左缩进。
     <br style=""/>
     Ctrl + Shift + V，粘贴过程中保持缩进。
     <br style=""/>
     Alt + F3，选中选择的词。
     <br style=""/>
     “&gt;”号，子元素选择符；“.”号，后面跟的是class名；“$”符，代表的是序号。
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     举个例子：输入 ul&gt;.item$*10,再按下Ctrl + E，则会一次性输出一个无需列表，列表中有10个带有class名为item的 li 行。
    </p>
    <p style='margin-top:0px; margin-bottom:1.1em; padding-top:0px; padding-bottom:0px; font-family:"Microsoft YaHei"; border:0px; outline:0px; font-size:15px; vertical-align:baseline; background:transparent; color:rgb(102,102,102); line-height:26px'>
     无疑这种一次性输入多行的方式十分快捷。如果多行有同一种操作，可以运用多行游标，Alt + F3会选中鼠标之前选择的词，此时多行被选中，按方向键可以选择
    </p>
    <h4 id="选择类" style='font-family:"Microsoft YaHei"; font-weight:100; line-height:1.1; color:rgb(153,153,153); margin:0.8em 0px; font-size:14px; padding:0px; border:0px; outline:0px; vertical-align:baseline; background:transparent'>
     选择类
    </h4>
    <ul style='margin:0px 0px 22px; padding:0px; font-family:"Microsoft YaHei"; list-style:square inside; border:0px; outline:0px; font-size:14px; vertical-align:baseline; background:transparent; color:rgb(153,153,153)'>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+D 选中光标所占的文本，继续操作则会选中下一个相同的文本。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Alt+F3 选中文本按下快捷键，即可一次性选择全部的相同文本进行同时编辑。举个栗子：快速选中并更改所有相同的变量名、函数名等。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+L 选中整行，继续操作则继续选择下一行，效果和 Shift+↓ 效果一样。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+L 先选中多行，再按下快捷键，会在每行行尾插入光标，即可同时编辑这些行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+M 选择括号内的内容（继续选择父括号）。举个栗子：快速选中删除函数中的代码，重写函数体代码或重写括号内里的内容。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+M 光标移动至括号内结束或开始的位置。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Enter 在下一行插入新行。举个栗子：即使光标不在行尾，也能快速向下插入一行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+Enter 在上一行插入新行。举个栗子：即使光标不在行首，也能快速向上插入一行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+[ 选中代码，按下快捷键，折叠代码。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+] 选中代码，按下快捷键，展开代码。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+K+0 展开所有折叠代码。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+← 向左单位性地移动光标，快速移动光标。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+→ 向右单位性地移动光标，快速移动光标。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      shift+↑ 向上选中多行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      shift+↓ 向下选中多行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Shift+← 向左选中文本。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Shift+→ 向右选中文本。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+← 向左单位性地选中文本。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+→ 向右单位性地选中文本。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+↑ 将光标所在行和上一行代码互换（将光标所在行插入到上一行之前）
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+↓ 将光标所在行和下一行代码互换（将光标所在行插入到下一行之后）。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Alt+↑ 向上添加多行光标，可同时编辑多行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Alt+↓ 向下添加多行光标，可同时编辑多行。
     </li>
    </ul>
    <h4 id="编辑类" style='font-family:"Microsoft YaHei"; font-weight:100; line-height:1.1; color:rgb(153,153,153); margin:0.8em 0px; font-size:14px; padding:0px; border:0px; outline:0px; vertical-align:baseline; background:transparent'>
     编辑类
    </h4>
    <ul style='margin:0px 0px 22px; padding:0px; font-family:"Microsoft YaHei"; list-style:square inside; border:0px; outline:0px; font-size:14px; vertical-align:baseline; background:transparent; color:rgb(153,153,153)'>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+J 合并选中的多行代码为一行。举个栗子：将多行格式的CSS属性合并为一行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+D 复制光标所在整行，插入到下一行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Tab 向右缩进。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Shift+Tab 向左缩进。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Shift+Tab 向左缩进。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+K+K 从光标处开始删除代码至行尾。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+K 删除整行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+/ 注释单行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+/ 注释多行。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+K+U 转换大写。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+K+L 转换小写。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Z 撤销。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Y 恢复撤销。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+U 软撤销，感觉和 Gtrl+Z 一样。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+F2 设置书签
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+T 左右字母互换。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      F6 单词检测拼写
     </li>
    </ul>
    <h4 id="搜索类" style='font-family:"Microsoft YaHei"; font-weight:100; line-height:1.1; color:rgb(153,153,153); margin:0.8em 0px; font-size:14px; padding:0px; border:0px; outline:0px; vertical-align:baseline; background:transparent'>
     搜索类
    </h4>
    <ul style='margin:0px 0px 22px; padding:0px; font-family:"Microsoft YaHei"; list-style:square inside; border:0px; outline:0px; font-size:14px; vertical-align:baseline; background:transparent; color:rgb(153,153,153)'>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+F 打开底部搜索框，查找关键字。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+shift+F 在文件夹内查找，与普通编辑器不同的地方是sublime允许添加多个文件夹进行查找，略高端，未研究。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+P 打开搜索框。举个栗子：1、输入当前项目中的文件名，快速搜索文件，2、输入@和关键字，查找文件中函数名，3、输入：和数字，跳转到文件中该行代码，4、输入#和关键字，查找变量名。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+G 打开搜索框，自动带：，输入数字跳转到该行代码。举个栗子：在页面代码比较长的文件中快速定位。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+R 打开搜索框，自动带@，输入关键字，查找文件中的函数名。举个栗子：在函数较多的页面快速查找某个函数。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+： 打开搜索框，自动带#，输入关键字，查找文件中的变量名、属性名等。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Shift+P 打开命令框。场景栗子：打开命名框，输入关键字，调用sublime text或插件的功能，例如使用package安装插件。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Esc 退出光标多行选择，退出搜索框，命令框等。
     </li>
    </ul>
    <h4 id="显示类" style='font-family:"Microsoft YaHei"; font-weight:100; line-height:1.1; color:rgb(153,153,153); margin:0.8em 0px; font-size:14px; padding:0px; border:0px; outline:0px; vertical-align:baseline; background:transparent'>
     显示类
    </h4>
    <ul style='margin:0px 0px 22px; padding:0px; font-family:"Microsoft YaHei"; list-style:square inside; border:0px; outline:0px; font-size:14px; vertical-align:baseline; background:transparent; color:rgb(153,153,153)'>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+Tab 按文件浏览过的顺序，切换当前窗口的标签页。
     </li>
     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+PageDown 向左切换当前窗口的标签页。

     <li style="margin:0px; padding:0px; list-style:none; border:0px; outline:0px; vertical-align:baseline; background:transparent">
      Ctrl+PageUp 向右切换当前窗口的标签页。
