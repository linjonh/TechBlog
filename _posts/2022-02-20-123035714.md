    <h2>
     <a id="_1">
     </a>
     实验背景
    </h2>
    <p>
     不少java编译的工具都只能兼容低版本java，而burpsuite2021等最新的工具只能在相对高版本环境下运行。故通过安装两个不同版本java实现。
    </p>
    <h2>
     <a id="_4">
     </a>
     准备
    </h2>
    <p>
     未安装java环境的windows10操作系统，jdk-8u201和jdk-11.0.12，冰蝎，burpsuite2021。
    </p>
    <h2>
     <a id="_7">
     </a>
     开灶
    </h2>
    <h3>
     <a id="step1_jdk_8">
     </a>
     step1_安装jdk
    </h3>
    <p>
     安装两个版本java，傻瓜式安装，且网上有很多教程，这里不做赘述。我安装的是在E盘java文件夹下，并创建了两个子文件夹：java8，java11。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f4bb9dadf74468cbd20148ee8e9734a3.png#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/74224460f1244fc31cb59c2b34fa12bc.png#pic_center">
       <br/>
       java11安装后可能没有jre，导致环境搭建出错。
       <br/>
       先到java11安装目录打开命令提示符
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c69a27fe1785755f95d8f00aa48aa697.png#pic_center">
        <br/>
        输入以下命令
        <br/>
        <code>
         bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre
        </code>
        <br/>
        检验
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f803b8b0ac84fea21e8960c351510410.png#pic_center">
         <br/>
         安装成功
        </img>
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="step2__19">
     </a>
     step2_配置环境变量
    </h3>
    <p>
     打开高级系统设置。以后经常会有要求配置系统环境变量，这里稍微详细一点。
     <br/>
     方案一，通过搜索框快速搜索
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a562379474972622edd1e99d28b31cd2.png#pic_center">
      <br/>
      方法二，打开控制面板搜索
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/455b1e4686fc209d1978b20fb287cb5f.png#pic_center"/>
     </img>
    </p>
    <p>
     打开后选择“环境变量”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d6bf4e74fcf4a29cbe7847c8d27a30e7.png#pic_center"/>
    </p>
    <p>
     <strong>
      这里修改的都是系统变量，不要修改到用户变量去了。
     </strong>
     <br/>
     步骤网上已经很详细了，这里略讲。
    </p>
    <p>
     <strong>
      创建变量名
      <code>
       CLASSPATH
      </code>
     </strong>
     <br/>
     变量值为`.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c1bd16c3158358b3850e393fed2a3c02.png#pic_center"/>
    </p>
    <p>
     <strong>
      创建JAVAHOME
     </strong>
     <br/>
     <code>
      JAVA_HOME8
     </code>
     ，值为java8安装的绝对路径；
     <br/>
     <code>
      JAVA_HOME11
     </code>
     ，值为java11安装的绝对路径；
     <br/>
     <code>
      JAVA_HOME
     </code>
     ,变量值为
     <code>
      %JAVA_HOME8%
     </code>
     或
     <code>
      JAVA_HOME11
     </code>
     ，其中8和11代表使用的java版本号。
     <br/>
     如下图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1f58236a39fa3d27afcc343d14413e67.png#pic_center"/>
    </p>
    <p>
     <strong>
      创建path
     </strong>
     <br/>
     添加如下两条
     <br/>
     <code>
      %JAVA_HOME\bin%
     </code>
     <br/>
     <code>
      %JAVA_HOME\jre\bin%
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7e6541abee1b8edffacd70772226ce8b.png#pic_center"/>
     <br/>
     安装了java11可能会出现以下变量，需要将其删去，不然只会解析java11。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7916d3fb51e02048ea154e84a1e47974.png#pic_center"/>
    </p>
    <h3>
     <a id="step3__50">
     </a>
     step3_检验
    </h3>
    <p>
     打开终端依次输入
     <code>
      java
     </code>
     ,
     <code>
      javac
     </code>
     检验是否安装成功，如果出现以下命令即代表安装成功
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1e63984fc72e9150171220c232dcdb4d.png#pic_center"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e88033c4667ddb5d78963b012ce51836.png#pic_center"/>
    </p>
    <p>
     接下来查看版本信息，如果需要切换就更改系统变量后面的版本号保存即可
     <br/>
     <code>
      java -version
     </code>
     <br/>
     <strong>
      java8
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0e98770c11f49dc80f6d47365e0dd39f.png#pic_center"/>
     <br/>
     冰蝎
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fd22165de32e2eabdb557ff9d26bd562.png#pic_center"/>
     <br/>
     <strong>
      java11
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a072b15b1e829a512845240dc94bbad0.png#pic_center"/>
     <br/>
     burpsuite2021
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/338b71b13af12aeb81f91af0f3474b98.png#pic_center"/>
     <br/>
     <strong>
      完结
     </strong>
    </p>
