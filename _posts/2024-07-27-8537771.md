         <li>
          <strong>
           专业文档搜索
          </strong>
          <br/>
          很多有价值的资料，在互联网上并非是普通的网页，而是以Word、PowerPoint、PDF等格式存在。百度支持对Office文档（包括Word、Excel、Powerpoint）、Adobe PDF文档、RTF文档进行了全文搜索。要搜索这类文档，很简单，在普通的查询词后面，加一个“filetype：”文档类型限定。“Filetype:”后可以跟以下文件格式：DOC、XLS、PPT、PDF、RTF、ALL。其中，ALL表示搜索所有这些文件类型。例如，查找张五常关于交易费用方面的经济学论文。“
          <a href="http://www.baidu.com/s?wd=%BD%BB%D2%D7%B7%D1%D3%C3%20%D5%C5%CE%E5%B3%A3%20filetype:doc" rel="nofollow noopener noreferrer" target="_blank">
           交易费用 张五常 filetype:doc
          </a>
          ”，点击结果标题，直接下载该文档，也可以点击标题后的“
          <a target="_blank">
           HTML版
          </a>
          ”快速查看该文档的网页格式内容。
          <br/>
          您也可以通过 百度文档搜索 界面（
          <a href="http://file.baidu.com/" rel="nofollow noopener noreferrer" target="_blank">
           http://file.baidu.com/
          </a>
          ） ，直接使用专业文档搜索功能。

         <li>
          <strong>
           股票、列车时刻表和飞机航班查询

           000001

           百度常用搜索
          </a>
          中，进行上述查询。

         <li>
          <strong>
           高级搜索语法
          </strong>
          <br/>
          <ul type="disc">
           <li>
            <br/>
           </li>
           <li>
            把搜索范围限定在网页标题中——intitle
            <br/>
            网页标题通常是对网页内容提纲挈领式的归纳。把查询内容范围限定在网页标题中，有时能获得良好的效果。使用的方式，是把查询内容中，特别关键的部分，用“intitle:”领起来。
            <br/>
            例如，找林青霞的写真，就可以这样查询：
            <a href="http://www.baidu.com/s?wd=%D0%B4%D5%E6%20intitle:%C1%D6%C7%E0%CF%BC" rel="nofollow noopener noreferrer" target="_blank">
             写真 intitle:林青霞
            </a>
            <br/>
            注意，intitle:和后面的关键词之间，不要有空格。
            <br/>
            <br/>
           </li>
           <li>
            把搜索范围限定在特定站点中——site
            <br/>
            有时候，您如果知道某个站点中有自己需要找的东西，就可以把搜索范围限定在这个站点中，提高查询效率。使用的方式，是在查询内容的后面，加上“site:站点域名”。
            <br/>
            例如，天空网下载软件不错，就可以这样查询：
            <a href="http://www.baidu.com/s?wd=msn%20site:skycn.com" rel="nofollow noopener noreferrer" target="_blank">
             msn site:skycn.com
            </a>
            <br/>
            注意，“site:”后面跟的站点域名，不要带“http://”；另外，site:和站点名之间，不要带空格。
            <br/>
            <br/>
           </li>
           <li>
            把搜索范围限定在url链接中——inurl
            <br/>
            网页url中的某些信息，常常有某种有价值的含义。于是，您如果对搜索结果的url做某种限定，就可以获得良好的效果。实现的方式，是用“inurl:”，后跟需要在url中出现的关键词。
            <br/>
            例如，找关于photoshop的使用技巧，可以这样查询：
            <a href="http://www.baidu.com/s?wd=photoshop%20inurl:jiqiao" rel="nofollow noopener noreferrer" target="_blank">
             photoshop inurl:jiqiao
            </a>
            <br/>
            上面这个查询串中的“photoshop”，是可以出现在网页的任何位置，而“jiqiao”则必须出现在网页url中。
            <br/>
            注意，inurl:语法和后面所跟的关键词，不要有空格。
            <br/>
            <span class="p1" style="font-weight:bold">
             <a id="0904" name="0904">
             </a>
            </span>
            <br/>
           </li>
           <li>
            精确匹配——双引号和书名号
            <br/>
            如果输入的查询词很长，百度在经过分析后，给出的搜索结果中的查询词，可能是拆分的。如果您对这种情况不满意，可以尝试让百度不拆分查询词。给查询词加上双引号，就可以达到这种效果。
            <br/>
            例如，搜索
            <a href="http://www.baidu.com/s?wd=%C9%CF%BA%A3%BF%C6%BC%BC%B4%F3%D1%A7" rel="nofollow noopener noreferrer" target="_blank">
             上海科技大学
            </a>
            ，如果不加双引号，搜索结果被拆分，效果不是很好，但加上双引号后，
            <a href="http://www.baidu.com/s?wd=%A1%B0%C9%CF%BA%A3%BF%C6%BC%BC%B4%F3%D1%A7%A1%B1" rel="nofollow noopener noreferrer" target="_blank">
             “上海科技大学”
            </a>
            ，获得的结果就全是符合要求的了。
            <br/>
            书名号是百度独有的一个特殊查询语法。在其他搜索引擎中，书名号会被忽略，而在百度，中文书名号是可被查询的。加上书名号的查询词，有两层特殊功能，一是书名号会出现在搜索结果中；二是被书名号扩起来的内容，不会被拆分。 书名号在某些情况下特别有效果，例如，查名字很通俗和常用的那些电影或者小说。比如，查电影“
            <a href="http://www.baidu.com/s?wd=%CA%D6%BB%FA" rel="nofollow noopener noreferrer" target="_blank">
             手机
            </a>
            ”，如果不加书名号，很多情况下出来的是通讯工具——手机，而加上书名号后，
            <a href="http://www.baidu.com/s?wd=%A1%B6%CA%D6%BB%FA%A1%B7" rel="nofollow noopener noreferrer" target="_blank">
             《手机》
            </a>
            结果就都是关于电影方面的了。
            <br/>
            <br/>
           </li>
           <li>
            要求搜索结果中不含特定查询词
            <br/>
            如果您发现搜索结果中，有某一类网页是您不希望看见的，而且，这些网页都包含特定的关键词，那么用减号语法，就可以去除所有这些含有特定关键词的网页。
            <br/>
            例如，搜
            <a href="http://www.baidu.com/s?wd=%C9%F1%B5%F1%CF%C0%C2%C2" rel="nofollow noopener noreferrer" target="_blank">
             神雕侠侣
            </a>
            ，希望是关于武侠小说方面的内容，却发现很多关于电视剧方面的网页。那么就可以这样查询：
            <a href="http://www.baidu.com/s?wd=%C9%F1%B5%F1%CF%C0%C2%C2%20-%B5%E7%CA%D3%BE%E7" rel="nofollow noopener noreferrer" target="_blank">
             神雕侠侣 -电视剧
            </a>
            <br/>
            注意，前一个关键词，和减号之间必须有空格，否则，减号会被当成连字符处理，而失去减号语法功能。减号和后一个关键词之间，有无空格均可。
           </li>
          </ul>

         </li>

          ，可以方便的做各种搜索查询。
          <br/>
          您还可以根据自己的习惯，在搜索框右侧的
          <a href="http://www.baidu.com/gaoji/preferences.html" rel="nofollow noopener noreferrer" target="_blank">
