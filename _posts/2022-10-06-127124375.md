    <h3>
     基本概述
    </h3>
    <h4>
     硬件组成
    </h4>
    <p>
     电机模块L9110S、循迹模块、红外避障模块、超声波模块、测速模块、OLED屏、蓝牙模块、4G，模块、语音模块SU-03T
    </p>
    <h4>
     功能
    </h4>
    <p>
     蓝牙控制小车、WiFi控制小车、4G控制小车、小车的避障、跟随、循迹
    </p>
    <h4>
     关键字
    </h4>
    <p>
     单片机中一般都有两块存储区域，ROM和RAM，程序代码存储在ROM中，程序要用的变量存储在RAM中。而“code”的作用就是将其修饰过的变量存储在ROM中而非RAM。 在单片机中，RAM空间都比较小，是比较宝贵的，当存放在RAM中的数据过多时，会导致编译不成功。
    </p>
    <ul>
     <li>
      exturn：使用exturn关键字修饰的全局变量或函数，作用域不再局限本文件，其他文件同样能访问到这些变量或函数，跟static关键字恰恰相反。
     </li>
     <li>
      code：使用code关键字修饰的变量（一般是初始化后，值保持不变的变量）后会被存放到ROM区，从而节省RAM的空间。
     </li>
    </ul>
    <h3>
     模块介绍
    </h3>
    <h4>
     电机模块L9110S
    </h4>
    <p>
     <img alt="aa5cb1e961d8406699bf75655fa90f4b.png" src="https://i-blog.csdnimg.cn/blog_migrate/481b3a75ac297873c886197096b18007.png"/>
    </p>
    <p>
     L0110S模块的A、B分别控制着两个电机，如果需要控制四个电机，则需要两个L0110S模块
    </p>
    <ul>
     <li>
      当B-1A为高电平，B-2A为低电平时，电机反转或正转
     </li>
     <li>
      当B-1A为低电平，B-2A为高电平时，电机正转反转
     </li>
     <li>
      当B-1A为低电平，B-2A为低电平时，电机不转
     </li>
     <li>
      电机的正转和反转与跟电机的接线不同而不同，注意自己调试
     </li>
    </ul>
    <h4>
     循迹模块（TCRT5000传感器）
    </h4>
    <p>
     <img alt="de5641f570fd4adba541c272a8063677.png" src="https://i-blog.csdnimg.cn/blog_migrate/5032323a2b07f61d379c663e477b90bf.png"/>
    </p>
    <ul>
     <li>
      当发射出的红外线没有被反射回来或被反射回来但强度不够大时，DO输出高电平，灯灭。 黑色吸收红外线，DO输出高电平，灯亮
     </li>
     <li>
      当发射出的红外线被反射回来或被反射回来且强度足够大，DO输出低电平，灯亮。 白色反射红外线，DO输出低电平，灯亮
     </li>
     <li>
      即黑色输出高电平，灯灭，白色输出低电平，灯亮
     </li>
    </ul>
    <h4>
     红外避障模块
    </h4>
    <p>
     <img alt="6265a10e74644db7b3c133fa870458c8.png" src="https://i-blog.csdnimg.cn/blog_migrate/0f7e72076b49cc859e824b221cc275aa.png"/>
    </p>
    <ul>
     <li>
      当发射出的红外线没有被反射回来或被反射回来但强度不够大时，DO输出高电平，灯灭。没有障碍物
     </li>
     <li>
      当发射出的红外线被反射回来，DO输出低电平，灯亮。有障碍物
     </li>
     <li>
      即有障碍物输出低电平，灯亮，没有障碍物输出高电平，灯灭
     </li>
    </ul>
    <h4>
     测速模块
    </h4>
    <p>
     <img alt="0877a6d59b6b4a61a22d38b3c0c7b352.png" src="https://i-blog.csdnimg.cn/blog_migrate/ae590218c149835af7935797a3ee8033.png"/>
    </p>
    <ul>
     <li>
      发射的红外线被物体遮挡时，输出高电平，发射的红外线没被物体遮挡时，输出低电平
     </li>
     <li>
      有物体高电平，没物体低电平
     </li>
     <li>
      当搭配小车测速盘，会形成下降沿（有遮挡高电平，没遮挡低电平）
     </li>
    </ul>

    <h4>
     调速小车（利用PWM波对电机进行调速）
    </h4>
    <ul>
     <li>
      利用定时器0软件模拟PWM波控制小车左轮速度，定时器1软件模拟PWM波控制小车右轮速度，通过控制轮子的速度来达到前进、停止、左转、右转
     </li>
     <li>
      在20ms的过程中，部分时间让电机正转，剩下时间让电机停止不动就能改变电机获得的功率，从而改变电机速度。

    </ul>

	Delay5us();  //延时5微秒后，为读取sda数据做准备

//IIC发送一个字节
void IIC_sendByte(char myData)
{
	int i;
	for ( i = 0; i &lt; 8; i++){
		//发生数据翻转，选择即将发送的是0还是1
		scl = 0;
		sda = myData &amp; 0x80;  //获取需要发送字节的最高位到SDA
		Delay5us();  //数据建立时间
		
		//开始发送数据
		scl = 1;
		Delay5us();  //数据发送时间
		
		scl = 0;  //发送完毕拉低，等待下1bit数据的传输
		Delay5us();
		
		myData = myData &lt;&lt; 1;
	}	
}

void olceClean()
{
	int i,j;
	for(i=0;i&lt;8;i++){
		oledWriteCmd(0xB0 + i);  //选择PAGE
		//选择PAGE的第0列开始显示
		oledWriteCmd(0x00);  
		oledWriteCmd(0x10);
		for(j = 0;j &lt; 128; j++){

		}
	}
}

//OLED显示一个字符
void oledShowByte(char rows,char columns,char oledByte)
{ 
	unsigned int i;
 
	//显示字符的上半部分
	oledWriteCmd(0xb0+(rows*2-2));  //选择行
	//选择列                        
	oledWriteCmd(0x00+(columns&amp;0x0f));                          
	oledWriteCmd(0x10+(columns&gt;&gt;4));      
	//显示数据                       
	for(i=((oledByte-32)*16);i&lt;((oledByte-32)*16+8);i++){
		oledWriteData(oledFont[i]);                            
	}
 
 	//显示字符的上半部分
	oledWriteCmd(0xb0+(rows*2-1));  //选择行  
	//选择列                    
	oledWriteCmd(0x00+(columns&amp;0x0f));                          
	oledWriteCmd(0x10+(columns&gt;&gt;4)); 
	//显示数据                           
	for(i=((oledByte-32)*16+8);i&lt;((oledByte-32)*16+8+8);i++){
		oledWriteData(oledFont[i]);                            
	}       
}
 
//OLED显示一个字符串
void oledShowString(char rows,char columns,char *str)
{
	while(*str != '\0'){
		oledShowByte(rows,columns,*str);
		str++;
		columns += 8;   
	}       
}

void main()
{
	uartInit();
	Time0Init();
	int0Init();

	oledInit();
	olceClean();  //清屏函数
	oledWriteCmd(0x20);  //设置内存
	oledWriteCmd(0x02);  //选择页寻址模式

	while(1){
		if(signal == 1){
			sprintf(speedMsg,"speed:%dcm/s ",speed);
			olceClean();
			oledShowString(2,5,speedMsg);
			signal = 0;
		}
	}
}

//定时器0产生的中断的处理函数
void Time0Handle() interrupt 1
{
	cnt++;
	TL0 = 0x66;		
	TH0 = 0xFC;
	//每过一秒统计一次产生了多少次下降沿，让main函数向串口发送当前小车速度
	if(cnt == 1000){
		cnt = 0;
		signal = 1;  
		speed = speedCnt;
		speedCnt = 0;
	}
}

//外部中断0产生的中断的处理函数
void int0Handle() interrupt 0
{
	speedCnt++;  //每产生一个下降沿让标志位+1
}

//IIC起始信号

}

void IIC_stop()
{
	scl = 0;
	sda = 0;
	scl = 1;
	Delay5us();
	sda = 1;
	Delay5us();
	sda = 0;
}
 
//IIC的ACK应答信号
char IIC_ack()
{
	char flag;
 
	scl = 0;
	sda = 1;  //在时钟脉冲9期间释放数据线
	Delay5us();  //延时5微秒后，为读取sda数据做准备
 
	scl = 1;  
	Delay5us();  
	flag = sda;  //读取数据线,0为应答
	Delay5us();  
	
	scl = 0;
	Delay5us();
	return flag;
}
 
//IIC发送一个字节
void IIC_sendByte(char myData)
{
	int i;
	for ( i = 0; i &lt; 8; i++){
		//发生数据翻转，选择即将发送的是0还是1
		scl = 0;
		sda = myData &amp; 0x80;  //获取需要发送字节的最高位到SDA
		Delay5us();  //数据建立时间
		
		//开始发送数据
		scl = 1;
		Delay5us();  //数据发送时间
		
		scl = 0;  //发送完毕拉低，等待下1bit数据的传输
		Delay5us();
		
		myData = myData &lt;&lt; 1;
	}	
}
