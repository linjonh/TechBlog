    <h3 style="padding:0px;">
     <span style="font-size:16px;">
      JavaScript是网页编程语言，而jQuery是一个基于js编写
      <br/>
      jquery就是基于JavaScript语言写出来的一个框架,实质上还是js而已
      <br/>
      So:
      <br/>
      1.jQuery 是一个 JavaScript 库。
      <br/>
      2.jQuery 极大地简化了 JavaScript 编程。（代码比沉重的JavaScript轻灵多了）
      <br/>
      3.jQuery 使JavaScript更好用。
      <br/>
      4.jquery就是要用更少的代码，漂亮的完成更多的功能。
     </span>
    </h3>
    <br/>
    <h4 style="padding:0px;">
     <span style="color:rgb(51,153,153);">
      一些方法对比:
      <br/>
      ⑴使用JavaScript和jQuery分别加载DOM
      <br/>
      JavaScript只会执行一次，而jQuery会执行多次
      <br/>
      <br/>
      ⑵使用JavaScript和jQuery获取ID
      <br/>
      JavaScript：
      <br/>
      document.getElementById('idName')
      <br/>
      JQuery：
      <br/>
      $('#idName')
      <br/>
      <br/>
      ⑶使用JavaScript和jQuery获取class
      <br/>
      JavaScript没有默认的获取class的方法
      <br/>
      JQuery：
      <br/>
      $('.className')
     </span>
    </h4>
    <br/>
    <h2 style="padding:0px;">
     <a style='background-image:url("img/anchor.gif");'>
     </a>
     <span style="color:rgb(255,0,0);">
      使用jQuery最大的好处是少量的代码做更多的事情，与JavaScript相比，jQuery的语法更加简单。
     </span>
    </h2>
    <br/>
    <h4 style="padding:0px;">
     <span style="color:rgb(204,51,204);">
      举个简单的栗子：获取元素的值
      <br/>
      Javascript代码
      <br/>
      document.getElementById(‘elementid').value
      <br/>
      <br/>
      使用jQuery：
      <br/>
      Jquery代码
      <br/>
      $(‘#elementid').val();
     </span>
