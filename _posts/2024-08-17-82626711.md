    <h4 id="导语">
     导语
    </h4>
    <p>
     最近在学习机器学习，发现需要恶补数学知识。总感觉国内大学的教材只适合考试，对数学知识的理解和运用不到位。现整理一些评价很高的书，总的来说较适合工科类的同学打基础和提高。
    </p>
    <h2 id="高等数学">
     <a name="t1">
     </a>
     高等数学
    </h2>
    <h4 id="入门">
     <a name="t2">
     </a>
     入门
    </h4>
    <p>
     <a href="https://book.douban.com/subject/1231399/" rel="nofollow">
      《托马斯微积分》
     </a>
     ：直观易读，强调建模应用和技巧训练，重要的是不失数学上的完整性，工科使用应该相当不错。作者：芬尼 / 韦尔 / 焦尔当诺
     <br/>
     <a href="https://book.douban.com/subject/4926707/" rel="nofollow">
      《普林斯顿微积分读本》
     </a>
     ：没看过，据说评价很高。 作者：阿德里安·班纳
    </p>
    <h4 id="进阶">
     <a name="t3">
     </a>
     进阶
    </h4>
    <p>
     <a href="https://book.douban.com/subject/1281343/?i=0" rel="nofollow">
      《微积分和数学分析引论》
     </a>
     ：数学经典之一。作者: Richard Courant / Fritz John
     <br/>
     <a href="https://book.douban.com/subject/1230288/" rel="nofollow">
      《数学分析原理》
     </a>
     ：分析学名著。作者：Walter Rudin
     <br/>
     <a href="https://book.douban.com/subject/4825571/" rel="nofollow">
      《数学分析八讲》
     </a>
     ：据说数理学科看着都有点困难。作者：辛钦
     <br/>
     <a href="https://book.douban.com/subject/1707158/" rel="nofollow">
      《微积分学教程》
     </a>
     （共三卷）：数学名著。作者：菲赫金哥尔茨
     <br/>
     <a href="https://book.douban.com/subject/3016613/" rel="nofollow">
      《微积分入门》
     </a>
     （共两卷）：世界级数学大师的优秀教材。作者：小平邦彦
    </p>
    <h2 id="线性代数">
     <a name="t4">
     </a>
     线性代数
    </h2>
    <h4 id="入门-1">
     <a name="t5">
     </a>
     入门
    </h4>
    <p>
     <a href="https://book.douban.com/subject/1425950/" rel="nofollow">
      《线性代数及其应用》
     </a>
     ：侧重于应用。作者：David C. Lay
     <br/>
     MIT 教授 Gilbert Strang 的公开课
     <a href="https://ocw.mit.edu/courses/mathematics/18-06-linear-algebra-spring-2010/index.htm" rel="nofollow">
      Linear Algebra
     </a>
     ，配合教材 《Introduction to Linear Algebra 》或《Linear Algebra and Its Applications》
    </p>
    <h4 id="进阶-1">
     <a name="t6">
     </a>
     进阶
    </h4>
    <p>
     <a href="https://book.douban.com/subject/3715623/" rel="nofollow">
      《线性代数应该这样学》
     </a>
     ：建立在普通线性代数基础上的，有泛函基础再看这本书会好些。作者：Sheldon Axler
     <br/>
     <a href="https://book.douban.com/subject/1281543/" rel="nofollow">
      《矩阵分析》
     </a>
     ：赫赫有名的一本书。作者：（美）合恩（Horn/R.A.）等/杨奇
     <br/>
     <a href="https://book.douban.com/subject/3745369/" rel="nofollow">
      《矩阵计算》
     </a>
     ：被称为数值线性代数圣经。作者：Gene H.Golub / Charles F.Van Loan
    </p>
    <h2 id="概率论与数理统计">
     <a name="t7">
     </a>
     概率论与数理统计：
    </h2>
    <p>
     <a href="https://book.douban.com/subject/1803862/" rel="nofollow">
      《概率论基础教程》
     </a>
     ：入门级的好教材。作者：罗斯（Sheldon M. Ross）
     <br/>
     <a href="https://book.douban.com/subject/4818683/" rel="nofollow">
      《概率论教程》
     </a>
     ：看版本号就知道很经典。作者：钟开莱
     <br/>
     <a href="https://book.douban.com/subject/2201479/" rel="nofollow">
      《概率论与数理统计》
     </a>
     ：用心写成的书，不光是教知识，更是教思维的方法。作者：陈希孺
     <br/>
     <a href="https://book.douban.com/subject/1577547/" rel="nofollow">
      《数理统计学导论（原书第7版） 》
     </a>
     ：不太了解。作者：霍格
     <br/>
     <a href="https://book.douban.com/subject/3996242/" rel="nofollow">
      《数理统计学教程 》
     </a>
     ：堪称最好的中文数理统计教材，中国科学院院士的又一杰作。作者：陈希孺
    </p>
