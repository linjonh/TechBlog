    <h5>
     <a id="2__40">

     2. 编写并测试应用代码
    </h5>
    <blockquote>
     <p>
      在Flutter中，一切都是控件（widget）。Flutter应用中的图像、图标和文本都是widget。布局元素
      <br/>
      例如行、列、栅格等用来安排其他widget的位置、大小和对齐，而这些布局元素本身也是widget。
     </p>
    </blockquote>
    <p>
     参考如下代码修改你的main.dart文件：
    </p>
    <pre><code>import 'package:flutter/material.dart';

void main() =&gt; runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Welcome to Flutter'),
        ),
        body: Center(
          child: Text('Hello World'),
        ),
      ),
    );
  }
}
</code></pre>
    <p>
     现在启动Android模拟器：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ff16b2c0946bfd259652a0e6d17e2151.png"/>
    </p>
    <p>
     注意widget的焦点，我们接下来将修改MaterialApp的内容。先看一下原始版本：
    </p>
    <pre><code>body: Center(
  child: Text('Hello World'),
)
</code></pre>
    <p>
     Flutter中的布局元素（也是widget）可以根据其是否支持包含多个widget，而简单地归类为两种类型。
     <br/>
     例如，Container、Padding只能包含一个子widget，而Row、Column则可以包含多个。
    </p>
    <p>
     现在我们在Row布局中引入三个文本widget：
    </p>
    <pre><code>body: Center(
  child: Row(
    mainAxisAlignment: MainAxisAlignment.center,
    children: &lt;Widget&gt;[
      Text('0'),
      Text('0'),
      Text('0'),
    ],
  ),
),
</code></pre>
    <p>
     现在看起来是这样：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dc4a16970b0937ab2bb5afb0155c57c2.png"/>
    </p>
    <p>
     在我们开始设置组件的样式之前，建议先创建一个新的widget来处理样式问题，以便遵循DRY（Don’t Repeat Yoursel）原则。
    </p>
    <p>
     将三个子widget用一个自定义widget（后面解释）替换。现在main.dart文件变成：
    </p>
    <pre><code>import 'package:flutter/material.dart';

void main() =&gt; runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Timer'),
        ),
        body: Center(
          child: Row(
            mainAxisAlignment: MainAxisAlignment.center,
            children: &lt;Widget&gt;[
              CustomTextContainer(),
              CustomTextContainer(),
              CustomTextContainer(),
            ],
          ),
        ),
      ),
    );
  }
}

class CustomTextContainer extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Text('00');
  }
}
</code></pre>
    <p>
     我不是很擅长用户界面，但是我们还是尽力吧，加点背景色，留点空白：

}

    <p>
     现在到了app最不可或缺的部分了：状态。状态将持有timer的当前值和运行标识。我
     <br/>
     补充了一个TimerState类，它负责维护状态，同时构建widget树。
    </p>
    <pre><code>class Timer extends StatefulWidget {
  @override
  State&lt;StatefulWidget&gt; createState() {
    return new TimerState();
  }
}

class TimerState extends State&lt;Timer&gt; {
  int secondsPassed = 0;
  bool isActive = false;
  
  @override
  Widget build(BuildContext context) {
    // Return Widget Tree
  }
}
</code></pre>
    <p>
     按钮用来切换timer的运行或者停止：
    </p>
    <pre><code>RaisedButton(
  child: Text(isActive ? 'STOP' : 'START'),
  onPressed: () {
    setState(() {
      isActive = !isActive;
    });
  },
)
</code></pre>
    <p>
     Dart有个用于异步操作的优雅的模块。我们可以使用其Timer类来实现读秒。修改后的
     <br/>
     TimerAppState代码如下，注意我们已经重构Timer类并改名为TimerApp以避免与异步
     <br/>
     模块中的Timer类混淆：
    </p>
    <pre><code>class TimerAppState extends State&lt;TimerApp&gt; {
  static const duration = const Duration(seconds:1);

  int secondsPassed = 0;
  bool isActive = false;

  Timer timer;

  void handleTick() {
    if (isActive) {
      setState(() {
        secondsPassed = secondsPassed + 1;
      });
    }
  }

  @override
  Widget build(BuildContext context) {
    if (timer == null)
      timer = Timer.periodic(duration, (Timer t) {
        handleTick();
      });

    int seconds = secondsPassed % 60;
    int minutes = secondsPassed ~/ 60;
    int hours = secondsPassed ~/ (60*60);

    return MaterialApp(...)
  }
}
</code></pre>
