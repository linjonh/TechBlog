      <p>
       在 Ubuntu 22.04 上安装 Python 可以使用以下步骤：
      </p>
      <ol>
       <li>
        <p>
         打开终端：按快捷键
         <code>
          Ctrl + Alt + T
         </code>
         打开终端。
        </p>
       </li>
       <li>
        <p>
         更新包管理器：在终端中输入
         <code>
          sudo apt update
         </code>
         并回车。
        </p>
       </li>
       <li>
        <p>
         安装 Python：在终端中输入
         <code>
          sudo apt install python3
         </code>
         并回车。
        </p>
       </li>
       <li>
        <p>
         验证安装：在终端中输入
         <code>
          python3
         </code>
         并回车，进入 Python 交互式环境，可以看到 Python 版本信息。
        </p>
       </li>
      </ol>
      <p>
       这样就完成了在 Ubuntu 22.04 上安装 Python 的过程。
      </p>
