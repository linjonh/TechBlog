     作者：Vijay Singh Khatri

     译者：geekpi

    <p>
     <strong>
      步骤 1
     </strong>
     ：首先，安装构建 Python 所需的开发包
     <br/>
     在 Debian 上
    </p>
    <pre><code class="prism language-cmd">$ sudo apt update　　
$ sudo apt install build-essential zlib1g-dev \　　
libncurses5-dev libgdbm-dev libnss3-dev \　　
libssl-dev libreadline-dev libffi-dev curl　　
</code></pre>
    <p>
     在 Fedora 上：
    </p>
    <pre><code class="prism language-sql">$ sudo dnf groupinstall development　　
</code></pre>
    <p>
     <strong>
      步骤 2：
     </strong>
     下载最新的稳定版本的 Python 3
     <br/>
     访问官方 Python 网站并下载最新版本的 Python 3。下载完成后，你会有一个 .tar.xz 归档文件（“tarball”），其中包含 Python 的源代码。
    </p>
    <p>
     <strong>
      步骤 3：
     </strong>
     解压 tarball
     <br/>
     下载完成后，使用解压程序或 Linux 的 tar 命令解压压缩包，
    </p>
    <pre><code class="prism language-powershell">$ tar <span class="token operator">-</span>xf Python<span class="token operator">-</span>3<span class="token punctuation">.</span>?<span class="token punctuation">.</span>?<span class="token punctuation">.</span>tar<span class="token punctuation">.</span>xz　　
</code></pre>
    <p>
     <strong>
      步骤 4：
     </strong>
     配置脚本
     <br/>
     解压 Python 压缩包后，进入 configure 脚本所在目录并在 Linux 终端中使用以下命令执行该脚本：
    </p>
    <pre><code class="prism language-powershell">$ cd Python<span class="token operator">-</span>3<span class="token punctuation">.</span><span class="token operator">*</span>　
<span class="token punctuation">.</span><span class="token operator">/</span>configure　　　
</code></pre>
    <p>
     <strong>
      步骤 5：
     </strong>
     开始构建过程
     <br/>
     如果你的系统上已经安装了某个版本的 Python，并且希望同时安装新版本的 Python，请使用以下命令：
    </p>
    <pre><code class="prism language-powershell">$ sudo make altinstall　
</code></pre>
    <p>
    </p>
    <p>
     如果要使用此版本替换当前版本的 Python，那么应使用包管理器（例如 apt 或 dnf）卸载当前的 Python 包，然后安装：
    </p>
    <pre><code class="prism language-powershell">$ sudo make install　
</code></pre>
    <p>
     <strong>
      步骤 6：
     </strong>
     验证安装
     <br/>
     如果你没有遇到任何错误，那么现在你的 Linux 系统上已安装了最新的 Python。要进行验证，请在终端中输入以下命令之一：
    </p>
    <pre><code class="prism language-powershell">python3 <span class="token operator">--</span>version　　
python <span class="token operator">--</span>version　　
</code></pre>
    <p>
     如果输出显示 Python 3.x，那么说明 Python 3 已成功安装。
    </p>
