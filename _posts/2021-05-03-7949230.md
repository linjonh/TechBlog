      人说,:IBM的今天, 是微软的明天; 微软的今天, 是古狗的明天; 古狗的今天, 是FACEBOOK的明天.”文章中真正耐人寻味的内容，其实是一千年前就有人写过的，原文如下：
     </p>
     <p>
      呜呼！灭六国者六国也，非秦也。族秦者秦也，非天下也。嗟乎！使六国各爱其人，则足以拒秦；使秦复爱六国之人，则递三世可至万世而为君，谁得而族灭也？秦人不暇自哀，而后人哀之；后人哀之而不鉴之，亦使后人而复哀后人也。
     </p>
     <p>
      虽然文章写了一千年，而今时间已经是2012-09-06日，地点已经是中国深圳，无论是公司还是国家中，一切都在重演。
      <strong>
       打败自己的，永远只有自己。
      </strong>
     </p>
     <p>
     </p>
     <p>
      原文如下：
     </p>
     <p>
      （1）
      <br/>
      <br/>
      微软现在内交外困, 大厦将倾, 摇摇欲坠. 但万足之虫, 死而未僵, 再撑个几年还不成问题, 甚至其中昙花一现, 回光反照也有可能. 借句股市里的话, 目前抵抗力最小的方向是向下.
      <br/>
      <br/>
      微软目前如同一盘肥肉, 旧敌(IBM, ORACLE)未除, 新敌来势凶猛, 都贪婪地欲吞微软为快.
      <br/>
      <br/>
      如果打一比方, 微软与GOOGLE之战, 好比国军对共军的内战. 目前大概是国军被迫从全面进攻转向重点进攻的1947年初. 虽然, 还有拿下延安的大捷在后, 微软已经天时地利人和全无, 已经是必败之势.
      <br/>
      <br/>
      微软内部, 各部门争风吃醋, 争权夺利, 互相下黑手, 那是平常事了. 跟当年蒋军人人自保, 看友军被围, 不仅不救, 反而抚掌大笑, 是惊人的类似.
      <br/>
      <br/>
      而微软投敌者甚重. 高级将领投敌更是屡屡发生. GOOGLE的语音识别, 从无到有, 从有到领先, 总共不到两年, 如果没有微软的叛将, 那根本就连做梦也别想.
      <br/>
      <br/>
      （2）
      <br/>
      <br/>
      人说, IBM的今天, 是微软的明天; 微软的今天, 是古狗的明天; 古狗的今天, 是FACEBOOK的明天.
      <br/>
      <br/>
      IBM, 微软, 古狗和脸书, 是四代IT公司的代表. 前三者都经历了从无到有, 从小到大, 都有过辉煌的时刻. 而脸书, 则是当下最火爆的公司.
      <br/>
      <br/>
      现在校园的童谣是这样说的,
      <br/>
      <br/>
      顶尖学校的顶尖学生, 去脸书;
      <br/>
      <br/>
      顶尖学校的二流学生, 去古狗或苹果;
      <br/>
      <br/>
      顶尖学校的三流学生或者是二流学校的顶尖学生, 去YAHOO或AMAZON;
      <br/>
      <br/>
      YAHOO或AMAZON不要的学生, 去微软;
      <br/>
      <br/>
      被以上公司辞退准备退休的人, 去IBM.
      <br/>
      <br/>
      梨叔说过, 21世纪, 最重要的财富是人才. 人才的流动方向, 正代表了公司的走势. 微软挖不到人才, 留不住人才, 已经到了震耳欲聋的地步了.
      <br/>
      <br/>
      从成立到现在, 微软已经经历了快四十年. 最近十年, 股票只跌不涨, 苦日子一眼看不到头. 如同一个早生华发的中年人, 脸上写满了人生的沧桑. 盖瓷几年前的无奈退休. 才五十岁的人, 看上去象一个七十多的老人. 其中的辛苦, 只有他自己知道. 天时已过, 幸运女神不再垂青, 花开花落, 人力有穷.
      <br/>
      <br/>
      如果你问爆沫(现微软总裁), 你最希望看见的是什么?
      <br/>
      <br/>
      他肯定说, 微软象苹果一样, 重新焕发第二春.
      <br/>
      <br/>
      如果你问他, 你最害怕看见的是什么?
      <br/>
      <br/>
      他肯定告诉你, 微软象NOVELL一样, 淡出市场, 成为其他公司的PET. 当然, 爆沫宁愿看见微软破产关门, 也不会让微软如果太阳(SUN)公司那样, 被下贱地吞并.
      <br/>
      <br/>
      我入微软数年, 正好躲开了它最辉煌的时刻. 看见了它从希望到失望到绝望到无所谓的过程, 见证了微软帝国的衰落. 真是有幸之极.
      <br/>
      <br/>
      虽然很多人都预料到了微软的下坡路, 但对其原因却众说纷纭. 有些人归咎于盖瓷的大方向错误, 有些人认为爆沫弃技术而走市场. 我不敢苟同这些观点. 我觉得, 主要有以下几个原因:
      <br/>
      <br/>
      1. 奖罚可笑, 人心涣散.
      <br/>
      <br/>
      2. 用人失当, 上行下效.
      <br/>
      <br/>
      3. 任人唯亲, 勾心斗角.
      <br/>
      <br/>
      4. 大方向 vs. 小方向.
      <br/>
      <br/>
      5. 山头林立, 合作困难.
      <br/>
      <br/>
      6. 领导力差, 无法贯彻.
      <br/>
      <br/>
      7. 政策多变, 大失人心.
      <br/>
      <br/>
      <br/>
      （3）
      <br/>
      <br/>
      在具体地谈微软的问题之前, 先讲讲微软的优点.
      <br/>
      <br/>
      我工作这么多年, 去过的公司用手都数不过来. 微软算是我比较喜欢的一个公司. 它的确有很多跟其它公司不一样的地方. 根据我的观察, 微软有以下的优点:
      <br/>
      <br/>
      1. 有很多员工热爱公司, 甚至到了盲目的程度.
      <br/>
      <br/>
      很多人从参加工作开始, 微软是他们的第一个也是唯一的公司. 从这些人的眼睛里, 微软的问题是小的, 优点是大大的. 跟我这样的外来户的看法完全不同. 而如果你问他们微软的优点是什么, 他们的回答往往让人吃惊. 因为他们是占在公司的角度上来回答的, 而不是占在雇员的角度上看问题的.
      <br/>
      <br/>
      2. 拿雇员当人, 民主气氛很高.
      <br/>
      <br/>
      说起来, 公司是个独裁的地方. 经理都是任命的, 而不是民选的. 只有董事们是股民选的, 那也跟雇员无关 (大股东们一般是基金). 实话说, 我到过的公司里, 无论大小, 微软是最民主的. 最重要的证明有四条:
      <br/>
      <br/>
      A. 微软有个所谓的微软调查.
      <br/>
      <br/>
      每年一次. 让雇员匿名自愿参加. 一般发生在三月份, 跟年中REVIEW同时发生. 该微软调查对官员们很重要, 因为这跟他们的REVIEW发生直接联系, 而且对他们以后的当官方向有重大影响. 当然, 这是对中低级官员来说的. 对于高级官员, 规则是可以被扭曲的.
      <br/>
      <br/>
      由于公司再三强调这是匿名的, 所以相对来说, 雇员比较安全, 回答也比较大胆. 为了防止官员猜出是哪个手下说了坏话, 还采取了一些措施, 例如, 少于一定数目的手下的官员, 是看不到他们的数据的.
      <br/>
      <br/>
      B. 微软有个MANAGER FEEDBACK.
      <br/>
      <br/>
      原来是每年两次. 由于众官员的强烈呼声, 现在改成一次了. 发生在八九月份, 跟年底REVIEW同时发生. 这个FEEDBACK 也对官员的仕图有一定的影响. 而且不仅是匿名, 官员不能直接看到Feed back, 除非是跟顶头上司勾结一气.
      <br/>
      <br/>
      C. 微软的民工造反, 成功率相对较高.
      <br/>
      <br/>
      我在其他公司的时候, 下级无论怎么不满, 只要上级没有不满, 最终的结果也是该官员的地位稳固, 下级继续忍耐. 在微软, 如果该官员的直接下级能联合起来, 通过各种途径表示不满, 那该官员的地位就有点不稳了. 在中低级官员中, 如果直接下级纷纷离去, 那么该官员就有可能失去领导的地位. 当然, 造反的成功率只是相对较高, 绝对数还是不大, 但已经让我这样的外来户吃惊了.
      <br/>
      <br/>
      D. 公司不反对, 甚至鼓励员工换组.
      <br/>
      <br/>
      这也是很令人吃惊的地方. 微软的基本想法是这样的, 如果你不喜欢你目前的工作, 与其让你跳到其他公司, 不如去其他的组, 这样公司还能留住你. 这是留住人才的一个巨大措施. 这也很大程度上限制了当官的, 尤其是中低级官僚的权力.
      <br/>
      <br/>
      微软不仅是相对民主一些, 而且对信息的流动, 比其他公司要透明很多, 到了令人惊诧的程度. 一般的公司, 对信息, 采取的都是"NEED KNOW" 策略. 如果你不需要了解, 你肯定不会了解. 独裁社会, 信息, 在很大程度上, 是权力的化身.
      <br/>
      <br/>
      微软, 首先对整个公司的ORG结构, 是很透明的. 即使是个新进公司的毕业生, 也可以查到某人的老板是谁. 某老板手下有多少空位等等. 这在很多公司是不可想象的.
      <br/>
      <br/>
      其次, 你的老板在来年要做什么, 优先级是什么, 也是很明确的, 黑字白底, 网上可查的. 别的公司很多拿你当牛, 需要你去哪里, 事先你不知道, 鞭子响处你才了解.
      <br/>
      <br/>
      最令人吃惊的是, 微软对各人的具体级别能从TITLE上猜出个范围. 而对每个中低级别的奖金比例, 股票比例, 都堂而皇之, 写在人力资源部的网页上. 各中低级别的工资范围, 几乎完全是透明的. 这在其他公司, 完全不可想象.
      <br/>
      <br/>
      微软还有一点, 其他公司比不了的, 就是对办公室的分配. 以前, 每个正式职工, 都一人一个独门的办公室. 门一关, 就是你自己的天下, 别人的眼睛, 包括老板的眼睛都进不来. 现在有些部门已经开始享受CUBICLE的待遇了, 但还有些部分, 继续有着独门OFFICE的特权. 而OFFICE中, 有临窗不临窗的区别. 这其中的分配, 居然是按照在微软的工龄来的. 我刚进去的时候, 看见一老兄在角落的一个大个办公室, 两面是窗, 不由肃然起敬. 后来发现, 这老哥只是个打杂的, 进公司年头略久.
      <br/>
      <br/>
      3. 员工比较有想法, 也比较敢说,
      <br/>
      <br/>
      因为官员的权力受到一定的限制, 当民工的, 就比较胆子大一点了. 因为顶撞, 被当官的当场解雇的情形, 在微软是不可想象的, 虽然在其他公司是比较常见的.
      <br/>
      <br/>
      我来微软的时候, 参加新职工培训. 当问到为什么加入微软的时候, 很多人说为能跟世界上最聪明的职工们一起工作而兴奋. 后来培训中, 也经常听到别人提这一点. 虽然有些人属于老奸巨滑的, 满嘴是政治正确的言论, 但大多数象是真心这么想的.
      <br/>
      <br/>
      实在说, 我没有感觉微软的人有多么聪明. 总体上的感觉, 在我工作过的公司中大概是中游偏上. 在其他公司, 我还碰上了若干让我很佩服的牛人. 在微软, 牛人也有, 但很佩服的曲指可数. 对产品很熟的人, 更是少到可怜. 这也许跟微软不重视CONSULTANT有关. 平心而论, 微软职工的平均水平, 恐怕也就刚到B的水准.
      <br/>
      <br/>
      其他公司, 你能看见的是每一个小组都有个小牛, 每个大组都有个中牛. 在微软, 很多组里是一群猪, 牛毛也没半个. 当然也有组里, 全是小牛. 但微软的中牛和领导型的大牛严重缺乏, 所以盲人走夜路的情况比比皆是.
      <br/>
      <br/>
      <br/>
      （4）
      <br/>
      <br/>
      说完了对老帮菜的好处, 接着说对小年轻的好处.
      <br/>
      <br/>
      1. 微软的图书馆相当不错. 十分方便而且种类巨多.
      <br/>
      <br/>
      买微软出版社的书, 只需要付点象征意义的钱, 就可以把大部头扛回家. 如果你是个书虫, 学习狂, 努力上进青年, 老得闲得发慌, 想去欧洲但打算先做点语言方面的准备工作, 都可以免费获得.
      <br/>
      <br/>
      2. 同样, 微软的各种培训特多. 学英文, 学中文, 学日文, 学数据库, 学写作, 应有尽有. 很多不需要任何批准, 报名就去, 完全免费.
      <br/>
      <br/>
      3. 微软的软件众多, 公司里用都是免费, 自己用也很便宜.
      <br/>
      <br/>
      4. 微软的工作机会种类很多. 当你干IE厌烦了, 你可以去干点OFFICE. WORD 写腻了, 可以去玩点XBOX的游戏. 不需要换公司, 工作年限继续增长.
      <br/>
      <br/>
      5. 微软不怎么管你上班下班的时间. 你在办公室, 门一关, 上网读中文, 下下棋, 打打牌, 没人管你. 你的活能交差就行.
      <br/>
      <br/>
      6. 几乎每座楼都有乒乓球桌, 台球桌, XBOX 360, WII, 游戏机等各种休闲设施. 你尽管放松好了.
      <br/>
      <br/>
      7.微软的所有验证, 对职工都是免费的. 有人一口气考了七十几个微软验证. 将来出去当CONSULTANT, 挣钱还是很多的.
      <br/>
      <br/>
      8. 微软的资历, 目前来说, 还是管用的. 简历上一写, 对进其它公司, 特别是古狗之类, 帮助还是很大的.
      <br/>
      <br/>
      有人说了, 你怎么不提医疗保险, 免费饮料, 免费PRO CLUB MEMBERSHIP 等等众多好处. 我想, 这些都不是主要吸引人的地方. 微软的医疗保险虽然比绝大多数美国公司的都好, 但比加拿大公司也差不多. 而且这些都是钱能买来的. 谈到钱, 大头是工资奖金股票, 这些都是小头, 没什么了不起的. 况且, 说不定将来的某天就消失不见了.
      <br/>
      <br/>
      说到微软的衰落, 我主要是侧重管理上的问题. 市场战略上, 的确也有问题. 主要体现在墨守成规, 企图一招吃遍天下等等. 不过, 我感觉, 管理上的问题导致了这些问题. 所以管理是因, 其他是果.
      <br/>
      <br/>
      下面呢, 就具体讲讲各种问题.
      <br/>
      <br/>
      1. 奖罚可笑, 人心涣散.
      <br/>
      <br/>
      我加入微软的时候, 每年两次REVIEW, 按五分制计算. 虽然理论上最高分是5, 但真正能拿到的凤毛麟角. 4.5 分就很罕见了. 在一般的组里, 4分就是最高分了.
      <br/>
      <br/>
      在低分方面, 2.5 分是最低分, 属于准备解雇的前奏. 拿2.5分的也不多, 比拿4.5分的可能略多一点.
      <br/>
      <br/>
      这样的话, 其实只有三档: 4分, 3.5分和3分. 大概按照25%, 35% 和 35% 的分配比例. 三分属于不及格, 惩罚也很严厉. 在奖金和股票中, 一项是空缺.
      <br/>
      <br/>
      35%的比例意味着, 整个部门, 有超过三分之一的人被打成右派. 这对民工的压力是巨大的, 对经理们来说, 压力也很大. 每年的REVIEW, 都是DRAMA上演的时候, 打得是不可开交.
      <br/>
      <br/>
      不是说笑话, 每年被这个3分逼傻, 逼疯, 逼退的人真是不少.
      <br/>
      <br/>
      当微软是顶点和接近顶点的时候, 3分就是最大的惩罚. 因为奖励带来的是股票, 而股票带来的是巨大的经济利益, 没有奖励就意味着痛入骨髓的惩罚. 因此, 绝大多数人为了避免惩罚而狂热加班. 他们的梦想是三年或者五年后的退休生活.
      <br/>
      <br/>

      微软有个五湖四海政策. 很多培训都强调这一点, 我已经背得滚瓜烂熟. 不过执行起来, 就很难讲是怎么一回事情了.
      <br/>
      <br/>
      大家都知道的是, 一个部门, 如果高级领导被换成是个印度人, 一年下来, 领导层的一半以上就会被染黑, 都是印度人的天下了. 中国人在这方面就比较顾忌一些. 某些中国领导要特意表现地对中国人严厉一些, 免得被人告状.
      <br/>
      <br/>
      微软里, 印度版块最集中的, 是ONLINE和MBS, 这两个地方是傻气牙曾经和现在战斗过的地方, 所以, 不让人惊讶. 俄罗斯人呢, 则很多集中在SQL部门. 他们数学学得好, 做算法是能手, 所以也是预料之中的. 中国人呢, 主力现在集中在ONLINE, 陆老爷这个大树下, 大家可以伸伸腿么. 其他地方, 除了RESEARCH, 基本是散兵游勇, 最多是小股游击队.
      <br/>
      <br/>
      此外, 巴西组织, 罗马尼亚匪帮, 犹太联盟, 意大利黑手党都很猖獗. 所以, 公司里, 人是轻易得罪不得的.
      <br/>
      <br/>
      以上或按民族, 或按国籍分的. 政治派系, 很多时候超越了这些NAIVE的分类, 所以先有盖瓷帮VS.爆沫帮, 后有WINDOWS党 VS. Office党 的说法.
      <br/>
      <br/>
      当年微软华人三巨头之一, 沈同学被盖瓷逼着来到总部做VP. 沈同学当然不情不愿. 在国内做土皇帝比在总部做个小VP舒服多了. 总部的水有多深, 做梦也会被吓醒的. 沈同学被盖瓷请到家里吃了顿饭. 饭桌上找话题, 大家就聊起了我党的江派对朱派以及江派对胡派的斗争和合作. 盖瓷越听越狐疑, 终于忍不住问道, 你这不是在演义微软的故事吧? 做为微软的盖CORE, 这方面的表现比江CORE要差上一个数量级. 跟爆沫斗了N年, 终于被迫退位. 爆沫明确宣布, 我才是SENIOR PARTNER. 我的决定, 不需要盖瓷批准. 排除了大家误以为盖瓷是幕后指挥的可能.
      <br/>
      <br/>
      盖瓷一退, OFFICE的地主JEFF 瑞克发现自己这辈子无望当上微软的总裁, 就只好退而求其次, 追随盖瓷, 去盖瓷的基金会当上了CEO. 这才引来了JUNIPER的CEO跑到OFFICE做总裁, WINDOWS的总裁跑到JUNIPER做CEO的喜剧故事.
      <br/>
      <br/>
      盖瓷的影响力的衰落到了令人发指的地步. 2007年, 盖瓷突然吃饱了撑的似的给全体职工发了一封EMAIL. 信中, 盖瓷如同一个愤青般的写道, 我们必须思考WINDOWS和OFFICE的前途是什么. 也许, 我们必须从头重写一个崭新的OFFICE. 信发得没头没脑, 莫名其妙. 过了半天, 若干个VP写信给大家, 企图解释这件事情. 某VP是这么写的, 盖瓷指的是将来的大方向, 大家不要误解. 可三年过去了, 大家看见的是, WINDOW 7 是一个VISTA的优化版, 而OFFICE 2010呢, 跟OFFICE 2007基本没多大变化. 下一个版本呢? WINDOWS 8 是号称加强对ONLINE的支持, OFFICE 2013则根本不可能重写.
