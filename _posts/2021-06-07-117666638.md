    <p>
     <strong>
      在服务器开启防火墙的状态下，若不开放端口，无法通过公网来访问我们的服务，可以选择关闭防火墙（不推荐）或者开启端口来访问我们的服务。
     </strong>
    </p>
    <hr/>
    <h4>
     <a id="Linux_httpsblogcsdnnetWindNolosearticledetails117665311_4">
     </a>
     需先设置防火墙–&gt;
     <a href="https://blog.csdn.net/WindNolose/article/details/117665311">
      Linux 启动关闭防火墙
     </a>
    </h4>
    <hr/>
    <h4>
     <a id="_6">
     </a>
     查看已经开启的端口：
    </h4>
    <pre><code class="prism language-bash"><span class="token function">netstat</span> -anp
</code></pre>
    <h4>
     <a id="_13">
     </a>
     开端口命令：
    </h4>
    <pre><code class="prism language-bash">firewall-cmd --add-port<span class="token operator">=</span>端口号/tcp --permanent
</code></pre>
    <h4>
     <a id="_19">
     </a>
     关端口命令：
    </h4>
    <pre><code class="prism language-bash"> firewall-cmd --permanent --remove-port<span class="token operator">=</span>端口号/tcp
</code></pre>
    <h4>
     <a id="_26">
     </a>
     保存设置：
    </h4>
    <pre><code class="prism language-bash">firewall-cmd --reload
</code></pre>
    <h4>
     <a id="_33">
     </a>
     查看端口是否已经开启：
    </h4>
    <pre><code class="prism language-bash">firewall-cmd --query-port<span class="token operator">=</span>端口号/tcp
</code></pre>
