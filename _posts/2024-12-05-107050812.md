    <h2 style="margin-left:0in;">
     <span style="color:#7c79e5;">
      方法、工具和过程
     </span>
    </h2>
    <ul>
     <li style="margin-left:0in;">
      软件工程方法为软件开发提供了“如何做”的技术
     </li>
     <li style="margin-left:0in;">
      软件工具为软件工程方法提供了自动的或半自动的软件支撑环境
     </li>
     <li style="margin-left:0in;">
      软件工程过程定义了：
     </li>
     <li style="margin-left:0in;">
      方法使用的顺序
     </li>
     <li style="margin-left:0in;">
      要求交付的文档资料
     </li>
     <li style="margin-left:0in;">
      为保证质量和适应变化所需要的管理
     </li>
     <li style="margin-left:0in;">
      软件开发各个阶段完成的里程碑
     </li>
    </ul>
    <h2 style="margin-left:0in;">
     <span style="color:#7c79e5;">
      软件工程项目的基本目标
     </span>
    </h2>
    <ul>
     <li style="margin-left:0in;">
      付出较低的开发成本
     </li>
     <li style="margin-left:0in;">
      达到要求的软件功能
     </li>
     <li style="margin-left:0in;">
      取得较好的软件性能
     </li>
     <li style="margin-left:0in;">
      开发的软件易于移植
     </li>
     <li style="margin-left:0in;">
      需要较低的维护费用
     </li>
     <li style="margin-left:0in;">
      能按时完成开发工作，及时交付使用
     </li>
    </ul>
    <h2 style="margin-left:0in;">
     <span style="color:#7c79e5;">
      软件工程的基本原理
     </span>
    </h2>
    <p style="margin-left:0in;">
     (1)用分阶段的生存周期计划严格管理
    </p>
    <p style="margin-left:0in;">
     （2)坚持进行阶段评审
    </p>
    <p style="margin-left:0in;">
     （3)实行严格的产品控制
    </p>
    <p style="margin-left:0in;">
     （4)米用现代程序设计技术
    </p>
    <p style="margin-left:0in;">
     （5)结果应能清楚地审查
    </p>
    <p style="margin-left:0in;">
     （6)合理安排软件开发小组的人
    </p>
    <p style="margin-left:0in;">
     （7）承认不断改进软件工程实践的重要性
    </p>
    <h2 style="margin-left:0in;">
     软件的特点
    </h2>
    <p style="margin-left:0in;">
     1.软件是一种逻辑实体，而不是具体的物理实体，因而它具有抽象性。
    </p>
    <p style="margin-left:0in;">
     2.软件是通过人们的智力活动，把知识与技术转化成信息的一种产品，是在研制、开发中被创造出来的
    </p>
    <p style="margin-left:0in;">
     3.在软件的运行和使用期间，没有硬件那样的机械磨损、老化问题。
    </p>
    <p style="margin-left:0in;">
     4.软件的开发和运行经常受到计算机系统的限制，对计算机系统有着不同程度的依赖性。在软件的开发和运行中必须以硬件提供的条件为基础。
    </p>
    <p style="margin-left:0in;">
     5.软件的开发至今尚未完全摆脱手工的开发方式。
    </p>
    <p style="margin-left:0in;">
     6.软件的开发费用越来越高，成本相当昂贵。
    </p>
