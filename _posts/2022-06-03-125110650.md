    <h3>
     <a id="_193">
     </a>
     布置地雷
    </h3>
    <br/>
    <p>
     这里使用了
     <code>
      rand()
     </code>
     函数在该方格没有地雷的情况下随机布置相应的地雷，有地雷的方格值为1，没有地雷的方格值为0，布置完地雷后再使用打印函数将界面布局打印出来
    </p>
    <br/>
    <p>
     <code>
      扩展：
     </code>
    </p>
    <p>
     在C语言中，我们一般使用
     <code>
      &lt;stdlib.h&gt;
     </code>
     头文件中的
     <code>
      rand()
     </code>
     函数来生成随机数
    </p>
    <pre><code>int rand (void);        //void 表示不需要传递参数。
</code></pre>
    <br/>
    <p>
     但是多次运行上面的代码，你会发现每次产生的随机数都一样
    </p>
    <p>
     实际上，
     <code>
      rand()
     </code>
     函数产生的随机数是伪随机数，是根据一个数值按照某个公式推算出来的，这个数值我们称之为
     <code>
      “种子”
     </code>
     ，我们可以通过
     <code>
      srand()
     </code>
     函数来重新
     <code>
      “播种”
     </code>
     ，这样种子就会发生改变。
    </p>
    <pre><code>void srand (unsigned int seed);
</code></pre>
    <br/>
    <p>
     它需要一个
     <code>
      unsigned int
     </code>
     类型的参数。在实际开发中，我们可以用时间作为参数，只要每次播种的时间不同，那么生成的种子就不同，最终的随机数也就不同。
    </p>
    <br/>
    <p>
     使用
     <code>
      &lt;time.h&gt;
     </code>
     头文件中的
     <code>
      time()
     </code>
     函数即可得到当前的时间（精确到秒），就像下面这样：
    </p>
    <pre><code>srand((unsigned)time(NULL));
</code></pre>
