     打开anaconda-environments
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/103ec019f4b0c2196355edfdd32322c0.png">

      我们可以看到目前只有两个环境，点击创建，创建一个新的python版本
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8dc1a5a8cd523806d6f8dd685b3094a0.jpeg#pic_center"/>

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/65d200900c589f1aad9efa11177b66ca.png">
      <br/>
      此时我们创建了一个新的Anacona环境下的python3.7.11（可以直接用这个版本，也可以覆盖为自己原来的版本），在你anaconda\envs处，可以找到新建的环境
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17c502f9365773e18ccdbd9c35f1354b.png">
       <br/>
       点击进去后，将除了conda-meta的文件全部删除，然后将原来的python3.7.4的文件复制到里面。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8428770e0f326a96467db55a2bde486d.png">
        <br/>
        使用不同环境之前先在environments激活环境。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bd4ea463cc39ea4494ac08af3d9a9a03.png">
         <br/>
         打开Noteboook-Edit-Edit Notebook Metadata查看当前环境下的python版本
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4355333e69c6b83b1eb8235523e559d8.png">
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f3067d04a5afdc6a878b5c570a0a414a.png">
           <br/>
           将python3.7.11 替换成了原本的python3.74，替换成功。
           <br/>
           这样我们就可以在anaconda里面用到原来python3.74版本的函数库，且可以切换不同python版本。
