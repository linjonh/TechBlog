<h5>
（4）编写“打开串口”槽函数
</h5>
<p style="text-align:center;">
<img alt="3ac63981166099d93d4020d7dad0af45.png" src="https://i-blog.csdnimg.cn/blog_migrate/04920ccec9e3a0ea2bd556dbefc292d0.png"/>

<p style="text-align:center;">
<img alt="5bc72e92d0f3792e3b92dc949ae4fd70.png" src="https://i-blog.csdnimg.cn/blog_migrate/d5b97b1fbd8c75d22fa541a43c59ddcb.png"/>
</p>
<p style="text-align:center;">
<img alt="d2c67cb12f3d9774707a1ba2358e7f11.png" src="https://i-blog.csdnimg.cn/blog_migrate/c6439e399e80cea54ebe4e227b67dc2d.png"/>
</p>
<p style="text-align:justify;">
信号和槽是用于对象之间的通信，它是 Qt 的核心机制。
</p>
<p style="text-align:justify;">
当某个事件发生之后，比如，按钮检测到自己被点击了一下，它就会发出一个信号（signal）。如果有对象对这个信号感兴趣，想要处理的信号和自己的一个函数（称为槽（slot））绑定来处理这个信号。也就是说，当信号发出时，被连接的槽函数会自动被回调。
</p>
<p style="text-align:justify;">
这里，我们点击
<code>
打开串口
</code>
按钮会发出
<code>
clicked 信号
</code>
，此时对应槽函数 on_openButton_clicked 会被调用。下面我们来实现这个槽函数：
</p>
<pre class="has"><code class="language-go">void Widget::on_openButton_clicked()
{
    /_  串口设置  _/
    serialPort-&gt;setPortName(ui-&gt;serialBox-&gt;currentText());
    serialPort-&gt;setBaudRate(ui-&gt;baudrateBox-&gt;currentText().toInt());
    serialPort-&gt;setDataBits(QSerialPort::Data8);
    serialPort-&gt;setStopBits(QSerialPort::OneStop);
    serialPort-&gt;setParity(QSerialPort::NoParity);

/_  打开串口提示框  _/
    if (true == serialPort-&gt;open(QIODevice::ReadWrite))
    {
        QMessageBox::information(this, "提示", "串口打开成功");
    }
    else
    {
        QMessageBox::critical(this, "提示", "串口打开失败");
    }
}</code></pre>
<p style="text-align:justify;">
这里我们写死数据位、停止位、求校验位；增加提示框。其中使用
<code>
QMessageBox
</code>
需要包含如下头文件：
</p>
<pre class="has"><code class="language-go">#include &lt;QMessageBox&gt;</code></pre>
<h5>
（5）编写“关闭串口”、“点灯”、“灭灯”槽函数
</h5>
<p style="text-align:justify;">

<code>
打开串口

<code>
关闭串口
</code>
、
<code>
点灯
</code>
、
<code>
灭灯
</code>
槽函数：
</p>
<pre class="has"><code class="language-go">void Widget::on_closeButton_clicked()
{
    serialPort-&gt;close();
}

void Widget::on_onButton_clicked()
{
    serialPort-&gt;write("ON\n");
    qDebug("ON\n");

void Widget::on_offButton_clicked()
{
    serialPort-&gt;write("OFF\n");
