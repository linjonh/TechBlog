    <h4 id="sun公司对java的描述如下java-is-a-simple-object-oriented-distributed-interpreted-robust-secure-architecture-neutral-portable-high-preformance-multithreaded-and-dynamic-language-大概意思是java是一门简单的面向对象的分布式的解释性的健壮的安全的结构合理的-便携-高性能多线程的动态的语言-具体而言有以下的优点">
     Sun公司对Java的描述如下:”
     <code>
      Java is a simple , object - oriented , distributed ,interpreted , robust , secure , architecture neutral , portable , high - preformance , multithreaded , and dynamic language "
     </code>
     ,大概意思是”
     <code>
      Java
     </code>
     是一门简单的,面向对象的,分布式的,解释性的,健壮的,安全的,结构合理的 ,便携 , 高性能,多线程的动态的语言 “,具体而言有以下的优点:
    </h4>
    <h4 id="1java为纯面向对象的语言java是一种everything-is-object的语言能够直接反映现实生活中的对象">
     1.
     <strong>
      Java为纯面向对象的语言
     </strong>
     .
     <code>
      Java
     </code>
     是一种
     <code>
      everything is object
     </code>
     的语言,能够直接反映现实生活中的对象.
    </h4>
    <h4 id="2平台无关性java语言可以一次编译到处运行编译后的代码可以在多种平台上运行由于java是解释性语言编译器会把java代码编程中间代码然后在jvm中运行因此java可以跨平台运行具有良好的可移植性">
     2.
     <strong>
      平台无关性
     </strong>
     .
     <code>
      Java
     </code>
     语言可以一次编译到处运行,编译后的代码可以在多种平台上运行.
     <strong>
      由于
      <code>
       Java
      </code>
      是解释性语言
     </strong>
     ,编译器会把Java代码编程中间代码,然后在
     <code>
      JVM
     </code>
     中运行,
     <strong>
      因此
      <code>
       Java
      </code>
      可以跨平台运行,具有良好的可移植性.
     </strong>
    </h4>
    <h4 id="3为完成特定任务内置的多种库通过使用这些库简化了开发者的开发难度例如java对多线程的支持网络通信的支持以及java中很重要的垃圾回收机制异常处理等">
     3.
     <strong>
      为完成特定任务内置的多种库.
     </strong>
     通过使用这些库,简化了开发者的开发难度,例如:
     <code>
      Java
     </code>
     对多线程的支持,网络通信的支持,以及
     <code>
      Java
     </code>
     中很重要的垃圾回收机制,异常处理等.
    </h4>
    <h4 id="4提供了对web应用开发的支持例如appletserver-jsp等">
     4.
     <strong>
      提供了对
      <code>
       WEB
      </code>
      应用开发的支持.
     </strong>
     例如:
     <code>
      Applet,Server , JSP
     </code>
     等
    </h4>
    <h4 id="5具有良好的安全性和健壮性网络是java的一个重要使用领域为了增强安全性-java提供例如防止恶意代码攻击的安全机制数组边界检测和bytecode校验等java的强类型机制垃圾回收器异常处理安全检查机制等都使得java编写的程序有着良好的健壮性">
     5.
     <strong>
      具有良好的安全性和健壮性.
     </strong>
     网络是
     <code>
      Java
     </code>
     的一个重要使用领域,为了增强安全性,
     <code>
      Java
     </code>
     提供例如
     <strong>
      防止恶意代码攻击的安全机制(数组边界检测和
      <code>
       ByteCode
      </code>
      校验等)
     </strong>
     ,
     <code>
      Java
     </code>
     的
     <strong>
      强类型机制
     </strong>
     ,
     <strong>
      垃圾回收器
     </strong>
     ,
     <strong>
      异常处理
     </strong>
     ,
     <strong>
      安全检查机制
     </strong>
     等都使得
     <code>
      Java
     </code>
     编写的程序有着良好的健壮性.
    </h4>
    <h4 id="6去除了c语言中难以理解容易混淆的特性指针结构单元运算符重载虚拟基础类多重继承等等使得程序更加严谨简洁">
     6.

      去除了

       C++
      </code>
