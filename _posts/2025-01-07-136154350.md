     <h4>
      如何运行Python程序
     </h4>
     <div id="content">
      不少新手朋友遇到的第一个问题，不是Python的数据结构，不是Python的类库使用，更不是第三方模块、框架的应用，而是简单的“如何运行”！
      <br/>
      Python可用以下三种方式运行，举例在Windows平台上，不过Linux平台亦是如此。
      <br/>
      <strong>
       1. 使用Python自带的IDLE
      </strong>
      <br/>
      在开始--&gt;程序--&gt;Python2.5(视你安装的版本而不同)中找到IDLE(Python GUI)，
      <br/>
      <br/>
      <p class="img-center">
       <img alt="" height="160" src="https://i-blog.csdnimg.cn/blog_migrate/e2262efe6c2b9d480a635a4dd92001e0.jpeg" width="462"/>
      </p>
      <br/>
      点击后弹出如下窗体：
      <br/>
      <p class="img-center">
       <img alt="" height="318" src="https://i-blog.csdnimg.cn/blog_migrate/8e6d28194b920ee7d19e641b8485f8cb.jpeg" width="584"/>
      </p>
      <br/>
      <br/>
      在&gt;&gt;&gt;提示符后输入代码，回车，就可以执行此代码
      <a href="http://yc.gptgaixie.com/article/37.html" rel="nofollow" title="用python绘制满天星100颗">
       用python绘制满天星100颗
      </a>
      。
      <br/>
      IDLE支持语法高亮，支持自动缩进，支持方法提示，不过提示的很慢。
      <br/>
      <p class="img-center">
       <img alt="" height="300" src="https://i-blog.csdnimg.cn/blog_migrate/3e16a334816ea9e7b6aa418e89e14b23.jpeg" width="585"/>
      </p>
      <br/>
      <br/>
      <strong>
       2. 在命令行窗口上运行
      </strong>
      <br/>
      这种方法的前提是：你在系统的PATH变量中配置了Python的安装路径。
      <br/>
      右键我的电脑--&gt;属性--&gt;高级--&gt;环境变量，在系统变量列表中找到Path项，点击编辑按钮，在其中追加“C:\Python25;”(路径及版本视你安装而定)，保存退出。
      <br/>
      <p class="img-center">
       <img alt="" height="487" src="https://i-blog.csdnimg.cn/blog_migrate/0be7d827c74656a3a6033ee89d45ecb0.jpeg" width="417"/>
      </p>
      <br/>
      <br/>
      开始--&gt;运行--&gt;输入cmd，回车，开启一个CMD窗口。
      <br/>
      <p class="img-center">
       <img alt="" height="168" src="https://i-blog.csdnimg.cn/blog_migrate/7c3c25eb9be41113fb00b593831c8ace.jpeg" width="347"/>
      </p>
      <br/>
      <br/>
      在DOS提示符&gt;后，输入python，回车，进入Python环境。
      <br/>
      <p class="img-center">
       <img alt="" height="201" src="https://i-blog.csdnimg.cn/blog_migrate/dde05831cff78d824c1f6b26016842d1.jpeg" width="436"/>
      </p>
      <br/>
      <br/>
      它的运行和IDLE基本一致，但是没有了语法高亮、自动缩进、方法提示，唯一的好处就是运行速度比IDLE快了些(如果你告诉我可以加参数运行python，那你就不算新手了，也不用看这篇文章了)，所以用处不大。
      <br/>
      退出此python环境使用Ctrl + Z，然后回车。
      <br/>
      <strong>
       3. 以脚本方式运行
      </strong>
      <br/>
      以上两种运行方式虽然简便，但是不适合大量代码的开发，只适合查看单句或少量几句代码的运行结果，或者验证某函数的调用方法，而这恰恰是我们平时调试、验证程序的常用方式。如果是正式的开发，则应该使用独立脚本的方式运行。
      <br/>
      打开你的文本编辑器(我是用EmEditor，当然你使用记事本、写字板也都可以)，输入python代码，保存成*.py文件，然后双击运行它就可以执行了，当然前提也是必须配置系统PATH变量。
      <br/>
      <p class="img-center">
       <img alt="" height="192" src="https://i-blog.csdnimg.cn/blog_migrate/a30cf921430352e35cea4f8c73d5b8b8.jpeg" width="329"/>
      </p>
      <br/>
      这个小程序的结果会一闪而过，如何让它停驻呢？有两个办法：
      <br/>
      l 在其所在目录下开启一个CMD窗口，输入python *.py运行
      <br/>
      <p class="img-center">
       <img alt="" height="145" src="https://i-blog.csdnimg.cn/blog_migrate/a4023d514c791fac28c4561672185dfc.jpeg" width="238"/>
      </p>
      <br/>
      l 在代码的最后增加如下语句：
      <br/>
      raw_input()
      <br/>
      <p class="img-center">
       <img alt="" height="192" src="https://i-blog.csdnimg.cn/blog_migrate/0110acb9ecbc790523805e91c4f02d96.jpeg" width="329"/>
      </p>
      <br/>
      然后你再双击运行，结果就会停留在那里，直到你敲击回车键才消失。
      <br/>
      <p class="img-center">
       <img alt="" height="89" src="https://i-blog.csdnimg.cn/blog_migrate/aa89f351060e9d97816e28e24ffbbb50.jpeg" width="244"/>
      </p>
      Linux下运行Python程序，一般说来有以下两种形式，其实和Windows下基本一样。
      <br/>
      <br/>
      一、在IDLE中运行
      <br/>
      在终端窗口输入
      <br/>
      $ python
      <br/>
      进入交互式运行环境，然后就可以边输入边执行代码了：
      <br/>
      &gt;&gt;&gt; print 'Hello Python'
      <br/>
      Hello Python
      <br/>
      &gt;&gt;&gt;
      <br/>
      退出使用Ctrl-D。
      <br/>
      <br/>
      二、以脚本方式运行
      <br/>
      在py脚本所在目录下输入
      <br/>
      $ python filename.py
      <br/>
      即可执行filename.py的程序。
