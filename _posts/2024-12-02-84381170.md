    <h2>
     在bat 文件开头指定java_home环境
    </h2>
    <p>
     <br/>
     set JAVA_HOME=D:\soft\jdk1.8
     <br/>
     set PATH=%JAVA_HOME%/bin;%JAVA_HOME%/jre/bin
     <br/>
     set CLASSPATH=.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
     <br/>
     注意:临时有效,过程结束则不再启作用
