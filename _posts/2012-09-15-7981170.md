      <p class="aligncenter" style="margin-top:5px; margin-bottom:20px; padding-top:5px; padding-bottom:5px; line-height:24px; width:589.5px">
       也许有些人不知道
       <span style="margin:0px; padding:0px; font-family:'Times New Roman'">
        Google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        ，
       </span>
       <span style="margin:0px; padding:0px; font-family:'Times New Roman'">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        百度都是搜索引擎网站。
        <br style="margin:0px; padding:0px"/>
       </span>
       百度是目前中国最大的中文搜索引擎
       <span style="margin:0px; padding:0px; font-family:'Times New Roman'">
        ,
       </span>
       Google
       <span style="margin:0px; padding:0px; font-family:宋体">
        是全球最大的并且最受欢迎的搜索引擎，主要的搜索服务有：网页搜索，图片搜索，视频搜索，地图搜索，新闻
       </span>
       搜索..
       <span style="margin:0px; padding:0px; font-family:宋体">
        等。
       </span>
      </p>
      <p style="margin-top:5px; margin-bottom:20px; padding-top:5px; padding-bottom:5px; line-height:24px; width:589.5px">
       暂且不谈
       <span style="margin:0px; padding:0px; font-family:Arial">
        Seo
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        等内容
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        就说说为什么
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        是全球最大而百度只是中国最大的搜索引擎。以前的时候，用得最多的是百度
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        因为浏览器首页是百度，而且当时也不知道百度谷歌有什么区别，而且谷歌还老是上不去，或者老是网络中断。
        <br style="margin:0px; padding:0px"/>
       </span>
       确实，百度对于平常未跟互联网打交道的人来说已经足够了，所以也不需要去了解什么谷歌或其他搜索引擎了。为什么会用到
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        呢，这肯定是百度搜不到的原因啦
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ..
        <br style="margin:0px; padding:0px"/>
       </span>
       现在从事
       <span style="margin:0px; padding:0px; font-family:Arial">
        IT
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        行业，有时候不懂的时候去问人家，人家就说一句
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        不懂就去
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        。然后我就跑到百度去搜索，可搜了半天也没搜到什么有用的东西出来。然后又去问，别人就说，去
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        啦，百度的度娘不给力的
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ..
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        然后我就去
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        了一下，搜索了一下，明显感觉出来了两个搜索引擎的不同之处，不说搜索速度，就说内容。搜索的时候，百度页面的前几条全是广告
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        而且还忒不明显，一不小心点错了。当然谷歌也有，只是相对于少些。你要赚钱，有广告，我想得通。但是对于搜索出来的内容
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        我就无语了。也许搜索常用关键字词还不是那么明显，但如果搜索一些不常有的或者技术性的文章，内容的差距就完全体现出来了，
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        搜索的时候，只要有相关的网页，它的前几页肯定都是你想看到的页面，至于页面展示顺序先不管。但是百度就不同了，我有好多次这样的感觉，有时候第一页根本啥都不是，全是些
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        SEO
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        页面或者广告，而第二页或第三页才开始有你想要的内容。谷歌搜索，第一是满足用户需求，而百度搜索第一却看的是该网页的优化，如果你优化够好，你网页里面什么内容都可以没有你都可以排到第一位，搜索用户很难搜索到想要的内容。我想这也是很多
       </span>
       <span style="margin:0px; padding:0px; font-family:Arial">
        IT
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        人士不想用百度的原因把。
        <br style="margin:0px; padding:0px"/>
       </span>
       好像
       <span style="margin:0px; padding:0px; font-family:Arial">
        google
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        在前几年也是中国最大的搜索引擎，后来被赶出了中国，至于为什么被赶出去，在网上了解了一下,得到谷歌退出中国最主要的原因是,谷歌浏览器不愿意对搜索结果进行筛选,就是只要能搜出来的都直接显示给用户，而中国这个社会有很多东西国家严格控制不想让我们普通人知道,比如一些负面新闻或者跟领导人有关的内容都不能显示,谷歌不想这样做，所以没办法只能退出了.百度没事就是因为百度对搜索做了很多筛选,所以你懂的。当前,这也仅仅是表面现象,这儿有两篇
       </span>
       <span style="margin:0px; padding:0px; font-size:12px">
        <a href="http://www.williamlong.info/archives/377.html" rel="nofollow noopener noreferrer" style="margin:0px; padding:0px; color:rgb(155,187,89); text-decoration:initial; outline:none" target="_blank" title="百度的原罪—封杀Google的黑内幕">
         <span style="margin:0px; padding:0px; color:rgb(255,0,0)">
          百度的原罪—封杀Google的黑内幕
         </span>
        </a>
        与
        <span style="margin:0px; padding:0px">
         <a href="http://zh.wikipedia.org/wiki/%E8%B0%B7%E6%AD%8C%E4%B8%AD%E5%9B%BD#.E6.B6.89.E9.BB.84.E4.BA.8B.E4.BB.B6" rel="nofollow noopener noreferrer" style="margin:0px; padding:0px; color:rgb(155,187,89); text-decoration:initial; outline:none" target="_blank" title="谷歌中国">
          谷歌中国
         </a>
         非常详细的诉说了Google退出中国的原因.
        </span>
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        现在谷歌中国的服务器放在香港，所以也导致谷歌很多时候掉线或者上不了，这是一个不得不说的悲剧。
        <br style="margin:0px; padding:0px"/>
       </span>
       百度在中国很出名，但只是中国，在国外很多人都没听过百度是神马咚咚
       <span style="margin:0px; padding:0px; font-family:Arial">
        ,
       </span>
       <span style="margin:0px; padding:0px; font-family:宋体">
        虽然这跟百度侧重于做中文搜索引擎也有关系，但这不排除它确实很有问题，因为在中国都有非常多的人不想用他，用他的原因只是因为谷歌上不去了或者其他。
       </span>
      </p>
