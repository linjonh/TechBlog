    <h3>
     <a id="_2">
     </a>
     管理/开发工具
    </h3>
    <ul>
     <li>
      <a href="https://www.pgadmin.org/" rel="nofollow">
       pgAdmin4
      </a>
      ，基于 Web 的 PostgreSQL 图形管理和开发工具。
     </li>
     <li>
      <a href="https://github.com/phppgadmin/phppgadmin" rel="nofollow">
       phpPgAdmin
      </a>
      ，基于 Web 的 PostgreSQL 图形管理工具。
     </li>
     <li>
      <a href="http://www.teampostgresql.com/index.jsp" rel="nofollow">
       TeamPostgreSQL
      </a>
      ，基于 Web 的 PostgreSQL 图形管理和开发工具。
     </li>
     <li>
      <a href="https://www.pgadmin.org/docs/pgadmin4/latest/pgagent.html" rel="nofollow">
       pgAgent
      </a>
      ，PostgreSQL 后台作业调度代理程序。
     </li>
     <li>
      <a href="http://www.bigsql.org/docs/pgcli/pgcli/" rel="nofollow">
       PGC
      </a>
      ，一个命令行工具，用于 PostgreSQL 和相关项目的安装、更新和管理。
     </li>
     <li>
      <a href="https://github.com/pgpartman/pg_partman" rel="nofollow">
       pg_partman
      </a>
      ，PostgreSQL 分区管理工具。
     </li>
     <li>
      <a href="https://www.pgaudit.org/" rel="nofollow">
       pgAudit
      </a>
      ，为 PostgreSQL 提供详细的会话/对象操作审计日志。
     </li>
     <li>
      <a href="https://github.com/citusdata/pg_cron" rel="nofollow">
       pg_cron
      </a>
      ，基于 cron 的后台作业调度程序。
     </li>
     <li>
      <a href="http://www.ora2pg.com/" rel="nofollow">
       Ora2Pg
      </a>
      ，Oracle/MySQL 到 PostgreSQL 迁移的免费工具。
     </li>
     <li>
      <a href="https://pgloader.io/" rel="nofollow">
       pgloader
      </a>
      ，支持各种数据源到 PostgreSQL 的数据加载和迁移。
     </li>
     <li>
      <a href="https://soft-builder.com/en/erbuilder-data-modeler/" rel="nofollow">
       ERBuilder
      </a>
      ，Windows 平台数据库建模工具，支持各种主流的数据库。
     </li>
     <li>
      <a href="https://pgmodeler.io/" rel="nofollow">
       pgModeler
      </a>
      ，PostgreSQL 数据库建模工具，支持跨平台。
     </li>
     <li>
      <a href="https://github.com/darold/pgFormatter" rel="nofollow">
       pgFormatter
      </a>
      ，SQL 语法格式化工具。
     </li>
     <li>
      <a href="http://pgbadger.darold.net/" rel="nofollow">
       pgBadger
      </a>
      ，PostgreSQL 日志分析工具。
     </li>
     <li>
      <a href="http://pgcluu.darold.net/" rel="nofollow">
       pgCluu
      </a>
      ，PostgreSQL 数据库性能监控和审计工具。
     </li>
     <li>
      <a href="https://www.postgrestats.com/" rel="nofollow">
       PostgreStats
      </a>
      ，PostgreSQL 可视化监控工具。
     </li>
    </ul>
    <h3>
     <a id="_22">
     </a>
     备份与恢复
    </h3>
    <ul>
     <li>
      <a href="http://www.pgbarman.org/" rel="nofollow">
       Barman
      </a>
      ，PostgreSQL 备份恢复管理器。
     </li>
     <li>
      <a href="https://pgbackrest.org/" rel="nofollow">
       pgBackRest
      </a>
      ，一个简单可靠的备份恢复工具。
     </li>
     <li>
      <a href="https://github.com/ossc-db/pg_bulkload" rel="nofollow">
       pg_bulkload
      </a>
      ，数据批量加载工具。
     </li>
    </ul>
    <h3>
     <a id="_28">
     </a>
     集群和复制
    </h3>
    <ul>
     <li>
      <a href="https://www.2ndquadrant.com/en/resources/postgres-bdr-2ndquadrant/" rel="nofollow">
       Postgres-BDR
      </a>
      ，分布式 PostgreSQL 多主集群方案。
     </li>
     <li>
      <a href="https://www.2ndquadrant.com/en/resources/pglogical/" rel="nofollow">
       pglogical
      </a>
      ，PostgreSQL 逻辑复制插件。
     </li>
     <li>
      <a href="https://github.com/zalando/patroni" rel="nofollow">
       Patroni
      </a>
      ，PostgreSQL HA 模板，支持 ZooKeeper、etcd、Consul 以及 Kubernetes。
     </li>
     <li>
      <a href="https://github.com/zalando/spilo/" rel="nofollow">
       spilo
      </a>
      ，PostgreSQL HA 集群 Docker 镜像。
     </li>
     <li>
      <a href="https://bucardo.org/Bucardo/" rel="nofollow">
       Bucardo
      </a>
      ，支持多主和多从节点的异步复制系统。
     </li>
     <li>
      <a href="https://pgchameleon.org/" rel="nofollow">
       pg_chameleon
      </a>
      ，从 MySQL 到 PostgreSQL 的复制系统。
     </li>
     <li>
      <a href="http://2ndquadrant.com/resources/pglogical/" rel="nofollow">
       pglogical
      </a>
      ，PostgreSQL 逻辑复制。
     </li>
     <li>
      <a href="https://www.pgpool.net/mediawiki/index.php/Main_Page" rel="nofollow">
       Pgpool-II
      </a>
      ，管理 PostgreSQL 集群的中间件。
     </li>
     <li>
      <a href="https://repmgr.org/" rel="nofollow">
       repmgr
      </a>
      ，PostgreSQL 集群复制和故障切换管理工具。
     </li>
     <li>
      <a href="http://slony.info/" rel="nofollow">
       Slony-I
      </a>
      ，支持 PostgreSQL 一主多从以及级联复制。
     </li>
     <li>
      <a href="http://sqlrelay.sourceforge.net/index.html" rel="nofollow">
       SQL Relay
      </a>
      ，数据库连接管理方案。
     </li>
    </ul>
    <h3>
     <a id="_43">
     </a>
     驱动和接口
    </h3>
    <ul>
     <li>
      <a href="http://www.pygresql.org/" rel="nofollow">
       PyGreSQL
      </a>
      ，Python 连接 PostgreSQL 数据库的模块。
     </li>
     <li>
      <a href="https://metacpan.org/release/DBD-Pg" rel="nofollow">
       DBD::Pg
      </a>
      ，Perl DBI 模块的 PostgreSQL 数据库驱动。
     </li>
     <li>
      <a href="http://pqxx.org/development/libpqxx/" rel="nofollow">
       libpqxx
      </a>
      ，C++ 连接 PostgreSQL 的官方接口。
     </li>
     <li>
      <a href="https://jdbc.postgresql.org/" rel="nofollow">
       PgJDBC
      </a>
      ，PostgreSQL JDBC 官方驱动。
     </li>
     <li>
      <a href="https://www.postgresql.org/ftp/odbc/versions/" rel="nofollow">
       psqlODBC
      </a>
      ，PostgreSQL ODBC 官方驱动。
     </li>
     <li>
      <a href="http://initd.org/psycopg/" rel="nofollow">
       psycopg
      </a>
      ，Python 连接 PostgreSQL 的驱动。
     </li>
    </ul>
    <h3>
     <a id="PostgreSQL__51">
     </a>
     PostgreSQL 衍生版
    </h3>
    <ul>
     <li>
      <a href="https://www.timescale.com/" rel="nofollow">
       TimescaleDB
      </a>
      ，基于 PostgreSQL 的时间序列数据库。
     </li>
     <li>
      <a href="https://www.2ndquadrant.com/en/resources/postgres-xl/" rel="nofollow">
       Postgres-XL
      </a>
      ，基于 PostgreSQL 的 MPP 数据库。
     </li>
     <li>
      <a href="https://bitnine.net/agensgraph-2-0-enterprise/" rel="nofollow">
       AgensGraph
      </a>
      ，基于 PostgreSQL 的图形数据库管理系统。
     </li>
     <li>
      <a href="https://github.com/greenplum-db/gpdb" rel="nofollow">
       Greenplum
      </a>
      ，基于 PostgreSQL 的 MPP 分析平台。
     </li>
    </ul>
    <h3>
     <a id="PostgreSQL__58">
     </a>
     PostgreSQL 扩展插件
    </h3>
    <ul>
     <li>
      <a href="http://postgis.net/" rel="nofollow">
       PostGIS
      </a>
      ，PostgreSQL 空间数据库扩展。
     </li>
     <li>
      <a href="https://github.com/orafce/orafce" rel="nofollow">
       Orafce
      </a>
      ，提供 Oracle 兼容的函数和程序包。
     </li>
     <li>
      <a href="https://pgxn.org/" rel="nofollow">
       https://pgxn.org/
      </a>
      ，提供了各种 PostgreSQL 插件和扩展包。
     </li>
     <li>
      <a href="https://github.com/HypoPG/hypopg" rel="nofollow">
       HypoPG
      </a>
      ，PostgreSQL 虚拟索引插件。
     </li>
     <li>
      <a href="https://github.com/powa-team/pg_qualstats/" rel="nofollow">
       pg_qualstats
      </a>
      ，统计 WHERE 和 JOIN 中的谓词，提供索引建议。
     </li>
     <li>
      <a href="https://github.com/rjuju/pg_track_settings" rel="nofollow">
       pg_track_settings
      </a>
      ，记录 PostgreSQL 配置参数的变更。
     </li>
     <li>
      <a href="https://www.pipelinedb.com/" rel="nofollow">
       PipelineDB
      </a>
      ，处理时间序列数据流。
     </li>
    </ul>
    <h3>
     <a id="_68">
     </a>
     外部数据库
    </h3>
    <ul>
     <li>
      <a href="http://www.bigsql.org/cassandra/fdw/" rel="nofollow">
       CassandraFDW
      </a>
      ，连接 Apache Cassandra 的 FDW。
     </li>
     <li>
      <a href="http://www.bigsql.org/athenafdw/" rel="nofollow">
       AthenaFDW
      </a>
      ，连接 Amazon Athena 的 FDW。
     </li>
     <li>
      <a href="https://github.com/EnterpriseDB/mysql_fdw" rel="nofollow">
       mysql_fdw
      </a>
      ，连接 MySQL 数据库的 FDW。
     </li>
     <li>
      <a href="https://github.com/citusdata/cstore_fdw" rel="nofollow">
       cstore_fdw
      </a>
      ，PostgreSQL 列式存储插件。
     </li>
     <li>
      <a href="https://github.com/laurenz/oracle_fdw" rel="nofollow">
       oracle_fdw
      </a>
      ，连接 Oracle 数据库的 FDW。
     </li>
     <li>
      <a href="https://github.com/tds-fdw/tds_fdw" rel="nofollow">
       tds_fdw
      </a>
      ，连接 Sybase 和 Microsoft SQL Server 的 FDW。
     </li>
    </ul>
    <h3>
     <a id="_77">
     </a>
     存储过程相关
    </h3>
    <p>
     PostgreSQL 默认支持以下过程语言：PL/pgSQL、PL/Tcl、PL/Perl 和 PL/Python。
    </p>
    <ul>
     <li>
      <a href="http://www.bigsql.org/docs/plprofiler/" rel="nofollow">
       plProfiler
      </a>
      ，PL/pgSQL 性能分析工具，可以分析存储过程和函数消耗的时间和运行次数。
     </li>
     <li>
      <a href="http://www.bigsql.org/docs/debugger/" rel="nofollow">
       plDebugger
      </a>
      ，PL/pgSQL 存储程序调式工具，支持断点调试。
     </li>
     <li>
      <a href="http://www.bigsql.org/docs/hintplan/" rel="nofollow">
       PG_HINT_PLAN
      </a>
      ，通过 hint 控制查询的执行计划。
     </li>
     <li>
      <a href="https://github.com/tada/pljava" rel="nofollow">
       PL/Java
      </a>
      ，Java 存储过程。
     </li>
     <li>
      <a href="https://github.com/postgres-plr/plr" rel="nofollow">
       PL/R
      </a>
      ，R 存储过程。
     </li>
    </ul>
