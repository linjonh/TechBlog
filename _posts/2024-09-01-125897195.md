    <h2>
     <a id="java_0">
     </a>
     java是什么
    </h2>
    <blockquote>
     <p>
      Java是一门
      <a href="https://baike.baidu.com/item/%E9%9D%A2%E5%90%91%E5%AF%B9%E8%B1%A1" rel="nofollow">
       面向对象
      </a>
      的编程语言，不仅吸收了
      <a href="https://baike.baidu.com/item/C%2B%2B" rel="nofollow">
       C++
      </a>
      语言的各种优点，还摒弃了C++里难以理解的多继承、
      <a href="https://baike.baidu.com/item/%E6%8C%87%E9%92%88/2878304" rel="nofollow">
       指针
      </a>
      等概念，因此Java语言具有功能强大和简单易用两个特征。Java语言作为静态面向对象编程语言的代表，极好地实现了面向对象理论，允许程序员以优雅的思维方式进行复杂的编程 [1] 。
      <br/>
      来源于百度
     </p>
    </blockquote>
    <p>
     java相对于其他语言来说有些特殊，他介于解释和编译型语言之间。
     <br/>
     扁你性语言的执行机制是直接翻译为机器码执行，这样做的缺点就是针对不同的指令集，就需要编译为不同的机器码，跨平台性不佳。而解释型语言则是由虚拟机直接解释执行源代码，这样做程序的运行效率变得很低。而java的机制则是先把Java编译为
     <code>
      .class
     </code>
     字节码文件，再由
     <code>
      jvm
     </code>
     虚拟机解释执行。根据不同的平台来开发不同的虚拟机。
    </p>
    <p>
     其实说java是一个程序设计语言不如说他是一个平台，他有虚拟机，有丰富的库，以及社区。
    </p>
    <h2>
     <a id="java_8">
     </a>
     java的几个版本
    </h2>
    <p>
     java发行以来，分出了三个版本
    </p>
    <ul>
     <li>
      java SE
     </li>
     <li>
      java EE
     </li>
     <li>
      java ME
     </li>
    </ul>
    <p>
     <code>
      java SE
     </code>
     是一个java的标准版，包含了基本的
     <code>
      jvm
     </code>
     和标准库。
     <br/>
     而
     <code>
      java EE
     </code>
     就是一个更加全面的java版本，包含了更多的库。但是基本的库和
     <code>
      jvm
     </code>
     虚拟机是不变的。
     <br/>
     <code>
      java ME
     </code>
     是一个适用于嵌入式系统的java，相当于轻量级的java。
     <code>
      java SE
     </code>
     的库在ME是不适用的，而且
     <code>
      jvm
     </code>
     虚拟机也是轻量级的。
    </p>
    <h2>
     <a id="jDKjRE_17">
     </a>
     jDK和jRE
    </h2>
    <p>
     初学java经常会听到;
    </p>
    <ul>
     <li>
      jdk
     </li>
     <li>
      jre
     </li>
    </ul>
    <p>
     首先是jre它是java运行环境。它的作用是运行已经编译好的java程序，说白了就是只负责运行不负责开发。
     <br/>
     而jdk是开发工具包，它包含了开发所需要的编译器，和基本库，也包含了jre的内容。
    </p>
