     微软 WPC 2014 合作伙伴keynote

    <p>
     本周一，2014 微软WPC (Worldwide Partner Conference) 合作者伙伴大会在美国华盛顿开幕，微软除了介绍了Azure、云端化的Office 365和Windows Phone移动平台之外，还简单提及了下一代Windows操作系统Windows 9的发展规划。
    </p>
    <p>
     做为微软金牌合作伙伴的
     <a href="http://www.gcpowertools.com.cn/" rel="nofollow noopener noreferrer" target="_blank">
      葡萄城控件
     </a>
     ，我们一直非常关注微软技术的发展趋势，下面就WPC 2014 Keynote进行分享：
    </p>
    <p>
    </p>
    <p>
     <a href="http://images.cnitblog.com/blog/139239/201407/161716458496429.jpg" rel="nofollow noopener noreferrer" target="_blank">
      <img alt="wpc-hero" border="0" height="274" src="http://images.cnitblog.com/blog/139239/201407/161716465371287.jpg" style="border-width: 0px; padding-top: 0px; padding-right: 0px; padding-left: 0px; display: inline; background-image: none;" title="wpc-hero" width="485"/>
     </a>
    </p>
    <ul>
     <li>
      <strong>
       Windows:
      </strong>
      最新数据显示，Windows占据着90%的PC市场；而在广阔的个人计算机市场Windows仅占14%(智能手机、平板电脑)。且公司已经在调整方向，主要应用已经从“仅适用于Windows”调整到“Windows第一，其他设备第二：Android、IOS”，如Office系列办公软件。Windows在未来还有很大的发展空间。
     </li>
     <li>
      <strong>
       13亿用户
      </strong>
      ： 微软指出Windows和Office的用户已经超过10亿用户。根据
      <a href="http://www.microsoft.com/en-us/news/bythenumbers/index.html" rel="nofollow noopener noreferrer" target="_blank">
       微软用户指数网站
      </a>
      看，更精确的数据是13亿。
     </li>
     <li>
      <strong>
       免费Windows：
      </strong>
      在今年4月的Build 2014，微软宣布它将提供Windows / Windows Phone的免费（9英寸或更小的屏幕）。“仅一周, 我们看到了31个新的设备获得了免费Windows许可” 特纳说，“而且每周都在加速增长，可以肯定的是，免费Windows战略已经初显成效。”
     </li>
     <li>
      <strong>
       Windows Phone的势头：
      </strong>
      微软首席运营官凯文·特纳说，对Windows Phone而言，过去的财年非常好。该平台已经比去年同期增长91％，增长率迄今为止位居行业第一。在14 个市场，位列智能手机第2 位; 在8 个市场，份额超过10%; 在24个市场：出货量超过iPhone；但是目前，Windows Phone 份额仅占 4%，未来市场机会很大。 在Windows Phone Store 应用，有27 万，且每天新增加 500 个, 总共有40 亿应用下载，自WP8 发布以来，付费应用每月收入增长 440%。 仅诺基亚 Lumia 520一款手机，就激活了1200 万部。
     </li>
     <li>
      <strong>
       通用的应用程序：
      </strong>
      随着Windows 8.1的发布，开发人员可以创建通用应用程序：手机、平板电脑、2合1和笔记本电脑、台式电脑。 特纳指出，让一个应用程序商店和一个开发的API是一个关键的是OS的API。但更多的是未来：随着新的Windows 9的发布，开发人员将能够创建一个通用的应用程序--可以在Xbox（TV）和大屏幕（PPI）的设备上运行。
     </li>
     <li>
      <strong>
       Windows 9：
      </strong>
      下一代Windows操作系统会统一开发接口，届时开发者只要编写一次程序，就可以让应用或者软件在不同设备上运行。他表示“下一代操作系统会满足一切用户对我们的期望。不过他并未透露新系统的发布时间，只是反复强调Windows 9的开发将会注意听取用户的心声。微软希望用户明白，下一代操作系统将不再采用目前Windows 8这种混合模式，而坚定走一体化方向。 (目前有传闻称，Windows 9正式版有望在2015年年初发布)
     </li>
     <li>
      <strong>
       物联网
      </strong>
      ：微软的做法，建立一个新的嵌入式操作系统, 将调用部分Windows的版本的API，物联网并专注于连通性。在WPC，特纳提供的只是一点点的详细信息。 “我们希望更多合作伙伴参与物联网”他说。 “如何将企业利用一切将要创建的数据，管理所有的设备。我们将物联网与信任的平台，一级功能强大的分析将会更加聪明。”
     </li>
     <li>
      <strong>
       新设备的到来：
      </strong>
      即将上市的新设备包括富士通的LIFEBOOK T904、HP Elitebook 1040 G1、W4 Acer Iconia 821、Dell XPS 13、Acer Aspire Switch 10、联想的 ThinkCentre M93p Tiny、东芝的Encore2、联想的 ThinkPad Yoga、HP的Beats 23、华硕的UX303、Dell Venue 11 Pro、三星的ATIV Book 9等等。微软在会上宣布惠普将推出一款199美元Windows
      <strong>
       笔记本
      </strong>
      ，名为Stream，主要竞争对手为Chromebook (约合人民币1200元)。
     </li>
    </ul>
    <p>
     关于新财年的业务发展战略，微软表示新的财年内会聚集5大业务，涉及云计算、移动、社交、大数据和商业智能以及安全业务。
    </p>
