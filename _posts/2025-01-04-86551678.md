    <h2>
     智能设备的研究与发展
    </h2>
    <ul>
     <li>
      嵌入式系统（Embedded System）也称作“嵌入式计算机系统（Embedded Computer System）”
     </li>
     <li>
      它是针对特定的应用，剪裁计算机的软件和硬件，以适应应用系统对功能、可靠性、成本、体积、功耗有严格要求的专用计算机系统
     </li>
     <li>
      嵌入式系统已经用于工业、农业、军事、家电等各个领域
     </li>
     <li>
      物联网环境中的各种智能设备都需要采用嵌入式技术来进行设计与制造
     </li>
     <li>
      嵌入式系统的设计理论、方法与工具是物联网研究的理论基础，具有重要的借鉴与指导作用。
     </li>
    </ul>

    <h3>
     微电子技术和产业发展的重要性
    </h3>
    <ul>
     <li>
      电子工业产值的增长速率是国民经济总产值增长速率的3倍，微电子产业的增长速率又是电子工业增长速率的2倍
     </li>
     <li>
      国民经济总产值每增加100~300元，就必须有10元电子工业和1元集成电路产值的支持
     </li>
     <li>
      集成电路对国民经济的贡献率远高于其他门类的产品，以单位质量钢筋对GDP的贡献为1计算，小汽车为5，彩电为30，计算机为1000，而集成电路的贡献率则高达2000
     </li>

    <h2>
     嵌入式技术的研究与发展
    </h2>
    <h3>
     环境智能化与嵌入式系统
    </h3>
    <ul>
     <li>
      未来98%的计算设备将工作在嵌入式系统中
     </li>
     <li>
      嵌入式系统是将计算与控制的概念联系在一起，并嵌入到物理系统之中，实现“环境智能化”的目的
     </li>
     <li>
      无线传感器网络、RFID应用系统是嵌入式技术在实现“环境智能化”方向迈出了一大步
     </li>
    </ul>
    <h3>
     嵌入式系统的基本概念
    </h3>
    <ul>
     <li>
      微型机应用和微处理器芯片技术的发展为嵌入式系统研究奠定了基础
     </li>
     <li>
      嵌入式系统的发展适应了智能控制的需求
     </li>
     <li>
      嵌入式系统的发展促进了芯片、操作系统、软件编程语言与体系结构研究的发展
     </li>
     <li>
      嵌入式系统的研究体现出多学科交叉融合的特点
     </li>
    </ul>
    <h3>
     嵌入式系统发展的过程
    </h3>
    <ul>
     <li>
      第一阶段：以可编程序控制器系统为核心的研究阶段
     </li>
     <li>
      第二阶段：以嵌入式中央处理器CPU为基础、简单操作系统为核心的阶段(eg.PLC)
     </li>
     <li>
      第三阶段：以嵌入式操作系统为标志的阶段
     </li>
     <li>
      第四阶段：基于网络操作的嵌入式系统发展阶段
     </li>
    </ul>

    <h3>
     可穿戴计算机在物联网中的应用
    </h3>
    <ul>
     <li>
      可穿戴计算机可以应用于物联网远程支援、抢险救灾、医疗救护、社会治安、新闻采访、社会娱乐与军事方面
     </li>
     <li>
      为了适应不同的用户需求，不同的可穿戴计算机根据其功能、与人交互方式的不同，而被设计成不同的内部结构和不同的外形
     </li>
    </ul>
    <h3>
     可穿戴计算概念产生的背景
    </h3>
    <ul>
     <li>
      可穿戴计算技术体现出“以人为本，人机合一”和“无处不在的计算”的理念，有力地支持着“从人围着计算机转，转向计算机围着人转” 这一计算机演变的重要趋势
     </li>
     <li>
      可穿戴计算系统与人类紧密结合成一个整体，能够拓展人的视觉、听觉，增强人的大脑记忆和应对外界环境变化的能力，延伸了人的大脑与四肢，代表着计算机的一个重要的发展趋势
     </li>
     <li>
      可穿戴计算技术研究涉及计算机科学、智能科学、光学工程、微电子技术、传感器技术、机械制造、通信科学、生物学、数学、生物医学、工业设计技术等多个交叉学科和技术领域
     </li>
    </ul>

    <h2>
     智能机器人研究及其在物联网中的应用
    </h2>
    <h3>
     机器人的基本概念
    </h3>
    <ul>
     <li>
      机器人学(robotics)是一个涉及计算机科学、人工智能方法、智能控制、精密机械、信息传感技术、生物工程的交叉学科
     </li>
     <li>
      按照应用领域进行分类，机器人可以分为民用和军用两大类
     </li>
     <li>
      民用机器人又可以进一步分为工业机器人、农业机器人、服务机器人、仿人机器人、特种机器人、微机器人与微操作机器人，以及空间机器人等
     </li>
     <li>
      军事机器人按照应用的目的分类，可以分为侦察机器人、监视机器人、排爆机器人、攻击机器人与救援机器人。按照工作环境分类，可以分为地面军用机器人、水下军用机器人、空中军用机器人
     </li>
    </ul>
    <p>
     <strong>
      智能机器人在物联网中的应用前景
     </strong>
    </p>
    <ul>
     <li>
      通过网络控制的智能机器人正在向我们展示出对世界超强的感知能力与智能处理能力
     </li>
     <li>
      智能机器人可以在物联网的环境保护、防灾救灾、安全保卫、航空航天、军事，以及工业、农业、医疗卫生等领域的应用中发挥重要的作用，必将成为物联网的重要成员
     </li>
     <li>
      物联网催生很多具有计算、通信、控制、协同和自治性能的智能设备，实现实时感知、动态控制和信息服务，智能机器人与物联网研究有很多共通之处
     </li>
     <li>
      智能机器人的研究成果与研究方法对于物联网理论研究具有极为重要的指导与借鉴意义
     </li>
    </ul>
