    <h3>
     <a id="_3">
     </a>
     嵌入式解决了什么问题?
    </h3>
    <p>
     我们有的时候写代码不是为了自己而写，而且是需要发给别人，比如别人拜托你帮忙写个什么文件批量重命名或者操作 excel 之类的，但是别人电脑上没有安装 python ,你想让一个小白安装 python 是一件麻烦事情不说，而且最终还可能各种报错
    </p>
    <p>
     这个时候先全部打包好，弄成一个可以直接运行 exe 或者 bat,不管什么格式，总之对方拿到之后双击就能运行就是成功，我们的 pyinstaller 就是前面一种，但是 pyinstaller 打出来的包巨大无比，而且有的时候打包单文件还会报错
    </p>
    <p>
     于是今天我们就来讲讲第二种，这一方法也是官方的推荐，官方看到我们使用 python 苦于打包，于是退出了嵌入式版本
    </p>
    <h3>
     <a id="_11">
     </a>
     什么是嵌入式？
    </h3>
    <p>
     嵌入式 python 是一个简化版的 python,这个版本的 python 省去了很多无用的东西，最终只留下一个非常精简的解释器，让其他电脑可以不用安装 python 也能执行 python ,这就是嵌入式的概念
    </p>
    <p>
     你可以理解为完整的 python 是食堂，你需要各种等待才能拿到需要的东西， python 嵌入式更像是快餐，你没办法在 python 嵌入式里面开发(大部分情况下),但是好在能够打开就用
    </p>
    <p>
     因为嵌入式需要一串启动咒语，所以我们一般是用
     <code>
      .bat
     </code>
     的方式来进行调用，方便新手能够一键启动，目前很火的 Ai 整合包也是采用同样的方法
    </p>
    <h3>
     <a id="_19">
     </a>
     如何安装嵌入式?
    </h3>
    <p>
     网上现在其实也有安装的教程， 但是不是特别详细，今天在这里完整的复述一遍，让小白能够一步步实现自己的打包梦
    </p>
    <h4>
     <a id="_23">
     </a>
     前往官网下载嵌入式
    </h4>
    <ol>
     <li>
      首先我们打开官网来到最新的官网页面
      <a href="https://www.python.org/" rel="nofollow">
       https://www.python.org/
      </a>
      <ul>
       <li>
        备注：
        <em>
         下载的页面排版更新了，如果你按老教程会找不到在什么地方
        </em>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     <img alt="image-20221019103253054" src="https://i-blog.csdnimg.cn/blog_migrate/cd99672c0f48f8e06a3c5de34f82baf6.png"/>
    </p>
    <ol start="2">
     <li>
      选择
      <code>
       Downloads
      </code>
      标签页，然后选择
      <code>
       Windows
      </code>
      ,点击
     </li>
    </ol>
    <p>
     <img alt="image-20221019103345590" src="https://i-blog.csdnimg.cn/blog_migrate/83266b8afa95538d080c05dc78089b0f.png"/>
    </p>
    <ol start="3">
     <li>
      在屏幕中寻找带
      <code>
       embeddable
      </code>
      字眼的安装包,版本不受限制，不过最好和本地现有的安装版本一致
      <ul>
       <li>
        (新手不要盲目安装3.10 +的版本，许多库都还没跟上最新版本，可能会导致少数库用不了)
       </li>
       <li>
        如果不知道自己的 python 版本，可以在命令行中输入
        <code>
         python
        </code>
        即可看到
       </li>
      </ul>
     </li>
    </ol>
    <p>
     <img alt="image-20221019103542791" src="https://i-blog.csdnimg.cn/blog_migrate/f053155b274bf740e383f21b7bb64263.png"/>
    </p>
    <h4>
     <a id="_40">
     </a>
     解压并配置嵌入式设置
    </h4>
    <blockquote>
     <p>
      我们现在从官方下载下来了嵌入式的版本，但是我们还需要进行一系列的配置我们才能够进行使用
     </p>
    </blockquote>
    <ol>
     <li>
      解压
     </li>
    </ol>
    <p>
     我们现在把刚刚下载的文件解压到一个地方
    </p>
    <p>
     <img alt="image-20221019103858883" src="https://i-blog.csdnimg.cn/blog_migrate/f10f4a73f09902795b71f52f4f900937.png"/>
    </p>
    <ol start="2">
     <li>
      下载
      <code>
       get-pip.py
      </code>
     </li>
    </ol>
    <p>
     我们发现这个版本的 python 没有 lib,我们需要自行下载，下载的方式也很简单，前往
     <a href="https://bootstrap.pypa.io/get-pip.py" rel="nofollow">
      https://bootstrap.pypa.io/get-pip.py
     </a>
     ,然后将所有内容拷贝下来，保存到同名的文件中
    </p>
    <p>
     <img alt="image-20221019104122649" src="https://i-blog.csdnimg.cn/blog_migrate/3e728c5ef62ccc0a77cead769933e916.png"/>
    </p>
    <p>
     你可以选择直接另存网站，或者选择全选拷贝，最终效果是一样的
    </p>
    <p>
     <img alt="image-20221019104205096" src="https://i-blog.csdnimg.cn/blog_migrate/a9563c8a8fefc56dd1ab15802db4a8f0.png"/>
    </p>
    <ol start="3">
     <li>
      执行
      <code>
       get-pip.py
      </code>
     </li>
    </ol>
    <p>
     我们拷贝完成之后选择退出，然后在标题栏输入
     <code>
      cmd
     </code>
    </p>
    <p>
     <img alt="image-20221019104331265" src="https://i-blog.csdnimg.cn/blog_migrate/b95de4943e91d20e7968571f44a5b0e5.png"/>
    </p>
    <p>
     在新打开的命令中输入
     <code>
      python get-pip.py
     </code>
    </p>
    <p>
     <img alt="image-20221019104421825" src="https://i-blog.csdnimg.cn/blog_migrate/3e9f218ec318060f4807b4fcf9398f94.png"/>
    </p>
    <p>
     此时我们的目录中多出两个文件夹
     <code>
      Lib
     </code>
     和
     <code>
      Scripts
     </code>
     ,我们后面会经常用到的只有
     <code>
      Lib
     </code>
     文件夹,我们在这里拷贝第三方的库
    </p>
    <p>
     <img alt="image-20221019104501843" src="https://i-blog.csdnimg.cn/blog_migrate/666b0320ceddd87fd03c6e7d3b35fb7d.png"/>
    </p>
    <ol start="4">
     <li>
      修改配置文件
     </li>
    </ol>
    <p>
     我们现在打开
     <code>
      python39._pth
     </code>
     ,使用记事本或者
     <code>
      notepad++
     </code>
    </p>
    <p>
     <img alt="image-20221019104624452" src="https://i-blog.csdnimg.cn/blog_migrate/6fed2dd7c27298993644a81830f38b13.png"/>
    </p>
    <p>
     可以看到刚刚打开的文件内容应该是这样的
    </p>
    <p>
     <img alt="image-20221019104711690" src="https://i-blog.csdnimg.cn/blog_migrate/8392ab93f15d67128e4e5135ccbb5fbe.png"/>
    </p>
    <p>
     去掉最后一行的注释
    </p>
    <pre><code class="prism language-python"><span class="token comment">#import site</span>
变成
<span class="token keyword">import</span> site
</code></pre>
    <p>
     <img alt="image-20221019104946258" src="https://i-blog.csdnimg.cn/blog_migrate/b734244c8dffef18d046ce5694d83051.png"/>
    </p>
    <p>
     到此为止，我们的安装和配置就结束了，接下来我们来介绍一下怎么使用
    </p>
    <h3>
     <a id="_96">
     </a>
     如何安装新的模块
    </h3>
    <p>
     切换到我们的
     <code>
      Scripts
     </code>
     文件夹里面，里面应该有一个
     <code>
      pip.exe
     </code>
     文件，这个时候在导航栏里面输入 cmd 打开控制台，然后正常的 pip 即可，最终文件会被安装到当前嵌入式环境的
     <code>
      Lib\site-packages
     </code>
     中
    </p>
    <p>
     <img alt="image-20221019143556682" src="https://i-blog.csdnimg.cn/blog_migrate/a4292141da3c0e36687cea95242dd614.png"/>
    </p>
    <h3>
     <a id="_102">
     </a>
     如何运行
    </h3>
    <p>
     在包含
     <code>
      python.exe
     </code>
     的主要目录下，打开在导航栏里面输入 cmd 打开控制台，然后输入 python 即可
    </p>
    <p>
     <img alt="image-20221019143622839" src="https://i-blog.csdnimg.cn/blog_migrate/63d7e788d609abbb8c5e6d37d4eae983.png"/>
    </p>
    <p>
     或者您可以把这个路径添加到系统环境变量中，这样也可以很方便的进行使用
    </p>
