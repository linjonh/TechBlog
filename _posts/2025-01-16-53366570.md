    <h2>
     <a id="4__81">
     </a>
     4. 具体使用
    </h2>
    <p>
     该款
     <a href="https://github.com/Carson-Ho/Search_Layout">
      SearchView
     </a>
     使用非常简单，仅需2步：
    </p>
    <h4>
     <a id="1_84">
     </a>
     步骤1：导入控件库
    </h4>
    <p>
     主要有
     <code>
      Gradle
     </code>
     &amp;
     <code>
      Maven
     </code>
     2种方式：
    </p>
    <ul>
     <li>
      方式1：
      <code>
       Gradle
      </code>
      引入依赖
      <br/>
      <em>
       build.Gradle
      </em>
     </li>
    </ul>
    <pre><code>dependencies {
    compile 'com.carson_ho:SearchLayout:1.0.1'
}
</code></pre>
    <ul>
     <li>
      方式2：
      <code>
       Maven
      </code>
      引入依赖
      <br/>
      <em>
       pom.xml
      </em>
     </li>
    </ul>
    <pre><code>&lt;dependency&gt;
  &lt;groupId&gt;com.carson_ho&lt;/groupId&gt;
  &lt;artifactId&gt;SearchLayout&lt;/artifactId&gt;
  &lt;version&gt;1.0.1&lt;/version&gt;
  &lt;type&gt;pom&lt;/type&gt;
&lt;/dependency&gt;
</code></pre>
    <h4>
     <a id="2_108">
     </a>
     步骤2：设置搜索框样式
    </h4>
    <ul>
     <li>
      具体属性设置
     </li>
    </ul>
    <p>
     <img alt="示意图" src="https://i-blog.csdnimg.cn/blog_migrate/ed0b215e436c90b4a70a71b412594cbd.png"/>
    </p>
    <ul>
     <li>
      使用示例
      <br/>
      在
      <code>
       XML
      </code>
      文件中进行设置
      <br/>
      <em>
       activity_main.xml
      </em>
     </li>
    </ul>
    <pre><code>&lt;scut.carson_ho.searchview.SearchView
        android:id="@+id/search_view"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        app:textSizeSearch="5dp"
        app:textColorSearch="#3F51B5"
        app:textHintSearch="输入查询关键字"
        app:searchBlockHeight="150"
        app:searchBlockColor="#ffffff"
        /&gt;
</code></pre>
    <h4>
     <a id="3___139">
     </a>
     步骤3：设置点击搜索按键 &amp; 返回按键后的操作
    </h4>
    <p>
     <em>
      MainActivity.java
     </em>
    </p>
    <pre><code>// 1. 初始化搜索框变量
    private SearchView searchView;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        
        // 2. 绑定视图
        setContentView(R.layout.activity_search);

        // 3. 绑定组件
        searchView = (SearchView) findViewById(R.id.search_view);

        // 4. 设置点击搜索按键后的操作（通过回调接口）
        // 参数 = 搜索框输入的内容
        searchView.setOnClickSearch(new ICallBack() {
            @Override
            public void SearchAciton(String string) {
                System.out.println("我收到了" + string);
            }
        });
        
        // 5. 设置点击返回按键后的操作（通过回调接口）
        searchView.setOnClickBack(new bCallBack() {
            @Override
            public void BackAciton() {
                finish();
            }
        });
    }
}
</code></pre>
