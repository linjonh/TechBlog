    <p>
     启动服务器F11进入boot manage界面，u盘的制作这里就不讲了，自行查找，主要说明安装步骤，进入界面如下。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5bca9cc0ddb3fc9ebb65ec455a4b9171.jpeg">
      <br/>
      选择Launch System Setup进入下一界面
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b084072938382b9cf8186fa0d45496fd.jpeg">
       <br/>
       选择System BIOS进入下一界面
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d5f7f835d6aedf16de7ac05b741092f2.jpeg">
        <br/>
        选择Integrated Devices进入下一界面
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69b8ea9c2e3738069d271d5e3e9d7814.jpeg">
         <br/>
         <strong>
          设置Embedded Video Controller为Enabled，否则无法使用虚拟KVM访问服务器进行u盘启动
         </strong>
         <br/>
         设置完毕后，点击Finish保存重启。重启后，F11进入boot manage界面，选择One-shot UEFI Boot Menu进入下一界面，
         <strong>
          注意这是UEFI模式启动u盘，u盘里要有EFI文件支持，也就是你制作的U盘要支持UEFI启动
         </strong>
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b596455efcfa1724c9b9ea8de73c5cad.jpeg">
          <br/>
          选择带USB字样按Enter键进行启动
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6664684248108b48cb45e3990a61b2ca.jpeg">
           <br/>
           启动成功后进入到如下界面
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b336dcc1c96f9854c373911e5116f570.jpeg">
            <br/>
            按键盘e改写盘名，删除多余，保证与你的优盘名相同，可参考
            <a href="https://blog.csdn.net/lianshaohua/article/details/88381039?utm_medium=distribute.pc_relevant.none-task-blog-OPENSEARCH-1&amp;depth_1-utm_source=distribute.pc_relevant.none-task-blog-OPENSEARCH-1">
             此篇
            </a>
            ，改完后如下
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>

    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/379fd5f949ed348eae9acdbbc995613a.jpeg">
      <br/>
      然后按Ctrl+x，等候进入图形安装界面
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/87af4b9593057b1b9a3d5db9c480b11a.jpeg">
       <br/>
       初次安装可以设置中文语言，点击继续进入下一界面
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8fe76614fbeac1e84d4ee7396c5b277.jpeg">
        <br/>
        日期和时间会自动获取，没有就选择上海时区，安装源自动获取为本地介质。在系统栏，选择安装位置，设置分区
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/635141296354ce5d9c32e0841477f85f.jpeg"/>
        <br/>
        选择✔项，再选择其它存储选项-分区-我要配置分区，点击左上完成
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b3292d20788cd1744ce47d947b108de0.jpeg"/>
        <br/>
        如果存在诸如
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/860cd081f4f3dcb877c1542f74962d47.png"/>
        ，点击下面的
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a0af5338f4c38c2ad5ce65edbb867b2f.png"/>
        <br/>
        按钮，逐一删除，如下所示
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa3beb18f3d658a37167b67e5f78a6ac.jpeg"/>
        <br/>
        完了之后，点击自动创建
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d14703f632ac31b2b068762f0ffd4096.jpeg"/>
        <br/>
        成功后，点击左上完成
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2b70bfa93a6befdf19145f87301f960e.jpeg"/>
        <br/>
        此后，会出现如下界面，点击接受更改
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/036757da390cc259a4efb5aa4b74c012.jpeg"/>
        <br/>
        成功后，点击开始安装，设置密码
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5e267da3046225e358890744e1126862.jpeg"/>
        <br/>
        设置完后，静待安装，安装完毕后，点击右下重启，什么都不用做，等候重启，进入longin界面
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8da2cae1ee2690b28bfaca06df37926.jpeg"/>
        <br/>
        输入用户名root和设置的密码，以及一些查看命令如下

        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/70fb4c90c7b6217c19c778f1cd067777.jpeg"/>

        至此，centOS 7.0 U盘安装完成
