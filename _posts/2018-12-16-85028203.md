    <h2>
     通过Docker或虚拟机快速搭建私有云平台
    </h2>
    <h4>
     --使用Nextcloud和ONLYOFFICE创建您的WebOffice云办公系统
    </h4>
    <p>
    </p>
    <p style="margin-left:0px;">
     <em>
      Web办公室是一种很好的工作方式，可以与您的团队协作，可以选择何时何地工作。本教程介绍如何使用
      <strong>
       <a href="https://nextcloud.com/" rel="nofollow">
        Nextcloud
       </a>
      </strong>
      和
      <strong>
       <a href="http://www.onlyoffice.com/" rel="nofollow">
        ONLYOFFICE
       </a>
      </strong>
      构建私有Web办公室，以
      <strong>
       <a href="https://nextcloud.com/" rel="nofollow">
        实时
       </a>
      </strong>
      管理，编辑和协作文档。开源和自托管这些工具都提供了高级安全功能，可以保护您的文档的私密性。
     </em>
    </p>
    <p style="margin-left:0px;">
    </p>
    <h3>
     <u>
      3种安装方式
     </u>
    </h3>
    <p style="margin-left:0px;">
     有几种方法可以使您的Web办公室正常运行。
    </p>
    <ol style="margin-left:20px;">
     <li>
      单独安装Nextcloud
     </li>
     <li>
      使用Docker compose一次安装与ONLYOFFICE集成的Nextcloud。
     </li>
     <li>
      使用Univention应用程序设备安装包含Univention Corporate Server，Nextcloud和ONLYOFFICE的VM。
     </li>
    </ol>
    <p>
    </p>
    <h4>
     1.单独安装Nextcloud
    </h4>
    <p style="margin-left:0px;">
     单独安装Nextcloud，然后安装并连接ONLYOFFICE。说明可
     <strong>
      <a href="https://www.itzgeek.com/how-tos/linux/edit-owncloud-nextcloud-documents-online-with-onlyoffice-online-editor.html" rel="nofollow">
       在此处获得
      </a>
     </strong>
     。
    </p>
    <p style="margin-left:0px;">
     <a href="https://blog.csdn.net/Aria_Miazzy/article/details/85028182">
      如何将OnlyOffice与NextCloud集成: https://blog.csdn.net/Aria_Miazzy/article/details/85028182
     </a>
    </p>
    <p style="margin-left:0px;">
    </p>
    <h4>
     2.使用Docker compose安装Nextcloud和ONLYOFFICE
    </h4>
    <p style="margin-left:0px;">
     这种类型的安装快速而轻松，因此您可以使用它首次尝试使用ONLYOFFICE进行Nextcloud。
    </p>
    <p style="margin-left:0px;">
     在开始安装Nextcloud和ONLYOFFICE之前，请确保在您的计算机上安装了
     <strong>
      <a href="https://www.itzgeek.com/tag/docker" rel="nofollow">
       Docker
      </a>
     </strong>
     和
     <strong>
      <a href="https://docs.docker.com/compose/install/" rel="nofollow">
       Docker Compose
      </a>
     </strong>
     。
    </p>
    <p style="margin-left:0px;">
     要开始安装过程，请使用以下命令下载最新版本的
     <strong>
      <a href="https://github.com/ONLYOFFICE/docker-onlyoffice-nextcloud">
       docker-onlyoffice-nextcloud存储库
      </a>
     </strong>
     ：
    </p>
    <pre class="has" style="margin-left:0px;"><code class="language-html">git clone --recursive https://github.com/ONLYOFFICE/docker-onlyoffice-nextcloud

cd docker-onlyoffice-nextcloud

git submodule update --remote</code></pre>
<p style="margin-left:0px;">
运行 Docker Compose：
</p>
<pre class="has" style="margin-left:0px;"><code class="language-html">docker-compose up -d</code></pre>
<p style="margin-left:0px;">
现在，您可以访问 Nextcloud，在浏览器地址栏中部署后，输入 Web 服务器地址。首次访问时，将出现设置向导，并要求您选择管理员帐户和密码。创建它并返回下载的文件夹。
</p>
<p style="margin-left:0px;">
运行
<code>
set_configuration.sh
</code>
脚本：
</p>
<pre class="has" style="margin-left:0px;"><code class="language-html">bash set_configuration.sh</code></pre>
<p style="margin-left:0px;">
然后重新启动页面。
</p>
<p style="margin-left:0px;">
</p>
<h4> 3.使用 Univention app 安装 Nextcloud 和 ONLYOFFICE
</h4>
<p style="margin-left:0px;">
Univention 应用程序是包含应用程序的 VirtualBox 或 VMware 映像 - Nextcloud 和 ONLYOFFICE - 以及 Univention Corporate Server。所有组件都是自动配置的：不需要管理员手动执行。
</p>
<p>
<u>
步骤 1.下载相应的 VM
</u>
</p>
<p style="margin-left:0px;">
您可以在
<strong>
<a href="https://www.univention.com/products/univention-app-center/onlyoffice-document-server-with-nextcloud/" rel="nofollow">
ONLYOFFICE-Nexctloud 中
</a>
</strong>
选择 VMware，VirtualBox，VMware ESX 和 KVM 映像   。在下载 VM 之前，请确保已为所选虚拟机
<strong>
<a href="https://my.vmware.com/en/web/vmware/free#desktop_end_user_computing/vmware_workstation_player/14_0" rel="nofollow">
VMware Workstation Player
</a>
</strong>
或
<strong>
<a href="https://www.itzgeek.com/tag/virtualbox" rel="nofollow">
VirtualBox
</a>
</strong>
安装了虚拟机监控程序    （请参阅文档以获取详细的安装指南）。
</p>
<p style="margin-left:0px;">
以下是下载地址：
</p>
<ul>
<li style="margin-left:0px;">
VirtualboxVM Image:
<a href="https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-virtualbox.ova" id="url" rel="nofollow">
https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-virtualbox.ova
</a>
</li>
<li style="margin-left:0px;">
VMware Image:
<a href="https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-vmware.zip" rel="nofollow">
https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-vmware.zip
</a>
</li>
<li style="margin-left:0px;">
KVM Image:
<a href="https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-KVM.qcow2" rel="nofollow">
https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-KVM.qcow2
</a>
</li>
<li style="margin-left:0px;">
<a href="https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-ESX.ova" rel="nofollow">
VMware ESX Image
</a>
:
<a href="https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-ESX.ova" rel="nofollow">
https://appcenter.software-univention.de/univention-apps/current/onlyoffice-ds/Univention-App-onlyoffice-ds-nextcloud-ESX.ova
</a>
</li>
</ul>
<p>
<u>
步骤 2.导入设备
</u>
</p>
<p style="margin-left:0px;">
对于 VMware，Workstation Player 解压缩下载的 VMware 设备文件，选择“打开虚拟机”选项，选择解压缩文件的子文件夹，然后打开 UCS-Demo-Image.vmx 文件，然后单击“打开电源”按钮以启动/启动虚拟机并运行预安装的 UCS 系统。
</p>
<p style="margin-left:0px;">
对于 VirtualBox，打开虚拟机管理程序，打开“文件”菜单，选择“导入设备”选项，接受软件许可协议，确认设置并开始导入。
</p>
<p>
<u>
步骤 3.安装应用程序
</u>
</p>
<p style="margin-left:0px;">
成功导入映像后，启动 VM。按照安装向导的步骤完成引导过程。
</p>
<ul style="margin-left:20px;">
<li>
调整正确的时区，键盘布局和其他参数，从可用列表中选择最近的城市。
</li>
<li>
指定系统的网络设置。自动获取 IP 地址或手动指定。
</li>
<li>
选择您的域设置：加入现有 UCS 或 Microsoft Active Directory 域，或者选择直接在此系统上选择“管理用户和权限”选项来设置新域。
</li>
<li>
填写您的帐户信息：公司名称，激活 ONLYOFFICE 的管理员电子邮件地址和新密码。
</li>
<li>
指定系统和 LDAP 基础的完整域名。
</li>
</ul>
<p style="margin-left:0px;">
最后，检查您的设置，然后单击“配置系统”按钮以启动配置过程。
</p>
<p style="margin-left:0px;">
配置完所有内容后，您的网络办公室就可以开始工作了。
</p>
<p style="margin-left:0px;">
</p>
<h3>
<u>
<u>
开始使用您的文档
</u>
</u>
</h3>
<p style="margin-left:0px;">
要创建文档，电子表格或演示文稿，只需单击 Nextcloud Web 界面顶部的
<strong> +
</strong>
按钮，然后选择适当的文件类型。
</p>
<p style="margin-left:0px;">
要开始编辑现有的文本文档，电子表格或演示文稿，请使用文件菜单中显示的“
<strong>
在 ONLYOFFICE 中打开”
</strong>
选项。
</p>

</p>

</p>

使用 Nextcloud 和 ONLYOFFICE - onlyoffice 文件菜单 nextcloud 创建您的 Web 办公室。
</p>
