     1. 目标相同
    </h6>
    <p>
     网络空间安全专业和信息安全专业都致力于保护计算机系统、网络和数据等信息资产的安全性。
    </p>
    <h6>
     <a id="httpsblogcsdnnetweixin_57514792articledetails140434580httpsblogcsdnnetSpringJavaMyBatisarticledetails139456680spm1001201430015502httpsblogcsdnnetlogic1001articledetails132448851spm10012014300155022_15">

     </a>
     2.基础知识相同
    </h6>
    <p>
     两个专业都需要学习计算机科学的基础知识，如计算机网络、操作系统、数据结构和算法等。
    </p>
    <h6>
     <a id="httpsblogcsdnnetweixin_57514792articledetails140434580httpsblogcsdnnetSpringJavaMyBatisarticledetails139456680spm1001201430015502httpsblogcsdnnetlogic1001articledetails132448851spm10012014300155023_19">

     </a>
     3.安全技术
    </h6>
    <p>
     两个专业都需要学习和了解各种安全技术，如网络安全、应用程序安全、密码学、漏洞分析与利用等。
    </p>

    <h6>
     <a id="httpsblogcsdnnetweixin_57514792articledetails140434580httpsblogcsdnnetSpringJavaMyBatisarticledetails139456680spm1001201430015502httpsblogcsdnnetlogic1001articledetails132448851spm10012014300155021_29">

     </a>
     1.侧重点不同
    </h6>
    <p>
     网络空间安全专业更加注重保护网络和网络空间的安全，包括网络防御、入侵检测与防御、渗透测试等。信息安全专业更加广泛，包括网络安全在内的各个方面，如应用程序安全、数据安全、物理安全等。
    </p>
    <h6>
     <a id="httpsblogcsdnnetweixin_57514792articledetails140434580httpsblogcsdnnetSpringJavaMyBatisarticledetails139456680spm1001201430015502httpsblogcsdnnetlogic1001articledetails132448851spm10012014300155022_33">

     </a>
     <a href="https://blog.csdn.net/logic1001/article/details/132448851?spm=1001.2014.3001.5502">
     </a>
     2.范围不同
    </h6>
    <p>
     网络空间安全专业更加专注于网络和网络空间的安全，而信息安全专业更加综合，涵盖了更广泛的安全领域。
    </p>
    <h6>
     <a id="httpsblogcsdnnetweixin_57514792articledetails140434580httpsblogcsdnnetSpringJavaMyBatisarticledetails139456680spm1001201430015502httpsblogcsdnnetlogic1001articledetails132448851spm10012014300155023_37">

     <a href="https://blog.csdn.net/weixin_57514792/article/details/140434580">

     </a>
     <a href="https://blog.csdn.net/logic1001/article/details/132448851?spm=1001.2014.3001.5502">
     </a>
     3.职业发展不同
    </h6>
    <p>
     网络空间安全专业更加偏向于网络安全工程师、渗透测试工程师、网络管理员等职业。
    </p>
    <p>
     信息安全专业的职业发展更加广泛，可以从事网络安全、应用程序安全、数据安全、安全顾问等多个领域的工作。
    </p>

    </p>
    <p>
     <strong>
      总的来说，网络空间安全专业和信息安全专业有许多相似之处，但也有一些不同之处。选择专业时，可以根据个人兴趣和职业发展方向来进行选择。
     </strong>

     学习计划
    </h3>
    <p>
     那么问题又来了，作为萌新小白，我应该先学什么，再学什么？
     <br/>
     既然你都问的这么直白了，我就告诉你，零基础应该从什么开始学起：
    </p>
    <h4>
     <a id="httpsblogcsdnnetm0_71744960articledetails131172765_77">
     </a>
     <a href="https://blog.csdn.net/m0_71744960/article/details/131172765">
     </a>
     阶段一：初级网络安全工程师
    </h4>
    <p>
     接下来我将给大家安排一个为期1个月的网络安全初级计划，当你学完后，你基本可以从事一份网络安全相关的工作，比如渗透测试、Web渗透、安全服务、安全分析等岗位；其中，如果你等保模块学的好，还可以从事等保工程师。
    </p>
    <p>
     <strong>
      综合薪资区间6k~15k
     </strong>
    </p>
    <p>
     1、网络安全理论知识（2天）
     <br/>
     ①了解行业相关背景，前景，确定发展方向。
     <br/>
     ②学习网络安全相关法律法规。
     <br/>
     ③网络安全运营的概念。
     <br/>
     ④等保简介、等保规定、流程和规范。（非常重要）
    </p>
    <p>
     2、渗透测试基础（1周）
     <br/>
     ①渗透测试的流程、分类、标准
     <br/>
     ②信息收集技术：主动/被动信息搜集、Nmap工具、Google Hacking
     <br/>
     ③漏洞扫描、漏洞利用、原理，利用方法、工具（MSF）、绕过IDS和反病毒侦察
     <br/>
     ④主机攻防演练：MS17-010、MS08-067、MS10-046、MS12-20等
    </p>
    <p>
     3、操作系统基础（1周）
     <br/>
     ①Windows系统常见功能和命令
     <br/>
     ②Kali Linux系统常见功能和命令
     <br/>
     ③操作系统安全（系统入侵排查/系统加固基础）
    </p>
    <p>
     4、计算机网络基础（1周）
     <br/>
     ①计算机网络基础、协议和架构
     <br/>
     ②网络通信原理、OSI模型、数据转发流程
     <br/>
     ③常见协议解析（HTTP、TCP/IP、ARP等）
     <br/>
     ④网络攻击技术与网络安全防御技术
     <br/>
     ⑤Web漏洞原理与防御：主动/被动攻击、DDOS攻击、CVE漏洞复现
    </p>
    <p>
     5、数据库基础操作（2天）
     <br/>
     ①数据库基础
     <br/>
     ②SQL语言基础
     <br/>
     ③数据库安全加固
    </p>
    <p>
     6、Web渗透（1周）
     <br/>
     ①HTML、CSS和JavaScript简介
     <br/>
     ②OWASP Top10
     <br/>
     ③Web漏洞扫描工具
     <br/>
     ④Web渗透工具：Nmap、BurpSuite、SQLMap、其他（菜刀、漏扫等）
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/9342a47116654b6fa263d98ddc1440ee.png#pic_center">
      <br/>
      <strong>
       那么，到此为止，已经耗时1个月左右。你已经成功成为了一名“脚本小子”。那么你还想接着往下探索吗？
      </strong>
     </img>
    </p>
    <h4>
     <a id="httpsblogcsdnnetm0_71744960articledetails131172765or_121">
     </a>
     <a href="https://blog.csdn.net/m0_71744960/article/details/131172765">
     </a>
     阶段二：中级or高级网络安全工程师（看自己能力）
    </h4>
    <p>
     <strong>
      综合薪资区间15k~30k
     </strong>
    </p>
    <p>
     7、脚本编程学习（4周）
     <br/>
     在网络安全领域。是否具备编程能力是“脚本小子”和真正网络安全工程师的本质区别。在实际的渗透测试过程中，面对复杂多变的网络环境，当常用工具不能满足实际需求的时候，往往需要对现有工具进行扩展，或者编写符合我们要求的工具、自动化脚本，这个时候就需要具备一定的编程能力。在分秒必争的CTF竞赛中，想要高效地使用自制的脚本工具来实现各种目的，更是需要拥有编程能力。
    </p>
    <p>
     零基础入门的同学，我建议选择脚本语言Python/PHP/Go/Java中的一种，对常用库进行编程学习
     <br/>
     搭建开发环境和选择IDE，PHP环境推荐Wamp和XAMPP，IDE强烈推荐Sublime；
    </p>
    <p>
     Python编程学习，学习内容包含：语法、正则、文件、 网络、多线程等常用库，推荐《Python核心编程》，没必要看完
    </p>
    <p>
     用Python编写漏洞的exp,然后写一个简单的网络爬虫
    </p>
    <p>
     PHP基本语法学习并书写一个简单的博客系统
    </p>
    <p>
     熟悉MVC架构，并试着学习一个PHP框架或者Python框架 (可选)
    </p>
    <p>
     了解Bootstrap的布局或者CSS。
    </p>
