    <p>
     音视频终端 SDK（腾讯云视立方）是腾讯云全新升级打造的音视频通信终端产品家族。集成了直播推流、视频播放、实时音视频、即时通信等音视频核心能力，为用户提供一体化 SDK，支持移动端、PC 端、Web 端、小程序端等多终端接入。根据时下火热的各类音视频场景，提供 SDK 快速集成方案，用户可根据业务需求定制化集成 SDK。
    </p>
    <h4 id=".E7.89.88.E6.9C.AC.E7.BB.93.E6.9E.84">
     音视频终端 SDK 产品家族介绍
    </h4>
    <p>
     音视频终端 SDK（腾讯云视立方） 产品家族分为基础 SDK 和含 UI 的低代码组件。
    </p>
    <p>
     基础 SDK
     <strong>
      仅提供终端基础能力
     </strong>
     ，包含了直播 SDK、短视频 SDK、播放器 SDK、全功能版 SDK 和腾讯特效 SDK 五个子产品 SDK。
    </p>
    <p>
     含 UI 的低代码组件
     <strong>
      提供包含后台服务及终端 SDK 一整套解决方案
     </strong>
     ，支持含 UI 资源快速集成，包括音视频通话 SDK 和多人音视频房间 SDK。
    </p>
