    <p>
     pycharm是一款强大的python集成开发环境，带有一整套python开发工具，今天就给大家介绍几款非常好用的插件。
    </p>
    <hr/>
    <p>
     <a href="https://mp.weixin.qq.com/s?__biz=Mzg3MDU0NjA0Mg==&amp;mid=2247483806&amp;idx=1&amp;sn=f888c0a69c8f9b773f531924862b3b1e&amp;chksm=ce8d6e8bf9fae79d5066633351e393ac53cf13f85821e4861536fe56f58be212977a8debb6ac#rd" rel="nofollow">

    <h3>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502_7">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     首先插件的下载方法
    </h3>
    <p>
     进入
     <strong>
      File -&gt; Settings -&gt; Plugins
     </strong>
     ，根据需要搜索插件名称（记得是在Marketplace中搜索），然后点击Install按钮，需要重启才能生效。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/313fb817380584f6e1d7d765d4d3c9b9.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Mongo_Plugin_14">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Mongo Plugin
    </h4>
    <p>
     MongoDB是一款非常流行的非关系型数据库，而Mongo Plugin是在Pycharm中对MongoDB进行可视化操作工具，通过它能可以很方便地对数据库进行可视化操作。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dbe01280981f91ba185d1e4bf55c58a3.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502IdeaVim_19">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     IdeaVim
    </h4>
    <p>
     linux中的vim大家该听过吧，而 IdeaVim 就可以让你在 Pycharm 中操作 Vim 的功能。
    </p>
    <p>
     在tools中勾选Vim Emulator即可使用，取消勾选回到正常编辑状态。关于vim的使用可以参考这篇文章
     <a href="http://mp.weixin.qq.com/s?__biz=MzI5NzIyMjQwNA==&amp;mid=2247484618&amp;idx=1&amp;sn=7249543c10550d1ebc2b00f2b766aa77&amp;chksm=ecb92ca1dbcea5b789b66cb6c024ad6ef131a1fc0c8d6dcdd26b2954cf37d457196eb4d6fb4d&amp;scene=21#wechat_redirect" rel="nofollow">
      Linux之vim快速入门
     </a>
     。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/96d9821b195d15d407afecf1b5d7b522.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Statistic_27">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Statistic
    </h4>
    <p>
     statistic项目统计插件，统计整体代码量，包括所有文件的统计数量和行数。点击左下角的
     <strong>
      statistic &gt;&gt;&gt;&gt;refresh
     </strong>
     即可。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/dab3a01f7e70f550ab4ab7c72feba665.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Json_Parser_33">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Json Parser
    </h4>
    <p>
     JSON Parser是一个用于验证和格式化JSON字符串的轻量级插件。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/44a1a6b339520132b0e46cc093dfee29.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502activatepowermode_39">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     activate-power-mode
    </h4>
    <p>
     炫酷的编码效果，简直停不下来，不过眼睛很容易疲劳，谨慎使用！
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/13b4abe8f0303a5dbc794cf659ad865d.gif"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Translation_45">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Translation
    </h4>
    <p>
     一款用于翻译的插件，包括谷歌翻译、有道翻译，直接在 Pycharm 中选择需要翻译的内容，右键点击翻译Translate进行翻译。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ed41f998b16f0bddf79fccefe1bdbf1d.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Rainbow_Brackets_51">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Rainbow Brackets
    </h4>
    <p>
     Rainbow Brackets可以让代码块之间清晰的显示出各种颜色，比如括号相同颜色，选中区域代码高亮的功能等，并且支持支持Java, Python, JavaScript,Go, PHP等多种编程语言。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/fdcd3f8e68f9d35e767b00f04b89d359.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Background_Image_Plus_57">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Background Image Plus
    </h4>
    <p>
     该插件是用来设置IDEA和PyCharm的背景图片，安装好后，点击
     <strong>
      View–&gt;Set Background Image
     </strong>
     选择自己喜欢的照片即可，还可以调节其透明度。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/19d0a1fb2441784ec0674f703a16951f.png"/>
    </p>
    <h4>
     <a id="httpsblogcsdnnethugo233articledetails129014587spm1001201430015502Material_Theme_UI_63">
     </a>
     <a href="https://blog.csdn.net/hugo233/article/details/129014587?spm=1001.2014.3001.5502">
     </a>
     Material Theme UI
    </h4>
    <p>
     Material Theme UI是一个主题插件，在很多编辑器中都可以使用，暗黑色，还是很有质感的，大家不喜欢默认主题的，可以尝试下。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f6030278878d3ed40d768c839be6c884.png"/>
     上面就是几个比较常用的idea插件，大家感兴趣的话，可以根据需要下载使用。
    </p>
