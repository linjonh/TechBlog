    <h2>
     <a id="2_6">
     </a>
     2、下载
    </h2>
    <p>
     到kafka官网下载你要学习的版本，新版本由于编程风格以及commit的人的风格不一。最好是学习后几个版本。下载好，放入指定位置解压
     <br/>
     官网：
     <strong>
      http://kafka.apache.org/
     </strong>
     <br/>
     下载地址：
     <strong>
      http://kafka.apache.org/downloads
     </strong>
     <br/>
     <code>
      需要补充的是，由于Kafka是gradle 构建的。所以，事先需要下载gradle。安装和下载我将不细说。就和maven一样。我下载的是3.1版本 bin.zip
     </code>
     <br/>
     gradle下载地址：
     <strong>
      https://services.gradle.org/distributions/
     </strong>
     <br/>
     下载到指定文件夹解压，设置环境变量，可以使用gradle -version 验证是否成功
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e8d096423f1ed96520e4bbfda9cb3302.png#pic_center"/>
    </p>
    <h2>
     <a id="3idea_16">
     </a>
     3、导入idea
    </h2>
    <p>
     1、打开idea ,
     <br/>
     2、点击open,
     <br/>
     3、选择build.gradle,
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/255b724e44434df8f80e47dc0d601f05.png#pic_center">
      <br/>
      4、
      <strong>
       如果没有设置gradle，他会自动提示让你设置gradle的home地址。
      </strong>
      <br/>
      5、
      <strong>
       选择 作为一个项目
      </strong>
      <br/>
      然后就是等待构建成功，需要我标记的这样才OK。
      <br/>
