    <p>
     <strong>
      <span style="font-size:24px">
       1.
       <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
        HTML5方案
       </span>
      </span>
     </strong>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
      <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
       <span style="font-size:18px">
        当前国内市场，有一些号称移动中间件的产品，但基本都采用HTML5方案。在技术实现上一般都是采用国外开源的PhoneGap项目进行了封装
        <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
         。HTML5标准在不同移动设备上的差异比较大，适配这些设备需要额外的工作。
        </span>
       </span>
      </span>
      <br/>
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
      <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
       <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px; background-color:rgb(255,255,255)">
        <span style="font-size:24px">
         <strong>
          2.
          <span style="color:rgb(51,51,51); font-family:宋体; line-height:24px">
           使用Web方式开发原生应用(基于原生控件而非HTML)
          </span>
         </strong>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
      <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
       <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px; background-color:rgb(255,255,255)">
        <span style="color:rgb(51,51,51); font-family:宋体; line-height:24px">
         <span style="font-size:18px">
          比较典型的有：
         </span>
         <span style="font-size:13px; color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
          <strong>
           <a href="http://www.imagapp.com/" rel="nofollow noopener noreferrer" target="_blank">
            iMAG
           </a>
          </strong>
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
      <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
       <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
        <span style="color:rgb(51,51,51); font-family:宋体; line-height:24px; background-color:rgb(255,255,255)">
         <span style="font-size:18px">
          <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
           很好地解决了兼容性问题，使应用在不同移动设备上具有同样的表现。
          </span>
          现有web开发人员可以转换成移动应用开发者。
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
      <span style="color:rgb(51,51,51); font-family:Helvetica,Tahoma,Arial,sans-serif; line-height:24px">
       <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
        <span style="color:rgb(51,51,51); font-family:宋体; line-height:24px; background-color:rgb(255,255,255)">
         <span style="font-size:18px">
          <span style="color:rgb(51,51,51); font-family:'Microsoft Yahei',Consolas,'Bitstream Vera Sans Mono','Courier New',Courier,monospace; line-height:19.5px">
           因为iMAG引擎底层封装的是原生UI控件，因此具有和原生应用相同的性能和用户体验。单就性能和用户体验这点是PhoneGap等基于HTML5的框架无法比拟的。
          </span>
          <br/>
         </span>
        </span>
