    <p>
     在 Windows 环境中，启动项目时如出现以下提示：
    </p>
    <pre><code>''NODE_ENV' 不是内部或外部命令，也不是可运行的程序或批处理文件。
</code></pre>
    <p>
     这是因为 Windows 系统不支持
     <code>
      NODE_ENV=dev
     </code>
     ,需要用
     <code>
      set
     </code>
     或者
     <code>
      cross-env
     </code>
     .
    </p>
    <p>
     解决方法如下：
    </p>
    <ol>
     <li>
      安装
      <code>
       cross-env
      </code>
      依赖：
      <code>
       npm install --save-dev cross-env
      </code>
     </li>
     <li>
      在
      <code>
       package.json
      </code>
      中的
      <code>
       'NODE_ENV=development ts-node'
      </code>
      前添加
      <code>
       cross-env
      </code>
     </li>
    </ol>
    <pre><code class="prism language-json"><span class="token string-property property">"scripts"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
