    <h3>
     <a id="_15">
     </a>
     一.爱你就像爱生命
    </h3>
    <p>
     爱你就像爱生命：
     <br/>
     我来这个世界，
     <br/>
     不是为了繁衍后代。
     <br/>
     而是来看花怎么开，水怎么流。
     <br/>
     太阳怎么升起，夕阳何时落下。
     <br/>
     我活在世上，
     <br/>
     无非想要明白些道理，
     <br/>
     遇见些有趣的事。
     <br/>
     生命是一场偶然，
     <br/>
     我在其中寻找因果。
    </p>
    <h3>
     <a id="_28">
     </a>
     二.枕上诗书
    </h3>
    <p>
     很久以前我以为，
     <br/>
     所爱隔山海，山海不可平。
     <br/>
     后来我知道了，
     <br/>
     海有舟可渡，山有路可行。
     <br/>
     现在我才发现，
     <br/>
     山海皆可平，难平是人心。
    </p>
    <h3>
     <a id="_37">
     </a>
     三.被讨厌的勇气
    </h3>
    <p>
     纵使被说坏话、被讨厌也没什么好在意的，
     <br/>
     因为那是对方的课题，擅自背负别人的课题
     <br/>
     只会让自己感到痛苦。
     <br/>
     所以
     <br/>
     别想太多，好好生活。
     <br/>
     日子过着过着就会有答案，
     <br/>
     努力走着走着就会有着落。
    </p>
    <h3>
     <a id="_47">
     </a>
     四.最高级的惩罚就是沉默
    </h3>
    <p>
     最高级的惩罚就是沉默，
     <br/>
     最矜持的报复就是无视少在烂事上纠缠，
     <br/>
     少为不值得的人生气，
     <br/>
     让自己活的高贵一点。
     <br/>
     学会放下，才能更好的前行。
     <br/>
     这个世界看似纷繁复杂，
     <br/>
     其实本质，
     <br/>
     却还是你一人的世界。
    </p>
    <h3>
     <a id="_58">
     </a>
     五.教育小孩
    </h3>
    <p>
     台湾作家龙应台在孩子教育上说过一句话：
     <br/>
     孩子，我要求你读书用功，
     <br/>
     不是因为我要你跟别人比成绩，
     <br/>
     而是因为，我希望你将来会拥有选择的权利，
     <br/>
     选择有意义、有时间的工作，
     <br/>
     而不是被迫谋生。
     <br/>
     当你的工作在你心中有意义，你就有成就感。
     <br/>
     当你的工作给你时间，不剥夺你的生活，你就有尊严。
     <br/>
     成就感和尊严，给你快乐。
    </p>
    <h3>
     <a id="_70">
     </a>
     六.天下事有难易乎
    </h3>
    <p>
     天下事有难易乎？
     <br/>
     为之，则难者亦易矣；
     <br/>
     不为，则易者亦难矣。
     <br/>
     人之为学有难易乎？
     <br/>
     学之，则难者亦易矣；
     <br/>
     不学，则易者亦难矣。
    </p>
