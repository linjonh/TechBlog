    <h3>
     <a id="CHMJava_Doc_1">
     </a>
     为什么之前使用CHM格式的Java Doc
    </h3>
    <p>
     以前读的
     <code>
      Java doc
     </code>
     文档都是
     <code>
      CHM
     </code>
     格式的,主要是
     <code>
      CHM
     </code>
     格式中有
     <code>
      索引
     </code>
     功能,通过索引可以快速查找需要的类,在线的
     <code>
      API
     </code>
     没有这个功能,还要打开浏览器根据包名查找,包名我不一定记得住,所以在线的不方便.
     <br/>
     所以我都是用的
     <code>
      CHM
     </code>
     格式的
     <code>
      java doc
     </code>
     文档,但是奈何英文不好,阅读起来比较繁琐.
    </p>
    <h3>
     <a id="Java_doc_4">
     </a>
     为什么现在推荐使用在线的Java doc
    </h3>
    <p>
     不过,最近我无意中发现,
     <code>
      java9
     </code>
     之后
     <code>
      Oracle
     </code>
     提供的
     <a href="https://docs.oracle.com/javase/9/docs/api/overview-summary.html" rel="nofollow">
      在线的
     </a>
     <code>
      java doc
     </code>
     文档提供了
     <code>
      搜索
     </code>
     功能,这样使用在线的
     <code>
      java doc
     </code>
     就很完美了,因为:
    </p>
    <ul>
     <li>
      你可以快速搜索到想要的类
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/208a3d60fe3603c72cb99447cf2a0811.png">
       - 然后就
       <code>
        可以使用浏览器的翻译插件进行翻译
       </code>
       .
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f2f71626d1b09311b655e7977ded5872.png"/>
      </img>
     </li>
    </ul>
    <h3>
     <a id="java9java_doc_9">
     </a>
     java9之前版本的在线的java doc中如何快速查找类
    </h3>
    <p>
     先在
     <code>
      java9
     </code>
     或者高级版本的
     <code>
      java doc
     </code>
     中查找到想要的类,然后修改浏览器地址栏上的版本号即可:
     <br/>
     例如:先在
     <code>
      java9
     </code>
     的
     <code>
      doc
     </code>
     中找到
     <code>
      System
     </code>
     类,浏览器地址栏中的地址如下:
    </p>
    <pre><code>https://docs.oracle.com/javase/9/docs/api/java/lang/System.html
</code></pre>
    <p>
     如果此时想看java8中的System类,这只需要把地址栏中的
     <code>
      9
     </code>
     改成
     <code>
      8
     </code>
     即可:
    </p>
    <pre><code>https://docs.oracle.com/javase/8/docs/api/java/lang/System.html
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/47d4d7a670099b0351eb389067565751.png"/>
    </p>
    <h3>
     <a id="_20">
     </a>
     总结
    </h3>
    <ol>
     <li>
      使用
      <code>
       java9
      </code>
      或者更高版本的在线
      <code>
       java doc
      </code>
      的搜索功能
     </li>
     <li>
      使用浏览器翻译插件翻译
     </li>
    </ol>
