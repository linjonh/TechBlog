     <br/>
     ESP32环境搭建(arduino)
     <br/>
     <a href="https://blog.csdn.net/qq_35174914/article/details/79328043">
      https://blog.csdn.net/qq_35174914/article/details/79328043
     </a>
     <br/>
     1 、首先安装 arduino IDE 1.8.5
     <br/>
     下载地址
     <br/>
     <a href="https://www.arduino.cc/en/Main/Software?setlang=en" rel="nofollow">
      https://www.arduino.cc/en/Main/Software?setlang=en
     </a>
     <br/>
     安装路径一般不装在C盘：D:\it\arduino
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e7989f5c537e656ab1bee0fc3783d6ca.png"/>
    </p>
    <p>
     2 、下载arduino的ESP32开发工具包
     <br/>
     进入espressif的仓库https://github.com/espressif
     <br/>
     选择arduino-esp32
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6a567004db1357ee8714d7327c773170.png">
      <br/>
      在github中选择Download ZIP
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0a386c719faa5941aeb3700f51fe419a.png">
       <br/>
       下载的文件为： arduino-esp32-master.zip 大小为30.5M字节
       <br/>
       3、安装arduino的ESP32开发工具包
       <br/>
       进入D:\it\arduino\hardware文件夹，
       <br/>
       新建一个名叫espressif的空白文件夹
       <br/>
       然后在espressif文件夹下新建一个名叫esp32的空白文件夹
       <br/>
       将下载下来的压缩包解压，内容复制到D:\it\arduino\hardware\espressif\esp32
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/968c7bea5dfd200f3fe07c146df7e31b.png"/>
      </img>
     </img>
    </p>
    <p>
     4、运行get.exe 程序
     <br/>
     需要运行tools文件夹下面的get.exe运行。该程序是基于Python的，确保电脑已经安装了Python。
     <br/>
     双击运行
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c0f5ee71666484f9074ca84a1335addd.png"/>
    </p>
    <p>
     会弹出cmd窗口直至程序下载完成。
     <br/>
     5、运行arduino IDE
     <br/>
     a.选择开发板
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/713d13420e81b7de86530400eed4edcc.png"/>
    </p>
    <p>
     b.选择串口
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/172d08af5024d3ef6333bb6cc69baa1f.png">
      <br/>
      C.编译下载
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/89e9a4fdf09e1fa682edd0d4417f4122.png"/>
     </img>
    </p>
