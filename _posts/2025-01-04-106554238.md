    <article id="main-content" style="font-size: 16px;">
     <p>
      <a href="http://www.java2html.com/" rel="nofollow noopener noreferrer" target="_blank">
       Java2HTML
      </a>
      是将Java源代码转换为彩色和可浏览HTML页面的工具，其步骤非常简单。
     </p>
     <h3>
      转换步骤
     </h3>
     <p>
      1）下载Java2HTML –
      <a href="http://www.java2html.com/download.html" rel="nofollow noopener noreferrer" target="_blank">
       http://www.java2html.com/download.html
      </a>
     </p>
     <p>
      2）解压缩，例如“ D：\ Java2HTML”
     </p>
     <p>
      3）编辑“ j2h.bat”文件并修改CLASSPATH变量。 确保它指向正确的j2h.jar库。
     </p>
     <pre class="has"><code class="language-bash">@echo off
REM Change this Script, to suit your needs

REM The next few line combines the command line args into one
set \_j2h_parameter=
:GetParameter
if "%1" == "" goto end
set \_j2h_parameter=%\_j2h_parameter% "%1"
shift
goto GetParameter
:end

REM runs Java2HTML
set MBACKUP=%CLASSPATH%
set CLASSPATH=D:\Java2HTML\j2h.jar;%CLASSPATH%
java j2h %\_j2h_parameter%
set CLASSPATH=%MBACKUP%</code></pre>
<p>
4）将“ j2h”设置为 Windows 环境变量。 （如果需要全局访问该命令）
</p>
<p>
5）发出以下命令
</p>
<pre class="has"><code class="language-bash">C:\&gt;j2h -js D:\Java2HTML\source -d D:\Java2HTML\output</code></pre>
<p>
<em>
D：\ Java2HTML \ source 是要转换的 Java 源代码
</em>
<em>
<br/>
</em>
<em>
D：\ Java2HTML \ output 是生成的 HTML 输出文件夹
</em>
</p>
