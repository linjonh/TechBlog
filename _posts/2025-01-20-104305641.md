     <h3>
      一、重温经典
     </h3>
     <h4>
      1.1 植物大战僵尸
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/86e554fbf99daca07014338e61a94c78.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/marblexu/PythonPlantsVsZombies
      </p>
     </blockquote>
     <p>
      PythonPlantsVsZombies 是 Python 语言编写的植物大战僵尸。
     </p>
     <ul>
      <li>
       <p>
        编程语言：Python
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          PythonPlantsVsZombies 支持的植物类型：太阳花、豌豆射手、寒冰射手、坚果、樱桃炸弹、双重豌豆射手等；
         </p>
        </li>
        <li>
         <p>
          PythonPlantsVsZombies 支持的僵尸类型：普通僵尸、棋子僵尸、路障僵尸、铁桶僵尸、毒爆僵尸；
         </p>
        </li>
        <li>
         <p>
          项目使用 Json 文件保存关卡信息，可独立设置僵尸出现的时间和位置；
         </p>
        </li>
        <li>
         <p>
          支持关卡开始时选择上场植物；
         </p>
        </li>
        <li>
         <p>
          支持夜间模式；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：风靡一时的植物大战僵尸关卡多、组合搭配多样，拿它来学习 Python 练手是一个不错的选择。
       </p>
      </li>
     </ul>
     <h4>
      1.2 地下城文字游戏
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/d638f5b3383daa7438affd305a8a9b93.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/AIDungeon/AIDungeon
      </p>
     </blockquote>
     <p>
      AIDungeon2 是一个基于机器学习的地下城文字游戏。
     </p>
     <ul>
      <li>
       <p>
        编程语言：Python
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          AI Dungeon 基于 OpenAI 打造，采用了最先进的开放源码文本生成功能，辅以自选冒险类特性，向玩家呈现了一段段独特的文字。在 AI Dungeon 中，玩家可以进行各式行为，且无论你的举动多么奇怪都能得到回应；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：此项目介绍了如何使用机器学习构建一个游戏，代码简单清晰适合 AI 爱好者深入研究哟~
       </p>
      </li>
     </ul>
     <h4>
      1.3 斗地主
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/7e2232dcdc7f20c9095893c92d7f6d02.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/ainilili/ratel
      </p>
     </blockquote>
     <p>
      Ratel 是一个命令行可联网的斗地主游戏。
     </p>
     <ul>
      <li>
       <p>
        编程语言：Java
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          Ratel 分客户端和服务端，你可以让小伙伴们的客户端都连接你的服务器进行游戏，也可以直接连接作者的公网服务器进行游戏；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：Ratel 使用 Java 语言开发，Netty 4.x 网络框架搭配 protobuf 数据协议，此外还可以学到 Server-Client、Client-Client 等交互流程。
       </p>
      </li>
     </ul>
     <h3>
      二、重温童年
     </h3>
     <h4>
      2.1 飞船射击
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/89257006ea40b22d1131624587e2bc92.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/gd4Ark/star-battle
      </p>
     </blockquote>
     <p>
      Star Battle 是一个 JavaScript ES6、Canvas 开发的飞船射击类游戏。
     </p>
     <ul>
      <li>
       <p>
        编程语言：JavaScript
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          使用 W、A、S、D 控制飞船，按下 Space 发射；
         </p>
        </li>
        <li>
         <p>
          燃料初始值为 15，每秒递减 1 点，当燃料值为 0 时，游戏结束；
         </p>
        </li>
        <li>
         <p>
          触碰掉下的燃料瓶可增加 15 点，最大值为 30 点；
         </p>
        </li>
        <li>
         <p>
          击中敌方飞船增加 5 分。行星需击中两次，增加 10 分。击中友方扣除 10 分；
         </p>
        </li>
        <li>
         <p>
          撞击敌方损失 15 点燃料，撞击友方扣除 10 分；
         </p>
        </li>
        <li>
         <p>
          游戏允许负分；
         </p>
        </li>
        <li>
         <p>
          按下 P 暂停游戏，按下 M 静音；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：作为单机游戏经典款太空大战，本项目除了能重温简单粗暴地击落得分之外，还可以学到 Canvas 实践技术，一举两得。
       </p>
      </li>
     </ul>
     <h4>
      2.2 俄罗斯方块
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/db6d43188386f1e9cdc8a392c39919b4.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/gd4Ark/star-battle
      </p>
     </blockquote>
     <p>
      Tinytetris 是一个用 C++ 编写的终端版俄罗斯方块游戏。
     </p>
     <ul>
      <li>
       <p>
        编程语言：C++
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          通过键位 A（左）、D（右）、W（旋转）、S（放下）和 Q（退出）来控制方块的方向和掉落；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：Tinytetris 提供了两个版本的源码，分为注释版和库版，注释较多易于理解和学习。
       </p>
      </li>
     </ul>
     <h4>
      2.3 扫雷
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/63e1f2a5031fb855bb14c55a86fc69e7.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/muan/emoji-minesweeper
      </p>
     </blockquote>
     <p>
      Emoji-minesweeper 是一款 Emoji 符号的扫雷游戏。代码很简短，游戏创意很酷。
     </p>
     <ul>
      <li>
       <p>
        编程语言：JavaScript
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          左键单击开启一个地点；
         </p>
        </li>
        <li>
         <p>
          右键单击以将一个点标记为炸弹；
         </p>
        </li>
        <li>
         <p>
          双击可打开目标附近的所有 8 个点（使用右键单击已标记为炸弹的点除外）；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：寥寥 300+ 行代码实现该游戏，还能学到 Javascript 处理 Emoji 的方法，你不来试试嘛~
       </p>
      </li>
     </ul>
     <h3>
      三、全家桶·模拟器
     </h3>
     <h4>
      3.1 Mac 上的红白机
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/78c54f289a3846974bbaab9d6fb2a72e.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/OpenEmu/OpenEmu
      </p>
     </blockquote>
     <p>
      OpenEmu 是一个可以玩各种复古游戏的游戏机。
     </p>
     <ul>
      <li>
       <p>
        编程语言：Objective-C
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          支持任天堂（NES）、索尼PSP、世嘉 32X 等 30 多种游戏引擎；
         </p>
        </li>
        <li>
         <p>
          支持外接游戏手柄、投屏等操作；
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：有了它，我们就可以在 Mac 系统上玩各种经典游戏，只需将下载到的游戏 ROM 拖到 NES 即可开心地玩耍 30+ 款掌机游戏。
       </p>
      </li>
     </ul>
     <h4>
      3.2 远程游戏
     </h4>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/0bca47d5247f12f88b2e2cd4eb9f59e2.png"/>
     </p>
     <blockquote>
      <p>
       GitHub 地址→https://github.com/HFO4/gameboy.live
      </p>
     </blockquote>
     <p>
      Gameboy.Live 是一个 Gameboy 模拟器，可以通过 Socket 远程玩像素游戏。
     </p>
     <ul>
      <li>
       <p>
        编程语言：Golang
       </p>
      </li>
      <li>
       <p>
        功能介绍
       </p>
      </li>
      <li>
       <ul>
        <li>
         <p>
          CPU指令仿真
         </p>
        </li>
        <li>
         <p>
          计时器和中断
         </p>
        </li>
        <li>
         <p>
          支持仅 ROM、MBC1、MBC2 磁带
         </p>
        </li>
        <li>
         <p>
          声音模拟
         </p>
        </li>
        <li>
         <p>
          图形仿真
         </p>
        </li>
        <li>
         <p>
          云游戏
         </p>
        </li>
        <li>
         <p>
          ROM 调试器
         </p>
        </li>
        <li>
         <p>
          游戏保存和恢复卡带级别
         </p>
        </li>
       </ul>
      </li>
      <li>
       <p>
        推荐理由：Gameboy.Live 是为学习目的而编写的 Gameboy 模拟器，所以通过这个项目你可以快速 Pick Golang 技术。
       </p>
      </li>
     </ul>
