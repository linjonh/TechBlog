     当人工智能遇上区块链
    </h1>

      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <strong>
        <span style="color:rgb(0,128,255);">
         导读：
        </span>
       </strong>
       <span style="color:rgb(0,128,255);">
        区块链可以解决AI应用中数据可信度的问题，让AI聚焦于算法；AI可以植入区块链智能合约中，让区块链智能合约更智能。
       </span>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       通信网络的不断升级，计算能力的持续提升，促使人工智能技术新一轮兴起。近年来，随着越来越多的力量进入这一市场，人工智能技术发展明显提速，包括
       <span style="color:rgb(0,128,255);">
        语音助手
       </span>
       、
       <span style="color:rgb(0,128,255);">
        同传翻译
       </span>
       、
       <span style="color:rgb(0,128,255);">
        无人驾驶
       </span>
       等在内的各式应用如雨后春笋般涌现。更为重要的是，人工智能正在加速与实体经济融合，助力更多的行业转型升级，
       <span style="color:rgb(0,128,255);">
        智慧交通
       </span>
       、
       <span style="color:rgb(0,128,255);">
        智慧医疗
       </span>
       、
       <span style="color:rgb(0,128,255);">
        智慧公安
       </span>
       、
       <span style="color:rgb(0,128,255);">
        智慧农业
       </span>
       ……这些不仅成了耳熟能详的名词，更为我们的生产和生活带来便利。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       人工智能（AI）、区块链无疑是当下最为热门的技术。随着两大技术的发展，越来越多的人开始将两者相提并论，探讨AI与区块链融合发展的可能性。那么，当AI遇上区块链，这个世界将发生哪些改变？
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       在日前举办的博鳌亚洲论坛上，AI同传让人们看到了人工智能技术的强大以及正在给我们生活带来的改变。同样，比特币等虚拟货币的兴起，也让人们认识了区块链这一技术，并且畅想其可能给金融、物流等多个行业带来的变革。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       正是因为人工智能、区块链在生产和生活中越来越多地应用与落地，使得人们开始探索人工智能和区块链的协同甚至融合发展。“区块链和人工智能实现共存是最有价值的。”中科院外籍院士张首晟日前在接受媒体采访时抛出了这一观点，他认为，
       <span style="color:rgb(0,128,255);">
        区块链可以让数据市场变得更加公平
       </span>
       。现代社会最大的不公平是人们容易歧视一些少数派。但在机器学习的过程中最需要的就是这些少数派拥有的数据。如果要将机器学习的精准率从90%提高到99%，它需要的不是已经学过的数据，而是和以前不一样的数据。因而，区块链与人工智能共生将创造无限可能。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       从技术本身看，区块链和人工智能是两种截然不同的技术，两者在各自的领域应用和落地。不过，两者也有一定的关联，
       <span style="color:rgb(0,128,255);">
        区块链是新型的分布式数据库技术，因而在“数据”上，区块链和人工智能有“合作”的空间
       </span>
       。在中科聚信公司首席技术官贺光忠看来，
       <span style="color:rgb(255,41,65);">
        区块链技术可以解决AI应用中数据可信度问题，有了区块链技术，AI可以更加聚焦于算法
       </span>
       。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       人工智能应用有三大要点。贺光忠表示，一是
       <span style="color:rgb(0,128,255);">
        数据
       </span>
       ，二是算法，三是
       <span style="color:rgb(0,128,255);">
        计算能力
       </span>
       。在数据领域，AI可以与区块链技术结合，一方面是从
       <span style="color:rgb(0,128,255);">
        应用层
       </span>
       面入手，两者各司其职，
       <span style="color:rgb(0,128,255);">
        AI负责自动化的业务处理和智能化的决策，区块链负责在数据层提供可信数据
       </span>
       ；另一方面是
       <span style="color:rgb(0,128,255);">
        数据层
       </span>
       ，两者可以
       <span style="color:rgb(0,128,255);">
        互相渗透
       </span>
       。区块链中的智能合约实际上也是一段实现某种
       <span style="color:rgb(0,128,255);">
        算法
       </span>
       的代码，既然是算法，那么
       <span style="color:rgb(0,128,255);">
        AI就能够植入其中，使区块链智能合约更加智能
       </span>
       。同时，将AI引擎训练模型结果和运行模型存放在区块链上，就能够确保模型不被篡改，降低了AI应用遭受攻击的风险。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       从上述观点中能够看到，人工智能和区块链有望基于双方各自的优势实现互补。事实上，目前业界已经有公司尝试将两者同时应用。区块链初创公司Everledger就正在探索将区块链与人工智能、物联网等技术结合起来，打造一站式的珠宝追踪及鉴定平台，在利用区块链技术实现珠宝流通记录真实、可溯源的基础上，通过人工智能实现追踪的自动化。
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       <br/>
      </p>
      <p style="line-height:1.75em;white-space:normal;text-align:justify;color:#7F7F7F;font-family:'微软雅黑';font-size:14px;margin-bottom:10px;">
       不过，需要看到的是，今天的区块链和人工智能还刚刚起步，如果将其比作一个人，那么他们还是嗷嗷待哺的婴儿，还不具备自立的能力。因此，对于区块链和人工智能而言，当前最重要的使命依然是完善和成长。未来，只有两大技术都相对成熟，并且有一定规模的应用落地，两者的协同与结合才更有价值和空间。
      </p>
