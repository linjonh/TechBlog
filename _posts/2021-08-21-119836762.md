    <p>
     制作docker是，需要下载python3.8。
    </p>
    <ol>
     <li>
      更换国内源
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">vim</span> /etc/apt/sources.list
</code></pre>
    <p>
     注释之前的，然后添加：
    </p>
    <pre><code class="prism language-bash">deb http://mirrors.aliyun.com/ubuntu/ xenial main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ xenial-security main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ xenial-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ xenial-backports main restricted universe multiverse
<span class="token comment">##测试版源</span>
deb http://mirrors.aliyun.com/ubuntu/ xenial-proposed main restricted universe multiverse
<span class="token comment"># 源码</span>
deb-src http://mirrors.aliyun.com/ubuntu/ xenial main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ xenial-security main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ xenial-updates main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ xenial-backports main restricted universe multiverse
<span class="token comment">##测试版源</span>
deb-src http://mirrors.aliyun.com/ubuntu/ xenial-proposed main restricted universe multiverse
<span class="token comment"># Canonical 合作伙伴和附加</span>
deb http://archive.canonical.com/ubuntu/ xenial partner

</code></pre>
    <p>
     更新：
    </p>
    <pre><code class="prism language-bash"><span class="token function">apt-get</span> update
</code></pre>
    <ol start="2">
     <li>
      安装依赖包
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">apt-get</span> <span class="token function">install</span> software-properties-common
</code></pre>
    <ol start="3">
     <li>
      添加 deadsnakes PPA 源
     </li>
    </ol>
    <pre><code class="prism language-bash">add-apt-repository ppa:deadsnakes/ppa
</code></pre>
    <p>
     4.下载python3.8
    </p>
    <pre><code class="prism language-bash"> <span class="token function">apt-get</span> update
 <span class="token function">apt-get</span> <span class="token function">install</span> python3.8
</code></pre>
    <p>
     5.建立软链接
     <br/>
     先删除旧的python连接。
     <br/>
     删除后建立新的连接关系，例如linux下python3默认在/usr/bin/下
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">rm</span> <span class="token parameter variable">-rf</span> /usr/bin/python
<span class="token function">ln</span> <span class="token parameter variable">-s</span> /usr/bin/python3.8 /usr/bin/python
</code></pre>
    <p>
     6.输入pip会找不到pip模块，这时候需要重新下载pip
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/96fb17f627e10d47a7ea389b279a0ea2.png#pic_center"/>
    </p>
    <p>
     进入python3.8的lib中：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /usr/lib/python3.8
<span class="token function">wget</span> https://bootstrap.pypa.io/get-pip.py  --no-check-certificate
python get-pip.py
</code></pre>
    <p>
     然后，安装成功！！
    </p>
