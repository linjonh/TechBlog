    <h2>
     <a id="IDEA___0">
     </a>
     IDEA 连接 数据库
    </h2>
    <h3>
     <a id="_1">
     </a>
     一、首先确保数据库服务是打开的
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f6bae3f032e27fd077feadaf213124f0.png#pic_center"/>
    </p>
    <ul>
     <li>
      使用
      <strong>
       mysql -u root -p
      </strong>
      连接数据库服务器。
     </li>
     <li>
      若不能进入到 mysql 里面则说明 没有启动服务器，使用
      <strong>
       net start mysql
      </strong>
      命令启动。
     </li>
     <li>
      如果 net start mysql 也无法启动，就需要 安置 MySQL：
      <ul>
       <li>
        <mark>
         以管理员身份运行 cmd，转到 mysql 的 bin 目录下
        </mark>
        ，使用命令：
        <strong>
         mysqld -install
        </strong>
        安置 MySQL，然后再次连接数据库。
       </li>
      </ul>
     </li>
     <li>
      直到出现
      <mark>
       mysql&gt;
      </mark>
      标志，才说明成功打开服务器。
     </li>
    </ul>
    <h3>
     <a id="_idea___mysql_9">
     </a>
     二、在 idea 中 连接 mysql
    </h3>
    <h4>
     <a id="1_idea__MySQL__10">
     </a>
     1、找到 idea 中 MySQL 数据库的连接方式。
    </h4>
    <p>
     第一步，找到 Database
     <br/>
     第二步，点击"+"
     <br/>
     第三步，点击 Data Source
     <br/>
     第四步，点击 MySQL
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ba349cb590848b889e9b9d3cd0be5ad1.png#pic_center"/>
    </p>
    <h4>
     <a id="2_16">
     </a>
     2、配置连接。
    </h4>
    <p>
     在
     <strong>
      第一次
     </strong>
     使用 的时候，除了要配置连接，还要配置相应的驱动，否则连接的时候会报错！！！
     <br/>
     图中的①②③④⑤分别表示为：
     <br/>
     ①、连接的名称
     <br/>
     ②、连接的数据库地址和端口号（连接本地的数据库就默认为localhost，端口号为3306）
     <br/>
     ③、MySQL数据库的用户名和密码
     <br/>
     ④、需要连接的数据库名称
     <br/>
     ⑤、配置完成之后，点击测试连接，查看是否连接成功。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ab863e9c2f6cf39fc6614d6359c4a21d.png#pic_center"/>

     回到 Date Sources，出现
