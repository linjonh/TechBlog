    <h2>

     </a>
     VS+QT开发环境搭建
    </h2>
    <h2>
     <a id="1_1">
     </a>
     1、工具准备
    </h2>
    <p>
     VisualStudio最新版直接
     <a href="https://visualstudio.microsoft.com/zh-hans/downloads/" rel="nofollow">
      官网
     </a>
     下载，根据需要进行下载，我下载的免费社区版本。日常开发完全够用。
     <br/>
     <img alt="免费社区版本" src="https://i-blog.csdnimg.cn/blog_migrate/a3187fa1db761b644fe96bd89d356da5.png">
      <br/>
      QT建议不要下最新版本，同样，为了减少病毒的风险，直接
      <a href="https://download.qt.io/archive/qt/" rel="nofollow">
       官网
      </a>
      下载。我下载的5.12版本
      <br/>
      <img alt="QT下载页" src="https://i-blog.csdnimg.cn/blog_migrate/b2c34ca7445628c67e1ab95914a2fea6.png">
       <br/>
       进入后，下载与电脑相对应的版本
       <br/>
       <img alt="windows下载的QT版本" src="https://i-blog.csdnimg.cn/blog_migrate/83c1a21f47d2de8614a7cb8fc53b5929.png">
        <br/>
        接下来就是等待下载时间
       </img>
      </img>
     </img>
    </p>
    <h2>
     <a id="2_9">
     </a>
     2、工具安装
    </h2>
    <h3>
     <a id="VS_10">
     </a>
     VS安装
    </h3>
    <p>
     建议不要安装全部软件包，太占用地方。如果只用C++，那么就可以只安装C++的软件包。
     <br/>
     双击VS的安装文件
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f39351752013c5e447af10511f91923e.png">
      <br/>
      选择自己需要的安装的类目。我只需要C++，因此仅勾选下图一个选项
      <br/>
      <img alt="只安装C++" src="https://i-blog.csdnimg.cn/blog_migrate/a954183da0fb5ac293ea96ecef8df107.png">
       <br/>
       接下来就是一值选择下一项，等待安装即可
      </img>
     </img>
    </p>
    <h3>
     <a id="QT_16">
     </a>
     QT安装
    </h3>
    <p>
     安装QT前，首先我们要断掉网络或者注册QT账户。（PS：不得不说一句：“断网大法好”）
     <br/>
     否则就无法进行下一步。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e2126fcbb13a72b161e508590a9a88c8.png">
      <br/>
      接下来同样是一直点下一步和同意各种条款即可完成安装。
      <br/>
      完成安装后，就可以重新将电脑接入网络了。
     </img>
    </p>
    <h2>
     <a id="3_22">
     </a>
     3、将两个工具进行联合
    </h2>
    <p>
     接下来就是如何使用VS进行QT开发的关键操作了。
    </p>
    <h3>
     <a id="QTMSVC_24">
     </a>
     将QT的MSVC运行库路径添加到电脑的环境变量
    </h3>
    <p>
     着重以Win11为例。。。。。
     <br/>
     我的电脑–&gt;属性–&gt;系统–&gt;关于–&gt;高级系统设置
     <br/>
     第一步
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/197f759ee8a176db77834ec594f5171d.png">
      <br/>
      第二步
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1176898cfa2b34ae1760db3840550404.png">
       <br/>
       第三步 选中Path变量，双击Path的值或者单击下方编辑按钮
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d23be895fa312a490f8188a16691523a.png"/>
       <br/>
       第四步 双击空白行或者点击新建按钮，然后数据MSVC的绝对路径。至此，单击所有的确定按钮。添加系统变量完成。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/43760e17e4bd800c8bf008827fdbe417.png"/>
      </img>
     </img>
    </p>
    <h3>
     <a id="VSQT_35">
     </a>
     为VS添加QT插件
    </h3>
    <ul>
     <li>
      启动VS，在顶部工具栏选择扩展-&gt;管理扩展-&gt;在联机下进行搜索“QT"-&gt;选择Qt Visual Studio Tools进行安装。安装完毕，重新启动一下VS软件。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/694d5633b623acf3b8ef6e19007da130.png"/>
     </li>
     <li>
      重启后，我们还不能直接创建QT工程。直接创建会在进行创建时，直接无法进行。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/634b5dfe4f252bb832cbf0e8d304a7e6.png"/>
      <br/>
      根据提示，我们知道需要让VS至少知道一个计算机上安装的QT版本。可以理解为qmake工具。（PS：虽然利用VS进行编程，但实际上对代码进行编译，链接运行的操作还是要基于qmake的）。
     </li>
     <li>
      选择扩展–&gt;Qt VS Tools–&gt;Options
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9ecd78b7701a7658cdb8104776cb7210.png"/>
      <br/>
      然后会进入如下界面。
     </li>
     <li>
      选择Qt–&gt;Versions–&gt;Path，然后去选择qmake文件。
      <br/>
      第一步：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b0b786ac59c532407cb31441fba3c5e.png"/>
      <br/>
      第二步：选择打开
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a671990fb2e56b54d4fcddcc2045f5a.png"/>
      <br/>
      此时，返回Options界面后我们会看到已经有Qt的Version信息了
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/afc41ec3c74ba5459b472e62692523de.png"/>
      <br/>
      接下来我们就可以去在VS中新建一个QT工程了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5b989d0dcc3f6d73e12ef44fbd7aaa8f.png"/>
     </li>
    </ul>
