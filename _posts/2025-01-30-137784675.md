    <p>
     将Excel文件转换为PDF可以方便储存表格数据，此外在打印或共享文档时也能确保表格样式布局等在不同设备和操作系统上保持一致。今天给大家分享一个使用第三方Python库Spire.XLS for Python 实现Excel转PDF的简单方法。
    </p>
    <h5>
     <a id="httpsblogcsdnnetmaiya_yayaarticledetails134500663spm1001201430015502PythonExcelPDF_4">
     </a>
     <a href="https://blog.csdn.net/maiya_yaya/article/details/134500663?spm=1001.2014.3001.5502">
     </a>
     Python中Excel转PDF的实现步骤
    </h5>
    <p>
     1. 首先，通过pip命令来安装依赖库；
    </p>
    <pre><code>pip install Spire.XLS

</code></pre>
    <p>
     2. 安装完成后，导入所需库；
    </p>
    <pre><code>from spire.xls import *
from spire.common import *

</code></pre>
    <p>
     3. 加载Excel文档，然后可以通过 Spire.XLS for Python 提供的
     <strong>
      Workbook.SaveToFile()
     </strong>
     或
     <strong>
      Worksheet.SaveToPdf()
     </strong>
     来将Excel工作簿或某个指定的Excel工作表转换为PDF格式。
    </p>
    <p>
     4. 在转换时还可以通过
     <strong>
      PageSetup
     </strong>
     类来进行页面设置。
    </p>
    <p>
     <strong>
      <em>
       代码示例
      </em>
     </strong>
    </p>
    <h5>
     <a id="httpsblogcsdnnetmaiya_yayaarticledetails134500663spm10012014300155021_PythonExcelPDF_27">
     </a>
     <a href="https://blog.csdn.net/maiya_yaya/article/details/134500663?spm=1001.2014.3001.5502">
     </a>
     1 .使用Python将Excel文件转为PDF格式（每个工作表显示为单独的一页）
    </h5>
    <pre><code>from spire.xls import *
from spire.common import *
 
# 创建Workbook对象
workbook = Workbook()
 
# 加载Excel文档
workbook.LoadFromFile("数据.xlsx")
 
# 设置工作表适应页面
workbook.ConverterSetting.SheetFitToPage = True
 
# 将Excel转换为PDF
workbook.SaveToFile("ToPDF.pdf", FileFormat.PDF)
workbook.Dispose()


</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/3cc89f2b05e952a1cea765c8c2e85e91.png"/>
    </p>
    <h5>
     <a id="httpsblogcsdnnetmaiya_yayaarticledetails134500663spm10012014300155022_PythonExcelPDF_51">
     </a>
     <a href="https://blog.csdn.net/maiya_yaya/article/details/134500663?spm=1001.2014.3001.5502">
     </a>
     2. 使用Python将Excel中每个工作表转换为单独的PDF文件
    </h5>
    <pre><code>from spire.xls import *
from spire.common import *
 
# 创建 Workbook 对象
workbook = Workbook()
 
# 加载Excel文件
workbook.LoadFromFile("数据.xlsx")
 
#遍历Excel工作簿中的所有工作表
for sheet in workbook.Worksheets:
 
#将每个工作表都保存为一个单独的PDF文件
    FileName =  sheet.Name + ".pdf"
    sheet.SaveToPdf(FileName)
workbook.Dispose()
  

</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/49aeb74c02b540bb06f6ec3197448b33.png"/>
    </p>
    <h5>
     <a id="httpsblogcsdnnetmaiya_yayaarticledetails134500663spm10012014300155023_PythonExcelPDF_76">
     </a>
     <a href="https://blog.csdn.net/maiya_yaya/article/details/134500663?spm=1001.2014.3001.5502">
     </a>
     3. 使用Python将指定Excel工作表转为PDF格式
    </h5>
    <pre><code>from spire.xls import *
from spire.common import *
 
# 创建 Workbook 对象
workbook = Workbook()
 
# 加载Excel文件
workbook.LoadFromFile("数据.xlsx")
 
# 获取第二个工作表
sheet = workbook.Worksheets[1]
 
# 获取 PageSetup 对象
pageSetup = sheet.PageSetup
 
# 设置页边距
pageSetup.TopMargin = 0.3
pageSetup.BottomMargin = 0.3
pageSetup.LeftMargin = 0.3
pageSetup.RightMargin = 0.3
 
# 设置页面尺寸
pageSetup.PaperSize = PaperSizeType.PaperA3
 
# 设置转换时工作表适应页面
workbook.ConverterSetting.SheetFitToPage = True
 
# 将该工作表保存为PDF文件
sheet.SaveToPdf("Excel工作表转PDF.pdf")
workbook.Dispose()

</code></pre>
