    <h4 id="一、通过apt-get安装指定版本的软件">
     一、通过apt-get安装指定版本的软件
    </h4>
    <pre><code>$ sudo apt-get install package=version</code></pre>
    <p>
     version是软件版本号，package是要安装的软件
    </p>
    <h4 id="二、查询指定软件有多少个版本">
     二、查询指定软件有多少个版本
    </h4>
    <p>
     <strong>
      1、通过网站搜索
     </strong>
    </p>
    <pre><code>https://packages.ubuntu.com/</code></pre>
    <p>
     <strong>
      2、使用 apt-cache madison 列出软件的所有来源
     </strong>
    </p>
    <pre><code>$ sudo apt-cache madison package</code></pre>
    <p>
     <img alt="file" src="https://i-blog.csdnimg.cn/blog_migrate/19a2e192e7a60f3b3ab9f881a0bdaa47.png"/>
    </p>
    <p>
     madison 是一个 apt-cache 子命令，可以通过man apt-cache查询更多apt-cache的用法。
    </p>
    <p>
     <strong>
      3、使用 apt-cache policy 列出软件的所有来源
     </strong>
    </p>
    <pre><code>$ sudo apt-cache policy vim</code></pre>
    <p>
     <img alt="file" src="https://i-blog.csdnimg.cn/blog_migrate/248bc849b166a44c563dd6e29a38237f.png"/>
    </p>
    <p>
     policy 是一个 apt-cache 子命令，可以通过 man apt-cache 查询更多apt-cache用法。
    </p>
    <p>
     <strong>
      4、使用 apt-cache showpkg 列出软件的所有来源
     </strong>
    </p>
    <pre><code>$ sudo apt-cache showpkg  vim </code></pre>
    <p>
     <img alt="file" src="https://i-blog.csdnimg.cn/blog_migrate/300439dda43d9e0280cd9f6942d6d8a0.png"/>
    </p>
    <p>
     <strong>
      5、使用 apt-get install -s 安装软件
     </strong>
    </p>
    <pre><code>$ sudo apt-get install -s package</code></pre>
    <p>
     <strong>
      6、使用 apt-show-versions 列出软件所有版本，并查看是否已经安装
     </strong>
    </p>
    <pre><code>$ sudo apt-get install apt-show-versions
$ apt-show-versions -a vim</code></pre>
    <p>
     <img alt="file" src="https://i-blog.csdnimg.cn/blog_migrate/2050d28b58d297df5947d0e501f983fc.png"/>
    </p>
    <p>
     还可以通过apt-show-versions -u package查询是否有升级版本。
    </p>
    <p>
     <strong>
      7、查询指定包的详情
     </strong>
    </p>
    <pre><code>$ sudo apt-cache show package</code></pre>
    <p>
     或者
    </p>
    <pre><code>$ dpkg -l package</code></pre>
    <p>
     <strong>
      8、显示已安装包的详情
     </strong>
    </p>
    <pre><code>$ dpkg -s package</code></pre>
    <p>
     或者
    </p>
    <pre><code>dpkg-query -s package</code></pre>
    <h4 id="三、使用技巧">
     三、使用技巧
    </h4>
    <p>
     在查询后面，我们可以带上一些参数来实现筛选
    </p>
    <pre><code>$ sudo apt-cache show package | grep version

$ sudo apt-show-versions | more</code></pre>
