    <h3>
     <a id="_6">
     </a>
     一：自然语言和编程语言基本概念
    </h3>
    <p>
     <strong>
      自然语言
     </strong>
     ：人与人之间进行沟通交流的表达方式（中文，英语，法语等）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4f53c6b6b4e46570e87588fa947493bd.png">
      <br/>
      <strong>
       编程语言（计算机语言）
      </strong>
      ：人与计算机之间进行沟通交流的表达方式（Python，C++等）
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1228be203e104d19ad3920af6e17a110.png">
       <br/>
       <strong>
        二：原理
       </strong>
       <br/>
       计算机只能识别二进制的0和1，假设需要计算机实现一个计算器功能，那第一步需要先按照对应编程语言（Python，C++，Java等）的语法实现该功能，然后交给编译器（解释器）将其翻译成二进制的0和1交给计算机解读，从而可以使计算机实现该功能。
       <br/>
       <strong>
        三：总结
       </strong>
       <br/>
       1，编程语言（C++，Python等）可以类似自然语言（英语，中文，法语等），需要遵循一定的语法规则，对方才可以读懂和理解。
       <br/>
       2，解释器（dea，Pycharm）可以类比翻译官
