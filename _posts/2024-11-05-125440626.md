     二、用ORM  Bee操作1万条数据测试性能
    </p>
    <p>
     <strong>
      简要解释下Suid, SuidRich.
     </strong>
    </p>
    <p>
     <strong>
      Suid是包含有简单的select,update,insert,delete操作的接口方法。
     </strong>
    </p>
    <p>
     <strong>
      SuidRich则有更多丰富多样的方法，像复杂查询，批量插入，聚合查询等。
     </strong>
    </p>
    <p>
     2.1批量插入1w数据
    </p>
    <pre><code class="language-java">	public int testBatchInsert10000Div2() {
		Tb_inaccount a;//命名不规范,忽学. 只是为了测更多场景
		String addr="aaa";
		List&lt;Tb_inaccount&gt; list=new ArrayList&lt;&gt;();
		for (int i = 1; i &lt;= 10000; i++) {
			a=new Tb_inaccount();
			a.setid(i);
			a.setMark(addr+i);
			a.setMoney(0.618);
			list.add(a);
		}
		return suidRich.insert(list,5000); 
	}</code></pre>
    <p>
    </p>
    <p>
     2.2 查询1w条数据
    </p>
    <pre><code class="language-java">	public void testQuery10000() {
//		return suidRich.select(new Tb_inaccount(), 0, 10000);
		for (int i = 0; i &lt; 10; i++) { //分10批查
			suidRich.select(new Tb_inaccount(), i*10000, 1000);
		}
	}</code></pre>
