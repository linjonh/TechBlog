     本文目前仅介绍导出所有数据库以及导入所有数据库
    </p>
    <h2>
     <a id="1_1">
     </a>
     1.导出所有数据库
    </h2>
    <p>
     如果是通过docker配置需要先进入docker的mysql容器内，在进行导出操作，即：
    </p>
    <ul>
     <li>
      1.进入mysql容器
     </li>
    </ul>
    <pre><code>docker exec -it 1e2e5584d721 /bin/sh
</code></pre>
    <p>
     注意：1e2e5584d721是你的镜像Id
    </p>
    <ul>
     <li>
      2.导出所有数据
     </li>
    </ul>
    <pre><code>mysqldump -uroot -p --all-databases &gt; /home/all.sql
</code></pre>
    <h2>
     <a id="2allsql_mysql_12">
     </a>
     2.将上一步导出的all.sql文件导入到其他 mysql服务器
    </h2>
    <h3>
     <a id="21_docker_13">
     </a>
     2.1 先进入到docker的镜像内，方法参照第一步
    </h3>
    <ul>
     <li>
      1.登录到mysql服务器
     </li>
    </ul>
    <pre><code>mysql -u 用户名 -p -h 远程主机 -P数据库端口
</code></pre>
    <h3>
     <a id="22__18">
     </a>
     2.2 导入
    </h3>
    <pre><code>source /home/all.sql;
</code></pre>
    <p>
     注意我是将第一步导出的all.sql文件放在了我mysql服务器的home目录了
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8755c24ffd32477acebc4e027bf492c2.png#pic_center"/>
    </p>
    <p>
     不要忘了后面的逗号
     <br/>
     等待导入结束即可
    </p>
