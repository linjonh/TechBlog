    <h1 class="title-article" id="articleContentId">
     下载网页视频的方法
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
    </p>
    <p>
     <span style="font-size:18px">
      随着技术的不断更新，现在小视频越来越火，有的时候想保存浏览的小视频，可不知道如何下载？
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      对于一些非专业的视频网站的小视频应该通过浏览器的选项是可以下载和保存的。下面就介绍使用浏览器下载的方法。
     </span>
    </p>
    <p>
     <span style="font-size:18px">
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      1
      <span style="font-family:宋体">
       用浏览器打开要播放的视频，并播放。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      2
      <span style="font-family:宋体">
       按
      </span>
      <span style="font-family:Calibri">
       F12
      </span>
      <span style="font-family:宋体">
       键，一般浏览器都是
      </span>
      <span style="font-family:Calibri">
       F12
      </span>
      <span style="font-family:宋体">
       ，如果没效果，那么就去找“开发人员工具”或者“开发者工具”，一般是在工具菜单下，或者帮助菜单下。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      3
      <span style="font-family:宋体">
       在出来的页面里选择
      </span>
      <span style="font-family:Calibri">
       Network
      </span>
      <span style="font-family:宋体">
       ，如果你用过的话，以后会默认到这里。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      4
      <span style="font-family:宋体">
       刷新页面，也就是页面重新载入（快捷键
      </span>
      <span style="font-family:Calibri">
       F5
      </span>
      <span style="font-family:宋体">
       ），并播放视频，如果你不刷新，或者先播放视频再点开“开发者工具”是捕获不到视频地址的。此时视频文件的地址已经暴露出来了，只需寻找了。一般的来讲，网络上
      </span>
      <span style="font-family:Calibri">
       mp4
      </span>
      <span style="font-family:宋体">
       文件用的是较多的，也有
      </span>
      <span style="font-family:Calibri">
       flv
      </span>
      <span style="font-family:宋体">
       的，看好视频后缀名称就可以了。如果你熟练了，你也可以先打开“开发者工具”在进入播放视频，一样的。一般来讲，同一网站的视频地址都会在一个地方，下回大概还是这个位置，就不用再浪费时间寻找了。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      <span style="font-family:宋体">
       如果确定视频是
      </span>
      MP4
      <span style="font-family:宋体">
       的，也可以在过滤框里输入
      </span>
      <span style="font-family:Calibri">
       MP4
      </span>
      <span style="font-family:宋体">
       进行搜索。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      5
      <span style="font-family:宋体">
       找到地址后，有两种方式下载：
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      1
      <span style="font-family:宋体">
       ）在找到的地址上右键，选择”
      </span>
      <span style="font-family:Calibri">
       Copy link address
      </span>
      <span style="font-family:宋体">
       ”
      </span>
      <span style="font-family:Calibri">
       (
      </span>
      <span style="font-family:宋体">
       复制链接地址
      </span>
      <span style="font-family:Calibri">
       )
      </span>
      <span style="font-family:宋体">
       ，可以把这个地址复制到迅雷，旋风等下载软件下载。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      2
      <span style="font-family:宋体">
       ） 在找到的地址上右键，“
      </span>
      <span style="font-family:Calibri">
       Open link in new tab
      </span>
      <span style="font-family:宋体">
       ”
      </span>
      <span style="font-family:Calibri">
       (
      </span>
      <span style="font-family:宋体">
       在新选项卡上打开链接
      </span>
      <span style="font-family:Calibri">
       )
      </span>
      <span style="font-family:宋体">
       ，有的浏览器打开的页面就是选择保存文件路径，有的是打开一个网页，在网页里选择另存为，然后再到选择文件路径选择框。选择好路径，就是下载保存文件了。
      </span>
     </span>
    </p>
