    <h3>
     <a id="_2">
     </a>
     执行命令
    </h3>
    <h4>
     <a id="roscore_3">
     </a>
     roscore
    </h4>
    <ul>
     <li>
      roscore是ROS的系统先决条件节点和程序的集合，必须运行roscore才能使ROS节点进行通信。roscore将启动：
      <ol>
       <li>
        ros master
       </li>
       <li>
        ros 参数服务器
       </li>
       <li>
        rosout日志节点
       </li>
      </ol>
     </li>
     <li>
      用法：
     </li>
    </ul>
    <pre><code class="prism language-powershell">roscore
</code></pre>
    <p>
     或指定端口号：
    </p>
    <pre><code class="prism language-powershell">roscore <span class="token operator">-</span>p 端口号
</code></pre>
    <h4>
     <a id="rosrun_19">
     </a>
     rosrun
    </h4>
    <ul>
     <li>
      用于运行指定的ROS节点，语法：
     </li>
    </ul>
    <pre><code class="prism language-powershell">rosrun 包名 可执行文件名
</code></pre>
    <ul>
     <li>
      例：
     </li>
    </ul>
    <pre><code class="prism language-powershell">rosrun turtlesim turtlesim_node
</code></pre>
    <h4>
     <a id="roslaunch_29">
     </a>
     roslaunch
    </h4>
    <ul>
     <li>
      用于执行某个包下的launch文件
     </li>
    </ul>
    <pre><code class="prism language-powershell">roslaunch 包名 launch文件名
</code></pre>
    <h3>
     <a id="_35">
     </a>
     增
    </h3>
    <h4>
     <a id="ROS_37">
     </a>
     创建新的ROS功能包
    </h4>
    <pre><code class="prism language-powershell">catkin_create_pkg 自定义包名 依赖包
</code></pre>
    <h4>
     <a id="_ROS_41">
     </a>
     安装 ROS功能包
    </h4>
    <pre><code class="prism language-powershell">sudo apt install xxx 
</code></pre>
    <h3>
     <a id="_46">
     </a>
     删
    </h3>
    <h4>
     <a id="_47">
     </a>
     删除某个功能包
    </h4>
    <pre><code class="prism language-powershell">sudo apt purge xxx 
</code></pre>
    <h3>
     <a id="_53">
     </a>
     查
    </h3>
    <h4>
     <a id="_54">
     </a>
     列出所有功能包
    </h4>
    <pre><code class="prism language-powershell">rospack list
</code></pre>
    <h4>
     <a id="_59">
     </a>
     查找指定功能包
    </h4>
    <ul>
     <li>
      查找某个功能包是否存在，如果存在返回安装路径
     </li>
    </ul>
    <pre><code class="prism language-powershell">rospack find 包名 
</code></pre>
    <h4>
     <a id="_65">
     </a>
     进入某个功能包
    </h4>
    <pre><code class="prism language-powershell">roscd 包名
</code></pre>
    <h4>
     <a id="_70">
     </a>
     列出某个包下的文件
    </h4>
    <pre><code class="prism language-powershell">rosls 包名
</code></pre>
    <h4>
     <a id="_75">
     </a>
     搜索某个功能包
    </h4>
    <pre><code class="prism language-powershell">apt search xxx
</code></pre>
    <h3>
     <a id="_81">
     </a>
     改
    </h3>
    <h4>
     <a id="_82">
     </a>
     修改功能包文件
    </h4>
    <ul>
     <li>
      需要安装 vim
     </li>
    </ul>
    <pre><code class="prism language-powershell">rosed 包名 文件名
</code></pre>
    <ul>
     <li>
      例:
     </li>
    </ul>
    <pre><code class="prism language-powershell">rosed turtlesim Color<span class="token punctuation">.</span>msg
</code></pre>
