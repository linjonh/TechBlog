    <p>
     方法一：cmd命令行执行pip install pandas
     <br/>
     1.Windows+R，输入cmd打开命令行窗口，输入pip install pandas。
     <br/>
     如下图所示
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dea6b32084427e092f718c44a14eaeef.png">
      <br/>
      2.若出现下图所示的告警，说明版本有冲突。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac751be1709f6b950253e7ca28e2b4ff.png">
       <br/>
       按照提示输入pip install --upgrade pip，对pip进行升级
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4072bead6e9120a77923c94e2eab2699.png">
        <br/>
        3.若出现下图所示的升级报错，输入python -m ensurepip，python -m pip install --upgrade pip
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ecfd08311ca66953050a7338a496cddf.png">
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14efe19b653a0a249b67b78515ae281b.png">
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0d64d5744f512c17740abb6dfbde0e49.png">
           <br/>
           4.再输入pip install pandas进行安装
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/39a8556dfbf87603a7f6e1f4176f8d34.png"/>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     5.若依然安装报错，可采取方法二进行安装
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/420bd58ad86491ada3c3f853edf96c90.png"/>
    </p>
    <p>
     方法二：找到pandas下载库，然后通过pip install [pandas包路径]安装
     <br/>
     1.进入网站 https://www.lfd.uci.edu/~gohlke/pythonlibs/
     <br/>
     Ctrl+F,输入pandas,找到需要的版本下载即可。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0d4b5ef238e385b03d88c5d501ee1c2d.png"/>
     <br/>
     比如我要下载的是pandas-1.4.3-cp310-cp310-win_amd64.whl。
     <br/>
     下载后保存到磁盘某一个路径下
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eb5050a977f8a97a6357895f60c20d25.png"/>
     <br/>
     2.打开cmd命令行，输入pip install [pandas包路径]
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0db926b863169dee458148b222adb65f.png"/>
    </p>
    <p>
     方法三：如果报超时的错，是因为国外的源文件在国内下载比较慢，可更换为国内的镜像。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f626cf8fb908051cb7622ec75f4b1274.png"/>
     <br/>
     使用清华大学的pandas镜像进行安装：
     <br/>
     pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pandas
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1060ee04e75778711e635ccca0ee3784.png"/>
     <br/>
     下载很快，且无报错。
     <br/>
     最后执行一下python -m pip list命令查看已经安装的包，可以看到不光pandas已经装好了，还安装了numpy库。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/75361e87363fa19bf9076e232e3b7222.png"/>
    </p>
