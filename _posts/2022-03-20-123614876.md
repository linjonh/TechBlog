    <h2>
     <a id="Visual_Studio_1">
     </a>
     Visual Studio下载
    </h2>
    <p>
     打开网址
     <a href="https://visualstudio.microsoft.com/zh-hans/vs/unity-tools/" rel="nofollow">
      https://visualstudio.microsoft.com/zh-hans/vs/unity-tools/
     </a>
    </p>
    <p>
     点击下载VisualStudio Community 2022版本
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1f942e1c85efdafb9cf03104df0cc8c1.png">
      <br/>
      下载后记得勾选Unity游戏开发
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/00c9e5b5b2fbd7964b732fd1d2fb86c5.png">
       <br/>
       下载完毕后重启电脑应该就OK啦
      </img>
     </img>
    </p>
    <h2>
     <a id="UnityHub_10">
     </a>
     UnityHub的下载
    </h2>
    <p>
     打开网址
     <a href="https://unity.cn/releases" rel="nofollow">
      https://unity.cn/releases
     </a>
     .
    </p>
    <p>
     进入后打开右上角注册，这里的注册可以直接用微信注册，接下来应该要你手机下个UnityHub啥的，然后用微信登陆，照着指示操作就完事er了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c43f3945d3b273a5a280e703c9efcd80.png">
      <br/>
      注册完毕后，再点击右上角的下载Unity
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a86e90f36321e53c0049b803d42f1ca6.png"/>
     </img>
    </p>
    <p>
     这里找到长期支持版本中的这个版本，点击下载UnityHub
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5f018aa42f548560fbe1def8a3fc454a.png">
      <br/>
      下载完毕后进入
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa5b17c728b794ee43cf1b3b78e17e55.png"/>
     </img>
    </p>
    <p>
     打开Hub后通行证就选个人非商业那种就好，也可以直接在安装里面找到这个版本来下载
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4005a954631095756eb687b229f08f1e.png"/>
    </p>
    <p>
     点击下一步，这里的内容暂时我们只需要勾选下面这一项，以后做项目需要的话再添加就OK了。千万不要勾选Documentation，我也不知道为什么这玩意就是下载不下来
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/92f84902beefc16f934d5ff38a653dff.png">
      <br/>
      然后打开项目打开新建来创建你的第一个项目
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4594aa3dff65ec236d05c831e871911b.png"/>
      <br/>
      创建项目后首先先选择一下unity的编译器，Edit中Preference中选择External Tools将编译器选择我们先下载好的VS，不然可能没有自动补全
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/107cf5d8b99e5466c34704b42d4b2c76.png"/>
      <br/>
      好了，接下来在Hierarchy窗口下随便创点东西玩玩吧。
      <br/>
      右键3D object 整个plane啥的都可以试试，你也可以在平面里随意拖动他，调整他玩一下，接下来就看你表演了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f43a44636f6643cbc86b55a5e7cb8988.png"/>
     </img>
    </p>
    <p>
     另外：如果第一次用可能全是英文会有点不习惯，可以在左上角的Edit找到Preferences里面点击language，切换为中文版先适应一下，不过还是最好适应英文版的，基本配置就完成了，接下来就可以进行游戏开发啦~
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/66df8dca7f71a966e198bd4e6d2a22c4.png"/>
    </p>
