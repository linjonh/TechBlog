     软考：软件工程:
    </h2>
    <p>
     <code>
      提示：系列被面试官问的问题，我自己当时不会，所以下来自己复盘一下，认真学习和总结，以应对未来更多的可能性
     </code>
    </p>
    <p>
     关于互联网大厂的笔试面试，都是需要细心准备的
     <br/>
     （1）自己的科研经历，
     <strong>
      科研内容
     </strong>
     ，学习的相关领域知识，要熟悉熟透了
     <br/>
     （2）自己的实习经历，做了
     <strong>
      什么内容
     </strong>
     ，学习的领域知识，要熟悉熟透了
     <br/>
     （3）除了科研，实习之外，平时自己关注的
     <strong>
      前沿知识
     </strong>
     ，也不要落下，仔细了解，面试官很在乎你是否喜欢追进新科技，跟进创新概念和技术
     <br/>
     （4）准备
     <strong>
      数据结构与算法
     </strong>
     ，有笔试的大厂，第一关就是手撕代码做算法题
     <br/>
     面试中，实际上，你准备数据结构与算法时以备不时之需，有足够的信心面对面试官可能问的算法题，很多情况下你的科研经历和实习经历足够跟面试官聊了，就不需要考你算法了。但很多大厂就会面试问你算法题，因此不论
     <strong>
      为了笔试面试
     </strong>
     ，数据结构与算法必须熟悉熟透了
     <br/>
     秋招提前批好多大厂不考笔试，直接面试，能否免笔试去面试，那就看你
     <strong>
      简历实力
     </strong>
     有多强了。
     <br/>
     （5）考网警的大佬，务必把软考：中级软件设计师的内容学好学透，学广，学多
    </p>
