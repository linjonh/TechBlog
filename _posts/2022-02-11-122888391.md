    <h3>
     <a id="1__0">
     </a>
     1. 简介
    </h3>
    <ul>
     <li>
      <p>
       WebAI.js 是一个基于
       <a href="https://docs.opencv.org/4.5.5/d5/d10/tutorial_js_root.html" rel="nofollow">
        OpenCV.js
       </a>
       和
       <a href="https://github.com/microsoft/onnxruntime/tree/master/js">
        ONNXRuntime
       </a>
       开发的一个 Web 前端 AI 模型部署工具
      </p>
     </li>
     <li>
      <p>
       项目 GitHub：
       <a href="https://github.com/AgentMaker/WebAI.js">
        AgentMaker/WebAI.js
       </a>
      </p>
     </li>
     <li>
      <p>
       可通过在线体验网站
       <a href="https://AgentMaker.github.io/WebAI.js" rel="nofollow">
        Hello WebAI.js
       </a>
       进行快速的体验试用
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f26e310f034dfab01081fc68d0af326d.png"/>
      </p>
     </li>
    </ul>
    <h3>
     <a id="2__7">
     </a>
     2. 特性
    </h3>
    <ul>
     <li>
      <p>
       WebAI.js 支持 HTML script 标签引入和 node.js 两种方式进行使用
      </p>
     </li>
     <li>
      <p>
       目前支持目标检测 (Yolo / ssd / …)、图像分类 (MobileNet / EfficientNet / …)、图像分割(BiseNet / PPSeg / …) 三类 CV 模型
      </p>
     </li>
     <li>
      <p>
       目前支持
       <a href="https://www.github.com/PaddlePaddle/PaddleDetection" rel="nofollow">
        PaddleDetection
       </a>
       /
       <a href="https://www.github.com/PaddlePaddle/PaddleClas" rel="nofollow">
        PaddleClas
       </a>
       /
       <a href="https://www.github.com/PaddlePaddle/PaddleSeg" rel="nofollow">
        PaddleSeg
       </a>
       三个套件部分导出模型的部署
      </p>
     </li>
    </ul>

  topK(number): 返回置信度前 K (K&gt;0) 个结果，如果 K&lt;0 返回所有结果

  bboxes({
      label: string, // 标签
      score: number, // 置信度
      color: number[], // 颜色（RGBA）
      x1: number, // 左上角 x 坐标
      y1: number, // 左上角 y 坐标
      x2: number, // 右下角 x 坐标
      y2: number // 右下角 y 坐标
  }[]): 目标检测包围框结果
  probs({
      label: string, // 标签
      prob: number // 置信度
  }[]): 图像分类置信度结果
  segResults({
      gray: cv.Mat, // 最大值索引图像（Gray）
      colorRGBA: cv.Mat, // 伪彩色图（RGBA）
      colorMap: { // 调色板
          lable: string, // 标签
          color: number[] // 颜色（RGBA）
      }[]
  }): 图像分割结果
</code></pre>
     </li>
     <li>

        API 参考
