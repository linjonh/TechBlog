      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-family:tahoma, arial, '宋体';">
      </p>
      <h2 style="padding:5px 0px;margin-top:0px;margin-bottom:0px;font-family:tahoma, arial, '宋体';">
       <span style="font-size:24px;">
        Linux系统有时候需要测试某个端口的连通性，用户可以参考如下方法来测试。
       </span>
      </h2>
      <div style="padding:0px;margin:0px;">
       <span style="font-size:24px;">
        <br/>
       </span>
      </div>
      <p>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        方法一、telnet法
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       telnet为用户提供了在本地计算机上完成远程主机工作的能力，因此可以通过telnet来测试端口的连通性，具体用法格式：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="line-height:21.6px;font-size:12px;color:rgb(255,0,0);">
        <span style="font-weight:700;">
         telnet ip port
        </span>
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       说明：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        ip：
       </span>
       是测试主机的ip地址
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        port：
       </span>
       是端口，比如80
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果telnet连接不存在的端口，那会如下图所示。
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/8396751a7b240ec318f8e0e02ddb2c99.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果telnet 连接存在端口会出现如下图所示的内容，下图中以80端口为例。
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a199b3518081125f22b8658cbd71d01e.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        方法二、ssh法
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       SSH 是目前较可靠，专为远程登录会话和其他网络服务提供安全性的协议,在linux上可以通过ssh命令来测试端口的连通性，具体用法格式如下：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       用法:
       <span style="line-height:21.6px;font-size:12px;color:rgb(255,0,0);">
        <span style="font-weight:700;">
         ssh -v -p port username@ip
        </span>
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       说明：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        -v
       </span>
       调试模式(会打印日志).
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        -p
       </span>
       指定端口
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        username:
       </span>
       远程主机的登录用户
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        ip:
       </span>
       远程主机
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果远程主机开通了相应的端口，会有如下图所示的建立成功的提示。
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f818fa49d1ff1d48cb298546db70b1c4.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果远程主机没有开通相应的端口，则如下图所示
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2d7ed837c98011fa01f3b5fc2ba1488f.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        方法三、curl法
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       curl是利用URL语法在命令行方式下工作的开源文件传输工具。也可以用来测试端口的连通性，具体用法:
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="line-height:21.6px;font-size:12px;color:rgb(255,0,0);">
        <span style="font-weight:700;">
         curl ip:port
        </span>
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       说明：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        ip
       </span>
       ：是测试主机的ip地址
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        port：
       </span>
       是端口，比如80
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果远程主机开通了相应的端口，都会输出信息，如果没有开通相应的端口，则没有任何提示，需要CTRL+C断开。
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e2f3c8d7f6af525289f76d98eb54eeaf.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        方法四、wget方法
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       wget是一个从网络上自动下载文件的自由工具
       <span style="font-weight:700;">
        ，支持通过HTTP、HTTPS、FTP三个最常见的TCP/IP协议下载
       </span>
       ，并可以使用HTTP代理。wget名称的由来是
       <span style="font-weight:700;">
        “World Wide Web”与“get”的结
       </span>
       合，它也可以用来测试端口的连通性具体用法:
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="line-height:21.6px;font-size:12px;color:rgb(255,0,0);">
        <span style="font-weight:700;">
         wget ip:port
        </span>
       </span>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       说明：
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        ip：
       </span>
       是测试主机的ip地址
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <span style="font-weight:700;">
        port：
       </span>
       是端口，比如80
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果远程主机不存在端口则会一直提示连接主机。
      </p>
      <p align="center" style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f4e1962050de412ecf0596b334875a0a.png" style="border:1px solid rgb(204,204,204);vertical-align:bottom;padding:1px;"/>
      </p>
      <p style="padding-top:5px;padding-bottom:5px;margin-bottom:0px;font-size:14px;text-align:left;font-family:tahoma, arial, '宋体';">
       如果远程主机存在端口则会看到相应的信息，如下图所示。
      </p>
