    <p>
     面试主要问一些什么
    </p>
    <p>
     板级测试
     <br/>
     有源测试：试电路板供电，这样可以测试电路板上的有源元器件（如晶体管、集成电路等），性能测试、环境测试和可靠性测试等方面。功能测试验证硬件的各项功能是否符合设计要求，性能测试评估硬件在不同工作负载下的性能表现
     <br/>
     无源测试：不需要给被测试电路板供电，主要用来测试电路板上的无源元器件（如电阻、电容、电感等）是否正常工作
     <br/>
     除了基本的有源和无源测试，板级测试还可能包括以下内容：
     <br/>
     1：信号完整性测试：测试单板上的各种信号质量，包括逻辑时序，以验证信号实际时序关系是否可靠，是否满足器件要求和设计要求。
     <br/>
     2：电源测试：测试电源部分是否高可靠，包括轻载、重载时的电源压降、纹波噪声等。
     <br/>
     3：极限情况测试：测试板子工作时的极限情况，比如所有设备都插上同时工作时的电压跌落、插拔动作造成的浪涌等。
     <br/>
     4：晶振测试：测试晶振出来时钟信号的频率及偏差。
     <br/>
     功耗，输入电流到输出电流的时间
    </p>
    <p>
     整机测试
     <br/>
     包含了功能测试+性能测试
    </p>
    <p>
     性能测试
     <br/>
     硬件电源电压测试 、电源纹波测试、上电时序、下电时序、逻辑功能电路测试、电源适应性测试、功耗测试、时钟测试、无线传输距离测试等
    </p>
    <p>
     硬件电源电压测试
     <br/>
     负载测试：使用电阻充电器或虚拟充电器给电源施加负载，检查电源是否能够稳定地处理负载而不会过热或关闭
     <br/>
     电压测试：使用万用表检查电源的输出电压是否符合规格，预期值为+3.3V、+5V和+12V等
     <br/>
     纹波和噪声测试：使用示波器检查电源输出的纹波和噪声，通常要求纹波小于输出电压的1%
    </p>
    <p>
     示波器 （分析电路中的波形信号）
     <br/>
     电源纹波测试
     <br/>
     测试目的：测试全部电源在各种负载条件下，电源的输出稳定性。
     <br/>
     测试方法：用示波器测量各电源的输出纹波，检测动态负载下的电压纹波峰峰值。
     <br/>
     测试环境：输入电源为固定，模拟用户使用环境，实现最大～最小动态负载，测试电源输出纹波要求
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6d3417fd154649609707ebfe0a8830b3.png">
      <br/>
      上电测试
      <br/>
      测试目的：测试上电时各路电压间相对时间关系（上电时序）。
      <br/>
      测试方法：模拟用户使用上下电，用示波器分别测量电压上电时间波形。
      <br/>
      测试环境：输入电源为固定，模拟用户使用上下电，测试各电源上电时序。
      <br/>
      测试标准：时序要求参考规格书。
     </img>
    </p>
    <p>
     时序测试
     <br/>
     测试目的：确保电子系统中的信号在正确的时间按照正确的顺序到达正确的位置，从而保证系统的正常运行和性能
     <br/>
     测试方法：建立时间的定义和测量方法，保持时间的定义和测量方法，传输延迟的定义和测量方法，相位偏移的定义和测量方法
    </p>
    <p>
     逻辑功能电路测试
     <br/>
     测试目的：逻辑功能电路测试是对电子系统中的数字逻辑电路进行的测试，目的是验证电路的功能是否符合设计要求
     <br/>
     测试方法：电源测试，验证电路板上的电源电压和电流是否符合规格要求，确保电源稳定
    </p>
    <p>
     电源适应性测试
     <br/>
     测试目的：用于评估电源设备在不同电源条件下的性能和稳定性的一项测试
     <br/>
     测试方法：输入输出特性测试，包括工作输入电压和电压变动范围、输入电压的频率和频率变动范围、额定输入电流、输入下陷和瞬间停电、冲击电流、漏电流和效率等
    </p>
    <p>
     功耗测试
     <br/>
     测试目的：是衡量电子设备在不同工作状态下能耗的一个重要手段
     <br/>
     测试方法：使用功率计或万用表直接测量设备在特定负载下的功耗。这种方法简单直接，常用于测量设备的总功耗
    </p>
    <p>
     时钟测试
     <br/>
     测试目的：验证时钟信号的质量和准确性，确保时钟信号在电子系统中的稳定性和可靠性
     <br/>
     测试方法：频率测量：通过测量时钟信号的频率来判断其准确性和稳定性
    </p>
    <p>
     无线传输距离测试
     <br/>
     测试目的：评估无线设备在不同环境下的通信范围和信号质量
     <br/>
     测试方法：Ping 包测试：通过发送Ping包来测试信号的连续性和丢包率，逐步增加设备之间的距离，直到信号连接断开
    </p>

    <p>
     作用： 电路的调试和维修
     <br/>
     品牌：西德宝
     <br/>
     作用：比如串口程序上电之后，会发送数据，但是没有就可以通过示波器查看是软件还是硬件问题
     <br/>
     带宽 100mhz
     <br/>
     采用率 每秒25亿个点
     <br/>
     存储深度 10k或者100k
     <br/>
     波形捕获率
     <br/>
     示波器的纹波和噪声：0.8v/1.1v纹波的要求是5%分别是40毫伏和50毫伏，20mhz看噪声的峰峰值是否在要求之内
     <br/>
     操作：交流带宽偶合限制，轮廓，打开测量，添加交流的有效值，然后测试一下高值和低值
     <br/>
     时序

    <h4>
     <a id="51__204">
     </a>
     5.1 测试晶振
    </h4>
    <p>
     为什么要测试晶振？
     <br/>
     晶振在电子设备中的应用非常广泛，包括时钟电路、通信系统、电子控制系统等。测试晶振可以确保这些系统能够正常、稳定地运行，避免因晶振问题而导致的系统故障或性能下降。
     <br/>
     怎么测试？
     <br/>
     （1）示波器检测法：
     <br/>
     晶振在激发时会产生正弦波，可以利用示波器来查看这个波形。
     <br/>
     将示波器的探头连接到晶振的时钟引脚上。
     <br/>
     观察示波器上显示的波形，如果是合适的正弦波，说明晶振正常工作。
     <br/>
     如果波形不稳定或者不是正弦波，可能说明晶振存在问题。
     <br/>
     （2）测试电路检测法：
     <br/>
     通过构建一个简单的测试电路来检查晶振是否工作。
     <br/>
     通常将晶振放置在晶体管的反馈网络中，并连接一个LED指示灯。
     <br/>
     如果晶振正常工作，LED灯会亮起；如果晶振不工作，LED灯将熄灭。
    </p>
    <p>
     电阻测量法：显示无穷大说明是好的，只能测试短路（没有经过用电器）的情况，如果晶振产生了断路（中途被断开）或者其他情况就测不出来
     <br/>
     电容测量法：没用
     <br/>
     验电笔：无效
     <br/>
     示波器：有用
    </p>
    <h4>
     <a id="52__223">
     </a>
     5.2 互感器：
    </h4>
    <p>
     （电流互感器的作用包括电流测量、电流保护和电流控制）
     <br/>
     为什么要测试互感器？
     <br/>
     电流不准，故障诊断与修复
     <br/>
     怎么测试互感器？
     <br/>
     满载测试：
     <br/>
     在额定负载条件下，通过施加电压或电流来测量互感器的负载损耗和温升。
     <br/>
     此方法用于确定互感器的负载能力和热稳定性。
    </p>
    <h4>
     <a id="53__232">
     </a>
     5.3 电阻
    </h4>
    <p>
     为什么要测试电阻？
     <br/>
     确保电路连接的正确性，判断是否存在断路、短路或其他连接问题，故障排查
     <br/>
     怎么测？
     <br/>
     直接测量法：
     <br/>
     使用专门的测量仪表，如万用表（欧姆档）进行电阻测量。
    </p>
