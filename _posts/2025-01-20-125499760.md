      </span>
      <span style="color:#FF0000;">
       ：开放2个超级入口，还能分享朋友圈
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="846" src="https://i-blog.csdnimg.cn/blog_migrate/1b58ce07fa35eee000a2cdade1704fea.jpeg" width="1200"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       2017
      </span>
      <span style="color:#595959;">
       年1月，微信小程序就上线了。为了保护朋友圈生态，小程序一直不支持被分享到朋友圈。而这样的一个规则，却被百度小程序打破了。
      </span>
     </span>
     <br/>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       5
      </span>
      <span style="color:#595959;">
       月22日，百度智能小程序上线，程序妹今天去体验了一番，结果真是吃鲸……接下来，程序妹给大家分享几个百度小程序的重大特点。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:0;">
     <strong>
      <span style="color:#00a0ea;">
       01
      </span>
     </strong>
     <strong>
      <span style="color:#00a0ea;">
       两大超级入口
      </span>
     </strong>
     <strong>
     </strong>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <strong>
       <span style="color:#00a0ea;">
        1.App
       </span>
      </strong>
      <strong>
       <span style="color:#00a0ea;">
        内的首页下拉
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       跟微信的首页下拉一样，百度App首页下拉出现小程序页面，而且是一个全屏页面。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="401" src="https://i-blog.csdnimg.cn/blog_migrate/2a068686766a42eeaea22b4c8cacbd16.png" width="280"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       根据“Talking Date”的数据，百度App的活跃率排名第7，这个超级入口流量不容小觑。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="716" src="https://i-blog.csdnimg.cn/blog_migrate/ff39746991667edc7aff36de8bee470b.jpeg" width="1048"/>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <strong>
       <span style="color:#00a0ea;">
        2.
       </span>
      </strong>
      <strong>
       <span style="color:#00a0ea;">
        搜索入口（正在测试）
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       百度的搜索入口开放给小程序了吗？答案应该是正在灰度测试中。程序妹试了几个，发现有一部分是有开放的，并且在搜索框中直接以最高级别显示。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="304" src="https://i-blog.csdnimg.cn/blog_migrate/b5ae139bfd75dc80e0a5cad137a1bfb2.png" width="416"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       这个超级入口对于品牌商，以及需要SEM推广的服务有非常大的意义。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:0;">
     <strong>
      <span style="color:#00a0ea;">
       02
      </span>
     </strong>
     <strong>
      <span style="color:#00a0ea;">
       分享朋友圈的能力
      </span>
     </strong>
     <strong>
     </strong>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       百度小程序有着强大的分享能力，可以直接分享给百度hi、微信好友和朋友圈。
      </span>
      <strong>
       <span style="color:#000000;">
        但由于百度始终是微信体系外的App，用户在微信中打开流程比较长。
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="430" src="https://i-blog.csdnimg.cn/blog_migrate/8da832960821338fb83dea98e8dc7da9.png" width="280"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       强大的分享功能，让百度小程序能够渗透到微信中，或许会返回来增加百度App的活跃度。
      </span>
     </span>
     <br/>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <strong>
       <span style="color:#00a0ea;">
        三.好玩的社交小游戏
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       微信小游戏采取的社交方案是通过社交，提高游戏的活跃度。而百度则可以放开来干：
      </span>
      <strong>
       <span style="color:#333333;">
        小游戏可以直接结识好友，达到更开放的社交效果。
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="1031" src="https://i-blog.csdnimg.cn/blog_migrate/5b1af5364eee87dbafcbdfc7216ca202.jpeg" width="1200"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       如图，百度小游戏通过匹配对战玩游戏。一般小姐姐都会匹配到小哥哥，反之亦然。游戏结束后，自动变为好友，可以与对方继续联系、继续玩游戏等等。
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <strong>
       <span style="color:#00a0ea;">
        四.智能的小程序
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:6pt;text-align:justify;">
    </p>
    <p style="margin-left:6pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       百度把擅长的AI能力赋能到小程序。让百度小程序在某些场景中更加好用。例如“长隆AR动物园”。在桌面上可以生成的一只大熊猫。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="672" src="https://i-blog.csdnimg.cn/blog_migrate/d7a94e79dfd74e64797e9a3ea5cfa932.jpeg" width="419"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <span style="color:#00a0ea;">
       05
      </span>
     </strong>
     <strong>
      <span style="color:#00a0ea;">
       BAT
      </span>
     </strong>
     <strong>
      <span style="color:#00a0ea;">
       都入局小程序，怎么选？
      </span>
     </strong>
     <strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#595959;">
       小程序作为互联网的新物种，已经经过大量的验证。但目前BAT都已入局小程序，想要升级服务的你，该如何选择呢？
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <ul>
     <li style="text-align:justify;">
      <span style="background-color:#FFFFFF;">
       <strong>
        <span style="color:#000000;">
         百度小程序，优势在于AI能力和广告推广能力
        </span>
       </strong>
       <span style="color:#000000;">
        。
       </span>
       <span style="color:#595959;">
        如果你的服务需要AI的支持，或者本身需要SEM付费推广的，可以考虑拥有一款百度小程序。
       </span>
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="background-color:#FFFFFF;">
       <strong>
        <span style="color:#000000;">
         支付宝小程序，优势在于信用能力。
        </span>
       </strong>
       <span style="color:#595959;">
        一般从本地生活服务切入，专注于垂直领域和信用消费领域。例如共享充电宝、酒店预定等。
       </span>
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="background-color:#FFFFFF;">
       <strong>
        <span style="color:#000000;">
         微信小程序，传播能力强，用户量大。
        </span>
       </strong>
       <span style="color:#595959;">
        几乎适合所有商家。如果你的服务不需要信用，不需要百度的AI能力，那么，你都应该优先考虑微信小程序。
       </span>
      </span>
     </li>
    </ul>
