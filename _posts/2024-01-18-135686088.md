    <h3>
     <a id="Android_StudioCtrlShiftF_0">
     </a>
     Android Studio全局搜索（Ctrl+Shift+F）不能用，产生快捷键冲突的两种情况，使用微软输入法和搜狗输入法。
    </h3>
    <p>
     <strong>
      1、使用的搜狗输入法
     </strong>
     <br/>
     需要进入按键选项中的系统功能快捷键，将第三个选项【简繁切换】关闭
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0783c915971c8b06da2428a757d2d126.png#pic_center">
      #pic_center))
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c43973b789eb0442716b6cba05009e8d.png#pic_center">
       )
      </img>
     </img>
    </p>
    <p>
     <strong>
      2、使用微软输入法
     </strong>
     <br/>
     同样需要关闭微软输入的简繁切换快捷键避免冲突
     <br/>
     首先：点击右下角的语言首选项
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/137299f9e113a3fbb9457965f3cf2045.png#pic_center">
      )
      <br/>
      然后：点击中文选项卡中的选项按钮
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9f02651c5f23ef8a8cbde7e2ab68061f.png#pic_center">
       )
       <br/>
       然后：继续点击微软拼音中的选项按钮
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0ccddef008cda78b9f14d245f4358a60.png#pic_center">
        )
        <br/>
        然后你应该自己也会了，点击按键，关闭简繁切换
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/be63c48fa8cc41382a81f6129c32b3e7.png#pic_center">
         )
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dba17968e57788b8a072a2ce85d24dd3.png#pic_center">
          )
          <br/>
          我的电脑在关闭简繁切换后，如果使用微软输入自动变成了繁体，还需要返回上一步在【常规】选项中将字符集切换为【简体中文】
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/aeec65f5895f7ec3135d1e1dd12d1669.png#pic_center"/>
          )
          <br/>
          然后返回Android studio 试一下 应该就可以使用Ctrl + Shift + F快捷键了。
          <br/>
          如果还没有解决可能需要下载检测热键冲突的软件找一下。
