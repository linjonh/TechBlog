    <h3>
     <a id="_0">
     </a>
     主要功能
    </h3>
    <ol>
     <li>
      HXDZ-30102-ACC采集心率血氧数据
     </li>
     <li>
      STM32通过串口将数据转发到air202模块
     </li>
     <li>
      air202将数据上传到阿里云平台进行展示与处理
     </li>
     <li>
      整合合宙air530GPS模块，将定位数据上传到阿里云
     </li>
    </ol>
    <h3>
     <a id="_5">
     </a>
     模块介绍
    </h3>
    <p>
     <strong>
      1. HXDZ-30102-ACC
     </strong>
     <br/>
     心率血氧传感器模块（HXDZ-30102-ACC）集成了LIS2DH12（ST的三轴加速传感器，用于记录运动数据）和MAX30102（血氧和心率检测记录）
     <br/>
     目前主要使用到MAX30102检测心率血氧信息
     <br/>
     开发主要参考资料： HXDZ-30102-ACC心率血氧传感器说明书、及相关demo、
     <a href="https://www.jianshu.com/p/39fac3a15493" rel="nofollow">
      参考博客
     </a>
     。
     <br/>
     获取途径：模块卖家提供。
     <br/>
     <strong>
      2. air202 S6开发版（Lua版）
     </strong>
     <br/>
     （1）接入阿里云物联网云平台
     <br/>
     （2）将采集到的数据进行数据处理上传至阿里云
     <br/>
     上海合宙通信科技有限公司开发的一款2G通信模块，支持OpenCPU二次开发支持阿里云、OneNet、百度天工、腾讯云等常见物联网平台，并有丰富的文档资料。
     <br/>
     <strong>
      3. STM32
     </strong>
     <br/>
     （1）心率血氧数据的采集
     <br/>
     （2）将心率血氧数据通过串口发送到air202
     <br/>
     <strong>
      4. 阿里云物联网平台
     </strong>
     <br/>
     （1）进行云端显示心率血氧数据
     <br/>
     <strong>
      5. Air530
     </strong>
     <br/>
     Air530模块是一款高性能、高集成度的多模卫星定位导航模块。体积小、功耗低，可用于车载导航、智能穿戴、无人机等GNSS 定位的应用中。而且提供了和其他模块厂商兼容的软、硬件接口，大幅减少了用户的开发周期。
    </p>
