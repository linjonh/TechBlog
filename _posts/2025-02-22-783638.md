        <span class="jj">
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           《
           <strong style="mso-bidi-font-weight: normal">
            信息系统工程
           </strong>
           》杂志决定从今年第六期增加一个栏目：《
           <strong style="mso-bidi-font-weight: normal">
            开源世界
           </strong>
           》，每期有固定的版面。那么，《
           <strong style="mso-bidi-font-weight: normal">
            开源世界
           </strong>
           》应该怎么办呢？
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           《
           <strong style="mso-bidi-font-weight: normal">
            信息系统工程
           </strong>
           》的英文刊名是《
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             CCnews
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           》，定位于“
          </span>
          <font face="Times New Roman">
           <strong style="mso-bidi-font-weight: normal">
            <span lang="EN-US" style="FONT-SIZE: 12pt">
             C
            </span>
           </strong>
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            hina
            <strong style="mso-bidi-font-weight: normal">
             C
            </strong>
            IO
            <strong style="mso-bidi-font-weight: normal">
             News
            </strong>
           </span>
          </font>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”。这表明刊物的主要读者群是“
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             CIO
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”（
          </span>
          <font face="Times New Roman">
           <strong style="mso-bidi-font-weight: normal">
            <span lang="EN-US" style="FONT-SIZE: 12pt">
             C
            </span>
           </strong>
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            hief
            <strong style="mso-bidi-font-weight: normal">
             I
            </strong>
            nformation
            <strong style="mso-bidi-font-weight: normal">
             O
            </strong>
            fficer
           </span>
          </font>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ），中文意思就是“
           <strong style="mso-bidi-font-weight: normal">
            首席信息官
           </strong>
           ”。很明显，
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             CIO
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           是信息系统工程的灵魂人物，因为这个“
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            CIO
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           组合词”的第一个字母是“
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             C
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”（首席之意）。用通俗一点的话来讲，“
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            CIO
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”就是单位主管信息化工作的“
           <strong style="mso-bidi-font-weight: normal">
            头
           </strong>
           ”（不能用于当面称呼）。《开源世界》，既然是为他们开办的，就得有自己的特点。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            CIO
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           都是精明人，对他们讲“套话”、“虚话”，丝毫没有用处。有时候，你说什么东西“
           <strong style="mso-bidi-font-weight: normal">
            好
           </strong>
           ”，他们一看就知道，这东西就是“不好”。为什么？因为，他们有一个“法宝”：
           <strong style="mso-bidi-font-weight: normal">
            实践经验
           </strong>
           。他们特别讲究实际，因为他们是干实事的人。因此，《开源世界》要面对精明实干的
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            CIO
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ，就必须要
           <strong style="mso-bidi-font-weight: normal">
            实事求是
           </strong>
           ，要有自觉服从科学真理的精神。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           我想，《开源世界》的内容必然涉及一些开源人物、开源理念、开源社区、开源技术、开源企业、开源经营模式和重大的历史事件，包括国内外的
           <strong style="mso-bidi-font-weight: normal">
            发展动态
           </strong>
           。栏目报道的内容要客观、准确，凡事要求“
           <strong style="mso-bidi-font-weight: normal">
            实事求是
           </strong>
           ”，不能误导读者。另一方面，《开源世界》栏目一定要有
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            CIO
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           的声音，不论正面、反面的意见都可以发表，争论有助于获得真知。《开源世界》栏目，会讲一些发生在我们身边的真人、真事，当然，这些“
           <strong style="mso-bidi-font-weight: normal">
            小故事
           </strong>
           ”必须涉及“开源”两字。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           “
           <strong style="mso-bidi-font-weight: normal">
            开源
           </strong>
           （
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             Open Source
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ）”一词，
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            1998
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           年年初，开始在美国媒体上出现。当时，国际“自由软件界”的某些领导人为了给“
           <strong style="mso-bidi-font-weight: normal">
            自由软件
           </strong>
           ”找到一个更为“
           <strong style="mso-bidi-font-weight: normal">
            好听
           </strong>
           ”
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            ”
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           的名字，创造了“
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             Open Source
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”（
           <strong style="mso-bidi-font-weight: normal">
            开源
           </strong>
           ）这个新词。那个时候，“
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             Open Source
            </font>
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”成了被美国媒体引用频率最高词汇之一，词汇的含义也逐渐发生演变，变得十分宽泛，让人不易捉摸。一时间，仿佛什么东西是开源了。至今，
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            8
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           个年头过去了，好像又回到了过去时光，“开源”软件一词与“自由软件”更加紧密相关，以至于在联合国的有关文件中也正式引用了“
           <strong style="mso-bidi-font-weight: normal">
            自由
           </strong>
          </span>
          <strong style="mso-bidi-font-weight: normal">
           <span lang="EN-US" style="FONT-SIZE: 12pt">
            <font face="Times New Roman">
             /
            </font>
           </span>
          </strong>
          <strong style="mso-bidi-font-weight: normal">
           <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
            开源软件
           </span>
          </strong>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”的说法。当今，开源软件与自由软件并没有完全脱离关系。但是，《开源世界》当然不是《自由世界》，这里指的是指：自由软件与开源软件的“世界”。简单说来，“
           <strong style="mso-bidi-font-weight: normal">
            开源软件
           </strong>
           ”是指：
           <strong style="mso-bidi-font-weight: normal">
            程序代码公开，软件自由使用
           </strong>
           。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           在《开源世界》里面，需要有一种正确观念：软件免费，不等于该软件的支持服务也一定免费。使用开源软件往往需要一定的技术支持服务为前提，而不是拿来就用。使用开源软件，需要购买相关的技术支持服务，而且要购买商业化的“支持服务”，这是一种新观念。在《开源世界》里面，“软件与服务”是相互紧密联系在一起的。一个开源软件是干什么用的，它需要在那些方面得到技术支持，这都是需要搞清楚的地方。笼统谈论“开源软件”的性能如何如何，往往是没有实际意义的。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
         <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 24pt">
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           《开源世界》就是基于自由
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            /
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           开源软件的“服务世界”，也就是“开源服务”的世界。让我们一起共同努力，把“
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
           <font face="Times New Roman">
            China CIO News
           </font>
          </span>
          <span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">
           ”月刊的《开源世界》栏目办得更好。
          </span>
          <span lang="EN-US" style="FONT-SIZE: 12pt">
          </span>
         </p>
        </span>
