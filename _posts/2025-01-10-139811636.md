    <h4>
     <a id="_UI__13">
     </a>
     老 UI 特点
    </h4>
    <p>
     老 UI 设计经典，包括：
    </p>
    <ul>
     <li>
      经典布局：顶部菜单栏、工具栏、侧边栏、中央编辑器区域和底部状态栏。
     </li>
     <li>
      配色主题：提供Light和Darcula主题。
     </li>
     <li>
      工具窗口：可固定或浮动。
     </li>
     <li>
      传统图标和字体：辨识度高，但可能不适合高分辨率屏幕。
     </li>
    </ul>
    <h4>
     <a id="UI_20">
     </a>
     启用新UI
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4af47f04f40248eea92d1a1e200c8121.png"/>
    </p>
    <h4>
     <a id="_UI__23">
     </a>
     新 UI 特点
    </h4>
    <p>
     新 UI 进行了全面改进，特点包括：
    </p>
    <ul>
     <li>
      现代化设计：扁平化图标和清晰布局。
     </li>
     <li>
      响应式设计：适应不同分辨率，特别是高分辨率屏幕。
     </li>
     <li>
      改进的配色方案：增加个性化选项，优化对比度和可读性。
     </li>
     <li>
      集成更多功能：内置终端、改进的版本控制和数据库工具。
     </li>
     <li>
      高度可定制化：用户可自定义布局、配色和快捷键。
     </li>
    </ul>
    <p>
     新UI旨在提升用户体验和开发效率，同时保持高度的可定制性。
    </p>
