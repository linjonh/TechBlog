     物联网云平台之大数据处理概念
    </h2>
    <p>
     1.私有云的核心属性是专有资源。
     <br/>
     2.云计算主流解决方案有四种：OpenStack，CloudStack，Eucalyptus和VMvCloudStuite。
     <br/>
     3.Keystone模块为OpenStack其他几个模块提供认证服务。
     <br/>
     4.Nova是OpenStack的核心模块之一负责管理。
     <br/>
     5.Glance模块提供OpenStack虚拟网络服务。
     <br/>
     6.Neutron模块提供OpenStack虚拟网络服务。
     <br/>
     7.在OpenStack的网络管理流程中，创建和删除网络步骤
     <br/>
     （1）创建一个网络
     <br/>
     （2）创建一个子网
     <br/>
     （3）启动一个虚拟机，将一块网卡对接到指定的网络上。
     <br/>
     （4）删除虚拟机
     <br/>
     （5）删除网络端口
     <br/>
     （6）删除网络
    </p>

    <p>
     2.网络协议模型
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c98814b8e5c0157c5e386afe74d4f12c.png#pic_center"/>

    <p>
     3.云平台的概念：平台即服务是指在云计算基础设施上为用户提供软件开发、运行和运营环境的服务，是把平台作为一种服务提供给用户的新型商业模式。
     <br/>
     4.云服务器也被称为计算单元，云服务器并不是真实的某一台或某一类服务器，而是由一组集群服务器利用虚拟化技术虚拟出来的，为用户提供弹性资源配置服务的虚拟服务器。
     <br/>
     5.数据仓库的特点：
     <br/>
     （1）面向主题特性是数据仓库和操作型数据库的根本区别。操作型数据库是为了支撑各种业务而建立的，而分析型数据库则是为了对从各种繁杂业务中抽象出来的分析主题（如用户、成本、商品等）进行分析而建立的。
     <br/>
     （2）集成性是指数据仓库会将不同源数据库中的数据汇总到一起。
     <br/>
     （3）数据仓库内的数据是面向公司全局的。
     <br/>
     （4）相比操作型数据库，数据仓库的时间跨度通常比较长。
     <br/>
     6.堆排序：堆排序（Heapsort）是指利用堆积树（堆）这种数据结构所设计的一种排序算法，它是选择排序的一种，可以利用数组的特点快速定位指定索引的元素。堆分为大顶堆和小顶堆，是完全二叉树。大顶堆的要求是每个节点的值都不大于其父节点的值，即A[PARENT[i]] &gt;= A[i]。在数组的非降序排序中，需要使用的就是大顶堆，因为根据大顶堆的要求可知，最大的值一定在堆顶。
     <br/>
     7.解决物理设备安全问题，通常采用：
     <br/>
     （1）尽可能加入已有的物理安全防护措施，防风雨、防雷电。
     <br/>
     （2）并非技术层面的问题，更应作为标准的一部分进行规范，发布物联网设备防盗、防破坏的规章制度。
     <br/>
     8.云计算主流的解决方案有四种：（1）云计算管理平台OpenStack
     <br/>
     （2）云计算平台CloudStack（3）开源的软件基础结构Eucalyptus（4）虚拟机组件VMware vCloud Suite
