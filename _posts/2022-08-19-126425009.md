    <h4 id="%E6%9C%AC%E6%AC%A1%E5%AE%9E%E9%AA%8C%E7%94%A8%E7%9A%84%E8%AE%BE%E5%A4%87%E6%98%AF%E7%A5%9E%E5%B7%9E%E6%95%B0%E7%A0%81CS6200%E3%80%81DCR2855%E3%80%81CRL1000" style="text-align:center;">
     <strong>
      本次实验用的设备是神州数码CS6200、DCR2855、CRL1000
     </strong>
    </h4>
    <h4 id="%E9%A6%96%E5%85%88%E7%A1%AE%E4%BF%9D%E8%AE%BE%E5%A4%87%E6%B2%A1%E6%9C%89%E4%BB%BB%E4%BD%95%E9%85%8D%E7%BD%AE" style="text-align:center;">
     <strong>
      首先确保设备没有任何配置
     </strong>
    </h4>
    <h4 id="%E6%8E%A5%E4%B8%8B%E6%9D%A5%E7%9A%84%E5%AE%9E%E9%AA%8C%E6%98%AF%E4%B8%8E%E4%B8%8A%E4%B8%80%E7%AB%A0%E7%BB%93%E5%90%88%E7%9A%84%EF%BC%8C%E5%9F%BA%E7%A1%80%E7%8E%AF%E5%A2%83%E8%AF%B7%E7%9C%8B%E4%B8%8A%E4%B8%80%E7%AB%A0%EF%BC%8C%E4%B8%8B%E9%9D%A2%E4%BC%9A%E7%9B%B4%E6%8E%A5%E9%85%8D%E7%BD%AE%E7%BD%91%E7%BB%9C%E7%8E%AF%E5%A2%83" style="text-align:center;">
     <strong>
      接下来的实验是与上一章结合的，基础环境请看上一章，下面会直接配置网络环境
     </strong>
    </h4>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4248a2289cda1a3b829b9b63e18676d4.png"/>
    </p>
    <p>
    </p>
    <p>
     <strong>
      1、首先创建一个VLAN配置上IP地址并分配端口
     </strong>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       CS6200(config)#vlan 200
       <br/>
       CS6200(config-vlan200)#switchport interface ethernet 1/0/24
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （这个端口是连接路由器的）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       CS6200(config-vlan200)#int vlan 200
       <br/>
       CS6200(config-if-vlan200)#ip add 172.16.34.253 255.255.255.0
      </strong>
     </span>
     <span style="color:#fe2c24;">
      <strong>
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （这个IP是内网用的也就是外网转换的）
      </strong>
     </span>
    </p>
    <p>
     <strong>
      2、配置交换机默认路由
     </strong>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       CS6200(config)#ip route 0.0.0.0/0 172.16.34.254
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （这里的/0代表四个0这是省略写法）
      </strong>
     </span>
    </p>
    <p>
     <strong>
      3、给路由器的端口配置IP并定义端口
     </strong>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#interface gigaEthernet 0/2
       <br/>
       DCR2855_config_g0/2#ip add 172.16.34.254 255.255.255.0
       <br/>
       DCR2855_config_g0/2#ip nat inside
      </strong>
     </span>
     <span style="color:#9c8ec1;">
      <strong>
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       (因为这个端口插得是内网端口所以给它设置为内部接口）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#interface gigaEthernet 0/0
       <br/>
       DCR2855_config_g0/0#ip add 192.168.1.96 255.255.255.0
       <br/>
       DCR2855_config_g0/0#ip nat outside
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （这个端口插的就是外网网线）
      </strong>
     </span>
    </p>
    <p>
     <strong>
      4、配置默认路由并允许相关网段访问
     </strong>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#ip route default 192.168.1.1
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （配置默认网关）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#ip route 192.168.100.1 255.255.255.0 172.16.34.253
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （配置默认路由指向交换机端口）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#ip access-list standard wl
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （创建访问列表wl是名称随意设置）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config_std_nacl#permit 192.168.100.1 255.255.255.0
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （允许访问）
      </strong>
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config_std_nacl#permit 172.16.34.253 255.255.255.0
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （这两个网段其实允许一个就行了允许两个也可以的）
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      <strong>
       5、配置NAT转换
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       DCR2855_config#ip nat inside source list nat interface gigaEthernet 0/0
      </strong>
     </span>
     <span style="color:#1c7331;">
      <strong>
       （做完这一步就算完成了）
      </strong>
     </span>
     <br/>
    </p>
