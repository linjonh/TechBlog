    <p>
     1、安装DevEco Studio
     <br/>
     直接安装即可
    </p>
    <p>
     2、配置开发环境
     <br/>
     1.运行已安装的DevEco Studio，首次使用，请选择Do not import settings，单击OK。
     <br/>
     2.安装Node.js与ohpm。注意使用默认路径，不可修改
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fd328d3c4bc6b162742f1078451ac5ce.png">
      <br/>
      3.设置SDK
      <br/>
      在SDK Setup界面，单击文件夹按钮，设置HarmonyOS SDK存储路径，单击Next进入下一步。
      <br/>
      可以自定义安装位置
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3caf5b8789e27df4c0096deeebb2ac7b.png">
       <br/>
       4.在弹出的SDK下载信息页面，单击Next，并在弹出的License Agreement窗口，阅读License协议，需同意License协议后，单击Next。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ba9000cd65ed7bdff88e58648e8e7b8.png">
        <br/>
        5.确认设置项的信息，点击Next开始安装。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f50ac96460592da4899984257a39ccba.png">
         <br/>
         6.等待Node.js、ohpm和SDK下载完成后，单击Finish，界面会进入到DevEco Studio欢迎页。
         <br/>
         安装需要较长时间，需要耐心等待。
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     3、设置环境变量
     <br/>
     HDC是为开发者提供HarmonyOS应用/服务的调试工具，为方便使用HDC工具，请为HDC端口号设置环境变量。
    </p>
    <p>
     在此电脑 &gt; 属性 &gt; 高级系统设置 &gt; 高级 &gt; 环境变量中，添加HDC端口变量名为：HDC_SERVER_PORT，变量值可设置为任意未被占用的端口，如7035。
     <br/>
     环境变量配置完成后，关闭并重启DevEco Studio。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cdc5f19c9d966fa22df0236019cb32c8.png"/>
    </p>
    <p>
     4、诊断开发环境
     <br/>
     在欢迎界面单击Help &gt; Diagnose Development Environment进行诊断。
     <br/>
     如果已经打开了工程开发界面，也可以在菜单栏单击Help &gt; Diagnostic Tools &gt; Diagnose Development Environment进行诊断。
    </p>
