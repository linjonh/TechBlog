    <p style="text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       首先需要安装与opencv-python 3.4.1.15版本对应的python版本，这里我下载的是python3.6.3版本
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       官网下载链接：
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <a href="http://www.python.org/download/" rel="nofollow" title="Download Python | Python.org">
       Download Python | Python.org
      </a>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       安装步骤不再进行详解，可自行进行搜索。要注意下载时选择的下载位置和添加路径
      </span>
     </span>
    </p>
    <p style="text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       打开python所在位置，找到Scripts文件夹，并打开它
      </span>
     </span>
    </p>
    <p style="text-align:justify;">
     <img alt="" height="334" src="https://i-blog.csdnimg.cn/blog_migrate/e58be8bfa9769027177f7f2176df8766.png" width="491"/>
    </p>
    <p>
     <span style="color:#000000;">
      <span style="color:#000000;">
       点击目录位置，输入cmd，并回车
      </span>
     </span>
    </p>
    <p>
     <img alt="" height="242" src="https://i-blog.csdnimg.cn/blog_migrate/a5ac5046e1d2a413af1d6fe8c7b86829.png" width="539"/>

    <p>

      <span style="color:#000000;">

    <p>
     <img alt="" height="196" src="https://i-blog.csdnimg.cn/blog_migrate/25a15081f0b24117b97a5d76180110c3.png" width="601"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       创建虚拟环境
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       输入pip install virtualenv
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="200" src="https://i-blog.csdnimg.cn/blog_migrate/7948b58fa420f801604aa8a71abf01ef.png" width="566"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       回车即可创建完成虚拟环境
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       在Scripts文件夹同目录下就会多出virtualenv文件
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="317" src="https://i-blog.csdnimg.cn/blog_migrate/dd30be29cc5ce207005880e79a92a8ac.png" width="544"/>

    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       我的虚拟环境更名为了venv
      </span>
     </span>
    </p>
    <p style="text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       打开venv中的Scripts文件，路径处输入cmd
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       输入activate启动虚拟环境
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="158" src="https://i-blog.csdnimg.cn/blog_migrate/26ceb23fbbc9a7e7e7d31e7f40be6dad.png" width="450"/>

    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       出现括号中的东西说明虚拟环境启动成功
      </span>
     </span>
    </p>
    <p style="text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       在虚拟环境中安装opencv3.4.1.15
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       这里推荐使用国内镜像进行下载，下载速度快，不会出现超时的异常
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">
      <span style="color:#000000;">
       输入
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#000000;">

       Simple Index
