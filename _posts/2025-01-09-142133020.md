     python3.8下载及安装详细教程
    </h2>
    <h3>
     <a id="Python38_1">
     </a>
     一：Python3.8下载
    </h3>
    <p>
     官网下载地址：
     <a href="https://www.python.org/ftp/python/3.8.0/python-3.8.0-amd64.exe" rel="nofollow">
      python3.8
     </a>
    </p>
    <h3>
     <a id="Python38_5">
     </a>
     二、Python3.8安装
    </h3>
    <p>
     1：双击打开.exe文件
     <br/>
     勾选如图，然后点击【Customize installation】（也可不勾选，安装后手动添加环境变量）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/981cff641a284de3a9dbc063a52cefa9.png"/>
    </p>
    <p>
     点击【Next】
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bae3c74914924140b5bba5645bb4bdff.png"/>
    </p>
    <p>
     可以修改安装地址（也可以不修改），然后点击【Install】
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/765e86a551eb4514b43f02ff7086c018.png">
      <br/>
      软件安装中，请稍等
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ad136c5928c2484fbde2933a4397b6bf.png">
       <br/>
       等待进度条走完，完成安装，点击【Close】
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0da2c9efc42f40dba4ba17d589ff82df.png"/>
    </p>
    <h3>
     <a id="_23">
     </a>
     *三、配置环境变量
    </h3>
    <p>
     若安装时已勾选自动配置环境变量，可略去此步骤
    </p>
    <p>
     1、右键我的电脑，点击【属性】
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e5d74a6fcfd04f6bac4098178265dec4.png">
      <br/>
      2、点击高【级系统设置】
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d0644df8f7b14bc4801d8b68d1570c90.png">
       <br/>
       3、点击【环境变量】
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a93b2bca634b44288277cdd561e6a35d.png">
        <br/>
        4、选中【系统变量】下的【Path】变量，点击【编辑】
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/55f6ee4909614ada870c1da2d2516f3b.png"/>
        <br/>
        5、点击【新建】，然后再点击【浏览】，选择你安装python是的文件夹
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6a25ecdeee1b4f2a84a09f32805b02d6.png"/>
        <br/>
        6、添加两个，一个是python的安装目录，一个是python下的scripts文件夹目录
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9eef435d05c44ae692f7081775c70577.png"/>
        <br/>
        7、添加完成后一定记得点击【确定】
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d6c9114fd2c946858d0da7af104a777c.png"/>
        <br/>
        7、继续点击【确定】
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bc229f00a8464cc697f18517767c355a.png"/>
        <br/>
        8、继续点击【确定】
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e568dcc58ff347e4bc643c1a70870577.png"/>
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="_44">
     </a>
     四、验证
    </h3>
    <h4>
     <a id="1win___Rcmd_45">
     </a>
     1：win + R，打开cmd命令行
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1611a1a55586431cb152f60253cfb329.png"/>
     <br/>
     输入
    </p>
    <pre><code>python --version
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c8da6fb1a54a4095b385fa32a04bda76.png"/>
     <br/>
     出现如图数据，代表 python3.8 安装完成
    </p>
    <h3>
     <a id="DjangoDjango_56">
     </a>
     五、安装Django（若无Django需求，不需要安装）
    </h3>
    <h4>
     <a id="1Django_58">
     </a>
     1、在命令行输入以下命令，安装最新版Django。
    </h4>
    <pre><code>py -m pip install Django
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c1e7c4663a6e4797b55dde6047b68414.png"/>
     <br/>
     安装完成后出现Successfully字样即安装成功。
    </p>
    <h4>
     <a id="2_67">
     </a>
     2、验证
    </h4>
    <p>
     在命令行依次输入以下命令

    <pre><code>python

import django

django.VERSION
</code></pre>
    <p>
     出现下图类似结果即安装成功
