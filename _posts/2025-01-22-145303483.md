    <p>
     双足机器人（也称为人形机器人或仿人机器人）是一个复杂的领域，涉及机械设计、电子工程、控制理论、计算机视觉等多个学科。对于想要探索或开发双足机器人的开发者来说，有许多开源项目可以提供帮助。这些项目通常包括硬件设计文件、固件代码以及高级软件框架，以实现运动控制、导航、感知等功能。
    </p>
    <h4 style="background-color:transparent">
     双足机器人开源项目推荐
    </h4>
    <h5>
     1.
     <strong>
      OpenHumanoids
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：由 Georgia Tech 的 AMBER 实验室开发的开源双足机器人平台。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：提供了详细的硬件设计文档和软件库，支持多种传感器和执行器。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://github.com/GeorgiaTechAMBER" title="OpenHumanoids GitHub">
       OpenHumanoids GitHub
      </a>
     </li>
    </ul>
    <h5>
     2.
     <strong>
      Cassie
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：Agility Robotics 开发的高性能双足机器人 Cassie 的开源版本。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：虽然 Cassie 本身不是完全开源的，但其控制算法和其他相关资源是公开的，并且可以为研究和教育目的使用。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://docs.agilityrobotics.com/" rel="nofollow" title="Cassie Documentation">
       Cassie Documentation
      </a>
     </li>
    </ul>
    <h5>
     3.
     <strong>
      ATRI
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：Advanced Telecommunications Research Institute International (ATR) 开发的小型双足机器人 ATRI。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：适合教育和研究用途，具有完整的开源硬件和软件包。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://github.com/ATR-Humanoid/atri" title="ATRI GitHub">
       ATRI GitHub
      </a>
     </li>
    </ul>
    <h5>
     4.
     <strong>
      Poppy Project
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：一个模块化的开源机器人平台，包括 Poppy Ergo Jr 和 Poppy Torso 等不同型号。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：特别适合教育和艺术应用，提供易于使用的 Python API 和丰富的社区支持。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="http://www.poppy-project.org/" rel="nofollow" title="Poppy Project Website">
       Poppy Project Website
      </a>
     </li>
    </ul>
    <h5>
     5.
     <strong>
      NAOqi SDK for NAO Robot
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：软银机器人公司提供的 NAO 机器人开发工具包。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：尽管 NAO 机器人本身不是免费的，但它的 SDK 是开源的，可用于学习和开发各种应用程序。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://developer.softbankrobotics.com/nao6/naoqi-sdk" rel="nofollow" title="NAOqi SDK">
       NAOqi SDK
      </a>
     </li>
    </ul>
    <h5>
     6.
     <strong>
      Robot Operating System (ROS) Packages
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：ROS 是一个广泛使用的机器人操作系统，拥有大量的社区贡献包，其中一些专门用于双足机器人。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：例如
      <code>
       bipedal-locomotion-framework
      </code>
      提供了针对双足行走的研究框架。
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://github.com/delphes/bipedal_locomotion_framework" title="ROS Bipedal Locomotion Framework">
       ROS Bipedal Locomotion Framework
      </a>
     </li>
    </ul>
    <h5>
    </h5>
    <p>
     大多数上述项目都托管在 GitHub 或其他代码托管平台上，您可以直接访问它们的官方页面下载源代码和相关文档。此外，许多项目还附带教程和示例程序，帮助新手快速上手。
    </p>
    <hr/>
    <h3>
     带大模型语音交互，大模型场景理解
    </h3>
    <p>
     带大模型语音交互和大模型场景理解的双足机器人项目是相对前沿的研究领域，结合了复杂的机械设计、先进的运动控制算法以及强大的人工智能技术。目前，这类高度集成的系统可能更多地出现在研究机构或大型科技公司的实验室中，但也有几个开源或部分开源的项目可以提供一定的功能和技术支持。
    </p>
    <h4>
     开源或部分开源项目
    </h4>
    <h5>
     1.
     <strong>
      Pepper and NAO by SoftBank Robotics
     </strong>
    </h5>
    <p>
    </p>
    <h5>
     3.
     <strong>
      Poppy Project with Cognitive Services
     </strong>
    </h5>
    <p>
    </p>
    <p>
    </p>
    <h5>
     4.
     <strong>
      Robot Operating System (ROS) + Hugging Face Transformers
     </strong>
    </h5>
    <p>
    </p>
    <p>
    </p>
    <h5>
     5.
     <strong>
      Cognitive Robotics at CMU
     </strong>
    </h5>
    <p>
    </p>
    <p>
    </p>
    <h5>
    </h5>
    <p>
     对于想要将大模型应用于双足机器人的开发者来说，最直接的方法可能是基于现有的开源框架（如 ROS）构建自己的系统，并结合像 Hugging Face 这样的 AI 模型库。这样不仅可以利用成熟的机器人控制技术，还能快速集成最新的 AI 成果。
    </p>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       简介
      </strong>
      ：虽然不是专门的双足机器人，但 Pepper 和 NAO 机器人在语音交互和场景理解方面表现出色，并且有丰富的社区资源。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        使用 NAOqi SDK 可以实现复杂的对话管理和环境感知。
       </li>
       <li>
        支持多种语言的自然语言处理（NLP）和语音识别。
       </li>
       <li>
        提供了用于开发高级应用的强大 API。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://developer.softbankrobotics.com/nao6/naoqi-sdk?spm=5176.28103460.0.0.21665d275vRQKf" rel="nofollow" title="NAOqi SDK">
       NAOqi SDK
      </a>
      <h5>
      </h5>
     </li>
     <li>
      <h5>
       2.
       <strong>
        MIT's HERMES Robot
       </strong>
      </h5>
     </li>
     <li>
      <strong>
       简介
      </strong>
      ：麻省理工学院开发的 HERMES 是一个专注于人机协作的双足机器人。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        结合了深度学习和强化学习来提高运动技能和环境互动能力。
       </li>
       <li>
        研究团队已经探索了使用大模型进行语音交互的可能性。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="http://groups.csail.mit.edu/robotics-center/main/hermes.php?spm=5176.28103460.0.0.21665d275vRQKf" rel="nofollow" title="HERMES Project">
       HERMES Project
      </a>
     </li>
     <li>
      <strong>
       简介
      </strong>
      ：Poppy 是一个模块化的开源机器人平台，特别适合教育和艺术应用。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        可以与 Microsoft Azure 或 Google Cloud 的认知服务集成，提供语音识别、文本到语音转换（TTS）、视觉识别等功能。
       </li>
       <li>
        社区活跃，提供了很多教程和示例代码。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="http://www.poppy-project.org/?spm=5176.28103460.0.0.21665d275vRQKf" rel="nofollow" title="Poppy Project Website">
       Poppy Project Website
      </a>
     </li>
     <li>
      <strong>
       简介
      </strong>
      ：通过 ROS 和 Hugging Face 的 Transformers 库相结合，可以在任何基于 ROS 的机器人平台上添加先进的自然语言处理和场景理解能力。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        利用 Hugging Face 提供的各种预训练模型，如 BERT、GPT 系列等，实现高质量的语音交互。
       </li>
       <li>
        可以利用 ROS 的传感器数据进行场景建模和理解。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <ul>
       <li>
        <a href="http://wiki.ros.org/" rel="nofollow" title="ROS">
         ROS
        </a>
       </li>
       <li>
        <a href="https://github.com/huggingface/transformers?spm=5176.28103460.0.0.21665d275vRQKf" title="Hugging Face Transformers">
         Hugging Face Transformers
        </a>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       简介
      </strong>
      ：卡内基梅隆大学的认知机器人实验室致力于开发能够理解和适应复杂环境的机器人系统。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        研究涉及大规模机器学习模型的应用，包括语音识别、对话管理、物体识别等。
       </li>
       <li>
        实验室的研究成果可能会被集成到实际的机器人平台中。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       链接
      </strong>
      ：
      <a href="https://www.ri.cmu.edu/research_areas/cognitive-robotics/?spm=5176.28103460.0.0.21665d275vRQKf" rel="nofollow" title="CMU Cognitive Robotics Lab">
       CMU Cognitive Robotics Lab
      </a>
     </li>
    </ul>
