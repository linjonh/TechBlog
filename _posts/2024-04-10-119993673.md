    <h2>
     <a id="_37">
     </a>
     部署教程：
    </h2>
    <h4>
     <a id="1__39">
     </a>
     1 源码导入微信开发者工具
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f4b8972fc0866eba9bbe7813a0fd5706.png"/>
    </p>
    <h4>
     <a id="2__45">
     </a>
     2 开通云开发环境
    </h4>
    <ul>
     <li>
      <p>
       参考微信官方文档：https://developers.weixin.qq.com/miniprogram/dev/wxcloud/basis/getting-started.html
      </p>
     </li>
     <li>
      <p>
       在使用云开发能力之前，需要先开通云开发。
      </p>
     </li>
     <li>
      <p>
       在开发者工具的工具栏左侧，点击 “云开发” 按钮即可打开云控制台，根据提示开通云开发，并且创建一个新的云开发环境。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b7f45675d38ab32afbdbec214a4f98e.png"/>
      </p>
     </li>
     <li>
      <p>
       每个环境相互隔离，拥有唯一的环境 ID(拷贝此ID，后面配置用到)，包含独立的数据库实例、存储空间、云函数配置等资源；
      </p>
     </li>
    </ul>
    <h5>
     <a id="3__54">
     </a>
     3 云函数及配置
    </h5>
    <ul>
     <li>
      <p>
       本项目使用到了一个云函数score_cloud
      </p>
     </li>
     <li>
      <p>
       在云函数cloudfunctions文件夹下选择云函数score_cloud , 右键选择在终端中打开,然后执行
      </p>
     </li>
     <li>
      <p>
       npm install –product
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/46d385fee54938fcde4f7fa92301f768.png"/>
      </p>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6828407b5954c44ccbb477fb89d42ef9.png"/>
    </p>
    <ul>
     <li>
      打开cloudfunctions/score_cloud/comm/ccmini_config.js文件，配置环境ID和后台管理员账号和密码
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/176066c42848dfff668318535c5690ae.png"/>
     </li>
    </ul>
    <h5>
     <a id="ExcelExcel_69">
     </a>
     本系统需要导入Excel文件，需要在云函数安装Excel内库支持，
    </h5>
    <ul>
     <li>
      在云函数cloudfunctions文件夹下选择云函数score_cloud , 右键选择在终端中打开,然后执行
     </li>
     <li>
      npm install node-xlsx
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7878d657d7d6793ac7b3f7ee8d1c1616.png"/>
    </p>
    <h5>
     <a id="4___76">
     </a>
     4 客户端配置
    </h5>
    <ul>
     <li>
      打开miniprogram/app.js文件，配置环境ID
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de4471d2dd38b24dea21cadae1f0561c.png"/>
     </li>
    </ul>
    <h5>
     <a id="5___83">
     </a>
     5 云函数配置
    </h5>
    <ul>
     <li>
      在微信开发者工具-》云开发-》云函数-》对指定的函数添加环境变量
     </li>
     <li>
      [服务端时间时区TZ] =&gt;Asia/Shanghai
     </li>
     <li>
      [函数内存] =&gt;128M
     </li>
     <li>
      [函数超时时间] =&gt; 60秒
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d0584dbfb123baeba6cb118e70c8405f.png"/>
    </p>
    <h5>
     <a id="6___93">
     </a>
     6 设置图片域名信任关系
    </h5>
    <ul>
     <li>
      进入小程序 开发管理=》开发设置=》服务器域名 =》downloadFile合法域名
     </li>
     <li>
      添加2个域名：
     </li>
     <li>
      1）你的云存储域名，格式类似：https://1234-test-pi5po-1250248.tcb.qcloud.la
     </li>
     <li>
      2）微信头像域名：https://thirdwx.qlogo.cn
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5d1ea89f11f9378cea9f247a300c478.png"/>
    </p>
