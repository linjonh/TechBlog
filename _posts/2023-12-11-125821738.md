     题目示例：
    </h3>
    <p>
     <strong>
      问题：求整数序列的元素最大跨度值
     </strong>
     <br/>
     <strong>
      描述：
     </strong>
     <br/>
     给定一个长度为n的非负整数序列，请计算序列的最大跨度值（最大跨度值 = 最大值减去最小值）。
     <br/>
     <strong>
      输入：
     </strong>
     <br/>
     一共2行，第一行为序列的个数n（1 &lt;= n &lt;= 1000)，第二行为序列的n个不超过1000的非负整数，整数之间以一个空格分隔。
     <br/>
     <strong>
      输出：
     </strong>
     <br/>
     输出一行，表示序列的最大跨度值。
     <br/>
     <strong>
      样例输入：
     </strong>
     <br/>
     6
     <br/>
     3 0 8 7 5 9
     <br/>
     <strong>
      样例输出：
     </strong>
     <br/>
     9
