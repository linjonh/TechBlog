      B．分式

      C．根式
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      二次根式命令：\sqrt{表达式} ．如果表达式是单个字符，则不需要花括号，但需要在字符和sqrt间加入一个空格。
      <br/>
      n次根式命令：\sqrt[n]{表达式}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      被开方表达式字符高度不一致时，根号上面的横线可能不在同一条直线上；为了使横线在同一直线上，可以在被开方表达式中插入一个只有高度没有宽度的数学支柱(\mathstrut)，例如：
      <br/>
      \[ \sqrt{a}+\sqrt{b}+\sqrt{c},     \qquad       \sqrt{\mathstrut a}+\sqrt{\mathstrut b}+\sqrt{\mathstrut c} \]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      (分别显示为：
      <img alt="[Latex学习笔记]数学公式基本命令 - 激进的猫 - 激进的猫" src="http://img0.ph.126.net/6s-HstJkvbpLyLDeKu7OWg==/3393462319323678201.jpg" style="border:0px; max-width:100%; margin:0px 10px 0px 0px"/>

     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      当被开方表达式较高时，开方次数的位置显得略低，解决办法为：将开方次数改为上标，并拉近与根式的水平距离，即将命令中的[n]改为[^n\!](^表示上标，间隔命令\!表示缩小间隔)，例如：
      <br/>
      \begin{eqnarray}
      <br/>
      \sqrt{1+\sqrt[p]{1+\sqrt[q]{1+a}}}\\
      <br/>
      \sqrt{1+\sqrt[^p\!]{1+\sqrt[^q\!]{1+a}}}
      <br/>
      \end{eqnarray}
     </p>

      举例：
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行内分式 \(\frac{x+y}{y+z} \)
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行间分式
      <br/>
      \[\frac{x+y}{y+z}\]
      <br/>
      上面的例子表明行内分式字体比行间分式字体小，因为行内分式使用的是角标字体。可以人工改变行内分式的字体大小，
      <br/>
      例如这个行内公式 $\displaystyle\frac{x+y}{y+z}$ 的大小和行间公式是一样的。
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      连分式：
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \begin{displaymath}
      <br/>
      x_0+\frac{1}{x_1+\frac{1}{x_2+\frac{1}{x_3+\frac{1}{x_4}}}}
      <br/>
      \end{displaymath}
      <br/>
      可以通过强制改变字体大小使得分子分母字体大小一致，例如：
      <br/>
      \newcommand{\FS}[2]{\displaystyle\frac{#1}{#2}}
      <br/>
      \begin{displaymath}
      <br/>
      x_0+\FS{1}{x_1+\FS{1}{x_2+\FS{1}{x_3+\FS{1}{x_4}}}}
      <br/>
      \end{displaymath}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      分数线长度值是预设为分子分母的最大长度，如果想要使分数线再长一点，可以在分子或分母两端添加一些间隔，例如：$\frac{1}{2},  \frac{\;1\;}{\;2\;}$
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \P\quad 3 根式
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      二次根式命令：$\backslash$sqrt\{表达式\} %%如果表达式是单个字符，则不需要花括号，但需要在字符和sqrt间加入一个空格
      <br/>
      n次根式命令：$\backslash$sqrt[n]\{表达式\}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      被开方表达式字符高度不一致时，根号上面的横线可能不在同一条直线上;为了使横线在同一直线上，可以在被开方表达式中插入一个
      <br/>
      只有高度没有宽度的数学支柱($\backslash$mathstrut)，例如：
      <br/>
      \[ \sqrt{a}+\sqrt{b}+\sqrt{c},\qquad \sqrt{\mathstrut a}+\sqrt{\mathstrut b}+\sqrt{\mathstrut c} \]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      当被开方表达式较高时，开方次数的位置显得略低，解决办法为：将开方次数改为上标，并拉近与根式的水平距离，即将命令中的$[n]$
      <br/>
      改为$[^n\!]$，例如：
      <br/>
      \begin{eqnarray}
      <br/>
      \sqrt{1+\sqrt[p]{1+\sqrt[q]{1+a}}}\\
      <br/>
      \sqrt{1+\sqrt[^p\!]{1+\sqrt[^q\!]{1+a}}}
      <br/>
      \end{eqnarray}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      命令$\backslash$surd生成根号上没有横线的根式，例如：$\surd{x+y+z}$
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \P\quad 4 求和与积分
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行内公式求和命令：$\backslash$ sum\_{}\{k=1\}\^{}n
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行内公式积分命令：$\backslash$ int\_{}a\^{}b
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行间公式求和命令：$\backslash$ [$\backslash$ sum\_{}\{k=1\}\^{}n ]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行间公式积分命令：$\backslash$ [$\backslash$ int\_{}a\^{}b ]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      例如：无穷级数$\sum_{k=1}^\infty \frac{x^n}{n!}$可化为积分$\int_0^\infty e^x$, 也即
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \[\sum_{k=1}^\infty \frac{x^n}{n!} =\int_0 ^\infty e^x\]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      改变上下限位置的命令：$\backslash$limits(强制上下限在上、下侧) 和 $\backslash$nolimits(强制上下限在右侧)
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行内公式上下限在积分、求和符号上侧：$\backslash$ sum$\backslash$limits\_{}\{k=1\}\^{}n 和 $\backslash$ int$\backslash$limits\_{}a\^{}b，例如$\sum\limits_{n=0}^\infty x^n$ 或 $\int\limits_a^b$
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      行间公式上下限在积分、求和符号右侧：$\backslash$[$\backslash$sum$\backslash$nolimits\_{}\{k=1\}\^{}n$\backslash$]，例如：
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \[\sum\nolimits_{k=1}^\infty x^n=\frac{1}{1+x}\]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \P\quad 5 下划线、上划线等
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      上划线命令：$\backslash$overline\{公式\}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      下划线命令：$\backslash$underline\{公式\}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      例如：$\overline{\overline{a^2}+\underline{ab}+\bar{a}^3}$
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      上花括弧命令：$\backslash$overbrace\{公式\}\^{}\{说明\}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      下花括弧命令：$\backslash$underbrace\{公式\}\_{}\{说明\}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      例如：
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \[ \underbrace{a+\overbrace{b+\dots+b}^{m\mbox{\scriptsize 个}}+c}_{20\mbox{\scriptsize 个}} \]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \P\quad 6 数学重音符号
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      %%以 a 为例；如果字母i或j带有重音，字母i、j应替换为\imath、\jmath
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \begin{tabbing}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      符号\hspace*{5bp} \= 命令\kill
      <br/>
      \(\hat{a}\) \&gt;  $\backslash$hat\{a\} \\
      <br/>
      \(\check{a}\) \&gt; $\backslash$check\{a\}\\
      <br/>
      \(\breve{a}\) \&gt; $\backslash$breve\{a\}\\
      <br/>
      \(\tilde{a}\) \&gt; $\backslash$tilde\{a\}\\
      <br/>
      \(\bar{a}\) \&gt; $\backslash$bar\{a\}\\
      <br/>
      \(\vec{a}\) \&gt; $\backslash$vec\{a\}\\
      <br/>
      \(\acute{a}\) \&gt; $\backslash$acute\{a\}\\
      <br/>
      \(\grave{a}\) \&gt; $\backslash$grave\{a\}\\
      <br/>
      \(\mathring{a}\) \&gt; $\backslash$mathring\{a\}\\
      <br/>
      \(\dot{a}\) \&gt; $\backslash$dot\{a\}\\
      <br/>
      \(\ddot{a}\) \&gt; $\backslash$ddot\{a\}\\
      <br/>
      \\
      <br/>
      \(\widehat{abc}\) \&gt; $\backslash$widehat\{abc\}\\
      <br/>
      \(\widetilde{xyz}\) \&gt; $\backslash$widetilde\{xyz\}
      <br/>
      \end{tabbing}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      <br/>
      \P\quad 7 堆积符号
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      符号堆积命令：$\backslash$stacrel\{上位符号\}\{基位符号\} %%基位符号大，上位符号小
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \qquad\qquad\qquad\quad \{上位公式 $\backslash$atop 下位公式\} %%上下符号一样大
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \qquad\qquad\qquad\quad \{上位公式 $\backslash$choose 下位公式\} %%上下符号一样大；上下符号被包括在圆括弧内
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      例如：
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      \begin{eqnarray*}
      <br/>
      \vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n}\\
      <br/>
      {n+1 \choose k}={n \choose k}+{n \choose k-1}\\
      <br/>
      \sum_{k_0,k_1,\ldots&gt;0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots
      <br/>
      \end{eqnarray*}
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      <br/>
      \P\quad 8 定界符
      <br/>
      \[
      <br/>
      ()
      <br/>
      \big(\big)
      <br/>
      \Big(\Big)
      <br/>
      \bigg(\bigg)
      <br/>
      \Bigg(\Bigg)
      <br/>
      \]
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
      自适应放大命令，自动随着公式内容大小调整符号大小：$\backslash$left 和 $\backslash$right
     </p>
     <p style="line-height:28px; margin-top:0px; margin-bottom:10px; padding-top:0px; padding-bottom:0px; color:rgb(56,56,56); font-family:Arial,Helvetica,simsun,u5b8bu4f53; font-size:16px; background-color:rgb(198,201,206)">
