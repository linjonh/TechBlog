    <p>
     <strong>
      VMware16虚拟机安装
     </strong>
    </p>
    <p>
     VMware16安装好之后，下载好我们要安装的操作系统的镜像文件，这里以centos7版本为例，就可以开始安装了。
    </p>
    <p>
     1、在主页选择创建新的虚拟机。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2f28cedc3ccd184d54f36d9559533ac2.png#pic_center">
      <br/>
      2、选择典型安装（自定义安装也可，有些小的功能可以选择）。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b08ae3d47a252d46f416e9962afb184.png#pic_center">
       <br/>
       3、选择稍后安装操作系统（方便后续安装可视化界面）。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/16122fe9d7bafe8e4c4b602a4b8bf0eb.png#pic_center">
        <br/>
        4、选择已经下载好的操作系统和版本（这里是Linux的CentOS 7）。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8587237d0dbace3aecfc7ee562cbd7ef.png#pic_center">
         <br/>
         5、给虚拟机命名并选择系统选择存储位置（可以默认）。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/57b2e00cf3eb89478f71f1bceb274ddc.png#pic_center">
          <br/>
          6、指定磁盘容量一般选择默认（虚拟磁盘存储为单个或者多个文件一般影响不大）。
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9563b733e6d661760577225fda123347.png#pic_center">
           <br/>
           7、选择自定义硬件，一般根据自己电脑设置（一般内存不能超过笔记本电脑内存的1/2，此处8G的电脑内存不能超过4G的内存；此处处理器4核8线程不能超过2核4线程）。
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e7bcd93616aa947b4595a9464b3da2f9.png#pic_center">
            <br/>
            8、CD/DVD(IDE)设置,选择使用ISO映像文件，填写CentOS7系统的保存路径。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3cb0eb3b591ee46f1d7cfd0d6a0b20d1.png#pic_center"/>
            <br/>
            9、这样就完成硬件设置了，同时已准备好创建虚拟机。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e3bf26bf3d083de5df53b4b32e8e15b4.png#pic_center"/>
            <br/>
            10、开启此虚拟机。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9c432de9bf4ff32b9f77a6c54fd2bf93.png#pic_center"/>
            <br/>
            11、选择install CentOS 7,等待安装。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3ffcb329ee48e76678a961518106ccb7.png#pic_center"/>
            <br/>
            12、选择安装中文语言。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b0acf02fccb198352762410a8cee098f.png#pic_center"/>
            <br/>
            13、点击软件选择，选择GNOME桌面。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/06be8c4921c4d2265b5e70b5ab210b3e.png#pic_center"/>
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/55fe1e3b833f1f22554e718cf7ca6360.png#pic_center"/>
            <br/>
            14、设置密码
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c60f35b046b1f0bfdf5a7750dfb0fad6.png#pic_center"/>
            15、等待安装。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/77489a716c2e9c8053e228685e6bbb7c.png#pic_center"/>
            16、完成，点击重启（还有一些用户名的设置等等，可根据提示完成）。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eaa53a664e5359794faaaecc188c7d9f.png#pic_center"/>
           </img>

     </img>
