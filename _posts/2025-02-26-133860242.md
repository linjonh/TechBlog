    <h3>
     <a id="_0">
     </a>
     前言
    </h3>
    <p>
     Java是一门十分流行的编程语言，许多程序员在学习编程时都会选择Java作为入门的第一门语言。本篇文章旨在为零基础的读者提供一些基础的语法知识并帮助他们开始编写Java程序。
    </p>
    <h3>
     <a id="Java_4">
     </a>
     Java语句
    </h3>
    <p>
     本文将介绍Java的语句，其中包括：赋值语句、条件语句、循环语句和跳转语句。我们将通过实例来演示这些语句的使用，并提供测试用例以帮助读者验证自己的代码。通过本文的学习，读者将会掌握Java程序中最基本的控制结构。
    </p>
    <h3>
     <a id="_8">
     </a>
     内容
    </h3>
    <h4>
     <a id="_10">
     </a>
     赋值语句
    </h4>
    <p>
     在Java中，赋值语句用来将一个值赋给一个变量。这个值可以是一个常量或者是一个表达式的结果。下面是一个简单的赋值语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">int</span> x <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     这条语句将整数值10赋给了变量x。在Java中，赋值语句使用等号（=）来表示。
    </p>
    <h4>
     <a id="_20">
     </a>
     条件语句
    </h4>
    <p>
     条件语句允许我们根据某个条件来选择不同的行为。在Java中，有两种基本的条件语句：if语句和switch语句。
    </p>
    <h5>
     <a id="if_24">
     </a>
     if语句
    </h5>
    <p>
     if语句用来测试一个布尔表达式的值，如果表达式为true，则执行if语句后面的语句块。下面是一个简单的if语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">int</span> x <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
<span class="token keyword">if</span><span class="token punctuation">(</span>x <span class="token operator">&gt;</span> <span class="token number">5</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is greater than 5"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="switch_35">
     </a>
     switch语句
    </h5>
    <p>
     switch语句用来根据不同的值来执行不同的操作。下面是一个简单的switch语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">int</span> x <span class="token operator">=</span> <span class="token number">2</span><span class="token punctuation">;</span>
<span class="token keyword">switch</span><span class="token punctuation">(</span>x<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">case</span> <span class="token number">1</span><span class="token operator">:</span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is 1"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token keyword">case</span> <span class="token number">2</span><span class="token operator">:</span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is 2"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token keyword">default</span><span class="token operator">:</span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is not 1 or 2"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_53">
     </a>
     循环语句
    </h4>
    <p>
     循环语句用来重复执行一段代码，直到某个条件被满足为止。在Java中，有三种基本的循环语句：while循环、do-while循环和for循环。
    </p>
    <h5>
     <a id="while_57">
     </a>
     while循环
    </h5>
    <p>
     while循环用来重复执行一段代码，直到某个条件不再成立为止。下面是一个简单的while循环的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">int</span> x <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token keyword">while</span><span class="token punctuation">(</span>x <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is "</span> <span class="token operator">+</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>
    x<span class="token operator">++</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="dowhile_69">
     </a>
     do-while循环
    </h5>
    <p>
     do-while循环和while循环的区别在于，do-while循环先执行一次代码块，然后再判断条件是否成立。下面是一个简单的do-while循环的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">int</span> x <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token keyword">do</span> <span class="token punctuation">{<!-- --></span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"x is "</span> <span class="token operator">+</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>
    x<span class="token operator">++</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token keyword">while</span><span class="token punctuation">(</span>x <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="for_81">
     </a>
     for循环
    </h5>
    <p>
     for循环用来重复执行一段代码，通常用于已知循环次数的情况。下面是一个简单的for循环的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">for</span><span class="token punctuation">(</span><span class="token keyword">int</span> i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"i is "</span> <span class="token operator">+</span> i<span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_91">
     </a>
     跳转语句
    </h4>
    <p>
     跳转语句用来改变程序的执行顺序。在Java中，有三种基本的跳转语句：break语句、continue语句和return语句。
    </p>
    <h5>
     <a id="break_95">
     </a>
     break语句
    </h5>
    <p>
     break语句用来跳出一个循环或switch语句。下面是一个简单的break语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">for</span><span class="token punctuation">(</span><span class="token keyword">int</span> i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">if</span><span class="token punctuation">(</span>i <span class="token operator">==</span> <span class="token number">5</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"i is "</span> <span class="token operator">+</span> i<span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="continue_108">
     </a>
     continue语句
    </h5>
    <p>
     continue语句用来跳过当前循环的剩余部分，并开始下一次循环。下面是一个简单的continue语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">for</span><span class="token punctuation">(</span><span class="token keyword">int</span> i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">if</span><span class="token punctuation">(</span>i <span class="token operator">==</span> <span class="token number">5</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">continue</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
    <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"i is "</span> <span class="token operator">+</span> i<span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="return_121">
     </a>
     return语句
    </h5>
    <p>
     return语句用来结束当前方法并返回一个值。下面是一个简单的return语句的例子：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">int</span> <span class="token function">add</span><span class="token punctuation">(</span><span class="token keyword">int</span> x<span class="token punctuation">,</span> <span class="token keyword">int</span> y<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> x <span class="token operator">+</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
