    <h2>
     <a id="1__1">
     </a>
     1. 任务背景
    </h2>
    <p>
     FISCO BCOS其实是有控制台管理工具，用来对区块链系统进行各种管理操作。
    </p>
    <h2>
     <a id="2__3">
     </a>
     2. 任务目标
    </h2>
    <p>
     配置并使用控制台对联盟链进行初步操作
    </p>
    <h2>
     <a id="3__5">
     </a>
     3. 相关知识点
    </h2>
    <p>
     在控制台链接FISCO BCOS节点，可以实现查询区块链状态、部署调用合约等功能，能够快速获取到所需要的信息。
    </p>
    <h2>
     <a id="4__7">
     </a>
     4. 任务实操
    </h2>
    <h3>
     <a id="41_JDK_8">
     </a>
     4.1 安装配置JDK
    </h3>
    <p>
     控制台依赖JDK，注意版本Oracle JDK 8 至JDK 14。
    </p>
    <p>
     使用FireFox从以下地址下载
     <code>
      jdk-8u202-linux-x64.tar.gz
     </code>
     ，当然其他符合条件的版本也行。
    </p>
    <pre><code>官网下载地址：https://www.oracle.com/java/technologies/downloads/archive/
镜像下载地址：https://repo.huaweicloud.com/java/jdk/8u202-b08/
</code></pre>
    <p>
     默认下载保存目录是
     <code>
      /home/chen/下载
     </code>
     ，所以执行下面命令，解压到
     <code>
      /usr/local
     </code>
    </p>
    <pre><code class="prism language-shell"><span class="token function">sudo</span> <span class="token function">tar</span> -zxvf /home/chen/下载/jdk-8u202-linux-x64.tar.gz -C /usr/local
</code></pre>
    <p>
     此时解压后jdk目录为
     <code>
      /usr/local/jdk1.8.0_202
     </code>
     ，如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6fc2c75bc24217be3563e4e89462f6e8.png">
      <br/>
      然后设置环境变量，执行
      <code>
       nano ~/.bashrc
      </code>
      ，修改配置文件，在开头添加下面内容后按
      <code>
       Ctrl+x
      </code>
      然后按
      <code>
       y
      </code>
      后回车保存。
     </img>
    </p>
    <pre><code>export JAVA_HOME=/usr/local/jdk1.8.0_202
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH
</code></pre>
    <p>
     最后执行
     <code>
      source ~/.bashrc
     </code>
     使配置马上生效，并可运行
     <code>
      java -version
     </code>
     查看安装是否成功。如下图为安装配置成功。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8506a15caadff36aa4ef36de02f32ab5.png"/>
    </p>
    <h3>
     <a id="42__31">
     </a>
     4.2 下载控制台程序
    </h3>
    <pre><code class="prism language-shell"><span class="token function">mkdir</span> /opt/fisco-console
<span class="token builtin class-name">cd</span> /opt/fisco-console
<span class="token function">curl</span> -<span class="token comment">#LO https://gitee.com/FISCO-BCOS/console/raw/master-2.0/tools/download_console.sh &amp;&amp; bash download_console.sh</span>
</code></pre>
    <h3>
     <a id="43___38">
     </a>
     4.3 拷贝配置程序
    </h3>
    <pre><code class="prism language-shell"><span class="token builtin class-name">cd</span> /opt/fisco-console
<span class="token function">cp</span> -n console/conf/config-example.toml console/conf/config.toml
</code></pre>
    <h3>
     <a id="44__44">
     </a>
     4.4 配置控制台证书
    </h3>
    <pre><code class="prism language-shell"><span class="token function">cp</span> -r /opt/fisco/nodes/127.0.0.1/sdk/* /opt/fisco-console/console/conf/
</code></pre>
    <h3>
     <a id="45_FISCO_BCOS_48">
     </a>
     4.5 启动FISCO BCOS
    </h3>
    <p>
     在开始之前，执行如下命令确保已启动FISCO BCOS
    </p>
    <pre><code class="prism language-shell"><span class="token function">bash</span> /opt/fisco/nodes/127.0.0.1/start_all.sh
</code></pre>
    <h3>
     <a id="46__53">
     </a>
     4.6 启动控制台
    </h3>
    <pre><code class="prism language-shell"><span class="token function">bash</span> /opt/fisco-console/console/start.sh
</code></pre>
    <p>
     输出如下信息表示成功：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b3a4a7b4d29c733d87727ecf0226ab4.png"/>
    </p>
    <h3>
     <a id="47__59">
     </a>
     4.7 使用控制台获取信息
    </h3>
    <pre><code class="prism language-shell"><span class="token comment"># 获取客户端版本</span>
<span class="token punctuation">[</span>group:1<span class="token punctuation">]</span><span class="token operator">&gt;</span> getNodeVersion
<span class="token comment"># 获取节点信息</span>
<span class="token punctuation">[</span>group:1<span class="token punctuation">]</span><span class="token operator">&gt;</span> getPeers
</code></pre>
    <h2>
     <a id="5__68">

     5. 小结
    </h2>

     控制台用起来还是可以的，YYDS
