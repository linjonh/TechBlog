    <h2>
     <a id="grep_1">
     </a>
     grep命令介绍
    </h2>
    <p>
     Linux操作系统中 grep 命令用于查找指定文件目录下文件里符合条件的字符串， grep 指令会把含有关键字的那一行显示出来。若不指定任何文件名称，则当前目录就是grep命令的需要查找的文件的目录，若所给予的文件名为 -，则 grep 命令会从标准输入设备读取数据。
    </p>
    <h3>
     <a id="_3">
     </a>
     参数选项
    </h3>
    <blockquote>
     <p>
      -r 是递归查找
      <br/>
      -n 是显示行号
      <br/>
      -R 查找所有文件包含子目录
      <br/>
      -i 忽略大小写
      <br/>
      -o 只输出匹配的内容（一般用于获取关键字在文件中出现的次数查找）
      <br/>
      -w 按单词位单位过滤
      <br/>
      -v 取反（获取不包含关键字的文件内容）
     </p>
    </blockquote>
    <h2>
     <a id="grep_12">
     </a>
     grep命令的使用
    </h2>
    <h3>
     <a id="1__14">
     </a>
     1. 在指定的文件中查找包含的关键字
    </h3>
    <p>
     <strong>
      查找方法：
     </strong>
    </p>
    <blockquote>
     <p>
      cat 文件名 | grep ‘关键字’ 或 grep ‘关键字’ 文件名
     </p>
    </blockquote>
    <p>
     事例：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> /airport/logs/202306272328/airport_admin/log_info.log<span class="token operator">|</span><span class="token function">grep</span> <span class="token string">'结束执行新增'</span>
<span class="token comment">#或</span>
<span class="token function">grep</span> <span class="token string">'结束执行新增'</span> /airport/logs/202306272328/airport_admin/log_info.log
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4e8fc1dfcad582536d9a9c23c9f8e6e6.png"/>
    </p>
    <h3>
     <a id="2__27">
     </a>
     2. 在指定目录下多个文件内容中查找包含的关键字
    </h3>
    <p>
     <strong>
      查找方法:
     </strong>
    </p>
    <blockquote>
     <p>
      grep -r ‘关键字’ 目录
     </p>
    </blockquote>
    <p>
     事例：
    </p>
    <pre><code class="prism language-bash"><span class="token function">grep</span> -r <span class="token string">'结束执行新增'</span> /airport/logs/202306272328
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6b0ce88576d37923ea74614fc0981b06.png">

     </img>
    </p>
    <h3>
     <a id="3_38">
     </a>
     3.在追加的文件内容中查找关键字
    </h3>
    <p>
     <strong>
      查找方法
     </strong>
     ：
    </p>
    <blockquote>
     <p>
      查找的文件实时更新，查找的关键字内容也在实时更新
     </p>
    </blockquote>
    <pre><code class="prism language-bash">  <span class="token function">tail</span> -f log_info.log <span class="token operator">|</span> <span class="token function">grep</span> <span class="token string">'销售订单'</span>
</code></pre>
    <p>
     <img alt="tail -f xxxxx.log | grep keyword 销售订单" src="https://i-blog.csdnimg.cn/blog_migrate/23bacb92bc33b8d5bac161c4501b569f.png"/>
    </p>
    <h3>
     <a id="4__47">
     </a>
     4. 统计文件中关键字出现的次数
    </h3>
    <p>
     <strong>
      查找方法:
     </strong>
    </p>
    <blockquote>
     <p>
      cat xxx.log | grep -o ‘关键字’ | wc -l
     </p>
    </blockquote>
    <p>
     事例：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> log_info.log <span class="token operator">|</span> <span class="token function">grep</span> -o <span class="token string">'销售订单'</span> <span class="token operator">|</span> <span class="token function">wc</span> -l
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/107825f890ff184d1add92de046369db.png"/>
    </p>
    <h3>
     <a id="5__vivim_57">
     </a>
     5. vi或vim打开的文件查找关键字(补充)
    </h3>
    <p>
     <strong>
      查找方法：
     </strong>
    </p>
    <blockquote>
     <p>
      在vi/vim命令行模式下直接输入： /关键字 （区分大小写）
      <br/>
      然后回车，定位到第一个关键字，之后通过n键向下查找，通过N键向上查找。
     </p>
    </blockquote>
