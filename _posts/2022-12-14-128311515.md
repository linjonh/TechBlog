    <h3>
     <a id="_9">
     </a>
     准备工作
    </h3>
    <ol>
     <li>
      注册一个小程序（一定得是企业认证账号，个人账号无法使用
      <code>
       web-view
      </code>
      ），具体注册就不演示了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5f011bd68577d916ce9e31d5e035883e.png"/>
     </li>
     <li>
      国内的备案域名（你
      <code>
       H5
      </code>
      页面和后台
      <code>
       API
      </code>
      域名）
     </li>
     <li>
      部署SSL证书（
      <code>
       web-view
      </code>
      必须使用https）
     </li>
    </ol>
    <h3>
     <a id="_15">
     </a>
     开始制作
    </h3>
    <ol>
     <li>
      <p>
       配置服务器域名
       <br/>
       开发管理 —— 开发设置 —— 服务器域名（主要是配置request）
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6bdaf916028e1097a6ed9206e46abdfb.png">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/462a700732ae209c940591499f62f85c.png">
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d13215f78a84d37ee1083c15b3f025c0.png">
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c33ff2993b646563a0508c37e98e4785.png"/>
         </img>
        </img>
       </img>
      </p>
     </li>
     <li>
      <p>
       配置业务域名
       <br/>
       开发管理 —— 开发设置 —— 业务域名（不配置
       <code>
        web-view
       </code>
       无法加载H5页面）
       <br/>
       <strong>
        注意：如果这里不配置，页面会提示 “小程序不支持打开非业务域名请重新配置”
       </strong>
       <br/>
       <strong>
        注意：个人认证没有这个配置，个人认证无法使用
        <code>
         web-view
        </code>
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/06bc91c3771cac569184d1a4c96fcac0.png"/>
      </p>
     </li>
     <li>
      <p>
       创建小程序
       <br/>
       在开发者设置里，找到
       <code>
        AppID
       </code>
       ，填写进创建小程序内的
       <code>
        AppID
       </code>
       ，然后模板就选择基础模板就可以了。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/19ad589c8d6206a775b97e031880ffac.png"/>
      </p>
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5fcd28f6399e6f6ade7b1b68feb6f9eb.png"/>
    </p>
    <ol start="4">
     <li>
      修改页面（加入
      <code>
       web-view
      </code>
      ）
      <br/>
      打开
      <code>
       page/index/index.wxml
      </code>
      文件，将里面的内容都删除，只填写下面内容。
      <br/>
      在
      <code>
       web-view
      </code>
      标签里面 填写H5页面地址即可。
     </li>
    </ol>
    <pre><code class="prism language-css">&lt;view&gt;
     &lt;web-view src=<span class="token string">"https://xxxxx"</span>&gt;&lt;/web-view&gt; 
&lt;/view&gt;
</code></pre>
    <p>
     这样就开发完成了，可以上传审核发布，也可以真机调试了。
    </p>
