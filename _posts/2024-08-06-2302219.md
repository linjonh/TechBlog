     </span>
     <span lang="EN-US" style="FONT-FAMILY: SimSun">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">

     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      是开放源码的项目，因此可以到
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      www.eclipse.org
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      下载
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse SDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。它包括
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      开发环境、
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      开发环境、
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Plug-in
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      开发环境及其所有的源代码和文档。但如果只是进行
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      应用，而不是插件开发或者研究
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      代码，则只需要下载
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Platform Runtime Binary+JDT Runtime Binary
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      下载之后，直接解压到指定目录即可，无需安装。但这之前，请确定你已安装好了
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，以及配置好了
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的环境变量。第一次使用，双击运行，会提示你设定工作空间
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      (Workspace)
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，它将作为文件存储的位置。此外，中文用户还可以安装
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      多国语言包插件，该插件可以根据操作系统的语言环境选择对应的语言包，对
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      进行本地区域化。可以到
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      www.eclipse.org
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      下载对应版本的语言包。下载之后，需要在关闭
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的前提下：
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      先对压缩包解压。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      然后，将获得的文件保存到
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      安装目录的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Language
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      子目录下。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      接着，还需要在
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的安装目录下新创建一个目录
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      links
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，并在其下新建一个文本文件，再修改其文件名及扩展名为
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      language.strat
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      （文件名可以任意，不一定要是
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      language
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ）。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      最后，打开这个文件键入
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      path=d://eclipse//language(
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      即多国语言包
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      language
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的目录路径
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      )
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      至此，重新启动
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，就进入了中文环境。如果，在这之前曾使用过
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，还可能会有部分英文。可以删除
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      configuration
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      子目录下面的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      org.eclipse.update
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      目录，以删除原有的记录信息后，再启动即可。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      为了使编写的程序文件都能正常的现实中文信息，需要设置文本文件编码类型为
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      UTF8
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。方法是选择
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      窗口
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      -&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      首选项
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，在弹出对话框中选择
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      常规
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      -&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      编辑器
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，在右侧文本文件编码设定中的“其它”中，从下拉列表选项中选择
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      UTF-8
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; FONT-FAMILY: SimSun; mso-outline-level: 1">

    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      2
     </span>
     <span lang="EN-US" style="FONT-FAMILY: SimSun">
      .Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      集成开发环境介绍
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
      <span lang="EN-US" style="FONT-FAMILY: Verdana">
       Eclipse
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
       开发环境在启动后，就进入了
      </span>
      <span lang="EN-US" style="FONT-FAMILY: Verdana">
       Eclipse
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
       的默认视图——
      </span>
      <span lang="EN-US" style="FONT-FAMILY: Verdana">
       Java
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
       透视图。
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: 宋体">
       可以
      </span>
      <span lang="EN-US" style="FONT-FAMILY: 宋体">
       浏览一下帮助中的《工作台用户指南》，熟悉一下概念
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: 宋体">
       。
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
       透视图包括以下
      </span>
      <span lang="EN-US" style="FONT-FAMILY: Verdana">
       4
      </span>
      <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
       个部分
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ：
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      编辑器窗格：代码编写和调试的核心区域。开发人员可以方便地进行编写和查看，并获得
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      提供的提示和帮助。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      包资源管理器窗格：显示在
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      中进行开发的项目的资源，随着开发的项目类型的不同而改变，例如开发
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目时，就显示包资源管理器。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      大纲窗格：以层次结构清晰地显示出程序源代码的基本结构，有利于程序的迅速定位和查看。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      任务窗格：在项目开发过程中会有不同的选项卡，例如：控制台、
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JavaDoc
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      文档等。通过这个窗格的信息，开发者可以在编写过程中或运行过程中，查看到程序的错误提示或者控制台输出等信息。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; FONT-FAMILY: SimSun; mso-outline-level: 1">

    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      3.
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      创建
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ■
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      直接创建：
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      在
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      中使用
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      开发工具进行
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目开发，简称为
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDT
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的开发。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      首先，要确认
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      中与
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      相关的环境变量是否正确：打开
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      窗口
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      -&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      首选项
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，选择树形窗格中的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      Java-&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      已安装的
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      JRE
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，检查对应路径的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JRE
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      安装。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      然后，再在菜单中选择
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      文件
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      -&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      新建
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      -&gt;
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      项目
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ，在弹出对话框向导中，选择向导中的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
      Java-&gt;Java
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      项目
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      接着，进入“下一步”，输入项目名称，其他可以默认，也可以设置项目创建的内容位置或来源、
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      一直性、项目布局等细节。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      再进入“下一步”，确认与项目相关的各个构建的设置，要注意检查以一下“
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      库
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      ”选项卡中与
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      相关的制作。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      最后，确定。如果原先的视图不是
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      透视图，屏幕会弹出对话框，提示你切换到
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      透视图，单击“是”。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      创建完成后，会发现由于是
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目，在包资源管理器中有相关的包资源的树状结构，即
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      JDK
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      相关的包。同时，在
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      的工作空间所在的硬盘目录中，
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      创建了项目对应的文件夹，生成了与项目相关的辅助文件。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; FONT-FAMILY: SimSun; mso-outline-level: 1">
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ■
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      打开已有
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      如果要把已有的
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Java
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      项目导入到
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      开发环境。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      在
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      包资源管理器
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      窗格空白处单击右键，在快捷菜单中选择
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      导入
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      在弹出的对话框向导中选择
     </span>
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      现有项目到工作空间
     </span>
     <span lang="EN-US" style="FONT-WEIGHT: bold; FONT-FAMILY: Verdana">
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      点击下一步，浏览选择已有项目文件夹目录位置。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">
     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      <span style="mso-spacerun: yes">
      </span>
     </span>
     <span lang="ZH-CN" style="FONT-WEIGHT: bold; FONT-FAMILY: SimSun">
      ·
     </span>
     <span lang="ZH-CN" style="FONT-FAMILY: SimSun">
      确定，即导入完成。
     </span>
    </p>
    <p style="FONT-SIZE: 10pt; MARGIN: 0in; mso-outline-level: 1">

      <span style="mso-spacerun: yes">
      </span>
     </span>

     <span lang="EN-US" style="FONT-FAMILY: Verdana">
      Eclipse
     </span>
