     <strong>
      一、form表单提交（常用）
     </strong>
    </h3>
    <p>
     from
     <a href="https://so.csdn.net/so/search?q=%E8%A1%A8%E5%8D%95&amp;spm=1001.2101.3001.7020" title="表单">
      表单
     </a>
     把所有属于表单中的内容提交给后台，例如输入框，单选框，多选框，文本域，文件域等。
    </p>
    <ol>
     <li>
      在后台可通过对应的name属性获取相应的值。
     </li>
     <li>
      from表单中的action属性标识提交数据的地址。
     </li>
     <li>
      method属性指明表单提交的方式。
     </li>
    </ol>
    <pre><code>&lt;form action="demo.do" method="post"&gt;
    用户名：&lt;br&gt;
    &lt;input type="text" name="username"&gt;&lt;br&gt;
    密码:&lt;br&gt;
    &lt;input type="password" name="password" &gt;&lt;br&gt;&lt;br&gt;
    &lt;input type="submit" value="提交"&gt;
&lt;/form&gt;
</code></pre>
    <h3 id="%E4%BA%8C%E3%80%81JQuery%E4%B8%AD%E7%9A%84ajax%E6%8F%90%E4%BA%A4%EF%BC%88%E5%B8%B8%E7%94%A8%EF%BC%89">

     </a>
     <a id="JQueryajax_16">
     </a>
     二、JQuery中的ajax提交（常用）
    </h3>
    <p>
     JavaScript中也有ajax提交，但是代码太多，所以JQuery对JS中的ajax进行了简化。引入JQuery相应的包即可使用。一般格式为：
    </p>
    <pre><code> $.ajax({
     url: "TestJsonServlet", //提价的路径
     type: "post",       //提交方式
     data: {
         //向后台提交的数据
     },
     dataType: "JSON",       //规定请求成功后返回的数据
     success: function (data) {
         //请求成功之后进入该方法，data为成功后返回的数据
     },
     error: function (errorMsg) {
         //请求失败之后进入该方法，errorMsg为失败后返回的错误信息
     }
 });
</code></pre>
    <p>
     总结：以上两种方式如果不显示的指定post提交方式，则默认的提交方式为get方式提交。此外，ajax中的url也可以直接通过字符串拼接，然后向后台提交数据，这种方式为get方式提交。下面详细说明
    </p>
    <h3 id="%E4%B8%89%E3%80%81%E9%80%9A%E8%BF%87url%E5%AD%97%E7%AC%A6%E4%B8%B2%E6%8B%BC%E6%8E%A5%E5%90%91%E5%90%8E%E5%8F%B0%E6%8F%90%E4%BA%A4%E6%95%B0%E6%8D%AE">
     <a name="t2">
     </a>
     <a id="url_36">
     </a>
     三、通过url字符串拼接向后台提交数据
    </h3>
    <ol>
     <li>
      直接在ajax中url拼接数据
     </li>
    </ol>
    <pre><code>$.ajax({
     url: "TestJsonServlet?id="+id+"&amp;gender="+"男", //提价的路径
     type: "get",       //提交方式
     dataType: "JSON",       //规定请求成功后返回的数据
     success: function (data) {
         //请求成功之后进入该方法，data为成功后返回的数据
     },
     error: function (errorMsg) {
         //请求失败之后进入该方法，errorMsg为失败后返回的错误信息
     }
 });

</code></pre>
    <ol>
     <li>
      JS提交数据，通过window.location.href指定路径提交数据。
     </li>
    </ol>
    <pre><code>var deleteUser = function (deleteId) {
    if (confirm("确认删除编号是【"+deleteId+"】的成员吗?")){
        window.location.href="DeleteUserServlet?deleteId="+deleteId;
    }
}
</code></pre>
    <ol>
     <li>
      通过a标签提交数据，通过a标签的href属性提交数据，和js提交数据类似。
     </li>
    </ol>
    <pre><code>&lt;a href="DeleteUserServlet?id='3'&amp;gender='男'"&gt;&lt;/a&gt;
</code></pre>
    <ol>
     <li>
      在后台中也可以互相访问相应的Servlet
     </li>
    </ol>
    <pre><code>request.setAttribute("id","3");
request.setAttribute("gender","男");
request.getRequestDispatcher("updateUser.jsp").forward(request,response);
</code></pre>
    <p>
     然后在对应的Servlet中接受参数，进行相应的处理
    </p>
    <pre><code> String id = request.getAttribute("id");
 String gender = request.getAttribute("gender");</code></pre>
