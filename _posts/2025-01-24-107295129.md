    <blockquote>
     <p>
      在学习使用华为云平台的过程中，使用MQTT客户端——MQTT.fx来将设备当前所处的状态作为MQTT主题发送给lot DA，在此之前我们先来学习如何使用MQTT.fx接入华为云lot DA
     </p>
     <p>
      主要步骤分为：
      <br/>
      一、创建互联网产品设备
      <br/>
      二、MQTT.fx接入前准备
      <br/>
      三、MQTT.fx 接入华为云lot DA
     </p>
    </blockquote>
    <blockquote>
     <p>
      1、MQTT.fx下载链接：
      <a href="http://www.jensd.de/apps/mqttfx/1.5.0/" rel="nofollow">
       link
      </a>
      <br/>
      2、CA证书百度网盘下载链接：
      <a href="https://pan.baidu.com/s/1xGVuRtbKADEn8v4mEgF1jA" rel="nofollow">
       link
      </a>
      <br/>
      提取码：csq1
      <br/>
      3、华为云连接信息生成工具：MqttClientIdGenerator.jar
      <br/>
      百度网盘下载链接：
      <a href="https://pan.baidu.com/s/1vTvvxL3kKnFSkNBPfmOXMQ" rel="nofollow">
       link
      </a>
      <br/>
      提取码：3f6g
      <br/>
      <strong>
       注：在window操作系统下运行jar包，需要先安装jdk1.7或以上版本（下载jdk），配置环境变量
      </strong>
     </p>
    </blockquote>
    <h3>
     <a id="_15">
     </a>
     一、创建互联网产品设备
    </h3>
    <p>
     <strong>
      1、注册 -&gt; 登录 -&gt; 服务列表 -&gt; 设备接入lot DA -&gt; 产品 -&gt; 创建产品
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1dd29eaf7d88ba3151355e8b317bb70e.png#pic_center">
      <br/>
      <strong>
       2、创建产品
      </strong>
      <br/>
      ①所属资源空间：华为云平台默认为用户划分一个资源空间，供用户在平台中创建资源（如：产品、设备等）
      <br/>
      ②产品名称：用户自定义产品名称
      <br/>
      ③协议类型：我们选择MQTT协议
      <br/>
      ④数据格式：选择JSON
      <br/>
      ⑤厂商名称：用户可自定义厂商名称
      <br/>
      ⑥选择模型：华为云在此为用户提供了免费的产品模型，用户可根据需求选择模型
      <br/>
      ⑦所属行业：用户根据需求自行选择
      <br/>
      ⑧设备类型：用户可自定义设备类型名称
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e691f04a241c9fbb7bb300e2d5cffcec.png#pic_center">
       <br/>
       <strong>
        3、创建设备 -&gt; 真实设备
       </strong>
       <br/>
       此处我们选择真实设备，如果选择虚拟设备，则云平台自动注册一个虚拟设备供用户直接在线试调，无需使用MQTT.fx接入平台
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b9f39ac91946bbebfb5dd5e7182d144d.png#pic_center"/>
      </img>
     </img>
    </p>
    <p>
     <strong>
      4、创建完成
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/44cab21e1a53396ae5c4e4e89898ef57.png#pic_center"/>
    </p>
    <h3>
     <a id="MQTTfx_35">
     </a>
     二、MQTT.fx接入前准备
    </h3>
    <p>
     <strong>
      1、查询平台设备接入协议（端口号）和域名
     </strong>
     <br/>
     <img alt="总览页面" src="https://i-blog.csdnimg.cn/blog_migrate/71e4f3ce4a7852623cadd15df2eaa33c.png#pic_center">
      2、打开设备-&gt;所有设备-&gt;查看所要接入的设备
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/831fd6b2ab233306ed275dbb5d47a566.png#pic_center"/>
     </img>
    </p>
    <p>
     <strong>
      3、下载华为云连接信息生成工具
     </strong>
     <br/>
     下载完成后，打开jar包，信息生成工具界面如下图
     <br/>
     ①deviceId：即设备ID
     <br/>
     ②device Secret：即密钥
     <br/>
     ③Message：即生成的连接信息（clientId、username、password）
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a7c79279f2ab1b92e175bfc850dc117.png#pic_center">
      <br/>
      <strong>
       4、生成连接信息
      </strong>
      <br/>
      把上述相关参数填入生成工具中，点击Generate
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2b8d788cab056273b24c4996b2deeed9.png"/>
     </img>
    </p>
    <h3>
     <a id="MQTTfxlotDA_51">
     </a>
     三、MQTT.fx接入华为云lotDA
    </h3>
    <p>
     <strong>
      1、打开MQTT.fx-1.7.1客户端，点击下方的齿轮按钮
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/44a20ee939a299a5f188e5be2b1a7402.png#pic_center"/>
    </p>
    <p>
     <strong>
      2、在打开的对话框中进行相关参数配置
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/663d17a9cc965b3a26855c54d59b9b50.png#pic_center"/>
     <br/>
     ① Profile Name：可以随便写入名称，支持中文
     <br/>
     ② Profile Type：保持默认设置，MQTT Broker
     <br/>
     ③ Broker Adress:填写从设备接入服务控制台获取的设备对接地址，此地址为域名信息。
     <br/>
     ④ Broker Port:接入协议，即端口号
     <br/>
     ⑤ Client ID：客户端ID，此处我们使用平台连接生成工具生成的ID信息
     <br/>
     ⑥ User Name：用户名，此处我们使用平台连接生成工具生成的User Name信息
     <br/>
     ⑦ Password：密码，即生成工具生成的Password
    </p>
    <p>
     <strong>
      3、下载并配置证书（解压后，用户根据实际需要选择对应语言的证书）
     </strong>
     <br/>
     说明：建议使用Java语言pem格式的证书
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/def90cfb3cf0d1941ee0526583782b39.png#pic_center"/>
     <br/>
     <strong>
      4、配置完成，点击“Connect”，设备鉴权成功后，在物联网平台可以看到设备处于在线状态
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f9af8f569bafbfc2d9d9cb655d8ac801.png#pic_center"/>
    </p>
