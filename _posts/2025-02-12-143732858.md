    <h4>
     <a id="11_Ollama_34">
     </a>
     1.1 Ollama介绍
    </h4>
    <p>
     <strong>
      Ollama
     </strong>
     是一个开源框架，专为在本地机器上便捷部署和运行大型语言模型（LLM）而设计。以下是关于Ollama的详细介绍：
    </p>
    <h5>
     <a id="111__38">
     </a>
     1.1.1 主要特点
    </h5>
    <ol>
     <li>
      <ol>
       <li>
        <strong>
         简化部署
        </strong>
        ：Ollama旨在简化在Docker容器中部署LLM的过程，使得非专业用户也能方便地管理和运行这些复杂的模型。
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        <strong>
         轻量级与可扩展
        </strong>
        ：作为轻量级框架，Ollama保持了较小的资源占用，同时具备良好的可扩展性，允许用户根据需要调整配置以适应不同规模的项目和硬件条件。
       </li>
      </ol>
     </li>
     <li>
      <ol start="3">
       <li>
        <strong>
         API支持
        </strong>
        ：提供了一个简洁的API，使得开发者能够轻松创建、运行和管理大型语言模型实例，降低了与模型交互的技术门槛。
       </li>
      </ol>
     </li>
     <li>
      <ol start="4">
       <li>
        <strong>
         预构建模型库
        </strong>
        ：包含一系列预先训练好的大型语言模型，用户可以直接选用这些模型应用于自己的应用程序，无需从头训练或自行寻找模型源。
       </li>
      </ol>
     </li>
     <li>
      <ol start="5">
       <li>
        <strong>
         跨平台支持
        </strong>
        ：提供针对macOS、Windows（预览版）、Linux以及Docker的安装指南，确保用户能在多种操作系统环境下顺利部署和使用Ollama。
       </li>
      </ol>
     </li>
    </ol>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/a8874981afe054b85bd4e26a9f8db1d9.png"/>
    </p>
    <h5>
     <a id="112__53">
     </a>
     1.1.2 使用场景
    </h5>
    <ol>
     <li>
      <ol>
       <li>
        <strong>
         聊天机器人
        </strong>
        ：利用Ollama部署的LLM，可以创建具有智能对话功能的聊天机器人。
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        <strong>
         文本生成
        </strong>
        ：可以用于生成各种文本内容，如新闻文章、博客文章、诗歌等。
       </li>
      </ol>
     </li>
     <li>
      <ol start="3">
       <li>
        <strong>
         问答系统
        </strong>
        ：能够回答用户提出的各种问题，适用于多种问答场景。
       </li>
      </ol>
     </li>
     <li>
      <ol start="4">
       <li>
        <strong>
         代码生成
        </strong>
        ：可以生成多种编程语言的代码，如Python、JavaScript等。
       </li>
      </ol>
     </li>
    </ol>
    <h4>
     <a id="12_OpenWebUI_64">
     </a>
     1.2 Open-WebUI介绍
    </h4>
    <p>
     <strong>
      Open WebUI
     </strong>
     （前身为Ollama WebUI）是一个专为大型语言模型（LLM）设计的可扩展、功能丰富且用户友好的自托管Web管理工具，旨在为用户提供直观、高效的大模型交互体验。以下是对Open WebUI的详细介绍：
    </p>
    <h5>
     <a id="121__68">
     </a>
     1.2.1 主要特点
    </h5>
    <ol>
     <li>
      <ol>
       <li>
        <strong>
         离线运行
        </strong>
        ：Open WebUI设计用于完全离线运行，无需依赖外部服务器或网络连接，提高了数据的安全性和隐私保护。
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        <strong>
         多模型支持
        </strong>
        ：支持各种LLM运行器，包括Ollama和兼容OpenAI的API，用户可以根据需要轻松集成和管理不同的大型语言模型。
       </li>
      </ol>
     </li>
     <li>
      <ol start="3">
       <li>
        <strong>
         直观界面
        </strong>
        ：聊天界面灵感来源于ChatGPT，确保了用户友好的体验。同时，提供响应式设计，在桌面和移动设备上都能享受无缝的体验。
       </li>
      </ol>
     </li>
     <li>
      <ol start="4">
       <li>
        <strong>
         轻松设置
        </strong>
        ：支持使用Docker或Kubernetes（kubectl、kustomize或helm）无缝安装，简化了部署和配置过程。
       </li>
      </ol>
     </li>
    </ol>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/9d2c5734fc0fe2d5f8eacf8c3e237afa.png"/>
    </p>
    <h5>
     <a id="122__81">
     </a>
     1.2.2 应用场景
    </h5>
    <p>
     Open WebUI适用于多种场景，包括但不限于：
    </p>
    <ol>
     <li>
      <ol>
       <li>
        <strong>
         聊天机器人
        </strong>
        ：利用Open WebUI部署的LLM，可以创建具有智能对话功能的聊天机器人，用于客户服务、娱乐互动等领域。
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        <strong>
         文本生成
        </strong>
        ：可以用于生成各种文本内容，如新闻文章、博客文章、诗歌等，满足内容创作的需求。
       </li>
      </ol>
     </li>
     <li>
      <ol start="3">
       <li>
        <strong>
         问答系统
        </strong>
        ：能够回答用户提出的各种问题，适用于教育、咨询、医疗等多个领域。
       </li>
      </ol>
     </li>
     <li>
      <ol start="4">
       <li>
        <strong>
         代码生成
        </strong>
        ：可以生成多种编程语言的代码，如Python、JavaScript等，辅助开发者进行代码编写和调试。
       </li>
      </ol>
     </li>
    </ol>

    <h4>
     <a id="32__120">
     </a>
     3.2 下载多模态大模型
    </h4>
    <p>
     可以在ollama官网找到非常多的大模型，然后根据自己的显卡（MacOS M芯片是内存）大小选择对应的模型，根据经验12G显存可以运行大概7B的模型。ollama的模型库包含了热门的
     <strong>
      llama3.2
     </strong>
     、
     <strong>
      qwen2.5
     </strong>
     、
     <strong>
      gemma2
     </strong>
     等模型，都非常优秀。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/ee318bfe7ca01394fd31bfd3f2cc3a7f.png"/>
    </p>
    <p>
     由于本文主要是介绍在本地部署多模态大模型，因此主要下载
     <strong>
      llava:13b
     </strong>
     和
     <strong>
      qwen:14b-chat
     </strong>
     这两个模型，其他模型大家可以自行探索，在命令行执行下面的命令进行下载：
    </p>
    <pre><code># 下载llava:13b模型   ollama pull llava:13b   # 下载qwen:14b-chat   ollama pull qwen:14b-chat
</code></pre>
    <p>
     下载完成后，提示如下：
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/ed41b29927b4c99001d43658b564b9aa.png"/>
    </p>
    <h4>
     <a id="33_ollama_136">
     </a>
     3.3 ollama开启远程访问
    </h4>
    <h5>
     <a id="331_windows_138">
     </a>
     3.3.1 windows设置
    </h5>
    <p>
     在「电脑」-&gt;「属性」-&gt;「高级系统设置」-&gt;「环境变量」中，添加一行记录：
    </p>
    <p>
     变量：OLLAMA_HOST，值：0.0.0.0
    </p>
    <h5>
     <a id="332_MacOS_144">
     </a>
     3.3.2 MacOS设置
    </h5>
    <p>
     在~/.bashrc中的最后一行添加下面的环境变量
    </p>
    <pre><code>export OLLAMA_HOST=0.0.0.0
</code></pre>
    <p>
     然后执行，下面的命令使环境变量在当前shell生效
    </p>
    <pre><code>source ~/.bashrc
</code></pre>
    <h5>
     <a id="333_ollama_158">
     </a>
     3.3.3 重启ollama
    </h5>
    <p>
     重启ollama使得刚才设置的环境变量生效。
    </p>

    <h3>
     <a id="OllamaDify_226">
     </a>
     六、Ollama和Dify结合，打造本地模型+智能体
    </h3>
    <p>
     ollama也可以和Dify进行结合，使用本地大模型来构建智能体
    </p>
    <p>
     进入到Dify首页，点击右上角的头像，选择「设置」：
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/de4bced6e0e87b7c1a718db4a6fbc13b.png"/>
    </p>
    <p>
     在「模型供应商」选择Ollama
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/f0739d7d85d6ac2b2397991000515a08.png"/>
    </p>
    <p>
     在弹出的界面中：
    </p>
    <ul>
     <li>
      <p>
       •
       <strong>
        模型名称
       </strong>
       ：填写用ollama list命令列出来的模型名
      </p>
     </li>
     <li>
      <p>
       •
       <strong>
        模型基础URL
       </strong>
       ：http://host.docker.internal:11434
      </p>
     </li>
    </ul>
    <p>
     点击保存即可。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/c849839c42ccda6fb0e2e484dc19bb8c.png"/>
    </p>
