      <p>
       在 Java 中，可以使用以下代码来求三个数的最大值：
      </p>
      <pre class="has"><code>int<span class="hljs-keyword">max</span> <span class="hljs-operator">=</span> a<span class="hljs-comment">;</span>
if (b &gt; <span class="hljs-keyword">max</span>) {
  <span class="hljs-keyword">max</span> <span class="hljs-operator">=</span> b<span class="hljs-comment">;</span>
}
if (<span class="hljs-keyword">c</span> &gt; <span class="hljs-keyword">max</span>) {
  <span class="hljs-keyword">max</span> <span class="hljs-operator">=</span> <span class="hljs-keyword">c</span><span class="hljs-comment">;</span>
}
</code></pre>
      <p>
       其中，a、b、c 分别表示三个数。上述代码中，我们首先将 a 赋值给变量 max，然后判断 b 是否大于 max，如果是，就将 b 赋值给 max；接着判断 c 是否大于 max，如果是，就将 c 赋值给 max。最终，max 的值就是三个数的最大值。
      </p>
      <p>
       也可以使用 Java 的 Math 库中的 max 函数来实现：
      </p>
      <pre class="has"><code><span class="hljs-built_in">int</span> <span class="hljs-built_in">max</span> = Math.<span class="hljs-built_in">max</span>(a, Math.<span class="hljs-built_in">max</span>(b, c));
</code></pre>
      <p>
       这种方法比较简单，但是会带来一些额外的开销。
      </p>
