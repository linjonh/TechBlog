    <h2>
     <a id="_Python__0">
     </a>
     方法一：切换 Python 版本
    </h2>
    <ol>
     <li>
      <p>
       查看系统可切换 Python 版本
      </p>
      <pre><code class="prism language-bash">update-alternatives <span class="token parameter variable">--list</span> python
<span class="token comment"># /usr/bin/python2.7</span>
<span class="token comment"># /usr/bin/python3.5</span>
<span class="token comment"># /usr/bin/python3.6</span>
</code></pre>
     </li>
     <li>
      <p>
       Python 版本切换：选择想要切换的版本序号即可
      </p>
      <pre><code class="prism language-bash">update-alternatives <span class="token parameter variable">--config</span> python
</code></pre>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/f7d71930c2e7d8ab453e1e28ab7541b5.png" width="60%">
      </img>
     </li>
    </ol>
    <h2>
     <a id="_Python__15">
     </a>
     修改 Python 软链接
    </h2>
    <ol>
     <li>
      <p>
       查看当前使用的版本
      </p>
      <pre><code class="prism language-bash">python <span class="token parameter variable">--version</span>
</code></pre>
     </li>
     <li>
      <p>
       查看当前所使用版本的位置
      </p>
      <pre><code class="prism language-bash"><span class="token function">which</span> python
</code></pre>
     </li>
     <li>
      <p>
       如果第二步结果是
      </p>
      <pre><code class="prism language-bash">/usr/bin/python
</code></pre>
      <p>
       则直接删除即可
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">rm</span> /usr/bin/python
</code></pre>
      <p>
       如果不在以上文件夹中，则修改名字，不要删除，如下
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mv</span> /home/ustc/anaconda3/bin/python /home/ustc/anaconda3/bin/python3.7.4
</code></pre>
     </li>
     <li>
      <p>
       创建新的软链接
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">ln</span> <span class="token parameter variable">-s</span> /usr/bin/python2.7 /usr/bin/python
</code></pre>
     </li>
     <li>
      <p>
       查看结果
      </p>
      <pre><code class="prism language-bash">python <span class="token parameter variable">--version</span>
</code></pre>
     </li>
    </ol>
