    <p>
     1.连接数据库
     <br/>
     sqlplus 用户名/密码 【@host_string】【用户身份】
     <br/>
     sqlplus sys/空格 as sysdba
     <br/>
     以sys用户登录，必须指定身份为 as sysdba 或 as sysoper，普通用户可以不要指定身份
     <br/>
     可以不输入密码的原因，是需要操作系统管理员用户。
    </p>

    <p>
     <strong>
      2.查询数据库所有用户
     </strong>
     <br/>
     select * from all_users;
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170419202532725?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemhvdTkyMDc4NjMxMg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
    </p>
    <p>
     <strong>
      3.创建用户（必须的要有创建用户的权限）
     </strong>
     <br/>
     create user 用户名 identified by 密码
     <br/>
     新创建的用户是没有权限的，需要授权.
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170419203015029?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemhvdTkyMDc4NjMxMg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
    </p>
    <p>
     <strong>
      4给用户连接赋予权限
     </strong>
     <br/>
     grant 权限 to 用户;
     <br/>
     grant connect,resource to zhou;
     <br/>
     connect,resource 是角色，角色有多个权限
