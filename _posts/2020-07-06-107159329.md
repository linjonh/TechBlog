    <p>
     执掌企业IT命脉的长红职业
     <br/>
     网络&amp; 安全技术人员可以在各种企业内担任网络工程师、信息安全工程师、运维工程师、首席技术官等职务，从事网络规划设计、IT 架构优化、数据库运营保障、网络安全攻防相关工作。
     <br/>
     凭一台笔记本可以攻城掠地、拯救地球，敲一个指令便可决定企业的生死存亡。
     <br/>
     得益于长期实践经验积累，以及发现/ 解决/ 预防问题的能力，网络&amp; 安全从业者具有超长的职业生涯，其薪酬水平、发展空间随技术年龄增长而快速攀升，真正的“越老越吃香”。
    </p>
    <p>
     网络运维工程师薪资
     <br/>
     根据职友集的数据显示，当前市场上需求量较大的几类网络安全岗位，如安全运维、渗透测试、等保测评等，平均薪资水平都在10k左右。
     <br/>
     随着经验和水平的不断增长，网络安全工程师可以胜任更高阶的安全架构、安全管理岗位，薪资更是可达30k。
    </p>
    <p>
     <br/>
     谁适合学网络运维？
     <br/>
     网络工程师的岗位核心技能要求：企业办公网组建、布线、企业信息应用、网络运维、规划设计、安全防护等，且易学。
     <br/>
     适合人群：
     <br/>
     1、 转行从事网络领域的人士，零基础学习网络技术，高薪就业。
     <br/>
     2、 面临加薪和跳槽瓶颈的在职工程师。
     <br/>
     3、 高职本科院校的计算机相关专业的学生。
     <br/>
     4、热爱网络技术，希望在网络行业长期发展的有志人士。
    </p>
    <p>
     <br/>
     网络运维职业发展
     <br/>
     桌面技术支持：能搞定企业办公组网、计算机组装/维修、系统&amp;软件安装、打印/扫描/投影等设备调试排错、解决常用软件问题、Windows服务器配置；
     <br/>
     网络工程师：能搞定交换机/路由器设备配置、中等规模企业网规划设计、办公楼综合布线、企业数据安全保障、公司云服务器等产品的选购及管理；
     <br/>
     具备进一步学习 Linux云计算/运维开发 课程的能力，向运维工程师岗位转型。
    </p>
