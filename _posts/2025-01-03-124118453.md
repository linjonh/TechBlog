    <h2>
     <a id="dockerdb2_2">
     </a>
     使用docker搭建db2数据库
    </h2>
    <h3>
     <a id="1docker_hubdb2_3">
     </a>
     1.进入docker hub查询db2版本
    </h3>
    <blockquote>
     <p>
      docker hub 官网:https://hub.docker.com/r/ibmcom/db2/tags
     </p>
    </blockquote>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d2f2e391400cfa03b96b411c78a43724.png">
      <br/>
      可以选择需要的版本。
     </img>
    </p>
    <h3>
     <a id="2_10">
     </a>
     2.运行命令执行搭建
    </h3>
    <p>
     我这里安装db2的最新版镜像
    </p>
    <pre><code>docker run -itd --name mydb2 --privileged=true -p 50000:50000 -e LICENSE=accept -e DB2INST1_PASSWORD=12345678 -e DBNAME=testdb -v /data/docker/db2:/database ibmcom/db2
</code></pre>
    <h3>
     <a id="3db2_16">
     </a>
     3.db2常用操作
    </h3>
    <h4>
     <a id="_17">
     </a>
     创建操作系统用户
    </h4>
    <p>
     db2用户使用的是操作系统系统用户，增加一个操作系统用户
    </p>
    <pre><code>groupadd ump ---添加用户组ump

useradd -m -g ump -d /home/ump ump ---添加用户ump

passwd ump ---修改ump的密码 
</code></pre>
    <h4>
     <a id="db2_27">
     </a>
     db2创建数据库
    </h4>
    <pre><code>[root@e986b568804e /]# su - db2inst1                                 
Last login: Tue Apr 12 14:44:22 UTC 2022
[db2inst1@e986b568804e ~]$ db2 create db ump_gsms
DB20000I  The CREATE DATABASE command completed successfully.
</code></pre>
    <h4>
     <a id="db2_34">
     </a>
     db2删除数据库
    </h4>
    <pre><code>db2 drop database ump_gsms
</code></pre>
    <h4>
     <a id="DB2_39">
     </a>
     赋予用户DB2权限
    </h4>
    <pre><code>#db2 连接数据库

db2 connect to ump_gsms

#db2 授权用户

db2 GRANT DBADM,CREATETAB,BINDADD,CONNECT,CREATE_NOT_FENCED_ROUTINE,IMPLICIT_SCHEMA,LOAD,CREATE_EXTERNAL_ROUTINE,QUIESCE_CONNECT ON DATABASE TO USER ump
</code></pre>
    <h4>
     <a id="db2__49">
     </a>
     db2 关闭连接
    </h4>
    <pre><code>db2 connect reset
</code></pre>
