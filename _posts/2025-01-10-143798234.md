    <p>
     数据结构和算法是计算机科学的核心，掌握它们不仅能提高代码的性能和效率，还能帮助你在面试中脱颖而出。对于 C# 开发者而言，了解常见的数据结构和算法，能帮助你在处理复杂问题时做出最优的选择。

    <hr/>
    <h3>
     1.
     <strong>
      数据结构概述
     </strong>
    </h3>
    <p>
     数据结构是计算机中存储和组织数据的方式。不同的数据结构适用于不同类型的问题，选择合适的数据结构可以提高程序的运行效率。
    </p>
    <h4>
     1.1 常见数据结构
    </h4>
    <ul>
     <li>
      <strong>
       数组（Array）
      </strong>
      ：线性数据结构，具有固定大小的元素集合。
     </li>
     <li>
      <strong>
       链表（Linked List）
      </strong>
      ：由一系列节点组成的线性数据结构，节点之间通过指针连接。
     </li>
     <li>
      <strong>
       栈（Stack）
      </strong>
      ：后进先出（LIFO）结构，适用于处理递归、撤销操作等问题。
     </li>
     <li>
      <strong>
       队列（Queue）
      </strong>
      ：先进先出（FIFO）结构，常用于任务调度、消息传递等场景。
     </li>
     <li>
      <strong>
       哈希表（Hash Table）
      </strong>
      ：基于哈希算法实现的映射结构，支持快速查找。
     </li>
     <li>
      <strong>
       树（Tree）
      </strong>
      ：层级数据结构，用于组织具有层次关系的数据。
     </li>
     <li>
      <strong>
       图（Graph）
      </strong>
      ：由节点和边组成的结构，适合表示网络关系。
     </li>
    </ul>
    <h4>
     1.2 C# 中的数据结构
    </h4>
    <p>
     C# 提供了许多内置的集合类型，如：
    </p>
    <ul>
     <li>
      <code>
       Array
      </code>
      ：内置数组类型，支持快速随机访问。
     </li>
     <li>
      <code>
       List&lt;T&gt;
      </code>
      ：动态数组，支持扩展。
     </li>
     <li>
      <code>
       LinkedList&lt;T&gt;
      </code>
      ：双向链表。
     </li>
     <li>
      <code>
       Stack&lt;T&gt;
      </code>
      ：栈实现。
     </li>
     <li>
      <code>
       Queue&lt;T&gt;
      </code>
      ：队列实现。
     </li>
     <li>
      <code>
       Dictionary&lt;TKey, TValue&gt;
      </code>
      ：哈希表（键值对映射）。
     </li>
    </ul>
    <h3>
     2.
     <strong>
      算法基础
     </strong>
    </h3>
    <p>
     算法是解决问题的一组步骤或规则。在解决问题时，算法的效率往往比代码的实现方式更为重要。常见的算法类型包括：
    </p>
    <ul>
     <li>
      <strong>
       排序算法
      </strong>
      ：如快速排序、归并排序、冒泡排序。
     </li>
     <li>
      <strong>
       查找算法
      </strong>
      ：如线性查找、二分查找。
     </li>
     <li>
      <strong>
       递归与迭代
      </strong>
      ：许多问题可以通过递归或迭代的方式解决。
     </li>
     <li>
      <strong>
       动态规划
      </strong>
      ：通过将问题分解为子问题来优化计算过程，常用于解决最优化问题。
     </li>
     <li>
      <strong>
       图算法
      </strong>
      ：如广度优先搜索（BFS）、深度优先搜索（DFS）、Dijkstra 算法等。
     </li>
    </ul>
    <hr/>
    <h3>
     3.
     <strong>
      C# 实现基本数据结构
     </strong>
    </h3>
    <h4>
     3.1 数组（Array）
    </h4>
    <p>
     数组是最简单的数据结构，它在 C# 中通过
     <code>
      Array
     </code>
     类型或
     <code>
      List&lt;T&gt;
     </code>
     类型实现。数组的大小固定，访问元素的时间复杂度为 O(1)。
    </p>
    <pre><code class="language-cs">int[] arr = { 1, 2, 3, 4, 5 };
Console.WriteLine(arr[2]);  // 访问数组元素，输出 3
</code></pre>
    <h4>
     3.2 链表（Linked List）
    </h4>
    <p>
     链表由一系列节点组成，每个节点包含数据和指向下一个节点的指针。C# 提供了
     <code>
      LinkedList&lt;T&gt;
     </code>
     类型来实现双向链表。
    </p>
    <pre><code class="language-cs">var linkedList = new LinkedList&lt;int&gt;();
linkedList.AddLast(1);
linkedList.AddLast(2);
linkedList.AddLast(3);

foreach (var item in linkedList)
{
Console.WriteLine(item); // 输出 1 2 3
}
</code></pre>
<h4>
3.3 栈（Stack）
</h4>
<p>
栈遵循后进先出（LIFO）原则，C# 提供了
<code>
Stack&lt;T&gt;
</code>
类。
</p>
<pre><code class="language-cs">Stack&lt;int&gt; stack = new Stack&lt;int&gt;();
stack.Push(1);
stack.Push(2);
stack.Push(3);

Console.WriteLine(stack.Pop()); // 输出 3
Console.WriteLine(stack.Peek()); // 输出 2（但不删除）
</code></pre>
<h4>
3.4 队列（Queue）
</h4>
<p>
队列遵循先进先出（FIFO）原则，C# 提供了
<code>
Queue&lt;T&gt;
</code>
类。
</p>
<pre><code class="language-cs">Queue&lt;int&gt; queue = new Queue&lt;int&gt;();
queue.Enqueue(1);
queue.Enqueue(2);
queue.Enqueue(3);

Console.WriteLine(queue.Dequeue()); // 输出 1
Console.WriteLine(queue.Peek()); // 输出 2（但不删除）
</code></pre>
<h4>
3.5 哈希表（Dictionary）
</h4>
<p>
哈希表通过键值对存储数据，C# 提供了
<code>
Dictionary&lt;TKey, TValue&gt;
</code>
类。
</p>
<pre><code class="language-cs">Dictionary&lt;string, int&gt; dictionary = new Dictionary&lt;string, int&gt;();
dictionary["apple"] = 5;
dictionary["banana"] = 10;

Console.WriteLine(dictionary["apple"]); // 输出 5
</code></pre>

<h3> 4.
<strong>
常见算法实现
</strong>
</h3>
<h4>
4.1 排序算法
</h4>
<h5>
4.1.1 冒泡排序（Bubble Sort）
</h5>
<p>
冒泡排序是一种简单的排序算法，通过重复交换相邻元素，直到整个序列有序。
</p>
<pre><code class="language-cs">void BubbleSort(int[] arr)
{
int n = arr.Length;
for (int i = 0; i &lt; n - 1; i++)
{
for (int j = 0; j &lt; n - i - 1; j++)
{
if (arr[j] &gt; arr[j + 1])
{
// 交换
int temp = arr[j];
arr[j] = arr[j + 1];
arr[j + 1] = temp;
}
}
}
}
</code></pre>
<h5>
4.1.2 快速排序（Quick Sort）
</h5>
<p>
快速排序是一种分治法排序算法，它选择一个“基准”元素，将数组分为两部分，一部分元素小于基准，另一部分元素大于基准，然后递归地对每部分进行排序。
</p>
<pre><code class="language-cs">void QuickSort(int[] arr, int low, int high)
{
if (low &lt; high)
{
int pivot = Partition(arr, low, high);
QuickSort(arr, low, pivot - 1);
QuickSort(arr, pivot + 1, high);
}
}

int Partition(int[] arr, int low, int high)
{
int pivot = arr[high];
int i = low - 1;
for (int j = low; j &lt; high; j++)
{
if (arr[j] &lt; pivot)
{
i++;
int temp = arr[i];
arr[i] = arr[j];
arr[j] = temp;
}
}
int temp2 = arr[i + 1];
arr[i + 1] = arr[high];
arr[high] = temp2;
return i + 1;
}
</code></pre>
<h4>
4.2 查找算法
</h4>
<h5>
4.2.1 线性查找
</h5>
<p>
线性查找从数组的第一个元素开始，逐个检查每个元素，直到找到目标元素或遍历完所有元素。
</p>
<pre><code class="language-cs">int LinearSearch(int[] arr, int target)
{
for (int i = 0; i &lt; arr.Length; i++)
{
if (arr[i] == target)
return i; // 返回元素的索引
}
return -1; // 如果没有找到元素
}
</code></pre>
<h5>
4.2.2 二分查找
</h5>
<p>
二分查找是一种高效的查找算法，要求数据已经排序。每次将查找范围减半，直到找到目标元素或范围为空。
</p>
<pre><code class="language-cs">int BinarySearch(int[] arr, int target)
{
int left = 0, right = arr.Length - 1;
while (left &lt;= right)
{
int mid = left + (right - left) / 2;
if (arr[mid] == target)
return mid; // 找到目标元素
if (arr[mid] &lt; target)
left = mid + 1; // 在右半部分继续查找
else
right = mid - 1; // 在左半部分继续查找
}
return -1; // 没有找到目标元素
}
</code></pre>

<h3> 5.
<strong>
递归与动态规划
</strong>
</h3>
<h4>
5.1 递归（Recursion）
</h4>
<p>
递归是函数调用自身的过程。许多问题可以通过递归来简化，常见的递归问题如斐波那契数列、阶乘计算等。
</p>
<h5>
5.1.1 斐波那契数列
</h5>
<pre><code class="language-cs">int Fibonacci(int n)
{
if (n &lt;= 1) return n;
return Fibonacci(n - 1) + Fibonacci(n - 2);
}
</code></pre>
<h4>
5.2 动态规划（Dynamic Programming）
</h4>
<p>
动态规划通过分解子问题并缓存结果来避免重复计算，常用于优化递归算法，如背包问题、最短路径问题等。
</p>
<h5>
5.2.1 斐波那契数列的动态规划解法
</h5>
<pre><code class="language-cs">int FibonacciDP(int n)
{
int[] dp = new int[n + 1];
dp[0] = 0;
dp[1] = 1;
for (int i = 2; i &lt;= n; i++)
{
dp[i] = dp[i - 1] + dp[i - 2];
}
return dp[n];
}
</code></pre>

<h3> 6.
<strong>
高级算法：图算法
</strong>
</h3>
<p>
图算法广泛应用于计算机网络、社交网络等领域。常见的图算法有
<strong>
广度优先搜索（BFS）
</strong>
、
<strong>
深度优先搜索（DFS）
</strong>
、
<strong>
Dijkstra 算法
</strong>
等。
</p>
<h4>
6.1 广度优先搜索（BFS）
</h4>
<p>
广度优先搜索是一种逐层遍历图的算法，通常用于寻找最短路径。
</p>
<pre><code class="language-cs">void BFS(Dictionary&lt;int, List&lt;int&gt;&gt; graph, int start)
{
var visited = new HashSet&lt;int&gt;();
var queue = new Queue&lt;int&gt;();
queue.Enqueue(start);
visited.Add(start);

    while (queue.Count &gt; 0)
    {
        int node = queue.Dequeue();
        Console.WriteLine(node);

        foreach (var neighbor in graph[node])
        {
            if (!visited.Contains(neighbor))
            {
                visited.Add(neighbor);
                queue.Enqueue(neighbor);
            }
        }
    }

}
</code></pre>
<h4>
6.2 深度优先搜索（DFS）
</h4>
<p>
深度优先搜索是通过递归或栈的方式逐层深入图中的每一个节点。
</p>
<pre><code class="language-cs">void DFS(Dictionary&lt;int, List&lt;int&gt;&gt; graph, int node, HashSet&lt;int&gt; visited)
{
visited.Add(node);
Console.WriteLine(node);

    foreach (var neighbor in graph[node])
    {
        if (!visited.Contains(neighbor))
        {
            DFS(graph, neighbor, visited);
        }
    }

}
</code></pre>
