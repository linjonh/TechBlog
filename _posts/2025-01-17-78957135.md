     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       今天，2018年第一个工作日。
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       区块宝研究院隆重推出
       <span style="font-size: 14px;font-family: 宋体;color: rgb(255, 76, 0);">
        《2017年区块链行业年度特别报告》
        <span style="font-size: 14px;font-family: 宋体;color: rgb(0, 0, 0);">
         ，
        </span>
       </span>
       该报告主要介绍了区块宝研究院近期在区块链行业研究方面的研究成果。
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       《2017年区块链行业年度特别报告》主要分为四个部分：
       <span style="font-family: 宋体;font-size: 14px;color: rgb(0, 128, 255);">
        区块链技术概述
       </span>
       、
       <span style="font-family: 宋体;font-size: 14px;color: rgb(0, 128, 255);">
        区块链行业概述
       </span>
       、
       <span style="font-family: 宋体;font-size: 14px;color: rgb(0, 128, 255);">
        2017年回顾
       </span>
       、
       <span style="font-family: 宋体;font-size: 14px;color: rgb(0, 128, 255);">
        2018年前瞻
       </span>
       。
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family: 宋体;font-size: 14px;">
       区块链技术概述部分，主要普及性概述了区块链的概念、区块的结构、区块链的基础架构及区块链的核心技术；
      </span>
     </p>
     <p style="text-indent:32px;">
      <br/>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family:宋体;">
       <span style="font-size: 14px;">
        区块链行业概述部分，主要解析了区块链行业的产业生态链、区块链行业的产业价值链及区块链行业特有的数字资产概况；
       </span>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family:宋体;">
       <span style="font-size: 14px;">
        <br/>
       </span>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family:宋体;">
       <span style="font-size: 14px;">
        2017年回顾部分，主要从专项政策、特色监管、行业发展三个方面对2017年区块链行业的发展情况进行了回顾；
       </span>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family:宋体;">
       <span style="font-size: 14px;">
        <br/>
       </span>
      </span>
     </p>
     <p style="text-indent: 0em;">
      <span style="font-family:宋体;">
       <span style="font-size: 14px;">
        2018年前瞻部分，主要依据区块宝研究院近年来对区块链行业的深度观察、持续跟踪及强大的数据库积累建设，提出了对区块链行业在企业层面、应用层面、技术层面的发展趋势的判断。
       </span>
      </span>
     </p>
