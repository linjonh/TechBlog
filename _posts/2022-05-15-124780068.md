    <p>
     前面几章其实已经把基础的Latex中文文本写的差不多了。这一章节针对我论文中用到的数学公式写一些心得。

    <hr/>
    <h2>
     <a id="_2">
     </a>
     数学公式
    </h2>
    <p>
     通常理工科写论文，少不了数学公式。在公式上，我个人认为相比word，latex还是有非常大的优势。
    </p>
    <h3>
     <a id="_4">
     </a>
     行内公式
    </h3>
    <p>
     行内公式通常用一对
     <code>
      $
     </code>
     来表示，例如：
    </p>
    <pre><code class="prism language-latex">行内公式：$x_2+y_2=1$
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1a92b85087f9c3dad1774c9a8dd36936.png">
      <br/>
      不过通常我们涉及到公式，在论文中都要进行标注，即要标记这是第几个公式等等。
      <br/>
      这种情况一般用到行间公式。
     </img>
    </p>
    <hr/>
    <h3>
     <a id="_13">
     </a>
     行间公式
    </h3>
    <p>
     行间公式通常用两对
     <code>
      $
     </code>
     来表示，例如：
    </p>
    <pre><code class="prism language-latex">行间公式：$$x_2+y_2=1$$
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac1a4a1f841e5675e17c4056627cb143.png">
      <br/>
      但是这里我们发现公式并没有标号，这时候需要用到环境这一词。
     </img>
    </p>
    <hr/>
    <h3>
     <a id="_21">
     </a>
     数学环境
    </h3>
    <h4>
     <a id="equation_22">
     </a>
     equation
    </h4>
    <p>
     比较正规的公式写法，后面都是有数学公式的标号。这时候需要用到环境
     <code>
      \begin{equation}
     </code>
    </p>
    <pre><code class="prism language-latex">数学环境：
	\begin{equation}
		y=a_0+a_1x+a_2x^2+...+a_nx^n\label{1}
	\end{equation}
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cbd4c90dafc0140d90ac2a0e78603448.png">
      <br/>
      这里需要注意的是，使用数学环境，在数学的环境中，不需要使用
      <code>
       $
      </code>
      来表示这是数学公式。
      <br/>
      上述代码段中，使用了
      <code>
       \label{}
      </code>
      的方式，对公式进行了标号。
     </img>
    </p>
    <hr/>
    <h4>
     <a id="gather_34">
     </a>
     gather
    </h4>
    <p>
     equation适用于单公式情况，这个环境直接翻译也就是等式的意思嘛，所以也就是一个等式的意思。
     <br/>
     如果要一次性多写几个，那需要用到gather环境。使用
     <code>
      gather
     </code>
     环境需要用到
     <code>
      \usepackage{amsmath}
     </code>
    </p>
    <pre><code class="prism language-latex">\begin{gather}
     x+y=2\\
     x-y=-4
\end{gather}
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/02b72675aad02adc677dc9677266fd1a.png">
      <br/>
      这里我的公式是自动居中排版，同时每一个公示自动编号。此数学环境适用于一次性写多个公式。如果不想编号，只需要在
      <code>
       gather
      </code>
      后面加
      <code>
       *
      </code>
     </img>
    </p>
    <hr/>
    <h4>
     <a id="cases_46">
     </a>
     cases
    </h4>
    <p>
     如果我想将上述的两个方程放到一个方程组进行编号，这时候需要用到
     <code>
      cases
     </code>
     环境。使用cases环境最好搭配上
     <code>
      equation
     </code>
     环境使用，具体情况如下所示。
    </p>
    <pre><code class="prism language-latex">\begin{equation}
	\begin{cases}
	x+y=2\\
	x-y=-4
	\end{cases}
\end{equation}
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ceacad441a5f60d8edfdb961e65a8f6.png"/>
    </p>
    <hr/>
    <p>
     这里会发现我们的标号是(3),所以可知道，使用数学环境编写的cases公式，会自带编号。
     <br/>
     如果我们不使用equation环境，单独使用cases环境进行书写，
     <strong>
      会报错！
     </strong>
     。原因在于cases是数学的环境，所以必须作为数学公式使用，如果不放在环境里面，需要我们使用一对
     <code>
      $
     </code>
     作为数学公式的声明。
     <br/>
     同时，使用
     <code>
      $
     </code>
     符号的数学与公式，是不会自动编号的
    </p>
    <pre><code class="prism language-latex">$$\begin{cases}

    <h3>
     <a id="Latex_72">
     </a>
     Latex数学公式的常用写法
    </h3>
    <p>
     刚才举例的情况中，以及涉及到了一些数学公式的写法。下面列举一些常用的写法。
    </p>
    <h4>
     <a id="_74">
     </a>
     上下标
    </h4>
    <blockquote>
     <p>
      上标用
      <code>
       ^
      </code>
      <br/>
      下标用
      <code>
       _
      </code>
     </p>
    </blockquote>
    <pre><code class="prism language-latex">\begin{equation}
	y=a^0+a^1+a^2+a^3+a^4+...+a^n\label{1}		
\end{equation}	
\begin{equation}
	y=a_0+a_1x+a_2x^2+...+a_nx^n\label{2}	
\end{equation}
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8e5dcc7a6db7f524b4d9af9f4962abd.png">
      <br/>
      这里所使用的数学环境最好放一个数学公式，不然后面对公式进行标号会出现问题。
      <br/>
      这两个例子就非常清晰的展示如何自己手写简单的上下标公式。
     </img>
    </p>

     <a id="_97">

     积分
    </h4>
    <p>
     不定积分的写法是使用
     <code>
      \int{被积函数}{积分单元}
     </code>
     <br/>
     定积分的写法是使用
     <code>
      \int_{下限}^{上限}{被积函数}{积分单元}
     </code>

     <code>
      {}
     </code>

$$\int_1^2 f(x)\,{dx} $$			%(2)
$$\int_{-\infty}^{+\infty}f(x){dx} $$	%(3)
$$ \int \!\!\!\!\! \int_D f(x,y)\,dx\,dy$$	%(4)
$$\int\int_D f(x,y)\,dx\,dy $$					%(5)
