       </span>
      </span>
      <span style="font-family:宋体">

     <span style="font-size:12px">
      <span style="font-family:宋体">
       在
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        ETL
       </span>
      </span>
      <span style="font-family:宋体">
       测试过程中，根据实际情况可以将
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        ETL
       </span>
      </span>
      <span style="font-family:宋体">
       的这些环节拆分或组合成多个测试面，根据不同的测试面进行不同重点的测试。目前可以将
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        ETL
       </span>
      </span>
      <span style="font-family:宋体">
       环节拆分成以下
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        4
       </span>
      </span>
      <span style="font-family:宋体">
       个测试面：
      </span>
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">

      <span style="font-family:宋体">

      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">

       </span>
      </span>
      <span style="font-family:宋体">
       、
      </span>

      </span>
      <span style="font-family:宋体">
       、
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        idl-&gt;adl
       </span>
      </span>
      <span style="font-family:宋体">
       、源数据
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        -&gt;adl

      </span>
      <span style="font-family:宋体">
       测试面，其中源数据

      <span lang="EN-US">
       <span style="font-family:Calibri">
        -&gt;bdl

    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span style="font-size:12px">
      <span style="font-family:宋体">

      </span>
      <span style="font-family:Calibri">
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:-18pt; margin:0cm 0cm 0pt 18pt">
     <span lang="EN-US" style="">
      <span style="">
       <span style="font-family:Calibri; font-size:12px">
        1.
       </span>
       <span style='font:7pt "Times New Roman"'>
       </span>
      </span>
     </span>
     <span style="font-size:12px">
      <span style="">
       <span style="font-family:宋体; background:yellow">
        第一面是源数据到
       </span>
       <span lang="EN-US" style="background:yellow">
        <span style="font-family:Calibri">
         odl
        </span>
       </span>
       <span style="font-family:宋体; background:yellow">
        的测试。
       </span>
      </span>
      <span style="font-family:宋体">
       主要关注以下几个方面（侧重物理数据测试）：
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span style="">
       <span style="font-family:宋体">
        对
       </span>
       <span lang="EN-US">
        <span style="font-family:Calibri">
         ETL
        </span>
       </span>
       <span style="font-family:宋体">
        抽取方案的测试。
       </span>
       <span lang="EN-US">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span style="font-family:宋体">
       首先是抽取方案稳定性，如果源体系表结构有改变，需要保证
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        ETL
       </span>
      </span>
      <span style="font-family:宋体">
       抽取方案不变或者微变。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span style="font-family:宋体">
       其次数据传送接口方案合理性。源系统以何种形式把数据提供给目标系统，源系统推送或者目标系统主动抽取。数据日期、数据大小、记录数、增量
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        or
       </span>
      </span>
      <span style="font-family:宋体">
       全量。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="font-size:12px">
       <span style="font-family:宋体">
        对于抽取策略的测试。
       </span>
       <span lang="EN-US">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span style="font-family:宋体">
       检测抽取策略的合理性。目前常用的抽取策略有全量抽取、增量抽取。对于增量抽取，捕捉变化的数据有如下几种：
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        1)
       </span>
      </span>
      <span style="font-family:宋体">
       采用快照方式。需要业务系统建立
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        insert,update,delete
       </span>
      </span>
      <span style="font-family:宋体">
       触发器。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        2)
       </span>
      </span>
      <span style="font-family:宋体">
       时间戳方式，在业务系统表建一个时间戳字段，一旦数据发生变化，则修改此字段。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        3)
       </span>
      </span>
      <span style="font-family:宋体">
       全表删除插入方式，每次
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        ETL
       </span>
      </span>
      <span style="font-family:宋体">
       操作先将目标表数据删除，然后抽取。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        4)hash
       </span>
      </span>
      <span style="font-family:宋体">
       比对，是全表比对的一个扩展，通过计算主要业务字段的
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        MD5
       </span>
      </span>
      <span style="font-family:宋体">
       校验码存入
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        hash
       </span>
      </span>
      <span style="font-family:宋体">
       维表，通过与
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        hash
       </span>
      </span>
      <span style="font-family:宋体">
       维表的比对进行抽取。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        5)
       </span>
      </span>
      <span style="font-family:宋体">
       日志表方式，跟进业务系统的日志表进行数据抽取。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        6)oracle
       </span>
      </span>
      <span style="font-family:宋体">
       变化数据捕捉，通过分析数据库自身日志判断变化的数据。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="font-size:12px">
       <span style="font-family:宋体">
        对转换规则的测试。
       </span>
       <span lang="EN-US">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <a name="OLE_LINK2" target="_blank">
     </a>
     <a name="OLE_LINK1" target="_blank">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         首先是数据格式的合法性。对于数据源中时间、数值、字符等数据的处理，是否符合数据仓库规则，是否进行统一的转换。
        </span>
       </span>
      </span>
     </a>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         其次是值域的有效性。是否有超出维表或者业务值域的范围。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         第三是空值的处理。是否捕获字段空值，或者需要对空值进行替换为其他含义值的处理。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         第四是主键的有效性。主键是否唯一。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         第五是乱码的检查。特殊符号或者乱码符号的护理规则。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="">
       <span style="font-family:宋体">
        <span style="font-size:12px">
         第六是脏数据的处理。比如不符合业务逻辑的数据，数据孤岛。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="font-size:12px">
       <span style="font-family:宋体">
        加载策略的测试。
       </span>
       <span lang="EN-US">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span lang="EN-US">
       <span style="font-family:Calibri">
       </span>
      </span>
      <span style="font-family:宋体">
       首先是加载策略的合理性。目前常用的加载策略有如下几种：
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        1) trunctae and insert.
       </span>
      </span>
      <span style="font-family:宋体">
       直接清空目标表，然后把新的数据加载进去。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        2) append.
       </span>
      </span>
      <span style="font-family:宋体">
       先根据规则清除当天的记录，然后把当天的新数据追加进去。
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        3) update and insert.
       </span>
      </span>
      <span style="font-family:宋体">
       用新数据与目标表中的历史数据进行比较，有变化的则更新，新记录则直接插入到目标表中。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-size:12px">
      <span lang="EN-US">
       <span style="font-family:Calibri">
        4) keep history change.
       </span>
      </span>
      <span style="font-family:宋体">
       保持历史的变化，通常是拉链记历史的方式实现。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:-18pt; margin:0cm 0cm 0pt 18pt">
     <span lang="EN-US" style="">
      <span style="">
       <span style="font-family:Calibri; font-size:12px">
        2.
       </span>
       <span style='font:7pt "Times New Roman"'>
       </span>
      </span>
     </span>
     <span style="font-size:12px">
      <span style="">
       <span style="font-family:宋体; background:yellow">
        第二面是
       </span>
       <span lang="EN-US" style="background:yellow">
        <span style="font-family:Calibri">
         odl
        </span>
       </span>
       <span style="font-family:宋体; background:yellow">
        到
       </span>
       <span lang="EN-US" style="background:yellow">
        <span style="font-family:Calibri">
         adl
        </span>
       </span>
       <span style="font-family:宋体; background:yellow">
        的测试。
       </span>
      </span>
      <span style="font-family:宋体">
       主要关注以下几个方面（物理数据测试
      </span>
      <span lang="EN-US">
       <span style="font-family:Calibri">
        +
       </span>
      </span>
      <span style="font-family:宋体">
       逻辑数据测试）：
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="">
      <span style="font-size:12px">
       <span style="font-family:宋体">
        对转换规则的测试。
       </span>
       <span lang="EN-US">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-family:宋体">
      <span style="font-size:12px">
       首先是数据格式的合法性。对于数据源中时间、数值、字符等数据的处理，是否符合数据仓库规则，是否进行统一的转换。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-family:宋体">
      <span style="font-size:12px">
       其次是值域的有效性。是否有超出维表或者业务值域的范围。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-family:宋体">
      <span style="font-size:12px">
       第三是空值的处理。是否捕获字段空值，或者需要对空值进行替换为其他含义值的处理。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-family:宋体">
      <span style="font-size:12px">
       第四是主键的有效性。主键是否唯一。
      </span>
     </span>
    </p>
    <p class="MsoListParagraph" style="text-indent:0cm; margin:0cm 0cm 0pt 18pt">
     <span style="font-family:宋体">
      <span style="font-size:12px">
       第五是乱码的检查。特殊符号或者乱码符号的护理规则。
      </span>
