    <h2>
     1.远程连接JQuery的库
    </h2>
    <p>
     站点是在国内，建议使用百度、新浪、又拍云等国内CDN地址，
    </p>
    <p>
     站点是在国外，建议使用谷歌和微软的CDN地址
     <br/>
     使用CDN地址引用jQuery的具体方法如下：
     <br/>
     &lt;head&gt;
     <br/>
     百度 CDN：
     <br/>
     &lt;script src="https://apps.bdimg.com/libs/jquery/2.1.4/jquery.min.js"&gt;
    </p>
    <p>
     新浪 CDN：
     <br/>
     &lt;script src="https://lib.sinaapp.com/js/jquery/2.0.2/jquery-2.0.2.min.js"&gt;
     <br/>
     又拍云 CDN：
     <br/>
     &lt;script src="https://upcdn.b0.upaiyun.com/libs/jquery/jquery-2.0.2.min.js"&gt;
     <br/>
     Staticfile CDN：
     <br/>
     &lt;script src="https://cdn.staticfile.org/jquery/1.10.2/jquery.min.js"&gt;
     <br/>
     Google CDN：
     <br/>
     &lt;script src="https://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"&gt;
     <br/>
     Microsoft CDN：
     <br/>
     &lt;script src="https://ajax.aspnetcdn.com/ajax/jquery/jquery-1.9.0.min.js"&gt;
    </p>
    <p>
     &lt;/script&gt;
     <br/>
     &lt;/head&gt;
    </p>
    <h2>
     2.借助前端页面的开发工具
    </h2>
    <p>
     也可以使用Hbuilder，创建前端页面的项目时，新建js文件选择带有Jquery库的模板，这样建立Js文件就可以直接使用Jquery,在该文件中写我们的Jquery代码，在需要的前端页面将此Js文件引入就可以了。
    </p>
