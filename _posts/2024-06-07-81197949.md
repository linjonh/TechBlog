    <p>
     <br/>
     最近在学习微信小程序开发，半个月学习下来，很想实战一下踩踩坑，于是就仿写了一个滴滴他们家的青桔单车小程序的前端实现，过程一言难尽，差不多两周时间过去了，发现小程序的坑远比想象的要多的多！！在实际练手中，完全是黑盒的，看到人家上线的小程序的效果，纯靠推测，部分效果在绞尽脑汁后能做出大致的实现，但是有些细节，费劲全力都没能做出来。很想一窥源码，查看究竟，看看大厂的前端大神们是如何规避了小程序的各种奇葩的坑。
    </p>
    <p>
    </p>
    <p>
     于是就想到获取到小程序地源文件，然后再对其进行反编译还原为源代码，来作为学习参考。我百度了各种关于小程序地反编译教程，但是感觉都不太适合像我这样地初学小白，踩了挺多坑。在这里把我重新简化好的，快速地获取一个微信小程序源码的方式记录下来。
    </p>
    <h2 id="一简单聊一下xxxxxwxapkg">
     <a name="t0">
     </a>
     一、简单聊一下
     <code>
      xxxxx.wxapkg
     </code>
    </h2>
    <p>
     <strong>
      先来想想一个很简单的问题
     </strong>
     ，小程序的源文件存放在哪
     <strong>
      ？
     </strong>
    </p>
    <ul>
     <li>
      当然是在微信的服务器上。
     </li>
    </ul>
    <p>
     <strong>
      但是在微信服务器上，普通用户想要获取到，肯定是十分困难的，有没有别的办法呢？
     </strong>
    </p>
    <ul>
     <li>
      简单思考一下我们使用小程序的场景就会明白，当我们点开一个微信小程序的时候，其实是微信已经将它的从服务器上下载到了手机，然后再来运行的。
     </li>
     <li>
      所以，虽然我们没能力从
      <code>
       服务器
      </code>
      上获取到，但是我们应该可以从
      <code>
       手机本地
      </code>
      找到到已经下载过的小程序源文件
     </li>
    </ul>
    <p>
     <strong>
      那么如何才能在手机里找到小程序的源文件包呢？
     </strong>
    </p>
    <ul>
     <li>
      这里只以安卓手机为例，毕竟穷逼不曾拥有过苹果手机
     </li>
     <li>
      具体目录位置直接给出：
      <code>
       /data/data/com.tencent.mm/MicroMsg/{一串16进制字符}/appbrand/pkg/
      </code>
     </li>
     <li>
      <p>
       在这个目录下，会发现一些
       <code>
        xxxxxxx.wxapkg
       </code>
       类型的文件，这些就是微信小程序的包
      </p>
      <p>
       <strong>
        微信小程序的格式就是:
        <code>
         .wxapkg
        </code>
       </strong>
      </p>
     </li>
     <li>
      <code>
       .wxapkg
      </code>
      是一个二进制文件，有其自己的一套结构。
     </li>
     <li>
      关于
      <code>
       .wxapkg
      </code>
      的详细内容可以参考lrdcq大神的博文：
      <a href="http://lrdcq.com/me/read.php/66.htm" rel="nofollow">
       微信小程序源码阅读笔记
      </a>
     </li>
     <li>
      但是这里有个
      <strong>
       坑
      </strong>
      ，想要进入到上面这个目录的话，用手机自带的文件管理器肯定是不行的，
      <code>
       安卓
      </code>
      或者
      <code>
       iPhone
      </code>
      都要要用到第三方的文件管理器，比如：
      <code>
       RE文件管理器
      </code>
      ，并且安卓需要取得root权限，而苹果手机肯定是要越狱的，且
      <strong>
       iphone的越狱难度&gt;&gt;安卓获取root
      </strong>
      ，不管越狱还是root，这都太费劲，当然有能力的同学可以直接从手机上来操作，但是这里
      <code>
       不推荐从真机上获取
      </code>
      。
     </li>
    </ul>

      <a href="https://github.com/qwerty472123/wxappUnpacker">

    <h2 id="三详细步骤">
     <a name="t2">
     </a>
     三、详细步骤
    </h2>
    <p>
     <strong>
      使用安卓模拟器获取到.wxapkg文件
     </strong>
     <br/>
     不用越狱，不用root，使用电脑端的
     <strong>
      安卓模拟器
     </strong>
     来获取是一个
     <strong>
      非常简单快捷且万能的
     </strong>
     获取方式，具体步骤如下：
     <br/>
     1.打开安装好的安卓模拟器，并在模拟器中安装
     <code>
      QQ
     </code>
     、
     <code>
      微信
     </code>
     、
     <code>
      RE管理器
     </code>
    </p>
    <ul>
     <li>
      <code>
       QQ
      </code>
      、
      <code>
       微信
      </code>
      在模拟器自带的应用商店里搜索下载安装即可
     </li>
     <li>
      <code>
       RE管理器
      </code>
      的下载地址：
      <a href="https://pan.baidu.com/s/1PPBx08rNutXxhlMMJbuTpQ" rel="nofollow">
       https://pan.baidu.com/s/1PPBx08rNutXxhlMMJbuTpQ
      </a>
     </li>
     <li>
      下载好后直接拖拽进打开的模拟器窗口就会自动安装
     </li>
    </ul>
    <p>
     2.设置一下模拟器
    </p>
    <ul>
     <li>
      以我个人认为比较好用的
      <code>
       夜神模拟器
      </code>
      举例
     </li>
     <li>
      首先到模拟器内部设置超级用户权限
     </li>
    </ul>
    <p>
     <img alt="这里写图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/dfcea862ca35b2823b5baa1579a72872.png">
      <br/>
      <img alt="这里写图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/a620f3486b17f2497dc448015f2574dc.png"/>
     </img>
    </p>
    <ul>
     <li>
      这些操作的目的都是为了能让
      <code>
       RE管理器
      </code>
      顺利的获取到ROOT权限
     </li>
    </ul>
    <p>
     3.接下来在模拟器里打开微信，然后在微信中运行你想要获取的下程序（这其实是让微信把小程序的源文件包从服务器下载到了本地了）
    </p>
    <ul>
     <li>
      就以我说的这款青桔单车的小程序举例（希望滴滴的大神不会想打死我~）
     </li>
     <li>
      在模拟器微信中运行一下后，
      <strong>
       直接切回模拟器桌面运行RE浏览器
      </strong>
      来到目录
     </li>
     <li>
      <code>
       /data/data/com.tencent.mm/MicroMsg//appbrand/pkg/
      </code>
     </li>
     <li>
      <p>
       就抵达了目的文件夹
       <br/>
       <img alt="这里写图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/ebe7292ef45d70276f523ebd7865dac3.png"/>
      </p>
     </li>
     <li>
      <p>
       你会看到发现里面的一些.wxapkg后缀的文件，就是它们没错啦，可以根据使用的时间来判断那个是你刚才从服务器下载过来的
      </p>
     </li>
     <li>
      一般小程序的文件不会太大，可以结合时间来判断，长按压缩所选文件,然后再将压缩好的包通过QQ发送到
      <code>
       我的电脑
      </code>
     </li>
     <li>
      <p>
       如果不进行压缩的话，是无法将这个文件通过QQ来发送的
       <br/>
       <img alt="这里写图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/369216a51476de00de40f537270c231f.png"/>
      </p>
     </li>
     <li>
      <p>
       所以QQ的这个功能可以让我们很方便的拿到源文件，而不必到电脑目录去找模拟器的文件目录。
      </p>
     </li>
     <li>
      解压。这样几步简单操作，就成功拿到了小程序的源文件了。
     </li>
    </ul>
    <h2 id="四使用反编译脚本解包-wxapkg">
     <a name="t3">

     </li>
     <li>
      打开nodejs命令窗口
     </li>
    </ul>
    <p>
     （node命令行如图）
     <br/>
     <img alt="这里写图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/322e2c00ae4af442f96d5fb012b9d40f.png">
      <br/>
      （不是上面那个绿色图标的Node.js）
     </img>
    </p>
    <p>
     <strong>
      打开node命令行之后，可以看到此时命令行所在的目录
     </strong>
     ：
     <br/>
     （图中的是我的命令行目录，
     <strong>

     <strong>

      里面,提示找不到那个文件，就将pages下注册的该页面语句暂时删除掉。删
      <br/>

      <br/>
      4、还是无法运行？那就自己想办法吧！

    <h2 id="五两步即可完成">

     五、两步即可完成
    </h2>
    <p>
     至此我们就通过非常简单的方式获取到了一个想要的小程序源文件，并对其进行了反编译还原 以后想要再反编译其他的小程序，非常快速，
     <strong>
      只需要两步
     </strong>
    </p>
    <p>
     1.
     <strong>
      使用模拟器找到小程序.wxapkg文件
     </strong>
     <br/>
     2.
     <strong>
      使用nodejs反编译脚本将.wxapkg文件反编译
     </strong>
    </p>
    <p>
     <strong>
      使用此方法，绝大部分的小程序都能正常反编译出来，但是也会有一些特殊的情况，具体可以查看
      <code>
       qwerty472123
      </code>
      大神的
      <code>
       readme
      </code>
      文件
     </strong>
    </p>
    <h2 id="六写在后面的话">
     <a name="t5">
     </a>
     六、写在后面的话
    </h2>
    <p>
     早在跳一跳小游戏火的时候，就有人通过小程序的appid和版本号，构造URL直接从服务器端下载该小程序的源码wxapkg文件。但是这种方法很快就被封了。
    </p>
    <p>
     上线的源代码能如此简单的被获取到，不得不说小程序的源码安全存在一些隐患，有些小程序会将重要的js逻辑代码揉在一个js文件里，这样，即使被获取了源码，也很难读懂，不过现在越来越多的小程序使用
     <strong>
      mpvue wepy
     </strong>
     之类的框架来开发，使得代码的安全性又提高不少。
    </p>
    <blockquote>
     <p>
      这里介绍的方法只想提供给像我一样的小程序新手用来学习和仿写练手，获
      <br/>
