    <p>
     Python是一种高级编程语言，它具有易学易用、跨平台等优点，因此在开发中得到了广泛的应用。
    </p>
    <p>
     然而，Python代码需要在Python解释器中运行，这对于一些用户来说可能不太方便。
    </p>
    <p>
     因此，将Python代码
     <span style="color:#fe2c24;">
      <strong>
       打包成可执行文件（exe）
      </strong>
     </span>
     是一种很好的解决方案。
    </p>
    <p>
     本文将介绍6种将Python代码打包成exe应用的方式。
    </p>
    <h4 id="%E4%BD%BF%E7%94%A8pyinstallerpyinstaller%E6%98%AF%E4%B8%80%E4%B8%AA%E6%B5%81%E8%A1%8C%E7%9A%84Python%E6%89%93%E5%8C%85%E5%B7%A5%E5%85%B7%EF%BC%8C%E5%AE%83%E5%8F%AF%E4%BB%A5%E5%B0%86Python%E4%BB%A3%E7%A0%81%E6%89%93%E5%8C%85%E6%88%90%E7%8B%AC%E7%AB%8B%E7%9A%84%E5%8F%AF%E6%89%A7%E8%A1%8C%E6%96%87%E4%BB%B6%E3%80%82">
     使用pyinstaller
     <br/>
     <span style="color:#fe2c24;">
      pyinstaller
     </span>
     是一个流行的Python打包工具，它可以将Python代码打包成独立的可执行文件。
    </h4>
    <p>
     使用pyinstaller打包Python代码非常简单，只需要在命令行中输入以下命令：
    </p>
    <p>
     pyinstaller your_script.py
     <br/>
     这将生成一个可执行文件，可以在Windows、Linux和MacOS上运行。
    </p>
    <p>
    </p>
    <h5 id="Python%20PyInstaller%E5%AE%89%E8%A3%85%E5%92%8C%E4%BD%BF%E7%94%A8%E6%95%99%E7%A8%8B%EF%BC%88%E8%AF%A6%E8%A7%A3%E7%89%88%EF%BC%89%C2%A0%E5%9C%A8%E5%88%9B%E5%BB%BA%E4%BA%86%E7%8B%AC%E7%AB%8B%E5%BA%94%E7%94%A8%EF%BC%88%E8%87%AA%E5%8C%85%E5%90%AB%E8%AF%A5%E5%BA%94%E7%94%A8%E7%9A%84%E4%BE%9D%E8%B5%96%E5%8C%85%EF%BC%89%E4%B9%8B%E5%90%8E%EF%BC%8C%E8%BF%98%E5%8F%AF%E4%BB%A5%E4%BD%BF%E7%94%A8%20PyInstaller%20%E5%B0%86%20Python%20%E7%A8%8B%E5%BA%8F%E7%94%9F%E6%88%90%E5%8F%AF%E7%9B%B4%E6%8E%A5%E8%BF%90%E8%A1%8C%E7%9A%84%E7%A8%8B%E5%BA%8F%EF%BC%8C%E8%BF%99%E4%B8%AA%E7%A8%8B%E5%BA%8F%E5%B0%B1%E5%8F%AF%E4%BB%A5%E8%A2%AB%E5%88%86%E5%8F%91%E5%88%B0%E5%AF%B9%E5%BA%94%E7%9A%84%20Windows%20%E6%88%96%20Mac%20OS%20X%20%E5%B9%B3%E5%8F%B0%E4%B8%8A%E8%BF%90%E8%A1%8C%E3%80%82%E5%AE%89%E8%A3%85%20PyInstallePython%20%E9%BB%98%E8%AE%A4%E5%B9%B6%E4%B8%8D%E5%8C%85%E5%90%AB%20PyInstaller%20%E6%A8%A1%E5%9D%97%EF%BC%8C%E5%9B%A0%E6%AD%A4%E9%9C%80%E8%A6%81%E8%87%AA%E8%A1%8C%E5%AE%89%E8%A3%85%20PyInstaller%20%E6%A8%A1%E5%9D%97%E3%80%82">
     <span style="color:#fe2c24;">
      Python PyInstaller安装和使用教程（详解版）
     </span>
     <span style="color:#6eaad7;">
      <br/>
      在创建了独立应用（自包含该应用的依赖包）之后，还可以使用 PyInstaller 将 Python 程序生成可直接运行的程序，这个程序就可以被分发到对应的 Windows 或 Mac OS X 平台上运行。
      <br/>
      安装 PyInstallePython 默认并不包含 PyInstaller 模块，因此需要自行安装 PyInstaller 模块。
     </span>
    </h5>
    <p>
     <span style="color:#6eaad7;">
      安装 PyInstaller 模块与安装其他 Python 模块一样，使用 pip 命令安装即可。在命令行输入如下命令：
      <br/>
      pip install pyinstaller
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      强烈建议使用 pip 在线安装的方式来安装 PyInstaller 模块，不要使用离线包的方式来安装，因为 PyInstaller 模块还依赖其他模块，pip 在安装 PyInstaller 模块时会先安装它的依赖模块。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      运行上面命令，应该看到如下输出结果：
      <br/>
      Successfully installed pyinstaller-x.x.x
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      其中的 x.x.x 代表 PyInstaller 的版本。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      在 PyInstaller 模块安装成功之后，在 Python 的安装目录下的 Scripts(D:\Python\Python36\Scripts) 目录下会增加一个 pyinstaller.exe 程序，接下来就可以使用该工具将 Python 程序生成 EXE 程序了。
      <br/>
      PyInstaller生成可执行程序PyInstaller 工具的命令语法如下：
      <br/>
      pyinstaller 选项 Python 源文件
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      不管这个 Python 应用是单文件的应用，还是多文件的应用，只要在使用 pyinstaller 命令时编译作为程序入口的 Python 程序即可。
      <br/>
      PyInstaller工具是跨平台的，它既可以在 Windows平台上使用，也可以在 Mac OS X 平台上运行。在不同的平台上使用 PyInstaller 工具的方法是一样的，它们支持的选项也是一样的。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      下面先创建一个 app 目录，在该目录下创建一个 app.py 文件，文件中包含如下代码：
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      from say_hello import *
      <br/>
      def main():
      <br/>
      print('程序开始执行')
      <br/>
      print(say_hello('孙悟空'))
      <br/>
      # 增加调用main()函数
      <br/>
      if __name__ == '__main__':
      <br/>
      main()
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#6eaad7;">
      接下来使用命令行工具进入到此 app 目录下，执行如下命令：
      <br/>
      pyinstaller -F app.py
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      执行上面命令，将看到详细的生成过程。当生成完成后，将会在此 app 目录下看到多了一个 dist 目录，并在该目
     </span>
     <span style="color:#fe2c24;">
      录下看到有一个 app.exe 文件，这就是使用 PyInstaller 工具生成的 EXE 程序
     </span>
     <span style="color:#6eaad7;">
      。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      在命令行窗口中进入 dist 目录下，在该目录执行 app.exe ，将会看到该程序生成如下输出结果：
      <br/>
      程序开始执行
      <br/>
      孙悟空，您好！
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      由于该程序没有图形用户界面，因此如果读者试图通过双击来运行该程序，则只能看到程序窗口一闪就消失了，这样将无法看到该程序的输出结果。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      在上面命令中使用了-F 选项，该选项指定生成单独的 EXE 文件，因此，在 dist 目录下生成了一个单独的大约为 6MB 的 app.exe 文件（在 Mac OS X 平台上生成的文件就叫 app，没有后缀）；与 -F 选项对应的是 -D 选项（默认选项），该选项指定生成一个目录（包含多个文件）来作为程序。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      下面先将 PyInstaller 工具在 app 目录下生成的 build、dist 目录删除，并将 app.spec 文件也删除，然后使用如下命令来生成 EXE 文件。
     </span>
     <br/>
     <span style="color:#fe2c24;">
      pyinstaller -D app.py
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      执行上面命令，将看到详细的生成过程。当生成完成后，将会在 app 目录下看到多了一个 dist 目录，并在该目录下看到有一个 app 子目录，在该子目录下包含了大量 .dll 文件和 .pyz 文件，它们都是 app.exe 程序的支撑文件。在命令行窗口中运行该 app.exe 程序，同样可以看到与前一个 app.exe 程序相同的输出结果。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      PyInstaller 不仅支持 -F、-D 选项，而且也支持如表 1 所示的常用选项。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      表 1 PyInstaller 支持的常用选项
      <br/>
      -h，--help    查看该模块的帮助信息
     </span>
     <br/>
     <span style="color:#fe2c24;">
      <strong>
       -F，-onefile    产生单个的可执行文件
      </strong>
     </span>
     <br/>
     <strong>
      <span style="color:#fe2c24;">
       -D，--onedir    产生一个目录（包含多个文件）作为可执行程序
      </span>
     </strong>
     <br/>
     <span style="color:#6eaad7;">
      -a，--ascii    不包含 Unicode 字符集支持
      <br/>
      -d，--debug    产生 debug 版本的可执行文件
      <br/>
      -w，--windowed，--noconsolc    指定程序运行时不显示命令行窗口（仅对 Windows 有效）
      <br/>
      -c，--nowindowed，--console    指定使用命令行窗口运行程序（仅对 Windows 有效）
      <br/>
      -o DIR，--out=DIR    指定 spec 文件的生成目录。如果没有指定，则默认使用当前目录来生成 spec 文件
      <br/>
      -p DIR，--path=DIR    设置 Python 导入模块的路径（和设置 PYTHONPATH 环境变量的作用相似）。也可使用路径分隔符（Windows 使用分号，Linux 使用冒号）来分隔多个路径
      <br/>
      -n NAME，--name=NAME    指定项目（产生的 spec）名字。如果省略该选项，那么第一个脚本的主文件名将作为 spec 的名字
      <br/>
      在表 1 中列出的只是 PyInstaller 模块所支持的常用选项，如果需要了解 PyInstaller 选项的详细信息，则可通过 pyinstaller -h 来查看。
     </span>
    </p>
    <p>
     <br/>
     <span style="color:#6eaad7;">
      下面再创建一个带图形用户界面，可以访问 MySQL 数据库的应用程序。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      在 app 当前所在目录再创建一个 dbapp 目录，并在该目录下创建 Python 程序，其中 exec_select.py 程序负责查询数据，main.py 程序负责创建图形用户界面来显示查询结果。
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      exec_select.py 文件包含的代码如下：
     </span>
     <br/>
    </p>
    <p>
     <span style="color:#6eaad7;">
      import mysql.connector
      <br/>
      def query_db():
      <br/>
      #
     </span>
     <span style="color:#fe2c24;">
      ①、连接数据库
     </span>
     <br/>
     <span style="color:#6eaad7;">
      conn = conn = mysql.connector.connect(user='root', password='32147',
      <br/>
      host='localhost', port='3306',
      <br/>
      database='python', use_unicode=True)
      <br/>
      #
     </span>
     <span style="color:#fe2c24;">
      ②、获取游标
     </span>
     <br/>
     <span style="color:#6eaad7;">
      c = conn.cursor()
      <br/>
      #
     </span>
     <span style="color:#fe2c24;">
      ③、调用执行select语句查询数据
     </span>
     <br/>
     <span style="color:#6eaad7;">
     </span>
     <span style="color:#fe2c24;">
      c.execute
     </span>
     <span style="color:#6eaad7;">
      ('select * from user_tb where user_id &gt; %s', (2,))
      <br/>
      # 通过游标的description属性获取列信息
      <br/>
      description = c.description
      <br/>
      # 使用fetchall获取游标中的所有结果集
      <br/>
      rows = c.fetchall()
      <br/>
      # ④、关闭游标
      <br/>
      c.close()
      <br/>
      # ⑤、关闭连接
      <br/>
      conn.close()
      <br/>
      return description, rows
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#6eaad7;">
      mian.py 文件包含的代码如下：
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      from exec_select import *
      <br/>
      from tkinter import *
     </span>
    </p>
    <p>
     <span style="color:#6eaad7;">
      def main():
      <br/>
      description, rows = query_db()
      <br/>
      # 创建窗口
      <br/>
      win = Tk()
      <br/>
      win.title('数据库查询')
      <br/>
      # 通过description获取列信息
      <br/>
      for i, col in enumerate(description):
      <br/>
      lb = Button(win, text=col[0], padx=50, pady=6)
      <br/>
      lb.grid(row=0, column=i)
      <br/>
      # 直接使用for循环查询得到的结果集
      <br/>
      for i, row in enumerate(rows):
      <br/>
      for j in range(len(row)):
      <br/>
      en = Label(win, text=row[j])
      <br/>
      en.grid(row=i+1, column=j)
      <br/>
      win.mainloop()
      <br/>
      if __name__ == '__main__':
      <br/>
      main()
     </span>
    </p>
