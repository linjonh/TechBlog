    <h2>
     <a id="_0">
     </a>
     一、文件配置
    </h2>
    <p>
     在桌面新建文件Start.txt 复制以下内容，保存后修改后缀名.txt为 .reg。
    </p>
    <pre><code class="prism language-bash">Windows Registry Editor Version <span class="token number">5.00</span>
<span class="token punctuation">[</span>HKEY_CLASSES_ROOT<span class="token punctuation">\</span>Start<span class="token punctuation">]</span>
<span class="token string">"URL Protocol"</span><span class="token operator">=</span><span class="token string">"D:<span class="token entity" title="\\">\\</span>Task<span class="token entity" title="\\">\\</span>app.exe"</span>
@<span class="token operator">=</span><span class="token string">"Start Protocol"</span>
<span class="token punctuation">[</span>HKEY_CLASSES_ROOT<span class="token punctuation">\</span>Start<span class="token punctuation">\</span>DefaultIcon<span class="token punctuation">]</span>
@<span class="token operator">=</span><span class="token string">"D:<span class="token entity" title="\\">\\</span>Task<span class="token entity" title="\\">\\</span>app.exe"</span>
<span class="token punctuation">[</span>HKEY_CLASSES_ROOT<span class="token punctuation">\</span>Start<span class="token punctuation">\</span>shell<span class="token punctuation">]</span>
<span class="token punctuation">[</span>HKEY_CLASSES_ROOT<span class="token punctuation">\</span>Start<span class="token punctuation">\</span>shell<span class="token punctuation">\</span>open<span class="token punctuation">]</span>

</code></pre>
    <h2>
     <a id="reg_15">
     </a>
     二、双击运行.reg，一路点确认
    </h2>
    <h2>
     <a id="_16">
     </a>
     三、页面打开
    </h2>
    <p>
     浏览器网址输入：Start://，回车，
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ca496ea12810430a91adf867ddaff9bb.png">
      <br/>
      点击打开app.exe按钮，exe桌面就打开了。
     </img>
    </p>
    <h2>
     <a id="_20">
     </a>
     四、前端按钮添加
    </h2>
    <p>
     在前端代码点击函数中输入以下代码
    </p>
    <pre><code class="prism language-css">window.location.href = <span class="token string">'Start://'</span>
</code></pre>
