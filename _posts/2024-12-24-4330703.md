               <h3>
                DELPHI的开源控件集
               </h3>
              </div>
              <div class="postbody">
               <p>
                <strong>
                 ZeosLib
                </strong>
                <a href="http://sourceforge.net/projects/zeoslib" rel="nofollow noopener noreferrer" target="_blank">
                 <strong>
                  <span style="color: #000000;">
                   http://sourceforge.net/projects/zeoslib
                  </span>
                 </strong>
                </a>
                <br/>
                一套开源的数据库控件，可以支持多数据库，相当不错：
                <br/>
                MSSQL、Interbase、FireBird、ProgreSQL、DB2、SYBASE、ORACLE、MySQL
                <br/>
                <br/>
                <strong>
                 RxLib
                 <a href="http://www.sourceforge.net/projects/rxlib" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.sourceforge.net/projects/rxlib
                  </span>
                 </a>
                </strong>
                <br/>
                老牌控件包，俄罗斯出品，很早以前的经典，包含60多个常用改良的控件。
                <br/>
                <br/>
                <strong>
                 Indy9 and Indy10
                 <a href="http://www.indyproject.org/Sockets/Download/DevSnapshot.en.aspx" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.indyproject.org/Sockets/Download/DevSnapshot.en.aspx
                  </span>
                 </a>
                </strong>
                <br/>
                这个就不用多说了，大家都知道的网络控件。
                <br/>
                <br/>
                <strong>
                 Internet Component Suite - ICS
                 <a href="http://www.overbyte.be/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.overbyte.be
                  </span>
                 </a>
                </strong>
                <br/>
                ICS是为构建客户端/服务器和因特网应用的组建集.它支持所有主要的TCP/IP协议,例如TCP,UDP,SMTP,POP3,NNTP,HTTP,PING,TELNET,FINGER等.
                <a href="http://www.turbopower.com/" rel="nofollow noopener noreferrer" target="_blank">
                 <span style="color: #000000;">
                  http://www.turbopower.com/
                 </span>
                </a>
                <br/>
                TurboPower 控件原来是收费的，都不错。
                <br/>
                <br/>
                <strong>
                 TwoDesk Components
                 <a href="http://www.twodesk.com/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.twodesk.com/
                  </span>
                 </a>
                </strong>
                <br/>
                这是著名的Castalia老家。TwoDesk也是开源的。
                <br/>
                <br/>
                <strong>
                 Fastcode Challenges
                 <a href="http://fastcode.sourceforge.net/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://fastcode.sourceforge.net/
                  </span>
                 </a>
                 <br/>
                 Fastcode Project
                 <a href="http://www.fastcodeproject.org/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.fastcodeproject.org/
                  </span>
                 </a>
                </strong>
                <br/>
                <br/>
                <strong>
                 JVCL
                 <a href="http://jvcl.sourceforge.net/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://jvcl.sourceforge.net
                  </span>
                 </a>
                 <br/>
                 JCL
                 <a href="http://jcl.sourceforge.net/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://jcl.sourceforge.net
                  </span>
                 </a>
                </strong>
                <br/>
                <br/>
                <strong>
                 Jordan Russell's Software
                 <a href="http://www.innosetup.org/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.innosetup.org
                  </span>
                 </a>
                </strong>
                <br/>
                开源的安装工具，拿过N多大奖，它出品和TB97和TB2000控件到现在为止还是最好的开源工具栏控件，delphi自带的工具栏控件和它一比就是垃圾。
                <br/>
                <br/>
                <strong>
                 Delphi Area
                 <a href="http://www.delphiarea.com/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.delphiarea.com
                  </span>
                 </a>
                </strong>
                <br/>
                有很多非常牛的控件，相信用过picshow的控件的程序让很多人都大吃一惊。
                <br/>
                <br/>
                <strong>
                 Project JEDI Portal
                 <a href="http://www.delphi-jedi.org/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://www.delphi-jedi.org
                  </span>
                 </a>
                </strong>
                <br/>
                <br/>
                <br/>
                <strong>
                 <span style="font-size: 14px;">
                  ┊常用精品工具┊
                 </span>
                </strong>
                <br/>
                <br/>
                <strong>
                 log4delphi
                 <a href="http://log4delphi.sourceforge.net/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://log4delphi.sourceforge.net/
                  </span>
                 </a>
                </strong>
                <br/>
                log4j的Delphi移植版，一个通用的日志类库。
                <br/>
                <br/>
                <strong>
                 FastMM
                 <a href="http://fastmm.sourceforge.net/" rel="nofollow noopener noreferrer" target="_blank">
                  <span style="color: #000000;">
                   http://fastmm.sourceforge.net/
                  </span>

                <br/>

                <br/>
                <br/>
                <strong>
                 DelphiSpeedUp
                 <a href="http://andy.jgknet.de/dspeedup" rel="nofollow noopener noreferrer" target="_blank">

                   http://andy.jgknet.de/dspeedup

                <br/>
                <strong>
                 cnPack组件包 CnWizards

                   http://www.cnpack.org/

                <br/>

                   http://www.tortoisesvn.org/
