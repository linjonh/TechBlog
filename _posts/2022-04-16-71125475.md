     <span style="margin:0px; padding:0px">

     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      <span style="margin:0px; padding:0px">
       （基础设施即服务）
      </span>
      ：提供虚拟机或者其他
     </span>
     基础资源作为服务提供给用户。用户可以从供应商那里获得虚拟机或者存储等资源来装载相关的应用，同时这些基础设施的繁琐的管理工作将由
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      IaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      供应商来处理。
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      IaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      主要的用户是企业的系统管理员和运维人员。
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">
    </p>

     <span style="margin:0px; padding:0px">
      PaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      <span style="margin:0px; padding:0px">
       （平台即服务）
      </span>
      ：把开发平台作为服务提供给用户。用户可以在一个包括
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      SDK
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      ，文档和测试环境等在内的开发平台上非常方便地编写应用，而且不论是在部署，或者在运行的时候，用户都无需为服务器，操作系统，网络和存储等资源的管理操心，这些繁琐的工作都由
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      PaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      供应商负责处理。
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      PaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      主要的用户是企业开发人员。
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">
     <span style="margin:0px; padding:0px">
      SaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      <span style="margin:0px; padding:0px">
       （软件即服务）
      </span>
      ：将应用作为服务提供给客户。用户只要接上网络，并通过浏览器，就能直接使用在云端上运行的应用，而不需要顾虑类似安装等琐事，并且免去初期高昂的软硬件投入。
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      SaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      主要面对的是普通的用户。
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">

     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      <span style="margin:0px; padding:0px">
       （容器即服务）

     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">

      PaaS

      两个层级的功能。相

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      IaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      和
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      PaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      服务，
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      CaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      PaaS
     </span>

      更灵活，而对上层应用的操控又比

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">

      VM
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      更细粒度的虚拟化服务，所以能够对计算资源做到更高效的利用。
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
      CaaS
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">

    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">

    </h2>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:19px; font-size:13px; font-family:Verdana,Arial,Helvetica,sans-serif; background-color:rgb(254,254,242)">
     技术关键字解析：
    </p>
    <ul style="margin:0px 0px 0px 30px; padding:0px; word-break:break-all; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19px; background-color:rgb(254,254,242)">
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       集群框架：
      </span>
      容器集群管理及编排系统。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       代码持续集成：
      </span>
      完全自动化的代码管理，包括源代码存储、编译、连接、测试、发布整个创建过程都应该自动完成。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       多租户管理：
      </span>
      实现多用户的环境下共用相同的系统或程序组件，并且仍可确保各用户间资源的隔离性。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       跨主机集群：
      </span>
      可以将本地云平台内部容器资源，平台外部的客户自有容器资源，及各种第三方公有云容器资源结合到一起形成混合容器云资源模式。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       镜像仓库：
      </span>
      提供本地公有/
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
       私有
      </span>
      镜像仓库，提供第三方镜像仓库或DockerHub
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
       镜像资源集成。
      </span>
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       应用编排服务：
      </span>
      使容器之间能够通信、彼此可以传递运行期
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
       ,
      </span>
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
       同时管理多个容器的行为。当容器集群共同构建应用架构时，需要考虑集群环境中的容器，哪些端口需要暴露、哪些卷需要挂载等信息。
      </span>
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       专属主机：
      </span>
      即从本地云平台中提供给用户的专属虚拟主机环境用于构建容器集群应用。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      <span style="margin:0px; padding:0px">
       APaaS
       <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
        服务
       </span>
       ：

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">

     <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
      容器云为开发者和企业提供了一个快速构建、集成、部署、运行容器化应用的平台，从而提高应用开发的迭代效率，简化运维环节，降低运维成本。

     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      1.首先作为公司内部技术纽带，多应用统一部署平台，提高开发效率，减少运维成本。
     </li>

      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">

      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">

      </span>
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:'Times New Roman'">
       CaaS
      </span>
      <span style="margin:0px; padding:0px; line-height:1.5; font-family:宋体">
       即使短期内经济上收益甚微，但会积累宝贵的客户资源及数据资源，这些无形资产最终会为公司赢得巨大战略价值（更广泛的资源整合，更多的合作伙伴加盟等）。
      </span>
