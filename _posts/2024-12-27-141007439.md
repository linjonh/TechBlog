    <h2>
     EhViewer 安装与使用指南
    </h2>
    <p>
     <a class="has-card" href="https://gitcode.com/gh_mirrors/ehvi/EhViewer/?utm_source=artical_gitcode&amp;index=top&amp;type=card&amp;webUrl" title="EhViewer">

    <h3>
     1. 项目介绍
    </h3>
    <p>
     EhViewer 是一个基于 Android 的开源应用程序，旨在为用户提供轻量级且高效的 E-Hentai 网站浏览体验。该项目由多个开发者共同维护，包括 Hippo Seven、NekoInverter 和 Tarsin Norbin。EhViewer 实现了Material Design 3界面风格，并支持动态色彩，同时还提供了画廊的阅读、下载、收藏、评分和评论等特性。该应用遵循 GPL v3 许可协议，可在多种设备上运行，并支持多国语言。
    </p>
    <h3>
     2. 项目快速启动
    </h3>
    <h4>
     环境要求
    </h4>
    <p>
     确保您的设备满足以下条件：
    </p>
    <ul>
     <li>
      Android 8.0+（对于较旧版本的 Android，支持可能有限）
     </li>
     <li>
      允许安装来自第三方来源的应用程序
     </li>
    </ul>
    <h4>
     安装步骤
    </h4>
    <ol>
     <li>
      打开浏览器，前往
      <a href="https://github.com/EhViewer-NekoInverter/EhViewer/releases">
       EhViewer 的 GitHub Release 页面
      </a>
      。
     </li>
     <li>
      下载适用于您设备的最新 APK 文件。
     </li>
     <li>
      在您的设备设置中，允许从未知来源安装应用程序（通常在安全设置中找到）。
     </li>
     <li>
      找到下载的 APK 文件并点击进行安装。
     </li>
    </ol>
    <h4>
     运行应用
    </h4>
    <ol>
     <li>
      安装完成后，打开 EhViewer 应用。
     </li>
     <li>
      输入 E-Hentai 的用户名和密码以登录您的账户（如果您有的话），或者直接开始浏览公共内容。
     </li>
     <li>
      探索应用的各种功能，如搜索漫画、下载画廊或管理已收藏的内容。
     </li>
    </ol>
    <h3>
     3. 应用案例和最佳实践
    </h3>
    <ul>
     <li>
      <strong>
       离线阅读
      </strong>
      ：通过提前下载画廊，可以在无网络连接时享受阅读体验。
     </li>
     <li>
      <strong>
       收藏夹管理
      </strong>
      ：定期整理收藏的画廊，保持有序性。
     </li>
     <li>
      <strong>
       设置偏好
      </strong>
      ：根据自己的喜好调整图像质量、加载速度和夜间模式等设置。
     </li>
     <li>
      <strong>
       隐私保护
      </strong>
      ：使用私人模式浏览以避免保存浏览记录。
     </li>
    </ul>
    <h3>
     4. 典型生态项目
    </h3>
    <ul>
     <li>
      <strong>
       Kotlin &amp; KotlinX
      </strong>
      : EhViewer 使用 Kotlin 作为主要编程语言，并利用 KotlinX 库提高开发效率。
     </li>
     <li>
      <strong>
       Arrow
      </strong>
      : 提供实用的函数式编程工具集，用于简化代码结构。
     </li>
     <li>
      <strong>
       Material Icons
      </strong>
      : 应用内使用的图标来源于 Google 的 Material Icons 设计。
     </li>
     <li>
      <strong>
       Ktor
      </strong>
      : 用来构建服务器端通信部分，处理网络请求。
     </li>
     <li>
      <strong>
       Coil
      </strong>
      : 图片加载库，负责高效地显示网络上的图片。
     </li>
     <li>
      <strong>
       Compose Destinations
      </strong>
      : 用于导航管理的 Jetpack Compose 插件。
     </li>
    </ul>
    <p>
     了解更多详情，您可以直接查看 EhViewer 的

     中的源码和文档。如果您遇到任何问题，建议参考 GitHub 上的 Issue 版块或加入官方社区寻找帮助。
    </p>
