    <h3>
     <a id="1__Python2_1">
     </a>
     1. 安装 Python2
    </h3>
    <p>
     可以到
     <a href="https://www.python.org/" rel="nofollow">
      Python 官网
     </a>
     去下载安装包进行安装，也可以直接使用如下命令安装 Python2：
    </p>
    <pre><code>$ sudo apt install python2
</code></pre>
    <p>
     安装完成后检查 Python 的版本，在这里我安装的 Python版本为 2.7.18：
    </p>
    <pre><code>$ python2 -V
Python 2.7.18
</code></pre>
    <p>
     一般 Ubuntu 是自带 python3 版本的：
    </p>
    <pre><code>$ python3 -V
Python 3.8.5
</code></pre>
    <p>
     安装完成后我们可以使用如下命令来检查目前可用的 Python 版本：
    </p>
    <pre><code>$ ls /usr/bin/python*
</code></pre>
    <p>
     <img alt="python-v" src="https://i-blog.csdnimg.cn/blog_migrate/5ead1020af56e6a3b505f038847aac59.png#pic_center">
      <br/>
      可以看到，我可用的 Python 版本有如上几个…
     </img>
    </p>
    <h3>
     <a id="2__27">
     </a>
     2. 设置默认方式(替代版本)
    </h3>
    <p>
     首先查看是否已经配置了 Python 的默认方式(替代版本)：
    </p>
    <pre><code>$ sudo update-alternatives --list python
</code></pre>
    <p>
     若没有设置，会显示：
    </p>
    <pre><code>update-alternatives: error: no alternatives for python
</code></pre>
    <p>
     若设置了，则是显示你替代的版本，可以以此确认你的备选方案是否可用：
    </p>
    <pre><code>$ sudo update-alternatives --list python
/usr/bin/python2
/usr/bin/python3
</code></pre>
    <p>
     然后使用如下命令设置默认方式(替代版本)：
    </p>
    <pre><code>$ sudo update-alternatives --install /usr/bin/python python /usr/bin/python2 1
$ sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 2
</code></pre>
    <p>
     选用可选的 Python 版本：
    </p>
    <pre><code>$ sudo update-alternatives --config python
There are 2 choices for the alternative python (providing /usr/bin/python).
  Selection    Path              Priority   Status
------------------------------------------------------------
* 0            /usr/bin/python3   2         auto mode
  1            /usr/bin/python2   1         manual mode
  2            /usr/bin/python3   2         manual mode
Press  to keep the current choice[*], or type selection number: 1
</code></pre>
    <p>
     在本例中，选择 1 来选择 Python2…
    </p>
    <p>
     最后，你可以检查你的 Python 版本来确认是否设置成功：
    </p>
    <pre><code>$ python -V
Python 2.7.18
</code></pre>
    <p>
     安装 Python 之后，肯定需要安装 pip，关于在 Ubuntu 下如何安装 pip 可以参考我的这篇博客：
     <a href="https://blog.csdn.net/My_CSDN_IT/article/details/113892209">
      Ubuntu 下安装 pip
     </a>
    </p>
