     </span>
    </p>
    <h4 style="background:#F8F8F8">
     <span style="color:#222222">
      <span style="font-size:18px">
       目前公司股权（股份）的法律现状？
      </span>
     </span>
    </h4>
    <p style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       国内公司分为两个大类：股份公司和有限公司。股份公司对应股份，有限公司对应股权。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="font-size:18px">
      <span style="color:#222222">
       股份公司有分为
      </span>
      <span style="color:#222222">
       “
      </span>
      <span style="color:#222222">
       上市公司
      </span>
      <span style="color:#222222">
       ”
      </span>
      <span style="color:#222222">
       和
      </span>
      <span style="color:#222222">
       “
      </span>
      <span style="color:#222222">
       非上市股份公司
      </span>
      <span style="color:#222222">
       ”
      </span>
      <span style="color:#222222">
       。上市公司股份是登记在中国证券登记结算公司；非上市公司的股份，按公司法规定，以股票为权利凭证，并配以股东名册来记录持股人。值得注意的是，非上市股份公司已发行的股份，不需要第三方机构登记，安全靠公司自身发行股票和维持股东名册来确定股权。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="font-size:18px">
      <span style="color:#222222">
       对于有限公司，因不存在股票这种凭证，股权证明就成了公司自行维护的
      </span>
      <span style="color:#222222">
       “
      </span>
      <span style="color:#222222">
       股东名册
      </span>
      <span style="color:#222222">
       ”
      </span>
      <span style="color:#222222">
       。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       综上所述，《公司法》对无论哪种公司的股权（股份）都给予了较大的自主权：让公司自行提供股权（股份）的最终证明。
      </span>
     </span>
    </p>
    <p align="left">
     <span style="color:#222222; background:#F8F8F8">
      <span style="font-size:18px">
      </span>
     </span>
    </p>
    <h4 style="background:#F8F8F8">
     <span style="color:#222222">
      <span style="font-size:18px">
       区块链股权（股份）法律效力如何？
      </span>
     </span>
    </h4>
    <p style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       《公司法》允许公司采取任何成文记录的形式来创建并维护股东名册，不局限于特定载体。区块链作为一种公开账本，不仅完全符合公司法规定，而且更是未来的趋势。做个比较，早年的股票是纸质的，且需要人工手动记账式交易，后来就电子化了；货币从纸钞到电子化也是这个进程。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       中国公司股权登记也越来越倾向于发达国家的成熟经验，股权不在国家机关登记，而是由商会、工会等民间自治组织来登记，以市场认可的第三方来提供关于股权（股份）的权利证明。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       所以，如果股权通过区块链来登记，就把可信的第三方彻底去中心化了，即不再局限于特定的第三方，靠全体参与者共同来维护一套登记系统。这套系统有四个特点：一、把登记权利交给了公司，体现了自治精神；二、确保了股权登记的公开透明、真实可信，减小争议；三、促进股权流通和资源共享，股权登记和转让更便捷安全；四、及时，数据永久安全保存。
      </span>
     </span>
    </p>
    <p align="left">
     <strong>
      <span style="color:#1F497D">
       <span style="font-size:18px">
       </span>
      </span>
     </strong>
    </p>
    <h4 style="background:#F8F8F8">
     <span style="color:#222222">
      <span style="font-size:18px">
       股权登记管理
      </span>
     </span>
    </h4>
    <p style="background:rgb(248,248,248)">
     <span style="font-size:18px">
      <span style="color:#222222">
       股权登记发挥着向社会公示作用，让潜在交易主体了解权属状态；登记也是交易的关键环节，记录所有权的转移。现在非上市公司股权往往需要人工处理，股东名册维护很繁琐，历史交易维护和跟踪也变得困难。
      </span>
      <strong>
       <span style="color:#1F497D">
        区块链股权登记完全数字化，有着安全透明、不可篡改、易于跟踪的特点，有很大的优势。
       </span>
      </strong>
     </span>
    </p>
    <h4 style="background:rgb(248,248,248)">
     <span style="color:#222222">
      <span style="font-size:18px">
       股权转让流通
      </span>
     </span>
    </h4>
    <p style="background:rgb(248,248,248)">
     <span style="font-size:18px">
      <span style="color:#222222">
       股权转让流通是个重要环节，传统
      </span>
      <span style="color:#222222">
       OTC
      </span>
      <span style="color:#222222">
       场外股权交易，以交易双方信用为基础，需要建立双边授信后才可进行交易，信用风险由交易双方自行承担，而交易平台集中承担市场交易参与者的信用风险。
      </span>
     </span>
    </p>
    <p style="background:rgb(248,248,248)">
     <span style="font-size:18px">
      <strong>
       <span style="color:#1F497D">
        区块链可以有效降低交易的信用风险。股权所有权登记在区块链中，股权交易必须要所有者私钥签名才能验证通过，交易确认后，股权变更也会记录在区块链中，保障了交易双方的利益
       </span>
      </strong>
