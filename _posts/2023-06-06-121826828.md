    <h3>
     <a id="HADOOP_32">
     </a>
     HADOOP概述
    </h3>
    <p>
     1.Hadoop是一个在计算机硬件的集群上储存数据，运行应用程序的开源软件框架
     <br/>
     2.解决大数据运算的框架方案
     <br/>
     3.大规模的存储/计算
     <br/>
     4.对于大型job处理速度非常快
     <br/>
     5.多种多样的处理引擎
    </p>
    <h3>
     <a id="Hadoop_38">

     <br/>

     Hive
    </h3>
    <p>
     基于Hadoop的一个数据仓库工具
    </p>
    <p>
     操作接口采用类SQL语法，学习成本低
     <br/>
     避免了去写MapReduce，开发效率高
     <br/>
     适用于海量结构化数据离线分析
     <br/>
     Web访问日志分析
     <br/>
     可运用于构建离线数据仓库
