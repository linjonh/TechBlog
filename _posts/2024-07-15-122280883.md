    <p>
     python append描述
     <br/>
     append函数可以在列表的末尾添加新的对象。函数无返回值，但是会修改列表。
    </p>
    <h3>
     <a id="append_3">
     </a>
     append语法
    </h3>
    <p>
     list.append(object)
     <br/>
     名称 说明 备注
     <br/>
     list 待添加元素的列表
     <br/>
     object 将要给列表中添加的对象 不可省略的参数
     <br/>
     <a href="https://www.howtouselinux.com/post/append-list-in-python" rel="nofollow">
      3 examples to append list in python
     </a>
    </p>
    <h3>
     <a id="append_10">
     </a>
     append举例
    </h3>
    <h3>
     <a id="1__12">
     </a>
     1. 给列表中添加整数、浮点数和字符串：
    </h3>
    <p>
     test = [‘Python’, ‘C’, ‘Java’]
    </p>
    <p>
     test.append(5)
     <br/>
     test.append(23.6)
     <br/>
     test.append(‘HTML’)
    </p>
    <p>
     print(test)
     <br/>
     输出结果为：
    </p>
    <p>
     [‘Python’, ‘C’, ‘Java’, 5, 23.6, ‘HTML’]
    </p>
    <h3>
     <a id="2__25">
     </a>
     2. 给列表中添加列表、元组和字典：
    </h3>
    <p>
     test = [‘Python’, ‘C’, ‘Java’]
    </p>
    <p>
     test.append([‘Windows’, 2018, ‘OpenStack’])
     <br/>
     test.append((‘Huawei’, ‘Tencent’))
     <br/>
     test.append({‘Nova’:‘virtual compute service’, ‘Neutron’:‘net service’})
    </p>
    <p>
     print(test)
     <br/>
     输出结果为：
    </p>
    <p>
     [‘Python’, ‘C’, ‘Java’, [‘Windows’, 2018, ‘OpenStack’], (‘Huawei’, ‘Tencent’), {‘Nova’: ‘virtual compute service’, ‘Neutron’: ‘net service’}]
    </p>
    <h3>
     <a id="3__38">
     </a>
     3. 给列表中添加空元素
    </h3>
    <p>
     test = [‘Python’, ‘C’, ‘Java’]
    </p>
    <p>
     test.append(None)
    </p>
    <p>
     print(test)
     <br/>
     输出结果为：
    </p>
    <p>
     [‘Python’, ‘C’, ‘Java’, None]
     <br/>
     注意事项
     <br/>
     object参数不能省略，否则Python会报错：
    </p>
    <p>
     test = [‘Python’, ‘C’, ‘Java’]
    </p>
    <p>
     test.append()
    </p>
    <p>
     print(test)
     <br/>
     Traceback (most recent call last):
     <br/>
     File “/Users/untitled3/Test2.py”, line 3, in
     <br/>
     test.append()
     <br/>
     TypeError: append() takes exactly one argument (0 given)
     <br/>
     如果想给列表末尾添加空元素，应该将参数写为None
    </p>
