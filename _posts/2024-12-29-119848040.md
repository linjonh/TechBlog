     <h2 id="rtvue-lowcode-简介">
      rtvue-lowcode 简介
     </h2>
     <p>

     <h2 id="项目运行安装">
      项目运行安装
     </h2>
     <p>
      克隆后直接安装即可
     </p>
     <pre class="has"><code>yarn install &amp;&amp; yarn serve</code></pre>
     <h2 id="自定义组件开发步骤">
      自定义组件开发步骤
     </h2>
     <p>
      准备自定义组件开发之前，需要将rtvue-lowcode项目克隆到本地。
      <br/>
      组件开发分为3个步骤：
     </p>
     <ol>
      <li>
       定义组件
      </li>
      <li>
       可拖拽组件
      </li>
      <li>
       配置组件
      </li>
     </ol>
     <h3 id="定义组件">
      定义组件
     </h3>
     <h4 id="创建组件文件">
      创建组件文件
     </h4>
     <p>
      首先在src/components 目录下定义组件，目录名和文件要严格保持一致，目录命名格式为
      <code>
       r-组件名称
      </code>
      ，文件命名格式为
      <code>
       r-组件名称.vue
      </code>
      。
      <br/>
      例如：文件目录名为
      <code>
       r-form-input
      </code>
      <br/>
      则文件名为：
      <code>
       r-form-input.vue
      </code>
     </p>
     <h4 id="混入组件代码">
      混入组件代码
     </h4>
     <p>
      创建组件后，需要将
      <code>
       rvuecomp
      </code>
      混入到组件内部这里。代码举例如下：
      <code>
       r-form-input
      </code>
      所示
     </p>
     <pre class="has"><code class="language-html">&lt;script&gt;
    import {rvuecomp} from '../mixins/r-vue-comp' //&lt;----在这里将组件混入进去
    export default {
        name: "r-form-input",
        mixins:[rvuecomp] //&lt;----在这里将组件混入进去
    };
&lt;/script&gt;</code></pre>
     <p>
      其中在
      <code>
       r-vue-comp
      </code>
      中主要定义了组件需要的参数类型，目前计划有两种参数类型：
      <code>
       option
      </code>
      和
      <code>
       compStyle
      </code>
      。
     </p>
     <p>
      其中
      <code>
       option
      </code>
      代表了组件的配置信息，例如一个输入框的左标签，默认数值等等。
     </p>
     <p>
      <code>
       compStyle
      </code>
      代表了控件容器的CSS。可以在控件的根
      <code>
       view
      </code>
      标签中，使用
      <code>
       :style='compStyle'
      </code>
      方法将
      <code>
       style
      </code>
      和
      <code>
       compStyle
      </code>
      绑定到一起。
     </p>
     <pre class="has"><code class="language-javascript">export const rvuecomp = {
    props: {
        option: {
            type: Object,
            require: true,
        },
        compStyle: {
            type: Object,
            require: false,
        },
    },
}</code></pre>
     <p>
      完成以上操作，即可定义了一个新的控件。随后我们的任务是将默认的数据放入
      <code>
       components/mixins/default_data.js
      </code>
      中，例如
      <code>
       r-form-input
      </code>
      中，默认数据如下所示：
     </p>
     <pre class="has"><code class="language-javascript">const rFormInput = {
  type: "r-form-input",
  option: {
    value: "",
    label: "普通输入框",
    placeholder: "请输入内容",
    btn:{
      getCode:()=&gt;{

      },
      codeText:'单击'
    }
  },
  compStyle: {
    height: "auto",
    width: "100%",
    "font-size": "24rpx",
    "background-color": "#fff",
    "margin-top": "0",
    "margin-right": "0",
    "margin-down": "0",
    "margin-left": "0",
  }
}</code></pre>
     <p>
      完成以上操作后就定义了一个组件。
     </p>
     <h3 id="可拖拽组件">
      可拖拽组件
     </h3>
     <h4 id="定义标签名称">
      定义标签名称
     </h4>
     <p>
      首先要在
      <code>
       src/pages/labels
      </code>
      中将
      <code>
       r-组件名称
      </code>
      放入
      <code>
       tags
      </code>
      数组中。
      <br/>
      例如在
      <code>
       r-form-input
      </code>
      中,标签可命名为
     </p>
     <pre class="has"><code class="language-javascript">{ text: "普通输入框", type: "r-form-input" },</code></pre>
     <h4 id="拖拽后加载默认数据">
      拖拽后加载默认数据
     </h4>
     <p>
      在
      <code>
       src/components/r-vue-edit
      </code>
      中在
      <code>
       methods:{addComp}
      </code>
      方法中加入
      <code>
       case:'组件类型'
      </code>
      选择器，并将默认数据加入到
      <code>
       options
      </code>
      里面，例如：
     </p>
     <pre class="has"><code class="language-javascript">case "r-form-input":
    this.options.splice(index, 0, this.getData(this.rFormInput, this));
    break;</code></pre>
     <p>
      完成以上操作后即可在屏幕上即可生成一个可拖拽组件，这个组件可以向容器中拖入。
     </p>
     <h3 id="配置组件">
      配置组件
     </h3>
     <p>
      在
      <code>
       pages/sidebar/option-bar/
      </code>
      中需要对每个组件的
      <code>
       option
      </code>
      做配置页面。
     </p>
     <p>
      命名格式为：
      <code>
       option-r-组件名称-bar.vue
      </code>
      <br/>
      例如:
      <code>
       option-r-form-input-bar.vue
      </code>
      即可自动将组件和配置页面绑定在一起。
     </p>
