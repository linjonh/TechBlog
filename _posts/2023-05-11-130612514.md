    <h2>
     <a id="1Pycharm_2">
     </a>
     1、Pycharm版本
    </h2>
    <p>
     连接远程服务器需要使用Pycharm专业版，免费的社区版本无法连接。
     <br/>
     <a href="https://www.jetbrains.com/pycharm/" rel="nofollow">
      Pycharm下载地址
     </a>
    </p>
    <h2>
     <a id="2Pycharm_7">
     </a>
     2、Pycharm设置
    </h2>
    <h3>
     <a id="1_8">
     </a>
     1、连接服务器，设置文件映射位置
    </h3>
    <p>
     首先菜单栏找到File—Settings—Deployment如下图，点击“+”创建SFTP
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7c803d0bc72606c19a54be512842369.png#pic_center">
      <br/>
      弹出的server name对话框可以填写要连接的服务器的名称，这里以0为例；
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/72102bd4b9e69aeb5e249610a6dd08b6.png#pic_center">
       <br/>
       选择设置SSH configuration：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7d9c156252fc72a2b976afa82f4e468.png#pic_center">
        <br/>
        填写对应服务器IP地址、用户名、密码，端口Port默认22，完成后点ok
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5c9b3a9dfb8a3ec96c6a0cd175f3c91.png#pic_center">
         <br/>
         再点击Mapping设置文件夹映射：
         <br/>
         Local path选择你的项目路径；Deployment path为服务器上对应同步的位置路径。
         <br/>
         设置完成后点击apply、ok即可。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d4e723d4a0a858057841e6e1adb76b45.png#pic_center"/>
        </img>
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="2Interpreter_21">
     </a>
     2、设置服务器上的解释器Interpreter
    </h3>
    <p>
     在File—Settings—Project找到Python Interpreter，点击add interpreter—on SSH
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1fae14aa6daa9f362d7e51c345e96553.png#pic_center">
      <br/>
      选择现有连接existing，点击下拉选项：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f20ec2efb534ea49082f39f584dba2b0.png#pic_center">
       <br/>
       应该会出现刚才已经配置好的连接，选择即可，点击next，等待一会再next，完成后设置选择服务器上的位置，选择System Interpreter：第一个Interpreter选择为你服务器上所使用的Python的位置；第二个Sync Folders点击右边文件浏览将选择Remote Path改为之前服务器上对应同步的位置路径。再点击ok、create即可。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dcf6def7acee1f8ee3a930ca44285f92.png#pic_center"/>
      </img>
     </img>
    </p>
    <h3>
     <a id="3_28">
     </a>
     3、设置文件自动同步控制
    </h3>
    <p>
     等待服务器进行文件同步就基本完成了Pycharm连接远程服务器的环境配置了。
     <br/>
     一般还会在Tools—Deployment—Options中选择每次修改同步服务器的选项，可以设置为每次保存后同步（ctrl+S）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/92bb9ed0a2393d4bcd16b93580f884ee.png#pic_center"/>
    </p>
