    <h3>
     <a id="1_15">
     </a>
     1.使用场景
    </h3>
    <p>
     在以下情况下，可能需要向 YAML 文件中写入数据：
    </p>
    <ol>
     <li>
      <p>
       配置文件更新：YAML 文件常用于存储应用程序的配置信息。当您需要更新应用程序的配置时，可以将新的配置数据写入 YAML 文件中，以便应用程序在下次启动时使用更新后的配置。
      </p>
     </li>
     <li>
      <p>
       数据持久化：如果您的应用程序需要将数据持久化到文件中，而且数据结构适合以键值对或结构化的方式进行表示，那么将数据写入 YAML 文件是一个不错的选择。您可以将数据转换为字典或列表的形式，并使用 YAML 格式将其写入文件。
      </p>
     </li>
     <li>
      <p>
       测试数据生成：在编写测试用例时，您可能需要生成一些测试数据。使用 YAML 格式可以方便地定义和管理测试数据。您可以编写一个包含各种测试场景和数据的 YAML 文件，并在测试过程中读取这些数据。
      </p>
     </li>
     <li>
      <p>
       数据交换：YAML 是一种轻量级的数据交换格式，易于阅读和编辑。如果您需要将数据从一个系统传输到另一个系统，可以使用 YAML 格式进行数据的序列化和反序列化，以实现数据的传输和共享。
      </p>
     </li>
     <li>
      <p>
       配置文件生成：有时候，您可能需要根据特定规则或逻辑生成配置文件。使用编程语言中的数据结构创建一个表示配置的对象，并将其转换为 YAML 格式写入文件，可以快速、灵活地生成配置文件。
      </p>
     </li>
    </ol>
    <p>
     当需要存储、传输或生成结构化数据时，将数据写入 YAML 文件是一种常见的选择。YAML 提供了一种简洁、易读的方式来表示数据，并且与许多编程语言都有良好的集成和支持。
    </p>
    <p>
     <img alt="image-20240404114214502" src="https://i-blog.csdnimg.cn/blog_migrate/956b36e04ef8f7561533cd022e4b7893.png#pic_center"/>
    </p>
    <h3>
     <a id="2_33">
     </a>
     2.使用方式
    </h3>
    <p>
     要在 YAML 文件中添加一个名为
     <code>
      agent_config
     </code>
     的数组，每个数组元素都是一个对象，对象包含
     <code>
      id
     </code>
     、
     <code>
      name
     </code>
     和
     <code>
      age
     </code>
     三个字段，可以使用以下示例：
    </p>
    <pre><code class="prism language-yaml"><span class="token key atrule">agent_config</span><span class="token punctuation">:</span>
  <span class="token punctuation">-</span> <span class="token key atrule">id</span><span class="token punctuation">:</span> <span class="token number">1</span>
    <span class="token key atrule">name</span><span class="token punctuation">:</span> John
    <span class="token key atrule">age</span><span class="token punctuation">:</span> <span class="token number">25</span>
  <span class="token punctuation">-</span> <span class="token key atrule">id</span><span class="token punctuation">:</span> <span class="token number">2</span>
    <span class="token key atrule">name</span><span class="token punctuation">:</span> Jane
    <span class="token key atrule">age</span><span class="token punctuation">:</span> <span class="token number">30</span>
  <span class="token punctuation">-</span> <span class="token key atrule">id</span><span class="token punctuation">:</span> <span class="token number">3</span>
    <span class="token key atrule">name</span><span class="token punctuation">:</span> Alex
    <span class="token key atrule">age</span><span class="token punctuation">:</span> <span class="token number">28</span>
</code></pre>
    <p>
     在这个示例中，
     <code>
      agent_config
     </code>
     是数组的键名，后面的每个缩进的部分表示数组的元素。每个元素都是一个对象，使用键值对的形式表示，其中
     <code>
      id
     </code>
     、
     <code>
      name
     </code>
     和
     <code>
      age
     </code>
     是字段名，后面是对应的值。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9f5387a682502e4086d77c8efaed315f.png#pic_center"/>
    </p>
    <p>
     如果你想在 Python 代码中动态添加这样的数组数据并将其写入 YAML 文件，可以使用
     <code>
      yaml
     </code>
     库提供的方法。以下是一个示例：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> yaml

data <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">'agent_config'</span><span class="token punctuation">:</span> <span class="token punctuation">[</span>
        <span class="token punctuation">{<!-- --></span><span class="token string">'id'</span><span class="token punctuation">:</span> <span class="token number">1</span><span class="token punctuation">,</span> <span class="token string">'name'</span><span class="token punctuation">:</span> <span class="token string">'John'</span><span class="token punctuation">,</span> <span class="token string">'age'</span><span class="token punctuation">:</span> <span class="token number">25</span><span class="token punctuation">}</span><span class="token punctuation">,</span>
        <span class="token punctuation">{<!-- --></span><span class="token string">'id'</span><span class="token punctuation">:</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token string">'name'</span><span class="token punctuation">:</span> <span class="token string">'Jane'</span><span class="token punctuation">,</span> <span class="token string">'age'</span><span class="token punctuation">:</span> <span class="token number">30</span><span class="token punctuation">}</span><span class="token punctuation">,</span>
        <span class="token punctuation">{<!-- --></span><span class="token string">'id'</span><span class="token punctuation">:</span> <span class="token number">3</span><span class="token punctuation">,</span> <span class="token string">'name'</span><span class="token punctuation">:</span> <span class="token string">'Alex'</span><span class="token punctuation">,</span> <span class="token string">'age'</span><span class="token punctuation">:</span> <span class="token number">28</span><span class="token punctuation">}</span>
    <span class="token punctuation">]</span>
<span class="token punctuation">}</span>

filename <span class="token operator">=</span> <span class="token string">'config.yaml'</span>
<span class="token keyword">with</span> <span class="token builtin">open</span><span class="token punctuation">(</span>filename<span class="token punctuation">,</span> <span class="token string">'w'</span><span class="token punctuation">,</span> encoding<span class="token operator">=</span><span class="token string">'utf-8'</span><span class="token punctuation">)</span> <span class="token keyword">as</span> yaml_file<span class="token punctuation">:</span>
    yaml<span class="token punctuation">.</span>dump<span class="token punctuation">(</span>data<span class="token punctuation">,</span> yaml_file<span class="token punctuation">)</span>
</code></pre>
    <p>
     在这个示例中，我们创建了一个字典
     <code>
      data
     </code>
     ，其中包含一个键为
     <code>
      agent_config
     </code>
     的数组，数组的元素是包含
     <code>
      id
     </code>
     、
     <code>
      name
     </code>
     和
     <code>
      age
     </code>
     字段的字典对象。然后，我们将
     <code>
      data
     </code>
     字典写入到名为
     <code>
      config.yaml
     </code>
     的 YAML 文件中。
    </p>
