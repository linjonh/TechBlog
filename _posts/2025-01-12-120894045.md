    <h3>
     <a id="Alibaba_7">
     </a>
     Alibaba代码规约插件
    </h3>
    <p>
     这个插件对于我们国人来说非常的好友，配合《阿里开发手册.pdf》文档相当不错的，下面介绍IDEA工具下安装插件步骤：
    </p>
    <ol>
     <li>
      <strong>
       File–&gt;Settings…
      </strong>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ee4aa83f810a46656a9b027da08c6c21.png#pic_center"/>
     </li>
     <li>
      <strong>
       Plugins—Marketplace搜索alibaba，选择对应插件—Install
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/47e03b68af0f30c750704dfb872bc6ff.png#pic_center"/>
     </li>
     <li>
      <strong>
       插件安装完成重启idea即可
      </strong>
     </li>
    </ol>
    <h3>
     <a id="_15">
     </a>
     使用方式
    </h3>
    <p>
     Alibaba Java Coding Guidelines
     <br/>
     直译就是阿里巴巴Java编码指南。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5eb61cb55a8c769ffc4e83cf618fac31.png#pic_center"/>
    </p>
    <ol>
     <li>
      <strong>
       单文件审查。右键需要审查的java文件，执行“代码规约扫描”
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bd51daee9d5f831751d3b8c6c14d31e1.png#pic_center"/>
     </li>
     <li>
      <strong>
       工程审查。右键整个工程，执行“代码规约扫描”
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cc59d09520ab06657a0bcd5552d8a744.png#pic_center"/>
     </li>
     <li>
      <strong>
       扫描结束，通常有三种结果，blocker，critical，major
       <br/>
       根据扫描结果及项目实际进行修改。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/43bd055952747c602573f8b786313b64.png#pic_center"/>
     </li>
     <li>
      <strong>
       展开到最里层，双击打开对应代码块地方，进行查看和修改。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b14530d303b51c69eaeda08e6670ad1d.png#pic_center"/>
     </li>
    </ol>
    <h3>
     <a id="SonarLint_30">
     </a>
     SonarLint
    </h3>
    <p>
     这个插件是英文版的哈，英文不好的也不用慌，里面的内容都是文本可以拷贝出来进行翻译的。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dd2d62cfca81c1c488a9f54d2436dba5.png#pic_center"/>
    </p>
    <h3>
     <a id="_32">
     </a>
     使用方式
    </h3>
    <ol>
     <li>
      <strong>
       插件安装方式同上
      </strong>
     </li>
     <li>
      <strong>
       和阿里巴巴Java编码指南插件一样可单文件和整个工程扫描。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/192a2a2098255473f35b355caecf0eed.png#pic_center"/>
     </li>
     <li>
      <strong>
       双击扫描处理的结果，定位到代码块。右侧有相应提示。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/be87e466cf56e91fb024062dd1362bab.png#pic_center"/>
     </li>
     <li>
      <strong>
       SonarLint的规则是可以禁用及自己定义的。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8f46c453a418de9b4526df053a4b49a0.png#pic_center"/>
     </li>
    </ol>
    <p>
     SonarLint功能还很多，有兴趣的同学自行去研究哈。此文章仅展示如何安装该插件和使用方式。
    </p>
