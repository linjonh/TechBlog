    <h3>
     <a id="_2">
     </a>
     一.背景：
    </h3>
    <p>
     鸿蒙项目开发需要使用模拟器进行开发测试，但目前想在DevEco Studio开发工具中使用模拟器就必须到华为官网进行报名申请，参加“
     <strong>
      鸿蒙模拟器（HarmonyOS Emulator）Beta活动申请
     </strong>
     ”。
     <br/>
     申请审核通过后，才可在DevEco Studio的设备管理器界面下载到模拟器镜像，下载后方可使用手机（包含折叠屏）模拟器、平板模拟器。
     <br/>
     提交
     <strong>
      模拟器的报名申请表，一般会在1-2天审核通过。这个非常重要哦！
     </strong>
    </p>
    <h3>
     <a id="_7">
     </a>
     二.报名方式：
    </h3>
    <p>
     申请报名有2种方式，一种是直接在官网活动链接申请报名，另一种是从开发工具DevEco Studio中，直接跳转至官网活动报名链接处，进行申请。
     <br/>
     <strong>
      在申请报名前时，需要进行注册并实名认证华为开发者账号
     </strong>
     ，可以点击链接，参考
     <a href="https://blog.csdn.net/sun_promise/article/details/134389955?spm=1001.2014.3001.5501">
      《注册并实名认证华为开发者账号》
     </a>
     。
    </p>
    <h3>
     <a id="_12">
     </a>
     三.申请步骤：
    </h3>
    <p>
     1.打开开发工具
     <strong>
      DevEco Studio
     </strong>
     编译器，然后找到“
     <strong>
      设备管理器
     </strong>
     ”，若没有申请报名过鸿蒙模拟器，会出现提示框，然后点击蓝色文字部分的链接，跳转至
     <strong>
      官网报名
     </strong>
     ，需要先注册并实名认证华为开发者账号，然后直接登录账号，进行报名。
     <br/>
     <img alt="1.鸿蒙模拟器.png" src="https://i-blog.csdnimg.cn/blog_migrate/d373e5279670d6a5451c78461fecd749.png">
      <br/>
      <img alt="2.提示.png" src="https://i-blog.csdnimg.cn/blog_migrate/d529c28fbd8ebaa61b08d4c63b5a6af6.png">
       <br/>
       <strong>
        2.在官网登录账号后，会出现活动报名申请。按实际情况，填写申请表，提交即可。
       </strong>
       <br/>
       <img alt="1.鸿蒙模拟器报名申请.png" src="https://i-blog.csdnimg.cn/blog_migrate/a7ca8b8cb5955ad7590d22a80c6254a0.png"/>
      </img>
     </img>
    </p>
    <p>
     <img alt="2.申请.png" src="https://i-blog.csdnimg.cn/blog_migrate/439423269679206add550a53f8938ce1.png"/>
    </p>
    <p>

    <p>
     **3.**在用户类型选择上，不必纠结，根据实际情况选择即可，不同类型，出现的信息略有不同。选择填写企业开发者最简单，也可选择其他的用户类型。

      目前有5种可供选择用户类型：

     <br/>
