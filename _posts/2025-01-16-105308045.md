     python安装cv2
    </h2>
    <p>
     打开cmd，直接pip install opencv-python安装会报错显示超时，但使用清华源安装会快很多。如下：
     <br/>
     pip install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python
     <br/>
     整个过程30s即可装好
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b06d5729117cfaf6160568e1c4edd343.png">
      <br/>
      下面检验是否安装成功，cmd中输入python。在python中输入import cv2，如果没报错（下图所示情况即为安装成功）
