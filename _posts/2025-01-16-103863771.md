    <p>
     QT中C++调用python文件
     <br/>
     参考： Qt中用C++调用Python文件的三种方法
     <br/>
     https://blog.csdn.net/qq_41800188/article/details/79709452
    </p>
    <p>
     1、打开Qt creator，在上方的菜单栏中找到：工具–&gt;选项，点击进入
     <br/>
     2、环境–&gt;外部工具–&gt;Add–&gt;添加目录，目录名字可以自己取，这里选择RunPy3作为目录名，回车
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a21a247e3a4d34e85fd21fc2079661e7.png">
      <br/>
      3、Add–&gt;添加工具，在新添加的目录下添加工具，工具名选择python3.6
      <br/>
      配置外部工具：
      <br/>
      Description：可以随意书写；
      <br/>
      Executable：即完成执行python文件的外部工具，这里和安装的PyCharm所用的python编译器一致
      <br/>
      Arguments：%{CurrentDocument:FilePath}
      <br/>
      Working directory：%{CurrentDocument:Path}
      <br/>
      Environment：QT_LOGGING_TO_CONSOLE = 1
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/823ce1a09d90eb8fd1ed383ca77cf6e0.png">
       <br/>
       4、右击项目名称–&gt;Add New，选择添加Python File，名称根据个人需求（例如testpython）
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/67e40bd455e90615b816f8e0da2ba679.png">
        <br/>
        5、编辑testpython.py如下：
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b0326352ade255baa1995b7f377f2bb.png">
      <br/>
      6、测试运行：Tools–&gt;External–&gt;RunPy3–&gt;python3.6，在概要信息中输出结果。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ed4631cb5c550de553147888e3d71d49.png">
       <br/>
       7、输出结果正常，接下来开始在QT的程序中配置python3.6
       <br/>
       进入项目的.pro文件，手动添加外部库的包含目录和库目录
       <br/>
       具体内容类似于在QTcreater项目中添加opencv库的方法
       <br/>
       a、右键项目，点击添加库；
       <br/>
       b、选择外部库，单击下一步；
       <br/>
       c、选择库文件， 手动添加python3.6的.so所在目录
       <br/>
       手动添加python3.6的包含目录（自动配好的包含路径往往有误，需要手动添加目录，若.so不好使，可以尝试m.so，参考附图所示）；
       <br/>
       d、单击下一步，直到结束；
       <br/>
       e、进入.pro文件检查；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2c9a64de766294ddcf09c24e5dc5eb9d.png">
        <br/>
        8、编译并运行main函数，若输出结果和单独运行python程序时一致，那么QT中C++调用python文件的配置工作就完成了
