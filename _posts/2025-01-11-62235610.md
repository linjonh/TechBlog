    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px">
     <span style="color:rgb(153,0,48)">
      <strong>
       常用工具：
      </strong>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px">
     1、3D Ripper DX：DX模型截取器，原名3D Ripper
     <br/>
     2、GameAssassin：屏幕模型截取
     <br/>
     3、sculptris-alpha：导出OBJ
     <br/>
     4、Susie：支持游戏图片格式最多的看图工具。建议配合ViX、Linar等支持Susie插件的看图工具一起使用，提取图片十分快捷。
     <br/>
     5、Grapholic：有一定年头的看图软件，不过有部分老游戏的图片格式只有它支持（如 May-BeSOFT），有一定使用价值。支持Susie插件。
     <br/>
     6、MamiList：只支持Elf的游戏图片格式。支持Susie插件
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px">
     <br/>
     <span style="color:rgb(153,0,48)">
      <strong>
       解包类：
       <br/>
      </strong>
     </span>
     1、3DExploration：可查看的3D文件包括3ds，prj、asc，cob、scn、lwo,lw、DirectX(*.x)、dxf、mdl、md2、off、raw、obj等，平面图型有bmp、rle、dib、gif、jpg、png、tif、tga、pcx等
     <br/>
     2、QuickBMS：通用解包器制作工具，解包利器，可以去官网看下~
     <br/>
     3、UMDGen：PSP映像专用处理工具
     <br/>
     4、RPGViewer：图片资源提取工具（LZP压缩格式）（无双5）
     <br/>
     5、PSP_AFS_Tool：PSP解AFS包的工具
     <br/>
     6、AFSExplorer：解AFS
     <br/>
     7、ChinaZip：可解PAK等多种一般类压缩文件
     <br/>
     8、Pak浏览器：可解PAK（部分）
     <br/>
     9、dbpack：图片打包器
     <br/>
     10、RPGViewer3.0：支持多游戏（如风色幻想XX）还能提取PNG格式（也就是说能提取出背景透明的立绘）此软件需要安装VisualC++ 2008 SP1 Redistributable Package
     <br/>
     11、ExtractData
     <wbr/>
     AVG：游戏提取工具，解AVG类游戏文件
     <br/>
     12、crass
     <wbr/>
     <wbr/>
     GalGame：提取工具，支持打包
     <br/>
     13、Susie32：电脑绘图观看程式，由日本的竹村嘉人制作的自由软件。本程序主要是透过对应的插件，使CG得以变成BMP格式。另外，亦可透过插件，使图像得以存储成为JPEG、TIFF等各种格式。此软件也是解包利器
     <br/>
     agth
     <wbr/>
     <wbr/>
     游戏文本实时提取工具，用途多多，使用方法网上铺天盖地，就不赘述了
     <br/>
     14、Extractor
     <wbr/>
     007：ADAT,APAK,MHW,MIX,MW4,NPAK,PACK,PAK,PBO,PFF,PKR,POD,RES,U,VDF,VPP,WDIR,WTN,XCR,ZWP等数十种文件包的解包、封包程序
     <br/>
     15、HCG Converter：对以像dat这样的归档方式的系统，HCGConverter有不俗的效果。同时它也一些自己的特行，如bmp强制提取，异或解密，alpha反转。
     <br/>
     16、MamiList：
     <wbr/>
     著名GALGAME图片浏览解压工具，针对ELF公司的GALGAME制作
     <br/>
     17、KID资源提取器：可以提取KID公司旗下大多数游戏（MO again无法提取）
    </p>
