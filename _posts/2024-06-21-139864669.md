    <p>
     WinForms，作为微软.NET框架下的一个桌面应用程序开发工具，自1999年首次亮相以来，已经走过了二十多年的发展历程。它以其简单直观的拖拽式界面设计和丰富的控件库，成为了大众喜爱的入门学习编程工具。由于它是比较基础的开发工具，在UI上并没有做太多优化，所以设置出来的界面相对比较简单。不过通过第三方的UI库，瞬间可以让Winform界面变得丰富多彩，焕然一新。
    </p>
    <h3>
     Winform 开源UI库
    </h3>
    <h4>
     一、AntdUI
    </h4>
    <p>
     Winform UI library use Ant Design (
     <a href="https://gitee.com/antdui/AntdUI" rel="nofollow" title="https://gitee.com/antdui/AntdUI">
      https://gitee.com/antdui/AntdUI
     </a>
     )目前有50+基础控件，完成所有动效/阴影，原生无边框窗口（拥有原生窗口动画和特性），占用资源低，控件纯GDI绘制，
     <strong>
      可商用
     </strong>
     。有以下特点：
    </p>
    <ul>
     <li>
      纯GDI绘制，没有图片，支持AOT。
     </li>
     <li>
      高质量绘图，高性能动画。
     </li>
     <li>
      Winform上最阴影的阴影效果。
     </li>
     <li>
      无边框窗口，拥有原生窗口特性。
     </li>
     <li>
      3D翻转效果。
     </li>
     <li>
      主题配置。
     </li>
     <li>
      SVG矢量图。
     </li>
     <li>
      DPI适配
     </li>
    </ul>

    <h4>
     二、SunnyUI
    </h4>
    <p>
     http://SunnyUI.Net是基于 C# 和 .Net WinForm 的开源控件库、工具类库、扩展类库、多页面开发框架。
     <strong>
      个人使用是免费的，企业商业用途需要授权
     </strong>
     。特点：
    </p>
    <p>
     主题：包含16个主题；
    </p>
    <ul>
     <li>
      国际化：UI控件默认是中文，可以自定义其他语言；
     </li>
     <li>
      字体图标：自定义按钮图标；
     </li>
     <li>
      控件：每个控件都内置了常用属性，以便自定义；
     </li>
     <li>
      窗体：普通窗体、登录界面窗体；
     </li>
     <li>
      窗体布局：内置了7个模板，也可以自定义；同时支持跟随分辨率适应；
     </li>
     <li>
      其他：内置常用的工具类，比如文件、json、ini文件操作。
     </li>
    </ul>
