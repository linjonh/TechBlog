     【离散数学】计算机考研复试问答题总结
    </h2>
    <p>
     复试专业课需要考察离散数学，针对线上复试总结出的一些问答题目~
     <br/>
     （如果有笔试，相关类型的题目一定要好好做）
    </p>
    <h2>
     <a id="__4">
     </a>
     第一章 命题逻辑
    </h2>
    <p>
     Q1. 什么是永真式，永假式和可满足式？
     <br/>
     （1）若一个命题公式在
     <mark>
      所有赋值下取值均为真
     </mark>
     ，则称该命题公式为重言式或永真式
     <br/>
     （2）若一个命题公式在
     <mark>
      所有赋值下取值均为假
     </mark>
     ，则称该命题公式为矛盾式或永假式
     <br/>
     （3） 若一个命题公式
     <mark>
      至少存在一组成真赋值
     </mark>
     ，则称该命题公式为可满足式
    </p>
    <p>
     Q2. 怎样判断两命题公式等值？什么是等值演算？
     <br/>
     <mark>
      等值演算
     </mark>
     ：根据已知的等值式，推演出与给定公式等值的公式的过程。
     <br/>
     <mark>
      判断两命题公式等值
     </mark>
     ：
     <br/>
     （1）由定义判断命题公式A和B是否等值，应判断A&lt;-&gt;B是否为重言式，若A&lt;-&gt;B的真值表最后一列全为1，则A&lt;-&gt;B为重言式，因此A&lt;=&gt;B。
     <br/>
     （2）看A经过等值演算能否推演出B。
    </p>
    <p>
     Q3. 什么是极小项，极大项以及主析取范式和主合取范式？
     <br/>
     <mark>
      极小项
     </mark>
     ：若在简单合取式中每个命题变项与其否定有且仅有一个出现一次，则这样的简单合取式称为极小项。
     <br/>
     <mark>
      极大项
     </mark>
     ：若在简单析取式中每个命题变项与其否定有且仅有一个出现一次，则这样的简单析取式称为极大项。
     <br/>
     <mark>
      主析取范式
     </mark>
     ：如果一个命题公式的析取范式中的简单合取式全是极小项，则称该析取范式为该命题公式的主析取范式。
     <br/>
     <mark>
      主合取范式
     </mark>
     ：如果一个命题公式的合取范式中的简单析取式全是极大项，则称该合取范式为该命题公式的主合取范式。
    </p>
    <p>
     Q4. 怎样用一个命题公式的主析取范式求主合取范式？
     <br/>
     （1）求出该命题公式的主析取范式；
     <br/>
     （2）写出以主析取范式中没出现的极小项的角码为角码的极大项；
     <br/>
     （3）由这些极大项构成的合取式即为该公式的的主合取范式
    </p>
    <p>
     Q5.怎样判断一个推理是否正确？
     <br/>
     设A1,A2,…,Ak，B为结论。
     <br/>
     若(A1∧A2∧…∧Ak)-&gt;B为重言式，则称A1,A2,…,Ak推出结论B的推理正确。
    </p>
    <h2>
     <a id="__31">
     </a>
     第二章 谓词逻辑
    </h2>
    <p>
     Q6.什么是指导变元，什么是自由出现和约束出现？
     <br/>
     在合式公式∀xA和∃xA中，称x为指导变项，A为相应量词的辖域；
     <br/>
     x的所有出现为约束出现，A中不是约束出现的其他变项的出现为自由出现。
    </p>
    <p>
     Q7.换名规则是什么？
     <br/>
     将一个指导变项及其在辖域中
     <mark>
      所有约束出现替换成公式中没有出现的个体变项符号
     </mark>
     。
    </p>
    <p>
     Q8.什么是前束范式？
     <br/>
     设A为一谓词公式，如果A具有如下形式：Q1x1Q2x2…QkxkB
     <br/>
     称A为前束范式，其中每个Qi为∀或∃，B为不含量词的谓词公式。
     <br/>
     一个公式的前束范式各指导变项应该是不同的，原公式中自由出现的个体变项在前束范式中还应是自由出现的。
    </p>
    <h2>
     <a id="__44">
     </a>
     第三章 集合的基本概念和运算
    </h2>
    <p>
     Q9.什么是幂集？
     <br/>
     设A为集合，
     <mark>
      把A的全体子集构成的集合
     </mark>
     称作A的幂集，记作P(A)。
    </p>
    <p>
     Q10.证明集合恒等式的方法有哪些？
     <br/>
     （1）命题演算法（两个方向的包含）
     <br/>
     （2）恒等变形法
     <br/>
     （3）反证法
    </p>
    <h2>
     <a id="__53">
     </a>
     第四章 二元关系与函数
    </h2>
    <p>
     Q11. 什么是集合的笛卡尔积？
     <br/>
     设A,B为两个集合，用
     <mark>
      A中元素作为第一元素，B中元素为第二元素
     </mark>
     都成有序对。所有这样的有序对组成的集合为A和B的笛卡尔积。
    </p>
    <p>
     Q12.什么是二元关系？
     <br/>
     如果
     <mark>
      一个集合为空集或者它的元素都是有序对
     </mark>
     ，则称这个集合是一个二元关系，一般记作R。
    </p>
    <p>
     Q13. 如何表示二元关系？
     <br/>
     集合表达式，关系矩阵，关系图
    </p>
    <p>
     Q14. 介绍关系的性质。
     <br/>
     <img alt="二元关系的性质" src="https://i-blog.csdnimg.cn/blog_migrate/61548fae58f155c7dc652ee4c0432895.png#pic_center"/>
    </p>
    <p>
     Q15. 介绍关系的自反（对称，传递）闭包，以及如何求各闭包？
     <br/>
     （1）
     <mark>
      唯一的
     </mark>
     ，包含R的
     <mark>
      最小的
     </mark>
     自反（对称，传递）关系；
     <br/>
     （2）若R是自反（对称，传递）的，则是R本身；
     <br/>
     （3）若R不是自反（对称，传递）的，
     <mark>
      补上最少序偶
     </mark>
     使之自反（对称，传递）。
     <br/>
     用关系图求闭包：
     <br/>
     检查R的关系图，哪一个结点没有环就加上一个环，
     <mark>
      得到自反闭包
     </mark>
     ；
     <br/>
     如果将R的关系图中的单向边全部改成双向边，其他都不变，
     <mark>
      得到对称闭包
     </mark>
     ；
     <br/>
     依次检查R的关系图的每个结点x，把从x出发的长度不超过n的所有路径的终点找到，如果x到这样的终点没有边，就加上一条边，
     <mark>
      得到传递闭包
     </mark>
     。
    </p>
    <p>
     Q16. 什么是集合上的等价关系？并举一个例子。
     <br/>
     设R为非空集合A上的关系，如果R是
     <mark>
      自反的，对称的和传递的
     </mark>
     ，则称R为A上的等价关系。
     <br/>
     举例：无向图的连通关系是等价关系。
    </p>
    <p>
     Q17. 什么是等价类和商集?
     <br/>
     设R是非空集合A上的等价关系，则
     <mark>
      A上互相等价的元素构成的A的若干个子集
     </mark>
     称作等价类。
     <br/>
     等价类:
     <br/>
     （1）任何等价类都是集合A的非空子集；
     <br/>
     （2）在A中任取两个元素，他们的等价类或是相等或是不交
     <br/>
     （3）所有等价类的并集就是A
     <br/>
     商集：
     <mark>
      等价类的集合
     </mark>
     。
    </p>
    <p>
     Q18. 什么是集合上的偏序关系？并举一个例子。
     <br/>
     设R为非空集合A上的关系，如果R是
     <mark>
      自反的，反对称的和传递的
     </mark>
     ，则称R为A上的偏序关系。
     <br/>
     举例：实数集上的小于等于关系。
    </p>
    <p>
     Q19. 什么是偏序集，什么是全序集？
     <br/>
     偏序集：一个
     <mark>
      集合A和A上的偏序关系R
     </mark>
     一起被称作偏序集，记作&lt;A,R&gt;
     <br/>
     全序集：设&lt;A,≤&gt;为偏序集，若对A中的
     <mark>
      任意元素x,y，x和y都可比
     </mark>
     ，则称≤为A上的全序关系，且&lt;A,≤&gt;为全序集。
    </p>
    <p>
     Q20. 如何描述偏序集？
     <br/>
     对于有穷的偏序集&lt;A,≤&gt;可以用
     <mark>
      哈斯图
     </mark>
     来描述，实际上哈斯图是简化的关系图，在哈斯图中每个结点表示A中的一个元素，结点位置按他们在偏序中的次序从底向上排列。如果y盖住x，则在x和y之间连一条线。
    </p>
    <p>
     Q21. 介绍集合的最小元，最大元，极小元，极大元的概念。
     <br/>
     （1）若对任意x∈B, 都有b≼x, 则称b为B的最小元；
     <br/>
     （2）若对任意x∈B,都有x≼b, 则称b 为B的最大元；
     <br/>
     （3）若对任意x∈B, 满足 x≼ b ⇒ x=b, 则称b为B的极小元；
     <br/>
     （4）若对任意x∈B, 满足 b ≼ x ⇒ x=b, 则称b为B的极大元。
    </p>
    <p>
     Q22. 介绍集合的上界，下界，上确界，下确界的概念。
     <br/>
     （1）若对任意x∈B,满足x≼a, 则称 a 为B的上界；
     <br/>
     （2）若对任意x∈B,满足a≼x , 则称 a 为B的下界；
     <br/>
     （3）若元素a '∈A是B的上界， 元素a∈A是B的任何一个上界， 若均有 a’≼a，则称 a '为B的上确界；
     <br/>
     （4）若元素a ‘∈A是B的下界， 元素a∈A是B的任何一个下界， 若均有a≼a’，则称 a ’ 为B的下确界。
    </p>
    <p>
     Q23. 介绍下列函数的性质：满射，单射，双射。
     <br/>
     设函数f：A-&gt;B
     <br/>
     （1）若ran f=B，则称f是满射的；
     <br/>
     （2）若对于任何的x1,x2∈A，x1≠x2，都有f(x1)≠f(x2)，则称f是单射的；
     <br/>
     （3）若f既是满射的又是单射的，则称f是双射的。
    </p>
    <p>
     Q24. 什么是自然映射？
     <br/>
     设R是A上的一个等价关系，顶不过以一个从A到A/R的函数g: A-&gt;A/R，且g(a)=[a],
     <mark>
      把A中的元素a映射到a的等价类[a]
     </mark>
     。
    </p>
    <h2>
     <a id="__120">
     </a>
     第五章 图
    </h2>
    <p>
     Q25. 简述并证明握手定理。
     <br/>
     <mark>
      握手定理
     </mark>
     ：任何图中所有顶点的度数之和等于边数的2倍。
     <br/>
     <mark>
      证明
     </mark>
     ：每一条边都有2个端点，所有顶点的度数之和等于它们作为端点的次数之和，因此恰好等于边数的2倍。
    </p>
    <p>
     Q26. 什么是图的连通分支？
     <br/>
     设G为一个无向图，R是G中顶点的连通关系，按照R可将图中顶点分成若干个等价类，由它们导出的导出子图称为G的连通分支，G的连通分支的个数记为p(G)。
    </p>
    <p>
     Q27. 什么是割点，什么是桥？
     <br/>
     （1）设无向图G&lt;V,E&gt;，V’⊂V，若p(G-V’)&gt;p(G)，且对V’的任何真子集V’‘，p(G-V‘’)=p(G)，则称V’为G的
     <mark>
      点割集
     </mark>
     。若点割集中只有一个顶点v，则称v为
     <mark>
      割点
     </mark>
     。
     <br/>
     (非专业理解：点割集是这样的集合，将图中一个点割集中的所有顶点删去，剩下的连通分支数大于原图中的连通分支数；若删去一个点割集的真子集中的顶点，剩下的连通分支数等于原图中连通分支数。若点割集中只有一个顶点，这个顶点就是割点。)
     <br/>
     （2）无向图G&lt;V,E&gt;，E’⊂E，若p(G-E’)&gt;p(G)，且对E’的任何真子集E’‘，p(G-E‘’)=p(G)，则称E’为G的
     <mark>
      边割集
     </mark>
     。若边割集中只有一条边e，则称e为
     <mark>
      桥
     </mark>
     。
     <br/>
     (非专业理解：边割集是这样的集合，将图中一个边割集中的所有边删去，剩下的连通分支数大于原图中的连通分支数；若删去一个边割集的真子集中的边，剩下的连通分支数等于原图中连通分支数。若边割集中只有一条边，这条边就是桥。)
    </p>
    <p>
     Q28. 介绍图的几种矩阵表示。
     <br/>
     关联矩阵（点和边的关系），邻接矩阵（点和点），可达矩阵
    </p>
    <h2>
     <a id="__137">
     </a>
     第六章 特殊的图
    </h2>
    <p>
     Q29. 什么是二部图？
     <br/>
     若能将无向图G=&lt;V,E&gt;的顶点集V划分成两个不相交的非空子集V1,V2，使得G中
     <mark>
      任何一条边的两个端点一个属于V1，另一个属于V2
     </mark>
     ，则成G为二部图。
    </p>
    <p>
     Q30. 怎样判断一个无向图是否是二部图？
     <br/>
     一个无向图G是二部图当且仅当G中
     <mark>
      没有长度为奇数的回路
     </mark>
     。
    </p>
    <p>
     Q31. 什么是欧拉图？
     <br/>
     经过图中每条边一次且仅一次并且行遍图中每个顶点的回路（通路），成为欧拉回路（通路），
     <mark>
      存在欧拉回路的图
     </mark>
     ，称为欧拉图。
    </p>
    <p>
     Q32. 怎样判断无向图和有向图是否存在欧拉回路（通路）。
     <br/>
     （1）无向图G有欧拉回路当且仅当G是
     <mark>
      连通图且无奇度顶点
     </mark>
     ；有欧拉通路但无欧拉回路，当且仅当
     <mark>
      G是连通图且恰好有两个奇度顶点
     </mark>
     ，这两个顶点是欧拉通路的端点。
     <br/>
     （2）有向图D有欧拉通路当且仅当D是
     <mark>
      连通的且每个顶点的入度等于出度
     </mark>
     ；有欧拉通路但无欧拉回路，当且仅当D是
     <mark>
      连通的，且除了这两个顶点外，其余顶点的入度均等于出度
     </mark>
     。这两个顶点，一个顶点的入度比出度大1，作为终点，另一个出度比入度大1，为始点。
    </p>
    <p>
     Q33. 什么是哈密顿图？
     <br/>
     经过图中每个顶点一次且仅一次的回路（通路）称为哈密顿回路（通路），
     <mark>
      存在哈密顿回路的图
     </mark>
     称为哈密顿图。
    </p>
    <p>
     Q34. 什么是平面图，什么是平面嵌入？
     <br/>
     如果图G能画在平面上使得
     <mark>
      除顶点处外没有边交叉
     </mark>
     ，则称G为平面图。画出的这种不出现边交叉的图称为G的一个平面嵌入。
    </p>
    <p>
     Q35. 介绍两个图同构和同胚。
     <br/>
     两个图同构的必要条件：顶点数相同，边数相同，度数序列相同
     <br/>
     同胚：如果两个图同构或者经过反复插入或消去2度顶点后同构，则称两个图同胚。
    </p>
    <p>
     Q36. 介绍基本割集系统和基本回路系统。
     <br/>
     （1）设T是连通图G=&lt;V,E&gt;的一颗生成树，对每一条弦e，存在
     <mark>
      唯一的由弦e和T的树枝
     </mark>
     构成的初级回路C，称其为对应于弦e的基本回路。所有基本回路的集合为对应生成树T的基本回路系统。（只有一条弦）
     <br/>
     （2）设T是连通图G=&lt;V,E&gt;的一颗生成树，对T的每一条树枝a，存在
     <mark>
      唯一的由树枝a，其余的边都是弦
     </mark>
     的割集，称这样的割集为对应树枝a的基本割集。所有基本割集的集合为对应生成树T的基本割集系统。（只有一条树枝）
    </p>
    <h2>
     <a id="__165">
     </a>
     第七章 代数系统
    </h2>
    <p>
     Q37. 如何验证一个运算是否为集合S上的二元运算？
     <br/>
     首先要保证
     <mark>
      参加运算的可以是S中的任意两个元素，而运算的结果也是S中的一个元素
     </mark>
     ，即S对该运算是封闭的。
    </p>

     Q41. 什么是子群和生成子群，怎样判断G的子集H能否构成G的子群？
     <br/>
     子群：设群&lt;G,
     <em>
      &gt;，H是G的非空子集。
      <mark>
       如果H关于G中的运算
      </mark>
     </em>
     构成群，称H为G的子群。
     <br/>
     设G为群，H是G的非空子集，如果对
     <mark>
      任意x,y∈H，都有xy^(-1)∈H
     </mark>
     ，则称H是G的子群。
