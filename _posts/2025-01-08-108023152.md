    <h3>
     <a id="13__28">
     </a>
     1.3 选择自己需要的配置：
    </h3>
    <p>
     <strong>
      Create Desktop Shortcut
     </strong>
     ：创建桌面快捷方式图标，根据需要勾选。
     <br/>
     <strong>
      Update context menu
     </strong>
     ：将从文件夹打开项目添加至鼠标右键，根据需要勾选。
     <br/>
     <strong>
      Create Associations
     </strong>
     ：关联文件格式，不勾选的话可以使用如 np++等文本编辑器打开，根据需要勾选。
     <br/>
     <strong>
      Download and install 32-bit JetBrains Runtime
     </strong>
     ：下载并安装 JetBrains 的 JRE，如果已经安装了JRE，则无需勾选此项。
     <br/>
     <strong>
      Update PATH variable (restart needed)
     </strong>
     ：将 IDEA 启动目录添加到环境变量中，勾选后可以从命令行中启动 IDEA，根据需要勾选。

    <h2>
     <a id="2_JAVA_51">
     </a>
     2. 配置JAVA运行环境
    </h2>
    <blockquote>
     <p>
      <strong>
       系统环境
      </strong>
      ：win10
     </p>
    </blockquote>
    <h3>
     <a id="21_JAVAJDK_54">
     </a>
     2.1 下载安装JAVA的JDK：
    </h3>
    <ul>
     <li>
      <a href="https://www.oracle.com/java/technologies/javase-downloads.html" rel="nofollow">
       JAVA-JDK下载地址
      </a>
     </li>
     <li>
      <a href="https://pan.baidu.com/s/1o6a6SkLNDyX0hKUAShsDVg" rel="nofollow">
       百度云JDK
      </a>
      提取码：gpcf
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       1. JDK、JRE、JVM三者之间的关系？
      </strong>
      <br/>
      JDK: Java开发工具箱。
      <br/>
      JRE: java运行环境。
      <br/>
      JVM: java虚拟机。
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       JDK包括JRE，JRE包括JVM。
       <br/>
       JVM是不能独立安装的。
       <br/>
       JRE和JDK都是可以独立安装的。
      </strong>
      <br/>
      有单独的JDK安装包。
      <br/>
      也有单独的JRE安装包。
      <br/>
      没有单独的JVM安装包。
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       安装JDK的时候
      </strong>
      ：JRE就自动安装了，同时JRE内部的JVM也就自动安装了。
      <br/>
      <strong>
       安装JRE的时候
      </strong>
      ：JVM也就自动安装了。
     </p>
    </blockquote>

     <a id="22__72">

    <p>
     <strong>
      此电脑——右键——属性——高级系统设置——点击——弹出系统属性窗口——环境变量
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/18493eb43074f229e7a8d0024ed65a65.png#pic_center"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a37b294f29a2b03e59b6d750331048c.png#pic_center"/>
    </p>
    <h3>
     <a id="23_JAVA_HOME_76">
     </a>
     2.3 配置第一个环境变量：JAVA_HOME
    </h3>
    <p>
     第一个是
     <font color="#dd0000">
      JAVA_HOME
     </font>
     变量，点击下面的【新建】，【变量名】为
     <font color="#dd0000">
      JAVA_HOME
     </font>
     ；【变量值】为JDK路径，如图，把JDK的路径复制到对话框中，本人的是
     <font color="#dd0000">
      C:\Program Files\Java\jdk1.8.0_121
     </font>
     ，点击【确定】即可。变量值也可以选择浏览目录进行填写。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0abffcdf9d2c78d6dfa5cdd1995845a2.png#pic_center"/>
    </p>
    <h3>
     <a id="24_PATH_79">
     </a>
     2.4 配置第一个环境变量：PATH
    </h3>
    <p>
     第二个是
     <font color="#dd0000">
      PATH
     </font>
     变量，因为系统已经预先配置有了这个变量，我们只需要修改即可，点击【编辑】，进入修改界面；
     <br/>
     点击【新建】，在最下方的位置把JDK目录下的bin文件夹的路径复制进去，本人的为
     <font color="#dd0000">
      C:\Program Files\Java\jdk1.8.0_121\bin
     </font>
     ，再点击【确定】即可。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8d0b396d02eb3e97a727f2660074cccd.png#pic_center"/>
    </p>
    <h3>
     <a id="25_CLASSPATH_83">
     </a>
     2.5 配置第一个环境变量：CLASSPATH
    </h3>
    <p>
     第三个是
     <font color="#dd0000">
      CLASSPATH
     </font>
     变量，和
     <font color="#dd0000">
      JAVA_HOME
     </font>
     变量的配置一样，我们点击【新建】，【变量名】为
     <font color="#dd0000">
      CLASSPATH
     </font>
     ；【变量值】为JDK文件夹下的lib目录。
     <font color="#dd0000">
      C:\Program Files\Java\jdk1.8.0_121\lib
     </font>
     ，如图所示。然后点击所有对话框中的【确定】。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f99d9f7446ef28b5ce2660af4227a920.png#pic_center"/>
    </p>
