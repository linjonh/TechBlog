    <h2>
     <a id="0_0">
     </a>
     0、序
    </h2>
    <p>
     HarmonyOS内核提供的Shell支持调试常用的基本功能，同时HarmonyOS内核的Shell支持添加新的命令，可以根据需求来进行定制。本文主要介绍已经实现了的常用的一些Shell命令，HarmonyOS内核提供的Shell命令操作主要有以下几类：系统相关命令、文件相关命令、网络相关命令，下面就三部分进行介绍。
    </p>
    <h2>
     <a id="1_2">
     </a>
     1、系统命令
    </h2>
    <p>
     <strong>
      1）help
     </strong>
     <br/>
     help命令用于显示当前操作系统内所有操作指令。
    </p>
    <p>
     <strong>
      2）kill
     </strong>
     <br/>
     命令用于发送特定信号给指定进程。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     kill [signo | -signo] [pid]。
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        signo
       </td>
       <td>
        信号ID
       </td>
      </tr>
      <tr>
       <td>
        pid
       </td>
       <td>
        进程ID
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      3）reset
     </strong>
     <br/>
     reset用于设备的重启，输入reset命令后，设备会立即重启。
    </p>
    <p>
     <strong>
      4）uname
     </strong>
     <br/>
     uname命令用于显示当前操作系统的名称，版本创建时间，系统名称，版本信息等。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      uname [-a | -s | -t | -v | --help]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        无参数
       </td>
       <td>
        默认显示操作系统名称
       </td>
      </tr>
      <tr>
       <td>
        -a
       </td>
       <td>
        显示全部信息
       </td>
      </tr>
      <tr>
       <td>
        -t
       </td>
       <td>
        显示版本创建的时间
       </td>
      </tr>
      <tr>
       <td>
        -s
       </td>
       <td>
        显示操作系统名称
       </td>
      </tr>
      <tr>
       <td>
        -v
       </td>
       <td>
        显示版本信息
       </td>
      </tr>
      <tr>
       <td>
        –help
       </td>
       <td>
        显示uname指令格式提示
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      5）watch
     </strong>
     <br/>
     watch命令用于周期性的监视一个命令的运行结果。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      watch [-c/-n/-t/–count/–interval/-no-title/–over] [command]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        -c / --count
       </td>
       <td>
        命令执行的总次数D
       </td>
      </tr>
      <tr>
       <td>
        -n / --interval
       </td>
       <td>
        命令周期性执行的时间间隔（s）
       </td>
      </tr>
      <tr>
       <td>
        -t / -no-title
       </td>
       <td>
        关闭顶端的时间显示
       </td>
      </tr>
      <tr>
       <td>
        command
       </td>
       <td>
        需要监测的命令
       </td>
      </tr>
      <tr>
       <td>
        –over
       </td>
       <td>
        关闭当前监测指令
       </td>
      </tr>
     </tbody>
    </table>
    <h2>
     <a id="2_44">
     </a>
     2、文件命令
    </h2>
    <p>
     <strong>
      1）cd
     </strong>
     <br/>
     cd命令用来改变当前目录。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      cd [path]
     </em>
    </p>
    <p>
     <strong>
      2）cat
     </strong>
     <br/>
     cat命令用于显示文本的具体内容。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      cat [pathname]
     </em>
    </p>
    <p>
     <strong>
      3）chmod
     </strong>
     <br/>
     chmod用于修改文件操作权限。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      chmod [mode] [pathname]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        mode
       </td>
       <td>
        文件或文件夹权限，用8进制表示对应User、Group、及Other（拥有者、群组、其他组）的权限。
       </td>
      </tr>
      <tr>
       <td>
        pathname
       </td>
       <td>
        文件路径
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      4）chown
     </strong>
     <br/>
     chmod用于将指定文件的拥有者改为指定的用户或组。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     chown
     <em>
      [owner] [group] [pathname]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        owner
       </td>
       <td>
        文件拥有者
       </td>
      </tr>
      <tr>
       <td>
        group
       </td>
       <td>
        文件群组
       </td>
      </tr>
      <tr>
       <td>
        pathname
       </td>
       <td>
        文件路径
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      5)cp
     </strong>
     <br/>
     拷贝文件。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      cp [SOURCEFILE] [DESTFILE]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        SOURCEFILE
       </td>
       <td>
        源文件路径
       </td>
      </tr>
      <tr>
       <td>
        DESTFILE
       </td>
       <td>
        目的文件路径
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      6)ls
     </strong>
     <br/>
     显示当前目录的内容。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      ls [path]
     </em>
    </p>
    <p>
     <strong>
      7)mkdir
     </strong>
     <br/>
     用于创建一个目录。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      mkdir [directory]
     </em>
    </p>
    <p>
     <strong>
      8)rm
     </strong>
     <br/>
     用于删除文件或文件夹。
     <br/>
     <strong>
      命令格式：
     </strong>
     <br/>
     <em>
      rm [-r] [dirname / filename]
     </em>
     <br/>
     <strong>
      参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        -r
       </td>
       <td>
        可选参数，若是删除目录则需要该参数。
       </td>
      </tr>
      <tr>
       <td>
        dirname/filename
       </td>
       <td>
        要删除文件或文件夹的名称，支持输入路径。
       </td>
      </tr>
     </tbody>
    </table>
    <h2>
     <a id="3_107">
     </a>
     3、网络命令
    </h2>
    <p>
     <strong>
      1)ifconfig
     </strong>
     <br/>
     用来查询和设置网卡的IP地址、网络掩码、网关、硬件mac地址等参数。并能够启用/关闭网卡。
     <br/>
     <strong>
      常用参数说明
     </strong>
     ：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        不带参数
       </td>
       <td>
        打印所有网卡信息
       </td>
      </tr>
      <tr>
       <td>
        up
       </td>
       <td>
        启动网卡
       </td>
      </tr>
      <tr>
       <td>
        down
       </td>
       <td>
        关闭网卡
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      2)ping
     </strong>
     <br/>
     用于测试网络连接是否正常。
     <br/>
     <strong>
      常用参数说明：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        IP
       </td>
       <td>
        网络IPv4地址
       </td>
      </tr>
     </tbody>
    </table>
