    <p>
     学习大数据需要的基础
    </p>
    <p>
     1、java SE、EE（SSM）
     <br/>
     90%的大数据框架都是Java写的
     <br/>
     2、MySQL
     <br/>
     SQL on Hadoop
     <br/>
     3、Linux
     <br/>
     大数据的框架安装在Linux操作系统上
     <br/>
     <br/>
     - 需要学什么
    </p>
    <p>
     大数据离线分析
    </p>
    <p>
     一般处理T+1数据（T：可能是1天、一周、一个月、一年）
     <br/>
     a、Hadoop ：一般不选用最新版本，踩坑难解决
     <br/>
     （common、HDES、MapReduce、YARN）
     <br/>
     环境搭建、处理数据的思想
     <br/>
     b、Hive：大数据的数据仓库
     <br/>
     通过写SQL对数据进行操作，类似于MySQL数据库的sql
     <br/>
     c、HBase：基于HDFS的NOSQL数据库
     <br/>
     面向列存储
     <br/>
     d、协作框架：
     <br/>
     sqoop（桥梁：HDFS《==》RDBMS）
     <br/>
     flume：搜集日志文件中的信息
     <br/>
     e、调度框架
     <br/>
     anzkaban
     <br/>
     了解：crotab（Linux自带）
     <br/>
     zeus（Alibaba）
     <br/>
     Oozie（cloudera）
     <br/>
     f、前沿框架扩展：
     <br/>
     kylin、impala、ElasticSearch（ES）
    </p>
    <p>
     <br/>
     大数据实时分析
    </p>
    <p>
     以spark框架为主
     <br/>
     Scala：OOP（面向对象程序设计）+FP（函数是程序设计）
     <br/>
     sparkCore：类比MapReduce
     <br/>
     sparkSQL：类比hive
     <br/>
     sparkStreaming：实时数据处理
     <br/>
     kafka：消息队列
     <br/>
     前沿框架扩展：flink
     <br/>
     阿里巴巴：blink
    </p>
    <p>
     大数据机器学习
    </p>
    <p>
     spark MLlib：机器学习库
     <br/>
     pyspark编程：Python和spark的结合
     <br/>
     推荐系统
     <br/>
     python数据分析
     <br/>
     python机器学习
     <br/>
     ---------------------
    </p>
