      <wbr/>
      <span style="FONT-SIZE: 12px">
       <wbr/>
      </span>
      <span style="FONT-SIZE: 12px">
       （１）关键字检索
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       在百度主页的检索栏内输入关键字串，单击“检索”按钮，百度搜索引擎会搜索中文分类条目、资料库中的网站信息以及新闻资料库，搜索完毕后将检索的结果显示出来，单击某一链接查看详细内容。
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       （２）使用双引号（“”）进行词组检索
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       利用双引号查询完全符合关键字串的网站。例如：输入“中国女足”找出包含“中国女足”的网站，而不会找出包含“中国男足”的网站。
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       （３）使用“＋”和“－”进行限制性检索
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       当我们需要检索结果中包含有两个或两个以上的内容，这时我们可以把几个条件之间用“＋“号连接。这样关键字串一定要出现在结果中，例如想查询那英的歌曲《
       <a href="http://www.sogou.com/sogoupedia?query=%D5%F7%B7%FE" rel="noopener noreferrer" style="TEXT-DECORATION: underline" target="_blank">
        <font color="#000066">
         征服
        </font>
       </a>
       》，你可以输入“
       <a href="http://www.sogou.com/sogoupedia?query=%C4%C7%D3%A2" rel="noopener noreferrer" style="TEXT-DECORATION: underline" target="_blank">
        <font color="#000066">
         那英
        </font>
       </a>
       ＋征服”。
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       在查询某个题材时并不希望在这个题材中包含另一个题材，这时你就可以使用“－”号。例如你想查找“水果”，但又不希望其中包含“苹果”。你就可以输入“水果－苹果”。减号的作用就在于可以使搜索的结果中反映你的需求，让你无须为大量无关的搜索结果而头疼。
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       （４）仅搜索网站的网址
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       在关键字前加“u:”，搜索引擎仅会查询网址，例如在网页中的搜索框中输入“ｕ：yahoo.com”，单击“搜索”按钮，则统一资源定位器中包含“yahoo.com”字符的网址全部显示出来。
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       （５）仅搜索网站标题
       <br/>
       <br/>
      </span>
      <wbr/>
      <wbr/>
      <wbr/>
      <span style="FONT-SIZE: 12px">
       在关键字前加“t:”，搜索引擎仅查询网站的名称。例如在网站搜索框中输入“t:网络技术”，单击“搜索”按钮，符合搜索条件的网站标题显示出来。
      </span>
