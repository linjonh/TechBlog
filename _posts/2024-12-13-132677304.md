    <h2>
     <a id="Google2_1">
     </a>
     Google官方提供2种测试人员参与测试方式
    </h2>
    <ul>
     <li>
      在 Android 设备上参与测试
     </li>
    </ul>
    <blockquote>
     <p>
      测试人员可以在 Android 设备上通过 Google Play 参与测试
      <br/>
      <a href="https://play.google.com/store/apps/details?id=com.huangbaoche.guide" rel="nofollow">
       https://play.google.com/store/apps/details?id=com.huangbaoche.guide
      </a>
     </p>
    </blockquote>
    <ul>
     <li>
      在网页中参与测试
     </li>
    </ul>
    <blockquote>
     <p>
      测试人员可以在相应网页中参与测试
      <br/>
      <a href="https://play.google.com/apps/testing/com.huangbaoche.guide" rel="nofollow">
       https://play.google.com/apps/testing/com.huangbaoche.guide
      </a>
     </p>
    </blockquote>
    <h2>
     <a id="_9">
     </a>
     纯粹英雄推荐步骤
    </h2>
    <ol>
     <li>
      <a href="https://play.google.com/apps/testing/com.huangbaoche.guide" rel="nofollow">
       点击此链接浏览器打开
      </a>
     </li>
     <li>
      打开的页面底部，显示邀请加入成为测试员的按钮，点击就可以加入，如图
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4171e183a87c8fbdbb80f1920db8d29.png"/>
     </li>
     <li>
      点击右下角“BECOME A TESTER”按钮，即可成功加入。如下图
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6e856029dfbf1f36b2db1a1d68bc4e06.png"/>
     </li>
     <li>
      点击上一步中"download it on Google Play"，即可打开GooglePlay进行安装。（步骤同普通安装过程）
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8f0e0c5e120e0b4828a0ef217011e22c.png">
       <br/>
       <strong>
        Tips
       </strong>
       ：如果GooglePlay账号没有登录，会要求先登录才能加入。
      </img>
     </li>
    </ol>
