     国内云计算第一股关停 IoT 物联网云服务，数据不保留，企业被迫迁移存量设备...
    </h1>

    <div id="js_content">
     <p style="text-align:left;">
      近日，有读者反馈，收到 UCloud 官方通知，称其
      <strong>
       物联网UIoT
      </strong>
      公有云平台即将在2022年3月31日0时
      <strong>
       下线
      </strong>
      ，
      <strong>
       企业需尽快完成存量设备迁移，下线后所有数据都不再保留。
      </strong>
      <br/>
     </p>
     <p style="text-align:center;">
      <img alt="71a7ee0987ccd4590231a37b2ef78622.png" src="https://i-blog.csdnimg.cn/blog_migrate/d0be04c8707534ae525d5b356ac210eb.png"/>
     </p>
     <p style="text-align:left;">
      UCloud 官方于2019年8月1发布
      <strong>
       物联网UIoT
      </strong>
      公有云服务，不到3年就宣布IoT云产品下线。
     </p>
     <p style="text-align:left;">
      这让当初选择UCloud物联网平台的中小企业，不得不面临
      <strong>
       设备迁移的窘境
      </strong>
      。刚过完春节的程序员们深陷加班的苦海，需在一个月时间内，
      <strong>
       重新选择云厂商IoT物联网平台
      </strong>
      ，完成设备端和业务系统的升级改造，并
      <strong>
       推送设备固件升级
      </strong>
      。
     </p>
     <p style="text-align:left;">
      作为中小企业都应从中吸取教训，优先选择实力雄厚的云厂商，避免遭遇到类似
      <strong>
       美团云
      </strong>
      <strong>
       关闭
      </strong>
      ，
      <strong>
       苏宁云
      </strong>
      <strong>
       关闭
      </strong>
      等黑天鹅事件。
     </p>
     <p style="text-align:center;">
      <strong>
       01
      </strong>
     </p>
     <p style="text-align:center;">
      <strong>
       UCloud IoT 云产品
      </strong>
     </p>
     <p style="text-align:left;">
      2020 年1月，UCloud正式在上交所科创板挂牌上市，成为
      <strong>
       中国云计算第一股
      </strong>
      ，也是中国A股市场第一家
      <strong>
       同股不同权
      </strong>
      的上市公司。开市后，风光无两，最高达到126元，市值接近570亿元。
     </p>
     <p style="text-align:left;">
      随后UCloud陷入困境，一边是资本失去信心，二是大股东以及高管开始纷纷减持。目前股价维持在22元，
      <strong>
       市值缩水80%
      </strong>
      。
     </p>
     <p>
      <img alt="672e37e40df11d5964b12195dde0c29c.png" src="https://i-blog.csdnimg.cn/blog_migrate/97b5f0f352e2beaecbedaeec40a6dbd7.png"/>
     </p>
     <p style="text-align:left;">
      UCloud
      <strong>
       物联网通信云平台
      </strong>
      （UIoT Core）为设备上云和产业互联网数字化转型提供重要支撑，架起设备与云服务连接的桥梁，平台支持海量设备的连接，方便设备数据上云；同时提供云端API，方便用户云端应用开发，缩短用户开发周期，减少运维成本。
     </p>
     <p style="text-align:left;">
      物联网通信云平台核心功能包括
      <strong>
       设备管理、物模型、设备影子、规则引擎、文件上传、固件远程升级
      </strong>
      等服务，可广泛应用于智慧城市、智能制造、智能家居、智慧农业、无人值守、新零售、智慧能源、智慧医疗等领域。
     </p>
     <p style="text-align:center;">
      <strong>
       02
      </strong>
     </p>
     <p style="text-align:center;">
      <strong>
       IoT 物联网平台倒闭潮
      </strong>
     </p>
     <p style="text-align:left;">
      UCloud并非第一个关闭自家 IoT 物联网服务的企业，早在2020年移动消息推送上市企业极光，就
      <strong>
       关闭
      </strong>
      了仅运营一年多的
      <strong>
       极光JIoT 物联网平台
      </strong>
      。
     </p>
     <p style="text-align:center;">
      <img alt="a786a1e0a2b0ec1dc5f89b303de66fd4.png" src="https://i-blog.csdnimg.cn/blog_migrate/8dddcfec42c40296892ebb194b5327a9.png"/>
     </p>
     <p style="text-align:left;">
      IoT物联网平台停止服务也不是国内独有的现象。据知名研究机构IoT Analytics 统计，在2021年全球范围出现了物联网平台“倒闭潮”，消失了近30%物联网平台型企业。
     </p>
     <p style="text-align:center;">
      <strong>
       03
      </strong>
     </p>
     <p style="text-align:center;">
      <strong>
       云厂商引领 IoT 平台发展
      </strong>
     </p>
     <p style="text-align:left;">
      物联网云平台为用户提供稳定、智能的服务和应用程序，需要巨大的投入和坚定的战略，而平台的盈利回报期往往很长，导致大量公司无法负担而关闭服务。
     </p>
     <p style="text-align:left;">
      除了关闭服务，一些IoT物联网平台公司选择转变商业模式，从连接设备和应用的PaaS云平台服务商，转为提供
      <strong>
       垂直领域应用解决方案的服务商
      </strong>
      ，以获得更快的盈利。
     </p>
     <p style="text-align:left;">
      这也导致
      <strong>
       IoT物联网平台
      </strong>
      市场进一步
      <strong>
       向头部云厂商集中
      </strong>
      。根据市场数据，
     </p>
     <p style="text-align:left;">
      物联网平台市场排名前10位的公司占有65%的份额，而在2019年和2016
     </p>
     <p style="text-align:left;">
      年，这一数字分别为58% 和44% 。
     </p>
     <p style="text-align:left;">
      <strong>
       AWS
      </strong>
      IoT Core、
      <strong>
       微软
      </strong>
      Azure IoT Hub 、
      <strong>
       阿里云
      </strong>
      IoT 物联网平台
      <strong>
       三巨
      </strong>
      <strong>
       头格局
      </strong>
      已形成，并以每年50%的增长率继续引领市场。三大云平台以服务的稳定性获得全球80%用户的信赖，不断发布新的产品和服务，以满足中小企业物联网应用的各种需求，使得物联网门槛不断降低，推动了整个物联网产业生态的形成。
     </p>
     <p style="text-align:left;">
      最后，赠送
      <strong>
       20元优惠券
      </strong>
      ，仅需一杯咖啡，加入
      <strong>
       IoT物联网最佳实践
      </strong>
      知识星球
      <strong>
       ，
      </strong>
      <strong>
       国内最大IoT物联网开发者互动社区，获取 1000+行业资料
      </strong>
      。
     </p>
