    <h2>
     <a id="nginx_0">
     </a>
     nginx-静态文件服务器
    </h2>
    <h5>
     <a id="1tomcattomcatnginxtomcatlocation_1">
     </a>
     1、代替tomcat作为静态文件服务器，缓解tomcat的访问压力，如果项目是前后端分离，前端使用nginx，后端使用tomcat，并且是在同一个电脑上，直接加一个location访问静态资源文件.
    </h5>
    <blockquote>
     <p>
      例如静态资源访问路径可用/statics/过滤出来。之后设置允许访问的文件类型，这里使用后缀名做限定，不符合的返回403。
      <br/>
      alias指定文件的绝对路径，确保alias + 请求路径/statics/之后的部分能够匹配到具体的文件就可以了
      <br/>
      因为是访问具体的文件，不想让别人访问到文件目录，就不用设置autoindex了，或者设置autoindex为off
     </p>
    </blockquote>
    <pre><code class="prism language-bash">location /statics/<span class="token punctuation">{<!-- --></span>
	<span class="token comment"># !~* 不区分大小写的匹配取非</span>
	<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token variable">$uri</span> <span class="token operator">!</span>~* <span class="token punctuation">\</span>.<span class="token punctuation">(</span>pdf<span class="token operator">|</span>txt<span class="token operator">|</span>doc<span class="token operator">|</span>docx<span class="token operator">|</span>xls<span class="token operator">|</span>xlsx<span class="token punctuation">)</span>$<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    	<span class="token builtin class-name">return</span> <span class="token number">403</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
	<span class="token builtin class-name">alias</span> D:/archive/<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>

    <h5>
     <a id="2nginxnginx_24">
     </a>
     2、如果前后端在不同的电脑上，静态资源文件跟随后端程序，那么需要在后端电脑上启动一个nginx,。在前端电脑的nginx，设置一个反向代理即可。
    </h5>
    <blockquote>
     <p>
      例如后端nginx这样设置，只允许本机访问和前端的nginx访问，location设置的路径匹配根据前端nginx反向代理拼接的路径设置。
     </p>
    </blockquote>
    <pre><code class="prism language-bash">location /archives/<span class="token punctuation">{<!-- --></span>
	<span class="token builtin class-name">alias</span> D:/archive/<span class="token punctuation">;</span>
	allow <span class="token number">127.0</span>.0.1<span class="token punctuation">;</span>     <span class="token comment">#允许本及访问</span>
	allow <span class="token number">192.168</span>.0.44<span class="token punctuation">;</span>  <span class="token comment">#允许前端nginx访问,前端nginx电脑的ip</span>
	deny all<span class="token punctuation">;</span>            <span class="token comment">#其他的都禁止</span>
<span class="token punctuation">}</span>
</code></pre>
    <blockquote>
     <p>
      例如前端nginx这样设置
     </p>
    </blockquote>
    <pre><code class="prism language-bash">location /statics/<span class="token punctuation">{<!-- --></span>
	<span class="token comment"># !~* 不区分大小写的匹配取非,根据后缀名过滤控制能够访问到的文件</span>
	<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token variable">$uri</span> <span class="token operator">!</span>~* <span class="token punctuation">\</span>.<span class="token punctuation">(</span>pdf<span class="token operator">|</span>txt<span class="token operator">|</span>doc<span class="token operator">|</span>docx<span class="token operator">|</span>xls<span class="token operator">|</span>xlsx<span class="token punctuation">)</span>$<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    	<span class="token builtin class-name">return</span> <span class="token number">403</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
	proxy_set_header Host <span class="token variable">$http_host</span><span class="token punctuation">;</span>
	proxy_set_header X-Real-IP <span class="token variable">$remote_addr</span><span class="token punctuation">;</span>
	proxy_set_header REMOTE-HOST <span class="token variable">$remote_addr</span><span class="token punctuation">;</span>
	proxy_set_header X-Forwarded-For <span class="token variable">$proxy_add_x_forwarded_for</span><span class="token punctuation">;</span>
	proxy_pass http://192.168.0:55/archives/<span class="token punctuation">;</span> <span class="token comment"># 后端nginx的ip,此处拼接了一个/archives/,便于后端nginx统一拦截处理。</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     3、显示过程中，如果出现中文乱码，设置一下charset
    </p>
    <pre><code class="prism language-bash">server <span class="token punctuation">{<!-- --></span>

        server_name  localhost<span class="token punctuation">;</span>

		charset utf-8<span class="token punctuation">;</span>
		<span class="token punctuation">..</span><span class="token punctuation">..</span><span class="token punctuation">..</span>
</code></pre>
