      <h2 id="python-解决-nameerror-name-reload-is-not-defined-问题">
       Python 解决 ：
       <code>
        NameError: name 'reload' is not defined
       </code>
       问题
      </h2>
      <h3 id="解决nameerror-name-reload-is-not-defined-问题">
       解决：
       <code>
        NameError: name 'reload' is not defined
       </code>
       问题
      </h3>
      <h4 id="对于-python-2x">
       对于
       <strong>
        Python 2.X
       </strong>
       ：
      </h4>
      <pre class="prettyprint"><code class="hljs erlang">import sys
<span class="hljs-function"><span class="hljs-title">reload<span class="hljs-params">(sys)
<span class="hljs-title">sys.<span class="hljs-title">setdefaultencoding<span class="hljs-params">(<span class="hljs-string">"utf-8")</span></span></span></span></span></span></span></code></pre>
      <hr/>
      <h4 id="对于-python-33">
       对于 &lt;=
       <strong>
        Python 3.3
       </strong>
       ：
      </h4>
      <pre class="prettyprint"><code class="hljs java"><span class="hljs-keyword">import imp
imp.reload(sys)</span></code></pre>
      <blockquote>
       <p>
        <strong>
         注意：
        </strong>
        <br/>
        1.
        <strong>
         Python 3
        </strong>
        与
        <strong>
         Python 2
        </strong>
        有
        <strong>
         很大的区别
        </strong>
        ，其中
        <strong>
         Python 3
        </strong>
        系统默认使用的就是
        <code>
         utf-8
        </code>
        编码。
        <br/>
        2. 所以，对于使用的是
        <strong>
         Python 3
        </strong>
        的情况，就不需要
        <code>
         sys.setdefaultencoding("utf-8")
        </code>
        这段代码。
        <br/>
        3.
        <strong>
         最重要的是
        </strong>
        ，
        <strong>
         Python 3
        </strong>
        的
        <strong>
         sys
        </strong>
        库里面已经
        <strong>
         没有
        </strong>
        <code>
         setdefaultencoding()
        </code>
        函数了。
       </p>
      </blockquote>
      <h4 id="对于-python-34">
       对于 &gt;=
       <strong>
        Python 3.4
       </strong>
       ：
      </h4>
      <pre class="prettyprint"><code class="hljs java"><span class="hljs-keyword">import importlib
importlib.reload(sys)</span></code></pre>
