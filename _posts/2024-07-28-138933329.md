     微信小程序接入阿里云物联网平台
    </h4>
    <h4>
     <a id="__22">
     </a>
     一 阿里云平台端
    </h4>
    <p>
     1.登录阿里云
     <br/>
     <a href="https://bbs.csdn.net/topics/618679757">
      阿里云物联网平台
     </a>
     <br/>
     点击进入
     <code>
      公共实例
     </code>
     ，之前没有的点进去申请
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/151482408ff160d97e6b36b59d1e061c.png">
      <br/>
      2.点击
      <strong>
       产品
      </strong>
      —&gt;
      <strong>
       创建产品
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b3f0b17c356087414c93498b902b383f.png">
       <br/>
       3.产品名称自定义，按项目选择类型，节点类型选择
       <strong>
        直连设备
       </strong>
       ，联网方式
       <strong>
        WiFi
       </strong>
       ，数据格式选择
       <strong>
        透传/自定义
       </strong>
       ，其他默认
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d380430cc0edc9e184d58cc899a2d3ca.png">
        <br/>
        4.
        <strong>
         添加设备
        </strong>
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fdbca0b5b450561b1120ca1ccb91491e.png">
         <br/>
         设备名称自定义，我这里是与微信通讯，这里设备名称定义为WeChat，（随便定义的）
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b344d76d042af9e1b977bbf9f7110ed6.png">
          <br/>
          5.进入刚刚添加的设备，点击
          <strong>
           查看
          </strong>
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/633b939bf4f66f8a3da960dbf91e99eb.png">
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d6f8ba0ccfaf87685f54b74fc3e44284.png"/>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/70fa094c6cf5cbd875391c42508954fe.png"/>
     <br/>
     记住上面的阿里云
     <code>
      三元组
     </code>
     ，这个是每一个设备具有的，微信小程序那边要用到
    </p>
    <pre><code>{
  "ProductKey": "a1vHYVO6GTw", 
  "DeviceName": "wechat",
  "DeviceSecret": "a29359d701c20387185125aa0ec19872"
}

</code></pre>
    <ol start="6">
     <li>
      进入
      <strong>
       产品
      </strong>
      ，选择
      <strong>
       自定义Topic
      </strong>
      ，然后点击
      <strong>
       定义Topic主题
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ed66619528fbc4b64909a320c0dd30fa.png"/>
      <br/>
      选择设备权限为
      <strong>
       发布和订阅
      </strong>
      ，Topic主题名称自定义，最好是规范命名，这里作为演示就把它命名为topic,记住我们创建的主题，点击
      <strong>
       确定
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9dabd1cc25bffeb003e5ba934c48784c.png"/>
     </li>
     <li>
      进入
      <strong>
       云产品流转
      </strong>
      ，点击
      <strong>
       创建规则
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/538209e4f262e131ac3f17b650422b57.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cea22e3677fba03b53c9813875244a76.png"/>
      <br/>
      填写信息，规则名称随意写，格式选择
      <code>
       json
      </code>
      格式
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/42972e0678a7b26b618004fbb353a2e9.png"/>
      <br/>
      创建之后进入，点击
      <strong>
       编写SQL
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3043832dae27811ce948d034d7d4bddb.png"/>
      <br/>
      <strong>
       字段
      </strong>
      很重要！！！
      <em>
       阿里云会根据这个来选择转发的数据，字段填的是json格式的键，如果不想那么麻烦就填
       <code>
        *
       </code>
       号
      </em>
      ，选择我篇创建的产品，设备，及刚刚创建的主题名称，点击确定
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ba45c624f59595d3f60c82a5185fe280.png"/>
      <br/>
      即可看到这个
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/68e329870604ee058af519c44eec91b5.png"/>
      <br/>
      双引号里面的内容就是我们的主题，记录下来，小程序要用
     </li>
    </ol>
    <pre><code>         /a1vHYVO6GTw/wechat/user/topic

</code></pre>
