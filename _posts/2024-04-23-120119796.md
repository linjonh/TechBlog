    <p>
     数学物理方程就是具有物理背景的数学方程。
    </p>
    <p>
     微分方程包含常微分方程（ODE）和偏微分方程（PDE）。前者在高等数学中简单的学过，ODE是指方程只对一个变量求导。PDE是指方程对多个变量求导(如x, y, t)。
     <br/>
     除此之外数学物理方程还有积分方程（不讨论）。
    </p>
    <p>
     二阶线性偏微分方程（二阶值最高求两次导）
    </p>
    <p>
     本章研究二阶线性偏微分方程。
    </p>

       定解问题
      </strong>
      。
      <br/>
      偏微分方程给出了场中时间空间的联系，偏微分方程称为
      <strong>
       泛定方程
      </strong>
      。但是还需要给出
      <strong>
       定解条件
      </strong>
      ，即初始条件（除非是稳定场）和边界条件（除非是无边界）才能求解数学物理方程。泛定条件和定解条件一起称为
      <strong>
       定解问题
      </strong>
      。
     </li>
    </ol>
    <p>
     求解方法：
     <br/>
     波动方程可以用-----行波法
     <br/>
     所有方程可以用-----
     <strong>
      分离变量法
     </strong>
     ，
     <br/>
     积分变换法（主要
     <strong>
      拉普拉斯积分
     </strong>
     变换，傅里叶积分变换）
     <br/>
     格林函数法
     <br/>
     其他：变分法，数值解（构造差分方程，求数值解）
    </p>

     <a id="_81">

     定解条件

    <h4>
     <a id="_82">
     </a>
     初始条件：
    </h4>
    <p>
     对于含有t的泛定方程，需要给出初始条件。初始条件是关于xyz的函数。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3a947d70df395e1304419204683ea1a9.png">

      泊松方程没有初始条件。因为不含t。
     </img>
    </p>
    <h4>
     <a id="_86">
     </a>
     边界条件
    </h4>
    <p>
     边界点上关于t的函数。
    </p>
    <p>
     第一类边界条件：给出了未知函数在边界上的值。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a86d8dbcd84edf47e76ea334dc17da3.png">
      <br/>
      第二类边界条件：
      <br/>
      给出边界法向的导数值。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4674ef663f8a4b8ec5b220aef9da0342.png">
       <br/>
       第三类边界条件：
       <br/>
       给出边界的值和法向导数的值的线性组合的值。
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c6ac715781548653db510022669fab9b.png"/>
    </p>
    <p>
     边界条件还可以分齐次非齐次，如果等号右边为0，则称为齐次，否则为非齐次。
    </p>
    <h3>
     <a id="_102">
     </a>
     定解问题的分类：
    </h3>
    <ol>
     <li>
      柯西问题：只有初始条件，没有边界条件。
      <br/>
      常常处理无界的问题
     </li>
     <li>
      边值问题：没有初始条件，只有边界条件。
      <br/>
      常常处理狄氏问题，如稳定的温度场。
     </li>
     <li>
      混合问题：有初始条件和边界条件。
     </li>
    </ol>
