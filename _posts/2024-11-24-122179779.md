    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <strong>
       第五讲 计算机病毒、木马
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <span style="color:#ff0066;">
       <strong>
        恶意代码
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        是指在
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        未经授权
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        的情况下，在信息系统中安装、
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        执行
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        并达到不正当目的的
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        程序
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        。
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        计算机病毒其实是一种
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        程序
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        ，这种程序具有
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        再生能力
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        ，它会
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        自动
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        地通过修改其他程序并把本身嵌入其他程序或者将自身
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        复制
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        到其他存储介质中，从而“感染”其他程序，在满足一定
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        条件
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        时，该程序就
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        干扰
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        电脑正常工作，搞乱或破坏已有存储的信息，甚至引起整个电脑系统不能正常工作。
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        计算机病毒的主要特点
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        ：
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        可执行性 传染性 非授权性 寄生性
       </strong>
      </span>
     </strong>
     <strong>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        隐蔽性  衍生性  破坏性
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      <span style="color:#ff0000;">
       <strong>
        木马不具有
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000;">
       <strong>
        自我复制性，而具有很强的

      针对性
