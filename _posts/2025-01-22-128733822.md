     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       在 Python 中拼接（连接）字符串很简单，可以直接将两个字符串紧挨着写在一起，具体格式为：
      </span>
     </p>
     <p style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       strname = "str1" "str2"
      </span>
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       strname 表示拼接以后的字符串变量名，str1 和 str2 是要拼接的字符串内容。使用这种写法，Python 会自动将两个字符串拼接在一起。
      </span>
     </p>
     <p style="">
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       【示例】以连续书写的形式拼接字符串：
      </span>
     </p>
     <pre class="kdocs-plaintext"><code class="language-plaintext">str1 ="Python教程""https://edu.csdn.net/skill/python/"
print(str1)
str2 ="Java""Python""C++""PHP"
print(str2)</code></pre>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       运行结果：
      </span>
     </p>
     <blockquote class="kdocs-blockquote" style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       Python教程
      </span>
      <a class="kdocs-link" href="https://edu.csdn.net/skill/python/" rel="noopener noreferrer" style="color:#0A6CFF;" target="_blank">
       <span class="kdocs-fontSize" style="font-size:9pt;">
        https://edu.csdn.net/skill/python
       </span>
      </a>
      <a class="kdocs-link" href="https://edu.csdn.net/skill/python/" rel="noopener noreferrer" style="color:#0A6CFF;" target="_blank">
       <span class="kdocs-fontSize" style="font-size:9pt;">
        /
       </span>
      </a>
      <br/>
      <span class="kdocs-fontSize" style="font-size:9pt;">
       JavaPythonC++PHP
      </span>
     </blockquote>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       需要注意的是，这种写法只能拼接字符串常量。
      </span>
     </p>
     <p style="">
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       如果需要使用变量，就得借助+运算符来拼接，具体格式为：
      </span>
     </p>
     <p style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       strname = str1 + str2
      </span>
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       当然，+运算符也能拼接字符串常量。
      </span>
     </p>
     <p style="">
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       【示例】使用+运算符拼接字符串：
      </span>
     </p>
     <pre class="kdocs-plaintext"><code class="language-plaintext">name ="C语言教程"
url ="https://edu.csdn.net/skill/c/"
info = name +"的网址是："+ url
print(info)</code></pre>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       运行结果：
      </span>
     </p>
     <blockquote class="kdocs-blockquote" style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       C语言教程的网址是：
      </span>
      <a class="kdocs-link" href="https://edu.csdn.net/skill/c/" rel="noopener noreferrer" style="color:#0A6CFF;" target="_blank">
       <span class="kdocs-fontSize" style="font-size:9pt;">
        https://edu.csdn.net/skill/c/
       </span>
      </a>
     </blockquote>
     <h3 style="text-align:left;">
      Python字符串和数字的拼接
     </h3>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       在很多应用场景中，我们需要将字符串和数字拼接在一起，而 Python 不允许直接拼接数字和字符串，所以我们必须先将数字转换成字符串。可以借助 str() 和 repr() 函数将数字转换为字符串，它们的使用格式为：
      </span>
     </p>
     <blockquote class="kdocs-blockquote" style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       str(obj)
      </span>
      <br/>
      <span class="kdocs-fontSize" style="font-size:9pt;">
       repr(obj)
      </span>
     </blockquote>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       obj 表示要转换的对象，它可以是数字、列表、元组、字典等多种类型的数据。
      </span>
     </p>
     <p style="">
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       请看下面的代码：
      </span>
     </p>
     <pre class="kdocs-plaintext"><code class="language-plaintext">name ="CSDN社区"
age =23
course =30
info = name +"已经"+str(age)+"岁了，共发布了"+repr(course)+"套教程。"
print(info)</code></pre>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       运行结果：
      </span>
     </p>
     <blockquote class="kdocs-blockquote" style="text-align:null;">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       CSDN社区已经23岁了，共发布了30套教程。
      </span>
     </blockquote>
     <h5 style="text-align:left;">
      str() 和 repr() 的区别
     </h5>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       str() 和 repr() 函数虽然都可以将数字转换成字符串，但它们之间是有区别的：
      </span>
     </p>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        str() 用于将数据转换成适合人类阅读的字符串形式。
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        repr() 用于将数据转换成适合解释器阅读的字符串形式（Python 表达式的形式），适合在开发和调试阶段使用；如果没有等价的语法，则会发生 SyntaxError 异常。
       </p>
      </li>
     </ul>
     <p style="">
     </p>
     <p style="">
      <span class="kdocs-fontSize" style="font-size:9pt;">
       请看下面的例子：
      </span>
     </p>
     <pre class="kdocs-plaintext"><code class="language-plaintext">s ="https://edu.csdn.net/skill/"
s_str =str(s)
s_repr =repr(s)

print(type(s_str))
print(s_str)
print(type(s_repr))
print(s_repr)</code></pre>
<p style="">
<span class="kdocs-fontSize" style="font-size:9pt;">
运行结果：
</span>
</p>
<blockquote class="kdocs-blockquote" style="text-align:null;">
<span class="kdocs-fontSize" style="font-size:9pt;">
&lt;class 'str'&gt;
</span>
<br/>
https://edu.csdn.net/skill/
<br/>
<span class="kdocs-fontSize" style="font-size:9pt;">
&lt;class 'str'&gt;
</span>
<br/>
https://edu.csdn.net/skill/
</blockquote>
<p style="">
<span class="kdocs-fontSize" style="font-size:9pt;">
本例中，s 本身就是一个字符串，但是我们依然使用 str() 和 repr() 对它进行了转换。从运行结果可以看出，str() 保留了字符串最原始的样子，而 repr() 使用引号将字符串包围起来，这就是 Python 字符串的表达式形式。
</span>
</p>
<p style="">
</p>
<p style="">
<span class="kdocs-fontSize" style="font-size:9pt;">
另外，在 Python 交互式编程环境中输入一个表达式（变量、加减乘除、逻辑运算等）时，Python 会自动使用 repr() 函数处理该表达式。
</span>
</p>
