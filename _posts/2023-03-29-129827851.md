    <h2>
     <a id="FFmpeg_1">
     </a>
     FFmpeg参数
    </h2>
    <h3>
     <a id="_3">
     </a>
     主要参数
    </h3>
    <ul>
     <li>
      <p>
       -i 设定输入流
      </p>
     </li>
     <li>
      <p>
       -f : 强制输入或输出文件的格式。
      </p>
      <p>
       <strong>
        ffmpeg -formats 可以列出所有的formats.
       </strong>
      </p>
      <p>
       对于输入，如果不指定-f, ffmpeg 会根据输入数据（来自于输入文件，或输入设备或其它协议）来判断数据的封装格式.
      </p>
      <p>
       对于输出，如果不指定-f， ffmpeg 也可以通过输出文件名进行推导.
      </p>
     </li>
     <li>
      <p>
       -ss 开始时间
      </p>
     </li>
     <li>
      <p>
       -t 时间长度
      </p>
     </li>
    </ul>
    <h3>
     <a id="_19">
     </a>
     音频参数
    </h3>
    <p>
     <strong>
      -a a指的是audio。
     </strong>
    </p>
    <ul>
     <li>
      -acodec &lt;codec_name&gt; ：设定声音编解码器，（编码器设定为”copy“，保持原有的编解码）
     </li>
     <li>
      copy：
      <em>
       copy:用于指定输出视频流和输入视频流的编码格式保持一致
      </em>
      。
     </li>
     <li>
      -an： （audio no）疏略音频，只保留视频。
     </li>
     <li>
      -ar &lt;音频采样率&gt;：指定音频的采样率。例
      <code>
       ffmpeg -i test.mp4 -ar 44100
      </code>
     </li>
     <li>
      -ac &lt;整数&gt;: (audio channel)：指定音频声道数量。 例
      <code>
       ffmpeg -i test.mpr -ac
      </code>
     </li>
     <li>
      -c:a &lt;编码器名字&gt;: 指定音频的编码器。
     </li>
    </ul>
    <h3>
     <a id="_30">
     </a>
     视频参数
    </h3>
    <p>
     -v v指的是video
    </p>
    <ul>
     <li>
      -vn：（video no）忽略视频，只保留音频。
     </li>
     <li>
      -vcodec &lt;codec_name&gt;: 设定视频编解码器（编码器设定为”copy“表示不进行编解码,即保持原来的编解码格式）。
     </li>
     <li>
      copy：
      <em>
       copy:用于指定输出视频流和输入视频流的编码格式保持一致
      </em>
      。
     </li>
     <li>
      -pix_fmt &lt;像素格式&gt;：指定视频像素格式
     </li>
     <li>
      -c:v &lt;编码器&gt;： 指定视频编码器。（-c:v copy。 保持原有的）
     </li>
     <li>
      -vf &lt;滤镜&gt;: 指定路径名字。例
      <code>
       ffmpeg -i input.mp4 -vf crop=in_w-100:in_h-100 output.mp4
      </code>
     </li>
     <li>
      -r &lt;帧率&gt;：设定帧速率。
     </li>
    </ul>
    <h4>
     <a id="_42">
     </a>
     其他参数
    </h4>
    <ul>
     <li>
      -re: 要求按照既定速率处理输入数据，这个速率即是输入文件的帧率
     </li>
    </ul>
    <h3>
     <a id="_48">
     </a>
     格式值
    </h3>
    <h4>
     <a id="_50">
     </a>
     声音格式
    </h4>
    <ul>
     <li>
      <p>
       s16le
      </p>
      <p>
       s16le是一种无压缩的
       <strong>
        声音格式
       </strong>
       ，
       <strong>
        全称为signed 16-bit little-endian
       </strong>
       ，
       <strong>
        意思是有符号的16位音频数据用小端字节序来存储
       </strong>
       。这种格式通常用于存储语音、音效等低质量的音频数据，因为它可以提供音频采样的全范围动态范围，从-32768到32767，能够提供足够的音量和质量，同时比其他无损或有损音频格式的文件大小较小。
      </p>
     </li>
    </ul>
    <h4>
     <a id="_56">
     </a>
     滤镜值
    </h4>
    <ul>
     <li>
      <p>
       crop
      </p>
      <p>
       FFmpeg 的 crop 滤镜可以用于裁剪视频，其使用方法如下:
      </p>
      <ol>
       <li>
        <p>
         基本语法：
         <code>
          crop=&lt;width&gt;:&lt;height&gt;:&lt;x&gt;:&lt;y&gt;
         </code>
        </p>
        <p>
         其中，
         <code>
          &lt;width&gt;
         </code>
         表示裁剪后视频的宽度，
         <code>
          &lt;height&gt;
         </code>
         表示裁剪后视频的高度，
         <code>
          &lt;x&gt;
         </code>
         表示从视频左侧开始裁剪的像素数量，
         <code>
          &lt;y&gt;
         </code>
         表示从视频顶部开始裁剪的像素数量。
        </p>
        <p>
         例如，下面的命令将从输入的视频文件中裁剪出宽度为 500，高度为 500，左上角坐标为 (100,100) 的视频，并将裁剪后的视频输出为 output.mp4：
         <br/>
         <code>
          ffmpeg -i input.mp4 -vf crop=500:500:100:100 output.mp4
         </code>
        </p>
       </li>
       <li>
        <p>
         输入视频时使用滤镜：
         <code>
          -vf "crop=&lt;width&gt;:&lt;height&gt;:&lt;x&gt;:&lt;y&gt;"
         </code>
        </p>
        <p>
         例如，要对
         <strong>
          输入的视频裁剪掉右侧和底部各 100 像素
         </strong>
         ，可以输入以下命令：
        </p>
        <pre><code>ffmpeg -i input.mp4 -vf crop=in_w-100:in_h-100 output.mp4
</code></pre>
        <p>
         在这个命令中，
         <strong>
          <code>
           in_w
          </code>
          和
          <code>
           in_h
          </code>
          表示输入视频的宽度和高度。
         </strong>
        </p>
       </li>
      </ol>
      <blockquote>
       <p>
        注意，crop 滤镜也可以用于裁剪音频或图片。如果要裁剪视频中的特定部分，请先使用 FFmpeg 的 trim 滤镜裁剪为所需的范围，然后再使用 crop 滤镜裁剪到所需的大小。例如，
        <code>
         trim=start_frame=100:end_frame=200,crop=320:240:100:100
        </code>
        表示从视频的第 100 帧开始裁剪，裁剪到第 200 帧，然后将宽度裁剪为 320 像素，高度为 240 像素，裁剪掉左边 100 像素和顶部 100 像素。
       </p>
      </blockquote>
     </li>
    </ul>

    <h3>
     <a id="_175">
     </a>
     分解/复用命令
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8575552c29ab434b3d64b4915621f1c7.png"/>
    </p>
    <p>
     一个已经封装格式的音视频文件，例如mp4、flv格式。Demuxer 拆解，解封装之后拿到编码数据，decoder解码后拿到原始数据（高度还原的数据，不是真正的原始数据，真正的原始数据是相机或者麦克风手机的数据），然后再指定编码格式encoder编码，最后指定封装格式muxer就变成最后的输出文件了。
    </p>
    <p>
     分解复用就是对解封装后的编码数据操作。可以转格式，单独抽取音频或者视频。
    </p>
    <ul>
     <li>
      mp4 转 flv
      <br/>
      ffmpeg -i out/media.mp4 -vcodec copy -acodec copy media.flv
     </li>
     <li>
      mp4 转 mov
      <br/>
      ffmpeg -i out/media.mp4 -vcodec copy -acodec copy media.mov
     </li>
     <li>
      从flv中抽取视频
      <br/>
      ffmpeg -i out/media.flv -vcodec copy -an out/meida_video.h264
     </li>
     <li>
      从flv中抽取音频
      <br/>
      ffmpeg -i out/media.flv -acodec copy -vn out/meida_audio.aac
     </li>
     <li>
      aac 转 mp3
      <br/>
      ffmpeg -i out/meida_audio.aac -acodec mp3 out/meida_audio.mp3
     </li>
    </ul>
    <h3>
     <a id="_201">
     </a>
     处理原始数据命令
    </h3>
    <p>
     原始数据在这里指的就是FFmpeg解码后的数据，对于视频来说就是YUV数据，属于音频来说就PCM数据。
    </p>
    <h4>
     <a id="YUV_207">
     </a>
     提前YUV数据
    </h4>
    <p>
     <code>
      ffmpeg -i input.mp4 -an -c:v rewvideo -pix_fmt yuv420p out.yuv
     </code>
    </p>
    <p>
     既然是提取原始数据，所以要指定格式。
    </p>
    <h4>
     <a id="ffmpegPCM_215">
     </a>
     ffmpeg提取PCM数据
    </h4>
    <p>
     提取PCM命令如下：
    </p>
    <p>
     <code>
      ffmpeg -i out.mp4 -vn -ar 44100 -ac 2 -f sl6le out.pcm
     </code>
    </p>
    <p>
     当播放的时候可能会出现错误：
     <code>
      out.pcm: Invalid data found when processing nput
     </code>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8fcd0bd8c487268781f58e99ca3a01f5.png"/>
    </p>
    <p>
     因为你是原始的数据嘛，播放器不知道你采样频率，声道数，这些信息，所以会出错。
    </p>
    <p>
     生成pcm数据需要指定这些参数，同样播放pcm数据也需要告诉ffplay这些对应的参数。
    </p>
    <p>
     播放命令
    </p>
    <p>
     <code>
      ffplay -ar 44100 -ac 2 -f s16le out.pcm
     </code>
    </p>
    <h3>
     <a id="_234">
     </a>
     滤镜命令
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2ad25dd4e32e76ebbac5bb54abbf1809.png"/>
    </p>
    <p>
     滤镜使用的大致流程：拿到解码后的数据帧，过滤，拿到过滤后的数据，编码再输出。
    </p>
    <p>
     命令使用：
    </p>
    <pre><code class="prism language-shell">ffmpeg -i input.mp4 -vf <span class="token operator">&lt;</span>指定滤镜<span class="token operator">&gt;</span> -c:v libx264 -c:a copy out.mp4
</code></pre>
    <p>
     ffmpeg -i input.mp4 -vf crop=500:500:0:0 -c:v libx264 -c:a copy out.mp4
    </p>
    <p>
     意思就是输入input.mp4文件，裁剪出宽度为 500，高度为 500，左上角坐标为 (0,0) 的视频，视频编码指定为 libx264 ,音频编码保持不变。并将裁剪后的视频输出为 output.mp4
    </p>
    <h3>
     <a id="_251">
     </a>
     裁剪与合并命令
    </h3>
    <pre><code class="prism language-shell">ffmpeg -i input.mp4 -ss 00:00:00 -t <span class="token number">10</span> out.mp4
</code></pre>
    <p>
     这个命令意思是 从 input.mp4的0时0分0秒开始截取10秒的视频。
    </p>
    <pre><code>ffmpeg -f concat -i input.txt out.flv
</code></pre>
    <p>
     input.txt内容是文件列表，记录了要合并的文件。内容为"file fileName"形式。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3d599352724e6b974ac25a948a34d4d8.png"/>
    </p>
    <p>
     这条命令就是把t1.ts和t2.ts合并。
    </p>
    <h3>
     <a id="_271">
     </a>
     图片/视频互转命令
    </h3>
    <h4>
     <a id="_273">
     </a>
     视频转图片
    </h4>
    <pre><code class="prism language-shell">ffmpeg -i input.mp4 -r <span class="token number">1</span> -f image2 image-%3d.jpeg
</code></pre>
    <ul>
     <li>
      -r 设定帧速率，视频转换出图片的帧率，-r 1 表示每秒转出为1张图片。
     </li>
     <li>
      -f 告诉ffmeg 转出的格式。 image2是一种图片格式。
     </li>
     <li>
      image-%3d.jpeg 是生成图片的命名。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4226f785b307d07356984d8dabbe6b6f.png"/>
     </li>
    </ul>
    <h4>
     <a id="_287">
     </a>
     图片转视频
    </h4>
    <pre><code class="prism language-shell">ffmpeg -i image-%3d.jpeg out.mp4
</code></pre>

    <h3>
     <a id="ffprobe_311">
     </a>
     ffprobe
    </h3>
    <p>
     <code>
      ffprobe
     </code>
     是
     <code>
      ffmpeg
     </code>
     提供的三大工具之一，用来查看音视频文件的各种信息，比如：封装格式、音频/视频流信息、数据包信息等。
    </p>
    <p>
     查看音视频文件的封装格式
    </p>
    <pre><code class="prism language-text">ffprobe -show_format inputFile
</code></pre>
    <p>
     查看音视频文件的流信息
    </p>
    <pre><code class="prism language-text">ffprobe -show_streams inputFile
</code></pre>
