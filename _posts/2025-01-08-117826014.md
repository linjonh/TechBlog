     <span style="color:#373a3c;">
      按照以下步骤绘制具有所需颜色的填充形状-
     </span>
    </p>
    <ol>
     <li>
      通过调用
      <code>
       fillcolor()
      </code>
      函数选择填充颜色并以#RRGGBB 格式传递颜色名称或颜色。
     </li>
     <li>
      在第 1 步之后，您必须调用
      <code>
       begin_fill()
      </code>
      并使用 Turtle 函数开始绘图。完成绘图后，调用
      <code>
       end_fill()
      </code>
      函数以使用所选颜色填充绘制的图形。
     </li>
    </ol>
    <h3>
     <span style="color:#373a3c;">
      在 Python Turtle 中绘制颜色填充的正方形
     </span>
    </h3>
    <pre class="has"><code class="language-brush: language-python">#Python program to draw color filled square in turtle programming
import turtle

t = turtle.Turtle()
t.fillcolor('blue')
t.begin_fill()
for i in range(4):
  t.forward(150)
  t.right(90)
t.end_fill()</code></pre>
