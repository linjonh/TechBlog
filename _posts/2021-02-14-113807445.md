    <h4 id="4-扩展">
     4. 扩展
    </h4>
    <ul>
     <li>
      <strong>
       grep
      </strong>
      命令还可以文本搜索管道中的内容，比如:
      <strong>
       ls / | grep ‘lib’
       <span style="color:#86ca5e;">
        【ls /bin | grep 'sh】
       </span>
      </strong>
     </li>
     <li>
      在使用
      <strong>
       grep
      </strong>
      命令的时候还可以省略搜索内容的引号，比如:
      <strong>
       ls / | grep lib
      </strong>
      ，
      <strong>
       grep hello 1.txt
      </strong>
     </li>
    </ul>
