     <h3>

      Python True keyword

       )
      </span>
     </h3>
     <p>
      <b>
       True
      </b>
      is a keyword (case-sensitive) in python, it is a Boolean value (a value of class type
      <b>
       bool
      </b>
      ).
      <b>
       True
      </b>
      is the result of a comparison operation.
     </p>
     <p>
      <b class="raw_b_node">
       True
      </b>
      是python中的关键字(区分大小写)，它是一个布尔值(类类型

       bool的值
      </b>
      )。 比较操作的结果为
      <b class="raw_b_node">
       True
