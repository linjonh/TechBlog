    <p>
     最近在Windows 10系统上安装了Python, 然后想到前段时间自己还安装了Git Bash, Git Bash的Linux命令很好用, 就想尝试一下在Git Bash上使用Python虚拟环境.
    </p>
    <h3>
     <a id="Git_BashPython_2">
     </a>
     Git Bash中使用Python
    </h3>
    <p>
     如果只想使用Python, 不需要虚拟环境的话, 直接键入
     <code>
      winpty python
     </code>
     即可, 后面的内容都不用看了. 这里加
     <code>
      winpty
     </code>
     原因是这样可以支持交互式操作, 参考
     <a href="https://www.cnblogs.com/cute/p/10303499.html" rel="nofollow">
      这篇文章
     </a>
    </p>
    <h3>
     <a id="Git_BashPython_5">
     </a>
     Git Bash中使用Python虚拟环境
    </h3>
    <p>
     探索发现安装虚拟环境时已经自动生成了一个适用于Bash的启动器, 因此可以用如下方法在Git Bash中启动虚拟环境:
    </p>
    <p>
     打开Git Bash, 运行启动器(这里需要替换成你自己的路径):
     <br/>
     <code>
      source /d/myenvs/test/Scripts/actitvate
     </code>
    </p>
    <p>
     出现
     <code>
      (虚拟环境名)
     </code>
     就表示进入了虚拟环境, 这里(test)表示虚拟环境名是test:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7d58d86641ec4d2504d3bcdce91d9993.png">
      <br/>
      为使用方便, 为上述命令设置别名, 即, 将以下语句
      <br/>
      <code>
       alias env_test='source /d/myenvs/test/Scripts/actitvate'
      </code>
      <br/>
      加入到~/.bashrc 中, 现在只需要在Git Bash中键入
      <code>
       env_test
      </code>
      即可打开test虚拟环境.
     </img>
    </p>
    <p>
     要退出虚拟环境, 键入
     <code>
      deactivate
     </code>
     即可.
    </p>
    <p>
     若创建了多个虚拟环境, 为了更方便地切换不同的虚拟环境, 可以在新建一个shell脚本, 比如myenv.sh(笔者的路径是/d/myenvs/myenv.sh, 可自己选择路径), 在其中键入如下内容
    </p>
    <pre><code>#!/bin/bash
cd 工作路径
ENV_NAME=$1 #虚拟环境名称
echo "You are in the virtualvenv '$ENV_NAME'"
source /d/myenvs/$ENV_NAME/Scripts/activate
</code></pre>
    <p>
     再为调用这个脚本的过程设置一个别名
    </p>
    <pre><code>alias myenv='source /d/myenvs/myenv.sh'
</code></pre>
    <p>
     这样要使用某个虚拟环境, 只需在Git Bash中键入
    </p>
    <pre><code>myenv 环境名
</code></pre>
    <h3>
     <a id="Git_Bashipython_36">
     </a>
     Git Bash中使用ipython
    </h3>
    <p>
     要使用ipython只需直接输入
     <code>
      winpty ipython
     </code>
     即可:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e4c3c26b0570e7060210240b747da332.png">
      <br/>
      PS: 此处如果省略
      <code>
       winpty
      </code>
      只键入
      <code>
       ipython
      </code>
      也能使用ipython, 但此时退出ipython的快捷键
      <code>
       Ctrl+d
      </code>
      失效了, 所以加上winpty比较方便.
     </img>
    </p>
