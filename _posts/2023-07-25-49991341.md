    嵌入式系统管理(ESM)日志也称为系统事件日志(SEL)或硬件日志，它报告Dell PowerEdge服务器中的潜在硬件问题。—— 有时，只需关闭服务器并断开其与电源插座的连接即可消除错误消息。 等待大约10秒钟，然后重新连接电源电缆并重新启动服务器。
    <br/>
    <br/>
    有时，您将需要清除ESM日志来消除错误消息。 例如，错误代码
    <strong>
     SEL FULL
    </strong>
    表示您将需要清除ESM日志。 （您将丢失服务器事件历史记录。）有多种不同方法可以清除ESM日志。’
    <br/>
    <br/>
    <strong>
     要使用DSET来清除ESM日志
    </strong>
    <ol>
     <li>
      下载并打开
      <a class="atagcolor" href="http://support.dell.com/dset" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:blue">
        <u>
         DSET实用程序
        </u>
       </span>
      </a>
      。
     </li>
     <li>
      单击
      <strong>
       System
      </strong>
      （系统）旁边的加号(
      <strong>
       +
      </strong>
      )，然后单击
      <strong>
       Logs
      </strong>
      （日志）旁边的加号(
      <strong>
       +
      </strong>
      )。
     </li>
     <li>
      单击
      <strong>
       ESM
      </strong>
      选中它。
     </li>
     <li>
      单击位于页面右上角的
      <strong>
       Clear ESM Log
      </strong>
      （清除ESM日志）。
     </li>
    </ol>
    <p>
     <strong>
      要使用OpenManage Server Administrator (OMSA)来清除ESM日志
     </strong>
    </p>
    <ol>
     <li>
      启动
      <strong>
       OpenManage Server Administrator (OMSA)
      </strong>
      。
     </li>
     <li>
      输入您的用户名和密码，然后按
      <strong>
       Enter
      </strong>
      ，登录OMSA。
     </li>
     <li>
      依次单击
      <strong>
       Logs
      </strong>
      （日志）选项卡和
      <strong>
       Hardware
      </strong>
      （硬件）。
     </li>
     <li>
      单击Options（“选项”）旁边的
      <strong>
       Clear Log
      </strong>
      （清除日志）。
     </li>
    </ol>
    <p>
     <strong>
      要使用Dell IT Assistant来清除ESM日志
     </strong>
    </p>
    <ol>
     <li>
      启动
      <strong>
       IT Assistant
      </strong>
      。
     </li>
     <li>
      从中间的System Tree（“系统树”）栏，单击选中您的服务器。
     </li>
     <li>
      单击
      <strong>
       Logs
      </strong>
      （日志）选项卡，然后单击位于页面右上角的
      <strong>
       Clear Log
      </strong>
      （清除日志）。
     </li>
    </ol>
