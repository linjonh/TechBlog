    </h1>

  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     要在本地启动前后端一体的项目，包括前端 Vue 应用和后端 Spring Boot 应用，您需要分别启动两个应用，前端和后端。
    </p>
    <p>
     下面是一般的步骤：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        启动后端 Spring Boot 应用
       </strong>
       ：
      </p>
      <ul>
       <li>
        在 IntelliJ IDEA 中打开您的 Spring Boot 项目。
       </li>
       <li>
        定位到 Spring Boot 应用的主类，通常是带有
        <code>
         public static void main(String[] args)
        </code>
        方法的类。
       </li>
       <li>
        右键点击该类，然后选择 "Run"。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        启动前端 Vue 应用
       </strong>
       ：
      </p>
      <ul>
       <li>
        打开命令行（如 Windows 的 cmd 或 PowerShell）。
       </li>
       <li>
        使用
        <code>
         cd
        </code>
        命令切换到您的前端 Vue 项目的目录，通常是包含
        <code>
         package.json
        </code>
        文件的目录。
       </li>
       <li>
        在命令行中运行
        <code>
         npm install
        </code>
        安装依赖。
       </li>
       <li>
        运行
        <code>
         npm run serve
        </code>
        启动 Vue 开发服务器。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     现在，您的前端和后端应用都应该在本地运行了。通常，前端应用会运行在
     <code>
      http://localhost:8080
     </code>
     上，而后端应用会运行在另一个端口（如
     <code>
      http://localhost:8081
     </code>
     ）上。
    </p>
    <p>
     确保前端应用可以正确地发起请求到后端，通常需要在前端代码中配置后端的 API 地址。
    </p>
    <p>
     在实际项目中，前端和后端的启动方式可能会有所不同，具体要根据您的项目结构和构建配置来确定启动方式。
    </p>
