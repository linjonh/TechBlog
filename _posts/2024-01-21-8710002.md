   <div class="htmledit_views" id="content_views">
    <p>
     在学习Python服务开发的过程中，我想找一个比较方便的Python云平台。
    </p>
    <p>
     在寻找的过程中，我尝试了
     <a href="http://sae.sina.com.cn/" rel="nofollow">
      SAE（新浪云平台）
     </a>
     ，被收费政策弄晕了，不能通过开发者审核，放弃了。
    </p>
    <p>
     尝试
     <a href="https://www.openshift.com/" rel="nofollow">
      OpenShift
     </a>
     ，这个平台可以创建3个免费的应用，Python的功能限制少，使用Git进行代码管理，Https进行数据传输，这些都比较好，不过因为它是国外的主机，速度上有点慢。
    </p>
    <p>
     最后找到了
     <a href="http://developer.baidu.com/bae" rel="nofollow">
      BAE（百度云平台）
     </a>
     ，有如下优点：
    </p>
    <ul>
     <li>
      使用容易：比SAE的这个平台通过开发者认证容易
     </li>
     <li>
      创建的应用比较多：首先可以创建10个应用，更多的应用可以再申请
     </li>
     <li>
      <a href="http://developer.baidu.com/dev#/price/charge" rel="nofollow">
       免费额度比较大
      </a>
      ：对于个人学习用途来说足够了
     </li>
     <li>
      国内访问速度快（SAE的访问速度也快），OpenShift的访问速度比较慢。
      <br/>
     </li>
    </ul>
    <p>
     缺点：
    </p>
    <ul>
     <li>
      大型应用就要付费了
     </li>
     <li>
      <a href="http://developer.baidu.com/wiki/index.php?title=docs/cplat/rt/python" rel="nofollow">
       Python
      </a>
      只使用2.7.2，而且有限制，一些模块和功能都受限。
     </li>
    </ul>
    <br/>
