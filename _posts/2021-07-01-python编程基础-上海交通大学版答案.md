    <blockquote>

     <p>
      【单选题】下列代码运行结果是?
      <br/>
      a = ‘a’
      <br/>
      print (a &gt; ‘b’ or ‘c’)
      <br/>
      A. a B. b C. c D. Y
      <br/>
      参考答案：c
      <br/>
      【答案解析】
      <br/>
      英文字母对应的Unicode编码
      <br/>
      A~Z 65~90
      <br/>
      a~z 97~122
      <br/>
      对python而言
      <br/>
      其一, 在不加括号时候, and优先级大于or
      <br/>
      其二, x or y 的值只可能是x或y. x为真就是x, x为假就是y
      <br/>
      第三, x and y 的值只可能是x或y. x为真就是y, x为假就是x
      <br/>
      此题里a &gt; “b"为假，所以返回"c”
     </p>
    </blockquote>
