     今年是 TechEd 进入中国的第 20 个年头，也是微软 Cloud OS 云操作系统及云计算平台突飞猛进的一年。会议期间，微软向各界来宾介绍了以云操作系统为核心的解决方案、技术、产品和服务，为众多开发者和 IT 专业人士创造了学习交流、经验分享和探索尖端技术的机会。微软全球资深副总裁、大中华区董事长兼首席执行官贺乐赋（Ralph Haupter）指出：“在过去的 20 年里，微软携手中国开发者、IT 专业人士、合作伙伴和广大用户，立足推动整个 IT 产业的发展和变革。云、大数据、移动、社交、现代业务应用，以及构建以人为本的 IT 构成了微软的核心竞争力，推动中国生态系统的可持续性发展。”

     更重要的是，微软 Cloud OS 云操作系统为广大开发者提供了更加广阔的发展平台。一方面，Cloud OS 提供了以 Windows 为基础的熟悉界面和统一的管理方式；而 Windows Azure 则同时支持 Windows 和 Linux 虚拟机，除了常见的 Java、.NET 之外，还支持 Python、Node.js、PHP 等大量开源工具，广泛的开放性让更多开发者能够轻松进入微软云平台。另一方面，Windows Azure 提供按需付费的计费体系和企业级软硬件服务，这意味着即使是创业团队乃至个人开发者，只需支付有限的费用，便能享有和大企业一致的一流服务，大大增强了初创企业和互联网企业在云计算时代的竞争实力。
