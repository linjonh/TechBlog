        <p style="margin:10px auto">
         1.弱智儿童欢乐多游戏源码完整版
         <br/>
         2  项目概要：弱智儿童欢乐多，这就是一款弱智的游戏……回归正题，这是一款仿IOS上TripTrap的关卡解锁模式的益智休闲游戏，代码中实际只有一关游戏，但是可以体验解锁，第一关和第二关都是一样的游戏场景。
         <br/>
         3  项目问题：虽然上传了文档，但不是参考，而是因为存在很多问题，欢迎大家指导；代码也很渣，欢迎吐槽；没有做适屏，所以打包的APK如果安装在大家的机子里maybe会混乱，我的机子是魅族MX3；微博分享功能是直接用的网上下载的例子，自己原理也没有弄清楚，所以这块有点乱；部分代码还在扫尾，比如退出还没有做所有退出。
         <br/>
         4  游戏功能：
         <br/>
         1. 关卡选择
         <br/>
         2. 游戏开始、暂停、重新开始、退出
         <br/>
         3. 游戏胜利和失败判定
         <br/>
         4. 评级功能（根据游戏剩余时间）
         <br/>
         6. 游戏重置设置（除第一关，其他关卡重新锁起）
         <br/>
         7. 游戏社交分享（腾讯微博）
         <br/>
         8. 观看故事背景（玩家进入第一关时自动播放）
         <br/>
         5 游戏玩法：
         <br/>
         通过点击跳转
         <a class="relatedlink" href="http://code.662p.com/list/2926_1.html" rel="nofollow noopener noreferrer" style="color:rgb(29,88,209); text-decoration:none" target="_blank">
          按钮
         </a>
         控制蜜蜂在不同草地间的跳转“采蜜”。（限定跳转区域和只能上下跳转）
         <br/>
         游戏主原型图如下（蜜蜂绕行到标示红色区域时，玩家点击
         <a class="relatedlink" href="http://code.662p.com/list/2926_1.html" rel="nofollow noopener noreferrer" style="color:rgb(29,88,209); text-decoration:none" target="_blank">
          Button
         </a>
         ，
         <br/>
         蜜蜂则会从一个草地飞跃到另一个草地（只允许上下跳转））

          6 游戏规则：
          <br/>
          主角在30s内吃掉所有得分道具且进入蜂巢时，游戏胜利（解锁下一关）
          <br/>
          障碍物同主角碰撞时，游戏失败
          <br/>
          游戏倒计时到达0s时，游戏失败
          <br/>
          游戏胜利时倒计时时间t在20-30s时，游戏评级为三颗星
          <br/>
          游戏胜利时倒计时时间t在10-20s时，游戏评级为两颗星
          <br/>
          游戏胜利时倒计时时间t在0-10s时，游戏评级为一颗星
