    <h2>
     <a id="_0">
     </a>
     脚本是一种计算机程序，通常用于自动化执行任务、设置操作系统或应用程序的配置选项等。
    </h2>
    <p>
     脚本编程语言（比如python、javascript、bash等）可以使编写脚本变得更加简单易懂，并且可以跨平台运行。脚本的优点在于它们可以很容易地修改和调试，同时还可以提高效率，减少人工操作的成本。常见的使用脚本的场景包括批量处理文件、数据分析、web开发自动化等。
    </p>
    <h2>
     <a id="_javascript__js_4">
     </a>
     浏览器的脚本文件可以理解为在网页上运行的代码，它通常以 javascript 文件的形式出现，后缀名为
     <code>
      .js
     </code>
     。
    </h2>
    <p>
     javascript 是一种基于对象和事件驱动的脚本语言，广泛应用于前端开发中。开发者可以使用 javascript 通过 dom 操作、事件绑定等方式实现与用户的交互，使得网页内容变得更加生动、丰富。
    </p>
    <p>
     除了 javascript 之外，浏览器也支持其他类型的脚本语言，例如 vbscript 和 actionscript 等。但由于 javascript 的兼容性和易学性较强，因此 javascript 成为了最广泛使用的浏览器脚本语言。
    </p>
