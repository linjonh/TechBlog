     本程序（闪拍系统），仅提供学习、研究使用，不得用于除此之外的其他用途。
     <br/>
     如需帮助
    </p>
    <p>
     服务器安装要求
     <br/>
     安装之前需要配置服务器安装环境，服务器环境要求：IIS/Apache/Nginx + PHP&gt;=5.4 + MySQL&gt;=5.5-5.7
     <br/>
     Web服务器：IIS、Apache或者Nginx都可以；
     <br/>
     PHP：PHP 5.4及以上版本；
     <br/>
     MySQL：MySQL 5及以上版本(建议5.5-5.7）
     <br/>
     openssl 、 fileinfo、Sg11，ionCube、Redis、Swoole扩展
     <br/>
     注意：如果是阿里云服务器，记得配置安全组开放8888端口、888端口、80端口、443端口、20端口和21端口这五个端口，
     <br/>
     参考：阿里云ECS怎么开放端口自定义安全组规则，不是阿里云服务器可以忽略。
     <br/>
     安装宝塔面板
     <br/>
     安装宝塔面板之前，确保你的系统是纯净的，如果之前使用其他的一键安装，建议重装系统为Centos7系统
     <br/>
     首先我们要进入自己的linux服务器，配带linux系统的远程软件，打开输入你的ip，其他的默认即可，点击连接即可，进入界面第一个会让你输入用户名，linux系统默认是root，输入完按回车会让你输入密码，注意：这里输入的密码你是看不见的，请不要觉得没有输对之类，输入完了回车，密码错误会让你重新输入。
     <br/>
     一进去可能你看不懂，什么都不用管，复制以下命令，在命令行输入以下命令安装宝塔面板，回车即可。会自动安装，等一会会出现一个y/n 输入y即可，回车
     <br/>
     yum install -y wget &amp;&amp; wget -O install.sh http://download.bt.cn/install/install_6.0.sh &amp;&amp; bash install.sh
     <br/>
     需要几分钟的时间，中途会停顿一下让选择web目录的位置，一般默认即可
     <br/>
     等待安装完成之后，命令行会显示面板的登录地址、账号以及密码，一定将这些信息保存下来，后期的管理都需要使用该信息登录面板
     <br/>
     安装网站环境
     <br/>
     宝塔面板安装完成后，我们就可以登录后台面板了，使用刚刚记录的后台地址登录(如果你是使用的阿里云或者腾讯云的话，可能会登录不上，这时候你需要在后台开启安全组8888端口，阿里云ECS开放安全组端口)
     <br/>
     接下来就会看到一键安装环境的界面，一般我们会选择左侧的LNMP架构，编译安装
     <br/>
     这里我要说下关于不同配置的服务器环境的设置，因为一般常用的有512M内存的，还有就是1G内存的，这两个的环境选择是不一样的
     <br/>
     如果是512M内存，那么使用默认的配置即可，Mysql 5.5 和 PHP 5.4
     <br/>
     如果是1G内存，推荐使用Mysql 5.6 和 PHP 5.6
    </p>
    <p>
     然后就是等待安装完成就可以了，一般会持续一段时间，根据你服务器的性能决定
     <br/>
     服务器新建站点
     <br/>
     等待所有的任务都完成以后，所有的环境就搭建好了，接下来就可以建立网站了
     <br/>
     宝塔面板新建一个网站，点击下图中绿色的按钮新建一个站点
    </p>
    <p>
     这里我们需要输入自己的网站域名，把带www和不带www的域名全部填写上，比如www.baidu.com，该步骤就是绑定域名
     <br/>
     然后FTP和数据库最好都创建，将这些信息都记录下来，以后我们都需要用到的
    </p>
    <p>
     所有设置完毕之后我们点击提交，网站即创建完毕(这里指服务器为网站准备出空间，类似开一个虚拟主机，只不过我们对此有控制权)
     <br/>
     域名解析到站点
     <br/>
     这个其实很简单，就是添加域名解析的ip地址，可能就是不同的域名商操作方法不一样，这里我演示下阿里云的域名解析
     <br/>
     我们先在后台管理中找到自己的域名，点击解析，然后添加解析
    </p>
    <p>
     这里需要添加2个解析到服务器的ip地址
     <br/>
     一个是 www 记录，这里设置www通用的，比如 www.baidu.com
    </p>
    <p>
     还有就是 @ 记录，这个是设置根域名，比如说主机笔记就是 baidu.com，虽然说有时候用不到，最好还是设置一下，毕竟还是有很多人直接输入域名不输入www的
    </p>
    <p>
     设置好解析以后，需要等待2分钟(其他的域名商可能会有延迟几个小时都有可能的，最好提前解析)
     <br/>
     然后访问网页看到一个创建成功的页面，就证明解析已经好了，可以进行下一步
     <br/>
     上传网站
    </p>
    <p>
     右上角上传点击一下，网站文件上传完成。
    </p>
    <p>
     环境搭建好了，网站上传之后就是根据每个网站的规则进行配置
    </p>
    <p>
     接下来，上传“微擎框架+后端文件”一体包，直接访问域名安装即可
    </p>
