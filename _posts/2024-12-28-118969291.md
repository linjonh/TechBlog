   <div class="htmledit_views" id="content_views">
    <p id="main-toc">
     1、IDEA（
     <a href="https://www.jetbrains.com/idea/" rel="nofollow">
      下载
     </a>
     ）
    </p>
    <ul>
     <li>
      在IDEA中，点击【File】-【Settings...】-【Plugins】，在里面搜索python，并安装插件。（安装后需要重启有效）
     </li>
     <li>
      在IDEA中，配置python SDK的路径
     </li>
    </ul>
    <p>
     【File】–【Project Structure】–【Platform Settings】–【SDKs】–【Python 】 –【Classpath 】
    </p>
    <p>
     <img alt="" height="1040" src="https://i-blog.csdnimg.cn/blog_migrate/439d0c35ab5e0f9ce9b5c21f44caf8ae.png" width="1200"/>
    </p>
    <ul>
     <li>
      （清除红色下划线）
     </li>
    </ul>
    <p>
     【Modules】-【Dependencies】-【New...】-【Python SDK】
    </p>
    <p>
     <img alt="" height="731" src="https://i-blog.csdnimg.cn/blog_migrate/e9d80bf7efc2ad341ac3ca65cd9cfb24.png" width="1026"/>
    </p>
    <p>
    </p>
    <p>
     2、Python（
     <a href="https://www.python.org/" rel="nofollow">
      下载
     </a>
     ）
    </p>
    <p>
     3、Anconda（
     <a href="https://www.anaconda.com/products/individual" rel="nofollow">
      下载
     </a>
     ）
    </p>
    <ul>
     <li>
      将Anconda安装目录添加到环境变量中
     </li>
    </ul>
    <p>
     右击【此电脑】–【属性】–【高级系统设置】–【环境变量】–【系统变量】–【Path】-【编辑】-【新增】
    </p>
    <pre><code>D:\Anaconda3
D:\Anaconda3\Scripts
D:\Library\bin</code></pre>
    <ul>
     <li>
      配置（pip升级到最新版）
     </li>
    </ul>
    <p>
     在Anaconda的安装目录下找到Scripts文件夹并进入，然后shift右键，选择在此处打开Powershell窗口，输入以下命令：
    </p>
    <pre><code>python -m pip install -U pip</code></pre>
    <pre><code class="language-python">pip install numpy</code></pre>
    <p>
     4、下载库
    </p>
    <p>
     <em>
      <strong>
       方法一：
      </strong>
     </em>
    </p>
    <ul>
     <li>
      requests：获取网页源代码
     </li>
     <li>
      lxml：得到网页源代码中的指定数据
     </li>
    </ul>
    <p>
     新建一个.py文件，其中输入：
    </p>
    <pre><code class="language-python">import requests
import lxml</code></pre>
    <p>
     此时，
     <strong>
      requests
     </strong>
     会出现红线。
    </p>
    <p>
     解决：将光标对准
     <strong>
      requests
     </strong>
     ，按快捷键：
     <strong>
      alt + enter
     </strong>
     ，IDEA会给出解决方法，选择
     <strong>
      install package requests
     </strong>
     ，IDEA就会自动安装。lxml的安装方式同理。
    </p>
    <p>
     将这两个库安装完成后，编译器就不会报红线。
    </p>
    <p>
     <em>
      <strong>
       方法二：
      </strong>
     </em>
    </p>
    <p>
     <img alt="" height="731" src="https://i-blog.csdnimg.cn/blog_migrate/6facf88fbc9e2bfb97360eb8c1f04b65.png" width="1026"/>
    </p>
    <p>
    </p>
    <p>
     至此，环境搭建完成，可以开始在IDEA中写Python了！
     <br/>
    </p>
