    <p style="margin-left:0;">
     微软基于角色的证书和技术专项证书都有规定有效期限。在2021年6月之前所获证书有效期是2年。6月起所获证书有效期为1年。在临近有效期6个月内，学员会收到提醒更新的邮件。
    </p>
    <p style="margin-left:0;">
     不少小伙伴已经收到了微软证书的更新邮件，仅需几步就可您的微软证书续期，而且无需任何费用。
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       如果认证过期而没有续期，则必须再次通过所需的考试才能获得认证
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:0;">
     分享一下微软证书更新的分步指南。
    </p>
    <p style="margin-left:0;">
     第一步，查看自己所获证书是否需要更新。以微软Azure管理员助理证书为例，打开认证页面。
     <a href="https://docs.microsoft.com/learn/certifications/azure-administrator/" rel="nofollow" title="Microsoft Certified: Azure Administrator Associate">
      Microsoft Certified: Azure Administrator Associate
     </a>
     <u>
     </u>
    </p>
    <p style="margin-left:0;">
     第二步：在认证页面最下方查看“详细了解续期”。
    </p>
    <p style="margin-left:0;">
     <img alt="" height="232" src="https://i-blog.csdnimg.cn/blog_migrate/aac65858b13090d065e80b34dd4688ce.png" width="1042"/>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     如果您的证书即将过期，微软也会主动通过邮件提醒续期。在续期邮件中也可以访问此链接。以下是续期邮件。
    </p>
    <p style="margin-left:0;">
     <img alt="" height="733" src="https://i-blog.csdnimg.cn/blog_migrate/29e20a5cdbd17f03f239c9f25029c2a0.png" width="677"/>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     第三步：转到续期详情页面，连接认证的个人资料已查看是否需要续期。如果您所持有的高级角色证书或者专项技能证书的有效期小于6个月，您就可以续期。
    </p>
    <p style="margin-left:0;">
     <img alt="" height="745" src="https://i-blog.csdnimg.cn/blog_migrate/938547e1f74a00fabdcad6bb724856eb.png" width="1102"/>
    </p>
    <p style="margin-left:0;">
     在个人的dashboard中可以查看是否有需证书需要续期，例如下面的提示。
    </p>
    <p style="margin-left:0;">
     <img alt="" height="741" src="https://i-blog.csdnimg.cn/blog_migrate/9d44ca7241a880a1c410be5ab5ed2c64.png" width="1200"/>
    </p>
    <p style="margin-left:0;">
     第四步：如果满足续期的时间条件，就可以通过完成在线评估题目来更新证书。在证书更新页面可看到“为续期评估做好准备”，这里列示了续期评估所需要补充学习的模块和路径。可以免费在线学习。学完后再做评估题。
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     <img alt="" height="725" src="https://i-blog.csdnimg.cn/blog_migrate/d8e7923b6c0b73d562a9903416486f07.png" width="830"/>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     第五步：通过续期邮件中的链接可以转到Microsoft Learn上的在线测试评估题。当您通过在线测试后，您的证书有效期会在原来的有效期基础上递延一年。
    </p>
    <p style="margin-left:0;">
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       如果第一次没有通过，可以立即再次尝试评估。
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       如果第二次也没通过，必须至少等待
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       24
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       小时才能进行任何其他尝试
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:0;">
     <img alt="" height="756" src="https://i-blog.csdnimg.cn/blog_migrate/b1ab3cb2860b4403ef599360c37bb930.png" width="1200"/>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     关于微软证书的有效期政策，可查看官方文档。
    </p>
    <p style="margin-left:0;">
     <a href="https://docs.microsoft.com/zh-cn/learn/certifications/renew-your-microsoft-certification?WT.mc_id=techcom_learn-blog-wwl" rel="nofollow" title="更新您的Microsoft认证 | Microsoft Docs">
      更新您的Microsoft认证 | Microsoft Docs
     </a>
    </p>
    <p style="margin-left:0;">
     <a href="https://docs.microsoft.com/zh-cn/learn/certifications/certification-expiration-policy" rel="nofollow" title="认证过期策略">
      认证过期策略
     </a>
    </p>
    <p style="margin-left:0;">
    </p>
    <p style="margin-left:0;">
     需要注意的是，Microsoft的基础（Fundamental）证书不会过期，

      <span style="color:#171717;">
       MOS

     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       、

     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       MTA
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       、
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       MCSA
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       、
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       MCSD
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       、
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       MCSE
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       和
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       MCE
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       认证不会过期
      </span>
     </span>
     <span style="background-color:#FFFFFF;">
      <span style="color:#171717;">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:0;">
     赶紧查看微软的认证面板，检查认证的有效期；一旦收到证书更新邮件，请尽快安排，以免延误续期。
    </p>
