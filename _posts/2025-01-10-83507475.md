     宇宙第一 IDE Visual Studio 开始支持 Java 了

       程序员的那些事
      </span>
      <span style="font-family:'-apple-system-font', 'system-ui', 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;letter-spacing:.544px;text-align:center;font-size:14px;color:rgb(255,41,65);line-height:22.4px;">
       加星标
      </span>
      <span style="font-family:'-apple-system-font', 'system-ui', 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;letter-spacing:.544px;text-align:center;color:rgb(255,41,65);font-size:14px;line-height:22.4px;">
       ）
      </span>
     </p>
     <p>
      <br/>
     </p>
     <blockquote>
      <p>
       <span style="font-size:14px;">
        综合自：CSDN/唐小引、IT之家
       </span>
      </p>
     </blockquote>
     <p>
      <br/>
     </p>
     <p>
      <span style="font-size:15px;">
       近日在 2018 微软技术暨生态大会上，微软宣布和介绍了 Visual Studio 和 Visual Studio Code 的一系列新特性，并对 Azure DevOps 研发云进行了整合升级，Visual Studio 与 Azure 云服务的关系将越来越密切。
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p style="text-align:center;">
      <span style="font-size:15px;">
       <img alt="640?wx_fmt=jpeg" height="546" src="https://i-blog.csdnimg.cn/blog_migrate/355b476a61198cda287c7c8ab6a98f8e.jpeg" style="color:rgb(51,51,51);font-family:'-apple-system-font', 'system-ui', 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;border-width:0px;vertical-align:bottom;background:rgb(231,231,231) none no-repeat center;" title="微软Visual Studio发布重磅特性：协作和智能辅助编程首支持Java/Python" width="728"/>
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p>
      <span style="font-size:15px;">
       微软全球资深副总裁、开发平台事业部总经理潘正磊（Julia Liuson）带来了一项颇让程序员震惊且兴奋的新服务 —— Visual Studio Live Share。
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       该服务不仅能够让开发者编程协作、跨终端共同调试，跨 Windows、macOS、Linux 等不同平台地协作，以及跨 Visual Studio 及 VS Code 地写代码，更重要的是，可以直接写  Java ！
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       至此，素有「宇宙第一 IDE」称号的 Visual Studio 终于开始支持 Java 了！！！
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <span style="font-size:15px;">
        潘正磊说道：
       </span>
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <span style="font-size:15px;">
        <br/>
       </span>
      </span>
     </p>
     <blockquote>
      <p>
       <span style="font-size:14px;">
        “我做 Visual Studio 已经很多年了，经常会被问到一个问题，那就是 Visual Studio 什么时候能支持 Java？
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        今天大家第一次先睹为快。Visual Studio Live Share 其中包含了几个相当重要的特性，
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        首先便是上述在不同的操作系统上分别使用 Visual Studio 和 VS Code 来协作；
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        其次，通过 Live Share  可以任意调用项目中的其他文件，它不是单纯的文件共享，我们可以使用 Go to Definition 来进行代码跳转；
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        更重要的是，它使用同一个 Debug Session，在多个不同的设备里，可实时显示变量值的变更、进行调试；
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        最令人激动的是，它支持多种语言，不论你使用何种语言，你都可以使用它来进行团队合作。
       </span>
      </p>
      <p>
       <br/>
      </p>
      <p>
       <span style="font-size:14px;">
        今天我们在此演示的是 Java，但对 Python、C# 等语言都有非常强大的支持，因此这个功能并不局限于某种语言。”
       </span>
      </p>
     </blockquote>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       在大会现场，微软的两位工程师分别在不同的操作系统上使用 Visual Studio 2017 和 Visual Studio Code 进行了 Visual Studio Live Share 的操作演示，这项功能不仅能够帮助研发团队实时交流，还可以协调开发者同时编写、更改、调试代码。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p>
      <span style="font-size:15px;">
       具体表现为：
      </span>
     </p>
     <p>
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <blockquote>
      <p>
       <span style="font-size:14px;">
        在 VS Code 中打开你的 Java 项目，点击状态栏上的分享，Visual Studio Live Share 会帮助你创建一个分享链接，将链接发给你的同事。当你同事打开链接的时候，它会自动关联到他的 Visual Studio，就此，你的同事就能和你共享你的代码，并且可以自由地浏览其他的项目源文件。不仅如此，你的同事，并不需要安装任何的 Java JDK 运行环境。
       </span>
      </p>
     </blockquote>
     <p>
      <span style="font-size:15px;">
