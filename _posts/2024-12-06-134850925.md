    <h4>
     <a id="1_2">
     </a>
     1、学硕
    </h4>
    <p>
     网络空间安全
     <strong>
      学硕
     </strong>
     一级学科专业代码为
     <strong>
      0839
     </strong>
     ，官方认定的二级学科也
     <strong>
      只有
     </strong>
     网络空间安全一个。除长春大学开设了**自设特色专业“网络文化安全”**外，再无其他二级学科。
    </p>
    <p>
     <strong>
      网络空间安全（0839）考研科目：
     </strong>
    </p>
    <ul>
     <li>
      (101)思想政治理论；
     </li>
     <li>
      (201)英语（一）；
     </li>
     <li>
      (301)数学（一）；
     </li>
     <li>
      网络空间安全相关专业课。
     </li>
    </ul>
    <p>
     <a href="https://mp.weixin.qq.com/s/V0gClqEmXNrVlnd7tXRJNQ" rel="nofollow">
      CSDN大礼包：《黑客&amp;网络安全入门&amp;进阶学习资源包》免费分享
     </a>
    </p>
    <p>
     其中，网络空间安全相关专业课：
    </p>
    <ul>
     <li>
      <strong>
       （1）全国统考科目
      </strong>
      ：
      <strong>
       (408)计算机学科专业基础
      </strong>
      ，含
      <strong>
       数据结构
      </strong>
      （45分，占比30%）、计算机组成原理（45分，占比30%）、
      <strong>
       操作系统
      </strong>
      （35分，占比23%）和
      <strong>
       计算机网络
      </strong>
      （25分，占比17%）四门课，据说整体难度很大。
     </li>
     <li>
      **（2）院校自主命题科目：**除408外的其他专业课，如清华大学的(912)计算机专业基础综合、北京交通大学的(906)计算机专业基础、南京航空航天大学的(829)计算机专业基础等。
     </li>
    </ul>
    <p>
     总体来说，网络空间安全学硕考自主命题科目的院校，
     <strong>
      多于
     </strong>
     统考(408)计算机学科专业基础的院校。暂
     <strong>
      未看到
     </strong>
     有不考数一，考数二的学校。
    </p>
    <h4>
     <a id="2_24">
     </a>
     2、专硕
    </h4>
    <p>
     <strong>
      专硕
     </strong>
     在一级学科**电子信息（专业代码：0854）**下，二级学科参见下图。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/297a33d2171c5d5ba30f6f1761c2d261.jpeg"/>
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/297a33d2171c5d5ba30f6f1761c2d261.jpeg"/>
    </p>
    <p>
     <strong>
      网络与信息安全专硕（085412）考研科目：
     </strong>
    </p>
    <ul>
     <li>
      (101)思想政治理论；
     </li>
     <li>
      (201)英语（一）或(204)英语（二）；
     </li>
     <li>
      (301)数学（一）或(302)数学（二）；
     </li>
     <li>
      网络与信息安全相关专业课。
     </li>
    </ul>
    <p>
     其中，网络与信息安全相关专业课也有全国统考科目（408）和院校自主命题科目之分。在考试科目上，
     <strong>
      与学硕基本一致
     </strong>
     。但网络与信息安全专硕考408的院校
     <strong>
      很少
     </strong>
     。
    </p>
    <p>
     <strong>
      较多
     </strong>
     院校网络安全的学硕、专硕
     <strong>
      考的科目是完全一样
     </strong>
     的。
    </p>

    <h3>
     <a id="_68">
     </a>
     二、建议
    </h3>
    <p>
     1、选择
     <strong>
      去高校、走学术道路
     </strong>
     的，考研读研是必经之路。
    </p>
    <p>
     2、选择
     <strong>
      求稳进体制内
     </strong>
     的，部分专业限硕士研究生以上学历报考，特别是
     <strong>
      专业技术类岗位
     </strong>
     。意向是上述岗位的，必须读研。
     <strong>
      选调、人才引进
     </strong>
     等途径可以缩小竞争范围，将竞争对手的范围缩小到高学历人才，甚至应届毕业生，
     <strong>
      需要读研
     </strong>
     。
    </p>
    <p>
     3、不同央企、国企对学历的
     <strong>
      要求不同
     </strong>
     ，需要进一步了解。一般来说，
     <strong>
      科研院所
     </strong>
     等国企单位对学历的要求很高，没有研究生学历基本
     <strong>
      难以
     </strong>
     拿到正式工编制。
    </p>
    <p>
     4、打算去私企就业的，
     <strong>
      本科985/211学校
     </strong>
     毕业，或者已经
     <strong>
      收到大厂offer
     </strong>
     的双非院校学生，
     <strong>
      可以不读研
     </strong>
     。准备去
     <strong>
      一线城市、新一线城市
     </strong>
     工作，本科就读于
     <strong>
      不是
     </strong>
     以计算机类专业为优势学科的双非院校的，并且本科阶段
     <strong>
      没有特别出彩的履历
     </strong>
     ，如实习经历、项目经历的，建议
     <strong>
      读研
     </strong>
     。
    </p>
    <blockquote>
     <p>
      我单位人力资源系统的提供方，对接我单位的2个员工，都是计算机类专业二本院校毕业的。问起他们薪资，
      <strong>
       每个月6k+
      </strong>
      。还是在新一线城市，不包吃住，基本存不下钱。
     </p>
    </blockquote>
    <p>
     5、网络空间安全专业读研，建议
     <strong>
      学校头衔和专业排名
     </strong>
     并重。重要程度上，先院校头衔（
     <strong>
      如985、211、双非）
     </strong>
     ，后专业排名**（A、B、C各类学科评级）**。
    </p>
    <p>
     6、很多院校的网络空间安全专业，跟计科、软工专业
     <strong>
      考一模一样的科目
     </strong>
     ，考试内容难度上是一样的。因此，可以考虑
     <strong>
      是否转战
     </strong>
     计科、软工专业。
    </p>
