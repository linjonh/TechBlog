    <h2 id="1%E3%80%81%E5%B0%86%E9%9B%86%E5%90%88list%E8%BD%AC%E6%8D%A2%E6%88%90string">
     1、将集合list转换成string
    </h2>
    <p>
     将一个Java集合List转换为String很简单，可以使用String.join()或StringBuilder等方法。下面是几个示例：
    </p>
    <h3 id="1.1%E3%80%81%E4%BD%BF%E7%94%A8String.join()%E6%96%B9%E6%B3%95%EF%BC%9A">
     1.1、使用String.join()方法：
    </h3>
    <pre><code class="language-java">List&lt;String&gt; list = new ArrayList&lt;&gt;();
list.add("A");
list.add("B");
list.add("C");
String joinedString = String.join(", ", list);
System.out.println(joinedString);  // 输出: A, B, C</code></pre>
    <h3 id="%C2%A01.2%E3%80%81%E4%BD%BF%E7%94%A8StringBuilder%EF%BC%9A">
     1.2、使用StringBuilder：
    </h3>
    <pre><code class="language-java">List&lt;String&gt; list = new ArrayList&lt;&gt;();
list.add("A");
list.add("B");
list.add("C");
StringBuilder sb = new StringBuilder();
for (String str : list) {
    sb.append(str);
    sb.append(", ");
}
sb.setLength(sb.length() - 2);  // 删除最后一个逗号和空格
String joinedString = sb.toString();
System.out.println(joinedString);  // 输出: A, B, C</code></pre>
    <h3 id="%C2%A01.3%E3%80%81%E4%BD%BF%E7%94%A8Java%208%E7%9A%84Stream%20API%20(%E6%8E%A8%E8%8D%90)%EF%BC%9A">
     1.3、使用Java 8的Stream API (推荐)：
    </h3>
    <pre><code class="language-java">List&lt;String&gt; list = new ArrayList&lt;&gt;();
list.add("A");
list.add("B");
list.add("C");
String joinedString = list.stream().collect(Collectors.joining(", "));
System.out.println(joinedString);  // 输出: A, B, C</code></pre>
    <h3 id="%E6%80%BB%E7%BB%93%EF%BC%9A">
     总结：
    </h3>
    <p>
     <strong>
      以上是转换Java集合List为字符串的几种方法。需要注意的是，当集合中包含null值时，使用String.join()方法将会抛出NullPointerException异常，而使用StringBuilder或Java 8的Stream API则可以正确处理null值。
     </strong>
    </p>
    <hr/>
    <h2 id="2%E3%80%81%E5%B0%86string%E8%BD%AC%E6%8D%A2%E6%88%90list">
     2、将string转换成list
    </h2>
    <p>
     将一个逗号分隔的字符串转换为Java集合List也很简单，可以使用String的split()方法或者使用Java 8的Stream API的splitAsStream()方法。下面是两个示例：
    </p>
    <h3 id="2.1%E3%80%81%E4%BD%BF%E7%94%A8String%E7%9A%84split()%E6%96%B9%E6%B3%95%EF%BC%9A">
     2.1、使用String的split()方法：
    </h3>
    <pre><code class="language-java">String str = "A, B, C";
List&lt;String&gt; list = Arrays.asList(str.split("\\s*,\\s*"));
System.out.println(list);  // 输出: [A, B, C]</code></pre>
    <h3 id="2.2%E3%80%81%C2%A0%E4%BD%BF%E7%94%A8Java%208%E7%9A%84Stream%20API%E7%9A%84splitAsStream()%E6%96%B9%E6%B3%95%EF%BC%9A">
     2.2、 使用Java 8的Stream API的splitAsStream()方法：
    </h3>
    <pre><code class="language-java">String str = "A, B, C";
List&lt;String&gt; list = Pattern.compile("\\s*,\\s*").splitAsStream(str).collect(Collectors.toList());
System.out.println(list);  // 输出: [A, B, C]</code></pre>
    <h3 id="%C2%A0%E6%80%BB%E7%BB%93%EF%BC%9A">
     总结：
    </h3>
    <p>
     <strong>
      需要注意的是，在使用String的split()方法时，分隔符需要使用正则表达式进行转义，例如上述代码中的"\s*,\s*"表示匹配以逗号为分隔符的字符串，并去除前后空格。使用Java 8的Stream API的splitAsStream()方法则不需要进行转义。
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:#fe2c24;">
       TIP:
      </span>
     </strong>
     以上是将逗号分隔的字符串转换为Java集合List的两种方法。需要注意的是，当字符串中包含空格或其他分隔符时，需要相应地
     <strong>
      <em>
       调整分隔符参数的正则表达式
      </em>
     </strong>
     。
    </p>
