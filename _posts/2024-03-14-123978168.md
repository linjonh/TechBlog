     <div>
      <span style="color:#000000;">

      </span>
     </div>
    </div>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/2d027df40fcd8e2abf389283bfadec37.png" width="500"/>

    <p>
     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">

     </span>
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/6ea1aa2ebb45ddc19734cf2f29a9c435.png" width="500"/>

    <div>
     <span style="color:#000000;">
      选择软件的安装路径，点击“更改”按钮，然后根据自己的实际需要选择合适路径
     </span>
     。
    </div>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/5b3b359fb39231d3eb578f6c383f510d.png" width="500"/>
    </p>
    <p>
     不建议选择启动时检测产品更新和加入VMware客户体验提示计划。
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/f5331284bc100189189572f8f1ffe91b.png" width="500"/>
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/20946d3496bda0963fbb2f47e462e055.png" width="500"/>
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/12467c03b56474b6c317c59968a3dafa.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      安装过程，耐心等待几分钟，等待安装完成。
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/8f322fbca7886afe593b9a8713e6955b.png" width="500"/>
    </p>
    <p>
     安装完成后，
     <span style="color:#000000;">
      第一次打开软件的时候会提示你输入许可证密钥
     </span>
     ，
     <span style="color:#000000;">
      VMware 是付费软件，是需要购买的，如果你购买了
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      的话就会有一串许可密钥，如果没有购买的话就选择“我希望试用 VMware WorkstationXX 30
     </span>
     <span style="color:#000000;">
      天”选项，这样你就可以体验 30
     </span>
     <span style="color:#000000;">
      天
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      。还有一种方法可以白嫖，
     </span>
     你懂得~
    </p>
    <p class="img-center">
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/blog_migrate/ac6d740bdcb3dfc9051eb4d091f8f54b.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      软件正式打开，界面如下所示：
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="205" src="https://i-blog.csdnimg.cn/blog_migrate/b3151964f2e87e102554c728aa32afd2.png" width="800"/>
    </p>
    <h2 id="3%E3%80%81%E5%88%9B%E5%BB%BA%E8%99%9A%E6%8B%9F%E6%9C%BA">
     3、
     <span style="color:#000000;">
      <strong>
       创建虚拟机
      </strong>
     </span>
    </h2>
    <div>
     <span style="color:#000000;">
      安装好
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      后可以在
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      上创建一个虚拟机，打开
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      ，选择：文件-&gt;
     </span>
     <span style="color:#000000;">
      新建虚拟机
     </span>
     。
    </div>
    <p class="img-center">
     <img alt="" height="276" src="https://i-blog.csdnimg.cn/blog_migrate/438b65b0e4cb1ad418f66cf698b881c7.png" width="800"/>
    </p>
    <div>
     <span style="color:#000000;">
      选中
     </span>
     <span style="color:#000000;">
      “自定义”选项，然后选择“下一步”，进入
     </span>
     <span style="color:#000000;">
      硬件兼容性选择界面。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/402167a1a5ae873a472edf14289cfe66.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      使用默认值就行了，直接点击“下一步”，进入
     </span>
     <span style="color:#000000;">
      操作系统安装界面。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/ad2a1caa4f80b01d4a6cc384a580f4d6.png" width="500"/>
    </p>
    <p>
     安装下载好的Ubuntu映像文件。
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/1e0ad2d71fd58f9d2bc1af101e70abcc.png" width="500"/>
    </p>
    <p>
     输入个人用户信息。
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/a8c0d7a2f2ec08fb1329e84e2e17951b.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      给虚拟机单独清理出一块磁盘，做嵌入式开发建议这块空磁盘的大小不小于100GB。
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/86820adff439928c3148ec5a0a0c8862.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      配置虚拟机所使用的处理器数量，以及每个处理器的内核数量，这个要根据自己实际使用的电脑 CPU
     </span>
     <span style="color:#000000;">
      配置来设置。比如我的电脑
     </span>
     <span style="color:#000000;">
      CPU
     </span>
     <span style="color:#000000;">
      是
     </span>
     <span style="color:#000000;">
      I7-4720HQ
     </span>
     <span style="color:#000000;">
      ，这是个
     </span>
     <span style="color:#000000;">
      4
     </span>
     <span style="color:#000000;">
      核
     </span>
     <span style="color:#000000;">
      8线程的CPU
     </span>
     <span style="color:#000000;">
      ，因此我就可以分
     </span>
     <span style="color:#000000;">
      2
     </span>
     <span style="color:#000000;">
      个核给
     </span>
     <span style="color:#000000;">
      VMware
     </span>
     <span style="color:#000000;">
      ，然后
     </span>
     <span style="color:#000000;">
      I7-4720HQ
     </span>
     <span style="color:#000000;">
      每个物理核有两个逻辑核，因此每个处理器的内核数量就是 2
     </span>
     <span style="color:#000000;">
      ，各位可以
     </span>
     <span style="color:#000000;">
      根据自己的实际电脑CPU
     </span>
     <span style="color:#000000;">
      配置来设置。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/1a8185b946071601c4237648621cd509.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      根据自己电脑的实际内存配置来设置分给虚拟机的内存大小，比如我的电脑是16GB
     </span>
     <span style="color:#000000;">
      的内存，因此我可以给虚拟机分配
     </span>
     <span style="color:#000000;">
      8GB
     </span>
     <span style="color:#000000;">
      的内存。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/515877efd4d8d464344828c8c1cdfe18.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      选择“使用桥接网络”，点击“下一步”。
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/f48dcccf6e3640b835da536a47367d9e.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      I/O
     </span>
     <span style="color:#000000;">
      控制器类型选择默认值：“
     </span>
     <span style="color:#000000;">
      LSI Logic
     </span>
     <span style="color:#000000;">
      ”，然后点击“下一步”。
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/6971a8cd7c9f909cb7436ce2174689cd.png" width="500"/>
    </p>
    <p>
     <span style="color:#000000;">
      选择磁盘类型，使用默认值：“
     </span>
     <span style="color:#000000;">
      SCSI
     </span>
     <span style="color:#000000;">
      ”，然后点击“下一步”。
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/b5ae679807f8c8cd5704b897c25eacfa.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      选择“创建新虚拟磁盘”，这样我们前面设置好的那个空的磁盘就会被创建为一个新的磁盘。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/31bbab1367388bdf5edb5b434f4a3b65.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      进入磁盘容量设置界面，
     </span>
     设置我们清出的空的磁盘多少是给虚拟机用的，我们清出了一个空磁盘肯
     <div>
      <span style="color:#000000;">
       定是全部给虚拟机用的，因此设置最大磁盘大小为空磁盘的大小。
      </span>
      <strong>
       <span style="color:#ff0000;">
        注意，磁盘空间尽量大一点，不要设置成建议的 20GB
       </span>
       <span style="color:#ff0000;">
        ，最好
       </span>
       <span style="color:#ff0000;">
        50GB
       </span>
       <span style="color:#ff0000;">
        以上，否则开发过程中很容易提示磁盘空间不够。
       </span>
      </strong>
     </div>
    </div>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/48b78b62d7f8622fdfcc6dc23071a958.png" width="500"/>
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/46b222d69a861476e0a421c9858396c3.png" width="500"/>
    </p>
    <p>
     进入设置“自定义硬件”。
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/e4a108b602ee9a5ab4385d1b03c9ae1d.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      选中虚拟机设置对话框中的 “CD/DVD(SATA)
     </span>
     <span style="color:#000000;">
      ”选项，然后在右侧选中“使用
     </span>
     <span style="color:#000000;">
      ISO
     </span>
     <span style="color:#000000;">
      映像文件”
     </span>
     ，这里的
     <span style="color:#000000;">
      ISO 映像文件就是之前下载并安装的
     </span>
     <span style="color:#000000;">
      Ubuntu系统镜像，当然也可以再这里替换成新系统镜像。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="477" src="https://i-blog.csdnimg.cn/blog_migrate/d88cef3501869d00ee082a8d65ec785f.png" width="500"/>
    </p>
    <div>
     <span style="color:#000000;">
      选择“
     </span>
     <span style="color:#000000;">
      USB
     </span>
     <span style="color:#000000;">
      控制器”选项，默认
     </span>
     <span style="color:#000000;">
      USB
     </span>
     <span style="color:#000000;">
      控制器的
     </span>
     <span style="color:#000000;">
      USB
     </span>
     <span style="color:#000000;">
      兼容性为
     </span>
     <span style="color:#000000;">
      USB2.0
     </span>
     <span style="color:#000000;">
      ，这样当你使用USB3.0 的设备的时候
     </span>
     <span style="color:#000000;">
      Ubuntu
     </span>
     <span style="color:#000000;">
      可能识别不出来，因此我们需要调整
     </span>
     <span style="color:#000000;">
      USB
     </span>
     <span style="color:#000000;">
      兼容性为
     </span>
     <span style="color:#000000;">
      USB3.1（最新版）。
     </span>
    </div>
    <p class="img-center">
     <img alt="" height="477" src="https://i-blog.csdnimg.cn/blog_migrate/2b36547e85379d8a5d90770ff8dd5da7.png" width="500"/>
    </p>
    <p>
     最后点击“完成” ，完成虚拟的创建和配置。
    </p>
    <p class="img-center">
     <img alt="" height="473" src="https://i-blog.csdnimg.cn/blog_migrate/adf034282d99171725ecff445a1da69b.png" width="500"/>
    </p>
    <h2 id="4%E3%80%81%E5%AE%89%E8%A3%85Ubuntu%E6%93%8D%E4%BD%9C%E7%B3%BB%E7%BB%9F">

     <span style="color:#000000;">
      <strong>
       安装
      </strong>
     </span>
     <span style="color:#000000;">
      <strong>
       Ubuntu
      </strong>
     </span>
     <span style="color:#000000;">
      <strong>
       操作系统
      </strong>
     </span>
    </h2>
    <p>
     点击“启动客户机”。
    </p>
    <p class="img-center">
     <img alt="" height="397" src="https://i-blog.csdnimg.cn/blog_migrate/d7c5b511fe0b843b67d1f87692305502.png" width="800"/>
    </p>
    <div>
     <span style="color:#000000;">
      Ubuntu
     </span>
     <span style="color:#000000;">
      开始安装以后首先是语言选择。
     </span>
    </div>
    <p class="img-center">
