    <p>
     <strong>
      一、Windows自建内网服务器安装
     </strong>
     <br/>
     首先去官网下载相应的安装包，https://github.com/rustdesk/rustdesk-server/releases到这个地址去下载windows版本的服务器程序
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e9685f2eb187ee24a43e492d6a8744e5.png#pic_center">
      <br/>
      下载完毕后，直接安装即可，不用任何设置。
      <br/>
      <strong>
       二、路由器端口转发设置
      </strong>
     </img>
    </p>
    <p>
     一般在路由器设置里面的高级选项-----》端口映射
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/870585b52afaf4475bacfd65bcf53413.png#pic_center">
      <br/>
      除去21116端口使用tcp,udp协议外，其他端口都是tcp协议
      <br/>
      上图中192.168.0.59是安装服务器软件的电脑的ip
      <br/>
      <strong>
       三、客户端设置
      </strong>
      <br/>
      只需将客户端中的ID/中继服务器选项设置为你的外网固定ip地址即可
     </img>
    </p>
