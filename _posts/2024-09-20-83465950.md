     </p>
     <p class="p0" style="text-indent:21pt;">
      <span style="font-weight:bold;font-size:10.5pt;color:#800080;font-family:'宋体';">
       软件工程的方法（开发方法）、工具（支持方法的工具）、过程（管理过程）称为软件工程的三要素
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       一、
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       软件的开发方法（面向过程、面向对象、面向数据）
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       1.1
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0">
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'Times New Roman';">
       a)
      </span>
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'宋体';">
       面向过程的方法：也被称为传统的软件开发方法
      </span>
      <span style="font-size:10.5pt;color:#0000ff;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       特点：程序的基本执行过程主要不是由用户控制，而是由程序控制的，并却按时序进行。面向过程的方法的优点是简单实用，缺点是维护困难。强调“‘自顶向下，逐步求精”。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       面向过程的方法在军事上的实时跟踪系统中就有很好的应用。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0">
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'Times New Roman';">
       b)
      </span>
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'宋体';">
       面向对象的方法：也称为现代的软件工程开发方法
      </span>
      <span style="font-size:10.5pt;color:#0000ff;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       面向对象是认识论和方法学的一个基本原则。面向对象方法（Ohject--Oriented Method，OOM）是一种运用对象、类、消息传递、继承、封装、聚合、多态性等概念来构造软件系统的软件开发方法。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0">
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'Times New Roman';">
       c)
      </span>
      <span style="font-weight:bold;font-size:10.5pt;color:#0000ff;font-family:'宋体';">
       面向数据的方法：面向“元数据”的方法，也是建设信息系统数据库和数据仓库的基本方法。元数据是关于数据的数据，组织数据的数据。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       例如，数据库概念设计中的实体名和属性名，数据库物力设计中的表名称和字段名。它们就是“元数据”，而某个具体的实例就不是元数据，它们叫做对象或记录，是被元数据组织或统帅的数据。概括起来，要点如下：
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       1.
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       数据位于企业信息系统的中国心。信息系统就是对数据的输入、处理、传输、查询和输出。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       2.
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       只要企业的业务方向不变和内容不变，企业的元数据室稳定的，由元数据构成的数据模型（Data Model）也是稳定的。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       3.
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       对元数据的处理方法是可变的，
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       4.
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       信息系统的核心是数据模型。数据模型包括概念数据模型（Comceptual Data Model，CDM）和物理数据模型（Physics Data Model，PDM）。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       5.
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       信息系统的实现主要是面向对象，其次才是面向数据和面向过程。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       特点：在程序的执行过程中，根据数据流动和处理的需要，有时由程序控制（如触发器和存储过程的执行），有时由用户控制（如用户在界面上的选择）。
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       1.2软件工程的5个面向理论：面向流程分析、面向数据设计、面向对象实现、面向功能测试、面向过程管理。
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;text-align:justify;">
     </p>
     <p class="p0" style="text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       二、
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       软件支持过程
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       软件工程的支持过程由支持软件生命周期各个阶段的生产工具组成。生产工具包括需求分析工具、设计工具、实现工具，维护工具，配置工具。软件开发环境又称CASE工具。
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-weight:bold;font-size:10.5pt;font-family:'宋体';">
       完整的CASE内容包括三个方面
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       ：CASE方法（快速原型法或迭代方法）、CASE技术（实体关系建模技术或面向对象建模技术）、CASE工具（软件开发环境）。CASE方法需要CASE技术支持，CASE技术需要CCASE工具的实现。
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-align:justify;">
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="text-indent:21pt;">
      <span style="font-size:10.5pt;font-family:'宋体';">
       三、
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
       软件管理过程
      </span>
      <span style="font-size:10.5pt;font-family:'宋体';">
      </span>
     </p>
     <p class="p0" style="margin-left:21pt;text-indent:21pt;">
      <span style="font-size:10.5pt;font-family:'宋体';">
