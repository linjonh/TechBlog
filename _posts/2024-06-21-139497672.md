    <p>
     在CentOS操作系统上安装Python并创建虚拟环境是Python开发过程中的常见操作。本文将详细介绍在CentOS上安装Python、创建虚拟环境以及常用的相关命令流程，帮助读者轻松上手Python开发环境的搭建。
    </p>
    <p>
    </p>
    <h4 id="1-在centos上安装python">
     1. 在CentOS上安装Python
    </h4>
    <p>
    </p>
    <p>
     在CentOS上安装Python可以通过包管理工具yum来实现。以下是安装Python的步骤：
    </p>
    <pre><code># 更新系统软件包
sudo yum update

# 安装Python 3
sudo yum install python3
</code></pre>
    <p>
    </p>
    <p>
     安装完成后，可以通过以下命令验证Python的安装情况：
    </p>
    <pre><code>python3 --version
</code></pre>
    <p>
    </p>
    <h4 id="2-创建python虚拟环境">
     2. 创建Python虚拟环境
    </h4>
    <p>
    </p>
    <p>
     Python虚拟环境可以帮助我们在项目之间隔离依赖包，保持项目的独立性。以下是在CentOS上创建Python虚拟环境的步骤：
    </p>
    <pre><code># 安装虚拟环境管理工具
sudo yum install python3-venv

# 创建虚拟环境
python3 -m venv myenv
</code></pre>
    <p>
    </p>
    <h4 id="3-激活和退出虚拟环境">
     3. 激活和退出虚拟环境
    </h4>
    <p>
    </p>
    <p>
     激活虚拟环境后，所有的Python命令将会在虚拟环境中执行。以下是激活和退出虚拟环境的命令：
    </p>
    <p>
    </p>
    <p>
    </p>
    <pre><code># 激活虚拟环境
source myenv/bin/activate

# 退出虚拟环境
deactivate
</code></pre>
    <p>
    </p>
    <h4 id="4-虚拟环境常用命令">
     4. 虚拟环境常用命令
    </h4>
    <p>
    </p>
    <p>
     在虚拟环境中，可以使用以下命令管理Python包和环境：
    </p>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       pip install package_name
      </strong>
      ：安装Python包。
     </li>
     <li>
      <strong>
       pip freeze
      </strong>
      ：列出当前环境安装的所有包。
     </li>
     <li>
      <strong>
       pip list
      </strong>
      ：列出当前环境安装的包及其版本。
     </li>
     <li>
      <strong>
       pip uninstall package_name
      </strong>
      ：卸载Python包。
     </li>
    </ul>
