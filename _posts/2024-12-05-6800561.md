     中国有哪些开源社区？
     <br/>
     <br/>
     http://www.RedOffice.com.cn发布时间：2008年04月18日红旗2000
     <br/>
     <br/>
     开源社区是一个具有“集体开发、合作创新、对等评估”，以及“源码公开、使用自由、资源整合、信息共享”特征的创新体系。作为开源运动的重要环节，中国开源社区的发展状况很大程度上反映了开源运动的现实情况。
     <br/>
     <br/>
     国内开源社区分类
     <br/>
     <br/>
     目前国内开源社区主要分三类：
     <br/>
     <br/>
     第一类是提供开源项目协同开发/发布平台、开源资源、交流论坛，这类社区以共创软件联盟为代表，包括武汉Linux协会、LinuxForum。
     <br/>
     <br/>
     第二类是提供开源资源、交流论坛、维护自主项目，主要huihoo、Opentop等。
     <br/>
     <br/>
     第三类是引进国外开源项目、提供开源资讯、交流论坛，这类社区以信息汇聚、技术交流为主，人气较旺、水平较低，是知识传播、人才汇聚、项目培育的良好平台，其中包括JavaUnion、LinuxSir、Java.com.cn等。
     <br/>
     <br/>
     社区的生存、发展需要网络带宽、硬件主机、维护人员等资源。因此，根据社区所需资源提供者进行分类，可以分为：
     <br/>
     <br/>
     第一类是公司或组织。这类社区一般有特定项目，具有可持续、稳定性好、资源丰富等特点，包括北京软件协会支持的共创软件联盟Cosoft、具有公司背景的LinuxAid等，这些社区也因为其信息量，往往能够吸引到足够的网民支持。
     <br/>
     <br/>
     第二类是松散团队。 这类社区人气旺盛、发展迅速，但稳定性、持续性较差，包括JavaUnion、Huihoo等，有些社区一旦缺乏管理核心团队或者核心团队无暇顾及网站的时候，立刻就会流失大量用户，不利于社区的发展。
     <br/>
     <br/>
     第三类是个人。 国内绝大部分社区是靠个人兴趣在维护，这些社区以交流论坛为主、随时可能消失，可以用前扑后续、层出不穷来描述，包括chinajavaworld、linuxSir等，一旦社区的负责人个人失去兴趣，立刻就会消失，不利于培养长期的开源项目。
     <br/>
     <br/>
     国内代表性开源社区
     <br/>
     <br/>
     国内有许多开源社区，与国际开源社区相比较，国内社区更加了解中文用户的实际需求，能够更快地解决中文使用方面遇到的问题，以下是一些有代表性的社区：
     <br/>
     <br/>
     1、共创软件联盟（http://cosoft.org.cn/html）
     <br/>
     <br/>
     共创软件联盟自2000年2月份成立运作迄今，通过灵活的开放源码策略实现广泛的智力汇聚和高效的成果传播，推进创新软件技术的迅速发育和成长，促进我国软件产业在先进的机制上实现跨越式发展。 一方面，充分继承国际上已经投入数千亿美元开发出来的开放源码软件；另一方面，以国家863计划为战略导向，组织研发目前尚没有、同时急需的软件并加以集成，按照联盟许可证规则进行开放。在联盟的环境中执行了一批新的863项目，以联盟许可证的方式来管理和支持这些项目。目前，共有14个课题源码在联盟网站上对社会公开发布，涉及12个科研院校和企业的研发成果，形成良好的社会效益和经济效益，取得令人瞩目的成果，并探索出一条在新形势下863项目运作及验收评估的可行途径，通过官、产、学、研的有机结合来促进软件产业在先进机制上实现跨越式发展。截至2007年12月26日，Cosoft.org.cn已有项目1474项，注册会员49565名。
     <br/>
     <br/>
     2、武汉自由软件协会（http://www.clinux.org）
     <br/>
     <br/>
     武汉自由软件协会是一个非营利性学术团体，由自由软件及相关信息技术的爱好者、研究者、开发者组成，宗旨是发展自由软件的标准、技术与精神，促进商业合作与共识。它成立于1998年6月23日，是国内历史最悠久的自由软件社区组织之一。致力于开源协作开发平台的建设，建有GRO.clinux.org，是国内最早建设的也是规模最大的开源软件协作开发平台之一，目前有近800个开源项目，近15000注册开发用户。
     <br/>
     <br/>
     3、开源中国社区（http://oss.org.cn）
     <br/>
     <br/>
     开源中国社区（Open Source Community of China，COC）是在信息产业部的大力支持与指导下，由中国开源软件推进联盟（COPU）、中国Linux产业战略联盟（CLIA）与CSIP共同组织成立的一个综合的、开放的、互动的开源软件社区。自成立起致力于普及、推广以Linux为代表的开源软件知识，使更多的人员认识开源，接触开源，参与开源，使用开源。社区提供了论坛、协同开发、知识库、资源库等资源，它的协同开发平台支持了国内第一个开源ERP项目—恩信ERP， 清华大学学位论文LaTeX模板等重要项目。据最新统计数据显示，社区共有软件22122个，总下载数为345107次。
     <br/>
     <br/>
     4、LUPA（http://www.lupaworld.com）
     <br/>
     <br/>
     LUPA 是开源高校推进联盟（ Leadership Of Open Source University Promotion Alliance ）的英文缩写，于2005 年6月12日在杭州成立。LUPA 是中国开源运动的探索者和实践者，也是“中国开源模式”的缔造者。LUPA 主张软件自主创新，围绕学生“就业与创业”搭建起学校与企业沟通的桥梁。给在校学生或社会群体提供一个直接与产业对话的平台， LUPA 融合国际最新前沿技术，打造新型、实用的标准化课件，促进中国高校教学教程改革，扶持高校学生自主创业和灵活就业，是解决目前我国“就业瓶颈”的理想模式。
     <br/>
     <br/>
     目前LUPA 已有清华大学、北京大学、浙江大学等 300 余所大专院校加盟，成为国内规模最大、拥有院校资源最多、影响最为广泛的开源社团组织。 截至2007年12月26日，注册会员110792名。
     <br/>
     <br/>
     5、ChinaUnix.net（http://chinaunix.net）
     <br/>
     <br/>
     ChinaUnix.net（以下简称CU）是一个以讨论Linux/Unix类操作系统技术、软件开发技术、数据库技术和网络应用技术等为主的开源技术社区网站。CU的宗旨是给所有爱好Linux/Unix技术、开源技术的朋友提供一个自由、开放、免费的交流空间。CU创办于2001年，发展到今天已经成为全球最大、人气最旺的Linux/Unix技术中文网站。
     <br/>
     <br/>
     2007年12月26日12：20，论坛共有 7201789 篇帖子(其中 920856 篇主题) 论坛共有 653732 位注册会员。论坛每天的平均在线交流人数超过3000多人；每天平均PaveView接近300万次；访问独立IP数超过20万。
     <br/>
     <br/>
     6、程序员大本营（http://www.csdn.net）
     <br/>
     <br/>
     CSDN.NET--中国最大的开发者网络，为开发人员和相关企业提供全面的信息服务和技术服务。其论坛人数到2002年3月注册人数已超过20万。目前总会员数达244万。
     <br/>
     <br/>
     7、Linux中国（http://www.linux-cn.com）
     <br/>
     <br/>
     Linux中国是广大的Linux爱好者自发建立的以讨论Linux技术，推动Linux及开源软件在中国的发展为目标的技术型社区网站。Linux中国的宗旨是给所有的Linux爱好者、开源技术的朋友提供一个自由、开放、平等、免费的交流空间。
     <br/>
     <br/>
     Linux中国的前身是Linux大本营，最初创办于1999年，经过多年的努力和发展，于2007年正式更名为Linux中国，启用全新的域名：http://www.linux-cn.com 。
     <br/>
     <br/>
     截至2007年12月26日，注册会员有44050名，主题10616项, 帖子37344个。
     <br/>
     <br/>
     8、Linux人社区（http://www.linux-ren.org）
     <br/>
     <br/>
     Linux人社区是由一群致力于推动国内Linux技术发展和应用的开发者创建的综合性技术社区，其目的是建立一个涵盖Linux基础技术和高端技术的交流平台，加强Linux开发者和使用者与开源社区的交流，提高国内的Linux开发和企业级应用的技术水平；通过社区汇聚国内的Linux人才，衍生更多本土化创新开源项目，为国际开源社区作出更大的贡献。
     <br/>
     <br/>
     依托Linux人社区，创建了社区Linux版本项目Everest（珠穆朗玛）。作为社区研发的基础平台，Everest项目已经随社区同时启动并正式发布第一个版本。项目的目标是提供一款最新、最酷、最快，轻量级、模块化的Linux操作系统；并以此为平台，探索各种Linux前沿技术，开发出更多具有影响力和生命力的软件项目。
     <br/>
     <br/>
     Linux人社区全部基于开源技术搭建，提供了新闻、论坛、wiki等服务。同时，在Linux人社区网站上提供了Everest项目的Bugzilla和CVS。
     <br/>
     <br/>
     9、LinuxFans社区（http://www.linuxfans.org）
     <br/>
     <br/>
     LinuxFans社区是国内比较有特色的一个综合站点，是一个真正的Linux爱好者的乐园，其站点提供的新闻、文档、软件下载和论坛的内容都比较活跃，并且依托该社区还开发了名为Magic Linux的发行版。LinuxFans社区就像是一个开源大家庭，用户可以更多地参与到该社区的项目中去。 目前注册会员244612人。
     <br/>
     <br/>
     10、LinuxSir社区（http://www.linuxsir.org）
     <br/>
     <br/>
     LinuxSir社区是一个纯论坛形式的社区，正是由于其形式专一，为该社区带来了很好的人气，开源爱好者在该社区中都很有礼貌地提问、回答，氛围特别适合初学者进行开源软件的学习。 目前注册会员为17万余人。
     <br/>
     <br/>
     LinuxSir中为常用的Linux发行版设置了子论坛，如RedHat、SuSE、Debian等都有独立的论坛。通过LinuxSir界面提供的信息统计功能可以看到，Debian论坛的在线人数通常是最多的，这也说明Debian Linux被越来越多的用户所接受。
     <br/>
     <br/>
     11、Linux伊甸园 （http://www.linuxeden.com）
     <br/>
     <br/>
     Linux伊甸园是中国最具影响力的Linux自由软件垂直互动门户网站之一，自2000年创立以来始终致力于自由软件和开源的资讯传播推进。
     <br/>
     <br/>
     经过七年的发展，到目前已经形成了7个专业频道，28个专业论坛板块，提供资讯、数据、互动三位一体的开源资讯网络平台。Linux伊甸园拥有20万注册用户，足于GNU/Linux 及开源软件推广，依仛各大Linux 厂商及开源社团，以提供及时, 丰富, 客观的开源软件及Linux 资讯及提供开源系统/ 网络解决方案作为自已坚定不移的长期发展战略。公司具备在系统集成、行业应用、开源项目开发等方面的强大的业务创新和技术创新能力，并已经在浙江各大高校开展广泛的合作教育、项目培训。
     <br/>
     <br/>
     12、红联Linux门户（http://www.linuxdiyf.com）
     <br/>
     <br/>
     红联是全球Linux阵营中的一员，加入与近10万名Linux爱好者共同讨论学习，携手为Linux在中国普及努力奋斗。红联与大家开发基于Linux的hOS。
     <br/>
     <br/>
     注册总人数是91472，每天在线用户人数超过3000。
     <br/>
     <br/>
     13、红旗Linux技术社区（http://www.redflag-linux.com）
     <br/>
     <br/>
     红旗linux技术社区是为了让更多的用户通过技术社区得到更好的用户体验而创立的社区。注册人数众多，发挥了促进产品技术创新和广泛吸引用户的作用。
     <br/>
     <br/>
     14、Linux技术中坚站论坛（http://www.chinalinuxpub.com）
     <br/>
     <br/>
     提供Linux环境下编程环境方面的技术、新手配置系统的详细文档、论坛。注册会员超过6万，帖子近30万篇。
     <br/>
     <br/>
     15、UPLinux（http://bbs.uplinux.com）
     <br/>
     <br/>
     UPLinux.com 提供分类清晰的高质量软件工程、系统管理、网络管理相关技术资源，内容涉及开发语言、开发平台、高级系统命令、整体解决方案、及网络分析、网络安全等，为Linux 技术工程师提供技术动力。
     <br/>
     <br/>
     截至2007年12月26日，共有9158篇主题，48299篇帖子，注册会员为28673名。
     <br/>
     <br/>
     16、Matrix（http://www.matrix.org.cn）
     <br/>
     <br/>
     一个计算机开源技术组织，原创内容多，主要涉及java基础、编程、工具与应用服务、j2ee,j2me技术、设计模式等，旨在让更多的人得到进步，他们的口号：Matrix is all！ 当前注册会员：153829名。
     <br/>
     <br/>
     17、PHPChina（http://bbs.phpchina.com）
     <br/>
     <br/>
     PHPChina是一个以PHP为中心，面向软件开发者、程序爱好者的开源技术网站及交流社区。作为PHP语言开发公司Zend Technology在大中华区的唯一授权官方网站，PHPChina同时也是Zend中国技术支持中心的官方社区，更是中国每个PHPer自己的开源社区。截至到2007年12月26日，PHPChina上共有40971主题，326204帖子，注册会员47746。
     <br/>
     <br/>
     18、灰狐动力（http://www.huihoo.com）
     <br/>
     <br/>
     成立于2001年9月已成为国内中间件行业极具权威性的站点，致力于企业计算的研究：CORBA、J2EE、Web Services、EAI等并组织开源中间件(huihoo.org)的开发：Orbas、JFox、OpenWeb、Link3。
     <br/>
     <br/>
     19、JAVA中文站（http://www.java-cn.com）
     <br/>
     <br/>
     一个老牌的站点提供大量的JAVA资料，是目前国内较好的java专业技术网站之一，在J友中已经具有相当的名气，已经拥有注册会员90000多人。
     <br/>
     <br/>
     20、JAVA世界中文论坛（http://bbs.chinajavaworld.com）
     <br/>
     <br/>
     该社区专为JAVA开源项目讨论和研究而设，几乎包含了所有流行的开源技术。每天在线人数超过3000人。
     <br/>
     <br/>
     21、Java研究组织（http://www.javaresearch.org）
     <br/>
     <br/>
     一个面向Java相关技术的专业性内容服务网站，于2002年4月成立，用户60000余人，原创文章多。
     <br/>
     <br/>
     22、EasyWine论坛（http://www.easywine.org）
     <br/>
     <br/>
     是中国第一个讨论Wine模拟应用技术的论坛。让Linux运行Windows程序，让新手更容易从Windows过渡到Linux丰富Linux的应用软件和游戏娱乐。注册用户超过万人。
     <br/>
     <br/>
     23、OpenDesktop.net （http://www.opendesktop.net）
     <br/>
     <br/>
     OpenDesktop.net是国内第一个以技术开发为主的大型开放源代码社区，致力于开发商品化的、与Windows高度兼容的、适合政府和企业商用的桌面Linux操作系统，这个系统的名称就叫OpenDesktop。社区由北京共创开源支持创办，当前在本社区发布的桌面Linux产品包括CoCreateLinux 2005SP2 和CoCreateOffice 2005SP2。
    </span>
