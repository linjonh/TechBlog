     下面从一个完全不懂的Python 的小白来安装Python、PyCharm等一系列工作的记录，遇到的问题也会写出，让完全不懂的小白也可上手安装，并且完成第一个Hello world代码。希望能够帮助到大家。

     一、Python下载与安装
    </h3>
    <p>
     真实在Windows上安装过Python的小伙伴会发现在 Windows 上安装Python真的和安装普通软件一样简单，我们只需要在官网下载个安装包，然后猛击“下一步”就可以了（真的秒~）！
    </p>
    <p>
     Python 安装包官网下载地址：
     <a href="https://www.python.org/downloads/" rel="nofollow">
      https://www.python.org/downloads/
     </a>
    </p>
    <p>
     <strong>
      第一步：点击首页的Downloads，并在下拉框中选择Windows
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c98898c4cc6e4ea99febada42d5dae70.jpeg#pic_center"/>
    </p>
    <p>
     <strong>
      第二步：选择指定版本
     </strong>
    </p>
    <p>
     此处我选择的是最3.11.4版本，但是如果你因为项目对Python版本有特定要求的话，可以此界面下滑选择需要的版本下载！
    </p>
    <p>
     需要插一嘴的是，为了防止你们下错而死在了安装上，简单对此界面一些前缀后缀进行说明（认真看看你下的是不是你想要的）：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        前后缀
       </th>
       <th>
        含义
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        64-bit
       </td>
       <td>
        64位的Python安装程序（现在的电脑一般都是64bit的）
       </td>
      </tr>
      <tr>
       <td>
        32-bit
       </td>
       <td>
        32位的Python安装程序
       </td>
      </tr>
      <tr>
       <td>
        Windows embeddable package
       </td>
       <td>
        .zip格式的绿色免安装版本，可直接嵌入（集成）到其他的应用程序中
       </td>
      </tr>
      <tr>
       <td>
        Windows installer
       </td>
       <td>
        .exe格式的可执行程序，这是完整的离线安装包（就选这个，别想了！）
       </td>
      </tr>
      <tr>
       <td>
       </td>
       <td>
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      第三步：下拉到最下面然后点击就在下载了哦
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3987b31ab0314a28bbf7d67e84236c08.png"/>
    </p>
    <p>
     <strong>
      第四步：双击下载的.exe文件就正式开始安装Python了
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3cef11c69f7d4dbf9f0509c75af6a69d.png"/>
    </p>
    <ul>
     <li>
      Install now 默认安装和默认安装路径（一般默认安装在C盘，不建议）
     </li>
     <li>
      Customize installation 自定义安装
     </li>
    </ul>
    <p>
     （友情提示：
     <strong>
      一定要勾选Add Python.exe to PATH
     </strong>
     ，这样的话Python命令工具所在的目录就会自动添加到系统环境变量中，就不需要你手动添加python系统环境变量啦，在执行Python命令时也不需要指定完整路径了！）
    </p>
    <p>
     <strong>
      Python 支持两种安装方式，默认安装和自定义安装：
     </strong>
    </p>
    <ul>
     <li>
      默认安装会勾选所有组件，并安装在 C 盘；
     </li>
     <li>
      自定义安装可以手动选择要安装的组件，并可自定义其安装位置。（正经人肯定选这个，我也推荐选这个，不然你啥东西都安C盘里，电脑会变卡哦~而且大家以后不管安装啥都看一眼安装路径，建议自定义哦！）
     </li>
    </ul>
    <p>
     ①点击Customize installation进入下一步，选择要安装的组件，建议保持默认即可，也就是全部勾选；
    </p>
    <p>
     ②点击“Next”，选择安装目录（这就是自定义Python的安装路径，自己找个盘放哦~）
    </p>
    <p>
     ③最后点击“Install”，静候一会就安装好啦！
    </p>
    <p>
     <strong>
      In the end：测试是否安装OK
     </strong>
    </p>
    <p>
     “win+r”打开Windows的命令行程序，输入python命令，如果出现下图所示则证明安装OK哦！
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c61da1ad0f4570f263d00479bf70fcbc.png"/>
    </p>
