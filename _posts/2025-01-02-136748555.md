    <p>
     本文将详细介绍Python中文件操作的基本概念，包括文件的读取、写入和追加。我们将深入探讨Python文件操作的核心知识，并通过代码示例来帮助我们更好地理解和掌握Python文件操作。

    <h3>
     <a id="2__4">
     </a>
     2. 文件读取
    </h3>
    <p>
     文件读取是指从文件中读取数据的过程。在Python中，文件读取通常使用内置的open()函数和文件对象的read()、readline()、readlines()等方法。
     <br/>
     <strong>
      2.1 open()函数
     </strong>
     <br/>
     open()函数是Python中用于打开文件的内置函数。它接受两个参数：文件路径和模式。文件路径是指文件的存储位置，可以是相对路径或绝对路径。模式是指打开文件的方式，包括只读模式(‘r’)、写入模式(‘w’)、追加模式(‘a’)等。
     <br/>
     例如，以下代码打开一个名为"example.txt"的文件，并以只读模式读取文件内容：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>
content <span class="token operator">=</span> <span class="token builtin">file</span><span class="token punctuation">.</span>read<span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>content<span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      2.2 read()方法
     </strong>
     <br/>
     read()方法是文件对象的一个方法，用于读取文件的全部内容。它接受一个可选的参数size，表示读取的字节数。如果不指定size参数，read()方法将读取整个文件内容。
     <br/>
     例如，以下代码读取"example.txt"文件的全部内容：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>
content <span class="token operator">=</span> <span class="token builtin">file</span><span class="token punctuation">.</span>read<span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>content<span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      2.3 readline()方法
     </strong>
     <br/>
     readline()方法是文件对象的一个方法，用于读取文件的一行内容。它返回读取到的行内容，如果已到达文件末尾，则返回空字符串。
     <br/>
     例如，以下代码逐行读取"example.txt"文件的内容：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>
<span class="token keyword">while</span> <span class="token boolean">True</span><span class="token punctuation">:</span>
    line <span class="token operator">=</span> <span class="token builtin">file</span><span class="token punctuation">.</span>readline<span class="token punctuation">(</span><span class="token punctuation">)</span>
    <span class="token keyword">if</span> <span class="token keyword">not</span> line<span class="token punctuation">:</span>
        <span class="token keyword">break</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span>line<span class="token punctuation">.</span>strip<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      2.4 readlines()方法
     </strong>
     <br/>
     readlines()方法是文件对象的一个方法，用于读取文件的所有行内容。它返回一个列表，列表中的每个元素都是文件的一行内容。
     <br/>
     例如，以下代码读取"example.txt"文件的所有行内容：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>

<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="3__48">
     </a>
     3. 文件写入
    </h3>
    <p>
     文件写入是指将数据写入文件的过程。在Python中，文件写入通常使用内置的open()函数和文件对象的write()、writelines()等方法。
     <br/>
     <strong>
      3.1 open()函数
     </strong>
     <br/>
     open()函数用于打开文件，并设置文件的模式。在写入模式下，如果文件已存在，则打开文件并清空文件内容；如果文件不存在，则创建新文件。
     <br/>
     例如，以下代码打开一个名为"example.txt"的文件，并以写入模式写入数据：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"w"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>write<span class="token punctuation">(</span><span class="token string">"Hello, World!"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      3.2 write()方法
     </strong>
     <br/>
     write()方法是文件对象的一个方法，用于向文件中写入一个字符串。它接受一个字符串作为参数，并将其写入文件。
     <br/>
     例如，以下代码向"example.txt"文件中写入一行文本：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"w"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>write<span class="token punctuation">(</span><span class="token string">"Hello, World!\n"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      3.3 writelines()方法
     </strong>
     <br/>
     writelines()方法是文件对象的一个方法，用于向文件中写入一个字符串列表。它接受一个字符串列表作为参数，并将列表中的每个字符串写入文件。
     <br/>
     例如，以下代码向"example.txt"文件中写入多行文本：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"w"</span><span class="token punctuation">)</span>
lines <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token string">"Hello, World!\n"</span><span class="token punctuation">,</span> <span class="token string">"Python is great!\n"</span><span class="token punctuation">,</span> <span class="token string">"I love Python!\n"</span><span class="token punctuation">]</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>writelines<span class="token punctuation">(</span>lines<span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="4__77">
     </a>
     4. 文件追加
    </h3>
    <p>
     文件追加是指在文件的末尾添加数据的过程。在Python中，文件追加通常使用内置的open()函数和文件对象的write()、writelines()等方法。
     <br/>
     <strong>
      4.1 open()函数
     </strong>
     <br/>
     open()函数用于打开文件，并设置文件的模式。在追加模式下，如果文件已存在，则打开文件并在文件末尾添加数据；如果文件不存在，则创建新文件。
     <br/>
     例如，以下代码打开一个名为"example.txt"的文件，并以追加模式写入数据：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"a"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>write<span class="token punctuation">(</span><span class="token string">"Hello, World!\n"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      4.2 write()方法
     </strong>
     <br/>
     write()方法是文件对象的一个方法，用于向文件中写入一个字符串。在追加模式下，write()方法会将字符串追加到文件的末尾。
     <br/>
     例如，以下代码向"example.txt"文件中追加一行文本：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"a"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>write<span class="token punctuation">(</span><span class="token string">"This is an appended line.\n"</span><span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      4.3 writelines()方法
     </strong>
     <br/>
     writelines()方法是文件对象的一个方法，用于向文件中写入一个字符串列表。在追加模式下，writelines()方法会将列表中的每个字符串依次追加到文件的末尾。
     <br/>
     例如，以下代码向"example.txt"文件中追加多行文本：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"a"</span><span class="token punctuation">)</span>
lines <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token string">"This is another line.\n"</span><span class="token punctuation">,</span> <span class="token string">"And another line.\n"</span><span class="token punctuation">]</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>writelines<span class="token punctuation">(</span>lines<span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="5__106">
     </a>
     5. 文件指针操作
    </h3>
    <p>
     文件指针是文件对象内部的一个指示器，用于指向当前读写位置。在Python中，文件对象提供了一些方法来操作文件指针。
     <br/>
     <strong>
      5.1 tell()方法
     </strong>
     <br/>
     tell()方法是文件对象的一个方法，用于获取当前文件指针的位置。
     <br/>
     例如，以下代码获取当前文件指针的位置：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>

<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      5.2 seek()方法
     </strong>
     <br/>
     seek()方法是文件对象的一个方法，用于设置文件指针的位置。它接受两个参数：偏移量和参考点。偏移量是指文件指针移动的字节数；参考点是指文件指针移动的参考位置，可以是文件开头(0)、当前位置(1)或文件末尾(2)。
     <br/>
     例如，以下代码将文件指针设置到文件开头：
    </p>
    <pre><code class="prism language-python"><span class="token builtin">file</span> <span class="token operator">=</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">"example.txt"</span><span class="token punctuation">,</span> <span class="token string">"r"</span><span class="token punctuation">)</span>

content <span class="token operator">=</span> <span class="token builtin">file</span><span class="token punctuation">.</span>read<span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>content<span class="token punctuation">)</span>
<span class="token builtin">file</span><span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>

     <br/>
