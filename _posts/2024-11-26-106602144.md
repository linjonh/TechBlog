    <h3>
     <a id="_2">

     二级目录
    </h3>
    <h4>
     <a id="_3">
     </a>
     三级目录
    </h4>
    <p>
     今天想把java的一个jar包放到linux上运行
     <br/>
     把jar包上传上去后发现
     <br/>
     -bash: java: command not found
     <br/>
     找不到java命令？
     <br/>
     看来linux上jdk有问题
    </p>
    <h2>
     <a id="1jdkjdk_10">
     </a>
     1、检查一下是否安装了jdk，看来还没有安装jdk
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7a0d3c5c416d7b120d79761f9e8f141f.png"/>
    </p>
    <h2>
     <a id="2linux_15">
     </a>
     2、检查linux位数
    </h2>
    <pre><code>[root@centos7 ~]# getconf LONG_BIT
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2bcda54888a22b3881813a5a719c4bd2.png">
      <br/>
      出现64，说明是64位的，如果是32就是32位的。
     </img>
    </p>
    <h2>
     <a id="3linux_JDKjdk8_22">
     </a>
     3、下载对应的linux JDK包，这里使用jdk8
    </h2>
    <p>
     下载jdk8 : 链接:
     <a href="https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html" rel="nofollow">
      https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
      <br/>
     </a>
     .
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/28899f48a8395d5ea5ddb288a4ec587e.png"/>
    </p>
    <h2>
     <a id="4_jdk8u251linuxx64targz_linux_28">
     </a>
     4、上传 jdk-8u251-linux-x64.tar.gz 到linux
    </h2>
    <p>
     把下载的jdk放在usr/java目录下，建议创建jdk8安装目录
    </p>
    <pre><code>[root@centos7 ~]# mkdir usr
[root@centos7 ~]# mkdir java
[root@centos7 ~]# mkdir jdk8
[root@centos7 ~]# cd usr/java/
[root@centos7 java]# ll
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8b3a8b7e77b3cae51f45760c9489ab89.png"/>
    </p>
    <h2>
     <a id="5jdk_40">
     </a>
     5、解压下载的jdk
    </h2>
    <pre><code>[root@centos7 java]# cd jdk8
[root@centos7 jdk8]# tar zxvf ../jdk-8u251-linux-x64.tar.gz ./
</code></pre>
    <p>
     解压成功后得到一个文件夹 jdk1.8.0_251
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/59b6d71ebaad20211128746fffe09540.png"/>
    </p>
    <h2>
     <a id="6_50">
     </a>
     6、编辑配置文件，配置环境变量
    </h2>
    <pre><code>[root@centos7 java]# vim /etc/profile
</code></pre>
    <p>
     在最后一行添加以下内容
    </p>
    <pre><code>export JAVA_HOME=(安装jdk的目录，可以先去该目录下使用 pwd 命令查看)

export PATH=$PATH:$JAVA_HOME/bin

export CLASSPATH=.:$JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9872f02222868bb6308161c9fabc8ad2.png"/>
    </p>
    <h2>
     <a id="7_65">
     </a>
     7、刷新配置文件
    </h2>
    <pre><code>  [root@centos7 java]# source /etc/profile
</code></pre>
    <p>
     重启命令：sudo shutdown -r now
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ad741106c7ce68264a4cf5cd06c20081.png"/>
    </p>
    <h2>
     <a id="8JDK_73">
     </a>
     8、最后检查JDK安装是否成功
    </h2>
    <p>
     java -version
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c63ede0fe0f71fb6c747ca334e27ebc9.png"/>
    </p>
    <p>
     到此，linux下jdk8安装成功了
    </p>
