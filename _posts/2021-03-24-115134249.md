    <blockquote>
     <p>
      微信用户数量已经突破12亿了，小程序现在已经成了主流的“轻应用”开发首选，你还在观望吗？本篇文章开始，带你走进小程序开发的大门。
     </p>
    </blockquote>
    <p>
     1、注册小程序账号
     <br/>
     1）首先，我们访问
     <a href="https://mp.weixin.qq.com/" rel="nofollow">
      微信公众平台
     </a>
     ，点击右上方的“立即注册”按钮，如下图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/be42c3ce2ab85824b1f65fb9f3a47637.png">
      <br/>
      2）这里我们可以看到有几个选项，我们选择小程序：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1ed9101581bdd1979ea181ea3e157c55.png">
       <br/>
       3）按照页面提示进行注册信息填写：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/087440bc4006e9b9d811d313647bf6a0.png">
        <br/>
        4）点击注册后，会向你填写的邮箱地址发送一封邮件，需要你登录邮箱查看，然后点击邮箱里面的链接进行下一步操作：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a2848cf81ba0c3bdfc910f2e2694f2e0.png">
         <br/>
         5）这里需要选择你要注册小程序的主体类型，这个按需要选择即可（ps：个人小程序无法微信认证）：
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9414b637542ccfabcc5808c1c3e98408.png">
          <br/>
          6）我这里选择的是个人主体，选择完主体后，下面会展示出信息补全页面，我们继续按提示把信息输入一下，这里需要使用你的微信号扫描页面上的二维码进行管理员绑定， 扫描成功会如下图所示，我们点击“继续”按钮进行下一步：
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1227407777c1448b6c4a8cbefd716a7a.png">
           <br/>
           7）页面会弹出提示框，仔细阅读，并确认填写信息无误即可点击“确定”按钮完成小程序账号的创建：
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4b2a3214f37c733b10a0114bb848f3cc.png">
            <br/>
            8）我们点击“前往小程序”按钮，即可进入到小程序管理页面，这个页面里包含所有小程序相关功能，我们可以依次点击一下，熟悉一下里面的页面功能：
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa153feab3d84ec56f0e1b2ff6cab32b.png">
             <br/>
             9）我们需要先完善一下小程序的信息，才能正常使用小程序，我们点击小程序信息一栏后面的“填写”按钮：
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8700dfd2d6bcde7174919935424ea73c.png">
              <br/>
              10）进入到小程序信息编辑页面，这里我们要填写小程序名称、简称、以及小程序的头像、简介、服务类目（
              <strong>
               ps：小程序不能重名，修改小程序名称需要审核，所以如果是要正式使用的小程序建议提前准备好确定使用的名字，不要后面再去修改
              </strong>
              ）：
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf99c9027c350c38cfe0beb47012652f.png">
               <br/>
               <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ccd94c87384968375be0cb6d4e397c7.png"/>
               <br/>
               11）点击提交，如果有信息填写错误，按提示修改完毕 即可完成小程序资料补全。
              </img>
             </img>
            </img>
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <blockquote>
     <p>
      至此，我们的微信小程序账号就已经注册完毕了，下一章，我们将学习如何使用微信开发者工具，see you soon！
     </p>
    </blockquote>
