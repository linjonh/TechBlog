    <p>
     最近在写小程序项目，遇到calc计算无效：
    </p>
    <pre><code class="language-css">.viewOne{
    height: calc(100vh - 30rpx);
}

.viewTwo{
    height: calc(100% - 30rpx);
}</code></pre>
    <p>
     在浏览器查看样式的时候，可见：
     <br/>
    </p>
    <pre><code class="language-css">.viewOne{
    height: calc(70vh);
}

.viewTwo{
    height: calc(70%);
}</code></pre>
    <p>
     上面的明显就不对，下面展示有效的写法：
    </p>
    <pre><code class="language-css">.viewOne{
    height: calc(~"100vh - 30rpx");
}

.viewTwo{
    height: calc(~"100% - 30rpx");
}</code></pre>
    <p>
     在浏览器查看页面相关样式属性：
    </p>
    <pre><code class="language-css">.viewOne{
    height: calc(100vh - 30rpx);
}

.viewTwo{
    height: calc(100% - 30rpx);
}</code></pre>
    <p>
     完美有效：
    </p>
    <p>
     <strong>
      最后小结一下calc的使用：
     </strong>
    </p>
    <ol>
     <li>
      运算符左右两边要有空格；
     </li>
     <li>
      数字和单位之间不能有空格；
     </li>
     <li>
      括号与减数，被减数之间不能有空格；
     </li>
     <li>
      以上无效记得加上英文~符和双引号。
     </li>
    </ol>
