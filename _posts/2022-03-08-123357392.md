    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      一. JavaScript简称js 是脚本语言
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <ol>
     <li style="text-align:justify;">
      <span style="color:#333333;">
       Javascript
      </span>
      <span style="color:#333333;">
       是一种具有函数优先的轻量级，解释型或即时编译型的编程语言。
      </span>
     </li>
    </ol>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <ol>
     <li style="text-align:justify;">
      <span style="color:#333333;">
       虽然他是作为开发web页面的脚本语言而出名，但是它也被用到了很多非浏览器的环境中，JavaScript基于原型编程，多范式的动态脚本语言，
      </span>
     </li>
    </ol>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <ol>
     <li style="text-align:justify;">
      <span style="color:#333333;">
       并且支持面向对象，命令式，声明式，函数式编程范式。
      </span>
     </li>
    </ol>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      二.主要功能可以镶嵌动态文本于HTML页面
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      1.
     </span>
     <span style="color:#333333;">
      对浏览器的事件做出响应
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      2.
     </span>
     <span style="color:#333333;">
      读写HTML元素
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      3.
     </span>
     <span style="color:#333333;">
      在数据被提交到服务器之前验证数据。
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      4.
     </span>
     <span style="color:#333333;">
      检测访客的浏览器信息
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      5.
     </span>
     <span style="color:#333333;">
      基于node.js技术进行服务器端编程
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      6.
     </span>
     <span style="color:#333333;">
      可以改变HTML中的图像
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      7.
     </span>
     <span style="color:#333333;">
      可以改变HTML中的样式
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      三．JavaScript脚本语言具有以下特点
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      1.
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#333333;">
       基于对象，JavaScript是一种基于对象的脚本语言，他不仅可以创建对象，也可以使用现有的对象
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#333333;">
       2.
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#333333;">
       简单。JavaScript语言采用的是弱类型的变量类型对使用数据类型未作出严格的要求，是基于Java基本语句和控制的脚本语言，其设计简单紧凑
      </span>
     </span>
    </p>
