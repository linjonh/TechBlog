    <p>
     在小程序开发过程中，经常遇到一种情况：页面中的一个按钮需要带有
     <code>
      open-type
     </code>
     ，同时还要灵活地设置这个按钮的尺寸和背景样式。
    </p>
    <p>
     这就意味着，你不能使用
     <code>
      &lt;view bind:tap="onBtn"&gt;&lt;/view
     </code>
     再加上css设置一张背景图来实现。
    </p>
    <p>
     然后系统的
     <code>
      &lt;button&gt;
     </code>
     也不是什么省油的灯。相信大家都干过这样的事情：
    </p>
    <pre class="has"><code>&lt;!--pages/demo/index.wxml--&gt;
&lt;view class="container"&gt;
    &lt;button&gt;button&lt;/button&gt;
    &lt;button class="btn-0" open-type="contact" plain="true"&gt;btn_0&lt;/button&gt;
    &lt;button class="btn-1" open-type="contact" plain="true"&gt;btn_1&lt;/button&gt;
    &lt;button class="btn-2" open-type="contact"&gt;btn_2&lt;/button&gt;
    &lt;button class="btn-3" open-type="contact"&gt;btn_3&lt;/button&gt;
    &lt;button class="btn-4" open-type="contact"&gt;btn_4&lt;/button&gt;
&lt;/view&gt;
</code></pre>
    <pre class="has"><code>/* pages/demo/index.wxss */

button {
width: 488rpx;
height: 65rpx;
margin-top: 20rpx;
}

.btn-0 {
border: none;
}

.btn-1::after {
border: none;
}

.btn-2 {
border: none;
}

.btn-3::after{
border: none;
}

.btn-4::after{
border: none;
background-image: url('https://res.groooo.com/yaj_home_btn_more.png');
background-size: 100%;
}
</code></pre>
<p>
<img alt="" class="has" src="//upload-images.jianshu.io/upload_images/14984314-bf1b33a804455368.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/377/format/webp"/>
</p>
<p>
image.png
</p>
<p>
哎呀，不小心透漏剧情。
</p>
<p>
所以要满足刚刚的需求，应该如此
</p>
<pre class="has"><code>&lt;button class="btn"&gt;&lt;/button&gt;
</code></pre>
<pre class="has"><code>.btn::after {
border: none;
background-image: url('https://res.groooo.com/yaj_home_btn_more.png');
background-size: 100%;
}
</code></pre>
<p>
当然，看了这么久，只知道这个，肯定不划算。你一定很想知道这里面的
<code>
::after
</code>
是干啥的。
</p>
