    <h2 class="entry-title" style='margin:0px 0px 30px; font-size:28px; font-family:"Roboto Slab",serif; line-height:1.1; color:rgb(68,68,68)'>
     Piwik——最好用的开源网站统计系统
    </h2>
    <div class="entry-content" style='margin:1.5em 0px 0px; font-family:"Open Sans"; font-size:14px; color:rgb(107,107,107)'>
     <p style="margin-top:0px; margin-bottom:10px">
      前段时间一直在寻找开源的网站统计系统，在网上发现了开源的Piwik,它简洁强大，有时一个开源的系统，可以部署在任何PHP+MYSQL上，可以说是最好用的网站统计系统，如果你不想使用国内其他厂商的网站统计系统，而想使用自己的统计系统，将网站统计数据留在自己的服务器上的话，Piwik完全满足你的要求。下面我们看看Piwki的官方介绍吧。
     </p>
     <h3 style='font-family:"Roboto Slab",serif; line-height:1.1; color:rgb(218,68,83); margin-top:20px; margin-bottom:10px; font-size:24px'>
      简介
     </h3>
     <div class="fusion-column content-box-column content-box-column-1 col-lg-12 col-md-12 col-sm-12" style="position:relative; min-height:1px; padding-right:15px; padding-left:15px; float:left; width:648px">
      <div class="col content-wrapper" style="">
       <blockquote style="padding:10px 20px; margin:0px 0px 20px; font-size:17.5px; border-left:5px solid rgb(238,238,238)">
        <p style="margin-top:0px; margin-bottom:0px">
         Piwik 是一套基于 Php+MySQL 技术构建，能够与 Google Analytics 相媲美的开源网站访问统计系统。Piwik 可以给你详细的统计信息，比如网页浏览人数, 访问最多的页面, 搜索引擎关键词等等，并且采用了大量的AJAX/Flash技术，使得在操作上更加便易。
        </p>
       </blockquote>
      </div>
      <blockquote style="padding:10px 20px; margin:0px 0px 20px; font-size:17.5px; border-left:5px solid rgb(238,238,238)">
       <div class="col content-wrapper" style="">
        <p style="margin-top:0px; margin-bottom:10px">
         Piwik 可以安装在你的服务器上面，数据就保存在你自己的服务器上 面。你可以非常容易的插入统计图表到你的博客或是网站后台的控制面板中。安装完成后，你只需将一小段代码放到将要统计的网页中即可。
        </p>
        <p style="margin-top:0px; margin-bottom:10px">
         Piwik还采用了插件扩展及开放API架构，拥有众多不同功能的插件，也可以让开发人员根据自已的实际需求创建更多的功能。
        </p>
        <p style="margin-top:0px; margin-bottom:0px">
        </p>
       </div>
      </blockquote>

      <h3 style='font-family:"Roboto Slab",serif; line-height:1.1; color:rgb(218,68,83); margin-top:20px; margin-bottom:10px; font-size:24px'>
       安装过程
      </h3>
      <p style="margin-top:0px; margin-bottom:10px">
       Piwik 的安装方式非常简单，只需要下载Piwik源码，将其解压后上传到PHP+MYSQL空间，然后按照提示一步步安装即可。
      </p>

             <div style="">
              一、查看需求（http://piwik.org/docs/requirements/）
             </div>
             <p style="margin-top:0px; margin-bottom:10px">
              安装软件包：
             </p>
             <div style="">
              sudo apt-get install php5
             </div>
             <div style="">
              sudo apt-get install php5-gd（图形库，不是必须）
             </div>
             <div style="">
              <p style="margin-top:0px; margin-bottom:10px">
               sudo apt-get install mysql-server
              </p>
              <div style="">
               sudo apt-get install php5-mysql（安装PDO_MYSQL扩展和MYSQLI 扩展）
              </div>
             </div>
             <div style="">
              二、安装piwik（http://piwik.org/docs/installation/）
             </div>
             <div style="">
              下载最新的 piwik
             </div>
             <div style="">
              把解压出来的 piwik 文件夹放到服务器上的 www 目录下（本人使用apache服务器，网站目录为/var/www/）
             </div>
             <div style="">
              使用浏览器进入 http://yourwebsite/piwik/（yourwebsite 是你的网站）开始 piwik 安装。
             </div>
             <div style="">
              安装过程中会提示出现的问题及解决方法。
