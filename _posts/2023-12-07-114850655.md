     一，环境准备：

    <p>
     1，
     <a href="https://download.csdn.net/download/qq_39225639/15816951?spm=1001.2014.3001.5501">
      植物大战僵尸安装包(CSDN下载链接)-无广告植入
     </a>
     <br/>
     2，用于修改读取和修改16进制文件的工具
     <br/>

      <br/>
      3，植物大战僵尸用户存档文件路径：
      <br/>
      C:\ProgramData\PopCap Games\PlantsVsZombies\userdata\user1.dat
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf35b94b418f2649ee7127147cf31745.png">

    <h3>
     <a id="_9">
     </a>
     二，任务说明：
    </h3>
    <p>
     修改《植物⼤战僵⼫》的存档⽂件，将⼀个新⽤户的进度直接修改到8-8关，⾦钱（能量）直接修改到1000；
     <br/>
     注：游戏金额 = 用户存档中所设置值的10倍，因此金币修改到1000 则值只需设置成100。
    </p>
    <h3>
     <a id="_14">
     </a>
     三，方法步骤：
    </h3>
    <p>
     1，首先进入游戏创建用户；
     <br/>
     2，通过玩游戏，按进度备份游戏进度文件，观察用户存档文件中数据的变化，找出存储游戏进度，和金币存储地址。（只有当游戏进度到达4-1商店功能才被开启）
     <br/>
     3，修改用户存档完成本次任务。
    </p>
    <h3>
     <a id="_20">
     </a>
     四，演示
    </h3>
    <p>
     1，起始状态
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/384b473af30628a9ef4c1e0a2c874f6f.png">
      <br/>
      2，通过对比发现了04这个地址存储的是游戏进度，每一层共10关，因此想要到达4-1的关卡则需要设置到31转换成16进制值是1f
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6c79b76fb76665fc72461de3361a59bb.png">
       <br/>
       此时游戏进度到达了4-1，商店功能已开启。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b0dac5fa15c3c199399a8248345cbf41.png">
        <br/>
        进入商店，查看到金额为 0；
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5e106377f0026fd69d3ab78a6a50d97e.png">
         <br/>
         3，继续游戏，通过对比发现，金币的存储地址为：01这个位置，十进制100就相当于游戏金币1000；因此16进制文件中值应该设置为64；
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7c9ab7a69e56ffc07c9fab336dc2b89.png">
          <br/>
          可以看到此时金币为1000了。
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7779d6a6b21f4da3d8b2ea86ab661d5f.png">
           <br/>
           4，通过观察游戏里的商品金额都挺大的，对于 09 这一个的内存空间最大值是ff 转换成十进制就是255对应的金币就是2550，因此存储游戏金币的值应该还有其他内存空间；
