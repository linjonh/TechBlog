    <h2>
     Java this关键字的功能说明:
    </h2>
    <p>
     在java代码中,this关键字用于引用当前对象的引用变量
    </p>
    <ol>
     <li>
      <code>
       this
      </code>
      关键字可用来引用当前类的实例变量。
     </li>
     <li>
      <code>
       this
      </code>
      关键字可用于调用当前类方法(隐式)。
     </li>
     <li>
      <code>
       this()
      </code>
      可以用来调用当前类的构造函数。
     </li>
     <li>
      <code>
       this
      </code>
      关键字可作为调用方法中的参数传递。
     </li>
     <li>
      <code>
       this
      </code>
      关键字可作为参数在构造函数调用中传递。
     </li>
     <li>
      <code>
       this
      </code>
      关键字可用于从方法返回当前类的实例。
     </li>
    </ol>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1fee29c8439213ce26acc76c79b7f502.jpeg"/>
    </p>
    <h3 id="h2-1-this-">
     1. this：引用当前类的实例变量
    </h3>
    <p>
     <code>
      this
     </code>
     关键字可以用来引用当前类的实例变量。
     <br/>
     如果实例变量和参数之间存在歧义，则
     <code>
      this
     </code>
     关键字可用于明确地指定类变量以解决歧义问题。
    </p>
    <p>
     <strong>
      了解没有 this 关键字的问题
     </strong>
    </p>
    <p>
     下面先来理解一个不使用
     <code>
      this
     </code>
     关键字的示例：
    </p>
    <pre><code>class testClass {
    int keyId;
    String info1;
  
    testClass(int keyId, String info1) {
        keyId = keyId;
        info1 = info1; 
    }

    void printInfo() {
        System.out.println(keyId+ " " + info1);
    }


     public static void main(String args[]) {
        testClass  s1 = new testClass(888,"java265.com");
        testClass  s2 = new testClass(999,"test");
        s1.printInfo();
        s2.printInfo();
    }
}
</code></pre>
    <p>
     ------运行以上代码，将输出以下信息----
    </p>
    <pre><code>0 null
0 null</code></pre>
    <p>
     <strong>
      在上面的java代码中，
      <br/>
      参数(形式参数)和实例变量(
      <code>
       rollno
      </code>
      和
      <code>
       name
      </code>
      )是相同的，
      <br/>
      此时我们需使用
      <code>
       this
      </code>
      关键字来区分局部变量和实例变量
     </strong>
    </p>
    <hr/>
    <p>
     <strong>
      使用 this 关键字解决了上面的问题
     </strong>
    </p>
    <pre><code>class testClass {
    int keyId;
    String info1;
  
    testClass(int keyId, String info1) {
        this.keyId = keyId;
        this.info1 = info1; 
    }

    void printInfo() {
        System.out.println(keyId+ " " + info1);
    }


     public static void main(String args[]) {
        testClass  s1 = new testClass(888,"java265.com");
        testClass  s2 = new testClass(999,"test");
        s1.printInfo();
        s2.printInfo();
    }
}
</code></pre>
    <p>
     ------运行以上代码，将输出以下信息----
    </p>
    <pre><code>888 java265.com
999 test 

</code><strong>注意事项:
   当形式参数和实例变量名称不同时，则无需使用this关键字，即可实现相应的功能</strong>
</pre>
    <h3>
     2. this：调用当前类方法
    </h3>
    <p>
     this关键字用于调用当前类的方法，当我们不使用this关键字时，编译器调用方法时，会自动添加this关键字
    </p>
    <h3 id="h2-3-this-">
     3. this()：调用当前类的构造函数
    </h3>
    <p>
     <code>
      this()
     </code>
     构造函数调用可以用来调用当前类的构造函数
    </p>
    <p>
     <strong>
      从参数化构造函数调用默认构造函数：
     </strong>
    </p>
    <pre><code>class testClass {
     testClass() {
        System.out.println("print testClass");
    }

    testClass(int x) {
        this(); //调用构造函数
        System.out.println(x);
    }

    public static void main(String args[]) {
        testClass a = new testClass(120);
    }
}</code></pre>
    <h4 id="h3--this-">
     例:
     <br/>
     使用this()构造函数调用
    </h4>
    <p>
     <code>
      this()
     </code>
     构造函数调用用于从构造函数重用构造函数。 它维护构造函数之间的链，即它用于构造函数链接。看看下面给出的示例，显示
     <code>
      this
     </code>
     关键字的实际使用。
    </p>
    <pre><code>class testClass {
    int keyId;
   String info1;
   String info2;
   String info3;

    testClass(int keyId, String info1, String info2) {
        this.keyId = keyId;
        this.info1 = info1;
        this.info2 = info2;
    }

  testClass(int keyId, String info1, String info2,String info3) { 
        this(keyId,info1,info2);//初始化构造函数
         this.info3 = info3;
    }

    void printInfo() {
        System.out.println(keyId + " " + info1 + " " + info2 + " " + info3);
    }
 
    public static void main(String args[]) {
        testClass s1 = new testClass(888, "java265", "java265-1");
        testClass s2 = new testClass(999, "java265", "java265-1","java265-2");
        s1.printInfo();
        s2.printInfo();
    }
}
</code></pre>
    <p>
     <strong>
      注意事项:
      <br/>
      调用this()必须是构造函数中的第一行代码
     </strong>
    </p>
    <p>
    </p>
    <h3 id="h2-4-this-">
     4. this：作为参数传递给方法
    </h3>
    <p>
     <code>
      this
     </code>
     关键字作为方法的参数进行传递，
     <br/>
     例：
    </p>
    <pre><code>class testClass {
    void p(testClass obj) {
        System.out.println(testClass被创建");
    }

    void show() {
        p(this);
    }

    public static void main(String args[]) {
        testClass s1 = new testClass();
        s1.show();
    }
}
</code></pre>
    <h3>
     6. this关键字用来返回当前类的实例
    </h3>
    <p>
     可以从方法中
     <code>
      this
     </code>
     关键字作为语句返回。 在这种情况下，方法的返回类型必须是类类型(非原始)。 看看下面的一个例子：
    </p>
    <p>
     <strong>
      作为语句返回的语法
     </strong>
    </p>
    <pre><code>return_type method_name(){  
    return this;  
}
</code></pre>
    <p>
    </p>
    <p>
     <strong>
      从方法中返回为语句的 this 关键字的示例
     </strong>
    </p>
    <pre><code>class testClass {
    testClass getThis() {
        return this;
    }

    void printInfo() {
        System.out.println("Hello java265.com!");
    }
 
    public static void main(String args[]) {
        new testClass().getThis().printInfo();
    }
}
</code></pre>
    <p>
     ------运行以上代码，将输出以下信息----
     <br/>
     Hello java265.com
     <br/>
    </p>
