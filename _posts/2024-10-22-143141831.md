     <strong>
      <strong>
       什么是LLaMA-Factory？
       <strong>
        <strong>
         <strong>
          LLaMA-Factory，全称Large Language Model Factory，即大型语言模型工厂。它支持
         </strong>
        </strong>
        多种预训练模型和微调算法
       </strong>
      </strong>
      ，提供了一套完整的工具和接口，使得用户能够轻松地对
     </strong>
     <strong>
      预训练的模型进行定制化的训练和调整
     </strong>
     **，以适应特定的应用场景，如
     <strong>
      <strong>
       智能客服、语音识别、机器翻译
      </strong>
     </strong>
     等。**
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/af64bfaa99b88b5610002d6d4b6fd472.jpeg"/>
    </p>
    <p>
     <strong>
      <strong>
       <strong>
        <strong>
         <strong>
          LLaMA-Factory
         </strong>
        </strong>
       </strong>
      </strong>
     </strong>
    </p>
    <ol>
     <li>
      <p>
       支持的模型：LLaMA-Factory支持多种大型语言模型，包括但不限于LLaMA、BLOOM、Mistral、Baichuan、Qwen、ChatGLM等。
      </p>
     </li>
     <li>
      <p>
       集成方法：包括（增量）预训练、指令监督微调、奖励模型训练、PPO训练、DPO训练和ORPO训练等多种方法。
      </p>
     </li>
     <li>
      <p>
       运算精度与优化算法：提供32比特全参数微调、16比特冻结微调、16比特LoRA微调和基于AQLM/AWQ/GPTQ/LLM.int8的2/4/8比特QLoRA微调等多种精度选择，以及GaLore、DoRA、LongLoRA、LLaMA Pro、LoRA+、LoftQ和Agent微调等先进算法。
      </p>
     </li>
    </ol>

    <p>
     <strong>
      <strong>
       二、模型微调（Fine-Tuning）
      </strong>
     </strong>
    </p>
    <p>
     <strong>
      如何使用
     </strong>
     LLaMA-Factory
     <strong>
      进行模型微调？
      <strong>
       <strong>
        使用LLaMA-Factory进行模型微调是一个涵盖从
       </strong>
      </strong>
      选择模型、数据加载、参数配置到训练、评估优化直至部署应用****的全面且高效的流程。
     </strong>
    </p>
    <p>
     1. 选择模型：
     <strong>
      根据应用场景和需求选择合适的预训练模型。
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        设置语言：
       </strong>
       进入WebUI后，可以切换到中文（zh）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置模型：
       </strong>
       选择LLaMA3-8B-Chat模型。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置微调方法：
        <strong>
         微调方法则保持默认值
        </strong>
        lora
       </strong>
       ，使用LoRA轻量化微调方法能极大程度地节约显存。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      <strong>
       <img alt="" src="https://img-blog.csdnimg.cn/img_convert/68250d83045497498852e57cfa9bf915.jpeg"/>
      </strong>
     </strong>
    </p>
    <p>
     2. 加载数据：
     <strong>
      将准备好的数据集加载到LLaMA-Factory中。
     </strong>
    </p>
    <ul>
     <li>
      LLaMA-Factory项目内置了丰富的数据集，放在了
      <code>
       data
      </code>
      目录下。同时也可以自己准备自定义数据集，将数据处理为框架特定的格式，放到指定的
      <code>
       data目录下。
      </code>
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/f57e70c650bba83116c3c3b738c088ae.jpeg"/>
     </strong>
    </p>
    <p>
     3. 配置参数：
     <strong>
      根据实际情况调整学习率、批次大小等训练参数。
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        学习率+梯度累积：
       </strong>
       设置学习率为1e-4，梯度累积为2，有利于模型拟合。
      </p>
     </li>
     <li>
      <p>
       <strong>
        计算类型：
       </strong>
       如果是NVIDIA V100显卡，计算类型保持为fp16；如果使用了AMD A10系列显卡，可以更改计算类型为bf16。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/d28f381523d59255ffeb75b91c0666cb.jpeg"/>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       LoRA参数设置：
      </strong>
      设置LoRA+学习率比例为16，LoRA+被证明是比LoRA学习效果更好的算法。在LoRA作用模块中填写all，即将LoRA层挂载到模型的所有线性层上，提高拟合效果。
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/3c3cb183387033933d7ff0112d409ef4.jpeg"/>
     </strong>
    </p>
    <p>
     4. 开始训练：
     <strong>
      启动训练过程，并监控模型的训练进度和性能表现。
     </strong>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        输出目录
       </strong>
       ：将输出目录修改为train_llama3，训练后的LoRA权重将会保存在此目录中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        预览命令：
       </strong>
       点击「预览命令」可展示所有已配置的参数，如果想通过代码运行微调，可以复制这段命令，在命令行运行。
      </p>
     </li>
     <li>
      <p>
       <strong>
        开始：
       </strong>
       点击「开始」启动模型微调。
      </p>
     </li>
    </ol>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/faad3e0e6d4333035dc8eb1af6aa9da2.jpeg"/>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       训练完毕：
      </strong>
      启动微调后需要等待一段时间，待模型下载完毕后可在界面观察到训练进度和损失曲线。模型微调大约需要20分钟，显示“训练完毕”代表微调成功。
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/7d4826c809491514c15382bb27415904.jpeg"/>
     </strong>
    </p>
    <p>
     5. 评估与优化：
     <strong>
      使用LLaMA-Factory提供的评估工具对模型性能进行评估，并根据评估结果进行针对性的优化。
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        刷新适配器：
       </strong>
       微调完成后，点击页面顶部的「刷新适配器」
      </p>
     </li>
     <li>
      <p>
       <strong>
        适配器路径：
       </strong>
       点击适配器路径，即可弹出刚刚训练完成的LoRA权重，点击选择下拉列表中的train_llama3选项，在模型启动时即可加载微调结果。
      </p>
     </li>
    </ul>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/0ceb556a8ee39d2ebf013f64e0e805d4.jpeg"/>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        评估模型：
       </strong>
       选择「Evaluate&amp;Predict」栏，在数据集下拉列表中选择「eval」（验证集）评估模型。
      </p>
     </li>
     <li>
      <p>
       <strong>
        输出目录：
       </strong>
       更改输出目录为
       <code>
        eval_llama3
       </code>
       ，模型评估结果将会保存在该目录中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        开始评估：
       </strong>
       最后点击开始按钮启动模型评估。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/7cc794df0668d7f291f6b263aea2e6c5.jpeg"/>
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        评估分数：
       </strong>
       模型评估大约需要5分钟左右，评估完成后会在界面上显示验证集的分数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        ROUGE分数：
       </strong>
       其中ROUGE分数衡量了模型输出答案（predict）和验证集中标准答案（label）的相似度，ROUGE分数越高代表模型学习得更好。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/9a2a2644eb360e42734bc0aeef80c177.jpeg"/>
     </strong>
    </p>
    <p>
     6. 部署应用：
     <strong>
      将训练好的模型部署到实际应用场景中，实现其功能和价值。
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       加载模型：
      </strong>
      选择「Chat」栏，确保适配器路径是
      <code>
       train_llama3
      </code>
      ，点击「加载模型」即可在Web UI中和微调模型进行对话。
     </li>
    </ul>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/558c46336d5d1cba1e72883f0ef820a3.jpeg"/>
    </p>
    <ul>
     <li>
      <strong>
       <strong>
        卸载模型：
       </strong>
      </strong>
      点击「卸载模型」，点击“×”号取消适配器路径，再次点击「加载模型」，即可与微调前的原始模型聊天。
     </li>
    </ul>
    <p>
     <strong>
      <img alt="" src="https://img-blog.csdnimg.cn/img_convert/d1be843e664f2c8dbf7187b9c3237af0.jpeg"/>
     </strong>
    </p>
