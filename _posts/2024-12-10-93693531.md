    <h1 class="title-article" id="articleContentId">
     理解“正反馈”和“负反馈”，学会系统性思考
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>

    <div class="rich_media_content" id="js_content">
     <p style="font-size:14px;color:rgb(160,160,160);line-height:1.8;">
      <span style="font-family:'-apple-system-font', 'system-ui', Arial, sans-serif;">
       “如果干得好，管理是最崇高的职业之一。没有哪一个职业能像管理一样为他人提供学习和成长的机会，让他们懂得承担责任并取得成绩，以及为团队的成功做出贡献”
      </span>
     </p>
     <p style="font-family:'-apple-system-font', 'system-ui', Arial, sans-serif;text-align:right;">
      <span style="color:rgb(160,160,160);font-size:14px;text-align:justify;">
       《你要如何衡量你的人生》
      </span>
     </p>
     <p style="font-family:'-apple-system-font', 'system-ui', Arial, sans-serif;text-align:right;">
      <span style="color:rgb(160,160,160);font-size:14px;text-align:justify;">
       Clayton M. Christensen
      </span>
     </p>
     <p>
      <br/>
     </p>
     管理心得
     <p>
      <br/>
     </p>
     <p>
     </p>
     <p style="line-height:1.75em;">
      <span style="letter-spacing:2px;font-size:16px;">
      </span>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       无论是个人还是组织，在解决问题时，要懂得系统性地思考。学习系统思维，有一对很重要的概念，叫做“正反馈”和“负反馈”。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       正反馈，也叫做放大反馈，
      </span>
      <span style="font-size:16px;letter-spacing:2px;color:rgb(255,104,39);">
       <strong>
        正反馈过程是增长的引擎，是一个驱动系统加速发生变化的过程
       </strong>
      </span>
      <span style="font-size:16px;letter-spacing:2px;">
       ，正反馈回路也叫自增强回路。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       锻炼能够让人感觉良好，然后继续坚持锻炼，于是愈发感觉良好，这就是一个正反馈；做好团队分享能够活跃技术氛围，良好的技术氛围能够激发团队成员的积极性，积极性高了会更踊跃实践并输出更多高质量分享，这也是一个正反馈的过程；当然，正反馈不是指“正能量”，正反馈回路不只是产生良性循环，如果“系统加速发生的变化”是坏的方向，那正反馈回路也会造成恶性循环；比如情绪不好会使得工作质量变差，工作质量不好会产生更多坏情绪，然后有继续影响工作成果。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       只要处在增长的局面，就一定会有正反馈作用，团队管理要学会系统思考，识别正反馈中的“积极因素”以及“消极因素”。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       负反馈，也叫平衡反馈，“负”的意思是“减少”，而不是“负能量”，
      </span>
      <span style="font-size:16px;letter-spacing:2px;color:rgb(255,104,39);">
       <strong>
        负反馈是一个调节的过程，总是让系统回到“正轨”上来
       </strong>
      </span>
      <span style="font-size:16px;letter-spacing:2px;">
       。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       目标导向就是一个负反馈的过程，管理活动中只要发现团队行为或者活动走向偏离了目标，就会采取各种措施使其回到正确的方向。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       做出领导决策时，负反馈会比正反馈更难理解，造成负反馈的因素会更隐蔽，比如组织要进行变革，这其实就是做一个负反馈的过程，组织这个系统中会有很多因素让其回归平衡，阻碍的组织的调整和改变。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       正确认识了“正反馈”和“负反馈”，有助于我们进行系统性的思考，就不容易出现“头疼医头，脚疼医脚”的片面方案，
      </span>
      <span style="font-size:16px;letter-spacing:2px;color:rgb(255,104,39);">
       <strong>
        因为系统的反馈会存在延迟和滞后性，所以学会找出系统关键因素，进行及时正确的反馈，减少滞后反馈带来的管理不确定性，是进行有效组织管理的技巧和手段。
       </strong>
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       有一本畅销书叫做《一分钟经理人》，就是讲到作为一个领导者，管理的关键是给出及时有效的的反馈。一分钟管理法的整个流程就是为了打造一个正确及时的反馈系统，分为三个部分：一分钟目标、一分钟称赞、一分钟更正。
      </span>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       <br/>
      </span>
     </p>
     <p style="line-height:2em;">
      <strong>
       <span style="font-size:16px;letter-spacing:2px;">
        一分钟目标：
       </span>
      </strong>
      <span style="font-size:16px;letter-spacing:2px;">
       领导者要主动关注小伙伴的动向，确定他对目标的理解是否清晰，重点在于达成共识，双方都要清晰接下来要完成的具体事项；
      </span>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       <br/>
      </span>
     </p>
     <p style="line-height:2em;">
      <strong>
       <span style="font-size:16px;letter-spacing:2px;">
        一分钟称赞：
       </span>
      </strong>
      <span style="font-size:16px;letter-spacing:2px;">
       领导者不要碍于面子，觉得称赞别人会不好意思，称赞是一种正向反馈，会激发小伙伴的内在动力，当然，称赞的方式不要太虚伪，要说具体事实，做得好是哪里做得好？大家都是成年人，都能看到对方的诚恳；
      </span>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       <br/>
      </span>
     </p>
     <p style="line-height:2em;">
      <strong>
       <span style="font-size:16px;letter-spacing:2px;">
        一分钟纠正：
       </span>
      </strong>
      <span style="font-size:16px;letter-spacing:2px;">
       小伙伴事情做得不好，要及时去引导，关键在于对事不对人，要懂得区分人的行为和人的自身价值，要有“事情没做好，双方都有责”的心态，说白了，好的领导者遇事要懂得反求诸己，出问题先从自身找起，客观对事，这样才能说服别人，做好引导；
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="line-height:2em;">
      <span style="font-size:16px;letter-spacing:2px;">
       以上就是我今天的分享，其实做这个分析主要是基于最近的两个思考，一个是为什么有些人、有些公司和产品能做的越来越好，另一个是团队管理中怎么应对变革，这两个问题都需要进行系统性的思考，我希望对大家也能有所启发。
      </span>
     </p>
