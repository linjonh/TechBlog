    <p>
     现在云技术高速发展。一些大型开发公司不仅在提供云的平台服务，同时提供云平台的API服务。我们的研发团队现在也很需要在云上封装 云 API的技术力量。
    </p>
    <p>
     这个职位的级别高（13级，普通开发者是10-11级）（意思那就是内什么很高）。如果您有您周围的人有兴趣，请简历我。 谢谢！
    </p>
    <p>
     <br/>
     Job Posting – API Designer
    </p>
    <p>
     <br/>
     <br/>
     This position is for an API Designer to join a team for building the object model of the next generation platform for DLS and Autodesk products.
     <br/>
     <br/>
     <br/>
     Job Title: API Designer
     <br/>
     Location: Shanghai
     <br/>
     <br/>
     <br/>
     Responsibilities:
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Collaborate with diverse teams of application end-users, developers, QA, and product management at different sites globally.
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Understand, collect and validate requirements and needs with application end users/customers.
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Design and write specifications for object model, and guide the development team in object model implementation.
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Write object model testing plug-in, and guide QA team to design object model test cases.
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Co-work with the client application teams on the related object model design, documentation and consultant.
     <br/>
     <br/>
     <br/>
     Requirements:
     <br/>
     •
     <span style="white-space:pre">
     </span>
     BS or higher in Mechanical Engineering or related engineering discipline
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Minimum 8 year experience as CAD application end user or related experience
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Excellent customer interaction and interpersonal skills
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Strong technical leadership skills
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Strong analytical capability and object modeling capability
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Strong English communication, in both written and oral
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Experienced in scripting programming and application plug-in development
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Experience in documentation for end-users
     <br/>
     <br/>
     <br/>
     Preferred Qualifications:
     <br/>
     •
     <span style="white-space:pre">
     </span>
     End user experience in CAM/CAE applications
     <br/>
     •
     <span style="white-space:pre">
     </span>
     Work experience on Mac platform
    </p>
