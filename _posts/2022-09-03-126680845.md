     前言

    <h3>
     <a id="_4">
     </a>
     效果
    </h3>
    <p>
     <mark>
      在服务器开机的情况下
     </mark>
     ，无论是在电脑还是手机，在浏览器输入自己搭建的网站名（公网），然后登录账号，即可使用。👀
    </p>
    <p>
     <strong>
      电脑：
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b1b108908c938b311a1b4ce70a3e6dea.png"/>
    </p>
    <p>
     <strong>
      手机：
     </strong>
     <br/>
     相册页面：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/73b042eac3891e5c2aefe7a230107351.png"/>
    </p>
    <p>
     相片页面：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4c10159995e2b3b5b1070ac880aec049.png"/>
    </p>
    <p>
     手机和电脑均可上传相片/创建相册（手机还能拍照上传相片）
    </p>
    <p>
     由于部署在服务器，运行速度很快，手机电脑会同步，而且功能齐全，太赞了👍
    </p>
    <h3>
     <a id="Lychee_24">
     </a>
     Lychee(荔枝)简介
    </h3>
    <p>
     一个免费、美观且易于使用的照片管理系统。
    </p>
    <p>
     <a href="https://github.com/LycheeOrg/Lychee">
      github传送门
     </a>
    </p>
    <h3>
     <a id="_29">
     </a>
     准备：一台服务器
    </h3>
    <p>
     要在服务器上创建相册，前提是必须有一台服务器。
    </p>
    <p>
     如已有服务器可以跳过这一步，如果没有可以参考接下来的教程，将会手把手教你搭建服务器：
    </p>
    <h2>
     <a id="1_37">
     </a>
     1.注册账号并选择配置服务器
    </h2>
    <p>
     这里选择性价比较高的
     <a href="https://curl.qcloud.com/pTv4r0NB" rel="nofollow">
      腾讯云服务器
     </a>
     👈
    </p>
    <p>
     🌟 新用户可以选择
     <strong>
      新人·专区
     </strong>
     ，折扣比较高。
    </p>
    <p>
     ❗️ 注意现在购买服务器都是必须
     <strong>
      实名注册
     </strong>
     。
    </p>
    <p>
     ⚡️ 注册完腾讯云账号后，可以根据自己的需要购买服务器。这里推荐使用
     <strong>
      轻量应用服务器
     </strong>
     。（轻量应用服务器 Lighthouse 是一种简单易用且适合运行轻量级应用的云服务器，具备入门简单、便捷高效、性价比高等优势，能帮助用户快速部署、配置和管理各类应用和开发测试环境，适用于快速建站等场景）
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7f1c040475ed747eeffd144e1cc9670f.png"/>
    </p>
    <p>
     服务器配置要求不用太高，
     <strong>
      2核2G4M
     </strong>
     （40GB SSD盘 300GB月流量 4M带宽）就够了。
     <br/>
     当然如果想要好一点的配置也是可以滴！上传的照片会占用系统盘（40GB SSD盘 云硬盘）的空间的。
     <br/>
     地域选择 北京/上海/广州 都可以，镜像选择
     <strong>
      宝塔Linux面板7.9.3腾讯云专享版
     </strong>
     。
     <br/>
     我这边用的是 2核4G6M（60GB SSD云硬盘）的轻量服务器，操作方法和 2核2G4M 的是相同的。
    </p>

    <h2>
     <a id="3_85">
     </a>
     3.添加服务器端口规则
    </h2>
    <p>
     首次访问上面的外网是没有响应的，因为默认浏览器访问的是 80 端口，宝塔Linux面板端口默认为
     <strong>
      8888
     </strong>
     ，需要添加8888端口规则。
    </p>
    <p>
     回到服务器面板，找到
     <strong>
      防火墙
     </strong>
     页面，点击
     <strong>
      添加规则
     </strong>
     ：
     <br/>
     <strong>
      限制来源
     </strong>
     勾选
     <strong>
      启用
     </strong>
     ，
     <strong>
      源IP地址
     </strong>
     填写
     <strong>
      0.0.0.0/0
     </strong>
     ，协议选择
     <strong>
      TCP
     </strong>
     ，端口选择
     <strong>
      8888
     </strong>
     ，策略选择
     <strong>
      允许
     </strong>
     ，备注写什么都行，最后点
     <strong>
      确定
     </strong>
     。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/efa6bffb823e8835babfa44d960ce90c.png"/>
    </p>
    <p>
     显示自定义的规则表示添加成功：

     选择编译安装
     <strong>
      LNMP
     </strong>
     ，然后点击一键安装，等待一段时间。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/946bfef512fdf1ec428e7c94e53ddc7d.png"/>
     <br/>
     二十多分钟安装好。中途面板退出了出现“请使用正确的入口登录面板”，重新进入外网地址然后登录就可以了，安装进程会继续。
    </p>

    <h2>
     <a id="7lychee_164">
     </a>
     7.安装lychee(荔枝)
    </h2>
    <p>
     回到服务器，借助 docker 下载 lychee 。
    </p>
    <p>
     <strong>
      （1）拉取lychee镜像：
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> pull lycheeorg/lychee
</code></pre>
    <p>
     如果出现报错：
    </p>
    <pre><code class="prism language-bash">Using default tag: latest
Got permission denied <span class="token keyword">while</span> trying to connect to the Docker daemon socket at unix:///var/run/docker.sock: Post <span class="token string">"http://%2Fvar%2Frun%2Fdocker.sock/v1.24/images/create?fromImage=lycheeorg%2Flychee&amp;tag=latest"</span><span class="token builtin class-name">:</span> dial unix /var/run/docker.sock: connect: permission denied
</code></pre>
    <p>
     在用户权限下docker 命令需要 sudo ，在命令前面加一个 sudo 就可以了：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">docker</span> pull lycheeorg/lychee
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a71bf50a2b7195beb8f85ebdedffd1e.png"/>
    </p>
    <p>
     <strong>
      （2）启动lychee镜像，在指定端口运行（端口映射：浏览器默认的80端口）
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> run -it -d -p <span class="token number">5120</span>:80 lycheeorg/lychee
</code></pre>
    <p>
     如果遇到报错：
    </p>
    <pre><code class="prism language-bash">docker: Got permission denied <span class="token keyword">while</span> trying to connect to the Docker daemon socket at unix:///var/run/docker.sock: Post <span class="token string">"http://%2Fvar%2Frun%2Fdocker.sock/v1.24/containers/create"</span><span class="token builtin class-name">:</span> dial unix /var/run/docker.sock: connect: permission denied.
See <span class="token string">'docker run --help'</span><span class="token builtin class-name">.</span>
</code></pre>
    <p>
     同样也是在前面加个 sudo 就可以了：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">docker</span> run -it -d -p <span class="token number">5120</span>:80 lycheeorg/lychee
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/12cb315ec468c2713f05a3739737e935.png"/>
    </p>
    <p>
     回到面板的软件商店：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ac7a082d92d95f31162de05fe1607dd.png"/>
     <br/>
     将会出现：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8e3602c5e3ef62980686bac0abcba6ad.png"/>
    </p>
