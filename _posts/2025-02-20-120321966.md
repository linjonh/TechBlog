    <p>
     1.嵌入式系统
     <br/>
     无OS(单片机)----简单OS—实时OS----和Internet结合
     <br/>
     2.组成（体系结构）
     <br/>
     嵌入式硬件平台+嵌入式OS+嵌入式应用软件
    </p>
    <p>
     RISC+CISC（可回顾计组）
     <br/>
     实验箱的主CPU:I.MAX6
     <br/>
     从CPU:STM32F103
    </p>
    <p>
     要接触的嵌入式OS：
     <br/>
     嵌入式linux，android
    </p>
    <p>
     嵌入式系统设计三阶段：
     <br/>
     1.分析
     <br/>
     2设计
     <br/>
     3.实现
    </p>
    <p>
     嵌入式系统开发流程
     <br/>
     选择嵌入式处理器（硬件平台） （主要是ARM处理器）
     <br/>
     选择嵌入式OS（软件平台）
     <br/>
     开发嵌入式应用软件
     <br/>
     系统测试
    </p>
    <p>
     嵌入式软件开发流程：
     <br/>
     写代码
     <br/>
     交叉编译
     <br/>
     交叉链接
     <br/>
     下载到目标板
     <br/>
     debug
    </p>
    <p>
     交叉开发：在环境1上开发能在环境2上运行的软件。
     <br/>
     因为宿主机（PC机 x86）和目标机（实际运行环境 arm环境等等）环境不同
     <br/>
     交叉调试：调试器run在宿主机的OS上，被调试进程run在特定硬件平台的嵌入式OS中。
    </p>
