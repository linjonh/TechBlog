    <h1 class="title-article" id="articleContentId">
     使用ACT(Microsoft Application Center Test) 进行压力测试
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     ACT(Microsoft Application Center Test) 是微软Microsoft visual studio.Net 2003 自带的
     <span class="t_tag">
      性能测试
     </span>
     <span class="t_tag">
      工具
     </span>
     ，使用方便,可以进行简单的
     <span class="t_tag">
      压力
     </span>
     测试；它可以独立安装，不需要将Microsoft visual studio.Net 2003中所有
     <span class="t_tag">
      功能
     </span>
     都安装；参见附件ACT.bmp
     <br/>
     <strong>
      1 创建测试
     </strong>
     <br/>
     *点击图标或通过菜单Actions ——New
     <span class="t_tag">
      test
     </span>
     ,然后就是Next ;
     <br/>
     *如果不愿意写
     <span class="t_tag">
      脚本
     </span>
     ，就可以
     <span class="t_tag">
      选择
     </span>
     Record a new test 来
     <span class="t_tag">
      录制
     </span>
     整个操作过程，ACT会
     <span class="t_tag">
      自动
     </span>
     生成脚本，且识别率高，一般不用修改即可顺利回放；
     <br/>
     <strong>
      2 脚本回放前
      <span class="t_tag">
       设置
      </span>
     </strong>
     <br/>
     *回放前可以通过菜单Actions ——Properties 来设置
     <span class="t_tag">
      性能
     </span>
     的指：参见附件Properties.bmp，General 选项卡的Test load level 来设置并发数，Users选项卡一般不用设置，Counters选项卡 来设置Collections interval（连接的
     <span class="t_tag">
      时间
     </span>
     间隔）添加或删除性能计数器;
     <br/>
     <strong>
      3 脚本回放
     </strong>
     <br/>
     *点击图标或通过菜单Actions——Start test 即可；
     <br/>
     *参见附件Run test.bmp ,
     <span class="t_tag">
      运行
     </span>
     过程中可以显示
     <span class="t_tag">
      数据
     </span>
     包的发送和接收情况（HTTP,DNS及Socket
     <span class="t_tag">
      error
     </span>
     s ）；通过点击Show details可以看到运行中的曲线图；
     <br/>
     <strong>
      4 查看测试
      <span class="t_tag">
       结果
      </span>
     </strong>
     <br/>
     * 点击左侧树型节点Results,可以查看测试结果；
    </p>
    <p>
    </p>
    <p>
     以下为原文的注解
    </p>
    <p>
     <span style="font-size: x-small;">
      <span style="font-size: large;">

       <span class="t_tag">
        软件
       </span>
       的适用范围小，不舒畅，现在使用WAS. 一个跟这个工具类似的软件。
      </span>
     </span>
    </p>
