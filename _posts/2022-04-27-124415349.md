    <h4>
     4. 配置我们自己项目的pages.json，值得注意的是，你在哪个页面进行语音视频通话的需求，就在哪个页面进行引入，注意目录路径，单词拼写，千万别整错了！

    <h4>
     5. 如何使用?
    </h4>
    <blockquote>
     <p>
      我先简单说一下这个demo的场景，模拟了两个人，需要两部手机， 一人为普通用户，一人为客服人员；这两个人可以进行视频或者是语音通话！
     </p>
    </blockquote>
    <p>
     首先进入应用选择，自己是什么角色
    </p>
    <p>
     index.vue
    </p>
    <pre><code>&lt;template&gt;
	&lt;view class="content"&gt;
		&lt;button type="primary" @click="setUser"&gt;我是用户&lt;/button&gt;
		&lt;button type="primary" plain @click="setCmr"&gt;我是客服&lt;/button&gt;
	&lt;/view&gt;
&lt;/template&gt;

&lt;script&gt;
import { mapMutations } from 'vuex';
export default {
	methods: {
		...mapMutations(['setUserInfo']),
		setUser() {
			this.setUserInfo('user');
			uni.redirectTo({
				url: '/pages/calling/calling?userID=cmr'
			});
		},
		setCmr() {
			this.setUserInfo('cmr');
			uni.redirectTo({
				url: '/pages/calling/calling?userID=wxy'
			});
		}
	}
};
&lt;/script&gt;

&lt;style lang="scss"&gt;
.content {
	height: 100vh;
	display: flex;
	align-items: center;
	justify-content: center;
}
&lt;/style&gt;</code></pre>
    <p>
     选择完成之后，我们把我们自己的用户数据设置在vuex里面
    </p>
    <pre><code>import Vue from 'vue';
import Vuex from 'vuex';


Vue.use(Vuex)

const userObj = {
	user: {
		userID: 'wxy',
		userSig: `aaa示例数据`
	},
	cmr: {
		userID: 'cmr',
		userSig: `bbb示例数据`
	}
}
const store = new Vuex.Store({
	state: {
		userInfo: {}
	},
	mutations: {
		setUserInfo(state, identity) {
			state.userInfo = userObj[identity]
		}
	}
})

export default store
</code></pre>
    <p>
     这个时候，可能得问，这个userID，userSig是从何而来的，我指条路，因为这种牵扯到 用户账号的私密性，你也知道，这种数据是不会放在前台的，得从自己的服务器请求得到，但是因为是示例，所以就直接写死在前台了，测试数据从那里来的话，看下面：
    </p>
    <p>
     <a href="https://console.cloud.tencent.com/trtc/usersigtool" rel="nofollow" title="登录 - 腾讯云">
      登录 - 腾讯云
     </a>
    </p>
    <p>
     <img alt="" height="287" src="https://i-blog.csdnimg.cn/blog_migrate/972de054b5975151bedabd1931dc7b15.png" width="661"/>
    </p>
    <p>
     自己去生成两个测试的账号就可以了！
    </p>
    <h4>
     <strong>
      最后,拨打通话的页面 calling.vue
     </strong>
    </h4>
    <p>
     config中的sdkAppId，是在这里获得的，创建应用之后，列表里面就会有这个sdkAppId
    </p>
    <p>
     <img alt="" height="330" src="https://i-blog.csdnimg.cn/blog_migrate/d01af34cf7aa0fc79e2bf14c05b4add9.png" width="553"/>
    </p>
    <pre><code>&lt;template&gt;
	&lt;view class="content"&gt;
		&lt;tuicalling ref="TUICalling" id="TUICalling-component" :config="config"&gt;&lt;/tuicalling&gt;
		&lt;view style="width: 100vw;text-align: center;font-size: 40rpx;"&gt;当前身份：{<!-- -->{ config.userID === 'cmr' ? '客服人员' : '普通用户' }}&lt;/view&gt;
		&lt;button type="primary" @click="voiceCall"&gt;语音通话&lt;/button&gt;
		&lt;button type="warn" @click="videoCall"&gt;视频通话&lt;/button&gt;
	&lt;/view&gt;
&lt;/template&gt;

&lt;script&gt;
import { mapState } from 'vuex';
export default {
	data() {
		return {
			userID: '',
			config: {
				sdkAppID: 999, // 开通实时音视频服务创建应用后分配的 SDKAppID
				userID: '', // 用户 ID，可以由您的帐号系统指定
				userSig: '' // 身份签名，相当于登录密码的作用
				// type: 2 //1.语音  2.视频
			},
			TUICalling: null
		};
	},
	onLoad({ userID }) {
		this.userID = userID;
		this.config.userID = this.userInfo?.userID;
		this.config.userSig = this.userInfo?.userSig;
	},
	onReady() {
		this.TUICalling = this.$refs.TUICalling;
		this.TUICalling.init();
	},
	onUnload() {
		this.$refs.TUICalling.destroyed();
	},
	computed: {
		...mapState(['userInfo'])
	},
	methods: {
		voiceCall() {
			this.TUICalling.call({
				userID: this.userID,
				type: 1
			});
		},
		videoCall() {
			this.TUICalling.call({
				userID: this.userID,
				type: 2
			});
		}
	}
};
&lt;/script&gt;

&lt;style&gt;
.content {
	height: 100vh;
	display: flex;
	flex-wrap: wrap;
	align-items: center;
	justify-content: center;
}
&lt;/style&gt;
</code></pre>
