    <h3>
     <a id="_220">
     </a>
     蛇的移动
    </h3>
    <p>
     这部分主要设计蛇的移动，
     <strong>
      拆分
     </strong>
     来看，要计算出蛇的
     <strong>
      蛇头和蛇尾
     </strong>
     的位置。
    </p>
    <p>
     这段代码定义了一个名为 moveSnake 的函数，用于控
     <strong>
      制蛇的移动
     </strong>
     。
    </p>
    <p>
     1.首先获取
     <strong>
      蛇尾部
     </strong>
     的坐标，以便在蛇吃到食物时使用。蛇尾总是在
     <strong>
      数组下标 -1
     </strong>
     的位置。
    </p>
    <p>
     2.通过循环移动蛇身体的每个部分，通过设置每个部分的
     <strong>
      top 和 left 样式
     </strong>
     来实现移动效果。也就是
     <strong>
      定位蛇身体每个部分的坐标
     </strong>
     ，进而实现移动的效果。
    </p>
    <p>
     3.根据移动方向
     <strong>
      更新蛇头
     </strong>
     的坐标。根据 direction 的值，
     <strong>
      增加或减少 snakeX 或 snakeY
     </strong>
     的值来改变蛇的位置。这里配合之前
     <strong>
      控制方向的函数
     </strong>
     实现。
    </p>
    <p>
     4.检查蛇是否超出游戏边界，如果超出则调用 gameOver 函数来结束游戏。这里是在后面
     <strong>
      游戏结束函数
     </strong>
     中定义。
    </p>
    <p>
     5.更新蛇头的位置，通过设置蛇头元素的
     <strong>
      top 和 left 样式来
     </strong>
     改变蛇头的位置。蛇头总是在
     <strong>
      数组为0
     </strong>
     的位置。
    </p>
    <p>
     6.检查蛇是否吃到食物，如果
     <strong>
      蛇头的坐标与食物的坐标相同
     </strong>
     ，则表示蛇吃到食物。随机生成新的食物位置，并创建一个新的蛇身体部分，将其
     <strong>
      添加到蛇身体数组中
     </strong>
     。最后，更新得分。
    </p>
    <p>
     7.继续移动蛇，设置下一个定时器，通过 setTimeout(moveSnake, speed) 来实现
     <strong>
      蛇的连续移动。
     </strong>
    </p>
    <pre><code>var moveSnakeTimeout = setTimeout(moveSnake, speed); //这行代码用于设置定时器，控制蛇的移动速度。后面用到
// 移动蛇的函数
function moveSnake() {
    // 获取蛇尾部的坐标
    var tailX = snakeBody[snakeBody.length - 1].offsetLeft;
    var tailY = snakeBody[snakeBody.length - 1].offsetTop;

    // 移动蛇身体的每个部分
    for (var i = snakeBody.length - 1; i &gt; 0; i--) {
        snakeBody[i].style.top = snakeBody[i - 1].style.top;
        snakeBody[i].style.left = snakeBody[i - 1].style.left;
    }

    // 根据移动方向更新蛇头的坐标
    if (direction === "right") {
        snakeX += 20;
    } else if (direction === "left") {
        snakeX -= 20;
    } else if (direction === "up") {
        snakeY -= 20;
    } else if (direction === "down") {
        snakeY += 20;
    }

    // 检查蛇是否超出游戏边界
    if (snakeX &lt; 0 || snakeX &gt;= 400 || snakeY &lt; 0 || snakeY &gt;= 400) {
        // 游戏结束
        gameOver();
        return;
    }

    // 更新蛇头的位置
    snakeBody[0].style.top = snakeY + "px";
    snakeBody[0].style.left = snakeX + "px";

    // 检查蛇是否吃到食物
    if (snakeX === foodX &amp;&amp; snakeY === foodY) {
        // 随机生成新的食物位置
        foodX = Math.floor(Math.random() * 20) * 20;
        foodY = Math.floor(Math.random() * 20) * 20;
        food.style.top = foodY + "px";
        food.style.left = foodX + "px";

        // 创建一个新的蛇身体部分
        var newSnakePart = document.createElement("div");
        newSnakePart.classList.add("snake");
        newSnakePart.style.top = tailY + "px";
        newSnakePart.style.left = tailX + "px";

        // 将新的蛇身体部分添加到蛇身体数组中
        snakeBody.push(newSnakePart);
        gameBoard.appendChild(newSnakePart);

        // 更新得分
        score++;
    }
 // 继续移动蛇，设置下一个定时器
    setTimeout(moveSnake, speed);
}

</code></pre>

function gameOver() {
    // 清除移动蛇的定时器
    clearTimeout(moveSnakeTimeout);

    var gameOverDiv = document.getElementById("game-over");

    <pre><code>// 检查蛇是否超出游戏边界
