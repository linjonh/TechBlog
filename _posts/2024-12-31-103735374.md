    <p>
     <strong>
      当我们 想用python3的opencv， 自己又装了ros，一运行就给你报个错，开心不
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d59c77b42525111c8dcaa3e66c6118fb.png">
      <br/>
      我们安装python3的cv2 用命令
     </img>
    </p>
    <pre><code class="prism language-powershell">pip3 install opencv<span class="token operator">-</span>python
</code></pre>
    <p>
     看了下，也安装成功了
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/61c2345df61e54e339ac0ada5a8a85fb.png"/>
    </p>
    <p>
     执行，还是报错
    </p>
    <p>
     打印下 python 的环境变量
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a90a3a1aeec2e2b44775156369d4402f.png"/>
    </p>
    <p>
     因为
     <code>
      /opt/ros/kinetic/lib/python2.7/dist-packages
     </code>
     这个路径 在
     <br/>
     <code>
      /home/utry/.local/lib/python3.5/site-packages
     </code>
     之前，按照字母排序的，这个导入模块的时候，每次都会先到`/opt/ros/kinetic/lib/python2.7/dist-packages 这里导入，这是因为在安装ros的时候添加的环境变量
    </p>
    <h4>
     <a id="_20">
     </a>
     有三个解决方案
    </h4>
    <ul>
     <li>
      有个很浪的方案，就是把ros自带的cv2 给删除掉，一劳永逸， 不能删除那个环境变量，不然ros会有大问题
     </li>
     <li>
      删掉ros 用 python3 装，传送门
      <a href="https://www.jianshu.com/p/5f60d11a1a86" rel="nofollow">
       https://www.jianshu.com/p/5f60d11a1a86
      </a>
      太慢了，太费事
     </li>
     <li>
      代码里加一段，调整下位置就行了， 我把python3.5 的环境变量整最前面去了
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token keyword">import</span> sys
<span class="token comment"># ----------------------</span>
<span class="token comment"># 消除ros自带opencv 对环境的影响,自带的是2.7 我们用的3.5</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>sys<span class="token punctuation">.</span>path<span class="token punctuation">)</span>
list_path <span class="token operator">=</span> sys<span class="token punctuation">.</span>path  <span class="token comment"># type:list</span>
<span class="token keyword">for</span> index <span class="token keyword">in</span> list_path<span class="token punctuation">:</span>
    <span class="token keyword">if</span> <span class="token string">'.local/lib/python3.5/site-packages'</span> <span class="token keyword">in</span> index<span class="token punctuation">:</span>
        list_path<span class="token punctuation">.</span>remove<span class="token punctuation">(</span>index<span class="token punctuation">)</span>
        list_path<span class="token punctuation">.</span>insert<span class="token punctuation">(</span><span class="token number">0</span><span class="token punctuation">,</span> index<span class="token punctuation">)</span>
        <span class="token keyword">break</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>sys<span class="token punctuation">.</span>path<span class="token punctuation">)</span>
<span class="token comment"># --------------------------</span>
</code></pre>
