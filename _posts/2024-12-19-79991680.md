    <p>
     1.使用pip安装OpenCV
    </p>
    <p>
     直接安装最新版：
    </p>
    <pre><code class="language-python">pip install opencv_python</code></pre>
    <p>
     注：有可能不好使，你可以上官网找OpenCV都有哪个版本
    </p>
    <p>
     然后（1）
    </p>
    <pre><code class="language-python">pip install opencv_python==版本号</code></pre>
    或者（2）直接
    <pre><code class="language-python">pip install opencv_python==随机写一个版本号</code></pre>
    那……它就会把所有存在的版本号都列出来，你找到你想要的，然后（1）就好了。
    <br/>
    <br/>
    <p>
     2.测试
    </p>
    <pre><code class="language-python">from cv2.cv2 import VideoCapture,imshow,waitKey,destroyAllWindows

# 创建摄像头对象
# 使用opencv自带的VideoCapture()函数定义摄像头对象，其参数0表示第一个摄像头，
# 一般就是笔记本的内建摄像头。

cap = VideoCapture(0)
while(1):
    ret,frame = cap.read()
    imshow("capture",frame)
    if waitKey(1) &amp; 0xFF == ord('q'):
        break
cap.release()
destroyAllWindows()</code></pre>
    <p>
     3.注意
    </p>
    <p>
     版本不同在调用CV2时的方法也略有不同，我目前遇到的有两种
    </p>
    <pre><code class="language-python">import cv2
from cv2.cv2 import *</code></pre>
