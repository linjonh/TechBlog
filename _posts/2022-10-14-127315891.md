    <h2>
     <a id="_source_0">
     </a>
     数据库导入 source指令
    </h2>
    <h4>
     <a id="cmd_2">
     </a>
     一、cmd命令提示符界面下的数据库文件导入
    </h4>
    <p>
     <strong>
      1. win+r打开运行窗口 输入cmd打开命令提示符 并登录mysql
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/acb2a249d638fda36be49409e23fe4f0.png">
      <br/>
      <br/>
      <br/>
      <strong>
       2. 输入source指令 || 格式：source 数据库文件的路径(可以直接用鼠标拖进去 会自动生成)
       <br/>
       tip：路径容易输入错误的朋友
       <mark>
        就到需要导入的.sql文件的目录下 然后拖鼠标拖进去
       </mark>
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7236b3dfcd54e867f391e272e9bc70e0.png">
       <br/>
       <br/>
       <br/>
       <strong>
        注：在可视化数据库软件中查询语句source指令会报错（这里以Navicat为例）
        <br/>
        <mark>
         所以source指令只能在cmd命令提示符里面登录mysql使用
        </mark>
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/25e082a60b7c746429bdf4202bdbc812.png">
        <br/>
        <br/>
        <br/>
       </img>
      </img>
     </img>
    </p>
    <h4>
     <a id="Navicat16_15">
     </a>
     二、数据库可视化软件下的数据库文件导入（此处以Navicat16为例）
    </h4>
    <p>
     <strong>
      1. Navicat右击localhost 选择运行SQL文件
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/aa180877c2563ab9c57291ad37d734d2.png">
      <br/>
      <br/>
     </img>
    </p>
    <p>
     <strong>
      2. 点击文件最右边这个 … 选择.sql文件的路径 导入即可
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c393350fbcf571cad9f2f2574ba19edf.png">
      <br/>
      <br/>
      <br/>
      <strong>
       3. 然后刷新界面 即可
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dd0870a2e458fe075ccca265e869600b.png"/>
     </img>
    </p>
