    <p>
     一、python 安装
    </p>
    <p>
     tensorflow最适合的版本是python3.5，而ubuntu14.04系统会自带python2.7，
     <strong>
      请不要卸载它
     </strong>
     。不同版本的python可以共存在一个系统上。卸载之后，桌面系统会被影响。
    </p>
    <p>
     1、安装python3.5
    </p>
    <p>
     <code>
      （1）sudo add-apt-repository ppa:fkrull/deadsnakes   #
     </code>
     <strong>
      PPA
     </strong>
     （Personal Package Archives）个人软件包集
    </p>
    <p>
     <code>
      （2）sudo apt-get update
     </code>
    </p>
    <p>
     <code>
      （3）sudo apt-get install python3.5
     </code>
    </p>
    <p>
     （4）sudo cp /usr/bin/python /usr/bin/python_bak，先备份
     <br/>
     （5）sudo rm /usr/bin/python，删除
     <br/>
     （6）sudo ln -s /usr/bin/python3.5 /usr/bin/python，默认设置成python3.5，重建软链接这样在终端中输入python默认就是 3.5版本了
    </p>
    <p>
     2、然后输入python时启动的是python2.7
     <br/>
     输入python3时启动的是python3.4
     <br/>
     输入python3.5时启动的才是python3.5
    </p>
    <p>
     3、使用以下命令即可更改python默认为3.5
    </p>
    <pre class="has"><code>sudo update-alternatives --install /usr/bin/python python /usr/bin/python2 100  
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 150  
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.5 200  </code></pre>
    <p>
     4、取消原本的 Python 3.4 ，并将 Python3 链接到最新的 3.5 上：
    </p>
    <pre class="has"><code>sudo mv /usr/bin/python3 /usr/bin/python3-old  
sudo ln -s /usr/bin/python3.5 /usr/bin/python3  </code></pre>
    <p>
     5、安装新版pip：
    </p>
    <pre class="has"><code>wget https://bootstrap.pypa.io/get-pip.py  
sudo python3 get-pip.py  
sudo pip3 install setuptools --upgrade  
sudo pip3 install ipython[all] </code></pre>
    <p>
     6、切换回来链接文件：
    </p>
    <pre class="has"><code>sudo rm /usr/bin/python3  
sudo mv /usr/bin/python3-old /usr/bin/python3  </code></pre>
