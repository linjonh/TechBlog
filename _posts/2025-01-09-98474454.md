    <h3>
     <a id="python_0">
     </a>
     本人是才开始学习python的小白一枚，以后在学习的过程中有些什么问题如果有时间会记录下来。
    </h3>
    <p>
     今天用windows安装下载了python3.6版本，以后打算用Visual Studio Code编辑器来编写python代码。
     <br/>
     python下载链接如下：
     <br/>
     <a href="https://www.python.org/downloads/windows/" rel="nofollow">
      https://www.python.org/downloads/windows/
     </a>
     <br/>
     首先打开VS Code选择好刚才下载好的python
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/355b95721a301b8481e010e27c7c9184.png">
      然后在商城搜索并安装python插件：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ebbd352f781299aa9bb05fd008da41cc.png">
       然后就可新建文件开始写代码，这时我发现代码不可自动补全，以下是解决办法：
       <br/>
       1.使用快捷键组合 Ctrl+Shift+p ，选中首选项的Open User Setting模式设置界面(没有的话输入setting就会弹出)
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/046c75b05963d11b1724cbf526298be6.png">
        在打开的Open User Settings文件界面，搜索：python.autoComplete.addBrackets
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3a651169f650286378f5882d5a46b80.png">
         在用户设置里面将代码后面的False 改为 True 即可（注意，有的是直接勾选，我的打开需要在右边的用户设置代码里面点击修改为true ）
         <br/>
         然后关闭重启则生效。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b0449c9a752b3a06378cae1d8454058c.png"/>
        </img>
