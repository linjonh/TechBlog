    <h4 id="webdriver-element-locator">
     WebDriver Element Locator
    </h4>
    <h5 id="安装">
     安装
    </h5>
    <ol>
     <li>
      打开firefox浏览器，进入网址
      <a href="https://addons.mozilla.org/en-US/firefox/" rel="nofollow">
       https://addons.mozilla.org/en-US/firefox/
      </a>
     </li>
     <li>
      在搜索框里输入WebDriver Element Locator
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151113154357517" title=""/>
     </li>
     <li>
      点击Add to firefox
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151113154847638" title=""/>
     </li>
     <li>
      会有个弹出框，点击install now
     </li>
     <li>
      可以看从firefox浏览器的menu -&gt; Tools -&gt; Add ons -&gt; Extension里看是否安装成功
     </li>
    </ol>
    <h5 id="使用">
     使用
    </h5>
    <p>
     1.在firefox里打开www.baidu.com
     <br/>
     2. 将光标定位在搜索框里，鼠标右键，如下图所示，会提供多个语言(C#,java,Python,Ruby等)的xpath。
     <br/>
     找到对应自己语言的xpath就可以使用了
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151113155856305" title=""/>
    </p>
    <h4 id="firebug">
     Firebug
    </h4>
    <p>
     这个插件大家一定不陌生，可以用了干很多事情，比如调试javascript等等。
     <br/>
     还有一个作用就是定位页面元素，这样你可以清晰的看到某个元素的id,class等等
     <br/>
     - 从
     <a href="https://addons.mozilla.org/en-US/firefox/addon/firebug/" rel="nofollow">
      https://addons.mozilla.org/en-US/firefox/addon/firebug/
     </a>
     里找到你的firefox浏览器对应的firebug的版本
     <br/>
     -
     <a href="https://getfirebug.com/downloads" rel="nofollow">
      https://getfirebug.com/downloads
     </a>
     ，下载并安装
     <br/>
     - 安装完后，你的firefox浏览器会有这个图标
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151020172439321" title=""/>
    </p>
    <h4 id="firepath">
     firepath
    </h4>
    <h5 id="安装-1">
     安装
    </h5>
    <ol>
     <li>
      下载安装：
      <a href="https://addons.mozilla.org/en-us/firefox/addon/firepath/" rel="nofollow">
       https://addons.mozilla.org/en-us/firefox/addon/firepath/
      </a>
     </li>
     <li>
      firepath需要结合firebug插件一起使用。安装完并重启浏览器后，点击firebug，在打开的firebug 窗口就能看到firepath安装成功。
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151116115657060" title=""/>
     </li>
    </ol>
    <h5 id="使用-1">
     使用
    </h5>
    <p>
     通过firebug+firepath可以得到元素的xpath和css path。
    </p>
    <h6 id="1-得到xpath">
     1. 得到xpath
    </h6>
    <p>
     1.1 打开firebug窗口，选择firepath tab。
     <br/>
     1.2 点击firebug窗口的inspector,然后选择要查看xpath的页面元素，xpath就会显示在firebug窗口，如下图
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151116120232033" title=""/>
    </p>
    <p>
     这种方法可以得到元素的相对xpath,如果想得到元素的绝对xpath，在FirePath旁边的小三角选择Generate Absolute xpath.
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151116120444413" title=""/>
    </p>
    <h6 id="2-得到css-path">
     2. 得到css path
    </h6>
    <p>
     方法和得到xpath一样，选择css就行，如下图所示
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20151116120659178" title=""/>
    </p>
    <h4 id="xpath-checker">
     Xpath Checker
    </h4>
    <p>
     这个插件用了检查你写的xpath是否正确。
     <br/>
     - 下载安装：
     <a href="https://addons.mozilla.org/zh-cn/firefox/addon/xpath-checker/" rel="nofollow">
      https://addons.mozilla.org/zh-cn/firefox/addon/xpath-checker/
     </a>
     <br/>
     - 安装完后，右键 -&gt; View Xpath， Xpath Checker就打开了
    </p>
