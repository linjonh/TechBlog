     <p>
      大家好，很高兴能够一起学习交流HarmonyOS应用开发。本节所演示的示例默认大家都已经安装了开发工具DevEco Studio并配置好了开发环境。
     </p>
    </blockquote>
    <h4>
     <a id="HelloHarmonyOS_4">
     </a>
     创建HelloHarmonyOS项目
    </h4>
    <p>
     打开DevEco Studio开发工具，在Welcome to DevEco左侧菜单栏中点击
     <strong>
      Create HarmonyOS Project
     </strong>
     ，创建一个项目。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3436fedf0d8bcf31a2e9e6c839c9466b.png#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cccc5e19ed204bf6df99780a7aa10fe6.png#pic_center">
       <br/>
       目前HarmonyOS应用开发仅开放了
       <strong>
        TV
       </strong>
       、
       <strong>
        Wearable
       </strong>
       、
       <strong>
        Lite Wearable
       </strong>
       三种设备，而Lite Wearable体验为最优。本次以TV为例演示操作。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e73dff4422af2cf784978c63b8359096.png#pic_center">
        <br/>
        由于项目是使用Gradle自动化构建工具进行依赖管理，初次创建项目完成后，会在
        <strong>
         C:\Users\admin.gradle\wrapper\dists
        </strong>
        目录下找是否存在项目需要的Gradle版本，若不存在，则会进行下载。
       </img>
      </img>
     </img>
    </p>
    <p>
     一般情况，我们没有做任何代理的话，下载是非常慢的。这里我们将Gradle需要的版本下载到本地，然后拷贝到指定的目录，然后刷新Gradle会自动加载下载相关项。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/379831748495e3f12cb17479a98783a2.png#pic_center">
      <br/>
      ① 查看当前使用Gradle版本：
      <strong>
       File --&gt; Project Structure --&gt; Project --&gt; Basic Info --&gt; Gradle Version --&gt; 5.4.1
      </strong>
     </img>
    </p>
    <p>
     ② 下载
     <strong>
      gradle-5.4.1-all.zip
     </strong>
     ：https://services.gradle.org/distributions/
    </p>
    <p>
     ③ 将下载的gradle-5.4.1-all.zip拷贝到
     <strong>
      C:\Users\admin.gradle\wrapper\dists\gradle-5.4.1-all\生成的随机字符串\
     </strong>
    </p>
    <p>
     ④ 点击Gradle选项卡刷新按钮。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/66721aff03bcf40c802d25673c061b46.png#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6f4dee3c11cbb05a7bc59602eea79103.png#pic_center">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ffc30711212bba21f16951857555a439.png#pic_center"/>
      </img>
     </img>
    </p>

     <a id="HelloHarmonyOS_25">

     运行HelloHarmonyOS项目名
    </h4>
    <p>
     我们对代码先不做任何修改，启动Tools --&gt; HVD Manager，进行授权认证后启动模拟器。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7267a49ea776f12f718903e5e360c30d.png#pic_center"/>
