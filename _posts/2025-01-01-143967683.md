    <p>
     在深度学习中，归一化是数据预处理中一个至关重要的步骤。归一化技术通过调整输入数据的尺度，使得数据具有相似的分布范围，提高模型的求解速度和泛化能力。
    </p>
    <p>
     常用的归一化方法，包括批量归一化（Batch Normalization）、层归一化（Layer Normalization）、组归一化（Group Normalization）和实例归一化（Instance Normalization）等。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/4f111cf84a4b438089726504f9b6e5c4.png"/>
    </p>
    <h2>
     <a id="_10">
     </a>
     <strong>
      一、归一化
     </strong>
    </h2>
    <h4>
     <a id="1Normalization_12">
     </a>
     1、
     <strong>
      归一化（Normalization）是什么？
     </strong>
    </h4>
    <p>
     归一化是一种数据预处理技术，旨在将数据的数值范围缩放到一个特定的尺度，通常是为了消除不同特征之间的量纲差异，使得数据更加适合进行后续的分析和处理。
    </p>
    <p>
     归一化是深度学习中不可或缺的数据预处理步骤。通过合理的归一化处理，可以显著提高模型的训练效率和性能，为深度学习模型的优化和泛化能力的提升提供有力支持。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/b53df2f09d338aa140af43c316e4a178.png"/>
    </p>
    <h4>
     <a id="2_20">
     </a>
     2、
     <strong>
      为什么需要归一化？
     </strong>
    </h4>
    <p>
     由于目标函数拥有众多参数且结构复杂，直接寻找最优参数变得十分困难。因此，我们需要借助优化器，它能够逐步调整参数，确保每次优化都朝着最快降低损失的方向前进。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/5011e6b77126b35395f2104c554b82c2.png"/>
    </p>
    <h4>
     <a id="3_26">
     </a>
     3、
     <strong>
      归一化的过程是什么？
     </strong>
    </h4>
    <p>
     归一化是通过确定数据的取值范围，*应用相应的归一化公式将数据转换到新的稳定尺度，从而得到更加适合后续分析和处理的数据集的过程。
    </p>
    <ol>
     <li>
      确定归一化的范围：确定数据的取值范围，
      <strong>
       即数据的最大值和最小值
      </strong>
      。在某些情况下，也可以根据需要选择其他统计量（如均值和标准差）来确定归一化的范围。
     </li>
     <li>
      应用归一化公式：根据选定的归一化范围，
      <strong>
       应用相应的归一化公式将数据转换到新的尺度
      </strong>
      。常见的归一化公式包括线性函数归一化（将数据映射到[0,1]或[-1,1]的范围）和Z-score标准化（将数据转换为标准正态分布）。
     </li>
     <li>
      得到归一化后的数据：应用归一化公式后，即
      <strong>
       可得到新的、经过归一化处理的数据集
      </strong>
      。这些数据集在数值上更加稳定，有利于后续的分析和处理。
     </li>
    </ol>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/cab4ce1bc4eb5f23d17955ecee730979.png"/>
    </p>
    <h2>
     <a id="_37">
     </a>
     <strong>
      二、常用的归一化
     </strong>
    </h2>
    <p>
     <strong>
      常用的归一化有哪些？
     </strong>
    </p>
    <p>
     常用的归一化主要包括批量归一化（BN）、层归一化（LN）、组归一化（GN）和实例归一化（IN）等，它们各有其独特的应用场景和优势，选择哪种归一化方法通常取决于具体任务和数据的特点。
    </p>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/5fdc1f8dbec854034844805997f19e8d.png"/>
    </p>
    <h4>
     <a id="1_Batch_Normalization_BN_45">
     </a>
     1. 批量归一化（Batch Normalization, BN）
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        方法
       </strong>
       ：在神经网络的每一层中，
       <strong>
        对每个mini-batch的输入进行归一化处理
       </strong>
       。通过减去均值，再除以标准差，将输入数据转化为均值为0，标准差为1的分布。
      </p>
     </li>
     <li>
      <p>
       <strong>
        优点
       </strong>
       ：
       <strong>
        加速网络训练、防止梯度问题
       </strong>
       、优化正则化效果、降低学习率要求，
       <strong>
        并有助于缓解过拟合
       </strong>
       ，从而显著提升神经网络的性能和稳定性。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景：
       </strong>
       适用于大多数神经网络场景，特别是在训练深层网络时。
      </p>
     </li>
    </ul>
    <h4>
     <a id="2_Layer_Normalization_LN_54">
     </a>
     2. 层归一化（Layer Normalization, LN）
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        方法
       </strong>
       ：在神经网络的每一层中，
       <strong>
        对每个样本的所有特征维度进行归一化处理
       </strong>
       。通过减去均值，再除以标准差，将每个样本的特征维度转化为均值为0，标准差为1的分布。
      </p>
     </li>
     <li>
      <p>
       <strong>
        优点：
       </strong>
       在训练样本较小、样本间相互影响较大的情况下更稳定。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景
       </strong>
       ：
       <strong>
        <strong>
         如循环神经网络（RNN）等场景。
        </strong>
       </strong>
      </p>
     </li>
    </ul>
    <h4>
     <a id="3_Instance_Normalization_IN_63">
     </a>
     3. 实例归一化（Instance Normalization, IN）
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        方法：
       </strong>
       对每个样本的特征维度进行归一化。通过减去均值，再除以标准差，将每个样本的特征维度转化为均值为0，标准差为1的分布。
      </p>
     </li>
     <li>
      <p>
       <strong>
        优点：
       </strong>
       更适用于图像生成等任务中，每个样本的特征维度独立于其他样本的情况。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景
       </strong>
       ：图像生成任务等。
      </p>
     </li>
    </ul>
    <h4>
     <a id="4_Group_Normalization_GN_72">
     </a>
     4. 组归一化（Group Normalization, GN）
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        方法：
       </strong>
       在神经网络的每一层中，
       <strong>
        将特征分成若干组
       </strong>
       ，对每个组的特征进行归一化处理。通过减去均值，再除以标准差，将每个组的特征维度转化为均值为0，标准差为1的分布。
      </p>
     </li>
     <li>
      <p>
       <strong>
        优点：
       </strong>
       适用于样本较小、样本间相互影响较大，但又不需要对整个mini-batch进行归一化的情况。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景：
       </strong>
       图像分割任务等。
      </p>
     </li>
    </ul>
    <p>
     <img alt="" src="https://img-blog.csdnimg.cn/img_convert/94989ada15c6da64b4b8dda9004f9532.png"/>
