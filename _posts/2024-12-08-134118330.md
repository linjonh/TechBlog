    <h2>
     <a id="SSH_0">
     </a>
     创建SSH连接
    </h2>
    <blockquote>
     <p>
      SSH用于与远程服务器建立连接
     </p>
    </blockquote>
    <p>
     Settings -&gt; Tools -&gt; SSH Configurations
     <br/>
     添加新的ssh连接，Host为ip地址，Username为用户名，认证方式这里选择密码验证
     <br/>
     全部填完后可以点击Test Connection测试连接是否成功
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b92bed1bcf69261ad88559078106a479.png" width="600"/>
    </p>
    <h2>
     <a id="Deployment_9">
     </a>
     创建Deployment
    </h2>
    <blockquote>
     <p>
      Deployment用于构建本地与远程服务器的路径映射
     </p>
    </blockquote>
    <p>
     Settings -&gt; Build,Execution,Deployment -&gt; Deployment
     <br/>
     添加新的Deployment，Type选择SFTP，SSH选择刚刚配置的SSH连接，根目录，URL如图所示
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d9fd3e599a4a8c217bdb8c191c3416d1.png" width="600">
      <br/>
      选择Mappings设置路径映射，Local path为项目本地目录，Deployed path是在远程服务器上的目录
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3ed56d907c5b176bcff111ec516a6160.png" width="600">
       <br/>
       将本地的文件上传到服务器
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/784482c46b21ce9a743f5fa79e08c309.png" width="600">
        <br/>
        设置每次保存文件时都上传到服务器，在删除本地文件时也同步删除服务器文件
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/034570290e10f97eef9f3becec1aad22.png" width="600"/>
       </img>
      </img>
     </img>
    </p>
    <h2>
     <a id="_21">
     </a>
     使用服务器环境编译
    </h2>
    <blockquote>
     <p>
      有的时候我们本地的环境与服务器不一致，导致项目在本地无法运行，在配置完上述步骤后可以修改为在服务器上进行编译
     </p>
    </blockquote>
    <p>
     在Run/Debug Configurations中将Run on设置为ssh连接的服务器，同时勾选Build on remote target
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8e6a5f050f081622bce88635f8905ca7.png" width="600"/>
    </p>
