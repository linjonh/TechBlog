    <h3>
     <a id="_3">
     </a>
     简介
    </h3>
    <p>
     Naive Ui Admin 是一个免费开源的中后台模版，使用了最新的
     <code>
      vue3
     </code>
     ,
     <code>
      vite2
     </code>
     ,
     <code>
      TypeScript
     </code>
     等主流技术开发，开箱即用的中后台前端解决方案，也可用于学习参考。
    </p>
    <h3>
     <a id="_7">
     </a>
     特性
    </h3>
    <ul>
     <li>
      <strong>
       最新技术栈
      </strong>
      ：使用 Vue3/vite2 等前端前沿技术开发
     </li>
     <li>
      <strong>
       TypeScript
      </strong>
      : 应用程序级 JavaScript 的语言
     </li>
     <li>
      <strong>
       主题
      </strong>
      ：可配置的主题
     </li>
     <li>
      <strong>
       Mock 数据
      </strong>
      内置 Mock 数据方案
     </li>
     <li>
      <strong>
       权限
      </strong>
      内置完善的动态路由权限生成方案
     </li>
     <li>
      <strong>
       组件
      </strong>
      二次封装了多个常用的组件
     </li>
    </ul>
    <h4>
     <a id="_15">
     </a>
     页面功能
    </h4>
    <ul>
     <li>
      <h4>
       <a id="_Features_16">
       </a>
       ✨ Features
      </h4>
     </li>
     <li>
      菜单模式（左侧/顶部）
     </li>
     <li>
      权限（固定角色/动态路由菜单）
     </li>
     <li>
      主题（深色/浅色）引用 naiveui 特性，黑暗主题/自定义组件，暗黑主题信手捏来
     </li>
     <li>
      系统主题（十种安全色，可切换预览，可自定义）
     </li>
     <li>
      导航风格（三种，可自由搭配，暗色侧边/白色侧边和通栏/暗色侧边和顶部通栏）
     </li>
     <li>
      多页签，持久化，拖拽，关闭，固定
     </li>
     <li>
      面包屑，图标，子菜单下拉联动展示
     </li>
     <li>
      漂亮的锁屏界面
     </li>
     <li>
      在线配置抽屉（布局预览功能）
     </li>
     <li>
      主控台，简单写了一下，能看了
     </li>
     <li>
      工作台随意撸了一下（不想写style）用 tailwindcss大开杀戒了
     </li>
     <li>
      echarts，官网例子很全了，没啥好撸的，偷懒再次引用vben作者的代码，再感谢一次，哈哈
     </li>
     <li>
      主控台，工作台，自适应支持
     </li>
     <li>
      左侧菜单屏幕可视宽度&lt;=950 自动收起
     </li>
     <li>
      页脚是否显示，可配置
     </li>
     <li>
      新增页面缓存，演示页面在&gt;工作台&gt;动态卡片&gt;输入框
     </li>
    </ul>
