    <h2>
     <a id="_1">
     </a>
     前言
    </h2>
    <blockquote>
     <p>
      <strong>
       JDK是 Java 语言的软件开发工具包，主要用于移动设备、嵌入式设备上的java应用程序。JDK是整个java开发的核心，它包含了JAVA的运行环境（JVM+Java系统类库）和JAVA工具。
      </strong>
     </p>
    </blockquote>

     一、Linux安装JDK及配置环境变量
    </h2>
    <h3>
     <a id="1JDK_13">
     </a>
     1.创建JDK的安装目录
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        /usr/local/目录下创建jdk文件夹
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">mkdir</span> /usr/local/jdk
</code></pre>
      <p>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6861c84fac5d1ade5ad7e4b55313dc63.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        切换到cd /usr/local/jdk/目录下
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /usr/local/jdk/
</code></pre>
      <p>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4b5dc43e2db308682af8a194b5bbb18f.png"/>
      </p>
     </li>
    </ul>

     <a id="3_35">

     3.解压缩

</code></pre>

    <h3>
     <a id="4jdk_47">
     </a>
     4.配置环境变量（根据自己jdk的实际安装位置进行配置）
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        /etc/profile文件最后添加如下内容，按
        <code>
         i
        </code>
        键进行编辑模式，添加完成后按esc键输入
        <code>
         :wq
        </code>
        保存退出
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">vim</span> /etc/profile
</code></pre>
      <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span>/usr/local/jdk/jdk-17.0.10
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token variable">$JAVA_HOME</span>/bin:<span class="token environment constant">$PATH</span>
</code></pre>
      <p>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/817597b5fe4bd095d9b339754d051148.png"/>
      </p>
     </li>
    </ul>
    <h3>
     <a id="5_63">
     </a>
     5.加载配置文件

    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> /etc/profile
</code></pre>
