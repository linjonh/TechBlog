    <p>
     Dell服务器出现 “cpu0000 cpu1 internal error (IERR) contact support “，可能会导致服务器死机或者重启或无法正常启动。这是因为CPU节能设置，需要在BIOS中关闭该设置即可。
     <br/>
     解决办法
     <br/>
     1、服务器关机，并断掉点源。
     <br/>
     2、按住电源键20秒，等待1分钟左右。
     <br/>
     3、接通电源，开机。
     <br/>
     4、开机后再dell图标出来的时候按下F2,进入BIOS设置。
     <br/>
     5、system Setup - System BIOS ，选择 system profile Settings，进入后将右边的 Performance Per Watt（DAPC）改为 Custom。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f88525d57f686d33f0839089493be9ae.png#pic_center"/>
    </p>
    <p>
     6、然后可发现，下方的其他选项变为可选。
     <br/>
     7、将 CIE 和 C States 选项改成 Disabled。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8aa74c950fe31474e3f0eca7d42df682.png#pic_center"/>
    </p>
    <p>
     8，ESC返回 保存设置，重启服务器。
    </p>
