    <p>
     知识点：
    </p>
    <p>
     一、网络空间安全
    </p>
    <p>
     1 分类
    </p>
    <ul>
     <li>
      内容安全（影响和煽动）
     </li>
     <li>
      信息安全&amp;数据安全&amp;业务安全（经济犯罪和国家利益）
     </li>
     <li>
      网络系统安全-物理安全、主机安全、应用系统安全、网络安全（物理破坏）
     </li>
    </ul>
    <p>
     2 基础的企业计算机网络架构
    </p>
    <ul>
     <li>
      互联网–&gt;DMZ区–&gt;生产网(大数据平台、内部数据库、在线数据库)–&gt;办公网（研发测试人员
      <br/>
      、数据管理和分析人员、业务人员、运维人员）–&gt;测试网（测试数据库）
     </li>
     <li>
      威胁来源：
      <br/>
      互联网匿名用户
      <br/>
      GUEST
      <br/>
      注册用户
      <br/>
      合作伙伴
      <br/>
      内部人员
     </li>
    </ul>
    <p>
     3 攻击入侵链条
    </p>
    <p>
     攻防研究：漏洞挖掘和利用、预置后门和检测
    </p>
    <ul>
     <li>
      步骤1：弱点发现
      <br/>
      漏洞扫描、情报收集、爬虫、暴力破解
     </li>
     <li>
      步骤2：弱点利用
      <br/>
      物理接触、后门启用、漏洞利用、劫持钓鱼&amp;社工、网络嗅探、拒绝服务
     </li>
     <li>
      步骤3：扩展&amp;隐藏
      <br/>
      植入控制、横向扩展、
      <a href="https://baike.baidu.com/item/%E6%91%86%E6%B8%A1%E6%94%BB%E5%87%BB/9430901?fr=aladdin" rel="nofollow">
       摆渡攻击
      </a>
      、痕迹抹除、隐蔽通道
     </li>
     <li>
      步骤4：获利
      <br/>
      窃取、滥用、破坏、勒索
     </li>
    </ul>
    <p>
     辅助体系
     <br/>
     匿名交易、匿名支付、匿名通讯、匿名身份
    </p>

    </p>

     <br/>
     病毒/木马检测、IDS/IPS、WAF、
     <a href="https://baike.baidu.com/item/DLP%E6%95%B0%E6%8D%AE%E6%B3%84%E9%9C%B2%E9%98%B2%E6%8A%A4%E7%B3%BB%E7%BB%9F/10522315?fr=aladdin" rel="nofollow">

      UEBA
