    <h2>
     <a id="_3">
     </a>
     下载方式
    </h2>
    <ol>
     <li>
      通过上方GitHub链接进入Releases页面下载
     </li>
     <li>

    <h2>
     <a id="_6">
     </a>
     使用方法
    </h2>
    <p>
     安装完成后右键单机需要验证的文件，点击“
     <strong>
      哈希信息
     </strong>
     ”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/25beed4b599c45e898b5521a92b3d85a.png#pic_center"/>
    </p>
    <p>
     把文件发布者提供的哈希值直接粘贴在校验栏即可。无需选择具体算法！
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3b1d96d0b8774c718332c98f90add742.png#pic_center"/>
    </p>
    <h3>
     <a id="_14">
     </a>
     设置显示其他支持的哈希算法
    </h3>
    <p>
     点击右侧齿轮图标进入设置界面勾选即可
    </p>

    <h3>
     <a id="_18">
     </a>
     特性
    </h3>
    <ul>
     <li>
      支持高达28种算法（详见下方）
     </li>
     <li>
      高性能哈希计算
     </li>
     <li>
      原生Windows界面
     </li>
     <li>
      支持高DPI屏幕
     </li>
     <li>
      支持长路径
     </li>
     <li>
      多语言
     </li>
     <li>
      Check hashes against VirusTotal with a button
     </li>
     <li>
      Hash checking against a checksum file (Supported: hex hash next to file, *sum output (hex or base64), corz .hash, SFV)
     </li>
     <li>
      Hash export to file or clipboard (Supported: *sum output, corz .hash, SFV)
     </li>
     <li>
      Optional context menu option for faster access
     </li>
     <li>
      File associations and standalone mode
     </li>
    </ul>
    <h3>
     <a id="_30">
     </a>
     支持的算法
    </h3>
    <ul>
     <li>
      CRC32, CRC64 (xz)
     </li>
     <li>
      xxHash (XXH32, XXH64)
     </li>
     <li>
      xxHash3 (64 and 128 bit variants)
     </li>
     <li>
      MD4, MD5
     </li>
     <li>
      RipeMD160
     </li>
     <li>
      Blake2sp
     </li>
     <li>
      SHA-1
     </li>
     <li>
      SHA-2 (SHA-224, SHA-256, SHA-384, SHA-512)
     </li>
     <li>
      SHA-3 (SHA3-224, SHA3-256, SHA3-384, SHA3-512)
     </li>
     <li>
      BLAKE3 (256 bit, 512 bit)
     </li>
     <li>
      KangarooTwelve (264 bit, 256 bit, 512 bit)
     </li>
     <li>
      ParallelHash128 (264 bit) and ParallelHash256 (528 bit)
     </li>
     <li>
      Streebog (GOST R 34.11-12) (256 bit, 512 bit)
     </li>
    </ul>
    <h3>
     <a id="_44">
     </a>
     系统要求
    </h3>
    <ul>
     <li>
      Windows 7或更高版本
     </li>
     <li>
      1G以上的空闲运行内存(for efficient hashing of more than 512 files at a time)
     </li>
    </ul>
