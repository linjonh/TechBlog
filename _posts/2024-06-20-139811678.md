    <h2 id="%C2%A0%20%C2%A0%20%C2%A0%20%C2%A0%201.0%20%E5%9F%BA%E4%BA%8E%E8%84%9A%E6%89%8B%E6%9E%B6%E5%88%9B%E5%BB%BA%E5%89%8D%E7%AB%AF%E5%B7%A5%E7%A8%8B">
     1.0 基于脚手架创建前端工程
    </h2>
    <p>
     <strong>
      基于脚手架可以快速的创建前端工程，让开发者可以更快速地开始实际开发工作。脚手架工具提供了一些预设的配置选项和常用的功能模块，开发人员可以根据项目需求选择适合的模板和插件，并生成一个完整的项目结构。
     </strong>
    </p>
    <p>
     <strong>
      除了创建项目，脚手架工具还通常提供了一些命令来帮助开发人员进行开发、构建和部署等操作，例如启动开发服务器、打包代码、进行代码检查等功能，大大提高了开发效率。
     </strong>
    </p>
    <p>
    </p>
    <h3 id="%C2%A0%20%C2%A0%20%C2%A0%20%C2%A0%201.1%20%E5%9F%BA%E4%BA%8E%20Vue%20%E5%BC%80%E5%8F%91%E5%89%8D%E7%AB%AF%E9%A1%B9%E7%9B%AE%E7%9A%84%E7%8E%AF%E5%A2%83%E8%A6%81%E6%B1%82">
     1.1 基于 Vue 开发前端项目的环境要求
    </h3>
    <p>
     <strong>
      1）node.js：前端项目的运行环境。
     </strong>
    </p>
    <p>
     <strong>
      2）npm：JavaScript 的包管理工具。
     </strong>
    </p>
    <p>
     <strong>
      3）Vue CLI：基于 Vue 进行快速开发的完整系统，实现交互式的项目脚手架。
     </strong>
    </p>
    <p>
     <strong>
      使用以下命令来全局安装 Vue CLI：
     </strong>
    </p>
    <blockquote>
     <pre><code class="language-java">npm install -g @vue/cli</code></pre>
    </blockquote>
    <p>
     <strong>
      安装完成后，您可以通过以下命令来验证 Vue CLI 的安装是否成功：
     </strong>
    </p>
    <blockquote>
     <pre><code class="language-java">vue --version</code></pre>
    </blockquote>
    <p>
     <strong>
      运行结果：
     </strong>
    </p>
    <blockquote>
     <p>
     </p>
     <pre><code class="language-java">C:\Users\86187&gt;vue --version
@vue/cli 5.0.8</code></pre>
    </blockquote>

     2.7 Axios
    </h3>
    <p>
     <strong>
      Axios 是一个基于 promise 的网络请求库，作用于浏览器和 node.js 中。

      </a>
     </strong>
    </p>
    <p>
     <strong>
      安装命令：
     </strong>
    </p>
    <p>
     <strong>
      在终端输入命令进行安装：
     </strong>
    </p>
    <blockquote>
     <pre><code class="language-html">npm install axios</code></pre>
    </blockquote>

     <strong>

    <h4 id="%C2%A0%20%C2%A0%20%C2%A0%20%C2%A0%202.7.1%20axiox%C2%A0%E5%88%9B%E5%BB%BA%E7%9A%84%20API%20%E4%B8%8E%E9%85%8D%E7%BD%AE%E4%BB%A3%E7%90%86">
     2.7.1 axiox 创建的 API 与配置代理
    </h4>
    <p>
     <strong>
      axiox 的 API 列表常见的请求方式：
     </strong>
    </p>
    <p>
     <strong>
      1）axiox.get(url[,config])：url 表示请求路径，config 表示配置对象，可以设置查询参数、请求头信息。
     </strong>
    </p>
    <p>
     <strong>
      2）axiox.post(url[,data[,config]])：url 表示请求路径，config 表示配置对象，可以设置查询参数、请求头信息，data 表示请求体数据，最常见的是 JSON 格式数据。
     </strong>
    </p>
    <p>
    </p>
    <p>
     <strong>
      配置代理：
     </strong>
    </p>
    <p>
     <strong>
      为了解决跨域问题，可以在 vue.config.js 文件中配置代理。
     </strong>
    </p>
    <p>
     <strong>
      代码演示：
     </strong>
    </p>
    <blockquote>
     <pre><code class="language-javascript">const { defineConfig } = require('@vue/cli-service')
module.exports = defineConfig({
  transpileDependencies: true,
  devServer: {
    port: 7070,
    proxy: {
      '/api' : {
        target: 'http://localhost:8080',
        pathRewrite: {
          '^/api': ''
        }
      }
    }
  }
})</code></pre>

      配置完代理后，就可以给后端发送请求了。
