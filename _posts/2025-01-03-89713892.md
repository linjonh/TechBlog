         前沿技术，目前为止功能最全最强大的PLC智能远程模块，物联网模块
        </strong>
       </span>
      </strong>
     </strong>
    </h2>
    <p style="margin-left:0pt;">
     <strong>
      <span style="color:#000000;">
       <strong>
        如下图，巨控PLC智能远程控制终端不同应用场合的不同型号，巨控GRM模块分为以下4大类，GRMOPC，GRM530，GRM230，GRM110
       </strong>
      </span>
     </strong>
     <span style="color:#000000;">
      。
     </span>
    </p>
    <p style="margin-left:0pt;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/64b80ecfee49766eb24c20625f05fde1.jpeg"/>
    </p>
    <p style="margin-left:0pt;">
     <strong>
      <span style="color:#000000;">
       <strong>
        一. 巨控GRMOPC系列的PLC智能远程模块的详细应用技术合应用场景。
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:0pt;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/e95c21df08d088b6ab7be44452065f2c.png"/>
    </p>
    <p style="margin-left:0pt;">
    </p>
    <ul>
     <li>
      <span style="color:#000000;">
       支持各种组态软件wincc，力控，组态王等和DCS（浙大，和利时）
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       组态软件作为OPCSERVER，模块和电脑通过网线，局域网联接， 模块作为OPCCLIENT从电脑获取数据
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       电脑无需接入公网
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       模块可通过WIFI,有线,4G上网
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       模块支持电源冗余和网络冗余
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       OPC远程通讯无需配置DCOM，整个方案无需固定IP和动态域名。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       解决OPC跨网段的访问难题，任何电脑能上网通过组态软件都能远程访问监控中心组态软件的数据。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       原有监控系统做OPCSERVER，无需任何更改。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       原有监控系统工控机不接入外网，从根本上杜绝病毒和网络入侵。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       GRMOPC可选择关闭或者断开联网。远端OPC访问需密码认证。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       采用网线和4G(全网通)上网，双线备份，双电源供电。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       全部数据可选读写权限，确保系统安全。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       开放接口，客户可以开发APP和网页后台。
      </span>
     </li>
    </ul>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      应用场景：通常使用现场有上位机软件，已经把PLC的数据采集到上位机系统，客户需要把上位机数据远程上传至手机APP或网页监控，需要做手机短信或微信故障信息推送。或采集数据上传给第三方系统
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/7eeea4b263a826fb7c81c01b41530285.png"/>
     </span>
    </p>
    <p style="margin-left:0pt;">
     <strong>
      <span style="color:#000000;">
       <strong>
        二. 巨控GRM530系列智能远程模块功能与技术应用
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:0pt;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/26b94b2bf068f746a00d69e4f964226e.png"/>
    </p>
    <p style="margin-left:0pt;">
    </p>
    <ol>
     <li>
      <span style="color:#000000;">
       电脑对远程 PLC 梯形图下载和监控
      </span>
     </li>
    </ol>
    <ul>
     <li>
      <span style="color:#000000;">
       工程师在只需能上网的电脑上登陆 GRM530的模块序号和密码，使用 PLC 编程软件如博途，即可远程下载，调试，上传梯形图。巨控科技成立了最先进的 PLC 远程通讯研究中心，多年来持续针对各种 PLC 实测优化。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       独创下载机制，对于文件较大触摸屏和要求响应速度的运动控制器均可流畅下载调试。
      </span>
     </li>
    </ul>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      2. 短信、网页、APP 远程监控
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#000000;">
       支持手机短信报警，多次拨号提醒，  确保重大故障及时通知到人，实现无人值守。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       免费提供易用的网页，手机 APP。无需任何开发基础，即可实现手机随时随地远程监控。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       提供可断线续传的云端历史 数据和报警记录，可以直接用网页查看历史数据报表，曲线图和报警记录
      </span>
     </li>
    </ul>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      应用场景：现在有PLC设备，需要远程上下在PLC程序，或远程把PLC数据传输到现场的上位机，WINCC,组态王，力控等组态软件监控数控。需要远程手机APP监控，PLC具备有网口或串口通信，或采集数据上传给第三方系统
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/49ee0e31bf78578207d058b5da7ce7b8.png"/>
     </span>
    </p>
    <p style="margin-left:0pt;">
     <strong>
      <span style="color:#000000;">
       <strong>
        三. 巨控GRM230系列智能远程模块功能与技术应用
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:0pt;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/edd6291dff3b5746d7dd0b9edf237b38.png"/>
    </p>
    <p style="margin-left:0pt;">
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      1.自带 IO 或模拟量和逻辑编程
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#000000;">
       GRM230 模块本机自带 12 路 IO，可以配置
      </span>
      <span style="color:#000000;">
       成开关量输入，模拟量输入，继电器输出，电流输出等。模块的全部 IO 都具备短路，过压、过流，静电保护，精度优于常用主流的中小型 PLC。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       内置逻辑编程功能，使用本机 IO 可独立当成 PLC 使用降低系统成本
      </span>
     </li>
    </ul>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      2.  短信、网页、APP 远程监控
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#000000;">
       支持手机短信报警，多次拨号提醒，  确保 重大故障及时通知到人，实现无人值守。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       免费提供易用的网页，手机 APP。无需任何开发基础，即可实现手机随时随地远程监控。
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       提供可断线续传的云端历史 数据和报警记录，可以直接用网页查看历史数据报表，曲
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       线图和报警记录，可以随时导出表格。
      </span>
     </li>
    </ul>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      应用场景：现在有带485接口的PLC设备或带有485通信仪，模拟量输出的仪器仪表设备，远程把PLC或仪器仪表数据传输到现场的上位机，WINCC,组态王，力控等组态软件监控数控，远程手机APP监控，或采集数据上传给第三方系统。还可以自带IO，支持编程逻辑控制
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/c9778254b3c65037039d8d5de7c801f9.png"/>
     </span>
    </p>
    <p style="margin-left:0pt;">
     <strong>
      <span style="color:#000000;">
       <strong>
        四. 巨控GRM110系列智能远程模块功能与技术应用
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:0pt;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/e943d1a7f6991931efaf700f5957716c.png"/>
    </p>
    <p style="margin-left:0pt;">
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      GRM110 系列无线通讯模采用最先进的创新性 LORA 无线通讯方式 ，相对传统的数传电台，距离增加  2 倍以上。能适应点对点、 点对多点的无线数据通信方式 ，  具有通过 无线方式完全隔 离现场危险信 号、安装方 便、使用 简单、抗干 扰能力强、稳 定可靠、高性价 比等特点。该 产品用于短距离（3KM）内组态软件和 PLC，触摸屏和 PLC，传感器和 PLC，传感器和触摸屏，PLC 和 PLC 之间的无线通讯。
     </span>
    </p>
    <p style="margin-left:0pt;">
     <span style="color:#000000;">
      应用场景：
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#000000;">
       PLC-----组态软件无线互联，支持组态软件和多个 PLC 之间无线传输数据(485 或者网口 PLC 都可以)
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       PLC------触摸屏无线互联，支持触摸屏和多个 PLC 之间无线传输数据(485 或者网口 PLC 都可以)
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       PLC------PLC 无线互联，可以支持分散在不同地方的 PLC 之间互相读写数据(485 或者网口 PLC 都可以)
      </span>
     </li>
     <li>
      <span style="color:#000000;">
       传感器------PLC 无线互联 ，主站可以采集远程从站的开关或模拟数据，一个从站可以带 8 个数字模拟输入，4 个继电器输出
      </span>
     </li>
    </ul>
