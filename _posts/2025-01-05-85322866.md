  <div class="article_content clearfix" id="article_content">

   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      C语言中，
      <a href="https://www.baidu.com/s?wd=%E6%95%B0%E5%AD%A6%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
       数学函数
      </a>
      是函数的一种。指专门进行数学运算的函数，一般都在&lt;math.h&gt;头文件下。头文件定义为#include&lt;math.h&gt;即可。c++则用#include&lt;math.h&gt;
     </strong>
     <br/>
     <strong>
      <a href="https://www.baidu.com/s?wd=%E6%95%B0%E5%AD%A6%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
       数学函数
      </a>
      列表：
     </strong>
     <br/>
     1）int abs(int i); 求整数的绝对值。
     <br/>
     2）long labs(long n); 求长整型数的绝对值。
     <br/>
     3）double fabs(double x); 求实数的绝对值。
     <br/>
     4）double floor(double x); 求不大于x的最大整数，它相当于
     <a href="https://www.baidu.com/s?wd=%E6%95%B0%E5%AD%A6%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      数学函数
     </a>
     [x]。
     <br/>
     5）double ceil(double x); 求不小于x的最小整数。
     <br/>
     6）double sqrt(double x); 求x的
     <a href="https://www.baidu.com/s?wd=%E5%B9%B3%E6%96%B9%E6%A0%B9&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      平方根
     </a>
     。
     <br/>
     7）double log10(double x); 求x的常用对数（10为底）。
     <br/>
     8）double log(double x); 求x的
     <a href="https://www.baidu.com/s?wd=%E8%87%AA%E7%84%B6%E5%AF%B9%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      自然对数
     </a>
     （e为底）
     <br/>
     9）double exp(double x); 求
     <a href="https://www.baidu.com/s?wd=%E6%AC%A7%E6%8B%89%E5%B8%B8%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      欧拉常数
     </a>
     e的x次方。
     <br/>
     10）double pow10(int p); 求10的p次方。
     <br/>
     11）double pow(double x, double y); 求x的y次方。
     <br/>
     12）double sin(double x); 正弦函数。
     <br/>
     13）double cos(double x); 余弦函数。
     <br/>
     14）double tan(double x); 正切函数。
     <br/>
     15）double asin(double x);
     <a href="https://www.baidu.com/s?wd=%E5%8F%8D%E6%AD%A3%E5%BC%A6%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      反正弦函数
     </a>
     。
     <br/>
     16）double acos(double x);
     <a href="https://www.baidu.com/s?wd=%E5%8F%8D%E4%BD%99%E5%BC%A6%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      反余弦函数
     </a>
     。
     <br/>
     17）double atan(double x);
     <a href="https://www.baidu.com/s?wd=%E5%8F%8D%E6%AD%A3%E5%88%87%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      反正切函数
     </a>
     。
     <br/>
     18）double atan2(double x);
     <a href="https://www.baidu.com/s?wd=%E5%8F%8D%E6%AD%A3%E5%88%87%E5%87%BD%E6%95%B0&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      反正切函数
     </a>
     2。
    </p>
    <p>
     <strong>
      反正切函数1和2的区别：
     </strong>
     <br/>
     float angle = atan( (y2-y1)/(x2-x1) );
    </p>
    <p>
     float angle = atan2( y2-y1, x2-x1 );
    </p>
    <p>
     1：参数的填写方式不同；
    </p>
    <p>
     2：atan2 的优点在于 如果 x2-x1等于0 依然可以计算，但是atan函数就会导致程序出错；
    </p>
