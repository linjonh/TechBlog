    <h5>
     <a id="9HarmonyOS_Connect_67">
     </a>
     9、下列哪一项不是HarmonyOS Connect的模组认证中的合规要求()
    </h5>
    <p>
     A）Wi-Fi认证
     <br/>
     B）RQB认证
     <br/>
     C）有毒有害物质检测
     <br/>
     D）OS兼容性测试
    </p>
    <blockquote>
     <p>
      答案：D
      <br/>
      解析：模组认证要求分为合规要求和技术要求，
      <br/>
      <strong>
       合规要求
      </strong>
      包含强制认证的有毒有害物质检测，型号核准，以及行业认证中的Wi-Fi认证、RQB认证;
      <br/>
      <strong>
       技术要求
      </strong>
      包含模组硬件测试、OS兼容性测试、安全测试、新服务体验测试这四种。
      <br/>
      题目中说的是合规要求，不是技术要求。
     </p>
    </blockquote>
