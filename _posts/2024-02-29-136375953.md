    </p>
    <h2 style="text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       1.作为运算符
      </span>
     </span>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       *表示乘法运算
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       **表示乘方运算
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       代码 示例：
      </span>
     </span>
    </p>
    <pre><code class="language-python">#1.* ** 作为运算符
print('\n------运算符测试------')
a = 3*2  #乘法
b = 3**2 #乘方
print("a = ", a)  #输出6
print("b = ", b)  #输出9
</code></pre>

    <h2 style="text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       2.作为函数形参
      </span>
     </span>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       *在python中做函数形参，表示的是一个
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       可变长度的序列，
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       类型为
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       tuple，
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       即将所有参数放入一个元组（tuple）中，被函数使用。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       **在python中做函数形参，表示的是一个可变长度的序列，类型为dict,即
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       将所有的关键字参数，放入一个字典（dict）
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       中， 被
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       函数使用
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       *与**作为形参同时出现时，**必须在最后面.
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     代码示例：
    </p>
    <pre><code class="language-python">#2.作为形参
print('\n------形参测试------')
def printx(*arg1, **arg2):
    print("arg1={0}, type:{1}".format(arg1, type(arg1)))
    print("arg2={0}, type:{1}".format(arg2, type(arg2)))

#调用
P1 = printx(1,2, key1=1, key2=2)</code></pre>

    <h2 style="text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       3.作为函数实参
      </span>
     </span>
    </h2>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       在list/tuple/set前加*，会把序列中的所有元素解包成位置参数
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       在dict前加*，会把dict的
      </span>
      <span style="color:#fe2c24;">
       <strong>
        键
       </strong>
      </span>
      <span style="color:#222222;">
       变成位置参数；加**，把字典的
      </span>
      <span style="color:#fe2c24;">
       <strong>
        键值对
       </strong>
      </span>
      <span style="color:#222222;">
       变成关键字参数。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       A = (a,b,c)
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       B = {“
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       key1
      </span>
     </span>

      <span style="color:#222222;">
       ”
      </span>

     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       :1,
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       “
      </span>
     </span>
     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
       key2

      <span style="color:#222222;">
       ”
      </span>

      <span style="color:#222222;">

     <span style="background-color:#ffffff;">
      <span style="color:#222222;">
