    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      概述
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      第一次使用IBM的云平台Bluemix，写一个blog记录一下。
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      我注册Bluemix挺早的，但是在工作中一直没有机会使用IBM的云平台。现在辞职创业，做自己喜欢的互联网，终于有机会用上了。Bluemix为开发人员、企业和最终用户提供基于云的应用程序和服务，轻松应对未来社交、移动和大数据需求的挑战。正适合我们这种创业者。
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      创建容器
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      1.登陆IBM的云平台Bluemix，进入如下图所示的仪表盘。点击“容器”，进入容器创建页面。
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205005835137?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      2.在容器创建页面，选择容器映像。IBM提供了很多映像模板，也可以添加自己的Docker容器，功能非常强大。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205005955963?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      3.我选择IBM提供的容器映像来创建第一个容器，类型为ibm-mobilefirst-starter。在弹出的窗口中，设置注册表名称空间为“shenlan",如下图所示。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205120420577?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      4.在创建页面中，可以在页面左边看到复制的映像的大小是734.67MB，页面中间能够看到公共端口。在页面右边能够看到系统内存的配额为2GB，已用256MB，公共IP地址配额为2个。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205120527290?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      5.在中间单个容器选项页中，输入容器名称为”shenlan”，点击“创建”按钮。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205120954826?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      6.创建成功后，会提示“Your container is running”，在页面中会实时显示当前的内存、存储使用状况。在页面右边底部容器运行状况，容器控制按钮，可以控制容器的停止、暂停、重新启动和删除。在页面中可以看到当前没有绑定公共IP。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205121052554?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      7.点击公共IP栏，选择绑定，提示正在绑定134.168.41.10。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205150717084?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      8.绑定成功后，公共IP栏显示为已经绑定的IP134.168.41.10。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205150633922?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%">
       <br/>
      </img>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      9.点击“监视和日志”，能够以图形化的方式实时展示CPU使用率，内存使用量、网络等系统参数。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205151032190?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%"/>
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      10.点击"高级视图"按钮，系统实时参数切换为高级视图显示。
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="https://img-blog.csdn.net/20151205151339503?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" style="border:none; max-width:100%"/>
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      总结
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      IBM Bluemix的容器创建非常简单，开发者
     </span>
     <span style="font-size:18px">
      不用为硬件、软件、网络、存储等其他因素分心。 不用担心硬件软件采购和安装就节省了程序开发人员的时间和精力，加快了程序应用的上线，也节省了开支。
     </span>
    </p>
