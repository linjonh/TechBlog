     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       JavaScript 是什么
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      JavaScript是世界上最流行的语言之一，是一种运行在客户端的脚本语言(Script是脚本的意思)
     </p>
     <p style="padding-left:1.4em;">
      脚本语言:不需要编译，运行过程中由js解释器(js 引擎）逐行来进行解释并执行。现在也可以基于Node.js技术进行服务器端编程
     </p>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       JavaScript 的作用
      </span>
     </h3>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        表单动态校验（密码强度检测)(JS产生最初的目的)
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        网页特效
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        服务端开发(Node.js)
       </p>
      </li>
     </ul>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       HTML/CSS/JS 的关系
      </span>
     </h3>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        html：决定网页结构和内容，相当于身体
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        css：决定网页呈现给用户的模样，相当于给人穿衣服、化妆
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        js：实现业务逻辑和页面控制，相当于人的各种动作
       </p>
      </li>
     </ul>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       浏览器执行JS
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      浏览器分成两部分:渲染引擎和JS引擎
     </p>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        <span class="kdocs-bold" style="font-weight:bold;">
         渲染引擎
        </span>
        :用来解析HTML与CSS，俗称内核，比如chrome浏览器的 blink，老版本的webkit
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        <span class="kdocs-bold" style="font-weight:bold;">
         JS引擎
        </span>
        :也称为JS解释器。用来读取网页中的JavaScript代码，对其处理后运行，比如chrome浏览器的V8
       </p>
      </li>
     </ul>
     <blockquote class="kdocs-blockquote" style="">
      <span class="kdocs-fontSize" style="font-size:15pt;">
       浏览器本身并不会执行JS代码，而是通过内置JavaScript引擎(解释器)来执行JS代码。JS引擎执行代码时逐行解释每一句源码(转换为机器语言)，然后由计算机去执行，所以JavaScript 语言归为脚本语言，会逐行解释执行.
      </span>
     </blockquote>

     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       ECMAScript
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      <span class="kdocs-bold" style="font-weight:bold;">
       ECMAScript
      </span>
      是由ECMA 国际（ 原欧洲计算机制造商协会）进行标准化的一门编程语言，这种语言在万维网上应用广泛，它往往被称为 JavaScript（网景公司） 或 JScript（微软公司），但实际上后两者是 ECMAScript 语言的实现和扩展。
     </p>
     <blockquote class="kdocs-blockquote" style="">
      <span class="kdocs-fontSize" style="font-size:15pt;">
       ECMAScript: ECMAScript规定了JS的编程语法和基础核心知识，是所有浏览器厂商共同遵守的一套JS语法工业标准。
      </span>
     </blockquote>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       DOM - 文档对象模型
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      <span class="kdocs-bold" style="font-weight:bold;">
       文档对象模型
      </span>
      (Document Object Model，简称DOM)，是W3C组织推荐的处理可扩展标记语言的
      <span class="kdocs-bold" style="font-weight:bold;">
       标准编程接口
      </span>
      。通过DOM提供的接口可以对页面上的各种元素进行操作(大小、位置、颜色等)。
     </p>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       BOM - 浏览器对象模型
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      <span class="kdocs-bold" style="font-weight:bold;">
       BOM
      </span>
      (Browser Object Model，简称BOM)是指浏览器对象模型，它提供了独立于内容的、可以与浏览器窗口进行互动的对象结构。通过BOM可以操作浏览器窗口，比如弹出框、控制浏览器跳转、获取分辨率等。
     </p>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       JS 使用
      </span>
     </h3>
     <p style="text-indent:1.4em;">
      JS有3种书写位置，分别为行内、内嵌和外部。
     </p>
     <h4 style="text-indent:1.4em;">
      <span class="kdocs-bold" style="font-weight:bold;">
       行内式
      </span>
     </h4>
     <pre class="kdocs-javascript"><code class="language-javascript">&lt;input type="button" value="点我试试" onclick="alert('Hello World')" /&gt;</code></pre>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        可以将单行或少量 JS 代码写在HTML标签的事件属性中（以 on 开头的属性），如：onclick
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        注意单双引号的使用：在
        <span class="kdocs-bold" style="font-weight:bold;">
         HTML
        </span>
        中我们推荐使用
        <span class="kdocs-bold" style="font-weight:bold;">
         双引号
        </span>
        ,
        <span class="kdocs-bold" style="font-weight:bold;">
         JS
        </span>
        中我们推荐使用
        <span class="kdocs-bold" style="font-weight:bold;">
         单引号
        </span>
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        行内式可读性差， 在html中编写JS大量代码时，不方便阅读；
       </p>
      </li>
     </ul>
     <h4 style="text-indent:1.4em;text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       内嵌 JS
      </span>
     </h4>
     <pre class="kdocs-javascript"><code class="language-javascript">&lt;script&gt;
     alert('Hello World~!');
&lt;/script&gt;</code></pre>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        可以将多行JS代码写到 &lt;script&gt; 标签中
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        内嵌 JS 是测试常用的方式
       </p>
      </li>
     </ul>
     <h4 style="text-indent:1.4em;text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       外部 JS 文件
      </span>
     </h4>
     <pre class="kdocs-javascript"><code class="language-javascript">&lt;script src="my.js"&gt;&lt;/script&gt;</code></pre>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        利于HTML页面代码结构化，把大段 JS代码独立到 HTML 页面之外，既美观，也方便文件级别的复用
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        引用外部 JS文件的 script 标签中间不可以写代码
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        适合于JS 代码量比较大的情况
       </p>
      </li>
     </ul>
     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       JS 注释
      </span>
     </h3>
     <h4 style="text-indent:1.4em;text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       单行注释
      </span>
     </h4>
     <pre class="kdocs-javascript"><code class="language-javascript">// 我是一行文字，不想被 JS引擎 执行，所以 注释起来</code></pre>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        // 用来注释单行文字（ 快捷键 ctrl + / ）
       </p>
      </li>
     </ul>
     <h4 style="text-indent:1.4em;text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">

     <h3 style="text-align:left;">
      <span class="kdocs-bold" style="font-weight:bold;">
       输入输出语句

     <p style="text-indent:1.4em;">
      为了方便信息的输入输出，JS中提供了一些输入输出语句
     </p>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">

        alert(msg)：浏览器弹出警示框

      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        console.log(msg)：浏览器控制台打印输出信息
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">

      </li>
     </ul>
     <h3 style="">
      变量
     </h3>
     <p style="text-indent:1.4em;">
      变量是用于存放数据的
      <span class="kdocs-bold" style="font-weight:bold;">
       容器
      </span>
      。我们通过
      <span class="kdocs-bold" style="font-weight:bold;">
       变量名
      </span>
      获取数据，甚至数据可以修改。
     </p>
     <p style="text-indent:1.4em;">
      本质:变量是程序在内存中申请的一块用来存放数据的空间。
     </p>
     <p style="text-align:null;">
      在使用时分为两步:1、声明变量，2、赋值
     </p>
     <p style="text-align:null;">
      声明一个变量并赋值， 我们称之为
      <span class="kdocs-bold" style="font-weight:bold;">
       变量的初始化
      </span>
      。
     </p>
     <h4 style="text-indent:1.4em;">
      语法
     </h4>
     <pre class="kdocs-javascript"><code class="language-javascript"> //声明变量 
var age; // 声明一个 名称为age 的变量 
age = 10; // 给 age 这个变量赋值为 10
//一个变量被重新复赋值后，它原有的值就会被覆盖，变量值将以最后一次赋的值为准。
age = 81;
//同时声明多个变量时
var age1 = 10, name = 'zs', sex = 2; </code></pre>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        <span class="kdocs-bold" style="font-weight:bold;">
         var
        </span>
        是一个JS关键字，用来声明变量( variable变量的意思)。使用该关键字声明变量后，计算机会自动为变量分配内存空间，不需要程序员管
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        age是程序员定义的变量名，我们要通过变量名来访问内存中分配的空间
       </p>
      </li>
     </ul>
     <h4 style="text-indent:1.4em;">
      命名规范
     </h4>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        由字母(A-Za-z)、数字(0-9)、下划线(_)、美元符号( $ )组成，如：usrAge, num01, _name
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        严格区分大小写。var app; 和 var App; 是两个变量
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        不能 以数字开头，不能 是关键字、保留字，遵守驼峰命名法
       </p>
      </li>
     </ul>
     <h3 style="">
      数据类型
     </h3>
     <p style="text-indent:1.4em;">
      在计算机中，不同的数据所需占用的存储空间是不同的，为了便于把数据分成所需内存大小不同的数据，充分利用存储空间，于是定义了不同的数据类型。简单来说，数据类型就是数据的类别型号。
     </p>
     <p style="text-indent:1.4em;">
      变量是用来存储值的所在处，它们有名字和数据类型。变量的数据类型决定了如何将代表这些值的位存储到计算机的内存中。
      <span class="kdocs-bold" style="font-weight:bold;">
       JavaScript 是一种弱类型或者说动态语言
      </span>
      。这意味着不用提前声明变量的类型，在程序运行过程中，类型会被自动确定。
     </p>
     <pre class="kdocs-javascript"><code class="language-javascript">var age = 10;             // 这是一个数字型
var areYouOk = '是的';  // 这是一个字符串</code></pre>
     <p style="">
      在代码运行时，变量的数据类型是由JS引擎
      <span class="kdocs-bold" style="font-weight:bold;">
       根据=右边变量值的数据类型来判断
      </span>
      的，运行完毕之后，变量就确定了数据类型。
      <span class="kdocs-bold" style="font-weight:bold;">
       JavaScript拥有动态类型，同时也意味着相同的变量可用作不同的类型
      </span>
      :
     </p>
     <pre class="kdocs-javascript"><code class="language-javascript">var x = 6; // x 为数字
var x = "Bill"; // x 为字符串 </code></pre>
     <h4 style="">
      数据类型的分类
     </h4>
     <p style="text-indent:1.4em;">
      JS 把数据类型分为两类：
     </p>
     <ul>
      <li style="margin-left:1.4em;list-style-type:disc;text-indent:0;">
       <p>
        简单数据类型(Number,string,Boolean,Undefined,Null)
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        Number,数字型，包含整型值和浮点型值，如21、0.21
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        JavaScript中数值的最大和最小值,
       </p>
      </li>
     </ul>
     <pre class="kdocs-javascript"><code class="language-javascript">alert(Number.MAX_VALUE); // 1.7976931348623157e+308
alert(Number.MIN_VALUE); // 5e-324</code></pre>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        进制有二进制、八进制、十进制、十六进制
       </p>
      </li>
     </ul>
     <pre class="kdocs-javascript"><code class="language-javascript"> // 1.八进制数字序列范围：0~7

var num1 = 07;     // 对应十进制的7
var num2 = 019; // 对应十进制的19
var num3 = 08;     // 对应十进制的8

 // 2.十六进制数字序列范围：0~9以及A~F

var num = 0xA;</code></pre>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        三个特殊值
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;">
       <p>
        Infinity ，代表无穷大，大于任何数值
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;">
       <p>
        -Infinity ，代表无穷小，小于任何数值
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;">
       <p>
        NaN ，Not a number，代表一个非数值
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        <span class="kdocs-bold" style="font-weight:bold;">
         isNaN(x)，用来判断一个变量是否为非数字的类型，返回true（x不是数字） 或者false（x是数字）
        </span>
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:2.8em;list-style-type:circle;text-indent:0;">
       <p>
        Boolean,布尔值类型，如true .false，等价于1和0
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        布尔类型有两个值: true和false，其中 true表示真(对)，而false表示假（错)
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
