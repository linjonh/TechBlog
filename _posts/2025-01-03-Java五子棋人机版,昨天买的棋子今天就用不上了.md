     实现思路
    </h2>
    <blockquote>
     <p>
      <font color="SeaGreen">
       1.创建运行窗口并添加背景色。
       <br/>
       2.绘制棋盘。
       <br/>
       3.用二维数组来控制起码落子位置、绘制指示器。
       <br/>
       4.鼠标在落子位置处点击可落子。
       <br/>
       5.落子后检查是否获得胜利。
       <br/>
       6.机器判断下一步，并落子。
       <br/>
       7.机器判断是否获得胜利。
      </font>
     </p>
