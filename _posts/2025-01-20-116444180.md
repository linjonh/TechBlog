    <h2>
     <a id="_0">
     </a>
     问题
    </h2>
    <p>
     使用网页版立创EDA画板子，看重立创EDA集成的元件库。但是网页版EDA运行很卡。
    </p>
    <h2>
     <a id="_2">
     </a>
     解决
    </h2>
    <ol>
     <li>
      <p>
       修改设置。参考
       <code>
        6楼
       </code>
       官方答复
       <a href="http://club.szlcsc.com/article/details_47833_1_9.html#floor_9" rel="nofollow">
        参考链接
       </a>
       。但是效果不大。
       <br/>
       <img alt="优化方法" src="https://i-blog.csdnimg.cn/blog_migrate/90def597f94812082b22a5edc8aabace.png#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       使用客户端。
       <a href="https://lceda.cn/page/download" rel="nofollow">
        下载链接
       </a>
       <br/>
       我的电脑上windows客户端无法运行，网上搜索是兼容性问题，尝试兼容性选项无果。之后转战deepin平台，成功运行。
       <br/>
       有一说一客户端是真的挺香的，不卡顿，元件库齐全，全平台兼容，一个帐号云同步。
      </p>
     </li>
     <li>
      <p>
       使用网页专业版
       <br/>
       使用过，但是目前感觉专业版还有些不完善：
      </p>
      <ul>
       <li>
        标准版EDA的原理图导入专业版后有些东西会乱掉（我的项目里的总线就乱掉了），封装会丢失，需要重新绑定封装。
       </li>
       <li>
        感觉专业版的元件库不好用，元件搜索的筛选设计十分脑残，界面是个长条而不是窗口，强烈要求改回原来窗口样式的元件库界面
       </li>
      </ul>
     </li>
    </ol>
    <p>
     当然真心希望立创EDA能再接再厉，提供更好的EDA，为我等硬件小白提供更友好的起点
    </p>
