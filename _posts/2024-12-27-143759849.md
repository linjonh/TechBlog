    <h4>
     1.1 小程序跳转小程序
    </h4>
    <p>
     小程序可跳转任意正常小程序(
     <a href="https://link.juejin.cn/?target=https%3A%2F%2Funiapp.dcloud.net.cn%2Fapi%2Fother%2Fopen-miniprogram.html%23navigatetominiprogram" rel="nofollow" title="uni.navigateToMiniProgram(OBJECT) | uni-app官网">
      uni.navigateToMiniProgram(OBJECT) | uni-app官网
     </a>
     )，使用
     <code>
      uni.navigateToMiniProgram
     </code>
    </p>
    <p>
     具体代码：
    </p>
    <pre><code class="hljs">// #ifdef MP-ALIPAY || MP-WEIXIN
uni.navigateToMiniProgram({
    appId: appId,
    path: 页面路径,
    envVersion: 'release',// 开发版、体验版、正式版
    success(res) { }
  });
// #endif
</code></pre>
    <h4>
     1.2 App跳转小程序
    </h4>
    <p>
     App跳转小程序需要使用
     <code>
      plus
     </code>
     的
     <code>
      launchMiniProgram
     </code>
     api,
     <a href="https://link.juejin.cn/?target=https%3A%2F%2Fwww.html5plus.org%2Fdoc%2Fzh_cn%2Fshare.html%23plus.share.ShareService.launchMiniProgram" rel="nofollow" title="HTML5+ API launchMiniProgram 文档">
      HTML5+ API launchMiniProgram 文档
     </a>
    </p>
    <p>
     具体代码：
    </p>
    <pre><code class="hljs">// #ifdef APP-PLUS
plus.share.getServices(
(data) =&gt; {
  let sweixin = null;
  data.forEach(item=&gt;{
    if(item.id==="weixin"){
      sweixin = item
    }
  })
  if (sweixin) {
    // 跳转到微信小程序
    sweixin.launchMiniProgram({
      id: "", //微信小程序原生id
      path: `/pages/mall/index`,
      // 可取值：0-正式版； 1-测试版； 2-体验版。默认值为0。
      type: 0, // 微信小程序版本类型，
    });
  } else {
    uni.showToast({
      title: "请安装微信",
      icon: "none",
    });
  }
},
(err) =&gt; {
  console.log("跳转失败");
}
);
// #endif

data值为：
[
    {
        "id": "sinaweibo",
        "description": "新浪微博",
        "authenticated": false,
        "accessToken": "",
        "nativeClient": false
    },
    {
        "id": "qq",
        "description": "QQ",
        "authenticated": false,
        "accessToken": "",
        "nativeClient": true
    },
    {
        "id": "weixin",
        "description": "微信",
        "authenticated": true,
        "accessToken": "",
        "nativeClient": true
    }
]
</code></pre>
    <h4>
     1.3 h5跳转小程序
    </h4>
    <p>
     h5跳转小程序，可以使用
     <code>
      URL Scheme
     </code>
     ,即通过
     <code>
      weixin://dl/business/?xxx
     </code>
     方式，拉起本地微信后进行跳转。
    </p>
    <p>
     <a href="https://link.juejin.cn/?target=https%3A%2F%2Fdevelopers.weixin.qq.com%2Fminiprogram%2Fdev%2Fframework%2Fopen-ability%2Furl-scheme.html" rel="nofollow" title="获取 URL Scheme | 微信开放文档">
      获取 URL Scheme | 微信开放文档
     </a>
    </p>
    <p>
     具体代码：
    </p>
    <pre><code class="hljs">// #ifdef WEB
window.location.href= "weixin://dl/business/?t=HZ2gBmGKRzp"
//"weixin://dl/business/?appid=wx7fdf86d7c67b2184&amp;path=pages/mall/index" 
// #endif

</code></pre>
    <h4>
     1.4 小程序跳转h5
    </h4>
    <p>
     小程序跳转h5通过微信的
     <code>
      web-view
     </code>
     ，跳转前需要后台配置业务域名。
    </p>
    <pre><code class="hljs">&lt;web-view
    :src="url"
    @message="bindmessage"
    @load="bindload"
  &gt;&lt;/web-view&gt;
</code></pre>

     2.2 公众号跳转小程序

    <h3>
     总结
    </h3>
    <p>
     最后总结一下：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        小程序之间跳转
       </strong>
       ：使用
       <code>
        uni.navigateToMiniProgram
       </code>
       方法，可以在小程序间跳转。参数包含目标小程序的
       <code>
        appId
       </code>
       和页面路径等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        App 跳转到小程序
       </strong>
       ：在 App 中可以通过
       <code>
        plus.share.getServices
       </code>
       获取分享服务，然后使用
       <code>
        launchMiniProgram
       </code>
       方法跳转到微信小程序。
      </p>
     </li>
     <li>
      <p>
       <strong>
        H5 跳转小程序
       </strong>
       ：H5 页面通过 URL Scheme (
       <code>
        weixin://dl/business/?xxx
       </code>
       ) 的形式调起本地微信，并跳转到指定小程序。
      </p>
     </li>
     <li>
      <p>
       <strong>
        小程序跳转 H5
       </strong>
       ：通过小程序内的
       <code>
        web-view
       </code>
       组件，能够嵌入和跳转到 H5 页面。
      </p>
     </li>
     <li>
      <p>
       <strong>
        H5 页面跳转小程序
       </strong>
       ：可以通过微信开放标签（如
       <code>
        wx-open-launch-weapp
       </code>
       ）或
       <code>
        wx.miniProgram.navigateTo
       </code>
       来实现 H5 页面跳转到小程序。
      </p>
     </li>
     <li>
      <p>
       <strong>
        小程序与公众号跳转
       </strong>
       ：
      </p>
      <ul>
       <li>
        小程序跳转到公众号：通过
        <code>
         official-account
        </code>
        组件，前提是公众号已关联小程序。
       </li>
       <li>
        公众号跳转到小程序：可以进行菜单和模板消息配置等。
       </li>
      </ul>
     </li>
    </ol>
