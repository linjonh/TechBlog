    <h4>
     1、
     <strong>
      C:\Users\10501\Desktop\RunPythonWork&gt;.\python-3.10.11-embed-amd64\python.exe .\PackageTest\main\main.py
      <br/>
      Traceback (most recent call last):
      <br/>

      <br/>

      <br/>
      ModuleNotFoundError: No module named 'PackageTest'

     <img alt="" height="626" src="https://i-blog.csdnimg.cn/blog_migrate/288298f9d3918add11dd654aee1d38a4.png" width="1113"/>

    <p>
     报错：找不到自己创建的包！！！
    </p>
    <p>
     main.py的代码如下：
    </p>
    <pre><code class="language-python"># ---encoding:utf-8---
# @Time    : 2023/6/22 17:35
# @Author  : CBAiotAigc
# @Email   ：1050100468@qq.com
# @Site    : 
# @File    : main.py
# @Project : Test
# @Software: PyCharm
# import os
# import sys
# BASE_PATH = os.path.dirname(os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
# print(BASE_PATH)
# sys.path.append(BASE_PATH)

from PackageTest.utils.utils import *

print("main")
</code></pre>
    <p>
     上面的PackageTest是我创建的包：
    </p>
    <p>
     <img alt="" height="491" src="https://i-blog.csdnimg.cn/blog_migrate/655e68ad49014884a7bbc527807d8bf2.png" width="1200"/>
    </p>
    <h4>
     解决方案：
    </h4>
    <p>
     代码最上面添加如下代码，即将sys.path追加项目根目录：
    </p>
    <pre><code class="language-python">import os
import sys
BASE_PATH = os.path.dirname(os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
print(BASE_PATH)
sys.path.append(BASE_PATH)</code></pre>
    <p>
     完整的main.py如下：
    </p>
    <pre><code class="language-python"># ---encoding:utf-8---
# @Time    : 2023/6/22 17:35
# @Author  : CBAiotAigc
# @Email   ：1050100468@qq.com
# @Site    : 
# @File    : main.py
# @Project : Test
# @Software: PyCharm
import os
import sys
BASE_PATH = os.path.dirname(os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
print(BASE_PATH)
sys.path.append(BASE_PATH)

from PackageTest.utils.utils import *

print("main")
</code></pre>
