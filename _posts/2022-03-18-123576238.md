    <h3>
     <a id="PythonPycharm_0">
     </a>
     Python—Pycharm，下载、安装、配置及使用教程
    </h3>
    <p>
     <em>
      使用Python需下载python解释器和编辑器，下面介绍解释器的安装过程。
     </em>
    </p>
    <p>
     PyCharm分为专业版，教育版和社区版，专业版收费的，教育版和社区版免费。专业版适用于企业，教育版适合教师和学生，对于Python初学小白来说，社区版足够。
    </p>
    <p>
     <strong>
      一、下载安装Pycharm
     </strong>
     <br/>
     官网链接：https://www.jetbrains.com/pycharm/
     <br/>
     点击首页右上方Download进入下载页面
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/425379db7f9eaffd460b063fdab3a9c6.png">
      <br/>
      双击图标进行安装
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/74c20dd1cea0ed23f3f6da3a4b5cf28c.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cb65d05b0f92b39bd40b94e88c1f0d89.png">
        <br/>
        选择安装路径，不建议放在C盘
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa876b15b96ece489fbf2808c430927b.png">
         <br/>
         Create Desktop Shortcut：创建桌面快捷方式
         <br/>
         Create Associations：创建py文件关联
         <br/>
         注意：建议不要勾选Update PATH variable(添加环境变量，安装后需重启电脑，会报错）
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8cc26861e4c15380be68fc69147bb05.png">
          <br/>
          默认开始菜单文件夹为“JetBrains”即可，开始安装
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c64376fe9e8fb880177aa3ed85140119.png">
           <br/>
           等待几分钟
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bb9d16292c17ccbb7fa5ee6c04b96d40.png">
            <br/>
            安装成功，Finish
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/825a8fabc150559577da02040a05ec5d.png">
             <br/>
             <strong>
              二、配置Pycharm
             </strong>
             <br/>
             新建一个文件夹，记住这个路径
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/33d26c9365f6a2eb60d9447ac7a7fd05.png">
              <br/>
              双击PyCharm图标，打开PyCharm进行安装
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4fc29eda8886519ec43e9a5f133b82fe.png"/>
              <br/>
              点击——“Don’t Send”
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7cbbad0c9d92f35e09bd006e4622d384.png"/>
              <br/>
              进入自定义UI界面，默认背景为“Darcula”，可将其调整为”Light”
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d87c0b1fd4e5c988e6d4ad3de6ed79c.png"/>
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a182993c21cb1661f7d5cc03475f600c.png"/>
              <br/>
              选择 —— “Open”
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6f185ec2158513657275bc3d375f5ea3.png"/>
              <br/>
              弹出窗口，浏览将刚才创建的文件路径
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fd286d49177bafe4604fa8f8cf07e211.png"/>
              <br/>
              点击 ——“Don’t show tips”
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/41cb878677a5770af7d97bf31fce08a3.png"/>
              <br/>
              选中“Code_Me”，右键点击 “New” —— “Python File”，新建Python文件
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/294bea1577e6091241934e5462db59fa.png"/>
              <br/>
              输入文件名：“test01"，双击创建
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/983931b279c849f6896fc6b5b58035e8.png"/>
              <br/>
              文件创建成功，Code_Me目录下生成一个test01.py文件。（以.py结尾的文件就是python文件）
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3aa12b11a7d9d821e1819a342d09068.png"/>
              <br/>
              <strong>
               三、添加Python解释器
              </strong>
              <br/>
              Python解释器设置，点击File——Settings——Project: Code_Me ——Python Interpreter——OK，添加Python解释器
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6d9f7f0c25b2505c9683cdc6e463d332.png"/>
              <br/>
              <strong>
               四、测试
              </strong>
              <br/>
              测试python程序运行是否成功，写一段代码，点击Run——Run,运行程序
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8218af7e3524cbbe9f742f76a5c768b3.png"/>
              <br/>
              <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/beaa8948851b54768bd1e0d00d285fb3.png"/>
              <br/>
              <strong>
               到这里就安装成功啦！！
              </strong>
