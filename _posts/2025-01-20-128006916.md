      目的：不同的程序需要不同的运行环境
     </li>
     <li id="ua55f21b8">
      方法：conda create -n 虚拟环境名 python3.8

      结果：在anaconda\envs 目录下有一个新的目录，里面是一个全新的python

    <h2 id="W962p">
     1、设置pip国内源
    </h2>
    <ul>
     <li id="ub113206c">
      原因：python编程经常需要调用其他的软件包，用pip命令安装软件包，默认情况下安装速度非常慢，所以需要设置pip国内源
     </li>
     <li id="uc4926441">
      在Anaconda配置里添加清华源，输入以下命令：
     </li>
    </ul>
    <p id="u3975dd6b">
     conda config --add channels
     <a href="https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/" rel="nofollow" title="Index of /anaconda/pkgs/free/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror">
      Index of /anaconda/pkgs/free/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
     </a>
    </p>
    <p id="ub654a7bb">
     conda config --add channels
     <a href="https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main" rel="nofollow" title="Index of /anaconda/pkgs/main/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror">
      Index of /anaconda/pkgs/main/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
     </a>
    </p>
    <p id="u5ed32505">
     conda config --add channels

    <h2 id="MBDpU">
     2、激活base环境
    </h2>
    <ul>
     <li id="u8ed44374">
      每次操作需要先激活base环境，才能创建或进入其他虚拟环境。
     </li>
    </ul>
    <p id="ua2ed5df1">
     1、输入conda active激活conda，进入base环境
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e4e3ad0b2656454e469757e29907f132.png"/>
    </p>
    <p id="u0bd6b354">
     2、输入python --version查看python版本，输入conda env list查看现有环境
    </p>
    <h2 id="nYz0L">
     3、方法
    </h2>
    <p id="ud3f0b7a8">
     1、输入d：切换到d盘，cd Anacoonda，cd ***（进入对应的地方）
    </p>
    <p id="u6428dbcc">
     2、输入conda create -n 虚拟环境名 python3.8
    </p>
    <p id="u0d4161e8">
     3、输入conda env list查看环境列表
    </p>
    <p id="u48aa36e7">
     4、激活（进入）虚拟环境（命令行有提示）比如：conda active 虚拟环境名
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/20e8a125c78a1c6ef30e37e235e6c0c4.png"/>
    </p>
    <p id="ua0718c3b">
     5、输入python检查虚拟环境python版本，并进入python开发环境，出现&gt;&gt;&gt;
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/68161f90d3111c1e4a00b7c572030b4d.png"/>
    </p>
    <p id="uff6cec03">
     6、退出python开发环境，键盘按Ctrl+z
    </p>
    <p id="uf849ef64">
     7、安装其他库，例如numpy，输入conda install numpy
    </p>
