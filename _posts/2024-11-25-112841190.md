     </a>
     一、控制系统框图
    </h2>
    <h3>
     <a id="1_17">
     </a>
     1.定义
    </h3>
    <p>
     由具有一定函数关系的环节组成的，且标有信号流向的图
    </p>
    <h3>
     <a id="2_20">
     </a>
     2.组成
    </h3>
    <p>
     1）信号线
     <br/>
     2）分支点：表示信号引出或测量的位置
     <br/>
     3）相加点：表示对两个以上的信号进行代数运算
     <br/>
     4）方框（环节）：表示对信号进行数学变换
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e96003310278ea24e7cda4e16e6f537c.png"/>
    </p>
    <h3>
     <a id="3_27">
     </a>
     3.方框图的等效变换
    </h3>
    <p>
     原则：1）前向通路传递函数乘积保持不变 2）回路中传递函数乘积保持不变

    <h2>
     <a id="_55">
     </a>
     二、信号流图
    </h2>
    <h3>
     <a id="1_56">
     </a>
     1.定义
    </h3>
    <p>
     用来表示一组线性代数方程的图
    </p>
    <h3>
     <a id="2_60">
     </a>
     2.组成
    </h3>
    <p>
     (1)节点:表示变量
     <br/>
     (2)支路:连接节点的有向线段
     <br/>
     (3)支路增益:表示两个变量的因果关系
    </p>
    <h3>
     <a id="3_64">
     </a>
     3.常用术语
    </h3>
    <p>
     (1) 输入节点: 只有输出支路的节点(又称源节点)。
     <br/>
     (2) 输出节点: 只有输入支路的节点(又称汇节点)。
     <br/>
     (3) 混合节点: 既有输入支路,又有输出支路的节点。
     <br/>
     (4) 前向通路: 从源节点开始到汇节点终止, 而且每个节点只通过一次的通路。
     <br/>
     (5) 不接触回路: 若一些回路没有任何公共节点和支路。
     <br/>
     (6) 回路增益: 回路中各支路增益(传输)的乘积。
    </p>
    <h3>
     <a id="4_71">
     </a>
     4.信号流图基本性质
    </h3>
    <p>
     （1）用节点表示变量
     <br/>
     其中：源节点代表输入量，
     <br/>
     汇节点代表输出量，
     <br/>
     混合节点代表各支路变量的代数和。
     <br/>
     （2）支路表示变量的传输和变换过程
     <br/>
     （3）增加一个具有单位传输的支路，可把混合节点
     <br/>
     化为汇节点
     <br/>
     （4）同一系统的信号流图不唯一。
    </p>
    <h3>
     <a id="5_80">
     </a>
     5.信号流图等效变换
    </h3>
    <p>
     （1）串联支路的总传输等于各支路传输的乘积；
     <br/>
     （2）并联支路的总传输等于各支路传输之和；
     <br/>
     （3）混合节点可以用移动支路的方法消去；
     <br/>
     （4）回环可用反馈连接化为等效支路。
    </p>
