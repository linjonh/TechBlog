    </h1>

 <article class="baidu_pl">

   <div class="htmledit_views" id="content_views">
    <p>
     用户体验设计中的场景服务设计，来自极客时间的课程归纳：
     <br/>
    </p>
    <div>
     <span style="color:#000000;">
      <strong>
       客户旅程：
      </strong>
     </span>
     <span style="color:#353535;">
      用户从接触产品的第一刻到完结的整个场景服务的完整旅程。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      <strong>
       前台服务：
      </strong>
     </span>
     <span style="color:#353535;">
      前台员工的服务流程设计与技术工具的流程设计。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      <strong>
       后台支撑：
      </strong>
     </span>
     <span style="color:#353535;">
      后台产品供应链、人员供给、安全保证等。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      <strong>
       平台运维：
      </strong>
     </span>
     <span style="color:#353535;">
      工具、数据、监控、基础设施搭建等。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      <strong>
       商业核算：
      </strong>
     </span>
     <span style="color:#353535;">
      成本收益控制、风险预估、分成体系、人力控制
     </span>
     <br/>
     <img alt="" height="747" src="https://i-blog.csdnimg.cn/blog_migrate/020c2c0eb28046d212bd518862e1b705.png" width="1200">
     </img>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#353535;">
      比如我们去宜家，从远远望见的“IKEA”Logo，到进门开始到逛店，然后选择自己喜欢的
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      物品去提货，到最终付款，预约送货或者自己取货离开，这个流程就是客户的旅程；而
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      IKEA 的工作人员，从进门的引导，到设置每个区域的服务人员，到最后取货、送货、结
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      账、一元冰淇淋的售卖等，这就是前台服务的设计；而后台的支撑，则是 IKEA 的产品供应
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      链，也就是其产品的生产和设计等；那么 IKEA 中台的数据工具、基础设施的种种建设，就
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      是平台的运维；最底层是商业核算，即 IKEA 整体在商业层面的控制和财务人力方面的支
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      撑。
     </span>
    </div>
    <div>
     <span style="color:#353535;">
      这样整体观察下来，就是一个商业场景服务设计的全部了。
     </span>
    </div>
   </div>
