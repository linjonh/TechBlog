     2、模块索引
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7aa52985382612bca9584aa2fe55bced.png#pic_center">
      <br/>
      在这个标准库中，有很多很强大的模块我们可以直接使用，
      <br/>
      并且标准库会随Python的安装一同安装。
     </img>
    </p>
    <pre><code class="prism language-python"><span class="token comment"># sys模块，它里面提供了一些变量和函数，使我们可以获取到Python解析器的信息</span>
<span class="token comment">#   或者通过函数来操作Python解析器</span>
<span class="token comment"># 引入sys模块</span>
<span class="token keyword">import</span> sys

<span class="token comment"># pprint 模块它给我们提供了一个方法 pprint() 该方法可以用来对打印的数据做简单的格式化</span>
<span class="token keyword">import</span> pprint

<span class="token comment"># sys.argv</span>
<span class="token comment"># 获取执行代码时，命令行中所包含的参数</span>
<span class="token comment"># 该属性是一个列表，列表中保存了当前命令的所有参数</span>
<span class="token comment"># print(sys.argv)</span>

<span class="token comment"># sys.modules</span>
<span class="token comment"># 获取当前程序中引入的所有模块</span>
<span class="token comment"># modules是一个字典，字典的key是模块的名字，字典的value是模块对象</span>
<span class="token comment"># pprint.pprint(sys.modules)</span>
<span class="token comment"># print(sys.modules)</span>

<span class="token comment"># sys.path</span>
<span class="token comment"># 他是一个列表，列表中保存的是模块的搜索路径</span>
<span class="token comment"># ['C:\\Users\\lilichao\\Desktop\\resource\\course\\lesson_06\\code',</span>
<span class="token comment"># 'C:\\dev\\python\\python36\\python36.zip',</span>
<span class="token comment"># 'C:\\dev\\python\\python36\\DLLs',</span>
<span class="token comment"># 'C:\\dev\\python\\python36\\lib',</span>
<span class="token comment"># 'C:\\dev\\python\\python36',</span>
<span class="token comment"># 'C:\\dev\\python\\python36\\lib\\site-packages']</span>
<span class="token comment"># pprint.pprint(sys.path)</span>

<span class="token comment"># sys.platform</span>
<span class="token comment"># 表示当前Python运行的平台</span>
<span class="token comment"># print(sys.platform)</span>

<span class="token comment"># sys.exit()</span>
<span class="token comment"># 函数用来退出程序</span>
<span class="token comment"># sys.exit('程序出现异常，结束！')</span>
<span class="token comment"># print('hello')</span>

<span class="token comment"># os 模块让我们可以对操作系统进行访问</span>
<span class="token keyword">import</span> os

<span class="token comment"># os.environ</span>
<span class="token comment"># 通过这个属性可以获取到系统的环境变量</span>
<span class="token comment"># pprint.pprint(os.environ)</span>
<span class="token comment"># pprint.pprint(os.environ['path'])</span>

<span class="token comment"># os.system()</span>
<span class="token comment"># 可以用来执行操作系统的名字</span>
os<span class="token punctuation">.</span>system<span class="token punctuation">(</span><span class="token string">'dir'</span><span class="token punctuation">)</span>
</code></pre>
