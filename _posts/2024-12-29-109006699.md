     <h3>
      简介
     </h3>
     <p style="text-align: left">
      基于 Vue 开发的界面可视化设计器，前端小白可以直接拖拽生成前端代码。
     </p>
     <h3>
      特性
     </h3>
     <ul>
      <li>
       <p>
        可视化拖拽布局
       </p>
      </li>
      <li>
       <p>
        支持数据关联和交互行为的可视化编辑
       </p>
      </li>
      <li>
       <p>
        支持扩展组件、组件属性、属性编辑器
       </p>
      </li>
     </ul>
     <h3>
      依赖
     </h3>
     <ul>
      <li>
       <p>
        vjform，可视化布局基于 vjform 扩展
       </p>
      </li>
      <li>
       <p>
        Lodash，使用了 get set forEach 等 API
       </p>
      </li>
      <li>
       <p>
        Vue，基于 v2.5.9 测试，理论上支持高于 v2.4.0 版本
       </p>
      </li>
      <li>
       <p>
        vuedraggable，实现拖拽布局
       </p>
      </li>
      <li>
       <p>
        vue-json-viewer，预览和复制设置定义的元数据
       </p>
      </li>
     </ul>
     <h3>
      Getting Start
     </h3>
     <p style="text-align: left">
      克隆项目，执行
     </p>
     <p style="text-align: left">
      <code>
       npm install
      </code>
     </p>
     <p style="text-align: left">
      之后
     </p>
     <p style="text-align: left">
      <code>
       npm run dev
      </code>
     </p>
