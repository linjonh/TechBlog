    <h2 align="center">
     <span style="font-family:Calibri">
      Windows
     </span>
     驱动程序开发相关
    </h2>
    <h3>
     <span style="font-family:Cambria">
      1.
     </span>
     驱动的注册表位置：
    </h3>
    <p>
     <strong>
      <span style="font-size:14px">
       1.
      </span>
     </strong>
     <span style="font-size:14px">
      <strong>
       硬件子键：
      </strong>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      HKEY_LOCATION_MACHINE\SYSTEM\ControlSet001\Enum
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      指明硬件的HID，VID，DID等跟硬件和硬件厂商相关的信息。
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       2.
      </span>
     </strong>
     <span style="font-size:14px">
      <strong>
       服务子健：
      </strong>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      HKEY_LOCATION_MACHINE\SYSTEM\ControlSet001\Services
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      它主要是跟驱动相关的，指明驱动的路径，以及驱动名字。
     </span>
    </p>
    <p>
     <strong>
      <span style="font-family:Calibri; font-size:14px">
       3.
      </span>
      <span style="font-size:14px">
       类子健：
      </span>
     </strong>
    </p>
    <p>
     <span style="font-size:14px">
      HKEY_LOCATION_MACHINE\SYSTEM\ControlSet001\Control\Class，
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      指明INF文件。
     </span>
    </p>
    <p>
     <span style="font-family:Calibri; font-size:14px">
     </span>
    </p>
    <h3>
     <span style="font-family:Cambria">
      2.
     </span>
     <span style="font-family:Cambria">
      INF
     </span>
     详解
    </h3>
    <p>
     <span style="font-family:Calibri; font-size:14px">
      INF
     </span>
     文件是一个文本文件，由许多按层次结构排列的节组成
    </p>
    <p>
     <strong>
      <span style="color:red">
       *
      </span>
      <span style="color:red">
       规则一：
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:black">
       INF
      </span>
      <span style="color:black">
       文件是分节的，每一个
      </span>
      <span style="color:black">
       INF
      </span>
      <span style="color:black">
       文件有许多的节组成，节名用方括号括起来。
      </span>
     </strong>
    </p>
    <p>
     <span style="color:black">
      这些节名有些是系统定义好的，有一些是用户自定义的。每一个节名最长为
     </span>
     <span style="color:black">
      <span style="font-family:Calibri">
       255
      </span>
     </span>
     <span style="color:black">
      个字符（
     </span>
     <span style="color:black">
      <span style="font-family:Calibri">
       Windows 2000/XP/2003
      </span>
     </span>
     <span style="color:black">
      操作系统中）或
     </span>
     <span style="color:black">
      <span style="font-family:Calibri">
       28
      </span>
     </span>
     <span style="color:black">
      个字符（
     </span>
     <span style="color:black">
      <span style="font-family:Calibri">
       Windows 98
      </span>
     </span>
     <span style="color:black">
      操作系统中）。节与节之间没有先后顺序的区别，另外，同一个
     </span>
     <span style="color:black">
      <span style="font-family:Calibri">
       INF
      </span>
     </span>
     <span style="color:black">
      文件中如果出现两个同样的节名，则系统会自动将这两个节名下面的条目合并到一起。
     </span>
     <span style="font-family:Calibri">
      <span style="color:black">
       <br/>
      </span>
      <strong>
       <span style="color:red">
        *
       </span>
      </strong>
     </span>
     <strong>
      <span style="color:red">
       规则二：
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:black">
       在节与节之间的内容叫条目，每一个节又是由许多的条目组成的，每一个条目都是由形如
      </span>
      <span style="color:black">
       “signature="$CHICAGO$"”
      </span>
      <span style="color:black">
       的形式组成的。如果每一个条目的等号后有多个值，则每一个值之间用
      </span>
      <span style="color:black">
       “
      </span>
      <span style="color:black">
       ，
      </span>
      <span style="color:black">
       ”
      </span>
      <span style="color:black">
       号分隔开。
      </span>
     </strong>
     <span style="color:black">
      <span style="font-family:Calibri">
      </span>
     </span>
    </p>
    <p>
     <strong>
      <span style="color:red">
       *
      </span>
      <span style="color:red">
       规则三：
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:black">
       INF
      </span>
      <span style="color:black">
       文件对大小写不敏感。
      </span>
     </strong>
     <span style="font-family:Calibri">
      <span style="color:black">
       <br/>
      </span>
      <strong>
       <span style="color:red">
        *
       </span>
      </strong>
     </span>
     <strong>
      <span style="color:red">
       规则四：
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:black">
       “
      </span>
      <span style="color:black">
       ；
      </span>
      <span style="color:black">
       ”
      </span>
      <span style="color:black">
       号后面的内容为注释。
      </span>
     </strong>
     <span style="color:black">
      <br/>
     </span>
     <strong>
      <span style="color:red">
       *
      </span>
      <span style="color:red">
       规则五：
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:black">
       如果一个条目的内容过多，在一行无法书写完全，则用
      </span>
      <span style="color:black">
       “\”
      </span>
      <span style="color:black">
       将一行内容书写为多行。
      </span>
     </strong>
    </p>

    <h4>
     <span style="font-family:Calibri; font-size:24px">
      2.2 SourceDisksNames
     </span>
    </h4>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      SourceDisksNames
     </span>
     <span style="color:black">
      节指明设备驱动程序所在的磁盘或
     </span>
     <span style="color:black">
      CD-ROM
     </span>
     <span style="color:black">
      ，其各项的格式如下：
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      diskid=disk-description,disk-label,disk-serial-number   (Windows 98)
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      diskid=disk-description[,[tagfile][,unused,path]]  (Windows 2000)
     </span>
    </p>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      其中，
     </span>
     <span style="color:black">
      “diskid”
     </span>
     <span style="color:black">
      指出磁盘驱动器的编号，它是正整型数，通常从
     </span>
     <span style="color:black">
      1
     </span>
     <span style="color:black">
      开始，且不能重复；
     </span>
     <span style="color:black">
      “disk-description”
     </span>
     <span style="color:black">
      表示磁盘的描述信息，他通常为一个字符串。对于
     </span>
     <span style="color:black">
      Windows 98
     </span>
     <span style="color:black">
      ，
     </span>
     <span style="color:black">
      “disk-label”
     </span>
     <span style="color:black">
      指出磁盘的卷标；
     </span>
     <span style="color:black">
      “disk-serial-numbe”
     </span>
     <span style="color:black">
      指出磁盘的序列号。对于
     </span>
     <span style="color:black">
      Windows 2000
     </span>
     <span style="color:black">
      ，
     </span>
     <span style="color:black">
      “tagfile”
     </span>
     <span style="color:black">
      指出磁盘标签文件的文件名；
     </span>
     <span style="color:black">
      “unused”
     </span>
     <span style="color:black">
      保留未用；
     </span>
     <span style="color:black">
      “path”
     </span>
     <span style="color:black">
      指出驱动程序所在的路径；
     </span>
     <span style="color:black">
      “[ ]”
     </span>
     <span style="color:black">
      表示可选参数。
     </span>
    </p>
    <h4>
     <span style="font-family:Calibri; font-size:24px">
      2.3 SourceDisksFiles
     </span>
    </h4>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      SourceDisksFiles
     </span>
     <span style="color:black">
      节指明设备驱动程序的文件全路径，其各项格式如下：
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      filename=diskid[,subdir] (Windows 98)
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      filename=diskid[,[,subdir][,size]]   (Windows 2000)
     </span>
    </p>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      其中，
     </span>
     <span style="color:black">
      “filename”
     </span>
     <span style="color:black">
      指出驱动程序的文件名；
     </span>
     <span style="color:black">
      “diskid”
     </span>
     <span style="color:black">
      指出磁盘驱动器的编号；
     </span>
     <span style="color:black">
      “subdir”
     </span>
     <span style="color:black">
      指出该文件在磁盘上的路径；
     </span>
     <span style="color:black">
      “size”
     </span>
     <span style="color:black">
      指出该文件未经压缩时的大小，以字节为单位。
     </span>
    </p>
    <h4>
     <span style="font-family:Calibri; font-size:24px">
      2.4 DestinationDirs
     </span>
    </h4>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      DestinationDirs
     </span>
     <span style="color:black">
      节指明
     </span>
     <span style="color:black">
      INF
     </span>
     <span style="color:black">
      文件和设备驱动程序的目标目录。
     </span>
     <span style="color:red">
      当
     </span>
     <span style="color:red">
      INF
     </span>
     <span style="color:red">
      文件使用
     </span>
     <span style="color:red">
      CopyFiles
     </span>
     <span style="color:red">
      、
     </span>
     <span style="color:red">
      DelFiles
     </span>
     <span style="color:red">
      、
     </span>
     <span style="color:red">
      RenFiles
     </span>
     <span style="color:red">
      指令时，则必须包含该节
     </span>
     <span style="color:black">
      。其各项的格式如下：
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      DefaultDestDir=dirid[,subdir]
     </span>
    </p>
    <p style="background:#fefef2">
     <span style="color:black">
      File-list-section=dirid[,subdir]
     </span>
    </p>
    <p align="left" style="background:#fefef2">
     <span style="color:black">
      其中，
     </span>
     <span style="color:black">
      “DefaultDestDir”
     </span>
     <span style="color:black">
      是一个项名称，代表文件复制、删除、重命名操作的缺省目标目录；
     </span>
     <span style="color:black">
      “file-list-section”
     </span>
     <span style="color:black">
      指出
     </span>
     <span style="color:black">
      CopyFiles
     </span>
     <span style="color:black">
      、
     </span>
     <span style="color:black">
      DelFiles
     </span>
     <span style="color:black">
      、
     </span>
     <span style="color:black">
      RenFiles
     </span>
     <span style="color:black">
      指令所引用的节；
     </span>
     <span style="color:black">
      “dirid”
     </span>
     <span style="color:black">
      指出目标目录值，见下表；
     </span>
     <span style="color:black">
      “subdir”
     </span>
     <span style="color:black">
      指出
     </span>
     <span style="color:black">
      dirid
     </span>
     <span style="color:black">
      目录下的子目录。
     </span>
    </p>
    <p align="center" style="background:#fefef2">
