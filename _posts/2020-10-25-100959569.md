     <p>
      倘若花了大价钱买了一些电子设备，却不知道其中的接口怎么使用，是不是非常苦恼。感觉浪费了一半的功能，笔者也不是很清楚，于是就查阅一些资料，总结一下。因为电子设备的更新换代又那么的快，我只能说现阶段的常见的电子设备接口，时间是2019年8月4日。
      <br/>
      <strong>
       1、图像音视频接口
      </strong>
      <br/>
      <strong>
       A、VGA接口（视频：模拟信号）
      </strong>
      <br/>
      <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/fd71b6778045c40b26e1297f791ba902.png">
       <br/>
       <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/aace4fb29a1693b9a65bfa1ed010d43e.png">
        <br/>
        VGA是IBM在1987年随PS/2机一起推出的一种视频传输标准。蓝色的插头也是最有辨识度的一个接口：VGA 接口有3排针脚，每排5个，共有15针。VGA接口采用的是模拟信号，不支持音频。
        <br/>
        <strong>
         B、DVI接口（视频：数字信号）
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/9c92621de1a14d049f875556785ca55c.png">
         <br/>
         DVI 接口共有三种规格，分别是 DVI-I（Integrated混合），DVI-D（Digital数字）和 DVI-A（Analog模拟），其中 DVI-I 和 DVI-D 还分为“单通道”和“双通道”两种类型。DVI接口的传输信号采用全数字格式，与之对应的是采用模拟信号的VGA接口。其为数字信号，略好于VGA，但仍不支持音频。
         <br/>
         <strong>
          C、HDMI接口（音视频：数字信号）
         </strong>
         <br/>
         <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/8e2e24c56f8ab239e833d21c27f371fc.png">
          <br/>
          HDMI 相比 DVI 接口支持了音频的输出，并且有着更高的带宽，它可以算是目前最主流的视频输出接口，由于接口变得更小，也被广泛应用于电视机、显示器、笔记本电脑等设备上。
          <br/>
          <strong>
           D、DP接口（音视频：数字信号）
          </strong>
          <br/>
          <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/e11df249a625714a2d673a6ebba6f511.png"/>
         </img>
        </img>
       </img>
      </img>
     </p>
     <p>
      在目前的显示器中几乎都标配了DP接口，它可以看做是 HDMI 接口的升级版，但是内部数据的传输方式与 DVI 和 HDMI 完全不同，有着更高的带宽。
      <br/>
      <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/72054132b3f27745579849dee4fc76d5.png">
       <br/>
       <strong>
        E、Type-C接口
       </strong>
       <br/>
       <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/69242d4273102998bd94bf1b13679cdc.png">
        <br/>
        在应用于手机上时，很多人称它为 Type-C 接口，但是应用在电脑上时，更多的人则叫它 USB-C 接口，虽然叫法不同，但是在本质上是同一种接口。
        <br/>
        <strong>
         “Type”指的是接口类型
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/098f40134038ed7efb9e2dc3e0e134c6.png"/>
        <br/>
        <strong>
         “USB”指的是接口规范标准
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/53c553e612c26c574ace0ee48c93a511.png"/>
        <br/>
        <strong>
         【拓展：type-c理论功能】
        </strong>
        <br/>
        <strong>
         ①手机直连网线
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/d9b71f0076aa44d798a815f28b5e4a45.png"/>
        <br/>
        <strong>
         ②手机连接U盘
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/c727528fdb4bf08f4380f070e7c87262.png"/>
        <br/>
        <strong>
         ③手机投屏
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/9f717239fd14235e8583e02b7f9cac0a.png"/>
        <br/>
        <strong>
         F、色差分量接口（视频）
        </strong>
        <br/>
        色差分量接口是色差接口使用不是很普遍，主要的原因是一些CRT电视机并没有提供色差分量的输入接口。可以在一些专业级视频工作站/编辑卡专业级视频设备或高档影碟机等家电上看到有YUV、YcbCr、Y/B-Y/R-Y等标记的接口标识，虽然其标记方法和接头外形各异但都是指的同一种接口——色差端口( 也称分量视频接口) 。
        <br/>
        简单的说，相比过去的AV和S端子，色差是将信号分为红、绿、蓝三种基色来输入的。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/13b3d90c3b32e0578ad1b6a2c144d92c.png"/>
        <br/>
        <strong>
         G、AV复合视频接口（音视频）
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/e3c67b43be06c8e4b69d2f9ad423aefa.png"/>
        <br/>
        简单理解就是，我们将色差分量接口的三个接线（绿亮度、蓝x轴方向、红y轴方向）用一个复合视频接口代替（×××的）。外加两个音频接口（也就是下面讲到的N、RCA模拟音频接口），就是AV复合视频接口。
        <br/>
        <strong>
         H、S-视频接口（视频）
        </strong>
        <br/>
        S端子也是非常常见的端子，其全称是Separate Video，也称为 SUPER VIDEO。S-Video连接规格是由日本开发的一种规格，S指的是“SEPARATE（分离）”，它将亮度和色度分离传输，避免了混合视频信号传输时亮度和色度的相互干扰。S端子实际上是一种五芯接口，由两路视频亮度信号、两路视频色度信号和一路公共屏蔽地线共五条芯线组成。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/b4afa43b3a0caecba66cfcf3455fc299.png"/>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/fdd6153d28cbcefb1b0be1ad92870570.png"/>
        <br/>
        <strong>
         I 、RS-232C接口
        </strong>
        <br/>
        其中EIA(Electronic Industry Association)代表美国电子工业协会，RS（Recommended standard）代表推荐标准，232是标识号，C代表RS232的最新一次修改。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/1df0ab6654331a50d4d1e916a393e5f9.png"/>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/a3d43d9499b4646f833a6d4aa626b0ed.png"/>
        <br/>
        <strong>
         【拓展：是不是很像VGA】
        </strong>
        <br/>
        ①VGA接口最早可以追溯到1987年，VGA 接口有3排针脚，每排5个，共有15针。
        <br/>
        ②VGA接口采用的是模拟信号，这种信号有一个明显缺点，就是抗干扰能力弱。
        <br/>
        ③RS232接口是计算机上的通讯接口的一种。
        <br/>
        ④通常 RS-232 接口以9个引脚或是25个引脚的型态出现，一般计算机上会有两组 RS-232 接口，分别为 COM1 和 COM2。
        <br/>
        <strong>
         L、光纤音频接口（音频）
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/9221c362dca91e61a9ef957d4608ee20.png"/>
        <br/>
        光纤音频接口TosLink，全名Toshiba Link，这是日本东芝(TOSHIBA)公司较早开发并设定的技术标准，在视听器材的背板上有Optical作标识。现在几乎所有的数字影音设备都具备这种格式的接头。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/276f03f18f724f9fc21ced92a6d5100d.png"/>
        <br/>
        <strong>
         M、3.5mm立体声接口（音频）
        </strong>
        <br/>
        它可以在实现耳机的功能的基础上，同时实现麦克风和线控等其他功能。目前市面上常见的，有OMTP和CTIA两种标准。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/8a244c3e0b86702ad287ab3e2fcd1cf8.png"/>
        <br/>
        <strong>
         N、RCA模拟音频接口（音频）
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/9c36678cef36e84fe9d6284bc5c07797.png"/>
        <br/>
        <strong>
         O、XLR接口
        </strong>
        <br/>
        最初端子为"Cannon X"系列，之后的版本加入了弹簧锁（Latch）成为"Cannon XL"系列，接着在端子接触面以橡胶包着（Rubber），成为了其缩写XLR的来源。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/441e97a42ebee4a752365af6c033bf7e.png"/>
        <br/>
        <strong>
         P、IEEE1394接口
        </strong>
        <br/>
        计算机接口IEEE1394，俗称火线接口，主要用于视频的采集，在INTEL高端主板与数码摄像机(DV)上可见。 IEEE 1394，别名火线（FireWire）接口，是由苹果公司领导的开发联盟开发的一种高速度传送接口，数据传输率一般为800Mbps。火线（FireWire）是苹果公司的商标。Sony的产品称这种接口为iLink。
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/ba7cf2e2b051a5092c38794097af3be1.png"/>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/3c53b63c37a9669033d73ed19f3f2fcd.png"/>
        <br/>
        <strong>
         Q、RF电视信号接口
        </strong>
        <br/>
        <img alt="电子设备常见的音视频接口" src="https://i-blog.csdnimg.cn/blog_migrate/07b664c1d218cb15275e03b4e0b5b580.png"/>
       </img>
      </img>
     </p>
