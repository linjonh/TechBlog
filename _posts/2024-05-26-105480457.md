    <p>
     <strong>
      数据结构静态的描述了数据之间的关系
     </strong>
    </p>
    <p>
     高效的程序需要在数据结构的基础上设计和选择算法
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/588a777e375a068f18ca0b82b3560b07.png"/>
    </p>
    <p>
     <strong>
      -算法是特定问题求解步骤的描述
      <br/>
      -在计算机中表现为指令的有限序列
     </strong>
    </p>
    <p>
     算法是独立存在的一种解决问题的方法和思想
     <br/>
     对于算法而言，语言并不重要，重要的是思想
    </p>
    <p>
     <strong>
      算法的特征
     </strong>
    </p>
    <ul>
     <li>
      -输入：算法具有0个或多个输入
     </li>
     <li>
      -输出：算法至少有一个或者多个输出
     </li>
     <li>
      -有穷性：算法在有限的步骤之后会自动结束而不会无限循环
     </li>
     <li>
      -确定性：算法中的每一步都有确定的含义，不会出现二义性
     </li>
     <li>
      -可行性：算法的每一步都是可行的
     </li>
    </ul>
    <p>
     <strong>
      正确性
     </strong>
    </p>
    <ul>
     <li>
      -算法对于合法数据能够得到满足要求的结果
     </li>
     <li>
      -算法能够处理非法输入，并得到合理的结果
     </li>
     <li>
      -算法对于边界数据和压力数据都能得到满足要求的结果
     </li>
    </ul>
    <p>
     <strong>
      可读性
     </strong>
    </p>
    <ul>
     <li>
      -算法要方便阅读，理解和交流
     </li>
    </ul>
    <p>
     <strong>
      健壮性
     </strong>
    </p>
    <ul>
     <li>
      -算法不应该产生莫名其妙的结果
     </li>
    </ul>
    <p>
     <strong>
      性价比
     </strong>
    </p>
    <ul>
     <li>
      -利用最少的资源得到满足要求的结果
     </li>
    </ul>
    <p>
     <strong>
      小结
     </strong>
     <br/>
     算法是为了解决实际问题而存在的
     <br/>
     数据结构是算法处理数据问题的载体
     <br/>
     数据结构和算法相辅相成，共同解决问题
    </p>
