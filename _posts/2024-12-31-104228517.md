    <h2 id="%E4%B8%80%E3%80%81%E5%9B%BD%E5%86%85%E5%B8%B8%E7%94%A8%E7%9A%84%E9%95%9C%E5%83%8F%E7%AB%99%E7%82%B9">
     一、国内常用镜像站点
    </h2>
    <p>
     Python默认的pip包来源是国外的Pypi服务站点：https://pypi.python.org/simple。对于国内用户来说，由于网络原因，下载速度太慢，因此推荐使用国内镜像源站点。目前，比较常用的pip镜像站点有：
    </p>
    <ul>
     <li>
      <strong>
       清华大学
       <a href="https://pypi.tuna.tsinghua.edu.cn/simple/" rel="nofollow">
        https://pypi.tuna.tsinghua.edu.cn/simple/
       </a>
      </strong>
     </li>
     <li>
      <strong>
       中国科大：
       <a href="https://mirrors.ustc.edu.cn/pypi/web/simple/" rel="nofollow">
        https://mirrors.ustc.edu.cn/pypi/web/simple/
       </a>
      </strong>
     </li>
     <li>
      <strong>
       豆瓣：
       <a href="http://pypi.doubanio.com/simple/" rel="nofollow">
        http://pypi.doubanio.com/simple/
       </a>
      </strong>
     </li>
     <li>
      <strong>
       阿里云：
       <a href="http://mirrors.aliyun.com/pypi/simple/" rel="nofollow">
        http://mirrors.aliyun.com/pypi/simple/
       </a>
      </strong>
     </li>
    </ul>
    <p>
     本人推荐优先使用清华大学的镜像站点。
    </p>
    <h2 id="%E4%BA%8C%E3%80%81pip%E4%BD%BF%E7%94%A8%E5%9B%BD%E5%86%85%E9%95%9C%E5%83%8F%E7%AB%99%E7%82%B9">
     二、pip使用国内镜像站点
    </h2>
    <p>
     <strong>
      方式一：直接在 pip install xxx 后加上镜像源地址。
     </strong>
     例如下载numpy包：
    </p>
    <pre class="has"><code class="language-bash">   pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple/</code></pre>
    <p>
     但是利用这种方式，每次下载python包，都需要在后面加上镜像源地址，操作繁琐，不建议。
    </p>
    <p>
     <strong>
      方式二：修改配置文件
     </strong>
    </p>
    <p>
     目前，在windows系统上使用python安装包安装时，都会自带安装pip软件，此时可以通过下述方式修改pip的配置文件。
    </p>
    <p>
     首先在C:\Users\×××目录中创建名为pip的文件夹（×××为用户名），然后在里面创建名为pip.ini的文件，最后在该文件中添加以下内容，保存即可。
    </p>
    <pre class="has"><code>[global]
index-url=https://pypi.tuna.tsinghua.edu.cn/simple
timeout = 6000
 
[install]
trusted-host=pypi.tuna.tsinghua.edu.cn
disable-pip-version-check = true</code></pre>
    <p>
     <em>
      上述方式对于本人配置（windows10系统，python 3.7.6）有效，其他windows系统和python版本可按此方式尝试一下。
     </em>
    </p>
    <p>
     <em>
      说明：请注意，该方式不适合于linux系统和MacOS。
     </em>
    </p>
    <h2 id="%E4%B8%89%E3%80%81Pycharm%E4%BD%BF%E7%94%A8%E5%9B%BD%E5%86%85%E9%95%9C%E5%83%8F%E7%AB%99%E7%82%B9">
     三、Pycharm使用国内镜像站点
    </h2>
    <p>
     目前，Pycharm作为主流的python集成开发环境，受到广泛应用。因此，介绍下如何在Pycharm中添加国内镜像站点。
    </p>
    <ol>
     <li>
      打开：File-&gt;setting...，在该配置页面中，找到Project Interpreter；
     </li>
     <li>
      点击package行右侧“+”符号，在打开界面中，点击底部的Manage Repositories；
     </li>
     <li>
      在打开的界面中，直接点击“+”号，添加想要使用的镜像站点地址即可。当然，也可以添加多个国内站点。
     </li>
    </ol>
    <p>
     <em>
      本人使用的Pycharm版本是：PyCharm 2019.3.2 (Community Edition) 64位
     </em>
    </p>
