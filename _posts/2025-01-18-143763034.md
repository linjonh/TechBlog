    <h3>
     项目基础介绍和主要编程语言
    </h3>
    <p>
     QSTrader 是一个基于 Python 的开源量化交易回测框架，由 QuantStart 公司开发。该项目旨在为量化交易策略提供一个模块化、松耦合的回测平台。QSTrader 的核心设计理念是通过模块化的方式，将信号生成、投资组合构建、风险管理、执行和模拟经纪账户等功能分离，使得用户可以根据自己的需求进行扩展和定制。
    </p>
    <h3>
     项目核心功能
    </h3>
    <ol>
     <li>
      <strong>
       模块化设计
      </strong>
      ：QSTrader 采用模块化设计，允许用户根据需要替换或扩展各个模块，从而实现高度定制化的回测功能。
     </li>
     <li>
      <strong>
       回测引擎
      </strong>
      ：项目提供了一个基于时间表驱动的回测引擎，支持长期和短期的股票和 ETF 交易策略。
     </li>
     <li>
      <strong>
       性能统计
      </strong>
      ：QSTrader 能够生成典型的“tearsheet”性能评估报告，并支持通过 JSON 导出统计数据，方便外部软件进行分析。
     </li>
     <li>
      <strong>
       开源免费
      </strong>
      ：QSTrader 采用 MIT 许可证，完全免费且无限制，适用于研究和商业应用。
     </li>
    </ol>
    <h3>
     项目最近更新的功能
    </h3>
    <p>
     QSTrader 最近更新了以下功能：
    </p>
    <ol>
     <li>
      <strong>
       Python 3.9 至 3.12 支持
      </strong>
      ：项目已更新，支持 Python 3.9、3.10、3.11 和 3.12 版本，确保与最新的 Python 生态系统兼容。
     </li>
     <li>
      <strong>
       依赖包更新
      </strong>
      ：更新了项目所需的依赖包，确保与最新版本的库兼容，减少潜在的依赖冲突。
     </li>
     <li>
      <strong>
       示例策略更新
      </strong>
      ：增加了新的示例策略，帮助用户更好地理解和使用 QSTrader 进行回测。
     </li>
     <li>
      <strong>
       文档改进
      </strong>
      ：改进了项目的文档，增加了详细的教程和示例，帮助新手用户快速上手。
     </li>
    </ol>
    <p>
     QSTrader 是一个功能强大且灵活的开源量化交易回测框架，适合量化交易爱好者和专业人士使用。通过其模块化的设计和丰富的功能，用户可以轻松地构建和测试自己的交易策略。
