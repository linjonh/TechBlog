    <h3>
     <a id="EFT_0">
     </a>
     安规测试之-群脉冲测试（EFT）
    </h3>
    <h3>
     <a id="_2">
     </a>
     目的
    </h3>
    <p>
     验证电子设备对
     <strong>
      感性负载切换
     </strong>
     、
     <strong>
      继电器触点弹跳
     </strong>
     或者
     <strong>
      高压开关切换
     </strong>
     等引起的瞬时扰动的
     <strong>
      抗干扰能力
     </strong>
    </p>
    <p>
     单个脉冲的能量是很小的，不会对设备造成影响，但是如果多个脉冲组成的脉冲群，它产生的干扰信号会对我们的设备线路
     <strong>
      结电容进行充电
     </strong>
     ，当它的能量积累到一定程度的时候，它就会引起我们整个设备的误动作。
    </p>
    <h3>
     <a id="_7">
     </a>
     脉冲群发生器
    </h3>
    <p>
     大概就长这样
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a92b03cabe9dc71749316c206156676d.png#pic_center">
      <br/>
      EUT的电源打开，我们的EUT才有一个电源输入
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1835c063d4f4e74e43aed5819326c7a6.png"/>
    </p>
    <h3>
     <a id="_14">
     </a>
     测试标准
    </h3>
    <ol>
     <li>
      GB17626.4-2008
     </li>
     <li>
      IEC61000-4-4-2004
     </li>
    </ol>
    <p>
     国标测试中分为五个等级：500V、1Kv、2Kv、4Kv、开放等级（客户指定），这几个等级是针对电源口的。
     <br/>
     I/O口（信号口和数据口）也有等级：250V、500V、1Kv、2Kv、开放等级
    </p>
    <h3>
     <a id="_22">
     </a>
     参数调整
    </h3>
    <p>
     耦合电源的输出方式：
     <strong>
      2pin的插头的话就选L1和N
     </strong>
     ，
     <strong>
      3pin插头的话就选L1、N、PE
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/77c05e8eb972bb20906912f6303364c8.png#pic_center">
      <br/>
      参数设置好了以后，就按右上角的EUT开关，然后按运行Run就可以自动测试了
     </img>
    </p>
    <h3>
     <a id="_29">
     </a>
     注意事项
    </h3>
    <ol>
     <li>
      测试过程中要打开同轴测试
     </li>
     <li>
      测试过程中测试产品的主要功能有没有损坏
     </li>
     <li>
      触摸设备显示屏，看有没有跳点现象
     </li>
     <li>
      观察有没有闪屏、雪花、关机、自动重启等现象
     </li>
    </ol>
    <p>
     ±2KV脉冲群测试时：
     <br/>
     1、触屏不会乱跳，屏幕不会闪屏，其中手指同轴（X轴或Y轴）时出现跳点是正常的；
     <br/>
     2、摄像头拍照录影正常，不会出现相机掉线或无法连接等报错；
     <br/>
     3、打印功能正常，不出现打印机无法连接和打印内容错乱等异常；
     <br/>
     4、以太网功能正常，不出现PING过程中掉线和掉线后不能恢复异常；
     <br/>
     备注：单项测试需按被测终端实际成品而定
    </p>
