    <p align="center" class="MsoNormal" style="margin:0cm 0cm 0pt; text-align:center">
     <span lang="EN-US" style="font-size:14pt">
      <span style="font-family:Calibri">
       SCCM2012
      </span>
     </span>
     <span style="font-family:宋体; font-size:14pt">
      系列之一
     </span>
    </p>
    <p align="center" class="MsoNormal" style="margin:0cm 0cm 0pt; text-align:center">
     <span style="font-family:宋体; font-size:14pt">
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       2012
      </span>
     </span>
     <span style="font-family:宋体">
      年
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       5
      </span>
     </span>
     <span style="font-family:宋体">
      月，微软发布了
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      。
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center
      </span>
     </span>
     <span style="font-family:宋体">
      系列产品对微软技术工程师来说并不陌生，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM
      </span>
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCOM
      </span>
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCVMM
      </span>
     </span>
     <span style="font-family:宋体">
      等产品都是微软解决方案中经常用到的。
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      较之前的产品有哪些改进呢？
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      能否被客户欣然接受呢？带着这个问题，我参加了微软的
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      产品训练营。
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
      </span>
     </span>
     <span style="font-family:宋体">
      参加完训练营后，总的感觉是有喜有忧。喜的是
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      中的产品性能确实有改进，在云端融合方面加强了很多，拿得出手的产品改进也不少，相信能忽悠不少客户。忧的是
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      改变了销售模式，不再单独出售
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM
      </span>
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCOM
      </span>
     </span>
     <span style="font-family:宋体">
      等产品，而是要把
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      中的八个产品当成一个套包进行销售。
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt; text-indent:21pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      整体销售当然有好处，强制用户使用
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       ITIL
      </span>
     </span>
     <span style="font-family:宋体">
      来管理
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       IT
      </span>
     </span>
     <span style="font-family:宋体">
      。同时强化了
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center
      </span>
     </span>
     <span style="font-family:宋体">
      的平台概念，淡化了产品概念。用户把这八个产品一买，基本上其他厂商在这个领域就没啥事干了。缺点当然也有，以前是推销一两个产品，现在是推销八个产品，这难度可是大大地不同了。不是所有的用户都会按照
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       ITIL
      </span>
     </span>
     <span style="font-family:宋体">
      的标准去考虑采购
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center
      </span>
     </span>
     <span style="font-family:宋体">
      中的所有产品，也不是所有的客户对
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center
      </span>
     </span>
     <span style="font-family:宋体">
      中的八个产品都感兴趣。而且
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      中的八个产品水准不同，其他厂商肯定会群起而攻之，打击
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center
      </span>
     </span>
     <span style="font-family:宋体">
      产品中最薄弱的环节。想到这里，我不禁暗暗担心。尤其担心的是有些客户已经针对
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCOM2007
      </span>
     </span>
     <span style="font-family:宋体">
      或
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM2007
      </span>
     </span>
     <span style="font-family:宋体">
      报了预算，但还没有下单。万一
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      正式发布后，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM2007
      </span>
     </span>
     <span style="font-family:宋体">
      之类的产品不能再单独下单，那就麻烦了。
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt; text-indent:21pt">
     <span style="font-family:宋体">
      咨询了微软的销售后，总算是打消了疑虑。
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      的产品优惠很大，基本上八个产品的价格也就是和之前版本两三个产品的价格差不多。而且购买一套
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      后，在虚拟化平台使用是不受限制的，这对致力于大型
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       IDC
      </span>
     </span>
     <span style="font-family:宋体">
      建设的企业来说有很大的诱惑。这样相当于告诉用户，你只要买上两三个产品，其他的基本都是送的。这样的话，用户何乐而不为呢？
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       System Center 2012
      </span>
     </span>
     <span style="font-family:宋体">
      中向用户推销两三个有特点的产品，并非难事。
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM
      </span>
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCOM
      </span>
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCVMM
      </span>
     </span>
     <span style="font-family:宋体">
      都是推销解决方案时攻城拔寨的利器。产品功能强大，口碑较好，有很好的群众基础。
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt; text-indent:21pt">
     <span style="font-family:宋体">
      于是我准备研究一下
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM2012
      </span>
     </span>
     <span style="font-family:宋体">
      等产品，研究一下新产品的部署，配置，使用，看看新产品的改进中有哪些亮点。我的研究工作准备从
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM2012
      </span>
     </span>
     <span style="font-family:宋体">
      开始，
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM
      </span>
     </span>
     <span style="font-family:宋体">
      是提高客户端管理效率的强大工具。我们一般会使用
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       SCCM
      </span>
     </span>
     <span style="font-family:宋体">
      的下列基本功能：
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       1、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      搜集客户机硬件资产
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       2、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      搜集客户端软件清单
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       3、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      客户机软件分发
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       4、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      客户端更新管理
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       5、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      客户端操作系统部署
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:-18pt">
     <span lang="EN-US">
      <span style="font-family:Calibri">
       6、
      </span>
      <span style="font:7pt/normal 'Times New Roman'">
      </span>
     </span>
     <span style="font-family:宋体">
      客户机远程协助
     </span>
     <span lang="EN-US">
     </span>
    </p>
    <p class="MsoListParagraph" style="margin:0cm 0cm 0pt 18pt; text-indent:0cm">
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       SCCM2012
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      的改进之处，按照微软的说法是这样的：
     </span>
     <span lang="EN-US" style="color:#0070c0">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       1
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      、以用户为中心：真正的以人为本，例如对于用户来说，利用
     </span>
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       SCCM 2012
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      可以识别用户在哪找设备下工作，从而给用户颁发哪些类型的软件；对于管理人员来说，可以控制员工在任何地点任何时间工作。此外，可以让用户从任何地方任何设备得到所需的软件。也可以根据不同的应用定制相应的管理解决方案。
     </span>
     <span lang="EN-US" style="color:#0070c0">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       2.
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      统一化的基础架构：减低了设备的运行和维护成本，统一的移动设备、桌面设备和虚拟设备的管理；统一的安全和配置管理；统一的架构管理，从而减少了基础架构的复杂程度。
     </span>
     <span lang="EN-US" style="color:#0070c0">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       3.
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      简化
     </span>
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       IT
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      部门的管理任务：完善的客户端管理技术从而提高了
     </span>
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
       IT
      </span>
     </span>
     <span style="font-family:宋体; color:#0070c0">
      管理人员的时间和价值。
     </span>
     <span lang="EN-US" style="color:#0070c0">
     </span>
    </p>
    <p class="MsoNormal" style="margin:0cm 0cm 0pt">
     <span lang="EN-US" style="color:#0070c0">
      <span style="font-family:Calibri">
      </span>
     </span>
     <span style="font-family:宋体">
      上述内容比较抽象，我还是通过一系列测试实验来学习吧。同时借助于
     </span>
     <span lang="EN-US">
      <span style="font-family:Calibri">
       MVA
      </span>
     </span>
     <span style="font-family:宋体">
      征文这个平台和大家分享一下学习心得，有不当之处还请大家多多指正。
     </span>
     <span lang="EN-US" style="color:#0070c0">
     </span>
    </p>
