    <h2>
     <span style="font-size:12px">
      原理说明和快速入门：

    <h4>
     配置文件为：
    </h4>
    <p>
     <span style="font-family:'Times New Roman'">
      Spring
     </span>
     <span style="font-family:宋体">
      的配置文件默认为
     </span>
     <span style="font-family:Times New Roman">
      WEB-INF/xxxx-servelet.xm
     </span>
     <span style="font-family:宋体">
      其中
     </span>
     <span style="font-family:Times New Roman">
      xxx
     </span>
     <span style="font-family:宋体">
      为
     </span>
     <span style="font-family:Times New Roman">
      web.xml
     </span>
     <span style="font-family:宋体">
      中
     </span>
     <span style="font-family:Times New Roman">
      org.springframework.web.servlet.DispatcherServlet
     </span>
     <span style="font-family:宋体">
      的
     </span>
     <span style="font-family:Times New Roman">
      servlet-name.
     </span>
    </p>
    <h4>
     <span style="font-family:Times New Roman">
      Action
     </span>
     <span style="font-family:宋体">
      分发：
     </span>
    </h4>
    <p>
     <span style="font-family:Times New Roman">
      Spring
     </span>
     <span style="font-family:宋体">
      将按照配置文件定义的
     </span>
     <span style="font-family:Times New Roman">
      URL
     </span>
     <span style="font-family:宋体">
      ，
     </span>
     <span style="font-family:Times New Roman">
      Mapping
     </span>
     <span style="font-family:宋体">
      到具体
     </span>
     <span style="font-family:Times New Roman">
      Controller
     </span>
     <span style="font-family:宋体">
      类，再根据
     </span>
     <span style="font-family:Times New Roman">
      URL
     </span>
     <span style="font-family:宋体">
      里的
     </span>
     <span style="font-family:Times New Roman">
      action= xxx
     </span>
     <span style="font-family:宋体">
      或其他参数，利用反射调用
     </span>
     <span style="font-family:Times New Roman">
      Controller
     </span>
     <span style="font-family:宋体">
      里对应的
     </span>
     <span style="font-family:Times New Roman">
      Action
     </span>
     <span style="font-family:宋体">
      方法。
     </span>
    </p>
    <h4>
     输入数据绑定：
    </h4>
    <p>
     <span style="font-family:Times New Roman">
      Spring
     </span>
     <span style="font-family:宋体">
      提供
     </span>
     <span style="font-family:Times New Roman">
      <span style="color:#ff0000">
       Binder
      </span>
     </span>
     <span style="font-family:宋体">
      通过名字的一一对应反射绑定
     </span>
     <span style="font-family:Times New Roman">
      Pojo
     </span>
     <span style="font-family:宋体">
      ，也可以直接从
     </span>
     <span style="font-family:Times New Roman">
      request.getParameter
     </span>
     <span style="font-family:宋体">
      （）取数据。
     </span>
    </p>
    <h4>
     输入数据验证
    </h4>
    <p>
     <span style="font-family:Times New Roman">
      Sping
     </span>
     <span style="font-family:宋体">
      提供了
     </span>
     <span style="font-family:Times New Roman">
      Validator
     </span>
     <span style="font-family:宋体">
      接口当然还可以使用开源的
     </span>
     <span style="font-family:Times New Roman">
      <span style="color:#ff0000">
       Commons-Validaor
      </span>
     </span>
     <span style="font-family:宋体">
      支持最好
     </span>
    </p>
    <h4>
     <span style="font-family:Times New Roman">
      Interceptor
     </span>
     <span style="font-family:宋体">
      （拦截器）
     </span>
    </h4>
    <p>
     <span style="font-family:Times New Roman">
      Spring
     </span>
     <span style="font-family:宋体">
      的拦截器提供接口需要自己编写，在这点不如
     </span>
     <span style="font-family:Times New Roman">
      <span style="color:#ff0000">
       WebWork
      </span>
     </span>
     <span style="font-family:宋体">
      做的好。全面
     </span>
    </p>
    <p>
     （这里提一下
     <span style="font-family:Times New Roman">
      WebWork
     </span>
     <span style="font-family:宋体">
      和
     </span>
     <span style="font-family:Times New Roman">
      Struts
     </span>
     <span style="font-family:宋体">
      的区别最主要的区别在于
     </span>
     <span style="font-family:Times New Roman">
      WebWork
     </span>
     <span style="font-family:宋体">
      在建立一个
     </span>
     <span style="font-family:Times New Roman">
      Action
     </span>
     <span style="font-family:宋体">
      时是新
     </span>
     <span style="font-family:Times New Roman">
      New
     </span>
     <span style="font-family:宋体">
      一个对象而
     </span>
     <span style="font-family:Times New Roman">
      Struts
     </span>
     <span style="font-family:宋体">
      是
     </span>
     <span style="font-family:Times New Roman">
      SingleMoule
     </span>
     <span style="font-family:宋体">
      所有的都继承它的一个
     </span>
     <span style="font-family:Times New Roman">
      Action
     </span>
     <span style="font-family:宋体">
      ，所以根据项目需要合适的选择。）
     </span>
    </p>

    <h3>
     <span style="font-family:Times New Roman">
      5
     </span>
     <span style="font-family:宋体">
      ）：
     </span>
     <span style="font-family:Times New Roman">
      OSCache
     </span>
    </h3>
    <p>
     <span style="font-family:Times New Roman">
      OSCache
     </span>
     <span style="font-family:宋体">
      是
     </span>
     <span style="font-family:Times New Roman">
      OpenSymphony
     </span>
     <span style="font-family:宋体">
      组织提供的一个
     </span>
     <span style="font-family:Times New Roman">
      J2EE
     </span>
     <span style="font-family:宋体">
      架构中
     </span>
     <span style="font-family:Times New Roman">
      Web
     </span>
     <span style="font-family:宋体">
      应用层的缓存技术实现组件，
     </span>
     <span style="font-family:Times New Roman">
      Cache
     </span>
     <span style="font-family:宋体">
      是一种用于提高系统响应速度、改善系统运行性能的技术。尤其是在
     </span>
     <span style="font-family:Times New Roman">
      Web
     </span>
     <span style="font-family:宋体">
      应用中，通过缓存页面的输出结果，可以很显著的改善系统的稳定性和运行性能。
     </span>
    </p>
    <p>
     它主要用在处理短时间或一定时间内一些数据或页面不会发生变化，或将一些不变的统计报表，缓冲在内存，可以充分的减轻服务器的压力，防治负载平衡，快速重启服务器（通过硬盘缓存）。
    </p>

     </span>
     <span style="font-family:Times New Roman">

     </span>
     <span style="font-family:Times New Roman">
      Filter
     </span>
