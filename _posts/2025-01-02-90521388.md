    <h5>
     <a id="_0">
     </a>
     背景
    </h5>
    <p>
     本地环境：Ubuntu18.04
     <br/>
     远程服务器：阿里云服务器ECS
     <br/>
     已通过
     <code>
      ssh
     </code>
     在VMware中远程连接至云服务器
    </p>
    <h5>
     <a id="_4">
     </a>
     需求
    </h5>
    <p>
     从远程服务器端下载文件至本地
    </p>
    <h5>
     <a id="_6">
     </a>
     问题
    </h5>
    <p>
     使用
     <code>
      scp
     </code>
     命令时，总是报错：‘No such file or dictionary’
    </p>
    <h5>
     <a id="_8">
     </a>
     可能的解决方案
    </h5>
    <ol>
     <li>
      确保scp命令的格式正确
     </li>
    </ol>
    <blockquote>
     <p>
      scp [option] root@
      <em>
       remoteHostAddress:/path
      </em>
      <em>
       /localPath
      </em>
     </p>
    </blockquote>
    <ol start="2">
     <li>
      是在本地机的终端中使用
      <code>
       scp
      </code>
      命令，而不是远程服务器的终端(更不是通过
      <code>
       ssh
      </code>
      连接到的远程服务器的终端）
     </li>
    </ol>
