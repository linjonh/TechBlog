    <h2>
     <a id="_0">
     </a>
     背景
    </h2>
    <p>
     最近在复盘Java基础知识，看到基础数据类型这一章节，有一张图片是这样的。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e35fde1a09eb1ab9cb98ea8530684b85.png">
      <br/>
      比较好奇，为什么其他类型都有字节标识，唯独boolean没有，不是只占用
      <code>
       1bit ~ 1位
      </code>
      的吗 ？
     </img>
    </p>
    <h2>
     <a id="_6">
     </a>
     找答案
    </h2>
    <h3>
     <a id="1_Java3_7">
     </a>
     出处1 ~《疯狂Java讲义（3版）》
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d3adbb400b1d0d724bcf6bfe2c76d7e4.png">
      <br/>
      一个boolean类型占用8位 = 1字节
     </img>
    </p>
    <h3>
     <a id="2__Java_11">
     </a>
     出处2 ~ Java官网
    </h3>
    <h4>
     <a id="1__12">
     </a>
     1 原始类型
    </h4>
    <p>
     <a href="https://docs.oracle.com/javase/tutorial/java/nutsandbolts/datatypes.html" rel="nofollow">
      Primitive Data Types
     </a>
    </p>
    <blockquote>
     <p>
      boolean: The boolean data type has only two possible values: true and false. Use this data type for simple flags that track true/false conditions. This data type represents one bit of information, but its “size” isn’t something that’s precisely defined.
     </p>
    </blockquote>
    <p>
     解释一下：
    </p>
    <blockquote>
     <p>
      布尔数据类型只有两个可能的值：true和false。 将此数据类型用于跟踪真/假条件的简单标志。 此数据类型表示一位信息，但其“大小”不是精确定义的内容。
     </p>
    </blockquote>
    <h4>
     <a id="2_jdk__19">
     </a>
     2 jdk 布尔类型
    </h4>
    <p>
     地址2：
     <a href="https://docs.oracle.com/javase/specs/#22909" rel="nofollow">
      jdk specs文档
     </a>
     <br/>
     我对比了jdk8， jdk11 和 jdk12 对于boolean类型的说明是完全一样的，故只贴出jdk12的图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c2824abcc1b4339610a66a5fe49115f2.png">
      <br/>
      翻译一下：
     </img>
    </p>
    <blockquote>
     <p>
      虽然Java虚拟机定义了一个
      <code>
       boolean
      </code>
      类型，但它只为它提供了非常有限的支持。没有Java虚拟机指令专门用于对
      <code>
       boolean
      </code>
      值的操作。相反，Java编程语言中对
      <code>
       boolean
      </code>
      值进行操作的表达式被编译为使用Java虚拟机
      <code>
       int
      </code>
      数据类型的值。
     </p>
     <p>
      Java虚拟机直接支持
      <code>
       boolean
      </code>
      数组。它的newarray指令可以创建
      <code>
       boolean
      </code>
      数组。使用
      <code>
       byte
      </code>
      数组指令baload和bastore访问和修改类型为
      <code>
       boolean
      </code>
      的数组。
     </p>
     <blockquote>
      <p>
       在Oracle的Java虚拟机实现中，Java编程语言中的
       <code>
        boolean
       </code>
       数组被编码为Java虚拟机
       <code>
        byte
       </code>
       数组，每个布尔元素使用8位。
      </p>
     </blockquote>
     <p>
      Java虚拟机使用1表示
      <code>
       boolean
      </code>
      数组组件的true，0表示false。其中Java编程语言布尔值由编译器映射到Java虚拟机类型int的值，编译器必须使用相同的编码。
     </p>
    </blockquote>
    <h2>
     <a id="_32">
     </a>
     结论
    </h2>
    <ol>
     <li>
      <code>
       boolean类型
      </code>
      被编译为int类型，等于是说JVM里占用字节和int完全一样，int是4个字节，于是boolean也是
      <code>
       4字节
      </code>
     </li>
     <li>
      boolean
      <code>
       数组
      </code>
      在Oracle的JVM中，编码为byte数组，每个boolean元素占用
      <code>
       8位
      </code>
      =
      <code>
       1字节
      </code>
     </li>
     <li>
      （待考证）boolean数组在非Oracle JVM里面，占用字节数
      <code>
       也许不是1字节
      </code>
     </li>
    </ol>
