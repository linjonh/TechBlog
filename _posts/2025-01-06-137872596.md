    <h4>
     <a id="5__38">
     </a>
     5. 设置全局环境变量
    </h4>
    <blockquote>
     <p>
      <code>
       Tips:
      </code>
      如果设置了
      <strong>
       全局环境变量
      </strong>
      也没有作用可以从以下两个方面寻找问题
     </p>
     <ul>
      <li>
       <code>
        ~/.bashrc
       </code>
       文件同样配置了
       <code>
        Java
       </code>
       将
       <code>
        /etc/profile
       </code>
       路径给覆盖了
      </li>
      <li>
       没有刷新配置 （使用指令
       <code>
        source /etc/profile
       </code>
       刷新文件即可）
      </li>
     </ul>
    </blockquote>
    <p>
     拿到第四步的
     <code>
      OpenJDK
     </code>
     安装路径后，在配置文件
     <code>
      /etc/profile
     </code>
     中配置环境变量
    </p>
    <pre><code class="prism language-sh">$ <span class="token function">vim</span> /etc/profile
</code></pre>
    <p>
     输入这两句即可
    </p>
    <pre><code class="prism language-sh"><span class="token builtin class-name">export</span> <span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span>/usr/lib/jvm/java-17-openjdk-amd64
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token variable">$JAVA_HOME</span>/bin:<span class="token environment constant">$PATH</span>
</code></pre>
