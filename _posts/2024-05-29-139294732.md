    <h2>
     <a id="3_48">
     </a>
     3.变量
    </h2>
    <ul>
     <li>
      在程序运行时，能储存计算结果或能表示值的抽象概念。 即在程序运行时，记录数据用的 变量的值可以发生改变！
     </li>
    </ul>
    <h3>
     <a id="31__52">
     </a>
     3.1 定义格式
    </h3>
    <p>
     <strong>
      变量名称 = 变量的值
     </strong>
    </p>
    <ul>
     <li>
      变量名称：每一个变量都有自己的名称，称为：变量名，也就是变量本身
     </li>
     <li>
      = ：赋值，表示将等号右边的值，赋予左边的变量
     </li>
     <li>
      变量的值：每一个变量都有自己存储的值（内容），称为：变量值
     </li>
    </ul>

    <h3>
     <a id="41_type_121">

     4.1 type()语句
    </h3>
    <p>
     通过type( )语句来得到数据的类型
     <br/>
     语法：type(被查看类型的数据)
    </p>
    <ul>
     <li>
      直接使用print输出
     </li>
     <li>
      用变量存储
     </li>
    </ul>
    <p>
     <strong>
      type(变量)查看的是变量存储数据的类型
      <br/>
      变量无类型，但存储的数据由类型
     </strong>
    </p>

    <h2>
     <a id="6_161">
     </a>
     6.标识符
    </h2>
    <p>
     用户在编程时，所使用的一系列名字，用于给变量、类、方法等命名。
    </p>
    <h3>
     <a id="61__165">
     </a>
     6.1 标识符命名规则
    </h3>
    <h4>
     <a id="611__167">
     </a>
     6.1.1 内容限定
    </h4>
    <p>
     只允许使用
     <strong>
      英文、中文、数字、下划线(_)
     </strong>
     ，其余任何内容都不被允许。
     <br/>
     ！注意：
    </p>
    <ul>
     <li>
      不推荐使用中文
     </li>
     <li>
      不允许数字开头
     </li>
    </ul>
    <h4>
     <a id="612__175">
     </a>
     6.1.2 大小写敏感
    </h4>
    <h4>
     <a id="613__177">
     </a>
     6.1.3 不可使用关键字
    </h4>
    <h3>
     <a id="62__179">
     </a>
     6.2 变量命名规范
    </h3>
    <h4>
     <a id="621__181">
     </a>
     6.2.1 见名知意
    </h4>
    <p>
     明了：尽量做到，看到名字，就知道什么意思
     <br/>
     简洁：尽量在确保“明了”的前提下，减少名字的长度
    </p>
    <h4>
     <a id="622__186">
     </a>
     6.2.2 下划线命名法
    </h4>
    <p>
     多个单词组合变量名，使用下划线做分隔
    </p>
    <h4>
     <a id="623__190">
     </a>
     6.2.3 英文字母全小写
    </h4>
    <p>
     命名变量中的英文字母，应全部小写
    </p>

      </code>

      <br/>
      使用变量接收它，它就是字符串，不适用变量接收它，它就可以作为多行注释使用。

     8.4 格式化的精度控制
    </h3>
    <p>
     使用辅助符号**“m.n”**来控制数据的宽度和精度
    </p>
    <ul>
     <li>
      m ，控制宽度，要求是数字（很少使用），如设置的宽度小于数字自身，则不生效
     </li>
     <li>
      n ，控制小数点精度，要求是数字，会进行小数的四舍五入
     </li>
    </ul>
    <p>
     示例：
    </p>
    <ul>
     <li>
      %5d：将整数的宽度控制在5位 如果数字为11，则%5d为[空格][空格][空格]11
     </li>
     <li>
      %5.2f：表示将宽度控制为5，将小数点精度设置为2 如果数字为1.2，则%5.2f为[空格]1.20
     </li>
    </ul>
