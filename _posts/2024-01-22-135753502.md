    <h2>
     前言
    </h2>
    <p>
     在渗透过程中，很多时候都需要执行一些 Python 编写的工具，例如 SQL 注入中，有一款名为SQLMap 自动化注入工具，该工具就是基于 Python 环境编写的；除此之外，有的时候还需要我们自己开发一些小工具，也需要 Python 环境。
    </p>
    <p>
     Python 在 Linux 平台有两种安装模式：
    </p>
    <p>
     1、通过源码编译安装
    </p>
    <p>
     2、使用集成环境（例如 Anaconda）
    </p>
    <p>
     本文主要介绍通过源码编译安装 Python 环境（安装包版本：Python-3.9.18）
    </p>
    <h2 id="%E7%8E%AF%E5%A2%83%E8%A6%81%E6%B1%82">
     环境要求
    </h2>
    <p>
     准备一台 CentOS 7 虚拟机
    </p>
    <p>
     详细请看我上一篇文章：
     <a href="https://blog.csdn.net/zatongtong/article/details/135666908" title="CentOS 7 环境准备">
      CentOS 7 环境准备
     </a>
    </p>
    <h3 id="Python%20%E6%BA%90%E7%A0%81%E5%AE%89%E8%A3%85%E5%8C%85">
     Python 源码安装包
    </h3>
    <p>
     <a class="link-info" href="https://www.python.org/downloads/source/" rel="nofollow" title="Linux Python 软件包下载目录">
      Linux Python 软件包下载目录

      本文使用的 Python 软件包下载链接

    <h2 id="%E5%BC%80%E5%A7%8B%E5%AE%89%E8%A3%85">
     开始安装
    </h2>
    <blockquote>
     <p>
      使用 root 账户登录系统
     </p>
    </blockquote>
    <h3 id="%E5%AE%89%E8%A3%85%E4%BE%9D%E8%B5%96%E7%8E%AF%E5%A2%83">
     安装依赖环境
    </h3>
    <p>
     执行以下命令，安装 Python 所需依赖环境
    </p>
    <pre><code class="language-bash">yum -y install libffi-devel zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc openssl-devel</code></pre>
    <p>
     如果下载速度比较慢，建议更换镜像源
    </p>
    <pre><code class="language-bash">mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.py.bak
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
yum clean all &amp;&amp; yum makecache

# 直接复制执行即可</code></pre>
    <p class="img-center">
     <img alt="" height="452" src="https://i-blog.csdnimg.cn/blog_migrate/c9615c9e3d9a2db44a5ec2401450c312.png" width="632"/>
    </p>
    <h3 id="%E4%B8%8A%E4%BC%A0%2F%E4%B8%8B%E8%BD%BD%20%E5%AE%89%E8%A3%85%E5%8C%85">
     上传/下载 安装包
    </h3>
    <blockquote>
     <p>
      两种方式都可以，看网络情况，怎么方便怎么来
     </p>
    </blockquote>
    <p>
     上传：通过 FTP 工具将下载好的 Python 安装包上传到 Linux /root/ 目录下
    </p>
    <p>
     <img alt="" height="472" src="https://i-blog.csdnimg.cn/blog_migrate/60aa665968153fee5922bc2d7b87e39f.png" width="1126"/>
    </p>
    <p>
     下载：通过 wget 命令，直接在 Linux 系统内下载软件包
    </p>
    <pre><code class="language-bash">wget https://www.python.org/ftp/python/3.9.18/Python-3.9.18.tgz</code></pre>
    <p>
     <img alt="" height="322" src="https://i-blog.csdnimg.cn/blog_migrate/35e5cbfa3bbef6364356e910c7a3a4c0.png" width="1053"/>
    </p>
    <h3 id="%E5%AE%89%E8%A3%85%20Python">
     安装 Python
    </h3>
    <p>
     1、创建 Python 目录
    </p>
    <pre><code class="language-bash">mkdir -p /usr/local/python3.9

# 这里的 python3.9 是我们稍后指定的 python 安装目录
# 可以是其他的名字，只要你找得到就好</code></pre>
    <p class="img-center">
     <img alt="" height="127" src="https://i-blog.csdnimg.cn/blog_migrate/8536d9c5b35000982df5c028fe36b189.png" width="535"/>
    </p>
    <p>
     2、解压 Python 软件安装包，并进入软件包目录内
    </p>
    <pre><code class="language-bash">tar -zxvf Python-3.9.18.tgz
# 解压源代码包

# 进入代码包目录
cd Python-3.9.18
</code></pre>
    <p class="img-center">
     <img alt="" height="300" src="https://i-blog.csdnimg.cn/blog_migrate/80908edd3dd5308758d7075095955f81.png" width="604"/>
    </p>
    <p>
     3、配置安装路径
    </p>
    <pre><code class="language-bash">./configure --prefix=/usr/local/python3.9

# /usr/local/python3.9 是我们在第一步创建的 Python 目录路径</code></pre>
    <p style="text-align:center;">
     <img alt="" height="405" src="https://i-blog.csdnimg.cn/blog_migrate/30a2920790a73cafb5c542dca8a11651.png" width="623"/>
    </p>
    <p>
     4、编译安装
    </p>
    <pre><code class="language-bash">make &amp;&amp; make install
# 编译安装

# 如果这一步报错，尝试执行下方命令进行处理
yum -y install gcc</code></pre>
    <p class="img-center">
     <img alt="" height="334" src="https://i-blog.csdnimg.cn/blog_migrate/6c5e6138befa03ac6f80efe7ddab0128.png" width="636"/>
    </p>
    <p class="img-center">
     <img alt="" height="177" src="https://i-blog.csdnimg.cn/blog_migrate/f766a0ced3c7b58143e1ef96d3007bbe.png" width="633"/>
    </p>
    <p style="text-align:center;">
     出现该图不是报错，它提示我们要添加环境变量，我们后面会提到如何添加环境变量
    </p>
    <p>
     5、验证是否安装成功，出现版本号即可
    </p>
    <pre><code class="language-bash">/usr/local/python3.9/bin/python3 -V
# 验证 Python 是否正确安装
# /usr/local/python3.9/ 是我们在第三步设置的路径

/usr/local/python3.9/bin/pip3 -V
# 验证 pip 是否正确安装
# pip 是 python 中管理第三方库的模块
# /usr/local/python3.9/ 是我们在第三步设置的路径
</code></pre>
    <p class="img-center">
     <img alt="" height="115" src="https://i-blog.csdnimg.cn/blog_migrate/aec2cc621c93a4426dfdfaa5479e00fd.png" width="616"/>
    </p>
    <h2 id="Python%20%E7%8E%AF%E5%A2%83%E9%85%8D%E7%BD%AE">
     Python 环境配置
    </h2>
    <h3 id="%E9%85%8D%E7%BD%AE%E7%8E%AF%E5%A2%83%E5%8F%98%E9%87%8F">
     配置环境变量
    </h3>
    <p>
     修改配置文件，添加环境变量
    </p>
    <pre><code class="language-bash">vi /etc/profile

# 在文件末尾处添加

export PATH=/usr/local/python3.9/bin:$PATH

# 其中 /usr/local/python3.9/ 是 Python 的安装目录
# 更改完后保存并退出</code></pre>
    <p class="img-center">
     <img alt="" height="151" src="https://i-blog.csdnimg.cn/blog_migrate/1dec5a6d63a86310faf89d6148d0fb0e.png" width="492"/>
    </p>
    <pre><code class="language-bash">source /etc/profile
# 使环境变量立刻生效

echo $PATH
# 查看环境变量</code></pre>
    <p class="img-center">
     <img alt="" height="115" src="https://i-blog.csdnimg.cn/blog_migrate/6f2be6574f4312057d113b88f6495f37.png" width="598"/>
    </p>
    <p>
     此时我们输入 python3 -V 和 pip3 -V 发现已经可以调用 Python 环境了，输入 python3 也能进入 python 交互界面
    </p>
    <p class="img-center">
     <img alt="" height="216" src="https://i-blog.csdnimg.cn/blog_migrate/4e2a8422da40ea30e915f0db28b648f3.png" width="639"/>
    </p>
    <h3 id="%E5%88%9B%E5%BB%BA%E8%BD%AF%E9%93%BE%E6%8E%A5">
     创建软链接
    </h3>
    <blockquote>
     <p>
      为了避免环境变量不生效，我们可以通过创建软链接的方式来调用 Python 环境
     </p>
    </blockquote>
    <pre><code class="language-bash">ln -s /usr/local/python3.9/bin/python3 /usr/bin/python3
# 创建 Python3 软链接

ln -s /usr/local/python3.9/bin/pip3 /usr/bin/pip3
# 创建 pip 软链接

# /usr/bin 目录主要存放的是 应用程序和工具，我们在这里创建一个软链接，就能调用了
# 软链接：相当于 Windows 操作系统中的快捷方式</code></pre>
    <p class="img-center">
     <img alt="" height="152" src="https://i-blog.csdnimg.cn/blog_migrate/f0ef7865c030b5c082e402203303a508.png" width="624"/>
    </p>
    <p>
     到这里，Python 环境就部署完成了
    </p>
    <h2 id="Python%20%E9%97%AE%E9%A2%98%E5%A4%84%E7%90%86">
     Python 问题处理
    </h2>
    <h3 id="pip%20%E4%B8%8B%E8%BD%BD%E7%AC%AC%E4%B8%89%E6%96%B9%E5%BA%93%E9%80%9F%E5%BA%A6%E8%BF%87%E6%85%A2">
     pip 下载第三方库速度过慢
    </h3>
    <blockquote>
     <p>
      Python 默认第三方库下载地址是国外地址，如果没有挂代理，下载速度会很慢，我们可以更换为国内地址，这样下载速度会明显提高
     </p>
    </blockquote>
    <p>
     创建 pip 第三方库下载目录
    </p>
    <pre><code class="language-bash">mkdir ~/.pip
# 创建 pip 地址目录

touch ~/.pip/pip.conf
# 创建配置文件</code></pre>
    <p class="img-center">
     <img alt="" height="82" src="https://i-blog.csdnimg.cn/blog_migrate/8a921cace3a6684c4a44d1bf3e9297f2.png" width="453"/>
    </p>
    <p>
     添加内容
    </p>
    <pre><code class="language-bash">[global]
trusted-host =  mirrors.aliyun.com
index-url = http://mirrors.aliyun.com/pypi/simple
# 更换为 阿里云 源


# 下面还有其他源可供选择，照葫芦画瓢替换 url 地址即可
# 阿里云：http://mirrors.aliyun.com/pypi/simple
# 清华：http://pypi.tuna.tsinghua.edu.cn/simple
# 中国科技大学 http://pypi.mirrors.ustc.edu.cn/simple/
# 华中理工大学：http://pypi.hustunique.com/
# 山东理工大学：http://pypi.sdutlinux.org/
# 豆瓣：http://pypi.douban.com/simple/</code></pre>
    <p>
     验证是否更换成功
    </p>
    <pre><code class="language-bash">pip3 config list
# 查看 pip 源</code></pre>
    <p class="img-center">
     <img alt="" height="147" src="https://i-blog.csdnimg.cn/blog_migrate/c329c16284b7af232bef2fe6a6ae1ec8.png" width="537"/>
    </p>
    <p>
     这样 pip 源就更换完成了，重新下载速度会块很多
    </p>
    <h3 id="pip%20%E7%BC%BA%E5%A4%B1">
     pip 缺失
    </h3>
    <blockquote>
     <p>
      安装完成后，发现 pip 无法使用，配置系统变量，软链接后还是不行
     </p>
     <p>
      甚至有找不到 pip 的情况出现
     </p>
    </blockquote>
    <p>
     解决方法：我们可以通过 Python 脚本进行 pip 安装
    </p>
    <pre><code class="language-bash">wget https://bootstrap.pypa.io/pip/get-pip.py
# 下载 pip 安装 py 脚本

python3 get-pip.py
# 安装 pip
# 如果安装速度过慢的话，可参考上一步，换个源试试

pip3 -V
# 验证</code></pre>

     <img alt="" height="341" src="https://i-blog.csdnimg.cn/blog_migrate/a8bda9d21d89edaaea2f55ddbd75907e.png" width="637"/>
    </p>
