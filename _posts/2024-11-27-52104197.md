    <p>
     <span style="font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      ECC的全称是Error Checking and Correction，是一种用于
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      Nand
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      的差错检测和修正算法。
     </span>
     <span style="font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      如果操作时序和电路稳定性不存在问题的话，
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      NAND Flash
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      出错的时候一般不会造成整个
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      Block
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      或是
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      Page
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      不能读取或是全部出错，而是整个
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      Page
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      （例如
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      512Bytes
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      ）中只有一个或几个
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      bit
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      出错。
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      ECC
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      能纠正
     </span>
     <span style="font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      1
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      个
     </span>
     <span style="font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      比特错误和检测2
     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      个
     </span>
     <span style="font-family:Verdana,Arial,Helvetica,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">
      比特错误，而且计算速度很快，但对
     </span>
     <span style="font-family:Times New Roman; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">

     </span>

     </span>
     <span style="font-family:宋体; margin:0px; padding:0px; font-size:13px; line-height:19.5px; background-color:rgb(254,254,242)">

     </span>
