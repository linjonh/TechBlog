    <h2>
     <a id="_1">
     </a>
     你要结婚了,女朋友分别对你说了这几句话,你好好捋一捋
    </h2>
    <h3>
     <a id="1__3">
     </a>
     1 如果你给我买钻戒,那么我嫁给你
    </h3>
    <p>
     一口价类型女友,买了就嫁给你
     <br/>
     p:买钻戒 q:嫁给你 p-&gt;q
    </p>
    <p>
     你可以再加一句同类型的话感受一下(这个方法对比下文)
     <br/>
     如果给我买房,那么我嫁给你
     <br/>
     你会觉得小婊砸,你是能分身还是怎么滴,
     <br/>
     明显感觉逻辑不通.
     <br/>
     说明 一个钻戒就可以嫁给你
     <br/>
     p-&gt;q 这个推理没毛病
    </p>
    <h4>
     <a id="__14">
     </a>
     补一下语文: 只有
    </h4>
    <p>
     1、连词，表示
     <strong>
      必需的条件
     </strong>
     ，下文常用“才”、“方”呼应。
     <br/>
     <strong>
      只有依靠群众，才能做好革命工作
     </strong>
     <br/>
     做好了工作肯定依靠了群众,
     <br/>
     依靠了群众,没有子弹 可能还是做不好工作
    </p>
    <h3>
     <a id="2__19">
     </a>
     2 只有你给我买钻戒,我才嫁给你
    </h3>
    <p>
     套路型,你很难成功
     <br/>
     p:买钻戒 q:嫁给你 q-&gt;p
    </p>
    <p>
     只有P才Q，说明只有P成立时，Q才能成立，注意是能成立不是一定成立
     <br/>
     <strong>
      说明P是Q成立的一个条件
     </strong>
     ，
     <br/>
     Q成立还可能有其他条件，
     <br/>
     比如说女方再加一句同类型的话试试
     <br/>
     <strong>
      只有你给我买房,我才嫁给你
     </strong>
     ,
     <br/>
     这个时候你会明显觉得好像他这么说没毛病,套路深啊,
     <br/>
     她也没说买了就嫁给你啊,都说了才,我才18,暗示自己很年轻.我才嫁给你,暗示你力度不够
     <br/>
     所以小伙子只买了钻戒是不能推理出她会嫁给你滴.
     <br/>
     但是只要Q成立就说明P一定成立，
     <br/>
     嫁给你了你肯定说明你买了钻戒买了房,不然怎能空手套白狼
     <br/>
     所以Q推出P.P是Q成立的必要条件。
    </p>
    <h4>
     <a id="__35">
     </a>
     补一下语文: 除非、才
    </h4>
    <p>
     除非
     <br/>
     1.连词。表示唯一的条件，
     <strong>
      相当于“只有
     </strong>
     ”，常与“才”“否则”“不然”等词呼应：～参加变革现实的实践，不然你是不能获得真知的。
     <br/>
     ps：我觉得这里解释有点矛盾：只有是指必需的条件，不是唯一的条件，把除非理解成 只有 更方便推理
    </p>
    <p>
     才
     <br/>
     6.表示只有在某种条件下然后怎样（前面常常用“只有、必须”或含有这类意思）：只有依靠群众，～能把工作做好。
    </p>
    <h4>
     <a id="__43">
     </a>
     补一下语文： 否则
    </h4>
    <p>
     否，不，代表一个假设性小句，相当于“
     <strong>
      如果不这样
     </strong>
     ”。 “则”，就，引出后果或结论，相当于“那么”。
     <br/>
     连词，是如果不这样的意思。
     <br/>
     《实践论》：“理性的东西所以靠得住，正是由于它来源于感性，否则理性的东西就成了无源之水，无本之木。”
    </p>
    <p>
     除非你给我100，否则我不 干这事
     <br/>
     同义为：
     <br/>
     （只有）你给我100，不这样我不 干这事
     <br/>
     就是
     <br/>
     不给100块就不干这个事
     <br/>
     我干这事了,说明你给我了一百块
    </p>
    <h3>
     <a id="3__56">
     </a>
     3 除非你给我买钻戒，否则我不嫁你
    </h3>
    <p>
     影视剧型女友，好好把握
     <br/>
     （只有）你给我买钻戒,(如果)不这样我不 嫁给你。
     <br/>
     A: 买钻戒 B:嫁给你
     <br/>
     否则我不 是双重否定 表肯定
     <br/>
     比如： 要不是你给了我钱，否则我不这么干。说明 我还是这么干了的
     <br/>
     所以 否则我不B 可以符号化为 ~ （~B) = B
     <br/>
     <strong>
      除非A，否则不B ：
      <sub>
       (
      </sub>
      B) -&gt;A 即 B -&gt; A
     </strong>
     。
     <br/>
     大白话就是,不买钻戒不嫁给你,嫁给你的条件是买钻戒
     <br/>
     也可同义为：
     <br/>
     只有你给我买钻戒，我才嫁你
     <br/>
     但是有可能 你买了钻戒还是不嫁给你
     <br/>
     所以只有嫁给你才能推出买钻戒, 光买了钻戒推不出嫁给你了
    </p>
    <p>
     再来个例子感受下
     <br/>
     除非P，否则不Q。
     <br/>
     <strong>
      “除非你给我五十万，否则我不会把女儿嫁给你”。
     </strong>
     <br/>
     就是“女儿嫁给你”可以，先给我五十万，否则免谈
     <br/>
     所以嫁给你了，可以推出来是“给了五十万聘礼”。
    </p>
    <p>
     理解除非p，否则不q，最容易出现的问题就是，这个最关键的“不”字，切记“不”字属于逻辑关联词的一部分，它不代表着q结论的否定，而是把 否则我不 看成整体表示肯定，也就是说，“除非P，否则不Q”和“除非P，否则Q”根本就不是代表着同一个逻辑意思。
    </p>
    <h3>
     <a id="4__79">
     </a>
     4 除非你给我买钻戒，否则我嫁你
    </h3>
    <p>
     白给型女友,恭喜你
     <br/>
     因为这么说话不好直接理解，所以先看一个常见的 除非A否则B 语句使用的场景
    </p>
    <h4>
     <a id="_AB__AB_82">
     </a>
     用报名理解 除非A,否则B 和 A，除非B
    </h4>
    <p>
     除非A,否则B,就是一般情况就是B情况，否则A发生了 ：~A-&gt;B, ~B-&gt;A
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/23b4ae8a375e1cc1bfac07ff14ca7ea5.png"/>
    </p>
    <p>
     再来看下这个例子：
     <br/>
     <strong>
      除非你买钻戒，否则我嫁你
     </strong>
     <br/>
     暗示嫁给你是常态，买了戒指就不一样了
     <br/>
     一般情况就是我嫁给你的，除非你给我买了戒指，我不嫁给你了
     <br/>
     就是 只要不是你给我买戒指了，我就嫁给你
     <br/>
     所以这女的是有多讨厌戒指啊！我就喜欢这样的！
    </p>
    <h4>
     <a id="_93">
     </a>
     用语义替换理解
    </h4>
    <p>
     <strong>
      把除非换成只有 ，否则 换成 如果不这样
     </strong>
     ，语义还是不变的。
     <br/>
     <strong>
      只有你给我买钻戒，如果不这样 我嫁给你
     </strong>
     <br/>
     <strong>
      这样
     </strong>
     指的是
     <strong>
      买钻戒
     </strong>
     <br/>
     <strong>
      只有你给我买钻戒，如果不买钻戒 我嫁给你
     </strong>
     <br/>
     就是说 买了钻戒，不嫁给你
     <br/>
     所以没嫁给你可以推出了你买了钻戒
     <br/>
     遇到这样的女朋友的好好珍惜人家吧
    </p>
    <p>
     除非A，否则B 关系：~B-&gt;A, ~A-&gt;B
     <br/>
     A:买钻戒
     <br/>
     B:嫁给你
     <br/>
     ~A-&gt;B:没买钻戒-&gt;你嫁给我
     <br/>
     ~B-&gt;A:你没嫁给我-&gt;买了钻戒
    </p>
    <p>
     转自:https://zhidao.baidu.com/question/362989416.html
    </p>
    <h3>
     <a id="5__114">
     </a>
     5 例题
    </h3>
    <p>
     【例】除非不把受到的巨大打击当回事，否则就会有心理疾病。
     <br/>
     以下各项都表达了与题干相同的含义，除了
     <br/>
     A.如果不把受到的巨大打击当回事，就不会有心理疾病。
     <br/>
     B.如果把受到的巨大打击当回事，就会有心理疾病。
     <br/>
     C.只有有心理疾病，才会把受到的巨大打击当回事。
     <br/>
     D.只有不把受到的巨大打击当回事，才不会有心理疾病。
     <br/>
     E.除非受到的巨大打击当回事，否则不会有心理疾病。
    </p>
    <p>
     【正确答案】A
    </p>
    <p>
     【中公解析】
     <br/>
     题干翻译：除非 不把受到的巨大打击当回事，否则 就会有心理疾病。
     <br/>
     p:不把打击当回事
     <br/>
     q:有疾病
     <br/>
     <br/>
     除非 p，否则 q 符号化为： ~p-&gt;q 和 ~q-&gt;p
     <br/>
     ~p-&gt;q：把打击当回事→有疾病
     <br/>
     ~q-&gt;p：没有疾病→不把打击当回事
    </p>
    <p>
     A翻译：不把受到的巨大打击当回事→不会有心理疾病。 p→~q
     <br/>
     B翻译：受到的巨大打击当回事→有心理疾病。 ~p→q
     <br/>
     C翻译：.只有有心理疾病，才会把受到的巨大打击当回事。 符号化为 只有q，才~p
     <br/>
     所以表示的逻辑为 ~p→q ：把受到的巨大打击当回事→有心理疾病
     <br/>
     D翻译：.只有不把受到的巨大打击当回事，才不会有心理疾病 C的逆否命题与C同义
     <br/>
     E翻译：除非受到的巨大打击当回事，否则不会有心理疾病。符号化为 除非~p，否则不q
     <br/>
     q→~p 有疾病-&gt;当回事 这个容易误解 常理是对的，但是推理是错的，好好理解一下。
     <br/>
     或者用 除非～p，否则～q 逻辑框架结果是 ～（～q）-&gt;（～p) 就是 q-&gt;～p 结果是一样的，和题干推理不一样。题干是当回事推出有病，E是有病推出当回事，反过来了
     <br/>
     所以表示的逻辑为：有心理疾病-&gt;当回事
     <br/>
     <br/>
     总结：只要选项符号化的逻辑与 ~p-&gt;q 和 ~q-&gt;p相同就是正确选项，明显A是错的 E也错的
    </p>

     补充 离散数学 命题符号化

    <h3>
     <a id="1__151">
     </a>
     1 常见命题符号化
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6f6eec829b06ad921f52bb45071b3388.png">
      <br/>
      x仅当y，y是x的必要非充分条件；x则(–&gt;)y表明x是y的充分非必要条件。
     </img>
    </p>
    <h3>
     <a id="2__155">
     </a>
     2 易混淆命题符号化总结
    </h3>
    <p>
     <em>
      <strong>
       例子：除非李联不怕吃苦，否则他不会取得这样的好成绩
      </strong>
     </em>
    </p>
    <h4>
     <a id="_AB________A__B_158">
     </a>
     法一：除非 A，否则B ~A -&gt; B
    </h4>
    <p>
     ~A-&gt;B 等价于 ~B-&gt;A 逆否命题
     <br/>
     A：李联不怕吃苦
     <br/>
     ~A：李联怕吃苦
     <br/>
     B：他不会取得好成绩
     <br/>
     ~B：她取得好成绩
     <br/>
     <strong>
      ~A -&gt; B：李联怕吃苦，所以她不会取得好成绩
     </strong>
     <br/>
     <strong>
      ~B -&gt; A：李联取得好成绩，所以李联不怕吃苦
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17bbbe0591d7f2544992955c6fde6e08.png"/>
    </p>
    <h4>
     <a id="p_q____qp_168">
     </a>
     法二：除非p，否则不 q q-&gt;p
    </h4>
    <p>
     除非李联不怕吃苦，否则他不会取得这样的好成绩
     <br/>
     p：李联不怕吃苦
     <br/>
     q：李联取得好成绩
     <br/>
     <strong>
      李联取得好成绩-&gt;李联不怕吃苦
     </strong>
    </p>
    <h4>
     <a id="3___174">
     </a>
     3 成功学例子 加强巩固
    </h4>
    <p>
     除非你努力了，否则不能成功 除非p，否则不q
     <br/>
     结论：q-&gt;p ~p-&gt; ~q
     <br/>
     p:努力了
     <br/>
     q：能成功
     <br/>
     q-&gt;p: 能成功说明你努力了
     <br/>
     ~p-&gt; ~q：不努力肯定不成功
     <br/>
     错误推理：努力了就成功了p-&gt;q
    </p>
    <p>
     除非你努力了，否则不能成功 除非A，否则B
     <br/>
     结论：~A-&gt;B ~B-&gt;A
     <br/>
     A：努力了
     <br/>
     B：不能成功
     <br/>
     ~A-&gt;B: 不努力一定不成功
     <br/>
     ~B-&gt;A:成功了说明努力了
     <br/>
     错误推理：努力了就成功了 A-&gt;~B
    </p>
    <p>
     推理也符合我们得常识：你努力了不一定成功,你不努力肯定不成功，你成功了一定努力了
    </p>
    <p>
     两种方法最终取得的结论是一样的，所以大家按照各自的习惯记忆不要搞混了,记住一种逻辑即可。
     <br/>
     平常用 除非p否则不q,比较多 就是q-&gt;p
     <br/>
     p是啥样的就直接写下来，不要被否定词迷惑了，例子的p就是一个否定结论
    </p>
    <p>
     复习一下：记住一个就行了
     <br/>
     除非你给我买戒指，否则我不跟你结婚
     <br/>
     跟你结婚-&gt;你买戒指
    </p>
    <p>
     除非A来了，否则B去参加
     <br/>
     B去参加-&gt;A没来
    </p>
    <h4>
     <a id="BA_205">
     </a>
     只有B，才A
    </h4>
    <p>
     只有B,才A 只有坚持下去,才能成功 成功-&gt;坚持 ，
     <br/>
     很多时候不是因为坚持了就能成功，但是坚持了才有成功的希望！大家加油
     <br/>
     就算没成功坚持下去也很有意义。
