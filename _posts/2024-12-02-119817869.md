    <p>
     如何从网页上下载内嵌的PDF文件？
     <br/>
     最近参加了百度的深度学习训练营，但是发现百度官方不提供其网页内嵌PDF文件的下载，所以我就给大家提供一个办法，用来下载这类网页内嵌的PDF文件。
     <br/>
     目录
     <br/>
     前言
     <br/>
     开始
    </p>
    <ol>
     <li>
      打开开发者工具
     </li>
     <li>
      选中Network栏目后再选择XHR
     </li>
     <li>
      Ctrl+R（刷新）
     </li>
     <li>
      在新标签页中打开该文件即可获取下载链接
      <br/>
      ￼
     </li>
    </ol>
    <p>
     开始
    </p>
    <ol>
     <li>
      打开开发者工具
      <br/>
      我以谷歌浏览器为例，其他浏览器也是一样操作。
     </li>
    </ol>
    <p>
     下图中就有一个内嵌的《卷积神经网络》PDF文件 👇

     ￼

     <li>
      选中Network栏目后再选择XHR
      <br/>
      此时可以看到XHR中是没有任何东西的。
      <br/>
