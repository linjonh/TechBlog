      <h3>
       1. fortune-sheet
      </h3>
      <p style="text-align:center;">
       <img alt="eb9847ce5bf4dc091f34b30e99089ae0.gif" src="https://i-blog.csdnimg.cn/blog_migrate/c944f04a798cf4983604abae16b6d92d.gif"/>
      </p>
      <p>
       <code>
        FortuneSheet
       </code>
       的目标是制造一个功能丰富，配置简单的在线表格组件，开箱即用。项目源于
       <code>
        Luckysheet
       </code>
       ，并继承了它的很多代码。作者为将其转换为
       <code>
        typescript
       </code>
       做了很多努力，并且解决了一些原项目设计层面的问题。
      </p>
      <p>
       但是我个人在研究和使用它的时候还是发现了很多问题，比如在
       <code>
        next
       </code>
       项目中无法更新和初始化数据，同时对图片的支持也不是特别友好，希望作者看到之后能正视这些问题。
      </p>
      <p>
       基础使用方式如下：
      </p>
      <pre class="has"><code class="language-go">import React from 'react';
import ReactDOM from 'react-dom';
import { Workbook } from "@fortune-sheet/react";
import "@fortune-sheet/react/dist/index.css"

ReactDOM.render(
  &lt;Workbook data={[{ name: "Sheet1" }]} /&gt;,
  document.getElementById('root')
);</code></pre>
<p>
github 地址：
<code>
https://github.com/ruilisi/fortune-sheet
</code>
</p>
<h3> 2. x-spreadsheet
</h3>
<p style="text-align:center;">
<img alt="102b002044afa49e2bbb95b390886061.gif" src="https://i-blog.csdnimg.cn/blog_migrate/d3c9108059a0398d2cb55e74e6ab92e4.gif"/>
</p>
<p>
<code>
x-spreadsheet
</code>
是一个基于 Web(es6) canvas 构建的轻量级 Excel 开发库, 我们可以使用原生 js 的方式在项目中引用它，也就意味着它可以在不同的前端框架中使用，比如
<code>
vue
</code>
，
<code>
react
</code>
，
<code>
angular
</code>
等。
</p>
<p>
基础使用方式如下：
</p>
<pre class="has"><code class="language-go">import Spreadsheet from "x-data-spreadsheet";
const s = new Spreadsheet("#x-spreadsheet-demo")
  .loadData({}) // load data
  .change(data =&gt; {
    // save data to db
  });

// data validation
s.validate()</code></pre>
<p>
github 地址：
<code>
https://github.com/myliang/x-spreadsheet
</code>
</p>
<h3> 3. univer
</h3>
<img alt="d6b9fb8485c2d9982144ee2a012e0b2f.png" src="https://i-blog.csdnimg.cn/blog_migrate/79c44ee34d496853955b93835863f398.png">
<p>
<code>
Univer
</code>
对
<code>
Luckysheet
</code>
底层进行了大量重构，支持非常多的功能，包括但不限于公式计算、条件格式、数据验证、筛选、协同编辑、打印、导入导出等等。
</p>
<p>
它有商业版本和开源版本，我也使用了一下它的开源版本，但是在
<code>
Nextjs
</code>
最新版本中仍然会报错，同时文档上希望能有更详细的 API 说明，如果作者看到了希望能重视这个问题哈，还是比较看好这个项目。
</p>
<p>
接下来附上一个在
<code>
vite
</code>
中使用的代码案例：
</p>
<pre class="has"><code class="language-go">import "./style.css";
import "@univerjs/design/lib/index.css";
import "@univerjs/ui/lib/index.css";
import "@univerjs/sheets-ui/lib/index.css";
import "@univerjs/sheets-formula/lib/index.css";

import { Univer, UniverInstanceType } from "@univerjs/core";
import { defaultTheme } from "@univerjs/design";
import { UniverDocsPlugin } from "@univerjs/docs";
import { UniverDocsUIPlugin } from "@univerjs/docs-ui";
import { UniverFormulaEnginePlugin } from "@univerjs/engine-formula";
import { UniverRenderEnginePlugin } from "@univerjs/engine-render";
import { UniverSheetsPlugin } from "@univerjs/sheets";
import { UniverSheetsFormulaPlugin } from "@univerjs/sheets-formula";
import { UniverSheetsUIPlugin } from "@univerjs/sheets-ui";
import { UniverUIPlugin } from "@univerjs/ui";

const univer = new Univer({
  theme: defaultTheme,
});

univer.registerPlugin(UniverRenderEnginePlugin);
univer.registerPlugin(UniverFormulaEnginePlugin);

univer.registerPlugin(UniverUIPlugin, {
  container: 'app',
  header: true,
  footer: true,
});

univer.registerPlugin(UniverDocsPlugin, {
  hasScroll: false,
});
univer.registerPlugin(UniverDocsUIPlugin);

univer.registerPlugin(UniverSheetsPlugin);
univer.registerPlugin(UniverSheetsUIPlugin);
univer.registerPlugin(UniverSheetsFormulaPlugin);

// create univer sheet instance
univer.createUnit(UniverInstanceType.UNIVER_SHEET, {});</code></pre>
<p>
github 地址：
<code>
https://github.com/dream-num/univer
</code>
</p>
<h3> 4. handsontable
</h3>
<img alt="b9b4a79bd27407dc4651040ade3d9e05.png" src="https://i-blog.csdnimg.cn/blog_migrate/78b84be119f71b03ce94a9e309cbea92.png">
<p>
<code>
handsontable
</code>
是一款完全开源的在线电子表格组件，他提供了详细的文档和丰富的 API 接口来保证我们能实现专业级电子表格：
</p>
<img alt="0370d10efcf886a8933c5bc5ff7f8471.png" src="https://i-blog.csdnimg.cn/blog_migrate/ed0f7dcf5ebb63374c1f26ccf94d3a60.png">
<p>
它同时支持多种前端框架，比如 vue2， vue3， react 等，非常适合有技术余力的团队经行二次开发。
</p>
<p>
一个简单的使用案例：
</p>
<pre class="has"><code class="language-go">import { HotTable } from '@handsontable/react';
import { registerAllModules } from 'handsontable/registry';
import 'handsontable/dist/handsontable.full.min.css';
registerAllModules();

// generate an array of arrays with dummy data
const data = new Array(100) // number of rows
  .fill()
  .map((_, row) =&gt; new Array(50) // number of columns
    .fill()
    .map((_, column) =&gt; `${row}, ${column}`)
  );

const ExampleComponent = () =&gt; {
  return (
    &lt;HotTable
      data={data}
      rowHeaders={true}
      colHeaders={true}
      contextMenu={true}
      mergeCells={[
        { row: 1, col: 1, rowspan: 3, colspan: 3 },
        { row: 3, col: 4, rowspan: 2, colspan: 2 }
      ]}
      autoWrapRow={true}
      autoWrapCol={true}
      licenseKey="non-commercial-and-evaluation"
    /&gt;
  );
};

export default ExampleComponent;</code></pre>
<p>
后面我也考虑基于它来实现一款类似
<code>
Excel
</code>
的在线电子表格，实现更强大的功能，并集成到我最近上线的
<code>
Nocode/WEP
</code>
项目中。
</p>
<p>
github 地址：
<code>
https://github.com/handsontable/handsontable
</code>
</p>
