     <p>
      小程序上线后，改版了很多次，包括一些 Api 接口也有改动。如果你学习一个很久之前的小程序项目是没有意义的，本文推荐的小程序都是最近有更新的。相信在你学习、部署的过程中，不会遇到很多问题。
     </p>
     <p>
      本文推荐的项目从入门到进阶都有，收藏起来吧。
      <br/>
     </p>
     <h2>
      <em>
       <strong>
        01.
       </strong>
      </em>
      <strong>
       WordPress 版微信小程序
      </strong>
     </h2>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/ccb39ee06839bbacc2128a0d0b13b9c9.png"/>
     </p>
     <h2>
      这个小程序功能几乎包含了「内容」产品涉及的功能，比如：
     </h2>
     <ul>
      <li>
       <h2>
        文章列表（包括显示文章分类、缩略图、发布时间，支持分页）
       </h2>
      </li>
      <li>
       <h2>
        轮播图、文章分类、文章内容页（包括文章站内链接跳转，站外链接复制到剪切板）
       </h2>
      </li>
      <li>
       <h2>
        猜你喜欢相关文章、文章评论（提交评论和回复评论，加载评论分页，显示微信用户评论者的头像）
       </h2>
      </li>
      <li>
       <h2>
        显示文章排行（浏览数、评论数、点赞数、赞赏数）、文章内容搜索
       </h2>
      </li>
      <li>
       <h2>
        通过微信支付对文章赞赏
       </h2>
      </li>
      <li>
       <h2>
        分享到朋友圈、小程序直播、支持微信搜一搜页面接入和内容搜索等等
       </h2>
      </li>
     </ul>
     <p>
      <em>
       <strong>
        02.
       </strong>
      </em>
      <strong>
       小程序书店
      </strong>
     </p>
     <p>
      技术栈：Taro + Taro UI + Redux + Webpack + ES6 + Mock。这个项目可以作为你第一个小程序练手项目。
      <br/>
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/442d68c4e3abf0957e5bd37ed8af7383.png"/>
     </p>
     <p>
      这个小程序非常简单，可以作为初学者第一个实战项目。帮助新手理解 Taro 与 Redux 的配合方式与 Taro 的基本使用。本项目还提供了一个快速搭建本地 mock 服务的解决方案。
     </p>
     <p>
      除此之外，这个项目还提供了学习文档：
      <br/>
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/042a26bfb69bf68b797d97c4306ab00b.png"/>
     </p>
     <p>
      <em>
       <strong>
        03.
       </strong>
      </em>
      <strong>
       高仿喜马拉雅
      </strong>
     </p>
     <p>
      如果你仅仅有 HTML CSS Js 的知识储备，想开发一个属于自己的小程序，这个 Demo 再适合你不过了，这个高仿喜马拉雅是一个初级项目完全使用微信小程序原生开发，没有使用自定义组件，非常的适合微信小程序开发新手。
     </p>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/f410ad258778068da384255c4d648704.png"/>
     </p>
     <p>
      <em>
       <strong>
        04.
       </strong>
      </em>
      <strong>
       网易云音乐小程序
      </strong>
     </p>
     <p>
      基于Taro与网易云音乐 api 开发，技术栈主要是：typescript+taro+taro-ui+redux+react-hooks。
     </p>
     <p>
      目前主要是着重小程序端的展示，主要也是借此项目强化下上述几个技术栈的使用，打造一个最佳实践项目，通过这个项目也可以帮助你快速使用 Taro 开发一个属于你自己的小程序。
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/b182cc1aa24bb294b0c783caf34f8296.png"/>
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/69fe2eb9e112826dada16bcdf0085651.png"/>
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/2ee0652ac5ecc3797f119c4e4d61c647.jpeg"/>
     </p>
     <p>
      这是该小程序实现的一些功能：
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/0df811773006742bc3b004dcd4e5726a.png"/>
     </p>
     <h2>
      <em>
       <strong>
        05.
       </strong>
      </em>
      <strong>
       小程序 GitHub Trending Hub
      </strong>
     </h2>
     <p>
      小程序 GitHub Trending Hub 是一个以 Feed 流形式查看 GitHub Trending 仓库集合的工具，通过它可以及时查看最近更新的热门仓库。
     </p>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/d9f093682599208335273533f9fb4106.png"/>
     </p>
     <h2>
      <em>
       <strong>
        06.
       </strong>
      </em>
      <strong>
       微信小程序商城
      </strong>
     </h2>
     <p>
      刚刚提及的项目适合新手入门，这个小程序适合进阶选手。这个程序涉及很多组件的使用比如：
      <br/>
     </p>
     <ul>
      <li>
       <p>
        接口 SDK
       </p>
      </li>
      <li>
       <p>
        api工厂
       </p>
      </li>
      <li>
       <p>
        vant-weapp
       </p>
      </li>
      <li>
       <p>
        小程序富文本插件（html 渲染）
       </p>
      </li>
      <li>
       <p>
        小程序海报组件-生成朋友圈分享海报并生成图片
       </p>
      </li>
      <li>
       <p>
        Apache ECharts
       </p>
      </li>
     </ul>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/f3c2ff8de6a4c2eda7024584aa6e0c90.png"/>
     </p>
     <p>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/e430c045cf00fb397d130ce876ee3384.png"/>
     </p>
     <p>
      <em>
       <strong>
        07.
       </strong>
      </em>
      <strong>
       情书站点
      </strong>
     </p>
     <p>
      第一个项目是校园小情书的微信小程序，该项目功能包括表白墙、树洞、校园论坛。
     </p>
     <p style="text-align: center">
      <img src="https://i-blog.csdnimg.cn/blog_migrate/65aaa9d13d0f7af33fb70b590c7f796a.png"/>
