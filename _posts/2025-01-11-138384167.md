    <p>
     微软收购ChatGPT，并将其用到了Bing搜索。而创业公司在搜索领域，如何利用AI并商业化变现？
    </p>
    <p>
     本期分享36个AI搜索(主要是国外)的网站。
    </p>
    <p>
     <strong>
      0、AI智能导航
     </strong>
     <br/>
     网址：https://codernav.com/ainav
     <br/>
     这是一个AI智能工具导航网站，网站收录了非常多的这类工具，不仅如此，网站还收录了众多AI问答、AI绘画、AI办公、AI论文等工具，大部分工具都提供了免费策略，非常值得推荐与收藏。
    </p>
    <p>
     <strong>
      1、Microsoft Bing
     </strong>
     <br/>
     网址：https://bing.com/new
     <br/>
     微软的New Bing将ChatGPT与搜索合成在一起。新的必应就像在你搜索时有一个研究助理、个人规划师和创意伙伴在身边。
    </p>
    <p>
     PS：中国大陆的用户打不开，会默认跳转bing，非New bing。
    </p>
    <p>
     <strong>
      2、OSS Insight
     </strong>
     <br/>
     网址：https://ossinsight.io/explore/
     <br/>
     OSS Insight 的数据资源管理器是一个 GPT 驱动的查询工具，用于 GitHub 实时数据探索。只需用自然语言提出问题，数据探索就会生成 SQL、查询数据并以可视化方式呈现结果。
    </p>
    <p>
     <strong>
      3、NeevaAI
     </strong>
     <br/>
     网址：https://neeva.com/
     <br/>
     NeevaAI 提供真实的实时 AI 搜索。它将 AI 的强大功能与搜索堆栈相结合，提供快速、准确和最新的结果。而且，它提供了信息提取的来源。
    </p>
    <p>
     <strong>
      4、Perplexity AI
     </strong>
     <br/>
     网址：https://www.perplexity.ai/
     <br/>
     perplexity AI是一个AI搜索引擎。这是一个受OpenAI WebGPT启发的演示，而不是商业产品。他们使用大型语言模型（OpenAI API）和搜索引擎。还可以通过将自然语言转换为SQL代码来回答Twitter图形查询。
    </p>
    <p>
     <strong>
      5、AnyPod
     </strong>
     <br/>
     网址：https://www.anypod.ai/
     <br/>
     为创作者打造的 AI 搜索引擎。轻松搜索喜欢的播客，还可以提交添加播客的表单。
    </p>
    <p>
     <strong>
      6、One More AI
     </strong>
     <br/>
     网址：https://onemoreai.com/
     <br/>
     搜索AI 生成的库存图片，查找由人工智能生成的数千张图片。
    </p>
    <p>
     <strong>
      7、Prompt Hunt
     </strong>
     <br/>
     网址：https://www.prompthunt.com/
     <br/>
     一个提供AI绘画提示词的搜索，并且还能生成图片。帮助更快获得想要的图片。
    </p>
    <p>
     <strong>
      8、Libraire
     </strong>
     <br/>
     网址：https://libraire.ai/
     <br/>
     AI 生成图像库。搜索使用AI绘画生成的 图像和提示。
    </p>
    <p>
     <strong>
      9、ChatGPT For Search Engines
     </strong>
     <br/>
     网址：https://chatonai.org/
     <br/>
     直接从搜索引擎结果访问 ChatGPT 语言模型，提出任何问题并获得自然语言响应。
    </p>
    <p>
     <strong>
      10、Ask an AI
     </strong>
     <br/>
     网址：https://www.askan.ai/
     <br/>
     一个接入AI的搜索引擎。
    </p>
    <p>
     <strong>
      11、Fathom.fm
     </strong>
     <br/>
     网址：https://hello.fathom.fm/
     <br/>
     Fathom.fm 是你的 AI 播客搜索。获取推荐的播客，分享最喜爱的剪辑，在播客中搜索，阅读成绩单，使用章节浏览剧集等等。
    </p>
    <p>
     <strong>
      12、Lexii.ai
     </strong>
     <br/>
     网址：https://lexii.ai/
     <br/>
     Lexii.ai 是一个人工智能搜索助手，可以回答问题并引用来源。它由 GPT-3 提供支持。
    </p>
    <p>
     <strong>
      13、Bookabout
     </strong>
     <br/>
     网址：https://bookabout.io/
     <br/>
     一个AI+搜索书籍的工具，帮助搜到想看的书。
    </p>
    <p>
     <strong>
      14、Imaiger
     </strong>
     <br/>
     网址：https://imaiger.com/
     <br/>
     用于生成的艺术和图像的AI图像搜索工具。
    </p>
    <p>
     <strong>
      15、Zevi.ai
     </strong>
     <br/>
     网址：https://www.zevi.ai/
     <br/>
     一个第三方搜索的解决方案方案，采用AI技术使结果更加准确。可以接入网站和APP。
    </p>
    <p>
     <strong>
      16、You
     </strong>
     <br/>
     网址：https://you.com/
     <br/>
     又一个采用AI技术的搜索引擎。
    </p>
    <p>
     <strong>
      17、PromptHero
     </strong>
     <br/>
     网址：https://prompthero.com/
     <br/>
     根据提示词，搜索图片。并提供如何生成该图片的提示词。
    </p>
    <p>
     <strong>
      18、Shaped
     </strong>
     <br/>
     网址：https://www.shaped.ai/
     <br/>
     又一个第三方搜索的解决方案，可接入网站和APP，使用AI使结果更准确。
    </p>
    <p>
     <strong>
      19、CrowdView
     </strong>
     <br/>
     网址：https://crowdview.ai/
     <br/>
     AI搜索引擎，可搜索各种论坛和社区，例如Reddit，HackerNews等。
    </p>
    <p>
     <strong>
      20、All Search AI
     </strong>
     <br/>
     网址：https://www.allsearch.ai/
     <br/>
     搜索 1000 多本书的内容。可以调整要显示结果的时间跨度，流派，获得的结果数量以及速度和质量。
    </p>
    <p>
     <strong>
      21、Everypixel
     </strong>
     <br/>
     网址：https://www.everypixel.com/
     <br/>
     一个由人工智能驱动的搜索引擎，搜索 50 个付费和免费的库存图片网站，并允许用户在几秒钟内搜索庞大的库存照片数据库。
    </p>
    <p>
     还可以按图像和各种搜索过滤器进行搜索，从而能够按颜色、方向和图像类型进行搜索。
    </p>
    <p>
     <strong>
      22、Steno
     </strong>
     <br/>
     网址：https://steno.ai/
     <br/>
     搜索喜爱的播客。
    </p>
    <p>
     <strong>
      23、Bria
     </strong>
     <br/>
     网址：https://bria.ai/
     <br/>
     集成 Bria 的 AI 驱动的 API，以自动化和扩展视频和图像的创建。
    </p>
    <p>
     <strong>
      24、Nuclia
     </strong>
     <br/>
     网址：https://nuclia.com/
     <br/>
     Nuclia 会自动索引来自任何内部和外部来源的非结构化数据，提供优化的搜索结果。它可以处理视频和音频转录、图像内容提取和文档解析。
    </p>
    <p>
     <strong>
      25、ArtHub
     </strong>
     <br/>
     网址：https://arthub.ai/
     <br/>
     搜索 AI 生成的设计、图像、艺术和顶级社区艺术家和设计师的提示提示词。
    </p>
    <p>
     <strong>
      26、Lexica
     </strong>
     <br/>
     网址：https://lexica.art/
     <br/>
     搜索由Stable Diffusion AI绘画生成的图片。
    </p>
    <p>
     <strong>
      27、Algolia
     </strong>
     <br/>
     网址：https://www.algolia.com/
     <br/>
     是一个第三方搜索的解决方案，可接入网站和APP，使用AI使结果更准确。有免费额度。
    </p>
    <p>
     <strong>
      28、Consensus
     </strong>
     <br/>
     网址：https://consensus.app/
     <br/>
     consensus是一个搜索引擎，它使用 AI 直接从科学研究中即时提取、汇总和提炼结果。
    </p>
    <p>
     <strong>
      29、Openart
     </strong>
     <br/>
     网址：https://openart.ai/
     <br/>
     搜索 AI绘画 生成所需的提示词。
    </p>
    <p>
     <strong>
      30、Dreamsands
     </strong>
     <br/>
     网址：https://dreamsands.ai/
     <br/>
     Dreamsands 是一个创意市场，可以在其中许可、收集和分享感兴趣的 AI 生成艺术品的图像。
    </p>
    <p>
     31、Stock AI
     <br/>
     网址：https://www.stockai.com/
     <br/>
     搜索想要的图片，如果没有就会利用AI绘画生成她。
    </p>
    <p>
     <strong>
      32、Krea
     </strong>
     <br/>
     网址：https://www.krea.ai/
     <br/>
     探索数以百万计的 AI 生成的图像并创建提示集合。
    </p>
    <p>
     <strong>
      33、Enterpix
     </strong>
     <br/>
     网址：https://enterpix.app/
     <br/>
     人工智能生成的图像搜索引擎。
    </p>
    <p>
     <strong>
      34、Glean
     </strong>
     <br/>
     网址：https://www.glean.com/
     <br/>
     一个企业级搜索解决方案。主要是搜索公司内部的文档。有点像公司内部Wiki，知识库。
    </p>
    <p>
     <strong>
      35、Looria
     </strong>
     <br/>
     网址：https://www.looria.com/
     <br/>
     Looria 最受信任的来源收集评论，过滤掉虚假评论，并总结调查结果，以帮助用户购买某个物品。
    </p>
    <p>
     可以说是，什么值得买的搜索版。
    </p>
    <p>
     <strong>
      36、Explore AI
     </strong>
     <br/>
     网址：https://exploreai.vercel.app/
     <br/>
     由 AI 提供支持的语义搜索引擎。直接在数以千计的YouTube视频中搜索答案。
    </p>
