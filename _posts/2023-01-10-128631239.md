    <h2>
     <a id="_24">
     </a>
     软件安装
    </h2>
    <h4>
     <a id="1Keil_25">
     </a>
     1.安装Keil
    </h4>
    <p>
     双击MDK535.exe进行安装，安装过程中大部分选项默认即可，注意安装环境不要出现
     <code>
      中文路径
     </code>
     。安装完成后关闭所有页面。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f9585ebcc0253e4eac1a745881b921e6.png"/>
    </p>
    <h4>
     <a id="2PACK_29">
     </a>
     2.安装必要的PACK包
    </h4>
    <p>
     双击Kei.STM32G4xx_DFP.1.2.0.pack，进行安装，一路Next，使用默认选项安装即可。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3914c69de74c867cd645d467cf1dd8d.png"/>
    </p>
    <h4>
     <a id="3Keil_32">
     </a>
     3.注册激活Keil
    </h4>
    <p>
     <strong>
      3.1 关闭电脑上的杀毒软件以及Windows实时保护
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9ac8e63b0aaf7b12dc241f1bdabee7da.png#pic_center">
      <br/>
      <strong>
       3.2 然后以管理员方式运行Keil
      </strong>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/71d0cb476dc062641f10a527fceabfa7.png#pic_center">
      <br/>
      <strong>
       3.3 打开Keil后, 点击
       <code>
        File
       </code>
       &gt;&gt;
       <code>
        License Managent ...
       </code>
      </strong>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/09e6299febccee9864c2d9d8857e9c30.png#pic_center"/>
    </p>
    <p>
     <strong>
      3.4 将出现如下页面，复制页面上的
      <code>
       CID
      </code>
      ：
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fcc13a3f0d113aa40063fa97528c2dee.png#pic_center"/>
     <br/>
     <strong>
      3.5 解压并打开Keil注册机
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0eebc62740d8412147372a028a647f2f.png"/>
     <br/>
     将Keil中的CID复制到注册机中，然后Target选择ARM，点击Generate生成注册码，再把注册码复制到Keil，点击Add LIC即可激活成功。
    </p>
    <p>
     下面给出了51版本激活动画参考，注意一定要把下图中的
     <code>
      Target从C51改成ARM
     </code>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c276eb1e9bfb4a7de11d6447ce306de0.gif#pic_center"/>
    </p>
    <h4>
     <a id="4STM32CubeMX_51">
     </a>
     4.安装STM32CubeMX
    </h4>
    <p>
     加压STM32CubeMX的安装文件，双击安装即可，使用默认安装选项，如果更改了安装路径，注意路径中不要出现中文。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8916b8d4c29abc022997090c0f0243b4.png"/>
    </p>
    <h2>
     <a id="_56">
     </a>
     开发环境验证
    </h2>
    <p>
     安装完成上述软件后，需要验证下是否安装正确。可以从QQ群或者百度云盘下载蓝桥杯官方提供的资料包，编译运行一下示例代码。
    </p>
    <p>
     <strong>
      1.下载资料包
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b079fdecc073755f011866321bdd1602.png"/>
     <br/>
     <strong>
      2.解压并打开示例代码
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1dc88cea6e51bbe959eb98ce72fd0bb4.png"/>
     <br/>
     <strong>
      3.配置工程
     </strong>
    </p>
    <p>
     点击keil中的
     <code>
      魔法棒
     </code>
     进入配置，选择
     <code>
      Debug
     </code>
     ，点击Use后面的
     <code>
      Setting
     </code>
     ，在新打开的页面中选择
     <code>
      Flash Download
     </code>
     ，点击
     <code>
      Add
     </code>
     ，选择
     <code>
      STM32G4xx_128_Flash
     </code>
     ，点击
     <code>
      Add
     </code>
     ，然后一路
     <code>
      OK
     </code>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6704c0c1c112f04f45298d8375fb2e14.png"/>
     <br/>
     配置完成后编译代码并运行，如果运行成功说明Keil安装完成。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/18f7b5dae44f5fe4832f659d5b2e3dc2.png"/>
     <br/>
     <strong>
      4.验证STM32CubeMX
     </strong>
    </p>
    <p>
     双击打开下图所示的文件，首次打开需要下载一个库，耐心等待。如果有提示请点击Continue。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/688751a7b6218c43b44c4478e6b5d8a3.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e155f479704248aacb398a82553c33d8.png"/>
     <br/>
     出现下图所示的页面，说明STM32Cube安装成功：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b2680cf0244d7a9278ccaaf8580cdfb4.png"/>
    </p>
