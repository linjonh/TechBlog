    <h4>
     <a id="100606000_0">
     </a>
     问题描述：当你在电脑上浏览京东网站时，官网只给你返回100页数据，每页60个，总共只显示6000个商品，但实际有几十万的相关产品没有展示出来。
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/20191015101508288.png"/>
    </p>
    <h6>
     <a id="font_colorred__3">
     </a>
     <font color="red">
      这算是京东的一种反扒机制，美团的电脑端也是这样，只给你返回固定页数的信息。遇到这个问题，可以选择
      <code>
       增加搜索维度、寻找其他接口
      </code>
      的方法，尽量多的去获取数据。
     </font>
    </h6>
    <h5>
     <a id="_4">
     </a>
     解决方法：
    </h5>
    <h5>
     <a id="_5">
     </a>
     一、在搜索的时候，缩小搜索范围，增加搜索维度
    </h5>
    <ul>
     <li>
      1、在搜索商品时，加上品牌、型号等信息，缩小范围
      <br/>
      <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/20191015102207350.png"/>
     </li>
     <li>
      2、利用好京东的导航栏，进行每一层的缩小范围
      <br/>
      <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/2019101510241834.png">
       <br/>
       这个方法更可取一些，可以循环导航栏的信息，去一层一层的获取。
       <br/>
       这个导航栏可以用xpath到每一个的链接。
      </img>
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token comment"># 提取商品页的所有相关商品 </span>
response<span class="token punctuation">.</span>xpath<span class="token punctuation">(</span><span class="token string">"//div[contains(@class,'J_selectorLine')]/@class"</span><span class="token punctuation">)</span>
<span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span><span class="token operator">-</span>
J_selectorLine s<span class="token operator">-</span>brand     <span class="token comment"># 商品的品牌栏</span>

J_selectorLine s<span class="token operator">-</span>line      <span class="token comment"># 商品的细分条件栏</span>
J_selectorLine s<span class="token operator">-</span>line s<span class="token operator">-</span>senior  <span class="token comment"># 商品的选项栏</span>
</code></pre>
    <h5>
     <a id="_23">
     </a>
     二、如果搜索固定的商品，可以找到它专属的商品页深度挖掘
    </h5>
    <ul>
     <li>
      1、利用好京东的导航栏，找到你想要抓取的信息
      <br/>
      <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/20191015103119741.png"/>
     </li>
     <li>
      2、跳转到二级域名下的商品列表里，他展示全部的页码。
      <br/>
      <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/201910151034131.png"/>
     </li>
     <li>
      3、这种方法可以用于全站抓取哦。
     </li>
    </ul>
    <h5>
     <a id="_29">
     </a>
     三、商品详情页的同款商品可以一并获取
    </h5>
    <ul>
     <li>
      <p>
       1、抓到具体到商品详情页后，会发现很多同款不同信号的商品，可以一起抓取
       <br/>
       <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/2019101511053669.png"/>
      </p>
     </li>
     <li>
      <p>
       2、这个商品列表，需要用到源代码正则匹配。
       <br/>
       <img alt="在这里插入图片描述" src="https://img-blog.csdnimg.cn/20191015110757694.png">
        <br/>
        <code>
         正则规则： re.findall(r"colorSize:(.*?}]),[ ]+?[warestatus:]+", response)
        </code>
       </img>
      </p>
      <p>
       这时，提取到的是每一个商品的
       <code>
        skuId
       </code>
       和
       <code>
        颜色信息
       </code>
       ，再用字符串拼接商品的URL
       <code>
        "https://item.jd.com/{skuId}.html"
       </code>
       ，去提取每一个商品的信息即可。
      </p>
     </li>
    </ul>
