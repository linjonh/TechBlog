     简单的的C语言编程小游戏——走迷宫
    </h2>
    <p>
     C语言是一种非常适合编写游戏的编程语言，今天我就用c语言编写一个十分简单的小游戏–走迷宫。
    </p>
    <h3>
     <a id="_2">
     </a>
     设计思路
    </h3>
    <h4>
     <a id="1_3">
     </a>
     1.数据分析：
    </h4>
    <p>
     定义变量 ： 迷宫：ｃｈａｒ[10][10];
     <br/>
     墙　‘＃’
     <br/>
     老鼠 ‘＠’
     <br/>
     出口 ‘ ’
     <br/>
     路 ‘ ’
     <br/>
     老鼠的坐标：　ｃｈａｒ　mx,my;
     <br/>
     出口的坐标：　　　ｃｈａｒ outx,outy;
    </p>
    <h4>
     <a id="2_13">
     </a>
     2.业务流程：
    </h4>
    <p>
     1.显示迷宫：先清理屏幕，再显示迷宫。
     <br/>
     2.判断是否到达出口：　判断老鼠的坐标与出口的坐标是否重合。
     <br/>
     3.获取方向键：
     <br/>
     4.先把原位置赋值为’’,新位置赋值为’＠’。
     <br/>
     5.不能穿墙，不能越界。
    </p>
    <h6>
     <a id="_19">
     </a>
     此时要调用两个函数：
    </h6>
    <ol>
     <li>
      system(“clear”）         清理屏幕函数
     </li>
     <li>
      getch()      调用键盘函数；
     </li>
    </ol>
