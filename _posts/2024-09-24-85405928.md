    <p style="margin-left:0px;">
     <span style="color:#333333;">
      记录设备接入阿里云的过程
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      五，一型一密
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      前几篇文章写的都是一机一密，但是实际项目中设备往往很多，不可能一一修改代码烧录，所以阿里云提供了一型一密的解决方案
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      一型一密安全认证方式，即为同一产品下所有设备烧录相同固件（固件中写入产品证书，即ProductKey和ProductSecret）。设备发送激活请求时，物联网平台根据产品证书进行认证。认证通过，物联网平台下发该设备对应的DeviceSecret。
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      1，开启一型一密
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      我们看一下我们之前创建的产品
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      <a href="https://i-blog.csdnimg.cn/blog_migrate/317dba22780f495854a5f3e27ca393f1.png" rel="nofollow">
       <img alt="" class="img-responsive" src="https://i-blog.csdnimg.cn/blog_migrate/317dba22780f495854a5f3e27ca393f1.png" width="644"/>
      </a>
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      我们只需要打开这个动态注册，我们的产品就具有了一型一密的功能
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      2，新增设备
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      新增设备的方法与之前一样，但是有一点，现在的DeviceName必须是设备的imei
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      3，修改代码
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      去掉这里注释，修改相应参数为自己的产品的
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      <a href="https://i-blog.csdnimg.cn/blog_migrate/fd26c24ab8e5e580e433b6ffe4457e11.png" rel="nofollow">
       <img alt="" class="img-responsive" src="https://i-blog.csdnimg.cn/blog_migrate/fd26c24ab8e5e580e433b6ffe4457e11.png" width="275"/>
      </a>
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      获取imei作为DeviceName
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      <a href="https://i-blog.csdnimg.cn/blog_migrate/6bec4219759cc7ee2375a700b96fdc7d.png" rel="nofollow">
       <img alt="" class="img-responsive" src="https://i-blog.csdnimg.cn/blog_migrate/6bec4219759cc7ee2375a700b96fdc7d.png" width="432"/>
      </a>
      <a href="https://i-blog.csdnimg.cn/blog_migrate/1697faf844996d344b293a8fa5f95a44.png" rel="nofollow">
       <img alt="" class="img-responsive" src="https://i-blog.csdnimg.cn/blog_migrate/1697faf844996d344b293a8fa5f95a44.png" width="520"/>
      </a>
      <br/>
      注释一机一密，去掉一型一密的注释
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      <a href="https://i-blog.csdnimg.cn/blog_migrate/515c3e75d12b7a9a66d6da2d65d8d218.png" rel="nofollow">
       <img alt="" class="img-responsive" src="https://i-blog.csdnimg.cn/blog_migrate/515c3e75d12b7a9a66d6da2d65d8d218.png" width="515"/>
      </a>
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      四，烧录
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      把代码下载进设备，可以看到刚才创建的设备已经激活
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      五，注意
     </span>
    </p>
    <p style="margin-left:0px;">
     <span style="color:#333333;">
      必须先在阿里云根据设备imei新建设备，一型一密只是安全认证方式，只用于设备认证不能自动添加设备
     </span>
    </p>
