    <h2>
     <a id="1_1">
     </a>
     1.背景
    </h2>
    <p>
     Mac使用lz rz命令时出现卡顿，所以想通过scp命令进行文件的上传和下载。
    </p>
    <h2>
     <a id="2_4">
     </a>
     2.解决方案
    </h2>
    <h3>
     <a id="21scp_5">
     </a>
     2.1.使用scp连接远程主机
    </h3>
    <pre><code class="prism language-shell"><span class="token function">ssh</span> root@47.92.xx.xx <span class="token comment">#连接时会提醒输入密码，正确输入密码即可远程连接</span>
</code></pre>
    <h3>
     <a id="22Mac_10">
     </a>
     2.2.将Mac的文件上传至远程服务器
    </h3>
    <pre><code class="prism language-shell"><span class="token function">scp</span> /Users/allenpandas/Downloads/test.txt root@47.92.xx.xx:/opt  <span class="token comment"># 将本地的test.txt上传至远程服务器的/opt目录</span>
</code></pre>
    <h3>
     <a id="23Mac_15">
     </a>
     2.3.将Mac的文件上传至远程服务器
    </h3>
    <pre><code class="prism language-shell"><span class="token function">scp</span> -r /Users/allenpandas/Downloads root@47.92.xx.xx:/downloads  <span class="token comment"># 将本地的Downloads整个目录上传至远程服务器的/downloads目录下</span>
</code></pre>
    <h3>
     <a id="24Mac_20">
     </a>
     2.4.从远程服务器上下载文件到Mac
    </h3>
    <pre><code class="prism language-shell"><span class="token function">scp</span> root@47.92.xx.xx:/opt/test.txt /Users/allenpandas/Desktop
</code></pre>
    <h3>
     <a id="25_25">
     </a>
     2.5.从远程服务器下载整个目录
    </h3>
    <pre><code class="prism language-shell"><span class="token function">scp</span> -r root@47.92.xx.xx:/downloads /Users/allenpandas/Desktop
</code></pre>
