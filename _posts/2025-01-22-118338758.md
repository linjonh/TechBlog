     小程序vant-weapp的日历组件的使用
    </h2>
    <p>
     话不多说，记录一下这个框架的使用~小程序使用轻量、可靠的小程序 UI 组件库 vant-weapp
     <br/>
     Github源码：
     <a href="https://links.jianshu.com/go?to=https%3A%2F%2Fgithub.com%2Fyouzan%2Fvant-weapp" rel="nofollow">
      https://github.com/youzan/vant-weapp
     </a>
     <br/>
     中文文档：
     <a href="https://links.jianshu.com/go?to=https%3A%2F%2Fyouzan.github.io%2Fvant-weapp%2F%23%2Fintro" rel="nofollow">
      https://youzan.github.io/vant-weapp/#/intro
     </a>
    </p>
    <hr/>
    <p>
     1：打开微信开发者工具，填写自己的appid和项目名称，选择不使用云服务，新建一个项目。
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/c51e95b9c37f04f05dbdc7a65ffdb878.png"/>
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/c1868a3bda4552f05aa4980046b3b4cc.png"/>
    </p>
    <p>
     2：右击在选择在终端打开
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/87aa5934a3be28dedf79dc97431d58f1.png"/>
    </p>
    <p>
     进入项目的根目录底下，注意，一定要进入根目录哦，使用cd …/返回上一级目录~
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/228cdb0791a06215a3c11339dfdf5f4e.png"/>
    </p>
    <p>
     3：小程序已经支持使用 npm 安装第三方包，
     <br/>
     这里通过 npm 安装
    </p>
    <pre><code>   1、第一步：npm init

   2、第二步：npm install --production

   3、第三步： npm i @vant/weapp -S --production
      或者  npm i vant-weapp -S --production

</code></pre>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/93721b8c19f78e2ae89f23a87dbae551.png"/>
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/11c8c4300b1db8b60e8cfa4851d7b766.png"/>
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/0044001cb98513bc5ddbc4f4da49882c.png"/>
    </p>
    <p>
     这里需要注意一下
     <br/>
     npm i vant-weapp -S --production或者npm i @vant/weapp -S --production
     <br/>
     引入的区别
    </p>
    <p>
     使用npm i vant-weapp安装的时候，到时候在在app.json或index.json中引入组件，需要使用这样的路径
    </p>
    <pre><code>{
  "usingComponents": {
    "van-button": "../../miniprogram_npm/vant-weapp/button/index"
  }
}

</code></pre>
    <p>
     使用npm i @vant/weapp安装的时候，到时候在在app.json或index.json中引入组件，需要使用这样的路径（推荐，因为这个可以直接抄文档，不需要改变引入路径的~）
    </p>
    <pre><code>{
"usingComponents": {
  "van-button": "@vant/weapp/button/index"
}
}

</code></pre>
    <p>
     4：在微信开发工具执行npm 构建，点击工具里面，构建npm
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/eb1688fc8284fa953ca5c7b784d72626.png"/>
    </p>
    <p>
     构建过程需要等待一会儿，不要捉急
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/7ad70cd94525714ec208692ef01ad49d.png"/>
    </p>
    <p>
     构建完会生成一个miniprogram_npm文件夹
     <br/>
     如果构建完如果编译报错，再构建一次就好了
    </p>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/2afd0bd064261916dce2189756c8328a.png"/>

    <p>
     5：使用vant-weapp日历组件
     <br/>
     我这里对日期的处理，是需要这样的格式YYYY-MM-dd
     <br/>
     所以在对选中的日期做了一些处理和判断~
    </p>
    <pre><code>2020-10-26

</code></pre>
    <p>
     wxml
    </p>
    <pre><code>&lt;form catchsubmit="confirmPublish"&gt;
  &lt;view class="cu-form-group " bindtap="showCalendar"&gt;
    &lt;view&gt;任务时限&lt;/view&gt;
    &lt;input disabled="disabled" placeholder="请选择任务时限" value="{<!-- -->{time}}" data-name="time"&gt;
    &lt;/input&gt;
  &lt;/view&gt;
  &lt;button class='btn1' form-type="submit"&gt;确认发布&lt;/button&gt;
&lt;/form&gt;
&lt;van-cell title="选择单个日期" value="{<!-- -->{ date }}" bind:click="onDisplay" /&gt;
&lt;van-calendar show="{<!-- -->{ show }}" bind:close="onClose" bind:confirm="onConfirm" /&gt;

</code></pre>
    <p>
     js
    </p>
    <pre><code>
const app = getApp()
Page({
  data: {
    maxDate: new Date().setFullYear(new Date().getFullYear() + 2),
    show: false,
    taskStartTime: '',
    time: '',
  },
  showCalendar() {
    this.setData({ show: true })
  },
  onClose() {
    this.setData({ show: false })
  },
  formatDate(date) {
    let taskStartTime
    if (date.getMonth() &lt; 9) {
      taskStartTime = date.getFullYear() + "-0" + (date.getMonth() + 1) + "-"
    } else {
      taskStartTime = date.getFullYear() + "-" + (date.getMonth() + 1) + "-"
    }
    if (date.getDate() &lt; 10) {
      taskStartTime += "0" + date.getDate()
    } else {
      taskStartTime += date.getDate()
    }
    this.setData({
      taskStartTime: taskStartTime,
    })
    return taskStartTime;
  },
  onConfirm(e) {
    this.setData({
      time: this.formatDate(e.detail),
      show: false
    })
  },
  onLoad: function (options) {
  },
  confirmPublish: function () {
    const data = {}
    data.taskStartTime = this.data.taskStartTime
    console.log(JSON.stringify(data))
  },
})

</code></pre>
