     1、服务器上安装git
    </h3>
    <p>
     <strong>
      安装命令：yum -y install git
     </strong>
    </p>
    <p>
     yum -y install 包名（支持*） ：自动选择y，全自动
    </p>
    <p>
     yum install 包名（支持*） ：手动选择y or n
    </p>
    <p>
     yum remove 包名（不支持*）
    </p>
    <p>
     rpm -ivh 包名（支持*）：安装rpm包
    </p>
    <p>
     rpm -e 包名（不支持*）：卸载rpm包
    </p>
    <p>
    </p>
    <p>
     <strong>
      安装完成后 git --version 查看版本
     </strong>
    </p>

     <img alt="" height="55" src="https://i-blog.csdnimg.cn/blog_migrate/80466e492b9fe2f5a838a2f90c70fe56.png" width="351"/>

    <h3>
     3、创建远程仓库
    </h3>
    <p>
     1.首先创建一个目录，在该目录下执行git init  即可创建一个可供远程访问的git服务器,如此即可创建一个git仓库。远程登陆即可使用。
    </p>
    <p>
     2.在本机，新建一个文件夹，使用cmd或者git.bash进入到该目录下，执行
     <br/>
     git clone daodao@192.168.214.001:/home/daodao/git/my.git
     <br/>
     然后根据提示输入登陆daodao的密码即可clone下来。
    </p>
    <h3>
     4、客户端安装git
    </h3>
    <p>
     客户端安装git完了之后创建文件夹并点击git bash Here，配置完用户邮箱执行git clone daodao@192.168.214.001:/home/daodao/git/my.git
     <br/>
     然后根据提示输入登陆daodao的密码即可clone下来。
    </p>
    <p>
     <img alt="" height="450" src="https://i-blog.csdnimg.cn/blog_migrate/4d9e262c38fddc68b1659df6387b7212.png" width="280"/>
    </p>
    <h3>
     5、获取公钥
    </h3>
    <p>
     本地仓库获取到公钥配置到服务器即可实现免密操作git，
    </p>
    <p>
     <strong>
      客户端生成私钥和公钥
     </strong>
     <br/>
     在bit bash中执行ssh-keygen -t rsa 并按回车3下
     <br/>
     （为什么按三下，是因为有提示你是否需要设置密码，
     <br/>
     如果设置了每次使用Git都会用到密码，
     <br/>
     一般都是直接不写为空，直接回车就好了）
     <br/>
     会在一个文件夹里面生成一个私钥 id_rsa和一个公钥id_rsa.pub，
     <br/>
     默认文件夹在你打开git bash的目录，或者在你的用户目录~/.ssh文件夹下
    </p>
    <p>
     <strong>
      添加公钥到服务器
     </strong>
    </p>
    <p>
     在服务器的仓库目录创建.ssh文件夹
    </p>
    <blockquote>
     <p>
      cd /home/git
     </p>
     <p>
      mkdir .ssh
     </p>
     <p>
      chmod 700 ./ssh 赋予权限
     </p>
     <p>
      cd .ssh
     </p>
     <p>
      vim authorized_keys  将你的公钥复制进去  :wq保存退出
     </p>
     <p>
      chmod 600 ./authorized_keys 赋予权限
     </p>
    </blockquote>

      <span style="color:#333333;">
       git push
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git push [variable name] master
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令可以将主分支上提交的变更发送到远程代码库。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <img alt="" height="144" src="https://i-blog.csdnimg.cn/blog_migrate/57a41af156fe7687d26101e059659700.png" width="640"/>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git push [variable name] [branch]
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令可以将指定分支上的提交发送到远程代码库。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <img alt="" height="123" src="https://i-blog.csdnimg.cn/blog_migrate/7e03801c9f59b913c1bc12df7b58bcd7.png" width="640"/>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git push –all [variable name]
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令可以将所有分支发送到远程代码库。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <img alt="" height="160" src="https://i-blog.csdnimg.cn/blog_migrate/4b97e8148c48579ed7d79bd890badb26.png" width="640"/>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git push [variable name] :[branch name]
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令可以删除远程代码库上的一个分支。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#333333;">
       git pull
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git pull [Repository Link]
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令将获取远程服务器上的变更，并合并到你的工作目录。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <img alt="" height="98" src="https://i-blog.csdnimg.cn/blog_migrate/9cfa6a98174ab1631e24d6af78350c83.png" width="640"/>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#333333;">
       git stash
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git stash save
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令将临时保存所有修改的文件。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git stash pop
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令将恢复最近一次
     </span>
     <span style="color:#333333;">
      stash
     </span>
     <span style="color:#333333;">
      （储藏）的文件。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <img alt="" height="133" src="https://i-blog.csdnimg.cn/blog_migrate/10acda179ca9d5062c561f2a8e68496a.png" width="640"/>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git stash list
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令将显示
     </span>
     <span style="color:#333333;">
      stash
     </span>
     <span style="color:#333333;">
      的所有变更。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      用法：
     </span>
     <span style="color:#333333;">
      git stash drop
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      该命令将丢弃最近一次
     </span>
     <span style="color:#333333;">
      stash
     </span>
     <span style="color:#333333;">
      的变更。
     </span>
    </p>
