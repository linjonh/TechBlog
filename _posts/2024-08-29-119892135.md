     1、电商项目业务流程

    <p>

    <h3 id="2-商品分类浏览包含的功能点">
     2、商品分类浏览包含的功能点
    </h3>
    <h3 id="2-1-挑选商品逻辑分析">
     2.1 挑选商品逻辑分析
    </h3>
    <p style="text-align:center;">
     <img alt="图片.png" src="https://i-blog.csdnimg.cn/blog_migrate/bc557e24356228275390f438cafb6a25.png"/>
    </p>
    <p>
    </p>
    <p style="text-align:center;">
     <img alt="图片.png" src="https://i-blog.csdnimg.cn/blog_migrate/6aa4f4ca1f48ac4f24b97c3d116a516d.png"/>
    </p>
    <ul>
     <li>
      <strong>
       商品分类
      </strong>
      <br/>
      显示所有一级商品分类、二级分类、三级分类
      <br/>
      鼠标移动到主分类时，右侧浮窗显示子分类
      <br/>
      点击子分类，跳转至商品筛选页面
     </li>
     <li>
      <strong>
       商品推荐
      </strong>
      <br/>
      显示首页推荐商品品类、商品列表
      <br/>
      点击商品分类，跳转至商品筛选页面
      <br/>
      点击具体商品，跳转至商品详情页面
     </li>
     <li>
      <strong>
       商品搜索
      </strong>
      <br/>
      商品搜索
      <br/>
      依据商品关键字搜索
      <br/>
      显示热门搜索关键字
     </li>
    </ul>
    <h3 id="2-2-商品筛选页面">
     2.2 商品筛选页面
    </h3>
    <p style="text-align:center;">
     <img alt="图片.png" src="https://i-blog.csdnimg.cn/blog_migrate/2c583031bded4930cba228f30bed5035.png"/>
    </p>
    <ul>
     <li>
      <strong>
       商品筛选
      </strong>
      <br/>
      可以按照品牌、分类、价格、尺寸等进行过滤
     </li>
     <li>
      <strong>
       商品排序
      </strong>
      <br/>
      可以按照 综合、销量、价格进行排序
     </li>
    </ul>
    <h3 id="2-3-商品详情页面">
     2.3 商品详情页面
    </h3>
    <p style="text-align:center;">
     <img alt="图片.png" src="https://i-blog.csdnimg.cn/blog_migrate/c5e942d364e9a53c474e140858d512b2.png"/>
    </p>
    <ul>
     <li>
      <strong>
       商品基本信息
      </strong>
      <br/>
      商品图片，多张支持鼠标浮窗查看大图
      <br/>
      商品标题、简述
      <br/>
      商品价格：原价、折扣价等
      <br/>
      累计销量、浏览次数、累计评论
      <br/>
      商品参数、尺码、规格、数量
      <br/>
      加入购物车、立即购买、收藏
     </li>
     <li>
      <strong>
       详情
      </strong>
      <br/>
      商品参数
      <br/>
      展示商品详情页信息
     </li>
     <li>
      <strong>
       评论
      </strong>
      <br/>
      展示评分、评论人头像、昵称(匿名/不匿名)、评论时间、评论内容、回复等
     </li>
     <li>
      <strong>

      <br/>
      依据开发给定的算法规则进行验证
