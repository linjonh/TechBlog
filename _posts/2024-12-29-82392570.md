     微软力挺 Go，宣布参与 Athens 项目和 GopherSource
    </h1>

     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;background-color:rgb(255,255,255);text-align:center;line-height:1.75em;">
      <img alt="640?wx_fmt=gif" class="__bg_gif" src="https://i-blog.csdnimg.cn/blog_migrate/9740fb08d32c71f4d4b35f3bdaa15340.gif" width="427"/>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       在最近举办的 Go 开发者大会 GopherCon 上，微软和大家了分享它为 Go 社区所做的工作，并宣布将要为 Athens 项目贡献代码，以及与 GopherSource 的合作。
      </span>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       在这么一个正式且重大的会议上，微软的这一系列动作，显然释放出了它要大力支持 Go 语言的信号。不过我们更愿意相信，微软更看重的是 Go 语言给自家云服务产品带来的强大“赋能”。
      </span>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       Athens 是一个开源项目，旨在为 Go 模块(Go modules)创建首个代理服务器。微软表示该项目目前仍处于 alpha 阶段，并将联合 Athens 社区继续致力于改善模块体验，重点是确保 Go 模块与所有代理服务器能无缝协作，并努力建立一个联合的、组织多样化的代理网络，其中包括：
      </span>
     </p>
     <ul class="list-paddingleft-2">
      <li>
       <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
        <span style="font-size:15px;letter-spacing:1px;">
         用于边缘部署的 Go 模块代理服务器；
        </span>
       </p>
      </li>
      <li>
       <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
        <span style="font-size:15px;letter-spacing:1px;">
         用于身份验证的模块代理的协议；
        </span>
       </p>
      </li>
      <li>
       <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
        <span style="font-size:15px;letter-spacing:1px;">
         模块公证服务以用于验证模块源码；
        </span>
       </p>
      </li>
      <li>
       <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
        <span style="font-size:15px;letter-spacing:1px;">
         在批准外部 Go 包时指定要包含和排除内容的功能。
        </span>
       </p>
      </li>
     </ul>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:center;background-color:rgb(255,255,255);line-height:1.75em;">
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/98edc603eeb7f1c2a551b0501f618659.jpeg" style="font-size:15px;letter-spacing:1px;">
       <br/>
      </img>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       Athens 是一个诞生自社区的项目，微软参与进来后，将会继续为社区的发展和技术贡献自己的力量，目前有数十名开发者在参与该项目的开发。
      </span>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       而文章刚开始所提到的 GopherSource，其实是一项新计划，旨在通过在社区内为上游项目和关键 Go 项目（如 Athens）带来更多用户和贡献者，以增强和实现 Go 生态系统的多样化。微软 Azure 的布道师 Aaron Schlesinger 表示：“Go 社区声名在外，但它仍然是一个令人困惑和陌生的领域，需要吸引新的和经验丰富的开发者。通过鼓励 Go 社区自己的优秀开发者为上游的 Go 项目做出贡献，微软确保 Go 生态系统能够满足整个社区的需求。”
      </span>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       微软还表示，公司内部也有一定的规模在使用 Go 来构建产品和服务。例如最近 Azure Kubernetes Service 背后的核心服务使用了 Go 来实现。
      </span>
     </p>
     <p style="min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;text-align:justify;background-color:rgb(255,255,255);line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       此外，微软正致力于在其自己的产品和服务中为 Go 开发者提供更好的开发体验，让他们能构建和运行更好的应用程序和服务，一些改进包括 Visual Studio Code 中原生 Go 支持的扩展、对 Azure 服务的 Go 支持，以及 Visual Studio Team Services 中 Go 应用程序的 CI/CD 功能。
      </span>
     </p>
