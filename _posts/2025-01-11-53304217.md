    <span style="font-family:Comic Sans MS; font-size:18px; background-color:rgb(255,255,255)">
     <span style="color:rgb(51,51,51)">
      create命令用于创建数据库。
     </span>
     <br style="color:rgb(51,51,51)"/>
     <br style="color:rgb(51,51,51)"/>
     <span style="color:rgb(51,51,51)">
      create命令格式：create database &lt;数据库名&gt;;
     </span>
     <br style="color:rgb(51,51,51)"/>
     <br style="color:rgb(51,51,51)"/>
     <span style="color:rgb(178,34,34)">
      注意：创建数据库之前要先连接Mysql服务器。
     </span>
     <br style="color:rgb(51,51,51)"/>
     <br style="color:rgb(51,51,51)"/>
     <span style="color:rgb(51,51,51)">
      1) 建立一个名为xhkdb的数据库：
     </span>
     <br style="color:rgb(51,51,51)"/>
     <span style="color:rgb(51,51,51)">
      mysql&gt; create database xhkdb;
     </span>
     <br style="color:rgb(51,51,51)"/>
     <br style="color:rgb(51,51,51)"/>
     <span style="color:rgb(51,51,51)">
      2) 创建数据库并分配用户：
     </span>
    </span>
    <ol style="margin-top:0px; margin-bottom:0px; list-style-position:initial; margin-left:0px; padding-left:20px; color:rgb(51,51,51)">
     <li>
      <span style="font-family:Comic Sans MS; font-size:18px; background-color:rgb(255,255,255)">
       CREATE DATABASE 数据库名;
      </span>
     </li>
     <li>
      <span style="font-family:Comic Sans MS; font-size:18px; background-color:rgb(255,255,255)">
       GRANT SELECT,INSERT,UPDATE,DELETE,CREATE,DROP,ALTER ON 数据库名.* TO 数据库名@localhost IDENTIFIED BY '密码';
      </span>
     </li>
     <li>
      <span style="font-family:Comic Sans MS; font-size:18px; background-color:rgb(255,255,255)">
       SET PASSWORD FOR '数据库名'@'localhost' = OLD_PASSWORD('密码');
      </span>
     </li>
    </ol>
    <span style="color:rgb(51,51,51); background-color:rgb(255,255,255)">
     <span style="font-family:Comic Sans MS; font-size:18px">
      依次执行3个命令完成数据库创建。
     </span>
