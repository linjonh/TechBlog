    <h2>
     <a id="_7">
     </a>
     介绍
    </h2>
    <p>
     写这篇的初衷是怕自己忘了这两个工具的名字。
    </p>
    <h3>
     <a id="Deskpins_104_KB_11">
     </a>
     Deskpins （功能单一，104 KB）
    </h3>
    <h4>
     <a id="1__13">
     </a>
     1 简介
    </h4>
    <p>
     优点：小，简单，适用于大部分 Windows 系统。
     <br/>
     缺点：钉子偶尔失灵、只有英文版。
     <br/>
     开源位置：
     <a href="https://github.com/thewhitegrizzli/DeskPins/releases">
      https://github.com/thewhitegrizzli/DeskPins/releases
     </a>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a5e79e5071c041f099beda8511696222.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1c7c471e1ae34c05a6fe6ef50a93424e.png"/>
    </p>
    <h4>
     <a id="2__23">
     </a>
     2 详细介绍
    </h4>
    <p>
     DeskPins 是由
     <strong>
      thewhitegrizzli
     </strong>
     开发的一个轻量级开源工具，允许用户将任意窗口置顶，简化多任务处理。用户可以通过点击系统托盘中的“插针”图标选择窗口固定在其他应用上方，并支持自定义快捷键。该软件设计简单、免费且占用极少资源，适用于所有 Windows 版本。
    </p>
    <p>
     关于开发者，
     <strong>
      thewhitegrizzli
     </strong>
     是一位 GitHub 用户，致力于开发实用的工具来提升桌面工作效率。
    </p>
    <p>
     了解更多并下载 DeskPins，请访问
     <a href="https://github.com/thewhitegrizzli/DeskPins">
      GitHub 页面
     </a>
     。
    </p>
    <h3>
     <a id="WindowTop_32MB__31">
     </a>
     WindowTop （功能全面，32MB 左右）
    </h3>
    <h4>
     <a id="1__33">
     </a>
     1 简介
    </h4>
    <p>
     优点：功能完美，有中文界面。
     <br/>
     缺点：比 Deskpins 大点，功能需要自己探索。
     <br/>
     开源位置：
     <a href="https://github.com/WindowTop/WindowTop-App/releases">
      https://github.com/WindowTop/WindowTop-App/releases
     </a>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6c27d8e07d7f4ae8b0007e08c13e9302.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/877c1b3d6a4d4dc59c59353f36cf69f0.png"/>
     </img>
    </p>
    <h4>
     <a id="2__42">
     </a>
     2 详细介绍
    </h4>
    <p>
     WindowTop 是一个增强生产力的工具，允许您将窗口置顶，并通过多种方式优化多任务处理。它具有“始终置顶”、“画中画（PiP）”、透明度调整和点击穿透、锚点、暗模式等功能。使用 WindowTop，您可以快速切换窗口、同时处理多个任务，甚至通过 PiP 模式在工作时观看视频。还可以启用暗模式或玻璃模式来减少屏幕亮度，提升工作舒适度。
    </p>
    <p>
     您可以访问官方网站了解更多信息:
     <a href="https://windowtop.info/" rel="nofollow">
      WindowTop
     </a>
    </p>
    <p>
     <strong>
      以下是它的主要功能列表以及对应的演示
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        窗口置顶（Always-on-Top）
       </strong>
       ：将任何窗口置于其他窗口之上，方便随时查看。
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/8c1f8c7f71cc407483e929668592fd69.gif"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        画中画模式（Picture-in-Picture, PiP）
       </strong>
       ：缩小窗口以便同时处理多个任务，例如工作时观看视频。
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/af9a14063c89471083458240e708aac0.gif"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        透明度和点击穿透（Opacity &amp; Click-Through）
       </strong>
       ：调整窗口透明度，允许点击窗口背后的内容。
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/9fb9548b38094ac7b5343f3ae7610bea.gif"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        窗口锚点（Anchors）
       </strong>
       ：将窗口固定在屏幕的特定位置，避免其意外移动。
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/a650fbc6d12645f1b92657f8e8cdf71b.gif"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        暗模式（Dark Mode）
       </strong>
       ：通过启用暗模式或玻璃模式减少眼睛疲劳。
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/60dc9e4ef4424d76ba73af4352d2c61c.gif"/>
      </p>
     </li>
    </ul>
