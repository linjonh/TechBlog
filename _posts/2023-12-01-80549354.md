    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     由于历史原因，Python有两个大的版本分支，Python2和Python3，又由于一些库只支持某个版本分支，所以需要在电脑上同时安装Python2和Python3，因此如何让两个版本的Python兼容，如何让脚本在对应的Python版本上运行，这个是值得总结的。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     对于Ubuntu 16.04 LTS版本来说，Python2（2.7.12）和Python3（3.5.2）默认同时安装，默认的python版本是2.7.12。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="73" src="https://i-blog.csdnimg.cn/blog_migrate/462327a0d72a8b65d158baea75278aad.png" style="padding:0px;border:0px;max-width:900px;" width="546"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     当然你也可以用python2来调用。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="80" src="https://i-blog.csdnimg.cn/blog_migrate/4277437361e73f36daf8d3d5e9600b4f.png" style="padding:0px;border:0px;max-width:900px;" width="583"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     如果想调用python3，就用python3.
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="71" src="https://i-blog.csdnimg.cn/blog_migrate/dbf34fca3f9e8ac8e03f6f136bc66342.png" style="padding:0px;border:0px;max-width:900px;" width="595"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     对于Windows，就有点复杂了。因为不论python2还是python3，python可执行文件都叫python.exe，在cmd下输入python得到的版本号取决于环境变量里哪个版本的python路径更靠前，毕竟windows是按照顺序查找的。比如环境变量里的顺序是这样的：
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d27f35ce74f6ba17c67808fa115bceef.png" style="padding:0px;border:0px;max-width:900px;"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     那么cmd下的python版本就是2.7.12。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="75" src="https://i-blog.csdnimg.cn/blog_migrate/f4aae34900e2a9371f421567e01f47f0.png" style="padding:0px;border:0px;max-width:900px;" width="707"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     反之，则是python3的版本号。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     这就带来一个问题了，如果你想用python2运行一个脚本，一会你又想用python3运行另一个脚本，你怎么做？来回改环境变量显然很麻烦。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     网上很多办法比较简单粗暴，把两个python.exe改名啊，一个改成python2.exe，一个改成python3.exe。这样做固然可以，但修改可执行文件的方式，毕竟不是很好的方法。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     我仔细查找了一些python技术文档，发现另外一个我觉得比较好的解决办法。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     借用py的一个参数来调用不同版本的Python。py -2调用python2，py -3调用的是python3.
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="61" src="https://i-blog.csdnimg.cn/blog_migrate/62eab788250bec54d9924d8f9b1dd618.png" style="padding:0px;border:0px;max-width:900px;" width="653"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     <img alt="" height="62" src="https://i-blog.csdnimg.cn/blog_migrate/c1ecd9be7b1c8d186a8aee9e42acc9fd.png" style="padding:0px;border:0px;max-width:900px;" width="625"/>
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     当python脚本需要python2运行时，只需在脚本前加上，然后运行py xxx.py即可。
    </p>

    </div>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     当python脚本需要python3运行时，只需在脚本前加上，，然后运行py xxx.py即可。
    </p>
    <div class="cnblogs_code" style="margin:5px 0px;padding:5px;background-color:rgb(245,245,245);border:1px solid rgb(204,204,204);color:rgb(0,0,0);text-align:left;font-family:'Courier New';font-size:12px;">

    </div>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     就这么简单。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     同时，这也完美解决了在pip在python2和python3共存的环境下报错，提示Fatal error in launcher: Unable to create process using '"'的问题。
    </p>
    <p style="margin:10px auto;line-height:1.5;color:rgb(0,0,0);font-size:13px;font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
     当需要python2的pip时，只需
    </p>

     python2和python3的pip package就这样可以完美分开了。
