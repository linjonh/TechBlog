    <h3>
     <a id="_2">
     </a>
     前言
    </h3>
    <p>
     本期介绍一个非常常用的前端插件 Clipboard.js，Clipboard.js 实现了纯 JavaScript （无 Flash）的浏览器内容复制到系统剪贴板的功能。
    </p>
    <h3>
     <a id="1clipboardjs_5">
     </a>
     1、clipboard.js介绍
    </h3>
    <p>
     clipboard.js 现代化的拷贝文字，不依赖 flash, 不依赖其他框架，gzip 压缩后只有 3kb 大小.
    </p>
    <p>
     「拷贝文字不应当是一件困难的事. 不需要过多繁杂的配置或者下载很多脚本文件. 最重要的，它不应该依赖flash或者其他框架，应该保持简洁」
    </p>
    <p>
     「这就是创造clipboard.js的原因和目的」
    </p>
    <h3>
     <a id="2clipboardjs_12">
     </a>
     2、clipboard.js的安装
    </h3>
    <p>
     两种安装方式
     <br/>
     通过npm工具安装
    </p>
    <pre><code class="prism language-javascript">npm install clipboard <span class="token operator">--</span>save
</code></pre>
    <p>
     CDN引入
    </p>
    <pre><code class="prism language-javascript"><span class="token operator">&lt;</span>script src<span class="token operator">=</span><span class="token string">"https://cdn.jsdelivr.net/npm/clipboard@2.0.6/dist/clipboard.min.js"</span><span class="token operator">&gt;</span><span class="token operator">&lt;</span><span class="token operator">/</span>script<span class="token operator">&gt;</span>
</code></pre>
    <h3>
     <a id="3_25">
     </a>
     3、使用
    </h3>
    <p>
     先引入clipboard.js
    </p>
    <pre><code class="prism language-javascript"><span class="token operator">&lt;</span>script src<span class="token operator">=</span><span class="token string">"https://cdn.jsdelivr.net/npm/clipboard@2.0.6/dist/clipboard.min.js"</span><span class="token operator">&gt;</span><span class="token operator">&lt;</span><span class="token operator">/</span>script<span class="token operator">&gt;</span>
</code></pre>
    <p>
     设置复制剪贴的内容
     <br/>
     从另一个元素复制内容。你可以给目标元素添加一个 data-clipboard-target 属性来实现这个功能
    </p>
    <pre><code class="prism language-javascript"><span class="token operator">&lt;</span>input id<span class="token operator">=</span><span class="token string">"foo"</span> value<span class="token operator">=</span><span class="token string">"https://github.com/zenorocha/clipboard.js.git"</span><span class="token operator">&gt;</span>

    <span class="token operator">&lt;</span>img src<span class="token operator">=</span><span class="token string">"assets/clippy.svg"</span> alt<span class="token operator">=</span><span class="token string">"Copy to clipboard"</span><span class="token operator">&gt;</span>

    <p>
     你还可以定义一个 data-clipboard-action 属性来指明你想要复制还是剪切内容
    </p>
    <pre><code class="prism language-javascript"><span class="token operator">&lt;</span>textarea id<span class="token operator">=</span><span class="token string">"bar"</span><span class="token operator">&gt;</span>Mussum ipsum cacilds<span class="token operator">...</span><span class="token operator">&lt;</span><span class="token operator">/</span>textarea<span class="token operator">&gt;</span>

    <pre><code class="prism language-javascript"><span class="token operator">&lt;</span>button <span class="token keyword">class</span><span class="token operator">=</span><span class="token string">"btn"</span> data<span class="token operator">-</span>clipboard<span class="token operator">-</span>action<span class="token operator">=</span><span class="token string">"cut"</span> data<span class="token operator">-</span>clipboard<span class="token operator">-</span>target<span class="token operator">=</span><span class="token string">"#bar"</span><span class="token operator">&gt;</span>

    <p>
     复制当前的内容，需要给目标元素设置一个data-clipboard-text 属性即可

<h3>
<a id="5_86">

5、其他用法
</h3>
<p>
如果你不想修改 HTML，提供了一个非常方面的命令式的 API 给你使用。你需要做的就是声明一个函数，做一些处理，并返回一个值。
</p>
<p>
例如, 你希望动态设置 target, 你需要返回一个 node 节点.
</p>
<pre><code class="prism language-javascript"><span class="token keyword">new</span> <span class="token class-name">ClipboardJS</span><span class="token punctuation">(</span><span class="token string">'.btn'</span><span class="token punctuation">,</span> <span class="token punctuation">{<!-- --></span>

<span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
<p>
