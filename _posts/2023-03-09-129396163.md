     <p style="">
      针对目前高级辅助驾驶，本文简要描述dSPACE产品Model Desk，可以支持的测试用例。
     </p>
     <h2 style="">
      1.AEB 自动紧急刹车系统；
     </h2>
     <p style="text-indent:1.4em;">
      --Euro-NCAP AEB测试，Mode Desk的道路场景搭建满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        在准备阶段，行驶在直径为30米的圆形车道，实现测试车的轮胎进入行驶状态。
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        三种测试用例的车道长度分别设置: 1) CCRs 为304米；2）CCRm为307米；3）CCRb为208米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        车道线宽设置为8米；
       </p>
      </li>
     </ul>
     <p style="padding-left:1.4em;">
      --Euro-NCAP AEB测试，Model Desk交通流场景设置满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        城市路况 或者快速公路
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        车辆速度的配置：10-50km/h; 30-80km/h; 50-80km/h
       </p>
      </li>
     </ul>
     <h2 style="">
      2.FCW 前撞预警系统；
     </h2>
     <p style="text-indent:1.4em;">
      --NHSTA FCW测试，Mode Desk的道路场景搭建满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        测试用例1：车道长度300米；测试用例2：车道长度246米；测试用例3：车道长度314米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        以上车道宽度大于8米，均未设置车道线
       </p>
      </li>
     </ul>
     <p style="padding-left:1.4em;">
      --NHSTA FCW测试，Mode Desk的交通流场景设置满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        测速主车Ego车速：32km/h；72km/h
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        测试用例1：同一车道内，follow车位于前方静止状态；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        测试用例2：同一车道内，测试主车追随的前方follow突然执行制动操作
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        测试用例3：同一车道内，测试主车前方出现低速行驶的follow车
       </p>
      </li>
     </ul>
     <h2 style="">
      3. ACC 自适应巡航系统；
     </h2>
     <p style="text-indent:1.4em;">
      --GBT ACC测试，Model Desk的道路场景搭建满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        准备阶段道路：大于320米
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case1：圆弧道路创建，半径125米，大于60米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case2：圆弧道路创建，半径250米，大于80米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case3：圆弧道路创建，半径500米，大于120米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case4：250米以上的减速道路（包含弯道/直道）。
       </p>
      </li>
     </ul>
     <p style="padding-left:1.4em;">
      -- GBT ACC测试，Model Desk的交通流场景设置满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case1：弯道半径125米，车速设置61km/h；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case2：弯道半径250米，车速设置86km/h；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        Case3：弯道半径500米，车速设置114km/h。
       </p>
      </li>
     </ul>
     <h2 style="">
      4. LDW 车道偏离预警系统；
     </h2>
     <p style="padding-left:1.4em;text-align:left;">
      -- GBT LDW测试，Model Desk的道路场景搭建满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        直道测试：车道线长度150米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        虚警测试：车道线长度1,000米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        弯道测试：半径为250米的圆弧，长度为150米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        弯道测试：半径为500米的圆弧，长度为220米；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        车道线：虚线（白色）、实线（白色）；
       </p>
      </li>
     </ul>
     <p style="padding-left:1.4em;text-align:left;">
      -- GBT LDW测试，Model Desk的交通流场景设置满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        在直道测试时，满足以下右拐弯时测试条件，排列组合各测试4次。
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;text-align:left;">
       <p>
        测试偏离速度：
        <img class="kdocs-latex-img" src="https://i-blog.csdnimg.cn/blog_migrate/246bb2e3163a7d4bc11061ec6b85a343.png">
         = 0.1m/s ~ 0.3m/s和
         <img class="kdocs-latex-img" src="https://i-blog.csdnimg.cn/blog_migrate/246bb2e3163a7d4bc11061ec6b85a343.png">
          = 0.6m/s ~ 0.8m/s
         </img>
        </img>
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;text-align:left;">
       <p>
        测试速度偏离方向：向左偏离和向右偏离
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;text-align:left;">
       <p>
        在弯道测试时，满足以下测试条件，排列组合各测试一次：
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;text-align:left;">
       <p>
        测试偏离速度：
        <img class="kdocs-latex-img" src="https://i-blog.csdnimg.cn/blog_migrate/246bb2e3163a7d4bc11061ec6b85a343.png">
         = 0.0m/s ~ 0.4m/s和
         <img class="kdocs-latex-img" src="https://i-blog.csdnimg.cn/blog_migrate/246bb2e3163a7d4bc11061ec6b85a343.png">
          = 0.4m/s ~ 0.8m/s；
         </img>
        </img>
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;text-align:left;">
       <p>
        测试车辆转向：右转弯和左转弯；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:5.6em;list-style-type:disc;text-indent:0;text-align:left;">
       <p>
        测试速度偏离方向：向右偏离/向左偏离
       </p>
      </li>
     </ul>
     <p style="padding-left:4.2em;">
     </p>
     <p style="text-indent:1.4em;">
      --NHSTA LDW测试，Model Desk的道路场景搭建满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        车道线类型：虚线（黄色）、实线（白色）、路标（凸起）；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        单侧车道线的设置；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        直道长度：150米
       </p>
      </li>
     </ul>
     <p style="padding-left:1.4em;">
      --NHSTA LDW测试，Mode Desk的交通流场景设置满足以下：
     </p>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        纵向车速 72km/h;
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        横向车速：低速，仿真模拟驾驶员低速偏离车道；
       </p>
      </li>
     </ul>
     <ul>
      <li style="margin-left:4.2em;list-style-type:square;text-indent:0;">
       <p>
        行驶方向：左/右;
       </p>
      </li>
     </ul>
