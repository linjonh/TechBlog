     一. 点击事件
     <br/>
     1. onclick：单击事件
     <br/>
     2. ondblclick：双击事件
     <br/>
     二. 焦点事件
     <br/>
     1. onblur：失去焦点
     <br/>
     2. onfocus:元素获得焦点。
     <br/>
     三. 加载事件
     <br/>
     1. onload：一张页面或一幅图像完成加载。
     <br/>
     四. 鼠标事件
     <br/>
     1. onmousedown    鼠标按钮被按下。
     <br/>
     2. onmouseup    鼠标按键被松开。
     <br/>
     3. onmousemove    鼠标被移动。
     <br/>
     4. onmouseover    鼠标移到某元素之上。
     <br/>
     5. onmouseout    鼠标从某元素移开。
     <br/>
     五. 键盘事件
     <br/>
     1. onkeydown    某个键盘按键被按下。
     <br/>
     2. onkeyup        某个键盘按键被松开。
     <br/>
     3. onkeypress    某个键盘按键被按下并松开。
     <br/>
     六. 选择和改变
     <br/>
     1. onchange    域的内容被改变。
     <br/>
     2. onselect    文本被选中。
     <br/>
     七. 表单事件
     <br/>
     1. onsubmit    确认按钮被点击。
     <br/>
     2. onreset    重置按钮被点击。
    </h3>
