    <h3>
     题目描述
    </h3>
    <p>
     奶牛们又在玩一种无聊的数字游戏。输得很郁闷的贝茜想请你写个程序来帮她在开局时预测结果。在游戏的开始，每头牛都会得到一个数N(1&lt;=N&lt;=1,000,000)。此时奶牛们的分数均为0。如果N是奇数，那么奶牛就会把它乘以3后再加1。如果N是偶数，那么这个数就会被除以2。数字每变动一次，这头奶牛就得到1分。当N的值等于1时，游戏结束，此时的分数就是这头奶牛在这局游戏中的最终得分。
     <br/>
     以下是N=5时，一局游戏的完整过程：
    </p>
    <blockquote>
     N   操作后所得数 注释   总分
     <br/>
     5        16          3*5+1       1
     <br/>
     16         8           16/2       2
     <br/>
     8         4            8/2       3
     <br/>
     4         2            4/2       4
     <br/>
     2         1            2/2       5
    </blockquote>
    <h3>
     输入描述
    </h3>
    <p>
     第1行: 一个正整数N。
    </p>
    <h3>
     输出描述
    </h3>
    <p>
     第1行: 输出一个正整数N，即奶牛在这局游戏中的最终得分。
    </p>
    <h3>
     输入
    </h3>
    <blockquote>
     <p>
      112
     </p>
    </blockquote>
    <h3>
     输出
    </h3>
    <blockquote>
     <p>
      20
     </p>
    </blockquote>
    <h3>
     解题思路
    </h3>
    <p>
     <strong>
      题意:
     </strong>
     求变动的次数。
     <br/>
     <strong>
      思路:
     </strong>
     直接模拟。
    </p>
    <p>
     <strong>
      Accepted Code:
     </strong>
    </p>
    <pre class="has"><code class="language-cpp">#include &lt;bits/stdc++.h&gt;
using namespace std;
int main() {
    int n, ans = 0;
    scanf("%d", &amp;n);
    while (n != 1) {
        if (n &amp; 1)
            n = n * 3 + 1;
        else n &gt;&gt;= 1;
        ans++;
    }
    printf("%d\n", ans);
    return 0;
}</code></pre>
