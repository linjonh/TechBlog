     <span style="font-size:16px; color:#cc0000">
      <strong>
       <u>
        2012年4月微软MVP申请开始了！
       </u>
      </strong>
     </span>
     <br/>
     <br/>
     CSDN与微软合作，长期为用户提供申请“微软最有价值专家”的平台，希望有兴趣、资历的朋友以及正在朝这个方向努力的朋友能够积极参与。
     <br/>
     <br/>
     申请微软MVP的朋友请下载申请表格：
     <a href="http://community.csdn.net/bbs/2011/09/2012mvp.rar" rel="nofollow noopener noreferrer" target="_blank" title="http://community.csdn.net/bbs/2011/09/2012mvp.rar">
      http://community.csdn.net/bbs/2012mvp.rar
     </a>
     （点击下载）
     <br/>
     <br/>
     <span style="color:#ff00">
      填写好相关内容后把申请表格发送到邮箱admin(at)csdn.net
     </span>
    </div>
    <div class="msgfont">
     <br/>
    </div>
    <div class="msgfont">
     我们会在每次MVP申请开始时把用户提交的申请表格推荐到微软（中国）相关部门！
    </div>
    <div class="msgfont">
     <br/>
    </div>
    <div class="msgfont">
     <span style="font-family:Arial; color:#333333; font-size:14px; line-height:26px">
      申请表填写示例可参考：
      <a href="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx" rel="noopener noreferrer" style="color:rgb(51,102,153); text-decoration:none" target="_blank" title="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx">
       http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx
      </a>
     </span>
     <br/>
     <br/>
     2012年4月微软MVP将由CSDN社区进行统一推荐。
     <br/>
     <br/>
     进行Renew的MVP可以将您的申请资料发送到CSDN社区，由社区统一转交，或将您的资料直接发给项目组，并CC给社区一份。如不在CSDN社区统一范围内申请的MVP，将不再默认为CSDN推荐申请。
     <br/>
     <br/>
     为了防止邮件丢失，请所有发送完邮件的人员，访问：
     <a href="http://community.csdn.net/bbs/2011/289.htm" rel="nofollow noopener noreferrer" target="_blank" title="http://community.csdn.net/bbs/2011/289.htm">
      http://community.csdn.net/bbs/2011/289.htm
     </a>
     进行登记。
     <br/>
     <br/>
     <span style="color:#ff00">
      <strong>
       申请截止时间2012年1月12日16点。
      </strong>
     </span>
     <br/>
     <br/>
     <br/>
     <strong>
      填写注意事项
     </strong>
     ：
     <br/>
     <br/>
     1、 CSDN推荐者必须填写项为：参与第三方技术社区。请务必标明您的个人社区ID。
     <br/>
     <br/>
     2、请详细填写您的自我介绍，最好使用英文，如果没有，微软将提供人员给您翻译，但难以保证翻译质量。
     <br/>
     <br/>
     3、
     <span style="color:#ff00">
      请尽量多的填写您在过去一年中对微软技术推广做出的贡献，但是此行动必须是无偿的。
     </span>
     <br/>
     <br/>
     4、如您提交的资料不够详细或者存在问题，CSDN或微软工作人员会第一时间联络您进行资料补充。
     <br/>
     <br/>
     <br/>
     注意：所有申请成功者将根据主要技术分享形式，获得CSDN论坛或者CSDN博客颁发的[微软最有价值专家]勋章，如勋章有误，请发邮件到admin@csdn.net进行说明。

    <div class="msgfont">
     <strong>
      其他问题补充:
     </strong>
     <br/>
     <br/>
     1 我已经申请过1月的MVP，现在当任务名单未出来，我是否需要再申请？
     <br/>
     <br/>
     答：您可以等待1月MVP名单出来后，再确认是否重新申请。
     <br/>
     <br/>
     2 我是Java,Oracle的领域的专家，是否可以申请？
     <br/>
     <br/>
     答：本申请针对微软技术领域专家，但如果您对微软技术领域有贡献，亦可进行申请。
     <br/>
     <br/>
     3 申请有对职业和年龄的限制吗？
     <br/>
     <br/>
     答：年龄必须满18岁。职业无限制。但不得从事非法职业。
     <br/>
     <br/>
     4 有哪些贡献算MVP贡献
     <br/>
     <br/>
     答：a 线上： 传播微软相关技术，如：社区发帖、回复。 写作BLOG
     <br/>
     <br/>
     b 线下： 写作、 组织微软技术相关活动 、微软课程培训。
     <br/>
     <br/>
     但所有一切不能为付费行为，均为免费行为。
     <br/>
     <br/>
     5 能给一个申请表填写的示例吗？
     <br/>
     <br/>
     答：请参考：
     <a href="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx" rel="noopener noreferrer" target="_blank" title="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx">
      http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx
     </a>
     <br/>
     <br/>
     再次提醒：本次申请截止时间为：2012年1月12日16点。
