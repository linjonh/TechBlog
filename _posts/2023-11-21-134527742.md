    <p>
     <strong>
      【问题描述】
     </strong>
     <br/>
     sql server 还数据库的时候，提示失败。
     <br/>
     点击左下角进度位置，可以得到详细信息：
     <br/>
     因为数据库正在使用，所以无法获得对数据库的独占访问权。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3e515f4a9886830b02daceb050ea00fe.png"/>
    </p>
    <p>
     <strong>
      【解决方法】
     </strong>
     <br/>
     针对数据库先后执行下述语句，获得独占访问权后，重新还原即可。
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> UFDATA_001_2015 <span class="token keyword">SET</span> OFFLINE  <span class="token keyword">WITH</span> <span class="token keyword">ROLLBACK</span> IMMEDIATE

<span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> UFDATA_001_2015 <span class="token keyword">SET</span> ONLINE 
</code></pre>
