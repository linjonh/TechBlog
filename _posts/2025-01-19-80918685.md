    <p style="line-height:1.5;color:rgb(0,0,0);font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);font-size:13px;">

    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:28px;font-weight:700;background-color:rgb(255,255,255);">
      安装升级Python
     </span>
     <br/>
    </p>
    <p>
     linux自带的python是2.6版本的，用习惯了2.7.x,所以想升级python2
    </p>
    <p>
     1.下载源码包,后面的下载链接直接在python官网找的，如果想安装更高的版本自行更换
     <br/>
    </p>
    <pre><code class="language-python">wget https://www.python.org/ftp/python/2.7.10/Python-2.7.10.tgz</code></pre>
    <p>
     2.解压压缩包
     <br/>
    </p>
    <pre><code class="language-python">tar xvf Python-2.7.10.tgz</code></pre>
    <p style="text-align:left;">
     3.指定安装目录
     <br/>
    </p>
    <pre><code class="language-python">cd Python-2.7.10
./configure --prefix=/usr/local/python2</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b9ca687309800a2d903083dae90e5352.png"/>
    </p>
    <p>
     <span style="color:rgb(0,0,0);font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
      <span style="color:rgb(0,0,0);font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:18px;text-align:left;background-color:rgb(255,255,255);">
       4.安装编译
      </span>
      <br/>
     </span>
    </p>
    <pre><code class="language-python">make
make install</code></pre>
    <p>
     5.创建链接将python指向2.7版本
     <br/>
     python默认是指向2.6版本的
     <br/>
    </p>
    <p>
     <span style="color:rgb(0,0,0);font-family:Verdana, Arial, Helvetica, sans-serif;font-size:13px;text-align:left;background-color:rgb(254,254,242);">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/247d8da4285e863854f147e8267bf723.png"/>
     </span>
    </p>
    <p>
     <span style="color:rgb(0,0,0);font-family:Verdana, Arial, Helvetica, sans-serif;text-align:left;background-color:rgb(254,254,242);">
      <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
       <span style="font-size:16px;">
        删除原有的指向2.6的链接，创建新的链接指向2.7，执行python可以看到已经指向2.7版本了
       </span>
      </span>
     </span>
    </p>
    <pre><code class="language-python">rm -f /usr/bin/python
ln -s /usr/local/python2/bin/python /usr/bin/python</code></pre>
    <p>
     这个时候，千万别手贱删除原有的python2.6.因为这个时候原来的yum已经不能用了。怎么修复呢？
     <br/>
     打开/usr/bin/yum文件，制指定python版本为原来系统的python2.6
    </p>
    <pre><code class="language-python">#! /usr/bin/python 修改为 #! /usr/bin/python2.6</code></pre>
    <h2 id="安装-setuptools" style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     安装 setuptools
    </h2>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     进入
     <a href="https://pypi.python.org/pypi/setuptools" rel="nofollow" title="setuptools 下载页面">
      setuptools 下载页面
     </a>
     ，下载 setuptools 源码包。
    </p>
    <pre><code class="language-python"># wget https://pypi.python.org/packages/6f/10/5398a054e63ce97921913052fde13ebf332a3a4104c50c4d7be9c465930e/setuptools-26.1.1.zip#md5=f81d3cc109b57b715d46d971737336db
</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      下载完成之后，解压缩，然后进入压缩后的目录：
     </span>
    </p>
    <pre><code class="language-python"># unzip setuptools-26.1.1.zip
# cd setuptools-26.1.1</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      安装：
     </span>
    </p>
    <pre><code class="language-python"># python setup.py install</code></pre>
    <h2 id="安装-pip" style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     安装 pip
    </h2>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     进入
     <a href="https://pypi.python.org/pypi/pip" rel="nofollow" title="pip 下载页面">
      pip 下载页面
     </a>
     ，下载 pip 源码包。
    </p>
    <pre><code class="language-python"># wget --no-check-certificate https://pypi.python.org/packages/e7/a8/7556133689add8d1a54c0b14aeff0acb03c64707ce100ecd53934da1aa13/pip-8.1.2.tar.gz#md5=87083c0b9867963b29f7aba3613e8f4a
</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      下载完成之后，解压缩，然后进入压缩后的目录：
     </span>
    </p>
    <pre><code class="language-python"># tar -zxvf pip-8.1.2.tar.gz
# cd pip-8.1.2</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      安装：
     </span>
    </p>
    <pre><code class="language-python"># python setup.py install</code></pre>
    <p>
    </p>
    <h2 id="错误处理" style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     错误处理
    </h2>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     在安装过程中，如果提示如下错误：
    </p>
    <pre><code class="language-python">RuntimeError: Compression requires the (missing) zlib module</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      需要先安装 zlib：
     </span>
    </p>
    <pre><code class="language-python"># yum install zlib
# yum install zlib-devel</code></pre>
    <p>
     <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      进入Python文件夹，再重新编译 python2.7
     </span>
    </p>
    <pre><code class="language-python"># make clean
# make
# make install</code></pre>
    <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     然后进入 setuptools 和 pip 目录，重新安装。
    </span>
