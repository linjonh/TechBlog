    <p>
     华为消费者业务软件部总裁王成录表示：“划时代的HarmonyOS重构了人、设备、场景的关系。未来不再是产品限制用户的使用场景，而是用户所处的场景定义设备的形态。HarmonyOS不但将为用户带来更多获得应用服务的设备入口，也将带来全新的超级服务、超级应用，为我们的生活创造颠覆式体验。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/06a78f568f5504b7b0700b40c0fc78de.png#pic_center"/>
    </p>
    <h5>
     <a id="HarmonyOS20hhttpsdeveloperharmonyoscomcnhomeSDKHarmonyOS_20HarmonyOS20_DevEco_Studio_16">
     </a>
     今天上午我访问HarmonyOS2.0官网发现，手机版SDK也已经同步上线，所以就迫不及待的准备体验一下HarmonyOS 2.0手机模拟器。首先肯定要下载华为HarmonyOS2.0 官方的开发工具DevEco Studio。
    </h5>
    <h5>
     <a id="_DevEco_Studiohttpswwwcztcmscnp1988_18">
     </a>
     下载和安装教程可以参考我以前的博客：
     <a href="https://www.cztcms.cn/?p=1988" rel="nofollow">
      华为鸿蒙系统应用开发工具介绍 DevEco Studio
     </a>
     。
    </h5>
    <h5>
     <a id="2012201help_19">
     </a>
     如果以前已经安装过，那要进行软件升级，目前最新版本为
     <code>
      2.0.12.201
     </code>
     。点击软件的
     <code>
      help
     </code>
     就可以进行软件更新。
    </h5>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/0ad97748c26dcc7085b40f0f2e0b0311.png"/>
    </p>
    <p>
     软件更新完成以后就可以新建
     <code>
      HarmonyOS 2.0
     </code>
     手机应用和平板应用了。
    </p>
    <h2>
     <a id="HarmonyOS_20__23">
     </a>
     那咱就开始吧——HarmonyOS 2.0 手机版使用初体验
    </h2>
    <h3>
     <a id="_25">
     </a>
     一、新建鸿蒙应用
    </h3>
    <h4>
     <a id="1DevEco_StudioCreate_HamonyOS_Project_27">
     </a>
     1、打开DevEco Studio软件，点击
     <code>
      Create HamonyOS Project
     </code>
     。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/f04504efcbe25f4109a72221b938a0b3.png"/>
    </p>
    <h4>
     <a id="2Phone_32">
     </a>
     2、可以看到相比上一代多了手机和平板应用，选择新建
     <code>
      Phone
     </code>
     (手机应用)，点击下一步。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/8ce1c4f8e2df3f082154adb9deb0753d.png"/>
    </p>
    <h4>
     <a id="3_36">
     </a>
     3、按照下图进行配置项目即可，点击完成。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/697783772c5e5b4c1869d0c8f4f88f3f.png"/>
    </p>
    <h4>
     <a id="4gradle_40">
     </a>
     4、首次运行项目会自动下载
     <code>
      gradle
     </code>
     （项目管理器），等待下载完成。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/bb05e890be390af894160667ddb4c2d0.png"/>
    </p>
    <h4>
     <a id="510_44">
     </a>
     5、接下来就是漫长的编译过程，大概10分钟左右，一般不会报错。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/755690416f535cf283758e7e195df84a.png"/>
    </p>
    <h3>
     <a id="HarmonyOS_20_49">
     </a>
     二、揭开HarmonyOS 2.0手机版的神秘面纱
    </h3>
    <h4>
     <a id="1ToolsHVD_Manager_51">
     </a>
     1、项目编译完成以后，我们先来看看鸿蒙操作系统长什么样，点击Tools——HVD Manager。
    </h4>
    <pre><code>这个时候浏览器会弹出窗口,在弹出的窗口登录华为账号，然后点击允许即可。可以参考文章前面提到的教程。
</code></pre>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/9c8319446dc62ae59e6357242e966140.png"/>
    </p>
    <h4>
     <a id="2HarmonyOS_20_P40MatePad_Pro_59">
     </a>
     2、进入到模拟器管理界面，终于见到了安装有
     <code>
      HarmonyOS 2.0
     </code>
     的手机和平板，虚拟型号分别为
     <code>
      P40
     </code>
     和
     <code>
      MatePad Pro
     </code>
     。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/b395cbbbcf982a0e00f3e9f471680628.png"/>
    </p>
    <h4>
     <a id="3P40DevEco_Studio_63">
     </a>
     3、点击模拟器管理界面的P40手机运行按钮，DevEco Studio软件右上方自动弹出手机模拟器的运行窗口。
    </h4>
    <pre><code>鸿蒙系统手机版的上层界面用的的EMUI 11，所以界面看起来和现在的华为手机没有区别；
操作系统底层使用的是华为自主研发的HarmonyOS 2.0，手机运行速度相较于Android大大提升。
</code></pre>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/90db9d9a7a44b837554017b7e4d363b2.png"/>
    </p>
    <h4>
     <a id="4Harmony_OSBeta_1_71">
     </a>
     4、打开手机设置，点击
     <code>
      关于手机
     </code>
     ，可以看到大大的
     <code>
      Harmony OS
     </code>
     几个字，系统版本为第一个测试版本（Beta 1）。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/72c3b9db5972e2c93390f6a4c7acae77.png"/>
    </p>
    <h4>
     <a id="5_75">
     </a>
     5、查看手机状态栏。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/7877cc5705e36ac0ee22b3fb15d91d5d.png"/>
    </p>
    <h4>
     <a id="6MatePad_ProDevEco_Studio_81">
     </a>
     6、查看了手机以后，也想来看看平板又是什么样子。点击模拟器管理界面的MatePad Pro运行按钮，DevEco Studio软件右上方自动弹出平板模拟器的运行窗口。
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2b7393270554631b4a78e436909cf22a.png#pic_center"/>
    </p>
    <h4>
     <a id="7Harmony_OSBeta_1_85">
     </a>
     7、打开平板设置，点击
     <code>
      关于平板电脑
     </code>
     ，也可以看到大大的
     <code>
      Harmony OS
     </code>
     几个字，系统版本为第一个测试版本（Beta 1）。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/39962435cb311f38f0815dd67345b7ff.png"/>
    </p>
    <h3>
     <a id="_90">
     </a>
     三、项目运行，测试运行结果
    </h3>
    <h4>
     <a id="1congfigjsonlabel_92">
     </a>
     1、修改
     <code>
      congfig.json
     </code>
     文件里面的
     <code>
      label
     </code>
     标签，进行应用自定义命名（当然也可以不做任何修改，直接运行项目）
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/e21de9f295298a7f5ebeaf3f259aacd5.png"/>
    </p>
    <h4>
     <a id="2_97">
     </a>
     2、修改主界面显示的文字。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/83975c8273db128b2688b7b45c3fe65a.png"/>
    </p>
    <h4>
     <a id="3_HarmonyOS_20__101">
     </a>
     3、点击软件右上角的运行按钮， 可以看到我的项目成功的运行在了鸿蒙系统上，
     <code>
      我的第一次HarmonyOS 2.0 手机应用开发成功
     </code>
     。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/4f8dfbe87cf7b09c2058357036685d51.png"/>
    </p>
    <h4>
     <a id="4_105">
     </a>
     4、回到桌面，也可以看到我们刚刚新建的项目在手机桌面上了。
    </h4>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/blog_migrate/7091c67ee2e988b059d55712259a3e5c.png"/>
    </p>
    <h4>
     <a id="5buildhap_109">
     </a>
     5、在
     <code>
      build
     </code>
     目录下也可以找到刚刚编译生成的项目软件安装包（hap文件）。
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cd0dad108c66d1ce42246d32603b5c36.png#pic_center"/>
    </p>
    <h3>
     <a id="_114">
     </a>
     四、使用体会与总结
    </h3>
    <h5>
     <a id="HarmonyOS_20__Beta_116">
     </a>
     上午观看了HarmonyOS 2.0 手机开发者 （Beta版）的发布会，下午自己又对鸿蒙系统进行实践体验，确实非常不错，我就感觉发现了新大陆一样。希望鸿蒙系统越做越好！遗憾的是现在的鸿蒙系统手机版只能提供给开发者，商用升级可能要到明年了。
    </h5>
