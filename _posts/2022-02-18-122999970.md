    <h3>
     <a id="rtmp_6">
     </a>
     rtmp流
    </h3>
    <p>
     视音频rtmp流,服务器nginx+rtmp-module
    </p>
    <h4>
     <a id="_8">
     </a>
     推流
    </h4>
    <pre><code class="prism language-bat">ffmpeg -f dshow -i audio="麦克风 (HIK 1080P Camera-Audio)" -f dshow -i video="HIK 1080P Camera" -acodec aac -ac 2 -ar 44100 -vcodec libx264 -f flv rtmp://192.168.1.142:2022/hls/live0
</code></pre>
    <h4>
     <a id="_12">
     </a>
     播放
    </h4>
    <pre><code class="prism language-bat">ffplay rtmp://192.168.1.142:2022/hls/live0
</code></pre>
    <h3>
     <a id="rtsp_16">
     </a>
     rtsp流
    </h3>
    <p>
     视音频rtsp流,服务器EasyDarwin
    </p>
    <h4>
     <a id="_19">
     </a>
     推流
    </h4>
    <pre><code class="prism language-bat">ffmpeg -f dshow -i audio="麦克风 (HIK 1080P Camera-Audio)" -f dshow -i video="HIK 1080P Camera" -acodec aac -ac 2 -ar 44100 -vcodec libx264 -rtsp_transport tcp -f rtsp rtsp://192.168.1.136:554/test
</code></pre>
    <h4>
     <a id="_23">
     </a>
     播放
    </h4>
    <pre><code class="prism language-bat">ffplay rtsp://192.168.1.136:554/test
