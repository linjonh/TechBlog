    <h2>
     <a id="_9">
     </a>
     网络安全的分类
    </h2>
    <pre><code>网络安全：包括防火墙、入侵检测/入侵防御、VPN、防病毒、网闸、抗DDOS等产品和部署。
Web安全：也称为Web应用安全，包括Web前端安全、Web后端安全、服务器安全、数据库安全等。
无线安全：Wifi密码破解
移动安全： IOS ，Andorid
桌面安全：Windows，Linux
工控安全：研究工控设施如电力、交通、化工、制造业等行业的安全。工控领域的安全一旦出现问题，则人们的日常生活将受到直接影响。
安全云
</code></pre>

    <h2>
     <a id="_74">
     </a>
     应急响应
    </h2>
    <h3>
     <a id="_76">
     </a>
     应急响应事件类型
    </h3>
    <p>
     数据安全：数据泄露，数据损坏，数据篡改
     <br/>
     应用安全：网页篡改，sql注入，XSS攻击
     <br/>
     系统安全：口令破解，系统提权，木马病毒
     <br/>
     网路安全：DDOS攻击，DNS劫持，ARP欺骗
    </p>
    <h3>
     <a id="_82">
     </a>
     应急响应事件级别，预警等级，响应级别
    </h3>
    <p>
     事件级别：特别重大，重大，较大，一般
     <br/>
     预警级别：红区预警，橙色，黄色，蓝色
     <br/>
     响应级别：I级响应，II，III，IV
     <br/>
     注：《国家网络安全事件应急预案》
    </p>
