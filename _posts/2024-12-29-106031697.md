    <h5>
     <a id="Denied_of_Service_63">
     </a>
     拒绝服务攻击(Denied of Service)
    </h5>
    <p>
     <strong>
      也称洪水攻击，使目标网络或系统资源耗尽，使服务暂时中断或停止，导致正常用户无法访问
     </strong>
     <br/>
     例如：
    </p>
    <ul>
     <li>
      长时间尝试访问网站或任何互联网服务时被拒绝
     </li>
     <li>
      网络异常缓慢
     </li>
     <li>
      特定网站无法访问
     </li>
     <li>
      无法访问任何网站
     </li>
     <li>
      垃圾邮件的数量急剧增加
     </li>
     <li>
      无线或有线网络连接异常断开
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4c9ad0d35fdff8ca32dc7aefe838384.png"/>
      <br/>
      <strong>
       Distributed DOS，分布式拒绝服务攻击
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bc3b1684bc5b9ddd771685b1dced6fef.png"/>
      <br/>
      <strong>
       近年典型DDOS攻击：
      </strong>
     </li>
     <li>
      Github：18年, 代码托管平台Github遭受有史以来最大的DDOS攻击，一瞬间收到1.35Tbps的流量访问；
     </li>
     <li>
      暴雪：16年，Lizard Squad组织对暴雪公司服务器发起DDoS攻击，包括《星际争霸2》、《魔兽世界》、《暗黑破坏神3》等重要游戏作品离线宕机，玩家无法登陆；
     </li>
     <li>
      摄像头僵尸网络攻击：16年，美国，一珠宝店遭遇DDOS，每秒钟35000次的HTTP请求，该网站宕机。这些垃圾请求全部来源于联网的监控摄像头，25000个摄像头组成僵尸网络发起DDoS攻击，成为已知最大的CCTV(闭路电视摄像头)僵尸网络。
     </li>
    </ul>
    <h4>
     <a id="2__79">
     </a>
     2. 信息的完整性
    </h4>
    <p>
     <strong>
      通常表现为信息被欺骗、被窃取等
     </strong>
    </p>
    <ul>
     <li>
      钓鱼网站：通常发送声称来自于银行或其他知名机构的欺骗性垃圾邮件，意图引诱收信人给出敏感信息
     </li>
     <li>
      Wifi钓鱼：搭建虚假热点，DDOS原热点，引诱目标访问虚假热点并诱取重要的个人隐私信息。
     </li>
     <li>
      短信诈骗：利用手机短信骗取金钱或财务。通常由成熟的产业链和细致的内部分工，负责策划诈骗业务、专打电话、专门群发短信
     </li>
    </ul>
    <p>
     <strong>
      社会工程学——“人类硬件漏洞”
     </strong>
    </p>
    <ul>
     <li>
      通过与他人的合法交流，来使其心理受到影响，做出某些动作或者是透露一些机密信息的方式。
     </li>
     <li>
      这通常被认为是欺诈他人以收集信息、行骗和入侵计算机系统的行为。
     </li>
    </ul>
    <p>
     <strong>
      常见方式:
     </strong>
    </p>
    <ul>
     <li>
      假托
     </li>
     <li>
      调虎离山
     </li>
     <li>
      在线聊天
     </li>
     <li>
      等价交换
     </li>
     <li>
      同情心
     </li>
    </ul>
    <h4>
     <a id="3__95">
     </a>
     3. 信息的保密性
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/213a424ad3f8a212e5c0792f9bfa3219.png"/>
    </p>
    <ul>
     <li>
      14年，大量12306用户数据在互联网疯传，包括用户帐号、明文密码、身份证号码、手机号码和电子邮箱等。黑客通过收集互联网其他网站泄漏的数据，对12306进行撞库攻击，导致13万用户数据泄漏。
     </li>
     <li>
      14年，eBay网站被黑客攻击，导致用户数据泄露。近1.28亿活跃用户重新设置密码。
     </li>
     <li>
      Google约有500万账户和密码产生泄露。
     </li>
    </ul>

    <h3>
     <a id="_147">
     </a>
     近距离身份识别技术
    </h3>
    <ul>
     <li>
      <strong>
       指纹识别
      </strong>
      ：指纹是灵长类动物手指末端指腹上由凹凸的皮肤所形成的纹路，细节特征有起点、终点、结合点和分叉点。可通过比较这些细节特征的区别来进行鉴别。
     </li>
     <li>
      <strong>
       面部识别
      </strong>
      ：包括人脸图像采集、人脸定位、人脸识别预处理、身份确认以及身份查找。具有自然性和不被被测个体察觉的特点。
     </li>
     <li>
      <strong>
       虹膜识别
      </strong>
      ：是基于眼睛中的虹膜进行身份识别，应用于安防设备（如门禁等），以及有高度保密需求的场所。
     </li>
    </ul>
    <h3>
     <a id="_151">
     </a>
     远距离身份识别技术
    </h3>
    <p>
     <strong>
      步态识别
     </strong>
     ：融合计算机视觉、模式识别与视频/图像序列处理的一门技术。通过分析人的身体体型和行走姿态来识别身份，具有
     <strong>
      非接触远距离
     </strong>
     和不容易伪装的优点。
    </p>
    <ul>
     <li>
      适用范围广。可对普通2K摄像机50米外人群实时步态识别。
     </li>
     <li>
      非受控识别，无需识别对象主动配合与参与，也能识别;并且是360度全视角识别，跨穿着、跨姿态。
     </li>
     <li>
      难以伪装。
