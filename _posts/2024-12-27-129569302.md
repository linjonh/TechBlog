    <p>
     在window桌面上找到
     <strong>
      此电脑,
      <strong>
       点击
      </strong>
      鼠标右键
     </strong>
     ,找到
     <strong>
      属性
     </strong>
     并打开
     <br/>
     下滑,找到
     <strong>
      高级系统设置
     </strong>
     ,点击
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/25e467211f388498de5a49436eef2aad.png">
      <br/>
      打开以后点击下方的环境变量设置
      <br/>
      这里需要在
      <strong>
       Path
      </strong>
      中填写 Java 的命令工具的路径。不过在这之前，需要先创建环境变量
      <code>
       JAVA_HOME
      </code>
      。设置
      <code>
       JAVA_HOME
      </code>
      的值为
      <code>
       E:\Program Files\Java\jdk-17
      </code>
      <br/>
      这里选择在系统变量当中进行修改
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/94e4e4b84c07868aaebbd595d9c5a03b.png">
       <br/>
       变量名设置为
       <code>
        JAVA_HOME
       </code>
       ,路径改成我们下载java时的路径
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8f3ae336c393927c6faa1dd39cbf7259.png">
        <br/>
        在设置完
        <code>
         JAVA_HOME
        </code>
        之后，在
        <strong>
         Path
        </strong>
        中添加新的一行
        <code>
         %JAVA_HOME%\bin
        </code>
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7ca7517a21db9a316abf60fcef348796.png">
