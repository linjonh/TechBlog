    <h2>
     <a id="Ubuntu_2">
     </a>
     虚拟机与Ubuntu安装
    </h2>
    <p>
     嵌入式移植通常使用Linux操作系统的环境，使用Linux下的交叉编译工具链对BootLoader、kernel以及应用程序进行编译，然后下载运行。当然也可以通过各类IDE或者Windows下的交叉编译工具链进行。
     <br/>
     可以直接使用带有Linux操作系统的电脑或者通过虚拟机运行Linux系统，毕竟目前世面上Windows操作系统用的人更多哈~。Ubuntu是一个以桌面应用为主的Linux发行版操作系统，可以使用VMware Workstations、ESXI或其它虚拟机方便地进行搭建。这里以ESXI为例安装ubuntu-22.04.2（推荐使用VMware Workstations，ESXI可能需要额外硬件设备）。
    </p>
    <h3>
     <a id="Ubuntu_6">
     </a>
     下载Ubuntu
    </h3>
    <p>
     可以在
     <a href="https://cn.ubuntu.com/" rel="nofollow">
      Ubuntu中文网站
     </a>
     或
     <a href="https://launchpad.net/ubuntu/+cdmirrors/" rel="nofollow">
      Ubuntu镜像站
     </a>
     中寻找国内镜像站进行下载，这里从
     <a href="https://mirrors.tuna.tsinghua.edu.cn/ubuntu-releases/" rel="nofollow">
      清华大学开源软件镜像站
     </a>
     下载ubuntu-22.04.3（kernel-6.2）。
    </p>
    <h3>
     <a id="_8">
     </a>
     创建虚拟机
    </h3>
    <p>
     点击“虚拟机”，并点击左上角“创建/注册虚拟机”，在弹出的页面中选择“创建新虚拟机”，并点击下一页。
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/74e98a40e3f72fd0c5c44238f8db6f92.jpeg#pic_center">
      <br/>
      填写虚拟机名称，选择操作系统系列和版本。
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf24ec2ee38dc1559651c5c343c1ff73.jpeg#pic_center">
       <br/>
       点击“下一页”。
       <br/>
       <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/82f354492d009dbde8eecac60376d58a.jpeg#pic_center">
        <br/>
        设置合适的硬件参数，主要为CPU核数、内存容量、硬盘容量等。
        <br/>
        <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fe596560035388136bdaf985bb11b713.jpeg#pic_center">
         <br/>
         在“CD/DVD驱动器1”中选择“数据存储ISO文件”，并在弹出的界面中点击“上载”，选择下载的ubuntu-22.04.3 ISO文件上传至存储中，并点击选择。
         <br/>
         <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d9dbd901c2c07179c8995c0a1c0a045a.jpeg#pic_center">
          <br/>
          最后确认参数设置并点击完成。
          <br/>
          <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17a82c9ee626087e1525c396c09fd46b.jpeg#pic_center">
           <br/>
           在新建的虚拟机上点击“打开电源”，开始ubuntu的安装，这里不再赘述。
           <br/>
           <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/35d83457c0569e297298fb2f5a59ef36.jpeg#pic_center"/>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="_23">
     </a>
     系统配置
    </h3>
    <p>
     首先将IP地址设置为静态，方便后续远程登录，这里IP地址根据具体情况进行设置。
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f7d955a654bba85c78db5b901bffd5dc.jpeg#pic_center">
      <br/>
      可以卸载一些用不到的组件
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> remove thunderbir rhythmbox libreoffice* <span class="token punctuation">..</span>.
<span class="token function">sudo</span> <span class="token function">apt</span> autoremove
</code></pre>
    <p>
     点击ubuntu镜像源右上角问号，复制
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ec3990d2293ebdf3c7107e7403d31b20.jpeg#pic_center"/>
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5f84ec659901d4057fb5ae8632d9400.jpeg#pic_center"/>
    </p>
    <p>
     然后备份sources.list文件，并打开sources.list文件，将上述内容粘贴并保存，并安装可用更新
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">cp</span> /etc/apt/source.list /etc/apt/source.list.bak
<span class="token function">sudo</span> gedit /etc/apt/sources.list
<span class="token punctuation">..</span>.
<span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> upgrade
</code></pre>
    <p>
     安装OpenSSH，并检查服务器状态
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> openssh-server
<span class="token function">service</span> <span class="token function">ssh</span> status
</code></pre>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ebf6e814d2fe4b5e5fe058a7ca36c35.jpeg#pic_center"/>
     <br/>
     此时可以通过SSH进行远程登录（局域网内）。
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/042e244e2fb2f9fd304c8e6ac3e5d307.jpeg#pic_center"/>
    </p>
