    <h2>
     <a name="_Toc370248931" target="_blank">
      1.  语义分析
     </a>
     <span style="color:rgb(51,51,51)">
      语义搜索
     </span>
     定义
    </h2>
    <p>
     <span style="color:rgb(51,51,51)">
      顾名思义，所谓语义搜索
     </span>
     <sup>
      <span style="color:#3366CC">
       [1]
      </span>
     </sup>
     <a name="ref_[1]_3469909" target="_blank">
     </a>
     <span style="color:rgb(51,51,51)">
      ，是指搜索引擎的工作不再拘泥于用户所输入请求语句的字面本身，而是透过现象看本质，准确地捕捉到用户所输入语句后面的真正意图，并以此来进行搜索，从而更准确地向用户返回最符合其需求的搜索结果
     </span>
    </p>
    <p>
    </p>
    <p align="left">
     <span style="color:#333333">
      <a href="http://zhidao.baidu.com/search?word=%E8%AF%AD%E4%B9%89%E7%BD%91&amp;fr=qb_search_exp&amp;ie=utf8" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#2D64B3">
        语义网
       </span>
      </a>
     </span>
     <span style="color:#333333">
      的基础是一种新型搜索技术，通过这一技术，用户只需使用日常语言输入问题，就可以获得想要的答案。下一代搜索引擎的开发者认为，他们的技术能
     </span>
     <span style="color:#333333">
      “
     </span>
     <span style="color:#333333">
      理解
     </span>
     <span style="color:#333333">
      ”
     </span>
     <span style="color:#333333">
      用户所使用的语言，在这方面远远强于谷歌的技术。新型搜索引擎随后遍历一个覆盖所有互联网内容的数据库，以寻找匹配度最高的内容。通过这一方式，它们可以针对用户提出的问题，例如
     </span>
     <span style="color:#333333">
      “
     </span>
     <span style="color:#333333">
      谁是全球最好的石油公司
     </span>
     <span style="color:#333333">
      ”
     </span>
     <span style="color:#333333">
      ，或者
     </span>
     <span style="color:#333333">
      “
     </span>
     <span style="color:#333333">
      女性最喜欢哪部电影
     </span>
     <span style="color:#333333">
      ”
     </span>
     <span style="color:#333333">
      ，给出最准确的答案。
     </span>
    </p>
    <p align="left">
     <a href="http://baike.baidu.com/view/3469909.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#136EC2">
       语义搜索
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      的
     </span>
     <a href="http://baike.baidu.com/view/781374.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#136EC2">
       实质
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      是
     </span>
     <a href="http://baike.baidu.com/view/229451.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#136EC2">
       自然语言
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      处理技术
     </span>
    </p>
    <p align="left">
     <span style="color:rgb(51,51,51)">
     </span>
    </p>
    <p align="left">
     <span style="color:rgb(51,51,51)">
      搜索电脑，排在前面的网页有可能出现一篇只提到计算机却没提到电脑。因为搜索引擎通过
     </span>
     <a href="http://baike.baidu.com/view/487035.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#136EC2">
       语义分析
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      知道这两个词是紧密相关的。
     </span>
    </p>
    <h2>
     <a name="_Toc370248932" target="_blank">
      2.   知识库（
     </a>
     <span style="color:#333333">
      领域术语
     </span>
     ）在作用
    </h2>
    <p align="left">
     <span style="color:rgb(51,51,51)">
      知识库是
     </span>
     <a href="http://baike.baidu.com/view/3729189.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#136EC2">
       语义搜索引擎
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      进行推理和知识积累的基础和关键，而Ontology
     </span>
     则是知识库的基础
    </p>
    <p align="left">
     <span style="color:#333333">
      ★
     </span>
     <span style="color:#333333">
      Ontology
     </span>
     <span style="color:#333333">
      为
     </span>
     <span style="color:#333333">
      <a href="http://baike.baidu.com/view/7976.htm" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#136EC2">
        知识库
       </span>
      </a>
     </span>
     <span style="color:#333333">
      的建立提供一个基本的结构；
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      ★
     </span>
     <span style="color:#333333">
      Ontology
     </span>
     <span style="color:#333333">
      提供一套概念和术语来描述某一领域，并且获取该领域的本质的概念结构；
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
      ★
     </span>
     <span style="color:#333333">
      <a href="http://baike.baidu.com/view/7976.htm" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#136EC2">
        知识库
       </span>
      </a>
     </span>
     <span style="color:#333333">
      就运用这些术语去表达现实或者虚拟世界中的正确知识。
     </span>
    </p>
    <p align="left">
     <span style="color:rgb(51,51,51)">
      因此，建设一个知识库的第一步就是对该领域进行有效的
     </span>
     <span style="color:rgb(51,51,51)">
      Ontology
     </span>
     <span style="color:rgb(51,51,51)">
      分析。通过本体支持语义，支持人机之间的交流，从而实现机器智能，为
     </span>
     <span style="color:rgb(51,51,51)">
      web
     </span>
     <span style="color:rgb(51,51,51)">
      的发展带来了新的契机。而本体在搜索引擎中的应用，必将对搜索引擎的易用性和效率，产生极大的改进，从而使得
     </span>
     <span style="color:rgb(51,51,51)">
      web
     </span>
     <span style="color:rgb(51,51,51)">
      用户能够更好的在浩如烟海的信息海洋中遨游
     </span>
    </p>
    <p align="left">
     <span style="color:#333333">
     </span>
    </p>
    <h2>
     <a name="_Toc370248933" target="_blank">
      3.  关联搜索，相关词
     </a>
    </h2>
    <p>
     <span style="color:rgb(51,51,51)">
      Concepts
     </span>
     <span style="color:rgb(51,51,51)">
      ），比如，搜索
     </span>
     <span style="color:rgb(51,51,51)">
      iPhone
     </span>
     <span style="color:rgb(51,51,51)">
      ，搜索结果中将会显示
     </span>
     <span style="color:rgb(51,51,51)">
      iPhone
     </span>
     <span style="color:rgb(51,51,51)">
      的具体参数和属性。搜索电影名称也会得到影片有关信息
     </span>
     <span style="color:rgb(51,51,51)">
      ——
     </span>
     <span style="color:rgb(51,51,51)">
      包括预告片、评论、摘录。搜索城市名字会得到该城市相关相信
     </span>
     <span style="color:rgb(51,51,51)">
      ——
     </span>
     <span style="color:rgb(51,51,51)">
      包括当地特色、活动、天气或酒店信息。
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51)">
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51)">
      当两个词或一组词大量出现在同一个文档中时，这些词之间就可以被认为是语义相关。
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51)">
      潜在语义索引并不依赖于语言，所以
     </span>
     <span style="color:rgb(51,51,51)">
      SEO
     </span>
     <span style="color:rgb(51,51,51)">
      和搜索引擎优化虽然一个是英语，一个是中文，但这两个词大量出现在相同的网页中，虽然搜索引擎还不能知道搜索引擎优化或
     </span>
     <span style="color:rgb(51,51,51)">
      SEO
     </span>
     <span style="color:rgb(51,51,51)">
      指的是什么，但是却可以从语义上把
     </span>
     <span style="color:rgb(51,51,51)">
      ”SEO”
     </span>
     <span style="color:rgb(51,51,51)">
      ，
     </span>
     <span style="color:rgb(51,51,51)">
      ”
     </span>
     <span style="color:rgb(51,51,51)">
      搜索引擎优化
     </span>
     <span style="color:rgb(51,51,51)">
      ”
     </span>
     <span style="color:rgb(51,51,51)">
      ，
     </span>
     <span style="color:rgb(51,51,51)">
      ”search engine optimization”
     </span>
     <span style="color:rgb(51,51,51)">
      ，
     </span>
     <span style="color:rgb(51,51,51)">
      ”SEM”
     </span>
     <span style="color:rgb(51,51,51)">
      等词紧紧的连在一起。
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51)">

    <h2>
     <a name="_Toc370248937" target="_blank">
      <span style="color:#333333">
       7.
      </span>
      潜在语义索引
     </a>
    </h2>
    <p>
     语义搜索的关键之一在于怎样通过海量文献找出词汇之间的关系。目前的解决方案是，当词或词组大量出现在同一个文档中时，这些词之间就可以被认为是语义相关，这种方式被称为。潜在语义索引并不依赖于语言，如SEO和搜索引擎优化，虽然一个是英语，一个是中文，但这两个词大量出现在相同的网页中，搜索引擎不知道SEO和搜索引擎优化分别指的是什么，但是可以从语义上把SEO、搜索引擎优化、search engine optimization、SEM等词和词组联系在一起
    </p>
