    <p>
     在标准英语中，查询query意味着对信息的请求。在计算机编程中，它指的是同一件事，只是信息是从数据库中检索的。
    </p>
    <p>
     但是，编写查询需要一组预定义的代码才能使数据库理解指令。这个概念也被称为查询语言。
    </p>
    <p>
     虽然数据库管理的标准语言是
     <strong>
      结构化查询语言 (SQL)
     </strong>
     ，但其他简化数据库通信的查询语言包括
     <strong>
      AQL
     </strong>
     、
     <strong>
      Datalog
     </strong>
     和
     <strong>
      DMX
     </strong>
     。
    </p>
    <p>
     <strong>
      重要的！
     </strong>
     请注意，SQL与MySQL不同——前者是查询语言，而后者是
     <strong>
      使用
     </strong>
     该语言的软件。
    </p>
    <p>
     本文将解释查询的工作原理、查询示例，并介绍将它们写入数据库的步骤。
    </p>
    <h4 id="h-what-is-a-query">
     什么是查询？
    </h4>
    <p>
     数据库查询是对来自数据库的数据的请求。请求应该来自数据库表或使用称为查询语言的代码的表组合。这样，系统可以相应地理解和处理查询。
    </p>
    <h4 id="h-how-does-query-work">
     查询如何工作？
    </h4>
    <p>
     假设您想在咖啡店点一份美式咖啡。你提出请求时说：“我可以来一杯美式咖啡吗？”。咖啡师会理解您
     <strong>
      的
     </strong>
     要求并给您订购的物品。
    </p>
    <p>
     查询的工作方式相同——它为代码增加了意义，允许系统相应地理解和执行操作。无论是 SQL 还是任何其他查询语言，只要用户和数据库使用相同的语言，就可以交换信息。
    </p>
    <p>
     同时，设计良好的数据库将数据存储在多个表中。它们由保存数据属性的列以及信息行或记录组成。然后，查询有助于从不同的表中检索数据，排列它们，并根据命令显示它们。
    </p>
    <p>
     查询可以是选择、操作或两者的组合。选择查询可以从数据源中检索信息，而操作查询用于数据操作，例如添加、更改或删除数据。
    </p>
    <p>
     高级用户还可以使用查询命令来执行各种编程任务，从创建MySQL用户和授予权限到
     <a href="https://www.wbolt.com/change-wordpress-urls-in-mysql.html" rel="nofollow" title="更改MySQL数据库中的WordPress URL">
      更改MySQL数据库中的WordPress URL
     </a>
     。
    </p>
    <p>
     以下是一些最常见的查询命令及其功能：
    </p>
    <ul>
     <li>
      <strong>
       SELECT
      </strong>
      – 从数据库中获取数据。它是最流行的命令之一，因为每个请求都以选择查询开始。
     </li>
     <li>
      <strong>
       AND
      </strong>
      – 组合来自一个或多个表的数据。
     </li>
     <li>
      <strong>
       CREATE TABLE
      </strong>
      – 构建不同的表并指定其中每列的名称。
     </li>
     <li>
      <strong>
       ORDER BY
      </strong>
      – 按数字或字母顺序对数据结果进行排序。
     </li>
     <li>
      <strong>
       SUM
      </strong>
      – 汇总特定列中的数据。
     </li>
     <li>
      <strong>
       UPDATE
      </strong>
      – 修改表中的现有行。
     </li>
     <li>
      <strong>
       INSERT
      </strong>
      – 向现有表中添加新数据或行。
     </li>
     <li>
      <strong>
       WHERE——
      </strong>
      过滤数据并根据设定的条件获取其值。
     </li>
    </ul>
    <p>
     如需更多变体，请结合上面的一些命令。例如，将
     <strong>
      SELECT
     </strong>
     查询与
     <strong>
      AND
     </strong>
     或
     <strong>
      SUM
     </strong>
     等其他命令配对以聚合数据或组合结果。
    </p>
    <p>
     除了使用查询语言从数据库中请求信息外，其他方法包括：
    </p>
    <ul>
     <li>
      <strong>
       使用可用参数
      </strong>
      。默认情况下，数据库软件具有用户可以根据需要定义的参数列表。这些参数处理
      <a href="https://www.wbolt.com/go?_=262c06897aaHR0cHM6Ly9kb2NzLm1pY3Jvc29mdC5jb20vZW4tdXMvc3FsL3JlbGF0aW9uYWwtZGF0YWJhc2VzL3VzZXItZGVmaW5lZC1mdW5jdGlvbnMvdXNlci1kZWZpbmVkLWZ1bmN0aW9u" rel="nofollow" title="用户定义函数 (UDF)">
       用户定义函数 (UDF)
      </a>
      和
      <a href="https://www.wbolt.com/go?_=d572daab48aHR0cHM6Ly9kb2NzLm1pY3Jvc29mdC5jb20vZW4tdXMvc3FsL3JlbGF0aW9uYWwtZGF0YWJhc2VzL3N0b3JlZC1wcm9jZWR1cmVzL3N0b3JlZC1wcm9jZWR1cmVzLWRhdGFiYXNlLWVuZ2luZT92aWV3PXNxbC1zZXJ2ZXItdmVyMTU%3D" rel="nofollow" title="存储过程 (SP)">
       存储过程 (SP)
      </a>
      之间的信息交换。
     </li>
     <li>
      <strong>
       示例查询 (QBE)
      </strong>
      。关系数据库使用图形查询语言。系统将显示一个代码模板，您可以在其中编写和指定数据的字段和值。因此，用户可以填写空白区域，而不是编写完整的SQL语句。
     </li>
     <li>
      <strong>
       安装数据库插件
      </strong>
      。初学者的理想解决方案 – 插件让用户只需单击几下即可执行各种数据库任务，包括查询。此外，一些插件带有优化功能，以确保最佳性能。
     </li>
    </ul>
    <p>
     除了数据库，搜索引擎还可以查询和检索信息。但是，这两种技术中的术语查询不同。
    </p>
    <p>
     <strong>
      网络搜索查询
     </strong>
     是指用户在搜索引擎中键入的关键字，而
     <strong>
      数据库查询
     </strong>
     是请求信息的特定动作。
    </p>
    <p>
     现在您已经了解了查询的基本原理，让我们研究一下您在查询数据库时可能会遇到的几个标准术语：
    </p>
    <ul>
     <li>
      <strong>
       <a href="https://www.wbolt.com/remove-query-strings-static-resources.html" rel="nofollow" title="查询字符串">
        查询字符串
       </a>
      </strong>
      – URL的一部分，用于将请求从Web传递到数据库。
     </li>
     <li>
      <strong>
       查询参数
      </strong>
      – 附加到URL末尾的元素，用于指定对数据库的特定查询。
     </li>
     <li>
      <strong>
       查询折叠
      </strong>
      – 是指启用
      <a href="https://www.wbolt.com/go?_=5cfcc8e00caHR0cHM6Ly9kb2NzLm1pY3Jvc29mdC5jb20vZW4tdXMvcG93ZXItcXVlcnkvcG93ZXItcXVlcnktd2hhdC1pcy1wb3dlci1xdWVyeQ%3D%3D" rel="nofollow" title="Power Query">
       Power Query
      </a>
      以转换复杂计算以进行查询优化的过程。
     </li>
     <li>
      <strong>
       查询包含
      </strong>
      ——当一个查询包含在另一个查询中时发生，如果它独立于存储的数据值。
     </li>
    </ul>
    <h4 id="h-query-languages">
     查询语言
    </h4>
    <p>
     如前所述，在处理查询时选择数据库及其语言至关重要。除了
     <a href="https://www.wbolt.com/what-is-mysql.html" rel="nofollow" title="SQL">
      SQL
     </a>
     之外，还有另一种类型的数据库，称为
     <strong>
      NoSQL
     </strong>
     （Not Only Structured Query Language）。两者的主要区别在于数据结构。
    </p>
    <p>
     SQL数据库是关系型数据库，并使用需要您指定数据结构的预定义模式。另一方面，NoSQL数据库是非关系型的，并且具有用于非结构化数据的动态模式。
    </p>
    <p>
     无论如何，SQL和NoSQL都提供了适用的选项。SQL数据库是
     <a href="https://www.wbolt.com/go?_=8f27384d5baHR0cHM6Ly9tYXJpYWRiLmNvbS9yZXNvdXJjZXMvYmxvZy9hY2lkLWNvbXBsaWFuY2Utd2hhdC1pdC1tZWFucy1hbmQtd2h5LXlvdS1zaG91bGQtY2FyZS8%3D" rel="nofollow" title="符合ACID">
      符合ACID
     </a>
     的数据结构的绝佳选择。相反，如果您有非结构化文档、键值或图形，那么NoSQL数据库可能是理想的选择。
    </p>

    <h4>
     小结
    </h4>
    <p>
     查询可以是选择查询或操作查询——选择查询选择部分数据，而操作查询操作检索到的数据。
    </p>
    <p>
     查询还可以结合使用这两种操作来执行更多不同的任务，例如，查看、插入、修改或删除数据，以及计算和组合来自多个表的数据。
    </p>
    <p>
     数据库查询表明，操作数据并不一定很复杂。大多数查询语言都很直观，一旦您了解了一些基本规则，就很容易学习。对于那些不喜欢编码的人，您可以使用数据库插件或
     <strong>
      Query by example
     </strong>
     作为替代方案。
    </p>
    <p>
     我们希望这篇文章对数据库查询及其工作原理有更多的了解。
    </p>
