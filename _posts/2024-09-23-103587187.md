    <h3>
     <a id="LinuxCentOS_6_0">
     </a>
     本文基于Linux环境(CentOS 6)操作
    </h3>
    <p>
     1、默认情况下，Linux会自带安装Python，可以运行python --version命令查看，如图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/99028ced424815350195eee38d95a1b7.png">
      <br/>
      我们发现Linux中已经自带了Python2.7.5。再次运行python命令后就可以使用python命令窗口了（Ctrl+D退出python命令窗口）。
      <br/>
      2、查看Linux默认安装的Python位置
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e96b749bf498575fcd2aa879cc1e5abf.png">
       <br/>
       看到/usr/bin/python和/usr/bin/python2都是软链接，/usr/bin/python指向/usr/bin/python2，而/usr/bin/python2最终又指向/usr/bin/python2.7。所以运行python/python2/python2.7是一样的，如图：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d12944805439a93d84340c713d8c5659.png">
        <br/>
        3、安装python3
       </img>
      </img>
     </img>
    </p>
    <p>
     （1）登录
     <a href="https://www.python.org/downloads/source/" rel="nofollow">
      https://www.python.org/downloads/source/
     </a>
     ，找到对应版本（我们以Python 3.7.5为例）如图：
     <br/>
     <img alt="下载Python-3.6.5.tgz" src="https://i-blog.csdnimg.cn/blog_migrate/ced407de5dfb879caaffdd3d172d963a.png">
      <br/>
      <strong>
       我选择的是Python3.7版本，3.8版本不久前也更新出来了，看自己的选择了。
      </strong>
      <br/>
      （2）文件上传
     </img>
    </p>
    <p>
     将文件上传到Linux系统的某个目录下，根据自己情况上传，本例上传到了/root/tools目录下。
    </p>
    <p>
     （3）解压
    </p>
    <p>
     执行tar -zxvf Python-3.7.5.tgz命令，将文件解压到当前目录，如图：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/830cd56e869a940048af631903256247.png"/>
    </p>
    <p>
     （4）准备编译环境
    </p>
    <p>
     执行如下命令：
     <br/>
     <strong>
      如果python是3.7以下版本，不需要需要安装libffi-devel
     </strong>
    </p>
    <pre><code>yum -y install zlib-devel libffi-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make
</code></pre>
    <p>
     安装python需要的依赖。成功后（Complete!），如图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9413f5c1ec2c6ef7fc6848968c4904d0.png">
      <br/>
      （5）编译安装
     </img>
    </p>
    <p>
     执行cd Python-3.7.5进入解压后的Python-3.7.5目录下，依次执行如下三个命令：
    </p>
    <pre><code>./configure --prefix=/root/training/Python-3.7.5
make
make install
</code></pre>
    <p>
     **其中–prefix是Python的安装目录，安装成功后会出现Successfuly提示，并且我们会看到，同时安装了setuptools和pip工具。**进入到/root/training/Python-3.7.5安装目录，如图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e4bc3ba131d4f206880e5175391e0181.png">
      <br/>
      （6）创建软链接
     </img>
    </p>
    <p>
     还记得开始，Linux已经安装了python2.7.5，这里我们不能将它删除，如果删除，系统可能会出现问题。我们只需要按照与Python2.7.5相同的方式为Python3.7.5创建一个软链接即可，我们把软链接放到/usr/local/bin目录下(执行下面两条命令)：
    </p>
    <pre><code>ln -s /root/training/Python-3.7.5/bin/python3.7 /usr/local/bin/python3
ls -l /usr/local/bin/
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7b330a178840d14f337bd9ef84501927.png">
      <br/>
      此时，我们在命令窗口运行python3，如图：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8643ede91bc5bc62fa0506ba328b823b.png">
       <br/>
       （7）配置环境变量
      </img>
     </img>
    </p>
    <p>
     执行 vi ~/.bash_profile，打开配置文件，添加如下配置：
    </p>
    <pre><code>#配置python
export PYTHON_HOME=/root/training/Python-3.7.5
export PATH=$PYTHON_HOME/bin:$PATH
</code></pre>
    <p>
     保存退出（:wq），执行source ~/.bash_profile命令使配置生效。执行echo $PATH命令，查看是否配置成功，如图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/78b557414316987813d4138aab9c69ca.png"/>
    </p>
