     <strong>
      1.什么是 HTML
     </strong>
    </p>
    <ol>
     <li>
      HTML 是用来描述网页的一种语言。
     </li>
     <li>
      HTML 指的是超文本标记语言 (Hyper Text Markup Language)
     </li>
     <li>
      HTML 不是一种编程语言，而是一种标记语言 (markup language)

    </ol>

     注：标记语言是一套标记标签 (markup tag)，HTML 使用标记标签来描述网页

     </strong>

    <p>
     <strong>
      1.HTML5的元素
     </strong>
    </p>
    <p>
     HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。
    </p>

      <tr>

     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         &lt;p&gt;
        </code>
       </td>
       <td align="center">
        This is a web page
       </td>
       <td align="right">
        <code>
         &lt;/p&gt;
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <code>
         &lt;a href="#" &gt;
        </code>
       </td>
       <td align="center">
        This is a link
       </td>
       <td align="right">
        <code>
         &lt;/a&gt;
        </code>
       </td>
      </tr>
      <tr>
       <td>

       <td align="center">
       </td>

       </td>
      </tr>
     </tbody>
    </table>
    <p>
     HTML 元素语法
    </p>
    <ol>
     <li>
      HTML 元素以开始标签起始
     </li>
     <li>
      HTML 元素以结束标签终止
     </li>
     <li>
      元素的内容是开始标签与结束标签之间的内容
     </li>
     <li>
      某些 HTML 元素具有空内容（empty content）
     </li>
     <li>
      空元素在开始标签中进行关闭（以开始标签的结束而结束）
     </li>
     <li>
      大多数 HTML 元素可拥有属性
     </li>
    </ol>
    <p>
     嵌套的 HTML 元素
    </p>
    <p>
     大多数 HTML 元素可以嵌套（可以包含其他 HTML 元素）。
     <br/>
     HTML 文档由嵌套的 HTML 元素构成。
    </p>
    <p>
     <strong>
      1.HTML5的属性
     </strong>
    </p>
    <p>
     HTML 标签可以拥有属性。
    </p>
    <ol>
     <li>
      属性总是以名称/值对的形式出现，比如：name=”value”。
     </li>
     <li>
      属性总是在 HTML 元素的开始标签中规定。
     </li>
    </ol>
    <p>
     常用属性：
    </p>
    <ol>
     <li>
      <code>
       &lt;h1&gt;
      </code>
      ：align 对齐方式
     </li>
     <li>
      <code>
       &lt;body&gt;
      </code>
      ：bgcolor 背景颜色
     </li>
     <li>
      <code>
       &lt;a&gt;
      </code>
      ：target 规定在何处打开链接
     </li>
    </ol>
    <p>
     通用属性：
    </p>
    <p>
     属性 值 描述
     <br/>
     class ：规定元素的类名
     <br/>
     id：规定元素的唯一 ID
     <br/>
     style：规定元素的行内样式
     <br/>
     title：规定元素的额外信息
    </p>
