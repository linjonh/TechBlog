    <p>
     嵌入式OCR：
     <br/>
     嵌入式OCR技术，采用全新的灰度算法OCR识别核心，将极大的增强了具有数码摄像功能的智能手机的应用能力。随着配有高像素摄像头的手机的普及，该技术将开辟嵌入式视觉文字图像采集识别的广阔应用领域。
    </p>
    <p>
     支持平台及操作系统
     <br/>
     Symbian
     <br/>
     Windows Mobile
     <br/>
     Linux等
    </p>
    <p>
     识别率
     <br/>
     对于主要字体的识别率如下：
     <br/>
     宋体 高
     <br/>
     仿宋 高
     <br/>
     黑体 高
     <br/>
     楷体 高
     <br/>
     圆体 高
     <br/>
     英文 较高
    </p>
    <p>
     识别速度
     <br/>
     识别速的和硬件配置有很大相关性，主要依赖于目标平台的CPU速度和内存访问速度，在CPU速度相近的平台上识别性能可能有一定差别。以下是一些平台上的测试数据：
    </p>
    <p>
     DELL PocketPC 2003 (Intel PXA270 CPU，312MHz)
     <br/>
     文字识别 40 字/秒
     <br/>
     名片识别(名片大小为1,000,000像素) 8秒/张
    </p>
    <p>
     Nokia 7610 (32-bit RISC CPU based on ARM-9 series, 123 MHz)
     <br/>
     大于15字/秒ARM 9, 100MHz
    </p>
    <p>
     占用空间
     <br/>
     动态内存
     <br/>
     对于百万像素图像，占用1M字节
    </p>
    <p>
     静态内存
     <br/>
     1.7M字节 GB2312 (6763 中文) + BIG5-1 (5401 中文) + 英文 + 数字+ 符号
     <br/>
     400K字节 英文 + 数字+ 符号
     <br/>
     设像最低要求
     <br/>
     支持近焦拍摄，可以清晰拍摄文字
     <br/>
     拍摄图像中每个字的高度大于 20 左右像素
    </p>
    <p>
     完全支持名片识别
     <br/>
     支持自动及手动版面分析和名片信息分类
    </p>
    <p>
     支持多种语言
    </p>

     <li>
      简体中文 Simplified Chinese (GB2312 – 6763 characters)
     </li>
     <li>
      繁体中文 Traditional Chinese (Big5 Level1 – 5401 characters)
     </li>
