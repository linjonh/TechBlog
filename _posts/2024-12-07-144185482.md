    <p>
     问题背景：
     <br/>
    </p>
    <p>
     现在有没有简单图形编程语言？
     <br/>
     最好手机能用，用C做了一个小游戏，觉得字符难看，决定改成图形。
    </p>
    <p>
     QPython 3，看了一下实例的贪吃蛇是用字符拼的。
    </p>
    <p>
     C4driod，能用但是麻烦，主要是有很多渲染器画笔的概念，文字还要使用字库，手机不确定能不能用文件。。printf和getchar之类似乎不能用
    </p>
    <p>
     有没有类似qbasic，logo，VB之类的语言。screen，line，pset等几个命令就能画图，
    </p>
    <p>
    </p>
    <p>
     在现代的编程环境中，虽然大多数主流编程语言更加注重性能和灵活性，但仍然有一些
     <strong>
      简单易用的图形编程语言
     </strong>
     和工具，适合快速开发图形化小游戏，尤其是在手机或简单环境下使用。以下是几个推荐方案，它们的重点是
     <strong>
      简单的绘图命令（如
      <code>
       line
      </code>
      、
      <code>
       circle
      </code>
      、
      <code>
       pset
      </code>
      等）
     </strong>
     ，且适合在移动设备或轻量环境中使用：
    </p>
    <hr/>
    <h3>
     <strong>
      1. Processing / p5.js
     </strong>
    </h3>
    <h4>
     <strong>
      简介
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Processing
      </strong>
      是一种专注于图形和交互的语言，语法简单直观，适合新手学习和快速开发小游戏。
     </li>
     <li>
      <strong>
       p5.js
      </strong>
      是 Processing 的 JavaScript 实现，可以在手机浏览器上运行。
     </li>
     <li>
      <strong>
       适合人群
      </strong>
      ：需要简单绘图命令，快速上手编程。
     </li>
    </ul>
    <h4>
     <strong>
      核心命令
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       line(x1, y1, x2, y2)
      </code>
      ：画线。
     </li>
     <li>
      <code>
       rect(x, y, w, h)
      </code>
      ：画矩形。
     </li>
     <li>
      <code>
       ellipse(x, y, w, h)
      </code>
      ：画椭圆。
     </li>
     <li>
      <code>
       point(x, y)
      </code>
      ：画点。
     </li>
     <li>
      <code>
       text("string", x, y)
      </code>
      ：绘制文字。
     </li>
    </ul>
    <h4>
     <strong>
      优点
     </strong>
    </h4>
    <ul>
     <li>
      入门简单，几乎不需要复杂的设置。
     </li>
     <li>
      可视化极强，直接支持图形绘制。
     </li>
     <li>
      <strong>
       p5.js
      </strong>
      在手机浏览器中可以直接运行，无需安装额外的软件。
     </li>
    </ul>
    <h4>
     <strong>
      示例：贪吃蛇
     </strong>
    </h4>
    <p>
     以下是一个简单的贪吃蛇实现的代码片段（使用 p5.js）：
    </p>
    <pre><code>let x = 250, y = 250, size = 10;

function setup() {
  createCanvas(500, 500); // 创建画布
}

function draw() {
  background(0); // 背景颜色
  fill(255);     // 填充颜色
  rect(x, y, size, size); // 绘制蛇的方块
}

function keyPressed() {
  if (keyCode === UP_ARROW) y -= size;
  if (keyCode === DOWN_ARROW) y += size;
  if (keyCode === LEFT_ARROW) x -= size;
  if (keyCode === RIGHT_ARROW) x += size;
}
</code></pre>
    <ul>
     <li>
      <strong>
       运行方法
      </strong>
      ：
      <ol>
       <li>
        在电脑上可以直接运行在
        <a href="https://editor.p5js.org/" rel="nofollow" title="p5.js 官方编辑器">
         p5.js 官方编辑器
        </a>
        。
       </li>
       <li>
        在手机上可以用浏览器访问运行，或者通过 p5.js 的离线支持。
       </li>
      </ol>
     </li>
    </ul>
    <hr/>
    <h3>
     <strong>
      2. BASIC 语言：QB64
     </strong>
    </h3>
    <h4>
     <strong>
      简介
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       QB64
      </strong>
      是经典的 QBasic 的现代实现，支持图形化命令（如
      <code>
       LINE
      </code>
      、
      <code>
       CIRCLE
      </code>
      、
      <code>
       PSET
      </code>
      等），同时扩展了对现代硬件的支持。
     </li>
     <li>
      <strong>
       适合人群
      </strong>
      ：怀旧 QBasic 的开发者，想用类似
      <code>
       SCREEN
      </code>
      和
      <code>
       LINE
      </code>
      的命令。
     </li>
    </ul>
    <h4>
     <strong>
      核心命令
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       SCREEN
      </code>
      ：设置图形模式。
     </li>
     <li>
      <code>
       LINE
      </code>
      ：画线。
     </li>
     <li>
      <code>
       CIRCLE
      </code>
      ：画圆。
     </li>
     <li>
      <code>
       PSET
      </code>
      ：画点。
     </li>
     <li>
      <code>
       PRINT
      </code>
      ：在屏幕上显示文字。
     </li>
    </ul>
    <h4>
     <strong>
      优点
     </strong>
    </h4>
    <ul>
     <li>
      和经典的 QBasic 非常类似，学习成本低。
     </li>
     <li>
      直接支持简单的图形操作，无需复杂配置。
     </li>
     <li>
      支持文件操作、音频、以及现代 API。
     </li>
    </ul>
    <h4>
     <strong>
      示例代码：贪吃蛇
     </strong>
    </h4>
    <pre><code>SCREEN 12
x = 320
y = 240
DO
    LINE (x, y)-(x + 10, y + 10), 15, BF ' 绘制蛇身体
    a$ = INKEY$ ' 获取键盘输入
    SELECT CASE a$
        CASE "w": y = y - 10
        CASE "s": y = y + 10
        CASE "a": x = x - 10
        CASE "d": x = x + 10
    END SELECT
    SLEEP 50
LOOP
</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        运行方法
       </strong>
       ：
      </p>
      <ol>
       <li>
        安装
        <a href="https://www.qb64.dev/" rel="nofollow" title="QB64">
         QB64
        </a>
        。
       </li>
       <li>
        运行代码，窗口会直接显示图形。
       </li>
      </ol>
     </li>
     <li>
      <p>
       <strong>
        适配手机
       </strong>
       ：
       <br/>
       虽然 QB64 主要适配桌面，但可以通过一些 DOS 模拟器（如
       <strong>
        Magic DOSBox
       </strong>
       ）在手机上运行。
      </p>
     </li>
    </ul>
    <hr/>
    <h3>
     <strong>
      3. Lua + Love2D
     </strong>
    </h3>
    <h4>
     <strong>
      简介
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Lua
      </strong>
      是一种轻量级嵌入式脚本语言，结合
      <strong>
       Love2D
      </strong>
      框架，可以非常方便地进行 2D 游戏开发。
     </li>
     <li>
      <strong>
       适合人群
      </strong>
      ：希望在手机或轻量环境中快速开发小游戏。
     </li>
    </ul>
    <h4>
     <strong>
      核心命令
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       love.graphics.line(x1, y1, x2, y2)
      </code>
      ：画线。
     </li>
     <li>
      <code>
       love.graphics.rectangle(mode, x, y, w, h)
      </code>
      ：画矩形。
     </li>
     <li>
      <code>
       love.graphics.circle(mode, x, y, r)
      </code>
      ：画圆。
     </li>
     <li>
      <code>
       love.graphics.print(text, x, y)
      </code>
      ：绘制文字。
     </li>
    </ul>
    <h4>
     <strong>
      优点
     </strong>
    </h4>
    <ul>
     <li>
      体积小，运行效率高。
     </li>
     <li>
      支持跨平台（包括手机）。
     </li>
     <li>
      语法简单，适合初学者。
     </li>
    </ul>
    <h4>
     <strong>
      示例：贪吃蛇
     </strong>
    </h4>
    <pre><code>function love.load()
    x, y = 100, 100
    size = 10
end

function love.update(dt)
    if love.keyboard.isDown("up") then y = y - size end
    if love.keyboard.isDown("down") then y = y + size end
    if love.keyboard.isDown("left") then x = x - size end
    if love.keyboard.isDown("right") then x = x + size end
end

function love.draw()
    love.graphics.rectangle("fill", x, y, size, size) -- 画蛇
end
</code></pre>
    <ul>
     <li>
      <strong>
       运行方法
      </strong>
      ：
      <ol>
       <li>
        下载并安装 Love2D 引擎。
       </li>
       <li>
        将代码保存为
        <code>
         main.lua
        </code>
        ，运行 Love2D 即可。
       </li>
       <li>
        在手机上，可以使用
        <strong>
         Love2D Player
        </strong>
        应用直接运行。
       </li>
      </ol>
     </li>
    </ul>
    <hr/>
    <h3>
     <strong>
      4. Small Basic
     </strong>
    </h3>
    <h4>
     <strong>
      简介
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Microsoft Small Basic
      </strong>
      是一款专为初学者设计的简单编程语言，支持基本的图形功能。
     </li>
     <li>
      <strong>
       适合人群
      </strong>
      ：希望用极少的命令实现小游戏。
     </li>
    </ul>
    <h4>
     <strong>
      核心命令
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       GraphicsWindow.DrawLine(x1, y1, x2, y2)
      </code>
      ：画线。
     </li>
     <li>
      <code>
       GraphicsWindow.FillRectangle(x, y, width, height)
      </code>
      ：画矩形。
     </li>
     <li>
      <code>
       GraphicsWindow.DrawEllipse(x, y, width, height)
      </code>
      ：画椭圆。
     </li>
     <li>
      <code>
       GraphicsWindow.ShowMessage(text, title)
      </code>
      ：显示消息框。
     </li>
    </ul>
    <h4>
     <strong>
      优点
     </strong>
    </h4>
    <ul>
     <li>
      极简语法，易学易用。
     </li>
     <li>
      内置图形窗口，适合图形化开发。
     </li>
    </ul>
    <h4>
     <strong>
      示例：贪吃蛇
     </strong>
    </h4>
    <pre><code>x = 100
y = 100
size = 10

While ("True")
  GraphicsWindow.Clear()
  GraphicsWindow.FillRectangle(x, y, size, size)
  key = GraphicsWindow.LastKey
  If (key = "W") Then y = y - size
  If (key = "S") Then y = y + size
  If (key = "A") Then x = x - size
  If (key = "D") Then x = x + size
  Program.Delay(100)
EndWhile
</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        运行方法
       </strong>
       ：
      </p>
      <ol>
       <li>
        安装
        <a href="https://smallbasic.github.io/" rel="nofollow" title="Small Basic">
         Small Basic
        </a>
        。
       </li>
       <li>
        直接运行代码。
       </li>
      </ol>
     </li>
     <li>
      <p>
       <strong>
        适配手机
       </strong>
       ：
       <br/>
       Small Basic 主要适配桌面，暂不支持手机。
      </p>
     </li>
    </ul>
    <hr/>
    <h3>
     <strong>
      5. BASIC-256
     </strong>
    </h3>
    <h4>
     <strong>
      简介
     </strong>
    </h4>
    <ul>
     <li>
      一种面向教育的 BASIC 实现，支持图形命令和交互。
     </li>
     <li>
      <strong>
       适合人群
      </strong>
      ：希望用简单的命令快速开发图形程序。
     </li>
    </ul>
    <h4>
     <strong>
      核心命令
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       color
      </code>
      ：设置颜色。
     </li>
     <li>
      <code>
       line
      </code>
      ：绘制线条。
     </li>
     <li>
      <code>
       circle
      </code>
      ：绘制圆形。
     </li>
     <li>
      <code>
       text
      </code>
      ：绘制文字。
     </li>
    </ul>
    <h4>
     <strong>
      示例代码
     </strong>
    </h4>
    <pre><code>clg
x = 160
y = 120

do
  color red
  rect x, y, 10, 10
  key = input("")
  if key = "w" then y = y - 10
  if key = "s" then y = y + 10
  if key = "a" then x = x - 10
  if key = "d" then x = x + 10
loop
</code></pre>
    <hr/>
    <h3>
     <strong>
      总结
     </strong>
    </h3>
    <p>
     根据你的需求（简单图形命令，适合手机运行），以下是推荐：
    </p>
    <ol>
     <li>
      <strong>
       p5.js
      </strong>
      ：适合手机浏览器运行，命令直观，在线支持。
     </li>
     <li>
      <strong>
       Lua + Love2D
      </strong>
      ：轻量且跨平台，适合制作小型游戏。
     </li>
     <li>
      <strong>
       BASIC-256 或 QB64
      </strong>
      ：复古风格，经典图形命令，适合怀旧开发者。
     </li>
    </ol>
    <p>
     如果你追求快速上手和移动设备支持，推荐
     <strong>
      p5.js
     </strong>
     或
     <strong>
      Love2D
     </strong>
     。
    </p>
