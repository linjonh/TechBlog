    <p>
     之前公司运行的docker 环境是 x86 基础 上的，但是由于为兼顾国产化，要求开发对应的 ARM 版docker 镜像；但是构架机上无法连接公网，导致一些之前的包下载不了；特此记录实现的过程。
    </p>
    <p>
     由于Python 是高级语言，对底层的依赖比较少，所以采用文件复制的办法实现x86 docker 到 arm64的迁移实现；
    </p>
    <pre><code class="prism language-shell"><span class="token comment"># 首先将x86 版本的基础镜像运行起来</span>
docker run --rm -ti  python_x86:v2.7 <span class="token function">bash</span>

<span class="token comment"># 查看 site-packages 所在位置</span>
root@8ed298a1b4ba:/<span class="token comment"># python -m site</span>
sys.path <span class="token operator">=</span> <span class="token punctuation">[</span>
    <span class="token string">'/'</span>,
    <span class="token string">'/usr/local/lib/python27.zip'</span>,
    <span class="token string">'/usr/local/lib/python2.7'</span>,
    <span class="token string">'/usr/local/lib/python2.7/plat-linux2'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-tk'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-old'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-dynload'</span>,
    <span class="token string">'/usr/local/lib/python2.7/site-packages'</span>,
<span class="token punctuation">]</span>

<span class="token comment"># 将安装包拷出 ，另起一个终端 将site-packages 目录拷贝出来</span>
docker <span class="token function">cp</span>  96f7f14e99ab:/usr/local/lib/python2.7/site-packages /tmp/

</code></pre>
    <p>
     原来的安装包拷贝出来了，存在 site-packages 中。
    </p>
    <p>
     然后将文件拷贝到 运行 构建 arm docker 镜像的机器上，将文件夹进行替换；
    </p>
    <pre><code class="prism language-shell">docker run --rm -ti  python_arm64:v2.7 <span class="token function">bash</span>

<span class="token comment"># 查看 site-packages 所在位置</span>
root@8ed298a1b4ba:/<span class="token comment"># python -m site</span>
sys.path <span class="token operator">=</span> <span class="token punctuation">[</span>
    <span class="token string">'/'</span>,
    <span class="token string">'/usr/local/lib/python27.zip'</span>,
    <span class="token string">'/usr/local/lib/python2.7'</span>,
    <span class="token string">'/usr/local/lib/python2.7/plat-linux2'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-tk'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-old'</span>,
    <span class="token string">'/usr/local/lib/python2.7/lib-dynload'</span>,
    <span class="token string">'/usr/local/lib/python2.7/site-packages'</span>,
<span class="token punctuation">]</span>

<span class="token comment"># 清理 site-packages 目录</span>
<span class="token function">rm</span> -rf /usr/local/lib/python2.7/site-packages

<span class="token comment"># 将原来文件包，拷贝进去</span>
docker <span class="token function">cp</span> /tmp/site-packages  96f7f14e99ab:/usr/local/lib/python2.7/

<span class="token comment"># docker commit </span>
docker commit 96f7f14e99ab python_arm64_new:v2.7

</code></pre>
