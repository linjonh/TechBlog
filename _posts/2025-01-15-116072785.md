     大家好！我是只谈技术不剪发的 Tony 老师。
    </p>
    <p>
     今天给大家介绍一款免费开源的智能数据库管理工具：
     <a href="https://dbgate.org/" rel="nofollow">
      DbGate
     </a>
     。
    </p>
    <p>
     如果觉得文章有用，欢迎评论📝、点赞👍、推荐🎁
    </p>
    <h3>
     <a id="_11">
     </a>
     功能特性
    </h3>
    <p>
     DbGate 是一个免费开源的软件，基于宽松的 MIT 协议分发，源码托管在
     <a href="https://github.com/dbgate/dbgate">
      GitHub
     </a>
     。
    </p>
    <p>
     DbGate 提供了多种数据库支持，目前包括 MySQL、PostgreSQL、SQL Server 以及 MongoDB。
    </p>
    <p>
     DbGate 支持不同的平台，包括 Windows、Linux、MacOS 以及 Web 浏览器。
    </p>
    <p>
     如果想要了解 DbGate，但是又没有安装 DbGate，可以直接使用官方提供的一个
     <a href="https://demo.dbgate.org/" rel="nofollow">
      在线演示应用
     </a>
     。
    </p>
    <h4>
     <a id="_20">
     </a>
     数据库连接
    </h4>
    <p>
     DbGate 支持同时连接多个不同的数据库，支持通过 SSH tunnel 连接数据库以及 SSL 连接。DbGate 提供了数据库对象结构浏览功能以及表、视图、存储过程和函数的查看功能。当用户打开多个窗口选项卡时，每个选项卡都会显示所属的数据库。
    </p>
    <p>
     <img alt="连接" src="https://i-blog.csdnimg.cn/blog_migrate/e349935d6f448e50ca77b989294b15eb.png#pic_center"/>
    </p>
    <h4>
     <a id="_25">
     </a>
     数据编辑器
    </h4>
    <p>
     DbGate 提供了强大的数据浏览器和编辑器，可以快速查找和修改数据。用户可以直接输入过滤表达式（例如字段值包含的字符子串），或者通过菜单过滤数据。
    </p>
    <p>
     <img alt="filter" src="https://i-blog.csdnimg.cn/blog_migrate/abb5466f6b4b664145e7a0e9e6bb08d4.png#pic_center">
      <br/>
      有时候我们想要查看外键关联中被引用字段的数据，DbGate 提供了查看相关数据的便捷方法，不需要编写额外的查询语句。用户甚至可以基于被引用的字段进行数据过滤。
     </img>
    </p>
    <p>
     <img alt="fk" src="https://i-blog.csdnimg.cn/blog_migrate/c4c7552e5218dc15afe81e1b123fc184.png#pic_center"/>
    </p>
    <p>
     DbGate 数据浏览器还提供了表单视图（Form view），可以一次查看一行数据。表单视图同样支持外键数据的查看。
    </p>
    <p>
     <img alt="form" src="https://i-blog.csdnimg.cn/blog_migrate/028c5436c19e11a67759a1252625fc9f.png#pic_center"/>
    </p>
    <p>
     DbGate 数据编辑器提供了类似于 Excel 的数据编辑功能，用户可以编辑数据并预览相应的 SQL 语句，然后使用 Ctrl+S 或者菜单命令保存数据。
    </p>
    <p>
     <img alt="edit" src="https://i-blog.csdnimg.cn/blog_migrate/b74e8149eda95bf34ae3b46b1a4d77cc.png#pic_center">
      <br/>
      除此之外，DbGate 还提供了许多数据令人惊喜的查看/编辑功能：
     </img>
    </p>
    <ul>
     <li>
      基于外键的主视图/详细视图；
     </li>
     <li>
      外键查找；
     </li>
     <li>
      一键分组统计功能；
     </li>
     <li>
      主视图/详细视图分组以及按照年、月、日分组；
     </li>
     <li>
      自由格式编辑器，以 JSON 文件编辑数据，不需要使用数据库；
     </li>
     <li>
      批量操作（被称为宏命令 macros）：
      <ul>
       <li>
        大小写转换；
       </li>
       <li>
        删除变音符号；
       </li>
       <li>
        当前数据；
       </li>
       <li>
        生成 UUID。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="_SQL__53">
     </a>
     查询编辑器、设计器以及 SQL 生成器
    </h4>
    <p>
     DbGate 提供了语法高亮、自动补全、连接查询向导（方便添加连接语句）、SQL 格式化、保存查询语句以及生成创建数据库结构/数据的 SQL 脚本等功能。
    </p>
    <p>
     <img alt="join" src="https://i-blog.csdnimg.cn/blog_migrate/d7d3e104174c715029ad7d1efaf967d3.png#pic_center">
      <br/>
      查询设计器提供了图形方式设计查询语句的功能，支持各种连接查询、分组聚合、数据过滤以及排序操作。
     </img>
    </p>
    <p>
     <img alt="designer" src="https://i-blog.csdnimg.cn/blog_migrate/8b54c5d89459d08c2dead8321e849041.png#pic_center">
      <br/>
      SQL 生成器可以用于创建数据库对象的脚本，支持表（包含外键）的创建、删除、数据插入以及截断脚本，支持视图、存储过程/函数的创建和删除脚本。
     </img>
    </p>
    <p>
     <img alt="sql" src="https://i-blog.csdnimg.cn/blog_migrate/4326e1f4867d1c73963acfd59c610777.png#pic_center"/>
    </p>
    <h4>
     <a id="_63">
     </a>
     导入与导出
    </h4>
    <p>
     数据的导入与导出支持 CSV、Excel、JSON 文件以及其他格式（通过插件进行扩展），可以实现不同数据库之间的数据复制。
    </p>
    <p>
     <img alt="export" src="https://i-blog.csdnimg.cn/blog_migrate/daaf0e60922156bf68f0474363b4ceeb.png#pic_center">
      <br/>
      导入文件之前支持数据预览。
     </img>
    </p>
    <p>
     <img alt="预览" src="https://i-blog.csdnimg.cn/blog_migrate/90db9d833b73f7229f01012f4d5e62ff.png#pic_center"/>
    </p>
    <p>
     支持使用 JavaScript 编写数据导入导出脚本。
    </p>
    <p>
     <img alt="js" src="https://i-blog.csdnimg.cn/blog_migrate/f15ebdb62eb62ad835b14a2d5332c266.png#pic_center"/>
    </p>
    <h4>
     <a id="NoSQL__75">
     </a>
     NoSQL 数据库支持
    </h4>
    <p>
     DbGate 提供了 MongoDB 数据库支持，它使用原生的 MongoDB 驱动，可以直接连接到数据库。
    </p>
    <p>
     DbGate 可以查看集合数据，支持表格视图和 JSON 视图，支持集合过滤的排序。
    </p>
    <p>
     <img alt="view" src="https://i-blog.csdnimg.cn/blog_migrate/5ede75fc720d4b9ffc64cee0353caf14.png#pic_center"/>
     <br/>
     用户可以使用 nodejs MongoDB API 运行 MongoDB 查询。
    </p>
    <p>
     <img alt="query" src="https://i-blog.csdnimg.cn/blog_migrate/474309369cd1a472e5548a8276cacb23.png#pic_center"/>
     <br/>
     数据编辑器提供了 MongoDB 脚本预览功能。
    </p>
    <p>
     <img alt="edit" src="https://i-blog.csdnimg.cn/blog_migrate/66ed26fea8d51f3b890ec867ec736cf7.png#pic_center"/>
     <br/>
     其他的功能包括数据导入导出、生成创建、删除、查找集合的脚本等。
    </p>
    <h4>
     <a id="_89">
     </a>
     扩展插件
    </h4>
    <p>
     DbGate 使用了可扩展的插件结构，通过
     <a href="https://www.npmjs.com/search?q=keywords:dbgateplugin" rel="nofollow">
      NPM 程序包
     </a>
     的方式提供插件，包括导入导出格式插件和数据库连接驱动插件。
    </p>
    <p>
     <img alt="插件" src="https://i-blog.csdnimg.cn/blog_migrate/007116fd0f919556f306c540af580a47.png#pic_center"/>
    </p>
    <p>
     用户也可以使用
     <a href="https://www.npmjs.com/package/generator-dbgate" rel="nofollow">
      yeoman 模板
     </a>
     创建自己的插件。
    </p>
    <h4>
     <a id="_96">
     </a>
     其他功能
    </h4>
    <p>
     DbGate 提供的其他功能包括：
    </p>
    <ul>
     <li>
      归档目录，可以将数据保存到本地。归档数据采用 JSON 格式存储。
     </li>
     <li>
      收藏对象，包括表（包含数据过滤以及主视图/详细视图），查询语句以及图表等。
     </li>
     <li>
      最近关闭选项卡，一周之内的未保存查询语句可以从这里查看。
     </li>
     <li>
      可配置的快捷键。
     </li>
     <li>
      深色主题。
     </li>
    </ul>
    <p>
     <img alt="收藏" src="https://i-blog.csdnimg.cn/blog_migrate/42270807b789b99b0d7c539de5f1b590.png#pic_center"/>
    </p>
    <p>
     另外，“保存文件”功能可以将文件保存到 DbGate 内部存储或者磁盘中，包括 SQL 查询、查询设计、JavaScript 数据脚本、Markdown 页面以及图片。
    </p>
    <p>
     图表功能可以将查询结果或者表格数据进行可视化展示，支持条形图、折线图、饼图、极区图以及自定义的图表颜色。
    </p>
    <p>
     <img alt="chart" src="https://i-blog.csdnimg.cn/blog_migrate/f3abb2453c5896a321cdd63844d3c360.png#pic_center"/>
    </p>
    <h3>
     <a id="_113">
     </a>
     下载安装
    </h3>
    <p>
     DbGate 提供了 Windows、Linux、MacOS 以及 Docker 等平台的安装文件，可以直接在
     <a href="https://dbgate.org/download/" rel="nofollow">
      官网
     </a>
     或者
     <a href="https://github.com/dbgate/dbgate/releases/latest/">
      GitHub
     </a>
     下载最新版本。
    </p>
