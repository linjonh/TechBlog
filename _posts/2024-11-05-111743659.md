    <h2>
     <a id="_13">
     </a>
     多选题
    </h2>
    <ol start="3">
     <li>
      在单处理机中，下面并行和并发现象哪些可能发生？
      <br/>
      <mark>
       A. 用户程序与用户程序之间的并发
      </mark>
      <br/>
      <mark>
       B. 设备与设备之间的并行
      </mark>
      <br/>
      <mark>
       C. 处理机与设备之间的并行
      </mark>
      <br/>
      D. 用户程序与用户程序之间的并行
     </li>
    </ol>
    <blockquote>
     <p>
      并发：【只是在宏观上同时】
      <br/>
      ①用户程序与用户程序之间
      <br/>
      ②用户程序与系统程序之间
      <br/>
      并行：【要求微观上时间的一点上同时】
      <br/>
      ①程序与设备之间
      <br/>
      ②设备与设备之间
      <br/>
      ③处理计与设备之间
     </p>
    </blockquote>

    <h2>
     <a id="_31">
     </a>
     主观题
    </h2>
    <ol start="5">
     <li>
      Windows、Mac OS、UNIX、Linux和Android等操作系统的特点是什么，分别适合在什么样的硬件环境中使用？
     </li>
    </ol>
    <blockquote>
     <p>
      请自行凝练语言，以Windows为例
      <br/>
      Windows主要特点有：
      <br/>
      1、起初是在MS-DOS之下的桌面环境，现有不同位数的多个版本；
      <br/>
      2、通过协作式多任务允许用户同时执行多个图形应用；
      <br/>
      3、基于内存分段的软件虚拟内存方案，可运行大于物理内存的应用（代码段和资源可在内存不足时交换）
      <br/>
      Windows适合于人电脑(PC)、移动设备、服务器（Server版）和嵌入式设备等。
     </p>
    </blockquote>
