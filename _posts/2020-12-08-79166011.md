     <p>
      题目描述
      <br/>
      输入一个整数数组，判断该数组是不是某二叉搜索树的后序遍历的结果。如果是则输出Yes,否则输出No。假设输入的数组的任意两个数字都互不相同。
     </p>
    </blockquote>
    <ul>
     <li>
      算法：后序遍历
     </li>
     <li>
      数据结构：二叉搜索树
     </li>
     <li>
      编程语言：C++
     </li>
    </ul>

    <span class="hljs-keyword">bool</span> VerifySquenceOfBST(<span class="hljs-stl_container"><span class="hljs-built_in">vector</span>&lt;<span class="hljs-keyword">int</span>&gt;</span> sequence) {
        <span class="hljs-keyword">if</span>(sequence.empty())
            <span class="hljs-keyword">return</span> <span class="hljs-keyword">false</span>;
        <span class="hljs-keyword">int</span> length=sequence.size();
        <span class="hljs-comment">//获取后续遍历的最后一个数，是二叉搜索树的最后一个元素</span>
        <span class="hljs-keyword">int</span> root=sequence[length-<span class="hljs-number">1</span>];

        <span class="hljs-keyword">for</span>(;i&lt;length-<span class="hljs-number">1</span>;i++)
        {
            <span class="hljs-comment">//左子树元素小于根节点</span>
            <span class="hljs-keyword">if</span>(sequence[i]&gt;root)
                <span class="hljs-keyword">break</span>;
            left.push_back(sequence[i]);
        }
        <span class="hljs-keyword">int</span> j=i;
        <span class="hljs-keyword">for</span>(;j&lt;length-<span class="hljs-number">1</span>;j++)
        {
            <span class="hljs-keyword">if</span>(sequence[j]&lt;root)

    }
};</code></pre>
