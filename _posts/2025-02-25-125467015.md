    <p style="margin-left:0;text-align:center;">
     <strong>
      第一部分          破冰游戏
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      1
     </strong>
     <strong>
      、对对碰
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     游戏类型:  破冰船                   参加人数:  不限              游戏时间:  5分钟或更长
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     所需材料:  无                          场地要求:  宽敞的会议室或户外进行
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     活动目的
    </p>
    <ol>
     <li style="text-align:justify;">
      活动筋骨，可替代早操，让学员在良好精神状态下上课。
     </li>
     <li style="text-align:justify;">
      享受放松和乐趣。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     操作程序
    </p>
    <ol>
     <li style="text-align:justify;">
      讲师让所有的学员围成一个大圆圈，讲师站在圆圈中心。
     </li>
     <li style="text-align:justify;">
      相邻的2个学员是1对拍档，拍档要面对面站立。
     </li>
     <li style="text-align:justify;">
      讲师站在圆圈中向大家说：这个活动会进行很多次，但每次圆圈中间会有1个人，这个人现在是我，待会儿可能是其它人。圆圈中的人发布命令，所有的命令都是让拍档们互相接触彼此身体的某个部位。听到命令后，拍档们必须去按要求尽快完成，同时嘴里还要重复命令内容。比如：命令是 “膝盖对膝盖”，每个人都要与你拍档的膝盖相碰，同时口中喊：“膝盖对膝盖”；当圈中人的命令是“眼睛对耳朵”的时候，你们就用一个人的眼睛接触另一个人的耳朵，边做边用语言重复：“眼睛对耳朵”……。
     </li>
     <li style="text-align:justify;">
      只有当圈中人的命令是：“人对人”的时候，圈中发命令的人可以下去抢一个伙伴，而其他所有的排挡必须迅速更换新伙伴，最终落单没有抢到伙伴的人将站到中间去成为发布命令的人，遵循相同的规则发布命令，直到获得新伙伴，另一个人成为新的发布命令的人。
     </li>
     <li style="text-align:justify;">
      所有被使用过的部位不能重复再次使用。
     </li>
     <li style="text-align:justify;">
      这时，讲师看学员的人数，如果是学员人数刚好是偶数，讲师就与学员一起参与这个游戏。若学员人数刚好是奇数，讲师示范完毕后，让学员推举出第一个发布命令的人，宣布开始游戏，而自己只在旁边观察。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     编者提示
    </p>
    <ol>
     <li style="text-align:justify;">
      这是一个纯粹的破冰船游戏，简单易行，而且气氛非常热烈。
     </li>
     <li style="text-align:justify;">
      如有必要，讲师可规定不能使用一些人体特别敏感的部位。
     </li>
     <li style="text-align:justify;">
      学员参与人数最好不要少与10人。
     </li>
    </ol>

     <strong>
      2
     </strong>
     <strong>
      、大风吹
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     游戏类型:  破冰船                    参加人数:  不限                      游戏时间:  5 分钟或更长
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     所需材料:  无                            场地要求:  宽敞的会议室或户外进行
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>

    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     活动目的
    </p>
    <ol>
     <li style="text-align:justify;">
      活动筋骨，可替代早操，让学员在良好精神状态下上课。
     </li>
     <li style="text-align:justify;">
      享受放松和乐趣。
     </li>
     <li style="text-align:justify;">
      活跃气氛，让学员换位置形成新的学习小组，但顺理成章，生动有趣。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     操作程序
    </p>
    <ol>
     <li style="text-align:justify;">
      每个学员都坐在自己的凳子上，拿掉多余的凳子，使凳子与学员的人数相等。
     </li>
     <li style="text-align:justify;">
      讲师站在讲台上宣布：现在我们要做一个游戏：当我说“大风吹”的时候，你们所有人齐声问：“吹什么？”“我回答：“吹鞋子”时，所有穿鞋子的人都要站起来，重新找凳子换位子，动作慢的那个人就可能没有凳子了，因为我说完之后也会加入这个游戏。如果我回答：“吹牙齿”，所有有牙齿的人就要重新换位子找凳子。以此类推，当我回答：“吹眼镜”时，所有带眼镜的人换位子，其它人做在原来的座位上不动。明白了吗？最后一个动作慢没有重新找到凳子的人，就要像我这样站到前面来叫口令，直到他抢到位置，产生新的叫口令的人。
     </li>
     <li style="text-align:justify;">
      此活动可持续3-4分钟，就按照最后一轮学员坐的位置开始继续培训课程。
     </li>
     <li style="text-align:justify;">
      一个完整的口令过程如下：
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
     叫口令者：“大风吹。”
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     学员：“吹什么？”
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     叫口令者：“吹衣服”。（物品由叫口令者自行决定。）
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     学员开始换位子。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     另一论开始：“大风吹”……
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     编者提示
    </p>
    <ol>
     <li style="text-align:justify;">
      是一个纯粹的破冰船游戏，简单易行，而且效果很好。
     </li>
     <li style="text-align:justify;">
      游戏开始前可能学员已经被分好小组了。
     </li>
     <li style="text-align:justify;">
      此游戏可应用在课程开始或任何讲师想调整小组成员并让大家放松的时候。
     </li>
    </ol>

    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     游戏时间:  5分钟或更长          所需材料:  现金或礼品           场地要求:  会议室

    <p style="margin-left:.0001pt;text-align:left;">
     活动目的
    </p>
    <ol>
     <li style="text-align:left;">
      鼓励新加入者及老相识之间彼此认识熟悉，使学员们融为一体。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     操作程序
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     在大型会议或聚会中，新成员往往会被冷落在一旁，较难与其他人认识并熟悉起来，原已形成的圈子很难被打破。还有一些成员会觉得完全被排斥在团队之外，而不是团队的一员。
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     为鼓励所有参与者友好对待其他各位学员，讲师可事先秘密地在学员中指定一位学员充当“神秘先生”或“神秘小姐”。培训或会议开始后，讲师向大家宣布要开展一项有趣的活动，“你们当中有一位神秘先生/小姐，如果你和他/她握手，他/她就会给你一元钱。”
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      相关讨论
     </strong>
    </p>
    <ol>
     <li style="text-align:left;">
      为什么我们总是不太愿意结识新朋友？
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      可能答案
     </strong>
     ：怕羞；习惯；本身不善交际……
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      总　　结
     </strong>
     ：每一次与陌生人交往都是一次我们推销自己并向对方学习的机会，同时对我们也是一个挑战，因为我们要更主动积极。
    </p>
    <ol>
     <li style="text-align:left;">
      假设的金钱奖励对你的行为有什么样的影响？
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:left;">

      可能答案
     </strong>
     ：可以促使我们结识更多人，但只局限于肤浅的谈话。

    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      总
     </strong>
     <strong>
      结：
     </strong>
     我们需要从内心里热爱做一件事，才会成功。
    </p>
    <ol>
     <li style="text-align:left;">
      我们可以使用哪些开场白来克服自己的沉默寡言？
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      可能答案

    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <strong>
      引导方向
     </strong>
     ：事实上每一次新的相遇都是一次我们推销自己并向对方学习的机会和对自我的挑战，因此我们要从内心改变自己，使自己喜欢与他人交往，这样我们就会增加更多的朋友，结识更多的人。

     <li style="text-align:left;">

     活动目的
    </p>
    <ol>
     <li style="text-align:justify;">
      帮助讲师和学员熟悉所有参加者的姓名，让整个课堂生动活泼起来。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     操作程序
    </p>
    <ol>
     <li style="text-align:justify;">
      在第一次登记时收集每位学员的照片，以供培训教程之用。
     </li>
     <li style="text-align:justify;">
      可在培训正式开始前召开一个学员介绍会，这活动对连续3天或5天的培训课程非常有用。
     </li>
     <li style="text-align:justify;">
      用一台拍立得照相机为每位学员拍照，并在相片的空白处或背面写上学员的名字。
     </li>
     <li style="text-align:justify;">
      在培训的第一堂课开始前，回顾全套照片。这有助于培训师快速记忆每位学员的相貌及名字，对于那些不能设置桌牌的课堂尤为有效。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      其它可选操作程序
     </strong>
    </p>
    <ol>
     <li style="text-align:justify;">
      将全套学员照片粘贴在一张大的张贴板上，在每张相片旁写上该学员的姓名及相关信息（如：公司，职位，头衔等）。将张贴板放在进口处或咖啡桌旁，以方便学员观看。
     </li>
     <li style="text-align:justify;">
      在第一堂课之前，雇一位当地擅长画漫画的人士，请他为每一位学员（及培训人员）画一张幽默的速写，并将这些画在合适的位置张贴起  来。由于这些画往往抓住人物的面部特征或性格特点，所以能帮助大家区分并记忆每位成员的相貌和名字。在课程结束后，也可用来作为特殊的“毕业”礼品赠送给学员。
     </li>
     <li style="text-align:justify;">
      如果你能够在培训开始前获得学员的姓名和照片，就可以在培训前把相片贴到张贴板上去，只是让学员在自己的照片下签名就行了。
     </li>
    </ol>

     </strong>
     <strong>
      、寻宝游戏
     </strong>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     游戏类型：破冰类及团队合作精神          参加人数：不限，5人一组为最佳
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     游戏时间：10分钟                所需材料：寻宝游戏工作表              场地要求：会议室
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>

    <p style="margin-left:.0001pt;text-align:left;">
     活动目的
    </p>
    <ol>
     <li style="text-align:left;">
      可利用此游戏来破冰，改善课堂气氛。
     </li>
     <li style="text-align:left;">
      也可以借此游戏让学员体会一下团队合作精神。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     操作程序
    </p>
    <ol>
     <li style="text-align:left;">
      培训师让全体学员组成5人一小组，每组选出一位代表作为组长。
     </li>

    </p>

     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     操作程序
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     在一些研讨会或项目小组中，成员可能来自不同的地方，相互之间并不了解，或者不清楚他们即将面临什么，互相之间的角色是怎样的等等。同时，讲师也不知道他们对此次活动有什么期望和要求，因此，可以做下列活动以加深相互的了解：
    </p>
    <ol>
     <li style="text-align:justify;">
      将成员分成4-6人的小组，在简报纸上写下他们各自的顾虑及对课程的期望。时间5分钟。
     </li>
     <li style="text-align:justify;">
      5分钟结束后，请各组中的一名成员代表发言，说明小组的情况。
     </li>
     <li style="text-align:justify;">
      在所有的小组发言都结束后，讲师总结归纳学员所提供的顾虑和期望的情况，略作解释和答复。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      其它可选择的操作程序
     </strong>
    </p>
    <ol>
     <li style="text-align:justify;">
      这个活动的操作方式也可改为由讲师提问学员，而后将每位学员的答案写到简报架上，如果学员的回答过于冗长，讲师可以适当将其归纳和概括成简短的一两句话。
     </li>
     <li style="text-align:justify;">
      课程结束后，讲师和学员一起回顾这些期望，看看这次培训是否达到了学员的期望。
     </li>
    </ol>

    </p>
    <ol>
     <li style="text-align:justify;">
      请一位自愿者协助你。
     </li>
     <li style="text-align:justify;">
      告诉大家你能神奇迅速的预知一道算术题的结果。
     </li>
     <li style="text-align:justify;">
      你走到一个看不见志愿者写题的地方。
     </li>
     <li style="text-align:justify;">
      让志愿者在黑板上任意写下一个三位数的数字（注意：该数字的个位和百位绝不能相同，如不能是：323，555，414）。
     </li>
     <li style="text-align:justify;">
      让志愿者将这个数字中的个位和百位数字互换，而十位上的数字保持不变，从而得到一个新的三位数（如：678变成876，234变成432），然后将原来的三位数和互换后的三位数进行比较，用大的三位数减去小的三位数得出两者之差，如821－128＝693。
     </li>
     <li style="text-align:justify;">
      再把得出的差（一个新三位数）的个位和百位再做最后一次互换，得到另一新数字；将此数字与上次的差进行相加得出最终结果：1089。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
     （如：差为693，交换个、百后位得出369，693+396＝1089）
    </p>
    <ol>
     <li style="text-align:justify;">
      当志愿者计算完毕后，讲师举起事先准备好写有1089的数字卡。（大多数情况下，学员们会对此举感到十分惊奇。）
     </li>
     <li style="text-align:justify;">
      如果碰到第一次相减得到一个两位数的时候，只要在它前面加零即可，和不变，依旧为“1089”。如786－687＝99，099＋990＝1089。
     </li>
    </ol>

    </p>
    <ol>
     <li style="text-align:justify;">
      讲师告诉学员要进行一个游戏，这个游戏和人类进化史有相关之处。但与我们以前在书本上学到的知识不同的是，此次进化的全过程是：蛋-鸡-凤凰-人。所有的人都必须用尽量短的时间由蛋进化成人。
     </li>

    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      具体过程：
     </strong>
     全体人员先蹲下扮演鸡蛋，而后相互找同伴进行猜拳，扮演蛋的人走路的时候必须保持蹲的姿势。赢者进化为小鸡，小鸡可以站起来一些，但不能完全直立而必须弯着腿和腰，走路的时候要跳着走。小鸡再找小鸡猜拳，赢者进化为凤凰，输者退化为蛋。凤凰可以站的更高一些，但还是不能完全直立，凤凰的腰是弯的，可以走路。凤凰与凤凰猜拳，输者退化为鸡，赢者进化为人。人可以直立行走。人与人继续猜拳，输者化为凤凰，赢者继续为人。
    </p>

     </strong>
     只有同级的人才可以猜拳，即蛋只能与蛋，鸡只能与鸡，凤凰只能与凤凰，人只能与人猜拳。

    </ol>

     </strong>
    </p>

     </li>

    <p style="margin-left:.0001pt;text-align:justify;">
     <strong>
      可能答案：
     </strong>
    </p>
    <ol>
     <li>
      <ol>
       <li style="text-align:justify;">
        团队达成共识的方式一般来说往往有自发、说服、强迫、表决、仲裁等5种方式。“自发”的方式最好但很不现实；“强迫”、“仲裁”可能会导致分裂；过多的使用“表决”则说明团队还不太默契，尽管它能“公正”的体现大多数人的意愿；“说服”是一个很温和的方式，但需要时间和合作的心态。
       </li>
       <li style="text-align:justify;">
        没有一种方法是最好的，而应根据时机采用不同的方法。
       </li>
       <li style="text-align:justify;">
        创意是一个团队生命活力的体现，但创意需要得到团队的认可才会有成绩。
       </li>
