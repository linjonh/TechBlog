     查看python的版本号有 四种方案
    </h2>
    <h3>
     <a id="_4">
     </a>
     命令行查看
    </h3>
    <p>
     命令行下查看，在配置好了开发运行环境的前提下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/53b3f05bd8be6f8d69fc50b8daf5e101.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6d0b6fb8e8159a89f9c8eaa1444d4d88.png"/>
     </img>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        输入 python -V
       </strong>
       ，默认是系统安装的python版本，如果环境中安装了2和3，分别区分
       <code>
        python2 -V 和python3 -V
       </code>
       ；
      </p>
     </li>
     <li>
      <p>
       ![](https://img-blog.csdnimg.cn/44055f0fc38a43c6aa6cd187ee1446b0.png
      </p>
     </li>
     <li>
      <p>
       <strong>
        输入 python --verison
       </strong>
       ，默认是系统安装的python版本，如果环境中安装了2和3，分别区分
       <code>
        python2 --version 和python3 --version
       </code>
       ；
      </p>
     </li>
     <li>
      <p>
       <strong>
        输入 python-X
       </strong>
       ，在命令行输入python3或是python2进入python的编译环境，可以看到相对应的python的版本
      </p>
     </li>
    </ol>
    <h3>
     <a id="sys_15">
     </a>
     sys模块
    </h3>
    <ol>
     <li>
      <strong>
       导入sys模块
      </strong>
      ，
     </li>
    </ol>
    <pre><code>import sys 
print(sys.version)
result如下:
3.6.9 (default, Nov  7 2019, 10:44:02) \n[GCC 8.3.0]
</code></pre>
    <p>
     总结：
    </p>
    <pre><code>在命令行中：
1、python -V
2、python --version
3、python
4、
import sys
sys.version
在python.py 文件中:
import sys
print(sys.version)
</code></pre>
