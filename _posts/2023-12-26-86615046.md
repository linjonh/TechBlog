      ，当时很好奇研究了一下怎么在浏览器中运行自己的JS。原来可以通过Chrome的扩展程序可以实现，废话就不多说了直接上代码。。。

    <p>
     <strong>
      新建一个manifest.json的文件（扩展程序的配置文件）该文件字符集格式必须为UTF-8
     </strong>
    </p>
    <p>
     <strong>
      manifest.json代码
     </strong>
    </p>
    <pre class="prettyprint"><code class="language-json hljs has-numbering">{
  "<span class="hljs-attribute">name</span>": <span class="hljs-value"><span class="hljs-string">"Flutter"</span></span>,
  "<span class="hljs-attribute">manifest_version</span>": <span class="hljs-value"><span class="hljs-number">2</span></span>,
  "<span class="hljs-attribute">version</span>": <span class="hljs-value"><span class="hljs-string">"1.0"</span></span>,
  "<span class="hljs-attribute">description</span>": <span class="hljs-value"><span class="hljs-string">"I Love Flutter ~~！"</span></span>,
  "<span class="hljs-attribute">browser_action</span>": <span class="hljs-value">{
    "<span class="hljs-attribute">default_icon</span>": <span class="hljs-value"><span class="hljs-string">"1.png"</span>
  </span>}</span>,
  "<span class="hljs-attribute">content_scripts</span>": <span class="hljs-value">[
    {
      "<span class="hljs-attribute">matches</span>": <span class="hljs-value">[<span class="hljs-string">"https://github.com/*"</span>,<span class="hljs-string">"http://www.csdn.net/*"</span>]</span>,
      "<span class="hljs-attribute">js</span>": <span class="hljs-value">[<span class="hljs-string">"script.js"</span>]
    </span>}
  ]
</span>}</code></pre>
    <blockquote>
     <p>
      <strong>
       default_icon ： 插件图标
      </strong>
      <br/>
      <strong>
       matches ： Content Script注入的Web页面
      </strong>
      <br/>
      <strong>
       js : 注入到匹配的Web页面中的JS文件
      </strong>
     </p>
    </blockquote>
    <p>
     <br/>
    </p>
    <p>
     <strong>
      script.js代码
     </strong>
    </p>
    <pre class="prettyprint"><code class="language-javascript hljs has-numbering">alert(<span class="hljs-string">"I Love Flutter"</span>);

document.getElementById(<span class="hljs-string">"user[password]"</span>).value=<span class="hljs-string">"123456"</span>;</code></pre>
