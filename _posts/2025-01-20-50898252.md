    <p style="word-wrap: break-word; margin-top: 0px; margin-bottom: 15pt; line-height: 1.5; letter-spacing: 1px; font-family: 'Microsoft YaHei', Verdana, sans-serif, SimSun; font-size: 14px;">
     <strong>
      <span style="font-size: 18px;">
       平台介绍：
      </span>
      <br/>
      JEECG（J2EE Code Generation）
     </strong>
     是一款基于代码生成器的智能开发平台。引领新的开发模式(OnlineCoding模式-&gt;代码生成器模式-&gt;手工MERGE智能开发)， 可以帮助解决Java项目60%的重复工作，让开发更多关注业务逻辑。既能快速提高开发效率，帮助公司节省成本，同时又不失灵活性。
    </p>
    <p style="word-wrap: break-word; margin-top: 0px; margin-bottom: 15pt; line-height: 1.5; letter-spacing: 1px; font-family: 'Microsoft YaHei', Verdana, sans-serif, SimSun; font-size: 14px;">
     <strong>
      JEECG宗旨
     </strong>
     是: 简单功能由Online在线开发;复杂功能由代码生成器生成进行手工Merge; 复杂流程业务采用表单自定义，业务流程使用工作流来实现、扩展出业务接口，供开发编写业务逻辑，既保证了公司流程的保密行，又减少了开发人员的工作量。
    </p>

     【1】
     <strong>
      新特性（亮点）
     </strong>
     <br/>
     1.平台插件支持（可插拔）
     <br/>
     2.移动报表配置能力（一次配置，7种展示风格）
     <br/>
     报表风格：折线图、面积图、条状图、3D柱状图、2D柱状图、3D饼图、2D饼图
     <br/>
     3.移动表单支持（提供多套移动模板）
     <br/>
     4.移动OA
    </p>

     5.通讯录

     【3】
     <strong>
      平台改进点

     2.升级spring
     <br/>
     3.支持单元测试
     <br/>
     4.集成cxf webservice接口，并实现一个服务端和客户端调用的demo
     <br/>
     5.数据库兼容性修改
     <br/>
     6.数据权限扩展支持写表达式，通过session取值
     <br/>
     7.电子签章
     <br/>
     8.online支持UE编辑器类型
     <br/>
     9.用户、组织机构，角色导入功能

     【4】

      Online样式（新增）
     </strong>

     1.单表通用模板（ace版）

     2.一对多通用模板（ace版）
     <br/>
     3.多tab样式模板
     <br/>
     4.单表移动模板
     <br/>
     5.一对多移动模板
