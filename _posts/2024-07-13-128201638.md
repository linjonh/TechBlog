    <h3>
     <a id="dumi_1">
     </a>
     一、dumi介绍
    </h3>
    <p>
     关于对
     <code>
      dumi
     </code>
     的介绍我们就可以简单的理解为快速开发文档的一种便捷开发工具，里面囊括了多种配置，我们不需要再去手动的编写组件这种，所以为我们开发组件库文档，官方文档，个人博客介绍这种网站提供了很便捷的帮助。
    </p>
    <p>
     有兴趣的同学也可以去看下
     <a href="https://github.com/hanbingxu82/tinkerbell-ui">
      tinkerbell-ui
     </a>
     这套手搭组件库文档，满满的😭血泪史😭。
    </p>
    <p>
     现在dumi推出了2的版本，当然我们在这里已经引入了dumi1的版本，所以此文我们针对dumi1进行简单介绍，2的版本，我们在后续的升级中会有所体现，文档地址参照如下。
    </p>
    <p>
     <a href="https://v1.d.umijs.org/zh-CN/guide" rel="nofollow">
      dumi1官方文档
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/38298cb52651f54e55a34673e5bad727.png"/>
    </p>
    <h3>
     <a id="_dumi_dumi_12">
     </a>
     二、使用 dumi 的两种方式（着重在已成型项目中使用dumi）
    </h3>
    <h4>
     <a id="21_dumi__13">
     </a>
     2.1、基于 dumi 官网带有的脚手架去进行开发
    </h4>
    <p>
     这一块不是咱们今天主要介绍的主题，当然官网也给了我们很好的实例，针对引用组件等等，我们可以直接根据路由等然后抛出、引用，然后把菜单完善好即可。
     <br/>
     参考如下：
     <br/>
     <a href="https://v1.d.umijs.org/zh-CN/guide#%E5%BF%AB%E9%80%9F%E4%B8%8A%E6%89%8B" rel="nofollow">
      dumi 快速上手
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2bf3a263e40eba4e4b2ea5d9a3e9ef4d.png">
      <br/>
      我们可以根据上方的流程，得到一个dumi的脚手架
     </img>
    </p>
    <h4>
     <a id="22_dumi__19">
     </a>
     2.2、在已成型的项目中引用 dumi 插件，运行项目
    </h4>
    <p>
     参照文档，我们执行两步操作
    </p>
    <pre><code>npm i dumi@1.1.48 -S
</code></pre>
    <p>
     完成之后我们在package.json的script中添加
    </p>
    <pre><code>    "dumi": "cross-env APP_ROOT=dumi dumi dev",
    "dumi-build": "cross-env APP_ROOT=dumi dumi build",
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fe18dc2f0af96f1e37d5f2b9f461d593.png">
      <br/>
      随即我们可以再项目的根目录创建一个dumi的文件夹，这个时候我们需要参照文档去进行对应的路由匹配，暴露组件等
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/33d084152d5c29b3499dbf31a668ef35.png">
       <br/>
       具体可以参照文档，对应的文件对照都在文档中有所体现
       <br/>
       <a href="https://v1.d.umijs.org/zh-CN/guide/basic#%E6%99%AE%E9%80%9A%E9%A1%B9%E7%9B%AE" rel="nofollow">
        目录介绍
       </a>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/26e93e3d5b8e91a60807ed215bb8d386.png">
        <br/>
        github 对应的实例（docs中的index.md,简单配置等可以直接copy过来）
        <br/>
        <a href="https://github.com/umijs/dumi-template">
         dumi-template
        </a>
       </img>
      </img>
     </img>
    </p>
