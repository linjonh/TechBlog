    </h2>
    <h3>
     <a id="1_2">
     </a>
     1、区块链思维的特点：
    </h3>
    <p>
     1、拒绝依赖。
     <br/>
     区块链不需要中间担保人就能顺利完成交易。
     <br/>
     2、不可反悔。
     <br/>
     用户在区块链上进行交易，如果转账完成后发现转错了地址，这时交易是无法取消的。
     <br/>
     3、加密授权。
     <br/>
     区块链上的数据都是被加密的，他人无法解密，更无法窃取。只有用户自身才能看到交易内容，相关方如果想阅读其中的内容，只能想用户申请授权，经过用户许可才能实现。
     <br/>
     4、消除不平等
     <br/>
     区块链是一个去中心化、分布式数据账本。
