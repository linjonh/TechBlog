    <p>
     在之前写了
     <a href="https://blog.csdn.net/zzwwjjdj1/article/details/85316961">
      微信小程序中如何使用promise
     </a>
     ，其实使用promise的最终目的是要能使用
     <span style="color:#f33b45;">
      async/await。
     </span>
    </p>
    <p>
     同时要使用async/await也必须在promise的基础之上。
    </p>
    <p>
     1、下载regenerator库
    </p>
    <p>
     npm i regenerator       或者   点击下载：
     <a href="http://nodejs999.com/regenerator-runtime.rar" rel="nofollow">
      http://nodejs999.com/regenerator-runtime.rar
     </a>
    </p>
    <p>
     然后将下载文件中的
     <span style="color:#f33b45;">
      regenerator-runtime
     </span>
     文件夹拿出来，放到小程序代码中去，一般是放在utils文件夹。
    </p>
    <p>
     2、在需要使用async/await的js文件顶部引入regenerator库：
    </p>
    <pre class="has"><code class="language-javascript">let regeneratorRuntime = require("../../utils/regenerator-runtime/runtime");</code></pre>
    <p>
     3、例子：保存canvas画布到手机
    </p>
    <pre class="has"><code class="language-javascript">async saveImg() {
    try {
      let { tempFilePath } = await wechat.api("canvasToTempFilePath", obj);
      await wechat.api("saveImageToPhotosAlbum", { filePath: tempFilePath });
    }
    catch (e) {
      console.log(e);
    }
  }</code></pre>
    <p>
     代码中的wechat.api方法看这里：
