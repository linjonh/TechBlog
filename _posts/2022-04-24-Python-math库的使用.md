    <h2>
     <a id="math__2">
     </a>
     math 库概述
    </h2>
    <p>
     <strong>
      math
     </strong>
     库是
     <strong>
      Python
     </strong>
     提供的内置数学类函数库，因为复数类型常用于科学计算，一般计算并不常用，因此
     <strong>
      math
     </strong>
     库不支持复数类型，仅支持整数和浮点数运算。
     <strong>
      math
     </strong>
     库一共提供了
     <strong>
      4
     </strong>
     个数学常数和
     <strong>
      44
     </strong>
     个函数。
     <strong>
      44
     </strong>
     个函数分为
     <strong>
      4
     </strong>
     类，包括
     <strong>
      16
     </strong>
     个数值表示函数、
     <strong>
      8
     </strong>
     个幂对数函数、
     <strong>
      16
     </strong>
     个三角对数函数和
     <strong>
      4
     </strong>
     个高等特殊函数。
    </p>
    <p>
     <strong>
      math
     </strong>
     库中函数数量较多，我们在学习过程中只需要逐个理解函数功能，记住个别常用函数即可。实际编程中，如果需要采用
     <strong>
      math
     </strong>
     库，可以随时查看
     <strong>
      math
     </strong>
     库快速参考。
    </p>
    <p>
     <strong>
      math
     </strong>
     库中的函数不能直接使用，需要首先使用保留字
     <strong>
      import
     </strong>
     引用该库，引用方式如下。
     <br/>
     第一种：
     <br/>
     <strong>
      import math
     </strong>
     <br/>
     对
     <strong>
      math
     </strong>
     库中函数采用
     <strong>
      math.&lt;
      <b>
       b&gt;()
      </b>
     </strong>
     形式使用，例如：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> math
a<span class="token operator">=</span>math<span class="token punctuation">.</span>ceil<span class="token punctuation">(</span><span class="token number">10.2</span><span class="token punctuation">)</span>		<span class="token comment">#向上取整</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>a<span class="token punctuation">)</span>
</code></pre>
    <blockquote>
     <p>
      11
     </p>
    </blockquote>
    <p>
     第二种：
     <br/>
     <strong>
      from math import &lt;函数名&gt;
     </strong>
     <br/>
     对
     <strong>
      math
     </strong>
     库中函数可以直接采用
     <strong>
      &lt;函数名&gt;()
     </strong>
     形式使用，例如：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> math <span class="token keyword">import</span> floor
a<span class="token operator">=</span>floor<span class="token punctuation">(</span><span class="token number">10.2</span><span class="token punctuation">)</span>		<span class="token comment">#向下取整</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>a<span class="token punctuation">)</span>
</code></pre>
    <blockquote>
     <p>
      10
     </p>
    </blockquote>
    <p>
     第二种的另一种形式是 from math import * 。如果采用这种方式引入
     <strong>
      math
     </strong>
     库，
     <strong>
      math
     </strong>
     库中所有函数可以采用
     <strong>
      &lt;函数名&gt;()
     </strong>
     形式直接使用。
    </p>
    <p>
     <strong>
      math
     </strong>
     库及其他函数库的引用都可以自由选取这两种方式实现。
    </p>

     <strong>
      math
     </strong>
     库提供的函数，而不直接使用
     <strong>
      Python
     </strong>
     提供的运算符。
