    </h4>
    <h4>
     （在学习了软件工程这门课，太多的知识点，而且有些知识点很容易混淆，下面就将之前经常混淆的一些知识点还有细节进行总结因为初学软件工程这本书，能力有限尚且不能完全归纳出来，望见谅！希望在一次次总结中不断增加巩固自己的知识点吧）
    </h4>
    <h2 id="1%E3%80%81%E5%B1%82%E6%AC%A1%E5%9B%BE(H%E5%9B%BE)">
     <span style="color:#000000;">
      <strong>
       1、层次图
      </strong>
     </span>
     <span style="color:#000000;">
      <strong>
       (H
      </strong>
     </span>
     <span style="color:#000000;">
      <strong>
       图
      </strong>
     </span>
     <span style="color:#000000;">
      <strong>
       )
      </strong>
     </span>
    </h2>
    <h4 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E7%94%A8%E6%9D%A5%E6%8F%8F%E7%BB%98%E8%BD%AF%E4%BB%B6%E7%9A%84%E5%B1%82%E6%AC%A1%E7%BB%93%E6%9E%84%E3%80%82%E5%BE%88%E9%80%82%E4%BA%8E%E5%9C%A8%E8%87%AA%E9%A1%B6%E5%90%91%E4%B8%8B%E8%AE%BE%E8%AE%A1%E8%BD%AF%E4%BB%B6%E7%9A%84%E8%BF%87%E7%A8%8B%E4%B8%AD%E4%BD%BF%E7%94%A8%E3%80%82">
     <span style="color:#000000;">
      层次图用来描绘软件的层次结构。很适于在自顶向下设计软件的过程中使用。用此图表示自顶向下分解所得系统的模块层次结构，H图又可称为模块层次图。H图中一个方形的方块代表一个模块，方块内可写出标识此模块的处理功能或模块名。模块间的调用关系，用联结两模块的直线表示。
     </span>
    </h4>
    <p>
    </p>
    <h2 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E5%92%8C%E5%B1%82%E6%AC%A1%E6%96%B9%E6%A1%86%E5%9B%BE%E7%9A%84%E5%8C%BA%E5%88%AB%EF%BC%9A">
     <span style="color:#775f55;">
      <strong>
       层次图和层次方框图的区别：
      </strong>
     </span>
    </h2>
    <p>
     <img alt="" class="has" height="255" src="https://i-blog.csdnimg.cn/blog_migrate/9f7da8c2c961734f61aad7f3eda40bd6.png" width="737"/>
    </p>
    <p style="margin-left:.35in;">
    </p>
    <h2 id="2%E3%80%81%E7%BB%93%E6%9E%84%E5%9B%BE">
     <strong>
      2、
      <span style="color:#000000;">
       结构图
      </span>
     </strong>
    </h2>
    <h4 id="Yourdon%E6%8F%90%E5%87%BA%E7%9A%84%E7%BB%93%E6%9E%84%E5%9B%BE%E6%98%AF%E8%BF%9B%E8%A1%8C%E8%BD%AF%E4%BB%B6%E7%BB%93%E6%9E%84%E8%AE%BE%E8%AE%A1%E7%9A%84%E5%8F%A6%E4%B8%80%E4%B8%AA%E6%9C%89%E5%8A%9B%E5%B7%A5%E5%85%B7%E3%80%82">
     <span style="color:#000000;">
      Yourdon提出的结构图是进行软件结构设计的另一个有力工具。
     </span>
    </h4>
    <h4 id="%E7%BB%93%E6%9E%84%E5%9B%BE%E5%92%8C%E5%B1%82%E6%AC%A1%E5%9B%BE%E7%B1%BB%E4%BC%BC%EF%BC%8C%E4%B9%9F%E6%98%AF%E6%8F%8F%E7%BB%98%E8%BD%AF%E4%BB%B6%E7%BB%93%E6%9E%84%E7%9A%84%E5%9B%BE%E5%BD%A2%E5%B7%A5%E5%85%B7%E3%80%82">
     <span style="color:#000000;">
      结构图和层次图类似，也是描绘软件结构的图形工具。
     </span>
    </h4>
    <h2>
    </h2>
    <h2 id="%E5%9F%BA%E6%9C%AC%E7%AC%A6%E5%8F%B7%EF%BC%9A">
     <span style="color:#775f55;">
      <strong>
       基本符号：
      </strong>
     </span>
    </h2>
    <ul>
     <li>
      <h4 id="%E6%96%B9%E6%A1%86%E4%BB%A3%E8%A1%A8%E4%B8%80%E4%B8%AA%E6%A8%A1%E5%9D%97%EF%BC%9B">
       <span style="color:#000000;">
        <strong>
         方框代表一个模块；
        </strong>
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E6%96%B9%E6%A1%86%E4%B9%8B%E9%97%B4%E7%9A%84%E7%9B%B4%E7%BA%BF%E8%A1%A8%E7%A4%BA%E6%A8%A1%E5%9D%97%E7%9A%84%E8%B0%83%E7%94%A8%E5%85%B3%E7%B3%BB%EF%BC%9B">
       <span style="color:#000000;">
        <strong>
         方框之间的直线表示模块的调用关系；
        </strong>
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E5%B0%BE%E9%83%A8%E6%98%AF%E7%A9%BA%E5%BF%83%E5%9C%86%E7%AE%AD%E5%A4%B4%E8%A1%A8%E7%A4%BA%E4%BC%A0%E9%80%92%E7%9A%84%E6%98%AF%E6%95%B0%E6%8D%AE%EF%BC%9B">
       <span style="color:#000000;">
        <strong>
         尾部是空心圆箭头表示传递的是数据；
        </strong>
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E5%B0%BE%E9%83%A8%E5%AE%9E%E5%BF%83%E5%9C%86%E7%AE%AD%E5%A4%B4%E8%A1%A8%E7%A4%BA%E4%BC%A0%E9%80%92%E7%9A%84%E6%98%AF%E6%8E%A7%E5%88%B6%E4%BF%A1%E6%81%AF%E3%80%82">
       <span style="color:#000000;">
        <strong>
         尾部实心圆箭头表示传递的是控制信息。
        </strong>
       </span>
      </h4>
     </li>
    </ul>
    <h2 style="margin-left:.35in;">
    </h2>
    <h2 id="%E5%9B%9B%E7%A7%8D%E6%A8%A1%E5%9D%97" style="margin-left:.35in;">
     <span style="color:#f33b45;">
      四种模块
     </span>
    </h2>
    <p>
     <span style="color:#e579b6;">
      <img alt="" class="has" height="508" src="https://i-blog.csdnimg.cn/blog_migrate/45b8bcd2588e20c3a9a4c59753d34211.png" width="534"/>
     </span>
    </p>
    <p>
    </p>
    <h2 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E5%92%8C%E7%BB%93%E6%9E%84%E5%9B%BE%E7%9A%84%E7%9B%B8%E5%90%8C%E7%82%B9%EF%BC%9A">
     <span style="color:#775f55;">
      <strong>
       层次图和结构图的相同点：
      </strong>
     </span>
    </h2>
    <ul>
     <li>
      <h4 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E5%92%8C%E7%BB%93%E6%9E%84%E5%9B%BE%E5%B9%B6%E4%B8%8D%E4%B8%A5%E6%A0%BC%E8%A1%A8%E7%A4%BA%E6%A8%A1%E5%9D%97%E7%9A%84%E8%B0%83%E7%94%A8%E6%AC%A1%E5%BA%8F%EF%BC%8C%E5%A4%9A%E6%95%B0%E4%BA%BA%E4%B9%A0%E6%83%AF%E6%8C%89%E8%B0%83%E7%94%A8%E6%AC%A1%E5%BA%8F%E4%BB%8E%E5%B7%A6%E5%88%B0%E5%8F%B3%E7%94%BB%E6%A8%A1%E5%9D%97%EF%BC%9B">
       <span style="color:#f33b45;">
        层次图和结构图并不严格表示模块的调用次序，多数人习惯按调用次序从左到右画模块；
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E5%92%8C%E7%BB%93%E6%9E%84%E5%9B%BE%E5%B9%B6%E4%B8%8D%E6%8C%87%E6%98%8E%E4%BD%95%E6%97%B6%E8%B0%83%E7%94%A8%E4%B8%8B%E5%B1%82%E6%A8%A1%E5%9D%97%EF%BC%9B">
       <span style="color:#f33b45;">
        层次图和结构图并不指明何时调用下层模块；
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E4%B8%A4%E8%80%85%E9%83%BD%E6%98%AF%E6%8F%8F%E7%BB%98%E8%BD%AF%E4%BB%B6%E7%BB%93%E6%9E%84%E7%9A%84%E5%9B%BE%E5%BD%A2%E5%B7%A5%E5%85%B7">
       <span style="color:#f33b45;">
        两者都是描绘软件结构的图形工具
       </span>
      </h4>
     </li>
     <li>
      <h4 id="%E5%B1%82%E6%AC%A1%E5%9B%BE%E5%92%8C%E7%BB%93%E6%9E%84%E5%9B%BE%E5%8F%AA%E8%A1%A8%E6%98%8E%E4%B8%80%E4%B8%AA%E6%A8%A1%E5%9D%97%E8%B0%83%E7%94%A8%E9%82%A3%E4%BA%9B%E6%A8%A1%E5%9D%97%EF%BC%8C%E6%B2%A1%E6%9C%89%E8%A1%A8%E7%A4%BA%E6%A8%A1%E5%9D%97%E5%86%85%E8%BF%98%E6%9C%89%E6%B2%A1%E6%9C%89%E5%85%B6%E4%BB%96%E6%88%90%E5%88%86%EF%BC%9B">
       <span style="color:#f33b45;">
        层次图和结构图只表明一个模块调用那些模块，没有表示模块内还有没有其他成分；
       </span>
      </h4>
     </li>
    </ul>
    <h2>
     <span style="color:#775f55;">
      <strong>
       层次图和系统结构图的不同点：
      </strong>
     </span>
    </h2>
    <ul>
     <li>
      <h4>
       <span style="color:#f33b45;">
        <strong>
         系统结构图的出发点是数据流图，但如果已具备H图，则在此基础，上加注模块间的控制与通信标志就成了系统结构图
        </strong>
       </span>
      </h4>
     </li>
    </ul>
    <h4>
     <span style="color:#3399ea;">
      <strong>
       （通常用层次图作为描绘软件结构的文档）
      </strong>
     </span>
    </h4>
