    <h4>
     一、TCP介绍
    </h4>
    <h3>
     TCP的特点：
    </h3>
    <ol>
     <li>
      <strong>
       面向连接的协议
      </strong>
      ：TCP是一种可靠的、面向连接的协议，在通信之前需要建立连接，以确保数据的可靠传输。这意味着在传输数据之前，发送方和接收方之间需要建立一条可靠的连接通道。
     </li>
     <li>
      <strong>
       流式协议
      </strong>
      ：TCP将数据看作是一连串的字节流，而不是独立的数据包。这意味着在传输过程中，数据可以被分割成多个数据包进行传输，并在接收端进行重新组装。
     </li>
     <li>
      <strong>
       可靠传输
      </strong>
      ：TCP通过使用校验和、序列号和确认应答等机制，确保数据的可靠传输。如果在传输过程中发生错误或丢失，TCP会进行重传，以确保数据的完整性和准确性。
     </li>
     <li>
      <strong>
       出错重传
      </strong>
      ：当TCP接收到错误的数据包时，它会要求发送方重新发送该数据包，以确保数据的正确性。
     </li>
     <li>
      <strong>
       确认应答
      </strong>
      ：TCP使用确认应答机制来确保数据的可靠传输。当接收方收到一个数据包时，它会向发送方发送一个确认应答，表示已经收到了该数据包。
     </li>
     <li>
      <strong>
       服务器被动连接，客户端主动连接
      </strong>
      ：在TCP连接中，服务器通常处于被动状态，等待客户端的连接请求。而客户端则处于主动状态，负责发起连接请求。一旦连接建立成功，双方就可以进行数据传输。
     </li>
    </ol>

    <h3>
     TCP编程流程
    </h3>
    <h4>
     服务器端流程：
    </h4>
    <ol>
     <li>
      <strong>
       创建套接字（socket）
      </strong>
      ：使用
      <code>
       socket()
      </code>
      函数创建一个套接字，指定使用的协议族（如IPv4或IPv6）和socket类型（如流式socket）。
     </li>
     <li>
      <strong>
       绑定套接字（bind）
      </strong>
      ：使用
      <code>
       bind()
      </code>
      函数将套接字与服务器的网络信息（如IP地址和端口号）进行绑定。
     </li>
     <li>
      <strong>
       监听套接字（listen）
      </strong>
      ：使用
      <code>
       listen()
      </code>
      函数将套接字设置为监听状态，准备接收客户端的连接请求。
     </li>
     <li>
      <strong>
       接受连接（accept）
      </strong>
      ：使用
      <code>
       accept()
      </code>
      函数阻塞等待客户端的连接请求，并返回一个新的套接字用于与该客户端进行通信。
     </li>
     <li>
      <strong>
       进行通信（recv/send）
      </strong>
      ：使用
      <code>
       recv()
      </code>
      和
      <code>
       send()
      </code>
      函数与客户端进行数据的接收和发送。
     </li>
     <li>
      <strong>
       关闭套接字（close）
      </strong>
      ：通信完成后，使用
      <code>
       close()
      </code>
      函数关闭套接字，释放资源。
     </li>
    </ol>
    <h4>
     客户端流程：
    </h4>
    <ol>
     <li>
      <strong>
       创建套接字（socket）
      </strong>
      ：与服务器端相同，使用
      <code>
       socket()
      </code>
      函数创建一个套接字。
     </li>
     <li>
      <strong>
       连接服务器（connect）
      </strong>
      ：使用
      <code>
       connect()
      </code>
      函数向服务器发起连接请求，指定服务器的IP地址和端口号。
     </li>
     <li>
      <strong>
       进行通信（send/recv）
      </strong>
      ：连接建立后，使用
      <code>
       send()
      </code>
      和
      <code>
       recv()
      </code>
      函数与服务器进行数据的发送和接收。
     </li>
     <li>
      <strong>
       关闭套接字（close）
      </strong>
      ：通信完成后，使用
      <code>
       close()
      </code>
      函数关闭套接字，释放资源。
     </li>
    </ol>
    <h2>
     二、TCP编程-socket
    </h2>
    <pre><code class="language-cpp">#include &lt;sys/types.h&gt;
#include &lt;sys/socket.h&gt;
int socket(int domain, int type, int protocol);

功能：
    创建一个套接字，返回一个文件描述符
参数：
    domain：通信域，协议族
      AF_UNIX 本地通信
      AF_INET ipv4网络协议
      AF_INET6 ipv6网络协议
      AF_PACKET 底层接口
    type：套接字的类型
      SOCK_STREAM 流式套接字（tcp）
      SOCK_DGRAM 数据报套接字（udp）
      SOCK_RAW 原始套接字（用于链路层）
    protocol：附加协议，如果不需要，则设置为0

返回值：
    成功：文件描述符
    失败：‐1</code></pre>
    <p>
     <strong>
      案例
     </strong>
    </p>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include &lt;sys/types.h&gt;

#include &lt;stdlib.h&gt;

int main(int argc, char const *argv[])
{
    //通过socket函数创建一个TCP套接字
    int sockfd;
    if((sockfd = socket(AF_INET, SOCK_STREAM, 0)) == -1)
    {
        perror("fail to socket");
        exit(1);
    }

    printf("sockfd = %d\n", sockfd);

    return 0;
}</code></pre>

    <h3>
     3.4 客户端code
    </h3>
    <p>
     使用windows下的网络调试助手作为服务器
    </p>
    <p>
     <img alt="" height="566" src="https://i-blog.csdnimg.cn/blog_migrate/b5678fa6038c16c5efa8d996f4dced50.png" width="588"/>
    </p>
    <p>
     客户端的程序
    </p>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;unistd.h&gt;
#include &lt;sys/types.h&gt;
#include &lt;sys/socket.h&gt;
#include &lt;arpa/inet.h&gt;
#include &lt;netinet/in.h&gt;
#include &lt;string.h&gt;

#define N 128

int main(int argc, char const *argv[])
{
    if(argc &lt; 3)
    {
        fprintf(stderr, "Usage: %s [ip] [port]\n", argv[0]);
        exit(1);
    }

    //第一步：创建套接字
    int sockfd;
    if((sockfd = socket(AF_INET, SOCK_STREAM, 0)) == -1)
    {
        perror("fail to socket");
        exit(1);
    }

    //printf("sockfd = %d\n", sockfd);

    //第二步：发送客户端连接请求
    struct sockaddr_in serveraddr;
    socklen_t addrlen = sizeof(serveraddr);

    serveraddr.sin_family = AF_INET;
    serveraddr.sin_addr.s_addr = inet_addr(argv[1]);
    serveraddr.sin_port = htons(atoi(argv[2]));

    if(connect(sockfd, (struct sockaddr *)&amp;serveraddr, addrlen) == -1)
    {
        perror("fail to connect");
        exit(1);
    }

    //第三步：进行通信
    //发送数据
    char buf[N] = "";
    fgets(buf, N, stdin);
    buf[strlen(buf) - 1] = '\0';
    if(send(sockfd, buf, N, 0) == -1)
    {
        perror("fail to send");
        exit(1);
    }

    //接收数据
    char text[N] = "";
    if(recv(sockfd, text, N, 0) == -1)
    {
        perror("fail to recv");
        exit(1);
    }

    printf("from server: %s\n", text);

    //第四步：关闭套接字文件描述符
    close(sockfd);

    return 0;
}</code></pre>

    <p>
     <strong>
      简单写个例子

    <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include &lt;sys/types.h&gt;
#include &lt;sys/socket.h&gt;
#include &lt;netinet/in.h&gt;

int main() {
    // 创建一个流式套接字
    int sockfd = socket(AF_INET, SOCK_STREAM, 0);
    if (sockfd == -1) {
        perror("socket");
        return -1;
    }

    // 初始化网络信息结构体
    struct sockaddr_in addr;
    addr.sin_family = AF_INET;
    addr.sin_port = htons(8080); // 绑定到8080端口
    addr.sin_addr.s_addr = htonl(INADDR_ANY); // 绑定到任意IP地址

    // 将套接字与网络信息结构体绑定
    socklen_t addrlen = sizeof(addr);
    if (bind(sockfd, (struct sockaddr *)&amp;addr, addrlen) == -1) {
        perror("bind");
        return -1;
    }

    printf("Bind successful\n");

    // 关闭套接字
    close(sockfd);

    return 0;
}</code></pre>

    <h3>
     4.5 TCP服务器例子
    </h3>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;unistd.h&gt;
#include &lt;sys/types.h&gt;
#include &lt;sys/socket.h&gt;
#include &lt;arpa/inet.h&gt;
#include &lt;netinet/in.h&gt;
#include &lt;string.h&gt;

#define N 128

int main(int argc, char const *argv[])
{
    if(argc &lt; 3)
    {
        fprintf(stderr, "Usage: %s [ip] [port]\n", argv[0]);
        exit(1);
    }

    //第一步：创建套接字
    int sockfd;
    if((sockfd = socket(AF_INET, SOCK_STREAM, 0)) == -1)
    {
        perror("fail to socket");
        exit(1);
    }

    //第二步：将套接字与服务器网络信息结构体绑定
    struct sockaddr_in serveraddr;
    socklen_t addrlen = sizeof(serveraddr);

    serveraddr.sin_family = AF_INET;
    serveraddr.sin_addr.s_addr = inet_addr(argv[1]);
    serveraddr.sin_port = htons(atoi(argv[2]));

    if(bind(sockfd, (struct sockaddr *)&amp;serveraddr, addrlen) == -1)
    {
        perror("fail to bind");
        exit(1);
    }

    //第三步：将套接字设置为被动监听状态
    if(listen(sockfd, 10) == -1)
    {
        perror("fail to listen");
        exit(1);
    }

    //第四步：阻塞等待客户端的链接请求
    int acceptfd;
    struct sockaddr_in clientaddr;
    if((acceptfd = accept(sockfd, (struct sockaddr *)&amp;clientaddr, &amp;addrlen)) == -1)
    {
        perror("fail to accept");
        exit(1);
    }

    //打印连接的客户端的信息
    printf("ip:%s, port:%d\n", inet_ntoa(clientaddr.sin_addr), ntohs(clientaddr.sin_port));

    //第五步：进行通信
    //tcp服务器与客户端通信时，需要使用accept函数的返回值
    char buf[N] = "";
    if(recv(acceptfd, buf, N, 0) == -1)
    {
        perror("fail to recv");
    }

    printf("from client: %s\n", buf);

    strcat(buf, " *_*");
    if(send(acceptfd, buf, N, 0) == -1)
    {
        perror("fail to send");
        exit(1);
    }

    //关闭套接字文件描述符
    close(acceptfd);
    close(sockfd);

    return 0;
