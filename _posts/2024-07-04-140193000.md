    <h4>
     <a id="_0">
     </a>
     注意：
    </h4>
    <p>
     安卓手机和苹果手机在测试unity游戏的过程中最大的区别在于：
     <strong>
      unity remote5软件天然对安卓手机不支持，需要在安卓手机上开启【开发者选项和USB调试】
     </strong>
    </p>
    <p>
     详细的区别可以参考我的另外一篇文章：
     <br/>
     <a href="https://blog.csdn.net/weixin_42069404/article/details/140181127?csdn_share_tail=%7B%22type%22:%22blog%22,%22rType%22:%22article%22,%22rId%22:%22140181127%22,%22source%22:%22weixin_42069404%22%7D">
      【苹果、Ipad使用unity remote5连接电脑测试unity游戏】
     </a>
    </p>
    <h4>
     <a id="_9">
     </a>
     步骤一：准备工作
    </h4>
    <h5>
     <a id="1__Unity_Remote_5_11">
     </a>
     1. 安装 Unity Remote 5
    </h5>
    <ol>
     <li>
      打开安卓手机上的 Google Play 商店。
     </li>
     <li>
      搜索 “Unity Remote 5”。
     </li>
     <li>
      点击安装并等待应用安装完成。
     </li>
    </ol>
    <h5>
     <a id="2__USB__17">
     </a>
     2. 启用开发者选项和 USB 调试
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        启用开发者选项
       </strong>
       :
      </p>
      <ul>
       <li>
        打开手机设置。
       </li>
       <li>
        进入
        <code>
         关于手机
        </code>
        或
        <code>
         系统
        </code>
        &gt;
        <code>
         关于手机
        </code>
        。
       </li>
       <li>
        连续点击
        <code>
         版本号
        </code>
        或
        <code>
         构建号
        </code>
        7 次，直到提示已进入开发者模式。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        启用 USB 调试
       </strong>
       :
      </p>
      <ul>
       <li>
        返回设置，进入
        <code>
         系统
        </code>
        &gt;
        <code>
         开发者选项
        </code>
        。
       </li>
       <li>
        找到并启用
        <code>
         USB 调试
        </code>
        。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_28">
     </a>
     步骤二：连接安卓设备和电脑
    </h4>
    <h5>
     <a id="1__30">
     </a>
     1. 连接设备
    </h5>
    <ol>
     <li>
      使用 USB 数据线将安卓设备连接到电脑。
     </li>
     <li>
      如果手机弹出 USB 连接模式提示，选择
      <code>
       传输文件
      </code>
      或
      <code>
       仅充电
      </code>
      模式。
     </li>
    </ol>
    <h5>
     <a id="2__35">
     </a>
     2. 检查设备连接状态
    </h5>
    <ol>
     <li>
      在电脑上，打开设备管理器（
      <code>
       Windows
      </code>
      +
      <code>
       X
      </code>
      或
      <code>
       设备管理器
      </code>
      ）。
     </li>
     <li>
      确保安卓设备已正确连接并识别。如果未识别，可以尝试更换 USB 端口或数据线。
     </li>
    </ol>
    <h4>
     <a id="_Unity__Unity_Remote_5_40">
     </a>
     步骤三：在 Unity 中配置 Unity Remote 5
    </h4>
    <h5>
     <a id="1__Unity__42">
     </a>
     1. 打开 Unity 项目
    </h5>
    <ol>
     <li>
      启动 Unity 编辑器。
     </li>
     <li>
      打开你想要测试的 Unity 项目。
     </li>
    </ol>
    <h5>
     <a id="2__Unity_Remote_5_47">
     </a>
     2. 配置 Unity Remote 5
    </h5>
    <ol>
     <li>
      进入
      <code>
       Edit
      </code>
      &gt;
      <code>
       Project Settings
      </code>
      &gt;
      <code>
       Editor
      </code>
      。
     </li>
     <li>
      在
      <code>
       Unity Remote
      </code>
      部分，进行以下设置：
      <ul>
       <li>
        <code>
         Device
        </code>
        ：选择
        <code>
         Any Android Device
        </code>
        。
       </li>
       <li>
        <code>
         Compression
        </code>
        ：选择合适的压缩选项，例如
        <code>
         JPEG
        </code>
        以减少延迟。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_Unity_Remote_5_54">
     </a>
     步骤四：在安卓设备上启动 Unity Remote 5
    </h4>
    <h5>
     <a id="1__56">
     </a>
     1. 启动应用
    </h5>
    <ol>
     <li>
      <p>
       打开安卓设备上的 Unity Remote 5 应用。
      </p>
     </li>
     <li>
      <p>
       在应用内选择与你的 Unity 编辑器版本相对应的 Unity 版本（如果应用要求选择）。
      </p>
     </li>
     <li>
      <p>
       等待应用提示已连接。你可以在 Unity 编辑器的控制台中看到类似 “Connected to Android Device” 的信息。
      </p>
     </li>
    </ol>
    <h4>
     <a id="_Unity__64">
     </a>
     步骤五：运行并测试 Unity 场景
    </h4>
    <h5>
     <a id="1__66">
     </a>
     1. 选择并运行场景
    </h5>
    <ol>
     <li>
      在 Unity 编辑器中选择要测试的场景。
     </li>
     <li>
      点击
      <code>
       Play
      </code>
      按钮以运行场景。
     </li>
    </ol>
    <h5>
     <a id="2__71">
     </a>
     2. 通过安卓设备进行交互
    </h5>
    <ol>
     <li>
      安卓设备上将显示 Unity 编辑器中的场景。
     </li>
     <li>
      使用手机的触摸屏进行交互，模拟实际的手机操作。
     </li>
    </ol>
    <h4>
     <a id="_76">
     </a>
     步骤六：调试和优化
    </h4>
    <h5>
     <a id="1__78">
     </a>
     1. 实时调试
    </h5>
    <ol>
     <li>
      在 Unity 编辑器的控制台中查看日志和调试信息。
     </li>
     <li>
      观察安卓设备上的运行效果，并及时进行调整。
     </li>
    </ol>
    <h5>
     <a id="2__83">
     </a>
     2. 优化游戏性能
    </h5>
    <ol>
     <li>
      根据测试结果调整游戏设置，例如分辨率和图像质量。
     </li>
     <li>
      优化代码，减少复杂计算和资源占用。
     </li>
    </ol>
    <h4>
     <a id="_88">
     </a>
     常见问题和解决方法
    </h4>
    <h5>
     <a id="1__90">
     </a>
     1. 设备未连接
    </h5>
    <ul>
     <li>
      <strong>
       检查 USB 连接
      </strong>
      : 确保数据线完好且设备连接正确。尝试更换 USB 端口或数据线。
     </li>
     <li>
      <strong>
       检查开发者选项
      </strong>
      : 确保
      <code>
       USB 调试
      </code>
      已启用。
     </li>
     <li>
      <strong>
       安装驱动程序
      </strong>
      : 安装手机厂商提供的 USB 驱动程序，或者使用安卓 SDK Manager 安装。
     </li>
    </ul>
    <h5>
     <a id="2__96">
     </a>
     2. 图像质量低
    </h5>
    <ul>
     <li>
      <strong>
       调整压缩选项
      </strong>
      : 在
      <code>
       Project Settings
      </code>
      &gt;
      <code>
       Editor
      </code>
      &gt;
      <code>
       Unity Remote
      </code>
      中尝试不同的压缩选项（JPEG, PNG）。
     </li>
     <li>
      <strong>
       检查连接速度
      </strong>
      : 使用高速 USB 端口。
     </li>
    </ul>
    <h5>
     <a id="3__101">
     </a>
     3. 高延迟
    </h5>
    <ul>
     <li>
      <strong>
       简化场景
      </strong>
      : 减少模型和特效的数量。
     </li>
     <li>
      <strong>
       优化代码
      </strong>
      : 优化脚本以减少处理时间。
     </li>
    </ul>
    <h5>
     <a id="4__106">
     </a>
     4. 应用崩溃或无法启动
    </h5>
    <ul>
     <li>
      <strong>
       重新启动设备
      </strong>
      : 尝试重新启动安卓设备。
     </li>
     <li>
      <strong>
       重新安装应用
      </strong>
      : 卸载并重新安装 Unity Remote 5。
     </li>
    </ul>
