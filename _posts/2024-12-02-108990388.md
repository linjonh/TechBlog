    <h2>
     <a id="_18">
     </a>
     前言
    </h2>
    <p>
     在前面的文章中，我们已经配置了DevEco Studio的开发环境。下面我们就开始实际的HarmonyOS应用开发，当然~ 我们的一个HarmonyOS项目还是用我们所熟悉的HelloWorld来开头，这里我们以Wearable工程为例，在华为提供的Wearable远程模拟器中运行。
    </p>
    <h2>
     <a id="_22">
     </a>
     新建工程
    </h2>
    <ol>
     <li>
      <p>
       打开DevEco Studio，在欢迎页点击
       <strong>
        Create HarmonyOS Project
       </strong>
       ，创建一个新工程
      </p>
      <p>
       <img alt="image-20201009154020727" src="https://i-blog.csdnimg.cn/blog_migrate/48a95f4d6ada6cacf926408e788cfefb.png"/>
      </p>
     </li>
     <li>
      <p>
       选择设备类型和模板，以
       <strong>
        Wearable
       </strong>
       为例，选择
       <strong>
        Empty Feature Ability(Java)
       </strong>
       ，点击
       <strong>
        Next
       </strong>
       。
      </p>
      <p>
       <img alt="image-20201009154107643" src="https://i-blog.csdnimg.cn/blog_migrate/4198ef475d41e350cc52ab4b7d3a2ed2.png"/>
      </p>
     </li>
     <li>
      <p>
       填写项目相关信息
      </p>
      <p>
       <img alt="image-20201009154556600" src="https://i-blog.csdnimg.cn/blog_migrate/193b8cd9fab46958441098696dbb4118.png"/>
      </p>
     </li>
     <li>
      <p>
       工程创建完成后，DevEco Studio会自动进行工程的同步，同步成功如下图所示。HarmonysOS使用Java开发应用默认使用Gradle进行编译构建，首次创建工程需要下载Gradle相关依赖，时间较长，需要耐心等待
      </p>
      <p>
       <img alt="image-20201009163640455" src="https://i-blog.csdnimg.cn/blog_migrate/f932ba8fd4c483ff0727eb299b0ca9a3.png"/>
      </p>
     </li>
     <li>
      <p>
       一个创建好的Java项目的目录结构如下图：
      </p>
      <p>
       <strong>
        .gradle
       </strong>
       ：Gradle配置文件，由系统自动生成，一般情况下不需要进行修改。
      </p>
      <ul>
       <li>
        <strong>
         entry
        </strong>
        ：默认启动模块（主模块），开发者用于编写源码文件以及开发资源文件的目录。
       </li>
       <li>
        <strong>
         entry&gt;libs
        </strong>
        ：用于存放entry模块的依赖文件。
       </li>
       <li>
        <strong>
         entry&gt;.gitignore
        </strong>
        ：标识git版本管理需要忽略的文件。
       </li>
       <li>
        <strong>
         entry&gt;build.gradle
        </strong>
        ：entry模块的编译配置文件。
       </li>
       <li>
        <strong>
         entry&gt;src&gt;main&gt;Java
        </strong>
        ：用于存放Java源码。
       </li>
       <li>
        <strong>
         entry&gt;src&gt;main&gt;resources
        </strong>
        ：用于存放资源文件。
       </li>
       <li>
        <strong>
         entry&gt;src&gt;main&gt;config.json
        </strong>
        ：HAP清单文件，配置信息
       </li>
       <li>
        <strong>
         entry&gt;src&gt;test
        </strong>
        ：编写测试文件的目录。
       </li>
      </ul>
      <p>
       <img alt="image-20201009170832351" src="https://i-blog.csdnimg.cn/blog_migrate/dd1c975d42cf6d4b3676e99383dc012f.png"/>
      </p>
     </li>
    </ol>
    <h2>
     <a id="_57">
     </a>
     运行应用
    </h2>
    <ol>
     <li>
      <p>
       在DevEco Studio菜单栏，点击
       <strong>
        Tools &gt; HVD Manager
       </strong>
       。首次使用模拟器，需下载模拟器相关资源，点击
       <strong>
        OK
       </strong>
       ，等待资源下载完成后，点击模拟器界面左下角的
       <strong>
        Refresh
       </strong>
       按钮。
      </p>
      <p>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/86ec2205d8a07aa3425241f1a30c213d.png"/>
      </p>
      <p>
       <img alt="image-20201009165815735" src="https://i-blog.csdnimg.cn/blog_migrate/bf7c3fcadfe67049bdaccac43ea38f93.png"/>
      </p>
      <p>
       <img alt="image-20201009165856263" src="https://i-blog.csdnimg.cn/blog_migrate/ce13beaff69ef163fa620e8e27e7bd4e.png"/>
      </p>
     </li>
     <li>
      <p>
       然后在浏览器会弹出华为账号登陆界面，输入已经实名认证过的账户和密码进行登录
      </p>
     </li>
     <li>
      <p>
       登录后，点击界面的
       <strong>
        允许
       </strong>
       按钮进行授权
      </p>
      <p>
       <img alt="image-20201009170031309" src="https://i-blog.csdnimg.cn/blog_migrate/9b98bdefad11912edb6e9f07e3c504bb.png"/>
      </p>
     </li>
     <li>
      <p>
       在设备列表中，选择Wearable设备，并点击三角形按钮，运行模拟器
      </p>
      <p>
       <img alt="image-20201009170149141" src="https://i-blog.csdnimg.cn/blog_migrate/a6f696557449fab7530ae0ae62515839.png"/>
      </p>
     </li>
     <li>
      <p>
       点击DevEco Studio工具栏中的三角形按钮运行工程，或使用默认快捷键Shift+F10运行
      </p>
      <p>
       <img alt="image-20201009170420195" src="https://i-blog.csdnimg.cn/blog_migrate/19e7e198ef76c197d4c9bf1a34f1a251.png"/>
      </p>
     </li>
     <li>
      <p>
       DevEco Studio会启动应用的编译构建，完成后应用即可运行在
       <strong>
        Remote Device
       </strong>
       上
      </p>
      <p>
       <img alt="image-20201009170546553" src="https://i-blog.csdnimg.cn/blog_migrate/f896fabdbcef31ffd05266289421fbfd.png"/>
      </p>
     </li>
    </ol>
