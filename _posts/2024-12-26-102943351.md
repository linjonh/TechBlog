     方法一（网速好时用）：
    </h4>
    <p>
     sudo pip3 install opencv-contrib-python==3.4.2.16
    </p>
    <h4>
     <a id="_4">
     </a>
     方法二（离线包方法,更靠谱）：
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        通过浏览器下载离线包：
       </strong>
       <br/>
       网址：终端运行sudo pip3 install opencv-contrib-python==3.4.2.16时可得到网址.
      </p>
     </li>
     <li>
      <p>
       <strong>
        在离线包所在路径下，执行终端命令：
       </strong>
      </p>
     </li>
    </ol>
    <pre><code>pip3 install opencv_contrib_python-3.4.2.16-cp36-cp36m-manylinux1_x86_64.whl
</code></pre>
    <ol start="3">
     <li>
      <p>
       <strong>
        检验安装结果：
       </strong>
       <br/>
       （1）python3
       <br/>
       （2）import cv2
       <br/>
       （3）print(cv2.__version) # 3.4.2
      </p>
     </li>
     <li>
      <p>
       <strong>
        卸载：
       </strong>
       <br/>
       终端执行
      </p>
     </li>
    </ol>
    <pre><code>pip3 uninstall opencv
</code></pre>
