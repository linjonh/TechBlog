     window系统下，如何使用sqlcipher工具对已加密数据库解密
    </h2>
    <h4>
     <a id="_1">
     </a>
     需求背景
    </h4>
    <p>
     <font size="4">
      目前我项目里数据库是使用sqlcipher加密的，本文介绍如何使用sqlcipher工具进行解密，已给测试人员以及其他开发人员使用。
     </font>
    </p>
    <h4>
     <a id="_4">
     </a>
     使用工具
    </h4>
    <p>
     <a href="https://pan.baidu.com/s/1wDF6FTR7iDJHP6_rbcjeCw" rel="nofollow">
      工具下载地址，提取码59jq
     </a>
    </p>
    <h4>
     <a id="_6">
     </a>
     具体解密操作
    </h4>
    <p>
     1.解压下载好的工具
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a873e6f02d8364769799f6147fac77c.png">
      <br/>
      2.打开cmd命令工具进入到bin目录下
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a41edd07235d772e2d39df56552ac00.png">
       <br/>
       3.解密已有的加密数据库，就以项目的universal_encrypt.db为例，执行
       <br/>
       sqlcipher-shell64.exe universal_encrypt.db 进入到sqlite中
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3cb2a0f496b5901d455b8846ce1004fb.png">
        <br/>
        4.对数据库进行解密,执行以下四部操作
       </img>
      </img>
     </img>
    </p>
    <pre><code class="prism language-javascript">     <span class="token number">1.</span><span class="token constant">PRAGMA</span> key <span class="token operator">=</span> <span class="token string">'123456'</span><span class="token punctuation">;</span>
     <span class="token number">2.</span><span class="token constant">ATTACH</span> <span class="token constant">DATABASE</span> <span class="token string">'plaintext.db'</span> <span class="token constant">AS</span> plaintext <span class="token constant">KEY</span> <span class="token string">''</span><span class="token punctuation">;</span>
     <span class="token number">3.</span><span class="token constant">SELECT</span> <span class="token function">sqlcipher_export</span><span class="token punctuation">(</span><span class="token string">'plaintext'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
     <span class="token number">4.</span><span class="token constant">DETACH</span> <span class="token constant">DATABASE</span> plaintext
</code></pre>
    <p>
     5.解密完成，这个plaintext.db就是解密好的数据库
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cbb4802b7b46ece6a4b8795769f685f5.png">

     </img>
    </p>
    <h4>
     <a id="_23">
     </a>
     其他使用
    </h4>
    <p>
     <strong>
      1.创建数据库
     </strong>
    </p>
    <pre><code> 1.sqlcipher-shell64.exe  encrypted .db
 2.PRAGMA key = 'password';
 3.create table encrypted (id integer, name text);
 4. .schema
     CREATE TABLE encrypted (id integer, name text);
</code></pre>
    <p>
     <strong>
      2.对已有数据库进行加密
     </strong>
    </p>
    <pre><code> 1.sqlcipher-shell64.exe  universal.db
 2.ATTACH DATABASE 'encrypted.db' AS encrypted KEY 'password';
 3.SELECT sqlcipher_export('encrypted');
 4.DETACH DATABASE encrypted;
</code></pre>
    <p>
     <strong>
      3.修改密码
     </strong>
     <br/>
     1.sqlite&gt; PRAGMA rekey = ‘newPassword’;
    </p>
    <p>
     <strong>
      4.打开加密的数据库
     </strong>
    </p>
    <p>
     1.sqlcipher-shell64.exe encrypted .db
     <br/>
     2.PRAGMA key = ‘password’;
     <br/>
     3. .schema
    </p>
