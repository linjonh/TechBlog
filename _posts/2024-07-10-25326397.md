     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">

     </span>
     <span style="color:#555555">
      Web
     </span>
     <span style="color:#555555">
      开发者来说要如何站在
      <a href="http://www.chinabyte.com/keyword/%E7%A7%BB%E5%8A%A8%E4%BA%92%E8%81%94%E7%BD%91/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        移动互联网
       </span>
      </a>
     </span>
     的
     <a href="http://www.chinabyte.com/keyword/%E6%B5%AA%E6%BD%AE/" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#333333">
       浪潮
      </span>
     </a>
     之巅呢？是选择学习原生开发，研究
     <span style="color:#555555">
      Java
     </span>
     <span style="color:#555555">
      、
     </span>
     <span style="color:#555555">
      Object-C
     </span>
     <span style="color:#555555">
      、
     </span>
     <span style="color:#555555">
      C#
     </span>
     <span style="color:#555555">
      等语言，还是选择继续使用网页开发，容忍
     </span>
     <span style="color:#555555">
      HTML5
     </span>
     <span style="color:#555555">
      功能的局限性？就在开发者左右为难的情况下
     </span>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      作为一个折中的解决
      <a href="http://solution.chinabyte.com/new/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        方案
       </span>
      </a>
     </span>

     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      呢？
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#555555">
       HybridApp
      </span>
      <span style="color:#555555">
       概念
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      ：
     </span>
     <span style="color:#555555">
      Hybrid App is amobile application that is coded in both browser-supported language andcomputer language. They are available through application distributionplatforms such as the Apple
      <a href="http://www.chinabyte.com/keyword/App+Store/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
       </span>
       <span style="color:#333333">
        App Store
       </span>
      </a>
      ,
      <a href="http://com.chinabyte.com/google/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        Google
       </span>
      </a>
      <a href="http://www.chinabyte.com/keyword/Play/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
       </span>
       <span style="color:#333333">
        Play
       </span>
      </a>
      etc.Usually, they are downloaded from the platform to a target device, such as
      <a href="http://www.chinabyte.com/keyword/iPhone/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
       </span>
       <span style="color:#333333">
        iPhone
       </span>
      </a>
      ,
      <a href="http://www.chinabyte.com/zq/android/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
       </span>
       <span style="color:#333333">
        Android
       </span>
      </a>
      phone or Windows Phone. Thesubscribers need to install to run them.
     </span>
    </p>
    <p>
     <span style="color:#555555">
     </span>
    </p>
    <p>
     <span style="color:#555555">
      总体来讲，
     </span>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      是同时采用网页语言与程序语言进行开发，通过不同的应用商店进行打包与分发，应用的特性更接近原生应用而且又区别与
     </span>
     <span style="color:#555555">
      Web
     </span>
     <span style="color:#555555">
      应用。但是在开发过程中同时使用了网页语言，所以开发成本与难度大大降低。也就是说
     </span>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      兼具了
     </span>
     <span style="color:#555555">
      Native App
     </span>
     <span style="color:#555555">
      与
     </span>
     <span style="color:#555555">
      Web App
     </span>
     <span style="color:#555555">
      两者的诸多优点。
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#555555">
       浅析
      </span>
      <span style="color:#555555">
       Hybrid App
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      主要以
     </span>
     <span style="color:#555555">
      JS+Native
     </span>
     <span style="color:#555555">
      两者相互调用为主，从开发层面实现“一次开发，多处运行”的机制，成为真正适合跨平台的开发。目前已经有众多
     </span>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      开发成功应用，比如
      <a href="http://com.chinabyte.com/baidu/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        百度
       </span>
      </a>
     </span>
     、
     <a href="http://com.chinabyte.com/netease/" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:#333333">
       网易
      </span>
     </a>
     、街旁等知名移动应用，都是采用
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      开发模式。
     </span>
    </p>
    <p>
     <span style="color:#555555">
     </span>
    </p>
    <p>
     <span style="color:#555555">
      经过众多开发者与成功案例证明
     </span>
     <span style="color:#555555">
      Hybrid App
     </span>
     <span style="color:#555555">
      兼具了
     </span>
     <span style="color:#555555">
      Native App
     </span>
     <span style="color:#555555">
      的良好用户体验的优势，也兼具了
     </span>
     <span style="color:#555555">
      Web App
     </span>
     <span style="color:#555555">
      使用
     </span>
     <span style="color:#555555">
      HTML5
     </span>
     <span style="color:#555555">
      跨平台开发低成本的优势。现在有更多的开发者在面临移动平台的选择，所以在这里根据开发中各个平台的使用情况，针对现在主流的平台进行分析。
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#555555">
       主流移动平台分析
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#555555">
      HybridApp
     </span>
     <span style="color:#555555">
      开发，现阶段主流的平台包括
     </span>
     <span style="color:#555555">
      PhoneGap
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      AppCan
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      appMobi
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Titanium
     </span>
     <span style="color:#555555">
      等，它们基于
     </span>
     <span style="color:#555555">
      webkit
     </span>
     <span style="color:#555555">
      开源内核，使用
     </span>
     <span style="color:#555555">
      HTML5
     </span>
     <span style="color:#555555">
      标准开发，适配机型简单，支持开发者自定义插件，并能很好的应用于商业，教育，娱乐等行业，成为移动开发者的首选开发平台。
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#555555">
       PhoneGap
      </span>
      <span style="color:#555555">
       （
      </span>
      <span style="color:#555555">
       Cordova
      </span>
      <span style="color:#555555">
       ）
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#555555">
      产品功能：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      PhoneGap
     </span>
     <span style="color:#555555">
      是一款国外的开源移动开发平台。目前已经将核心代码贡献给
     </span>
     <span style="color:#555555">
      Apache cordova
     </span>
     <span style="color:#555555">
      ，最新版本是
     </span>
     <span style="color:#555555">
      2.6.0
     </span>
     <span style="color:#555555">
      ，它是基于
     </span>
     <span style="color:#555555">
      HTML
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      CSS
     </span>
     <span style="color:#555555">
      和
     </span>
     <span style="color:#555555">
      JavaScript
     </span>
     <span style="color:#555555">
      的，可以使用一些开源的框架比如
     </span>
     <span style="color:#555555">
      jQuery Mobile
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Dojo Mobile
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Sencha Touch
     </span>
     <span style="color:#555555">
      等等来提高用户体验，也提供了比较丰富的原生插件调用。
     </span>
    </p>
    <p>
     <span style="color:#555555">
     </span>
    </p>
    <p>
     <span style="color:#555555">
      特性：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      1.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      可以使用
     </span>
     <span style="color:#555555">
      DreamWeaver 5.5
     </span>
     <span style="color:#555555">
      编码，现在使用
     </span>
     <span style="color:#555555">
      appMobi
     </span>
     <span style="color:#555555">
      提供的
     </span>
     <span style="color:#555555">
      xdk
     </span>
     <span style="color:#555555">
      进行模拟器开发。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      2.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      代码开源，开发者可以放心使用。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      3.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      兼容性，一次开发，多处运行。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      4.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      使用
     </span>
     <span style="color:#555555">
      JS+HTML5
     </span>
     <span style="color:#555555">
      ，成本低。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      优点：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      1.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      Native
     </span>
     <span style="color:#555555">
      接口比较丰富，通过
      <a href="http://www.chinabyte.com/keyword/%E5%B0%81%E8%A3%85/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        封装
       </span>
      </a>
     </span>
     的
     <span style="color:#555555">
      API
     </span>
     <span style="color:#555555">
      可以直接访问硬件，比如说加速，相机，指南针，
     </span>
     <span style="color:#555555">
      GPS
     </span>
     <span style="color:#555555">
      ，文件访问等。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      2.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      接口文档描述非常详细。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      3.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      支持平台多，包括
     </span>
     <span style="color:#555555">
      iOS
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Android
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Blackberry
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Sym
      <a name="_GoBack" target="_blank">
      </a>
      bian
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      bada
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Windows Phone 7
     </span>
     <span style="color:#555555">
      ，
     </span>
     <span style="color:#555555">
      Windows Phone 8
     </span>
     <span style="color:#555555">
      等。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      缺点：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      1.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      需要针对相应的平台环境配置，进行编译，打包测试，发布等等。由于使用
     </span>
     <span style="color:#555555">
      Hybrid
     </span>
     <span style="color:#555555">
      开发的用户群，大部分是
     </span>
     <span style="color:#555555">
      web
     </span>
     <span style="color:#555555">
      开发者，对原生开发基本不了解，这无疑给每一个开发者增加了沉重的负担，需要对各个平台的开发都要需要了解，对硬件等等都要配置，加大开发成本。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      2.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      使用效果启动慢，页面切换响应慢，数据请求慢。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      3.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      文档虽比较详细但是基本是英文，对于国内大部分用户英文水平较差的是比较大的挑战。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      4.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      因为是国外的框架，技术支持不够到位，出现问题，无法排解，成为技术攻关的难点。
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#555555">
       AppCan
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#555555">
      产品功能：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      AppCan
     </span>
     <span style="color:#555555">
      是本土移动开发中使用最广的移动平台，网络舆论而言，
     </span>
     <span style="color:#555555">
      AppCan
     </span>
     <span style="color:#555555">
      是
     </span>
     <span style="color:#555555">
      PhoneGap
     </span>
     <span style="color:#555555">
      的中国化，但是从对
     </span>
     <span style="color:#555555">
      AppCan
     </span>
     <span style="color:#555555">
      实际使用，以及转向移动开发的朋友们互相交流反馈，他们是截然不同的两个移动平台，
     </span>
     <span style="color:#555555">
      AppCan
     </span>
     <span style="color:#555555">
      不仅封装了类似于
     </span>
     <span style="color:#555555">
      PhoneGap
     </span>
     <span style="color:#555555">
      的本地调用功能，而且封装了
     </span>
     <span style="color:#555555">
      uexWindow
     </span>
     <span style="color:#555555">
      多窗口机制，实现了移动端的
     </span>
     <span style="color:#555555">
      iframe
     </span>
     <span style="color:#555555">
      效果，虽然不是开源项目，但一直都有面向开发者的免费版，并且也有定位于
      <a href="http://com.chinabyte.com/" rel="nofollow noopener noreferrer" target="_blank">
       <span style="color:#333333">
        企业
       </span>
      </a>
     </span>
     用户的企业版套装，目前最新版本为
     <span style="color:#555555">
      2.2.X
     </span>
     <span style="color:#555555">
      。
     </span>
    </p>
    <p>
     <span style="color:#555555">
     </span>
    </p>
    <p>
     <span style="color:#555555">
      特性：
     </span>
    </p>
    <p>
     <span style="color:#555555">
      1.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      提供的集成开发环境的
     </span>
     <span style="color:#555555">
      IDE
     </span>
     <span style="color:#555555">
      进行模拟器开发。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      2.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      兼容性，一次开发，多处运行。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      3.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      使用
     </span>
     <span style="color:#555555">
      JS+HTML5
     </span>
     <span style="color:#555555">
      ，成本低。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      4.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
      在线打包。
     </span>
    </p>
    <p>
     <span style="color:#555555">
      5.
     </span>
     <span style="color:#555555">
     </span>
     <span style="color:#555555">
     </span>
