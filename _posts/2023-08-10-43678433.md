     安装配置Thrift
     <br/>
     Thrift的编译器使用C++编写的，在安装编译器之前，首先应该保证操作系统基本环境支持C++的编译，安装相关依赖的软件包，如下所示
     <br/>
    </p>
    <pre><code class="language-html">sudo yum install automake libtool flex bison pkgconfig gcc-c++ boost-devel libevent-devel zlib-devel python-devel ruby-devel openssl-devel</code></pre>
    <br/>
    下载Thrift的软件包，并解压缩：
    <br/>
    <p>
     wget http://mirrors.cnnic.cn/apache/thrift/0.9.0/thrift-0.9.0.tar.gz
    </p>
    <p>
     <br/>
    </p>
    <span style="background-color:rgb(240,240,240)">
     tar -xvzf thrift-0.9.0.tar.gz
    </span>
    <br/>
    配置、编译、安装Thrift，如下所示：
    <br/>
    <br/>
    <pre><code class="language-html">sudo ./configure
sudo make
sudo make install</code></pre>
    <br/>
    <br/>
    <p>
     如果在配置的时候报错，请参考官方安装指导
    </p>
    <p>
     <a href="http://thrift.apache.org/docs/install/centos" rel="nofollow noopener noreferrer" target="_blank">
      http://thrift.apache.org/docs/install/centos
     </a>
     <br/>
    </p>
