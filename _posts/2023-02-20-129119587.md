    <p>
     在linux系统上安装了Python后，Python自带了一个WEB服务器 SimpleHTTPServer
     <br/>
     可以使用 python -m SimpleHTTPServer 快速搭建一个http服务，提供一个文件浏览的web服务。
    </p>
    <p>
     注意：
     <br/>
     SimpleHTTPServer是Python 2自带的一个模块，是Python的Web服务器。它在Python 3已经合并到http.server模块中。SimpleHTTPServer在Python 3的用法与在Python 2的用法相似(python3 -m http.server 6789), 本文以Python 2为例。
     <br/>
     ————————————————
    </p>
    <h2>
     <a id="SimpleHTTPServer_6">
     </a>
     SimpleHTTPServer使用方法
    </h2>
    <ol>
     <li>
      进入待分享的目录
     </li>
     <li>
      执行命令python -m SimpleHTTPServer 端口号。 注意：不填端口号则默认使用8000端口
     </li>
     <li>
      浏览器访问该主机的地址：http://IP:端口号/
     </li>
    </ol>
    <p>
     执行命令如下：
    </p>
    <pre><code class="prism language-sql">cd <span class="token keyword">data</span><span class="token operator">/</span>logs<span class="token operator">/</span>fulfillment<span class="token operator">-</span>service
lsof <span class="token operator">-</span>i:<span class="token number">8000</span><span class="token operator">|</span>grep <span class="token operator">-</span>v PID<span class="token operator">|</span>awk <span class="token string">'{print $2}'</span><span class="token operator">|</span>xargs <span class="token operator">-</span>r <span class="token keyword">kill</span> <span class="token operator">-</span><span class="token number">9</span>
<span class="token comment"># python -m SimpleHTTPServer 8000        #把当前目录发布到8000端口</span>
nohup python <span class="token operator">-</span>m SimpleHTTPServer <span class="token number">8000</span> <span class="token operator">&amp;</span>  <span class="token comment">#nohup忽略所有的挂断信号，防止关闭bash或者ctrl c kill掉进程</span>
</code></pre>
    <p>
     浏览器打开，可以看到如下内容：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d14285a02921ffd7f48d1f62f8a4f68.png">
      <br/>
      注意：使用时还要注意Linux的防火墙因素。
      <br/>
      当服务器没有开通web访问权限时，可以使用curl命令访问一下本服务器的8000端口
      <br/>
      例如，curl 192.168.2.67:8000
