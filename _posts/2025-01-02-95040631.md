    <h2>
     <a id="word_2">
     </a>
     word内置了公式编辑器
    </h2>
    <ul>
     <li>
      选择“插入”→“公式”可进入到公式编辑的页面，或者直接按快捷键“Alt加上+”
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cc45f067d4b953e70eec46b75dfb24e7.png"/>
     </li>
     <li>
      用于编辑一些短的公式符号还凑合，但公式一旦复杂就显得麻烦了
     </li>
    </ul>
    <h2>
     <a id="LaTeXword_6">
     </a>
     LaTeX格式的数学公式怎么写到word里？
    </h2>
    <p>
     经过对比，这里推荐用在线Markdown编辑器写好数学公式，然后转换格式写入word里面。有两款编辑器可供参考
     <a href="https://www.zybuluo.com/mdeditor#1509100" rel="nofollow">
      作业部落
     </a>
     和
     <a href="https://stackedit.io/app#" rel="nofollow">
      StackEdit
     </a>
     ，由于自己使用StackEdit点不出“Show Math As→MathML Code”，故这里以作业部落为例，StackEdit如果可以的话方法类似。
    </p>
    <ul>
     <li>
      选择“文件”→“新文稿”，创建一个空白页，在左侧编辑区输入LaTeX公式。
     </li>
     <li>
      在右侧的预览窗口右键单击公式，选择
      <code>
       Show Math As→MathML Code
      </code>
      ，将MathML格式的公式复制下来。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7f5cb9e79b22ae62c5de14ae0cca9f20.png">
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/411933913511fd5e1887ef98984d0ff4.png"/>
      </img>
     </li>
     <li>
      打开word，右键单击要插入公式的位置，选择“只保留文本”。这个例子有点瑕疵，稍微修改一下就好了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fbb6136c7407e79f1cbef7118c8a7bb6.png"/>
     </li>
    </ul>
    <h2>
     <a id="LaTeX_13">
     </a>
     问题来了，怎么得到LaTeX格式的数学公式呢？
    </h2>
    <ul>
     <li>
      这里建议安装一个
      <a href="https://mathpix.com/" rel="nofollow">
       Mathpix
      </a>
      ，他可以识别图片上的数学公式，并且转化为LaTeX格式。图片来源就可以很丰富了，比如论文、手写公式、网络图片等。
     </li>
     <li>
      以手写公式为例
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4e395b8d12e4d75bba1343fcf1c47575.png"/>
     </li>
     <li>
      安装了Mathpix以后，按快捷键“Ctrl+Alt+M”（windows系统）会出现一个截图的框，选中要转化的公式即可
     </li>
     <li>
      生成公式后，点击“copy”即可复制到剪切板，粘贴到markdown编辑器中即可（一般选第2个或者第3个copy）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e7ce927b6499db9221741e79ed712d2b.png"/>
     </li>
    </ul>
