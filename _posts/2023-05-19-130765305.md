    </h1>

  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     1.硬件准备
    </p>
    <div>
     <span style="color:#000000;">
      （1）正点原子 ATK-ESP-01 WIFI 模块
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （2）正点原子 STM32F103ZET6精英板子
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （3）USB转TTL模块
     </span>
    </div>
    <div>
     2.烧录固件
    </div>
    <div>
     （1）烧录软件和固件都可以在正点原子增值资料包找到。
    </div>
    <div>
    </div>
    <div>
     <img alt="" height="120" src="https://i-blog.csdnimg.cn/blog_migrate/d22aee7abc285acb90e5d0a8858490fa.png" width="316">
     </img>
    </div>
    <div>
    </div>
    <div>
     （2）烧录注意事项
    </div>
    <ul>
     <li>
      串口波特率要选择正确：115200
     </li>
     <li>
      IO_0引脚接地为下载模式；悬空为运行模式
     </li>
     <li>
      点击START后，拔插一下VCC或者硬件的RST引脚接地复位
     </li>
     <li>
      下载完数据后，要记得拔掉IO_0引脚，然后RST引脚接地复位，退出下载模式
     </li>
    </ul>
    <p>
     3.
     <span style="color:#0d0016;">
      ATK_ESP8266 模块与串口调试助手通讯。
     </span>
    </p>
    <div>
     <span style="color:#000000;">
      测试模块在 STA
     </span>
     <span style="color:#000000;">
      模式下的透传设置：
     </span>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#000000;">
     </span>
     <span style="color:#fe2c24;">
      恢复出厂设置：由于ATK_ESP8266 模块可能之前烧录过其他固件，所以可以恢复一下出厂设置。（很重要）：AT+RESTORE。
     </span>
     <span style="color:#000000;">
     </span>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#000000;">
      1
     </span>
     <span style="color:#000000;">
      ） 设置
     </span>
     <span style="color:#000000;">
      WIFI
     </span>
     <span style="color:#000000;">
      模式：
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      AT+CWMODE=1 //
     </span>
     <span style="color:#000000;">
      设置为
     </span>
     <span style="color:#000000;">
      STA
     </span>
     <span style="color:#000000;">
      模式
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      响应
     </span>
     <span style="color:#000000;">
      :OK
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      2
     </span>
     <span style="color:#000000;">
      ） 重启生效
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      AT+RST
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      响应
     </span>
     <span style="color:#000000;">
      :OK
     </span>
    </div>
    <div>
     <img alt="" height="689" src="https://i-blog.csdnimg.cn/blog_migrate/681dea312a3d7a78d5d4a41d4e0c60e8.png" width="1200">
     </img>
    </div>
    <div>
     <span style="color:#000000;">
      3
     </span>
     <span style="color:#000000;">
      ） 连接路由器
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      AT+CWJAP=”LH”,”lianhaokeji” //
     </span>
     <span style="color:#000000;">
      连接对象路由器的
     </span>
     <span style="color:#000000;">
      ssid
     </span>
     <span style="color:#000000;">
      和
     </span>
     <span style="color:#000000;">
      password
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      响应
     </span>
     <span style="color:#000000;">
      :OK
     </span>
    </div>
    <div>
    </div>
    <div>
     <img alt="" height="195" src="https://i-blog.csdnimg.cn/blog_migrate/ee9b80861103a8763fe4f53e1eee3290.png" width="833">
     </img>
    </div>
    <p>
     <strong>
      要出现“WIFI GOT IP  OK”，说明ESP8266模块成功连接上路由器 。
     </strong>
    </p>
    <p>
     <strong>
      如果出现“+CWJAP:+3 FAIL”，说明网络可能不是2.4G频段
     </strong>
    </p>
    <div>
     如果出现“+CWJAP :+1  FAIL”，说明固件可能烧录出错，要重新烧录固件，恢复出厂设置一下
    </div>
    <div>
    </div>
    <div>
     <span style="color:#000000;">
      4
     </span>
     <span style="color:#000000;">
      ） 查询模块
     </span>
     <span style="color:#000000;">
      IP
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      AT+CIFSR
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      响应:
     </span>
     “192.168.xx.xxx”
    </div>
    <div>
    </div>
    <div>
     如果STAIP出现“2.0.255.255”这样或者其他，可以恢复出厂设置一下。
    </div>
    <div>
     如果STAIP出现出现“192.168.xx.xxx”，才能说明模块正确分配到IP地址。
    </div>
    <div>
     5）
     <span style="color:#000000;">
      在电脑上面使用网络调试助手，创建一个
     </span>
     <span style="color:#000000;">
      server
     </span>
     <span style="color:#000000;">
      ，如图
     </span>
    </div>
    <div>
     <img alt="" height="857" src="https://i-blog.csdnimg.cn/blog_migrate/a9f001d52e3ec42a761855740c0ef5b3.png" width="1004">
     </img>
    </div>
    <div>
     <div>
      <span style="color:#000000;">
       6）模块连接到
      </span>
      <span style="color:#000000;">
       server
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       AT+CIPSTART="TCP","192.168.31.154",8086  //填写正确的IP地址和端口
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       响应
      </span>
      <span style="color:#000000;">
       :CONNECT
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       OK
      </span>
     </div>
     <div>
      <div>
       <span style="color:#000000;">
        7
       </span>
       <span style="color:#000000;">
        ） 开启透传模式
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        AT+CIPMODE=1
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        响应
       </span>
       <span style="color:#000000;">
        :OK
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        8
       </span>
       <span style="color:#000000;">
        ） 开始透传
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        AT+CIPSEND
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        响应
       </span>
       <span style="color:#000000;">
        :OK
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        &gt;
       </span>
      </div>
      <div>
       <img alt="" height="652" src="https://i-blog.csdnimg.cn/blog_migrate/71a9ac0ebf0b12f94c467628fa132965.png" width="1200">
       </img>
      </div>
      <p>
      </p>
      <div>
       <span style="color:#000000;">
        9
       </span>
       <span style="color:#000000;">
        ） 退出透传模式
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        在透传模式中，若识别到单独的一包数据“
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        +++
       </span>
       <span style="color:#000000;">
        ”，则退出透传模式
       </span>
      </div>
      <div>
       <span style="color:#000000;">
        4.
       </span>
       TLink 云平台创建
      </div>
      <div>
       （1）登录
       <a href="https://www.tlink.io/index.htm" rel="nofollow" title="Tlink物联网-工业物联网平台">
        Tlink物联网-工业物联网平台
       </a>
       ，注册TLink账号
      </div>
      <div>
       （2）创建设备
      </div>
      <div>
       <img alt="" height="486" src="https://i-blog.csdnimg.cn/blog_migrate/28bf814ddee80e4dd2a1e2c0ea27ffcc.png" width="1200">
       </img>
      </div>

      <div>
       （3）连接设备
      </div>

       <li>
        <span style="color:#fe2c24;">
         编辑协议：十分重要，要跟STM32代码的协议要一致。注意看协议说明
        </span>
       </li>
       <li>
        代码里面的协议是“"FM:%d,%d,%d,%d,%d.#"”
