    <h4>
     <a id="2_6">
     </a>
     2、使用方式
    </h4>
    <h5>
     <a id="_7">
     </a>
     安装完成后的使用步骤
    </h5>
    <ul>
     <li>
      打开小程序开发者工具，用微信扫码登录开发者工具点击左侧
     </li>
     <li>
      菜单中的小程序选项
     </li>
     <li>
      点击+号新建小程序项目
     </li>
     <li>
      填写项目名称
     </li>
     <li>
      选择项目存放路径（必须选择空目录）
     </li>
     <li>
      填写AppID（AppID是在微信公众平台进行申请的）
     </li>
     <li>
      点击新建按钮
     </li>
    </ul>
