    <h2>

    <p>
     Mybatis一个基于Java的持久层框架，它通过XML或注解的方式，将SQL语句和Java方法进行映射，使得开发者可以轻松地进行数据库操作。下面我会演示mybatis的配置与使用并实现数据库的增删改查。

    <h2>
     <a id="3mybatis_10">
     </a>
     3.导入mybatis依赖
    </h2>
    <p>
     <strong>
      如果在创建项目时已导入依赖可以跳过这一步。
     </strong>
     <br/>
     在
     <code>
      pom.xml
     </code>
     文件中导入mybatis依赖：
    </p>
    <pre><code>&lt;dependency&gt;
    &lt;groupId&gt;org.mybatis.spring.boot&lt;/groupId&gt;
    &lt;artifactId&gt;mybatis-spring-boot-starter-test&lt;/artifactId&gt;
    &lt;version&gt;3.0.3&lt;/version&gt;
    &lt;scope&gt;test&lt;/scope&gt;
&lt;/dependency&gt;
</code></pre>
    <p>
     导入
     <code>
      sql server
     </code>
     的依赖：
    </p>
    <pre><code>&lt;dependency&gt;
    &lt;groupId&gt;com.microsoft.sqlserver&lt;/groupId&gt;
    &lt;artifactId&gt;mssql-jdbc&lt;/artifactId&gt;
    &lt;scope&gt;runtime&lt;/scope&gt;
&lt;/dependency&gt;
</code></pre>
    <p>
     <strong>
      如果想要使用其他版本的mybatis依赖，请前往
      <code>
       https://mvnrepository.com/
      </code>
     </strong>
    </p>
    <h2>
     <a id="4mybatis_31">
     </a>
     4.如何使用mybatis
    </h2>
    <h3>
     <a id="41_mybatis_32">
     </a>
     4.1 配置mybatis
    </h3>
    <h4>
     <a id="411__33">
     </a>
     4.1.1 配置数据源
    </h4>
    <p>
     在
     <code>
      application.properties
     </code>
     中配置数据库：
    </p>
    <pre><code>spring.datasource.driver-class-name=com.microsoft.sqlserver.jdbc.SQLServerDriver
spring.datasource.url=jdbc:sqlserver://localhost:1433;database=数据库名称
spring.datasource.username=sa
spring.security.user.password=123456
</code></pre>
    <p>
     也可以在
     <code>
      application.yaml
     </code>
     或
     <code>
      application.yml
     </code>
     文件中配置数据库：
    </p>
    <pre><code>spring:
  datasource:
    driver-class-name: com.microsoft.sqlserver.jdbc.SQLServerDriver
    url: jdbc:sqlserver://localhost:1433;database=数据库名称
    username: sa
    password: 123456
</code></pre>
    <h4>
     <a id="412__50">
     </a>
     4.1.2 相关配置
    </h4>
    <pre><code>#配置映射文件路径（使用xml配置文件映射SQL语句时设置）,此处的mapper是自己在resources目录下创建的子目录
mybatis.mapper-locations=classpath:/mapper/*.xml
</code></pre>
    <h3>
     <a id="42_mybatis_55">
     </a>
     4,2 使用mybatis

    <h4>
     <a id="421_entity_56">

      <code>

public class Person {

    private String name;
    private int age;
    private String gender;
}

     <a id="_xml_81">

     ① xml形式

    <p>
     在方法名处按下

      alt + enter

     后选择
     <code>

     ② 注解形式

    <h4>
     <a id="424_controllermapper_93">

     4.2.4 创建controller并调用mapper
    </h4>
    <p>
     在启动类的所在目录下创建controller：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fd6295477c727d1c79fb0655f474c2b1.png"/>

     中：

    <pre><code>@RestController

    <h2>
     <a id="5_117">
     </a>
     5.实现数据库的增删改查
    </h2>
    <h3>
     <a id="51__118">
     </a>
     5.1 查询
    </h3>
    <p>
     上面演示了获取全部person数据，这里举例
     <code>
      通过id查询数据
     </code>
     ，我们可以模仿这个例子实现其他方式的查询。
     <br/>
     定义查询方法并编写SQL语句
    </p>
    <pre><code>//使用@Param("id")接收前端传来的参数
@Select("select * from person where id = #{id}")
Person getPersonById(@Param("id") int id);
</code></pre>
    <p>
     调用：
    </p>
    <pre><code>@GetMapping("/byId/{id}")
    public Person getPersonById(@PathVariable("id") int id) {
        return personMapper.getPersonById(id);
    }
</code></pre>
    <p>
     结果：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0db86152504e4e8814613ddf269039f9.png"/>
    </p>
    <h3>
     <a id="52__135">
     </a>
     5.2 新增
    </h3>
    <p>
     定义新增方法并编写SQL语句：
    </p>
    <pre><code>//由于id设置了自增，所以这里不需要传入id
@Insert("insert into person values (#{name},#{age},#{gender})")
void addPerson(Person person);
</code></pre>
    <p>
     调用：
    </p>
    <pre><code>@GetMapping("/add")
    public void addPerson(@RequestParam("name") String name, 
                          @RequestParam("age") int age,
                          @RequestParam("gender") String gender) {
        Person person = new Person();
        person.setName(name);
        person.setAge(age);
        person.setGender(gender);
        personMapper.addPerson(person);
    }
</code></pre>
    <p>
     使用：

    <h3>
     <a id="53__159">
     </a>
     5.3 修改
    </h3>
    <p>
     <strong>
      这里我通过id修改person的信息，我们可以模仿这个例子通过名字来修改person信息，也可以修改person一个或多个字段信息。
     </strong>
     <br/>
     定义修改方法并编写SQL语句：
    </p>
    <pre><code>//通过id修改person信息
@Update("update person set name = #{name}, age = #{age}, gender = #{gender} where id = #{id}")
void updatePerson(Person person);
</code></pre>
    <p>
     调用：
    </p>
    <pre><code>@GetMapping("/update")
    void updatePerson(@RequestParam("id") int id,
                      @RequestParam("name") String name,
                      @RequestParam("age") int age,
                      @RequestParam("gender") String gender) {
        Person person = new Person();
        person.setId(id);
        person.setName(name);
        person.setAge(age);
        person.setGender(gender);
        personMapper.updatePerson(person);
    } 
</code></pre>

     5.4 删除

    <pre><code>@GetMapping("/del")
