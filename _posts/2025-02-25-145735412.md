    <h3>
     <a id="__One_API_Docker___92">
     </a>
     <strong>
      🚀 快速部署 One API（支持 Docker &amp; 手动安装）
     </strong>
    </h3>
    <h4>
     <a id="1_Docker__94">
     </a>
     <strong>
      1️⃣ Docker 一键部署（推荐）
     </strong>
    </h4>
    <pre><code>docker run --name one-api -d --restart always -p 3000:3000 \
    -e TZ=Asia/Shanghai \
    -v /home/ubuntu/data/one-api:/data \
    justsong/one-api
</code></pre>
    <p>
     然后访问
     <code>
      http://localhost:3000
     </code>
     进行配置。
    </p>
    <h4>
     <a id="2__105">
     </a>
     <strong>
      2️⃣ 手动部署（适用于服务器）
     </strong>
    </h4>
    <pre><code>git clone https://github.com/songquanpeng/one-api.git
cd one-api
go build -o one-api
./one-api
</code></pre>
    <p>
     默认 One API 运行在
     <code>
      http://localhost:3000
     </code>
     。
    </p>
    <h4>
     <a id="3__OpenAI_API__116">
     </a>
     <strong>
      3️⃣ 配置 OpenAI API 兼容模式
     </strong>
    </h4>
    <pre><code>export OPENAI_API_BASE=https://your-one-api.com/v1
</code></pre>
    <h4>
     <a id="4_Nginx__122">
     </a>
     <strong>
      4️⃣ Nginx 反向代理（可选）
     </strong>
    </h4>
    <pre><code>server {
    server_name api.yourdomain.com;
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-For $remote_addr;
    }
}
</code></pre>
