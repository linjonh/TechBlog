      </span>
     </p>
     <h3 class="md-end-block md-heading" style="font-size:1.2em;font-weight:bold;line-height:1.225;color:rgb(62,192,131);border-bottom:5px solid rgb(62,192,131);width:inherit;">
      开源框架
     </h3>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        1、 mpvue
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       mpvue 是美团点评开源的一个使用 Vue.js 开发小程序的前端框架。框架基于 Vue.js 核心，mpvue 修改了 Vue.js 的  runtime 和  compiler 实现，使其可以运行在小程序环境中，从而为小程序开发引入了整套 Vue.js 开发体验。使用  mpvue 开发小程序，你将在小程序技术体系的基础上获取到这样一些能力：
      </span>
     </p>
     <ul class="ul-list list-paddingleft-2">
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         彻底的组件化开发能力：提高代码复用性
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         完整的 Vue.js 开发体验
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         方便的 Vuex 数据管理方案：方便构建复杂应用
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         快捷的 webpack 构建机制：自定义构建策略、开发阶段 hotReload
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         支持使用 npm 外部依赖
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         使用 Vue.js 命令行工具 vue-cli 快速初始化项目
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span class="md-line md-end-block">
         H5 代码转换编译成小程序目标代码的能力
        </span>
       </p>
      </li>
     </ul>
     <blockquote style="margin-left:1em;border-left-width:4px;border-left-color:rgb(62,192,131);color:rgb(136,136,136);font-size:15px;line-height:2;">
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        Github：https://github.com/Meituan-Dianping/mpvue
       </span>
      </p>
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        官网：http://mpvue.com/
       </span>
      </p>
     </blockquote>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        2、Tina.js
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       Tina.js 一款轻巧的渐进式微信小程序框架。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       特性： 轻盈小巧。 极易上手，保留 MINA (微信小程序官方框架) 的大部分 API 设计；无论你有无小程序开发经验，都可以轻松过渡上手。 渐进增强，既有状态管理器，也有路由增强，还可以自己编写插件。
      </span>
     </p>
     <blockquote style="margin-left:1em;border-left-width:4px;border-left-color:rgb(62,192,131);color:rgb(136,136,136);font-size:15px;line-height:2;">
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        Tina.js 开源框架地址：https://github.com/tinajs/tina
       </span>
      </p>
     </blockquote>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        3、Taro
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       Taro 是由京东 - 凹凸实验室打造的一套遵循 React 语法规范的多端统一开发框架。我要没记错的话，是最近刚刚开源的。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       使用 Taro，我们可以只书写一套代码，再通过 Taro 的编译工具，将源代码分别编译出可以在不同端（微信小程序、H5、App 端等）运行的代码。同时 Taro 还提供开箱即用的语法检测和自动补全等功能，有效地提升了开发体验和开发效率。
      </span>
     </p>
     <blockquote style="margin-left:1em;border-left-width:4px;border-left-color:rgb(62,192,131);color:rgb(136,136,136);font-size:15px;line-height:2;">
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        官网：http://taro.aotu.io/
       </span>
      </p>
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        GitHub: http://github.com/nervjs/taro
       </span>
      </p>
     </blockquote>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        4、wepy
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       WePY 是一款让小程序支持组件化开发的框架，通过预编译的手段让开发者可以选择自己喜欢的开发风格去开发小程序。框架的细节优化，Promise，Async Functions 的引入都是为了能让开发小程序项目变得更加简单，高效。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        特性：
       </strong>
      </span>
     </p>
     <ul class="ul-list list-paddingleft-2">
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         类 Vue 开发风格
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持自定义组件开发
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持引入 NPM 包
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持 Promise
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持 ES2015 + 特性，如 Async Functions
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持多种编译器，Less/Sass/Styus、Babel/Typescript、Pug
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持多种插件处理，文件压缩，图片压缩，内容替换等
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         支持 Sourcemap，ESLint 等
        </span>
       </p>
      </li>
      <li>
       <p style="color:rgb(79,79,79);width:inherit;">
        <span style="font-size:15px;">
         小程序细节优化，如请求列队，事件优化等
        </span>
        <span class="md-line md-end-block">
        </span>
       </p>
      </li>
     </ul>
     <blockquote style="margin-left:1em;border-left-width:4px;border-left-color:rgb(62,192,131);color:rgb(136,136,136);font-size:15px;line-height:2;">
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        Github ：https://github.com/Tencent/wepy
       </span>
      </p>
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        官网：https://tencent.github.io/wepy
       </span>
      </p>
     </blockquote>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        5、weweb
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       weweb 是一个兼容小程序语法的前端框架，你可以用小程序的写法，来写 web 应用。如果你已经有小程序了，通过它你可以将你的小程序运行在浏览器中。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        特性
       </strong>
       ： 跨平台，一套代码多端运行（小程序、h5、未来直接打包成安卓、ios app 也不是梦) 自带常用组件，完美继承了小程序内置组件 兼容小程序 rpx 语法，使页面更容易适配各种机型
      </span>
     </p>
     <blockquote style="margin-left:1em;border-left-width:4px;border-left-color:rgb(62,192,131);color:rgb(136,136,136);font-size:15px;line-height:2;">
      <p style="line-height:2;margin-left:1em;width:inherit;">
       <span class="md-line md-end-block">
        GitHub：https://github.com/wdfe/weweb
       </span>
      </p>
     </blockquote>
     <h3 class="md-end-block md-heading" style="font-size:1.2em;font-weight:bold;line-height:1.225;color:rgb(62,192,131);border-bottom:5px solid rgb(62,192,131);width:inherit;">
      组件库
     </h3>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       还有各种开源的组件库，比如有以下这几个：
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       有赞的
       <strong>
        zanui-weapp
       </strong>
       和
       <strong>
        iview
       </strong>
       这两个比较出名。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       这两个组件库我之前在我的公众号都推荐过，具体用法和功能我就不多介绍了，自己找我以前的推文或者自己搜索一下就很清楚了。用法很简单。
      </span>
     </p>
     <h3 class="md-end-block md-heading" style="font-size:1.2em;font-weight:bold;line-height:1.225;color:rgb(62,192,131);border-bottom:5px solid rgb(62,192,131);width:inherit;">
      大餐来了
     </h3>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       上面介绍了 5 个开源框架和两个组件库，我并不是说上面的不好，上面的绝对都很好，不管上面的几个开源框架还是组件库都是大厂产品，都很好。但是他们仅仅就是开源框架就是开源框架，组件库就是组件库。一直没有满足我单独针对微信小程序开发的痛点。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       小程序开发一个是开源框架的好用，另外一个就是丰富的组件库，单纯官方推出的并不能满足我们的需要，上面的框架和组件库都是分开的。所以并不能解决我的痛点。
       <strong>
        而今天我推荐的这个开源框架既能提供框架也丰富扩展了很多组件，使我们开发的时候不用担心再自己封装组件了
       </strong>
       。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       其实上面分享的组件库有一个重大的缺点，他们组件库足够丰富，但是对于样式的自定义很死板，不灵活，但是今天的这个就灵活多了。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       我给大家推荐的这个开源框架就是：
       <strong>
        touchwx
       </strong>
       。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       Touch WX 是一套完全免费的微信小程序开发框架，包含丰富的 UI 控件用于官方组件的补充。特点如下：
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        1、组件扩充：
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       增加了 30 多种常用的组件用于官方组件的补充。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        2、功能扩充：
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       兼容阿里的 iconfont 图标库，海量矢量图标随意使用；补充了常用样式库、支持 less 语法、支持全局配置主题色等
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        3、开发体验改善：
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       四文件方式改为单文件方式，通过 VSCode 编辑器 + 插件的方式开发，拥有 web 开发体验；
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        4、小程序转为 H5 应用：
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       可以与 H5 开发框架 Touch UI 工程相互转换，发布成 webApp。开发一套代码，拥有两套应用。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        这套框架的原理是：
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       将 Touch WX 工程中所写的代码进行编译，直接输出为微信小程序工程原始代码。扩充的 30 多种组件，完全是基于小程序官方的自定义组件机制实现（row&amp;col 除外）。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       所以它支持小程序的全部语法，怎么开发小程序，就怎么开发 Touch WX。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       不过因为是单文件的开发方式，在文件的代码结构上稍有不同。请注意这一点。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       这样好处在于：
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        1、开发者迁移成本很小。
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       可以轻松的将已有的小程序移植为 Touch WX 工程，来使用它的扩展能力；
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        2、便于排查错误。
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       当遇到问题时，开发者也可以随时查看输出的小程序原始代码来定位问题所在。不会搞不清楚到底是框架问题还是自己代码的问题；
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        3、按需编译
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       由于小程序对体积有限制，在使用框架开发时，只有使用到的组件才会编译输出为小程序源码。没用到的不会输出。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       <strong>
        4、不会对框架产生依赖。
       </strong>
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       以后不想用了这套框架，可以直接对已经输出的小程序工程进行维护。
      </span>
     </p>
     <p style="color:rgb(79,79,79);font-size:15px;width:inherit;">
      <span class="md-line md-end-block">
       目前我认为单独针对小程序开发的话，这个开源框架是最适合我们的，效率也是最高的。
      </span>
     </p>
