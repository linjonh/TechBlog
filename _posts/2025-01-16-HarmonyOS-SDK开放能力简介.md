     HarmonyOS SDK开放能力简介
    </h1>
   </div>
  </div>

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <blockquote>
     <p>
      HarmonyOS SDK是HarmonyOS面向应用和服务开发的开放能力合集，基于HarmonyOS SDK前沿技术，广大开发者可以构建焕然一新的HarmonyOS应用。能力覆盖应用框架、应用服务、系统、媒体、AI、图形六大领域。
     </p>
    </blockquote>
    <p>
     <img alt="" height="357" src="https://i-blog.csdnimg.cn/direct/edf7f57885404305be68274595ba5b4c.png" width="634"/>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p10869182624318">
     HarmonyOS应用是能够在HarmonyOS设备（例如，手机、平板等）上运行的应用程序，支持两种形态：
    </p>
    <ul id="ZH-CN_TOPIC_0000001943775237__ul17869122616436">
     <li id="li11869172654317">
      需要下载安装的应用
     </li>
     <li id="li17726322181511">
      轻量级，具备免安装，即用即走，服务直达，自由流转等关键特征的元服务
     </li>
    </ul>
    <p id="ZH-CN_TOPIC_0000001943775237__p20734446191516">
     当前HarmonyOS SDK开放能力包括以下两大类：
    </p>
    <ul id="ZH-CN_TOPIC_0000001943775237__ul1373464617153">
     <li id="li1073424681519">
      端侧能力开放：开放能力通过SDK方式呈现，应用通过集成SDK的方式进行集成
     </li>
     <li id="li17734646111517">
      云侧能力开放：通过服务器开放接口（如： RESTful接口）给应用进行集成
     </li>
    </ul>
    <h2 id="section171056368811">
     <strong>
      一、应用框架
     </strong>
    </h2>
    <p>
     <strong>
      高效开发，助力构建高端精致、稳定流畅的HarmonyOS应用
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p188371636984">
     应用框架作为HarmonyOS应用开发的基石，提供了易用的原生UI框架、强大的分布式能力、以及高效的编程语言，让开发者轻松打造出精美、流畅的HarmonyOS应用。
    </p>
    <ol id="ZH-CN_TOPIC_0000001943775237__ol09798226920">
     <li id="li12979622591">
      提供了丰富的HarmonyOS UI组件，通过智能的布局和交互，一次开发，多设备适配，极大提升开发效率，并通过立体的光影、模糊和动效，助力开发者打造高端精致的HarmonyOS应用。
     </li>
     <li id="li169793228917">
      提供统一的数据语言和跨设备同步，提供分布式能力，场景化的服务扩展和数据绑定，共享库的代码复用，简化开发过程和提高代码质量。
     </li>
     <li id="li18979222095">
      提供高效的编程语言，低开销、精细的后台管理，统一渲染和调度，打造稳定流畅的应用。
     </li>
    </ol>
    <h2 id="section688712910513">
     二、应用服务
    </h2>
    <p>
     <strong>
      使能应用提供便捷用户体验，助力经营和商业闭环
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p11871164316402">
     应用服务开放能力提供从开发上架到变现运营能力，覆盖应用的整个生命周期。
    </p>
    <ul id="ZH-CN_TOPIC_0000001943775237__ul4739115812401">
     <li id="li1173925874010">
      Account Kit、Game Service Kit提供华为帐号相关的个人信息、实名、游戏等数据
     </li>
     <li id="li107391582403">
      Location Kit、Map Kit、Contacts Kit等服务则帮助实现各种业务场景功能
     </li>
     <li id="li12739155818403">
      Ads Kit、IAP Kit和Payment Kit让应用实现商业闭环
     </li>
     <li id="li9739135844014">
      Analytics Kit等服务通过全面的运营数据改善经营
     </li>
     <li id="li773905884010">
      Push Kit、Notification Kit、Live View Kit等服务则更好地触达用户
     </li>
    </ul>
    <h2 id="section5361153215515">
     三、系统
    </h2>
    <p>
     <strong>
      助力构建纯净安全、稳定快速的HarmonyOS应用
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p189776282417">
     系统领域，为HarmonyOS应用构筑安全、流畅、全场景协同的可靠能力底座。
    </p>
    <ul id="ZH-CN_TOPIC_0000001943775237__ul18592123844115">
     <li id="li4592203894113">
      提供强大的身份验证、密钥管理、证书服务和安全环境检测等能力，助力开发者实现可信开发执行和应用安全管理，保护用户数据
     </li>
     <li id="li1592173811412">
      提供丰富的通信服务开放能力和全面的设备发现、连接和管理能力，实现更多HarmonyOS硬件设备更快连接、更广覆盖
     </li>
     <li id="li1059215389415">
      提供多样化的系统基础服务，让应用更快响应，为用户打造简单易用的多端互联服务
     </li>
     <li id="li1459223813417">
      提供高效的测试和问题定位工具，帮助应用敏捷开发、快速迭代
     </li>
    </ul>
    <h2 id="section167734405111">
     <strong>
      四、媒体
     </strong>
    </h2>
    <p>
     <strong>
      助力HarmonyOS应用打造超清、流畅、高能效的音视频体验
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p1654215175315">
     媒体开放能力，作为HarmonyOS能力底座，基于“软、硬、芯”协同、编解码硬件加速、轻量级播放引擎、统一场景化接口等，为开发者提供应用快速构建音视频相关业务能力，助力鸿蒙生态应用打造超清、流畅、高能效的音视频体验。
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p2542812531">
     媒体领域主要提供用户视觉、听觉信息的处理能力，包括音视频和相机图像信息的采集、压缩存储、解压播放等，为影音娱乐类、视频制作类、音视频通话类等业务领域应用提供基础媒体能力，为开发者提供鸿蒙生态媒体领域的全场景解决方案。
    </p>
    <h2 id="section8356128105317">
     五、图形
    </h2>
    <p>
     <strong>
      提供丰富的图形游戏开放能力，使能应用高流畅，高画质，低功耗
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p11725201675317">
     图形游戏开放能力，充分发挥华为已有硬件的渲染能力，包括2D、3D渲染，针对游戏场景的插桢、超分、场景化渲染以及GPU硬件的原子化接口，同时还有面向AR、VR场景的运动跟踪，人体识别等丰富能力，助力开发者实现高画质、高流畅、低功耗的图形及游戏体验。
    </p>
    <h2 id="section9828152612536">
     六、AI开放能力
    </h2>
    <p>
     <strong>
      分层开放，各类场景皆可原生智能
     </strong>
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p1841816211436">
     AI领域提供了分层的开放能力，满足不同开发者的诉求。
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p1580317420437">
     首先，从底层硬件加速和推理框架出发，利用先进的技术手段，实现了高效的计算能力和强大的处理性能。
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p056511674315">
     其次，在中间层提供了丰富的API，涵盖了语音、图像和自然语言处理等多个领域，助力开发者快速实现各种复杂的场景功能。
    </p>
    <p id="ZH-CN_TOPIC_0000001943775237__p930663314536">
     最后，在上层提供了控件化的能力，助力开发者更加便捷地开发出各种创新的应用。
    </p>
