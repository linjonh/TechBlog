    <h3>

     1、更新 iptv 数据源
    </h3>
    <h4>
     <a id="A_29">
     </a>
     A、默认测速方式
    </h4>
    <p>
     打开步骤二 1 下载的更新工具，在这里暂时使用默认的配置即可
     <br/>
     <img alt="更新工具" src="https://i-blog.csdnimg.cn/direct/b3ca3ae5345145b58ea26e654b37cb82.png">
      <br/>
      点击开始更新，进行更新，这里等待时间取决于当前的网络
      <br/>
      <img alt="开始更新" src="https://i-blog.csdnimg.cn/direct/66e90d29670a4cd7836c0b818188c3b2.png">
       <br/>
       更新完毕之后，会提示更新完成，更新完成的提示信息如下
       <br/>
       <img alt="更新完成" src="https://i-blog.csdnimg.cn/direct/1e448b19c27949a881af60fac1a08b01.png"/>
      </img>
     </img>
    </p>
    <h4>
     <a id="B_ffmpeg__36">
     </a>
     B、使用 ffmpeg 作为测速工具
    </h4>
    <p>
     TV 工具默认勾选了 ffmpeg 作为更新工具，但是在使用过程中，会判断是否存在 ffmpeg 工具，如果存在，则使用 ffmpeg 工具进行测试，如果不存在，则调用默认的测速方式，逻辑如下
     <br/>
     <img alt="测试逻辑" src="https://i-blog.csdnimg.cn/direct/694dd457a3fa44fea7952494aec7f1a1.png">
      <br/>
      <img alt="判断是否有安装 ffmpeg 的判断" src="https://i-blog.csdnimg.cn/direct/c72a1632b9d84e5abc3897c5f25f131e.png">
       <br/>
       按照判断逻辑，如需要使用 ffmpeg 测试，则需要安装 ffmpeg 工具即可
       <br/>
       下载 ffmpeg 工具，官网地址：
       <a href="https://www.ffmpeg.org/" rel="nofollow">
        https://www.ffmpeg.org/
       </a>
       <br/>
       <img alt="ffmpeg 官网" src="https://i-blog.csdnimg.cn/direct/93866bbf436c48f5b5e7dae7ad13e0ed.png">
        <br/>
        选择 windows
        <br/>
        <img alt="选择 windows" src="https://i-blog.csdnimg.cn/direct/f833fffd75bd4cee91531d6b5c226f2c.png">
         <br/>
         这里建议使用 release 版本
         <br/>
         <img alt="建议下载" src="https://i-blog.csdnimg.cn/direct/53388626d7ac4433a8d6e69c57dbb22c.png"/>
         <br/>
         解压下载的 ffmpeg 工具，解压后的文件目录如下
         <br/>
         <img alt="解压后的文件目录" src="https://i-blog.csdnimg.cn/direct/f4178fba543c4b13ab5142baba504b1c.png"/>
         <br/>
         将解压后的目录，放到你自己喜欢的位置，我放置到了 C:\Program Files\ffmpeg 目录下
         <br/>
         <img alt="ffmpeg 放置目录" src="https://i-blog.csdnimg.cn/direct/38bad41ad8444e02b9e6ff51bc3868a4.png"/>
         <br/>
         将工具添加到系统环境中
         <br/>
         <img alt="添加到系统环境" src="https://i-blog.csdnimg.cn/direct/39249c3e840c46f38e02f91d91fe2b7e.png"/>
         <br/>
         保存后，通过 WIN + R 键，打开 cmd ，输入 ffmpeg -version 看是否安装完成
         <br/>
         <img alt="ffmpeg 安装完成" src="https://i-blog.csdnimg.cn/direct/7d29325d1fd04afd90e5b77256bc7f43.png"/>
         <br/>
         安装成功后，重新启动 TV 更新工具，重新更新即可，测试使用的测试工具，即为 ffmpeg。
        </img>
       </img>
      </img>
     </img>
    </p>

     总结
    </h2>
    <p>
     1、通过开源的方式，可以获取的相关的 IPTV 源并实现播放，比较简单，快捷
