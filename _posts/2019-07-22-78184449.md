    <p align="left" style="margin: 0px; text-align: left;">
     <strong>
      <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
       无穷小微积分是植根于现代数学基础上的一朵鲜花，而不是鬼话、毒草
       <span lang="EN-US" style="margin: 0px;">
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 28.1pt;">
     <strong>
      <span style="margin: 0px; font-family: 宋体; font-size: 14pt;">
       如果不懂得现代数学发展史，我们就是数学文明的文盲。
       <span lang="EN-US" style="margin: 0px;">
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      数学悖论的产生导致数学的第三次理论危机。也就是说，悖论（
      <span lang="EN-US" style="margin: 0px;">
       Paradox
      </span>
      ）会导致理论系统的逻辑矛盾，产生理论系统的不一致性。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      1897
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      年，福尔蒂揭示了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E9%9B%86%E5%90%88%E8%AE%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          集合论
         </span>
        </span>
       </a>
      </span>
      中的第一个悖论。两年后，康托发现了很相似的悖论。
      <span lang="EN-US" style="margin: 0px;">
       1902
      </span>
      年，
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%BD%97%E7%B4%A0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          罗素
         </span>
        </span>
       </a>
      </span>
      又发现了一个悖论，它除了涉及
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E9%9B%86%E5%90%88%E6%A6%82%E5%BF%B5" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          集合概念
         </span>
        </span>
       </a>
      </span>
      本身外不涉及别的概念。
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%BD%97%E7%B4%A0%E6%82%96%E8%AE%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          罗素悖论
         </span>
        </span>
       </a>
      </span>
      曾被以多种形式通俗化。其中最著名的是罗素于
      <span lang="EN-US" style="margin: 0px;">
       1919
      </span>
      年给出的，它涉及到某村理发师的困境。理发师宣布了这样一条原则：他给所有不给自己刮脸的人刮脸，并且，只给村里这样的人刮脸。当人们试图回答下列疑问时，就认识到了这种情况的悖论性质：
      <span lang="EN-US" style="margin: 0px;">
       "
      </span>
      理发师是否自己给自己刮脸？
      <span lang="EN-US" style="margin: 0px;">
       "
      </span>
      如果他不给自己刮脸，那么他按原则就该为自己刮脸；如果他给自己刮脸，那么他就不符合他的原则。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      罗素悖论的精确表述：
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      如果存在一个集合
      <span lang="EN-US" style="margin: 0px;">
       A={x | x
      </span>
     </span>
     <span style="margin: 0px; font-family: 'MS Gothic'; font-size: 12pt;">
      ∉
     </span>
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      x}
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      ，那么
      <span lang="EN-US" style="margin: 0px;">
       A∈A
      </span>
      是否成立？如果它成立，那么
      <span lang="EN-US" style="margin: 0px;">
       A∈A
      </span>
      ，不满足
      <span lang="EN-US" style="margin: 0px;">
       A
      </span>
      的特征性质。如果它不成立，
      <span lang="EN-US" style="margin: 0px;">
       A
      </span>
      就满足了特征性质。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      罗素悖论使整个数学大厦动摇了。无怪乎
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%BC%97%E9%9B%B7%E6%A0%BC" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          弗雷格
         </span>
        </span>
       </a>
      </span>
      在收到罗素的信之后，在他刚要出版的《算术的基本法则》第
      <span lang="EN-US" style="margin: 0px;">
       2
      </span>
      卷末尾写道：
      <span lang="EN-US" style="margin: 0px;">
       "
      </span>
      一位科学家不会碰到比这更难堪的事情了，即在工作完成之时，它的基础垮掉了，当本书等待印出的时候，罗素先生的一封信把我置于这种境地
      <span lang="EN-US" style="margin: 0px;">
       "
      </span>
      。于是终结了近
      <span lang="EN-US" style="margin: 0px;">
       12
      </span>
      年的刻苦钻研。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      承认无穷集合，承认无穷基数，就好像一切灾难都出来了，这就是第三次数学危机的实质。尽管悖论可以消除，矛盾可以解决，然而数学的确定性却在一步一步地丧失。现代
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%85%AC%E7%90%86%E9%9B%86%E5%90%88%E8%AE%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          公理集合论
         </span>
        </span>
       </a>
      </span>
      的大堆公理，简直难说孰真孰假，可是又不能把它们都消除掉，它们跟整个数学是血肉相连的。所以，第三次危机表面上解决了，实质上更深刻地以其它形式延续着。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <a name="2" target="_blank">
     </a>
     <a name="sub930502_2" target="_blank">
     </a>
     <a name="背景" target="_blank">
     </a>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      第三次数学危机产生于十九世纪末和二十世纪初，当时正是数学空前兴旺发达的时期。首先是逻辑的
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%95%B0%E5%AD%A6%E5%8C%96" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          数学化
         </span>
        </span>
       </a>
      </span>
      ，促使了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%95%B0%E7%90%86%E9%80%BB%E8%BE%91" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          数理逻辑
         </span>
        </span>
       </a>
      </span>
      这门学科诞生。
     </span>
     <strong>
      <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 18pt;">
      </span>
     </strong>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      十九世纪七十年代
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%BA%B7%E6%89%98%E5%B0%94" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          康托尔
         </span>
        </span>
       </a>
      </span>
      创立的集合论是
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%8E%B0%E4%BB%A3%E6%95%B0%E5%AD%A6" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          现代数学
         </span>
        </span>
       </a>
      </span>
      的基础，也是产生危机的直接来源。十九世纪末，戴德金及皮亚诺对
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%AE%97%E6%9C%AF" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          算术
         </span>
        </span>
       </a>
      </span>
      及
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%AE%9E%E6%95%B0%E7%90%86%E8%AE%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          实数理论
         </span>
        </span>
       </a>
      </span>
      进行
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%85%AC%E7%90%86" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          公理
         </span>
        </span>
       </a>
      </span>
      化，推动了公理化运动。而公理化运动的最大成就则是
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%B8%8C%E5%B0%94%E4%BC%AF%E7%89%B9" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          希尔伯特
         </span>
        </span>
       </a>
      </span>
      在
      <span lang="EN-US" style="margin: 0px;">
       1899
      </span>
      年对于初等几何的公理化。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <a name="3" target="_blank">
     </a>
     <a name="sub930502_3" target="_blank">
     </a>
     <a name="定义" target="_blank">
     </a>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      为了讲清楚第三次数学危机的来龙去脉，我们首先要说明什么是数学危机。一般来讲，危机是一种激化的、非解决不可的矛盾。从哲学上来看，矛盾是无处不在的、不可避免的，即便以确定无疑著称的数学也不例外。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      数学中有大大小小的许多矛盾，比如正与负、加法与减法、微分与积分、有理数与
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%97%A0%E7%90%86%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          无理数
         </span>
        </span>
       </a>
      </span>
      、
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%AE%9E%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          实数
         </span>
        </span>
       </a>
      </span>
      与
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E8%99%9A%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          虚数
         </span>
        </span>
       </a>
      </span>
      等等。但是整个数学发展过程中还有许多深刻的矛盾，例如有穷与无穷，连续与离散，乃至存在与构造，逻辑与直观，具体对象与抽象对象，概念与计算等等。在整个数学发展的历史上，贯穿着矛盾的斗争与解决。而在矛盾激化到涉及整个数学的基础时，就产生数学危机。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      矛盾的消除，危机的解决，往往给数学带来新的内容，新的进展，甚至引起革命性的变革，这也反映出矛盾斗争是事物发展的历史动力这一基本原理。整个数学的发展史就是矛盾斗争的历史，斗争的结果就是数学领域的发展。
      <a name="4" target="_blank">
      </a>
      <a name="sub930502_4" target="_blank">
      </a>
      <a name="研究" target="_blank">
      </a>
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      人类最早认识的是自然数。从引进零及
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E8%B4%9F%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          负数
         </span>
        </span>
       </a>
      </span>
      就经历过斗争：要么引进这些数，要么大量的数的减法就行不通；同样，引进分数使
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E4%B9%98%E6%B3%95" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          乘法
         </span>
        </span>
       </a>
      </span>
      有了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E9%80%86%E8%BF%90%E7%AE%97" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          逆运算
         </span>
        </span>
       </a>
       ——
       <a href="https://baike.baidu.com/item/%E9%99%A4%E6%B3%95" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          除法
         </span>
        </span>
       </a>
      </span>
      ，否则许多实际问题也不能解决。但是接着又出现了这样的问题，是否所有的量都能用有理数来表示
      <span lang="EN-US" style="margin: 0px;">
       ?
      </span>
      于是发现
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%97%A0%E7%90%86%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          无理数
         </span>
        </span>
       </a>
      </span>
      就导致了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%AC%AC%E4%B8%80%E6%AC%A1%E6%95%B0%E5%AD%A6%E5%8D%B1%E6%9C%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          第一次数学危机
         </span>
        </span>
       </a>
      </span>
      ，而危机的解决也就促使逻辑的发展和几何学的体系化。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      <a href="https://baike.baidu.com/item/%E6%96%B9%E7%A8%8B" rel="nofollow noopener noreferrer" target="_blank">
       <span lang="EN-US" style="margin: 0px; color: blue;">
        <span lang="EN-US" style="margin: 0px;">
         方程
        </span>
       </span>
      </a>
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      的解导致了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E8%99%9A%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          虚数
         </span>
        </span>
       </a>
      </span>
      的出现，虚数从一开始就被认为是
      <span lang="EN-US" style="margin: 0px;">
       “
      </span>
      不实的
      <span lang="EN-US" style="margin: 0px;">
       ”
      </span>
      。可是这种不实的数却能解决
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%AE%9E%E6%95%B0" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          实数
         </span>
        </span>
       </a>
      </span>
      所不能解决的问题，从而为自己争得存在的权利。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      几何学的发展从
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%AC%A7%E5%87%A0%E9%87%8C%E5%BE%97%E5%87%A0%E4%BD%95" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          欧几里得几何
         </span>
        </span>
       </a>
      </span>
      的一统天下发展到各种非欧几何学也是如此。在十九世纪发现了许多用传统方法不能解决的问题，如五次及五次以上代数方程不能通过加、减、乘、除、
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E4%B9%98%E6%96%B9" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          乘方
         </span>
        </span>
       </a>
      </span>
      、开方求出根来；古希腊几何三大问题，即三等分
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E4%BB%BB%E6%84%8F%E8%A7%92" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          任意角
         </span>
        </span>
       </a>
      </span>
      、倍
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%AB%8B%E6%96%B9%E4%BD%93" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          立方体
         </span>
        </span>
       </a>
      </span>
      、
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%8C%96%E5%9C%86%E4%B8%BA%E6%96%B9" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          化圆为方
         </span>
        </span>
       </a>
      </span>
      不能通过
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%9C%86%E8%A7%84" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          圆规
         </span>
        </span>
       </a>
      </span>
      、直尺作图来解决等等。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      <span style="margin: 0px;">
      </span>
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      否定的结果表明了传统方法的局限性，也反映了人类认识的深入。这种发现给这些学科带来极大的冲击，几乎完全改变了它们的方向。比如说，
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E4%BB%A3%E6%95%B0%E5%AD%A6" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          代数学
         </span>
        </span>
       </a>
      </span>
      从此以后向抽象代数学方面发展，而求解方程的根变成了分析及计算数学的课题。在第三次数学危机中，这种情况也多次出现，尤其是包含整数算术在内的
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%BD%A2%E5%BC%8F%E7%B3%BB%E7%BB%9F" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          形式系统
         </span>
        </span>
       </a>
      </span>
      的不完全性、许多问题的不可判定性都大大提高了人们的认识，也促进了
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%95%B0%E7%90%86%E9%80%BB%E8%BE%91" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          数理逻辑
         </span>
        </span>
       </a>
      </span>
      的大发展。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <a name="5" target="_blank">
     </a>
     <a name="sub930502_5" target="_blank">
     </a>
     <a name="发展" target="_blank">
     </a>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      这种矛盾、危机引起的发展，改变面貌，甚至引起革命，在数学发展历史上是屡见不鲜的。
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E6%95%B0%E5%AD%A6%E5%8D%B1%E6%9C%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          第二次数学危机
         </span>
        </span>
       </a>
      </span>
      是由
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E6%97%A0%E7%A9%B7%E5%B0%8F%E9%87%8F" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          无穷小量
         </span>
        </span>
       </a>
      </span>
      的矛盾引起的，它反映了数学内部的有限与无穷的矛盾。数学中也一直贯穿着计算方法、分析方法在应用与概念上清楚及逻辑上严格的矛盾。在这方面，比较注意实用的数学家盲目应用。而比较注意严密的数学家及哲学家则提出批评。只有这两方面取得协调一致后，矛盾才能解决。后来算符演算及
      <span lang="EN-US" style="margin: 0px;">
       δ
      </span>
      函数也重复了这个过程，开始是形式演算、任意应用，直到施瓦尔兹才奠定
      <span lang="EN-US" style="margin: 0px;">
       <a href="https://baike.baidu.com/item/%E5%B9%BF%E4%B9%89%E5%87%BD%E6%95%B0%E8%AE%BA" rel="nofollow noopener noreferrer" target="_blank">
        <span lang="EN-US" style="margin: 0px; color: blue;">
         <span lang="EN-US" style="margin: 0px;">
          广义函数论
         </span>
        </span>
       </a>
      </span>
      的严整系统。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <a name="6" target="_blank">
     </a>
     <a name="sub930502_6" target="_blank">
     </a>
     <a name="危机" target="_blank">
     </a>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      对于第三次数学危机，有人认为只是数学基础的危机，与数学无关。这种看法是片面的。诚然，问题涉及数理逻辑和集合论，但它一开始就牵涉到无穷集合，而现代数学如果脱离无穷集合就可以说寸步难行。因为如果只考虑有限集合或至多是可数的集合，那绝大部分数学将不复存在。而且即便这些有限数学的内容，也有许多问题要涉及无穷的方法，比如解决数论中的许多问题都要用解析方法。由此看来，第三次数学危机是一次深刻的数学危机。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <a name="7" target="_blank">
     </a>
     <a name="sub930502_7" target="_blank">
     </a>
     <a name="解决" target="_blank">
     </a>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      数学家们通过将集合的构造公理化来排除了这样的集合的存在性。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      <span style="margin: 0px;">
      </span>
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      例如，在策梅洛（
      <span lang="EN-US" style="margin: 0px;">
       Zermelo
      </span>
      ）和弗伦克尔（
      <span lang="EN-US" style="margin: 0px;">
       Fraenkel
      </span>
      ）等提出的
      <span lang="EN-US" style="margin: 0px;">
       ZF
      </span>
      公理系统（也称
      <span lang="EN-US" style="margin: 0px;">
       ZFC
      </span>
      公理系统）中，严格规定了一个集合存在的条件（简单地说，存在一个空集【空集公理】；每个集合存在幂集【幂集公理】；每个集合里所有的集合取并也形成集合【并集公理】；每个集合的满足某条件的元素构成子集【子集公理】；一个
      <span lang="EN-US" style="margin: 0px;">
       ”
      </span>
      定义域
      <span lang="EN-US" style="margin: 0px;">
       “
      </span>
      为
      <span lang="EN-US" style="margin: 0px;">
       A
      </span>
      的
      <span lang="EN-US" style="margin: 0px;">
       ”
      </span>
      函数
      <span lang="EN-US" style="margin: 0px;">
       “
      </span>
      存在
      <span lang="EN-US" style="margin: 0px;">
       “
      </span>
      值域
      <span lang="EN-US" style="margin: 0px;">
       ”
      </span>
      【替换公理】等），这样无法定义出悖论中的集合。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left; text-indent: 24pt;">
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      总而言之，在集合论公理化基础上，第三次数学危机完美解决。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
    <p align="left" style="margin: 0px; text-align: left;">
     <span lang="EN-US" style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      <span style="margin: 0px;">
      </span>
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      实际情况是，无穷小微积分在上述
     </span>
     <span style="margin: 0px; color: rgb(51, 51, 51); font-family: 宋体; font-size: 11pt;">
      磐
     </span>
     <span style="margin: 0px; font-family: 宋体; font-size: 12pt;">
      若坚石的公理化集合论的沃土上茁壮成长出的一朵鲜花，而不是鬼话、邪说、毒草。
      <span lang="EN-US" style="margin: 0px;">
      </span>
     </span>
    </p>
