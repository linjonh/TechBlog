        <p>
         <span style="color:#000000">
         </span>
         <span style="color:#444444">
          Android
         </span>
         <span style="color:#444444">
          操作系统介绍
         </span>
         <span style="color:#000000">
          <br/>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           是
          </span>
          <span style="font-family:Tahoma">
           Google
          </span>
          <span style="font-family:宋体">
           于
          </span>
          <span style="font-family:Tahoma">
           2007
          </span>
          <span style="font-family:宋体">
           年
          </span>
          <span style="font-family:Tahoma">
           11
          </span>
          <span style="font-family:宋体">
           月
          </span>
          <span style="font-family:Tahoma">
           05
          </span>
          <span style="font-family:宋体">
           日宣布的基于
          </span>
          <span style="font-family:Tahoma">
           Linux
          </span>
          <span style="font-family:宋体">
           平台的开源手机操作系统的名称，该平台由操作系统、中间件、用户界面和应用软件组成。它采用软件堆层（
          </span>
          <span style="font-family:Tahoma">
           Software Stack
          </span>
          <span style="font-family:宋体">
           ，又名软件叠层）的架构，主要分为三部分。底层以
          </span>
          <span style="font-family:Tahoma">
           Linux
          </span>
          <span style="font-family:宋体">
           内核工作为基础，由
          </span>
          <span style="font-family:Tahoma">
           C
          </span>
          <span style="font-family:宋体">
           语言开发，只提供基本功能；中间层包括函数库
          </span>
          <span style="font-family:Tahoma">
           Library
          </span>
          <span style="font-family:宋体">
           和 虚拟机
          </span>
          <span style="font-family:Tahoma">
           Virtual Machine
          </span>
          <span style="font-family:宋体">
           ，由
          </span>
          <span style="font-family:Tahoma">
           C++
          </span>
          <span style="font-family:宋体">
           开发。最上层是各种应用软件，包括通话程序，短信程序等，应用软件则由各公司自行开发，以
          </span>
          <span style="font-family:Tahoma">
           Java
          </span>
          <span style="font-family:宋体">
           作为编写程序的一部分。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           一词的本义指“
          </span>
         </span>
         <span style="color:#336699">
          <a href="http://robot.ofweek.com/#tc_qz_original=917815313" rel="nofollow noopener noreferrer" target="_blank">
           机器人
          </a>
         </span>
         ”，最初的
         <span style="font-family:Tahoma">
          Android
         </span>
         <span style="font-family:宋体">
          主要支持手机，后来经过开发改良，逐渐扩展到平板电脑及其他一些领域上，是首个为移动终端打造的真正的开放和完整的移动软件。
         </span>
        </p>
        <p>
         <span style="color:#444444">
          系统架构：
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           的系统架构和其操作系统一样，采用了分层的架构。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           分为四个层，从高层到低层分别是应用程序层、应用程序框架层、系统运行库层和
          </span>
          <span style="font-family:Tahoma">
           Linux
          </span>
          <span style="font-family:宋体">
           内核层。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          一、应用程序
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           会同一系列核心应用程序包一起发布，该应用程序包包括客户端，
          </span>
          <span style="font-family:Tahoma">
           SMS
          </span>
          <span style="font-family:宋体">
           短消息程序，日历，地图，浏览器，联系人管理程序等。所有的应用程序都是使用
          </span>
          <span style="font-family:Tahoma">
           JAVA
          </span>
          <span style="font-family:宋体">
           语言编写的。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          二、应用程序框架
         </span>
        </p>
        <p>
         <span style="color:#444444">
          开发人员也可以完全访问核心应用程序所使用的
          <span style="font-family:Tahoma">
           API
          </span>
          <span style="font-family:宋体">
           框架。该应用程序的架构设计简化了组件的重用
          </span>
          <span style="font-family:Tahoma">
           ;
          </span>
          <span style="font-family:宋体">
           任何一个应用程序都可以发布它的功能块并且任何其它的应用程序都可以使用其所发布的功能块（不过得遵循框架的安全性）。同样，该应用程序重用机制也使用户可以方便的替换程序组件。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          三、系统运行库
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           包含一些
          </span>
          <span style="font-family:Tahoma">
           C/C++
          </span>
          <span style="font-family:宋体">
           库，这些库能被
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           系统中不同的组件使用。它们通过
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           应用程序框架为开发者提供服务。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          四、
          <span style="font-family:Tahoma">
           Linux
          </span>
          <span style="font-family:宋体">
           内核层
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           内核是基于
          </span>
          <span style="font-family:Tahoma">
           Linux
          </span>
          <span style="font-family:宋体">
           内核的修改的内核版本，它提供了用于支持
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           平台的设备驱动。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          系统结构：
         </span>
        </p>
        <p>
         <span style="color:#444444">
          一、系统内核
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           是运行于
          </span>
          <span style="font-family:Tahoma">
           Linux kernel
          </span>
          <span style="font-family:宋体">
           之上，但并不是
          </span>
          <span style="font-family:Tahoma">
           GNU/Linux
          </span>
          <span style="font-family:宋体">
           。因为在一般
          </span>
          <span style="font-family:Tahoma">
           GNU/Linux
          </span>
          <span style="font-family:宋体">
           里支持的功能，
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           大都没有支持，包括
          </span>
          <span style="font-family:Tahoma">
           Cairo
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           X11
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           Alsa
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           FFmpeg
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           GTK
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           Pango
          </span>
          <span style="font-family:宋体">
           及
          </span>
          <span style="font-family:Tahoma">
           Glibc
          </span>
          <span style="font-family:宋体">
           等都被移除掉了。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           又以
          </span>
          <span style="font-family:Tahoma">
           Bionic
          </span>
          <span style="font-family:宋体">
           取代
          </span>
          <span style="font-family:Tahoma">
           Glibc
          </span>
          <span style="font-family:宋体">
           、以
          </span>
          <span style="font-family:Tahoma">
           Skia
          </span>
          <span style="font-family:宋体">
           取代
          </span>
          <span style="font-family:Tahoma">
           Cairo
          </span>
          <span style="font-family:宋体">
           、再以
          </span>
          <span style="font-family:Tahoma">
           opencore
          </span>
          <span style="font-family:宋体">
           取代
          </span>
          <span style="font-family:Tahoma">
           FFmpeg
          </span>
          <span style="font-family:宋体">
           等等。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           为了达到商业应用，必须移除被
          </span>
          <span style="font-family:Tahoma">
           GNU GPL
          </span>
          <span style="font-family:宋体">
           授权证所约束的部份，例如
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           将驱动程序移到
          </span>
          <span style="font-family:Tahoma">
           Userspace
          </span>
          <span style="font-family:宋体">
           ，使得
          </span>
          <span style="font-family:Tahoma">
           Linux driver
          </span>
          <span style="font-family:宋体">
           与
          </span>
          <span style="font-family:Tahoma">
           Linux kernel
          </span>
          <span style="font-family:宋体">
           彻底分开。
          </span>
          <span style="font-family:Tahoma">
           Bionic/Libc/Kernel/
          </span>
          <span style="font-family:宋体">
           并非标准的
          </span>
          <span style="font-family:Tahoma">
           Kernel header files
          </span>
          <span style="font-family:宋体">
           。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           的
          </span>
          <span style="font-family:Tahoma">
           Kernel header
          </span>
          <span style="font-family:宋体">
           是利用工具由
          </span>
          <span style="font-family:Tahoma">
           Linux Kernel header
          </span>
          <span style="font-family:宋体">
           所产生的，这样做是为了保留常数、数据结构与宏。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          二、硬件抽像层
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           的
          </span>
          <span style="font-family:Tahoma">
           HAL
          </span>
          <span style="font-family:宋体">
           （硬件抽像层）是能以封闭源码形式提供硬件驱动模块。
          </span>
          <span style="font-family:Tahoma">
           HAL
          </span>
          <span style="font-family:宋体">
           的目的是为了把
          </span>
          <span style="font-family:Tahoma">
           Android framework
          </span>
          <span style="font-family:宋体">
           与
          </span>
          <span style="font-family:Tahoma">
           Linux kernel
          </span>
          <span style="font-family:宋体">
           隔开，让
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           不至过度依赖
          </span>
          <span style="font-family:Tahoma">
           Linux kernel
          </span>
          <span style="font-family:宋体">
           ，以达成
          </span>
          <span style="font-family:Tahoma">
           Kernel independent
          </span>
          <span style="font-family:宋体">
           的概念，也让
          </span>
          <span style="font-family:Tahoma">
           Android framework
          </span>
          <span style="font-family:宋体">
           的开发能在不考量驱动程序实现的前提下进行发展。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          三、中介软件
         </span>
        </p>
        <p>
         <span style="color:#444444">
          操作系统与应用程序的沟通桥梁，应用分为两层：函数层（
          <span style="font-family:Tahoma">
           Library
          </span>
          <span style="font-family:宋体">
           ）和虚拟机（
          </span>
          <span style="font-family:Tahoma">
           Virtual Machine
          </span>
          <span style="font-family:宋体">
           ）。
          </span>
          <span style="font-family:Tahoma">
           Bionic
          </span>
          <span style="font-family:宋体">
           是
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           改良
          </span>
          <span style="font-family:Tahoma">
           libc
          </span>
          <span style="font-family:宋体">
           的版本。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           同时包含了
          </span>
          <span style="font-family:Tahoma">
           Webkit
          </span>
          <span style="font-family:宋体">
           ，所谓的
          </span>
          <span style="font-family:Tahoma">
           Webkit
          </span>
          <span style="font-family:宋体">
           就是
          </span>
          <span style="font-family:Tahoma">
           Apple Safari
          </span>
          <span style="font-family:宋体">
           浏览器背后的引擎。
          </span>
          <span style="font-family:Tahoma">
           Surface flinger
          </span>
          <span style="font-family:宋体">
           是就
          </span>
          <span style="font-family:Tahoma">
           2D
          </span>
          <span style="font-family:宋体">
           或
          </span>
          <span style="font-family:Tahoma">
           3D
          </span>
          <span style="font-family:宋体">
           的内容显示到屏幕上。
          </span>
          <span style="font-family:Tahoma">
           Android
          </span>
          <span style="font-family:宋体">
           使用工具链（
          </span>
          <span style="font-family:Tahoma">
           Toolchain)
          </span>
          <span style="font-family:宋体">
           为
          </span>
          <span style="font-family:Tahoma">
           Google
          </span>
          <span style="font-family:宋体">
           自制的
          </span>
          <span style="font-family:Tahoma">
           BionicLibc
          </span>
          <span style="font-family:宋体">
           。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           采用
          </span>
          <span style="font-family:Tahoma">
           OpenCORE
          </span>
          <span style="font-family:宋体">
           作为基础多媒体框架。
          </span>
          <span style="font-family:Tahoma">
           Open CORE
          </span>
          <span style="font-family:宋体">
           可分
          </span>
          <span style="font-family:Tahoma">
           7
          </span>
          <span style="font-family:宋体">
           大块：
          </span>
          <span style="font-family:Tahoma">
           PVPlayer
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           PVAuthor
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           Codec
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           PacketVideo Multimedia Framework
          </span>
          <span style="font-family:宋体">
           （
          </span>
          <span style="font-family:Tahoma">
           PVMF)
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           Operating System Compatibility Library
          </span>
          <span style="font-family:宋体">
           （
          </span>
          <span style="font-family:Tahoma">
           OSCL)
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           Common
          </span>
          <span style="font-family:宋体">
           、
          </span>
          <span style="font-family:Tahoma">
           OpenMAX
          </span>
          <span style="font-family:宋体">
           。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           使用
          </span>
          <span style="font-family:Tahoma">
           skia
          </span>
          <span style="font-family:宋体">
           为核心图形引擎，搭配
          </span>
          <span style="font-family:Tahoma">
           OpenGL/ES
          </span>
          <span style="font-family:宋体">
           。
          </span>
          <span style="font-family:Tahoma">
           skia
          </span>
          <span style="font-family:宋体">
           与
          </span>
          <span style="font-family:Tahoma">
           LinuxCairo
          </span>
          <span style="font-family:宋体">
           功能相当，但相较于
          </span>
          <span style="font-family:Tahoma">
           Linux Cairo, skia
          </span>
          <span style="font-family:宋体">
           功能还只是雏形的。
          </span>
          <span style="font-family:Tahoma">
           2005
          </span>
          <span style="font-family:宋体">
           年
          </span>
          <span style="font-family:Tahoma">
           Skia
          </span>
          <span style="font-family:宋体">
           公司被
          </span>
          <span style="font-family:Tahoma">
           Google
          </span>
          <span style="font-family:宋体">
           收购，
          </span>
          <span style="font-family:Tahoma">
           2007
          </span>
          <span style="font-family:宋体">
           年初，
          </span>
          <span style="font-family:Tahoma">
           Skia GL
          </span>
          <span style="font-family:宋体">
           源码被公开，
          </span>
          <span style="font-family:Tahoma">
           Skia
          </span>
          <span style="font-family:宋体">
           也是
          </span>
          <span style="font-family:Tahoma">
           Google Chrome
          </span>
          <span style="font-family:宋体">
           的图形引擎。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">
          Android
          <span style="font-family:宋体">
           的多媒体数据库采用
          </span>
          <span style="font-family:Tahoma">
           SQLite
          </span>
          <span style="font-family:宋体">
           数据库系统。数据库又分为共用数据库及私用数据库。用户可通过
          </span>
          <span style="font-family:Tahoma">
           ContentResolver
          </span>
          <span style="font-family:宋体">
           类（
          </span>
          <span style="font-family:Tahoma">
           Column
          </span>
          <span style="font-family:宋体">
           ）取得共用数据库。
          </span>
         </span>
        </p>
        <p>
         <span style="color:#444444">

          </span>
          <span style="font-family:Tahoma">
           Java
          </span>

           Dalvik Virtual Machine
          </span>
          <span style="font-family:宋体">
           ）。
          </span>
          <span style="font-family:Tahoma">
           Dalvik
          </span>
          <span style="font-family:宋体">
           虚拟机是一种“暂存器型态”（
          </span>
          <span style="font-family:Tahoma">

          </span>
          <span style="font-family:宋体">
           ）的
          </span>
          <span style="font-family:Tahoma">
           Java
          </span>
          <span style="font-family:宋体">

          </span>
         </span>
        </p>

         <span style="color:#444444">

          </span>

          </span>
          <span style="font-family:Tahoma">
           Dalvik
          </span>
          <span style="font-family:宋体">
           虚拟机来运行，让系统在运行程序时可达到优化。
          </span>
          <span style="font-family:Tahoma">
           Dalvik
          </span>
          <span style="font-family:宋体">
           虚拟机并非运行
          </span>
          <span style="font-family:Tahoma">
           Java
          </span>
          <span style="font-family:宋体">
           字节码（
          </span>

           Bytecode
          </span>

           ），而是运行一种称为
          </span>
          <span style="font-family:Tahoma">
           .dex
          </span>
          <span style="font-family:宋体">
           格式的文件。

           ID
