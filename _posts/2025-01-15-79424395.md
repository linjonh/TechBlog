    <p style="margin-top:0px;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     大家都知道，在不同操作系统、不同游览器里面默认显示的字体是不一样的，并且相同字体在不同操作系统里面渲染的效果也不尽相同，那么如何设置字体显示效果会比较好呢？下面我们逐步的分析一下：
    </p>
    <p style="margin-top:0px;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     例1（小米米官网）：font-family: "Arial","Microsoft YaHei","黑体","宋体",sans-serif;
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     例2（淘宝技术研发中心）：font: 12px/1.5 Tahoma,Helvetica,Arial,'宋体',sans-serif;
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     例3（加网 ）：font: 14px/1.5 'Microsoft YaHei',arial,tahoma,\5b8b\4f53,sans-serif;
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     例4（淘宝UED）：font: 12px/1 Tahoma,Helvetica,Arial,"\5b8b\4f53",sans-serif;
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     例5（一淘UX）:font-family: Helvetica, 'Hiragino Sans GB', 'Microsoft Yahei', '微软雅黑', Arial, sans-serif;
    </p>
    <p style="margin-top:0px;margin-bottom:15px;padding-top:0px;padding-bottom:0px;color:rgb(99,106,115);font-family:'Microsoft Yahei', Tahoma, Helvetica, Arial, SimSun, sans-serif;font-size:14px;line-height:21px;background-color:rgb(238,238,238);">
     font: 12px/1 Tahoma,Helvetica,Arial,"\5b8b\4f53",sans-serif;
    </p>
    <h4 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px 0px;font-size:1.5em;">
     一、首先我们看看各平台的默认字体情况
    </h4>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     1、Window下：
    </h6>
    <ul style="margin:1.5em 0px 1.5em 3em;padding-left:0px;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       宋体（SimSun）：Win下大部分游览器的默认字体，
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        宋体
       </code>
       在小字号下（如12px、14px）的显示效果还可以接受，但是字号一大就非常糟糕了，所以使用的时候要注意。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       微软雅黑（"Microsoft Yahei"）：从 Vista 开始，微软提供了这款新的字体，一款无衬线的黑体类字体，并且拥有Regular、Bold 两种粗细的字重，显著提高了字体的显示效果。现在这款字体已经成为Windows游览器中最值得使用的中文字体。从Win8开始，
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        微软雅黑
       </code>
       又加入了 Light 这款更细的字重，对于喜欢细字体的设计、开发人员又多了一个新的选择。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Arial：Win平台上默认的无衬线西文字体（为什么要说英文字体后面会解释），有多种变体，显示效果一般。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Tahoma：十分常见的无衬线字体，被采用为Windows 2000、Windows XP、Windows Server 2003及Sega游戏主机Dreamcast等系统的预设字型，显示效果比
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Arial
       </code>
       要好。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Verdana：无衬线字体，优点在于它在小字上仍结构清晰端整、阅读辨识容易。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       其他：Windows 下默认字体列表：
       <a href="https://www.microsoft.com/typography/fonts/product.aspx?pid=161" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
        微软官网
       </a>
       、
       <a href="https://en.wikipedia.org/wiki/List_of_typefaces_included_with_Microsoft_Windows" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
        维基百科
       </a>
       、
       <a href="https://support.office.com/zh-cn/article/%E4%B8%8D%E5%90%8C%E7%89%88%E6%9C%AC%E7%9A%84-Office-%E6%8F%90%E4%BE%9B%E7%9A%84%E5%AD%97%E4%BD%93-db1101fc-5cc0-4300-91cd-de7c79d907cd?CorrelationId=e2918255-27c6-4f99-b24c-6789900e8cb2&amp;ui=zh-CN&amp;rs=zh-CN&amp;ad=CN&amp;ocmsassetID=HA010282644" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
        Office字体
       </a>
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       结论：
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        微软雅黑
       </code>
       为Win平台上最值得选择的中文字体，但非游览器默认，需要设置；西文字体的选择以
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Arial
       </code>
       、
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Tahoma
       </code>
       等无衬线字体为主。
      </p>
     </li>
    </ul>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     2、Mac OS下：
    </h6>
    <ul style="margin:1.5em 0px 1.5em 3em;padding-left:0px;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       华文黑体（STHeiti）、华文细黑（STXihei）：属于同一字体家族系列，OS X 10.6 之前的简体中文系统界面默认字体，也是目前Chrome游览器下的默认字体，有 Regular 和 Bold 两个字重，显示效果可以接受，
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        华文细黑
       </code>
       也曾是我最喜欢的字体之一。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       黑体-简（Heiti SC）：从 10.6 开始，
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        黑体-简
       </code>
       代替
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        华文黑体
       </code>
       用作简体中文系统界面默认字体，苹果生态最常用的字体之一，包括iPhone、iPad等设备用的也是这款字体，显示效果不错，但是喇叭口设计遭人诟病。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       冬青黑体（ Hiragino Sans GB ）：听说又叫
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        苹果丽黑
       </code>
       ，日文字体
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Hiragino KakuGothic
       </code>
       的简体中文版，简体中文有 常规体和 粗体 两种，
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        冬青黑体
       </code>
       是一款清新的专业印刷字体，小字号时足够清晰，拥有很多人的追捧。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Times New Roman：Mac平台Safari下默认的字体，是最常见且广为人知的西文衬线字体之一，众多网页浏览器和文字处理软件都是用它作为默认字体。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Helvetica、Helvetica Neue：一种被广泛使用的传奇般的西文字体（这货还有专门的记录片呢），在微软使用山寨货的
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Arial
       </code>
       时，乔布斯却花费重金获得了
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Helvetica
       </code>
       苹果系统上的使用权，因此该字体也一直伴随着苹果用户，是苹果生态中最常用的西文字体。
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Helvetica Neue
       </code>
       为
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Helvetica
       </code>
       的改善版本，且增加了更多不同粗细与宽度的字形，共拥有51种字体版本，极大的满足了日常的使用。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       苹方（PingFang SC）：在Mac OS X EL Capitan上，苹果为中国用户打造了一款全新中文字体--
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        苹方
       </code>
       ，去掉了为人诟病的喇叭口，整体造型看上去更加简洁，字族共六枚字体：极细体、纤细体、细体、常规体、中黑体、中粗体。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       San Francisco：同样是Mac OS X EL Capitan上最新发布的西文字体，感觉和
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Helvetica
       </code>
       看上去差别不大，目前已经应用在Mac OS 10.11+、iOS 9.0+、watch OS等最新系统上。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       其他：Mac下默认字体列表：
       <a href="https://support.apple.com/zh-cn/HT202408" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
        苹果官网
       </a>
       、
       <a href="https://en.wikipedia.org/wiki/List_of_typefaces_included_with_OS_X" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
        维基百科
       </a>
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       结论：目前
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        苹方
       </code>
       和
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        San Francisco
       </code>
       为苹果推出的最新字体，显示效果也最为优雅，但只有最新系统才能支持，而
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        黑体-简
       </code>
       和
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
        Helvetica
       </code>
       可以获得更多系统版本支持，显示效果也相差无几，可以接受。
      </p>
     </li>
    </ul>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     3、Android系统：
    </h6>
    <ul style="margin:1.5em 0px 1.5em 3em;padding-left:0px;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       Droid Sans、Droid Sans Fallback：
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">

       </code>
       为安卓系统中默认的西文字体，是一款人文主义无衬线字体，而
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">

       </code>
       则是包含汉字、日文假名、韩文的文字扩展支持。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       结论：
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">

       </code>

      </p>
     </li>
    </ul>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     4、iOS系统：
    </h6>
    <ul style="margin:1.5em 0px 1.5em 3em;padding-left:0px;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       iOS系统的字体和Mac OS系统的字体相同，保证了Mac上的字体效果，iOS设备就没有太大问题。
      </p>
     </li>
    </ul>

     5、Linux：
    </h6>
    <ul style="margin:1.5em 0px 1.5em 3em;padding-left:0px;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       文泉驿点阵宋体：类似
       <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">

       </code>
       的衬线字体，一般不推荐使用。
      </p>
     </li>
     <li style="margin:.3em 0px;">
      <p style="margin-top:0px;margin-bottom:0px;">
       文泉驿微米黑：几乎是 Linux 社区现有的最佳简体中文字体。
      </p>
     </li>
    </ul>
    <h4 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px 0px;font-size:1.5em;">
     二、选择字体需要注意的问题
    </h4>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     1、字体的中英文写法：
    </h6>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     我们在操作系统中常常看到
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      宋体
     </code>

    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     小米公司优先使用
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      Helvetica Neue
     </code>
     这款字体以保证最新版本Mac用户的最佳体验，选择了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      Arial
     </code>
     作为Win下默认英文字体及Mac的替代英文字体；中文字体方面首选了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      微软雅黑
     </code>
     ，然后选择了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      冬青黑体
     </code>
     及
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      黑体-简
     </code>
     作为Mac上的替代方案；最后使用
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      文泉驿微米黑
     </code>
     兼顾了Linux系统。
    </p>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     2、淘宝
    </h6>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     鉴于淘宝网改版频率较频繁，这里截图保存了一下，
     <a href="http://oamfqhi9c.bkt.clouddn.com/blog_201607_taobao.jpg" rel="nofollow" style="color:rgb(0,154,97);text-decoration:none;border-bottom-width:1px;border-bottom-style:solid;padding-bottom:1px;background:transparent;">
      点此查看
     </a>
     。
    </p>
    <pre style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:1em;margin-top:1.5em;margin-bottom:1.5em;line-height:1.45;color:rgb(51,51,51);border:none;max-height:35em;"><code class="CSS" style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:1em;padding:0px;color:inherit;background:none;"><span class="hljs-selector-tag" style="font-weight:bold;">font</span>: 12<span class="hljs-selector-tag" style="font-weight:bold;">px</span>/1<span class="hljs-selector-class">.5</span> <span class="hljs-selector-tag" style="font-weight:bold;">tahoma</span>,<span class="hljs-selector-tag" style="font-weight:bold;">arial</span>,'<span class="hljs-selector-tag" style="font-weight:bold;">Hiragino</span> <span class="hljs-selector-tag" style="font-weight:bold;">Sans</span> <span class="hljs-selector-tag" style="font-weight:bold;">GB</span>','\5<span class="hljs-selector-tag" style="font-weight:bold;">b8b</span>\4<span class="hljs-selector-tag" style="font-weight:bold;">f53</span>',<span class="hljs-selector-tag" style="font-weight:bold;">sans-serif</span>;</code></pre>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     其实从图中明显看出淘宝网的导航及内容有着大量的衬线字体，鉴于淘宝网站大部分字号比较小，显示效果也还可以接受。代码中可以看出淘宝使用了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      Tahoma
     </code>
     、
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      Arial
     </code>
     作为首选英文字体，中文字体首选了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      冬青黑体
     </code>
     ，由于Win下没有预装该款字体，所以显示出了后面的宋体（
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      5b8b4f53
     </code>
     为汉字
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      宋体
     </code>
     用 unicode 表示的写法，不用
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      SimSun
     </code>
     是因为 Firefox 的某些版本和 Opera 不支持
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      SimSun
     </code>
     的写法）
    </p>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     3、简书
    </h6>
    <pre style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:1em;margin-top:1.5em;margin-bottom:1.5em;line-height:1.45;color:rgb(51,51,51);border:none;max-height:35em;"><code class="CSS" style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:1em;padding:0px;color:inherit;background:none;"><span class="hljs-selector-tag" style="font-weight:bold;">font-family</span>: "<span class="hljs-selector-tag" style="font-weight:bold;">lucida</span> <span class="hljs-selector-tag" style="font-weight:bold;">grande</span>", "<span class="hljs-selector-tag" style="font-weight:bold;">lucida</span> <span class="hljs-selector-tag" style="font-weight:bold;">sans</span> <span class="hljs-selector-tag" style="font-weight:bold;">unicode</span>", <span class="hljs-selector-tag" style="font-weight:bold;">lucida</span>, <span class="hljs-selector-tag" style="font-weight:bold;">helvetica</span>, "<span class="hljs-selector-tag" style="font-weight:bold;">Hiragino</span> <span class="hljs-selector-tag" style="font-weight:bold;">Sans</span> <span class="hljs-selector-tag" style="font-weight:bold;">GB</span>", "<span class="hljs-selector-tag" style="font-weight:bold;">Microsoft</span> <span class="hljs-selector-tag" style="font-weight:bold;">YaHei</span>", "<span class="hljs-selector-tag" style="font-weight:bold;">WenQuanYi</span> <span class="hljs-selector-tag" style="font-weight:bold;">Micro</span> <span class="hljs-selector-tag" style="font-weight:bold;">Hei</span>", <span class="hljs-selector-tag" style="font-weight:bold;">sans-serif</span>;</code></pre>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     自认为简书的阅读体验很棒，我们看看简书所用的字体，简书优先选择了
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      lucida
     </code>
     家族的系列字体作为英文字体，该系列字体在Mac和Win上都是预装的，并且有着不俗的表现；中文字体方面将
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      冬青黑体
     </code>
     作为最优先使用的字体，同样考虑了Linux系统。
    </p>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     各大公司的字体设置大同小异，这里不再一一举例查看，有兴趣的可以自己多多查看。
    </p>
    <h4 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px 0px;font-size:1.5em;">
     四、其他的一些注意点
    </h4>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     1、字体何时需要添加引号
    </h6>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     当字体具体某个取值中若有一种样式名称包含空格，则需要用双引号或单引号表示，如：
    </p>
    <pre style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:1em;margin-top:1.5em;margin-bottom:1.5em;line-height:1.45;color:rgb(51,51,51);border:none;max-height:35em;"><code class="CSS" style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:1em;padding:0px;color:inherit;background:none;"><span class="hljs-selector-tag" style="font-weight:bold;">font-family</span>: "<span class="hljs-selector-tag" style="font-weight:bold;">Microsoft</span> <span class="hljs-selector-tag" style="font-weight:bold;">YaHei</span>", "<span class="hljs-selector-tag" style="font-weight:bold;">Arial</span> <span class="hljs-selector-tag" style="font-weight:bold;">Narrow</span>", <span class="hljs-selector-tag" style="font-weight:bold;">sans-serif</span>;</code></pre>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     如果书写中文字体名称为了保证兼容性也会添加引号，如：
    </p>
    <pre style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:1em;margin-top:1.5em;margin-bottom:1.5em;line-height:1.45;color:rgb(51,51,51);border:none;max-height:35em;"><code class="CSS" style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:1em;padding:0px;color:inherit;background:none;"><span class="hljs-selector-tag" style="font-weight:bold;">font-family</span>: "黑体<span class="hljs-selector-tag" style="font-weight:bold;">-</span>简", "微软雅黑", "文泉驿微米黑";</code></pre>
    <h6 style="font-weight:500;line-height:1.2;color:rgb(51,51,51);margin:1.5em 0px;font-size:1em;">
     2、引用外部字体
    </h6>
    <p style="margin-top:1.5em;margin-bottom:1.5em;color:rgb(51,51,51);font-size:14px;line-height:22.4px;">
     大多数的中文字体由于版权原因不能随意使用，这里推荐一个免版权而且漂亮的中文字体
     <code style="font-family:'Source Code Pro', Consolas, Menlo, Monaco, 'Courier New', monospace;font-size:.93em;padding:2px 4px;color:rgb(199,37,78);background-color:rgb(249,242,244);">
      思源黑体
     </code>
     ，该字体为Adobe与Google推出的一款开源字体， 有七种字体粗细（ExtraLight、Light、Normal、Regular、Medium、Bold 和 Heavy），完全支持日文、韩文、繁体中文和简体中文，字形优美，依稀记得小米公司好像有使用过。
     <br/>
     鉴于中文字体的体积比较大（一般字库全一点的中文字体动辄几Mb），所以较少人会使用外部字体，如果真的需要引入，也可以考虑通过工具根据页面文字的使用多少单独生成中文字体，以减小文件大小。
    </p>
