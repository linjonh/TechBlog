    <h2>
     <strong>
      <span style="color:rgb(255,0,0)">
       一.码率（
       <span style="font-family:Calibri">
        Data Rate
       </span>
       <span style="font-family:宋体">
        ）
       </span>
      </span>
     </strong>
    </h2>
    <p>
     码率是指视频图像经过编码压缩后在单位时间内的数据流量，也叫
     <span style="color:rgb(255,0,0)">
      码流
     </span>
     ，是视频编码中画面质量控制中最重要的部分。同样分辨率下，压缩比越小，视频图像的码率就越大，画面质量就越高。
    </p>
    <p>
     也叫
     <span style="color:rgb(255,0,0)">
      比特率
     </span>
     ，表示经过压缩编码后的视音频数据每秒需要用多少个比特来表示，即把每秒显示的图像进行压缩后的数据量，一般采用的单位是
     <span style="font-family:Calibri">
      kbps
     </span>
     <span style="font-family:宋体">
      即千位每秒。
     </span>
    </p>
    <p>
     <span style="color:rgb(255,0,0)">
      码流
     </span>
     <span style="color:rgb(255,0,0)">
      /
     </span>
     <span style="color:rgb(255,0,0)">
      比特率
     </span>
     <span style="color:rgb(255,0,0)">
      /
     </span>
     <span style="color:rgb(255,0,0)">
      码率计算公式 基本的算法是：
     </span>
    </p>
    <p>
     <span style="color:rgb(46,117,181)">
      文件体积
     </span>
     <span style="color:rgb(46,117,181)">
      （
     </span>
     <span style="color:rgb(46,117,181)">
      kb
     </span>
     <span style="color:rgb(46,117,181)">
      ）
     </span>
     <span style="color:rgb(46,117,181)">
      =
      <span style="font-family:宋体">
       时间
      </span>
      <span style="font-family:Calibri">
       X
      </span>
      <span style="font-family:宋体">
       码率
      </span>
      <span style="font-family:Calibri">
       /8
      </span>
     </span>
     <span style="color:rgb(46,117,181)">
      &lt;!--
     </span>
     <span style="color:rgb(46,117,181)">
      这里时间单位是秒，码率除以
      <span style="font-family:Calibri">
       8
      </span>
      <span style="font-family:宋体">
       ，就不用说了
      </span>
     </span>
     <span style="color:rgb(46,117,181)">
      --&gt;
     </span>
    </p>
    <p>
     <span style="color:rgb(46,117,181)">
      码率
      <span style="font-family:Calibri">
       (kbps)=
      </span>
      <span style="font-family:宋体">
       文件大小
      </span>
      <span style="font-family:Calibri">
       (
      </span>
      <span style="font-family:宋体">
       字节
      </span>
      <span style="font-family:Calibri">
       )X8/
      </span>
      <span style="font-family:宋体">
       时间
      </span>
      <span style="font-family:Calibri">
       (
      </span>
      <span style="font-family:宋体">
       秒
      </span>
      <span style="font-family:Calibri">
       )/1000
      </span>
     </span>
    </p>
    <h2>
     <span style="color:rgb(255,0,0)">
      二.流量
     </span>
    </h2>
    <p>
     流量是什么？
     <span style="color:rgb(0,176,80)">
      流量是单位时间内收发的数据包的数量
     </span>
     。也就是说在我们观看视频的时候，流量就随之而来。
    </p>
    <p>
     <span style="color:rgb(91,155,213)">
      流量（
      <span style="font-family:Calibri">
       kb
      </span>
      <span style="font-family:宋体">
       ）
      </span>
      <span style="font-family:Calibri">
       =
      </span>
      <span style="font-family:宋体">
       观看视频时长（
      </span>
      <span style="font-family:Calibri">
       s
      </span>
      <span style="font-family:宋体">
       ）
      </span>
      <span style="font-family:Calibri">
       *
      </span>
      <span style="font-family:宋体">
       当前视频码率（
      </span>
      <span style="font-family:Calibri">
       kbps
      </span>
      <span style="font-family:宋体">
       ）
      </span>
      <span style="font-family:Calibri">
       /8
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(91,155,213)">
     </span>
    </p>
    <h2>
     <span style="color:rgb(255,0,0)">
      三.编码方式
     </span>
    </h2>
    <p>
     <span style="color:rgb(51,51,51)">
      所谓编码方式就是指通过特定的
     </span>
     <a href="http://baike.baidu.com/view/1508717.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:rgb(19,110,194)">
       压缩技术
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      ，将某个视频格式的文件转换成另一种视频格式文件的方式
     </span>
     <span style="color:rgb(51,51,51)">
      。
     </span>
    </p>
    <p>
     <span style="color:rgb(51,51,51)">
      目前
     </span>
     <a href="http://baike.baidu.com/view/2722588.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:rgb(19,110,194)">
       视频流
      </span>
     </a>
     <span style="color:rgb(51,51,51)">
      传输中最为重要的编解码标准有
     </span>
     <span style="color:rgb(51,51,51)">
      ：
     </span>
    </p>
    <p>
     <span style="color:rgb(91,155,213)">
      国际电联的
      <span style="font-family:Arial">
       H.261
      </span>
      <span style="font-family:宋体">
       、
      </span>
      <span style="font-family:Arial">
       H.263
      </span>
      <span style="font-family:宋体">
       ，
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(0,176,80)">
      运动静止图像专家组的
     </span>
     <a href="http://baike.baidu.com/view/254405.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:rgb(0,176,80)">
       M-JPEG
      </span>
     </a>
    </p>
    <p>
     <span style="color:rgb(46,117,181)">
      国际标准化组织运动图像专家组的
      <span style="font-family:Arial">
       MPEG
      </span>
      <span style="font-family:宋体">
       系列标准
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(63,63,63)">
      互联网上被广泛应用的还有
      <span style="font-family:Arial">
       Real-Networks
      </span>
      <span style="font-family:宋体">
       的
      </span>
      <span style="font-family:Arial">
       RealVideo
      </span>
     </span>
    </p>
    <p>
     <a href="http://baike.baidu.com/view/39784.htm" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:rgb(63,63,63)">
       微软公司
      </span>
     </a>
     <span style="color:rgb(63,63,63)">
      的
      <span style="font-family:Arial">
       WMT
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(63,63,63)">
      以及
      <span style="font-family:Arial">
       Apple
      </span>
      <span style="font-family:宋体">
       公司的
      </span>
      <span style="font-family:Arial">
       QuickTime
      </span>
      <span style="font-family:宋体">
       等。
      </span>
