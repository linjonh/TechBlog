    <h3>
     <a id="_3">
     </a>
     一、复习方面
    </h3>
    <h4>
     <a id="1_4">
     </a>
     1.数学
    </h4>
    <ul>
     <li>
      <p>
       因为2020年初疫情一直在家，闲着也是闲着，开始的就比较早，大概3月左右就开始了，基本上零基础开始，大一大二学的数学都忘光了。 刚开始看的武忠祥的基础班，开二倍速，感觉听的很舒服，讲的也很好，全是干货。张宇讲课比较幽默，汤家凤比较注重基础，选自己听着舒服的老师就好。
      </p>
     </li>
     <li>
      <p>
       划个重点：
       <strong>
        复习数学一定是做题！做题！做题！
        <br/>
        跟哪个老师不重要，看多少网课也不重要，一定是多做题，到后期多做模拟卷，题海战术才是王道！！！
       </strong>
      </p>
     </li>
     <li>
      <p>
       我考的是21年的数二，很简单，应该是近五六年以来最简单的一年了，前期准备的时候写完了660和330，然后在8月左右的时候二刷了一遍。多说一句，李永乐660真的推荐，能认认真真把660做下来的，基础绝对过关了，没有偏题怪题，全都是最基本的概念和定义，但同时难度也不小，刚开始会比较吃力，但认真理解后会有醍醐灌顶的感觉。
      </p>
     </li>
     <li>
      <p>
       我最后复习的时候基本上刷完了市面上所有知名的预测卷和模拟卷，张宇8+4，李林6+4，李永乐6套，汤家凤8套，合工大共创+超越，李正元5套，模拟卷平均120左右，最后考研140+（因为题确实简单。。）模拟卷的话，基本上李林6+4，合工大共创+超越是必做的，因为质量真的高，其他的例如李永乐、汤家凤有时间肯定多做，至于张宇的模拟卷…如果时间很充裕可以做，时间很紧张的话就算了吧。
      </p>
     </li>
     <li>
      <p>
       说到做题再多写一点吧，不需要特别看重真题，真题一定要刷，但没必要三遍四遍的刷，刷真题只是感受真题的难度和做真题的感觉，对于整体的提升并不大。对于想稳稳的考到120以上的同学，应该在把真题刷过一遍的基础上，多做模拟题和预测题。然后别忘了纠错整理。
       <strong>
        到了考前最后一两周的时候，也别忘了每天都做几道题，保持做题的感觉，这个很重要！！！
       </strong>
      </p>
     </li>
     <li>
      <p>
       最后，数学绝对绝对是拉开差距最大的学科了，我基本上学数学的时间比学其他所以科目加起来的时间还要长，按照真题出题的规律，一年简单一年难，2022年的数学肯定又要尸横遍野了，学弟学妹们加油努力吧。
      </p>
     </li>
    </ul>
    <h4>
     <a id="2__14">
     </a>
     2. 英语
    </h4>
    <p>
     英语最重要的就是两个东西：单词和真题，没有其他的。背单词建议APP和纸质书同时进行，效果最好，前期只背单词，单词背的差不多了，就可以开始写真题了，从后往前写，2010年之前的就当普通练习题随便写写就好了，2010年之后的就一定要认真写了，每一套至少写两遍，2015年之后的，每一套至少写4遍。我21年英二81，基本上就是按照这个路线，作文和翻译不用急着准备，11月准备就来得及。
    </p>
    <p>
     排几个坑吧：
    </p>
    <ul>
     <li>
      <h5>
       <a id="_20">
       </a>
       别去看背单词的网课…效率低的离谱，还没啥用，光听段子了。
      </h5>
     </li>
     <li>
      <h5>
       <a id="_21">
       </a>
       没必要做模拟题，英语刷真题就足够。
      </h5>
     </li>
     <li>
      <h5>
       <a id="_22">
       </a>
       别信什么万能作文模板，没有那种好事，想得高分还得自己努力。
      </h5>
     </li>
    </ul>
    <h4>
     <a id="3__24">
     </a>
     3. 专业课
    </h4>
    <p>
     直接写专业课吧，政治不想写了…想起那玩意就头疼。
    </p>
    <ul>
     <li>
      专业课是847 ：计算机网络和原理。 直接划重点：**不要相信考纲！！！**可以把这个考纲理解为考察重点， 很多考纲上没写的内容也会考，虽然考的分值不大，但如果不复习的话肯定是不会写的。
     </li>
     <li>
      谢希仁的计算机网络和唐朔飞的计算机组成原理，看这两个就够了。王道天勤之类的，基础差的可以看看网课，没必要做上面的题，那个是为408准备的，差别其实挺大的。
     </li>
     <li>
      复习内容的话，计组里面太偏硬件的东西不用看，然后计网建议认真学，多做课后题，因为复试还要考。
     </li>
    </ul>
    <h3>
     <a id="_32">
     </a>
     二、关于学院
    </h3>
    <ol>
     <li>
      <p>
       网安学院目前还在陕西西安的长安校区，但往太仓校区搬肯定也就是最近的事了，我们21届应该是研一在西安，研二去太仓。明年22届肯定是要搬的，至于什么时候搬就没人知道了。
      </p>
     </li>
     <li>
      <p>
       学院里的老师的话，有一部分是自动化过来的，也有很多新招的老师，总体都挺好的，没有特别坑的老师。
      </p>
     </li>
     <li>
      <p>
       西工大与计算机相关的学院有三个：计算机学院、软件学院还有就是网安，这三个学院考研的专业课基本上一样（网安是固定，计算机和软件是4选2，4个里面包括网安考的这两个）所以可以先正常复习，最后根据复习情况决定考哪个学院，根据往年情况的话，考研难度：计算机 &gt; 软件 &gt; 网安。
      </p>
     </li>
    </ol>
