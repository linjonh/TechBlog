     这一节我们来继续学习：S7-1200的编程语言和plc的工作原理和程序结构
     <br/>
     一、编程语言
     <br/>
     编程语言有梯形图（LAD）、函数块图（FBD）和结构化控制语言（SCL）这三种编程语言
     <br/>
     1.1.梯形图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6626ef993250046235911645bb5b9e04.png#pic_center">
      <br/>
      a：人员使用最多
      <br/>
      b：梯形图由触点、线圈和用方框表示的指令框组成，触点和线框组成的电路称程序段（也叫网络）
      <br/>
      c:分析时，可以想象在梯形图的左右两侧有一个左正右负的电压，触点工作时，有一个假想的“能流”流过线圈。能流只能从左向右流动
      <br/>
      1.2.函数块图
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f19755ee471f342912f85b4ff52b8c5d.png#pic_center">
       <br/>
       国内很少人用
       <br/>
       1.3.结构化控制语言
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a688a58c14632d6ec4acd00e353acce7.png#pic_center">
        <br/>
        高级语言
        <br/>
        适用于数据管理、过程优化、配方管理和数据计算、统计任务
        <br/>
        其中三个语言，梯形图和函数块图可以切换，只能在“添加新块”中选择SCL语言
        <br/>
        二、工作原理和用户结构
        <br/>
        2.1 逻辑运算
        <br/>
        触点串联实现可“与”运算
        <br/>
        触点并联可实现“或”运算
        <br/>
        常闭触点可实现“非运算”
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e71642ae70ce086a60ab5366fd099240.png#pic_center">
         <br/>
         这个逻辑运算为：
         <br/>
         KM=(KM+SB1)×SB2×FR (+：或；×：与)
         <br/>
         2.2.用户程序结构
         <br/>
         2.2.1.模块化编程
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/049108610e53ca5abb69e68673f24dd5.png#pic_center">
          <br/>
          块的数量无限制，受到存储器的影响
          <br/>
          被调用的代码块又可以调用别的代码块。从OB循环OB或启动OB开始，嵌套深度为16；从中断OB开始，嵌套深度为6
          <br/>
          2.2.2.组织块
          <br/>
          （1）程序循环组织块
          <br/>
          QB1是主程序，每次循环，调用一次OB1
          <br/>
          （2）启动组织块
          <br/>
          （3）中断组织块
          <br/>
          2.2.3函数和函数块（FB,FC）
          <br/>
          用户编写的子程序
          <br/>
          FB和FC块调用完返回它的代码块
          <br/>
          2.2.4数据块
          <br/>
          存放执行代码所需的数据区，无指令，自动分配地址
          <br/>
          有两种类型：
          <br/>
          a:供所有代码块使用数据，OB,FC,FB都可访问
          <br/>
          b:背景数据块供FB使用，但FB临时数据不用背景数据存储
          <br/>
          三、工作过程
          <br/>
          3.1CPU工作模式
          <br/>
          三种：RUN、STOP、STARTUP
          <br/>
          3.1冷启动与暖启动
          <br/>
          冷启动：执行冷启动，初始化所有输出，清除2所有数据
          <br/>
          暖启动：不对存储器复位，其他和冷启动一样
