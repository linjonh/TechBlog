    <p>
     EST : expressed sequences tags 表达序列标签
     <br/>
     WGS：whole genome sequencing
     <br/>
     miRNA(microRNA): 真核生物中一类内源性的具有调节功能的非编码基因，其大小大约为长度为20-25的核苷酸序列
     <br/>
     Run:
     <br/>
     Bioproject
     <br/>
     Biosample
     <br/>
     Experiment
     <br/>
     Accession list: 就是run号，可以根据这个号来下载raw data 数据
    </p>

     light, nitrate, phosphate, temperature

    <ol>
     <li>
      <p>
       登录SRA网页 https://www.ncbi.nlm.nih.gov/sra
      </p>
     </li>
     <li>
      <p>
       输入要搜索的物种名字，点击search
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/66e08722047aaf70e6728eb72c4ae7ad.png#pic_center">
        <br/>
        3、然后在Send to 下拉框中点击Run Selector, 然后点击Go
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/372c6d57f5d9cb04f330d7944b8e661a.png#pic_center">
         <br/>
         4、对搜索出来的结果打钩，如正方形方框所示，然后点击metadata, 就会下载每个run所对应的详细信息
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0a57ef7d1a0aafa2a41dc540fd479eb5.png#pic_center">
          <br/>
          5、对于下载下来的metadata数据，从excel表格打开，选择用逗号隔开，就可以把table数据转化成excel的表格数据。然后用关键词来搜索感兴趣的信息。
         </img>
        </img>
       </img>

    <h2>
     <a id="_27">
     </a>
     如何查找一类藻的转录组信息
    </h2>
    <p>
     1、登录sra网站，然后输入一个门的名称，如Dinoflagellates，单击search
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ffd36bf7e2c57925b2bb362a50af85b.png#pic_center">
      <br/>
      2、在右边的信息里，点击tree，就会显示出物种的分类信息，然后点击dinoflagellates, 如下面方框显示所示。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9edf54c7b1e80cee12a0f39ab6c12d42.png#pic_center">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d2b709c9791322b6b8882300a1019a17.png#pic_center"/>
       <br/>
       3、选择send to, 然后点击run selector, 最后点击Go
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/429f4711c0ff1a3c2ada64ab3c0d9871.png#pic_center"/>
     <br/>
     3、勾选好items，然后点击metadata，就下自动下载所有选定run的相关信息。

    </p>
