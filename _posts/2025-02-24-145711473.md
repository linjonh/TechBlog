    <p>
     华为近日正式推送
     <strong>
      原生鸿蒙操作系统 HarmonyOS NEXT 5.0.0.126
     </strong>
     ，带来了众多功能更新与优化。本文将详细介绍本次更新内容，帮助用户更好地了解和使用新功能。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/145d540c1c3249f599df70097acde059.png"/>
    </p>
    <hr/>
    <h3>
     🎨
     <strong>
      主题随动：个性化你的鸿蒙体验
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       心情空间主题
      </strong>
      ：新增更多表情，支持 DIY 搭配，带来更丰富的趣味体验。
     </li>
     <li>
      <strong>
       空气投篮主题
      </strong>
      ：新增交互玩法，可点击篮球进行投篮，为用户提供更畅快的解压方式。
     </li>
    </ul>
    <p>
    </p>
    <h3>
     🔄
     <strong>
      华为分享：增强跨设备传输能力
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       支持单次分享万余张图片
      </strong>
      ，适用于搭载 HarmonyOS 3 及以上版本的设备。
     </li>
    </ul>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/3450369144304fd0b7c4860c432a43c8.png"/>
    </p>
    <p>
    </p>
    <h3>
     📞
     <strong>
      通话功能优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       新增电源键通话控制功能
      </strong>
      ：
      <ul>
       <li>
        <strong>
         来电时
        </strong>
        ：按两下电源键拒接。
       </li>
       <li>
        <strong>
         去电时
        </strong>
        ：按一下电源键取消拨号。
       </li>
       <li>
        <strong>
         通话中
        </strong>
        ：按一下电源键挂断。
       </li>
       <li>
        <strong>
         静音铃声
        </strong>
        ：可通过电源键或音量键静音。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       设置路径
      </strong>
      ：
      <code>
       通话 &gt; 设置 &gt; 高级 &gt; 按电源键结束通话
      </code>
     </li>
    </ul>
    <p>
    </p>
    <h3>
     🔐
     <strong>
      隐私和安全中心
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       新增文件保密柜入口
      </strong>
      ：便捷加密保存图片、音视频及文档。
     </li>
     <li>
      <strong>
       开机铃声可手动开关
      </strong>
      ：路径
      <code>
       设置 &gt; 声音和振动 &gt; 开机铃声
      </code>
     </li>
    </ul>
    <p>
    </p>
    <h3>
     📌
     <strong>
      桌面优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       支持新增空白页
      </strong>
      ：
      <ul>
       <li>
        桌面捏合进入编辑界面后，可添加空白页。
       </li>
       <li>
        长按桌面页可整页拖动调整顺序。
       </li>
      </ul>
     </li>
    </ul>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/daba6a0ab8554cae9a78b75f5b3481c8.jpeg"/>
    </p>
    <p>
    </p>
    <h3>
     🔄
     <strong>
      系统优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       新增定时开关机功能
      </strong>
      ：
      <ul>
       <li>
        允许用户设置
        <strong>
         自动开关机时间
        </strong>
        。
       </li>
       <li>
        设置路径：
        <code>
         设置 &gt; 系统 &gt; 定时开关机
        </code>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       新增隔空传送功能
      </strong>
      ：
      <ul>
       <li>
        开启后，一抓一放即可
        <strong>
         隔空分享文件
        </strong>
        。
       </li>
       <li>
        设置路径：
        <code>
         设置 &gt; 系统 &gt; 快捷启动和手势 &gt; 隔空传送
        </code>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       超级中转站功能升级
      </strong>
      ：支持将其中的图片拖入图库保存。
     </li>
     <li>
      <strong>
       分屏/悬浮窗截屏优化
      </strong>
      ：
      <ul>
       <li>
        可自动识别分屏或悬浮窗。
       </li>
       <li>
        允许用户选择仅截取分屏或悬浮窗内容。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       新增隐私打码功能
      </strong>
      ：
      <ul>
       <li>
        截屏包含敏感信息时，将
        <strong>
         自动提醒
        </strong>
        并可一键打码。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       任务管理增强
      </strong>
      ：
      <ul>
       <li>
        在多任务中心
        <strong>
         下拉任务卡片可锁定任务
        </strong>
        ，清理时不会删除。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       新增 USB 连接 HUAWEI HiCar
      </strong>
      。
     </li>
    </ul>
    <p>
    </p>
    <h3>
     📝
     <strong>
      备忘录优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       智能文本识别
      </strong>
      ：
      <ul>
       <li>
        开启后可
        <strong>
         智能识别手机号、地址、日期、邮箱
        </strong>
        。
       </li>
       <li>
        支持拨号、导航、新建日程、邮件操作。
       </li>
       <li>
        设置路径：
        <code>
         右上角四点图标 &gt; 设置 &gt; 智能文本识别
        </code>
       </li>
      </ul>
     </li>
    </ul>
    <p>
    </p>
    <h3>
     🔗
     <strong>
      畅连优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       新增短信文件夹
      </strong>
      ，可便捷查看分类短信。
     </li>
    </ul>
    <p>
    </p>
    <h3>
     📂
     <strong>
      文件管理
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       新增大文件清理功能
      </strong>
      ：
      <ul>
       <li>
        通过
        <code>
         最近 &gt; 右上角扫把图标
        </code>
        进入
        <strong>
         大文件清理
        </strong>
        ，释放存储空间。
       </li>
      </ul>
     </li>
    </ul>
    <p>
    </p>
    <h3>
     📅
     <strong>
      日历优化
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       重要日卡片支持自定义背景图
      </strong>
      ：
      <ul>
       <li>
        可选择自己喜欢的照片作为背景。
       </li>
       <li>
        操作路径：
        <code>
         长按重要日卡片 &gt; 编辑
        </code>
       </li>
      </ul>
     </li>
    </ul>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/53cc9e0b914340f8ae500ad42966b0ad.png"/>

    </p>

     🏠
     <strong>
      智慧生活

        <strong>
         智慧生活 App

        远程访问 PC。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       新增智慧屏远程看家功能
      </strong>
      ：
      <ul>
       <li>
        可通过
        <strong>
         智慧生活 App
        </strong>
        连接家中智慧屏，随时随地查看家中情况。
       </li>
      </ul>
     </li>
    </ul>

    </p>
