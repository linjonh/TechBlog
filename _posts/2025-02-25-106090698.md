    <h3>
     <a id="_5">
     </a>
     创建数据库
    </h3>
    <pre><code>用Oracle Database Configuration Assistant创建数据库
</code></pre>
    <p>
     开始-&gt;所有程序-&gt;Oracle-……-&gt;配置和移植工具-&gt;Database Configuration Assistant（记住创建数据库时使用的用户名和密码）。
     <br/>
     也可以直接搜索Database Configuration Assistant。
    </p>
    <h5>
     <a id="1Database_Configuration_Assistant_11">
     </a>
     1.打开Database Configuration Assistant
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/903edb97d3e538399f1a815ade94c5fa.png"/>
    </p>
    <h5>
     <a id="2_14">
     </a>
     2.选择创建数据库
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a39998310ef6b0de3ed63de97579cd5.png">
      下一步
     </img>
    </p>
    <h5>
     <a id="3_17">
     </a>
     3.输入数据库名，设置口令，输入可插入数据库名
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5c04404d53b484c8eef4fce81987491.png">
      <em>
       数据库名和可插入数据库名
      </em>
      自己取，oracle口令设置有以下规范和建议。
      <br/>
      <strong>
       口令至少包含一个小写字母
      </strong>
      ，
      <strong>
       至少包含一个大写字母。 至少包含一个数字。 长度至少为 8 个字符。
      </strong>
      使用可包括下划线 (_), 美元符号 ($) 和井号 (#) 字符的数据库字符集。 如果包含特殊字符 (包括以数字或符号作为口令的开头), 请将口令加双引号。 不应为实际单词。
     </img>
    </p>
    <p>
     这里有涉及到CDB和PDB，就介绍一下：
     <br/>
     CDB全称为Container Database，中文翻译为数据库容器，PDB全称为Pluggable Database，即可插拔数据库。
     <br/>
     Oracle 12C引入了CDB与PDB的新特性，在ORACLE 12C数据库引入的多租用户环境（Multitenant Environment）中，
     <strong>
      允许一个数据库容器（CDB）承载多个可插拔数据库（PDB）
     </strong>
     。
    </p>
    <p>
     选择CDB数据库或PDB数据库
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ecbfe386dc1c11b72ba8655dc68b8d98.png">
      如上图，在
      <strong>
       创建为容器数据库
      </strong>
      前面打勾，就是CDB，不打勾就是PDB。
      <br/>
      如果你选择了CDB就输入一个自己取的
      <em>
       可插入数据库名
      </em>
      。
     </img>
    </p>
    <p>
     当然数据库文件位置等也是可以设置的。
    </p>

    <h3>
     <a id="_39">
     </a>
     连接数据库
    </h3>
    <h5>
     <a id="1SQL_Developer_40">
     </a>
     1.找到SQL Developer并打开
    </h5>
    <p>
     依然是在开始菜单里面找，也可以直接搜索。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9e16b3313ad1be7de0f3614f54ea48f3.png"/>
    </p>
    <h5>
     <a id="2_44">
     </a>
     2.新建连接
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/38c57afdc459089eea45b99174c94ebd.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3ac36c415d35dd1d7aef2062aa559f4b.png"/>
      取一个
      <em>
       连接名
      </em>
      ，
      <br/>
      <em>
       用户名
      </em>
      是sys，
      <br/>
      <em>
       口令
      </em>
      是你刚刚设置的口令，
      <br/>
      <em>
       角色
      </em>
      选择SYSDBA，
      <br/>
      <em>
       主机号
      </em>
      和
      <em>
       端口
      </em>
      默认，
      <br/>
      <em>
       SID
      </em>
      是你刚刚设置的数据库名，输入它。
     </img>
    </p>
    <p>
     <em>
      输入完成就测试一下，没问题就连接。
     </em>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a73f7eef5e9129c09dd7a6e13b017294.png"/>
     类似于插上电就是连接成功了。
    </p>
