    <h1 class="title-article" id="articleContentId">
     认证的服务号可以快速注册并认证小程序，但是有额度限制
    </h1>
   </div>
  </div>
 </div>

    <div style="margin-left:0;text-align:left;">
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        为方便公众号快捷接入小程序，并在各功能中使用小程序的服务，上线复用公众号资质注册小程序流程。快速注册认证小程序，
       </span>
      </span>
      <strong>
       <span style="color:#ff9900;">
        <span style="background-color:#ffffff;">
         无需重新提交主体材料、无需对公打款、无需支付300元认证费用
        </span>
       </span>
      </strong>
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        。
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        <strong>
         条件：
        </strong>
        已认证的企业、媒体、政府、其他组织类型公众号
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        <strong>
         限制：
        </strong>
       </span>
      </span>
      <strong>
       <span style="color:#fe2c24;">
        <span style="background-color:#ffffff;">
         非个体户类型（企业、媒体、政府、其他组织类型）公众号一个月可以复用资质注册5个小程序，个体户一个月可以复用资质注册1个小程序。
        </span>
       </span>
      </strong>
     </div>
     <div style="margin-left:0;">
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        <strong>
         开通入口：
        </strong>
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        登录公众号-小程序-小程序管理-添加-快速注册并认证小程序
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        <strong>
         相关规则：
        </strong>
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        1、复用资质创建的小程序默认与该公众号关联；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        2、复用资质创建的小程序默认关联该公众号，不占一个月新增关联13个小程序的限制；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        3、复用资质创建的小程序默认关联公众号，不下发模板消息，不默认出现在公众号资料页；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        4、一个公众号若存在“待完成注册”的小程序，不可发起复用资质创建小程序；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        5、一个公众号一个月内可复用资质注册小程序不多于5个。（创建成功才占用）；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        6、使用复用公众号资质注册小程序，将以本公众号的主体作为该小程序的开发者，可选择以本公众号运营者作为该小程序的管理员；
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        7、
       </span>
      </span>
      <strong>
       <span style="color:#fe2c24;">
        <span style="background-color:#ffffff;">
         若选择了复用认证资质，则小程序完成注册后即是“已认证”状态。
        </span>
       </span>
      </strong>
     </div>
     <div style="margin-left:0;">
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        <strong>
         创建流程：
        </strong>
       </span>
      </span>
     </div>
     <div style="margin-left:0;">
      <span style="color:#666666;">
       <span style="background-color:#ffffff;">
        第一步：登录公众号-&gt;小程序-&gt;小程序管理-&gt;添加-&gt;快速注册并认证小程序
       </span>
      </span>
     </div>
