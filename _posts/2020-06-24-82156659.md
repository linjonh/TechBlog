    <h1 class="title-article" id="articleContentId">
     撇去 Windows 的微软，又掉进了贪污贿赂的深坑
    </h1>
   </div>
  </div>
 </div>

  <div class="article_content clearfix" id="article_content">

    <div class="rich_media_content" id="js_content">
     <p style="min-height:1em;color:rgb(51,51,51);text-align:center;line-height:1.75em;">
      <img alt="640?wx_fmt=gif" class="__bg_gif" src="https://i-blog.csdnimg.cn/blog_migrate/9740fb08d32c71f4d4b35f3bdaa15340.gif" width="427"/>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       前一阵子，微软刚刚“舍弃”了运营多年的
       <a href="http://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&amp;mid=2650703396&amp;idx=1&amp;sn=6e884fd88c887770e72ac3705f0fc9a4&amp;chksm=bea6f7f789d17ee173d59b4176a1096a75c9de9f51fea50d99ea806299935006545ca5e8d9cb&amp;scene=21#wechat_redirect" rel="nofollow">
        Windows 众多业务
       </a>
       ，导致众人唏嘘不已。现在，一起几年前的事件又将 Word 和 Excel 等软件业务拉下了水。
      </span>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/c069ab4005ef9ee55813bb341d02ed58.jpeg"/>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <br/>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/8902df2bdffaccbf1c9b314d89e22746.png"/>
      </strong>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <span style="font-size:18px;letter-spacing:1px;text-align:justify;">
        微软
        <span style="font-size:18px;letter-spacing:1px;text-align:justify;">
         软件业务
        </span>
        涉及贪污贿赂问题，遭美国当局调查
       </span>
      </strong>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <br/>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       据
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        华尔街日报、路透社、Engadget 等多家外媒近日报道，
        <span style="text-align:justify;">
         2013 和 2014 年期间
        </span>
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         微软曾以极低的折扣价格将其 Word 和 Excel 等软件出售给了匈牙利的中间商公司，而这些中间商公司又
         <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
          以接近全额的价格将软件卖给了当地政府机构，并利用差价回扣贿赂政府官员接受合同——
         </span>
        </span>
       </span>
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       因涉及潜在贪污贿赂问题，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        微软
       </span>
       遭到了美国当局的调查。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       报道称，2010 年的时候微软曾大力拓展新兴市场，包括匈牙利等小型中等收入国家。
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       2012 年，微软就说服了匈牙利政府不要使用竞争对手的免费文字处理和电子表格软件。一份政府声明也显示，2012 年 11 月，当时微软的首席运营官 Kevin Turner 会见了匈牙利总理 Viktor Orban，后者在会面时还带上了一批小型企业的企业主。
      </span>
     </p>
     <p style="min-height:1em;text-align:center;line-height:normal;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/ff92fc8349f35735203817fb699bef8a.jpeg"/>
     </p>
     <p style="min-height:1em;text-align:center;line-height:normal;">
      <span style="font-size:14px;letter-spacing:1px;text-align:justify;color:rgb(136,136,136);">
       左图为 Kevin Turner，右图为 Viktor Orban
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       在接下来的两年时间里，微软每年向这些公司出售了大约 3000 万美元的软件，有的折扣率甚至高达 30%——但却无法得知这些销售中究竟有多少笔受到过严格审查，而这些“野蛮”的开拓举动毫不意外地带来了法律方面的挑战。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       据
       <span style="text-align:justify;">
        华尔街日报报道，
        <span style="text-align:justify;">
         2014 年的时候微软就曾获知匈牙利业务中存在潜在不当行为，公司当时就
        </span>
        已迅速采取行动并展开了内部调查。微软副首席法律顾问 David Howard 向
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         华尔街日报
        </span>
        表示，“我们一直遵守商业道德规范，绝不会在这些标准上让步。”为此，微软也一直在积极配合司法部和 SEC 的调查。
       </span>
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       “我们已经解雇了四名员工，包括匈牙利的负责经理 Istvan Papp。我们还在去年终止了与匈牙利 4 个合作伙伴的业务关系。”
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        David Howard 还
       </span>
       提到，
       <span style="text-align:justify;font-size:15px;letter-spacing:1px;">
        “我们一直在为自己提出的法律诉讼辩护”，并且加大了审查力度，提高了折扣方面的透明度。
       </span>
       从 2014 年开始微软已经彻底改变了处理折扣的方式，并向政府客户披露了更多信息。 “（我们已经）要求合作伙伴向客户清晰地传递折扣信息，并确保客户确认他们知道折扣。”
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        微软方面
       </span>
       还表示，它们严格要求合作伙伴遵守美国和当地法律的合同，并增加了基于人工智能的技术，以帮助识别合规风险。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <br/>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/35681423b96fcd60bcc7a569defc40bd.png"/>
      </strong>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <span style="font-size:18px;">
        微软不是第一次遭遇贿赂指控了
       </span>
      </strong>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       <br/>
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       虽然微软此次的涉嫌侵权行为是在海外进行的，但美国的“反海外腐败法”却赋予了美国当局在其他地区追究美国公司腐败行为的权力。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       根据美国 1977 年“反海外腐败法”（FCPA，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        Foreign Corrupt Practices Act
       </span>
       ）的规定，美国企业以及在美国股票交易所上市的企业禁止向外国官员行贿。因此，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        美国当局职权范围甚为广泛，可以调查和起诉任何地方发生的腐败指控。所以
       </span>
       ，包括微软在内的许多美国大公司都制定了流程来征求内部举报人的投诉，并向美国联邦机构报告潜在的违规行为。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        FCPA
       </span>
       专家指出，内部调查的成本各有不同，大型跨国企业平均要在每个国家花 200 万美元调查自己的运营，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        看有没有违反反贪污法，然后将结果交给政府，以求从轻罚款或者避免罚款。
       </span>
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       自 2009 年起，美国司法部发起了一百多宗诉讼，累计罚款超 20 亿美元。一般情况下，美国联邦调查人员并不会出现审查错误。近年来，微软也一直在调查各个国家的业务销售状况，这也不是微软第一次受到这样的审查。
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       早在 2013 年华尔街日报就曾报道，美国当局已经调查了微软在
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       中国、意大利、罗马尼亚、巴基斯坦和俄罗斯的潜在贿赂交易。
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       目前还不清楚美国是否会继续调查微软在其他国家的业务活动。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       微软还表示，有 170 多位员工负责调查和监管公司政微是否符合规定，包括符合反海外腐败法的规定。微软副总裁 John Frank 曾公开表示：“我们郑重看待指控，不管指控源来自何处，我们都会调查。我们还会进行前期培训、监管并审计，以确保全球的业务运营达到最高的法律和伦理标准。”
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       除去微软，事实上，技术圈的腐败问题一直很严重。仅去年，就有
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       三星、小米、华为等企业曾涉及贿赂和反腐指控。2017 年 2 月，三星集团的家族继承人李在镕因
       <a href="http://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&amp;mid=2650693988&amp;idx=1&amp;sn=7735987467e45315e639d25fbeb61558&amp;chksm=bea610b789d199a13fbc874d886f0d15629de00d9f08ee8a082489574661b27829cac880ad89&amp;scene=21#wechat_redirect" rel="nofollow">
        贿赂韩国前总统而
       </a>
       被判五年徒刑，引起业界轰动。2017
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       年 9 月
       <a href="http://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&amp;mid=2650689733&amp;idx=1&amp;sn=70992e31b86e94a582cbd594664ee018&amp;chksm=bea6211689d1a800e9436ae031aebffcbaf187111260d08f099b2bf1855e9ca08697142817a8&amp;scene=21#wechat_redirect" rel="nofollow">
        小米爆出内部腐败案
       </a>
       ，6 名内部员工利用职务便利索取供应商财物，严重损害公司利益，最终因涉嫌触犯“非国家工作人员受贿罪”而被公安机关批准逮捕。2017 年 12 月，华为消费者 BG 大中华区执行副总裁腾鸿飞因
       <a href="http://mp.weixin.qq.com/s?__biz=MjM5MjAwODM4MA==&amp;mid=2650692275&amp;idx=1&amp;sn=77e97da74069ac14eb6e4c282bb97bb5&amp;chksm=bea62b6089d1a276643a1dd9509ddf3ab6ce2c290d66462aa7ce6d35575456754c297728c778&amp;scene=21#wechat_redirect" rel="nofollow">
        涉嫌非国家工作人员受贿罪
       </a>
       ，被公安机关采取强制措施。近日，京东也未能幸免，多名内部员工因涉嫌非国家工作人员受贿罪，而被公安机关刑事拘留......
      </span>
     </p>
     <p style="min-height:1em;text-align:center;line-height:normal;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/487eba0703f582c01ddede199db21e68.jpeg">
       <br/>
      </img>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
      </span>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/71d6090e79cf441419041a7759a6a6b8.png">
        <br/>
       </img>
      </strong>
     </p>
     <p style="min-height:1em;text-align:center;line-height:1.75em;">
      <strong>
       <span style="font-size:18px;">
        微软和匈牙利政府各执一词，具体情况仍有待考究
       </span>
      </strong>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       据
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        华尔街日报报道，
       </span>
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       从 2012 年到 2014 年，微软向匈牙利第三方公司的销售额约为 3000 万美元——但是据微软称，这些折扣是由已离职的销售人员签署的。而匈牙利则称，这些
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        匈牙利
       </span>
       雇员的举动是被腐败的个人所误导的。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       对此，司法部和 SEC 都对华尔街日报的询问拒绝置评。在
      </span>
      <span style="font-size:15px;letter-spacing:1px;">
       对微软
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        匈牙利业务的负责经理 Istvan Papp
       </span>
       询问中，后者也只是告诉他们自己“收到了管理层的反馈”，但却拒绝进一步发表评论。后续调查具体如何，还有待各方跟进。
      </span>
     </p>
     <p style="min-height:1em;text-align:justify;line-height:1.75em;">
      <span style="font-size:15px;letter-spacing:1px;">
       但可以肯定的是，微软此次的反腐败调查对所有想要以资金收贿方式意谋“野蛮”扩展的互联网企业来说，都是一记警钟。
      </span>
     </p>
