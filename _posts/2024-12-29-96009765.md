    <h1 class="title-article" id="articleContentId">
     计算机编程语言（机器语言，汇编语言，高级语言）
    </h1>
   </div>
  </div>
 </div>

    <p>
     计算机语言：计算机语言通常是一个能完整、准确和规则地表达人们的意图,并用以指挥或控制计算机工作的“符号系统”。
     <br/>
     计算机语言通常分为三类：即机器语言，汇编语言和高级语言。
    </p>
    <h3>
     1. 机器语言
    </h3>
    <p>
     机器语言是用二进制代码表示的计算机能直接识别和执行的一种机器指令的集合。
     <span style="color:#ff0000;">
      这种指令集，称
     </span>
     <a href="http://baike.baidu.com/view/52.htm" rel="nofollow">
      <span style="color:#ff0000;">
       <u>
        机器码
       </u>
      </span>
     </a>
     <span style="color:#ff0000;">
      （machine code），
     </span>
     <span style="color:#ff0000;">
      是电脑的CPU可直接解读的数据。
     </span>
     <span style="color:#000000;">
      它是计算机
     </span>
     的设计者通过计算机的硬件结构赋予计算机的操作功能。机器语言具有灵活、直接执行和速度快等特点。
     <span style="color:#ff0000;">
      不同型号的计算机其机器语言是不相通的，按着一种计算机的机器指令编制的
     </span>
     <a href="http://baike.baidu.com/view/17674.htm" rel="nofollow">
      <span style="color:#ff0000;">
       <u>
        程序
       </u>
      </span>
     </a>
     <span style="color:#ff0000;">
      ，不能在另一种计算机上执行。
     </span>
    </p>
    <p>
     用机器语言编写程序,编程人员要首先熟记所用计算机的全部指令代码和代码的涵义。手编程序时,程序员得自己处理每条指令和每一数据的存储分配和输入输出,还得记住编程过程中每步所使用的工作单元处在何种状态。这是一件十分繁琐的工作,编写程序花费的时间往往是实际运行时间的几十倍或几百倍。而且,
     <span style="color:#ff0000;">
      编出的程序全是些0和1的指令代码,
     </span>
     直观性差,还容易出错。现在,除了计算机生产厂家的专业人员外,绝大多数程序员已经不再去学习机器语言了。
    </p>
    <p>
     一条指令就是
     <a href="http://baike.baidu.com/view/111847.htm" rel="nofollow">
      <u>
       <span style="color:#0066cc;">
        机器语言
       </span>
      </u>
     </a>
     的一个语句，它是一组有意义的
     <span style="color:#ff0000;">
      <strong>
       二进制代码
      </strong>
     </span>
     ，
     <span style="color:#006600;">
      <strong>
       指令的基本格式如，操作码字段和地址码字段，
      </strong>
     </span>
     <span style="color:#006600;">
      <strong>
       操作码（OperationCode，OP）用来表示该指令所要完成的操作（如加、减、乘、除、数据传送等），其长度取决于指令系统中的指令条数。地址码用来描述该指令的操作对象，它或者直接给出操作数，或者指出操作数的存储器地址或寄存器地址（即寄存器名）。
      </strong>
     </span>
    </p>
    <p>
     一条指令就是机器语言的一个语句，它是一组有意义的二进制代码。各计算机公司设计生产的计算机，其指令的数量与功能、指令格式、
     <a href="http://baike.baidu.com/view/889427.htm" rel="nofollow">
      <u>
       <span style="color:#0066cc;">
        寻址方式
       </span>
      </u>
     </a>
     、
     <a href="http://baike.baidu.com/view/286817.htm" rel="nofollow">
      <u>
       <span style="color:#0066cc;">
        数据格式
       </span>
      </u>
     </a>
     是差别的。从计算机的发展过程可以看到，由于构成计算机的基本硬件发展迅速，
     <strong>
      <span style="color:#ff0000;">
       计算机的更新换代很快，这就存在软件如何跟上硬件的问题
      </span>
     </strong>
     。
     <span style="color:#000099;">
      一台新机器推出交付使用时，只有少量
     </span>
     <a href="http://baike.baidu.com/view/7860.htm" rel="nofollow">
      <span style="color:#ff0000;">
       <u>
        系统软件
       </u>
      </span>
     </a>
     <span style="color:#000099;">
      （
      <span style="color:#ff0000;">
       如操作系统
      </span>
      等）可提交用户，大量软件是不断充实的，尤其是
     </span>
     <a href="http://baike.baidu.com/view/330120.htm" rel="nofollow">
      <span style="color:#000099;">
       <u>
        应用程序
       </u>
      </span>
     </a>
     <span style="color:#000099;">
      ，有相当一部分是用户在使用机器时不断产生的。为了缓解新机器的推出致使原有应用程序不能继续使用的问题，各个计算机公司生产的同一系列的计算机时，尽管其硬件实现方法可以不同，但
     </span>
     <a href="http://baike.baidu.com/view/178189.htm" rel="nofollow">
      <span style="color:#000099;">
       <u>
        指令系统
       </u>
      </span>
     </a>
     <span style="color:#000099;">
      、数据格式、I/O系统等保持相同，因而软件完全兼容。
     </span>
     当研制该系列计算机的新型号时，尽管指令系统可以有较大的扩充，但仍保留了原来的全部指令，
     <span style="color:#ff0000;">
      使软件向上兼容，即旧机型上的软件不加修改即可在新机器上运行。
     </span>
    </p>
    <p>
     说到这里大家明白为什么电脑即计算机，
     <span style="color:#ff0000;">
      <strong>
       计算机即电脑
      </strong>
     </span>
     了吧，原来无论它表现得多么地智能，其
     <strong>
      实质也不过是这些二进制代码运算的结果
     </strong>
     。但是二进制代码使用起来太费劲了，于是后来就在其基础上发展出的
     <span style="color:#ff0000;">
      汇编语言
     </span>
     ，但是汇编依然看起来不直观，于是最终就出现了现代编程所采用的
     <span style="color:#ff0000;">
      高级语言
     </span>
     。这是计算机编程语言的发展。有了编程语言，按说就可以使这些机器为我所用了。实际还没那么简单。
     <strong>
      <span style="color:#006600;">
       首先每做一件事，都要重复编一大堆的操作程序，工作量很大
      </span>
     </strong>
     。其次，现在咱们都看到了，使用计算机的人，大大多数，都不是计算机专业的，甭说编程了，操作步骤复杂一点都要抵制。
     <strong>
      <span style="color:#009900;">
       所以，还需要一个
       <span style="color:#ff0000;">
        操作系统
       </span>
       ，将复杂的操作留给自己，把简单的操作，呈现给用户。
      </span>
     </strong>
    </p>
    <h3>
     2.汇编语言
    </h3>
    <p>
     为了克服机器语言难读、难编、难记和易出错的缺点,人们就用与代码指令实际含义相近的英文缩写词、字母和数字等符号来取代指令代码(如用ADD表示运算符号“＋”的机器代码),于是就产生了汇编语言。所以说,汇编语言是一种用助记符表示的仍然面向机器的计算机语言。汇编语言亦称符号语言。汇编语言由于是采用了助记符号来编写程序,比用机器语言的二进制代码编程要方便些,在一定程度上简化了编程过程。
     <span style="color:#ff0000;">
      汇编语言的特点是用符号代替了机器指令代码, 而且助记符与指令代码一一对应,基本保留了机器语言的灵活性。
     </span>
     使用汇编语言能面向机器并较好地发挥机器的特性,得到质量较高的程序。
    </p>
    <p>
     汇编语言中由于使用了助记符号,用汇编语言编制的程序送入计算机,计算机不能象用机器语言编写的程序一样直接识别和执行（
     <strong>
      <span style="color:#ff0000;">
       机器语言程序,它一经被安置在内存的预定位置上,就能被计算机的CPU处理和执行。）
      </span>
     </strong>
     必须通过
     <span style="color:#009900;">
      预先放入计算机的
      <strong>
       <span style="color:#ff0000;">
        “汇编程序“
       </span>
      </strong>
      的加工和翻译,
     </span>
     才能变成能够被计算机识别和处理的二进制代码程序。用汇编语言等非机器语言书写好的符号程序称源程序,
     <span style="color:#000099;">
      <strong>
       运行时汇编程序要将源程序翻译成目标程序。
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#ff0000;">
      汇编语言
     </span>
     像机器指令一样,是硬件操作的控制信息,因而
     <span style="color:#ff0000;">
      仍然是面向机器的语言,
     </span>
     使用起来还是比较繁琐费时,通用性也差。汇编语言是低级语言。但是,汇编语言用来编制系统软件和过程控制软件,其目标程序占用内存空间少,运行速度快,有着高级语言不可替代的用途。
    </p>
    <h3>
     3.高级语言
    </h3>
    <p>
     <span style="color:#ff0000;">
      不论是机器语言还是汇编语言都是面向硬件的具体操作的
     </span>
     ，语言对机器的过分依赖，要求使用者必须对硬件结构及其工作原理都十分熟悉，这对非计算机专业人员是难以做到的,对于计算机的推广应用是不利的。计算机事业的发展,促使人们去寻求一些与人类自然语言相接近且能为计算机所接受的语意确定、规则明确、自然直观和通用易学的计算机语言。这种与自然语言相近并为计算机所接受和执行的计算机语言称高级语言。
     <span style="color:#ff0000;">
      高级语言是
     </span>
     <span style="color:#ff0000;">
      面向用户的语言
     </span>
     。
     <span style="color:#000099;">
      无论何种机型的计算机, 只要配备上相应的
     </span>
     <span style="color:#ff0000;">
      高级语言的编译或解释程序
     </span>
     ,
     <span style="color:#333399;">
      则用该高级语言编写的程序就可以通用
     </span>
     。
    </p>
    <p>
     <strong>
      总结：
     </strong>
     机器语言是用二进制代码也就是0或1用于程序设计，汇编语言就是用助记符等一些符号代替二进制代码。这里就有一个问题了，计算机能读懂的只有机器指令，也就是二进制代码，那么计算机如何能理解用汇编语言编写的程序呢？这时，就需要一个能够将汇编指令转换成机器指令的翻译程序---编译器，程序员用汇编语言写出源程序，在用汇编编译器将其编译为机器指令，由计算机最终执行。
    </p>
    <p>
     汇编语言在机器语言的基础上引入了助记符，但是这些助记符量大难记，因此，我们又有了高级语言。定义上很清楚的表达了高级语言，他以人类日常语言为基础，这样的话，更易于编写，而且有较高的可读性。
    </p>
    <h3>
     <span style="color:#7c79e5;">
      <strong>
       高级语言可以实现计算机语言跨平台运行。
      </strong>
     </span>
    </h3>
    <h3>
     三种计算机语言如何在计算机上执行？
    </h3>
    <p>
     机器语言：机器语言可以直接在计算机上执行。
    </p>
    <p>
     汇编语言和高级语言：用某种汇编语言或高级语言编写的程序称为源程序，源程序不能直接在计算机上执行，如果源程序是用汇编语言编写的，则需要一个汇编程序将其翻译成目标程序后才能执行。如果源程序使用某种高级语言编写的，则需要对应的解释程序或编译程序对其进行翻译，然后在计算器上运行。
    </p>
    <h3>
     解释程序与编译程序比较？
    </h3>
    <p>
     解释程序也成为解释器，他或者直接执行源程序，或者将源程序直接翻译成中间代码后再加以执行，而编译程序也称为编译器则是将源程序翻译成目标语言程序，然后在计算机上运行目标程序。
    </p>
    <p>
     两种语言处理程序的
     <strong>
      根本区别
     </strong>
     是：在编译方式下，机器上运行的是与源程序等价的目标程序，编译程序和源程序都不参加目标程序的执行过程；而在解释方式下，解释程序和源程序要参与到程序的运行过程中，运行程序的控制权在解释程序。
    </p>
