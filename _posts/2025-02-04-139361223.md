    <h3>
     <a id="OllamaLlama_19">
     </a>
     什么是
     <code>
      Ollama
     </code>
     ，它与
     <code>
      Llama
     </code>
     是什么关系？
    </h3>
    <p>
     <code>
      Ollama
     </code>
     是一个开源的 LLM（大型语言模型）服务工具，用于简化在本地运行大语言模型，降低使用大语言模型的门槛，使得大模型的开发者、研究人员和爱好者能够在本地环境快速实验、管理和部署最新大语言模型，包括如
     <code>
      Llama 3
     </code>
     、
     <code>
      Phi 3
     </code>
     、
     <code>
      Mistral
     </code>
     、
     <code>
      Gemma
     </code>
     等开源的大型语言模型。
    </p>
    <p>
     <code>
      Ollama
     </code>
     目前支持以下大语言模型：https://ollama.com/library
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f205db97317c080aa0d108e1ad3213af.jpeg"/>
    </p>
    <p>
     Ollama支持的大模型列表
    </p>
    <p>
     因此，
     <code>
      Ollama
     </code>
     与
     <code>
      Llama
     </code>
     的关系：
     <code>
      Llama
     </code>
     是大语言模型，而
     <code>
      Ollama
     </code>
     是大语言模型（不限于
     <code>
      Llama
     </code>
     模型）便捷的管理和运维工具
    </p>
    <h3>
     <a id="Ollama_32">
     </a>
     安装
     <code>
      Ollama
     </code>
     大语言模型工具
    </h3>
    <p>
     <code>
      Ollama
     </code>
     提供了
     <strong>
      MacOS
     </strong>
     、
     <strong>
      Linux
     </strong>
     和
     <strong>
      Windows
     </strong>
     操作系统的安装包，大家可根据自己的操作系统，下载安装即可：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e493209b63af8fa143ab59bfef52e102.jpeg"/>
    </p>
    <p>
     Ollama下载页面
    </p>
    <p>
     安装包下载之后的安装过程，和日常安装其他软件没有差别，包括点击
     <code>
      Next
     </code>
     以及
     <code>
      Install
     </code>
     等安装
     <code>
      ollama
     </code>
     到命令行。安装后续步骤中，我们可无需安装任何模型（默认是
     <code>
      Llama 3
     </code>
     ），因为我们在上文中已经安装了
     <code>
      Llama 3 8B
     </code>
     大模型，后面可以直接使用。
    </p>
    <p>
     当然，假如没有根据我的前面博文安装
     <code>
      Llama 3 8B
     </code>
     模型，在安装
     <code>
      Ollama
     </code>
     过程中，也可以一起进行安装。

    <h3>
     <a id="Llama_3_8BGGUF_Ollama_62">
     </a>
     基于
     <code>
      Llama 3 8B
     </code>
     GGUF 模型文件创建
     <code>
      Ollama
     </code>
     模型
    </h3>
    <p>
     在我们存放
     <code>
      Llama 3 8B
     </code>
     的 GGUF 模型文件目录中，创建一个文件名为
     <code>
      Modelfile
     </code>
     的文件，该文件的内容如下：
    </p>
    <pre><code class="prism language-c">FROM <span class="token punctuation">.</span><span class="token operator">/</span>Meta<span class="token operator">-</span>Llama<span class="token operator">-</span><span class="token number">3</span><span class="token operator">-</span><span class="token number">8</span>B<span class="token operator">-</span>Instruct<span class="token punctuation">.</span>Q4_K_M<span class="token punctuation">.</span>gguf   
</code></pre>
    <p>
     然后在控制台，使用这个文件创建
     <code>
      Ollama
     </code>
     模型，这里我把
     <code>
      Ollama
     </code>
     的模型取名为
     <strong>
      Llama-3-8B
     </strong>
     ：
    </p>
    <pre><code class="prism language-c">$ ollama create Llama<span class="token operator">-</span><span class="token number">3</span><span class="token operator">-</span><span class="token number">8</span>B <span class="token operator">-</span>f <span class="token punctuation">.</span><span class="token operator">/</span>Modelfile   transferring model data   using existing layer sha256<span class="token operator">:</span><span class="token number">647</span>a2b64cbcdbe670432d0502ebb2592b36dd364d51a9ef7a1387b7a4365781f   creating new layer sha256<span class="token operator">:</span><span class="token number">459</span>d7c837b2bd7f895a15b0a5213846912693beedaf0257fbba2a508bc1c88d9   writing manifest   success   
</code></pre>
    <p>
     最后，通过
     <code>
      Ollama
     </code>
     启动我们刚创建的大语言模型：
    </p>
    <pre><code class="prism language-c">ollama run Llama<span class="token operator">-</span><span class="token number">3</span><span class="token operator">-</span><span class="token number">8</span>B   
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/12d9f6afadb8751aa7fcc397237d6aca.png"/>
    </p>
    <p>
     Ollama启动成功输出
    </p>
    <p>
     启动完毕，其实我们已经有了一个控制台聊天界面，可以通过控制台与
     <code>
      Llama-3-8B
     </code>
     聊天了
    </p>
    <p>
     如果我们不想要这个模型了，也可以通过命令行删除模型文件：
     <code>
      ollama rm Llama-3-8B
     </code>
    </p>
    <p>
     <code>
      Ollama
     </code>
     存放模型文件根目录：
     <code>
      ~/.ollama
     </code>
    </p>
    <h3>
     <a id="Ollama_Web__94">
     </a>
     部署
     <code>
      Ollama
     </code>
     大模型 Web 可视化聊天界面
    </h3>
    <p>
     控制台聊天对话界面体验总归是不太好，接下来部署 Web 可视化聊天界面。
    </p>
    <p>
     首先，下载
     <code>
      ollama-webui
     </code>
     Web 工程代码：
     <code>
      git clone https://github.com/ollama-webui/ollama-webui-lite
     </code>
    </p>
    <p>
     然后切换
     <code>
      ollama-webui
     </code>
     代码的目录：
     <code>
      cd ollama-webui-lite
     </code>
    </p>
    <p>
     设置 Node.js 工具包镜像源，以接下来下载 Node.js 的依赖包更加快速：
     <code>
      npm config set registry http://mirrors.cloud.tencent.com/npm/
     </code>
    </p>
    <p>
     安装 Node.js 依赖的工具包：
     <code>
      npm install
     </code>
    </p>
    <p>
     最后，启动 Web 可视化界面：
     <code>
      npm run dev
     </code>
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/7e59dca858489fb947a0f9065b3c47e2.jpeg"/>
    </p>
    <p>
     Web对话机器人部署成功输出
