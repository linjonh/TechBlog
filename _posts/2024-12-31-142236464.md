    <h2>
     <a id="__BL___1">
     </a>
     <strong>
      解锁BL锁
     </strong>
    </h2>
    <p>
     1. 下载安装 miflash_unlock：
     <a href="https://miuiver.com/miunlock/" rel="nofollow">
      https://miuiver.com/miunlock/
     </a>
     ，登录小米账号（需要和解锁设备绑定的账号一致）
     <br/>
     <img alt="image.png" src="https://img-blog.csdnimg.cn/img_convert/23fb26bcd74fa59c4d69cf34d78c03d5.png"/>
    </p>
    <p>
     2. 连接手机，关机后按住 “音量-” + 开机键进入 fastboot 模式，根据提示解锁手机
     <br/>
     <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/0e269314644db5eed7afb0fe3d780557.png"/>
    </p>
    <p>
     如果连接不上设备，点击右上角设置按钮【驱动检测】，手机进入recovery模式下重新连接，等待驱动安装完成。
     <br/>
     <img alt="image.png" src="https://img-blog.csdnimg.cn/img_convert/28d1098ec350ef8033a24d0155573788.png"/>
    </p>
    <p>
     如果还是连不上，参考下面链接的解决办法
     <br/>
     <a href="https://miuiver.com/usb3-fix/" rel="nofollow">
      https://miuiver.com/usb3-fix/
     </a>
    </p>
    <p>
     3. 重启手机开发者模式中查看解锁状态
     <br/>
     <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/e9f80cde83194f1114f19da3f6bb53d3.png"/>
    </p>
    <h2>
     <a id="_____19">
     </a>
     <strong>
      刷机
     </strong>
    </h2>
    <p>
     ROM下载：
     <a href="https://magiskcn.com/rom" rel="nofollow">
      https://magiskcn.com/rom
     </a>
    </p>
    <p>
     1. 下载小米线刷工具：
     <a href="https://xiaomirom.com/download-xiaomi-flash-tool-miflash/" rel="nofollow">
      https://xiaomirom.com/download-xiaomi-flash-tool-miflash/
     </a>
    </p>
    <p>
     2.
     <a href="https://xiaomirom.com/rom/mi-5c-meri-china-fastboot-recovery-rom/#%E4%B8%8B%E8%BD%BD-%E5%B0%8F%E7%B1%B3-5c-%E5%BC%80%E5%8F%91%E7%89%88%E5%86%85%E6%B5%8B%E7%89%88-fastboot-%E7%BA%BF%E5%88%B7%E5%8C%85" rel="nofollow">
      下载设备开发版线刷包
     </a>
     ，并解压线刷包
     <br/>
     <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/6d9b0419249e833dd8882a8f2249cb25.png"/>
    </p>
    <p>
     3. 打开miflash
    </p>
    <p>
     4. 连接手机，关机，按住 “开机” + “音量-” 进入fastboot模式
    </p>
    <p>
     5. 选择线刷包解压目录开始刷机
     <br/>
     <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/ca39292c1b8bced684fc79593e0c4f96.png">
      <br/>
      等待刷机成功
      <br/>
      <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/6abfbe53c20d263489e91cfed6bef0b3.png"/>
     </img>
    </p>
    <p>
     6. 刷机成功后，重新解除BL锁
    </p>
    <p>
     7. 打开手机管家——应用管理——权限——ROOT权限——开启ROOT权限
     <br/>
     <img alt="截图.png" src="https://img-blog.csdnimg.cn/img_convert/0e34fddfd9f7e7e585988a796d82ad6e.png"/>
     <br/>
     结果发现root工具一直下载失败
    </p>
    <h2>
     <a id="__Magiskroot___44">
     </a>
     <strong>
      使用Magisk进行root
     </strong>
    </h2>
    <p>
     下载 Magisk：
     <a href="https://github.com/topjohnwu/Magisk/releases">
      https://github.com/topjohnwu/Magisk/releases
     </a>
    </p>
    <p>
     1. 安装 Magisk
    </p>
    <pre><code>adb install Magisk-v27.0.apk
</code></pre>
    <p>
     2. 解压线刷包中的 boot.img，并 push 到手机
    </p>
    <pre><code>adb push boot.img /sdcard/boot.img
</code></pre>
    <p>
     3. Magisk中【安装】【下一步】【选择并修补一个文件】【选择boot.img】，把处理完后的magisk_patched 下载到电脑 Android sdk\platform-tools 目录（
     <a href="https://developer.android.com/tools/releases/platform-tools?hl=zh-cn" rel="nofollow">
      这里下载 Android SDK Platform Tools
     </a>
     ）
    </p>
    <pre><code>adb pull /sdcard/Download/magisk_patched-27000_RIC3F.img D:\App\android\sdk\platform-tools
</code></pre>
    <p>
     4. 使用 adb 命令将设备重启到 fastboot 模式
    </p>
    <pre><code>adb reboot bootloader
</code></pre>
    <p>
     5. 把 magisk_patched 刷新到系统
    </p>
    <pre><code># 切换到platform-tools目录
cd D:\App\android\sdk\platform-tools

# 刷入 magisk_patched
fastboot flash boot magisk_patched-27000_RIC3F.img

# 完成后运行重启手机命令
fastboot reboot
</code></pre>
    <p>
     6. 检测SU权限
    </p>
    <pre><code>adb shell
su
</code></pre>
