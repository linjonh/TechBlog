    <h2>
     <a id="_5">
     </a>
     一、算法
    </h2>
    <p>
     1.
     <code>
      程序=算法+数据结构
     </code>
     （由科学家尼克劳斯·沃斯提出）
     <br/>
     2.算法的设计一般采用
     <code>
      由粗到细，由抽象到具体的逐步求精
     </code>
     的方法。
     <br/>
     3.算法的必要满足条件：
     <code>
      确定性，有穷性，可行性，0个或多个输入，至少1个或多个输出
     </code>
     。
     <br/>
     4.关于算法需要考虑的方面：算法设计，算法表示，算法分析。
     <br/>
     5.算法与程序的区别
    </p>
    <ul>
     <li>
      算法是解决问题的方法。
     </li>
     <li>
      程序是算法的具体代码实现
     </li>
     <li>
      算法是程序设计的核心，算法的好坏直接决定了程序的效率。
     </li>
     <li>
      算法与程序是相应的，但不是一一对应的。
     </li>
     <li>
      程序可以不用满足算法的有穷性。
     </li>
    </ul>
    <p>
     6.算法的表示
     <br/>
     1.
     <code>
      自然语言
     </code>
     （缺点：容易产生二义性）
     <br/>
     2.流程图
     <br/>
     3.程序设计语言
     <br/>
     4.
     <code>
      伪代码（PDL）
     </code>
    </p>
    <p>
     7.评价算法的标准
     <br/>
     1.
     <code>
      正确性
     </code>
     ：评价一个算法优劣的最重要的标准。
     <br/>
     2.算法的复杂度
    </p>
    <ul>
     <li>
      <code>
       时间复杂度
      </code>
      ：执行算法所需要的计算工作量。
     </li>
     <li>
      <code>
       空间复杂度
      </code>
      ：执行算法所需要的内存空间。
     </li>
    </ul>
    <p>
     3.
     <code>
      可读性
     </code>
     <br/>
     4.
     <code>
      健壮性
     </code>
    </p>
    <h2>
     <a id="_33">
     </a>
     二、数据结构
    </h2>
    <h3>
     <a id="1_34">
     </a>
     1.基本概念
    </h3>
    <p>
     1.数据元素：
     <code>
      数据元素
     </code>
     是
     <code>
      数据
     </code>
     的基本单位。
     <br/>
     2.数据对象：数据对象是性质相同的数据元素的集合。
     <br/>
     3.数据结构：是指由某一数据对象中所有数据成员之间的关系组成的集合。
     <br/>
     4.常见的逻辑结构（考试考到了，选错了呜呜呜）：
    </p>
    <ul>
     <li>
      集合结构
     </li>
     <li>
      线性结构
     </li>
     <li>
      树形结构
     </li>
     <li>
      网状结构
      <br/>
      也可以分为
      <code>
       线性
      </code>
      结构（线性表，栈和队列，串，一维数组）和
      <code>
       非线性
      </code>
      结构（集合，树形，网状）。
      <br/>
      5.常见的存储结构
     </li>
     <li>
      <code>
       顺序存
      </code>
      储结构：把逻辑上
      <code>
       相邻
      </code>
      的数据元素，存储在
      <code>
       物理位置也相邻
      </code>
      的存储单元里。
     </li>
     <li>
      <code>
       链式
      </code>
      存储结构：各个数据元素存储位置随意，不要求逻辑上相邻的数据元素在物理位置上相邻。
     </li>
    </ul>
    <h3>
     <a id="2_47">
     </a>
     2.数据结构
    </h3>
    <p>
     1.没有前继结点的称为“
     <code>
      根结点
     </code>
     ”。
     <br/>
     2.没有后继结点的称为“
     <code>
      叶子结点
     </code>
     ”。
     <br/>
     3.没有根结点或叶子结点的数据结构一定是非线性结构。
     <br/>
     4.有且只有一个根结点的数据结构可能是线性结构，也可能是非线性结构。
    </p>
    <h3>
     <a id="3_52">
     </a>
     3.顺序表的插入和删除操作
    </h3>
    <p>
     1.顺序表的
     <code>
      插入操作
     </code>
    </p>
    <ul>
     <li>
      最好情况：插入的是尾元素，移动次数为
      <code>
       0
      </code>
      。
     </li>
     <li>
      最坏情况：插入的是第一个元素，移动次数为
      <code>
       n
      </code>
      。
     </li>
     <li>
      平均情况：在i位置插入新元素，移动次数为
      <code>
       n-i+1
      </code>
      。
     </li>
     <li>
      算法的平均时间复杂度为O(n)。
     </li>
    </ul>
    <p>
     2.顺序表的
     <code>
      删除操作
     </code>
    </p>
    <ul>
     <li>
      最好情况：删除的是尾元素，移动次数为
      <code>
       0
      </code>
      。
     </li>
     <li>
      最坏情况：删除的是第一个元素，移动次数为
      <code>
       n-1
      </code>
      。
     </li>
     <li>
      平均情况：在i位置删除一个元素，移动次数为
      <code>
       n-i
      </code>
      。
     </li>
     <li>
      算法的平均时间复杂度为O(n)。
     </li>
    </ul>
    <h3>
     <a id="4_65">
     </a>
     4.栈和队列
    </h3>
    <p>
     1.栈：先进后出
     <br/>
     2.队列：先进先出
     <br/>
     3.
     <code>
      循环队列
     </code>
     的相关操作
    </p>
    <ul>
     <li>
      初始状态：
      <code>
       Q.front=Q.rear
      </code>
     </li>
     <li>
      队首指针进1（出队）：
      <code>
       Q.front=（Q.front+1）%Maxsize
      </code>
     </li>
     <li>
      队尾指针进1（入队）：
      <code>
       Q.rear=（Q.rear+1）%Maxsize
      </code>
     </li>
     <li>
      队列长度：
      <code>
       （Q.rear-Q.front+Maxsize）%Maxsize
      </code>
     </li>
     <li>
      队满：
      <code>
       （Q.rear+1）%Maxsize == Q.front
      </code>
     </li>
     <li>
      队空：
      <code>
       Q.front == Q.rear
      </code>
     </li>
    </ul>
    <h3>
     <a id="5_75">
     </a>
     5.二叉树
    </h3>
    <p>
     1.性质
    </p>
    <ul>
     <li>
      在二叉树的k层上，最多有
      <code>
       2^(k-1)
      </code>
      个结点。
     </li>
     <li>
      在深度为m的二叉树最多有2^m-1个结点。
     </li>
     <li>
      在任意一颗二叉树中，度为0的结点总是比度为2的结点多一个。
      <code>
       n0=n2+1
      </code>
      。
      <br/>
      还有结点数为n，其深度不小于（log2底n）+1。
     </li>
    </ul>
    <p>
     2.前序遍历：根-左-右
     <br/>
     中序遍历：左-根-右
     <br/>
     后序遍历：左-右-根
     <br/>
     层次遍历：按层来，由左向右，一层层来。
     <br/>
