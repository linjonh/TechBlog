      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        (ICE)
       </font>
      </span>

       仍是进行嵌入式应用系统调试最有效的开发工具。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        ICE
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       首先可以通过实际执行，对应用程序进行原理性检验，排除以人的思维难以发现的设计逻辑错误。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        ICE
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       的另一个主要功能是在应用系统中仿真微控制器的实时执行，发现和排除由于硬件干扰等引起的异常执行行为。此外，高级的
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        ICE
       </font>
      </span>

      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        ICE
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       的跟踪功能对系统进行记录和分析是一个快而有效的方法。
      </span>
     </font>
     <font size="3">
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        <br/>
        <br/>
       </font>
      </span>

      </span>

     <font size="3">
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">

       </font>
      </span>

      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        C
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">

      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        C
       </font>
      </span>

       <font face="Times New Roman">

       </font>
      </span>

       <font face="Times New Roman">
        C
       </font>
      </span>

      </span>

        C++

      </span>

        EC++

      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">

     </span>
     <font size="3">

       </font>
      </span>

      <br/>
     </span>
     <font size="3">
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">

      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">

      </span>

       </font>
      </span>

       很可能将成为未来的主流工具。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
       </font>
      </span>
     </font>
    </p>

     <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">

      <font face="Times New Roman" size="3">
       3.
      </font>
     </span>
     <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
      <font size="3">

      </font>
     </span>
     <font size="3">
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">

        <br/>
        <br/>
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">

      </span>

      </span>

       </font>

       电气信号。不同档次和功能模拟器工具价格差距巨大。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
       </font>
      </span>
     </font>

     </span>
     <font size="3">

       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       一样，仿真嵌入式系统在应用系统中的实际执行情况。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        (
       </font>
      </span>

      </span>

      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
       </font>
      </span>
     </font>

    <p class="MsoNormal" style="MARGIN: 0cm 0cm 0pt">
     <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
      <br/>
      <font face="Times New Roman" size="3">
       4.
      </font>
     </span>
     <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
      <font size="3">
       实时多任务操作系统
      </font>
     </span>
     <font size="3">
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        <br/>
        (Real Time multi-tasking Operation System, RTOS)
        <br/>
        <br/>
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       实时多任务操作系统
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        (RTOS)
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       是嵌入式应用软件的基础和开发平台。目前在中国大多数嵌入式软件开发还是基于处理器直接编写，没有采用商品化的
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        RTOS
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       ，不能将系统软件和应用软件分开处理。
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        RTOS
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       是一段嵌入在目标代码中的软件，用户的其它应用程序都建立在
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        RTOS
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       之上。不但如此，
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        RTOS
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       还是一个可靠性和可信性很高的实时内核，将
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        CPU
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">

      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        I/O
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       、定时器等资源都包装起来，留给用户一个标准的
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        API
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       ，并根据各个任务的优先级，合理地在不同任务之间分配
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        CPU
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       时间。
      </span>
     </font>
     <font size="3">
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        <br/>
        <br/>
        RTOS
       </font>
      </span>
      <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-bidi-font-size: 10.5pt">
       是针对不同处理器优化设计的高效率实时多任务内核，优秀商品化的
      </span>
      <span lang="EN-US" style="mso-bidi-font-size: 10.5pt">
       <font face="Times New Roman">
        RTOS
