    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE()宏一般是用在mfc中的，用于将调试信息输出到vs的输出窗口中(这是关键), 这在使用vs作为开发工具的时候，是非常方便的。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      <wbr/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      然而在开发一般c++程序时，却貌似无法获得这样的便利，其实只要经过几个步骤同样可以实现：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      一：#include&lt;atltrace.h&gt;
      <wbr/>
      <wbr/>
      <wbr/>
      <wbr/>
      <wbr/>
      <wbr/>
      <wbr/>
      //此头文件包含ATLTRACE宏，而其实MFC做的就是#define TRACE ATLTRACE
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      二：使用ATLTRACE("error");
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      三：包含atl链接库：
      <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:10px">
       <span style="font-size:10px; word-wrap:normal; word-break:normal">
        <span lang="EN-US" style="word-wrap:normal; word-break:normal">
         <span style="word-wrap:normal; word-break:normal; font-family:'Times new roman'">
          atlsd.lib
         </span>
        </span>
        <span style="word-wrap:normal; word-break:normal; font-family:宋体">
         （
        </span>
        <span lang="EN-US" style="word-wrap:normal; word-break:normal">
         <span style="word-wrap:normal; word-break:normal; font-family:'Times new roman'">
          debug
         </span>
        </span>
        <span style="word-wrap:normal; word-break:normal; font-family:宋体">
         ）或者
        </span>
        <span lang="EN-US" style="word-wrap:normal; word-break:normal">
         <span style="word-wrap:normal; word-break:normal; font-family:'Times new roman'">
          atls.lib
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      <wbr/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="font-weight:bold">
      <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:10px">
       <span style="font-size:10px; word-wrap:normal; word-break:normal">
        <span style="word-wrap:normal; word-break:normal; font-family:宋体; background-color:rgb(255,255,255)">
         trace详细教程：
        </span>
       </span>
      </span>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      <wbr/>
      <wbr/>
      TRACE宏对于VC下程序调试来说是很有用的东西，有着类似printf的功能；该宏仅仅在程序的DEBUG版本中出现，当RELEASE的时候该宏就完全消息了，从而帮助你调式也在RELEASE的时候减少代码量。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      使用非常简单，格式如下：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE("DDDDDDDDDDD");
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE("wewe%d",333);
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      同样还存在TRACE0，TRACE1，TRACE2。。。分别对应0，1，2。。个参数
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE信息输出到VC IDE环境的输出窗口（该窗口是你编译项目出错提示的哪个窗口），但仅限于你在VC中运行你的DEBUG版本的程序。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE信息还可以使用DEBUGVIEW来捕获到。这种情况下，你不能在VC的IDE环境中运行你的程序，而将BUILD好的DEBUG版本的程序单独运行，这个时候可以在DEBUGVIEW的窗口看到DEBUGVIE格式的输出了。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      VC中TRACE的用法有以下四种：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      1:
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE
      <wbr/>
      <wbr/>
      ，就是不带动态参数输出字符串,
      <wbr/>
      <wbr/>
      类似C的printf("输出字符串");
      <br/>
      <wbr/>
      <wbr/>
      <wbr/>
      <br/>
      2:
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE
      <wbr/>
      <wbr/>
      中的字符串可以带一个参数输出
      <wbr/>
      <wbr/>
      , 类似C的printf("...%d",变量);
      <br/>
      <br/>
      3:
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE
      <wbr/>
      <wbr/>
      可以带两个参数输出，类似C的printf("...%d...%f",变量1,变量2);
      <br/>
      <br/>
      4:
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE 可以带三个参数输出，类似C的printf("...%d，%d,%d",变量1,变量2，变量3);
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      TRACE 宏有点象我们以前在C语言中用的Printf函数，使程序在运行过程中输出一些调试信息，使我们能了解程序的一些状态。但有一点不同的是：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      <br/>
      TRACE 宏只有在调试状态下才有所输出，而以前用的Printf 函数在任何情况下都有输出。和Printf 函数一样，TRACE函数可以接受多个参数如：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      int x = 1;
      <br/>
      int y = 16;
      <br/>
      float z = 32.0;
      <br/>
      TRACE( "This is a TRACE statement/n" );
      <br/>
      TRACE( "The value of x is %d/n", x );
      <br/>
      TRACE( "x = %d and y = %d/n", x, y );
      <br/>
      TRACE( "x = %d and y = %x and z = %f/n", x, y, z );
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:5px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:21px; font-family:simsun; font-size:14px">
     <span style="background-color:rgb(255,255,255)">
      要注意的是TRACE宏只对Debug 版本的工程产生作用，在Release 版本的工程中，TRACE宏将被忽略。
     </span>
    </p>
