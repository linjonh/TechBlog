    <h3>
     <a id="visual_studio2019mdfidf_0">
     </a>
     在visual studio2019中连接数据库（mdf文件，idf文件）
    </h3>
    <p>
     在搜索框中输入
     <code>
      资源管理器
     </code>
     ，然后点击连接到数据库，选择你要连接的数据库类型，然后点击下一步，之后选择你的mdf文件。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/086e98905cdf91113329891d4c48f9dc.png#pic_center"/>
    </p>
    <p>
     之后，如果连接成功，会在服务器资源管理器出现数据连接，在这里会显示你的表
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/56ae5bde1d857dbbb6488cb59252ffa9.png#pic_center"/>
    </p>
    <p>
     在asp.net 连接数据库，使用
     <code>
      SqlDataSource
     </code>
     组件
    </p>
    <pre><code class="prism language-asp">&lt;asp:SqlDataSource runat="server" ID="bookCategory" ConnectionString="&lt;%$ ConnectionStrings:bookInfoConnnect %&gt;" SelectCommand="select * from Category;"&gt;
                
&lt;/asp:SqlDataSource&gt;
</code></pre>
    <p>
     这里的
     <code>
      ConnectionStrings
     </code>
     需要自己在
     <code>
      Web.config
     </code>
     中配置，全部写在
     <code>
      &lt;configuration&gt;
     </code>
     中，
     <code>
      bookInfoConnnect
     </code>
     为自己定义的名字，
     <code>
      SelectCommand
     </code>
     为自己设置的SQL查询语句，他会将查询的数据返回
    </p>
    <pre><code class="prism language-xml"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>configuration</span><span class="token punctuation">&gt;</span></span>
	<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>connectionStrings</span><span class="token punctuation">&gt;</span></span>
		<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>add</span> <span class="token attr-name">name</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>bookInfoConnnect<span class="token punctuation">"</span></span> <span class="token attr-name">connectionString</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>Data Source=(LocalDB)\MSSQLLocalDB;AttachDbFilename=C:\Users\86176\OneDrive\桌面\asp作业\学习通作业\WebApplication1\BookStore.mdf;Integrated Security=True;Connect Timeout=30<span class="token punctuation">"</span></span><span class="token punctuation">/&gt;</span></span>
	<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>connectionStrings</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>configuration</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     其中，
     <code>
      connectionString
     </code>
     的参数为：首先，打开自己的服务器资源管理器，找到自己的数据库表，右键它点击属性
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f219aae414131a2eedf77d18ccba153c.png#pic_center"/>
    </p>
    <p>
     数据的显示
    </p>
    <pre><code class="prism language-asp">&lt;asp:GridView runat="server" ID="showBookCategory" DataSourceID="bookCategory" AllowPaging="false" AllowSorting="true" AutoGenerateColumns="false"&gt;
    &lt;Columns&gt;
        &lt;asp:BoundField DataField="id" HeaderText="id" SortExpression="id" /&gt;
        &lt;asp:BoundField DataField="Name" HeaderText="Name" SortExpression="Name" /&gt;
    &lt;/Columns&gt;
&lt;/asp:GridView&gt;
</code></pre>
    <p>
     <code>
      DataSourceID
     </code>
     绑定数据源，该列表会显示如下内容，点击
     <code>
      id
     </code>
     ，
     <code>
      name
     </code>
     会根据内容进行排序，此外
     <code>
      AllowPaging
     </code>
     还可以设置分页，请将
     <code>
      AutoGenerateColumns
     </code>
     设置为
     <code>
      false
     </code>
     ，否则将会自动生成表格内容，这样你在自己设置时，将会显示出两个表格内容
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e191a0c85f07c3e9e0f08a6046a8409b.png#pic_center"/>
    </p>
    <p>
     跳转链接
    </p>
    <pre><code class="prism language-asp">&lt;asp:HyperLink runat="server" NavigateUrl="https://www.baidu.com"&gt;跳转&lt;/asp:HyperLink&gt;
</code></pre>
    <h5>
     <a id="GridView_56">
     </a>
     关于GridView的用法
    </h5>
    <h6>
     <a id="GridViewColumnsHyperLinkField_60">
     </a>
     GridView&gt;Columns&gt;HyperLinkField
    </h6>
    <h6>
     <a id="HyperLinkField_62">
     </a>
     对于其下属的HyperLinkField属性
    </h6>
    <p>
     使用方法
    </p>
    <p>
     在
     <code>
      Columns
     </code>
     内部
    </p>
    <pre><code class="prism language-asp">&lt;Columns&gt;
	&lt;asp:HyperLinkField HeaderText="dd" DataTextField="id" DataNavigateUrlFields="id"/&gt;
    &lt;asp:HyperLinkField HeaderText="Name" DataTextField="Name" DataNavigateUrlFields="Name" /&gt;
&lt;/Columns&gt;
</code></pre>
    <p>
     参数详解：
    </p>
    <ul>
     <li>
      HeaderText：列表的头部名称
     </li>
     <li>
      DataTextField：与数据库所绑定的字段
     </li>
     <li>
      DataNavigateUrlFields：将字段内容绑定到链接，之后可以点击链接进行跳转
     </li>
     <li>
      DataNavigateUrlFormatString：对于链接的设置
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4bd2f20fb776b1f0b2566515dacb07d8.png#pic_center"/>
    </p>
    <p>
     处理
     <code>
      HyperLinkField
     </code>
     给出的
     <code>
      DataNavigateUrlFormatString
     </code>
     链接设置，读取后面传递的参数
    </p>
    <pre><code class="prism language-c#">protected void Page_Load(object sender, EventArgs e)
{
	if(!IsPostBack)
    {
    	String name = Request.QueryString["Name"];
        // 例如你将DataNavigateUrlFormatString设置为Details.aspx?Name={0}，中括号内的内容将会被HyperLinkField中的Name所替换掉
    }
}
</code></pre>
