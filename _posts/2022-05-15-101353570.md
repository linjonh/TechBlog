    <h2>
     <a id="_0">
     </a>
     智能运维
    </h2>
    <p>
     AIOps(Artificial Intelligence for IT Operations) 即智能运维，其目标是给予已有的运维数据（日志，监控信息，应用信息等），通过机器学习的方式来进一步解决自动化运维所未能解决的问题，提高系统的预判能力及稳定性。
    </p>
    <p>
     KPI：Key Performance Indicators
     <br/>
     SRT：search response time
     <br/>
     HSRT: High SRT (搜索时间大于一秒的搜索)
    </p>
    <h3>
     <a id="_10">
     </a>
     已定义的智能运维科研问题
    </h3>
    <ul>
     <li>
      <p>
       针对历史事件
      </p>
      <ul>
       <li>
        瓶颈分析
       </li>
       <li>
        热点分析
       </li>
       <li>
        KPI聚类
       </li>
       <li>
        KPI关联关系挖掘
       </li>
       <li>
        异常事件关联关系挖掘
       </li>
       <li>
        全链路模块调用链分析
       </li>
       <li>
        故障传播关系图构建
       </li>
      </ul>
     </li>
     <li>
      <p>
       针对当前事件
      </p>
      <ul>
       <li>
        异常检测
       </li>
       <li>
        异常定位
       </li>
       <li>
        异常报警聚合
       </li>
       <li>
        快速止损
       </li>
       <li>
        故障根因分析
       </li>
      </ul>
     </li>
     <li>
      <p>
       针对未来事件
      </p>
      <ul>
       <li>
        故障预测
       </li>
       <li>
        容量预测
       </li>
       <li>
        趋势预测
       </li>
       <li>
        热点分析
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <a id="_34">
     </a>
     案例
    </h3>
    <h4>
     <a id="_36">
     </a>
     故障预测
    </h4>
    <ul>
     <li>
      <p>
       问题描述
      </p>
      <ul>
       <li>
        对于发生轻微异常的组件，需要预测在未来一段时间内是否会发生故障。
       </li>
      </ul>
     </li>
     <li>
      <p>
       输入
      </p>
      <ul>
       <li>
        KPI数据
       </li>
       <li>
        告警
       </li>
      </ul>
     </li>
     <li>
      <p>
       输出
      </p>
      <ul>
       <li>
        是否会发生故障
        <ul>
         <li>
          1：会发生故障
         </li>
         <li>
          0：不会发生故障
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     转换为机器学习问题（监督学习）
    </p>
    <ul>
     <li>
      <p>
       问题定义
      </p>
      <ul>
       <li>
        二分类问题
       </li>
      </ul>
     </li>
     <li>
      <p>
       关键步骤
      </p>
      <ul>
       <li>
        样本构造（历史样本集）
       </li>
       <li>
        特征构造
       </li>
       <li>
        分类算法（随机森林）
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="_60">
     </a>
     故障定界（异常定位）
    </h4>
    <ul>
     <li>
      <p>
       问题描述
      </p>
      <ul>
       <li>
        需要判断是哪一个设备故障导致的异常
       </li>
      </ul>
     </li>
     <li>
      <p>
       输入
      </p>
      <ul>
       <li>
        KPI数据
       </li>
       <li>
        网络拓扑
       </li>
      </ul>
     </li>
     <li>
      <p>
       输出
      </p>
      <ul>
       <li>
        故障位置
       </li>
      </ul>
     </li>
    </ul>
    <p>
     转换为机器学习问题（非监督学习）
    </p>
    <ul>
     <li>
      <p>
       问题定义
      </p>
      <ul>
       <li>
        聚类问题
       </li>
      </ul>
     </li>
     <li>
      <p>
       关键步骤
      </p>
      <ul>
       <li>
        样本构造（历史样本集）
       </li>
       <li>
        特征构造
       </li>
       <li>
        聚类算法（层次聚类）
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="_82">
     </a>
     异常事件关联关系挖掘（告警关联）
    </h4>
    <ul>
     <li>
      <p>
       问题描述
      </p>
      <ul>
       <li>
        对网络每天产生的大量告警进行关联，把属于同一个故障的告警压缩为一组
       </li>
      </ul>
     </li>
     <li>
      <p>
       输入
      </p>
      <ul>
       <li>
        告警
       </li>
       <li>
        网络拓扑
       </li>
      </ul>
     </li>
     <li>
      <p>
       输出
      </p>
      <ul>
       <li>
        告警组
       </li>
      </ul>
     </li>
    </ul>
    <p>
     转换为数据挖掘问题
    </p>
    <ul>
     <li>
      <p>
       问题定义
      </p>
      <ul>
       <li>
        序列频繁项集挖掘问题
       </li>
      </ul>
     </li>
     <li>
      <p>
       关键步骤
      </p>
      <ul>
       <li>
        <p>
         告警预处理
        </p>
        <ul>
         <li>
          清除重复告警
         </li>
         <li>
          清除闪断告警
         </li>
        </ul>
       </li>
       <li>
        <p>
         告警空间关联
        </p>
       </li>
       <li>
        <p>
         告警时间分组
        </p>
       </li>
       <li>
        <p>
         频繁项集挖掘
        </p>
       </li>
       <li>
        <p>
         挖掘出频繁项集后，有专家人工确认有关联的故障规则
        </p>
       </li>
      </ul>
     </li>
    </ul>
