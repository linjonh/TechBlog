    <h1 class="title-article" id="articleContentId">
     智能硬件开发怎么做？机智云全套自助式开发工具助力高效开发
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       智能硬件开发是一个交叉学科，通常在入门的时候，是以
       <a class="relatedlink" href="http://club.gizwits.com/thread-5326-1-1.html" rel="nofollow" title="STM32">
        STM32
       </a>
       为主，搭建云平台或接入云平台，以此实现智能产品开发和设计。
      </span>
     </span>
    </div>
    <div>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       如果是自己搭建一个私有云平台，通常需要了解云技术框架及技术、通信服务搭建和设计、mysql服务对接、项目通信实验、web前后端搭建、项目验收等项目，无论是对于嵌入式开发者还是企业来说，都需要花费大量时间。往往我们会选择一个现成的
       <a class="relatedlink" href="http://www.gizwits.com/" rel="nofollow" title="智能硬件云服务">
        智能硬件云服务
       </a>
       平台，以此提高效率。
      </span>
     </span>
    </div>
    <div>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       <strong>
        以智能硬件接入机智云平台为例
       </strong>
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       1、先准备一个开发套件或联网模块
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       可以是
       <a class="relatedlink" href="http://club.gizwits.com/forum-173-1.html" rel="nofollow" title="esp8266">
        esp8266
       </a>
       、
       <a class="relatedlink" href="http://docs.gizwits.com/zh-cn/developer/resource/Gokit.html" rel="nofollow" title="gokit">
        gokit
       </a>
       、STM32、
       <a class="relatedlink" href="http://club.gizwits.com/thread-6332-1-1.html" rel="nofollow" title="arduino">
        arduino
       </a>
       、GE211、正点原子开发套件、模组等。
      </span>
     </span>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       这里我们用可同时支持WiFi和低功耗蓝牙模组GE211为例。
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       “GE211是一款内置ESP32-C3-WROOM-02（简称ESP32-C3）的转接板，是一款专为物联网应用场景设计的通用型WiFi和低功耗蓝牙模组，安全性高、内存充足、外设接口多、应用场景丰富、尺寸小，可用于智能家居、工业自动化、医疗、消费电子产品等领域。”
      </span>
     </span>
    </div>
    <p>
     <br/>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       2、熟悉机智云智能硬件云平台
      </span>
     </span>
    </div>
    <div>
     <p class="img-center">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e24920f880371816cbc121bb0fabd03d.jpeg"/>
     </p>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       <strong>
        机智云
        <a class="relatedlink" href="http://www.gizwits.com/" rel="nofollow" title="物联网平台">
         物联网平台
        </a>
        接入
       </strong>
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       关于APP开发
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       机智云提供了APPSDK、开源框架、微信SDK，便于开发者快速构建手机应用端系统，只需根据工具指引实现业务功能，无需实现底层功能。
      </span>
     </span>
    </div>
    <div>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       APPSDK开发：即用APP代码自动生成工具生成的appsdk包。机智云封装了配网、云平台数据调用等功能，保证数据通信的准确性和私密性，这使开发者仅需关注APP的UI和UE设计即可，而相对复杂的协议与错误处理等事项可忽略。这种需要有一点Java、Swift语言基础。
      </span>
     </span>
    </div>
    <div>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       在线配置式开发：即赛博坦工具开发，开发者只需要在线进行选择相应的控制方式、控制大小，替换页面主题方式、主题背景等信息，来定义一款自己喜欢的APP。
      </span>
     </span>
    </div>
    <p>
     <br/>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       下图所示，就是通过“在线配置”的方式，在机智云物联网平台做的一款APP
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
     <p class="img-center">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5dca75963c324aed63fe576ce7d1d5ec.jpeg"/>
     </p>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       <strong>
        关于设备端开发
       </strong>
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       为了让大家快速接入机智云，机智云文档中心提供了基于开源套件Gokit编写的MCU和SOC两套方案的快速接入教程。
      </span>
     </span>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       值得一提的是需要了解
       <a class="relatedlink" href="http://docs.gizwits.com/zh-cn/deviceDev/gagent_info.html" rel="nofollow" title="GAgent">
        GAgent
       </a>
       是什么？它是机智云为了让不同通信模组接入机智云而推出的一个固件，主要的作用是数据转发，是设备数据、机智云、应用端（APP）的数据交互桥梁。换言之，只要在机智云指定的模块中烧写Gagent，就能让模组实现云服务轻松联网，给开发者减压提效。
      </span>
     </span>
    </div>
    <p>
    </p>
    <div>
     <p class="img-center">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/885dd7fe66433764a7dbd585c9d41ae5.jpeg"/>
     </p>
    </div>
    <p>
    </p>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       <strong>
        关于机智云物联网云端介绍
       </strong>
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       3、进行实操
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       实操部分由于截图太多，可以参考已有案例《动手做一个4孔插座，顺便搞懂智能产品开发》案例，此案例比较通俗易懂，通过公开设备端最小系统图、记录云端开发设计流程，实现了无计量4孔插座的开发全过程。实现功能如下：
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       1.控制4路插孔，可单控和全控。
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       2.定时任务创建
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       3.控制5v2.1A手机充电孔开关
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       4.硬件接线开源已经参考设计
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       5.掉电记忆开关状态
      </span>
     </span>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       6.远程升级
      </span>
     </span>
    </div>
    <div>
    </div>
    <div>
     <p class="img-center">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6d0be9448a6421e898bc9d293961e0e9.jpeg"/>
     </p>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#444444;">
      <span style="background-color:#ffffff;">
       通过借助机智云自助开发平台，对于我们个人开发者来说做一款智能硬件产品，非常方便。尤其是机智云物联网平台的“免费工具”很多，很实用。尤其是对于我不擅长APP开发的个人来说，用机智云做一款智能插座、改造老式电风扇并不难。
      </span>
     </span>
    </div>
