     什么是MCSE(微软认证系统工程师）？
    </h1>
   </div>
  </div>

    <p>
     微软公司提供的对于网络专业人员的微软系统工程师认证。MCSE是指被证明能够在用Microsoft Windows NT Server 和Microsoft Backoffice服务器产品家庭构建的广泛的计算环境中进行有效地规划、实现、维护和支持信息系统的合格人员。
     <br/>
     <br/>
     微软认证系统工程师（MCSE+Internet）
     <br/>
     <br/>
     是指技术资格的专业人员，能够部署、管理包括浏览器、代理服务器、主机服务器、数据库、消息系统及电子商务等部件的Internet/Intranet解决方案。管理和分析Web站点。
     <br/>
     <br/>
     微软认证方案开发人员（MCSD）
     <br/>
     <br/>
     微软公司提供的对于开发人员的微软方案开发人员认证。MCSD是指被证明能够用微软的开发工具、技术和平台，包括Microsoft Office和Microsoft BackOffice，设计和开发提供定制业务方案的合格人员。
     <br/>
     <br/>
     微软认证产品专家（MCP）
     <br/>
     <br/>
     由微软公司提供的对于精通一种微软技术产品的人员的认证。MCP是被证明至少对一种微软操作系统有深入了解的人。此人可能需要通过附加的微软认证考试，以证明他在Microsoft Backoffice产品、开发工具或桌面应用程序方面有更多的技能。
     <br/>
     <br/>
     微软认证Internet专家（MCP+Internet）
     <br/>
     <br/>
     能胜任服务器系统的安装、配置和管理，通过及ISAP脚本语言充分发挥服务器的扩展功能。
     <br/>
     <br/>
     微软认证Site Building专家（MCP+Site Building）
     <br/>
     <br/>
     在建立Web站点方面有专业能力。
     <br/>
     <br/>
     微软认证数据库管理员（MCDBA）
     <br/>
     <br/>
     微软认证题库资料，请参考成为MCSE需要掌握
     <br/>
     1、 Windows 2003网络和操作系统基础；
     <br/>
     <br/>
     2、 Windows2003Sever；
     <br/>
     <br/>
     3、 Windows 2003网络基础结构；
     <br/>
     <br/>
     4、 Windows 2003活动目录服务基础结构设计与管理；
     <br/>
     <br/>
     5、 Windows 2003目录服务基础结构设计；
     <br/>
     <br/>
     6、 SQL2005数据库；
     <br/>
     <br/>
     7、 邮件服务器ExChange；
     <br/>
     <br/>
     MCSE 2003常规科目及常用科目组合　　Exam 70–290: Managing and Maintaining a Microsoft Windows Server 2003 Environment
     <br/>
     <br/>
     Exam 70–291:Implementing, Managing, and Maintaining a Microsoft Windows Server 2003 Network Infrastructure
     <br/>
     <br/>
     Exam 70–293:Planning and Maintaining a Microsoft Windows Server 2003 Network Infrastructure
     <br/>
     <br/>
     Exam 70–294:Planning, Implementing, and Maintaining a Microsoft Windows Server 2003 Active Directory Infrastructure
     <br/>
     <br/>
     Exam 70–298:Designing Security for a Microsoft Windows Server 2003 Network
     <br/>
     <br/>
     Exam 70–270:Installing, Configuring, and Administering Microsoft Windows XP Professional
     <br/>
     <br/>
     Exam 70–228:Installing, Configuring, and Administering Microsoft SQL Server 2000 Enterprise Edition

     <br/>
