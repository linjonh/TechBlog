    <h2>
     INAV 开源飞行控制软件教程
    </h2>
    <p>
     项目地址:
     <a href="https://gitcode.com/gh_mirrors/in/inav/?utm_source=artical_gitcode&amp;index=top&amp;type=href&amp;">
      https://gitcode.com/gh_mirrors/in/inav
     </a>
    </p>
    <h3>
     项目介绍
    </h3>
    <p>
     INAV 是一个开源的飞行控制软件，广泛应用于多旋翼飞行器（如四轴、六轴、八轴飞行器）和固定翼飞行器。INAV 提供了强大的配置工具 INAV Configurator，支持多种飞行器类型，并且不断更新以适应新的硬件和技术。
    </p>
    <h3>
     项目快速启动
    </h3>
    <h4>
     安装 INAV Configurator
    </h4>
    <h5>
     Windows 平台
    </h5>
    <ol>
     <li>
      访问
      <a href="https://github.com/iNavFlight/inav-configurator/releases">
       INAV Configurator 发布页面
      </a>
      。
     </li>
     <li>
      下载适用于 Windows 平台的 Configurator（ia32）。
     </li>
     <li>
      解压下载的文件并运行 INAV Configurator。
     </li>
    </ol>
    <h5>
     Mac 平台
    </h5>
    <ol>
     <li>
      访问
      <a href="https://github.com/iNavFlight/inav-configurator/releases">
       INAV Configurator 发布页面
      </a>
      。
     </li>
     <li>
      下载适用于 Mac 平台的 Configurator。
     </li>
     <li>
      解压 ZIP 文件并运行 INAV Configurator，或者使用 DMG 包进行安装。
     </li>
    </ol>
    <h4>
     配置 INAV 软件
    </h4>
    <ol>
     <li>
      连接你的飞行控制器到电脑。
     </li>
     <li>
      打开 INAV Configurator。
     </li>
     <li>
      选择你的飞行控制器型号并进行配置。
     </li>
    </ol>

# 打开 INAV Configurator 并选择相应的端口
</code></pre>
    <h3>
     应用案例和最佳实践
    </h3>
    <h4>
     多旋翼飞行器配置
    </h4>
    <ol>
     <li>
      <strong>
       选择合适的飞行模式
      </strong>
      ：根据飞行需求选择如定高、定位、手动等模式。
     </li>
     <li>
      <strong>
       调整 PID 参数
      </strong>
      ：根据飞行器的响应调整 PID 参数以获得最佳飞行性能。
     </li>
     <li>
      <strong>
       校准传感器
      </strong>
      ：确保加速度计和陀螺仪校准正确，以提高飞行稳定性。
     </li>
    </ol>
    <h4>
     固定翼飞行器配置
    </h4>
    <ol>
     <li>
      <strong>
       调整飞行控制参数
      </strong>
      ：根据固定翼的特性调整飞行控制参数，如副翼、升降舵和方向舵的响应。
     </li>
     <li>
      <strong>
       设置导航模式
      </strong>
      ：使用 GPS 导航模式进行自动起飞和降落。
     </li>
     <li>
      <strong>
       优化电池管理
      </strong>
      ：设置电池警告和自动降落功能，确保飞行安全。
     </li>
    </ol>
    <h3>
     典型生态项目
    </h3>
    <h4>
     INAV Blackbox Tools
    </h4>
    <p>
     INAV Blackbox Tools 提供命令行工具（如
     <code>
      blackbox_decode
     </code>
     和
     <code>
      blackbox_render
     </code>
     ）用于黑匣子日志的转换和分析，帮助开发者调试和优化飞行性能。
    </p>
    <h4>
     OpenTX Telemetry Widget
    </h4>
    <p>
     OpenTX Telemetry Widget 允许使用 EdgeTX 和 OpenTX 无线电（如 Taranis、Horus、Jumper、Radiomaster、Nirvana）的用户显示 INAV 的遥测数据，提供实时飞行信息。
    </p>
    <h4>
     INAV OSD Switcher Tool
    </h4>
    <p>
     INAV OSD Switcher Tool 是一个方便的工具，允许用户在 INAV 中轻松切换 OSD 布局，支持复制、移动或替换布局文件。
    </p>
    <p>
     通过以上模块的介绍和实践，用户可以快速上手并充分利用 INAV 开源飞行控制软件的功能。
    </p>
