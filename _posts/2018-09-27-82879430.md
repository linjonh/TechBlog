     <h3>
     </h3>
     <p style="text-align:justify;">
      <strong>
       <span style="font-size:15px;color:rgb(136,136,136);">
        2018年中的区块链行业呈现出与年初不同的形态。
       </span>
      </strong>
     </p>
     <p>
      <br/>
     </p>
     <p style="text-align:justify;">
      <strong>
       <span style="font-size:15px;color:rgb(136,136,136);">
        一方面ICO渐冷，传统VC转淡，整个融资环境趋于严峻；另一方面各国纷纷布局区块链，从政策上大力发展区块链技术。
       </span>
      </strong>
     </p>
     <p>
      <br/>
     </p>
     <p style="text-align:justify;">
      <strong>
       <span style="font-size:15px;color:rgb(136,136,136);">
        整个区块链行业的变动，使得区块链人才招聘市场也呈现出不同的变化，最为明显的是，整个行业的职位增速逐渐放缓。
       </span>
      </strong>
     </p>
     <p>
      <br/>
     </p>
     <p style="text-align:justify;">
      <strong>
       <span style="font-size:15px;color:rgb(136,136,136);">
        一季度的人才泡沫逐渐褪去，区块链人才供需比逐渐趋于理性。在第二季度，区块链技术人员的平均月薪为3万元，相比第一季度上涨5.3%，涨幅趋于稳定，
        <strong style="color:rgb(51,51,51);font-family:'微软雅黑';font-size:16px;letter-spacing:.544px;text-align:justify;background-color:rgb(244,244,244);">
         <span style="font-size:15px;color:rgb(136,136,136);">
          行业呈现新的招聘现状。
         </span>
        </strong>
       </span>
      </strong>
     </p>
     <p>
      <br/>
     </p>
     <p style="text-align:justify;">
      <strong>
       <span style="font-size:15px;color:rgb(136,136,136);">
        基于此，本份报告从拉勾网覆盖的1954余万互联网从业者，36万+互联网公司大数据中筛选信息，分析区块链招聘市场现状，并预测区块链人才市场发展的趋势。为政府、企业、高校、人才等市场相关方提供参考，共同促进区块链人才市场和整体行业持续健康发展。
       </span>
      </strong>
     </p>
     <p style="text-align:justify;">
