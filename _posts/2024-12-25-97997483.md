     </p>
     <span style="color:rgb(89,89,89);">
      周内，朋友黄工分享了一篇我的文章，关于职场的几个建议.
     </span>
     <br/>
     <span style="color:rgb(89,89,89);">
      <br/>
     </span>
     <a href="https://mp.weixin.qq.com/s?__biz=MzI4MDI4MDE5Ng==&amp;mid=2247487638&amp;idx=1&amp;sn=0033be2bbecba476b8e8d829bdbcdc48&amp;scene=21#wechat_redirect" rel="nofollow">
      <span style="color:rgb(120,172,254);">
       给初入职场朋友的8条走心建议
      </span>
     </a>
     <span style="color:rgb(89,89,89);">
      <br/>
     </span>
     <br/>
     <span style="color:rgb(89,89,89);">
      因为我给错了二维码，所以很惊喜的是加了一些新的朋友。
     </span>
     <br/>
     <span style="color:rgb(89,89,89);">
      加入过来的朋友，问的最多的一个问题是：
     </span>
     <span style="color:rgb(89,89,89);">
      你也是做嵌入式开发吗？
     </span>
     <p>
      <br/>
     </p>
     <span style="color:rgb(89,89,89);">
      说是也不是，那么这篇文章简单说说这个嵌入式的故事，如果大家感兴趣，当下在学习嵌入式开发，可以关注黄工的公号:
     </span>
     <span style="color:rgb(89,89,89);">
      <br/>
     </span>
     <img alt="640?wx_fmt=jpeg" class="rich_pages" src="https://i-blog.csdnimg.cn/blog_migrate/0ef1da97ae72053af45a7605e8795210.jpeg" style="width:167px;">
      <p style="text-align:center;">
       <br/>
      </p>
      <strong>
       <span style="color:rgb(89,89,89);">
        一位从事嵌入式软件开发多年的攻城狮，内容聚焦在
       </span>
       <span style="color:rgb(89,89,89);">
        嵌入式、 底层软件。
       </span>
       <span style="color:rgb(89,89,89);">
       </span>
       <span style="color:rgb(89,89,89);">
        一个90后的帅小伙陪在你成长的道路，也是值得幸福的一件事。
       </span>
      </strong>
      <span style="color:rgb(89,89,89);">
      </span>
      <strong>
       <span style="color:rgb(89,89,89);">
        <br/>
       </span>
      </strong>
      <span style="color:rgb(89,89,89);">
      </span>
      <span style="color:rgb(89,89,89);">
       大学时期，我所选择的本专业是电子信息工程，大一时候学习了 C语言 和 数据结构 ，开始渐渐对计算机编程感兴趣。
      </span>
      <span style="color:rgb(89,89,89);">
       因为大二的时候，学习了 计算机组成原理 和 数字电路。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       主要是 数字电路 这个课程，自己产生了浓厚的兴趣，对于与或非，逻辑矩阵，以及状态机理解的都比较透彻，从这个时候，我便想学习编程了。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       并且我们开的课程有 FPGA ，VLOG编程。
      </span>
      <span style="color:rgb(89,89,89);">
       那时候便在想学什么，当时单片机比较流行，就学习了 C51编程。
      </span>
      <span style="color:rgb(89,89,89);">
       那时候对于逻辑地址，以及中断，ADC这些都不懂，于是就从图书馆借了很多这类书，开始一个个攻克。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       吃力的学习，慢慢消化，几度要想着放弃。
      </span>
      <span style="color:rgb(89,89,89);">
       而我想要坚持的理由很俗，就是认为这个方向能赚钱。
      </span>
      <span style="color:rgb(89,89,89);">
       没有太多其他的理由，因为其他的都很假，什么兴趣，都是随之而来的。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       于是就学习这个，用的模拟器调试，开发，做交通灯软件。
      </span>
      <span style="color:rgb(89,89,89);">
       慢慢的学习完这个C51，便开始进入ARM7的学习阶段，这个期间报了学校的课程，在这个里面接触到了操作系统，当时学习的是uscosii，使用的是周立功的开发板子。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       在这个里面学习了操作系统的很多知识，中断上下文，现场保护，FLAG，MSGBOX等一些IPC的方式。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       学习操作系统比较难，这个过程枯燥无味，但往往困难的，你突破了就算是胜利。
      </span>
      <span style="color:rgb(89,89,89);">
       于是斗志昂扬，开始几年泡在操作系统书籍里面，同时学习了操作系统编程(UNIX系统编程)。
      </span>
      <span style="color:rgb(89,89,89);">
       在这段时间里面，为了理解透彻汇编，将C语言反编译学习了一些，
      </span>
      <span style="color:rgb(89,89,89);">
       主要是 PE 文件破解，ARM 汇编指令优化。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       在不断的探索之后，算是把基础结结实实的打了下来。
      </span>
      <span style="color:rgb(89,89,89);">
       说实话，很多时候别人给你的快方法，实际对你没啥用处。
      </span>
      <span style="color:rgb(89,89,89);">
       因为你没有建立自学能力，知识体系，而这个过程都是需要长时间的训练，琢磨，在试错之后才能出成绩。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       在回过头去看之前的努力，一切都变得很值得。
      </span>
      <span style="color:rgb(89,89,89);">
       不要陷入一直在别人的总结，分享之下成长。
      </span>
      <span style="color:rgb(89,89,89);">
       这种只算是记忆的东西，没有迁移的能力。
      </span>
      <span style="color:rgb(89,89,89);">
       要抽丝剥茧，把一些细枝末节过滤掉，建立自己的知识体系，丰富自己的体系。
      </span>
      <span style="color:rgb(89,89,89);">
       抓住事物的本质，才不会被GUI迷惑，此后才是一片明亮。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       以上就跟大家唠唠我的嵌入式时代，毕业之后就在手机行业做开发，往后我们继续来聊。
      </span>
      <p>
       <br/>
      </p>
      <span style="color:rgb(89,89,89);">
       与其相濡以沫，不如相忘江湖。
      </span>
      <span style="color:rgb(89,89,89);">
       有酒有茶，摇曳星辰，就在明哥的江湖。
      </span>
      <span style="color:rgb(89,89,89);">
