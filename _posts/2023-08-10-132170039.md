    <p>
     <strong>
      【问题描述】
     </strong>
     <br/>
     当使用U8客户端电脑作为天联高级版软件服务器的时候，
     <br/>
     即：U8应用服务器和远程服务器不是同一台电脑。
     <br/>
     每次新建天高用户后，
     <br/>
     新的天高用户，登录天高客户端后并打开U8登录界面，
     <br/>
     用友U8的登录窗口服务器地址即为天高服务器的计算机名称，
     <br/>
     而不是用友U8应用服务器的计算机名称。
     <br/>
     这样就需要操作者进行修改，虽然该位置只需要修改一次，下次就会记忆。
     <br/>
     但也觉得这样很麻烦，毕竟作为使用者，有的时候是搞不明白这一点点的小操作。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c0c78100e47a1b37f1f3decc9399c1ec.png"/>
    </p>
    <p>
     <strong>
      【解决方法】
     </strong>
     <br/>
     在天高服务器上，打开注册表，
     <br/>
     按下述路径，找到Server参数，
     <br/>
     双击Server，在其【数值数据】中录入U8应用服务器的ip地址或者计算机名称。
     <br/>
     路径：HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Ufsoft\WF\V8.700\Login
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6360f2851b274651c2862bd4918d4b3c.png">
      <br/>
      后续新的用户，通过天高客户端打开U8登录界面，
      <br/>
      U8的服务器地址即为应用服务器地址。
      <br/>
      不再需要人工修改了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4d5369d360fcb4d19e20e6bcc1d03539.png"/>
     </img>
    </p>
