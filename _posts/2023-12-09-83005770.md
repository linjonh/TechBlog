      </span>
     </p>

      </span>

     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">

        <span style="font-size:15px;">
         第一，明确学习目标。
        </span>
       </strong>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">

       <strong>
        <span style="font-size:15px;">
         第二，系统性学习。
        </span>
       </strong>
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">
      <span style="font-size:15px;letter-spacing:1px;">
       明确了目标，还需要掌握学习方法，任何一门编程语言都应该先系统性学习，而不是碎片化地学习，毕竟一门语言包含太多细节，碎片化的学习，顾此失彼，浅尝辄止，无法从全局把握。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">
      <span style="font-size:15px;letter-spacing:1px;">
       当然，系统学习也要讲究方法，一头扎进书本里，动辄数百页的教材难免让人望而却步，很多时候效果并不好。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">
      <span style="font-size:15px;letter-spacing:1px;">
       因此，系统学习需要选择精炼的资料，篇幅短，知识点全面，并辅以生动的实例，事半而功倍。
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">
      <span style="letter-spacing:1px;">
       <strong>
        <span style="font-size:15px;">
         第三，阶段性成果。
        </span>
       </strong>
       <span style="font-size:15px;">
        既然是学习，必须有阶段性成果，比如，动手写一个抢票软件、爬虫、或小工具......
       </span>
      </span>
     </p>

      </span>

       那么，如果想要追赶 Python 的热潮，应该如何学习呢？
      </span>
     </p>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">

          </span>
         </strong>
        </span>
       </p>
      </li>
     </ul>
     <p style="margin-left:8px;min-height:1em;color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;background-color:rgb(255,255,255);">
      <span style="font-size:15px;letter-spacing:1px;">
       如果课程里面，只是对着屏幕把项目代码敲一遍，那其实看了也不能自己真正掌握，重要的是自己在课程作业里动手实践，所以课程中有没有循序渐进的练手项目，帮你提升能力，这是很重要的。
      </span>
     </p>

       <p style="margin-left:8px;min-height:1em;line-height:1.75em;">
        <span style="letter-spacing:1px;">
         <strong>
          <span style="font-size:15px;">

          </span>
         </strong>
        </span>
       </p>

      <span style="font-size:15px;letter-spacing:1px;">
       比如是否有学习到：Python基础；Python网络爬虫；数据挖掘与机器学习；Python Web开发，Python自动化运维。
      </span>
     </p>

          <span style="font-size:15px;">

          </span>

        </span>
