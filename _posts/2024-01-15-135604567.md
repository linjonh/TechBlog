    <p>
     目的：将python项目的开发环境迁移至离线网络中进行开发
     <br/>
     开发工具：pycharm
     <br/>
     环境：windows（有网络）----&gt; windows(离线）
     <br/>
     第一步：查看python版本项目所使用的python版本，在pycharm中点击下方的Terminal,输入命令
     <code>
      python
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f21936d2af0de3ddfdadc480f75aa64c.png">
      <br/>
      然后在Python 官方网站： https://www.python.org下载对应版本的python，例如本文是在windows环境中进行安装，因此下载
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3beb8e63a2e8d3aaea2cd0eaf3415c6b.png">
       <br/>
       将下载文件传输至离线环境中，双击，按照以下方式进行安装。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3cd81a1054ee15becb8fde103ea34812.png">
        <br/>
        打开cmd，输入命令
        <code>
         python
        </code>
        ，如果出现python版本信息则说明安装成功。
        <br/>
        第二步：在pycharm中，点击下方的Terminal,输入命令
        <code>
         pip freeze &gt; requirements.txt
        </code>
        ，提取第三方库目录到requirements.txt中，执行完毕在项目路径下生成requirements.txt文档。
       </img>
      </img>
     </img>
    </p>
    <p>
     第三步：将项目所需库，requirements.txt文档中的库下载至本地。使用命令
     <code>
      pip download -d Mypackages -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
     </code>
     其中
     <code>
      Mypackages
     </code>
     为下载库所存放的位置。
    </p>
    <p>
     第四步：将项目文件，
     <code>
      requirements.txt
     </code>
     、
     <code>
      Mypackages
     </code>
     传输到离线环境中，使用命令
     <code>
      pip install -r requirements.txt --no-index -f Mypackages
     </code>
     即可安装成功。
    </p>
