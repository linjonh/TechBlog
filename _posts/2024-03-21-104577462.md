      如果进入后是下图样式，是有问题的，需要在后台修改配置（如果正常的，跳过下面修改步骤）

      <br/>
      进入Zstack系统界面（vmware装的那个系统）或使用crt等工具登陆该服务器：vi /etcmodprobe.d/kvm-nested.conf，在最后面加上ept=0，保存退出
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d49b50cc622931839f4f1cbcde3e7d54.png"/>
      <br/>
      然后关闭控制台界面并停止云主机
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4e4bfb1ae66f5ce94597ea17e93019c3.png"/>
      <br/>
      再次回到系统后台或crt工具登陆后（vmware装的那个系统）输入：rmmod kvm-intel ; modprobe kvm-intel ept=0 unrestricted_guest=0
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8faa5cbe83ae31521156ad8667471a94.png"/>
      <br/>
      然后再启动云主机

     </li>
