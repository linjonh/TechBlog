    <p style="margin-left:0px;">
     <strong>
      随着人工智能
      <strong>
      </strong>
      大数据的火热
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      成为了广大科学家和普通大众的学习语言。在学习
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的过程中，我感到迷茫不知道自己该从什么地方入手，我在学习
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      之前了解过这门语言，也知道
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      有很多的学习方向，比如说数据采集方向
     </strong>
     <strong>
      (
     </strong>
     <strong>
      爬虫
     </strong>
     <strong>
      )
     </strong>
     <strong>
      ，或者
     </strong>
     <strong>
      Web
     </strong>
     <strong>
      开发方向，也可能是最近特别火热的人工智能方向。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      就像学习其他编程语言或者是学习一门外语一样
     </strong>
     <strong>
     </strong>
     <strong>
      ，我从
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的基础语法开始学习
     </strong>
     <strong>
     </strong>
     <strong>
      ，了解什么是
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的变量
     </strong>
     <strong>
     </strong>
     <strong>
      什么是循环
     </strong>
     <strong>
     </strong>
     <strong>
      什么是函数，什么是模块。类等等。总之，基础是学习以后高级开发的基石。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      学习任何一门语言，选择操作系统，搭建开发环境是必不可少的。学习
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      ，我是用
     </strong>
     <strong>
      windows
     </strong>
     <strong>
      系统，要自己下载并配置好
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      开发环境，老师推荐的是
     </strong>
     <strong>
      AnaConda Navigator
     </strong>
     <strong>
      。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      选择一门重视大脑学习的书籍，有一本好用的课外书：明日科技的《
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      编程入门指南》，这是我看过最注重大脑学习的书籍，对于我入门有很大帮助。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      仔细阅读有关
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的书籍，好记心不如烂不同，一定要做笔记，过后回顾笔记。书上给出的练习，一定要做，并且争取全部弄懂。书上有的练习代码，一定要自己调试一遍，知其所以然。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      每天的编码必不可少，既然选择学习编程，学习
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      ，坚持编码应该是必须做到的。没有代码积累，要写出高质量的代码，几乎不可能。在写过不少基础代码之后，可以去各大
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      相关的网站阅读别人的代码，多阅读别人的也是提高自己的编码水平的很好途径，同时，有把自己的代码分享给别人阅读，一边互相交流促进。例如
     </strong>
     <strong>
      CSDN
     </strong>
     <strong>
      ，牛客，洛谷等平台都是比较好用的。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      最重要的是明白学
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的目的是什么，是数据挖掘还是想刷个火车票啥的，是机器学习还是搞个小脚本装下，是想成为厉害的
     </strong>
     <strong>
      harker
     </strong>
     <strong>
      还是强大的
     </strong>
     <strong>
      web
     </strong>
     <strong>
      开发者。只有明白了目的，才有学下去的动力。
     </strong>
     <strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      下载并安装好
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      及
     </strong>
     <strong>
      IDE,
     </strong>
     <strong>
      你需要明白什么是
     </strong>
     <strong>
      IDE
     </strong>
     <strong>
      ，什么是命令行等，并配置好路径及环境变量。在这里推荐大家找下免费的
     </strong>
     <strong>
      PyCharm
     </strong>
     <strong>
      ，风格比较舒服，但是会比较慢。基础练习。没有编程基础的可以先过一下基础，可以找本书练，也可以在网上寻找视频练习。现在网络这么发达，应该加强在网上找答案的能力。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <strong>
      根据自己的目标去深耕。知识的学习都是由浅入深的，先掌握基础，再根据自己的目标去练习，才会有效果。不要三心二意，今天学十分钟爬虫，明天学
     </strong>
     <strong>
      5
     </strong>
     <strong>
      钟机器学习什么的。不断练习，不断练习。量变到质变的过程。只有在敲代码的过程中才会遇到自己的问题，解决了就提升了。
     </strong>
    </p>
    <p style="margin-left:0;text-align:justify;">
     Python是一门非常有潜力的高级语言，历经多年的发展，其在编程上发挥着越来越大的作用。而且，在字符串上的处理，
     <strong>
      Python
     </strong>
     <strong>
      相对于
     </strong>
     <strong>
      C
     </strong>
     <strong>
      语言，
     </strong>
     <strong>
      Java
     </strong>
     <strong>
      等也是给我的使用极大的便利。而且
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      不仅如此，它的库也很多，正因为它强大的库，让编程变得不再艰难。我对
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的学习也仅仅只是它的基础方面，但
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      的强大，却是足足地吸引着我，希望自己能够在不断地学习中，将
     </strong>
     <strong>
      Python
     </strong>
     <strong>
      学习的更加好。
     </strong>
    </p>
