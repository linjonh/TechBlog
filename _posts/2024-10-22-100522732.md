    <p>
     1.查看数据库，选中使用数据库，并查看数据库表，具体操作命令如下：
    </p>
    <p>
     show databases;
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/03f90fec7929ec2f9c5dff0bacced862.png">
      <br/>
      use mysql;
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6e37d45c33a948f2faf128afa35965d5.png">
       <br/>
       show tables;
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e597e176f9766416b0ae9dc547671a58.png">
        <br/>
        2.查看数据库表数据，并利用explain分析数据库表
       </img>
      </img>
     </img>
    </p>
    <p>
     select * from user;
    </p>
    <p>
     explain select * from user;
    </p>
    <p>
     3.查看数据库使用索引的情况，使用命令：
     <br/>
     show status like ‘Handler_read%’;
    </p>
    <p>
     4.用于分析和存储表的关键字，分析的结果可以得到精准的信息，利用命令analyze，
     <br/>
     analyze table user;
    </p>
    <p>
     5.检查数据库表user,检查表是否有错误，利用命令：
     <br/>
     check table user;
    </p>
    <p>
     6.优化数据库表，利用命令：
     <br/>
     optimize table user;
    </p>
    <p>
     扩展资料：
     <br/>
     可以使用命令行工具管理 MySQL 数据库（命令 mysql 和 mysqladmin)，也可以从 MySQL 的网站下载图形管理工具 MySQL Administrator, MySQL Query Browser 和 MySQL Workbench。
    </p>
    <p>
     phpMyAdmin是由 php 写成的 MySQ L资料库系统管理程程序，让管理者可用 Web 界面管理 MySQL 资料库。
    </p>
    <p>
     phpMyBackupPro也是由 PHP 写成的，可以透过 Web 界面创建和管理数据库。它可以创建伪 cronjobs，可以用来自动在某个时间或周期备份 MySQL 数据库。
    </p>
    <p>
     另外，还有其他的 GUI 管理工具，例如 mysql-front 以及 ems mysql manager, navicat等等。
    </p>
