      <br/>
      Python列表当中删除数据的操作方法主要包含四个，分别是del、pop()、remove()、clear()。下面分别对这4个方法进行介绍和代码体验。
     </img>
    </p>
    <h3>
     <a id="del_5">
     </a>
     一、del
    </h3>
    <p>
     删除列表或删除指定数据
    </p>
    <p>
     <strong>
      1、 语法
     </strong>
    </p>
    <blockquote>
     <p>
      del 目标 或 del(目标)
     </p>
    </blockquote>
    <p>
     <strong>
      2、快速体验
     </strong>
    </p>
    <p>
     2.1删除列表
    </p>
    <pre><code>list1 = ['python', 'java', 'php']
# 2种写法
del list1
# del(list1 )
print(list1) # 报错提示 NameError: name 'list1' is not defined
</code></pre>
    <p>
     2.2 删除指定数据
    </p>
    <pre><code>list1 = ['python', 'java', 'php']
# del可以删除指定下标的数据
del list1[0]
print(list1) # ['java', 'php'] ---- 'python'数据被删除
</code></pre>
    <hr/>
    <h3>
     <a id="pop_43">
     </a>
     二、pop()
    </h3>
    <p>
     删除指定下标的数据，如果不指定下标，默认删除最后一个数据，无论是按照下标还是删除最后一个，pop函数都会返回这个被删除的数据
    </p>
    <p>
     <strong>
      1、语法：
     </strong>
    </p>
    <blockquote>
     <p>
      列表序列.pop()
     </p>
    </blockquote>
    <p>
     <strong>
      2、快速体验
     </strong>
    </p>
    <pre><code># 不指定下标
list1 = ['python', 'java', 'php']

del_list = list1.pop()
print(del_list) # php
print(list1) # ['python', 'java']

# 指定下标
list2 = ['python', 'java', 'php']

del_list2 = list2.pop(1)
print(del_list2) # java
print(list2) # ['python', 'php']
</code></pre>
    <hr/>
    <h3>
     <a id="remove_78">
     </a>
     三、remove()
    </h3>
    <p>
     移除列表中某个数据的第一个匹配项
    </p>
    <p>
     <strong>
      1、 语法
     </strong>
    </p>
    <blockquote>
     <p>
      列表序列.remove(数据)
     </p>
    </blockquote>
    <p>
     <strong>
      2、快速体验
     </strong>
    </p>
    <pre><code>list1 = ['python', 'java', 'php']
list1.remove('python')
# list1.remove(list1[0]) # 和上面效果一样
print(list1)
</code></pre>
    <hr/>
    <h3>
     <a id="clear__102">
     </a>
     四、clear() ：清空列表
    </h3>
    <p>
     <strong>
      1、 语法
     </strong>
    </p>
    <blockquote>
     <p>
      列表序列.clear()
     </p>
    </blockquote>
    <p>
     <strong>
      2、快速体验
     </strong>
    </p>
    <pre><code>list1 = ['python', 'java', 'php']
list1.clear()
print(list1) # [] --- 空列表
</code></pre>
    <p>
     以上是最简单的列表删除操作方法，都是属于python入门教程范畴，所以多敲敲代码去看看再去看看官方文档就能搞懂了，对数据的操作在实际开发中还是很常见的。
    </p>
