     江西微软技术中心实训小结
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <div align="left" style="TEXT-INDENT: 24pt">
     <span style="FONT-SIZE: 12pt">
      转眼之间在江西微软技术中心已经度过了将近两个月的实训生活，在这段时间里太多的事情令人难忘，太多的事情值得我思考，太多的知识在这里获得，于是写下了这份实训小结。
     </span>
    </div>
    <div align="left" style="TEXT-INDENT: 24pt">
     <span style="FONT-SIZE: 12pt">
      先谈谈这段时间的经历吧。来之前，非常担心这里的生活会令我不适应，住宿、火食、学生环境，但是来了之后却发现这些问题技术中心早就为我们做好了挺好的安排，让我们能够迅速投入学习，解决后顾之忧。于是学习就这样开始，我们先在二十天内学习了《
     </span>
     <span style="FONT-SIZE: 12pt">
      C#
     </span>
     <span style="FONT-SIZE: 12pt">
      程序设计语言》、《高级编程技术》两本书，接着是十天做一个小项目图书馆管理系统，接着也是在二十天内学习《基于
     </span>
     <span style="FONT-SIZE: 12pt">
      C#
     </span>
     <span style="FONT-SIZE: 12pt">
      的
     </span>
     <span style="FONT-SIZE: 12pt">
      Windows
     </span>
     <span style="FONT-SIZE: 12pt">
      应用程序设计》、《面向
     </span>
     <span style="FONT-SIZE: 12pt">
      .NET
     </span>
     <span style="FONT-SIZE: 12pt">
      的
     </span>
     <span style="FONT-SIZE: 12pt">
      Web
     </span>
     <span style="FONT-SIZE: 12pt">
      应用程序设计》（现在还在这个阶段）。上课老师是从北京过来的
     </span>
     <span style="FONT-SIZE: 12pt">
      Microsoft MCT MVP
     </span>
     <span style="FONT-SIZE: 12pt">
      ，上课质量当然没有话说，在学习阶段基本上是半天上课半天上机，在做项目阶段，是全天在机器前工作。上课的教室刚好是四十八个有桌子和椅子的位置，其余是没有桌子的，供旁听用。平时经常有学校的老师、研究生还有技术中心的工作人员过来旁听。上机则是每人一台
     </span>
     <span style="FONT-SIZE: 12pt">
      Haier
     </span>
     <span style="FONT-SIZE: 12pt">
      极光
     </span>
     <span style="FONT-SIZE: 12pt">
      A6-B010
     </span>
     <span style="FONT-SIZE: 12pt">
      ，还加了内存和独显，工作空间和技术中心员工一样。我就坐在一抬头就可以看到技术中心员工的位置。最近技术中心员工还轮流给我们讲自己在平时项目开发中的一些心得经验，非常实用。
     </span>
    </div>
    <div align="left" style="TEXT-INDENT: 24pt">
     <span style="FONT-SIZE: 12pt">
      然后谈谈个人感受。既然是实训，那当然是学到了不少知识，这也不需多说。在这段时间除了学了很多知识外，也了解了许多东西。从老师跟我们的谈话中，从技术中心员工的工作气氛中，自己的思想也发生了很大的改变。曾经也学过
     </span>
     <span style="FONT-SIZE: 12pt">
      C#
     </span>
     <span style="FONT-SIZE: 12pt">
      ，了解过
     </span>
     <span style="FONT-SIZE: 12pt">
      .NET
     </span>
     <span style="FONT-SIZE: 12pt">
      ，但是没有发现过好的优势，但是现在我越来越看好
     </span>
     <span style="FONT-SIZE: 12pt">
      .NET
     </span>
     <span style="FONT-SIZE: 12pt">
      ，也非常的佩服微软公司；曾经也经常和人谈论
     </span>
     <span style="FONT-SIZE: 12pt">
      Java
     </span>
     <span style="FONT-SIZE: 12pt">
      和
     </span>
     <span style="FONT-SIZE: 12pt">
      C#
     </span>
     <span style="FONT-SIZE: 12pt">
      的优劣，但现在知到语言之间是不能比的，如果去比了这是烂废时间；现在也越来越注意为人处事学会做人和团队协做，很多事情自己一个人是做不出来的。
     </span>
    </div>
    <div align="left" style="TEXT-INDENT: 24pt">
     <span style="FONT-SIZE: 12pt">
      感谢
     </span>
     <span style="FONT-SIZE: 12pt">
      Microsoft
     </span>
     <span style="FONT-SIZE: 12pt">
      和江西政府给我们提供了这么好的一个学习、实践机会，我想我要尽自己的最大努力拿出最多的时间参与到以后的学习、实训中去。这是一个难得的机会，错过了会非常的可惜。
     </span>
    </div>
