    <p>

    </p>
    <h3>
     功能亮点
    </h3>
    <p>
     本软件包涵盖了一系列核心功能，以支持全面的质量控制流程，具体包括：
    </p>
    <ul>
     <li>
      <strong>
       频率分布分析
      </strong>
      ：帮助理解数据的分布特性。
     </li>
     <li>
      <strong>
       运行图与控制图
      </strong>
      ：如X图、R图，用于监测过程中数据的变化趋势和稳定性。
     </li>
     <li>
      <strong>
       X条和S图、R条和R图
      </strong>
      ：适用于更细致的数据波动分析。
     </li>
     <li>
      <strong>
       X和MR图
      </strong>
      （平均与移动极差图）：评估过程中的系统性变化和随机变异。
     </li>
     <li>
      <strong>
       标准偏差计算
      </strong>
      ：量化数据分散程度，是过程能力分析的基础。
     </li>
     <li>
      <strong>
       控制限设定
      </strong>
      ：基于历史数据定义正常操作界限，识别出需介入调整的异常情况。
     </li>
     <li>
      <strong>
       过程能力指数
      </strong>
      （如Cpk, Ppk）：衡量过程满足规格要求的能力。
     </li>
     <li>
      <strong>
       p图和np图
      </strong>
      ：针对不合格品率的统计分析。
     </li>
     <li>
      <strong>
       c图和u图
      </strong>
      ：适用于计件不良或缺陷的监控。
     </li>
    </ul>
