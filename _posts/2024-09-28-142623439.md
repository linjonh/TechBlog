   <div class="htmledit_views" id="content_views">
    <p>
     公司app是用纯flutter开发的，目前支持android和iOS，后续估计也会支持鸿蒙harmonyos。目前谷歌flutter并没有支持咱们国产手机操作系统鸿蒙harmonyos，于是乎国内有个叫OpenHarmony-SIG的组织，去做了鸿蒙harmonyos适配flutter开发的工作。本文就是基于OpenHarmony-SIG/flutter_flutter去搭建纯flutter开发环境。
    </p>
    <ol>
     <li>
      鸿蒙开发环境搭建，鸿蒙开发套件官方下载地址如下：
      <a href="https://gitee.com/link?target=https%3A%2F%2Fdeveloper.huawei.com%2Fconsumer%2Fcn%2Fdownload%2F" rel="nofollow" title="https://developer.huawei.com/consumer/cn/download/">
       https://developer.huawei.com/consumer/cn/download/
      </a>
     </li>
     <li>
      下载鸿蒙版flutter
      <pre><code>git clone -b dev https://gitee.com/openharmony-sig/flutter_flutter.git</code></pre>
     </li>
     <li>
      下载FlutterEngine构建产物（非必选项），网盘下载地址
      <a href="https://docs.qq.com/sheet/DUnljRVBYUWZKZEtF?tab=BB08J2" rel="nofollow" title="Flutter Engine构建产物">
       Flutter Engine构建产物
      </a>
      ，FlutterEngine构建产物方便用来在android studio来运行、调试、打包等鸿蒙harmonyos项目flutter侧代码，当然你也可以通过DevEco-Studio来运行、调试、打包等鸿蒙代码。构建命令如下，执行改命令之前需要签名。
      <pre><code class="language-Dart">flutter build hap --local-engine=/Users/administrator/Documents/src/out/ohos_debug_unopt_arm64</code></pre>
     </li>
     <li>
      安装jdk，版本需为jdk 17，下载地址为
      <a href="https://www.oracle.com/cn/java/technologies/downloads/#java17" rel="nofollow" title="Java Downloads | Oracle 中国">
       Java Downloads | Oracle 中国
      </a>
     </li>
     <li>
      环境变量配置，注意：拉取下来的flutter_flutter/bin目录必须为第二步鸿蒙版flutter下载下来的bin目录。
      <pre><code class="language-Dart">export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

# 拉取下来的flutter_flutter/bin目录
export PATH=/Users/administrator/Documents/flutter_flutter/bin:$PATH

# jdk-17.jdk jdk-11.0.17.jdk zulu-8.jdk
JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home
PATH=$JAVA_HOME/bin:$PATH:.
export JAVA_HOME
export PATH

# HamonyOS SDK
export TOOL_HOME=/Applications/DevEco-Studio.app/Contents # mac环境
export DEVECO_SDK_HOME=$TOOL_HOME/sdk # command-line-tools/sdk
export PATH=$TOOL_HOME/tools/ohpm/bin:$PATH # command-line-tools/ohpm/bin
export PATH=$TOOL_HOME/tools/hvigor/bin:$PATH # command-line-tools/hvigor/bin
export PATH=$TOOL_HOME/tools/node/bin:$PATH # command-line-tools/tool/node/bin</code></pre>
     </li>
     <li>
      检查环境，运行flutter doctor -v检查环境变量配置是否正确，Futter与OpenHarmony应都为ok
      <img alt="8cfdf6d2a493455095c5f80f75e7ff30.png" src="https://i-blog.csdnimg.cn/direct/8cfdf6d2a493455095c5f80f75e7ff30.png"/>
     </li>
     <li>
      创建Flutter工程
      <pre><code class="language-Dart"># 创建工程 方式一 该方式只创建了ohos平台
flutter create --platforms ohos &lt;projectName&gt; 

# 创建工程 方式二 该方式创建了android,ios,ohos三个平台
flutter create  &lt;projectName&gt; 

# 进入工程根目录编译hap包
flutter build hap --debug</code></pre>
     </li>
     <li>
      鸿蒙真机运行Flutter项目
      <pre><code class="language-Dart"> flutter run --debug -d &lt;deviceId&gt;

 flutter build hap --debug
 hdc -t &lt;deviceId&gt; install &lt;hap file path&gt;</code></pre>
     </li>
     <li>
      当然我们也可以通过DevEco-Studio模拟器来运行flutter项目
     </li>
    </ol>
