     <h2>
      <a id="font_face_color44cef6_size5AIfont_38">
      </a>
      <font color="#44cef6" face="微软雅黑" size="5">
       AI概述
      </font>
     </h2>
     <p>
      在这个时代
      <strong>
       AI
      </strong>
      <strong>
       与
      </strong>
      我们每个
      <strong>
       人
      </strong>
      <strong>
       息息相关
      </strong>
      <br/>
      <strong>
       1956年
      </strong>
      在美国召开了
      <strong>
       第一场人工智能研讨会
      </strong>
      ，由此人类
      <strong>
       开始了
      </strong>
      对人工智能道路探索。
      <br/>
      在这场
      <strong>
       会议上
      </strong>
      ，
      <strong>
       纽维尔
      </strong>
      和
      <strong>
       西蒙
      </strong>
      演示了一个名为"逻辑学家"的
      <strong>
       程序
      </strong>
      充分展示了
      <strong>
       机器能做类似推理
      </strong>
      的工作。在这个会议上
      <strong>
       人工智能获得了定义
      </strong>
      。
     </p>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e9e0325ca57dffe4d83859f15f17924b.png#pic_center"/>
     </p>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a138158a4067b60e05eb50ce297f15bc.png#pic_center"/>
     </p>
     <p>
      <strong>
       1978年
      </strong>
      国内
      <strong>
       第一所
      </strong>
      <strong>
       人工智能与智能控制研究
      </strong>
      组在清华大学成立，并且同年招收了第1批硕士生。那时主要以
      <strong>
       智能机器人
      </strong>
      作为主要研究方向。
      <br/>
      <strong>
       1990年
      </strong>
      ，
      <strong>
       智能技术与系统国家重点实验室
      </strong>
      正式建立，标志着中国第1次开始正式
      <strong>
       开展人工智能相关研究
      </strong>
      。
      <br/>
      时间发展
      <strong>
       至今
      </strong>
      ，人工智能已经有
      <strong>
       三个阶段
      </strong>
      ：
      <strong>
       第一代
      </strong>
      人工智能，
      <strong>
       第二代
      </strong>
      人工智能，
      <strong>
       第三代
      </strong>
      人工智能。
      <br/>
      从
      <strong>
       最初的
      </strong>
      第一代人工智能，让
      <strong>
       机器像人一样思考
      </strong>
      ，培养
      <strong>
       从已知知识出发
      </strong>
      ，
      <strong>
       推出新的结论
      </strong>
      、
      <strong>
       新的知识
      </strong>
      的能力。
      <br/>
      <strong>
       第二代人工智能
      </strong>
      主要基于
      <strong>
       人工神经网络模拟人脑脑神经网络的工作原理
      </strong>
      <br/>
      <strong>
       但是
      </strong>
      第二代人工智能由于所有
      <strong>
       训练的数据
      </strong>
      <strong>
       均来自客观世界
      </strong>
      ，从而它的识别
      <strong>
       只能识别
      </strong>
      不同的物体，并
      <strong>
       不能真正的认识
      </strong>
      物体。
      <br/>
      <strong>
       第三代人工智能
      </strong>
      则是依靠
      <strong>
       模型和算法
      </strong>
      来支持发展，并在此过程中发展了
      <strong>
       一系列人工智能理论
      </strong>
      。
      <br/>
      而目前市面上的
      <strong>
       AI大语言模型
      </strong>
      ，则是将第一代人工智能的
      <strong>
       知识
      </strong>
      为驱动，和第二代人工智能的
      <strong>
       数据
      </strong>
      以及提炼出的
      <strong>
       算法和模型以及算力
      </strong>
      同时运用
      <strong>
       而成
      </strong>
      。
      <br/>
      大语言模型的
      <strong>
       大
      </strong>
      ，来自于
      <strong>
       两个"大"
      </strong>
      <br/>
      第1个"大"是
      <strong>
       大的人工神经网络
      </strong>
      <br/>
      人工神经网络可以用来
      <strong>
       分类
      </strong>
      学习
      <strong>
       数据中间
      </strong>
      的
      <strong>
       关联关系
      </strong>
      ，也可以
      <strong>
       用来预测
      </strong>
      。
      <br/>
      第2个"大"是
      <strong>
       大的文本
      </strong>
      <br/>
      由于第1个大的发展，导致所有文本
      <strong>
       不用经过任何预处理
      </strong>
      就可以学习，所以文本就由最初的
      <strong>
       GB量级
      </strong>
      发展为
      <strong>
       TB量级
      </strong>
      。
      <br/>
      大模型的
      <strong>
       局限性
      </strong>
      <br/>
      <strong>
       缺乏主动性
      </strong>
      （依赖于提示工程）且
      <strong>
       输出质量不可控
      </strong>
      （会出现计算机"幻觉"）， 且AI工具尚
      <strong>
       不能准确分辨对错
      </strong>
      ，也
      <strong>
       难以主动
      </strong>
      进行
      <strong>
       自我迭代
      </strong>
      （也需要不断花钱去砸算力）。
      <br/>
      但目前人工智能
      <strong>
       最大的问题
      </strong>
      是：
      <br/>
      即
      <strong>
       专用人工智能
      </strong>
      （在
      <mark>
       <strong>
        特定
       </strong>
      </mark>
      的领域用
      <mark>
       <strong>
        特定
       </strong>
      </mark>
      的模型完成
      <mark>
       <strong>
        特定
       </strong>
      </mark>
      的任务）
      <br/>
      而
      <strong>
       接下来
      </strong>
      人工智能将向
      <strong>
       通用人工智能
      </strong>
      进行发展
      <br/>
      <strong>
       除此以外
      </strong>
      ，人工智能
      <strong>
       应要具有身体
      </strong>
      ，所以说必须
      <strong>
       通过机器人与客观世界连在一起
      </strong>
      <br/>
      <strong>
       在未来
      </strong>
      ，越来越多的人
      <strong>
       学习AI是大势所趋
      </strong>
      ，而
      <strong>
       学习AI的人
      </strong>
      要么向各行各业转移，为各行各业
      <strong>
       进行赋能
      </strong>
      ；要么就
      <strong>
       和其他技术结合
      </strong>
      ，发展出
      <strong>
       新的产业
      </strong>
      。
      <br/>
      各位，人工智能对各行各业都有重大影响，但
      <strong>
       大多数
      </strong>
      是
      <strong>
       帮助人类提高工作质量和效率
      </strong>
      ，
      <strong>
       而非取代人类进行工作
      </strong>
      。（引用中国科学院张钹院士的
      <a href="https://mp.weixin.qq.com/s/TwyFwYW69A72YZGVuA89jw" rel="nofollow">
       《走进“无人区”，探索人工智能之路》
      </a>
      ）
      <br/>
      送上喜欢的一句话:
      <br/>
      <mark>
       <strong>
        “让混沌重生，然后掌握混沌”
       </strong>
      </mark>
      <br/>
      <strong>
       未来已来
      </strong>
      ，
      <strong>
       不因物喜
      </strong>
      ，
      <strong>
       不以己悲
      </strong>
      ，需要的是
      <strong>
       坚持不懈的努力
      </strong>
      ，
      <strong>
       天道酬勤
      </strong>
     </p>
