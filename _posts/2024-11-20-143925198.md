     Eagle-OJ 开源的在线编程训练平台
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     Eagle-OJ 是一个开源的在线编程训练平台，旨在为学校和机构提供一个简单易用的解决方案，以便他们能够轻松搭建自己的编程训练平台。Eagle-OJ 不仅支持多种编程语言的判题，还提供了一系列便捷的功能，使其成为教学和竞赛的理想选择。以下是 Eagle-OJ 的详细介绍，包括其主要特点、功能模块和技术架构。
    </p>
    <h4>
     主要特点
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        开源和灵活
       </strong>
       :
      </p>
      <ul>
       <li>
        Eagle-OJ 是一个完全开源的项目，这意味着任何人都可以免费使用和修改它。项目托管在 GitHub 上，社区活跃，持续更新和改进。
       </li>
       <li>
        项目支持 Docker 一键快速部署，极大地简化了安装和配置过程，使得即使是技术背景较弱的用户也能轻松上手。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        多语言支持
       </strong>
       :
      </p>
      <ul>
       <li>
        支持多种编程语言，包括 Python 2.7、Python 3.5、C++、C 和 Java 8。这使得用户可以在平台上进行不同语言的编程练习和竞赛。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        丰富的功能
       </strong>
       :
      </p>
      <ul>
       <li>
        <strong>
         题库管理
        </strong>
        : 管理员可以轻松创建、编辑和删除题目，支持多种题型，包括传统题、交互题、提答题等。
       </li>
       <li>
        <strong>
         用户管理
        </strong>
        : 支持用户注册、登录、权限管理和个人信息管理。
       </li>
       <li>
        <strong>
         比赛管理
        </strong>
        : 支持创建和管理编程比赛，可以设置比赛时间和规则，支持 ACM 和 OI 模式。
       </li>
       <li>
        <strong>
         小组功能
        </strong>
        : 管理员可以创建和管理学生小组，开设小组赛，方便教学和竞赛组织。
       </li>
       <li>
        <strong>
         代码管理
        </strong>
        : 记录用户每次提交的代码，支持代码版本管理，方便用户追溯和分析代码改动。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        高性能和稳定性
       </strong>
       :
      </p>
      <ul>
       <li>
        使用高效的判题机制，确保代码在安全的沙箱环境中执行，防止恶意代码对系统造成损害。
       </li>
       <li>
        采用微服务架构，支持高并发和水平扩展，保证系统的稳定性和性能。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     技术架构
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        前端
       </strong>
       :
      </p>
      <ul>
       <li>
        使用 Vue.js 框架构建，提供响应式的用户界面，用户体验良好。
       </li>
       <li>
        项目地址:
        <a href="https://github.com/Eagle-OJ/eagle-oj-web" title="Eagle-OJ/eagle-oj-web - GitHub">
         Eagle-OJ/eagle-oj-web - GitHub
        </a>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        后端
       </strong>
       :
      </p>
      <ul>
       <li>
        使用 Spring Boot 框架构建，提供 RESTful API 接口，支持前后端分离。
       </li>
       <li>
        项目地址:
        <a href="https://github.com/Eagle-OJ/eagle-oj-api" title="Eagle-OJ/eagle-oj-api - GitHub">
         Eagle-OJ/eagle-oj-api - GitHub
        </a>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        判题沙箱
       </strong>
       :
      </p>
      <ul>
       <li>
        使用 Seccomp 和其他安全机制，确保用户提交的代码在安全的环境中执行。
       </li>
       <li>
        项目地址:
        <a href="https://github.com/Eagle-OJ/eagle-oj-judger" title="Eagle-OJ/eagle-oj-judger - GitHub">
         Eagle-OJ/eagle-oj-judger - GitHub
        </a>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        数据库
       </strong>
       :
      </p>
      <ul>
       <li>
        使用 MySQL 数据库存储用户信息、题目信息和比赛信息等。
       </li>
       <li>
        使用 Redis 缓存常用数据，提高系统性能。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        消息队列
       </strong>
       :
      </p>
      <ul>
       <li>
        使用 RabbitMQ 处理异步任务，如代码判题、邮件发送等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        Docker 化部署
       </strong>
       :
      </p>
      <ul>
       <li>
        提供完整的 Dockerfile 和 Docker Compose 文件，支持一键部署。
       </li>
       <li>
        项目地址:
        <a href="https://github.com/Eagle-OJ/eagle-oj" title="Eagle-OJ/eagle-oj - GitHub">
         Eagle-OJ/eagle-oj - GitHub
        </a>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     部署步骤
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        克隆仓库
       </strong>
       :
      </p>
      <pre></pre>
      bash
      <p>
       深色版本
      </p>
      <pre><code>git clone https://github.com/Eagle-OJ/eagle-oj.git
cd eagle-oj</code></pre>
     </li>
     <li>
      <p>
       <strong>
        修改配置文件
       </strong>
       :
      </p>
      <ul>
       <li>
        修改
        <code>
         docker-compose.yml
        </code>
        文件中的配置，如数据库连接信息、端口映射等。
       </li>
       <li>
        修改
        <code>
         .env
        </code>
        文件中的环境变量，如数据库用户名、密码等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        启动 Docker 容器
       </strong>
       :
      </p>
      <pre></pre>
      bash
      <p>
       深色版本
      </p>
      <pre><code>docker-compose up -d</code></pre>
     </li>
     <li>
      <p>
       <strong>
        访问系统
       </strong>
       :
      </p>
      <ul>
       <li>
        打开浏览器，访问
        <code>
         http://localhost
        </code>
        ，即可看到 Eagle-OJ 的登录页面。
       </li>
      </ul>
     </li>
    </ol>
