    <h2>
     <a id="_0">
     </a>
     整体思路：
    </h2>
    <p>
     <font size="4">
      想要从前端传输数据到后端，大体分三步：首先要从前端获取值，然后用ajax的方式传输到后端，接着在后端接收数据
      <br/>
      我这里实现的是从html页传数据到ashx中使用
     </font>
    </p>
    <h2>
     <a id="_3">
     </a>
     详细实现：
    </h2>
    <h3>
     <a id="1_4">
     </a>
     1、从前端获取值
    </h3>
    <p>
     <font size="4">
      （1）先为自己想要获取值的那个控件起一个名id
     </font>
    </p>
    <pre><code>&lt;div&gt;
	&lt;input id="shuru" type="text"  /&gt;
 &lt;/div&gt;
</code></pre>
    <p>
     <font size="4">
      （2）然后使用js方式获取id为“shuru”的控件
     </font>
    </p>
    <pre><code>document.getElementById("shuru");
</code></pre>
    <p>
     <font size="4">
      （3）再获取控件中的值，并将其放在一个变量中
      <br/>
      (放在一个变量中，是方便后边data后对应的数据，当然也可以不放在变量中，直接对应到data中：data: { $("#shuru").val()},效果是一样的)
     </font>
    </p>
    <pre><code>var shuru=$("#shuru").val();
</code></pre>
    <h3>
     <a id="2ajax_24">
     </a>
     2、用ajax的方式传入后端
    </h3>
    <pre><code>$.ajax({
         type: "post",
         url: "Handler1.ashx",
         data: { shuru},
         success: function (data) {
         document.getElementById("content").innerHTML = data;
         }
});
</code></pre>
    <h3>
     <a id="3_36">
     </a>
     3、在后端获取所传过来的数据
    </h3>
    <pre><code>public string shuru;
shuru=context.Request["shuru"].ToString(); 
</code></pre>
    <p>
     这样就可以使用前端所传过来的数据了。
     <br/>
     跳转页面使用：
     <br/>
     context.Response.Write(###);
     <br/>
     context.Response.End();
    </p>
    <h2>
     <a id="_46">
     </a>
     总结
    </h2>
    <p>
     <font size="4">
      其实从前端传到后端的方法有很多种，除了这种ajax方式传数据的，还有表单form也是可以传数据的。使用哪种方式，需要看自己的需要。form这种形式，它里边会有个action，这样就会定向到某一个页面，不能多次转换页面了，而ajax就不会是定向到一个网页，它只是把数据传过去了，你在后端可以根据判断条件转向多个页面，使用write就可。
     </font>
    </p>
