    <h2>
     <a id="ubuntuubuntupython39_0">
     </a>
     ubuntu篇—ubuntu安装python3.9
    </h2>
    <p>
     在ubuntu上安装Python有两种方法:在线安装和源码编译安装。
    </p>
    <h3>
     <a id="1apt_3">
     </a>
     方法1：使用apt在线安装
    </h3>
    <ul>
     <li>
      1.更新软件包列表并安装必备组件：
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo apt update
$ sudo apt install software<span class="token operator">-</span>properties<span class="token operator">-</span>common
</code></pre>
    <ul>
     <li>
      2.将Deadsnakes PPA添加到系统的来源列表中：
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo add<span class="token operator">-</span>apt<span class="token operator">-</span>repository ppa<span class="token punctuation">:</span>deadsnakes<span class="token operator">/</span>ppa
</code></pre>
    <ul>
     <li>
      3.启用存储库后，请使用以下命令安装Python 3.9：
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo apt install python3<span class="token punctuation">.</span><span class="token number">9</span>
</code></pre>
    <ul>
     <li>
      3.通过键入以下命令验证安装是否成功：
     </li>
    </ul>
    <pre><code class="prism language-python">$ python3<span class="token punctuation">.</span><span class="token number">9</span> <span class="token operator">-</span><span class="token operator">-</span>version
</code></pre>
    <h3>
     <a id="2_30">
     </a>
     方式2：源码编译安装
    </h3>
    <ul>
     <li>
      1.更新软件包列表并安装构建Python所需的软件包：
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo apt update
$ sudo apt install build<span class="token operator">-</span>essential zlib1g<span class="token operator">-</span>dev libncurses5<span class="token operator">-</span>dev libgdbm<span class="token operator">-</span>dev libnss3<span class="token operator">-</span>dev libssl<span class="token operator">-</span>dev libreadline<span class="token operator">-</span>dev libffi<span class="token operator">-</span>dev wget
</code></pre>
    <ul>
     <li>
      2.下载源代码
     </li>
    </ul>
    <pre><code class="prism language-python">$ wget https<span class="token punctuation">:</span><span class="token operator">//</span>www<span class="token punctuation">.</span>python<span class="token punctuation">.</span>org<span class="token operator">/</span>ftp<span class="token operator">/</span>python<span class="token operator">/</span><span class="token number">3.9</span><span class="token number">.16</span><span class="token operator">/</span>Python<span class="token operator">-</span><span class="token number">3.9</span><span class="token number">.16</span><span class="token punctuation">.</span>tgz
</code></pre>
    <ul>
     <li>
      3.下载完成后，解压缩下载的文件：
     </li>
    </ul>
    <pre><code class="prism language-python">$ tar <span class="token operator">-</span>xf Python<span class="token operator">-</span><span class="token number">3.9</span><span class="token number">.16</span><span class="token punctuation">.</span>tgz
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/148c8c7f12ccec9c9dcaa4e1a74a22b0.png"/>
    </p>
    <ul>
     <li>
      4.切换到Python源目录并执行configure脚本，该脚本执行许多检查以确保系统上的所有依赖项都存在：
     </li>
    </ul>
    <pre><code class="prism language-python">$ cd Python<span class="token operator">-</span><span class="token number">3.9</span><span class="token number">.16</span>
$ <span class="token punctuation">.</span><span class="token operator">/</span>configure <span class="token operator">-</span><span class="token operator">-</span>enable<span class="token operator">-</span>optimizations
</code></pre>
    <p>
     –enable-optimizations选项通过运行多个测试来优化Python二进制文件，这会使构建过程变慢。
    </p>
    <ul>
     <li>
      5.启动Python 3.9构建过程：
     </li>
    </ul>
    <pre><code class="prism language-python">$ make
</code></pre>
    <ul>
     <li>
      6.构建过程完成后，输入以下命令安装Python二进制文件：
     </li>
    </ul>
    <pre><code class="prism language-python">$ sudo make altinstall
</code></pre>
    <p>
     不要使用标准的make install，因为它将覆盖默认的系统python3二进制文件。
    </p>
    <ul>
     <li>
      7.Python 3.9已安装并准备使用，通过运行以下命令进行验证：
     </li>
    </ul>
    <pre><code class="prism language-python">$ python3<span class="token punctuation">.</span><span class="token number">9</span> <span class="token operator">-</span><span class="token operator">-</span>version
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e6ec3cfda3272b2fafa16eae3bbc3765.png">
      <br/>
      由于后面带了尾巴使用不便，所以想把它去除，方法是：
      <br/>
      先找到python的位置
     </img>
    </p>
    <pre><code class="prism language-python">which python3<span class="token punctuation">.</span><span class="token number">9</span>

update<span class="token operator">-</span>alternatives <span class="token operator">-</span><span class="token operator">-</span>install <span class="token operator">/</span>usr<span class="token operator">/</span><span class="token builtin">bin</span><span class="token operator">/</span>python python <span class="token operator">/</span>usr<span class="token operator">/</span>local<span class="token operator">/</span><span class="token builtin">bin</span><span class="token operator">/</span>python3<span class="token punctuation">.</span><span class="token number">9</span> <span class="token number">1</span>
</code></pre>
