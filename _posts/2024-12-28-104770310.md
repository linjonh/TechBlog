    <p>
     前言：开发过程、测试过程、生产过程使用的接口地址不能，还有执行的操作可能也不一样，也就需要实现配置好开发环境、测试环境、生产环境，需要什么环境下的配置直接使用即可。
    </p>
    <p>
     <strong>
      1、根目录下新建文件：
      <code>
       .env.development（开发环境）
      </code>
      、
      <code>
       .env.test（测试环境）
      </code>
      、
      <code>
       .env.production
      </code>
      文件（生产环境）
     </strong>
    </p>
    <p>
     <img alt="" height="292" src="https://i-blog.csdnimg.cn/blog_migrate/5d77a5576bb3b99f3eedcd7df30cabe5.png" width="318"/>
    </p>
    <p>
     <strong>
      2、三个配置文件的配置内容如下：
     </strong>
    </p>
    <p>
     <strong>
      <code>
       .env.development（开发环境）配置内容
      </code>
     </strong>
    </p>
    <pre><code>NODE_ENV = 'development'  //模式
VUE_APP_MODE = 'development'  //通过"VUE_APP_MODE"变量来区分环境
VUE_APP_API_URL = 'http://192.****:8008/api/' //api地址</code></pre>
    <p>
     <strong>
      <code>
       .env.test（测试环境）配置内容
      </code>
     </strong>
    </p>
    <pre><code>NODE_ENV = 'production'
VUE_APP_MODE = 'test'
VUE_APP_API_URL = 'http://xxx.xxx.xxx.xx:8008/api/'
outputDir = test</code></pre>
    <p>
     <strong>
      <code>
       .env.production
      </code>
      文件（生产环境）配置内容
     </strong>
    </p>
    <pre><code>NODE_ENV = 'production'
VUE_APP_MODE = 'production'
VUE_APP_API_URL = 'http://xxx.xxx.xxx.xx:8008/api/'</code></pre>
    <p>
     <strong>
      3、修改
      <code>
       vue.config.js
      </code>
     </strong>
    </p>
    <p>
     <img alt="" height="131" src="https://i-blog.csdnimg.cn/blog_migrate/d638f86dccfdf85df1ff907fd9bfdd6f.png" width="360"/>
    </p>
    <pre><code>module.exports = {

    // 基本路径,相对路径
    publicPath: "./",
    // 输出文件目录
    outputDir: process.env.outputDir,</code></pre>
    <p>
     <strong>
      4、修改
      <code>
       package.json
      </code>
      文件
     </strong>
    </p>
    <p>
     <img alt="" height="143" src="https://i-blog.csdnimg.cn/blog_migrate/c4521cc3a63bdcc592cd3a16f181fc57.png" width="546"/>
    </p>
    <pre><code>    "test": "vue-cli-service build --mode test", //打包测试环境
    "publish": "vue-cli-service build &amp;&amp; vue-cli-service build --mode test", //测试和生产一起打包</code></pre>
    <p>
     <strong>
      5、判断并使用不用的开发环境配置
     </strong>
    </p>
    <pre><code>if(process.env.VUE_APP_MODE==='development'){
    //开发环境下的执行操作
}else if(process.env.VUE_APP_MODE==='test'){
    //测试环境下的执行操作
}else{
    //生产环境下的执行操作
}
</code></pre>
