    <h3>
     <a id="_10">
     </a>
     一.学习感悟
    </h3>
    <p>
     经过从 2021 年跟着 b 站河北王校长学习,从最开始的集合,并发编程,JVM,到后面的 mysql,redis,spring,自己学习到了很多.
    </p>
    <p>
     第一个方面自己在 2021 年之前也在不断学习,但是那个时候没有总结的习惯,很多东西看过一遍,当时明白了,但是过了一个星期就忘记了,更不用说过了几个月,忘的影都没了,其实忘记了并不可怕,忘记了我们需要知道如何在自己的知识库在最短的时间内检索出来,进行复习巩固之前学过的知识,这是我觉得 2021 年中旬以来,自己最大的进步,用 typora 写技术博客,记录开发过程中遇到的问题,并总结自己学习网课或者 B 站视频的知识,不断完善自己的知识库,更有深度的去理解技术.不仅要全,还需要有深度.
     <br/>
     可以光顾下私人博客,使用的阿里云白嫖的服务器,
     <a href="http://qinyingjie.top/" rel="nofollow">
      kuan 的解忧杂货铺
     </a>
     ,欢迎指正.
    </p>
    <p>
     第二个方面是自己跟着河北王校长学习到一种学习方法,这个方法就是书要多读,即使比较枯燥无味,我们也可以带着问题去读,然后通过多次读书,更深刻的理解问题.如果读了书之后,我们可以先通过 B 站视频,先了解下要学的这个技术怎么使用,再通过书籍了解底层的原理.
    </p>
    <p>
     再说下临近几个月的学习,在二月份的时候,结束了 kubernetes 的学习,知道了 kubernetes 的组件和架构,会使用一些命令,部署微服务到 kubernetes 在 2022 年 12 月和 2023 年 1 月主要在学习 SpringCloud
     <br/>
     alibaba 的知识,并对常用组件熟悉了原理并进行实战和总结,对自己的提升还是蛮大的.
    </p>

    <h3>
     <a id="_35">
     </a>
     三.之前的学习
    </h3>
    <p>
     以下是之前已经学习总结的知识点：
    </p>
    <ol>
     <li>
      学习 Java 基础知识，包括语法、面向对象编程、集合框架等。
     </li>
     <li>
      学习 Java Web 开发，包括 Servlet、JSP、Spring 框架等。
     </li>
     <li>
      学习数据库编程，包括 SQL 语言、JDBC 等。
     </li>
     <li>
      学习常用的开发工具，如 Eclipse、IntelliJ IDEA 等。
     </li>
     <li>
      学习常用的版本控制工具，如 Git。
     </li>
     <li>
      参加 Java 相关的在线课程，以加速学习进程。
     </li>
     <li>
      阅读 Java 相关的书籍和博客，了解行业最新动态和技术趋势。
     </li>
     <li>
      参加技术交流会议和社区活动，扩展人脉和了解最新技术动态。比如 Sentinel 开源讨论群,会有双周会
     </li>
     <li>
      持续关注 Java 技术的发展趋势，以保持对行业的敏感度和竞争力。
     </li>
     <li>
      持续练习和实践，不断提升自己的编程能力和解决问题的能力。
     </li>
     <li>
      学习 Java 多线程编程，了解线程池、锁、并发编程等相关知识。
     </li>
     <li>
      学习 Java 虚拟机（JVM）的原理和调优技巧，以提升应用程序的性能。
     </li>
     <li>
      学习 Java 网络编程，了解 TCP/IP 协议、Socket 编程、HTTP 协议等。
     </li>
     <li>
      学习 Java 安全编程，包括数字证书、加密算法、安全传输协议等。
     </li>
     <li>
      学习 Java 容器化技术，如 Docker、Kubernetes 等。
     </li>
     <li>
      学习 Java 微服务开发，包括 Spring Cloud、Dubbo 等。
     </li>
     <li>
      学习 Java 大数据技术，如 Hadoop、Spark 等。
     </li>
    </ol>
