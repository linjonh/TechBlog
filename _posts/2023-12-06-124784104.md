    <p style="margin-left:.0001pt;text-align:center;">
     D2D通信技术
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      一、
      <strong>
       <strong>
        什么是D2D通信技术
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      D2D即
      <strong>
       <strong>
        Device-to-Device
       </strong>
      </strong>
      ，也称之为
      <strong>
       <strong>
        终端直通
       </strong>
      </strong>
      。D2D通信技术是指两个对等的用户节点之间直接进行通信的一种通信方式。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="327" src="https://i-blog.csdnimg.cn/blog_migrate/bea1ce84b7d70b3a3c941dc2c8320eff.png" width="431"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      如图中所示。在由D2D通信用户组成的分散式网络中，每个用户节点都能发送和接收信号，并具有自动路由(转发消息)的功能。网络的参与者共用它们所拥有的一部分硬件资源，包括信息处理、存储以及网络连接能力等。这些共用资源向网络提供服务和资源，能被其它用户直接访问而不需要经过中间实体。在D2D通信网路中，用户节点同时扮演伺服器和客户端的角色，用户能够意识到彼此的存在，自组织地构成一个虚拟或者实际的群体。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      二、
      <strong>
       <strong>
        D2D通信技术
       </strong>
      </strong>
      <strong>
       <strong>
        的
       </strong>
      </strong>
      <strong>
       <strong>
        优势
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （1）大幅度提供频谱利用率
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      在该技术的应用下，用户通过D2D 进行通信连接，避开了使用蜂窝无线通信，因此不使用频带资源。而且，D2D 所连接的用户设备可以共享蜂窝网络的资源，提高资源利用率。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （2）改善用户体验
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      随着移动互联网的发展，相邻用户进行资源共享，小范围社交以及本地特色业务等，逐渐成为一个重要的业务增长点。D2D 在该场景的应用是可以提高用户体验的。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （3）拓展应用
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      传统的通信网需要进行基础设置建设等，要求较高，设备损耗或影响整个通信系统。而D2D 的引入使得网络的稳定性增强，并具有一定灵活性，传统网络可借助D2D 进行业务拓展。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        5G D2D通信主要应用场景
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （1）本地业务
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       社交应用
      </strong>
      通过D2D通信功能，可以进行如内容分享、互动游戏等邻近用户之间数据的传输，用户通过D2D的发现功能寻找邻近区域的感兴趣用户。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       本地数据传输
      </strong>
      利用D2D的邻近特性及数据直通特性实现本地数据传输，在节省频谱资源的同时扩展移动通信应用场景。如基于邻近特性的本地广告服务可向用户推送商品打折促销、影院新片预告等信息，通过精确定位目标用户使得效益最大化。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       蜂窝网络流量卸载
      </strong>
      <strong>
      </strong>
      利用D2D通信的本地特性开展的本地多媒体业务，可以大大节省网络核心层及频谱的资源。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （2）应急通信
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      D2D通信可以解决极端自然灾害引起通信基础设施损坏导致通信中断而给救援带来障碍的问题。在D2D通信模式下，两个邻近的移动终端之间仍然能够建立无线通信，为灾难救援提供保障。另外，在无线通信网络覆盖盲区，用户通过一跳或多跳D2D通信可以连接到无线网络覆盖区域内的用户终端，借助该用户终端连接到无线通信网络。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （3）物联网增强
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      如果D2D通信技术与物联网结合，则有可能产生真正意义上的互联互通无线通信网络。车联网中的V2V（Vehicle-to-Vehicle）通信就是典型的物联网增强的D2D通信应用场景。基于终端直通的D2D由于在通信时延、邻近发现等方面的特性，使得其应用于车联网车辆安全领域具有先天优势。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       三、
      </strong>
      <strong>
       <strong>
        D2D系统架构
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      按照蜂窝网络覆盖范围区分，可以把D2D通信分成3种场景：
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      （1）蜂窝网络覆盖下的D2D通信，LTE基站首先需要发现D2D通信设备，建立逻辑连接，然后控制D2D设备的资源分配，进行资源调度和干扰管理，用户可以获得高质量的通信。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      （2）部分蜂窝网络覆盖下的D2D通信，基站只需引导设备双方建立连接，而不再进行资源调度，其网络复杂度比第一类D2D通信有大幅降低。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      （3）完全没有蜂窝网络覆盖下的D2D通信，用户设备直接进行D2D通信，该场景对应于蜂窝网络瘫痪的时候，用户可以经过多跳，相互通信或者接入网络。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       四、
      </strong>
      <strong>
       <strong>
        5G 通信网中应用D2D 技术存在的问题
       </strong>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （1）传统蜂窝网络需要全面修改和升级。
       </strong>
      </strong>
      要想在5G 通信网中应用D2D 技术，首先要确保其不会与D2D 通信技术产生冲突。然而，传统蜂窝网络比较封闭，无法支持D2D 通信的有效应用。因此对传统蜂窝网络进行全面的修改与升级非常有必要，其中包括元件升级，控制平面修改、数据平面修改等，是一个极大的工程，必须要有足够先进的技术支持和大量的资金投入。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （2）频谱资源共享造成的干扰。
       </strong>
      </strong>
      D2D 技术在5G 通信网中的应用可以有效解决频谱资源不足的问题，依靠设备之间的直接连接进行通信，大幅提高了频谱资源的利用率。但频谱资源的共享，可能会对用户的通信造成干扰，从而影响用户通信体验。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#ffffff;">
      <strong>
       <strong>
        （3）通信高峰造成的通信问题。
       </strong>
      </strong>
      与4G 网络相比，5G 网络在传输速度、效率等方面都有所提升，尤其是对延迟、资源使用率和可扩展性等提出了较高要求。但是当大量用户同时通过D2D 设备连接入网时，很可能造成5G 网络通信延迟大幅提升，对用户的实际使用造成影响。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     总之，作为面向5G 关键技术之一的D2D 通信技术, 在依托未来移动通信技术的超高速率、超大带宽、超大规模接入能力和超大数据处理能力等特征的基础上, 通过发挥自身的核心技术优势，将在5G 移动通信运用中充分彰显自身优势。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
