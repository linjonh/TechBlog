     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      <strong>
       1、下载python3.5
      </strong>
     </span>
    </p>
    <p style='padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; color:rgb(80,80,80); font-family:宋体,"Arial Narrow",arial,serif; line-height:28px'>
     wget https://www.python.org/ftp/python/3.5.0/Python-3.5.0.tgz
    </p>
    <p style='padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; color:rgb(80,80,80); font-family:宋体,"Arial Narrow",arial,serif; line-height:28px'>
     注：如果在Linux中下载较慢，可以在Windows操作系统中去Python官网下载：
     <a href="https://www.python.org/downloads/release/python-350/" rel="nofollow noopener noreferrer" target="_blank">
      https://www.python.org/downloads/release/python-350/
     </a>
    </p>
    <p style='padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="color:rgb(80,80,80); line-height:28px">
      注意要下载
      <a href="https://www.python.org/ftp/python/3.5.0/Python-3.5.0.tgz" rel="nofollow noopener noreferrer" style="" target="_blank">
       Gzipped source tarball
      </a>
      版本的，然后使用WinSCP将下载好的文件
     </span>
     <span style='color:rgb(80,80,80); line-height:28px; font-family:宋体,"Arial Narrow",arial,serif; font-size:14px'>
      Python-3.5.0.tgz拖拽到Linux中，后续步骤还是不变的。
     </span>
     <br/>
    </p>
    <p style='color:rgb(80,80,80); padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="line-height:21px">
      <strong>
       2、解压
      </strong>
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     tar zxvf Python-3.5.0.tgz
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <strong>
      3、进入
      <span style='color:rgb(80,80,80); font-family:宋体,"Arial Narrow",arial,serif; font-size:14px; line-height:28px'>
       Python-3.5.0文件夹
      </span>
     </strong>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     cd Python-3.5.0
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <strong>
      4、配置安装位置
     </strong>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      ./configure --prefix=/usr/local/python3.5
     </span>
    </p>
    <p style='color:rgb(80,80,80); padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="line-height:21px">
      注：如果没有安装C语言编译器会提示错误。如果出现错误，在联网的情况下使用 yum install gcc 命令安装gcc编译器
     </span>
    </p>
    <p style='color:rgb(80,80,80); padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="line-height:21px">
      <strong>
       5、编译
      </strong>
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      make
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      <strong>
       6、安装
      </strong>
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      make install
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <strong>
      7、下载并安装setuptools 18.5
     </strong>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     wget https://bootstrap.pypa.io/ez_setup.py -O - | python
    </p>
    <p style='line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="color:#505050">
      注：如果提示错误
     </span>
     <span style="color:rgb(240,255,240); font-family:Verdana,Arial,Tahoma; font-size:14px; line-height:25px; background-color:rgb(0,0,0)">
      --no-check-certificate
     </span>
     <span style="color:#ffffff">
      <span style="font-family:Verdana,Arial,Tahoma; font-size:14px; line-height:25px; background-color:rgb(0,0,0)">
      </span>
     </span>
    </p>
    <p style='line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     在wget后加上 --no-check-certificate :
     <span style='color:rgb(80,80,80); font-family:宋体,"Arial Narrow",arial,serif; font-size:14px; line-height:28px'>
      wget
      <span style='font-family:宋体,"Arial Narrow",arial,serif; font-size:14px; line-height:28px'>
       --no-check-certificate
       <span style='color:rgb(80,80,80); font-family:宋体,"Arial Narrow",arial,serif; font-size:14px; line-height:28px'>
        https://bootstrap.pypa.io/ez_setup.py -O - | python
       </span>
      </span>
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <span style="padding:0px; margin:0px; line-height:1.5; background-color:inherit">
      <strong>
       8、备份原有python命令执行文件
      </strong>
     </span>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     mv /usr/bin/python /usr/bin/pythonbak
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     <strong>
      9、创建新python软连接
     </strong>
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
     ln -s /usr/local/python3.5/bin/python3.5 /usr/bin/python
    </p>
    <p style='color:rgb(80,80,80); line-height:28px; padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:0px; font-size:14px; clear:both; height:auto; overflow:hidden; font-family:宋体,"Arial Narrow",arial,serif'>
