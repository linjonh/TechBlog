     证书很多的.软件工程师和MCSD最好.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     系统分析师.软件设计师 .网络工程师 .电子商务设计师 .数据库系统工程师 .信息系统项目管理师 .系统架构师 .软件程序员 .电子商务设计员 .信息系统监理师 .软件评测师.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     微软认证是国际上计算机科学最权威最专业的.当然也是最难和最贵的.主要有:MCTS 「微软认证技术工程师 (MCTS) 」认证能够让专业人员将目标放在特定的技术上.在工作中展示他们深度的知识及独特的专长.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCITP 「微软认证 IT专家 (MCITP)」认证可表明您在规划.部署.支持.维护和优化 IT 基础架构的特殊专长.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCPD 「微软认证开发专家 (MCPD)」认证可表明您是专家级的 Windows 应用程序开发人员.Web 应用程序开发人员或企业应用程序开发人员.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCAP 「微软认证架构师计划 (MCAP)」的对象是解决方案架构设计师与基础结构架构设计师.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCDST 「微软认证操作系统技术支持工程师 (MCDST)」认证证明您拥有技术和客户服务技巧.可以疑难解答Windows环境中的硬件和软件作业问题.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCSA 「微软认证系统管理员 (MCSA)」认证表明您懂得管理以 Windows 平台为基础的网络与系统环境.特别包含有「MCSA:信息处理」和「MCSA:信息安全」两个特别科目.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCSE 「微软认证系统工程师 (MCSE)」认证表明您懂得以 Windows 平台与 Microsoft Server 软件为基础的基础结构解决方案.特别包含有「MCSE:信息处理」和「MCSE:信息安全」两个特别科目.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCDBA 「微软认证数据库管理员 (MCDBA)」认证表明您懂得设计.操作及管理 Microsoft SQL Server 数据库.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCT 「微软认证讲师 (MCT)」是微软认可的专门讲师.负责为 IT 专业人员与开发人员讲授微软培训课程.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCAD 「微软认证应用程序开发专家 (MCAD)」认证表明您能使用微软技术来开发及维护部门层次的应用程序.组件.Web 或桌面客户端.或是后端数据服务.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
     MCSD 「微软认证解决方案开发专家 (MCSD)」认证表明您能使用微软开发工具.技术.平台与 Windows 结构来设计及开发尖端企业级解决方案.
    </span>
    <br style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px"/>
    <p>
     <span style="color:rgb(69,69,69); font-family:arial,宋体,sans-serif,tahoma,'Microsoft YaHei'; font-size:14px; line-height:24px">
      MOS 「微软 Office 专家 (MOS)」认证展现您使用微软办公软件的技能.
     </span>
    </p>

       现在中英文都有
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       MCSE中文考试介绍
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       对MCSE认证的信息，想必大家已经知道不少了，可大多数人考的都是英文认证。有些朋友们很关心有没有中文考试的MCSE的认证，它的认证和英文认证又有什么区别。我有幸获得MCSE的中文考试证书，我现在将我所知道的一些信息与大家分享。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       相同点：
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       1．英文考试和中文考试所获得的证书都是一样的国际通行认证，都有ID号,都有比尔盖茨先生的签名。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       2．英文考试费为450元一门,中文考试的费用相同为300元一门。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       3．英文考试和中文考试的题目差不多，不一样的只是语言。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       不同点：
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       1．英文考试和中文考试的考试站不同。中文考试站为ATA站，英文考试为Prometric站。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       2．ATA站的考试必须在10个人以上才可以开考。
      </span>
      <br style="word-wrap:break-word; outline:none; color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px"/>
      <span style="color:rgb(68,68,68); font-family:'Microsoft yahei',微软雅黑,Arial,Helvetica,sans-serif,宋体; font-size:14px; line-height:21px">
       3．英文考试和中文考试，分科MCP证书的样式不同。这里的不同不针对MCSE,MCDBA,MCSA,MCP证书，只针对分科的MCP证书，就是后面有成绩的那个证书。中文证书的样式与英文的基本相同，只是将考试科目翻译成了中文。举个例子，比如说我考取了70-216，英文证书会写dbking has successfully completed a Microsoft Certified Professional 。而中文证书会写dbking has successfully completed a Microsoft Certified Professional exam on Windows 2000 网络基本架构的实现。再次强调无论中英文，最终证书——MCSE,MCSA,MCDBA,MCP——的样式价值功能都是一样的。
