    <h2>
     1. 引言
    </h2>
    <p>
     在网络上，有许多精彩的视频资源，其中包括了Bilibili上的各种视频。有时候我们可能需要将这些视频下载到本地进行观看或进一步编辑。本篇博客将介绍如何使用Python爬虫来下载Bilibili视频，并且将其音视频文件合并成一个完整的视频文件。
    </p>
    <h3>
     2. 代码功能解析
    </h3>
    <p>
     我们先来看一下代码的主要功能：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        提取标题信息
       </strong>
       ：通过正则表达式从 Bilibili 视频页面中提取视频的标题信息。
      </p>
     </li>
     <li>
      <p>
       <strong>
        获取音视频信息
       </strong>
       ：使用正则表达式提取页面中的 JavaScript 数据，并使用 JSON 解析得到音视频的 URL。
      </p>
     </li>
     <li>
      <p>
       <strong>
        下载音视频文件
       </strong>
       ：使用
       <code>
        requests
       </code>
       库下载音视频文件，并保存到本地。
      </p>
     </li>
     <li>
      <p>
       <strong>
        合并音视频文件
       </strong>
       ：通过调用
       <code>
        ffmpeg
       </code>
       命令，将下载的音视频文件合并为一个新的视频文件。
      </p>
     </li>
    </ul>
    <h3>
     3. 使用说明
    </h3>
    <p>
     在运行代码之前，需要做以下准备：
    </p>
    <ul>
     <li>
      安装 Python 环境，并确保已安装以下库：
      <code>
       requests
      </code>
      、
      <code>
       ffmpeg-python
      </code>
      。
     </li>
     <li>
      获取 Bilibili 视频的页面 URL，例如
      <code>
       https://www.bilibili.com/video/BV11V411M7Fi?p=1
      </code>
      。
     </li>
    </ul>
    <p>
     运行代码时，需要传入一个 Response 对象作为参数，该对象包含从 Bilibili 视频页面获取的响应内容。同时，需要指定下载的视频页数范围。
    </p>
    <p>
     下载的视频文件将保存在当前目录下的
     <code>
      video
     </code>
     文件夹中，合并后的视频文件也保存在
     <code>
      video
     </code>
     文件夹中。
    </p>
    <h3>
     4. 依赖说明
    </h3>
    <p>
     该代码依赖以下库和工具：
    </p>
    <ul>
     <li>
      <code>
       requests
      </code>
      ：用于发送 HTTP 请求和获取响应内容。
     </li>
     <li>
      <code>
       re
      </code>
      ：用于正则表达式匹配。
     </li>
     <li>
      <code>
       json
      </code>
      ：用于解析 JSON 格式的数据。
     </li>
     <li>
      <code>
       ffmpeg
      </code>
      ：用于音视频处理，包括合并音视频文件。
     </li>
    </ul>
    <p>
     在运行代码之前，请确保已安装好这些依赖项。你可以使用以下命令来安装它们：
    </p>
    <p>
     <code>
      pip install requests ffmpeg-python
     </code>
    </p>
