    <p>
     python安装炒鸡详细的教程。
     <br/>
     一个电脑小白的自我成长之路^_*.
     <br/>
     现在回归一下自己的安装之路，劈开了一些坑。
    </p>
    <p>
     打开浏览器，输入python。如下图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/431415a583b565971dfd5bf73c6f0f83.png#pic_center">
      <br/>
      找到welcome to python.org 官网，双击打开官网。如下：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/98edf0f17b0f5e47835342700ab3dc37.png#pic_center">
       <br/>
       点一下 Downloads （或者右键），在跳出的选项中点击 All release 会跳出如下界面：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9e81973341c6c1b446c338ca58028c3c.png#pic_center">
        <br/>
        你可以直接下载最新版本，如果你想下载安装其他版本，滚动鼠标往下，找到Looking for a specific release?选项卡中，选择你想要安装的版本。（本文以python3.7.7为例）博主选择的python3.7.7
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d1be6f8d7d97439321e529b9bf4032b6.png#pic_center">
         <br/>
         双击你要安装的版本，跳出网页：
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/678c8816fcc756e650734c24275bf325.png#pic_center"/>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     鼠标滚动最底部，如果你是window10的系统，选择Windows x86-64 executable installe，如果你是macOS操作系统，选择macOS 64-bit installer。下载到你的电脑的某个文件夹中，打开文件夹。
     <br/>
     找到python-3.7.7-amd64.exe这个执行文件。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ed30bcf117895c3f838cc316fef98bf6.png#pic_center">
      <br/>
      双击运行，界面如下：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8afa899cd6eaa6c9e373cb7b1073a431.png#pic_center">
       <br/>
       选择customize installation(也就是自定义安装，install now 是默认安装，为了安装后自己容易找到文件夹 推荐自定义安装，同时可以减少C盘的占用率)
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14ef4a453a14359c7226988226e76436.png#pic_center"/>
       <br/>
       点击next，跳出界面如下：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17afa16361dc200b99ba19a7d88b5fa0.png#pic_center"/>
       <br/>
       点击browse ， 选择自己需要安装的路径，（博主选择的是D盘下的安装程序文件夹），然后点击install
       <br/>
       等待安装完成，在自己电脑的开始中找到IDLE
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7ce7f7f996d7fb5850248e4473295057.png#pic_center"/>
       <br/>
       点击打开python shell 命令框。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1beb634b798dd92b517ddba5401262e0.png#pic_center"/>
       <br/>
       输入a = ‘hello’
       <br/>
       回车再输入 print(a)
       <br/>
       再回车看一下结果
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a263f04435ec1ea0bf2996c107efa02f.png#pic_center"/>
       <br/>
       恭喜你，python安装成功。。。。。。。
      </img>
     </img>
    </p>
