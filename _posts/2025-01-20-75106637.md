    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       对于0
      </span>
      基础想进入这一行的朋友们来说，肯定会有很多的疑惑，前端到底要学些什么内容？学习前端到底难不难？前端又是具体做些什么的？等等问题。今天我们就来详细说说什么是前端。
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#595959">
       <span style="font-size:14px">
        <br/>
       </span>
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       到底什么叫前端？
      </span>
     </strong>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       Web
      </span>
      <span style="color:rgb(89,89,89)">
       前端开发是从网页制作演变而来的，互联网的演变进程，网站的前端也由此发生了翻天覆地的变化。网页不再只是承载单一的文字和图片，各种富媒体让网页的内容更加生动，网页上软件化的交互形式为用户提供了更好的使用体验，这些都是基于前端技术实现的。无论是开发难度上，还是开发方式上，现在的网页制作都更接近传统的网站后台开发，所以现在不再叫网页制作，而是叫Web
      </span>
      前端开发。
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       简单来说,
      </span>
      <span style="color:#595959">
       <br/>
      </span>
      <span style="color:#595959">
       前：代表与人直接展示的部分，包括界面与用户的交互等
       <br/>
      </span>
      端：代表输出终端，如pc浏览器，手机浏览器，和有的app、应用程序等
      <br/>
      合起来的意思也就是这些浏览器，app，应用程序的界面展现以及用户交互就是前端
     </span>
    </p>
    <p>
     <span style="color:#595959">
      <span style="font-size:14px">
       前端的主要职能：把网页界面更好的呈现给用户，与后端相比更加注重页面性能与用户体验。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       <br/>
      </span>
      <strong>
       前端工程师主要做什么？
      </strong>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       <br/>
      </span>
      <span style="color:#595959">
       前端工程师主要利用HMTL
      </span>
      与CSS建构页面，用JavaScript完善交互以及用户体验。
      <span style="color:#595959">
       互动效果包括，弹出层，页签切换，图片滚动，ajax
      </span>
      异步互动等。
     </span>
    </p>
    <p>
     <span style="color:#595959">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       前端开发要学哪些内容？
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:#595959">
       <span style="font-size:14px">
       </span>
      </span>
     </strong>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       Web
      </span>
      <span style="color:#595959">
       三项组成：
       <strong>
        结构、表现、行为
       </strong>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#595959">
      <span style="font-size:14px">
       前端人员遵守的第一原则：“结构与表现分离”
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       <strong>
        结构标准（
       </strong>
       HTML
      </span>
      、XHTML、XML
      <strong>
       ）
      </strong>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       HTML
      </span>
      <span style="color:#595959">
       ：超文本标记语言
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       XML
      </span>
      <span style="color:#595959">
       ： 可扩展标记语言
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       XHTML
      </span>
      <span style="color:#595959">
       ：可扩展超文本标记语言
      </span>
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       <span style="color:#595959">
        表现标准（CSS
       </span>
       ）
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       <span style="color:#595959">
        行为标准（JavaScript
       </span>
       ）
      </span>
     </strong>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <strong>
       <span style="color:rgb(89,89,89)">
        HTML
       </span>
       <span style="color:rgb(89,89,89)">
        ：
       </span>
      </strong>
      <span style="color:rgb(89,89,89)">
       掌握HTML
      </span>
      是网页的核心，web应用的基石
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       这种语言是目前web
      </span>
      领域应用最广泛的语言，我们只需要在html文档中插入对应的标签，即可实现web页面的编写与排列。
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <strong>
       <span style="color:rgb(89,89,89)">
        CSS
       </span>
       <span style="color:rgb(89,89,89)">
        ：
       </span>
      </strong>
      <span style="color:rgb(89,89,89)">
       程序员的画笔，实现网页外观的重要一点，可以将网页外观做得更加美观。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       可以为html
      </span>
      标签自定义样式，通过设置html标签的样式，来改变其外观，达到美化与排版web页面的目地。
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <strong>
       <span style="color:#595959">
        JavaScript
       </span>
       <span style="color:#595959">
        ：
       </span>
      </strong>
      <span style="color:#595959">
       实现网页实时的、动态的、可交互式的表达能力。
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       作为一种网页脚本语言，它可以在html
      </span>
      中运行，设置并修改浏览器中的对象与变量。
     </span>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       学习HTML
      </span>
      ，CSS首先应该跟着书，资料进行基础系统的学一遍。作为前端，html+css，是基础中的基础。只有先把基础学扎实了，再来谈余下的。理论知识学透了，就要开始来进行实战演练，只有大量的练习，做各种各样的布局练习来巩固自己所学的理论知识，将理论得到实现。
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       而学习JavaScript
      </span>
      ，首先我们要对他有个基本的了解，梳理Js的只是结构，知道这门语言能做什么，不能做什么。记住大部分JavaScript的函数，知道如何实现，并多加练习，花大量的时间在浏览器控制台上写代码，测试if-else语句，数组，函数等。
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">
       建议在
      </span>
      <a href="http://www.f-z.cn/" rel="nofollow noopener noreferrer" target="_blank">
       www.f-z.cn
      </a>
      <span style="color:rgb(89,89,89)">
       进行学习，边学边练。入门教程学习完后，可以直接在训练营进行实战。
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
       学习过程中，你要锻炼和掌握大力写代码，持续地练习下去，每次一点点的进步积累起来你会发现效果惊人，最重要的是持之以恒。
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
       不要妄想通过看书掌握前端，前端是做出来的，不是看出来的。
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <span style="color:rgb(89,89,89)">
      <span style="font-size:14px">
      </span>
     </span>
    </p>
    <p>
     <strong>
      <span style="font-size:14px">
       除此之外作为一名前端工程师还需要掌握哪些？
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:#595959">
       <span style="font-size:14px">
       </span>
      </span>
     </strong>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       一、
      </span>
      <span style="color:#595959">
       计算机
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       1.
      </span>
      <span style="color:#595959">
       计算机组成原理
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       2.
      </span>
      <span style="color:#595959">
       计算机操作系统
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       3.
      </span>
      <span style="color:#595959">
       数据结构
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       4.
      </span>
      <span style="color:#595959">
       计算机网络编程
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       5.
      </span>
      <span style="color:#595959">
       计算机网络
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       二、
      </span>
      <span style="color:#595959">
       切图技能
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       1.
      </span>
      <span style="color:#595959">
       Ps/Ai
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">
       三、
      </span>
      <span style="color:rgb(89,89,89)">
       软能力：
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:rgb(89,89,89)">

      </span>
      <span style="color:rgb(89,89,89)">

    </p>

     <span style="font-size:14px">

       英语能力

     </span>
    </p>
    <p>
     <span style="font-size:14px">
      <span style="color:#595959">

      </span>
      <span style="color:rgb(89,89,89)">
