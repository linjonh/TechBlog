    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       一，如何免费制作小程序
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     可能有人会问，那么如果我想免费申请小程序可以吗？答案当然是OK的。以商业用途的小程序为例，接下来我给大家详细介绍一下免费申请小程序的方法：
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       1，注册
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     制作小程序的第一步，都是先到微信公众平台里注册。在公众平台点击“立即注册”，选择账号注册类型。（“小程序”认证过的公众号可以免费申请小程序）在微信公众平台上点击右上角：“立即注册”。选择微信小程序，使用一个没有注册过公众平台的邮箱来注册小程序，填写小程序账号信息后，系统会发送一个邮件，你登陆相应邮箱激活账号完成注册流程即可。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       2，完善信息
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     接下来比较关键，激活后，选择“主体类型”：个人、企业、政府、媒体或者其他组织。按照你的用途选择。注册完成后即可使用小程序账号密码登入小程序后台。在【设置】里填写完善小程序信息的名称、头像和账号等信息，提交后等待微信审核。
    </p>
    <p style="margin-left:0;text-align:left;">
     比如如果在选择主体的时候如果你是需要商业用途的小程序，在选择主体类型的时候选择：“企业”，企业类型选择：“个体工商户”，然后填写企业名称和营业执照。注册方式选择：“微信认证”。进入小程序后台根据自己需要设置一下小程序就可以了。
    </p>
    <p style="margin-left:0;text-align:left;">
     完成注册后，回到小程序第三方工具的编辑器，点击授权微信小程序账号即可。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       tips：如果你是已认证的企业类型公众号，直接登录自己的公众账号，进入小程序管理的“快速注册并认证小程序”就行了。
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       3，生成小程序
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     如果你想快速上线发布一个自己的小程序，那么使用第三方开发工具来做比较方便。还是以商用的“xx线了”电商模板为例，先选择模板，进入到“xx线了”后台，选择创建小程序里的电商，然后再选中一个你喜欢的模板直接导入就行了，选择好模板后，就可以添加商品、装修店面了。最后一个程序就是分页设置，添加各种板块的快捷方式，就完成了一个成熟小程序的大体了。
    </p>

     2.第二步上面设计好页面风格以后转到小程序发布页面，发布体验版本，然后直接小程序扫码就可以体验小程序了

    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       二、Saas模板开发；
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       1，saas是什么
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     小程序saas模板现在是非常常见的一种通过互联网提供软件服务的模式，用户不用购买软件，而是向提供商租用软件，进行小程序的编辑，我们可以看出，企业使用saas版的小程序，是无需购买服务器的，当然，其数据都储存在小程序提供方的服务器上。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       举个
      </strong>
     </strong>
     <strong>
      <strong>
       梨子
      </strong>
     </strong>
     <strong>
      <strong>
       ：
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     如果把小程序制作比喻成PPT制作，小程序的saas就像是有的PPT软件会提供做好的PPT模板，你只需要选择想要的款式，然后直接把内容套进去就行。
    </p>
    <p style="margin-left:0;text-align:left;">
     但是在使用的时候你要注意，你有对它的使用权，可以添加你想添加的内容或者图案，但是你无法对它本有的一些内容格式进行更改，因为它的格式在后台是定死了的。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       2，租用saas模板会遇到的问题
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     没错saas模板都是租用的，正是因为是租用，saas模板的最终解释权都是在供应商的手里，所以这里面有很深的水，作为消费者和购买者，我们一定要小心的避开其中经常会踩到的雷，不要被供应商的小优小惠给迷惑了，否则到了后期需要进一步深化自己的小程序的时候，才发现原来自己的小程序用的是商家给的模板，进入进退维艰的地步。下面我给大家列举几种，我们经常会遇到的一些雷点：
    </p>
    <ul>
     <li style="text-align:justify;">
      <strong>
       <strong>
        雷点1：收费混乱，被商家牵着鼻子走
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:0;text-align:left;">
     对于大多数人来说，怕就怕商家不但骗你用的是模板，还给你来一大堆模模糊糊的收费规则。往往saas模板都是按年收费的，所以很多商家会在第一时间告诉你，这个免费用，但这个免费，通常都没有说完，一般都只是指免费注册和免费试用，就是第一年免费，第二年之后就要开始收取高昂的租金了，那个时候你已经把小程序用的比较熟了，依赖性也比较强，你这个时候不给钱也没办法。
    </p>
    <p style="margin-left:0;text-align:left;">
     其次，他还会有很多二次收费，或者是隐藏收费。我之前遇到的一个朋友购买的saas模板，说好的是多商户入驻，但是当人数越来越多的时候，供应商就开始限制商户入住，并要收取人头费，这种霸王条款的强制性收费给他带来了很大的困扰，相当影响企业后期发展，大家注意小心甄别。
    </p>
    <ul>
     <li style="text-align:justify;">
      <strong>
       <strong>
        雷点2：无法升级也无法维护自己顾客的数据
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:0;text-align:left;">
     之前提到了saas模板因为是采用的供应商的模板，所以数据是定死的，你无法进行过多的更改。
    </p>
    <p style="margin-left:0;text-align:left;">
     此外，除了无法升级功能以外，还有一个重要的地方，就是无法打通数据库。在前期你积累的所有客户的数据，想要进行进一步的更新的时候，你会发现你无法获取这些数据，因为这些数据都保存在供应商的模板当中，你没有权限去访问和获取，所以一旦你需要给客户提供更好的体验申请的模板，或者是想要制作APP的时候，这些限制将会极大遏制你对客户的服务体验。
    </p>
    <ul>
     <li style="text-align:justify;">
      <strong>
       <strong>
        雷点3：供应商跑路，购买商急
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:0;text-align:left;">
     正是因为这些数据都掌握在供应商手里，所以当供应商的服务出现问题，不能给你提供较好的模板服务，或者由于经营不善倒闭的时候，你的公司也会受到很大牵连，那个时候你想换一个公司，或者想导出自己的数据，你基本就是啥也没有了，那个时候你应该怎么办呢？————你品仔细品。
    </p>
    <ul>
     <li style="text-align:justify;">
      <strong>
       <strong>
        雷点4：成交佣金和犯规罚款。
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:0;text-align:left;">
     没错，供应商还会以各种理由强制收取你在小程序上的成交额，并美其名曰佣金抽成。也就是说你赚的每一笔钱，每一单交易都是在为了供应商打工，这种模板套路相当常见，但是很多人也都是在之后才追悔莫及。
    </p>
    <p style="margin-left:0;text-align:left;">
     有的供应商还会先给你免费用小程序，等你把小程序的内容和资料完善之后，开始处处对你的小程序设置限制，甚至是封禁，有的会告诉你说“你的内容违规！”或者一些内容要会员才可开通，这个时候你就只能在乖乖的交钱花钱买平安。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       三，源码开发小程序
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     完全不同于saas的模板化开发源码，开发小程序是自己掌握数据库和核心的开发代码，是很多大型企业会考虑选择的一种开发方式。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       1，源码开发小程序是什么
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     小程序源码是小程序的程序代码，技术人员开发并添加功能等等。是制作小程序的源码，利用一些开源的源代码进行升级开发。如果你懂一点代码知识，会 JS 的话，就可以尝试自己下载官方小程序开发工具，然后写代码开发。
    </p>
    <p style="margin-left:0;text-align:left;">
     源码系统小程序是技术公司程序员从零开始搭建的系统，所以是有自主知识产权的。并且源码系统也可以开发许多saas系统的小程序。所以很多卖模板小程序的公司，不一定能卖源码系统，因为不知道模板中间倒卖过多少次了。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       2，遇到的问题
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     版权问题是源码开发过程当中，作为常见的一类问题，上面我们讲过了，有些小作坊公司会从中作梗，通过网上下载的源码直接售卖，甚至有的人直接去破解源码，这样一来，你自己的源码就很可能是盗版，自主性也无法得到保证。
    </p>
    <p style="margin-left:0;text-align:left;">
     随之而来的就是版权纠纷和源码使用当中存在的bug，这种bug通常会导致你的小程序无法升级或是数据搜集出问题，在后期的维护和处理当中，会给你增添许多沉没成本，同时也会有很大的法律纠纷风险。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       四，定制小程序
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       1，定制小程序是什么
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     首先，定制小程序服务由于是完全根据企业的需求来进行定制开发的，所以其中的功能、页面可以任意增加、删除，修改。完全按照自己的需求来更改。
    </p>
    <p style="margin-left:0;text-align:left;">
     同时定制小程序使得服务器也是独立的，源代码完全自己把握，所以拥有较高的数据安全性。也就不用担心后期的维护问题和版权纠纷问题。
    </p>
    <p style="margin-left:0;text-align:left;">
     此外企业也无需再如模板小程序一样长期续费，更加方便。
    </p>
    <p style="margin-left:0;text-align:left;">
     所以正是因为安全、稳定，功能性强，并且量身定制，哪怕收费周期和收费高一些，但是定制可以完美切合需求，所以在市场上还是受到普遍推崇。
    </p>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <strong>
       2，定制遇到的问题
      </strong>
     </strong>
    </p>
    <p style="margin-left:0;text-align:left;">
     定制有一些对于企业来说必要的优点，但是也不可忽视其中存在的一些弊端要注意：
    </p>
    <ul>
     <li style="text-align:justify;">
      首先，想更改功能要通过第三方团队，并且每次的更改都会有成本压力。
     </li>
     <li style="text-align:justify;">
      其次，如果你找的定制开发团队不靠谱，那么就白白浪费大量时间，无法保证小程序的质量。
     </li>
     <li style="text-align:justify;">
      最后定制小程序一定要谨慎。在双方交流的时候，要先把需求说清楚，尽量减少给企业带来的损失。并且找定制平台时要注意选择有可靠的技术和拥有丰富经验的团队，保证业界口碑好、从业经验丰富。
     </li>
    </ul>
