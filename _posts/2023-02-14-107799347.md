    <div style="font-size: 16px;">
     <p>
      python列表中随机选择
     </p>
     <p>
      Python
      <b>
       random module
      </b>
      provides an inbuilt method
      <b>
       choice()
      </b>
      has an ability to select a random item from the list and other sequences. Using the
      <b>
       choice()
      </b>
      method, either a single random item can be chosen or multiple items. The below set of examples illustrate the behavior of
      <b>
       choice()
      </b>
      method.
     </p>
     <p>
      Python
      <b class="raw_b_node">
       随机模块
      </b>
      提供了一种内置方法
      <b class="raw_b_node">
       choice()
      </b>
      ，可以从列表和其他序列中选择一个随机项。 使用
      <b class="raw_b_node">
       choice()
      </b>
      方法，可以
      <b class="raw_b_node">
       选择
      </b>
      一个随机项目或多个项目。 下面的示例集说明了
      <b class="raw_b_node">
       choice()
      </b>
      方法的行为。
     </p>
     <p>
      <b>
       Syntax:
      </b>
     </p>
     <p>
      <b class="raw_b_node">
       句法：
      </b>
     </p>
     <pre><code class="has">    random.choice(sequence) 
</code>
</pre>
     <p>
      Here,
      <span class="w3-codespan">
       sequence
      </span>
      can be a list, set, dictionary, string or tuple.
     </p>
     <p>
      在这里， 序列可以是列表，集合，字典，字符串或元组。
     </p>
     <p>
      <b>
       Example 1: Choose a single item randomly from the list
      </b>
     </p>
     <p>
      <b class="raw_b_node">
       示例1：从列表中随机选择一个项目
      </b>
     </p>
     <pre><code class="has">&gt;&gt;&gt; import random
&gt;&gt;&gt; test_list = ['include_help', 'wikipedia', 'google']
&gt;&gt;&gt; print(random.choice(test_list))
wikipedia
&gt;&gt;&gt; print(random.choice(test_list))
google
&gt;&gt;&gt; print(random.choice(test_list))
wikipedia
&gt;&gt;&gt;
</code>
</pre>
     <p>
      <b>
       Example 2: Choose multiple items randomly from the list
      </b>
     </p>
     <p>
      <b class="raw_b_node">
       示例2：从列表中随机选择多个项目
      </b>
     </p>
     <p>
      In order to select multiple items from list, random module provides a method called choices.
     </p>
     <p>
      为了从列表中选择多个项目，随机模块提供了一种称为choices的方法。
     </p>
     <pre><code class="has">&gt;&gt;&gt; import random
&gt;&gt;&gt; print(random.choices(test_list, k=2))
['wikipedia', 'include_help']
&gt;&gt;&gt; print(random.choices(test_list, k=1))
['google']
&gt;&gt;&gt; print(random.choices(test_list, k=3))
['google', 'google', 'google']
&gt;&gt;&gt;
</code>

</pre>
