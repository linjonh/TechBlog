    <h2>
     <a id="_0">
     </a>
     前言
    </h2>
    <p>
     💻随着大数据、人工智能的发展，人们步入了新的时代，逐渐走上科技的巅峰。
    </p>
    <p>
     ⚔科技是一把双刃剑，网络安全不容忽视，人们的
     <code>
      隐私
     </code>
     在大数据面前暴露无遗，账户被盗、资金损失、网络诈骗、隐私泄露，种种迹象表明，随着互联网的发展，
     <code>
      网络安全
     </code>
     需要引起人们的重视。
    </p>
    <p>
     💂互联网安全从其本质上来讲就是互联网上的
     <code>
      信息安全
     </code>
     。从广义来说，凡是涉及到互联网上
     <code>
      信息的保密性、完整性、可用性、真实性和可控性
     </code>
     的相关技术和理论都是网络安全的研究领域。
    </p>
    <p>
     👨‍👨‍👧‍👦网络安全需要一群
     <code>
      网络安全技术人员
     </code>
     的维护。而CTF，就是这些人
     <code>
      技术竞技
     </code>
     的比赛。
     <code>
      网络安全大赛
     </code>
     或许听上去很熟悉，它到底是什么呢？
    </p>
    <h2>
     <a id="CTF_11">
     </a>
     CTF概况
    </h2>
    <h3>
     <a id="CTF_14">
     </a>
     CTF简介
    </h3>
    <p>
     📚CTF（Capture The Flag），中文名
     <code>
      夺旗赛
     </code>
     。
    </p>
    <p>
     🆚网络安全人员之间进行竞技的一种比赛。
    </p>
    <h3>
     <a id="CTF_21">
     </a>
     CTF的含义
    </h3>
    <p>
     🔤CTF的英文名可以直接翻译为
     <code>
      夺得Flag
     </code>
     。
    </p>
    <p>
     💯参赛团队之间通过进行攻防对抗等形式率先从主办方给出的比赛环境中得到一串
     <code>
      具有一定格式的字符串或其它内容
     </code>
     ，并提交给主办方，从而夺取分数。
    </p>
    <p>
     🚩为了方便称呼，将需要夺得的内容称为
     <code>
      Flag
     </code>
     。
    </p>
    <h3>
     <a id="CTF_30">
     </a>
     CTF的发展历史
    </h3>
    <h4>
     <a id="CTF_33">
     </a>
     CTF的起源
    </h4>
    <p>
     💻CTF起源于1996年DEFCON全球黑客大会，以代替之前黑客们通过互相发起真实攻击进行技术比拼的方式。发展至今
    </p>
    <p>
     🆚1996年第四届DEF CON官方举办了网络技术比拼。
    </p>
    <h4>
     <a id="CTF_39">
     </a>
     早期的CTF
    </h4>
    <p>
     早期的CTF：
    </p>
    <ul>
     <li>
      没有明确的比赛规则
     </li>
     <li>
      没有专业搭建的比赛平台和环境
     </li>
    </ul>
    <p>
     而是：
    </p>
    <ul>
     <li>
      参赛队伍自己准备比赛目标
     </li>
     <li>
      组织者是非专业的志愿者
     </li>
     <li>
      参加者需要接受参赛队伍手动计分的规则
     </li>
    </ul>
    <h4>
     <a id="CTF_52">
     </a>
     现代CTF竞赛
    </h4>
    <p>
     现在的CTF比赛一般由
     <code>
      专业队伍
     </code>
     承担比赛平台、命题、赛事组织以及拥有
     <code>
      自动化积分系统
     </code>
     。
    </p>
    <p>
     参赛队伍需提交参赛申请，并且由DEF CON会议组织者们进行评选。
    </p>
    <p>
     比赛侧重于对
     <code>
      计算机底层和系统安全
     </code>
     的核心能力。
    </p>
    <h3>
     <a id="CTF_60">
     </a>
     CTF的比赛赛制
    </h3>
    <h4>
     <a id="Jeopardy_63">
     </a>
     解题模式（Jeopardy）
    </h4>
    <p>
     在解题模式的CTF赛制中，参赛队伍可以通过
     <code>
      互联网或者现场网络
     </code>
     参与，这种模式的CTF竞赛与
     <code>
      ACM编程竞赛、信息学奥赛
     </code>
     比较类似，以解决网络安全技术挑战题目的
     <code>
      分值和时间
     </code>
     来排名，通常用于在线选拔赛。
    </p>
    <h4>
     <a id="AttackDefense_67">
     </a>
     攻防模式（Attack-Defense）
    </h4>
    <p>
     在攻防模式CTF赛制中，参赛队伍在网络空间互相进行
     <code>
      攻击和防守
     </code>
     ，挖掘网络服务漏洞并
     <code>
      攻击
     </code>
     对手服务来得分，修补自身服务漏洞进行
     <code>
      防御
     </code>
     来避免丢分。此模式CTF赛制是一种竞争激烈，具有很强
     <code>
      观赏性
     </code>
     和高度
     <code>
      透明性
     </code>
     的网络安全赛制。
    </p>
    <h4>
     <a id="Mix_71">
     </a>
     混合模式（Mix）
    </h4>
    <p>
     结合了
     <code>
      解题模式
     </code>
     和
     <code>
      攻防模式
     </code>
     的CTF赛制，比如参赛队伍通过解题可以获取一些初始分数，然后通过攻防对抗进行得分增减的零和游戏，最终以得分高低分出胜负。采用混合模式CTF赛制的典型代表如iCTF国际CTF竞赛。
    </p>
    <h3>
     <a id="CTF_75">
     </a>
     CTF著名赛事
    </h3>
    <p>
     由于互联网的兴盛，网络安全也越来越重要，网络安全人才也越来越受到重视。为了选拔人才，国际上都会举行大赛来选拔人才。
    </p>
    <p>
     著名赛事：
    </p>
    <ul>
     <li>
      DEF CON CTF：DEF CON作为
      <code>
       CTF赛制的发源地
      </code>
      ，DEF CON CTF也成为了目前拥有全球最高技术水平和影响力的CTF竞赛，相当于
      <code>
       CTF赛事之中的“世界杯”
      </code>
      。考验参赛团队在
      <code>
       逆向分析、漏洞挖掘、漏洞利用、漏洞修补加固
      </code>
      等方面的综合能力。
     </li>
     <li>
      百度杯CTF夺旗大战：由百度安全应急响应中心和i春秋联合举办的CTF比赛，国内现今为止首次历时最长（半年）、频次最高的CTF大赛。赛题丰富且突破了技术和网络的限制。
     </li>
     <li>
      RuCTF：RuCTF是由俄罗斯Hackerdom组织一年一度的国家级竞赛，解题模式资格面向全球竞赛，解题攻防混合模式的决赛面向俄罗斯队伍的国家级竞赛。比赛按照经典的攻击/防御CTF规则进行。
     </li>
    </ul>
    <h3>
     <a id="CTF_86">
     </a>
     CTF的意义
    </h3>
    <p>
     💻⽐赛形式与内容拥有浓厚的
     <code>
      ⿊客精神
     </code>
     和
     <code>
      ⿊客⽂化
     </code>
     。
    </p>
    <p>
     🛠近年来，CTF已经成为了学习锻炼
     <code>
      信息安全技术
     </code>
     ，展现
     <code>
      安全能⼒和⽔平
     </code>
     的绝佳平台。
    </p>
