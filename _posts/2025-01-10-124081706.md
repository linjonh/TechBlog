    <h3>
     <a id="_3">
     </a>
     一、桌面操作系统
    </h3>
    <h4>
     <a id="1_5">
     </a>
     1、基本开发工具
    </h4>
    <h5>
     <a id="_7">
     </a>
     更新软件源
    </h5>
    <p>
     <code>
      apt update
     </code>
    </p>
    <h5>
     <a id="11__11">
     </a>
     1.1 安装编译所需软件包
    </h5>
    <p>
     <code>
      apt install build-essential
     </code>
    </p>
    <p>
     通常会安装
     <code>
      gcc, g++, make
     </code>
     等软件及开发依赖的库
    </p>
    <h5>
     <a id="12__QT5_17">
     </a>
     1.2 安装QT5
    </h5>
    <p>
     <code>
      apt install qtcreator qtbase5-dev -y
     </code>
    </p>
    <p>
     qt配置:
    </p>
    <pre><code>cd /usr/lib/loongarch64-linux-gnu/qtchooser
cp qt5.conf default.conf
</code></pre>
    <h5>
     <a id="13_JAVA_28">
     </a>
     1.3 安装JAVA
    </h5>
    <p>
     目前龙芯平台支持的java 环境有
     <code>
      openjdk-8, openjdk-11, openjdk-17
     </code>
    </p>
    <p>
     <strong>
      java8:
     </strong>
    </p>
    <p>
     <code>
      apt install openjdk-8-jdk
     </code>
    </p>
    <p>
     <strong>
      java11:
     </strong>
    </p>
    <p>
     <code>
      apt install openjdk-8-jdk
     </code>
    </p>
    <p>
     <strong>
      java 17:
     </strong>
    </p>
    <p>
     从http://www.loongnix.cn/zh/api/java/ 中下载
     <code>
      java 17
     </code>
     软件包
    </p>
    <p>
     将压缩包解压到/opt
    </p>
    <p>
     <code>
      sudo tar xf loongson17.1.0-jdk17_35-linux-loongarch64.tar.gz -C /opt
     </code>
    </p>
    <p>
     配置环境变量：
    </p>
    <p>
     <code>
      vim /etc/profile
     </code>
    </p>
    <pre><code>JAVA_HOME=/opt/jdk-17  (jdk包解压路径)

CLASSPATH=$JAVA_HOME/lib

PATH=$JAVA_HOME/bin:$PATH
</code></pre>
    <p>
     添加以上三行后执行
     <code>
      source /etc/profile
     </code>
     使配置文件生效
    </p>
    <h5>
     <a id="14_Go_62">
     </a>
     1.4 安装Go
    </h5>
    <p>
     目前go 支持两个版 go-1.15 和 go-1.18， 默认安装的是go-1.15
    </p>
    <p>
     <code>
      apt install golang
     </code>
    </p>
    <p>
     <strong>
      go-1.18
     </strong>
    </p>
    <p>
     <code>
      apt install golang-1.18
     </code>
    </p>
    <h5>
     <a id="15_pyhton3_74">
     </a>
     1.5 安装pyhton3
    </h5>
    <p>
     <code>
      apt install python3
     </code>
    </p>
    <h5>
     <a id="16_rust_78">
     </a>
     1.6 安装rust
    </h5>
    <p>
     rust 支持 rust-1.41 和 rust-1.57
    </p>
    <p>
     <code>
      apt install rustc
     </code>
    </p>
    <h4>
     <a id="2IDE_84">
     </a>
     2、常用IDE
    </h4>
    <p>
     获取方式：操作系统的应用商店或应用软件官网
    </p>
    <p>
     IDEA 全家桶（IDEA, GoLand, PyCharm）是基于java 开发的，都可以直接从软件官网下载安装（需要系统安装java 11）
    </p>
    <p>
     其他主流的IDE 有VSCode, Eclipse 等可以通过应用商店安装。
    </p>

yum groupinstall "Development Tools"
