     目的：
     <br/>
     通过综合性实验的设计，使学生综合训练自己的需求分析、系统设计、编程、测试等各种能力，积累软件开发的经验，提高学生进行软件开发所需具备的专业素质以及团队协作能力。
     <br/>
     要求：
     <br/>
     1.编写一个音视频播放器，实现流畅的播放音频、视频；包括添加多个播放文件、播放、暂停、控制进度，播放列表，自定义添加歌词等功能；
     <br/>
     3.实现友好的UI设计，增强用户体验；
    </p>
    <p>
     该实验在HTML5中调用了Audio标签与Video标签对视频与音乐进行的设计
    </p>
    <p>
     效果图为：
     <br/>
     音乐播放器：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5b301b5a0f404c13051ee2b85a5a8911.png#pic_center">
      <br/>
      视频页面：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/643236e441862316c373598cff53b714.png#pic_center">
       <br/>
       文件类型：
       <br/>
       视频文件类型为MP4文件；
       <br/>
       音乐文件类型为MP3文件。
       <br/>
       模块描述
       <br/>
       视频模块：
       <br/>
       1)功能：用户选择视频文件并播放，也可以跳转到音乐播放器页面。
       <br/>
       2)接口：用户可以通过点击音乐按钮跳转到音乐页面。
       <br/>
       音乐模块：
       <br/>
       1)功能：用户选择音乐文件并播放，也可以跳转到视频播放器页面。
       <br/>
       2)接口：用户可以通过点击视频按钮跳转到音乐页面。
       <br/>
       关键代码：
      </img>

    <pre><code>视频播放器HTML代码：
&lt;!DOCTYPE html&gt;
&lt;html&gt;
    &lt;head&gt;
        &lt;meta charset="UTF-8"/&gt;
        &lt;title&gt;视频播放器&lt;/title&gt;
&lt;!--连接css文件--&gt;
        &lt;link rel="stylesheet" href="Video.css"/&gt;
    &lt;/head&gt;
    &lt;body&gt;
    &lt;div class="view"&gt;
      &lt;div class="left-side"&gt;&lt;!-- 创建左边列表--&gt;
        &lt;div class="control"&gt;
          &lt;div class="o-btns"&gt;&lt;!--控制按钮--&gt;
           &lt;a href="FileAudio.html"&gt;&lt;button id="audio"&gt;切换音频&lt;/button&gt;&lt;/a&gt;
             &lt;button class="add-video"&gt;本地视频&lt;/button&gt;
&lt;input type="file" class="video-file" id="file" accept="video/mp4"
multiple="true"/&gt;
           &lt;/div&gt;
         &lt;/div&gt;
		&lt;!--创建视频列表--&gt;
        &lt;div class="video-contain"&gt;
           &lt;ul class="video-list"&gt;&lt;/ul&gt;
         &lt;/div&gt;
       &lt;/div&gt;
			&lt;!--创建视频控制控件--&gt;
     &lt;video  preload="auto"	 style="width: 960px;height: 660px; margin-left: 450px;margin-top:50px"	controls="controls" &gt;&lt;/video&gt;
        &lt;/div&gt;
		&lt;!--连接js文件--&gt;
        &lt;script src="jquery-3.6.0.min.js"&gt;&lt;/script&gt;
        &lt;script src="video_main.js"&gt;&lt;/script&gt;
    &lt;/body&gt;
&lt;/html&gt;
