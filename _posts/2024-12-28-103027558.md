     <br/>

     <br/>
     二、键盘操作在UiPath中的使用
     <br/>
     1.打开设计器，在设计库中新建一个Sequence，为序列命名及设置Sequence存放的路径
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a39d2e7b6b026995414429dffeec447d.png">
      <br/>
      2.在Activities中搜索open browser，并将其拖至设计区，且设置打开网站，运行该流程“https://www.baidu.com/”
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/022209a9b28c1e76a1169501e5b311ad.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f235a2ad98c3160ac79ceab822240164.png">
        <br/>
        3.在Activities中搜索 输入信息 （Typeinto），并将其拖至设计区，拾取百度搜索的输入框且设置键盘输入的文本为"机器人"
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e3476bbd106a5ed20c74ef22502a66ae.png">
         <br/>
         #拾取区域，红色框选区域
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a849dae76cac2e949b0316af359dc02a.png">
          <br/>
          4.在Activities中搜索 发送热键 （Sendhotkey），并将其拖至设计区，选择enter按键
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/932f04f9f43955fc37b022fa3d8cce4a.png">
