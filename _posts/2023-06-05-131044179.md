    <h2>
     <a id="_1">
     </a>
     本文简介
    </h2>
    <p>
     博主在学习到双变量数学归纳法时候发现网上很难找到清晰明白的教程，就把自己学习和理解的一些知识和大家分享一下。博主不是数学专业，讲解也是从个人理解出发，如有错误还请各位批评指正。
    </p>
    <p>
     本文从初高中学的数学归纳法（单变量弱归纳）开始，复习数学归纳法基本术语和步骤，然后过渡到强归纳法，用一个例子来说明强归纳和弱归纳的区别，然后再推进到两个变量甚至多变量的数学归纳法。基础较好的同学建议直接跳到双变量部分。
    </p>
    <hr/>

    </p>
    <h2>
     <a id="_induction_6">
     </a>
     单变量数学归纳 induction
    </h2>
    <p>
     在讲双变量数学归纳之前，我们先来复习一下对于单变量的数学归纳法。
    </p>
    <blockquote>
     <p>
      数学归纳法（Mathematical Induction, MI）是一种数学证明方法，通常被用于证明某个给定命题在整个（或者局部）自然数范围内成立。
      <a href="#refer-anchor-1" rel="nofollow">
       <sup>
        1
       </sup>
      </a>
     </p>
    </blockquote>
    <p>
     数学归纳法又可以分为
     <strong>
      弱归纳法
     </strong>
     （也叫普通数学归纳法，第一数学归纳法）和
     <strong>
      强归纳法
     </strong>
     （也叫第二数学归纳法）。他们的区别在于对归纳假设的不同：
    </p>
    <blockquote>
     <p>
      <strong>
       弱归纳法假设：
       <span class="katex--inline">
        <span class="katex">
         <span class="katex-mathml">
          n 
          
         
           = 
          
         
           k 
          
         
        
          n=k
         </span>
         <span class="katex-html">
          <span class="base">
           <span class="strut" style="height: 0.4306em;">
           </span>
           <span class="mord mathnormal">
            n
           </span>
           <span class="mspace" style="margin-right: 0.2778em;">
           </span>
           <span class="mrel">
            =
           </span>
           <span class="mspace" style="margin-right: 0.2778em;">
           </span>
          </span>
          <span class="base">
           <span class="strut" style="height: 0.6944em;">
           </span>
           <span class="mord mathnormal" style="margin-right: 0.0315em;">
            k
           </span>
          </span>
         </span>
        </span>
       </span>
       时命题成立
      </strong>
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       强归纳法假设：
       <span class="katex--inline">
        <span class="katex">
         <span class="katex-mathml">
          n 
          
         
           &lt;

           k 
          
         
        
          n&lt;=k
         </span>
         <span class="katex-html">
          <span class="base">

           </span>
           <span class="mspace" style="margin-right: 0.2778em;">
           </span>
           <span class="mrel">
            &lt;=
           </span>
           <span class="mspace" style="margin-right: 0.2778em;">
           </span>
          </span>
          <span class="base">
           <span class="strut" style="height: 0.6944em;">

          </span>
         </span>
        </span>
       </span>
       时命题成立
      </strong>
     </p>
    </blockquote>
