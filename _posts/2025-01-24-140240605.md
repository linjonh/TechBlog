     <p>
      <strong>
       <strong>
       </strong>
       JetLinks 物联网平台
      </strong>
     </p>
     <p style="text-align:center;">
      <img alt="b56d759610e6d40ffb9f93062edd4789.png" src="https://i-blog.csdnimg.cn/blog_migrate/e262182e286664eb02421389c4e9076c.png"/>
     </p>
     <p style="text-align:left;">
      <strong>
       JetLinks
      </strong>
      是一个基于Java
      <strong>
       国产
      </strong>
      <strong>
       开源的企业级的物联网平台
      </strong>
      ，它集成了设备管理、数据安全通信、消息订阅、规则引擎等一系列物联网核心能力，支持以平台适配设备的方式连接海量设备，采集设备数据上云，提供云端API，通过调用云端API实现远程控制。JetLinks物联网平台还支持多种设备接入协议，并提供了丰富的协议库。
     </p>
     <p style="text-align:center;">
      <img alt="cfdac29431a9ae2c546635d4396d077a.png" src="https://i-blog.csdnimg.cn/blog_migrate/9a187c6aff9460749cf0ccbe9454b0eb.png"/>
     </p>
     <p>
      JetLinks 基于Java8，Spring-boot 2.7，Webflux，Netty，Vert.x等开发， 是一个全响应式(Reactor)的物联网基础平台。使用 Maven多模块，实现组件化开发，可按需引入需要的模块。
     </p>
     <p>
      <strong>
       性能优势
      </strong>
     </p>
     <p style="text-align:left;">
      JetLinks支持私有化或容器化以单机、集群或微服务的方式部署在任意平台服务器上，支持横向拓展，具有极佳的性能表现。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         <strong>
          千万级设备连接
         </strong>
        </strong>
       </p>
      </li>
     </ul>
     <p>
      架构水平扩容，支持千万级设备连接。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         <strong>
          百万级数据并发
         </strong>
        </strong>
       </p>
      </li>
     </ul>
     <p>
      架构水平扩容，支持百万级数据并发。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         <strong>
          10w+任务调度处理
         </strong>
        </strong>
       </p>
      </li>
     </ul>
     <p>
      自研RactorQL实现规则引擎任务处理，具有极佳的处理性能。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         <strong>
          ＜30ms设备延迟
         </strong>
        </strong>
       </p>
      </li>
     </ul>
     <p>
      全响应式非阻塞网络编程，具有极低的设备延迟。
     </p>
     <p>
      <strong>
       核心特性
      </strong>
     </p>
     <ul>
      <li>
       <p>
        完全开源(社区版免费),基于事件驱动,拓展性强.
       </p>
      </li>
      <li>
       <p>
        支持统一物模型管理,多种设备,多种厂家,多种报文,统一管理。
       </p>
      </li>
      <li>
       <p>
        统一设备连接管理,多协议适配(TCP,MQTT,UDP,CoAP,HTTP等),屏蔽网络编程复杂性,灵活接入不同厂家不同协议的设备。
       </p>
      </li>
      <li>
       <p>
        灵活的规则引擎,设备告警,消息通知,数据转发,场景联动。
       </p>
      </li>
      <li>
       <p>
        强大的ReactorQL引擎,使用SQL来处理实时数据.可拓展自定义函数。
       </p>
      </li>
      <li>
       <p>
        地理位置:统一管理地理位置信息,支持区域搜索。
       </p>
      </li>
      <li>
       <p>
        灵活的非侵入多租户数据权限控制
       </p>
      </li>
      <li>
       <p>
        支持灵活的设备数据存储策略，可将不同类型的设备数据存储到不同的地方。
       </p>
      </li>
      <li>
       <p>
        自研hsweb、easyorm、reactorQL、规则引擎、消息总线、注册中心等物联网平台核心组件。
       </p>
      </li>
      <li>
       <p>
        基于自研组件与开放协议，可快速二次开发，适配各行业物联网业务。
       </p>
      </li>
      <li>
       <p>
        支持InFluxDB、ClickHouse、Elasticsearch、Cassandra、TDengine五种时序数据库。
       </p>
      </li>
      <li>
       <p>
        全响应式企业级物联网平台，面向数据流和变化传递，更优雅的多线程和异步编程方式。
       </p>
      </li>
      <li>
       <p>
        支持可视化规则引擎编辑器，实现拖拉拽的方式处理数据输入、清洗、计算、输出、推送、存储。
       </p>
      </li>
      <li>
       <p>
        支持鲲鹏等国产化服务器部署。
       </p>
      </li>
      <li>
       <p>
        水平扩容架构，支持集群微服务部署。
       </p>
      </li>
     </ul>
     <p style="text-align:left;">
      <strong>

     <p>
      <strong>
       <strong>
        <strong>
        </strong>
        JetLinks 产品功能和架构
       </strong>
      </strong>
     </p>
     <p style="text-align:center;">
      <img alt="328e19fab0271a526741dc38497de050.png" src="https://i-blog.csdnimg.cn/blog_migrate/34bf2ec29f7f437dfc0a0247f531d103.png"/>
     </p>
     <p>
      ‍Jetlinks 的定位为 Paas 服务的物联网平台，降低物联网企业研发、运营和运维成本，提高物联网项目的落地能力。
     </p>
     <p>
      <strong>
       功能清单
      </strong>
     </p>
     <ul>
      <li>
       <p>
        <strong>
         设备接入
        </strong>
        <br/>
       </p>
      </li>
     </ul>
     <p>
      多元化设备接入方式，提供基于网络协议、云平台、边缘网关、视频通道等多种设备接入方式。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         数据处理
        </strong>
       </p>
      </li>
     </ul>
     <p>
      提供基于ReactorQL、自定义脚本计算、规则引擎等强大的数据处理功能。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         数据储存
        </strong>
       </p>
      </li>
     </ul>
     <p>
      平台支持Elasticsearch、ClickHouse、InfluxDB、TDengine、Cassandra等数据存储中间件，可自定义配置。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         应用集成
        </strong>
       </p>
      </li>
     </ul>
     <p>
      平台提供所有功能API接口，第三方应用可轻松获取平台数据；同时支持Oauth2单点登录。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         规则引擎
        </strong>
       </p>
      </li>
     </ul>
     <p>
      提供基于可视化规则编辑器，以拖拉拽的方式处理数据输入、清洗、计算、输出、存储等业务。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         通知管理
        </strong>
       </p>
      </li>
     </ul>
     <p>
      平台支持钉钉、微信、邮箱、短信、webhook、语音6大类通知方式。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         告警中心
        </strong>
       </p>
      </li>
     </ul>
     <p>
      支持设备和定时维度实时触发告警规则。支持告警推送、告警阈值、告警解除、告警限频等功能。
     </p>
     <ul>
      <li>
       <p>
        <strong>
         视频管理
        </strong>
       </p>
      </li>
     </ul>
     <p>
      平台基于GB/T28181/2016协议支持多路视频直播、回放、录制、云台控制、国标级联。
     </p>

      <strong>
       <strong>
        <strong>
         <strong>
          JetLinks
         </strong>
        </strong>
        开发实战
       </strong>
      </strong>
     </p>
     <p style="text-align:center;">
      <img alt="5e4c879def66112700e3c62b7a77ba87.png" src="https://i-blog.csdnimg.cn/blog_migrate/4e898a8cb5f5d9cd49fd621a9b0dbe7e.png"/>
     </p>
     <p>
      <strong>
       设备接入流程
      </strong>
     </p>
     <p style="text-align:center;">
      <img alt="6d702db32686f484627c844a426851af.png" src="https://i-blog.csdnimg.cn/blog_migrate/6bc97650b5654c684643017a075cf82f.png"/>
     </p>
     <p>
      <strong>
       产品管理
      </strong>
     </p>
     <p style="text-align:left;">
      产品管理对具有相同功能的设备模型进行统一的维护管理，包括产品的基本信息、物模型、设备接入等功能。
     </p>
     <p style="text-align:center;">
      <img alt="29c9508cc42d9d1a5688da45a6700987.png" src="https://i-blog.csdnimg.cn/blog_migrate/9e3d99e833c0a94a3d6882e395930fd5.png"/>
     </p>
     <p>
      <strong>
       设备管理
      </strong>
     </p>
     <p style="text-align:left;">
      设备管理可对设备实例进行全生命周期管理，包括设备基本信息、物模型、运行状态、设备功能、日志管理、设备诊断、数据解析、点位绑定等管理功能。
     </p>
     <p style="text-align:center;">
      <img alt="303db6c80a6a971790ca550709c39255.png" src="https://i-blog.csdnimg.cn/blog_migrate/b8a2470b4641935c72ec3619ffec152b.png"/>
     </p>
     <p style="text-align:center;">
      <img alt="685563f9b5dcaf99350a6a057aef0833.png" src="https://i-blog.csdnimg.cn/blog_migrate/86d552af8e841fbb5f1fe6b5e94f27d1.png"/>
     </p>
     <p>
      <strong>
       网关接入
      </strong>
     </p>
     <p style="text-align:left;">
      设备接入网关聚合了设备接入Jetlinks物联网平台的所需的相关配置信息，支持自定义接入、视频类设备接入、云平台接入、通道类设备接入、官方接入等方式。
     </p>
     <p style="text-align:center;">
      <img alt="f2c3a7ab00043354134401135a90827c.png" src="https://i-blog.csdnimg.cn/blog_migrate/d58aae4d5281ab854fe200bbc61562b2.png"/>
     </p>
     <p>
      <strong>
       规则引擎
      </strong>
     </p>
     <p style="text-align:left;">
      规则引擎前端编辑界面基于node-red，后端使用纯Java实现。灵活的规则模型配置，支持多种规则模型以及自定义规则模型。
      <strong>
       设备告警，场景联动
      </strong>
      均由统一的规则引擎管理。
     </p>
     <p style="text-align:center;">
      <img alt="587f31c3967ecf2bb02bdc5658c8eb39.png" src="https://i-blog.csdnimg.cn/blog_migrate/452f85d292bf5524c5400ed2ca374766.png"/>
     </p>
     <p>
      <strong>
       日志管理
      </strong>
     </p>
     <p style="text-align:left;">
      记录访问日志与系统日志记录，支撑查看和排查系统异常。
     </p>
     <p style="text-align:left;">
      <img alt="036ae9e0d5ddf7642af64714299191d4.png" src="https://i-blog.csdnimg.cn/blog_migrate/70d21e95b1e33a951bdbb6510d3809b7.png"/>
     </p>
     <p style="text-align:center;">
      <img alt="ea0b7c7500ad920f765db43abd811be7.png" src="https://i-blog.csdnimg.cn/blog_migrate/7852985ee849b127d044cca64d4718cd.png"/>
     </p>
     <p>
      <strong>
       告警配置
      </strong>
     </p>
     <p style="text-align:left;">
      统一管理系统内的告警规则配置，触发告警规则时可产生对应的告警记录数据。
     </p>
     <p style="text-align:center;">
      <img alt="9b23a8193fabb3e492798f0596f1df17.png" src="https://i-blog.csdnimg.cn/blog_migrate/4d93546cbb9d20068dc370c52630ae29.png"/>
     </p>
     <p>
      <strong>
       可视化大屏
      </strong>
     </p>
     <p style="text-align:center;">
