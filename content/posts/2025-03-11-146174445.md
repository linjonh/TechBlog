    </h2>

      <strong>

       <strong>

       </strong>
       小于 num 时
      </strong>

    <blockquote>

      </strong>

      <br/>
      psx_buf[7]：右摇杆的水平值（X轴）。
      <br/>
      psx_buf[8]：右摇杆的垂直值（Y轴）。

      <strong>
       偏移方向和幅度。
      </strong>
     </p>
    </blockquote>

      </strong>
      ，确
      <strong>
       定摇杆的偏移方向和幅度
      </strong>
      。
     </li>
     <li>
      根据摇杆的偏移方向和幅度，调整四个电机的转速，实现小车的前后、左右、旋转等运动。
     </li>
     <li>
      通过设置
      <strong>
       死区（num）
      </strong>
      ，避免摇杆微小变化导致电机频繁调整。
     </li>
     <li>
      最终通过 motor_speed_set 函数将计算出的转速值传递给电机控制器，实现小车的运动控制。
     </li>
    </ol>
    <h2>
     <a id="8__157">
     </a>
     8. 参数转化示例
    </h2>
    <h3>
     <a id="_158">
     </a>
     假设
    </h3>
    <blockquote>
     <p>
      psx_buf[7] = 100（右摇杆向上推）。
      <br/>
      psx_buf[8] = 150（右摇杆向右推）。
      <br/>
      psx_buf[5] = 80（左摇杆向左推）。
      <br/>
      psx_buf[6] = 127（左摇杆中立）。
     </p>
    </blockquote>
    <h3>
     <a id="_164">
     </a>
     计算
    </h3>
    <blockquote>
     <p>
      右摇杆垂直值：127 - 100 = 27，差值为正，小车向前运动。
      <br/>
      右摇杆水平值：127 - 150 = -23，差值为负，小车向右平移。
      <br/>
      左摇杆水平值：127 - 80 = 47，差值为正，小车逆时针旋转。
      <br/>
      左摇杆垂直值：127 - 127 = 0，无变化。
     </p>
    </blockquote>
