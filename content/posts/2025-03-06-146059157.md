    <h3 id="convert-markdown-to-onenote">
     使用 C# 将 Obsidian Markdown 转换为 OneNote
    </h3>
    <p>
     使用 C# 将 Obsidian 中的 Markdown 文件转换为 OneNote 文档分为两个步骤。首先，我们将使用 Aspose.Html for .NET 将 MD 文件转换为 HTML 格式。之后，我们将使用 Aspose.Note for .NET 将转换后的 HTML 文件转换为 OneNote 支持的格式。
    </p>
    <blockquote>
     <p>
      <em>
       注意：
       <strong>
        Aspose.HTML
       </strong>
       和
       <strong>
        Aspose.Note都是
       </strong>
       <strong>
        Aspose.Total
       </strong>
       库的一部分。但它们也可以单独使用。
      </em>
     </p>
    </blockquote>
    <p>
     请按照以下步骤在 C# 中将 Obsidian 转换为 OneNote：
    </p>
    <ol>
     <li>
      <strong>
       使用 Aspose.HTML 的Converter.ConvertMarkdown()
      </strong>
      方法将 Markdown 文件转换为 HTML 。
     </li>
     <li>
      创建Aspsoe.Note的
      <strong>
       Document
      </strong>
      类的实例。
     </li>
     <li>
      使用转换后的 HTML 文档和 HtmlImportOptions 类对象调用
      <strong>
       Import() 。
      </strong>
     </li>
     <li>
      <strong>
       使用Save()
      </strong>
      方法将文档保存为 .one 文件。
     </li>
    </ol>
