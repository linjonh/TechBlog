    <h2>
     <a id="Jenkins__0">
     </a>
     [Jenkins] 即将关闭，剩余生成将不会被执行
    </h2>
    <h2>
     <a id="_1">
     </a>
     问题现象
    </h2>
    <p>
     该问题是最近才出现的，状态如下图所示，会提示
     <code>
      The Jenkins Controller is preparing for shutdown. No new builds can be started.
     </code>
     ​。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d00ac91ee40b446890a790ee5b3845b5.png#pic_center"/>
    </p>
    <h2>
     <a id="_4">
     </a>
     具体原因
    </h2>
    <p>
     在知乎上找到了有人和我遇到一样的问题，并且找到了问题所在，是由于构建超时和ThinBackup备份有关，ThinBackup默认设置等待空闲才开始备份（超时2小时会将Jenkins强制设置为quiet模式，而该模式下不会生成新的构建队列）。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b418e09feb8a4e51944a130c9411d8fc.png#pic_center">
      <br/>
      Reference：
      <a href="https://www.zhihu.com/question/444408386/answer/2224553578" rel="nofollow">
       https://www.zhihu.com/question/444408386/answer/2224553578
      </a>
     </img>
    </p>
    <h2>
     <a id="_8">
     </a>
     解决方法
    </h2>
    <ol>
     <li>
      将超时的任务关闭
     </li>
     <li>
      点击取消按钮，将剩余任务继续执行完
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d37b167dbb74417484f4b09800487263.png#pic_center"/>
     </li>
     <li>
      进入ThinBackup插件设置目录，将等待空闲备份功能关闭，将图中的红框取消勾选就可以了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/71d54038369f4e3a95b2b6720944a676.png#pic_center"/>
     </li>
     <li>
      保存后，最好在
      <code>
       系统管理
      </code>
      ​-&gt;
      <code>
       工具与动作
      </code>
      ​-&gt;
      <code>
       读取设置
      </code>
      ​，将系统重新读取基础设置以使用最新的配置。
      <br/>
      ​​
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/2fff96faee62435fa533aceb2b591701.png#pic_center"/>
     </li>
    </ol>
