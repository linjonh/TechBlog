    <h3>
     <a id="JDK21_0">
     </a>
     一、下载JDK21
    </h3>
    <p>
     访问
     <code>
      Oracle
     </code>
     官方网站或选择
     <code>
      OpenJDK
     </code>
     作为替代品。
     <code>
      Oracle JDK
     </code>
     从
     <code>
      11
     </code>
     版本开始是商业的，可能需要支付费用。
     <code>
      OpenJDK
     </code>
     是一个免费开源选项。
    </p>
    <p>
     <code>
      Oracle JDK
     </code>
     官方网站：
     <a href="https://www.oracle.com/java/technologies/downloads/#jdk21-mac" rel="nofollow">
      Oracle JDK Downloads
     </a>
     <br/>
     <code>
      OpenJDK
     </code>
     官方网站：
     <a href="https://jdk.java.net/" rel="nofollow">
      OpenJDK Downloads
     </a>
    </p>
    <p>
     这里以
     <code>
      JDK21
     </code>
     为例：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f9c266c24195f9f17422866e94b0ca2a.png">
      <br/>
      我选择的是
      <code>
       JDK21
      </code>
      进行安装：
      <code>
       ARM64
      </code>
      和
      <code>
       X64
      </code>
      代表
      <code>
       CPU
      </code>
      架构类型，执行
      <code>
       uname -m
      </code>
      命令可以查看电脑是什么类型的
      <code>
       cpu
      </code>
      架构，选择对应的包，不匹配会提示安装失败或者不可用；
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/35765a61afe6330836c5c4b539609393.png">
       <br/>
       <strong>
        <code>
         bin.tar.gz
        </code>
        和
        <code>
         dmg
        </code>
        包的区别：
       </strong>
       <code>
        tar.gz
       </code>
       直接解压就可以用，解压的路径就是安装的路径；
       <code>
        dmg
       </code>
       是
       <code>
        mac
       </code>
       安装文件规范，默认会安装到
       <code>
        /Library/Java/JavaVirtualMachines/
       </code>
       路径下；
      </img>
     </img>
    </p>
    <h3>
     <a id="JDK_12">
     </a>
     二、安装JDK
    </h3>
    <p>
     进入下载目录，解压文件并将文件移动到指定的目录中。因为我们是手动安装，所以需要自己创建
     <code>
      ~/Library/Java/JavaVirtualMachines
     </code>
     目录，我这里保持和自动安装的目录相同。
    </p>
    <pre><code class="prism language-shell"><span class="token builtin class-name">cd</span> ~/Downloads
<span class="token comment"># 记得将命令中的`openjdk-21.0.2_macos-x64_bin.tar.gz`、`jdk-21.0.2.jdk` 修改为你实际的名称：</span>
<span class="token function">tar</span> <span class="token parameter variable">-zxf</span> openjdk-21.0.2_macos-x64_bin.tar.gz
<span class="token function">mv</span> jdk-21.0.2.jdk ~/Library/Java/JavaVirtualMachines
</code></pre>
    <h3>
     <a id="_20">
     </a>
     三、设置环境变量
    </h3>
    <p>
     使用终端编辑你的
     <code>
      shell
     </code>
     配置文件（如
     <code>
      ~/.bash_profile
     </code>
     、
     <code>
      ~/.bashrc
     </code>
     或
     <code>
      ~/.zshrc
     </code>
     等），我们选择
     <code>
      ~/.zshrc
     </code>
     文件，在文件末尾加入以下内容：第一行设置了
     <code>
      JAVA_HOME
     </code>
     环境变量，指向
     <code>
      JDK 21
     </code>
     的安装目录。第二行将
     <code>
      JDK 21
     </code>
     的
     <code>
      bin
     </code>
     目录添加到了
     <code>
      PATH
     </code>
     环境变量中，以便在终端中直接使用
     <code>
      Java
     </code>
     命令。
    </p>
    <pre><code class="prism language-shell"><span class="token comment"># 记得将命令中的`jdk-21.0.2.jdk` 修改为你实际的名称：</span>
<span class="token builtin class-name">export</span> <span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span><span class="token string">"~/Library/Java/JavaVirtualMachines/jdk-21.0.2.jdk/Contents/Home"</span> 
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token variable">$JAVA_HOME</span>/bin:<span class="token environment constant">$PATH</span>
</code></pre>
    <p>
     2、应用配置：
     <code>
      source ~/.zshrc
     </code>
     。
    </p>
    <h3>
     <a id="_28">
     </a>
     四、验证安装
    </h3>
    <p>
     打开命令行界面，运行以下命令：
    </p>
    <pre><code class="prism language-java">java <span class="token operator">-</span>version
javac <span class="token operator">-</span>version
</code></pre>
    <p>
     如果显示了
     <code>
      java
     </code>
     和
     <code>
      javac
     </code>
     的版本信息，而不是错误消息，则表明
     <code>
      JDK
     </code>
     已成功安装。
    </p>
