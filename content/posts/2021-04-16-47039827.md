    <p>
     调试程序的时候出现了java.lang.NullPointerException错误
    </p>
    <span style="color:#FF0000">
     原因分析：
    </span>
    <br/>
    这个错误一般是数据库连接的错误。
    <br/>
    <br/>
    <span style="color:#FF0000">
     处理方法：
    </span>
    <br/>
    1、检查JAVA环境配置是否正确
    <br/>
    2、数据库的连接设置是否正确
    <br/>
    3、JDBC驱动文件是否包含在项目中
    <br/>
    加载JDBC驱动文件的具体方法是：
    <br/>
    在项目上点击右键-&gt;Properties-&gt;java Build Path
    <br/>
    然后选择Add External JARs找到准备好的jar包,选择打开，OK。
