    <h4>
     使用 Nginx 部署到服务器
    </h4>
    <h5>
     1. 上传文件
    </h5>
    <p>
     将本地项目构建生成的
     <code>
      dist
     </code>
     文件夹上传到服务器的指定目录，例如
     <code>
      /var/www/vue3-app
     </code>
     。你可以使用
     <code>
      scp
     </code>
     命令进行上传：
    </p>
    <pre><code class="language-bash">scp -r dist user@server_ip:/var/www/vue3-app
</code></pre>
    <p>
     其中
     <code>
      user
     </code>
     是服务器用户名，
     <code>
      server_ip
     </code>
     是服务器的 IP 地址。
    </p>
    <h5>
     2. 安装和配置 Nginx
    </h5>
    <p>
     在服务器上安装 Nginx：
    </p>
    <pre><code class="language-bash"># Ubuntu/Debian
sudo apt-get update
sudo apt-get install nginx

# CentOS/RHEL
sudo yum install nginx
</code></pre>
    <p>
     配置 Nginx 以指向你的项目文件。创建一个新的 Nginx 配置文件，例如
     <code>
      /etc/nginx/conf.d/vue3-app.conf
     </code>
     ：
    </p>
    <pre><code class="language-bash">server {
    listen 80;
    server_name your_domain_or_ip;  # 替换为你的域名或服务器 IP

    root /var/www/vue3-app;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
</code></pre>
    <h5>
     3. 验证和重启 Nginx
    </h5>
    <p>
     验证 Nginx 配置文件的语法是否正确：
    </p>
    <pre><code class="language-bash">sudo nginx -t
</code></pre>
    <p>
     如果语法正确，重启 Nginx 服务：
    </p>
    <pre><code class="language-bash">sudo systemctl restart nginx
</code></pre>
    <p>
     现在，你可以在浏览器中访问你的域名或服务器 IP 来查看部署好的 Vue 3 项目。
    </p>
    <h4>
     使用 Docker 部署
    </h4>
    <h5>
     1. 编写 Dockerfile
    </h5>
    <p>
     在项目根目录下创建一个
     <code>
      Dockerfile
     </code>
     ，内容如下：
    </p>
    <pre><code class="language-Dart"># 基础镜像，使用 Node.js 作为构建环境
FROM node:18-alpine as build-stage
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

# 第二阶段，使用 Nginx 作为生产环境服务器
FROM nginx:1.23-alpine as production-stage
COPY --from=build-stage /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
</code></pre>
    <p>
     同时，你可以创建一个
     <code>
      nginx.conf
     </code>
     文件来配置 Nginx：
    </p>
    <pre><code class="language-bash">server {
    listen 80;
    server_name _;
    root /usr/share/nginx/html;
    index index.html;
    location / {
        try_files $uri $uri/ /index.html;
    }
}
</code></pre>
    <h5>
     2. 构建 Docker 镜像
    </h5>
    <p>
     在项目根目录下执行以下命令构建 Docker 镜像：
    </p>
    <pre><code class="language-bash">docker build -t my-vue3-app .
</code></pre>
    <h5>
     3. 运行 Docker 容器
    </h5>
    <p>
     构建好镜像后，使用以下命令运行 Docker 容器：
    </p>
    <pre><code class="language-bash">docker run -d -p 8080:80 my-vue3-app
</code></pre>
    <p>
     这里将容器的 80 端口映射到主机的 8080 端口，你可以根据需要调整。
    </p>
    <h4>
     使用云平台部署（以 Vercel 为例）
    </h4>
    <h5>
     1. 注册和登录 Vercel
    </h5>
    <p>
     访问
     <a href="https://vercel.com/" rel="nofollow" title="Vercel 官网">
      Vercel 官网
     </a>
     注册并登录账号。
    </p>
    <h5>
     2. 关联代码仓库
    </h5>
    <p>
     将你的 Vue 3 项目代码托管到 GitHub、GitLab 或 Bitbucket 等代码托管平台。在 Vercel 中关联你的代码仓库。
    </p>
    <h5>
     3. 配置部署
    </h5>
    <p>
     Vercel 会自动检测到你的 Vue 3 项目，并提供默认的构建和部署配置。你可以根据需要进行自定义配置，例如指定构建命令和输出目录。
    </p>
    <h5>
     4. 部署项目
    </h5>
    <p>
     完成配置后，Vercel 会自动触发部署流程，部署完成后会提供一个访问链接，你可以在浏览器中访问该链接查看部署好的项目。
    </p>
