    <h2 style="text-align:justify">
     <strong>
      01 组件介绍
     </strong>
    </h2>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在数据可视化大屏搭建中，
     <strong>
      <strong>
       地图组件
      </strong>
     </strong>
     是使用最频繁且重要的组件之一。它不但能直观展现地理方位相关信息，还能通过与
     <strong>
      <strong>
       标记点、飞线、图表
      </strong>
     </strong>
     等各类组件巧妙搭配，达成更加多元、生动的数据呈现效果，在众多行业领域都得到了广泛应用。鉴于其强大的适配能力，面对包含地理或区域属性的数据可视化需求时，借助搭载地图组件的大屏方案，往往能以较低成本获取出色成效，极具性价比。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     下面小鲸鱼将为大家介绍相关行业的案例模板，并且帮助大家掌握地图可视化大屏的常见搭建流程。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/879686b49a6242a1842478def81d64c3.png"/>
    </p>
    <h2 style="text-align:justify">
     <strong>
      02 行业模板
     </strong>
    </h2>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在
     <strong>
      <strong>
       教育行业
      </strong>
     </strong>
     中，通常使用地图组件来展示某个地区的教育数据，如：各区域招生计划、学校人员数量、各区域人口流动等。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/6f14027fb15f4aa69b9a309825a82793.png"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在
     <strong>
      <strong>
       金融行业
      </strong>
     </strong>
     中，根据各地区的经济数据，使用不同颜色的底图层来进行划分，简单清晰地展示各地区地经济发展水平。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/4884a4459a0f4ae8ae35d9c2fae2e8b3.png"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在
     <strong>
      <strong>
       水利行业
      </strong>
     </strong>
     中，通过标绘组件来展示大到省区小到具体流域的水利情况，使用线段来勾勒出具体的河流路径，为防洪排涝工作提供决策支持。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/6a7c61abca3042d19bd4929875706239.png"/>
    </p>
    <h2 style="text-align:justify">
     <strong>
      03 搭建流程
     </strong>
    </h2>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     地图组件的搭建流程，一般可归纳为三大核心要素：
     <strong>
      <strong>
       组件的搭配、整体的UI设计（背景），数据的联动
      </strong>
     </strong>
     ，巧妙融合这三要素，能够极大地丰富和增强地图大屏的展示效果，使其更加生动且贴合多样化的数据可视化需求。
    </p>
    <h3 style="text-align:justify">
     <strong>
      1. 组件的搭配
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在
     <a class="link-info" href="https://www.shanhaibi.com/" rel="nofollow" title="山海鲸">
      <strong>
       山海鲸
      </strong>
     </a>
     <span style="display:none">
     </span>
     中，按展示效果来看，地图组件主要划分为 2D 与 3D 两大类别。在2D地图中，常见的功能有：
     <strong>
      <strong>
       地图下钻、标记点展示、路径展示、热力图
      </strong>
     </strong>
     等，想要实现这些效果，需要对地图的样式进行设置，包括
     <strong>
      <strong>
       底图层、呼吸点层、热力层、标记层
      </strong>
     </strong>
     和
     <strong>
      <strong>
       飞线层
      </strong>
     </strong>
     等。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/62c6d851d3204c29979cd5bdd50f6ab2.png"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       三维地图
      </strong>
     </strong>
     通过高度、深度、立体感等表现形式，能够真实还原地形地貌、城市建筑和空间结构。相比二维地图，它能够更清晰地展示复杂的地理数据，帮助用户快速理解空间关系，如地形起伏、建筑高度等。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在3D地图中，建筑模型是大屏的核心部分，通常使用
     <strong>
      <strong>
       倾斜摄影模型
      </strong>
     </strong>
     或
     <strong>
      <strong>
       3D城市白模
      </strong>
     </strong>
     来进行搭建，此外，还可通过添加
     <strong>
      <strong>
       标记、飞线、电子围栏、流光
      </strong>
     </strong>
     等组件，进一步丰富画面内容。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/d7a2099d3c3649efa5e21bb3dac354a7.png"/>
    </p>
    <h3 style="text-align:justify">
     <strong>
      2. 整体的UI设计
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在完成地图及其子组件的样式雕琢后，我们还需要对整体的UI进行设计，如
     <strong>
      <strong>
       调整看板背景
      </strong>
     </strong>
     。以添加
     <strong>
      <strong>
       相同配色
      </strong>
     </strong>
     的圆环特效为例，可根据地图主色调或关键数据所代表的颜色，选取与之匹配的圆环。这些圆环能以半透明、渐变等形式分布在看板角落或边缘，通过巧妙的动态设计，如缓慢缩放、旋转，不仅能吸引用户视线，还能与地图组件及其他子组件的色彩相互呼应，从视觉上构建起紧密的联系，让整个大屏
     <strong>
      <strong>
       画面过渡更加自然流畅
      </strong>
     </strong>
     。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/1147a82b2b934d5d82ea75256ebf4e05.png"/>
    </p>
    <h3 style="text-align:justify">
     <strong>
      3. 数据的联动
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     数据作为可视化大屏的重要组成部分，通常使用各种类型的图表组件来承载数据，常见的包括
     <strong>
      <strong>
       折线图、条形图、柱状图、饼图
      </strong>
     </strong>
     等，并且通过
     <strong>
      <strong>
       数据联动、选中传递
      </strong>
     </strong>
     等功能，使图表和地图组件进行交互，例如：
     <strong>
      <strong>
       点击表格某一行后，切换到对应标记点所在的地图区域
      </strong>
     </strong>
     。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/faf8b2da44c849f0a84c68d92431987f.gif"/>
    </p>
