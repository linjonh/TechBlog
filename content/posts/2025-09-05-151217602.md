# vscode连接SSH



### 前提：已正确安装VScode（若是将使用最好是

### 一、安装 Remote - SSH 扩展并连接远程主机

打开 VScode，点击左侧的扩展图标（或使用快捷键 `Ctrl+Shift+X`），在搜索框中输入 “SSH”，可以看到官方提供的 “Remote - SSH” 扩展，点击 “Install” 安装。

![](https://i-blog.csdnimg.cn/direct/fe4b133cb5084ae885bdbccc69a16bc8.png)

安装好插件后，左边的侧边栏上多一个远程连接的图标（如箭头1），点进去之后，出现 SSH 连接界面，点击小加号可以连接远程服务器。

![](https://i-blog.csdnimg.cn/direct/4d377bf1e78d494ab8855290121de627.png)

点击+号之后，页面上方出现如下命令框：

![](https://i-blog.csdnimg.cn/direct/99725c6242404fbcb42685c411e3f5dc.png)

这个命令框是用来输入连接命令的，使用 如下格式的`ssh` 命令连接到远程服务器：

```

ssh username@remote_host
ssh -p port_number username@remote_host

```

第一行是默认端口为 22 的连接命令，第二行是自定义端口号，`port_number` 为端口号，`username` 为用户名，`remote_host` 为 IP 地址。我使用的是默认端口，所以我的命令是：

```

ssh root@125.122.33.xxx
```

输入后回车执行，此时 会弹出选择更新配置文件，点击第一个路径即可：

![](https://i-blog.csdnimg.cn/direct/cacdb38801034ec0a2435462a71cd2f8.png)

然后软件的右下角会出现提示信息，点击 “Open Config” 打开配置文件：

![](https://i-blog.csdnimg.cn/direct/1cc85d964d414d7f907fb747ab407d74.png)

显示如下内容：

![](https://i-blog.csdnimg.cn/direct/8b59ae9d02284f699af403ab114d4b73.png)

确认配置文件没错的话，就可以保存关闭了。（Host后面也是IP地址）

Host：这是一个用户定义的别名，用于指代远程主机。可以在在终端中执行 ssh 加上这个别名来代替替代 ssh username@remote_host。  
HostName：指定远程主机的实际地址或主机名。  
Port：指定 SSH 连接使用的端口号。  
User：指定连接到远程主机时使用的用户名。  
一般这时候可以看到左边树状图中的 SSH 分支上，会多出一个 Ubuntu（如果没出现，可以点一下上面的刷新按钮）。点击 Ubuntu 旁边类似文件夹的小窗口（含义是“用新窗口打开”），就可以连接到 Ubuntu 了。

![](https://i-blog.csdnimg.cn/direct/8a8d41a6045c4b4eb949ee6a083d01c5.png)

点击之后会有新窗口，点击Linux之后，在最上方的命令行中输入密码并且回车

![](https://i-blog.csdnimg.cn/direct/f8e5b01c735e46b8a21ace4b52bc62e2.png)

连接成功后在左下角会出现如下内容：

![](https://i-blog.csdnimg.cn/direct/2b9e6ec988ca4bd1a522ed1aca8af809.png)

点击旁边的小三角形，再点击 “ TERMINAL”，可以看到 Ubuntu 的终端界面，在这里可以使用命令行操作 Ubuntu。

![](https://i-blog.csdnimg.cn/direct/692468f34565433e8f3f953f049e3769.png)

#### 完成！！！
