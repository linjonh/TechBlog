    <h4>
     贪心算法通俗解释
    </h4>
    <p>
     <strong>
      贪心算法
     </strong>
     是一种"每一步都选择当前最优解"的算法策略。它不关心全局是否最优，而是通过局部最优的累积来逼近最终解。优点是简单高效，缺点是可能无法得到全局最优解。
    </p>
    <h4>
     <strong>
      一句话秒懂
     </strong>
    </h4>
    <p>
     自动售货机找零钱：
     <strong>
      用最少数量的硬币凑出指定金额
     </strong>
     。比如找零198美分，它会优先用25美分的大硬币，不够再用小的，直到凑够金额。
    </p>
    <hr/>
    <h4>
     <strong>
      背景故事
     </strong>
    </h4>
    <p>
     想象你在加拿大超市当收银员（CAD场景）：
    </p>
    <ol>
     <li>
      <p>
       顾客买了东西
      </p>
     </li>
     <li>
      <p>
       你需要快速找出零钱
       <strong>
        198分
       </strong>
      </p>
     </li>
     <li>
      <p>
       收银台硬币有：50分、25分、10分、5分、1分
      </p>
     </li>
     <li>
      <p>
       <strong>
        目标
       </strong>
       ：用最少的硬币数量凑出1300分
      </p>
     </li>
    </ol>
    <pre><code class="hljs">using System;
using System.Collections.Generic;

public class GreedyAlgorithm
{
    [CommandMethod("xx")]
   public static void 贪心算法之硬币找零()
    {
        // 场景模拟：在 CAD 系统中自动计算最优找零方案
       List&lt;int&gt; coins = new List&lt;int&gt; { 1, 5, 10, 25,50 }; // 硬币面额（美分）
        int amount = 198; // 需要找零的金额（美分）

        List&lt;int&gt; result = CoinChange(coins, amount);

        Env.Editor.WriteMessage($"找零 {amount} 美分需要的最少硬币：");
        foreach (int coin in result)
        {
            Env.Editor.WriteMessage(coin + " "); 
        }
    }

    /// &lt;summary&gt;
    /// 贪心算法实现硬币找零
    /// &lt;/summary&gt;
    /// &lt;param name="coins"&gt;可用硬币面额数组&lt;/param&gt;
    /// &lt;param name="amount"&gt;目标金额&lt;/param&gt;
    /// &lt;returns&gt;硬币组合列表&lt;/returns&gt;
    static List&lt;int&gt; CoinChange(List&lt;int&gt; coins, int amount)
    {
        var sortCoins = coins.OrderByDescending(x=&gt;x).ToList();
       // Array.Sort(coins, (a, b) =&gt; b.CompareTo(a)); // 降序排序（关键贪心步骤）
        List&lt;int&gt; change = new List&lt;int&gt;();

        foreach (int coin in sortCoins)
        {
            while (amount &gt;= coin)
            {
                // 在 CAD 系统中，这里可以记录交易日志
                change.Add(coin);
                amount -= coin;
            }
        }

        return change;
    }
}</code></pre>

    <h5>
     代码注释说明：
    </h5>
    <ol>
     <li>
      <p>
       <code>
        Array.Sort(coins, (a, b) =&gt; b.CompareTo(a))
       </code>
       <br/>
       将硬币按面额从大到小排序，这是贪心算法的核心——优先使用大面额硬币
      </p>
     </li>
     <li>
      <p>
       <code>
        while (amount &gt;= coin)
       </code>
       <br/>
       只要当前硬币可以用就持续使用，体现贪心的"局部最优"特性
      </p>
     </li>
     <li>
      <p>
       时间复杂度为 O(n log n)，主要来自排序操作
      </p>
     </li>
