    <h4>
     <a id="CentOS__Conda_0">
     </a>
     CentOS 安装 Conda
    </h4>
    <h5>
     <a id="1__Miniconda__2">
     </a>
     1. 下载 Miniconda 安装脚本
    </h5>
    <p>
     Miniconda 是 Anaconda 的轻量版，只包含 Conda 和 Python。你可以从
     <a href="https://docs.conda.io/en/latest/miniconda.html" rel="nofollow">
      Miniconda 官方网站
     </a>
     下载适合你系统的安装脚本。
    </p>
    <pre><code class="prism language-bash"><span class="token function">wget</span> https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
</code></pre>
    <h5>
     <a id="2__9">
     </a>
     2. 运行安装脚本
    </h5>
    <p>
     下载完成后，运行安装脚本：
    </p>
    <pre><code class="prism language-bash"><span class="token function">bash</span> Miniconda3-latest-Linux-x86_64.sh
</code></pre>
    <h5>
     <a id="3__16">
     </a>
     3. 按照提示进行安装
    </h5>
    <p>
     安装过程中，你会看到一些提示：
    </p>
    <ul>
     <li>
      <strong>
       阅读许可协议
      </strong>
      ：按
      <code>
       Enter
      </code>
      键浏览许可协议，然后输入
      <code>
       yes
      </code>
      同意。
     </li>
     <li>
      <strong>
       选择安装路径
      </strong>
      ：默认路径是
      <code>
       ~/miniconda3
      </code>
      ，你可以按
      <code>
       Enter
      </code>
      使用默认路径，或指定其他路径。
     </li>
     <li>
      <strong>
       初始化 Conda
      </strong>
      ：安装完成后，会提示你是否要初始化 Conda。建议选择
      <code>
       yes
      </code>
      ，这样 Conda 会自动配置环境变量。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a7e0a32c5951422984e681d8fc483898.png#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/60e31cc7156042d98d0c382f00d2fb2f.png#pic_center"/>
     </img>
    </p>
    <p>
     安装好
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9cb506415d194487a1f1dfdbc4bddd13.png#pic_center"/>
    </p>
    <h5>
     <a id="4__Conda_32">
     </a>
     4. 激活 Conda
    </h5>
    <p>
     安装完成后，你需要激活 Conda。如果你选择了初始化 Conda，可以重新打开终端或运行以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.bashrc
</code></pre>
    <h5>
     <a id="5__39">
     </a>
     5. 验证安装
    </h5>
    <p>
     你可以通过以下命令验证 Conda 是否安装成功：
    </p>
    <pre><code class="prism language-bash">conda <span class="token parameter variable">--version</span>
</code></pre>
    <p>
     如果安装成功，你会看到 Conda 的版本号。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9604769119f64cdfb8abf9741af3be0e.png#pic_center"/>
    </p>
    <h5>
     <a id="6__Conda_51">
     </a>
     6. 更新 Conda（可选）
    </h5>
    <p>
     你可以通过以下命令更新 Conda 到最新版本：
    </p>
    <pre><code class="prism language-bash">conda update conda
</code></pre>
    <h5>
     <a id="7__58">
     </a>
     7. 创建和管理环境
    </h5>
    <p>
     现在你可以使用 Conda 创建和管理 Python 环境了。例如，创建一个名为
     <code>
      myenv
     </code>
     的环境并安装 Python 3.8：
    </p>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> myenv <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.12</span>
</code></pre>
    <p>
     <code>
      The following NEW packages will be INSTALLED:
     </code>
     会提示需要安装这些 packages，y 就好
    </p>
    <p>
     激活环境：
    </p>
    <pre><code class="prism language-bash">conda activate myenv
</code></pre>
    <p>
     退出环境：
    </p>
    <pre><code class="prism language-bash">conda deactivate
</code></pre>
    <h5>
     <a id="8__Conda_79">
     </a>
     8. 卸载 Conda（可选）
    </h5>
    <p>
     如果你需要卸载 Conda，可以删除安装目录：
    </p>
    <pre><code class="prism language-bash"><span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/miniconda3
</code></pre>
    <p>
     然后从
     <code>
      .bashrc
     </code>
     或
     <code>
      .zshrc
     </code>
     中删除 Conda 的初始化代码。
    </p>
    <hr/>
    <h4>
     <a id="Windowsconda_90">
     </a>
     Windows安装conda
    </h4>
    <blockquote>
     <p>
      conda 5.6G，安装包 1G
     </p>
    </blockquote>
    <h5>
     <a id="1__Miniconda__Anaconda_94">
     </a>
     1. 下载 Miniconda 或 Anaconda
    </h5>
    <ul>
     <li>
      <strong>
       Miniconda
      </strong>
      ：轻量版，仅包含 Conda 和 Python。
     </li>
     <li>
      <strong>
       Anaconda
      </strong>
      ：完整版，包含 Conda、Python 和大量科学计算库。
     </li>
    </ul>
    <h5>
     <a id="2__Miniconda_99">
     </a>
     2. 下载 Miniconda
    </h5>
    <ol>
     <li>
      访问
      <a href="https://docs.conda.io/en/latest/miniconda.html" rel="nofollow">
       Miniconda 下载页面
      </a>
      。
     </li>
     <li>
      选择适合 Windows 的 Python 版本（推荐 Python 3.x）。
     </li>
     <li>
      下载
      <code>
       .exe
      </code>
      安装文件。
     </li>
    </ol>
    <h5>
     <a id="3__Anaconda_105">
     </a>
     3. 下载 Anaconda
    </h5>
    <ol>
     <li>
      访问
      <a href="https://www.anaconda.com/products/distribution" rel="nofollow">
       Anaconda 下载页面
      </a>
      。
     </li>
     <li>
      选择适合 Windows 的 Python 版本。
     </li>
     <li>
      下载
      <code>
       .exe
      </code>
      安装文件。
     </li>
    </ol>
    <h5>
     <a id="4__Path__111">
     </a>
     4. 编辑 Path 变量
    </h5>
    <blockquote>
     <p>
      如果安装完后，bash 执行 conda 说没有找到指令的
     </p>
    </blockquote>
    <ul>
     <li>
      在“环境变量”窗口中，找到“系统变量”部分，选择
      <code>
       Path
      </code>
      ，然后点击“编辑”。
     </li>
     <li>
      点击“新建”，然后添加以下路径（根据你的 Conda 安装路径）：
      <ul>
       <li>
        <code>
         C:\Users\&lt;你的用户名&gt;\Anaconda3\Scripts
        </code>
       </li>
       <li>
        <code>
         C:\Users\&lt;你的用户名&gt;\Anaconda3\
        </code>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="powershellconda_122">
     </a>
     在powershell上用conda
    </h4>
    <h5>
     <a id="1__PowerShell__124">
     </a>
     1. 检查 PowerShell 执行策略
    </h5>
    <p>
     PowerShell 的执行策略可能阻止了 Conda 初始化脚本的运行。你可以通过以下命令查看当前策略：
    </p>
    <pre><code class="prism language-powershell"><span class="token function">Get-ExecutionPolicy</span>
</code></pre>
    <p>
     如果策略是
     <code>
      Restricted
     </code>
     ，你需要更改为
     <code>
      RemoteSigned
     </code>
     或
     <code>
      Unrestricted
     </code>
     ：
    </p>
    <pre><code class="prism language-powershell"><span class="token function">Set-ExecutionPolicy</span> RemoteSigned <span class="token operator">-</span>Scope CurrentUser
</code></pre>
    <h5>
     <a id="2__Conda_138">
     </a>
     2. 重新初始化 Conda
    </h5>
    <p>
     确保 Conda 已正确初始化。运行以下命令重新初始化：
    </p>
    <pre><code class="prism language-powershell">conda init powershell
</code></pre>
    <p>
     然后关闭并重新打开 PowerShell。
    </p>
