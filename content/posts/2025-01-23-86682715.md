# linux上测试网速带宽

主要的测试思路是通过开源工具speedtest测试。

下面是操作步骤：

1.从github克隆包到本地

# git clone https://github.com/sivel/speedtest-cli.git

2.执行speedtest这个python脚本

# python speedtest.py
