    <h2>
     <a id="python_3">
     </a>
     python安装
    </h2>
    <p>
     要确保已经安装了python的一个版本。如果没有安装可以参考我之前的一篇
     <a href="https://blog.csdn.net/qq_40024275/article/details/102801509">
      文章
     </a>
     。
     <br/>
     去
     <a href="https://www.python.org/" rel="nofollow">
      python官网
     </a>
     下载最新的与自己电脑匹配的python版本，我这里下载的是python3.8.0版本，64位。下载exe可以直接安装，安装时记得选中
     <code>
      Add Python 3.8 to Path
     </code>
     ，否则需要后续手动配置环境变量。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa7ec9f4ef3feab3f93a883da34eca9b.png#pic_center"/>
    </p>
    <h2>
     <a id="python_8">
     </a>
     查看python版本信息
    </h2>
    <p>
     首先，查看自己python的版本等信息。在命令行输入python，即可查看python的版本信息啦！
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d98de5cb31632c93a677b41cdb772883.png#pic_center">
      <br/>
      比如我这个显示，就是说我的python是3.8.0版本的，并且python是64位的，后面的on Win32是在Windows系统上。
     </img>
    </p>
    <h2>
     <a id="pip_12">
     </a>
     检查pip
    </h2>
    <p>
     在命令行输入
     <code>
      python -m pip --version
     </code>
     ，确保已经安装pip。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d91ef909678ab6597194436878f90f3f.png">
      <br/>
      这里可以看到pip对应的存放位置，以及python的版本。我这里的pip是19.2.3，存放位置是上面的路径
      <code>
       C:\Users\Wanjia Zhang\AppData\Local\Programs\Python\Python38\lib\site-packages\pip
      </code>
      。
      <br/>
      如果发现没有安装pip，可以到
      <a href="https://www.python.org/" rel="nofollow">
       python官网
      </a>
      下载：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/470040c2abd3ee0ca79d701d02f40772.png">
       <br/>
       也可以直接访问
       <a href="https://pypi.org/" rel="nofollow">
        pip
       </a>
       来进行下载。
      </img>
     </img>
    </p>

     <a id="_31">
     </a>
     地址二
    </h3>
    <p>
     我习惯使用第二个地址下载：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d6506da55c39e7ab1a1aad9e1cff21c.png#pic_center">
      <br/>
      上图中的1.9.6指的是pygame的版本，cp后面的数字表示的是pygame的版本，需要根据自己的python版本进行选择，再后面就是看你python是32位的还是64位的，这里注意看的是python对应的位，不是系统的。
      <br/>
      我选择下载
      <code>
       pygame-1.9.6-cp38-cp38-win_amd64.whl
      </code>

    <h2>
     <a id="pygame_36">
     </a>
     pygame安装
    </h2>
    <h3>
     <a id="_37">
     </a>
     压缩包版本的安装
    </h3>
    <p>
     如果你下载的是一个压缩包，则将该压缩包解压并且放在刚才pip所在的lib目录下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/879f0bbf2e3e6d34e7d19fae94cb821f.png#pic_center"/>
     <br/>
     然后在命令行输入
     <code>
      python -m pip install --user pygame
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1659cd8d5dd84081905a63ad57d367ff.png#pic_center"/>
    </p>
    <h3>
     <a id="whl_42">
     </a>
     .whl版本的安装
    </h3>
    <p>
     将下载的.whl文件放在与上面pip文件夹并列的位置：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d4e162e7c37ef1ee0829d0df2d5c0fed.png#pic_center"/>
     <br/>
     然后在空白处按住shift点击右键，也就是在这个目录下打卡命令行窗口，当然你也可以自己寻路过去。然后在命令行输入：
     <code>
      pip install pygame-1.9.6-cp38-cp38-win_amd64.whl
     </code>
     后面是你自己下载的whl的名字。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a7d7412b008990fb2480d549d5fb1f26.png"/>
     <br/>
     （没错儿我就是进行到这一步之前把原来的python卸载了，换到E盘中重新装了一次，忽略路径的问题）
    </p>
    <p>
     安装过程中可能出现的问题：
     <br/>
     问题一：You are using pip version 19.2.3, however version 19.3.1 is available.
     <br/>
     You should consider upgrading via the ‘python -m pip install --upgrade pip’ command.
     <br/>
     这个问题主要是版本没有更新的意思，输入python -m pip install --upgrade pip命令更新一下就好啦~
    </p>
    <p>
     问题二：pygame-1.9.6-cp38-cp38-win_amd64.whl is not a supported wheel on this platform.
     <br/>
     产生这位问题的原因就是版本不对应。检查一下是否已经安装了对应的python版本，并确定是32位还是64位的。
    </p>

     <code>
      pip install pygame

     就可以。
    </p>
