    <h4>
     <a id="3_Swagger_UI_87">
     </a>
     3. 启动项目并访问Swagger UI
    </h4>
    <p>
     完成上述配置后，启动你的Spring Boot项目。然后，你可以通过以下URL访问Swagger UI界面：
    </p>
    <ul>
     <li>
      使用
      <code>
       springfox-swagger2
      </code>
      ：
      <code>
       http://localhost:8080/swagger-ui.html
      </code>
     </li>
     <li>
      使用
      <code>
       springdoc-openapi
      </code>
      ：
      <code>
       http://localhost:8080/swagger-ui.html
      </code>
      或
      <code>
       http://localhost:8080/swagger-ui/index.html
      </code>
     </li>
    </ul>
    <h4>
     <a id="4_Swagger_94">
     </a>
     4. 使用Swagger注解
    </h4>
    <p>
     你可以在Controller和Model上使用Swagger的注解来进一步定制API文档。
    </p>
    <h5>
     <a id="_98">
     </a>
     常用注解
    </h5>
    <ul>
     <li>
      <code>
       @Api
      </code>
      : 用于类上，表示这个类是Swagger的资源。
     </li>
     <li>
      <code>
       @ApiOperation
      </code>
      : 用于方法上，表示一个HTTP请求的操作。
     </li>
     <li>
      <code>
       @ApiParam
      </code>
      : 用于参数上，表示对参数的说明。
     </li>
     <li>
      <code>
       @ApiModel
      </code>
      : 用于模型类上，表示对模型的说明。
     </li>
     <li>
      <code>
       @ApiModelProperty
      </code>
      : 用于模型类的属性上，表示对属性的说明。
     </li>
    </ul>
