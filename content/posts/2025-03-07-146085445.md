       <li>
        <code>
         y
        </code>
        ：存储上个月剩余的零钱（未存入妈妈账户的部分）。
       </li>
       <li>
        <code>
         ans
        </code>
        ：累计存入妈妈账户的整百金额总和。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        循环处理每月数据
       </strong>
       ：
      </p>
      <ul>
       <li>
        遍历12个月，逐月输入开销
        <code>
         x
        </code>
        。
       </li>
       <li>
        <strong>
         资金不足判断
        </strong>
        ：若上个月剩余的钱
        <code>
         y
        </code>
        加上本月300元不足以支付本月开销，直接输出负的当前月份并终止程序。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        存款计算
       </strong>
       ：
      </p>
      <ul>
       <li>
        计算本月总剩余资金
        <code>
         (y + 300 - x)
        </code>
        。
       </li>
       <li>
        将总剩余资金中整百的部分存入妈妈账户（
        <code>
         q1 * 100
        </code>
        ），并更新累计存款
        <code>
         ans
        </code>
        。
       </li>
       <li>
        剩余零钱更新为总剩余资金对100取模的结果，即
        <code>
         y = (y + 300 - x) % 100
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        年终结算
       </strong>
       ：
      </p>
      <ul>
       <li>
        输出最终总金额，包括妈妈账户中的本金与20%利息（
        <code>
         ans * 1.2
        </code>
        ）以及手中剩余的零钱
        <code>
         y
        </code>
        。
       </li>
      </ul>
     </li>
    </ol>

     <a id="_137">

    </h4>

     </li>

      <strong>
       整百存款计算
      </strong>

     </li>
     <li>

      </strong>
