    <h4>
     <strong>
      <code>
       1、wget
      </code>
     </strong>
     : 这是一个用于从网络上下载文件的命令行工具。它支持 HTTP、HTTPS 和 FTP 协议。
    </h4>
    <pre><code>wget http://download.qt.io/archive/qt/5.12/5.12.9/qt-opensource-linux-x64-5.12.9.run</code></pre>
    <h4>
     2、下载完成后，你可以通过以下命令使文件可执行并运行安装程序：
    </h4>
    <pre><code>chmod +x qt-opensource-linux-x64-5.12.9.run
sudo ./qt-opensource-linux-x64-5.12.9.run</code></pre>
    <h4>
     3、安装五笔输入法&amp;安装拼音输入法
    </h4>
    <pre><code>sudo apt-get install fcitx-table-wubi // 五笔输入法
sudo apt-get install fcitx-sunpinyin // 拼音输入法请装 sunpinyin 或者 fcitx-googlepinyin</code></pre>
    <h4>
     4、安装git
    </h4>
    <pre><code>sudo apt-get install git</code></pre>
    <h4>
     5、克隆 Git 仓库
    </h4>
    <pre><code>git clone https://gitee.com/QQ1252699831/fcitx-qt5-1.1.1.git</code></pre>
    <h4>
     6、运行以下命令来执行脚本
    </h4>
    <pre><code>./fcitx-install.sh</code></pre>
    <h4>
     7、运行的命令需要管理员权限，可以在命令前加上
     <code>
      sudo
     </code>
    </h4>
    <pre><code>sudo ./fcitx-install.sh

//只有具有 sudo 权限的用户才能使用 sudo。如果你是普通用户且没有 sudo 权限，请联系系统管理员。

//使用 sudo 时要小心，因为它可以执行任何系统操作，可能会导致系统损坏。</code></pre>
    <h4>
     8、如何查看版本，以fcitx为例
    </h4>
    <pre><code>fcitx --version</code></pre>
    <h4>
     9、安装
     <code>
      make
     </code>
     构建工具
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        <code>
         make
        </code>
       </strong>
       : 这是一个常用的构建工具，用于自动化编译和安装软件。它读取
       <code>
        Makefile
       </code>
       文件中的指令来执行编译任务。
      </p>
     </li>
    </ul>
    <pre><code class="hljs">sudo apt-get install make</code></pre>
    <h4>
     10、安装 OpenGL 核心库
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        <code>
         libxcb-cursor0
        </code>
       </strong>
       : 这是 XCB（X C Binding）库的一部分，用于处理光标相关的功能。XCB 是一个用于与 X Window 系统通信的库。
      </p>
     </li>
    </ul>
    <pre><code class="hljs">sudo apt-get install libgl1-mesa-dev #安装OpenGL核心库</code></pre>
