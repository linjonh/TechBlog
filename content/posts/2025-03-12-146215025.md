    <h2>
     <a id="_0">
     </a>
     概述
    </h2>
    <blockquote>
     <p>
      <strong>
       装饰器模式
      </strong>
      :
      <strong>
       Decorator Pattern
      </strong>
      : 是一种结构型设计模式.
      <br/>
      <strong>
       作用
      </strong>
      ： 允许你
      <code>
       动态地给对象添加功能或职责
      </code>
      ，而
      <code>
       无需修改其原始类的代码
      </code>
      ,非常的符合
      <code>
       开闭原则
      </code>
      。
      <br/>
      <strong>
       实现思路
      </strong>
      ：通过创建一个包装对象（即装饰器），来包裹原始对象，并在运行时动态地为这个对象增加新的行为。
      <br/>
      简单理解 ： 就是一层一层的包装起来。
     </p>
    </blockquote>
    <h2>
     <a id="_6">
     </a>
     核心思想
    </h2>
    <blockquote>
     <p>
      1、
      <strong>
       开放封闭原则
      </strong>
      ：软件实体（类、模块、函数等）应该对扩展开放，对修改封闭。
      <br/>
      装饰器模式很好地体现了这一原则，它允许我们在不改变已有代码的情况下，通过添加新的装饰器类来扩展功能。
      <br/>
      2、
      <strong>
       避免继承层次过多
      </strong>
      ：如果使用继承来实现功能扩展，可能会导致子类数量爆炸式增长。而使用装饰器模式可以有效地减少这种复杂性。
     </p>
    </blockquote>
    <h2>
     <a id="_11">
     </a>
     主要角色
    </h2>
    <blockquote>
     <p>
      1、
      <strong>
       Component（组件）
      </strong>
      ：定义了被装饰的对象接口，可以是一个具体类或抽象类。
      <br/>
      2、
      <strong>
       ConcreteComponent（具体组件）
      </strong>
      ：实现了Component接口的具体类，是我们希望装饰的对象。
      <br/>
      3、
      <strong>
       Decorator（装饰器）
      </strong>
      ：持有一个Component对象的引用，并且定义了一个与Component相同的接口，使得Decorator可以替代具体的组件。
      <br/>
      4、
      <strong>
       ConcreteDecorator（具体装饰器）
      </strong>
      ：实现了Decorator接口，并在其基础上添加了额外的行为或功能。
     </p>
    </blockquote>
    <h2>
     <a id="_17">
     </a>
     情景模式
    </h2>
    <blockquote>
     <p>
      1、有一些企业是 做电子产品的，可以生产各种各样的电子产品 -
      <strong>
       抽象的组件
      </strong>
      ；
      <br/>
      2、有一家企业是 专门生产电脑的 -
      <strong>
       具体的组件
      </strong>
      ；
      <br/>
      3、有一些企业是做 音响的，可以给电脑配置音响 -
      <strong>
       装饰器
      </strong>
      ；
      <br/>
      4、有一家企业是 专门给电脑定制音响的 -
      <strong>
       具体装饰器
      </strong>
      。
      <br/>
      5、有一些企业是做 软件的，可以给电脑安装软件 -
      <strong>
       装饰器
      </strong>
      ；
      <br/>
      6、有一家企业是 专门给电脑做 聊天软件的 -
      <strong>
       具体的装饰器
      </strong>
      。
     </p>
    </blockquote>
