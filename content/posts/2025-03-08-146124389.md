    <h3>
     <a id="1__SQLite3_20">
     </a>
     1. 什么是 SQLite3？
    </h3>
    <p>
     SQLite3 是一个轻量级的
     <strong>
      嵌入式数据库
     </strong>
     ，无需安装独立服务，所有数据存储在一个文件中。
    </p>
    <ul>
     <li>
      ✅
      <strong>
       无需配置
      </strong>
      ：像操作普通文件一样使用。
     </li>
     <li>
      ✅
      <strong>
       跨平台
      </strong>
      ：数据库文件可在 Windows/macOS/Linux 共享。
     </li>
     <li>
      ✅
      <strong>
       适合小型项目
      </strong>
      ：APP 配置、本地缓存、小型网站等。
     </li>
    </ul>
    <hr/>

    <h3>
     <a id="6__168">

     6. 事务与错误处理
    </h3>
    <h4>
     <a id="61__169">
     </a>
     6.1 什么是事务？
    </h4>
    <ul>
     <li>
      <strong>
       事务
      </strong>
      ：一组原子性的操作，要么全部成功，要么全部失败。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：转账操作（A 扣款，B 加款必须同时成功）。
     </li>
    </ul>
    <h4>
     <a id="62__173">
     </a>
     6.2 手动提交与回滚
    </h4>
    <pre><code class="prism language-python"><span class="token keyword">try</span><span class="token punctuation">:</span>
    cursor<span class="token punctuation">.</span>execute<span class="token punctuation">(</span><span class="token string">"INSERT INTO users (name) VALUES ('David')"</span><span class="token punctuation">)</span>
    cursor<span class="token punctuation">.</span>execute<span class="token punctuation">(</span><span class="token string">"INSERT INTO users (name) VALUES ('Eve')"</span><span class="token punctuation">)</span>
    conn<span class="token punctuation">.</span>commit<span class="token punctuation">(</span><span class="token punctuation">)</span>  <span class="token comment"># 成功则提交</span>
<span class="token keyword">except</span> sqlite3<span class="token punctuation">.</span>Error <span class="token keyword">as</span> e<span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"出错啦:"</span><span class="token punctuation">,</span> e<span class="token punctuation">)</span>
    conn<span class="token punctuation">.</span>rollback<span class="token punctuation">(</span><span class="token punctuation">)</span>  <span class="token comment"># 失败则回滚（撤销所有操作）</span>
</code></pre>

    <h3>
     <a id="8__215">
     </a>
     8. 常见问题
    </h3>
    <h4>
     <a id="Q1_216">
     </a>
     Q1：代码运行后找不到数据库文件？
    </h4>
    <ul>
     <li>
      <strong>
       检查当前目录
      </strong>
      ：在代码中添加：
      <pre><code class="prism language-python"><span class="token keyword">import</span> os
<span class="token keyword">print</span><span class="token punctuation">(</span>os<span class="token punctuation">.</span>getcwd<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span>  <span class="token comment"># 查看文件实际保存路径</span>
</code></pre>
     </li>
     <li>
      <strong>
       使用绝对路径
      </strong>
      ：如
      <code>
       sqlite3.connect('/User/name/data.db')
      </code>
     </li>
    </ul>
    <h4>
     <a id="Q2_224">
     </a>
     Q2：插入数据后查询不到？
    </h4>
    <ul>
     <li>
      <strong>
       忘记提交事务
      </strong>
      ：确保执行
      <code>
       conn.commit()
      </code>
      。
     </li>
     <li>
      <strong>
       查询条件错误
      </strong>
      ：检查 WHERE 子句是否拼写正确。
     </li>
    </ul>

     <a id="Q3_228">
     </a>
     Q3：如何备份数据库？
