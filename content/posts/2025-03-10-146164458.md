    <h3>
     <a id="_3">
     </a>
     前言
    </h3>
    <p>
     在现代前端开发中，单页面应用（SPA）是非常流行的开发模式，而路由管理是 SPA 中的核心功能之一。Vue 提供了官方的路由管理工具——Vue Router，帮助我们轻松地实现页面导航、动态路由、嵌套路由等功能。
    </p>
    <p>
     今天，我们将学习 Vue Router 的基础用法，并通过一个简单的项目实践来掌握以下内容：
    </p>
    <ol>
     <li>
      路由配置与导航。
     </li>
     <li>
      动态路由与嵌套路由。
     </li>
     <li>
      路由守卫与懒加载。
     </li>
    </ol>
    <p>
     最后，我们会为项目添加路由支持，实现页面切换功能，比如任务列表页和任务详情页。

    <h3>
     <a id="_24">
     </a>
     一、路由配置与导航
    </h3>
    <h4>
     <a id="11__Vue_Router_26">
     </a>
     1.1 什么是 Vue Router？
    </h4>
    <p>
     Vue Router 是 Vue.js 的官方路由管理库，用于在单页面应用中实现页面导航。它通过监听 URL 的变化，动态加载对应的组件，从而实现页面的切换。
    </p>
    <h4>
     <a id="12__Vue_Router_30">
     </a>
     1.2 安装 Vue Router
    </h4>
    <p>
     在 Vue 3 中，可以通过以下命令安装 Vue Router：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span> vue-router
</code></pre>
    <h4>
     <a id="13__38">
     </a>
     1.3 基本路由配置
    </h4>
    <h5>
     <a id="_40">
     </a>
     步骤
    </h5>
    <ol>
     <li>
      创建路由配置文件
      <code>
       router/index.js
      </code>
      。
     </li>
     <li>
      定义路由规则（
      <code>
       routes
      </code>
      ），每个路由规则都需要指定路径（
      <code>
       path
      </code>
      ）和对应的组件（
      <code>
       component
      </code>
      ）。
     </li>
     <li>
      创建路由实例并将其添加到 Vue 应用中。
     </li>
    </ol>

    <h4>
     <a id="14__95">
     </a>
     1.4 路由导航
    </h4>
    <p>
     在模板中使用
     <code>
      &lt;router-link&gt;
     </code>
     实现页面跳转：
    </p>
    <pre><code class="prism language-vue">&lt;template&gt;
  &lt;div&gt;
    &lt;nav&gt;
      &lt;router-link to="/"&gt;首页&lt;/router-link&gt;
      &lt;router-link to="/tasks"&gt;任务列表&lt;/router-link&gt;
    &lt;/nav&gt;
    &lt;router-view&gt;&lt;/router-view&gt;
  &lt;/div&gt;
&lt;/template&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        &lt;router-link&gt;
       </code>
      </strong>
      ：用于创建导航链接，
      <code>
       to
      </code>
      属性指定目标路径。
     </li>
     <li>
      <strong>
       <code>
        &lt;router-view&gt;
       </code>
      </strong>
      ：用于显示匹配的组件。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_116">
     </a>
     二、动态路由与嵌套路由
    </h3>
    <h4>
     <a id="21__118">
     </a>
     2.1 动态路由
    </h4>
    <p>
     动态路由允许我们在路径中使用动态参数，例如
     <code>
      /tasks/:id
     </code>
     ，其中
     <code>
      :id
     </code>
     是一个动态参数。
    </p>
    <h5>
     <a id="_122">
     </a>
     示例：任务详情页
    </h5>
    <p>
     <strong>
      路由配置：
     </strong>
    </p>
    <pre><code class="prism language-javascript"><span class="token punctuation">{<!-- --></span> <span class="token literal-property property">path</span><span class="token operator">:</span> <span class="token string">'/tasks/:id'</span><span class="token punctuation">,</span> <span class="token literal-property property">component</span><span class="token operator">:</span> TaskDetail <span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      <code>
       TaskDetail.vue
      </code>
      ：
     </strong>
    </p>
    <pre><code class="prism language-vue">&lt;template&gt;
  &lt;div&gt;
    &lt;h1&gt;任务详情&lt;/h1&gt;
    &lt;p&gt;任务 ID：{<!-- -->{ taskId }}&lt;/p&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script&gt;
import { useRoute } from 'vue-router';

export default {
  setup() {
    const route = useRoute();
    const taskId = route.params.id; // 动态参数
    return { taskId };
  },
};
&lt;/script&gt;
</code></pre>
    <hr/>
    <h4>
     <a id="22__154">
     </a>
     2.2 嵌套路由
    </h4>
    <p>
     嵌套路由允许我们在一个路由中嵌套子路由。例如，在任务列表页中，我们可以嵌套一个任务详情的子路由。
    </p>
    <h5>
     <a id="_158">
     </a>
     示例：嵌套任务详情
    </h5>
    <p>
     <strong>
      路由配置：
     </strong>
    </p>
    <pre><code class="prism language-javascript"><span class="token punctuation">{<!-- --></span>
  <span class="token literal-property property">path</span><span class="token operator">:</span> <span class="token string">'/tasks'</span><span class="token punctuation">,</span>
  <span class="token literal-property property">component</span><span class="token operator">:</span> TaskList<span class="token punctuation">,</span>
  <span class="token literal-property property">children</span><span class="token operator">:</span> <span class="token punctuation">[</span>
    <span class="token punctuation">{<!-- --></span> <span class="token literal-property property">path</span><span class="token operator">:</span> <span class="token string">':id'</span><span class="token punctuation">,</span> <span class="token literal-property property">component</span><span class="token operator">:</span> TaskDetail <span class="token punctuation">}</span><span class="token punctuation">,</span> <span class="token comment">// 嵌套路由</span>
  <span class="token punctuation">]</span><span class="token punctuation">,</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      <code>
       TaskList.vue
      </code>
      ：
     </strong>
    </p>
    <pre><code class="prism language-vue">&lt;template&gt;
  &lt;div&gt;

      tasks: [
        { id: 1, name: '学习 Vue Router' },

  },

&lt;/script&gt;
</code></pre>

     三、路由守卫与懒加载
    </h3>
    <h4>
     <a id="31__203">
     </a>

    </h4>

    <pre><code class="prism language-javascript">router<span class="token punctuation">.</span><span class="token function">beforeEach</span><span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token parameter">to<span class="token punctuation">,</span> from<span class="token punctuation">,</span> next</span><span class="token punctuation">)</span> <span class="token operator">=&gt;</span> <span class="token punctuation">{<!-- --></span>
