# 科普 | 数据安全与网络安全（一）概念篇

当我们介绍天空卫士的时候，通常会强调我们是一家专注于数据安全技术的企业。这时，“童鞋们”的脑子里是不是产生了一个大大的问号？

概念、应用

局限、融合

来为大家解析网络安全和数据安全的区别，希望能为您答疑解惑。

首先小编从概念上来进行对比：

基本概念

网络安全Cyber Security

指网络系统的硬件、软件及其系统中的数据受到保护，不因偶然或恶意的原因遭到破坏、更改、泄露。系统连续可靠地运行，网络服务不中断，保障网络信息的存储安全，以及信息的产生、传输和使用过程中的安全。

数据安全
  
Data Security

是指通过采取必要措施，确保数据处于有效保护和合法利用的状态，以及具备保障持续安全状态的能力。数据安全要保证从数据收集到销毁的全生命周期的安全。

观点提炼

网络安全保护是指：系统不被攻击、网络不中断、系统正常运行。

数据安全保护是指：数据做为资产，对其安全的保护。

概念侧重点

网络安全Cyber Security

网络安全以“网络安全体系为中心”，侧重于计算的资源与环境，是有边界的，通常受限于主权边界。网络安全也泛指整个安全体系，侧重于网络空间安全、网络访问控制、安全通信、防御网络攻击或入侵等。通过保障计算环境（网络空间）的安全，从而最终保障计算对象（数据）的安全。

数据安全
  
Data Security

数据安全是以“数据为中心”，主要侧重于数据全生命周期的安全和合规性，以此来保护数据的安全，特别是敏感数据的安全与合规。

观点提炼

网络安全是基于边界的，边界内就是网络安全要守护的范围。网络安全对于环境与资源的作用，就好像城墙对于城堡的作用。只要城墙不被攻破，城堡内就是安全的。如果外面的人员想要进“城”，需要守门人打开城门，合法进入，翻墙或者破门而入则被视为“攻击”。

数据安全是没有边界的，数据资产在哪里，数据安全就在哪里。还是以城堡举例，城墙内是网络安全构筑的安全边界。城堡内有多少棵树、多少张桌子、多少珠宝、多少古玩等，对于这些资产的保护，属于数据安全要做的事情。
