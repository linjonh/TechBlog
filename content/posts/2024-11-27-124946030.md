    <h2>
     <a id="_0">
     </a>
     总览
    </h2>
    <p>
     经常需要在服务器上跑一些代码，主要用到xshell、xftp以及vscode。
     <br/>
     其中：
    </p>
    <ul>
     <li>
      xshell用于连接服务器之后执行命令行命令；
     </li>
     <li>
      xftp主要用于本地与服务器之间拖动文件；
     </li>
     <li>
      vscode用于直接修改服务器端的代码内容。
     </li>
    </ul>
    <h2>
     <a id="xshell_6">
     </a>
     xshell连接服务器
    </h2>
    <ol>
     <li>
      <p>
       新建会话
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3de0c7fa9428782f21222e444b34f2bf.png"/>
      </p>
     </li>
     <li>
      <p>
       输入名称、主机地址以及端口号 点击确定
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e1bec6846f4110a830a9dc11029df7dc.png"/>
      </p>
     </li>
     <li>
      <p>
       选中新建的会话点击连接
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3d5c360c862da19326c9cd0060555db.png"/>
      </p>
     </li>
     <li>
      <p>
       接受并保存主机密钥
      </p>
     </li>
     <li>
      <p>
       输入用户 输入密码
      </p>
     </li>
     <li>
      <p>
       连接成功
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e6d60ff0c3c4a76152ea32c561f187fb.png"/>
      </p>
     </li>
    </ol>
    <p>
     接着就可以直接输入命令行执行
    </p>
    <h2>
     <a id="xftp_21">
     </a>
     xftp连接服务器
    </h2>
    <p>
     点击红框所示图标，自动启动xftp
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0bedcafb65669b61066c25f8221f7910.png">
      <br/>
      即可以在本地与服务器之间进行文件传输（直接拖拽即可）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b22aa96bee22d86c257f04544596f8fd.png">
       <br/>
       后续也可以点击红框图标查看已连接过的服务器快速进行连接
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b78b90b86c48590eb7004e193b6f5ad5.png"/>
      </img>
     </img>
    </p>
    <h2>
     <a id="vscode_28">
     </a>
     vscode连接服务器
    </h2>
    <ol>
     <li>
      下载安装Remote -SSH 插件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/884a59b6195cb79ed29e7ab51a162684.png"/>
     </li>
     <li>
      如图所示点击远程连接图标以及SSH TARGETS
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bfab84d638ac038096d81769cdab539f.png"/>
     </li>
     <li>
      点击齿轮
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ed5422fc7523cb730f1b01228aa10f2.png"/>
     </li>
     <li>
      点击第一个config
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6c288984a239a2243716ee80e1e60e20.png"/>
     </li>
     <li>
      按格式输入要连接的服务器的相关信息，包括自定义的名称、主机地址、端口号以及登录用户
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac4637e99b783d75b06238939d303d8f.png"/>
     </li>
     <li>
      保存以后左边SSH TARGETS便会显示这个服务器，点击红框图标建立连接
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8b154cbde02f0b5903bae4e27d1aa7ea.png"/>
     </li>
     <li>
      选择服务器平台 一般是Linux
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4e264cd7ad5193528ea35618511776f4.png"/>
     </li>
     <li>
      输入yes 并输入密码
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/73e124571c8f8f1b0e4fd337120c4d24.png"/>
      <br/>
      注意输入密码的时候是没显示的 输入完之后回车就行
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b6c8d0cde95c2f88a9c83a9794e3304f.png"/>
     </li>
     <li>
      回车后会下载相关数据 下载完以后点击红框的文件图标
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4e23b846d44256485c29a2603211fb1.png"/>
     <br/>
     点击打开文件夹
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0ac65eabbd3697c2d00abe5c1cef776c.png"/>
     <br/>
     选择要打开的文件夹并确定
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0da0c49c324ffe607e25c93e235b9886.png"/>
     <br/>
     再次输入密码
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1a64785d14ff1c95b5148e6ebd93e7c1.png"/>
     <br/>
     即可查看文件并直接修改啦
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/07e7e0680c42520ecc547fd9ce06a919.png"/>
     <br/>
     这样我们就可以直接通过vscode修改服务器端的代码，而不用在本地修改以后再覆盖服务器端文件。
    </p>
