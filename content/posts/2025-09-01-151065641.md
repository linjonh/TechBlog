# 查看服务器设备是否为物理机



有些情况下要确认机器是否为物理机还是虚拟机，以安排对应的事项可以通过命令判断

```shell
#查看系统 CPU 硬件信息（物理机通常显示CPU厂商型号，inter等）

lscpu

# 查看系统硬件产品名称（物理机通常显示服务器厂商型号，如 Dell、HPE 等）

sudo dmidecode -s system-product-name

# 查看硬件制造商（物理机显示真实厂商，虚拟机显示虚拟化厂商）

cat /sys/class/dmi/id/sys_vendor

```
