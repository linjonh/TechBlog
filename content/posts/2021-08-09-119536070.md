      sdb和sdc是分别作为cinder服务和swift服务的存储磁盘。

     <strong>
      （这个一定要打上√）
     </strong>

     <strong>
      （可以选择不打√）
     </strong>
     <br/>
     第三项会提升io性能，但需要物理机支持io虚拟化，对于bios和主板、CPU有一定要求。（
     <strong>
      可以选择不打√）

     <h4>
      3、配置网络、主机名
     </h4>
    </div>
    <p>
    </p>
    <p>
     （1）controller节点
     <br/>
     #vi /etc/sysconfig/network-scripts/ifcfg-en*（具体的网口）文件
     <br/>
     配置网络（第一块网卡）：
    </p>
    <p>
     改：
     <br/>
     ONBOOT=yes
     <br/>
     BOOTPROTO=static
    </p>
    <p>
     添：
     <br/>
     IPADDR=192.168.100.10
     <br/>
     PREFIX=24
     <br/>
     GATEWAY=192.168.100.1
    </p>
    <p>
     配置网络（第二块网卡）：
     <br/>
     #vi /etc/sysconfig/network-scripts/ifcfg-en*（具体的网口）文件
     <br/>
     改：
     <br/>
     ONBOOT=yes
     <br/>
     BOOTPROTO=static
    </p>
    <p>
     添：
     <br/>
     IPADDR=192.168.200.10
     <br/>
     PREFIX=24
    </p>
    <p>
     重启网卡
     <br/>
     #systemctl restart NetworkManager
    </p>
    <p>
     配置主机名：
     <br/>
     #hostnamectl set-hostname controller
     <br/>
     按ctrl+d 退出 重新登陆
    </p>
    <p>
     （2）compute 节点
    </p>
    <p>
     配置网络（第一块网卡）：
     <br/>
     #vi /etc/sysconfig/network-scripts/ifcfg-en*（具体的网口）文件
     <br/>
     改：
     <br/>
     ONBOOT=yes
     <br/>
     BOOTPROTO=static
    </p>
    <p>
     添：
     <br/>
     IPADDR=192.168.100.20
     <br/>
     PREFIX=24
     <br/>
     GATEWAY=192.168.100.1
    </p>
    <p>
     配置网络（第二块网卡）：
     <br/>
     #vi /etc/sysconfig/network-scripts/ifcfg-en*（具体的网口）文件
     <br/>
     改：
     <br/>
     ONBOOT=yes
     <br/>
     BOOTPROTO=static
    </p>
    <p>
     添：
     <br/>
     IPADDR=192.168.200.20
     <br/>
     PREFIX=24
    </p>
    <p>
     重启网卡
     <br/>
     #systemctl restart NetworkManager
    </p>
    <p>
     配置主机名：
     <br/>
     #hostnamectl set-hostname compute
    </p>
    <p>
     按ctrl+d 退出 重新登陆
    </p>

     <h4>
      6、配置yum源
     </h4>
     <br/>
     【controller】
     <br/>
     #rm -rf /etc/yum.repos.d/*
     <br/>
     #cd /etc/yum.repos.d/
     <br/>
     #vi centos.repo
    </div>
    <p>
    </p>
    <p>
     增：
     <br/>
     [centos]
     <br/>
     name=centos
     <br/>
     baseurl=file:///opt/centos
     <br/>
     gpgcheck=0
     <br/>
     enabled=1
     <br/>
     [iaas]
     <br/>
     name=iaas
     <br/>
     baseurl=file:///opt/iaas-repo
     <br/>
     gpgcheck=0
     <br/>
     enabled=1
    </p>
    <p>
     【compute】
    </p>
    <p>
     #mv /etc/yum.repos.d/* /opt/
     <br/>
     #cd /etc/yum.repos.d/
     <br/>
     #vi centos.repo
     <br/>
     增：
     <br/>
     [centos]
     <br/>
     name=centos
     <br/>
     baseurl=ftp://192.168.100.10/centos
     <br/>
     gpgcheck=0
     <br/>
     enabled=1
     <br/>
     [iaas]
     <br/>
     name=iaas
     <br/>
     baseurl=ftp://192.168.100.10/iaas-repo
     <br/>
     gpgcheck=0
     <br/>
     enabled=1
    </p>

    <div class="toc">
     <h4>
      7、挂载iso镜像文件
     </h4>
    </div>
    <p>
    </p>
    <p>
     [root@controller ~]# mkdir /opt/centos
     <br/>
     [root@controller ~]# mount -o loop CentOS-7-x86_64-DVD-1804.iso /mnt/
     <br/>
     [root@controller ~]# cp -rvf /mnt/* /opt/centos/
     <br/>
     [root@controller ~]# umount /mnt/
    </p>
    <p>
     [root@controller ~]# mount -o loop chinaskills_cloud_iaas.iso /mnt/
     <br/>
     [root@controller ~]# cp -rvf /mnt/* /opt/
     <br/>
     [root@controller ~]# umount /mnt/
    </p>
    <p>
     <strong>
      挂载一定要这样挂，不能直接挂载到opt/centos和/opt/，不然后期可能会出现一些bug
     </strong>
    </p>
    <p>
    </p>
    <div class="toc">
     <h4>
      8、搭建ftp服务器，开启并设置自启
     </h4>
     <br/>
     [root@controller ~]# yum install vsftpd -y
     <br/>
     [root@controller ~]# vi /etc/vsftpd/vsftpd.conf
     <br/>
     添加anon_root=/opt/
     <br/>
     保存退出
    </div>
    <p>
    </p>
    <p>
     [root@controller ~]# systemctl start vsftpd &amp;&amp; systemctl enable vsftpd
    </p>

    <div class="toc">
     <h4>
      9、配置防火墙和Selinux
     </h4>
     <br/>
     【controller和compute都要操作】
    </div>
    <p>
    </p>
    <p>
     关闭防火墙并设置开机不自启
     <br/>
     #systemctl stop firewalld.service
     <br/>
     #systemctl disable firewalld.service
    </p>
    <p>
     #setenforce 0
     <br/>
     设置SELinux 成为permissive模式 临时关闭selinux
    </p>
    <p>
     写入配置文件
     <br/>
     #vi /etc/selinux/config
     <br/>
     改：
     <br/>
     SELINUX=permissive
    </p>
    <p>
     永久关闭selinux
    </p>
    <p>
     <strong>
      以下是对上面的操作做个验证：
     </strong>
     <br/>
     在文件资源管理器中输入ftp://192.168.100.10/ 看看有没有这个页面
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/16089ed89b81aac1ddfc73d525d73094.png"/>
    </p>
    <p>
     <strong>
      清除缓存，验证yum源
     </strong>
     <br/>
     #yum clean all
     <br/>
     #yum repolist（
     <strong>
      这个最好在controller和compute中都验证一下）
     </strong>
    </p>

     <h4>
      10、编辑环境变量（controller和compute节点都要操作）
     </h4>
    </div>
    <p>
    </p>
    <p>
     编辑文件/etc/xiandian/openrc.sh,此文件是安装过程中的各项参数，根据每项参数上一行的说明及服务器实际情况进行配置。

     HOST_PASS=000000
     <br/>
     HOST_NAME=controller
     <br/>
     HOST_IP_NODE=192.168.100.20
     <br/>
     HOST_PASS_NODE=000000
     <br/>
     HOST_NAME_NODE=compute
     <br/>
     network_segment_IP=192.168.100.0/24
     <br/>
     RABBIT_USER=openstack
     <br/>
     RABBIT_PASS=000000
     <br/>
     DB_PASS=000000
     <br/>
     DOMAIN_NAME=demo
     <br/>
     ADMIN_PASS=000000
     <br/>
     DEMO_PASS=000000
     <br/>
     KEYSTONE_DBPASS=000000
     <br/>
     GLANCE_DBPASS=000000
     <br/>
     GLANCE_PASS=000000
     <br/>
     NOVA_DBPASS=000000
     <br/>
     NOVA_PASS=000000
     <br/>
     NEUTRON_DBPASS=000000
     <br/>
     NEUTRON_PASS=000000
     <br/>
     METADATA_SECRET=000000
     <br/>
     <strong>
      INTERFACE_IP=192.168.100.10/192.168.100.20（controllerIP/computeIP）

     SWIFT_PASS=000000

     HEAT_PASS=000000

     KURYR_PASS=000000
