         微软总部面试归来
         <br/>
         从北京回来已有几天了，呵呵，当时就想马上写个面经，可是后来要忙发表论文什么的，没写，现在补上。将此面经献给武汉理工所有将来可能去微软面试的师弟师妹，祝你们面试成功，进入微软。
         <br/>
         我的这份面经其实不完美，因为微软HR没有给我OFFER，而是用了很委婉的语言：“回去等消息”。所以我就把我的整个微软过程记录下来，给大家一种借鉴吧。11.9号做飞机回来后，碰巧又参加了武汉理工的一个微软创新杯座谈会，当时微软公司的高校负责人问我面得怎么样，呵呵，为了维护武汉理工的尊严，俺说：“还可以吧！”
         <br/>
         微软历程：
         <br/>
         1. 微软9月28日宣讲后投简历
         <br/>
         2. 参加笔试  10月16日
         <br/>
         3. 电话面试  11.12
         <br/>
         4. 报销机票去北京微软中国――希格码大厦进行终面。12.8
         <br/>
         如果三年前我去找工作，估计连个小公司，开个600块让我做开发，那我都会开心死了，可惜我那时也这种公司都没机会。因为那时的我学的交通学院船舶工程专业，还没编过上百行的代码，不懂C,不懂开发，这就是那个三年前真真实实的我。那时我渴望得到和计算机专业学生一样的学习环境。(在给导师带实验课的时候，我却发现其实不少计算机专业的本科学生却也和非计算机专业学生一样，连基本代码都不会写，真是国家计算机教育的悲哀！)
         <br/>
         但是，我依旧热爱计算机编程，依旧对梦想永不放弃，依旧乐观，我坚信，只要给我和别人一样锻炼的机会，我会和别人一样出色！
         <font color="#ff0000">
          于是，后来我考研换成了自己喜爱的计算机专业，其中研二算是过得比较充实，所以遇到了蛮多公司的笔和面。有一篇记载我读研教训和经验的帖子，希望也能给将要读研的师弟师妹一些参考。大家有兴趣可以点机－》
          <br/>
          <a href="http://forum.whut.edu.cn/bbs/article.aspx?topicid=232439&amp;forumid=10608&amp;page=1" rel="nofollow noopener noreferrer" target="_blank">
           <font color="#002c99">
            http://forum.whut.edu.cn/bbs/article.aspx?topicid=232439&amp;forumid=10608&amp;page=1
           </font>
          </a>
         </font>
         <br/>
         一、起步，华科宣讲后投简历
         <br/>
         华科的宣讲我挤了进去，挨着微软研究院附院长洪小文博士和工程院附院长张益肇博士在讲台旁听宣讲，我在会上问了他们两个问题，我用的是JAVA，微软开发用的是C/c++，请问微软给我机会么？我来自武汉理工，开发过XXX, XXX, 有两项技术专利申请，微软筛选简历会根据名校教育背景筛选么？当时可能问题比较长，华科的人都开始起嘘声了，不过后来两个月我用行动回击了他们的嘘声。其实，我认为在武汉，很多武汉理工的优秀生不比武大华科的差！接着我就回去网投，简历我把每个部门都投了：MSN, ATC,MSRA
         <br/>
         网投填的简历大家要注意，要把自己做的项目，参加全国比赛，含有对应专业的专利或者有创新的论文写上，这些指标很重要，另外，网投的简历不要低调，人人都高调，自己低调会在第一关找死的！当然有些企业很变态，有学校歧视，会在学校名称这关卡掉！这种企业如果有学校歧视，不去也罢！
         <br/>
         二、武大工学部参加笔试  10月16日
         <br/>
         参加笔试要带简历，而我没带，幸亏实验室的兄弟帮我火速打印送过来，谢谢jungle。考两个半小时，编程基本都是C/C++，要把数据结构，网络，C/C++看看，我本来做JAVA的，为了这些大公司特意在那个月突击了C++才对一些公司幸免过关，趋势在微软前一天考的,权当微软的模拟考了。把大公司当模拟考了，这是惨呀！！趋势月薪可是和微软差不多的哟。
         <br/>
         考试在同一个教室见到了以前同在一个公司兼职的武大读研GG，QQ昵称“至尊宝”，后来和我一样参加了电话面试，但运气不是很好，挂面。一般去有名的大公司笔面基本上都碰到了他，真是巧，呵呵，趋势，上海贝尔，北电，连群硕这种名气大工资少的也一起碰到了，还一起拿了OFFER。至尊宝是个蛮优秀的小伙，签了上海贝尔。
         <br/>
         三、 电话面试  11.12
         <br/>
         记得是周六，同学在用我手机玩游戏，突然来电话，问我方便么，是微软的，又问我有没有微软其他部门面我了，我说没，问我投的是什么，我说分别MSN, ATC,MSRA，接着他就说他是MSN的，
         <br/>
         我说等等，就把充电器插上，手机话筒打开，开始被面了，接着HR就开始说用英文，introduce yourself,  project，project 难点，还问如果经理给你一个任务，你是无法完成的，但有必须完成怎么办，我说，找别人帮助，他说别人很忙了，我说说服别人和我一起挑战这个任务，后来急了一下，就用中文补充，我会组一个团队挑战这个任务，确保任务完成!接着他问你为什么开发搜索引擎，我用了李开复演讲用过的标准发音：“I want to change the world!”,后来就面技术问题了，数组和联表的区别，c，c++开辟和释放内存的函数或操作符，线程与进程区别，怎么快速释放多个内存空间，（是不是栈和堆的本质），线程与进程区别一时没答上，同学就快速百度了一个答案，我边看边练，基本上靠速度迷惑他我没用电脑，呵呵。就这样最后问我下周有时间没，我说下周要去上海参加比赛，没时间，有的话也是24号以后或下个月才有空，他就重复了下11月不能去，OK, 24号以后才有空。
         <br/>
         <br/>
         四、 报销机票去北京微软中国――希格码大厦进行终面。12.8
         <br/>
         <br/>
         我向导师借了3千块，呵呵，人真穷，研二打工的积蓄都花完了，用1千七百多买了来回机票。人生第一次坐的飞机，没想到是微软请的客，踏出了机舱，我就感受到了北京-8度的冷。
         <br/>
         11：40我踏上了京城，兴奋极了！坐了个机场大巴直接去中关村，到了北航的门口下，北航离希格码大厦很近，我先问了问旁边的5星宾馆，一晚1200，靠，借的路费不够，而且没进微软，就先不要花他太多吧，虽然报销宾馆住宿，：），于是我找了个近的北航招待所。
         <br/>
         下午3点面试，我提前去了，第一眼看到希格码大厦外观，觉得不是很豪华，不想一个IT老大的亚洲研发总部，上了希格码大厦六楼侧厅，发现已经有人在等了，呵呵，也是面微软的，问了一下，一个北京理工，两个清华的，我和北理坐在一边，两个清华GG坐在另一边。一个清华GG好像不知道武汉理工，以为她是华中理工，我和他解释了一下，那个原来的理工已经叫华科了。清华GG和我应聘的是开发,北理聘的是测试。我问清华GG清华来的这里（最终面）多不多，他说不是很多。
         <br/>
         3点．HR出现了，要我们去保安那签个名，我在签名的时候看了看，上午是中科院和清华的好像有10个左右，HR还给我一个报销表单，说回去以后贴好发票寄过来就可以了，尽量把能报的都报上去。微软HR态度很好，感觉希望每个人都能进微软那样的和我们招呼。
         <br/>
         HR很给大家点了下名，握了握手，发了个胸牌interviewer, 其中和一个清华GG特别聊了几句，说你在对面那家公司做了多久，做得怎么样。看来清华GG对希格码大厦很熟悉，读研就在附件干活了。感觉是比较牛的那种，HR接着又进微软门里进行安排了。
         <br/>
         我问了那个小“牛”，你们导师应该比较多项目吧，还要出去工作么，他说导师的也做，外面干活也干。我又问了下微软月薪大概会有多少，他说1W多吧。
         <br/>
         过了一会，HR把我们四个召集起来说流程: 等下面试都是in English, 有develop， test， design环节,我们会根据不同人的情况面试不同次数,希望大家不要紧张,好好发挥。接着，我们就被他逐个逐个领到不同的办公室，进行了面试。
         <br/>
         1. 第一面  华人面试官（估计是开发主管）
         <br/>
         听了清华GG的话，感觉微软的高薪就离我这么近，我还是变得紧张。
         <br/>
         我被领进了一个办公室，见到一个40来岁的面试官，他先介绍了自己的名字，接着就问我的中文名字，填写在那个英文表格里，接着就开始用英语聊了，首先是introduce yourself, 我介绍了，来自武汉理工，做了搜索引擎开发，参加了全国挑战杯比赛，是两项集体专利的发明人之一，做过手机游戏项目，爱好开发搜索引擎，他接着根据我的介绍又问我问题，你爱好开发搜索引擎那投了google没？我说我当时投的是中文简历，google只收英文简历，漏过去了（确实也是这样），接着他又问，google和microsoft同时要你，你怎么选择，我听了后，说我选择microsoft，因为他是第一个给我visit北京的公司（知遇之恩，当然要报答），然后他就比较满意的继续要我详细描述项目，可以用中文，这时我就把蹩脚的口语丢开了，直接说我们开发了一个翻译搜索引擎，叫译吧，他接着问域名是多少，可以访问么，我说可以，不过这几天服务器可能人多会当掉，我说我们是租一个800快的虚拟空间，他边问边记录在那个表格上，接着又问一些项目的细节技术难点，他偶尔看表，搞了估计40分钟，他最后问我你用JAVA开发搜索引擎和游戏的，对C,C++了解怎么样，我就老实的说，对c还掌握，对C++熟悉，但就没做过大项目，接着，他给我一张纸，要我在15分钟内写出一个程序，上面写了一个英文题目，是关于在一个长的字符串中找出数字串变成数字返回，当时我接过这个英文题目，不知为何变得很紧张，毕竟那张桌很近，在他眼皮低下我头脑一片空白，他不时的看了看我，不时的把笔记本电脑打开上译吧去试用，我只写了个开头，他说时间到了，说有思路么，我就说了下思路，说能不能在给我两分钟我继续把他写出来，他说时间紧，还要面别人，说不要紧的，要我出去找一下HR吧，我还锲而不舍的在想那个问题，他就改说他去找HR，要我在office等，接着出去找带路的HR。我在office，掏出了自己的草稿纸，接着有开始写代码，等他回来的时候，我写好了这个函数，把我的草稿纸递给了他，跟着HR走了（但在走出微软希格码大厦后，我又回想我写的有bug）。知道发挥不好后，接着我就光脚的不怕穿鞋，心想剩下的时间就只为武汉理工的尊言继续奋斗吧，不想这个高薪OFFER了。
         <br/>
         2. 第二面  华人面试官（估计是测试主管）
         <br/>
         进了另一个办公室，见到的也是个40来岁的面试官，他看了看我的简历，问我的第一句话竟然是：“你是清华的吧”。可能他今天面多了清华GG，给我第一句话居然是这个，我用坚毅的眼神望着他，说：“不是，我是武汉理工的”。他没有用英文和我交流，全部都讲中文，这样我奇怪，也让我轻松，我心想难道第一面已经记录了我的oral english不强的特点，他就不在为难我？ 接着他很友善的让我选择讲项目的方式，说白板和纸都可以，我就拿着大笔在白板上讲课一样的把项目讲了一遍。他边作记录边问专业问题，我想个讲师一样边讲解，边比划。然后和第一个面试官不同的是问我开发的每个系统有多少行代码，我分别写了多少行代码（靠，我哪记得呀，但我发现原来这时外企很看重的指标）我说哪个系统有40M,我写了30页。哪个系统又是多少，说个大概。问答蛮久，最后他说你用JAVA做开发的，那就写个JAVA程序吧，4分钟，随便写个系统用的算法。我说时间太少了，我就写个排序吧。接着2分钟把选择排序写个他，他让我解释，我说了一下，他说马马虎虎吧。让我在OFFICE等HR来领我去面下一个。
         <br/>
         3. 第三面  美国面试官（估计是设计的负责人）
         <br/>
         等了一会，HR带我休息间，带另一个人走，又在带我去另一个办公室，面试官是个lady，有点象我们研一的外教Rose, 这次铁定全english，HR和老外聊得太快了，以至我没听清面试官的名字，到了这面我已变得不紧张了，步骤也是先introduce myself，于是“我来自武汉理工”又在微软响了一边！接着再描述project detail。老外问我哪些搜索专利的状态，我说在审批中，因为一个专利审下来要花一年甚至更长时间，接着说你设计开发过游戏，下面来讨论一下，假如我是客户想要开发一款游戏，你是开发设计者，你怎么细化需求（requirement），让客户更准确表达出他的意思，然后，就给了一个游戏开发的主题，是帮助小孩阅读识字的，我说着，说着就忘词了，就大胆的说可以给我笔和纸，就在纸上有比划来，比划去，把游戏界面和功能给她逐个逐个细化，聊了蛮久，老外最后问，说你有什么free question？可以问，我就问她觉得微软怎么样？MSN搜索部在希格码大厦规模怎么样呀？第一次很老外聊这面专业，这么久，估计这个聊了1个小时吧，接着老外就又到外面找HR。我看了一下手机18:00多了，微软一般18:00 下班的，估计可能要面完了，果然，HR过了说今天就面完了，问我感觉怎么样，我说很紧张，没发挥好，他笑着安慰我，说：“说不定别人也发挥不好，回去等消息,两周后会有结果”，接着带我走出了六楼的微软开发部。
         <br/>
         整个面试我在微软开发部的格子间差点迷路了，中间在休息间等的时候坐在了宣讲曾经提到过的休息间里，里面很多沙发和玻璃桌，环境挺不错的，我一个人在空荡荡的休息，心想，没想到我居然来到在宣讲中的希格码大厦休息厅，传说中微软牛人休息和讨论的地方！
         <br/>
         出了六楼的门，我又把胸牌还给保安，在登记纸上写上离开时间，我看到和我一起的清华小“牛”还没签名走，估计offer属他了，同时我又看到了我的下面有新的面试名单，是下午4：00开始的，单位写的都是北大，我翻了一下上一页，有北理工，清华，计算所的，没看到武汉的，可能这两天在北京希格码面MSN的估计武汉就我一个了。不过回来后听说华科武大今年有进ATC的, 进了微软的才是牛人呀，我只是来微软旅游的，呵呵，我在希格码大厦出来，看到几个老外和几个GG一块象是边聊技术边上去microsoft，觉得希格码大厦楼的外观不怎么牛，但里面的人真是感觉比较牛。
         <br/>
         微软，我相信还是会再次登临的，但下次登临，我用什么身份来呢……活着，就是一柄刀，只有不断的拼杀，才能印证自身的存在！
