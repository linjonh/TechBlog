     云创智城YunCharge 新能源-云快充协议模拟器使用手册
    </h3>
    <p>
     <strong>

    <h3>
     <a id="1__3">
     </a>
     1. 介绍
    </h3>
    <p>
     云快充协议模拟器用于测试和调试充电设备的通讯协议，实现设备的连接、数据交互以及状态模拟。本手册将指导用户如何正确配置和使用该模拟器。
     <br/>
     <img alt="模拟器" src="https://i-blog.csdnimg.cn/direct/1013f2cff47c49c18a8dd98a9a64a76e.png"/>
    </p>
    <h3>
     <a id="2__8">
     </a>
     2. 进入模拟器
    </h3>
    <p>
     2.1 访问管理后台
    </p>
    <p>
     在浏览器中输入云快充后台管理系统的网址，并使用管理员账号登录。
    </p>
    <p>
     2.2 进入模拟器
    </p>
    <p>
     在左侧导航栏，找到 “虚拟桩模拟器” 并点击进入，加载协议模拟器界面。
     <br/>
     <img alt="虚拟桩模拟器" src="https://i-blog.csdnimg.cn/direct/7bde461c84bc43f9b482c762c93b2bcc.png"/>
    </p>
    <h3>
     <a id="3__19">
     </a>
     3. 连接设备
    </h3>
    <p>
     3.1 设置连接参数
    </p>
    <p>
     在 “连接设置” 页面，填写以下信息：
     <br/>
     • 连接地址：tcp://127.0.0.1:5003
     <br/>
     • 设备编号：输入模拟设备的唯一编号，如 24102413849856
     <br/>
     • 设备类型：
     <br/>
     • 选择 直流电 或 交流电
     <br/>
     • 设备型号选择 1030（示例）
     <br/>
     <img alt="连接配置" src="https://i-blog.csdnimg.cn/direct/858ce41e19e944b99a71231a3f4899e9.png"/>
    </p>
    <p>
     3.2 启动连接
    </p>
    <p>
     填写完毕后，点击 “连接” 按钮，等待系统建立连接。成功后，状态会显示 “已连接”。
    </p>
    <h3>
     <a id="4__35">
     </a>
     4. 设备状态与日志监控
    </h3>
    <p>
     4.1 设备状态查看
    </p>
    <p>
     连接成功后，系统会显示：
     <br/>
     • 设备可用性：Operative
     <br/>
     • 设备状态：可用/不可用
     <br/>
     • 连接状态：已连接（绿色）
    </p>
    <p>
     4.2 交互日志
    </p>
    <p>
     在 “交互日志” 区域，可以查看设备与服务器的通讯数据，格式为 JSON。例如：
    </p>
    <p>
     {
     <!-- -->
     <br/>
     “check”: “0471”,
     <br/>
     “command”: “55”,
     <br/>
     “data”: “241024138498567A781917E90319”,
     <br/>
     “serialNum”: “0000”
     <br/>
     }
    </p>
    <p>
     日志包含：
     <br/>
     • command：指令码
     <br/>
     • data：设备发送的数据
     <br/>
     • serialNum：序列号
    </p>
    <p>
     日志能帮助开发者分析设备通讯情况，排查异常。
    </p>
    <h3>
     <a id="5__62">
     </a>
     5. 断开连接
    </h3>
    <p>
     如需断开设备，可点击 “断开连接” 按钮。断开后，设备状态将变为 “未连接”。
    </p>
    <h3>
     <a id="6__66">
     </a>
     6. 其他功能
    </h3>
    <p>
     6.1 设备管理
    </p>
    <p>
     在 “设备管理” 页面，可以查看所有模拟设备，并进行配置。
    </p>
    <p>
     6.2 订单管理
    </p>
    <p>
     可用于测试充电订单的创建、支付和结算流程。
    </p>
