      HBuilder将uniapp项目运行到微信开发者工具进行小程序调试
     </p>
    </blockquote>
    <h3>
     <a id="_2">
     </a>
     前言
    </h3>
    <ul>
     <li>
      有时候我们的项目开发是使用uniapp，而最终是要发布到微信小程序的。
     </li>
     <li>
      在开发uniapp项目过程中，我们可以使用HBuilderX运行到微信小程序模拟器（微信开发者工具），一边写代码一边调试、预览效果。
     </li>
    </ul>
    <h3>
     <a id="_5">
     </a>
     准备
    </h3>
    <p>
     电脑安装以下软件
    </p>
    <ul>
     <li>
      HbuilderX（写代码）
     </li>
     <li>
      微信开发者工具（调试，预览）
     </li>
    </ul>
    <h3>
     <a id="_11">
     </a>
     具体步骤
    </h3>
    <h3>
     <a id="1__12">
     </a>
     1. 小程序运行路径配置
    </h3>
    <ul>
     <li>
      打开HbuilderX里面的【工具】→【设置】
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f3b91c20f3972a414eec0e8a3a7ffeb3.png"/>
     </li>
     <li>
      在运行配置里面，把微信开发者工具的安装路径填上。
      <br/>
      <code>
       注意!!
      </code>
      ：这里路径要选到安装路径的最后一级，把【微信开发者工具】文件夹也选上
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1889727ce03bb103f2fa3f8cc9c41396.png">
       <br/>
       如果只配置了这个，运行到小程序模拟器的时候会提示服务端口没有打开，这里的服务端口指的是微信开发工具里面的服务端口
      </img>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/98ecb71c8d605e6ec6ca195e66ef8e84.png"/>
    </p>
    <h3>
     <a id="2__22">
     </a>
     2. 打开服务端口
    </h3>
    <p>
     我们现在只需在微信开发者工具中的设置→安全 里面 把服务器端口打开
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ba21544d3715b26ec39d85e731ed0f6c.png">
      <br/>
      我们再进行运行编译
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/802e8cc10a909029fdf579a7bfc1a343.png">
       <br/>
       编译成功后，会自动打开微信开发者工具，这时候就能看到我们的uniapp在小程序的运行效果了。在HBuilderX中修改文件并保存，会自动刷新微信模拟器，可以一边写代码一边调试了。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/65f51fd4f7a3f9a70496078158f6c2dc.png"/>
      </img>
     </img>
    </p>
