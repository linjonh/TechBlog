    <h2>
     <a id="kafkazookeeperzookeeper_0">
     </a>
     因为kafka是基于zookeeper做的，所以必须要有zookeeper
    </h2>
    <h2>
     <a id="Zookeeper_1">
     </a>
     一、Zookeeper
    </h2>
    <h3>
     <a id="1_2">
     </a>
     1.拉取镜像
    </h3>
    <pre><code class="prism language-powershell">docker pull zookeeper:3<span class="token punctuation">.</span>7<span class="token punctuation">.</span>0
</code></pre>
    <h2>
     <a id="2_7">
     </a>
     2.运行
    </h2>
    <pre><code class="prism language-powershell">docker run <span class="token operator">--</span>restart=always \
  <span class="token operator">--</span>log-driver json-file \
  <span class="token operator">--</span>log-opt max-size=100m \
  <span class="token operator">--</span>log-opt max-file=2  \
  <span class="token operator">--</span>name zookeeper <span class="token operator">-</span>p 2181:2181 \
  <span class="token operator">-</span>v <span class="token operator">/</span>etc/localtime:<span class="token operator">/</span>etc/localtime:ro \
  <span class="token operator">-</span>d zookeeper:3<span class="token punctuation">.</span>7<span class="token punctuation">.</span>0
</code></pre>
    <h2>
     <a id="3_17">
     </a>
     3.检查是否运行成功
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/37d273fd182d45dcb1c0707bf5d2097b.png"/>
    </p>
    <pre><code class="prism language-powershell">docker exec <span class="token operator">-</span>it zookeeper <span class="token operator">/</span>bin/sh
zkCli<span class="token punctuation">.</span>sh <span class="token operator">-</span>server 192<span class="token punctuation">.</span>168<span class="token punctuation">.</span>101<span class="token punctuation">.</span>129:2181
</code></pre>
    <h2>
     <a id="Kafka_23">
     </a>
     二、Kafka
    </h2>
    <h3>
     <a id="1_24">
     </a>
     1.拉取镜像
    </h3>
    <p>
     wurstmeister是一个官方的镜像仓库直接拉失败了，所以包了一层，要是还不行，去阿里弄个镜像加速器配上
    </p>
    <pre><code class="prism language-powershell">docker pull wurstmeister/kafka
</code></pre>
    <p>
     就是这里
     <br/>
     https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e8d7746ca323466c8adc2028087fc672.png"/>
    </p>
    <h3>
     <a id="2__32">
     </a>
     2. 创建挂载目录好日志
    </h3>
    <h4>
     <a id="21_33">
     </a>
     2.1创建挂载目录
    </h4>
    <pre><code class="prism language-powershell">mkdir <span class="token operator">-</span>p <span class="token operator">/</span>home/kafka/<span class="token punctuation">{<!-- --></span><span class="token keyword">data</span><span class="token punctuation">,</span>config<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="3_37">
     </a>
     3.启动
    </h3>
    <p>
     <strong>
      注意：192.168.101.129这个地址需要替换成你自己的服务器IP
     </strong>
    </p>
    <pre><code class="prism language-powershell">docker run <span class="token operator">--</span>privileged=true \
<span class="token operator">--</span>log-driver json-file \
<span class="token operator">--</span>log-opt max-size=100m \
<span class="token operator">--</span>log-opt max-file=2 \
<span class="token operator">--</span>name kafka <span class="token operator">-</span>p 9092:9092 \
<span class="token operator">-</span>e KAFKA_BROKER_ID=0 \
<span class="token operator">-</span>e KAFKA_ZOOKEEPER_CONNECT=192<span class="token punctuation">.</span>168<span class="token punctuation">.</span>101<span class="token punctuation">.</span>129:2181/kafka \
<span class="token operator">-</span>e KAFKA_ADVERTISED_LISTENERS=PLAINTEXT:<span class="token operator">/</span><span class="token operator">/</span>192<span class="token punctuation">.</span>168<span class="token punctuation">.</span>101<span class="token punctuation">.</span>129:9092 \
<span class="token operator">-</span>e KAFKA_LISTENERS=PLAINTEXT:<span class="token operator">/</span><span class="token operator">/</span>0<span class="token punctuation">.</span>0<span class="token punctuation">.</span>0<span class="token punctuation">.</span>0:9092 \
<span class="token operator">-</span>e ALLOW_PLAINTEXT_LISTENER=yes \
<span class="token operator">-</span>e KAFKA_HEAP_OPTS=<span class="token string">'-Xms512M -Xmx4G'</span> \
<span class="token operator">-</span>v <span class="token operator">/</span>home/kafka/<span class="token keyword">data</span>:<span class="token operator">/</span>wurstmeister/kafka/<span class="token keyword">data</span> \
<span class="token operator">-</span>v <span class="token operator">/</span>home/kafka/config:<span class="token operator">/</span>wurstmeister/kafka/config \
<span class="token operator">-</span>v <span class="token operator">/</span>etc/localtime:<span class="token operator">/</span>etc/localtime \
<span class="token operator">-</span>d wurstmeister/kafka
</code></pre>
    <h3>
     <a id="4_56">

     4.测一下
