    <h2>
     <a id="gitssh_gitcodessh_0">
     </a>
     【git】ssh配置提交 gitcode-ssh提交
    </h2>
    <p>
     之前一直用的是gitee和阿里云的仓库，前两天想在gitcode上面备份一下我的打洞代码和一些资料 就直接使用http克隆了下来 。
     <br/>
     在提交的时候他一直会让我输入账号和密码，但是我之前根本没有设置过这个，根本没法搞，后来试了登陆的账号和密码也不是，找了半天也找不到，后来还是干脆配置一下ssh算了
     <br/>
     在这里设置：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/63a47b0a26c14b1da5ccb1ed37bcf8a5.png">
      <br/>
      我用的是RSA类型
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/be7e53c42fb242ccb194bbf6545e0933.png"/>
     </img>
    </p>
    <h4>
     <a id="1_7">
     </a>
     1.输入生成密钥的命令
    </h4>
    <p>
     运行以下命令，将 your_email@example.com 替换为您的邮箱地址：
    </p>
    <pre><code class="prism language-bash">ssh-keygen <span class="token parameter variable">-t</span> rsa <span class="token parameter variable">-b</span> <span class="token number">4096</span> <span class="token parameter variable">-C</span> <span class="token string">"your_email@example.com"</span>
</code></pre>
    <p>
     -t rsa：指定密钥类型为 RSA。
    </p>
    <p>
     -b 4096：指定密钥长度为 4096 位（推荐）。
    </p>
    <p>
     -C “your_email@example.com”：添加注释，通常使用您的邮箱地址，便于识别密钥。
    </p>
    <h4>
     <a id="2_20">
     </a>
     2.选择密钥保存位置
    </h4>
    <p>
     命令执行后，您将看到以下提示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9f1c51e7d5f04648b9ccc8a21bb1c569.png"/>
    </p>
    <pre><code class="prism language-bash">Generating public/private rsa key pair.
Enter <span class="token function">file</span> <span class="token keyword">in</span> <span class="token function">which</span> to save the key <span class="token punctuation">(</span>/home/user/.ssh/id_rsa<span class="token punctuation">)</span>:
</code></pre>
    <h4>
     <a id="3__Enter_sshid_rsa_29">
     </a>
     3 按 Enter 键接受默认位置（~/.ssh/id_rsa）
    </h4>
    <p>
     如果需要自定义保存路径，可以输入新的路径，例如：~/.ssh/my_custom_key
    </p>
    <h4>
     <a id="4__33">
     </a>
     4 设置密钥密码（可选但推荐）
    </h4>
    <p>
     接下来，系统会提示您设置密码：
    </p>
    <pre><code class="prism language-bash">Enter passphrase <span class="token punctuation">(</span>empty <span class="token keyword">for</span> no passphrase<span class="token punctuation">)</span>:
Enter same passphrase again:
</code></pre>
    <p>
     输入一个安全的密码以保护您的私钥（建议使用强密码）
    </p>
    <p>
     如果不需要密码，直接按 Enter 键跳过
    </p>
    <h4>
     <a id="5__46">
     </a>
     5 确认密钥生成成功
    </h4>
    <p>
     如果操作成功，您将看到类似以下的输出：
    </p>
    <pre><code class="prism language-bash">Your identification has been saved <span class="token keyword">in</span> /Users/.ssh/id_rsa
Your public key has been saved <span class="token keyword">in</span> /Users/.ssh/id_rsa.pub

The key's randomart image is:

<span class="token operator">|</span>         <span class="token punctuation">..</span><span class="token punctuation">..</span>o<span class="token operator">==</span>B<span class="token operator">|</span>
<span class="token operator">|</span>        <span class="token punctuation">..</span>o.o.*O<span class="token operator">=</span><span class="token operator">|</span>
<span class="token operator">|</span>        .<span class="token operator">=</span> o.E+*+<span class="token operator">|</span>
<span class="token operator">|</span>        o.+ <span class="token punctuation">..</span>. o<span class="token operator">|</span>
<span class="token operator">|</span>        S. <span class="token punctuation">..</span>    <span class="token operator">|</span>
<span class="token operator">|</span>          o* o <span class="token builtin class-name">.</span> <span class="token operator">|</span>

    <p>
     私钥：~/.ssh/id_rsa（切勿泄露）。
    </p>
    <p>
     公钥：~/.ssh/id_rsa.pub（可上传到 GitCode）。
    </p>
    <h4>
     <a id="6__notepad__73">
     </a>
     6 使用 notepad 打开文本文件
    </h4>
    <p>
     切换到指定目录并且打开文件：
    </p>
    <pre><code class="prism language-bash">notepad id_rsa.pub
</code></pre>
    <p>
     适用于：只想用 记事本 打开 TXT 文件。
