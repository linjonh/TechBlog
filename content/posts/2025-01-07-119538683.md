    <h3>
     一、Nginx-RTMP服务器搭建
    </h3>
    <h4>
     <strong>
      1、下载
      <a href="http://nginx-win.ecsds.eu/download/" rel="nofollow">
       Nginx
      </a>
     </strong>
    </h4>
    <p>
     <img alt="" height="111" src="https://i-blog.csdnimg.cn/blog_migrate/6d6a796ed06756b46c3ed48bf9aa568c.png" width="264"/>
    </p>
    <p>
     该版本是带rtmp模块且编译过的nginx版本，可以直接使用
    </p>
    <p>
     下载完成后解压，解压后的目录命名为 nginx
    </p>
    <p>
    </p>
    <h4>
     2、下载
     <a href="https://github.com/arut/nginx-rtmp-module">
      Nginx的RTMP扩展包
     </a>
    </h4>
    <p>
     下载完成后解压，将解压后的文件夹放入 nginx 目录下
    </p>
    <p>
     <img alt="" height="401" src="https://i-blog.csdnimg.cn/blog_migrate/69f66d956491db5a81856e894bbc4de0.png" width="671"/>
    </p>
    <h4>
     3、修改配置文件
    </h4>
    <p>
     在conf目录下，新建一个文件“nginx.conf”，内容如下
    </p>
    <pre><code>worker_processes  1;
 
events {
    worker_connections  1024;
}

#RTMP服务
rtmp {
	server { 
		listen		1935;	#监听端口
		chunk_size	4096;	#数据传输块大小
		application live{	#创建名为"live"的应用
		   live on;
		}
    }
}
 
#HTTP服务，可以通过浏览器访问http://localhost/stat 或者 http://localhost:80/stat 查看服务器状态
http {
    include       mime.types;
    default_type  application/octet-stream;
    sendfile      on;
    keepalive_timeout  65;
	
    server {
        listen       80;
        server_name  localhost;
		
		location /stat {
			rtmp_stat all;
			rtmp_stat_stylesheet stat.xsl;
		}
 
		location /stat.xsl { 
			root ./nginx-rtmp-module-master/;	#rtmp拓展包目录
		}
 
        location / {
            root   html;
            index  index.html index.htm;
        }
 
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
    }
 
}                             </code></pre>
    <p>
     该文件主要是直播配置，录播配置、直播回放配置各有不同
    </p>
    <h4>
     4、启动服务器
    </h4>
    <p>
     在nginx目录下打开命令行界面，输入
     <strong>
      <span style="color:#0d0016;">
       nginx.exe -c conf\nginx.conf
      </span>
     </strong>
    </p>
    <p>
     启动后，可以在任务管理器中看到nginx进程
    </p>
    <p>
     <img alt="" height="43" src="https://i-blog.csdnimg.cn/blog_migrate/e697177f4bd86cb6f4d86fbe4cff10f0.png" width="388"/>
    </p>
    <h4>
     5、查看服务器状态
    </h4>
    <p>
     在浏览器地址栏输入
     <a href="http://localhost/stat" rel="nofollow">
      localhost/stat
     </a>
     ，可以看到服务器状态信息
    </p>
    <p>
     <img alt="" height="268" src="https://i-blog.csdnimg.cn/blog_migrate/29bebf4fa9651b48b856c390d600a525.png" width="1200"/>
    </p>
    <h4>
     6、其它ngnix命令
    </h4>
    <pre><code>nginx.exe -s stop    //快速终止服务器，可能不保存相关信息
nginx.exe -s quit    //完整有序停止服务器，保存相关信息
nginx.exe -s reload  //重新载入Nginx，当配置信息修改，需要重新载入这些配置时使用此命令</code></pre>
    <p>
    </p>
    <h3>
     二、OBS推流
    </h3>
    <p>
     官网下载地址：
     <a href="https://obsproject.com/zh-cn/download" rel="nofollow">
      https://obsproject.com/zh-cn/download
     </a>
    </p>
    <p>
     其它下载地址：
     <a href="http://www.pc6.com/down.asp?id=410276" rel="nofollow">
      http://www.pc6.com/down.asp?id=410276
     </a>
    </p>
    <p>
     <img alt="" height="684" src="https://i-blog.csdnimg.cn/blog_migrate/63c9379068482042d2101d999682608a.png" width="1081"/>
    </p>
    <p>
     大致步骤如图所示，分为四步
    </p>
    <h4>
     1、添加场景
    </h4>
    <p>
     软件默认有一个场景，我们可以不用再去添加，直接去步骤2添加来源即可
    </p>
    <h4>
     2、添加来源
    </h4>
    <p>
     可以添加显示器捕获、视频捕获、音频捕获、媒体源等，此处根据需要进行选择。
    </p>
    <p>
     本文将本地视频video.mp4作为媒体源进行推流
    </p>
    <p>
     <img alt="" height="612" src="https://i-blog.csdnimg.cn/blog_migrate/a0dfdd65b0833e82bc4b3275fdcb9051.png" width="722"/>
    </p>
    <h4>
     3、推流设置
    </h4>
    <p>
     <img alt="" height="780" src="https://i-blog.csdnimg.cn/blog_migrate/e1da863825ad16a8e4f1c6611ef1b731.png" width="983"/>
    </p>
    <p>
     服务：自定义
    </p>
    <p>
     服务器：  rtmp://127.0.0.1:1935/live
    </p>
    <p>
     串流秘钥： stream
    </p>
    <h4>
     4、开始推流
    </h4>
    <p>
     点击开始推流按钮
    </p>
    <h4>
     5、查看服务器状态
    </h4>
    <p>
     打开浏览器输入 localhost/stat 可以看到推流信息
    </p>
    <p>
     <img alt="" height="214" src="https://i-blog.csdnimg.cn/blog_migrate/6e09348c4870bb2b185b46ff6ab82513.png" width="692"/>
