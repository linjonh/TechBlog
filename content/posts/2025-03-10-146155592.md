     Golang ｜* 和 &amp;
    </h1>

    <p>
     在 Golang（Go 语言）中，
     <code>
      *
     </code>
     和
     <code>
      &amp;
     </code>
     主要用于指针操作，它们的作用如下：
    </p>
    <h4>
     <a id="1__2">
     </a>
     1.
     <code>
      &amp;
     </code>
     （取地址符）
    </h4>
    <p>
     <code>
      &amp;
     </code>
     用于获取变量的内存地址（即指针）。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main
<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    a <span class="token operator">:=</span> <span class="token number">10</span>
    p <span class="token operator">:=</span> <span class="token operator">&amp;</span>a  <span class="token comment">// 获取变量 a 的地址，p 是一个指向 a 的指针</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span><span class="token string">"a 的值:"</span><span class="token punctuation">,</span> a<span class="token punctuation">)</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span><span class="token string">"a 的地址:"</span><span class="token punctuation">,</span> <span class="token operator">&amp;</span>a<span class="token punctuation">)</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span><span class="token string">"p 的值（即 a 的地址）:"</span><span class="token punctuation">,</span> p<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      输出示例（地址因环境不同而变化）：
     </strong>
    </p>
    <pre><code>a 的值: 10
a 的地址: 0xc000014088
p 的值（即 a 的地址）: 0xc000014088
</code></pre>
    <hr/>
    <h4>
     <a id="2__27">
     </a>
     2.
     <code>
      *
     </code>
     （指针解引用）
    </h4>
    <p>
     <code>
      *
     </code>
     用于指针解引用，即通过指针访问指针指向的值。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main
<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    a <span class="token operator">:=</span> <span class="token number">10</span>
    p <span class="token operator">:=</span> <span class="token operator">&amp;</span>a    <span class="token comment">// p 是指向 a 的指针</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span><span class="token string">"p 指向的值:"</span><span class="token punctuation">,</span> <span class="token operator">*</span>p<span class="token punctuation">)</span> <span class="token comment">// 解引用 p，获取 a 的值</span>

    <span class="token operator">*</span>p <span class="token operator">=</span> <span class="token number">20</span>   <span class="token comment">// 通过指针修改 a 的值</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span><span class="token string">"修改后 a 的值:"</span><span class="token punctuation">,</span> a<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      输出：
     </strong>
    </p>
    <pre><code>p 指向的值: 10
修改后 a 的值: 20
</code></pre>
    <hr/>
    <h4>
     <a id="3___52">
     </a>
     3.
     <code>
      *
     </code>
     还用于声明指针类型
    </h4>
    <p>
     在声明指针变量时，
     <code>
      *
     </code>
     用于指定变量是指针类型。例如：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">var</span> p <span class="token operator">*</span><span class="token builtin">int</span>  <span class="token comment">// 声明一个 int 类型的指针变量 p</span>
</code></pre>
    <p>
     只有在解引用时，
     <code>
      *p
     </code>
     才表示指针指向的值。
    </p>
    <hr/>
    <h4>
     <a id="4__61">
     </a>
     4. 总结区别
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        符号
       </th>
       <th>
        作用
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         &amp;
        </code>
       </td>
       <td>
        取地址，获取变量的内存地址
       </td>
      </tr>
      <tr>
       <td>
        <code>
         *
        </code>
       </td>
       <td>
        解引用指针，获取指针指向的值，也用于声明指针类型
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     如果你对指针的理解还不太清楚，可以思考：
    </p>
    <ul>
     <li>
      <code>
       &amp;a
      </code>
      相当于 “我要 a 的地址”。
     </li>
     <li>
      <code>
       *p
      </code>
      相当于 “我要 p 指向的值”。
     </li>
    </ul>
    <p>
     希望这能帮助你理解
     <code>
      *
     </code>
     和
     <code>
      &amp;
     </code>
     的区别！
    </p>
