    <blockquote>
     <p>
      删除时，每删除一个元素，后边的元素都会左移一位，也就是下标会减1
     </p>
    </blockquote>
    <pre><code>		
	List&lt;String&gt; list = new ArrayList&lt;String&gt;();
   	list.add("a");
   	list.add("a");
   	list.add("b");
   	list.add("c");
   	list.add("d");
   	System.out.println("删除list前的size:"+list.size());
   	System.out.println("list的所有元素:"+list);
   	for (int i = 0; i &lt; list.size(); i++) {
   		if (list.get(i).equals("a")) {
   			list.remove(i);
   		}
   		
   	}
   	System.out.println("删除list前的size:"+list.size());
   	System.out.println("list的所有元素:"+list);
</code></pre>
    <blockquote>
     <p>
      输出：
      <code>
       删除list前的size:5 list的所有元素:[a, a, b, c, d] 删除list前的size:4 list的所有元素:[a, b, c, d]
      </code>
     </p>
    </blockquote>
    <hr/>
    <h3>
     <a id="_28">
     </a>
     删除时，下标自减
    </h3>
    <pre><code>		List&lt;String&gt; list = new ArrayList&lt;String&gt;();
		list.add("a");
		list.add("a");
		list.add("b");
		list.add("c");
		list.add("d");
		System.out.println("删除list前的size:"+list.size());
		System.out.println("list的所有元素:"+list);
		for (int i = 0; i &lt; list.size(); i++) {
			if (list.get(i).equals("a")) {
				list.remove(i);
				i--;
			}
			
		}
		System.out.println("删除list前的size:"+list.size());
		System.out.println("list的所有元素:"+list);
</code></pre>
    <blockquote>
     <p>
      <code>
       删除list前的size:5 list的所有元素:[a, a, b, c, d] 删除list前的size:3 list的所有元素:[b, c, d]
      </code>
     </p>
    </blockquote>
    <hr/>
    <h3>
     <a id="_53">
     </a>
     反向删除
    </h3>
    <pre><code>		List&lt;String&gt; list = new ArrayList&lt;String&gt;();
		list.add("a");
		list.add("a");
		list.add("b");
		list.add("c");
		list.add("d");
		System.out.println("删除list前的size:"+list.size());
		System.out.println("list的所有元素:"+list);

		for (int i = list.size()-1; i &gt;= 0; i--) {
			if (list.get(i).equals("a")) {
				list.remove(i);
			}
		}
		System.out.println("删除list前的size:"+list.size());
		System.out.println("list的所有元素:"+list);
		}
</code></pre>
    <blockquote>
     <p>
      <code>
       删除list前的size:5 list的所有元素:[a, a, b, c, d] 删除list前的size:3 list的所有元素:[b, c, d]
      </code>
     </p>
    </blockquote>
    <hr/>
    <h3>
     <a id="iterator_79">
     </a>
     使用迭代器删除（iterator）（推荐）
    </h3>
    <pre><code>	List&lt;String&gt; list = new ArrayList&lt;String&gt;();
		list.add("a");
		list.add("a");
		list.add("b");
		list.add("c");
		list.add("d");
		System.out.println("删除list前的size:"+list.size());
		System.out.println("list的所有元素:"+list);
	Iterator&lt;String&gt; iterator = list.iterator();
	while (iterator.hasNext()) {
		if (iterator.next().equals("a")) {
			iterator.remove();
		}
		
	}
</code></pre>
    <blockquote>
     <p>
      <code>
       删除list前的size:5 list的所有元素:[a, a, b, c, d] 删除list前的size:3 list的所有元素:[b, c, d]
      </code>
     </p>
    </blockquote>
    <blockquote>
     <p>
      iterator内部帮我们做了类似i-1的操作
     </p>
    </blockquote>
    <h3>
     <a id="list_103">

     赋值给新list

    <p>
     <em>
      <strong>
       注意：在进行普通for循环删除时，不要把list.size()抽离出去赋值给变量，然后用此变量做为for条件, 因为删除时，list.size()的值是会改变的，要把list.size作为for条件。
      </strong>
