# python离线安装

[Python Releases for Windows | Python.org](https://www.python.org/downloads/windows/ "Python Releases for Windows | Python.org")

下载包地址widows一般是64bit的包

下载完成后双击，在弹出的首个页面会看到下面的图

第一步：建议手动安装

第二步：
一定要勾选把版本加入到Path路径

![](https://i-blog.csdnimg.cn/direct/10cd0cbf4bff4e1681ab9d26a9618c0a.png)

然后就是无脑下一步，到这一步就可以选择安装文件的放的位置

![](https://i-blog.csdnimg.cn/direct/b276714a4faa4db29ee3e1c060c62c76.png)

然后install，基本就完成了

cmd验证

python --version

pip --version

关于卸载：先找到高级系统设置里把环境变量关于python的删除干净

然后卸载，卸载不掉的时候会有个提示看日志，点进去，把能看得到的文件路径内的文件删除干净

重启，在卸载，重复几遍就可以卸载干净
