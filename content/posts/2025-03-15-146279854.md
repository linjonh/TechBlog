    <h4>
     <a id="Swap__Linux__0">
     </a>
     <strong>
      Swap 分区在 Linux 系统中的作用
     </strong>
    </h4>
    <p>
     Swap 分区（交换分区）相当于虚拟内存，在物理内存（RAM）不够用时，系统会将一部分磁盘空间作为虚拟内存使用。主要作用如下：
    </p>
    <ul>
     <li>
      <strong>
       缓解内存不足
      </strong>
      ：当物理内存耗尽时，系统会将部分进程数据临时存放在 Swap 以防止系统崩溃。
     </li>
     <li>
      <strong>
       休眠（Hibernate）支持
      </strong>
      ：部分 Linux 发行版支持系统休眠（suspend-to-disk），需要足够的 Swap 来存储 RAM 中的数据。
     </li>
     <li>
      <strong>
       缓存和后台进程管理
      </strong>
      ：当某些进程长时间未被访问，内核可能会将其数据移入 Swap，以释放 RAM 给更活跃的任务。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="Swap__8">
     </a>
     <strong>
      Swap 分区的大小建议
     </strong>
    </h4>
    <p>
     Swap 分区的大小通常与系统 RAM 相关，推荐配置如下：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        物理内存（RAM）
       </th>
       <th>
        Swap 推荐大小（桌面系统）
       </th>
       <th>
        Swap 推荐大小（服务器）
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        小于 2GB
       </td>
       <td>
        物理内存的 2 倍
       </td>
       <td>
        物理内存的 2 倍
       </td>
      </tr>
      <tr>
       <td>
        2GB - 8GB
       </td>
       <td>
        物理内存的 1.5 倍
       </td>
       <td>
        物理内存的 1~2 倍
       </td>
      </tr>
      <tr>
       <td>
        8GB - 16GB
       </td>
       <td>
        物理内存的 1 倍
       </td>
       <td>
        物理内存的 1~2 倍
       </td>
      </tr>
      <tr>
       <td>
        16GB 以上
       </td>
       <td>
        4GB~8GB
       </td>
       <td>
        物理内存的 0.5~1 倍
       </td>
      </tr>
      <tr>
       <td>
        休眠模式
       </td>
       <td>
        至少等于 RAM 大小
       </td>
       <td>
        至少等于 RAM 大小
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     如果服务器是
     <strong>
      高性能数据库
     </strong>
     或
     <strong>
      高 I/O 负载
     </strong>
     应用（如 NoSQL），尽量减少 Swap，以避免磁盘 I/O 影响性能。
    </p>
    <hr/>
    <h4>
     <a id="_Swap_22">
     </a>
     <strong>
      如何创建和管理 Swap
     </strong>
    </h4>
    <h5>
     <a id="1__Swap__23">
     </a>
     <strong>
      1. 创建 Swap 分区（传统方式）
     </strong>
    </h5>
    <p>
     如果系统安装时没有创建 Swap 分区，可以手动创建：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 查看当前分区信息</span>
lsblk
<span class="token function">fdisk</span> <span class="token parameter variable">-l</span>

<span class="token comment"># 创建新的 Swap 分区（假设 /dev/sdb1 为空闲分区）</span>
<span class="token function">mkswap</span> /dev/sdb1  <span class="token comment"># 格式化为 swap</span>
<span class="token function">swapon</span> /dev/sdb1  <span class="token comment"># 启用 swap</span>

<span class="token comment"># 持久化配置（编辑 /etc/fstab）</span>
<span class="token builtin class-name">echo</span> <span class="token string">'/dev/sdb1 swap swap defaults 0 0'</span> <span class="token operator">&gt;&gt;</span> /etc/fstab
</code></pre>
    <h5>
     <a id="2__Swap__Swap__38">
     </a>
     <strong>
      2. 使用 Swap 文件代替 Swap 分区
     </strong>
    </h5>
    <p>
     如果磁盘已经分区完成，但需要增加 Swap，可以使用 Swap 文件：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 创建 2GB 的 Swap 文件</span>
<span class="token function">sudo</span> fallocate <span class="token parameter variable">-l</span> 2G /swapfile  <span class="token comment"># 也可以用 dd 命令： sudo dd if=/dev/zero of=/swapfile bs=1M count=2048</span>
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token number">600</span> /swapfile  <span class="token comment"># 确保安全性</span>
<span class="token function">sudo</span> <span class="token function">mkswap</span> /swapfile  <span class="token comment"># 创建 Swap 文件</span>
<span class="token function">sudo</span> <span class="token function">swapon</span> /swapfile  <span class="token comment"># 启用 Swap</span>

<span class="token comment"># 持久化配置（编辑 /etc/fstab）</span>
<span class="token builtin class-name">echo</span> <span class="token string">'/swapfile swap swap defaults 0 0'</span> <span class="token operator">&gt;&gt;</span> /etc/fstab
</code></pre>
    <h5>
     <a id="3__Swap__51">
     </a>
     <strong>
      3. 调整 Swap 交换策略
     </strong>
    </h5>
    <p>
     Linux 通过
     <strong>
      swappiness
     </strong>
     参数控制 Swap 交换策略，值范围是
     <strong>
      0~100
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       swappiness=0
      </strong>
      ：尽量避免使用 Swap（适用于大内存服务器）。
     </li>
     <li>
      <strong>
       swappiness=10-30
      </strong>
      ：适用于普通桌面或服务器（默认通常是 60）。
     </li>
     <li>
      <strong>
       swappiness=60+
      </strong>
      ：系统更积极地使用 Swap（适用于低内存设备）。
     </li>
    </ul>
    <p>
     查看当前 swappiness 值：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> /proc/sys/vm/swappiness
</code></pre>
    <p>
     临时修改（重启后失效）：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">sysctl</span> <span class="token assign-left variable">vm.swappiness</span><span class="token operator">=</span><span class="token number">10</span>
</code></pre>
    <p>
     永久修改（重启后生效）：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token string">'vm.swappiness=10'</span> <span class="token operator">|</span> <span class="token function">sudo</span> <span class="token function">tee</span> <span class="token parameter variable">-a</span> /etc/sysctl.conf
<span class="token function">sudo</span> <span class="token function">sysctl</span> <span class="token parameter variable">-p</span>  <span class="token comment"># 使修改生效</span>
</code></pre>

    <h4>
     <a id="_85">
     </a>
     <strong>
      总结
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       桌面系统
      </strong>
      ：如果 RAM 大于 16GB，Swap 可以设小一些（4GB~8GB）。
     </li>
     <li>
      <strong>
       服务器
      </strong>
      ：建议至少 1 倍 RAM，尤其是数据库和虚拟机环境。
     </li>
     <li>
      <strong>
       休眠功能
      </strong>
      ：Swap 需大于或等于 RAM 大小。
     </li>
     <li>
      <strong>
       调整 swappiness
      </strong>
      以优化性能，避免不必要的磁盘 I/O。
     </li>
    </ul>
    <h4>
     <a id="_Swap_91">
     </a>
     增加
     <strong>
      Swap
     </strong>
     分区
    </h4>
    <p>
     增加
     <strong>
      Swap
     </strong>
     分区的方式主要有两种：
    </p>
    <ol>
     <li>
      <strong>
       扩展现有的 Swap 分区
      </strong>
      （如果有未使用的磁盘空间）。
     </li>
     <li>
      <strong>
       创建新的 Swap 文件
      </strong>
      （适用于无法调整分区的情况）。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_1_Swap__Swap__98">
     </a>
     <strong>
      方法 1：扩展 Swap 分区（适用于已有 Swap 分区）
     </strong>
    </h3>
    <p>
     如果你当前的 Swap 是一个独立分区，并且磁盘有剩余空间，可以按照以下步骤进行扩展。
    </p>
    <h4>
     <a id="_1_Swap_101">
     </a>
     <strong>
      步骤 1：关闭当前 Swap
     </strong>
    </h4>
    <p>
     先检查 Swap 设备：
    </p>
    <pre><code class="prism language-bash"><span class="token function">swapon</span> <span class="token parameter variable">--show</span>
</code></pre>
    <p>
     然后关闭当前 Swap：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> swapoff <span class="token parameter variable">-a</span>
</code></pre>
    <h4>
     <a id="_2_Swap__111">
     </a>
     <strong>
      步骤 2：调整 Swap 分区
     </strong>
    </h4>
    <p>
     查看磁盘信息：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">fdisk</span> <span class="token parameter variable">-l</span>
</code></pre>
    <p>
     如果你有未分配的磁盘空间，可以使用
     <code>
      fdisk
     </code>
     或
     <code>
      parted
     </code>
     进行扩展：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">fdisk</span> /dev/sdX  <span class="token comment"># 选择你的 Swap 分区所在磁盘</span>
</code></pre>
    <p>
     删除现有的 Swap 分区并重新创建一个更大的分区（数据会丢失，谨慎操作）。
    </p>
    <p>
     使用
     <code>
      resize2fs
     </code>
     （对于某些文件系统，如 ext4）或
     <code>
      parted
     </code>
     命令进行调整。
    </p>
    <h4>
     <a id="_3_Swap__124">
     </a>
     <strong>
      步骤 3：格式化新的 Swap 分区
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mkswap</span> /dev/sdXn  <span class="token comment"># 重新格式化 Swap 分区（例如 /dev/sda3）</span>
<span class="token function">sudo</span> <span class="token function">swapon</span> /dev/sdXn  <span class="token comment"># 重新启用 Swap</span>
</code></pre>
    <h4>
     <a id="_4_etcfstab_130">
     </a>
     <strong>
      步骤 4：更新 /etc/fstab
     </strong>
    </h4>
    <p>
     编辑
     <code>
      /etc/fstab
     </code>
     ，确保 Swap 分区挂载信息正确：
    </p>
    <pre><code class="prism language-bash">/dev/sdXn swap swap defaults <span class="token number">0</span> <span class="token number">0</span>
</code></pre>
    <p>
     然后执行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mount</span> <span class="token parameter variable">-a</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_2_Swap__Swap_142">
     </a>
     <strong>
      方法 2：使用 Swap 文件增加 Swap（推荐，安全、不影响分区）
     </strong>
    </h3>
    <p>
     如果你不想修改已有的 Swap 分区，可以直接增加 Swap 文件，这种方法不会影响系统现有分区布局。
    </p>
    <h4>
     <a id="_1_Swap_145">
     </a>
     <strong>
      步骤 1：检查当前 Swap
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">free</span> <span class="token parameter variable">-m</span>
<span class="token function">swapon</span> <span class="token parameter variable">--show</span>
</code></pre>
    <h4>
     <a id="_2_Swap__151">
     </a>
     <strong>
      步骤 2：创建 Swap 文件
     </strong>
    </h4>
    <p>
     假设你想增加
     <strong>
      4GB Swap
     </strong>
     ，执行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> fallocate <span class="token parameter variable">-l</span> 4G /swapfile  <span class="token comment"># 创建 4GB 空间的 Swap 文件</span>
</code></pre>
    <p>
     （如果
     <code>
      fallocate
     </code>
     命令不可用，可以使用
     <code>
      dd
     </code>
     命令）
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">dd</span> <span class="token assign-left variable">if</span><span class="token operator">=</span>/dev/zero <span class="token assign-left variable">of</span><span class="token operator">=</span>/swapfile <span class="token assign-left variable">bs</span><span class="token operator">=</span>1M <span class="token assign-left variable">count</span><span class="token operator">=</span><span class="token number">4096</span>  <span class="token comment"># 创建 4GB 文件</span>
</code></pre>
    <h4>
     <a id="_3_161">
     </a>
     <strong>
      步骤 3：设置权限
     </strong>
    </h4>
    <p>
     确保只有 root 用户能访问：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token number">600</span> /swapfile
</code></pre>
    <h4>
     <a id="_4_Swap__167">
     </a>
     <strong>
      步骤 4：格式化 Swap 文件
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mkswap</span> /swapfile
</code></pre>
    <h4>
     <a id="_5_Swap__172">
     </a>
     <strong>
      步骤 5：启用 Swap 文件
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">swapon</span> /swapfile
</code></pre>
    <h4>
     <a id="_6_177">
     </a>
     <strong>
      步骤 6：永久生效
     </strong>
    </h4>
    <p>
     编辑
     <code>
      /etc/fstab
     </code>
     ，添加以下内容：
    </p>
    <pre><code class="prism language-bash">/swapfile swap swap defaults <span class="token number">0</span> <span class="token number">0</span>
</code></pre>
    <p>
     然后执行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mount</span> <span class="token parameter variable">-a</span>
</code></pre>
    <h4>
     <a id="_7_Swap__187">
     </a>
     <strong>
      步骤 7：调整 Swap 策略（可选）
     </strong>
    </h4>
    <p>
     检查当前
     <code>
      swappiness
     </code>
     值：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> /proc/sys/vm/swappiness
</code></pre>
    <p>
     如果值较高（如 60），可以降低它（推荐 10~30，减少对 Swap 的依赖）：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token string">'vm.swappiness=10'</span> <span class="token operator">|</span> <span class="token function">sudo</span> <span class="token function">tee</span> <span class="token parameter variable">-a</span> /etc/sysctl.conf
<span class="token function">sudo</span> <span class="token function">sysctl</span> <span class="token parameter variable">-p</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_3_Swap__200">
     </a>
     <strong>
      方法 3：增加多个 Swap 文件
     </strong>
    </h3>
    <p>
     如果需要进一步增加 Swap，可以创建多个 Swap 文件：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> fallocate <span class="token parameter variable">-l</span> 2G /swapfile2
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token number">600</span> /swapfile2
<span class="token function">sudo</span> <span class="token function">mkswap</span> /swapfile2
<span class="token function">sudo</span> <span class="token function">swapon</span> /swapfile2
</code></pre>
    <p>
     并添加到
     <code>
      /etc/fstab
     </code>
     ：
    </p>
    <pre><code class="prism language-bash">/swapfile2 swap swap defaults <span class="token number">0</span> <span class="token number">0</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_Swap__215">
     </a>
     <strong>
      如何验证 Swap 是否增加成功？
     </strong>
    </h3>
    <pre><code class="prism language-bash"><span class="token function">free</span> <span class="token parameter variable">-m</span>   <span class="token comment"># 显示 Swap 总大小</span>

</code></pre>

     <strong>
      总结
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      修改分区（方法 1）
     </strong>
    </p>
    <ul>
