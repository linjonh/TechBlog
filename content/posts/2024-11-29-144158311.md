     <p style="text-align:center;">
      <img alt="7cf44162fcbab405228de7abd58ba70e.gif" src="https://img-blog.csdnimg.cn/img_convert/7cf44162fcbab405228de7abd58ba70e.gif"/>

     <p>
      浏览器大战升级，微软 Edge 正面临着来自 Chrome、Opera、Vivaldi、Waterfox、Wavebox 的五重夹击。
     </p>
     <p style="text-align:left;">
      整理 | 苏宓
      <br/>
     </p>
     <p style="text-align:left;">
      出品 | CSDN（ID：CSDNnews）
     </p>
     <p style="text-align:justify;">
      出乎众人意料的是，就在 Google 因涉嫌违反反垄断法规被美国司法部调查，并有可能面临被迫出售 Chrome 浏览器的局面时，Chrome 却联合 Opera、Vivaldi、Waterfox 和 Wavebox 成立了一个名为“浏览器选择联盟”（简称 BCA）的组织，剑指微软的 Edge 浏览器。
     </p>
     <p style="text-align:justify;">
      同时该组织还特地上线了一个网站（https://browserchoicealliance.org/），其在首页直接写道，「在过去十年中，微软一直在使用各种转变策略，包括暗中操作和误导性用户界面，阻止用户可靠地访问他们喜欢的浏览器或将其设置为默认浏览器。」于是，该“浏览器选择联盟”呼吁欧盟委员会将微软的 Edge 列为《数字市场法》（DMA）下的守门人。
     </p>
     <p style="text-align:justify;">
      <img alt="a2ffc916d8132b855330cb0539cde0f3.png" src="https://img-blog.csdnimg.cn/img_convert/a2ffc916d8132b855330cb0539cde0f3.png"/>
     </p>
     <p style="text-align:justify;">
      此举也让人担忧，Edge 是否会重蹈曾经被判定为垄断的 IE 浏览器的覆辙。
     </p>
     <p style="text-align:justify;">
      <strong>
       <img alt="c24200fd2bbceeb160ca4259b88d946e.png" src="https://img-blog.csdnimg.cn/img_convert/c24200fd2bbceeb160ca4259b88d946e.png"/>
      </strong>
     </p>
     <p style="text-align:justify;">
      <strong>
       浏览器选择联盟究竟是什么？
      </strong>
     </p>
     <p style="text-align:justify;">
      众所周知，Edge 是 IE 之后微软推出的 Web 浏览器，自 2015 年以来包含在每台 Windows 安装中（IE 自 1998 年以来）。
     </p>
     <p style="text-align:justify;">
      虽然该浏览器现在基于 Chromium 开源项目构建，但这个浏览器选择联盟组织称，微软仍在使用可疑策略试图强迫用户避免安装第三方浏览器。
     </p>
     <p style="text-align:justify;">
      为此，他们想要联合起来，指出微软在 Windows PC 上推广其 Edge 浏览器时存在的问题，进而抵制以及促使微软修正这种行为。
     </p>
     <p style="text-align:justify;">
      截至目前，浏览器选择联盟成员包括谷歌 Chrome、Opera、Vivaldi、BrowserWorks（Waterfox）和 Wavebox（一款优质、无广告的浏览器）一共五家公司。
     </p>
     <p style="text-align:justify;">
      对此，不少人也发现，相较之下，Google Chrome 在这个成员名单中显得有些突兀。毕竟据 Statcounter 数据显示，Chrome 可是全球第一大浏览器，占比高达 66.68%，为何要现身与微软“拉仇恨”？此举究竟是 Chrome 为了转移外界对于其受到反垄断调查的注意，还是为了真正改善用户体验？
     </p>
     <p style="text-align:center;">
      <img alt="54fb473bb847c5c307b8453ed1c9425b.png" src="https://img-blog.csdnimg.cn/img_convert/54fb473bb847c5c307b8453ed1c9425b.png"/>
     </p>
     <p style="text-align:justify;">
      Google Chrome 浏览器副总裁兼总经理 Parisa Tabriz 在加入该组织时说道：“消费者在选择浏览器时有很多选择。当他们选择下载浏览器时，他们的选择应该受到尊重。我们很自豪能成为倡导消费者选择和尊重的团体的一员。”
     </p>
     <p style="text-align:justify;">
      那要问为什么苹果的 Safari 不在名单之列，外媒 ghacks 分析称，其实也不难理解，首先因为 Apple Safari 无法在 Windows 上使用，自然没有必要去针对 Window 平台上用户该用谁家的浏览器去争论。
     </p>
     <p style="text-align:justify;">
      不过，尚不清楚 Brave Browser、Mozilla Firefox 和其他浏览器为何不加入浏览器选择联盟。但是值得注意的是，现有的浏览器选择联盟成员透露，也还有一部分公司担心自己会遭到微软的直接或间接报复，他们不愿公开发表自己的观点。
     </p>
     <p style="text-align:justify;">
      <img alt="5210dbee8ff5aec97eb949ddfdbd3934.png" src="https://img-blog.csdnimg.cn/img_convert/5210dbee8ff5aec97eb949ddfdbd3934.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       剑指微软暗箱操作，阻止用户在 Windows 上设置自己喜欢的浏览器
      </strong>
     </p>
     <p style="text-align:justify;">
      至于微软 Edge 是怎么成为其他浏览器厂商的“眼中钉”的？
     </p>
     <p style="text-align:justify;">
      浏览器选择联盟的成员表示，Windows 是领先的桌面操作系统，根据 StatCounter 的数据显示，全球超过 73% 的计算机用户使用微软 Windows 系统。
     </p>
     <p style="text-align:center;">
      <img alt="6206d6007b2eff13e002b1fa1ed94643.png" height="720" src="https://img-blog.csdnimg.cn/img_convert/6206d6007b2eff13e002b1fa1ed94643.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      由于 Edge 是 Windows 的装机必备浏览器，浏览器选择联盟认为，这种权利使微软能够通过对竞争对手设置欺骗性限制，以此来限制竞争对手软件在 Windows 上的竞争能力。
     </p>
     <p style="text-align:justify;">
      此外，浏览器选择联盟成员声称，微软通过暗箱操作、设置技术障碍和基于欺骗行为，阻止消费者找到并使用他们想要的浏览器，并妨碍他们将其设置为 Window PC 的首选默认浏览器。具体行为包括：
     </p>
     <ul>
      <li>
       <p style="text-align:justify;">
        <strong>
         为下载不同的浏览器设置障碍。
        </strong>
        即当用户使用 Edge 浏览器中的 Bing 搜索引擎去搜索第三方浏览器时，你可能会看到一条消息，上面写着“无需下载新的 Web 浏览器。微软建议使用 Microsoft Edge 获得快速、安全和现代的网络体验，这可以帮助您节省时间和金钱”；
       </p>
      </li>
     </ul>
     <p style="text-align:center;">
      <img alt="c45f232c9e1eb9cd17b9efcff0877420.png" src="https://img-blog.csdnimg.cn/img_convert/c45f232c9e1eb9cd17b9efcff0877420.png"/>
     </p>
     <p style="text-align:center;">
      <img alt="ba3960829c460ea9c7ee266e0a5346c0.png" src="https://img-blog.csdnimg.cn/img_convert/ba3960829c460ea9c7ee266e0a5346c0.png"/>
     </p>
     <ul>
      <li>
       <p style="text-align:justify;">
        <strong>
         在定期更新期间将用户的默认设置更改回 Edge；
        </strong>
       </p>
      </li>
      <li>
       <p style="text-align:justify;">
        <strong>
         强制在 Edge 或 Bing 中打开微软自己的服务（Teams/Outlook 等）中的链接；
        </strong>
       </p>
      </li>
      <li>
       <p style="text-align:justify;">
        <strong>
         围绕浏览器选择的强制性信息，鼓励用户“恢复微软推荐的浏览器设置”。
        </strong>
       </p>
      </li>
     </ul>
     <p style="text-align:justify;">
      「这些行为不仅影响浏览器开发人员，还影响整个网络生态系统」，浏览器选择联盟说道。
     </p>
     <p style="text-align:justify;">
      其实就这些行为，浏览器选择联盟成员之一的 Vivaldi 首席执行官兼联合创始人 Jon von Tetzchner 在 2021 年发文公开批评过微软，同时其还在 2023 年 9 月《数字市场法案》出台后，成为第一家向欧盟递交公开信表达对微软反竞争行为担忧的浏览器公司。然而，其表示，尽管微软的 Edge 浏览器符合 DMA 加强监管的门槛，但欧盟决定不将其列入名单。
     </p>
     <p style="text-align:justify;">
      不止如此，另一家浏览器公司 Opera 同样在 2024 年 7 月向欧盟提起过上诉，反对欧盟委员会不根据《数字市场法》指定 Microsoft Edge 为守门人的决定，然而至今也没有什么结果。
     </p>
     <p style="text-align:justify;">
      这一次，希望人多力量大的浏览器厂商们集 Chrome、Opera、Vivaldi、BrowserWorks（Waterfox）和 Wavebox 之力，呼吁全球监管机构紧急干预，解决微软在 Edge 方面的各种问题，捍卫消费者的选择权。该组织还特别提到，
      <strong>
       呼吁欧盟委员会将微软的 Edge 列为《数字市场法案》（DMA）下的“守门人”。
      </strong>
     </p>
     <p style="text-align:justify;">
      <strong>
       <img alt="066011e27f0b9474e5c4bd54bd315d41.png" src="https://img-blog.csdnimg.cn/img_convert/066011e27f0b9474e5c4bd54bd315d41.png"/>
      </strong>
     </p>
     <p style="text-align:justify;">
      <strong>
       Chrome 今天的窘境，Edge 明日的镜鉴？
      </strong>
     </p>
     <p style="text-align:justify;">
      我们也曾报道过，如果被选为 DMA 的守门人，这意味着入选的公司需要遵守一系列特定的规定，以确保其行为不会对市场竞争造成不公平的影响。倘若“守门人”违反了这些规定，欧盟委员会可以对其处以罚款，最高可达该公司全球年度总营业额的 10%，对于屡次违规的情况，罚款可提高至 20%。此外，还可能采取结构性补救措施，如强制出售业务或部分业务。
     </p>
     <p style="text-align:justify;">
      此前，欧盟委员会根据 DMA 规定的守门人标准（即过去三年市值至少达到 750 亿欧元，在欧盟年营业额超过 75 亿欧元，拥有至少 4500 万月活跃用户以及超过 1 万名年度活跃用户，至少在三个欧盟成员国提供核心平台服务等），认定了六家守门人，包括 Alphabet（Google 母公司）、亚马逊、苹果、字节跳动、Meta、微软。
     </p>
     <p style="text-align:justify;">
      其中，委员会通过这几家公司的特定核心平台服务确立了守门人地位。具体来看，微软的 Edge 确实不在监管范围，涉及浏览器方面的只有 Google Chrome 和苹果 Safari：
     </p>
     <p style="text-align:center;">
      <img alt="b1f51ae698cb85350db128e845c4b523.png" src="https://img-blog.csdnimg.cn/img_convert/b1f51ae698cb85350db128e845c4b523.png"/>
     </p>
     <p style="text-align:justify;">
      前段时间，美国司法部对 Google 提起的反垄断诉讼中，其他搜索引擎制造商的高管证实，大多数用户没有更改浏览器中的默认搜索引擎的习惯，因为他们要么不知道如何更改，要么是因为修改选项深藏在设置中。这事儿不仅限于搜索引擎，浏览器也是一样。这也就是美国司法部希望 Google 出售 Chrome 的原因，这凸显了垄断如何不公平地统治市场。
     </p>
     <p>
      这一次，倘若微软因为 Edge 被列为《数字市场法案》（DMA）下的“守门人”，必然也会迎来来自欧盟委员会的反垄断调查，而 Google Chrome 如今面临的窘境极有可能是 Edge 明天的镜鉴。
     </p>
     <p>
      在浏览器选择联盟成员看来，微软在 Edge 设置的层层门槛，已经让其自身在浏览器大战中占据不公平的优势。
     </p>
     <p style="text-align:justify;">
      浏览器选择联盟表示，“浏览器是通向互联网的门户，它们还支持各种其他重要服务——从人工智能到生产力工具等等。作为更大互联网经济的一部分，这些服务直接影响着全球许多人的生活和生计。特别是在教育和企业领域，桌面电脑上的浏览器的使用仍将是未来几年用户体验的基本组成部分。因此，促进浏览器的公平竞争至关重要：随着技术的不断发展，竞争可以确保消费者拥有创新和选择。 ”
     </p>
     <p style="text-align:justify;">
      对此，Vivaldi 首席执行官兼联合创始人 Jon von Tetzchner 斥责道：“长期以来，Vivaldi 一直公开谴责微软使用卑鄙手段破坏消费者选择。Edge 在 Windows 中受到如此多的青睐，像我们这样的独立浏览器根本无法与它公平竞争。监管机构需要紧急干预。”
     </p>
     <p>
      Opera 浏览器执行副总裁 Krystian Kolondra 表示：“近 30 年来，Opera 一直捍卫开放标准和公平使用造福用户的技术。用户选择是自由开放网络不可或缺的一部分，这种自由必须得到保护。这就是我们加入浏览器选择联盟的原因。”
     </p>
     <p style="text-align:justify;">
      Wavebox 联合创始人 Susan Davies 称：“企业和商务用户之所以付费使用Wavebox，是因为它具有提高生产力的功能。但微软的做法一直都在削弱用户的偏好选择，例如通过设置链接自动在 Edge 中打开，并在更新过程中更改默认设置。”
     </p>
     <p style="text-align:justify;">
      BrowserWorks（Waterfox）创始人 Alex Kontos：“微软的做法让注重隐私的用户难以发现并选择像 Waterfox 这样的替代浏览器，实际上限制了消费者的选择。通过加入这个产业联盟，我们正在为使网络变得伟大的基本原则——开放性、竞争性和用户赋权——发声和支持。。”
     </p>
     <p style="text-align:justify;">
      那么，他们能否成功？未来还将取决于反垄断监管机构。
     </p>
