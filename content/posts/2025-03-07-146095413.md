    <p>
     <strong>
      1. rocky 9.4版本会有一个podman影响docker，先删除这个（其他版本也许没有，但执行以下没坏处）
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> dnf remove <span class="token function">podman</span>
</code></pre>
    <p>
     <strong>
      2.删除podman残余文件
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">rm</span> <span class="token parameter variable">-rf</span> /etc/containers /etc/registries.conf /etc/containers/policy.json /etc/containers/storage.conf ~/.config/containers ~/.local/share/containers
</code></pre>
    <p>
     <strong>
      3.卸载旧版本docker
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> dnf remove <span class="token function">docker</span> docker-client docker-client-latest docker-common docker-latest docker-latest-logrotate docker-logrotate docker-engine
</code></pre>
    <p>
     <strong>
      4. 安装docker所需依赖
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> dnf <span class="token function">install</span> <span class="token parameter variable">-y</span> yum-utils device-mapper-persistent-data lvm2
</code></pre>
    <p>
     <strong>
      5.添加docker官方和阿里的YUM仓库
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

<span class="token function">sudo</span> yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
</code></pre>
    <p>
     <strong>
      6.安装 Docker Engine
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> dnf <span class="token function">install</span> docker-ce docker-ce-cli containerd.io
</code></pre>
    <p>
     <strong>
      7.启动Docker
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl start <span class="token function">docker</span>
</code></pre>
    <p>
     8.设置docker开机启动
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl <span class="token builtin class-name">enable</span> <span class="token function">docker</span>
</code></pre>
    <p>
     <strong>
      9.验证安装
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">docker</span> <span class="token parameter variable">--version</span>
</code></pre>
