# 图形界面操作系统发展史

Alto

1973年4月，第一个可操作的Alto电脑在Xerox PARC完成。Alto是第一个把计算机所有元素结合到一起的图形界面操作系统。它使用3键鼠标、位运算显示器、图形窗口、以太网络连接。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_01Alto.gif)

Perq

1980年，Three Rivers Computer Corporation推出Perq图形工作站。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_02Perq.gif)

Star

1981年7月，Xerox推出Star系统，它是Alto的商用版本。特点：双击图标技术、多窗口、对话框概念、1024\*768的分辨率。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_03Star.jpg)

Lisa

1983年1月，Apple推出Lisa系统。特点：下拉菜单和菜单条技术。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_04Lisa.jpg)

Visi On

1983年，Visi推出Visi On，它是第一款为IBM PC设计的集成图形软件环境。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_05Visi.gif)

Windows

1983年，微软宣布它为IBM PC设计的Windows程序，不过这个系统到1985年才发布。特点：多窗口、可改变大小窗口。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_06Windows.gif)

Macintosh

1984年1月，Apple推出Macintosh。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_07Macintosh.gif)

GEM icon/desktop

1984年9月，Digital Research针对8086和DOS计算机推出GEM icon/desktop用户界面系统，这个系统稍后演变为AtariST。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_08GEM.gif)

Window System X

1984年7月，MIT推出“Window System X”。它1－6版本使用单色画，使用DEC VS100显示器。8－10版本开始使用彩色显示。VAXstation II/GPX.X10是第一个被广泛使用的版本。第11个版本被设计成一个很高性能的系统，有很好的窗口管理风格，良好的图形表现和系统扩展能力。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_09SystemX.gif)

Commodore 64

1985年，Geos发布Commodore 64，稍后又发布了Apple II。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_10Commodore.gif)

Amiga 100

1985年7月，Commodore推出Amiga 100（Amiga Workbench Version 1.0）。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_12Amiga100.gif)

Windows 1.0

1985年8月，微软最终发布第一个Windows版本。特点：窗口不可重叠，但可平铺；窗口不会覆盖屏幕下方的图标区域。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_13Windows.gif)

GEM

1986年，Apple表示会控告Digital Research，因为其GEM桌面系统跟自己的Macintosh非常相像。Digital Reseach最终修改了自己的桌面系统。新的桌面系统现在只有用两个无法移动、无法改变大小的窗口来浏览文件。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_14GEM.gif)

Macintosh II

1987年，Apple推出Macintosh II，它是第一款彩色Mac系统。特点：640\*480，56色，24位显卡。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_15Macintosh2.gif)

Windows 2.0

1987年，微软发布第二个Windows版本，V2.03。特点：可改变大小/可重叠的窗口、加进了其他窗口控制功能。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_16Windows2.gif)

Arthur

1987年，Acorn发布应用于Acorn电脑的Arthur系统，它是RISC系统的基础，RISC OS 2和3和它有类似的外观，但有很大的改善。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_17Arthur.gif)

GS/OS

1988年9月，Apple发布GS/OS，是为Apple IIGS设计的16位操作系统，这个系统跟Macintosh的外观很相像。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_18GS.gif)

OS/2

1988年10月，IBM发布OS/2 1.10 Standard Edition。OS2增加了叫做“Presentation Manager”的图形用户界面。1.10 GUI是由微软开发的，看上去有点像Windows 2。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_19OS2.gif)

NeXT

1988年10月，NeXT Computer发布了，价格为6500美元。它使用25MHz’30 processor/8MB RAM/250MB optical disk drive，数字处理器，声卡、fax modem、17寸显示器。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_20NeXT.jpg)

Amiga 2 for A3000

1990年，Commodore发布Amiga Workbench 2 for A3000。特点：新3D效果，可定制菜单。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_21Amiga2.gif)

Windows 3.0

1990年5月，微软发布Windows 3.0。特点：程序管理SHELL。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_22Windows3.gif)

PG-GEOS

1990年11月，GeoWorks发布PG-GEOS。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_23PG.gif)

OS/2 2.0

1992年春，IBM发布OS/2 version2.0，真正的32为操作系统。特点：Workplace SHELL、面向对象的用户界面。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_24OS22.gif)

Windows 3.1

1992年3月，微软推出Windows 3.1，界面跟3.0差不多，但这个版本包含了多媒体支持。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_25Windows31.gif)

Amiga 3

1992年5月，Amiga Workbench 3发布。特点：使用图片做背景、色彩调节。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_26Amiga3.gif)

Windows NT

1993年5月，微软发布Windows NT的第一个版本，是32位操作系统。同样使用3.1的版本号，它为Intel，Power PC，Alpha和MIPS系统设计。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_27WindowsNT.gif)

Photon microGUI

1994年QNX Software Systems发布他们第一个微核心窗口操作系统：Photon microGUI。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_28Photon.gif)

Windows 95

1995年8月24日，微软发布Windows 95。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_29Windows95.gif)

BeOS

1995年，BeOS发布，这是第一个为定制的多处理器系统（BeBox）开发的。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_30BeOS.gif)

New Deal Office 2.5

1996年，New Deal发布New Deal Office 2.5，它的前身是PC-GEOS。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_31DealOffice.gif)

OS/2 Warp 4

1996年，IBM发布OS/2 Warp 4，这个系统上Workplace Shell技术有重大应用。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_32OS24.gif)

Windows NT 4.0

1996年，微软发布Windows NT 4.0，它使用和Windows 95相同的用户界面。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_33WindowsNT4.gif)

Mac OS 8

1997年6月，Mac OS 8发布。在两周内销售125万套，是那个时期销售最好的操作系统。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_34Mac8.gif)

Windows 98

1998年7月25日，微软发布Windows 98。特点：IE，浏览器取代了Windows shell和帮助系统。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_35Windows98.gif)

Windows 98 Lite

1998年11月22日，Shane Brooks发布98 Lite，这个版本移去了IE。特点：没有IE，与win98相同的硬件支持，快速启动。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_36Win98NoIE.gif)

Mac OS X

1999年，Apple发布Mac OS X Server，是基于UNIX的操作系统，使用Macintosh GUI。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_37MacX.gif)

RISC OS 4

1999年7月，RISC OS Ltd发布RISC OS 4，使用于RiscPC A7000 或 A7000+。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_38RISC4.jpg)

Aqua

2000年1月5日，Apple发布Aqua，界面风格有很大改善。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_39Aqua.jpg)

Windows 2000

2000年2月17日，微软发布Windows 2000（Windows NT 5）。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_40Windows2000.gif)

Windows XP

2001年10月25日，微软发布Windows XP（Windows NT 5.1）。特点：激活技术...汗

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_41WindowsXP.jpg)

Windows Server 2003

2003年4月24日，微软发布Windows Server 2003（Windows NT 5.2，也叫Windows.NET SERVER）。特点：只发布服务器版本。

![](https://p-blog.csdn.net/images/p_blog_csdn_net/tjianliang/432059/o_42Windows2003.jpg)
