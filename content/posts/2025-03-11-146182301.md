## 文件概述
对miniz压缩库的C++封装扩展，提供ZIP文件读写的高级接口。主要特性包括：
- 面向对象的ZIP存档管理
- UTF-8文件名支持
- 错误状态追踪机制
- 安全的资源生命周期管理

## 核心组件

### 全局函数
```cpp
bool open_zip_reader(mz_zip_archive*, const std::string&amp; fname_utf8);   // 打开ZIP读取器
bool open_zip_writer(mz_zip_archive*, const std::string&amp; fname_utf8);   // 打开ZIP写入器
bool close_zip_reader(mz_zip_archive*);                                // 关闭读取器
bool close_zip_writer(mz_zip_archive*);                                // 关闭写入器
</code></pre>

    <h3>
     <a id="_90">
     </a>
     设计特点
    </h3>
    <ol>
     <li>
      <strong>
       资源管理
      </strong>
      ：要求显式调用关闭函数，避免资源泄漏
     </li>
     <li>
      <strong>
       跨平台支持
      </strong>
      ：通过UTF-8路径处理Windows/Linux兼容性
     </li>
     <li>
      <strong>
       状态追踪
      </strong>
      ：
      <code>
       is_alive()
      </code>
      实时反映存档操作状态
     </li>
     <li>
      <strong>
       错误追溯
      </strong>
      ：自动记录最后一次操作的错误代码
     </li>
    </ol>

    <h3>
     <a id="_102">
     </a>
     注意事项
    </h3>
    <ol>
     <li>
      <strong>
       线程安全
      </strong>
      ：单个MZ_Archive实例不应跨线程使用
     </li>
     <li>
      <strong>
       异常处理
      </strong>
      ：所有函数返回bool状态，需手动检查
     </li>
     <li>
      <strong>
       内存管理
      </strong>
      ：
      <code>
       mz_zip_archive
      </code>
      结构体由miniz库内部管理
     </li>
     <li>
      <strong>
       性能建议
      </strong>
      ：批量文件操作时保持存档开启状态
     </li>
    </ol>
