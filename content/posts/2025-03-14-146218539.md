    <p>
     可以使用
     <strong>
      滑动窗口
     </strong>
     方法来解决这个问题。步骤如下：
    </p>
    <ol>
     <li>
      <strong>
       初始化
      </strong>
      ：计算前
      <code>
       k
      </code>
      个字符中元音字母的个数，作为初始窗口的值。
     </li>
     <li>
      <strong>
       滑动窗口
      </strong>
      ：遍历字符串，每次右移窗口，删除窗口左端字符，添加窗口右端字符，同时更新元音字母的计数。
     </li>
     <li>
      <strong>
       维护最大元音计数
      </strong>
      ：在滑动过程中，记录最大值。
     </li>
    </ol>
    <h4>
     <strong>
      Python 代码
     </strong>
    </h4>
    <pre><code class="language-python">def maxVowels(s, k):
    vowels = {'a', 'e', 'i', 'o', 'u'}  # 使用集合加速查找
    count = max_count = sum(1 for c in s[:k] if c in vowels)  # 计算初始窗口的元音数量

    for i in range(k, len(s)):
        count += (s[i] in vowels) - (s[i - k] in vowels)  # 右移窗口，更新元音计数
        max_count = max(max_count, count)  # 记录最大元音计数
    
    return max_count
</code></pre>
    <h4>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code>s = "abciiidef"
k = 3
print(maxVowels(s, k))  # 输出 3
</code></pre>

       s
