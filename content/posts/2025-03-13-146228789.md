    <p>
     时间戳（Timestamp）是计算机中表示时间的常用方式，通常指自
     <strong>
      协调世界时（UTC）1970年1月1日0时0分0秒（Unix纪元）
     </strong>
     起经过的秒数（或毫秒/微秒）。以下是时间戳的核心操作和示例：
    </p>
    <hr/>
    <h4>
     <a id="_5">
     </a>
     <strong>
      一、获取时间戳
     </strong>
    </h4>
    <h5>
     <a id="1__6">
     </a>
     1.
     <strong>
      当前时间戳（秒级）
     </strong>
    </h5>
    <p>
     使用
     <code>
      time.time()
     </code>
     返回浮点数（秒 + 小数部分为微秒）：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> time

current_ts <span class="token operator">=</span> time<span class="token punctuation">.</span>time<span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>current_ts<span class="token punctuation">)</span>  <span class="token comment"># 输出：1690200000.123456 （示例值）</span>
</code></pre>
    <h5>
     <a id="2__15">
     </a>
     2.
     <strong>
      当前时间戳（毫秒级）
     </strong>
    </h5>
    <p>
     将秒转换为毫秒：
    </p>
    <pre><code class="prism language-python">current_ts_ms <span class="token operator">=</span> <span class="token builtin">int</span><span class="token punctuation">(</span>time<span class="token punctuation">.</span>time<span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token operator">*</span> <span class="token number">1000</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>current_ts_ms<span class="token punctuation">)</span>  <span class="token comment"># 输出：1690200000123</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_24">
     </a>
     <strong>
      二、时间戳与日期互转
     </strong>
    </h4>
    <h5>
     <a id="1____25">
     </a>
     1.
     <strong>
      时间戳 → 日期时间
     </strong>
    </h5>
    <p>
     使用
     <code>
      time.localtime()
     </code>
     或
     <code>
      time.gmtime()
     </code>
     ：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> time

<span class="token comment"># 将时间戳转为本地时间（结构化对象）</span>
ts <span class="token operator">=</span> <span class="token number">1690200000</span>
local_time <span class="token operator">=</span> time<span class="token punctuation">.</span>localtime<span class="token punctuation">(</span>ts<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>local_time<span class="token punctuation">)</span>
<span class="token comment"># 输出：time.struct_time(tm_year=2023, tm_mon=7, tm_mday=24, ...)</span>

<span class="token comment"># 转为可读字符串</span>
formatted_time <span class="token operator">=</span> time<span class="token punctuation">.</span>strftime<span class="token punctuation">(</span><span class="token string">"%Y-%m-%d %H:%M:%S"</span><span class="token punctuation">,</span> local_time<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>formatted_time<span class="token punctuation">)</span>  <span class="token comment"># 输出：2023-07-24 12:00:00</span>
</code></pre>
    <h5>
     <a id="2____41">
     </a>
     2.
     <strong>
      日期时间 → 时间戳
     </strong>
    </h5>
    <p>
     使用
     <code>
      time.mktime()
     </code>
     ：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> time

<span class="token comment"># 将日期字符串转为时间戳</span>
date_str <span class="token operator">=</span> <span class="token string">"2023-07-24 12:00:00"</span>
struct_time <span class="token operator">=</span> time<span class="token punctuation">.</span>strptime<span class="token punctuation">(</span>date_str<span class="token punctuation">,</span> <span class="token string">"%Y-%m-%d %H:%M:%S"</span><span class="token punctuation">)</span>
ts <span class="token operator">=</span> time<span class="token punctuation">.</span>mktime<span class="token punctuation">(</span>struct_time<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>ts<span class="token punctuation">)</span>  <span class="token comment"># 输出：1690200000.0</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_55">
     </a>
     <strong>
      三、时间戳计算
     </strong>
    </h4>
    <h5>
     <a id="1__56">
     </a>
     1.
     <strong>
      计算时间差
     </strong>
    </h5>
    <p>
     记录两个时间戳的差值：
    </p>
    <pre><code class="prism language-python">start_ts <span class="token operator">=</span> time<span class="token punctuation">.</span>time<span class="token punctuation">(</span><span class="token punctuation">)</span>
time<span class="token punctuation">.</span>sleep<span class="token punctuation">(</span><span class="token number">3</span><span class="token punctuation">)</span>  <span class="token comment"># 模拟耗时操作</span>
end_ts <span class="token operator">=</span> time<span class="token punctuation">.</span>time<span class="token punctuation">(</span><span class="token punctuation">)</span>

elapsed <span class="token operator">=</span> end_ts <span class="token operator">-</span> start_ts
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"耗时：</span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>elapsed<span class="token punctuation">:</span><span class="token format-spec">.2f</span><span class="token punctuation">}</span></span><span class="token string">秒"</span></span><span class="token punctuation">)</span>  <span class="token comment"># 输出：耗时：3.00秒</span>
</code></pre>
    <h5>
     <a id="2__67">
     </a>
     2.
     <strong>
      时间戳加减
     </strong>
    </h5>
    <p>
     对时间戳直接进行数学运算：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> time

now_ts <span class="token operator">=</span> time<span class="token punctuation">.</span>time<span class="token punctuation">(</span><span class="token punctuation">)</span>
future_ts <span class="token operator">=</span> now_ts <span class="token operator">+</span> <span class="token number">3600</span>  <span class="token comment"># 1小时后</span>
past_ts <span class="token operator">=</span> now_ts <span class="token operator">-</span> <span class="token number">1800</span>    <span class="token comment"># 30分钟前</span>

<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"1小时后："</span><span class="token punctuation">,</span> time<span class="token punctuation">.</span>strftime<span class="token punctuation">(</span><span class="token string">"%H:%M"</span><span class="token punctuation">,</span> time<span class="token punctuation">.</span>localtime<span class="token punctuation">(</span>future_ts<span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"30分钟前："</span><span class="token punctuation">,</span> time<span class="token punctuation">.</span>strftime<span class="token punctuation">(</span><span class="token string">"%H:%M"</span><span class="token punctuation">,</span> time<span class="token punctuation">.</span>localtime<span class="token punctuation">(</span>past_ts<span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_82">
     </a>
     <strong>
      四、格式化时间戳
     </strong>
    </h4>
    <h5>
     <a id="1__83">
     </a>
     1.
     <strong>
      自定义日期格式
     </strong>
    </h5>
    <p>
     使用
     <code>
      time.strftime()
     </code>
     的格式符：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> time

ts <span class="token operator">=</span> <span class="token number">1690200000</span>
local_time <span class="token operator">=</span> time<span class="token punctuation">.</span>localtime<span class="token punctuation">(</span>ts<span class="token punctuation">)</span>

<span class="token comment"># 常用格式符</span>
formats <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">"日期"</span><span class="token punctuation">:</span> <span class="token string">"%Y-%m-%d"</span><span class="token punctuation">,</span>          <span class="token comment"># 2023-07-24</span>
    <span class="token string">"时间"</span><span class="token punctuation">:</span> <span class="token string">"%H:%M:%S"</span><span class="token punctuation">,</span>          <span class="token comment"># 12:00:00</span>
    <span class="token string">"完整"</span><span class="token punctuation">:</span> <span class="token string">"%Y-%m-%d %H:%M:%S"</span><span class="token punctuation">,</span> <span class="token comment"># 2023-07-24 12:00:00</span>
    <span class="token string">"周几"</span><span class="token punctuation">:</span> <span class="token string">"%A"</span><span class="token punctuation">,</span>                <span class="token comment"># Monday（英文）</span>
    <span class="token string">"月份"</span><span class="token punctuation">:</span> <span class="token string">"%B"</span><span class="token punctuation">,</span>                <span class="token comment"># July（英文）</span>
<span class="token punctuation">}</span>

<span class="token keyword">for</span> key<span class="token punctuation">,</span> fmt <span class="token keyword">in</span> formats<span class="token punctuation">.</span>items<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"</span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>key<span class="token punctuation">}</span></span><span class="token string">: </span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>time<span class="token punctuation">.</span>strftime<span class="token punctuation">(</span>fmt<span class="token punctuation">,</span> local_time<span class="token punctuation">)</span><span class="token punctuation">}</span></span><span class="token string">"</span></span><span class="token punctuation">)</span>
</code></pre>
    <h5>
     <a id="2__104">
     </a>
     2.
     <strong>
      时区处理
     </strong>
    </h5>
    <p>
     默认
     <code>
      time
     </code>
     模块使用本地时区，若需处理其他时区，建议用
     <code>
      datetime
     </code>
     模块：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> datetime <span class="token keyword">import</span> datetime<span class="token punctuation">,</span> timezone

<span class="token comment"># UTC时间</span>
utc_time <span class="token operator">=</span> datetime<span class="token punctuation">.</span>fromtimestamp<span class="token punctuation">(</span><span class="token number">1690200000</span><span class="token punctuation">,</span> tz<span class="token operator">=</span>timezone<span class="token punctuation">.</span>utc<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>utc_time<span class="token punctuation">)</span>  <span class="token comment"># 输出：2023-07-24 12:00:00+00:00</span>

<span class="token comment"># 转换为东八区时间</span>
utc_time <span class="token operator">=</span> datetime<span class="token punctuation">.</span>fromtimestamp<span class="token punctuation">(</span><span class="token number">1690200000</span><span class="token punctuation">,</span> tz<span class="token operator">=</span>timezone<span class="token punctuation">.</span>utc<span class="token punctuation">)</span>
beijing_time <span class="token operator">=</span> utc_time<span class="token punctuation">.</span>astimezone<span class="token punctuation">(</span>timezone<span class="token punctuation">(</span>timedelta<span class="token punctuation">(</span>hours<span class="token operator">=</span><span class="token number">8</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>beijing_time<span class="token punctuation">)</span>  <span class="token comment"># 输出：2023-07-24 20:00:00+08:00</span>
</code></pre>
