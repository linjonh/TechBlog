    <p>
     检查的代码：
    </p>
    <pre><code class="language-php">&lt;?php

$dir = __DIR__;
$files = new RecursiveIteratorIterator(new RecursiveDirectoryIterator($dir));

foreach ($files as $file) {
    if ($file-&gt;isFile() &amp;&amp; pathinfo($file, PATHINFO_EXTENSION) === 'php') {
        $content = file_get_contents($file-&gt;getPathname());
        if (substr($content, 0, 3) === "\xEF\xBB\xBF") {
            echo "BOM found in: " . $file-&gt;getPathname() . "\n";
        } else {
            echo "No BOM in: " . $file-&gt;getPathname() . "\n";
        }
    }
}
</code></pre>
    <p>
     批量去除Bom头的代码：
    </p>
    <pre><code class="language-php">&lt;?php



$dir = __DIR__; // 当前目录
$files = new RecursiveIteratorIterator(new RecursiveDirectoryIterator($dir));

foreach ($files as $file) {
    if ($file-&gt;isFile() &amp;&amp; pathinfo($file, PATHINFO_EXTENSION) === 'php') {
        echo "Checking: " . $file-&gt;getPathname() . "\n"; // 打印正在处理的文件
        $content = file_get_contents($file-&gt;getPathname());
        if (substr($content, 0, 3) === "\xEF\xBB\xBF") {
            $content = substr($content, 3);
            file_put_contents($file-&gt;getPathname(), $content);
            echo "Removed BOM from: " . $file-&gt;getPathname() . "\n";
        }
    }
}
</code></pre>
    <p>
     但是最后的方法是：
    </p>
    <pre><code class="language-bash">xxd index3.php | head -n 1</code></pre>
    <p>
     <img alt="" height="39" src="https://i-blog.csdnimg.cn/direct/f72b86decda24924a45334b2d2b2942b.png" width="607"/>

    <p>
     发现没有bom头，用PHPStorm右下角的 CR utf-8 改成  CRLF utf-8 就好了。
    </p>
