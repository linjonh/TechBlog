    <h2>
     <a id="_0">
     </a>
     样式
    </h2>
    <p>
     在
     <strong>
      LVGL
     </strong>
     中，
     <strong>
      样式（Style）
     </strong>
     是用来控制对象（控件）外观的核心机制,涵盖了
     <strong>
      颜色、边框、背景、文字、阴影、渐变、圆角
     </strong>
     等视觉特性。通过样式，你可以轻松地为不同状态（如正常、按下、禁用等）指定不同的外观效果。
    </p>
    <h3>
     <a id="1__4">
     </a>
     1. 样式的基本概念
    </h3>
    <h4>
     <a id="11_lv_style_t_6">
     </a>
     1.1 样式对象（lv_style_t）
    </h4>
    <ul>
     <li>
      样式对象中包含若干可配置的视觉属性，例如
      <strong>
       背景色
      </strong>
      、
      <strong>
       边框宽度
      </strong>
      、
      <strong>
       文字颜色
      </strong>
      、
      <strong>
       阴影
      </strong>
      、
      <strong>
       圆角
      </strong>
      等。
     </li>
     <li>
      每个属性都可以通过相关的 API（如
      <code>
       lv_style_set_bg_color()
      </code>
      、
      <code>
       lv_style_set_border_width()
      </code>
      ）来设置。
     </li>
    </ul>
    <h4>
     <a id="12__11">
     </a>
     1.2 属性类型
    </h4>
    <ul>
     <li>
      <strong>
       背景（background）
      </strong>
      ：如背景颜色、背景透明度、背景渐变等。
     </li>
     <li>
      <strong>
       边框（border）
      </strong>
      ：如边框颜色、边框宽度、边框圆角等。
     </li>
     <li>
      <strong>
       文本（text）
      </strong>
      ：如字体、文字颜色、文字对齐方式等。
     </li>
     <li>
      <strong>
       阴影（shadow）
      </strong>
      ：如阴影偏移、阴影颜色、阴影宽度等。
     </li>
     <li>
      <strong>
       布局（layout）
      </strong>
      ：如内边距（padding）、外边距（margin）等（主要用于布局系统）。
     </li>
     <li>
      <strong>
       轮廓（outline）
      </strong>
      ：与边框类似，但通常用于获得“聚焦”效果。
     </li>
     <li>
      <strong>
       其他
      </strong>
      ：如图像（image）、透明度（opa）、转换（transform）等高级属性。
     </li>
    </ul>
    <h4>
     <a id="13_lv_state_t_21">
     </a>
     1.3 状态（lv_state_t）
    </h4>
    <ul>
     <li>
      LVGL 中，每个对象可能处于不同的状态(
      <mark>
       <strong>
        状态可以组合
       </strong>
      </mark>
      )，如
      <strong>
       正常（LV_STATE_DEFAULT）
      </strong>
      、
      <strong>
       按下（LV_STATE_PRESSED）
      </strong>
      、
      <strong>
       禁用（LV_STATE_DISABLED）
      </strong>
      、
      <strong>
       聚焦（LV_STATE_FOCUSED）
      </strong>
      等。
     </li>
     <li>
      同一个对象可以在不同状态下应用不同的样式，例如按钮在按下时改变背景色。
     </li>
    </ul>
    <h4>
     <a id="14__26">
     </a>
     1.4 优先级和叠加
    </h4>
    <ul>
     <li>
      对象可以同时应用多个样式，这些样式将“叠加”到一起，后添加的样式优先级更高。
     </li>
     <li>
      样式属性可以相互覆盖，例如后添加的样式如果设置了
      <code>
       bg_color
      </code>
      ，则会覆盖先前样式的
      <code>
       bg_color
      </code>
      。
     </li>
    </ul>
    <h3>
     <a id="2__31">
     </a>
     2. 样式的创建与使用
    </h3>
    <h4>
     <a id="21__33">
     </a>
     2.1 创建样式
    </h4>
    <p>
     ​ 样式存储在 lv_style_t 变量中。LVGL 的样式系统采用了动态分配与静态分配相结合的方式，目的是尽可能地节省内存。
    </p>
    <pre><code class="prism language-c"><span class="token keyword">static</span> <span class="token class-name">lv_style_t</span> style_btn_default<span class="token punctuation">;</span>
<span class="token function">lv_style_init</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 初始化样式,确保处于一个已知的、干净的初始状态</span>
<span class="token function">lv_style_set_bg_color</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">,</span> <span class="token function">lv_color_hex</span><span class="token punctuation">(</span><span class="token number">0x000000</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <blockquote>
     <p>
      <strong>
       只存储实际设置的属性
      </strong>
      <br/>
      样式（lv_style_t）不会为所有可能的视觉属性预留固定空间，而是仅在需要时，通过动态分配（或扩展已有内存）来存储实际设置的属性值。这避免了为未使用的属性浪费内存。
     </p>
    </blockquote>
    <h4>
     <a id="22__46">
     </a>
     2.2 给对象添加样式
    </h4>
    <p>
     完成样式初始化后，你可以通过以下方式为对象应用样式：
    </p>
    <pre><code class="prism language-c"><span class="token class-name">lv_obj_t</span> <span class="token operator">*</span>btn <span class="token operator">=</span> <span class="token function">lv_btn_create</span><span class="token punctuation">(</span><span class="token function">lv_scr_act</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 创建一个按钮对象            </span>
<span class="token function">lv_obj_add_style</span><span class="token punctuation">(</span>btn<span class="token punctuation">,</span> <span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">,</span> LV_STATE_DEFAULT<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 应用于默认状态</span>
</code></pre>
    <ul>
     <li>
      第三个参数表示应用到哪种状态或场景。传 0 表示应用到
      <strong>
       LV_STATE_DEFAULT
      </strong>
      （即正常、释放的状态）。
     </li>
    </ul>
    <h4>
     <a id="23__57">
     </a>
     2.3 设置不同状态下的样式
    </h4>
    <p>
     如果你想在
     <strong>
      按钮按下（PRESSED）
     </strong>
     时使用不同的颜色，可以再次创建一个样式用于按下状态：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">static</span> <span class="token class-name">lv_style_t</span> style_btn_pressed<span class="token punctuation">;</span>
<span class="token function">lv_style_set_bg_color</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn<span class="token punctuation">,</span> <span class="token function">lv_color_hex</span><span class="token punctuation">(</span><span class="token number">0x007ACC</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>    <span class="token comment">// 改变背景色</span>
<span class="token function">lv_obj_add_style</span><span class="token punctuation">(</span>btn<span class="token punctuation">,</span> <span class="token operator">&amp;</span>style_btn_pressed<span class="token punctuation">,</span> LV_STATE_PRESSED<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 应用于按下状态</span>
</code></pre>
    <p>
     这样，当按钮处于按下状态时，就会显示新的背景色。
    </p>
    <h4>
     <a id="24__69">
     </a>
     2.4 样式的内存分配
    </h4>
    <p>
     假设你要创建一个按钮样式，只改变背景颜色和圆角属性。初始时，你可能这样写
    </p>
    <pre><code class="prism language-c"><span class="token keyword">static</span> <span class="token class-name">lv_style_t</span> style_btn<span class="token punctuation">;</span>
<span class="token function">lv_style_init</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 初始化样式</span>

<span class="token comment">// 设置背景颜色为红色</span>
<span class="token function">lv_style_set_bg_color</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn<span class="token punctuation">,</span> <span class="token function">lv_color_hex</span><span class="token punctuation">(</span><span class="token number">0xFF0000</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token comment">// 设置圆角半径为 5 像素</span>
<span class="token function">lv_style_set_radius</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn<span class="token punctuation">,</span> <span class="token number">5</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     <strong>
      实际情况：
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       初始状态
      </strong>
      <br/>
      当调用
      <code>
       lv_style_init(&amp;style_btn)
      </code>
      时，
      <code>
       style_btn.values_and_props
      </code>
      可能为空或分配了一个很小的内存块，因为还没有任何属性被设置。
     </li>
     <li>
      <strong>
       设置背景颜色
      </strong>
      <br/>
      当你调用
      <code>
       lv_style_set_bg_color(&amp;style_btn, lv_color_hex(0xFF0000));
      </code>
      时，内部会检查这个样式是否已经为背景颜色属性分配了内存。如果没有，它就会分配一个足够存储“背景颜色”这一属性的数据块，然后把属性 ID 和对应的值（红色）存储进去。
     </li>
     <li>
      <strong>
       设置圆角
      </strong>
      <br/>
      接下来，调用
      <code>
       lv_style_set_radius(&amp;style_btn, 5);
      </code>
      会让系统检查
      <code>
       style_btn.values_and_props
      </code>
      是否有足够空间存储另一个属性（圆角）。如果空间不够，系统会动态扩展该内存块，把新的属性和值也存储进去。
     </li>
    </ol>
    <p>
     这样，
     <strong>
      只为你真正设置的属性分配内存
     </strong>
     ，而不是预先为所有可能属性保留大量空间。最终，
     <code>
      lv_style_t
     </code>
     只占用较小内存，同时能够高效地存储你实际用到的样式属性。这就是 LVGL 为了节省内存而采用动态分配和按需扩展的设计理念。
    </p>
    <h3>
     <a id="3__95">
     </a>
     3. 普通样式和本地样式
    </h3>
    <h4>
     <a id="31__97">
     </a>
     3.1 普通样式（共享/常量样式）
    </h4>
    <p>
     ​ 普通样式通常是以全局定义的方式存在，常见于主题或默认样式中。这类样式在编译时就已经确定好，并且被多个对象共享，内存开销小，因为它们是只读的。
    </p>
    <p>
     示例：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">static</span> <span class="token class-name">lv_style_t</span> style_btn_default<span class="token punctuation">;</span>
<span class="token function">lv_style_init</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 初始化</span>
<span class="token function">lv_style_set_bg_color</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">,</span> <span class="token function">lv_color_hex</span><span class="token punctuation">(</span><span class="token number">0x000000</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">lv_style_set_border_width</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>style_btn_default<span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <h4>
     <a id="32_Local_styles_110">
     </a>
     3.2 本地样式（Local styles）
    </h4>
    <p>
     ​ 当你需要对单个对象进行定制化修改，而不希望影响到其他使用相同普通样式的对象时，LVGL 会为该对象创建一个本地样式副本。
    </p>
    <ul>
     <li>
      修改本地样式不会改变原始的普通样式，因此只会影响当前对象，从而实现个性化的外观设置。
     </li>
    </ul>
    <pre><code class="prism language-c"><span class="token class-name">lv_obj_t</span> <span class="token operator">*</span> btn_local <span class="token operator">=</span> <span class="token function">lv_btn_create</span><span class="token punctuation">(</span><span class="token function">lv_scr_act</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">lv_obj_set_style_bg_color</span><span class="token punctuation">(</span>btn_local<span class="token punctuation">,</span> <span class="token function">lv_color_hex</span><span class="token punctuation">(</span><span class="token number">0xFF5722</span><span class="token punctuation">)</span><span class="token punctuation">,</span> LV_PART_MAIN<span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">lv_obj_set_style_radius</span><span class="token punctuation">(</span>btn_local<span class="token punctuation">,</span> <span class="token number">8</span><span class="token punctuation">,</span> LV_PART_MAIN<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
