    <h3>
     <a id="_0">
     </a>
     引言
    </h3>
    <p>
     在 Flutter 开发中，Dart 作为其编程语言，采用了面向对象的编程范式。面向对象编程（OOP）将数据和操作数据的方法封装在一起，形成对象，以提高代码的可维护性、可扩展性和可重用性。本文将详细介绍 Dart 面向对象编程的核心概念，包括类、对象、继承、多态、抽象类和接口等，并结合代码示例进行说明。
    </p>
    <h3>
     <a id="1__4">
     </a>
     1. 类和对象
    </h3>
    <p>
     类是对象的蓝图，它定义了对象的属性和方法。对象是类的实例，通过类可以创建多个不同的对象。
    </p>
    <h4>
     <a id="_7">
     </a>

    </h4>

    <h3>
     <a id="2__42">
     </a>
     2. 继承
    </h3>
    <p>
     继承是面向对象编程的重要特性之一，它允许一个类（子类）继承另一个类（父类）的属性和方法。子类可以扩展父类的功能，也可以重写父类的方法。
    </p>
    <h4>
     <a id="_45">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token comment">// 定义父类</span>
<span class="token keyword">class</span> <span class="token class-name">Animal</span> <span class="token punctuation">{<!-- --></span>
  <span class="token class-name">String</span> name<span class="token punctuation">;</span>

  <span class="token class-name">Animal</span><span class="token punctuation">(</span><span class="token keyword">this</span><span class="token punctuation">.</span>name<span class="token punctuation">)</span><span class="token punctuation">;</span>

  <span class="token keyword">void</span> <span class="token function">makeSound</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'动物发出声音'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token comment">// 定义子类</span>

  <span class="token metadata function">@override</span>
  <span class="token keyword">void</span> <span class="token function">makeSound</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'汪汪汪'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>

<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_74">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      <code>
       class Animal
      </code>
      是父类，包含一个属性
      <code>
       name
      </code>
      和一个方法
      <code>
       makeSound
      </code>
      。
     </li>
     <li>
      <code>
       class Dog extends Animal
      </code>
      表示
      <code>
       Dog
      </code>
      类继承自
      <code>
       Animal
      </code>
      类。
     </li>
     <li>
      <code>
       Dog(String name) : super(name)
      </code>
      是
      <code>
       Dog
      </code>
      类的构造函数，使用
      <code>
       super(name)
      </code>
      调用父类的构造函数。
     </li>
     <li>
      <code>
       @override
      </code>
      注解表示重写父类的方法，
      <code>
       Dog
      </code>

      <code>
       makeSound
      </code>

      </code>

     </li>
     <li>

      <code>

      <code>
       Dog
      </code>

      <code>
       dog

      <code>
       makeSound
      </code>
      方法。
     </li>
    </ul>
    <h3>
     <a id="3__81">
     </a>
     3. 多态
    </h3>
    <p>
     多态是指同一个方法调用可以根据对象的不同类型表现出不同的行为。在 Dart 中，多态主要通过继承和方法重写来实现。
    </p>
    <h4>
     <a id="_84">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token comment">// 定义父类</span>
<span class="token keyword">class</span> <span class="token class-name">Shape</span> <span class="token punctuation">{<!-- --></span>
  <span class="token keyword">void</span> <span class="token function">draw</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'绘制图形'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token comment">// 定义子类</span>
<span class="token keyword">class</span> <span class="token class-name">Circle</span> <span class="token keyword">extends</span> <span class="token class-name">Shape</span> <span class="token punctuation">{<!-- --></span>
  <span class="token metadata function">@override</span>
  <span class="token keyword">void</span> <span class="token function">draw</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'绘制圆形'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token keyword">class</span> <span class="token class-name">Square</span> <span class="token keyword">extends</span> <span class="token class-name">Shape</span> <span class="token punctuation">{<!-- --></span>
  <span class="token metadata function">@override</span>
  <span class="token keyword">void</span> <span class="token function">draw</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'绘制正方形'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token class-name">Shape</span> shape1 <span class="token operator">=</span> <span class="token class-name">Circle</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token class-name">Shape</span> shape2 <span class="token operator">=</span> <span class="token class-name">Square</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

  shape1<span class="token punctuation">.</span><span class="token function">draw</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  shape2<span class="token punctuation">.</span><span class="token function">draw</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_117">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      <code>
       class Shape
      </code>
      是父类，定义了一个
      <code>
       draw
      </code>
      方法。
     </li>
     <li>
      <code>
       class Circle extends Shape
      </code>
      和
      <code>
       class Square extends Shape
      </code>
      分别定义了两个子类，并重写了
      <code>
       draw
      </code>
      方法。
     </li>
     <li>
      在
      <code>
       main
      </code>
      函数中，将
      <code>
       Circle
      </code>
      和
      <code>
       Square
      </code>
      类的对象赋值给
      <code>
       Shape
      </code>
      类型的变量
      <code>
       shape1
      </code>
      和
      <code>
       shape2
      </code>
      。
     </li>
     <li>
      调用
      <code>
       shape1.draw()
      </code>
      和
      <code>
       shape2.draw()
      </code>
      时，会根据对象的实际类型调用相应的
      <code>
       draw
      </code>
      方法，体现了多态性。
     </li>
    </ul>
    <h3>
     <a id="4__123">
     </a>
     4. 抽象类和抽象方法
    </h3>
    <p>
     抽象类是一种不能被实例化的类，它主要用于定义一些通用的属性和方法，供子类继承和实现。抽象方法是在抽象类中声明但没有实现的方法，子类必须实现这些抽象方法。
    </p>
    <h4>
     <a id="_126">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token comment">// 定义抽象类</span>
<span class="token keyword">abstract</span> <span class="token keyword">class</span> <span class="token class-name">Vehicle</span> <span class="token punctuation">{<!-- --></span>
  <span class="token comment">// 抽象方法</span>
  <span class="token keyword">void</span> <span class="token function">start</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

  <span class="token comment">// 普通方法</span>
  <span class="token keyword">void</span> <span class="token function">stop</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'车辆停止'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token comment">// 定义子类</span>

  <span class="token metadata function">@override</span>
  <span class="token keyword">void</span> <span class="token function">start</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'汽车启动'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token class-name">Car</span> car <span class="token operator">=</span> <span class="token class-name">Car</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  car<span class="token punctuation">.</span><span class="token function">start</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_154">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      <code>
       abstract class Vehicle
      </code>
      定义了一个抽象类
      <code>
       Vehicle
      </code>
      。
     </li>
     <li>
      <code>
       void start()
      </code>
      是抽象方法，没有具体的实现，子类必须实现该方法。
     </li>
     <li>
      <code>
       void stop()
      </code>
      是普通方法，有具体的实现。
     </li>
     <li>
      <code>
       class Car extends Vehicle
      </code>
      表示
      <code>
       Car
      </code>
      类继承自
      <code>
       Vehicle
      </code>
      类，并实现了
      <code>
       start
      </code>
      方法。
     </li>
     <li>
      在
      <code>
       main
      </code>
      函数中，创建了一个
      <code>
       Car
      </code>
      类的对象
      <code>
       car
      </code>
      ，并调用其
      <code>
       start
      </code>
      和
      <code>
       stop
      </code>
      方法。
     </li>
    </ul>
    <h3>
     <a id="5__161">
     </a>
     5. 接口
    </h3>
    <p>
     在 Dart 中，接口的概念与抽象类类似，但接口只包含抽象方法，不包含属性和具体实现的方法。类可以实现一个或多个接口。
    </p>
    <h4>
     <a id="_164">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token comment">// 定义接口</span>

<span class="token keyword">abstract</span> <span class="token keyword">class</span> <span class="token class-name">Swimmable</span> <span class="token punctuation">{<!-- --></span>

<span class="token punctuation">}</span>
