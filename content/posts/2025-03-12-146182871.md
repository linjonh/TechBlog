    <h2 id="%E4%B8%80%E3%80%81moment().isValid()%E9%AA%8C%E8%AF%81%E6%97%B6%E9%97%B4%E6%98%AF%E5%90%A6%E6%9C%89%E6%95%88" name="%E4%B8%80%E3%80%81moment().isValid()%E9%AA%8C%E8%AF%81%E6%97%B6%E9%97%B4%E6%98%AF%E5%90%A6%E6%9C%89%E6%95%88">
     一、moment().isValid()验证时间是否有效
    </h2>
    <blockquote>
     <p>
      验证当前时间是否有效，有效返回true,反之返回false
     </p>
    </blockquote>
    <pre><code class="language-TypeScript">    console.log(moment().isValid());//不传时间默认为当前时间 输出true
    console.log(moment(null).isValid());//输出false</code></pre>
    <h2 id="%E4%BA%8C%E3%80%81moment().second()%E8%8E%B7%E5%8F%96%E7%A7%92%E6%95%B0%E6%88%96%E8%80%85%E8%AE%BE%E7%BD%AE%E7%A7%92%E6%95%B0" name="%E4%BA%8C%E3%80%81moment().second()%E8%8E%B7%E5%8F%96%E7%A7%92%E6%95%B0%E6%88%96%E8%80%85%E8%AE%BE%E7%BD%AE%E7%A7%92%E6%95%B0">
     二、moment().second()获取秒数或者设置秒数
    </h2>
    <blockquote>
     <p>
      moment().second()和moment().seconds()都可以获取或者设置秒数，如果超过59则会增加分钟数，以此类推:获取分钟为moment().minute()，获取小时
     </p>
    </blockquote>
    <pre><code class="language-TypeScript">console.log(moment().second(67));//返回当前时间加一分钟秒数为7秒
console.log(moment().second(4));//返回当前时间且秒数为4s
console.log(moment().second());//当前时间的秒数，每秒变化</code></pre>
    <pre><code class="language-TypeScript">moment().minute(Number);
moment().minute(); // Number
moment().minutes(Number);
moment().minutes(); // Number
moment().hour(Number);
moment().hour(); // Number
moment().hours(Number);
moment().hours(); // Number</code></pre>
    <h2 id="%C2%A0%E4%B8%89%E3%80%81moment().day()%E8%8E%B7%E5%8F%96%E6%98%9F%E6%9C%9F%E6%88%96%E8%80%85%E8%AE%BE%E7%BD%AE%E6%98%9F%E6%9C%9F" name="%C2%A0%E4%B8%89%E3%80%81moment().day()%E8%8E%B7%E5%8F%96%E6%98%9F%E6%9C%9F%E6%88%96%E8%80%85%E8%AE%BE%E7%BD%AE%E6%98%9F%E6%9C%9F">
     三、moment().day()获取星期或者设置星期
    </h2>
    <blockquote>
     <p>
      该方法可用于设置星期几，星期日为 0，星期六为 6。
     </p>
    </blockquote>
    <pre><code class="language-TypeScript"> console.log(moment().day(0));//返回日期为星期天
 console.log(moment().day(1));//返回日期为星期一</code></pre>

    <h2 id="%E5%9B%9B%E3%80%81moment().add()%E5%8A%A0%E6%B3%95%E6%93%8D%E4%BD%9C" name="%E5%9B%9B%E3%80%81moment().add()%E5%8A%A0%E6%B3%95%E6%93%8D%E4%BD%9C">
     四、moment().add()加法操作
    </h2>
    <pre><code>moment().add(Number, String);
moment().add(Duration);
moment().add(Object);</code></pre>
    <pre><code class="language-TypeScript">moment().add(7, 'days');//当前时间加7天
moment().add(7, 'days').add(1, 'months'); // 当前时间加一个月零7天
moment().add({ days: 7, months: 1 }); //传参使用对象的形式传</code></pre>
    <p>
     当前时间为2025-3-11加7天后返回值为3.18日的实例

    <h2 id="%E4%BA%94%E3%80%81moment().subtract()%E5%87%8F%E6%B3%95%E6%93%8D%E4%BD%9C" name="%E4%BA%94%E3%80%81moment().subtract()%E5%87%8F%E6%B3%95%E6%93%8D%E4%BD%9C">
     五、moment().subtract()减法操作
    </h2>
    <pre><code>moment().subtract(Number, String);
moment().subtract(Duration);
moment().subtract(Object);</code></pre>
    <pre><code class="language-TypeScript">moment().subtract(7, 'days');//当前时间减7天
moment().subtract(7, 'days').add(1, 'months'); // 当前时间减一个月零7天
moment().subtract({ days: 7, months: 1 }); //传参使用对象的形式传</code></pre>
    <h2 id="%E5%85%AD%E3%80%81moment.max()%E6%9C%80%E5%A4%A7%E5%80%BC%C2%A0" name="%E5%85%AD%E3%80%81moment.max()%E6%9C%80%E5%A4%A7%E5%80%BC%C2%A0">
     六、moment.max()最大值
    </h2>
    <blockquote>
     <p>
      返回给定实例的最大值，可以找到给定实例中最大的时间
     </p>
    </blockquote>
    <pre><code>moment.max(Moment[,Moment...]);
moment.max(Moment[]);</code></pre>
    <pre><code class="language-TypeScript">  const a = moment();
  const b = moment(a).add(7, 'd');
  console.log(moment.max(a, b));//返回较大值b</code></pre>
    <h2 id="%E4%B8%83%E3%80%81moment.min()%E6%9C%80%E5%B0%8F%E5%80%BC" name="%E4%B8%83%E3%80%81moment.min()%E6%9C%80%E5%B0%8F%E5%80%BC">
     七、moment.min()最小值
    </h2>
    <pre><code class="language-TypeScript">moment.min(Moment[,Moment...]);
moment.min(Moment[]);</code></pre>
    <pre><code class="language-TypeScript">  const a = moment();
  const b = moment(a).add(7, 'd');
  console.log(moment.min(a, b));//返回较小值a</code></pre>
    <h2 id="%E5%85%AB%E3%80%81%E5%85%8B%E9%9A%86%E6%97%B6%E9%97%B4" name="%E5%85%AB%E3%80%81%E5%85%8B%E9%9A%86%E6%97%B6%E9%97%B4">
     八、克隆时间
    </h2>
    <blockquote>
     <p>
      moment中调任何操作方法都会改变时间，所以如果不想修改原时间就复制一个时间进行具体操作
     </p>
    </blockquote>
    <pre><code class="language-TypeScript"> //第一种方法
 const a = moment();
 const b = moment(a);
 //第二种方法
 const a = moment();
 const b = a.clone();</code></pre>
