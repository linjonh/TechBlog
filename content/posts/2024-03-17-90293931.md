    <p>
     <strong>
      Win7下按详细信息显示:
     </strong>
     <br/>
     1.打开一个文件夹，默认是图标显示的，我们把视图更改为"详细信息"，然后我们点击"组织"以及"文件夹和搜索选项"：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eb022ca74d4c9194e4ed83c861902bcf.png">
      <br/>
      2.然后我们点击"查看"和"应用到文件夹"然后点击确定就可以使文件默认按详细信息进行显示了，非常简单：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a17c2f3685883bcd9d21e478fda07388.png">
       <br/>
       <strong>
        Win10下按详细信息显示:
       </strong>
       <br/>
       1.点击查看，选择详细信息:
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac7598ae8c7f36287faabf6cc9806847.png">
        <br/>
        2.依次点击查看、选项、更改文件夹和搜索选项:
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/78eb3104203be804da91a7bacf6529a5.png">
         <br/>
         3.在弹出的窗口依次点击查看、应用到文件夹最后确定就可以了:
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bbf36804d9aa66164e6e849d2f8e71a1.png"/>
