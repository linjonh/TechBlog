    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px; text-align:center">
     <strong>
      <span style="font-size:18px">
       音视频同步问题
      </span>
     </strong>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      音视频流里都包含了播放速率的信息，音频使用采样率来表示，而视频则采用
      <span style="font-family:'Times New Roman'">
       f/s
      </span>
      <span style="font-family:宋体">
       来表示，但是我们却不能简单地用这两个数据来对音视频进行同步，我们需要使用
      </span>
      <span style="font-family:'Times New Roman'">
       DTS
      </span>
      <span style="font-family:宋体">
       （解码时间戳）和
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       （播放时间戳）这两个数据；我们知道影视数据在存储时，会存在多种帧形式，例如
      </span>
      <span style="font-family:'Times New Roman'">
       MPEG
      </span>
      <span style="font-family:宋体">
       中就采用了
      </span>
      <span style="font-family:'Times New Roman'">
       I
      </span>
      <span style="font-family:宋体">
       ，
      </span>
      <span style="font-family:'Times New Roman'">
       B
      </span>
      <span style="font-family:宋体">
       和
      </span>
      <span style="font-family:'Times New Roman'">
       P
      </span>
      <span style="font-family:宋体">
       ，由于
      </span>
      <span style="font-family:'Times New Roman'">
       B
      </span>
      <span style="font-family:宋体">
       帧的存在使得
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       和
      </span>
      <span style="font-family:'Times New Roman'">
       DTS
      </span>
      <span style="font-family:宋体">
       存在不同（原因见附录），如图
      </span>
      <span style="font-family:'Times New Roman'">
       1
      </span>
      <span style="font-family:宋体">
       所示为一个简单的例子；当然真正影响我们音视频同步的是
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      我们可以从影视文件中获得包的
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       ，但是我们无法直接获得帧（我们真正关心的）的
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       ，解决办法是用一帧的第一个包的
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       作为这一帧的
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       ，这是可行的，因为当一个包开始一帧时，
      </span>
      <span style="font-family:'Times New Roman'">
       avcodec_decode_video()
      </span>
      <span style="font-family:宋体">
       会调用为帧申请存储空间的函数，我们可以重写这个函数，在其中加入获取包
      </span>
      <span style="font-family:'Times New Roman'">
       DTS
      </span>
      <span style="font-family:宋体">
       的方法（这是很容易的），因为
      </span>
      <span style="font-family:'Times New Roman'">
       ffmpeg
      </span>
      <span style="font-family:宋体">
       会重新给包进行排序，因此被
      </span>
      <span style="font-family:'Times New Roman'">
       avcodec_decode_video()
      </span>
      <span style="font-family:宋体">
       处理过的包的
      </span>
      <span style="font-family:'Times New Roman'">
       DTS
      </span>
      <span style="font-family:宋体">
       和返回的帧的
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       是相同的，这样就可以得到帧的
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       了。
      </span>
     </span>
     <span style="font-size:18px">
      当然有时我们可能没法得到这个
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       ，那样我们使用内部
      </span>
      <span style="font-family:'Times New Roman'">
       video_clock
      </span>
      <span style="font-family:宋体">
       （记录视频已经过去的时间）；
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      这样我们就可以使用
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       来重新队列化我们的帧（
      </span>
      <span style="font-family:'Times New Roman'">
       queue_picture
      </span>
      <span style="font-family:宋体">
       ）；
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <span style="font-family:宋体">
      </span>
     </span>
    </p>
    <p class="p0" style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px; margin-bottom:0pt; margin-top:0pt">
     <img alt="" src="http://hi.csdn.net/attachment/201111/19/0_1321713231EDuX.gif" style="border:none; max-width:100%">
      <br/>
     </img>
    </p>
    <br style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px"/>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      然后，我们可以去获得音频的播放时间
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      get_audio_clock
      <span style="font-family:宋体">
       （）：
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      把
      <span style="font-family:'Times New Roman'">
       audio_clock
      </span>
      <span style="font-family:宋体">
       作为音频的
      </span>
      <span style="font-family:'Times New Roman'">
       pts
      </span>
      <span style="font-family:宋体">
       ，但是在
      </span>
      <span style="font-family:'Times New Roman'">
       audio_decode_frame
      </span>
      <span style="font-family:宋体">
       （）计算的
      </span>
      <span style="font-family:'Times New Roman'">
       audio_clock
      </span>
      <span style="font-family:宋体">
       的是假定缓冲满的情况，而实际上可能缓冲是不满的，我们需要减去空闲的部分的时间：
      </span>
      <span style="font-family:'Times New Roman'">
       pts -= (double)hw_buf_size / bytes_per_sec;
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      audio_decode_frame
      <span style="font-family:宋体">
       （）：音频解码
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      读取包时，获取其
      <span style="font-family:'Times New Roman'">
       pts
      </span>
      <span style="font-family:宋体">
       ，放入
      </span>
      <span style="font-family:'Times New Roman'">
       audio_clock
      </span>
      <span style="font-family:宋体">
       中；
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      播放时根据缓冲大小和播放速率计算播放时间；
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      获得这两个pts
      <span style="font-family:宋体">
       后，我们有三个选择：视频同步音频
      </span>
      <span style="font-family:'Times New Roman'">
       (
      </span>
      <span style="font-family:宋体">
       计算音视频
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       只差，来决定视频是否有延迟
      </span>
      <span style="font-family:'Times New Roman'">
       )
      </span>
      <span style="font-family:宋体">
       、音频同步视频（根据音视频
      </span>
      <span style="font-family:'Times New Roman'">
       PTS
      </span>
      <span style="font-family:宋体">
       差值调整音频取的样值，即改变音频缓冲区的大小）和音频视频同步外部时钟（同前一个）。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <img alt="" src="http://hi.csdn.net/attachment/201111/19/0_132171299311Z5.gif" style="border:none; max-width:100%"/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      <br/>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      附录：
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      I frame
      <span style="font-family:宋体">
       ：帧内编码帧 又称
      </span>
      <span style="font-family:'Times New Roman'">
       intra
      </span>
      picture
      <span style="font-family:宋体">
       ，
      </span>
      <span style="font-family:'Times New Roman'">
       I
      </span>
      <span style="font-family:宋体">
       帧通常是每个
      </span>
      <span style="font-family:'Times New Roman'">
       GOP
      </span>
      <span style="font-family:宋体">
       （
      </span>
      <span style="font-family:'Times New Roman'">
       MPEG
      </span>
      <span style="font-family:宋体">
       所使用的一种视频压缩技术）的第一个帧，经过适度地压缩，做为随机访问的参考点，可以当成图象。
      </span>
      <span style="font-family:'Times New Roman'">
       I
      </span>
      <span style="font-family:宋体">
       帧可以看成是一个图像经过压缩后的产物。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      P frame:
      <span style="font-family:宋体">
       前向预测编码帧 又称
      </span>
      <span style="font-family:'Times New Roman'">
       predictive-frame
      </span>
      <span style="font-family:宋体">
       ，通过充分将低于图像序列中前面已编码帧的时间冗余信息来压缩传输数据量的编码图像，也叫预测帧；
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      B frame:
      <span style="font-family:宋体">
       双向预测内插编码帧 又称
      </span>
      <span style="font-family:'Times New Roman'">
       bi-directional interpolated prediction frame
      </span>
      <span style="font-family:宋体">
       ，既考虑与源图像序列前面已编码帧，也顾及源图像序列后面已编码帧之间的时间冗余信息来压缩传输数据量的编码图像，也叫双向预测帧；
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      I frame:  自身可以通过视频解压算法解压成一张单独的完整的图片。
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      P frame
      <span style="font-family:宋体">
       ：需要参考其前面的一个
      </span>
      <span style="font-family:'Times New Roman'">
       I frame
      </span>
      <span style="font-family:宋体">
       或者
      </span>
      <span style="font-family:'Times New Roman'">
       B frame
      </span>
      <span style="font-family:宋体">
       来生成一张完整的图片。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      B frame:
      <span style="font-family:宋体">
       则要参考其前一个
      </span>
      <span style="font-family:'Times New Roman'">
       I
      </span>
      <span style="font-family:宋体">
       或者
      </span>
      <span style="font-family:'Times New Roman'">
       P
      </span>
      <span style="font-family:宋体">
       帧及其后面的一个
      </span>
      <span style="font-family:'Times New Roman'">
       P
      </span>
      <span style="font-family:宋体">
       帧来生成一张完整的图片。
      </span>
     </span>
    </p>
    <p style="color:rgb(51,51,51); font-family:Arial; font-size:14px; line-height:26px">
     <span style="font-size:18px">
      I frame
      <span style="font-family:宋体">
       的解码不依赖于任何的其它的帧
      </span>
      <span style="font-family:'Times New Roman'">
       .
      </span>
      <span style="font-family:宋体">
       而
      </span>
      <span style="font-family:'Times New Roman'">
       p frame
      </span>
      <span style="font-family:宋体">
       的解码则依赖于其前面的
      </span>
      <span style="font-family:'Times New Roman'">
       I frame
      </span>
      <span style="font-family:宋体">
       或者
      </span>
      <span style="font-family:'Times New Roman'">
       P frame.B frame
      </span>
      <span style="font-family:宋体">
       的解码则依赖于其前的最近的一个
      </span>
      <span style="font-family:'Times New Roman'">
       I frame
      </span>
      <span style="font-family:宋体">
       或者
      </span>
      <span style="font-family:'Times New Roman'">
       P frame
      </span>
      <span style="font-family:宋体">
       及其后的最近的一个
      </span>
      <span style="font-family:'Times New Roman'">
       P frame.
      </span>
     </span>
    </p>
