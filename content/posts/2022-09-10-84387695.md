    <h4>
     <a id="idea_0">
     </a>
     一、在idea中出现如下错误
    </h4>
    <pre><code>Error:(27, 35) java: -source 1.5 中不支持 multi-catch 语句
  (请使用 -source 7 或更高版本以启用 multi-catch 语句)
</code></pre>
    <p>
     <strong>
      解决办法：
     </strong>
     <br/>
     <strong>
      在idea中重新设置
     </strong>
     <br/>
     File -&gt;Project Structure-&gt;Project Settings -&gt; Modules -&gt; Module名字 -&gt; Sources -&gt; Language Level-&gt;选择project default
    </p>
    <h4>
     <a id="_java_source_15__lambda__9">
     </a>
     二、 java: -source 1.5 中不支持 lambda 表达式
    </h4>
    <pre><code>Error:(26, 38) java: -source 1.5 中不支持 lambda 表达式
 (请使用 -source 8 或更高版本以启用 lambda 表达式)
</code></pre>
    <p>
     <strong>
      解决办法:
     </strong>
     <br/>
     打开 File -&gt; settings，搜索 Java compiler
     <br/>
     此时会发现右下角的每一个module裡的Compilation options中，都多加了一个-version
     <br/>
     把每个module的-version都删除掉，就可以正常运行了！
     <br/>
     （或者改为你使用的java版本就可以了（我的是java1.8））
    </p>
