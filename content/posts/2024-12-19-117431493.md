    <h3>
     <a id="_0">
     </a>
     是时候为新入门的运维精英门，留下一笔宝贵的财富了
    </h3>
    <blockquote>
     <p>
      突然感觉自己老了，真的，有时候精力都快跟不上了，想想自己刚入这行的时候，为了能通过面试，全网找面试题，面试资料。不知道自己还能在运维界蹦跶多久。蹭着还有些时日，自己也是想了好久，搭建一个平台，收集和补录大家提供的面试题，这个平台可能初期对了些大佬门没有任何的作用，但是小编保证后期与新技术的接入，大佬们也会稍微的关注点的
     </p>
    </blockquote>
    <h4>
     <a id="_2">
     </a>
     小编的想法
    </h4>
    <h5>
     <a id="_3">
     </a>
     第一，搭建一个关于运维面试题之类的网站，自己维护，当然了也欢迎大家可以和小编一起维护
    </h5>
    <h5>
     <a id="7030_4">
     </a>
     第二、所有的面试题，70%来自网上，还有30%要做到金三银四实时更新，当然了，大家也可以提供一下面试题，
     <em>
      <strong>
       小编在向平台录入题目的同时也会在每道题的后方留下面试题提供着的博客地址
      </strong>
     </em>
    </h5>
    <h5>
     <a id="_5">
     </a>
     第三、小编尽量针对中高难度的面试题，会出视频进行讲解，关于这个还是希望大家可以一起来参加，毕竟小编的精力是有限的
    </h5>
    <h5>
     <a id="_7">
     </a>
     第四、平台在运行期间百分之百会遇见更种各样的问题。比如病毒，等等问题，这个时候就是锻炼技术的时候了，大家也可以广发的参与进来
    </h5>
    <h5>
     <a id="_9">
     </a>
     第五、目前小编计划运行平台为一年，如果运行的可以的话，就让他运行下去，毕竟小编口袋也不是很充足
    </h5>
    <h5>
     <a id="_11">
     </a>
     第六、平台采用的源码是开源的，也就是面临的挑战很大，平台只是用来学习和技术提升。
    </h5>
    <h4>
     <a id="_12">
     </a>
     小编的几个担心
    </h4>
    <h5>
     <a id="bug_13">
     </a>
     第一、毕竟是个人能力有限，本平台也就是类似与考试平台，大家可以到上面做做题，发现bug的时候，还是希望大家稍微温柔点，如果能告知小编，小编将感激不尽
    </h5>
    <h5>
     <a id="_14">
     </a>
     第二、大佬请绕走，小编也只是想提升一下自己的技术
    </h5>
    <h5>
     <a id="_15">
     </a>
     第三、平台无人问津，希望大家还是要积极踊跃参与，同时也想找一个同事共同参与整个平台的维护中
    </h5>
    <h5>
     <a id="_17">
     </a>
     第四、也是小编最怕的，由于是第一次，没有太多的经验，小编也会保证，哪怕只有一个用户，小编也会前方百计的维护平台的运行
    </h5>
    <h4>
     <a id="_20">
     </a>
     小编的几个构想
    </h4>
    <h5>
     <a id="_21">
     </a>
     第一、如果可以，小编要扩展到其他的领域，现在要求的是一专多能，平台做大做强了，小编要把平台做到公认
    </h5>
    <h5>
     <a id="_23">
     </a>
     第二、针对可以复现的面试题，小编也会找台服务器做到问题复现，俗话说得好，题做前遍，不如手练一遍，针对个别特殊的题，大家可以@小编，小编应大家要求，问题复现
    </h5>
    <h4>
     <a id="_25">
     </a>
     结束语
    </h4>
    <h5>
     <a id="_26">
     </a>
     其实还有好多想说的，小编是一个实干家，当初也层暗恋过一个女研发，但是吧，呵呵，当时。。。。都是泪，最后祝愿大家可以心想事成，么么哒
    </h5>
