    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       摄像头模块的ISP（图像信号处理器）处理流程是对图像传感器输出的原始信号
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       进行系统性优化的过程，主要分为以下关键步骤及对应功能模块：
      </span>
     </span>
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        一、原始信号输入与预处理
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       传感器信号捕获
      </strong>
     </strong>
     ‌
     <br/>
     CMOS/CCD传感器将光信号转换为模拟电信号，并通过A/D转换生成Bayer阵列格式的原始数据（RAW-RGB）‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       黑电平校正（BLC）
      </strong>
     </strong>
     ‌
     <br/>
     消除传感器暗电流噪声，通过标定偏移量确保暗区细节保留，维持信号线性一致性‌。
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        二、光学与传感器缺陷补偿
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       镜头阴影校正（Lens Shading）
      </strong>
     </strong>
     ‌
     <br/>
     补偿因镜头光学特性导致的边缘亮度衰减和颜色偏差‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       坏点校正（BPC）
      </strong>
     </strong>
     ‌
     <br/>
     检测并修复传感器坏点或异常像素，避免图像中出现固定噪声点‌。
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        三、颜色与细节恢复
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       去马赛克（Demosaic/CIP）
      </strong>
     </strong>
     ‌
     <br/>
     将Bayer阵列的单一颜色通道插值恢复为全彩色RGB图像‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       去噪（DNS）
      </strong>
     </strong>
     ‌
     <br/>
     通过空域或时域滤波算法降低图像噪声，平衡细节保留与降噪效果‌。
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        四、色彩与亮度优化
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       自动白平衡（AWB）
      </strong>
     </strong>
     ‌
     <br/>
     根据环境光源调整色温，使白色物体在不同光照下呈现自然效果‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       颜色矫正（Color Correction）
      </strong>
     </strong>
     ‌
     <br/>
     修正因传感器或滤镜引起的色彩偏差，提升色彩还原准确性‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       Gamma校正
      </strong>
     </strong>
     ‌
     <br/>
     调整亮度曲线以适配人眼视觉特性，优化局部与全局对比度‌。
    </p>
    <h4 style="margin-left:0pt; margin-right:0pt; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/0cf86227d0f043d19c4ee59be184fc85.jpeg"/>
    </h4>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        五、输出与后处理
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       色彩空间转换
      </strong>
     </strong>
     ‌
     <br/>
     将RGB数据转换为YUV等标准色彩空间，便于显示、编码或传输‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       动态范围压缩与锐化
      </strong>
     </strong>
     ‌
     <br/>
     针对高动态场景进行色调映射，并通过锐化增强边缘清晰度‌。
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        六、实时控制算法（3A算法）
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       自动曝光（AE）
      </strong>
     </strong>
     ‌：动态调整曝光时间和增益，防止过曝或欠曝‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       自动对焦（AF）
      </strong>
     </strong>
     ‌：通过对比度或相位检测实现精准对焦‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       自动白平衡（AWB）
      </strong>
     </strong>
     ‌：如前述，贯穿色彩处理阶段‌。
    </p>
    <h4 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <strong>
      <strong>
       <strong>
        流程架构差异
       </strong>
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       外置ISP
      </strong>
     </strong>
     ‌：独立芯片处理，灵活性高，适用于专业影像设备‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ‌
     <strong>
      <strong>
       内置ISP
      </strong>
     </strong>
     ‌：集成于SoC中，成本低，多见于手机等消费级设备‌。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       以上流程各模块顺序可能因ISP芯片设计或场景需求略有调整，但核心目标均为提升图像质量与视觉一致性
      </span>
     </span>
