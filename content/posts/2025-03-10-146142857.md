    <h2>
     SQL 语法
    </h2>
    <h3>
     引言
    </h3>
    <p>
     SQL（Structured Query Language）是一种用于管理关系型数据库的编程语言。它被广泛用于数据库的创建、查询、更新和管理。SQL语法是数据库操作的基础，掌握SQL语法对于数据库管理员和开发者来说至关重要。本文将详细介绍SQL的基本语法，包括数据定义语言（DDL）、数据操作语言（DML）和数据控制语言（DCL）。
    </p>
    <h3>
     数据定义语言（DDL）
    </h3>
    <h4>
     1. 创建数据库（CREATE DATABASE）
    </h4>
    <pre><code class="language-markdown">CREATE DATABASE 数据库名;
</code></pre>
    <h4>
     2. 创建表（CREATE TABLE）
    </h4>
    <pre><code class="language-markdown">CREATE TABLE 表名 (
  列名1 数据类型,
  列名2 数据类型,
  ...
);
</code></pre>
    <h4>
     3. 修改表结构（ALTER TABLE）
    </h4>
    <pre><code class="language-markdown">ALTER TABLE 表名
ADD 列名 数据类型;
ALTER TABLE 表名
DROP COLUMN 列名;
</code></pre>
    <h4>
     4. 删除数据库（DROP DATABASE）
    </h4>
    <pre><code class="language-markdown">DROP DATABASE 数据库名;
</code></pre>
    <h4>
     5. 删除表（DROP TABLE）
    </h4>
    <pre><code class="language-markdown">DROP TABLE 表名;
</code></pre>
    <h3>
     数据操作语言（DML）
    </h3>
    <h4>
     1. 插入数据（INSERT）
    </h4>
    <pre><code class="language-markdown">INSERT INTO 表名 (列名1, 列名2, ...)
VALUES (值1, 值2, ...);
</code></pre>
    <h4>
     2. 更新数据（UPDATE）
    </h4>
    <pre><code class="language-markdown">UPDATE 表名
SET 列名1 = 值1, 列名2 = 值2, ...
WHERE 条件;
</code></pre>
    <h4>
     3. 删除数据（DELETE）
    </h4>
    <pre><code class="language-markdown">DELETE FROM 表名
WHERE 条件;
</code></pre>
    <h4>
     4. 查询数据（SELECT）
    </h4>
    <pre><code class="language-markdown">SELECT 列名1, 列名2, ...
FROM 表名
WHERE 条件;
</code></pre>
    <h3>
     数据控制语言（DCL）
    </h3>
    <h4>
     1. 授予权限（GRANT）
    </h4>
    <pre><code class="language-markdown">GRANT 权限 ON 表名 TO 用户;
</code></pre>
    <h4>
     2. 撤销权限（REVOKE）
    </h4>
    <pre><code class="language-markdown">REVOKE 权限 ON 表名 FROM 用户;
</code></pre>
    <h4>
     3. 查看权限（SHOW）
    </h4>
    <pre><code class="language-markdown">SHOW GRANTS;
</code></pre>
    <h3>
     分页查询
    </h3>
    <h4>
     1. MySQL
    </h4>
    <pre><code class="language-markdown">SELECT 列名1, 列名2, ...
FROM 表名
LIMIT 页码, 每页显示条数;
</code></pre>
    <h4>
     2. SQL Server
    </h4>
    <pre><code class="language-markdown">SELECT 列名1, 列名2, ...
FROM 表名
OFFSET 页码 * 每页显示条数 ROWS
FETCH NEXT 每页显示条数 ROWS ONLY;
</code></pre>
    <h3>
     总结
    </h3>
    <p>
     本文简要介绍了SQL的基本语法，包括DDL、DML和DCL。掌握SQL语法对于数据库操作至关重要。在实际应用中，还需要根据具体需求进行学习和实践。希望本文能对您有所帮助。
    </p>
