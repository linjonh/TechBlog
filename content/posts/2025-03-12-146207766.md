    <h2>
     <a id="vscode_uniapp_Vue_3__Vite__Pinia__uView_Plus_0">
     </a>
     在vscode搭建 uni-app 项目（Vue 3 + Vite + Pinia + uView Plus）
    </h2>
    <h2>
     <a id="_1">
     </a>
     一、环境准备
    </h2>
    <h3>
     <a id="1__Nodejs_2">
     </a>
     1. 安装 Node.js
    </h3>
    <p>
     确保已安装 Node.js（需≥14版本），可通过以下命令检查版本：
    </p>
    <pre><code class="prism language-bash"><span class="token function">node</span> <span class="token parameter variable">-v</span>
</code></pre>
    <h3>
     <a id="2__VSCode_10">
     </a>
     2. 安装 VSCode
    </h3>
    <p>
     从 VSCode 官网 下载并安装适合你操作系统的版本。
    </p>
    <h3>
     <a id="3__uniapp__12">
     </a>
     3. 安装 uni-app 官方插件
    </h3>
    <p>
     在 VSCode 中打开扩展视图（快捷键 Ctrl+Shift+X），搜索 “uni-app”，找到官方插件并安装。
    </p>
    <h3>
     <a id="4__Volar__14">
     </a>
     4. 安装 Volar 插件
    </h3>
    <p>
     Volar 是 Vue 3 的官方语言服务，提供智能感知、语法检查等功能。在扩展视图中搜索 “Volar” 并安装。
    </p>
    <h2>
     <a id="_17">
     </a>
     二、创建项目
    </h2>
    <h3>
     <a id="1__18">
     </a>
     1. 初始化项目
    </h3>
    <p>
     首先，确保你已经安装了 Node.js 和 npm。然后，打开终端，使用以下命令创建一个新的 uni-app 项目：(个人node 版本 18.18.2)
    </p>
    <p>
     <strong>
      注意：
     </strong>
     <code>
      Vue3/Vite
     </code>
     版要求 node 版本
     <code>
      ^14.18.0 || &gt;=16.0.0
     </code>
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> init uni-app my-uniapp-project <span class="token parameter variable">--template</span><span class="token operator">=</span>vue3-vite-uni
</code></pre>
    <h3>
     <a id="2__26">
     </a>
     2. 进入项目目录
    </h3>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> my-uniapp-project
</code></pre>
    <h3>
     <a id="3__32">
     </a>
     3. 安装依赖
    </h3>
    <p>
     运行以下命令安装项目依赖：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span>
</code></pre>
    <h3>
     <a id="4__uView_Plus_38">
     </a>
     4. 安装 uView Plus
    </h3>
    <p>
     安装 uView Plus UI 库：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span> uview-plus
</code></pre>
    <h3>
     <a id="5__uView_Plus_45">
     </a>
     5. 配置 uView Plus
    </h3>
    <p>
     在 main.js 中引入并配置 uView Plus：
    </p>
    <pre><code class="prism language-bash">// main.js
<span class="token function">import</span> <span class="token punctuation">{<!-- --></span> createSSRApp <span class="token punctuation">}</span> from <span class="token string">'vue'</span><span class="token punctuation">;</span>

    <h3>
     <a id="6__Pinia_69">
     </a>
     6. 安装 Pinia
    </h3>
    <p>
     安装 Pinia 状态管理库：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span> pinia
</code></pre>
    <p>
     在 main.js 中引入并创建 Pinia 实例：
    </p>
    <pre><code class="prism language-bash">// main.js
<span class="token function">import</span> <span class="token punctuation">{<!-- --></span> createPinia <span class="token punctuation">}</span> from <span class="token string">'pinia'</span><span class="token punctuation">;</span>

    <pre><code class="prism language-bash">// vite.config.js

    <h3>
     <a id="10__184">
     </a>
     10.安装对应依赖并 运行项目
    </h3>
    <p>
     使用以下命令运行项目：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span>
<span class="token function">npm</span> run dev:h5  // 运行 H5 平台

    <p>
     这将启动开发服务器，并在浏览器中打开项目。
    </p>
    <h3>
     <a id="11__196">
     </a>
     11. 条件编译
    </h3>
    <p>
     uni-app 支持条件编译，允许你根据不同平台编写特定的代码。例如：
    </p>
    <pre><code class="prism language-bash">// <span class="token comment">#ifdef H5</span>
console.log<span class="token punctuation">(</span><span class="token string">'This code will only run on H5 platform'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
// <span class="token comment">#endif</span>

// <span class="token comment">#ifdef MP-WEIXIN</span>
console.log<span class="token punctuation">(</span><span class="token string">'This code will only run on WeChat Mini Program'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
// <span class="token comment">#endif</span>

// <span class="token comment">#ifdef APP-PLUS</span>
console.log<span class="token punctuation">(</span><span class="token string">'This code will only run on App platform'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
// <span class="token comment">#endif</span>
</code></pre>
    <p>
     这样你可以根据需要在
     <em>
      请求拦截
     </em>
     或者
     <em>
      路径处理
     </em>
     时区分不同的运行环境
    </p>
    <h3>
     <a id="12__213">
     </a>
     12. 构建项目
    </h3>
    <p>
     要构建项目，使用以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token function">npm</span> run build:h5  // 对应的命令
</code></pre>
    <p>
     这将生成一个构建后的文件夹，其中包含可以部署的静态文件。
    </p>
