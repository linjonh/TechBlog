## 好用的在线 java 编译网站，编辑器（亲测）

置顶

2019年01月03日 18:23:31
[春风一慕](https://me.csdn.net/w584212179)

阅读数 2133

1. https://www.jdoodle.com/online-java-compiler

这个支持 Java 10，并且能够保存代码，还支持导入外部库。

但有时候国内登不上，真不明白这个学习网站也封。

2. https://www.tutorialspoint.com/compile\_java8\_online.php

这个网站比较美观，也支持 java 10， 默认是运行 HelloWorld.class, 所以运行时需要对 javac 与 java 命令 专门设置一下，可以保存代码到 GitHub, dropbox 等，但不支持导入外部库。

3. http://www.beta.browxy.com/

它不仅运行速度快，而且还能保存代码，要是再美观些就完美了。

新版本已经支持 java 10 了，可以用 lambda 表达式， 而且美观多了，也能导入外部库，但有大小限制。

4. https://www.compilejava.net/
