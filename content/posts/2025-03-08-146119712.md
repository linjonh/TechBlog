    <h4>
     <a id="Eigenvalue_Decomposition_0">
     </a>
     <strong>
      特征值分解（Eigenvalue Decomposition）
     </strong>
    </h4>
    <h5>
     <a id="_2">
     </a>
     <strong>
      课程目标：
     </strong>
    </h5>
    <ul>
     <li>
      理解
      <strong>
       特征值（Eigenvalue）
      </strong>
      和
      <strong>
       特征向量（Eigenvector）
      </strong>
      的概念
     </li>
     <li>
      掌握
      <strong>
       特征值分解（Eigendecomposition）
      </strong>
      的基本原理
     </li>
     <li>
      学会计算
      <strong>
       矩阵的特征值和特征向量
      </strong>
     </li>
     <li>
      了解特征值分解在
      <strong>
       数据降维（PCA）、物理、计算机视觉
      </strong>
      等领域的应用
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_10">
     </a>
     <strong>
      第一部分：特征值和特征向量
     </strong>
    </h3>
    <h4>
     <a id="11__12">
     </a>
     <strong>
      1.1 特征值和特征向量的定义
     </strong>
    </h4>
    <p>
     对于一个 n x n 方阵 A ，如果存在一个非零向量 v 和一个标量 λ，使得：
     <br/>
     <span class="katex--display">
      <span class="katex-display">
       <span class="katex">
        <span class="katex-mathml">
         A 
         
        
          v 
         
        
          = 
         
        
          λ 
         
        
          v 
         
        
       
         A v = \lambda v
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 0.6833em;">
          </span>
          <span class="mord mathnormal">
           A
          </span>
          <span class="mord mathnormal" style="margin-right: 0.0359em;">
           v
          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>
          <span class="mrel">
           =
          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>
         </span>
         <span class="base">
          <span class="strut" style="height: 0.6944em;">
          </span>
          <span class="mord mathnormal">
           λ
          </span>
          <span class="mord mathnormal" style="margin-right: 0.0359em;">
           v
          </span>
         </span>
        </span>
       </span>
      </span>
     </span>
     <br/>
     那么：
     <br/>
     v称为矩阵A的特征向量Eigenvector
     <br/>
     λ称为矩阵A的特征值Eigenvalue
    </p>
    <h5>
     <a id="_21">
     </a>
     <strong>
      直观理解：
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       特征向量
      </strong>
      是在矩阵变换时
      <strong>
       方向不变
      </strong>
      的向量（只会被放大或缩小）。
     </li>
     <li>
      <strong>
       特征值
      </strong>
      描述了变换时向量的缩放比例。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="12__27">
     </a>
     <strong>
      1.2 计算特征值
     </strong>
    </h4>
    <p>
     从方程 A v = λ v 变形得到：(A - λI) v = 0 为了求解 v，需要 A - λI 是
     <strong>
      奇异矩阵
     </strong>
     （即行列式为 0）：
     <br/>
     <span class="katex--display">
      <span class="katex-display">
       <span class="katex">
        <span class="katex-mathml">
         det 
         
        
          ⁡ 
         
        
          ( 
         
        
          A 
         
        
          − 
         
        
          λ 
         
        
          I 
         
        
          ) 
         
        
          = 
         
        
          0 
         
        
       
         \det(A - \lambda I) = 0
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 1em; vertical-align: -0.25em;">
          </span>
          <span class="mop">
           det
          </span>
          <span class="mopen">
           (
          </span>
          <span class="mord mathnormal">
           A
          </span>
          <span class="mspace" style="margin-right: 0.2222em;">
          </span>
          <span class="mbin">
           −
          </span>
          <span class="mspace" style="margin-right: 0.2222em;">
          </span>
         </span>
         <span class="base">
          <span class="strut" style="height: 1em; vertical-align: -0.25em;">
          </span>
          <span class="mord mathnormal">
           λ
          </span>
          <span class="mord mathnormal" style="margin-right: 0.0785em;">
           I
          </span>
          <span class="mclose">
           )
          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>
          <span class="mrel">
           =
          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>
         </span>
         <span class="base">
          <span class="strut" style="height: 0.6444em;">
          </span>
          <span class="mord">
           0
          </span>
         </span>
        </span>
       </span>
      </span>
     </span>
     <br/>
     这个方程称为
     <strong>
      特征方程（Characteristic Equation）
     </strong>
     ，解出 λ 即得到
     <strong>
      特征值
     </strong>
     。
    </p>

     </strong>

     <br/>

          </span>
