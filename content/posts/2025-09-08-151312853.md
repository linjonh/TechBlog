# 7,000 星！AutoMQ 开源再创里程碑

![](https://i-blog.csdnimg.cn/direct/e36f25d8347a49a3afa3ba60beea43ea.png)

AutoMQ GitHub Star 正式突破 7,000 🌟！

从第一行代码到全球数千开发者的认可，AutoMQ 再次迎来开源旅程的重要里程碑。感谢每一次 Star、PR 与深度讨论，让开源 Kafka 在云时代焕发新活力。

**为什么越来越多的企业和开发者选择 AutoMQ？**

● **100% Kafka API 兼容：**平滑迁移，无供应商锁定，兼容 Kafka 全生态。

● **Diskless 架构：**Broker 无状态，秒级扩缩容，弹性伸缩成本更低。

● **低延迟写入：**P99 延迟可低至 10ms，满足金融、微服务等实时场景。

● **零跨 AZ 成本：**共享存储架构，避免高额跨可用区流量费用。

● **自动数据平衡：**内置 Self-Balancing，1 秒迁移分区，消除热点。

7000 Star 里程碑达成，既是认可，更是动力！AutoMQ 将持续聚焦成本、弹性与效率，为企业和开发者提供更优质的云原生 Kafka 体验。
