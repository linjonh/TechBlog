    <h2>
     <a id="_1">
     </a>
     微信小程序文件上传失败解决方法
    </h2>
    <h3>
     <a id="1__3">
     </a>
     1. 现象
    </h3>
    <p>
     今天更新了下小程序，发现文件上传无法拉起拍照或者选择照片，点击上传按钮，没有反应，也没有任何报错，顿时就一顿蒙蔽。最后灵机一动想起了，微信最近更新了隐私协议，文件上传需要用到
    </p>
    <pre><code>wx.chooseImage、wx.chooseMedia、wx.chooseVideo
wx.authorize({scope: ‘scope.camera’})、wx.createVKSession、
</code></pre>
    <p>
     这些接口，而这些接口都是需要先在隐私协议中声明，才可以使用的，最终我在隐私协议添加了这两个权限，文件就可以重新上传了。
    </p>
    <p>
     隐私协议弹窗如何创建参考上一篇博客
    </p>
    <p>
     <a href="https://blog.csdn.net/qq_35921773/article/details/132638793?spm=1001.2014.3001.5501">
      <strong>
       微信小程序新版隐私协议弹窗实现最新版
      </strong>
     </a>
    </p>
    <h3>
     <a id="2__18">
     </a>
     2. 解决方法
    </h3>
    <ol>
     <li>
      <p>
       登录微信开放平台，进入小程序后台
      </p>
     </li>
     <li>
      <p>
       选择版本管理，选择最新的版本，提交审核
      </p>
     </li>
     <li>
      <p>
       进入审核页面，拉到最下方“更新隐私协议”
      </p>
     </li>
     <li>
      <p>
       在隐私协议页面，**[增加信息类型]**增加摄像头、选中的照片或视频信息这两个权限。更新用户隐私协议
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3f3f4deb366da6fc6e8b06dc0c4c08a.png">
        <br/>
        <strong>
         注意
        </strong>
        <br/>
        目前隐私协议是需要单独审核的，小程序是另外的审核。一定要等到隐私协议通过审核后，再提交小程序的审核，这样才会有效。
        <br/>
        隐私协议的审核状态，在微信公众平台》小程序管理后台》设置》拉到最下边可以看到。
        <br/>
        隐私协议通过审核后，小程序后台会收到消息通知。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1a2449b467cc5cc47429e4c7bdd09c0a.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       重新发布小程序
      </p>
     </li>
    </ol>
