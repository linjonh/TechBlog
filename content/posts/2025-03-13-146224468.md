    <h2>
     1. ES6 中的函数概述
    </h2>
    <p>
     ES6 提供了多种定义函数的方法，包括传统的函数声明、函数表达式、匿名函数、箭头函数等。这些不同的方式可以帮助我们更加灵活地编写代码。
    </p>
    <h3>
     2. 函数的基本定义
    </h3>
    <h4>
     2.1 传统函数声明
    </h4>
    <p>
     ES6 仍然支持使用
     <code>
      function
     </code>
     关键字定义函数。
    </p>
    <pre><code>function web(){
    return "https://blog.csdn.net/Theodore_1022";
}
console.log(web()); // 输出："https://blog.csdn.net/Theodore_1022"</code></pre>
    <h4>
     2.2 传递参数的函数
    </h4>
    <p>
     函数可以接收参数，并在函数体内部使用它们。
    </p>
    <pre><code>function dev(a, b){
    return a - b;
}
console.log(dev(4, 5)); // 输出：-1</code></pre>
    <h3>
     3. 参数默认值
    </h3>
    <p>
     在 ES6 中，函数参数可以有默认值，如果调用时未传入参数，则使用默认值。
    </p>
    <pre><code>function plus1(a = 1, b = 2){
    return a * b;
}
console.log(plus1()); // 输出：2 (1*2)
console.log(plus1(2, 3)); // 输出：6 (2*3)</code></pre>
    <h3>
     4. 匿名函数（Function Expression）
    </h3>
    <p>
     匿名函数是一种没有名称的函数，可以赋值给变量。
    </p>
    <pre><code>let plus2 = function(a, b){
    return a * b;
};
console.log(plus2(3, 4)); // 输出：12</code></pre>
    <h3>
     5. 箭头函数（Arrow Function）
    </h3>
    <h4>
     5.1 基本箭头函数
    </h4>
    <p>
     ES6 引入了箭头函数，使用
     <code>
      =&gt;
     </code>
     语法，使代码更加简洁。
    </p>
    <pre><code>let plus3 = (a, b) =&gt; {
    return a * b;
};
console.log(plus3(4, 2)); // 输出：8</code></pre>
    <h4>
     5.2 省略
     <code>
      {}
     </code>
     并隐式返回
    </h4>
    <p>
     当函数体只有一条返回语句时，可以省略
     <code>
      {}
     </code>
     和
     <code>
      return
     </code>
     关键字。
    </p>
    <pre><code>let plus4 = (a, b) =&gt; a * b;
console.log(plus4(5, 6)); // 输出：30</code></pre>
    <h3>
     6. 结语
    </h3>
    <p>
     ES6 提供了多种定义函数的方法，使 JavaScript 代码更加简洁、高效。掌握函数的不同写法，有助于更好地编写现代 JavaScript 代码。在开发过程中，推荐使用箭头函数简化回调，提高可读性，同时在适当场景下使用默认参数，减少冗余代码。
    </p>
