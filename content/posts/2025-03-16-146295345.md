    <h2>
     <a id="GitHub_0">
     </a>
     一、在GitHub上创建仓库
    </h2>
    <h4>
     <a id="1Your_repositories_1">
     </a>
     1.点击右上角头像–&gt;点击Your repositories
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9dc34ed2fa934732ad6e0f722cef41ec.png"/>
    </p>
    <h4>
     <a id="2New_3">
     </a>
     2.点击New
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7376c1dbcd0e43b7bfd5061ada36bf5c.png"/>
    </p>
    <h4>
     <a id="3_5">
     </a>
     3.创建仓库
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/380cd0135300488b94f9b0fb9c5627a5.png">
      <br/>
      网址复制一下，在后面git上传时会用到
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/941affc1bf6845af9a9d303475fe9aff.png"/>
     </img>
    </p>
    <h2>
     <a id="Git_Bash_9">
     </a>
     二、打开Git Bash
    </h2>
    <h4>
     <a id="1cd__10">
     </a>
     1.cd 进入项目所在路径
    </h4>
    <h4>
     <a id="2git_init_11">
     </a>
     2.输入git init
    </h4>
    <p>
     在当前项目的目录中生成本地的git管理（当前目录下出现.git文件夹）
    </p>
    <h4>
     <a id="3git_add__13">
     </a>
     3.输入git add .
    </h4>
    <p>
     将项目上的文件添加到仓库中
     <br/>
     注意add后面有个空格，后面还有个点
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f2ed7d2269fb44009a267c76879ab338.png"/>
    </p>
    <h4>
     <a id="4git_commit_m_first_commit_17">
     </a>
     4.输入git commit -m “first commit”
    </h4>
    <p>
     本次提交的注释
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ac13d2a1dc674cebba294061857e52fe.png"/>
    </p>
    <h4>
     <a id="5git_remote_add_origin_httpsurl_20">
     </a>
     5.输入git remote add origin https://刚复制的url地址
    </h4>
    <p>
     将本地的仓库关联到github上
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/21a24d89fcb64dca86c4df7aad8fb64b.png"/>
    </p>
    <h4>
     <a id="6git_push_u_origin_master_23">
     </a>
     6.输入git push -u origin master
    </h4>
    <p>
     把代码上传到github仓库
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/67f43966335e455e9b591fd58fa8b991.png"/>
    </p>
