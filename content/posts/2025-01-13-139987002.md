    <p id="ZH-CN_TOPIC_0000001959471585__p1842113319365">
     该命令行工具集合了HarmonyOS应用开发所用到的系列工具，包括代码检查codelinter、三方库的包管理ohpm、命令行解析hstack、编译构建hvigorw。
    </p>
    <h3 id="section21298572437">
     命令行工具获取
    </h3>
    <p id="ZH-CN_TOPIC_0000001959471585__p5564122918275">
     请前往下载中心获取并下载命令行工具Command Line Tools。
    </p>
    <h3 id="section17776863449">
     配置环境变量
    </h3>
    <h4 id="section1645505610110">
     Windows
    </h4>
    <p id="ZH-CN_TOPIC_0000001959471585__p480594115416">
     将解压后command-line-tools文件夹的bin目录配置到环境变量中。
    </p>
    <p id="ZH-CN_TOPIC_0000001959471585__p15675716183">
    </p>
    <p class="img-center">
     <img alt="" height="557" src="https://i-blog.csdnimg.cn/blog_migrate/fcce6c44f97cade8f52ded5800757aad.png" width="518"/>
    </p>
    <h4 id="section12362025216">
     macOS/Linux
    </h4>
    <ol id="ZH-CN_TOPIC_0000001959471585__ol15513515345">
     <li id="ZH-CN_TOPIC_0000001959471585__li949012913396">
      将下载后的命令行工具解压到本地。
     </li>
     <li id="ZH-CN_TOPIC_0000001959471585__zh-cn_topic_0000001169160500_zh-cn_topic_0000001056725590_li56571525162613">
      打开终端工具，执行以下命令，根据输出结果分别执行不同命令。
      <pre><code class="hljs">echo $SHELL </code></pre>
     </li>
    </ol>
    <ul>
     <li>
      如果输出结果为/bin/bash，则执行以下命令，打开.bash_profile文件。
     </li>
    </ul>
    <pre><code class="hljs">vi ~/.bash_profile</code></pre>
    <ul>
     <li>
      如果输出结果为/bin/zsh，则执行以下命令，打开.zshrc文件。
     </li>
    </ul>
    <pre><code class="hljs">vi ~/.zshrc</code></pre>
    <ol>
     <li id="ZH-CN_TOPIC_0000001959471585__zh-cn_topic_0000001169160500_zh-cn_topic_0000001056725590_li25486521518">
      单击字母“i”，进入
      <strong>
       Insert
      </strong>
      模式。
     </li>
     <li id="ZH-CN_TOPIC_0000001959471585__zh-cn_topic_0000001169160500_zh-cn_topic_0000001056725590_li782154061516">
      输入以下内容，在PATH路径下添加环境变量。请以实际命令行工具存储路径为准。
      <pre><code class="hljs">export PATH=~/command-line-tools/bin:$PATH  </code></pre>
     </li>
     <li id="ZH-CN_TOPIC_0000001959471585__zh-cn_topic_0000001169160500_zh-cn_topic_0000001056725590_li15568193111618">
      编辑完成后，单击
      <strong>
       Esc
      </strong>
      键，退出编辑模式，然后输入“:wq”，单击
      <strong>
       Ente
      </strong>
      <strong>
       r
      </strong>
      键保存。
     </li>
     <li id="ZH-CN_TOPIC_0000001959471585__zh-cn_topic_0000001169160500_zh-cn_topic_0000001056725590_li85075510917">
      执行以下命令，使配置的环境变量生效。
     </li>
    </ol>
    <ul>
     <li>
      如果步骤a时打开的是.bash_profile文件，请执行如下命令：
     </li>
    </ul>
    <pre><code class="hljs">source ~/.bash_profile</code></pre>
    <p>
    </p>
    <ul>
     <li>
      如果步骤a时打开的是.zshrc文件，请执行如下命令：
     </li>
    </ul>
    <pre><code class="hljs">source ~/.zshrc</code></pre>
