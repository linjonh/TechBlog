   <div class="htmledit_views" id="content_views">
    <p>
     Ubuntu 20.04 LTS已经移除Python2，默认安装python3，但是很多时候还是会经常用到python2.7版本，这里首先单独安装python2.7，然后配置python2和python3，以达到手动切换系统默认python版本。
    </p>
    <p>
     第一步：安装python2.7使用命令：
    </p>
    <pre><code class="hljs">sudo apt install python2</code></pre>
    <p>
     第二步：下面两条命令将/usr/bin/python2.7优先级设置为1，将/usr/bin/python3.8的优先级设置为2
    </p>
    <pre><code class="hljs">sudo update-alternatives --install /usr/bin/python python /usr/bin/python2.7 1
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.8 2</code></pre>
    <p>
     第三步：显示python可替换版本
    </p>
    <pre><code class="hljs">sudo update-alternatives --config python</code></pre>
    <p>
     其中输入1则将python2.7设置为默认版本，输入2则将python3.8设置为默认版本
    </p>
    <p>
     <img alt="" height="294" src="https://i-blog.csdnimg.cn/blog_migrate/370357d67fb1bcdc26f0ff6bd398541d.png" width="966"/>
    </p>
    <p>
     第四步：python3安装pip（先切换到python3下）
    </p>
    <pre><code class="hljs">sudo apt install python3-pip</code></pre>
    <p>
     python2安装pip，sudo apt install python-pip显示系统已经无法找到安装包地址，所以：
    </p>
    <pre><code class="hljs">sudo apt-get install curl  #安装curl下载工具
sudo curl https://bootstrap.pypa.io/pip/2.7/get-pip.py -o get-pip.py  #下载安装脚本
sudo python get-pip.py  #运行安装脚本</code></pre>
    <p>
     第五步：检查（什么python环境，检查对应pip版本对不对）
    </p>
