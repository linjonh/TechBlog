# python运行一个.py文件

1. [编写文件cal.py](http://xn--cal-tu9do4jw84btn1b.py)

```
print(100+200+300)

```

执行：
  
D:\File\python>python
[cal.py](http://cal.py)
  
600
  
这边需要注意，需要进入到.py文件的目录下。

2. [在看一个helloworld.py](http://xn--helloworld-pf2pypj97ghk7g.py)

```
print('hello world')

```

执行：
  
D:\File\python>python
[helloworld.py](http://helloworld.py)
  
hello world
