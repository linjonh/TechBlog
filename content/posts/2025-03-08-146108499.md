# 基于SpringBoot的智慧停车场小程序（源码+论文+部署教程）

#### 运行环境

• 前端：小程序 + Vue

• 后端：Java

• IDE工具：IDEA（可自行选择） + HBuilderX + 微信开发者工具

• 技术栈：小程序 + SpringBoot + Vue + MySQL

#### 主要功能

智慧停车场微信小程序主要包含小程序端和管理端。

**管理端**

• 停车场管理

• 公告信息管理

• 用户信息管理

• 预定信息管理

• 用户反馈管理

**小程序端**

• 登录注册

• 预约停车位

• 停车导航

• 停车缴费

• 用户信息

• 车辆信息

• 钱包充值

• 意见反馈
