     <strong>
      <span style="background-color:#fbd4d0;">
       简介

     <img alt="" height="516" src="https://i-blog.csdnimg.cn/blog_migrate/3f40c166285791bce6addc113fdea3b9.png" width="1200"/>

    <p>
     <br/>
     Midjourney是一个由同名研究实验室开发的人工智能程式，可根据文本生成图像，于2022年7月12日进入公开测试阶段，使用者可透过Discord 的机器人指令进行操作，该研究实验室由Leap Motion的创办人大卫·霍尔兹负责领导。
     <br/>
     官方网站
     <br/>
     传送门: https:// www.midjourney.com/home(需要科学上网)
     <br/>
     用途和使用场景
     <br/>
     主要用途:可用于壁纸、插画、漫画、平面设计、logo、APP图标等的灵感创作。
     <br/>
     适合人群:设计师 插画师 美术指导 创意总监 自媒体人 创意行业视觉行业一切内容创作者AI爱好者
     <br/>
     行业投资人；
     <br/>
     可以延伸职业:AI图像生成师/AI视觉创意指导/AI艺术家。
    </p>
    <p>
     <em>
      <strong>
       关于如何使用Discord机器人生成图像：
      </strong>
     </em>
    </p>
    <p>
     - 要使用Discord机器人生成图像，你需要首先加入Midjourney的官方Discord服务器，或者邀请机器人到你自己的服务器。你可以在Midjourney的网站上找到加入或邀请的链接。
     <br/>
     - 在Discord上，你可以通过直接给机器人发送消息或者在任何频道中@机器人来与它交流。要生成图像，你需要使用/imagine命令并输入一个提示。例如，/imagine a blue cat wearing a hat。
     <br/>
     - 机器人会返回四张图像供你选择。你可以点击图像下方的表情符号来放大、下载、收藏或分享图像。你也可以输入/retry命令来重新生成一组新的图像。
     <br/>
     - 除了/imagine命令之外，还有其他一些命令可以帮助你使用Discord机器人。例如，/help命令可以显示所有可用的命令和说明；/account命令可以显示你的订阅信息和剩余作品数量；/gallery命令可以显示你收藏的图像等。
    </p>

    <p>
     <strong>
      <em>
       玩好Midjourney需要具备的能力
      </em>
     </strong>
     <br/>
     英语:Midjourney是纯英文操作，输入英文关键词进行Ai绘图，能看懂基本的英语指令，输入英语关键词，进行AI 绘画创作。
     <br/>
     知识管理:关键词决定了你输图的质量，需要你具备知识管理能力，建立积累关键词资源库的力，比如用印象笔记，EXCEL表格等工具建立自己的专属图库。
     <br/>
     艺术鉴赏:你至少应该懂得基本的绘画风格，摄影参数，光影效果，3D技术，超现实主义，波普艺术，立体主义，达达主义，未来主义，现实主义，极简主义，抽象派，新艺术，超现实主义，表现主义，构成主义各种等各种艺术风格流派
     <br/>
     修图能力:在AI绘画的基础上能修图改图，二次创作创新。青出于蓝而胜于蓝。
    </p>
    <p>
     搜索能力:能够熟练使用google wiki chatgtp搜索，谷歌翻译等创建关键词词库。
     <br/>
     <br/>
     逻辑思维:兵器是身体的延伸，AI是思维的衍生，需要你具备逻辑树形思维创建图像
     <br/>
     创意思维:需要你脑洞打开，无远弗届，创意--创造--创世。

    <p>
     <br/>
     <em>
      <strong>
       <span style="color:#0d0016;">
        Midjourney升级到V5发布的变化
       </span>
      </strong>
     </em>
     <br/>
     <br/>
     V5基本型有什么新特点?
     <br/>
     -风格范围更广，对提示的反应更灵敏
     <br/>
     -图像质量高得多(分辨率提高⒉倍)，动态范围得到改善-图像更细致。细节更有可能是正确的。更少的不需要的文本。
     <br/>
     -改进了图像提示的性能
     <br/>
     -支持--tile参数，用于无缝拼接（实验性的)。
     <br/>
     -支持--ar长宽比大于2:1(试验性)。
     <br/>
     -支持--iw，以权衡图像提示与文本提示的关系
     <br/>
     <strong>
      V5的风格和提示
     </strong>
     <br/>
     -今天的测试基本上是该模型的"专业"模式。
     <br/>
     -它比V3和V4更"无主见"，并被调整为提供多种多样的输出，对你的输入反应非常灵敏。
     <br/>
     -这里的权衡是，它可能更难使用。短的提示语可能不那么好用。你应该尝试写更长、更明确的文字，说明你想要什么(即:"具有戏剧性灯光的电影照片")。
     <br/>
     -请在提示聊天室里互相交流，以了解如何使用v5。
     <br/>
     -我们希望在将v5切换为默认样式之前，能有一个"友好"的默认样式。当这种情况发生时，我们仍然会让你把它关掉，回到像今天这种'原始'模式。
     <br/>
     请注意
     <br/>
     -这是一个alpha测试，事情会发生变化。不要依赖这个确切的模式在未来可用。当我们把V5推向全面发布时，它将被大幅修改。
     <br/>
     -现在还没有V5的升频器，V5的默认分辨率与升频后的V4相同。如果你点击升尺度，它就会立即给你那张图片本身。

    <p>
     简介
     <br/>
     <br/>
     Notion于2022年11月首次宣布其基于AI的文案写作助手Notion AI，并于23年2月22日推出正式版本，具备生成内容、修改内容的功能，可运用在文学、会议日程、新闻稿、销售文案等内容的辅助创作中，用户可通过网页端、移动端、PC端登陆，据Notion官网截至2月22日累计用户超200万。生成内容方面，可精细化便捷修改。用户可在指令框内输入希望写作的内容，或直接从指令框的选项中选择，Notion AI将快速生成相应内容，用户在此基础上翻译为特定语言，或对特定段落的行文风格、段落长度、标点符号等进行精细化的修改。此外,Notion AI还具备检查拼写和语法错误、创建想法列表的能力，以及写Excel公式、写Python 代码等复杂任务的处理能力。已开启订阅变现。Notion AI目前允许用户免费输入20次指令，超出20次或23年4月5日后，用户使用时需付费订阅，年度套餐为每年96美元，或每月8美元，月度套餐则为每月10美元，均提供无限指令次数。

    <p>
     2023年3月6日
     <br/>
     [中科院院士潘云鹤:从ChatGPT谈AI发展方向，建议布局AIGC]
     <br/>
     <br/>
     3月6日，中国青年报杭州电消息，中国工程院院士、浙江大学教授、计算机应用专家潘云鹤主讲2023年首期"之江院士讲坛”,从大热的人工智能聊天程序ChatGPT出发,剖析未来AI技术发展的趋势。
     <br/>
     潘院士分析，ChatGPT 火热的原因，首先是因为它回答问题的水平超乎人们的预期，組表现出实事求是的特点;欺, ChatGPT可能会对记者编辑、作家等文字工作者，以及画家、篆刻家、作曲家等创造文化工作者有影响;此外，OpenAI的限时开放使用造就了巨大的人气，也引发了许多大企业的竞争，比如微软入股OpenAI,谷歌宣布开发Bard, Meta 推出LLaMa。但AIGC -定不会只用于聊天、画画，而是会转向更有价值的应用领域，因此我们要提前布局实体经济的AIGC、文化艺术的AIGC、城乡发展的AIGC。他举例说，比如布局新药物的智能设计生成、儿童教育的智能内容生成、智能社区和生态推演的智能模拟生成。
     <br/>
     2023年3月1日
     <br/>
     [国内首个AI生成内容检测平台正式公测，准确率超90%]3月1日,人民网消息, AIGC-X正式开始全网公测。据悉,这是由人民日报社主管、依托人民网建设的传播内容认知全国重点实验室，中国科学技术大学，合肥综合性国家科学中心人工智能研究院联合推出的国内首个AI生成内容检测工具。据介绍，AIGC-X 可快速区分机器生成文本与人工生成文本，目前对中文文本检测的准确率已超过90%。
     <br/>
     从AI绘画、AI作曲到虚拟人，再到近期爆火的ChatGPT，伴随人工智能生成内容(AIGC) 技术发展加速，内容铲方式迎巨大变革。与此同时，在不断"生长”的互联网内容世界，找到一把明晰虚实、分辨真假的技术标尺，也成为关乎行业长远健康发展的关键。
     <br/>
     2023年2月13日
     <br/>
     <br/>
     [政策支持落地，AIGC有望迎来加速发展]
     <br/>
     2023年2月13日，北京市经信局表示:“支持头部企业打造对标ChatGPT的大模型，着力构建开源框架和通用大模型的应用生态。加强人工智能算力基础设施布局。加速人工智能基础数据供给。支持人工智能优势企业在自动驾驶、智能制造、智慧城市等优势领域开展创新应用，全面构筑人工智能场景创新高地”。北京信局明确表示支持头部企业打造对标ChatGPT的大模型，支持算力、数据、应用等相关产业协同发展，AIGC行业从市场关注上升至政策支持层面。
