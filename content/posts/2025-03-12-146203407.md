    <p>
     今天我们继续来看看网络变压器的2个重要电性参数与它的测试方法：
    </p>
    <p style="text-align:justify">
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       1.反射损耗（Return loss）:
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       又称回波损耗，它是
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       衡量插入网络变压器后系统阻抗失配程度与信号频率之间的关系曲线Return Loss=20Xlog(Vr/Vi)的，其中Vr为反射信号幅度，Vi为入射信号幅度. 其单位为dB,并以“-”来表示衰减, 理想的网络变压器其Return Loss应趋近于无限大。
      </span>
     </span>
    </p>
    <p class="img-center">
     <img alt="" height="192" src="https://i-blog.csdnimg.cn/direct/d7376e73dc0c44ee99f8ba943a1129ab.png" width="387"/>
    </p>
    <p>
     2.
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       交越干扰（Cross Talk）:
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       也称串扰，
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       是指磁性元件对附近元件所干扰的程度，即两个单元电路中的一个单元电路中的信号V1与感应到另一个单元电路中的信号V2之比值Cross Talk =20Xlog(V2/V1) ，其单位为dB,并以“-”来表示衰减。不良的CT易造成客户端传输有
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       数据包丢失（
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       packet loss
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       ）
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
       的现象。
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#666666">
      </span>
     </span>
    </p>
