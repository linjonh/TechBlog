    <h2>
     <a id="1_VSCode_0">
     </a>
     1. 下载VSCode
    </h2>
    <p>
     进入VSCode的官网：
    </p>
    <blockquote>
     <p>
      https://code.visualstudio.com/
     </p>
    </blockquote>
    <p>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/735122b4d3560097799a532cf4b5559a.png#pic_center">
      <br/>
     </img>
    </p>
    <p>
     点击为Linux系统准备的.deb下载，下载完成后按默认选项install即可。
     <br/>
     <br/>
    </p>
    <h2>
     <a id="2__13">
     </a>
     2. 安装插件
    </h2>
    <p>
     点击插件，搜索Java，安装相关的插件。这里用户可以选择，不用每个都装。Language support for Java一定要安装，一些常用的如Java Extension Pack, Code Runner最好装一下。
     <br/>
     <br/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/08fc4c2ccbaa8b6dbb0165949557f026.png#pic_center"/>
    </p>
    <br/>
    <h2>
     <a id="3_JDK_23">
     </a>
     3. JDK
    </h2>
    <p>
     在终端运行以下命令：
    </p>
    <blockquote>
     <p>
      sudo apt-get install openjdk-8-jdk
     </p>
    </blockquote>
    <br/>
    <p>
     之后，到vscode里preference-&gt;settings，搜索java.home，并把路径加入到.json文件中。
     <br/>
     <br/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d08ba9e30d1acf31a63b161a56f0b005.png#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c3c0bfef1cdd937d5406d67928a8b2ed.png#pic_center"/>
     </img>
    </p>
    <br/>
    <h2>
     <a id="4__41">
     </a>
     4. 运行程序
    </h2>
    <p>
     新建一个.java文件，写完代码后点击run code，会生成.class文件并输出结果。
     <br/>
     <br/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f4b1e3c237520ad99494351417b36100.png#pic_center">
      <br/>
      <br/>
     </img>
    </p>
    <p>
     到此为止，我们就成功地在Ubuntu里用VSCode配置编写了Java！
     <br/>
     如果对你有帮助的话，求一波点赞~
    </p>
