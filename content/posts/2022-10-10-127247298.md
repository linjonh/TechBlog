    <h2>
     <a id="_10">
     </a>
     一、简介
    </h2>
    <p>
     在
     <a href="https://blog.csdn.net/weixin_33005117/article/details/127212910?spm=1001.2014.3001.5501">
      钉钉小程序入门1—区分企业内部应用、第三方企业应用、第三方个人应用
     </a>
     一文中我解释了钉钉的三种应用以及它们的区别，以企业内部应用为例，点击创建应用按钮后会弹出了如下的弹窗：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/caa68d036277774fa2f3c3d0e3df12f6.png">
      <br/>
      这里有一个应用类型：
      <strong>
       H5微应用
      </strong>
      和
      <strong>
       小程序
      </strong>
      ，很多小伙伴不知道这二者有什么区别，下面我来给大家介绍一下。
     </img>
    </p>
    <h2>
     <a id="_14">
     </a>
     二、区别介绍
    </h2>
    <h3>
     <a id="_15">
     </a>
     区别一：运行原理
    </h3>
    <p>
     <strong>
      小程序主页面
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b406efd34b8c4913df0afe7b1b064441.png"/>
    </p>
    <p>
     <strong>
      H5微应用主页面
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4c7bffe20adc0f2ff939d49e0fcf1fe9.png">
      <br/>
      从左侧菜单可以看出：在基础信息菜单下小程序比H5微应用多一个人员管理；在安全与监控菜单下小程序比H5微应用多一个安全中心。
     </img>
    </p>
    <p>
     在解释这两个菜单有什么作用之前，大家需要了解一下小程序与H5的一些基本概念：
    </p>
    <blockquote>
     <p>
      <strong>
       1. 不一样的运行环境
      </strong>
      <br/>
      H5仅能借助浏览器实现启动，但小程序虽然包含H5技术，但在运行环境方面与H5不同，它基于解析器得以运行，结合自定义的开发语言标准，其性能更是有了长足的进步。简单讲小伙伴们可以利用浏览器打开H5网页，运用微信、钉钉、支付宝平台打开小程序。
     </p>
     <p>
      <strong>
       2. 不一样的成本
      </strong>
      <br/>
      在开发方面，很多企业借助各类开发软件针对H5开发，定制化出众。但这种开发需要结合多方面技术实现，因此相对比较复杂，开发的成本也是比较高的。然而小程序则不一样，开发完全借助官方下载的开发工具即可实现，小程序开发价格相对低廉很多。
     </p>
     <p>
      <strong>
       3. 不一样的系统权限
      </strong>
      <br/>
      从程序本身来看，这个小程序能得到更多的系统权限，而对于H5则不同，权限很低。很简单举几个例子，比如网络当前的状态，甚至是一些数据的缓存应用等，都能够得以与小程序进行连接。
     </p>
     <p>
      <strong>
       4. 不一样的运行状态
      </strong>
      <br/>
      现在的人们总是强调人性化，用户高体验度，那么这一点就被小程序抢先拥有了。要知道，相较于H5，这个小程序拥有更为流畅的使用速度。
     </p>
     <p>
      <strong>
       5. 不一样的开发方式
      </strong>
      <br/>
      小程序的开发都需要借助于专门的开发工具，如微信小程序需要微信开发者工具，钉钉也需要小程序开发者工具。而H5不同，可以使用VsCode、HBuilder开发。
     </p>
    </blockquote>
    <p>
     很多使用过小程序和H5的用户，都很印象深刻，前者的打开速度要快上很多，利用一些软件就也可以测试出两者的差距。
    </p>
    <p>
     所以小程序多出来的
     <strong>
      人员管理
     </strong>
     管的就是开发工具的权限，只有在人员管理中添加了的用户才能在小程序开发工具中找到这个小程序。多出来的
     <strong>
      安全中心
     </strong>
     则是小程序容器外跳链接时做的一个网址备案管理，防止被钓鱼网站利用。
    </p>
    <h3>
     <a id="_44">
     </a>
     区别二：打开方式
    </h3>
    <p>
     <strong>
      小程序开发管理
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b119883e059b31f2751c677de7e35f68.png">
      <br/>
      <strong>
       H5微应用开发管理
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/118f292bc6eed05ece57217b8a6ea7f4.png">
       <br/>
       在开发管理中，H5微应用比小程序多一个
       <strong>
        应用首页地址
       </strong>
       ，这个地址相当于网站的首页地址，H5微应用是必须配置这个地址的，不配置的话小程序是无法打开的。
      </img>
     </img>
    </p>
    <p>
     由于H5微应用没有运行在钉钉容器中，可能没法通过api拿不到当前组织的CorpId，而CorpId又是动态的，所以H5微应用的链接我一般配置成如下格式：
     <br/>
     <img alt="https://xxx.com/index?corpId=&amp;appId=xxx&amp;clientId=suitepwismulvpcgilrk4" src="https://i-blog.csdnimg.cn/blog_migrate/baa8d30f7b13be1f870d80fdb489a5e9.png"/>
    </p>
    <p>
     这样我们在打开H5微应用时，钉钉就会自动把当前这个组织的CorpId替换到链接上去，前端同学就可以通过参数拿到CorpId了。
    </p>
    <p>
     而小程序就没有这样的问题了，直接调用原生api即可获取当前组织的CorpId。
    </p>
    <h3>
     <a id="_58">
     </a>
     区别三：安全中心
    </h3>
    <p>
     H5微应用可以理解为就是一个网页，只不过是做成小程序的样子，它的跳转完全不受容器控制，除非自己写代码控制跳转，否则都不会进行拦截。
     <br/>
     而小程序不是这样，钉钉小程序的安全中心如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7cea862df6a93dbd75721b3a28011552.png">
      <br/>
      你想跳转的任何一个非当前域名的页面都需要在这里维护，有些光配置还不行还得进行安全校验，例如微信的业务域名就非常的恶心，必须去域名下面放一个文件，否则就无法配置。
     </img>
    </p>
    <h3>
     <a id="_65">
     </a>
     区别四：版本管理
    </h3>
    <p>
     小程序分为开发版—体验版—线上版，其中体验版发布到线上版需要官方审核，有时还有可能审核不通过。
     <br/>
     而H5微应用只需要后台维护一下js、css的版本号即可，遇到bug只需要改完代码发布一下就行了。
     <br/>
     在版本管理这块H5完胜小程序。
    </p>
    <h3>
     <a id="_70">
     </a>
     区别五：适配性
    </h3>
    <p>
     钉钉是分为PC端和移动端的，在PC端打不开小程序，点击会显示“该应用仅支持移动端打开”，则H5微应用都可以打开。
     <br/>
     小程序打开是不可以放大全屏的，宽高都被限制了，而H5微应用打开后会有一个放大按钮，点击可以全屏，利用这个特点可以适配平板和PC端都行。
    </p>
