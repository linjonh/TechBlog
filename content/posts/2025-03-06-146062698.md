    <h2>
     <a id="Android__1">
     </a>
     Android 多用户相关
    </h2>
    <p>
     本文主要记录下android 多用户相关的adb 命令操作.
    </p>
    <h3>
     <a id="1__5">
     </a>
     1: 获取用户列表
    </h3>
    <p>
     命令: adb shell pm list users
    </p>
    <p>
     输出如下:
    </p>
    <pre><code>Users:
	UserInfo{0:机主:c13} running
</code></pre>
    <p>
     默认只有一个用户, id为0 ，用户状态为运行
    </p>
    <h3>
     <a id="2__18">
     </a>
     2: 创建新用户
    </h3>
    <p>
     命令： adb shell pm create-user ‘zh’
    </p>
    <p>
     创建用户zh，输出如下：
    </p>
    <pre><code>Success: created user id 11
</code></pre>
    <p>
     可以看到创建成功，用户的id是11.
    </p>
    <p>
     此时我们继续查询下用户列表:
    </p>
    <pre><code>Users:
	UserInfo{0:机主:c13} running
	UserInfo{11:zh:400}
</code></pre>
    <p>
     可以看到11这个用户是非运行状态的.
    </p>
    <h3>
     <a id="3__40">
     </a>
     3: 启用新用户
    </h3>
    <p>
     命令: adb shell am start-user 11
    </p>
    <p>
     注意: start-user 后使用的是user id
    </p>
    <p>
     输出结果如下:
    </p>
    <pre><code>Success: user started
</code></pre>
    <p>
     pm list user:
    </p>
    <pre><code>Users:
	UserInfo{0:机主:c13} running
	UserInfo{11:zh:410} running
</code></pre>
    <p>
     可以看到zh这个用户的状态已经更改成功.
    </p>
    <h3>
     <a id="4__62">
     </a>
     4: 切换新用户
    </h3>
    <p>
     命令: adb shell am switch-user 11(user id)
    </p>
    <p>
     执行后,可以看到手机已经自动切换用户,提示开始使用了.
    </p>
    <h3>
     <a id="5__68">
     </a>
     5: 删除新用户
    </h3>
    <p>
     命令: adb shell pm remove-user 11
    </p>
    <p>
     结果:
    </p>
    <pre><code>Success: removed user
</code></pre>
    <p>
     注意: 当已经切换到user11 时, 执行删除会失败.
    </p>
    <pre><code>Error: couldn't remove user id 11
</code></pre>
    <h3>
     <a id="6__84">
     </a>
     6: 获取当前用户
    </h3>
    <p>
     命令: adb shell am get-current-user
    </p>
    <h3>
     <a id="7__88">
     </a>
     7: 安装应用到指定用户
    </h3>
    <p>
     命令: adb install-multiple --user 12 test.apk
    </p>
