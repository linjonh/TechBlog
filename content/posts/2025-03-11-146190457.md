    <h3>
     标签定义及使用说明
    </h3>
    <p>
     &lt;i&gt; 定义与文本中其余部分不同的部分，并把这部分文本呈现为斜体文本。
    </p>
    <p>
     &lt;i&gt; 标签被用来表示科技术语、其他语种的成语俗语、想法、宇宙飞船的名字等等。
    </p>
    <p>
     在没有其他适当语义的元素可以使用时，请使用 &lt;i&gt; 元素。其他语义的元素如下：
    </p>
    <ul>
     <li>
      <a href="https://www.w3cschool.cn/htmltags/tag-em.html" rel="nofollow" title="&lt;em&gt;">
       &lt;em&gt;
      </a>
      （被强调的文本）
     </li>
     <li>
      <a href="https://www.w3cschool.cn/htmltags/tag-strong.html" rel="nofollow" title="&lt;strong&gt;">
       &lt;strong&gt;
      </a>
      （重要的文本）
     </li>
     <li>
      <a href="https://www.w3cschool.cn/htmltags/tag-mark.html" rel="nofollow" title="&lt;mark&gt;">
       &lt;mark&gt;
      </a>
      （被标记的/高亮显示的文本）
     </li>
     <li>
      <a href="https://www.w3cschool.cn/htmltags/tag-cite.html" rel="nofollow" title="&lt;cite&gt;">
       &lt;cite&gt;
      </a>
      （作品的标题）
     </li>
     <li>
      <a href="https://www.w3cschool.cn/htmltags/tag-dfn.html" rel="nofollow" title="&lt;dfn&gt;">
       &lt;dfn&gt;
      </a>
      （一个定义项目）
     </li>
    </ul>

    <h3>
     HTML 4.01 与 HTML5之间的差异
    </h3>
    <p>
     在 HTML 4.01 中，&lt;i&gt; 标签呈现斜体的文本。然而，在 HTML5 中没有必要这么做，可以使用样式表来格式化 &lt;i&gt; 元素中的文本。
    </p>

    </p>

     一个内联框架被用来在当前 HTML 文档中嵌入另一个文档。

    <h3>
     提示和注释
    </h3>
    <p>
     <strong>
      提示：
     </strong>
     您可以把需要的文本放置在 &lt;iframe&gt; 和 &lt;/iframe&gt; 之间，这样就可以应对不支持 &lt;iframe&gt; 的浏览器。
    </p>
    <p>
     <strong>
      提示：
     </strong>
     使用 CSS 为 &lt;iframe&gt; （包括滚动条）定义样式。
    </p>

    <h2>
     HTML5 &lt;input&gt; 标签
    </h2>
    <h3>
     实例
    </h3>

    <h3>
     标签定义及使用说明
    </h3>
    <p>
     &lt;input&gt; 标签规定了用户可以在其中输入数据的输入字段。
    </p>
    <p>
     &lt;input&gt; 元素在
     <a href="https://www.w3cschool.cn/html5/html5-form.html" rel="nofollow" title="&lt;form&gt; 元素">
      &lt;form&gt; 元素
     </a>
     中使用，用来声明允许用户输入数据的 input 控件。
    </p>
    <p>
     输入字段可通过多种方式改变，取决于 type 属性。
    </p>
    <hr/>
    <h3>
     提示和注释
    </h3>
    <p>
     <strong>
      注意:
     </strong>
     &lt;input&gt; 元素是空的,它只包含标签属性。
    </p>
    <p>
     <strong>
      提示:
     </strong>
     你可以使用
     <a href="https://www.w3cschool.cn/htmltags/tag-label.html" rel="nofollow" title="&lt;label&gt;">
      &lt;label&gt;
     </a>
     元素来定义 &lt;input&gt; 元素的标注。
    </p>
    <hr/>
    <h3>
     HTML 4.01 与 HTML5之间的差异
    </h3>
    <p>
     在 HTML 4.01 中, "align" 数据已经不再使用。HTML5 中不支持该属性。 可以使用CSS来定义 &lt;input&gt; 元素的对齐方式。
    </p>
    <p>
     在 HTML5中, &lt;input&gt; 添加了几个属性，并且添加了对应的值。
    </p>
    <hr/>
    <h3>
     HTML 与 XHTML 之间的差异
    </h3>
    <p>
     在 HTML 中，&lt;input&gt; 标签没有结束标签。
    </p>
    <p>
     在 XHTML 中，&lt;input /&gt; 标签必须被正确地关闭。
    </p>

    <h3>
     标签定义及使用说明
    </h3>
    <p>
     &lt;img&gt; 标签定义 HTML 页面中的图像。
    </p>
    <p>
     &lt;img&gt; 标签有两个必需的属性：src 和 alt。
    </p>
    <p>
     <strong>
      注释：
     </strong>
     从技术上讲，图像并不会插入 HTML 页面中，而是链接到 HTML 页面上。&lt;img&gt; 标签的作用是为被引用的图像创建占位符。
    </p>
    <p>
     <strong>
      提示：
     </strong>
     通过在 &lt;a&gt; 标签中嵌套 &lt;img&gt; 标签，给图像添加到另一个文档的链接。
    </p>
    <hr/>
    <h3>
     HTML 4.01 与 HTML5之间的差异
    </h3>
    <p>
     HTML5 中不支持以下属性：align、border、hspace、longdesc、vspace。
    </p>
    <p>
     在 HTML 4.01 中，以下属性：align、border、hspace、vspace 已废弃。
    </p>
    <hr/>
    <h3>
     HTML 与 XHTML 之间的差异
    </h3>
    <p>
     在 HTML 中，&lt;img&gt; 标签没有结束标签。
    </p>
    <p>
     在 XHTML 中，&lt;img /&gt; 标签必须被正确地关闭。
    </p>
