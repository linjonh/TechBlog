   <div class="htmledit_views" id="content_views">
    <p>
     MacOS Big Sur 11,算是很老的系统了，所以装起来brew有点费劲。
    </p>
    <h3>
     首先安装brew
    </h3>
    <h4>
     官网：
    </h4>
    <pre><code>/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"</code></pre>
    <h4>
     官网加速：
    </h4>
    <p>
     按照官网的方法，直接执行下面语句即可安装：
    </p>
    <pre><code class="hljs">/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"</code></pre>
    <p>
     可以设定brew加速，在bash shell下的指令：
    </p>
    <pre><code>export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.ustc.edu.cn/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.ustc.edu.cn/homebrew-core.git"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.ustc.edu.cn/homebrew-bottles"
 
# brew4.x API加速
export HOMEBREW_API_DOMAIN="https://mirrors.ustc.edu.cn/homebrew-bottles/api"

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"</code></pre>
    <p>
     但是由于系统刚装出来，所以有些openssl等库可能比较老，所以有可能无法安装成功，这时候就要git clone下载homebrew的源代码，然后执行源码里的执行文件即可，文件在brew/bin/brew （但是这个未安装版好像是有些问题的）。
    </p>
    <h4>
     也可以用中科大这个源来安装：
    </h4>
    <pre><code>/bin/bash -c "$(curl -fsSL https://cdn.jsdelivr.net/gh/ineo6/homebrew-install/install.sh)"</code></pre>
    <h4>
     也可以直接在github网站下载pkg的安装包：
    </h4>
    <p>
     <a href="https://github.com/Homebrew/brew/releases" title="Releases · Homebrew/brew · GitHub">
      Releases · Homebrew/brew · GitHub
     </a>
    </p>
    <p>
     下载后直接安装（后来才知道，当前的版本需要Mac13以上的系统，并不适合）！
    </p>
    <p>
    </p>
    <h4>
     brew国内源（推荐使用）
    </h4>
    <p>
     比较推荐用这个，速度最快，而且安装的时候顺便设置好了国内镜像。
    </p>
    <p>
     <a href="https://gitee.com/cunkai/HomebrewCN" rel="nofollow" title="HomebrewCN: Homebrew 国内安装脚本，快速部署 brew ，国内镜像">
      HomebrewCN: Homebrew 国内安装脚本，快速部署 brew ，国内镜像
     </a>
    </p>
    <p>
     苹果安装脚本
    </p>
    <pre><code>/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
</code></pre>
    <p>
     苹果卸载脚本
    </p>
    <pre><code>/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/HomebrewUninstall.sh)"
</code></pre>
    <p>
    </p>
    <h3>
     再用brew安装wget
    </h3>
    <pre><code>brew install wget
</code></pre>
    <p>
     装了半天，安了一大堆东西，非常长的时间才装完。 如果安装不顺利，可以尝试先安装cmake或者curl试试。很多时候是卡在了github下载方面。
    </p>
    <pre><code>brew install cmake</code></pre>
    <p>
     也可以到cmake.og官网下载安装包：
     <a href="https://cmake.org/download/" rel="nofollow" title="Download CMake">
      Download CMake
     </a>
    </p>
    <p>
     总之有了cmake，后面软件实在不行编译安装也可以装上。
    </p>
    <p>
    </p>
    <h3>
     安装Miniconda 的python环境
    </h3>
    <p>
     再用wget获取miniconda
    </p>
    <pre><code>wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-py312_24.7.1-0-MacOSX-x86_64.sh</code></pre>
    <p>
     没有wget的时候，也可以直接到清华源网页下载：
     <a href="https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda" rel="nofollow" title=" Index of /anaconda/miniconda/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror">
      Index of /anaconda/miniconda/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
     </a>
    </p>
    <p>
    </p>
    <p>
     安装miniconda
    </p>
    <pre><code>sh Miniconda3-py312_24.7.1-0-MacOSX-x86_64.sh</code></pre>
    <p>
     安装conda后，选择激活环境（也可以使用source xx/bin/activate手动激活），（可选）安装uv
    </p>
    <pre><code>pip install pip -U
pip install uv
</code></pre>
    <p>
     注意如果pip网速慢的话，可以加上镜像:
    </p>
    <p>
     -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
    </p>
    <p>
     也可以设定镜像的环境变量：
    </p>
    <pre><code>pip config set global.index-url https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
</code></pre>
    <h3>
     安装exo AI并行框架
    </h3>
    <p>
     克隆下载exo源代码
    </p>
    <pre><code>git clone https://github.com/exo-explore/exo
</code></pre>
    <h4>
     安装exo
    </h4>
    <pre><code># 进入exo目录执行
pip install -e .
</code></pre>
    <p>
     如果安装的时候tinygrad不好下载，那么可以先到setup.py文件中屏蔽tinygrad这个句，稍后再单独安装它: pip install tinygrad
    </p>
    <p>
     opencv-python需要改成"opencv-python&gt;4.10.0.84"
    </p>
    <h4>
     启动exo
    </h4>
    <pre><code>exo
</code></pre>
    <p>
     好啦！用52415端口就可以管理exo啦！
    </p>
