# mysql查询当前库的实例名\_连接数据库，查询当前的数据库名、服务名、实例名、表空间、用户名...

1.使用plsql：

sqlplus /nolog

(Linux：su - oracle)

2.连接数据库：

连接本地数据库：conn / as sysdba

远程连接数据库：conn username/[email protected]地址:端口/orcl;

3.查询当前的数据库名、服务名、实例名、表空间、用户名：

数据库：select name from V$database;

服务名：show parameter service\_names;

实例名：select instance\_name from V$instance;

表空间：select default\_tablespace from dba\_users where username=‘登录用户‘(用户名要大写)

用户名：select \* from user\_users;

4.用户管理

创建用户：create user ukey identified by aisino;

赋予权限：

grant connect,resource,dba to ukey;

grant create session to ukey;

查询该用户下的所有表：

select \* from all\_tables where owner=‘登录用户‘(用户名要大写)

select table\_name from user\_tables;

修改用户默认表空间：alter user dbcenter default tablespace DBCENTER\_DATA

5.新建表空间：

create tablespace user\_temp

tempfile 'D:\oracle\oradata\Oracle9i\user\_temp.dbf'

size 50m

autoextend on

next 50m maxsize 20480m;

6.退出oracle：

quit
