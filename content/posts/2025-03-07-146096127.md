    <p>
     在开发uniapp应用时，可能会遇到图片无法加载的情况。这可能是由于图片路径错误、资源未正确打包、网络图片地址失效等原因导致的。
    </p>
    <h3>
     <a id="_2">
     </a>
     方案一：
    </h3>
    <p>
     使用require方式引入图片
    </p>
    <pre><code>&lt;template&gt;
    &lt;image :src="list[0].url" mode="scaleToFill" class="grid_img"&gt;&lt;/image&gt;
&lt;/template&gt;

&lt;script&gt;
	export default {
		data() {
			return {
				list: [
					{
						url: require('./img/cat_01.png')
					}
				]
			}
		}
	}
&lt;/script&gt;
</code></pre>
    <h3>
     <a id="_26">
     </a>
     方案二：
    </h3>
    <p>
     使用属性background-image加载图片
    </p>
    <pre><code>background-image:url(../../static/picture.png);
</code></pre>
