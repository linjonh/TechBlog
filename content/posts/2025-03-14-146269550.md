    <blockquote>
     <h4>
      【题目描述】
     </h4>
     <p>
      给定一个5×5的矩阵(数学上，一个r×c的矩阵是一个由r行c列元素排列成的矩形阵列)，将第n行和第m行交换，输出交换后的结果。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      输入共6行，前5行为矩阵的每一行元素,元素与元素之间以一个空格分开。
     </p>
     <p>
      第6行包含两个整数m、n，以一个空格分开（1≤m,n≤5）。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      输出交换之后的矩阵，矩阵的每一行元素占一行，元素之间以一个空格分开。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      1 2 2 1 2
      <br/>
      5 6 7 8 3
      <br/>
      9 3 0 5 3
      <br/>
      7 2 1 4 6
      <br/>
      3 0 8 2 4
      <br/>
      1 5
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      3 0 8 2 4
      <br/>
      5 6 7 8 3
      <br/>
      9 3 0 5 3
      <br/>
      7 2 1 4 6
      <br/>
      1 2 2 1 2
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e3 + 10;
int nums[N][N];

int main()
{
	for (int i = 1; i &lt;= 5; i++)
	{
		for (int j = 1; j &lt;= 5; j++)
		{
			cin &gt;&gt; nums[i][j];
		}
	}
	int n, m; cin &gt;&gt; n &gt;&gt; m;
	for (int i = 1; i &lt;= 5; i++)
	{
		swap(nums[n][i], nums[m][i]);
	}
	for (int i = 1; i &lt;= 5; i++)
	{
		for (int j = 1; j &lt;= 5; j++)
		{
			cout &lt;&lt; nums[i][j] &lt;&lt; ' ';
		}
		cout &lt;&lt; endl;
	}

	return 0;
}</code></pre>
