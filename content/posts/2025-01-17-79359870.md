    <p>
     tqdm 是一个快速，可扩展的Python进度条，可以在 Python 长循环中添加一个进度提示信息，用户只需要封装任意的迭代器 tqdm(iterator)。

      https://pypi.python.org/pypi/tqdm

      pip install tqdm

    <p>
     <strong>
      安装报错

    <pre><code class="language-python">(C:\ProgramData\Anaconda3) C:\Users\Administrator&gt;pip install tqdm

      下载
     </strong>
    </p>
