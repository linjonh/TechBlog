    <h2>
     <a id="_59">
     </a>
     二、属性值计算过程
    </h2>
    <ul>
     <li>
      <mark>
       明确知识
      </mark>
      ：页面的元素
      <code>
       都具备一套css样式
      </code>
      ，并非仅有开发人员设定的css属性
     </li>
     <li>
      过程：
      <mark>
       无属性值
      </mark>
      -
      <code>
       1.确定声明值
      </code>
      -
      <code>
       2.层叠冲突
      </code>
      -
      <code>
       3.使用继承
      </code>
      -
      <code>
       4.使用默认值
      </code>
      -
      <mark>
       每个属性都有值
      </mark>
     </li>
    </ul>
    <h3>
     <a id="1_65">
     </a>
     1.确定声明值
    </h3>
    <blockquote>
     <p>
      参考样式表中没有冲突的声明，作为css属性值
     </p>
    </blockquote>
    <pre><code class="prism language-css"><span class="token selector">.container</span><span class="token punctuation">{<!-- --></span>
	<span class="token property">color</span><span class="token punctuation">:</span>red<span class="token punctuation">;</span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>16px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token selector">h1</span><span class="token punctuation">{<!-- --></span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>20px
<span class="token punctuation">}</span>
<span class="token selector">.container h1</span><span class="token punctuation">{<!-- --></span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>12px<span class="token punctuation">;</span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>16px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token comment">/* 浏览器默认样式*/</span>
<span class="token selector">h1</span><span class="token punctuation">{<!-- --></span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>2em<span class="token punctuation">;</span><span class="token comment">/*举例*/</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     上述通过步骤一，可以提取出
     <code>
      未存在冲突
     </code>
     的 color：red；
    </p>
    <blockquote>
     <p>
      理解好上述流程后，打开控制台我们可以看到a标签存在浏览器的默认样式，属于
      <code>
       未存在冲突
      </code>
      样式,color使用浏览器
      <code>
       默认值在此环节生效
      </code>
      ,如有疑问，继续向下看，核心看
      <code>
       继承样式生效时机
      </code>
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4ef90c128f774eb9946d628678f9a6c3.png"/>
    </p>
    <h3>
     <a id="2_92">
     </a>
     2.层叠冲突
    </h3>
    <blockquote>
     <p>
      对样式表有冲突的声明使用层叠规则，确定CSS属性值
     </p>
    </blockquote>
    <ul>
     <li>
      层叠冲突处理过程
      <ul>
       <li>
        比较重要性 - 作者样式表&gt;浏览器默认样式表
        <code>
         解决上述问题1
        </code>
       </li>
       <li>
        比较特殊性 - css优先级
       </li>
       <li>
        比较源次序 - 后写的css覆盖先写的
       </li>
      </ul>
     </li>
    </ul>
    <blockquote>
     <p>
      通过比较重要性：浏览器默认的font-size:2em被覆盖
      <br/>
      通过比较特殊性：确认 container h1样式被最终生效
      <br/>
      通过比较源次序：container h1中的font-size:16px;为最终生效值
     </p>
    </blockquote>
    <p>
     上述通过步骤二，可以提取出的 font-size:16px;
    </p>
    <h3>
     <a id="3_106">
     </a>
     3.使用继承
    </h3>
    <blockquote>
     <p>
      对仍然没有值得属性，若可以继承，则继承父元素的值
     </p>
    </blockquote>
    <p>
     <code>
      需要明确可继承属性！！！
     </code>
    </p>
    <h3>
     <a id="4_110">
     </a>
     4.使用默认值
    </h3>
    <blockquote>
     <p>
      对仍然没有值的属性，使用默认值！
     </p>
    </blockquote>
