# uniapp 微信小程序 升级 uniad插件版本号

问题描述：

> 每次提交代码升级的时候会弹窗提示：uniad插件版本太低…

解决办法

### 一、使用微信小程序开发工具点击右上角

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/40fd1dd19eef4ec8a3fbc04df77f51ec.png)
  
查看到最新版本：1.3.4

### 二、在app.json中改为最新的版本即可

```bash
"uni-ad": {
      "version": "1.3.4",
      "provider": "wxf72d316417b6767f"
    }

```
