    <h1 class="title-article" id="articleContentId">
     CentOS 8 停止维护后通过 rpm 包手动安装 docker
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     根据
     <a href="https://docs.docker.com/engine/install/centos/#install-from-a-package" rel="nofollow" title="Docker官方文档">
      Docker官方文档
     </a>
     的指引，进入 Docker rpm 包下载的地址，根据自己系统的架构和具体版本选择对应的路径
    </p>
    <p>
    </p>
    <p class="img-center">
     <a href="https://img2024.cnblogs.com/blog/1772049/202410/1772049-20241008231407820-967132668.png" rel="nofollow">
      <img alt="image" height="641" src="https://i-blog.csdnimg.cn/img_convert/ec23c918882a4da143f34e00d0d997df.png" width="1236"/>
     </a>
    </p>
    <p>
     这里使用
     <a href="https://download.docker.com/linux/centos/7/x86_64/stable" rel="nofollow" title="Index of linux/centos/7/x86_64/stable/">
      Index of linux/centos/7/x86_64/stable/
     </a>
     版本，根据 docker 官方的给出的安装命令选择性的下载对应的 rpm 包
    </p>
    <p>
    </p>
    <p class="img-center">
     <a href="https://img2024.cnblogs.com/blog/1772049/202410/1772049-20241008232220019-1524942668.png" rel="nofollow">
      <img alt="image" height="986" src="https://i-blog.csdnimg.cn/img_convert/305f01520ab041bfd7a5df30a3a34a0c.png" width="1542"/>
     </a>
    </p>
    <p>
     最终使用 yum 命令安装下载好的 rpm 包即可
    </p>
    <pre><code># 如果你不清楚都下载那些rpm包，可以参考这段命令
yum localinstall -y containerd.io-1.6.32-3.1.el8.x86_64.rpm \
                     docker-ce-cli-26.1.3-1.el8.x86_64.rpm \
                     docker-ce-26.1.3-1.el8.x86_64.rpm \
                     docker-buildx-plugin-0.14.0-1.el8.x86_64.rpm \
                     docker-compose-plugin-2.27.0-1.el8.x86_64.rpm
</code></pre>
    <p>
     下载完成后就可以直接使用 docker 了
    </p>
    <pre><code># 设置Docker开机自启
[root@localhost docker-engine]# systemctl enable docker

# 启动Docker服务
[root@localhost docker-engine]# systemctl start docker

# Docker安装完成
[root@localhost docker-engine]# docker ps
CONTAINER ID   IMAGE     COMMAND   CREATED   STATUS    PORTS     NAMES
</code></pre>
