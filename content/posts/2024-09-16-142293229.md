### 1. 项目介绍

Hanime1Plugin 是一个针对 Android 平台的插件项目，主要用于增强在 Hanime1.me 网站上的观看体验。该项目旨在提供一种高效、便捷的方式来改善在线动画的播放质量，消除广告干扰，使得观影更加流畅舒适。Hanime1Plugin 是一个开源项目，遵循 GPL-3.0 许可证，用户可以自由使用并参与到项目的改进中。

### 2. 项目快速启动

#### 2.1 环境准备

在开始之前，请确保你已经安装了以下工具：

* Git
* Android Studio
* JDK 8 或更高版本

#### 2.2 克隆项目

首先，克隆 Hanime1Plugin 项目到本地：

```bash
git clone https://github.com/feiyeyuanye/Hanime1Plugin.git

```

#### 2.3 导入项目

1. 打开 Android Studio。
2. 选择
   `File`
   ->
   `Open`
   ，然后导航到你克隆项目的目录并选择
   `Hanime1Plugin`
   文件夹。
3. 等待 Android Studio 完成项目的导入和构建。

#### 2.4 运行项目

1. 连接你的 Android 设备或启动模拟器。
2. 在 Android Studio 中，点击
   `Run`
   按钮（绿色三角形）。
3. 选择你的设备或模拟器，项目将会自动安装并运行。
