    <h2>
     <a id="1__0">
     </a>
     1. 下载与安装
    </h2>
    <p>
     以
     <code>
      ideaIU-2021.3.1
     </code>
     为例，安装步骤：
    </p>
    <ol>
     <li>
      以管理员身份启动
      <code>
       ideaIU-2021.3.1
      </code>
     </li>
     <li>
      修改安装路径为：
      <code>
       D:\Program Files\JetBrains\IntelliJ IDEA 2021.3.1
      </code>
     </li>
     <li>
      勾选【创建桌面快捷方式】（可选）、【打开文件夹作为项目】（可选）。
     </li>
     <li>
      解压crack文件，双击里面的【破解】文件，出现
      <code>
       success
      </code>
      的提示。
     </li>
     <li>
      双击打开IDEA2021，选择Get license from:【Activation code】，其下方
      <strong>
       有一个面板
      </strong>
      。
     </li>
     <li>
      将解压的crack文件夹中的激活码拖拽到
      <strong>
       上述面板
      </strong>
      中。
     </li>
     <li>
      点击【Activate】。
     </li>
    </ol>
    <p>
     至此，英文版安装好了。更多细节和汉化步骤参考：
     <a href="https://mp.weixin.qq.com/s?__biz=MzA4MjU4MTg2Ng==&amp;mid=2247552352&amp;idx=1&amp;sn=ba9828dfafa2f9ecd016485a066bdaea" rel="nofollow">
      软件管家中的IDEA下载与安装
     </a>
     。
    </p>
    <blockquote>
     <p>
      其他安装教程参考：
      <a href="https://www.bilibili.com/video/BV1g4411k7UJ" rel="nofollow">
       https://www.bilibili.com/video/BV1g4411k7UJ
      </a>
     </p>
    </blockquote>

     2. 配置maven及国内镜像
    </h2>
    <h3>
     <a id="21__16">
     </a>
     2.1 下载与安装
    </h3>
    <p>
     方式1：可以下载与安装非idea自带的maven，可以
     <a href="https://blog.csdn.net/liqian_ken/article/details/105650143">
      参考该文1
     </a>
     ，或者
     <a href="https://blog.csdn.net/2301_76357803/article/details/136013359">
      参考该文2
     </a>
     。
    </p>
    <p>
     方式2：先使用idea自带的maven。
    </p>
    <blockquote>
     <p>
      idea自带maven的本地路径参考：
      <code>
       D:\Program Files\JetBrains\IntelliJ IDEA 2021.3.1\plugins\maven\lib\maven3
      </code>
      。
     </p>
    </blockquote>
    <p>
     可以在【Settings】-【Build, Execution, Deployment】-【Build Tools】-【Maven】中，找到（或重新指定）默认的maven配置文件地址：即【User settings file】的值（某settings.xml）。
    </p>

    <h2>

     3. 创建简单工程（简单示例）
    </h2>
    <h3>
     <a id="31__57">
     </a>
     3.1 创建一个空白工程
    </h3>
    <ol>
     <li>
      新建工程New Project，选择【Empty Project】，Project name设置为LearnJava，路径默认，点击确认，由此创建一个空白工程。
     </li>
    </ol>
    <h4>
     <a id="311_Java_60">
     </a>
     3.1.1 创建一个Java项目
    </h4>
    <ol>
     <li>
      选择【File】-【New】-【Module】，选择【Java】模块，可以设置模块的SDK（暂且使用默认的OpenJDK1.8.0_144，下同），设置模块名为demo。
     </li>
     <li>
      在src文件夹内创建一个TestDemo类，输入main并回车，自动创建main方法，输入sout并回车快速创建控制台打印。
     </li>
     <li>
      在控制台打印字符，并运行/调试。
     </li>
    </ol>
    <h4>
     <a id="312_maven_65">
     </a>
     3.1.2 创建一个maven项目

      <code>
       org.apache.maven.archetypes:maven-archetype-quickstart
      </code>
      .
     </li>
     <li>
      项目名设置为demo-maven，其余默认，maven仓库如果需要配置则使用上面配置过的maven。

     4.1 Setting相关
    </h3>
    <h4>
     <a id="411git_bashTerminal_106">
     </a>
     4.1.1配置git bash终端（Terminal）
    </h4>
    <p>
     【Settings】-【Tools】-【Terminal】，将【shell path】（默认值
     <code>
      powershell.exe
     </code>
     ）修改为Git安装目录中的
     <code>
      bin\bash.exe
     </code>
     路径，例如：
     <code>
      D:\Program Files\Git\bin\bash.exe
     </code>
     。然后新建一个终端就是git bash了。
    </p>
