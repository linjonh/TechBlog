    <p>
     jsonb会打乱原json属性顺序，但是支持索引
    </p>
    <p>
     原json：
    </p>
    <pre><code class="language-javascript">[{
        "ITEM": "W(nm)",
        "HZ1": 0,
        "HZ2": 0
    }, {
        "ITEM": "OL(nm)",
        "HZ1": 0,
        "HZ2": 0
    }, {
        "ITEM": "OR(nm)",
        "HZ1": 0,
        "HZ2": 0
    }
]</code></pre>
    <pre><code class="language-sql">CREATE TABLE mc.test (
	id int8 GENERATED BY DEFAULT AS IDENTITY NOT NULL,
	json_detail jsonb NOT NULL
);</code></pre>
    <p>
    </p>
    <p>
     <strong>
      json类型保存数据库后：
     </strong>
    </p>
    <blockquote>
     <p>
      [{"HZ1": 0, "HZ2": 0, "ITEM": "W(nm)"}, {"HZ1": 0, "HZ2": 0, "ITEM": "OL(nm)"}, {"HZ1": 0, "HZ2": 0, "ITEM": "OR(nm)"}]
     </p>
    </blockquote>
    <p>
     <img alt="" height="145" src="https://i-blog.csdnimg.cn/direct/d7006aefe68349e88975cfda48eb9e52.png" width="914"/>
    </p>
    <p>
    </p>
    <p>
     <strong>
      改成json类型后：
     </strong>
    </p>
    <pre><code class="language-sql">CREATE TABLE mc.test (
	id int8 GENERATED BY DEFAULT AS IDENTITY NOT NULL,
	json_detail json NOT NULL
);</code></pre>
    <blockquote>
     <p>
      [{"ITEM":"W(nm)","HZ1":0,"HZ2":0},{"ITEM":"OL(nm)","HZ1":0,"HZ2":0},{"ITEM":"OR(nm)","HZ1":0,"HZ2":0}]
     </p>
    </blockquote>

    </p>
