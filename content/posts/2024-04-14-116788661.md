# linux各版本大小,linux系统多大

![0b38c620c6db2b18d5c4eeeb606c90f7.png](https://i-blog.csdnimg.cn/blog_migrate/d74a75e468e2da3bcbed4aaf3ed0035e.jpeg)

linux系统多大

Linux具有很多发行版，从桌面到服务器版本，各个系统的大小也是不同的。总的来说范围是十几兆到4G以上。

桌面版多则达到4个G，涵盖了日常学习办公的大部分软件及漂亮的UI界面。少则只需要几百兆，甚至十几兆，当然只有普通的终端界面。

下面介绍几款非常小的Linux发行版：

1、Tiny Core Linux：终端版本只有 11 MB，图形界面版本只有 16 MB。不使用图形界面，运行 Tiny Core 就只需要 64 MB 的内存。

2、SliTaz Linux：SliTaz Linux 的镜像文件有大约 51 MB 大小，差不多是 Tiny Core 的四倍，但是包含一整套出色的驱动程序和应用程序。

3、Porteus Linux：Porteus 提供了不同的桌面环境可供选择，最小的镜像文件大约在 270 MB 左右，最大的有 350 MB。

4、Bodhi Linux：Bodhi Linux 的 ISO 镜像文件有 740 MB 大小，初看之下并不是很“微型”，不过一旦安装完成之后，你就会惊讶于它是多么微型了。

5、Puppy Linux：早在 Tiny Core、SliTaz、 AntiX 或者是 Porteus 诞生之前，就已经有 Puppy Linux 了。作为最早的微型 Linux 发行版之一，Puppy 已经历经了十五年风霜，并且无论是对于老爷机还是新用户来说始终都是一个可靠的、可启动的操作系统。
