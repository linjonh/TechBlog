题目：
  
Python 提供的 sun() 函数可以接受一个list并求和，请编写一个prod() 函数，可以接受一个list 并利用 reduce() 求积。

答案：

```
from functools import reduce 
def prod(n):
    def product_rule(x,y):
        return x*y
    return reduce(product_rule,n)
print('3*5*7=',prod([3,5,7]))   
```
