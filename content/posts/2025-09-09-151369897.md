# 图像金字塔---图像上采样下采样



![](https://i-blog.csdnimg.cn/direct/33785184825643a19ff253d8e824bd2f.png)

我们先展示一个索隆图片

```

import cv2
face = cv2.imread('zl.png',cv2.IMREAD_GRAYSCALE)#G0
cv2.resize(face,(600,600))
cv2.imshow('face',face)
cv2.waitKey(0)
```

![](https://i-blog.csdnimg.cn/direct/d02a77aed4b94f0aaec86b3eadf818de.png)

## 下采样

这里下采样就是

向金字塔顶部移动时，图像的尺寸和分辨率都不断地降低。通常情况下，每向上移动一级，图像的宽和高都降低为原来的1/2。

做法：

1、高斯滤波

 2、删除其偶数行和偶数列 OpenCV函数cv2.pyrDown()

```

face_down_1 = cv2.pyrDown(face)#下采样G1
cv2.imshow('down_1',face_down_1)
cv2.waitKey(0)
```

![](https://i-blog.csdnimg.cn/direct/1155b359bd844d2ebcfc526a314a9fb7.png)

## 上采样

通常将图像的宽度和高度都变为原来的2倍。这意味着，向上采样的结果图像的大小是原始图像的4倍。因此，要在结果图像中补充大量的像素点。对新生成的像素点进行赋值的行为，称为插值。 做法：

1、插值

2、高斯滤波

![](https://i-blog.csdnimg.cn/direct/cf7fe37522fb42e8a00d04f8a18ebadf.png)

```

face_up_1 = cv2.pyrUp(face)
cv2.imshow('up_1',face_up_1)#G1'
cv2.waitKey(0)
```

![](https://i-blog.csdnimg.cn/direct/f4fc1290668348e182d63ccb77e0382a.png)

我们可以看到这个看着模糊了一点。

## 拉普拉斯金字塔

![](https://i-blog.csdnimg.cn/direct/ec7a2660af6c475492c00a71ec115d3a.png)

这个很简单就是我们下采样之后是不能变成原图的，就算尺寸变回之后也会非常模糊。![](https://i-blog.csdnimg.cn/direct/dff328c4292d4641ac81d44af462c2ae.png)

这个时候我们用原图减去这个下采样之后上采样的图像

![](https://i-blog.csdnimg.cn/direct/b8ebbdce8dc149f29cb20e14a721b4f9.png)

得到了这个，然后我们再加一起，是不是就变成原图了，聪明！！！

![](https://i-blog.csdnimg.cn/direct/d02a515d065949cd8df8b147fe3a4e88.png)

这个就是图像金字塔。

完整代码

```

# dst = cv2.pyrDown(src[,dst,dstsize[,borderType]])
# dst: 目标图像
# src: 原始图像
# dstsize: 目标图像的大小
import cv2
face = cv2.imread('zl.png',cv2.IMREAD_GRAYSCALE)#G0
cv2.resize(face,(600,600))
cv2.imshow('face',face)
cv2.waitKey(0)
face_down_1 = cv2.pyrDown(face)#下采样G1
cv2.imshow('down_1',face_down_1)
cv2.waitKey(0)
face_down_2 = cv2.pyrDown(face_down_1)#G2
cv2.waitKey(0)
#高斯金字塔操作中的向上采样
# dst = cv2.pyrUp(src[,dst,dstsize[,borderType]])
# dst: 目标图像
# src: 原始图像
# dstsize: 目标图像的大小
face_up_1 = cv2.pyrUp(face)
cv2.imshow('up_1',face_up_1)#G1'
cv2.waitKey(0)
face_up_2 = cv2.pyrUp(face_up_1)
cv2.imshow('up_2',face_up_2)#G2'
### 对下采用后图像进行上采样，图像变模糊，无法复原
face_down_1_up = cv2.pyrUp(face_down_1)#下采样G1
face_down_2_up = cv2.pyrUp(face_down_2)#下采样G2
cv2.imshow('down_1_up',face_down_1_up)
cv2.imshow('down_2_up',face_down_2_up)
cv2.waitKey(0)
### 拉普拉斯金字塔
L0 = face - face_down_1_up
L1 = face_down_1 - face_down_2_up
fuyuan = face_down_1_up + L0
cv2.imshow('L0',L0)
cv2.imshow('L1',L1)
cv2.waitKey(0)
cv2.imshow('fuyuan',fuyuan)
cv2.waitKey(0)
```
