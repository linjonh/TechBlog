# Centos 7.6离线安装docker



在内网环境下，一般不能联网在线部署，这时候就需要以离线的方式安装docker。本节内容主要总结一下在CentOS 7.6环境中离线安装docker的步骤。  
1、下载docker安装包  
https://pan.baidu.com/share/init?surl=PaUllQZ-dwpgJ7quA5IkcQ&pwd=4sfc  
2、上传到服务器/home/目录下，解压缩

```bash
cd /home/
tar -zxvf docker-20.10.13.tgz

```

3、将解压后docker目录内容拷贝到/usr/bin/

```bash
cp docker/* /usr/bin/

```

4、复制docker.service到/etc/systemd/system/目录

```bash
cp docker.service /etc/systemd/system/

```

5、启动docker

```bash
systemctl daemon-reload
systemctl start docker

```

6、查看docker状态

```bash
systemctl status docker

```

7、设置开机启动

```bash
systemctl enable docker

```

8、查看docker版本

```bash
docker version

```

若需要配置docker-compose，继续执行后续操作  
9、将docker-compose文件放到全局并添加权限

```bash
cd /home/
mv docker-compose /usr/local/bin/
sudo chmod +x /usr/local/bin/docker-compose

```
