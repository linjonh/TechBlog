# python连接新大陆物联网云平台

主流的 API 开发模式下，第一步是要获取ACCESSTOKEN的，包括在新大陆物联网云平台中，也是需要先获取到ACCESSTOKEN的。

在新大陆物联网云平台中也给过单个查询传感器状态

```
GET 
   http://api.nlecloud.com/devices/{deviceId}/Sensors/{apiTag}
```

当填写正确的设备ID，跟设备标识时，会获取到这个传感器的状态
