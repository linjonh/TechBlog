# pycharm 最新版上一次编辑位置

![](https://i-operation.csdnimg.cn/images/cf31225e169b4512917b2e77694eb0a2.png)PyCharm返回上一次编辑位置技巧

pycharm

##### 方法一：用快捷键（最方便）

* **跳回上一次编辑位置**：

  + Windows/Linux: `Ctrl + Alt + ←`
  + macOS: `⌘ + Option + ←`
* **跳到前一次位置**：

  + Windows/Linux: `Ctrl + Alt + →`
  + macOS: `⌘ + Option + →`

##### 方法二：显示工具栏按钮

1. 在菜单里点 **View → Appearance → Toolbar**，确保工具栏是显示的。
2. 在工具栏空白处右键 → **Customize Toolbar...**
3. 在 “Available Actions” 里搜索 **Navigate Back** 和 **Navigate Forward**，把它们加回去。

##### 方法三：用 “最近编辑位置”

* 快捷键：

  + Windows/Linux: `Ctrl + Shift + Backspace`
  + macOS: `⌘ + Shift + Delete`
* 这个会直接带你回到最近修改过的地方，比导航更精准。
