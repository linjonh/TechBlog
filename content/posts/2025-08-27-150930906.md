## 四、配置并安装

### 1、安装wget

```bash
yum install wget

```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/02775029415642719a8a2158c8b2d3ed.png)

### 2、安装 yum-utils

```bash
yum install -y yum-utils

```

### 3、添加软件源信息

```bash
yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

```

### 4、查询可用版本

```bash
yum list docker-ce --showduplicates

```

### 5、安装Docker

```bash
yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

```

### 6、在CentOS9中启用 systemd

直接启动会报错：  
 ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/d618dbbf5e594edd955cabbfa49e4ab3.png)  
 解决：

```bash
vi /etc/wsl.conf

# 添加内容：
[boot]
systemd=true

```

保存后，重启系统

### 7、启动docker

```bash
systemctl start docker

```

设置开机自启

```bash
systemctl enable docker

```

取消开机自启

```bash
systemctl disable docker

```

如果docker无法search或者pull镜像，请去看我的另一篇文章：  
 [docker拉取镜像报错解决方式](https://blog.csdn.net/qq_43388893/article/details/149254022?spm=1011.2415.3001.5331)

## 五、安装ssh

### 1、安装

```bash
# 安装
yum install -y openssh-server
# 启动
systemctl start sshd
# 设置开机自启
systemctl enable sshd

```

### 2、配置第三方工具使用root用户连接

```bash
vi /etc/ssh/sshd_config

```

在文件中找到关键字：PermitRootLogin  
 有可能是被注释掉的：  
 ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/579db43df8bd45c5a5f2715c84d17fc0.png)  
 取消注释，并将后面的值修改为yes  
 ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/eb39efe0667d4287910209802eebcd35.png)  
 保存后，重启sshd，

```bash
systemctl restart sshd

```
