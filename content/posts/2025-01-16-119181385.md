    <h4 id="h3--windows-" style="margin-left:0;text-align:left;">
     <span style="color:#333333;">
      企业微信Windows版本调试
     </span>
    </h4>
    <ol style="margin-left:0;">
     <li>
      把
      <a href="https://dldir1.qq.com/foxmail/wwopen_docFile/devtools_resources.pak" rel="nofollow">
       devtools_resources.pak
      </a>
      放到企业微信的安装目录下（
      <span style="color:#FF0000;">
       复制的文件名，要保证为 devtools_resources.pak
      </span>
      ），注意安装目录带有版本号（4.0.1304.400是浏览器内核的版本号）；
     </li>
    </ol>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#333333;">
      <img alt="devtools_resources.pak存储路径" src="https://i-blog.csdnimg.cn/blog_migrate/fdf66b3bc1829530edcbfcf8ae7d81fa.png" width="600"/>
     </span>
    </p>
    <ol style="margin-left:0;">
     <li>
      关闭企业微信，重新启动；
     </li>
     <li>
      按快捷键 ctrl + alt + shift + D，进入调试模式；
     </li>
     <li>
      在需要调试的页面上点击右键，点“开发者工具”，进入调试模式。
      <br/>
      <img alt="win开发者工具" src="https://i-blog.csdnimg.cn/blog_migrate/a0acf9f89f95d6b8b758aa37254b0f46.png"/>
     </li>
    </ol>
    <hr/>
    <h4 id="h3--mac-" style="margin-left:0;text-align:left;">
     <span style="color:#333333;">
      <a class="reference-link" name="企业微信Mac版本调试">
      </a>
      企业微信Mac版本调试
     </span>
    </h4>
    <ol style="margin-left:0;">
     <li>
      按下command+shift+control+D进入调试模式
      <br/>
      <img alt="Mac打开调试模式" src="https://i-blog.csdnimg.cn/blog_migrate/2d69247dad467fb9b6f1889ef22ef828.png"/>
     </li>
     <li>
      帮助-&gt;开发调试选项-&gt;开启webView元素审查
      <br/>
      <img alt="Mac开启元素审查" src="https://i-blog.csdnimg.cn/blog_migrate/5215db117519604ce8b960c206d51367.png"/>
     </li>
     <li>
      内置浏览器右键
      <br/>
      <img alt="Mac检查页面元素" src="https://i-blog.csdnimg.cn/blog_migrate/bfb1fc26acc62469be35cd2054473079.png"/>
     </li>
    </ol>
