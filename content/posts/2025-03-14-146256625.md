# Modbus TCP到RTU：轻松转换指南！

**Modbus TCP 到 RTU：轻松转换指南！**

在现代工业自动化领域，
****Modbus TCP和Modbus RTU****
两种通信协议因其高效、稳定的特点被广泛应用。然而，随着技术的发展和设备升级的需求，经常会遇到需要将这两种协议进行互相转换的场景。这时，
****Modbus TCP转Modbus RTU****
网关便扮演了至关重要的角色，它不仅实现了
****Modbus TCP与Modbus RTU****
之间的无缝连接，还为数据的集中管理和统一监控提供了可能。

![](https://i-blog.csdnimg.cn/direct/7d820eb451564becb0d37c98d723fb51.png)

下面是具体操作步骤：

1、登陆到配置网页修改串口参数，实例个数是10的话（每一个 Modbus TCP 服务最多同时支持 5 个TCP 主站连接访问，最多可创建 10 个本地主站）：

![](https://i-blog.csdnimg.cn/direct/31291aa1e8f048af9fc750dd3271a4e5.png)

2、Modbus RTU转Modbus TCP 网关作为本地从站，有两种通讯方式：网络通讯和串口通讯，三种通讯类型：服务器、客户端、串口，其中服务器和客户端属于网通通讯。

![](https://i-blog.csdnimg.cn/direct/cc5f265d732240b7a5e0fecc5cd4b4f5.png)

3、填入相应的参数，点击保存，提示创建成功，如下图：

![](https://i-blog.csdnimg.cn/direct/a2727ee3ae204c75b22380e2167e1864.png)

4、添加完本地从站参数后点击保存：

![](https://i-blog.csdnimg.cn/direct/98d8801868e74e0c88f4f8b48ab22a44.png)

5、打开 Modbus Poll  工具，在菜单 File  中选择 New-Setup-Read/Write  Definition 并创建从站 A（Mbpoll1）:

![](https://i-blog.csdnimg.cn/direct/d5e98d60180c4564bc190d818d138cd4.png)

6、在菜单 File 中选择New-Setup-Read/Write Definition 并创建从站 B（Mbpoll2）:

![](https://i-blog.csdnimg.cn/direct/471f49a5d0334e2b88003849161fb600.png)

7、所有从站创建完之后，设置采集相应的数据，这里就列举其中 A、B两个从站，如下图：

![](https://i-blog.csdnimg.cn/direct/dc176e7190564b279cf36131a00249ed.png)

****Modbus TCP到Modbus RTU****
的转换是连接不同通信世界的重要桥梁。随着工业自动化技术的发展和多样化需求的增加，这种转换技术的重要性将会日益凸显。无论是硬件制造商还是系统集成商，了解和掌握这项技术都将极大地提升他们在市场中的竞争力。
