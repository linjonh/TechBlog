    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="background-color:white">
      <span style="color:#2f5df2">
       读取Excel文件
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       在pandas模块中，读取Excel文件主要使用
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        pd.read_excel()
       </span>
      </span>
      <span style="color:#707070">
       函数。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       将 必选参数 - Excel文件的
      </span>
      <u>
       <span style="color:#2f5df2">
        路径
       </span>
      </u>
      <span style="color:#707070">
       传入该函数中，便可以得到对应的DataFrame格式的数据。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="920" src="https://i-blog.csdnimg.cn/direct/b63019242d89448cafcb4b297f6390c7.png" width="1267"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="background-color:white">
      <span style="color:#707070">
       下面是一个简单的示例：将一份2019年的超市销售订单数据的Excel文件
      </span>
     </span>
     <u>
      <span style="background-color:white">
       <span style="color:#2f5df2">
        路径
       </span>
      </span>
     </u>
     <span style="background-color:white">
      <span style="color:#707070">
       传入
      </span>
     </span>
     <code>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        pd.read_excel()
       </span>
      </span>
     </code>
     <span style="background-color:white">
      <span style="color:#707070">
       函数后，得到了该数据对应的DataFrame格式的数据。
      </span>
     </span>
     <br/>
     <span style="background-color:white">
      <span style="color:#707070">
       此时，读取出的数据会被自动添加默认从0开始的行索引index和列索引columns，并且columns默认是原来数据的第一行。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <strong>
      <span style="color:#00b0f0">
       面对特殊情况的处理
      </span>
     </strong>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       和读取CSV文件一样，
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        pd.read_excel()
       </span>
      </span>
      <span style="color:#707070">
       函数也为我们提供了大量的可选参数来处理形形色色的情况。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       之前学习的参数也可以用在
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        pd.read_excel()
       </span>
      </span>
      <span style="color:#707070">
       函数中。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       除此之外，还有1个特殊的场景是只在处理Excel文件时能遇到的。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       接下来，我们将通过分析这个场景来帮助大家理解
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        pd.read_excel()
       </span>
      </span>
      <span style="color:#707070">
       函数中另一个最常用的可选参数。
      </span>
     </span>
    </p>

     <span style="color:#2f5df2">

           <span style="color:#ff502c">
            pd.read_excel()

           sheet_name

        <p style="margin-left:0; margin-right:0; text-align:left">

         <span style="background-color:#fff5f5">
          <span style="color:#ff502c">
           pd.read_excel()
          </span>
         </span>
         函数提供了一个参数：
         <span style="background-color:#fff5f5">
