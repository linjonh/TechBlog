    <h1 class="title-article" id="articleContentId">
     终身成长，不枉此生
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <p align="justify">
     <span style="font-family:宋体">
      最近看了一本书
     </span>
     ——《终身成长》，受益颇丰。这本书上主要是写了一种思维模式，即成长型思维模式。接下来呢，就对成长型思维模式和它的死对头——固定性思维模式进行讲解，愿和你分享
     <span style="font-family:Calibri">
      ~~
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      固定型思维模式认为，聪明才智等能力是天生的，后天无法改变。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      成长型思维模式认为，天赋只是起点，人的才智通过锻炼可以提高，只要努力就可以做的更好。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      固定型思维模式中，成功来源于
     </span>
     <span style="font-family:宋体">
      事情的结果，能够证明自己的天赋和能力
     </span>
     <span style="font-family:宋体">
      。而成长型思维模式中，成功来源于
     </span>
     <span style="font-family:宋体">
      尽自己最大努力做事
     </span>
     <span style="font-family:宋体">
      ，来源于学习和自我提高。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      固定型思维模式者认为，一次考试失利、输掉一场比赛、被拒绝等等，都意味着
     </span>
     <span style="font-family:宋体">
      自己是一个失败者，不具有这方面天赋。
     </span>
     <span style="font-family:宋体">
      而且，他们不会从失败中学习并纠正自己，相反，他们可能只是去尝试去修复自尊，不愿意面对挑战。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      成长型思维模式者则认为，
     </span>
     <span style="font-family:宋体">
      自身没有成长或者没有尽到最大努力，才意味着失败。
     </span>
     <span style="font-family:宋体">
      遇到挫折和挑战，会看做是提高自己的机会，不断改善和进步。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      那么，成长型思维模式到底有什么作用呢？
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      在职场中，具有成长型思维的领导更善于
     </span>
     <span style="font-family:宋体">
      倾听
     </span>
     <span style="font-family:宋体">
      他人的批判性建议，更重视员工的
     </span>
     <span style="font-family:宋体">
      个人发展
     </span>
     <span style="font-family:宋体">
      。具有成长型思维的企业和团队更具有
     </span>
     <span style="font-family:宋体">
      包容性
     </span>
     <span style="font-family:宋体">
      ，能够更加开放自由的交流。而具有成长型思维的员工，在
     </span>
     <span style="font-family:宋体">
      具有挑战性的合作谈判等任务
     </span>
     <span style="font-family:宋体">
      中更有竞争力。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      在人际关系中，具有成长型思维的人更具
     </span>
     <span style="font-family:宋体">
      同理心，会主动经营关系
     </span>
     <span style="font-family:宋体">
      ，相信爱情、友谊等一切关系都是可以培养和成长的。在社交中也更多的
     </span>
     <span style="font-family:宋体">
      关注他人，而非别人对自己的评价
     </span>
     <span style="font-family:宋体">
      ，因此也更容易突破害羞等性格方面导致的社交障碍，积极地与他人互动。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      《终身成长》这本书中，提供了获得成长型思维的四步法：
     </span>
    </p>
    <p align="justify">
     ▌
     <span style="font-family:宋体">
      第一步：接受
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      首先要接受并拥抱自己的固定型思维模式。
     </span>
     <span style="font-family:宋体">
      大部分人并不是只拥有一种思维模式，而是两种思维模式的混合体，
     </span>
     <span style="font-family:宋体">
      在某些情况下处于固定型思维模式，另一些情况下处于成长型思维模式，这是很正常的。但接受它的存在，并不代表接受让它频繁出现和它带来的危害。
     </span>
    </p>
    <p align="justify">
     ▌
     <span style="font-family:宋体">
      第二步：观察
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      通过观察，明确是
     </span>
     <span style="font-family:宋体">
      什么激发了自己的固定型思维模式。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      观察它通常在
     </span>
     <span style="font-family:宋体">
      什么时候出现？
     </span>
     <span style="font-family:宋体">
      也许是在面对一个巨大挑战的时候，它开始出现并劝你退缩。也许是在遭遇失败时，它突然出现并泼你冷水。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      观察自己在固定型思维模式下，
     </span>
     <span style="font-family:宋体">
      是怎么对待他人的？
     </span>
     <span style="font-family:宋体">
      是否经常会评判别人的才能？是否经常会给孩子压力？
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      还有想一想，
     </span>
     <span style="font-family:宋体">
      最近一次
     </span>
     <span style="font-family:宋体">
      激发固定型思维模式人格的是什么事？当时它说了什么，自己有什么感觉？
     </span>
     <span style="font-family:宋体">
      先观察一下，不要急于评价。
     </span>
    </p>
    <p align="justify">
     ▌
     <span style="font-family:宋体">
      第三步：命名
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      给自己的固定型思维模式人格起个名字。
     </span>
     <span style="font-family:宋体">
      然后
     </span>
     <span style="font-family:宋体">
      描述
     </span>
     <span style="font-family:宋体">
      一下它是什么样的：什么时候出现，是什么性格，以及如何影响我们。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      比如书中提到一位财务经理，给自己的固定型思维模式人格起名为
     </span>
     “
     <span style="font-family:宋体">
      杜安
     </span>
     <span style="font-family:Calibri">
      ”
     </span>
     <span style="font-family:宋体">
      ，他说：
     </span>
     <span style="font-family:Calibri">
      “
     </span>
     <span style="font-family:宋体">
      当我们遇到困境的时候，我的杜安就会出现。他让我变得对每个人都非常挑剔，我变得蛮横、苛刻，而不是去支持我的员工。
     </span>
     <span style="font-family:Calibri">
      ”
     </span>
     <span style="font-family:宋体">
      他的团队成员，在相互了解对方的固定型思维模式人格后，彼此的合作进入了一个新的阶段，团队的士气也大幅上升。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      如果你并不想把自己的固定型思维模式人格介绍给他人，也可以只在心中默默记住。
     </span>
    </p>
    <p align="justify">
     ▌
     <span style="font-family:宋体">
      第四步：教育
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      给固定型思维模式人格命名之后，
     </span>
     <span style="font-family:宋体">
      我们就可以开始教育他。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      具体做法是，在遇到可能激发自己固定型思维模式人格出现的事情时，首先要
     </span>
     <span style="font-family:宋体">
      保持警惕
     </span>
     <span style="font-family:宋体">
      。在他出现并阻止我们时，尝试让自己说服他，
     </span>
     <span style="font-family:宋体">
      告诉他为什么不这样做的原因，并邀请他和你一起面对挑战
     </span>
     <span style="font-family:宋体">
      ：
     </span>
     “
     <span style="font-family:宋体">
      我知道这可能会失败，但是我愿意试一试。你能不能对我有耐心一些？
     </span>
     <span style="font-family:Calibri">
      ”
     </span>
    </p>
    <p align="justify">
     So,
     <span style="font-family:宋体">
      我们怎么才能保持住成长型的思维模式呢？
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      当你犯错的时候，以前你会对自己说：
     </span>
     “我犯错了，我真失败！”，那么以后你就要对你自己说：“犯错能使我更好！”
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      当你遇到挑战的时候，以前的你会说：
     </span>
     “我不擅长这个。”那么，以后的你要说：“我正在提高。”
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      当你想放弃的时候，以前的你会说：
     </span>
     “我想放弃了。”那么以后的你要说：“我得试试其他的办法
     <span style="font-family:Calibri">
      ~
     </span>
     <span style="font-family:宋体">
      ”
     </span>
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      当你否定自己的时候，以前的你就会说：
     </span>
     “我没有这个天赋。”那么，以后的你就要对自己说：“只是训练不够而已，坚持训练一段时间看看。”
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      坚持以后在日常的生活中这样悄悄的训练自己，成长型的思维模式会慢慢养成的
     </span>
     ~~
    </p>
    <p align="justify">
     <span style="font-family:宋体">
      人的一生无法重来，每一天都无比珍贵，用成长型思维和这个世界相处，也许每天都会变得更有意义一些。
     </span>
     <span style="font-family:宋体">
      愿我们能在有限的生命里，终身成长，不枉此生。
     </span>
    </p>
