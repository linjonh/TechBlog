    <h2>
     1.元类
    </h2>
    <p>
     在 Python 中，
     <code>
      abc
     </code>
     模块（Abstract Base Classes，抽象基类）提供了创建抽象基类的机制，这些抽象基类可以用来定义接口，
     <strong>
      强制子类实现特定的方法
     </strong>
     ，有助于实现多态性和代码的可维护性。下面从多个方面详细介绍
     <code>
      abc
     </code>
     元类相关内容。
    </p>
    <p>
     <strong>
      <code>
       ABCMeta
      </code>
      是
      <code>
       abc
      </code>
     </strong>
     模块中的元类，
     <code>
      ABC
     </code>
     类实际上就是使用
     <code>
      ABCMeta
     </code>
     作为元类创建的。你也可以直接使用
     <code>
      ABCMeta
     </code>
     来创建抽象基类。
    </p>
    <pre><code class="language-python">class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

    @abstractmethod
    def perimeter(self):
        pass

class Rectangle(Shape):
    def __init__(self, length, width):
        self.length = length
        self.width = width

    def area(self):
        return self.length * self.width

    def perimeter(self):
        return 2 * (self.length + self.width)

# 创建 Rectangle 实例
rect = Rectangle(5, 3)
print(f"Area: {rect.area()}")
print(f"Perimeter: {rect.perimeter()}")

# 尝试创建 Shape 实例会引发 TypeError
try:
    shape = Shape()
except TypeError as e:
    print(f"Error: {e}")</code></pre>
    <h2>
     <strong>
      2.Registry
     </strong>
    </h2>
    <p>
     在深度学习和计算机视觉领域，
     <code>
      Registry
     </code>
     是一种常见的设计模式，用于管理和注册不同的组件，而
     <code>
      PIPELINES
     </code>
     通常用于存储数据处理流程中的各个操作步骤
    </p>
    <h4>
     <code>
      2.1Registry
     </code>
     类的概念
    </h4>
    <p>
     <code>
      Registry
     </code>
     类是一个注册表，它的主要作用是将不同的组件（如函数、类等）注册到一个全局的字典中，方便后续根据名称来查找和使用这些组件。这种设计模式在很多深度学习框架（如 MMDetection、MMCV 等）中被广泛使用，有助于实现代码的模块化和可扩展性。
    </p>
    <h5>
     2.2 注册组件
    </h5>
    <pre><code class="language-cpp">@PIPELINES.register_module()
class Resize:
    def __init__(self, size):
        self.size = size

    def __call__(self, data):
        # 实现图像缩放逻辑
        print(f"Resizing data to {self.size}")
        return data

@PIPELINES.register_module(module_name='Flip')
class HorizontalFlip:
    def __init__(self, flip_ratio):
        self.flip_ratio = flip_ratio

    def __call__(self, data):
        # 实现水平翻转逻辑
        print(f"Flipping data with ratio {self.flip_ratio}")
        return data</code></pre>
    <p>
     上述代码中，我们定义了两个数据处理类
     <code>
      Resize
     </code>
     和
     <code>
      HorizontalFlip
     </code>
     ，并使用
     <code>
      @PIPELINES.register_module()
     </code>
     装饰器将它们注册到
     <code>
      PIPELINES
     </code>
     注册表中。
     <code>
      Resize
     </code>
     类用于图像缩放，
     <code>
      HorizontalFlip
     </code>
     类用于水平翻转。
    </p>
    <h5>
     2.3 获取组件
    </h5>
    <pre><code class="language-cpp"># 根据名称从注册表中获取组件
resize_op = PIPELINES.get('Resize')
flip_op = PIPELINES.get('Flip')

# 创建组件实例并调用
resize = resize_op(size=(224, 224))
flip = flip_op(flip_ratio=0.5)

# 模拟数据
data = {'image': [1, 2, 3]}

# 调用组件
resized_data = resize(data)
flipped_data = flip(resized_data)</code></pre>
    <h2>
     3.one-hot
    </h2>
    <p>
     One-hot（独热编码）是一种在机器学习和数据处理中广泛使用的编码方式，主要用于将分类数据转换为计算机能够理解和处理的数字格式。下面从多个方面详细介绍 One-hot 编码。
    </p>
    <p>
     在 One-hot 编码中，每个类别都会被表示为一个二进制向量，向量的长度等于类别的总数。在这个向量中，只有对应类别的位置为 1，其余位置都为 0。例如，假设有三个类别：苹果、香蕉、橙子，那么它们的 One-hot 编码分别为：
    </p>
    <ul>
     <li>
      苹果：[1, 0, 0]
     </li>
     <li>
      香蕉：[0, 1, 0]
     </li>
     <li>
      橙子：[0, 0, 1]
     </li>
    </ul>
    <h2>
     4.装饰器
    </h2>
    <p>
     在神经网络当中，目前而言，其实也就是把函数封装成属性成员来使用或者元类和注册表搭配一下。
    </p>
    <pre><code class="language-cpp">
@property 用于定义属性的获取方法（getter），
@属性名.setter 用于定义属性的设置方法（setter），
@属性名.deleter 用于定义属性的删除方法（deleter）。</code></pre>
    <pre><code class="language-python">class Person:
    def __init__(self, age):
        self._age = age

    @property
    def age(self):
        """获取 age 属性的方法"""
        print("获取 age 属性的值")
        return self._age

    @age.setter
    def age(self, value):
        """设置 age 属性的方法"""
        if value &lt; 0:
            raise ValueError("年龄不能为负数")
        print("设置 age 属性的值")
        self._age = value

    @age.deleter
    def age(self):
        """删除 age 属性的方法"""
        print("删除 age 属性")
        del self._age

# 创建 Person 类的实例
p = Person(25)

# 获取 age 属性
print(p.age)

# 设置 age 属性
p.age = 30
print(p.age)

# 删除 age 属性
del p.age</code></pre>
    <h2>
     5.for
    </h2>
    <h4>
     <code>
      for i, c in train_data
     </code>
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <p>
       假设
       <code>
        train_data
       </code>
       是一个包含元组的可迭代对象，例如列表或元组本身。在这种遍历方式中，每个元组被解包，其元素分别赋值给
       <code>
        i
       </code>
       和
       <code>
        c
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       这种方式适用于
       <code>
        train_data
       </code>
       中的元组本身就包含了有意义的、需要分别处理的两个元素的情况。比如，元组可能表示坐标点
       <code>
        (x, y)
       </code>
       ，或者表示某种数据的键值对
       <code>
        (key, value)
       </code>
       。
      </p>
     </li>
    </ul>
    <h4>
     <code>
      for i, c in enumerate(train_data)
     </code>
    </h4>
    <ul>
     <li>
      <p>
       <code>
        enumerate()
       </code>
       函数用于将一个可迭代对象转换为一个枚举对象，其中每个元素都是一个包含索引和原始元素的元组。在这种遍历方式中，
       <code>
        i
       </code>
       被赋值为元组在
       <code>
        train_data
       </code>
       中的索引，
       <code>
        c
       </code>
       被赋值为
       <code>
        train_data
       </code>
       中的元组本身。
      </p>
     </li>
     <li>
      <p>
       这种方式更侧重于在遍历
       <code>
        train_data
       </code>
       时，同时获取元组的索引和元组内容，索引可以用于标识元组的位置或作为其他计算的依据。
      </p>
     </li>
    </ul>

    <h2>
     7.联合体
    </h2>
    <pre><code class="language-python">from typing import Union, Tuple

# 定义一个函数，参数 data 可以是一个包含四个整数的元组，也可以是一个整数
def process_data(data: Union[Tuple[int, int, int, int], int]) -&gt; None:
    if isinstance(data, tuple):
        print(f"Received a tuple: {data}")
    elif isinstance(data, int):
        print(f"Received an integer: {data}")

# 测试传入元组
process_data((1, 2, 3, 4))
# 测试传入整数
process_data(5)</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        类型注解含义
       </strong>
       ：
       <code>
        Union[Tuple[int, int, int, int], int]
       </code>
       表明
       <code>
        data
       </code>
       参数可以是
       <code>
        (int, int, int, int)
       </code>
       形式的元组（即包含四个整数的元组），也可以是一个单独的整数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        函数内部逻辑
       </strong>
       ：在
       <code>
        process_data
       </code>
       函数内部，使用
       <code>
        isinstance
       </code>
       函数来判断传入的
       <code>
        data
       </code>
       具体是元组类型还是整数类型，然后进行相应的处理。
      </p>
     </li>
    </ul>
