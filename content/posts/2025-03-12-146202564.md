   <div class="htmledit_views" id="content_views">
    <p>
     外表统计信息的收集方式和收集内容与内表基本一致，目前支持对 Hive，Iceberg 和 Hudi 等外部表的收集。
    </p>
    <p>
     自 2.0.3 版本之后，Hive 外表支持了自动和采样收集。
    </p>
    <h3 id="注意事项">
     注意事项
     <a href="https://doris.apache.org/zh-CN/docs/lakehouse/external-statistics#%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9" rel="nofollow" title="​">
      ​
     </a>
    </h3>
    <ol>
     <li>
      <p>
       HMS 类型的 Iceberg 和 Hudi 外表，以及 JDBC 外表只支持手动全量收集。其他类型的外表暂不支持统计信息收集。
      </p>
     </li>
     <li>
      <p>
       外表默认关闭自动统计信息收集功能，需要在创建 Catalog 的时候添加属性来打开，或者通过设置 Catalog 属性来开启或关闭。
      </p>
     </li>
    </ol>
    <h4 id="创建-catalog-时打开自动收集的属性-默认是-false">
     创建 Catalog 时打开自动收集的属性 (默认是 false）：
     <a href="https://doris.apache.org/zh-CN/docs/lakehouse/external-statistics#%E5%88%9B%E5%BB%BA-catalog-%E6%97%B6%E6%89%93%E5%BC%80%E8%87%AA%E5%8A%A8%E6%94%B6%E9%9B%86%E7%9A%84%E5%B1%9E%E6%80%A7-%E9%BB%98%E8%AE%A4%E6%98%AF-false" rel="nofollow" title="​">
      ​
     </a>
    </h4>
    <pre><code>'enable.auto.analyze' = 'true'
</code></pre>
    <p>
    </p>
    <h4 id="通过修改-catalog-属性控制是否开启自动收集">
     通过修改 Catalog 属性控制是否开启自动收集：
     <a href="https://doris.apache.org/zh-CN/docs/lakehouse/external-statistics#%E9%80%9A%E8%BF%87%E4%BF%AE%E6%94%B9-catalog-%E5%B1%9E%E6%80%A7%E6%8E%A7%E5%88%B6%E6%98%AF%E5%90%A6%E5%BC%80%E5%90%AF%E8%87%AA%E5%8A%A8%E6%94%B6%E9%9B%86" rel="nofollow" title="​">
      ​
     </a>
    </h4>
    <pre><code>ALTER CATALOG external_catalog SET PROPERTIES ('enable.auto.analyze'='true'); // 打开自动收集
ALTER CATALOG external_catalog SET PROPERTIES ('enable.auto.analyze'='false'); // 关闭自动收集
</code></pre>
    <p>
    </p>
