    <p>
     在麒麟系统（Kylin OS）上使用 PyCharm 进行 Python 开发并生成
     <code>
      .deb
     </code>
     可安装软件包，可以按照以下步骤进行操作：
    </p>
    <h4>
     <a id="1__2">
     </a>
     1. 准备工作
    </h4>
    <ul>
     <li>
      <strong>
       安装 PyCharm
      </strong>
      ：确保已经在麒麟系统上安装了 PyCharm，可以使用官方提供的安装包进行安装。
     </li>
     <li>
      <strong>
       安装必要的工具和依赖
      </strong>
      ：确保系统中安装了
      <code>
       dpkg
      </code>
      、
      <code>
       debhelper
      </code>
      等工具，这些工具用于创建和管理
      <code>
       .deb
      </code>
      包。
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> dpkg debhelper
</code></pre>
     </li>
     <li>
      <strong>
       安装 Python 开发环境
      </strong>
      ：确保系统中安装了 Python 和
      <code>
       setuptools
      </code>
      ，用于 Python 项目的构建和安装。
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> python3 python3-setuptools
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="2__Python__13">
     </a>
     2. 创建 Python 项目
    </h4>
    <ul>
     <li>
      <strong>
       在 PyCharm 中创建项目
      </strong>
      ：打开 PyCharm，创建一个新的 Python 项目。
     </li>
     <li>
      <strong>
       项目结构
      </strong>
      ：创建一个标准的 Python 项目结构，例如：
      <pre><code>myapp/
├── setup.py
├── myapp
│   ├── __init__.py
│   └── main.py
└── ...
</code></pre>
     </li>
     <li>
      <strong>
       编写
       <code>
        setup.py
       </code>
      </strong>
      ：在项目根目录下创建
      <code>
       setup.py
      </code>
      文件，用于定义项目的元数据和依赖关系。例如：
      <pre><code class="prism language-python"><span class="token keyword">from</span> setuptools <span class="token keyword">import</span> setup<span class="token punctuation">,</span> find_packages

setup<span class="token punctuation">(</span>
    name<span class="token operator">=</span><span class="token string">"myapp"</span><span class="token punctuation">,</span>
    version<span class="token operator">=</span><span class="token string">"0.1"</span><span class="token punctuation">,</span>
    packages<span class="token operator">=</span>find_packages<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">,</span>
    install_requires<span class="token operator">=</span><span class="token punctuation">[</span>
        <span class="token comment"># 列出项目依赖的 Python 库</span>
        <span class="token string">"requests"</span><span class="token punctuation">,</span>
        <span class="token string">"flask"</span><span class="token punctuation">,</span>
    <span class="token punctuation">]</span><span class="token punctuation">,</span>
    entry_points<span class="token operator">=</span><span class="token punctuation">{<!-- --></span>
        <span class="token string">"console_scripts"</span><span class="token punctuation">:</span> <span class="token punctuation">[</span>
            <span class="token string">"myapp=myapp.main:main"</span><span class="token punctuation">,</span>
        <span class="token punctuation">]</span><span class="token punctuation">,</span>
    <span class="token punctuation">}</span><span class="token punctuation">,</span>
<span class="token punctuation">)</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="3__stdeb__deb__45">
     </a>
     3. 使用
     <code>
      stdeb
     </code>
     生成
     <code>
      .deb
     </code>
     包
    </h4>
    <ul>
     <li>
      <strong>
       安装
       <code>
        stdeb
       </code>
      </strong>
      ：
      <code>
       stdeb
      </code>
      是一个用于将 Python 包转换为 Debian 包的工具。
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> python3-stdeb
</code></pre>
     </li>
     <li>
      <strong>
       生成
       <code>
        .deb
       </code>
       包
      </strong>
      ：在项目根目录下运行以下命令，生成
      <code>
       .deb
      </code>
      包：
      <pre><code class="prism language-bash">python3 setup.py --command-packages<span class="token operator">=</span>stdeb.command bdist_deb
</code></pre>
      这将生成一个
      <code>
       deb
      </code>
      文件，位于
      <code>
       dist
      </code>
      目录下。
     </li>
    </ul>
    <h4>
     <a id="4__56">
     </a>
     4. 打包和分发
    </h4>
    <ul>
     <li>
      <strong>
       测试
       <code>
        .deb
       </code>
       包
      </strong>
      ：在其他麒麟系统或兼容的 Debian 系统上测试生成的
      <code>
       .deb
      </code>
      包，确保其能够正确安装和运行。
     </li>
     <li>
      <strong>
       分发
       <code>
        .deb
       </code>
       包
      </strong>
      ：将生成的
      <code>
       .deb
      </code>
      包分发给用户，用户可以通过以下命令进行安装：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> dpkg <span class="token parameter variable">-i</span> myapp.deb
</code></pre>
     </li>
    </ul>
