    <p>
     Elasticsearch（简称 ES）提供了多种预置的分词器（Analyzer），用于对文本进行分词处理。分词器通常由字符过滤器（Character Filters）、分词器（Tokenizer）和词元过滤器（Token Filters）组成。以下是一些常用的预置分词器及其示例：
    </p>
    <hr/>
    <h4>
     <a id="1_Standard_Analyzer_4">
     </a>
     1.
     <strong>
      Standard Analyzer（标准分词器）
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       默认分词器
      </strong>
      ，适用于大多数语言。
     </li>
     <li>
      处理步骤：
      <ol>
       <li>
        使用标准分词器（Standard Tokenizer）按空格和标点符号分词。
       </li>
       <li>
        应用小写过滤器（Lowercase Token Filter）将词元转换为小写。
       </li>
      </ol>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"standard"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog's"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2_Simple_Analyzer_24">
     </a>
     2.
     <strong>
      Simple Analyzer（简单分词器）
     </strong>
    </h4>
    <ul>
     <li>
      按非字母字符（如数字、标点符号）分词，并将词元转换为小写。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"simple"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog"</span><span class="token punctuation">,</span> <span class="token string">"s"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3_Whitespace_Analyzer_41">
     </a>
     3.
     <strong>
      Whitespace Analyzer（空格分词器）
     </strong>
    </h4>
    <ul>
     <li>
      仅按空格分词，不转换大小写，不处理标点符号。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"whitespace"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"The"</span><span class="token punctuation">,</span> <span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"QUICK"</span><span class="token punctuation">,</span> <span class="token string">"Brown-Foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog's"</span><span class="token punctuation">,</span> <span class="token string">"bone."</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4_Keyword_Analyzer_58">
     </a>
     4.
     <strong>
      Keyword Analyzer（关键词分词器）
     </strong>
    </h4>
    <ul>
     <li>
      将整个文本作为一个单独的词元，不做任何分词处理。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"keyword"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="5_Stop_Analyzer_75">
     </a>
     5.
     <strong>
      Stop Analyzer（停用词分词器）
     </strong>
    </h4>
    <ul>
     <li>
      类似于简单分词器，但会过滤掉常见的停用词（如 “the”, “and”, “a” 等）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"stop"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog"</span><span class="token punctuation">,</span> <span class="token string">"s"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="6_Pattern_Analyzer_92">
     </a>
     6.
     <strong>
      Pattern Analyzer（正则分词器）
     </strong>
    </h4>
    <ul>
     <li>
      使用正则表达式定义分词规则。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"pattern"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      默认按非字母字符分词，并转换为小写：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog"</span><span class="token punctuation">,</span> <span class="token string">"s"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="7_Language_Analyzer_109">
     </a>
     7.
     <strong>
      Language Analyzer（语言分词器）
     </strong>
    </h4>
    <ul>
     <li>
      针对特定语言优化，支持多种语言（如英语、中文、法语等）。
     </li>
     <li>
      <strong>
       示例（英语）
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"english"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"fox"</span><span class="token punctuation">,</span> <span class="token string">"jump"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"lazi"</span><span class="token punctuation">,</span> <span class="token string">"dog"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="8_ICU_Analyzer_126">
     </a>
     8.
     <strong>
      ICU Analyzer（国际化分词器）
     </strong>
    </h4>
    <ul>
     <li>
      基于 ICU（International Components for Unicode）库，支持多语言分词。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"icu_analyzer"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"dog's"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="9_Fingerprint_Analyzer_143">
     </a>
     9.
     <strong>
      Fingerprint Analyzer（指纹分词器）
     </strong>
    </h4>
    <ul>
     <li>
      对文本进行分词、去重、排序，并生成唯一的“指纹”。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token constant">POST</span> _analyze
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"analyzer"</span><span class="token operator">:</span> <span class="token string">"fingerprint"</span><span class="token punctuation">,</span>
  <span class="token string-property property">"text"</span><span class="token operator">:</span> <span class="token string">"The 2 QUICK Brown-Foxes jumped over the lazy dog's bone."</span>
<span class="token punctuation">}</span>
</code></pre>
      <strong>
       输出
      </strong>
      ：
      <pre><code class="prism language-json"><span class="token punctuation">[</span><span class="token string">"2"</span><span class="token punctuation">,</span> <span class="token string">"bone"</span><span class="token punctuation">,</span> <span class="token string">"brown"</span><span class="token punctuation">,</span> <span class="token string">"dog"</span><span class="token punctuation">,</span> <span class="token string">"foxes"</span><span class="token punctuation">,</span> <span class="token string">"jumped"</span><span class="token punctuation">,</span> <span class="token string">"lazy"</span><span class="token punctuation">,</span> <span class="token string">"over"</span><span class="token punctuation">,</span> <span class="token string">"quick"</span><span class="token punctuation">,</span> <span class="token string">"the"</span><span class="token punctuation">]</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_160">
     </a>
     总结
    </h4>
    <p>
     Elasticsearch 的预置分词器适用于不同的场景，开发者可以根据需求选择合适的分析器，或者自定义分词器以满足特定需求。
    </p>
