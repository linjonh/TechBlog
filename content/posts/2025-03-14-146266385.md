    <p>
     <code>
      axis=0
     </code>
     和
     <code>
      axis=1
     </code>
     是用来指定操作的维度或轴的参数。
    </p>
    <h5>
     <a id="_1">
     </a>
     记忆方法
    </h5>
    <ul>
     <li>
      <strong>
       <code>
        axis=0
       </code>
      </strong>
      ：想象一个二维数组，沿着列方向（垂直方向）操作。
     </li>
     <li>
      <strong>
       <code>
        axis=1
       </code>
      </strong>
      ：想象一个二维数组，沿着行方向（水平方向）操作。
     </li>
    </ul>
    <h4>
     <a id="1_NumPy__axis__4">
     </a>
     1.
     <strong>
      NumPy 中的
      <code>
       axis
      </code>
      参数
     </strong>
    </h4>
    <p>
     在 NumPy 中，
     <code>
      axis
     </code>
     参数用于指定操作的维度。对于二维数组（矩阵），
     <code>
      axis=0
     </code>
     表示沿着列操作，而
     <code>
      axis=1
     </code>
     表示沿着行操作。
    </p>
    <h5>
     <a id="eg_7">
     </a>
     eg：
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">import</span> numpy <span class="token keyword">as</span> np

<span class="token comment"># 创建一个 3x4 的二维数组</span>

</code></pre>
    <h5>
     <a id="_25">
     </a>
     解释
    </h5>
    <ul>
     <li>
      <strong>
       <code>
        axis=0
       </code>
      </strong>
      ：沿着列操作，即对每一列进行操作。例如，
      <code>
       np.sum(array, axis=0)
      </code>
      会对每一列的元素求和，结果是一个包含每列和的数组。
     </li>
     <li>
      <strong>
       <code>
        axis=1
       </code>
      </strong>
      ：沿着行操作，即对每一行进行操作。例如，
      <code>
       np.sum(array, axis=1)
      </code>
      会对每一行的元素求和，结果是一个包含每行和的数组。
     </li>
    </ul>
    <h4>
     <a id="2_Pandas__axis__29">
     </a>
     2.
     <strong>
      Pandas 中的
      <code>
       axis
      </code>
      参数
     </strong>
    </h4>
    <p>
     在 Pandas 中，
     <code>
      axis
     </code>
     参数用于指定操作的维度。对于 DataFrame，
     <code>
      axis=0
     </code>
     表示沿着行操作（即对每一列进行操作），而
     <code>
      axis=1
     </code>
     表示沿着列操作（即对每一行进行操作）。
    </p>

    <h5>
     <a id="_55">
     </a>
     解释
    </h5>
    <ul>
     <li>
      <strong>
       <code>
        axis=0
       </code>
      </strong>
      ：沿着行操作，即对每一列进行操作。例如，
      <code>
       df.sum(axis=0)
      </code>
      会对每一列的元素求和，结果是一个包含每列和的 Series。
     </li>
     <li>
      <strong>
       <code>
        axis=1
       </code>
      </strong>
      ：沿着列操作，即对每一行进行操作。例如，
      <code>
       df.sum(axis=1)
      </code>
      会对每一行的元素求和，结果是一个包含每行和的 Series。
     </li>
    </ul>
    <h4>
     <a id="3__59">
     </a>
     3.
     <strong>
      总结
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        axis=0
       </code>
      </strong>
      ：沿着列操作，对每一列进行操作（在 NumPy 中是沿着列，在 Pandas 中是沿着行）。
     </li>
     <li>
      <strong>
       <code>
        axis=1
       </code>
      </strong>
      ：沿着行操作，对每一行进行操作（在 NumPy 中是沿着行，在 Pandas 中是沿着列）。
     </li>
    </ul>
