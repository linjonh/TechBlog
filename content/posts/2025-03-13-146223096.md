    <p>
     在STM32中配置定时器以产生1毫秒中断的步骤如下：
    </p>
    <h4>
     <a id="1__2">
     </a>
     1. 确定定时器时钟频率
    </h4>
    <p>
     假设系统主频为72MHz，定时器挂载在APB1总线（如TIM2），且APB1预分频系数为1，则定时器时钟为72MHz。
    </p>
    <h4>
     <a id="2_ARR_5">
     </a>
     2. 计算预分频器和自动重载值（ARR）
    </h4>
    <ul>
     <li>
      <strong>
       目标周期
      </strong>
      ：1ms = 0.001秒
     </li>
     <li>
      <strong>
       定时器时钟
      </strong>
      ：72MHz → 72,000,000 Hz
     </li>
     <li>
      <strong>
       计数值
      </strong>
      ：72,000,000 Hz × 0.001s = 72,000
     </li>
     <li>
      <strong>
       分频与ARR设置
      </strong>
      ：
      <ul>
       <li>
        预分频器（Prescaler）：71 → 分频后时钟 = 72MHz / (71+1) = 1MHz（每微秒计数一次）
       </li>
       <li>
        ARR：999 → 计数1000次（0到999）后溢出，时间 = 1000 × 1µs = 1ms
       </li>
      </ul>
     </li>
    </ul>
