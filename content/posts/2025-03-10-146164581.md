# 四、子串——10. 和为 K 的子数组

## 题目描述

给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。

子数组是数组中元素的连续非空序列。

## 示例

### 示例1：

> 输入：nums = [1,1,1], k = 2
>   
> 输出：2

### 示例2：

> 输入：nums = [1,2,3], k = 3
>   
> 输出：2

## 思路

双重for循环，暴力法。

## 代码

```java
class Solution {
        public int subarraySum(int[] nums, int k) {
            int sum =0;
            int ans = 0;
            for (int i = 0; i < nums.length; i++) {
                for (int j = i; j < nums.length; j++) {
                    sum+=nums[j];
                    if (sum == k){
                        ans++;
                    }
                }
                sum = 0;
            }
            return ans;
        }
}
```
