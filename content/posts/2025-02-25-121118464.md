# LSGO软件技术团队招新 && 线下组队学习

![](https://i-blog.csdnimg.cn/blog_migrate/a1de0c22f3af65459e190aa126f17baa.jpeg)

## 团队招新

LSGO软件技术团队（Dreamtech算法组）成立于2010年09月，团队主要从事地理信息系统、管理信息系统、计算机视觉等领域的应用开发，团队同时具有培养学生的重要职能，毕业学生分布在IBM、百度、阿里、腾讯、京东、华为、小米、今日头条、链家网、拼多多等IT企业。

欢迎认可我们且达到条件的同学加入（教十一D406），我们结伴学习，分享知识，共同成长。

**条件**
：

* 至少完成一次Dreamtech联合Datawhale组织的线上组队学习活动并取得证书。
* 至少完成一次Kaggle、讯飞、天池、和鲸或CCF等组织的数据科学比赛。
* 至少完成一次面向线下组队学习群的自己参加比赛的技术类分享。

你也可以通过以下二维码加入到
**线下组队学习群**
，参加我们组织的活动，比如：每周知识分享、大学生创新创业项目、蓝桥杯比赛、数学建模比赛、阿里天池比赛、线上组队学习、论文分享等等，以便大家有个相互了解（
已经进入其它“线下组队学习群”的伙伴不需要重复进入。
）。

![](https://i-blog.csdnimg.cn/blog_migrate/8df45e9b6f179869a284e9e7129df344.png)

## 学习路线

**数据科学方向**
：

* Python 编程语言
* Python 数据可视化
* Python 爬虫技术
* Numpy 科学计算
* Pandas 数据分析
* Sklearn 机器学习
* Pytorch 深度学习
* 天池比赛、创新实验项目、数学建模比赛

**算法方向**
：

* Python 编程语言
* C# 编程语言
* C++ 编程语言
* 设计模式
* 算法设计
* Leetcode刷题、每周周赛、蓝桥杯比赛、ACM比赛

## 成员要求

进入Dreamtech算法组的同学，必须做到以下两点，否则视为退出。

* 每个月必须参与一门Dreamtech联合Datawhale组织的线上组队学习活动。
* 每个月必须面向线下组队学习群进行一场技术分享活动，以检验自己这段时间的学习的效果。
