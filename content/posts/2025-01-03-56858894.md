     本文档将带你一步步创建完成一个微信小程序，并可以在手机上体验该小程序的实际效果。这个小程序的首页将会显示欢迎语以及当前用户的微信头像，点击头像，可以在新开的页面中查看当前小程序的启动日志
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
     1. 获取微信小程序的 AppID
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
     如果你是收邀请的开发者，我们会提供一个帐号，利用提供的帐号，登录 https://mp.weixin.qq.com ，就可以在网站的“设置”-“开发者设置”中，查看到微信小程序的 AppID 了，注意不可直接使用服务号或订阅号的 AppID 。
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
     如果是游客模式，可以跳过本步骤
    </p>
    <img alt="" src="">
     <br/>
     <p>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      注意：如果我们不是用注册时绑定的管理员微信号，在手机上体验该小程序。那么我们还需要操作“绑定开发者”。即在“用户身份”-“开发者”模块，绑定上需要体验该小程序的微信号。本教程默认注册帐号、体验都是使用管理员微信号。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      2. 创建项目
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      我们需要通过开发者工具，来完成小程序创建和代码编辑。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      开发者工具安装完成后，打开并使用微信扫码登录。选择创建“项目”，填入上文获取到的 AppID ，设置一个本地项目的名称（非小程序名称），比如“我的第一个项目”，并选择一个本地的文件夹作为代码存储的目录，点击“新建项目”就可以了。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      为方便初学者了解微信小程序的基本代码结构，在创建过程中，如果选择的本地文件夹是个空文件夹，开发者工具会提示，是否需要创建一个 quick start 项目。选择“是”，开发者工具会帮助我们在开发目录里生成一个简单的 demo。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      <img alt="" src="">
       <br/>
      </img>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      项目创建成功后，我们就可以点击该项目，进入并看到完整的开发者工具界面，点击左侧导航，在“编辑”里可以查看和编辑我们的代码，在“调试”里可以测试代码并模拟小程序在微信客户端效果，在“项目”里可以发送到手机里预览实际效果。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      3. 编写代码
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      创建小程序实例
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      点击开发者工具左侧导航的“编辑”，我们可以看到这个项目，已经初始化并包含了一些简单的代码文件。最关键也是必不可少的，是 app.js、app.json、app.wxss 这三个。其中，.js后缀的是脚本文件，.json后缀的文件是配置文件，.wxss后缀的是样式表文件。微信小程序会读取这些文件，并生成
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       小程序实例
      </span>
      。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      下面我们简单了解这三个文件的功能，方便修改以及从头开发自己的微信小程序。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      app.js是小程序的脚本代码。我们可以在这个文件中监听并处理小程序的生命周期函数、声明全局变量。调用框架提供的丰富的 API，如本例的同步存储及同步读取本地数据。想了解更多可用 API，可参考
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       API 文档
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      //app.jsApp({ onLaunch: function () { //调用API从本地缓存中获取数据var logs = wx.getStorageSync('logs') || [] logs.unshift(Date.now()) wx.setStorageSync('logs', logs) }, getUserInfo:function(cb){ var that = this; if(this.globalData.userInfo){ typeof cb == "function" &amp;&amp; cb(this.globalData.userInfo) }else{ //调用登录接口 wx.login({ success: function () { wx.getUserInfo({ success: function (res) { that.globalData.userInfo = res.userInfo; typeof cb == "function" &amp;&amp; cb(that.globalData.userInfo) } }) } }); } }, globalData:{ userInfo:null } })
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      app.json 是对整个小程序的全局配置。我们可以在这个文件中配置小程序是由哪些页面组成，配置小程序的窗口背景色，配置导航条样式，配置默认标题。注意该文件不可添加任何注释。更多可配置项可参考
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       配置详解
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      { "pages":[ "pages/index/index", "pages/logs/logs" ], "window":{ "backgroundTextStyle":"light", "navigationBarBackgroundColor": "#fff", "navigationBarTitleText": "WeChat", "navigationBarTextStyle":"black" } }
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      app.wxss 是整个小程序的公共样式表。我们可以在页面组件的 class 属性上直接使用 app.wxss 中声明的样式规则。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      /**app.wxss**/.container { height: 100%; display: flex; flex-direction: column; align-items: center; justify-content: space-between; padding: 200rpx 0; box-sizing: border-box; }
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      创建页面
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      在这个教程里，我们有两个页面，index 页面和 logs 页面，即欢迎页和小程序启动日志的展示页，他们都在 pages 目录下。微信小程序中的每一个页面的【路径+页面名】都需要写在 app.json 的 pages 中，且 pages 中的第一个页面是小程序的首页。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      每一个
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       小程序页面
      </span>
      是由同路径下同名的四个不同后缀文件的组成，如：index.js、index.wxml、index.wxss、index.json。.js后缀的文件是脚本文件，.json后缀的文件是配置文件，.wxss后缀的是样式表文件，.wxml后缀的文件是页面结构文件。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      index.wxml 是页面的结构文件：
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      &lt;!--index.wxml--&gt; &lt;view class="container"&gt; &lt;view bindtap="bindViewTap" class="userinfo"&gt; &lt;image class="userinfo-avatar" src="{
      <!-- -->
      {userInfo.avatarUrl}}" background-size="cover"&gt;&lt;/image&gt; &lt;text class="userinfo-nickname"&gt;{
      <!-- -->
      {userInfo.nickName}}&lt;/text&gt; &lt;/view&gt; &lt;view class="usermotto"&gt; &lt;text class="user-motto"&gt;{
      <!-- -->
      {motto}}&lt;/text&gt; &lt;/view&gt; &lt;/view&gt;
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      本例中使用了
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       &lt;view/&gt;
      </span>
      、
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       &lt;image/&gt;
      </span>
      、
      <span style="margin:0px; padding:0px; max-width:100%; word-wrap:break-word!important; list-style-type:none; border:0px; display:inline-block">
       &lt;text/&gt;
      </span>
      来搭建页面结构，绑定数据和交互处理函数。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      index.js 是页面的脚本文件，在这个文件中我们可以监听并处理页面的生命周期函数、获取小程序实例，声明并处理数据，响应页面交互事件等。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      //index.js//获取应用实例var app = getApp() Page({ data: { motto: 'Hello World', userInfo: {} }, //事件处理函数 bindViewTap: function() { wx.navigateTo({ url: '../logs/logs' }) }, : function () { console.log('') var that = this//调用应用实例的方法获取全局数据 app.getUserInfo(function(userInfo){ //更新数据 that.setData({ userInfo:userInfo }) }) } })
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:10px; padding-bottom:20px; max-width:100%; clear:both; min-height:1em; font-size:16px; list-style-type:none; text-indent:2em; border:0px; color:rgb(25,25,25); font-family:'PingFang SC',Arial,微软雅黑,宋体,simsun,sans-serif; line-height:30px; word-wrap:break-word!important">
      index.wxss 是页面的样式表：
     </p>
