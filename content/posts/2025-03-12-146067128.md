    <h3 name="1.%20%E5%BC%95%E8%A8%80">
     <span style="color:#ed7976">
      1. 引言
     </span>
    </h3>
    <p>
     <code>
      Image
     </code>
     组件是 Flutter 中用于显示图片的核心组件，它支持多种图片来源（本地、网络、资源文件等），并提供丰富的调整选项，如缩放、适应模式、缓存等。本文将详细介绍
     <code>
      Image
     </code>
     组件的使用方式及其重要参数。
    </p>
    <h3 id="2.%20%E5%8A%A0%E8%BD%BD%E5%9B%BE%E7%89%87%E7%9A%84%E6%96%B9%E5%BC%8F" name="2.%20%E5%8A%A0%E8%BD%BD%E5%9B%BE%E7%89%87%E7%9A%84%E6%96%B9%E5%BC%8F">
     <span style="color:#ed7976">
      2. 加载图片的方式
     </span>
    </h3>
    <p>
     Flutter
     <code>
      Image
     </code>
     组件支持多种方式加载图片：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        类型
       </th>
       <th>
        说明
       </th>
       <th>
        代码示例
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         Asset 资源
        </strong>
       </td>
       <td>
        加载本地资源图片
       </td>
       <td>
        <code>
         Image.asset('assets/images/logo.png')
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Network 网络
        </strong>
       </td>
       <td>
        加载远程图片
       </td>
       <td>
        <code>
         Image.network('https://example.com/img')
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         File 文件
        </strong>
       </td>
       <td>
        加载设备存储图片
       </td>
       <td>
        <code>
         Image.file(File('path/to/image.jpg'))
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Memory 内存
        </strong>
       </td>
       <td>
        加载二进制数据
       </td>
       <td>
        <code>
         Image.memory(bytes)
        </code>
       </td>
      </tr>
     </tbody>
    </table>
    <h4 id="2.1%C2%A0%E6%9C%AC%E5%9C%B0%E5%9B%BE%E7%89%87" name="2.1%C2%A0%E6%9C%AC%E5%9C%B0%E5%9B%BE%E7%89%87">
     <span style="color:#956fe7">
      2.1 本地图片
     </span>
    </h4>
    <p>
     首先，需要在
     <code>
      pubspec.yaml
     </code>
     文件中添加资源路径：
    </p>
    <pre><code class="language-Dart">flutter:
  assets:
    - assets/images/ic_input_et.png</code></pre>
    <p>
     然后，在代码中使用：
    </p>
    <pre><code class="language-Dart">Image.asset(
    'assets/images/ic_input_et.png',
    width: 100,          // 显式设置宽度
    height: 200,         // 显式设置高度
),</code></pre>
    <p class="img-center">
     <img alt="" height="192" src="https://i-blog.csdnimg.cn/direct/79c386a12bc34389a959eee1ecc322b3.png" width="500"/>
    </p>
    <h4 id="2.2%20%E7%BD%91%E7%BB%9C%E5%9B%BE%E7%89%87" name="2.2%20%E7%BD%91%E7%BB%9C%E5%9B%BE%E7%89%87">
     <span style="color:#956fe7">
      2.2 网络图片
     </span>
    </h4>
    <pre><code class="language-Dart">Image.network(
    'https://i-blog.csdnimg.cn/direct/943bcea21349401399ae61542b3e9f44.png',
    width: 500,          // 显式设置宽度
    height: 200,         // 显式设置高度
)</code></pre>
    <p class="img-center">
     <img alt="" height="112" src="https://i-blog.csdnimg.cn/direct/e7db1dd647c04d3b8a1e0e065c6dccdc.png" width="500"/>
    </p>
    <h4 id="2.3%20%E6%9C%AC%E5%9C%B0%E6%96%87%E4%BB%B6%E5%9B%BE%E7%89%87" name="2.3%20%E6%9C%AC%E5%9C%B0%E6%96%87%E4%BB%B6%E5%9B%BE%E7%89%87">
     <span style="color:#956fe7">
      2.3 本地文件图片
     </span>
    </h4>
    <p>
     适用于加载存储在设备本地的图片（此处需要权限，咱们后面统一写一下）：
    </p>
    <pre><code class="language-Dart">import 'dart:io';
Image.file(
    File('/storage/emulated/0/Download/ReBang1101.jpg'),
    width: 200,          // 显式设置宽度
    height: 100,         // 显式设置高度
),
</code></pre>
    <h4 id="2.4%20%E5%86%85%E5%AD%98%E5%9B%BE%E7%89%87" name="2.4%20%E5%86%85%E5%AD%98%E5%9B%BE%E7%89%87">
     <span style="color:#956fe7">
      2.4 内存图片
     </span>
    </h4>
    <p>
     适用于需要动态加载的图片数据，如相机捕获或网络流数据：
    </p>
    <pre><code class="language-Dart">Image.memory(uint8ListData);
</code></pre>
    <h3 id="3.%20fit%20%E5%8F%82%E6%95%B0%EF%BC%9A%E6%8E%A7%E5%88%B6%E5%9B%BE%E7%89%87%E9%80%82%E5%BA%94%E6%96%B9%E5%BC%8F" name="3.%20fit%20%E5%8F%82%E6%95%B0%EF%BC%9A%E6%8E%A7%E5%88%B6%E5%9B%BE%E7%89%87%E9%80%82%E5%BA%94%E6%96%B9%E5%BC%8F">
     <span style="color:#ed7976">
      3.
      <code>
       fit
      </code>
      参数：控制图片适应方式
     </span>
    </h3>
    <p>
     <code>
      BoxFit
     </code>
     枚举控制图片如何适应
     <code>
      Image
     </code>
     组件的边界。
    </p>
    <table>
     <thead>
      <tr>
       <th>
        <code>
         BoxFit
        </code>
        值
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         fill
        </code>
       </td>
       <td>
        拉伸填充整个容器，可能变形
       </td>
      </tr>
      <tr>
       <td>
        <code>
         contain
        </code>
       </td>
       <td>
        适应容器，保持原比例，可能留白
       </td>
      </tr>
      <tr>
       <td>
        <code>
         cover
        </code>
       </td>
       <td>
        填充容器，可能裁剪
       </td>
      </tr>
      <tr>
       <td>
        <code>
         fitWidth
        </code>
       </td>
       <td>
        宽度填满，高度按比例缩放
       </td>
      </tr>
      <tr>
       <td>
        <code>
         fitHeight
        </code>
       </td>
       <td>
        高度填满，宽度按比例缩放
       </td>
      </tr>
      <tr>
       <td>
        <code>
         none
        </code>
       </td>
       <td>
        原始大小显示，不缩放
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     示例：
    </p>
    <pre><code>Image.network(
  'https://example.com/sample.jpg',
  fit: BoxFit.cover,
)
</code></pre>
    <h3 id="4.%20%E9%AB%98%E7%BA%A7%E5%BA%94%E7%94%A8%E6%8A%80%E5%B7%A7" name="4.%20%E9%AB%98%E7%BA%A7%E5%BA%94%E7%94%A8%E6%8A%80%E5%B7%A7" style="background-color:transparent">
     <span style="color:#ed7976">
      4. 高级应用技巧
     </span>
    </h3>
    <h4 id="4.1%C2%A0%E5%8D%A0%E4%BD%8D%E7%AC%A6%E4%B8%8E%E6%B7%A1%E5%85%A5%E6%95%88%E6%9E%9C" name="4.1%C2%A0%E5%8D%A0%E4%BD%8D%E7%AC%A6%E4%B8%8E%E6%B7%A1%E5%85%A5%E6%95%88%E6%9E%9C">
     <span style="color:#956fe7">
      4.1 占位符与淡入效果
     </span>
    </h4>
    <pre><code class="language-Dart">FadeInImage.assetNetwork(
  placeholder: 'assets/loading.gif',  // 占位图
  image: 'https://example.com/photo.jpg', // 实际图片
  fadeInDuration: Duration(milliseconds: 500), // 淡入时长
  fit: BoxFit.cover,
)</code></pre>
    <h4 id="4.2%C2%A0%E5%9B%BE%E7%89%87%E7%BC%93%E5%AD%98%E7%AE%A1%E7%90%86" name="4.2%C2%A0%E5%9B%BE%E7%89%87%E7%BC%93%E5%AD%98%E7%AE%A1%E7%90%86">
     <span style="color:#956fe7">
      4.2 图片缓存管理
     </span>
    </h4>
    <p>
     Flutter 默认对网络图片进行缓存，可以使用
     <code>
      cached_network_image
     </code>
     插件优化加载性能。
    </p>
    <p>
     <strong>
      安装插件
     </strong>
     ：
    </p>
    <pre><code class="language-Dart">dependencies:
  cached_network_image: ^3.4.1
</code></pre>
    <p>
     <strong>
      使用示例
     </strong>
     ：
    </p>
    <pre><code class="language-Dart">CachedNetworkImage(
  imageUrl: 'https://i-blog.csdnimg.cn/direct/943bcea21349401399ae61542b3e9f44.png',
  placeholder: (context, url) =&gt; LoadingSpinner(),
  errorWidget: (context, url, error) =&gt; Icon(Icons.error),
  memCacheHeight: 400,  // 内存缓存高度
  maxWidthDiskCache: 1024, // 磁盘缓存最大宽度
)</code></pre>
    <h4 id="4.3%C2%A0%E5%9B%BE%E7%89%87%E8%A3%81%E5%89%AA%E4%B8%8E%E6%BB%A4%E9%95%9C" name="4.3%C2%A0%E5%9B%BE%E7%89%87%E8%A3%81%E5%89%AA%E4%B8%8E%E6%BB%A4%E9%95%9C">
     <span style="color:#956fe7">
      4.3 图片裁剪与滤镜
     </span>
    </h4>
    <p>
    </p>
    <pre><code class="language-Dart">ClipRRect( // 圆角裁剪
  borderRadius: BorderRadius.circular(16),
  child: Image.asset('assets/product.jpg'),
),

ShaderMask( // 添加渐变蒙版
  shaderCallback: (rect) =&gt; LinearGradient(
    colors: [Colors.black, Colors.transparent]
  ).createShader(rect),
  child: Image.network('https://example.com/banner.jpg'),
),

ImageFiltered( // 高斯模糊
  imageFilter: ImageFilter.blur(sigmaX: 2, sigmaY: 2),
  child: Image.file(File('path/to/image.jpg')),
)</code></pre>
    <pre></pre>
    <h3 id="5.%20%E6%80%A7%E8%83%BD%E4%BC%98%E5%8C%96%E6%8C%87%E5%8D%97" name="5.%20%E6%80%A7%E8%83%BD%E4%BC%98%E5%8C%96%E6%8C%87%E5%8D%97" style="background-color:transparent">
     <span style="color:#ed7976">
      5. 性能优化指南
     </span>
    </h3>
    <h4 id="5.1%C2%A0%E8%B5%84%E6%BA%90%E5%9B%BE%E7%89%87%E8%A7%84%E8%8C%83" name="5.1%C2%A0%E8%B5%84%E6%BA%90%E5%9B%BE%E7%89%87%E8%A7%84%E8%8C%83">
     <span style="color:#956fe7">
      5.1 资源图片规范
     </span>
    </h4>
    <p>
     使用
     <code>
      2.0x
     </code>
     、
     <code>
      3.0x
     </code>
     目录实现多分辨率适配。
    </p>
    <pre><code class="language-Dart"># pubspec.yaml 配置示例
flutter:
  assets:
    - assets/images/2.0x/ic_input_et.png  # 适配 @2x 屏幕
    - assets/images/3.0x/ic_input_et.png  # 适配 @3x 屏幕</code></pre>
    <h4 id="2.%20%E5%A4%A7%E5%9B%BE%E5%8A%A0%E8%BD%BD%E7%AD%96%E7%95%A5" name="2.%20%E5%A4%A7%E5%9B%BE%E5%8A%A0%E8%BD%BD%E7%AD%96%E7%95%A5">
     <span style="color:#956fe7">
      2. 大图加载策略
     </span>
    </h4>
    <pre><code class="language-Dart">Image.asset(
  'assets/large_map.jpg',
  cacheWidth: 800,   // 指定解码尺寸
  cacheHeight: 600,
  isAntiAlias: true, // 开启抗锯齿
)</code></pre>
    <h4 id="5.3%C2%A0%E5%86%85%E5%AD%98%E7%AE%A1%E7%90%86%E6%96%B9%E6%A1%88" name="5.3%C2%A0%E5%86%85%E5%AD%98%E7%AE%A1%E7%90%86%E6%96%B9%E6%A1%88">
     <span style="color:#956fe7">
      5.3 内存管理方案
     </span>
    </h4>
    <pre><code class="language-Dart">// 使用 ResizeImage 限制解码尺寸
ResizeImage.resizeIfNeeded(
  width: 400,
  height: 300,
  imageProvider: AssetImage('assets/high_res.jpg'),
)</code></pre>

     <span style="color:#ed7976">
      6. 最佳实践总结
     </span>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        资源管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         使用
         <code>
          2.0x
         </code>
         、
         <code>
          3.0x
         </code>
         目录实现多分辨率适配
        </p>
       </li>
       <li>
        <p>
         优先选择 WebP 格式减小包体积
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        网络图片
       </strong>
      </p>
      <ul>
       <li>
        <p>
         必须处理加载错误和超时情况
        </p>
       </li>
       <li>
        <p>
         推荐使用 CDN 加速并开启 HTTPS
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        性能优化
       </strong>
      </p>
      <ul>
       <li>
        <p>

         <code>
          cacheWidth/cacheHeight
         </code>
         限制解码尺寸
        </p>

       <li>
        <p>
         列表视图采用
         <code>
          ListView.builder
         </code>
         懒加载
        </p>

     <li>
      <p>
       <strong>
        内存安全
       </strong>
      </p>

        </p>
       </li>
       <li>
        <p>
         监控内存使用量（可通过 DevTools）
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        高级效果
       </strong>
      </p>

        <p>
         组合使用

          ShaderMask
         </code>

         <code>
          ClipPath
         </code>
         实现复杂形状
        </p>

       <li>
        <p>
         通过
         <code>
          ImageFilter

       </li>
      </ul>
     </li>
