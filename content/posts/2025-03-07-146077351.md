    <p>
     1.对于右移操作符来说，有两种右移方式：
    </p>
    <p>
     （1）算术右移：右边丢弃，左边补原符号位；
    </p>
    <p>
     （2）逻辑右移：右边丢弃，左边补0；
    </p>
    <p>
     2.
    </p>
    <p>
     （1）&amp;（按位与）操作符：
    </p>
    <p>
     这里的按位是按二进制位，如：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int a = 3;  //00000000000000000000000000000011
     <br/>
     int b = 5;  //00000000000000000000000000000101
     <br/>
     int c = a &amp; b;  //00000000000000000000000000000001
     <br/>
     printf("%d\n", c);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     输出结果为1；这里的规则是，两个数都为真，c才为真；
    </p>
    <p>
     （2）|（按位或）：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int a = 3;  //00000000000000000000000000000011
     <br/>
     int b = 5;  //00000000000000000000000000000101
     <br/>
     int c = a | b;  //00000000000000000000000000000111
     <br/>
     printf("%d\n", c);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     只要两个数的二进制位上其中有一个为真，c就为真；
    </p>
    <p>
     （3）^(按位异或）：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int a = 3;      //00000000000000000000000000000011
     <br/>
     int b = 5;      //00000000000000000000000000000101
     <br/>
     int c = a ^ b;  //00000000000000000000000000000110
     <br/>
     printf("%d\n", c);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     两个数的相应二进制位如果相同，则为假，不相同为真；
    </p>
    <p>
     <img alt="" height="72" src="https://i-blog.csdnimg.cn/direct/b512e6a57b0b4000b01eeb5ddcf2929a.png" width="632"/>
    </p>
    <p>
     <span style="color:#fe2c24">
      奇技淫巧：a^a = 0   a^0 = a
     </span>
    </p>
    <p>
     <span style="color:#0d0016">

      <br/>
      }

     <br/>

    <p>
     5.sizeof操作符（注意不是函数）：用于计算对象所占空间的大小，单位是字节；
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int a = 10;
     <br/>
     int arr[10] = {0};
     <br/>
     printf("%d\n", sizeof(arr));
     <br/>
     printf("%d\n", sizeof a);
     <br/>
     printf("%d\n", sizeof(int [10]));
     <br/>
     return 0;
     <br/>
     }
    </p>
    <p>
     <img alt="" height="374" src="https://i-blog.csdnimg.cn/direct/8bb786600a2f4080aee9446ba1e8ca45.png" width="916"/>
    </p>
    <p>
     sizeof后可以不加（），int [10]代表的是数组类型，相当于sizeof(int),定义变量时去掉变量名就是变量类型的代表方式。
    </p>
    <p>
     例如：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     short s = 5;
     <br/>
     int a = 10;
     <br/>
     printf("%d\n", sizeof(s = a + 2));
     <br/>
     printf("%d\n", s);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     <span style="color:#fe2c24">
      注意：sizeof里面的表达式不进行计算，所以s的值还是5，这涉及到内部程序编译顺序的问题：
     </span>
    </p>
    <p>
     <span style="color:#fe2c24">
      首先，我们写的程序.c文件是源程序，它经过编译时，执行了sizeof指令，然后表达式就消失了，之后再进行链接，成为可执行程序.exe文件，这时进行执行的时候才进行表达式的计算。
     </span>
    </p>
    <p>
     <img alt="" height="159" src="https://i-blog.csdnimg.cn/direct/27a405873271475bac19edca626b095b.png" width="873"/>
    </p>
    <p>
     6.逻辑与（&amp;&amp;）和逻辑或（||）：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int i = 0, a = 0, b = 2, c = 3, d = 4;
     <br/>
     i = a++ &amp;&amp; ++b &amp;&amp; d++;
     <br/>
     printf("%d %d %d %d", a, b, c, d);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     输出结果为1,2,3,4；原因是逻辑与语句遇到第一个为假，则直接返回假值，不进行后面一系列运算，所以b、d的值不变；
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     int i = 0, a = 0, b = 2, c = 3, d = 4;
     <br/>
     i = a++ || ++b || d++;
     <br/>
     printf("%d %d %d %d", a, b, c, d);
     <br/>
     return 0;
     <br/>
     }
     <br/>
     结果为1,3,3,4；原因是逻辑或语句遇到真值时会立刻返回真值，不进行后续运算。
    </p>

    <p>
     9.隐式类型转换：
    </p>
    <p>
     c的整型算术运算总是至少以缺省整型类型的精度来进行的；
    </p>
    <p>
     为了获得这个精度，表达式中的字符和短整型操作数在使用之前被转换为普通整型，这种转换称为整型提升；
    </p>
    <p>
     （1） 整型提升的意义：
    </p>
    <p>
     表达式的整型运算要在CPU的相应运算器件内执行，CPU内整型运算器（ALU）的操作数的字节长度一般就是int的字节长度，同时也是CPU的通用寄存器的长度；
    </p>
    <p>
     因此，即使两个char类型的相加，在CPU执行时实际上也是要先转换为CPU内整型操作符的标准长度；
    </p>
    <p>
     通用CPU是难以直接实现两个8比特字节直接相加运算（虽然机器指令中可能有这种字节相加指令）。所以，表达式中各种长度可能小于int长度的整型值，都必须先转换成int或unsigned int，然后才能送入CPU去执行运算。
    </p>
    <p>
     <span style="color:#fe2c24">
      整型提升是按照变量的数据类型的符号位来提升的
     </span>
     。
    </p>
    <p>
     <img alt="" height="1310" src="https://i-blog.csdnimg.cn/direct/b9fadffcf96941699723382bda455fea.png" width="1021"/>
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     char a = 3;  //00000011  -&gt;   00000000000000000000000000000011
     <br/>
     char b = 127;//01111111  -&gt;   00000000000000000000000001111111
     <br/>
     char c = a + b;             //00000000000000000000000010000010  然后截断成10000010存到c中；
     <br/>
     printf("%d\n", c);         //输出时再进行整形提升：11111111111111111111111110000010
     <br/>
     return 0;                  //转换为原码10000000000000000000000001111101  结果为-126
     <br/>
     }
    </p>
    <p>
     发生整型提升的条件：
    </p>
    <p>
     （1）算术运算：+ - * / %
    </p>
    <p>
     （2）比较运算：&gt; &lt; &gt;= &lt;= == !=
    </p>
    <p>
     （3）位运算：&amp;   |   ^    &lt;&lt;   &gt;&gt;
    </p>
    <p>
     例如：
    </p>
    <p>
     #include &lt;stdio.h&gt;
     <br/>
     #include &lt;stdlib.h&gt;
    </p>
    <p>
     int main()
     <br/>
     {
     <!-- -->
     <br/>
     char c = 1;
     <br/>
     printf("%d\n", sizeof(c));
     <br/>
     printf("%d\n", sizeof(+c));
     <br/>
     printf("%d\n", sizeof(-c));
     <br/>
     return 0;
     <br/>
     }
    </p>
    <p>
     结果为1 4 4；
    </p>
    <p>
     （2）算术转换
    </p>
    <p>
     <img alt="" height="653" src="https://i-blog.csdnimg.cn/direct/173afb44143a42eaaba117e65695d80f.png" width="1195"/>

    <p>
     一般是精度低的向精度高的转换，或者内存小的向内存大的转换。
    </p>
