# # 如何查看 Ubuntu 版本？

## 如何查看 Ubuntu 版本？

要查看‌Ubuntu版本，你可以通过以下几种方法：

#### 1. 使用‌lsb_release 命令‌查看

使用 lsb_release -a 命令可以查看Ubuntu的详细版本信息，包括发行版ID、版本号以及版本代号。‌

‌### 2、查看 /etc/issue 文件‌：

通过 cat /etc/issue 命令可以查看系统的欢迎信息，其中通常包含操作系统的版本号。‌

‌### 3、查看 /proc/version 文件‌：

使用 cat /proc/version 命令可以查看当前系统运行的各种数据，其中包括Ubuntu版本信息。‌

‌### 4、其他命令查看ubuntu版本‌：

1)如uname -r：查看内核版本号。

2）hostnamectl：查看系统的hostname、操作系统、内核等信息，其中“Operating System”行会显示Ubuntu的版本。

3）cat /etc/os-release：查看操作系统识别数据，包括多行信息，其中“PRETTY_NAME”是一个友好的完整版本描述。

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/0de8a7df23d0445dbe9da2c88b661138.png)

#### 5、 使用图形界面查看ubuntu版本

‌依次打开：
  
系统设置（Settings），然后滚动到底部选择“关于”（About）或【详细信息】，在这里你可以看到Ubuntu的版本信息。‌

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/f78a768fdabf4094b7add98ec8d5443b.png)

#### 6、使用第三方工具Neofetch查看ubuntu版本

‌‌Neofetch‌：
  
如果你安装了neofetch（可通过sudo apt install neofetch安装），运行neofetch命令，它可以提供一个包含系统版本信息在内的丰富的系统信息概览。‌
