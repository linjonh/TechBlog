    <h4>
     一、Mythalion 13B 模型简介
    </h4>
    <p>
     Mythalion 13B 是由 PygmalionAI 与 Gryphe 合作开发的强大语言模型。它通过融合 Pygmalion-2 13B 和 MythoMax L2 13B 模型而成，旨在增强角色扮演（RP）和聊天场景中的表现。由于其基于 Llama-2 架构构建，Mythalion 13B 在处理文本生成、对话模拟、创意写作等任务中展现出优异的表现。
    </p>
    <h4>
     二、类似模型对比与介绍
    </h4>
    <p>
     除了 PygmalionAI/mythalion-13b，还有以下类似的模型：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        Pygmalion 7B/13B/30B 系列：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         专为角色扮演与对话生成优化。
        </p>
       </li>
       <li>
        <p>
         基于 Llama-2 和 LLaMA 架构。
        </p>
       </li>
       <li>
        <p>
         具有较好的对话连贯性和情景互动能力。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        MythoMax L2 13B：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         基于 Llama-2 13B 进行微调。
        </p>
       </li>
       <li>
        <p>
         强调长文本生成与创意写作。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        Llama-2 7B/13B/70B：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         Meta 发布的开源模型。
        </p>
       </li>
       <li>
        <p>
         具备较强的通用语言处理能力，但对中文的支持相对较弱。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        Mistral 7B：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         一个强大的开源模型，特别优化了架构以增强推理与生成性能。
        </p>
       </li>
       <li>
        <p>
         在小规模参数下表现优异。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        ChatGLM 6B/130B：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         专为中文优化的开源对话模型。
        </p>
       </li>
       <li>
        <p>
         支持中英文对话与创意写作。
        </p>
       </li>
       <li>
        <p>
         性能上适合需要中文支持的场景。
        </p>
       </li>
      </ul>
     </li>
    </ol>

    <h4>
     四、本地搭建指南
    </h4>
    <h5>
     1. 环境准备
    </h5>
    <ul>
     <li>
      <p>
       **Python 版本：**3.8 及以上。
      </p>
     </li>
     <li>
      <p>
       <strong>
        安装必要的 Python 库：
       </strong>
      </p>
     </li>
    </ul>
    <pre><code>pip install transformers torch accelerate</code></pre>
    <h5>
     2. 下载并加载模型
    </h5>
    <pre><code>from transformers import AutoTokenizer, AutoModelForCausalLM

tokenizer = AutoTokenizer.from_pretrained("PygmalionAI/mythalion-13b")
model = AutoModelForCausalLM.from_pretrained("PygmalionAI/mythalion-13b", device_map="auto")</code></pre>
    <h5>
     3. 与模型对话
    </h5>
    <pre><code>input_text = "Hello! How are you today?"
inputs = tokenizer(input_text, return_tensors="pt").to("cuda")  # 如果有GPU支持
outputs = model.generate(**inputs, max_length=100)
response = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(response)</code></pre>
    <h4>
     五、最低配置要求
    </h4>
    <h5>
     Windows / Linux / Mac (CPU 或 M1/M2)
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        操作系统：
       </strong>
       Windows 10 / Linux (Ubuntu 20.04+) / MacOS 12.0 及以上。
      </p>
     </li>
     <li>
      <p>
       <strong>
        CPU：
       </strong>
       至少 4 核心（推荐 8 核心以上）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        内存 (RAM)：
       </strong>
       16GB (建议 32GB 以获得更好的性能)。
      </p>
     </li>
     <li>
      <p>
       <strong>
        存储：
       </strong>
       至少 20GB 可用磁盘空间。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Python 版本：
       </strong>
       3.8 或更高。
      </p>
     </li>
     <li>
      <p>
       <strong>
        GPU (可选)：
       </strong>
      </p>
      <ul>
       <li>
        <p>
         Windows / Linux：NVIDIA GPU，支持 CUDA (如 RTX 3060 或更高)。
        </p>
       </li>
       <li>
        <p>
         Mac：M1 或 M2 (使用 MPS 加速)。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     六、优化建议
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        使用
       </strong>
       <code>
        <strong>
         torch.float16
        </strong>
       </code>
       <strong>
        或
       </strong>
       <code>
        <strong>
         torch.bfloat16
        </strong>
       </code>
       <strong>
        加载模型，减少显存占用。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        使用
       </strong>
       <code>
        <strong>
         load_in_8bit=True
        </strong>
       </code>
       <strong>
        或
       </strong>
       <code>
        <strong>
         load_in_4bit=True
        </strong>
       </code>
       <strong>
        提高内存效率。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        分布式加载 (如果在多 GPU 上运行)。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        模型微调：对中文数据集进行微调以增强中文支持。
       </strong>
      </p>
     </li>
    </ol>
