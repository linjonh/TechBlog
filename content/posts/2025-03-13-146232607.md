    <h4>
     <a id="Flex_6">
     </a>
     <strong>
      一、Flex布局核心概念
     </strong>
    </h4>
    <h5>
     <a id="1__7">
     </a>
     1.
     <strong>
      基本原理
     </strong>
    </h5>
    <p>
     •
     <strong>
      一维布局
     </strong>
     ：Flex布局专注于单行或单列的布局（通过
     <code>
      flex-direction
     </code>
     控制方向），适合处理行内或列内的弹性排列。
     <br/>
     •
     <strong>
      容器与项目
     </strong>
     ：
     <br/>
     •
     <strong>
      容器
     </strong>
     ：使用
     <code>
      display: flex
     </code>
     或
     <code>
      display: inline-flex
     </code>
     的元素。
     <br/>
     •
     <strong>
      项目
     </strong>
     ：容器内的子元素自动成为Flex项目。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/032f228556b047e6ba17d42e0a70e43f.png"/>
    </p>
    <h5>
     <a id="2__14">
     </a>
     2.
     <strong>
      主轴与交叉轴
     </strong>
    </h5>
    <p>
     •
     <strong>
      主轴
     </strong>
     （默认水平方向）：项目沿此轴排列。
     <br/>
     •
     <strong>
      交叉轴
     </strong>
     ：与主轴垂直的方向。
     <br/>
     •
     <strong>
      方向控制
     </strong>
     ：通过
     <code>
      flex-direction
     </code>
     切换主轴方向（如
     <code>
      row
     </code>
     、
     <code>
      column
     </code>
     、
     <code>
      row-reverse
     </code>
     ）。
    </p>
    <hr/>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c97e1a52706a49d6bf7b69ad3214f665.png"/>
    </p>
    <h4>
     <a id="Flex_22">
     </a>
     <strong>
      二、Flex容器关键属性
     </strong>
    </h4>
    <h5>
     <a id="1_flexdirection_23">
     </a>
     1.
     <strong>
      flex-direction
     </strong>
    </h5>
    <p>
     •
     <code>
      row
     </code>
     （默认）：水平左对齐 → 主轴为水平方向。
     <br/>
     •
     <code>
      row-reverse
     </code>
     ：水平右对齐。
     <br/>
     •
     <code>
      column
     </code>
     ：垂直上对齐 → 主轴为垂直方向。
     <br/>
     •
     <code>
      column-reverse
     </code>
     ：垂直下对齐。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">display</span><span class="token punctuation">:</span> flex<span class="token punctuation">;</span>
  <span class="token property">flex-direction</span><span class="token punctuation">:</span> column-reverse<span class="token punctuation">;</span> <span class="token comment">/* 垂直反向排列 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="2_flexwrap_36">
     </a>
     2.
     <strong>
      flex-wrap
     </strong>
    </h5>
    <p>
     •
     <code>
      nowrap
     </code>
     （默认）：不换行，溢出隐藏。
     <br/>
     •
     <code>
      wrap
     </code>
     ：换行，项目自动换到下一行/列。
     <br/>
     •
     <code>
      wrap-reverse
     </code>
     ：换行且反向排列。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">flex-wrap</span><span class="token punctuation">:</span> wrap<span class="token punctuation">;</span> <span class="token comment">/* 多列布局自动换行 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="3_justifycontent_47">
     </a>
     3.
     <strong>
      justify-content
     </strong>
    </h5>
    <p>
     • 主轴方向上的对齐方式：
     <br/>
     •
     <code>
      flex-start
     </code>
     ：靠左/上对齐（默认）。
     <br/>
     •
     <code>
      flex-end
     </code>
     ：靠右/下对齐。
     <br/>
     •
     <code>
      center
     </code>
     ：居中。
     <br/>
     •
     <code>
      space-between
     </code>
     ：元素间均匀分布，首尾留空。
     <br/>
     •
     <code>
      space-around
     </code>
     ：元素周围均匀分布（包括首尾）。
     <br/>
     •
     <code>
      space-evenly
     </code>
     ：元素间均匀分布，间距相等。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">justify-content</span><span class="token punctuation">:</span> space-between<span class="token punctuation">;</span> <span class="token comment">/* 水平均匀分布 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="4_alignitems_62">
     </a>
     4.
     <strong>
      align-items
     </strong>
    </h5>
    <p>
     • 交叉轴方向上的对齐方式（单行/单列时与
     <code>
      justify-content
     </code>
     效果相同）：
     <br/>
     •
     <code>
      stretch
     </code>
     （默认）：按比例拉伸填充空间。
     <br/>
     •
     <code>
      flex-start
     </code>
     ：靠左/上对齐。
     <br/>
     •
     <code>
      flex-end
     </code>
     ：靠右/下对齐。
     <br/>
     •
     <code>
      center
     </code>
     ：居中。
     <br/>
     •
     <code>
      baseline
     </code>
     ：基线对齐（文字底部对齐）。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">align-items</span><span class="token punctuation">:</span> center<span class="token punctuation">;</span> <span class="token comment">/* 垂直居中 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="5_aligncontent_76">
     </a>
     5.
     <strong>
      align-content
     </strong>
    </h5>
    <p>
     • 多行/多列时的交叉轴对齐（仅当
     <code>
      flex-wrap: wrap
     </code>
     时生效）：
     <br/>
     •
     <code>
      stretch
     </code>
     （默认）：行/列拉伸填充容器。
     <br/>
     •
     <code>
      flex-start
     </code>
     ：行/列顶部对齐。
     <br/>
     •
     <code>
      flex-end
     </code>
     ：行/列底部对齐。
     <br/>
     •
     <code>
      center
     </code>
     ：行/列居中。
     <br/>
     •
     <code>
      space-between
     </code>
     ：行/列间均匀分布。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">align-content</span><span class="token punctuation">:</span> space-between<span class="token punctuation">;</span> <span class="token comment">/* 多行垂直均匀分布 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/05435c47cfb94347b51e889547524cd4.png"/>

    <h4>
     <a id="Flex_93">
     </a>
     <strong>
      三、Flex项目关键属性
     </strong>
    </h4>
    <h5>
     <a id="1_flexgrow_94">
     </a>
     1.
     <strong>
      flex-grow
     </strong>
    </h5>
    <p>
     •
     <strong>
      定义
     </strong>
     ：项目在主轴方向上的扩展比例（默认
     <code>
      0
     </code>
     ）。
     <br/>
     •
     <strong>
      作用
     </strong>
     ：当剩余空间存在时，按比例分配额外空间。
     <br/>
     •
     <strong>
      示例
     </strong>
     ：
     <code>
      flex: 1
     </code>
     表示占用全部可用空间，
     <code>
      flex: 2
     </code>
     占据两倍空间。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.item</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">flex-grow</span><span class="token punctuation">:</span> 1<span class="token punctuation">;</span> <span class="token comment">/* 平均分配剩余空间 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="2_flexshrink_105">
     </a>
     2.
     <strong>
      flex-shrink
     </strong>
    </h5>
    <p>
     •
     <strong>
      定义
     </strong>
     ：项目在主轴方向上的收缩比例（默认
     <code>
      1
     </code>
     ）。
     <br/>
     •
     <strong>
      作用
     </strong>
     ：当空间不足时，按比例缩小项目。
     <br/>
     •
     <strong>
      示例
     </strong>
     ：
     <code>
      flex-shrink: 0
     </code>
     表示不缩小。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.item</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">flex-shrink</span><span class="token punctuation">:</span> 0<span class="token punctuation">;</span> <span class="token comment">/* 空间不足时不缩小 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="3_flexbasis_116">
     </a>
     3.
     <strong>
      flex-basis
     </strong>
    </h5>
    <p>
     •
     <strong>
      定义
     </strong>
     ：项目在主轴方向上的初始大小（默认
     <code>
      auto
     </code>
     ）。
     <br/>
     •
     <strong>
      作用
     </strong>
     ：作为伸缩的基准值。
     <br/>
     •
     <strong>
      示例
     </strong>
     ：
     <code>
      flex: 0 0 200px
     </code>
     →
     <code>
      flex-grow=0
     </code>
     ,
     <code>
      flex-shrink=0
     </code>
     ,
     <code>
      flex-basis=200px
     </code>
     。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.item</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">flex-basis</span><span class="token punctuation">:</span> 150px<span class="token punctuation">;</span> <span class="token comment">/* 初始宽度为150px */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="4_flex_127">
     </a>
     4.
     <strong>
      flex简写属性
     </strong>
    </h5>
    <p>
     •
     <code>
      flex: grow shrink basis
     </code>
     → 用于同时设置三个属性。
     <br/>
     •
     <strong>
      示例
     </strong>
     ：
     <code>
      flex: 1 1 auto
     </code>
     → 等价于
     <code>
      flex-grow: 1
     </code>
     ,
     <code>
      flex-shrink: 1
     </code>
     ,
     <code>
      flex-basis: auto
     </code>
     。
    </p>
    <pre><code class="prism language-css"><span class="token selector">.item</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">flex</span><span class="token punctuation">:</span> 1<span class="token punctuation">;</span> <span class="token comment">/* 简写：均分剩余空间，可收缩可扩展 */</span>
<span class="token punctuation">}</span>
</code></pre>

    <h4>
     <a id="_191">
     </a>
     <strong>
      五、常见问题与误区
     </strong>
    </h4>
    <h5>
     <a id="1_flex_192">
     </a>
     1.
     <strong>
      flex布局为何无法居中？
     </strong>
    </h5>
    <p>
     •
     <strong>
      原因
     </strong>
     ：未正确设置交叉轴对齐属性。
     <br/>
     •
     <strong>
      解决
     </strong>
     ：单行布局用
     <code>
      justify-content: center
     </code>
     ，单列布局用
     <code>
      align-items: center
     </code>
     。
    </p>
    <h5>
     <a id="2__196">
     </a>
     2.
     <strong>
      子元素高度不一致导致布局错乱
     </strong>
    </h5>
    <p>
     •
     <strong>
      原因
     </strong>
     ：未设置
     <code>
      align-items
     </code>
     或
     <code>
      align-self
     </code>
     。
     <br/>
     •
     <strong>
      解决
     </strong>
     ：使用
     <code>
      align-items: stretch
     </code>
     让项目拉伸填充容器高度。
    </p>
    <h5>
     <a id="3_flexshrink_0__200">
     </a>
     3.
     <strong>
      flex-shrink: 0 导致溢出
     </strong>
    </h5>
    <p>
     •
     <strong>
      原因
     </strong>
     ：项目固定大小且容器空间不足时拒绝收缩。
     <br/>
     •
     <strong>
      解决
     </strong>
     ：合理设置
     <code>
      flex-shrink
     </code>
     或调整容器尺寸。
    </p>
