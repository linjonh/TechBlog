    <h2>
     <a id="MongoDB_0">
     </a>
     什么是MongoDB
    </h2>
    <p>
     MongoDB 是一个基于分布式文件存储的数据库，由 C++ 语言编写。它是一个 NoSQL 数据库，旨在为现代应用提供高性能、高可用性和可扩展性。
    </p>
    <h3>
     <a id="_4">
     </a>
     特点
    </h3>
    <ul>
     <li>
      <strong>
       文档存储
      </strong>
      ：MongoDB 使用 BSON（类似 JSON 的二进制格式）存储数据。
     </li>
     <li>
      <strong>
       灵活的模式
      </strong>
      ：无需预定义数据结构，支持动态模式。
     </li>
     <li>
      <strong>
       高性能
      </strong>
      ：支持索引、复制和分片，提供高效的数据访问和存储。
     </li>
     <li>
      <strong>
       高可用性
      </strong>
      ：通过复制集实现数据的高可用性和自动故障恢复。
     </li>
     <li>
      <strong>
       可扩展性
      </strong>
      ：支持水平扩展，通过分片技术处理大规模数据。
     </li>
    </ul>
    <h3>
     <a id="_12">
     </a>
     使用场景
    </h3>
    <ul>
     <li>
      <strong>
       大数据应用
      </strong>
      ：适用于需要处理大量数据的应用，如日志分析、实时数据处理等。
     </li>
     <li>
      <strong>
       内容管理系统
      </strong>
      ：适用于需要灵活数据模型的内容管理系统。
     </li>
     <li>
      <strong>
       物联网
      </strong>
      ：适用于需要存储和处理大量传感器数据的物联网应用。
     </li>
    </ul>
    <h3>
     <a id="_18">
     </a>
     基本命令
    </h3>
    <ul>
     <li>
      <code>
       show dbs
      </code>
      ：显示所有数据库。
     </li>
     <li>
      <code>
       use &lt;database&gt;
      </code>
      ：切换到指定数据库。
     </li>
     <li>
      <code>
       show collections
      </code>
      ：显示当前数据库中的所有集合。
     </li>
     <li>
      <code>
       db.&lt;collection&gt;.find()
      </code>
      ：查询集合中的所有文档。
     </li>
    </ul>
