    <p>
     前言
    </p>
    <p style="text-indent:50px;">
     在业务中遇到需要获取商品的二维码的信息返回商品的二维码信息，在调用后台的接口，首先想到用小程序的Api,wx.scanCode(Object object)获取二维码信息。
    </p>
    <p style="text-indent:50px;">
     支持相机扫描或者相册获取二维码。
    </p>
    <pre class="has"><code>  getScancode: function () {
    var _this = this;
    // 允许从相机和相册扫码
    wx.scanCode({
      success: (res) =&gt; {
        var result = res.result;
        var scanType = res.scanType;
        var charSet = res.charSet;
        var path = res.path;
        _this.setData({
          result: result,
          scanType: scanType,
          charSet: charSet,
          path: path
        })
      }
    })
  }</code></pre>
    <p>
     返回参数：
    </p>
    <pre class="has"><code>属性	    类型	说明	支持版本
result	    string	所扫码的内容	
scanType    string	所扫码的类型	
charSet	    string	所扫码的字符集	
path	    string	当所扫的码为当前小程序的合法二维码时，会返回此字段，内容为二维码携带的 path	
rawData	    string	原始数据，base64编码</code></pre>
    <p style="text-indent:50px;">
     通过返回的参数，即可完成相关的业务。
    </p>
