     AI Enterprise 3.1 将企业推向人工智能的前沿

    <p>
     生成人工智能标志着人工智能革命历程中的一个重要里程碑。我们正处于一个根本性的转折点，企业不仅要涉足其中，还要深入其中。
    </p>
    <p>
     NVIDIA 人工智能平台的软件层
     <a href="https://www.nvidia.com/en-us/data-center/products/ai-enterprise/" rel="nofollow" title="NVIDIA AI Enterprise">
      NVIDIA AI Enterprise
     </a>
     拥有 50 多个框架、预训练的模型和开发工具，旨在加速企业进入人工智能的前沿，同时简化人工智能，使每个企业都能访问它。
    </p>
    <p>
     另一个令人兴奋的消息是宣布了一项
     <a href="https://nvidianews.nvidia.com/news/at-t-supercharges-operations-with-nvidia-ai" rel="nofollow" title="AT&amp;T Corp. 与 NVIDIA 之间的合作">
      AT&amp;T Corp. 与 NVIDIA 之间的合作
     </a>
     ， AT &amp; T 将继续通过使用 NVIDIA 提供的人工智能来改变其运营并增强可持续性。这包括使用
     <a href="https://www.nvidia.com/en-us/data-center/products/ai-enterprise/" rel="nofollow" title="NVIDIA AI Enterprise">
      NVIDIA AI Enterprise
     </a>
     软件套件增强其数据处理，该软件套件现在包括
     <a href="https://www.nvidia.com/en-us/deep-learning-ai/solutions/data-science/apache-spark-3/" rel="nofollow" title="NVIDIA RAPIDS Accelerator for Apache Spark">
      NVIDIA RAPIDS Accelerator for Apache Spark
     </a>
     。
    </p>
    <h3>
     发布亮点：加速数据处理、人工智能工作流程、扩大云计算可用性
    </h3>
    <p>
     在 2023 年的 NVIDIA GTC 2023 上， NVIDIA 宣布了最新版本的 NVIDIA AI Enterprise ，加速了内部部署和云计算的价值实现。更新添加了以下功能和工具等：
    </p>
    <ul>
     <li>
      RAPIDS Apache 加速器 Spark
     </li>
     <li>
      用于个性化产品推荐和路线优化的 NVIDIA AI 工作流
     </li>
     <li>
      扩展了云中的可用性
     </li>
     <li>
      支持最新的 NVIDIA Ada Lovelace 架构 GPU
     </li>
    </ul>
    <h4>
     使用 RAPIDS Apache 加速器 Spark 实现 5 倍的数据处理加速
    </h4>
    <p>
     数据处理将数据集转换为有意义和有信息的形式，为人工智能提供燃料。但随着组织积累越来越多的数据，处理成本和时间也在增长。
    </p>
    <p>
     NVIDIA AI Enterprise 3.1 包括
     <a href="https://www.nvidia.com/en-us/deep-learning-ai/solutions/data-science/apache-spark-3/" rel="nofollow" title="RAPIDS Accelerator for Apache Spark">
      RAPIDS Accelerator for Apache Spark
     </a>
     ，使数据科学家和数据工程师能够将 Apache Spark 3.x 加速 5 倍，并在不更改代码的情况下，在仅使用 CPU 的平台上节省高达 75% 的基础设施成本。
    </p>
    <p>
     您可以在经过认证的平台上获得 Spark 部署的优化性能，并完全访问企业级支持，包括 Azure 和 AWS 上的 Amazon EMR 、 Google Cloud Dataproc 和 Databricks 。
    </p>
    <h4>
     新的人工智能工作流程：下一个项目预测和路线优化
    </h4>
    <p>
     NVIDIA 人工智能工作流是云原生的预打包参考示例，展示了如何使用 NVIDIA AI 框架构建人工智能解决方案。 NVIDIA AI Enterprise 3.1 提供了两种新的人工智能工作流程：
    </p>
    <ul>
     <li>
      <a href="https://www.nvidia.com/en-us/ai-data-science/ai-workflows/next-item-prediction/" rel="nofollow" title="NVIDIA next item prediction AI workflow">
       NVIDIA next item prediction AI workflow
      </a>
      用于建立推荐渠道，以推动客户保留和促销。
     </li>
     <li>
      <a href="https://www.nvidia.com/en-us/ai-data-science/ai-workflows/route-optimization/" rel="nofollow" title="NVIDIA route optimization AI workflow">
       NVIDIA route optimization AI workflow
      </a>
      用于优化配送和配送车队，以在动态约束下减少成本和时间。
     </li>
    </ul>
    <p>
     这些参考应用程序为您提供了一条快速开始使用人工智能推荐和路线优化解决方案的途径，并满足您的业务目标。
    </p>
    <h4>
     NVIDIA AI Enterprise 可在云市场上使用
    </h4>
    <p>
     为了提高企业支持的人工智能软件在云中的可访问性， NVIDIA AI enterprise 现已在
     <a href="https://console.cloud.google.com/marketplace/product/nvidia/nvidia-ai-enterprise-vmi" rel="nofollow" title="Google Cloud Marketplace">
      Google Cloud Marketplace
     </a>
     上提供，并将很快在 AWS 和 Microsoft Azure Marketplace 上提供。
    </p>
    <p>
     您可以按需立即开始，或者使用您承诺的云支出协议在几分钟内访问全栈软件套件。 NVIDIA AI Enterprise 包括人工智能框架和工具，用于在 NVIDIA 提供的企业级支持下构建数据科学管道和人工智能应用程序。
    </p>
    <p>
     它还被认证可以在谷歌云、 AWS 、 Azure 和甲骨文云上运行，供携带自己许可证的客户使用。
    </p>
    <h4>
     使用 CSP 容器编排服务实现 Kubernetes 自动化
    </h4>
    <p>
     NVIDIA AI Enterprise 3.1 支持 Kubernetes 容器托管环境，包括 Amazon 弹性 Kubernete 服务（ Amazon EKS ）和谷歌 Kubernets 引擎（ GKE ）。
    </p>
    <p>
     通过使用集成的 GPU 实例，您可以利用完全支持、优化的人工智能软件堆栈和简单的部署，在云中高效地运行分布式人工智能作业。
    </p>
    <h4>
     使用最新的 GPU 架构加速，增强混合云基础设施
    </h4>
    <p>
     最新的 NVIDIA AI Enterprise 版本支持以下 NVIDIA Ada Lovelace 架构 GPU ，以增强计算密集型人工智能和推理工作负载：
    </p>
    <ul>
     <li>
      NVIDIA L4 公司
     </li>
     <li>
      NVIDIA L40 环境
     </li>
     <li>
      NVIDIA RTX 6000 Ada 一代
     </li>
    </ul>
