    <p>
     又来博客留下我的足迹了，哈哈哈，这次是对于双向链表的理解
    </p>
    <hr id="hr-toc" name="tableOfContents"/>
    <p>
    </p>
    <p>
     在双向链表中有头双向循环，无头双向循环，有头双向不循环，无头双向不循环，而我将要介绍的是有头双向循环，别看名字长，其实就是只纸老虎，只要我们理解它的结构，问题自然迎刃而解了结构图如下：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/9fade68d8e174d44b90add55107a882f.png"/>
    </p>
    <h4 id="%E5%88%9B%E5%BB%BA%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%EF%BC%9A" name="%E5%88%9B%E5%BB%BA%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%EF%BC%9A">
     创建双向链表：
    </h4>
    <p>
     从上面的结构图我们不然发现，我们创建需要定义什么指针域和数据域
    </p>
    <pre><code>typedef int LTDataType;
typedef struct ListNode
{
	struct ListNode* prev;//指针域
	struct ListNode* next;//指针域
	LTDataType data;//数据域
}LTNode;</code></pre>
    <h4 id="%E7%94%B3%E8%AF%B7%E7%BB%93%E7%82%B9%EF%BC%9A" name="%E7%94%B3%E8%AF%B7%E7%BB%93%E7%82%B9%EF%BC%9A">
     申请结点：
    </h4>
    <p>
     与单链表代码差不多，将其指针域置空，就不再过多赘述
    </p>
    <pre><code>LTNode* BuyNode(LTDataType x)
{
	LTNode* newnode = (LTNode*)malloc(sizeof(LTNode));//申请空间
	if (newnode == NULL)
	{
		perror("malloc fail");
		return;
	}
	newnode-&gt;data = x;//赋值
	newnode-&gt;next = NULL;
	newnode-&gt;prev = NULL;
	return newnode;//返回创建的结点
}
</code></pre>
    <p>
    </p>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E5%88%9D%E5%A7%8B%E5%8C%96%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E5%88%9D%E5%A7%8B%E5%8C%96%EF%BC%9A">
     双向链表初始化：
    </h4>
    <p>
     我们只需将自己连向自己，下一个指向上一个，上一个指向下一个，如图：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/d9a04d77742e4c7cb0920e5580d68d27.png"/>
    </p>
    <pre><code>LTNode* LTInit()
{
	LTNode* phead = BuyNode(-1);//传空，为phead申请空间
	phead-&gt;next = phead-&gt;prev;
	phead-&gt;prev = phead-&gt;next;
	return phead;//返回头结点
}</code></pre>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E6%8F%92%E5%85%A5%E7%BB%93%E7%82%B9%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E6%8F%92%E5%85%A5%E7%BB%93%E7%82%B9%EF%BC%9A">
     双向链表插入结点：
    </h4>
    <p>
     头插：
    </p>
    <p>
     我们先将新结点的后指针指向第一个结点的数据域，第一个结点的前指针指向新结点的数据域，新结点的前指针亦然，可能文字可能形容有点模糊，所以我们看下图：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/4ce2047513d74fd892c6dad303f73ea8.png"/>
    </p>
    <pre><code>void LTPushFront(LTNode* phead, LTDataType x)
{
	assert(phead);
	LTNode* newnode = BuyNode(x);
	LTNode* Next = phead-&gt;next;//保存下一个结点的地址，防止丢失
	//新结点与后结点链接
	newnode-&gt;next = Next;
	Next-&gt;prev = newnode;
	//新结点与头节点链接
	newnode-&gt;prev = phead;
	phead-&gt;next = newnode;
}</code></pre>
    <p>
     <strong>
      温馨提示：如果没有保存下一个结点的地址，则需先跟后结点链接，在与头结点相连
     </strong>
    </p>
    <p>
     尾插：
    </p>
    <p>
     双链表尾插相对于单链表的尾插来说要容易许多，因为我们可以轻松找到尾，然后改变指针指向即可，如下图：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/6c5c3c4d71a34dc2aed3e44eb9c4cdca.png"/>
    </p>
    <pre><code>void LTPushBack(LTNode* phead, LTDataType x)
{
	assert(phead);
	LTNode* newnode = BuyNode(x);//创建新结点
	LTNode* tail = phead-&gt;prev;//找尾
	//尾结点与新结点相连
	newnode-&gt;prev = tail;
	tail-&gt;next = newnode;
	//新结点与头结点相连
	newnode-&gt;next = phead;
	phead-&gt;prev = newnode;
}</code></pre>
    <p>
     在指定位置插入：
    </p>
    <p>
     我们通常会在指定位置之前插入，找到指定位置前一个，然后改变指针方向即可，如图：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/3cea9221316348198387c906f8e3c37b.png"/>
    </p>
    <pre><code>void LTInsert(LTNode* phead, LTNode* pos, LTDataType x)
{
	assert(phead);
	LTNode* newnode = BuyNode(x);
	LTNode* front = pos-&gt;prev;//找到指定位置前一个结点
	//新结点与指定结点相连
	newnode-&gt;next = pos;
	pos-&gt;prev = newnode;
	//新节点与指定结点前一个结点相连
	front-&gt;next = newnode;
	newnode-&gt;prev = front;
}</code></pre>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E5%88%A0%E9%99%A4%E7%BB%93%E7%82%B9%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E5%88%A0%E9%99%A4%E7%BB%93%E7%82%B9%EF%BC%9A">
     双向链表删除结点：
    </h4>
    <p>
     头删：
    </p>
    <p>
     我们先要判断链表是否为空，如果为空就不用删除了；因为之后要释放删除的结点，所以我们还需保存一下，这样就可以了
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/93c59188723147bb827e750c8053315c.png"/>
    </p>
    <pre><code>bool LTEmpty(LTNode* phead)
{
	return phead == phead-&gt;next;
}
void LTPopFront(LTNode* phead)
{
	assert(phead);
	assert(!LTEmpty(phead));//判断链表是否为空
	LTNode* del = phead-&gt;next;
	LTNode* Next = phead-&gt;next-&gt;next;
	//第一结点的下一个结点与头结点相连
	Next-&gt;prev = phead;
	phead-&gt;next = Next;
	free(del);//释放掉这个结点
	del = NULL;
}</code></pre>
    <p>
     尾删：
    </p>
    <p>
     尾删就比较容易了，将尾结点释放，然后改变指针指向，就这样完成了^ - ^
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/4d833a9d73784d45973603732a34fcc8.png"/>
    </p>
    <pre><code>void LTPopBack(LTNode* phead)
{
	assert(phead);
	assert(phead != phead-&gt;next);
	//或assert(!LTEmpty(phead))
	LTNode* tail = phead-&gt;prev;
	LTNode* Pretail = tail-&gt;prev;
	//头节点与尾结点的前一个结点相连
	Pretail-&gt;next = phead;
	phead-&gt;next = Pretail;
	free(tail);//释放尾结点
	tail = NULL;
}</code></pre>
    <p>
     在指定位置删除：
    </p>
    <p>
     找到要删除结点的前一个和后一个，然后两个结点相互链接，这样就能够删除了，如图：
    </p>
    <pre><code>void LTErase(LTNode* phead, LTNode* pos)
{
	assert(phead);
	LTNode* front = pos-&gt;prev;//找到前结点
	LTNode* back = pos-&gt;next;//找到后结点
	//前结点和后结点相连
	front-&gt;next = back;
	back-&gt;prev = front;
	free(pos);//释放指定节点
	pos = NULL;
}</code></pre>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E6%89%93%E5%8D%B0%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E6%89%93%E5%8D%B0%EF%BC%9A">
     双向链表的打印：
    </h4>
    <p>
     提到打印，我们会用到遍历循环，那循环结束的标志是什么呢？有一个好主意，我们可以先从第一个结点开始打印，然后向后循环遍历，直至遍历到头结点，循环结束^_^，如下图：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/8c5e1b37425e4135bcc236d041af836b.png"/>
    </p>
    <pre><code>void LTPrint(LTNode* phead)
{
	LTNode* begin = phead-&gt;next;
	while (begin != phead)//循环继续条件
	{
		printf("%d&lt;=&gt;", begin-&gt;data);
		begin = begin-&gt;next;
	}
	printf("\n");
}</code></pre>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E6%9F%A5%E6%89%BE%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E6%9F%A5%E6%89%BE%EF%BC%9A">
     双向链表的查找：
    </h4>
    <p>
     遍历一遍链表，然后找要查找的元素
    </p>
    <pre><code>LTNode* LTFind(LTNode* phead, LTDataType x)
{
	assert(phead);
	LTNode* cur = phead-&gt;next;
	while (cur != phead)
	{
		if (cur-&gt;data == x)
			return cur;
		cur = cur-&gt;next;
	}
	return NULL;//没找到
}</code></pre>
    <h4 id="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E9%94%80%E6%AF%81%EF%BC%9A" name="%E5%8F%8C%E5%90%91%E9%93%BE%E8%A1%A8%E7%9A%84%E9%94%80%E6%AF%81%EF%BC%9A">
     双向链表的销毁：
    </h4>
    <p>
     将动态申请的空间释放掉，并循环释放每一个结点，防止内存泄漏的风险
    </p>
    <pre><code>void LTDestroy(LTNode* phead)
{
	assert(phead);
	LTNode* cur = phead-&gt;next;
	LTNode* next = cur-&gt;next;
	while (cur != phead)
	{
		LTNode* next = cur-&gt;next;//防止找不到下一个结点
		free(cur);
		cur = next;
	}
	free(phead);//释放头结点
}</code></pre>
