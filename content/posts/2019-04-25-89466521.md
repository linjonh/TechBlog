## 七、登录禅道数据库

*禅道Windows一键安装包默认只能在服务器本机登录。*

（1）浏览器里输入
**http://127.0.0.1/adminer/**
，进入禅道数据库登录页面：

![](https://i-blog.csdnimg.cn/blog_migrate/b9fca684b74b48c9d192a62190626827.png)

---

（2）在登录页面填写
**xampp/zentao/config/my.php**
里的对应参数，即可进入禅道数据库：

```java
系  统：默认选择 MySQL 。

服务器：对应填写 $config->db ->host 和 port 的参数。
// 比如：$config->db ->host = '127.0.0.1'， $config->db ->port = '3306'。
// 那么服务器填写：127.0.0.1:3306。

用户名：对应填写 $config->db ->user 的参数。
// 比如：$config->db ->user = 'zentao' 。
// 那么用户名就应该填写 zentao 。

密  码：对应填写 $config->db ->password = ' ' 中单引号里的值。
// 每个人的数据库密码会不一样，单引号里显示的值是什么，密码就填写什么

数据库：对应填写 $config->db ->name 的参数。
// 比如：$config->db ->name = 'zentao '。
// 那么数据库就应该填写 zentao 。
```

## 八、非本机访问禅道

*禅道部署在内网服务器上，那么内网可以通过ip访问，外网无法访问；若是禅道部署在外网服务器上，那就可以在外网通过ip访问。*

禅道访问地址为
**http://禅道机器的ip地址:端口号/zentao/**
，ip地址换成禅道机器实际的ip地址，端口号换成实际的端口号。
