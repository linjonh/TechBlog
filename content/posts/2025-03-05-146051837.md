    <h2>
     <a id="_8">
     </a>
     算法原理（双指针）
    </h2>
    <ul>
     <li>
      在数组中操作元素的题目，一般都用
      <em>
       <strong>
        双指针算法
       </strong>
      </em>
      <ul>
       <li>
        现根据“异地”操作，再优化双指针下的“就地”操作
        <ul>
         <li>
          <p>
           例子
          </p>
         </li>
         <li>
          <p>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/30f1bba5cecb48118ba87f56df7bcb0b.png"/>
          </p>
          <ul>
           <li>
            <p>
             定义两个指针
             <strong>
              cur和dest
             </strong>
             ，分别指向两个数组的开头，当cur识别非零元素直接拷贝到dest指向的数组，遇到0元素直接写两遍，直到dest的数组元素的最大值
            </p>
           </li>
           <li>
            <p>
             例子优化成一个数组
            </p>
            <ul>
             <li>
              该题目
              <strong>
               从左到右
              </strong>
              无法进行复写，所以最好的方法是
              <strong>
               从右到左
              </strong>
             </li>
            </ul>
           </li>
           <li>
            <p>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5d0b9d0c031c48e3a25f163a18a4f972.png"/>
            </p>
            <ul>
             <li>
              <p>
               先找到最后一个“复写”的数
               <br/>
               <strong>
                1. 用双指针算法，dest定义是否为结果中的最后一个位置，cur从前到后遍历数组，
                <br/>
                2. 第一步 先cur先进行遍历，判断cur是否为0，
                <br/>
                3. 第二步 移动dest一步或两步，
                <br/>
                4. 第三步 判断dest是否已经结束为止。
                <br/>
                5. 第四步 cur++
               </strong>
               **
               <br/>
               6.
               <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b4e20c6469ff4f2db216be0e6c16df4b.png"/>
              </p>
              <ul>
               <li>
                <p>
                 非零元素移动一步
                </p>
               </li>
               <li>
                <p>
                 0元素移动两步
                 <br/>
                 <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bec772fba20e4cceb73413124ff6acd0.png"/>
                </p>
               </li>
               <li>
                <p>
                 dest结尾结束
                 <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f6223f79bd2e4b68a99e829395b75a14.png"/>
                </p>
               </li>
              </ul>
              <p>
               <strong>
                特例：
               </strong>
              </p>
             </li>
             <li>
              <p>
               当cur遍历过程中，遇到0元素，dest正好超出边界
              </p>
             </li>
             <li>
              <p>
               <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ff7d332a6b5e47a1a9575730332fc601.png"/>
              </p>
              <p>
               <strong>
                - 处理情况
               </strong>
               <br/>
               <strong>
                - 让cur退一步，dest退两步，然后正常遍历
               </strong>
               <br/>
               <strong>
                - 从后向前完成复写步操作
               </strong>
              </p>
             </li>
            </ul>
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <pre><code>			**1. 双指针**
				- cur位于“复写”数位置从左往右遍历
				- dest执行复写指令
			**2. 操作步骤**
				- 判断cur的值
				- dest进行复写数组值
					- cur为非零，直接复写cur所指向的值
					- cur为0，复写0，dest往前走一步，每一步所指向的值都复写为0
				- dest--，cur--;
</code></pre>
