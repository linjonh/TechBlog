     基础问题简答

    <h4>
     <a id="1__11">
     </a>
     1. 内存对齐
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：提高 CPU 访问效率，避免跨对齐边界访问导致性能下降或硬件错误。
     </li>
     <li>
      <strong>
       实现
      </strong>
      ：编译器通过插入填充字节，使数据按类型对齐（如
      <code>
       int
      </code>
      按 4 字节对齐）；可用
      <code>
       #pragma pack(n)
      </code>
      或
      <code>
       alignas(n)
      </code>
      控制。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">struct</span> <span class="token class-name">A</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">char</span> a<span class="token punctuation">;</span> <span class="token keyword">int</span> b<span class="token punctuation">;</span> <span class="token keyword">short</span> c<span class="token punctuation">;</span> <span class="token punctuation">}</span><span class="token punctuation">;</span> <span class="token comment">// sizeof(A) = 12，填充 3+2 字节</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="2_this__19">
     </a>
     2.
     <code>
      this
     </code>
     指针
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：指向当前对象实例，用于访问成员变量和方法，支持链式调用。
     </li>
     <li>
      <strong>
       实现
      </strong>
      ：编译器隐式传递，成员函数接收
      <code>
       this
      </code>
      作为隐藏参数（如
      <code>
       void A::func(A* this)
      </code>
      ）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">A</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">int</span> x<span class="token punctuation">;</span> <span class="token keyword">void</span> <span class="token function">set</span><span class="token punctuation">(</span><span class="token keyword">int</span> x<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">this</span><span class="token operator">-&gt;</span>x <span class="token operator">=</span> x<span class="token punctuation">;</span> <span class="token punctuation">}</span> <span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="3__this_27">
     </a>
     3. 在成员函数中删除
     <code>
      this
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       后果
      </strong>
      ：释放对象内存，后续访问会导致未定义行为（如崩溃）。
     </li>
     <li>
      <strong>
       注意
      </strong>
      ：仅适用于动态分配对象，需立即返回避免继续使用。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">A</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">void</span> <span class="token function">destroy</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">delete</span> <span class="token keyword">this</span><span class="token punctuation">;</span> <span class="token keyword">return</span><span class="token punctuation">;</span> <span class="token punctuation">}</span> <span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="4__35">
     </a>
     4. 引用占用内存吗？
    </h4>
    <ul>
     <li>
      <strong>
       答案
      </strong>
      ：通常不占额外内存（作为别名优化掉），但在类成员等场景可能占用指针大小（4/8 字节）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span> <span class="token keyword">int</span><span class="token operator">&amp;</span> ref <span class="token operator">=</span> a<span class="token punctuation">;</span> <span class="token comment">// ref 不占额外空间</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="5_C__42">
     </a>
     5. C++ 越界访问场景
    </h4>
    <ul>
     <li>
      数组越界（如
      <code>
       arr[10]
      </code>
      访问大小为 5 的数组）。
     </li>
     <li>
      野指针或已释放内存访问。
     </li>
     <li>
      容器操作（如
      <code>
       vector
      </code>
      的
      <code>
       operator[]
      </code>
      未检查边界）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">int</span> arr<span class="token punctuation">[</span><span class="token number">5</span><span class="token punctuation">]</span><span class="token punctuation">;</span> arr<span class="token punctuation">[</span><span class="token number">10</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token number">1</span><span class="token punctuation">;</span> <span class="token comment">// 未定义行为</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="6__51">
     </a>
     6. 进程通信方式
    </h4>
    <ul>
     <li>
      管道（匿名/命名）、消息队列、共享内存、信号量、套接字、信号、mmap 文件映射、Unix 域套接字。
     </li>
    </ul>
    <h4>
     <a id="7__54">
     </a>
     7. 无锁队列实现
    </h4>
    <ul>
     <li>
      <strong>
       方法
      </strong>
      ：基于原子操作（如 CAS）管理头尾指针。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp">std<span class="token double-colon punctuation">::</span>atomic<span class="token operator">&lt;</span>Node<span class="token operator">*</span><span class="token operator">&gt;</span> head<span class="token punctuation">;</span> <span class="token comment">// 使用 CAS 更新</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="8_ping__61">
     </a>
     8.
     <code>
      ping
     </code>
     在哪一层？实现原理？
    </h4>
    <ul>
     <li>
      <strong>
       层级
      </strong>
      ：网络层（ICMP 协议）。
     </li>
     <li>
      <strong>
       原理
      </strong>
      ：发送 ICMP Echo 请求，接收 Echo 响应，计算 RTT 判断连通性。
     </li>
    </ul>
    <h4>
     <a id="9_HTTPS__65">
     </a>
     9. HTTPS 流程
    </h4>
    <ol>
     <li>
      客户端发送
      <code>
       ClientHello
      </code>
      （TLS 版本、加密套件）。
     </li>
     <li>
      服务器返回
      <code>
       ServerHello
      </code>
      和证书。
     </li>
     <li>
      客户端验证证书，生成预主密钥并加密发送。
     </li>
     <li>
      双方派生对称密钥，加密通信。
     </li>
    </ol>
    <h4>
     <a id="10_GDB__CPU__71">
     </a>
     10. GDB 使用及 CPU 高使用定位
    </h4>
    <ul>
     <li>
      <strong>
       使用
      </strong>
      ：
      <code>
       gdb attach &lt;PID&gt;
      </code>
      ，
      <code>
       thread apply all bt
      </code>
      查看堆栈。
     </li>
     <li>
      <strong>
       定位
      </strong>
      ：
      <code>
       top -H
      </code>
      找高 CPU 线程，
      <code>
       gdb
      </code>
      分析热点代码。
     </li>
    </ul>
    <h4>
     <a id="11__75">
     </a>
     11. 智能指针
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        shared_ptr
       </code>
      </strong>
      ：引用计数，共享所有权。
     </li>
     <li>
      <strong>
       <code>
        unique_ptr
       </code>
      </strong>
      ：独占所有权，支持移动。
     </li>
     <li>
      <strong>
       <code>
        weak_ptr
       </code>
      </strong>
      ：解决循环引用。
     </li>
    </ul>
    <h4>
     <a id="12__80">
     </a>
     12. 右值引用与移动语义
    </h4>
    <ul>
     <li>
      <strong>
       右值引用（
       <code>
        &amp;&amp;
       </code>
       ）
      </strong>
      ：绑定临时对象。
     </li>
     <li>
      <strong>
       移动语义
      </strong>
      ：通过右值引用转移资源，避免深拷贝。
     </li>
     <li>
      <strong>
       差异
      </strong>
      ：右值引用是语法手段，移动语义是优化策略。
     </li>
    </ul>

    <h3>
     <a id="_95">
     </a>
     扩展问题详解
    </h3>
    <h4>
     <a id="0______97">
     </a>
     0. 自我介绍 + 项目介绍 + 问题解决
    </h4>
    <ul>
     <li>
      <strong>
       示例
      </strong>
      ：我是某公司开发，参与过高并发服务器项目，遇到内存泄漏问题。用
      <code>
       valgrind
      </code>
      定位，优化资源释放逻辑解决。
     </li>
    </ul>
    <h4>
     <a id="1__100">
     </a>
     1. 智能指针详解
    </h4>
    <ul>
     <li>
      <strong>
       出现原因
      </strong>
      ：避免裸指针的内存泄漏和悬空指针问题。
     </li>
     <li>
      <strong>
       性能开销
      </strong>
      ：
      <code>
       shared_ptr
      </code>
      有引用计数开销（原子操作），
      <code>
       unique_ptr
      </code>
      无额外开销。
     </li>
     <li>
      <strong>
       支持类型
      </strong>
      ：任意类型（内置类型、类、指针等）。
     </li>
    </ul>
    <h4>
     <a id="2_STL__105">
     </a>
     2. STL 容器底层原理
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        vector
       </code>
      </strong>
      ：动态数组，连续内存，容量翻倍增长。
     </li>
     <li>
      <strong>
       <code>
        list
       </code>
      </strong>
      ：双向链表，支持快速插入删除。
     </li>
     <li>
      <strong>
       <code>
        map
       </code>
      </strong>
      ：红黑树，键值对有序存储。
     </li>
    </ul>
    <h4>
     <a id="3_B__110">
     </a>
     3. B+ 树详解
    </h4>
    <ul>
     <li>
      <strong>
       结构
      </strong>
      ：多叉树，叶子节点存储数据，非叶子节点存储索引。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：叶子节点双向链表（InnoDB），支持范围查询。
     </li>
     <li>
      <strong>
       优势
      </strong>
      ：减少 IO，提升查询效率。
     </li>
    </ul>
    <h4>
     <a id="4__115">
     </a>
     4. 线程创建方式
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        std::thread
       </code>
      </strong>
      ：C++11 标准。
     </li>
     <li>
      <strong>
       <code>
        pthread_create
       </code>
      </strong>
      ：POSIX 线程。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp">std<span class="token double-colon punctuation">::</span>thread <span class="token function">t</span><span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token punctuation">]</span><span class="token punctuation">{<!-- --></span> std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Thread\n"</span><span class="token punctuation">;</span> <span class="token punctuation">}</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="5_DFS__BFS__123">
     </a>
     5. DFS 和 BFS 过程
    </h4>
    <ul>
     <li>
      <strong>
       DFS（深度优先搜索）
      </strong>
      ：递归或栈，深入一条路径。
     </li>
     <li>
      <strong>
       BFS（广度优先搜索）
      </strong>
      ：队列，按层遍历。
     </li>
    </ul>
    <h4>
     <a id="malloc__free_vs_new__delete_127">
     </a>
     <code>
      malloc
     </code>
     /
     <code>
      free
     </code>
     vs
     <code>
      new
     </code>
     /
     <code>
      delete
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        malloc
       </code>
       /
       <code>
        free
       </code>
      </strong>
      ：C 风格，仅分配/释放内存，不调用构造/析构。
     </li>
     <li>
      <strong>
       <code>
        new
       </code>
       /
       <code>
        delete
       </code>
      </strong>
      ：C++ 风格，分配内存并调用构造/析构。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token keyword">int</span><span class="token operator">*</span> p <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token keyword">int</span><span class="token punctuation">(</span><span class="token number">10</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token keyword">delete</span> p<span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="shared_ptr__135">
     </a>
     <code>
      shared_ptr
     </code>
     线程安全吗？
    </h4>
    <ul>
     <li>
      <strong>
       引用计数
      </strong>
      ：线程安全（原子操作）。
     </li>
     <li>
      <strong>
       对象访问
      </strong>
      ：不安全，需外部加锁。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp">std<span class="token double-colon punctuation">::</span>shared_ptr<span class="token operator">&lt;</span><span class="token keyword">int</span><span class="token operator">&gt;</span> sp <span class="token operator">=</span> std<span class="token double-colon punctuation">::</span><span class="token generic-function"><span class="token function">make_shared</span><span class="token generic class-name"><span class="token operator">&lt;</span><span class="token keyword">int</span><span class="token operator">&gt;</span></span></span><span class="token punctuation">(</span><span class="token number">10</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h3>
     <a id="_143">
     </a>
     内存对齐详解
    </h3>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：提高 CPU 访问效率，避免硬件错误。
     </li>
     <li>
      <strong>
       实现
      </strong>
      ：
      <ul>
       <li>
        默认对齐：按类型大小（如
        <code>
         int
        </code>
        4 字节）。
       </li>
       <li>
        控制：
        <code>
         #pragma pack(n)
        </code>
        或
        <code>
         alignas(n)
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">pragma</span> <span class="token expression"><span class="token function">pack</span><span class="token punctuation">(</span><span class="token number">2</span><span class="token punctuation">)</span></span></span>
