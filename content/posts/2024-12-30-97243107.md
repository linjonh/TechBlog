    <p>
     <strong>
      1.进入到终端输入
      <code>
       java
      </code>
      命令
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b9834986832b70c9f3b1b781449f73d5.png">
      <br/>
      <strong>
       2.点击‘更多信息’去到官网，下载jdk,往下滑动，选择自己需要的Jdk版本
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/463fce00c6e6bc1226f4998ce38d39aa.png">

       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/213891c992693c788f4773de128808de.png">
        <br/>
        <strong>
         3.接受协议，下载镜像
        </strong>
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/83f23d80d3739cf507248353f436a64a.png">
         <br/>
         <strong>
          4.进行安装完成之后，在终端输入
          <code>
           java
          </code>
         </strong>
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a34db0d371390e1fb18ac4eebc0c5ef.png">
          <br/>
          <strong>
           5.配置java环境成功
          </strong>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     安装的默认路径：
    </p>
    <pre><code>/Library/Java/JavaVirtualMachines/jdk1.8.0_201.jdk
</code></pre>
    <p>
     <strong>
      6.打开terminal（终端）
     </strong>
     <br/>
     然后输入
     <code>
      vi .bash_profile
     </code>
     <br/>
     将这段文字输入：
     <br/>
     <code>
      JAVA_HOME= /Library/Java/JavaVirtualMachines/jdk1.8.0_201.jdk/Contents/Home export JAVA_HOME
     </code>
     <br/>
     在vi编辑器使用
     <code>
      :wq
     </code>
     命令保存退出
     <br/>
     输入
     <code>
      source .bash_profile
     </code>
     这样就可以使得环境变量起作用了
     <br/>
     你可以
     <code>
      echo $JAVA_HOME
     </code>
     看看有没有输出刚才配置的路径，如果输出成功代表你的环境变量已经配置好了
     <br/>
