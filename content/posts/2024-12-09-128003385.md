## 三、微信小游戏发布的几个功能

1、添加分享

```javascript
var id = '' // 通过 MP 系统审核的图片编号
var url = '' // 通过 MP 系统审核的图片地址
wx.shareAppMessage({
  imageUrlId: id,
  imageUrl: url
})

wx.onShareAppMessage(function () {
return {
imageUrlId: id,
imageUrl: url
}
})


```

2、添加广告

```javascript
et bannerAd = wx.createBannerAd({
adUnitId: 'xxxx',
style: {
left: 10,
top: 76,
width: 320
}
})

bannerAd.show()


```
