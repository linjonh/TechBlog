    <h2>
     <strong>
      题目
     </strong>
    </h2>
    <p>
     <img alt="" height="1223" src="https://i-blog.csdnimg.cn/direct/c482f04ba39044e0bb1053ab43105202.png" width="912"/>

    <h2>
     思路
    </h2>
    <p>
     因为两个链表都是逆序存储数字的，所以我们可以通过直接将各位的数字相加求得结果即可。
    </p>
    <p>
     我们可以用head以及tail保存相加后链表的头和尾。
    </p>
    <p>
     （1）若l1链表不为空，则val1 = l1-&gt;val，否则为0。
    </p>
    <p>
     （2）若l2链表不为空，则val2 = l2-&gt;val，否则为0。
    </p>
    <p>
     sum等于两个数字和进位相加。
    </p>
    <p>
     首先判断其第一位数字是否已经相加，若没有，则给head，tail创建一个值为sum%10的ListNode新节点。
    </p>
    <p>
     否则，将创建一个ListNode新节点与tail相连。
    </p>
    <p>
     若链表不为空，则继续遍历。
    </p>
    <p>
     最后，如果所有的数都加完了且carry不为0，则在最后补上值为carry的节点。
    </p>
    <h2>
     代码
    </h2>
    <pre><code class="language-cpp">class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* head = nullptr, *tail = nullptr;
        int carry = 0;
        while(l1 || l2)
        {
            int val1 = l1 ? l1-&gt;val : 0;
            int val2 = l2 ? l2-&gt;val : 0;
            int sum = carry + val1 + val2;
            if(!head)
            {
                head = tail = new ListNode(sum % 10);
            }
            else
            {
                tail-&gt;next = new ListNode(sum % 10);
                tail = tail-&gt;next;
            }
            carry = sum /10;
            if(l1)
            {
                l1 = l1-&gt;next;
            }
            if(l2)
            {
                l2 = l2-&gt;next;
            }
        }
        if(carry != 0)
        tail-&gt;next = new ListNode(carry);
        
        return head;
    }
};</code></pre>
