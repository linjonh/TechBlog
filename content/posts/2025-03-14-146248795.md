    <h1 class="title-article" id="articleContentId">
     C++ 左值（lvalue）和右值（rvalue）
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     在 C++ 中，左值（lvalue）和右值（rvalue）是指对象的不同类别，区分它们对于理解 C++ 中的表达式求值和资源管理非常重要，尤其在现代 C++ 中涉及到移动语义（Move Semantics）和完美转发（Perfect Forwarding）时。
    </p>
    <h2>
     <strong>
      一、左值（Lvalue）
     </strong>
    </h2>
    <h3>
     <strong>
      1. 定义
     </strong>
    </h3>
    <p>
     左值（lvalue）表示的是一个有名称的、持久的内存位置，可以在表达式的左侧，也可以在右侧使用。简单来说，左值是可以通过引用来访问的对象，它在程序的生命周期中有一个持久的存储位置。
    </p>
    <h3>
     <strong>
      2. 特征
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       可修改
      </strong>
      ：大多数左值是可修改的（当然，常量左值不可修改）。
     </li>
     <li>
      <strong>
       有持久地址
      </strong>
      ：左值有内存地址，可以通过
      <code>
       &amp;
      </code>
      取地址。
     </li>
    </ul>
    <h3>
     <strong>
      3. 例子
     </strong>
    </h3>
    <p>
    </p>
    <table>
     <tbody>
      <tr>
       <td>
        <pre>int x = 10;   // x 是一个左值，代表存储它的内存位置
x = 20;       // x 作为左值出现在赋值表达式的左侧</pre>
       </td>
      </tr>
     </tbody>
    </table>
    <p>
    </p>
    <p>
     <strong>
      4. 常见的左值类型
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       普通变量
      </strong>
      ：如
      <code>
       int x = 10;
      </code>
      中的
      <code>
       x
      </code>
      。
     </li>
     <li>
      <strong>
       数组元素
      </strong>
      ：如
      <code>
       arr[3]
      </code>
      。
     </li>
     <li>
      <strong>
       对象成员
      </strong>
      ：如
      <code>
       obj.member
      </code>
      。
     </li>
     <li>
      <strong>
       解引用指针
      </strong>
      ：如
      <code>
       ptr
      </code>
      。
     </li>
    </ul>
    <h2>
     <strong>
      二、右值（Rvalue）
     </strong>
    </h2>
    <h3>
     <strong>
      1. 定义
     </strong>
    </h3>
    <ul>
     <li>
      右值（rvalue）表示的是临时对象或不具有持久内存位置的对象，通常是表达式的结果。右值可以出现在赋值表达式的右侧，但不能出现在左侧（除非作为右值引用）。
     </li>
    </ul>
    <h3>
     <strong>
      2. 特征
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       无持久地址
      </strong>
      ：右值通常是一个临时对象，它在某些情况下会被销毁。
     </li>
     <li>
      <strong>
       不能修改
      </strong>
      ：右值本身不表示一个持久的存储位置，所以不能被赋值或取地址。
     </li>
    </ul>
    <h3>
     <strong>
      3. 例子
     </strong>
    </h3>
    <table>
     <tbody>
      <tr>
       <td>
        <pre>int x = 10;  // x 是左值
int y = 20;  // y 是左值
y = x + 5;   // (x + 5) 是右值，表示一个临时结果</pre>
       </td>
      </tr>
     </tbody>
    </table>
    <h3>
     <strong>
      4. 常见的右值类型
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       字面量
      </strong>
      ：如
      <code>
       5
      </code>
      、
      <code>
       3.14
      </code>
      、
      <code>
       'a'
      </code>
      等。
     </li>
     <li>
      <strong>
       临时对象
      </strong>
      ：如表达式的返回值，例如
      <code>
       x + y
      </code>
      返回一个临时结果。
     </li>
     <li>
      <strong>
       函数返回值
      </strong>
      ：如返回一个非引用的临时值
      <code>
       int foo() { return 42; }
      </code>
      。
     </li>
     <li>
      <strong>
       类型转换
      </strong>
      ：如
      <code>
       (int)3.14
      </code>
      或
      <code>
       std::move(x)
      </code>
      。
     </li>
    </ul>
    <h2>
     <strong>
      三、右值引用（Rvalue Reference）
     </strong>
    </h2>
    <p>
     C++11 引入了右值引用（
     <code>
      T&amp;&amp;
     </code>
     ），使得我们能够有效地使用右值（临时对象）进行资源转移（例如移动语义）。右值引用允许对象的资源不需要复制，而是可以直接“移动”到新的对象中，这样能提高程序的效率，避免不必要的资源复制。
    </p>
    <h3>
     <strong>
      1. 右值引用的使用
     </strong>
    </h3>
    <ul>
     <li>
      右值引用的声明通常为
      <code>
       T&amp;&amp;
      </code>
      （例如，
      <code>
       int&amp;&amp;
      </code>
      ）。
     </li>
     <li>
      <code>
       std::move
      </code>
      将一个左值转换为右值引用，从而启用移动语义。
     </li>
     <li>
      移动构造函数和移动赋值运算符通常采用右值引用作为参数，允许从临时对象中“偷取”资源。
     </li>
    </ul>
    <h3>
     <strong>
      2. 示例
     </strong>
    </h3>
    <table>
     <tbody>
      <tr>
       <td>
        <pre>#include &lt;iostream&gt;#include &lt;vector&gt;void printVector(std::vector&lt;int&gt;&amp;&amp; v) {
    for (auto i : v) {
        std::cout &lt;&lt; i &lt;&lt; " ";
    }
    std::cout &lt;&lt; std::endl;
}

int main() {
    printVector({1, 2, 3, 4, 5});  // 使用右值传递临时对象
    return 0;
}</pre>
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     在这个例子中，
     <code>
      {1, 2, 3, 4, 5}
     </code>
     是一个右值，可以作为右值引用参数传递给
     <code>
      printVector
     </code>
     函数。
    </p>
    <h2>
     <strong>
      四、区别与联系
     </strong>
    </h2>
    <ul>
     <li>
      <strong>
       左值（Lvalue）
      </strong>
      ：表示一个持久的对象，它有地址并且可以被修改。例如，变量、数组元素、解引用指针等。
     </li>
     <li>
      <strong>
       右值（Rvalue）
      </strong>
      ：表示一个临时对象或没有持久地址的值，通常出现在赋值语句的右边。它们通常不可修改。
     </li>
     <li>
      <strong>
       右值引用（Rvalue Reference）
      </strong>
      ：C++11 引入的一种新类型，允许我们将右值传递给函数，从而避免资源的复制（通过移动语义）。右值引用通过
      <code>
       T&amp;&amp;
      </code>
      来表示。
     </li>
    </ul>
    <h2>
     <strong>
      五、C++11 中的扩展：完美转发和移动语义
     </strong>
    </h2>
    <h3>
     <strong>
      1. 完美转发（Perfect Forwarding）
     </strong>
    </h3>
    <p>
     通过右值引用，我们可以将函数的参数完美地转发到另一个函数，无论是左值还是右值。使用
     <code>
      std::forward
     </code>
     可以实现完美转发。
    </p>
    <table>
     <tbody>
      <tr>
       <td>
        <pre>template&lt;typename T&gt;
void wrapper(T&amp;&amp; arg) {
    func(std::forward&lt;T&gt;(arg));  // 完美转发 arg 到 func 函数
}</pre>
       </td>
      </tr>
     </tbody>
    </table>
    <h3>
     <strong>
      2. 移动语义（Move Semantics）
     </strong>
    </h3>
    <p>
     右值引用可以用于“移动”资源，而不是复制它们。移动构造函数和移动赋值运算符允许通过“转移”资源来避免不必要的内存复制。
    </p>
    <table>
     <tbody>
      <tr>
       <td>
        <pre>std::vector&lt;int&gt; getVector() {
    std::vector&lt;int&gt; v = {1, 2, 3, 4};
    return v;  // 返回一个右值
}

int main() {
    std::vector&lt;int&gt; v = getVector();  // 通过移动语义，避免了不必要的复制
}</pre>
       </td>
      </tr>
     </tbody>
    </table>
    <h2>
     <strong>
      六、总结
     </strong>
    </h2>
    <ul>
     <li>
      <strong>
       左值（Lvalue）
      </strong>
      ：有持久存储位置，通常表示变量或对象。
     </li>
     <li>
      <strong>
       右值（Rvalue）
      </strong>
      ：没有持久存储位置，通常表示临时对象或值。
     </li>
     <li>
      <strong>
       右值引用（Rvalue Reference）
      </strong>
      ：用于支持移动语义，允许我们移动而不是复制资源。
     </li>
    </ul>
    <p>
     通过区分左值和右值，C++ 提供了更高效的内存管理方式，尤其在现代 C++ 中，移动语义和完美转发能够显著提高性能。
    </p>
