    <h3>
     <a id="876_Middle_of_the_Linked_List_0">
     </a>
     876. Middle of the Linked List
    </h3>
    <p>
     Given the
     <code>
      head
     </code>
     of a singly linked list, return
     <em>
      the middle node of the linked list
     </em>
     .
    </p>
    <p>
     If there are two middle nodes, return
     <strong>
      the second middle
     </strong>
     node.
    </p>
    <p>
     <strong>
      Example 1:
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bff99fc9f14845f3b5f840029a953bb2.png"/>
    </p>
    <pre><code>Input: head = [1,2,3,4,5]
Output: [3,4,5]
Explanation: The middle node of the list is node 3.
</code></pre>
    <p>
     <strong>
      Example 2:
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f12fad50c8824253aeb3b40481add32c.png"/>
    </p>
    <pre><code>Input: head = [1,2,3,4,5,6]
Output: [4,5,6]
Explanation: Since the list has two middle nodes with values 3 and 4, we return the second one.
</code></pre>
    <h3>
     <a id="_21">
     </a>
     问题描述
    </h3>
    <p>
     给定一个单链表的头节点
     <code>
      head
     </code>
     ，返回链表的中间节点。如果链表有两个中间节点（即链表长度为偶数），则返回第二个中间节点。
    </p>
    <h3>
     <a id="_23">
     </a>
     解题思路
    </h3>
    <p>
     要找到链表的中间节点，可以使用
     <strong>
      快慢指针法
     </strong>
     ：
    </p>
    <ul>
     <li>
      使用两个指针，一个快指针（每次走两步）和一个慢指针（每次走一步）。
     </li>
     <li>
      当快指针到达链表末尾时，慢指针正好指向链表的中间节点。
     </li>
     <li>
      如果链表长度为偶数，慢指针会指向第二个中间节点。
     </li>
    </ul>
