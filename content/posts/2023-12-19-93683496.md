     <p style="margin-left:16px;line-height:25.6px;">
      <span style="color:rgb(89,89,89);font-size:16px;">
       社交到视频会议，从金融到医疗，云端通信能力正在与传统的电话/IM、呼叫中心、企业通信融合，打通各个端与渠道的数据，实现实时互联与掌控。而音视频技术在其中扮演着重要的角色，如何为用户提供更低延迟、更佳体验的实时视频？如何保障海外业务的稳定性和高可用？AI技术如何为音视频赋能？
      </span>
      <br/>
     </p>
     <p style="margin-left:16px;line-height:1.5em;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:1.5em;">

        6月29日·北京
       </span>

      <span style="line-height:1.6;color:rgb(89,89,89);font-size:16px;">
       | 云+社区邀请了多位腾讯资深工程师，全面解析腾讯云音视频及融合通信技术，分享最新的低延迟技术、全新的商业直播方案技术实现以及音视频AI技术落地。

      <span style="font-size:16px;">
       <strong style="text-align:justify;">
        <span style="color:rgb(89,89,89);">
         Topic：移动直播连麦技术实践
        </span>
       </strong>

     <p style="margin-left:16px;line-height:25.6px;text-align:left;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;text-align:left;">
      <span style="color:rgb(89,89,89);font-size:16px;">
       连麦服务是直播中十分重要的一项增值业务，如何实现高质量的连麦服务，一直是众多业务开发者关注的话题，腾讯云基于自身十多年的音视频终端技术积累，结合强大的视频云服务，在连麦技术上进行了有效的实践，在此分享给大家。
      </span>
     </p>

      <span style="text-align:left;color:rgb(89,89,89);font-size:16px;">

      </span>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <strong>

        Topic：腾讯云直播PCDN加速方案
       </span>
      </strong>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <span style="font-size:16px;color:rgb(89,89,89);">
       腾讯云X-P2P是业内领先成熟的P2P产品，其中多个产品线均已成熟，包括不同平台、不同延迟场景下的P2P直播、点播P2P等，现已推广到斗鱼、企鹅电竞、英雄联盟等直播平台使用，经受住了大流量阅兵活动直播、赛事直播的考验。现在腾讯云视频P2P技术，在P2P网络传输、P2P技术对视频的侵入性、P2P网络拓扑组建上有了新的进展，在此分享给大家。
      </span>

      </span>
     </p>
     <p style="margin-left:16px;line-height:1.5em;">

     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <strong>
       <span style="font-size:16px;color:rgb(89,89,89);">
        Topic：腾讯视频云海外直播系统架构设计与最佳实践
       </span>
      </strong>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <span style="font-size:16px;color:rgb(89,89,89);">
       近几年，视频直播业务呈爆发态势，国内直播逐渐呈现疲软的态势下，越来越多的厂商把目标指向了海外，海外直播逐步发展为当前最热的风口之一。本次分享，从技术层面详细讲解腾讯视频云海外直播系统，并结合直播场景详细阐述了海外视频云直播系统架构中的实践案例。帮助开发者快速梳理海外云直播核心知识点以及部分场景化架构方案。
      </span>
     </p>

      <strong>
       <span style="font-size:16px;color:rgb(89,89,89);">
        Topic：实时音视频与PSTN结合的解决办法
       </span>
      </strong>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <span style="font-size:16px;color:rgb(89,89,89);">
       融合通信具体是指实时音视频与PSTN结合，实现多端、多人的实时通话技术。本次分享主要介绍实时音频与传统PSTN语音业务，如何融合在一起，以及融合过程中的碰到的难点和解决方案。
      </span>
     </p>

      </span>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">

     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <strong>
       <span style="font-size:16px;color:rgb(89,89,89);">
        Topic：音视频AI技术落地实践
       </span>
      </strong>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <br/>
     </p>
     <p style="margin-left:16px;line-height:25.6px;">
      <span style="font-size:16px;color:rgb(89,89,89);">
       腾讯智眸基于腾讯领先的音视频 AI 技术，为广电新媒体、短视频、泛娱乐及在线视频场景提供了一整套的视频解决方案，主要包括极速高清、视频智能识别和分析、视频在线编辑等产品，视频智能识别和分析又是其中将AI技术结合得最为紧密的解决方案，广泛应用于视频审核、视频检索、视频分析等场景中。本次分享也主要围绕AI技术在视频智能识别和分析的应用，以及实际落地过程中遇到的挑战以及解决办法。
      </span>
     </p>
