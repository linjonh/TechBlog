    <h3>
     <a id="_2">
     </a>
     前言
    </h3>
    <p>
     Conda 是一个开源的包管理系统和环境管理系统，广泛应用于数据科学和机器学习领域。本文将详细介绍如何在 CentOS 系统上安装 Conda，帮助您快速搭建开发环境。
    </p>
    <hr/>
    <h3>
     <a id="_8">
     </a>
     准备工作
    </h3>
    <p>
     在开始安装之前，请确保您的 CentOS 系统已经满足以下条件：
    </p>
    <ul>
     <li>
      已连接到互联网
     </li>
     <li>
      拥有
      <code>
       sudo
      </code>
      权限
     </li>
     <li>
      系统已安装
      <code>
       wget
      </code>
      和
      <code>
       bash
      </code>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_17">
     </a>
     安装步骤
    </h3>
    <h4>
     <a id="1__Miniconda__19">
     </a>
     1. 下载 Miniconda 安装脚本
    </h4>
    <pre><code class="prism language-bash"><span class="token function">wget</span> https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
</code></pre>
    <h4>
     <a id="2__25">
     </a>
     2. 运行安装脚本
    </h4>
    <pre><code class="prism language-bash"><span class="token function">bash</span> Miniconda3-latest-Linux-x86_64.sh
</code></pre>
    <h4>
     <a id="3__31">
     </a>
     3. 阅读并同意许可协议
    </h4>
    <p>
     在安装过程中，您将看到许可协议。按照提示输入
     <code>
      yes
     </code>
     同意协议。
    </p>
    <h4>
     <a id="4__35">
     </a>
     4. 选择安装位置
    </h4>
    <p>
     系统会询问您安装 Miniconda 的位置。默认位置通常是用户的 home 目录，如
     <code>
      ~/miniconda3
     </code>
     。您可以按回车接受默认位置，或指定其他路径。
    </p>
    <h4>
     <a id="5__Conda_39">
     </a>
     5. 初始化 Conda
    </h4>
    <p>
     安装脚本会询问是否初始化 Miniconda3。建议选择
     <code>
      yes
     </code>
     ，这将自动将 Conda 添加到您的
     <code>
      PATH
     </code>
     中。
    </p>
    <h4>
     <a id="6__43">
     </a>
     6. 激活安装
    </h4>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.bashrc
</code></pre>
    <h4>
     <a id="7__49">
     </a>
     7. 验证安装
    </h4>
    <pre><code class="prism language-bash">conda <span class="token parameter variable">--version</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_Conda__57">
     </a>
     常用 Conda 命令
    </h3>
    <ul>
     <li>
      创建新环境:
      <code>
       conda create --name myenv python=3.8
      </code>
     </li>
     <li>
      激活环境:
      <code>
       conda activate myenv
      </code>
     </li>
     <li>
      查看已有环境:
      <code>
       conda env list
      </code>
     </li>
     <li>
      安装包:
      <code>
       conda install numpy
      </code>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_66">
     </a>
     注意事项
    </h3>
    <ol>
     <li>
      建议定期更新 Conda:
      <code>
       conda update conda
      </code>
     </li>
     <li>
      谨慎使用
      <code>
       conda update --all
      </code>
      命令
     </li>
     <li>
      创建虚拟环境时指定 Python 版本
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_74">
     </a>
     高级用法
    </h3>
    <h4>
     <a id="_Python__76">
     </a>
     创建不同 Python 版本环境
    </h4>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> py38 <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.8</span>
conda create <span class="token parameter variable">-n</span> py39 <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.9</span>
conda create <span class="token parameter variable">-n</span> py310 <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.10</span>
</code></pre>
    <h4>
     <a id="_84">
     </a>
     查看依赖树
    </h4>
    <pre><code class="prism language-bash">conda list <span class="token parameter variable">--explicit</span>
</code></pre>
    <h4>
     <a id="_90">
     </a>
     解决依赖冲突
    </h4>
    <pre><code class="prism language-bash">conda <span class="token function">install</span> --no-deps packagename
</code></pre>
    <h4>
     <a id="_Mamba_96">
     </a>
     安装 Mamba
    </h4>
    <pre><code class="prism language-bash">conda <span class="token function">install</span> <span class="token parameter variable">-c</span> conda-forge mamba
</code></pre>
    <h4>
     <a id="_Mamba__Conda_102">
     </a>
     使用 Mamba 替代 Conda
    </h4>
    <pre><code class="prism language-bash">mamba create <span class="token parameter variable">-n</span> fastenv <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.9</span> numpy pandas
</code></pre>
    <h4>
     <a id="_108">
     </a>
     添加清华镜像
    </h4>
    <pre><code class="prism language-bash">conda config <span class="token parameter variable">--add</span> channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config <span class="token parameter variable">--add</span> channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
</code></pre>
    <h4>
     <a id="_115">
     </a>
     导出完整环境
    </h4>
    <pre><code class="prism language-bash">conda <span class="token function">env</span> <span class="token builtin class-name">export</span> <span class="token operator">&gt;</span> environment.yml
</code></pre>
    <h4>
     <a id="_YAML__121">
     </a>
     从 YAML 文件创建环境
    </h4>
    <pre><code class="prism language-bash">conda <span class="token function">env</span> create <span class="token parameter variable">-f</span> environment.yml
</code></pre>
    <h4>
     <a id="_127">
     </a>
     跨平台环境导出
    </h4>
    <pre><code class="prism language-bash">conda <span class="token function">env</span> <span class="token builtin class-name">export</span> --from-history <span class="token operator">&gt;</span> environment.yml
</code></pre>
    <h4>
     <a id="_base__133">
     </a>
     禁用自动激活 base 环境
    </h4>
    <pre><code class="prism language-bash">conda config <span class="token parameter variable">--set</span> auto_activate_base <span class="token boolean">false</span>
</code></pre>
    <h4>
     <a id="_139">
     </a>
     设置并发下载数
    </h4>
    <pre><code class="prism language-bash">conda config <span class="token parameter variable">--set</span> download_threads <span class="token number">5</span>
</code></pre>
    <h4>
     <a id="_145">
     </a>
     配置缓存目录
    </h4>
    <pre><code class="prism language-bash">conda config <span class="token parameter variable">--set</span> pkgs_dirs /path/to/conda/packages
</code></pre>
    <hr/>
    <h3>
     <a id="Dockerfile__153">
     </a>
     Dockerfile 示例
    </h3>
    <pre><code class="prism language-Dockerfile">FROM continuumio/miniconda3

# 复制环境文件
COPY environment.yml /tmp/
RUN conda env create -f /tmp/environment.yml
</code></pre>

    <h3>
     <a id="_181">

     检查环境状态
    </h3>

conda list
conda doctor
</code></pre>

     <a id="_191">

    </h3>

    <h3>
     <a id="_221">
     </a>
     安全建议
    </h3>
    <ul>
     <li>
      定期更新 Conda 和包
     </li>
     <li>
      使用虚拟环境隔离项目依赖
     </li>
     <li>
      避免在 base 环境直接安装包
     </li>
     <li>
      使用
      <code>
       --no-pin
      </code>
      谨慎更新
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_230">

     常见陷阱与解决方案
    </h3>

     <li>
      依赖冲突处理
     </li>

      使用
      <code>
       conda list --revisions
      </code>
      回溯
     </li>
     <li>
      创建新环境而非修改现有环境
     </li>
     <li>
      优先使用
      <code>
       conda-forge
      </code>
      频道
     </li>
    </ul>
