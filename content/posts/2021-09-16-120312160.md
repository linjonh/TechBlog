    <h3>
     <a id="_30">
     </a>
     第一步
    </h3>
    <p>
     个人建议先把反编译脚本文件夹放在根目录下，哪个盘都行，一是便于操作，二是避免了路径名存在中文字符，省的出麻烦；推荐放在C盘，但强烈建议
     <font color="red">
      不要放在
      <code>
       D盘
      </code>
     </font>
     ，原因后面再说。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e43075a37c2a078c9eaff2c64d8e24e7.png#pic_center"/>
    </p>
    <h3>
     <a id="_34">
     </a>
     第二步
    </h3>
    <p>
     找到自己想要编译的小程序包。不知道跟微信主程序安装位置有没有关系，我的是微信PC端的安装位置是默认的。小程序位置为
    </p>
    <pre><code>C:\Users\yourname\Documents\WeChat Files\Applet
</code></pre>
    <p>
     这里面有很多以wx开头的数字与字母组合的文件夹（当然前提是你打开过加载完全的小程序），点进去还会有一个以数字命名的文件夹，再点进去，你会发现有一个
     <code>
      __APP__.wxapkg
     </code>
     的文件，有的不只有它一个文件，还有一两个其他的文件，这个不清楚是什么情况，而且这种的包我不会破，抱歉。
     <br/>
     <strong>
      Tip：yourname是你电脑的用户名，例如本地账户名是zhangsan，那这里就是zhangsan；要是当时新系统用微软账户登录的，那就是你微软的登录账号的前一部分，例如123456789@qq.com，这里会显示12345，不知道为什么，大概就是这么个思路。
     </strong>
     或者直接点击右侧的
     <code>
      文档
     </code>
     也是里面直接找
     <code>
      \WeChat Files\Applet
     </code>
     也是可以的
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c0cae197b0f4339459ac5addf01a259e.png#pic_center"/>
    </p>
    <h3>
     <a id="_42">
     </a>
     第三步
    </h3>
    <p>
     打开小程序解密程序。可能有一部分人会弹出安装
     <code>
      .NET Framework
     </code>
     的弹窗，点击安装就好了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ddc8550efced6f6c5607797bf660cbc9.png#pic_center">
      <br/>
      安装完打开解密程序，点击唯一的一个按钮“选择加密小程序包”，将你想反编译的程序包的所在目录粘过去，点击打开，进行解密
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c4e7c2779cf8c0fed5b47b4a78bd975b.png#pic_center"/>
     </img>
    </p>

     第四步

    <p>

     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/56f25b0cc89e50b6a7d45d600d923be0.png#pic_center"/>
