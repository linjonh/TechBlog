    <h4>
     <a id="_0">
     </a>
     区块链起源
    </h4>
    <p>
     一个故事，讲懂什么是区块链。
     <br/>
     我们假设在很多年以前，那时候科技还不那么的发达，没有手机没有电脑，甚至还没有出现一个统一的有效的货币。在一个叫做
     <strong>
      大河村
     </strong>
     的村子里住着很多村民，因为没有货币所以他们交易的方式便是通过记账的方式。
    </p>
    <p>
     <strong>
      就像下面这张图
     </strong>
     ，村民B向村民A买了100元的猪肉，那么相应的A 的账本上便多了100，记为+100。B的账本上便少了100，记为-100。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fe7306a453ef5ee4e9330ba04751e289.png">
      <br/>
      这就是最原始的交易，村民A和B的账本上都会对这次交易记录一笔。但是如果人
      <strong>
       数量达到一定程度呢？
       <strong>
        每产生一笔交易都会实时记录下这一笔，如果当天产生
       </strong>
       很多笔交易
      </strong>
      的话那不得把人累死？所以大河村的村民就想出来了一个办法，那就是把每一天的所有交易
      <strong>
       集中起来
      </strong>
      放在
      <mark>
       公告栏
      </mark>
      里，每天晚上定时所有村民都来公告栏处查看，然后确认无误了再写进自己的账本里。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/52b481aae0e5e90fe736e51a0c3463a0.png">
       <br/>
       后来这群村民觉得每天都要自己记账，好麻烦啊，怎么办呢？于是他们就想到了一个办法，选举
       <strong>
        大河村村长
       </strong>
       来当这个抄送人，并且共同商讨出了一套记账机制，这时候他们又会想既然都有抄送人了，那是不是每天就不用去公告栏那里去看了呀？这样可以省下很多事情。于是
       <strong>
        村长
       </strong>
       便尽心尽力的当起了这个记账人
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/abe7af9bf1f03d392e94645171b044c0.png">
      <br/>
      虽然这样一下子生活方式有了很大的改变，但是账本上的支出收入照常进行的，那，村长呢？村长专心当起了这个记账人之后那他的收入哪里来呢？那是因为我们的村长在每进行一笔交易记账的时候都会收取5元作为报酬，这便成了他的收入来源。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eb6f17cc77367aec1579c36e3356674d.png">
       <br/>
       <strong>
        注：
       </strong>
       知道挖矿吗？这里你看出来什么吗？
      </img>
     </img>
    </p>
    <p>
     区块链，每个村民的账本就是这里面的一个块，
     <mark>
      那什么是链？
     </mark>
     <br/>
     每一天记账的当日余额就是你第二天账本的预期余额
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e3f0f0439ff66b266bd47e9df5ea513c.png">
      <br/>
      通过中间箭头，让每一个块，每一天的数据给联系起来共同组成了一个链状，这，就是区块链。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e0fa93dc0d3fd174c27b02cc4d4b9337.png">
       <br/>
       ① 六个村民之间每两个村民之间都会存在相互联系，都能够产生交易，这就是熟悉的
       <mark>
        点对点——P2P模式。
       </mark>
       <br/>
       ② 对于相同的账本，每一个村民手中的账本数据都是相同的，存储的数据不是在哪一个村民手中，而是在每一个人手中，这就体现了区块链的本质
       <mark>
        去中心化。
       </mark>
       <br/>
       ③ 上面说到的每一笔交易、每一笔数据都是同时存在每个人手中的，这也意味着你如果想要去修改其中一条数据的话就必须同时修改掉所有人手中的数据才能够实现，这就体现了
       <mark>
        不可篡改性
       </mark>
       <br/>
       ④ 每个人的手里存储着同样的数据，这些数据的来源不是因为你我之间的信任所以才相同，而是依赖于一套共识的机制去写入数据，也就是说我们相信的不是交易对面的你，而是交易过程的一套规范机制。这就是另一大特性
       <mark>
        共识信任机制
       </mark>
      </img>
     </img>
    </p>
    <h5>
     <a id="_25">
     </a>
     区块链四大特性
    </h5>
    <p>
     （一）
     <strong>
      点对点——P2P模式
     </strong>
     <br/>
     （二）
     <strong>
      去中心化
     </strong>
     <br/>
     （三）
     <strong>
      不可篡改
     </strong>
     <br/>
     （四）
     <strong>
      共识信任机制
     </strong>
    </p>
    <p>
     个人思考不代表绝对，仅供参考。
    </p>
