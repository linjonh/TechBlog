     3.删除（通过git bash）
    </h4>
    <blockquote>
     <p>
      在clone的项目目录下，右击打开git bash，然后先删除到暂存区，再commit,再push到远程仓库，即gitee，然后就可以看到gitee删除了之前提交的文件。
     </p>
     <p>
      git rm yxxtest.txt
      <br/>
      git commit -m '删除yxxtest'
      <br/>
      git push origin master
     </p>
    </blockquote>
