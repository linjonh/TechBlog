    <h3>
     1.电阻(主要是贴片电阻)
    </h3>
    <p>
     <a href="https://twd6onxsxva.feishu.cn/docx/UPsnd0VLLoL97fx7aXrcnW1Dn0e" rel="nofollow" title="01 基础课程-电阻">
      01 基础课程-电阻

    </p>
    <h4>
     1.电阻封装
     <img alt="" height="402" src="https://i-blog.csdnimg.cn/direct/361b45f088614eacba8b3fbef3664ced.png" width="806"/>
    </h4>
    <h4>
     2.相关参数
    </h4>
    <p>
     1.功率额定值：
    </p>
    <p>
     电阻能够长期承受的最大功率，功率过大可能导致电阻过热或损坏。封装尺寸越大，散热能力越强，功率额定值通常越高。
    </p>
    <p>
     2.容差：
    </p>
    <p>
     电阻实际值与标称阻值之间的允许偏差范围。
    </p>
    <p>
     3.电压额定值：
    </p>
    <p>
     电阻能承受的最大工作电压，超过此电压可能导致电阻击穿或损坏。封装尺寸越大的电阻通常电压额定值越大。
    </p>
    <h4>
     3.怎么通过封装上的数字看贴片电阻阻值
    </h4>
    <p>
     <img alt="" height="387" src="https://i-blog.csdnimg.cn/direct/9eea454bf01242fbac87ec970c30f88b.png" width="567"/>
    </p>
    <p>
     简单来说，3位数字和4位数字标识的最后一位数字为10的乘幂，而出现字母”R“的，字母R代表小数点。如下图所示：
    </p>
    <p>
     <img alt="" height="79" src="https://i-blog.csdnimg.cn/direct/062d9163698d4cc7849d8040bcf594fd.png" width="567"/>
    </p>
    <h4>
     4.0
     <strong>
      Ω电阻
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        0欧姆电阻
       </strong>
       ，也被称为
       <strong>
        0欧姆
       </strong>
       <strong>
        跳线
       </strong>
       或
       <strong>
        0欧姆
       </strong>
       <strong>
        桥接
       </strong>
       <strong>
        电阻
       </strong>
       ，在电路设计中主要用作一种导线或跳线，尽管其电阻接近于零。
      </p>
     </li>
     <li>
      <p>
       阻抗的作用
      </p>
      <ul>
       <li>
        <p>
         在交流电路中，阻抗决定了电流与电压之间的关系。具体来说，阻抗不仅影响电流的大小，还影响电压与电流之间的相位关系。
        </p>
       </li>
       <li>
        <p>
         在直流电路中，阻抗退化为电阻，因为电抗在直流条件下为零。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       什么是阻抗匹配
      </p>
      <ul>
       <li>
        <p>
         <strong>
          阻抗
         </strong>
         <strong>
          匹配
         </strong>
         是电子和通信工程中的一个重要概念，指的是在电路或系统中，使信号源的输出阻抗与负载的输入阻抗相匹配，以最大限度地传输功率或减少反射。阻抗匹配在射频、电缆传输、音频和其他高频电路中尤为关键。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       为什么需要阻抗匹配
      </p>
      <p>
       <strong>
        最大功率传输
       </strong>
       ：
      </p>
      <p>
       <strong>
        减少信号反射
       </strong>
       ：
      </p>
      <p>
       <strong>
        优化
       </strong>
       <strong>
        信噪比
       </strong>
       <strong>
        （
       </strong>
       <strong>
        SNR
       </strong>
       <strong>
        ）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在电路中，特别是在高频电路和射频系统中，最大化信号传输的功率非常重要。通过阻抗匹配，可以确保从源头传递到负载的功率损失最小，增加系统的效率。
        </p>
       </li>
       <li>
        <p>
         在传输线（如同轴电缆、微带线）中，如果源阻抗和负载阻抗不匹配，信号会在接口处产生反射。这种反射会导致信号失真、损耗和干扰。通过阻抗匹配，可以将反射最小化，保证信号的完整性。
        </p>
       </li>
       <li>
        <p>
         在音频和通信系统中，阻抗匹配可以优化信噪比，减少信号传输过程中的噪声干扰，提升信号的质量。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       阻抗匹配的原理
      </p>
      <p>
       <strong>
        阻抗
       </strong>
       <strong>
        匹配
       </strong>
       的原理是为了确保在电路中最大化功率传输并最小化信号反射。其核心思想是使信号源的输出阻抗与负载的输入阻抗相等或相互共轭。当阻抗匹配时，以下两个主要效应得以实现：
      </p>
      <h4>
       5.经验取值
      </h4>
      <ul>
       <li>
        <p>
         <strong>
          最大功率传输
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           根据最大功率传输定理，当信号源的输出阻抗（Zs）与负载的输入阻抗（ZL）相等时，负载可以接收到最大的功率。这意味着在此条件下，电路中的功率传输效率最高，能量损耗最小。
          </p>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          最小化信号反射
         </strong>
         ：
         <img alt="" height="328" src="https://i-blog.csdnimg.cn/direct/d8750f46156640a5913b1e80d8c58e28.png" width="517"/>
        </p>
        <ol>
         <li>
          <p>
           在高频或高速信号传输中，如果源阻抗和负载阻抗不匹配，信号在传输线上会发生反射，导致信号失真和功率损失。通过阻抗匹配，可以消除或显著减少反射，确保信号完整地传输到负载端。
          </p>
         </li>
        </ol>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     上拉电阻我们一般都选择的是 4.7K、10K等
    </p>
    <h3>
     2.电容
    </h3>
    <p>
     <a href="https://twd6onxsxva.feishu.cn/docx/VcjjdpMAGouAKFxd8DZcwlMGnbg" rel="nofollow" title="02 基础课程-电容">
      02 基础课程-电容
     </a>
    </p>
    <h4>
     1.特性
    </h4>
    <p>
     隔直通交。
    </p>
    <p>
     小电容：通高频，阻低频
    </p>
    <p>
     打电容：通低频，阻高频
    </p>
    <h4>
     2.电容选型的经验取值
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        电源
       </strong>
       <strong>
        滤波
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 在电源电路中，滤波电容用于平滑电压纹波，减少噪声。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           <strong>
            输入
           </strong>
           <strong>
            滤波
           </strong>
           : 对于AC-DC电源转换器的输入滤波，常用的电解电容取值在220μF到1000μF之间，具体取值与输入电压和负载电流有关。
          </p>
         </li>
         <li>
          <p>
           <strong>
            输出
           </strong>
           <strong>
            滤波
           </strong>
           : 对于DC-DC转换器或线性稳压器的输出滤波，通常使用较小的电容，如10μF到100μF的电解电容，同时并联一个0.1μF到1μF的陶瓷电容以滤除高频噪声。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        去耦电容
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 去耦电容用于电源与地之间，减少电源线上的噪声，稳定器件的电压。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           在数字电路中，通常每个IC的电源引脚旁边放置一个0.1μF的陶瓷电容。
          </p>
         </li>
         <li>
          <p>
           对于高速数字电路，可能会增加1μF到10μF的电容以提供更强的去耦效果。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        振荡电路
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 在振荡电路中，电容与电感或电阻一起决定振荡频率。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           对于RC振荡器或LC振荡器，电容的取值通常在1pF到100nF之间，具体取值需要根据所需的振荡频率计算。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        定时电路
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 电容器用于定时电路，如555定时器，用于决定脉冲的时间长度。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           定时电容通常选择在1nF到100μF之间，根据公式T=RC计算需要的时间常数。较大电容值产生较长的定时时间。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        信号
       </strong>
       <strong>
        耦合
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 电容器用于耦合交流信号，阻断直流分量。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           在音频电路中，耦合电容的取值通常为0.1μF到10μF，以确保音频信号能够完整通过。
          </p>
         </li>
         <li>
          <p>
           在射频电路中，电容值可能在pF级别，以允许高频信号通过而阻止低频或直流成分。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        高频
       </strong>
       <strong>
        滤波
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 用于抑制高频噪声，保护敏感元件。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           通常选择0.01μF到0.1μF的陶瓷电容，这些值在高频情况下能有效滤除不需要的信号。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        旁路
       </strong>
       <strong>
        电容
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 用于给放大器、比较器等电路中的器件提供稳定的直流工作点，同时消除直流偏置。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           旁路电容值通常较大，如10μF到100μF，用于滤除电源上的低频纹波和噪声。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        储能电容
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          应用
         </strong>
         : 在电源断开或电流突增时，提供瞬间电能。
        </p>
       </li>
       <li>
        <p>
         <strong>
          经验取值
         </strong>
         :
        </p>
        <ul>
         <li>
          <p>
           储能电容的取值较大，通常在100μF以上，甚至达到数千μF，用于存储足够的电荷应对突发需求。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     3.耐压值
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        工作电压
       </strong>
       : 电容的耐压值应至少是工作电压的1.5倍到2倍。这是为了保证在正常工作时电容不会过早老化或击穿。
      </p>
     </li>
     <li>
      <p>
       <strong>
        峰值电压
       </strong>
       : 如果电路中存在较大的电压波动或瞬态电压（例如电源滤波或耦合应用），耐压值还需考虑到这些情况，通常要更高一些。
      </p>
     </li>
    </ul>
    <h4>
     4.应用
    </h4>
    <p>
     电容具体电路应用查看引用文章。
    </p>
    <h3>
     3.电感
    </h3>
    <p>
     <a href="https://twd6onxsxva.feishu.cn/docx/Cw7md9rOSoe02Nx4oi3cb0Lxnrh" rel="nofollow" title="03 基础课程电感">
      03 基础课程电感
     </a>
    </p>
    <h4>
     1.特性
    </h4>
    <p>
     <strong>
      通直阻交：
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        电感阻交流
       </strong>
       ：电感器对交流电产生感抗，阻碍其通过，尤其是高频交流信号。
      </p>
     </li>
     <li>
      <p>
       <strong>
        电感通直流
       </strong>
       ：电感器对直流电没有感抗，因此直流电可以自由通过。
      </p>
     </li>
    </ul>
    <p>
    </p>
    <p>
     <strong>
      阻高频，通低频：
     </strong>
    </p>
    <p>
     $$X_L=2\pi fl$$
    </p>
    <ul>
     <li>
      <p>
       XL 是感抗，单位是欧姆（Ω）。
      </p>
     </li>
     <li>
      <p>
       f 是交流电的频率，单位是赫兹（Hz）。
      </p>
     </li>
     <li>
      <p>
       L 是电感量，单位是亨利（H）。
      </p>
     </li>
    </ul>
    <p>
