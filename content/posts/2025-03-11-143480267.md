    <h3>
     一、环境变量设置
    </h3>
    <h5>
     1.1 命令行环境变量设置
    </h5>
    <p>
     为了打印XML拓扑文件，需要设置NCCL的环境变量NCCL_TOPO_DUMP_FILE。这个环境变量指定了XML拓扑文件的输出路径和文件名。例如，可以在命令行中设置此环境变量：
    </p>
    <pre><code class="language-bash">export NCCL_TOPO_DUMP_FILE=/path/to/output/topo.xml</code></pre>
    <h5>
     1.2 代码中环境变量设置
    </h5>
    <p>
     或者，在代码中设置此环境变量（假设使用的是C/C++）：
    </p>
    <pre><code class="language-cpp">setenv("NCCL_TOPO_DUMP_FILE", "/path/to/output/topo.xml", 1);</code></pre>
    <pre><code class="language-cpp">#include &lt;stdlib.h&gt;
#include &lt;nccl.h&gt;

int main() {

    // 设置环境变量
    setenv("NCCL_TOPO_DUMP_FILE", "/path/to/output/topo.xml", 1);

 

    // 初始化NCCL通信器（这里以单GPU为例）
    ncclComm_t comm;
    int nDevices = 1;
    cudaSetDevice(0); // 设置CUDA设备
    ncclCommInitAll(comm, nDevices, NULL);


    // ... 进行其他NCCL操作 ...


    // 销毁NCCL通信器
    ncclCommDestroy(comm);

    return 0;
}</code></pre>

    <h3>
     四、拓扑文件保存代码
    </h3>
    <p>
     1、如果设置了环境变量
     <strong>
      NCCL_TOPO_DUMP_FIL
     </strong>
     E调用保存拓扑文件函数
     <strong>
      ncclTopoDumpXmlToFile
     </strong>
    </p>
    <pre><code class="language-cpp">  xmlTopoFile = ncclGetEnv("NCCL_TOPO_DUMP_FILE");
  if (xmlTopoFile &amp;&amp; comm-&gt;rank == ncclParamTopoDumpFileRank()) {
    INFO(NCCL_ENV, "NCCL_TOPO_DUMP_FILE set by environment to %s", xmlTopoFile);
    NCCLCHECK(ncclTopoDumpXmlToFile(xmlTopoFile, xml));
  }
</code></pre>
    <p>
     2、ncclTopoDumpXmlToFile中打开文件，并调用
     <strong>
      ncclTopoDumpXmlRec
     </strong>
     向文件中写入内容
    </p>
    <pre><code class="language-cpp">ncclResult_t ncclTopoDumpXmlToFile(const char* xmlTopoFile, struct ncclXml* xml) {
  FILE* file = fopen(xmlTopoFile, "w");
  if (file == NULL) {
    WARN("Unable to open %s, not dumping topology.", xmlTopoFile);
    return ncclSuccess;
  }
  NCCLCHECK(ncclTopoDumpXmlRec(0, file, xml-&gt;nodes));
  fclose(file);
  return ncclSuccess;
}</code></pre>
    <p>
     3、ncclTopoDumpXmlRec向文件中写入内容
    </p>
    <pre><code class="language-cpp">ncclResult_t ncclTopoDumpXmlRec(int indent, FILE* file, struct ncclXmlNode* node) {
  for (int i=0; i&lt;indent; i++) fprintf(file, " ");
  fprintf(file, "&lt;%s", node-&gt;name);

  for (int a=0; a&lt;node-&gt;nAttrs; a++) {
    fprintf(file, " %s=\"%s\"", node-&gt;attrs[a].key, node-&gt;attrs[a].value);
  }
  if (node-&gt;nSubs == 0) {
    fprintf(file, "/&gt;\n");
  } else {
    fprintf(file, "&gt;\n");
    for (int s=0; s&lt;node-&gt;nSubs; s++) {
      NCCLCHECK(ncclTopoDumpXmlRec(indent+2, file, node-&gt;subs[s]));
    }
    for (int i=0; i&lt;indent; i++) fprintf(file, " ");
    fprintf(file, "&lt;/%s&gt;\n", node-&gt;name);
  }
  return ncclSuccess;
}</code></pre>
