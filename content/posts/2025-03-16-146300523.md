     <br/>
     转矩。直流电机是通过两个磁场的相互作用产生旋转。定子上装设了一对直流励磁的静止的主磁极
     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>

     <br/>
     用得更为广泛。

     <br/>
