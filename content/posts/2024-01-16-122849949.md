    <h4>
     <a id="_0">
     </a>
     问题
    </h4>
    <p>
     idea 导入 java 项目，java文件图标显示带“J”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0e864b767446be41114268d84810d17a.png#pic_center"/>
    </p>
    <h4>
     <a id="_3">
     </a>
     解决方案
    </h4>
    <p>
     点击File &gt;&gt; Project Structure：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b4c2775793489f137588a336810039e.png#pic_center"/>
    </p>
    <p>
     在Project Structure页面选择 Project Settings 下的 Modules，点击右侧删除项目 Contect Root
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5ad5d6a83190ba82500f8156000e0e61.png#pic_center"/>
    </p>
    <p>
     点击 Add Content Root 选择项目路径 java 文件夹
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7f80338809a25b0a67ac8b457170969.png#pic_center"/>
    </p>
    <p>
     添加后如图标识完成
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e4534d11649ec07884469deb7ac5a910.png#pic_center"/>
    </p>
    <p>
     确认会到项目界面，显示正常
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a17cdf819739ff9e62b7cb29c5243e34.png#pic_center"/>
    </p>
