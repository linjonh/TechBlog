    <h3>
     预处理指令与宏定义
    </h3>
    <h4>
     1. 预处理指令
    </h4>
    <p>
     <br/>
     预处理指令以" # "开头，是在 编译之前 由预处理器处理的指令。预处理器负责以下任务：
     <br/>
     - 头文件包含
     <br/>
     - 宏定义
     <br/>
     - 条件编译
    </p>
    <h4>
     2. 头文件包含
    </h4>
    <p>
     <br/>
     - #include：用于包含头文件。
     <br/>
     - 头文件通常包含以下内容：
     <br/>
     - 函数声明
     <br/>
     - 宏定义
     <br/>
     - 类型定义
     <br/>
     - 功能：使得程序可以使用这些功能。
    </p>
    <h4>
     3. 宏定义
    </h4>
    <p>
     <br/>
     - #define：用于定义宏。
     <br/>
     - 对象宏：用于定义常量。
     <br/>
     - 函数宏：用于定义简单的函数。
    </p>
    <p>
     <span style="color:#fe2c24">
      #define  代表名称  被代表  --&gt;#define  name  stuff
     </span>
     <br/>
     - 示例：
     <br/>
    </p>
    <pre><code>  #define M 100          // 定义常量 M，值为 100
  #define STR "hehe"     // 定义字符串常量 STR
  #define reg register   // 定义关键字 reg

  printf("M 的值为：%d\n", M);       // 输出：M 的值为：100
  printf("STR 的值为：%s\n", STR);   // 输出：STR 的值为：hehe
</code></pre>
    <p>
    </p>
    <h3>
     位运算
    </h3>
    <p>
     位运算符用于 **直接操作整数在内存中存储的二进制位**。以下是常见的位运算符及其功能：
    </p>
    <h4>
     1. 按位与（`&amp;`）
    </h4>
    <p>
     <br/>
     - 功能：将两个操作数的对应位进行比较，只有当两个操作数对应位都为 1 时，结果的对应位才为 1，否则为 0。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5; // 二进制：00000101
  int b = 3; // 二进制：00000011
  int result = a &amp; b; // result 的值为 1，二进制：00000001
  printf("a &amp; b = %d\n", result); // 输出：a &amp; b = 1</code></pre>
    <h4>
     2. 按位或（`|`）
    </h4>
    <p>
     <br/>
     - 功能：将两个操作数的对应位进行比较，只要两个操作数对应位中有一个为 1，结果的对应位就为 1，否则为 0。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5; // 二进制：00000101
  int b = 3; // 二进制：00000011
  int result = a | b; // result 的值为 7，二进制：00000111
  printf("a | b = %d\n", result); // 输出：a | b = 7</code></pre>
    <h4>
     3. 按位异或（`^`）
    </h4>
    <p>
     <br/>
     - 功能：将两个操作数的对应位进行比较，当两个操作数对应位不同时，结果的对应位为 1，相同时为 0。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5; // 二进制：00000101
  int b = 3; // 二进制：00000011
  int result = a ^ b; // result 的值为 6，二进制：00000110
  printf("a ^ b = %d\n", result); // 输出：a ^ b = 6</code></pre>
    <h4>
     4. 按位取反（`~`）
    </h4>
    <p>
     <br/>
     - 功能：将操作数的每一位进行取反，即 0 变为 1，1 变为 0。
     <br/>
     - 注意：在有符号整数中，取反会影响符号位，需要考虑补码表示。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5; // 二进制：00000101 (8位)
  int result = ~a; // result 的值为 -6，二进制：11111010 (补码)
  printf("~a = %d\n", result); // 输出：~a = -6</code></pre>
    <h4>
     5. 左移（`&lt;&lt;`）
    </h4>
    <p>
     <br/>
     - **功能**：将操作数的二进制位向左移动指定的位数，右边补 0。左移一位相当于乘以 2。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5; // 二进制：00000101
  int result = a &lt;&lt; 2; // result 的值为 20，二进制：00010100
  printf("a &lt;&lt; 2 = %d\n", result); // 输出：a &lt;&lt; 2 = 20</code></pre>
    <h4>
     6. 右移（`&gt;&gt;`）
    </h4>
    <p>
     <br/>
     - 功能：将操作数的二进制位向右移动指定的位数。
     <br/>
     - 逻辑右移：左边补 0。
     <br/>
     - 算术右移：左边补符号位。
     <br/>
     - 注意：具体使用哪种右移方式取决于编译器和数据类型。
     <br/>
     - 示例：
     <br/>
    </p>
    <pre><code>  int a = 5; // 二进制：00000101
  int result = a &gt;&gt; 2; // result 的值为 1，二进制：00000001
  printf("a &gt;&gt; 2 = %d\n", result); // 输出：a &gt;&gt; 2 = 1

  int b = -5; // 二进制：11111011 (补码)
  int result2 = b &gt;&gt; 2; // result2 的值取决于编译器，可能是 -2 或其他值
  printf("b &gt;&gt; 2 = %d\n", result2); // 输出：b &gt;&gt; 2 = -2 (取决于编译器)</code></pre>
    <p>
     注意： 运算符优先级最⾼的是（⼀元运算符），如 ++ 、 -- 、 ! 等，其次是（算术运算符），最后是 赋值运算符。（
     <span style="color:#fe2c24">
      计算机会按照优先级来计算，就跟数学上的加减乘除一样
     </span>
     <span style="color:#0d0016">
      ）
     </span>
    </p>

    <h3>
     位操作技巧
    </h3>
    <p>
     这里是我新认识的通过二进制位进行操作的。
    </p>
    <h4>
     1. 位设置
    </h4>
    <p>
     <br/>
     - 功能：将一个整数的特定位设置为 1。
     <br/>
     - 示例：
    </p>
    <pre><code>  unsigned char num = 0b00000000; // 初始值为 0
  unsigned char mask = 0b00000100; // 掩码
  unsigned char newNum = num | mask; // 设置第 2 位为 1
  printf("New number: %d\n", newNum); // 输出：New number: 4</code></pre>
    <h4>
     2. 位清除
    </h4>
    <p>
     <br/>
     - 功能：将一个整数的特定位设置为 0。
     <br/>
     - 示例：
    </p>
    <pre><code>  unsigned char num = 0b11111111; // 初始值为 255
  unsigned char mask = ~0b00000100; // 掩码
  unsigned char newNum = num &amp; mask; // 清除第 2 位
  printf("New number: %d\n", newNum); // 输出：New number: 251</code></pre>
    <h4>
     3. 判断奇偶数
    </h4>
    <p>
     <br/>
     - **功能**：使用位运算快速判断一个整数是奇数还是偶数。
     <br/>
     - **原理**：如果 `num &amp; 1` 的结果为 1，则为奇数；否则为偶数。
     <br/>
     - 示例：
    </p>
    <pre><code>  int num = 7;
  if (num &amp; 1) {
      printf("%d 是奇数\n", num); // 输出：7 是奇数
  } else {
      printf("%d 是偶数\n", num);
  }

  int num2 = 6;
  if (num2 &amp; 1) {
      printf("%d 是奇数\n", num2);
  } else {
      printf("%d 是偶数\n", num2); // 输出：6 是偶数
  }</code></pre>
    <h4>
     4. 交换两个变量的值
    </h4>
    <p>
     <br/>
     - 功能：使用位运算在不使用额外变量的情况下交换两个变量的值。
     <br/>
     - 示例：
    </p>
    <pre><code>  int a = 5;
  int b = 3;
  printf("交换前：a = %d, b = %d\n", a, b); // 输出：交换前：a = 5, b = 3

  a ^= b; // a = 5 ^ 3 = 6 (0101 ^ 0011 = 0110)
  b ^= a; // b = 3 ^ 6 = 5 (0011 ^ 0110 = 0101)
  a ^= b; // a = 6 ^ 5 = 3 (0110 ^ 010</code></pre>
