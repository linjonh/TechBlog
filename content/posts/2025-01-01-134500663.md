    <p>
     将Excel文件转换为PDF可以方便储存表格数据，此外在打印或共享文档时也能确保表格样式布局等在不同设备和操作系统上保持一致。今天给大家分享一个使用第三方Python库Spire.XLS for Python 实现Excel转PDF的简单方法。
    </p>
    <h4>
     <a id="PythonExcelPDF_4">
     </a>
     Python中Excel转PDF的实现步骤
    </h4>
    <p>
     1. 首先，通过pip命令来安装依赖库；
    </p>
    <pre><code class="prism language-py">pip install Spire<span class="token punctuation">.</span>XLS
</code></pre>
    <p>
     2. 安装完成后，导入所需库；
    </p>
    <pre><code class="prism language-py"><span class="token keyword">from</span> spire<span class="token punctuation">.</span>xls <span class="token keyword">import</span> <span class="token operator">*</span>
<span class="token keyword">from</span> spire<span class="token punctuation">.</span>common <span class="token keyword">import</span> <span class="token operator">*</span>
</code></pre>
    <p>
     3. 加载Excel文档，然后可以通过 Spire.XLS for Python 提供的
     <strong>
      Workbook.SaveToFile()
     </strong>
     或
     <strong>
      Worksheet.SaveToPdf()
     </strong>
     来将Excel工作簿或某个指定的Excel工作表转换为PDF格式。
    </p>
    <p>
     4. 在转换时还可以通过
     <strong>
      PageSetup
     </strong>
     类来进行页面设置。
    </p>
    <p>
     <strong>
      <em>
       代码示例
      </em>
     </strong>
    </p>
    <h4>
     <a id="1_PythonExcelPDF_25">
     </a>
     1 .使用Python将Excel文件转为PDF格式（每个工作表显示为单独的一页）
    </h4>
    <pre><code class="prism language-py"><span class="token keyword">from</span> spire<span class="token punctuation">.</span>xls <span class="token keyword">import</span> <span class="token operator">*</span>
<span class="token keyword">from</span> spire<span class="token punctuation">.</span>common <span class="token keyword">import</span> <span class="token operator">*</span>
 
<span class="token comment"># 创建Workbook对象</span>
workbook <span class="token operator">=</span> Workbook<span class="token punctuation">(</span><span class="token punctuation">)</span>
 
<span class="token comment"># 加载Excel文档</span>
workbook<span class="token punctuation">.</span>LoadFromFile<span class="token punctuation">(</span><span class="token string">"数据.xlsx"</span><span class="token punctuation">)</span>
 
<span class="token comment"># 设置工作表适应页面</span>
workbook<span class="token punctuation">.</span>ConverterSetting<span class="token punctuation">.</span>SheetFitToPage <span class="token operator">=</span> <span class="token boolean">True</span>
 
<span class="token comment"># 将Excel转换为PDF</span>
workbook<span class="token punctuation">.</span>SaveToFile<span class="token punctuation">(</span><span class="token string">"ToPDF.pdf"</span><span class="token punctuation">,</span> FileFormat<span class="token punctuation">.</span>PDF<span class="token punctuation">)</span>
workbook<span class="token punctuation">.</span>Dispose<span class="token punctuation">(</span><span class="token punctuation">)</span>

</code></pre>
<p>
<img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/3cc89f2b05e952a1cea765c8c2e85e91.png"/>
</p>
<h4>
<a id="2_PythonExcelPDF_52">
</a> 2. 使用 Python 将 Excel 中每个工作表转换为单独的 PDF 文件
</h4>
<pre><code class="prism language-py"><span class="token keyword">from</span> spire<span class="token punctuation">.</span>xls <span class="token keyword">import</span> <span class="token operator">_</span>
<span class="token keyword">from</span> spire<span class="token punctuation">.</span>common <span class="token keyword">import</span> <span class="token operator">_</span>

<span class="token comment"># 创建 Workbook 对象</span>
workbook <span class="token operator">=</span> Workbook<span class="token punctuation">(</span><span class="token punctuation">)</span>

<span class="token comment"># 加载 Excel 文件</span>
workbook<span class="token punctuation">.</span>LoadFromFile<span class="token punctuation">(</span><span class="token string">"数据.xlsx"</span><span class="token punctuation">)</span>

<span class="token comment">#遍历 Excel 工作簿中的所有工作表</span>
<span class="token keyword">for</span> sheet <span class="token keyword">in</span> workbook<span class="token punctuation">.</span>Worksheets<span class="token punctuation">:</span>

<span class="token comment">#将每个工作表都保存为一个单独的 PDF 文件</span>
FileName <span class="token operator">=</span> sheet<span class="token punctuation">.</span>Name <span class="token operator">+</span> <span class="token string">".pdf"</span>
sheet<span class="token punctuation">.</span>SaveToPdf<span class="token punctuation">(</span>FileName<span class="token punctuation">)</span>
workbook<span class="token punctuation">.</span>Dispose<span class="token punctuation">(</span><span class="token punctuation">)</span>

</code></pre>
<p>

</p>
<h4>
<a id="3_PythonExcelPDF_78">
</a> 3. 使用 Python 将指定 Excel 工作表转为 PDF 格式
</h4>
<pre><code class="prism language-py"><span class="token keyword">from</span> spire<span class="token punctuation">.</span>xls <span class="token keyword">import</span> <span class="token operator">_</span>
<span class="token keyword">from</span> spire<span class="token punctuation">.</span>common <span class="token keyword">import</span> <span class="token operator">_</span>

<span class="token comment"># 创建 Workbook 对象</span>
workbook <span class="token operator">=</span> Workbook<span class="token punctuation">(</span><span class="token punctuation">)</span>

<span class="token comment"># 加载 Excel 文件</span>
workbook<span class="token punctuation">.</span>LoadFromFile<span class="token punctuation">(</span><span class="token string">"数据.xlsx"</span><span class="token punctuation">)</span>

<span class="token comment"># 获取第二个工作表</span>
sheet <span class="token operator">=</span> workbook<span class="token punctuation">.</span>Worksheets<span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span>

<span class="token comment"># 获取 PageSetup 对象</span>
pageSetup <span class="token operator">=</span> sheet<span class="token punctuation">.</span>PageSetup

<span class="token comment"># 设置页边距</span>
pageSetup<span class="token punctuation">.</span>TopMargin <span class="token operator">=</span> <span class="token number">0.3</span>
pageSetup<span class="token punctuation">.</span>BottomMargin <span class="token operator">=</span> <span class="token number">0.3</span>
pageSetup<span class="token punctuation">.</span>LeftMargin <span class="token operator">=</span> <span class="token number">0.3</span>
pageSetup<span class="token punctuation">.</span>RightMargin <span class="token operator">=</span> <span class="token number">0.3</span>

<span class="token comment"># 设置页面尺寸</span>
pageSetup<span class="token punctuation">.</span>PaperSize <span class="token operator">=</span> PaperSizeType<span class="token punctuation">.</span>PaperA3

<span class="token comment"># 设置转换时工作表适应页面</span>
workbook<span class="token punctuation">.</span>ConverterSetting<span class="token punctuation">.</span>SheetFitToPage <span class="token operator">=</span> <span class="token boolean">True</span>

<span class="token comment"># 将该工作表保存为 PDF 文件</span>
sheet<span class="token punctuation">.</span>SaveToPdf<span class="token punctuation">(</span><span class="token string">"Excel 工作表转 PDF.pdf"</span><span class="token punctuation">)</span>
