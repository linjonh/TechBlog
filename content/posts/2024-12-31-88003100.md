     </h5>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo" rel="nofollow">
       matomo
      </a>
      - 类似于googel分析的项目

     <h5 class="heading">
      5. RPC 框架
     </h5>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgrpc.io%2F" rel="nofollow">
       grpc
      </a>
      - googel开源的rpc框架，基于protobuf,跨语言．
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fbrpc%2Fbrpc" rel="nofollow">
       brpc
      </a>
      - 百度开源的rpc框架
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2FTencent%2FTars" rel="nofollow">
       tars
      </a>
      - 腾讯开源的rpc框架
      <br/>
      <a href="https://link.juejin.im?target=http%3A%2F%2Fdubbo.incubator.apache.org%2F" rel="nofollow">
       dubbo
      </a>
      - 阿里开源的rpc框架，国内java圈很火．
      <br/>
     </p>
     <h5 class="heading">
      6. 分布式协调系统
     </h5>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fapache%2Fzookeeper" rel="nofollow">
       zookeeper
      </a>
      - 简单来说就是一个java实现的强一致的kv存储系统．
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fcoreos%2Fetcd" rel="nofollow">
       etcd
      </a>
      - go版zookeeper.
      <br/>
     </p>
     <h5 class="heading">
      7. 负载均衡
     </h5>
     <p>
      <a href="https://link.juejin.im?target=http%3A%2F%2Fnginx.org%2F" rel="nofollow">
       nginx
      </a>
      - 四层，七层代理
      <br/>
      <a href="https://link.juejin.im?target=http%3A%2F%2Fwww.haproxy.org%2F" rel="nofollow">
       haproxy
      </a>
      - 类似于nginx
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Ftraefik.io%2F" rel="nofollow">
       traefik
      </a>
      - go语言编写，天然支持集成k8s.
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgetkong.org%2F" rel="nofollow">
       kong
      </a>
      - 网关，基于nginx的扩展，更方便进行管理．
      <br/>
     </p>
     <h5 class="heading">
      8. 服务器运维工具
     </h5>
     <p>
      <a href="https://link.juejin.im?target=http%3A%2F%2Fwww.jumpserver.org%2F" rel="nofollow">
       jumpserver
      </a>
      - 服务器登录堡垒机
      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fdocs.saltstack.com%2Fen%2Flatest%2F" rel="nofollow">
       saltstack
      </a>
      - 服务器批量管理工具
      <br/>
      <a href="https://link.juejin.im?target=http%3A%2F%2Fdocs.ansible.com%2F" rel="nofollow">
       ansible
      </a>
      - 服务器批量管理工具，基于ssh,相比saltstack来说，不用安装agent.
      <br/>
     </p>

      9.1 对象存储
     </h6>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fminio%2Fminio" rel="nofollow">

      <br/>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fceph%2Fceph" rel="nofollow">
       ceph

      <br/>

      9.2 缓存
     </h6>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fantirez%2Fredis" rel="nofollow">
       redis
      </a>
      - 支持多少数据结构，集群性扩展，高性能．
      <br/>
     </p>
     <h6 class="heading">
      9.3 NewSql 数据库
     </h6>

      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fpingcap%2Ftidb" rel="nofollow">
       tidb
      </a>

      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fcockroachdb%2Fcockroach" rel="nofollow">
       cockroach

      - 类似tidb
      <br/>
     </p>

      10. 容器与资源编排
     </h5>
     <p>
      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby" rel="nofollow">
       moby
      </a>
      - 容器，曾用名docker.

     </h5>

      <a href="https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2Fprometheus%2Fprometheus" rel="nofollow">
       prometheus
