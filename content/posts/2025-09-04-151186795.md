# 内网穿透的应用-小白也能建博客：Halo+cpolar让个人网站从梦想变现实

### 前言

“想建个博客记录生活，却被域名、服务器、代码搞得头大？”——大学生小林的烦恼道出了很多普通人的心声。商业建站平台要么按流量计费，要么塞满广告，免费版功能阉割；自己买服务器又要学Linux命令，改个主题都要查半天教程。Halo与cpolar内网穿透服务的组合，就像给建站小白配备了"自动导航"，鼠标点点就能搭建专业博客，手机都能管理文章，成本几乎为零，彻底告别"想建站却不敢动手"的尴尬。

普通人对博客的需求其实很简单：写文章方便、页面好看、别人能访问。但现实总是骨感：WordPress后台像迷宫，Typecho要改代码，GitHub Pages连评论功能都没有。Halo首先解决"易用性"问题——打开网页就是熟悉的编辑器，拖拖拽拽就能换主题，内置几十种插件一键安装，连大学生都能快速搞定。配合cpolar后，"别人能访问"的问题迎刃而解：在家用旧电脑搭建博客，生成一个公网链接就能分享给全世界，不用买域名不用备案，甚至能绑定自己的个性域名。所有操作就像发朋友圈一样简单，却拥有完全属于自己的数字空间。

操作简单到难以置信是最大亮点。很多人一听"本地部署+内网穿透"就觉得高深，其实Halo的Docker安装只需复制一条命令，cpolar配置更是点几下鼠标的事，全程不用碰代码。“这种"零门槛"的体验，让建博客从程序员的专利变成普通人的玩具，每个人都能轻松拥有自己的"小网站”，记录生活、分享知识、连接同好——这或许就是互联网最初的美好模样。

本篇文章介绍如何在CentOS下使用Docker部署开源建站工具Halo,并结合cpolar内网穿透工具实现个人博客的公网访问。

![0b78a413f513086c5dd5095f540f9fd](https://i-blog.csdnimg.cn/img_convert/60817c26da27f92256961dc4659f19f6.png)

### 1. Docker部署Halo

#### 1.1 检查Docker版本

* **本地环境操作系统：**CentOS 7 安装Docker

  #### 如果未安装Docker可参考：

  安装软件包（提供实用程序）并设置存储库

  ```shell
  $ sudo yum install -y yum-utils
  $ sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

  ```

  安装Docker引擎

  ```shell
  sudo yum install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

  ```

  启动Docker

  ```
  sudo systemctl start docker

  ```

  通过运行映像来验证 Docker 引擎安装是否成功

  ```shell
  sudo docker run hello-world

  ```

#### 已安装Docker步骤：

* **启动Docker：**

```shell
 systemctl start docker

```

* **检查docker版本**

```shell
docker -v

```

* **检查docker compose版本：**确保2.0以上版本

```shell
docker compose version

```

* **下载Halo镜像：**在docker hub下载Halo-V2.10版本镜像

```shell
docker pull halohub/halo:2.10

```

#### 1.2 在Docker中部署Halo

* **创建挂载目录**

```shell
mkdir -p /data/halo && cd /data/halo

```

![07a005be5cd19fdf5a377210cf5cc4c](https://i-blog.csdnimg.cn/img_convert/d83f04108dd166e1e250fb2ef95826a1.png)

* **创建Halo容器:**

Halo 2与1.x 版本不兼容；

此命令默认使用自带的 H2 Database 数据库，另外可以额外单独使用Mysql数据库进行连接。

```shell
docker run -it -d --name halo -p 8090:8090 -v /data/halo/.halo2:/root/.halo2 halohub/halo:2.10

```

![dd6d2a9d946f02352574cd914fe3413](https://i-blog.csdnimg.cn/img_convert/94c1e98edfa559f112cb787f17d7f286.png)

* **查看halo容器状态**：检查halo容器状态，确保halo容器正常启动

```shell
docker ps

```

![0e79a5a0e6b242e6a6e73543831ccd7](https://i-blog.csdnimg.cn/img_convert/2f8d892e1bb102495c38161e516b8bc7.png)

* **检查halo容器日志:** 检查halo容器运行日志，确保halo服务正常启动。

```shell
docker logs halo

```

![d2379dad6f69ba006bea146afc6b874](https://i-blog.csdnimg.cn/img_convert/b284e7ec4f076dad328d7a01e65953a8.png)

* **查看本地IP地址**

```shell
ifconfig

```

* **进入Halo初始化页面**

> 访问地址：http://192.168.149.142:8090/console/setup，将IP替换为自己服务器IP地址。

**站点名称：**myweb  
 **邮箱：**admin@qq.com  
 **用户名：**admin  
 **密码：**自定义  
 点击初始化即可

![a0f433681aae3d111b7523508784e38](https://i-blog.csdnimg.cn/img_convert/1aa07cf7e8226c03c3da04003d75bb2d.png)

* **登录Halo：**输入账号和密码，登录halo，进入到仪表盘界面

![e05facc1f24d0ea066d9a9757fd4fe8](https://i-blog.csdnimg.cn/img_convert/274cb9960e302bb0ef1fe30e9a1576e0.png)

![4e37c2c4e250ebfa4bc69eb97dc3b6b](https://i-blog.csdnimg.cn/img_convert/a4abbe2f4d65f6efd8b01810c0e51765.png)

接下来举个例子我们创作一篇文章，在Halo后台管理页面，文章模块，点击“新建”

![05f5f06359138c94a5dd5e9e743b14b](https://i-blog.csdnimg.cn/img_convert/23ffbb29c3adb3cacbac005c945c2d52.png)

编辑完文章后，点击发布，将文章进行发布，在文章列表中即可看到发布过的文章

我们成功在本地部署了Halo，通过访问挂载的8089端口即可看到Halo首页界面，并创作了第一篇文章，如果我们想把创作好的个人博客发布至公网分享给身边人点击查看，那么就需要借助cpolar内网穿透工具了，接下来我们安装cpolar内网穿透工具，实现无公网环境远程访问！

### 2. Linux安装Cpolar

#### 2.1 打开服务器防火墙

启动防火墙

```shell
systemctl start firewalld

```

查看防火墙状态

```shell
systemctl status firewalld

```

centos7 防火墙添加端口：【单个】

```shell
firewall-cmd --zone=public --add-port=9200/tcp --permanent

```

重新载入

```shell
firewall-cmd --reload

```

查看

```shell
firewall-cmd --zone=public --list-ports

```

#### 2.2 安装cpolar内网穿透

上面在本地Docker中成功部署了Halo,并局域网访问成功,下面我们在Linux安装Cpolar内网穿透工具,通过Cpolar 转发本地端口映射的http公网地址,我们可以很容易实现远程访问,而无需自己注册域名购买云服务器.下面是安装cpolar步骤

> cpolar官网地址: <https://www.cpolar.com>

* 使用一键脚本安装命令

```shell
curl -L https://www.cpolar.com/static/downloads/install-release-cpolar.sh | sudo bash

```

* 向系统添加服务

```shell
sudo systemctl enable cpolar

```

* 启动cpolar服务

```shell
sudo systemctl start cpolar

```

cpolar安装成功后，在外部浏览器上访问Linux 的9200端口 即:【http://局域网ip:9200】，使用cpolar账号登录(如没有账号,可以点击下面免费注册),登录后即可看到cpolar web 配置界面,结下来在web 管理界面配置即可

![](https://i-blog.csdnimg.cn/img_convert/ef691e1331002b2c5313362ad6395147.png)

### 3. 配置Halo个人博客公网地址

点击左侧仪表盘的隧道管理——创建隧道，创建一个Halo的公网http地址隧道!

* 隧道名称：可自定义命名，注意不要与已有的隧道名称重复
* 协议：选择http
* 本地地址：8090 (本地访问的地址)
* 域名类型：免费选择随机域名
* 地区：选择China

点击`创建`

![51428fa8d798a6f09244ff9567aa312](https://i-blog.csdnimg.cn/img_convert/c6bdb55e31d7f08f4b7bfca59ba804c7.png)

隧道创建成功后，点击左侧的状态——在线隧道列表,查看所生成的公网访问地址，有两种访问方式,一种是http 和https

![d8fb5e5727a167bc7a527fbebfbd944](https://i-blog.csdnimg.cn/img_convert/c3c8d09a8820b8f13c921a89200773f2.png)

使用上面的Cpolar https公网地址,在任意设备的浏览器进行访问,即可成功看到Halo首页界面,这样一个公网地址且可以远程访问就创建好了,使用了cpolar的公网域名,无需自己购买云服务器,即可发布到公网进行远程访问！

![image-20240109105141668](https://i-blog.csdnimg.cn/img_convert/708a30b3c4c9473f6585d897ec680598.png)

**小结**

如果我们需要把自己的个人博客长期发布至公网，分享给别人查看，由于刚才创建的是随机的地址，24小时会发生变化。另外它的网址是由随机字符生成，不容易记忆。如果想把域名变成固定的二级子域名，并且不想每次都重新创建隧道，可以选择创建一个固定的http地址来解决这个问题。

### 4. 固定Halo公网地址

我们接下来为其配置固定的HTTP端口地址，该地址不会变化，方便分享给别人长期查看你的博客，而无需每天重复修改服务器地址。

> 配置固定http端口地址需要将cpolar升级到专业版套餐或以上。

[登录cpolar官网](https://dashboard.cpolar.com/)，点击左侧的预留，选择保留二级子域名，设置一个二级子域名名称，点击保留,保留成功后复制保留的二级子域名名称

![bf1dd6a42148579307e2dae72055abc](https://i-blog.csdnimg.cn/img_convert/b9b495be25c4c1fd0d8b65f044a2f504.png)

保留成功后复制保留成功的二级子域名的名称

![edd09daaed4aa7183b057b271331157](https://i-blog.csdnimg.cn/img_convert/29481edd615cd193fd956afb612909f4.png)

返回登录Cpolar web UI管理界面，点击左侧仪表盘的隧道管理——隧道列表，找到所要配置的隧道，点击右侧的编辑

![98b73f50633de7c58fe3484439751ed](https://i-blog.csdnimg.cn/img_convert/df7bd444a042efae70ddba011be84055.png)

修改隧道信息，将保留成功的二级子域名配置到隧道中

* 域名类型：选择二级子域名
* Sub Domain：填写保留成功的二级子域名

点击`更新`(注意,点击一次更新即可,不需要重复提交)

![d2eeffd306da1d18e1c98884a2e45fa](https://i-blog.csdnimg.cn/img_convert/488c9ca2891ffaca7132e107a14eb432.png)

更新完成后,打开在线隧道列表,此时可以看到公网地址已经发生变化,地址名称也变成了固定的二级子域名名称的域名

![d68fa5856079ee39d18b5e909fb6697](https://i-blog.csdnimg.cn/img_convert/89cf10119130b5011ca827a65f28be0b.png)

最后,我们使用固定的公网https地址访问,可以看到访问成功,这样一个固定且永久不变的公网地址就设置好了,随时随地都可以把个人博客分享给其他人了！

![ade1b323d51ab81f7458d0fa22cf9ee](https://i-blog.csdnimg.cn/img_convert/b42073c554634440775d45a2e50d4473.png)
