    <h2 style="background-color:transparent;">
     第七部分：移动开发 —— 触手可及的未来
    </h2>
    <h3 id="%E5%BC%95%E8%A8%80">
     引言
    </h3>
    <p>
     移动开发也称为
     <a href="https://baike.baidu.com/item/%E6%89%8B%E6%9C%BA%E5%BC%80%E5%8F%91/222789?fromModule=lemma_inlink" rel="nofollow" title="手机开发">
      手机开发
     </a>
     ，或叫做移动互联网开发，移动应用开发等。是指以手机、PDA、UMPC等便携终端为基础，进行相应的开发工作，由于这些随身设备基本都采用无线上网的方式，因此，业内也称作为无线开发。移动应用包括早期的WM掌上助理设备，Web os设备、java设备、塞班设备，以及现在现在火热的安卓和苹果设备等移动应用的开发。移动应用的形成对移动设备的功能有了长足的拓展。设备可以不单单只靠自带的简陋功能，而是可以像计算机一样通过安装应用程序、游戏程序等进行扩展，使移动设备成为更能帮助人们解决事物的个人
     <a href="https://baike.baidu.com/item/%E6%99%BA%E8%83%BD%E7%BB%88%E7%AB%AF/10147151?fromModule=lemma_inlink" rel="nofollow" title="智能终端">
      智能终端
     </a>
     。
    </p>
    <p>
     随着智能手机的普及，移动应用已成为人们日常生活中不可或缺的一部分。移动开发，即开发可在移动设备上运行的应用程序，已成为软件开发领域的一个重要分支。本文将探讨移动开发平台，包括Android和iOS，以及跨平台开发技术，如React Native和Flutter。
    </p>
    <p>
     <img alt="" height="754" src="https://i-blog.csdnimg.cn/blog_migrate/a1650ee213856fc65c81895facb1ec29.png" width="1200"/>
    </p>
    <h3 id="1.%20%E7%A7%BB%E5%8A%A8%E5%BC%80%E5%8F%91%E5%B9%B3%E5%8F%B0" style="background-color:transparent;">
     1. 移动开发平台
    </h3>
    <h4 id="1.1%20Android" style="background-color:transparent;">
     1.1 Android
    </h4>
    <p>
     安卓（Android）是一种基于
     <a href="https://baike.baidu.com/item/Linux/27050?fromModule=lemma_inlink" rel="nofollow" title="Linux">

      开放源代码
     </a>
     的移动操作系统。主要应用于
     <a href="https://baike.baidu.com/item/%E7%A7%BB%E5%8A%A8%E8%AE%BE%E5%A4%87/9157757?fromModule=lemma_inlink" rel="nofollow" title="移动设备">
      移动设备

     <a href="https://baike.baidu.com/item/%E6%99%BA%E8%83%BD%E6%89%8B%E6%9C%BA/94396?fromModule=lemma_inlink" rel="nofollow" title="智能手机">

     <a href="https://baike.baidu.com/item/%E5%B9%B3%E6%9D%BF%E7%94%B5%E8%84%91/1348389?fromModule=lemma_inlink" rel="nofollow" title="平板电脑">
      平板电脑

     <a href="https://baike.baidu.com/item/Google/86964?fromModule=lemma_inlink" rel="nofollow" title="Google">
      Google
     </a>
     公司和
     <a href="https://baike.baidu.com/item/%E5%BC%80%E6%94%BE%E6%89%8B%E6%9C%BA%E8%81%94%E7%9B%9F/9064338?fromModule=lemma_inlink" rel="nofollow" title="开放手机联盟">
      开放手机联盟
     </a>
     领导及开发。Android操作系统最初由

      安迪·鲁宾
     </a>
     开发，主要支持手机。2005年8月由
     <a href="https://baike.baidu.com/item/Google/86964?fromModule=lemma_inlink" rel="nofollow" title="Google">
      Google
     </a>
     收购注资。2007年11月，Google与84家硬件制造商、软件开发商及电信营运商组建开放手机联盟共同研发改良Android系统。随后

      Google
     </a>

     </a>

      源代码
     </a>

     <a href="https://baike.baidu.com/item/%E6%95%B0%E7%A0%81%E7%9B%B8%E6%9C%BA/277472?fromModule=lemma_inlink" rel="nofollow" title="数码相机">
      数码相机
     </a>
     、游戏机、
     <a href="https://baike.baidu.com/item/%E6%99%BA%E8%83%BD%E6%89%8B%E8%A1%A8/71070?fromModule=lemma_inlink" rel="nofollow" title="智能手表">
      智能手表
     </a>
     等。2011年第一季度，Android在全球的
     <a href="https://baike.baidu.com/item/%E5%B8%82%E5%9C%BA%E4%BB%BD%E9%A2%9D/7026134?fromModule=lemma_inlink" rel="nofollow" title="市场份额">
      市场份额
     </a>
     首次超过
     <a href="https://baike.baidu.com/item/%E5%A1%9E%E7%8F%AD%E7%B3%BB%E7%BB%9F/8506777?fromModule=lemma_inlink" rel="nofollow" title="塞班系统">
      塞班系统
     </a>
     ，跃居全球第一。 2013年的第四季度，Android平台手机的全球市场份额已经达到78.1%。2013年09月24日
     <a href="https://baike.baidu.com/item/%E8%B0%B7%E6%AD%8C/117920?fromModule=lemma_inlink" rel="nofollow" title="谷歌">
      谷歌
     </a>
     开发的
     <a href="https://baike.baidu.com/item/%E6%93%8D%E4%BD%9C%E7%B3%BB%E7%BB%9F/192?fromModule=lemma_inlink" rel="nofollow" title="操作系统">
      操作系统
     </a>
     Android在迎来了5岁生日，全世界采用这款系统的设备数量已经达到10亿台。
    </p>
    <p>
     <img alt="" height="500" src="https://i-blog.csdnimg.cn/blog_migrate/a263235674a6d5a3a5b883154b12ff8f.png" width="1200"/>

     <li>
      <strong>
       开发语言
      </strong>

      <strong>
       开发环境

      ：Android Studio提供了集成的开发环境。

     2.1 React Native
    </h4>
