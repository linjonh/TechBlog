    <h3>
     与在其他设备上训练的模型相比，在mac上使用gpu训练的模型的质量明显较低，因此我们暂时使用cpu代替。

     <li>
      Install Xcode command-line tools by running
      <code>
       xcode-select --install
      </code>

     </li>
     <li>

      <code>
       brew install ffmpeg

     <li>
      Install the program by running the following commands:
     </li>
    </ol>
    <pre>conda create -n GPTSoVits python=3.9
conda activate GPTSoVits
pip install -r requirements.txt</pre>

    <h2>
     在macos上训练自己的声音

    <p>
     激活环境：

    <h3>
     第一踩坑点
    </h3>
    <p>
     启动命令：
    </p>
    <pre>python3 webui.py</pre>
    <p>
     之后，无法跳转，显示超时
    </p>
    <pre>File "/opt/anaconda3/envs/GPTSoVits/lib/python3.9/site-packages/pydantic/_internal/_generate_schema.py", line 558, in _unknown_type_schema
    raise PydanticSchemaGenerationError(
pydantic.errors.PydanticSchemaGenerationError</pre>
    <p>
     解决方法：
    </p>
    <pre>pip uninstall fastapi
pip install fastapi==0.112.2</pre>
    <p>
     成功跑起：
    </p>
    <p>
    </p>

     第二个踩坑点

     是在微调训练gpt报错

    <pre>return torch.zeros_like(preds_oh, dtype=torch.int32).scatter_(-1, result.unsqueeze(1).unsqueeze(1), 1)

     解决方法：
    </p>
    <pre>pip install torchmetrics==1.5</pre>

    <h3>
     CondaError: Run 'conda init' before 'conda activate'
    </h3>
    <p>
     利用
     <a href="https://zhida.zhihu.com/search?content_id=239934276&amp;content_type=Article&amp;match_order=1&amp;q=Mobaxterm&amp;zhida_source=entity" rel="nofollow" title="Mobaxterm">
      Mobaxterm
     </a>
     连接远程服务器，在创建conda环境后报错。
    </p>
    <p>
     输入conda init后返回No action taken.
    </p>
    <h4>
     解决方法
    </h4>
    <ol>
     <li>
      <p>
       输入
       <a href="https://zhida.zhihu.com/search?content_id=239934276&amp;content_type=Article&amp;match_order=1&amp;q=source+activate&amp;zhida_source=entity" rel="nofollow" title="source activate">
        source activate
       </a>
       ，后发现前面多了(base)
      </p>
     </li>
     <li>
      <p>
       输入conda activate xxx(此处为自定义的环境名称)，后发现(base)变为(xxx)
      </p>
     </li>
    </ol>
    <p>
     参考：
     <a href="https://zhuanlan.zhihu.com/p/683118485" rel="nofollow" title="https://zhuanlan.zhihu.com/p/683118485">
      https://zhuanlan.zhihu.com/p/683118485
