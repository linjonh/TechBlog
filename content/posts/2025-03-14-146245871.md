    <p>
     使用 vue create ss 创建个名为 ss 的 vue3 项目 (vue2项目不知道为什么不可以）
    </p>
    <p>
     App.vue中添加方法和事件，并打断点
    </p>
    <pre><code class="hljs">&lt;button @click="ha"&gt;asdfasdfasdfsdf&lt;/button&gt;

methods: {
    ha() {
      console.log('111111111')
      console.log('222222222222')
    }
  }</code></pre>
    <p>
     <img alt="" height="157" src="https://i-blog.csdnimg.cn/direct/eccc3877df7c4eeba3c8ba5b9f41cccb.png" width="499"/>
    </p>
    <p>
     vue.config.js 添加
    </p>
    <pre><code class="hljs">
  configureWebpack: {
    devtool: "source-map",
  },</code></pre>
    <p>
     启动 npm run serve
    </p>
    <p>
     添加 launch.json
    </p>
    <pre><code class="hljs">{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "chrome",
            "request": "launch",
            "name": "Launch Chrome against localhost",
            "url": "http://localhost:8080",
            "webRoot": "${workspaceFolder}"
        }
    ]
}</code></pre>
    <p>
     启动调试，点击按钮会进入到调试里。
    </p>
