    <h4>
     1.
     <strong>
      HTML文档结构
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <code>
        &lt;!DOCTYPE html&gt;
       </code>
       ：声明文档类型为HTML5。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;html lang="en"&gt;
       </code>
       ：定义HTML文档的根元素，并设置语言为英语。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;head&gt;
       </code>
       ：包含文档的元数据，如字符集、标题和外部样式表链接。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;body&gt;
       </code>
       ：包含网页的可见内容。
      </p>
     </li>
    </ul>
    <h4>
     2.
     <strong>
      元数据
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <code>
        &lt;meta charset="UTF-8"&gt;
       </code>
       ：指定文档的字符编码为UTF-8。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;title&gt;千年之恋&lt;/title&gt;
       </code>
       ：设置网页的标题，显示在浏览器的标签页上。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;link href="css/style06.css" type="text/css" rel="stylesheet"&gt;
       </code>
       ：链接外部CSS样式表文件
       <code>
        style06.css
       </code>
       ，用于定义页面的样式。
      </p>
     </li>
    </ul>
    <h4>
     3.
     <strong>
      页面结构
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        头部 (
        <code>
         &lt;div id="head"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         包含一个图片 (
         <code>
          &lt;img src="images/logo.jpg"/&gt;
         </code>
         )，用于显示网站的Logo。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        导航栏 (
        <code>
         &lt;div id="nav_bg"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         使用无序列表 (
         <code>
          &lt;ul&gt;
         </code>
         ) 创建导航栏，每个列表项 (
         <code>
          &lt;li&gt;
         </code>
         ) 包含一个链接 (
         <code>
          &lt;a&gt;
         </code>
         )，指向不同的页面或功能。
        </p>
       </li>
       <li>
        <p>
         最后一个链接 (
         <code>
          &lt;a href="#" class="bg"&gt;注册&lt;/a&gt;
         </code>
         ) 有一个额外的类
         <code>
          bg
         </code>
         ，可能是用于特殊样式。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        横幅 (
        <code>
         &lt;div id="banner"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         包含一个图片 (
         <code>
          &lt;img src="images/banner.jpg"/&gt;
         </code>
         )，用于显示网站的横幅广告或宣传图。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        内容区域 (
        <code>
         &lt;div id="content"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         包含一个表单 (
         <code>
          &lt;form&gt;
         </code>
         )，用于用户注册。
        </p>
       </li>
       <li>
        <p>
         表单分为两个部分：
        </p>
        <ul>
         <li>
          <p>
           <strong>
            账号信息
           </strong>
           ：包括注册方式（邮箱或手机）、注册邮箱、注册手机、登录密码和昵称。
          </p>
         </li>
         <li>
          <p>
           <strong>
            个人信息
           </strong>
           ：包括性别、学历、所在城市、兴趣爱好和自我介绍。
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         使用表格 (
         <code>
          &lt;table&gt;
         </code>
         ) 来布局表单内容，每个表行 (
         <code>
          &lt;tr&gt;
         </code>
         ) 包含表头 (
         <code>
          &lt;td class="left"&gt;
         </code>
         ) 和输入字段 (
         <code>
          &lt;td&gt;
         </code>
         )。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        页脚 (
        <code>
         &lt;div id="footer"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         包含版权信息 (
         <code>
          &lt;p&gt;
         </code>
         )，显示网站的版权声明和备案信息。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     4.
     <strong>
      表单元素
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        单选按钮 (
        <code>
         &lt;input type="radio"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于选择注册方式（邮箱或手机）和性别（男或女）。
        </p>
       </li>
       <li>
        <p>
         <code>
          name="sex"
         </code>
         用于将单选按钮分组，确保同一组中只能选择一个选项。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        文本输入框 (
        <code>
         &lt;input type="text"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于输入注册邮箱、注册手机和昵称。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        密码输入框 (
        <code>
         &lt;input type="password"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于输入登录密码，输入内容会被隐藏。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        下拉菜单 (
        <code>
         &lt;select&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于选择学历和所在城市。
        </p>
       </li>
       <li>
        <p>
         <code>
          &lt;option&gt;
         </code>
         定义下拉菜单中的选项，
         <code>
          selected="selected"
         </code>
         设置默认选中的选项。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        复选框 (
        <code>
         &lt;input type="checkbox"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于选择兴趣爱好，用户可以多选。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        文本域 (
        <code>
         &lt;textarea&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于输入自我介绍，用户可以输入多行文本。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        按钮 (
        <code>
         &lt;input type="button"&gt;
        </code>
        )
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用于提交表单，但当前代码中没有指定按钮的文本或功能。
        </p>
       </li>
      </ul>
     </li>
    </ul>

    <h4>
     6.
     <strong>
      HTML属性
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <code>
        id
       </code>
       ：用于唯一标识元素，如
       <code>
        id="head"
       </code>
       、
       <code>
        id="nav_bg"
       </code>
       等。
      </p>
     </li>
     <li>
      <p>
       <code>
        name
       </code>
       ：用于表单元素的名称，如
       <code>
        name="sex"
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       <code>
        maxlength
       </code>
       ：限制输入框的最大字符数，如
       <code>
        maxlength="8"
       </code>
       限制密码长度为8个字符。
      </p>
     </li>
     <li>
      <p>
       <code>
        cols
       </code>
       和
       <code>
        rows
       </code>
       ：定义文本域的列数和行数。
      </p>
     </li>
    </ul>
