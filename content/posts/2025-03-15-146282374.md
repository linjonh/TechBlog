    <h2>
     <a id="Mysql_2">
     </a>
     Mysql的正常启动
    </h2>
    <p>
     我使用的是Mac电脑，电脑上的Mysql8.0是处于运行状态，如果关闭状态也可以。
    </p>
    <h3>
     <a id="Msql_5">
     </a>
     设置Msql的无密码登陆前操作
    </h3>
    <p>
     我们对Markdown编辑器进行了一些功能拓展与语法支持，除了标准的Markdown编辑器功能，我们增加了如下几点新功能，帮助你用它写博客：
    </p>
    <ol>
     <li>
      <strong>
       mysqld_safe --skip-grant-tables &amp;
      </strong>
      ，设置无密码登陆的操作，看命令执行成功了就行。
     </li>
     <li>
      <strong>
       mysql -u root
      </strong>
      ，然后是无密码登陆的命令，即可直接访问。
     </li>
     <li>
      无密码登陆之后，直奔mysql库
      <strong>
       use mysql
      </strong>
      ，在当前库中修改用户表的root和密码
      <strong>
       alter user ‘root’@‘localhost’ IDENTIFIED BY ‘123456’
      </strong>
      ，再刷新一下更改的内容即可****
     </li>
     <li>
      一般情况下密码就可以更改了，下次启动还是通过mysql -u root -p的形式输入密码就行；
     </li>
    </ol>
