    <h3 id="一具体构成描述">
     <strong>
      一、具体构成描述
     </strong>
    </h3>
    <h4 id="1具体构成描述构成因特网的基本硬件和软件组件">
     <strong>
      1、具体构成描述
     </strong>
     （构成因特网的基本硬件和软件组件）
    </h4>
    <ol>
     <li>
      主机（host）或端系统(end system)：与因特网相连，处在边缘的计算机等设备。
     </li>
     <li>
      <p>
       分组(packet)：当一台端系统有数据要向另一台端系统发送时，发送端系统将数据分段，并为每段加上首部字节。由此形成的信息包称为分组
      </p>
      <blockquote>
       <p>
        <strong>
         端系统通过通信链路和分组交换机连接到一起
        </strong>
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       通信链路(communication link)：通信链路由不同的物理媒体组成（同轴电缆，铜线，光纤，无线电频谱）。不同的链路以不同的速率传输数据，链路的传输速率是以
       <a href="http://baike.baidu.com/link?url=HoJXV6M2rR-6cKtZ8kRLBHfnuOlxylAW6RsOIY5G4Qv-E_-JRKAoLaoaGfCFpv9ZL7sna7McVtHReKm9N6S8uq" rel="nofollow">
        bps
       </a>
       度量的。
      </p>
     </li>
     <li>
      分组交换机(packet switch)：从它的一条入通信链路接受到达的分组，并从它的一条出通信链路转发该分组。
     </li>
     <li>
      <p>
       路径(route)：从发送端系统到接收端系统，一个分组所经历的一系列通信链路和分组交换机称为通过该网络的路径。
      </p>
      <blockquote>
       <p>
        <strong>
         端系统通过因特网服务提供商接入因特网
        </strong>
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       ISP（Internet Service Provider）：由多个分组交换机和多段通信链路组成的网络
      </p>
      <blockquote>
       <p>
        端系统，分组交换机和其他因特网部件，都要运行控制因特网中信息接收和发送的一系列协议，其中TCP/IP是两个最为重要的协议
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       内联网：例如许多公司和政府网络，这些网络内的主机不能与专用网络外部的主机交换信息（除非这些信息通过了所谓的防火墙）。
      </p>
      <h4 id="2服务描述为分布式应用提供服务的网络基础设施">
       <strong>
        2、服务描述
       </strong>
       （为分布式应用提供服务的网络基础设施）
      </h4>
      <p>
       1.分布式应用程序：涉及多台相互交换数据的端系统。
       <br/>
       2.API：规定了运行在一个端系统上的软件请求因特网基础设施向运行在另一个端系统上的特定目的地软件交付数据的方式
      </p>
     </li>
    </ol>
    <h3 id="二网络边缘">
     <strong>
      二、网络边缘
     </strong>
    </h3>
    <blockquote>
     <p>
      与因特网相连的计算机，因位于因特网的边缘，通常被称为端系统，也称为主机，但主机有时又被进一步分为客户机（client）和服务器（server）.
     </p>
    </blockquote>
    <h4 id="1客户机和服务器程序">
     <strong>
      1、客户机和服务器程序
     </strong>
    </h4>
    <p>
     1.客户机—服务器模式：
     <strong>
      客户机程序是运行在一个端系统上的程序，它发出请求，并从运行在另一个端系统上的服务器程序接收服务。
     </strong>
     <br/>
     2.对等应用程序（p2p）：端系统互相作用并运行执行客户机和服务器功能的程序。
    </p>
    <h4 id="2接入网">
     <strong>
      2、接入网
     </strong>
    </h4>
    <p>
     1.接入网
    </p>
    <blockquote>
     <p>
      <strong>
       即将端系统接连到其边缘路由器的物理链路
      </strong>
      <br/>
      边缘路由器是端系统到任何其他远程端系统的路径上的第一台路由器
     </p>
    </blockquote>
    <p>
     2.网络接入的分类
    </p>
    <ul>
     <li>
      住宅接入
     </li>
     <li>
      公司接入
     </li>
     <li>
      无线接入
     </li>
    </ul>
    <h3 id="三物理媒体">
     <strong>
      三、物理媒体
     </strong>
    </h3>
    <p>
     <strong>
      物理媒体分类
     </strong>
     <br/>
     - 引导型媒体
    </p>
    <blockquote>
     <p>
      电波沿着固体媒体（如光缆，双绞铜线，同轴电缆）被引导。
     </p>
    </blockquote>
    <p>
    </p>
    <ul>
     <li>
      <br/>
     </li>
     <li>
      非引导型媒体
      <br/>
     </li>
    </ul>
    <p>
    </p>
    <blockquote>
     <p>
      电波在空气或外层空间（如无线局域网，数字卫星频道）中传播。
     </p>
    </blockquote>
