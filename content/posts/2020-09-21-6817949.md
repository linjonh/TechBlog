    <h1 class="title-article" id="articleContentId">
     如何理解《道德经》开篇，决定一个软件工作者的前程
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <span style="font-size:18px">
     <span style="font-family:KaiTi_GB2312">
      摘要：道德经 软件之道 软件工程师的发展                        参阅：
     </span>
     <a href="http://blog.csdn.net/xabcdjon/article/details/6792276">
      <span style="font-family:KaiTi_GB2312">
       道德经开篇该如何翻译
      </span>
     </a>
    </span>
    <p align="left">
     <span style="font-size:18px">
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      最近在网上看了一篇文章，说老师布置了一道古文断句翻译题，同学们回答各异，最终发展也各不相同，当然有一定的调侃成分。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      原文如下：“
      <strong>
       道可道非常道名可名非常名无名天地之始有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这是道德经的开篇，我也借此调侃一下我们做IT的，希望能带来一些思考和启发。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      1.  “
      <strong>
       道可,道非,常道。名可,名非,常名。无名天地之始,有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      DAO可用，DAO不好用，常有的事。名称起的好，名称起的不好，经常起名字。刚开始大家都没什么名气，等到有点名气已经是为人父母了。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学做了一辈子
      <strong>
       <span style="color:#ff0000">
        低端程序员
       </span>
      </strong>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      2.  “
      <strong>
       道可道非常道。名可名非常名。无名天地之始,有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      编程规范可用说是非常有道理的，命名可以说也是非常重要的。无名的那些数据是原始类型，有名的那些是整个系统的基础。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学成长为一名
      <strong>
       <span style="color:#ff0000">
        优秀的程序员
       </span>
      </strong>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      3.  “
      <strong>
       道可道，非常道。名可名，非常名。无名天地之始,有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      软件设计的原则是存在的，但不是一成不变的；为它们命名是可以的，但通常代表不了它的内涵。那些没有名的原则（公理）是软件设计的起点，这些有名的原则是构建软件系统各部分的基础。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学成长为优秀的
      <strong>
       <span style="color:#ff0000">
        软件设计师
       </span>
      </strong>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      4.  “
      <strong>
       道可道，非常道。名可名，非常名。无，名天地之始，有，名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      软件之道是客观存在的，但并不是我们通常设置的那些条条框框；为这个软件之道命名是可以的，但不能用常规的方式命名；它需要从“无”和“有”两个层面命名，从无的层面，它可以称为一切软件的起始，从有的层面，它可以称为所有软件系统的造物主。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学成长为卓越的
      <strong>
       <span style="color:#ff0000">
        软件大师
       </span>
      </strong>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      5.  “
      <strong>
       道可，道非常，道。名可,名非常，名。无名天地之始，有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      说你有道理你就有道理，说你没道理你就没道理，就看想怎么说了；能让你声誉好，也能让你名声不好，就看怎么弄你的名声了。可以说天地间没有一样东西是有名的，也可以说万物都是有名的。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学成长为一个
      <span style="color:#ff0000">
       <strong>
        不太靠谱的项目经理
       </strong>
      </span>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      6.  “
      <strong>
       道可，道非常，道。 名可，名非常，名。 无名天地之始，有名万物之母
      </strong>
      。”
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      说它好也行，说它非常好也行，尽管往好了说；说它有名也行，说它非常有名也行，尽管让它出名。起初谁都没有名气，等创出名气来，大家就会像对母亲一样尊重你了。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">
      这位同学成长为优秀的软件
      <span style="color:#ff0000">
       <strong>
        售前工程师
       </strong>
      </span>
      。
     </span>
    </p>
    <p align="left">
     <span style="font-size:18px">

    </p>
