    <p>
     ### **二、FFmpeg基础使用**
    </p>
    <p>
     #### **1. 转换视频格式**
     <br/>
     ```bash
     <br/>
     ffmpeg -i input.mp4 output.avi
     <br/>
     ```
     <br/>
     - `-i`：指定输入文件。
    </p>
    <p>
     ---
    </p>
    <p>
     #### **2. 提取音频**
     <br/>
     ```bash
     <br/>
     ffmpeg -i input.mp4 -vn -acodec copy output.mp3
     <br/>
     ```
     <br/>
     - `-vn`：忽略视频流。
     <br/>
     - `-acodec copy`：直接复制音频流（无需重新编码）。
    </p>
    <p>
     ---
    </p>
    <p>
     #### **3. 调整视频分辨率**
     <br/>
     ```bash
     <br/>
     ffmpeg -i input.mp4 -vf scale=1280:720 output.mp4
     <br/>
     ```
     <br/>
     - `-vf scale=宽度:高度`：设置分辨率。
    </p>
    <p>
     ---
    </p>
    <p>
     #### **4. 剪切视频片段**
     <br/>
     ```bash
     <br/>
     ffmpeg -ss 00:01:30 -i input.mp4 -to 00:02:00 -c copy output.mp4
     <br/>
     ```
     <br/>
     - `-ss`：起始时间（时:分:秒）。
     <br/>
     - `-to`：结束时间。
     <br/>
     - `-c copy`：直接复制流（快速无质量损失）。
    </p>
    <p>
     ---
    </p>
    <p>
     #### **5. 合并多个视频**
     <br/>
     1. 创建 `list.txt` 文件，内容如下：
     <br/>
     ```
     <br/>
     file 'video1.mp4'
     <br/>
     file 'video2.mp4'
     <br/>
     ```
     <br/>
     2. 执行合并命令：
     <br/>
     ```bash
     <br/>
     ffmpeg -f concat -i list.txt -c copy output.mp4
     <br/>
     ```
    </p>
    <p>
     ---
    </p>
    <p>
     ### **三、FFmpeg高级功能**
    </p>
    <p>
     #### **1. 添加水印**
     <br/>
     ```bash
     <br/>
     ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=10:10" output.mp4
     <br/>
     ```
     <br/>
     - `overlay=10:10`：水印位于左上角，距离左边和顶部各10像素。
    </p>
    <p>
     ---
    </p>
    <p>
     #### **2. 调整视频速度**
     <br/>
     ```bash
     <br/>
     # 视频加速2倍，音频加速2倍
     <br/>
     ffmpeg -i input.mp4 -vf "setpts=0.5*PTS" -af "atempo=2.0" output.mp4
     <br/>
     ```
    </p>
    <p>
     ---
    </p>
    <p>
     #### **3. 提取视频帧为图片**
     <br/>
     ```bash
     <br/>
     ffmpeg -i input.mp4 -r 1 -f image2 image-%03d.jpg
     <br/>
     ```
     <br/>
     - `-r 1`：每秒提取1帧。
     <br/>
     - `image-%03d.jpg`：生成文件名（如 `image-001.jpg`）。
    </p>
    <p>
     ---
    </p>
    <p>
     ### **四、常用参数说明**
     <br/>
     | 参数          | 功能                           |
     <br/>
     |---------------|--------------------------------|
     <br/>
     | `-c:v libx264`| 指定视频编码器（如H.264）。    |
     <br/>
     | `-b:v 1M`     | 设置视频比特率（1Mbps）。      |
     <br/>
     | `-preset fast`| 编码速度与质量的平衡参数。     |
     <br/>
     | `-an`         | 禁用音频。                     |
    </p>
