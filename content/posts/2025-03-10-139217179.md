# Git下载安装

### 1.git下载

打开
[git官网](https://git-scm.com/)
，点击downloads。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/4c932e8ee5a4ddcd14a7031528623c58.png)
  
选择对应的操作系统，这里我们使用的是windows操作系统。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/edd7c6aca69dea8508eef39aeeefde74.png)
  
选择对应的操作系统是32维还是64位，点击及自动下载。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/275d29f5bacdf1e9dfc722903680cfcd.png)

### 2.Git安装

运行下载好的git安装包。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/991a76ad97038fa7a37e23f7d8c78fce.png)
  
选择next。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/f6a2c4cfa67dd31b246380dfbdcab705.png)
  
选择安装的位置，然后next。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/7127e90a2ddde3d16ef46719aae09b43.png)
  
选择需要安装的组件默认即可，然后next。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/7fec68e60394f8e38337e094e4adbeb8.png)
  
然后一直默认选项选next到install，直接选择install。
  
安装完成后都不勾选，点击finish，安装完成。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/943881c7e29284759c1b7a1238b4a1ba.png)
  
验证是否安装完成，在桌面右键看是否有Open Git GUI here和Open Git Bash here。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/5673cb94ac0c4ea89ed8ec2015f7edd3.png)
