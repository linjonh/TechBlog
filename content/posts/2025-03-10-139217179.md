    <h3>
     <a id="1git_0">
     </a>
     1.git下载
    </h3>
    <p>
     打开
     <a href="https://git-scm.com/" rel="nofollow">
      git官网
     </a>
     ，点击downloads。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4c932e8ee5a4ddcd14a7031528623c58.png">
      <br/>
      选择对应的操作系统，这里我们使用的是windows操作系统。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/edd7c6aca69dea8508eef39aeeefde74.png">
       <br/>
       选择对应的操作系统是32维还是64位，点击及自动下载。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/275d29f5bacdf1e9dfc722903680cfcd.png"/>
      </img>
     </img>
    </p>
    <h3>
     <a id="2Git_9">
     </a>
     2.Git安装
    </h3>
    <p>
     运行下载好的git安装包。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/991a76ad97038fa7a37e23f7d8c78fce.png">
      <br/>
      选择next。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f6a2c4cfa67dd31b246380dfbdcab705.png">
       <br/>
       选择安装的位置，然后next。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7127e90a2ddde3d16ef46719aae09b43.png">
        <br/>
        选择需要安装的组件默认即可，然后next。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7fec68e60394f8e38337e094e4adbeb8.png">
         <br/>
         然后一直默认选项选next到install，直接选择install。
         <br/>
         安装完成后都不勾选，点击finish，安装完成。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/943881c7e29284759c1b7a1238b4a1ba.png">
          <br/>
          验证是否安装完成，在桌面右键看是否有Open Git GUI here和Open Git Bash here。
