    <h2>
     <a id="_2">
     </a>
     一、简介
    </h2>
    <p>
     有些开源项目将少儿可视化编程环境scratch与Arduino相结合，可以进行积木一样拖拉式编程。常见的有:
    </p>
    <h4>
     <a id="1_S4A_4">
     </a>
     1. S4A
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/48b5ec84ef6a8220414894200908ddfe.jpeg#pic_center">
      <br/>
      不过其Scratch是基于2.0开发的，界面比较老。
     </img>
    </p>
    <h4>
     <a id="2_ArduinoScratch_7">
     </a>
     2. ArduinoScratch
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d3ac52afa2028bfada5bf1486dd60049.png"/>
    </p>
    <h4>
     <a id="3_OpenBlock_10">
     </a>
     3. OpenBlock
    </h4>
    <p>
     OpenBlock是一款优秀的国产开源免费Scratch+Arduino开发环境，更新迭代速度快，操作容易上手，本文将介绍其入门操作。准备工作有：
    </p>
    <ul>
     <li>
      安装Windows系统的PC机
     </li>
     <li>
      一块Arduino UNO
     </li>
     <li>
      一个LED灯
     </li>
    </ul>
    <h2>
     <a id="_17">
     </a>
     二、操作步骤
    </h2>
    <h3>
     <a id="1__Arduino__18">
     </a>
     1. 准备 Arduino 开发板
    </h3>
    <p>
     我这里准备了一块Arduino UNO，使用USB线连接到电脑。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1fa669a85d088454c13a1e40af20055a.jpeg#pic_center"/>
    </p>
    <h4>
     <a id="1__Arduino_IDE_22">
     </a>
     (1) 安装 Arduino 开发IDE
    </h4>
    <p>
     <a href="https://www.arduino.cc/en/guide/windows" rel="nofollow">
      下载地址
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e3b2ee74e5de359e2133711404ebaeaf.jpeg#pic_center"/>
    </p>
    <p>
     下载后双击，按提示进行安装。此教程网上比较多，这里不再缀述。
    </p>
    <h4>
     <a id="2_arduino_28">
     </a>
     (2) 从系统-设备管理器找到arduino的端口
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9acea2cf84e317addcaa98804687abce.jpeg#pic_center">
      <br/>
      这里端口号是USB-SERIAL CH340(COM4)。
     </img>
    </p>
    <h4>
     <a id="3_Arduino_IDE_31">
     </a>
     (3) 下面测试Arduino IDE
    </h4>
    <p>
     打开Arduino IDE：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a60795051946287e7c96f4eae40234f7.jpeg#pic_center">
      <br/>
      在工具-端口，选择COM4（以上面第2步查到的端口号一致）：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9864fd60010565be562264024a88f330.jpeg#pic_center">
       <br/>
       在菜单选择：文件-示例-01.Basics-Blink：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/53e2152b648a6a3634ccf7814a6b098f.jpeg#pic_center">
        <br/>
        在工具栏点击上传按钮：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14ce9d6ff2cbb25923943f6160b4d746.jpeg#pic_center"/>
        <br/>
        等待下方的上传成功。
       </img>
      </img>
     </img>
    </p>
    <p>
     观察Arduino电脑板L灯会一闪一闪。
    </p>
    <h3>
     <a id="2__OpenBlock_44">
     </a>
     2. 下载安装 OpenBlock软件
    </h3>
    <h4>
     <a id="1_httpsgiteecomopenblockcc_45">
     </a>
     (1) 开源地址：https://gitee.com/openblockcc/
    </h4>
    <p>
     本文下载安装其桌面版，到其
     <a href="https://openblockcc.gitee.io/wiki/zh/download-software/" rel="nofollow">
      中文文档
     </a>
     找到下载地址，下载后双击按提示进行安装。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f12c27c871c96d159a5959cd6122ae73.jpeg#pic_center"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a5e62076dad3d7be630a5dbe69152a7.jpeg#pic_center"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ba8835f7bd3cbbaf23c5eb8c68252557.jpeg#pic_center"/>
    </p>
    <h4>
     <a id="2_OpenBlock_54">
     </a>
     (2) 启动OpenBlock
    </h4>
    <p>
     如果前一步没有钩选 运行OpenBlock，可以从开始菜单启动OpenBlock。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/df45f821cc39d4b63fe13f23e0fb2184.png"/>
    </p>
    <h3>
     <a id="3_OpenBlockArduino_58">
     </a>
     3. 在OpenBlock中连接Arduino
    </h3>
    <h4>
     <a id="1__59">
     </a>
     (1) 选择设备
    </h4>
    <p>
     注意这时OpenBlock上方有未选择设备-没有连接的字样，点击未选择设备：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b90a0e393f60749edb7b85c8e20f62b8.jpeg#pic_center"/>
     <br/>
     选择Arduino Uno。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2c9ed84aaa4f918cb9c84511b9dc5c9a.jpeg#pic_center"/>
    </p>
    <h4>
     <a id="2__64">
     </a>
     (2) 连接设备
    </h4>
    <p>
     点击连接按钮。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/238f1dbb312de55da817621d836f51b2.jpeg#pic_center"/>
    </p>
    <h4>
     <a id="3__67">
     </a>
     (3) 下载固件
    </h4>
    <p>
     然后点击返回编辑器，点击界面的下载固件按钮：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4b5bdf23d6b62d870fa64b50e99382aa.jpeg#pic_center"/>
     <br/>
     等待界面显示上传成功，点击下方的 “关闭” 按钮。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/149803714d7fe2106be91591354f190c.jpeg#pic_center"/>
    </p>
    <h2>
     <a id="__72">
     </a>
     三、 入门程序示例
    </h2>
    <h3>
     <a id="1__73">
     </a>
     1. 放一些积木模块
    </h3>
    <p>
     首先选择事件-“当 绿旗 被点击”，把它拖到右侧程序区：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4481ae90b8a0a93918ce9860b3f3239d.jpeg#pic_center"/>
     <br/>
     类似操作按下面图标把其它积木块拼在一起：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/91a18442f75dc95d5a9483d8a0a070b9.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="2_LED_80">
     </a>
     2. 接入LED灯
    </h3>
    <p>
     把LED插到靠近USB一侧插孔的第4、5脚，LED灯的长脚插入标13的孔，短脚插入标GND的孔。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/84ec4500c6ae514ff538980a98ac3507.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="3__84">
     </a>
     3. 测试运行
    </h3>
    <p>
     点击绿旗运行按钮：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6b850185bd284da5b4b3f3feeb5ef3c1.jpeg#pic_center"/>
     <br/>
     观察LED会闪烁。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a665eaf12404c35cb420f9fa4ef6a61.gif#pic_center"/>
    </p>
    <h2>
     <a id="_89">
     </a>
     四、使用上传模式
    </h2>
    <p>
     注意刚才的界面右上角有个“实时模式”的开关，这时候如果关闭OpenBlock，则LED灯不再闪烁。如果要程序脱离软件仍能运行，则可以使用上传模式：
     <br/>
     点击切换为上传模式：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/163ee952f7df02b5c2a2c2b4258727a3.jpeg#pic_center"/>
     <br/>
     点击中间的上传按钮，等待上传成功：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f906c69523c4cd97aa1fe7f9356aacea.jpeg#pic_center"/>
     <br/>
     观察这时候LED正常闪烁。关闭OpenBlock软件，LED灯仍然可以正常闪烁。
    </p>
