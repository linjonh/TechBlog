    <p>
     最近刚接触阿里云的堡垒机，一开始不熟悉，然后自己进行摸索，为了防止自己忘记和方便大家，遂写个文章。
    </p>
    <p>
     首先，你要进行购买云堡垒机，会有账号和密码。由于我是用的是已经注册好的，所以关于怎么获取堡垒机不做描述了。（因为我没看这部分）
     <br/>
     首先看一下云堡垒运维界面
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/68f7a016e349678535f3651932a60578.png">
      <br/>
      然后进入运维界面下的主机运维界面
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/39fa703ac048987cd9d3a1fb522e709a.png">
       <br/>
       下面进入运维工具配置，在这个页面的右上角，
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f9531240ffa116d33e950ad5fb025179.png">
        <br/>
        然后对运维工具进行配置，
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bbad766bb855a86714f0364fa933e4c8.png">
         <br/>
         描述：RDP是正常的远程桌面，也就是电脑自带的mstsc.exe
         <br/>
         SSH &amp; TELNET &amp; Rlogin：由于我操作的是ubuntu ，所以使用的是SSH，命令执行
         <br/>
         FTP/SFTP：两个都是进行文件传输的。
         <br/>
         VNC：远程桌面，类似mstsc.exe，向日葵。
         <br/>
         下面进行配置，以我使用的操作系统举例
         <br/>
         1.首先下载管理工具，下载位置在
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cee4e462fbc26d8946ec92b7691fe5a5.png">
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/aa0a2c2479d7d7af35bf33b859773e7d.png">
           <br/>
           下载以后运行。修改配置点击保存，
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ca1649e0bf58ef90f76cca5a3912b10.png">
            <br/>
            然后浏览器是弹出提示。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/186c08bb1ad8012d1baa5a00f6016a11.png">
             <br/>
             选择“打开 运维审计系统”，
             <br/>
             这就把你的保存了，关闭这个设置界面关了就行了。
             <br/>
             然后就是连接我们的堡垒机了，点击要远程的堡垒机。
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/10d081ee2ed20f35d8b0fe0b92b41129.png"/>
             <br/>
             在弹出的串口中选择远程方式，然后输入用户名和密码。
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3c176beadeeb854e1b5a47ecf78a5558.png"/>
             <br/>
             点击确定，会弹出
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/186c08bb1ad8012d1baa5a00f6016a11.png"/>
             <br/>
             选择“打开 运维审计系统”，然后就会在你的本地弹出之前配置好的工具了。
            </img>
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>

     这个页面找到对应的工具。
