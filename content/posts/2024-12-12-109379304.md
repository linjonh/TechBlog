    <p>
     <strong>
      前提：电脑中安装了node和mongodb
     </strong>
    </p>
    <h3>
     <a id="_2">
     </a>
     一、准备工作
    </h3>
    <p>
     1、要在全局环境下安装express以及它的生成器
    </p>
    <pre><code class="prism language-javascript">npm install express <span class="token operator">-</span>g
npm install <span class="token operator">-</span>g express<span class="token operator">-</span>generator
</code></pre>
    <p>
     2、创建项目
    </p>
    <pre><code class="prism language-javascript">npm init <span class="token operator">-</span>y
express <span class="token operator">-</span>e  project（e表示模板引擎 是ejs   project代表项目的名称）
</code></pre>
    <p>
     3、安装依赖包以及mongoose
    </p>
    <pre><code class="prism language-javascript">npm install
npm install mongoose <span class="token operator">--</span>save
</code></pre>
    <p>
     4、在启动项目之前可以修改两个地方
    </p>
    <ul>
     <li>
      修改端口号：
      <br/>
      bin/www 文件夹中修改端口号
      <code>
       var port = normalizePort(process.env.PORT || '50');
      </code>
     </li>
     <li>
      修改后自动启动项目，这个必须提前在全局安装好nodemon
      <br/>
      package.json 文件夹中将node改为nodemon：
      <code>
       "start": "nodemon ./bin/www"
      </code>
      <br/>
      5、启动项目
      <br/>
      <code>
       npm start
      </code>
     </li>
    </ul>
    <p>

    <p>
     三、由于使用express -e生成的项目是ejs模版引擎，修改为后缀为：.html
    </p>
    <ol>
     <li>
      首先框架已经搭建好，然后下载ejs模块并保存到package.josn中去
      <br/>
      <code>
       npm intsall ejs --save
      </code>
     </li>
     <li>
      在app.js文件下，引用ejs的那一行代码改成如下代码:
      <br/>
      <code>
       var ejs = require('ejs')
      </code>
      <br/>
      在将配置模版引擎的代码修改如下。
     </li>
    </ol>
    <pre><code class="prism language-javascript">app<span class="token punctuation">.</span><span class="token keyword">set</span><span class="token punctuation">(</span><span class="token string">'views'</span><span class="token punctuation">,</span> path<span class="token punctuation">.</span><span class="token function">join</span><span class="token punctuation">(</span>__dirname<span class="token punctuation">,</span> <span class="token string">'views'</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

      最后把views下的ejs后缀的文件改成html后缀就行了
