    <h2>
     <a id="1__0">
     </a>
     1. 前言
    </h2>
    <p>
     微信小程序已经非常普及，但是遗憾的是官方提供的 SDK 等没有Delphi语言的，这样使用Delphi语言开发微信小程序后台就相对比较困难，需要研究平台API，费时费力，特别是调试API很费时间，有鉴于此，我用Delphi开发的微信小程序控件后台控件，使用该控件，只需要设置微信小程序的平台相关参数，就可以实现微信小程序后台功能。
    </p>
    <h2>
     <a id="2__2">
     </a>
     2. 微信小程序控件介绍
    </h2>
    <h3>
     <a id="21__3">
     </a>
     2.1 控件实现基本功能
    </h3>
    <p>
     <strong>
      控件自带 Web 服务，无需安装任何 Web 服务控件！！
     </strong>
     <br/>
     小程序控件实现的功能：
     <br/>
     1.JWT 支持
     <br/>
     2.微信小程序前端登录Login，通过登录获取openid，unionid以及JWT_Token
     <br/>
     3.获取用户电话号码功能
     <br/>
     4.订阅消息功能
     <br/>
     5.客服消息功能
     <br/>
     6.动态消息功能
     <br/>
     7.OCR 识别功能
     <br/>
     1）身份证识别（正反面）
     <br/>
     2）银行卡识别
     <br/>
     3）驾驶证识别
     <br/>
     4）行驶证识别
     <br/>
     5）营业执照识别
     <br/>
     6）通用印刷体识别
     <br/>
     8.敏感图片鉴别（涉黄、涉爆）
     <br/>
     9.敏感词鉴别
     <br/>
     10.获取数量限制和非数量限制小程序（透明或非透明）
     <br/>
     11.物流接口
     <br/>
     12.上传文件接口
     <br/>
     13.通用下载文件接口
     <br/>
     14.文件下载 URL 匿名访问功能
     <br/>
     15.通用 JWT_token 机密函数
    </p>
