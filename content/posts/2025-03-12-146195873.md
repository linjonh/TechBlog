    <p>
     一、项目介绍
     <br/>
     440.基于SpringBoot的美食分享平台，系统包含两种角色：管理员、用户,系统分为前台和后台两大模块，主要功能如下。
    </p>
    <p>
     <br/>
     【前台功能】
    </p>
    <p>
     1. 首页：提供用户进入系统的入口。
     <br/>
     2. 菜谱：用户可以浏览系统中分享的各种菜谱。
     <br/>
     3. 论坛：用户可以在论坛上进行饮食交流、分享心得等。
     <br/>
     4. 饮食资讯：提供最新的饮食资讯、健康饮食建议等信息。
     <br/>
     5. 个人中心：用户可以管理个人信息。
    </p>
    <p>
     【后台功能】
     <br/>
     1. 首页：提供管理员进入后台管理的入口。
     <br/>
     2. 个人中心：管理员可以管理个人信息。
     <br/>
     3. 管理员管理：进行管理员权限的分配和管理。
     <br/>
     4. 用户管理：查看、编辑、冻结或删除用户账号。
     <br/>
     5. 菜谱管理：添加、编辑、删除系统中的菜谱信息。
     <br/>
     6. 论坛管理：管理论坛板块。
     <br/>
     7. 饮食资讯管理：发布、编辑、删除系统的饮食资讯和健康建议。
     <br/>
     8. 基础数据管理：管理系统的基础数据。
     <br/>
     9. 轮播图信息：管理员可以设置首页轮播图。
    </p>
    <p>
     二、项目技术
     <br/>
     编程语言：Java
     <br/>
     数据库：MySQL
     <br/>
     项目管理工具：Maven
     <br/>
     前端技术：Vue
     <br/>
     后端技术：SpringBoot
    </p>
    <p>
     三、运行环境
     <br/>
     操作系统：Windows、macOS都可以
     <br/>
     JDK版本：JDK1.8以上都可以
     <br/>
     开发工具：IDEA、Ecplise、Myecplise都可以
     <br/>
     数据库: MySQL5.7以上都可以
     <br/>
     Maven：任意版本都可以
