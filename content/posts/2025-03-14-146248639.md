    <p>
     2025年1月 ——
     <a rel="nofollow">
      字节跳动（TikTok 的母公司）推出了 Trae
     </a>
     ，这款创新的 AI 驱动代码编辑器，旨在大幅提升开发者的工作效率。Trae 将强大的人工智能与简洁直观的界面结合，帮助开发者更快速、轻松地编写、调试和优化代码。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e306a3f0788349548cec0b713989215e.jpeg"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/93cd05159f494a118c47a8123e482fd6.png"/>
    </p>
    <p>
     Trae 是一款专为国际市场打造的一体化开发环境（IDE）。
    </p>
    <p>
     Trae 最吸引人的特点是它集成了人工智能，能够在编写代码的同时实时为开发者提供帮助。编辑器会在你输入代码的过程中进行分析，提供智能建议、解答问题，甚至检测并修复常见错误。这种 AI 辅助功能能让开发者把更多时间花在解决复杂问题上，避免浪费时间排查错误。
    </p>
    <p>
     Trae 的实时代码建议功能会根据你输入的内容，自动提供上下文相关的建议，帮助你避免犯错，确保代码质量。此外，Trae 的 AI 还能根据自然语言描述生成代码片段，只要开发者简单描述需求，AI 就能自动生成对应的代码。
    </p>
    <hr/>
    <p>
     <strong>
      侧边聊天
     </strong>
    </p>
    <p>
     聊天模式作为一个全能 AI 搭档贯穿你的编码过程。它能够解答编码问题、解释代码仓库、生成代码片段、修复错误等，满足你在开发过程中提出的各种需求。
    </p>
    <h4>
     <a id="_18">
     </a>
     启用聊天模式
    </h4>
    <ul>
     <li>
      使用快捷键
      <strong>
       Command + U
      </strong>
      打开侧边聊天框。
     </li>
     <li>
      在聊天框左上角点击
      <strong>
       Chat
      </strong>
      以进入聊天模式。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a9b3eb58b31643ad8d191e91a019a42e.png"/>
     </li>
    </ul>
    <h4>
     <a id="_24">
     </a>
     探索聊天模式
    </h4>
    <h5>
     <a id="_26">
     </a>
     添加上下文
    </h5>
    <p>
     AI 助手可以理解当前项目的上下文信息，包括代码、文件和文件夹，从而更精准地协助你满足开发需求。
    </p>
    <h5>
     <a id="_30">
     </a>
     输入多模态内容
    </h5>
    <p>
     你可以在聊天中添加图片，例如错误截图、设计草图、参考样式等，以更准确、高效地表达需求。
    </p>
    <h5>
     <a id="_34">
     </a>
     生成和处理代码变更
    </h5>
    <p>
     根据你的需求，AI 助手会自动创建新文件或编辑现有文件，并保存生成的代码。你可以选择接受或拒绝代码变更，包括：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        接受代码变更时
       </strong>
       ，AI 助手会创建新文件或将变更应用于现有文件。所应用的代码将以差异（diff）格式显示。
      </p>
     </li>
     <li>
      <p>
       <strong>
        处理多个文件中的所有代码变更
       </strong>
       <br/>
       点击输入框上方文件列表中的
       <strong>
        Accept All
       </strong>
       或
       <strong>
        Reject All
       </strong>
       按钮，可批量接受或拒绝当前项目中所有代码变更。
      </p>
     </li>
     <li>
      <p>
       <strong>
        处理单个文件中的所有代码变更
       </strong>
       <br/>
       在编辑器中打开文件，点击顶部提示栏中的
       <strong>
        Accept
       </strong>
       （快捷键：Command + Enter）或
       <strong>
        Reject
       </strong>
       （快捷键：Command + Backspace）按钮，可批量接受或拒绝当前文件中的所有代码变更。
      </p>
     </li>
     <li>
      <p>
       <strong>
        处理单个代码变更
       </strong>
       <br/>
       在编辑器中，点击某个代码变更右上角的
       <strong>
        ^Y
       </strong>
       或
       <strong>
        ^N
       </strong>
       按钮，以接受或拒绝该特定变更。
      </p>
     </li>
    </ul>
    <p>
     此外，对于聊天中生成的代码片段，你可以按如下方式处理：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/64567e8df5494f2583e1be4626810292.png"/>
    </p>
    <p>
     <strong>
      说明：
     </strong>
    </p>
    <ol>
     <li>
      点击
      <strong>
       Copy
      </strong>
      按钮，将代码片段中的所有代码复制下来。
     </li>
     <li>
      在当前编辑器中打开的文件中，点击任意位置，然后点击
      <strong>
       Insert at Cursor
      </strong>
      按钮，将代码片段中的代码插入到光标所在位置。
     </li>
     <li>
      点击
      <strong>
       Add to New File
      </strong>
      按钮，新建一个文件并将代码片段中的代码添加到该文件中。
     </li>
     <li>
      对于代码变更，点击
      <strong>
       Apply
      </strong>
      按钮，将其应用到相应的文件中。
     </li>
    </ol>
    <blockquote>
     <p>
      注意：与当前项目无关的代码不能被应用。
     </p>
    </blockquote>
    <hr/>
    <p>
     <strong>
      生成和运行命令
     </strong>
    </p>
    <p>
     在聊天模式中，根据你的需求，AI 助手将提供可执行的 Shell 命令，你可以一键运行。
    </p>
    <p>
     你可以根据需要执行以下操作：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e32eada0a8694f6785e45a32b5b8d109.png"/>
    </p>
    <p>
     <strong>
      说明：
     </strong>
    </p>
    <ol>
     <li>
      点击
      <strong>
       Copy
      </strong>
      按钮，将命令复制下来。
     </li>
     <li>
      点击
      <strong>
       Add to Terminal
      </strong>
      按钮，将命令添加到可用终端。
     </li>
     <li>
      点击
      <strong>
       Run
      </strong>
      按钮，在可用终端中执行该命令。
     </li>
    </ol>
    <hr/>
    <p>
     <strong>
      回退到先前版本
     </strong>
    </p>
    <p>
     在聊天模式中，你可以将项目回退到某一轮聊天之前的版本。
    </p>
    <p>
     回退操作不可撤销；
     <br/>
     仅支持在最新的聊天窗口中回退项目；
     <br/>
     仅支持将项目回退到最新 10 轮聊天中的某个版本。
    </p>
    <ul>
     <li>
      <strong>
       找到目标聊天气泡
      </strong>
      <br/>
      在左侧，点击
      <strong>
       Revert
      </strong>
      按钮。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6e66d8beee46406385ecc8cf9096540c.png"/>
    </p>
    <ul>
     <li>
      系统会弹出一个对话框，显示此次回退将影响的文件。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/339e22bec5cf4044ac93d7676a8b3780.png"/>
    </p>
    <ul>
     <li>
      <p>
       点击列表中某个文件的名称，该文件将在编辑器中打开，并显示将要发生的变更。
      </p>
     </li>
     <li>
      <p>
       <strong>
        确认此次回退的影响
       </strong>
       <br/>
       如果影响符合预期，点击
       <strong>
        Confirm
       </strong>
       按钮。
       <br/>
       AI 助手将开始回退该项目。
      </p>
     </li>
    </ul>
    <hr/>
    <p>
     <strong>
      管理历史聊天记录
     </strong>
    </p>
    <p>
     点击聊天窗口右上角的
     <strong>
      Show History
     </strong>
     按钮，左侧将出现历史面板，显示来自聊天模式和构建者模式的所有历史聊天记录。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/489780f840334bdca3783a75734c0d58.png"/>
    </p>
    <ul>
     <li>
      若要查看特定的历史聊天，点击相应条目即可跳转至该部分。
     </li>
     <li>
      若需删除某条历史聊天，悬停至目标条目，点击右侧的
      <strong>
       Delete
      </strong>
      按钮。
     </li>
     <li>
      已删除的聊天记录不可恢复。
     </li>
    </ul>
    <hr/>
    <p>
     <strong>
      内联聊天
     </strong>
    </p>
    <p>
     Trae 提供了嵌入在代码编辑器中的内联聊天框。你可以在编码过程中随时调用内联聊天，与 AI 助手交流的同时保持流畅的编码流程。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b95ba90f30fa457297098fe3819f4b9c.png"/>
    </p>
    <h4>
     <a id="_130">
     </a>
     调用内联聊天
    </h4>
    <p>
     你可以通过两种方式调用内联聊天：
    </p>
    <ul>
     <li>
      在编辑器中将光标置于任意位置，然后使用快捷键
      <strong>
       Command + I
      </strong>
      。
     </li>
     <li>
      在编辑器中选中任意代码，然后使用快捷键
      <strong>
       Command + I
      </strong>
      ，或点击浮动菜单中的
      <strong>
       Edit
      </strong>
      按钮。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/2d894548f78c4e9b9a269e96b6a7819e.png"/>
    </p>
    <h4>
     <a id="_140">
     </a>
     表达你的需求
    </h4>
    <p>
     在内联聊天框中输入你的请求，例如为选中代码添加注释、解释选中代码、优化选中代码等，然后点击右侧的
     <strong>
      Send
     </strong>
     按钮或按键盘上的
     <strong>
      Enter
     </strong>
     键。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/109f7d385576477ebf43861f96795405.png"/>
    </p>
    <h4>
     <a id="_147">
     </a>
     处理响应
    </h4>
    <p>
     发送请求后，AI 助手生成的内容将以差异（diff）格式显示在编辑器中。你可以审阅这些代码变更，并选择接受或拒绝它们。
    </p>
    <ul>
     <li>
      若要接受或拒绝所有变更，点击对话框左下角的
      <strong>
       Accept
      </strong>
      按钮（快捷键：Command + Enter）或
      <strong>
       Reject
      </strong>
      按钮（快捷键：Command + Backspace）。
     </li>
     <li>
      若要接受或拒绝特定变更，点击内容片段右上角的
      <strong>
       ^Y
      </strong>
      （快捷键：Control + Y）或
      <strong>
       ^N
      </strong>
      （快捷键：Control + N）按钮。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ddc5855c1f8042e9af02829cb5d8f2f0.png"/>
    </p>
    <h4>
     <a id="_157">
     </a>
     重试聊天
    </h4>
    <p>
     如果 AI 助手生成的内容不是你想要的，你可以点击聊天框左下角的
     <strong>
      Retry
     </strong>
     按钮，针对同一问题发起另一轮聊天，让 AI 助手重新作答。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9c1f0d1a79a54cfe8dd542e4e2ae4308.png"/>
    </p>
