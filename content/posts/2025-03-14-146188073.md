    <h4>
     <a id="DataSpell__76">
     </a>
     DataSpell 使用
    </h4>
    <p>
     创建 jupyter notebook文件，文档的扩展名为
     <code>
      .ipynb
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/90c27cc4df3649099c4458374b28819a.png"/>
     <br/>
     常用快捷键
    </p>
    <ul>
     <li>
      两种模式通用快捷键
      <ul>
       <li>
        shit+enter，执行本单元代码，并跳转到下一单元格
       </li>
       <li>
        ctrl+enter，执行本单元代码，留在本单元格
       </li>
      </ul>
     </li>
     <li>
      命令模式，按esc进入
      <ul>
       <li>
        Y，cell切换到code模式
       </li>
       <li>
        M，cell切换到markdown模式
       </li>
       <li>
        A，在当前cell的上面添加cell
       </li>
       <li>
        B，在当前cell的下面添加cell
       </li>
       <li>
        双击D：删除当前cell
       </li>
      </ul>
     </li>
     <li>
      编辑模式，按enter进入
      <ul>
       <li>
        多光标操作：ctrl键点击鼠标
       </li>
       <li>
        回退：ctrl+z
       </li>
       <li>
        重做:ctrl+y
       </li>
       <li>
        补全代码：变量、方法后跟tab
       </li>
       <li>
        为一行或多行添加、取消注释 ctrl+/
       </li>
      </ul>
     </li>
    </ul>
