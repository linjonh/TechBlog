    <h3>
     <a id="_5__148">
     </a>
     第 5 步：关键帧动画的定义
    </h3>
    <p>
     在
     <code>
      style.css
     </code>
     的下方，可以看到
     <code>
      @keyframes animate
     </code>
     ：
    </p>
    <pre><code class="prism language-css"><span class="token atrule"><span class="token rule">@keyframes</span> animate</span> <span class="token punctuation">{<!-- --></span>
    <span class="token selector">0%</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">transform</span><span class="token punctuation">:</span> <span class="token function">rotate</span><span class="token punctuation">(</span>0deg<span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
    <span class="token selector">100%</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">transform</span><span class="token punctuation">:</span> <span class="token function">rotate</span><span class="token punctuation">(</span>360deg<span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_163">
     </a>
     解释
    </h4>
    <ul>
     <li>
      我们定义了一个名为
      <code>
       animate
      </code>
      的动画：
      <ul>
       <li>
        在动画开始（
        <code>
         0%
        </code>
        ）时，元素不旋转 (
        <code>
         rotate(0deg)
        </code>
        )；
       </li>
       <li>
        在动画结束（
        <code>
         100%
        </code>
        ）时，元素旋转 360°。
       </li>
      </ul>
     </li>
     <li>
      因为
      <code>
       .ring
      </code>
      引用了
      <code>
       animation: animate 1s linear infinite;
      </code>
      ，所以这个动画会
      <strong>
       每 1 秒旋转一整圈
      </strong>
      ，反复无限次执行。
     </li>
    </ul>
    <blockquote>
     <p>
      若想加快或减慢旋转速度，可以调整
      <code>
       1s
      </code>
      为更小或更大数值；如果想让旋转有加速或减速，可以把
      <code>
       linear
      </code>
      替换为
      <code>
       ease-in-out
      </code>
      等。
     </p>
    </blockquote>

      </strong>
      <ul>
       <li>
        浏览器解析
        <code>
         index.html
        </code>
        ，在
        <code>
         &lt;body&gt;
        </code>
        中找到
        <code>
         .center
        </code>
        容器，并把它居中。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       环与文字
      </strong>
      <ul>
       <li>
        <code>
         .text
        </code>
        文字固定不动，静静地呆在圆心。
       </li>
       <li>
        <code>
         .ring
        </code>
        元素利用关键帧动画不断执行
        <code>
         rotate(360deg)
        </code>
        。由于它有阴影（
        <code>
         box-shadow
        </code>
        ），在旋转时会呈现“环绕”视觉效果。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       无限循环
      </strong>
      <ul>
       <li>
        通过
        <code>
         animation: animate 1s linear infinite;
        </code>
        ，该环会不停地旋转，直到页面被关闭或跳转。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_7__185">
     </a>
     第 7 步：常见问题与可选扩展
    </h3>
    <ol>
     <li>
      <strong>
       如何改变环的颜色？
      </strong>
      <ul>
       <li>
        在
        <code>
         .ring
        </code>
        的
        <code>
         box-shadow: 0 4px 0 #262626;
        </code>
        里，把
        <code>
         #262626
        </code>
        换成其他颜色即可。
       </li>
       <li>
        你也可以给
        <code>
         .ring
        </code>
        一个略带半透明的背景，以达到其他视觉效果。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       想让文字也旋转吗？
      </strong>
      <ul>
       <li>
        只要给
        <code>
         .text
        </code>
        同样设置
        <code>
         animation: animate 1s linear infinite;
        </code>
        即可，不过通常加载指示器的文字会静止显示，故这里选择让文字不旋转。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       加载文本换成图片Logo？
      </strong>
      <ul>
       <li>
        可以把
        <code>
         &lt;div class="text"&gt;LOADING&lt;/div&gt;
        </code>
        换成
        <code>
         &lt;img src="logo.png" class="text"&gt;
        </code>
        ，同时注意
        <code>
         width/height
        </code>
        等，或使用 background-image。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       缓动效果
      </strong>
      <ul>
       <li>
        如果想让环转动时有
        <strong>
         渐慢再加速
        </strong>
        的效果，可以把
        <code>
         animation-timing-function
        </code>
        改为
        <code>
         ease-in-out
        </code>
        ，或在
        <code>
         @keyframes
        </code>
        中设置更多细分点（如
        <code>
         50% { transform: rotate(180deg) }
        </code>
        等）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       自适应屏幕
      </strong>
      <ul>
       <li>
        如果想让环形大小随屏幕变化，可把固定的
        <code>
         300px
        </code>
        改为百分比或相对单位，比如
        <code>
         width: 20vmin; height: 20vmin;
        </code>
        ，让它在小屏上自动缩小，在大屏上放大。
       </li>
      </ul>
     </li>
    </ol>

    <h3>
     <a id="_201">
     </a>
     总结
    </h3>
    <p>
     通过这份拆解式教程，你可以了解本示例的整体思路与各部分功能：
    </p>
    <ol>
     <li>
      <strong>
       HTML 结构
      </strong>
      ：由一个父容器
      <code>
       .center
      </code>
      ，内部包含旋转环
      <code>
       .ring
      </code>
      与文字
      <code>
       .text
      </code>
      。
     </li>
     <li>
      <strong>
       CSS 样式
      </strong>
      ：
      <ul>
       <li>
        绝对定位 +
        <code>
         translate(-50%, -50%)
        </code>
        实现元素在页面中心。
       </li>
       <li>
        圆形 + 阴影 + 关键帧动画，实现旋转的环形效果。
       </li>
       <li>
        层叠两个 300×300 的圆（一个做环旋转，一个做文字展示）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       动画原理
      </strong>
      ： 通过
      <code>
       @keyframes
      </code>
      +
      <code>
       animation
      </code>
      属性，让
      <code>
       .ring
      </code>
      不断地做 0° → 360° 的循环旋转。
     </li>
    </ol>
    <p>
     你可以在此基础上继续完善或扩展，比如换文本、调配颜色、改动旋转速度，甚至加上更多环等等。这样，就能轻松制作一个别致又简洁的“LOADING”动画！祝你学习愉快。
