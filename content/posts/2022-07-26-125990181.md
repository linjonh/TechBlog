    <h4>
     方案一：
    </h4>
    <ul>
     <li>
      关于后端返回时间格式带T问题
     </li>
     <li>
      有时候后端在返回时间数据的时候会格式化一次，前端接收到的数据就是带T格式的，页面展示的时候肯定不要这个带T的，所以需要我们处理一下在进行展示
     </li>
     <li>
      这个是返回数据的效果图（带T的）
     </li>
     <li style="text-align:center;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/915457e378adc407907bc941ccd3e8a4.png"/>
     </li>
     <li>
      我用的elementUI的表格组件，里边有**formatter**属性，借助这个属性将T替换掉。
     </li>
     <li>
      可以直接看代码
     </li>
     <li>
      <pre><code class="language-javascript">&lt;el-table-column
  align="center"
  prop="lastSyncTime"
  label="最新同步时间"
  show-overflow-tooltip
  :formatter="formatter"/&gt;
// 借助formatter对该列数据进行编写</code></pre>
      <p>
      </p>
     </li>
     <li>
      <pre><code class="language-javascript">methods:{
    formatter (row) {
      const zoneDate = new Date(row.lastSyncTime).toJSON()
      const date = new Date(+new Date(zoneDate) + 8 * 3600 * 1000).toISOString().replace(/T/g, ' ').replace(/\.[\d]{3}Z/, '')
      return date
    }
}</code></pre>
     </li>
    </ul>
    <h4>
     方案二
    </h4>
    <blockquote>
     <ul>
      <li>
       利用replace去除T用空格代替  可以用v-if判断一下
      </li>
     </ul>
     <pre><code class="language-javascript">&lt;el-table-column
              align="center"
              label="最新同步时间"
              show-overflow-tooltip&gt;
              &lt;template v-slot="{row}"&gt;
                &lt;span v-if="row.lastSyncTime!==null"&gt;{<!-- -->{row.lastSyncTime.replace('T',' ')}}&lt;/span&gt;
                &lt;span v-else&gt;{<!-- -->{row.lastSyncTime}}&lt;/span&gt;
              &lt;/template&gt;
&lt;/el-table-column&gt;</code></pre>
     <p>
     </p>
