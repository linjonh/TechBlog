     区块链游戏的 1.0  2.0  3.0  4.0 时代
    </h1>
   </div>
  </div>
 </div>

   <div class="htmledit_views" id="content_views">
    <p>
     游戏是区块链技术涉及较广的领域，并且发展效果显著，近一年，区块链游戏掀起了一阵阵热潮，这样的结果来源于区块链游戏的一次次蜕变。
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1dc0fa028683a49b315ff3fcc122775c.png">
      <br/>
     </img>
    </p>
    <span style="font-size:16px;color:#cc0000;">
     <strong>
      1.0 时代：区块链技术开发应用，《加密猫》游戏深受欢迎
     </strong>
    </span>
    <br/>
    <br/>
    当区块链技术最开始应用到游戏中时，开发的游戏仅仅是以收集型游戏为主，最具有代表性的就是《加密猫》，玩家通过充值收集等操作饲养属于自己的宠物，这样的游戏模式虽然较为简单，游戏意义不强，但是依旧保持着非常稳定的发展。《加密猫》相较于其他的游戏，最大的优势就是不断地征集玩家和各方的意见，这也是其可以长期发展的最主要原因。
    <br/>
    <br/>
    <strong>
     <span style="font-size:16px;color:#cc0000;">
      2.0 时代：区块链游戏加入新鲜“血液”，内容不断丰富
     </span>
    </strong>
    <br/>
    <br/>
    随着区块链游戏不断的风靡，也让更多的游戏公司看到了发展前景，也纷纷的加入到了区块链游戏的大军当中，而且游戏模式也有所变化，不只局限于单一的游戏，而是不断地加入新内容，让玩家更好的感受区块链游戏的意义。例如公信宝公司最新开发的“币得”，加入了“一元夺宝”新内容，就是对“新血液”最好的诠释。而“元链星系”和之前的《加密猫》等游戏相比，虽然也是以能量收集为主，但是收集之外，增添了好友互赞，微信步数、知识答题、“矿产”新增等丰富的游戏内容，令人耳目一新。
    <br/>
    <br/>
    <strong>
     <span style="font-size:16px;color:#cc0000;">
      3.0 时代：链上+链下有效结合，玩法多样化出现
     </span>
    </strong>
    <br/>
    <br/>
    2.0时代是成功的，但是，太多收集型游戏的不断涌现，也让许多玩家腻而生厌，这也就激发了更多的游戏开发者努力开发更多有趣的游戏。区块链游戏也加入了模拟经营类游戏、体育竞猜类游戏、放置类游戏等多样化的游戏，将链上和链下进行结合，尽最大的可能满足玩家需求，但当前多处于内测阶段，距离上市还需一段时间。
    <br/>
    <strong>
     <br/>
     <span style="font-size:16px;color:#cc0000;">
      4.0 时代：区块链游戏全面升级，向传统游戏看齐
     </span>
    </strong>
    <br/>
    <br/>
    就目前来看，区块链游戏的发展是成功的，如今，众多的游戏公司都将区块链游戏和传统游戏进行有效结合，吸取传统游戏中的精华部分，将区块链游戏做大、做全、做精。这样的游戏将是游戏界首屈一指的游戏应用，其发展前景也是不可估量的。随着区块链游戏向传统游戏看齐，全面升级的这份期许，将最终得以实现。
    <br/>
    <br/>
    <p>
     区块链游戏的不断发展和蜕变，将其不可估量的发展前景和实力展现在世人面前，虽然区块链游戏在发展过程中还需要不断地探索和改革，但短短一年内就取得如此惊人的成绩。相信区块链游戏在不久的未来，一定会让玩家“称心满意”。
    </p>
