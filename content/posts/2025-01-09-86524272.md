    </h1>

    <h4>
     <strong>
      计算机硬件包括：
      <span style="color:#f33b45;">
       CPU、内存以及IO设备
      </span>
      。
     </strong>
    </h4>
    <p>
     <a href="https://www.baidu.com/s?wd=%E6%93%8D%E4%BD%9C%E7%B3%BB%E7%BB%9F&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      操作系统
     </a>
     有五大功能：
     <br/>
     <br/>
     1、处理机管理：主要控制和管理
     <strong>
      CPU
     </strong>
     的工作。
     <strong>
      （CPU管理）
     </strong>
    </p>
    <p>
     <strong>
      单CPU下和多CPU下。
     </strong>
     <br/>
     <br/>
     2、存储管理：主要进行
     <strong>
      内存
     </strong>
     的分配和管理
     <strong>
      （内存管理）
     </strong>
    </p>
    <p>
     <span style="color:#f33b45;">
      <strong>
       内存分配、地址映射、内存保护与共享、虚拟内存等。
      </strong>
     </span>
     <br/>
     <br/>
     3、设备管理：主要管理基本的
     <strong>
      输入输出
     </strong>
     设备
     <strong>
      （IO管理）
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:#f33b45;">
       完成用户的 I/O 请求，方便用户使用各种设备，并提高设备的利用率。
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <span style="color:#f33b45;">
       主要包括缓冲管理、设备分配、设备处理、虛拟设备等。
      </span>
     </strong>
     <br/>
     <br/>
     4、
     <a href="https://www.baidu.com/s?wd=%E6%96%87%E4%BB%B6%E7%AE%A1%E7%90%86&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      文件管理
     </a>
     ：负责对计算机
     <strong>
      文件
     </strong>
     的组织、存储、操作和保护等。
     <strong>
      （文件管理）
     </strong>
    </p>
    <p>
     <span style="color:#f33b45;">
      <strong>
       文件存储空间的管理、目录管理、文件读写管理和保护等。
      </strong>
     </span>
     <br/>
     <br/>
     5、
     <a href="https://www.baidu.com/s?wd=%E8%BF%9B%E7%A8%8B%E7%AE%A1%E7%90%86&amp;tn=SE_PcZhidaonwhc_ngpagmjz&amp;rsv_dl=gh_pc_zhidao" rel="nofollow">
      进程管理
     </a>
     ：也称为
     <strong>
      作业
     </strong>
     管理，是指对计算机所进行的操作进行管理。
     <strong>
      （作业管理或者进程管理）
     </strong>
    </p>
    <p>
     <span style="color:#f33b45;">
      <strong>
       进程控制、进程同步、进程通信、死锁处理、处理机调度等。
      </strong>
     </span>
    </p>
