    <h4>
     使用SQL语句，在MySQL数据库的表中搜索出固定时间格式相关的数据：
    </h4>
    <p>
    </p>
    <p>
     <strong>
      格式
     </strong>
     ：
     <span style="color:#3399ea;">
      SELECT * FROM
     </span>
     <span style="color:#f33b45;">
      表名
     </span>
     <span style="color:#3399ea;">
      WHERE  DATE_FORMAT(
     </span>
     <span style="color:#f33b45;">
      列名
     </span>
     <span style="color:#3399ea;">
      ,'%Y-%m-%d')=‘
     </span>
     <span style="color:#f33b45;">
      时间格式
     </span>
     <span style="color:#3399ea;">
      ’
     </span>
    </p>
    <p>
     <strong>
      举例
     </strong>
     ：
     <span style="color:#7c79e5;">
      SELECT * FROM  node  WHERE  DATE_FORMAT(Datatime,'%Y-%m-%d')=‘2018-12-5’
     </span>
    </p>
    <p>
     <strong>
      解释
     </strong>
     ： 在
     <span style="color:#ffbb66;">
      node
     </span>
     表中找到时间格式是
     <span style="color:#f33b45;">
      ‘2018-12-5’
     </span>
     格式的所有数据（你得到的是显示时间当天的数据），当然你也可以找到某月份的数据  只要把时间格式  改为
     <span style="color:#f33b45;">
      ‘2018-12’
     </span>
     ，这样你得到的是 显示时间为
     <span style="color:#f33b45;">
      2018年12月份
     </span>
     的所有数据。
    </p>
    <p>
     <strong>
      解释一下：
     </strong>
    </p>
    <p>
     <strong>
     </strong>
     这里的列名代表的列，它的类型  应该是 时间类型，而时间格式也必须是该列部分显示格式，不可能数据库的数据格式是
     <span style="color:#f33b45;">
      2012-5-4  10:12:30
     </span>
     这样的，然后你肯定不能找
     <span style="color:#f33b45;">
      2012/5/4
     </span>
     这种格式的数据，因为它不是该列的部分 显示格式。
    </p>
