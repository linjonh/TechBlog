    <p>
     ‌Pandoc‌是一个开源的文档转换工具，由John MacFarlane开发，旨在提供一个通用的文档转换解决方案。它支持多种输入和输出格式，能够高效地将不同格式的文档进行转换‌
    </p>
    <p>
     功能
     <br/>
     Pandoc支持以下格式之间的转换：
    </p>
    <p>
     **Markdown、reStructuredText、Textile、EmacsOrg-Mode、EmacsMuse、djot、HTML、LaTeX、EPUB、FictionBook2、OpenDocument XML、BibTeX、BibLaTeX、CSL JSON、CSL YAML、RISC、EndNote XML、Microsoft Word docx、Rich Text Format (RTF)、OpenOffice/LibreOffice ODT、Jupyter notebook (ipynb)、PDF、XML (DocBook, JATS, TEI)等‌
    </p>
    <p>
     安装与使用
     <br/>
     Pandoc的安装方法因操作系统而异：
    </p>
    <p>
     ‌Windows‌：可以从Pandoc官方网站下载安装程序并按照提示完成安装。
     <br/>
     ‌macOS‌：可以使用Homebrew (brew install pandoc) 或者直接下载.pkg文件进行安装。
     <br/>
     ‌Linux‌：可以通过包管理器安装，如使用snap (snap install pandoc)。
     <br/>
     安装完成后，可以通过命令行验证是否安装成功：pandoc --version。
    </p>
    <p>
     基本用法示例
     <br/>
     ‌Markdown转HTML‌：pandoc input.md -o output.html
     <br/>
     ‌Markdown转PDF‌：需要安装LaTeX，命令为 pandoc input.md -o output.pdf
     <br/>
     ‌合并多个文件‌：pandoc file1.md file2.md -o combined.docx
     <br/>
     ‌TXT转Markdown‌：pandoc input.txt -t markdown -o output.md
     <br/>
     ‌DOCX转Markdown‌：pandoc input.docx -t markdown -o output.md
     <br/>
     ‌XLSX转Markdown‌：虽然Pandoc不直接支持XLSX格式，但可以先将XLSX转换为CSV，然后使用Pandoc进行转换，命令为 pandoc output.csv -t markdown -o output.md‌
    </p>
    <p>
     下载地址：
     <br/>
     https://github.com/jgm/pandoc/releases/tag/3.6.3
    </p>
