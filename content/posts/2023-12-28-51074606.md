     (三) 数据库安全检查点
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     1. 系统数据是否加密
    </p>
    <ul style="margin:0px 0px 0px 30px; padding:0px; word-break:break-all; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      尽量不要使用sa 账号
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      严格控制数据库用户的权限,不要轻易给用户直接的查询\更改\插入\删除的权限
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      数据库的账号和密码(还有端口号),是不是直接写在配置文件里未进行加密
     </li>
    </ul>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     2. 系统数据的完整性
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     3. 系统该数据的可管理性
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     4. 系统数据的独立性
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     5. 系统数据的可备份和恢复能力
    </p>
    <ul style="margin:0px 0px 0px 30px; padding:0px; word-break:break-all; font-family:'Helvetica Neue',Helvetica,Arial,sans-serif; font-size:13px; line-height:19.5px; background-color:rgb(245,245,245)">
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      服务器突然断电--配置文件的错误导致无法访问或数据丢失
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      重做日志发生损坏--测试数据库管理员能否把数据恢复到故障恢复之前
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      硬盘发生故障--测试备份文件异地存放的有效性
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:disc">
      数据批量更新的错误处理--主要测试数据库管理员在更新之前是否有对数据库进行备份的习惯.
     </li>
    </ul>
