    <p>
     1、嵌入式系统概念：用于控制、监视或者辅助操作机器和设备的装置（不一定是电路，或者计算机），此为广义的嵌入式系统
     <br/>
     2、嵌入式系统是以应用为中心，以计算机技术为基础，软件硬件可裁剪，适应应用系统对功能、可靠性、成本、体积、功耗严格要求的专用计算机系统，此为狭义的嵌入式系统
     <br/>
     可理解为：所有以专用CPU芯片（或者类CPU芯片）为核心，只为实现某一特定功能的一类计算机系统。
     <br/>
     3、主要特点：
     <br/>
     a、是一个“计算机”系统
     <br/>
     b、嵌在其他设备或商品中。使得商品变得更自动、智能、好用或多功能
     <br/>
     c、针对应用定值，也叫专用计算机。满足应用要求，如功能、成本、可靠性、体积、冲击和震动，离开环境或载体系统会失去意义，一辈子就做一件事。
     <br/>
     d、系统硬件软件一般需要重新设计或裁剪。满足应用要求即可，一般性能越低越好，和通用计算机相反。
     <br/>
     e、一个不可能被垄断的行业。数量、计算能力
     <br/>
     4、嵌入式技术分类：
     <br/>
     a、计算机分为通用，专用（嵌入式）
     <br/>
     b、嵌入式：包括单片机系统（&gt;50%），DSP系统（数字信号处理），ASIC系统（专用集成电路，用电路来解决复杂问题），嵌入式计算机系统（ARM&gt;90%）
     <br/>
     c、这里要注意广义嵌入式系统和狭义嵌入式系统之间的关系，狭义的嵌入式系统一般指的是嵌入式计算机系统。
     <br/>
     5、嵌入式系统形式分类：
     <br/>
     分为设备级、板级（有一个电路板）、片级（如耳机，很小）
     <br/>
     6、嵌入式计算机系统技术结构：包含嵌入式系统技术专家、嵌入式硬件工程师、嵌入式软件工程师，以及三方面都有涉猎的嵌入式操作系统工程师
    </p>
