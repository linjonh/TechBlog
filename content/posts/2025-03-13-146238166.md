    <h2>
     <a id="CSS__0">
     </a>
     CSS 知识点总结１
    </h2>
    <p>
     今天写了两个页面,用到的知识点,总结一下
    </p>
    <h3>
     <a id="1_Flexbox__2">
     </a>
     1. Flexbox 布局
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        display: flex;
       </code>
      </strong>
      ：启用 Flexbox 布局，用于创建灵活的容器。
     </li>
     <li>
      <strong>
       <code>
        flex-direction: column;
       </code>
      </strong>
      ：将子元素垂直排列。
     </li>
     <li>
      <strong>
       <code>
        justify-content
       </code>
      </strong>
      ：控制子元素在主轴上的对齐方式。
      <ul>
       <li>
        <code>
         justify-content: space-between;
        </code>
        ：子元素在主轴两端对齐，剩余空间均匀分布。
       </li>
       <li>
        <code>
         justify-content: center;
        </code>
        ：子元素在主轴居中。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        align-items
       </code>
      </strong>
      ：控制子元素在交叉轴上的对齐方式。
      <ul>
       <li>
        <code>
         align-items: center;
        </code>
        ：子元素在交叉轴居中。
       </li>
       <li>
        <code>
         align-items: flex-end;
        </code>
        ：子元素在交叉轴末端对齐。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        gap: 10px;
       </code>
      </strong>
      ：设置子元素之间的间距。
     </li>
    </ul>
    <h3>
     <a id="2_Grid__13">
     </a>
     2. Grid 布局
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        display: grid;
       </code>
      </strong>
      ：启用 Grid 布局，用于创建网格容器。
     </li>
     <li>
      <strong>
       <code>
        grid-template-columns: repeat(2, 1fr);
       </code>
      </strong>
      ：定义两列，每列宽度相等。
     </li>
     <li>
      <strong>
       <code>
        gap: 30px;
       </code>
      </strong>
      ：设置网格项之间的间距。
     </li>
    </ul>
    <h3>
     <a id="3__18">
     </a>
     3. 盒模型
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        padding: 20px;
       </code>
      </strong>
      ：设置内边距。
     </li>
     <li>
      <strong>
       <code>
        margin: 20px;
       </code>
      </strong>
      ：设置外边距。
     </li>
     <li>
      <strong>
       <code>
        margin: 0 auto;
       </code>
      </strong>
      ：水平居中元素。
     </li>
     <li>
      <strong>
       <code>
        border-radius: 10px;
       </code>
      </strong>
      ：设置圆角。
     </li>
     <li>
      <strong>
       <code>
        border: 1px solid #ddd;
       </code>
      </strong>
      ：设置边框。
     </li>
    </ul>
    <h3>
     <a id="4__25">
     </a>
     4. 定位
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        position: relative;
       </code>
      </strong>
      ：设置相对定位，为子元素的绝对定位提供参考。
     </li>
     <li>
      <strong>
       <code>
        position: absolute;
       </code>
      </strong>
      ：设置绝对定位，脱离文档流。
      <ul>
       <li>
        <code>
         left: 12px;
        </code>
        ：距离左侧 12px。
       </li>
       <li>
        <code>
         top: 50%;
        </code>
        ：距离顶部 50%。
       </li>
       <li>
        <code>
         transform: translateY(-50%);
        </code>
        ：精确垂直居中。
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <a id="5__32">
     </a>
     5. 过渡效果
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        transition: box-shadow 0.3s ease;
       </code>
      </strong>
      ：设置
      <code>
       box-shadow
      </code>
      属性的过渡效果。
     </li>
     <li>
      <strong>
       <code>
        transition: border-color 0.3s ease;
       </code>
      </strong>
      ：设置
      <code>
       border-color
      </code>
      属性的过渡效果。
     </li>
    </ul>
    <h3>
     <a id="6__36">
     </a>
     6. 字体和颜色
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        font-family: Arial, sans-serif;
       </code>
      </strong>
      ：设置字体。
     </li>
     <li>
      <strong>
       <code>
        font-size: 16px;
       </code>
      </strong>
      ：设置字体大小。
     </li>
     <li>
      <strong>
       <code>
        color: #333;
       </code>
      </strong>
      ：设置字体颜色。
     </li>
     <li>
      <strong>
       <code>
        background-color: #fbfbfadd;
       </code>
      </strong>
      ：设置背景颜色（带透明度）。
     </li>
    </ul>
    <h3>
     <a id="7__42">
     </a>
     7. 响应式设计
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        max-width: 800px;
       </code>
      </strong>
      ：限制容器的最大宽度，确保在不同屏幕尺寸下保持布局。
     </li>
     <li>
      <strong>
       <code>
        width: 90%;
       </code>
      </strong>
      ：设置宽度为父容器的 90%。
     </li>
    </ul>
    <h3>
     <a id="8__46">
     </a>
     8. 其他
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        min-height: 100vh;
       </code>
      </strong>
      ：设置容器的最小高度为视口高度。
     </li>
     <li>
      <strong>
       <code>
        cursor: pointer;
       </code>
      </strong>
      ：设置鼠标悬停时为手型，表示可点击。
     </li>
     <li>
      <strong>
       <code>
        outline: none;
       </code>
      </strong>
      ：移除聚焦时的默认边框。
     </li>
    </ul>
    <h3>
     <a id="_51">
     </a>
     总结
    </h3>
    <ul>
     <li>
      <strong>
       布局
      </strong>
      ：广泛使用 Flexbox 和 Grid 布局，创建灵活的页面结构。
     </li>
     <li>
      <strong>
       定位
      </strong>
      ：使用相对定位和绝对定位，精确控制元素的位置。
     </li>
     <li>
      <strong>
       盒模型
      </strong>
      ：通过
      <code>
       padding
      </code>
      、
      <code>
       margin
      </code>
      和
      <code>
       border
      </code>
      控制元素的间距和边框。
     </li>
     <li>
      <strong>
       响应式设计
      </strong>
      ：通过
      <code>
       max-width
      </code>
      和百分比宽度，确保页面在不同设备上表现良好。
     </li>
     <li>
      <strong>
       过渡效果
      </strong>
      ：为交互元素添加平滑的过渡效果，提升用户体验。
     </li>
    </ul>
