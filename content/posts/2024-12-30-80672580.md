     解决linux配置JAVA_HOME 不生效问题
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      centos7,遇到安装的java版本(想要jdk1.7)和执行java -version看到的版本(自带的jdk1.8)不一致的问题
     </span>
     <br/>
     <span style="font-weight:700;">
      通过查看多方资料,发现了问题所在
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
    </p>
    <p>
     <span style="font-weight:700;">
      1. 使用which java发现出来的是/usr/bin/java
     </span>
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e19ff628bab18f3d9c9b5aa5c5df8cb5.png"/>
    </p>
    <p>
     <span style="font-weight:700;">
      进入到/usr/bin后查看,发现java是一个链接.
     </span>
    </p>
    <p>
     <span style="font-weight:700;">
      链接到了/etc/alternatives
     </span>
    </p>
    <p>
     <span style="font-weight:700;">
      而进入到/etc/alternatives下查看,发现java又是一个链接,如下:
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0ff232107f6c5f58b4b48dbf61f6b84e.png">
      <br/>
     </img>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      其实/etc/alternatives里的
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="color:#ff0000;">
      /usr/lib/jvm/java-1.8.0-openjdk-1.8.0.171-8.b10.el7_5.x86_64/jre/bin/java
     </span>
     <br/>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      才是真正自带的Java.
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <strong>
      解决的方法如下:
     </strong>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      ______________________________________________________________________________________
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      1.删除/etc/alternatives和/usr/bin中的java;
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/687350932f11cfd1dc705e08666f521d.png">
       <br/>
      </img>
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      2.到官网下载JDK,并上传解压到/opt
     </span>
     <br/>
     <span style="font-weight:700;">
      (此路径时楼主自己定的,你也可以选择其他的路径进行解压,都一样)中;
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      最后,将安装路径添加到配置文件/etc/profile
     </span>
    </p>
    <pre class="prettyprint" style="font-size:14px;line-height:22px;"><code class="hljs bash has-numbering">vi /etc/profile

<span class="hljs-keyword">export</span>  JAVA_HOME=/opt/jdk1.<span class="hljs-number">7.0</span>_45
<span class="hljs-keyword">export</span>  PATH=<span class="hljs-variable">$JAVA_PATH</span>/bin:<span class="hljs-variable">$PATH</span>

<span class="hljs-built_in">source</span>  /etc/profile</code></pre>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:700;">
      问题即可解决.
     </span>
    </p>
