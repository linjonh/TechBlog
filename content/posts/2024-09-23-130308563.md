    <h3>
     概述
    </h3>
    <p>
     在这篇短文中，我们将会展示如何把 Map 中的值取出来，转换为一个
     <em>
      Array,
     </em>

     </em>

     <em>
      Set
     </em>

    <p>

     当然，你可以使用 Java JDK 来进行转换，你也可以使用 Guava 来进行转换。

    </p>
    <p>
     首先，让我们来看看，如何使用原生的 Java JDK把一个 Map 的值换行为 Array。
    </p>
    <pre><code>    @Test
    public final void givenUsingCoreJava_whenMapValuesConvertedToArray_thenCorrect() {
        final Map&lt;Integer, String&gt; sourceMap = createMap();

        final Collection&lt;String&gt; values = sourceMap.values();
        final String[] targetArray = values.toArray(new String[0]);
    }

</code></pre>
<p>
在上面的代码中，我们使用了
<code>
values.toArray(new String[0]);
</code>
来对数组变量进行初始化。
</p>
<p>
根据：
<a class="has-card" href="https://shipilev.net/blog/2016/arrays-wisdom-ancients/#_conclusion" rel="nofollow" title="Arrays of Wisdom of the Ancients">

<em>

</p>
<h3>
Map 的值转换为 List
</h3>
<p>
下面，让我们看看如何使用原生 Java 来把一个 Map 中的值转换为 List。
</p>
<pre><code> @Test
public final void givenUsingCoreJava_whenMapValuesConvertedToList_thenCorrect() {
final Map&lt;Integer, String&gt; sourceMap = createMap();

        final List&lt;String&gt; targetList = new ArrayList&lt;&gt;(sourceMap.values());
    }

</code></pre>
<p>
可以看到，我们还是使用了 Map 中提供的 values 方法。
</p>
<p>
使用 Guava
</p>
<pre><code> @Test
public final void givenUsingGuava_whenMapValuesConvertedToList_thenCorrect() {
final Map&lt;Integer, String&gt; sourceMap = createMap();

        final List&lt;String&gt; targetList = Lists.newArrayList(sourceMap.values());
    }

</code></pre>
<p>
简单来说，我们只是使用 Guava 的 Lists 方法进行了下包装，核心还是使用 Map 自带的 values 方法。
</p>
