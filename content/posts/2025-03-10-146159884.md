    <h3 style="margin-left:0; margin-right:0; text-align:start">
     <strong>
      <span style="color:#34495e">
       1. 题目
      </span>
     </strong>
    </h3>
    <h4 style="margin-left:0; margin-right:0; text-align:start">
     <strong>
      <span style="color:#34495e">
       <strong>
        描述
       </strong>
      </span>
     </strong>
    </h4>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      给定一个单链表的头结点pHead(该头节点是有值的，比如在下图，它的val是1)，长度为n，反转该链表后，返回新链表的表头。
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      数据范围： 0&lt;≤
      <em>
       n
      </em>
      ≤1000
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      要求：空间复杂度 O(1) ，时间复杂度 O(n)。
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      如当输入链表{1,2,3}时，
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      经反转后，原链表变为{3,2,1}，所以对应的输出为{3,2,1}。
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      以上转换过程如下图所示：
     </span>
    </p>
    <p style="margin-left:0.8em; margin-right:0.8em; text-align:center">
     <span style="color:#34495e">
      <img alt="" src="https://i-blog.csdnimg.cn/direct/080bbbc3083d475e8a022c7093fe921d.png"/>
     </span>
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:start">
     <strong>
      <span style="color:#34495e">
       <strong>
        示例1
       </strong>
      </span>
     </strong>
    </h4>
    <p>
     输入：
    </p>
    <pre>  {1,2,3}</pre>
    <p>
     返回值：
    </p>
    <pre>   {3,2,1}</pre>
    <h4>
     <strong>
      示例2
     </strong>
    </h4>
    <p>
     输入：

    <pre>  {}</pre>
    <p>
     返回值：
    </p>
    <pre>  {}</pre>
    <p>
     说明：
    </p>
    <pre>  空链表则输出空                 </pre>
    <h3 style="margin-left:0; margin-right:0; text-align:start">
     <strong>
      <span style="color:#34495e">
       2. 解题思路
      </span>
     </strong>
    </h3>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      假如说这是我们的链表，结构如下图所示：
     </span>
    </p>
    <p style="margin-left:0.8em; margin-right:0.8em; text-align:center">
     <span style="color:#34495e">
      <img alt="" src="https://i-blog.csdnimg.cn/direct/bc08670c40cf486387908e7f93681304.png"/>
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      <strong>
       第一步
      </strong>
      ：定义3个指针变量，pre（序节点）、cur（当前操作的节点）和nxt（当前操作的下一个节点），结构如下图所示：
     </span>
    </p>
    <p style="margin-left:0.8em; margin-right:0.8em; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/0832da1e8ec64cf5bc93444ac8602260.png"/>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      <strong>
       第二步
      </strong>
      ：通过更改刚刚定义的3个指针变量反转链表节点。
     </span>
    </p>
    <ul style="margin-left:.8em; margin-right:.8em">
     <li>
      <p style="margin-left:.5rem; margin-right:0">
       更改pre的指针域（next）指向：
      </p>
     </li>
    </ul>
    <p style="margin-left:0.5rem; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/3021deaf64de4599abdc982992316b9a.png"/>
    </p>
    <ul style="margin-left:.8em; margin-right:.8em">
    </ul>
    <ul style="margin-left:.8em; margin-right:.8em">
     <li>
      <p style="margin-left:.5rem; margin-right:0">
       更改cur的指针域（next）指向：
      </p>
      <p style="margin-left:.5rem; margin-right:.5rem">
       首先更改cur的指针域（next）指向，让它指向pre；之后再移动pre到cur除，最后移动cur到nxt处。
      </p>
     </li>
    </ul>
    <p style="margin-left:0.5rem; margin-right:0.5rem; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/6b275b3ef515459aa3e8882c773faf02.png"/>
    </p>
    <ul style="margin-left:.8em; margin-right:.8em">
    </ul>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     这时，已经完成了链表的第一个和第二节点的反转。
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      <strong>
       第三步：
      </strong>
      重复第二步的操作，直到链表的所有节点反转完成。
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      如果cur指针变量指向的节点为Null时，就说明所有节点都完成了反转，循环退出，因此反转链表循环的条件：cur！=NUll。
     </span>
    </p>
    <p style="margin-left:.8em; margin-right:.8em; text-align:start">
     <span style="color:#34495e">
      最后返回反转之后的头结点，头结点就是pre所指向的内容。
     </span>
    </p>
    <p style="margin-left:0.8em; margin-right:0.8em; text-align:center">

func ReverseList(pHead *ListNode) *ListNode {
	if pHead == nil {
		return pHead //节点为空，直接返回
	}
	var pre *ListNode //（操作的）前序节点
	cur := pHead      //（操作的）当前节点
	nxt := pHead      //（操作的）下一个节点
	for cur != nil {
		nxt = cur.Next //移动nxt指针
		cur.Next = pre // 更改当前节点（cur）指针域的指向
		pre = cur      //移动pre指针
		cur = nxt      //移动cur指针
	}
	return pre //返回反转之后的头结点
}

// Insert 从链表节点尾部添加节点
func (ln *ListNode) Insert(val int) {
	if ln == nil {
		return
	}
	//创建一个新节点
	newNode := &amp;ListNode{Val: val}
	cur := ln
	//找到链表的末尾节点
	for cur.Next != nil {
		cur = cur.Next
	}
	//末尾节点的next指针域连接新节点
	cur.Next = newNode
}

// Print 从链表头结点开始打印链表的值
func (ln *ListNode) Print() {
	if ln == nil {
		return
	}
	cur := ln
	//遍历每一个节点
	for cur != nil {
		fmt.Print(cur.Val, "\t")
		cur = cur.Next //更改指针变量的指向
	}
	fmt.Println()
}
