    <h3>
     1 宾语从句
    </h3>
    <ol>
     <li>
      <h4>
       <strong>
        定义
       </strong>
      </h4>
      <ol>
       <li>
        在主从复合句中充当宾语，位于及物动词、介词或复合谓语之后的从句。
       </li>
      </ol>
     </li>
     <li>
      <h4>
       <strong>
        引导词
       </strong>
      </h4>
      <ul>
       <li>
        <span style="color:#fe2c24">
         <strong>
          综述：
         </strong>
        </span>
        <ul>
         <li>
          that（可省略）、if/whether、连接代词（what、which、who、whom、whose 等）和连接副词（when、where、how、why 等）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         that
        </strong>
        ：无实际意义，在从句中不作成分，常可省略。
        <ul>
         <li>
          例如：I think (that) he is right.
         </li>
        </ul>
       </li>
       <li>
        <strong>
         if/whether
        </strong>
        ：表示 “是否”，在从句中不作成分。
        <ul>
         <li>
          例如：I don't know if/whether he will come.
         </li>
        </ul>
       </li>
       <li>
        <strong>
         连接代词（
        </strong>
        在从句中作主语、宾语、定语等
        <strong>
         ）
        </strong>
        ：
        <ul>
         <li>
          <span style="color:#fe2c24">
           <strong>
            what（什么）
           </strong>
          </span>
         </li>
         <li>
          <span style="color:#fe2c24">
           <strong>
            which（哪一个）
           </strong>
          </span>
         </li>
         <li>
          <span style="color:#fe2c24">
           <strong>
            who（谁）
           </strong>
          </span>
         </li>
         <li>
          <span style="color:#fe2c24">
           <strong>
            whom（谁，宾格）
           </strong>
          </span>
         </li>
         <li>
          <span style="color:#fe2c24">
           <strong>
            whose（谁的）
           </strong>
          </span>
         </li>
         <li>
          例如
          <ul>
           <li>
            Do you know what he wants？
           </li>
          </ul>
         </li>
        </ul>
       </li>
       <li>
        <strong>
         连接副词（
        </strong>
        在从句中作状语
        <strong>
         ）
        </strong>
        <ul>
         <li>
          <strong>
           <span style="color:#fe2c24">
            when（什么时候）,
           </span>
          </strong>
         </li>
         <li>
          <strong>
           <span style="color:#fe2c24">
            where（在哪里）,
           </span>
          </strong>
         </li>
         <li>
          <strong>
           <span style="color:#fe2c24">
            how（怎样）,
           </span>
          </strong>
         </li>
         <li>
          <strong>
           <span style="color:#fe2c24">
            why（为什么）
           </span>
          </strong>
         </li>
         <li>
          例如
          <ul>
           <li>
            Can you tell me how I can get to the station?
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h4>
       <strong>
        语序
       </strong>
      </h4>
      <ul>
       <li>
        宾语从句要用陈述语序，即 “主语 + 谓语 + 其他”。例如：
        <ul>
         <li>
          错误：Do you know where does he live?
         </li>
         <li>
          正确：Do you know where he lives?
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h4>
       <strong>
        时态
       </strong>
      </h4>
      <ul>
       <li>
        主句是一般现在时，从句根据实际情况用各种时态。
        <ul>
         <li>
          例如：He says (that) he will go to Beijing tomorrow.
         </li>
        </ul>
       </li>
       <li>
        主句是一般过去时，从句要用相应的过去时态（一般过去时、过去进行时、过去将来时、过去完成时）。
        <ul>
         <li>
          例如：He said (that) he was reading a book at that time.
         </li>
         <li>
          He asked me if I could help him.（“if” 引导宾语从句，作 “asked” 的宾语）
         </li>
        </ul>
       </li>
       <li>
        从句表示客观真理、科学事实时，无论主句是什么时态，从句都用一般现在时。
        <ul>
         <li>
          例如：The teacher told us that the earth moves around the sun.
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     2 定语从句
    </h3>
    <ol>
     <li>
      <h4>
       <strong>
        定义
       </strong>
      </h4>
      <ol>
       <li>
        在句中作定语，用来修饰一个名词或代词（即先行词）的从句。
       </li>
      </ol>
     </li>
     <li>
      <h4>
       <strong>
        关系词
       </strong>
      </h4>
      <ul>
       <li>
        <strong>
         概述
        </strong>
        <ul>
         <li>
          关系词包括关系代词（who、whom、whose、which、that）和关系副词（when、where、why）。
         </li>
         <li>
          关系代词在从句中可作主语、宾语、定语等成分，关系副词在从句中作状语。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         关系代词
        </strong>
        <ul>
         <li>
          who（指人，作主语或宾语）
         </li>
         <li>
          whom（指人，作宾语）
         </li>
         <li>
          whose（指人或物，作定语）
         </li>
         <li>
          which（指物，作主语或宾语）
         </li>
         <li>
          that（指人或物，作主语或宾语）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         关系副词
        </strong>
        ：
        <ul>
         <li>
          when（指时间，在从句中作时间状语）
         </li>
         <li>
          where（指地点，在从句中作地点状语）
         </li>
         <li>
          why（指原因，在从句中作原因状语）。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h4>
       <strong>
        示例
       </strong>
      </h4>
      <ul>
       <li>
        The man who is standing there is my teacher.（who 指人，在从句中作主语）
       </li>
       <li>
        The book which I bought yesterday is very interesting.（which 指物，在从句中作宾语）
       </li>
       <li>
        I still remember the day when I first met him.（when 指时间，作时间状语）
       </li>
       <li>
        This is the house where I lived last year.（where 指地点，作地点状语）
       </li>
       <li>
        I like the book that/which is very interesting.（“that/which” 引导定语从句，修饰 “book”，在从句中作主语）
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     3 状语从句
    </h3>
    <ol>
     <li>
      <h4>
       <strong>
        定义
       </strong>
      </h4>
      <ol>
       <li>
        在复合句中，由从句表示的状语称作状语从句。
       </li>
       <li>
        它可以用来修饰谓语、非谓语动词、定语、状语或整个句子。
       </li>
       <li>
        根据其表达的含义，状语从句可分为时间、地点、原因、目的、结果、条件、让步、比较等多种类型。
       </li>
      </ol>
     </li>
     <li>
      <h4>
       <strong>
        引导词及分类
       </strong>
      </h4>
      <ol>
       <li>
        时间状语从句（when、while、as 等）；
       </li>
       <li>
        地点状语从句（where、wherever 等）；
       </li>
       <li>
        原因状语从句（because、since、as 等）；
       </li>
       <li>
        目的状语从句（so that、in order that 等）；
       </li>
       <li>
        结果状语从句（so...that...、such...that... 等）；
       </li>
       <li>
        条件状语从句（if、unless 等）；
       </li>
       <li>
        让步状语从句（although、though、even though、even if 等）。
       </li>
      </ol>
     </li>
    </ol>
    <p>
    </p>
    <ol>
     <li>
      <h5>
       <strong>
        时间状语从句
       </strong>
      </h5>
      <ul>
       <li>
        since（自从）, until（直到）等。
        <ul>
         <li>
          When I got home, my mother was cooking.（when 引导时间状语从句）
         </li>
         <li>
          When I grow up, I want to be a teacher.（“When” 引导时间状语从句）
         </li>
        </ul>
       </li>
       <li>
        before（在…… 之前）,
       </li>
       <li>
        after（在…… 之后）,
       </li>
       <li>
        as（当…… 时；一边…… 一边……）,
       </li>
       <li>
        while（在…… 期间）,
       </li>
       <li>
        when（当…… 时候）,
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        地点状语从句
       </strong>
      </h5>
      <ul>
       <li>
        where（在…… 地方）,
       </li>
       <li>
        wherever（无论在哪里）等。
        <ul>
         <li>
          Where there is a will, there is a way.（where 引导地点状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        原因状语从句
       </strong>
      </h5>
      <ul>
       <li>
        because（因为）,
       </li>
       <li>
        since（既然）,
       </li>
       <li>
        as（由于）等。
        <ul>
         <li>
          I didn't go to school because I was ill.（because 引导原因状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        目的状语从句
       </strong>
      </h5>
      <ul>
       <li>
        so that（以便）,
       </li>
       <li>
        in order that（为了）等。
        <ul>
         <li>
          She gets up early so that she can catch the first bus.（so that 引导目的状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        结果状语从句
       </strong>
      </h5>
      <ul>
       <li>
        so...that...（如此…… 以至于……）,
       </li>
       <li>
        such...that...（如此…… 以至于……）等。
        <ul>
         <li>
          He is so young that he can't go to school.（so...that... 引导结果状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        条件状语从句
       </strong>
      </h5>
      <ul>
       <li>
        if（如果）,
       </li>
       <li>
        unless（除非）等。
        <ul>
         <li>
          If it rains tomorrow, we will stay at home.（if 引导条件状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h5>
       <strong>
        让步状语从句
       </strong>
      </h5>
      <ul>
       <li>
        although/though（虽然）,
       </li>
       <li>
        even though/even if（即使）等。
        <ul>
         <li>
          Although it was raining hard, he still went to school.（although 引导让步状语从句）
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ol>
    <h3>
    </h3>
    <h3>
     4 主语从句
    </h3>
    <ol>
     <li>
      <h4>
       <strong>
        定义：
       </strong>
       在复合句中充当主语的从句
      </h4>
     </li>
     <li>
      <h4>
       引导词
      </h4>
      <ol>
       <li>
        that
       </li>
       <li>
        whether
       </li>
       <li>
        连接代词（what、who、which、whose 等）
       </li>
       <li>
        连接副词（when、where、how、why 等）
       </li>
      </ol>
     </li>
     <li>
      <h4>
       <strong>
        示例
       </strong>
      </h4>
      <ol>
       <li>
        What he said is very important.（“What he said” 是主语从句，作句子的主语）
       </li>
      </ol>
     </li>
    </ol>
    <h3>
     5 表语从句
    </h3>
    <p>
    </p>
    <ol>
     <li>
      <h4>
       <strong>
        定义：
       </strong>
       在复合句中作表语，位于系动词之后
      </h4>
     </li>
     <li>
      <h4>
       <strong>
        引导词
       </strong>
      </h4>
      <ol>
       <li>
        that
       </li>
       <li>
        whether
       </li>
       <li>
        as if、as though
       </li>
       <li>
        连接代词（what、who、which、whose 等）
       </li>
       <li>
        连接副词（when、where、how、why 等）。
       </li>
      </ol>
     </li>
     <li>
      <h4>
       <strong>
        示例
       </strong>
      </h4>
      <ol>
       <li>
        The problem is that we don't have enough money.（“that we don't have enough money” 是表语从句，作 “is” 的表语）
       </li>
      </ol>
     </li>
    </ol>
