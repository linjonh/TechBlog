    <p>
     1． 使用对象资源管理器建立名为“book”的数据库。数据文件和日志文件采用默认方式。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a983564fb7dd3e78d1f991eabf040380.png">
      <br/>
      2．使用T-SQL命令建立名为“bookmanager”的数据库，并自定义数据文件和日志文件。
      <br/>
      create database bookmanager
      <br/>
      on primary
      <br/>
      (name=bookmanager_data,
      <br/>
      filename=‘d:\bookmanager_data.mdf’,
      <br/>
      size=5,
      <br/>
      maxsize=10,
      <br/>
      filegrowth=10%
      <br/>
      )
      <br/>
      log on
      <br/>
      (name=bookmanager_log,
      <br/>
      filename=‘d:\bookmanager_log.ldf’,
      <br/>
      size=10,
      <br/>
      maxsize=100,
      <br/>
      filegrowth=10%
      <br/>
      )
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d102c116900cb32057095ddc988da50.png">
       <br/>
       1．使用对象资源管理器编辑、修改“book”数据库。
       <br/>
       （1）添加一个名为book_data.ndf的数据文件，文件的初始大小为2MB，最大尺寸为10MB，增长速度为20%。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/249aa31b5f38b1de3e926f4174d57496.png">
        <br/>
        （2）添加一个名为book_log1.ldf的日志文件，文件的初始大小为1MB，最大尺寸无穷大，增长速度为1MB。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3675aa973bd1c81624e4e08fa1e9993b.png">
         <br/>
         （3）把主数据文件的最大尺寸修改为50MB，增长速度设置为1MB。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9df5b39bdebdf895919ba2ed2eb0305d.png">
          <br/>
          2．使用T-SQL命令编辑、修改“bookmanager”数据库。
          <br/>
          （1）添加一个名为bookmanager1.ndf数据文件，文件的初始大小为3MB，最大尺寸为无穷大，增长速度为10%。
          <br/>
          alter database bookmanager
          <br/>
          add file
          <br/>
          (name=bookmanager1,
          <br/>
          filename=‘d:\bookmanager1.ndf’,
          <br/>
          size=3,
          <br/>
          maxsize=unlimited,
          <br/>
          filegrowth=10%
          <br/>
          )
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/907626d127a3c5682a071e32daf54bc3.png">
           <br/>
           （2）添加一个名为bookmanager_log1.ldf日志文件，文件的初始大小为3MB，最大尺寸为10MB，增长速度为1MB。
           <br/>
           alter database bookmanager
           <br/>
           add log file
           <br/>
           (name=bookmanager_log1,
           <br/>
           filename=‘d:\bookmanager_log.ldf’,
           <br/>
           size=3,
           <br/>
           maxsize=10,
           <br/>
           filegrowth=1
           <br/>
           )
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dd51312cd39d9bbe27a0f23808d069ab.png">
            <br/>
            （3）把主数据文件的最大尺寸修改为20MB，增长速度设置为1MB。
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     alter database bookmanager
     <br/>
     modify file
     <br/>
     (name=bookmanager,
     <br/>
     maxsize=20,
     <br/>
     filegrowth=1
     <br/>
     )
    </p>
    <p>
     项目3：删除数据库
     <br/>
     1． 使用对象资源管理器删除“book”数据库。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ff44a978c8ebb2c045317d50df83a597.png">
      <br/>
      2．使用T-SQL命令删除“bookmanager”数据库。
     </img>
    </p>
    <p>
     drop database bookmanager
     <br/>
     go
