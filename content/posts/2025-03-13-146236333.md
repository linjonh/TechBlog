    <h3>
     <a id="1fsstat__fsstatSync_5">
     </a>
     1.fs.stat() / fs.statSync()：获取文件或目录的详细信息（大小、类型、修改时间等）。
    </h3>
    <p>
     语法格式：
    </p>
    <pre><code class="prism language-bash">fs.stat<span class="token punctuation">(</span>path<span class="token punctuation">[</span>, options<span class="token punctuation">]</span>, callback<span class="token punctuation">)</span>
</code></pre>
    <p>
     参数解读:
     <br/>
     path: 文件路径
     <br/>
     options: 选项对象，可选
     <br/>
     callback: 回调函数，参数为 (err, stats)
     <br/>
     stats: fs.Stats 对象，包含文件状态信息
     <br/>
     stats.size: 文件大小（字节）
     <br/>
     stats.birthtime: 创建时间
     <br/>
     stats.mtime: 修改时间
     <br/>
     stats.isFile(): 是否为文件
     <br/>
     stats.isDirectory(): 是否为目录

    <h3>
     <a id="2fsaccess_73">
     </a>
     2.fs.access()：检查文件是否存在或是否有访问权限。
    </h3>
    <p>
     语法格式
    </p>
    <pre><code class="prism language-bash">fs.access<span class="token punctuation">(</span>path<span class="token punctuation">[</span>, mode<span class="token punctuation">]</span>, callback<span class="token punctuation">)</span>
</code></pre>
    <p>
     参数解读:
     <br/>
     path:必选参数，文件路径
     <br/>
     mode:可选参数，文件的操作权限callback:回调函数，传递一个参数，参数为可能的异常信息，如果没有异常信息，则为null
     <br/>
     fs.access()方法用于检查指定的文件是否存在。如果文件存在，或者权限符合要求，回调函数将会被调用，否则将会抛出异常。
     <br/>
     fs.access()方法是fs.stat()方法的简化版，用于测试指定路径的用户权限。
