    <h2>
     <a id="DockerElasticsearch_1">
     </a>
     基于Docker安装Elasticsearch
    </h2>
    <ol>
     <li>
      拉取镜像
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> pull elasticsearch:8.13.0
</code></pre>
    <ol start="2">
     <li>
      创建配置
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">vim</span> /root/package/es/config/elasticsearch.yml
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7f57b13ca8084500adc171e9e4ae9c9d.png"/>
    </p>
    <pre><code class="prism language-bash">cluster.name: <span class="token string">"nft-cluster"</span>
network.host: <span class="token number">0.0</span>.0.0
http.cors.enabled: <span class="token boolean">true</span>
http.cors.allow-origin: <span class="token string">"*"</span>
xpack.security.enabled: <span class="token boolean">false</span>
</code></pre>
    <ol start="3">
     <li>
      启动镜像
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> run <span class="token parameter variable">--name</span> elasticsearch <span class="token parameter variable">-p</span> <span class="token number">9200</span>:9200  <span class="token parameter variable">-p</span> <span class="token number">9300</span>:9300  <span class="token parameter variable">-e</span> 
<span class="token string">"discovery.type=single-node"</span> <span class="token parameter variable">-e</span> 
<span class="token assign-left variable">ES_JAVA_OPTS</span><span class="token operator">=</span><span class="token string">"-Xms256m -Xmx512m"</span> <span class="token parameter variable">-v</span> 
/root/package/es/config/elasticsearch.yml:/usr/share/elasticsearch8/config/elasticsearch.yml <span class="token parameter variable">-v</span> 
/home/package/es/data:/usr/share/elasticsearch8/data <span class="token parameter variable">-v</span> 
/home/package/es/plugins:/usr/share/elasticsearch8/plugins <span class="token parameter variable">-d</span> elasticsearch:8.13.0
</code></pre>
    <p>
     这里面的
     <code>
      /root/package/es/config/elasticsearch.yml
     </code>
     也可以改成你自己的目录文件
    </p>
    <ol start="4">
     <li>
      检查是否启动成功
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> logs elasticsearch
</code></pre>
    <ol start="5">
     <li>
      通过9200端口访问：
     </li>
    </ol>
    <pre><code class="prism language-bash">http://192.168.19.142:9200/
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3db5efe3aa604b4089348ab739fe12ec.png"/>
    </p>
    <h2>
     <a id="DockerKibana_52">
     </a>
     基于Docker安装Kibana
    </h2>
    <ol>
     <li>
      使用docker下载kibana
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> pull kibana:8.13.0
</code></pre>
    <ol start="2">
     <li>
      查看es的ip
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> inspect a46fc9482c69 <span class="token operator">|</span><span class="token function">grep</span> IPAddress
</code></pre>
    <p>
     这里的
     <code>
      a46fc9482c69
     </code>
     换成你自己的容器 ID。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ec0748e18a4142d2bf3387a89806e611.png"/>
    </p>
    <ol start="3">
     <li>
      创建并配置kibana.yml
      <br/>
      需要把刚刚查到的es的ip设置到elasticsearch.hosts中，其他的配置自己适当调整即可。
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">vim</span> /root/package/es/config/kibana.yml
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/199acee0b7ea40d09e2d0abd7f6631cc.png"/>
    </p>
    <pre><code class="prism language-bash">server.name: kibana
server.port: <span class="token number">5601</span>
server.host: <span class="token number">0.0</span>.0.0
elasticsearch.hosts: <span class="token punctuation">[</span> <span class="token string">"http://172.17.0.2:9200"</span> <span class="token punctuation">]</span> 
elasticsearch.username: <span class="token string">"elastic-user"</span>
elasticsearch.password: <span class="token string">"123456"</span>
xpack.monitoring.ui.container.elasticsearch.enabled: <span class="token boolean">true</span>
i18n.locale: <span class="token string">"zh-CN"</span>
</code></pre>
    <ol start="4">
     <li>
      启动Kibana
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">docker</span> run <span class="token parameter variable">--name</span> kibana <span class="token parameter variable">-d</span> <span class="token parameter variable">-p</span> <span class="token number">5601</span>:5601 <span class="token parameter variable">-v</span> 
/root/package/es/config/kibana.yml:/usr/share/kibana/config/kibana.yml kibana:8.13.0
</code></pre>
    <ol start="5">
     <li>
      通过页面访问：
      <code>
       http://ip:5601/app/home#/
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/58d51b86db5c4abf8f66b5238a40524e.png">
       <br/>
       输入用户名（elastic-user）、密码（123456））即可访问
      </img>
     </li>
    </ol>
