    <h2>
     一、实现
    </h2>
    <p>
     在Vue-flow官网中，关于
     <a class="link-info" href="https://vueflow.dev/examples/layout/animated.html" rel="nofollow" title="动态流程图">
      动态流程图
     </a>
     的部分长这样
    </p>
    <p>
     <img alt="" class="left" height="243" src="https://i-blog.csdnimg.cn/direct/6e5ec94eaa754d68add7be9204243330.png" width="281">
      <img alt="" class="right" height="240" src="https://i-blog.csdnimg.cn/direct/bbc1f69bacf749328f406757576cbdf6.png" width="284"/>
     </img>
    </p>
    <p>
     他可以让你的流程变得可动，更加容易理解
    </p>
    <p>
     Examples中提供了各个文件的代码以及importMap，但是当我复制文件过来之后发现无法渲染,控制台报警告
     <img alt="" height="40" src="https://i-blog.csdnimg.cn/direct/f394bf02e63e461ea3d5bd523a5b2a24.png" width="942"/>
    </p>
    <p>
     我们只需要在index.vue中引入
    </p>
    <pre><code class="hljs">import '@vue-flow/core/dist/style.css'</code></pre>
    <h2>
     二、修改为自定义样式
    </h2>
    <p>
     在官方例子中，根节点长这样
     <img alt="" height="38" src="https://i-blog.csdnimg.cn/direct/41d35493785c4c7aa085c01af79638f9.png" width="37">
      ，如果我们想要将根节点中的图标修改为字符串，我们只需要在ProcessNode.vue中将
      <span style="color:#ed7976">
       processLabel
      </span>
      计算属性修改，如下
     </img>
    </p>
    <pre><code class="language-javascript">const processLabel = computed(() =&gt; {
  if (isStartNode.value) {
    return 'node'
  }

  switch (status.value) {
    case ProcessStatus.ERROR:
      return 'faild'
    case ProcessStatus.SKIPPED:
      return '🚧'
    case ProcessStatus.CANCELLED:
      return '🚫'
    case ProcessStatus.FINISHED:
      return '😎'
    default:
      return '🏠'
  }
})</code></pre>
    <p>
     下面switch中的几个case分别对应各个节点根据状态不同显示的图案（字符串）
    </p>
    <p>
     而bgColor则代表各个节点更具不同状态显示的背景颜色
    </p>
    <pre><code class="language-javascript">const bgColor = computed(() =&gt; {
  if (isStartNode.value) {
    return '#2563eb'
  }

  switch (status.value) {
    case ProcessStatus.ERROR:
      return '#f87171'
    case ProcessStatus.FINISHED:
      return '#42B983'
    case ProcessStatus.CANCELLED:
      return '#fbbf24'
    default:
      return '#4b5563'
  }
})</code></pre>
    <p>
     如果想要去除右上角控制器，只需要在index.vue中删除&lt;Panel&gt;标签中的内容
    </p>
    <pre><code class="language-html">      &lt;Panel class="process-panel" position="top-right"&gt;
        &lt;div class="layout-panel"&gt;
          &lt;button v-if="isRunning" class="stop-btn" title="stop" @click="stop"&gt;
            &lt;Icon name="stop" /&gt;
            &lt;span class="spinner" /&gt;
          &lt;/button&gt;
          &lt;button v-else title="start" @click="run(nodes)"&gt;
            &lt;Icon name="play" /&gt;
          &lt;/button&gt;

          &lt;button title="set horizontal layout" @click="layoutGraph('LR')"&gt;
            &lt;Icon name="horizontal" /&gt;
          &lt;/button&gt;

          &lt;button title="set vertical layout" @click="layoutGraph('TB')"&gt;
            &lt;Icon name="vertical" /&gt;
          &lt;/button&gt;
        &lt;/div&gt;

        &lt;div class="checkbox-panel"&gt;
          &lt;label&gt;Cancel on error&lt;/label&gt;
          &lt;input v-model="cancelOnError" type="checkbox" /&gt;
        &lt;/div&gt;
      &lt;/Panel&gt;</code></pre>
    <p>
     整个流程的开启函数则是在useRunProcess.js中修改，在index.vue中通过
     <span style="color:#ed7976">
      run(nodes)
     </span>
     调用
    </p>
