# VS Code-常用插件-下载迁移-前端开发

#### 背景

前端使用的开发工具一般是Visual Studio Code，很多辅助功能，比如字体高亮、单词拼写检查、预览图片等需要安装插件。但是插件在原来的电脑，不想下载或者自己是新人，想迁移同事的插件，或者新电脑没有外网。

以下是工具安装，插件下载，迁移插件流程

#### 1.下载Visual Studio Code,到visualstudio官网，按步骤安装

[Visual Studio Code - Code Editing. Redefined](https://code.visualstudio.com/ "Visual Studio Code - Code Editing. Redefined")

![](https://i-blog.csdnimg.cn/blog_migrate/77e8c4b49fb7035d45e2e9ca4b6ff4c2.png)

#### 2.打开工具，进入EXTENSIONS界面，搜索插件

![](https://i-blog.csdnimg.cn/blog_migrate/e672f3881d69e9d80992ea2e997ede14.png)

#### 3.常用插件-web开发，安装以下标题，搜索，安装

![](https://i-blog.csdnimg.cn/blog_migrate/34671b56c2b9f96b7cd4b7e572c04bd5.png)
![](https://i-blog.csdnimg.cn/blog_migrate/d58d87815b9da54f08dc2615927f1d33.png)

插件详细解释

```bash
1.3 Color Highlight
用于给我们代码中的颜色进行高亮展示的插件，直接将代码中颜色值高亮展示。

1.5 Error Gutters
报错提示，报错的地方都有大红波浪线提示，可以很快的定位错误。

1.7 indent-rainbow
彩虹缩进，就是把代码不同的缩进展示不同的颜色。

1.8 Trailing Spaces
强迫症福音，这个插件会把尾随空格显示出来。

2.1 Code Spell Checker
检查代码中单词拼写是否正确，当单词不正常的时候，就会在下方出现波浪线进行提示，还可以自定义词典，忽略某个单词的检查等。

2.2 Git History
右键单击文件选择 Git：View File History 来以列表的形式查看所有的提交记录。可以更好的溯源，知道谁修改过代码。

3.6 vetur / volar
使用 vue 进行开发的这两个插件必不可免，volar 是跟 vue3 更配的，功能也能多，由于这两个插件功能过于庞大，感兴趣的自行搜索使用。


```

#### 3.找到下载的插件存放地址

路径一般在C盘下,例如我的C:\Users\xxx_0249\.vscode\extensions

![](https://i-blog.csdnimg.cn/blog_migrate/53a1a7a859fb91587fc61bcbe1b75bfe.png)

#### 4.把extensions里对应的插件，复制到新电脑相同的路径下

#### 5.重新启动新电脑的Visual studio code开发工具，可以正常使用了，也可以看到插件已经安装了。
