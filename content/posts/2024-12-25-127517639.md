# 解决特殊字符前端传入后端

先用encodeURI将含有特殊符号的信息转码成test，然后传入后端

```bash
var test=encodeURI(value)

```

然后后端使用一个正则表达式来进行判断,是特殊符号能通过,比如:

```bash
test= test.replaceAll("%(?![0-9a-fA-F]{2})", "%25");

```

后端获取test信息然后使用URLDecoder.docode()解码

```bash
test= URLDecoder.decode(test);

```
