    <h2>
     安装Rust
    </h2>
    <p>
     Rust直接从官网下载安装文件：
     <a href="https://www.rust-lang.org/tools/install" rel="nofollow" title="Install Rust - Rust Programming Language">
      Install Rust - Rust Programming Language
     </a>
    </p>
    <p>
     下载，运行安装即可。
    </p>
    <p>
     安装好提示：
    </p>
    <p>
     Rust is installed now. Great!
    </p>
    <p>
     To get started you may need to restart your current shell.
     <br/>
     This would reload its PATH environment variable to include
     <br/>
     Cargo's bin directory (%USERPROFILE%\.cargo\bin).
    </p>
    <p>
     Press the Enter key to continue.
    </p>
    <p>
    </p>
    <h3>
     设置cargo加速
    </h3>
    <p>
     进入目录，如~/.cargo
    </p>
    <p>
     删除cache文件
    </p>
    <pre><code>rm .package-cache</code></pre>
    <p>
     创建config.toml文件
    </p>
    <pre><code>[source.crates-io]
replace-with = 'tuna' # 可以根据需要替换为其他源

# 中国科学技术大学
[source.ustc]
registry = "sparse+https://mirrors.ustc.edu.cn/crates.io-index/"

# 阿里云
[source.aliyun]
registry = "sparse+https://mirrors.aliyun.com/crates.io-index/"

# 上海交通大学
[source.sjtu]
registry = "https://mirrors.sjtug.sjtu.edu.cn/git/crates.io-index/"

# 清华大学
[source.tuna]
registry = "https://mirrors.tuna.tsinghua.edu.cn/git/crates.io-index.git"

# rustcc社区
[source.rustcc]
registry = "https://code.aliyun.com/rustcc/crates.io-index.git"</code></pre>
    <p>
     再使用cargo，就会发现下载文件快很多！
    </p>
    <h2 style="background-color:transparent">
     安装Zed
    </h2>
    <p>
     Zed的官网：
     <a href="https://github.com/zed-industries/zed" title="GitHub - zed-industries/zed: Code at the speed of thought – Zed is a high-performance, multiplayer code editor from the creators of Atom and Tree-sitter.">
      GitHub - zed-industries/zed: Code at the speed of thought – Zed is a high-performance, multiplayer code editor from the creators of Atom and Tree-sitter.
     </a>
    </p>
    <p>
     Zed有LInux和Mac OS的安装包，Windows下反而要自己编译安装。
    </p>
    <p>
    </p>
    <h3>
     安装条件
    </h3>
    <p>
     需要rustup、VisualStudo、CMake 以及Windows 10 SDK version 2104 (10.0.20348.0)等
    </p>
    <h3 style="background-color:transparent">
     开始安装
    </h3>
    <h4>
     下载源代码
    </h4>
    <pre><code>git clone https://github.com/zed-industries/zed</code></pre>
    <p>
     进入zed目录，然后
    </p>
    <h4>
     cargo 安装
    </h4>
    <p>
     执行：
    </p>
    <p>
     cargo run
    </p>
    <p>
     编译碰到报错：Unable to init GPU context: NoSupportedDeviceFound
    </p>
    <p>
     无法解决问题。
    </p>
    <p>
     zed安装失败。
    </p>
    <h2>
     总结
    </h2>
    <p>
     大约是显卡太老了，导致zed编译的时候报错Unable to init GPU context: NoSupportedDeviceFound
    </p>

      NVIDIA RTX / Quadro Desktop and Notebook Driver Release 470

     <strong>
      以前的是： NVIDIA-SMI 474.82       Driver Version: 474.82       CUDA Version: 11.4

    <p>
     更新了也不行，而且现在nvidia-smi也不能执行了，它不在path里了
    </p>
    <p>
    </p>
    <h3>
     查看zed手册，提到需要Vulkan
    </h3>
    <p>
     下载Vulan驱动：
     <a href="https://developer.nvidia.com/vulkan-driver" rel="nofollow" title="Vulkan Driver Support | NVIDIA Developer">
      Vulkan Driver Support | NVIDIA Developer
     </a>
    </p>
    <p>
     If you see a notification saying
     <code>
      Zed failed to open a window: NoSupportedDeviceFound
     </code>
     this means that Vulkan cannot find a compatible GPU. You can begin troubleshooting Vulkan by installing the
     <code>
      vulkan-tools
     </code>
     package and running:
    </p>
    <p>
     <code>
      vkcube
     </code>
    </p>
    <p>
     但是最新的Vulkan1.4是支持到572版本，474版本的没找到。而且这块卡quadro k4200 不在支持列表。
    </p>
