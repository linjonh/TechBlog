     windows下安装Open Web UI
    </h1>

  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <p>
     windows下安装openwebui有三种方式,docker,python+node.js,整合包.
    </p>
    <p>
     这里我选择的是第二种,非docker.
    </p>
    <h4>
     非Docker方式安装
    </h4>
    <h4>
     <strong>
      1. 安装Python
     </strong>
     ：
    </h4>
    <p>
     下载并安装Python 3.11，建议安装路径中不要包含中文字符，并勾选“Add python 3.11 to Path”选项。
     <br/>
     安装完成后，在命令提示符中输入python，若显示Python版本信息则表示安装成功。
    </p>
    <h4>
     <strong>
      2. 安装Node.js和npm
     </strong>
     ：
    </h4>
    <p>
     访问Node.js中文网，下载并安装较新版本的Node.js（如20.12.2），安装过程中会自动安装npm。在命令提示符中输入npm -v，若显示npm版本信息则表示安装成功。
    </p>
    <h4>
     <strong>
      3. 克隆OpenWebUI项目
     </strong>
     ：
    </h4>
    <p>
     打开命令行工具（如CMD、PowerShell或Git Bash），执行以下命令克隆OpenWebUI项目：
    </p>
    <pre><code>git clone https://github.com/open-webui/open-webui.git</code></pre>
    <p>
     导航到OpenWebUI项目目录，
    </p>
    <h4>
     <strong>
      4. 安装npm依赖
     </strong>
     ：
    </h4>
    <pre><code>npm install</code></pre>
    <h4>
     <strong>
      5. 安装Python依赖
     </strong>
     ：
    </h4>
    <pre><code>pip install -r ./backend/requirements.txt</code></pre>
    <p>
     (我下载到的项目requirements.txt依赖管理文件在backend目录下，具体看文件路径)
    </p>
    <h4>
     <strong>
      6. 安装uv
     </strong>
    </h4>
    <p>
     macOS/Linux：
    </p>
    <pre><code>curl -LsSf https://astral.sh/uv/install.sh | sh</code></pre>
    <p>
     windows：
    </p>
    <pre><code>powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"</code></pre>
    <h4>
     <strong>
      7.
     </strong>
     安装与
     <strong>
      <code>
       pip
      </code>
     </strong>
    </h4>
    <p>
     安装Open WebUI：
    </p>
    <pre><code>pip install open-webui</code></pre>
    <p>
     更新Open WebUI
    </p>
    <p>
     要更新到最新版本，只需运行：
    </p>
    <pre><code>pip install --upgrade open-webui</code></pre>
    <h4>
     <strong>
      8. 运行Open WebUI
     </strong>
    </h4>
    <p>
     <strong>
      通过指定的 Python 版本（3.11）来启动
      <code>
       open-webui
      </code>
      项目，并且将项目的数据存储路径设置为
      <code>
       C:\open-webui\data
      </code>
      (示例路径)，最后启动该项目的 Web 服务器，以便用户可以通过浏览器访问。
     </strong>
    </p>
    <p>
     macOS/Linux：
    </p>
    <pre><code> DATA_DIR=~/.open-webui uvx --python 3.11 open-webui@latest serve</code></pre>
    <p>
     windows powershell：
    </p>
    <pre><code>$env:DATA_DIR="C:\open-webui\data"; uvx --python 3.11 open-webui@latest serve</code></pre>
    <p>
     <img alt="" height="623" src="https://i-blog.csdnimg.cn/direct/5e098e0473a047819ab67a8f2d3d0d99.png" width="1200"/>
    </p>
    <p>
     <img alt="" height="342" src="https://i-blog.csdnimg.cn/direct/b529594520ee4b5da93e3858bf3f6941.png" width="1200"/>
    </p>
    <h4>
     <img alt="" height="750" src="https://i-blog.csdnimg.cn/direct/f4c27b97915746289d0b14b6952e7652.png" width="1200"/>
    </h4>
    <p>
     此方法安装所有必要的依赖项并启动Open WebUI，允许简单高效的设置。安装后，可以在http://localhost:8080访问Open WebUI。
    </p>
    <p>
     以后启动可以使用:
    </p>
    <pre><code>open-webui serve</code></pre>
    <p>
     参考
     <a href="https://docs.openwebui.com/" rel="nofollow" title="🏡 Home | Open WebUI">
      🏡 Home | Open WebUI
     </a>
    </p>
    <p>
     或者是在OpenWebUI项目的
     <strong>
      backend
     </strong>
     目录下，找到
     <strong>
      start_windows.bat
     </strong>
     文件并双击运行。
    </p>
    <p>
