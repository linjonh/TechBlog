    <h3>
     <a id="_0">
     </a>
     问题与处理策略
    </h3>
    <h5>
     <a id="_2">
     </a>
     问题描述
    </h5>
    <ul>
     <li>
      在前端项目中，执行
      <code>
       pnpm install
      </code>
      指令安装依赖时，报如下错误
     </li>
    </ul>
    <pre><code>node_modules/.pnpm/node-sass@4.14.1/node_modules/node-sass: Running postinstall script, failed in 748ms
</code></pre>
    <h5>
     <a id="_10">
     </a>
     问题原因
    </h5>
    <ul>
     <li>
      <code>
       Node.js
      </code>
      和
      <code>
       node-sass
      </code>
      版本不兼容，
      <code>
       node-sass
      </code>
      对
      <code>
       Node.js
      </code>
      的版本有严格的要求
     </li>
    </ul>
    <h5>
     <a id="_14">
     </a>
     处理策略
    </h5>
    <ul>
     <li>
      检查
      <code>
       Node.js
      </code>
      版本是否符合
      <code>
       node-sass
      </code>
      的要求，访问
      <code>
       node-sass
      </code>
      的 npm 页面查看支持的
      <code>
       Node.js
      </code>
      版本：
      <code>
       https://www.npmjs.com/package/node-sass
      </code>
     </li>
    </ul>
