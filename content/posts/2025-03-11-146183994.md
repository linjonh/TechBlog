    <h3>
     <a id="flask_20">
     </a>
     flask的响应机制
    </h3>
    <p>
     flask的响应机制主要是将视图函数的返回值转换为HTTP响应。当视图函数返回一个字符串时，flask会将其封装成一个Response对象，并设置默认的
     <code>
      Content-Type
     </code>
     为
     <code>
      text/html; charset=utf-8
     </code>
     ；当返回字典或列表时，flask会使用jsonify函数将其转换为JSON响应。
    </p>
    <h3>
     <a id="jsonfy_23">
     </a>
     jsonfy
    </h3>
    <p>
     jsonify 是flask提供的一个便捷函数，用于将 python 对象（如字典、列表）转换为JSON响应。它会自动创建一个Response对象，默认设置
     <code>
      Content-Type
     </code>
     为
     <code>
      application/json
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/99c58af620724b94bcdfded2c6911362.png"/>
    </p>
    <h3>
     <a id="jsondumps_27">
     </a>
     json.dumps
    </h3>
    <p>
     在flask里，
     <code>
      jsonify
     </code>
     函数其实是基于python的
     <code>
      json.dumps
     </code>
     来实现的。
     <code>
      json.dumps
     </code>
     默认会把非 ASCII 字符转义成 unicode 编码形式。
    </p>
    <p>
     可以通过设置
     <code>
      ensure_ascii
     </code>
     参数为
     <code>
      False
     </code>
     来避免这种转义，并将
     <code>
      json.dumps
     </code>
     的结果包装为
     <code>
      Response
     </code>
     对象，设置响应头。
    </p>
