# 【HarmonyOS NEXT】是否支持calc计算宽高

##### 【关键字】

API 10 / calc计算 / 尺寸设置 / 宽高 / width / height / size

##### 【问题描述】

HarmonyOS是否支持calc计算宽高?

##### 【解决方案】

从API version 10开始，width、height、size接口支持calc计算特性。

详细信息请参见尺寸设置：
[https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size-0000001774280842](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size-0000001774280842 "https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size-0000001774280842")
