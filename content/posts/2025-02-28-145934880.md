     </a>
     前言
    </h3>
    <blockquote>
     <p>

    <h3>
     <a id="_8">
     </a>
     技术选择
    </h3>
    <p>
     该项目基于Vue Flask Sqlalchemy Yolov11 实现了水稻病害检测系统，含有数据大屏展示
    </p>
    <h3>
     <a id="_11">
     </a>
     文件结构
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/2fbe69ab346a432ea120ae062dc77ae9.png"/>
    </p>
    <ul>
     <li>
      <p>
       configs文件夹
       <br/>
       该文件夹放入的为一些配置文件，例如：数据库配置、路由配置等
      </p>
     </li>
     <li>
      <p>
       modules文件夹
       <br/>
       该文件夹放入的为一些业务模块，例如：用户模块、检测模块等
      </p>
     </li>
     <li>
      <p>
       resource文件夹
       <br/>
       该文件夹放入一些静态资源，例如：配置文件、图片等
      </p>
     </li>
     <li>
      <p>
       templates文件夹
       <br/>
       该文件夹放入一些HTML文件，例如：邮箱模板、前后端不分离的时候放入前端文件
      </p>
     </li>
     <li>
      <p>
       utils文件夹
       <br/>
       该文件夹放入一些封装好的工具函数
      </p>
     </li>
     <li>
      <p>
       vo文件夹
       <br/>
       该文件夹放入封装好的响应数据
      </p>
     </li>
    </ul>
