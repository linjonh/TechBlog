# centos操作系统上传和下载百度网盘内容

探序基因 整理

进入百度网盘官网[百度网盘 客户端下载](https://pan.baidu.com/download#pan "百度网盘 客户端下载")

下载linux的rpm格式的安装包

![](https://i-blog.csdnimg.cn/direct/37fcc3e7279b4ab1b00ad60cf2859829.png)

在linux命令行中输入：rpm -ivh baidunetdisk_4.17.7_x86_64.rpm

出现报错：

错误：依赖检测失败：

libXScrnSaver 被 baidunetdisk-4.17.7-1.x86_64 需要

于是：

yum install libXScrnSaver, yum install libXScrnSaver-devel

再rpm -ivh baidunetdisk_4.17.7_x86_64.rpm就可以了

安装bypy命令行工具

pip3.9 install bypy --user

(--user的意思，是安装在用户目录下，通常是 `~/.local/bin`（对于可执行脚本）)

重置关联账号：rm -rf ~/.bypy

#获取授权码并关联账号

~/.local/bin/bypy info

#该命令会生成一个 URL，你需要在浏览器中打开并获取授权码，然后粘贴回命令行。

#关联成功后，可以运行以下命令查看账号信息：

~/.local/bin/bypy info

下载：~/.local/bin/bypy downfile 网盘文件 文件夹

提交后台下载nohup ~/.local/bin/bypy downfile 网盘文件 文件夹 &
