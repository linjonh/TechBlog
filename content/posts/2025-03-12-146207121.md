    <h3>
     <a id="_0">
     </a>
     引言
    </h3>
    <p>
     在 Flutter 开发中，Dart 语言的流程控制语句起着至关重要的作用。它们允许开发者根据不同的条件执行不同的代码块，或者重复执行特定的代码，从而实现程序的逻辑控制。下面将详细介绍 Dart 中常见的流程控制语句，并结合代码示例进行说明。
    </p>
    <h3>
     <a id="1_if__else__4">
     </a>
     1. if - else 语句
    </h3>
    <p>
     <code>
      if - else
     </code>
     语句用于根据条件的真假来执行不同的代码块。它是最基本的条件控制语句，在很多场景下都会用到。
    </p>
    <h4>
     <a id="_7">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  int score <span class="token operator">=</span> <span class="token number">75</span><span class="token punctuation">;</span>

  <span class="token keyword">if</span> <span class="token punctuation">(</span>score <span class="token operator">&gt;=</span> <span class="token number">90</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'成绩优秀'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token keyword">if</span> <span class="token punctuation">(</span>score <span class="token operator">&gt;=</span> <span class="token number">70</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'成绩良好'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>

  <span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'成绩不及格'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>

     <a id="_24">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      首先定义了一个整数变量
      <code>
       score
      </code>
      并赋值为 75。
     </li>
     <li>
      程序会依次判断
      <code>
       if
      </code>
      和
      <code>
       else if
      </code>
      后面的条件表达式。如果
      <code>
       score &gt;= 90
      </code>
      为
      <code>
       true
      </code>
      ，则执行对应的代码块；若为
      <code>
       false
      </code>
      ，则继续判断下一个
      <code>
       else if
      </code>
      条件。
     </li>
     <li>
      当
      <code>
       score &gt;= 70
      </code>
      为
      <code>
       true
      </code>
      时，执行
      <code>
       print('成绩良好');
      </code>
      语句，之后不再继续判断其他条件。
     </li>
     <li>
      如果所有
      <code>
       if
      </code>
      和
      <code>
       else if
      </code>
      条件都不满足，则执行
      <code>
       else
      </code>
      后面的代码块。
     </li>
    </ul>
    <h3>
     <a id="2_switch__case__30">
     </a>
     2. switch - case 语句
    </h3>
    <p>
     <code>
      switch - case
     </code>
     语句用于根据一个表达式的值来选择执行不同的代码块，通常用于多个固定值的匹配场景。
    </p>
    <h4>
     <a id="_33">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

      <span class="token keyword">break</span><span class="token punctuation">;</span>

      <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token keyword">case</span> <span class="token string-literal"><span class="token string">'Wednesday'</span></span><span class="token punctuation">:</span>
      <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'星期三，工作过半啦'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
      <span class="token keyword">break</span><span class="token punctuation">;</span>

      <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'星期四，快到周末啦'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>

      <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'星期六，好好休息'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
      <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token keyword">case</span> <span class="token string-literal"><span class="token string">'Sunday'</span></span><span class="token punctuation">:</span>
      <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'星期日，放松一下'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
      <span class="token keyword">break</span><span class="token punctuation">;</span>
    <span class="token keyword">default</span><span class="token punctuation">:</span>
      <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'输入的不是有效的星期'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_66">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      定义了一个字符串变量
      <code>
       day
      </code>
      并赋值为
      <code>
       'Monday'
      </code>
      。
     </li>
     <li>
      <code>
       switch
      </code>
      语句会将
      <code>
       day
      </code>
      的值与各个
      <code>
       case
      </code>
      后面的值进行比较。
     </li>
     <li>
      当找到匹配的
      <code>
       case
      </code>
      时，执行该
      <code>
       case
      </code>
      下的代码块。注意，每个
      <code>
       case
      </code>
      块末尾需要使用
      <code>
       break
      </code>
      语句来跳出
      <code>
       switch
      </code>
      语句，否则会继续执行下一个
      <code>
       case
      </code>
      块的代码。
     </li>
     <li>
      如果没有匹配到任何
      <code>
       case
      </code>
      ，则执行
      <code>
       default
      </code>
      后面的代码块。
     </li>
    </ul>
    <h3>
     <a id="3_for__72">
     </a>
     3. for 循环
    </h3>
    <p>
     <code>
      for
     </code>
     循环用于重复执行一段代码，通常在已知循环次数的情况下使用。
    </p>
    <h4>
     <a id="_75">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token keyword">for</span> <span class="token punctuation">(</span>int i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">5</span><span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'当前数字是: </span><span class="token interpolation"><span class="token punctuation">$</span><span class="token expression">i</span></span><span class="token string">'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_84">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      <code>
       for
      </code>
      循环由三个部分组成：初始化语句
      <code>
       int i = 0
      </code>
      、循环条件
      <code>
       i &lt; 5
      </code>
      和迭代语句
      <code>
       i++
      </code>
      。
     </li>
     <li>
      首先执行初始化语句，将变量
      <code>
       i
      </code>
      初始化为 0。
     </li>
     <li>
      然后检查循环条件
      <code>
       i &lt; 5
      </code>
      ，如果为
      <code>
       true
      </code>
      ，则执行循环体中的代码
      <code>
       print('当前数字是: $i');
      </code>
      。
     </li>
     <li>
      执行完循环体后，执行迭代语句
      <code>
       i++
      </code>
      ，将
      <code>
       i
      </code>
      的值加 1。
     </li>
     <li>
      再次检查循环条件，重复上述过程，直到循环条件为
      <code>
       false
      </code>
      时，循环结束。
     </li>
    </ul>
    <h4>
     <a id="_for__91">
     </a>
     遍历列表的 for 循环示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'我喜欢吃 </span><span class="token interpolation"><span class="token punctuation">$</span><span class="token expression">fruit</span></span><span class="token string">'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

    <p>
     这里使用
     <code>
      for - in
     </code>
     语法来遍历列表
     <code>
      fruits
     </code>
     中的每个元素，将元素依次赋值给变量
     <code>
      fruit
     </code>
     并执行循环体。
    </p>
    <h3>
     <a id="4_while__102">
     </a>
     4. while 循环
    </h3>
    <p>
     <code>
      while
     </code>
     循环会在指定条件为
     <code>
      true
     </code>
     时不断执行代码块，直到条件变为
     <code>
      false
     </code>
     。
    </p>
    <h4>
     <a id="_105">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  int count <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
  <span class="token keyword">while</span> <span class="token punctuation">(</span>count <span class="token operator">&lt;</span> <span class="token number">3</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'当前计数: </span><span class="token interpolation"><span class="token punctuation">$</span><span class="token expression">count</span></span><span class="token string">'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    count<span class="token operator">++</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_116">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      首先定义一个整数变量
      <code>
       count
      </code>
      并初始化为 0。
     </li>
     <li>
      检查
      <code>
       while
      </code>
      后面的条件
      <code>
       count &lt; 3
      </code>
      ，如果为
      <code>
       true
      </code>
      ，则执行循环体中的代码
      <code>
       print('当前计数: $count');
      </code>
      和
      <code>
       count++;
      </code>
      。
     </li>
     <li>
      每次循环结束后，再次检查条件，直到条件为
      <code>
       false
      </code>
      时，循环结束。
     </li>
    </ul>
    <h3>
     <a id="5_do__while__121">
     </a>
     5. do - while 循环
    </h3>
    <p>
     <code>
      do - while
     </code>
     循环与
     <code>
      while
     </code>
     循环类似，但它会先执行一次循环体，然后再检查条件。
    </p>
    <h4>
     <a id="_124">
     </a>
     代码示例
    </h4>
    <pre><code class="prism language-dart"><span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  int num <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
  <span class="token keyword">do</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string-literal"><span class="token string">'当前数字: </span><span class="token interpolation"><span class="token punctuation">$</span><span class="token expression">num</span></span><span class="token string">'</span></span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    num<span class="token operator">++</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span> <span class="token keyword">while</span> <span class="token punctuation">(</span>num <span class="token operator">&lt;</span> <span class="token number">2</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_135">
     </a>
     代码解释
    </h4>
    <ul>
     <li>
      先执行
      <code>
       do
      </code>
      后面的循环体，即打印
      <code>
       当前数字: 0
      </code>
      并将
      <code>
       num
      </code>
      的值加 1。
     </li>
     <li>
      然后检查
      <code>
       while
      </code>
      后面的条件
      <code>
       num &lt; 2
      </code>
      ，如果为
      <code>
       true
      </code>
      ，则继续执行循环体；否则循环结束。
     </li>
    </ul>
    <h3>
     <a id="6_break__continue__139">
     </a>
     6. break 和 continue 语句
    </h3>
    <ul>
     <li>
      <code>
       break
      </code>
      语句用于跳出当前的循环或
      <code>
       switch
      </code>
      语句。
     </li>
     <li>
      <code>
       continue
      </code>
      语句用于跳过当前循环的剩余部分，直接开始下一次循环。
     </li>
    </ul>
    <h4>
     <a id="_143">
     </a>
     代码示例
    </h4>
