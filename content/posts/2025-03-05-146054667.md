    <p>
     流程控制是编程中的一个核心概念，它决定了程序执行的顺序和路径。通过流程控制，我们可以让程序根据条件选择执行不同的代码块，或者重复执行某些代码块。流程控制结构有顺序结构、分支结构和循环结构三种。其中顺序结构表示代码从上往下按照顺序执行，很简单，不用多讲。本文主要讲分支结构，下一篇文章讲循环结构。
    </p>
    <h3>
     <a id="_12">
     </a>
     一、分支结构简介
    </h3>
    <p>
     分支结构（也称为条件结构、选择结构）允许程序根据条件表达式的值执行不同的代码块。在Python中，分支结构主要通过
     <code>
      if
     </code>
     语句实现。条件表达式通常涉及比较运算符（如
     <code>
      ==
     </code>
     、
     <code>
      &gt;
     </code>
     、
     <code>
      &lt;
     </code>
     等）和逻辑运算符（如
     <code>
      and
     </code>
     、
     <code>
      or
     </code>
     、
     <code>
      not
     </code>
     等）。
    </p>
    <h3>
     <a id="Python_18">
     </a>
     二、Python中的分支结构
    </h3>
    <h4>
     <a id="21_if__20">
     </a>
     2.1 if 语句
    </h4>
    <p>
     <code>
      if
     </code>
     语句是最基本的分支结构，用于在条件为真时执行特定的代码块。适用于只有一种分支的情况下。
    </p>
    <h5>
     <a id="1_24">
     </a>
     （1）语法结构
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件表达式<span class="token punctuation">:</span>
    <span class="token comment"># 条件为真时执行的代码块</span>
</code></pre>
    <h5>
     <a id="2_31">
     </a>
     （2）执行规律
    </h5>
    <ol>
     <li>
      判断条件表达式的结果
     </li>
     <li>
      如果条件表达式为True，执行里面的代码块，执行完之后继续执行if结构后面的代码
     </li>
     <li>
      如果条件表达式为False，跳过if结构，执行if结构后面的代码
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：条件表达式可以是布尔值、关系运算符执行结果和逻辑运算符执行结果
    </p>

    <h4>
     <a id="22_ifelse__59">
     </a>
     2.2 if-else 语句
    </h4>
    <p>
     <code>
      if-else
     </code>
     语句用于在条件为真时执行一个代码块，否则执行另一个代码块。适用于两种分支的情况下
    </p>
    <h5>
     <a id="1_63">
     </a>
     （1）语法结构
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件表达式<span class="token punctuation">:</span>
    <span class="token comment"># 条件为真时执行的代码块</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token comment"># 条件为假时执行的代码块</span>
</code></pre>
    <h5>
     <a id="2_72">
     </a>
     （2）执行规律
    </h5>
    <ol>
     <li>
      判断条件表达式的结果
     </li>
     <li>
      如果条件表达式为True，执行条件表达式后面的代码块，执行完之后继续执行if-else结构后面的代码
     </li>
     <li>
      如果条件表达式为False，执行else后面的代码块，执行完之后继续执行if-else结构后面的代码
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：条件表达式可以是布尔值、关系运算符执行结果和逻辑运算符执行结果
    </p>

    <h4>
     <a id="23_ifelifelse__104">
     </a>
     2.3 if-elif-else 语句
    </h4>
    <p>
     <code>
      if-elif-else
     </code>
     语句用于检查多个条件，并根据第一个满足的条件执行相应的代码块。如果没有条件满足，则执行
     <code>
      else
     </code>
     部分的代码块。
    </p>
    <h5>
     <a id="1_108">
     </a>
     （1）语法结构
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件表达式<span class="token number">1</span><span class="token punctuation">:</span>
    <span class="token comment"># 条件表达式1为真时执行的代码块</span>
<span class="token keyword">elif</span> 条件表达式<span class="token number">2</span><span class="token punctuation">:</span>
    <span class="token comment"># 条件表达式2为真时执行的代码块</span>
<span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token comment"># 可以添加多个elif语句</span>
<span class="token keyword">elif</span> 条件表达式n<span class="token punctuation">:</span>
    <span class="token comment"># 条件表达式n为真时执行的代码块</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token comment"># 所有条件都为假时执行的代码块</span>
</code></pre>
    <h5>
     <a id="2_122">
     </a>
     （2）执行规律
    </h5>
    <ol>
     <li>
      判断条件表达式1的结果
     </li>
     <li>
      如果条件表达式1为True，执行条件表达式1后面的代码块，执行完之后继续执行if-elif-else结构后面的代码
     </li>
     <li>
      如果条件表达式1为False，继续判断条件表达式2
     </li>
     <li>
      如果条件表达式2为True，执行条件表达式2后面的代码块，执行完之后继续执行if-elif-else结构后面的代码
     </li>
     <li>
      如果条件表达式2为False，继续判断…
     </li>
     <li>
      如果条件表达式n为True，执行条件表达式n后面的代码块，执行完之后继续执行if-elif-else结构后面的代码
     </li>
     <li>
      如果条件表达式n为False，执行else后面代码块
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：else语句可以省略不写，如果所有条件表达式都为False，又省略了else，则if-elif-else结构不被执行。
    </p>
    <p>
     <strong>
      注意
     </strong>
     ：条件表达式可以是布尔值、关系运算符执行结果和逻辑运算符执行结果
    </p>

    <h4>
     <a id="24_if_158">
     </a>
     2.4 嵌套if语句
    </h4>
    <p>
     嵌套
     <code>
      if
     </code>
     语句是指在一个
     <code>
      if
     </code>
     语句的代码块中再嵌套另一个
     <code>
      if
     </code>
     语句。这允许程序根据多个条件进行更复杂的判断。
    </p>
    <h5>
     <a id="1_162">
     </a>
     （1）语法结构
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件表达式<span class="token number">1</span><span class="token punctuation">:</span>
    <span class="token comment"># 条件表达式1为真时执行的代码块</span>
    <span class="token keyword">if</span> 条件表达式<span class="token number">2</span><span class="token punctuation">:</span>
        <span class="token comment"># 条件表达式1和条件表达式2都为真时执行的代码块</span>
    <span class="token keyword">else</span><span class="token punctuation">:</span>
        <span class="token comment"># 条件表达式1为真，但条件表达式2为假时执行的代码块</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token comment"># 条件表达式1为假时执行的代码块</span>
</code></pre>
    <h5>
     <a id="2_175">
     </a>
     （2）执行规律
    </h5>
    <ol>
     <li>
      <p>
       判断条件表达式1的结果
      </p>
     </li>
     <li>
      <p>
       如果条件表达式1为True，执行条件表达式1后面的代码块，然后判断条件表达式2
      </p>
      <p>
       如果条件表达式2为True，执行条件表达式2后面的代码块，执行完之后继续执行嵌套if结构后面的代码
      </p>
      <p>
       如果条件表达式2为False，执行里面else后面的代码块，执行完之后继续执行嵌套if结构后面的代码
      </p>
     </li>
     <li>
      <p>
       如果条件表达式1为False，则执行外面else后面的代码块
      </p>
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：嵌套if理论上可以任意层次的嵌套，但是不要过多嵌套，一般不超过三层，否则不利于代码阅读。
    </p>
    <p>
     <strong>
      注意
     </strong>
     ：条件表达式可以是布尔值、关系运算符执行结果和逻辑运算符执行结果
    </p>

    <h4>
     <a id="25__213">
     </a>
     2.5 条件表达式
    </h4>
    <p>
     在Python中，条件表达式（也称为三元操作符或条件运算符）与许多其他编程语言（如C、C++、Java等）中的形式略有不同。在其他语言中，条件表达式通常写成
     <code>
      a &gt; b ? a : b
     </code>
     ，这意味着“如果
     <code>
      a
     </code>
     大于
     <code>
      b
     </code>
     ，则结果为
     <code>
      a
     </code>
     ，否则结果为
     <code>
      b
     </code>
     ”。
    </p>
    <p>
     然而，在Python中，这种语法是不被支持的。相反，Python使用了一种更冗长但更具可读性的语法来实现相同的功能。
    </p>
    <h5>
     <a id="1_219">
     </a>
     （1）语法结构
    </h5>
    <pre><code>result = 表达式1 if 条件表达式 else 表达式2
</code></pre>
    <h5>
     <a id="2_225">
     </a>
     （2）执行规律
    </h5>
    <ol>
     <li>
      <p>
       判断条件表达式的结果
      </p>
     </li>
     <li>
      <p>
       如果条件表达式为True，将表达式1的值存储在变量result中
      </p>
      <p>
       如果条件表达式为False，将表达式2的值存储在变量result中
      </p>
     </li>
    </ol>
    <h5>
     <a id="3_233">
     </a>
     （3）演示示例
    </h5>
