# 【SpringMVC】常用注解：@RequestHeader

## 1.作用

用于获取请求头

## 2.属性

value：提供消息头名称

required：是否必须有次消息头。

## 3.示例

先编写JSP代码

```
a href="demo1/useRequestHeader">获取请求消息头</a>
```

然后编写控制器代码

```
@RequestMapping("useRequestHeader")
 public String useRequestHeader(@RequestHeader(value="Accept
Language") String requestHeader){
 System.out.println(requestHeader);
 return "success";
 }
```

结果

```
zh-CN,zh;q=0.9
```
