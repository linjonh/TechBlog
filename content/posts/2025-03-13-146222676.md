    <p>
     指针代表内存中地址标识符，变量，数组都是存储内存中的数据。所以可以获得它们的地址，用指针来表示这块内存。
    </p>
    <p>
     如图输出内存中的地址。
    </p>
    <p>
     <img alt="" height="376" src="https://i-blog.csdnimg.cn/direct/1e1023c0d15746b3a91c3051d9d00f59.png" width="1258"/>
    </p>
    <p>
     <img alt="" height="187" src="https://i-blog.csdnimg.cn/direct/8b33a7b7365f4fe2b6274eea907e9c77.png" width="1111"/>
    </p>
    <p>
     对于一个函数来说，也是内存中存储这段数据，所以我们也可以获取函数的地址。
    </p>
    <p>
     函数指针的声明方式
    </p>
    <p>
     <img alt="" height="97" src="https://i-blog.csdnimg.cn/direct/e416771a539a49d6a5f1000293a53ec9.png" width="741"/>
    </p>
    <p>
     下面我们来看一个简单函数指针使用
    </p>
    <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
double mutiply(double a, double b)
{
	return a * b;
}
double add(double a, double b)
{
	return a + b;
}
int main()
{
	double(*ptr)(double,double ) = mutiply;
	double res = ptr(2, 3);
	double res1 = (*ptr)(2, 3);
	cout &lt;&lt; "ress=" &lt;&lt; res &lt;&lt; " res1=" &lt;&lt; res1 &lt;&lt; endl;

	ptr = &amp;add;   //凡是符合这样接口的函数都可以使用这样的指针来调用
	double res3 = ptr(1, 2);
	cout &lt;&lt; "res3=" &lt;&lt; res3 &lt;&lt; endl;

}</code></pre>
    <p>
     这种函数的定义方式看上去比较繁琐，我们在函数指针的声明前面加上typedefine,这样ptr就不在是一个变量而是一个类型别名，代表指向这样一个函数的指针类型，为了方便讲typedefine定义的指针类型别名与函数指针区分开来，我们可以将别名用大写字母作为区分。
    </p>
    <p>
     代码如下
    </p>
    <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
//typedef double (*FuncTypeptr)(double ,double);
typedef double (FuncTypeptr)(double, double);
double mutiply(double a, double b)
{
	return a * b;
}
int main()
{
	//FuncTypeptr ptr1= mutiply;
	FuncTypeptr* ptr1 = mutiply;
	double res = ptr1(1,2);
	cout &lt;&lt; "res=" &lt;&lt; res &lt;&lt; endl;
}</code></pre>
    <p>
     <img alt="" height="367" src="https://i-blog.csdnimg.cn/direct/d0ce1d7211f048e9bc90e83fe447c980.png" width="838"/>
    </p>
    <p>
     函数指针在C语言中的使用
    </p>
    <p>
     在排序函数中传入compare函数指针，这样排序函数就可以通过闯入的指针来判断数值的大小。
    </p>
    <p>
     代码如下
    </p>
    <pre><code class="language-cpp">#include&lt;vector&gt;
#include&lt;iostream&gt;
#include&lt;algorithm&gt;
using namespace std;
bool compare(float a, float b)
{
	return a &lt; b;
}
int main()
{
	vector&lt;float&gt;num{ 0,1,2,3,6,3,7,11 };
	sort(num.begin(), num.end(), compare);
	for (auto a : num)
		cout &lt;&lt; a &lt;&lt; " ";
}</code></pre>
    <p>
     在c++中我们可以通过重载一个类的函数运算符，来实现函数对象，使用函数对象代替函数指针。
    </p>
    <p>
     代码如下
    </p>
    <pre><code class="language-cpp">#include&lt;vector&gt;
#include&lt;iostream&gt;
#include&lt;algorithm&gt;
using namespace std;
class Compare
{
public:
	bool operator()(float a, float b)
	{
		return a &lt; b;
	}
};
int main()
{
	vector&lt;float&gt;num{ 0,1,2,3,6,3,7,11 };
	sort(num.begin(), num.end(), Compare());
	for (auto a : num)
		cout &lt;&lt; a &lt;&lt; " ";
}</code></pre>
    <p>
    </p>
    <p>
     <img alt="" height="354" src="https://i-blog.csdnimg.cn/direct/6b8cd699317345168ddf46970bfd06ab.png" width="1224"/>
    </p>
    <p>
    </p>
    <pre><code class="language-cpp">#include&lt;vector&gt;
#include&lt;iostream&gt;
using namespace std;


class DemoClass {
public:
	double add(double a, double b) { return a + b; };
	double multiply(double a, double b) { return a * b; };
};

int main()
{
	double (DemoClass:: * ptrMemberFunc)(double,double);
	ptrMemberFunc = &amp;DemoClass::add;
	DemoClass obj;
	double result = (obj.*ptrMemberFunc)(1,2);
	cout &lt;&lt; result &lt;&lt; endl;

	DemoClass* pobj = &amp;obj;
	ptrMemberFunc = &amp;DemoClass::multiply;
	result = (pobj-&gt;*ptrMemberFunc)(1, 2);
	cout &lt;&lt; result &lt;&lt; endl;
}</code></pre>
    <p>
     同样适用于多态类中的虚函数
