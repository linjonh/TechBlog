     Oracle19c进入EM Express（Oracle企业管理器）详细步骤
    </h1>

    <p>
     以下是使用Oracle 19c进入Oracle Enterprise Manager Database Express（EM Express）的详细步骤：
    </p>
    <p>
     ### **步骤 1：确认EM Express配置状态**
     <br/>
     1. **登录数据库服务器**
     <br/>
     使用Oracle用户或管理员权限账户登录操作系统。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/746a2853d23445a5821b24e0d609b6db.png"/>
    </p>
    <p>
     2. **查看EM Express端口配置**
     <br/>
     通过SQL*Plus连接到数据库，执行以下SQL查询确认HTTP/HTTPS端口：
     <br/>
     ```sql
     <br/>
     SELECT DBMS_XDB_CONFIG.getHttpsPort() AS HTTPS_PORT FROM DUAL;
     <br/>
     ```
     <br/>
     - 若返回`0`，表示未配置HTTPS端口。
     <br/>
     - 若返回非零值（如5500），则为当前HTTPS端口。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/330c09aac6414193a0c2d307a0c1ac2a.png"/>
    </p>
    <p>
     3. **配置端口（如未设置）**
     <br/>
     - **设置HTTPS端口**：
     <br/>
     ```sql
     <br/>
     EXEC DBMS_XDB_CONFIG.SETHTTPSPORT(5500);
     <br/>
     COMMIT;
     <br/>
     ```
     <br/>
     - **设置HTTP端口（不推荐）**：
     <br/>
     ```sql
     <br/>
     EXEC DBMS_XDB_CONFIG.SETHTTPPORT(5501);
     <br/>
     COMMIT;
     <br/>
     ```
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/e77c9f118cf74a1c8c3a63752e869092.png"/>
    </p>
    <p>
     ### **步骤 2：检查监听器与数据库状态**
     <br/>
     1. **确认监听器运行**（win+R输入cmd)
     <br/>
     ```bash
     <br/>
     lsnrctl status
     <br/>
     ```
     <br/>
     - 确保监听器状态为`READY`，且PORT列中包含EM Express的端口（如5500）。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/a8b4418cb4394434b61e085a87c75e61.png"/>
    </p>
    <p>
     2. **确认数据库实例运行**
     <br/>
     ```bash
     <br/>
     sqlplus / as sysdba
     <br/>
     SQL&gt; SELECT status FROM v$instance;
     <br/>
     ```
     <br/>
     - 确保输出为`OPEN`状态。
    </p>
    <p>
     ---
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/9590f630570a485c92ca322f3a6c10ed.png"/>
    </p>
    <p>
     ### **步骤 3：访问EM Express**
     <br/>
     1. **生成访问URL**
     <br/>
     - 默认格式：
     <br/>
     ```
     <br/>
     https://&lt;hostname&gt;:&lt;https_port&gt;/em
     <br/>
     ```
     <br/>
     （例如：`https://localhost:5500/em`）
    </p>
    <p>
     2. **浏览器访问**
     <br/>
     - 输入URL，首次访问需忽略浏览器的SSL证书警告（点击“高级”→“继续前往”）。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/7b06ce88e3f74f31bcc91b13edc7df6f.png"/>
    </p>
    <p>
     3. **登录凭据**
     <br/>
     - **用户名**：具有`SYSDBA`或`SYSOPER`权限的账户（如`SYS`）。
     <br/>
     - **密码**：安装数据库时设置的密码。
     <br/>
     - **连接身份**：选择`SYSDBA`（推荐）或`NORMAL`（需权限）。
    </p>
    <p>
     ---
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/b8c1f8e28fc24032b2ab1c1788689c31.png"/>
    </p>
    <p>
     ### **步骤 4：常见问题解决**
     <br/>
     - **端口无法访问**
     <br/>
     - 检查防火墙是否开放端口：
     <br/>
     ```bash
     <br/>
     # Linux/Unix
     <br/>
     sudo firewall-cmd --zone=public --add-port=5500/tcp --permanent
     <br/>
     sudo firewall-cmd --reload
    </p>
    <p>
     # Windows
     <br/>
     通过“控制面板”➔“Windows Defender 防火墙”➔ 添加允许端口5500的入站规则。
     <br/>
     ```
    </p>
    <p>
     - **EM Express页面显示空白/错误**
     <br/>
     - 重启Oracle相关服务：
     <br/>
     ```bash
     <br/>
     # Linux/Unix
     <br/>
     emctl stop dbconsole   # 如果配置了dbconsole（旧版本）
     <br/>
     emctl start dbconsole
    </p>
    <p>
     # Windows
     <br/>
     通过“服务”重启OracleService&lt;SID&gt;和OracleOraDB19Home1TNSListener。
     <br/>
     ```
    </p>
    <p>
     - **端口冲突或占用**
     <br/>
     - 修改HTTPS端口（例：改为5502）：
     <br/>
     ```sql
     <br/>
     EXEC DBMS_XDB_CONFIG.SETHTTPSPORT(5502);
     <br/>
     COMMIT;
     <br/>
     ```
    </p>
    <p>
     ---
    </p>
    <p>
     ### **示例操作流程（Linux环境）**
     <br/>
     1. 连接数据库并设置端口：
     <br/>
     ```sql
     <br/>
     sqlplus / as sysdba
     <br/>
     SQL&gt; EXEC DBMS_XDB_CONFIG.SETHTTPSPORT(5500);
     <br/>
     SQL&gt; COMMIT;
     <br/>
     SQL&gt; EXIT;
     <br/>
     ```
    </p>
    <p>
     2. 检查监听器状态：
     <br/>
     ```bash
     <br/>
     lsnrctl status
     <br/>
     ```
    </p>
    <p>
     3. 开放防火墙端口：
     <br/>
     ```bash
     <br/>
     sudo firewall-cmd --add-port=5500/tcp --permanent
     <br/>
     sudo firewall-cmd --reload
     <br/>
     ```
    </p>
    <p>
     4. 浏览器访问：
     <br/>
     ```
     <br/>
     https://localhost:5500/em
     <br/>
     ```
     <br/>
     - 使用账户`SYS`及密码登录，选择`SYSDBA`角色。
    </p>
    <p>
     ---
    </p>
    <p>
     通过以上步骤，您应能成功访问Oracle 19c的企业管理器EM Express。若仍遇到问题，请检查`$ORACLE_HOME/rdbms/log/em*.log`日志文件排查详细错误。
    </p>
