    <h3>
     说明
    </h3>
    <p>
     尼克与格莱尔很喜欢玩“加加乐”游戏，游戏规则是一方报出一个四位数，另一方说出该数的各个数位之和。如尼克说1212，格莱尔就说6；尼克说5675，格莱尔就说23。
     <br/>
     试编一程序，输入一个整数，输出它的各个数位之和。
    </p>
    <h3>
     输入格式
    </h3>
    <p>
     输入一个整数。
    </p>
    <h3>
     输出格式
    </h3>
    <p>
     输出各个数位之和。
    </p>
    <h3>
     样例
    </h3>
    <h3>
     输入数据 1
    </h3>
    <pre><code>1234</code></pre>
    <p>
     <a rel="nofollow">
      Copy
     </a>
    </p>
    <h3>
     输出数据 1
    </h3>
    <pre><code>10</code></pre>
    <h2>
     思路：
    </h2>
    <h6>
     这道题并不难，只需要把这个四位数进行拆分即可，用四个不同的变量存储四个不同的数组后相加并输出就可以个。要注意，拆分细节自己在下面看。
    </h6>
    <h6>
    </h6>
    <h2>
     题解：
    </h2>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;
int a,b,c,d,e,f,g;
int main(){
	cin&gt;&gt;a;
	int ge=a%10;
	int shi=a/10%10;
	int bai=a/100%10;
	int qian=a/1000%10;
	cout&lt;&lt;ge+shi+bai+qian;
	return 0;
}</code></pre>
