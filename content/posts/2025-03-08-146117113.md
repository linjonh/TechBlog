    <h2 style="background-color:transparent">
     一、网页
    </h2>
    <h3>
     1.什么是网页：
    </h3>
    <p>
     <span style="color:#fe2c24">
      <strong>
       网站
      </strong>
     </span>
     是指在因特网上根据一定的规则，使用 HTML 等制作的用于展示特定内容相关的
     <span style="color:#fe2c24">
      <strong>
       网页集合
      </strong>
     </span>
     。
    </p>
    <p>
     <span style="color:#fe2c24">
      <strong>
       网页
      </strong>
     </span>
     是网站中的一“页”，通常是
     <span style="color:#fe2c24">
      <strong>
       HTML 格式的文件
      </strong>
     </span>
     ，它要通过浏览器来阅读。
    </p>
    <p>
     网页是构成网站的基本元素，它通常由图片、链接、文字、声音、视频等元素组成。通常我们看到的网页，常见以 .htm 或 .html 后缀结尾的文件，因此将其俗称为 HTML 文件。
    </p>
    <h3>
     2.什么是 HTML
    </h3>
    <p>
     HTML 指的是
     <span style="color:#fe2c24">
      <strong>
       超文本标记语言 (Hyper Text Markup Language)
      </strong>
     </span>
     ，它是用来描述网页的一种语言。
     <br/>
     HTML 不是一种编程语言，而是一种
     <span style="color:#fe2c24">
      <strong>
       标记语言 (markup language)
      </strong>
     </span>
     。
     <br/>
     标记语言是一套标记标签 (markup tag)。
     <br/>
     <br/>
     所谓
     <span style="color:#fe2c24">
      <strong>
       超文本，有
      </strong>
     </span>
     2 层含义：
     <br/>
     1. 它可以加入图片、声音、动画、多媒体等内容（超越了文本限制 ）。
     <br/>
     2. 它还可以从一个文件跳转到另一个文件，与世界各地主机的文件连接（超级链接文本 ）。
    </p>
    <p>
    </p>
    <h3>
     3.网页的形成
    </h3>
    <p>
     网页是由网页元素组成的，这些元素是利用 html 标签描述出来，然后通过浏览器解析来显示给用户的。
     <img alt="" height="441" src="https://i-blog.csdnimg.cn/direct/71f014d6081c4a7692a69cded04a3dda.png" width="994"/>
    </p>
    <p>
     前端人员开发代码 ----&gt; 浏览器显示代码（解析、渲染） -----&gt; 生成最后的 Web 页面
    </p>
    <h3>
     4.网页总结
    </h3>
    <p>
     网页是图片､链接､文字､声音､视频等元素组成, 其实就是一个html文件(后缀名为html)
    </p>
    <p>
     网页生成制作: 有前端人员书写 HTML 文件, 然后浏览器打开,就能看到了网页.
    </p>
    <p>
     HTML: 超文本标记语言, 用来制作网页的一门语言. 有标签组成的. 比如 图片标签 链接标签 视频标签等…
    </p>
    <h2>
     二、常用浏览器
    </h2>
    <p>
     网页是通过浏览器来展示的, 关于浏览器我们要介绍以下两点:
    </p>
    <p>
     1. 常用的浏览器
    </p>
    <p>
     2. 浏览器内核
    </p>
    <p>
     浏览器是网页显示､运行的平台｡常用的浏览器有 IE､火狐(Firefox)､谷歌(Chrome)､Safari和Opera等｡ 平时称为五大浏览器｡
    </p>
    <p>
     <img alt="" height="314" src="https://i-blog.csdnimg.cn/direct/19e40a754ea44cb3814702ac94a08975.png" width="450"/>
    </p>
    <p>
     查看浏览器市场份额:
     <a class="link-info" href="http://tongji.baidu.com/data/browser" rel="nofollow" title="http://tongji.baidu.com/data/browser">
      http://tongji.baidu.com/data/browser
     </a>
    </p>
    <h3>
     1.浏览器内核
    </h3>
    <p>
     浏览器内核(渲染引擎): 负责读取网页内容,整理讯息,计算网页的显示方式并显示页面｡
    </p>
    <p>
     <img alt="" height="244" src="https://i-blog.csdnimg.cn/direct/a07eb0458efc41be8b152ec95421134d.png" width="966"/>
    </p>
    <p>
     目前国内一般浏览器都会采用 Webkit/Blink 内核,如 360､UC､QQ､搜狗等｡
    </p>
    <h2>
     三、Web 标准(重点)
    </h2>
    <p>
     接下来围绕web标准,我们学习以下两点: Web 标准是由 W3C 组织和其他标准化组织制定的一系列标准的集合｡
    </p>
    <p>
     W3C(万维网联盟)是国际最著名的标准化组织｡
    </p>
    <p>
     接下来围绕web标准,我们学习以下两点:
    </p>
    <blockquote>
     <p>
      1. 为什么需要web标准
     </p>
     <p>
      2. Web标准的构成
     </p>
    </blockquote>
    <h3>
     1.为什么需要 Web 标准
    </h3>
    <p>
     浏览器不同,它们显示页面或者排版就有些许差异｡
    </p>
    <p>
     <img alt="" height="460" src="https://i-blog.csdnimg.cn/direct/75c5fadf6b944e35b27eafcca30c09fb.png" width="736"/>
    </p>
    <blockquote>
     <p>
      遵循 Web 标准除了可以让不同的开发人员写出的页面更标准、更统一外，还有以下优点：
      <br/>
      1. 让 Web 的发展前景更广阔。
      <br/>
      2. 内容能被更广泛的设备访问。
      <br/>
      3. 更容易被搜寻引擎搜索。
      <br/>
      4. 降低网站流量费用。
      <br/>
      5. 使网站更易于维护。
      <br/>
      6. 提高页面浏览速度。
      <br/>
     </p>
    </blockquote>
    <h3 style="background-color:transparent">
     2.Web 标准的构成
    </h3>
    <p>
     主要包括
     <span style="color:#fe2c24">
      <strong>
       结构(Structure) ､表现(Presentation)和行为(Behavior)
      </strong>
     </span>
     三个方面｡
    </p>
    <p>
     <img alt="" height="211" src="https://i-blog.csdnimg.cn/direct/01eef24da78844419c80016aecaa2f76.png" width="979"/>
    </p>
    <p>
     Web 标准提出的最佳体验方案:结构､样式､行为相分离
    </p>
    <p>
     简单理解:
     <span style="color:#fe2c24">
      <strong>
       结构写到 HTML 文件中, 表现写到 CSS 文件中, 行为写到 JavaScript 文件中｡
      </strong>
     </span>
    </p>
    <p>
     <img alt="" height="604" src="https://i-blog.csdnimg.cn/direct/578c251d716f45bea45ba0a4532b1da7.png" width="480"/>
    </p>
