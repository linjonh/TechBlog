    <h2>
     <a id="1_7">
     </a>
     1.条件判断语句
    </h2>
    <h3>
     <a id="11_if__8">
     </a>
     1.1 if 语句
    </h3>
    <p>
     if语句是常见的条件判断语句
    </p>
    <blockquote>
     <p>
      语法格式：if 条件表达式 ：语句
      <br/>
      默认情况下，if语句只会控制紧随其后的那条语句，
      <br/>
      如果希望控制多条语句，可以使用代码块（即在冒号后另起一行，带有一个缩进）
     </p>
    </blockquote>
    <p>
     <img alt="代码块" src="https://i-blog.csdnimg.cn/direct/dfb21516d9ed494aaf5cb94b4168be02.png"/>
    </p>
    <blockquote>
     <p>
      执行流程：if语句在执行时，先对条件表达式进行求值判断，
      <br/>
      如果为True，则执行if后的语句，如果为False，则不执行
     </p>
    </blockquote>
    <p>
     <strong>
      注意
     </strong>
     <br/>
     在if的条件表达式中可以使用逻辑运算符连接多个条件
    </p>
    <p>
     缩进的方式
    </p>
    <blockquote>
     <ol>
      <li>
       一种使用Tab键，一种使用空格（4个）
      </li>
      <li>
       Python的官方文档中推荐使用空格来缩进
      </li>
      <li>
       Python代码中使用的缩进方式必须统一
      </li>
     </ol>
     <hr/>
     <p>
      为了统一缩进方式，可以在sublime中设置按Tab键时生成4个空格
      <br/>
      设置步骤：
      <br/>
      首选项-设置，在弹出的文档中搜索 translate_tabs_to_spaces，将这个值设置为true
     </p>
    </blockquote>

     2. input 函数
    </h3>
    <blockquote>
     <p>

      <li>
       input函数中可以设置一个字符串参数，这个字符串将会作为提示文字显示
      </li>
      <li>
       用户输入完成后，输入的内容会以返回值的形式返回
      </li>
     </ol>
    </blockquote>

    <h3>
     <a id="4ifelifelse__57">
     </a>
     4.if-elif-else 语句
    </h3>
    <p>
     多条件判断语句，语法结构如下：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件表达式 <span class="token punctuation">:</span> 
	代码块
<span class="token keyword">elif</span> 条件表达式 <span class="token punctuation">:</span> 
	代码块
<span class="token keyword">elif</span> 条件表达式 <span class="token punctuation">:</span> 
	代码块
	<span class="token punctuation">.</span>
	<span class="token punctuation">.</span>
	<span class="token punctuation">.</span>
<span class="token keyword">else</span> <span class="token punctuation">:</span> 
	代码块

</code></pre>
    <p>
     代码执行流程:
    </p>
    <blockquote>
     <p>
      if-elif-else 语句在执行时，会自上而下一次对条件表达式进行求值判断，
     </p>
     <ul>
      <li>
       如果表达式的结果为True，则执行当前代码块，然后语句结束
      </li>
      <li>
       如果表达式结果为False，则继续向下判断，直到找到True为止
      </li>
      <li>
       如果所有的表达式都是False，则执行else后的代码块
      </li>
     </ul>
     <hr/>
     <p>
      <strong>
       if-elif-else语句只有一个代码块会被执行
      </strong>
     </p>
    </blockquote>
    <p>
     eg:

     2.循环语句
    </h2>
    <p>
     循环语句可以指定的代码块重复指定的次数，循环语句分为：

    <ul>
     <li>
      while循环
     </li>
     <li>
      for循环
     </li>
    </ul>
    <h3>
     <a id="21_while_93">
     </a>
     2.1 while语句
    </h3>
    <p>
     语法结构
    </p>
    <pre><code class="prism language-python"> <span class="token keyword">while</span> 条件表达式 <span class="token punctuation">:</span> 
 	代码块
</code></pre>
    <blockquote>
     <p>
      执行流程:
     </p>
     <ul>
      <li>
       while语句在执行时,会先对while后的条件表达式进行求值判断,
      </li>
      <li>
       如果判断结果为True，则执行循环体（代码块）
      </li>
      <li>
       循环体执行完毕，继续对条件表达式进行求值判断，以此类推，直到判断结果为False，则循环终止
      </li>
      <li>
       如果循环有对应的else，则执行else后的代码块
      </li>
     </ul>
    </blockquote>
    <ul>
     <li>
      循环三要素
     </li>
    </ul>
    <ol>
     <li>
      初始化表达式，通过初始化表达式初始化一个变量
     </li>
     <li>
      条件表达式，用来设置循环执行的条件
     </li>
     <li>
      更新表达式，修改初始化变量的值
     </li>
    </ol>
    <p>
     eg：
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 1.初始化表达式</span>
i <span class="token operator">=</span> <span class="token number">0</span>
<span class="token comment"># 2.条件表达式,设置循环执行条件</span>

	<span class="token comment"># 3.更新表达式,修改初始化值</span>
	i <span class="token operator">+=</span> <span class="token number">1</span>

</code></pre>
