    <h5>
     <a id="Linux_2">
     </a>
     一、Linux软件包管理
    </h5>
    <h6>
     <a id="1__4">
     </a>
     1. 什么是软件包？
    </h6>
    <p>
     在Linux下安装软件，一种常见的方法是下载程序的源代码，并进行编译，得到可执行程序。然而，这种方法既耗时又繁琐。
     <mark>
      为了方便，人们将一些常用的软件提前编译好，做成软件包，存放在服务器上。通过
      <strong>
       包管理器
      </strong>
      ，用户可以方便地获取这些编译好的软件包，直接进行安装
     </mark>
     。
    </p>
    <h6>
     <a id="2__8">
     </a>
     2. 软件包和软件包管理器
    </h6>
    <p>
     软件包和软件包管理器之间的关系，类似于“App”和“应用商店”。Linux中有多种软件包管理器，
     <code>
      yum
     </code>
     是其中非常常用的一种。
    </p>
    <ul>
     <li>
      <strong>
       yum（Yellow dog Updater, Modified）
      </strong>
      ：主要应用于Fedora、RedHat、CentOS等发行版。
     </li>
    </ul>
    <h6>
     <a id="3_yum_14">
     </a>
     3. yum的使用
    </h6>
    <h6>
     <a id="_16">
     </a>
     注意事项
    </h6>
    <ul>
     <li>
      使用yum的所有操作必须保证主机（虚拟机）网络畅通，可以通过
      <code>
       ping
      </code>
      命令验证。
     </li>
    </ul>
    <h6>
     <a id="_20">
     </a>
     查看软件包
    </h6>
    <p>
     使用
     <code>
      yum list
     </code>
     命令可以罗列出当前可用的软件包。由于包的数量可能非常多，可以使用
     <code>
      grep
     </code>
     命令筛选关注的包。
    </p>
    <pre><code class="prism language-bash">bash复制代码

yum list <span class="token operator">|</span> <span class="token function">grep</span> lrzsz
</code></pre>
    <p>
     输出示例：
    </p>
    <pre><code>复制代码

lrzsz.x86_64 0.12.20-36.el7 @base
</code></pre>
    <p>
     软件包名称解释：
    </p>
    <ul>
     <li>
      <code>
       lrzsz
      </code>
      ：软件包名称
     </li>
     <li>
      <code>
       0.12.20
      </code>
      ：主版本号
     </li>
     <li>
      <code>
       36.el7
      </code>
      ：次版本号.源程序发行号-软件包的发行号
     </li>
     <li>
      <code>
       x86_64
      </code>
      ：主机平台.CPU架构（64位系统安装包）
     </li>
     <li>
      <code>
       base
      </code>
      ：软件源名称
     </li>
    </ul>
    <h6>
     <a id="_46">
     </a>
     安装软件
    </h6>
    <p>
     使用
     <code>
      yum install
     </code>
     命令安装软件，例如安装
     <code>
      gcc
     </code>
     ：
    </p>
    <pre><code class="prism language-bash">bash复制代码

<span class="token function">sudo</span> yum <span class="token function">install</span> gcc
</code></pre>
    <p>
     yum会自动找到并下载所有依赖的软件包，确认安装后敲“y”。
    </p>
    <h6>
     <a id="_58">
     </a>
     卸载软件
    </h6>
    <p>
     使用
     <code>
      yum remove
     </code>
     命令卸载软件，例如卸载
     <code>
      lrzsz
     </code>
     ：
    </p>
    <pre><code class="prism language-bash">bash复制代码

<span class="token function">sudo</span> yum remove lrzsz
</code></pre>
    <p>
     注意：安装软件时需要向系统目录中写入内容，一般需要
     <code>
      sudo
     </code>
     权限或切换到
     <code>
      root
     </code>
     账户。
    </p>
    <h6>
     <a id="_70">
     </a>
     切换软件源
    </h6>
    <p>
     切换软件源本质上是更改配置文件。例如，将CentOS 7的yum镜像源切换到163镜像源：
    </p>
    <p>
     操作清单：
    </p>
    <ol>
     <li>
      备份原配置文件：
     </li>
    </ol>
    <pre><code class="prism language-bash">bash复制代码

<span class="token function">sudo</span> <span class="token function">cp</span> /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
</code></pre>
    <ol>
     <li>
      下载新的配置文件：
     </li>
    </ol>
    <pre><code class="prism language-bash">bash复制代码

<span class="token function">sudo</span> <span class="token function">wget</span> <span class="token parameter variable">-O</span> /etc/yum.repos.d/CentOS-Base.repo http://mirrors.163.com/.help/CentOS7-Base-163.repo
</code></pre>
    <ol>
     <li>
      清理yum缓存并更新：
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum clean all
<span class="token function">sudo</span> yum makecache
</code></pre>
    <h5>
     <a id="LinuxVim_99">
     </a>
     二、Linux编辑器——Vim
    </h5>
    <p>
     Vim是一个强大的
     <strong>
      文本编辑器
     </strong>
     ，广泛用于编写和修改配置文件。Vim有
     <strong>
      多种模式
     </strong>
     ，下面介绍三种核心模式：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cd619c20a20d4cfeb27b7ee17fc9b49d.png"/>
    </p>
    <h6>
     <a id="1_Vim_106">
     </a>
     1. Vim的三种模式
    </h6>
    <ul>
     <li>
      <strong>
       命令模式（Command Mode）
      </strong>
      ：控制屏幕光标的移动，字符、字或行的删除，移动复制某区段及进入插入模式或底行模式。
     </li>
     <li>
      <strong>
       插入模式（Insert Mode）
      </strong>
      ：只有在此模式下，才可以进行文字输入。按
      <code>
       ESC
      </code>
      键可回到命令模式。
     </li>
     <li>
      <strong>
       底行模式（Last Line Mode）
      </strong>
      ：文件保存或退出，也可以进行文件替换，找字符串，列出行号等操作。在命令模式下，按
      <code>
       Shift + :
      </code>
      即可进入该模式。
     </li>
    </ul>
    <h6>
     <a id="2_Vim_112">
     </a>
     2. Vim的基本操作
    </h6>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c2ac960f65384aca935a1d3bf7cc7695.png"/>
    </p>
    <h6>
     <a id="Vim_117">
     </a>
     进入Vim
    </h6>
    <pre><code class="prism language-bash">bash复制代码

<span class="token function">vim</span> test.c
</code></pre>
    <p>
     进入Vim后，默认处于命令模式。
    </p>
    <h6>
     <a id="_127">
     </a>
     模式切换
    </h6>
    <ul>
     <li>
      <strong>
       命令模式切换到插入模式
      </strong>
      ：
      <ul>
       <li>
        <code>
         i
        </code>
        ：在当前光标位置开始插入。
       </li>
       <li>
        <code>
         a
        </code>
        ：在当前光标位置的下一个字符开始插入。
       </li>
       <li>
        <code>
         o
        </code>
        ：在当前行的下一行开始插入。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       插入模式切换到命令模式
      </strong>
      ：按
      <code>
       ESC
      </code>
      键。
     </li>
     <li>
      <strong>
       命令模式切换到底行模式
      </strong>
      ：按
      <code>
       Shift + :
      </code>
      。
     </li>
    </ul>
    <h6>
     <a id="_136">
     </a>
     移动光标
    </h6>
    <ul>
     <li>
      <code>
       h
      </code>
      ：光标左移一格。
     </li>
     <li>
      <code>
       j
      </code>
      ：光标下移一格。
     </li>
     <li>
      <code>
       k
      </code>
      ：光标上移一格。
     </li>
     <li>
      <code>
       l
      </code>
      ：光标右移一格。
     </li>
     <li>
      <code>
       G
      </code>
      ：移动到文章最后。
     </li>
     <li>
      <code>
       $
      </code>
      ：移动到光标所在行的行尾。
     </li>
     <li>
      <code>
       ^
      </code>
      ：移动到光标所在行的行首。
     </li>
     <li>
      <code>
       w
      </code>
      ：光标跳到下一个字的开头。
     </li>
     <li>
      <code>
       e
      </code>
      ：光标跳到下一个字的字尾。
     </li>
     <li>
      <code>
       b
      </code>
      ：光标回到上一个字的开头。
     </li>
     <li>
      <code>
       gg
      </code>
      ：移动到文本开始。
     </li>
     <li>
      <code>
       Shift + G
      </code>
      ：移动到文本末端。
     </li>
     <li>
      <code>
       Ctrl + b
      </code>
      ：屏幕往后移动一页。
     </li>
     <li>
      <code>
       Ctrl + f
      </code>
      ：屏幕往前移动一页。
     </li>
     <li>
      <code>
       Ctrl + u
      </code>
      ：屏幕往后移动半页。
     </li>
     <li>
      <code>
       Ctrl + d
      </code>
      ：屏幕往前移动半页。
     </li>
    </ul>
    <h6>
     <a id="_155">
     </a>
     删除文字
    </h6>
    <ul>
     <li>
      <code>
       x
      </code>
      ：删除光标所在位置的一个字符。
     </li>
     <li>
      <code>
       #x
      </code>
      ：例如
      <code>
       6x
      </code>
      ，表示删除光标所在位置的后面6个字符。
     </li>
     <li>
      <code>
       X
      </code>
      ：删除光标所在位置的前一个字符。
     </li>
     <li>
      <code>
       #X
      </code>
      ：例如
      <code>
       20X
      </code>
      ，表示删除光标所在位置的前面20个字符。
     </li>
     <li>
      <code>
       dd
      </code>
      ：删除光标所在行。
     </li>
     <li>
      <code>
       #dd
      </code>
      ：从光标所在行开始删除#行。
     </li>
    </ul>
    <h6>
     <a id="_164">
     </a>
     复制与粘贴
    </h6>
    <ul>
     <li>
      <code>
       yw
      </code>
      ：将光标所在之处到字尾的字符复制到缓冲区中。
     </li>
     <li>
      <code>
       #yw
      </code>
      ：复制#个字到缓冲区。
     </li>
     <li>
      <code>
       yy
      </code>
      ：复制光标所在行到缓冲区。
     </li>
     <li>
      <code>
       #yy
      </code>
      ：例如
      <code>
       6yy
      </code>
      ，表示拷贝从光标所在的该行往下数6行文字。
     </li>
     <li>
      <code>
       p
      </code>
      ：将缓冲区内的字符贴到光标所在位置。
     </li>
    </ul>
    <h6>
     <a id="_172">
     </a>
     替换
    </h6>
    <ul>
     <li>
      <code>
       r
      </code>
      ：替换光标所在处的字符。
     </li>
     <li>
      <code>
       R
      </code>
      ：替换光标所到之处的字符，直到按下
      <code>
       ESC
      </code>
      键为止。
     </li>
    </ul>
    <h6>
     <a id="_177">
     </a>
     撤销与重做
    </h6>
    <ul>
     <li>
      <code>
       u
      </code>
      ：撤销上一次操作。
     </li>
     <li>
      <code>
       Ctrl + r
      </code>
      ：撤销的恢复更改。
     </li>
    </ul>
    <h6>
     <a id="_182">
     </a>
     查找与替换
    </h6>
    <ul>
     <li>
      <code>
       /关键字
      </code>
      ：从当前光标位置向后查找。
     </li>
     <li>
      <code>
       ?关键字
      </code>
      ：从当前光标位置向前查找。
     </li>
    </ul>
    <h6>
     <a id="_187">
     </a>
     末行模式命令集
    </h6>
    <ul>
     <li>
      <code>
       set nu
      </code>
      ：在文件中的每一行前面列出行号。
     </li>
     <li>
      <code>
       #G
      </code>
      ：例如
      <code>
       15G
      </code>
      ，表示移动光标至文章的第15行行首。
     </li>
     <li>
      <code>
       :w
      </code>
      ：保存当前文件。
     </li>
     <li>
      <code>
       :wq
      </code>
      ：保存并退出Vim。
     </li>
     <li>
      <code>
       :q!
      </code>
      ：不保存强制退出Vim。
     </li>
    </ul>
    <h6>
     <a id="_195">
     </a>
     批量化注释
    </h6>
    <ul>
     <li>
      <strong>
       批量注释
      </strong>
      （适用于 C/Java/JS 等语言）
      <ul>
       <li>
        块选择模式：
        <code>
         Ctrl + v
        </code>
        进入可视块模式
       </li>
       <li>
        选择行范围：用
        <code>
         j/k
        </code>
        或
        <code>
         10j
        </code>
        （向下选10行）快速选中多
       </li>
       <li>
        插入注释符：按
        <code>
         Shift + i
        </code>
        进入插入模式
        <br/>
        输入注释符：输入
        <code>
         //
        </code>
        （不同语言可能用 # 或 "）
        <br/>
        生效操作：按
        <code>
         Esc
        </code>
        （需等待 1-2 秒，Vim 会将操作应用到所有选中行）
       </li>
      </ul>
     </li>
    </ul>
    <p>
     注意：若操作后只有首行被注释，说明未正确退出插入模式，需确保按
     <code>
      Esc
     </code>
     后看到所有行同步变化
    </p>
    <ul>
     <li>
      <strong>
       批量去注释
      </strong>
      （需精确列选择）
      <ul>
       <li>
        块选择模式：
        <code>
         Ctrl + v
        </code>
       </li>
       <li>
        精准选中注释符：用
        <code>
         l
        </code>
        或
        <code>
         h
        </code>
        横向移动光标，确保只选中注释符（如
        <code>
         //
        </code>
        ）
       </li>
       <li>
        删除操作：按
        <code>
         d
        </code>
        或
        <code>
         x
        </code>
        （若选中单列）
       </li>
      </ul>
     </li>
    </ul>
    <p>
     注意：若注释前有不定量空格，建议先用
     <code>
      :left
     </code>
     命令对齐代码，或用正则替换更高效
    </p>

    <h6>
     <a id="3_Vim_214">
     </a>
     3. Vim配置
    </h6>
    <p>
     Vim的配置文件位于
     <code>
      /etc/vimrc
     </code>
     （公共配置文件）和用户主目录下的
     <code>
      .vimrc
     </code>
     （私有配置文件）。
    </p>
    <h6>
     <a id="_218">
     </a>
     常用配置选项
    </h6>
    <ul>
     <li>
      <code>
       syntax on
      </code>
      ：启用语法高亮。
     </li>
     <li>
      <code>
       set nu
      </code>
      ：显示行号。
     </li>
     <li>
      <code>
       set shiftwidth=4
      </code>
      ：设置缩进的空格数为4。
     </li>
    </ul>
    <h6>
     <a id="_224">
     </a>
     插件安装
    </h6>
    <ul>
     <li>
      <strong>
       TagList插件
      </strong>
      ：提供代码标签的浏览功能。
     </li>
     <li>
      <strong>
       WinManager插件
      </strong>
      ：提供文件浏览器和窗口管理器功能。
     </li>
    </ul>
    <p>
     安装插件步骤：
    </p>
    <ol>
     <li>
      下载并解压插件。
     </li>
     <li>
      将解压后的
      <code>
       doc
      </code>
      目录内容放到
      <code>
       ~/.vim/doc
      </code>
      。
     </li>
     <li>
      将解压后的
      <code>
       plugin
      </code>
      目录内容放到
      <code>
       ~/.vim/plugin
      </code>
      。
     </li>
     <li>
      在
      <code>
       ~/.vimrc
      </code>
      中添加相应的配置。
     </li>
    </ol>
    <p>
     例如，安装TagList插件后，在
     <code>
      ~/.vimrc
     </code>
     中添加：
    </p>
    <pre><code class="prism language-vim">let Tlist_Show_One_File=1
let Tlist_Exit_OnlyWindow=1
let Tlist_Use_Right_Window=1
</code></pre>
    <p>
     安装WinManager插件后，在
     <code>
      ~/.vimrc
     </code>
     中添加：
    </p>
    <pre><code class="prism language-vim">let g:winManagerWindowLayout='FileExplorer|TagList'
nmap wm :WMToggle&lt;cr&gt;
</code></pre>
    <p>
     重启Vim后，在命令模式下输入
     <code>
      wm
     </code>
     即可看到效果。
    </p>
