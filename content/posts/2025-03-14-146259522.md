### 🏭版本类型示意

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/acffcaedd8de4115b4daf93f9c91a276.png#pic_center)

| 字段类型 | **可能取值** |
| --- | --- |
| **处理器架构**   (architecture) | `i686` （32位 x86）   `x86_64` （64位 x86） |
| **线程模型**   (thread model) | `posix` (Portable Operating System Interface)（跨平台兼容）   `win32` (Windows 32-bit API)（Windows 原生，直接掉系统 API）   `mcf` (Mingw Cooperative Fibers)（特定优化配置） |
| **异常模型**   (exception model) | `dwarf` (Debugging With Attributed Record Formats)（32位架构使用，一种调试信息格式，也与异常处理有关）   `seh` (Structured Exception Handling)（64位架构使用，结构化异常处理） |
| **C运行时库**   (C runtime) | `ucrt` (Universal C Runtime)（Windows 10+ 通用运行时）   `msvcrt` (Microsoft Visual C++ Runtime)（旧版 Microsoft Visual C++ 运行时，用于向后兼容） |
| **运行时版本**   (runtime version) | `rt_v12` (Runtime Version )（版本号 12） |
| **修订号**   (build revision) | `rev1` (Revision)（修订号 1） |
