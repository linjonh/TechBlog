    <p>
     本内容是建立在有java的基础上去学习Kotlin的这门语言的，所以更多的是记录一些与java不同的之处，或者是Kotlin的特性等。
    </p>
    <p>
    </p>
    <h2 id="基本类型" style="margin-left:0; margin-right:0">
     基本类型
    </h2>
    <p>
     <span style="color:#333333">
      在 Kotlin 中，所有东西都是对象，在这个意义上讲我们可以在任何变量上调用成员函数和属性。 一些类型可以有特殊的内部表示——例如，数字、字符和布尔值可以
     </span>
     <span style="color:#333333">
      在运行时表示为原生类型值，但是对于用户来说，它们看起来就像普通的类。 在本节中，我们会描述 Kotlin 中使用的基本类型：数字、字符、布尔值、数组与字符串。
     </span>
    </p>
    <p>
    </p>
    <p>
    </p>
    <p>
    </p>
    <p>
    </p>
    <h3 id="数字" style="margin-left:0; margin-right:0">
     数字
    </h3>
    <p>
     <span style="color:#333333">
      Kotlin 处理数字在某种程度上接近 Java，但是并不完全相同。例如，对于数字没有隐式拓宽转换（如 Java 中
     </span>
     <code>
      int
     </code>
     <span style="color:#333333">
      可以隐式转换为
     </span>
     <code>
      long
     </code>
     <span style="color:#333333">
      ——译者注)，另外有些情况的字面值略有不同。
     </span>
    </p>
    <p>
    </p>
    <p>
    </p>
    <p>
    </p>
    <p>
    </p>
    <h4 id="字面常量" style="margin-left:0; margin-right:0">
     字面常量
    </h4>
    <p>
    </p>
    <p>
     数值常量字面值有以下几种:
    </p>
    <ul style="margin-left:30px; margin-right:0">
     <li>
      十进制:
      <code>
       123
      </code>
      <ul style="margin-left:30px; margin-right:0">
       <li>
        Long 类型用大写
        <code>
         L
        </code>
        标记:
        <code>
         123L
        </code>
       </li>
      </ul>
     </li>
     <li>
      十六进制:
      <code>
       0x0F
      </code>
     </li>
     <li>
      二进制:
      <code>
       0b00001011
      </code>
     </li>
    </ul>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
       </span>
      </span>
     </span>
    </p>
    <p>
     Kotlin 同样支持浮点数的常规表示方法:
    </p>
    <ul style="margin-left:30px; margin-right:0">
     <li>
      默认 double：
      <code>
       123.5
      </code>
      、
      <code>
       123.5e10
      </code>
     </li>
     <li>
      Float 用
      <code>
       f
      </code>
      或者
      <code>
       F
      </code>
      标记:
      <code>
       123.5f
      </code>
     </li>
    </ul>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <strong>
         注意：
        </strong>
        在 Kotlin 中字符不是数字，且
        <span style="color:#333333">
         不支持八进制，且字符不是数值
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
    </p>
    <h4 id="表示方式" style="margin-left:0; margin-right:0">
     表示方式
    </h4>
    <p>
    </p>
    <p>
     在 Java 平台数字是物理存储为 JVM 的原生类型，除非我们需要一个可空的引用（如
     <code>
      Int?
     </code>
     ）或泛型。 后者情况下会把数字装箱
    </p>
    <p>
     <span style="color:#333333">
      注意数字装箱不必保留同一性:
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <strong>
          val
         </strong>
         a
         <span style="background-color:#f7f7f7">
          :
         </span>
         Int=
         <span style="color:#0000ff">
          10000
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         print
         <span style="background-color:#f7f7f7">
          (
         </span>
         a===a
         <span style="background-color:#f7f7f7">
          )
         </span>
         <span style="color:gray">
          // 输出“true”
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <strong>
          val
         </strong>
         boxedA
         <span style="background-color:#f7f7f7">
          :
         </span>
         Int?=a
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <strong>
          val
         </strong>
         anotherBoxedA
         <span style="background-color:#f7f7f7">
          :
         </span>
         Int?=a
        </span>
       </span>

     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">

     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <span style="color:#333333">
          另一方面，它保留了相等性:
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">

         <strong>

    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         print
         <span style="background-color:#f7f7f7">
          (
         </span>
         a==a
         <span style="background-color:#f7f7f7">
          )
         </span>
         <span style="color:gray">
          // 输出“true”
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <strong>
          val
         </strong>
         boxedA
         <span style="background-color:#f7f7f7">
          :
         </span>
         Int?=a

    </p>
    <p>

    <h4 id="表示方式" style="margin-left:0; margin-right:0">
     类型转换
    </h4>
    <p>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         Kotlin 不能像java那样类型那样强制转换，int类型数据不能默认和long类型的数值进行比较大小。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         比如在java中：
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         int a;
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         long b = 100;
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         a = b;（在kotlin中这样是不允许的,必须写成：a = b.toInt()）
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <span style="color:#333333">
          每个数字类型支持如下的转换:
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
    </p>
    <ul style="margin-left:30px; margin-right:0">
     <li>
      <code>
       toByte(): Byte
      </code>
     </li>
     <li>
      <code>
       toShort(): Short
      </code>
     </li>
     <li>
      <code>
       toInt(): Int
      </code>
     </li>
     <li>
      <code>
       toLong(): Long
      </code>
     </li>
     <li>
      <code>
       toFloat(): Float
      </code>
     </li>
     <li>
      <code>
       toDouble(): Double
      </code>
     </li>
     <li>
      <code>
       toChar(): Char
      </code>
     </li>
    </ul>
    <p>
     但是在做计算的过程中，隐式转换又存在，比如：
    </p>
    <p>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         <strong>
          val
         </strong>
         l=
         <span style="color:#0000ff">
          1L
         </span>
         +
         <span style="color:#0000ff">
          3
         </span>
         <span style="color:gray">
          // Long + Int =&gt; Long
         </span>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
    </p>
    <h4 id="运算" style="margin-left:0; margin-right:0">
     运算
    </h4>
    <p>
    </p>
    <p>
     Kotlin支持数字运算的标准集，运算被定义为相应的类成员（但编译器会将函数调用优化为相应的指令）。 参见
     <a href="https://www.kotlincn.net/docs/reference/operator-overloading.html" rel="nofollow" title="运算符重载">
      运算符重载
     </a>
     。
    </p>
    <p>
     对于位运算，没有特殊字符来表示，而只可用中缀方式调用命名函数，例如:
    </p>
    <pre><code><strong>val</strong> x = (<span style="color:#0000ff">1</span> shl <span style="color:#0000ff">2</span>) and <span style="color:#0000ff">0x000FF000</span>
</code></pre>
    <p>
     这是完整的位运算列表（只用于
     <code>
      Int
     </code>
     和
     <code>
      Long
     </code>
     ）：
    </p>
    <ul style="margin-left:30px; margin-right:0">
     <li>
      <code>
       shl(bits)
      </code>
      – 有符号左移 (Java 的
      <code>
       &lt;&lt;
      </code>
      )
     </li>
     <li>
      <code>
       shr(bits)
      </code>
      – 有符号右移 (Java 的
      <code>
       &gt;&gt;
      </code>
      )
     </li>
     <li>
      <code>
       ushr(bits)
      </code>
      – 无符号右移 (Java 的
      <code>
       &gt;&gt;&gt;
      </code>
      )
     </li>
     <li>
      <code>
       and(bits)
      </code>
      – 位与
     </li>
     <li>
      <code>
       or(bits)
      </code>
      – 位或
     </li>
     <li>
      <code>
       xor(bits)
      </code>
      – 位异或
     </li>
     <li>
      <code>
       inv()
      </code>
      – 位非
     </li>
    </ul>
    <p>
    </p>
    <h4 id="浮点数比较" style="margin-left:0; margin-right:0">
     浮点数比较
    </h4>
    <p>
     相等和大小的 比较与java相同，Kotlin增加了一种区间实例与区间检测，如下：
    </p>
    <p>
    </p>
    <p>
    </p>
    <ul style="margin-left:30px; margin-right:0">
     <li>
      区间实例以及区间检测：
      <code>
       a..b
      </code>
      、
      <code>
       x in a..b
      </code>
      、
      <code>
       x !in a..b
      </code>
     </li>
    </ul>
    <p>
    </p>
    <p>
    </p>
    <h3 id="字符" style="margin-left:0; margin-right:0">
     字符
    </h3>
    <p>
     之前有提到过，字符不是数字
     <br/>
     字符用 Char 类型表示。它们不能直接当作数字
    </p>
    <p>
    </p>
    <p>
    </p>
    <p>
     fun check(c: Char) {
     <!-- -->
     <br/>
     if (c == 1) { // 错误：类型不兼容
     <br/>
     // ……
     <br/>
     }
     <br/>
     }
    </p>
    <p>
     <span style="color:#333333">
      字符字面值用单引号括起来:
     </span>
     <code>
      '1'
     </code>
     <span style="color:#333333">
      。 特殊字符可以用反斜杠转义。 支持这几个转义序列：
     </span>
     <code>
      \t
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \b
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \n
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \r
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \'
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \"
     </code>
     <span style="color:#333333">
      、
     </span>
     <code>
      \\
     </code>
     <span style="color:#333333">
      和
     </span>
     <code>
      \$
     </code>
     <span style="color:#333333">
      。 编码其他字符要用 Unicode 转义序列语法：
     </span>
     <code>
      '\uFF00'
     </code>
     <span style="color:#333333">
      。
     </span>
    </p>
    <p>
    </p>
    <h3 id="数组" style="margin-left:0; margin-right:0">
     数组
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#333333">
      数组在 Kotlin 中使用 Array 类来表示，它定义了 get 和 set 函数（按照运算符重载约定这会转变为 []）和 size 属性，以及一些其他有用的成员函数：
      <br/>
      class Array&lt;T&gt; private constructor() {
      <!-- -->
      <br/>
      val size: Int
      <br/>
      operator fun get(index: Int): T
      <br/>
      operator fun set(index: Int, value: T): Unit
      <br/>
      operator fun iterator(): Iterator&lt;T&gt;
      <br/>
      // ……
      <br/>
      }
     </span>
    </p>
    <p>
     <span style="color:#333333">
      <span style="color:#333333">
       <span style="color:#333333">
        <span style="color:#333333">
         我们可以使用库函数 arrayOf() 来创建一个数组并传递元素值给它，这样 arrayOf(1, 2, 3) 创建了 array [1, 2, 3]。 或者，库函数 arrayOfNulls() 可以用于创建一个指定大小的、所有元素都为空的数组。
         <br/>
         另一个选项是用接受数组大小和一个函数参数的工厂函数，用作参数的函数能够返回给定索引的每个元素初始值：
         <br/>
         // 创建一个 Array&lt;String&gt; 初始化为 ["0", "1", "4", "9", "16"]
         <br/>
         val asc = Array(5, { i -&gt; (i * i).toString() })
         <br/>
         如上所述，[] 运算符代表调用成员函数 get() 和 set()。
         <br/>
         注意: 与 Java 不同的是，Kotlin 中数组是不型变的（invariant）。这意味着 Kotlin 不让我们把 Array&lt;String&gt; 赋值给 Array&lt;Any&gt;，以防止可能的运行时失败（但是你可以使用 Array&lt;out Any&gt;, 参见类型投影）。
         <br/>
         Kotlin 也有无装箱开销的专门的类来表示原生类型数组: ByteArray、 ShortArray、IntArray 等等。这些类和 Array 并没有继承关系，但是它们有同样的方法属性集。它们也都有相应的工厂方法:
         <br/>
         val x: IntArray = intArrayOf(1, 2, 3)
         <br/>
         x[0] = x[1] + x[2]
        </span>
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="color:#333333">
      字符串
     </span>
    </p>
    <p>
    </p>
    <div>
     <span style="color:#333333">
      字符串除了具有java的特性意外，字符串的元素——字符可以使用索引运算符访问:
     </span>
     <code>
      s[i]
     </code>
     <span style="color:#333333">
      。 可以用

    <h3 id="when-表达式" style="margin-left:0; margin-right:0">

    </h3>
    <h3 id="when-表达式" style="margin-left:0; margin-right:0">
     when 取代了类 C 语言的 switch 操作符。其最简单的形式如下：
    </h3>
    <div>
     <strong>
      when
     </strong>
     <span style="background-color:#f7f7f7">
      (
     </span>
     x
     <span style="background-color:#f7f7f7">
      )
     </span>
     <span style="background-color:#f7f7f7">
      {
      <!-- -->
     </span>
    </div>
    <div>
     <span style="color:#0000ff">
      1
     </span>
     -&gt;print
     <span style="background-color:#f7f7f7">
      (
     </span>
     <span style="color:#009c00">
      "x == 1"
     </span>
     <span style="background-color:#f7f7f7">
      )
     </span>
    </div>
    <div>
     <span style="color:#0000ff">
      2
     </span>
     -&gt;print
     <span style="background-color:#f7f7f7">
      (
     </span>
     <span style="color:#009c00">
      "x == 2"
     </span>
     <span style="background-color:#f7f7f7">
      )
     </span>
    </div>
    <div>
     <strong>
      else
     </strong>
     -&gt;
     <span style="background-color:#f7f7f7">
      {
      <!-- -->
     </span>
     <span style="color:gray">
      // 注意这个块
     </span>
    </div>
    <div>
     print
     <span style="background-color:#f7f7f7">
      (
     </span>
     <span style="color:#009c00">
      "x is neither 1 nor 2"
     </span>
     <span style="background-color:#f7f7f7">
      )
     </span>
    </div>
    <div>
     <span style="background-color:#f7f7f7">
      }
     </span>
    </div>
    <div>
     <span style="background-color:#f7f7f7">
      }
     </span>
    </div>
    <p>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#0000c0">
      when
     </span>
     将它的参数和所有的分支条件顺序比较，直到某个分支满足条件。
     <span style="color:#0000c0">
      when
     </span>
     既可以被当做表达式使用也可以被当做语句使用。如果它被当做表达式， 符合条件的分支的值就是整个表达式的值，如果当做语句使用， 则忽略个别分支的值。（像
     <span style="color:#0000c0">
      if
     </span>
     一样，每一个分支可以是一个代码块，它的值是块中最后的表达式的值。）
    </p>
    <p>
     如果其他分支都不满足条件将会求值
     <span style="color:#0000c0">
      else
     </span>
     分支。 如果
     <span style="color:#0000c0">
      when
     </span>
     作为一个表达式使用，则必须有
     <span style="color:#0000c0">
      else
     </span>
     分支， 除非编译器能够检测出所有的可能情况 都已经覆盖了。
    </p>
    <p>
     <span style="color:#333333">
      如果很多分支需要用相同的方式处理，则可以把多个分支条件放在一起，用逗号分隔：
     </span>
    </p>
    <p>
