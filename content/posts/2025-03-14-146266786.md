    <h2>
     <a id="_2">
     </a>
     最好用的方法！
    </h2>
    <p>
     <strong>
      使用 nb_conda_kernels 添加所有环境
     </strong>
    </p>
    <p>
     第二种方法其实也挺不错的。有个缺点是，你新建一个环境，就要重复操作一次。
     <br/>
     而这个方法就是一键添加所有 conda 环境，且不妙哉！
    </p>
    <pre><code>conda activate my-conda-env    # this is the environment for your project and code
conda install ipykernel
conda deactivate

conda activate base      # could be also some other environment
conda install nb_conda_kernels
jupyter notebook
</code></pre>
    <p>
     推荐指数： ⭐️⭐️⭐️⭐️⭐️
    </p>
    <p>
     注意：这里的 conda install nb_conda_kernels 是在 base 环境下操作的。
    </p>
    <p>
     安装好后，打开 jupyter notebook 就会显示所有的 conda 环境啦，点击随意切换。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d2b669f3d74c4a308f1068c4dcbd3635.png"/>
    </p>
    <h2>
     <a id="2_26">
     </a>
     2、每次如何使用
    </h2>
    <p>
     首先输入jupyter book，然后下面会出现很多行，复制里面出现的带有http:的url，
     <br/>
     然后点击添加kernel，选择“现有Jupyter服务器”，
     <br/>
     然后输入密码，
     <br/>
     接下来就可以选择你想运行这个.ipynb的多个环境了。
    </p>
