     用户激励体系：等级体系应如何设计？

    <p style="margin-left:0cm;">
     <span style="color:#999999;">
      “
     </span>
     <span style="color:#999999;">
      用户激励体系到底应该怎么落地，应该怎么执行？这篇文章，想和大家聊聊用户激励体系的
     </span>
     <span style="color:#999999;">
      “
     </span>
     <span style="color:#999999;">
      术
     </span>
     <span style="color:#999999;">
      ”
     </span>
     <span style="color:#999999;">
      。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/3e9627425fc03e9f5130551335625696.jpeg" class="has" height="360" src="https://i-blog.csdnimg.cn/blog_migrate/3e9627425fc03e9f5130551335625696.jpeg" width="640"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      一、用户激励体系的原因及整体思路是啥？
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      设计激励体系，一方面是为了提升用户活跃，增加留存，粘住用户，另一方面也是为了度量每个用户在产品中的成长过程和价值，便于分群运营。当然，也有个大前提，就是企业愿意付出较低的成本来激励用户。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      影响用户的行为模型
     </span>
     <span style="color:#333333;">
      B=MAT
     </span>
     <span style="color:#333333;">
      ，让用户完成设定期望动作，需要满足三个条件：充分的动机；完成这一行为的能力；促使人们付诸行动的触发。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      而决定用户动机主要有
     </span>
     <span style="color:#333333;">
      8
     </span>
     <span style="color:#333333;">
      个核心的驱动力：使命感、发展及成就感、创造力的发挥、占有欲、社交影响及联系、稀缺性、未知及好奇、恐惧及失去。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      国内的互联网产品在搭建用户激励体系时，基本思路都是一致的，根据影响用户行为的因素，来设计等级体系、积分体系、徽章体系、会员体系等等。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/118b520a532cd09999fc87c4272130c0.jpeg" class="has" height="370" src="https://i-blog.csdnimg.cn/blog_migrate/118b520a532cd09999fc87c4272130c0.jpeg" width="640"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      接下来，我会分篇讲一下用户成长体系的具体落地设计，这里主要讲一下用户等级体系。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      二、等级体系怎么创建？
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      相信绝大多数人拥有过这样一种经历：为了提高自己的
     </span>
     <span style="color:#333333;">
      QQ
     </span>
     <span style="color:#333333;">
      号等级，每天打开电脑第一件事儿就是挂上
     </span>
     <span style="color:#333333;">
      QQ
     </span>
     <span style="color:#333333;">
      号，看到自己
     </span>
     <span style="color:#333333;">
      QQ
     </span>
     <span style="color:#333333;">
      号等级不断上升，从星星变成月亮、从月亮变成太阳，内心充满了成就感，一定要跟小伙伴炫耀比拼一下，这应该是绝大多数人体验的第一个互联网产品的等级体系。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      现在大火的王者荣耀设置了青铜、白银、黄金、铂金、钻石、星耀及王者荣耀几个段位。玩家为了升段位，沉迷于游戏不能自拔。等级体系的设计正是抓住了用户获得成就的满足感、喜欢炫耀等心理。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      具体来讲，等级体系主要包括三个方面：定义影响等级的行为、等级成长值计算、不同等级对应的权益。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#000000;">
       1.
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       定义影响等级的行为
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      影响用户等级的行主要可分为：
     </span>
     <span style="color:#333333;">
      “
     </span>
     <span style="color:#333333;">
      主动行为
     </span>
     <span style="color:#333333;">
      ”
     </span>
     <span style="color:#333333;">
      和
     </span>
     <span style="color:#333333;">
      “
     </span>
     <span style="color:#333333;">
      被动行为
     </span>
     <span style="color:#333333;">
      ”
     </span>
     <span style="color:#333333;">
      。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#333333;">
       主动行为：
      </span>
     </strong>
     <span style="color:#333333;">
      一般指用户主动付出的行为，以汽车之家为例：可以是完善个人信息、绑定手机号、添加车辆、添加行驶证、发表评论、浏览文章、直播充值、直播打赏等。
      <strong>
       被动行为：
      </strong>
      一般指的是其他人对其的认可行为，以汽车之家为例：这些行为可以是获得点赞、回答问题被评选优质解答、发表口碑被评为精华口碑、游记被推荐到每日推荐等。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      一般的用户等级体系都会对用户行为进行量化，对产品贡献越大的行为，分值越大，没有固定方法论，具体分值自行设置，并且根据实际情况调整。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      如马蜂窝积累方式如下：
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/d7aae78ba4c21c53697d4109bdc15e63.jpeg" class="has" height="466" src="https://i-blog.csdnimg.cn/blog_migrate/d7aae78ba4c21c53697d4109bdc15e63.jpeg" width="640"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      当然有些等级体系并没有对应的量化数值，比如汽车之家问答产品中的
     </span>
     <span style="color:#333333;">
      KOL
     </span>
     <span style="color:#333333;">
      等级体系：
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/eb5b9031a5531be012a5fd7cab2ca39a.jpeg" class="has" height="255" src="https://i-blog.csdnimg.cn/blog_migrate/eb5b9031a5531be012a5fd7cab2ca39a.jpeg" width="640"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#000000;">
       2.
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       等级成长值的计算
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#333333;">
       （
      </span>
     </strong>
     <strong>
      <span style="color:#333333;">
       1
      </span>
     </strong>
     <strong>
      <span style="color:#333333;">
       ）计算函数类型
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      等级成长值计算一般会有几种比较通用的计算方式：幂函数、指数函数及分段函数。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      对于想要设置更多的用户等级的产品，大多采用幂函数。通过拿到马蜂窝及汽车之家的等级规则，并做一下简单的回归分析，可以看到其等级成长值计算类似于幂函数。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/f76aad52c97c4ab707f250b7c0e5d70b.jpeg" class="has" height="216" src="https://i-blog.csdnimg.cn/blog_migrate/f76aad52c97c4ab707f250b7c0e5d70b.jpeg" width="364"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/dacf3f809e3f7b577957adccdcc5fb74.jpeg" class="has" height="218" src="https://i-blog.csdnimg.cn/blog_migrate/dacf3f809e3f7b577957adccdcc5fb74.jpeg" width="362"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#333333;">
       （
      </span>
     </strong>
     <strong>
      <span style="color:#333333;">
       2
      </span>
     </strong>
     <strong>
      <span style="color:#333333;">
       ）成长值计算逻辑
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      确定产品类型，选择计算成长值的函数类型
     </span>
     <span style="color:#333333;">
      F
     </span>
     <span style="color:#333333;">
      （指数函数、幂函数或分段函数）；根据产品生命周期，自定义用户成长到最高级所用时间
     </span>
     <span style="color:#333333;">
      Tmax
     </span>
     <span style="color:#333333;">
      ；根据前面
     </span>
     <span style="color:#333333;">
      “
     </span>
     <span style="color:#333333;">
      定义影响等级的行为
     </span>
     <span style="color:#333333;">
      ”
     </span>
     <span style="color:#333333;">
      ，计算出用户每日可获得最大分值
     </span>
     <span style="color:#333333;">
      Smax
     </span>
     <span style="color:#333333;">
      ；根据
     </span>
     <span style="color:#333333;">
      b&amp;c
     </span>
     <span style="color:#333333;">
      ，计算出用户在该产品中整个生命周期中一共可积累的最大分值
     </span>
     <span style="color:#333333;">
      Mmax =Tmax * Smax
     </span>
     <span style="color:#333333;">
      ；根据最大分值
     </span>
     <span style="color:#333333;">
      Mmax
     </span>
     <span style="color:#333333;">
      和函数
     </span>
     <span style="color:#333333;">
      F
     </span>
     <span style="color:#333333;">
      ，计算出用户可达到的最大等级
     </span>
     <span style="color:#333333;">
      Dmax
     </span>
     <span style="color:#333333;">
      ，于是可以得出用户成长的等级区间
     </span>
     <span style="color:#333333;">
      [1,Dmax]
     </span>
     <span style="color:#333333;">
      ；根据等级区间
     </span>
     <span style="color:#333333;">
      [1,Dmax]
     </span>
     <span style="color:#333333;">
      ，计算出每个等级对应的成长值。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      按照上面的步骤，用户等级成长值基本的
     </span>
     <span style="color:#333333;">
      demo
     </span>
     <span style="color:#333333;">
      就可以制作出来了
     </span>
     <span style="color:#333333;">
      !
     </span>
    </p>
    <p style="margin-left:0cm;">
     <strong>
      <span style="color:#000000;">
       3.
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       不同等级对应的权益
      </span>
     </strong>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      为了给用户足够的激励，不同的等级体系，一般对应着不同的特权，而具体的特权需要根据不同的产品与业务来制定。常规来讲，对应的特权一般会从物质及精神两个层面进行考虑，大家可以参看激励用户行为的
     </span>
     <span style="color:#333333;">
      8
     </span>
     <span style="color:#333333;">
      个核心驱动力，对具体设计会带来有一定的理论依据及启发。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      举个例子，网易云音乐的不同等级的用户，会享受不同的特权：
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#000000;">
      <img alt="https://i-blog.csdnimg.cn/blog_migrate/8a3c125d982736259336376b97c88456.jpeg" class="has" height="412" src="https://i-blog.csdnimg.cn/blog_migrate/8a3c125d982736259336376b97c88456.jpeg" width="640"/>
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      目前，越来越多的互联网公司开始设置用户等级来激励用户，淘宝、京东等开始用大数据建模的方式来设计等级体系，避免了人工大量的计算，模型也更加完美，有兴趣的同学可以仔细研究一下。
     </span>
    </p>
    <p style="margin-left:0cm;">
     <span style="color:#333333;">
      这里主要是讲了设计用户等级大概的一个设计思路，要做出完美的方案，还是需要深入研究产品及业务，不断迭代优化的。
     </span>
    </p>
