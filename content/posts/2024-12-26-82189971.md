    <h2 id="如果有能力支付正版请购买正版">
     如果有能力支付正版，请购买正版
    </h2>
    <hr/>
    <p>
     目前最新的IDEA版本
     <code>
      2018.2
     </code>
     已经不支持该破解服务器，如果想继续使用，需要下载之前的版本
    </p>
    <hr/>
    <p>
     搭建的工具以及方法
     <a href="https://github.com/a252937166/new-idea-server">
      GitHub
     </a>
     上有。这里只是在linux上搭建，不过由于是用别人已经写好的软件，实际上只是下载运行一下就可以了。
     <br/>
     如果你拥有一台云主机，你就可以自己在Linux上搭建一个IDEA授权服务器，不用每一次都在本地开。
     <br/>
     下载Linux对应的文件
     <code>
      IntelliJIDEALicenseServer_linux_amd64
     </code>
     ,以及对应的启动脚本和终止脚本，当然你自己直接启动不借助脚本也是可以的，
     <strong>
      注意
     </strong>
     要给每一个文件
     <strong>
      执行权限
     </strong>
     <code>
      chmod +x 文件名
     </code>
     .
    </p>
    <hr/>
    <p>
     下面附上Linux所需工具的下载地址，博主放在自己的主机上，速度应该会比百度云快一点，。，
     <br/>
     <a href="https://madongyu.ml/download/IntelliJIDEALicenseServer_linux_amd64" rel="nofollow">
      IntelliJIDEALicenseServer_linux_amd64
     </a>
     <br/>
     <a href="https://madongyu.ml/download/start.sh" rel="nofollow">
      start.sh
     </a>
     <br/>
     <a href="https://madongyu.ml/download/stop.sh" rel="nofollow">
      stop.sh
     </a>
    </p>
    <hr/>
    <p>
     启动成功后可以查看当前目录（即自己下载完后文件所在的目录）下的info.log这个文件的日志报告，查看是否启动成功。
     <br/>
     在当前目录下新建
     <code>
      IntelliJIDEALicenseServer.html
     </code>
     ，可以定义自己在1027端口下的默认页面。。。
    </p>
    <hr/>
    <p>
     最后附上博主已经搭好的服务器

      https://madongyu.ml
     </a>
    </p>
