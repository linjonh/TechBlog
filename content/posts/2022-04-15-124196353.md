# [Win 10]Python降版本方法、Python3.9降至3.7

Win 10
  
环境：Pycharm、anaconda、python3.9
  
一、打开cmd
  
二、执行下列命令
  
conda install python==3.7
  
三、查看python版本，是否降版本成功
  
python --version(两个短横） 或 python -V
