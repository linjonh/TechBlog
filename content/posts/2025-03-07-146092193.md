    <h3>
     一、NBMA网络
    </h3>
    <h4>
     <strong>
      NBMA网络的特点
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        连接方式：
       </strong>
      </p>
      <ol>
       <li>
        <p>
         支持多台设备连接到同一个网络段，但网络本身
         <strong>
          不支持广播或组播
         </strong>
         。典型例子：帧中继、ATM。
        </p>
       </li>
      </ol>
     </li>
     <li>
      <p>
       <strong>
        DR/BDR选举
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         由于网络不支持广播，OSPF需要
         <strong>
          手动配置邻居
         </strong>
         。
        </p>
       </li>
       <li>
        <p>
         仍然会选举DR（指定路由器）和BDR（备份指定路由器），以减少LSA泛洪。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        邻居发现
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         路由器无法通过广播自动发现邻居，必须
         <strong>
          手动配置邻居的IP地址
         </strong>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        LSA泛洪
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         DR和BDR负责与其他路由器建立邻接关系，并代表网络发送LSA。
        </p>
       </li>
       <li>
        <p>
         非DR/BDR的路由器（DROTHER）只与DR和BDR建立邻接关系。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        链路开销
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         OSPF根据链路带宽计算开销（Cost），NBMA网络的链路开销通常较高。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     二、P2MP
    </h3>
    <h4>
     <strong>
      P2MP网络特点
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        网络类型
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         支持一台设备与多台设备连接，但
         <strong>
          不支持广播
         </strong>
         。
        </p>
       </li>
       <li>
        <p>
         典型场景：部分帧中继配置、无线网络。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        DR/BDR选举
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          无需DR/BDR选举
         </strong>
         ，所有路由器直接与中心路由器建立邻接关系。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        邻居发现
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         路由器通过Hello报文
         <strong>
          自动发现邻居
         </strong>
         ，无需手动配置。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        Hello报文
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         Hello间隔通常为30秒。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        LSA泛洪
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         LSA直接在所有路由器之间泛洪，无需DR/BDR。
        </p>
       </li>
      </ul>
     </li>
    </ol>

    <h4>
     IP及OSPF配置
    </h4>
    <blockquote>
     <p>
      [R1]int s1/0/0
      <br/>
      [R1-Serial1/0/0]link-protocol fr    //二层封装协议为FR
      <br/>
      [R1-Serial1/0/0]fr map ip 10.1.1.2 102 broadcast   //去10.1.1.2打上102的标记然后广播
      <br/>
      [R1-Serial1/0/0]fr map ip 10.1.1.3 103 broadcast
      <br/>
      [R1-Serial1/0/0]ip add 10.1.1.1 24
      <br/>
      [R1]int LoopBack 0
      <br/>
      [R1-LoopBack0]ip add 1.1.1.1 24
      <br/>
      [R1]ospf router-id 1.1.1.1
      <br/>
      [R1-ospf-1]area 0
      <br/>
      [R1-ospf-1-area-0.0.0.0]network 10.1.1.0 0.0.0.255
      <br/>
      [R1-ospf-1-area-0.0.0.0]network 1.1.1.0 0.0.0.255
     </p>
     <p>
     </p>
     <p>
      [R2]int s1/0/0
      <br/>
      [R2-Serial1/0/0]link-protocol fr
      <br/>
      [R2-Serial1/0/0]fr map ip 10.1.1.1 201 broadcast
      <br/>
      [R2-Serial1/0/0]ip add 10.1.1.2 24
      <br/>
      [R2]int LoopBack 0
      <br/>
      [R2-LoopBack0]ip add 2.2.2.2 24
      <br/>
      [R2]ospf router-id 2.2.2.2
      <br/>
      [R2-ospf-1]area 0
      <br/>
      [R2-ospf-1-area-0.0.0.0]network 10.1.1.0 0.0.0.255
      <br/>
      [R2-ospf-1-area-0.0.0.0]network 2.2.2.0 0.0.0.255
     </p>
     <p>
     </p>
     <p>
      [R3]int s1/0/0
      <br/>
      [R3-Serial1/0/0]link-protocol fr
      <br/>
      [R3-Serial1/0/0]fr map ip 10.1.1.1 301 broadcast
      <br/>
      [R3-Serial1/0/0]ip add 10.1.1.3 24
      <br/>
      [R3]int LoopBack 0
      <br/>
      [R3-LoopBack0]ip add 3.3.3.3 24
      <br/>
      [R3]ospf router-id 3.3.3.3
      <br/>
      [R3-ospf-1]area 0
      <br/>
      [R3-ospf-1-area-0.0.0.0]network 10.1.1.0 0.0.0.255
      <br/>
      [R3-ospf-1-area-0.0.0.0]network 3.3.3.0 0.0.0.255
     </p>
    </blockquote>
    <h4>
     配置完在R1上查看ospf的邻接状态，可以看到R1没有邻接关系
    </h4>
    <blockquote>
     <p>
      [R1]dis ospf peer brief
     </p>
     <p>
      OSPF Process 1 with Router ID 1.1.1.1
      <br/>
      Peer Statistic Information

      ----------------------------------------------------------------------------
      <br/>
      Area Id          Interface                        Neighbor id      State
      <br/>
      ----------------------------------------------------------------------------
     </p>
    </blockquote>
    <h4>
     再去查看一下网络类型，网络类型为NBMA
    </h4>
    <blockquote>
     <p>
      [R1]dis ospf int s1/0/0
     </p>
     <p>
      OSPF Process 1 with Router ID 1.1.1.1
      <br/>
      Interfaces
     </p>
     <p>
      <br/>
      Interface: 10.1.1.1 (Serial1/0/0)
      <br/>
      Cost: 48      State: DR        Type: NBMA      MTU: 1500
      <br/>
      Priority: 1
      <br/>
      Designated Router: 10.1.1.1
      <br/>
      Backup Designated Router: 0.0.0.0
      <br/>
      Timers: Hello 30 , Dead 120 , Poll  120 , Retransmit 5 , Transmit Delay 1
     </p>
    </blockquote>
    <h4>
     接下来建立邻居
    </h4>
    <blockquote>
     <p>
      [R1]ospf
      <br/>
      [R1-ospf-1]peer 10.1.1.2   //和10.1.1.2单播建立邻居
      <br/>
      [R1-ospf-1]peer 10.1.1.3
      <br/>
      [R2]ospf
      <br/>
      [R2-ospf-1]peer 10.1.1.1
     </p>
     <p>
      [R3]ospf
      <br/>
      [R3-ospf-1]peer 10.1.1.1
     </p>
    </blockquote>
    <h4>
     然后将R1配置为DR，不选举BDR
    </h4>
    <blockquote>
     <p>
      [R2]int s1/0/0
      <br/>
      [R2-Serial1/0/0]ospf dr-priority 0
      <br/>
      [R3]int s1/0/0
      <br/>
      [R3-Serial1/0/0]ospf dr-priority 0
     </p>
    </blockquote>
    <h4>
     然后在R1上查看邻接关系
    </h4>
    <blockquote>
     <p>
      [R1]dis ospf peer brief
     </p>
     <p>
      OSPF Process 1 with Router ID 1.1.1.1
      <br/>
      Peer Statistic Information
      <br/>
      ----------------------------------------------------------------------------
      <br/>
      Area Id          Interface                        Neighbor id      State

      0.0.0.0          Serial1/0/0                      3.3.3.3          Full
      <br/>
      ----------------------------------------------------------------------------
     </p>
    </blockquote>
    <h4>
     接下来要将NBMA改为P2MP
    </h4>
    <h4>
     将刚才建立的邻居删除
    </h4>
    <blockquote>
     <p>
      [R1]ospf
      <br/>
      [R1-ospf-1]undo peer 10.1.1.2
      <br/>
      [R1-ospf-1]undo peer 10.1.1.3
     </p>
     <p>
      [R2]ospf
      <br/>
      [R2-ospf-1]undo peer 10.1.1.1
     </p>
     <p>
      [R3]ospf
      <br/>
      [R3-ospf-1]undo peer 10.1.1.1
     </p>
    </blockquote>
    <h4>
     然后去查看R1的邻接关系
    </h4>
    <blockquote>
     <p>
      [R1]dis ospf peer brief
     </p>
     <p>
      OSPF Process 1 with Router ID 1.1.1.1
      <br/>
      Peer Statistic Information
      <br/>
      ----------------------------------------------------------------------------
      <br/>
      Area Id          Interface                        Neighbor id      State
      <br/>
      ----------------------------------------------------------------------------
     </p>
    </blockquote>
    <h4>
     接下来将网络的类型改为P2MP
    </h4>
    <blockquote>
     <p>
      [R1]int s1/0/0
      <br/>
      [R1-Serial1/0/0]ospf network-type p2mp   //将网络类型改成p2mp
     </p>
     <p>
      [R2]int s1/0/0
      <br/>
      [R2-Serial1/0/0]ospf network-type p2mp
     </p>
     <p>
      [R3]int s1/0/0
      <br/>
      [R3-Serial1/0/0]ospf network-type p2mp
     </p>
    </blockquote>
