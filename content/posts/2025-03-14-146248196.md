    <p>
     这段代码是一个基于 C++ 的均匀 B 样条（Uniform B-spline）实现，专门用于表示 SE(3) 变换（即三维空间中的刚体变换，包括旋转和平移）。以下是对代码的总结：
    </p>

    <h4>
     <a id="2__7">
     </a>
     2.
     <strong>
      功能概述
     </strong>
    </h4>
    <ul>
     <li>
      文件定义了一个模板类
      <code>
       Se3Spline
      </code>
      ，用于在 SE(3) 空间中构造和操作均匀 B 样条。
     </li>
     <li>
      SE(3) 表示三维空间的刚体变换，内部通过分离表示法实现：
      <ul>
       <li>
        <strong>
         旋转部分
        </strong>
        ：使用 SO(3) 样条（
        <code>
         So3Spline
        </code>
        ）表示。
       </li>
       <li>
        <strong>
         平移部分
        </strong>
        ：使用三维欧几里得样条（
        <code>
         RdSpline
        </code>
        ）表示。
       </li>
      </ul>
     </li>
     <li>
      支持 N 阶样条（
      <code>
       N
      </code>
      为模板参数），阶数和次数（degree）分别为
      <code>
       N
      </code>
      和
      <code>
       N-1
      </code>
      。
     </li>
     <li>
      参考文献：
      <a href="https://arxiv.org/abs/1911.08860" rel="nofollow">
       arXiv:1911.08860
      </a>
      。
     </li>
    </ul>
    <h4>
     <a id="3__15">
     </a>
     3.
     <strong>
      主要特性
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       时间参数化
      </strong>
      ：样条基于均匀时间间隔（
      <code>
       time_interval_ns
      </code>
      ），支持纳秒级时间戳。
     </li>
     <li>
      <strong>
       控制点（Knots）
      </strong>
      ：存储 SE(3) 变换的控制点，支持动态添加、删除和修改。
     </li>
     <li>
      <strong>
       NURBS 支持
      </strong>
      ：增加了非均匀有理 B 样条（NURBS）的功��，通过
      <code>
       knts
      </code>
      （knot vector）和混合矩阵（
      <code>
       blending_mats
      </code>
      、
      <code>
       cumu_blending_mats
      </code>
      ）实现。
     </li>
     <li>
      <strong>
       评估功能
      </strong>
      ：
      <ul>
       <li>
        计算指定时间的位姿（
        <code>
         poseNs
        </code>
        ）、位置（
        <code>
         positionWorld
        </code>
        ）、速度（
        <code>
         transVelWorld
        </code>
        ）、加速度（
        <code>
         transAccelWorld
        </code>
        ）等。
       </li>
       <li>
        支持 NURBS 版本的评估（如
        <code>
         poseNsNURBS
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       雅可比矩阵
      </strong>
      ：支持计算位姿对控制点的导数（Jacobian），用于优化或分析。
     </li>
     <li>
      <strong>
       随机轨迹生成
      </strong>
      ：提供生成随机轨迹的功能（
      <code>
       genRandomTrajectory
      </code>
      ）。
     </li>
    </ul>
    <h4>
     <a id="4__25">
     </a>
     4.
     <strong>
      核心方法
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       构造函数
      </strong>
      ：初始化样条，指定时间间隔和起始时间。
     </li>
     <li>
      <strong>
       控制点操作
      </strong>
      ：
      <ul>
       <li>
        <code>
         setKnot
        </code>
        ：设置特定控制点的 SE(3) 位姿。
       </li>
       <li>
        <code>
         knots_push_back
        </code>
        /
        <code>
         knots_pop_back
        </code>
        ：添加或移除控制点。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       时间管理
      </strong>
      ：
      <ul>
       <li>
        <code>
         minTimeNs
        </code>
        /
        <code>
         maxTimeNs
        </code>
        ：获取样条的时间范围。
       </li>
       <li>
        <code>
         extendKnotsTo
        </code>
        ：扩展控制点到指定时间。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       调试工具
      </strong>
      ：
      <code>
       print_knots
      </code>
      用于输出控制点信息。
     </li>
     <li>
      <strong>
       混合矩阵
      </strong>
      ：
      <code>
       InitBlendMat
      </code>
      和
      <code>
       AddBlendMat
      </code>
      用于计算 NURBS 的混合矩阵。
     </li>
    </ul>
    <h4>
     <a id="5__36">
     </a>
     5.
     <strong>
      数据结构
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       模板参数
      </strong>
      ：
      <ul>
       <li>
        <code>
         _N
        </code>
        ：样条阶数。
       </li>
       <li>
        <code>
         _Scalar
        </code>
        ：数据类型（如
        <code>
         double
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       内部成员
      </strong>
      ：
      <ul>
       <li>
        <code>
         pos_spline
        </code>
        ：平移部分的样条（
        <code>
         RdSpline
        </code>
        ）。
       </li>
       <li>
        <code>
         so3_spline
        </code>
        ：旋转部分的样条（
        <code>
         So3Spline
        </code>
        ）。
       </li>
       <li>
        <code>
         knts
        </code>
        ：NURBS 的节点向量。
       </li>
       <li>
        <code>
         blending_mats
        </code>
        /
        <code>
         cumu_blending_mats
        </code>
        ：混合矩阵及其累积形式。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       数学类型
      </strong>
      ：使用 Eigen 库定义向量和矩阵（如
      <code>
       Vec3
      </code>
      、
      <code>
       Mat3
      </code>
      ），以及 Sophus 库的
      <code>
       SO3
      </code>
      和
      <code>
       SE3
      </code>
      。
     </li>
    </ul>

     </strong>
    </h4>

    <h4>

     7.
     <strong>
      NURBS扩展实现
     </strong>
     ：
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        核心改动
       </strong>
       ：
      </p>
      <ul>
       <li>
        引入非均匀节点向量
        <code>
         knts
        </code>
        存储时间戳（纳秒），替代原有均匀间隔
        <code>
         dt_ns_
        </code>
       </li>
       <li>
        添加混合矩阵
        <code>
         blending_mats
        </code>
        和累积混合矩阵
        <code>
         cumu_blending_mats
        </code>
        用于NURBS基函数计算
       </li>
       <li>
        新增
        <code>
         poseNsNURBS
        </code>
        系列方法实现非均匀时间插值
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        关键实现细节
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         节点查询
        </strong>
        ：通过
        <code>
         GetCtrlIndexNURBS
        </code>
        和
        <code>
         GetIdxT
        </code>
        在
        <code>
         knts
        </code>
        中二分查找时间区间
       </li>
       <li>
        <strong>
         混合矩阵生成
        </strong>
        ：
        <ul>
         <li>
          <code>
           InitBlendMat
          </code>
          和
          <code>
           AddBlendMatForGen
          </code>
          根据节点间隔动态计算4x4混合矩阵
         </li>
         <li>
          矩阵元素基于NURBS递推公式，考虑相邻节点的时间差
         </li>
         <li>
          累积混合矩阵通过行累加简化后续计算
         </li>
        </ul>
       </li>
       <li>
        <strong>
         特殊处理
        </strong>
        ：
        <ul>
         <li>
          平移和旋转分别调用
          <code>
           RdSpline::evaluateNURBS
          </code>
          和
          <code>
           So3Spline::evaluateNURBS
          </code>
         </li>
         <li>
          速度计算需传入delta_t和混合矩阵以处理非均匀间隔
         </li>
        </ul>
       </li>
      </ul>
     </li>
