     二、大数据的简单介绍
    </h2>
    <h3>
     <a id="21__35">
     </a>
     2.1 大数据的定义
    </h3>
    <p>
     大数据（Big Data）：指无法在一定时间范围内用常规软件工具进行捕捉、管理和处理的数据集合，是需要新处理模式才能具有更强的决策力、洞察发现力和流程优化能力的海量、高增长率和多样化的信息资产。
    </p>
    <h3>
     <a id="22_5V_37">
     </a>
     2.2 大数据的特点(5V)
    </h3>
    <p>
     <img alt="9" src="https://i-blog.csdnimg.cn/blog_migrate/d3163f345bdc31a1ad1c7e8d1d74ebab.png#pic_center"/>
    </p>
    <ul>
     <li class="task-list-item">
      <input checked="true" class="task-list-item-checkbox" disabled="disabled" type="checkbox"/>
      1、大量
      <br/>
      采集、存储和计算的数据量都非常大。
      <br/>
      <br/>
      随着科学技术的发展和互联网的普及，全球的数据量已经大到爆了，而传统的关系型数据库根本处理不了如此海量的数据。
      <br/>
      <br/>
      截至目前，人类生产的所有印刷材料的数据量是200PB，而历史上全人类总共说过的话的数据量大约是5EB。当前，典型个人计算机硬盘的容量为TB量级，而一些大企业的数据量已经接近EB量级。
     </li>
     <li class="task-list-item">
      <input checked="true" class="task-list-item-checkbox" disabled="disabled" type="checkbox"/>
      2、高速
      <br/>
      根据IDC的“数字宇宙”的报告，预计到2025年，全球数据使用量将达到163ZB。在如此海量的数据面前，处理数据的效率就是企业的生命。
      <br/>
      在大数据时代，数据的创建、存储、分析都要求被高速处理，比如电商网站的个性化推荐尽可能要求实时完成推荐，这也是大数据区别于传统数据挖掘的显著特征。
     </li>
     <li class="task-list-item">
      <input checked="true" class="task-list-item-checkbox" disabled="disabled" type="checkbox"/>
      3、多样
      <br/>
      数据形式和来源多样化，包括结构化、半结构化和非结构化数据，具体表现为网络日志、音频、视频、图片、地理位置信息等，多类型的数据对数据的处理能力提出了更高的要求。其数据类型按照分类可划分为：
      <br/>
      <br/>
      ①结构化数据：
      <br/>
      Excel文件、csv文件等。
      <br/>
      结构化数据为表格形式的数据：
      <strong>
       每列数据类型相同，切不可再细分。
      </strong>
      <br/>
      ②半结构化数据：
      <br/>
      邮件、网页、JSON文件、日志文件等。
      <br/>
      <strong>
       这些数据的结构和内容混在一起，没有明显的区分。
      </strong>
      <br/>
      ③非结构化数据：
      <br/>
      图片、视频等
     </li>
     <li class="task-list-item">
      <input checked="true" class="task-list-item-checkbox" disabled="disabled" type="checkbox"/>
      4、真实
      <br/>
      确保数据的真实性，才能保证数据分析的正确性。
     </li>
     <li class="task-list-item">
      <input checked="true" class="task-list-item-checkbox" disabled="disabled" type="checkbox"/>
      5、低价值
      <br/>
      数据价值密度相对较低，或者说是浪里淘沙却又弥足珍贵 。
      <br/>
      <br/>
      互联网发展催生了大量数据，信息海量，但价值密度较低，如何结合业务逻辑并通过强大的机器算法来挖掘数据价值，是大数据时代最需要解决的问题，也是一个有难度的课题。
     </li>
    </ul>

     2.8.1 数据采集
    </h4>
    <p>
     <strong>

     </strong>
    </p>
    <p>
     下面为一些常见的数据采集工具：
    </p>
    <p>
     1️⃣
     <strong>
      普通
     </strong>
     <br/>
     1、
     <strong>
      python网络爬虫
     </strong>
     ：
    </p>
    <p>
     Urllib：HTTP请求库，用于请求、下载网页
    </p>
    <p>
     Requests：基于Urllib，但更加快捷方便
    </p>
    <p>
     Scrapy：爬虫框架，用于从网页中提取数据
    </p>
    <p>
     2、
     <strong>
      Webmagic(Java)
     </strong>
     ：垂直爬虫框架，简单爬虫开发流程，专注于逻辑功能的开发。
    </p>
    <p>
     3、
     <strong>
      BotnetSpider(C#)
     </strong>
     ：国人开源的跨平台、高性能、轻量级爬虫软件，.NET最为优秀的开源爬虫之一
    </p>
    <p>
     2️⃣
     <strong>
      分布式采集工具
     </strong>
    </p>
    <p>
     1、
     <strong>
      Chukwa(hadoop)
     </strong>
     ：用于监控大型分布式系统的数据收集系统，构建在HDFS和Mapreduce之上。
    </p>
    <p>
     2、
     <strong>
      Kafka
     </strong>
     ：由Apache开源，高吞吐量的分布式发布订阅消息系统。
    </p>
    <p>
     3、
     <strong>
      Flume
     </strong>
     ：分布式的海量日志采集、集合和传输系统。
    </p>

     🔍
     <strong>
      数据存储：数据以某种格式记录在计算机内部或外部存储介质上。
     </strong>
    </p>
    <p>
     🔍
     <strong>
      数据存储不一定是以文件的形式存储，它可以是以数据库的形式进行存储，由数据库决定，怎样写入到文件中或者怎样保存在内存中。
     </strong>
    </p>
    <p>
     下列为常见的数据库系统：
     <br/>
     1️⃣
     <strong>
      关系型数据库(SQL)
     </strong>
     <br/>
     1、
     <strong>
      MySQL(小型数据库)
     </strong>
     ：最流行的关系型数据库管理系统之一，将数据保存于不同表中。
    </p>
    <p>
     2、
     <strong>
      Oracle(大型数据库)
     </strong>
     ：甲骨文公司数据库管理系统，高效率、可靠性高、适应高吞吐量。
    </p>
    <p>
     3、
     <strong>
      SQL Server
     </strong>
     ：Microsoft公司的数据库管理系统，使用方便、可伸缩性好、与相关软件集成程度高。
    </p>
    <p>
     2️⃣
     <strong>
      非关系型数据库(NOSQL）
     </strong>
     <br/>
     1、
     <strong>
      Hbase
     </strong>
     ：高可靠性、高性能、可伸缩的分布式存储系统，可在PC Server上搭建大规模结构化存储集群。
    </p>
    <p>
     2、
     <strong>
      MongoDB
     </strong>
     ：介于关系型数据库和非关系型数据库之间的产品，功能丰富；基于分布式文件存储数据库，由C++编写。
    </p>
    <p>
     3、
     <strong>
      Redis
     </strong>
     ：一个日志型、高性能Key-Value数据库，数据可以从主服务器向任意数量的从服务器同步。
    </p>
    <h4>
     <a id="283__142">
     </a>
     2.8.3 数据清洗
    </h4>
    <p>
     <strong>
      数据清洗：为了便于后续的处理和分析，对数据进行的质量诊断、数据整合、数据转换、缺失值处理和异常值处理等操作。
     </strong>
    </p>
    <p>
     🔍数据清洗的方法
    </p>
    <p>
     1、
     <strong>
      缺失值处理
     </strong>
     ：对存在缺失的数据进行插补
    </p>
    <p>
     2、
     <strong>
      异常值处理
     </strong>
     ：对数据集中存在的不合理值进行处理
    </p>
    <p>
     3、
     <strong>
      数据转换
     </strong>
     ：将数据从一种表现形式转换成另一种表现形式
    </p>
    <p>
     🔍数据清洗的工具
    </p>
    <p>
     1、
     <strong>
      Mapreduce(Hadoop)
     </strong>
     ：基于集群的高性能并行计算框架；并行计算与运行软件框架；并行程序设计模型与方法。
    </p>
    <p>
     2、
     <strong>
      Pandas(Python)
     </strong>
     ：解决数据分析任务的Python库，提供了诸多数据清洗的函数和方法。
    </p>
    <p>
     3、
     <strong>
      OpenRefine
     </strong>
     ：数据清洗工具，能够对数据进行可视化操作，类似Excel，但其工作方式更像数据库。
    </p>

     3.1 大数据的影响
    </h3>
    <p>
     图灵奖获得者、著名数据库专家Jim Gray 博士观察并总结人类自古以来，在科学研究上，先后历经了实验、理论、计算和数据四种范式
     <br/>
     <img alt="10" src="https://i-blog.csdnimg.cn/blog_migrate/ff48ab52adfbdeb972b7a01ce371a375.png#pic_center"/>
     <br/>
     在思维方式方面，大数据完全颠覆了传统的思维方式：
     <br/>
     👋全样而非抽样
     <br/>
     👋效率而非精确
     <br/>
     👋相关而非因果
    </p>
    <p>
     💫在社会发展方面，大数据决策逐渐成为一种新的决策方式，大数据应用有力促进了信息技术与各行业的深度融合，大数据开发大大推动了新技术和新应用的不断涌现
    </p>
    <p>
     💫在就业市场方面，大数据的兴起使得数据科学家成为热门职业
    </p>
    <p>
     💫在人才培养方面，大数据的兴起，将在很大程度上改变中国高校信息技术相关专业的现有教学和科研体制
     <br/>
     大数据行业发展趋势的两个层面：国家层面和高校教育和就业层面。

     3.2.2 高校教育和就业层面

    </p>
    <h3>
     <a id="33__280">
     </a>
     3.3 未来展望
    </h3>
    <p>
     现在大数据正处于下一个风口
    </p>
    <p>
     💖2020年是5G的元年，国家在大力铺设5G设备，2021年就是5G手机应用的开始，也是大数据要爆发的1年。5G带来的是每秒钟10g的数据，会给每家公司都带来海量的数据。那么传统的Java工具根本解决不了海量数据的存储。就更不用说海量数据的计算了。如果你对5G的感触不够深，可以回忆一下3G和4G的区别。3G时只能打电话、发短信，当时还觉得很好，觉得3G不错。但是4G来了后，大家很少打电话和发短信了，都改为语音、视频、直播、网上购物等生活方式，带火了淘宝、京东、美团、字节跳动等企业。没有跟上节奏的百度，有点摇摇欲坠。当然百度ai还是很牛逼的。
    </p>
