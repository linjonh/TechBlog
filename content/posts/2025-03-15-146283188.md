    <h2>
     1、题目描述
    </h2>
    <p>
     数字
     <code>
      n
     </code>
     代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且
     <strong>
      有效的
     </strong>
     括号组合。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <pre><strong>输入：</strong>n = 3
<strong>输出：</strong>["((()))","(()())","(())()","()(())","()()()"]</pre>
    <h2>
     2、初始思路
    </h2>
    <h3>
     2.1 思路
    </h3>
    <p>
     全排列+筛选
    </p>
    <h3>
     2.2 犯错点
    </h3>
    <p>
     全排列，时间复杂度高，且易读性较差
    </p>
    <h2>
     3 优化算法
    </h2>
    <h3>
     3.1 思路
    </h3>
    <p>
     在构造的过程中直接确保括号的正确匹配：
    </p>
    <ul>
     <li>
      当左括号数量&lt;n时，可以继续添加左括号；
     </li>
     <li>
      当右括号数量&lt;左括号数量时，才可以添加右括号。
     </li>
    </ul>
    <h3>
     3.2 代码
    </h3>
    <pre><code class="hljs">class Solution:
    def generateParenthesis(self, n: int) -&gt; List[str]:
        res = []
        path = []
        def backtracking(path, left, right):
            if len(path) == 2*n:
                res.append(''.join(path.copy()))
                return
            if left&lt;n:
                path.append('(')
                backtracking(path, left+1, right)
                path.pop()
            if right&lt;left:
                path.append(')')
                backtracking(path, left, right+1)
                path.pop()
        backtracking(path, 0, 0)
        return res</code></pre>
