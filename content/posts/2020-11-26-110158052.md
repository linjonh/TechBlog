    </p>

     HaaS600 Kit 套件包含以下物料：

    <ul id="ul-hz4-7p5-cim">

      <p id="p-qbr-d7b-sll">

      <p id="p-7me-i31-yni">
       micro-USB线缆 x 1

       5V/2A电源适配器 x 1

     <li id="li-h9x-t1i-xfu">
      <p id="p-nr2-j90-geq">
       4G天线 x 1
      </p>
     </li>
     <li id="li-f5s-k14-lhp">

    <h3>
     系统环境
    </h3>
    <h4 id="p-uiy-903-qxb">
     <strong>
      连接设备
     </strong>
    </h4>
    <p id="p-hqm-2kc-p6x">
     按照下图所示将开发电脑跟设备的调试/用户串口通过MicroUSB连接，连接后，将在PC设备管理器中出现两个串口，分别用于调试和用户串口输入输出。
    </p>
    <p id="p-szt-zck-n5i">
     <a href="https://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/zh-CN/8011773061/p176734.png" rel="nofollow">

     </a>
    </p>
    <p id="p-2zw-rki-bmk">
    </p>
    <h4 id="p-lq4-vz1-45u">
     <strong>
      安装USB转串口驱动

    <p id="p-2xa-hza-to5">
     1. 不同的电脑生成的串口号和名称不一样，以实际生成的为准。

    <p id="p-h8z-4lh-uf0">
     2. 生成的两个串口，一个是debug串口(查看运行日志，只读)，另一个是用户串口(JavaScript交互式解释器或其他用途)

        Mac版
       </a>
      </p>
     </li>
     <li id="li-tlf-gou-l4m">
      <p id="p-8he-6db-63h">
       <a href="https://aos-oss-cn-shanghai.oss-cn-shanghai.aliyuncs.com/amp/amp-win.zip?spm=a2c4g.11186623.2.25.46801a2ds93qcV&amp;file=amp-win.zip" rel="nofollow">
        Windows版
       </a>
      </p>
     </li>
    </ul>

    <p id="p-5zx-w4z-qkr">
     请参考：
