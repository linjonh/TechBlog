     方法一：使用额外的数组
    </p>
    <p>
     我们可以使用额外的数组来将每个元素放至正确的位置。用 n 表示数组的长度，我们遍历原数组，将原数组下标为 i 的元素放至新数组下标为 (i+k)mod n 的位置，最后将新数组拷贝至原数组即可。
    </p>
    <pre><code class="language-javascript">
var rotate = function(nums, k) {
    const n = nums.length;
    const newArr = new Array(n);
    for (let i = 0; i &lt; n; ++i) {
        newArr[(i + k) % n] = nums[i]
    }
    for (let i = 0; i &lt; n; ++i) {
        nums[i] = newArr[i];
    }
};</code></pre>
