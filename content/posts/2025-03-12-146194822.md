    <h3>
     <a id="1_Package_16">
     </a>
     <strong>
      1. 为什么需要包（Package）？如何组织代码？
     </strong>
    </h3>
    <h4>
     <a id="__17">
     </a>
     <strong>
      🔹 什么是包？
     </strong>
    </h4>
    <p>
     <strong>
      包（Package）
     </strong>
     是 Java 用来组织类和接口的一种机制，它可以：
     <br/>
     ✅
     <strong>
      避免类名冲突
     </strong>
     （如
     <code>
      com.mysql.Connection
     </code>
     和
     <code>
      java.sql.Connection
     </code>
     ）
     <br/>
     ✅
     <strong>
      提高代码的可维护性
     </strong>
     （模块化结构更清晰）
     <br/>
     ✅
     <strong>
      控制访问权限
     </strong>
     （
     <code>
      public
     </code>
     、
     <code>
      protected
     </code>
     、
     <code>
      private
     </code>
     配合
     <code>
      package-private
     </code>
     ）
    </p>
    <h4>
     <a id="__Java__23">
     </a>
     <strong>
      🔹 如何定义 Java 包？
     </strong>
    </h4>
    <p>
     在 Java 中，
     <strong>
      包名通常以公司域名倒写+项目结构
     </strong>
     ，例如：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">package</span> <span class="token namespace">com<span class="token punctuation">.</span>example<span class="token punctuation">.</span>utils</span><span class="token punctuation">;</span> <span class="token comment">// 定义包</span>

<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">MyUtils</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">printMessage</span><span class="token punctuation">(</span><span class="token class-name">String</span> message<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Message: "</span> <span class="token operator">+</span> message<span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     📌
     <strong>
      注意：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       包声明必须放在文件的第一行
      </strong>
     </li>
     <li>
      <strong>
       包路径必须和实际目录结构一致
      </strong>
      （如
      <code>
       com/example/utils/MyUtils.java
      </code>
      ）
     </li>
    </ul>
    <h4>
     <a id="__38">
     </a>
     <strong>
      🔹 如何使用其他包中的类？
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">import</span> <span class="token import"><span class="token namespace">com<span class="token punctuation">.</span>example<span class="token punctuation">.</span>utils<span class="token punctuation">.</span></span><span class="token class-name">MyUtils</span></span><span class="token punctuation">;</span> <span class="token comment">// 导入包中的类</span>

<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Main</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">MyUtils</span><span class="token punctuation">.</span><span class="token function">printMessage</span><span class="token punctuation">(</span><span class="token string">"Hello, Java Package!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     如果不
     <code>
      import
     </code>
     ，可以用
     <strong>
      全限定类名
     </strong>
     ：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Main</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name"><span class="token namespace">com<span class="token punctuation">.</span>example<span class="token punctuation">.</span>utils<span class="token punctuation">.</span></span>MyUtils</span><span class="token punctuation">.</span><span class="token function">printMessage</span><span class="token punctuation">(</span><span class="token string">"Hello, Java Package!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <h3>
     <a id="2_JAR__58">
     </a>
     <strong>
      2. JAR 存档文件是什么？如何创建和运行？
     </strong>
    </h3>
    <h4>
     <a id="__JAR__59">
     </a>
     <strong>
      🔹 什么是 JAR 文件？
     </strong>
    </h4>
    <p>
     <strong>
      JAR（Java Archive）
     </strong>
     文件是
     <strong>
      Java 类及资源的压缩包
     </strong>
     ，它的作用类似 ZIP 文件，主要用于：
    </p>
    <ul>
     <li>
      <strong>
       封装 Java 类文件（.class）
      </strong>
     </li>
     <li>
      <strong>
       包含资源文件（如图片、配置文件）
      </strong>
     </li>
     <li>
      <strong>
       支持数字签名和权限控制
      </strong>
     </li>
     <li>
      <strong>
       加快 Java 程序的加载和运行
      </strong>
     </li>
    </ul>
    <h4>
     <a id="__JAR__66">
     </a>
     <strong>
      🔹 如何创建 JAR 文件？
     </strong>
    </h4>
    <p>
     1️⃣
     <strong>
      编译 Java 代码
     </strong>
    </p>
    <pre><code class="prism language-sh">javac <span class="token parameter variable">-d</span> out Main.java
</code></pre>
    <p>
     2️⃣
     <strong>
      创建 JAR 文件
     </strong>
    </p>
    <pre><code class="prism language-sh">jar cvf myapp.jar <span class="token parameter variable">-C</span> out <span class="token builtin class-name">.</span>
</code></pre>
    <p>
     参数说明：
    </p>
    <ul>
     <li>
      <code>
       c
      </code>
      ：创建 JAR 文件
     </li>
     <li>
      <code>
       v
      </code>
      ：显示创建过程
     </li>
     <li>
      <code>
       f
      </code>
      ：指定 JAR 文件名
     </li>
     <li>
      <code>
       -C
      </code>
      ：切换目录（保证正确的包路径）
     </li>
    </ul>
    <h4>
     <a id="__JAR__81">
     </a>
     <strong>
      🔹 如何运行 JAR 文件？
     </strong>
    </h4>
    <p>
     如果 JAR 里面有
     <code>
      Main-Class
     </code>
     ，可以直接运行：
    </p>
    <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-jar</span> myapp.jar
</code></pre>
    <p>
     但如果
     <code>
      MANIFEST.MF
     </code>
     <strong>
      未指定
      <code>
       Main-Class
      </code>
     </strong>
     ，则需要：
    </p>
    <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-cp</span> myapp.jar com.example.Main
</code></pre>
    <hr/>
    <h3>
     <a id="3__JAR__92">
     </a>
     <strong>
      3. 如何管理 JAR 依赖？
     </strong>
    </h3>
    <p>
     在实际项目中，我们通常需要引用第三方 JAR，例如
     <code>
      mysql-connector-java.jar
     </code>
     ，如何正确管理这些 JAR 依赖呢？
    </p>
    <h4>
     <a id="__1_JAR_95">
     </a>
     <strong>
      🔹 方法 1：手动管理 JAR
     </strong>
    </h4>
    <p>
     将
     <code>
      mysql-connector-java.jar
     </code>
     复制到
     <code>
      libs/
     </code>
     目录，然后运行：
    </p>
    <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-cp</span> myapp.jar<span class="token punctuation">;</span>libs/mysql-connector-java.jar com.example.Main
</code></pre>
    <p>
     ⚠
     <strong>
      问题：
     </strong>
    </p>
    <ul>
     <li>
      需要手动下载和维护 JAR
     </li>
     <li>
      JAR 依赖管理混乱
     </li>
    </ul>
    <h4>
     <a id="__2_Maven__JAR__104">
     </a>
     <strong>
      🔹 方法 2：使用 Maven 进行 JAR 依赖管理
     </strong>
    </h4>
    <p>
     使用
     <strong>
      Maven
     </strong>
     可以自动管理 JAR 依赖，示例
     <code>
      pom.xml
     </code>
     ：
    </p>
    <pre><code class="prism language-xml"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>dependencies</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>dependency</span><span class="token punctuation">&gt;</span></span>
        <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>groupId</span><span class="token punctuation">&gt;</span></span>mysql<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>groupId</span><span class="token punctuation">&gt;</span></span>
        <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>artifactId</span><span class="token punctuation">&gt;</span></span>mysql-connector-java<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>artifactId</span><span class="token punctuation">&gt;</span></span>
        <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>version</span><span class="token punctuation">&gt;</span></span>8.0.33<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>version</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>dependency</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>dependencies</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     然后直接：
    </p>
    <pre><code class="prism language-sh">mvn package
<span class="token function">java</span> <span class="token parameter variable">-jar</span> target/myapp.jar
</code></pre>
    <p>
     ✅
     <strong>
      Maven 自动下载依赖，避免手动管理 JAR！
     </strong>
    </p>

    <h3>
     <a id="4_Java__124">
     </a>
     <strong>
      4. Java 项目如何部署？
     </strong>
    </h3>
    <h4>
     <a id="__125">
     </a>
     <strong>
      🔹 本地部署
     </strong>
    </h4>
    <p>
     1️⃣
     <strong>
      手动运行 JAR
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-jar</span> myapp.jar
</code></pre>
    <p>
     2️⃣
     <strong>
      后台运行（Linux）
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">nohup</span> <span class="token function">java</span> <span class="token parameter variable">-jar</span> myapp.jar <span class="token operator">&gt;</span> log.txt <span class="token operator"><span class="token file-descriptor important">2</span>&gt;</span><span class="token file-descriptor important">&amp;1</span> <span class="token operator">&amp;</span>
</code></pre>
    <p>
     3️⃣
     <strong>
      注册为 Linux 服务
     </strong>
     <br/>
     创建
     <code>
      /etc/systemd/system/myapp.service
     </code>
     ：
    </p>
    <pre><code class="prism language-ini">[Unit]
Description=MyApp Service
After=network.target

[Service]
ExecStart=/usr/bin/java -jar /home/user/myapp.jar
Restart=always
User=user
Group=user

[Install]
WantedBy=multi-user.target
</code></pre>
    <p>
     然后：
    </p>
    <pre><code class="prism language-sh"><span class="token function">sudo</span> systemctl <span class="token builtin class-name">enable</span> myapp
<span class="token function">sudo</span> systemctl start myapp
</code></pre>
    <h4>
     <a id="__156">
     </a>
     <strong>
      🔹 服务器部署
     </strong>
    </h4>
    <p>
     1️⃣
     <strong>
      上传 JAR
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">scp</span> myapp.jar user@server:/home/user/
</code></pre>
    <p>
     2️⃣
     <strong>
      运行 JAR
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">ssh</span> user@server
<span class="token function">java</span> <span class="token parameter variable">-jar</span> myapp.jar
</code></pre>
    <p>
     📌
     <strong>
      推荐结合 Docker/Kubernetes 实现自动化部署！
     </strong>
    </p>

    <h3>
     <a id="5__JAR__Java__170">
     </a>
     <strong>
      5. 实战演练：从零创建 JAR 并运行 Java 程序
     </strong>
    </h3>
    <h4>
     <a id="Step_1_Java__171">
     </a>
     <strong>
      Step 1：编写 Java 代码
     </strong>
    </h4>
    <p>
     创建
     <code>
      Main.java
     </code>
     ：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">package</span> <span class="token namespace">com<span class="token punctuation">.</span>example</span><span class="token punctuation">;</span>

<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Main</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Hello, Java JAR!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

     <strong>

     </strong>
    </h4>
    <pre><code class="prism language-sh">javac <span class="token parameter variable">-d</span> out Main.java
</code></pre>
    <h4>
     <a id="Step_3_MANIFESTMF_188">

     <strong>
      Step 3：创建

</code></pre>
    <h4>
     <a id="Step_4_JAR_193">
     </a>
     <strong>

     </strong>
    </h4>
    <pre><code class="prism language-sh">jar cvfm myapp.jar MANIFEST.MF <span class="token parameter variable">-C</span> out <span class="token builtin class-name">.</span>
</code></pre>
    <h4>
     <a id="Step_5_JAR_198">
     </a>
     <strong>
      Step 5：运行 JAR
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-jar</span> myapp.jar
</code></pre>
    <p>
     ✅
     <strong>
      成功运行 Java JAR 应用！
     </strong>
    </p>
