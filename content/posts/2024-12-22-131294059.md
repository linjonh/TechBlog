    <h2>
     <a id="JavaJar_6">
     </a>
     将Java程序打包成Jar包
    </h2>
    <p>
     <strong>
      1. file -&gt; Project Structure 或者直接使用全局快捷键（Ctrl + Alt + Shift + S）
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/788770958ba7e7fee970d4a34c770370.png">
      <br/>
      <strong>
       2. 新建JAR 并且选择有依赖关系的模块
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b291d42f05da2344287d1232ff320611.png">
       <br/>
       <strong>
        3. 选择要打包的项目，选择要执行的 main 函数和jar包路径
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b5dd9ce9d7c37ea511bb1c3b51722a38.png">
        <br/>
        <strong>
         3.1 选择启动的主函数
        </strong>
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3c4739c29873bb4bcb1d1e06d43ad1cf.png">
         <br/>
         4. 成功界面
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a5e4580c3ab97a5729e5500579c1b46b.png">
          <br/>
          <strong>
           4.1 你已经在这个路径下已经建过一个module了的话再次新建module就会报这个错，只要删除对应的文件夹即可
          </strong>
          <br/>
          <img alt="加粗样式" src="https://i-blog.csdnimg.cn/blog_migrate/81fbc0a31c8f16f2915bb3f84ab52e64.png">
           <br/>
           5. 点击上方栏的Build生成Build Artifacts
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c95f8745aa5b89c12da527d6e693ff7f.png">
            <br/>
            <strong>
             6. 选择点击build就会在前面设置的路径下生成我们的jar包，其实这个jar包就已经可以运行了，继续看下去我们把它变成exe可执行文件。
            </strong>
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f4b3b4b0cad8f9207d42211edc3b7be7.png">
             <br/>
             <strong>
              去指定的目录下可以看到我们的Jar包已经生成了
             </strong>
             <br/>
             <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a97ba0c54711b1b35449c9038c19169a.png"/>
            </img>
