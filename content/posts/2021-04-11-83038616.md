    <p>
     导入一个新项目后常会出现
     <br/>
     The import java.io cannot be resolved
     <br/>
     String cannot be resolved to a type
     <br/>
     其原因在于没有导入需要的包。
    </p>
    <p>
     解决办法：
    </p>
    <p>
     1、选中项目点击右键找到Properties-&gt;java build path-&gt;Libraries-&gt;Add External Jars-&gt;选中本项目所需要的jar包。
    </p>
    <p>
     2、选中项目点击右键找到Properties-&gt;java build path-&gt;Libraries 看看有没有 JRE System Library,如果没有，点击Add Library，然后选中 JRE System Library-&gt;Next-&gt;Finish即可。
