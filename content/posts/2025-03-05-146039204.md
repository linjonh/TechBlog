    <h2>
     一、软件简介
    </h2>
    <p>
     DBeaver 是
     <strong>
      开源跨平台数据库管理工具
     </strong>
     ，支持 MySQL、PostgreSQL、Oracle、SQLite 等 80+ 种数据库，提供可视化SQL编辑、数据导入导出、ER图表生成等功能。社区版（CE）完全免费，是企业级数据库管理的轻量级替代方案。
    </p>
    <hr/>

    <h4>
     2. Java环境验证
    </h4>
    <p>
     打开CMD执行命令：
    </p>
    <pre><code class="language-bash">java -version</code></pre>
    <hr/>
    <h3>
     三、安装流程详解
    </h3>
    <h4>
     步骤1：获取安装包
    </h4>
    <p>
     访问
     <a href="https://pan.quark.cn/s/6495c52e1580" rel="nofollow" title="官网安装包下载页">
      官网安装包下载页
     </a>
     ，下载
     <strong>
      dbeaver-ce-25.0.0-x86_64-setup.zip
     </strong>
    </p>
    <h4>
     步骤2：启动安装向导
    </h4>
    <p>
     解压压缩包，双击
     <code>
      dbeaver-ce-25.0.0-x86_64-setup.exe
     </code>
     ，启动安装程序：
     <img alt="" height="450" src="https://i-blog.csdnimg.cn/direct/d47a2cae7f544200ae433fa7cc3d5491.png" width="623"/>
    </p>
    <h4>
     步骤3：组件选择
    </h4>
    <p>
     勾选以下核心组件：
    </p>
    <ul>
     <li>
      主程序文件（必选）
     </li>
     <li>
      桌面快捷方式
     </li>
     <li>
      文件关联（.sql/.db等）
      <img alt="" height="450" src="https://i-blog.csdnimg.cn/direct/0d037529a430473390dad70c9cbe994a.png" width="623"/>
     </li>
    </ul>
    <h4>
     步骤4：指定安装路径
    </h4>
    <p>
     建议修改默认路径（示例）：
    </p>
    <pre><code>D:\DatabaseTools\DBeaverCE 
</code></pre>
    <h4>
     <img alt="" height="450" src="https://i-blog.csdnimg.cn/direct/0e077f3ed7534572914cd66a00b2e557.png" width="623"/>
    </h4>
    <h4>
     步骤5：完成安装
    </h4>
    <p>
     点击
     <strong>
      "安装"
     </strong>
     ，即可进行软件安装进程：
     <img alt="" height="450" src="https://i-blog.csdnimg.cn/direct/70a19707dff64dd49ea0b3bd958821da.png" width="623"/>
    </p>
    <hr/>
    <h3>
     四、数据库连接配置
    </h3>
    <h4>
     1. 新建MySQL连接
    </h4>
    <p>
     点击工具栏
     <strong>
      New Database Connection
     </strong>
     图标，选择MySQL数据库类型
    </p>
    <p>
     填写关键参数：
    </p>
    <ul>
     <li>
      Server Host：localhost（本地服务器）
     </li>
     <li>
      Port：3306
     </li>
     <li>
      Database：test_db
     </li>
     <li>
      Username：root
     </li>
     <li>
      Password：******
     </li>
    </ul>
    <h4>
     2. 驱动管理
    </h4>
    <p>
     若首次使用某数据库类型，软件会自动下载驱动
    </p>
    <hr/>
    <h3>
     五、核心功能验证
    </h3>
    <h4>
     测试1：SQL编辑器
    </h4>
    <p>
     在SQL面板执行查询语句：
    </p>
    <pre><code class="language-sql">SELECT * FROM users WHERE status=1;</code></pre>
    <h4>
     测试2：数据导出
    </h4>
    <p>
     右键表选择
     <strong>
      Export Data
     </strong>
     ，支持CSV/Excel/SQL等格式
    </p>
    <hr/>
    <h3>
     六、常见问题解答
    </h3>
    <h4>
     Q1：启动时报"Java environment not found"错误？
    </h4>
    <ul>
     <li>
      检查JDK/JRE是否安装
     </li>
     <li>
      在dbeaver.ini 中添加JVM路径：
     </li>
    </ul>
    <pre><code>-vm
C:\Program Files\Java\jdk-17\bin 
</code></pre>
    <h4>
     Q2：如何连接云数据库？
    </h4>
    <p>
     在连接配置中修改：
    </p>
    <ul>
     <li>
      Host改为云服务器公网IP
     </li>
     <li>
      开放云服务器安全组对应端口
     </li>
     <li>
      使用SSH隧道连接（高级设置页签）
     </li>
    </ul>
    <h4>
     Q3：插件扩展安装
    </h4>
    <p>
     通过
     <strong>
      Help &gt; Install New Software
     </strong>
     添加插件市场源：
    </p>
    <pre><code>https://dbeaver.com/update/ce/latest  
</code></pre>
