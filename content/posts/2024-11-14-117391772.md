### 一.前言

* Anaconda 工具集合 –
  **第三方库集合**
* Pycharm 开发工具，俗称 IDE –
  **写代码工具**

### 二.Anaconda 是什么？

Anaconda 是一个提供
[Python](https://www.codersrc.com)
开发所需的工具包，包括 Python / IPython / CPython / Conda 等 180 个依赖项，同时支持 Linux，Mac，Windows 三大平台。

![](https://i-blog.csdnimg.cn/blog_migrate/58febd976e995cf5e0f4e95b31ecabb3.png)

**想做 Python 开发，搭建开发环境，有两种方式**

**1.秃顶方案**
：你除了安装 Pycharm 开发工具之外还得安装各种依赖性环境，配置超级麻烦 ；

**2.懒人方案**
：安装了 Anaconda 之后，只需要在安装开发 Pycharm 即可开始
[Hello World](https://www.codersrc.com/archives/4410.html)
入门。

打个形象的比喻：你想在纸上做笔记或者写字，那么你就得用铅笔/毛笔/钢笔/水彩笔….总之需要笔才能完成，Anaconda 就可以看作是文具盒，里面什么玩意的笔都有，随便你挑。不用文具盒 (Anaconda) 也行，你自己得跑到文具店一样一样的买。

![](https://i-blog.csdnimg.cn/blog_migrate/8ebc66647fa8f5e65cf256a10021c584.jpeg)

​

由于包含了大量得安装包，安装包大约 500M 左右。大是大了点，一应俱全。使用 Anaconda，省时省心省力，你值得拥有！

### 三.Anaconda 下载地址

1.Anaconda 官方下载地址：
[Anaconda 下载地址](https://www.anaconda.com/products/individual#Downloads)
，根据你机器的操作系统选择不同版本，比如选择 Python3 版本的 Windows 64 位的安装包。

![](https://i-blog.csdnimg.cn/blog_migrate/e9e33a2e598049502adf67baec9581f2.png)

```
64-bit：对应64位操作系统
32-bit：对应32位操作系统

```

2.如果 Anaconda 官网下载比较慢呢，可以在
[清华大学开源软件镜像站](https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/)
下载，自行选择版本。

![](https://i-blog.csdnimg.cn/blog_migrate/3c85ffdc46c7f71a0f59d20125e886f5.png)

```
x86_64：对应64位操作系统
x86：对应32位操作系统

```
