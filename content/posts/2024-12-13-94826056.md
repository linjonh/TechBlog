    <h2>
     <a id="_ubuntu__0">
     </a>
     虚拟机下 ubuntu 显示屏幕太小问题
    </h2>
    <p>
     转载
     <a href="https://blog.csdn.net/dcrmg/article/details/74090307">
      https://blog.csdn.net/dcrmg/article/details/74090307
     </a>
    </p>
    <p>
     解决方法是安装 VMware Tool 插件解决。
    </p>
    <h3>
     <a id="1__ISO__7">
     </a>
     1. 首先修改 ISO 文件路径
    </h3>
    <blockquote>
     <p>
      将虚拟机下的操作系统执行
      <strong>
       关机操作
      </strong>
      ，然后在VMware菜单栏上选择虚拟机-&gt;设置-&gt;CD/DVD(SATA)，在 "链接"栏勾选 “使用物理驱动器”。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/422c7d2a2cb3295c30f0f339f10d2975.png"/>
    </p>
    <p>
     修改 ISO 文件路径的原因是防止出现
     <code>
      客户机操作客户机操作系统己将 CD-ROM 门锁定，并且可能正在使用 CD-ROM.....
     </code>
     的问题。
    </p>
    <h3>
     <a id="2__VMware_Tool_16">
     </a>
     2. 安装 VMware Tool
    </h3>
    <ol>
     <li>
      <p>
       启动并进入操作系统
      </p>
     </li>
     <li>
      <p>
       在 VMware 菜单栏上选择
       <strong>
        虚拟机-&gt;安装 VMware Tools
       </strong>
       ，点击 ubuntu 左侧列表里的 DVD 图标，就会出现 VMware Tools 的安装文件 VMwareTools-10.1.6-5214329.tar.gz
      </p>
     </li>
     <li>
      <p>
       将压缩包拷贝到桌面并解压
      </p>
     </li>
    </ol>
    <blockquote>
     <p>
      解压命令：tar -zxvf 压缩包名
     </p>
    </blockquote>
    <ol start="4">
     <li>
      解压完成后进入 vmware-tools-distrib 中，用root身份执行文件夹里vmware-install.pl的文件。
     </li>
    </ol>
    <blockquote>
     <p>
      sudo ./vmware-install.pl
     </p>
    </blockquote>
    <ol start="5">
     <li>
      接着就一直接Enter键就行了，如果一遍没有安装好，可以再执行一次
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4cb465ab79ee174dc107554a3bf2df80.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de17a7105fc2843ef8416980b0a7b588.png"/>
    </p>
    <p>
     因为我已经安装过 tool ，这里就是重新安装。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/75b801d29bf1dee841176e80207d4085.png"/>
    </p>
    <p>
     最后，如果你的显示屏幕还是比较奇怪，可以选择重新安装 VMware tool。
    </p>
