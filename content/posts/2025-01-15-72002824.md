      <blockquote>
       <p>
        未来物联网全栈开发 ——
        <a class="replace_word" href="http://lib.csdn.net/base/javascript" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="JavaScript知识库">
         JavaScript
        </a>
        OR
        <a class="replace_word" href="http://lib.csdn.net/base/python" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="Python知识库">
         Python
        </a>
        ？
        <br/>
        物联网开发涉及面庞杂，开发周期长，所以我们必须寻找一种覆盖面广的编程语言和方法。
        <br/>
        JavaScript 支持 HTTP 和 JSON 、支持函数式编程、可提供交互式环境等特点堪称适用于物联网全栈开发；
        <br/>
        Python 作为一种胶水语言，可在物联网及
        <a class="replace_word" href="http://lib.csdn.net/base/embeddeddevelopment" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="嵌入式开发知识库">
         嵌入式
        </a>
        系统中承担大量任务，并部分替代以上语言。
       </p>
      </blockquote>
      <hr/>
      <p>
       物联网是新一代信息技术的重要组成部分，也是“信息化”时代的重要发展阶段，不太清楚的可以看上篇文章《
       <a href="http://blog.csdn.net/djy1992/article/details/52670655" rel="noopener noreferrer" target="_blank">
        IoT领域的故事、经历、技术实战
       </a>
       》，正所谓语言无国度，无论是 Js 还是 Python ， 都有各自优点，最主要还是要看个人选择自身比较合适的语言。但是在未来物联网全栈开发的基础上，个人觉得 Python 作为主流且更具有优势，不信？请看文章最后的一句话。
       <br/>
       <br/>
       <br/>
       <img alt="米奇云科技奥特曼超人杜锦阳" src="https://img-blog.csdn.net/20170512105659204?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvREpZMTk5Mg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
        <br/>
        <br/>
       </img>
      </p>
      <h3 id="先来说说-javascript">
       <a name="t0" target="_blank">
       </a>
       <strong>
        先来说说 Javascript
       </strong>
      </h3>
      <p>
       可能很少人知道
       <strong>
        js 的前身
       </strong>
       ，Javascript 的前身叫LiveScript，是Sun公司推出
       <a class="replace_word" href="http://lib.csdn.net/base/javase" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="Java SE知识库">
        Java
       </a>
       后 Netscape 公司引进的，当时因为某些原因，它将原有的 LiveScript 重新进行设计，参考了面向对象的概念，并更名为 JavaScript，据说是因为一位小职员在无聊时开发出来的自身版本 。
       <br/>
       之所以取名JavaScript，是因为JavaScript是作为一种嵌入HTML文档的、基于对象的脚本设计语言。其中很大一部分的语法同Java语言很相似，而且JavaScript的设计可以使它很容易地同Java语言一同工作，它可以充分支持Java的applet小应用程序，Java的 applet小应用程序也可以很方便地访问已有的JavaScript代码。所以，
       <strong>
        也可以JavaScript看成是Java语言的某种简化版本。
       </strong>
      </p>
      <p>
       <img alt="米奇云科技奥特曼超人杜锦阳" src="https://img-blog.csdn.net/20170512105520535?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvREpZMTk5Mg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
      </p>
      <p>
       <a class="replace_word" href="http://lib.csdn.net/base/javascript" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="JavaScript知识库">
        前端开发
       </a>
       人员都比较清楚 JavaScript 的优点，来排列下：
      </p>
      <ol>
       <li>
        JavaScript 可以减少网络传输。
       </li>
       <li>
        JavaScript 拥有跨平台性。
       </li>
       <li>
        JavaScript 简单、方便操纵HTML对象。
       </li>
       <li>
        JavaScript 支持分布式运算。
       </li>
       <li>
        JavaScript 简单弱类型，并且相对安全。
       </li>
      </ol>
      <p>
       可是它也有
       <span style="color:red">
        <strong>
         局限性
        </strong>
        ，如各浏览器厂商对JavaScript支持程度不同，“Web安全性”对JavaScript一些功能牺牲，如 .NET 大多数攻击都是通过 Js 运行脚本注入。
       </span>
      </p>
      <blockquote>
       <p>
        目前在互联网上有很多浏览器，如Firefox、Internet
        <br/>
        Explorer、Opera等，但每种浏览器支持JavaScript的程度是不一样的，不同的浏览器在浏览一个带有JavaScript脚本的主页时，由于对JavaScript的支持稍有不同，其效果会有一定的差距，有时甚至会显示不出来。
       </p>
      </blockquote>
      <p>
       <br/>
      </p>
      <h3 id="再来说说-python">
       <a name="t1" target="_blank">
       </a>
       <strong>
        再来说说 Python
       </strong>
      </h3>
      <p>
       <strong>
        胶水语言
       </strong>
       ，同样的也是面向对象，但我更喜欢称它为“蛇”语言，
       <strong>
        一种 C 和 shell 之间的语言
       </strong>
       ，功能全面，易学易用，可拓展的语言，为什么这么说，在你会 Java 的基础上，如果你同时学 Js 和 Python 的话，我想，Python 会更快胜任。
      </p>
      <p>
       开发游戏里有一种常见的应用情形，需要快速生成模型。如何操作？
      </p>
      <p>
       <span style="color:red">
        使用 Python 快速生成程序的原型（有时甚至是程序的最终界面），然后对其中有特别要求的部分，用更合适的语言改写
       </span>
       。
       <br/>
       <img alt="米奇云科技奥特曼超人杜锦阳" src="https://img-blog.csdn.net/20170512105203650?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvREpZMTk5Mg==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
        <br/>
        <strong>
         比如3D游戏中的图形渲染模块，性能要求特别高，就可以用C/C++重写，而后封装为Python可以调用的扩展类库。需要注意的是在您使用扩展类库时可能需要考虑平台问题，某些可能不提供跨平台的实现。
        </strong>
       </img>
      </p>
      <blockquote>
       <p>
        Python 在吉多手中诞生了。实际上,第一个实现是在 Mac 机上。可以说,Python 是从 ABC 发展起来,主要受到了Modula-3(另一种相当优美且强大的语言,为小型团体所设计的)的影响。并且结合了
        <strong>
         Unix shell 和C
        </strong>
        的习惯。
       </p>
      </blockquote>
      <p>
       如果你使用过，应该对它所支持的扩展库印象会很深刻吧，它可扩展性和可嵌入性，包括实用性非常强。
       <span style="color:red">
        如著名的
        <strong>
         <a class="replace_word" href="http://lib.csdn.net/base/computervison" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="计算机视觉知识库">
          计算机视觉
         </a>
         库
         <a class="replace_word" href="http://lib.csdn.net/base/opencv" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="OpenCV知识库">
          OpenCV
         </a>
         、三维可视化库VTK、医学图像处理库ITK
        </strong>
        ，还有最近在帮公司开发的高级反编译读取库（暂定命名YHK）。
       </span>
      </p>
      <p>
       <br/>
      </p>
      <h3 id="未来谁是王者">
       <a name="t2" target="_blank">
       </a>
       <strong>
        未来谁是王者
       </strong>
      </h3>
      <p>
       那对比优缺点和使用规则之后我们发现，两者各有千秋，未来物联网发展离不开一样东西，那就是
       <a class="replace_word" href="http://lib.csdn.net/base/aiplanning" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="人工智能规划与决策知识库">
        智能
       </a>
       化与科学计算，
       <strong>
        国外这几年比较主流的是 MATLAB
       </strong>
       ，记得上家公司带领的团队里有个英国留学回来的哥们，和他聊过几次，了解到，其实在国外，MATLAB 一直都是比较主流的，但它是商用，和
       <a class="replace_word" href="http://lib.csdn.net/base/oracle" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="Oracle知识库">
        Oracle
       </a>
       一样，需要付费，且价格不菲。由此可见，国外走的不是“程序、产品、项目”，而是“
       <a class="replace_word" href="http://lib.csdn.net/base/robot" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="机器人知识库">
        机器人
       </a>
       、智能、科学计算、图形、
       <a class="replace_word" href="http://lib.csdn.net/base/datastructure" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="算法与数据结构知识库">
        算法
       </a>
       ”这些，当然国内也有，却是非主流的趋势，今年才感觉到市场的趋势改变，也许未来会着重发展，所以单从这一点来看，科学算法占用了很大的比例。
      </p>
      <p>
       <strong>
        用 Python 做科学计算有如下优点：
       </strong>
      </p>
      <ul>
       <li>
        首先，MATLAB 是商用的，且价格不菲。而 Python 完全免费，众多开源的
        <strong>
         科学计算库
        </strong>
        都提供了Python的调用接口。用户可以在任何计算机上免费安装Python及其绝大多数扩展库。
       </li>
       <li>
        其次，与MATLAB相比，Python是一门更易学、更严谨的程序设计语言。它能让用户编写出更易读、易维护的代码。
       </li>
       <li>
        最后，MATLAB主要专注于工程和科学计算。然而即使在计算领域，也经常会遇到文件管理、界面设计、网络通信等各种需求。而Python有着丰富的扩展库，可以轻易完成各种高级任务，开发者可以用Python实现完整应用程序所需的各种功能。
       </li>
      </ul>
      <p>
       <br/>
      </p>
      <blockquote>
       <p>
        <span style="font-size:14px">
         由于Python语言的简洁性、易读性以及可扩展性，在国外用Python做科学计算的研究机构日益增多，一些知名大学已经采用Python来教授程序设计课程。
         <strong>
          例如卡耐基梅隆大学的编程基础、麻省理工学院的计算机科学及编程导论就使用Python语言讲授。
         </strong>
        </span>
       </p>
       <p>
        众多开源的科学计算软件包都提供了Python的调用接口，例如著名的计算机视觉库OpenCV、三维可视化库VTK、医学图像处理库ITK。
        <br/>
        Python专用的科学计算扩展库就更多了，
        <strong>
         例如如下3个十分经典的科学计算扩展库：NumPy、SciPy和matplotlib
        </strong>
        ，它们分别为Python提供了快速数组处理、数值运算以及绘图功能。因此Python语言及其众多的扩展库所构成的开发环境十分适合工程技术、科研人员处理实验数据、制作图表，甚至开发科学计算应用程序。
       </p>
      </blockquote>
      <p>
       <br/>
       <br/>
       电商、游戏、爬虫、数据分析等商业应用，再到 分布式系统、网络系统、大规模数据存储、web基础
       <a class="replace_word" href="http://lib.csdn.net/base/architecture" rel="nofollow noopener noreferrer" style="color:#df3434; font-weight:bold" target="_blank" title="大型网站架构知识库">
        架构
       </a>
       等进行性能分析，再过来就是 图形渲染、科学计算、智能化、再到运行速度、快速处理、支持的扩展库，
       <strong>
        如果是你，你会选什么呢？
       </strong>
      </p>
