     Windows “上云”后，Linux 有望“捡漏”成最大赢家？
    </h1>

  </div>
 </div>

      <img alt="5533edbc24214828b0c7bbdcf4f2bb56.gif" src="https://i-blog.csdnimg.cn/blog_migrate/3633af23d783e20b6eaed22f874137a2.gif"/>

     <p style="text-align:left;">
      整理 | 郑丽媛
      <br/>
     </p>
     <p style="text-align:left;">
      出品 | CSDN（ID：CSDNnews）
     </p>
     <p style="text-align:justify;">
      今年，凭借 OpenAI 市值飞升的微软，无疑是 AI 行业的领军者。然而微软 2023 年第四财季财报，却揭示了其辉煌背后的危机：卖不动的 Windows 和硬件设备、“道阻且难”的 AI 商业化之路。
     </p>
     <p style="text-align:justify;">
      正如 CEO 萨提亚 · 纳德拉所说，当前 AI 仍是微软加码的重点，但实际上对于 Windows，微软也一直有个明确的长期发展目标：将 Windows 迁移到云端。
     </p>
     <p style="text-align:center;">
      <img alt="7aed94c977e33a4626c652ea5656296c.png" src="https://i-blog.csdnimg.cn/blog_migrate/960e0737f0cb47cb1c6fd1e728fd2397.png"/>
     </p>
     <p style="text-align:justify;">
      <img alt="ae5118817deefdbcee6003f82d3e30c6.png" src="https://i-blog.csdnimg.cn/blog_migrate/ff686a5624963fffa79e0eb161177949.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       在云端完整体验 Windows 10/Windows 11
      </strong>
     </p>
     <p style="text-align:justify;">
      “将 Windows 迁移到云端”这项长期计划的曝光，源自于一份微软内部文件，是近期与美国联邦贸易委员会（FTC）在进行 Xbox 争夺战时披露的。
     </p>
     <p style="text-align:justify;">
      在这份文件中的“现代生活战略和优先级”中，微软将“Windows 11 迁移到云端”的战略视为一个长期机会：
     </p>
     <blockquote>
      <p style="text-align:justify;">
       将 Windows 11 越来越多地迁移到云端：以 Windows 365 为基础，将完整的 Windows 操作系统从云端流式传输到任何设备。利用云和客户端的强大功能，改进 AI 驱动的服务，实现人们数字体验的全面漫游。
      </p>
     </blockquote>
     <p style="text-align:center;">
      <img alt="abd2e7e9e5d2bc03e3886f6e50584b06.png" src="https://i-blog.csdnimg.cn/blog_migrate/124fb8f22b335f68896dfeb958a3b100.png"/>
     </p>
     <p style="text-align:justify;">
      其中提到的 Windows 365，是微软在 Microsoft Inspire 2021 合作伙伴大会上正式发布的新云服务，允许企业在云端体验 Windows 10 或 Windows 11 系统。
     </p>
     <p style="text-align:justify;">
      “我们以 Windows 365 创造出了一个全新的类别：Cloud PC。”当时，萨提亚 · 纳德拉对于 Windows 365 的解释是：“如同以 SaaS 方式将应用程序带到云端一样，我们现在把操作系统也带到了云上，这将为组织提供一种更加灵活、安全的工作方式，让员工无论置身何地，都能保持连接，并更好地释放生产力。”
     </p>
     <p style="text-align:justify;">
      简单来说，Windows 365 并不是一款可以在本地安装的传统桌面操作系统，但它可在任何现代 Web 浏览器上运行，并允许用户从各种设备访问 Cloud PC。值得一提的是，这个“设备”不仅可以是 Windows 设备，还可以是 iPad、Android 手机等几乎任何可以连接到 Internet 的设备。
     </p>
     <p style="text-align:justify;">
      不仅如此，Microsoft 365 总经理 Wangui McKelvey 还表示，Windows 365 提供即时启动体验，即用户可以从云端将 App、工具、数据和设置，传输到 Mac、iPad 、Linux 设备和 Android 设备上：“你可以从上次中断的地方继续，因为即使切换设备，Cloud PC 的状态仍然保持不变。”
     </p>
     <p style="text-align:justify;">
      也就是说，只要设备能连接到 Internet、具有 Web 浏览器，用户就可以通过 Windows 365 设备访问 Cloud PC，在云端完整体验 Windows 10 或 Windows 11，包括应用、数据和各项设置。
     </p>
     <p style="text-align:left;">
      <img alt="82d3de04ba55a4c62c5d37d79b2e7394.png" src="https://i-blog.csdnimg.cn/blog_migrate/b53e7528d09b80dc5de91be14da1e2b1.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       “价格不菲”的 Windows 365
      </strong>
     </p>
     <p style="text-align:justify;">
      自 2021 年 8 月 2 日起，Windows 365 已正式面向企业提供，共有两个版本：Windows 365 Business 和 Windows 365 Enterprise。
     </p>
     <p style="text-align:center;">
      <img alt="a727b4c821a1b670c154b6d8ea18f997.png" src="https://i-blog.csdnimg.cn/blog_migrate/946e5eb76a7d6fdfa20c47ab543e9dda.png"/>
     </p>
     <p style="text-align:justify;">
      不论是哪个版本，Windows 365 的订阅费都不算便宜，起价便为每位用户每月 31 美元（约人民币 223 元），该价位将提供一个 4GB 的 Azure 虚拟机和 128GB 的存储空间，以及 Microsoft 365 应用程序、Outlook 和 OneDrive。
     </p>
     <p style="text-align:center;">
      <img alt="4113a264a2c9c1c03b9d28f9ad505d28.png" src="https://i-blog.csdnimg.cn/blog_migrate/30828ccfbcdbde007eddd75147263799.png"/>
     </p>
     <p style="text-align:justify;">
      除此之外，外媒 Windows Central 高级编辑 Zac Bowden 透露，他曾看过一些内部文件，其中详细介绍了微软一直在考虑推出“家庭版” Windows 365，即父母可以在孩子的 Cloud PC 上辅导功课或一起玩视频游戏等。
     </p>
     <p style="text-align:justify;">
      而关于“家庭版” Windows 365 的定价，Zac Bowden 表示：“根据我看到的文档，其价格为每月 10 美元，但我的消息来源称，这只是一个占位数字，实际价格尚未最终确定。”
     </p>
     <p style="text-align:justify;">
      对此有人猜测，如果未来 Windows 365 “家庭”账户真是每月 10 美元，这显然是一个亏本价格，估计微软是想通过低价去鼓励人们试用 Windows 365。
     </p>
     <p style="text-align:left;">
      <img alt="7791543c28ec8fefb9b9a40c8e678db3.png" src="https://i-blog.csdnimg.cn/blog_migrate/7d053bb669411f3db0c4b43ccee74b0e.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       Windows 上云后，Linux 将成桌面系统最大赢家？
      </strong>
     </p>
     <p style="text-align:justify;">
      大体上来看，微软推出 Windows 365 的意图很很明显：在任何设备上都能从云端体验 Windows PC，以此让每个人都成为 Windows 用户——正如微软所说的：“我们相信云将为未来的工作提供动力，我们绝大多数的客户都选择云技术来增强员工的能力。”
     </p>
     <p style="text-align:justify;">
      当然，本地版本的 Windows 永远不会消失，但谁也说不准，或许几十年后，大多数 Windows 用户都将习惯使用基于云的 Windows，毕竟“能连接 Internet”+“有浏览器”，几乎是现在每台电子设备都具备的条件。
     </p>
     <p style="text-align:justify;">
      那么，如果未来 Windows 完全迁移至云端，届时桌面操作系统将会是谁的天下？对于这个问题，外媒 The Register 认为答案是 Linux：微软作为 Linux 的对手，将 Windows 迁移到云端即放弃了其在桌面操作系统上的竞争，与此同时苹果 macOS 也有上云趋势。
     </p>
     <blockquote>
      <p style="text-align:justify;">
       “那些注重隐私的人、那些希望控制自己电脑上数据的人以及 PC 高级用户，是不愿意上云的。那么这些人将何去何从？没错，就是 Linux，它将是他们的第一选择。这就是 Linux 最终将成为顶级 PC 操作系统的原因。”
      </p>
     </blockquote>
     <p style="text-align:justify;">
      但对此，诸多网友并不认同：“在我看来，Linux 永远不会在桌面上占据主导地位。人们更多关心的是应用程序，操作系统只是“推动者”。虽然很抱歉，但我还是想说，这就是几十年来这么多人远离 Linux 的原因之一。Linux 开发人员需要加强他们的最终用户程序设计，不要只关注操作系统。”
     </p>
     <p style="text-align:justify;">
      那么对于这件事，你又有什么看法呢？
     </p>
