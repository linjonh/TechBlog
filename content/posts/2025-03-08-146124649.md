    <h2 id="ZsFyj">
     运行环境
    </h2>
    <p id="ube1ca88c">
     智慧医药系统运行环境如下：
    </p>
    <ul>
     <li id="ub1d2db71">
      前端： HTML+CSS
     </li>
     <li id="u772f5325">
      后端：Java + AIGC+Deepseek
     </li>
     <li id="uec3d6e6e">
      IDE工具：IDEA
     </li>
     <li id="u1f3b2b9d">
      技术栈：Springboot + HTML+CSS + MySQL
     </li>
    </ul>
    <h2 id="exiNC">
     主要角色
    </h2>
    <p id="u5909fe54">
     智慧医药系统主要分为两个角色。
    </p>
    <ul>
     <li id="ub4c595c5">
      游客
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="u5e9b5bf0">
        尚未进行注册和登录。具备登录注册、疾病、药品的搜索、查询详情权限。
       </li>
      </ul>
     </li>
    </ul>
    <ul>
     <li id="u2e8023af">
      用户
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="u088c9863">
        除游客权限外，还具备登录、个人资料的修改、登陆密码修改、意见反馈、智能医生咨询的权限。
       </li>
      </ul>
     </li>
    </ul>
    <ul>
     <li id="u0cc96037">
      管理员
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="u40c05dbd">
        除用户的权限外，还具备疾病管理、药品管理、反馈管理等权限。
       </li>
      </ul>
     </li>
    </ul>
    <h2 id="R40oc">
     主要功能
    </h2>
    <p id="u59eb474e">
     智慧医药系统主要功能如下。
    </p>
    <ul>
     <li id="u79759717">
      游客
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="ua636f360">
        登录注册：注册成为系统用户
       </li>
       <li id="u4f836806">
        系统主页：浏览系统主页、疾病、药品的搜索
       </li>
      </ul>
     </li>
    </ul>
    <ul>
     <li id="udcc3fb01">
      用户
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="ub1b6291a">
        个人资料：修改用户的个人资料
       </li>
       <li id="ufb2b1634">
        系统反馈：提交用户反馈
       </li>
       <li id="u244cd24f">
        智能医生：集成 DeepSeek，智能问诊功能
       </li>
      </ul>
     </li>
    </ul>
    <ul>
     <li id="u3d15abdb">
      管理员
     </li>
    </ul>
    <ul>
     <li>
      <ul>
       <li id="u2195f29d">
        个人资料：修改个人资料
       </li>
       <li id="u6c1bec78">
        系统反馈：提交反馈意见
       </li>
       <li id="u04e302b0">
        智能医生：与智能医生进行交流聊天
       </li>
       <li id="u77b4a9ec">
        疾病管理：发布疾病、编辑疾病、删除药品等
       </li>
       <li id="ud94bd1f9">
        药品管理：发布药品、编辑药品、关联疾病、删除药品等
       </li>
       <li id="u0abea4f0">
        反馈管理：管理用户提交的反馈
       </li>
      </ul>
     </li>
    </ul>
