     EL与 JSTL 实践
    </h4>
    <hr/>
    <h4>
     <a id="_ELExpression_Language_4">
     </a>
     <strong>
      一. EL（Expression Language）
     </strong>
    </h4>
    <p>
     EL（表达式语言）是 JSP 2.0 中引入的一种简单的脚本语言，用于在 JSP 页面中简化数据的访问和显示。它通过一种类似于 JavaScript 的语法，允许开发者在 JSP 页面中直接访问 JavaBean 的属性、集合、甚至是 Java 类的静态字段和方法。
    </p>
    <hr/>
    <h5>
     <a id="1_EL__10">
     </a>
     <strong>
      1、 EL 的基本语法
     </strong>
    </h5>
    <p>
     EL 表达式的语法格式为
     <code>
      ${}
     </code>
     ，例如：
    </p>
    <pre><code class="prism language-jsp">${requestScope.userName}  
</code></pre>
    <h5>
     <a id="2_EL__18">
     </a>
     <strong>
      2、 EL 的变量
     </strong>
    </h5>
    <p>
     EL 变量是自动创建的，您无需显式声明。以下是一些常见的变量：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        隐式变量
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         pageScope
        </code>
        ：当前页面的作用域。
       </li>
       <li>
        <code>
         requestScope
        </code>
        ：请求作用域。
       </li>
       <li>
        <code>
         sessionScope
        </code>
        ：会话作用域。
       </li>
       <li>
        <code>
         applicationScope
        </code>
        ：应用作用域。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        用户定义变量
       </strong>
       ：
       <br/>
       使用 JSTL 的
       <code>
        &lt;c:set&gt;
       </code>
       标签可以自定义变量，例如：
      </p>
      <pre><code class="prism language-jsp">&lt;c:set var="title" value="EL 示例" scope="page"/&gt;  
${title}  
</code></pre>
     </li>
    </ol>
    <hr/>
    <h5>
     <a id="3EL__39">
     </a>
     <strong>
      3、EL 的操作符
     </strong>
    </h5>
    <p>
     EL 支持多种操作符，包括：
    </p>
    <ul>
     <li>
      <strong>
       算术操作符
      </strong>
      ：
      <code>
       +
      </code>
      ,
      <code>
       -
      </code>
      ,
      <code>
       *
      </code>
      ,
      <code>
       /
      </code>
      ,
      <code>
       %
      </code>
      ,
      <code>
       div
      </code>
      ,
      <code>
       mul
      </code>
      。
     </li>
     <li>
      <strong>
       比较操作符
      </strong>
      ：
      <code>
       ==
      </code>
      ,
      <code>
       eq
      </code>
      ,
      <code>
       !=
      </code>
      ,
      <code>
       neq
      </code>
      ,
      <code>
       &gt;
      </code>
      ,
      <code>
       gt
      </code>
      ,
      <code>
       &lt;
      </code>
      ,
      <code>
       lt
      </code>
      。
     </li>
     <li>
      <strong>
       逻辑操作符
      </strong>
      ：
      <code>
       &amp;&amp;
      </code>
      ,
      <code>
       and
      </code>
      ,
      <code>
       ||
      </code>
      ,
      <code>
       or
      </code>
      ,
      <code>
       not
      </code>
      ,
      <code>
       !
      </code>
      。
     </li>
     <li>
      <strong>
       空检查操作符
      </strong>
      ：
      <code>
       empty
      </code>
      （检查对象是否为空）。
     </li>
    </ul>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-jsp">${10 + 5}  &lt;!-- 算术操作 --&gt;  
${user.age &gt; 18}  &lt;!-- 比较操作 --&gt;  
${!empty user.name}  &lt;!-- 空检查 --&gt;  
</code></pre>
    <hr/>
    <h5>
     <a id="4EL__58">
     </a>
     <strong>
      4、EL 的使用场景
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        访问 JavaBean 的属性
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;jsp:useBean id="user" class="com.example.User" scope="request"/&gt;  
用户名称：${user.name}  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        遍历集合
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;c:forEach items="${users}" var="user"&gt;  
    ${user.name}  
&lt;/c:forEach&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        条件判断
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;c:if test="${user.age &gt; 18}"&gt;  
    成人用户  
&lt;/c:if&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        格式化输出
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:formatDate value="${user.birthday}" pattern="yyyy-MM-dd"/&gt;  
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_JSTLJavaServer_Pages_Standard_Tag_Library_91">
     </a>
     <strong>
      二. JSTL（JavaServer Pages Standard Tag Library）
     </strong>
    </h4>
    <p>
     JSTL 是 Java 社区为了简化 JSP 页面开发而推出的一组标准标签库。它提供了多种功能标签，涵盖了常见的 Web 开发需求。
    </p>
    <hr/>
    <h5>
     <a id="1_JSTL__97">
     </a>
     <strong>
      1、 JSTL 的核心标签库
     </strong>
    </h5>
    <p>
     JSTL 分为四个主要模块：
    </p>
    <ol>
     <li>
      <strong>
       核心标签库
      </strong>
      ：用于流程控制、集合操作、数据处理等。
     </li>
     <li>
      <strong>
       格式化标签库
      </strong>
      ：用于日期、时间、数字等格式化。
     </li>
     <li>
      <strong>
       数据库标签库
      </strong>
      ：用于简化 JDBC 操作。
     </li>
     <li>
      <strong>
       XML 标签库
      </strong>
      ：用于 XML 数据的处理。
     </li>
    </ol>
    <h6>
     <a id="core_106">
     </a>
     <strong>
      核心标签库（core）
     </strong>
    </h6>
    <p>
     以下是核心标签库中常用的标签：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        &lt;c:out&gt;
       </strong>
       ：
       <br/>
       用于输出数据，支持 HTML 转义。
      </p>
      <pre><code class="prism language-jsp">&lt;c:out value="${user.name}" default="未知"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:forEach&gt;
       </strong>
       ：
       <br/>
       用于遍历集合或数组。
      </p>
      <pre><code class="prism language-jsp">&lt;c:forEach items="${users}" var="user" varStatus="status"&gt;  
    ${status.index}: ${user.name}  
&lt;/c:forEach&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:forTokens&gt;
       </strong>
       ：
       <br/>
       用于遍历字符串标记。
      </p>
      <pre><code class="prism language-jsp">&lt;c:forTokens items="a,b,c" delims=","&gt;  
    ${token}  
&lt;/c:forTokens&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:if&gt;
       </strong>
       ：
       <br/>
       用于条件判断。
      </p>
      <pre><code class="prism language-jsp">&lt;c:if test="${user.age &gt; 18}"&gt;  
    成人用户  
&lt;/c:if&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:choose&gt;
       </strong>
       ：
       <br/>
       用于多分支条件判断。
      </p>
      <pre><code class="prism language-jsp">&lt;c:choose&gt;  
    &lt;c:when test="${user.age &gt; 18}"&gt;成人用户&lt;/c:when&gt;  
    &lt;c:otherwise&gt;未成年用户&lt;/c:otherwise&gt;  
&lt;/c:choose&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:set&gt;
       </strong>
       ：
       <br/>
       用于设置变量。
      </p>
      <pre><code class="prism language-jsp">&lt;c:set var="title" value="JSTL 示例" scope="page"/&gt;  
${title}  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:remove&gt;
       </strong>
       ：
       <br/>
       用于删除变量。
      </p>
      <pre><code class="prism language-jsp">&lt;c:remove var="title" scope="page"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:import&gt;
       </strong>
       ：
       <br/>
       用于导入其他资源（如 HTML 文件、JSP 页面）。
      </p>
      <pre><code class="prism language-jsp">&lt;c:import url="http://example.com/header.html"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:param&gt;
       </strong>
       ：
       <br/>
       用于在导入资源时传递参数。
      </p>
      <pre><code class="prism language-jsp">&lt;c:import url="detail.jsp"&gt;  
    &lt;c:param name="id" value="123"/&gt;  
&lt;/c:import&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        &lt;c:redirect&gt;
       </strong>
       ：
       <br/>
       用于页面跳转。
      </p>
      <pre><code class="prism language-jsp">&lt;c:redirect url="/login.jsp"/&gt;  
</code></pre>
     </li>
    </ol>
    <hr/>
    <h6>
     <a id="fmt_194">
     </a>
     <strong>
      格式化标签库（fmt）
     </strong>
    </h6>
    <ol>
     <li>
      <p>
       <strong>
        fmt:setLocale
       </strong>
       ：
       <br/>
       设置Locale。
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:setLocale value="zh_CN"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        fmt:bundle
       </strong>
       ：
       <br/>
       加载资源文件。
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:bundle basename="messages"&gt;  
    &lt;fmt:message key="welcome.message"/&gt;  
&lt;/fmt:bundle&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        fmt:formatDate
       </strong>
       ：
       <br/>
       格式化日期。
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:formatDate value="${user.birthday}" pattern="yyyy-MM-dd"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        fmt:formatNumber
       </strong>
       ：
       <br/>
       格式化数字。
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:formatNumber value="12345.678" pattern="#,###.##"/&gt;  
</code></pre>
     </li>
    </ol>
    <hr/>
    <h6>
     <a id="sql_228">
     </a>
     <strong>
      数据库标签库（sql）
     </strong>
    </h6>
    <ol>
     <li>
      <p>
       <strong>
        sql:setDataSource
       </strong>
       ：
       <br/>
       设置数据源。
      </p>
      <pre><code class="prism language-jsp">&lt;sql:setDataSource var="dataSource"   
                   driver="com.mysql.cj.jdbc.Driver"   
                   url="jdbc:mysql://localhost:3306/testdb"  
                   user="root"   
                   password="password"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        sql:query
       </strong>
       ：
       <br/>
       执行查询。
      </p>
      <pre><code class="prism language-jsp">&lt;sql:query dataSource="${dataSource}" sql="SELECT * FROM users"&gt;  
    ${users}  
&lt;/sql:query&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        sql:update
       </strong>
       ：
       <br/>
       执行更新（插入、删除、修改）。
      </p>
      <pre><code class="prism language-jsp">&lt;sql:update dataSource="${dataSource}" sql="INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com')"/&gt;  
</code></pre>
     </li>
    </ol>
    <hr/>
    <h5>
     <a id="2_JSTL__259">
     </a>
     <strong>
      2、 JSTL 的使用场景
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        简化流程控制
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;c:forEach items="${users}" var="user"&gt;  
    &lt;p&gt;用户 ${user.name}&lt;/p&gt;  
&lt;/c:forEach&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        格式化数据
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:formatDate value="${user.birthday}" pattern="yyyy-MM-dd"/&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        数据库操作
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;sql:query var="users" sql="SELECT * FROM users"&gt;  
    &lt;c:forEach items="${users.rows}" var="user"&gt;  
        ${user.name}  
    &lt;/c:forEach&gt;  
&lt;/sql:query&gt;  
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        国际化
       </strong>
       ：
      </p>
      <pre><code class="prism language-jsp">&lt;fmt:setLocale value="${param.locale}"/&gt;  
&lt;fmt:bundle basename="messages"&gt;  
    &lt;h1&gt;&lt;fmt:message key="welcome.message"/&gt;&lt;/h1&gt;  
&lt;/fmt:bundle&gt;  
</code></pre>
     </li>
    </ol>

    <h5>
     <a id="2_JSP__341">
     </a>
     <strong>
      2、 JSP 页面实现
     </strong>
    </h5>
    <p>
     以下是一个完整的 JSP 页面示例，展示数据显示和分页功能。
    </p>
    <pre><code class="prism language-jsp">&lt;%@ page language="java" contentType="text/html; charset=UTF-8"  
         pageEncoding="UTF-8" import="java.util.*" %&gt;  
&lt;%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c" %&gt;  
&lt;%@ taglib uri="http://java.sun.com/jsp/jstl/fmt" prefix="fmt" %&gt;  
&lt;%@ taglib uri="http://java.sun.com/jsp/jstl/sql" prefix="sql" %&gt;  

&lt;html&gt;  
&lt;head&gt;  
    &lt;title&gt;用户管理 - 数据显示与分页&lt;/title&gt;  
&lt;/head&gt;  
&lt;body&gt;

    &lt;%  
        // 定义每页显示的记录数  
        int pageSize = 5;  

        // 获取当前页码（默认为第 1 页）  
        int currentPage = 1;  
        if (request.getParameter("currentPage") != null) {  
            currentPage = Integer.parseInt(request.getParameter("currentPage"));  
        }  

        // 总记录数  
        int totalRecords = 0;  

        // 计算总记录数  
        try {  
            Class.forName("com.mysql.cj.jdbc.Driver");  
            Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/testdb", "root", "password");  
            Statement stmt = conn.createStatement();  
            ResultSet rs = stmt.executeQuery("SELECT COUNT(*) FROM users");  
            if (rs.next()) {  
                totalRecords = rs.getInt(1);  
            }  
            rs.close();  
            stmt.close();  
            conn.close();  
        } catch (SQLException e) {  
            e.printStackTrace();  
        }  
    %&gt;  

    &lt;c:set var="pageSize" value="&lt;%=pageSize%&gt;"/&gt;  
    &lt;c:set var="currentPage" value="&lt;%=currentPage%&gt;"/&gt;  
    &lt;c:set var="totalRecords" value="&lt;%=totalRecords%&gt;"/&gt;  

    &lt;c:if test="${totalRecords &gt; 0}"&gt;  
        &lt;sql:setDataSource var="dataSource"  
                          driver="com.mysql.cj.jdbc.Driver"  
                          url="jdbc:mysql://localhost:3306/testdb"  
                          user="root"  
                          password="password"/&gt;  

        &lt;sql:query dataSource="${dataSource}" sql="SELECT * FROM users LIMIT ${currentPage-1}, ${pageSize}" var="users"/&gt;  

        &lt;div&gt;  
            &lt;h2&gt;用户列表（第 ${currentPage} 页 / 共 ${totalRecords} 条记录）&lt;/h2&gt;

                    &lt;th&gt;用户姓名&lt;/th&gt;  
                    &lt;th&gt;用户邮箱&lt;/th&gt;  
                &lt;/tr&gt;  
                &lt;c:forEach items="${users.rows}" var="user" varStatus="status"&gt;  
                    &lt;tr&gt;  
                        &lt;td&gt;${status.index + 1}&lt;/td&gt;  
                        &lt;td&gt;${user.name}&lt;/td&gt;  
                        &lt;td&gt;${user.email}&lt;/td&gt;  
                    &lt;/tr&gt;  
                &lt;/c:forEach&gt;  
            &lt;/table&gt;  
        &lt;/div&gt;  

        &lt;!-- 分页导航 --&gt;  
        &lt;div style="margin-top: 20px;"&gt;  
            &lt;c:if test="${currentPage &gt; 1}"&gt;  
                &lt;a href="?currentPage=${currentPage-1}"&gt;上一页&lt;/a&gt;  
            &lt;/c:if&gt;  
            &lt;c:forEach begin="1" end="${totalRecords / pageSize + 1}" var="page"&gt;  
                &lt;c:choose&gt;  
                    &lt;c:when test="${page == currentPage}"&gt;  
                        &lt;span style="color: red;"&gt;${page}&lt;/span&gt;  
                    &lt;/c:when&gt;  
                    &lt;c:otherwise&gt;  
                        &lt;a href="?currentPage=${page}"&gt;${page}&lt;/a&gt;  
                    &lt;/c:otherwise&gt;  
                &lt;/c:choose&gt;  
            &lt;/c:forEach&gt;  
            &lt;c:if test="${currentPage &lt; totalRecords / pageSize + 1}"&gt;  
                &lt;a href="?currentPage=${currentPage+1}"&gt;下一页&lt;/a&gt;  
            &lt;/c:if&gt;  
        &lt;/div&gt;  
    &lt;/c:if&gt;
