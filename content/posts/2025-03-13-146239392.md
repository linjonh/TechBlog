     题目思路:
    </h3>
    <blockquote>
     <p>
      本题解题思路如下:
     </p>
     <p>
      滑动窗口!双指针一个left,一个right,同向向后滑动即可.窗口大于x就缩紧left向后移动,窗口小于x就扩大right向后移动.注意,left的下标从0开始,输出位置时要加1,right表示的是和left的相对距离,所以输出时不用加1.思路图示如下:
     </p>
     <p>
      <img alt="" height="255" src="https://i-blog.csdnimg.cn/direct/9b68cbc3ee3a440e8663c3a957ff3e2d.png" width="400"/>
     </p>
    </blockquote>
    <hr/>
    <h3 id="%E8%A7%A3%E9%A2%98%E4%BB%A3%E7%A0%81%3A" name="%E8%A7%A3%E9%A2%98%E4%BB%A3%E7%A0%81%3A">
     解题代码:
    </h3>
    <blockquote>
     <p>
      本题解题代码如下:
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
#include&lt;vector&gt;

using namespace std;
int main()
{
    //滑动窗口,同向双指针
    //输入数据
    int n,x;
    cin&gt;&gt;n&gt;&gt;x;
    vector&lt;int&gt; vi;
    vi.resize(n);
    for(int i=0;i&lt;n;i++)
    {
        cin&gt;&gt;vi[i];
    }
    //开始解题
    int left=0,right=0;
    int min_left=0,min_right= n;
    int sum=0;
    while(left&lt;n &amp;&amp; right&lt;n &amp;&amp; left&lt;=right)
    {
        if(sum&lt;x)
        {
            //扩大窗口
            sum+=vi[right];
            right++;
        }
        else
        {
            //如果新窗口长度比老窗口小,就更新最小区间
            if((right-left) &lt; (min_right-min_left))
            {
                min_right=right;
                min_left=left;
            }
            //缩小窗口
            sum-=vi[left];
            left++;
        }
    }
    
    cout&lt;&lt;min_left+1&lt;&lt;" "&lt;&lt;min_right;
    return 0;
}</code></pre>
    </blockquote>

     题目思路:
    </h3>
    <blockquote>
     <p>
      本题解题思路如下:
     </p>
     <p>
      一开始输入的时候把奇数直接加到sum里,偶数直接push进大根堆,后面循环k次(条件是堆不为空)把堆顶的数据除2,堆顶除2后还是偶数就继续push进堆,如果不是偶数就直接加进sum.循环完毕把堆里剩下的偶数加进sum后输出即可.
     </p>
    </blockquote>
    <hr/>
    <h3 id="%E8%A7%A3%E9%A2%98%E4%BB%A3%E7%A0%81%3A" name="%E8%A7%A3%E9%A2%98%E4%BB%A3%E7%A0%81%3A" style="background-color:transparent">
     解题代码:
    </h3>
    <blockquote>
     <p>
      本题解题代码如下:
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
#include&lt;queue&gt;
using namespace std;

int main()
{
    int n,k,tmp;
    long long sum;
    cin&gt;&gt;n&gt;&gt;k;
    priority_queue&lt;int&gt; pq;
    for(int i=0;i&lt;n;i++)
    {
        cin&gt;&gt;tmp;
        if(tmp%2==0)
            pq.push(tmp);
        else
            sum+=tmp;
    }
    for(int i=0;i&lt;k;i++)
    {
        if(pq.empty())
            break;
        tmp = pq.top()/2;
        pq.pop();
        if(tmp%2==0)
            pq.push(tmp);
        else
            sum+=tmp;
    }
    while(!pq.empty())
    {
        sum+=pq.top();
        pq.pop();
    }
    cout&lt;&lt;sum;
    return 0;
}</code></pre>
    </blockquote>
