    <p>
     1.获取公众号的账号和登录密码，并关注公众号；
     <br/>
     左侧菜单-接口权限，找到网页授权，看此项是否‘已获得’，如未获得，可能是公众号未认证，或者公众号不是服务号，而是订阅号，则需要注销当前公众号，重新申请已认证的服务号；
     <br/>
     简单看看其他接口是否开通了，如支付，如需要支付的还需要申请开通支付权限；
     <br/>
     2.进入公众号-左侧菜单-人员设置：先把自己设置成【长期运营人员】，此处需要管理员扫码确认，还需要自己微信上面‘公众平台安全助手’上面点击按钮确认；
     <br/>
     3.左侧菜单-基本配置：启用开发人员设置，启用appSecret，此处需要管理员扫码，扫码成功后，
     <br/>
     需要输入公众号登录密码，图片验证码，确认后出现AppSecret，注意先不要关闭弹框；
     <br/>
     4.将AppID和AppSecret复制到本地文件中保存，以后项目授权配置需要；
     <br/>
     5.接着设置ip白名单，加入服务器ip，现在可以使用长期运营者扫码确认了；
     <br/>
     6.点击右上角的头像-功能设置，进入后分别设置业务域名、js接口安全域名、网页授权域名；
     <br/>
     注意，设置前请先将对应的txt文件放入服务器，让域名+文件名.txt可以访问（自己先在浏览器访问一下，不能访问则之前的三个域名设置不能成功）；
     <br/>
     7.左侧菜单-开发者工具-点击web开发者工具，点击绑定开发者微信号，输入自己的微信号，同意绑定，需要本人在微信上面‘公众平台安全助手’上面点击按钮确认；
     <br/>
     8.左侧菜单-新的功能，点击进入，在广告与服务中找到‘模板消息’，点击开通，选择主行业，副行业，填写申请理由：
     <br/>
     如：为搭建业务员、客户、驾驶员等之间的便利沟通，各自及时掌握订单的动态；让企业更好的为用户提供服务，特申请使用微信模板消息！
     <br/>
     点击提交申请，等待2-3个工作日即可，时间可能会更早，看微信那边操作；
     <br/>
     9.左侧菜单-小程序管理：点击开通小程序：（注意：公众号可关联10个“同主体或关联主体”的小程序，3个“非同主体”小程序；主体的解释不是很清楚，
     <br/>
     我理解的大概意思就是注册公众号人的身份证信息吧，比如公众号和小程序都是用的一个人的信息注册的表示同主体；如需更专业的就百度或者问微信了）
     <br/>
     9.1 如果没有小程序，则可以点击‘快速注册并认证小程序’，这一操作注册的小程序使用微信公众号的资质，不会再支付认证费用，如果没有公众号，单独开通小程序，是需要认证的，即给300认证费/年；
     <br/>
     这一步按照之后的操作，此处不做介绍，主要是忘记操作了，一点点麻烦而已，似乎还需要管理员的协助；
     <br/>
     9.2 点击‘关联小程序’，这一步表示你已经有小程序了(此处正好要操作，就记录一下步骤)；
     <br/>
     需要管理员扫码确认，然后输入小程序的AppID，即可以关联，一般情况是不需要再做关联确认；
     <br/>
     9.3 找服务商开发小程序（不作说明）；
     <br/>
     10.左侧菜单-自定义菜单，添加一个菜单，输入名称，菜单内容可以选择网页和小程序，点击小程序会让你选择哪个小程序，然后可以配置小程序的地址，即可在公众号上面点击菜单链接到对应的小程序；
    </p>
