    <h3>
     <a id="_0">
     </a>
     前言
    </h3>
    <blockquote>
     <p>
      当你在使用微信小程序时，是否曾经遇到过左滑操作是如此的顺畅，让你想起了
      <strong>
       iPhone
      </strong>
      上的滑动操作？左滑操作在微信小程序中已经成为了一个常见的交互方式，它能够帮助用户快速地执行某些操作，提高了用户的使用体验。那么，你是否好奇这个左滑操作是如何实现的呢？本文将会为你揭秘微信小程序左滑操作背后的技术原理。
     </p>
    </blockquote>
    <p>
     <strong>
      实现效果：
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a6a6ce466256b8ca769ee0fee87f0764.gif#pic_center"/>
    </p>
    <h3>
     <a id="_9">
     </a>
     实现思路
    </h3>
    <ol>
     <li>
      首先我们需要为容器添加
      <code>
       bindtouchstart
      </code>
      (手指触摸屏幕触发)、
      <code>
       bindtouchmove
      </code>
      (手指在屏幕上滑动的时候连续触发)；
     </li>
     <li>
      <code>
       bindtouchstart
      </code>
      事件第一次触发时就是手指点击屏幕的开始，也就是起点x坐标；其中，我们需要清除左滑的所有内容，只操作
      <code>
       isTouchMove
      </code>
      值为
      <code>
       true
      </code>
      的；
     </li>
     <li>
      <code>
       bindtouchmove
      </code>
      事件中开始滑动，记录开始滑动的
      <code>
       x
      </code>
      、
      <code>
       y
      </code>
      坐标、滑动时变化的
      <code>
       x
      </code>
      、
      <code>
       y
      </code>
      坐标以及滑动的角度，通过
      <code>
       Math.abs
      </code>
      函数返回指定数字
      <code>
       x
      </code>
      的绝对值；
     </li>
     <li>
      计算滑动的角度，
      <code>
       start
      </code>
      为起点坐标，
      <code>
       end
      </code>
      为终点坐标，通过
      <code>
       Math.atan()
      </code>
      函数返回一个数值的反正切（以弧度为单位）；
     </li>
     <li>
      为左滑出来的内容添加点击事件操作，在js中触发点击事件
      <code>
       passEvent
      </code>
      (审核通过)、
      <code>
       notGoEvent
      </code>
      (审核不通过)执行操作。
     </li>
    </ol>
