    <p>
     WebStorm 是开发人员最常使用的一种网页开发工具，它是 JetBrains 公司旗下的一款网页开发工具，其功能非常强大，支持各种前端和 JavaScript 库的代码补全，被广大开发者誉为 Web 前端开发神器、最强大的 HTML5 编辑器、最智能的 JavaScript IDE 等。本文将对 WebStorm 的下载、安装及使用进行讲解。
    </p>
    <h2>
     <a id="WebStorm_3">
     </a>
     一、WebStorm的下载
    </h2>
    <p>
     点击链接
     <a href="https://www.jetbrains.com.cn/webstorm/" rel="nofollow">
      https://www.jetbrains.com.cn/webstorm/
     </a>
     ，进入 WebStorm 官网，单击页面中的
     <strong>
      <code>
       下载
      </code>
     </strong>
     按钮，即可下载 WebStorm 的安装文件，如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5da37c8e30924ea8b0acfce8bca1978e.png">
      <br/>
      下载完成的 WebStorm 安装文件如下图所示，其命名格式为
      <strong>
       <code>
        "WebStorm-版本号.exe"
       </code>
       。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6bedab28f66f4db794dabca3f3a7a103.png">
       <br/>
       笔者在编写本文时，WebStorm 的最新版本是
       <strong>
        <code>
         2024.2.4
        </code>
        ，
       </strong>
       该版本会随着时间的推移不断更新，读者在使用时，不用纠结版本的变化，直接下载最新版本即可。
      </img>
     </img>

    <h2>
     <a id="WebStorm_9">
     </a>
     二、WebStorm的安装
    </h2>
    <p>
     安装 WebStorm 开发工具的步骤如下：
    </p>
    <ol>
     <li>
      双击下载完成的 WebStorm 安装文件，开始安装 WebStorm，如下图所示，单击
      <strong>
       <code>
        下一步
       </code>
      </strong>
      按钮。
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/2df28c3db7f2454988fed9d011198adb.png" width="60%"/>
     </li>
     <li>
      进入路径设置对话框，如下图所示，在该对话框中单击
      <strong>
       <code>
        "浏览(B)..."
       </code>
      </strong>
      按钮去选择 WebStorm 的安装路径，然后单击
      <strong>
       <code>
        下一步(N)
       </code>
      </strong>
      按钮。
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/b0d63c77532f4fd09e89942222f2a0f5.png" width="60%"/>
     </li>
     <li>
      进入安装配置对话框，在该对话框中首先创建桌面快捷方式，并添加系统环境变量，然后创建右键菜单快捷方式，最后单击
      <strong>
       <code>
        下一步(N)
       </code>
      </strong>
      按钮，如下图所示：
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/1281c0a006a84708baf55305b4a48e21.png" width="60%"/>
     </li>
     <li>
      进入确认安装对话框，如下图所示，直接单击
      <strong>
       <code>
        安装(I)
       </code>
      </strong>
      按钮开始安装。
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/d10d808c11e44622afb2a4fb49f24b9b.png" width="60%"/>
     </li>
     <li>
      此时会进入 WebStorm 的安装对话框，该对话框中显示当前的安装进度，如下图所示：
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/8f665f123b7742409f1d77fa337e529b.png" width="60%"/>
     </li>
     <li>
      安装完成后自动进入安装完成对话框，单击
      <strong>
       <code>
        完成(F)
       </code>
      </strong>
      按钮即可完成 WebStorm 的安装，如下图所示：
      <br/>
      <img src="https://i-blog.csdnimg.cn/direct/3e3a6ef7b4494637b84a4c7e4f8f3942.png" width="60%">
       <br/>
       安装完成对话框中有两个单选按钮，其中
       <strong>
        <code>
         "是，立即重新启动(Y)"
        </code>
       </strong>
       表示立即重启，而
       <strong>
        <code>
         "否，我会在之后重新启动(N)"
        </code>
       </strong>
       表示稍后重启，用户可以根据自己的实际情况进行选择，默认选中的是
       <strong>
        <code>
         "否，我会在之后重新启动(N)"
        </code>
       </strong>
       。
      </img>
     </li>
    </ol>

    <h3>
     <a id="81_WebStormNodejs_52">
     </a>
     8.1 使用WebStorm创建第一个Node.js程序
    </h3>
    <p>
     使用 WebStorm 创建 Node.js 程序的步骤如下：
    </p>
    <ol>
     <li>
      <p>
       在 WebStrom 的欢迎对话框中，单击左侧的 Projects，然后单击右侧的 New Project 按钮，如下图所示：
       <br/>
       <img src="https://i-blog.csdnimg.cn/direct/5f95c89a0048450887603540a75ceb66.png" width="70%"/>
      </p>
     </li>
     <li>
      <p>
       弹出 New Project 对话框，该对话框的左侧显示的是可以创建的项目类型，右侧是关于项目的一些配置信息，这里选择左侧的 Node.js，然后在右侧的 Location 文本框中输入或者选择项目的位置，单击 Create 按钮，如下图所示：
       <br/>
       <img src="https://i-blog.csdnimg.cn/direct/9a6f24f4d7f5412ba58d26c275a5d829.png" width="70%"/>
       <br/>
       <strong>
        注意：
       </strong>
       在创建 Node.js 项目时，设置的 Location 目录中不能含有大写字母，否则会出现错误提示，并且 Create 按钮不可用。
      </p>
     </li>
     <li>
      <p>
       创建完的 Node.js 项目如下图所示，该项目中默认包含一个 package.json 项目描述文件，以及 Node.js 依赖包。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/09d1c705ddc34bdf9dfa633a9320b0a6.png"/>
      </p>
     </li>
     <li>
      <p>
       在创建的 Node.js 项目的左侧目录结构中单击鼠标右键，在弹出的快捷菜单中选择 New→JavaScript File 命令，如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5dd1c15b10464e42b688da07bfd953e8.png"/>
      </p>
     </li>
     <li>
      <p>
       弹出 New JavaScript file 对话框，在该对话框中输入文件名，这里输入 index，按 Enter 键即可，如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4e5af2f528d946c085c2602b7a2e73c1.png"/>
      </p>
     </li>
     <li>
      <p>
       在创建的 index.js 文件中输入以下代码：
      </p>
      <pre><code class="prism language-javascript"><span class="token comment">//加载http模块</span>
<span class="token keyword">var</span> http <span class="token operator">=</span> <span class="token function">require</span><span class="token punctuation">(</span><span class="token string">'http'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">"请打开浏览器，输入地址 http://127.0.0.1:3000/"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token comment">//创建http服务器，监听网址127.0.0.1 端口号3000</span>
http<span class="token punctuation">.</span><span class="token function">createServer</span><span class="token punctuation">(</span><span class="token keyword">function</span><span class="token punctuation">(</span><span class="token parameter">req<span class="token punctuation">,</span> res</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    res<span class="token punctuation">.</span><span class="token function">end</span><span class="token punctuation">(</span><span class="token string">'Hello World!'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">"right"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">)</span><span class="token punctuation">.</span><span class="token function">listen</span><span class="token punctuation">(</span><span class="token number">3000</span><span class="token punctuation">,</span><span class="token string">'127.0.0.1'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
      <p>
       上面的代码中，第2行用来加载 http 模块，在 Node.js 程序中，要使用哪个模块，就使用 require 加载该模块；第3行用来在控制台中输出日志提示，其中 console 是 Node.js 中的控制台类，其 log 方法用来输出日志；第5行的 http.createServer 用来创建一个 http 服务器，该方法中定义了一个 JavaScript 函数，用来处理网页请求和响应，其中有两个参数，req 表示请求，res 表示响应，该函数中使用 res.end 方法在页面上输出要显示的文字信息，并使用 console.log 方法在控制台中输出日志提示；最后一行的 listen 方法用来设置要监听的网址以及端口号。输入完成的效果如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/98680a942f4a4a1a81bd8a42ea972bc2.png"/>
      </p>
     </li>
    </ol>
    <h3>
     <a id="82_WebStormNodejs_79">
     </a>
     8.2 在WebStorm中运行Node.js程序

    <p>

    <p>
     <strong>

      <code>

     </strong>
     命令，即可运行 Node.js 程序，如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dec4ca783e234d0baf3e5bcd71eb977a.png"/>
     <br/>
     <strong>
      ②
     </strong>
