    <h4>
     <a id="Git_93">
     </a>
     Git的常见操作
    </h4>
    <p>
     <strong>
      克隆远程仓库到本地
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone 仓库地址
</code></pre>
    <p>
     <strong>
      拉取最新代码
     </strong>
    </p>
    <p>
     这里是拉取远程仓库最新代码到本地,在仓库下面执行命令
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> pull <span class="token operator">&lt;</span>远程主机名<span class="token operator">&gt;</span> <span class="token operator">&lt;</span>远程分支名<span class="token operator">&gt;</span>:<span class="token operator">&lt;</span>本地分支名<span class="token operator">&gt;</span>
</code></pre>
    <p>
     可以简写为：
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> pull
</code></pre>
    <p>
     <strong>
      状态查看
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> status
</code></pre>
    <p>
     查看工作区、暂存区状态。
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69b5f39f39862c53a187126daf759439.jpeg"/>
    </p>
    <hr/>
    <p>
     <strong>
      git add添加
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> <span class="token function">add</span> <span class="token punctuation">[</span>file name<span class="token punctuation">]</span>
</code></pre>
    <p>
     将工作区的“新建/修改”添加到暂存区，如下图创建一个good.txt并提交：
    </p>
    <p>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/201d8bab1835b9f39e4a9432ab05e6b3.jpeg"/>
    </p>
    <hr/>
    <p>
     <strong>
      提交文件
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> commit  <span class="token parameter variable">-m</span> <span class="token string">"commit message"</span>  <span class="token punctuation">[</span>file name<span class="token punctuation">]</span>
</code></pre>
    <p>
     将暂存区的内容提交到本地库。
    </p>
    <p>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b9b10146298f9460e96887682647af63.jpeg"/>
    </p>
    <hr/>
    <p>
     <strong>
      提交多个文件(团队协作时谨慎使用)：
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> commit <span class="token parameter variable">-m</span> <span class="token string">"commit message"</span> *
</code></pre>
    <p>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4273cd9b774018ec4a97f769d6b9d40a.jpeg"/>
    </p>
    <hr/>
    <p>
     <strong>
      推送本地代码到远程仓库
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> push 
</code></pre>
    <p>
     当你新增/修改一个文件想要推送到远程时，可以参考如下步骤：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 查看一下状态</span>
<span class="token function">git</span> status

<span class="token comment"># 添加</span>
<span class="token function">git</span> <span class="token function">add</span> <span class="token punctuation">[</span>file name<span class="token punctuation">]</span>

<span class="token comment">#提交</span>
<span class="token function">git</span> commit  <span class="token parameter variable">-m</span> <span class="token string">"commit message"</span>  <span class="token punctuation">[</span>file name<span class="token punctuation">]</span>

<span class="token comment"># 推送</span>
<span class="token function">git</span> push
</code></pre>
