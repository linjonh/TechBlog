    <h2>

     一、简洁版
    </h2>
    <h3>
     <a id="11__4">
     </a>
     1.1 下载并安装
    </h3>
    <p>
     下载：
     <code>
      mysql-8.0.40-1.el7.x86_64.rpm-bundle.tar
     </code>
    </p>
    <p>
     官网地址：
    </p>
    <pre><code class="prism language-shell">https://downloads.mysql.com/archives/community/
</code></pre>
    <p>
     也可以在本文最上方资源绑定里下载。介绍：资源包里是裁剪后的，也是安装mysql 的四个必要安装包，直接安装即可。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/71a1a84efce247a894a16527d7299b41.png"/>
    </p>
    <ol>
     <li>
      <p>
       在安装前需要先卸载 mariadb
      </p>
      <p>
       输入以下指令查看 mariadb
      </p>
      <pre><code class="prism language-shell"><span class="token function">rpm</span> <span class="token parameter variable">-qa</span> <span class="token operator">|</span> <span class="token function">grep</span> mariadb
</code></pre>
      <p>
       卸载以上列出的所有的 mariadb 包
      </p>
      <pre><code class="prism language-shell"><span class="token function">rpm</span> <span class="token parameter variable">-e</span> <span class="token parameter variable">--nodeps</span> 安装包1
<span class="token function">rpm</span> <span class="token parameter variable">-e</span> <span class="token parameter variable">--nodeps</span> 安装包2
</code></pre>
     </li>
     <li>
      <p>
       在/usr/local目录下新建 mysql 文件夹，并将下载的文件拖拽到
       <code>
        /usr/local/mysql
       </code>
       下：
      </p>
      <pre><code class="prism language-shell"><span class="token builtin class-name">cd</span> /usr/local
<span class="token function">mkdir</span> mysql
<span class="token builtin class-name">cd</span> mysql
</code></pre>
     </li>
     <li>
      <p>
       使用以下命令解压
      </p>
      <pre><code class="prism language-shell"><span class="token function">tar</span> <span class="token parameter variable">-zxvf</span> mysql-8.0.40-1.el7.x86_64.rpm-bundle.tar
</code></pre>
     </li>
     <li>
      <p>
       解压后会有很多 *.rpm 的安装包，只需安装 common、libs、client、server 即可（本文最上方资源绑定里只有四个，下载后直接解压安装即可），安装指令如下：
      </p>
      <pre><code class="prism language-shell"><span class="token function">rpm</span> <span class="token parameter variable">-ivh</span> <span class="token parameter variable">--nodeps</span> <span class="token parameter variable">--force</span> mysql-community-common-8.0.40-1.el7.x86_64.rpm

<span class="token function">rpm</span> <span class="token parameter variable">-ivh</span> <span class="token parameter variable">--nodeps</span> <span class="token parameter variable">--force</span> mysql-community-libs-8.0.40-1.el7.x86_64.rpm

<span class="token function">rpm</span> <span class="token parameter variable">-ivh</span> <span class="token parameter variable">--nodeps</span> <span class="token parameter variable">--force</span> mysql-community-client-8.0.40-1.el7.x86_64.rpm

<span class="token function">rpm</span> <span class="token parameter variable">-ivh</span> <span class="token parameter variable">--nodeps</span> <span class="token parameter variable">--force</span> mysql-community-server-8.0.40-1.el7.x86_64.rpm
</code></pre>
     </li>
     <li>
      <p>
       初始化 MySQL
      </p>
      <pre><code class="prism language-shell">mysqld <span class="token parameter variable">--initialize</span>
</code></pre>
     </li>
     <li>
      <p>
       授权防火墙
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 将 MySQL 授权给防火墙</span>
<span class="token function">chown</span> mysql:mysql /var/lib/mysql -R<span class="token punctuation">;</span>
<span class="token comment"># 启动 MySQL 服务</span>
systemctl start mysqld.service<span class="token punctuation">;</span>
<span class="token comment"># 使 MySQL能用</span>
systemctl <span class="token builtin class-name">enable</span> mysqld<span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       查看数据库的初始密码
      </p>
      <pre><code class="prism language-shell"><span class="token function">cat</span> /var/log/mysqld.log <span class="token operator">|</span> <span class="token function">grep</span> password
</code></pre>
      <p>
       <code>
        root@localhost: rzgz0mzU7Z-f
       </code>
       ，冒号之后的就是初始密码。
      </p>
     </li>
     <li>
      <p>
       登录数据库
      </p>
      <pre><code class="prism language-shell">mysql <span class="token parameter variable">-uroot</span> <span class="token parameter variable">-p</span>
</code></pre>
      <p>
       回车，输入初始密码。（莫慌，密码是不显示的）
      </p>
     </li>
     <li>
      <p>
       修改密码为：
       <code>
        123456
       </code>
      </p>
      <pre><code class="prism language-shell">ALTER <span class="token environment constant">USER</span> <span class="token string">'root'</span>@<span class="token string">'localhost'</span> IDENTIFIED WITH mysql_native_password BY <span class="token string">'123456'</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ol>
    <h3>
     <a id="12__98">
     </a>
     1.2 解决三个问题
    </h3>
    <ol>
     <li>
      <p>
       时区问题，登录数据库，输入以下内容。MySQL 以美国时区时间为准，北京时间比美国时间晚8个小时，所以
       <code>
        +8:00
       </code>
       。
      </p>
      <pre><code class="prism language-shell"><span class="token builtin class-name">set</span> global <span class="token assign-left variable">time_zone</span><span class="token operator">=</span><span class="token string">'+8:00'</span><span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       开启远程服务。目的：让主机客户端也能访问 Linux 中的 MySQL
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 创建一个用户， % 是通配符，代表所有用户</span>
create user <span class="token string">'root'</span>@<span class="token string">'%'</span> identified with mysql_native_password by <span class="token string">'123456'</span><span class="token punctuation">;</span>
<span class="token comment"># 授予所有用户权限</span>
grant all privileges on *.* to <span class="token string">'root'</span>@<span class="token string">'%'</span> with grant option<span class="token punctuation">;</span>
<span class="token comment"># 刷新权限</span>
flush privileges<span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       <code>
        exit
       </code>
       退出数据库，开放 3306 端口
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 开放 3306 端口</span>
firewall-cmd <span class="token parameter variable">--zone</span><span class="token operator">=</span>public --add-port<span class="token operator">=</span><span class="token number">3306</span>/tcp <span class="token parameter variable">--permanent</span>
<span class="token comment"># 重新启动防火墙</span>
systemctl restart firewalld.service
<span class="token comment"># 重新载入防火墙配置</span>
firewall-cmd <span class="token parameter variable">--reload</span>
</code></pre>
     </li>
    </ol>
