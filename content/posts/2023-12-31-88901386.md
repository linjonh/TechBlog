    <h2>
     <a id="11__2">
     </a>
     1.1 什么是嵌入式？
    </h2>
    <p>
     <strong>
      嵌入式系统（Embedded System）：
     </strong>
     一种嵌入机械或电气系统内部、具有专一功能和实时计算性能的计算机系统。
     <br/>
     嵌入式开发由传统的单片机开发发展而来，但与传统的单片机开发又有所不同。
    </p>
    <h3>
     <a id="111_5">
     </a>
     1.1.1单片机开发：
    </h3>
    <p>
     传统的
     <strong>
      “单片机开发”
     </strong>
     是直接在硬件系统上烧录程序代码。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2ba729c5bb229f316add00a29c22d293.png#pic_center"/>
    </p>
    <h3>
     <a id="112_8">
     </a>
     1.1.2嵌入式开发：
    </h3>
    <p>
     <strong>
      “嵌入式开发”
     </strong>
     则是由嵌入式操作系统作为中间媒介，嵌入软件层与硬件层，将软硬件进行解耦。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fb590f81ec7b60dadb840a7977745d7d.png#pic_center">
      <br/>
      所谓嵌入式开发，即
      <strong>
       在软件层与硬件层嵌入
       <font color="red">
        操作系统
       </font>
       层
      </strong>
      。
      <br/>
      <br/>
     </img>
    </p>
    <h2>
     <a id="12__12">
     </a>
     1.2 单片机开发与嵌入式开发分区别
    </h2>
    <p>
     单片机开发（也叫
     <strong>
      裸机开发
     </strong>
     ）是根据硬件结构直接将写好的代码烧录到硬件系统上。嵌入式开发则是在硬件上移植嵌入式操作系统，在操作系统上进行应用软件安装。
    </p>
    <h3>
     <a id="121_14">
     </a>
     1.2.1单片机开发
    </h3>
    <p>
     <strong>
      优点：
     </strong>
    </p>
    <ol>
     <li>
      硬件成本较低。
     </li>
    </ol>
    <p>
     <strong>
      缺点：
     </strong>
    </p>
    <ol>
     <li>
      软件开发人员必须懂硬件；
     </li>
     <li>
      软硬件耦合度过高，移植性差；
     </li>
     <li>
      功能性和用户体验较差。
     </li>
    </ol>
    <h3>
     <a id="122_22">
     </a>
     1.2.2嵌入式开发
    </h3>
    <p>
     <strong>
      优点：
     </strong>
    </p>
    <ol>
     <li>
      解决了传统单片机开发软硬件耦合度过高、移植性差等缺陷；
     </li>
     <li>
      具有多线程机制；
     </li>
     <li>
      提供有丰富的网络协议栈；
     </li>
     <li>
      提供了丰富的开源软件和工具。
     </li>
    </ol>
    <p>
     <strong>
      缺点：
     </strong>
    </p>
    <ol>
     <li>
      硬件成本高。
     </li>
    </ol>
    <h2>
     <a id="13__32">
     </a>
     1.3 嵌入式操作系统
    </h2>
    <p>
     所谓嵌入式操作系统即为运行在嵌入式系统上的操作系统。它们通常被设计非常紧凑有效，抛弃了运行在它们之上的特定的应用程序所不需要的各种功能。嵌入式操作系统多数也是即时操作系统。
    </p>
    <h3>
     <a id="131_34">
     </a>
     1.3.1什么是开源？
    </h3>
    <p>
     开源即“开放源代码”分为部分开源和完全开源。
     <br/>
     <br/>
     <strong>
      部分开源：
     </strong>
     <br/>
     可免费试用，源代码部分公开。
     <br/>
     <strong>
      完全开源：
     </strong>
     <br/>
     可免费试用，所有源代码完全公开。
    </p>

     <a id="132_40">
     </a>

    </ol>
    <h2>
     <a id="14__52">
     </a>
     1.4 嵌入式开发需要学习哪些内容？
    </h2>
    <p>
     嵌入式开发人员可分为
     <strong>
      应用软件开发
     </strong>
     和
     <strong>
      系统软件开发
     </strong>
     两类。
    </p>
    <h3>
     <a id="141_54">
     </a>
     1.4.1应用软件开发：
    </h3>
    <ol>
     <li>
      精通一门语言；（C/C++）
     </li>
     <li>
      熟悉一款操作系统；
     </li>
     <li>
      掌握数据结构和算法。
     </li>
    </ol>
    <h3>
     <a id="142_59">
     </a>
     1.4.2系统软件开发：
    </h3>
    <ol>
     <li>
      精通C语言；
     </li>
     <li>
      熟悉汇编语言；
     </li>
     <li>
      理解操作系统的实现；（理解各种接口如何实现、能进行重写）
     </li>
     <li>
      熟悉硬件工作原理。
     </li>
    </ol>
