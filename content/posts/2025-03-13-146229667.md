    <h2>
     1.理解张量
    </h2>
    <h4>
     从简单到复杂逐步理解张量
    </h4>
    <p class="img-center">
     <img alt="" height="355" src="https://i-blog.csdnimg.cn/direct/65d1651e7496441fb3da2b04633f084f.png" width="474"/>
    </p>
    <ul>
     <li>
      <strong>
       标量
      </strong>
      ：可以将标量看作是零维张量，它只有一个数值，比如温度、质量等。例如，今天的气温是 25 摄氏度，这里的 25 就是一个标量。
     </li>
     <li>
      <strong>
       向量
      </strong>
      ：一维张量就是向量，它由一组有序的数值组成，可以表示具有大小和方向的量。比如在平面直角坐标系中，向量 (3, 4) 表示从原点到点 (3, 4) 的有向线段。
     </li>
     <li>
      <strong>
       矩阵
      </strong>
      ：二维张量是矩阵，它是一个由行和列组成的二维数组。矩阵在图像处理、线性代数等领域有广泛应用。例如，一个表示图像的矩阵，每个元素可以代表图像中对应像素的灰度值或颜色值。
     </li>
     <li>
      <p>
       <strong>
        多维张量
       </strong>
       ：当维度超过二维时，就进入了多维张量的范畴。例如，三维张量可以想象成一个立方体，每个元素都有三个索引来确定其位置。在视频数据中，三维张量可以表示一个视频片段，其中三个维度分别可以是时间、高度和宽度。而四维张量则可以在三维张量的基础上，再增加一个维度，比如颜色通道，这样就可以更全面地表示视频数据的信息。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      1.直接创建张量
     </strong>
     ：使用
     <code>
      torch.tensor()
     </code>
     函数可以直接将 Python 列表转换为 PyTorch 张量。
    </p>
    <pre><code class="language-python">import torch

# 创建一个一维张量
tensor_1d = torch.tensor([1, 2, 3, 4, 5])
print("一维张量:")
print(tensor_1d)

# 创建一个二维张量
tensor_2d = torch.tensor([[1, 2, 3], [4, 5, 6]])
print("\n二维张量:")
print(tensor_2d)

# 创建一个三维张量
tensor_3d = torch.tensor([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print("\n三维张量:")
print(tensor_3d)</code></pre>

    <h3>
     2.操作张量
    </h3>
    <h3>
     2.1shape和size
    </h3>
    <p>
     在 PyTorch 中，
     <code>
      shape
     </code>
     和
     <code>
      size()
     </code>
     都与张量（Tensor）的维度信息相关，但它们在使用形式和表现上存在一些差异，下面为你详细介绍：
    </p>
    <p>
     <strong>
      相同点
     </strong>
    </p>
    <p>
     <code>
      shape
     </code>
     和
     <code>
      size()
     </code>
     本质上都用于获取张量各维度的大小，返回的结果是一致的，都能反映出张量的维度信息。以下是一个示例代码：
    </p>
    <pre><code class="language-python">import torch

# 创建一个二维张量
tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])

# 使用 shape 属性获取张量的形状
print("使用 shape 属性:", tensor.shape)

# 使用 size() 方法获取张量的形状
print("使用 size() 方法:", tensor.size())</code></pre>
    <p>
     <strong>
      不同点
     </strong>
    </p>
    <p>
     1. 类型和调用方式
    </p>
    <h5>
    </h5>
    <p>
     <strong>
      <code>
       shape
      </code>
     </strong>
     ：它是张量的一个属性，使用时直接通过点号访问，就像访问对象的普通属性一样。
    </p>
    <pre><code class="language-python">import torch
tensor = torch.tensor([1, 2, 3])
print(tensor.shape)</code></pre>
    <p>
     <strong>
      <code>
       size()
      </code>
     </strong>
     ：这是张量的一个方法，使用时需要在后面加上括号，括号内可以传入参数来获取特定维度的大小。
    </p>
    <pre><code class="language-python">import torch
tensor = torch.tensor([1, 2, 3])
print(tensor.size())</code></pre>
    <p>
     2
     <strong>
      .关于参数
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        shape[0]
       </code>
      </strong>
      ：
      <code>
       shape
      </code>
      是张量的一个属性，它返回一个
      <code>
       torch.Size
      </code>
      对象，该对象类似于元组，
      <code>
       shape[0]
      </code>
      是对这个
      <code>
       torch.Size
      </code>
      对象进行索引操作，获取其第一个元素。
     </li>
     <li>
      <strong>
       <code>
        size(0)
       </code>
      </strong>
      ：
      <code>
       size()
      </code>
      是张量的一个方法，通过传入参数 0 来指定获取第一维的大小
     </li>
    </ul>
    <ul>
     <li>
      <strong>
       <code>
        shape[0]
       </code>
      </strong>
      ：只能获取第一维的大小，如果要获取其他维度的大小，需要通过索引
      <code>
       shape[1]
      </code>
      、
      <code>
       shape[2]
      </code>
      等方式。
     </li>
    </ul>
    <pre><code class="language-python">import torch
tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])
print("第二维的大小 (使用 shape):", tensor.shape[1])</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        size()
       </code>
      </strong>
      ：方法更加灵活，它可以不传入参数，直接返回整个张量的形状信息（返回值类型为
      <code>
       torch.Size
      </code>
      ），也可以传入不同的整数参数来获取指定维度的大小。
     </li>
    </ul>
    <pre><code class="language-python">import torch
tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])
h,w=tensor.size()
print(h,w)</code></pre>
    <p>
     <strong>
      2.2 操作维度
     </strong>
    </p>
    <p>
     <strong>
      <code>
       1.torch.unsqueeze()
      </code>
      方法
     </strong>
    </p>
    <p>
     <code>
      torch.unsqueeze()
     </code>
     函数用于在指定的维度上插入一个新的维度，其语为
     <code>
      torch.unsqueeze(input, dim)
     </code>
     ，其中
     <code>
      input
     </code>
     是输入的张量，
     <code>
      dim
     </code>
     是要插入新维度的位置（索引从 0 开始）
    </p>
    <pre><code class="language-python">import torch

# 创建一个一维张量
tensor_1d = torch.tensor([1, 2, 3])
print("原一维张量形状:", tensor_1d.shape)

# 在第 0 维插入一个新维度
tensor_2d_0 = torch.unsqueeze(tensor_1d, dim=0)
print("在第 0 维插入新维度后的形状:", tensor_2d_0.shape)

# 在第 1 维插入一个新维度
tensor_2d_1 = torch.unsqueeze(tensor_1d, dim=1)
print("在第 1 维插入新维度后的形状:", tensor_2d_1.shape)</code></pre>
    <p>
     2. 在 PyTorch 中，可以使用
     <code>
      None
     </code>
     索引来增加维度，这与
     <code>
      torch.unsqueeze()
     </code>
     效果类似。示例如下
    </p>
    <pre><code class="language-python">import torch

# 创建一个一维张量
tensor_1d = torch.tensor([1, 2, 3])
print("原一维张量形状:", tensor_1d.shape)

# 在第 0 维插入一个新维度
tensor_2d_0 = tensor_1d[None, :]
print("在第 0 维插入新维度后的形状:", tensor_2d_0.shape)

# 在第 1 维插入一个新维度
tensor_2d_1 = tensor_1d[:, None]
print("在第 1 维插入新维度后的形状:", tensor_2d_1.shape)</code></pre>

     </code>

    </p>

    </p>

try:
    # 尝试对第 0 维（大小为 2）进行扩展，会报错

except RuntimeError as e:
    print("错误信息:", e)</code></pre>

     5.
     <strong>
      squeeze
     </strong>
    </p>
    <p>
     在 PyTorch 中，
     <code>
      squeeze()
     </code>
     方法用于移除张量中维度大小为 1 的维度。这在处理一些维度冗余的数据时非常有用，可以让张量的形状更加简洁，便于后续的计算和处理。下面详细介绍
     <code>
      squeeze()
     </code>
     方法的使用、参数和注意事项。

    <pre><code class="language-python"># 形式一：不指定维度，移除所有大小为 1 的维度

    <p>

    </p>
    <pre><code class="language-python">import torch

    </p>

     6.
     <strong>
      permute
     </strong>
    </p>
    <p>
     <code>
      permute
     </code>
     方法用于对张量的维度进行重新排列，也就是改变张量各维度的顺序。这在处理不同维度排列的数据时非常有用，比如在图像处理、深度学习模型的数据输入处理等场景中经常会用到。

     <code>
      permute
     </code>
     方法的基本语法如下：
    </p>

# 交换第 0 维和第 1 维的顺序
permuted_tensor_2d = tensor_2d.permute(1, 0)
print("维度重排后二维张量形状:", permuted_tensor_2d.shape)</code></pre>

     <code>

     ，通过
     <code>
      permute(1, 0)
     </code>
     把第 0 维和第 1 维的顺序进行了交换，重排后的张量形状变为
     <code>
      (3, 2)
     </code>
    </p>

     <code>

    </p>

    <h3>
     2.3 加乘法
    </h3>
    <h4>
     广播机制的规则
    </h4>
    <p>
     广播机制的核心规则是从右向左比较两个张量的形状，对于每一个维度，需要满足以下条件之一：
    </p>
    <ol>
     <li>
      <strong>
       维度大小相等
      </strong>
      ：两个张量在该维度上的大小相同。
     </li>
     <li>
      <strong>
       其中一个维度大小为 1
      </strong>
      ：如果一个张量在某一维度上的大小为 1，而另一个张量在该维度上有非 1 的大小，那
      <strong>
       么大小为 1 的维度会被扩展为与另一个张量该维度大小相同
      </strong>
      。
     </li>
     <li>
      <strong>
       其中一个张量缺少该维度
      </strong>
      ：
      <strong>
       如果一个张量在某一维度上不存在
      </strong>
      （即形状中没有对应位置），则可以将其视为该维度大小为 1，然后按照规则 2 进行扩展。
     </li>
    </ol>
    <p>
     <strong>
      1.加法
     </strong>
    </p>
    <p>
     如果两个张量形状不同，但满足广播机制的规则，PyTorch 会自动进行广播操作，使它们的形状匹配后再进行逐元素相加。广播规则是从右向左比较两个张量的维度，要么维度大小相等，要么其中一个维度大小为 1 或缺少该维度。
    </p>
    <pre><code class="language-python">import torch

# 创建一个二维张量和一个一维张量
tensor_2d = torch.tensor([[1, 2, 3], [4, 5, 6]])
tensor_1d = torch.tensor([1, 1, 1])

# 利用广播机制相加
result = tensor_2d + tensor_1d
print("广播相加结果:")
print(result)</code></pre>
    <p>
     这里
     <code>
      tensor_1d
     </code>
     形状是
     <code>
      (3,)
     </code>
     ，
     <code>
      tensor_2d
     </code>
     形状是
     <code>
      (2, 3)
     </code>
     。
     <code>
      tensor_1d
     </code>
     在第 0 维上会广播，逻辑上扩展为
     <code>
      [[1, 1, 1], [1, 1, 1]]
     </code>
     ，也就是复用原来的数据，然后和
     <code>
      tensor_2d
     </code>
     逐元素相加
    </p>
    <p>
     2.乘法
    </p>
    <h5>
     矩阵乘法
    </h5>
    <p>
     在 PyTorch 中，使用
     <code>
      torch.matmul()
     </code>
     函数或
     <code>
      @
     </code>
     运算符进行矩阵乘法。矩阵乘法要求左矩阵的列数等于右矩阵的行数。
    </p>
    <pre><code class="language-python">import torch

# 创建两个二维张量用于矩阵乘法
tensor1 = torch.tensor([[1, 2], [3, 4]])
tensor2 = torch.tensor([[5, 6], [7, 8]])

# 矩阵乘法
result = torch.matmul(tensor1, tensor2)
# 或者使用 @ 运算符
# result = tensor1 @ tensor2
print("矩阵乘法结果:")
print(result)</code></pre>
