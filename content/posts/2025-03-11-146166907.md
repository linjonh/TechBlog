    <h2>
     一、题目
    </h2>
    <p>
     <a href="https://leetcode.cn/problems/container-with-most-water/" rel="nofollow" title="11. 盛最多水的容器">
      11. 盛最多水的容器
     </a>
    </p>
    <p>
     给定一个长度为
     <code>
      n
     </code>
     的整数数组
     <code>
      height
     </code>
     。有
     <code>
      n
     </code>
     条垂线，第
     <code>
      i
     </code>
     条线的两个端点是
     <code>
      (i, 0)
     </code>
     和
     <code>
      (i, height[i])
     </code>
     。
    </p>
    <p>
     找出其中的两条线，使得它们与
     <code>
      x
     </code>
     轴共同构成的容器可以容纳最多的水。
    </p>
    <p>
     返回容器可以储存的最大水量。你不能倾斜容器。
    </p>
    <h2>
     二、思路
    </h2>
    <p>
     容纳最多的水——(j-i) * min(height[i], height[j])取到最大值
    </p>
    <p>
     如果直接遍历下去，要比较的次数太多了（也就是指针移动的次数太多了），怎么办呢？
    </p>
    <p>
     我们要找到简化比较过程的一个规律，可以利用最大储水量依赖的容器高来判断指针如何移动：
    </p>
    <p>
     <strong>
      1. 初始化两个指针
     </strong>
     ：
     <code>
      left
     </code>
     指向数组的开头，
     <code>
      right
     </code>
     指向数组的末尾。
    </p>
    <p>
     <strong>
      2. 初始化最大水量
     </strong>
     ：
     <code>
      most
     </code>
     用于存储最大的水量。
    </p>
    <p>
     <strong>
      3. 循环条件
     </strong>
     ：当
     <code>
      left
     </code>
     小于
     <code>
      right
     </code>
     时，继续循环。
    </p>
    <p>
     <strong>
      4. 计算当前水量
     </strong>
     ：
    </p>
    <ul>
     <li>
      <span style="color:#494949">
       <code>
        h
       </code>
       是两个高度中的较小值。
      </span>
     </li>
     <li>
      <span style="color:#494949">
       <code>
        width
       </code>
       是两个指针之间的距离。
      </span>
     </li>
     <li>
      <span style="color:#494949">
       <code>
        water
       </code>
       是当前容器可以容纳的水量。
      </span>
     </li>
    </ul>
    <p>
     <strong>
      5. 更新最大水量
     </strong>
     ：用
     <code>
      Math.max
     </code>
     更新
     <code>
      most
     </code>
     。
    </p>
    <p>
     <strong>
      6. 移动指针
     </strong>
     ：
    </p>
    <ul>
     <li>
      <span style="color:#494949">
       如果
       <code>
        height[left]
       </code>
       小于
       <code>
        height[right]
       </code>
       ，增加
       <code>
        left
       </code>
       指针。
      </span>
     </li>
     <li>
      <span style="color:#494949">
       否则，减小
       <code>
        right
       </code>
       指针。
      </span>
     </li>
    </ul>
    <p>
     <strong>
      7. 返回最大水量
     </strong>
     ：
     <span style="color:#494949">
      在循环结束后返回
      <code>
       most
      </code>
      。
     </span>
    </p>
    <h2>
     三、代码
    </h2>
    <p>
     ① JavaScript：
    </p>
    <pre><code class="language-javascript">function mostWater(height) {
    let left = 0;
    let right = height.length - 1;
    let most = 0;

    while (left &lt; right) {
        const h = Math.min(height[left], height[right]);
        const width = right - left;
        const water = h * width;
        most = Math.max(most, water);

        if (height[left] &lt; height[right]) {
            left++;
        } else {
            right--;
        }
    }

    return most;
};</code></pre>
    <p>
     ② python：
    </p>
    <pre><code class="language-python">def most_water(height):
    left = 0
    right = len(height) - 1
    most = 0

    while left &lt; right:
        h = min(height[left], height[right])
        width = right - left
        water = h * width

        if water &gt; most:
            most = water

        if height[left] &lt; height[right]:
            left += 1
        else:
            right -= 1

    return most
</code></pre>
    <p>
     ③ C++：
    </p>
    <pre><code class="language-cpp">int mostWater(const vector&lt;int&gt;&amp; height) {
    int left = 0;
    int right = height.size() - 1;
    int most = 0;

    while (left &lt; right) {
        int h = min(height[left], height[right]);
        int width = right - left;
        int water = h * width;

        if (water &gt; most) {
            most = water;
        }

        if (height[left] &lt; height[right]) {
            left++;
        } else {
            right--;
        }
    }

    return most;
}
</code></pre>
