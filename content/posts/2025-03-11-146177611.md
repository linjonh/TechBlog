    <p>
     在Ubuntu中，有多种方式可以查看目录或文件的权限，以下为你详细介绍常见的指令及其使用方法：
    </p>
    <h4>
     <a id="1_ls_l__2">
     </a>
     1.
     <code>
      ls -l
     </code>
     命令
    </h4>
    <p>
     这是最常用的查看文件和目录权限的命令，
     <code>
      -l
     </code>
     选项用于以长格式列出文件和目录的详细信息，其中就包含权限信息。
    </p>
    <h5>
     <a id="_5">
     </a>
     基本语法
    </h5>
    <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-l</span> <span class="token punctuation">[</span>文件或目录路径<span class="token punctuation">]</span>
</code></pre>
    <ul>
     <li>
      <strong>
       不指定路径
      </strong>
      ：如果不指定路径，默认会列出当前工作目录下的文件和目录的权限信息。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-l</span>
</code></pre>
    <p>
     示例输出：
    </p>
    <pre><code class="prism language-plaintext">total 24
-rw-r--r-- 1 user user  0 Mar 11 16:30 example.txt
drwxr-xr-x 2 user user 4096 Mar 11 16:31 example_dir
</code></pre>
    <p>
     这里，第一列表示权限信息，
     <code>
      -rw-r--r--
     </code>
     表示这是一个文件（开头的
     <code>
      -
     </code>
     表示文件），文件所有者有读写权限，所属组和其他用户只有读权限；
     <code>
      drwxr-xr-x
     </code>
     表示这是一个目录（开头的
     <code>
      d
     </code>
     表示目录），目录所有者有读、写、执行权限，所属组和其他用户有读和执行权限。
    </p>
    <ul>
     <li>
      <strong>
       指定路径
      </strong>
      ：可以指定具体的文件或目录路径来查看其权限信息。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-l</span> /home/user/Documents
</code></pre>
    <h4>
     <a id="2_stat__26">
     </a>
     2.
     <code>
      stat
     </code>
     命令
    </h4>
    <p>
     <code>
      stat
     </code>
     命令可以提供更详细的文件或目录状态信息，包括权限、大小、创建时间、修改时间等。
    </p>
    <h5>
     <a id="_29">
     </a>
     基本语法
    </h5>
    <pre><code class="prism language-bash"><span class="token function">stat</span> <span class="token punctuation">[</span>文件或目录路径<span class="token punctuation">]</span>
</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="prism language-bash"><span class="token function">stat</span> example.txt
</code></pre>
    <p>
     输出示例：
    </p>
    <pre><code class="prism language-plaintext">  File: example.txt
  Size: 0               Blocks: 0          IO Block: 4096   regular empty file
Device: 801h/2049d      Inode: 123456      Links: 1
Access: (0644/-rw-r--r--)  Uid: ( 1000/  user)   Gid: ( 1000/  user)
Access: 2025-03-11 16:30:00.000000000 +0800
Modify: 2025-03-11 16:30:00.000000000 +0800
Change: 2025-03-11 16:30:00.000000000 +0800
 Birth: -
</code></pre>
    <p>
     其中
     <code>
      Access: (0644/-rw-r--r--)
     </code>
     明确显示了文件的权限，
     <code>
      0644
     </code>
     是权限的数字表示形式，
     <code>
      -rw-r--r--
     </code>
     是符号表示形式。
    </p>
    <h4>
     <a id="3_find__50">
     </a>
     3.
     <code>
      find
     </code>
     命令
    </h4>
    <p>
     <code>
      find
     </code>
     命令主要用于在文件系统中查找文件和目录，同时也可以结合
     <code>
      -ls
     </code>
     选项来查看查找到的文件和目录的权限信息。
    </p>
    <h5>
     <a id="_53">
     </a>
     基本语法
    </h5>
    <pre><code class="prism language-bash"><span class="token function">find</span> <span class="token punctuation">[</span>查找路径<span class="token punctuation">]</span> <span class="token parameter variable">-ls</span>
</code></pre>
    <p>
     例如，要查看
     <code>
      /home/user
     </code>
     目录下所有文件和目录的权限信息，可以使用以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token function">find</span> /home/user <span class="token parameter variable">-ls</span>
</code></pre>
    <p>
     该命令会递归地列出
     <code>
      /home/user
     </code>
     目录下的所有文件和目录的详细信息，包括权限。
    </p>
