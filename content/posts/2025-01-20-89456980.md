# B站后端源码"泄露"

## B 站后端源码"泄露"

作为 Java 程序员，听到后端源码泄露，还是 B 站这种名厂，想必是十分激动的。

但是上 github 上瞅了一眼，是 GO 开发的，顿时就感觉凉了。

B 站方面反映也很及时，十个小时不到，现在就已经搜不到了，不过我在浏览器上下载项目的 zip 包，还是在缓缓进行中。

事情具体是怎么样的也不清楚，看了 github 上最初的几条留言以及知乎上的回答，大概是离职员工做的吧。挺可怕的，这种感觉是要负法律责任啊！
