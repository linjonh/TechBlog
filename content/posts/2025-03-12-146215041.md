     </path>
    </svg>
    <p>
     在Mac上安装MySQL后，如果终端中找不到mysql命令，通常是 因为MySQL的命令行工具（如mysql客户端）没有被正确地添加到你的环境变量中。
    </p>
    <ol>
     <li>
      检查 MySQL 是否已安装
     </li>
    </ol>
    <pre><code class="prism language-javascript"> 	ps <span class="token operator">-</span>ef<span class="token operator">|</span>grep mysql
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b99b34898a6c4731861456f4f415eac7.png">
      <br/>
      查看到路径在
      <strong>
       /usr/local/mysql/bin
      </strong>
     </img>
    </p>
    <ol start="2">
     <li>
      查看 .bash_profile 文件是否存在 （cd /usr/local/mysql/bin 在mysql的安装目录去查看）
     </li>
    </ol>
    <pre><code class="prism language-javascript">	ls <span class="token operator">-</span>al 
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/557bd62598014811a9a68844b83866db.png">
      <br/>
      如果
      <strong>
       .bash_profile
      </strong>
      文件不存在，通过以下指令创建
     </img>
    </p>
    <pre><code class="prism language-javascript"> sudo touch <span class="token punctuation">.</span>bash_profile
</code></pre>
    <p>
     3.修改.bash_profile 文件
    </p>
    <pre><code class="prism language-javascript">sudo vim <span class="token punctuation">.</span>bash_profile
</code></pre>
    <p>
     <strong>
      进入bash_profile如何去修改文件
     </strong>
     ，输入i键进行插入，esc键退出，最后:wq去保存这个文件
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f8cceb1003694c79ac065888cfa1285f.png"/>
    </p>
    <p>
     添加mysql的环境变量 /usr/local/mysql/bin 为之前查找到的mysql的安装位置
    </p>
    <pre><code class="prism language-javascript"><span class="token keyword">export</span> <span class="token constant">PATH</span><span class="token operator">=</span>$<span class="token punctuation">{<!-- --></span><span class="token constant">PATH</span><span class="token punctuation">}</span><span class="token operator">:</span><span class="token operator">/</span>usr<span class="token operator">/</span>local<span class="token operator">/</span>mysql<span class="token operator">/</span>bin 
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c959470ef88742ef93090da1cde7d8c3.png"/>
    </p>
    <ol start="4">
     <li>
      让.bash_profile文件生效
     </li>
    </ol>
    <pre><code class="prism language-javascript">source <span class="token punctuation">.</span>bash_profile
</code></pre>
    <p>
     5.再次运行mysql看是否配置成功
    </p>
    <pre><code class="prism language-javascript">mysql <span class="token operator">-</span>u root <span class="token operator">-</span>p     输入mysql密码验证
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cefbb44443774d959260efc6928e7dfa.png"/>
    </p>
    <h3>
     <a id="_54">
     </a>
     方案二
    </h3>
    <pre><code>1. vim ~/.zshrc 打开配置文件
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a93b663b3b16433da0eb4ed136ca6dd9.png">
      <br/>
      2. 加入这段配置代码
      <strong>
       export PATH=“/usr/local/mysql/bin:$PATH”
      </strong>
      ，退出保存
     </img>
    </p>
    <p>
     3 . 输入
     <strong>
      source ~/.zshrc
     </strong>
     让文件生效
     <br/>
     4. which mysql查看
    </p>
