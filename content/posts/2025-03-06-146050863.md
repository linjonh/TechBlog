    <h4>
     <a id="Rust_0">
     </a>
     什么是Rust
    </h4>
    <p>
     <code>
      Rust
     </code>
     是由Mozilla研究院开发的一种系统级编程语言，旨在提供更好的内存安全保证，同时保持高性能，自2010年首次发布以来，Rust以其安全性、并发性和实用性迅速获得了广泛的关注。Rust最独特的特性之一是其
     <code>
      所有权模型
     </code>
     ，这使得编写的代码能够在编译期就避免许多常见的内存错误，比如空指针异常、数据竞争等，Rust 速度极快且内存高效：
     <code>
      没有运行时
     </code>
     或
     <code>
      垃圾收集器
     </code>
     ，它可以为性能关键型服务提供支持，并与其他语言轻松集成，能够消除许多类 编译时的错误。
     <br/>
     <a href="https://www.rust-lang.org/" rel="nofollow">
      Rust官方网站
     </a>
    </p>
    <h4>
     <a id="Rust_7">
     </a>
     如何安装Rust
    </h4>
    <p>
     以Windows系统为例，在Rust官网中点击
     <code>
      马上开始
     </code>
     按钮
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0ead8e449d304e99a5f85824e80bc90a.png">
      <br/>
      随后根据电脑系统来选择
      <code>
       下载 RUSTUP-INIT.EXE(32位)
      </code>
      或
      <code>
       下载 RUSTUP-INIT.EXE(32位)
      </code>
      ，如果不清楚应该如何查看电脑位数，可以直接唤醒
      <code>
       cmd
      </code>
      窗口，在窗口中输入以下命令来查看：
     </img>
    </p>
    <pre><code class="prism language-bash">wmic os get osarchitecture
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9ad7e74b201b4599ab36cb79e71eeaf3.png">
      <br/>
      随后双击下载好的安装文件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8acf98c645aa4f28bfb719c5e11b64f9.png">
       <br/>
       接下来会提示三个安装选项
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f852baf2c5fb43f9b4bd84c9d6390b98.png"/>
      </img>
     </img>
    </p>
    <ul>
     <li>
      <code>
       标准安装
      </code>
      ：直接按下Enter键即可进行默认设置的安装。
     </li>
     <li>
      <code>
       自定义安装
      </code>
      ：允许你对安装过程进行一些定制，比如改变安装路径、选择不同的工具链版本等。
     </li>
     <li>
      <code>
       取消安装
      </code>
      ：如果你现在不打算安装，可以选择此选项退出安装程序。
     </li>
    </ul>
    <p>
     我们可以直接选择
     <code>
      标准安装
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0fc4cfc1e6a7420299ee5c040f8fef2e.png">
      <br/>
      安装完成后可以在终端中输入命令来检查是否安装成功：
     </img>
    </p>
    <pre><code class="prism language-bash">rustc <span class="token parameter variable">--version</span>
<span class="token function">cargo</span> <span class="token parameter variable">-V</span>
</code></pre>
    <p>
     当看到以下输出时代表安装成功
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4bd8d740dcf9438282c6ca6b30b45e33.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/080db9dad38943a8a33226bb5a969c9e.png"/>
     </img>
    </p>
    <h4>
     <a id="RustHello_World_33">
     </a>
     Rust输出
     <code>
      Hello World
     </code>
    </h4>
    <p>
     我们可以先找萨满问问米，挑选一个风水好的文件夹，随后使用命令新建rust项目：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cargo</span> new ggbond
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8d4a98ce282d4c3cb02f315e228ccc66.png">
      <br/>
      其中
      <code>
       src
      </code>
      文件夹中的
      <code>
       main.rs
      </code>
      文件就是程序主文件，我们在项目的根目录使用命令，即可运行并构建项目：
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">cargo</span> run
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/414932a740254a98b1c2f288a0cf8a7c.png"/>
     <br/>
     运行后，我们可以在项目文件夹中看到一个新建的
     <code>
      target
     </code>
     文件夹，构建的可运行程序就放在这个文件夹里
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3d92a85d064742cca50e2f73381082a3.png"/>

    <h4>
     <a id="cargo_47">
     </a>
     什么是cargo
    </h4>
    <p>
     <code>
      cargo
     </code>
     是Rust 构建工具和包管理器，它简化了使用Rust进行开发的过程。无论是创建新项目、编译代码、运行测试还是发布包，Cargo都提供了相应的命令来帮助开发者高效地完成任务，通过Cargo可以轻松管理依赖关系、构建配置以及项目的元数据。
    </p>
    <p>
     以下是日常开发中最常用的几个Cargo命令：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        创建新项目
       </strong>
      </p>
      <pre><code>cargo new project_name
</code></pre>
      <p>
       创建一个名为
       <code>
        project_name
       </code>
       的新Rust项目。
      </p>
     </li>
     <li>
      <p>
       <strong>
        构建项目
       </strong>
      </p>
      <pre><code>cargo build
</code></pre>
      <p>
       编译当前项目，输出在
       <code>
        target/debug/
       </code>
       目录下。
      </p>
     </li>
     <li>
      <p>
       <strong>
        构建并优化项目
       </strong>
      </p>
      <pre><code>cargo build --release
</code></pre>
      <p>
       使用优化选项编译项目，适用于生产环境，输出位于
       <code>
        target/release/
       </code>
       目录中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        运行项目
       </strong>
      </p>
      <pre><code>cargo run
</code></pre>
      <p>
       编译并执行当前项目，方便快速测试代码变化。
      </p>
     </li>
     <li>
      <p>
       <strong>
        检查代码
       </strong>
      </p>
      <pre><code>cargo check
</code></pre>
      <p>
       快速检查代码是否有错误，但不生成可执行文件，适合频繁验证代码正确性。
      </p>
     </li>
     <li>
      <p>
       <strong>
        运行测试
       </strong>
      </p>
      <pre><code>cargo test
</code></pre>
      <p>
       执行项目中的所有测试，确保代码按预期工作。
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看文档
       </strong>
      </p>
      <pre><code>cargo doc --open
</code></pre>
      <p>
       为当前项目及其所有依赖生成并打开API文档。
      </p>
     </li>
     <li>
      <p>
       <strong>
        更新依赖
       </strong>
      </p>
      <pre><code>cargo update
</code></pre>
      <p>
       更新
       <code>
        Cargo.lock
       </code>
       以反映最新版本的依赖。
      </p>
     </li>
     <li>
      <p>
       <strong>
        添加依赖
       </strong>
      </p>
      <pre><code>cargo add crate_name
</code></pre>
      <p>
       将指定的crate作为依赖添加到你的项目中（需要安装额外插件如
       <code>
        cargo-edit
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        清理构建文件
       </strong>
      </p>
      <pre><code>cargo clean
</code></pre>
      <p>
       清除之前构建生成的所有文件，释放磁盘空间。
      </p>
     </li>
    </ol>
