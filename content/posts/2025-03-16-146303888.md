    <p>
     git 是一个版本控制工具，真正开发项目中是多个人开发一个项目的，需要进行代码的管理，每个人在自己电脑上面开发，然后提交代码，互相之间是不影响的。

    <p>
     实习的时候 早上进入公司，ld 给我代码地址让我拉到本地看代码，然后给在新的需求，让我新建一个分支在上面写，写完之后在提交上去，最后 pull request，请求 ld 进行代码的合并。
    </p>
    <p>
     如果不会 git，在实习的时候我感觉是很减分，需要熟练使用 git。
    </p>
    <p>
     master |dev 分支是项目最完善的代码分支，其他分支是用来开发的。这个分支可能会绑定一些自动化部署的比如 jekins 等，实现代码的 CI\CD 流程。
    </p>

    <h5>
     <a id="_73">
     </a>
     拉去主分支代码
    </h5>
    <p>
     首先 master 分支上面是最新的代码，每次都要从 dev 分支上面去拷贝新的代码下来。 先
    </p>
    <pre><code class="prism language-java">git clone https<span class="token operator">:</span><span class="token operator">/</span><span class="token operator">/</span>gitee<span class="token punctuation">.</span>com<span class="token operator">/</span>z<span class="token operator">-</span>zhou<span class="token operator">-</span>xin<span class="token operator">/</span>linux<span class="token operator">-</span>testing<span class="token punctuation">.</span>git
</code></pre>

    <h5>
     <a id="_86">
     </a>
     查看本地分支的状态
    </h5>
    <pre><code class="prism language-java">git status  <span class="token comment">// 查看本地分支的信息</span>
</code></pre>
    <p>
     可能会和远程的分支最新不一样，在开发的时候
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/109bc5f3b123362edf789a3e55b4ebab.png"/>

    <h5>
     <a id="_95">
     </a>
     查看远程分支
    </h5>
    <pre><code class="prism language-java">git remote <span class="token operator">-</span>v  <span class="token comment">//查看远程分支的信息</span>
</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/d1e423cd71abccc59d5f3be841ff48c7.png"/>
    </p>
    <h5>
     <a id="_102">
     </a>
     查看远程的仓库信息
    </h5>
    <pre><code class="prism language-java">git branch <span class="token operator">-</span>rv   
</code></pre>
    <p>
     <code>
      git branch -rv
     </code>
     是用来显示本地和远程仓库的分支信息的命令。
    </p>
    <ul>
     <li>
      <code>
       **git branch**
      </code>
      ：列出本地所有的分支。
     </li>
     <li>
      <code>
       **-r**
      </code>
      ：表示列出远程的分支。
     </li>
     <li>
      <code>
       **-v**
      </code>
      ：显示每个分支的最新提交信息。
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/70441e05a464467da2e1519c01435b32.png"/>
    </p>
    <h5>
     <a id="_115">
     </a>
     本地分支切换
    </h5>
    <p>
     切换到本地的分支上面去，不会创建远程的分支(切换到已经存在的分支 )
    </p>
    <pre><code class="prism language-java"> git checkout dev  
</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/9efaafcc481e49c4ae45578d67259af9.png"/>
    </p>
    <h5>
     <a id="_124">
     </a>
     切换并创建分支
    </h5>
    <p>
     创建并切换到一个新分支
     <code>
      xxx
     </code>
     ，这个分支会基于当前所在的分支。并在远程创建分支
    </p>
    <pre><code class="prism language-java">  git checkout <span class="token operator">-</span>b  zxtest
</code></pre>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/4510aed2136219cd882392ea39d08a4e.png"/>
    </p>
    <p>
     创建一个文件，git bash 里面可以使用 linux 的命令，可以直接当作 linux 去使用。

     <a id="_139">
     </a>
     提交代码

</code></pre>

    <pre><code class="prism language-java">git push origin   xxx 新建的分支名称
