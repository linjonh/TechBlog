    <p>
     我们在
     <a href="https://blog.csdn.net/weixin_71719718/article/details/145960815" title="1.Qt写简单的登录界面(c++)_c++ qt 设计一个简单界面-CSDN博客">
      1.Qt写简单的登录界面(c++)_c++ qt 设计一个简单界面-CSDN博客
     </a>
     中写了个简单的登录界面，但没有槽，在这里写一个带有槽的界面。
    </p>
    <h3>
     1.代码
    </h3>
    <p>
     代码目录如下：
    </p>
    <p class="img-center">
     <img alt="" height="398" src="https://i-blog.csdnimg.cn/direct/91a9c4d4218b488da7814ccae1a746d1.png" width="400"/>
    </p>
    <p>
     main.cpp的代码如下：
    </p>
    <pre><code class="hljs">#include "MainWindow.h"
#include &lt;QtWidgets/QApplication&gt;

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    MainWindow w;
    w.show();
    return a.exec();
}
</code></pre>
    <p>
     MainWindow.h的代码如下：
    </p>
    <pre><code class="hljs">#pragma once

#include &lt;QtWidgets/QMainWindow&gt;
#include "ui_MainWindow.h"
#include &lt;QLabel&gt;
#include &lt;QtWidgets&gt;

class MainWindow : public QMainWindow
{
    Q_OBJECT

public:
    MainWindow(QWidget *parent = nullptr);
    ~MainWindow();

public slots: // 或者直接在类定义中定义槽函数。
    void handleMySignal();

private:
    Ui::MainWindow *ui;
};</code></pre>
    <p>
     MainWindow.cpp的代码如下：
    </p>
    <pre><code class="hljs">#include "MainWindow.h"

#include &lt;QLabel&gt;
#include &lt;QtWidgets&gt;

MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
    ,ui(new Ui::MainWindow)
{
    ui-&gt;setupUi(this);
    this-&gt;setWindowTitle("The first UI display of wang");//设置窗口标题
    this-&gt;setWindowIcon(QIcon("D:\\AI\\vs\\ui\\Demo1\\Demo1\\buttonss\\ss.png"));//设置窗口图标
    //this-&gt;setWindowFlag(Qt::FramelessWindowHint);//设置纯净标题
    this-&gt;resize(800, 600);//设置大小
    this-&gt;setStyleSheet("background-color:white");//设置背景颜色
    
    // --------------------按钮设置-------------------
    QPushButton* btn = new QPushButton(" register", this);
    btn-&gt;setGeometry(350, 400, 120, 50);
    btn-&gt;setStyleSheet("background-color:rgb(8,189,253);border-radius:5px;color:white");//设置背景，按钮样式,字体颜色

    // 建立关联当点击pushButton时信号clicked 发送给槽on_pushButton_clicked
    btn-&gt;connect(btn, &amp;QPushButton::clicked, this, &amp;MainWindow::handleMySignal);

}

MainWindow::~MainWindow()
{
    delete ui;
}


void MainWindow::handleMySignal()
{
    this-&gt;setWindowTitle("wang");//设置窗口标题
    
}</code></pre>

    <h3>
     2.注意事项
    </h3>
    <p>
     在该代码中，可能会出现 ui(new Ui::MainWindow) 的报错，解决办法如下：
    </p>
    <p>
     1.打开项目中的MainWindow.ui文件如下：
    </p>
    <p>
     <img alt="" height="1143" src="https://i-blog.csdnimg.cn/direct/ea0efafad9c5425c93596e08ac0fda17.png" width="1539"/>
    </p>
    <p>
     修改上图中右下角的"objectName"右边的名称为"MainWindow",然后保存该文件，退出后进入vs重新生成解决方案，这个时候还会继续报错。
    </p>
    <p>
     2.进入MainWindow.h文件
    </p>
    <p>
     如下图所示，将工具栏的debug修改为Release,然后将该文件最下面一行修改为Ui::MainWindow *ui;（自动生成的代码不是这样的，需要修改成这样）。
    </p>
    <p>
     <img alt="" height="1273" src="https://i-blog.csdnimg.cn/direct/ea5765c79d8248639b1db695ddbefe35.png" width="1511"/>
    </p>
    <p>
     最后再次生成解决方案，就不会报错了。
    </p>
