    <h1 class="title-article" id="articleContentId">
     Pycharm修改默认执行框架为unittest
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
    </p>
    <p>
     在 PyCharm 中修改默认的测试框架为
     <code>
      unittest
     </code>
     的步骤如下：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 PyCharm 设置
       </strong>
       :
      </p>
      <ul>
       <li>
        在 Windows 或 Linux 上，您可以通过菜单栏中的
        <code>
         File
        </code>
        &gt;
        <code>
         Settings
        </code>
        来打开设置。
        <br/>
        <img alt="" height="810" src="https://i-blog.csdnimg.cn/direct/441b466fede54f64a981fa2a0becd97e.png" width="448"/>
       </li>
       <li>
        在 macOS 上，您可以通过
        <code>
         PyCharm
        </code>
        &gt;
        <code>
         Preferences
        </code>
        来打开设置。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        找到测试框架设置
       </strong>
       :
      </p>
      <ul>
       <li>
        在设置窗口中，导航到
        <code>
         Tools
        </code>
        &gt;
        <code>
         Python Integrated Tools
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        修改默认测试框架
       </strong>
       :
      </p>
      <ul>
       <li>
        在
        <code>
         Python Integrated Tools
        </code>
        页面，您会看到一个名为
        <code>
         Default test runner
        </code>
        的选项。
       </li>
       <li>
        点击该选项的下拉菜单，并选择
        <code>
         unittest
        </code>
        作为默认测试框架。
        <br/>
        <img alt="" height="1069" src="https://i-blog.csdnimg.cn/direct/df9f8dca67b447429f56427177545a82.png" width="1472"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        应用更改
       </strong>
       :
      </p>
      <ul>
       <li>
        点击
        <code>
         Apply
        </code>
        或
        <code>
         OK
        </code>
        按钮以保存设置。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     这样设置后，PyCharm 应该会默认使用
     <code>
      unittest
     </code>
     作为新的测试框架进行测试运行。如果还没有看到相关选项，请确保已经安装并配置 Python SDK。
    </p>
    <p>
     如果在特定项目中没有应用更改，您还可以检查项目级别的配置，以确保设置正确应用于项目中。
    </p>
    <p>
    </p>
