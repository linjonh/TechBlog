# Pycharm修改默认执行框架为unittest

在 PyCharm 中修改默认的测试框架为
`unittest`
的步骤如下：

1. **打开 PyCharm 设置**
   :

   * 在 Windows 或 Linux 上，您可以通过菜单栏中的
     `File`
     >
     `Settings`
     来打开设置。
       
     ![](https://i-blog.csdnimg.cn/direct/441b466fede54f64a981fa2a0becd97e.png)
   * 在 macOS 上，您可以通过
     `PyCharm`
     >
     `Preferences`
     来打开设置。
2. **找到测试框架设置**
   :

   * 在设置窗口中，导航到
     `Tools`
     >
     `Python Integrated Tools`
     。
3. **修改默认测试框架**
   :

   * 在
     `Python Integrated Tools`
     页面，您会看到一个名为
     `Default test runner`
     的选项。
   * 点击该选项的下拉菜单，并选择
     `unittest`
     作为默认测试框架。
       
     ![](https://i-blog.csdnimg.cn/direct/df9f8dca67b447429f56427177545a82.png)
4. **应用更改**
   :

   * 点击
     `Apply`
     或
     `OK`
     按钮以保存设置。

这样设置后，PyCharm 应该会默认使用
`unittest`
作为新的测试框架进行测试运行。如果还没有看到相关选项，请确保已经安装并配置 Python SDK。

如果在特定项目中没有应用更改，您还可以检查项目级别的配置，以确保设置正确应用于项目中。
