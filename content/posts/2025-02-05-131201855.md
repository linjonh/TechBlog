   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      需求：
     </strong>
    </p>
    <p>
     需求是小程序的首页banner图来自pc端配置生产。
    </p>
    <p>
     在
     <strong>
      小程序点击轮播的banner图后，跳到对应的linkurl
     </strong>
     ，但是我们和客户约定好，只能跳转公众号。
    </p>
    <p>
     一开始看了小程序的文档，需要配置什么业务域名，下载校验文件，其实不需要。
     <strong>
      只需要一个webview页面即可。
     </strong>
    </p>
    <p>
    </p>
    <p>
     <strong>
      实现：
     </strong>
    </p>
    <p>
    </p>
    <p>
     <img alt="" height="698" src="https://i-blog.csdnimg.cn/blog_migrate/161ce98d8d71671c6c96d379716c77b0.png" width="412"/>
    </p>
    <p>
    </p>
    <p>
     跳转之后：
    </p>
    <p>
     <img alt="" height="694" src="https://i-blog.csdnimg.cn/blog_migrate/20c8a955e748b05c98c1c71ebe817c17.png" width="404"/>
    </p>
    <p>
     <strong>
      具体步骤：
     </strong>
    </p>
    <p>
     1.首先创建要一个webview的文件夹
    </p>
    <p>
     <img alt="" height="100" src="https://i-blog.csdnimg.cn/blog_migrate/54d8c665624430dd91efc79908132be6.png" width="204"/>
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-javascript">&lt;template&gt;
  &lt;web-view :src="url"&gt;&lt;/web-view&gt;
&lt;/template&gt;
 
&lt;script&gt;
export default {
  data() {
    return {
      url: ''
    }
  },
  onLoad(item) {
    // 传入需要跳转的链接 使用web-view标签进行跳转
    this.url = decodeURIComponent(item.url)
  }
}
&lt;/script&gt;</code></pre>
    <p>
     2.在业务点击按钮层面直接跳转传入url：
    </p>
    <p>
     注意：公众号链接可以打开你的公众号文章，然后点击右上角，再点击复制链接就可以了，一般我们都是和用户约定好，只能配置公众号文章，所以我们是动态跳转至用户配置的url；
    </p>
    <pre><code class="language-javascript">skipLink(e) {
   // 你的公众号链接
   let url = 'https://mp.weixin.qq.com/s/xxxxxxx';
   uni.navigateTo({
	 url: '/pages/index/webview?url=' + url
   })
},</code></pre>
    <p>
     动态配置业务代码：
    </p>
    <pre><code class="language-javascript">		skipLink(e) {
			if(this.urlList[e]) {
				uni.navigateTo({
					url: '/pages/index/webview?url=' + this.urlList[e]
				})
			}
		},</code></pre>
