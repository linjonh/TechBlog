    <h2>
     <a id="Python_0">
     </a>
     一、Python的基本操作
    </h2>
    <h3>
     <a id="11_Python_2">
     </a>
     1.1 Python基本语法
    </h3>
    <ul>
     <li>
      Python中严格区分大小写
     </li>
     <li>
      Python中每一行就是一条语句，每条语句以换行结束
     </li>
     <li>
      Python中每一行语句不要过长（建议每行不超过80个字符）
     </li>
     <li>
      一条语句可以分成多行编写，语句后以 \ 结尾
     </li>
     <li>
      Python是缩进严格的语言，在python中不要随意缩进
     </li>
    </ul>
    <h3>
     <a id="12__12">
     </a>
     1.2 注释
    </h3>
    <h4>
     <a id="121_14">
     </a>
     <strong>
      1.2.1注释的作用
     </strong>
    </h4>
    <p>
     使用自己熟悉的语言，在程序开发中对某些代码进行标注说明。
    </p>
    <p>
     <strong>
      目的
     </strong>
     ：
     <strong>
      增加代码的可读性
     </strong>
    </p>
    <h4>
     <a id="122__20">
     </a>
     1.2.2 注释的分类
    </h4>
    <p>
     <strong>
      (1) 单行注释
     </strong>
    </p>
    <ul>
     <li>
      以#开头，#右边不会被执行，仅仅起到辅助说明的作用，为了保证可读性#后加空格
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token comment"># 这是hello python 的注释</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"hello python"</span><span class="token punctuation">)</span>
</code></pre>
    <ul>
     <li>
      在代码后面增加的单行注释，注意代码与注释之间至少有2个空格
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"hello python"</span><span class="token punctuation">)</span> <span class="token comment"># 这是hello python的注释</span>
</code></pre>
    <p>
     <strong>
      (2) 多行注释
     </strong>
    </p>
    <ul>
     <li>
      一对连续的三个引号（单引号、双引号都可以）
     </li>
    </ul>
    <pre><code class="prism language-Python">"""
这是一个多行注释
这是一个多行注释
"""
print("hello python")
</code></pre>
    <h3>
     <a id="13__51">
     </a>
     1.3 字面量和变量
    </h3>
    <ul>
     <li>
      字面量就是一个个的值，表示的意思是它字面的值，在程序中可以直接使用。
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token number">1</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">,</span><span class="token number">3</span><span class="token punctuation">,</span><span class="token string">'hello'</span><span class="token punctuation">,</span><span class="token string">'Python'</span>
</code></pre>
    <ul>
     <li>
      变量（variable）可以用来保存字面量，并且变量中保存的字面量是不定的
     </li>
    </ul>
    <pre><code class="prism language-Python">a = 100
print(a)
</code></pre>
    <ul>
     <li>
      通常开发中，是将字面量保存在变量中，通过变量来引用字面量
     </li>
    </ul>
    <h3>
     <a id="14_72">
     </a>
     1.4变量
    </h3>
    <ul>
     <li>
      python中标量不需要声明，直接为变量赋值即可
     </li>
    </ul>
    <pre><code class="prism language-python">a <span class="token operator">=</span> <span class="token number">100</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>a<span class="token punctuation">)</span>
</code></pre>
    <ul>
     <li>
      不能够使用没有赋值的变量
     </li>
    </ul>
    <pre><code class="prism language-python"><span class="token operator">&gt;&gt;</span><span class="token operator">&gt;</span> <span class="token keyword">print</span><span class="token punctuation">(</span>b<span class="token punctuation">)</span>

    <ul>
     <li>
      python是一个动态类型的语言，可以为变量赋任何类型的值，也可以任意修改变量的值;

    </ul>
    <pre><code class="prism language-python"><span class="token operator">&gt;&gt;</span><span class="token operator">&gt;</span> a <span class="token operator">=</span> <span class="token number">100</span>

    <h3>
     <a id="16__131">
     </a>
     1.6 命名规范
    </h3>
    <ul>
     <li>
      <p>
       下划线命名法
      </p>
      <ul>
       <li>
        所有字母小写，单词之间使用_分割，通常用于变量
       </li>
      </ul>
      <pre><code class="prism language-python">hello_world
</code></pre>
     </li>
     <li>
      <p>
       帕斯卡命名法 ( 驼峰命名法 )
      </p>
     </li>
     <li>
      <p>
       首字母大写，每个单词首字母大写，其余字母小写，通常用于类名
      </p>
     </li>
    </ul>
    <pre><code class="prism language-Python">HelloWord
</code></pre>
