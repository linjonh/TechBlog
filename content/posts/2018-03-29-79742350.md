    <p align="justify">
     在微信小程序里面的弹出框有时候不能满足我们的需求，需要自己写弹出框，但是要保证页面不要上下滑动，解决方法是在遮盖层上用绑定手指触摸事件
     <span style="color:rgb(62,62,62);">
      catchtouchstart
     </span>
     。
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      微信小程序有
     </span>
     bind
     <span style="font-family:'宋体';">
      和
     </span>
     <span style="font-family:Calibri;">
      catch
     </span>
     <span style="font-family:'宋体';">
      两种事件绑定方法，
     </span>
     <span style="color:rgb(51,51,51);background:rgb(247,247,247);">
      bind
     </span>
     <span style="color:rgb(51,51,51);">
      <span style="font-family:'宋体';">
       事件绑定不会阻止冒泡事件向上冒泡，
      </span>
     </span>
     <span style="color:rgb(51,51,51);background:rgb(247,247,247);">
      catch
     </span>
     <span style="color:rgb(51,51,51);">
      <span style="font-family:'宋体';">
       事件绑定可以阻止冒泡事件向上冒泡。
      </span>
     </span>
     <span style="color:rgb(51,51,51);">
      <span style="font-family:'宋体';">
       我们使用
      </span>
     </span>
     <span style="color:rgb(51,51,51);">
      catch绑定手指触摸事件，
     </span>
     <span style="color:rgb(62,62,62);">
      <span style="font-family:'宋体';">
       然后他的父节点就不会收到事件冒泡，这样就实现了
      </span>
     </span>
     <span style="color:rgb(62,62,62);">
      <span style="font-family:'宋体';">
       禁止页面上下滑动
      </span>
     </span>
    </p>
    <p align="justify">
     <span style="color:rgb(62,62,62);">
      <span style="font-family:'宋体';">
       <br/>
      </span>
     </span>
    </p>
