    <h2>
     <a id="SecureCRT__0">
     </a>
     SecureCRT 文件上传下载操作指南
    </h2>
    <h3>
     <a id="SFTP_2">
     </a>
     一、开启SFTP会话
    </h3>
    <ol>
     <li>
      <strong>
       连接SFTP
      </strong>
      <br/>
      右键点击SecureCRT会话窗口标题栏 → 选择
      <strong>
       “连接 SFTP 会话”
      </strong>
      ，或使用快捷键
      <code>
       Alt+P
      </code>
      打开SFTP面板。
      <ul>
       <li>
        <strong>
         确认连接成功
        </strong>
        ：成功连接后，命令行提示符变为
        <code>
         sftp&gt;
        </code>
        ，输入
        <code>
         help
        </code>
        可查看支持的命令列表。
       </li>
       <li>
        <strong>
         前提条件
        </strong>
        ：确保远程服务器已启用SSH服务并支持SFTP（默认端口22）。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_10">
     </a>
     二、目录操作命令
    </h3>
    <h4>
     <a id="_11">
     </a>
     本地目录操作
    </h4>
    <ul>
     <li>
      <strong>
       查看本地当前路径
      </strong>
      <pre><code class="prism language-bash">lpwd
</code></pre>
     </li>
     <li>
      <strong>
       列出本地目录内容
      </strong>
      <pre><code class="prism language-bash">lls <span class="token punctuation">[</span>选项<span class="token punctuation">]</span> <span class="token punctuation">[</span>目录路径<span class="token punctuation">]</span>
</code></pre>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-bash">lls <span class="token parameter variable">-l</span> /home/user/docs  <span class="token comment"># 详细列表格式</span>
lls *.txt              <span class="token comment"># 列出当前目录下所有.txt文件</span>
</code></pre>
     </li>
     <li>
      <strong>
       切换本地目录
      </strong>
      <pre><code class="prism language-bash">lcd <span class="token punctuation">[</span>目录路径<span class="token punctuation">]</span>
</code></pre>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-bash">lcd ~/Downloads  <span class="token comment"># 切换到用户下载目录</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="_34">
     </a>
     服务器目录操作
    </h4>
    <ul>
     <li>
      <strong>
       查看服务器当前路径
      </strong>
      <pre><code class="prism language-bash"><span class="token builtin class-name">pwd</span>
</code></pre>
     </li>
     <li>
      <strong>
       列出服务器目录内容
      </strong>
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token punctuation">[</span>选项<span class="token punctuation">]</span> <span class="token punctuation">[</span>目录路径<span class="token punctuation">]</span>
</code></pre>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-al</span> /var/log  <span class="token comment"># 列出隐藏文件及权限信息</span>
</code></pre>
     </li>
     <li>
      <strong>
       切换服务器目录
      </strong>
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> <span class="token punctuation">[</span>目录路径<span class="token punctuation">]</span>
</code></pre>
      <strong>
       注意
      </strong>
      ：若目录含空格或特殊字符，需用引号包裹，如
      <code>
       cd "My Documents"
      </code>
      。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_55">
     </a>
     三、文件传输命令
    </h3>
    <h4>
     <a id="___56">
     </a>
     上传文件（本地 → 服务器）
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        上传单个/多个文件
       </strong>
      </p>
      <pre><code class="prism language-bash">put <span class="token punctuation">[</span>选项<span class="token punctuation">]</span> 本地文件 <span class="token punctuation">[</span>服务器路径<span class="token punctuation">]</span>
</code></pre>
      <p>
       <strong>
        常用选项
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         -r
        </code>
        ：递归上传目录（需SecureCRT 8.0+）
       </li>
       <li>
        <code>
         -l 1000
        </code>
        ：限制传输速度为1000 KB/s
       </li>
      </ul>
      <p>
       <strong>
        示例
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">put report.pdf                  <span class="token comment"># 上传单个文件</span>
put *.jpg /var/www/images/      <span class="token comment"># 上传所有.jpg到指定目录</span>
put <span class="token parameter variable">-r</span> project_folder           <span class="token comment"># 递归上传整个目录</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="___72">
     </a>
     下载文件（服务器 → 本地）
    </h4>
    <ul>
     <li>
      <strong>
       下载单个/多个文件
      </strong>
      <pre><code class="prism language-bash">get <span class="token punctuation">[</span>选项<span class="token punctuation">]</span> 服务器文件 <span class="token punctuation">[</span>本地路径<span class="token punctuation">]</span>
</code></pre>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-bash">get backup.zip                 <span class="token comment"># 下载到本地当前目录</span>
get /var/log/*.log ~/logs/     <span class="token comment"># 下载所有.log文件到本地logs目录</span>
get <span class="token parameter variable">-r</span> configs/                <span class="token comment"># 递归下载整个目录</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_86">
     </a>
     四、高级功能与技巧
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        传输模式设置
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         二进制模式
        </strong>
        （默认）：适用于图片、压缩包等
        <pre><code class="prism language-bash">binary
</code></pre>
       </li>
       <li>
        <strong>
         ASCII模式
        </strong>
        ：用于文本文件，自动转换换行符
        <pre><code class="prism language-bash">ascii
</code></pre>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        断点续传
       </strong>
       <br/>
       使用
       <code>
        reput
       </code>
       或
       <code>
        reget
       </code>
       继续中断的传输：
      </p>
      <pre><code class="prism language-bash">reput interrupted_file.zip
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        文件校验
       </strong>
       <br/>
       传输后建议校验文件完整性：
      </p>
      <pre><code class="prism language-bash"><span class="token comment"># 服务器端</span>
md5sum server_file.txt
<span class="token comment"># 本地</span>
certutil <span class="token parameter variable">-hashfile</span> local_file.txt MD5
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        图形化拖放传输
       </strong>
      </p>
      <ul>
       <li>
        直接拖拽文件到SFTP窗口上传，或从窗口拖出文件到本地下载。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_117">
     </a>
     五、错误处理与常见问题
    </h3>
    <ul>
     <li>
      <strong>
       权限被拒绝
      </strong>
      <br/>
      使用
      <code>
       sudo
      </code>
      或联系管理员调整目录权限。
     </li>
     <li>
      <strong>
       文件不存在
      </strong>
      <br/>
      检查路径拼写，使用
      <code>
       lls
      </code>
      或
      <code>
       ls
      </code>
      确认文件是否存在。
     </li>
     <li>
      <strong>
       连接超时
      </strong>
      <br/>
      检查网络或调整会话设置：
      <strong>
       Options → Session Options → Terminal → Send protocol NO-OP
      </strong>
      设为每120秒。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="SFTP_127">
     </a>
     六、退出SFTP
    </h3>
    <pre><code class="prism language-bash"><span class="token builtin class-name">exit</span>
或
quit
</code></pre>
    <p>
     <strong>
      注意
     </strong>
     ：退出前确保传输完成，避免数据丢失。
    </p>
