# Visual Studio 2022新建c语言项目的详细步骤

### 步骤1：点击创建新项目

![](https://i-blog.csdnimg.cn/direct/e704e1a78861448f846bcc73472b015b.png)

### 步骤2：到了项目模板

--> 选择“
控制台应用
” (在window终端运行代码。默认打印"Hello World")

--> 点击 “
下一步
”

![](https://i-blog.csdnimg.cn/direct/9e8b791416f049a49e56912f6182b398.png)

### 步骤3：到了配置新项目模块

--> 输入“
项目名称
”

--> 更改“
位置
”路径，一般建议存放除C盘外的盘，例如D盘新建“vs\_projects”文件夹，然后存放

--> 点击“
创建
”

![](https://i-blog.csdnimg.cn/direct/fc0ade40ffbf4c1696cbea6265041e32.png)

### 步骤4：到了编辑模块

--> 找到
源文件
下默认新建好的
.cpp文件
，然后点击“
右键
”

--> 选择“
移除
”

--> 在弹窗中选择“
删除
”

![](https://i-blog.csdnimg.cn/direct/b8e7821323394f658ad5e8680b9e281f.png)

### 步骤5：自己添加用于编辑c语言代码的文件夹

--> 点击“
源文件
”，然后右键

-->选择 “
添加
”  “新建项”

-->输入自己想取的文件名后，点击“
添加
”

![](https://i-blog.csdnimg.cn/direct/f3c46b069efa45fca5c397f87f7756fb.png)

![](https://i-blog.csdnimg.cn/direct/22141a300af747d3bc5c1ee133db9344.png)

### 注意点：

如果还想重复步骤5进行添加.cpp文件夹，记得把之前源文件下的main()函数注释掉,否则新建的编辑环境运行会报错！！！

（一个源文件下只能有一个入口函数 ，即源文件下main()函数只能有一个）
