    <h3>
     <a id="_136_25">
     </a>
     一、只出现一次的数字（题目 136）
    </h3>
    <h4>
     <a id="1__27">
     </a>
     1. 题目描述
    </h4>
    <p>
     给定一个非空整数数组，其中除了一个元素只出现一次外，其他元素都出现两次。找到那个只出现一次的元素。
    </p>
    <h4>
     <a id="2__31">
     </a>
     2. 示例
    </h4>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     输入：
     <code>
      nums = [2, 2, 1]
     </code>
    </p>
    <p>
     输出：
     <code>
      1
     </code>
    </p>
    <h4>
     <a id="3__39">
     </a>
     3. 解题思路
    </h4>
    <p>
     这道题主要考察位运算的应用。我们可以使用异或运算来找到只出现一次的数字。异或运算的性质是：一个数异或自己为 0，一个数异或 0 为自己。因此，遍历数组，将所有元素异或在一起，最终结果就是只出现一次的数字。
    </p>

     二、多数元素（题目 169）
    </h3>
    <h4>
     <a id="1__64">
     </a>
     1. 题目描述
    </h4>
    <p>
     给定一个大小为 n 的数组，找到出现次数大于 ⌊ n/2 ⌋ 的元素。
    </p>
    <h4>
     <a id="2__68">
     </a>
     2. 示例
    </h4>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     输入：
     <code>
      nums = [3, 2, 3]
     </code>
    </p>
    <p>
     输出：
     <code>
      3
     </code>
    </p>
    <h4>
     <a id="3__76">
     </a>
     3. 解题思路
    </h4>
    <p>
     这道题主要考察 Boyer-Moore 投票算法的应用。我们可以维护一个候选元素和其计数，遍历数组，如果当前元素与候选元素相同，则计数加 1，否则计数减 1。当计数为 0 时，更新候选元素为当前元素。

    <h3>
     <a id="_31_159">
     </a>
     四、下一个排列（题目 31）
    </h3>
    <h4>
     <a id="1__161">
     </a>
     1. 题目描述
    </h4>
    <p>
     给定一个整数数组，重新排列成字典序的下一个排列。如果不存在下一个排列，则重新排列成最低的排列。
    </p>
    <h4>
     <a id="2__165">
     </a>
     2. 示例
    </h4>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     输入：
     <code>
      nums = [1, 2, 3]
     </code>
    </p>
    <p>
     输出：
     <code>
      [1, 3, 2]
     </code>
    </p>
    <h4>
     <a id="3__173">
     </a>
     3. 解题思路
    </h4>
    <p>
     这道题主要考察数组的操作和排列的生成。我们可以按照以下步骤生成下一个排列：
    </p>
    <ol>
     <li>
      从后向前找到第一个可以增大的数。
     </li>
     <li>
      从后向前找到第一个比该数大的数，并交换。
     </li>
     <li>
      将该数后面的部分反转，使其最小。
     </li>
    </ol>
    <h4>
     <a id="4_Java_181">
     </a>
     4. 代码实现（Java）

    <h3>
     <a id="_287_221">
     </a>
     五、寻找重复数（题目 287）
    </h3>
    <h4>
     <a id="1__223">
     </a>
     1. 题目描述
    </h4>
    <p>
     给定一个包含 n + 1 个整数的数组，其中每个元素在 1 到 n 之间，至少存在一个重复元素。找到那个重复的元素。
    </p>
    <h4>
     <a id="2__227">
     </a>
     2. 示例
    </h4>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     输入：
     <code>
      nums = [1, 3, 4, 2, 2]
     </code>
    </p>
    <p>
     输出：
     <code>
      2
     </code>
    </p>
    <h4>
     <a id="3__235">
     </a>
     3. 解题思路
    </h4>
    <p>
     这道题主要考察二分查找的应用。我们可以使用二分查找来确定重复元素。具体步骤如下：
    </p>
    <ol>
     <li>
      初始化左右边界为 1 和 n。
     </li>
     <li>
      计算中间值 mid，统计数组中小于等于 mid 的元素个数。
     </li>
     <li>
      如果个数大于 mid，则重复元素在左半部分，否则在右半部分。
     </li>
     <li>
      重复步骤 2 和 3，直到找到重复元素。
     </li>
    </ol>
