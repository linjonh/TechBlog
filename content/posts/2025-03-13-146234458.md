    <h1 class="title-article" id="articleContentId">
     linux centos 忘记root密码拯救
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     在CentOS 7中，如果忘记root密码，可以通过修改系统启动参数进入单用户模式或紧急模式进行重置。以下是两种常用方法，适用于物理机或虚拟机环境：
    </p>
    <h4>
     <strong>
      方法一：通过
      <code>
       rd.break
      </code>
      参数重置密码
     </strong>
    </h4>
    <p>
     <strong>
      步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        重启系统并进入GRUB菜单
       </strong>
       在开机界面出现时，按任意键停止自动启动，按方向键选择默认启动项（通常为第一行），按
       <code>
        e
       </code>
       进入编辑模式
      </p>
     </li>
     <li>
      <p>
       <strong>
        修改启动参数
       </strong>
       找到以
       <code>
        linux16
       </code>
       或
       <code>
        linux
       </code>
       开头的行，在行末添加
       <code>
        rd.break
       </code>
       （注意与前后参数用空格隔开），按
       <code>
        Ctrl+X
       </code>
       启动系统
      </p>
     </li>
     <li>
      <p>
       <strong>
        挂载文件系统并切换环境
       </strong>
      </p>
      <pre><code>mount -o remount,rw /sysroot # 挂载根目录为可读写
chroot /sysroot # 切换到系统根目录环境</code></pre>
     </li>
     <li>
      <strong>
       重置密码
      </strong>
     </li>
    </ol>
    <pre><code>passwd root # 输入两次新密码
touch /.autorelabel # 若系统启用SELinux需执行此命令 
exit # 退出chroot环境
reboot # 重启系统</code></pre>
    <h4>
     <strong>
      方法二：通过
      <code>
       init=/bin/sh
      </code>
      参数重置密码
     </strong>
    </h4>
    <p>
     <strong>
      步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        进入GRUB编辑模式
       </strong>
       同样在开机界面按
       <code>
        e
       </code>
       键编辑启动项，找到
       <code>
        linux16
       </code>
       行，将
       <code>
        ro
       </code>
       改为
       <code>
        rw init=/bin/sh
       </code>
       （或直接在行末追加
       <code>
        init=/bin/sh
       </code>
       ），按
       <code>
        Ctrl+X
       </code>
       启动
      </p>
     </li>
     <li>
      <p>
       <strong>
        挂载根目录并重置密码
       </strong>
      </p>
      <pre><code>mount -o remount,rw / # 确保根目录可写
passwd root # 输入新密码
touch /.autorelabel # 处理SELinux标签
exec /sbin/init # 或直接输入reboot 重启</code></pre>
     </li>
    </ol>
    <p>
    </p>
    <h4>
     <strong>
      注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        SELinux处理
       </strong>
       若系统启用了SELinux，必须执行
       <code>
        touch /.autorelabel
       </code>
       以更新安全上下文，否则可能导致无法登录
      </p>
     </li>
    </ol>
    <p>
