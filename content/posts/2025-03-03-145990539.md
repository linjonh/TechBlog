     使用SDKMAN!安装springboot
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="markdown_views prism-atom-one-dark" id="content_views">
    <svg style="display: none;" xmlns="http://www.w3.org/2000/svg">
     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     在 Ubuntu 环境中使用
     <code>
      sdk install springboot
     </code>
     命令之前，您需要先安装
     <strong>
      SDKMAN!
     </strong>
     （Software Development Kit Manager）。以下是详细的安装步骤：
    </p>
    <hr/>
    <h4>
     <a id="_SDKMAN_4">
     </a>
     <strong>
      安装 SDKMAN!
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        打开终端
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        运行以下命令以安装 SDKMAN!
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token function">curl</span> <span class="token parameter variable">-s</span> <span class="token string">"https://get.sdkman.io"</span> <span class="token operator">|</span> <span class="token function">bash</span>
</code></pre>
      <p>
       这个脚本会自动下载并安装 SDKMAN!，并将其配置到您的用户目录中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        初始化 SDKMAN!
       </strong>
       ：
       <br/>
       安装完成后，需要初始化 SDKMAN!。运行以下命令：
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> <span class="token string">"<span class="token environment constant">$HOME</span>/.sdkman/bin/sdkman-init.sh"</span>
</code></pre>
      <p>
       或者，您也可以重新打开终端窗口，让初始化脚本自动加载。
      </p>
     </li>
     <li>
      <p>
       <strong>
        验证安装
       </strong>
       ：
       <br/>
       运行以下命令验证 SDKMAN! 是否正确安装：
      </p>
      <pre><code class="prism language-bash">sdk version
</code></pre>
      <p>
       如果安装成功，您将看到 SDKMAN! 的版本信息。
      </p>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_Java_32">
     </a>
     <strong>
      安装 Java
     </strong>
    </h4>
    <p>
     Spring Boot 需要 Java 环境才能运行。如果您尚未安装 Java，可以通过 SDKMAN! 安装 OpenJDK。推荐使用 Java 17 或更高版本。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        安装 OpenJDK
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">sdk <span class="token function">install</span> <span class="token function">java</span> <span class="token number">17.0</span>.1-open
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        设置默认 Java 版本
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">sdk default <span class="token function">java</span> <span class="token number">17.0</span>.1-open
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        验证 Java 安装
       </strong>
       ：
       <br/>
       运行以下命令验证 Java 是否正确安装：
      </p>
      <pre><code class="prism language-bash"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_Spring_Boot_54">
     </a>
     <strong>
      安装 Spring Boot
     </strong>
    </h4>
    <p>
     完成上述步骤后，您就可以使用 SDKMAN! 安装 Spring Boot 了。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        安装 Spring Boot
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">sdk <span class="token function">install</span> springboot
</code></pre>
      <p>
       如果您想安装特定版本的 Spring Boot，可以指定版本号，例如：
      </p>
      <pre><code class="prism language-bash">sdk <span class="token function">install</span> springboot <span class="token number">3.1</span>.0
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        验证 Spring Boot 安装
       </strong>
       ：
       <br/>
       运行以下命令验证 Spring Boot 是否正确安装：
      </p>
      <pre><code class="prism language-bash">spring <span class="token parameter variable">--version</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_76">
     </a>
     <strong>
      完整步骤总结
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        安装 SDKMAN!
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token function">curl</span> <span class="token parameter variable">-s</span> <span class="token string">"https://get.sdkman.io"</span> <span class="token operator">|</span> <span class="token function">bash</span>
<span class="token builtin class-name">source</span> <span class="token string">"<span class="token environment constant">$HOME</span>/.sdkman/bin/sdkman-init.sh"</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        安装 Java
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">sdk <span class="token function">install</span> <span class="token function">java</span> <span class="token number">17.0</span>.1-open
sdk default <span class="token function">java</span> <span class="token number">17.0</span>.1-open
<span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        安装 Spring Boot
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">sdk <span class="token function">install</span> springboot
spring <span class="token parameter variable">--version</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_99">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       网络问题
      </strong>
      ：如果您的网络环境访问外网受限，可能需要配置代理或使用国内镜像。
     </li>
     <li>
      <strong>
       权限问题
      </strong>
      ：确保您有足够的权限执行上述命令。如果遇到权限问题，可以尝试使用
      <code>
       sudo
      </code>
      。
     </li>
    </ul>
    <p>
     完成这些步骤后，您就可以在 Ubuntu 环境中使用 Spring Boot 进行开发了。
    </p>
