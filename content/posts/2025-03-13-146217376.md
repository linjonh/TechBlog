    <h3>
     <span style="color:#0d0016">
      1.接受一个工作表
     </span>
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      在 11 案例用到的 Excel 工作簿中，数据是从第一张工作表的 A1 单元格开始的。但在实际场景中， Excel 文件可能并没有这么规整。所以 panda 提供了一些参数来优化读取过程。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      比如 store.xlsx 文件的三个工作表都是这样：
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <img alt="" height="455" src="https://i-blog.csdnimg.cn/direct/1cf0c925799a4d08998fc93f682961b7.png" width="1483"/>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      通过 sheet_name、skiprows 和 usecols 这些参数，可以告诉 pandas 关于我们想要读取的列的详细信息。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <img alt="" height="779" src="https://i-blog.csdnimg.cn/direct/2cf1fe5783c34ad2ae7cfcfdfa026135.png" width="1749"/>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        sheet_name="2019"
       </code>
      </strong>
      :指定了要读取的工作表名称。sheet_name 接受工作表切片。闭区间，包括起始和结束的工作表。
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#0d0016">
       单个工作表:
       <code>
        sheet_name="Sheet1"
       </code>
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       多个工作表（列表）:
       <code>
        sheet_name=["Sheet1", "Sheet2"]
       </code>
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       工作表切片:
       <code>
        sheet_name="Sheet1:Sheet3"
       </code>
       或
       <code>
        sheet_name=0:3
       </code>
       （使用索引）
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        skiprows=1
       </code>
      </strong>
      :用于跳过文件开头指定数量的行。在这个例子中，
      <code>
       skiprows=1
      </code>
      表示跳过文件的第一行，从第二行开始读取数据。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        usecols="B:F"
       </code>
      </strong>
      :用于指定要读取的列范围。读取从列 B 到列 F 的数据（包括 B 和 F 列）。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <img alt="" height="1081" src="https://i-blog.csdnimg.cn/direct/6d4e97d665a947c082620b13ff7030a2.png" width="2531"/>
     </span>
    </p>
    <blockquote>
     <p>
      <span style="color:#0d0016">
       补充：Flagship 的数据类型应该是 bool 而不是 object。要修正这一问题，需要提供一个转换函数来处理某列中发生冲突的单元格。（编写 fix_missing 函数或者写一个lambda 表达式）
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       将
       <code>
        Flagship
       </code>
       列中的非布尔值处理为
       <code>
        False
       </code>
       ，而将有效的布尔值保留为
       <code>
        True
       </code>
       或
       <code>
        False
       </code>
       。
      </span>
     </p>
     <p>
     </p>
     <p class="img-center">
      <img alt="" height="403" src="https://i-blog.csdnimg.cn/direct/f6200232b0c04082909686871327ed25.png" width="1489"/>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <code>
        Flagship
       </code>
       列的数据类型是
       <code>
        object
       </code>
       ，而不是
       <code>
        bool
       </code>
       。是因为该列中包含了一些非布尔值（如
       <code>
        MISSING
       </code>
       和
       <code>
        NaN
       </code>
       ），导致 pandas 无法自动将其转换为布尔类型。
      </span>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <strong>
        1.编写函数
       </strong>
      </span>
     </p>
     <p>
     </p>
     <pre><code class="language-python">import pandas as pd
import numpy as np

# 定义 fix_missing 函数
def fix_missing(value):
    if isinstance(value, bool):
        return value
    else:
        return False

# 读取数据
df = pd.read_excel("xl/stores.xlsx", sheet_name="2019", skiprows=1, usecols="B:F")

# 应用 fix_missing 函数到 Flagship 列
df['Flagship'] = df['Flagship'].apply(fix_missing)

# 打印修正后的 DataFrame
print(df)</code></pre>
     <p>
      <span style="color:#0d0016">
       <strong>
        2.使用 lambda 表达式
       </strong>
      </span>
     </p>
     <p>
     </p>
     <pre><code class="language-python">import pandas as pd

# 读取数据
df = pd.read_excel("xl/stores.xlsx", sheet_name="2019", skiprows=1, usecols="B:F")

# 使用 lambda 表达式将 Flagship 列中非布尔值转换为 False
df['Flagship'] = df['Flagship'].apply(lambda x: x if isinstance(x, bool) else False)

# 打印修正后的 DataFrame
print(df)</code></pre>
     <p>
      <span style="color:#0d0016">
       <code>
        apply()
       </code>
       方法用于对 DataFrame 的每一列应用一个函数。
      </span>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <code>
        lambda x: x if isinstance(x, bool) else False
       </code>
       是一个匿名函数，它检查每个元素
       <code>
        x
       </code>
       。
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       如果
       <code>
        x
       </code>
       是布尔类型（
       <code>
        True
       </code>
       或
       <code>
        False
       </code>
       ），则保留原值。如果
       <code>
        x
       </code>
       不是布尔类型（例如，
       <code>
        MISSING
       </code>
       或
       <code>
        NaN
       </code>
       ），则将其转换为
       <code>
        False
       </code>
       。
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       <code>
        isinstance(x, bool)
       </code>
       是一个内置函数，用于检查变量
       <code>
        x
       </code>
       是否是布尔类型 (
       <code>
        bool
       </code>
       )。
      </span>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <strong>
        3.结合使用
       </strong>
      </span>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <code>
        converters
       </code>
       参数用于指定列的自定义转换函数。这允许你在数据读取过程中，对特定列的数据进行预处理或转换。
       <code>
        converters
       </code>
       是一个字典，其中键是列名（或列索引），值是应用于该列的转换函数。当
       <code>
        pandas.read_excel
       </code>
       读取数据时，会使用这些转换函数来处理指定列的数据。
      </span>
     </p>
     <pre><code class="language-python">def fix_missing(x):
    return False if x in ["", "MISSING"] else x

df = pd.read_excel(
    "xl/stores.xlsx",
    sheet_name="2019",
    skiprows=1,
    usecols="B:F",
    converters={"Flagship": fix_missing})</code></pre>
     <p>
      <span style="color:#0d0016">
       <strong>
        <code>
         fix_missing
        </code>
       </strong>
       函数：这个函数用于检查传入的值
       <code>
        x
       </code>
       。
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       如果
       <code>
        x
       </code>
       是一个空字符串
       <code>
        ""
       </code>
       或字符串
       <code>
        "MISSING"
       </code>
       ，则返回
       <code>
        False
       </code>
       。否则，返回原始值
       <code>
        x
       </code>
       。
      </span>
     </p>
     <p>
     </p>
     <p>
      <span style="color:#0d0016">
       <strong>
        <code>
         converters={"Flagship": fix_missing}
        </code>
       </strong>
       :
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       这里指定了一个转换字典，其中键是
       <code>
        "Flagship"
       </code>
       ，表示要对
       <code>
        Flagship
       </code>
       列应用转换。值是
       <code>
        fix_missing
       </code>
       函数，这意味着在读取
       <code>
        Flagship
       </code>
       列的数据时，每个单元格的值都会通过
       <code>
        fix_missing
       </code>
       函数进行处理。
      </span>
     </p>
    </blockquote>

    <h3>
     <span style="color:#0d0016">
      2.接受一个工作表名称列表
     </span>
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      在
      <code>
       pandas
      </code>
      库中，
      <code>
       read_excel
      </code>
      函数可以读取 Excel 文件中的一个或多个工作表。当你想要读取多个工作表时，可以通过传递一个工作表名称的列表或者使用
      <code>
       sheet_name=None
      </code>
      来读取所有的工作表。这将返回一个字典，其中键是工作表名称，值是对应的
      <code>
       DataFrame
      </code>
      。
     </span>
    </p>
    <p>
    </p>
    <p id="">
     <span style="color:#0d0016">
      假设你有一个 Excel 文件
      <code>
       data.xlsx
      </code>
      ，其中包含以下三个工作表：
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      Sheet1: 包含学生基本信息。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      Sheet2: 包含学生的成绩。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      Sheet3: 包含学生的出勤记录。
     </span>
    </p>
    <pre><code class="language-python">import pandas as pd

# 读取 Excel 文件中的所有工作表
sheets = pd.read_excel("data.xlsx", sheet_name=None)

# `sheets` 是一个字典，键是工作表名称，值是对应的 DataFrame
print(sheets.keys())  # 输出: dict_keys(['Sheet1', 'Sheet2', 'Sheet3'])

# 访问特定工作表的 DataFrame
students_info = sheets['Sheet1']
students_grades = sheets['Sheet2']
students_attendance = sheets['Sheet3']

# 打印 Sheet1 的数据
print(students_info)</code></pre>

# 读取 Excel 文件中的两个工作表 "2019" 和 "2020"

# 打印 2019 年工作表的前两行

      3.设置列名

    <p>
    </p>
    <p>
     <span style="color:#0d0016">

df = pd.read_excel(

    usecols="B:C,F", 
    header=None,

)</code></pre>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        sheet_name=0：
       </code>
      </strong>
      指定读取 Excel 文件的第一个工作表。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        skiprows=2：
       </code>
      </strong>
      跳过文件开头的 2 行。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        skipfooter=3：
       </code>
      </strong>
      跳过文件末尾的 3 行。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       <code>
        usecols="B:C,F"：
       </code>
      </strong>
      指定要读取的列。
      <code>
       "B:C"
      </code>
      表示读取从 B 列到 C 列的所有列（包括 B 和 C 列）。

       <code>

      </strong>
      ：由于
      <code>
       header=None
      </code>
      ，我们需要通过
      <code>
       names
      </code>
      参数提供自定义的列名。

      <code>
       "Branch"

      、
      <code>

      </code>
      和
      <code>
       "Is_Flagship"
      </code>
      。

    <h3>
     <span style="color:#0d0016">
      4.处理缺失值
     </span>
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      使用
      <code>
       na_values
      </code>
      和
      <code>
       keep_default_na 参数。
      </code>
     </span>
    </p>
    <pre><code class="language-python">import pandas as pd
 
df = pd.read_excel(
    "xl/stores.xlsx", 
    sheet_name="2019", 
    skiprows=1, 
    usecols="B,C,F", 
    skipfooter=2, 
    na_values="MISSING", 
    keep_default_na=False
)</code></pre>
    <div>
     <p>
      <span style="color:#0d0016">
       <strong>
        <code>
         na_values="MISSING"
        </code>
       </strong>
       :指定将
       <code>
        "MISSING"
       </code>
       视为缺失值（
       <code>
        NaN
       </code>
       ）。任何在数据中出现的
       <code>
        "MISSING"
       </code>
       都会被转换为
       <code>
        NaN
       </code>
       。
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       <strong>
        <code>
         keep_default_na=False
        </code>
       </strong>
       :指定不保留
       <code>
        pandas
       </code>
       默认的缺失值标识符。默认情况下，
       <code>
        pandas
       </code>
       会将空字符串
       <code>
        ""
       </code>
       、
       <code>
        'nan'
       </code>
       、
       <code>
        'null'
       </code>
       、
       <code>
        'NULL'
       </code>
       等识别为缺失值并转换为
       <code>
        NaN
       </code>
       。设置为
       <code>
        False
       </code>
       后，只有
       <code>
        na_values
       </code>
       中指定的值（如
       <code>
        "MISSING"
       </code>
       ) 会被视为缺失值。
      </span>
     </p>

    <h2>
     <span style="color:#0d0016">
      二 ExcelFile 类
     </span>
    </h2>
    <p>
    </p>
    <h3>
     <span style="color:#0d0016">
      1.什么是 ExcelFile 类
     </span>
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      使用
      <code>
       ExcelFile
      </code>
      类，你可以一次性加载整个 Excel 文件，然后根据需要选择性地解析各个工作表，从而提高数据处理的效率。
     </span>
    </p>
    <p>
    </p>
    <h3>
     <span style="color:#0d0016">
      2.为什么有 ExcelFile 类
     </span>
    </h3>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      当需要从同一个 Excel 文件中读取多个工作表时，直接使用
      <code>
       read_excel
      </code>
      函数可能会多次打开和读取整个文件，导致性能下降。而
      <code>
       ExcelFile
      </code>
      类允许你一次性打开文件，并在内存中保持打开状态，从而避免了重复的文件读取操作。
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="color:#0d0016">
      <code>
       ExcelFile
      </code>
      类可以被用作上下文管理器（使用
      <code>
       with
      </code>
      语句），这意味着文件会在使用后被自动关闭，从而避免了资源泄漏的问题。当使用
      <code>
       with
      </code>
      语句来打开
      <code>
       ExcelFile
      </code>
      时，
      <code>
       pandas
      </code>
