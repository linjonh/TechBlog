    <h4>
     <a id="Hadoop_FS__1">
     </a>
     <strong>
      一、Hadoop FS 基础操作命令
     </strong>
    </h4>
    <h5>
     <a id="1__3">
     </a>
     1. 查看帮助
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-help</span> <span class="token punctuation">[</span>命令名<span class="token punctuation">]</span>  <span class="token comment"># 查看具体命令的帮助文档</span>
<span class="token comment"># 示例：</span>
hadoop fs <span class="token parameter variable">-help</span> <span class="token function">mkdir</span>
</code></pre>
    <h5>
     <a id="2__10">
     </a>
     2. 目录操作
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-mkdir</span> /path          <span class="token comment"># 创建目录</span>
hadoop fs <span class="token parameter variable">-mkdir</span> <span class="token parameter variable">-p</span> /path/a/b   <span class="token comment"># 递归创建多级目录</span>
hadoop fs <span class="token parameter variable">-rmdir</span> /path          <span class="token comment"># 删除空目录</span>
</code></pre>
    <h5>
     <a id="3__17">
     </a>
     3. 文件操作
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-put</span> local_file /hdfs_path   <span class="token comment"># 上传本地文件到HDFS</span>
hadoop fs <span class="token parameter variable">-get</span> /hdfs_file local_path   <span class="token comment"># 下载HDFS文件到本地</span>
hadoop fs <span class="token parameter variable">-cat</span> /hdfs_file              <span class="token comment"># 查看文件内容</span>
hadoop fs <span class="token parameter variable">-tail</span> /hdfs_file             <span class="token comment"># 查看文件尾部内容</span>
hadoop fs <span class="token parameter variable">-text</span> /hdfs_file             <span class="token comment"># 查看文件内容（自动解压）</span>
hadoop fs <span class="token parameter variable">-cp</span> /src /dest               <span class="token comment"># 复制文件/目录</span>
hadoop fs <span class="token parameter variable">-mv</span> /src /dest               <span class="token comment"># 移动文件/目录</span>
hadoop fs <span class="token parameter variable">-rm</span> /file                    <span class="token comment"># 删除文件</span>
hadoop fs <span class="token parameter variable">-rm</span> <span class="token parameter variable">-r</span> /directory            <span class="token comment"># 递归删除目录</span>
</code></pre>
    <h5>
     <a id="4__30">
     </a>
     4. 查看目录/文件信息
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-ls</span> /path          <span class="token comment"># 列出目录内容</span>
hadoop fs <span class="token parameter variable">-ls</span> <span class="token parameter variable">-h</span> /path       <span class="token comment"># 友好格式显示文件大小（KB/MB/GB）</span>
hadoop fs <span class="token parameter variable">-ls</span> <span class="token parameter variable">-R</span> /path       <span class="token comment"># 递归列出子目录</span>
hadoop fs <span class="token parameter variable">-du</span> <span class="token parameter variable">-h</span> /path       <span class="token comment"># 查看目录/文件占用空间</span>
hadoop fs <span class="token parameter variable">-stat</span> /file        <span class="token comment"># 查看文件详细信息</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_41">
     </a>
     <strong>
      二、权限管理
     </strong>
    </h4>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-chmod</span> <span class="token number">755</span> /path        <span class="token comment"># 修改权限（八进制模式）</span>
hadoop fs <span class="token parameter variable">-chmod</span> <span class="token parameter variable">-R</span> a+rwx /path   <span class="token comment"># 递归修改权限（符号模式）</span>
hadoop fs <span class="token parameter variable">-chown</span> user:group /path <span class="token comment"># 修改文件所属用户和组</span>
hadoop fs <span class="token parameter variable">-chgrp</span> group /path      <span class="token comment"># 修改文件所属组</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_52">
     </a>
     <strong>
      三、集群管理与状态查询
     </strong>
    </h4>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-df</span> <span class="token parameter variable">-h</span>          <span class="token comment"># 查看HDFS存储空间使用情况</span>
hadoop fs <span class="token parameter variable">-du</span> <span class="token parameter variable">-h</span> /path    <span class="token comment"># 统计目录占用空间</span>
hadoop fs <span class="token parameter variable">-count</span> /path    <span class="token comment"># 查看目录下的文件/目录数量</span>
hadoop fsadmin <span class="token parameter variable">-safemode</span> enter  <span class="token comment"># 进入安全模式（管理员）</span>
hadoop fsadmin <span class="token parameter variable">-safemode</span> leave  <span class="token comment"># 退出安全模式（管理员）</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_64">
     </a>
     <strong>
      四、高级功能
     </strong>
    </h4>
    <h5>
     <a id="1__66">
     </a>
     1. 回收站操作（需开启回收站功能）
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-rm</span> <span class="token parameter variable">-skipTrash</span> /file   <span class="token comment"># 直接删除（不进入回收站）</span>
<span class="token comment"># 默认删除操作会移动到回收站（路径：/user/${username}/.Trash）</span>
</code></pre>
    <h5>
     <a id="2_HAR_72">
     </a>
     2. 文件归档（HAR）
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-archive</span> <span class="token parameter variable">-archiveName</span> myhar.har <span class="token parameter variable">-p</span> /src /dest  <span class="token comment"># 创建归档文件</span>
hadoop fs <span class="token parameter variable">-ls</span> har:///myhar.har   <span class="token comment"># 查看归档内容</span>
</code></pre>
    <h5>
     <a id="3__78">
     </a>
     3. 设置副本数
    </h5>
    <pre><code class="prism language-bash">hadoop fs <span class="token parameter variable">-setrep</span> <span class="token parameter variable">-w</span> <span class="token number">2</span> /file   <span class="token comment"># 修改副本数为2（-w表示等待完成）</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_85">
     </a>
     <strong>
      五、注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <code>
       hadoop fs
      </code>
      是通用命令，兼容HDFS和其他Hadoop支持的文件系统。
     </li>
     <li>
      删除目录时需谨慎使用
      <code>
       -rm -r
      </code>
      ，建议先确认路径。
     </li>
     <li>
      安全模式操作需管理员权限。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_92">
     </a>
     <strong>
      结语
     </strong>
    </h4>
    <p>
     掌握这些
     <code>
      hadoop fs
     </code>
     命令是Hadoop开发与运维的基础，建议结合实践场景多操作演练。后续可深入学习HDFS架构与API开发技巧！
    </p>
