    <h3 id="w-e-element-395">
     <strong>
      一、
     </strong>
     Ollama+Deepseek-R1+Continue
     <strong>
      本地集成VScode
     </strong>
    </h3>
    <h5 id="w-e-element-399">
     1）安装前知识点
    </h5>
    <p id="w-e-element-401">
     Continue 介绍
    </p>
    <p id="w-e-element-403">
     详情可参照官网：
     <span contenteditable="false" tabindex="-1">
      <a href="https://www.continue.dev/" rel="nofollow noopener noreferrer" target="_blank" title="continue官网">
       continue官网
      </a>
     </span>
    </p>
    <p id="w-e-element-408">
     Continue 是 Visual Studio Code 和 JetBrains 中领先的开源 AI 代码助手。
    </p>
    <p>
     •在侧边栏中进行聊天以理解和迭代代码。
    </p>
    <p>
     •自动补全，以便在输入时接收内联代码建议。
    </p>
    <p>
     •编辑功能可让你无需离开当前文件即可修改代码。
    </p>
    <p>
     •操作功能用于为常见用例建立快捷方式。
    </p>
    <h5 id="w-e-element-418">
     2）安装 VSCode
    </h5>
    <p id="w-e-element-420">
     首先，确保你已经安装了 VS Code 编辑器。如果尚未安装，可以从 VSCode 官网（
     <span contenteditable="false" tabindex="-1">
      <a href="https://code.visualstudio.com/download" rel="nofollow noopener noreferrer" target="_blank" title="Download Visual Studio Code - Mac, Linux, Windows">
       Download Visual Studio Code - Mac, Linux, Windows
      </a>
     </span>
     ）下载并安装。
    </p>
    <h5 id="w-e-element-425">
     3）安装conitinue插件
    </h5>
    <p id="w-e-element-427">
     a.打开 VSCode 编辑器。
    </p>
    <p id="w-e-element-429">
     b.在 VSCode 的插件中心，搜索 “Continue”。
    </p>
    <p id="w-e-element-431">
     c.找到 “Continue” 插件后，点击 “安装” 按钮进行安装。安装完成后，在 VSCode 左侧侧栏中会增加一个对应的图标，这就是 Continue 的主界面。
    </p>
    <p id="w-e-element-18">
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="1164" src="https://i-blog.csdnimg.cn/img_convert/986238bc12ef5d28a3e6adce98cfe80f.png" width="2794"/>
    </p>
    <p>
    </p>
    <p id="w-e-element-437">
     安装完成之后，重启下continue，vscode上就会出现continue图标，点击“continue”出现主界面
    </p>
    <h5 id="w-e-element-439">
     4）本地集成VScode配置
    </h5>
    <p id="w-e-element-441">
     点击continue输入框的左下角，点击“+ Add Chat model”，然后进行配置
    </p>
    <p id="w-e-element-19">
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="768" src="https://i-blog.csdnimg.cn/img_convert/94fbf98aa4a2417c189dcb5d42830447.png" width="1024"/>
    </p>
    <p>
    </p>
    <p id="w-e-element-447">
     配置Provider为“Ollama”，Modal选择“Autodetect”，点击“Connect”
    </p>
    <p id="w-e-element-20">
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="768" src="https://i-blog.csdnimg.cn/img_convert/8ffacf4323b1fa93871bed1664ae0dd9.png" width="1024"/>
    </p>
    <p>
    </p>
    <p id="w-e-element-453">
     到这里就配置完成了，让我们测试下成果吧，输入问题“你是什么模型？”
    </p>
    <p id="w-e-element-21">
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="768" src="https://i-blog.csdnimg.cn/img_convert/44b15eeca1143588d3b5d4b38b99e0d3.png" width="1024"/>
    </p>
    <p>
    </p>
    <p id="w-e-element-459">
     通过以上步骤，你就完成了 Continue 在 VSCode 中的本地集成，能够借助其强大的功能提升编程效率，享受更加智能、便捷的开发体验。
    </p>
    <h3 id="w-e-element-461">
     二、总结
    </h3>
    <p id="w-e-element-463">
     成功把 Ollama、Deepseek - R1 和 Continue 集成到 VScode 之后，确实避免了很多问题！先说说数据安全，以前总在网络搜索，心里总有点不踏实，现在都在本地搞定，再也不用担心信息泄露了，做涉及敏感信息的项目，安全感满满，对咱开发者来说，这可太香了。
    </p>
    <p id="w-e-element-465">
     不过，这集成也不是十全十美的。对电脑硬件要求有点高，如果电脑配置不太给力，模型响应速度就会变慢，有时候等得还挺着急，多少还是会影响点效率。但总体来说，这次集成真的让我学到了不少，绝对是一次超值得尝试的技术实践！强烈推荐大家也试试！
    </p>
