    <h3>
     一、C 语言的前世今生
    </h3>
    <p>
     C 语言诞生于 20 世纪 70 年代，由贝尔实验室的丹尼斯・里奇（Dennis Ritchie）开发。它最初是为了配合 UNIX 操作系统的开发，旨在提供一种高效、灵活且可移植的编程语言。在那个硬件资源有限的年代，C 语言凭借其对硬件的直接操作能力和简洁的语法，迅速成为了系统编程的首选语言。
    </p>
    <p>
     从发展历程来看，C 语言的出现并非一蹴而就。它的前身是 B 语言，而 B 语言又是基于 BCPL 语言发展而来。丹尼斯・里奇在 B 语言的基础上，对语法进行了改进，引入了数据类型等重要概念，从而诞生了 C 语言。随着 UNIX 操作系统的广泛传播，C 语言也逐渐被更多人所熟知和使用。1989 年，美国国家标准协会（ANSI）制定了 C 语言的第一个标准 ——ANSI C，这为 C 语言的规范化和广泛应用奠定了坚实的基础。
    </p>
    <h3>
     二、C 语言转义字符详解
    </h3>
    <p>
     在 C 语言中，转义字符是一种特殊的字符常量，用于表示一些无法直接通过键盘输入的字符，或者具有特殊含义的字符。转义字符以反斜杠 “\” 开头，后面紧跟一个或多个字符，通过这种组合来实现特定的功能。
    </p>
    <h4>
     常见转义字符类型及作用
    </h4>
    <ol>
     <li>
      <strong>
       换行符（
      </strong>
      <strong>
       \n
      </strong>
      <strong>
       ）
      </strong>
      ：在输出时，使光标移动到下一行的开头，常用于格式化输出。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("第一行\n第二行");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     上述代码执行后，会在控制台分两行输出 “第一行” 和 “第二行”。
    </p>
    <ol>
     <li>
      <strong>
       制表符（
      </strong>
      <strong>
       \t
      </strong>
      <strong>
       ）
      </strong>
      ：用于在输出时插入一个水平制表位，通常用于对齐文本。比如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("姓名\t年龄\t性别\n");
     </code>
    </p>
    <p>
     <code>
      printf("张三\t20\t男\n");
     </code>
    </p>
    <p>
     <code>
      printf("李四\t22\t女\n");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     运行结果会以表格形式呈现，使各列数据对齐。
    </p>
    <ol>
     <li>
      <strong>
       回车符（
      </strong>
      <strong>
       \r
      </strong>
      <strong>
       ）
      </strong>
      ：将光标移到当前行的开头，但不会换行。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("Hello\rWorld");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     输出结果为 “World”，因为 “\r” 将光标移到行首后，“World” 覆盖了 “Hello”。
    </p>
    <ol>
     <li>
      <strong>
       退格符（
      </strong>
      <strong>
       \b
      </strong>
      <strong>
       ）
      </strong>
      ：使光标向左移动一个字符位置，常用于删除前一个字符。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("Hello\b World");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     输出结果为 “Hell World”，“\b” 删除了 “o”。
    </p>
    <ol>
     <li>
      <strong>
       反斜杠（
      </strong>
      <strong>
       \\
      </strong>
      <strong>
       ）
      </strong>
      ：用于表示反斜杠字符本身，因为反斜杠在 C 语言中有特殊用途，需要用双反斜杠来表示一个普通的反斜杠。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("C:\\Program Files");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     会输出 “C:\Program Files”。
    </p>
    <ol>
     <li>
      <strong>
       单引号（
      </strong>
      <strong>
       \'
      </strong>
      <strong>
       ）
      </strong>
      ：用于表示单引号字符，在字符常量或字符串中，单引号有界定作用，使用\'可输出单引号。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("He said, \'Hello!\'");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     输出 “He said, 'Hello!'”。
    </p>
    <ol>
     <li>
      <strong>
       双引号（
      </strong>
      <strong>
       \"
      </strong>
      <strong>
       ）
      </strong>
      ：用于表示双引号字符，在字符串中，双引号界定字符串，使用\"可在字符串中输出双引号。例如：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("She said, \"Goodbye!\"");
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     输出 “She said, "Goodbye!"”。
    </p>
    <ol>
     <li>
      <strong>
       八进制转义字符（
      </strong>
      <strong>
       \ooo
      </strong>
      <strong>
       ）
      </strong>
      ：\后面跟 1 到 3 位八进制数字，代表一个字符的八进制 ASCII 码值。例如，\101表示字符A，因为101（八进制）对应的十进制是 65，而 65 是字符A的 ASCII 码值。示例代码：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("%c", '\101');
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     输出字符A。
    </p>
    <ol>
     <li>
      <strong>
       十六进制转义字符（
      </strong>
      <strong>
       \xhh
      </strong>
      <strong>
       ）
      </strong>
      ：\x后面跟 1 到 2 位十六进制数字，代表一个字符的十六进制 ASCII 码值。如\x41也表示字符A，因为41（十六进制）对应的十进制是 65。示例代码：
     </li>
    </ol>
    <pre></pre>
    <p>
     <code>
      #include &lt;stdio.h&gt;
     </code>
    </p>
    <p>
     <code>
      int main() {
      <!-- -->
     </code>
    </p>
    <p>
     <code>
      printf("%c", '\x41');
     </code>
    </p>
    <p>
     <code>
      return 0;
     </code>
    </p>
    <p>
     <code>
      }
     </code>
    </p>
    <p>
     同样输出字符A。
    </p>
    <p>
     转义字符在 C 语言中是非常重要的组成部分，熟练掌握它们的使用，能够帮助开发者更好地控制输出格式、处理特殊字符，使程序更加灵活和强大。
    </p>
