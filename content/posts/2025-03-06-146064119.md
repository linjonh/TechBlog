    <h1 class="title-article" id="articleContentId">
     SQL分几种
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     SQL（Structured Query Language）是用于管理关系型数据库的标准语言。根据功能，SQL 语句可以分为以下几类：
    </p>
    <hr/>
    <h4>
     <strong>
      1. 数据查询语言（DQL，Data Query Language）
     </strong>
    </h4>
    <p>
     用于从数据库中查询数据。
     <br/>
     <strong>
      核心语句
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       SELECT
      </code>
      ：查询数据。
      <br/>
      <strong>
       示例
      </strong>
      ：
     </li>
    </ul>
    <pre id="code-fqbscswfy"><code><code>SELECT name, age FROM users WHERE age &gt; 18;
</code></code></pre>
    <hr/>
    <h4>
     <strong>
      2. 数据操作语言（DML，Data Manipulation Language）
     </strong>
    </h4>
    <p>
     用于对数据库中的数据进行增、删、改操作。
     <br/>
     <strong>
      核心语句
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       INSERT
      </code>
      ：插入数据。
     </li>
     <li>
      <code>
       UPDATE
      </code>
      ：更新数据。
     </li>
     <li>
      <code>
       DELETE
      </code>
      ：删除数据。
      <br/>
      <strong>
       示例
      </strong>
      ：
     </li>
    </ul>
    <pre id="code-ltncc4dsy"><code><code>INSERT INTO users (name, age) VALUES ('Alice', 25);
UPDATE users SET age = 26 WHERE name = 'Alice';
DELETE FROM users WHERE name = 'Alice';
</code></code></pre>
    <hr/>
    <h4>
     <strong>
      3. 数据定义语言（DDL，Data Definition Language）
     </strong>
    </h4>
    <p>
     用于定义或修改数据库结构（如表、索引、视图等）。
     <br/>
     <strong>
      核心语句
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       CREATE
      </code>
      ：创建数据库对象（如表、索引、视图等）。
     </li>
     <li>
      <code>
       ALTER
      </code>
      ：修改数据库对象。
     </li>
     <li>
      <code>
       DROP
      </code>
      ：删除数据库对象。
     </li>
     <li>
      <code>
       TRUNCATE
      </code>
      ：清空表数据。
      <br/>
      <strong>
       示例
      </strong>
      ：
     </li>
    </ul>
    <p>
     <code>
      sql
     </code>
    </p>
    <pre id="code-jh0zt6wok"><code><code>CREATE TABLE users (id INT PRIMARY KEY, name VARCHAR(50));
ALTER TABLE users ADD COLUMN age INT;
DROP TABLE users;
TRUNCATE TABLE users;
</code></code></pre>
    <hr/>
    <h4>
     <strong>
      4. 数据控制语言（DCL，Data Control Language）
     </strong>
    </h4>
    <p>
     用于控制数据库访问权限。
     <br/>
     <strong>
      核心语句
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       GRANT
      </code>
      ：授予权限。
     </li>
     <li>
      <code>
       REVOKE
      </code>
      ：撤销权限。
      <br/>
      <strong>
       示例
      </strong>
      ：
     </li>
    </ul>
    <p>
    </p>
    <pre id="code-ej73nx969"><code><code>GRANT SELECT ON users TO 'user1';
REVOKE SELECT ON users FROM 'user1';
</code></code></pre>
    <hr/>
    <h4>
     <strong>
      5. 事务控制语言（TCL，Transaction Control Language）
     </strong>
    </h4>
    <p>
     用于管理数据库事务。
     <br/>
     <strong>
      核心语句
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       BEGIN
      </code>
      或
      <code>
       START TRANSACTION
      </code>
      ：开始事务。
     </li>
     <li>
      <code>
       COMMIT
      </code>
      ：提交事务。
     </li>
     <li>
      <code>
       ROLLBACK
      </code>
      ：回滚事务。
     </li>
     <li>
      <code>
       SAVEPOINT
      </code>
      ：设置保存点。
      <br/>
      <strong>
       示例
      </strong>
      ：
     </li>
    </ul>
    <pre><code>
          </code></pre>
    <pre id="code-s2xr5dpte"><code><code>START TRANSACTION;
UPDATE users SET age = 30 WHERE name = 'Alice';
COMMIT;
</code></code></pre>
    <hr/>
    <h4>
     <strong>
      6. 其他 SQL 语句
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       游标操作
      </strong>
      ：如
      <code>
       DECLARE CURSOR
      </code>
      、
      <code>
       FETCH
      </code>
      等。
     </li>
     <li>
      <strong>
       存储过程和函数
      </strong>
      ：如
      <code>
       CREATE PROCEDURE
      </code>
      、
      <code>
       CREATE FUNCTION
      </code>
      等。
     </li>
     <li>
      <strong>
       触发器
      </strong>
      ：如
      <code>
       CREATE TRIGGER
      </code>
      。
     </li>
     <li>
      <strong>
       其他辅助语句
      </strong>
      ：如
      <code>
       USE
      </code>
      （选择数据库）、
      <code>
       SHOW
      </code>
      （显示数据库信息）等。
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      总结
     </strong>
    </h4>
    <p>
     SQL 语句主要分为以下六类：
    </p>
    <ol>
     <li>
      <strong>
       DQL
      </strong>
      ：用于查询数据（
      <code>
       SELECT
      </code>
      ）。
     </li>
     <li>
      <strong>
       DML
      </strong>
      ：用于操作数据（
      <code>
       INSERT
      </code>
      、
      <code>
       UPDATE
      </code>
      、
      <code>
       DELETE
      </code>
      ）。
     </li>
     <li>
      <strong>
       DDL
      </strong>
      ：用于定义或修改数据库结构（
      <code>
       CREATE
      </code>
      、
      <code>
       ALTER
      </code>
      、
      <code>
       DROP
      </code>
      ）。
     </li>
     <li>
      <strong>
       DCL
      </strong>
      ：用于控制权限（
      <code>
       GRANT
      </code>
      、
      <code>
       REVOKE
      </code>
      ）。
     </li>
     <li>
      <strong>
       TCL
      </strong>
      ：用于管理事务（
      <code>
       BEGIN
      </code>
      、
      <code>
       COMMIT
      </code>
      、
      <code>
       ROLLBACK
      </code>
      ）。
     </li>
     <li>
      <strong>
       其他
      </strong>
      ：如游标、存储过程、触发器等。
     </li>
    </ol>
    <p>
     掌握这些 SQL 语句类型，可以更好地管理和操作关系型数据库。
    </p>
