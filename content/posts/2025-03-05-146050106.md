    <h4>
     <a id="_1">
     </a>
     <strong>
      一、基础操作
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全选/取消全选
      </strong>
      ：
      <code>
       A
      </code>
      （全选）、
      <code>
       AA
      </code>
      （连续按两次A取消全选）
     </li>
     <li>
      <strong>
       复制物体
      </strong>
      ：
      <code>
       Shift + D
      </code>
      （复制后需点击确认位置）
     </li>
     <li>
      <strong>
       移动物体
      </strong>
      ：
      <code>
       G
      </code>
      （按X/Y/Z可约束轴向移动）
     </li>
     <li>
      <strong>
       旋转物体
      </strong>
      ：
      <code>
       R
      </code>
      （按X/Y/Z可约束轴向旋转）
     </li>
     <li>
      <strong>
       缩放物体
      </strong>
      ：
      <code>
       S
      </code>
      （按X/Y/Z可约束轴向缩放）
     </li>
     <li>
      <strong>
       撤销
      </strong>
      ：
      <code>
       Ctrl + Z
      </code>
     </li>
     <li>
      <strong>
       重做
      </strong>
      ：
      <code>
       Ctrl + Shift + Z
      </code>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_13">
     </a>
     <strong>
      二、视图与导航
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       切换三视图
      </strong>
      ：
      <code>
       Shift +
      </code>
      ``（FPS自由视角模式）
     </li>
     <li>
      <strong>
       前/后视图
      </strong>
      ：
      <code>
       Ctrl + 1
      </code>
      （小键盘）/
      <code>
       Ctrl + 3
      </code>
      （右视图）
     </li>
     <li>
      <strong>
       顶/底视图
      </strong>
      ：
      <code>
       Ctrl + 7
      </code>
      （顶视图）/
      <code>
       Ctrl + 9
      </code>
      （底视图）
     </li>
     <li>
      <strong>
       放大/缩小视图
      </strong>
      ：
      <code>
       Ctrl + +
      </code>
      （放大）、
      <code>
       Ctrl + -
      </code>
      （缩小）
     </li>
     <li>
      <strong>
       平移相机视角
      </strong>
      ：
      <code>
       Shift + 鼠标中键
      </code>
     </li>
     <li>
      <strong>
       旋转镜头视角
      </strong>
      ：
      <code>
       鼠标中键拖动
      </code>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_24">
     </a>
     <strong>
      三、选择与编辑
     </strong>
    </h4>
    <p>
     <strong>
      取消选择
     </strong>
     ：
     <code>
      Alt + A
     </code>
    </p>
    <p>
     <strong>
      随机选择面
     </strong>
     ：
     <code>
      Select Random
     </code>
     （需手动打开相关工具）
    </p>
    <p>
     <strong>
      合并顶点
     </strong>
     ：
     <code>
      Alt + M
     </code>
    </p>
    <p>
     <strong>
      连顶点成线/面
     </strong>
     ：
     <code>
      F
     </code>
     （两个顶点连线，四个顶点构成面）
    </p>
    <p>
     <strong>
      分离模型部分
     </strong>
     ：
     <code>
      P
     </code>
     （Edit Mode下使用）
    </p>
    <p>
     <strong>
      标记接缝
     </strong>
     ：
     <code>
      Ctrl + E → Mark Seam
     </code>
    </p>
    <hr/>
    <h4>
     <a id="_40">
     </a>
     <strong>
      四、建模工具
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       环切（插入循环边）
      </strong>
      ：
      <code>
       Ctrl + R
      </code>
     </li>
     <li>
      <strong>
       倒角
      </strong>
      ：
      <code>
       Ctrl + B
      </code>
      （拖拽调节倒角程度）
     </li>
     <li>
      <strong>
       挤出操作
      </strong>
      ：
      <code>
       E
      </code>
      （按区域挤压）
     </li>
     <li>
      <strong>
       斜角切割
      </strong>
      ：
      <code>
       Ctrl + Shift + B
      </code>
      （顶点模式）
     </li>
     <li>
      <strong>
       创建斜面
      </strong>
      ：
      <code>
       Ctrl + B（Bevel）
      </code>
     </li>
     <li>
      <strong>
       光滑模型
      </strong>
      ：
      <code>
       W → Shade Smooth
      </code>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_51">
     </a>
     <strong>
      五、材质与雕刻
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       调整笔刷强度
      </strong>
      ：
      <code>
       Shift + F
      </code>
      （雕刻模式下）
     </li>
     <li>
      <strong>
       切换X-Ray模式
      </strong>
      ：
      <code>
       Alt + Z
      </code>
      （方便选中隐藏点）
     </li>
     <li>
      <strong>
       添加材质链接
      </strong>
      ：
      <code>
       Ctrl + L
      </code>
      （Make Links to Material）
     </li>
     <li>
      <strong>
       节点编辑器缩放
      </strong>
      ：
      <code>
       S
      </code>
      （蓝图节点缩放）
     </li>
     <li>
      <strong>
       合并材质节点组
      </strong>
      ：
      <code>
       Ctrl + G
      </code>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_61">
     </a>
     <strong>
      六、渲染与动画
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       插入关键帧
      </strong>
      ：
      <code>
       I
      </code>
      （可为位置、旋转属性插入动画帧）
     </li>
     <li>
      <strong>
       渲染队列
      </strong>
      ：
      <code>
       Ctrl + Shift + /
      </code>
     </li>
     <li>
      <strong>
       切换粒子系统视图
      </strong>
      ：
      <code>
       Alt + H
      </code>
      （显示隐藏粒子）
     </li>
     <li>
      <strong>
       应用模型缩放
      </strong>
      ：
      <code>
       Ctrl + A → Apply Scale
      </code>
      （归一化模型尺寸）
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_70">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       操作系统差异
      </strong>
      ：macOS需用
      <code>
       Cmd
      </code>
      替代
      <code>
       Ctrl
      </code>
      ，部分快捷键可能冲突。
     </li>
     <li>
      <strong>
       版本差异
      </strong>
      ：部分快捷键在Blender 2.8及以上版本中功能可能调整（如视图切换逻辑）。
     </li>
    </ul>
