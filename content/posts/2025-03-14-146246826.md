    <p>
     RadialGradientExponential 径向梯度指数
    </p>
    <p>
     常用于制作
     <strong>
      从圆心到四周产生渐变的效果
     </strong>
    </p>
    <p>
     <img alt="" height="391" src="https://i-blog.csdnimg.cn/direct/3828133f04864dd0a434449774826e79.png" width="388">
      <img alt="" height="254" src="https://i-blog.csdnimg.cn/direct/41ad587d6de6499aaa36b33008a98081.png" width="432">
       <br/>
       RadialGradientExponential有5个参数：
      </img>
     </img>
    </p>
    <ul>
     <li>
      <p style="margin-left:0px; margin-right:0px">
       <strong>
        UV
       </strong>
       <br/>
       纹理坐标（默认直接采样值）
      </p>
     </li>
     <li>
      <p style="margin-left:0px; margin-right:0px">
       <strong>
        CenterPosition  中心位置
       </strong>
       <br/>
       渐变圆心坐标（默认画布中心
       <code>
        (0.5,0.5)
       </code>
       ）
      </p>
     </li>
     <li>
      <p style="margin-left:0px; margin-right:0px">
       <strong>
        Radius  半径
       </strong>
       <br/>
       渐变半径比例（取值范围
       <code>
        (0,1]
       </code>
       ，1=画布边长）
      </p>
     </li>
     <li>
      <p style="margin-left:0px; margin-right:0px">
       <strong>
        Density  密度
       </strong>
       <br/>
       渐变锐度系数（值越大边缘过渡越尖锐）
      </p>
     </li>
     <li>
      <p style="margin-left:0px; margin-right:0px">
       <strong>
        InvertDensity  反转密度
       </strong>
       <br/>
       亮度反转开关（
       <code>
        true
       </code>
       时中心暗边缘亮）
      </p>
     </li>
    </ul>
