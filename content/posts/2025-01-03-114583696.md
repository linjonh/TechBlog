    <h2>
     <a id="_2">
     </a>
     清除前
    </h2>
    <p>
     查看A_Controller.java的Local History，可见历史记录，如下所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2850a03c7d22a966508479887977be79.png#pic_center">
      <br/>
      上面的截图说明了该文件有1个历史记录。
     </img>
    </p>
    <h2>
     <a id="_8">
     </a>
     开始清除
    </h2>
    <p>
     点击顶部的Help &gt; FindAction，在Action栏中输入Registry，可见以下搜索结果：
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7c9ec4ceb4d404b1ce8b07b11d1dee9a.png#pic_center">
      <br/>
      双击搜索结果，进入以下页面。寻找
      <mark>
       localHistory.daysToKeep
      </mark>
      参数，将5
      <mark>
       改成0
      </mark>
      即可。
      <br/>
      然后点击对话窗的“close”，重启IDEA。
      <br/>
      <strong>
       意义
      </strong>
      ：
      <br/>
      5是默认值，默认保留最近5天的历史记录。
      <br/>
      改成0，意味着禁用本地历史记录，对应的，历史记录也随之删除。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/73b49da8e275c0e6e23408f764a49c59.png#pic_center"/>
     </img>
    </p>
    <h2>
     <a id="_16">
     </a>
     清除后
    </h2>
    <p>
     IDEA重启后，再看A_Controller.java的Local History。发现已没有任何记录，如下所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b325f3fefa8d4511cf2e71225a87932e.png#pic_center"/>
    </p>
    <h2>
     <a id="_19">
     </a>
     补充
    </h2>
    <p>
     如果清除了本地记录后，又想使用Local History功能，那就重新将
     <mark>
      localHistory.daysToKeep
     </mark>
     参数设置成5或者其他值即可。
     <br/>
     本方法来自IDEA官方文档：
     <a href="https://www.jetbrains.com/help/idea/2019.3/local-history.html#location" rel="nofollow">
      https://www.jetbrains.com/help/idea/2019.3/local-history.html#location
     </a>
     .
     <br/>
     本人IDEA版本 2019.3.4
    </p>
