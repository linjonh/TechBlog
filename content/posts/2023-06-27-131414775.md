    <h3>
     <a id="python__15">
     </a>
     一.python 安装
    </h3>
    <p>
     在 centos 服务器上安装 python3.10
    </p>
    <h4>
     <a id="1_19">
     </a>
     1.安装依赖
    </h4>
    <pre><code class="prism language-apl">#安装依赖库
yum install wget zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make zlib zlib-devel libffi-devel -y
</code></pre>
    <h4>
     <a id="2_python_26">
     </a>
     2.下载 python
    </h4>
    <pre><code class="prism language-apl">#下载python
wget https://www.python.org/ftp/python/3.10.0/Python-3.10.0.tgz
</code></pre>
    <p>
     <code>
      网页下载
     </code>
    </p>
    <p>
     https://www.python.org/downloads
    </p>
    <h4>
     <a id="3_37">
     </a>
     3.解压
    </h4>
    <pre><code class="prism language-apl">#解压
tar -zxvf Python-3.10.0.tgz
</code></pre>
    <h4>
     <a id="4_44">
     </a>
     4.进入目录
    </h4>
    <pre><code class="prism language-apl">#进入目录
cd Python-3.10.0
</code></pre>
    <h4>
     <a id="5_51">
     </a>
     5.校验环境
    </h4>
    <pre><code class="prism language-apl">#校验
./configure --enable-optimizations
</code></pre>
    <h4>
     <a id="6_58">
     </a>
     6.编译
    </h4>
    <pre><code class="prism language-apl">#编译
make -j 8
</code></pre>
    <h4>
     <a id="7_65">
     </a>
     7.安装
    </h4>
    <pre><code class="prism language-apl">#安装
sudo make altinstall
</code></pre>
    <h4>
     <a id="8_72">
     </a>
     8.验证
    </h4>
    <pre><code class="prism language-apl">#验证
python3.10 --version
</code></pre>
    <h3>
     <a id="_79">
     </a>
     二.老版本处理
    </h3>
    <h4>
     <a id="1_81">
     </a>
     1.解压
    </h4>
    <pre><code class="prism language-apl"># 删除系统自带的老版本(python2)的软链接
rm -f /usr/bin/python
</code></pre>
    <h4>
     <a id="2_88">
     </a>
     2.创建软连接
    </h4>
    <pre><code class="prism language-apl"># 创建软链接
ln -s /usr/local/python3.10.4/bin/python3.10 /usr/bin/python
</code></pre>
    <h4>
     <a id="3_python2__95">
     </a>
     3.系统只认 python2 问题
    </h4>
    <p>
     <code>
      修改2个文件
     </code>
    </p>
    <pre><code class="prism language-apl">/usr/bin/yum
/usr/libexec/urlgrabber-ext-down
</code></pre>
    <p>
     <code>
      修改方式
     </code>
    </p>
    <pre><code>将
#!/usr/bin/python
改为
#!/usr/bin/python2
</code></pre>
    <h4>
     <a id="4conda__113">
     </a>
     4.conda 虚拟环境
    </h4>
    <pre><code class="prism language-apl">#创建环境
conda create -n chatbot python=3.10

#激活环境
conda activate chatbot

#查看环境
conda info --envs
</code></pre>
    <h3>
     <a id="_126">
     </a>
     三.依赖安装
    </h3>
    <h4>
     <a id="1_128">
     </a>
     1.查看已安装
    </h4>
    <pre><code class="prism language-apl">#查看所有已安装
pip3 list
</code></pre>
    <h4>
     <a id="2_135">
     </a>
     2.安装包
    </h4>
    <pre><code class="prism language-apl">#安装pip
python3.6  get-pip.py
</code></pre>
    <h4>
     <a id="3_142">
     </a>
     3.指定版本
    </h4>
    <pre><code class="prism language-apl"># 指定版本安装
pip install langchain==0.0.211
</code></pre>
    <h4>
     <a id="4_149">
     </a>
     4.更新依赖
    </h4>
    <pre><code class="prism language-apl">#更新版本
pip install langchain --upgrade -i  https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/
</code></pre>
    <h4>
     <a id="5_156">
     </a>
     5.卸载包
    </h4>
    <pre><code class="prism language-apl">#卸载包
pip uninstall 第三方包名
</code></pre>
    <h4>
     <a id="6_163">
     </a>
     6.文件安装依赖
    </h4>
    <pre><code class="prism language-python"><span class="token comment"># 使用脚本安装依赖</span>
pip install <span class="token operator">-</span>r ci<span class="token operator">-</span>requirements<span class="token punctuation">.</span>txt
</code></pre>
    <p>
     ci-requirements.txt
    </p>
    <pre><code class="prism language-apl">langchain==0.0.211
responses==0.23.1
aioresponses==0.7.4
matplotlib==3.7.1
uvicorn==0.22.0
python-dotenv==1.0.0
python-graphql-client==0.4.3
tomli==2.0.1
python-socketio==5.8.0
fastapi==0.95.1
fastapi-socketio==0.0.10
watchfiles==0.19.0
asyncer==0.0.2
uptrace==1.18.0
openai==0.27.4
aiofiles==23.1.0
</code></pre>
