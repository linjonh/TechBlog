### 一.python 安装

在 centos 服务器上安装 python3.10

#### 1.安装依赖

```apl
#安装依赖库
yum install wget zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make zlib zlib-devel libffi-devel -y

```

#### 2.下载 python

```apl
#下载python
wget https://www.python.org/ftp/python/3.10.0/Python-3.10.0.tgz

```

`网页下载`

https://www.python.org/downloads

#### 3.解压

```apl
#解压
tar -zxvf Python-3.10.0.tgz

```

#### 4.进入目录

```apl
#进入目录
cd Python-3.10.0

```

#### 5.校验环境

```apl
#校验
./configure --enable-optimizations

```

#### 6.编译

```apl
#编译
make -j 8

```

#### 7.安装

```apl
#安装
sudo make altinstall

```

#### 8.验证

```apl
#验证
python3.10 --version

```

### 二.老版本处理

#### 1.解压

```apl
# 删除系统自带的老版本(python2)的软链接
rm -f /usr/bin/python

```

#### 2.创建软连接

```apl
# 创建软链接
ln -s /usr/local/python3.10.4/bin/python3.10 /usr/bin/python

```

#### 3.系统只认 python2 问题

`修改2个文件`

```apl
/usr/bin/yum
/usr/libexec/urlgrabber-ext-down

```

`修改方式`

```
将
#!/usr/bin/python
改为
#!/usr/bin/python2

```

#### 4.conda 虚拟环境

```apl
#创建环境
conda create -n chatbot python=3.10

#激活环境
conda activate chatbot

#查看环境
conda info --envs

```

### 三.依赖安装

#### 1.查看已安装

```apl
#查看所有已安装
pip3 list

```

#### 2.安装包

```apl
#安装pip
python3.6  get-pip.py

```

#### 3.指定版本

```apl
# 指定版本安装
pip install langchain==0.0.211

```

#### 4.更新依赖

```apl
#更新版本
pip install langchain --upgrade -i  https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/

```

#### 5.卸载包

```apl
#卸载包
pip uninstall 第三方包名

```

#### 6.文件安装依赖

```python
# 使用脚本安装依赖
```
