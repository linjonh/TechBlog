    <p>
     对于浮点数，舍⼊到最接近的整数是什么意思呢？
     <br/>
     ⽐如:
     <br/>
     4.3舍⼊成4
     <br/>
     4.65舍⼊成5
     <br/>
     这种情况可以使⽤+0.5后强制类型转换成int类型的值处理。
     <br/>
     float f = 4.65;
     <br/>
     int n = (int)(f+0.5);
    </p>
