# 微信小程序滚动监听

微信小程序是没有真实DOM节点的，也就是说没有window对象，怎么实现类似web网页的滚动监听事件呢？
  
在页面Page()函数里，有个onPageScroll方法，可以输出滚动距离：

```
onPageScroll:function(e){
    console.log(e);//{scrollTop:99}
  }
```

开发者可以通过监听scrollTop的变化来操作页面显示。
  
现有bug：有时响应滚动监听存在延迟。
