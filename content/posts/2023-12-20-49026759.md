     <span style="line-height:18px">
      使用MPEG4IP库接口实现音视频存储
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      MP4WriteSample   写音视频帧（需要精确控制时间戳，时间戳可以使用相对值，即当前帧时间戳减去上一帧的时间戳）
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      MP4AddH264VideoTrack    添加H264 track（timescale时间因子参数很重要，根据视频帧的时间戳类型定义，90000（视频标准采样时间戳）或1000（ms时间戳））
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      MP4AddAudioTrack   添加音频AAC track (timesacle时间因子参数设置为1000（ms时间戳）)
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      音频AAC
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      采样频率是指将模拟声音波形进行数字化时，每秒钟抽取声波幅度样本的次数。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      。正常人听觉的频率范围大约在20Hz~20kHz之间，根据奈奎斯特采样理论，为了保证声音不失真，采样频率应该在40kHz左右。常用的音频采样频率有8kHz、11.025kHz、22.05kHz、16kHz、37.8kHz、44.1kHz、48kHz等，如果采用更高的采样频率，还可以达到DVD的音质
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      对采样率为44.1kHz的AAC音频进行解码时，一帧的解码时间须控制在23.22毫秒内。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      背景知识:
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      <span style="word-wrap:normal; word-break:normal; line-height:19px; font-family:Verdana,Geneva,Arial,Helvetica,sans-serif; background-color:rgb(249,252,233)">
       (一个AAC原始帧包含一段时间内1024个采样及相关数据)
      </span>
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      分析：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      1 AAC
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      音频帧的播放时间=一个AAC帧对应的采样样本的个数/采样频率(单位为s)
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      一帧 1024个 sample。采样率 Samplerate 44100KHz，每秒44100个sample, 所以 根据公式
      <wbr/>
      <wbr/>
      音频帧的播放时间=一个AAC帧对应的采样样本的个数/采样频率
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      当前AAC一帧的播放时间是= 1024*1000000/44100= 22.32ms(单位为ms)
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      2 MP3
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <wbr/>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      mp3 每帧均为1152个字节， 则：
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      frame_duration = 1152 * 1000000 / sample_rate
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      例如：sample_rate = 44100HZ时， 计算出的时长为26.122ms，这就是经常听到的mp3每帧播放时间固定为26ms的由来。
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; line-height:26px; color:rgb(70,70,70); font-size:14px; font-family:Arial">
     <span style="word-wrap:normal; word-break:normal; line-height:18px; font-size:12px">
      <br/>
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      视频H264
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
     <span style="line-height:18px">
      视频帧时间戳，分两种类型标准采样时间戳（毫秒时间戳*90）和毫秒时间戳（按帧率计算，1000/帧率=播放持续时间）
     </span>
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; border:0px; list-style:none; word-wrap:normal; word-break:normal; color:rgb(70,70,70); font-family:Arial">
