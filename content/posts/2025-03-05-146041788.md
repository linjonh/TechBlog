     一、串的基本概念
    </h2>
    <h3 id="1.%E4%B8%B2%E7%9A%84%E6%A6%82%E5%BF%B5" name="1.%E4%B8%B2%E7%9A%84%E6%A6%82%E5%BF%B5">
     1.串的概念
    </h3>
    <p>
     串，又叫字符串，是由一系列字符组成的数据结构，用于表示文本或字符序列。在编程中，字串，符串通常用双引号括起来，如"Hello,World！"
    </p>
    <p>
     <img alt="" height="63" src="https://i-blog.csdnimg.cn/direct/c2175be50d6f4946a0820fb3f4b83baf.png" width="860"/>
    </p>
    <p>
     而实际在内存中存储的时候，字符串最后还会有一个 '\0' 的字符（'\0'的ASCII的值为 0），它代表了这个串的结尾
    </p>
    <hr/>
    <h3 id="2.%E8%8E%B7%E5%8F%96%E4%B8%B2%E7%9A%84%E9%95%BF%E5%BA%A6" name="2.%E8%8E%B7%E5%8F%96%E4%B8%B2%E7%9A%84%E9%95%BF%E5%BA%A6">
     2.获取串的长度
    </h3>
    <p>
     串的长度是指字符串中包含的字符个数。每个语言都有内置方法来获取字符串的长度(
     <strong>
      注意：
     </strong>
     字符串长度是不包含最后一一个 ' \0 ' 的)
    </p>
    <hr/>
    <h3 id="3.%E4%B8%B2%E7%9A%84%E6%8B%B7%E8%B4%9D" name="3.%E4%B8%B2%E7%9A%84%E6%8B%B7%E8%B4%9D">
     3.串的拷贝
    </h3>
    <p>
     有时需要创建一个字符串的副本，而不修改原始字符串。一般可通过
     <span style="color:#fe2c24">
      遍历串
     </span>
     的方式实现串的拷贝。
    </p>
    <p>
     <img alt="" height="150" src="https://i-blog.csdnimg.cn/direct/5bf8cf5e4a6448b39d5bf653990a111d.png" width="865"/>
    </p>
    <hr/>
    <h3 id="4.%E4%B8%B2%E7%9A%84%E6%AF%94%E8%BE%83" name="4.%E4%B8%B2%E7%9A%84%E6%AF%94%E8%BE%83">
     4.串的比较
    </h3>
    <p>
     串的比较是指比较两个串是否相等，一般可以采用遍历的方式逐个比较，当然一些语言提供了内置比较函数
    </p>
    <hr/>
    <h3 id="5.%E4%B8%B2%E7%9A%84%E6%8B%BC%E6%8E%A5" name="5.%E4%B8%B2%E7%9A%84%E6%8B%BC%E6%8E%A5">
     5.串的拼接
    </h3>
    <p>
     将两个串拼接在一起，一般是申请一块新的内存，新的内存的大小是两个需要拼接的字符串的长度之和加一，然后第一个串的字符挨个赋值到新的内存，再将第二个串的字符挨个赋值到新的内存，最后加上一个 ‘ \0 ' ，就实现完毕了。
    </p>
    <p>
     <img alt="" height="207" src="https://i-blog.csdnimg.cn/direct/42570a761b5f45469ac4798a2a7a8a14.png" width="859"/>
    </p>
    <hr/>
    <h3 id="6.%E4%B8%B2%E7%9A%84%E7%B4%A2%E5%BC%95" name="6.%E4%B8%B2%E7%9A%84%E7%B4%A2%E5%BC%95">
     6.串的索引
    </h3>
    <p>
     可以使用索引来访问字符串中的特定字符。索引从 0 开始，表示第一个字符的位置，依次递增，可以使用 [] 并传入索引值来获取字符
    </p>
    <p>
     <img alt="" height="128" src="https://i-blog.csdnimg.cn/direct/7bc947dfaa024f14a6aa33329813b2d1.png" width="848"/>
    </p>

    <h2 id="%E4%B8%89%E3%80%81%E5%AE%9E%E6%88%98" name="%E4%B8%89%E3%80%81%E5%AE%9E%E6%88%98">
     三、实战

    <h3 id="1.344.%20%E5%8F%8D%E8%BD%AC%E5%AD%97%E7%AC%A6%E4%B8%B2" name="1.344.%20%E5%8F%8D%E8%BD%AC%E5%AD%97%E7%AC%A6%E4%B8%B2">
     1.
     <a href="https://leetcode.cn/problems/reverse-string/" rel="nofollow" title="344. 反转字符串">
      344. 反转字符串
     </a>
    </h3>
    <blockquote>
     <p>
      编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组
      <code>
       s
      </code>
      的形式给出。
     </p>
     <p>
      不要给另外的数组分配额外的空间，你必须
      <strong>
       <a href="https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95" rel="nofollow" title="原地">
        原地
       </a>
       修改输入数组
      </strong>
      、使用 O(1) 的额外空间解决这一问题。
     </p>
     <p>
      <strong>
       示例 1：
      </strong>
     </p>
     <pre><strong>输入：</strong>s = ["h","e","l","l","o"]
<strong>输出：</strong>["o","l","l","e","h"]
</pre>
     <p>
      <strong>
       示例 2：
      </strong>
     </p>
     <pre><strong>输入：</strong>s = ["H","a","n","n","a","h"]
<strong>输出：</strong>["h","a","n","n","a","H"]
</pre>
     <p>
      <strong>
       提示：
      </strong>
     </p>
     <ul>
      <li>
       <code>
        1 &lt;= s.length &lt;= 105
       </code>
      </li>
      <li>
       <code>
        s[i]
       </code>
       都是
       <a href="https://baike.baidu.com/item/ASCII" rel="nofollow" title="ASCII">
        ASCII
       </a>
       码表中的可打印字符
      </li>
     </ul>
    </blockquote>
    <h4 id="%E6%96%B9%E6%B3%95%E4%B8%80%E3%80%81%E5%BA%93%E5%87%BD%E6%95%B0" name="%E6%96%B9%E6%B3%95%E4%B8%80%E3%80%81%E5%BA%93%E5%87%BD%E6%95%B0">
     方法一、库函数
    </h4>
    <h5 id="%E6%80%9D%E8%B7%AF%E4%B8%8E%E7%AE%97%E6%B3%95" name="%E6%80%9D%E8%B7%AF%E4%B8%8E%E7%AE%97%E6%B3%95">
     <span style="color:#38d8f0">
      思路与算法
     </span>
    </h5>
    <p>
     ​
     <strong>
      原地反转
     </strong>
     ​：
     <code>
      list.reverse()
     </code>
     是Python列表的内置方法，能够直接修改原列表，将元素顺序完全倒置。例如：列表
     <code>
      ['h','e','l','l','o']
     </code>
     调用
     <code>
      reverse()
     </code>
     后变为
     <code>
      ['o','l','l','e','h']
     </code>
     ，且不返回新列表，完全符合题目“原地修改”的要求。​
    </p>
    <p>
     ​​
     <strong>
      时间复杂度
     </strong>
     ：
     <code>
      O(n)
     </code>
     ，需要遍历列表一半长度的元素进行对称交换。
    </p>
    <p>
     <strong>
      空间复杂度
     </strong>
     ：
     <code>
      O(1)
     </code>
     ，仅通过指针操作交换元素，不占用额外空间。
    </p>
    <p>
     <span style="color:#fe2c24">
      <strong>
       列表.reverse()：
      </strong>
     </span>
     Python 中用于反转列表元素顺序的内置方法。它直接在原列表上进行操作，不会返回新的列表，而是直接修改原列表的顺序。
    </p>
    <pre><code class="language-python">class Solution:
    def reverseString(self, s: List[str]) -&gt; None:
        """
        Do not return anything, modify s in-place instead.
        """
        s.reverse()
        </code></pre>

    </p>

    <h4 id="%E6%96%B9%E6%B3%95%E4%BA%8C%E3%80%81%E5%8F%8C%E6%8C%87%E9%92%88" name="%E6%96%B9%E6%B3%95%E4%BA%8C%E3%80%81%E5%8F%8C%E6%8C%87%E9%92%88">
     方法二、双指针
    </h4>
    <h5 id="%E6%80%9D%E8%B7%AF%E4%B8%8E%E7%AE%97%E6%B3%95" name="%E6%80%9D%E8%B7%AF%E4%B8%8E%E7%AE%97%E6%B3%95">
     <span style="color:#38d8f0">
      思路与算法
     </span>
    </h5>
    <p>
     <strong>
      双指针策略
     </strong>
     ：设置两个指针
     <code>
      l
     </code>
     和
     <code>
      r
     </code>
     ，分别指向字符串列表的头部（初始为0）和尾部（初始为
     <code>
      len(s)-1
     </code>
     ）​
    </p>
    <p>
     <strong>
      对称交换
     </strong>
     <span style="color:#4d4d4d">
      ：每次循环交换
     </span>
     <code>
      l
     </code>
     <span style="color:#4d4d4d">
      和
     </span>
     <code>
      r
     </code>
     <span style="color:#4d4d4d">
      位置的字符，随后
     </span>
     <code>
      l
     </code>
     <span style="color:#4d4d4d">
      右移、
     </span>
     <code>
      r
     </code>
     <span style="color:#4d4d4d">
      左移，直到
     </span>
     <code>
      l &gt;= r
     </code>
     <span style="color:#4d4d4d">
      时终止循环。​
     </span>
    </p>
    <p>
     <strong>
      原地操作
     </strong>
     <span style="color:#4d4d4d">
      ：直接在输入的列表
     </span>
     <code>
      s
     </code>
     <span style="color:#4d4d4d">
      上修改，不占用额外空间，符合题目要求。
     </span>
    </p>

    def countSegments(self, s: str) -&gt; int:
        n = len(s)
        preIsBlank = False
        sum = 0

            if s[i] == " ":

    def isYuanYin(self, s):

    def halvesAreAlike(self, s: str) -&gt; bool:
        n = len(s)
        if n % 2 == 1:
            return False
        count = 0
        for i in range(n // 2):
            if self.isYuanYin(s[i]):
                count += 1
            
        for i in range(n // 2, n):
            if self.isYuanYin(s[i]):
                count -= 1
        return count == 0
