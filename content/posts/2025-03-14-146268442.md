# SQL99 多表查询

## 内连接：

```sql
select name, depart_name, city
from employee e join department d
on e.depart_id = d.depart_id
join location l
on d.locat_id = l.locat_id;
```

最下边两种查询情况需要用关键字"union all"进行连表，这里以左边满外连接为例：

```sql
select ......
from 表1 as a
left join 表2 as b
on a.列名 = b.列名
union all
select ......
from 表1 as a
right join 表2 as b
on a.列名 = b.列名
where a.列名 is null
```
