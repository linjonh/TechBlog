     <a id="_259">

     <strong>
      推荐方式
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       使用
       <code>
        ThreadPoolTaskExecutor
       </code>
       结合
       <code>
        @Autowired
       </code>
       来管理异步任务（推荐）。
      </strong>
     </li>
     <li>
      <strong>
       使用
       <code>
        ThreadPoolTaskScheduler
       </code>
       进行定时任务调度。
      </strong>
     </li>
     <li>
      <strong>
       避免直接使用
       <code>
        ExecutorService
       </code>
       ，因为它不受 Spring 管理，不能动态调整线程池参数。
      </strong>
     </li>
    </ul>

     <strong>
