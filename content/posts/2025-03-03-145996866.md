    <h5>
     查看目录和文件
    </h5>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        ls
       </code>
      </strong>
      ：列出目录内容。
      <ul>
       <li>
        常用选项：
        <ul>
         <li>
          <code>
           -l
          </code>
          ：以长格式显示，显示文件的权限、所有者、大小、修改时间等详细信息。
         </li>
         <li>
          <code>
           -a
          </code>
          ：显示所有文件和目录，包括隐藏文件（以
          <code>
           .
          </code>
          开头的文件）。
         </li>
         <li>
          <code>
           -h
          </code>
          ：与
          <code>
           -l
          </code>
          一起使用，以人类可读的格式显示文件大小（如 KB、MB 等）。
         </li>
        </ul>
       </li>
       <li>
        示例：
        <code>
         ls -lh
        </code>
        会以长格式和人类可读的大小显示当前目录下的文件和文件夹。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        pwd
       </code>
      </strong>
      ：显示当前工作目录的完整路径。
      <ul>
       <li>
        示例：执行
        <code>
         pwd
        </code>
        命令，会输出当前所在目录的绝对路径。
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     创建目录和文件
    </h5>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        mkdir
       </code>
      </strong>
      ：创建新目录。
      <ul>
       <li>
        示例：
        <code>
         mkdir new_directory
        </code>
        会在当前目录下创建一个名为
        <code>
         new_directory
        </code>
        的新目录。
       </li>
       <li>
        若要创建多级目录，可以使用
        <code>
         -p
        </code>
        选项，如
        <code>
         mkdir -p parent/child
        </code>
        会创建
        <code>
         parent
        </code>
        目录及其子目录
        <code>
         child
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        touch
       </code>
      </strong>
      ：创建一个新的空文件，或者更新已有文件的访问和修改时间。
      <ul>
       <li>
        示例：
        <code>
         touch new_file.txt
        </code>
        会在当前目录下创建一个名为
        <code>
         new_file.txt
        </code>
        的空文件。
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     切换目录
    </h5>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        cd
       </code>
      </strong>
      ：用于切换当前工作目录。
      <ul>
       <li>
        示例：
        <ul>
         <li>
          <code>
           cd /home/user/Documents
          </code>
          会切换到
          <code>
           /home/user/Documents
          </code>
          目录。
         </li>
         <li>
          <code>
           cd ..
          </code>
          会切换到当前目录的父目录。
         </li>
         <li>
          <code>
           cd ~
          </code>
          会切换到当前用户的主目录。
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     删除目录和文件
    </h5>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        rm
       </code>
      </strong>
      ：删除文件或目录。
      <ul>
       <li>
        示例：
        <ul>
         <li>
          <code>
           rm file.txt
          </code>
          会删除当前目录下的
          <code>
           file.txt
          </code>
          文件。
         </li>
         <li>
          若要删除目录，需要使用
          <code>
           -r
          </code>
          选项，如
          <code>
           rm -r directory
          </code>
          会递归删除
          <code>
           directory
          </code>
          目录及其所有内容。
         </li>
         <li>
          为了避免误删除，可以使用
          <code>
           -i
          </code>
          选项，它会在删除每个文件或目录之前询问确认，如
          <code>
           rm -ri directory
          </code>
          。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        rmdir
       </code>
      </strong>
      ：删除空目录。
      <ul>
       <li>
        示例：
        <code>
         rmdir empty_directory
        </code>
        会删除当前目录下名为
        <code>
         empty_directory
        </code>
        的空目录。
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     复制和移动文件 / 目录
    </h5>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        cp
       </code>
      </strong>
      ：复制文件或目录。
      <ul>
       <li>
        示例：
        <ul>
         <li>
          <code>
           cp file.txt new_file.txt
          </code>
          会将
          <code>
           file.txt
          </code>
          文件复制为
          <code>
           new_file.txt
          </code>
          。
         </li>
         <li>
          若要复制目录，需要使用
          <code>
           -r
          </code>
          选项，如
          <code>
           cp -r source_directory destination_directory
          </code>
          会将
          <code>
           source_directory
          </code>
          目录及其内容复制到
          <code>
           destination_directory
          </code>
          。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        mv
       </code>
      </strong>
      ：移动文件或目录，也可用于重命名文件或目录。
      <ul>
       <li>
        示例：
        <ul>
         <li>
          <code>
           mv file.txt /home/user/Documents
          </code>
          会将
          <code>
           file.txt
          </code>
          文件移动到
          <code>
           /home/user/Documents
          </code>
          目录。
         </li>
         <li>
          <code>
           mv old_name.txt new_name.txt
          </code>
          会将
          <code>
           old_name.txt
          </code>
          文件重命名为
          <code>
           new_name.txt
          </code>
          。
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     文件内容查看
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        cat
       </code>
      </strong>
      ：将文件内容输出到标准输出（通常是终端）。适用于查看较小的文件。
      <ul>
       <li>
        示例：
        <code>
         cat file.txt
        </code>
        会将
        <code>
         file.txt
        </code>
        文件的内容显示在终端上。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        more
       </code>
      </strong>
      ：分页显示文件内容，按空格键翻页，按
      <code>
       q
      </code>
      键退出。
      <ul>
       <li>
        示例：
        <code>
         more large_file.txt
        </code>
        会分页显示
        <code>
         large_file.txt
        </code>
        的内容。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        less
       </code>
      </strong>
      ：也是分页显示文件内容，但功能比
      <code>
       more
      </code>
      更强大，支持向前和向后翻页，搜索等操作。
      <ul>
       <li>
        示例：
        <code>
         less large_file.txt
        </code>
        会打开
        <code>
         large_file.txt
        </code>
        文件供查看，按
        <code>
         PageUp
        </code>
        和
        <code>
         PageDown
        </code>
        键翻页，按
        <code>
         /
        </code>
        键可进行搜索。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        head
       </code>
      </strong>
      ：显示文件的前几行，默认显示前 10 行。
      <ul>
       <li>
        示例：
        <code>
         head -n 5 file.txt
        </code>
        会显示
        <code>
         file.txt
        </code>
        文件的前 5 行。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        tail
       </code>
      </strong>
      ：显示文件的后几行，默认显示后 10 行。常用于查看日志文件的最新内容。
      <ul>
       <li>
        示例：
        <code>
         tail -n 5 file.txt
        </code>
        会显示
        <code>
         file.txt
        </code>
        文件的后 5 行。
       </li>
       <li>
        使用
        <code>
         -f
        </code>
        选项可以实时跟踪文件的更新，如
        <code>
         tail -f log.txt
        </code>
        会实时显示
        <code>
         log.txt
        </code>
        文件新增的内容。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     权限管理
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        chmod
       </code>
      </strong>
      ：修改文件或目录的权限。
      <ul>
       <li>
        权限分为读（
        <code>
         r
        </code>
        ）、写（
        <code>
         w
        </code>
        ）、执行（
        <code>
         x
        </code>
        ），可以针对文件所有者（
        <code>
         u
        </code>
        ）、所属组（
        <code>
         g
        </code>
        ）、其他用户（
        <code>
         o
        </code>
        ）进行设置。
       </li>
       <li>
        示例：
        <ul>
         <li>
          <code>
           chmod u+x file.sh
          </code>
          会给文件
          <code>
           file.sh
          </code>
          的所有者添加执行权限。
         </li>
         <li>
          也可以使用数字表示权限，如
          <code>
           chmod 755 file.sh
          </code>
          表示所有者具有读、写、执行权限（7），所属组和其他用户具有读、执行权限（5）。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        chown
       </code>
      </strong>
      ：更改文件或目录的所有者和所属组。
      <ul>
       <li>
        示例：
        <code>
         chown user:group file.txt
        </code>
        会将
        <code>
         file.txt
        </code>
        文件的所有者更改为
        <code>
         user
        </code>
        ，所属组更改为
        <code>
         group
        </code>
        。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     进程管理
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        ps
       </code>
      </strong>
      ：显示当前运行的进程信息。
      <ul>
       <li>
        常用选项：
        <ul>
         <li>
          <code>
           -ef
          </code>
          ：显示所有进程的详细信息，包括进程的父进程 ID、启动时间等。
         </li>
        </ul>
       </li>
       <li>
        示例：
        <code>
         ps -ef
        </code>
        会列出系统中所有正在运行的进程。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        top
       </code>
      </strong>
      ：实时显示系统中各个进程的资源占用情况，按
      <code>
       q
      </code>
      键退出。
      <ul>
       <li>
        示例：执行
        <code>
         top
        </code>
        命令后，会实时更新显示系统中 CPU、内存占用较高的进程。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        kill
       </code>
      </strong>
      ：向进程发送信号，通常用于终止进程。
      <ul>
       <li>
        示例：
        <code>
         kill 1234
        </code>
        会向进程 ID 为 1234 的进程发送默认的终止信号（SIGTERM）。
       </li>
       <li>
        若进程无法正常终止，可以使用
        <code>
         -9
        </code>
        选项发送强制终止信号（SIGKILL），如
        <code>
         kill -9 1234
        </code>
        。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     网络操作
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        ping
       </code>
      </strong>
      ：用于测试与目标主机的网络连通性。
      <ul>
       <li>
        示例：
        <code>
         ping www.example.com
        </code>
        会向
        <code>
         www.example.com
        </code>
        发送 ICMP 回显请求数据包，并显示响应信息。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        ifconfig
       </code>
      </strong>
      ：用于查看和配置网络接口的信息，如 IP 地址、子网掩码等。在较新的 Linux 系统中，
      <code>
       ip
      </code>
      命令逐渐取代了它。
      <ul>
       <li>
        示例：
        <code>
         ifconfig eth0
        </code>
        会显示
        <code>
         eth0
        </code>
        网络接口的详细信息。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        ip
       </code>
      </strong>
      ：功能更强大的网络配置命令。
      <ul>
       <li>
        示例：
        <code>
         ip addr show
        </code>
        会显示所有网络接口的 IP 地址信息。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        netstat
       </code>
      </strong>
      ：用于显示网络连接、路由表、网络接口统计等信息。在较新的 Linux 系统中，
      <code>
       ss
      </code>
      命令逐渐取代了它。
      <ul>
       <li>
        示例：
        <code>
         netstat -tuln
        </code>
        会显示所有 TCP 和 UDP 监听端口。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        ss
       </code>
      </strong>
      ：用于显示网络套接字信息，功能类似于
      <code>
       netstat
      </code>
      ，但速度更快。
      <ul>
       <li>
        示例：
        <code>
         ss -tuln
        </code>
        会显示所有 TCP 和 UDP 监听端口。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     系统信息查看
    </h4>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        uname
       </code>
      </strong>
      ：显示系统的基本信息，如内核名称、版本等。
      <ul>
       <li>
        常用选项：
        <ul>
         <li>
          <code>
           -a
          </code>
          ：显示所有信息。
         </li>
        </ul>
       </li>
       <li>
        示例：
        <code>
         uname -a
        </code>
        会显示系统的详细信息，包括内核版本、主机名等。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        df
       </code>
      </strong>
      ：显示文件系统的磁盘使用情况。
      <ul>
       <li>
        常用选项：
        <ul>
         <li>
          <code>
           -h
          </code>
          ：以人类可读的格式显示磁盘空间大小。
         </li>
        </ul>
       </li>
       <li>
        示例：
        <code>
         df -h
        </code>
        会显示各个文件系统的使用情况，以 KB、MB、GB 等单位显示。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        du
       </code>
      </strong>
      ：显示文件和目录的磁盘使用情况。
      <ul>
       <li>
        常用选项：
        <ul>
         <li>
          <code>
           -h
          </code>
          ：以人类可读的格式显示磁盘空间大小。
         </li>
         <li>
          <code>
           -s
          </code>
          ：只显示总计大小。
         </li>
        </ul>
       </li>
       <li>
        示例：
        <code>
         du -sh directory
        </code>
        会显示
        <code>
         directory
        </code>
        目录的总大小。
       </li>
      </ul>
     </li>
    </ul>
