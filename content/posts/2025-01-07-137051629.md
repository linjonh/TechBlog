# 推荐几款常用的流程图开源库

大家好我是辉子，关注 公众号: 【罗米笔记】，有更好的笔记会及时更新

开始

今天推荐的是流程图创建开源库，因为在工作中可能需要用到审批流之类的。

1. jsPlumb

jsPlumb 是一个开源的 JavaScript 库，可以用来创建交互式流程图和数据可视化。它支持各种节点类型，可以自定义节点和连接的样式，还提供了丰富的 API 用于实现复杂的交互效果。

![图片](https://i-blog.csdnimg.cn/blog_migrate/8f01731814110c349d8b409d8d51f594.png)

https://github.com/jsplumb/jsplumb

2. GoJS

GoJS 是一个 JavaScript 和 TypeScript 的库，用于构建交互式图表和图形，包括流程图、组织结构图、状态图等等。该库非常强大，具有灵活的模板和丰富的 API，可以满足各种复杂的流程图需求。

![图片](https://i-blog.csdnimg.cn/blog_migrate/22021caa67504b556c33b4d97eef90fd.png)

https://github.com/NorthwoodsSoftware/GoJS

3. Mermaid

Mermaid 是一种轻量级、简单易用的图表绘制语言，使用markdown形式进行书写，它可以生成流程图、序列图、甘特图等多种图表。Mermaid 使用 JavaScript 实现，其语法清晰简洁，易于学习和使用。

![图片](https://i-blog.csdnimg.cn/blog_migrate/5965cd2ce24f2a82fbeccde161932dbb.png)

https://github.com/mermaid-js/mermaid

4. LogicFlow

是一款基于javascript的流程图编辑框架，提供了一系列流程图相关的交互、编辑功能，也支持简单灵活的节点自定义、插件等拓展机制，开发者可以快速在业务系统内实现类似于流程图的需求

![图片](https://i-blog.csdnimg.cn/blog_migrate/c2316c70fdc6c2cc39f802eabd4f2074.png)

https://github.com/didi/LogicFlow

5. X6

X6 是基于 HTML 和 SVG 的图编辑引擎，提供低成本的定制能力和开箱即用的内置扩展，方便我们快速搭建 DAG 图、ER 图、流程图、血缘图等应用。

![图片](https://i-blog.csdnimg.cn/blog_migrate/eac9189fc1a3729a008b66323cf622ed.png)

https://github.com/antvis/X6

6. mxGraph

mxGraph 是一个开源的 JavaScript 图形库，用于创建交互式图形和图表，包括流程图、组织结构图、网络拓扑图等。mxGraph 具有优秀的性能和灵活的 API，可以满足各种复杂的图形绘制需求。文档太少了，draw.io是基于这个开发

![图片](https://i-blog.csdnimg.cn/blog_migrate/e689a7833d1a7fcf970b85e03885ef78.png)

https://github.com/jgraph/mxgraph
