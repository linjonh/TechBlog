# 微软把狐狸（FoxPro）放回牧场

记得


18

年前，在


DOS

平台上，我曾经使用一种“语言”来处理数据表单，得心应手，乐此不疲，简直入了迷，那就是


dBASE

（数据库管理系统），后来，转向了微软的“狐狸”数据库，即


FoxPro

。近日，微软宣布即将把


FoxPro

的核心部件“开源”，同时还宣布，在


FoxPro 9.0

版本之后，不再发布新的


FoxPro

版本，但是，微软对


FoxPro

提供的技术支持将继续延续到


2015

年。

微软把


FoxPro

“开源”，如同放“狐”归山，还其自由，应该受到欢迎。微软自然明白，许多人都喜欢应用


FoxPro

。微软此举非常英明，将促使“开源


FoxPro

社区”的成长和壮大，使得


FoxPro

能够再度复兴。我们把话说明白了，由此，在


Windows

平台上也有了“开源社区”，开源（


Open Source

）不是


Linux

领域的专利，因此，即使站在私有软件立场上，也不可轻视“开源”。

这半年来，


Java

开源，


FoxPro

也开源了，这是什么原因造成的？没有什么特别“原因”，只是大势所趋而已。
