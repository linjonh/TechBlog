    <h2>
     <a id="Windows__RabbitMQ__1">
     </a>
     Windows 系统下安装 RabbitMQ 的详细指南
    </h2>
    <blockquote>
     <p>
      <strong>
       Erlang版本
      </strong>
      ：26.2
      <br/>
      <strong>
       RabbitMQ 版本
      </strong>
      ：3.13.7
      <br/>
      <br/>
      确保
      <strong>
       RabbitMQ
      </strong>
      的版本与已安装的
      <strong>
       Erlang
      </strong>
      版本兼容。您可以在
      <strong>
       RabbitMQ
      </strong>
      的
      <a href="https://www.rabbitmq.com/which-erlang.html" rel="nofollow">
       <strong>
        版本兼容性页面
       </strong>
      </a>
      查看详细信息。
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9679e94a0efa4b57bd9431efd617d21e.png"/>
     </p>
    </blockquote>
    <h3>
     <a id="1__6">
     </a>
     1. 前言
    </h3>
    <p>
     RabbitMQ 是一个流行的开源消息代理，采用 AMQP（高级消息队列协议）标准。它允许应用程序之间以异步方式交换数据，确保消息的可靠性和灵活性。RabbitMQ 适用于各种场景，如微服务架构、异步处理、工作队列等。
    </p>
    <h3>
     <a id="2__10">
     </a>
     2. 安装前的准备
    </h3>
    <p>
     在安装 RabbitMQ 之前，请确保您的 Windows 系统符合以下要求：
    </p>
    <ul>
     <li>
      <strong>
       操作系统
      </strong>
      ：Windows 7、8、10 或更高版本
     </li>
     <li>
      <strong>
       架构
      </strong>
      ：x64（64 位）
     </li>
    </ul>
    <p>
     此外，RabbitMQ 依赖于 Erlang 环境，因此需要先安装 Erlang。
    </p>
    <h3>
     <a id="3__19">
     </a>
     3. 安装步骤
    </h3>
    <h4>
     <a id="31__Erlang_21">
     </a>
     3.1 下载并安装 Erlang
    </h4>
    <ol>
     <li>
      <strong>
       访问 Erlang 官方下载页面
      </strong>
      ：
      <a href="https://www.erlang.org/downloads" rel="nofollow">
       https://www.erlang.org/downloads
      </a>
     </li>
     <li>
      <strong>
       选择适合您系统的安装程序
      </strong>
      ，通常为
      <code>
       Windows 64-bit Installer
      </code>
      ，选择匹配的版本。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/19bab0e631e64256bb63a4b4d6db028c.png"/>
     </li>
     <li>
      <strong>
       运行安装程序
      </strong>
      ，按照提示完成安装，默认
      <code>
       next
      </code>
      即可。
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：安装路径中请勿包含空格，以避免潜在的问题。
    </p>
    <h4>
     <a id="32__RabbitMQ_30">
     </a>
     3.2 下载并安装 RabbitMQ
    </h4>
    <ol>
     <li>
      <strong>
       访问 RabbitMQ 官方下载页面
      </strong>
      ：
      <a href="https://www.rabbitmq.com/download.html" rel="nofollow">
       https://www.rabbitmq.com/download.html
      </a>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e0c6f91a9d144d9b9f7e2bc8201be80b.png"/>
     </li>
     <li>
      <strong>
       下载适用于 Windows 的安装程序
      </strong>
      ，选择合适的版本，例如最新的
      <code>
       rabbitmq-server-4.0.7.exe
      </code>
      。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/27edd34f583542e7ab03657a4e3fa357.png"/>
     </li>
     <li>
      <strong>
       运行安装程序
      </strong>
      ，按照提示完成安装，可以自定义安装路径。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5eb998acd34b470884b6ebde83553124.png"/>
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：确保
     <strong>
      RabbitMQ
     </strong>
     的版本与已安装的
     <strong>
      Erlang
     </strong>
     版本兼容。您可以在
     <strong>
      RabbitMQ
     </strong>
     的
     <a href="https://www.rabbitmq.com/which-erlang.html" rel="nofollow">
      <strong>
       版本兼容性页面
      </strong>
     </a>
     查看详细信息。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/35a4e4ffac4c4e6bb6d4f1a46daa93d2.png"/>
    </p>
    <h4>
     <a id="33__45">
     </a>
     3.3 配置环境变量
    </h4>
    <ol>
     <li>
      <strong>
       打开系统环境变量设置
      </strong>
      ：
      <ul>
       <li>
        右键点击“此电脑”或“计算机”，选择“属性”
       </li>
       <li>
        点击“高级系统设置”
       </li>
       <li>
        点击“环境变量”
       </li>
      </ul>
     </li>
     <li>
      <strong>
       在“系统变量”中找到
       <code>
        Path
       </code>
      </strong>
      ，并点击“编辑”
     </li>
     <li>
      <strong>
       添加 Erlang 和 RabbitMQ 的安装路径
      </strong>
      ，例如：
      <ul>
       <li>
        <code>
         C:\Program Files\Erlang\bin
        </code>
       </li>
       <li>
        <code>
         C:\Program Files\RabbitMQ Server\rabbitmq_server-4.0.7\sbin
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="34__56">
     </a>
     3.4 验证安装
    </h4>
    <ol>
     <li>
      <strong>
       打开命令提示符
      </strong>
      ，输入
      <code>
       erl
      </code>
      ，如果显示 Erlang 的版本信息，则表示 Erlang 安装成功。
     </li>
     <li>
      <strong>
       输入
       <code>
        rabbitmqctl status
       </code>
      </strong>
      ，如果显示 RabbitMQ 的状态信息，则表示 RabbitMQ 安装成功。
     </li>
    </ol>
    <h4>
     <a id="35__RabbitMQ__61">
     </a>
     3.5 启用 RabbitMQ 管理插件
    </h4>
    <p>
     RabbitMQ 提供了一个基于 Web 的管理界面，方便用户管理和监控。
    </p>
    <ol>
     <li>
      进入安装目录的
      <strong>
       /sbin
      </strong>
      目录下，
      <strong>
       在命令提示符中输入
      </strong>
      ：
      <code>
       rabbitmq-plugins enable rabbitmq_management
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dae2d3aa113d4ed4b71d3efde8796c57.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e4e990e7fc294b619596aec0514a1b2d.png"/>
      </img>
     </li>
     <li>
      <strong>
       重新启动 RabbitMQ 服务
      </strong>
      ：
      <ul>
       <li>
        打开“服务”管理器
       </li>
       <li>
        找到
        <code>
         RabbitMQ
        </code>
        服务
       </li>
       <li>
        右键选择“重新启动”
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bb1573534b004c1bb1fcd81578592ec9.png"/>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       访问管理界面
      </strong>
      ：在浏览器中输入
      <code>
       http://localhost:15672/
      </code>
      <ul>
       <li>
        <strong>
         默认用户名
        </strong>
        ：
        <code>
         guest
        </code>
       </li>
       <li>
        <strong>
         默认密码
        </strong>
        ：
        <code>
         guest
        </code>
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d976a6f094a542bead119c3b104fe358.png"/>
        <br/>
        <strong>
         注意
        </strong>
        ：出于安全考虑，默认的
        <code>
         guest
        </code>
        用户只能本地访问，建议在生产环境中创建新的用户并赋予相应权限。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="4__79">
     </a>
     4. 常见问题解决
    </h3>
    <h4>
     <a id="41_RabbitMQ__81">
     </a>
     4.1 RabbitMQ 服务无法启动
    </h4>
    <ul>
     <li>
      <strong>
       检查 Erlang 版本
      </strong>
      ：确保 Erlang 版本与 RabbitMQ 兼容。参考：
      <a href="https://www.rabbitmq.com/which-erlang.html" rel="nofollow">
       https://www.rabbitmq.com/which-erlang.html
      </a>
     </li>
     <li>
      <strong>
       检查端口占用
      </strong>
      ：RabbitMQ 默认使用端口
      <code>
       5672
      </code>
      和
      <code>
       15672
      </code>
      ，确保这些端口未被占用。
     </li>
    </ul>
    <h4>
     <a id="42__86">
     </a>
     4.2 无法访问管理界面
    </h4>
    <ul>
     <li>
      <strong>
       确保管理插件已启用
      </strong>
      ：执行
      <code>
       rabbitmq-plugins enable rabbitmq_management
      </code>
      。
     </li>
     <li>
      <strong>
       检查防火墙设置
      </strong>
      ：确保端口
      <code>
       15672
      </code>
      未被防火墙阻止。
     </li>
    </ul>
