# python安装目录查看\_怎么查看python的安装目录

![]()

怎么查看python的安装目录？

linux中一般安装后的默认路径如下#!/usr/bin/env python

或者

#!/usr/locat/bin/python

如果没有找到，可以通过命令查询python路径whereis python

或者

which python

Windows OS

可以通过以下方式查询python路径或者第三方模块安装路径。>>> import sys

>>> sys.path
