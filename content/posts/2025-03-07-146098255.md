# 体验OpenManus

这里两天Manus非常火，但是根本抢不到邀请码。

发现有个开源项目：

```
https://github.com/mannaandpoem/OpenManus
```

于是迫不及待地体验了一下，上手很简单，就按官方文档来就行。

然后记得修改下config.toml的配置，我修改成openai的代理方式。

```python
# Global LLM configuration
[llm]
model = "gpt-4-turbo"
base_url = "https://api.openai-proxy.org/v1"
api_key = "sk-*****"
max_tokens = 4096
temperature = 0.0

# Optional configuration for specific LLM models
[llm.vision]
model = "gpt-4-turbo"
base_url = "https://api.openai-proxy.org/v1"
api_key = "sk-*****"
```

之前的介绍国内使用代理访问openai的文章：

[国内访问OpenAI API\_openai代理-CSDN博客](https://blog.csdn.net/weyson/article/details/142656848?fromshare=blogdetail&sharetype=blogdetail&sharerId=142656848&sharerefer=PC&sharesource=weyson&sharefrom=from_link "国内访问OpenAI API_openai代理-CSDN博客")

等启动后输入指令：

使用python制作一款类似flybird的小游戏，输出到game目录

![](https://i-blog.csdnimg.cn/direct/8db22d66d3ff4d1eaf1a0d24d63b71b9.png)

然后等他一顿疯狂输出。。。。。。

然后game目录下就生成了一个py文件

![](https://i-blog.csdnimg.cn/direct/8e4779a6e87c45ad8be960ba1de28d7f.png)

然后执行这个py文件

```bash
python flybird.py
```

游戏就直接运行起来了

![](https://i-blog.csdnimg.cn/direct/a12206f5a6a441c0ad386f67df3adf6d.png)

虽然画面有点low，但起码程序是可以跑的，体验下来，感觉还是很强大的。
