    <h4>
     git 源码泄露
    </h4>
    <p>
     Git 是一个开源的分布式版本控制系统，在执行 git init 初始化目录时，会在当前目录下自动创建 .git 目录，记录代码的变更记录等。发布代码时，如果没把 .git 目录删除，就直接发布到了服务器上，攻击者就可以通过它来恢复源代码。
    </p>
