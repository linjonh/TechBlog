    <p>
     hdc（HarmonyOS Device Connector）是 HarmonyOS 为开发人员提供的用于调试的命令行工具，通过该工具可以在 windows/linux/mac 系统上与真实设备或者模拟器进行交互。
    </p>
    <p>
     hdc 工具通过 HarmonyOS SDK 获取，存放于 /Huawei/Sdk/openharmony/版本号/toolchains/ 目录下。
    </p>
    <h3>
     <a id="Windows__hdc__4">
     </a>
     一、Windows 系统 hdc 环境变量设置方法
    </h3>
    <h4>
     <a id="1_5">
     </a>
     1、打开环境变量
    </h4>
    <pre><code>右键 此电脑 &gt; 属性 &gt; 高级系统设置 &gt; 高级 &gt; 环境变量
</code></pre>
    <h4>
     <a id="2_10">
     </a>
     2、新建系统变量
    </h4>
    <pre><code>环境变量 &gt; 系统环境 &gt; 新建
变量名：OHOS_HOME
变量值：D:\Huawei\Sdk
</code></pre>
    <h4>
     <a id="3_Path_16">
     </a>
     3、添加到 Path
    </h4>
    <pre><code>找到 环境变量 &gt; 系统环境 &gt; Path 变量
然后双击打开，点击新建
在最后一行填写 %OHOS_HOME%\openharmony\版本号\toolchains
</code></pre>
    <h4>
     <a id="4_hdc_22">
     </a>
     4、检查是否已经成功配置 hdc
    </h4>
    <pre><code>在终端输入 hdc version
</code></pre>
    <p>
     如果出现类似如下内容证明 hdc 配置成功：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/40f0da56e39fbf7e2c17b8453dd02414.png#pic_center"/>
    </p>
    <h3>
     <a id="macOS__hdc__29">
     </a>
     二、macOS 系统 hdc 环境变量设置方法
    </h3>
    <h4>
     <a id="1_bash_profile__30">
     </a>
     1、编辑 .bash_profile 文件
    </h4>
    <pre><code>vim ~/.bash_profile
</code></pre>
    <h4>
     <a id="2_bash_profile__34">
     </a>
     2、在 .bash_profile 文件中添加以下内容
    </h4>
    <pre><code>export PATH=${PATH}:/Users/用户名/Library/Huawei/Sdk/openharmony/版本/toolchains
</code></pre>
    <p>
     <code>
      如果是下载的SDK包可以在几个文件夹下面找一下toolchains文件夹，找到后copy这个文件夹的路径即可
     </code>
    </p>
    <h4>
     <a id="3_bash_profile__40">
     </a>
     3、保存 .bash_profile 文件，并退出
    </h4>
    <pre><code>点击 esc
输入 :wq
回车
</code></pre>
    <h4>
     <a id="4_bash_profile__46">
     </a>
     4、加载并执行 .bash_profile 文件
    </h4>
    <pre><code>source ~/.bash_profile
</code></pre>
    <h4>
     <a id="5_hdc_50">
     </a>
     5、检查是否已经成功配置 hdc
    </h4>
    <pre><code>在终端输入 hdc version
</code></pre>
    <p>
     如果出现类似如下内容证明 hdc 配置成功：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/520ed7a6afe3ef0c7e0327eb8d641c4d.png"/>
    </p>
    <p>
     注：如果配置成功就不用看第6步。
    </p>
    <h4>
     <a id="6_59">
     </a>
     6、问题修复
    </h4>
    <p>
     给 ~/.bash_profile 文件一个启动命令
    </p>
    <pre><code>vim ~/.zshrc

配置增加中增加
source ~/.bash_profile

然后刷新配置命令
source ~/.zshrc

再次执行 hdc 命令一切正常
</code></pre>
    <h4>
     <a id="7_73">
     </a>
     7、注意
    </h4>
    <p>
     不知道为什么，我这个mac的环境变量配置到zshrc不行，但是配置到bash_profile就可以了，很诡异，有大佬了解的可以说一下原因。
    </p>
