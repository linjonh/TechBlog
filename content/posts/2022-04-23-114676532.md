    <p>
     黑轴键盘是更适合玩游戏的一款键盘。
     <br/>
     黑轴，多数游戏玩家的利器。无段落感，唯一的确认感就是把按键按压到底的触碰感。
     <br/>
     选黑轴还是红轴这些点很重要看过你就懂了
     <br/>
     <a href="http://www.adiannao.cn/dy" rel="nofollow">
      http://www.adiannao.cn/dy
     </a>
     <br/>
     干脆，直接，行云流水般畅快，非常适合竞技类的游戏。它很高效，虽然因为无段落感造成键程很长的错觉（其实全都一样），但是其实只需蜻蜓点水般点击，即可触发。
     <br/>
     黑轴的压力系数最大，按键键程最短，所以按起来反应迅速，是玩魔兽，星际，DOTA等RTS游戏的好选择。
     <br/>
     这不是我个人主观的一个印象，是有科学依据的，以下是详细对比：
     <br/>
     一、黑轴：压力克数：80
     <br/>
     触发：1.5mm
     <br/>
     特点：无段落感
     <br/>
     寿命：5000万-6000万次
     <br/>
     重度爱好者、长时间玩游戏的玩家首选，如果打字会很累的。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/afe5ad2d5c00bbb3b11aa32b90b49801.jpeg#pic_center"/>
    </p>
    <p>
     二、红轴：
     <br/>
     压力克数：40
     <br/>
     触发：2.0mm
     <br/>
     特点：无段落感
     <br/>
     寿命：5000万次
     <br/>
     长时间文字输入，红轴是个非常不错的选择，红轴可看作一个轻量级的黑轴：和黑轴一样的直上直下，但是压克力数更小。
