    <h3>
     <a id="JavaScript_1">
     </a>
     一、JavaScript基础
    </h3>
    <h4>
     <a id="1_3">
     </a>
     1.变量（重点）
    </h4>
    <ul>
     <li>
      <p>
       变量指的是在程序中保存数据的一个容器
      </p>
      <blockquote>
       <p>
        语法：var 变量名 = 值
       </p>
      </blockquote>
     </li>
    </ul>
    <h5>
     <a id="11__9">
     </a>
     1-1 定义变量及赋值
    </h5>
    <pre><code class="prism language-javascript"><span class="token comment">//定义一个变量</span>
<span class="token keyword">var</span> num

<span class="token comment">//给一个变量赋值</span>
num <span class="token operator">=</span> <span class="token number">100</span>

<span class="token comment">//定义一个变量的同时给其赋值</span>
<span class="token keyword">var</span> num2 <span class="token operator">=</span> <span class="token number">200</span>
</code></pre>
    <ul>
     <li>
      注意：
      <ol>
       <li>
        一个变量名只能存储一个值
       </li>
       <li>
        当再次给一个变量赋值的时候，前面一次的值就没有了
       </li>
       <li>
        变量名称区分大小写（
        <code>
         js
        </code>
        严格区分大小写）
       </li>
      </ol>
     </li>
    </ul>
    <h5>
     <a id="12__27">
     </a>
     1-2 变量的命名规则和命名规范
    </h5>
    <ul>
     <li>
      <p>
       规则：必须需遵守的
      </p>
      <blockquote>
       <ol>
        <li>
         一个变量名称可以由**数字、字母、英文下划线（__）、美元符号（$）**组成
        </li>
        <li>
         严格区分大小写
        </li>
        <li>
         不能由数字开头
        </li>
        <li>
         不能是
         <strong>
          保留字
         </strong>
         或者
         <strong>
          关键字
         </strong>
        </li>
        <li>
         不要出现空格
        </li>
       </ol>
      </blockquote>
     </li>
     <li>
      <p>
       规范：建议遵守
      </p>
      <blockquote>
       <ol>
        <li>
         变量名尽量有意义（语义化）
        </li>
        <li>
         遵循骆驼命名规则，有多个单词组成的时候，从第二个单词开始首字母大写
        </li>
        <li>
         不要使用中文
        </li>
       </ol>
      </blockquote>
     </li>
     <li>
      <h6>
       <a id="_43">
       </a>
       判断数据类型：
      </h6>
      <blockquote>
       <ul>
        <li>
         <p>
          使用
          <code>
           typeof
          </code>
          关键字来进行判断
         </p>
         <pre><code class="prism language-javascript"><span class="token comment">//第一种使用方式</span>
<span class="token keyword">var</span> n1 <span class="token operator">=</span> <span class="token number">100</span>
console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token keyword">typeof</span> n1<span class="token punctuation">)</span>

<span class="token comment">//第二种使用方式</span>
<span class="token keyword">var</span> s1 <span class="token operator">=</span> <span class="token string">'abc'</span>
console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token keyword">typeof</span><span class="token punctuation">(</span>s1<span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
        </li>
       </ul>
      </blockquote>
     </li>
    </ul>
    <h4>
     <a id="2_57">
     </a>
     2.数据类型转换
    </h4>
    <ul>
     <li>
      数据类型之间的转换，比如数字转成字符串，字符串转成布尔，布尔转成数字等
     </li>
    </ul>
    <h5>
     <a id="21__61">
     </a>
     2-1 其他数据类型转成数值
    </h5>
    <ol>
     <li>
      <p>
       <code>
        Number(变量)
       </code>
      </p>
      <blockquote>
       <p>
        可以把一个变量强制转换成数值类型
       </p>
       <p>
        可以转换小数，会保留小数
       </p>
       <p>
        可以转换成布尔值
       </p>
       <p>
        以下都是：遇到不可转换的都会返回
        <code>
         NaN
        </code>
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       <code>
        parseInt(变量)
       </code>
      </p>
      <blockquote>
       <p>
        从第一位开始检查，是数字就转换，直到一个不是数字的内容
       </p>
       <p>
        开头不是数字就直接返回
        <code>
         NaN
        </code>
       </p>
       <p>
        不认识小数点，只能保留整数
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       <code>
        parseFloat(变量)
       </code>
      </p>
      <blockquote>
       <p>
        从第一位开始检查，是数字就转换，直到一个不是数字的内容
       </p>
       <p>
        开头不是数字就直接返回
        <code>
         NaN
        </code>
       </p>
       <p>
        认识一次小数点
       </p>
      </blockquote>
     </li>
    </ol>
    <h5>
     <a id="22__89">
     </a>
     2-2 其他数据类型转成字符串
    </h5>
    <ol>
     <li>
      <p>
       <code>
        变量.toString()
       </code>
      </p>
      <blockquote>
       <p>
        有一些数据类型不能用
        <code>
         toString()
        </code>
        方法，如：
        <code>
         undefined
        </code>
        和
        <code>
         null
        </code>
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       <code>
        String(变量)
       </code>
      </p>
      <blockquote>
       <p>
        所有数据类型都能用
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       使用加法运算
      </p>
      <blockquote>
       <p>
        字符串拼接：有一方是字符串就会拼接
       </p>
       <p>
        加法运算：只有两边都是数字才会进行数学运算
       </p>
      </blockquote>
     </li>
    </ol>
    <h5>
     <a id="23__105">
     </a>
     2-3 其他数据类型转成布尔
    </h5>
    <ol>
     <li>
      <p>
       <code>
        Boolean(变量)
       </code>
      </p>
      <blockquote>
       <p>
        在js中，只有
        <code>
         ''
        </code>
        、
        <code>
         0
        </code>
        、
        <code>
         null
        </code>
        、
        <code>
         undefined
        </code>
        、
        <code>
         NaN
        </code>
        ，这些是
        <code>
         false
        </code>
        ，其他都是
        <code>
         true
        </code>
       </p>
      </blockquote>
     </li>
    </ol>
    <h4>
     <a id="3_111">
     </a>
     3.函数
    </h4>
    <ul>
     <li>
      <p>
       对于
       <code>
        js
       </code>
       来说，函数就是把任意一段代码放在一个
       <strong>
        盒子
       </strong>
       里面
      </p>
     </li>
     <li>
      <p>
       代码：
      </p>
      <pre><code class="prism language-javascript"><span class="token comment">//这个是我们以前写的一段代码</span>
<span class="token keyword">for</span> <span class="token punctuation">(</span><span class="token keyword">var</span> i <span class="token operator">=</span> <span class="token number">0</span> <span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span> <span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>i<span class="token punctuation">)</span>
<span class="token punctuation">}</span>

<span class="token comment">//函数，这个 {} 就是那个“盒子”</span>
<span class="token keyword">function</span> <span class="token function">fu</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">//这个函数我们以前写的代码</span>
    <span class="token keyword">for</span> <span class="token punctuation">(</span><span class="token keyword">var</span> i <span class="token operator">=</span> <span class="token number">0</span> <span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span> <span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    	console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>i<span class="token punctuation">)</span>
	<span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
      <h5>
       <a id="31_132">
       </a>
       3-1函数定义阶段
      </h5>
      <p>
       <strong>
        声明式
       </strong>
      </p>
      <ul>
       <li>
        <p>
         使用
         <code>
          function
         </code>
         这个关键字来声明一个函数
        </p>
       </li>
       <li>
        <p>
         语法：
        </p>
        <pre><code class="prism language-javascript"><span class="token keyword">function</span> <span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">//一段代码</span>
<span class="token punctuation">}</span>
<span class="token comment">//function：声明函数的关键字</span>
<span class="token comment">//fn：函数的名字，自己定义</span>
<span class="token comment">//（）：必须写，放参数的位置</span>
<span class="token comment">//{} ： 就是放一段代码的位置也就是刚刚说的“盒子”</span>
</code></pre>
       </li>
      </ul>
      <p>
       <strong>
        赋值式
       </strong>
      </p>
      <ul>
       <li>
        <p>
         其实就是使用
         <code>
          var
         </code>
         关键字一样
        </p>
       </li>
       <li>
        <p>
         首先使用
         <code>
          var
         </code>
         定义一个变量，把一个函数当做值直接复制给这个变量就可以了
        </p>
       </li>
       <li>
        <p>
         语法：
        </p>
        <pre><code class="prism language-javascript"><span class="token keyword">var</span> <span class="token function-variable function">fn</span> <span class="token operator">=</span> <span class="token keyword">function</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">//一段代码</span>
<span class="token punctuation">}</span>
</code></pre>
       </li>
      </ul>
      <h5>
       <a id="32_164">
       </a>
       3-2函数调用阶段
      </h5>
      <p>
       <strong>
        调用一个函数
       </strong>
      </p>
      <ul>
       <li>
        函数调用就是直接谢
        <code>
         函数名（）
        </code>
        即可
       </li>
      </ul>
      <pre><code class="prism language-javascript"><span class="token comment">//声明式函数</span>
<span class="token keyword">function</span> <span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">"我是fn函数"</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>

<span class="token comment">//调用函数</span>
<span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span>

<span class="token comment">//赋值式函数</span>
<span class="token keyword">var</span> <span class="token function-variable function">f1</span> <span class="token operator">=</span> <span class="token keyword">function</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
    console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">"我是f1函数"</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>

<span class="token comment">//调用函数</span>
<span class="token function">f1</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
      <ul>
       <li>
        注意：
        <strong>
         定义完一个函数以后，如果没有调用函数，那么写在{}里面的代码没有意义，只有调用以后才会执行
        </strong>
       </li>
      </ul>
      <h5>
       <a id="33__190">
       </a>
       3-3 调用上的区别
      </h5>
      <ul>
       <li>
        声明式函数：调用可以在
        <strong>
         定义之前或者定义之后
        </strong>
       </li>
       <li>
        赋值式函数：调用只能在
        <strong>
         定义之后
        </strong>
       </li>
      </ul>
      <h5>
       <a id="34__195">
       </a>
       3-4 函数的参数
      </h5>
      <ul>
       <li>
        <p>
         参数分为两种
         <strong>
          形参
         </strong>
         和
         <strong>
          实参
         </strong>
        </p>
        <pre><code class="prism language-javascript"><span class="token comment">//声明式</span>
<span class="token keyword">function</span> <span class="token function">fn</span><span class="token punctuation">(</span><span class="token parameter">形参</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    一段代码
<span class="token punctuation">}</span>

<span class="token function">fn</span><span class="token punctuation">(</span>实参<span class="token punctuation">)</span>

<span class="token comment">//赋值式一样的位置，写在"()"里面</span>
</code></pre>
       </li>
      </ul>
      <h5>
       <a id="35__211">
       </a>
       3-5 函数的定义域
      </h5>
      <ul>
       <li>
        <p>
         <strong>
          访问规则
         </strong>
        </p>
        <ul>
         <li>
          首先，在自己的作用域内部查找，如果有，就直接拿来用
         </li>
         <li>
          如果没有，就去上一级作用于查找，如果有，就拿来用
         </li>
         <li>
          以此类推
         </li>
         <li>
          如果一直没有这个变量就会直接报错
         </li>
        </ul>
        <pre><code class="prism language-javascript"><span class="token keyword">var</span> num <span class="token operator">=</span> <span class="token number">100</span>

<span class="token keyword">function</span> <span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">var</span> num2 <span class="token operator">=</span> <span class="token number">200</span>
    
    <span class="token keyword">function</span> <span class="token function">fn2</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">var</span> num3 <span class="token operator">=</span> <span class="token number">300</span>
        
        console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>num3<span class="token punctuation">)</span><span class="token comment">//300</span>
        console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>num2<span class="token punctuation">)</span><span class="token comment">//200</span>
        console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>num<span class="token punctuation">)</span><span class="token comment">//100</span>
        console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>a<span class="token punctuation">)</span><span class="token comment">//都没有就是直接报错</span>
    <span class="token punctuation">}</span>
    
    <span class="token function">fn2</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>

<span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
        <ul>
         <li>
          变量的访问规则也叫做作用域的查询机制
         </li>
         <li>
          作用域的查找机制只能是向上查找不能向下找
         </li>
        </ul>
        <pre><code class="prism language-javascript"><span class="token keyword">function</span> <span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">var</span> num <span class="token operator">=</span> <span class="token number">100</span>
<span class="token punctuation">}</span>
<span class="token function">fn</span><span class="token punctuation">(</span><span class="token punctuation">)</span>

console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>num<span class="token punctuation">)</span><span class="token comment">//发现自己作用域没有，自己就是全局作用域，没有上一级了，直接报错</span>
</code></pre>
       </li>
       <li>
        <p>
         <strong>
          赋值规则
         </strong>
        </p>
        <ul>
         <li>
          先在自己作用域内部查找，有就直接赋值
         </li>
         <li>
          没有就去上一级作用域内部查找，有就直接赋值
         </li>
         <li>
          以此类推
         </li>
         <li>
          如果一直找到全局作用域都没有，那么就把这个变量定义为全局变量，再给他赋值
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="4_260">
     </a>
     4.对象
    </h4>
    <ul>
     <li>
      <p>
       对象是一个复杂数据类型
      </p>
     </li>
     <li>
      <p>
       存储了一些基本数据类型的一个集合
      </p>
      <pre><code class="prism language-javascript"><span class="token keyword">var</span> obj <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
    <span class="token literal-property property">num</span> <span class="token operator">:</span> <span class="token number">100</span><span class="token punctuation">,</span>
    <span class="token literal-property property">str</span> <span class="token operator">:</span> <span class="token string">'hello world'</span><span class="token punctuation">,</span>
    <span class="token literal-property property">boo</span> <span class="token operator">:</span> <span class="token boolean">true</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       这里的
       <code>
        {}
       </code>
       和函数中的
       <code>
        {}
       </code>
       不一样
      </p>
     </li>
     <li>
      <p>
       函数里面的是写代码的，而对象里面是写一些数据的
      </p>
     </li>
     <li>
      <p>
       <strong>
        对象就是一个键值对的集合
       </strong>
      </p>
     </li>
     <li>
      <p>
       <code>

     6. DOM节点
    </h4>
    <ul>
     <li>
      <code>
       DOM
      </code>
      的节点一般分为常用的三大类

      </strong>

      </strong>

     <li>

     </li>

       <code>
        ()
       </code>
      </p>
     </li>
     <li>
      <p>
       函数体只有一句话或者只有返回值的时候，可以省略
       <code>
        {}
       </code>
      </p>
     </li>
     <li>
      <p>
       函数体只有一句
       <code>
        return
       </code>
       时，可以省略
       <code>
        return
       </code>
      </p>
     </li>
     <li>
      <p>
       注意如果返回的是一个对象类型的时候，省略了
       <code>
        return
       </code>

       <code>
        {}
       </code>

       <code>
        {}
       </code>

       <code>
        ()
       </code>

       <code>

      </p>
     </li>
     <li>
      <p>

       <code>
        arguments
       </code>
      </p>
      <blockquote>
       <p>

        <code>
         arguments[0]、arguments[1]....
        </code>
        获取
       </p>
      </blockquote>
     </li>
     <li>
      <p>
       箭头函数没有
       <code>
        this
       </code>

       <code>

      </p>
     </li>
     <li>
      <p>

    <h4>
     <a id="7_jsonp_2589">
     </a>
     7. jsonp
    </h4>
    <ul>
     <li>
      <p>
       <code>
        Jsonp
       </code>
       是
       <code>
        json
       </code>
       的一种"使用模式"，可以让网页从别的域名（网站）获取资料，即跨域读取数据。
      </p>
     </li>
     <li>
      <p>
       因为同源策略，才需要这个特殊的技术
      </p>
      <pre><code class="prism language-javascript"><span class="token keyword">const</span> script <span class="token operator">=</span> document<span class="token punctuation">.</span><span class="token function">createElement</span><span class="token punctuation">(</span><span class="token string">'script'</span><span class="token punctuation">)</span>
script<span class="token punctuation">.</span>src <span class="token operator">=</span> <span class="token string">'./kerwin.txt'</span><span class="token comment">//这里是要传的地址</span>
document<span class="token punctuation">.</span>body<span class="token punctuation">.</span><span class="token function">appendChild</span><span class="token punctuation">(</span>script<span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <p>
       <code>
        jsonp
       </code>
       原理：动态创建script标签，src属性指向没有跨域限制，指向一个接口，接口返回的格式一定是***()函数表达式，函数的参数就是接口返回的数据
      </p>
     </li>
     <li>
      <p>
       注意：
      </p>
      <ol>
       <li>
        <p>
         后端接口形式必须***()，需要后端配合
        </p>
       </li>
       <li>
        <p>
         jsonp缺点：
        </p>
        <ul>
         <li>
          <p>
           获取了数据之后删除
          </p>
          <pre><code class="prism language-javascript"><span class="token comment">//因为它获取数据是每次都插入了一个script标签，然后获取数据，获取完数据之后就可以将这个标签删除了，直接用onload删除</span>
script<span class="token punctuation">.</span><span class="token function-variable function">onload</span> <span class="token operator">=</span> <span class="token keyword">function</span> <span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	script<span class="token punctuation">.</span><span class="token function">remove</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
         </li>
         <li>
          <p>
           只能
           <code>
            get
           </code>
           请求，不能
           <code>
            post
           </code>
           、
           <code>
            put
           </code>
           、
           <code>
            delete
           </code>
          </p>
         </li>
        </ul>
       </li>
      </ol>
     </li>
    </ul>
