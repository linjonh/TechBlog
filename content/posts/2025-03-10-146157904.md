    <h6>
     <a id="Redis__4">
     </a>
     Redis 命令
    </h6>
    <p>
     Redis 命令用于在 redis 服务上执行操作。
    </p>
    <p>
     要在 redis 服务上执行命令需要一个 redis 客户端。Redis 客户端在我们之前下载的的 redis 的安装包中。
    </p>
    <h6>
     <a id="_10">
     </a>
     语法
    </h6>
    <p>
     Redis 客户端的基本语法为：
    </p>
    <pre><code>$ redis-cli
</code></pre>
    <h6>
     <a id="_17">
     </a>
     实例
    </h6>
    <p>
     以下实例讲解了如何启动 redis 客户端：
    </p>
    <p>
     启动 redis 服务器，打开终端并输入命令
     <strong>
      redis-cli
     </strong>
     ，该命令会连接本地的 redis 服务。
    </p>
    <pre><code>$ redis-cli
redis 127.0.0.1:6379&gt;
redis 127.0.0.1:6379&gt; PING

PONG
</code></pre>
    <p>
     在以上实例中我们连接到本地的 redis 服务并执行
     <strong>
      PING
     </strong>
     命令，该命令用于检测 redis 服务是否启动。
    </p>
    <hr/>
    <h6>
     <a id="_34">
     </a>
     在远程服务上执行命令
    </h6>
    <p>
     如果需要在远程 redis 服务上执行命令，同样我们使用的也是
     <strong>
      redis-cli
     </strong>
     命令。
    </p>
    <h6>
     <a id="_38">
     </a>
     语法
    </h6>
    <pre><code>$ redis-cli -h host -p port -a password
</code></pre>
    <h6>
     <a id="_43">
     </a>
     实例
    </h6>
    <p>
     以下实例演示了如何连接到主机为 127.0.0.1，端口为 6379 ，密码为 mypass 的 redis 服务上。
    </p>
    <pre><code>$redis-cli -h 127.0.0.1 -p 6379 -a "mypass"
redis 127.0.0.1:6379&gt;
redis 127.0.0.1:6379&gt; PING

PONG
</code></pre>
    <h6>
     <a id="_54">
     </a>
     有时候会有中文乱码。
    </h6>
    <p>
     要在 redis-cli 后面加上 --raw
    </p>
    <pre><code>redis-cli --raw
</code></pre>
    <p>
     就可以避免中文乱码了。
    </p>
