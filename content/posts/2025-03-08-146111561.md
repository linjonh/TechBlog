    <ul>
     <li>
      <p>
       <code>
        npm install eslint -D
       </code>
       安装eslint库
      </p>
     </li>
     <li>
      <p>
       <code>
        npx eslint --init
       </code>
       初始化配置，按项目实际情况选
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/66f3023c5dc749989652caa19e534345.png"/>
      </p>
     </li>
     <li>
      <p>
       自动生成
       <code>
        eslint.config.js
       </code>
       ，可以添加自定义rules
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/37289c604aa94df8b1edc7eadde95cc8.png"/>
      </p>
     </li>
     <li>
      <p>
       安装
       <code>
        ESLint
       </code>
       插件
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/68b6bae21cfc4fedb8ae22bf46e2e836.png">
        <br/>
        此时打开vue文件就会标红有问题的位置
       </img>
      </p>
     </li>
     <li>
      <p>
       安装prettier
       <code>
        npm install prettier eslint-config-prettier eslint-plugin-prettier -D
       </code>
      </p>
     </li>
     <li>
      <p>
       创建
       <code>
        .prettierrc.json
       </code>
      </p>
      <pre><code>{
  "semi": false,
  "singleQuote": false,
  "tabWidth": 2,
  "printWidth": 80,
  "trailingComma": "none"
}
</code></pre>
     </li>
     <li>
      <p>
       安装prettier插件
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a70f210051164c6ebccd098e66a6d9bc.png"/>
      </p>
     </li>
     <li>
      <p>
       打开vscode用户配置
       <code>
        Ctrl + Shift + p
       </code>
       -&gt;
       <code>
        Settings
       </code>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/02857a317b794e3fbf46857820c670e4.png"/>
      </p>
     </li>
     <li>
      <p>
       添加内容：
      </p>
      <pre><code>{
	  "editor.formatOnSave": true,
	  "editor.defaultFormatter": "esbenp.prettier-vscode",
	  "[javascript]": {
	    "editor.defaultFormatter": "esbenp.prettier-vscode"
	  },
	  "[vue]": {
	    "editor.defaultFormatter": "esbenp.prettier-vscode"
	  },
	  "eslint.validate": ["javascript", "vue"],
	  "eslint.run": "onType",
	  "eslint.format.enable": true
}
</code></pre>
     </li>
     <li>
      <p>
       到这里应该就配好了，在vue文件中执行
       <code>
        Ctrl + s
       </code>
       验证保存自动格式化
      </p>
     </li>
     <li>
      <p>

        参考链接
       </a>
      </p>
     </li>
    </ul>
