    <h2>
     JavaScript 的组成
    </h2>
    <p>
    </p>
    <p>
     JavaScript 是一种广泛应用于网页开发的脚本语言，它主要由 ECMAScript、文档对象模型（DOM）和浏览器对象模型（BOM）三部分组成。下面将详细介绍这三个组成部分。
    </p>
    <h3>
     1. ECMAScript
    </h3>
    <h4 style="background-color:transparent">
     定义
    </h4>
    <p>
     ECMAScript 是 JavaScript 的核心，它是由欧洲计算机制造商协会（ECMA）制定的标准，规定了 JavaScript 的基本语法、数据类型、控制结构、对象模型等核心语言特性。可以把它看作是 JavaScript 的基础语法规则，为 JavaScript 程序的编写提供了基本框架。
    </p>
    <h4 style="background-color:transparent">
     发展历程
    </h4>
    <p>
     ECMAScript 不断发展和演进，目前已经有多个版本，如 ES5、ES6（ES2015）、ES7 等。每个版本都引入了一些新的特性和语法糖，以提高开发效率和代码的可读性。例如，ES6 引入了箭头函数、
     <code>
      let
     </code>
     和
     <code>
      const
     </code>
     声明、Promise 对象等。
    </p>
    <h4 style="background-color:transparent">
     核心特性
    </h4>
    <ul>
     <li>
      <strong>
       变量与数据类型
      </strong>
      ：支持多种数据类型，包括基本数据类型（如
      <code>
       Number
      </code>
      、
      <code>
       String
      </code>
      、
      <code>
       Boolean
      </code>
      、
      <code>
       Null
      </code>
      、
      <code>
       Undefined
      </code>
      、
      <code>
       Symbol
      </code>
      ）和引用数据类型（如
      <code>
       Object
      </code>
      ）。可以使用
      <code>
       var
      </code>
      、
      <code>
       let
      </code>
      和
      <code>
       const
      </code>
      来声明变量。
     </li>
    </ul>
    <p>
     javascript
    </p>
    <pre><code>// 使用 let 声明变量
let num = 10;
// 使用 const 声明常量
const str = "Hello, ECMAScript!";
</code></pre>
    <ul>
     <li>
      <strong>
       函数
      </strong>
      ：函数是 ECMAScript 中的一等公民，可以作为参数传递、返回值返回，还可以嵌套定义。支持普通函数和箭头函数两种定义方式。
     </li>
    </ul>
    <p>
     javascript
    </p>
    <pre><code>// 普通函数
function add(a, b) {
    return a + b;
}
// 箭头函数
const subtract = (a, b) =&gt; a - b;
</code></pre>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       控制语句
      </strong>
      ：包括
      <code>
       if...else
      </code>
      、
      <code>
       switch
      </code>
      、
      <code>
       for
      </code>
      、
      <code>
       while
      </code>
      、
      <code>
       do...while
      </code>
      等控制语句，用于控制程序的执行流程。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>for (let i = 0; i &lt; 5; i++) {
    console.log(i);
}
</code></pre>
    <ul>
     <li>
      <strong>
       对象和类
      </strong>
      ：ES6 引入了类的概念，使得面向对象编程在 JavaScript 中更加直观和方便。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    greet() {
        console.log(`Hello, my name is ${this.name}`);
    }
}
const person = new Person("Alice", 25);
person.greet();
</code></pre>
    <h3 style="background-color:transparent">
     2. 文档对象模型（DOM）
    </h3>
    <h4 style="background-color:transparent">
     定义
    </h4>
    <p>
     文档对象模型（DOM）是 HTML 和 XML 文档的编程接口，它将网页文档表示为一个树形结构，树中的每个节点都可以被访问和操作。通过 DOM，JavaScript 可以动态地修改网页的内容、结构和样式。
    </p>
    <h4 style="background-color:transparent">
     常见操作
    </h4>
    <ul>
     <li>
      <strong>
       选择元素
      </strong>
      ：可以使用多种方法选择 HTML 元素，如
      <code>
       getElementById
      </code>
      、
      <code>
       getElementsByClassName
      </code>
      、
      <code>
       getElementsByTagName
      </code>
      、
      <code>
       querySelector
      </code>
      和
      <code>
       querySelectorAll
      </code>
      。
     </li>
    </ul>
    <p>
     html
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;body&gt;
    &lt;div id="myDiv"&gt;This is a div&lt;/div&gt;
    &lt;script&gt;
        const div = document.getElementById('myDiv');
        console.log(div);
    &lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       修改元素内容
      </strong>
      ：使用
      <code>
       innerHTML
      </code>
      、
      <code>
       textContent
      </code>
      等属性来修改元素的内容。
     </li>
    </ul>
    <p>
     javascript
    </p>
    <pre><code>const element = document.querySelector('p');
element.textContent = 'New paragraph text';
</code></pre>
    <ul>
     <li>
      <strong>
       修改元素样式
      </strong>
      ：通过
      <code>
       style
      </code>
      属性来修改元素的 CSS 样式。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>const element = document.getElementById('myDiv');
element.style.color = 'red';
</code></pre>
    <ul>
     <li>
      <strong>
       事件处理
      </strong>
      ：为元素添加事件监听器，响应用户的操作，如
      <code>
       click
      </code>
      、
      <code>
       mouseover
      </code>
      等事件。
     </li>
    </ul>
    <p>
    </p>
    <p>
     html
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;body&gt;
    &lt;button id="myButton"&gt;Click me&lt;/button&gt;
    &lt;script&gt;
        const button = document.getElementById('myButton');
        button.addEventListener('click', function () {
            alert('Button clicked!');
        });
    &lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <h3 style="background-color:transparent">
     3. 浏览器对象模型（BOM）
    </h3>
    <h4>
     定义
    </h4>
    <p>
     浏览器对象模型（BOM）提供了与浏览器窗口进行交互的对象和方法，它允许 JavaScript 控制浏览器窗口的各个方面，如浏览器窗口的大小、位置、历史记录、弹出框等。不过，BOM 并没有标准化，不同浏览器可能会有一些差异。
    </p>
    <h4 style="background-color:transparent">
     常见对象
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        window
       </code>
       对象
      </strong>
      ：是 BOM 的顶层对象，代表浏览器窗口，包含了许多有用的方法和属性。
     </li>
    </ul>
    <p>
     javascript
    </p>
    <pre><code>// 弹出警告框
window.alert('This is an alert');
// 定时执行函数
window.setTimeout(() =&gt; {
    console.log('This message is shown after 2 seconds');
}, 2000);
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        location
       </code>
       对象
      </strong>
      ：用于获取或设置当前页面的 URL 信息。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>// 获取当前页面的 URL
console.log(window.location.href);
// 跳转到新页面
window.location.href = 'https://www.example.com';
</code></pre>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        history
       </code>
       对象
      </strong>
      ：允许操作浏览器的历史记录。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>// 后退一页
window.history.back();
// 前进一页
window.history.forward();
</code></pre>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        navigator
       </code>
       对象
      </strong>
      ：包含有关浏览器的信息，如浏览器的名称、版本等。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>console.log(navigator.userAgent); // 获取浏览器的用户代理信息
</code></pre>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       <code>
        screen
       </code>
       对象
      </strong>
      ：包含有关用户屏幕的信息，如屏幕的宽度、高度等。
     </li>
    </ul>
    <p>
    </p>
    <p>
     javascript
    </p>
    <pre><code>console.log(screen.width); // 获取屏幕的宽度
</code></pre>
    <p>
     综上所述，ECMAScript 为 JavaScript 提供了基本的语法和编程结构，DOM 用于操作网页文档，BOM 用于与浏览器窗口进行交互，三者共同构成了完整的 JavaScript 体系，使得开发者可以创建出交互性强、动态性高的网页应用。
    </p>

    <h2>
     JavaScript的使用
    </h2>
    <h3 style="background-color:transparent">
     1、JavaScript介绍
    </h3>
    <p>
     JavaScript是运行在浏览器端的脚步语言，JavaScript主要解决的是前端与用户交互的问题，包括使用交互与数据交互。JavaScript是浏览器解释执行的，前端脚本语言还有JScript（微软，IE独有），ActionScript(Adobe公司，需要插件）等。
    </p>
    <h4>
     前端三大块：
    </h4>
    <ol>
     <li>
      HTML：页面结构
     </li>
     <li>
      CSS:页面表现：元素大小、颜色、位置、隐藏或显示，部分动画效果
     </li>
     <li>
      JavaScript：页面行为：部分动画效果、页面与用户的交互、页面功能
     </li>
    </ol>
    <h3>
     2.JavaScript嵌入页面的方式
    </h3>
    <p>
     1.行间事件（主要用于事件）
    </p>
    <pre><code class="language-html">&lt;input type="button" name="" onclick="alert('ok！');"&gt;</code></pre>
    <p>
     2.页面script标签嵌入
    </p>
    <pre><code class="language-html">&lt;script type="text/javascript"&gt;        
    alert('ok！');
&lt;/script&gt;</code></pre>
    <p>
     3.外部引入
    </p>
    <pre><code class="language-html">&lt;script type="text/javascript" src="js/index.js"&gt;&lt;/script&gt;</code></pre>
    <h3>
     3.变量
    </h3>
    <p>
     JavaScript是一种弱类型语言，javascript的变量类型由它的值来决定。定义变量需要用关键词'var'
    </p>
    <pre><code class="language-javascript"> var iNum = 123;
 var sTr = 'asd';

 //同时定义多个变量可以用","隔开，公用一个‘var’关键字

 var iNum = 45,sTr='qwe',sCount='68';</code></pre>
    <h4>
     3.1 变量类型
    </h4>
    <p>
     <strong>
      5种基本数据类型：
     </strong>
    </p>
    <p>
     1、number 数字类型
    </p>
    <p>
     2、string 字符串类型
    </p>
    <p>
     3、boolean 布尔类型 true 或false
    </p>
    <p>
     4、undefined undefined 类型，变量声明未初始化，它的值就是undefined
    </p>
    <p>
     5、null 类型，表示空对象，如果定义的变量将来准备保存对象，可以将变量初始化为null，在页面上获取不到对象，返回的值就是null
    </p>
    <p>
     <strong>
      1钟复合类型：
     </strong>
    </p>
    <p>
     object
    </p>
    <p>
    </p>
    <h4>
     <strong>
      3.2 变量、函数、属性、函数参数命名规范
     </strong>
    </h4>
    <p>
     1、区分大小写
     <br/>
     2、第一个字符必须是字母、下划线（_）或者美元符号（$）
     <br/>
     3、其他字符可以是字母、下划线、美元符或数字
    </p>
    <p>
    </p>
    <p>
     <strong>
      匈牙利命名风格：
     </strong>
    </p>
    <p>
     对象o Object 比如：oDiv
     <br/>
     数组a Array 比如：aItems
     <br/>
     字符串s String 比如：sUserName
     <br/>
     整数i Integer 比如：iItemCount
     <br/>
     布尔值b Boolean 比如：bIsComplete
     <br/>
     浮点数f Float 比如：fPrice
     <br/>
     函数fn Function 比如：fnHandler
     <br/>
     正则表达式re RegExp 比如：reEmailCheck
    </p>
    <h3 style="background-color:transparent">
     4.获取元素方法一(通过id属性）
    </h3>
    <p>
     可以使用内置对象document上的getElementById方法来获取页面上设置了id属性的元素，获取到的是一个html对象，然后将它赋值给一个变量，比如：
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script type="text/javascript"&gt;
			// 通过ID名获取元素赋值给oDiv变量
			var oDiv = document.getElementById('div1');

			// 改变元素的属性
			oDiv.style.color = "red";
			oDiv.style.fontSize = "20px";


		&lt;/script&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="div1"&gt;这是一个元素&lt;/div&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     上面的语句，如果把javascript写在元素的上面，就会出错，因为页面上从上往下加载执行的，javascript去页面上获取元素div1的时候，元素div1还没有加载，解决方法有两种：
    </p>
    <p>
     第一种方法：将javascript放到页面最下边
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="div1"&gt;这是一个元素&lt;/div&gt;
		&lt;script type="text/javascript"&gt;
			// 通过ID名获取元素赋值给oDiv变量
			var oDiv = document.getElementById('div1');
		
			// 改变元素的属性
			oDiv.style.color = "red";
			oDiv.style.fontSize = "20px";
		&lt;/script&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     第二种方法：将javascript语句放到window.onload触发的函数里面，获取元素的语句会在页面加载完后才执行，就不会出错了。
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script type="text/javascript"&gt;
			//当整个页面加载完之后再执行花括号里面的语句
					window.onload = function(){
			
						// 通过ID名获取元素赋值给oDiv变量
						var oDiv = document.getElementById('div1');
			
						// 改变元素的属性
						oDiv.style.color = "red";
						oDiv.style.fontSize = "20px";
			
					}
		&lt;/script&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="div1"&gt;这是一个元素&lt;/div&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <h3>
     5.操作元素属性
    </h3>
    <p>
     获取的页面元素，就可以对页面元素的属性进行操作，属性的操作包括属性的读和写
    </p>
    <h4>
     5.1 操作方法
    </h4>
    <p>
     1、“.”操作
    </p>
    <p>
     2、“[ ]”操作
    </p>
    <h4>
     5.2 属性写法
    </h4>
    <p>
     1、html的属性和js里面属性写法一样
    </p>
    <p>
     2、“class”属性写成“className”
    </p>
    <p>
     3、“style”属性里面的属性，有横杠的改成驼峰式，比如：“font-size”，改成“style.fontSize”
    </p>
    <p>
     通过“.”操作属性：
    </p>
    <h4>
     5.3 通过 . 操作属性
    </h4>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script type="text/javascript"&gt;
			window.onload=function(){
				var oDiv=document.getElementById('div1')
				var oA=document.getElementById('link')
				var oDiov2=document.getElementById('div2')
				
				// 读取属性
				var sId=oDiv.id;
				alert(sId);
				
				// 写属性
				oDiv.style.color="yellow";
				oA.href="http://www.baidu.com";
				oA.title="这是去到百度往的链接";
				
				// 操作class属性需要写成className
				oDiov2.className="box2"
			}
		&lt;/script&gt;
		
		&lt;style type="text/css"&gt;
			.box{
				font-size: 20px;
				color: aquamarine;
			}
			.box2{
				font-size:30px;
				color: beige;
			}
		&lt;/style&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="div1"&gt;这是一个div元素&lt;/div&gt;

     <strong>

     </strong>
    </h4>
    <p>
     1、获取数组的长度：aList.length;
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];

    <p>
     2、用下标操作数组的某个数据：aList[0];
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];

    <p>
     3、join() 将数组成员通过一个分隔符合并成字符串
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];
alert(aList.join('-')); // 弹出 1-2-3-4</code></pre>
    <p>
     4、push() 和 pop() 从数组最后增加成员或删除成员
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];
aList.push(5);
alert(aList); //弹出1,2,3,4,5
aList.pop();
alert(aList); // 弹出1,2,3,4</code></pre>
    <p>
     5、unshift()和 shift() 从数组前面增加成员或删除成员
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];
aList.unshift(5);
alert(aList); //弹出5,1,2,3,4
aList.shift();
alert(aList); // 弹出1,2,3,4</code></pre>
    <p>
     6、reverse() 将数组反转
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];
aList.reverse();
alert(aList);  // 弹出4,3,2,1</code></pre>
    <p>
     7、indexOf() 返回数组中元素
     <strong>
      第一次
     </strong>
     出现的索引值

    <pre><code class="language-javascript">var aList = [1,2,3,4,1,3,4];

    <p>
     8、splice() 在数组中增加或删除成员
    </p>
    <pre><code class="language-javascript">var aList = [1,2,3,4];

    <h4 id="a9a12983">
     11.2. 定时器的使用：
    </h4>
    <p id="udb0b4606">
     <code>
      js
     </code>
     定时器有两种创建方式：
    </p>
    <ol>
     <li id="u8d31bb50">
      <code>
       setTimeout(func[, delay, param1, param2, ...])
      </code>
      ：以指定的时间间隔（以毫秒计）调用一次函数的定时器
     </li>
     <li id="u20f1454e">
      <code>
       setInterval(func[, delay, param1, param2, ...])
      </code>
      ：以指定的时间间隔（以毫秒计）重复调用一个函数的定时器
     </li>
    </ol>
    <p id="ufc172e40">
     <code>
      <strong>
       setTimeout
      </strong>
     </code>
     <strong>
      函数的参数说明:
     </strong>
    </p>
    <ul>
     <li id="uebcaccbe">
      第一个参数
      <code>
       func
      </code>
      , 表示定时器要执行的函数名
     </li>
     <li id="u139f3514">
      第二个参数 delay, 表示时间间隔，默认是0，单位是毫秒
     </li>
     <li id="u729f8f1e">
      第三个参数
      <code>
       param1
      </code>
      , 表示定时器执行函数的第一个参数，一次类推传入多个执行函数对应的参数。
     </li>
    </ul>
    <pre><code class="language-javascript">function hello() {
      console.log('hello');
  }
  
  // 执行一次函数的定时器
  setTimeout(hello, 500);</code></pre>
    <p id="u8d9c33d9">
     <code>
      <strong>
       setInterval
      </strong>
     </code>
     <strong>
      函数的参数说明:
     </strong>
    </p>
    <ul>
     <li id="ucabce376">
      第一个参数
      <code>
       func
      </code>
      , 表示定时器要执行的函数名
     </li>
     <li id="u81a9c585">
      第二个参数 delay, 表示时间间隔，默认是0，单位是毫秒
     </li>
     <li id="u8bdfaf0f">
      第三个参数
      <code>
       param1
      </code>
      , 表示定时器执行函数的第一个参数，一次类推传入多个执行函数对应的参数。
     </li>
    </ul>
    <pre><code class="language-javascript">function hello() {
      console.log('hello');
  }
  
  // 重复执行函数的定时器
  setInterval(hello, 1000);</code></pre>
