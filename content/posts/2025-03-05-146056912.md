      3.我们看一下上面代码里黄色背景的部分，就是写入DataTable的部分，其实简单的判断一下，如果是数字类型又能格式化成时间的情况，就按照时间格式读取就行了，如下图：
     </p>
     <div class="cnblogs_code">
      <pre><code class="has">for (int j = row.FirstCellNum; j &lt; cellCount; ++j)
{
    ICell cell = row.GetCell(j);
    if (cell != null)
    {
        if (cell.CellType == CellType.Numeric &amp;&amp; DateUtil.IsCellDateFormatted(cell))
            dataRow[j] = cell.DateCellValue.ToString();
        else
            dataRow[j] = cell.ToString();
    }
}</code>
