    <p>
     第一步：打开网址，https://www.mysql.com，点击downloads之后跳转到https://www.mysql.com/downloads
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/921ed39e95b9e9208efb70fd47d6ea5c.png#pic_center"/>
    </p>
    <p>
     第二步 ：跳转至网址https://dev.mysql.com/downloads/，选择Community选项
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6c07eec87759517b9f66b16ad41e9f77.png#pic_center"/>
    </p>
    <p>
     第三步 ：点击MySQL Community Server进入https://dev.mysql.com/downloads/mysql/页面，再点击5.6版本的数据库
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/96a98570c98440c5d14e66c291a00825.png#pic_center"/>
    </p>
    <p>
     第四步：windows操作系统 点击5.6版本之后会跳转到https://dev.mysql.com/downloads/mysql/5.6.html#downloads 网址，页面如下，确认好要下载的版本和操作系统，点击Download
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6e63ac014a4cbf3376eee59904d63c9a.png#pic_center"/>
    </p>
    <p>
     第五步：可以不用登陆或者注册，直接点击No thanks,just start my download就可以下载了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/46658a8101f78b484de71360cc34d5da.png#pic_center"/>
    </p>
    <p>
     解压
     <br/>
     下载的zip文件解压，将解压之后的文件夹放到任意目录下，这个目录就是mysql的安装目录。
    </p>
    <p>
     配置
     <br/>
     打开目录，会看到my-default.ini配置文件，复制这个配置文件可以重命名为my.ini或者my.cnf
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/367f004cfd0287cac354361b15bd8990.png#pic_center"/>
    </p>
    <pre><code class="prism language-python"><span class="token punctuation">[</span>mysql<span class="token punctuation">]</span>
<span class="token comment"># 设置mysql客户端默认字符集</span>
default<span class="token operator">-</span>character<span class="token operator">-</span><span class="token builtin">set</span><span class="token operator">=</span>utf8 
<span class="token punctuation">[</span>mysqld<span class="token punctuation">]</span>
<span class="token comment">#设置3306端口</span>
port <span class="token operator">=</span> <span class="token number">3306</span> 
<span class="token comment"># 设置mysql的安装目录</span>
basedir<span class="token operator">=</span>C<span class="token punctuation">:</span>\Program Files\mysql<span class="token operator">-</span><span class="token number">5.6</span><span class="token number">.39</span><span class="token operator">-</span>winx64 
<span class="token comment"># 设置mysql数据库的数据的存放目录</span>
datadir<span class="token operator">=</span>C<span class="token punctuation">:</span>\Program Files\mysql<span class="token operator">-</span><span class="token number">5.6</span><span class="token number">.39</span><span class="token operator">-</span>winx64\data 
<span class="token comment"># 允许最大连接数</span>
max_connections<span class="token operator">=</span><span class="token number">200</span>
<span class="token comment"># 服务端使用的字符集默认为8比特编码的latin1字符集</span>
character<span class="token operator">-</span><span class="token builtin">set</span><span class="token operator">-</span>server<span class="token operator">=</span>utf8
<span class="token comment"># 创建新表时将使用的默认存储引擎</span>
default<span class="token operator">-</span>storage<span class="token operator">-</span>engine<span class="token operator">=</span>INNODB

my<span class="token punctuation">.</span>ini 参考配置
</code></pre>
<p>
环境变量
<br/>
在系统变量 PATH 后面添加: 你的 mysql bin 文件夹的路径（如 C:\Program Files\mysql-5.6.41-winx64\bin）
</p>
<p>
安装 MySQL 服务
<br/>
以管理员身份打开 cmd 窗口后，将目录切换到你解压文件的 bin 目录，输入 mysqld install 回车运行
</p>
<p>
启动 mysql 服务
<br/>
以管理员身份在 cmd 中输入:net start mysql
</p>
<p>
服务启动成功之后，就可以登录了，输入 mysql -u root -p（第一次登录没有密码，直接按回车过）
</p>
<pre><code class="prism language-python">net stop mysql <span class="token comment"># 停止 mysql</span>

<span class="token comment">#在 windows 操作系统上没有重启 mysql 服务的命令</span>
<span class="token comment">#如果要重启服务，只能先 stop 再 start</span>
</code></pre>
