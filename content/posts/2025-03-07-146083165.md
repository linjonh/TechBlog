    <h2>
     <a id="ffplay_0">
     </a>
     一、ffplay简单过滤器
    </h2>
    <ul>
     <li>
      视频旋转：借助transpose滤镜
     </li>
    </ul>
    <pre><code class="prism language-shell">ffplay <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-vf</span> <span class="token assign-left variable">transpose</span><span class="token operator">=</span><span class="token number">1</span>
</code></pre>
    <p>
     这里选择不同的数字是不同的方向：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/50994ab43a2942dca93870f8983657db.gif"/>
    </p>
    <ul>
     <li>
      视频翻转：借助hflip/vflip实现水平和垂直翻转：
     </li>
    </ul>
    <p>
     水平翻转
    </p>
    <pre><code class="prism language-shell">ffplay <span class="token number">1</span>.mp4 <span class="token parameter variable">-vf</span> hflip
</code></pre>
    <p>
     垂直翻转
    </p>
    <pre><code class="prism language-shell">ffplay <span class="token number">1</span>.mp4 <span class="token parameter variable">-vf</span> vflip
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e8c50f2de03548688d76e6c692712356.gif"/>
    </p>
    <ul>
     <li>
      音频变速播放：借助atempo滤镜，设置1表示原速度，设置2为2倍数
      <br/>
      下面语句是设置播放为二倍数
     </li>
    </ul>
    <pre><code class="prism language-shell">ffplay <span class="token number">1</span>.mp4 <span class="token parameter variable">-af</span> <span class="token assign-left variable">atempo</span><span class="token operator">=</span><span class="token number">2</span>
</code></pre>
    <ul>
     <li>
      视频变速播放：借助setpts滤镜，setpts=PTS/2，表示原来的pts变为现在的1/2，即2倍数播放
     </li>
    </ul>
    <pre><code class="prism language-shell">ffplay <span class="token number">1</span>.mp4 <span class="token parameter variable">-vf</span> <span class="token assign-left variable">setpts</span><span class="token operator">=</span>PTS/2
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/366220d44e5648cfad9dca0ee8916373.gif"/>
    </p>
    <ul>
     <li>
      音视频同时变速：使用atempo+setpts即可
     </li>
    </ul>
    <pre><code class="prism language-shell">ffplay mv.mp4 <span class="token parameter variable">-af</span> <span class="token assign-left variable">atempo</span><span class="token operator">=</span><span class="token number">2</span> <span class="token parameter variable">-vf</span> <span class="token assign-left variable">setpts</span><span class="token operator">=</span>PTS/2
</code></pre>
