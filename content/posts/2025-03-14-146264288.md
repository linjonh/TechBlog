    <h2>
     <a id="CropperJS_4">
     </a>
     一、CropperJS
    </h2>
    <p>
     <code>
      cropperjs
     </code>
     是一个轻量级、功能强大的
     <strong>
      图片裁剪插件
     </strong>
     ，专为前端开发者设计。它支持图片的裁剪、旋转、缩放等操作，适用于头像上传、图片编辑等场景，具有响应式、触摸屏友好和丰富的 API 配置。
    </p>
    <hr/>
    <h4>
     <a id="_10">
     </a>
     <strong>
      核心特性
     </strong>
    </h4>
    <p>
     •
     <strong>
      裁剪功能
     </strong>
     ：支持自由比例、固定比例或固定尺寸裁剪。
     <br/>
     •
     <strong>
      交互友好
     </strong>
     ：拖拽、缩放、旋转（支持触摸屏设备）。
     <br/>
     •
     <strong>
      响应式
     </strong>
     ：自动适应容器尺寸变化。
     <br/>
     •
     <strong>
      事件回调
     </strong>
     ：提供裁剪区域坐标变化、图片加载等事件。
     <br/>
     •
     <strong>
      API 丰富
     </strong>
     ：可通过代码动态调整裁剪区域、导出结果等。
    </p>

    <h4>
     <a id="_73">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <p>
     • 用户头像上传（如社交平台）。
     <br/>
     • 电商商品图片编辑（裁剪主图、细节图）。
     <br/>
     • 在线设计工具中的图片处理模块。
    </p>
    <hr/>
    <h4>
     <a id="_80">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <p>
     •
     <strong>
      依赖项
     </strong>
     ：纯 JavaScript 实现，无 jQuery 依赖。
     <br/>
     •
     <strong>
      兼容性
     </strong>
     ：支持现代浏览器（Chrome、Firefox、Safari、Edge）。
     <br/>
     •
     <strong>
      性能
     </strong>
     ：处理大图时建议先压缩再裁剪，避免卡顿。
    </p>
    <p>
     如果需要更高级的功能（如滤镜、贴图），可以结合其他库（如
     <code>
      fabric.js
     </code>
     或
     <code>
      konva.js
     </code>
     ）使用。
    </p>
