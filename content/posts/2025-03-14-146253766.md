# 使用vite脚手架搭建react项目

> 首选使用npm安装react vite
>
> **npm install -g create vite**
> 替换npm install -g create-react-app
>
> 新的脚手架据说很快，是原来的webpack的700倍。。。

![](https://i-blog.csdnimg.cn/direct/9b00f145927b43419f96e8a14d951b15.png)

> 安装完vite后，使用
> **npm create vite**
> my-react-app
>
> (my-react-app 这是我自己的项目名，你自己可以随便取名字)
>
> 先会问你，确定继续吗？继续你就y一下。
>
> 然后回自动提示一些选择项，例如framework你要用哪个？variant 用哪个？ 你舅妈慢慢根据自己情况来选择就行了。都选好了，就会告诉你DONE了，可以开始进入项目启动了。
>
> ![](https://i-blog.csdnimg.cn/direct/448406b47cf241618a547451644c7bcb.png)

> 下面就是进入你创建的项目，我的是my-react-app，然后npm install  ，install结束后，就可以启动项目了。npm run dev
