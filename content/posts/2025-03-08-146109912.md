    <h2>
     《WebForms 实例》
    </h2>
    <h3>
     引言
    </h3>
    <p>
     WebForms 是微软推出的一种用于构建动态Web应用程序的技术。它基于ASP.NET框架，允许开发者使用服务器端控件来构建用户界面，并通过事件驱动模型来响应用户交互。本文将通过一些实例，详细介绍WebForms的使用方法，帮助开发者更好地理解和应用这一技术。
    </p>
    <h3>
     实例一：简单的登录表单
    </h3>
    <p>
     以下是一个简单的登录表单实例，用于验证用户名和密码。
    </p>
    <pre><code class="language-html">&lt;%@ Page Language="C#" AutoEventWireup="true" CodeBehind="Login.aspx.cs" Inherits="WebApplication1.Login" %&gt;
&lt;!DOCTYPE html&gt;
&lt;html xmlns="http://www.w3.org/1999/xhtml"&gt;
&lt;head runat="server"&gt;
    &lt;title&gt;登录表单&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;form id="form1" runat="server"&gt;
        &lt;div&gt;
            &lt;asp:Label ID="Label1" runat="server" Text="用户名:"&gt;&lt;/asp:Label&gt;
            &lt;asp:TextBox ID="TextBox1" runat="server"&gt;&lt;/asp:TextBox&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;asp:Label ID="Label2" runat="server" Text="密码:"&gt;&lt;/asp:Label&gt;
            &lt;asp:TextBox ID="TextBox2" runat="server" TextMode="Password"&gt;&lt;/asp:TextBox&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;asp:Button ID="Button1" runat="server" Text="登录" OnClick="Button1_Click" /&gt;
        &lt;/div&gt;
    &lt;/form&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <pre><code class="language-csharp">protected void Button1_Click(object sender, EventArgs e)
{
    if (TextBox1.Text == "admin" &amp;&amp; TextBox2.Text == "admin")
    {
        Response.Redirect("Welcome.aspx");
    }
    else
    {
        Label3.Text = "用户名或密码错误！";
    }
}
</code></pre>
    <p>
     在这个例子中，我们使用
     <code>
      asp:Label
     </code>
     控件来显示提示信息，使用
     <code>
      asp:TextBox
     </code>
     控件来获取用户输入的用户名和密码。当用户点击登录按钮时，会触发
     <code>
      Button1_Click
     </code>
     事件处理程序，检查用户名和密码是否正确。
    </p>

    </p>
    <h3>
     实例三：动态表单
    </h3>
    <p>
     以下是一个动态表单的实例，根据用户选择的表单类型，显示不同的字段。
    </p>
    <pre><code class="language-html">&lt;%@ Page Language="C#" AutoEventWireup="true" CodeBehind="DynamicForm.aspx.cs" Inherits="WebApplication1.DynamicForm" %&gt;
&lt;!DOCTYPE html&gt;
&lt;html xmlns="http://www.w3.org/1999/xhtml"&gt;
&lt;head runat="server"&gt;
    &lt;title&gt;动态表单&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;form id="form1" runat="server"&gt;
        &lt;div&gt;
            &lt;asp:Label ID="Label1" runat="server" Text="选择表单类型:"&gt;&lt;/asp:Label&gt;
            &lt;asp:DropDownList ID="DropDownList1" runat="server" AutoPostBack="true" OnSelectedIndexChanged="DropDownList1_SelectedIndexChanged"&gt;
                &lt;asp:ListItem Value="个人资料"&gt;个人资料&lt;/asp:ListItem&gt;
                &lt;asp:ListItem Value="工作经历"&gt;工作经历&lt;/asp:ListItem&gt;
            &lt;/asp:DropDownList&gt;
        &lt;/div&gt;
        &lt;div id="PersonalInfo"&gt;
            &lt;asp:Label ID="Label2" runat="server" Text="姓名:"&gt;&lt;/asp:Label&gt;
            &lt;asp:TextBox ID="TextBox1" runat="server"&gt;&lt;/asp:TextBox&gt;
        &lt;/div&gt;
        &lt;div id="WorkExperience" style="display:none;"&gt;
            &lt;asp:Label ID="Label3" runat="server" Text="公司名称:"&gt;&lt;/asp:Label&gt;
            &lt;asp:TextBox ID="TextBox2" runat="server"&gt;&lt;/asp:TextBox&gt;
            &lt;asp:Label ID="Label4" runat="server" Text="职位:"&gt;&lt;/asp:Label&gt;
            &lt;asp:TextBox ID="TextBox3" runat="server"&gt;&lt;/asp:TextBox&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;asp:Button ID="Button1" runat="server" Text="提交" OnClick="Button1_Click" /&gt;
        &lt;/div&gt;
    &lt;/form&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <pre><code class="language-csharp">protected void DropDownList1_SelectedIndexChanged(object sender, EventArgs e)
{
    if (DropDownList1.SelectedItem.Value == "个人资料")
    {
        Panel1.Controls.Add(PersonalInfo);
        Panel1.Controls.Remove(WorkExperience);
    }
    else
    {
        Panel1.Controls.Add(WorkExperience);
        Panel1.Controls.Remove(PersonalInfo);
    }
}
</code></pre>
    <p>
     在这个例子中，我们使用
     <code>
      asp:DropDownList
     </code>
     控件来让用户选择表单类型。根据用户的选择，我们通过JavaScript和CSS控制不同表单字段的显示和隐藏。
    </p>
    <h3>
     总结
    </h3>
    <p>
     以上是三个简单的WebForms实例，通过这些实例，我们可以看到WebForms的强大功能和易用性。在实际开发中，我们可以根据需求，使用WebForms构建各种类型的Web应用程序。希望本文能帮助您更好地理解和应用WebForms技术。
    </p>
