    <h2>
     <a id="_0">
     </a>
     原因
    </h2>
    <p>
     由于官方只提供了docker部署，基于开源代码需要实现自己内部得逻辑，所以需要本地pycharm能访问，且docker运行依赖得其余组件，均需要使用开发服务器得配置。
    </p>
    <h2>
     <a id="_3">
     </a>
     修改过程
    </h2>
    <ol>
     <li>
      安装python
      <br/>
      项目依赖于Python 版本：&gt;=3.10, &lt; 3.13
     </li>
     <li>
      包管理
      <br/>
      项目使用Poetry来管理包，所以需要先初始化环境后初始化项目
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token comment"># 确认当前目录为ragflow当层目录</span>
<span class="token comment"># 创建虚拟环境</span>
virtualenv hello
<span class="token comment"># 激活虚拟环境</span>
.<span class="token punctuation">\</span>hello<span class="token punctuation">\</span>Scripts<span class="token punctuation">\</span>activate
<span class="token comment"># 安装poetry</span>
pip <span class="token function">install</span> poetry
<span class="token comment"># 验证安装是否成功</span>
poetry <span class="token parameter variable">--version</span>
<span class="token comment"># 初始化项目</span>
poetry <span class="token function">install</span>
</code></pre>
    <p>
     <strong>
      注1
     </strong>
     ：如果初始化过慢则需要在pyproject.toml文件中新增国内镜像
    </p>
    <pre><code class="prism language-txt">
[[tool.poetry.source]]
name = "mirrors"
url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
priority = "primary"
</code></pre>
    <p>
     <strong>
      注2
     </strong>
     ：会存在无法安装pyicu得问题，本地运行若只需要聊天/知识库可以先忽略
    </p>
    <ol start="3">
     <li>
      下载项目必备模型
      <br/>
      找到目录下得download_depts.py文件，开启加速器运行，当前目录下会下载这些文件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f2ee0dc0c1ef4855bd108613a2c46e5a.png">

       注：如果存在虚拟环境则需要把nltk_data文件夹移到虚拟环境文件目录下
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/837d0f0e50b246969b0e7a555f453043.png"/>
      </img>
     </li>
     <li>
