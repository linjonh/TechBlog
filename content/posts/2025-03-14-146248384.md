    <h2>
     <a id="flutter__0">
     </a>
     flutter 简介
    </h2>
    <p>
     Flutter是谷歌的移动UI框架，可以快速在iOS和Android上构建高质量的原生用户界面。
    </p>
    <ol>
     <li>
      具有跨平台开发特性，支持IOS、Android、Web三端。
     </li>
     <li>
      支持热重载特性，大大提高了开发效率。
     </li>
     <li>
      自绘UI引擎和编译成原生代码的方式，使得系统的运行时的高性能成为了可能。 使用Dart语言，目前已经支持同时编译成Web端代码。
     </li>
    </ol>
    <h2>
     <a id="_flutter_6">
     </a>
     安装 flutter
    </h2>
    <h4>
     <a id="1_sdk__7">
     </a>
     1. sdk 安装
    </h4>
    <p>
     首先，确保您的系统满足以下要求：
    </p>
    <ul>
     <li>
      Windows 10 或更高版本
     </li>
     <li>
      至少 4GB 内存
     </li>
     <li>
      磁盘空间：1.64 GB（不包括 IDE/tools）
     </li>
    </ul>
    <p>
     进入官方下载站：
     <a href="https://docs.flutter.dev/get-started/install/windows/mobile" rel="nofollow">
      下载地址
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6c03308a2ea940e4b84e8a31d462d7e8.png">
      <br/>
      下载后解压到一个文件夹下就行，例如我的在
      <code>
       e:\dev\flutter\bin
      </code>
      下面
      <br/>
      也可以一句命令行傻瓜式安装
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 下载并解压 Flutter SDK</span>
<span class="token function">mkdir</span> e:<span class="token punctuation">\</span>dev<span class="token punctuation">\</span>flutter<span class="token punctuation">\</span>bin
<span class="token builtin class-name">cd</span> e:<span class="token punctuation">\</span>dev<span class="token punctuation">\</span>flutter<span class="token punctuation">\</span>bin
<span class="token function">curl</span> <span class="token parameter variable">-o</span> flutter_windows_3.19.0-stable.zip https://storage.googleapis.com/flutter_infra_release/releases/stable/windows/flutter_windows_3.19.0-stable.zip
<span class="token function">tar</span> <span class="token parameter variable">-xf</span> flutter_windows_3.19.0-stable.zip
</code></pre>
    <p>
     然后添加环境目录，将 sdk 的bin目录地址添加 PATH 环境变量当中。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1036b06f98ba4aa6b4a53cb1efd16dcb.png#pic_center"/>
    </p>
    <h4>
     <a id="2___Android_Studio_29">
     </a>
     2. 安装 Android Studio
    </h4>
    <p>
     确认系统配置
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/83a05a794a544ccaba5b50196a25e723.png">
      <br/>
      找到安装包下载地址
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d71fbd26f179421e83878d3125d2b412.png">
       <br/>
       安装时选择 “Standard” 安装类型，安装完成后，打开 Android Studio 并安装 Android SDK，进入 settings ，在 SDK maneger 里安装 Android SDK。
      </img>
     </img>
    </p>
    <h4>
     <a id="_vscode_36">
     </a>
     安装 vscode
    </h4>
    <p>
     （可选但推荐）：
    </p>
    <ul>
     <li>
      下载并安装 VS Code
     </li>
     <li>
      安装 Flutter 和 Dart 扩展
     </li>
    </ul>
    <h4>
     <a id="_40">
     </a>
     验证安装
    </h4>
    <p>
     <code>
      WIN+R
     </code>
     进入 CMD，输入
     <code>
      flutter doctor
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/298b2a77276645d5949fb96a385845de.png">
      <br/>
      有打 x 的项就是需要配置的项。我这边显示 Chrome 和 Visual Studio 需要配置，我主要是开发 Android app，所以不需要配置，有需要配置的可以根据提示配置即可。
      <br/>
      network 问题可以参考这篇文章：
      <br/>
      <a href="https://blog.csdn.net/PxFuture/article/details/134112244">
       network 问题
      </a>
     </img>
    </p>
    <h4>
     <a id="_46">
     </a>
     创建项目
    </h4>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> d:<span class="token punctuation">\</span>Dev<span class="token punctuation">\</span>flutter_study
flutter create my_first_app
<span class="token builtin class-name">cd</span> my_first_app
flutter run
</code></pre>
