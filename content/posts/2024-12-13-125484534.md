    <h3>
     <a id="_Chrome__2">
     </a>
     下载 Chrome 扩展程序安装包
    </h3>
    <h4>
     <a id="1_Crx4Chrome_4">
     </a>
     <strong>
      1. Crx4Chrome
     </strong>
    </h4>
    <p>
     地址：
     <a href="https://link.zhihu.com/?target=https%3A//www.crx4chrome.com/" rel="nofollow">
      https://www.crx4chrome.com/
     </a>
    </p>
    <p>
     该网站相当于一个 Chrome 扩展资料库，里面收集了几乎全部 Chrome 扩展，并且做了分类和介绍，还有历史版本可供下载。
    </p>
    <p>
     进入主页直接搜索你要下载的插件，然后点击下载。
    </p>
    <h4>
     <a id="2_Chrome_Extension_Downloader_12">
     </a>
     <strong>
      2. Chrome Extension Downloader
     </strong>
    </h4>
    <p>
     地址：
     <a href="https://link.zhihu.com/?target=https%3A//chrome-extension-downloader.com/" rel="nofollow">
      https://chrome-extension-downloader.com/
     </a>
    </p>
    <p>
     该网站适合已经有了扩展程序的链接地址的，将地址输进去，点击 Download extension 下载到本地
    </p>
    <h4>
     <a id="3_Crx_18">
     </a>
     <strong>
      3. Crx离线安装包下载
     </strong>
    </h4>
    <p>
     地址：
     <a href="https://link.zhihu.com/?target=http%3A//yurl.sinaapp.com/crx.php" rel="nofollow">
      http://yurl.sinaapp.com/crx.php
     </a>
    </p>
    <p>
     和上面的类似，不过是输入扩展程序的 ID 进行下载。
    </p>
    <p>
     **示例：**扩展链接
     <a href="https://link.zhihu.com/?target=https%3A//chrome.google.com/webstore/detail/google-translate/" rel="nofollow">
      https://chrome.google.com/webstore/detail/google-translate/
     </a>
     <strong>
      aapbdbdomjkkjkaonfhkkikfgjllcleb
     </strong>
    </p>
    <p>
     末尾这段即扩展 ID，将其复制进输入框，点击生成，右键红字部分链接另存为。
    </p>
    <h4>
     <a id="4_Google_Chrome__28">
     </a>
     <strong>
      4. Google Chrome 插件下载
     </strong>
    </h4>
    <p>
     地址：
     <a href="https://link.zhihu.com/?target=https%3A//www.gugeapps.net/" rel="nofollow">
      https://www.gugeapps.net/
     </a>
    </p>
    <p>
     这是一个扩展商店的镜像网站，页面和官方商店很像。
    </p>
    <h4>
     <a id="5_Chrome__34">
     </a>
     <strong>
      5. Chrome 插件网
     </strong>
    </h4>
    <p>
     地址：
     <a href="https://link.zhihu.com/?target=http%3A//www.cnplugins.com/" rel="nofollow">
      http://www.cnplugins.com/
     </a>
    </p>
    <p>
     除了扩展下载，还有优秀扩展推荐和使用技巧分享。（下载需要获取验证码）
    </p>
    <h3>
     <a id="_Chrome__40">
     </a>
     离线安装 Chrome 扩展程序
    </h3>
    <p>
     下载好 CRX 文件，将后缀名
     <code>
      .crx
     </code>
     修改为
     <code>
      .zip
     </code>
     或
     <code>
      .rar
     </code>
     ，然后解压到一个文件夹中。
    </p>
    <p>
     进入 Chrome 扩展程序管理页面（选项 - 更多工具 - 扩展程序）。打开 “开发者模式” -&gt; 加载已解压的扩展程序 -&gt; 选择刚才解压的文件夹，完成。
    </p>
    <h3>
     <a id="end_48">
     </a>
     end
    </h3>
    <p>
     最好还是通过 Chrome 应用商店直接在线安装。无法在线安装的通过离线安装倒也方便。
    </p>
    <p>
     使用老版本 Chrome 的用户还可以试下直接将 crx 文件拖进去直接安装，新版本已经不支持这一方式了。
    </p>
