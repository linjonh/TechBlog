    <div>
     <span style="font-size:18px">
      Java编程语言定义了八种基本的数据类型，共分为四类：整数类(byte、short、int、long)、文本类(char)、浮点类(double、float)和逻辑类(boolean)
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <br/>
    </div>
    <div>
     <span style="font-size:18px">
      <img alt="" src="">
       <br/>
      </img>
     </span>
    </div>
    <div>
     <div>
      <div>
       <span style="font-size:18px">
        <strong>
         1. 整数类
        </strong>
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (1) 采用三种进制——十进制、八进制和十六进制。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        2 —— 十进制值是2；
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        077 —— 首位的0表示这是一个八进制的数值；
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        0xBAAC —— 首位的0x表示这是一个十六进制的数值。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (2) 具有缺省int。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (3) 用字母“L”和“l”定义long。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (4) 所有Java编程语言中的整数类型都是带符号的数字。
       </span>
      </div>
     </div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <div>
      <div>
       <span style="font-size:18px">
        <strong>
         2. 文本类
        </strong>
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (1) 代表一个16 bit Unicode字符。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (2) 必须包含用单引号(' ')引用的文字。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        (3) 使用下列符号：
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        'a'——一个字符。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        '\t'--一个制表符。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        '\u???? '--一个特殊的Unicode字符，????应严格使用四个十六进制数进行替换。
       </span>
      </div>
     </div>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <div>
      <div>
       <span style="font-size:18px">
        <strong>
         3. 浮点类
        </strong>
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        默认为double类型，如果一个数字包括小数点或指数部分，或者在数字后带有字母F或f(float)、D或d(double)，则该数字为浮点数。
       </span>
      </div>
     </div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <div>
      <div>
       <span style="font-size:18px">
        <strong>
         4. 逻辑类
        </strong>
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        boolean数据类型有两种值：true和false。
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        例如：boolean flag = true;
       </span>
      </div>
      <div>
       <span style="font-size:18px">
        上述语句声明变量flag为boolean 类型，它被赋予的值为true。
       </span>
      </div>
     </div>
     <br/>
