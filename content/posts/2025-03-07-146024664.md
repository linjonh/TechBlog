    <h2>
     1. 基础介绍
    </h2>
    <p>
     <strong>
      1、线性结构：
     </strong>
    </p>
    <p>
     如果一个数据元素序列满足：
    </p>
    <p>
     （1）除第一个和最后一个数据元素外，每个数据元素只有一个前驱数据元素和一个后继数据元素；
    </p>
    <p>
     （2）第一个数据元素没有前驱数据元素；
    </p>
    <p>
     （3）最后一个数据元素没有后继数据元素。
    </p>
    <p>
     则称这样的数据结构为线性结构。
    </p>
    <p>
     <strong>
      2. 线性表
     </strong>
    </p>
    <div>
     <span style="color:#646a73">
      线性表
     </span>
     <span style="color:#646a73">
      <em>
       （linear list）
      </em>
     </span>
     <span style="color:#646a73">
      是n个具有相同特性的数据元素的有限序列。 线性表是⼀种在实际中⼴泛使⽤ 的数据结构，常⻅的线性表：顺序表、链表、栈、队列。
     </span>
    </div>
    <div>
     <div>
      <span style="color:#646a73">
       线性表在逻辑上是线性结构，也就说是连续的⼀条直线。但是在物理结构上并不⼀定是连续的，线性表在物理上存储时，通常以数组和链式结构的形式存储
      </span>
     </div>
     <div>
     </div>
     <p>
      <strong>
       <span style="color:#646a73">
        3. 顺序表
       </span>
      </strong>
     </p>
     <div>
      <span style="color:#1f2329">
       顺序表是⽤⼀段物理地址连续的存储单元依次存储数据元素的线性结构，⼀般情况下采⽤数组存储。在数组上完成数据的增删查改。
      </span>
     </div>
     <div>
     </div>

        <p>
         这种结构在Java中表示一个包含多个列表的列表
        </p>

         <strong>
          <code>
           List&lt;List&lt;E&gt;&gt;
          </code>
         </strong>
         : 这是一个泛型声明，表示一个列表的列表。外层列表的每个元素都是一个
         <code>
          List&lt;E&gt;
         </code>

         <code>

         </span>

        </div>
        <div>
         <span style="color:#1456f0">
          1.
         </span>
         <span style="color:#1f2329">
          ArrayList底层使⽤连续的空间，任意位置插⼊或删除元素时，需要将该位置后序元素整体往前或者往后搬移，故时间复杂度为O(N)
         </span>
        </div>
        <div>
         <span style="color:#1456f0">
          2.
         </span>
         <span style="color:#1f2329">

         </span>

          3.
         </span>
         <span style="color:#1f2329">

         </span>

      <pre><code class="language-java">import java.util.ArrayList;
import java.util.List;
import java.util.Random;

public class CardDom {
    static class Card{
        public int rank;//牌面值
        public String suit;//花色

        public Card(int rank,String suit){
            this.rank = rank;
            this.suit = suit;
        }
        @Override
        public String toString() {
            return "["+rank + " "+ suit+"]" ;
        }
    }
    public static final String[] SUITS = {"♠","♥","♣","♦"};
    //有一副牌
    private static List&lt;Card&gt; buyDeck(){
        List&lt;Card&gt; deck = new ArrayList&lt;&gt;();//创建一个链表存放牌
        for(int i=0;i&lt;SUITS.length;i++){
            for(int j=1;j&lt;=13;j++){
                Card card = new Card(j,SUITS[i]);
                deck.add(card);//将牌插入
            }
        }
        return deck;
    }
    //将交换牌的位置
    private static void swap(List&lt;Card&gt; deck, int i, int j){
        Card t = deck.get(i);
        deck.set(i,deck.get(j));
        deck.set(j,t);
    }
    //洗牌
    private static void shuffle(List&lt;Card&gt; deck){
        Random random = new Random();
        for(int i=deck.size()-1;i&gt;0;i--){
            int j = random.nextInt(i);
            swap(deck,i,j);
        }
    }
    //每人发五张牌
    public static void main(String[] args) {
        List&lt;Card&gt; desk = buyDeck();
        System.out.print("买回来的牌:");
        System.out.println(desk);
        shuffle(desk);
        System.out.print("洗过的牌：");
        System.out.println(desk);
        List&lt;List&lt;Card&gt;&gt; heads = new ArrayList&lt;&gt;();//三个人玩牌
        List&lt;Card&gt; head0 = new ArrayList&lt;&gt;();//存放第一个人的牌
        List&lt;Card&gt; head1 = new ArrayList&lt;&gt;();//存放第二个人的牌
        List&lt;Card&gt; head2 = new ArrayList&lt;&gt;();//存放第三个人的牌
        heads.add(head0);
        heads.add(head1);
        heads.add(head2);
        //控制次数
        for(int i=0;i&lt;5;i++){
            for (int j=0;j&lt;3;j++){
                //每次拿零下标的牌
                Card card=desk.remove(0);//从牌中拿出第一张牌
                heads.get(j).add(card);//每个人拿到的牌存入对应的顺序表
            }
        }
        System.out.print("第一个人拿的牌");
        System.out.println(head0);
        System.out.print("第二个人拿的牌");
        System.out.println(head1);
        System.out.print("第三个人拿的牌");
        System.out.println(head2);
        System.out.print("剩下的牌：");
        System.out.println(desk);
    }
