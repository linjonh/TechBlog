    咱们看看,怎么解决这个问题:
    <br/>
    检查一下,发现:
    <br/>
    E:\apache-tomcat-6.0.10\webapps\hello\WEB-INF\web.xml
    <br/>
    里面:
    <br/>
    &lt;servlet-mapping&gt;
    <br/>
    &lt;servlet-name&gt;HelloServlet&lt;/servlet-name&gt;
    <br/>
    &lt;url-pattern&gt;hello.do&lt;/url-pattern&gt;
    <br/>
    &lt;/servlet-mapping&gt;
    <br/>
    &lt;/web-app&gt;
    <br/>
    这里的配置:
    <br/>
    &lt;servlet-mapping&gt;
    <br/>
    &lt;servlet-name&gt;HelloServlet&lt;/servlet-name&gt;
    <br/>
    //1.注意应该把hello.do改成/hello.do
    <br/>
    &lt;url-pattern&gt;/hello.do&lt;/url-pattern&gt;
    <br/>
    &lt;/servlet-mapping&gt;
    <br/>
    &lt;/web-app&gt;
    <br/>
    <br/>
    <br/>
    这样就可以了.
