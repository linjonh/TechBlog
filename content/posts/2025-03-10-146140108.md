     https://blog.csdn.net/qq_45956730/article/details/137601010

    <h3>
     2.5 设置运行脚本参数
    </h3>
    <p>
     根目录下：
     <code>
      .vscode/launch.json
     </code>
    </p>
    <p>
     如果没有就选择debug栏点这个配置：
    </p>
    <p>
     <img alt="" height="413" src="https://i-blog.csdnimg.cn/direct/4b3d8a518fc1431392a7eaca671de92a.png" width="410"/>
    </p>
    <ul>
     <li>
      <p>
       <code>
        launch.json
       </code>
       的内容
      </p>
     </li>
    </ul>
    <pre><code>{
    "version": "0.2.0",
    "configurations": [
      {
        "name": "Python: 当前文件",
        "type": "debugpy",
        "request": "launch",
        "program": "libs/chatchat-server/chatchat/cli.py", # 运行文件
        "console": "integratedTerminal",
        "cwd": "/xxx/codes/langchain-chatchat-0.3.0",  # 工作目录
        "justMyCode": true,
        "args": ["start", "-a"]  # 运行参数
      }
    ]
  }</code></pre>
    <p>
     然后点运行：
    </p>
    <p>
     <img alt="" height="105" src="https://i-blog.csdnimg.cn/direct/5db36282d9a14c249fe2573fb1cb1301.png" width="554"/>
    </p>
    <h3>
     2.6 设置环境变量
    </h3>
    <p>
     仍然是在前面的
     <code>
      launch.json
     </code>
     中增加内容：
    </p>
    <pre><code>{
    "version": "0.2.0",
    "configurations": [
      {
        "name": "Python: 当前文件",
        "type": "debugpy",
        "request": "launch",
        "program": "cli.py",
        "console": "integratedTerminal",
        "cwd": "/xxx/codes/langchain-chatchat-0.3.0/libs/chatchat-server/chatchat",  
        "justMyCode": true,
        "args": ["start", "-a"],
        "env": {
                "CHATCHAT_ROOT": "/xxx/codes/langchain-chatchat-0.3.0/libs/chatchat-server/chatchat/data"
        } # 这里则是增加环境变量
      }
    ]
  }</code></pre>
    <p>
    </p>
    <p>
     然后仍然按照上述方式启动即可。
    </p>
