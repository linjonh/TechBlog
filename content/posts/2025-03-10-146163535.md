    <h5>
     一、HTML 基本结构
    </h5>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;meta charset="utf-8"&gt;
    &lt;title&gt;页面标题&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;!-- 可见内容区域 --&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        &lt;!DOCTYPE html&gt;
       </code>
      </strong>
      ：声明文档类型为 HTML5。
     </li>
     <li>
      <strong>
       <code>
        &lt;html&gt;
       </code>
      </strong>
      ：根标签，包裹所有 HTML 内容。
     </li>
     <li>
      <strong>
       <code>
        &lt;head&gt;
       </code>
      </strong>
      ：包含元数据（标题、CSS、字符集等）。
     </li>
     <li>
      <strong>
       <code>
        &lt;body&gt;
       </code>
      </strong>
      ：页面可见内容的容器。
     </li>
    </ul>

    <h5>
     三、超链接与图像
    </h5>
    <h6>
     1. 超链接
     <code>
      &lt;a&gt;
     </code>
    </h6>
    <pre><code>&lt;a href="https://www.baidu.com" target="_blank" rel="noopener"&gt;百度&lt;/a&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         href
        </code>
        ：链接地址（支持 URL、本地文件、邮箱
        <code>
         mailto:
        </code>
        ）。
       </li>
       <li>
        <code>
         target
        </code>
        ：打开方式（
        <code>
         _blank
        </code>
        新窗口，
        <code>
         _self
        </code>
        当前页）。
       </li>
       <li>
        <code>
         download
        </code>
        ：强制下载文件（需文件名）。
       </li>
      </ul>
     </li>
    </ul>
    <h6>
     2. 图像
     <code>
      &lt;img&gt;
     </code>
    </h6>
    <pre><code>&lt;img src="image.jpg" alt="图片描述" width="200" height="100"&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         src
        </code>
        ：图片路径（支持相对/绝对路径）。
       </li>
       <li>
        <code>
         alt
        </code>
        ：替代文本（图片无法加载时显示）。
       </li>
       <li>
        <code>
         width
        </code>
        /
        <code>
         height
        </code>
        ：尺寸（像素或百分比，高度百分比可能无效）。
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h5>
     四、列表
    </h5>
    <h6>
     1. 无序列表
     <code>
      &lt;ul&gt;
     </code>
    </h6>
    <pre><code>&lt;ul type="square"&gt;
    &lt;li&gt;苹果&lt;/li&gt;
    &lt;li&gt;香蕉&lt;/li&gt;
&lt;/ul&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       type
      </strong>
      ：
      <code>
       disc
      </code>
      （默认实心圆）、
      <code>
       circle
      </code>
      （空心圆）、
      <code>
       square
      </code>
      （方块）。
     </li>
    </ul>
    <h6>
     2. 有序列表
     <code>
      &lt;ol&gt;
     </code>
    </h6>
    <pre><code>&lt;ol type="A" start="3"&gt;
    &lt;li&gt;第一步&lt;/li&gt;
    &lt;li&gt;第二步&lt;/li&gt;
&lt;/ol&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       type
      </strong>
      ：
      <code>
       1
      </code>
      （数字）、
      <code>
       A
      </code>
      （大写字母）、
      <code>
       a
      </code>
      （小写字母）、
      <code>
       I
      </code>
      （罗马数字）。
     </li>
     <li>
      <strong>
       start
      </strong>
      ：起始序号（支持数字）。
     </li>
    </ul>
    <hr/>
    <h5>
     五、表格
    </h5>
    <pre><code>&lt;table border="1"&gt;
    &lt;tr&gt;
        &lt;th colspan="2"&gt;标题&lt;/th&gt;

        &lt;td&gt;单元格1&lt;/td&gt;

         &lt;th&gt;

       </li>

     <li>
      <strong>
       属性

       <li>

     </li>
    </ul>

    <hr/>
    <h5>
     九、注意事项
    </h5>
    <ol>
     <li>
      <strong>
       语义化标签
      </strong>
      ：优先使用
      <code>
       &lt;strong&gt;
      </code>
      、
      <code>
       &lt;em&gt;
      </code>
      等语义化标签。
     </li>
     <li>
      <strong>
       属性弃用
      </strong>
      ：避免使用
      <code>
       align
      </code>
      、
      <code>
       bgcolor
      </code>
      等过时属性，改用 CSS。
     </li>
     <li>
      <strong>
       表单安全
      </strong>
      ：敏感数据（如密码）使用
      <code>
       method="post"
      </code>
      。
     </li>
     <li>
      <strong>
       无障碍访问
      </strong>
      ：为图片添加
      <code>
       alt
      </code>
      属性，表单元素关联
      <code>
       &lt;label&gt;
      </code>
      。
     </li>
    </ol>
