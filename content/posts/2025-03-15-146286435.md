    <p>
     在面对无法清除U盘分区，无法写入，无法抹掉情况时，在 Mac 上可以使用终端彻底格式化 U 盘，可以使用
     <code>
      diskutil
     </code>
     命令。以下是详细步骤：
    </p>
    <h4>
     <a id="__4">
     </a>
     <strong>
      ⚠ 注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       格式化会清空 U 盘上的所有数据
      </strong>
      ，请先备份重要文件。
     </li>
     <li>
      确保你选择了正确的 U 盘，否则可能误删其他数据。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="__U__11">
     </a>
     <strong>
      🛠️ 终端格式化 U 盘教程
     </strong>
    </h3>
    <h4>
     <a id="1__13">
     </a>
     <strong>
      1. 打开终端
     </strong>
    </h4>
    <p>
     在 Mac 上按
     <code>
      Command + Space
     </code>
     ，输入
     <strong>
      “终端”
     </strong>
     ，然后按
     <code>
      Enter
     </code>
     。
    </p>
    <hr/>
    <h4>
     <a id="2__U__19">
     </a>
     <strong>
      2. 查看 U 盘信息
     </strong>
    </h4>
    <p>
     输入以下命令查看所有磁盘设备：
    </p>
    <pre><code class="prism language-bash">diskutil list
</code></pre>
    <p>
     你会看到类似以下的输出：
    </p>
    <pre><code>/dev/disk0 (internal, physical):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:      GUID_partition_scheme                        *500.3 GB   disk0
   1:             Apple_APFS_ISC Container disk1         524.3 MB   disk0s1
   2:                 Apple_APFS Container disk3         494.4 GB   disk0s2
   3:        Apple_APFS_Recovery Container disk2         5.4 GB     disk0s3

/dev/disk3 (synthesized):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:      APFS Container Scheme -                      +494.4 GB   disk3
                                 Physical Store disk0s2
   1:                APFS Volume Macintosh HD - Data     397.3 GB   disk3s1
   2:                APFS Volume Macintosh HD            13.3 GB    disk3s3
   3:              APFS Snapshot com.apple.os.update-... 13.3 GB    disk3s3s1
   4:                APFS Volume Preboot                 12.3 GB    disk3s4
   5:                APFS Volume Recovery                1.9 GB     disk3s5
   6:                APFS Volume VM                      8.6 GB     disk3s6

/dev/disk5 (external, physical):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:      GUID_partition_scheme                        *124.1 GB   disk5
   1:       Microsoft Basic Data                         6.2 GB     disk5s1
   2:                        EFI ESP                     5.2 MB     disk5s2
   3:       Microsoft Basic Data                         307.2 KB   disk5s3
   4:       Microsoft Basic Data 新加卷                  117.9 GB   disk5s4
</code></pre>
    <p>
     在这个例子中，U 盘的
     <code>
      IDENTIFIER
     </code>
     是
     <strong>
      disk5
     </strong>
     （你的可能不同）。
    </p>
    <hr/>
    <h4>
     <a id="3__U__61">
     </a>
     <strong>
      3. 卸载 U 盘
     </strong>
    </h4>
    <p>
     格式化前，先卸载 U 盘：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> diskutil umountDisk /dev/disk5
</code></pre>
    <hr/>
    <h4>
     <a id="4__U__71">
     </a>
     <strong>
      4. 覆盖 U 盘
     </strong>
    </h4>
    <pre><code class="prism language-shell"><span class="token function">sudo</span> diskutil zeroDisk /dev/disk5
</code></pre>
    <h4>
     <a id="5__U__77">
     </a>
     <strong>
      5. 彻底格式化 U 盘
     </strong>
    </h4>
    <p>
     格式化并抹除 U 盘，使用以下命令：
    </p>
    <pre><code class="prism language-bash">diskutil eraseDisk FAT32 USB_NAME MBRFormat /dev/disk5
</code></pre>
    <ul>
     <li>
      <code>
       FAT32
      </code>
      —— 兼容 Windows 和 macOS（可改为
      <code>
       ExFAT
      </code>
      或
      <code>
       APFS
      </code>
      ）。
     </li>
     <li>
      <code>
       USB_NAME
      </code>
      —— 你想给 U 盘起的名字（可替换为你喜欢的）。
     </li>
     <li>
      <code>
       MBRFormat
      </code>
      —— 使用
      <strong>
       主引导记录 (MBR)
      </strong>
      ，适用于 U 盘。
     </li>
     <li>
      <code>
       /dev/disk5
      </code>
      ——
      <strong>
       确保这里是你的 U 盘编号
      </strong>
      ！
     </li>
    </ul>
    <p>
     如果要使用
     <strong>
      ExFAT
     </strong>
     格式（支持大文件，兼容性好）：
    </p>
    <pre><code class="prism language-bash">diskutil eraseDisk ExFAT USB_NAME MBRFormat /dev/disk2
</code></pre>
    <p>
     如果要格式化为
     <strong>
      macOS 专用格式 (APFS)
     </strong>
     ：
    </p>
    <pre><code class="prism language-bash">diskutil eraseDisk APFS USB_NAME GPT /dev/disk2
</code></pre>
    <hr/>
    <h4>
     <a id="5__104">
     </a>
     <strong>
      5. 确保格式化完成
     </strong>
    </h4>
    <p>
     输入：
    </p>
    <pre><code class="prism language-bash">diskutil list
</code></pre>
    <p>
     检查 U 盘是否已正确格式化。
    </p>
    <hr/>
    <h4>
     <a id="6__116">
     </a>
     <strong>
      6. 安全擦除（可选）
     </strong>
    </h4>
    <p>
     如果要彻底清除数据，使其难以恢复，可以用
     <code>
      secureErase
     </code>
     ：
    </p>
    <pre><code class="prism language-bash">diskutil secureErase <span class="token number">2</span> /dev/disk2
</code></pre>
    <p>
     参数解释：
    </p>
    <ul>
     <li>
      <code>
       0
      </code>
      – 快速擦除（不安全）。
     </li>
     <li>
      <code>
       1
      </code>
      – 单次写入 0。
     </li>
     <li>
      <code>
       2
      </code>
      –
      <strong>
       （推荐）
      </strong>
      7 次写入随机数据。
     </li>
     <li>
      <code>
       3
      </code>
      – 35 次写入随机数据（极端安全）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="7__U__133">
     </a>
     <strong>
      7. 弹出 U 盘
     </strong>
    </h4>
    <p>
     格式化完成后，可以安全弹出：
    </p>
    <pre><code class="prism language-bash">diskutil <span class="token function">eject</span> /dev/disk2
</code></pre>
    <p>
     然后拔掉 U 盘即可。
    </p>
    <hr/>
    <h3>
     <a id="_145">
     </a>
     <strong>
      总结
     </strong>
    </h3>
    <ul>
     <li>
      <code>
       diskutil list
      </code>
      → 找到 U 盘的编号。
     </li>
     <li>
      <code>
       diskutil unmountDisk /dev/diskX
      </code>
      → 卸载 U 盘。
     </li>
     <li>
      <code>
       diskutil eraseDisk FORMAT NAME MBRFormat /dev/diskX
      </code>
      → 彻底格式化。
     </li>
     <li>
      （可选）
      <code>
       diskutil secureErase
      </code>
      → 安全擦除数据。
     </li>
     <li>
      <code>
       diskutil eject /dev/diskX
      </code>
      → 安全弹出 U 盘。
     </li>
    </ul>
    <p>
     这样， U 盘就被彻底格式化并可用啦！
    </p>
