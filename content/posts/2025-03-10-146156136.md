# ubuntu 安装 Zotero

## 下载安装包

https://www.zotero.org/download/

## 安装zeotero

参考 https://www.zotero.org/support/installation

### 解压并移动到opt，修改权限

将解压得到的 Zotero\_linux-x86\_64 文件夹移动到 opt 文件夹下:

```bash
sudo cp -r Zotero_linux-x86_64/ /opt/ 

```

进入/opt/Zotero\_linux-x86\_64，给 zotero、zotero-bin、set\_launcher\_icon 添加执行权限。

```bash
sudo chmod +x zotero
 
sudo chmod +x zotero-bin
 
sudo chmod +x set_launcher_icon

```

下一步在终端输入：

```bash
sudo ./set_launcher_icon
./zotero

```

即可打开软件Zotero

### 桌面启动

将 zotero.desktop 链接到 ~/.local/share/applications/

```bash
ln -s /opt/zotero/zotero.desktop ~/.local/share/applications/zotero.desktop

```
