    <h3>
     <strong>
      1.
      <code>
       exports
      </code>
      对象介绍
     </strong>
    </h3>
    <p>
     在 Node.js 中，
     <code>
      exports
     </code>
     是
     <code>
      module.exports
     </code>
     的一个引用（别名），用于导出模块的内容。
    </p>
    <p>
     <strong>
      主要特点：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       exports
      </code>
      和
      <code>
       module.exports
      </code>
      最初指向同一个对象。
     </li>
     <li>
      只能使用
      <code>
       exports.属性 = 值
      </code>
      形式来添加属性，不能直接赋值
      <code>
       exports = {...}
      </code>
      ，否则会断开与
      <code>
       module.exports
      </code>
      的关联。
     </li>
     <li>
      推荐使用
      <code>
       module.exports
      </code>
      ，因为它更清晰，避免
      <code>
       exports
      </code>
      赋值带来的问题。
     </li>
    </ul>
    <hr/>
    <h3>
     <strong>
      2. 基础示例：使用
      <code>
       exports
      </code>
      导出对象
     </strong>
    </h3>
    <p>
     创建一个
     <code>
      math.js
     </code>
     文件，使用
     <code>
      exports
     </code>
     对象导出数学运算函数。
    </p>
    <h4>
     <strong>
      文件名：math.js
     </strong>
     （JavaScript）
    </h4>
    <pre><code class="language-javascript">// math.js
exports.add = (a, b) =&gt; a + b;
exports.subtract = (a, b) =&gt; a - b;
exports.multiply = (a, b) =&gt; a * b;
exports.divide = (a, b) =&gt; (b !== 0 ? a / b : "Error: Division by zero");
</code></pre>
    <hr/>
    <h3>
     <strong>
      3. 在另一个文件中使用
      <code>
       require
      </code>
      进行导入
     </strong>
    </h3>
    <p>
     创建
     <code>
      app.js
     </code>
     文件，并使用
     <code>
      require
     </code>
     来引入
     <code>
      math.js
     </code>
     模块。
    </p>
    <h4>
     <strong>
      文件名：app.js
     </strong>
     （JavaScript）
    </h4>
    <pre><code class="language-javascript">// app.js
const math = require("./math"); // 引入 math.js 模块

console.log("加法 5 + 3 =", math.add(5, 3)); 
console.log("减法 10 - 4 =", math.subtract(10, 4)); 
console.log("乘法 6 * 7 =", math.multiply(6, 7)); 
console.log("除法 20 / 5 =", math.divide(20, 5)); 
console.log("除法 10 / 0 =", math.divide(10, 0)); // 测试除数为0的情况
</code></pre>
    <hr/>
    <h3>
     <strong>
      4. 运行程序
     </strong>
    </h3>
    <p>
     在终端（Terminal 或 CMD）中运行：
    </p>
    <pre><code class="language-bash">node app.js
</code></pre>
    <p>
     <strong>
      运行结果：
     </strong>
    </p>
    <pre><code class="language-bash">加法 5 + 3 = 8
减法 10 - 4 = 6
乘法 6 * 7 = 42
除法 20 / 5 = 4
除法 10 / 0 = Error: Division by zero
</code></pre>
    <hr/>
    <h3>
     <strong>
      5.
      <code>
       exports
      </code>
      的限制
     </strong>
    </h3>
    <p>
     <strong>
      错误示例
     </strong>
     （不建议使用）：
    </p>
    <pre><code class="language-javascript">// math.js
exports = {  // ❌ 这样会导致导出失败
    add: (a, b) =&gt; a + b,
    subtract: (a, b) =&gt; a - b
};
</code></pre>
    <p>
     <strong>
      原因：
     </strong>
     这样做会让
     <code>
      exports
     </code>
     变量指向一个新对象，但
     <code>
      module.exports
     </code>
     仍然指向原始对象，导致
     <code>
      require('./math')
     </code>
     取不到新赋值的对象。
    </p>
    <hr/>
    <h3>
     <strong>
      6. 其他
      <code>
       exports
      </code>
      使用方式
     </strong>
    </h3>
    <h4>
     <strong>
      示例1：导出单个函数
     </strong>
    </h4>
    <p>
     <strong>
      文件名：greet.js
     </strong>
     （JavaScript）
    </p>
    <pre><code class="language-javascript">// greet.js
exports.greet = function(name) {
    return `Hello, ${name}!`;
};
</code></pre>
    <p>
     <strong>
      文件名：app.js
     </strong>
     （JavaScript）
    </p>
    <pre><code class="language-javascript">// app.js
const greetModule = require("./greet");
console.log(greetModule.greet("鼠鼠"));
</code></pre>
    <p>
     <strong>
      运行结果：
     </strong>
    </p>
    <pre><code class="language-bash">Hello, 鼠鼠!
</code></pre>
    <hr/>
    <h4>
     <strong>
      示例2：导出类
     </strong>
    </h4>
    <p>
     <strong>
      文件名：Person.js
     </strong>
     （JavaScript）
    </p>
    <pre><code class="language-javascript">// Person.js
exports.Person = class {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    introduce() {
        return `My name is ${this.name}, I am ${this.age} years old.`;
    }
};
</code></pre>
    <p>
     <strong>
      文件名：app.js
     </strong>
     （JavaScript）
    </p>
    <pre><code class="language-javascript">// app.js
const { Person } = require("./Person");

const person1 = new Person("Alice", 25);
console.log(person1.introduce());

const person2 = new Person("Bob", 30);
console.log(person2.introduce());
</code></pre>
    <p>
     <strong>
      运行结果：
     </strong>
    </p>
    <pre><code class="language-javascript">My name is Alice, I am 25 years old.
My name is Bob, I am 30 years old.
</code></pre>
    <hr/>
    <h3>
     <strong>
      7. 结论
     </strong>
    </h3>
    <ul>
     <li>
      <code>
       exports
      </code>
      是
      <code>
       module.exports
      </code>
      的一个引用，默认指向相同的对象。
     </li>
     <li>
      只能使用
      <code>
       exports.属性 = 值
      </code>
      形式，不能直接
      <code>
       exports = {...}
      </code>
      赋值，否则会失去作用。
     </li>
     <li>
      一般推荐使用
      <code>
       module.exports
      </code>
      ，避免
      <code>
       exports
      </code>
      赋值带来的问题。
     </li>
     <li>
      适用于导出多个函数或对象时使用。
     </li>
    </ul>

     希望这篇文章能帮助你理解
     <code>
      exports
     </code>
     的用法！
