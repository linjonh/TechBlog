    <h4>
     <a id="Python_0_0">
     </a>
     Python 自动给数字前面补0
    </h4>
    <p>
     为了排版方便或者是输出文件命名整洁，通常需要给数字前面补0来做统一。
     <br/>
     Python中有一个zfill函数用来给字符串前面补0，非常有用，这个zfill看起来也就是zero fill的缩写吧，看一下如何使用：
    </p>
    <h5>
     <a id="10_3">
     </a>
     1.字符串前面补0
    </h5>
    <pre><code>n = "123"
s = n.zfill(5)
print(s)

# 结果：
00123
</code></pre>
    <h5>
     <a id="20_11">
     </a>
     2.整型数据前面补0
    </h5>
    <pre><code>a = 123
s = '%05d' % a
print(s)

# 结果：
00123
</code></pre>
