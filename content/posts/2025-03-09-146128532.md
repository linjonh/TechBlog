    <h1 class="title-article" id="articleContentId">
     如何在 Windows 10 启用卓越性能模式及不同电源计划对比
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="markdown_views prism-github-gist" id="content_views">
    <svg style="display: none;" xmlns="http://www.w3.org/2000/svg">
     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     在使用
     <code>
      powercfg -duplicatescheme
     </code>
     命令启用
     <strong>
      “卓越性能模式”
     </strong>
     （即
     <strong>
      Ultimate Performance
     </strong>
     模式）之前，有几个前提条件需要注意：
    </p>
    <h4>
     <a id="_2">
     </a>
     前提条件：
    </h4>
    <ol>
     <li>
      <strong>
       系统版本要求
      </strong>
      ：
      <strong>
       卓越性能模式
      </strong>
      仅在
      <strong>
       Windows 10 专业版
      </strong>
      或更高版本（如 Windows 10 企业版、Windows 10 教育版）中提供。如果你使用的是 Windows 10 家庭版，无法启用此模式。
     </li>
     <li>
      <strong>
       硬件要求
      </strong>
      ：此模式主要用于桌面系统，而某些笔记本电脑可能不支持该功能。如果你的系统不支持此模式，可能会收到错误提示。
     </li>
     <li>
      <strong>
       管理员权限
      </strong>
      ：你需要以管理员身份运行 PowerShell，才能执行
      <code>
       powercfg
      </code>
      命令。
     </li>
    </ol>
    <h4>
     <a id="_7">
     </a>
     启用卓越性能模式的步骤：
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        右击开始菜单
       </strong>
       ，选择
       <strong>
        “Windows PowerShell (管理员)”
       </strong>
       。确保你有管理员权限。
      </p>
     </li>
     <li>
      <p>
       输入以下命令，然后按
       <strong>
        Enter
       </strong>
       ：
      </p>
      <pre><code class="prism language-powershell">powercfg <span class="token operator">-</span>duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
</code></pre>
      <p>
       该命令会复制
       <strong>
        “卓越性能模式”
       </strong>
       （
       <code>
        Ultimate Performance
       </code>
       ）到你的电源计划中，并使其成为可选项。
      </p>
     </li>
    </ol>
    <h4>
     <a id="_19">
     </a>
     开启高性能模式（“卓越性能”模式）
    </h4>
    <ol>
     <li>
      <strong>
       右键点击开始菜单
      </strong>
      ，选择
      <strong>
       “电源选项”
      </strong>
      。
     </li>
     <li>
      在电源选项窗口中，点击
      <strong>
       “显示附加计划”
      </strong>
      。
     </li>
     <li>
      选择
      <strong>
       “高性能”
      </strong>
      计划，或者选择
      <strong>
       “平衡”
      </strong>
      计划并通过点击
      <strong>
       “更改计划设置”
      </strong>
      ，然后选择
      <strong>
       “更改高级电源设置”
      </strong>
      ，进一步调整高级设置以提高性能。
      <ul>
       <li>
        对于某些系统，特别是笔记本电脑，可能没有直接的高性能计划。此时，可以启用
        <strong>
         “卓越性能”
        </strong>
        模式（如果支持）。
       </li>
      </ul>
     </li>
     <li>
      在
      <strong>
       高级设置
      </strong>
      中，你可以进一步调整 CPU、电池使用情况、硬盘、显示等多个项的策略。例如：
      <ul>
       <li>
        将
        <strong>
         处理器的最大状态
        </strong>
        设置为
        <strong>
         100%
        </strong>
        ，以确保 CPU 始终以最高频率运行。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_28">
     </a>
     不同模式的性能对比
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        高性能模式
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         性能
        </strong>
        ：优先保证性能，CPU 会持续保持在较高的频率，避免频繁降频。适合高负载任务（如游戏、视频编辑、大规模计算）。
       </li>
       <li>
        <strong>
         功耗
        </strong>
        ：较高，笔记本电池寿命会缩短，台式机的电力消耗增加。
       </li>
       <li>
        <strong>
         应用场景
        </strong>
        ：需要全速运行的应用程序或游戏。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        平衡模式
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         性能
        </strong>
        ：根据当前的使用需求动态调整 CPU 性能。低负载时降频，减少能耗，负载高时提高性能。
       </li>
       <li>
        <strong>
         功耗
        </strong>
        ：中等，适合日常使用，既可以保证性能，又能优化电池寿命。
       </li>
       <li>
        <strong>
         应用场景
        </strong>
        ：日常办公、浏览网页、看视频等常见操作。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        节能模式
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         性能
        </strong>
        ：降低 CPU 和硬盘性能，优先考虑电池寿命和系统冷却，适合不需要高性能的操作。
       </li>
       <li>
        <strong>
         功耗
        </strong>
        ：非常低，适合需要延长电池使用时间的场景。
       </li>
       <li>
        <strong>
         应用场景
        </strong>
        ：轻度使用，保持电池寿命，适合长时间外出时使用。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_45">
     </a>
     小结
    </h4>
    <ul>
     <li>
      <strong>
       卓越性能模式（高性能模式）
      </strong>
      最大化了系统的计算能力，适合需要处理大量数据或执行高强度任务的场景。
     </li>
     <li>
      <strong>
       平衡模式
      </strong>
      是最常见的选择，适合大多数日常使用，能在性能和功耗之间取得良好平衡。
     </li>
     <li>
      <strong>
       节能模式
      </strong>
      主要用于节省电力，但牺牲了一些性能，适用于轻度任务或延长笔记本的电池使用时间。
     </li>
    </ul>
    <p>
     你可以根据自己的需求选择合适的电源计划，特别是在需要高负载时使用高性能模式，而日常使用则选择平衡模式以延长设备的电池寿命。
    </p>
