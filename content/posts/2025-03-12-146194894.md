    <h2>
     <a id="IDEAJDKIDEA__0">
     </a>
     <strong>
      IDEA社区版如何使用JDK原生命令：从IDEA到命令行的开发技巧
     </strong>
     🚀
    </h2>
    <p>
     在 Java 开发中，我们通常使用 IDE（如 IntelliJ IDEA）来提高开发效率。但有时，我们也需要了解如何使用 JDK 原生命令来完成一些开发任务。尤其是对于
     <strong>
      IDEA 社区版
     </strong>
     （没有集成高级工具）的用户，熟悉如何从命令行使用 JDK 原生命令会为你提供更多的灵活性与便利性。
    </p>
    <p>
     本篇文章将为你详细介绍
     <strong>
      如何在 IDEA 社区版中使用 JDK 原生命令
     </strong>
     ，以及如何从命令行编译和运行 Java 程序。让我们一起掌握这些基本技能，提升开发效率！
    </p>
    <hr/>
    <h3>
     <a id="_1__JDK__8">
     </a>
     <strong>
      📌 1. 什么是 JDK 原生命令？
     </strong>
    </h3>
    <p>
     JDK 原生命令（如
     <code>
      javac
     </code>
     、
     <code>
      java
     </code>
     、
     <code>
      javadoc
     </code>
     等）是 JDK 提供的基础命令行工具，用于：
    </p>
    <ul>
     <li>
      <strong>
       编译 Java 程序
      </strong>
      （
      <code>
       javac
      </code>
      ）
     </li>
     <li>
      <strong>
       运行 Java 程序
      </strong>
      （
      <code>
       java
      </code>
      ）
     </li>
     <li>
      <strong>
       查看 Java 类的文档
      </strong>
      （
      <code>
       javadoc
      </code>
      ）
     </li>
    </ul>
    <p>
     这些命令是 JDK 自带的，通常通过命令行终端运行。当你需要了解或调试 Java 程序时，这些命令将非常有用。
    </p>
    <hr/>
    <h3>
     <a id="_2__IDEA__JDK__18">
     </a>
     <strong>
      📌 2. 在 IDEA 社区版中配置 JDK 环境
     </strong>
    </h3>
    <h4>
     <a id="__1_JDK_19">
     </a>
     <strong>
      🎯 步骤 1：安装 JDK
     </strong>
    </h4>
    <p>
     确保你的机器已经安装了 JDK（推荐使用 JDK 17 或更高版本）。你可以从 Oracle 官网或者 OpenJDK 项目下载 JDK。
    </p>
    <h4>
     <a id="__2_22">
     </a>
     <strong>
      🎯 步骤 2：配置环境变量
     </strong>
    </h4>
    <p>
     安装完成后，需要将 JDK 的
     <code>
      bin
     </code>
     目录添加到系统的环境变量中。这样，你就可以在任何终端窗口中运行 JDK 原生命令了。
    </p>
    <h5>
     <a id="Windows__25">
     </a>
     <strong>
      Windows 环境配置：
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       右键点击“此电脑” &gt; “属性”
      </strong>
     </li>
     <li>
      <strong>
       点击“高级系统设置” &gt; “环境变量”
      </strong>
     </li>
     <li>
      在
      <strong>
       系统变量
      </strong>
      中找到
      <code>
       Path
      </code>
      变量，点击
      <strong>
       编辑
      </strong>
      ，然后添加
      <code>
       C:\Program Files\Java\jdk-17\bin
      </code>
      （假设 JDK 安装在该目录）。
     </li>
    </ol>
    <h5>
     <a id="MacLinux__30">
     </a>
     <strong>
      Mac/Linux 环境配置：
     </strong>
    </h5>
    <p>
     在终端中执行以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span>/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token variable">$JAVA_HOME</span>/bin:<span class="token environment constant">$PATH</span>
</code></pre>
    <p>
     <strong>
      ⚡ 通过以上配置，你就可以在命令行中使用 JDK 原生命令了！
     </strong>
    </p>
    <hr/>
    <h3>
     <a id="_3__JDK__Java__40">
     </a>
     <strong>
      📌 3. 如何使用 JDK 原生命令编译和运行 Java 程序？
     </strong>
    </h3>
    <h4>
     <a id="__Java__42">
     </a>
     <strong>
      🎯 编写 Java 程序
     </strong>
    </h4>
    <p>
     首先，在 IDEA 社区版中创建一个简单的 Java 项目并编写一个 Java 类（例如
     <code>
      HelloWorld.java
     </code>
     ）：
    </p>
    <pre><code class="prism language-java"><span class="token comment">// HelloWorld.java</span>
<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">HelloWorld</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Hello, World!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_1__Java__53">
     </a>
     <strong>
      🎯 1️⃣ 编译 Java 程序
     </strong>
    </h4>
    <p>
     接下来，我们将使用 JDK 提供的
     <code>
      javac
     </code>
     命令来编译这个 Java 文件。
     <br/>
     在 IDEA 中，右键点击
     <code>
      HelloWorld.java
     </code>
     ，选择
     <strong>
      Show in Explorer
     </strong>
     ，进入文件所在目录。
    </p>
    <p>
     然后，在命令行中执行：
    </p>
    <pre><code class="prism language-sh">javac HelloWorld.java
</code></pre>
    <p>
     该命令会在当前目录生成一个
     <code>
      HelloWorld.class
     </code>
     文件，这是编译后的字节码文件。
    </p>
    <h4>
     <a id="_2__Java__63">
     </a>
     <strong>
      🎯 2️⃣ 运行 Java 程序
     </strong>
    </h4>
    <p>
     现在，可以使用
     <code>
      java
     </code>
     命令来运行编译后的 Java 程序。执行以下命令：
    </p>
    <pre><code class="prism language-sh"><span class="token function">java</span> HelloWorld
</code></pre>
    <p>
     你会看到控制台输出：
    </p>
    <pre><code>Hello, World!
</code></pre>
    <p>
     这就意味着你成功地编译并运行了 Java 程序！
    </p>
    <hr/>
    <h3>
     <a id="_4__IDEA__JDK__76">
     </a>
     <strong>
      📌 4. 使用 IDEA 社区版的终端集成 JDK 命令
     </strong>
    </h3>
    <p>
     IDEA 社区版本身没有集成终端窗口，但你可以通过外部终端（如 Windows 的命令提示符、PowerShell 或 Mac/Linux 的终端）来运行 JDK 命令。
     <br/>
     如果你更喜欢直接在 IDEA 内部进行命令行操作，可以使用以下方法：
    </p>
    <h4>
     <a id="__1_80">
     </a>
     <strong>
      🎯 方法 1：使用系统终端
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       Windows
      </strong>
      ：打开
      <strong>
       命令提示符
      </strong>
      或
      <strong>
       PowerShell
      </strong>
      。
     </li>
     <li>
      <strong>
       Mac/Linux
      </strong>
      ：打开
      <strong>
       终端
      </strong>
      。
     </li>
    </ol>
    <p>
     然后进入到项目的工作目录，使用
     <code>
      javac
     </code>
     和
     <code>
      java
     </code>
     命令编译和运行 Java 程序。
    </p>
    <h4>
     <a id="__2_IDEA__86">
     </a>
     <strong>
      🎯 方法 2：使用 IDEA 外部工具配置终端
     </strong>
    </h4>
    <p>
     虽然 IDEA 社区版没有集成终端，但你可以配置外部工具来便捷地运行 JDK 原生命令：
    </p>
    <ol>
     <li>
      打开
      <strong>
       File
      </strong>
      &gt;
      <strong>
       Settings
      </strong>
      &gt;
      <strong>
       Tools
      </strong>
      &gt;
      <strong>
       External Tools
      </strong>
      。
     </li>
     <li>
      添加一个新工具，配置为调用
      <strong>
       命令行工具
      </strong>
      ，例如
      <code>
       javac
      </code>
      和
      <code>
       java
      </code>
      。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_5__JDK__93">
     </a>
     <strong>
      📌 5. 使用 JDK 命令调试和生成文档
     </strong>
    </h3>
    <h4>
     <a id="_javadoc_API__95">
     </a>
     <strong>
      🎯
      <code>
       javadoc
      </code>
      ：生成 API 文档
     </strong>
    </h4>
    <p>
     <code>
      javadoc
     </code>
     命令可以帮助我们生成 API 文档，特别适合团队协作时使用。
    </p>
    <p>
     例如，执行以下命令来为
     <code>
      HelloWorld.java
     </code>
     生成文档：
    </p>
    <pre><code class="prism language-sh">javadoc HelloWorld.java
</code></pre>
    <p>
     此命令会在当前目录下生成一个
     <code>
      index.html
     </code>
     文件，通过浏览器查看详细文档。
    </p>
    <h4>
     <a id="_javap_104">
     </a>
     <strong>
      🎯
      <code>
       javap
      </code>
      ：查看编译后的字节码
     </strong>
    </h4>
    <p>
     <code>
      javap
     </code>
     命令可以反编译
     <code>
      .class
     </code>
     文件，查看字节码内容：
    </p>
    <pre><code class="prism language-sh">javap HelloWorld
</code></pre>
    <p>
     这对于调试和学习 Java 字节码非常有用。
    </p>
    <hr/>
    <h3>
     <a id="_6__113">
     </a>
     <strong>
      📌 6. 总结与小技巧
     </strong>
    </h3>
    <p>
     🔹
     <strong>
      IDEA 社区版没有内置终端，但我们可以通过外部终端使用 JDK 原生命令
     </strong>
     ，如
     <code>
      javac
     </code>
     、
     <code>
      java
     </code>
     、
     <code>
      javadoc
     </code>
     等。
     <br/>
     🔹
     <strong>
      配置 JDK 环境变量
     </strong>
     后，你就可以在命令行中使用这些工具。
     <br/>
     🔹
     <strong>
      使用
      <code>
       javac
      </code>
      编译 Java 文件，使用
      <code>
       java
      </code>
      运行 Java 程序
     </strong>
     。
     <br/>
     🔹
     <strong>
      通过
      <code>
       javadoc
      </code>
      和
      <code>
       javap
      </code>
      命令进行文档生成和字节码分析
     </strong>
     。
    </p>
    <p>
     掌握 JDK 原生命令将帮助你更加灵活地在不同环境下开发和调试 Java 程序，即使是没有完整 IDE 功能的社区版。你还可以通过命令行进行批处理和自动化任务，让你的开发更高效！
    </p>
    <p>
     🚀
     <strong>
      试试看，在 IDEA 社区版中开始使用 JDK 原生命令，提升你的开发效率！
     </strong>
    </p>
