     （1）安装依赖
    </h3>
    <p>
     选中前端目录输入cmd回车就会打开控制台

    <p>
     <img alt="" height="447" src="https://i-blog.csdnimg.cn/direct/d9daff9968784d118348f31ee8fa19c9.png" width="586"/>

    <p>
     输入
     <span style="background-color:#fbd4d0">
      npm install
     </span>
     后等待安装依赖，安装成功后再输入
     <span style="background-color:#fbd4d0">
      code .
     </span>
     <span style="background-color:null">
      自动打开vscode。
     </span>
    </p>

     （3）补充：

    <p>

     <span style="background-color:#fbd4d0">

     <span style="background-color:#fbd4d0">
      npm run dev

     <span style="background-color:null">
