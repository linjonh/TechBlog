    <h2>
     1.控制系统定义
    </h2>
    <p>
     控制系统是指通过监测和调整系统的行为，以达到预期目标的一套系统。它由一组相互关联的组件组成，这些组件协同工作，用于控制物理过程、机械设备、电子设备或其他系统。例如，一个简单的温控系统可以通过监测房间温度并调整加热器的工作状态，使温度保持在设定值。
    </p>
    <p>
     <strong>
      古典控制理论
     </strong>
     主要以
     <strong>
      传递函数
     </strong>
     为基础，研究
     <strong>
      单输入、单输出
     </strong>
     一类自动控制系统的分析和设计问题；
    </p>
    <p>
     <strong>
      现代控制理论
     </strong>
     主要以
     <strong>
      状态空间法
     </strong>
     为基础，研究
     <strong>
      多输入、多输出、多参数、非线性、高精度、高效能
     </strong>
     等控制系统的分析和设计问题。如最优控制、最佳滤波、系统识别、自适应控制等课题。
    </p>
    <ul>
     <li>
      <strong>
       系统：
      </strong>
      以完成一定任务的一些部件或原件的组合；
     </li>
     <li>
      <strong>
       控制量：
      </strong>
      系统的输入变量；
     </li>
     <li>
      <strong>
       被控制量：
      </strong>
      系统的输出变量；
     </li>
     <li>
      <strong>
       扰动量：
      </strong>
      所有妨碍控制量对被控质量按要求进行正常控制的因素叫做扰动量；
     </li>
     <li>
      <strong>
       控制过程：
      </strong>
      消除扰动因素影响从而保持被控制量按预期要求变化的过程叫做控制过程；
     </li>
     <li>
      <strong>
       反馈量：
      </strong>
      反馈信号
     </li>
     <li>
      <strong>
       偏差量：
      </strong>
      偏差信号，反馈量和控制量的偏差。
     </li>
    </ul>
    <p>
    </p>
    <h2>
     2.控制系统的分类
    </h2>
    <p>
     根据控制信号的生成方式，控制系统可分为两种主要类型：
    </p>
    <ul>
     <li>
      <strong>
       开环控制系统（Open-loop Control System）
      </strong>
      <br/>
      在开环系统中，控制器直接根据参考输入生成控制信号，不考虑被控制量的实际输出；即被控制量只能受控于控制量，而不能对控制量反施任何影响。
      <ul>
       <li>
        开环系统结构简单，但对外部干扰或系统变化较敏感，准确性较低。
       </li>
       <li>
        控制精度取决于高精度元件和有效的抗干扰措施；
       </li>
      </ul>
     </li>
     <li>
      <strong>
       闭环控制系统（Closed-loop Control System）
      </strong>
      <br/>
      也称为反馈控制系统，控制器根据被控对象的实际输出与参考输入的误差，动态调整控制信号。控制量通过控制器控制被控制量，而被控制量又被反馈到输入端和控制量进行相减运算产生偏差量，再经过适当变换去控制被控制量以消除偏差。
      <ul>
       <li>
        闭环控制系统能自动校正扰动量，具有更高的准确性和稳定性；
       </li>
       <li>
        可以采用不太精密和成本低廉的元件构成控制质量较高系统；
       </li>
       <li>
        随动系统：控制量为时变信号，被控质量复现控制量；
       </li>
       <li>
        稳定系统：控制量为常数；
       </li>
      </ul>
     </li>
    </ul>
    <h2>
     3.控制系统的基本元件及基本要求
    </h2>
    <h3>
     基本元件
    </h3>
    <ul>
     <li>
      测量元件：测量被控制量和控制量；其精度直接影响控制系统的精度；
     </li>
     <li>
      比较元件：比较被控制量和控制量，以产生偏差信号；
     </li>
     <li>
      放大元件：由于偏差信号比较微弱，需要进行变换放大；
     </li>
     <li>
      执行元件：根据偏差信号产生的控制作用，使被控对象按控制信号的变化规律动作；
     </li>
     <li>
      校正元件：串联校正以解决前向通道延迟反应，反馈校正以解决反馈通道震荡反应；
     </li>
    </ul>
    <h3>
     基本要求
    </h3>
    <ul>
     <li>
      过渡过程：被控制量在受到控制量或扰动量作用下，由原来的稳态变化到新稳态的过程，有如下动态指标
      <ul>
       <li>
        过渡时间：表征系统反应输入信号速度的一项重要性能指标，表征快速性能；
       </li>
       <li>
        达到第一个极值时间：过渡过程达到第一个极值所需要的时间来表征反应输入信号的快速性能；
       </li>
       <li>
        超调量：第一个极值减去稳态值，越小表征阻尼性能越好；
       </li>
       <li>
        震荡次数：反馈量穿越预设控制量的次数的一半；震荡次数越小说明系统的阻尼性能越好；
       </li>
      </ul>
     </li>
     <li>
      稳态：稳态误差是表征稳态性能的一项指标
      <ul>
       <li>
        单调上升
       </li>
       <li>
        不稳定系统
       </li>
       <li>
        衰减震荡
       </li>
       <li>
        发散震荡
       </li>
       <li>
        等幅震荡-自震荡：取决于系统固有特性；
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     性能指标
    </h3>
    <ul>
     <li>
      <strong>
       稳定性（Stability）
      </strong>
      ：系统在受到扰动后，能否恢复到稳态。
     </li>
     <li>
      <strong>
       准确性（Accuracy）
      </strong>
      ：系统输出与参考输入之间的偏差大小。
     </li>
     <li>
      <strong>
       响应速度（Speed of Response）
      </strong>
      ：系统从接收输入到达到稳态所需的时间。
     </li>
     <li>
      <strong>
       鲁棒性（Robustness）
      </strong>
      ：系统在面对参数变化或外部干扰时，维持性能的能力。
     </li>
    </ul>
    <p>
     这些指标在设计控制系统时需权衡优化，以满足实际需求。
    </p>
    <h2>
     4.控制系统的传递函数
    </h2>
    <ol>
     <li>
      运动方程式
     </li>
     <li>
      非线性运动方程的线性化
     </li>
     <li>
      传递函数
      <ol>
       <li>
        描述控制系统或元件固有的动态特性的一种关系式，与运动方程式一一对应；
       </li>
       <li>
        是复变量s的有理分式，分子多项式和分母多项式的各项系数均为实数；
       </li>
       <li>
        分母多项式的最高阶次高于或等于分子多项式的最高阶次；
       </li>
       <li>
        零点和极点、复数零点和复数极点；
       </li>
       <li>
        串联环节-并联环节-反馈回路环节的传递函数的求取
       </li>
      </ol>
     </li>
     <li>
      方块图及其简化：函数方块、想加点、分支点
     </li>
     <li>
      信号流图：梅森增益公式
     </li>
    </ol>
    <h2>
     5.控制系统的数学模型
    </h2>
    <p>
     为了分析和设计控制系统，需要建立其数学模型，常用的模型包括：
    </p>
    <ul>
     <li>
      <strong>
       微分方程（Differential Equations）
      </strong>
      ：描述系统随时间变化的动态行为，例如牛顿力学方程。
     </li>
     <li>
      <strong>
       传递函数（Transfer Function）
      </strong>
      ：在拉普拉斯域中表示输入与输出之间的关系，便于频率分析。
     </li>
     <li>
      <strong>
       状态空间模型（State-Space Model）
      </strong>
      ：用一组一阶微分方程描述系统状态的演化，适合多输入多输出系统。
     </li>
    </ul>
    <p>
     这些模型为控制系统的分析和优化提供了数学基础。
    </p>
