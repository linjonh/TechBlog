    <h3>
     <a id="APT____advanced_package_Tool____5">
     </a>
     APT 》》 advanced package Tool 高级软件包工具
    </h3>
    <ol>
     <li>
      apt-get update
      <strong>
       更新软件包列表
      </strong>
     </li>
     <li>
      apt-get install
      <strong>
       安装软件包
      </strong>
     </li>
     <li>
      apt-get remove
      <strong>
       卸载软件包
      </strong>
     </li>
     <li>
      apt-get upgrade 升级有可用更新的系统（慎用）
     </li>
     <li>
      apt-cache search 搜索软件包
     </li>
     <li>
      apt-cache show 获取包信息
     </li>
     <li>
      apt-get remove --purge 删除包及配置文件 purge 是清洗意思
     </li>
     <li>
      apt-cache depends 了解使用依赖
     </li>
     <li>
      apt-cache rdepends 查看被那些包依赖
     </li>
     <li>
      apt-get build-dep 安装相关的编译环境
     </li>
     <li>
      apt-get source 下载源代码
     </li>
     <li>
      apt-get clean &amp;&amp; apt-get autoclean
      <strong>
       清理无用的包
      </strong>
     </li>
     <li>
      apt-get check 检查是否有损坏的依赖
     </li>
    </ol>
