    <h3>
     <a id="vscode___1">
     </a>
     vscode - 操作整理
    </h3>
    <h3>
     <a id="_3">
     </a>
     概述
    </h3>
    <p>
     不同的文本文件后缀，编辑时可能有不同要求。
     <br/>
     e.g. .bat文件，就要求用ansi格式保存(否则中文乱码)
     <br/>
     e.g. 行缩进要求用空格键代表TAB键，一个TAB键为4个空格。
     <br/>
     vscode通过配置文件，是可以自动设置这些的. 针对不同文本文件类型, 设置不同的配置，方便编辑这些文件。
    </p>
    <h3>
     <a id="_8">
     </a>
     笔记
    </h3>
    <h3>
     <a id="_9">
     </a>
     打开文件后，编码另存为
    </h3>
    <p>
     如果打开文件后，不确定文件编码是啥，可以另存一下靠谱。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/25903b8c20504369af75bff1441a2910.png">

        在vscode中，用gb2312保存的文件。再用记事本打开，模拟另存时，就能看到是ansi格式。
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="_16">
     </a>
     配置指定后缀的文件的语言模式
    </h3>
    <p>
     新建一个.abc的文件，用vscode打开
     <br/>
     不管是怎样的文本文件，打开后，在vscode的右下角都能看到默认的配置，左击这个配置。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c396c1cfc6d34e4cbaa5fd551156f12e.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5165f4e04b744e7a8961b32b16ae23e9.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/912736707beb401ca4f06d32e760b59e.png">
        <br/>
        一般我们打开一个文件，都是一个已知类型，或者我们知道他是啥文件。
        <br/>
        虽然我打开了.abc文件，但是我是要按照bat脚本文件来打开的，我就选Batch语言模式。
        <br/>
        vscode这么强大，一般的语言模式都有了。如果没有，也能选一个相近的语言模式来打开。
       </img>
      </img>
     </img>
    </p>
    <p>
     每种语言模式都有特定的规则，如果默认的规则不符合自己要求，可以将不同的规则配置到全局规则。
     <br/>
     那么以后在本机打开的这种文件，就按照我们自己定义的规则来(e.g. 缩进用空格，TAB键按下时，转为4个空格)。
    </p>
    <h4>
     <a id="__Batch_29">
     </a>
     语言模式配置 - Batch
    </h4>
    <p>
     bat文件的后缀为.bat, 默认打开就是Batch语言模式。如果不是，将语言模式选为 Batch
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9a31e47e9c594b6eb69bbaa74c4d40c7.png"/>

    <h3>
     <a id="vscode_77">
     </a>
     如果用vscode打开的文件没有显示回车，原因及处理
    </h3>
    <h4>
     <a id="vscode_78">
     </a>
     vscode启用了信任模式，需要信任工作区才行。
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e92d9dd5c06b419abd4d48f94f050ec9.png"/>
     <br/>
     如果看到打开的文档的上下提示条有"受限模式"的提示，就点击上图的红框处的按钮，进行工作区信任。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b6b6c09dddde44c3a6bb60bb1ef93660.png"/>
     <br/>
     直接信任此打开的文件，或者添加要信任的文件夹都行。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e723bc46be644abd88de6f00e9bf1b65.png"/>
     <br/>
     被信任的文件，就会应用设置，此时，回车就由插件显示出来了。
    </p>
