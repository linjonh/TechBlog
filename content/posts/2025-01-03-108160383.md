# 创建springboot项目时出现Selected Java version 11 is not supported by SDK (maximum 8)

创建springboot项目的时候出现这个问题

> Selected Java version 11 is not supported by SDK (maximum 8)

说明springboot项目不支持java Version11的版本。只需要把java version改成java version8就行了、
