    <h2>
     <strong>
      1、添加 Nginx 官方仓库
     </strong>
    </h2>
    <p>
     <code>
      epel-release
     </code>
     是 Extra Packages for Enterprise Linux 的仓库，包含了
     <code>
      nginx
     </code>
     等常用软件。
    </p>
    <pre><code class="hljs">sudo yum install -y epel-release
sudo yum install -y nginx</code></pre>
    <h2>
     2、
     <strong>
      手动添加 Nginx 仓库
     </strong>
    </h2>
    <p>
     如果
     <code>
      epel-release
     </code>
     不可用，可以手动添加 Nginx 官方仓库
    </p>
    <pre><code class="hljs">sudo vi /etc/yum.repos.d/nginx.repo</code></pre>
    <p>
     添加以下内容：
    </p>
    <pre><code class="hljs">[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true</code></pre>
    <p>
     保存并退出，然后运行
    </p>
    <pre><code class="hljs">sudo yum clean all
sudo yum makecache
sudo yum install -y nginx</code></pre>
