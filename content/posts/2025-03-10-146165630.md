    <p>
     对于 CentOS
    </p>
    <h2>
     <a id="_Docker__1">
     </a>
     安装 Docker 的前置条件
    </h2>
    <p>
     首先，需要安装一些必要的软件包， 对于 CentOS 7，可以使用以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum <span class="token function">install</span> <span class="token parameter variable">-y</span> yum-utils device-mapper-persistent-data lvm2
</code></pre>
    <h2>
     <a id="_Docker__9">
     </a>
     添加 Docker 仓库
    </h2>
    <p>
     设置 Docker 的官方仓库。对于 CentOS
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
</code></pre>
    <h2>
     <a id="_Docker_16">
     </a>
     安装 Docker
    </h2>
    <p>
     安装 Docker：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum <span class="token function">install</span> <span class="token parameter variable">-y</span> docker-ce docker-ce-cli containerd.io
</code></pre>
    <h2>
     <a id="_Docker__23">
     </a>
     启动 Docker 服务
    </h2>
    <p>
     启动 Docker 服务并设置开机启动：
     <br/>
     启动
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl start <span class="token function">docker</span>
</code></pre>
    <p>
     设置开机启动
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl <span class="token builtin class-name">enable</span> <span class="token function">docker</span>
</code></pre>
    <h2>
     <a id="_35">
     </a>
     验证安装
    </h2>
    <p>
     检查 Docker 是否正确安装并运行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token parameter variable">--version</span>
</code></pre>
    <h2>
     <a id="_43">
     </a>
     更换镜像源
    </h2>
    <p>
     检查路径 /etc/docker/ 有没有 daemon.json 文件
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">touch</span> /etc/docker/daemon.json
</code></pre>
    <p>
     编辑文件
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">vim</span> /etc/docker/daemon.json
</code></pre>
    <p>
     添加内容：
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">{<!-- --></span>
<span class="token string">"registry-mirrors"</span> <span class="token builtin class-name">:</span>
<span class="token punctuation">[</span>
<span class="token string">"https://mirrors.tencent.com/docker"</span>,
<span class="token string">"https://docker.m.daocloud.io"</span>,
<span class="token string">"https://noohub.ru"</span>,
<span class="token string">"https://huecker.io"</span>,
<span class="token string">"https://dockerhub.timeweb.cloud"</span>,
<span class="token string">"https://docker.rainbond.cc"</span>
<span class="token punctuation">]</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     查看docker
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> info
</code></pre>
    <p>
     重启
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl daemon-reload
<span class="token function">sudo</span> systemctl restart <span class="token function">docker</span> 
</code></pre>
