# python 查看变量内存地址

1.代码：

[Python id() 函数 | 菜鸟教程](https://www.runoob.com/python/python-func-id.html "Python id() 函数 | 菜鸟教程")

id()函数返回10进制地址

[Python hex() 函数 | 菜鸟教程](https://www.runoob.com/python/python-func-hex.html "Python hex() 函数 | 菜鸟教程")

hex()函数将10进制地址转为16进制

```
#!/usr/bin/python3

# id(变量)查看内存地址，10进制
# hex(10进制地址),把10进制转为16进制

a = 1
print(hex(id(a)))
# 0x7fb4b9f138

del a
a = 2
print(hex(id(a)))
# 0x7fb4b9f158

b = 2.0
print(hex(id(b)))
# 0x7fb417e378
```
