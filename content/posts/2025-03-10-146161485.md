    <h3>
     <a id="font_color__E8A87C_face___CSSfont_0">
     </a>
     <font color="#E8A87C" face="楷体">
      一、初识 CSS
     </font>
    </h3>
    <h4>
     <a id="font_color__F76C6C1_CSS_font_2">
     </a>
     <font color="#F76C6C">
      1, CSS 定义
     </font>
    </h4>
    <p>
     <font color="red">
      层叠样式表(Cascading Style Sheets，缩写为 CSS)
     </font>
     ，是一种 样式表 语言，用来描述 HTML 文档的呈现（美化内容）
    </p>
    <p>
     CSS 能够对网页中元素位置的排版进行
     <strong>
      像素级
     </strong>
     精确控制，实现美化页面的效果. 能够做到页面的样式和结构分离
    </p>
    <p>
     <img alt="image-20250225164123718" src="https://i-blog.csdnimg.cn/img_convert/20d93b735b511752f617208b68354317.png"/>
    </p>
    <h4>
     <a id="font_color__F76C6C2_font_10">
     </a>
     <font color="#F76C6C">
      2, 基本语法规范
     </font>
    </h4>
    <p>
     选择器 +{一条/N条声明)
    </p>
    <ul>
     <li>
      选择器决定针对谁修改(找谁)
     </li>
     <li>
      声明决定修改啥 (干啥)
     </li>
     <li>
      声明的属性是键值对，使用：区分键值对，使用：区分键和值
     </li>
    </ul>
    <p>
     <strong>
      书写规则：
     </strong>
     选择器{属性名：属性值}
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
     <span class="token comment">/*这里写的都是css,css通常写在style标签中*/</span>         
      <span class="token selector">p</span><span class="token punctuation">{<!-- --></span> <span class="token comment">/*p:选择器:查找标签*/</span>
         <span class="token comment">/*设置文字颜色变红*/</span>
         <span class="token comment">/* color: red; */</span>
         <span class="token property">color</span><span class="token punctuation">:</span>  <span class="token function">rgb</span><span class="token punctuation">(</span>172<span class="token punctuation">,</span>25<span class="token punctuation">,</span>44<span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token comment">/*字变大 px：像素 */</span>
        <span class="token property">font-size</span><span class="token punctuation">:</span> 30px<span class="token punctuation">;</span>
      <span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>

<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>这是一个p标签<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <blockquote>
     <p>
      注意：
     </p>
     <ul>
      <li>
       css 要写到
       <code>
        style
       </code>
       标签中(后面还会介绍其他写法)
      </li>
      <li>
       <code>
        style
       </code>
       标签可以放到页面任意位，一般放到
       <code>
        head
       </code>
       标签内
      </li>
      <li>
       CSS 使用 /* */ 作为注释.(使用 ctrl +/快速切换)
      </li>
     </ul>
    </blockquote>
    <h4>
     <a id="font_color__F76C6C3_CSS_font_43">
     </a>
     <font color="#F76C6C">
      3, CSS 引入
     </font>
    </h4>
    <p>
     <strong>
      ① 内部样式表：
     </strong>
     <font color="red">
      学习使用
     </font>
    </p>
    <ul>
     <li>
      CSS 代码在 style 标签里面，嵌入到 html 内部
     </li>
     <li>
      <strong>
       优点
      </strong>
      ：这样做能够让样式和页面结构分离.
     </li>
     <li>
      <strong>
       缺点
      </strong>
      ：分离的还不够彻底. 尤其是 css 内容多的时候.
     </li>
    </ul>
    <blockquote>
     <p>
      前面写的代码主要都是使用了这种方式. 实际开发中不常用.
     </p>
     <ul>
      <li>
       PS: 搜狗搜索中仍然保留着这种写法.
      </li>
     </ul>
     <p>
      <img alt="image-20250225164947394" src="https://i-blog.csdnimg.cn/img_convert/c62c825ea6b3dec390d920269e80d8d4.png"/>
     </p>
    </blockquote>
    <p>
     <strong>
      ② 行内样式表：
     </strong>
     <font color="red">
      配合 JS 使用
     </font>
    </p>
    <ul>
     <li>
      通过 style 属性, 来指定某个标签的样式.
     </li>
     <li>
      只适合于写简单样式. 只针对某个标签生效.
     </li>
     <li>
      <strong>
       缺点:
      </strong>
      不能写太复杂的样式，这种写法优先级较高, 会覆盖其他的样式
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
    <span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
   <span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token special-attr"><span class="token attr-name">style</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span><span class="token value css language-css"><span class="token property">color</span><span class="token punctuation">:</span>green</span><span class="token punctuation">"</span></span></span><span class="token punctuation">&gt;</span></span>想要生活过的去, 头上总得带点绿<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     效果如下：
    </p>
    <p>
     <img alt="image-20250225165421411" src="https://i-blog.csdnimg.cn/img_convert/736661c4251bd659853e82aad1bed4ce.png"/>
    </p>
    <ul>
     <li>
      此时可以看到 red 被覆盖了
     </li>
    </ul>
    <p>
     <strong>
      ③ 外部样式表：
     </strong>
     <font color="red">
      开发使用
     </font>
    </p>
    <ul>
     <li>
      <p>
       CSS 代码写在单独的 CSS 文件里面 (.css)
      </p>
     </li>
     <li>
      <p>
       在 HTML 使用
       <code>
        link
       </code>
       标签引入 CSS
      </p>
     </li>
     <li>
      <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>link</span> <span class="token attr-name">rel</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>stylesheet<span class="token punctuation">"</span></span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>[CSS文件路径]<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        优点:
       </strong>
       样式和结构彻底分离了.
      </p>
     </li>
     <li>
      <p>
       <strong>
        缺点:
       </strong>
       受到浏览器缓存影响, 修改之后 不一定 立刻生效
      </p>
     </li>
    </ul>
    <p>
     【案例】：
    </p>
    <pre><code class="prism language-html"><span class="token comment">&lt;!--创建 my.css文件--&gt;</span>
p{
    color: red;
}

<span class="token comment">&lt;!--创建 test.html文件--&gt;</span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>head</span><span class="token punctuation">&gt;</span></span>
<span class="token comment">&lt;!-- link 引入外部样式表，rel: 关系，样式表 --&gt;</span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>link</span> <span class="token attr-name">rel</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>stylesheet<span class="token punctuation">"</span></span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>./my.css<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>head</span><span class="token punctuation">&gt;</span></span>

<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>body</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>这是一个p标签<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>body</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     <font color="red">
      注意：不要忘记 link 标签调用 CSS, 否则不生效
     </font>
    </p>
    <p>
     <strong>
      关于缓存：
     </strong>
    </p>
    <blockquote>
     <ul>
      <li>
       这是计算机中一种常见的提升性能的技术手段.
      </li>
      <li>
       网页依赖的资源(图片/CSS/JS等)通常是从服务器上获取的。如果频繁访问该网站，那么这些外部资源就没必要反复从服务器获取. 就可以使用缓存先存起来(就是存在本地磁盘上了)。从而提高访问效率
      </li>
      <li>
       可以通过
       <code>
        ctrl + F5
       </code>
       强制刷新页面，强制浏览器重新获取 css 文件
      </li>
     </ul>
    </blockquote>
    <h4>
     <a id="font_color__F76C6C4_font_124">
     </a>
     <font color="#F76C6C">
      4, 代码风格
     </font>
    </h4>
    <p>
     <font color="#7F3CFF">
      样式风格
     </font>
    </p>
    <p>
     ① 紧凑风格
    </p>
    <pre><code class="prism language-html">p { color: red; font-size: 30px;}
</code></pre>
    <p>
     ② 展开风格
    </p>
    <pre><code class="prism language-html">p {
    color: red;
    font-size: 30px;
}
</code></pre>
    <p>
     <font color="#7F3CFF">
      样式大小写
     </font>
    </p>
    <p>
     虽然 CSS 不区分大小写, 我们开发时统一使用小写字母
    </p>
    <p>
     <font color="#7F3CFF">
      空格规范
     </font>
    </p>
    <ul>
     <li>
      冒号后面带空格
     </li>
     <li>
      选择器和
      <code>
       {
       <!-- -->
      </code>
      之间也有一个空格
     </li>
    </ul>
    <h3>
     <a id="font_color__E8A87C_face__font_154">
     </a>
     <font color="#E8A87C" face="楷体">
      二、选择器
     </font>
    </h3>
    <h4>
     <a id="font_color__7ED3211_font_156">
     </a>
     <font color="#7ED321">
      1. 基本概念
     </font>
    </h4>
    <p>
     <strong>
      选择器功能
     </strong>
    </p>
    <p>
     选中页面中指定的标签元素，来设置样式
    </p>
    <blockquote>
     <ul>
      <li>
       要先选中元素, 才能设置元素的属性.
      </li>
      <li>
       就好比 SC2, War3 这样的游戏, 需要先选中单位, 再指挥该单位行动.
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      选择器种类
     </strong>
    </p>
    <p>
     以下内容只是 CSS2 标准中支持的选择器, 在 CSS3 中还做出了一些补充, 咱们后面再说
    </p>
    <p>
     <font color="#7F3CFF">
      基础选择器：单个选择器构成的
     </font>
    </p>
    <ol>
     <li>
      标签选择器
     </li>
     <li>
      类选择器
     </li>
     <li>
      id 选择器
     </li>
     <li>
      通配符选择器
     </li>
    </ol>
    <p>
     <font color="#7F3CFF">
      复合选择器：把多种基础选择器综合运用起来
     </font>
    </p>
    <ol>
     <li>
      后代选择器
     </li>
     <li>
      子选择器
     </li>
     <li>
      并集选择器
     </li>
     <li>
      伪类选择器
     </li>
    </ol>
    <p>
     参考文档：
     <a href="https://www.w3school.com.cn/cssref/css_selectors.asp" rel="nofollow">
      w3school
     </a>
    </p>
    <h4>
     <a id="font_color__7ED3212_font_187">
     </a>
     <font color="#7ED321">
      2. 基础选择器
     </font>
    </h4>
    <h5>
     <a id="font_color__chocolate21_font_189">
     </a>
     <font color="chocolate">
      2.1 标签选择器
     </font>
    </h5>
    <p>
     **标签选择器：**使用 标签名 作为选择器 -&gt; 选中
     <strong>
      同名标签
     </strong>
     设置 相同的样式
    </p>
    <p>
     特点: 能快速为同一类型的标签都选择出来，但是不能差异化选择。相当于 选中所有的这个标签都生效css
    </p>
    <p>
     例如：p，h1，div，a，img…
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
<span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>猫<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>咬人<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span><span class="token punctuation">&gt;</span></span>君<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span><span class="token punctuation">&gt;</span></span>阿叶<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span><span class="token punctuation">&gt;</span></span>阿叶君<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="font_color__chocolate22_font_215">
     </a>
     <font color="chocolate">
      2.2 类选择器
     </font>
    </h5>
    <p>
     特点:
     <strong>
      差异化
     </strong>
     表示不同的标签，可以让多个标签的都使用同一个标签.
    </p>
    <p>
     步骤：
    </p>
    <ol>
     <li>
      <p>
       定义类选择器 -&gt; .类名
      </p>
     </li>
     <li>
      <p>
       使用类选择器 -&gt; 标签添加
       <strong>
        class= “类名”
       </strong>
      </p>
     </li>
    </ol>
    <pre><code class="prism language-css"><span class="token selector">&lt;style&gt;
    .red</span><span class="token punctuation">{<!-- --></span>
        <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
    <span class="token selector">.size</span><span class="token punctuation">{<!-- --></span>
        <span class="token property">font-size</span><span class="token punctuation">:</span> 66px<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
&lt;/style&gt;

&lt;p&gt;111&lt;/p&gt; &lt;!--黑色--&gt;

&lt;!-- 一个标签可以使用多个类名，用空号隔开就行 --&gt;
&lt;p class=<span class="token string">"red size "</span>&gt;222&lt;/p&gt; &lt;!--红色--&gt;
&lt;div class=<span class="token string">"red"</span>&gt;333&lt;/div&gt;
</code></pre>
    <p>
     <strong>
      注意：
     </strong>
    </p>
    <ul>
     <li>
      类名用
      <code>
       .
      </code>
      开头的
     </li>
     <li>
      一个类可以被多个标签使用, 一个标签也能使用多个类(多个类名要使用空格分割, 这种做法可以让 代码更好复用)
     </li>
     <li>
      类名自定义，不要用 纯数字或者中文，尽量用英文命名
     </li>
     <li>
      类名可以由数字、字母、下划线、中划线组成，但是不能用数字、中划线开头
     </li>
     <li>
      一个标签可以同时使用多个类名 ，用空号隔开就行，这样做可以把相同的属性提取出来，达到
      <strong>
       简化代码
      </strong>
      的效果.
     </li>
    </ul>
    <p>
     **开发习惯：**类名见名知意，多个单词可以用 - 连接
    </p>
    <h5>
     <a id="font_color__chococlate23_id_font_254">
     </a>
     <font color="chococlate">
      2.3 id 选择器
     </font>
    </h5>
    <p>
     和类选择器类似.
    </p>
    <ul>
     <li>
      CSS 中使用
      <code>
       #
      </code>
      开头表示 id 选择器
     </li>
     <li>
      id 选择器的值和 html 中某个元素的 id 值相同
     </li>
     <li>
      html 的元素 id 不必带
      <code>
       #
      </code>
     </li>
     <li>
      id 是
      <strong>
       唯一
      </strong>
      的, 不能被多个标签使用 (是和 类选择器 最大的区别)
     </li>
    </ul>
    <p>
     <strong>
      步骤：
     </strong>
    </p>
    <ul>
     <li>
      定义 id 选择器 -&gt;
      <strong>
       #id
      </strong>
      名
     </li>
     <li>
      使用 id 选择器 -&gt; 标签添加
      <strong>
       id= “id名”
      </strong>
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">&lt;style&gt;
    #blue</span><span class="token punctuation">{<!-- --></span>
        <span class="token property">color</span><span class="token punctuation">:</span> skyblue<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
&lt;/style&gt;

&lt;div id=<span class="token string">"blue"</span>&gt;这个div文字是蓝色&lt;/div&gt;
&lt;p id=<span class="token string">"blue"</span>&gt;111&lt;/p&gt;以后加js会容易出错
</code></pre>
    <p>
     <strong>
      注意：
     </strong>
    </p>
    <ol>
     <li>
      所有标签都有id属性
     </li>
     <li>
      id 属性值在一个页面中是唯一的，不可重复
     </li>
     <li>
      一个标签只能有一个id属性
     </li>
     <li>
      一个id选择器只能选中一个标签
     </li>
    </ol>
    <p>
     类比：姓名是类选择器, 可以重复。 身份证号码是 id 选择器，是唯一的.
    </p>
    <p>
     **id 选择器 VS 类选择器：**前者优先级更高
    </p>

    </p>

     </li>

    </ul>

     </font>
    </h5>
    <p>

    </p>
    <p>

     </a>
     <font color="chocolate">
      2.1 文本颜色
     </font>
    </h5>
    <p>
     <strong>
      认识 RGB
     </strong>
    </p>
    <p>
     我们的显示器是由很多很多的 “像素” 构成的. 每个像素视为一个点, 这个点就能反映出一个具体的颜色.
    </p>
    <ul>
     <li>
      我们使用 R (red), G (green), B (blue) 的方式表示颜色(色光三原色). 三种颜色按照不同的比例搭配, 就能混合出各种五彩斑斓的效果.
     </li>
     <li>
      计算机中针对 R, G, B 三个分量, 分别使用一个字节表示(8个比特位, 表示的范围是 0-255, 十六进制表示 为 00-FF).
     </li>
     <li>
      数值越大, 表示该分量的颜色就越浓. 255, 255, 255 就表示白色; 0, 0, 0 就表示黑色.
     </li>
    </ul>
    <p>
     <strong>
      设置文本颜色
     </strong>
    </p>
    <pre><code class="prism language-css"><span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token property">color</span><span class="token punctuation">:</span> #ff0000<span class="token punctuation">;</span>
<span class="token property">color</span><span class="token punctuation">:</span> <span class="token function">rgb</span><span class="token punctuation">(</span>255<span class="token punctuation">,</span> 0<span class="token punctuation">,</span> 0<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <blockquote>
     <p>
      鼠标悬停在 vscode 的颜色上, 会出现颜色选择器, 可以手动调整颜色.
     </p>
    </blockquote>
    <p>
     color 属性值的写法：
    </p>
    <ul>
     <li>
      预定义的颜色值(直接是单词)
     </li>
     <li>
      [最常用] 十六进制形式
     </li>
     <li>
      RGB 方式
     </li>
    </ul>
    <blockquote>
     <p>
      十六进制形式表示颜色, 如果两两相同, 就可以用一个来表示.
     </p>
     <p>
      <code>
       #ff00ff =&gt; #f0f
      </code>
     </p>
    </blockquote>
    <pre><code class="prism language-css"><span class="token selector">&lt;style&gt;
    .color</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
        <span class="token comment">/* color: rgb(255, 0, 0); */</span>
        <span class="token comment">/* color: #ff0000; */</span>
   <span class="token punctuation">}</span>
&lt;/style&gt;
&lt;div class=<span class="token string">"color"</span>&gt;这是一段话&lt;/div&gt;
</code></pre>

     <a id="font_color__chocolate25_font_989">

     <font color="chocolate">
      2.5 行高
     </font>
    </h5>
    <p>
     行高指的是上下文本行之间的基线距离.
    </p>
    <p>
     HTML 中展示文字涉及到这几个
     <strong>
      基准线
     </strong>
    </p>
    <ul>
     <li>
      顶线
     </li>
     <li>
      中线
     </li>
     <li>
      基线 (相当于英语四线格的倒数第二条线)
     </li>
     <li>
      底线
     </li>
    </ul>
    <p>
     <strong>
      内容区
     </strong>
     ：底线和顶线包裹的区域，即下图深灰色背景区域
    </p>
    <p>
     <img alt="image-20250226163611742" src="https://i-blog.csdnimg.cn/img_convert/8da4a904ca6f43d81116f5396cd7e48e.png"/>
    </p>
    <blockquote>
     <p>
      其实基线之间的距离 = 顶线间距离 = 底线间距离 = 中线间距离
     </p>
    </blockquote>
    <pre><code class="prism language-css"><span class="token property">line-height</span><span class="token punctuation">:</span> [值]
</code></pre>
    <p>
     <font color="#ED1C24">
      注意1: 行高 = 上边距 + 下边距 + 字体大小
     </font>
    </p>
    <ul>
     <li>
      上下边距是相等的, 此处字体大小是 16px, 行高 40px, 上下边距就分别是 12px
     </li>
    </ul>
    <p>
     【案例】
    </p>
    <pre><code class="prism language-css"><span class="token selector">&lt;style&gt;
    .line-height .one</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">line-height</span><span class="token punctuation">:</span> 40px<span class="token punctuation">;</span>
        <span class="token property">font-size</span><span class="token punctuation">:</span> 16px<span class="token punctuation">;</span>
   <span class="token punctuation">}</span>
&lt;/style&gt;

    <p>
     平铺：就是盒子太大，但是图片只有一张，那么就会复制图片填充这个盒子（默认平铺效果）
    </p>
    <pre><code class="prism language-css"><span class="token property">background-repeat</span><span class="token punctuation">:</span> [平铺方式]
</code></pre>
    <p>
     重要取值:
    </p>
    <ul>
     <li>
      <code>

      </code>
      : 平铺
     </li>
     <li>
      <code>
       no-repeat
      </code>
      : 不平铺，只显示一张背景图片，显示在盒子的左上角
     </li>
     <li>
      <code>
       repeat-x
      </code>
      : 水平平铺
     </li>
     <li>
      <code>
       repeat-y
      </code>
      : 垂直平铺
     </li>
    </ul>
    <p>
     默认是 repeat. 背景颜色和背景图片可以同时存在. 背景图片在背景颜色的上方.

    <h5>
     <a id="font_color__chocolate34_font_1156">
     </a>
     <font color="chocolate">
      3.4 背景位置
     </font>
    </h5>
    <p>
     由于背景图默认是在左上角的，但是我们可以通过这个来调整背景图的位置
    </p>
    <pre><code class="prism language-css"><span class="token property">background-position</span><span class="token punctuation">:</span> x<span class="token punctuation">,</span> y<span class="token punctuation">;</span>
</code></pre>
    <p>
     作用：修改图片的位置，参数有三种风格:
    </p>
    <ol>
     <li>
      <strong>
       方位名词
      </strong>
      : (top, left, right, bottom)
     </li>
     <li>
      <strong>
       精确单位
      </strong>
      : 坐标或者百分比(以左上角为原点) ，数字 +px（正负都可以）
     </li>
     <li>
      <strong>
       混合单位
      </strong>
      : 同时包含方位名词和精确单位
     </li>
    </ol>
    <pre><code class="prism language-css"><span class="token selector">&lt;style&gt;
    .bgp .one</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">background-image</span><span class="token punctuation">:</span> <span class="token url"><span class="token function">url</span><span class="token punctuation">(</span>rose.jpg<span class="token punctuation">)</span></span><span class="token punctuation">;</span>
        <span class="token property">height</span><span class="token punctuation">:</span> 500px<span class="token punctuation">;</span>
        <span class="token property">background-repeat</span><span class="token punctuation">:</span> no-repeat<span class="token punctuation">;</span>
        <span class="token property">background-color</span><span class="token punctuation">:</span> purple<span class="token punctuation">;</span>
        
        <span class="token property">background-position</span><span class="token punctuation">:</span> center bottom<span class="token punctuation">;</span>
        <span class="token property">background-position</span><span class="token punctuation">:</span> 50px -100px<span class="token punctuation">;</span>
        <span class="token property">background-position</span><span class="token punctuation">:</span> 50px center<span class="token punctuation">;</span>
   <span class="token punctuation">}</span>
&lt;/style&gt;

&lt;div class=<span class="token string">"bgp"</span>&gt;
    &lt;div class=<span class="token string">"one"</span>&gt;背景居中&lt;/div&gt;
&lt;/div&gt;
</code></pre>
    <p>
     <strong>
      注意
     </strong>
    </p>
    <ul>
     <li>
      如果参数的两个值都是方位名词, 则前后顺序无关. (top left 和 left top 等效)
     </li>
     <li>
      如果只指定了一个方位名词, 则第二个默认居中. (left 则意味着水平居中, top 意味着垂直居中. )
     </li>
     <li>
      如果参数是精确值, 则的的第一个肯定是 x , 第二个肯定是 y. (100 200 意味着 x 为 100, y 为 200)
     </li>
     <li>
      如果参数是精确值, 且只给了一个数值, 则该数值一定是 x 坐标, 另一个默认垂直居中. 如果参数是混合单位, 则第一个值一定为 x, 第二个值为 y 坐标. (100 center 表示横坐标为 100, 垂 直居中)
     </li>
    </ul>
    <p>
     <strong>
      关于坐标系
     </strong>
    </p>
    <p>
     计算机中的平面坐标系, 一般是左手坐标系(和高中数学上常用的右手系不一样. y轴是往下指的).
    </p>
    <p>
     <img alt="image-20250226180412081" src="https://i-blog.csdnimg.cn/img_convert/cb9446f32a032be901b45a18bd6d655f.png"/>

     <li>

        <span class="token property">background-image</span><span class="token punctuation">:</span> <span class="token url"><span class="token function">url</span><span class="token punctuation">(</span>rose.jpg<span class="token punctuation">)</span></span><span class="token punctuation">;</span>
        <span class="token property">background-repeat</span><span class="token punctuation">:</span> no-repeat<span class="token punctuation">;</span>
        <span class="token property">background-position</span><span class="token punctuation">:</span> center<span class="token punctuation">;</span>
        <span class="token property">background-size</span><span class="token punctuation">:</span> contain<span class="token punctuation">;</span>
   <span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>

<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>bgs<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>one<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>背景尺寸<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     注意体会
     <code>
      contain
     </code>
     和
     <code>

    <h4>
     <a id="font_color__7ED3216_font_1390">
     </a>
     <font color="#7ED321">
      6. 元素的显示模式
     </font>
    </h4>
    <p>
     网页中的 HTML 标签非常多，在不同地方会用到不同类型的标签，了解他们的特点可以更好的布局我们的网页。其中元素显示模式就是元素(标签)以什么方式进行显示，比如
    </p>
    <div>
     自己占一行，比如一行可以放多个
     <code>
      &lt;span&gt;
     </code>
     等。
    </div>
    <p>
    </p>
    <ul>
     <li>
      HTML 元素一般分为块元素和行内元素两种类型。有的资料也有行内块元素的说法。
     </li>
    </ul>
    <h5>
     <a id="font_color__chocolate61_font_1396">
     </a>
     <font color="chocolate">
      6.1 块级元素
     </font>
    </h5>
    <ul>
     <li>
      h1 - h6、p、 div、 ul、 ol、 li …，其中 其中
      <div>
       标签是最典型的 块元素
      </div>
     </li>
    </ul>
    <p>
     <strong>
      特点:
     </strong>
    </p>
    <ul>
     <li>
      独占一行
     </li>
     <li>
      高度, 宽度, 内外边距, 行高都可以控制.
     </li>
     <li>
      宽度默认是父级元素宽度的 100% (和父元素一样宽)
     </li>
     <li>
      是一个
      <strong>
       容器(盒子)
      </strong>
      ， 里面可以放
      <strong>
       行内
      </strong>
      和
      <strong>
       块级
      </strong>
      元素
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
<span class="token selector">.demo1 .parent</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">width</span><span class="token punctuation">:</span> 500px<span class="token punctuation">;</span>
    <span class="token property">height</span><span class="token punctuation">:</span> 500px<span class="token punctuation">;</span>
    <span class="token property">background-color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token selector">.demo1 .child</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">/* 不写 width, 默认和父元素一样宽 */</span>
    <span class="token comment">/* 不写 height, 默认为 0 (看不到了) */</span>
    <span class="token property">height</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span>
    <span class="token property">background-color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>

<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>demo1<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>parent<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
        <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>child<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>child1<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
        <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>child<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>child2<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     注意:
    </p>
    <ul>
     <li>
      文字类的元素内不能使用块级元素
     </li>
     <li>
      p 标签主要用于存放文字, 内部不能放块级元素, 尤其是 div
     </li>
     <li>
      同理，
      <code>
       &lt;h1&gt;~&lt;h6&gt;
      </code>
      等都是文字类块级标签，里面也不能放其他块级元素
     </li>
    </ul>
    <p>
     比如：
    </p>
    <pre><code class="prism language-css">&lt;body&gt;
    &lt;p&gt;&lt;div&gt;蛤蛤&lt;/div&gt;&lt;/p&gt;
&lt;/body&gt;
</code></pre>
    <ul>
     <li>
      从上面的示例可以看到当把块级元素
      <code>
       &lt;div&gt;
      </code>
      放到文字内的元素
      <code>
       &lt;p&gt;
      </code>
      中时，浏览器解析时会把它单独拿出来，同时设置的样式也不能生效。
     </li>
    </ul>
    <h5>
     <a id="font_color__chocolate62_font_1446">
     </a>
     <font color="chocolate">
      6.2 行内元素/内联元素
     </font>
    </h5>
    <p>
     常见的元素:
    </p>
    <ul>
     <li>
      a、strong、b、em、i、del、s、ins、u、span，其中
      <code>
       &lt;span&gt;
      </code>
      标签是最典型的行内元素。有的地方也将行内元素称为内联元素。
     </li>
    </ul>
    <p>
     特点:
    </p>
    <ul>
     <li>
      不独占一行，一行可以显示多个
     </li>
     <li>
      设置高度、宽度、行高无效
     </li>
     <li>
      <strong>
       左右外边距
      </strong>
      有效(上下无效)，
      <strong>
       内边距
      </strong>
      有效,
     </li>
     <li>
      默认宽度就是本身的内容
     </li>
     <li>
      行内元素只能容纳文本和其他行内元素，不能放块级元素
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
    <span class="token selector">.demo2 span</span> <span class="token punctuation">{<!-- --></span>
        <span class="token property">width</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span>
        <span class="token property">height</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span>
        <span class="token property">background-color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>

<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>demo2<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>span</span><span class="token punctuation">&gt;</span></span>child1<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>span</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>span</span><span class="token punctuation">&gt;</span></span>child2<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>span</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     <strong>
      注意:
     </strong>
    </p>
    <blockquote>
     <ul>
      <li>
       <code>
        a
       </code>
       标签中不能再放a 标签(虽然 chrome 不报错, 但是最好不要这么做).
      </li>
      <li>
       特殊情况下：
       <code>
        a
       </code>
       标签里可以放块级元素，但是更建议先把
       <code>
        a
       </code>
       转换成块级元素.
      </li>
     </ul>
    </blockquote>
    <p>
     <font color="blue">
      行内元素和块级元素的区别
     </font>
    </p>
    <ul>
     <li>
      块级元素独占一行，行内元素不独占一行
     </li>
     <li>
      块级元素可以设置宽高，行内元素不能设置宽高,
     </li>
     <li>
      块级元素四个方向都能设置内外边距，行内元素垂直方向不能设置
     </li>
    </ul>
    <h5>
     <a id="font_color__chocolate63_font_1490">
     </a>
     <font color="chocolate">
      6.3 行内块元素
     </font>
    </h5>
    <p>
     在行内元素中有几个特殊的标签例如
     <code>
      &lt;img/&gt;
     </code>
     、
     <code>
      &lt;input/&gt;
     </code>
     、
     <code>
      &lt;td&gt;
     </code>
     ，它们同时具有块元素和行内元素的特点。 称它们为行内块元素。
    </p>
    <p>
     <strong>
      行内块元素特点
     </strong>
    </p>
    <ol>
     <li>
      <p>
       和相邻行内元素(行内块)在一行上，但是他们之间会有空白缝隙。一行可以显示多个(行内元素特点)。
      </p>
     </li>
     <li>
      <p>
       默认宽度就是它本身内容的宽度(行内元素特点)。
      </p>
     </li>
     <li>
      <p>
       高度，行高、外边距以及内边距都可以控制(块级元素特点)。
      </p>
     </li>
    </ol>
    <p>
     【案例】
    </p>

     <font color="chocolate">
      6.4 改变显示模式
     </font>
    </h5>
    <p>
     特殊情况下，我们需要元素模式的转换，简单理解: 一个模式的元素需要另外一种模式的特性。比如想要增加链接
     <code>
      &lt;a&gt;
     </code>
     的触发范围
    </p>
    <p>
     使用 display 属性可以修改元素的显示模式，可以把 div 等变成行内元素,也可以把 a,span 等变成块级元素
    </p>
    <ul>
     <li>
      <code>
       display:block
      </code>
      改成块级元素[常用]
     </li>
     <li>
      <code>
       display:in1ine
      </code>
      改成行内元素[很少用]
     </li>
     <li>
      <code>
       display:inline-block
      </code>
      改成行内块元素
     </li>
    </ul>
    <p>
     【案例】
    </p>
