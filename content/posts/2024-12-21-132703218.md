     <p style="text-align:justify;">
      怎么回事？突然间 Windows 12 就来了，还是网页版的？
     </p>
     <p style="text-align:justify;">
      <img alt="9c4cebc32b0535ebbe07063f1f863469.png" src="https://i-blog.csdnimg.cn/blog_migrate/31eba24d8b677f1f8829ab21c789a81a.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      仔细一瞧，原来这是一位开发者看到 Windows 12 概念版（Power by PowerPoint）后深受启发，因而才决定做一个 Windows12 网页版：
     </p>
     <p style="text-align:justify;">
      “此项目基于目前 Windows 版本创造，与微软的 Windows 12 正式版本不一致。此项目绝不附属于微软，且不应与微软操作系统或产品混淆，这也不是 Windows365 cloud PC。”
     </p>
     <p style="text-align:justify;">
      虽然这个网页与 Windows 12 正式版并无关系，但正如项目作者所说，创建「Windows 12 网页版」是希望让用户在网络上预先体验 Windows 12，因此基本上该有的都有了。
     </p>
     <p style="text-align:justify;">
      那么接下来就让我们来“超前体验”一下 Windows12 网页版吧！
     </p>
     <blockquote>
      <p style="text-align:justify;">
       Windows12 网页版体验地址：https://tjy-gitnub.github.io/win12/desktop.html
      </p>
     </blockquote>
     <p style="text-align:justify;">
      <img alt="2152d06507588f9dc49db4acece0df5b.png" src="https://i-blog.csdnimg.cn/blog_migrate/3321e2d8d233d9250c9512df0b6a6504.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       “超前体验” Windows12 ！
      </strong>
     </p>
     <p style="text-align:justify;">
      在最初决定创建这个 Windows 12 网页版时，项目作者谭景元大致拟定了一个“前景规划”：
     </p>
     <p style="text-align:center;">
      <img alt="ba61c1df1d4cd12a481d63a7e0fa911e.png" src="https://i-blog.csdnimg.cn/blog_migrate/b0b477f37974c9d8c0a491857154139b.png"/>
     </p>
     <p style="text-align:justify;">
      从路径规划前的打勾状态来看，这一年来谭景元已陆续实现其计划中的大部分功能。虽与 Windows 12 正式版不同，但他使用了标准网络技术，如 HTML，CSS 和 JS，在网络上对操作系统进行模拟并创新。
     </p>
     <p style="text-align:justify;">
      据作者介绍，Windows 12 网页版是一个开源项目，采用 EPL V2.0 开源许可，根据该许可用户可对项目进行传播、分发、修改以及二次发布，但必须注明源码来源，包括作者和项目链接等，并使用相同的协议开源。
     </p>
     <p style="text-align:justify;">
      下面来看看 Windows 12 网页版的实际效果。
     </p>
     <p style="text-align:justify;">
      （1）开始菜单
     </p>
     <p style="text-align:justify;">
      <img alt="bc8689156d4e5cf2b7ff25659c860c94.png" src="https://i-blog.csdnimg.cn/blog_migrate/74e32ff4a858a419cb1dc229be2a251f.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      与当前的 Windows 11 相比，Windows 12 网页版的界面变化不小，开始菜单中的布局也有明显变化。
     </p>
     <p style="text-align:justify;">
      （2）丰富应用
     </p>
     <p style="text-align:justify;">
      <img alt="35773d7bf190156d1d086d63d3ef03cf.png" src="https://i-blog.csdnimg.cn/blog_migrate/4bd7ab1b1e87d83575469331379bb30c.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      <img alt="8664afcd9c6717d19e5f006afaf393b7.png" src="https://i-blog.csdnimg.cn/blog_migrate/819dd9a48418f20fc1ad0542f8d3918b.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      虽然只是用于体验的网页版，但它依旧拥有不少的基础应用：计算机、相机、文件资源管理器、Python Editor、Python 3.10.2……尝试了一下，不少应用都能在其中使用。
     </p>
     <p style="text-align:justify;">
      （3）AI Copilot
     </p>
     <p style="text-align:justify;">
      <img alt="2fc70d27404f389aa8d1d9ad64c6ad56.png" src="https://i-blog.csdnimg.cn/blog_migrate/ff92cb580d0bfd89266ee22cb588ebd3.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      不仅如此，在今年席卷全球的 AI 热潮下，作者还在 v7.0.0 更新中，给 Windows 12 网页版新增了个 AI 助手。
     </p>
     <p style="text-align:justify;">
      不过有一点需要明确，由于该 Windows 12 网页版仅用于体验，因此许多功能并不能真实应用至你的电脑当中。例如搜索只能搜系统中的虚拟文件，新闻部分只有作者虚构出来的新闻，设置页面的许多选项目前也不能点击，后续作者可能会持续更新完善。
     </p>
     <p style="text-align:justify;">
      <img alt="e26e108c3f487e83cbd072393ec30a03.png" src="https://i-blog.csdnimg.cn/blog_migrate/62d7fb03d717fef3df070987cf0cef4d.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      <img alt="cea6a7b282b87f057dcc4ed68484f79e.png" src="https://i-blog.csdnimg.cn/blog_migrate/8d3cd676d4a10296d873729bb1d77bab.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      在实现 Windows 12 的基本功能后，谭景元进一步做了一些畅想，目前也实现了几个：
     </p>
     <p style="text-align:center;">
      <img alt="e94e0f360b3afa929600d4d93d525c8c.png" src="https://i-blog.csdnimg.cn/blog_migrate/666c350db9a323483f467ef88286cd06.png"/>
     </p>
     <p style="text-align:justify;">
      总体来说，这个 Windows 12 网页版从精美的 UI 设计、丰富的功能以及丝滑的动画等方面来看，体验感还不错，因此该项目开源至 GitHub 刚满一年，已获得 1300+ Star（注：作者提醒，目前移动端适配不太完善，桌面版网站还“凑合着用”）。
     </p>
     <p style="text-align:center;">
      <img alt="dc0cd53b0ea7e25ded68b7b183f7ba21.png" src="https://i-blog.csdnimg.cn/blog_migrate/f84a4906ba2cb016a8fc00a2f140403a.png"/>
     </p>
     <p style="text-align:justify;">
      按理说，接下来这个项目只要继续进行功能更新，自然会吸引更多人前来体验网页版 Windows 12，但最近项目作者谭景元突然宣布要请个“长假”——因为他初三了。
     </p>
     <p style="text-align:center;">
      <img alt="ff5af79f518f74cc532092d6cffec99b.png" src="https://i-blog.csdnimg.cn/blog_migrate/1f10297f54dd1fc59bf3aa192f5b50f8.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      是的，你没看错，这个 Windows 12 网页版的作者谭景元，是国内一名刚上初三的中学生！
     </p>
     <p style="text-align:justify;">
      <img alt="25e96ef9456f20ac5339e02a7a897e53.png" src="https://i-blog.csdnimg.cn/blog_migrate/97d5b72127afc60098832aa24a708bf5.png"/>
     </p>
     <p style="text-align:justify;">
      <strong>
       Windows 12 网页版的背后，是一名 14 岁的初中生
      </strong>
     </p>
     <p style="text-align:justify;">
      从谭景元创建的个人主页「星源的网站」来看，他出生于 2009 年 5 月，其小学和目前就读的初中都在成都，而目前刚上初三的他，就已曾获得两个重量级奖项：
     </p>
     <ul>
      <li>
       <p style="text-align:justify;">
        CSP 普及组一等奖（CSP 是 CCF 面向社会非专业人士推出非专业级别的能力认证，考核内容均涉及算法和编程）
       </p>
      </li>
      <li>
       <p style="text-align:justify;">
        蓝桥杯国赛三等奖（蓝桥杯全国软件和信息技术专业人才大赛，是由工业和信息化部人才交流中心举办的国内最大的信息技术竞赛）。
       </p>
      </li>
     </ul>
     <p style="text-align:justify;">
      <img alt="bdf81ea3be3cdd38c8911e780411379a.png" src="https://i-blog.csdnimg.cn/blog_migrate/4656d41f1ed9afbecc87d3734c4afeb6.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      在他的个人主页中，谭景元介绍道，其擅长安装各种系统、把各种软件都换成预览版、剪视频（B 站主页：https://space.bilibili.com/2010692096）以及“做作业”（？？？），并精通各种语言：
     </p>
     <p style="text-align:justify;">
      <img alt="f6aa743a4841ce88537ea77942a5c653.png" src="https://i-blog.csdnimg.cn/blog_migrate/fdf62d80d58698800131f72a95254396.png" width="1200"/>
     </p>
     <p style="text-align:justify;">
      目前谭景元所做的项目，除了 Windows 12 网页版，还有另外两个：
     </p>
     <ul>
      <li>
       <p style="text-align:justify;">
        尼科夫在线工具：“一个专用于坑人的网站，在注册过程中设置了重重关卡，无人能够成功。”
       </p>
      </li>
      <li>
       <p style="text-align:justify;">
        Tapple 滚动检测：“一个滚动监测样式库，灵感来源于苹果官网，本网站就是使用的此样式库。”
       </p>
      </li>
     </ul>
     <p style="text-align:justify;">
      实际上在今年 5 月，也就是谭景元初二下学期时，他就已经发布过公告，宣布要暂别这个项目一段时间，以专心学习备战中考：“我...呃...立志要考七林，从今开始要好好学习了。所以我将有很长一段时间因太忙而无暇顾及项目，对此表示抱歉 (初三上可能还有时间，在过年之后就直接甩手了。”
     </p>
     <p style="text-align:justify;">
      不过虽然短时间内不会再有什么功能更新，但谭景元还是将项目的维护工作进行了交接：“目前暂定将合作开发权限（push 权限）交给 @User782Tec 以在期间维护项目，在此提前表示非常感谢（如有意见欢迎提出）。”所以，期间开发者们体验过后如对该项目有任何意见或建议，可以在 GitHub 上提交 issues。
     </p>
     <p style="text-align:center;">
      <img alt="cb82af26283e3032d9f25c51f2da475b.png" src="https://i-blog.csdnimg.cn/blog_migrate/c6ce38f92d3c359e7d30746342663f95.png"/>
     </p>
     <p style="text-align:justify;">
      那么最后，让我们预祝“星源”谭景元明年中考顺利，考上心仪高中，也期待着他的回归~
     </p>
