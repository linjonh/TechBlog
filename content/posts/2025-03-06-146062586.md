    <h4>
     <a id="_0">
     </a>
     一、问题描述以及产生原因
    </h4>
    <ul>
     <li>
      问题描述：当
      <code>
       el-table
      </code>
      表格有横向滚动条和纵向滚动条，把横向滚动条拉到最右边，表格的表头会和内容错位（表头和内容列不对齐）
     </li>
     <li>
      问题产生原因：在
      <code>
       el-table
      </code>
      有纵向滚动条时，
      <mark>
       el-table__body-wrapper + 纵向滚动条的宽度是100%
      </mark>
      ，故表格
      <code>
       内容区域宽度不足100%，而表头el-table__header-wrapper的宽度仍为100%
      </code>
      ，表格内容实际宽度小于表头，因此造成错位
     </li>
    </ul>
    <h4>
     <a id="_3">
     </a>
     二、解决问题
    </h4>
    <ul>
     <li>
      思路：给表格表头的宽度设置和表格内容一样即可
      <code>
       100% - 纵向滚动条宽度
      </code>
     </li>
     <li>
      代码实现
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">::v-deep</span> <span class="token punctuation">{<!-- --></span>
  <span class="token selector">.el-table__header-wrapper</span> <span class="token punctuation">{<!-- --></span>
  	// 这里我设置的纵向滚动条宽度为8px
    <span class="token property">width</span><span class="token punctuation">:</span> <span class="token function">calc</span><span class="token punctuation">(</span>100% - 8px<span class="token punctuation">)</span> 
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
