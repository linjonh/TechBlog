    <h3 id="title-c9x-7ci-vqt">
     什么是魔笔
    </h3>
    <p>
     介绍什么是魔笔低代码应用开发平台。
    </p>
    <p>
     魔笔是一款面向全端（Web、H5、全平台小程序、App）场景的模型驱动低代码开发平台，提供一站式的应用全生命周期管理，包括可视化开发、发布、运维。基于云原生架构的弹性扩展和低成本运维服务，帮助客户高效地解决应用研发、迭代、运维的问题
     <a href="https://developer.aliyun.com/article/1332713" rel="nofollow" title="1">
      1
     </a>
     。
    </p>
    <p>
     魔笔是为
     <strong>
      专业开发者
     </strong>
     量身打造的
     <strong>
      AI 原生化
     </strong>
     的
     <strong>
      低代码应用开发平台
     </strong>
     ，集成了
     <strong>
      通义千问大模型，
     </strong>
     让开发更智能、应用更智能。魔笔提供一站式的应用全生命周期管理，包括可视化开发、发布、运维；基于
     <strong>
      云原生架构
     </strong>
     的弹性扩展和低成本运维服务，帮助开发者高效地解决应用研发、迭代、运维的问题。
    </p>
    <h4>
     易用性
    </h4>
    <p>
     魔笔提供了可视化的应用搭建工具，无需编写代码，只需拖拽组件、配置属性等方式，即可完成应用的搭建。同时，魔笔提供了丰富的组件库和模板，可以满足用户的不同需求
     <a href="https://developer.aliyun.com/article/1260351" rel="nofollow" title="3">
      3
     </a>
     。
    </p>
    <h4>
     效率
    </h4>
    <p>
     相较于传统的开发方式，使用魔笔进行应用搭建可以大大降低开发成本和开发周期，提高开发效率。同时，魔笔提供了应用动态更新功能，可以实时更新应用内容，提高了应用的可维护性
     <a href="https://developer.aliyun.com/article/1260351" rel="nofollow" title="3">
      3
     </a>
     。
    </p>
    <h4>
     稳定性
    </h4>
    <p>
     使用魔笔进行应用开发时，平台的稳定性非常高，无论是在应用搭建、环境管理、应用发布和版本管理等方面，都非常稳定可靠
     <a href="https://developer.aliyun.com/article/1260351" rel="nofollow" title="3">
      3
     </a>
     。
    </p>
    <h4>
     扩展性
    </h4>
    <p>
     魔笔提供了物料集成和API集成功能，可以轻松实现应用的功能扩展。同时，魔笔还支持多端应用生成，可以满足用户在不同设备上的使用需求
     <a href="https://developer.aliyun.com/article/1260351" rel="nofollow" title="3">
      3
     </a>
     。
    </p>
    <p id="e3e85db34043k">
     您可以通过阅读本文，快速了解在魔笔低代码应用开发平台中构建应用的流程。
    </p>
    <p>
     <strong>
      模板市场
     </strong>
     <br/>
     魔笔内置了适用于各行业、场景和应用类型的应用模板。通过使用这些模板，您可以更轻松地编辑和发布应用，迅速启动项目。

    <h3 id="ff14e9bc71pjo">
     逻辑开发
    </h3>
    <p id="c281b55592gyb">
     逻辑开发是指开发者通过可视化配置或编写代码，描述应用的交互流程和行为逻辑，使其能够根据用户输入（例如：点击页面上某个按钮）或其他一些系统事件（例如：调用某个后端接口并成功返回了数据），完成相应的处理与响应（例如：刷新页面上某个组件，并弹出消息提示）。
    </p>
    <p id="6b9f91c3d9ah9">
     在魔笔低代码开发平台中，大部分常规逻辑都可以通过简单的可视化配置方式完成，从而大幅提升开发者的搭建效率，同时降低人员沟通与维护成本。对于更复杂和定制的业务逻辑，魔笔也提供了表达式、前端函数等开发者所熟悉的代码级能力，在效率提升基础上进一步满足开发者的灵活性和可控性需求。
    </p>
    <h3 id="1a032e05829y1">
     创建数据库集成
    </h3>
    <h4 id="30393796afp0f">
     1. 创建数据库集成
    </h4>
    <ol id="3a174b6b76uma">
     <li id="a314cd8babnnh">
      <p id="0263a4a49c2hk">
       登录魔笔平台，进入管控台。
      </p>
     </li>
     <li id="d6b79763c6dn4">
      <p id="5e5709c2f9zbr">
       导航至“资源”-&gt;“集成”，点击“创建集成”。
      </p>
     </li>
     <li id="627a2de95er5p">
      <p id="24a531a59fuzi">
       选择“数据库类型的集成”（当前支持 MySQL 和 PostgreSQL），并填写集成名称及描述。
      </p>
     </li>
     <li id="804b5a8a56lvh">
      <p id="26a9351d379ps">
       配置不同环境的数据库连接信息，包括
       <code>
        host
       </code>
       、
       <code>
        port
       </code>
       、数据库名称、用户名和密码。自定义参数中支持使用
       <code>
        serverTimezone
       </code>
       指定数据库时区，例如使用
       <code>
        GMT%2B8
       </code>
       指定东八区。
      </p>
      <p class="img-center">
       <img alt="image" height="133" id="5351100a3a3g9" src="https://i-blog.csdnimg.cn/img_convert/129c8dac49e8a3a2ea3a28510a26165b.png" width="439"/>
      </p>
      <p>
       <strong>
        说明
       </strong>
      </p>
      <p id="5b66c56beeyys">
       自定义参数需要使用 URL 编码，例如
       <code>
        GMT+8
       </code>
       需要使用
       <code>
        GMT%2B8
       </code>
       。
      </p>
     </li>
     <li id="b1f9300c364hv">
      <p id="32063e33794z0">
       若账号下已有阿里云 RDS 数据库，可选择“从当前云账号录入”，快捷选择已有的数据库资源。
      </p>
     </li>
    </ol>
    <p id="9a619f1cc61jj">
    </p>
    <p class="img-center">
     <img alt="image" height="900" id="86779fe6femdb" src="https://i-blog.csdnimg.cn/img_convert/46d673fac5ace1c3a9af585e1b4c4da8.png" width="889"/>
    </p>
    <ol id="8d2e4530b8g56">
     <li id="c143943faeop8">
      <p id="d26df42102yfa">
       点击“确定”以创建数据库集成。
      </p>
     </li>
    </ol>
    <h4 id="17291adc048nl">
     2. 在应用内部创建集成操作
    </h4>
    <ol id="3b34ff2869ubd">
     <li id="e13f0d4cccqxd">
      <p id="9045229c4c4mw">
       打开“代码”面板，点击对应作用域的“+”后选中“集成操作”。
      </p>
      <p id="dfc3408d773lv">
      </p>
      <p class="img-center">
       <img alt="image" height="660" id="ca49cdc198lym" src="https://i-blog.csdnimg.cn/img_convert/483f9c81ed3e5d5290693056fa4fa5d3.png" width="714"/>
      </p>
     </li>
     <li id="cf5ce9dea000z">
      <p id="9b24a770e1skj">
       选择之前创建的数据库集成作为资源，输入 SQL 查询语句后，根据需求，选择集成操作的触发方式。
      </p>
     </li>
    </ol>
