    <h3>
     一、游戏介绍
    </h3>
    <p>
     <strong>
      1、历史背景
     </strong>
     <br/>
     数字华容道脱胎于传统华容道，后者源自三国时期“曹操败走华容道”的故事。传统玩法是通过移动不同形状的木块，帮助“曹操”从出口逃脱。而数字华容道将棋子替换为数字，目标是通过滑动方块，将乱序的数字按1至N的顺序排列整齐（例如4×4棋盘需排列1-15，留一空格作为移动空间）。
    </p>
    <p>
     <strong>
      2、基本规则
     </strong>
    </p>
    <ul>
     <li>
      棋盘分为3×3、4×4、5×5等不同难度等级，数字范围随棋盘大小递增。
     </li>
     <li>
      每次只能移动与空格相邻的数字方块，通过滑动调整顺序，最终达成从左到右、从上到下的数字连贯性
     </li>
    </ul>
    <h3>
     二、代码生成游戏盘面
    </h3>
    <p>
     <strong>
      1、实现思路：
     </strong>
    </p>
    <p>
     1.按游戏难度生成一个目标数字盘面；
    </p>
    <p>
     2.随机打乱整个盘面的数字；
    </p>
    <p>
     3.判断盘面可解性，无解则重新生成直到有解为止。
    </p>
    <p>
     <strong>

    <pre><code class="language-TypeScript">/** 难度：3x3、4x4、5x5 */
const level = ref(4);
/** 数字盘面 */
const list = ref([]);

/**
 * 随机打乱数组
 * @param {number[]} array 要打乱的数组
 */
function shuffleArray(array: number[]) {
	for (let i = array.length - 1; i &gt; 0; i--) {
		const j = Math.floor(Math.random() * (i + 1));
		[array[i], array[j]] = [array[j], array[i]];
	}
	return array;
}

/**
 * 判断数字华容道的可解性
 * @param {number[]} array 数字盘面
 * @param {number} row 盘面行数
 */
function isSolvable(array: number[], row: number = 4) {
	// 去掉空格（0)
	const puzzles = array.filter(item =&gt; item &gt; 0);
	// 逆序数：例如，在序列 [2, 3, 1] 中，2 和 1 构成一个逆序对，3 和 1 也构成一个逆序对。
	let n = 0;
	for (var i = 0; i &lt; puzzles.length; i++) {
		for (var j = i + 1; j &lt; puzzles.length; j++) {
			if (puzzles[i] &gt; puzzles[j]) {
				n++;
			}
		}
	}
	/**
	 * 若格子列数为奇数，则逆序数必须为偶数；
	 * 若格子列数为偶数，且逆序数为偶数，则当前空格所在行数与初始空格所在行数的差为偶数；
	 * 若格子列数为偶数，且逆序数为奇数，则当前空格所在行数与初始空格所在行数的差为奇数
	 */
	if (level.value % 2 !== 0) return n % 2 === 0;
	// 空格所在的行
	let emptyRow = Math.ceil((array.findIndex(item =&gt; item === 0) + 1) / row);
	return emptyRow % 2 === 0 ? n % 2 === 0 : n % 2 !== 0;
}

/**
 * 创建游戏
 * @param {number} n 游戏难度
 */
function createGame(n?: number) {
	level.value = n || level.value;
	const ls = [];
	for (var i = 0; i &lt; level.value * level.value; i++) {
		ls.push(i);
	}
	const array = shuffleArray(ls);
	if (isSolvable(array, level.value)) { // 是否可解
		list.value = array;
	} else {
		createGame();
	}
}

/** 首次点击位置 */
const first = ref(-1);
/** 游戏是否暂停 */
const isStop = ref(false);

/** 是否完成游戏 */
function isSuccess(array: number[]) {
	return array.every((n, m) =&gt; (n === 0 &amp;&amp; m === level.value * level.value - 1) || n === m + 1)
}

/** 交换两个数组元素 */
function exchange(array: number[], i: number, j: number) {
	[array[i], array[j]] = [array[j], array[i]];
	return array;
}

function onItemClick(i: number) {
	if (first.value === -1) {
		first.value = i;
	} else if (first.value !== i) {
		if ([list.value[first.value], list.value[i]].includes(0) &amp;&amp; (Math.abs(first.value - i) === 1 || (Math.abs(first.value - i) &lt;= level.value &amp;&amp; first.value % level.value === i % level.value))) {
			exchange(list.value, i, first.value);
			step.value++;
			if (isSuccess(list.value)) {
				uni.showToast({
					title: '挑战成功',
					icon: 'success'
				});
				isStop.value = true;
			}
		}
		first.value = -1;
	}

		isStop.value = false;

		timer = null;

/** 倒计时 */
function timeDown() {
	timer = setTimeout(() =&gt; {
		if (isStop.value) return;
		time.value++;
		timeDown();
	}, 1000);

    <h3>
     七、
     <strong>
      玩法与技巧
     </strong>
    </h3>
    <p>
     <strong>
      1、分阶段排列法
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       逐行推进
      </strong>
      ：优先排列第一行（如1-3），再依次完成后续行。例如，在4×4棋盘中，先固定1-4，再处理5-8，以此类推210。
     </li>
     <li>
      <strong>
       处理底层数字
      </strong>
      ：若目标数字位于底层，需通过“绕行”策略，将上层数字暂时移开，腾出空间调整1012。
     </li>
    </ul>
    <p>
     <strong>
      2、高阶技巧
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       循环移动法
      </strong>
      ：利用空格形成循环路径，将目标数字逐步推至目标位置（如将4从底层移至第一行时，需反复调整1-3的位置）。
     </li>
     <li>
      <strong>
       预留通道
      </strong>
      ：在排列后几行时，需为后续调整保留移动通道，避免死锁
     </li>
    </ul>
