    <p>
     警告信息
     <strong>
      “Array and string offset access syntax with curly braces is deprecated”
     </strong>
     是 PHP 中的一个弃用警告（Deprecation Notice），表明在 PHP 中使用花括号
     <code>
      {}
     </code>
     来访问数组或字符串的偏移量已经被标记为过时。
    </p>
    <h4>
     <a id="_3">
     </a>
     背景
    </h4>
    <p>
     在 PHP 的早期版本中，可以使用花括号
     <code>
      {}
     </code>
     来访问数组或字符串的偏移量，例如：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$array</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$array</span><span class="token punctuation">{<!-- --></span><span class="token number">0</span><span class="token punctuation">}</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 1</span>

<span class="token variable">$string</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Hello"</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$string</span><span class="token punctuation">{<!-- --></span><span class="token number">1</span><span class="token punctuation">}</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 e</span>
</code></pre>
    <p>
     然而，从 PHP 7.4 开始，这种语法已经被标记为过时，并且在未来的版本中可能会被移除。
    </p>
    <h4>
     <a id="_14">
     </a>
     解决方法
    </h4>
    <p>
     为了避免这个警告，您应该使用方括号
     <code>
      []
     </code>
     来访问数组或字符串的偏移量。以下是修改后的代码示例：
    </p>
    <h5>
     <a id="_17">
     </a>
     修改数组访问
    </h5>
    <p>
     将：
    </p>
    <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token variable">$array</span><span class="token punctuation">{<!-- --></span><span class="token number">0</span><span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     改为：
    </p>
    <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token variable">$array</span><span class="token punctuation">[</span><span class="token number">0</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_27">
     </a>
     修改字符串访问
    </h5>
    <p>
     将：
    </p>
    <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token variable">$string</span><span class="token punctuation">{<!-- --></span><span class="token number">1</span><span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     改为：
    </p>
    <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token variable">$string</span><span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
    <h4>
     <a id="_37">
     </a>
     示例代码
    </h4>
    <p>
     以下是完整的示例代码，展示如何修改以避免这个弃用警告：
    </p>
    <pre><code class="prism language-php"><span class="token delimiter important">&lt;?php</span>
<span class="token variable">$array</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
<span class="token variable">$string</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Hello"</span><span class="token punctuation">;</span>

<span class="token comment">// 错误的用法（PHP 7.4+ 会报弃用警告）</span>
<span class="token comment">// echo $array{0};</span>
<span class="token comment">// echo $string{1};</span>

<span class="token comment">// 正确的用法</span>
<span class="token keyword">echo</span> <span class="token variable">$array</span><span class="token punctuation">[</span><span class="token number">0</span><span class="token punctuation">]</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 1</span>
<span class="token keyword">echo</span> <span class="token variable">$string</span><span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">;</span> <span class="token comment">// 输出 e</span>
</code></pre>
    <h4>
     <a id="_53">
     </a>
     其他注意事项
    </h4>
    <ol>
     <li>
      <strong>
       PHP 版本
      </strong>
      ：如果您正在使用 PHP 7.4 或更高版本，建议尽快修改代码以避免未来的兼容性问题。
     </li>
     <li>
      <strong>
       代码审查
      </strong>
      ：如果您正在维护一个大型项目，可以使用代码分析工具（如 PHPStan 或 Psalm）来帮助查找和修复这种弃用的语法。
     </li>
     <li>
      <strong>
       性能影响
      </strong>
      ：虽然这种语法的弃用主要是为了代码的一致性和可维护性，但使用方括号
      <code>
       []
      </code>
      通常也是更推荐的实践。
     </li>
    </ol>
    <p>
     在代码文件里面通过将
     <code>
      {}
     </code>
     替换为
     <code>
      []
     </code>
     ，或者更换低版本的php就可以轻松解决这个警告，并确保代码在未来的 PHP 版本中仍然可以正常运行。
    </p>
