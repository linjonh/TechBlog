# 利用python 5分钟制作一款小游戏

1.安装pygame
  
在命令行cmd中输入：pip install pygame
  
（ 注：如果安装不成功，需要输入：python -m pip install --user --upgrade pip 然后再安装即可。）
  
2.接着在cmd中输入：python -m pygame.exmaples.aliens
