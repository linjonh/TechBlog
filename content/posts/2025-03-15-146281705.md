    <h3>
     <a id="docker_network_0">
     </a>
     拉取镜像并创建docker network
    </h3>
    <p>
     按照官方文档提供的方式拉取镜像：
     <code>
      docker pull apache/rocketmq:4.9.6
     </code>
     <br/>
     创建一个docker网络：
     <code>
      docker network create rocketmq
     </code>
    </p>
    <h3>
     <a id="NameServerBroker_3">
     </a>
     启动NameServer以及Broker
    </h3>
    <h4>
     <a id="NameServer_4">
     </a>
     启动NameServer
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 启动NameServer</span>
<span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token parameter variable">--name</span> rmqnamesrv <span class="token parameter variable">-p</span> <span class="token number">9876</span>:9876 <span class="token parameter variable">--net</span> rocketmq apache/rocketmq:4.9.6 <span class="token function">sh</span> mqnamesrv
<span class="token comment"># 验证NameServer是否启动成功</span>
<span class="token function">docker</span> logs <span class="token parameter variable">-f</span> rmqnamesrv
<span class="token comment"># 启动完成后 查看rmqnamesrv在网络中的ip</span>
<span class="token function">docker</span> inspect rmqnamesrv <span class="token operator">|</span> <span class="token function">grep</span> <span class="token string">"IPAddress"</span>
</code></pre>
    <p>
     <img alt="查看ip" src="https://i-blog.csdnimg.cn/direct/d94de17315f24454a0a1213df34df144.png">
      <br/>
      如上图所示，我这里的ip是
      <code>
       172.23.0.2
      </code>
     </img>
    </p>
    <h4>
     <a id="Broker_15">
     </a>
     启动Broker
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 创建broker.conf文件</span>
<span class="token function">cat</span> <span class="token operator">&lt;&lt;</span><span class="token string">EOF<span class="token bash punctuation"> <span class="token operator">&gt;</span> broker.conf</span>
brokerClusterName=DefaultCluster
brokerName=broker-a
brokerId=0
brokerIP1=这里替换为服务器的外网地址
namesrvAddr=这里替换为上一步的ip地址:9876
EOF</span>
<span class="token comment"># 启动rocketmq broker</span>
<span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token punctuation">\</span>
<span class="token parameter variable">--name</span> rmqbroker <span class="token punctuation">\</span>
<span class="token parameter variable">--net</span> rocketmq <span class="token punctuation">\</span>
<span class="token parameter variable">-p</span> <span class="token number">10912</span>:10912 <span class="token parameter variable">-p</span> <span class="token number">10911</span>:10911 <span class="token parameter variable">-p</span> <span class="token number">10909</span>:10909 <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> ./broker.conf:/home/rocketmq/rocketmq-4.9.6/conf/broker.conf <span class="token punctuation">\</span>
apache/rocketmq:4.9.6 <span class="token function">sh</span> mqbroker <span class="token punctuation">\</span>
<span class="token parameter variable">-c</span> /home/rocketmq/rocketmq-4.9.6/conf/broker.conf
<span class="token comment"># 查看日志</span>
<span class="token function">docker</span> logs rmqbroker
</code></pre>
    <p>
     当日志出现以下语句时，即启动成功
     <br/>
     The broker XXX boot success. serializeType=JSON and name server is 172.23.0.2:9876
    </p>
    <h3>
     <a id="RocketMQ_Dashboard_39">
     </a>
     安装RocketMQ Dashboard可视化面板
    </h3>
    <pre><code class="prism language-bash"><span class="token comment"># 拉取镜像</span>
<span class="token function">docker</span> pull apacherocketmq/rocketmq-dashboard:latest
<span class="token comment"># 运行</span>
<span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token parameter variable">--name</span> rocketmq-dashboard <span class="token parameter variable">--net</span> rocketmq <span class="token parameter variable">-e</span> <span class="token string">"JAVA_OPTS=-Drocketmq.namesrv.addr=172.23.0.2:9876"</span> <span class="token parameter variable">-p</span> <span class="token number">8080</span>:8080 <span class="token parameter variable">-t</span> apacherocketmq/rocketmq-dashboard:latest
<span class="token comment"># 查看</span>
<span class="token function">docker</span> <span class="token function">ps</span> <span class="token operator">|</span> <span class="token function">grep</span> rocketmq-dashboard
</code></pre>
    <h3>
     <a id="_49">
     </a>
     最后，记得在云服务器的防火墙中开放以下端口。
    </h3>
    <p>
     默认端口配置下，需开放端口号：控制台：8080；namesrv：9876；broker：10909、10911
    </p>
