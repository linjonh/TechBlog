    <h2>
     一、前言
    </h2>
    <h4>
     实现功能：
    </h4>
    <ul>
     <li>
      <span style="color:#494949;">
       显示多张图片，每张图片占据轮播图容器的一部分空间。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       实现向左和向右的切换按钮，可以点击按钮切换到上一张或下一张图片。
      </span>
     </li>
     <li>
      <span style="color:#494949;">
       在底部显示小圆点，表示当前显示的图片，点击小圆点可以跳转到对应的图片。
      </span>
     </li>
    </ul>
    <h4>
     实现逻辑：
    </h4>
    <ol>
     <li>
      <p>
       首先，使用
       <code>
        querySelectorAll
       </code>
       方法获取所有的轮播项元素，并将其保存在
       <code>
        items
       </code>
       变量中。
      </p>
     </li>
     <li>
      <p>
       定义一个全局变量
       <code>
        current
       </code>
       ，用于记录当前显示的轮播项的索引，默认为0。通过操作
       <code>
        current
       </code>
       变量来控制显示的轮播项。并通过调用相关函数来更新显示和样式
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        showSlide
       </code>
       函数，用于显示当前的轮播项。通过遍历所有轮播项，设置其
       <code>
        transform
       </code>
       属性来实现水平滑动效果。同时，调用
       <code>
        updateDots
       </code>
       函数更新小圆点的样式。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        prevSlide
       </code>
       函数，用于切换到上一张轮播项。如果
       <code>
        current
       </code>
       大于0，则将
       <code>
        current
       </code>
       减1；否则，将
       <code>
        current
       </code>
       设置为最后一张轮播项的索引。然后，调用
       <code>
        showSlide
       </code>
       函数显示对应的轮播项。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        nextSlide
       </code>
       函数，用于切换到下一张轮播项。如果
       <code>
        current
       </code>
       小于轮播项的数量减1，则将
       <code>
        current
       </code>
       加1；否则，将
       <code>
        current
       </code>
       设置为0。然后，调用
       <code>
        showSlide
       </code>
       函数显示对应的轮播项。
      </p>
     </li>
     <li>
      <p>
       使用
       <code>
        setInterval
       </code>
       方法定义一个定时器
       <code>
        timer
       </code>
       ，每隔3秒自动调用一次
       <code>
        nextSlide
       </code>
       函数，实现自动播放功能。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        pauseTimer
       </code>
       函数，用于暂停定时器。通过调用
       <code>
        clearInterval
       </code>
       方法，清除定时器
       <code>
        timer
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        resumeTimer
       </code>
       函数，用于恢复定时器。重新设置定时器
       <code>
        timer
       </code>
       ，调用
       <code>
        nextSlide
       </code>
       函数实现轮播功能。
      </p>
     </li>
     <li>
      <p>
       添加鼠标悬停和离开事件监听器，当鼠标悬停在轮播图上时，调用
       <code>
        pauseTimer
       </code>
       函数暂停自动播放；当鼠标离开时，调用
       <code>
        resumeTimer
       </code>
       函数恢复自动播放。
      </p>
     </li>
     <li>
      <p>
       使用
       <code>
        querySelectorAll
       </code>
       方法获取所有的小圆点元素，并将其保存在
       <code>
        dots
       </code>
       变量中。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        updateDots
       </code>
       函数，用于更新小圆点的样式。遍历所有小圆点元素，移除它们的
       <code>
        active
       </code>
       类名，然后根据
       <code>
        current
       </code>
       变量为当前轮播项对应的小圆点添加
       <code>
        active
       </code>
       类名。
      </p>
     </li>
     <li>
      <p>
       定义
       <code>
        jumpToSlide
       </code>
       函数，用于跳转到指定的轮播项。将
       <code>
        current
       </code>
       变量设置为指定轮播项的索引，然后调用
       <code>
        showSlide
       </code>
       和
       <code>
        updateDots
       </code>
       函数，显示对应的轮播项并更新小圆点的样式。
      </p>
     </li>
    </ol>

    &lt;script&gt;
        let items = document.querySelectorAll('.item');
        let current = 0;
        function showSlide() {
            items.forEach(item =&gt; {
                item.style.transform = `translateX(-${current * 100}%)`;
            });
            updateDots();
        }

        function prevSlide() {
            if (current &gt; 0) {
                current--;
            } else {
                current = items.length - 1;
            }
            showSlide();
        }

        function nextSlide() {
            if (current &lt; items.length - 1) {
                current++;
            } else {
                current = 0;
            }
            showSlide();
        }

        let timer = setInterval(nextSlide, 3000);

        function pauseTimer() {
            clearInterval(timer);
        }
        function resumeTimer() {
            timer = setInterval(nextSlide, 3000);
        }

        document.getElementById('carousel').addEventListener('mouseover', pauseTimer);
        document.getElementById('carousel').addEventListener('mouseout', resumeTimer);

        let dots = document.querySelectorAll('.dots li');

        function updateDots() {
            dots.forEach(dot =&gt; {
                dot.classList.remove('active');
            });
            dots[current].classList.add('active');
        }

        function jumpToSlide(index) {
            current = index;
            showSlide();
            updateDots();
        }
    &lt;/script&gt;
