     Python自动点击器开发教程 - 支持键盘连按和鼠标连点
    </h2>
    <p>
    </p>
    <p>
    </p>
    <h3>
     <a id="_3">
     </a>
     项目介绍
    </h3>
    <p>
     这是一个使用Python开发的自动点击器，支持键盘按键连续点击和鼠标连点功能。主要特点：
    </p>
    <ul>
     <li>
      支持键盘按键和鼠标两种模式
     </li>
     <li>
      可自定义按键序列（支持多个按键连续点击）
     </li>
     <li>
      精确的时间控制（最小可到1毫秒）
     </li>
     <li>
      简洁的图形界面
     </li>
     <li>
      全局快捷键支持
     </li>
     <li>
      ⚠️鼠标左键连点，点运行按钮会导致卡住，建议按快捷键F8
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/04e1401e394c4e52ba699f6828746fca.png"/>
     </li>
    </ul>
    <h3>
     <a id="_15">
     </a>
     开发环境
    </h3>
    <ul>
     <li>
      Python 3.x
     </li>
     <li>
      依赖库：
      <ul>
       <li>
        pywin32
       </li>
       <li>
        keyboard
       </li>
       <li>
        tkinter (Python自带)
       </li>
      </ul>
     </li>
    </ul>

     <a id="_23">

     安装依赖

    <h3>
     <a id="_61">
     </a>
     开发要点
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        键盘事件模拟
       </strong>
      </p>
      <ul>
       <li>
        使用
        <code>
         win32api.keybd_event
        </code>
        实现键盘事件模拟
       </li>
       <li>
        通过扫描码映射表支持多种按键
       </li>
       <li>
        精确控制按键按下和释放时间
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        鼠标事件模拟
       </strong>
      </p>
      <ul>
       <li>
        使用
        <code>
         ctypes.windll.user32.mouse_event
        </code>
        实现鼠标点击
       </li>
       <li>
        支持左键和右键点击
       </li>
       <li>
        可调节点击间隔和按住时间
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        多线程处理
       </strong>
      </p>
      <ul>
       <li>
        使用线程实现连续点击功能
       </li>
       <li>
        确保UI响应不被阻塞
       </li>
       <li>
        安全的线程终止机制
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        用户界面设计
       </strong>
      </p>
      <ul>
       <li>
        使用tkinter创建简洁的图形界面
       </li>
       <li>
        提供直观的参数调节控件
       </li>
       <li>
        实时状态显示
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_83">
     </a>
     使用说明
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        键盘连点模式
       </strong>
       ：
      </p>
      <ul>
       <li>
        在输入框中输入要连续点击的按键，多个按键用逗号分隔
       </li>
       <li>
        例如：
        <code>
         w,a,s,d
        </code>
        将依次点击这些按键
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        鼠标连点模式
       </strong>
       ：
      </p>
      <ul>
       <li>
        选择要点击的鼠标按键（左键或右键）
       </li>
       <li>
        调整点击间隔和按住时间
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        时间设置
       </strong>
       ：
      </p>
      <ul>
       <li>
        按住时间：每次点击按住的时长
       </li>
       <li>
        间隔时间：两次点击之间的间隔
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        快捷键
       </strong>
       ：
      </p>
      <ul>
       <li>
        F8：开始/停止
       </li>
       <li>
        F9：退出程序
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_101">
     </a>
     注意事项
    </h3>
    <ol>
     <li>
      程序使用了Windows API，仅支持Windows系统
     </li>
     <li>
      部分游戏可能有反作弊系统，使用需谨慎
     </li>
     <li>
      建议先用较长的间隔时间测试
     </li>
     <li>
      如果点击太快可能会被某些程序屏蔽
     </li>
    </ol>
    <h3>
     <a id="_108">
     </a>
     优化建议
    </h3>

     </li>
     <li>
      可以增加按键组合功能
     </li>
     <li>
      可以添加点击次数限制
     </li>
     <li>
      可以保存和加载配置文件
     </li>
    </ol>
