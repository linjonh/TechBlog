    <h2>
     <a id="inux_33">
     </a>
     四、inux部署数据库
    </h2>
    <ul>
     <li>
      Linux准备环境：vmware虚拟软件 --&gt;加载虚拟机 --&gt;启动
     </li>
    </ul>
    <h4>
     <a id="1__35">
     </a>
     1. 设置网卡–&gt;虚拟网络编辑器
    </h4>
    <p>
     注：此处需高度匹配
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0dcdad029d216d6fdd9ab85c9f6f60a6.png"/>
    </p>
    <h4>
     <a id="2__37">
     </a>
     2. 配置网卡命令
    </h4>
    <p>
     操作命令：
    </p>
    <blockquote>
     <p>
      cat /etc/sysconfig/network-scripts/ifcfg-eth0
     </p>
    </blockquote>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b1e2b24dbf4ecd9a738440622564f74c.png">
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b1f4ebdb25c2556f077b4ce01a5bcf5e.png"/>
     </img>
    </p>
    <h4>
     <a id="3__44">
     </a>
     3. 配置成功后重启网卡
    </h4>
    <p>
     操作命令：
    </p>
    <blockquote>
     <p>
      shutdown -r now
     </p>
    </blockquote>
    <h4>
     <a id="4__47">
     </a>
     4. 部署数据库服务
    </h4>
    <p>
     操作命令：
    </p>
    <blockquote>
     <p>
      yum install -y mariadb mariadb-server
     </p>
    </blockquote>
    <h4>
     <a id="6__50">
     </a>
     6. 启动数据库服务
    </h4>
    <p>
     操作命令:
    </p>
    <blockquote>
     <p>
      systemctl start mariadb
     </p>
    </blockquote>
    <h2>
     <a id="_54">
     </a>
     五、数据库管理
    </h2>
    <h4>
     <a id="1__55">
     </a>
     1. 进入数据库
    </h4>
    <p>
     操作命令：
    </p>
    <blockquote>
     <p>
      mysql -uroot -p***
     </p>
    </blockquote>
    <p>
     u后头为账号，p为密码，新设置的数据库默认没有密码，则不用输入密码直接：mysql -uroot
    </p>
    <h4>
     <a id="2__60">
