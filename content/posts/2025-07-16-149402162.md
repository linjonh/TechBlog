# 阿里云云计算三大认证详解

## 一、云计算工程师 ACA 认证

![](https://i-blog.csdnimg.cn/direct/2db4a2eef79548fc849113bbc64b4535.png)

#### （1）证书介绍

阿里云云计算工程师 ACA 认证（Alibaba Cloud Certified Associate - Cloud Computing）主要面向使用阿里云基础产品的专业技术人员，其认证内容涉及阿里云的计算、存储、网络、安全类核心产品，能全面检验和认证学员对阿里云主要产品技术技能的掌握水平。该认证的目标群体包括学生群体、开发者，也可作为运维人员的入门证书。需要注意的是，本认证仅支持持有中国大陆身份证的用户参与，且同一账号在同一认证有效期内仅可领取一次证书。

#### （2）认证考试信息

* **适合人群**：云计算初学者、基础一般的小白，主要面向学生群体及开发者，也可作为运维人员的入门证书。
* **考试内容**：涵盖云计算基础知识、阿里云基础产品和服务的使用，主要涉及阿里云的计算、存储、网络、安全类核心产品。
* **考试形式**：线上考试，时长 60 分钟，共 50 道选择题，其中 35 道单选、15 道多选，满分 100 分，80 分及格。
* **考试费用**：600 元。
* **考试方式**：线上考试，无需预约，备考好后可直接进入官网考试。
* **考试地点**：考生自备电脑和网络，在封闭环境中进行考试。
* **证书到手时间**：考试完立即出分数，随后可访问阿里云认证网站下载电子证书。
* **证书形式**：电子版证书。
* **证书有效期**：2 年。
* **价值体现**：获得 ACA 认证，意味着具备了阿里云基础产品的使用能力，能够胜任云计算运维工程师等初级岗位。

#### （3）考试知识点

| 考试内容 | 占比 |
| --- | --- |
| 云计算基础 | 10% |
| 弹性计算 | 25% |
| 云容器 | 10% |
| 云存储 | 15% |
| 云数据库 | 10% |
| 云网络 | 20% |
| 云安全 | 10% |

### 二、云计算高级工程师 ACP 认证

#### 

#### （1）证书介绍

阿里云云计算高级工程师 ACP 认证（Alibaba Cloud Certified Professional - Cloud Computing）的面向群体为使用阿里云云计算产品的架构、开发、运维类人员，主要考核考生利用阿里云云计算技术服务体系设计稳定、安全、高性能、易扩展、低成本的企业云计算架构的能力。

#### （2）认证考试信息

* **适合人群**：已经具备一定云计算基础，想进一步提升技能的专业人士，主要面向具备 3-5 年工作经验的技术人员。
* **考试内容**：包括云架构设计、云安全、云计算部署和管理等方面，主要考核考生利用阿里云云计算技术服务体系设计稳定、安全、高性能、易扩展、低成本的企业云计算架构的能力。
* **考试形式**：主要是线下考试（通过培训机构报名可线上），考笔试 + 实验，考试时间 120 分钟，完成 100 道客观题，满分 100 分，80 分及格。
* **考试费用**：官网报名费用为 1200 元。
* **考试方式**：个人登录官网报名需参加线下考试；通过授权培训机构报名可参加线上考试，一般每周四进行（节假日除外），需提前两周预约。
* **考试地点**：笔试是线下考试，需要考生去所在城市的线下考试中心参加；实验是线上考试，考生需自行准备电脑和网络环境。
* **证书到手时间**：考试结束后 2 个工作日内公布成绩，考后第 3 个工作日起，可登录阿里云官网领取电子证书。
* **证书形式**：电子版证书。
* **证书有效期**：2 年。
* **价值体现**：能证明个人可基于阿里云产品解决企业业务问题，提升职业竞争力，是云计算方向热门认证。

#### （3）考试知识点

| --- | --- |
| 基础云应用架构 | 35% |
| 复杂云网络架构 | 25% |
| 云上高可用、高性能架构 | 25% |
| 云上安全体系架构 | 15% |

### 三、云计算架构师 ACE 认证

#### 

#### （1）证书介绍

阿里云云计算架构师 ACE 认证（Alibaba Cloud Certified Expert - Cloud Computing）是阿里云的高级别技术认证，针对云架构师、解决方案架构师、云运维专家等技术人员。通过该认证，能有效证明认证人员具备丰富的需求分析能力、云架构设计能力、云架构部署实施能力、云架构诊断和调优能力，能够洞察业务痛点，制定有效的技术改进方案，最大限度地提高系统架构的可扩展性、安全性、可靠性、耐用性和成本效益等特性。

#### （2）认证考试信息

* **适合人群**：面向高级云计算工程师和架构师，主要针对具备 5 年以上工作经验的云计算技术专家。
* **考试内容**：涵盖复杂的云计算架构设计、高级运维和优化等内容，要求考生具备深厚的技术背景和丰富的实际操作经验。
* **考试形式**：难度最大，包括笔试、实验和面试三个环节，且这三个环节必须依次进行，需在 1 年内全部通过，否则只能重考。笔试主要考核云架构相关理论知识，为 100 题选择题；实验考试以云速搭（CADT）为载体，给定场景要求、预算和时限（8 个小时），考核考生云上架构设计等能力；面试由阿里云 P9 技术专家组建的面试官团队进行，时长 1 小时，重点考察考生在架构设计、实施、优化等方面的思考。
* **考试费用**：9600 元。
* **考试地点**：笔试在普尔文国际考试中心进行；实验和面试在阿里云园区进行。
* **证书到手时间**：当笔试、实验和面试全部通过后，即可领取证书。
* **证书形式**：通常为电子版证书。
* **证书有效期**：2 年，在证书失效前，可通过再认证方式刷新有效期。
* **价值体现**：ACE 认证含金量最高，持证人数少，对标阿里云 P8 级别架构师水平，持证者有机会成为企业争抢的对象，有机会拿到较高的薪资待遇。
