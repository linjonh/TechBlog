    <p>
     <span style="font-size:24px">
      <span style="font-family:Microsoft YaHei">
       1.公司或国家的高层勾心斗角的来源就是因为制度，方案的不透明，所以才会有勾心
      </span>
     </span>
    </p>
    <p>

    <span style="font-size:24px">
     <br/>
     <br/>
     <span style="font-family:Microsoft YaHei">

     <br/>
     <br/>
    </span>
    <p>
     <span style="font-size:24px">
      <span style="font-family:Microsoft YaHei">

      </span>
     </span>

       乐意的加班了，当然是在适度的条件下。

     <br/>
     <br/>
     <span style="font-family:Microsoft YaHei">
      9.
      <strong>
       在政府做事，
      </strong>

     </span>
