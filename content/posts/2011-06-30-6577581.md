    <p>
     各位 CSDN博客用户：
     <br/>
     <br/>
     大家好！
     <br/>
     <br/>
     微软 2011 年 10 月份的MVP 申请已经开始！ CSDN 与微软合作，长期为用户提供申请 “ 微软最有价值专家 ”（ MVP ） 的平台，希望 申请“微软最有价值专家”并已经具备相应条件 的 博客用户 积极参与 ， CSDN可代为推荐。
     <br/>
     <br/>
     申请微软MVP 的 用户 请下载申请表格，我们会在MVP 申请开始时把用户提交的申请表格推荐到微软（中国）相关部门 。
     <br/>
     <br/>
     下载表格 地址 ：
     <a href="http://community.csdn.net/bbs/2011/06/2011MVPChina.rar" rel="nofollow">
      http://community.csdn.net/bbs/2011/06/2011MVPChina.rar
     </a>
     (点击下载 )
     <br/>
     <br/>
     填写注意事项：
     <br/>
     <br/>
     1、“ 姓名 ” 这一项在填写的时候请注明 您的 CSDN   ID ， 格式如：姓名 (CSDN ID: id )  。
     <br/>
     <br/>
     2、 对 CSDN 博客用户来讲， “ 社区贡献 ” 部分的第3 项 ：“ 参与技术博客，积极推广微软技术 ” 是必填的，其他部分依据个人情况填写。
     <br/>
     <br/>
     CSDN论坛用户请按照如下公告进行申请：
     <br/>
     <br/>
     <a href="http://topic.csdn.net/u/20110630/14/5d5dbca2-f19e-4dd2-b962-ea0f739c87f3.html" rel="nofollow">
      http://topic.csdn.net/u/20110630/14/5d5dbca2-f19e-4dd2-b962-ea0f739c87f3.html
     </a>
     <br/>
     <br/>
     3 、请填写您的自我介绍，最好使用英文，如果没有，微软将提供人员给您翻译，但不保证翻译质量。
     <br/>
     <br/>
     4 、
     <span style="color: #ff0000;">
      请尽量 详细地描述 您在过去一年中对微软技术推广做出的贡献，但是此行动必须是无偿的。
     </span>
     <br/>
     <br/>
     5 、如您 的 资料不够详细或者有问题，CSDN 或微软工作人员会第一时间联络您进行资料补充。
     <br/>
     <br/>
     6、 申请表填写完毕后，请重命名该文件为《MVP 申请  -  姓名》，如：《 MVP 申请  -  姓名》。
     <br/>
     <br/>
     申请表 填写 完毕后，请以附件形式发邮件至 admin  # csdn.net 。邮件标题请注明： MVP-申请 - 您的姓名
     <br/>
     <br/>
     <strong>
      <span style="color: #ff0000;">
       申请截止时间7 月 8 日 16 点。
      </span>
     </strong>
     <br/>
     <br/>
     <br/>
     为了 避免 邮件丢失 我们联系不到申请用户 ，请 申请者发完邮件后访问：
     <a href="http://community.csdn.net/bbs/2011/289.htm" rel="nofollow">
      http://community.csdn.net/bbs/2011/289.htm
     </a>
     进行登记。
    </p>
    <p>
     <br/>
     <br/>
     其他问题补充:
     <br/>
     <br/>
     1、我已经申请过 7 月的 MVP ，现在当任务名单未出来，我是否需要再申请？
     <br/>
     <br/>
     答：您可以等待7 月 MVP 名单出来后，再确认是否重新申请。
     <br/>
     <br/>
     2、我是 Java,Oracle 的领域的专家，是否可以申请？
     <br/>
     <br/>
     答：本申请针对微软技术领域专家，但如果您对微软技术领域有贡献，亦可进行申请。
     <br/>
     <br/>
     3、申请有对职业和年龄的限制吗？
     <br/>
     <br/>
     答：年龄必须满18 岁。职业无限制。但不得从事非法职业。
     <br/>
     <br/>
     4、有哪些贡献算 MVP 贡献
     <br/>
     <br/>
     答：a  线上： 传播微软相关技术，如：社区发帖、回复。 写作 BLOG
     <br/>
     <br/>
     b 线下： 写作、 组织微软技术相关活动 、微软课程培训。
     <br/>
     <br/>
     但所有一切不能为付费行为，均为免费行为。
     <br/>
     <br/>
     5、能给一个申请表填写的示例吗？
     <br/>
     <br/>
     答：请参考：
     <a href="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx">
      http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx
     </a>
    </p>
