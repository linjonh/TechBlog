      题目描述
      <br/>
      输入一个链表，输出该链表中倒数第k个结点。
     </p>
    </blockquote>
    <ul>
     <li>
      算法：前后两个指针

      数据结构：链表
     </li>

      编程语言：C++
     </li>
    </ul>
    <pre class="prettyprint"><code class="hljs cpp"><span class="hljs-comment">/*
struct ListNode {

<span class="hljs-keyword">class</span> Solution {
<span class="hljs-keyword">public</span>:
    ListNode* FindKthToTail(ListNode* pListHead, <span class="hljs-keyword">unsigned</span> <span class="hljs-keyword">int</span> k) {
       <span class="hljs-keyword">if</span>(pListHead==<span class="hljs-keyword">nullptr</span>||k==<span class="hljs-number">0</span>)
           <span class="hljs-keyword">return</span> <span class="hljs-keyword">nullptr</span>;
        ListNode* p=pListHead;
        <span class="hljs-comment">//第一个节点先走k步</span>
        <span class="hljs-keyword">for</span>(<span class="hljs-keyword">int</span> i=<span class="hljs-number">0</span>;i&lt;k-<span class="hljs-number">1</span>;i++)
        {

            <span class="hljs-keyword">if</span>(p-&gt;next==<span class="hljs-keyword">nullptr</span>)
                <span class="hljs-keyword">return</span> <span class="hljs-keyword">nullptr</span>;

        }

            behind=behind-&gt;next;
        }
        <span class="hljs-keyword">return</span> behind;
    }
