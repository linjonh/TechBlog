     在 IntelliJ IDEA 中配置 Git
    </h1>

  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <h4>
     <strong>
      1. 确保已安装 Git
     </strong>
    </h4>
    <p>
     在配置之前，确保你的系统已经安装了 Git。
    </p>
    <ul>
     <li>
      <p>
       检查是否已安装 Git：
      </p>
      <p>
       bash
      </p>
      <p>
       复制
      </p>
      <pre>git --version</pre>
      <p>
       如果未安装，请前往
       <a href="https://git-scm.com/" rel="nofollow" title="Git 官网">
        Git 官网
       </a>
       下载并安装。
      </p>
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      2. 在 IntelliJ IDEA 中配置 Git
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        打开 IntelliJ IDEA
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        进入设置
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         Windows/Linux：点击菜单栏的
         <code>
          File
         </code>
         &gt;
         <code>
          Settings
         </code>
         。
        </p>
       </li>
       <li>
        <p>
         macOS：点击菜单栏的
         <code>
          IntelliJ IDEA
         </code>
         &gt;
         <code>
          Preferences
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        找到 Git 配置
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在设置窗口中，导航到
         <code>
          Version Control
         </code>
         &gt;
         <code>
          Git
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        配置 Git 路径
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在
         <code>
          Path to Git executable
         </code>
         字段中，输入 Git 的可执行文件路径。
        </p>
        <ul>
         <li>
          <p>
           Windows：通常是
           <code>
            C:\Program Files\Git\bin\git.exe
           </code>
           。
          </p>
         </li>
         <li>
          <p>
           macOS/Linux：通常是
           <code>
            /usr/bin/git
           </code>
           。
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         如果 IDEA 自动检测到 Git 路径，可以直接使用默认值。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        测试 Git 配置
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击
         <code>
          Test
         </code>
         按钮，确保 IDEA 能够正确识别 Git。
        </p>
       </li>
       <li>
        <p>
         如果成功，会显示 Git 版本号。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      3. 配置 GitHub/GitLab 账户（可选）
     </strong>
    </h4>
    <p>
     如果你需要与 GitHub、GitLab 等远程仓库交互，可以配置账户信息。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        进入设置
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         导航到
         <code>
          Version Control
         </code>
         &gt;
         <code>
          GitHub
         </code>
         或
         <code>
          GitLab
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        添加账户
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击
         <code>
          Add account
         </code>
         ，选择登录方式（Token 或密码）。
        </p>
       </li>
       <li>
        <p>
         按照提示完成登录。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      4. 创建或导入 Git 仓库
     </strong>
    </h4>
    <h5>
     <strong>
      创建新仓库
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       打开你的项目。
      </p>
     </li>
     <li>
      <p>
       点击顶部菜单的
       <code>
        VCS
       </code>
       &gt;
       <code>
        Import into Version Control
       </code>
       &gt;
       <code>
        Create Git Repository
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       选择项目根目录，点击
       <code>
        OK
       </code>
       。
      </p>
     </li>
    </ol>
    <h5>
     <strong>
      导入现有仓库
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       打开你的项目。
      </p>
     </li>
     <li>
      <p>
       点击顶部菜单的
       <code>
        VCS
       </code>
       &gt;
       <code>
        Get from Version Control
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       输入远程仓库的 URL，选择本地目录，点击
       <code>
        Clone
       </code>
       。
      </p>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      5. 使用 Git 功能
     </strong>
    </h4>
    <p>
     配置完成后，你可以在 IDEA 中使用 Git 的所有功能：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        提交更改
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击顶部菜单的
         <code>
          VCS
         </code>
         &gt;
         <code>
          Commit
         </code>
         。
        </p>
       </li>
       <li>
        <p>
         选择要提交的文件，输入提交信息，点击
         <code>
          Commit
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        推送更改
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击顶部菜单的
         <code>
          VCS
         </code>
         &gt;
         <code>
          Git
         </code>
         &gt;
         <code>
          Push
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        拉取更改
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击顶部菜单的
         <code>
          VCS
         </code>
         &gt;
         <code>
          Git
         </code>
         &gt;
         <code>
          Pull
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        查看历史
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         右键点击项目或文件，选择
         <code>
          Git
         </code>
         &gt;
         <code>
          Show History
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        分支管理
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         点击底部工具栏的
         <code>
          Git
         </code>
         ，切换到
         <code>
          Branches
         </code>
         标签，可以创建、切换或合并分支。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      6. 其他设置
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        忽略文件
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在项目根目录创建
         <code>
          .gitignore
         </code>
         文件，列出需要忽略的文件或目录。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        自动导入更改
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在
         <code>
          Settings
         </code>
         &gt;
         <code>
          Version Control
         </code>
         &gt;
         <code>
          Confirmation
         </code>
         中，可以配置是否自动导入外部更改。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      总结
     </strong>
    </h4>
    <p>
     通过以上步骤，你可以在 IntelliJ IDEA 中成功配置 Git，并使用其强大的版本控制功能。如果遇到问题，可以检查 Git 路径是否正确，或者查看 IDEA 的日志文件以获取更多信息。
    </p>
