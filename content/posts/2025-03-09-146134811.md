      <span style="color:#0d0016">
       （2）总结。
      </span>
     </h5>
     <ul>
      <li>
       选项 A，
       <span style="background-color:#ffd900">
        search
       </span>
       类型的输入框主要用于搜索功能。通常带有清除按钮，不是滑块控件。
      </li>
     </ul>
     <ul>
      <li>
       选项 B，
       <span style="background-color:#ffd900">
        controls
       </span>
       不是&lt;input&gt;元素的type属性值，它常用与&lt;video&gt;、&lt;audio&gt;等媒体元素中添加默认的播放控制条。
      </li>
      <li>
       选项 C，HTML 中没有
       <span style="background-color:#ffd900">
        slider
       </span>
       这种&lt;input&gt;的type值。
      </li>
      <li>
       选项 D，
       <span style="background-color:#ed7976">
        range
       </span>
       类型的&lt;input&gt;元素定义了一个滑块控件。用户可以在指定的范围内选择一个值 。
      </li>
     </ul>

     <ul>

       <span style="color:#0d0016">

        </span>

       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#38d8f0">
         reset
        </span>
        ：不是HTML5新增的，在HTML4就已经存在，用于重置表单。
       </span>
      </li>
     </ul>
     <hr/>
     <ul>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#fbd4d0">
         search
        </span>
        ：HTML5新增。用于搜索输入框，功能类似于text。但在某些浏览器中会有特殊的样式和功能。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#fbd4d0">
         number
        </span>
        ：HTML5新增。专门用于数字输入，可以设置min、max等数值范围。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#fbd4d0">
         tel
        </span>
        ：HTML5新增。用于电话号码输入，在移动设备上会调出数字键盘。
       </span>
      </li>
     </ul>

    <h4 id="%E4%BA%8C%E3%80%81%E8%A1%A8%E6%A0%BC(%3Ctable%3E)%E4%B8%8E%E6%8F%8F%E8%BF%B0%E5%88%97%E8%A1%A8(%3Cdl%3E)%E3%80%82" name="%E4%BA%8C%E3%80%81%E8%A1%A8%E6%A0%BC(%3Ctable%3E)%E4%B8%8E%E6%8F%8F%E8%BF%B0%E5%88%97%E8%A1%A8(%3Cdl%3E)%E3%80%82" style="background-color:transparent">
     <span style="color:#0d0016">
      二、表格（table）与
      <strong>
       描述列表（dl）。
      </strong>
     </span>
    </h4>
    <blockquote>
     <h5 id="%EF%BC%881%EF%BC%89%E8%A1%A8%E6%A0%BC%3Ctable%3E%E7%BB%84%E6%88%90%E6%A0%BC%E5%BC%8F%E3%80%82" name="%EF%BC%881%EF%BC%89%E8%A1%A8%E6%A0%BC%3Ctable%3E%E7%BB%84%E6%88%90%E6%A0%BC%E5%BC%8F%E3%80%82">
      <span style="color:#0d0016">
       （1）表格（input）组成格式。
      </span>
     </h5>
     <ul>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;table&gt;
        </span>
        ：表示整个表格的容器，是
        <span style="background-color:#ed7976">
         表格的根标签
        </span>
        。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;tr&gt;
        </span>
        ：代表表格中的一行，一个&lt;table&gt;标签内可包含多个&lt;tr&gt;标签来定义多行内容。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;th&gt;
        </span>
        ：用于定义表头单元格，一般位于表格的第一行，用来描述每列数据的含义，内容默认加粗居中显示。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;td&gt;
        </span>
        ：表示表格的数据单元格，每个&lt;tr&gt;标签内可包含一个或多个&lt;td&gt;标签，用于存放具体的数据。
       </span>
      </li>
     </ul>
     <hr/>
     <h5 id="%EF%BC%882%EF%BC%89%E6%8F%8F%E8%BF%B0%E5%88%97%E8%A1%A8%3Cdl%3E%E7%9A%84%E7%BB%84%E6%88%90%E6%A0%BC%E5%BC%8F%E3%80%82" name="%EF%BC%882%EF%BC%89%E6%8F%8F%E8%BF%B0%E5%88%97%E8%A1%A8%3Cdl%3E%E7%9A%84%E7%BB%84%E6%88%90%E6%A0%BC%E5%BC%8F%E3%80%82">
      <span style="color:#0d0016">
       （2）描述列表（dl）的组成格式。
      </span>
     </h5>
     <ul>
      <li>
       <span style="color:#0d0016">
        描述列表在 HTML 中用于
        <span style="background-color:#ffd900">
         展示术语及其对应的描述
        </span>
        。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;dl&gt;
        </span>
        ：是
        <span style="background-color:#ed7976">
         描述列表的根标签
        </span>
        ，用于包裹整个描述列表内容。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;dt&gt;
        </span>
        ：定义术语，即需要解释的名词或概念。
       </span>
      </li>
      <li>
       <span style="color:#0d0016">
        <span style="background-color:#faa572">
         &lt;dd&gt;
        </span>
        ：定义&lt;dd&gt;描述，用于对&lt;dt&gt;中的术语进行详细解释或说明。一个&lt;dt&gt;可以对应多个&lt;dd&gt;。
       </span>
      </li>
     </ul>

     <h5 id="%EF%BC%882%EF%BC%89%E5%AE%98%E6%96%B9%E8%A7%A3%E6%9E%90%E4%B8%8E%E4%BB%A3%E7%A0%81%E7%A4%BA%E4%BE%8B%E3%80%82" name="%EF%BC%882%EF%BC%89%E5%AE%98%E6%96%B9%E8%A7%A3%E6%9E%90%E4%B8%8E%E4%BB%A3%E7%A0%81%E7%A4%BA%E4%BE%8B%E3%80%82" style="background-color:transparent">

      </span>

     <ul>
      <li>
       <span style="background-color:#a2e043">
        &lt;label&gt;元素是HTML中非常重要的表单相关元素,主要用于提升表单的可访问性和用户体验
       </span>
       。
      </li>
     </ul>
     <hr/>
     <ul>
      <li>
       选项A正确：&lt;label&gt;元素有两种使用方式：一是
       <span style="background-color:#ffd900">
        使用for属性与表单控件建立关联
       </span>
       。二是
       <span style="background-color:#ffd900">
        直接将表单控件嵌套在label元素内部
       </span>
       。这两种方式都能使label元素正确关联到表单控件。
      </li>
      <li>
       代码示例。
      </li>
     </ul>
     <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;body&gt;

    &lt;!-- for与id一致 --&gt;
    &lt;input type="checkbox" id="myCheckbox"&gt;
    &lt;label for="myCheckbox"&gt;这是一个复选框&lt;/label&gt;

    &lt;!-- 嵌套 --&gt;
    &lt;label&gt;
        &lt;!-- radio：定义单选框 --&gt;
        &lt;input type="radio" name="gender"&gt; 男
    &lt;/label&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
     <hr/>
     <ul>
      <li>
       选项B正确：
       <span style="background-color:#fbd4d0">
        当使用for属性关联方式时，&lt;label&gt;元素的for属性值必须与目标表单控件的id属性值完全一致
       </span>
       ！这样才能建立正确的关联关系。具体例子如上。
      </li>
     </ul>
     <hr/>
     <ul>
      <li>
       选项C错误：&lt;label&gt;元素内部的&lt;a&gt;链接仍然具有其正常的链接功能，点击时会触发页面跳转。所以&lt;label&gt;的点击行为不会阻止或覆盖链接的默认行为。
      </li>
      <li>
       代码示例。
      </li>
     </ul>
