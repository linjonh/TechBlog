    <h3>
     <a id="MySQL__11">
     </a>
     一.MySQL 下载
    </h3>
    <p>
     下载地址：
     <a href="https://dev.mysql.com/downloads/mysql/" rel="nofollow">
      MySQL :: Download MySQL Community Server
     </a>
    </p>
    <p>
     建议下载 dmg 后缀的文件
    </p>
    <p>
     可以用移动硬盘保存
    </p>
    <p>
     可以保存在 csdn
    </p>
    <p>
     可以使用迅雷下载
    </p>
    <p>
     <img alt="image-20230416110027318" src="https://i-blog.csdnimg.cn/blog_migrate/4ff9625054043c3ac75438faa24834a9.png"/>
    </p>
    <p>
     <img alt="img" src="https://i-blog.csdnimg.cn/blog_migrate/6e124406de4cc78322e1b1eaba6c39ed.png"/>
    </p>
    <h3>
     <a id="MySQL__27">
     </a>
     二.MySQL 安装
    </h3>
    <p>
     安装很简单，一直 下一步 就可以啦，安装过程会让你设置
     <strong>
      初始密码，这个要自己记录下来
     </strong>
    </p>
    <p>
     我的 mysql 的密码暂时设为:xxxx
    </p>
    <h3>
     <a id="MySQL__33">
     </a>
     三.MySQL 配置
    </h3>
    <h4>
     <a id="1_MySQL_35">
     </a>
     1.打开启动 MySQL
    </h4>
    <p>
     打开系统偏好设置，在最下方找到 MySQL，然后点击
     <code>
      Start MySQL Server
     </code>
    </p>
    <h4>
     <a id="2MySQL__39">
     </a>
     2.MySQL 的目录文件
    </h4>
    <pre><code>cd /usr/local/mysql/bin
ls
</code></pre>
    <p>
     <img alt="img" src="https://i-blog.csdnimg.cn/blog_migrate/77c25a2da468c78c69d644fca6887bc3.png"/>
    </p>
    <h4>
     <a id="3_48">
     </a>
     3.在终端加入环境路径
    </h4>
    <pre><code>cd ~
open .bash_profile
</code></pre>
    <p>
     在打开的文件中，末尾添加如下内容，保存
    </p>
    <pre><code>export PATH=$PATH:/usr/local/mysql/bin
alias mysql=/usr/local/mysql/bin/mysql
alias mysqladmin=/usr/local/mysql/bin/mysqladmin
</code></pre>
    <h4>
     <a id="4_MySQL_63">
     </a>
     4.登录 MySQL
    </h4>
    <pre><code>mysql -u root -p
</code></pre>
    <p>
     提示输入密码，注意输入的密码是不显示的。
    </p>
    <p>
     然后我们也可以看到它的版本号：Server version: 8.0.19 MySQL Community Server - GPL
    </p>
    <p>
     <img alt="img" src="https://i-blog.csdnimg.cn/blog_migrate/e5800cb1112c9d2e71d2f97f7c31a797.png"/>
    </p>
    <h4>
     <a id="5_75">
     </a>
     5.退出,强制关闭
    </h4>
    <h5>
     <a id="1_MySQL_77">
     </a>
     1.退出 MySQL
    </h5>
    <p>
     执行 quit
    </p>
    <p>
     [外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传(img-QHcIGN8s-1661690758815)(http://qinyingjie.cn/pic/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L1l1REJM,size_16,color_FFFFFF,t_70-20220828204157881.png)]
    </p>
    <h5>
     <a id="2_MySQL_83">
     </a>
     2.关闭 MySQL
    </h5>
    <p>
     同理如果你要关闭 MySQL： 打开系统偏好设置，在最下方找到 MySQL，然后点击
     <code>
      Stop MySQL Server
     </code>
    </p>
    <h5>
     <a id="3_MySQL_87">
     </a>
     3.强制关闭 MySQL
    </h5>
    <p>
     有时候执行
     <strong>
      （2）关闭 MySQL
     </strong>
     ，不起作用，那么这样操作试试
    </p>
    <p>
     前往，使用工具，活动监视器，搜索 mysql，点击左上角的叉，依次关掉这两个进程
    </p>
    <p>
     <img alt="img" src="https://i-blog.csdnimg.cn/blog_migrate/e6e37fecfce10a83426fe36026937860.png"/>
    </p>
    <p>
     另外：如果想重新启动 MySQL 不好使的话，不妨先重启电脑再试
    </p>
