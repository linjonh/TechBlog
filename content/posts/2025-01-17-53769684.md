    <p>
     Windows下使用DOS命令进入MySQL数据库
     <br/>
     <strong>
      前提条件：
     </strong>
     <br/>
     MySQL数据库5.5.25（已完成安装并配置环境变量）
     <br/>
     win7或win10
     <br/>
     <strong>
      具体步骤：
     </strong>
     <br/>
     1.桌面左下角windows图标–搜索框内输入cmd，结果如图所示，点击cmd.exe，或者使用快捷键Windows键（在键盘上有个Windows标志的按键）+R输入cmd后回车。
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220213926890?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220213955203?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
       <br/>
       2.启动mysql数据库：在出来的DOS命令窗口中输入 net start mysql，或者使用快捷键Windows键（在键盘上有个Windows标志的按键）+ R直接输入net start mysql后回车。（另附：关闭的命令为net stop mysql）
       <br/>
       <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220214013974?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
        <br/>
        <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220214034209?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
         <br/>
         3.在DOS命令窗口输入 mysql -hlocalhost -uroot -p回车 进入mysql数据库，其中-h表示服务器名，localhost表示本地；-u为数据库用户名，root是mysql默认用户名；-p为密码，如果设置了密码，可直接在-p后链接输入，如：-p123456，用户没有设置密码，显示Enter password时，直接回车即可。注意，如果你的mysql没有安装在C盘下，你需要先使用DOS命令进入mysql的安装目录下的bin目录中。以我的电脑为例，方法如下：输入D:进入D盘，在输入cd D:\Tools\MySQL5.5.25\bin进入到mysql的bin目录下才可以输入 mysql -hlocalhost -uroot -p
         <br/>
         <img alt="在dos下运行net  start mysql 不能启动mysql！提示发生系统错误 5；拒绝访问！切换到管理员模式就可以启动了。所以要以管理员身份来运行cmd程序来启动mysql。" src="https://img-blog.csdn.net/20161220214057209?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
          <br/>
          4.当显示此界面时表示成功链接到了mysql数据库
          <br/>
          <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220214134721?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
           <br/>
           5.输入show databases；显示你有的数据库（mysql数据库中的命令必须以分号结尾“；”）
           <br/>
           <img alt="这里写图片描述" src="https://img-blog.csdn.net/20161220214153116?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYTI1NDcyMzA3Nw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">

    <p>
     <strong>
      注意事项
     </strong>
     <br/>
     在dos下运行net start mysql 不能启动mysql！提示
     <strong>
      发生系统错误 5；拒绝访问！
     </strong>
     <br/>
     切换到管理员模式就可以启动了。所以要以管理员身份来运行cmd程序来启动mysql。
