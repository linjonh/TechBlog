    <p>
     将HTML网页内容转换为 PDF 格式能方便文档的后续打印、存档和分享等。之前介绍过如何
     <a href="https://blog.csdn.net/Eiceblue/article/details/135823964">
      通过QT插件将HTML转为PDF文件
     </a>
     ，本文将介绍另一个新的转换方法，通过谷歌浏览器Chrome插件将HTML网页转PDF文件。
    </p>
    <blockquote>
     <ul>
      <li>
       <p>
        首先需要安装
        <a href="https://www.e-iceblue.cn/Introduce/Spire-PDF-NET.html" rel="nofollow">
         <strong>
          Spire.PDF for .NET 库
         </strong>
        </a>
        （10.7.21版本及以上）。可以通过此链接下载产品包后手动添加引用，或者直接通过
        <a href="https://www.nuget.org/packages/Spire.PDF/" rel="nofollow">
         NuGet
        </a>
        安装。
        <br/>
        <a href="https://www.e-iceblue.cn/Downloads/Spire-PDF-NET.html" rel="nofollow">
         https://www.e-iceblue.cn/Downloads/Spire-PDF-NET.html
        </a>
       </p>
      </li>
      <li>
       <p>
        此外还需要用到谷歌浏览器插件，请确保系统中安装了
        <strong>
         chrome.exe
        </strong>
        。
       </p>
      </li>
     </ul>
    </blockquote>
    <h3>
     <a id="C_ChromeHTMLPDF_9">
     </a>
     C# 通过Chrome插件将HTML网页转换为PDF文件
    </h3>
    <p>
     Spire.PDF for .NET新增了
     <strong>
      <code>
       ChromeHtmlConverter.ConvertToPdf()
      </code>
     </strong>
     方法，支持使用 Chrome 浏览器插件将 HTML 网页转换为 PDF。该方法中的 3 个参数为：
    </p>
    <ul>
     <li>
      <code>
       string input
      </code>
      ：输入 HTML 文件路径
     </li>
     <li>
      <code>
       string output
      </code>
      ：输出 PDF 文件路径
     </li>
     <li>
      <code>
       ConvertOptions
      </code>
      ：转换设置，可自定义设置转换超时、PDF 纸张大小和页边距等
     </li>
    </ul>
