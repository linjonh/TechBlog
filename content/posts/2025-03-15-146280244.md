    <h2>
     <strong>
      1. 快速释放空间的临时措施
     </strong>
    </h2>
    <ol>
     <li>
      <h3>
       <strong>
        清理 APT 缓存
       </strong>
       ：
      </h3>
      <pre><code class="language-bash">sudo apt clean # 删除所有已下载的软件包缓存（释放约数百MB）</code></pre>
     </li>
     <li>
      <h3>
       <strong>
        删除旧内核
       </strong>
       ：
      </h3>
      <pre><code class="language-bash">sudo apt autoremove --purge   # 卸载旧内核和无用依赖包</code></pre>
     </li>
     <li>
      <h3>
       <strong>
        清理日志文件
       </strong>
       ：
      </h3>
      <pre><code class="language-bash">sudo journalctl --vacuum-time=7d  # 删除超过7天的系统日志
sudo rm -rf /var/log/*.gz /var/log/*.old  # 删除压缩和旧日志</code></pre>
     </li>
    </ol>
    <h2>
     2.
     <strong>
      扩展根分区
     </strong>
    </h2>
    <p>
     <strong>
      扩展虚拟机磁盘
     </strong>
     （在宿主机操作，如 VMware/VirtualBox）：
    </p>
    <p>
     关闭虚拟机 → 调整磁盘大小（如从 20GB 扩展到 100GB）。
    </p>
    <p>
     <img alt="" height="546" src="https://i-blog.csdnimg.cn/direct/e06f51eb719b478a96375e2a75cfb1c6.png" width="725"/>
    </p>
    <p>
     <strong>
      扩展分区和文件系统
     </strong>
     （在虚拟机内操作）：
    </p>
    <p>
     <strong>
      使用
      <code>
       gparted
      </code>
      工具：
     </strong>
    </p>
    <pre><code class="language-bash">sudo apt install gparted       # 安装分区工具
sudo gparted                  # 启动图形化界面</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        操作步骤
       </strong>
       ：
       <br/>
       <strong>
        选择磁盘（如
        <code>
         /dev/sda3
        </code>
        ）
       </strong>
      </p>
     </li>
    </ul>
    <p>
     <img alt="" height="362" src="https://i-blog.csdnimg.cn/direct/615d05a1e8b445d2940603e9f5da3807.png" width="776"/>
    </p>
    <p>
     <strong>
      右键根分区（如
      <code>
       /dev/sda3
      </code>
      ）→ Resize/Move → 拖动调整大小(或输入具体大小)。
     </strong>
    </p>
    <p>
     <img alt="" height="524" src="https://i-blog.csdnimg.cn/direct/dac8c964be1549a996e0c002023ccfb4.png" width="800"/>
    </p>
    <p>
     <img alt="" height="460" src="https://i-blog.csdnimg.cn/direct/33237650f8174814b8f071579cebc8aa.png" width="744"/>
    </p>
    <p>
     <strong>
      点击界面上的绿色“✔”，完成扩展！
     </strong>
