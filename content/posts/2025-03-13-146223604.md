    <h2>
     题目
    </h2>
    <p>
     给定一个非负整数
     <em>
      <code>
       numRows
      </code>
      ，
     </em>
     生成「杨辉三角」的前
     <em>
      <code>
       numRows
      </code>
     </em>
     行。
    </p>
    <p>
     在「杨辉三角」中，每个数是它左上方和右上方的数的和。
    </p>
    <h2>
     示例
    </h2>
    <blockquote>
     <p>
      <strong>
       示例 1:
      </strong>
     </p>
     <pre><strong>输入:</strong> numRows = 5
<strong>输出:</strong> [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
</pre>
     <p>
      <strong>
       示例 2:
      </strong>
     </p>
     <pre><strong>输入:</strong> numRows = 1
<strong>输出:</strong> [[1]]</pre>
    </blockquote>
    <h2>
     分析
    </h2>
    <h3>
     数学构造法
    </h3>
    <p>
     首先创建一个二维向量
     <code>
      triangle
     </code>
     用于存储杨辉三角的每一行。
    </p>
    <p>
     杨辉三角的第一行只有一个元素
     <code>
      1
     </code>
     ，将其添加到
     <code>
      triangle
     </code>
     中。
    </p>
    <p>
     从第二行开始，使用
     <code>
      for
     </code>
     循环逐行生成。对于每一行，先创建一个长度为
     <code>
      i + 1
     </code>
     的向量
     <code>
      row
     </code>
     ，并将其首尾元素初始化为
     <code>
      1
     </code>
     。然后，使用内层
     <code>
      for
     </code>
     循环计算中间元素的值，每个中间元素等于上一行相邻两个元素之和。最后，将生成的行添加到
     <code>
      triangle
     </code>
     中。
    </p>
    <p>
     时间复杂度：O(
     <img alt="numRows^{2}" class="mathcode" src="https://latex.csdn.net/eq?numRows%5E%7B2%7D">
      )
     </img>
    </p>
    <p>
     空间复杂度：O(1)
    </p>
    <pre><code class="language-cpp">class Solution {
public:
    std::vector&lt;std::vector&lt;int&gt;&gt; generate(int numRows) {
        std::vector&lt;std::vector&lt;int&gt;&gt; triangle;
        if (numRows == 0) return triangle;
        // 第一行
        triangle.push_back({1});
        for (int i = 1; i &lt; numRows; ++i) {
            std::vector&lt;int&gt; row(i + 1, 1);
            for (int j = 1; j &lt; i; ++j) {
                row[j] = triangle[i - 1][j - 1] + triangle[i - 1][j];
            }
            triangle.push_back(row);
        }
        return triangle;
    }
};  </code></pre>
