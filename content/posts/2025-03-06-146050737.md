    <br/>
    在 C++20 中，聚合初始化（Aggregate Initialization）得到了进一步的扩展和改进，尤其是在使用圆括号进行初始化时。这一特性不仅提升了代码的灵活性，还为开发者提供了更简洁的初始化方式。本文将详细介绍 C++20 中聚合初始化的新特性，以及如何在实际开发中使用括号进行聚合初始化。
    <p>
    </p>
    <h3>
     <a id="1__5">
     </a>
     1. 聚合初始化简介
    </h3>
    <p>
     聚合初始化是一种使用大括号
     <code>
      {}
     </code>
     或圆括号
     <code>
      ()
     </code>
     对聚合类型（如数组、结构体、类等）进行初始化的方法。聚合类型必须满足以下条件：
    </p>
    <ul>
     <li>
      没有用户定义的构造函数。
     </li>
     <li>
      没有私有或受保护的非静态数据成员。
     </li>
     <li>
      没有虚函数。
     </li>
     <li>
      没有基类。
     </li>
    </ul>
    <p>
     例如，以下结构体是一个典型的聚合类型：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">struct</span> <span class="token class-name">Point</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">int</span> x<span class="token punctuation">;</span>
    <span class="token keyword">int</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     在 C++20 之前，聚合初始化只能使用大括号
     <code>
      {}
     </code>
     ，例如：
    </p>
    <pre><code class="prism language-cpp">Point p <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">}</span><span class="token punctuation">;</span> <span class="token comment">// 使用大括号初始化</span>
</code></pre>
    <p>
     然而，C++20 引入了使用圆括号
     <code>
      ()
     </code>
     进行聚合初始化的能力，这使得语法更加灵活。
    </p>
    <h3>
     <a id="2_C20__31">
     </a>
     2. C++20 中的括号聚合初始化
    </h3>
    <p>
     C++20 允许使用圆括号进行聚合初始化，这在某些场景下可以减少代码的复杂性。例如：
    </p>
    <pre><code class="prism language-cpp">Point <span class="token function">p</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 使用圆括号初始化</span>
</code></pre>
    <p>
     这种初始化方式与大括号初始化的效果相同，但语法上更加简洁。
    </p>
    <h4>
     <a id="21_Designated_Initializers_41">
     </a>
     2.1 指定初始化器（Designated Initializers）
    </h4>
    <p>
     C++20 还引入了指定初始化器（Designated Initializers），允许开发者通过成员名称显式指定初始化值。这种方式不仅提高了代码的可读性，还避免了因成员顺序错误而导致的问题。
    </p>
    <p>
     例如：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">struct</span> <span class="token class-name">AudioInfo</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">int</span> channel_num<span class="token punctuation">;</span>
    <span class="token keyword">int</span> channel_layout<span class="token punctuation">;</span>
    <span class="token keyword">int</span> sample_rate<span class="token punctuation">;</span>
    <span class="token keyword">int</span> frame_count<span class="token punctuation">;</span>
    <span class="token keyword">int</span> bit_depth<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

AudioInfo info<span class="token punctuation">{<!-- --></span><span class="token punctuation">.</span>channel_num <span class="token operator">=</span> <span class="token number">1</span><span class="token punctuation">,</span> <span class="token punctuation">.</span>bit_depth <span class="token operator">=</span> <span class="token number">16</span><span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     在上述代码中，
     <code>
      info
     </code>
     的
     <code>
      channel_num
     </code>
     和
     <code>
      bit_depth
     </code>
     被显式初始化，而其他成员将使用默认值（如果未提供默认值，则初始化为零）。
    </p>
    <h4>
     <a id="22__61">
     </a>
     2.2 嵌套聚合初始化
    </h4>
    <p>
     对于嵌套的聚合类型，C++20 支持递归地使用指定初始化器。例如：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">struct</span> <span class="token class-name">Point</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">int</span> x<span class="token punctuation">;</span>
    <span class="token keyword">int</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">struct</span> <span class="token class-name">Color</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">int</span> r<span class="token punctuation">;</span>
    <span class="token keyword">int</span> g<span class="token punctuation">;</span>
    <span class="token keyword">int</span> b<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">struct</span> <span class="token class-name">ComplexPoint</span> <span class="token punctuation">{<!-- --></span>
    Point pt<span class="token punctuation">;</span>
    Color color<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

ComplexPoint cp<span class="token punctuation">{<!-- --></span>
   <span class="token punctuation">.</span>pt <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">.</span>x <span class="token operator">=</span> <span class="token number">66</span><span class="token punctuation">,</span> <span class="token punctuation">.</span>y <span class="token operator">=</span> <span class="token number">99</span><span class="token punctuation">}</span><span class="token punctuation">,</span>
   <span class="token punctuation">.</span>color <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">.</span>r <span class="token operator">=</span> <span class="token number">255</span><span class="token punctuation">,</span> <span class="token punctuation">.</span>g <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">,</span> <span class="token punctuation">.</span>b <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     这种方式使得嵌套结构的初始化更加直观。
    </p>
    <h3>
     <a id="3__90">
     </a>
     3. 使用括号初始化数组
    </h3>
    <p>
     C++20 还允许使用圆括号初始化数组，这在某些情况下可以减少代码的冗余。例如：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">int</span> arr<span class="token punctuation">[</span><span class="token number">3</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 使用圆括号初始化数组</span>
</code></pre>
    <p>
     这种方式与大括号初始化的效果相同，但语法上更加简洁。
    </p>
    <h3>
     <a id="4__100">
     </a>
     4. 注意事项
    </h3>
    <p>
     尽管 C++20 提供了更灵活的聚合初始化方式，但在使用时仍需注意以下几点：
    </p>
    <ol>
     <li>
      <strong>
       窄化转换
      </strong>
      ：在聚合初始化中，C++20 允许窄化转换（如从
      <code>
       double
      </code>
      转换为
      <code>
       int
      </code>
      ），但这种转换可能导致数据丢失。
     </li>
     <li>
      <strong>
       成员顺序
      </strong>
      ：即使使用指定初始化器，成员的初始化顺序仍需与声明顺序一致。
     </li>
     <li>
      <strong>
       联合体初始化
      </strong>
      ：对于联合体，只能为其一个成员提供初始化值。
     </li>
    </ol>
    <h3>
     <a id="5__108">
     </a>
     5. 实际应用场景
    </h3>
    <p>
     C++20 的聚合初始化特性在以下场景中特别有用：
    </p>
    <ul>
     <li>
      <strong>
       数据类初始化
      </strong>
      ：对于仅包含数据成员的类或结构体，聚合初始化提供了一种简洁的初始化方式。
     </li>
     <li>
      <strong>
       嵌套配置类
      </strong>
      ：在处理多层嵌套的配置结构时，指定初始化器可以显著提高代码的可读性。
     </li>
     <li>
      <strong>
       数组初始化
      </strong>
      ：使用圆括号初始化数组可以减少代码的复杂性。
     </li>
    </ul>
    <h3>
     <a id="6__116">
     </a>
     6. 总结
    </h3>
    <p>
     C++20 对聚合初始化的扩展，尤其是引入圆括号初始化和指定初始化器，为开发者提供了更灵活、更直观的初始化方式。这些特性不仅简化了代码，还减少了因成员顺序错误而导致的潜在问题。然而，在使用时仍需注意窄化转换和成员顺序等限制。
    </p>
    <p>
     通过合理利用 C++20 的聚合初始化特性，开发者可以编写出更加简洁、高效且易于维护的代码。
    </p>
