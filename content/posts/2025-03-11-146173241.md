    <h4>
     <a id="_CSS__1">
     </a>
     什么是 CSS 权重？
    </h4>
    <p>
     CSS 权重（Specificity）是浏览器用来决定哪个样式规则优先级更高的机制。当多个样式规则同时作用于同一个元素时，权重高的规则会覆盖权重低的规则。
    </p>
    <p>
     权重的计算基于选择器的类型，通常用四组数字表示，例如
     <code>
      0,0,0,0
     </code>
     。从左到右，每组数字分别代表：
    </p>
    <ol>
     <li>
      <strong>
       行内样式
      </strong>
      （如
      <code>
       style="color: red;"
      </code>
      ）：权重为
      <code>
       1,0,0,0
      </code>
      。
     </li>
     <li>
      <strong>
       ID 选择器
      </strong>
      （如
      <code>
       #myDiv
      </code>
      ）：权重为
      <code>
       0,1,0,0
      </code>
      。
     </li>
     <li>
      <strong>
       类选择器、伪类选择器、属性选择器
      </strong>
      （如
      <code>
       .myClass
      </code>
      、
      <code>
       :hover
      </code>
      、
      <code>
       [type="text"]
      </code>
      ）：权重为
      <code>
       0,0,1,0
      </code>
      。
     </li>
     <li>
      <strong>
       元素选择器、伪元素选择器
      </strong>
      （如
      <code>
       div
      </code>
      、
      <code>
       ::before
      </code>
      ）：权重为
      <code>
       0,0,0,1
      </code>
      。
     </li>
     <li>
      <strong>
       通配符和继承
      </strong>
      ：权重为
      <code>
       0,0,0,0
      </code>
      。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_15">
     </a>
     权重计算规则
    </h4>
    <ol>
     <li>
      <strong>
       从左到右逐级比较
      </strong>
      ：如果某一位数值相同，则比较下一位。
     </li>
     <li>
      <strong>
       权重相同
      </strong>
      ：后出现的样式规则会覆盖前面的规则。
     </li>
     <li>
      <strong>
       <code>
        !important
       </code>
      </strong>
      ：优先级最高，会覆盖其他声明，但应谨慎使用。
     </li>
     <li>
      <strong>
       继承的样式
      </strong>
      ：权重为
      <code>
       0
      </code>
      ，低于直接样式和通配符。
     </li>
     <li>
      <strong>
       复合选择器
      </strong>
      ：权重叠加，但不会进位。
     </li>
    </ol>

    <h4>
     <a id="_209">
     </a>
     总结
    </h4>
    <p>
     通过以上案例，我们可以清晰地理解 CSS 权重的计算方式和优先级规则：
    </p>
    <ol>
     <li>
      <strong>
       权重比较
      </strong>
      ：从左到右逐级比较，数值大的优先级高。
     </li>
     <li>
      <strong>
       <code>
        !important
       </code>
      </strong>
      ：优先级最高，但应谨慎使用。
     </li>
     <li>
      <strong>
       继承
      </strong>
      ：权重为
      <code>
       0
      </code>
      ，低于直接样式和通配符。
     </li>
     <li>
      <strong>
       就近原则
      </strong>
      ：继承样式距离更近的祖先优先级更高。
     </li>
    </ol>
