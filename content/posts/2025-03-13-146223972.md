    <h3>
     <a id="1_Tuple_3">
     </a>
     1 元组（Tuple）
    </h3>
    <p>
     元组是有序且不可更改的集合。在 Python 中，元组是用
     <strong>
      圆括号
     </strong>
     编写的。
    </p>
    <p>
     实例 创建元组：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span>  <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="2__16">
     </a>
     2 访问元组元素
    </h3>
    <p>
     可以通过引用方括号内的索引号来访问元组元素：
    </p>
    <p>
     实例 打印元组中的第二个项目：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span>   <span class="token string">"banana"</span>

</code></pre>
    <h3>
     <a id="3__29">
     </a>
     3 负索引
    </h3>
    <p>
     负索引表示从末尾开始，-1 表示最后一个项目，-2 表示倒数第二个项目，依此类推。
    </p>
    <p>
     实例 打印元组的最后一个项目：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">[</span><span class="token operator">-</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span>   <span class="token string">"cherry"</span>

</code></pre>
    <h3>
     <a id="4__42">
     </a>
     4 索引范围
    </h3>
    <p>
     可以通过指定范围的起点和终点来指定索引范围。指定范围后，返回值将是带有指定项目的新元组。
    </p>
    <p>
     实例 返回第三、第四、第五个项目：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">,</span> <span class="token string">"orange"</span><span class="token punctuation">,</span> <span class="token string">"kiwi"</span><span class="token punctuation">,</span> <span class="token string">"melon"</span><span class="token punctuation">,</span> <span class="token string">"mango"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">[</span><span class="token number">2</span><span class="token operator">:</span><span class="token number">5</span><span class="token punctuation">]</span><span class="token punctuation">)</span>  <span class="token punctuation">(</span><span class="token string">"cherry"</span><span class="token punctuation">,</span> <span class="token string">"orange"</span><span class="token punctuation">,</span> <span class="token string">"kiwi"</span><span class="token punctuation">)</span>

</code></pre>
    <p>
     <strong>
      注释：搜索将从索引 2（包括）开始，到索引 5（不包括）结束。
     </strong>
    </p>
    <p>
     <strong>
      请记住，第一项的索引为 0。
     </strong>
    </p>
    <h3>
     <a id="5__59">
     </a>
     5 负索引范围
    </h3>
    <p>
     如果要从元组的末尾开始搜索，请指定负索引：
    </p>
    <p>
     实例 此例将返回从索引 -4（包括）到索引 -1（排除）的项目：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">,</span> <span class="token string">"orange"</span><span class="token punctuation">,</span> <span class="token string">"kiwi"</span><span class="token punctuation">,</span> <span class="token string">"melon"</span><span class="token punctuation">,</span> <span class="token string">"mango"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">[</span><span class="token operator">-</span><span class="token number">4</span><span class="token operator">:</span><span class="token operator">-</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span>  <span class="token punctuation">(</span><span class="token string">"orange"</span><span class="token punctuation">,</span> <span class="token string">"kiwi"</span><span class="token punctuation">,</span> <span class="token string">"melon"</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="6__72">
     </a>
     6 更改元组值
    </h3>
    <p>
     创建元组后，您将无法更改其值。元组是不可变的，或者也称为恒定的。
    </p>
    <p>
     但是有一种解决方法。您可以将元组转换为列表，更改列表，然后将列表转换回元组。
    </p>
    <p>
     实例 把元组转换为列表即可进行更改：
    </p>
    <pre><code class="prism language-java">x <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
y <span class="token operator">=</span> <span class="token function">list</span><span class="token punctuation">(</span>x<span class="token punctuation">)</span>
y<span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token string">"kiwi"</span>
x <span class="token operator">=</span> <span class="token function">tuple</span><span class="token punctuation">(</span>y<span class="token punctuation">)</span>

<span class="token function">print</span><span class="token punctuation">(</span>x<span class="token punctuation">)</span>    <span class="token punctuation">(</span><span class="token char">'apple'</span><span class="token punctuation">,</span> <span class="token char">'kiwi'</span><span class="token punctuation">,</span> <span class="token char">'cherry'</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="7__91">
     </a>
     7 遍历元组
    </h3>
    <p>
     使用
     <strong>
      for 循环
     </strong>
     遍历元组项目。
    </p>
    <p>
     实例 遍历项目并打印值：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token keyword">for</span> x in thistuple<span class="token operator">:</span>
    <span class="token function">print</span><span class="token punctuation">(</span>x<span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="8__105">
     </a>
     8 检查元素是否存在
    </h3>
    <p>
     要确定元组中是否存在指定的项，请使用
     <code>
      in
     </code>
     关键字：
    </p>
    <p>
     实例 检查元组中是否存在 “apple”：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token keyword">if</span> <span class="token string">"apple"</span> in thistuple<span class="token operator">:</span>
    <span class="token function">print</span><span class="token punctuation">(</span><span class="token string">"Yes, 'apple' is in the fruits tuple"</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="9__119">
     </a>
     9 元组长度
    </h3>
    <p>
     要确定元组有多少项，请使用
     <code>
      len()
     </code>
     方法：
    </p>
    <p>
     实例 打印元组中的项目数量：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span><span class="token function">len</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span><span class="token punctuation">)</span>    <span class="token number">3</span>

</code></pre>
    <h3>
     <a id="10__132">
     </a>
     10 添加元素
    </h3>
    <p>
     元组一旦创建，您就无法向其添加元素。
     <strong>
      元组是不可改变的
     </strong>
     。
    </p>
    <p>
     实例 您无法向元组添加项目：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
thistuple<span class="token punctuation">[</span><span class="token number">3</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token string">"orange"</span> 会引发错误：<span class="token class-name">TypeError</span><span class="token operator">:</span> <span class="token char">'tuple'</span> object does not support item assignment
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span>    <span class="token punctuation">(</span><span class="token char">'apple'</span><span class="token punctuation">,</span> <span class="token char">'banana'</span><span class="token punctuation">,</span> <span class="token char">'cherry'</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="11__146">
     </a>
     11 创建有一个元素的元组
    </h3>
    <p>
     如需创建仅包含一个元素的元组，您
     <strong>
      必须在该元素后添加一个逗号
     </strong>
     ，否则 Python 无法将变量识别为元组。
    </p>
    <p>
     实例 单项元组，别忘了逗号：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span><span class="token function">type</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span><span class="token punctuation">)</span>  <span class="token operator">&lt;</span><span class="token keyword">class</span> <span class="token char">'tuple'</span><span class="token operator">&gt;</span>

#不是元组
thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">)</span>
<span class="token function">print</span><span class="token punctuation">(</span><span class="token function">type</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span><span class="token punctuation">)</span>   <span class="token operator">&lt;</span><span class="token keyword">class</span> <span class="token char">'str'</span><span class="token operator">&gt;</span>

</code></pre>
    <h3>
     <a id="12__163">
     </a>
     12 删除元素
    </h3>
    <p>
     <strong>
      注释：无法删除元组中的元素。
     </strong>
    </p>
    <p>
     元组是不可更改的，因此无法从中删除元素，但可以完全删除元组：
    </p>
    <p>
     实例
     <code>
      del
     </code>
     关键字可以完全删除元组：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>
del thistuple

<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span> 这会引发错误，因为元组已不存在。

</code></pre>
    <h3>
     <a id="13__180">
     </a>
     13 合并两个元组
    </h3>
    <p>
     如需连接两个或多个元组，您可以使用 + 运算符：
    </p>
    <p>
     实例 合并这个元组：
    </p>
    <pre><code class="prism language-java">tuple1 <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token string">"a"</span><span class="token punctuation">,</span> <span class="token string">"b"</span> <span class="token punctuation">,</span> <span class="token string">"c"</span><span class="token punctuation">)</span>
tuple2 <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">)</span>

tuple3 <span class="token operator">=</span> tuple1 <span class="token operator">+</span> tuple2
<span class="token function">print</span><span class="token punctuation">(</span>tuple3<span class="token punctuation">)</span>    <span class="token punctuation">(</span><span class="token char">'a'</span><span class="token punctuation">,</span> <span class="token char">'b'</span><span class="token punctuation">,</span> <span class="token char">'c'</span><span class="token punctuation">,</span> <span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">)</span>

</code></pre>
    <h3>
     <a id="14_tuple__196">
     </a>
     14 tuple() 构造函数
    </h3>
    <p>
     也可以使用
     <code>
      tuple()
     </code>
     构造函数来创建元组。
    </p>
    <p>
     实例 使用tuple() 方法来创建元组：
    </p>
    <pre><code class="prism language-java">thistuple <span class="token operator">=</span> <span class="token function">tuple</span><span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span><span class="token punctuation">)</span> 请注意双括号
<span class="token function">print</span><span class="token punctuation">(</span>thistuple<span class="token punctuation">)</span>  <span class="token punctuation">(</span><span class="token string">"apple"</span><span class="token punctuation">,</span> <span class="token string">"banana"</span><span class="token punctuation">,</span> <span class="token string">"cherry"</span><span class="token punctuation">)</span>

</code></pre>
