    <h4>
     <a id="everything_0">
     </a>
     everything是什么
    </h4>
    <p>
     是运行于Windows系统，基于
     <code>
      文件、文件夹名称
     </code>
     的快速搜索引擎。
     <br/>
     安装后会自动搜索电脑上的各个盘，并在搜索之前把所用的文件和文件夹都列出来，所以其搜索速度非常快！！！
    </p>
    <h4>
     <a id="everything_3">
     </a>
     安装everything
    </h4>
    <p>
     去官网下载，
     <a href="https://www.voidtools.com/zh-cn/" rel="nofollow">
      https://www.voidtools.com/zh-cn/
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a145d41726aef773cbfe863df6eb9e1f.png">
      <br/>
      下载安装版即可，若是64位电脑下载安装版64位。
     </img>
    </p>
    <h4>
     <a id="1_9">
     </a>
     1、根据文件名进行搜索
    </h4>
    <p>
     最常用的，输入要搜索的文件/文件夹的名称即可，不到一秒结果都会列在下面。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c2900adfe6175f8c721d17a1689c5faf.png">
      <br/>
      这里有几种不同的搜索方式：
     </img>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        搜索选项
       </th>
       <th>
        作用
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        区分大小写
       </td>
       <td>
        搜索windows与 Windows不一样
       </td>
      </tr>
      <tr>
       <td>
        全字匹配
       </td>
       <td>
        勾选后，会搜索出和关键字一样的文件，并不包含含有关键字的文件
       </td>
      </tr>
      <tr>
       <td>
        匹配路径
       </td>
       <td>
        勾选后，会搜索出路径中含有搜索关键字的文件
       </td>
      </tr>
      <tr>
       <td>
        是否匹配变音标识
       </td>
       <td>
        用的较少
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     可在“搜索”选项中勾选相应的选项。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/164e596e7058428de2e19d17c33a57cc.png">
      <br/>
      <strong>
       还有，就是使用
       <code>
        通配符
       </code>
       的搜索：
      </strong>
     </img>
    </p>
    <pre><code>* ： 匹配 0 个或多个字符. 	 
? ： 匹配 1 个字符.
</code></pre>
    <p>
     搜索
     <code>
      a*.c
     </code>
     ,会列出所有以a开头，以.c结尾的文件。
     <br/>
     搜索
     <code>
      ?.c
     </code>
     ,会列出名字只有一个字符，以.c结尾的文件。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c73d33593e43229282654d8d571bab8c.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/315ef0721fdaea84cec26b922b8dfda0.png"/>
     </img>
    </p>
    <h4>
     <a id="2_31">
     </a>
     2、在指定的搜索路径中查找相应的文件
    </h4>
    <p>
     在这里，先要知道everything的搜索规则：
    </p>
    <pre><code>操作符  				含义
space（即空格）  		与 (AND) ，表示需要同时满足多个条件
|	     			或 (OR)，条件相或，只需满足其中一个条件
!	     			非 (NOT)，与条件相反
</code></pre>
    <p>
     只需要将要搜索的文件名后面加上指定的路径即可，不要忘了两者之间
     <strong>
      用空格隔开
     </strong>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/92e8ce92477913ab38b695d1031ee480.png"/>
    </p>
    <h4>
     <a id="3_42">
     </a>
     3、根据文件的修改时间进行搜索
    </h4>
    <p>
     这里就需要了解一下everything相关函数的语法了：
     <br/>
     <strong>
      帮助选项
     </strong>
     –&gt;
     <strong>
      搜索语法
     </strong>
     ： 里面有everything详细的语法，有关于函数的说明。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/86a7b10b363297f5a137dd01d6ad1642.png">
      <br/>
      我们需要的是
      <strong>
       dm: 函数
      </strong>
      ，具体使用如下：
      <br/>
      <code>
       关键字 dm:20200402T1215
      </code>
      <br/>
      即在关键字后面加上一个时间条件（
      <strong>
       空格隔开
      </strong>
      ）；
      <br/>
      20200402T1215表示2020年4月2日，12点12分（如果要精确到小时和分，需在前面加上"T"）。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f0191d69b58624e75c067750449a95e6.png">
       <br/>
       <code>
        关键字 dm:20200402T1215-20200402T2000
       </code>
       <br/>
       即搜索在两个时间点之间的文件。
      </img>
     </img>
    </p>
    <h4>
     <a id="4_53">
     </a>
     4、按文件大小进行搜索
    </h4>
    <p>
     加上条件
     <code>
      size:100kb-300kb
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2a2d0b3100f5dcad71ed1621371ea119.png"/>
     <br/>
     大小单位有：kb、mb、gb
    </p>
    <blockquote>
     <p>
      <em>
       如果需要多个条件进行约束，只需将条件加在后面，用空格隔开即可。
      </em>
     </p>
    </blockquote>
    <p>
     以上基本可以满足日常的使用了，希望可以对大家有所帮助~~
    </p>
