# harmonyos演示,HarmonyOS 2.0 跨屏演示：手机编辑内容跨到平板，自动优化排版

原标题：HarmonyOS 2.0 跨屏演示：手机编辑内容跨到平板，自动优化排版

IT之家12月16日消息 华为 Harmony OS 2.0 手机应用开发者 Beta 版如期而至，华为消费者业务软件部总裁王成录表示将开放 HarmonyOS 2.0 系统能力、开发工具与 API 等，将为大家带来智能家居、智慧办公、出行、运动健康、影音娱乐等不同场景的全新体验。

![c4119fa668cf5a3b8d272d7520ff7873.png](https://i-blog.csdnimg.cn/blog_migrate/13271820bda9f45f794acc0adcbac320.jpeg)

IT之家曾报道，华为方面还公布了 HarmonyOS 2.0 的分布式 UI 控件内容展示，适配多种屏幕类型与交互方式。而现在多个数码博主放出了华为 HarmonyOS 2.0 的跨屏协作方式，展示了搭载 HarmonyOS 2.0 的华为手机、平板电脑、车载大屏、智能穿戴等设备的跨屏交互。

对于华为 Harmony OS 手机版本和鸿蒙平板之间的跨屏来说，发送邮件时编辑文本的体验是一件不可忽视的事。利用 Harmony OS，用户可在手机端邮箱进行编辑，并通过界面上方跨屏按钮将手机上已编辑的内容直接整体跨屏转到平板电脑中，内容和格式可以实现同步，同时还能在大屏端实现优化排版。

▲ 来源：机氪

而对于车载智慧屏与手机的跨屏来说，导航是用户体验极强的一种日常。借助 Harmony OS，用户上车后手机导航可自动实现跨越至车机界面继续导航，利用车载设备进行播报，下车的时候则反之。当然，除车机外，华为手机、大屏设备、穿戴设备等等产品之间都可以实现更「无缝」的交互。

▲ 来源：极客之选
