    <p>
     Scrapy是一个为了爬取网站数据，提取结构性数据而编写的Python应用框架。 可以应用在包括数据挖掘，信息处理或存储历史数据等一系列的程序中。
     <br/>
     Scrapy安装环境必备项：python、pip工具、wheel模块、lxml模块、twisted模块、pyOpenSSL模块（需要OpenSSL）、scrapy模块
    </p>
    <blockquote>
     <p>
      1、pip是一个安装和管理 Python 包的工具
      <br/>
      2、wheel本质上是一个 zip 包格式，用于 python 模块的安装、模块的发布
      <br/>
      3、lxml是python中处理xml的一个非常强大的库，可以非常方便的解析和生成xml文件
      <br/>
      4、twisted是用Python实现的基于事件驱动的网络引擎框架。
      <br/>
      5、pyOpenSSL是python的密码库（OpenSSL 是一个安全套接字层密码库）
     </p>
    </blockquote>
    <p>
     接下来我们看看这些工具如何安装。
    </p>
    <h2 id="一基础安装">
     一、基础安装
    </h2>
    <h3 id="1python开发环境搭建">
     1、python开发环境搭建
    </h3>
    <p>
     1）python安装
     <br/>
     python编译工具官网的地址为：
     <a href="https://www.python.org/downloads/" rel="nofollow">
      https://www.python.org/downloads/
     </a>
     ，下载好工具点击运行安装即可。
     <br/>
     2）python环境配置
     <br/>
     安装成功后在 “环境变量”-》“path”中将python安装目录和安装目录下的Scripts文件夹放入即可（如：在path环境变量中设置如下内容：C:\Program Files (x86)\Python36-32\Scripts\;C:\Program Files (x86)\Python36-32\）
     <br/>
     3）python开发工具安装及配置
     <br/>
     python 常用的IDE开发工具是PyCharm，其下载网址是：
     <a href="https://www.jetbrains.com/pycharm/download/#section=windows" rel="nofollow">
      https://www.jetbrains.com/pycharm/download/#section=windows
     </a>
     ，在改地址下载安装即可。
     <br/>
     安装好后，PyCharm工具的python环境配置如下，File-》settings打开配置窗口，按如下图配置即可
     <br/>
     <img alt="PyCharm" src="https://i-blog.csdnimg.cn/blog_migrate/56533efa79a1c92344310d3ec95c4309.png" title=""/>
    </p>
    <h3 id="2pip工具">
     2、pip工具
    </h3>
    <p>
     python安装包里默认包含了pip工具。
     <br/>
     由于本人使用的python版本是3.6的，其pip工具的版本是9.0.3，使用时会提示需要更新pip（如下图）
     <br/>
     <img alt="python" src="https://i-blog.csdnimg.cn/blog_migrate/4e14d7b64fe588c4d2eb76669cbb46ea.png" title="">
      <br/>
      所以输入提示更新命令
     </img>
    </p>
    <pre class="prettyprint"><code class="language-python hljs">python -m pip install -upgrade pip</code></pre>
    <p>
     输出如下：
     <br/>
     <img alt="pip out" src="https://i-blog.csdnimg.cn/blog_migrate/9e95e2a07c6dd0b0bbecc2b30e62c095.jpeg" title=""/>
    </p>
    <h3 id="3wheel模块">
     3、wheel模块
    </h3>
    <p>
     wheel模块安装，使用指令
    </p>
    <pre class="prettyprint"><code class="language-python hljs">pip3 install wheel</code></pre>
    <p>
     输出如下：
     <br/>
     <img alt="wheel" src="https://i-blog.csdnimg.cn/blog_migrate/f9628f00e2287e109f97371b8498a677.jpeg" title=""/>
    </p>
    <h3 id="4lxml模块">
     4、lxml模块
    </h3>
    <p>
     lxml模块，用命令 pip3 install lxml 安装，会提示缺少包的错误，因此该模块安装需要采取使用本地.whl文件，安装，该文件可以在
     <a href="https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml" rel="nofollow">
      https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml
     </a>
     下载。本人python版本是V3.6.5 32位的，因此选择了 lxml‑4.2.1‑cp36‑cp36m‑win32.whl，如下图
     <br/>
     <img alt="lxml.whl" src="https://i-blog.csdnimg.cn/blog_migrate/b95e1299fd18521fc928b04136a4e9c2.png" width="60%"/>
    </p>
    <p>
     执行如下命令，安装lxml模块
    </p>
    <pre class="prettyprint"><code class="hljs lasso">pip3 install lxml<span class="hljs-subst">-</span><span class="hljs-number">4.2</span><span class="hljs-number">.1</span><span class="hljs-attribute">-cp36</span><span class="hljs-attribute">-cp36m</span><span class="hljs-attribute">-win32</span><span class="hljs-built_in">.</span>whl</code></pre>
    <p>
     命令执行结果如下：
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/05eca0afceac95c0bb66567b23cb1b0b.jpeg" title=""/>
    </p>
    <h3 id="5pyopenssl模块">
     5、pyOpenSSL模块
    </h3>
    <p>
     pyOpenSSL安装前需要先安装了OpenSSL库，window未集成OpenSSL库，所以需要下载并且安装。
     <br/>
     1）OpenSSL安装
     <br/>
     下载地址：
     <a href="http://slproweb.com/products/Win32OpenSSL.html" rel="nofollow">
      http://slproweb.com/products/Win32OpenSSL.html
     </a>
     ，本人下载的是 Win32 OpenSSL v1.1.0h该版本（如下图）
     <br/>
     <img alt="pyOpenSSL" src="https://i-blog.csdnimg.cn/blog_migrate/3e62dba27074a0c553ea33c718d67dff.png" title="">
      <br/>
      该安装包需要VC2008以上的库支持，本人有vs2017，可以通过vs2017安装包，自选如下组件，
      <br/>
      <img alt="vs2017" src="https://i-blog.csdnimg.cn/blog_migrate/3d05708aa2263831a1bc9edcdc191c2a.png" width="50%">
       <br/>
       2）pyOpenSSL安装
       <br/>
       输入命令进行安装
      </img>
     </img>
    </p>
    <pre class="prettyprint"><code class="hljs cmake">pip3 <span class="hljs-keyword">install</span> pyOpenSSL</code></pre>
    <p>
     安装结果如下图：
     <br/>
     <img alt="pyOpenSSL" src="https://i-blog.csdnimg.cn/blog_migrate/4da184f2fc50af52b8d3889f804103c6.jpeg" title=""/>
    </p>
    <p>
     安装完成后，进入python，import OpenSSL没错误报错误，则成功。如下图验证
     <br/>
     <img alt="验证" src="https://i-blog.csdnimg.cn/blog_migrate/acfd8b5b39bc8fba57380f4bc6889b55.jpeg" title=""/>
    </p>
    <h3 id="6twisted安装">
     6、twisted安装
    </h3>
    <p>
     twisted模块依赖PyOpenSSL、Zope.Interface、PyWin32，使用pip命令安装即可
    </p>
    <pre class="prettyprint"><code class="language-python hljs"><span class="hljs-comment">#安装Zope.Interface</span>
pip3 install zope.interface
<span class="hljs-comment">#安装PyWin32</span>
pip3 install PyWin32
<span class="hljs-comment">#安装twisted</span>
pip3 install twisted</code></pre>
    <p>
     命令运行结果如下：
     <br/>
     <img alt="zope.interface" src="https://i-blog.csdnimg.cn/blog_migrate/d554f396a3435330cf1c12856fee5b7e.jpeg" title=""/>
     <br/>
     <img alt="pywin32" src="https://i-blog.csdnimg.cn/blog_migrate/3d922347d123827f262efcfd8276bd1d.jpeg" title=""/>
     <br/>
     <img alt="twisted" src="https://i-blog.csdnimg.cn/blog_migrate/753998ee6b32d6a4d54f433de5050992.jpeg" title=""/>
    </p>
    <h3 id="7安装scrapy">
     7、安装scrapy
    </h3>
    <p>
     scrapy所依赖的环境在上面6个步骤中已经安装完成，下面使用pip命令安装scrapy即可
    </p>
    <pre class="prettyprint"><code class="hljs cmake">pip3 <span class="hljs-keyword">install</span> scrapy</code></pre>
    <p>
     命令执行结果如下：
     <br/>
     <img alt="scrapy" src="https://i-blog.csdnimg.cn/blog_migrate/25a487929ab17ec071849eb05820c181.jpeg" title=""/>
    </p>
    <p>
     通过上述步骤，即将scrapy开发环境搭建完成。
    </p>
    <h2 id="二pychram安装python模块">
     二、PyChram安装python模块
    </h2>
    <h3 id="1python模块通用安装手法">
     1、python模块通用安装手法
    </h3>
    <p>
     在上述 基础安装-&gt;python开发环境搭建中已经讲到PyChram的安装和在PyCharm中的python库的设置，在此将介绍，PyChram中如何添加如pyOpenSSL、lxml等模块的方式。
     <br/>
     通过File -&gt; Settings，在Settings弹出框左侧找到 Project Interpreter，点击 Project Interpreter，在右侧找到 ”+“进行模块的添加即可，如下图
     <br/>
     <img alt="添加模块" src="https://i-blog.csdnimg.cn/blog_migrate/854d1cce992e6573c8a1c798f31b99be.png" title=""/>
    </p>
    <h3 id="2pychram集成scrapy框架">
     2、PyChram集成Scrapy框架
    </h3>
    <p>
     1）在工程根目录找到
     <strong>
      init
     </strong>
     .py，填入如下程序
    </p>
    <pre class="prettyprint"><code class="language-python hljs"><span class="hljs-comment">#!/usr/bin/env python</span>
<span class="hljs-comment"># -*- coding:utf-8 -*-</span>

<span class="hljs-keyword">from</span> scrapy <span class="hljs-keyword">import</span> cmdline
<span class="hljs-comment">#scrapy crawl itcast （house为爬虫名）</span>
<span class="hljs-comment">#此处的语句即是采用的twisted框架库</span>
cmdline.execute(<span class="hljs-string">"sscrapy crawl house"</span>.split())</code></pre>
    <p>
     2) 在 Run -&gt; Edit Configurations…工具中创建python运行设置，如下图
     <br/>
     <img alt="run" src="https://i-blog.csdnimg.cn/blog_migrate/eef1e6adf953a1f96e5c189a62e358fe.png" title=""/>
    </p>
    <p>
     创建好后，OK保存，点击运行即可
     <br/>
     <img alt="运行" src="https://i-blog.csdnimg.cn/blog_migrate/0ad305a17414aea4896f8ff0db8d8cef.png" title=""/>
    </p>
