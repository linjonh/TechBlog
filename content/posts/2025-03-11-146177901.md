    <h2>
     <a id="macOS__Oh_My_Zsh___0">
     </a>
     macOS 安装、优化、还原、升级 Oh My Zsh 完全指南 🚀
    </h2>
    <p>
     Oh My Zsh 是 macOS 终端增强的利器，它能提供强大的自动补全、主题定制和插件支持，让你的终端更高效、更炫酷。本文将全面介绍
     <strong>
      如何安装、优化、还原、重新安装和升级 Oh My Zsh
     </strong>
     ，以及如何恢复安装前的
     <code>
      .zshrc
     </code>
     配置文件。
    </p>
    <hr/>
    <h3>
     <a id="_1_macOS__Oh_My_Zsh_6">
     </a>
     🔹 1. macOS 安装 Oh My Zsh
    </h3>
    <p>
     在 macOS 上，你可以使用
     <code>
      curl
     </code>
     或
     <code>
      wget
     </code>
     来安装
     <strong>
      Oh My Zsh
     </strong>
     。
    </p>
    <h4>
     <a id="__10">
     </a>
     📥 安装步骤
    </h4>
    <h5>
     <a id="__1_curl_12">
     </a>
     ✅ 方式 1：使用
     <code>
      curl
     </code>
     （推荐）
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sh</span> <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <h5>
     <a id="__2_wget_17">
     </a>
     ✅ 方式 2：使用
     <code>
      wget
     </code>
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sh</span> <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">wget</span> -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <h5>
     <a id="__3_Homebrew_22">
     </a>
     ✅ 方式 3：使用 Homebrew（可选）
    </h5>
    <pre><code class="prism language-bash">brew <span class="token function">install</span> oh-my-zsh
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c8bf58be99554595b99a453d686e2f8f.png"/>
    </p>
    <h4>
     <a id="__29">
     </a>
     🛠 安装后需要做的事情
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        检查是否安装成功
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token environment constant">$SHELL</span>
</code></pre>
      <p>
       如果输出
       <code>
        /bin/zsh
       </code>
       ，说明安装成功。
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看
        <code>
         .zshrc
        </code>
        是否存在
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-la</span> ~/.zshrc
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        重启终端或运行
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="__47">
     </a>
     🎨 启用自动补全和语法高亮
    </h4>
    <p>
     为了提升终端使用体验，我们可以安装
     <strong>
      zsh-autosuggestions
     </strong>
     （自动补全） 和
     <strong>
      zsh-syntax-highlighting
     </strong>
     （语法高亮）。
    </p>
    <h5>
     <a id="__Homebrew__51">
     </a>
     ✅ 使用 Homebrew 安装自动补全和语法高亮
    </h5>
    <pre><code class="prism language-bash">brew <span class="token function">install</span> zsh-autosuggestions zsh-syntax-highlighting
</code></pre>
    <h5>
     <a id="__git__56">
     </a>
     ✅ 使用 git 安装自动补全和语法高亮
    </h5>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions
<span class="token function">git</span> clone https://github.com/zsh-users/zsh-syntax-highlighting ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting
</code></pre>
    <h5>
     <a id="__62">
     </a>
     ✅ 启用插件
    </h5>
    <p>
     编辑
     <code>
      ~/.zshrc
     </code>
     ，找到
     <code>
      plugins=()
     </code>
     这一行，修改为：
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">plugins</span><span class="token operator">=</span><span class="token punctuation">(</span>git zsh-autosuggestions zsh-syntax-highlighting<span class="token punctuation">)</span>
</code></pre>
    <p>
     或者在 .zshrc 末尾手动添加：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 自动补全</span>
<span class="token builtin class-name">source</span> /opt/homebrew/share/zsh-autosuggestions/zsh-autosuggestions.zsh

<span class="token comment"># 语法高亮</span>
<span class="token builtin class-name">source</span> /opt/homebrew/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh
</code></pre>
    <p>
     <strong>
      ps
     </strong>
     ：
     <code>
      Homebrew
     </code>
     安装使用
     <code>
      source
     </code>
     的方式，
     <code>
      git
     </code>
     使用
     <code>
      plugins
     </code>
     的方式。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8769e99b707d448192e8d532b07ccea6.png"/>
    </p>
    <h5>
     <a id="__zshrc_80">
     </a>
     ✅ 重新加载
     <code>
      .zshrc
     </code>
    </h5>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <p>
     启用后，你的终端将支持
     <strong>
      命令自动补全
     </strong>
     和
     <strong>
      语法高亮
     </strong>
     ，提高输入效率！🚀
    </p>
    <hr/>
    <h3>
     <a id="_2__macOS__shell_89">
     </a>
     🔄 2. 还原 macOS 默认 shell
    </h3>
    <p>
     如果你不想使用
     <strong>
      Oh My Zsh
     </strong>
     ，可以按以下步骤还原
     <strong>
      macOS 默认 shell
     </strong>
     （通常是
     <code>
      zsh
     </code>
     或
     <code>
      bash
     </code>
     ）。
    </p>
    <h4>
     <a id="__93">
     </a>
     🔄 还原方法
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        运行 Oh My Zsh 自带卸载脚本
       </strong>
      </p>
      <pre><code class="prism language-bash">uninstall_oh_my_zsh
</code></pre>
      <p>
       这个命令会：
      </p>
      <ul>
       <li>
        删除
        <code>
         ~/.oh-my-zsh/
        </code>
        目录
       </li>
       <li>
        删除
        <code>
         ~/.zshrc
        </code>
       </li>
       <li>
        还原 macOS 预设 shell
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        手动切换回默认 shell（可选）
       </strong>
      </p>
      <pre><code class="prism language-bash">chsh <span class="token parameter variable">-s</span> /bin/bash
</code></pre>
      <p>
       或者切换回 macOS 自带的 Zsh：
      </p>
      <pre><code class="prism language-bash">chsh <span class="token parameter variable">-s</span> /bin/zsh
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        删除 Oh My Zsh 相关文件（可选）
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/.oh-my-zsh
<span class="token function">rm</span> <span class="token parameter variable">-f</span> ~/.zshrc ~/.zsh_history
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        重启终端
       </strong>
       关闭并重新打开终端，确保
       <strong>
        Zsh
       </strong>
       变回 macOS 自带的配置。
      </p>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_3__Oh_My_Zsh_123">
     </a>
     ♻ 3. 重新安装 Oh My Zsh
    </h3>
    <p>
     如果你之前卸载了
     <strong>
      Oh My Zsh
     </strong>
     ，想要重新安装，可以按照
     <strong>
      安装步骤
     </strong>
     再次执行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sh</span> <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_4__Oh_My_Zsh_132">
     </a>
     🔼 4. 升级 Oh My Zsh
    </h3>
    <p>
     如果你已经安装了
     <strong>
      Oh My Zsh
     </strong>
     ，可以使用以下命令升级到最新版本：
    </p>
    <h4>
     <a id="__136">
     </a>
     🚀 自动升级
    </h4>
    <pre><code class="prism language-bash">omz update
</code></pre>
    <p>
     或者：
    </p>
    <pre><code class="prism language-bash">upgrade_oh_my_zsh
</code></pre>
    <h4>
     <a id="__145">
     </a>
     📝 手动升级
    </h4>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~/.oh-my-zsh
<span class="token function">git</span> pull origin master
</code></pre>
    <hr/>
    <h3>
     <a id="_5__zshrc_153">
     </a>
     🔍 5. 找回安装前的
     <code>
      .zshrc
     </code>
    </h3>
    <h4>
     <a id="__zshrcpreohmyzsh_155">
     </a>
     📂 如何找到
     <code>
      .zshrc.pre-oh-my-zsh
     </code>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        检查是否存在
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-la</span> ~ <span class="token operator">|</span> <span class="token function">grep</span> zshrc
</code></pre>
      <p>
       如果输出如下：
      </p>
      <pre><code class="prism language-bash">-rw-r--r--   <span class="token number">1</span> user  staff   <span class="token number">1234</span> Mar <span class="token number">10</span> <span class="token number">10</span>:00 .zshrc
-rw-r--r--   <span class="token number">1</span> user  staff   <span class="token number">1100</span> Mar 09 <span class="token number">12</span>:00 .zshrc.pre-oh-my-zsh
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        恢复
        <code>
         .zshrc
        </code>
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">mv</span> ~/.zshrc.pre-oh-my-zsh ~/.zshrc
<span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="__zshrcpreohmyzsh__173">
     </a>
     🛠 如果
     <code>
      .zshrc.pre-oh-my-zsh
     </code>
     不存在
    </h4>
    <p>
     如果你已经手动删除了
     <code>
      .zshrc.pre-oh-my-zsh
     </code>
     ，你可以手动创建一个基础的
     <code>
      ~/.zshrc
     </code>
     文件：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token string">'export PATH="/usr/local/bin:$PATH"'</span> <span class="token operator">&gt;</span> ~/.zshrc
<span class="token builtin class-name">echo</span> <span class="token string">'export LANG="en_US.UTF-8"'</span> <span class="token operator">&gt;&gt;</span> ~/.zshrc
<span class="token builtin class-name">echo</span> <span class="token string">'export ZSH_THEME="robbyrussell"'</span> <span class="token operator">&gt;&gt;</span> ~/.zshrc
<span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
