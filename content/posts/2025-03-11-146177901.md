# macOS 终端优化

## macOS 安装、优化、还原、升级 Oh My Zsh 完全指南 🚀

Oh My Zsh 是 macOS 终端增强的利器，它能提供强大的自动补全、主题定制和插件支持，让你的终端更高效、更炫酷。本文将全面介绍
**如何安装、优化、还原、重新安装和升级 Oh My Zsh**
，以及如何恢复安装前的
`.zshrc`
配置文件。

---

### 🔹 1. macOS 安装 Oh My Zsh

在 macOS 上，你可以使用
`curl`
或
`wget`
来安装
**Oh My Zsh**
。

#### 📥 安装步骤

##### ✅ 方式 1：使用 `curl` （推荐）

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

```

##### ✅ 方式 2：使用 `wget`

```bash
sh -c "$(wget -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

```

##### ✅ 方式 3：使用 Homebrew（可选）

```bash
brew install oh-my-zsh

```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/c8bf58be99554595b99a453d686e2f8f.png)

#### 🛠 安装后需要做的事情

1. **检查是否安装成功**

   ```bash
   echo $SHELL

   ```

   如果输出
   `/bin/zsh`
   ，说明安装成功。
2. **查看
   `.zshrc`
   是否存在**

   ```bash
   ls -la ~/.zshrc

   ```
3. **重启终端或运行**

   ```bash
   source ~/.zshrc

   ```

#### 🎨 启用自动补全和语法高亮

为了提升终端使用体验，我们可以安装
**zsh-autosuggestions**
（自动补全） 和
**zsh-syntax-highlighting**
（语法高亮）。

##### ✅ 使用 Homebrew 安装自动补全和语法高亮

```bash
brew install zsh-autosuggestions zsh-syntax-highlighting

```

##### ✅ 使用 git 安装自动补全和语法高亮

```bash
git clone https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting

```

##### ✅ 启用插件

编辑
`~/.zshrc`
，找到
`plugins=()`
这一行，修改为：

```bash
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)

```

或者在 .zshrc 末尾手动添加：

```bash
# 自动补全
source /opt/homebrew/share/zsh-autosuggestions/zsh-autosuggestions.zsh

# 语法高亮
source /opt/homebrew/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

```

**ps**
：
`Homebrew`
安装使用
`source`
的方式，
`git`
使用
`plugins`
的方式。

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/8769e99b707d448192e8d532b07ccea6.png)

##### ✅ 重新加载 `.zshrc`

```bash
source ~/.zshrc

```

启用后，你的终端将支持
**命令自动补全**
和
**语法高亮**
，提高输入效率！🚀

### 🔄 2. 还原 macOS 默认 shell

如果你不想使用
**Oh My Zsh**
，可以按以下步骤还原
**macOS 默认 shell**
（通常是
`zsh`
或
`bash`
）。

#### 🔄 还原方法

1. **运行 Oh My Zsh 自带卸载脚本**

   ```bash
   uninstall_oh_my_zsh

   ```

   这个命令会：

   * 删除
     `~/.oh-my-zsh/`
     目录
   * 删除
     `~/.zshrc`
   * 还原 macOS 预设 shell
2. **手动切换回默认 shell（可选）**

   ```bash
   chsh -s /bin/bash

   ```

   或者切换回 macOS 自带的 Zsh：

   ```bash
   chsh -s /bin/zsh

   ```
3. **删除 Oh My Zsh 相关文件（可选）**

   ```bash
   rm -rf ~/.oh-my-zsh
   rm -f ~/.zshrc ~/.zsh_history

   ```
4. **重启终端**
   关闭并重新打开终端，确保
   **Zsh**
   变回 macOS 自带的配置。

---

### ♻ 3. 重新安装 Oh My Zsh

如果你之前卸载了
**Oh My Zsh**
，想要重新安装，可以按照
**安装步骤**
再次执行：

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

```

---

### 🔼 4. 升级 Oh My Zsh

如果你已经安装了
**Oh My Zsh**
，可以使用以下命令升级到最新版本：

#### 🚀 自动升级

```bash
omz update

```

或者：

```bash
upgrade_oh_my_zsh

```

#### 📝 手动升级

```bash
cd ~/.oh-my-zsh
git pull origin master

```

---

### 🔍 5. 找回安装前的 `.zshrc`

#### 📂 如何找到 `.zshrc.pre-oh-my-zsh`

1. **检查是否存在**

   ```bash
   ls -la ~ | grep zshrc

   ```

   如果输出如下：

   ```bash
   -rw-r--r--   1 user  staff   1234 Mar 10 10:00 .zshrc
   -rw-r--r--   1 user  staff   1100 Mar 09 12:00 .zshrc.pre-oh-my-zsh

   ```
2. **恢复
   `.zshrc`**

   ```bash
   mv ~/.zshrc.pre-oh-my-zsh ~/.zshrc
   source ~/.zshrc

   ```

#### 🛠 如果 `.zshrc.pre-oh-my-zsh` 不存在

如果你已经手动删除了
`.zshrc.pre-oh-my-zsh`
，你可以手动创建一个基础的
`~/.zshrc`
文件：

```bash
echo 'export PATH="/usr/local/bin:$PATH"' > ~/.zshrc
echo 'export LANG="en_US.UTF-8"' >> ~/.zshrc
echo 'export ZSH_THEME="robbyrussell"' >> ~/.zshrc
source ~/.zshrc

```
