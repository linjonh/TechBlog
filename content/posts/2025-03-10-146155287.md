    <p>
     flutter的HTTP headers用法介绍
    </p>
    <p>
     在 Flutter 中，HTTP headers 是用于在发送 HTTP 请求时传递额外信息的关键部分。它们可以用于身份验证、缓存控制、内容类型声明等。以下是关于 Flutter 中 HTTP headers 的详细说明和用法。
    </p>
    <hr/>
    <h4>
     1. 什么是 HTTP Headers？
    </h4>
    <p>
     HTTP headers 是键值对（key-value pairs），用于在客户端和服务器之间传递附加信息。例如：
    </p>
    <ul>
     <li>
      <p>
       <code>
        Content-Type
       </code>
       : 指定请求或响应的数据类型（如
       <code>
        application/json
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       <code>
        Authorization
       </code>
       : 用于身份验证（如
       <code>
        Bearer &lt;token&gt;
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       <code>
        Cache-Control
       </code>
       : 控制缓存行为（如
       <code>
        no-cache
       </code>
       ）。
      </p>
     </li>
    </ul>
    <hr/>
    <h4>
     2. 在 Flutter 中使用 HTTP Headers
    </h4>
    <p>
     Flutter 提供了多种方式发送 HTTP 请求并设置 headers，以下是常见的几种方式：
    </p>
    <h5>
     2.1 使用
     <code>
      http
     </code>
     包
    </h5>
    <p>
     <code>
      http
     </code>
     是 Flutter 中常用的 HTTP 请求库。可以通过
     <code>
      headers
     </code>
     参数设置 headers。
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>import 'package:http/http.dart' as http;

Future&lt;void&gt; fetchData() async {
  final response = await http.get(
    Uri.parse('https://example.com/api/data'),
    headers: {
      'Authorization': 'Bearer your_token',
      'Content-Type': 'application/json',
      'Cache-Control': 'no-cache',
    },
  );

  if (response.statusCode == 200) {
    print('Data: ${response.body}');
  } else {
    print('Failed to load data');
  }
}</pre>
    <h5>
     2.2 使用
     <code>
      dio
     </code>
     包
    </h5>
    <p>
     <code>
      dio
     </code>
     是一个功能更强大的 HTTP 客户端库，支持拦截器、文件上传等功能。可以通过
     <code>
      Options
     </code>
     设置 headers。
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>import 'package:dio/dio.dart';

Future&lt;void&gt; fetchData() async {
  final dio = Dio();
  final response = await dio.get(
    'https://example.com/api/data',
    options: Options(
      headers: {
        'Authorization': 'Bearer your_token',
        'Content-Type': 'application/json',
        'Cache-Control': 'no-cache',
      },
    ),
  );

  if (response.statusCode == 200) {
    print('Data: ${response.data}');
  } else {
    print('Failed to load data');
  }
}</pre>
    <h5>
     2.3 使用
     <code>
      HttpClient
     </code>
    </h5>
    <p>
     Flutter 的
     <code>
      dart:io
     </code>
     库提供了
     <code>
      HttpClient
     </code>
     ，可以直接设置 headers。
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>import 'dart:io';
import 'dart:convert';

Future&lt;void&gt; fetchData() async {
  final HttpClient client = HttpClient();
  final HttpClientRequest request = await client.getUrl(Uri.parse('https://example.com/api/data'));
  request.headers.add('Authorization', 'Bearer your_token');
  request.headers.add('Content-Type', 'application/json');
  request.headers.add('Cache-Control', 'no-cache');

  final HttpClientResponse response = await request.close();
  final String body = await response.transform(utf8.decoder).join();

  if (response.statusCode == 200) {
    print('Data: $body');
  } else {
    print('Failed to load data');
  }
}</pre>
    <h5>
     2.4 在
     <code>
      NetworkImage
     </code>
     中使用 headers
    </h5>
    <p>
     <code>
      NetworkImage
     </code>
     是 Flutter 中用于加载网络图片的类，可以通过
     <code>
      headers
     </code>
     参数设置 headers。
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  headers: {
    'Authorization': 'Bearer your_token',
    'Cache-Control': 'no-cache',
  },
)</pre>
    <hr/>
    <h4>
     3. 常见的 HTTP Headers
    </h4>
    <p>
     以下是一些常见的 HTTP headers 及其用途：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        Header 名称
       </th>
       <th>
        用途
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         Authorization
        </code>
       </td>
       <td>
        用于身份验证，通常携带 token（如
        <code>
         Bearer &lt;token&gt;
        </code>
        ）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         Content-Type
        </code>
       </td>
       <td>
        指定请求或响应的数据类型（如
        <code>
         application/json
        </code>
        、
        <code>
         text/plain
        </code>
        ）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         Cache-Control
        </code>
       </td>
       <td>
        控制缓存行为（如
        <code>
         no-cache
        </code>
        、
        <code>
         max-age=3600
        </code>
        ）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         Accept
        </code>
       </td>
       <td>

        <code>
         application/json
        </code>
        ）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         User-Agent
        </code>
       </td>

       </td>
      </tr>
      <tr>
       <td>

        <code>
         en-US
        </code>
        ）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         Referer
        </code>
       </td>
       <td>
        标识请求的来源页面。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         If-Modified-Since
        </code>
       </td>
       <td>
        用于条件请求，仅在资源修改后返回。
       </td>
      </tr>
     </tbody>
    </table>
    <hr/>
    <h4>
     4. 自定义 Headers
    </h4>
    <p>
     你可以根据需求自定义 headers。例如，添加一个自定义的
     <code>
      X-Custom-Header
     </code>
     ：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>final response = await http.get(
  Uri.parse('https://example.com/api/data'),
  headers: {
    'X-Custom-Header': 'custom_value',
  },
);</pre>
    <hr/>
    <h4>
     5. 使用拦截器（Interceptors）
    </h4>
    <p>
     如果你需要为所有请求添加统一的 headers，可以使用拦截器。
     <code>
      dio
     </code>
     包支持拦截器：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>import 'package:dio/dio.dart';

void setupDio() {
  final dio = Dio();
  dio.interceptors.add(InterceptorsWrapper(
    onRequest: (options, handler) {
      options.headers['Authorization'] = 'Bearer your_token';
      options.headers['Content-Type'] = 'application/json';
      return handler.next(options);
    },
  ));

  // 使用 dio 发送请求
  dio.get('https://example.com/api/data').then((response) {
    print('Data: ${response.data}');
  });
}</pre>
    <hr/>
    <h4>
     6. 处理 Headers 的响应

    <p>
     服务器可能会在响应中返回一些 headers，你可以通过以下方式获取它们：

print('Response Headers: ${response.headers}');</pre>

print('Response Headers: ${response.headers}');</pre>

    <p>
     复制
    </p>
    <pre>final HttpClientResponse response = await request.close();
print('Response Headers: ${response.headers}');</pre>
    <hr/>
    <h4>
     7. 注意事项
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        大小写敏感
       </strong>
       ：HTTP headers 的名称是大小写不敏感的，但建议统一使用首字母大写的格式（如
       <code>
        Content-Type
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        安全性
       </strong>
       ：不要在 headers 中直接传递敏感信息（如密码），使用 HTTPS 加密传输。
      </p>
     </li>
     <li>
      <p>
       <strong>
        缓存控制
       </strong>
       ：如果需要禁用缓存，确保服务器和客户端都正确设置了
       <code>
        Cache-Control
       </code>
       头。
      </p>
     </li>
    </ul>
    <hr/>
    <h4>
     总结
    </h4>
    <p>
     在 Flutter 中，HTTP headers 是发送请求时的重要部分。你可以通过
     <code>
      http
     </code>
     、
     <code>
      dio
     </code>
     或
     <code>
      HttpClient
     </code>
     设置 headers，并根据需求自定义它们。合理使用 headers 可以实现身份验证、缓存控制、内容类型声明等功能。
    </p>
