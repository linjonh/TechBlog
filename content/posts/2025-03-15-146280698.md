# 配置blender的python环境

  1. 在blender的脚本出输入：

> > > import sys  
>  print(sys.executable)

`2. 通过上述命令我们得到blener的python版本，下面我们在conda配置一个同样版本的python环境。

    
    
    conda create -n blenderpy python=3.11.9
    

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/7b5bcc8e979a4c0d9bf39d7e9a4f470e.png)

  3. 找到blender安装路径下的python文件夹，将它删除或更换为其他文件名
  4. ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/74544274c08143ed94f3265d4d438085.png)  
4.链接自己在conda创建的python环境

    
    
      Mklink /j python D:\software\anaconda\install\envs\blenderpy
      或者
      New-Item -ItemType Junction -Path "python" -Target "D:\software\anaconda\install\envs\blenderpy"
    
    

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/f20af76eeeed46bba0c089c8efb5f81a.png)  
5\. 可以看到链接成功了  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/32124decf63848cfa5b9561d354685bf.png)  
6\. 后面就可以在创建的conda环境中使用blender了  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/4ea71345df514a31afb80aad3a730327.png)
