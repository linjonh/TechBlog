    <p>
     根据课程设计目标，本课程具有以下 3 个特点：
    </p>
    <ul>
     <li>
      <strong>
       难度低：
      </strong>
      只保留核心技术点，并对每一步实现做深入讲解，舍弃复杂难懂的部分；
     </li>
     <li>
      <strong>
       上手快：
      </strong>
      只介绍最核心内容，帮助你快速掌握要点；
     </li>
     <li>
      <strong>
       功能相对全面：
      </strong>
      在降低难度的同时，仍涵盖 Go 项目的核心功能。
     </li>
    </ul>

    <h3>
     <a id="_19">
     </a>
     本课程写作背景
    </h3>
    <p>
     在学习并使用一门新的语言时，很多开发者都期望能够有一套很入门的课程，来快速学习、快速入门。另外，我自己最近也在学习 AI，因为平时工作比较忙，所以学习过程中也很期待能有一个 AI 实战 Hello World 课程，能够以很低的学习难度，快速带我入门 AI。
    </p>
    <p>
     考虑到开发者的真实诉求，以及自己的学习体验，我决定出一个极速入门课程。该课程类似于一个 Go 项目开发 Hello World，能够让你在极短的时间内入门 Go 项目开发。并由此在未来展开更多的深度学习。
    </p>
    <h3>
     <a id="_23">
     </a>
     课程目标
    </h3>
    <p>
     基于课程的写作目的，本课程的目标如下：
     <strong>
      以最小的学习难度，让初学者快速了解如何开发一个功能相对完备的高质量 Go 项目。
     </strong>
    </p>
    <p>
     如果你想进阶学习 Go 项目开发技术，可以学习本课程的进阶课程：
    </p>
    <ul>
     <li>
      <strong>
       Go 项目开发理论课（22 节）：
      </strong>
      <a href="https://konglingfei.com/cloudai/catalog/methodology.html" rel="nofollow">
       Go 项目开发方法论课
      </a>
      可以让你掌握开发一个优秀 Go 项目的方法；
     </li>
     <li>
      <strong>
       中级工程师进阶课（40 节课）：
      </strong>
      <a href="https://konglingfei.com/cloudai/catalog/intermediate.html" rel="nofollow">
       Go 项目开发中级实战课
      </a>
      。本套课程的进阶课程，包含了更多的技术点，例如：gRPC、gRPC-Gateway、授权、性能测试、性能分析、静态代码检查、Makefile、OpenAPI、自定义日质包、自定义错误包，更强大的校验机制等。可以，让你全方位的掌握如何开发一个优秀的 Go 项目。本套课程包含了一个拥有 16w 行代码的高质量 Go 项目：
      <a href="https://github.com/onexstack/miniblog">
       miniblog
      </a>
      ；
     </li>
     <li>
      <strong>
       专家级工程师进阶课（100 节课）：
      </strong>
      <a href="https://konglingfei.com/cloudai/catalog/expert.html" rel="nofollow">
       Go 项目开发专家级实战课
      </a>
      。专家级实战课可以直接让你进阶为 Go 开发专家。本套课程包含了一个拥有 20w 行代码的高质量 Go 项目：
      <a href="https://github.com/onexstack/onex">
       onex
      </a>
      。
     </li>
    </ul>
    <p>
     上述 3 门课程会使你直接进阶为 Go 项目开发专家。另外，本课程（4.5 万字）是
     <a href="https://konglingfei.com/cloudai/catalog/intermediate.html" rel="nofollow">
      Go 项目开发中级实战课
     </a>
     （22 万字，是极速入门课内容量的 5 倍） 的精简版，学完之后，你可以无缝切入
     <a href="https://konglingfei.com/cloudai/catalog/intermediate.html" rel="nofollow">
      Go 项目开发中级实战课
     </a>
     课程的学习。上述课程均采用了相同的开发风格、开发规范及软件架构等。并且有配套的项目开发脚手架：
     <a href="https://github.com/onexstack/osbuilder.git">
      osbuilder
     </a>
     。
    </p>
    <br/>
    <p>
     <strong>
      因为本课程目标是极速入门课，所以不会介绍太多、太详细的 Go 开发技术。
     </strong>
     如果想学习更多的 Go 开发技术，可以加入我的云原生 AI 实战营星球进行学习，12+ 高质量体系课、20+ 高质量实战项目助你在 AI 时代建立技术竞争力。
    </p>
    <h3>
     <a id="_37">
     </a>
     课程特点
    </h3>
    <p>
     因为本课程的目标是能够让你在极端的时间内，相对全面的学习 Go 项目开发，所以本课程具有以下特点：
    </p>
    <ul>
     <li>
      <strong>
       学习时间短：
      </strong>
      课程能够让你在很短时间内快速入门 Go 项目开发，预估时间为 2 个小时；
     </li>
     <li>
      <strong>
       内容全面：
      </strong>
      虽然是极速入门课课，但课程的内容仍然相对全面，包含了 Go 项目开发中的重要开发内容；
     </li>
     <li>
      <strong>
       内容基础：
      </strong>
      因为课程内容相对全面，要压缩学习时间，所以，本课程，不会讲太多、太复杂的开发知识，只会讲开发过程中比较基础的知识；
     </li>
     <li>
      <strong>
       符合最佳实践：
      </strong>
      虽然课程内容基础，但仍然符合最佳实践。
     </li>
    </ul>
