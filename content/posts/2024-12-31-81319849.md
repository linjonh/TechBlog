    <p>
     **
     <font size="6">
      Vim查找命令及快捷键
     </font>
     <br/>
     **
    </p>
    <p>
     <strong>
      一、通过指令查找
      <br/>
     </strong>
     <br/>
     <strong>
      1、左斜杠/ 　从光标所在位置向文件尾搜索
      <br/>
     </strong>
     <br/>
     <strong>
      例
     </strong>
     ：／hello
     <br/>
     <br/>
     <strong>
      2、问号？　从光标所在位置向文件头搜索
      <br/>
     </strong>
     <br/>
     <strong>
      例
     </strong>
     ：？hello
     <br/>
     <br/>
     <strong>
      注：／和？均可加参数做指定查找
     </strong>
     （两者搜索方向不同，此处以／举例进行说明）
     <br/>
     <br/>
     <strong>
      １）　&lt;匹配单词的开头　\&gt;匹配单词的结尾
      <br/>
     </strong>
     <br/>
     <strong>
      例
     </strong>
     :　&lt; hel 可以找到所有以hel开头的单词 （此处为了编辑方便我在&lt;和hel中加了空格，实际不应有此空格）
     <br/>
     \&gt;llo 可以找到所有以llo结尾的单词
     <br/>
     &lt; for&gt;可以找到所有的for单词，forever中的for不会被找到 （此处为了编辑方便我在&lt;和for中加了空格，实际
     <br/>
     不应有此空格）
    </p>
    <p>
     <strong>
      ２）　＾代表行首　＄代表行尾
      <br/>
     </strong>
     <br/>
     <strong>
      例
     </strong>
     :　＾hello 只会匹配行首的hello
     <br/>
     $hello 只会匹配行尾的hello
     <br/>
     <br/>
     <strong>
      另：按"n"或者“N”可对找到的词进行后一个或者前一个的跳转
     </strong>
    </p>
    <p>
     <strong>
      二、通过快捷键查找
      <br/>
     </strong>
     <br/>
     1、#　开始向文件头的方向搜索光标所在位置的单词的下一个出现位置
     <br/>
     2、*　开始向文件尾的方向搜索光标所在位置的单词的下一个出现位置
    </p>
    <p>
     <strong>
      另：被查找到的单词会高亮显示，如想要去除该高亮显示，可使用命令 ：nohl （即no high light的意思）
     </strong>
    </p>
