# Python || 青蛙跳台阶

一只青蛙一次可以跳上1级台阶，也可以跳上2级。请问该青蛙跳上一个n级的台阶总共有多少种跳法。

输入台阶数，输出一共有多少种跳法。

**输入：**

3

**输出：**

3

```python
#一只青蛙一次可以跳上1级台阶，也可以跳上2级。请问该青蛙跳上一个n级的台阶总共有多少种跳法。
#输入台阶数，输出一共有多少种跳法
#coding:utf-8
def jump1(n):
    if n==1:
        return 1
    elif n==2:
        return 2
    else:
        return jump1(n-1)+jump1(n-2)
x=eval(input())
print(jump1(x))




```
