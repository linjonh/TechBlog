     使用Shotcut为视频添加马赛克效果

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <svg style="display: none;" xmlns="http://www.w3.org/2000/svg">
     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     在
     <strong>
      Shotcut
     </strong>
     中添加马赛克效果可以通过视频特效和遮罩工具实现。以下是详细步骤：
    </p>
    <hr/>
    <h4>
     <a id="_1_4">
     </a>
     <strong>
      步骤 1：导入视频到时间线
     </strong>
    </h4>
    <ol>
     <li>
      打开
      <strong>
       Shotcut
      </strong>
      ，点击顶部菜单栏的
      <strong>
       文件 &gt; 导入文件
      </strong>
      ，选择要编辑的视频文件。
     </li>
     <li>
      将视频拖到下方的
      <strong>
       时间线
      </strong>
      中的视频轨道上。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_2_10">
     </a>
     <strong>
      步骤 2：选择要添加马赛克的剪辑
     </strong>
    </h4>
    <ul>
     <li>
      在时间线中，单击选择要添加马赛克的视频剪辑，或按住鼠标左键拖拽选中片段范围。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_3_15">
     </a>
     <strong>
      步骤 3：添加马赛克效果
     </strong>
    </h4>
    <ol>
     <li>
      点击左侧面板的
      <strong>
       效果
      </strong>
      （Effect）标签。
     </li>
     <li>
      展开
      <strong>
       视频特效
      </strong>
      （Video Effects）目录：
      <ul>
       <li>
        在
        <strong>
         Pixelation
        </strong>
        或
        <strong>
         Pixelate
        </strong>
        类别中找到马赛克效果（例如：
        <strong>
         Pixelate Color Blocks
        </strong>
        、
        <strong>
         Pixelate Grid
        </strong>
        等）。
       </li>
      </ul>
     </li>
     <li>
      将选中的马赛克效果拖拽到时间线的视频剪辑上（可直接覆盖剪辑或添加到效果堆栈）。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_4_23">
     </a>
     <strong>
      步骤 4：调整马赛克参数
     </strong>
    </h4>
    <ol>
     <li>
      在主界面中，右侧会自动显示
      <strong>
       效果控制面板
      </strong>
      （如果没有，点击菜单栏的
      <strong>
       显示 &gt; 效果控制
      </strong>
      ）。
     </li>
     <li>
      在效果控制面板中找到刚添加的马赛克效果（如
      <strong>
       Pixelate Color Blocks
      </strong>
      ）。
     </li>
     <li>
      <strong>
       调整参数
      </strong>
      （根据效果名称可能略有不同）：
      <ul>
       <li>
        <strong>
         块大小（Block Size）
        </strong>
        ：数值越大，马赛克颗粒越粗；反之则越细。
       </li>
       <li>
        <strong>
         辣块偏移（Offset）
        </strong>
        ：调整马赛克块的位置（可选，用于动态效果）。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_5_32">
     </a>
     <strong>
      步骤 5：添加遮罩限定马赛克区域
     </strong>
    </h4>
    <p>
     如果马赛克只应用于视频的特定区域（如人脸），需使用遮罩工具：
    </p>
    <ol>
     <li>
      点击菜单栏的
      <strong>
       滤镜
      </strong>
      &gt;
      <strong>
       遮罩工具
      </strong>
      ，或直接使用快捷键
      <strong>
       M
      </strong>
      。
     </li>
     <li>
      选择遮罩类型（矩形、椭圆或手绘）：
      <ul>
       <li>
        <strong>
         矩形/椭圆遮罩
        </strong>
        ：适用于规则区域（如人脸）。
       </li>
       <li>
        <strong>
         自由手绘
        </strong>
        ：适用于复杂形状。
       </li>
      </ul>
     </li>
     <li>
      在预览窗口中绘制遮罩区域：
      <ul>
       <li>
        绘制完成后，遮罩会自动应用到马赛克效果。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       调整遮罩属性
      </strong>
      （通过效果控制面板）：
      <ul>
       <li>
        <strong>
         羽化（Feather）
        </strong>
        ：让遮罩边缘更柔和。
       </li>
       <li>
        <strong>
         反转（Invert）
        </strong>
        ：选择遮罩内外区域的显示。
       </li>
       <li>
        <strong>
         定位
        </strong>
        ：拖动遮罩框或微调坐标参数。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_6_47">
     </a>
     <strong>
      步骤 6：预览与调整
     </strong>
    </h4>
    <ul>
     <li>
      点击播放按钮测试效果。
     </li>
     <li>
      若遮罩偏移，可通过
      <strong>
       效果控制面板
      </strong>
      的位置参数或直接在预览窗口拖动遮罩框调整。
     </li>
     <li>
      调整马赛克块大小或遮罩羽化值，直到效果满意。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_7_54">
     </a>
     <strong>
      步骤 7：导出视频
     </strong>
    </h4>
    <ol>
     <li>
      点击顶部菜单栏的
      <strong>
       文件 &gt; 导出 &gt; 为媒体
      </strong>
      。
     </li>
     <li>
      设置输出格式、编码参数（如 H.264 MP4）。
     </li>
     <li>
      点击
      <strong>
       开始
      </strong>
      开始渲染导出。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_61">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       遮罩跟随移动对象
      </strong>
      ：
      <ul>
       <li>
        如果被马赛克的物体在移动，需要手动调整遮罩的位置关键帧（点击效果控制面板中的
        <strong>
         添加关键帧
        </strong>
        按钮，在不同时间点设置遮罩位置）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       实时预览卡顿
      </strong>
      ：
      <ul>
       <li>
        可暂时关闭遮罩或降低预览分辨率，渲染时再启用完整效果。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       效果叠加顺序
      </strong>
      ：
      <ul>
       <li>
        马赛克效果的位置会影响其他特效，确保其在效果堆栈的正确位置。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_71">
     </a>
     <strong>
      替代方法（高级）
     </strong>
    </h4>
    <ul>
     <li>
      如果需要动态马赛克效果（如渐变出现），可以在效果控制面板中对
      <strong>
       块大小
      </strong>
      参数添加
      <strong>
       关键帧动画
      </strong>
      。
     </li>
    </ul>
    <hr/>
    <p>
     通过以上步骤，您可以在 Shotcut 中成功为视频添加马赛克效果。若过程中遇到问题，可参考
     <a href="https://www.shotcut.org/" rel="nofollow">
      Shotcut 官方文档
     </a>
     或社区论坛。
    </p>
