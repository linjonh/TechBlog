    <h3>
     <a id="DApp_10">
     </a>
     DApp架构
    </h3>
    <p>
     <code>
      DApp
     </code>
     ，即
     <code>
      Decentralized Application
     </code>
     ，去
     <strong>
      中心化应用
     </strong>
     。
    </p>
    <p>
     我们知道，一般的，一个完整的项目包括：
     <strong>
      展示操作层
     </strong>
     、
     <strong>
      逻辑层
     </strong>
     、
     <strong>
      存储层
     </strong>
     。区块链是可以作为
     <strong>
      逻辑层
     </strong>
     （通过
     <code>
      智能合约
     </code>
     实现逻辑处理）、
     <strong>
      存储层
     </strong>
     （通过
     <code>
      区块链结构
     </code>
     存储数据）的实现的。简单的，我们可以这么说：如果应用中使用到了
     <code>
      区块链
     </code>
     等去中心化技术，那么我们就可以认为这个应用是
     <code>
      DApp
     </code>
     应用，不同的只是说这个去中心化技术在应用中的应用占比不同而已。
    </p>
    <ul>
     <li>
      <p>
       纯粹的
       <code>
        DApp
       </code>
       应用，
       <strong>
        逻辑层
       </strong>
       、
       <strong>
        存储层
       </strong>
       都是采用去中心化技术实现的，架构如图：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/701a12709e0ca5e85b12127ed0f65e5c.png"/>
      </p>
     </li>
     <li>
      <p>
       考虑到区块链存储成本太大，在实际项目中，我们只把关键的、需要去中心化的数据存储到区块链上，大部分不必要去中心化的数据还是按照传统的方式存入数据库中的，架构如图：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c1301b48ad478d65148c768a1f03f9cb.png"/>
      </p>
     </li>
    </ul>
    <h3>
     <a id="_22">
     </a>
     公链与联盟链
    </h3>
    <p>
     区块链发展到现在，主要分为两个方向：
     <code>
      公链
     </code>
     和
     <code>
      联盟链
     </code>
     。公链和联盟链都是区块链，都具备区块链的基础特征，他们的不同之处在于：
    </p>
    <ul>
     <li>
      公链
      <ul>
       <li>
        开放式的，不限制节点的加入。
       </li>
       <li>
        公链需要有对应的激励机制，对矿工节点进行激励，以确保区块链被持续维护。
       </li>
       <li>
        公链的节点往往比较多。
       </li>
      </ul>
     </li>
     <li>
      联盟链
      <ul>
       <li>
        非开放式的，属于团体组件的网络；对于新节点的加入有准入机制。
       </li>
       <li>
        不需要激励机制，由组建联盟链的团体进行维护。
       </li>
       <li>
        联盟链的节点往往没有公链的节点多。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     截至目前（2021-08-11），常见的公链技术（应用）有：比特币、
     <code>
      以太坊
     </code>
     、
     <code>
      EOS
     </code>
     等；常见的联盟链技术（应用）有：
     <code>
      Hyperledger Fabric
     </code>
     、
     <code>
      Quorum
     </code>
     、各
     <code>
      BaaS(Blockchain as a Service)平台
     </code>
     等。
    </p>
