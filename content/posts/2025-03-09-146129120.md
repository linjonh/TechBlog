## 二、创建表t\_hero

表属性包括（id，name，nickname，age，gender，address，weapon，types）

```sql
mysql> create table t_hero(
    ->  id int,
    -> name varchar(100),
    -> age int,
    -> gender varchar(2),
    -> address varchar(100),
    -> weapon varchar(100),
    -> types varchar(100));
Query OK, 0 rows affected (0.06 sec)
 
mysql> desc t_hero;
+----------+--------------+------+-----+---------+-------+
| Field    | Type         | Null | Key | Default | Extra |
```
