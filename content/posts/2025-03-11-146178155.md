    <h3 id="开始">
     开始
    </h3>
    <h4 id="1安装-iterm2">
     1.安装 iTerm2
    </h4>
    <p>
     仅有 macOS 版版本的（当然了，人家都说是替换 macOS 原生终端的了，嘿嘿）
    </p>
    <p>
     地址：
     <a href="https://iterm2.com/" rel="nofollow" title="iTerm2 - macOS Terminal Replacement">
      iTerm2 - macOS Terminal Replacement
     </a>
    </p>
    <p>
     下载完成后，把程序拖入“访达”的“应用程序“目录里就安装完成了
    </p>
    <h4 id="2安装-oh-my-zsh">
     2.安装 oh-my-zsh
    </h4>
    <p>
     因为 macOS 已经使用 zsh 为默认的命令控制台了，所以可以装个这玩意儿
    </p>
    <p>
     地址：
     <a href="https://ohmyz.sh/" rel="nofollow" title="Oh My Zsh - a delightful &amp; open source framework for Zsh">
      Oh My Zsh - a delightful &amp; open source framework for Zsh
     </a>
    </p>
    <p>
     是使用命令安装，网页里有说明，注意一定要可以连接 github
    </p>
    <p>
     对于像我这种无法连接 github 的可怜人，使用了 ”watt工具箱“ 这个工具才连上，虽然网速可怜，但总比没有强，这个工具的地址：
     <a href="https://steampp.net/" rel="nofollow" title="Watt Toolkit">
      Watt Toolkit
     </a>
    </p>
    <p>
     使用时打开程序，将 “网络加速” 中加速服务里 github 给勾选后再开始加速就可以了（详细操作可以看提示）
    </p>
    <h4 id="3设置-iterm2">
     3.设置 iTerm2
    </h4>
    <p>
     从 iTerm2 里找了一个自带的主题，个人比较喜欢，要是有其他喜欢的也可以安装，安装方法自己百度，也是很简单
    </p>
    <p>
     打开 iTerm2 -&gt; Settings ... (Command + ,) 进入设置
    </p>
    <ol>
     <li>
      <p>
       设置 Appearance -&gt; General -&gt; Theme 修改为 Minimal （这样标题跟窗口会比较融合）
      </p>
     </li>
     <li>
      <p>
       设置 Profiles （修改个人的配置，也可以另外加新的，不过本人就修改这个默认的就行了，具体数值也可以看个人喜好自行修改） Text -&gt; Font 文字大小 15 Window -&gt; Transparency 透明度 20，Blur 毛玻璃效果勾选并设置 20，Columns 128，Rows 36
      </p>
     </li>
    </ol>
    <h4 id="4设置-oh-my-zsh-主题">
     4.设置 oh-my-zsh 主题
    </h4>
    <p>
     在命令里进行修改配置文件（在 iTerm2 里执行命令就行，当然在终端里也一样，就是要修改 zsh 的配置）
    </p>
    <pre><code>vim .zshrc
# 修改 ZSH_THEME 并保存
ZSH_THEME="jonathan"
# 也可以修改为其他主题，主题截图在官网上有，自带主题都在 ~/.oh-my-zsh/themes 中</code></pre>
    <p>
     这样设置也算是挺好看的，以后看看再修改修改看看别的样式。
    </p>
    <h4 id="5配置-iterm2-连接-ssh">
     5.配置 iTerm2 连接 SSH
    </h4>
    <p>
     打开 iTerm2 -&gt; Settings ... (Command + ,) 进入设置
    </p>
    <p>
     注：添加新的 Profiles ，可以复制已经设置完成样式的 Default ，这样省事儿
    </p>
    <ol>
     <li>
      <p>
       先选中 Profiles 中的 Default
      </p>
     </li>
     <li>
      <p>
       点下方”Other Actions ...“中的 Duplicate Profiles，将原配置文件复制一份
      </p>
     </li>
     <li>
      <p>
       选中复制的新配置文件，在右边修改 Name 比如”远程 SSH“
      </p>
     </li>
     <li>
      <p>
       往下有一个 Command 中原 Login Shell 修改为 Command ，并在后面的文本框中输入 ssh 用户名@服务器IP -p 端口 -p 端口 这个可以不输入，不输入时默认是 22
      </p>
     </li>
    </ol>
    <p>
     <strong>
      设置自动输入密码登录，要不然每次都输入密码太麻烦
     </strong>
    </p>
    <ol>
     <li>
      <p>
       接上面步骤，选 Advanced 设置项
      </p>
     </li>
     <li>
      <p>
       点击 Triggers 中的 Edit 按钮
      </p>
     </li>
     <li>
      <p>
       在弹出的窗口中点击左下角的 +（加号）添加一个要发送的字符串
      </p>
     </li>
     <li>
      <p>
       下面就是要填写的参数
       <u>
        <strong>
         <em>
          Regular Expression
         </em>
        </strong>
       </u>
       输入一个值，说是应该是一个正则表达试，但没看明白怎么设置，但也不用太纠结，因为要输入密码，所以这里输入 password 即可
       <u>
        <strong>
         <em>
          Action
         </em>
        </strong>
       </u>
       选择 Send Text ...，因为本来也是为了发送密码字符串
       <u>
        <strong>
         <em>
          Parameters
         </em>
        </strong>
       </u>
       输入登录时要输入的密码
       <u>
        <strong>
         <em>
          Instant 与 Enabled
         </em>
        </strong>
       </u>
       都要勾选
      </p>
     </li>
    </ol>
    <p>
     要连接 SSH 时，可以从菜单 Profiles 中选择新建立完成的配置就可以打开了
    </p>
    <p>
     也可以快捷键 Shift + Command + o 然后输入部分新配置的名称就可以找到并打开SSH 连接。
    </p>
    <h4 id="6配置-iterm2-支持-lrzsz">
     6.配置 iTerm2 支持 lrzsz
    </h4>
    <p>
     这个设置还是有点麻烦的，要先安装 homebrew，可以参照下面的文档中的说明安装
     <a href="https://blog.csdn.net/unmobile/article/details/133916617?spm=1001.2014.3001.5501" title="开源免费 macOS 读写 NTFS 格式磁盘方法_mac ntfs读写工具免费-CSDN博客">
      开源免费 macOS 读写 NTFS 格式磁盘方法_mac ntfs读写工具免费-CSDN博客
     </a>
    </p>
    <p>
     （1）开始正式安装
    </p>
    <pre><code>brew install lrzsz  # 这个是为了在本地 macOS 上安装 rz / sz 命令，必要的</code></pre>
    <p>
     （2）下载并安装脚本，安装方法其实 github 都写的很清楚，照着做就好了
    </p>
    <ul>
     <li>
      <p>
       脚本下载位置：
       <a href="https://github.com/kuoruan/iterm2-zmodem" title="GitHub - kuoruan/iterm2-zmodem: Zmodem(lrzsz) for iTerm2">
        GitHub - kuoruan/iterm2-zmodem: Zmodem(lrzsz) for iTerm2
       </a>
      </p>
     </li>
     <li>
      <p>
       要下载其中的 iterm2-zmodem.sh 文件，下载地址如下，怎么下载都行能下载下来就行
       <a href="https://raw.githubusercontent.com/kuoruan/iterm2-zmodem/refs/heads/master/iterm2-zmodem.sh" rel="nofollow" title="https://raw.githubusercontent.com/kuoruan/iterm2-zmodem/refs/heads/master/iterm2-zmodem.sh">
        https://raw.githubusercontent.com/kuoruan/iterm2-zmodem/refs/heads/master/iterm2-zmodem.sh
       </a>
      </p>
     </li>
     <li>
      <p>
       将下载的 iterm2-zmodem.sh 改名为 iterm2-zmodem（mv iterm2-zmodem.sh iterm2-zmodem） 然后将文件放到 /usr/local/bin ，要是没有这个目录就新建立一个（sudo mkdir /usr/local/bin，有就不用管了，也要确保 PATH 中有这个目录，可以使用 export | grep PATH，没有就定义加一个。
      </p>
     </li>
     <li>
      <p>
       修改文件的属性 sudo chmod +x /usr/local/bin/item2-zmodem
      </p>
     </li>
    </ul>
    <p>
     （3）配置 iTerm2 中的触发器 打开 iTerm2 -&gt; Settings ... (Command + ,) -&gt; Profiles -&gt; Advanced -&gt; Triggers -&gt; Edit Regular expression: rz waiting to receive.**B0100 Action: Run Silent Coprocess Parameters: /usr/local/bin/iterm2-zmodem send Instant: 选中
    </p>
    <p>
     Regular expression: **B00000000000000 Action: Run Silent Coprocess Parameters: /usr/local/bin/iterm2-zmodem recv Instant: 选中
    </p>
    <p>
     （4）都关闭后就可以使用了
    </p>
    <h4 id="7添加自动补全">
     7.添加自动补全
    </h4>
    <p>
     同步插件
    </p>
    <pre><code>git clone https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions</code></pre>
    <p>
     修改 .zshrc 配置 vim .zshrc
    </p>
    <blockquote>
     <p>
      查找plugins=(git) 修改成 plugins=( git zsh-autosuggestions )
     </p>
    </blockquote>
    <p>
     重新载入配置文件即可 source .zshrc
    </p>
    <h4 id="8添加关键字高亮">
     8.添加关键字高亮
    </h4>
    <p>
     同步插件
    </p>
    <pre><code>git clone https://github.com/zsh-users/zsh-syntax-highlighting ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting</code></pre>
    <p>
     修改 .zshrc 配置
    </p>
    <blockquote>
     <p>
      查找plugins= 在段的下面新添加 plugins=( git zsh-syntax-highlighting )
     </p>
    </blockquote>
    <p>
     重新载入配置文件即可 source .zshrc
    </p>
