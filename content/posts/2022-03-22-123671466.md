    <h2>
     <a id="0__word__2">
     </a>
     0. 调出 word 自带的公式编辑器
    </h2>
    <p>
     插入公式：
     <code>
      alt
     </code>
     +
     <code>
      =
     </code>
     <br/>
     <strong>
      以下各公式均在此步骤之后键入
     </strong>
    </p>
    <h2>
     <a id="1__6">
     </a>
     1. 键入希腊字母
    </h2>
    <p>
     键入希腊字母：
     <strong>
      \+(字母英文读音)
     </strong>
     <br/>
     e.g. \alpha
     <br/>
     \Delta
     <br/>
     注：首字母为大写则键入结果为大写希腊字母，首字母为小写同理。
     <br/>
     附：希腊字母读音表
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2a7a253ec0cff0a3435fa1dc6c7ba91e.png"/>
    </p>
    <h2>
     <a id="2__13">
     </a>
     2. 键入上标下标
    </h2>
    <p>
     上标：
     <code>
      ^
     </code>
     <br/>
     e.g. x^3
     <br/>
     下标：
     <code>
      _
     </code>
     <br/>
     e.g. a_2
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1cf5019fedf36915dcd2b59a0100b463.png">
      <br/>
      注：上下标可联用，顺序无所谓x^2_3=x_3^2
     </img>
    </p>
    <h2>
     <a id="3__20">
     </a>
     3. 键入分数
    </h2>
    <p>
     键入分数：
     <code>
      /
     </code>
     <br/>
     e.g. a/b
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/af675b8d42482f19a9f980a16c905b24.png"/>
    </p>
    <h2>
     <a id="4__25">
     </a>
     4. 键入积分
    </h2>
    <p>
     键入积分：
     <code>
      \int
     </code>
     <br/>
     e.g.\int x
     <br/>
     \int_0^1 x
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2a60624b6b869793a8151b088980003d.png"/>
    </p>
    <h2>
     <a id="5__30">
     </a>
     5. 键入根式
    </h2>
    <p>
     键入根式：
     <code>
      \sqrt
     </code>
     <br/>
     e.g. \sqrt(2) \sqrt(5&amp;2)
    </p>
    <h2>
     <a id="6__34">
     </a>
     6. 键入矩阵
    </h2>
    <p>
     键入矩阵：
     <code>
      \matrix(a&amp;b@c&amp;d)
     </code>
     <br/>
     注：输入完成后要按一下空格，使得括号与矩阵的长度相等。
     <br/>
     e.g.
     <code>
      \matrix(a&amp;b@c&amp;d)
     </code>
     <br/>
     <code>
      (\matrix(a&amp;b@c&amp;d))
     </code>
     <br/>
     <code>
      [\matrix(a&amp;b@c&amp;d)]
     </code>
     <br/>
     <code>
      |\matrix(a&amp;b@c&amp;d)
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2b372dc8581565bbb1928f3bc86e031f.png"/>
    </p>
    <h2>
     <a id="7__43">
     </a>
     7. 键入累加和
    </h2>
    <p>
     键入累加和：
     <code>
      \sum
     </code>
     <br/>
     e.g.\sum (i=1)^n x_i
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/728c1ea7d3f50722b3eb6091e7e89c8b.png"/>
    </p>
    <h2>
     <a id="8__48">
     </a>
     8. 键入方程组
    </h2>
    <p>
     e.g.
     <code>
      \eqarray(a+b+c&amp;=3@b+c&amp;=1@c&amp;=2)
     </code>
     <strong>
      等号对齐
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a194e8f77c8ca2b6daed878035b6237.png">
      <br/>
      e.g. \eqarray(&amp;a+b+c=3@&amp;b+c=1@&amp;c=2)
      <strong>
       头对齐
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a85493085976c184cbb50530ce7f3e40.png">
       <br/>
       e.g.
       <code>
        {\eqarray(a+b &amp;=1@b &amp;=2)\close
       </code>
       <strong>
        带大括号的方程组
       </strong>
       <br/>
       (注：键入后，仍需在结尾加空格键，使得大括号长度与方程式一致。)
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5d2361fb219204eb6549bbac20cefff5.png"/>
      </img>
     </img>
    </p>
