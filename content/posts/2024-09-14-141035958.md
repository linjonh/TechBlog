### 一.安装软件

#### 1.安装 1

```shell
yum -y install zsh

```

#### 2.安装 2

```shell
#下载oh-my-zsh
git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

#移动文件
cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

```

#### 3.快捷命令

```shell
#编辑
vim ~/.zshrc

#生效
source ~/.zshrc

alias cdtmp='cd `mktemp -d /tmp/sorrycc-XXXXXX`'
alias ip="ipconfig getifaddr en0 && ipconfig getifaddr en1"
alias hosts="vi /etc/hosts"

function mcd {
  mkdir $1 && cd $1;
}

```

#### 2.修改 zsh 主题

`编辑文件:`

```shell
sudo vim ~/.zshrc

```

将 ZSH\_THEME=“” 修改为 random 可以每次开启终端都随机一个，就比较变态，完美解决选择困难症患者

```sql
#好看的主题
[oh-my-zsh] Random theme 'sonicradish'
robbyrussell 默认
gnzh 好用
sonicradish
intheloop
gallois   待选
fox
itchy

```

`生效`

```shell
source ~/.zshrc

```

### 三.powerlevel10k

#### 1.安装主题

```shell
git clone https://github.com/romkatv/powerlevel10k.git ~/.oh-my-zsh/themes/powerlevel10k

```

#### 2.查看主题

```shell
ls ~/.oh-my-zsh/themes

```

#### 3.设置主题

```shell
# 配置文件
sudo vim ~/.zshrc

# 主题设置
ZSH_THEME="powerlevel10k/powerlevel10k".

# 生效
source ~/.zshrc

```

#### 4.配置 p10k

根据提示下载字体,按照自己喜欢的风格配置就好

```shell
p10k configure

```

### 四.配置插件

#### 1.查看插件

```apl
ls ~/.oh-my-zsh/plugins

```

#### 2.配置插件

```apl
# 配置文件
sudo vim ~/.zshrc

# 生效
source ~/.zshrc

```

#### 3.安装的插件

```apl
plugins=(
git
pip
sudo
thefuck
autojump
last-working-dir
zsh-autosuggestions
zsh-syntax-highlighting
)

```

#### 4.安装插件

```apl
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git  ~/.oh-my-zsh/plugins/zsh-syntax-highlighting

```
