     iPad 上最强 IDE，支持 20 多种开发语言，还开源了

     <p>
      作者丨小集
     </p>
     <p>
      来源丨小集（ID：zsxjtip）
     </p>
     <p>
      https://mp.weixin.qq.com/s/36KacwiKsXYNY_H1foxhQA
     </p>
     <p>
      如今 iPad 的功能越来越强劲，最新 iPad Pro 还上了 M1 芯片，内存最高升到 16GB，硬盘加到了 2TB。强大的性能让 iPad 在越来越多的场景下，成为生产力工具，如设计、协同办公、教学领域等等。不过 iPad 对编程的支持，却始终差强人意。之前一直盛传的 iPad 版 Xcode 也还没个影，而 Swift Playground 也只是个小工具而已。不过，我们这边给大家安利一款开源的 iPad 应用，可以称得上是目前 iPad 上最强的 IDE：
      <strong>
       「Code App」
      </strong>
      。
     </p>
     <p>
      Code App 是 MIT 许可的桌面级代码编辑器，适用于 iPadOS，内置了 Node.js、Python、C、C++、PHP 运行时。目前这款应用已经在 App Store 上线，应用介绍里面列出了其强大的能力，一起来看看。
     </p>
     <p style="text-align:center;">
      <img alt="515db1376145679884c79685f7347dc6.png" src="https://i-blog.csdnimg.cn/blog_migrate/f7f93f44b6be831b7e8d9f32148a90c2.png"/>
     </p>
     <h4>
      特性
     </h4>
     <p>
      Code App 研发团队为这款应用添加了 App Store 同类应用所没有的强大功能，包括以下：
     </p>
     <ul>
      <li>
       <p style="text-align:left;">
        强大的高性能文本编辑器（来自 Visual Studio Code 的 Monaco 编辑器）
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        一流的本地文件系统支持
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        嵌入式仿真终端
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        本地后端开发环境（Node 和 PHP）
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        本地 Python 运行时
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        本地 Clang 编译器
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        Git 版本控制
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        包管理器支持（Pip 和 NPM）
       </p>
      </li>
      <li>
       <p style="text-align:left;">
        远程连接支持（文件和终端）
       </p>
      </li>
     </ul>
     <p>
      不过开发团队也称，虽然他们希望能让 Code App 的编辑体验与桌面提供的一样接近，但 Code App 仍然受到 iOS 的限制。例如，我们不能下载带有本机组件的任意命令或模块，也无法创建子进程。
     </p>
     <h4>
      支持的语言
     </h4>
     <p>
      Code App 支持多种开发语言，包括当前市面上最流程的那些语言，如 Python、Java、Go、Swift、Kotin 等等。以下是其所支持的语言清单：
     </p>
     <p>
      <img alt="592c8910cdfb5b236667dccee6fbb172.png" src="https://i-blog.csdnimg.cn/blog_migrate/c08d5117c17a31ed6d92ce278dd8076c.png"/>
     </p>
     <p>
      <img alt="4472f3f49f6e3d49d6265ff27f28a913.png" src="https://i-blog.csdnimg.cn/blog_migrate/01da9a3c72522ca0b64ceb44c64a5fb3.png"/>
     </p>
     <p>
      这就基本囊括了我们常用的语言了。接下来，我们以 Python 和 Node.js 为例，看看如何使用这款应用。
     </p>
     <h4>
      构建 Python 应用
     </h4>
     <p>
      这里，我们以传统的
      <code>
       Hello World!
      </code>
      为起点，来开始我们的 Code App 之旅。
     </p>
     <p>
      首先创建一个 Python 源文件。展开文件资源管理器侧栏并选择新文件图标，我们可以使用内置模板或输入自定义文件名来创建一个空文件。这里我们暂且使用内置模板。
     </p>
     <p style="text-align:center;">
      <img alt="1f1c5e06d46423e1e2f6140c3378c200.png" src="https://i-blog.csdnimg.cn/blog_migrate/45f4e4915c6174efe8dd03743af954f8.png"/>
     </p>
     <p>
      我们现在可以看到用 Python 编写的
      <code>
       Hello World
      </code>
      程序。点击右上角的运行按钮运行。
     </p>
     <p>
      <img alt="d9f9948235167ebcbfe1de16065fd13c.png" src="https://i-blog.csdnimg.cn/blog_migrate/751bfa32bea85dc83bfc9623b8a5749d.png"/>
     </p>
     <p>
      在 Terminal 面板中，可以看到输出了我们想要的结果。怎么样，就是这么简单。
     </p>
     <p>
      当然，真正的应用还有很多重要的事情要做。其中一个就是第三方依赖包下载。第三方模块可以让我们做很多事情，包括向 Web 服务器发出网络请求、绘制图表以分析数据等等，可以为我们省去很多事情。
     </p>
     <p>
      Code App 集成了 pip 命令。与桌面端一样，要安装所需要的模块，只需要在终端中键入
      <code>
       pip install &lt;module name&gt;
      </code>
      。这里我们试用一下 request 模块来写个简单的网络请求。
     </p>
     <p>
      首先在终端中键入
      <code>
       pip install requests
      </code>
      ，可以看到模块的安装过程。
     </p>
     <img alt="e03ca790b6d57da6bfe81b9d2f8f6dec.png" src="https://i-blog.csdnimg.cn/blog_migrate/4977c49b9fed0b8732a41a64d7a11f41.png">
      <p>
       然后在代码编辑区，我们输入以下代码
      </p>
      <pre class="has"><code class="language-go">import requests

res = requests.get('http://api.open-notify.org/astros.json')
print(res.text)</code></pre>
<p>
运行可以看到相应的输出
</p>
<p style="text-align:center;">
<img alt="a0cee09cf21156d56ffd8549348ecd69.png" src="https://i-blog.csdnimg.cn/blog_migrate/341c121cb07da39d015eecaf4d04d32a.png"/>
</p>
<p>
酷吧！！！
</p>
<h4>
创建 Node.js 工程
</h4>
<p>
接下来，我们在 Code App 中，使用  Node.js 和 Express 构建一个简单的 Web 服务器。
</p>
<p>
首先创建一个新的文件夹，以便更好的组织文件。在在文件资源管理器部分，点击创建文件夹按钮以创建一个新文件夹。按住文件夹单元格并重命名文件夹。然后长按文件夹，在弹出菜单中选中
<code>
Assign as workplace foler
</code>
，将当前文件夹指定为工作区文件夹。
</p>
<p style="text-align:center;">
<img alt="74a1dd732e9e03fb10086e1b3c6f25dc.png" src="https://i-blog.csdnimg.cn/blog_migrate/67358e8a274972f97ca12d00212d916d.png"/>
</p>
<p>
然后我们通过 npm 来安装 Express 模块。在终端中运行
<code>
npm i express
</code>
，我们可以看到整个安装过程。安装完成后，我们可以看到一个名为 node_modules 的文件夹，这个大家应该很熟悉了。
</p>
<p>
然后创建 package.json 文件作为项目的配置文件。
</p>
<p>
使用 Express 创建 Web 服务器非常简单。创建一个新文件 index.js 并添加以下代码。
</p>
<pre class="has"><code class="language-go">const express = require('express');

const app = express();
const port = 3000;

app.get('/', (req, res) =&gt; {
    res.send('Hello from Express')
})

app.listen(port, () =&gt; console.log(`app listening on port ${port}`))</code></pre>
<p>
然后我们可以点击右上角的运行按钮，来运行服务器。可以看到 Code App 会并排启动 Safari 并访问 localhost:3000。神奇吧！！！
</p>
<p style="text-align:center;">
<img alt="4219fe4c28c75fc042ff37a75f9f6773.png" src="https://i-blog.csdnimg.cn/blog_migrate/66a068f0b471f7390dcc3ea513e11e88.png"/>
</p>
<p>
如果要停止服务器，可以点击终端右上角的停止按钮即可。
</p>
<h4>
版本控制
</h4>
<p>
Code App 还内置了 Git，以支持代码的版本控制。在使用 Git 之前，我们需要设置相关的账号信息。这个可以在
<code>
ettings &gt; Version Control &gt; Identity
</code>
完成。
</p>
<p>
如果需要拉取私有库或推送代码到私有库，还需要设置凭据，这个可以在
<code>
Settings &gt; Version Control &gt; Authentication
</code>
中完成。
</p>
<p>
要拉取代码库时，可以在侧边栏选择
<code>
Source Control
</code>
，输入存储库的 url 或通过在搜索字段中键入关键字来搜索 GitHub 上的存储库。目前只支持 HTTPS。
</p>
<p style="text-align:center;">
<img alt="3966dd0039ed05a9bd92643610d3b48a.png" src="https://i-blog.csdnimg.cn/blog_migrate/704df7b6e1f63f4604a28f215677100c.png"/>
</p>
<p>
当然，后续各种提交、推送、切换分支操作，Code App 都是支持的。
</p>
<p>
<img alt="6f43996e2ae9c03cdeb0a15f018dbe28.png" src="https://i-blog.csdnimg.cn/blog_migrate/475a87932db10f2c8fe43b961177815c.png"/>
</p>
<p>
<img alt="784a5d70918a3e20102042135d7f1602.png" src="https://i-blog.csdnimg.cn/blog_migrate/e003dcb3d49c098d75728a033249d2aa.png"/>
</p>
<h4>
连接到远程服务器
</h4>
<p>
Code App 还支持连接到自托管服务器并访问其文件和终端。要使用此功能，我们需要在我们的个人电脑或服务器上设置 ssh 或 ftp 服务器。在 Mac 上，我们可以通过在“系统偏好设置”的“共享”部分启用远程登录和完整磁盘访问来实现此目的。
</p>
<p style="text-align:center;">
<img alt="039ac19d715d8dc0958e7f44543d6ff7.png" src="https://i-blog.csdnimg.cn/blog_migrate/4d06d2682e48f6ec7a26e534aedd357a.png"/>
</p>
<p>
然后，在 Code App 的侧边栏中打开
<code>
Remote
</code>
，并输入服务器的信息。开启
<code>
Remember address
</code>
以保存傻气。后续再次连接服务器时，Code App 会要求进行 Face ID 或 Touch ID 身份验证。
</p>
<p style="text-align:center;">
<img alt="f427723bfef6e3a18404adf322057aac.png" src="https://i-blog.csdnimg.cn/blog_migrate/f3fea699e1d1b297342b6018cf9a6a7e.png"/>
</p>
<p>
我们可能还想使用密钥身份验证而不是密码。为此，首先在终端中运行
<code>
ssh-keygen
</code>
来生成 SSH 密钥。远程部分现在将显示一个
<code>
Show public key
</code>
按钮。点击它并将公钥复制到我们的远程服务器
<code>
~/.ssh/authorized_keys
</code>
。我们可以通过在远程服务器的终端中运行
<code>
open ~/.ssh/authorized_keys
</code>
来打开此文件。
</p>
<p>
现在我们就可以访问远程服务器的文件和终端了。
</p>
<p style="text-align:center;">
<img alt="3d8da33ab6cd7fe27252b7eb311cb807.png" src="https://i-blog.csdnimg.cn/blog_migrate/a5681ed9c7cfaaefb438f5bdb65178ed.png"/>
</p>
<h4>
安装
</h4>
<p>
我们可以直接在 App Store 上下载 Code App，地址是 [1] ，这款应用目前是收费的，价格是 $5.99，大家可以考虑支持一下。
</p>
<p>
当然，如果不想花这个钱，也可以自行编译构建，具体方法请参考 [2]。
</p>
<h4>
小结
</h4>
<p>
目前 Code App 还在不断的改进，后续还会添加更多的版本控制支持、更多的终端命令、带有 WebAssembly 的 C/C++ 运行时 等等。Code App 为 iPad 成为开发效率工具提供了更多的可能，值得尝试一下。
</p>
