     css3有哪些新属性
    </h1>

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <p>
     CSS3 相较于之前版本，增加了很多新属性和特性，使网页设计更具表现力和灵活性。以下是一些重要的 CSS3 新属性和特性，按照功能分类列举：
    </p>
    <h4>
     一、布局相关
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        Flexbox 布局
       </strong>
      </p>
      <ul>
       <li>
        <code>
         display: flex;
        </code>
       </li>
       <li>
        常用属性：
        <code>
         justify-content
        </code>
        ,
        <code>
         align-items
        </code>
        ,
        <code>
         flex-direction
        </code>
        ,
        <code>
         flex-wrap
        </code>
        ,
        <code>
         align-content
        </code>
        ,
        <code>
         flex-grow
        </code>
        ,
        <code>
         flex-shrink
        </code>
        ,
        <code>
         flex-basis
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        Grid 布局
       </strong>
      </p>
      <ul>
       <li>
        <code>
         display: grid;
        </code>
       </li>
       <li>
        常用属性：
        <code>
         grid-template-columns
        </code>
        ,
        <code>
         grid-template-rows
        </code>
        ,
        <code>
         grid-gap
        </code>
        ,
        <code>
         grid-area
        </code>
        ,
        <code>
         justify-items
        </code>
        ,
        <code>
         align-items
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     二、背景和渐变
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        渐变背景
       </strong>
      </p>
      <ul>
       <li>
        线性渐变：
        <code>
         background: linear-gradient();
        </code>
       </li>
       <li>
        径向渐变：
        <code>
         background: radial-gradient();
        </code>
       </li>
       <li>
        重复渐变：
        <code>
         background: repeating-linear-gradient();
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        多重背景
       </strong>
      </p>
      <ul>
       <li>
        <code>
         background-image: url(image1), url(image2);
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     三、边框和阴影
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        圆角边框
       </strong>
      </p>
      <ul>
       <li>
        <code>
         border-radius: 10px;
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        边框图片
       </strong>
      </p>
      <ul>
       <li>
        <code>
         border-image: url(border.png) 30 round;
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        盒阴影
       </strong>
      </p>
      <ul>
       <li>
        <code>
         box-shadow: 10px 10px 5px #888;
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        文本阴影
       </strong>
      </p>
      <ul>
       <li>
        <code>
         text-shadow: 2px 2px 5px #333;
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     四、动画和过渡
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        过渡效果
       </strong>
      </p>
      <ul>
       <li>
        <code>
         transition: all 0.3s ease;
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        动画效果
       </strong>
      </p>
      <ul>
       <li>
        <code>
         animation: slide 2s infinite;
        </code>
       </li>
       <li>
        <code>
         @keyframes slide { from { left: 0; } to { left: 100px; } }
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     五、文本效果
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        自定义字体
       </strong>
      </p>
      <ul>
       <li>
        <code>
         @font-face { font-family: 'MyFont'; src: url('myfont.woff2'); }
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        文字换行控制
       </strong>
      </p>
      <ul>
       <li>
        <code>
         word-wrap: break-word;
        </code>
       </li>
       <li>
        <code>
         word-break: break-all;
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     六、颜色和透明度
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        RGBA 和 HSLA 颜色
       </strong>
      </p>
      <ul>
       <li>
        <code>
         color: rgba(255, 0, 0, 0.5);
        </code>
       </li>
       <li>
        <code>
         color: hsla(120, 100%, 50%, 0.3);
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        不透明度
       </strong>
      </p>
      <ul>
       <li>
        <code>
         opacity: 0.8;
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     七、盒模型增强
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        盒子大小控制
       </strong>
      </p>
      <ul>
       <li>
        <code>
         box-sizing: border-box;
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        轮廓线增强
       </strong>
      </p>
      <ul>
       <li>
        <code>
         outline-offset: 5px;
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     八、媒体查询和响应式设计
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        媒体查询
       </strong>
      </p>
      <ul>
       <li>
        <code>
         @media screen and (max-width: 600px) { ... }
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        视口单位
       </strong>
      </p>
      <ul>
       <li>
        <code>
         vw
        </code>
        ,
        <code>
         vh
        </code>
        ,
        <code>
         vmin
        </code>
        ,
        <code>
         vmax
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     九、过滤效果
    </h4>
    <ol>
     <li>
      <strong>
       滤镜效果
      </strong>
      <ul>
       <li>
        <code>
         filter: blur(5px);
        </code>
       </li>
       <li>
        <code>
         filter: grayscale(50%);
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     十、变换和透视
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        2D 变换
       </strong>
      </p>
      <ul>
       <li>
        <code>
         transform: rotate(45deg);
        </code>
       </li>
       <li>
        <code>
         transform: scale(1.5);
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        3D 变换
       </strong>
      </p>
      <ul>
       <li>
        <code>
         transform: rotateX(45deg) rotateY(30deg);
        </code>
       </li>
       <li>
        <code>
         perspective: 1000px;
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     总结
    </h4>
    <p>
     CSS3 引入了大量增强页面布局和视觉效果的新特性，尤其是在响应式布局、动画效果和视觉美化方面有了很大提升。作为前端开发者，掌握这些特性可以让页面效果更加生动和现代化。
    </p>
