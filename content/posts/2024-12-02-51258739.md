# python绘制折线图示例

绘制结果：

![](http://i.imgur.com/l2Y2hLX.png)

python代码：

```
import numpy as np
import pylab as pl

x = [1, 2, 3, 4, 5]  # Make an array of x values
y = [1, 4, 9, 16, 25]  # Make an array of y values for each x value

pl.plot(x, y)  # use pylab to plot x and y
pl.show()   # show the plot on the screen

```
