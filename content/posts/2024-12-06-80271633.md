    <h3 id="encoding">
     encoding
    </h3>
    <p>
     根据Python官方文档中有关字符串的部分
     <a href="https://docs.python.org/3/library/stdtypes.html?highlight=encode#string-methods" rel="nofollow">
      1
     </a>
     ：
    </p>
    <blockquote>
     <p>
      str.encode(encoding=”utf-8”, errors=”strict”)
      <br/>
      Return an encoded version of the string as a bytes object. Default encoding is ‘utf-8’. errors may be given to set a different error handling scheme. The default for errors is ‘strict’, meaning that encoding errors raise a UnicodeError. Other possible values are ‘ignore’, ‘replace’, ‘xmlcharrefreplace’, ‘backslashreplace’ and any other name registered via codecs.register_error(), see section Error Handlers. For a list of possible encodings, see section Standard Encodings.
     </p>
    </blockquote>
    <p>
     encoding是编码的意思，在python中，Unicode类型是作为编码的基础类型。
    </p>
    <h3 id="unicode">
     Unicode
    </h3>
    <p>
     Unicode是一种标准，包括了字符集、编码方案等。因为ASCII码只能编码英文字符，具有很大的局限性，而Unicode 为每种语言中的每个字符设定了统一并且唯一的二进制编码，以满足跨语言、跨平台进行文本转换、处理的要求
     <a href="https://baike.baidu.com/item/Unicode" rel="nofollow">
      2
     </a>
     。在
     <a href="https://unicode-table.com/en/" rel="nofollow">
      https://unicode-table.com/en/
     </a>
     可以查到所有的Unicode字符。
    </p>
    <h3 id="utf-8">
     utf-8
    </h3>
    <p>
     UTF-8以字节为单位对Unicode进行编码。
    </p>
    <h3 id="utf-16">
     utf-16
    </h3>
    <p>
     UTF-16编码以16位无符号整数为单位。
    </p>
    <h3 id="utf-32">
     utf-32
    </h3>
    <p>
     UTF-32编码以32位无符号整数为单位。
    </p>
    <h3 id="gbk">
     gbk
    </h3>
    <p>
     GBK全称《汉字内码扩展规范》（GBK即“国标”、“扩展”汉语拼音的第一个字母，英文名称：Chinese Internal Code Specification）
     <a href="https://baike.baidu.com/item/GBK字库?fromtitle=GBK&amp;fromid=481954" rel="nofollow">
      3
     </a>
     。GBK是采用单双字节变长编码，英文使用单字节编码，完全兼容ASCII字符编码，中文部分采用双字节编码。
    </p>
    <h3 id="ascii-码">
     ASCII 码
    </h3>
    <p>
     ASCII 码使用指定的7 位或8 位二进制数组合来表示128 或256 种可能的字符
     <a href="https://baike.baidu.com/item/ASCII/309296" rel="nofollow">
      4
     </a>
     。
    </p>
    <h3 id="decoding">
     decoding
    </h3>
    <p>
     根据Python官方文档中有关字符串的部分
     <a href="https://docs.python.org/3/library/stdtypes.html?highlight=encode#string-methods" rel="nofollow">
      1
     </a>
     ：
    </p>
    <blockquote>
     <p>
      bytes.decode(encoding=”utf-8”, errors=”strict”)
      <br/>
      bytearray.decode(encoding=”utf-8”, errors=”strict”)
      <br/>
      Return a string decoded from the given bytes. Default encoding is ‘utf-8’. errors may be given to set a different error handling scheme. The default for errors is ‘strict’, meaning that encoding errors raise a UnicodeError. Other possible values are ‘ignore’, ‘replace’ and any other name registered via codecs.register_error(), see section Error Handlers. For a list of possible encodings, see section Standard Encodings.
      <br/>
      Note Passing the encoding argument to str allows decoding any bytes-like object directly, without needing to make a temporary bytes or bytearray object.
     </p>
    </blockquote>
    <p>
     decode()就是将“字节流”按照某种规则转换成“文本”，可以理解为是encode()的逆过程。
    </p>
    <h3 id="例子">
     例子
    </h3>
    <p>
     现在用一个小小的例子来体验一下编码和解码。

     首先可以去
     <a href="https://learnpythonthehardway.org/python3/languages.txt" rel="nofollow">
      https://learnpythonthehardway.org/python3/languages.txt
     </a>
     上将上面的字符保存在一个txt文件中，并以utf-8的形式保存，命名为languages。写下代码：
    </p>
    <pre class="prettyprint"><code class="hljs python"><span class="hljs-comment">#####encoding&amp;decoding#####</span>
<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">main</span><span class="hljs-params">(file,encoding,errors)</span>:</span>
    line = file.readline()
    <span class="hljs-keyword">if</span> line:
        print_file(line,encoding,errors)
        <span class="hljs-keyword">return</span> main(file,encoding,errors)

<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">print_file</span><span class="hljs-params">(line,encoding,errors)</span>:</span>
    next_lang = line.strip() <span class="hljs-comment">###strip() 方法用于移除字符串头尾指定的字符（默认为空格）</span>
    raw_bytes = next_lang.encode(encoding, errors=errors) <span class="hljs-comment">###encode()函数中errors默认为strict，还可以设置为ignore,replace等</span>
    cooked_string = raw_bytes.decode(encoding, errors=errors) <span class="hljs-comment">###decode()函数为解码</span>

    print(raw_bytes, <span class="hljs-string">"&lt;===&gt;"</span>, cooked_string)
