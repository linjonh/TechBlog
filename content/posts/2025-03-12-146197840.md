     📁 文件/目录操作
    </h3>
    <h4>
     <a id="1__3">
     </a>
     1. 基础操作
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 递归复制目录（含子目录）</span>
<span class="token function">cp</span> <span class="token parameter variable">-r</span> source_dir/ target_dir/

<span class="token comment"># 递归删除目录（强制删除不提示）</span>
<span class="token function">rm</span> <span class="token parameter variable">-rf</span> dir_name/

<span class="token comment"># 查看当前路径</span>
<span class="token builtin class-name">pwd</span>

<span class="token comment"># 创建多级目录</span>
<span class="token function">mkdir</span> <span class="token parameter variable">-p</span> parent_dir/child_dir
</code></pre>
    <h4>
     <a id="2__17">
     </a>
     2. 权限管理
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 修改文件权限（755 = rwxr-xr-x）</span>
<span class="token function">chmod</span> <span class="token number">755</span> filename

<span class="token comment"># 递归修改目录权限</span>
<span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">755</span> dir_name/

<span class="token comment"># 修改文件所有者</span>
<span class="token function">chown</span> user:group filename
</code></pre>
    <h3>
     <a id="__29">
     </a>
     🌐 网络传输
    </h3>
    <h4>
     <a id="1_SCP__30">
     </a>
     1. SCP 安全拷贝
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 基础格式（注意大写 -P 指定端口）</span>
<span class="token function">scp</span> <span class="token parameter variable">-P</span> <span class="token number">3178</span> local_file user@remote_host:/remote/dir

<span class="token comment"># 递归传输目录（示例来自用户需求）</span>
<span class="token function">scp</span> <span class="token parameter variable">-r</span> <span class="token parameter variable">-P</span> <span class="token number">3178</span> /path/salicon user@ip:/path/dataset

<span class="token comment"># 从远程服务器下载文件</span>
<span class="token function">scp</span> <span class="token parameter variable">-P</span> <span class="token number">2222</span> user@host:/remote/file /local/dir
</code></pre>
    <h4>
     <a id="2_RSYNC_SCP_42">
     </a>
     2. RSYNC 增量同步（推荐替代SCP）
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 带压缩和进度显示的同步</span>
<span class="token function">rsync</span> <span class="token parameter variable">-avzP</span> <span class="token parameter variable">-e</span> <span class="token string">"ssh -p 3178"</span> /local/dir user@host:/remote/dir
</code></pre>
