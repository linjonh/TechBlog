    <p>

    <h3>
     <a id="12__5">
     </a>
     1.2 编辑虚拟机
    </h3>
    <p>
     创建完成后，点击编辑虚拟机设置，可对虚拟机内存、处理器、硬盘等各再次进行编辑设置。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b68e099019524f3ba933970a23c9cc34.jpeg#pic_center">

     </img>
    </p>
    <h2>
     <a id="_13">
     </a>
     二、虚拟机开关机
    </h2>
    <h3>
     <a id="21__14">
     </a>
     2.1 打开虚拟机
    </h3>
    <p>
     虚拟机创建成功后，点击
     <strong>
      开启此虚拟机
     </strong>
     ，可打开创建的虚拟机。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/07a90a012d3146a8852917526dad6409.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="22__20">
     </a>
     2.2 关闭虚拟机
    </h3>
    <p>
     方法1：使用开始菜单关机。
     <br/>
     在虚拟机操作系统中点击开始菜单，选择关机，可正常关闭虚拟机。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/55dd0de8eaab4893a7e820c9b0958243.jpeg#pic_center"/>
    </p>
    <p>
     方法2：使用菜单栏中虚拟机进行关机。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a4249cc159da4e0a9da065a96d2c8f27.jpeg#pic_center"/>
    </p>
    <h2>
     <a id="_29">
     </a>
     三、虚拟机快照功能
    </h2>
    <h3>
     <a id="31__30">
     </a>
     3.1 拍摄快照
    </h3>
    <p>
     拍摄快照相当于对当前系统运行状态进行暂存。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dd63fce8dbb543c7913967dc260246a5.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="32__35">
     </a>
     3.2 恢复快照
    </h3>
    <p>
     可使用回复快照功能恢复到之前保存的系统运行状态。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/10663b5e6357465a995cb5132003d5ac.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="33__40">
     </a>
     3.3 管理快照
    </h3>
    <p>
     使用快照管理器可对保存的快照进行删除等功能。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/537fc694b1e94a179227d9374524b1ca.jpeg#pic_center"/>
    </p>
    <h2>
     <a id="_44">
     </a>
     四、虚拟机导出和克隆
    </h2>
    <h3>
     <a id="41__45">
     </a>
     4.1 虚拟机导出
    </h3>
    <p>
     使用导出功能，可对当前虚拟机进行保存导出为OVF文件，可选择此文件用于直接打开虚拟机。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3c28c639bf4c41c09204de3150ae09a0.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="42__51">
     </a>
     4.2 虚拟机克隆
    </h3>
    <p>
     使用克隆功能，可对当前虚拟机进行克隆并导出文件到本地，可选择此文件，直接打开虚拟机。
    </p>
    <h2>
     <a id="_54">
     </a>
     五、打开虚拟机
    </h2>
    <p>
     可使用打开虚拟机功能，直接打开使用保存的克隆文件或者导出的OVF文件。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/138258ff6d0a45ea918f34f7357487a9.jpeg#pic_center"/>
    </p>
