    <h2>
     项目介绍
    </h2>
    <p>
     SafeLine，中文名 "雷池"，是一款简单好用, 效果突出的
     <strong>
      <code>
       Web 应用防火墙(WAF)
      </code>
     </strong>
     ，可以保护 Web 服务不受黑客攻击。
    </p>
    <p>
     雷池通过过滤和监控 Web 应用与互联网之间的 HTTP 流量来保护 Web 服务。可以保护 Web 服务免受
     <code>
      SQL 注入
     </code>
     、
     <code>
      XSS
     </code>
     、
     <code>
      代码注入
     </code>
     、
     <code>
      命令注入
     </code>
     、
     <code>
      CRLF 注入
     </code>
     、
     <code>
      ldap 注入
     </code>
     、
     <code>
      xpath 注入
     </code>
     、
     <code>
      RCE
     </code>
     、
     <code>
      XXE
     </code>
     、
     <code>
      SSRF
     </code>
     、
     <code>
      路径遍历
     </code>
     、
     <code>
      后门
     </code>
     、
     <code>
      暴力破解
     </code>
     、
     <code>
      CC
     </code>
     、
     <code>
      爬虫
     </code>
     等攻击。
    </p>
    <p>
    </p>
    <h2 id="核心能力">
     核心能力
    </h2>
    <p>
     对于你的网站而言, 雷池可以实现如下效果:
    </p>
    <ul>
     <li>
      <strong>
       <code>
        阻断 Web 攻击
       </code>
      </strong>
      <ul>
       <li>
        可以防御所有的 Web 攻击，例如
        <code>
         SQL 注入
        </code>
        、
        <code>
         XSS
        </code>
        、
        <code>
         代码注入
        </code>
        、
        <code>
         操作系统命令注入
        </code>
        、
        <code>
         CRLF 注入
        </code>
        、
        <code>
         XXE
        </code>
        、
        <code>
         SSRF
        </code>
        、
        <code>
         路径遍历
        </code>
        等等。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        限制访问频率
       </code>
      </strong>
      <ul>
       <li>
        限制用户的访问速率，让 Web 服务免遭
        <code>
         CC 攻击
        </code>
        、
        <code>
         暴力破解
        </code>
        、
        <code>
         流量激增
        </code>
        和其他类型的滥用。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        人机验证
       </code>
      </strong>
      <ul>
       <li>
        互联网上有来自真人用户的流量，但更多的是由爬虫, 漏洞扫描器, 蠕虫病毒, 漏洞利用程序等自动化程序发起的流量，开启雷池的人机验证功能后真人用户会被放行，恶意爬虫将会被阻断。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        身份认证
       </code>
      </strong>
      <ul>
       <li>
        雷池的 "身份认证" 功能可以很好的解决 "未授权访问" 漏洞，当用户访问您的网站时，需要输入您配置的用户名和密码信息，不持有认证信息的用户将被拒之门外。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        动态防护
       </code>
      </strong>
      <ul>
       <li>
        在用户浏览到的网页内容不变的情况下，将网页赋予动态特性，对 HTML 和 JavaScript 代码进行动态加密，确保每次访问时这些代码都以随机且独特的形态呈现。
       </li>
      </ul>
     </li>
    </ul>
