    <h2>
     <span style="color:#0d0016">
      1.基本语法规范
     </span>
    </h2>
    <p>
     <span style="color:#0d0016">
      选择器 + {⼀条/N条声明}
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      • 选择器决定针对谁修改 (找谁)
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      • 声明决定修改啥. (⼲啥)
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      • 声明的属性是键值对. 使⽤ ; 区分键值对, 使⽤ : 区分键和值
     </span>
    </p>
    <p>
     比如：
    </p>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;CSS_Test&lt;/title&gt;
    &lt;style&gt;
        h1{
            color: red;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;h1&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <img alt="" height="144" src="https://i-blog.csdnimg.cn/direct/906f968382b94473952bfbe2421890c4.png" width="784"/>
    </p>
    <blockquote>
     <p>
      <span style="color:#646a73">
       style 标签可以放到⻚⾯任意位置. ⼀般放到 head 标签内.
      </span>
     </p>
    </blockquote>

    <h2>
     3.CSS选择器
    </h2>
    <p>
     3.1标签选择器
    </p>
    <blockquote>
     <p>
      注意所有的
      <span style="color:#646a73">
       CSS 都要写到 style 标签中
      </span>
     </p>
    </blockquote>
    <pre><code class="hljs">    &lt;style&gt;
        div{
            color: red;
        }
    &lt;/style&gt;</code></pre>
    <p>
    </p>
    <p>
     <img alt="" height="91" src="https://i-blog.csdnimg.cn/direct/521330f52fef45109de982027e7d4dc6.png" width="782">
     </img>
    </p>
    <h3>
     3.2类选择器
    </h3>
    <pre><code class="hljs">    &lt;style&gt;
        .test {
            color: red;
        }
    &lt;/style&gt;   
 &lt;div class = "test"&gt;
        Hello World
    &lt;/div&gt;</code></pre>
    <p>
     <span style="color:#646a73">
      ⼀个类可以被多个标签使⽤, ⼀个标签也能使⽤多个类
     </span>
    </p>
    <h3>
     <span style="color:#646a73">
      3.3 ID选择器
     </span>
    </h3>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;CSS_Test&lt;/title&gt;
    &lt;style&gt;
        #test {
            color: red;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div id = "test"&gt;
        Hello World
    &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <img alt="" height="114" src="https://i-blog.csdnimg.cn/direct/2abbb2ee458345a489fd2083dcd078d7.png" width="664"/>
    </p>
    <blockquote>
     <p>
      注意id比class优先
     </p>
    </blockquote>
    <h3>
     <span style="color:#0d0016">
      3.4 通配符选择器
     </span>
    </h3>
    <pre><code class="hljs">    &lt;style&gt;
        * {
            color: red;
        }
    &lt;/style&gt;</code></pre>
    <p>
     设置所有元素为红色

    <h2>
     4.常用CSS
    </h2>
    <h3>
     <span style="color:#0d0016">
      4.1 color
     </span>
    </h3>
    <p>
     <span style="color:#0d0016">
      color: 设置字体颜⾊
     </span>
    </p>
    <pre><code class="hljs">.text1{
 font-size: 32px;
}</code></pre>
    <div>
     <p>
      <span style="color:#0d0016">
       颜⾊有如下⼏种表达⽅式:
      </span>
     </p>

      <span style="color:#0d0016">
       • 英⽂单词,如red,blue
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       • rgb代码的颜⾊ 如rgb(255,0,0)
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       • ⼗六进制的颜⾊ 如#ff00ff
