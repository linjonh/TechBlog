    <p>
     1. 执行以下SQL：
    </p>
    <p>
     SELECT
     <br/>
     TABLE_NAME 表名,
     <br/>
     COLUMN_NAME 字段,
     <br/>
     COLUMN_COMMENT 字段名称,
     <br/>
     COLUMN_TYPE 数据类型,
     <br/>
     IS_NULLABLE 是否为空,
     <br/>
     COLUMN_KEY 约束条件,
     <br/>
     COLUMN_COMMENT 备注
     <br/>
     FROM
     <br/>
     INFORMATION_SCHEMA.COLUMNS
     <br/>
     where
     <br/>
     -- test 为数据库名称，复制粘贴改为你的数据库名称即可
     <br/>
     table_schema ='databasename'
     <br/>
     -- 输出所有该数据库所有表含字段
     <br/>
     order by table_name
    </p>
    <p>
     2. 执行查询除结构后，点击导出结果，选择想要的格式进行导出。
