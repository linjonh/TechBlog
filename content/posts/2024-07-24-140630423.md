    <h2>
     <a id="DevEco_Studio__6">
     </a>
     DevEco Studio 简介
    </h2>
    <p>
     DevEco Studio是华为为HarmonyOS开发者提供的一款集成开发环境（IDE）。它基于IntelliJ IDEA开发，专门用于构建、调试和优化HarmonyOS应用程序和服务。DevEco Studio为开发者提供了丰富的工具和功能，旨在简化开发流程，提高开发效率，并帮助开发者快速构建高质量的HarmonyOS应用。
    </p>
    <h2>
     <a id="DevEco_Studio__9">
     </a>
     一、DevEco Studio 主要功能
    </h2>
    <p>
     <strong>
      多设备支持：
     </strong>
     DevEco Studio支持HarmonyOS的多设备开发，包括智能手机、智能手表、智能电视等。开发者可以在同一项目中为多个终端设备编写和调试代码，充分利用HarmonyOS的分布式特性。
    </p>
    <p>
     <strong>
      高效的开发工具：
     </strong>
     DevEco Studio集成了代码编辑器、项目管理工具、调试器、仿真器和构建工具，提供了全方位的开发支持。其智能代码补全、代码重构、静态代码分析等功能，使开发过程更加高效和便捷。
    </p>
    <p>
     <strong>
      HarmonyOS特性支持：
     </strong>
     DevEco Studio内置了对HarmonyOS特性的支持，如Ability、Distributed Data Management、Atomic Services等，开发者可以方便地使用这些特性来构建复杂的分布式应用。
    </p>
    <p>
     <strong>
      丰富的插件和扩展：
     </strong>
     DevEco Studio支持多种插件和扩展，开发者可以根据自己的需求安装和使用这些插件，以扩展IDE的功能和提高工作效率。
    </p>
    <p>
     <strong>
      完善的调试和测试工具：
     </strong>
     DevEco Studio提供了强大的调试工具和丰富的测试框架，支持多种调试方式，如本地调试、远程调试和分布式调试。开发者可以使用这些工具快速发现和解决代码中的问题，确保应用的稳定性和性能。
    </p>
    <p>
     <strong>
      集成化的服务：
     </strong>
     DevEco Studio集成了华为云服务和其他第三方服务，开发者可以方便地调用这些服务来实现更丰富的应用功能，如数据存储、消息推送、用户认证等。
    </p>
    <p>
     DevEco Studio为HarmonyOS开发者提供了一个功能强大且易用的开发平台，助力开发者高效地构建和优化多设备、多场景的HarmonyOS应用。
    </p>

    <h2>
     <a id="DevEco_Studio__52">
     </a>
     三、DevEco Studio 配置开发环境
    </h2>
    <p>
     应用/服务支持API Version 4至9，首次使用DevEco Studio，工具的配置向导会引导您下载SDK及工具链。配置向导默认下载 API Version 9的SDK及工具链，如需下载API Version 4至8，可在工程配置完成后，进入HarmonyOS SDK界面手动下载，方法如下：
     <br/>
     在DevEco Studio欢迎页，单击Configure（或
     <img alt="或图标" src="https://i-blog.csdnimg.cn/direct/2b05698778c747ccacae82d5577499e0.png">
      图标）&gt; Settings &gt; SDK &gt; HarmonyOS页签（macOS系统为Configure &gt; Preferences &gt; SDK &gt; HarmonyOS）。
      <br/>
      在DevEco Studio打开工程的情况下，单击Files &gt; Settings &gt; SDK &gt; HarmonyOS页签进入（macOS系统为DevEco Studio &gt; Preferences &gt; SDK &gt; HarmonyOS）。
     </img>
    </p>
    <p>
     接下来介绍首次启动DevEco Studio的配置向导：
    </p>
    <ol>
     <li>
      运行已安装的DevEco Studio，首次使用，请选择Do not import settings，单击OK。
     </li>
     <li>
      安装Node.js与ohpm。可以指定本地已安装的Node.js或ohpm（Node.js版本要求为v14.19.1及以上，且低于v17.0.0；对应的npm版本要求为6.14.16及以上）路径位置；如果本地没有合适的版本，可以选择Install按钮，选择下载源和存储路径后，进行在线下载，单击Next进入下一步。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      如果配置向导界面出现的是设置HTTP Proxy Setup，说明网络受限，请根据参考信息配置DevEco Studio代理后，再下载Node.js、ohpm和SDK。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/95bbb927d2e144828836d2d77b40fabc.png#pic_center">
      <br/>
      3. 在SDK Setup界面，单击
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0141efe8e37c459ebaa5444b26193c52.png">
       按钮，设置HarmonyOS SDK存储路径，单击Next进入下一步。
      </img>
     </img>
    </p>
    <blockquote>
     <p>
      说明
      <br/>

    </ol>

    <h2>
     <a id="Hello_World_94">
     </a>
     四、创建和运行Hello World
    </h2>
    <p>
     DevEco Studio配置开发环境完成后，可以通过运行Hello World工程来验证环境设置是否正确。接下来以创建一个Phone设备的工程为例进行介绍。
    </p>
    <h3>
     <a id="41__97">
     </a>
     4.1 创建一个新工程
    </h3>
    <ol>
     <li>
      打开DevEco Studio，在欢迎页单击Create Project，创建一个新工程。
     </li>
     <li>
      根据工程创建向导，选择创建Application应用或Atomic Service元服务。选择“Empty Ability”模板，然后单击Next。关于工程模板的介绍和支持的设备类型，请参考工程模板和开发语言介绍。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dcbfe646c9b44b4c9799ce395eb0e399.png#pic_center"/>
     </li>
     <li>
      填写工程相关信息，保持默认值即可，单击Finish。关于各个参数的详细介绍，请参考创建一个新的工程。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      默认情况下，新建工程的Compile SDK为9，在设备中运行该工程时，请选择API 9及以上的设备才能运行该工程。
     </p>
    </blockquote>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/7caee16081154dc0a6992b04c3eda9c5.png#pic_center"/>
     工程创建完成后，DevEco Studio会自动进行工程的同步。
    </p>
    <h3>
     <a id="42_Hello_World_108">
     </a>
     4.2 运行Hello World
    </h3>
    <p>
     Hello World工程可以运行在模拟器中，或者运行在真机设备中。本示例以选择将Hello World工程运行在模拟器中进行说明，如果选择运行在真机设备中，需要先对工程进行签名，然后才能运行在真机设备中，详细指导请参考在Phone设备中运行应用/服务。
    </p>
    <p>
     DevEco Studio提供远程模拟器和本地模拟器，本示例以远程模拟器为例进行说明。关于本地模拟器的使用请参考使用Local Emulator运行应用/服务。
    </p>
    <ol>
     <li>
      在DevEco Studio菜单栏，单击Tools &gt; Device Manager。
     </li>
     <li>
      在Remote Emulator页签中单击Sign In，在浏览器中弹出华为开发者联盟帐号登录界面，请输入
      <a href="https://developer.huawei.com/consumer/cn/doc/start/ibca-0000001062388135" rel="nofollow">
       已实名认证
      </a>
      的华为开发者联盟帐号的用户名和密码进行登录。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      a. 推荐使用最新版本Chrome浏览器，如果使用Safari、360等其他浏览器，要取消阻止跨站跟踪和阻止所有Cookie功能。
      <br/>
      b. 点击Sign in之后，如果DevEco Studio一直显示进度条等待弹框，并未有浏览器弹出并打开开发者联盟登录界面，此时您可以主动切换到浏览器查看相应的界面。更多问题可以查看远程模拟器登录常见问题。
     </p>
    </blockquote>
    <ol start="3">
     <li>
      登录后，请单击界面的允许按钮进行授权，授权完成后，切换回Device Manager界面。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/910cde0376af40c8bb480e0906292990.png#pic_center"/>
     </li>
     <li>
      在设备列表中，选择Phone设备P50，并单击
