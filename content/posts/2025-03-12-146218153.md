    <h4 id="PWLVL">
     验证优化效果
    </h4>
    <p>
     <img alt="" height="365" src="https://i-blog.csdnimg.cn/direct/db2aaa7a43234633aeb40fcfaf1867da.png" width="1370"/>
    </p>
    <p>
     <strong>
      方案 1：临时使用国内镜像源（推荐）
     </strong>
    </p>
    <pre id="tkpjQ"><code>pip install uv -i https://pypi.tuna.tsinghua.edu.cn/simple</code></pre>
    <ul>
     <li id="ud03e3946">
      <strong>
       速度提升
      </strong>
      ：镜像源服务器位于国内，带宽充足，通常可达 1-10MB/s
     </li>
     <li id="u8959fd68">
      <strong>
       支持源列表
      </strong>
      ：
     </li>
    </ul>
    <pre id="Qj1jH"><code># 清华源（推荐）
-i https://pypi.tuna.tsinghua.edu.cn/simple
# 阿里云源
-i https://mirrors.aliyun.com/pypi/simple
# 豆瓣源
-i https://pypi.douban.com/simple</code></pre>
    <h5 id="xnWCs">
     方案 2：永久配置镜像源
    </h5>
    <pre id="h8KU2"><code># 创建配置文件（Linux/macOS）
mkdir -p ~/.pip
echo "[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn" &gt; ~/.pip/pip.conf</code></pre>
    <ul>
     <li id="uff1f0a37">
      <strong>
       验证生效
      </strong>
      ：后续所有
      <code>
       pip install
      </code>
      命令自动使用该源
     </li>
    </ul>
    <hr/>
    <h4 id="ND7GN">
     常见问题补充
    </h4>
    <ol>
     <li id="u8a1fa86a">
      <strong>
       SSL 证书错误
      </strong>
      ：添加
      <code>
       --trusted-host
      </code>
      参数
     </li>
    </ol>
    <pre id="BApLT"><code>pip install uv -i https://mirrors.aliyun.com/pypi/simple --trusted-host mirrors.aliyun.com</code></pre>
    <ol>
     <li id="ube2ba95c">
      <strong>
       包哈希校验失败
      </strong>
      ：尝试更换其他镜像源
     </li>
     <li id="u05491ec7">
      <strong>
       版本兼容性问题
      </strong>
      ：使用
      <code>
       pip install uv==0.6.5
      </code>
      指定旧版本
     </li>
    </ol>
