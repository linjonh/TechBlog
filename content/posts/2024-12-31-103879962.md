    <h4>
     <a id="_1">
     </a>
     简介
    </h4>
    <p>
     apt-get，是一条linux命令，适用于deb包管理式的操作系统（例如Ubuntu系统），主要用于自动从互联网的软件仓库中搜索、安装、升级、卸载软件或操作系统。
    </p>
    <p>
    </p>
    <h4>
     <a id="_6">
     </a>
     参数
    </h4>
    <pre><code class="prism language-bash"> -h 		帮助文件。  

 -q 		输出到日志 - 无进展指示  

 -qq 		不输出信息，错误除外  

 -d 		仅下载 - 不安装或解压归档文件  

 -s 		不实际安装。模拟执行命令  

 -y 		在需要确认的场景中回应 <span class="token function">yes</span>

 -f 		尝试修正系统依赖损坏处  

 -m 		如果归档无法定位，尝试继续  

 -u 		同时显示更新软件包的列表  

 -b 		获取源码包后编译  

 -V 		显示详细的版本号  

 -c<span class="token operator">=</span>? 		阅读此配置文件  

 -o<span class="token operator">=</span>? 		设置自定的配置选项，如 -o dir::cache<span class="token operator">=</span>/tmp  
</code></pre>
    <p>
    </p>
    <h4>
     <a id="___38">
     </a>
     升级 &amp; 安装
    </h4>
    <pre><code class="prism language-bash"><span class="token function">apt-get</span> update						// 更新源文件，并不会做任何安装升级操作

<span class="token function">apt-get</span> upgrade						// 升级所有已安装的包

<span class="token function">apt-get</span> <span class="token function">install</span> packagename				// 安装指定的包

<span class="token function">apt-get</span> <span class="token function">install</span> packagename --only-upgrade		// 仅升级指定的包

<span class="token function">apt-get</span> <span class="token function">install</span> packagename --reinstall   		// 重新安装包

<span class="token function">apt-get</span> -f <span class="token function">install</span>   					// 修复安装

<span class="token function">apt-get</span> build-dep packagename				// 安装相关的编译环境

<span class="token function">apt-get</span> <span class="token function">source</span> packagename  				// 下载该包的源代码

<span class="token function">apt-get</span> dist-upgrade 					// 升级系统

<span class="token function">apt-get</span> dselect-upgrade 				// 使用 dselect 升级
</code></pre>
    <p>
    </p>
    <h4>
     <a id="___64">
     </a>
     查询 &amp; 显示
    </h4>
    <pre><code class="prism language-bash">apt-cache search packagename 				// 查询指定的包  　　

apt-cache show packagename 				// 显示包的相关信息，如说明、大小、版本等 

apt-cache depends packagename 				// 了解使用该包依赖哪些包

apt-cache rdepends packagename 				// 查看该包被哪些包依赖
</code></pre>
    <p>
    </p>
    <h4>
     <a id="_78">
     </a>
     删除
    </h4>
    <pre><code class="prism language-bash"><span class="token function">apt-get</span> remove packagename				// 删除包  　　

<span class="token function">apt-get</span> remove packagename -- purge 			// 删除包，包括删除配置文件等 

<span class="token function">apt-get</span> autoremove packagename --purge 			// 删除包及其依赖的软件包+配置文件等（只对6.10有效，推荐使用）
</code></pre>
    <p>
    </p>
    <h4>
     <a id="___90">
     </a>
     清理 &amp; 检查
    </h4>
    <pre><code class="prism language-bash"><span class="token function">apt-get</span> clean 						// 清理无用的包 

<span class="token function">apt-get</span> autoclean 					// 清理无用的包 

<span class="token function">apt-get</span> check 						// 检查是否有损坏的依赖
</code></pre>
    <p>
    </p>
    <h4>
     <a id="aptget_upgrade_102">
     </a>
     apt-get upgrade更新时忽略指定包
    </h4>
    <p>
     // 使用这个命令可以将指定的包的版本
     <code>
      hold
     </code>
     住，这样在更新的时候就会忽略掉这个包。
    </p>
    <pre><code class="prism language-bash">apt-mark hold packagename		
</code></pre>
    <p>
     // 将
     <code>
      hold
     </code>
     替换为
     <code>
      unhold
     </code>
     就可以取消对这个包版本的锁定了。
    </p>
    <pre><code class="prism language-bash">apt-mark unhold packagename
</code></pre>
