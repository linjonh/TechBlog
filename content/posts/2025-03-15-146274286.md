    <h4>
     <a id="_43">
     </a>
     部署运行
    </h4>
    <h5>
     <a id="_45">
     </a>
     后端运行步骤
    </h5>
    <p>
     (1) 安装python 3.8
    </p>
    <p>
     (2) 安装依赖。进入server目录下，执行 pip install -r requirements.txt
    </p>
    <p>
     (3) 安装mysql 5.7数据库，并创建数据库，命名为xxx，创建SQL如下：
    </p>
    <pre><code>CREATE DATABASE IF NOT EXISTS python_hospital DEFAULT CHARSET utf8 COLLATE utf8_general_ci
</code></pre>
    <p>
     (4) 恢复xxx.sql数据。在mysql下依次执行如下命令：
    </p>
    <pre><code>mysql&gt; use python_hospital;
mysql&gt; source D:/xxx/xxx/xxx.sql;
</code></pre>
    <p>
     (5) 启动django服务。在server目录下执行：
    </p>
    <pre><code>python manage.py runserver
</code></pre>
    <h5>
     <a id="_67">
     </a>
     前端运行步骤
    </h5>
    <p>
     (1) 安装node 16.14
    </p>
    <p>
     (2) 进入web目录下，安装依赖，执行:
    </p>
    <pre><code>npm install 
</code></pre>
    <p>
     (3) 运行项目
    </p>
    <pre><code>npm run dev
</code></pre>
