     题目：
    </h2>

    </p>
    <pre><code class="hljs">                &lt;!-- TODO: 待补充代码 --&gt;

    <p>
     题目要求：
    </p>
    <blockquote>
     <p>
      1. 将可供筛选的语言列表 `languages` 在 `select` 标签下的 `option` 元素进行渲染， **`option` 的 `value` 值必须对应绑定 `languages` 数组中的值**。效果如下：
     </p>
     <p>
      2. 完善 `changeHandle` 函数，当用户选择语言和输入显示位次的时候都会调用此函数。当用户改变 `select` 筛选器的选项或改变展示项目的位次输入时，根据用户的**选择**和**输入的位次**重新渲染图表。注意：**重新渲染图表必须通过修改 `xData` 和 `yData` 的值进行，不要修改变量名称**。
     </p>
    </blockquote>
    <p>
     答案：
    </p>
    <pre><code class="hljs">                &lt;!-- TODO: 待补充代码 --&gt;
                &lt;select name="language" id="language" @change="changeHandle" v-model="language"&gt;
                    &lt;option v-for="lang in languages" :value="lang" :index="lang"&gt;{<!-- -->{lang}}&lt;/option&gt;
                &lt;/select&gt;</code></pre>
    <pre><code class="hljs">            // 语言筛选改变时或页面数字输入框数字改变时的处理函数
            const changeHandle = () =&gt; {
                // TODO：待补充代码

                let newData = chartData.value.filter((item) =&gt; {
                    return item.language == language.value
                })
                if (language.value === 'All') newData = chartData.value;
                newData = newData.slice(pageStart.value - 1, pageEnd.value);
                xData.value = newData.map(item =&gt; item.name);
                yData.value = newData.map(item =&gt; item.stars);

                initChart();
            };</code></pre>

    <h2>
     拓展学习
    </h2>
    <h4>
     本题作者想说
    </h4>
    <p>
     目标1：
    </p>
    <ol>
     <li>
      <code>
       v-for="lang in languages"
      </code>
      ：这是一个Vue.js的循环指令，用于遍历
      <code>
       languages
      </code>
      数组，为数组中的每个元素生成一个
      <code>
       &lt;option&gt;
      </code>
      元素。
     </li>
     <li>
      <code>
       :value="lang"
      </code>
      ：将每个
      <code>
       &lt;option&gt;
      </code>
      元素的
      <code>
       value
      </code>
      属性设置为当前循环到的
      <code>
       lang
      </code>
      值。
     </li>
     <li>
      <code>
       :index="lang"
      </code>
      ：将每个
      <code>
       &lt;option&gt;
      </code>
      元素的
      <code>
       index
      </code>
      属性设置为当前循环到的
      <code>
       lang
      </code>
      值。注意，这里
      <code>
       :index
      </code>
      并不是Vue.js的标准属性，可能是自定义属性，用于在JavaScript中获取当前选项的索引。
     </li>
     <li>
      <code>
       {
       <!-- -->
       {lang}}
      </code>
      ：将当前循环到的
      <code>
       lang
      </code>
      值显示为
      <code>
       &lt;option&gt;
      </code>
      元素的文本内容。
     </li>
    </ol>
    <p>
     目标2：
    </p>
    <p>
     实现原理
    </p>
    <ol>
     <li>
      <p>
       <strong>
        过滤数据
       </strong>
       ：首先，通过
       <code>
        chartData.value.filter
       </code>
       方法，根据
       <code>
        language.value
       </code>
       的值来过滤
       <code>
        chartData
       </code>
       中的数据。如果
       <code>
        language.value
       </code>
       等于
       <code>
        'All'
       </code>
       ，则不过滤，直接使用所有数据。
      </p>
     </li>
     <li>
      <p>
       <strong>
        分页数据
       </strong>
       ：接着，使用
       <code>
        slice
       </code>
       方法对过滤后的数据进行分页处理。
       <code>
        pageStart.value - 1
       </code>
       和
       <code>
        pageEnd.value
       </code>
       定义了要显示的数据范围。
      </p>
     </li>
     <li>
      <p>
       <strong>
        更新图表数据
       </strong>
       ：通过
       <code>
        map
       </code>
       方法，将分页后的数据映射到
       <code>
        xData.value
       </code>
       和
       <code>
        yData.value
       </code>
       中，分别表示图表的横轴和纵轴数据。
      </p>
     </li>
     <li>
      <p>
       <strong>
        初始化图表
       </strong>
       ：最后，调用
       <code>
        initChart
       </code>
       函数来重新初始化图表，以反映最新的数据。
      </p>
     </li>
    </ol>
