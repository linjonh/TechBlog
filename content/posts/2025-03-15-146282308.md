    <h3>
     题目描述
    </h3>
    <p>
     小 Z 有一个大整数 s，整数 s 的位数最大可能有 100 位。现在小 Z 将要将这个整数 s 分成若干段，至少两段。
    </p>
    <p>
     问是否存在一种分段方法，可以使得分段后的每一段整数的各个数位之和相同。
    </p>
    <p>
     「注意」
    </p>
    <p>
     本题中的大整数 s 比较特殊，可能含有前导 0，例如输入有可能为 0134 这种整数。
    </p>
    <h3>
     输入
    </h3>
    <p>
     第一行输入测试数据组数t,t&gt;=1 且 t&lt;=10;
    </p>
    <p>
     每组数据输入有2行：
    </p>
    <p>
     每组数据第一行输入一个整数 n，表示大整数 s 的位数。2≤n≤100。
    </p>
    <p>
     每组数据第二行输入一个位数为 n 的大整数，即输入 n 个数字字符，每个数字字符都为 0 - 9 之间的字符，可能含有前导 0。
    </p>
    <h3>
     输出
    </h3>
    <p>
     输出t行
    </p>
    <p>
     每行输出 YES 或者 NO。
    </p>
    <h3>
     样例输入
    </h3>
    <pre><code class="language-cpp">3
5
73452
6
000000
3
232
</code></pre>
    <h3>
     样例输出
    </h3>
    <pre><code class="language-cpp">YES
YES
NO</code></pre>
    <p>
    </p>
    <p>
     示例代码：
    </p>
    <pre><code class="language-cpp">#include &lt;bits/stdc++.h&gt;
using namespace std;
typedef long long ll;
bool check(ll x,string b){
    ll s=0;
    for(int i=0;i&lt;b.size();i++){
        s=s+b[i]-'0';
        if(s==x) s=0; 
    } 
    if(s==0) return 1;
    else return 0;
} 
int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);cout.tie(0);
    ll t;
    cin&gt;&gt;t;
    while(t--){
        ll n,sum=0;
        cin&gt;&gt;n; 
        string a;
        cin&gt;&gt;a;
        for(int i=0;i&lt;a.size();i++){
            sum+=(a[i]-'0');
        }
        if(sum==0){
            cout&lt;&lt;"YES\n";continue;
        }
        bool v=0;
        for(int i=1;i&lt;sum;i++){
            if(sum%i==0){
                if(check(i,a)==1){
                    v=1;break;
                } 
            }
        } 
        if(v==0) cout&lt;&lt;"NO\n";
        else cout&lt;&lt;"YES\n";
    }
    return 0;
}</code></pre>
