     <p>
      做项目中的一些小经验，方便后续
     </p>
    </blockquote>
    <p>
     （1）前端代码中，请求的 URL 是通过查询参数（
     <code>
      ?id=${articleId}
     </code>
     ）传递的
    </p>
    <p>
     后端接口：
    </p>
    <pre><code class="language-java">@GetMapping("/knowledgeDetail")
public Result getKnowledgeByid(@RequestParam Integer id) {
    Knowledge knowledge = knowledgeService.getKnowledgeByid(id);
    return Result.success(knowledge);
}</code></pre>
    <p>
    </p>
    <p>
     (2) 前端代码这样传递参数
    </p>
    <pre><code>url: `${baseUrl}/knowledge/knowledgeDetail/${articleId}`</code></pre>
    <p>
     后端接口：
    </p>
    <pre><code class="language-java">@GetMapping("/knowledgeDetail/{id}")
public Result getKnowledgeByid(@PathVariable Integer id) {
    Knowledge knowledge = knowledgeService.getKnowledgeByid(id);
    if (knowledge == null) {
        return Result.error("Knowledge not found");
    }
    return Result.success(knowledge);
}</code></pre>
