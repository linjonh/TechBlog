# 测试docker里面的GPU是否可用

**背景：**
安装好docker和nvidia-docker以后，需要测试容器里面GPU是否可用：

命令1：

```shell
sudo docker run --runtime=nvidia --rm nvidia/cuda nvidia-smi

```

命令2：

```shell
sudo docker run --gpus all --rm nvidia/cuda nvidia-smi

```

有时候第一个命令不好使，可以测试一下第二个命令
