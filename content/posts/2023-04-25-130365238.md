    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        前言：
       </span>
      </span>
      <span style="color:#000000;">
       开放原子训练营旨在通过多种形式的技术培训，丰富开发者社区活动，扩大项目影响力，提高项目在开发者群体中的知名度。本次开放原子训练营，通过开展RT-Thread Nano动手学习活动，旨在帮助广大嵌入式应用开发者，特别是具有嵌入式应用基础，但需要增加OS应用开发经验的人员，快速地掌握开源IoT OS。
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        随着物联网行业的快速发展，越来越多的MCU或MPU厂商通过RTOS或中间件推动产品的销售，设备终端制造商通过RTOS提升工作效率加快产品迭代，云服务商通过提供RTOS为产品开发者提供更加便捷的云端-终端连接。在航空航天、医疗等高可靠领域，稳定的RTOS依然是不可或缺的部分。
       </span>
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        RT-Thread Nano
       </span>
      </span>
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        诞生于2006年，是国内以开源中立、 社区化发展起来的一款高可靠实时操作系统 ，由睿赛德科技负责开发维护和运营 。经过十五年的沉淀积累，专业化的运营推广，其高可靠性、高安全性、高伸缩性，以及丰富易用的中间组件等特性极大地满足了市场需求，被广泛应用于航空、电力、轨道交通、车载、工业自动化、消费电子等众多行业领域。
       </span>
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <img alt="" height="597" src="https://i-blog.csdnimg.cn/blog_migrate/0585eb3c8651f76fcc6c2737ffd57fbf.png" width="945"/>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        2023
       </span>
      </span>
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        年4月22日，开放原子第二季-RT-Thread Nano 动手学习营线下培训在北京圆满结束！
       </span>
      </span>
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        本次线下培训以摩斯密码为背景，通过对电报员生活的体验，使广大开发者感受摩尔斯密码里的故事，并且掌握RTOS 基础知识。
       </span>
      </span>
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        活动受到参会开发者的广泛好评，参会嘉宾一致反馈，讲师的讲解细致，内容难易适中，收获满满！
       </span>
      </span>
     </span>
    </p>
    <p style="margin-left:0;text-align:justify;">
     <span style="color:#333333;">
      <span style="background-color:#ffffff;">
       <span style="color:#000000;">
        活动开场，社区生态负责人介绍了社区生态的基本情况，包括RT-Thread的发展历程、RT-Thread优秀的软件生态以及丰富的社区活动等。在实践环节，工程师介绍了RTOS编程思想，指导开发者根据实验手册完成开发环境的搭建，并挑战摩尔斯密码的实现。针对动手实验环节开发者遇到的问题，讲师耐心地给予开发者一对一的指导，让开发者们能够更好地完成实验。
       </span>
      </span>
     </span>
    </p>
