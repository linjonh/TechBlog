     前言
    </h3>
    <p>

     <strong>
      API
     </strong>
    </h3>
    <p>
     <strong>
      APISpace
     </strong>
     的
     <strong>
      未来7天生活指数API
     </strong>
     ，支持国内3400+个城市以及国际4万个城市的天气指数数据，同时也支持国内任意经纬度查询，接口会返回该经纬度最近的天气指数数据，包括晨练、洗车、穿衣、感冒、运动、旅游、舒适度、紫外线、钓鱼、晾晒、过敏、啤酒（12项，有详细说明）。支持到未来七天。

    <h5>

     相关指数说明：
    </h5>
    <ul>
     <li>
      <strong>
       晨练指数:
      </strong>
      根据温度、湿度、风速等气象因素，判断是否适宜晨练，并提供穿衣建议。
     </li>
     <li>
      <strong>
       洗车指数:
      </strong>
      根据未来是否有降水、沙尘等天气情况，判断未来几天是否适宜洗车。
     </li>
     <li>
      <strong>
       穿衣指数:
      </strong>
      根据温度、湿度、风速等气象因素，推荐适宜的穿衣搭配。
     </li>
     <li>
      <strong>
       感冒指数:
      </strong>
      根据温度变化、湿度等气象因素，评估感冒发生的可能性，并提供预防建议。
     </li>
     <li>
      <strong>
       运动指数:
      </strong>
      根据温度、湿度、风速、空气质量等气象因素，判断是否适宜户外运动，并提供运动建议。
     </li>
     <li>
      <strong>
       旅游指数:
      </strong>
      综合天气、温度、湿度、风速、空气质量等因素，评估未来几天是否适宜旅游，并提供旅游建议。
     </li>
     <li>
      <strong>
       舒适度指数:
      </strong>
      根据温度、湿度、风速等气象因素，评估人体的舒适感受。
     </li>
     <li>
      <strong>
       紫外线指数:
      </strong>
      根据紫外线强度，提醒用户注意防晒。
     </li>
     <li>
      <strong>
       钓鱼指数:
      </strong>
      根据温度、气压、风速等气象因素，判断是否适宜钓鱼。
     </li>
     <li>
      <strong>
       晾晒指数:
      </strong>
      根据未来是否有降水、湿度等天气情况，判断未来几天是否适宜晾晒衣物。
     </li>
     <li>
      <strong>
       过敏指数:
      </strong>
      根据花粉浓度、空气质量等气象因素，评估过敏发生的可能性，并提供预防建议。
     </li>
     <li>
      <strong>
       啤酒指数:
      </strong>
      根据温度、湿度等气象因素，推荐适宜的啤酒类型。
     </li>
    </ul>
