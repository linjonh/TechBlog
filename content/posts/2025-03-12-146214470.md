    <h4>
     <a id="1__255">
     </a>
     1. 依赖下载失败排查步骤
    </h4>
    <ol>
     <li>
      检查网络连接与代理设置
     </li>
     <li>
      验证仓库镜像配置（settings.xml）
     </li>
     <li>
      清理本地仓库缓存：
      <code>
       mvn dependency:purge-local-repository
      </code>
     </li>
    </ol>
    <h4>
     <a id="2__260">
     </a>
     2. 构建速度优化方案
    </h4>
    <ul>
     <li>
      增加并行线程：
      <code>
       mvn -T 4 clean install
      </code>
     </li>
     <li>
      跳过测试：
      <code>
       -DskipTests
      </code>
     </li>
     <li>
      使用构建缓存工具：Gradle Build Cache（混合构建）
     </li>
    </ul>
