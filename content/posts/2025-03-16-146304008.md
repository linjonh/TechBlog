     </a>
     <code>
      update_plan_status()
     </code>
    </h5>
    <p>
     <strong>
      功能
     </strong>
     ：基于已完成的工具执行更新当前计划进度。
     <br/>
     <strong>
      实现细节
     </strong>
     ：
    </p>
    <ul>
     <li>
      检查工具调用是否在跟踪器中
     </li>
     <li>
      验证工具调用已成功完成
     </li>
     <li>
      执行规划工具的 “mark_step” 命令将步骤标记为已完成
     </li>

    <h5>
     <a id="_get_current_step_index_151">
     </a>

      _get_current_step_index()
     </code>
    </h5>
    <p>
     <strong>
      功能
     </strong>
     ：解析当前计划以识别第一个未完成步骤的索引。
     <br/>
     <strong>
      实现细节
     </strong>
     ：
    </p>
    <ul>
     <li>
      获取当前计划
     </li>
     <li>
      查找 “Steps:” 行的索引
     </li>
     <li>
      查找第一个未完成或进行中的步骤
     </li>
     <li>
      将当前步骤标记为进行中
     </li>
    </ul>

     </a>
     <code>
      think()
     </code>
    </h5>
    <p>
     <strong>
      功能
     </strong>
     ：处理当前状态并决定下一步行动。
     <br/>
     <strong>
      实现细节
     </strong>
     ：
    </p>
    <ul>
     <li>
      更新当前工作目录
     </li>
     <li>
      使用当前工作目录格式化
      <code>
       next_step_prompt
      </code>
     </li>
     <li>
      调用父类的
      <code>
       think()
      </code>
      方法
     </li>
    </ul>
