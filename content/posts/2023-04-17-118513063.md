    <p>
     （ ）是存储在计算机内有结构的数据的集合。
     <br/>
     A. 数据库系统
     <br/>
     B. 数据库
     <br/>
     C. 数据库管理系统
     <br/>
     D. 数据结构
     <br/>
     正确答案： B
    </p>
    <blockquote>
     <p>
      数据库是存储在计算机内有结构的数据的集合，数据库管理系统是管理数据库的软件，数据库系统包含数据库、DBMS和DB。
     </p>
    </blockquote>
    <p>
     数据库(DB)、数据库系统(DBS)和数据库管理系统(DBMS)三者之间的关系是（ ） 。
     <br/>
     A. DBS包括DB和DBMS
     <br/>
     B. DBMS包括DB和DBS
     <br/>
     C. DB包括DBS和DBMS
     <br/>
     D. DBS就是DB，也就是DBMS
     <br/>
     正确答案：A
    </p>
    <blockquote>
     <p>
      DBMS是管理数据库的一个系统软件，而DBS包含DBMS、DB和数据库管理员等
     </p>
    </blockquote>
    <p>
     在数据管理中，同一数据重复存储的现象，称为（ ）
     <br/>
     A. 不一致
     <br/>
     B. 数据冗余
     <br/>
     C. 规范化
     <br/>
     D. 异常
     <br/>
     正确答案： B
    </p>
    <p>
     数据冗余是指（ ）。
     <br/>
     A. 数据与数据之间没有联系
     <br/>
     B. 数据有丢失
     <br/>
     C. 数据量太大
     <br/>
     D. 存在重复数据
     <br/>
     正确答案： D
    </p>
    <p>
     在数据管理技术的发展过程中，经历了人工管理阶段、文件系统阶段和数据库系统阶段。在这几个阶段中，数据独立性最高的是（ ）阶段。
     <br/>
     A. 数据库系统
     <br/>
     B. 文件系统
     <br/>
     C. 人工管理
     <br/>
     D. 数据项管理
     <br/>
     正确答案： A
    </p>
    <blockquote>
     <p>
      数据独立性是指数据结构有变化是否需要修改程序。人工管理和文件管理阶段数据结构有变化，程序必须修改，数据独立性很差。
     </p>
    </blockquote>
    <p>
     数据库系统与文件系统的主要区别是（ ） 。
     <br/>
     A. 数据库系统复杂，而文件系统简单
     <br/>
     B. 文件系统不能解决数据冗余和数据独立性问题，而数据库系统可以解决
     <br/>
     C. 文件系统只能管理程序文件，而数据库系统能够管理各种类型的文件
     <br/>
     D. 文件系统管理的数据量较少，而数据库系统可以管理庞大的数据量
     <br/>
     正确答案：B
    </p>
    <blockquote>
     <p>
      文件系统也能管理庞大数据量的数据，文件系统适合管理定长的、结构固定，而且不需要共享的数据。数据库管理比文件管理最大的优势是可数据共享，减少数据冗余和数据独立性高。
     </p>
    </blockquote>
    <p>
     数据库的基本特点是（ ） 。
     <br/>
     A. (1)数据可以共享(或数据结构化) (2)数据独立性
     <br/>
     (3)数据冗余大，易移植 (4)统一管理和控制
     <br/>
     B. (1)数据可以共享(或数据结构化) (2)数据独立性
     <br/>
     (3)数据冗余小，易扩充 (4)统一管理和控制
     <br/>
     C. (1)数据可以共享(或数据结构化) (2)数据互换性
     <br/>
     (3)数据冗余小，易扩充 (4)统一管理和控制
     <br/>
     D. (1)数据非结构化 (2)数据独立性
     <br/>
     (3)数据冗余小，易扩充 (4)统一管理和控制
     <br/>
     正确答案：B
    </p>
    <blockquote>
     <p>
      数据库系统的特点是：数据结构化；数据的共享性高，冗余度低，易扩充；数据独立性高；数据由DBMS统一管理和控制。
     </p>
    </blockquote>
    <p>
     在数据库中，下列说法（ ）是不正确的。
     <br/>
     A. 数据库避免了一切数据的重复
     <br/>
     B. 若系统是完全可以控制的，则系统可确保更新时的一致性
     <br/>
     C. 数据库中的数据可以共享
     <br/>
     D. 数据库减少了数据冗余
     <br/>
     正确答案：A
    </p>
    <blockquote>
     <p>
      数据库减少了数据冗余，但并不是没有冗余，表与表之间还需要靠相同的数据来关联。可以说数据库减少了不必要的重复数据，但有些重复数据是必须的。
     </p>
    </blockquote>
    <p>
     下述关于数据库系统的正确叙述是（ ）。
     <br/>
     A. 数据库系统减少了数据冗余
     <br/>
     B. 数据库系统避免了一切冗余
     <br/>
     C. 数据库系统中数据的一致性是指数据类型一致
     <br/>
     D. 数据库系统比文件系统能管理更多的数据
     <br/>
     正确答案：A
    </p>
    <blockquote>
     <p>
      数据库减少了数据冗余，但并不是没有冗余，表与表之间还需要靠相同的数据来关联，有些重复数据是必须的。
     </p>
    </blockquote>
    <p>
     下述关于数据库系统的正确叙述是（ ）。
     <br/>
     A. 数据库中只存在数据项之间的联系
     <br/>
     B. 数据库的数据项之间和记录之间都存在联系
     <br/>
     C. 数据库的数据项之间无联系，记录之间存在联系
     <br/>
     D. 数据库的数据项之间和记录之间都不存在联系
     <br/>
     正确答案： B
    </p>
    <blockquote>
     <p>
      数据库可以实现整体结构化，不仅能表示数据项之间的联系，也能表示记录间的联系。
     </p>
    </blockquote>
    <p>
     数据库管理系统(DBMS)最主要功能是（ ）。
     <br/>
     A. 修改数据库
     <br/>
     B. 定义数据库
     <br/>
     C. 应用数据库
     <br/>
     D. 保护数据库
     <br/>
     正确答案： B
    </p>
    <blockquote>
     <p>
      数据库管理系统最主要的功能是定义数据库
     </p>
    </blockquote>
    <p>
     数据库管理系统的工作不包括（ ）。
     <br/>
     A. 定义数据库
     <br/>
     B. 对已定义的数据库进行管理
     <br/>
     C. 为定义的数据库提供操作系统
     <br/>
     D. 数据通信
     <br/>
     正确答案： C
    </p>
    <blockquote>
     <p>
      数据库管理系统是介于用户和操作系统之间的系统软件，专门管理数据库。
     </p>
    </blockquote>
    <p>
     数据库系统的特点是（ ）、数据独立、减少数据冗余、避免数据不一致和加强了数据保护。
     <br/>
     A. 数据共享
     <br/>
     B. 数据存储
     <br/>
     C. 数据应用
     <br/>
     D. 数据保密
     <br/>
     正确答案： A
    </p>
    <blockquote>
     <p>
      数据库系统的特点是：数据的共享性高，冗余度低，易扩充；数据结构化；数据独立性高；数据由DBMS统一管理和控制。
     </p>
    </blockquote>
    <p>
     数据的管理方法主要有（ ）。
     <br/>
     A. 批处理和文件系统
     <br/>
     B. 文件系统和分布式系统
     <br/>
     C. 分布式系统和批处理
     <br/>
     D. 数据库系统和文件系统
     <br/>
     正确答案： D
    </p>
    <blockquote>
     <p>
      目前数据管理方法主要有数据库系统和文件系统，人工管理已经很少使用。
     </p>
    </blockquote>
    <p>
     数据库管理系统是（ ）。
     <br/>
     A. 操作系统的一部分
     <br/>
     B. 在操作系统支持下的系统软件
     <br/>
     C. 一种编译程序
     <br/>
     D. 一种操作系统
     <br/>
     正确答案： B
    </p>
    <blockquote>
     <p>
      数据库管理系统是介于用户和操作系统之间的系统软件，专门管理数据库。
     </p>
    </blockquote>
    <p>
     数据库的特点之一是数据的共享，严格地讲，这里的数据共享是指（ ）。
     <br/>
     A. 同一个应用中的多个程序共享一个数据集合
     <br/>
     B. 多个用户、同一种语言共享数据
     <br/>
     C. 多个用户共享一个数据文件
     <br/>
     D. 多种应用、多种语言、多个用户相互覆盖地使用数据集合
     <br/>
     正确答案：D
    </p>
    <blockquote>
     <p>
      数据库中的数据可以共享给多种应用、多个用户，使用不同的编程语言来访问。
     </p>
    </blockquote>
    <p>
     （ ）可以减少相同数据重复存储的现象。
     <br/>
     A. 记录
     <br/>
     B. 字段
     <br/>
     C. 文件
     <br/>
     D. 数据库
     <br/>
     正确答案： D
    </p>
    <blockquote>
     <p>
      相同数据重复存储就是指冗余数据多，使用数据库可以可以共享数据，减少了数据冗余。
     </p>
    </blockquote>
