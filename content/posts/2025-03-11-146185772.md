    <h3>
     <a id="_0">
     </a>
     问题与处理策略
    </h3>
    <h5>
     <a id="_2">
     </a>
     问题描述
    </h5>
    <ul>
     <li>
      在微信小程序项目中，通过 require 引入图片文件，报如下错误
     </li>
    </ul>
    <pre><code>Error: module 'assets/img/topImg.jpg.js' is not defined, 
require args is '../../assets/img/topImg.jpg'
</code></pre>
    <pre><code># 翻译

错误：未定义模块“assets/img/topImg.jpg.js”
require 参数为“../../assets/img/topImg.jpg”
</code></pre>
    <h5>
     <a id="_18">
     </a>
     问题原因
    </h5>
    <ol>
     <li>
      <p>
       在微信小程序中，不支持直接使用 require 引入图片文件，例如，jpg、png 等
      </p>
     </li>
     <li>
      <p>
       微信小程序的模块系统不支持将图片作为模块加载，这个报错是系统尝试将图片文件当作 JavaScript 模块来加载，但失败了
      </p>
     </li>
    </ol>
    <h5>
     <a id="_24">
     </a>
     处理策略
    </h5>
    <h6>
     <a id="1_WXML__26">
     </a>
     （1）在 WXML 中使用
    </h6>
    <ul>
     <li>
      在 WXML 中直接使用图片路径
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>image</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>../../assets/img/topImg.jpg<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>image</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h6>
     <a id="2_JS__34">
     </a>
     （2）在 JS 中使用
    </h6>
    <ul>
     <li>
      在 WXML 中直接使用图片路径，这样可以动态设置图片路径
     </li>
    </ul>
    <pre><code class="prism language-javascript"><span class="token function">Page</span><span class="token punctuation">(</span><span class="token punctuation">{<!-- --></span>
