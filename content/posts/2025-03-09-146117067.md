    <h3>
     <a id="_105">
     </a>
     题解
    </h3>
    <p>
     <font size="3">
      1. 函数头：
      <br/>
      TreeNode* dfs(root)
      <br/>
      2. 函数体：
      <br/>
      后序遍历，先处理左子树，再处理右子树，最后判断根是否为0
      <br/>
      <strong>
       子问题：如果左子树为空，右子树为空，根的值是0，可以进行剪枝，然后返回空节点
      </strong>
      <br/>
      3.递归的出口：
      <br/>
      root == nullptr return nullptr
      <br/>
      子问题可以做为题目的突破口
     </font>
    </p>

    <h3>
     <a id="_159">

     题解
    </h3>
    <p>
     <font size="3">
      1.
      <strong>

      </strong>
      ，
      <font color="Red">

      </font>

      <font color="Red">

      <font color="Blue">
       4. 这题利用中序遍历和一个全局变量和剪枝来完成，保证左右子树和根都是二叉搜索树
      </font>
     </font>
    </p>

     二叉树的所有路径
    </h2>
    <p>
     <a href="https://leetcode.cn/problems/binary-tree-paths/" rel="nofollow">
      题目链接
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/56641aac5f5f427d8ca7578164d2d477.png"/>

    <h3>
     <a id="_248">
     </a>
     题解
    </h3>
    <p>
     <font size="3">
      1. 函数头：void dfs(root,string)
      <br/>
      字符串用来每次存路径
      <br/>
      2. 函数体：如果是叶子节点就把前面的路径加入到数组中，然后返回，如果是非叶子节点就递归左子树和右子树找出所有的路径
      <br/>
      3. 递归出口：root == nullptr return;
     </font>
     <br/>
     <font size="3">
      <font color="Red">
       4. 回溯：需要恢复现场，比如所有路径那题，如果是
       <strong>
        全局变量
       </strong>
       的string就需要
       <strong>
        恢复现场
       </strong>
       ，避免后面的加入的数字影响前面的，叶子节点恢复现场是去掉数字，非叶子节点恢复现场是去掉数字和箭头，用全局的恢复现场比较麻烦，所以使用了传参的string，每次函数都是自动恢复现场，因为是局部的变量，每次string都是上次的模样
       <br/>
       5.剪枝：上面这题就不需要再写root == nullptr的情况返回了，因为左右子树递归那里写了不等于空才去递归，相当于等于空的那部分递归剪掉了
       <br/>
      </font>
     </font>
    </p>
