# jsp页面获取后端返回的request.setAttribute

一不小心遇到个不分离的项目,然后后端给我一个编码,返回参数: request.setAttribute(“xx\_no”,xx\_no);
  
这时候可以:

```
<input type="hidden" value="${xx_no}" id="xx_Id" />

var  number  = $('#xx_Id').val();

```
