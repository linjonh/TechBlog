    <h2>
     1、题目描述
    </h2>
    <p>
     给你一个字符串
     <code>
      s
     </code>
     ，请你将
     <em>
     </em>
     <code>
      s
     </code>
     <em>
     </em>
     分割成一些 子串，使每个子串都是
     <strong>
      回文串
     </strong>
     。返回
     <code>
      s
     </code>
     所有可能的分割方案。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <pre><strong>输入：</strong>s = "aab"
<strong>输出：</strong>[["a","a","b"],["aa","b"]]</pre>
    <h2>
     2、初始思路
    </h2>
    <h3>
     2.1 思路
    </h3>
    <p>
     本题也是一道经典的回溯题，首先可以给出本题的树状图：
    </p>
    <p>
     <img alt="" height="1141" src="https://i-blog.csdnimg.cn/direct/2cb939f564534363b759a913d2a791ca.jpeg" width="1116"/>
    </p>
    <p>
     按照前文给出的回溯模板进行分析：
    </p>
    <p>
     1、终止条件：当切割到词尾时，不能够再进行切割，此时结束；
    </p>
    <pre><code>if startIndex == len(s):
    res.append(path.copy())
    return</code></pre>
    <p>
     2、本文的关键是要将字符串分割为都为回文的子串，因此对子串的回文判断也很重要，可以使用反转进行判断：
    </p>
    <pre><code>if substr == substr[::-1]:</code></pre>
    <p>
     3、回溯的参数：首先，字符串s为一个参数；其次，判断切割位置的startIndex可作为另一个参数：
    </p>
    <pre><code>def backtracking(s,startIndex):
    backtracking(s,i+1)</code></pre>
    <h3>
     2.2 代码
    </h3>
    <p>
     根据上述分析，可以得到本题的完整代码：
    </p>
    <pre><code>class Solution:
    def partition(self, s: str) -&gt; List[List[str]]:
        res = []
        path = []
        def backtracking(s,startIndex):
            #切到字符串结尾结束
            if startIndex == len(s):
                res.append(path.copy())
                return
            for i in range(startIndex, len(s)):
                substr = s[startIndex:i+1]
                print("substr:",substr)
                if substr == substr[::-1]:
                    path.append(substr)
                    print("path:",path)
                    backtracking(s,i+1)
                    path.pop()
        backtracking(s,0)
        return res</code></pre>
