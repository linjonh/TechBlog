    <h4>
     页面配置
    </h4>
    <p>
     小程序的配置可以配置页面路径、窗口表现、tabBar等，分为全局配置和页面配置，全局配置针对所有页面生效，页面配置只针对当前页生效。
    </p>
    <h5>
     <a name="t8">
     </a>
     全局配置 (app.json)
    </h5>
    <p>
     <strong>
      (1) 路径配置
     </strong>
    </p>
    <p>
     <code>
      pages
     </code>
     配置页面路径，未配置路径的页面无法被访问到
    </p>
    <p>
     <strong>
      技巧 :
     </strong>
     创建一个页面文件时, 在pages中添加一个配置项, 小程序开发工具自动把文件创建出来,不需要手动创建
    </p>
    <pre><code>//app.json 中
{
   "pages": [
    "pages/index/index",
    "pages/home/home",
    "pages/info/info",
  ],
}
</code></pre>
    <p>
     <strong>
      (2) 起始页配置
     </strong>
    </p>
    <p>
     <code>
      entryPath
     </code>
     配置小程序默认的启动页面（首页） 如果不填，将默认为
     <code>
      pages
     </code>
     列表的第一项
    </p>
    <pre><code> {
   "entryPagePath": "pages/home/home",
 }
</code></pre>
    <p>
     <strong>
      (3) window窗口配置
     </strong>
    </p>
    <p>
     <code>
      window
     </code>
     用于设置小程序的状态栏、导航条、标题、窗口背景色。
    </p>
    <pre><code> "window": {
    "enablePullDownRefresh": true,
     
    //窗口配置
    "backgroundColor": "#eee",
    "backgroundTextStyle": "dark",
     
    //导航栏的配置  
    "navigationBarBackgroundColor": "#FFA500", 
    "navigationBarTextStyle": "white",
    "navigationBarTitleText": "我的小程序",
    "navigationStyle": "default"
  },
</code></pre>
    <p>
     <strong>
      窗口的配置
     </strong>
    </p>
    <ul>
     <li>
      <code>
       backgroundColor
      </code>
      窗口的背景色
     </li>
    </ul>
    <p>
     <strong>
      导航栏的配置
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <code>
        navigationBarBackgroundColor
       </code>
       导航栏背景颜色 (颜色是十六进制)
      </p>
     </li>
     <li>
      <p>
       <code>
        navigationBarTextStyle
       </code>
       导航栏标题颜色，仅支持
       <code>
        black
       </code>
       /
       <code>
        white
       </code>
      </p>
     </li>
     <li>
      <p>
       <code>
        navigationBarTitleText
       </code>
       导航栏标题文字内容
      </p>
     </li>
     <li>
      <p>
       <code>
        navigationStyle
       </code>
       导航栏样式，仅支持以下值:
       <code>
        default
       </code>
       默认样式
       <code>
        custom
       </code>
       自定义导航栏
      </p>
     </li>
    </ul>
    <p>
     <strong>
      下拉刷新的配置
     </strong>
    </p>
    <ul>
     <li>
      <code>
       backgroundTextStyle
      </code>
      下拉 loading 的样式，仅支持
      <code>
       dark
      </code>
      /
      <code>
       light
      </code>
     </li>
     <li>
      <code>
       enablePullDownRefresh
      </code>
      是否开启全局的下拉刷新。
     </li>
    </ul>
    <p>
     <strong>
      (4) tabBar 配置
     </strong>
    </p>
    <p>
     <code>
      tabBar
     </code>
     底部 Tab 栏（tabBar 中的页面必须配置在
     <code>
      pages
     </code>
     中）
    </p>
    <pre><code> "tabBar": {
    "color": "#999",
    "selectedColor": "#5792AF",
    "backgroundColor": "#e0e0e0",
    "borderStyle": "white",
    "position": "bottom",
    "list": [
      {
        "pagePath": "pages/home/home",
        "text": "我的",
        "iconPath": "/assets/icon/home.png",
        "selectedIconPath": "/assets/icon/home-active.png"
      },
    ]
  },
</code></pre>
    <ul>
     <li>
      <code>
       color
      </code>
      文字颜色，仅支持十六进制颜色
     </li>
     <li>
      <code>
       selectedColor
      </code>
      文字选中时的颜色，仅支持十六进制颜色
     </li>
     <li>
      <code>
       backgroundColor
      </code>
      tab 的背景色，仅支持十六进制颜色
     </li>
     <li>
      <code>
       borderStyle
      </code>
      tabbar上边框的颜色 , 仅支持
      <code>
       black
      </code>
      /
      <code>
       white
      </code>
     </li>
     <li>
      <code>
       list
      </code>
      tabBar 页面跳转的
      <strong>
       路径
      </strong>
      和
      <strong>
       图标
      </strong>
      配置
     </li>
     <li>
      <code>
       pagePath
      </code>
      跳转的页面路径
     </li>
     <li>
      <code>
       text
      </code>
      tabBar 上按钮文字
     </li>
     <li>
      <code>
       iconPath
      </code>
      图片路径 icon 大小限制为 40kb，不支持网络图片。
      <strong>
       当
       <code>
        position
       </code>
       为
       <code>
        top
       </code>
       时，不显示 icon。
      </strong>
     </li>
     <li>
      <code>
       selectedIconPath
      </code>
      选中时的图片路径
     </li>
     <li>
      <code>
       position
      </code>
      tabBar 的位置，仅支持
      <code>
       bottom
      </code>
      /
      <code>
       top
      </code>
     </li>
    </ul>
    <h5>
     <a name="t9">
     </a>
     页面配置
    </h5>
    <p>
     页面配置大部分与全局配置中的
     <code>
      window
     </code>
     是重叠的，页面配置的优先级大于全局配置的。
    </p>
    <ul>
     <li>
      <code>
       disableScroll
      </code>
      只能用在在页面配置中，禁用页面滚动
     </li>
     <li>
      <code>
       usingComponents
      </code>
      后续学到自定义组件时再介绍
     </li>
    </ul>
