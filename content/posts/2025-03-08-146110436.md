    <h1 class="title-article" id="articleContentId">
     LabVIEW基于双通道FFT共轭相乘的噪声抑制
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     对于双通道采集的含噪信号，通过FFT获取复数频谱后，对第二通道频谱取共轭并与第一通道频谱相乘，理论上可增强相关信号成分并抑制非相关噪声。此方法适用于
     <strong>
      通道间信号高度相关、噪声独立
     </strong>
     的场景（如共模干扰抑制）。以下为LabVIEW实现方案及案例验证。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="478" id="0.5761389087223558" src="https://i-blog.csdnimg.cn/img_convert/2b6c1d0eb9e84bb8e76f3dc1c7152973.png" width="1073"/>
    </p>
    <p>
     ​
    </p>
    <hr/>
    <h4>
     <strong>
      实现原理与步骤
     </strong>
    </h4>
    <h5>
     <strong>
      1. 核心数学推导
     </strong>
    </h5>
    <p>
     设两通道信号为：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        通道1
       </strong>
       ：
       <code>
        S1(t) = Signal(t) + Noise1(t)
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        通道2
       </strong>
       ：
       <code>
        S2(t) = Signal(t) + Noise2(t)
       </code>
      </p>
     </li>
    </ul>
    <p>
     假设信号
     <code>
      Signal(t)
     </code>
     完全相关，噪声
     <code>
      Noise1
     </code>
     与
     <code>
      Noise2
     </code>
     独立。在频域中：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        FFT(S1)
       </strong>
       ：
       <code>
        A + jW1 = FFT(Signal) + FFT(Noise1)
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        FFT(S2)
       </strong>
       ：
       <code>
        B + jW2 = FFT(Signal) + FFT(Noise2)
       </code>
      </p>
     </li>
    </ul>
    <p>
     对通道2频谱取共轭后相乘：
    </p>
    <p>
     (A + jW1) × (B - jW2) = AB + W1W2 + j(BW1 - AW2)
    </p>
    <p>
     若信号成分占主导（A≈B, W1≈W2），则实部增强（AB + W1W2 ≈ A² + W1²），虚部噪声项（BW1 - AW2）因噪声不相关趋于抵消。最终通过逆FFT恢复时域信号，噪声被部分抑制。
    </p>
    <h5>
     <strong>
      2. LabVIEW实现流程
     </strong>
    </h5>
    <p>
     <strong>
      关键VI
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       <code>
        FFT.vi
       </code>
       （信号处理→变换→FFT）
      </p>
     </li>
     <li>
      <p>
       <code>
        Complex Conjugate.vi
       </code>
       （数学→复数→复共轭）
      </p>
     </li>
     <li>
      <p>
       <code>
        Multiply.vi
       </code>
       （数值→乘）
      </p>
     </li>
     <li>
      <p>
       <code>
        Inverse FFT.vi
       </code>
       （信号处理→变换→逆FFT）
      </p>
     </li>
    </ul>
    <p>
     <strong>
      代码步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        信号采集与预处理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         双通道同步采集信号（如DAQmx读取）。
        </p>
       </li>
       <li>
        <p>
         加窗处理（推荐汉宁窗，减少频谱泄漏）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        FFT变换
       </strong>
      </p>
      <p>
       <strong>
        (A + jW1) × (B - jW2) = AB + W1W2 + j(BW1 - AW2)
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        共轭相乘处理
       </strong>
      </p>
      <p>
       B+jW2 → Complex Conjugate.vi → B-jW2   A+jW1 × B-jW2 → Multiply.vi → 乘积频谱
      </p>
     </li>
     <li>
      <p>
       <strong>
        逆FFT恢复时域信号
       </strong>
      </p>
      <p>
       <strong>
        乘积频谱 → Inverse FFT.vi → 降噪后时域信号
       </strong>
      </p>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      3. 应用案例：工业电机振动信号降噪
     </strong>
    </h4>
    <h5>
     <strong>
      场景需求
     </strong>
    </h5>
    <p>
     某电机振动监测系统中，两加速度传感器对称安装，采集轴承振动信号。目标：抑制环境噪声，提取真实振动特征频率。
    </p>
    <h5>
     <strong>
      实现细节
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        信号生成（仿真）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          信号成分
         </strong>
         ：10 Hz基频 + 50 Hz谐波，幅值1 V。
        </p>
       </li>
       <li>
        <p>
         <strong>
          噪声
         </strong>
         ：通道1加入20-100 Hz高斯白噪声（SNR=10 dB），通道2加入相同频带但独立生成的噪声。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        LabVIEW代码设计
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          FFT设置
         </strong>
         ：采样率1 kHz，FFT点数1024，汉宁窗。
        </p>
       </li>
       <li>
        <p>
         <strong>
          频域处理
         </strong>
         ：共轭相乘后，取乘积频谱的模值开方作为最终幅值（避免能量倍增）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        结果对比
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          原始信号
         </strong>
         ：频谱中10 Hz与50 Hz被噪声淹没（峰值不明显）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          处理后信号
         </strong>
         ：10 Hz与50 Hz幅值清晰，背景噪声降低约6 dB。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      4. 注意事项与优化
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        通道同步性
       </strong>
       ：确保两通道严格同步采样，避免时延导致相位偏差。
      </p>
     </li>
     <li>
      <p>
       <strong>
        频谱对称性
       </strong>
       ：仅处理FFT结果的前半部分（0~Nyquist频率），避免重复计算。
      </p>
     </li>
     <li>
      <p>
       <strong>
        幅值修正
       </strong>
       ：逆FFT后需缩放幅值（除以FFT点数N），恢复物理单位。
      </p>
     </li>
     <li>
      <p>
       <strong>
        窗函数补偿
       </strong>
       ：若加窗处理，需在频域幅值中除以窗函数的相干增益（如汉宁窗为0.5）。
      </p>
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      总结
     </strong>
    </h4>
    <p>
     通过双通道FFT共轭相乘处理，LabVIEW可有效抑制独立噪声并增强相关信号。此方法适用于
     <strong>
      对称传感器布局
     </strong>
     或
     <strong>
      共模干扰场景
     </strong>
     （如工业振动监测、声学阵列），但需注意通道一致性校准与频谱对称性处理。实际测试表明，在SNR≥10 dB时，信噪比可提升3-8 dB，显著改善特征频率识别能力。
    </p>
