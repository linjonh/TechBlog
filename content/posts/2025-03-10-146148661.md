    <h2>
     <a id="Git__0">
     </a>
     Git 设置全局代理或项目代理
    </h2>
    <p>
     <code>
      git config
     </code>
     ：
    </p>
    <h4>
     <a id="git_2">
     </a>
     全局配置，设置git代理服务器
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 设置 HTTP 代理</span>
<span class="token function">git</span> config <span class="token parameter variable">--global</span> http.proxy http://127.0.0.1:7897

<span class="token comment"># 设置 HTTPS 代理</span>
<span class="token function">git</span> config <span class="token parameter variable">--global</span> https.proxy http://127.0.0.1:7897

<span class="token comment"># 设置所有协议的代理（如果需要）</span>
<span class="token function">git</span> config <span class="token parameter variable">--global</span> core.gitProxy <span class="token string">'socks5://127.0.0.1:7897'</span>
</code></pre>
    <h4>
     <a id="_global_14">
     </a>
     如果想仅仅在当前项目配置，需要去掉
     <code>
      --global
     </code>
     ：
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 设置 HTTP 代理（仅当前项目）</span>
<span class="token function">git</span> config http.proxy http://127.0.0.1:7897

<span class="token comment"># 设置 HTTPS 代理（仅当前项目）</span>
<span class="token function">git</span> config https.proxy http://127.0.0.1:7897

<span class="token comment"># 设置所有协议的代理（仅当前项目）</span>
<span class="token function">git</span> config core.gitProxy <span class="token string">'socks5://127.0.0.1:7897'</span>
</code></pre>
    <h4>
     <a id="_26">
     </a>
     测试延迟：
    </h4>
    <pre><code class="prism language-bash"><span class="token function">ping</span> <span class="token parameter variable">-c</span> <span class="token number">5</span> github.com
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/552a976cad0d474a82fb493dfec0fdb0.png"/>
    </p>
    <h2>
     <a id="_33">
     </a>
     取消设置
    </h2>
    <pre><code class="prism language-bash"><span class="token function">git</span> config <span class="token parameter variable">--global</span> --unset-all http.proxy
<span class="token function">git</span> config <span class="token parameter variable">--global</span> --unset-all https.proxy
</code></pre>
