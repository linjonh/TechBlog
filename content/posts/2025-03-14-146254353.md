    <p>
     <strong>
      图（Graph）
     </strong>
     是一种非线性数据结构，用于表示对象之间的关系。图由
     <strong>
      顶点（Vertex）
     </strong>
     和
     <strong>
      边（Edge）
     </strong>
     组成，其中顶点表示对象，边表示对象之间的关系。图广泛应用于计算机科学、数学、物理、生物、社交网络等领域。
     <br/>
    </p>
    <p>
    </p>
    <h2>
     <a id="1__2">
     </a>
     1. 图的基本概念
    </h2>
    <ul>
     <li>
      <strong>
       顶点（Vertex）
      </strong>
      ：也称为
      <strong>
       节点（Node）
      </strong>
      ，表示图中的对象。例如，在社交网络中，顶点可以表示人。
     </li>
     <li>
      <strong>
       边（Edge）
      </strong>
      ：表示顶点之间的关系。例如，在社交网络中，边可以表示两个人是朋友。
     </li>
     <li>
      <strong>
       有向图（Directed Graph）
      </strong>
      ：边有方向，表示从一个顶点指向另一个顶点。例如，A → B 表示从 A 到 B 的关系。
     </li>
     <li>
      <strong>
       无向图（Undirected Graph）
      </strong>
      ：边没有方向，表示两个顶点之间的双向关系。例如，A — B 表示 A 和 B 是相互关联的。
     </li>
     <li>
      <strong>
       权重（Weight）
      </strong>
      ：边可以带有权重，表示关系的强度或成本。例如，在地图中，边的权重可以表示两个城市之间的距离。
     </li>
    </ul>

    <h3>
     <a id="_16">
     </a>
     按边是否有权重
    </h3>
    <ul>
     <li>
      <strong>
       带权图（Weighted Graph）
      </strong>
      ：
      <ul>
       <li>
        边带有权重，表示关系的强度或成本。
       </li>
       <li>
        示例：地图（边的权重表示两个城市之间的距离）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       无权图（Unweighted Graph）
      </strong>
      ：
      <ul>
       <li>
        边没有权重，只表示顶点之间是否存在关系。
       </li>
       <li>
        示例：社交网络（只表示两个人是否是朋友）。
       </li>
      </ul>
     </li>
    </ul>

    <h3>
     <a id="_30">
     </a>
     按图的连通性
    </h3>
    <ul>
     <li>
      <strong>
       连通图（Connected Graph）
      </strong>
      ：
      <ul>
       <li>
        无向图中，任意两个顶点之间都存在路径。
       </li>
       <li>
        示例：完全连通的社交网络。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       非连通图（Disconnected Graph）
      </strong>
      ：
      <ul>
       <li>
        无向图中，存在至少两个顶点之间没有路径。
       </li>
       <li>
        示例：孤立的社交网络群体。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       强连通图（Strongly Connected Graph）
      </strong>
      ：
      <ul>
       <li>
        有向图中，任意两个顶点之间都存在双向路径。
       </li>
       <li>
        示例：完全连通的网页链接图。
       </li>
      </ul>
     </li>
    </ul>
    <h2>
     <a id="3__40">
     </a>
     3. 图的表示方法
    </h2>
    <p>
     图可以通过多种方式表示，常见的有：
    </p>
    <ul>
     <li>
      <strong>
       邻接矩阵（Adjacency Matrix）
      </strong>
      ：
      <ul>
       <li>
        使用二维数组表示顶点之间的连接关系。
       </li>
       <li>
        适合稠密图。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       邻接表（Adjacency List）
      </strong>
      ：
      <ul>
       <li>
        使用数组或链表存储每个顶点的邻接顶点。
       </li>
       <li>
        适合稀疏图。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       边列表（Edge List）
      </strong>
      ：
      <ul>
       <li>
        直接存储所有边的列表。
       </li>
       <li>
        适合某些特定算法（如 Kruskal 算法）。
       </li>
      </ul>
     </li>
    </ul>
    <h2>
     <a id="4__51">
     </a>
     4. 图的算法
    </h2>
    <p>
     图论中有许多经典算法，例如：
    </p>
    <ul>
     <li>
      <strong>
       遍历算法
      </strong>
      ：
      <ul>
       <li>
        <strong>
         深度优先搜索（DFS）
        </strong>
        ：用于遍历或搜索图。
       </li>
       <li>
        <strong>
         广度优先搜索（BFS）
        </strong>
        ：用于最短路径问题。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       最短路径算法
      </strong>
      ：
      <ul>
       <li>
        <strong>
         Dijkstra 算法
        </strong>
        ：用于带权图的最短路径。
       </li>
       <li>
        <strong>
         Floyd-Warshall 算法
        </strong>
        ：用于所有顶点对之间的最短路径。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       最小生成树算法
      </strong>
      ：
      <ul>
       <li>
        <strong>
         Kruskal 算法
        </strong>
        ：基于边列表的最小生成树。
       </li>
       <li>
        <strong>
         Prim 算法
        </strong>
        ：基于顶点的最小生成树。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       拓扑排序
      </strong>
      ：
      <ul>
       <li>
        用于
        <strong>
         有向无环图（DAG）
        </strong>
        的排序。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       强连通分量
      </strong>
      ：
      <ul>
       <li>
        <strong>
         Kosaraju 算法
        </strong>
        ：用于查找有向图的强连通分量。
       </li>
      </ul>
     </li>
    </ul>
