     小写字母 - 32 = 大写字母
     <br/>
     大写字母 + 32 = 小写字母
    </p>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;

const int N = 110;
char a[N] = { 0 };

{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin &gt;&gt; a;

    int i = 0;
    while (a[i] != '\0')
    { 
        if (a[i] &gt;= 'a' &amp;&amp; a[i] &lt;= 'z')
        {
            a[i] -= 32;
        }
        cout &lt;&lt; a[i];
        i++;
    }
    cout &lt;&lt; endl;
    
    return 0;
}

    <p>
     这⾥再给⼤家介绍两个函数： islower 和 tolower ，需要的头⽂件是
     <code>
      &lt;cctype&gt;
     </code>
     <br/>
     字符分类函数和字符转换函数：https://legacy.cplusplus.com/reference/cctype/
    </p>
    <pre><code class="prism language-c++">int islower ( int c ); //判断字符是否是⼩写字⺟  
int tolower ( int c ); //转换成⼩写字⺟
</code></pre>
    <p>
     islower 是C/C++中提供的⼀个判断字符是否是⼩写字⺟的函数，如果参数 c 是⼩写字⺟，函数返回⼀个⾮0的数字，如果不是⼩写字⺟，函数返回0，其实还有⼀个函数是 isupper ，是判断⼤写字⺟的。
     <br/>
     tolower 是C/C++中提供的⼀个将参数 c 从⼤写字⺟转化成⼩写字⺟的函数，通过返回值返回转换后的⼩写字⺟。如果 c 本⾝就是⼩写字⺟，则什么都不发⽣。还有⼀个函数是 toupper ,是⼩写字⺟转换成⼤写的。
    </p>
    <pre><code class="prism language-c++">#include &lt;iostream&gt;  
#include &lt;cctype&gt;  

using namespace std;  
const int N = 110;  
char s[N];  
int main()  
{  
	cin &gt;&gt; s;
	  
	for(int i = 0; s[i] != '\0'; i++)  
	{  
		if(islower(s[i]))  
		{  
			s[i] = toupper(s[i]);  
		}  
	}  
	cout &lt;&lt; s &lt;&lt;endl;  

	return 0;
}

</code></pre>

    <p>
     如果参数 c 是数字字符，则返回⾮ 0 的值，如果不是数字字符，则返回 0 。
    </p>
    <pre><code class="prism language-c++">#include &lt;iostream&gt;

	int i = 0;  
	int c = 0;  
	while (arr[i] != '\0') //这⾥判断是否等于\0，来觉得是否结束  
	{  
		if (isdigit(arr[i]))

    <p>
     哈希
     <br/>
     题⽬说字符串中只有⼩写字⺟，⼩写字⺟的ASCII值的范围是：97
     <sub>
      122，根据前⾯我们学习的知识，我们知道在C和C++中每个字符都有ASCII值，标准的ASCII码表中有128个字符，ASCII值的范围是0
     </sub>
     127.
     <br/>
     所以我们创建⼀个128元素的整型数组，下标分别是0~127，下标正好和字符的ASCII值范围对应，那么整型的数组的⼀个元素就为⼀个字符计数就可以。每读取⼀个字符，根据字符的ASCII值将数组中下标为字符ASCII值的元素值+1，相当于统计这个字符出现的次数。
     <br/>
     <img alt="![[Pasted image 20250306191825.png]]" src="https://i-blog.csdnimg.cn/direct/ed020f96134c47b5a08bba74fe3b0e47.png"/>
    </p>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;

    ios::sync_with_stdio(false);

    {

        i++;        
    }

    }
    if (a[i] == 0)
        cout &lt;&lt; "no" &lt;&lt; endl;
    
    return 0;
