    <h3>
     <a id="_0">
     </a>
     概述
    </h3>
    <p>
     <a href="https://developer.huawei.com/consumer/cn/download/deveco-studio" rel="nofollow">
      HUAWEI DevEco Studio
     </a>
     （以下简称
     <strong>
      DevEco Studio
     </strong>
     ）是基于
     <strong>
      IntelliJ IDEA Community
     </strong>
     开源版本打造的一站式开发平台，专为
     <strong>
      HarmonyOS
     </strong>
     系统上的应用和元服务（以下简称
     <strong>
      应用/元服务
     </strong>
     ）提供高效的开发环境。
    </p>
    <p>
     作为一款专业的开发工具，DevEco Studio 不仅具备
     <strong>
      代码开发、编译构建、调试
     </strong>
     等基本功能，还提供了一系列增强特性，以优化开发体验和提升效率：
    </p>
    <ul>
     <li>
      <strong>
       高效智能代码编辑
      </strong>
      ：支持
      <strong>
       ArkTS、JS、C/C++
      </strong>
      语言，提供
      <strong>
       代码高亮、智能补全、错误检查、自动跳转、格式化、快速查找
      </strong>
      等功能，提高开发效率。
     </li>
     <li>
      <strong>
       多端双向实时预览
      </strong>
      ：支持
      <strong>
       UI 代码的双向预览、实时预览、动态预览、组件预览、多设备预览
      </strong>
      ，便于快速查看 UI 效果。
     </li>
     <li>
      <strong>
       多端设备模拟仿真
      </strong>
      ：内置
      <strong>
       HarmonyOS 本地模拟器
      </strong>
      ，支持
      <strong>
       Phone
      </strong>
      等设备的模拟仿真，方便调试应用。
     </li>
     <li>
      <strong>
       DevEco Profiler 性能调优
      </strong>
      ：提供
      <strong>
       实时监控能力
      </strong>
      和
      <strong>
       场景化调优模板
      </strong>
      ，覆盖多个维度的设备资源监测，帮助开发者快速优化代码性能。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_13">
     </a>
     开发准备
    </h3>
    <p>
     您可以通过单击
     <a href="https://developer.huawei.com/consumer/cn/download/" rel="nofollow">
      此处下载 DevEco Studio
     </a>
     ，并按照安装向导完成开发工具的安装。
    </p>
    <blockquote>
     <p>
      <strong>
       注意
      </strong>
      ：DevEco Studio 依赖网络环境，为确保工具的正常使用，请确保设备已连接互联网。若企业网络存在访问限制，可能需要手动配置代理信息。
     </p>
    </blockquote>

    <h3>
     <a id="_DevEco_Studio_32">
     </a>
     安装 DevEco Studio
    </h3>
    <h4>
     <a id="Windows__34">
     </a>
     Windows 环境
    </h4>
    <ol>
     <li>
      下载完成后，双击
      <strong>
       deveco-studio-xxxx.exe
      </strong>
      ，进入安装向导。
     </li>
     <li>
      选择安装路径（默认路径为
      <code>
       C:\Program Files
      </code>
      ），或单击
      <strong>
       浏览（B）…
      </strong>
      选择其他路径，点击
      <strong>
       Next
      </strong>
      继续。
     </li>
     <li>
      在安装选项界面，勾选
      <strong>
       DevEco Studio
      </strong>
      ，然后依次单击
      <strong>
       Next
      </strong>
      直到安装完成。
     </li>
     <li>
      安装完成后，单击
      <strong>
       Finish
      </strong>
      退出安装向导。
     </li>
    </ol>
    <h4>
     <a id="macOS__41">
     </a>
     macOS 环境
    </h4>
    <ol>
     <li>
      在安装界面中，将
      <strong>
       DevEco-Studio.app
      </strong>
      拖拽到
      <strong>
       Applications
      </strong>
      文件夹，等待安装完成。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_47">
     </a>
     特别说明
    </h3>
    <ol>
     <li>
      <strong>
       开箱即用
      </strong>
      ：DevEco Studio 集成了
      <strong>
       HarmonyOS SDK、Node.js、Hvigor、OHPM、模拟器平台
      </strong>
      等，无需额外配置，大幅简化安装流程。
     </li>
     <li>
      <strong>
       SDK 位置
      </strong>
      ：HarmonyOS SDK 已嵌入 DevEco Studio，可在以下路径查看：
      <ul>
       <li>
        <strong>
         Windows
        </strong>
        ：
        <code>
         Settings &gt; OpenHarmony SDK
        </code>
       </li>
       <li>
        <strong>
         macOS
        </strong>
        ：
        <code>
         DevEco Studio &gt; Preferences &gt; OpenHarmony SDK
        </code>
       </li>
      </ul>
     </li>
     <li>
      如需开发
      <strong>
       OpenHarmony
      </strong>
      应用，可在
      <strong>
       OpenHarmony SDK
      </strong>
      页面下载对应的 SDK。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_Hello_World_57">
     </a>
     创建并运行 Hello World
    </h3>
    <p>
     安装 DevEco Studio 后，建议通过运行
     <strong>
      Hello World
     </strong>
     工程来验证环境是否配置正确。以下以创建
     <strong>
      Phone 设备
     </strong>
     工程为例。
    </p>
    <h4>
     <a id="_61">
     </a>
     创建新工程
    </h4>
    <ol>
     <li>
      打开
      <strong>
       DevEco Studio
      </strong>
      ，在欢迎页单击
      <strong>
       Create Project
      </strong>
      创建新工程。
     </li>
     <li>
      根据向导，选择
      <strong>
       Application
      </strong>
      或
      <strong>
       Atomic Service
      </strong>
      ，选择
      <strong>
       Empty Ability
      </strong>
      模板，单击
      <strong>
       Next
      </strong>
      。
     </li>
    </ol>
    <table>
     <thead>
      <tr>
       <th>
        模板名称
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         Empty Ability
        </strong>
       </td>
       <td>
        适用于
        <strong>
         Phone、Tablet、2in1、Car
        </strong>
        设备，展示
        <strong>
         Hello World
        </strong>
        功能。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Native C++
        </strong>
       </td>
       <td>
        适用于
        <strong>
         Phone、Tablet、2in1、Car
        </strong>
        设备，示例工程调用 C++ 代码，界面显示 “Hello World”。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         [CloudDev]Empty Ability
        </strong>
       </td>
       <td>
        端云一体化开发通用模板。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         [Lite]Empty Ability
        </strong>
       </td>
       <td>
        适用于
        <strong>
         Lite Wearable
        </strong>
        设备，展示
        <strong>
         Hello World
        </strong>
        ，可用于小型嵌入式设备开发。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Flexible Layout Ability
        </strong>
       </td>
       <td>
        用于创建跨设备应用的
        <strong>
         三层架构工程
        </strong>
        （公共能力层、基础特性层、产品定制层）。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Embeddable Ability
        </strong>
       </td>
       <td>
        用于开发
        <strong>
         可嵌入到其他应用
        </strong>
        运行的元服务。
       </td>
      </tr>
     </tbody>
    </table>
    <ol start="3">
     <li>
      填写项目信息，单击
      <strong>
       Finish
      </strong>
      完成创建。
     </li>
    </ol>
    <table>
     <thead>
      <tr>
       <th>
        设置项
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         Project Name（项目名称）
        </strong>
       </td>
       <td>
        可自定义项目名称。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Bundle Name（包名称）
        </strong>
       </td>
       <td>
        影响应用 ID，需与应用发布 ID 保持一致。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Save Location（保存位置）
        </strong>
       </td>
       <td>
        项目存放路径，建议自定义。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Compile SDK（编译 SDK）
        </strong>
       </td>
       <td>
        选择合适的 API 版本。
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         Model（模型）
        </strong>
       </td>
       <td>
        选择
        <strong>
         Stage 模型
        </strong>
        ，其余保持默认。
