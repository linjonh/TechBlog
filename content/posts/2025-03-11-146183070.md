   <div class="htmledit_views" id="content_views">
    <p>
     一、使用国内的源（下载速度起飞）
    </p>
    <p>
     打开终端，设置国内源：
    </p>
    <p>
     &gt;pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
    </p>
    <p>
     官方源 https://pypi.org/simple
    </p>
    <p>
     本地文件查看：C:\Users\levy\AppData\Roaming\pip\pip.ini
    </p>
    <p>
     <br/>
     二、使用pip安装重要包文件及版本：
     <br/>
     tensorflow有点大需要等待一下，按照顺序执行，版本之间的匹配非常重要，版本对应不上就会有各种报错。
    </p>
    <p>
     pip install numpy==1.21.6
     <br/>
     pip install pandas==1.0.3
     <br/>
     pip install scipy==1.1.0
     <br/>
     pip install scikit-learn==1.0.2
     <br/>
     pip install tensorflow==2.11.0
     <br/>
     pip install keras==2.11.0
    </p>
    <p>
    </p>
    <p>
     三、查看tensorboard界面
    </p>
    <p>
     在pyCharm终端输入命令：
    </p>
    <p>
     &gt;tensorboard.exe --logdir=E:\\PycharmProjects\\tensorboard
    </p>
