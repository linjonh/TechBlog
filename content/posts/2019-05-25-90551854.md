# SQL Server 查询数据库中所有数据库名表名

1.查询数据库中的所有数据库名：

SELECT Name FROM Master..SysDatabases ORDER BY Name

2.查询某个数据库中所有的表名：

SELECT Name FROM SysObjects Where XType='U' ORDER BY Name
