    <h2>
     <a id="2__31">
     </a>
     2 思路解析
    </h2>
    <p>
     此题考查图的
     <code>
      DFS
     </code>
     和
     <code>
      BFS
     </code>
     遍历。
     <br/>
     针对某个结点，我们除了要知道它是属于哪个连通块以外，还要能判断它是否已经被遍历过，然后从小编号的结点到大编号的结点依次遍历，如果当前结点已经被遍历过，则跳过，否则从这个结点开始进行
     <code>
      DFS
     </code>
     和
     <code>
      BFS
     </code>
     遍历。
    </p>
    <h3>
     <a id="21__34">
     </a>
     2.1 图的存储
    </h3>
    <p>
     一般来说，常用的图的存储方式有
     <code>
      邻接矩阵
     </code>
     和
     <code>
      邻接表
     </code>
     两种，在图的遍历中，一般选择后者存储，不过当图的结点比较少时，选择前者代码更简单。这里我们选择以
     <code>
      邻接矩阵
     </code>
     的方式存图（题目所给的
     <code>
      n
     </code>
     不超过
     <code>
      10
     </code>
     ，很小，更重要的原因是题目要求结点按从小到大的顺序遍历，邻接表不好控制这个）。
    </p>
    <h3>
     <a id="22_DFS_36">
     </a>
     2.2 DFS
    </h3>
    <p>
     给你一个结点编号
     <code>
      i
     </code>
     表示要从
     <code>
      i
     </code>
     开始遍历，除此之外你还需要知道邻接矩阵和标记数组等信息，才能完成遍历，由此可得出如下算法步骤：
     <br/>
     1. 输出
     <code>
      i
     </code>
     ；
     <br/>
     2. 遍历当前结点
     <code>
      i
     </code>
     的所有邻接结点
     <code>
      j
     </code>
     ：
     <br/>
     2.1 如果
     <code>
      j
     </code>
     已经被遍历过则跳过
     <code>
      j
     </code>
     ，否则标记
     <code>
      j
     </code>
     ，表示
     <code>
      j
     </code>
     现在已经被遍历了，然后以
     <code>
      j
     </code>
     为起点再进行
     <code>
      DFS
     </code>
     遍历。
    </p>
    <h3>
     <a id="23_BFS_41">
     </a>
     2.3 BFS
    </h3>
    <p>
     给你一个结点编号
     <code>
      i
     </code>
     表示要从
     <code>
      i
     </code>
     开始遍历，除此之外你还需要知道邻接矩阵和标记数组等信息，同时还需要借助队列作为辅助才能完成遍历，由此可得出如下算法步骤：
     <br/>
     1. 判断
     <code>
      i
     </code>
     是否被遍历，如果没有被遍历，则标记并将其加入队列
     <code>
      q
     </code>
     ；
     <br/>
     2. 如果队列不为空，循环：
     <br/>
     2.1 取出队首元素
     <code>
      j
     </code>
     ，输出
     <code>
      j
     </code>
     ；
     <br/>
     2.2 将
     <code>
      j
     </code>
     的所有未被遍历的邻接结点都加入队列，并标记。
     <br/>
     队列的实现可以参考
     <a href="https://blog.csdn.net/m0_52319522/article/details/145533381">
      这篇文章
     </a>
     ，此题的代码实现并没有使用单独的队列数据结构，而是怎么方便怎么来，具体见代码。
    </p>
