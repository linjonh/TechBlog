# Caused by: java.io.FileNotFoundException: Could not open ServletContext resource [/WEB-INF/springDis

```objectivec
Caused by: 
java.io.FileNotFoundException:
Could not open ServletContext resource [/WEB-INF/springDispatcherServlet-servlet.xml]

```

**这个错误就是告诉我这个
  
无法打开ServletContext资源[/WEB-INF/springDispatcherServlet-servlet.xml]这个文件打不开，开启不了。我去检查了一下我的web.xml里的相应的文件的配置发现是路径有问题：**
  
1.1、我最开始是用的这个默认路径下，我是直接注释掉了下面的这个语句的，因为我放到了默认路径下就不用写路径了，但是报错：

```xml
	<!-- 下面这个不指定的话，必须在其同级目录下有一个 -->
		<init-param>
			<param-name>contextConfigLocation</param-name>
			<param-value>/local</param-value>
		</init-param> 

```

1.2、这个是我改进之后，我把路径加上了，就没有错误了：

```xml
<!-- springmvc -->
	<!-- The front controller of this Spring Web application, responsible for handling all application requests -->
	<servlet>
		<servlet-name>springDispatcherServlet</servlet-name>
		<servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
		<!-- 下面这个不指定的话，必须在其同级目录下有一个 -->
<!-- 		<init-param>
			<param-name>contextConfigLocation</param-name>
			<param-value>/local</param-value>
		</init-param> -->
		<init-param>
			<param-name>contextConfigLocation</param-name>
			<param-value>/WEB-INF/dispatcherServlet-servlet.xml</param-value>
		</init-param>
		<load-on-startup>1</load-on-startup>
	</servlet>

    <!-- Map all requests to the DispatcherServlet for handling -->
    <servlet-mapping>
    	<servlet-name>springDispatcherServlet</servlet-name>
    	<url-pattern>/</url-pattern>
    </servlet-mapping>


```
