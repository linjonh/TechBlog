    <p>
     1.首先确保没有安装docker，若安装了yum remove docker 卸载docker

    </p>
    <pre><code class="prism language-c">yum install docker

</code></pre>

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9f3e09d7dc55fe4c5dcbd17b7f48ba81.png">

      3.查看docker版本
     </img>
    </p>
    <pre><code class="prism language-c">docker <span class="token operator">-</span>v

      <br/>
      4. 查看镜像，测试下载一个nginx镜像
     </img>
    </p>
    <pre><code class="prism language-c">docker images
