    <h2 id="%E4%B8%80%E3%80%81%E6%97%B6%E9%97%B4%E5%A4%8D%E6%9D%82%E5%BA%A6%E4%B8%8E%E7%A9%BA%E9%97%B4%E5%A4%8D%E6%9D%82%E5%BA%A6">
     一、时间复杂度与空间复杂度
    </h2>
    <h3 id="1.%E6%97%B6%E9%97%B4%E5%A4%8D%E6%9D%82%E5%BA%A6">
     1.时间复杂度
    </h3>
    <p>
     时间复杂度的定义：在计算机科学中，算法的时间复杂度是一个函数，它定量描述了该算法的运行时间。一 个算法执行所耗费的时间，从理论上说，是不能算出来的，只有你把你的程序放在机器上跑起来，才能知 道。但是我们需要每个算法都上机测试吗？是可以都上机测试，但是这很麻烦，所以才有了时间复杂度这个 分析方式。一个算法所花费的时间与其中语句的执行次数成正比例，
     <span style="color:#fe2c24;">
      算法中的基本操作的执行次数，为算法 的时间复杂度
     </span>
     。
    </p>
    <pre><code class="language-cpp">void Func1(int N)
{
int count = 0;
for (int i = 0; i &lt; N ; ++ i)
{
 for (int j = 0; j &lt; N ; ++ j)
 {
 ++count;
 }
}
for (int k = 0; k &lt; 2 * N ; ++ k)
{
 ++count;
}
int M = 10;
while (M--)
{
 ++count;
}
printf("%d\n", count);
}</code></pre>
    <p>
     以上代码Func1的执行次数为第一个for循环嵌套for循环就是N*N就是N^2再加for循环2*N次再加10次
    </p>
    <p>
     ：F(N)=N^2+2*N+10。
    </p>
    <p>
     如果N=10 F(N)=130               如果N=100 F(N)=10210             如果N=1000 F(N)=1002010
    </p>
    <p>
     实际中我们计算时间复杂度时，我们其实并不一定要计算精确的执行次数，而只需要
     <span style="color:#fe2c24;">
      大概执行次数，那么这 里我们使用大O的渐进表示法
     </span>
    </p>
    <p>
     推导大O阶的方法：
    </p>
    <p>
     1.用常数1代取运行时间中的所有
     <span style="color:#fe2c24;">
      加法
     </span>
     常数
    </p>
    <p>
     2.再修改后的运行次数函数中，只保留次数最多的阶项(次数最多的一项)
    </p>
    <p>
     3.如果最高阶项存在且不是1，则去除与这个项目相乘的常数。得到的结果就是大O阶
    </p>
    <p>
     使用大O的渐进表示法以后，Func1的时间复杂度为：O(N^2)
    </p>
    <p>
     因N^2是此项目里执行次数最多的一项
    </p>
    <h3 id="2.%E7%A9%BA%E9%97%B4%E5%A4%8D%E6%9D%82%E5%BA%A6">
     2.空间复杂度
    </h3>
    <p>
     空间复杂度是对一个算法在运行过程中
     <span style="color:#fe2c24;">
      临时占用存储空间大小的量度
     </span>
     (
     <span style="color:#ff9900;">
      临时变量的个数
     </span>
     )。空间复杂度不是程序占用了多少 bytes的空间，因为这个也没太大意义，所以空间复杂度算的是变量的个数。空间复杂度计算规则基本跟实践 复杂度类似，也使用大O渐进表示法。
    </p>
    <pre><code class="language-cpp">实例1
void BubbleSort(int* a, int n)
{
 assert(a);
 for (size_t end = n; end &gt; 0; --end)
 {
 int exchange = 0;
 for (size_t i = 1; i &lt; end; ++i)
 {
 if (a[i-1] &gt; a[i])
 {
 Swap(&amp;a[i-1], &amp;a[i]);
 exchange = 1;
 }
 }
 if (exchange == 0)
 break;
 }
}




实例2
long long* Fibonacci(size_t n)
{
 if(n==0)
 return NULL;
 
 long long * fibArray = (long long *)malloc((n+1) * sizeof(long long));
 fibArray[0] = 0;
 fibArray[1] = 1;for (int i = 2; i &lt;= n ; ++i)
 {
 fibArray[i ] = fibArray[ i - 1] + fibArray [i - 2];
 }
 return fibArray ;
}</code></pre>
    <p>
     1. 实例1使用了常数个额外空间，所以空间复杂度为 O(1)
    </p>
    <p>
     2. 实例2动态开辟了N个空间，空间复杂度为 O(N)
    </p>
    <h2 id="%E4%BA%8C%E3%80%81%E9%A1%BA%E5%BA%8F%E8%A1%A8">
     二、顺序表和链表(线性表)
    </h2>
    <p>
     线性表（linear list）是n个具有相同特性的数据元素的有限序列。
    </p>
    <p>
     线性表在逻辑上是线性结构，也就说是连续的一条直线。但是在物理结构上并不一定是连续的，线性表在物 理上存储时，通常以数组和链式结构的形式存储。
    </p>
    <p>
     <img alt="" height="377" src="https://i-blog.csdnimg.cn/blog_migrate/f5618e06e88d4ad1d12956f1ef1e65f1.png" width="685"/>
    </p>
    <h3 id="1.%E7%BA%BF%E6%80%A7%E8%A1%A8-%E7%BA%BF%E6%80%A7%E8%A1%A8%E5%8A%9F%E8%83%BD%E5%AE%9E%E7%8E%B0">
     1.顺序表-顺序表功能实现
    </h3>
    <p>
     顺序表是用一段物理地址连续的存储单元依次存储数据元素的线性结构，一般情况下采用数组存储。在数组 上完成数据的增删查改。
    </p>
    <p>
     头文件
    </p>
    <pre><code class="language-cpp">#define _CRT_SECURE_NO_WARNINGS 1
#include &lt;stdio.h&gt;
#include &lt;string.h&gt;
#include &lt;assert.h&gt;
#include &lt;windows.h&gt;
#include &lt;math.h&gt;
#include &lt;errno.h&gt;
#define MAX 100//定义顺序表的容量大小
#define ElemType int//定义顺序表的类型
//顺序表的结构体实现
typedef struct SqList
{
	ElemType data[MAX];///顺序表的数组元素
	int capacity;//顺序表的有效长度
}SqList;
void SqListInit(SqList* con);//初始化顺序表
void SqListPushBack(SqList* con, ElemType val);//尾插
void SqListPopBack(SqList* con);//尾删
void SqListPushFront(SqList* con,ElemType val);//头插
void SqListPopFront(SqList* con);//头删
void SqListInsert();//指定位置插入
void SqListDelete();//删除元素
void SqListFind();//查找元素
void SqListFind();//修改元素
void SqListShow(SqList* con);//显示所有元素</code></pre>
    <p>
     函数实现
    </p>
    <pre><code class="language-cpp">void SqListInit(SqList* con)
{
	for (int i = 0; i &lt; MAX; i++)
	{
		con-&gt;data[i] = 0;//将所有元素初始化为0
	}
	con-&gt;capacity = 0;//长度初始化为0
}
void SqListPushBack(SqList* con,ElemType val)
{
	if (con-&gt;capacity == MAX)//判断顺序表是否为满
	{
		printf("该顺序表已满\n");
		return;
	}
	con-&gt;data[con-&gt;capacity++] = val;//将下标为capacity的元素复制为val然后capacity加1
}
void SqListPopBack(SqList* con)
{
	if (con-&gt;capacity == 0)//判断顺序表是否为空
	{
		printf("该顺序表为空\n");
		return;
	}
	con-&gt;capacity--;//长度个数减1完成逻辑上的删除
}
void SqListPushFront(SqList* con,ElemType val)
{
	if (con-&gt;capacity == MAX)//判断顺序表是否为满
	{
		printf("该顺序表已满\n");
		return;
	}
	int num = con-&gt;capacity;
	while (num)//将所有元素后移
	{
		con-&gt;data[num] = con-&gt;data[num-1];
		num--;
	}
	con-&gt;data[0] = val;//在第一个元素赋值val
	con-&gt;capacity++;//长度加1
}
void SqListPopFront(SqList* con)
{
	if (con-&gt;capacity == 0)//判断顺序表是否为空
	{
		printf("该顺序表为空\n");
		return;
	}
	for (int i = 0; i &lt; con-&gt;capacity - 2; i++)//将所有元素往前移
	{
		con-&gt;data[i] = con-&gt;data[i + 1];
	}
	con-&gt;capacity--;
}
void SqListInsert(SqList* con,ElemType val,int pos)
{
	if (con-&gt;capacity == MAX)//判断顺序表是否为满
	{
		printf("该顺序表已满\n");
		return;
	}
	if (pos&lt;1 || pos&gt;con-&gt;capacity + 1)//判断指定位置是否有效
	{
		printf("位置无效\n");
		return;
	}
	for (int i = con-&gt;capacity; i &gt;= pos; i--)
	{
		con-&gt;data[i] = con-&gt;data[i - 1];
	}
	con-&gt;data[pos - 1] = val;
	con-&gt;capacity++;
}
void SqListDelete(SqList* con,ElemType val)
{
	if (con-&gt;capacity == 0)//判断顺序表是否为空
	{
		printf("该顺序表为空\n");
		return;
	}
	int num = -1;
	for (int i = 0; i &lt; con-&gt;capacity; i++)
	{

{

	if (con-&gt;top - con-&gt;data == con-&gt;StackSize)//判断栈顶到栈底的元素如果等于长度说明栈满 则执行扩容操作
	{
		STDataType* ptr = (STDataType*)realloc(con-&gt;data, sizeof(STDataType)*(con-&gt;StackSize + 2));//重新指定一个原有数组的长度+2个数组
		if (ptr == NULL)
		{
			printf("%s ", strerror(errno));
			return;
		}
		con-&gt;data = ptr;//复制数据
		con-&gt;StackSize += 2;//长度+2
	}
	*con-&gt;top++ = x;//解引用栈顶找到栈顶的空间放入元素栈顶++则完成入栈
}
STDataType StackPop(Stack* con)//出栈 返回值用元素类型来接受弹出的元素
{
	assert(con);//判断栈有效性
	con-&gt;top--;//栈顶向下移动
	return *(con-&gt;top);//返回下移后的元素 返回后该位置变为栈顶则弹出成功
}
void StackClear(Stack* con)//清空栈
{
	assert(con);//判断有效性
	con-&gt;top = con-&gt;data;//将栈顶指向栈底完成逻辑上的删除则完成清空
}
void StackDel(Stack* con)//销毁栈
{
	assert(con);//判断栈有效性
	free(con-&gt;data);//释放栈的数组空间
	con-&gt;StackSize = 0;//将长度设为0
	con-&gt;top = con-&gt;data = NULL;//将栈的栈顶指针和数组置空
}</code></pre>

{

	obj-&gt;tail = con;//新结点变成新的队尾

ElemType QueuePop(LinkQueue* obj)//出队列

	if (QueueEmpty(obj))
	{
		printf("队列为空\n");
		return -1;

	ElemType x = cur-&gt;data;//存储队头结点的元素准备返回
	obj-&gt;front-&gt;next = cur-&gt;next;//队头指向第二个结点
	if (cur-&gt;next == NULL)//判断队列是否只有一个结点 特殊处理
	{
		obj-&gt;tail = obj-&gt;front;
	}
	free(cur);

		printf("队列为空\n");
		return;
	}
	Queue* cur = obj-&gt;front-&gt;next;//存队列的第一个结点
	while (cur)
	{
		printf("%d ", cur-&gt;data);

		free(obj-&gt;front);
		obj-&gt;front = obj-&gt;tail;//队头迭代
	}

	free(obj);//销毁队列

    <h2 id="%E5%9B%9B%E3%80%81%E4%BA%8C%E5%8F%89%E6%A0%91">
     四、二叉树

    <h3 id="1.%E4%BA%8C%E5%8F%89%E6%A0%91-%E4%BA%8C%E5%8F%89%E6%A0%91%E5%8A%9F%E8%83%BD%E5%AE%9E%E7%8E%B0">
     1.二叉树-初阶二叉树功能实现
    </h3>
    <p>
     树是一种非线性的数据结构,它是由n（n不小于0）个有限结点组成一个具有层次关系的集合
    </p>
    <p>
     <strong>
      把它叫做树是因 为它看起来像一棵倒挂的树，也就是说它是根朝上，而叶朝下的
     </strong>
    </p>
    <p>
     有一个特殊的结点，称为根结点，根节点没有前驱结点 除根节点外，其余结点被分成M(M&gt;0)个互不相交的集合T1、T2、……、Tm，其中每一个集合Ti(1&lt;= i &lt;= m)又是一棵结构与树类似的子树。每棵子树的根结点有且只有一个前驱，可以有0个或多个后继 因此，树是递归定义的。
    </p>
    <p>
     <img alt="" height="323" src="https://i-blog.csdnimg.cn/blog_migrate/7e4d071db872ceb6c83915d85e850204.png" width="608"/>
    </p>
    <p>
     节点的度：一个节点含有的子树的个数称为该节点的度； 如上图：A的为6
    </p>
    <p>
     叶节点或终端节点：度为0的节点称为叶节点； 如上图：B、C、H、I...等节点为叶节点
    </p>
    <p>
     非终端节点或分支节点：度不为0的节点； 如上图：D、E、F、G...等节点为分支节点
    </p>

    </p>
    <p>
     孩子节点或子节点：一个节点含有的子树的根节点称为该节点的子节点； 如上图：B是A的孩子节点
    </p>
    <p>
     兄弟节点：具有相同父节点的节点互称为兄弟节点； 如上图：B、C是兄弟节点
    </p>

    </p>
    <p>
     二叉树的特点：
    </p>
    <p>
     1. 每个结点最多有两棵子树，即二叉树不存在度大于2的结点。
    </p>
    <p>
     2. 二叉树的子树有左右之分，其子树的次序不能颠倒。
    </p>
    <p>
     <img alt="" height="321" src="https://i-blog.csdnimg.cn/blog_migrate/35b493de9449eb2a56b8b7db1d485c6e.png" width="381"/>

    <p>
     1. 满二叉树：一个二叉树，如果每一个层的结点数都达到最大值，则这个二叉树就是满二叉树。也就是 说，如果一个二叉树的层数为K，且结点总数是(2^k) -1 ，则它就是满二叉树。
    </p>
    <p>
     <img alt="" height="736" src="https://i-blog.csdnimg.cn/blog_migrate/2bc1972608253f66c618fe50f449ee69.png" width="1200"/>
    </p>
    <p>
     2. 完全二叉树：完全二叉树是效率很高的数据结构，完全二叉树是由满二叉树而引出来的。对于深度为K 的，有n个结点的二叉树，当且仅当其每一个结点都与深度为K的满二叉树中编号从1至n的结点一一对 应时称之为完全二叉树。 要注意的是满二叉树是一种特殊的完全二叉树
    </p>

     <img alt="" height="739" src="https://i-blog.csdnimg.cn/blog_migrate/f2f2d5cc91a355267062044284f975d6.png" width="1076"/>
    </p>
    <p>
     二叉树一般可以使用两种结构存储，一种顺序结构，一种链式结构。
    </p>
    <p>
     二叉树的性质：
    </p>
    <p>
     1. 若规定根节点的层数为1，则一棵非空二叉树的第i层上最多有2^(i-1) 个结点.
    </p>
    <p>
     2. 若规定根节点的层数为1，则深度为h的二叉树的最大结点数是2^h- 1.
    </p>
    <p>
     <img alt="" height="649" src="https://i-blog.csdnimg.cn/blog_migrate/30ababb89c6d61050e992f82e0027149.png" width="1200"/>
