    <h5>
     <a id="_3">
     </a>
     <strong>
      三步快速实现远程调试
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        硬件部署
       </strong>
      </p>
      <ul>
       <li>
        准备西门子S7-1200 PLC、巨控GRM552YW-C模块及编程电脑。
       </li>
       <li>
        GRM552YW-C通过网口与PLC连接，支持4G/5G/Wi-Fi/有线网络接入，无需复杂布线。
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b86b9392d6a24be8bfb7c70d9d14405f.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        软件配置
       </strong>
      </p>
      <ul>
       <li>
        安装
        <strong>
         GVCOM3配置软件
        </strong>
        ，注册模块（输入唯一序列号与密码，初始密码需修改）。
       </li>
       <li>
        配置网口IP（与PLC同一网段，例如PLC IP为
        <code>
         192.168.2.143
        </code>
        ，模块IP设为
        <code>
         192.168.2.22
        </code>
        ）。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/32002ea9cb0347f58a53df828e0f1cb5.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        远程操作
       </strong>
      </p>
      <ul>
       <li>
        使用西门子TIA Portal（博图）软件，通过虚拟网卡（TAP-Win32 Adapter V9）搜索在线PLC。
       </li>
       <li>
        直接远程上传/下载程序，支持实时监控变量、在线调试梯形图逻辑。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/71be9b8e9b4f4640aa56fc4f540d6621.png"/>
       </li>
      </ul>
     </li>
    </ol>
