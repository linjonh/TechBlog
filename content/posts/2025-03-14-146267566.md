# 微信开发者工具内建终端使用不了npm，但是cmd可以

下载cnpm并配置镜像源

终端cmd：  npm install -g cnpm --registry=http://registry.npmmirror.com

打开微信开发者工具，找到方框的文件右击选择内建终端打开

![](https://i-blog.csdnimg.cn/direct/0eb130953c814bc9873ea640bbd203bb.png)

初始化：  npm init -y    发现npm没有此命令

![](https://i-blog.csdnimg.cn/direct/031dbe8dd338495885c98c46797a41df.png)
关闭微信开发工具，用
**管理员的身份**
重新打开微信开发者工具

npm init -y

配置ui库

cnpm install @vant/weapp -S
