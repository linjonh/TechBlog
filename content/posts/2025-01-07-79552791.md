    <p>
     <span style="line-height:24px;">
      <span style="font-family:'Microsoft YaHei';background-color:rgb(255,255,255);">
       <span style="font-size:16px;color:#666666;">
        Selenium Python提供了一个简单的绑定式API来使用Selenium WebDriver编写功能/验收测试。通过Selenium Python API，您可以以一种直观的方式访问Selenium WebDriver的所有功能。
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:'Microsoft YaHei';background-color:rgb(255,255,255);">
      <span style="font-size:16px;color:#666666;">
       Selenium Python提供了一个便利的绑定式API去接入火狐、IE、Chrome、 Remote等浏览器。目前支持的python
       <span style="color:rgb(102,102,102);font-family:'Microsoft YaHei';font-size:18px;background-color:rgb(255,255,255);">
        版本
       </span>
       有2.7或3.5以上。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:'Microsoft YaHei';background-color:rgb(255,255,255);">
      <span style="font-size:16px;color:#666666;">
       <span style="text-align:left;">
        这份文档介绍了
       </span>
       <span style="text-align:left;">
        Selenium 2 WebDriver API. Selenium 1，但不包括Selenium RC API 。
       </span>
      </span>
     </span>
    </p>

      </span>

       PyPI
      </a>
      <span style="font-family:'Microsoft YaHei';text-align:left;">
       包下载
      </span>
      <span style="font-family:'Microsoft YaHei';text-align:left;">
       Selenium。当然，最好使用python自带
       <a href="https://pip.pypa.io/en/latest/installing/" rel="nofollow">
        pip
       </a>
       命令去安装Selenium包。Python3.6
       <a href="https://docs.python.org/3.6/installing/index.html" rel="nofollow">
        标准库
       </a>
       中自带pip命令，而pthon2.x需要自己单独安装。通过如下操作使用pip命令安装Selenium:
