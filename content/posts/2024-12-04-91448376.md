     Java中类名，方法，变量，包名等大小写规范
    </h3>
    <p>
     类名：首字母大写 例如： Class Person 中的P需要大写
    </p>
    <p>
     方法名： 一般用动词开头 采用驼峰方式 首字母小写 例如: int driveCar();
    </p>
    <p>
     变量名： 一般全是小写 例如: int count; int age;如果使用关键字static final定义的就应该全部都用大写，如：
     <br/>
     static final int VALUE_ONE = 11;
    </p>
    <p>
     包名： 一般采用公司域名倒过来写 例如：Package com.sina.java…
    </p>
    <p>
     tips:
     <br/>
     建议javaer都是用idea开发，在里边可以下载插件
     <strong>
      Alibaba Java Coding Guidelines
     </strong>
     来约束自己的命名规范，还有书写代码的格式，以养成良好的习惯。
    </p>
