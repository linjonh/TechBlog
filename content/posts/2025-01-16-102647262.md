    <h2>
     <a id="vscode_python_0">
     </a>
     vscode 如何导入python库
    </h2>
    <p>
     首先，我们要知道，VScode和本地运行的并不是同一个python，反正我的是这样，所以导致了在本地下载好库后，在VScode运行时还是报错。那么如何在VScode中导入python库呢？
    </p>
    <h4>
     <a id="1vscodepython_4">
     </a>
     1.已经在vscode中装了python
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3f8ff850ec9ae7d2287f0558fb9e945f.jpeg"/>
    </p>
    <h4>
     <a id="2_6">
     </a>
     2.打开终端
    </h4>
    <p>
     这个就相当于本地的 cmd 命令一样
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d706f8a600d8c18a0d7f6f38975c4a1d.jpeg"/>
    </p>
    <h4>
     <a id="3vscodepython_9">
     </a>
     3.找到vscode中python的路径
    </h4>
    <p>
     打个比方，我现在还没有Pillow这个库，我点击运行以后，它会报错。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0cfe09cd94b88b0992533a1b043e7b6c.png">
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b58a107d379c539f7d3fad89cdbe4164.png">
       <br/>
       蓝色的路径就是vscode中python的路径，我们要的路径就到Python37_64为止，也就是
      </img>
     </img>
    </p>
    <pre><code>C:/Program Files (x86)/Microsoft Visual Studio/Shared/Python37_64
</code></pre>
    <p>
     把这个复制下来
    </p>
    <h4>
     <a id="4pippython_18">
     </a>
     4.运行pip，安装python库
    </h4>
    <p>
     首先声明，高版本的python中都是自带pip的，如果没有请先安装pip
     <br/>
     如何看有没有pip呢？ 在本地打开刚才复制的路径，进入到这个Script文件夹下
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c10f7246145d157a20c9b22e976ec3b9.png">
      <br/>
      看看里面有没有pip.exe文件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/460ea46796fd5d9585c0025146c13a99.jpeg">
       <br/>
       如果有的话就可以运行pip了。
      </img>
     </img>
    </p>
    <p>
     .
     <br/>
     .
     <br/>
     .
    </p>
    <p>
     下面正式开始
     <br/>
     输入
    </p>
    <pre><code>cd "C:\Program Files (x86)\Microsoft Visual Studio\Shared\Python37_64\Scripts"
</code></pre>
    <p>
     即 cd+格式+“刚才复制的路径+\Scripts”
     <br/>
     注意：这里的路径一定要用双引号括起来不然在路径里有(x86)的情况下会报错
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/341399305c64a33508a1b72e59d437c6.png">
      <br/>
      然后输入 .\pip install 库名
     </img>
    </p>
    <pre><code>.\pip install Pillow
</code></pre>
    <p>
     注意前面一定要加 .\
    </p>
    <p>
     然后开始下载了
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ae0637cf1eb165bf54d0f16a0f615f42.png">
      <br/>
      显示 Successfully installed Pillow 就是下载成功了
      <br/>
      .
     </img>
    </p>
