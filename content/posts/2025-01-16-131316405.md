     3、检验堆空间修改
    </h2>
    <h3>
     3.1 测试类
    </h3>
    <pre><code class="language-java">public class StackTest {
    public static void main(String[] args) {
        //返回Java虚拟机中的堆内存总量
        long xmsMemory = Runtime.getRuntime().totalMemory() / 1024 / 1024;
        //返回Java虚拟机中使用的最大堆内存
        long xmxMemory = Runtime.getRuntime().maxMemory() / 1024 / 1024;
        System.out.println("-Xms:" + xmsMemory + "M");
        System.out.println("-Xmx:" + xmxMemory + "M");

    }

}</code></pre>

<h2>
4、检验栈空间修改
</h2>
<h3>
4.1 测试类
</h3>
<pre><code class="language-java">public class StackSizeTest {
private static int count = 1;

    public static void stack() {
        System.out.println(count++);
        stack();
    }

    public static void main(String[] args) {
        StackSizeTest.stack();
    }

}</code></pre>
