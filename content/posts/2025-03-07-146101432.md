    <p id="">
     步骤实现：
    </p>
    <ol>
     <li>
      <p id="">
       <strong>
        定义全局 CSS 变量
       </strong>
       ：在你的主样式文件（通常是
       <code>
        App.vue
       </code>
       或一个单独的 CSS 文件）中定义全局 CSS 变量。
      </p>
     </li>
     <li>
      <p id="">
       <strong>
        创建 Vue 组件或组合式 API 来管理这些变量
       </strong>
       ：你可以使用一个 Vue 组件或 Vue 的组合式 API（Composition API）来管理这些全局变量的值。
      </p>
     </li>
     <li>
      <p id="">
       <strong>
        使用 CSS 变量
       </strong>
       ：在你的组件中使用这些 CSS 变量来设置样式。
      </p>
     </li>
     <li>
      <p id="">
       <strong>
        动态调整 CSS 变量
       </strong>
       ：通过 Vue 的响应式数据来动态调整这些 CSS 变量的值。
      </p>
     </li>
    </ol>
    <h4>
     1. 定义全局 CSS 变量
    </h4>
    <p id="">
     在你的
     <code>
      main.css
     </code>
     或
     <code>
      App.vue
     </code>
     中的
     <code>
      &lt;style&gt;
     </code>
     标签内定义全局 CSS 变量：
    </p>
    <pre><code class="language-css">/* main.css 或 App.vue 中的 &lt;style&gt; 标签 */
:root {
  --font-size-small: 12px;
  --font-size-medium: 16px;
  --font-size-large: 20px;
  --current-font-size: var(--font-size-medium); /* 默认字号 */
}</code></pre>
    <h4>
     2. 创建 Vue 组件或组合式 API 来管理这些变量
    </h4>
    <p id="">
     在你的 Vue 组件中，使用 Vue 的组合式 API 来管理这些变量的值。例如，在
     <code>
      FontSizeController.vue
     </code>
     中：
    </p>
    <pre><code class="language-javascript">&lt;template&gt;
  &lt;div&gt;
    &lt;button @click="setFontSize('small')"&gt;小&lt;/button&gt;
    &lt;button @click="setFontSize('medium')"&gt;中&lt;/button&gt;
    &lt;button @click="setFontSize('large')"&gt;大&lt;/button&gt;
    &lt;slot&gt;&lt;/slot&gt; &lt;!-- 用于插槽，以便在其他组件中使用 --&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script setup&gt;
import { ref, onMounted, watch } from 'vue';

const fontSizeMap = {
  small: '12px',
  medium: '16px',
  large: '20px'
};

const currentFontSize = ref(window.getComputedStyle(document.documentElement).getPropertyValue('--font-size-medium'));

const setFontSize = (size) =&gt; {
  currentFontSize.value = fontSizeMap[size];
  document.documentElement.style.setProperty('--current-font-size', currentFontSize.value);
};

// 监听 currentFontSize 的变化（可选，用于调试或其他逻辑）
watch(currentFontSize, (newVal) =&gt; {
  console.log('Current font size:', newVal);
});
&lt;/script&gt;

&lt;style scoped&gt;
/* 样式可以根据需要调整 */
button {
  margin: 5px;
}
&lt;/style&gt;</code></pre>
    <h4>
     3. 使用 CSS 变量
    </h4>
    <p id="">
     在你的其他组件中使用这些 CSS 变量。例如，在
     <code>
      MyComponent.vue
     </code>
     中：
    </p>
    <pre><code class="language-javascript">&lt;template&gt;
  &lt;FontSizeController&gt;
    &lt;div class="text"&gt;
      这是一段文本，字号会根据按钮点击而变化。
    &lt;/div&gt;
  &lt;/FontSizeController&gt;
&lt;/template&gt;

&lt;script setup&gt;
&lt;/script&gt;

&lt;style scoped&gt;
.text {
  font-size: var(--current-font-size);
}
&lt;/style&gt;</code></pre>
    <h4>
     4. 动态调整 CSS 变量
    </h4>
    <p id="">
     通过点击
     <code>
      FontSizeController
     </code>
     组件中的按钮，你可以动态调整全局 CSS 变量
     <code>
      --current-font-size
     </code>
     的值，从而影响使用了该变量的组件的样式。
    </p>
