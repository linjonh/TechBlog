    <h2 id="2.%20%E5%AD%97%E5%85%B8" name="2.%20%E5%AD%97%E5%85%B8">
     2. 字典
    </h2>
    <p>
     （Dictionary）是一种内置的数据结构，以
     <a href="https://so.csdn.net/so/search?q=%E9%94%AE%E5%80%BC%E5%AF%B9&amp;spm=1001.2101.3001.7020" title="键值对">
      键值对
     </a>
     （key-value pair）的形式存储数据。在许多Python或PyTorch中常常被使用，例如nn.Sequential等等。因此猫猫有必要写一篇文章，详细讲讲字典的生命全周期。
    </p>
    <h3 id="2.1%20%E5%AD%97%E5%85%B8%E7%9A%84%E5%88%9B%E5%BB%BA" name="2.1%20%E5%AD%97%E5%85%B8%E7%9A%84%E5%88%9B%E5%BB%BA">
     2.1 字典的创建
    </h3>
    <p>
     字典创建的方法本质上就两种：
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       大括号法
      </span>
     </span>
     、
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       dict方法
      </span>
     </span>
    </p>
    <h4 id="2.1.1%20%E5%A4%A7%E6%8B%AC%E5%8F%B7%2B%E7%9B%B4%E6%8E%A5%E8%B5%8B%E5%80%BC" name="2.1.1%20%E5%A4%A7%E6%8B%AC%E5%8F%B7%2B%E7%9B%B4%E6%8E%A5%E8%B5%8B%E5%80%BC">
     2.1.1 大括号+直接赋值
    </h4>
    <p>
     这是最常见和直接的方式，直接使用
     <code>
      {}
     </code>
     包裹键值对：
    </p>
    <pre><code class="language-python">my_dict = {'name': 'Alice', 'age': 25, 'city': 'New York'}
</code></pre>
    <h4 id="2.1.2%C2%A0%E5%A4%A7%E6%8B%AC%E5%8F%B7%2B%E5%AD%97%E5%85%B8%E6%8E%A8%E5%AF%BC%E5%BC%8F" name="2.1.2%C2%A0%E5%A4%A7%E6%8B%AC%E5%8F%B7%2B%E5%AD%97%E5%85%B8%E6%8E%A8%E5%AF%BC%E5%BC%8F">
     2.1.2 大括号+字典推导式
    </h4>
    <p>
     字典推导式允许你根据某种条件或从其他可迭代对象创建字典：
    </p>
    <pre><code class="language-python">my_dict = {x: x**2 for x in range(5)}
# 输出: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
</code></pre>
    <h4 id="2.1.3%C2%A0dict%2B%E5%85%B3%E9%94%AE%E5%AD%97%E5%8F%82%E6%95%B0" name="2.1.3%C2%A0dict%2B%E5%85%B3%E9%94%AE%E5%AD%97%E5%8F%82%E6%95%B0">
     2.1.3 dict+关键字参数
    </h4>
    <p>
     你可以使用
     <code>
      dict()
     </code>
     函数来创建字典，通过关键字参数来初始化：
    </p>
    <pre><code class="language-python">my_dict = dict(name='Alice', age=25, city='New York')
</code></pre>
    <h4 id="2.1.4%C2%A0dict%2B%E5%85%83%E7%BB%84%E5%88%97%E8%A1%A8" name="2.1.4%C2%A0dict%2B%E5%85%83%E7%BB%84%E5%88%97%E8%A1%A8">
     2.1.4 dict+元组列表
    </h4>
    <p>
     你可以使用
     <code>
      dict()
     </code>
     函数来创建字典，通过元组列表来初始化：
    </p>
    <pre><code class="language-python">my_dict = dict([('name', 'Alice'), ('age', 25), ('city', 'New York')])
</code></pre>
    <h4 id="2.1.5%C2%A0dict%2Bzip%E6%96%B9%E6%B3%95" name="2.1.5%C2%A0dict%2Bzip%E6%96%B9%E6%B3%95">
     2.1.5 dict+zip方法
    </h4>
    <p>
     <code>
      zip()
     </code>
     可以将两个可迭代对象组合成一个字典，前一个对象作为键，后一个对象作为值：
    </p>
    <p>
     <strong>
      示例一：
     </strong>
    </p>
    <pre><code class="language-python">keys = ['name', 'age', 'city']
values = ['Alice', 25, 'New York']
my_dict = dict(zip(keys, values))
# 输出: {'name': 'Alice', 'age': 25, 'city': 'New York'}
</code></pre>
    <p>
     <strong>
      示例二：
     </strong>
    </p>
    <pre><code class="language-python">dic = dict(zip('abc', [1, 2, 3]))
print(dic)
# 输出结果：{'a': 1, 'b': 2, 'c': 3}
</code></pre>
    <h4 id="2.1.6%20dict%2Bfromkeys%E6%96%B9%E6%B3%95" name="2.1.6%20dict%2Bfromkeys%E6%96%B9%E6%B3%95">
     2.1.6 dict+fromkeys方法
    </h4>
    <p>
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       通常用来初始化字典, 设置value的默认值
      </span>
     </span>
    </p>
    <pre><code class="language-python">my_dict = dict.fromkeys(['name', 'age', 'city'], None)
# 输出: {'name': None, 'age': None, 'city': None}
</code></pre>
    <h3 id="2.2%20%E5%AD%97%E5%85%B8%E7%9A%84%E8%AE%BF%E9%97%AE" name="2.2%20%E5%AD%97%E5%85%B8%E7%9A%84%E8%AE%BF%E9%97%AE">
     2.2 字典的访问
    </h3>
    <p>
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       字典访问使用的是中括号（[ ])
      </span>
     </span>
    </p>
    <h4 id="2.2.1%C2%A0%E4%BD%BF%E7%94%A8%E9%94%AE%E8%AE%BF%E9%97%AE%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0" name="2.2.1%C2%A0%E4%BD%BF%E7%94%A8%E9%94%AE%E8%AE%BF%E9%97%AE%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0">
     2.2.1 使用键访问字典元素
    </h4>
    <p>
     通过字典的键来访问对应的值。
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       如果键不存在，会抛出
      </span>
      <code>
       <span style="background-color:#fefcd8">
        KeyError
       </span>
      </code>
      <span style="background-color:#fefcd8">
       错误
      </span>
     </span>
     ：
    </p>
    <pre><code class="language-python">my_dict = {'name': 'Alice', 'age': 25, 'city': 'New York'}
# 访问 'name' 键对应的值
print(my_dict['name'])  # 输出: Alice
</code></pre>
    <blockquote>
     <p>
      <span style="color:#fe2c24">
       利用[]中括号来访问键值对的值
      </span>
     </p>
    </blockquote>
    <h4 id="2.2.2%C2%A0%E4%BD%BF%E7%94%A8%C2%A0get()%C2%A0%E6%96%B9%E6%B3%95%E8%AE%BF%E9%97%AE%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0" name="2.2.2%C2%A0%E4%BD%BF%E7%94%A8%C2%A0get()%C2%A0%E6%96%B9%E6%B3%95%E8%AE%BF%E9%97%AE%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0">
     2.2.2 使用
     <code>
      get()
     </code>
     方法访问字典元素
    </h4>
    <p>
     <code>
      get()
     </code>
     方法可以通过键来访问字典的值。如果键不存在，它会返回
     <code>
      None
     </code>
     ，也可以指定一个默认值：
    </p>
    <pre><code class="language-python"># 使用 get() 方法
print(my_dict.get('name'))  # 输出: Alice
# 使用 get() 方法访问不存在的键
print(my_dict.get('gender'))  # 输出: None
# 使用 get() 方法访问不存在的键并设置默认值
print(my_dict.get('gender', 'Not specified'))  # 输出: Not specified
</code></pre>
    <h4 id="2.2.3%C2%A0%E5%85%83%E7%BB%84%E5%88%97%E8%A1%A8%E5%88%99%E7%94%A8%E5%85%83%E7%BB%84%E8%AE%BF%E9%97%AE%E7%9A%84%E6%96%B9%E6%B3%95%E8%AE%BF%E9%97%AE" name="2.2.3%C2%A0%E5%85%83%E7%BB%84%E5%88%97%E8%A1%A8%E5%88%99%E7%94%A8%E5%85%83%E7%BB%84%E8%AE%BF%E9%97%AE%E7%9A%84%E6%96%B9%E6%B3%95%E8%AE%BF%E9%97%AE">
     2.2.3 元组列表则用元组访问的方法访问
    </h4>
    <p>
     字典的输出有时候就是元组形式，因此访问字典可能也就是访问元组：
    </p>
    <pre><code class="language-python">tuple_list = [(1, 'a'), (2, 'b'), (3, 'c')]
print(tuple_list[0])  # 输出: (1, 'a')
print(tuple_list[1])  # 输出: (2, 'b')
print(tuple_list[0][0])  # 输出: 1 (访问第一个元组中的第一个元素)
print(tuple_list[1][1])  # 输出: 'b' (访问第二个元组中的第二个元素)
</code></pre>
    <h3 id="2.3%20%E5%AD%97%E5%85%B8%E7%9A%84%E4%BF%AE%E6%94%B9" name="2.3%20%E5%AD%97%E5%85%B8%E7%9A%84%E4%BF%AE%E6%94%B9">
     2.3 字典的修改
    </h3>
    <h4 id="2.3.1%C2%A0%E7%9B%B4%E6%8E%A5%E4%BF%AE%E6%94%B9%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0" name="2.3.1%C2%A0%E7%9B%B4%E6%8E%A5%E4%BF%AE%E6%94%B9%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0">
     2.3.1 直接修改字典元素
    </h4>
    <p>
     你可以通过直接指定键来修改字典中的值。
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       如果键不存在，会新增该键值对：
      </span>
     </span>
    </p>
    <pre><code class="language-python">my_dict = {'name': 'Alice', 'age': 25, 'city': 'New York'}

# 修改 'age' 键对应的值
my_dict['age'] = 26
print(my_dict)  # 输出: {'name': 'Alice', 'age': 26, 'city': 'New York'}

# 如果键不存在，添加新的键值对
my_dict['gender'] = 'Female'
print(my_dict)  # 输出: {'name': 'Alice', 'age': 26, 'city': 'New York', 'gender': 'Female'}
</code></pre>
    <h4 id="2.3.2%C2%A0%E4%BD%BF%E7%94%A8%C2%A0update()%C2%A0%E6%96%B9%E6%B3%95%E4%BF%AE%E6%94%B9%E5%AD%97%E5%85%B8" name="2.3.2%C2%A0%E4%BD%BF%E7%94%A8%C2%A0update()%C2%A0%E6%96%B9%E6%B3%95%E4%BF%AE%E6%94%B9%E5%AD%97%E5%85%B8">
     2.3.2 使用
     <code>
      update()
     </code>
     方法修改字典
    </h4>
    <p>
     <code>
      update()
     </code>
     方法可以用于修改字典中的元素。你可以传入一个字典或键值对（可以是一个可迭代对象，例如
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       关键字参数
      </span>
     </span>
     或
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       元组列表
      </span>
     </span>
     ）：
    </p>
    <pre><code class="language-python"># 修改现有的键值对
my_dict.update({'age': 27, 'city': 'Los Angeles'})
print(my_dict)  # 输出: {'name': 'Alice', 'age': 27, 'city': 'Los Angeles', 'gender': 'Female'}

# 通过传入键值对的元组列表来修改或添加元素
my_dict.update([('age', 28), ('gender', 'Male')])
print(my_dict)  # 输出: {'name': 'Alice', 'age': 28, 'city': 'Los Angeles', 'gender': 'Male'}
</code></pre>
    <h4 id="2.3.3%C2%A0%E5%88%A0%E9%99%A4%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0" name="2.3.3%C2%A0%E5%88%A0%E9%99%A4%E5%AD%97%E5%85%B8%E5%85%83%E7%B4%A0">
     2.3.3 删除字典元素
    </h4>
    <p>
     使用
     <code>
      del
     </code>
     或
     <code>
      pop()
     </code>
     可以删除字典中的键值对：
    </p>
    <pre><code class="language-python"># 使用 del 删除键值对
del my_dict['city']
print(my_dict)  # 输出: {'name': 'Alice', 'age': 28, 'gender': 'Male', 'country': 'USA'}
</code></pre>
    <pre><code class="language-python"># 使用 pop() 删除并返回值
removed_value = my_dict.pop('age')
print(removed_value)  # 输出: 28
print(my_dict)  # 输出: {'name': 'Alice', 'gender': 'Male', 'country': 'USA'}
</code></pre>
    <h4 id="2.3.4%C2%A0%E6%B8%85%E7%A9%BA%E5%AD%97%E5%85%B8" name="2.3.4%C2%A0%E6%B8%85%E7%A9%BA%E5%AD%97%E5%85%B8">
     2.3.4 清空字典
    </h4>
    <p>
     <code>
      clear()
     </code>
     方法可以清空字典中的所有键值对：
    </p>
    <pre><code class="language-python"># 使用 clear() 清空字典
my_dict.clear()
print(my_dict)  # 输出: {}
</code></pre>
    <h3 id="2.4%20%E5%AD%97%E5%85%B8%E6%96%B9%E6%B3%95" name="2.4%20%E5%AD%97%E5%85%B8%E6%96%B9%E6%B3%95">
     2.4 字典方法
    </h3>
    <h4 id="2.4.1%C2%A0keys()" name="2.4.1%C2%A0keys()">
     2.4.1 keys()
    </h4>
    <p>
     返回字典中所有的键：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
print(my_dict.keys())  # 输出: dict_keys(['a', 'b'])
</code></pre>
    <h4 id="2.4.2%C2%A0values()" name="2.4.2%C2%A0values()">
     2.4.2 values()
    </h4>
    <p>
     返回字典中所有的值：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
print(my_dict.values())  # 输出: dict_values([1, 2])
</code></pre>
    <h4 id="2.4.3%C2%A0update()" name="2.4.3%C2%A0update()">
     2.4.3 update()
    </h4>
    <p>
     更新字典，可以传入另一个字典、键值对或可迭代的键值对序列。
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       如果键已经存在，会更新值；如果键不存在，会添加新键值对
      </span>
     </span>
     ：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
my_dict.update({'b': 3, 'c': 4})
print(my_dict)  # 输出: {'a': 1, 'b': 3, 'c': 4}
</code></pre>
    <h4 id="2.4.4%C2%A0fromkeys()" name="2.4.4%C2%A0fromkeys()">
     2.4.4 fromkeys()
    </h4>
    <p>
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       创建一个新字典，以指定的键创建字典，并为每个键设置默认值
      </span>
     </span>
     。可以传递一个可迭代对象作为键和一个可选的默认值：
    </p>
    <pre><code class="language-python">keys = ['a', 'b', 'c']
new_dict = dict.fromkeys(keys, 0)
print(new_dict)  # 输出: {'a': 0, 'b': 0, 'c': 0}
</code></pre>
    <h4 id="2.4.5%C2%A0setdefault()" name="2.4.5%C2%A0setdefault()">
     2.4.5 setdefault()
    </h4>
    <p>
     如果键不存在于字典中，则将其添加，并将值设置为提供的默认值。如果键已经存在，则不对该键做任何修改：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
print(my_dict.setdefault('c', 3))  # 输出: 3
print(my_dict.setdefault('a', 3))  # 输出: 1
</code></pre>
    <h4 id="2.4.6%C2%A0clear()" name="2.4.6%C2%A0clear()">
     2.4.6 clear()
    </h4>
    <p>
     清空字典中的所有元素：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
my_dict.clear()
print(my_dict)  # 输出: {}
</code></pre>
    <h4 id="2.4.6%C2%A0copy()" name="2.4.6%C2%A0copy()">
     2.4.6 copy()
    </h4>
    <p>
     返回字典的浅拷贝。拷贝后的字典不会影响原字典：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
new_dict = my_dict.copy()
print(new_dict)  # 输出: {'a': 1, 'b': 2}
</code></pre>
    <h4 id="2.4.7%C2%A0items()" name="2.4.7%C2%A0items()">
     2.4.7
     <code>
      items()
     </code>
    </h4>
    <p>
     返回字典中所有键值对，以
     <span style="color:#fe2c24">
      <span style="background-color:#fefcd8">
       元组列表
      </span>
     </span>
     的形式返回。可以用于遍历字典：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
print(my_dict.items())  # 输出: dict_items([('a', 1), ('b', 2)])
</code></pre>
    <h4 id="2.4.8%C2%A0get()" name="2.4.8%C2%A0get()">
     2.4.8 get()
    </h4>
    <p>
     返回指定键的值，如果键不存在，返回
     <code>
      None
     </code>
     （或指定的默认值）：
    </p>
    <pre><code class="language-python">my_dict = {'a': 1, 'b': 2}
print(my_dict.get('a'))  # 输出: 1
print(my_dict.get('c', 'Not Found'))  # 输出: 'Not Found'
</code></pre>
    <h2 id="3.%20%E5%AE%8C%E6%95%B4%E7%A4%BA%E4%BE%8B" name="3.%20%E5%AE%8C%E6%95%B4%E7%A4%BA%E4%BE%8B">
     3. 完整示例
    </h2>
    <p>
     下面是一个完整的 Python 示例，展示了字典的创建、修改、访问以及使用一些常见的字典方法：
    </p>
    <pre><code class="language-python"># 创建字典
student = {
    "name": "Alice",        # 键值对: "name" -&gt; "Alice"
    "age": 20,              # 键值对: "age" -&gt; 20
    "courses": ["Math", "Science"],  # 键值对: "courses" -&gt; ["Math", "Science"]
}

# 访问字典的值
print("Name:", student["name"])  # 输出: Alice
print("Age:", student.get("age"))  # 输出: 20

# 使用 get 方法访问一个不存在的键，提供默认值
print("Gender:", student.get("gender", "Not Specified"))  # 输出: Not Specified

# 修改字典中的值
student["age"] = 21  # 修改 age 的值
print("Updated Age:", student["age"])  # 输出: 21

# 向字典中添加新的键值对
student["address"] = "123 Main St"
print("Address:", student["address"])  # 输出: 123 Main St

# 使用 update() 方法修改多个键值对
student.update({"name": "Bob", "age": 22})
print("Updated Name:", student["name"])  # 输出: Bob
print("Updated Age:", student["age"])  # 输出: 22

# 使用 keys(), values(), items() 方法
print("Keys:", student.keys())  # 输出: dict_keys(['name', 'age', 'courses', 'address'])
print("Values:", student.values())  # 输出: dict_values(['Bob', 22, ['Math', 'Science'], '123 Main St'])
print("Items:", student.items())  # 输出: dict_items([('name', 'Bob'), ('age', 22), ('courses', ['Math', 'Science']), ('address', '123 Main St')])

# 删除字典中的键值对
student.pop("address")  # 删除 "address" 键值对
print("After pop:", student)  # 输出: {'name': 'Bob', 'age': 22, 'courses': ['Math', 'Science']}

# 使用 popitem() 删除并返回字典中的最后一个键值对
last_item = student.popitem()
print("Popped item:", last_item)  # 输出: ('courses', ['Math', 'Science'])

# 清空字典
student.clear()
print("After clear:", student)  # 输出: {}

# 使用 fromkeys 创建一个新字典
new_dict = dict.fromkeys(["a", "b", "c"], 0)
print("New Dictionary:", new_dict)  # 输出: {'a': 0, 'b': 0, 'c': 0}
</code></pre>
    <ol>
     <li>
      <strong>
       字典创建
      </strong>
      ：我们使用大括号
      <code>
       {}
      </code>
      来创建字典，并指定键值对。字典的键是唯一的，值可以是任何数据类型。
     </li>
     <li>
      <strong>
       访问字典的值
      </strong>
      ：我们通过
      <code>
       dict[key]
      </code>
      或
      <code>
       dict.get(key)
      </code>
      来访问字典中的值。
      <code>
       get()
      </code>
      方法可以提供默认值以避免键不存在时报错。
     </li>
     <li>
      <strong>
       修改字典的值
      </strong>
      ：通过
      <code>
       dict[key] = value
      </code>
      可以修改字典中的值。
     </li>
     <li>
      <strong>
       添加新的键值对
      </strong>
      ：使用相同的方式通过键来添加新的键值对。
     </li>
     <li>
      <strong>
       使用
       <code>
        update()
       </code>
       方法
      </strong>
      ：可以批量更新字典中的多个键值对。
     </li>
     <li>
      <strong>
       遍历字典
      </strong>
      ：通过
      <code>
       keys()
      </code>
      获取所有的键，
      <code>
       values()
      </code>
      获取所有的值，
      <code>
       items()
      </code>
      获取所有的键值对。
     </li>
     <li>
      <strong>
       删除字典元素
      </strong>
      ：使用
      <code>
       pop()
      </code>
      删除指定的键值对，
      <code>
       popitem()
      </code>
      删除并返回最后一个键值对。
     </li>
     <li>
      <strong>
       清空字典
      </strong>
      ：
      <code>
       clear()
      </code>
      方法会删除字典中的所有元素。
     </li>
     <li>
      <strong>
       创建新字典
      </strong>
      ：
      <code>
       fromkeys()
      </code>
      方法通过指定的键创建新字典，并可以为每个键指定一个默认值。
     </li>
    </ol>
