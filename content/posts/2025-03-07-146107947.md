    <h4>
     <a id="1_jsondumpjsondumps_23">
     </a>
     1. json.dump()和json.dumps()
    </h4>
    <p>
     用于对python对象（字典、列表等）
     <strong>
      序列化
     </strong>
     为JSON格式的数据
    </p>
    <h5>
     <a id="11_jsondump_27">
     </a>
     1.1 json.dump()
    </h5>
    <p>
     用于将Python对象（如字典、列表）直接序列化为JSON格式数据，并写入‌文件‌，需指定文件句柄作为参数‌。
    </p>
    <pre><code class="prism language-python">data <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token string">"name"</span><span class="token punctuation">:</span> <span class="token string">"张三"</span><span class="token punctuation">,</span> <span class="token string">"age"</span><span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
<span class="token keyword">with</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">'data.json'</span><span class="token punctuation">,</span> <span class="token string">'w'</span><span class="token punctuation">)</span> <span class="token keyword">as</span> f<span class="token punctuation">:</span> 
    json<span class="token punctuation">.</span>dump<span class="token punctuation">(</span>data<span class="token punctuation">,</span> f<span class="token punctuation">)</span>
</code></pre>
    <h5>
     <a id="11_jsondumps_37">
     </a>
     1.1 json.dumps()
    </h5>
    <p>
     将Python对象序列化为JSON格式的‌字符串‌。
    </p>
    <pre><code class="prism language-python">data <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token string">"name"</span><span class="token punctuation">:</span> <span class="token string">"张三"</span><span class="token punctuation">,</span> <span class="token string">"age"</span><span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
json_str <span class="token operator">=</span> json<span class="token punctuation">.</span>dumps<span class="token punctuation">(</span>data<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="2_jsonloadjsonloads_47">
     </a>
     2. json.load()和json.loads()
    </h4>
    <p>
     将JSON格式的数据
     <strong>
      反序列化
     </strong>
     为Python对象（如字典或列表）
    </p>
    <h5>
     <a id="21_jsonload_50">
     </a>
     2.1 json.load()
    </h5>
    <p>
     用于从
     <strong>
      ‌文件对象
     </strong>
     ‌中读取JSON数据，并将其反序列化为Python对象（如字典或列表）。需通过open()等操作获取
     <strong>
      文件句柄
     </strong>
     作为参数‌。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">with</span> <span class="token builtin">open</span><span class="token punctuation">(</span><span class="token string">'data.json'</span><span class="token punctuation">,</span> <span class="token string">'r'</span><span class="token punctuation">)</span> <span class="token keyword">as</span> f<span class="token punctuation">:</span>  <span class="token comment"># r表示只读，使用dump时如果参数为r会报错</span>
    data <span class="token operator">=</span> json<span class="token punctuation">.</span>load<span class="token punctuation">(</span>f<span class="token punctuation">)</span>  <span class="token comment"># 从文件读取并解析为Python对象</span>
</code></pre>
    <h5>
     <a id="22_jsonloads_59">

     2.2. json.loads()
    </h5>
    <p>
     用于将‌JSON格式的字符串‌反序列化为Python对象，其参数是

    <pre><code class="prism language-python">json_str <span class="token operator">=</span> <span class="token string">'{"name": "李四", "age": 25}'</span>

    <h4>
     <a id="4__76">
     </a>
     4. 区分
    </h4>
    <p>
     为了便于区分，可以从函数命名理解。
    </p>
    <ul>
     <li>
      将
      <code>
       loads
      </code>
      理解为
      <code>
       load str
      </code>
      的缩写，也就是加载（反序列化）字符串，
     </li>
     <li>
      将
      <code>
       dumps
      </code>
      理解为
      <code>
       dump st
      </code>
      r的缩写，表示将 Python 对象序列化为‌字符串。
     </li>
     <li>
      不包含
      <code>
       s
      </code>
      的是对文件进行操作。
     </li>
    </ul>
