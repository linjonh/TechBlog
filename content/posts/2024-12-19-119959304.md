    <h2>
     <a id="_3">
     </a>
     前言
    </h2>
    <p>
     国内部分车厂会用到arxml格式的数据库，有时为了做二次开发工作，需要用到数据库的各个signals，由于存在的分支节点太多，不利用调用；将数据库转换成dbc或者excel格式能够更加方便我们去调用，例如Labview，Python都有现成的模块调用；
    </p>
    <h2>
     <a id="_5">
     </a>
     一、工具安装；
    </h2>
    <p>
     1、
     <a href="http://www.winwin7.com/soft/xtgj-8239.html" rel="nofollow">
      点击下载安装
     </a>
     --python 3.8工具
     <br/>
     2、
     <a href="https://github.com/ebroecker/canmatrix">
      点击下载安装
     </a>
     --Canmatrix工具
    </p>
    <h2>
     <a id="_8">
     </a>
     二、操作步骤；
    </h2>
    <h3>
     <a id="1python_9">
     </a>
     1、配置python环境变量；
    </h3>
    <p>
     我的电脑→右键属性→高级系统设置→环境变量→系统变量→选择path→编辑→新建→复制自己python文件安装路径→确认
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a96b7f29e0b6b8518229b33c347168b0.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/365f7d96261b526d59f5bfcf3feda478.png"/>
    </p>
    <blockquote>
     <p>
      综上，已完成python的环境变量配置；
     </p>
    </blockquote>
    <h3>
     <a id="2Canmatrix_17">
     </a>
     2、安装Canmatrix工具；
    </h3>
    <p>
     <strong>
      PS：直接在Dos窗口输入命令就好了，不用切换到特定路径
     </strong>
    </p>
    <p>
     1、打开命令窗口DOS，在命令行输入
     <strong>
      pip install canmatrix
     </strong>
     ，安装canmatrix；完成后会在/python/script文件路径下生成canconvert和cancompare（用于转换和比较数据库），并且会提醒pip插件版本太旧；
    </p>
    <p>
     2、在命令窗口输入
     <strong>
      python -m pip install --upgrade pip
     </strong>
     （更新pip插件，否则无法继续安装其他python插件）
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2fc93d89711f7ad2f3abd526a3db598a.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/098ef44c3c43e2720ecaebc4f84e68f1.png"/>
     </img>
    </p>
    <h3>
     <a id="3python_27">
     </a>
     3、安装python功能插件；
    </h3>
    <p>
     1、安装完
     <strong>
      catmatrix
     </strong>
     后不能直接进行arxml、dbc和excel之间的转换，运行canmatrix会提醒缺少部分pyhton插件；
     <br/>
     2、请依次在命令窗口输入以下命令（逐个安装）
     <br/>
     1）
     <strong>
      pip install XlsxWriter
     </strong>
     ----- 解决Xlsx
     <br/>
     2）
     <strong>
      pip install pyaml
     </strong>
     -----解决yaml
     <br/>
     3）
     <strong>
      pip install lxml
     </strong>
     -----解决arxml、kcd、fibex
     <br/>
     4）
     <strong>
      pip install xlwt
     </strong>
     -----解决xls
     <br/>
     5）
     <strong>
      pip install xlrd
     </strong>
     -----解决读取旧版Excel数据
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/669e7b2fd2834a099d325c3bad6f95a7.png"/>
    </p>
    <blockquote>
     <p>
      发送上述命令后python插件缺少问题将会解决；
     </p>
    </blockquote>
    <h3>
     <a id="4Canmatrixarxmldbc_39">
     </a>
     4、使用Canmatrix将arxml转换dbc；
    </h3>
    <p>
     <strong>
      PS：目标文件不需要模板，如下述target.dbc
     </strong>
     ；
    </p>
    <p>
     1、发送命令将arxml转换成dbc：
     <strong>
      canconvert D:\Desktop\Test\target.arxml D:\Desktop\Test\tatget.dbc
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d76f828821c4d91988bd287c1bc155fc.png"/>
    </p>
    <blockquote>
     <p>
      综上，已完成arxml到dbc的转换，同时也支持arxml、dbc、excel格式之间的转换；
     </p>
    </blockquote>
