    <h3>
     <a id="KEPServerEX6OPCUA_2">
     </a>
     采用：KEPServerEX6获取OPCUA服务数据
    </h3>
    <p>
     1.新建甬道3
     <br/>
     2.驱动程序选择：OPC UA client
     <br/>
     3.设置UA服务器-端点URL：opc.tcp://ip:502
     <br/>
     安装教程：https://blog.csdn.net/weixin_41425107/article/details/140861485
    </p>
    <h3>
     <a id="noderedS7OPCUA_10">
     </a>
     采用nodered采集S7协议/OPCUA服务
    </h3>
    <p>
     1.安装nodered
     <br/>
     参考：https://www.cnblogs.com/yaopengfei/p/17159691.html
     <br/>
     2.安装opcua节点
     <br/>
     参考：https://blog.csdn.net/SugarToffee/article/details/138231603
     <br/>
     3.安装S7
     <br/>
     参考：https://blog.csdn.net/dongfangxinding/article/details/135263743
     <br/>
     4.java程序采集S7
     <br/>
     参考：https://blog.csdn.net/XS_YOUYOU/article/details/127775881
    </p>
    <h3>
     <a id="_20">
     </a>
     部署
    </h3>
    <p>
     node-red
     <br/>
     node-v18.15.0-x64
     <br/>
     双击node-v18.15.0-x64安装
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/38868c4d818f43768bf9707755560e56.png"/>
    </p>
    <p>
     安装PM2
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/705d92bab40c45b5b03cde6e4198cb5a.png"/>
    </p>
    <p>
     使用命令启动nodered
    </p>
    <pre><code>pm2 start c:\iot\node-red\node-red\packages\node_modules\node-red\red.js
</code></pre>
    <p>
     使用命令保存当前启动任务，关机重启
    </p>
    <pre><code>pm2 save
</code></pre>
    <p>
     node-red访问地址
    </p>
    <pre><code>http://IP:1880
</code></pre>
