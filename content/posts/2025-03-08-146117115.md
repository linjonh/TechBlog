     Windows下安装kafka
    </h1>

     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     在 Windows 系统下安装 Kafka 可以按照以下步骤进行：
    </p>
    <h4>
     <a id="1__Java__2">
     </a>
     1. 安装 Java 环境
    </h4>
    <p>
     Kafka 是基于 Java 开发的，因此需要先安装 Java 环境。
    </p>
    <ul>
     <li>
      <strong>
       下载 Java
      </strong>
      ：访问
      <a href="https://www.oracle.com/java/technologies/javase-downloads.html" rel="nofollow">
       Oracle Java 下载页面
      </a>
      或
      <a href="https://jdk.java.net/" rel="nofollow">
       OpenJDK 下载页面
      </a>
      ，选择适合你系统的 Java 版本（建议 Java 8 及以上）进行下载。
     </li>
     <li>
      <strong>
       安装 Java
      </strong>
      ：运行下载的安装程序，按照提示完成安装。
     </li>
     <li>
      <strong>
       配置环境变量
      </strong>
      ：
      <ul>
       <li>
        右键点击“此电脑”，选择“属性”。
       </li>
       <li>
        点击“高级系统设置”，在弹出的窗口中点击“环境变量”。
       </li>
       <li>
        在“系统变量”中找到“Path”变量，点击“编辑”，将 Java 的
        <code>
         bin
        </code>
        目录路径（例如
        <code>
         C:\Program Files\Java\jdk-11\bin
        </code>
        ）添加到“Path”变量中。
       </li>
       <li>
        新建“JAVA_HOME”系统变量，变量值为 Java 的安装目录（例如
        <code>
         C:\Program Files\Java\jdk-11
        </code>
        ）。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="2__Kafka_13">
     </a>
     2. 下载 Kafka
    </h4>
    <ul>
     <li>
      访问
      <a href="https://kafka.apache.org/downloads" rel="nofollow">
       Apache Kafka 官方下载页面
      </a>
      ，选择合适的 Kafka 版本（建议选择最新的稳定版本），下载
      <code>
       Binary downloads
      </code>
      下的压缩包。
     </li>
     <li>
      将下载的压缩包解压到你想要安装 Kafka 的目录，例如
      <code>
       C:\kafka
      </code>
      。
     </li>
    </ul>
    <h4>
     <a id="3__Zookeeper_17">
     </a>
     3. 配置 Zookeeper
    </h4>
    <p>
     Kafka 依赖 Zookeeper 来管理集群元数据和协调节点。Kafka 自带了 Zookeeper，你可以直接使用。
    </p>
    <ul>
     <li>
      打开
      <code>
       C:\kafka\config
      </code>
      目录，找到
      <code>
       zookeeper.properties
      </code>
      文件。
     </li>
     <li>
      通常情况下，你可以使用默认配置，不需要进行修改。
     </li>
    </ul>
    <h4>
     <a id="4__Kafka_23">
     </a>
     4. 配置 Kafka
    </h4>
    <ul>
     <li>
      打开
      <code>
       C:\kafka\config
      </code>
      目录，找到
      <code>
       server.properties
      </code>
      文件。
     </li>
     <li>
      可以根据需要修改以下配置：
      <ul>
       <li>
        <code>
         log.dirs
        </code>
        ：指定 Kafka 消息日志的存储目录，例如
        <code>
         log.dirs=C:/kafka/kafka-logs
        </code>
        。
       </li>
       <li>
        <code>
         listeners
        </code>
        ：指定 Kafka 监听的地址和端口，默认是
        <code>
         PLAINTEXT://:9092
        </code>
        。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="5__Zookeeper_29">
     </a>
     5. 启动 Zookeeper
    </h4>
    <ul>
     <li>
      打开命令提示符（CMD），切换到 Kafka 的安装目录：
     </li>
    </ul>
    <pre><code class="prism language-sh"><span class="token builtin class-name">cd</span> C:<span class="token punctuation">\</span>kafka
</code></pre>
    <ul>
     <li>
      启动 Zookeeper 服务：
     </li>
    </ul>
    <pre><code class="prism language-sh">bin<span class="token punctuation">\</span>windows<span class="token punctuation">\</span>zookeeper-server-start.bat config<span class="token punctuation">\</span>zookeeper.properties
</code></pre>
    <p>
     如果一切正常，你会看到 Zookeeper 成功启动的日志信息。
    </p>
    <h4>
     <a id="6__Kafka_40">
     </a>
     6. 启动 Kafka
    </h4>
    <ul>
     <li>
      打开另一个命令提示符（CMD），切换到 Kafka 的安装目录：
     </li>
    </ul>
    <pre><code class="prism language-sh"><span class="token builtin class-name">cd</span> C:<span class="token punctuation">\</span>kafka
</code></pre>
    <ul>
     <li>
      启动 Kafka 服务：
     </li>
    </ul>
    <pre><code class="prism language-sh">bin<span class="token punctuation">\</span>windows<span class="token punctuation">\</span>kafka-server-start.bat config<span class="token punctuation">\</span>server.properties
</code></pre>
    <p>
     如果启动成功，你会看到 Kafka 服务的启动日志。
    </p>
    <h4>
     <a id="7__51">
     </a>
     7. 验证安装
    </h4>
    <ul>
     <li>
      <strong>
       创建主题
      </strong>
      ：打开一个新的命令提示符，切换到 Kafka 安装目录，执行以下命令创建一个名为
      <code>
       test
      </code>
      的主题：
     </li>
    </ul>
    <pre><code class="prism language-sh">bin<span class="token punctuation">\</span>windows<span class="token punctuation">\</span>kafka-topics.bat <span class="token parameter variable">--create</span> --bootstrap-server localhost:9092 --replication-factor <span class="token number">1</span> <span class="token parameter variable">--partitions</span> <span class="token number">1</span> <span class="token parameter variable">--topic</span> <span class="token builtin class-name">test</span>
</code></pre>
    <ul>
     <li>
      <strong>
       发送消息
      </strong>
      ：执行以下命令启动一个生产者，向
      <code>
       test
      </code>
      主题发送消息：
     </li>
    </ul>
    <pre><code class="prism language-sh">bin<span class="token punctuation">\</span>windows<span class="token punctuation">\</span>kafka-console-producer.bat --bootstrap-server localhost:9092 <span class="token parameter variable">--topic</span> <span class="token builtin class-name">test</span>
</code></pre>
    <p>
     在命令行中输入一些消息，按回车键发送。
    </p>
    <ul>
     <li>
      <strong>
       接收消息
      </strong>
      ：打开另一个命令提示符，切换到 Kafka 安装目录，执行以下命令启动一个消费者，接收
      <code>
       test
      </code>
      主题的消息：
     </li>
    </ul>
    <pre><code class="prism language-sh">bin<span class="token punctuation">\</span>windows<span class="token punctuation">\</span>kafka-console-consumer.bat --bootstrap-server localhost:9092 <span class="token parameter variable">--topic</span> <span class="token builtin class-name">test</span> --from-beginning
</code></pre>
    <p>
     如果你能在消费者的命令行中看到之前生产者发送的消息，说明 Kafka 安装和配置成功。
    </p>
    <h4>
     <a id="8__68">
     </a>
     8. 关闭服务
    </h4>
    <p>
     当你不再需要使用 Kafka 时，可以按以下步骤关闭服务：
    </p>
    <ul>
     <li>
      先关闭 Kafka 服务：在 Kafka 服务的命令提示符窗口中，按
      <code>
       Ctrl + C
      </code>
      停止服务。
     </li>
     <li>
      再关闭 Zookeeper 服务：在 Zookeeper 服务的命令提示符窗口中，按
      <code>
       Ctrl + C
      </code>
      停止服务。
     </li>
    </ul>
    <p>
     通过以上步骤，你就可以在 Windows 系统下成功安装和使用 Kafka。
    </p>
