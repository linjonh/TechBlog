     电脑如何拦截端口号，实现阻断访问？
    </h1>
   </div>

 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <blockquote>
     <p>
      如果你弟弟喜欢玩游戏，你可以查询该应用占用的端口，结合以下方法即可阻断端口号，让弟弟好好学习，天天向上！
     </p>
    </blockquote>
    <p>
     拦截端口可以通过防火墙和路由器进行拦截 ，以下是常用方法：
    </p>
    <p>
     <strong>
      方法 1：使用 Windows 防火墙拦截端口
     </strong>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开防火墙
       </strong>
      </p>
      <ul>
       <li>
        <p>
         按
         <strong>
          Win + R
         </strong>
         ，输入
         <code>
          wf.msc
         </code>
         ，回车打开
         <strong>
          Windows Defender 防火墙
         </strong>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        创建出站规则
       </strong>
       （阻止外发连接）
      </p>
      <ul>
       <li>
        <p>
         点击
         <strong>
          “出站规则”
         </strong>
         &gt;
         <strong>
          “新建规则”
         </strong>
         。
        </p>
       </li>
       <li>
        <p>
         选择
         <strong>
          “端口”
         </strong>
         &gt;
         <strong>
          “下一步”
         </strong>
         。
        </p>
       </li>
       <li>
        <p>
         选择
         <strong>
          “TCP” 或 “UDP”
         </strong>
         ，然后输入要拦截的端口（如
         <code>
          8393-8400, 5000-5500
         </code>
         ）。
        </p>
       </li>
       <li>
        <p>
         选择
         <strong>
          “阻止连接”
         </strong>
         &gt; 适用于
         <strong>
          所有网络
         </strong>
         &gt;
         <strong>
          命名规则并完成
         </strong>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        创建入站规则
       </strong>
       （阻止进入连接）
      </p>
      <ul>
       <li>
        <p>
         同上，在
         <strong>
          “入站规则”
         </strong>
         里重复相同步骤。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     <img alt="" height="579" src="https://i-blog.csdnimg.cn/direct/e4f4e6ff19d84af4b359865270a42dd7.png" width="1305"/>
    </p>
    <p>
    </p>
    <p>
     <strong>
      方法 2：使用路由器拦截端口
     </strong>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        登录路由器后台
       </strong>
       （通常是
       <code>
        192.168.1.1
       </code>
       或
       <code>
        192.168.0.1
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       找到
       <strong>
        “防火墙” 或 “安全设置”
       </strong>
       选项。
      </p>
     </li>
     <li>
      <p>
       选择
       <strong>
        “端口过滤” 或 “端口封锁”
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       添加要拦截的端口（如
       <code>
        8393-8400, 5000-5500
       </code>
       ），然后保存。
      </p>
     </li>
     <li>
      <p>
       重启路由器，使设置生效。
      </p>
     </li>
    </ol>
