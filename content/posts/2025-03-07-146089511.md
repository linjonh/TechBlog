# 电脑如何拦截端口号，实现阻断访问？

> 如果你弟弟喜欢玩游戏，你可以查询该应用占用的端口，结合以下方法即可阻断端口号，让弟弟好好学习，天天向上！

拦截端口可以通过防火墙和路由器进行拦截 ，以下是常用方法：

**方法 1：使用 Windows 防火墙拦截端口**

1. **打开防火墙**

   * 按
     **Win + R**
     ，输入
     `wf.msc`
     ，回车打开
     **Windows Defender 防火墙**
     。
2. **创建出站规则**
   （阻止外发连接）

   * 点击
     **“出站规则”**
     >
     **“新建规则”**
     。
   * 选择
     **“端口”**
     >
     **“下一步”**
     。
   * 选择
     **“TCP” 或 “UDP”**
     ，然后输入要拦截的端口（如
     `8393-8400, 5000-5500`
     ）。
   * 选择
     **“阻止连接”**
     > 适用于
     **所有网络**
     >
     **命名规则并完成**
     。
3. **创建入站规则**
   （阻止进入连接）

   * 同上，在
     **“入站规则”**
     里重复相同步骤。

![](https://i-blog.csdnimg.cn/direct/e4f4e6ff19d84af4b359865270a42dd7.png)

**方法 2：使用路由器拦截端口**

1. **登录路由器后台**
   （通常是
   `192.168.1.1`
   或
   `192.168.0.1`
   ）。
2. 找到
   **“防火墙” 或 “安全设置”**
   选项。
3. 选择
   **“端口过滤” 或 “端口封锁”**
   。
4. 添加要拦截的端口（如
   `8393-8400, 5000-5500`
   ），然后保存。
5. 重启路由器，使设置生效。
