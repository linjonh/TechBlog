    <p>
     Java泛型是Java 5引入的一项重要特性，旨在增强类型安全、减少代码冗余，并支持更灵活的代码设计。以下是对泛型的详细介绍及使用指南：
    </p>
    <hr/>
    <h4>
     <a id="_4">
     </a>
     <strong>
      一、泛型核心概念
     </strong>
    </h4>
    <p>
     泛型允许在
     <strong>
      类、接口、方法
     </strong>
     中使用
     <strong>
      类型参数
     </strong>
     （如
     <code>
      &lt;T&gt;
     </code>
     ），使得代码可以处理多种数据类型，而无需重复编写逻辑。
    </p>
    <h5>
     <a id="_7">
     </a>
     <strong>
      解决的问题
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       类型安全
      </strong>
      ：避免运行时
      <code>
       ClassCastException
      </code>
      。
     </li>
     <li>
      <strong>
       消除强制类型转换
      </strong>
      ：编译器自动处理类型转换。
     </li>
     <li>
      <strong>
       代码复用
      </strong>
      ：同一逻辑可处理不同类型的数据。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_14">
     </a>
     <strong>
      二、泛型的基本使用
     </strong>
    </h4>

     <a id="1__16">
     </a>
     <strong>

     </strong>

     通过
     <code>

    <h4>
     <a id="_67">
     </a>
     <strong>
      三、类型通配符
     </strong>
    </h4>
    <p>
     用于增强泛型的灵活性，处理未知类型。
    </p>
    <h5>
     <a id="1___70">
     </a>
     <strong>
      1. 无界通配符
      <code>
       &lt;?&gt;
      </code>
     </strong>
    </h5>
    <p>
     表示任意类型，常用于只读操作。
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">void</span> <span class="token function">printList</span><span class="token punctuation">(</span><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token operator">?</span><span class="token punctuation">&gt;</span></span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">for</span> <span class="token punctuation">(</span><span class="token class-name">Object</span> item <span class="token operator">:</span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span>item<span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="2___extends_T_80">
     </a>
     <strong>
      2. 上界通配符
      <code>
       &lt;? extends T&gt;
      </code>
     </strong>
    </h5>
    <p>
     接受
     <code>
      T
     </code>
     及其子类，适合读取数据（Producer）。
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">double</span> <span class="token function">sum</span><span class="token punctuation">(</span><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token operator">?</span> <span class="token keyword">extends</span> <span class="token class-name">Number</span><span class="token punctuation">&gt;</span></span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">double</span> sum <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
    <span class="token keyword">for</span> <span class="token punctuation">(</span><span class="token class-name">Number</span> num <span class="token operator">:</span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        sum <span class="token operator">+=</span> num<span class="token punctuation">.</span><span class="token function">doubleValue</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="3___super_T_92">
     </a>
     <strong>
      3. 下界通配符
      <code>
       &lt;? super T&gt;
      </code>
     </strong>
    </h5>
    <p>
     接受
     <code>
      T
     </code>
     及其父类，适合写入数据（Consumer）。
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">void</span> <span class="token function">addNumbers</span><span class="token punctuation">(</span><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token operator">?</span> <span class="token keyword">super</span> <span class="token class-name">Integer</span><span class="token punctuation">&gt;</span></span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    list<span class="token punctuation">.</span><span class="token function">add</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    list<span class="token punctuation">.</span><span class="token function">add</span><span class="token punctuation">(</span><span class="token number">2</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="PECSProducer_Extends_Consumer_Super_101">
     </a>
     <strong>
      PECS原则（Producer Extends, Consumer Super）
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       生产者（读取数据）
      </strong>
      ：使用
      <code>
       &lt;? extends T&gt;
      </code>
      。
     </li>
     <li>
      <strong>
       消费者（写入数据）
      </strong>
      ：使用
      <code>
       &lt;? super T&gt;
      </code>
      。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_107">
     </a>
     <strong>
      四、类型擦除与限制
     </strong>
    </h4>
    <p>
     Java泛型通过
     <strong>
      类型擦除
     </strong>
     实现，编译时擦除泛型信息，替换为原生类型（如
     <code>
      Object
     </code>
     ）或边界类型。
    </p>
    <h5>
     <a id="_110">
     </a>
     <strong>
      关键限制
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       无法实例化类型参数
      </strong>
      ：
      <code>
       new T()
      </code>
      是非法的。
     </li>
     <li>
      <strong>
       不能使用基本类型
      </strong>
      ：如
      <code>
       List&lt;int&gt;
      </code>
      需改为
      <code>
       List&lt;Integer&gt;
      </code>
      。
     </li>
     <li>
      <strong>
       静态上下文限制
      </strong>
      ：静态方法/变量不能引用类的类型参数。
     </li>
     <li>
      <strong>
       泛型数组问题
      </strong>
      ：不能直接创建泛型数组（如
      <code>
       new T[10]
      </code>
      ）。
     </li>
    </ul>

    <h4>
     <a id="_140">
     </a>
     <strong>
      六、应用场景
     </strong>
    </h4>

     <li>
      <strong>

      </strong>

       回调机制

      接口。

    </ul>
