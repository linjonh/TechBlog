    <h2>
     <a id="1YUV_0">
     </a>
     1、提取YUV数据
    </h2>
    <ul>
     <li>
      提取yuv数据，并保持分辨率与原视频一致
     </li>
    </ul>
    <p>
     使用-pix_fmt或-pixel_format指定yuv格式提取数据，并保持原来的分辨率
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> music.mp4 <span class="token parameter variable">-t</span> <span class="token string">"01:00"</span> <span class="token parameter variable">-pixel_format</span> yuv420p music.yuv
</code></pre>
    <p>
     提取成功后，可以使用ffplay指定yuv数据的帧率、分辨率来播放：
    </p>
    <pre><code class="prism language-shell">ffplay .<span class="token punctuation">\</span>music.yuv <span class="token parameter variable">-framerate</span> <span class="token number">30</span> <span class="token parameter variable">-video_size</span> 1920x1080
</code></pre>
    <ul>
     <li>
      提取视频并且改变分辨率，比如改变分辨率为1280x720
     </li>
    </ul>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> music.mp4 <span class="token parameter variable">-pix_fmt</span> yuv420p <span class="token parameter variable">-s</span> 1280x720 music.yuv
</code></pre>
    <p>
     使用ffplay播放：
    </p>
    <pre><code class="prism language-shell">ffplay .<span class="token punctuation">\</span>music.yuv <span class="token parameter variable">-framerate</span> <span class="token number">30</span> <span class="token parameter variable">-video_size</span> 1280x720
</code></pre>
    <p>
     效果如下：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cb2a9cd3999846e4813039f14c49c7c8.gif"/>
    </p>
    <ul>
     <li>
      提取RGB数据
     </li>
    </ul>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> music.mp4 <span class="token parameter variable">-pix_fmt</span> rgb24 <span class="token parameter variable">-s</span> 1280x720 music.rgb
</code></pre>
    <p>
     使用ffplay播放rgb数据，如果指定为yuv就会出现花屏
    </p>
    <pre><code class="prism language-shell">ffplay .<span class="token punctuation">\</span>music.rgb <span class="token parameter variable">-pixel_format</span> rgb24 <span class="token parameter variable">-framerate</span> <span class="token number">30</span> <span class="token parameter variable">-video_size</span> 1280x720
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d4419aaba9fe4fa182247b1467dba3aa.gif"/>
    </p>
    <ul>
     <li>
      rgb与yuv互转
     </li>
    </ul>
    <p>
     指定输入源格式和输出源格式即可实现格式互转，如把rgb转换为yuv：
    </p>
    <p>
     <em>
      注意这里一开始需要给定输入源的帧率和分辨率、像素格式
     </em>
    </p>
    <pre><code class="prism language-shell">ffmpeg  <span class="token parameter variable">-pixel_format</span> rgb24 <span class="token parameter variable">-framerate</span> <span class="token number">30</span> <span class="token parameter variable">-s</span> 1920x1080 <span class="token parameter variable">-i</span> music.rgb <span class="token parameter variable">-pixel_format</span> yuv420p music_rgbtoyuv.yuv 
</code></pre>
    <p>
     使用ffplay播放即可，如果指定播放格式不对，就会出现各种问题
    </p>
    <pre><code class="prism language-shell">ffplay music_rgbtoyuv.yuv <span class="token parameter variable">-pixel_format</span> yuv420p <span class="token parameter variable">-video_size</span> 1920x1080
</code></pre>
    <p>
     如出现花屏、多个画面等问题
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3e9f27f1360343ae9ffc65232b5100cf.gif"/>
    </p>
    <h2>
     <a id="pcm_63">
     </a>
     二、提取pcm数据
    </h2>
    <p>
     提取pcm数据需要指定采样率、采样通道、音频格式等数据
    </p>
    <p>
     这个命令就是把mp3文件中的pcm数据采集出来，需要指定adpcm_ima_alp 编解码器
    </p>
    <pre><code class="prism language-shell"> ffmpeg <span class="token parameter variable">-i</span> music.mp3 <span class="token parameter variable">-codec</span> adpcm_ima_alp <span class="token parameter variable">-ar</span> <span class="token number">44</span>.1k <span class="token parameter variable">-ch_layout</span> stereo music.pcm
</code></pre>
