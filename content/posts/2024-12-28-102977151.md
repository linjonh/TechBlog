    <p>
     一、本人安装的是java13 和java8 。以这两个为例子。
     <br/>
     二、首先本人萌新 之前没安装过java ，首次用的brew安装java推荐的版本 java13
     <br/>
     三、homebrew brew是很好的工具
     <br/>
     1.卸载 brew
     <br/>
     ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall)”
    </p>
    <p>
     2.安装 brew
     <br/>
     /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3f7f9b81c7fb7b28070ddff139991f96.png">
      出现这个代表安装成功。
      <br/>
      2.安装brew cask
      <br/>
      brew install brew-cask-completion
      <br/>
      四、安装java
      <br/>
      1.首先安装的是brew推荐安装java13
      <br/>
      brew search java
      <br/>
      brew info java
      <br/>
      brew install java
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ebd421968bdb9429ba633fd1f3d61055.png">
       非常的简单
       <br/>
       2.安装java8
       <br/>
       下载包所有版本
       <br/>
       brew tap AdoptOpenJDK/openjdk
       <br/>
       下载Java8
       <br/>
       brew cask install adoptopenjdk8
       <br/>
       或者java9
       <br/>
       brew cask install adoptopenjdk9
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/40fe84818f4c568a4bf4cf4ca2ea4454.png">
        mac需要输入密码 出现这个安装完成！！！
        <br/>
        还没完 继续看
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/288141fc1c7f08440dd483e4dfcb7b66.png">
         此时看/Library/Java/JavaVirtualMachines有两个包 一个java8 一个java13
         <br/>
         命令行打开文件
         <br/>
         sudo vim /etc/profile
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7d9d85ca720a1c504980df799c8d9422.png">
          我把绝对路径放在里边 用jdk13和jdk8表示
          <br/>
          保存并推出，刷新文件：
          <br/>
          source /etc/profile
          <br/>
          可以清楚的看到改变版本
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/da1ad642c004f8cfe60a31ef19183d18.png">
           <br/>
           到此就为止了Java mac系统 双版本安装完成 。 本人也是萌新 ，写的不对高手勿喷，留言即可。
