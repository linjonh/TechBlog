# 为 AI 让路，Windows 12 将推迟至 2025 年发布

![058d9c1f793855781741f1a54bb6f0d5.png](https://i-blog.csdnimg.cn/blog_migrate/7c81c08f3461343d199977cf872fb811.png)

近日，微软正在对其 Windows 操作系统的发布策略进行重大调整。根据 Windows Central 爆料，微软将推迟原计划于今年发布的 Windows 12，转而专注于对现有 Windows 11 的重大更新。预计Windows 12 的登场可能将推迟到 2025 年。

![5e7e5a2e2369d0f78ddaa03f8337b5e2.jpeg](https://i-blog.csdnimg.cn/blog_migrate/65da5823988d13c31588eb11700a5aac.jpeg)

我们来看一个问题，Windows 11 能否称得上是一代成功的系统？

从微软的角度来看，Windows 11是微软的核心产品，肩负着重大的使命。在Windows 11发布后，微软的市值超过2万亿，仅次于苹果，从这方面说Windows 11对微软来说是非常成功的。此外，Windows 11的预览版也受到了很多用户的关注和期待，这也证明了它的成功。

但也有一些用户对Windows 11并不买账，虽然Windows 11相比之前的Windows版本有很多改进，但是也有一些不足之处，比如兼容性差、界面不美观、操作不方便等等。

另外从市场份额的角度来看，Windows 11目前的市场份额略高于23%，相比之前的数据有所增长，但仍然落后于Windows 10。从这方面看Windows 11在市场份额方面尚未完全取代Windows 10，可能还不算是非常成功。

![8ca57e74f3881f4641f132b81de477e2.jpeg](https://i-blog.csdnimg.cn/blog_migrate/59839d92d9f99c9478efd549603d7e69.jpeg)

近两年微软一直在探索 AI 时代 Windows 的最佳发展路径，Windows 11 24H2 甚至被定位为“AI 驱动”的重大更新，与全新一代 AI PC 同步亮相。这也说明微软正在通过专注于当前版本的更新来推进 AI 技术在 Windows 操作系统中的应用。

另外前 Windows 和 Surface 业务负责人 Panos Panay 的离职也可能是微软推迟 Windows 12 发布的一个重要因素。他的离职可能导致微软在 Windows 操作系统的开发和管理上做出调整，从而影响新版本的发布计划。

这一策略的调整，表明微软公司在不断探索 AI 时代 Windows 的最佳发展路径。微软希望为用户提供更加智能、高效的使用体验，同时也为自己的 AI 技术探索和发展赢得了更多时间。尽管 Windows 12 的发布被推迟，但微软对 AI 技术在操作系统中的应用的探索并未停止。随着更多“AI PC”的推出，我们可以期待 Windows 11 24H2 将带来更多令人兴奋的新功能和体验。

同时，微软的这一策略调整也提醒我们，在科技行业中，适应变化、持续创新是保持竞争力的关键。

![bbcb1b5f4f6ca833811e8ea3b4b44ebe.jpeg](https://i-blog.csdnimg.cn/blog_migrate/e32c4bc3be3ed3d6cfa2657580251b90.jpeg)

**以下是目前近期 Windows 12 的新功能汇总**

**全新的桌面用户界面**
：Windows 12将发布一种前所未见的全新桌面用户界面，具有浮动式任务栏和类似macOS的状态栏。

**深度融合AI技术**
：微软将加强对AI技术的整合，可能会在Windows 12中看到ChatGPT和DALL-E 2等技术的强大潜力。

**资源管理器和记事本改进**
：选项卡支持将出现在记事本中，用户可以同时打开多个不同的记事本窗口，并且可以同步编辑多个不同的文本文件。

**支持触摸屏设备及键盘**
：Windows 12将支持触摸屏设备，并且控制面板会包含在设置的选项里面。
