     <span style="font-family:'宋体';font-size:14px;">
      1.
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       对异常
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的敏锐
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">

     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       软件测试需要善于发现产品中存在的不足。不仅仅
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       是
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       从系统使用的正向看产品是否有问题，还要从用户角度出发
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       ，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       去设想产品的行为是否存在问题。正如
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">

       许多软件测试
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       的

      <span style="font-family:'宋体';">
       教材中说的那样，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       软件
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       测试不
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       是去证明
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       软件是
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       正确
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       而是
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       去证明错误的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       ，对于一些似是而非的现象，要如猎狗般快速嗅探出其中的
      </span>
      “不对劲”。比如，文档描述
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       上下

      <span style="font-family:'宋体';">
       文
      </span>
     </span>

     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       消息
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       传递中，一个似乎无伤大雅的换行
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       。
      </span>
     </span>
    </p>

     <span style="font-family:'宋体';font-size:14px;">
      2.
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       对
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       技术的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       旺盛
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       好奇心
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       软件测试的日常工作是枯燥的，保持技术的好奇心，可以提高自己工作中的积极性，也能够促使自己用技术的手段提高测试效能。人们
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       常说好奇心是人类进步的源泉，对
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       技术
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       好奇
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       ，诱发
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       学习
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的兴趣
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       。
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       这
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       或许
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       是
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       能够
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       在测试这一行当中长久
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       走下去的
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       ，最为关键的品质了
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Calibri;font-size:14px;">
     </span>
    </p>
    <p style="margin-left:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      3.
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       扎实
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的技术基础
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       众所周知，国内的软件测试行业门槛普遍
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       较
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       低，然而软件测试的工作性质
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       ，又
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       不得不
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       要求
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       从业
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       人员
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       拥有
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       足够高的技术
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       素养。试想，一个技术素养低下的人，又如何能够指出一个技术能力强的人在设计和开发中的不足呢？于是乎，大量的软件测试人员停留在黑盒测试、手工测试层面，很难再
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       有所
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       提升。
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       <br/>
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       而作为一名合格的软件测试工程师，应当
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       要做到的是
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       ：能阅读开发的代码，能理清开发代码的设计思路、实现逻辑；对于系统
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       新增需求，会改变原有的哪些模块，会影响哪些地方的回归，这些都了然于胸。
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       <br/>
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       因此，如果你自认自己技术能力弱，那么多在这上面下些功夫吧。在日常工作中，多去看设计文档，看开发的代码，甚至自己动手去实现一些工具。如果过不了这个坎，软件测试中的高级岗位基本和你无缘，就等着被后来者顶替你吧。
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:Calibri;font-size:14px;">
     </span>
    </p>
    <p style="margin-left:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      4.
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       系统性
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       质量意识
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       在软件研发过程中，问题发现的
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       越早，修复的成本越低
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       。在
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       整个软件测试过程中，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       质量
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       意识应当贯穿
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       于
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       所有环节，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       从
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       质量的角度
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       去
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       思考问题。
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       专家级的软件测试工程师往往都拥有
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       一套
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       自己的质量体系思路，能够在日常工作中，通过架构设计的优化、设计实现有效的测试工具平台、研发流程优化等等手段，去最大限度的降低软件开发过程中的风险。一名专家级的软件测试工程师，基本要具备和开发架构对等的能力，可以和开发架构平等对话设计产品的质量能力。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Calibri;font-size:14px;">
     </span>
    </p>
    <p style="margin-left:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      5.
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       对产品
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的思考
      </span>
     </span>
    </p>

      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       的
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       质量意识
      </span>
     </span>
    </p>
    <p style="text-indent:28px;">
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       在软件研发过程中，问题发现的
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       越早，修复的成本越低

     </span>
     <span style="font-family:'宋体';font-size:14px;">

       。在
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       整个软件测试过程中，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       质量
      </span>

     <span style="font-family:Calibri;font-size:14px;">

       意识应当贯穿

     </span>
     <span style="font-family:'宋体';font-size:14px;">

       于
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       所有环节，
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">

       从
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       质量的角度
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       去
      </span>

     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       思考问题。
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">

     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       一套

     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       自己的质量体系思路，能够在日常工作中，通过架构设计的优化、设计实现有效的测试工具平台、研发流程优化等等手段，去最大限度的降低软件开发过程中的风险。一名专家级的软件测试工程师，基本要具备和开发架构对等的能力，可以和开发架构平等对话设计产品的质量能力。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Calibri;font-size:14px;">

      5.
     </span>
     <span style="font-family:'宋体';font-size:14px;">

       对产品

     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       这或许不应当
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">
      <span style="font-family:'宋体';">
       是软件测试工程师所需要做的工作，然而
      </span>
     </span>
     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       《
      </span>
      How Google Test Softwore
      <span style="font-family:'宋体';">
       》一书中
      </span>
     </span>
     <span style="font-family:Calibri;font-size:14px;">

      </span>
     </span>
    </p>

     <span style="font-family:'宋体';font-size:14px;">
      <span style="font-family:'宋体';">
       <br/>
      </span>
     </span>
    </p>

       我们

       需要

       构想

       ，并尽可能的从
