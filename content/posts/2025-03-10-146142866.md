    <h2>
     R 基础运算
    </h2>
    <h3>
     引言
    </h3>
    <p>
     R 是一种广泛用于统计分析和图形表示的编程语言和软件环境。R 的基础运算功能是其核心组成部分，它为用户提供了强大的数据处理和分析能力。本文将详细介绍 R 中的基础运算，包括数值运算、逻辑运算、字符串运算等，旨在帮助读者快速掌握 R 的基础运算技巧。
    </p>
    <h3>
     数值运算
    </h3>
    <p>
     R 中的数值运算主要包括加法、减法、乘法、除法、乘方等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 加法
result &lt;- 2 + 3
print(result)  # 输出 5

# 减法
result &lt;- 5 - 2
print(result)  # 输出 3

# 乘法
result &lt;- 2 * 3
print(result)  # 输出 6

# 除法
result &lt;- 6 / 2
print(result)  # 输出 3

# 乘方
result &lt;- 2^3
print(result)  # 输出 8
</code></pre>
    <h3>
     逻辑运算
    </h3>
    <p>
     R 中的逻辑运算包括逻辑与（
     <code>
      &amp;
     </code>
     ）、逻辑或（
     <code>
      |
     </code>
     ）、逻辑非（
     <code>
      !
     </code>
     ）等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 逻辑与
result &lt;- (2 &gt; 1) &amp; (3 &gt; 2)
print(result)  # 输出 TRUE

# 逻辑或
result &lt;- (2 &gt; 1) | (3 &gt; 4)
print(result)  # 输出 TRUE

# 逻辑非
result &lt;- !(2 &gt; 1)
print(result)  # 输出 FALSE
</code></pre>
    <h3>
     字符串运算
    </h3>
    <p>
     R 中的字符串运算主要包括连接、子串提取、替换等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 字符串连接
result &lt;- paste("Hello", "World", sep = " ")
print(result)  # 输出 "Hello World"

# 子串提取
result &lt;- substr("Hello World", 1, 5)
print(result)  # 输出 "Hello"

# 字符串替换
result &lt;- gsub("World", "R", "Hello World")
print(result)  # 输出 "Hello R"
</code></pre>
    <h3>
     向量运算
    </h3>
    <p>
     R 中的向量运算包括向量的创建、向量的元素访问、向量的运算等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 向量创建
vector &lt;- c(1, 2, 3, 4, 5)
print(vector)  # 输出 1 2 3 4 5

# 向量元素访问
print(vector[2])  # 输出 2

# 向量运算
result &lt;- vector * 2
print(result)  # 输出 2 4 6 8 10
</code></pre>
    <h3>
     数组与矩阵运算
    </h3>
    <p>
     R 中的数组与矩阵运算包括数组的创建、数组的元素访问、数组的运算等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 数组创建
array &lt;- array(c(1, 2, 3, 4, 5, 6, 7, 8, 9), c(3, 3))
print(array)

# 数组元素访问
print(array[1, 2])  # 输出 3

# 数组运算
result &lt;- array + 2
print(result)
</code></pre>
    <h3>
     数据框运算
    </h3>
    <p>
     R 中的数据框运算包括数据框的创建、数据框的元素访问、数据框的运算等。以下是一些示例：
    </p>
    <pre><code class="language-R"># 数据框创建
data.frame &lt;- data.frame(name = c("Alice", "Bob", "Charlie"), age = c(25, 30, 35))
print(data.frame)

# 数据框元素访问
print(data.frame[1, "name"])  # 输出 Alice

# 数据框运算
result &lt;- data.frame[age &gt; 28, ]
print(result)
</code></pre>
    <h3>
     总结
    </h3>
    <p>
     本文介绍了 R 中的基础运算，包括数值运算、逻辑运算、字符串运算、向量运算、数组与矩阵运算、数据框运算等。通过掌握这些基础运算，读者可以更好地利用 R 进行数据处理和分析。希望本文能对您有所帮助。
    </p>
