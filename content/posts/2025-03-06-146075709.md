    <p>
     📢 STM32F103xC/D/E 系列是一款高性能、低功耗的 32 位 MCU，适用于工业、汽车、消费电子等领域；
     <strong>
      基于 ARM Cortex-M3，主频最高 72MHz，支持 512KB Flash、64KB SRAM
     </strong>
     ，适合复杂嵌入式应用，
     <strong>
      提供丰富的外设（USB、CAN、SDIO、I²C、SPI、USART），支持低功耗模式（睡眠、停止、待机）
     </strong>
     ，适用于电池供电设备。
    </p>
    <p>
     提供官方手册：
     <br/>
     <a href="https://www.st.com/en/microcontrollers-microprocessors/stm32f103.html" rel="nofollow">
      STM32F103 数据手册
     </a>
     <br/>
     <a href="https://www.st.com/resource/en/reference_manual/cd00171190.pdf" rel="nofollow">
      STM32F103 参考手册
     </a>
     <br/>

      STM32 开发指南
     </a>

    <h4>
     <a id="STM32F103xCSTM32F103xDSTM32F103xE_6">
     </a>
     STM32F103xC/STM32F103xD/STM32F103xE
    </h4>
    <p>
     <img alt="STM32F103xC" src="https://i-blog.csdnimg.cn/direct/5a3bd6f316d241b6a9b914a9b526ee66.png">
      <br/>
      STM32F103RCT6 是意法半导体（STMicroelectronics）推出的
      <strong>
       基于 ARM Cortex-M3 内核的 32 位 MCU
      </strong>
      ，属于
      <strong>
       STM32F1 系列的高性能互联型产品
      </strong>
      ，具有较高的主频、丰富的外设和低功耗特性。这是一款基于 ARM Cortex-M3 的高性能 MCU 微控制器，提供丰富的外设和通信接口，适用于各种嵌入式系统。
     </img>
    </p>
    <h4>
     <a id="MCU__9">
     </a>
     MCU 基本信息
    </h4>
    <blockquote>
     <p>
      <strong>
       型号
      </strong>
      ：STM32F103RCT6
      <br/>
      <strong>
       制造商
      </strong>
      ：STMicroelectronics（意法半导体）
      <br/>
      <strong>
       核心
      </strong>
      ：ARM Cortex-M3（32 位）
      <br/>
      <strong>
       最高主频
      </strong>
      ：72MHz
      <br/>
      <strong>
       Flash 存储
      </strong>
      ：256KB
      <br/>
      <strong>
       SRAM
      </strong>
      ：48KB
      <br/>
      <strong>
       封装类型
      </strong>
      ：LQFP64（64 引脚）
      <br/>
      <strong>
       工作电压
      </strong>
      ：2.0V ~ 3.6V
      <br/>
      <strong>
       I/O 口
      </strong>
      ：最多 51 个 GPIO
      <br/>
      <strong>
       工作温度
      </strong>
      ：-40°C ~ 85°C（工业级）
     </p>
    </blockquote>
    <h5>
     <a id="_21">
     </a>
     主要功能特点
    </h5>
    <p>
     <strong>
      时钟系统：高达72M
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       外部高速晶振
      </strong>
      ：4MHz ~ 16MHz
      <br/>
      <strong>
       内部高速 RC 振荡器
      </strong>
      ：8MHz
      <br/>
      <strong>
       内部低速 RC 振荡器
      </strong>
      ：40kHz（用于看门狗）
      <br/>
      <strong>
       内部锁相环（PLL）
      </strong>
      ：可对 HSE（外部高速晶振）或 HSI（内部高速 RC 振荡器）进行倍频，提供系统时钟
      <br/>
      <strong>
       外部低速 32.768 kHz 晶振
      </strong>
      ：用于 RTC（实时时钟）
     </p>
    </blockquote>
    <p>
     <strong>
      低功耗模式：
     </strong>
    </p>
    <blockquote>
     <p>
      支持
      <strong>
       3 种低功耗模式
      </strong>
      ：
     </p>
     <ol>
      <li>
       <strong>
        睡眠模式（Sleep）
       </strong>
       ：CPU 停止，但外设可继续运行
      </li>
      <li>
       <strong>
        停止模式（Stop）
       </strong>
       ：所有时钟停止，仅保留 SRAM 和寄存器数据
      </li>
      <li>
       <strong>
        待机模式（Standby）
       </strong>
       ：最低功耗模式，仅保留 RTC 和备份寄存器，可用电池供电
      </li>
     </ol>
    </blockquote>
    <p>
     <strong>
      ADC / DAC：
     </strong>
    </p>
    <blockquote>
     <ul>
      <li>
       <strong>
        ADC（模数转换）
       </strong>
       ：
       <ul>
        <li>
         3 个 12 位 ADC
        </li>
        <li>
         最多 16 个外部测量通道
        </li>
        <li>
         支持内部温度测量
        </li>
        <li>
         内置参考电压
        </li>
       </ul>
      </li>
      <li>
       D
       <strong>
        AC（数模转换）
       </strong>
       ：
       <ul>
        <li>
         2 个 12 位 DAC
        </li>
        <li>
         可用于音频信号生成或电压控制
        </li>
       </ul>
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      DMA（直接存储器存取）：
     </strong>
    </p>
    <blockquote>
     <p>
      12 个 DMA 通道
      <br/>
      支持的外设：
     </p>
     <ul>
      <li>
       定时器
      </li>
      <li>
       ADC
      </li>
      <li>
       SDIO
      </li>
      <li>
       I²S
      </li>
      <li>
       SPI
      </li>
      <li>
       I²C
      </li>
      <li>
       USART
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      定时器：
     </strong>
    </p>
    <blockquote>
     <p>
      多达 11 个定时器
     </p>
     <ul>
      <li>
       4 个通用定时器：16 位，支持 PWM、输入捕获、编码器模式
      </li>
      <li>
       2 个基本定时器：用于 DAC 触发
      </li>
      <li>
       2 个高级定时器： 支持 6 通道 PWM，死区时间管理
      </li>
      <li>
       1 个系统定时器：SysTick 24 位倒计时
      </li>
      <li>
       2 个看门狗定时器： 独立看门狗（IWDG），窗口看门狗（WWDG）
      </li>
     </ul>
     <p>
      <strong>
       支持 PWM 输出、输入捕获、编码器接口等功能
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      存储器：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       Flash 存储器
      </strong>
      ：256KB ~ 512KB（用于代码和数据存储）
      <br/>
      <strong>
       SRAM
      </strong>
      ：48KB ~ 64KB（高速存取存储器）
      <br/>
      <strong>
       FMC（灵活存储控制器）
      </strong>
      ：
     </p>
     <ul>
      <li>
       <strong>
        支持 NOR Flash、SRAM、PSRAM、NAND 存储器
       </strong>
      </li>
      <li>
       <strong>
        支持 LCD 并行接口（8080/6800 模式）
       </strong>
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      GPIO：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       最多 112 个 I/O 口
      </strong>
      <br/>
      <strong>
       所有 GPIO 支持 5V 容忍（FT）
      </strong>
      <br/>
      <strong>
       支持外部中断（EXTI）
      </strong>
      <br/>
      <strong>
       支持 AF（复用功能）
      </strong>
      <br/>
      <strong>
       支持位带操作（Bit-Banding）
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      通信接口：
     </strong>
    </p>
    <blockquote>
     <p>
      多达 13 个通信接口
     </p>
     <ul>
      <li>
       2 × I²C
      </li>
      <li>
       5 × USART（串口）
      </li>
      <li>
       3 × SPI
      </li>
      <li>
       1 × CAN 2.0
      </li>
      <li>
       1 × USB OTG（全速 2.0）
      </li>
      <li>
       1 × SDIO（用于 SD 卡接口）
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      封装信息：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       LQFP64
      </strong>
      （10×10 mm，64 引脚）
      <br/>
      <strong>
       LQFP100
      </strong>
      （14×14 mm，100 引脚）
      <br/>
      <strong>
       LQFP144
      </strong>
      （20×20 mm，144 引脚）
      <br/>
      <strong>
       LFBGA100
      </strong>
      （10×10 mm，100 球 BGA）
      <br/>
      <strong>
       LFBGA144
      </strong>
      （10×10 mm，144 球 BGA）
     </p>
    </blockquote>
    <p>
     <strong>
      调试接口：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       支持 SWD（串行调试）和 JTAG 调试
      </strong>
      <br/>
      <strong>
       SWD 仅需 2 根数据线
      </strong>
      <br/>
      <strong>
       支持最多 8 个断点和 4 个观察点
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      STM32F103RCT6 的引脚信息：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       封装类型
      </strong>
      ：LQFP64（64 引脚）
      <br/>
      <strong>
       GPIO 数量
      </strong>
      ：最多 51 个 I/O 口
      <br/>
      <strong>
       部分 I/O 口支持 5V 容忍
      </strong>
      （标注
      <strong>

      <br/>
      <strong>
       电源引脚
      </strong>

     </p>
     <ul>

       <strong>
        VDD（主电源）

        VSS（地）
       </strong>
      </li>
      <li>
       <strong>
        VBAT（备用电池电源）
       </strong>
      </li>
     </ul>
     <p>
      <strong>
       时钟引脚
      </strong>
      ：
     </p>
     <ul>
      <li>
       <strong>
        OSC_IN / OSC_OUT（外部晶振）
       </strong>
      </li>
      <li>
       <strong>
        OSC32_IN / OSC32_OUT（RTC 低速晶振）
       </strong>
      </li>
     </ul>
     <p>
      <strong>
       调试引脚
      </strong>
      ：
     </p>
     <ul>
      <li>
       <strong>
        SWDIO / SWCLK（SWD 调试）
       </strong>
      </li>
      <li>
       <strong>
        JTDI / JTDO / JTAG（JTAG 调试）
       </strong>
      </li>
     </ul>
    </blockquote>

     STM32F103xC/D/E 微控制器的命名规则解析

    <p>
     STM32F103xC/D/E 系列微控制器的命名规则，用于识别不同型号的产品规格。
    </p>
    <p>
     📌
     <strong>
      命名格式：（每个部分代表特定的产品特性：）
     </strong>
     <br/>
     <img alt="STM32F103xC/D/E 微控制器的命名规则" src="https://i-blog.csdnimg.cn/direct/e0d2f7b8cf484836922ce42386efbea8.png"/>
    </p>
    <pre><code class="prism language-bash">STM32   F   <span class="token number">103</span>   R   C   T   <span class="token number">6</span>   xxx
  │      │    │    │   │   │   │   └── 选项（如编程版本、卷带包装）
  │      │    │    │   │   │   └── 工作温度范围
  │      │    │    │   │   └── 封装类型<span class="token punctuation">(</span>H<span class="token operator">=</span>BGA <span class="token assign-left variable">T</span><span class="token operator">=</span>LQFP<span class="token punctuation">)</span>
  │      │    │    │   └── Flash 存储大小<span class="token punctuation">(</span>C:256k D:384k E:512k<span class="token punctuation">)</span>
  │      │    │    └── 引脚数量<span class="token punctuation">(</span>引脚是芯片的重要资源，因为它直接决定有多少通道和外界进行数据交互<span class="token punctuation">)</span>

    <h4>
     <a id="STM328_196">
     </a>
     STM32：8大系列产品
    </h4>
    <p>
     STM32 微控制器系列 的产品分类，涵盖 高性能、低功耗、互联、基础型等不同应用场景。
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/7f7cfbcfd967438bb70606b7bb1fe6b9.png"/>
    </p>
    <h6>
     <a id="STM32__199">
     </a>
     STM32 系列的共同特性
    </h6>
    <blockquote>
     <p>
      <strong>
       通信接口
      </strong>
      ：支持
      <strong>
       USART、SPI、I²C
      </strong>
      <br/>
      <strong>
       通用定时器
      </strong>
      ：适用于
      <strong>
       PWM、输入捕获、输出比较
      </strong>
      <br/>
      <strong>
       DMA 控制器
      </strong>
      ：提高数据传输效率
      <br/>
      <strong>
       RTC（实时时钟）
      </strong>
      ：支持低功耗计时
      <br/>
      <strong>
       PLL 和时钟管理
      </strong>
      ：动态调整系统频率
      <br/>
      <strong>
       FSMC（外部存储控制器）
      </strong>
      ：支持外部 Flash、SRAM、LCD 接口
      <br/>
      <strong>
       ADC/DAC
      </strong>
      ：支持
      <strong>
       12 位 ADC 和 12 位 DAC
      </strong>
      <br/>
      <strong>
       低功耗支持：
      </strong>
     </p>
     <ul>
      <li>
       <strong>
        32.768kHz 振荡器
       </strong>
      </li>
      <li>
       <strong>
        低速/高速 RC 振荡器
       </strong>
      </li>
      <li>
       <strong>
        待机、睡眠、停止模式
       </strong>
      </li>
     </ul>
     <p>
      <strong>
       工业级温度范围
      </strong>
      （-40°C ~ +85°C / -40°C ~ +105°C）
      <br/>
      <strong>
       工作电压
      </strong>
      ：2.0V ~ 3.6V（部分支持 1.65V ~ 3.6V）
      <br/>
      <strong>
       5V 容忍 I/O 口
      </strong>
      <br/>
      <strong>
       温度传感器
      </strong>
     </p>
    </blockquote>
    <h5>
     <a id="STM32__216">
     </a>
     STM32 主要产品系列
    </h5>
    <p>
     STM32 按照不同应用需求，划分为
     <strong>
      8 大系列
     </strong>
     ：
    </p>
    <h6>
     <a id="_1_STM32F4__DSP__218">
     </a>
     ⭐ 1. STM32F4 系列：高性能 DSP 计算
    </h6>
    <pre><code>核心：Cortex-M4（168MHz）
特点：
	DSP + FPU（浮点运算）
	192KB~1MB SRAM
	2 个 USB 2.0 OTG
	3 轴电机定时器
	2 个 CAN 2.0B
	SDIO、I²S、以太网
	加密/哈希处理器
应用：高性能计算、信号处理、工业控制
</code></pre>
    <h6>
     <a id="_2_STM32F2__229">
     </a>
     ⭐ 2. STM32F2 系列：高性能嵌入式
    </h6>
    <pre><code>核心：Cortex-M3（120MHz）
特点：
	128KB SRAM，1MB Flash
	USB 2.0 OTG
	2 个 CAN 2.0B
	SDIO、I²S、以太网
	加密/哈希处理
应用：工业控制、嵌入式系统
</code></pre>
    <h6>
     <a id="_3_STM32F1_STM32F105107_238">
     </a>
     ⭐ 3. STM32F1 互联型（STM32F105/107）
    </h6>
    <pre><code>核心：Cortex-M3（72MHz）
特点：
	64KB SRAM，256KB Flash
	USB 2.0 OTG
	2 个 CAN 2.0B
	SDIO、I²S、以太网
应用：物联网（IoT）、通信设备
</code></pre>
    <h6>
     <a id="_4_STM32F1_STM32F103_246">
     </a>
     ⭐ 4. STM32F1 典型型（STM32F103）
    </h6>
    <pre><code>核心：Cortex-M3（72MHz）
特点：
	大容量 Flash
	USB 2.0
	CAN 2.0B
	SDIO、I²S
应用：消费电子、机器人控制
</code></pre>
    <h6>
     <a id="_5_STM32F1_USB_STM32F102_254">
     </a>
     ⭐ 5. STM32F1 USB 基本型（STM32F102）
    </h6>
    <pre><code>核心：Cortex-M3（48MHz）
特点：
	16KB SRAM，128KB Flash
	USB 2.0
应用：USB 外设、低成本设备
</code></pre>
    <h6>
     <a id="_6_STM32F1_STM32F101_260">
     </a>
     ⭐ 6. STM32F1 基本型（STM32F101）
    </h6>
    <pre><code>核心：Cortex-M3（36MHz）
特点：
	1MB Flash
	基础定时器
应用：基础嵌入式应用
</code></pre>
    <h6>
     <a id="_7_STM32F1_STM32F100_266">
     </a>
     ⭐ 7. STM32F1 超值型（STM32F100）
    </h6>
    <pre><code>核心：Cortex-M3（24MHz）
特点：
	512KB Flash
	3 轴电机定时器
	CEC（消费电子控制）
应用：成本敏感型应用
</code></pre>
    <h6>
     <a id="_8_STM32_L1_STM32L151152_273">
     </a>
     ⭐ 8. STM32 L1 超低功耗系列（STM32L151/152）
    </h6>
    <pre><code>核心：Cortex-M3（32MHz）
特点：
	48KB SRAM，384KB Flash
	USB 2.0
	2KB 数据 EEPROM
	LCD 驱动（8×40 段 / 4×44 段）
	低功耗特性（BOR、MSI、VScal）
应用：便携式设备、低功耗 IoT
</code></pre>
    <p>
     以上，
    </p>
    <pre><code>STM32F4：高性能 DSP 计算
STM32F2：嵌入式高性能应用
STM32F1 互联型：物联网通信
STM32F1 典型型：主流嵌入式控制
STM32F1 USB 基本型：USB 应用
STM32F1 基本型：基础嵌入式
STM32F1 超值型：低成本应用
STM32 L1：超低功耗 IoT
</code></pre>
