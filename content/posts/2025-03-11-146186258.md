     </strong>
     <br/>
     将复杂的数据结构简化为二维表格形式
     <br/>
     大型:Oracle、DB2
     <br/>
     中型:MySql、SQLServer
     <br/>
     小型:Sqlite

    <p>
     <strong>
      2.非关系型数据库
     </strong>
     <br/>
     以键值对存储，且结构不固定
     <br/>
     //JSON
     <br/>
     Redis
     <br/>
     MongoDB
     <br/>
     <strong>
      嵌入式数据库：
     </strong>
     <br/>
     sqlite3：
     <br/>
     stu.db
     <br/>
     1.开源免费， c语言开发
     <br/>

     <br/>
     3.文件型数据库，可以移动
     <br/>
     4.数据容量最大2T
