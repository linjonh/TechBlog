    <h2 id="Java%E5%92%8CJavaScript%E7%9A%84%E4%B8%BB%E8%A6%81%E5%8C%BA%E5%88%AB%E4%B9%8B%E5%A4%84%E5%9C%A8%E5%93%AA%E9%87%8C%EF%BC%9F" style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        Java和JavaScript的主要区别之处在哪里？
       </strong>
      </span>
     </strong>
    </h2>
    <p style="margin-left:0;text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        JavaScript和java分别是什么？
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        JavaScript是一种轻量级编程语言（“脚本语言”），用于使网页交互。它可以将动态文本插入到HTML里。JavaScript也称为浏览器语言。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="margin-left:0;text-align:left;">
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java是一种面向对象的编程语言，具有虚拟机平台，允许你创建几乎在每个平台上运行的编译程序。Java承诺，“一次编写，随处运行”。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="margin-left:0;text-align:left;">
    </p>
    <p style="margin-left:0;text-align:left;">
    </p>
    <p style="margin-left:0;text-align:left;">
    </p>
    <h3 style="margin-left:0px;text-align:left;">
     <strong>
      <span style="background-color:#ffff00;">
       <span style="color:#000000;">
        <strong>
         JavaScript和java的区别有：
        </strong>
       </span>
      </span>
     </strong>
    </h3>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        1.  Java是强类型语言，而JavaScript是弱类型语言；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java是强类型语言，使用变量前必须首先在程序中声明；在Java中，在编译时会检查变量的类型。而，JavaScript是弱类型语言，具有更宽松的语法和规则。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        2. 数据类型不同；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java的数据类型有8种，分别是：byte类型、short类型、int类型、long类型、float类型、double类型、char类型以及Boolean类型。而，JavaScript的数据类型只有三种，分别为：number类型、string类型和Boolean类型。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        3. 执行环境不同；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java应用程序可以在任何虚拟机（JVM）或Web浏览器中运行，执行时会使用更多内存。而，JavaScript代码仅在浏览器上运行，因为JavaScript是仅针对浏览器开发的。它运行时只需要很少的内存，因此它可以在网页中被使用。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        4. 定位不同；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java是一种完全面向对象的OOP编程语言，我们想要运用Java写东西就必须先设计好对象；而，JavaScript不同，它是基于对象的一种OOP脚本语言，它为开发者提供了自带的很多内部对象，要更为简易省时。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        5. 联编方式不同；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java采用静态联编，是一种静态类型语言。Java的对象引用必须在编译时就进行，编译器需要实现强类型检查。而JavaScript不同，它采用动态联编，是一种动态类型语言，它的对象引用可以在运行时检查。
       </span>
      </span>
     </p>
    </blockquote>
    <p style="text-align:left;">
     <strong>
      <span style="color:#000000;">
       <strong>
        6. 并发方法不同
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000;">
       <strong>
        ；
       </strong>
      </span>
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:0;text-align:left;">
      <span style="background-color:#eeeeee;">
       <span style="color:#000000;">
        Java具有一种基于线程的并发方法。而，Javascript具有基于事件的并发方法。
       </span>
      </span>
     </p>
    </blockquote>
