     1. CSS计数器
    </h2>
    <p>
     CSS 计数器就像“变量”。变量值可以通过 CSS 规则递增（将跟踪它们的使用次数）。
    </p>
    <p>
     如需使用 CSS 计数器，我们将使用以下属性：
    </p>
    <ul>
     <li>
      <p>
       <code>
        counter-reset
       </code>
       - 创建或重置计数器
      </p>
     </li>
     <li>
      <p>
       <code>
        counter-increment
       </code>
       - 递增计数器值
      </p>
     </li>
     <li>
      <p>
       <code>
        content
       </code>
       - 插入生成的内容
      </p>
     </li>
     <li>
      <p>
       <code>
        counter()
       </code>
       或
       <code>
        counters()
       </code>
       函数 - 将计数器的值添加到元素
      </p>
     </li>
    </ul>
    <p>
     如需使用 CSS 计数器，必须首先使用
     <code>
      counter-reset
     </code>
     创建它。
    </p>
    <h3 id="%E5%B5%8C%E5%A5%97%E8%AE%A1%E6%95%B0%E5%99%A8%EF%BC%9A" name="%E5%B5%8C%E5%A5%97%E8%AE%A1%E6%95%B0%E5%99%A8%EF%BC%9A">
     <strong>
      嵌套计数器：
     </strong>
    </h3>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
&lt;style&gt;
body {
  counter-reset: section;
}

h1 {
  counter-reset: subsection;
}

h1::before {
  counter-increment: section;
  content: "Section " counter(section) ". ";
}

h2::before {
  counter-increment: subsection;
  content: counter(section) "." counter(subsection) " ";
}
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;

    <h2 id="2.%E5%8D%95%E4%BD%8D" name="2.%E5%8D%95%E4%BD%8D">
     2.单位
    </h2>
    <h3 id="%E7%BB%9D%E5%AF%B9%E9%95%BF%E5%BA%A6%EF%BC%9A" name="%E7%BB%9D%E5%AF%B9%E9%95%BF%E5%BA%A6%EF%BC%9A">
     绝对长度：
    </h3>
    <p>
     绝对长度单位是固定的，用任何一个绝对长度表示的长度都将恰好显示为这个尺寸。
    </p>
    <p>
     不建议在屏幕上使用绝对长度单位，因为屏幕尺寸变化很大。
    </p>
    <table>
     <tbody>
      <tr>
       <th>
        单位
       </th>
       <th>
        描述
       </th>
      </tr>

       <td>
        厘米
       </td>
      </tr>
      <tr>
       <td>
        mm

       <td>
        毫米
       </td>
      </tr>
      <tr>
       <td>
        in
       </td>
       <td>
        英寸 (1in = 96px = 2.54cm)
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         px *
        </strong>
       </td>
       <td>
        像素 (1px = 1/96th of 1in)

     相对长度：
    </h3>
    <p>
     相对长度单位规定相对于另一个长度属性的长度。相对长度单位在不同渲染介质之间缩放表现得更好。
    </p>
    <table>
     <tbody>
      <tr>
       <th>
        单位
       </th>
       <th>
        描述
       </th>
      </tr>
      <tr>
       <td>
        <strong>
         em
        </strong>
       </td>
       <td>
        <strong>
         相对于元素的字体大小（font-size）（2em 表示当前字体大小的 2 倍）
        </strong>
       </td>
      </tr>
      <tr>
       <td>
        ex
       </td>
       <td>
        相对于当前字体的 x-height(极少使用)
       </td>
      </tr>
      <tr>
       <td>
        ch
       </td>
       <td>
        相对于 "0"（零）的宽度
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         rem
        </strong>
       </td>
       <td>
        <strong>
         相对于根元素的字体大小（font-size）
        </strong>
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         vw

       </td>
       <td>

        </strong>
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         vh
        </strong>
       </td>
       <td>
        <strong>
         相对于视口*高度的 1%
        </strong>
       </td>
      </tr>
      <tr>
       <td>
        vmin
       </td>
       <td>
        相对于视口*较小尺寸的 1％
       </td>
      </tr>
      <tr>
       <td>
        vmax
       </td>
       <td>
        相对于视口*较大尺寸的 1％
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         %
        </strong>
       </td>
       <td>
        <strong>
         相对于父元素
        </strong>
       </td>

    <p>
     视口（Viewport）= 浏览器窗口的尺寸。如果视口为 50 厘米宽，则 1vw = 0.5 厘米。
    </p>

    <h2 id="%C2%A03.%40media%E5%AA%92%E4%BD%93%E6%9F%A5%E8%AF%A2" name="%C2%A03.%40media%E5%AA%92%E4%BD%93%E6%9F%A5%E8%AF%A2" style="background-color:transparent">
     3.@media媒体查询
    </h2>
    <p>
     相当于在不同媒体设备类型（如手机、平板、电脑、打印机等）的不同页面大小，显示格式有区别。
    </p>
    <p>
     <strong>
      应用：
     </strong>
    </p>
    <ul>
     <li>
      <p>
       隐藏元素
      </p>
     </li>
     <li>
      <p>
       改变元素
      </p>
     </li>
     <li>
      <p>
       弹性布局
      </p>
     </li>
    </ul>
    <p>
     <strong>
      css3媒体类型：
     </strong>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        值
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        all
       </td>
       <td>
        用于所有媒体类型设备。
       </td>
      </tr>
      <tr>
       <td>
        print
       </td>
       <td>
        用于打印机。
       </td>
      </tr>
      <tr>
       <td>
        screen
       </td>
       <td>
        用于计算机屏幕、平板电脑、智能手机等等。
       </td>
      </tr>
      <tr>
       <td>
        speech
       </td>
       <td>
        用于大声“读出”页面的屏幕阅读器。
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      语法与常用的设备断点
     </strong>
     <strong>
      ：
     </strong>
    </p>
    <pre><code class="language-html">@media not|only mediatype and (expressions) {
  CSS-Code;
}</code></pre>
    <pre><code class="language-css">/* 超小型设备（电话，600px 及以下） */
@media only screen and (max-width: 600px) {...} 

/* 小型设备（纵向平板电脑和大型手机，600 像素及以上） */
@media only screen and (min-width: 600px) {...} 

/* 中型设备（横向平板电脑，768 像素及以上） */
@media only screen and (min-width: 768px) {...} 

/* 大型设备（笔记本电脑/台式机，992px 及以上） */
@media only screen and (min-width: 992px) {...} 

/* 超大型设备（大型笔记本电脑和台式机，1200px 及以上） */
@media only screen and (min-width: 1200px) {...}</code></pre>
    <p>
     当使用媒体查询
     <code>
      min-device-width
     </code>
     而不是
     <code>
      min-width
     </code>
     来
     <strong>
      检查设备宽度
     </strong>
     ，而
     <strong>
      不是浏览器宽度
     </strong>
     。然后，当您调整浏览器窗口的大小时，图像将不会变化
    </p>
    <pre><code class="language-css">/* 针对小于 400 像素的设备： */
body {
  background-image: url('img_smallflower.jpg'); 
}

/* 针对 400 像素及更大的设备： */
@media only screen and (min-device-width: 400px) {
  body { 
    background-image: url('img_flowers.jpg'); 
  }
}</code></pre>
    <pre><code class="language-css">/* 针对小于 400 像素的宽度： */
body {
  background-image: url('img_smallflower.jpg'); 
}

/* 针对 400 像素或更大的宽度： */
@media only screen and (min-width: 400px) {
  body { 
    background-image: url('img_flowers.jpg'); 
  }
}</code></pre>
