   <div class="htmledit_views" id="content_views">
    <ol>
     <li>
      深拷贝
      <br/>
      是指创建一个新对象，并递归地复制原对象中所有层级的属性和值，从而确保新对象与原对象完全独立
      <br/>
      <br/>
      深拷贝的实现方法 ：
      <br/>
      1.
      <strong>
       使用 `JSON.parse(JSON.stringify())
      </strong>
      <br/>
      函数会被忽略复制，比如，下面的对象的formatter函数深拷贝以后就不在了
      <br/>
      <pre><code>    {
      name: "笔数",
      type: "value",
      axisLabel: {
        formatter: function (value) {
          console.log('value: ', value);
          return value + 'kg';
        }
      }
    },</code></pre>
     </li>
    </ol>
    <ul>
     <li>
      <p>
       <strong>
        JSON.parse(JSON.stringify())会导致：
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        无法处理函数
       </strong>
       ：函数会被忽略。
      </p>
     </li>
     <li>
      <p>
       <strong>
        无法处理特殊对象类型
       </strong>
       ：如
       <code>
        Date
       </code>
       、
       <code>
        RegExp
       </code>
       、
       <code>
        Map
       </code>
       、
       <code>
        Set
       </code>
       等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        无法处理循环引用
       </strong>
       ：会抛出错误。
      </p>
     </li>
     <li>
      <p>
       <strong>
        丢失原型链
       </strong>
       ：新对象的原型链会被重置为
       <code>
        Object.prototype
       </code>
      </p>
     </li>
    </ul>
    <ol>
     <li>
      浅拷贝
      <br/>
      只复制对象的第一层属性。如果属性值是对象（如
      <code>
       address
      </code>
      ），它会复制该属性值的对象的引用，而不是对象的内容
      <pre><code>const original = {
  age: 25,
  address: {
    city: "Moe",
  }
};


shallowCopy.address.street = "New Street";
console.log(original.address.street); // 输出：New Street</code></pre>
      <p>
      </p>
     </li>
    </ol>
