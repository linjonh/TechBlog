    <h2>
     <span style="font-size:18px">
      DBUtils工具类介绍
     </span>
    </h2>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       DBUtils是java编程中的数据库操作实用工具，小巧简单使用；
      </span>
     </p>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       DBUtils封装了对JDBC的操作，简化了JDBC操作，可以少写代码；
      </span>
     </p>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       DBUtils就是JDBC的简化开发工具包。需要导入common-dbutils-xx.jar才能够正常使用DBUtils工具。
      </span>
     </p>
    </blockquote>
    <h3>
     <span style="font-size:18px">
      三个核心类
     </span>
    </h3>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       QueryRunner类
      </span>
     </p>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       ResultSetHandler接口
      </span>
     </p>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <p>
      <span style="font-size:18px">
       DBUtils类
      </span>
     </p>
    </blockquote>
    <h2>
     <span style="font-size:18px">
      DBUtils工具类处理结果集的方式
     </span>
    </h2>
    <p>
     <span style="font-size:18px">
      QueryRunner类实现查询操作
     </span>
    </p>
    <p>
     <span style="font-size:18px">
      ResultSetHandler 结果集处理类
     </span>
    </p>
    <p>
    </p>
    <ol>
     <li>
      <span style="font-size:18px">
       ArrayHandler
       <br/>
       将结果集中的第一条记录封装到一个Object[ ] 数组中，数组中的每一个元素就是这条记录中的每一个字段的值。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       ArrayListHandler
       <br/>
       将结果集中的每一条记录都封装到一个Object[ ] 数组中，再将这些数组封装到List集合中。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       BeanHandler
       <br/>
       将结果集中第一条记录封装到一个指定的JavaBean中。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       BeanListHander
       <br/>
       将结果集中每一条记录封装到指定的JavaBean中，再将这些JavaBean封装到List集合中。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       ColumnListHandler
       <br/>
       将结果集中指定的列的字段值，封装到一个List集合中。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       ScalarHandler
       <br/>
       用于单数据。例如 select count(*) from 操作。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       MapHandler
       <br/>
       将结果集第一行封装到Map集合中，key 列名，value 该列数据。
      </span>
     </li>
     <li>
      <span style="font-size:18px">
       MapListHandler
       <br/>
       将结果集第一行封装到Map集合中，
       <span style="font-size:18px">
        key 列名，value 该列数据，在将Map集合存储到List集合中。
       </span>
      </span>
     </li>
    </ol>
    <p>
    </p>
