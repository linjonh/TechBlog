        let display = document.getElementById('display');

    <p>
     在现代生活中，计算器是我们日常工作和学习中不可或缺的工具。无论是进行简单的加减乘除运算，还是处理复杂的数学表达式，计算器都能帮助我们快速得到结果。本文将介绍如何使用HTML、CSS和JavaScript创建一个简单的网页计算器，并解释其实现原理。
    </p>
    <h3>
     1. 项目结构
    </h3>
    <p>
     这个简单的计算器项目主要由三部分组成：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        HTML
       </strong>
       ：用于构建计算器的基本结构和布局。
      </p>
     </li>
     <li>
      <p>
       <strong>
        CSS
       </strong>
       ：用于美化计算器的外观，使其更加用户友好。
      </p>
     </li>
     <li>
      <p>
       <strong>
        JavaScript
       </strong>
       ：用于实现计算器的逻辑功能，如输入、删除、清空和计算等。
      </p>
     </li>
    </ul>
    <h3>
     2. HTML结构
    </h3>
    <p>
     HTML部分定义了计算器的基本结构。我们使用了一个
     <code>
      &lt;div&gt;
     </code>
     容器来包裹整个计算器，并在其中放置了一个输入框和一个按钮网格。
    </p>
    <pre><code class="language-html">&lt;div class="calculator"&gt;
    &lt;input type="text" id="display" disabled&gt;
    &lt;div class="buttons"&gt;
        &lt;button onclick="clearDisplay()"&gt;C&lt;/button&gt;
        &lt;button onclick="appendToDisplay('/')" class="operator"&gt;/&lt;/button&gt;
        &lt;button onclick="appendToDisplay('*')" class="operator"&gt;*&lt;/button&gt;

    <ul>
     <li>
      <p>
       <strong>
        输入框
       </strong>
       ：
       <code>
        &lt;input type="text" id="display" disabled&gt;
       </code>
       用于显示用户输入和计算结果。
       <code>
        disabled
       </code>
       属性确保用户不能直接通过键盘输入。
      </p>
     </li>
     <li>
      <p>
       <strong>
        按钮
       </strong>
       ：每个按钮都绑定了
       <code>
        onclick
       </code>
       事件，用于处理用户的点击操作。按钮分为数字按钮、运算符按钮和功能按钮（如清空、删除、等于）。
      </p>
     </li>
    </ul>
    <h3>
     3. CSS样式
    </h3>
    <p>
     CSS部分用于美化计算器的外观。我们使用了Flexbox布局来使计算器在页面中居中显示，并使用Grid布局来排列按钮。
    </p>

     <li>

       <code>

       和
       <code>

       <code>

     <li>
      <p>
       <strong>
        按钮布局
       </strong>
       ：使用
       <code>
        grid-template-columns: repeat(4, 1fr)

       将按钮排列成4列的网格布局。
      </p>
     </li>
     <li>
      <p>
       <strong>
        按钮样式
       </strong>
       ：不同类型的按钮（如运算符、等于按钮）有不同的背景颜色和悬停效果，以提高用户体验
      </p>
     </li>
    </ul>
    <h3>
     4. JavaScript逻辑
    </h3>
    <p>
     JavaScript部分负责处理计算器的逻辑功能。我们定义了四个函数来实现计算器的基本操作：
    </p>
    <pre><code class="language-javascript">function appendToDisplay(value) {
    document.getElementById('display').value += value;
}

function clearDisplay() {
    document.getElementById('display').value = '';
}

function deleteLast() {
    let display = document.getElementById('display');
    display.value = display.value.slice(0, -1);
}

function calculate() {
    let display = document.getElementById('display');
    try {
        // 使用 eval 计算表达式，支持运算符优先级
        display.value = eval(display.value);
    } catch (e) {
        display.value = '错误';
    }
}</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        <code>
         appendToDisplay(value)
        </code>
       </strong>
       ：将用户点击的按钮值追加到输入框中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         clearDisplay()
        </code>
       </strong>
       ：清空输入框中的内容。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         deleteLast()
        </code>
       </strong>
       ：删除输入框中的最后一个字符。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         calculate()
        </code>
       </strong>
       ：使用
       <code>
        eval()
       </code>
       函数计算输入框中的表达式，并将结果显示在输入框中。如果表达式无效，则显示“错误”。
      </p>
     </li>
    </ul>
    <h3>
     5. 总结
    </h3>
    <p>
     通过这个简单的项目，我们学习了如何使用HTML、CSS和JavaScript创建一个功能齐全的网页计算器。这个计算器不仅支持基本的加减乘除运算，还具备清空、删除和错误处理功能。虽然这个计算器相对简单，但它为我们提供了一个良好的起点，可以在此基础上进一步扩展功能，如支持更复杂的数学运算、添加历史记录等。
    </p>

     希望这篇文章能帮助你理解如何构建一个简单的网页计算器，并激发你对前端开发的兴趣！
    </p>
