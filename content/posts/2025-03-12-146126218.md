    <h2>
     一、数据库的操作
    </h2>
    <p>
     打开MySQL命令行客户端，安装完MySQL后会有两个客户端，这两个客户端的区别是编码方式不同。建议使用 -Unicode这个客户端，该客户端使用的是utf8编码格式。
    </p>
    <p>
     <img alt="" height="534" src="https://i-blog.csdnimg.cn/direct/be888bf2b7c3441cb8af31579ef1e51a.png" width="1235"/>
    </p>
    <p>
     输入设置好的密码，就可以操作数据库了。
    </p>
    <p>
     <img alt="" height="927" src="https://i-blog.csdnimg.cn/direct/0f7bde0f069b42fbb6f43363da784df2.png" width="1734"/>
    </p>
    <hr/>
    <h3 id="1.1%20%E5%B1%95%E7%A4%BA%E6%95%B0%E6%8D%AE%E5%BA%93" name="1.1%20%E5%B1%95%E7%A4%BA%E6%95%B0%E6%8D%AE%E5%BA%93">
     1.1 展示数据库
    </h3>
    <pre><code class="language-sql">SHOW DATABASES;</code></pre>
    <p>
     注意：
     <span style="color:#ed7976">
      <strong>
       sql 的关键字是大小写不敏感的，使用大写或小写都可以。
      </strong>
     </span>
    </p>
    <p>
     使用该 sql 语句可以查看当前计算机中所有的数据库
     <img alt="" height="626" src="https://i-blog.csdnimg.cn/direct/9773c421475542cca6171b872e983b8d.png" width="640"/>
    </p>
    <hr/>
    <h3 id="1.2%20%E5%88%9B%E5%BB%BA%E6%95%B0%E6%8D%AE%E5%BA%93" name="1.2%20%E5%88%9B%E5%BB%BA%E6%95%B0%E6%8D%AE%E5%BA%93">
     1.2 创建数据库
    </h3>
    <pre><code class="language-sql">CREATE DATABASE [IF NOT EXISTS] db_name [CHARACTER SET charset_name];</code></pre>
    <p>
     【说明】：
    </p>
    <ul>
     <li>
      [ ] 是可选项。在写sql语句时可以带上[ ]里的内容也可以不带。
     </li>
     <li>
      <p>
       CHARACTER SET：指定数据库采用的字符集。注意：
       <span style="color:null">
        MySQL的utf8编码不是真正的utf8，没有包含某些复杂的中文字符。MySQL真正的utf8是使用utf8mb4，建议大家都使用utf8mb4。
       </span>
       <img alt="" height="756" src="https://i-blog.csdnimg.cn/direct/fef52e7d0c9e4cb4a5dcc6fbdff70ed9.png" width="1089"/>
      </p>
     </li>
    </ul>
    <ul>
     <li>
      IF NOT EXISTS 是指当计算机中不存在要创建的数据库时，允许创建。如果存在该数据库了，则不会继续创建，能避免在批量操作时因为报错而影响之后的 sql 语句。
      <img alt="" height="123" src="https://i-blog.csdnimg.cn/direct/65fc266f5de845d38e4e8834a149507a.png" width="954"/>
     </li>
    </ul>
    <hr/>
    <h3 id="%C2%A01.3%20%E4%BD%BF%E7%94%A8%E6%95%B0%E6%8D%AE%E5%BA%93" name="%C2%A01.3%20%E4%BD%BF%E7%94%A8%E6%95%B0%E6%8D%AE%E5%BA%93">
     1.3 使用数据库
    </h3>
    <pre><code class="language-sql">use 数据库名；</code></pre>
    <p>
     <img alt="" height="73" src="https://i-blog.csdnimg.cn/direct/7438c71406734e24b213d9e1eb97c65b.png" width="669"/>
    </p>
    <p>
     这表示正在使用该数据库，可以在该库中进一步操作。
    </p>
    <hr/>
    <h3 id="1.4%20%E6%9F%A5%E7%9C%8B%E5%BD%93%E5%89%8D%E6%95%B0%E6%8D%AE%E5%BA%93" name="1.4%20%E6%9F%A5%E7%9C%8B%E5%BD%93%E5%89%8D%E6%95%B0%E6%8D%AE%E5%BA%93">
     1.4 查看当前数据库
    </h3>
    <pre><code class="language-sql">select database();</code></pre>
    <p>
     该sql语句可以查看我们当前正在使用的数据库名
     <img alt="" height="339" src="https://i-blog.csdnimg.cn/direct/20ed38cb684a48a8b56dd9f005330c11.png" width="702"/>
    </p>
    <hr/>
    <h3 id="1.5%C2%A0%E5%88%A0%E9%99%A4%E6%95%B0%E6%8D%AE%E5%BA%93" name="1.5%C2%A0%E5%88%A0%E9%99%A4%E6%95%B0%E6%8D%AE%E5%BA%93" style="background-color:transparent">
     1.5 删除数据库
    </h3>
    <pre><code class="language-sql">DROP DATABASE [IF EXISTS] db_name;</code></pre>
    <p>
     <img alt="" height="234" src="https://i-blog.csdnimg.cn/direct/27f4a4870d5b4cb1adac484c4d533157.png" width="656"/>
    </p>
    <p>
     此时，我们刚刚创建的 shop 数据库已经从计算机上删除。
    </p>
    <hr/>
    <h3 id="1.6%C2%A0%E5%B0%8F%E7%BB%93" name="1.6%C2%A0%E5%B0%8F%E7%BB%93">
     1.6 小结
    </h3>
    <pre><code class="language-sql">展示数据库
SHOW DATABASES;

创建数据库
CREATE DATABASE [IF NOT EXISTS] db_name [CHARACTER SET charset_name];

使用数据库
use db_name;

查看当前数据库
select database();

删除数据库
DROP DATABASE [IF EXISTS] db_name;</code></pre>
