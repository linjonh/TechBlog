# 金蝶k3远程组件配置连接服务器,金蝶K3服务器配置工具

K3服务器配置工具

最新版本: 1.0.0

发布作者: admin

文件标签: 便捷、快速

正式版:仅供金蝶服务期内的客户使用(需联网)。

文件名称：K3服务器配置工具

最新版本：1.0.0

更新日期：2015-09-07

适用的K/3版本：11.0及以上版本

适用的操作系统版本：windows server 2008/R2 Windows server 2003

开发者：philio\_cao

功能简介：

一键设置功能方便快速部署K3服务器，工具涉及：

1、服务器角色添加，防止因角色添加不完全引起WEB以及使用K3报错。

2、防火墙设置，针对08服务器防火墙设置需要添加K3相关出入站规则以及如需开启防火墙添加K3所需端口。

3、服务器参数设置，涉及08服务器与相关所有参数以及服务器安全策略，方便快速便捷部署K3服务器，单独信任信任注册功能，可一键标识k3组件；以及提供网络修复功能，用于解决局域网网络故障。

4、集成组件修复功能，用于修复K3相关服务无法启用等情况。

5、WEB设置，涉及WEB网页常见报错修复，快速设置修复。

6、具体涉及服务器修改操作，请查看工具界面使用说明页签。

操作说明

1.  使用背景

在安装完成K3之后，为保证K3能正常使用，需要对服务器进行众多参数设置，包括防火墙、DTC参数、服务器安全策略等等，K3服务器配置工具提供了一键配置功能，部署完成K3之后无需担忧操作系统其它设置，也提供了单点设置功能，可分别对防火墙、服务器参数、WEB配置针对不同情况做出对应的修复。

2. 使用环境

1.服务器操作系统：windows server 2003/windows server 2008/windows server 2008R2

2.K3适用版本：11.0及以上版本

3.注意：工具自动检测操作系统版本，根据操作系统版本不同，将显示对应的程序界面。

3. 功能简介

使用须知

双击打开K3服务器配置工具，然后点击立即体验，程序将根据操作系统不同，出现对应的工具界面，如操作系统是Windows server 2008/2008R2,将出现如下工具界面。

显示该工具界面，点击使用说明可以了解具体对操作系统参数等修改操作，点击【我已仔细阅读使用说明页签】一键配置显示为可点击状态，程序将自动配置服务器参数以及对应服务器角色，安装中间层部件或WEB服务部件将自动调出中间层配置界面以及站点及远程组件配置界面。如不需进行一键配置功能，可进入下面②③④⑤⑥页签进行对应操作。

详细使用步骤，请下载《K3工具包帮助文件》，里面有详细步骤。
