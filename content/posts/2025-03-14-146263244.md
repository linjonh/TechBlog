    <h3>
     <a id="1__0">
     </a>
     <strong>
      1. 简介
     </strong>
    </h3>
    <p>
     Zsh（Z Shell）是一款功能强大的 Shell，相比 Bash 提供了更强的
     <strong>
      自动补全、命令高亮、插件支持
     </strong>
     等功能。而
     <strong>
      Oh My Zsh
     </strong>
     是一个
     <strong>
      Zsh 的增强管理工具
     </strong>
     ，让你可以轻松安装插件和主题，极大提高开发效率。
    </p>
    <p>
     本教程将详细介绍如何在 Linux 上安装 Oh My Zsh，并配置
     <strong>
      命令补全和语法高亮
     </strong>
     。
    </p>
    <hr/>
    <h3>
     <a id="2__Zsh_7">
     </a>
     <strong>
      2. 安装 Zsh
     </strong>
    </h3>
    <p>
     Oh My Zsh 依赖于 Zsh，因此需要先安装 Zsh。
    </p>
    <h4>
     <a id="21__Zsh_10">
     </a>
     <strong>
      2.1 安装 Zsh
     </strong>
    </h4>
    <p>
     如果系统没有安装 Zsh，可以使用以下命令安装。
    </p>
    <p>
     <strong>
      Ubuntu / Debian
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> <span class="token function">zsh</span> <span class="token parameter variable">-y</span>
</code></pre>
    <p>
     <strong>
      CentOS / RHEL
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> yum <span class="token function">install</span> <span class="token function">zsh</span> <span class="token parameter variable">-y</span>
</code></pre>
    <p>
     <strong>
      Arch Linux
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> pacman <span class="token parameter variable">-S</span> <span class="token function">zsh</span> <span class="token parameter variable">--noconfirm</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/594c4a7a6d9548b187178b17445f7ddc.png"/>
    </p>
    <h4>
     <a id="22__Zsh_31">
     </a>
     <strong>
      2.2 检查系统是否已安装 Zsh
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">zsh</span> <span class="token parameter variable">--version</span>
</code></pre>
    <p>
     如果输出类似
     <code>
      zsh 5.8
     </code>
     ，说明 Zsh 已安装，可以跳过安装步骤。
    </p>
    <h4>
     <a id="23__Shell__Zsh_37">
     </a>
     <strong>
      2.3 切换默认 Shell 为 Zsh
     </strong>
    </h4>
    <pre><code class="prism language-bash">chsh <span class="token parameter variable">-s</span> <span class="token variable"><span class="token variable">$(</span><span class="token function">which</span> <span class="token function">zsh</span><span class="token variable">)</span></span>
</code></pre>
    <p>
     然后
     <strong>
      注销并重新登录
     </strong>
     ，或者运行
     <code>
      zsh
     </code>
     进入 Zsh。
    </p>
    <hr/>
    <h3>
     <a id="3__Oh_My_Zsh_45">
     </a>
     <strong>
      3. 安装 Oh My Zsh
     </strong>
    </h3>
    <p>
     Oh My Zsh 官方提供了自动安装脚本。
    </p>
    <h4>
     <a id="31__48">
     </a>
     <strong>
      3.1 在线安装（推荐）
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sh</span> <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <p>
     如果
     <code>
      curl
     </code>
     命令不存在，可以安装它：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> <span class="token function">curl</span> <span class="token parameter variable">-y</span>   <span class="token comment"># Ubuntu / Debian</span>
<span class="token function">sudo</span> yum <span class="token function">install</span> <span class="token function">curl</span> <span class="token parameter variable">-y</span>   <span class="token comment"># CentOS / RHEL</span>
</code></pre>
    <h4>
     <a id="32__GitHub__59">
     </a>
     <strong>
      3.2 离线安装（适用于无法访问 GitHub 的环境）
     </strong>
    </h4>
    <p>
     如果你的服务器无法直接访问 GitHub，可以先在
     <strong>
      其他电脑
     </strong>
     上下载 Oh My Zsh，然后手动安装。
    </p>
    <p>
     <strong>
      步骤 1
     </strong>
     ：在
     <strong>
      可以联网的电脑
     </strong>
     上下载 Oh My Zsh：
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone <span class="token parameter variable">--depth</span><span class="token operator">=</span><span class="token number">1</span> https://github.com/ohmyzsh/ohmyzsh.git ~/ohmyzsh
</code></pre>
    <p>
     <strong>
      步骤 2
     </strong>
     ：将
     <code>
      ~/ohmyzsh
     </code>
     目录
     <strong>
      拷贝到目标服务器
     </strong>
     （比如使用 SCP）：
    </p>
    <pre><code class="prism language-bash"><span class="token function">scp</span> <span class="token parameter variable">-r</span> ~/ohmyzsh user@your_server:/home/user/
</code></pre>
    <p>
     <strong>
      步骤 3
     </strong>
     ：在目标服务器上执行安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">mv</span> ~/ohmyzsh ~/.oh-my-zsh
<span class="token function">cp</span> ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
chsh <span class="token parameter variable">-s</span> <span class="token variable"><span class="token variable">$(</span><span class="token function">which</span> <span class="token function">zsh</span><span class="token variable">)</span></span>
<span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9683da26623b4683a506889aaf6ff806.png"/>
    </p>
    <hr/>
    <h3>
     <a id="4__Oh_My_Zsh__84">
     </a>
     <strong>
      4. 配置 Oh My Zsh 插件（补全和高亮）
     </strong>
    </h3>
    <p>
     Oh My Zsh 提供了多个增强功能的插件，其中最常用的是：
    </p>
    <ul>
     <li>
      <strong>
       <code>
        zsh-autosuggestions
       </code>
      </strong>
      ：命令自动建议
     </li>
     <li>
      <strong>
       <code>
        zsh-syntax-highlighting
       </code>
      </strong>
      ：语法高亮
     </li>
     <li>
      <strong>
       <code>
        zsh-completions
       </code>
      </strong>
      ：补全增强
     </li>
    </ul>
    <h4>
     <a id="41__zshautosuggestions_90">
     </a>
     <strong>
      4.1 安装
      <code>
       zsh-autosuggestions
      </code>
      （命令自动建议）
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone <span class="token parameter variable">--depth</span><span class="token operator">=</span><span class="token number">1</span> https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions
</code></pre>
    <p>
     然后编辑
     <code>
      ~/.zshrc
     </code>
     ，找到
     <code>
      plugins=()
     </code>
     ，添加
     <code>
      zsh-autosuggestions
     </code>
     ：
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">plugins</span><span class="token operator">=</span><span class="token punctuation">(</span>git zsh-autosuggestions<span class="token punctuation">)</span>
</code></pre>
    <p>
     使配置生效：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <h4>
     <a id="42__zshsyntaxhighlighting_105">
     </a>
     <strong>
      4.2 安装
      <code>
       zsh-syntax-highlighting
      </code>
      （语法高亮）
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone <span class="token parameter variable">--depth</span><span class="token operator">=</span><span class="token number">1</span> https://github.com/zsh-users/zsh-syntax-highlighting.git ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting
</code></pre>
    <p>
     同样修改
     <code>
      ~/.zshrc
     </code>
     ，添加插件：
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">plugins</span><span class="token operator">=</span><span class="token punctuation">(</span>git zsh-autosuggestions zsh-syntax-highlighting<span class="token punctuation">)</span>
</code></pre>
    <p>
     应用更改：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <h4>
     <a id="43__zshcompletions_120">
     </a>
     <strong>
      4.3 安装
      <code>
       zsh-completions
      </code>
      （补全增强）
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone <span class="token parameter variable">--depth</span><span class="token operator">=</span><span class="token number">1</span> https://github.com/zsh-users/zsh-completions ~/.oh-my-zsh/custom/plugins/zsh-completions
</code></pre>
    <p>
     修改
     <code>
      ~/.zshrc
     </code>
     ，添加插件：
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">plugins</span><span class="token operator">=</span><span class="token punctuation">(</span>git zsh-autosuggestions zsh-syntax-highlighting zsh-completions<span class="token punctuation">)</span>
</code></pre>
    <p>
     使配置生效：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/42de95cff2c841a6b38ddd98d554b89d.png"/>
    </p>
    <hr/>
    <h3>
     <a id="5__Zsh__139">
     </a>
     <strong>
      5. 选择和配置 Zsh 主题
     </strong>
    </h3>
    <p>
     Oh My Zsh 自带多个主题，可以在
     <code>
      ~/.zshrc
     </code>
     里修改
     <code>
      ZSH_THEME
     </code>
     进行设置。
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">ZSH_THEME</span><span class="token operator">=</span><span class="token string">"robbyrussell"</span>  <span class="token comment"># 默认主题</span>
</code></pre>
    <p>
     如果想使用
     <strong>
      更高级的 Powerlevel10k 主题
     </strong>
     ，请参考
     <strong>
      下一篇教程
     </strong>
     。
    </p>
    <p>
     列出所有可用主题：
    </p>
    <pre><code class="prism language-bash"><span class="token function">ls</span> ~/.oh-my-zsh/themes/
</code></pre>
    <p>
     然后修改
     <code>
      ~/.zshrc
     </code>
     ，修改
     <code>
      ZSH_THEME
     </code>
     为你喜欢的主题：
    </p>
    <pre><code class="prism language-bash"><span class="token assign-left variable">ZSH_THEME</span><span class="token operator">=</span><span class="token string">"agnoster"</span>
</code></pre>
    <p>
     应用更改：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <hr/>
    <h3>
     <a id="6__165">
     </a>
     <strong>
      6. 常见问题
     </strong>
    </h3>
    <h4>
     <a id="61_zsh_command_not_found_zsh_167">
     </a>
     <strong>
      6.1
      <code>
       zsh: command not found: zsh
      </code>
     </strong>
    </h4>
    <p>
     说明 Zsh 没有安装，运行以下命令安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> <span class="token function">zsh</span> <span class="token parameter variable">-y</span>  <span class="token comment"># Ubuntu / Debian</span>
<span class="token function">sudo</span> yum <span class="token function">install</span> <span class="token function">zsh</span> <span class="token parameter variable">-y</span>  <span class="token comment"># CentOS / RHEL</span>
</code></pre>
    <h4>
     <a id="62_chsh_Warning_shell_not_changed_174">
     </a>
     <strong>
      6.2
      <code>
       chsh: Warning: shell not changed
      </code>
     </strong>
    </h4>
    <ul>
     <li>
      可能是因为当前用户没有权限更改 Shell，可以尝试：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> chsh <span class="token parameter variable">-s</span> <span class="token variable"><span class="token variable">$(</span><span class="token function">which</span> <span class="token function">zsh</span><span class="token variable">)</span></span> <span class="token environment constant">$USER</span>
</code></pre>
     </li>
     <li>
      或者
      <strong>
       重新登录
      </strong>
      ，然后手动运行
      <code>
       zsh
      </code>
      进入 Zsh。
     </li>
    </ul>
