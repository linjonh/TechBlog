     前端静态文件下载处理方式
    </h3>
    <h4>
     <a id="a_1">
     </a>
     a标签自带下载
    </h4>
    <ul>
     <li>
      实现方式
     </li>
    </ul>
    <pre><code class="prism language-javascript"><span class="token comment">// vue下载本地文件时，一定要把本地静态文件存放在配置文件指定的文件夹下，小编是assets目录</span>
<span class="token comment">// ie浏览器发现a标签的download属性无效</span>
<span class="token operator">&lt;</span>a href<span class="token operator">=</span><span class="token string">'./assets/download/template.pdf'</span> download<span class="token operator">=</span><span class="token string">'template'</span><span class="token operator">&gt;</span>下载模板<span class="token operator">&lt;</span><span class="token operator">/</span>a<span class="token operator">&gt;</span>
</code></pre>
    <ul>
     <li>
      问题1
      <br/>
      在a标签上添加target="_blank" 熟悉，让ie浏览器直接打开新的标签页，预览文件后下载。
     </li>
     <li>
      问题2
      <br/>
      把文件压缩成为压缩包，这样ie和谷歌都会默认下载文件！！！
     </li>
    </ul>
    <h4>
     <a id="iframe_16">
     </a>
     iframe方法
    </h4>
    <ul>
     <li>
      实现方式
     </li>
    </ul>
    <pre><code class="prism language-javascript"><span class="token keyword">let</span> eleIF <span class="token operator">=</span> document<span class="token punctuation">.</span><span class="token function">createElement</span><span class="token punctuation">(</span><span class="token string">'iframe'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <ul>
     <li>
      问题1
      <br/>
      下载文件地址路径非常重要，要确定好配置文件中区分环境后的地址，不然会导致运行可以，打包后失败
     </li>
    </ul>
