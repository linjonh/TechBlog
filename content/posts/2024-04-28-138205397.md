    <h2>
     <a id="switch_2">
     </a>
     一、switch的应用场景
    </h2>
    <p>
     在分支结构中，我们除了使用if-else或者是if-else if之外，当分支量过大的时候，我们即可选择使用switch语句进行选择
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3ee282da897028a2306a55c7d882f396.png"/>
    </p>
    <h2>
     <a id="switch_6">
     </a>
     二、switch三种语法
    </h2>
    <p>
     Java中switch的三种用法详解：
    </p>
    <ul>
     <li>
      switch 标准方式
     </li>
     <li>
      switch - &gt; 用法
     </li>
     <li>
      switch yield 用法
     </li>
    </ul>
    <h3>
     <a id="21switch__13">
     </a>
     2.1、switch 标准方式
    </h3>
    <p>
     语法：
    </p>
    <pre><code>switch (表达式) {
	case 值1:
		// 执行语句块1
		break;
	case 值2:
		// 执行语句块2
		break;
	...
	case 值N:
		// 执行语句块N
		break;
	default:
		// 如果以上所有case都不匹配，执行默认语句块
}
</code></pre>
    <p>
     case语句中指定的每个值必须具有与表达式兼容的类型。switch语句的执行顺序是，首先计算表达式的值，然后将该值与每个case的值进行比较，如果找到匹配的case，则执行该case下的语句，
     <code>
      直到遇到break语句或switch语句结束
     </code>
     。如果没有找到匹配的case，且存在default子句，则执行default子句中的语句。如果没有匹配的case且没有default子句，程序将不执行任何操作。
    </p>
    <blockquote>
     <p>
      注意：若表达式的值与case后面的值匹配则会执行该case下的语句，但是若该case下没有break，则会一直往下执行，直到遇到break语句或switch语句结束
     </p>
    </blockquote>
    <p>
     <mark>
      注意
     </mark>
     <br/>
     <strong>
      1.表达式支持的参数类型：
     </strong>
     <br/>
     JDK1.0-1.4 数据类型接受 byte、short、int、char
    </p>
    <p>
     JDK1.5 数据类型接受 byte、short、int、char、enum(枚举)、(包装类型Character、Byte、Short、Integer)
    </p>
    <p>
     JDK1.7 数据类型接受 byte、short、int、char、enum(枚举)、包装类型、String 六种类型
    </p>
    <blockquote>
     <p>
      （1）对四个包装类的支持是因为java编译器在底层手动进行拆箱。
      <br/>
      （2）而对枚举类的支持是因为枚举类有一个ordinal方法,该方法实际上是一个int类型的数值。
      <br/>
      （3）jdk1.7开始支持String类型,但实际上String类型有一个hashCode算法,结果也是int类型。
      <br/>
      （4）而byte short char类型可以在不损失精度的情况下向上转型成int类型.所以总的来说,可以认为switch中只支持int类型。
     </p>
    </blockquote>
    <p>
     <strong>
      <code>
       float、double都不能用于switch语句。
      </code>
     </strong>
    </p>
    <p>
     <strong>
      2.default 子句是可选的，当没有匹配的 case 时，执行default
     </strong>
     <br/>
     <strong>
      3.break语句用来在执行完一个 case 分支后使程序跳出 switch 语句块；如果没有写 break，程序会顺序执行到 switch 结尾，除非遇到break。
     </strong>
    </p>
    <h3>
     <a id="22switch____56">
     </a>
     2.2、switch - &gt; 用法
    </h3>
    <blockquote>
     <p>
      jdk12开始switch语法支持函数式接口和lambda表达式
     </p>
    </blockquote>
    <p>
     使用switch标准方式编写代码太多的break造成代码冗余可读性不高 可以借助函数式接口和lambda表达式简化书写
    </p>
    <p>
     代码如下：
    </p>
    <pre><code>switch (表达式) {
	case 值1 -&gt; 执行语句块1;
	case 值2 -&gt; 执行语句块2;
	......
	case 值N -&gt; 执行语句块N;	
	default:
		// 如果以上所有case都不匹配，执行默认语句块
}
</code></pre>
    <p>
     使用 - &gt; 方法更加简单，
     <code>
      -&gt;的用法其实只是相当于不用输入break就可以跳出判断
     </code>
     , 保证其只有一条分支可走.
    </p>
    <p>
     示例：
    </p>
    <pre><code>int dayOfWeek = 2;
switch (dayOfWeek) {
    case 1 -&gt; System.out.println("星期一");
    case 2 -&gt; System.out.println("星期二");
    case 3 -&gt; System.out.println("星期三");
    case 4 -&gt; System.out.println("星期四");
    case 5 -&gt; System.out.println("星期五");
    case 6 -&gt; System.out.println("星期六");
    default -&gt; System.out.println("星期日");
}
</code></pre>
    <p>
     输出结果为：
    </p>
    <pre><code>星期二
</code></pre>
    <p>
     jdk12之后还可以多条件合并
    </p>
    <pre><code>int dayOfWeek = 2;
switch (dayOfWeek) {
    case 1,2,3,4,5 -&gt; System.out.println("工作日");
    default -&gt; System.out.println("其他");
}
</code></pre>
    <h3>
     <a id="22switch_yield__103">
     </a>
     2.2、switch yield 用法
    </h3>
    <blockquote>
     <p>
      jdk14新特性，增加了yield用法
     </p>
    </blockquote>
    <pre><code>int numLetters = switch(seasonName){
	case "Spring" -&gt;{
		System.out.println("spring time");
		yield 6;
		}
	case "Summer","Winter" -&gt; 6; //可以使用yield关键字，也可以省略
	case "Fall" -&gt; 4;
	default -&gt; -1;
};
</code></pre>
    <p>
     这里来说，
     <code>
      switch 作为了 一条语句， 所以末尾要加冒号
     </code>
     ， 其他用法相同
    </p>
    <p>
     示例：
    </p>
    <pre><code>int dayOfWeek = 2;
String weekday = switch (dayOfWeek) {
    case 1:
        yield "星期一";
    case 2:
        yield "星期二";
    case 3:
        yield "星期三";
    case 4:
        yield "星期四";
    case 5:
        yield "星期五";
    case 6:
        yield "星期六";
    default:
        yield "星期日";
};
System.out.println(weekday);
</code></pre>
    <h2>
     <a id="case_140">
     </a>
     三、什么是case穿透？
    </h2>
    <p>
     在java的switch非简化语句中，当匹配到的case中没加break；的时候，那就会执行当前case里面的语句，以及这个switch语句中当前case下面的所有case中或者是default中的语句。直到遇到break；为止
    </p>
    <p>
     以下是有关case穿透的实验代码：
    </p>
    <pre><code>public static void main(String[] args) {
    String date = "周二";

    switch(date){
        case "周一":
            System.out.println("周一");
        case "周二":
            System.out.println("周二");
        case "周三":
            System.out.println("周三");
        default:
            System.out.println("其他");
    }
}

</code></pre>
    <p>
     输出周二、周三、其他
    </p>

     四、示例
    </h2>
    <h3>
     <a id="41_163">
     </a>
     4.1、标准示例
    </h3>
    <pre><code>public static void main(String[] args) {
    String date = "周一";

    switch(date){
        case "周一":
            System.out.println("工作日");
            break;
        case "周二":
            System.out.println("工作日");
            break;
        case "周三":
            System.out.println("工作日");
            break;
        default:
            System.out.println("其他");
            break;
    }
}
</code></pre>
    <h3>
     <a id="42_184">
     </a>
     4.2、错误示例
    </h3>
    <p>
     由于switch语句中整体都没加break， 所以语句会在遇到第一个符合的case开始执行，直到遇到break或switch语句结束
    </p>
    <pre><code>public static void main(String[] args) {
    String date = "周二";

    switch(date){
        case "周一":
            System.out.println("周一");
        case "周二":
            System.out.println("周二");
        case "周三":
            System.out.println("周三");
        default:
            System.out.println("其他");
    }
}
</code></pre>
    <p>
     输出周二、周三、其他

    <h3>
     <a id="43_206">
     </a>
     4.3、引申用法（多条件合并）
    </h3>
    <p>
     上面写的案例感觉很多逻辑重复的，能否多个条件合并在一起呢？当然是可以的
    </p>
    <p>
     <strong>
      方式一：
     </strong>
    </p>
    <pre><code>public static void main(String[] args) {
    String date = "周一";

    switch(date){
        case "周一":
        case "周二":
        case "周三":
            System.out.println("工作日");
            break;
        default:
            System.out.println("其他");
            break;
    }
}
</code></pre>
    <p>
     由于case "周一"后面没有跟break;所以会一直走下去，直到遇到break。因此可以利用这种方式实现少些代码。
    </p>
    <p>
     <strong>
      方式二：java14引入
     </strong>
    </p>
    <blockquote>
     <p>
      注意：此种方式需要jdk版本是jdk14及以上
     </p>
    </blockquote>
    <pre><code>public static void main(String[] args) {
   String date = "周一";

    switch(date){
        case "周一","周二","周三":
            System.out.println("工作日");
            break;
        default:
            System.out.println("其他");
            break;
    }
}
</code></pre>
