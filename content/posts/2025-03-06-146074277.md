     <a id="windowsspleeter_240_0">

     windows部署spleeter 版本2.4.0：分离音频的人声和背景音乐
    </h3>
    <h4>
     <a id="Spleeter__2">
     </a>
     <strong>
      一、Spleeter 是什么？
     </strong>
    </h4>
    <p>
     <strong>
      Spleeter
     </strong>
     是由法国音乐流媒体公司
     <strong>
      Deezer
     </strong>
     开发并开源的一款基于深度学习的音频分离工具。它能够将音乐中的不同音轨（如人声、鼓、贝斯、钢琴等）分离为独立的音频文件，适用于音乐制作、学术研究、音频处理等领域。
    </p>
    <h4>
     <a id="_6">
     </a>
     <strong>
      二、核心功能
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       多音轨分离
      </strong>
      ：
      <ul>
       <li>
        <strong>
         2stems
        </strong>
        ：分离为
        <strong>
         人声（vocals）
        </strong>
        和
        <strong>
         伴奏（accompaniment）
        </strong>
        。
       </li>
       <li>
        <strong>
         4stems
        </strong>
        ：分离为
        <strong>
         人声
        </strong>
        、
        <strong>
         鼓
        </strong>
        、
        <strong>
         贝斯
        </strong>
        和
        <strong>
         其他
        </strong>
        。
       </li>
       <li>
        <strong>
         5stems
        </strong>
        ：分离为
        <strong>
         人声
        </strong>
        、
        <strong>
         鼓
        </strong>
        、
        <strong>
         贝斯
        </strong>
        、
        <strong>
         钢琴
        </strong>
        和
        <strong>
         其他
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       高效处理
      </strong>
      ：
      <ul>
       <li>
        支持 CPU 和 GPU（需 TensorFlow GPU 版本）加速。
       </li>
       <li>
        单曲处理仅需数秒至数分钟（取决于硬件配置）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       开源免费
      </strong>
      ：
      <ul>
       <li>
        代码和预训练模型完全开源（GitHub MIT 协议）。
       </li>
       <li>
        无需商业授权，适合个人和学术用途。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_21">
     </a>
     <strong>
      三、安装过程
     </strong>
    </h4>
    <p>
     github地址: https://github.com/deezer/spleeter/tree/master
    </p>
    <pre><code>1.创建conda虚拟环境  python使用3.9
conda create -n spleeter python=3.9
conda activate spleeter

2.安装依赖ffmpeg和libsndfile
conda install ffmpeg libsndfile

3.安装最新版spleeter
pip install spleeter

4.下载测试文件
wget https://github.com/deezer/spleeter/raw/master/audio_example.mp3

5.执行
spleeter separate -p spleeter:2stems -o output audio_example.mp3
最后在命令执行路径下生成目录output/audio_example，内部有两个文件，人声音文件vocals.wav和背景音乐文件accompaniment.wav
</code></pre>
    <p>
     各种模型下载：2stems、4stems、5stems
    </p>
    <p>
     https://github.com/deezer/spleeter/releases
    </p>

    <h4>
     <a id="Windows_111">
     </a>
     <strong>
      五、Windows用户注意
     </strong>
    </h4>
    <p>
     命令
     <code>
      spleeter
     </code>
     在 Windows可能上无法正常工作。这是一个已知问题，我们希望很快修复。在命令行中替换
     <code>
      spleeter separate
     </code>
     为
     <code>
      python -m spleeter separate
     </code>
     ，应该可以正常工作。
    </p>
