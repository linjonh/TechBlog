    <h3>
     题目描述
    </h3>
    <p>
     给定一个整数 N，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例 2）。
    </p>
    <h3>
     输入格式
    </h3>
    <p>
     一个整数 N。
    </p>
    <h3>
     输出格式
    </h3>
    <p>
     一个整数，表示反转后的新数。
    </p>
    <h3>
     输入输出样例
    </h3>
    <p>
     <strong>
      输入 #1
     </strong>
     复制
    </p>
    <pre>123</pre>
    <p>
     <strong>
      输出 #1
     </strong>
     复制
    </p>
    <pre>321</pre>
    <p>
     <strong>
      输入 #2
     </strong>
     复制
    </p>
    <pre>-380</pre>
    <p>
     <strong>
      输出 #2
     </strong>
     复制
    </p>
    <pre>-83</pre>
    <h3>
     说明/提示
    </h3>
    <p>
     <strong>
      【数据范围】
     </strong>
    </p>
    <p>
     −1,000,000,000≤N≤1,000,000,000。
    </p>
    <p>
     noip2011 普及组第一题
    </p>
    <p>
    </p>
    <h3>
     <strong>
      题解
     </strong>
    </h3>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;
int main(){
    int N,x,sum=0;
    cin&gt;&gt;N;
    for(int n=N;n!=0;){
        x=abs(n%10);
        n/=10;
        sum=sum*10+x;//类栈思想,先进的数×10的次数最多,运算结束后为最高位
    }
    if(N&lt;0)
        cout&lt;&lt;-sum&lt;&lt;endl;
    else
        cout&lt;&lt;sum&lt;&lt;endl;
    return 0;
}</code></pre>
