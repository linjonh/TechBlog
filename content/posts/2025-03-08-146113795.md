    <h2>
     <a id="1Windows_0">
     </a>
     1、Windows环境
    </h2>
    <ul>
     <li>
      Jenkins进行Build steps的使用Execute Windows batch command启动微服务（Jar包），Jenkins会默认终止Shell产生的服务进程，而在命令行能够正常运行的服务进程。
     </li>
    </ul>

    <h3>
     <a id="13__21">
     </a>
     1.3、 解决方案
    </h3>
    <ul>
     <li>
      修改Execute Windows batch command的脚本，在脚本首行加入
      <code>
       BUILD_ID=dontKillMe
      </code>
      ，告诉Jenkins不要关闭进程。
     </li>
    </ul>
