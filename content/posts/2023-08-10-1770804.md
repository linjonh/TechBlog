# 参加微软创新日，技术的方向在哪里

**．引言**

今天，
2007
年
9
月
3
日

，有幸得空参见微软技术创新日（北京）。因为时程计算失误，我几乎第一个到场，公司入驻在东三环的现代汽车大厦，办公环境简洁而大方，不是风范。不过，后来会场组织不力，差不多晚半个小时开场，算是一点失误。从下午
2
点到
5
点左右，基本上一切顺利，临走领了件
T
恤留作纪念，结束了一次技术创新之旅。

这就是简短的一次参与，匆匆茫茫间，关于技术的感想油然而生。

**2**
**．与会**

下面将今天参会的流程做以梳理，希望给自己与大家更多的资源、积累和启示。会议大致分了
5
个板块，按顺序分别为：

ü

Dev-Wow
活动介绍

ü

Windows Vista
技术相关

ü

SaaS/SOA
架构相关

ü

Office Business Application
相关

ü

SQL Server 2008
相关

下面分别做以简述，思路较乱，望见谅。

**2.1 Dev-WoW**
**活动**

MS
近期将举办
Dev-WoW
活动，其目的是调动广大开发人员的创新思想，按照微软设计的大赛规则，实现基于以下技术的创新项目，并注册参见即可。好像参加即可获赠微软邮寄的相应资料，但是注意填写详细的可达地址信息。

<http://www.dev-wow.com/default.aspx>

主要关注的技术包括：

Windows Vista: WPF, Gadget

Office System: Smart Document, SharePoint

SQL Serve: Reporting Service

有意参加的尽快报名，奖品还是很诱人的，空闲的时候玩玩
WPF
、
Gadget
等玩意还能参加大赛，何乐不为呢？

参赛的情况还是参加网站信息吧，希望博客园的同志们都能拿点东西回来，

J

。

**2.2 Windows Vista**
**技术相关**

第二位出场的讲师很有口才，使得现场气氛很好，而且是
Vista
新技术的品味，自然能够吸引更多的眼球。

讲师主要从用户体验的角度分析了
.NET3.0
的方方面面，主要包括：
WPF-
表现层、
WF—
工作流、
WCF—
通信层和
Card Space—
安全认证。当然重点还是
WPF
的超酷感官。

因为我基本还没有关注过
WPF
方面的东西，所以值得分享的主要有几点：

ü


在
Expression Blend
来实现表现层，而业务逻辑的实现由
Visual Studio
来完成，在
Expression Blend
中生成的项目就是
VS
项目，因此完全将美工和设计人员分离开发；

ü


讲师谈到了对
Siverlight
和
Flash
的比较，首先
Siverlight
的开发可以将美工和设计人员相分离，而且实现的脚本语言不局限于
ActionScript
，使用
C#
、
VB
甚至
Ruby
均可实现；

ü


没有玩儿过
Gadget
，原来其原理还是相当的简单：几个
CSS
文件、几个
JS
文件、一些图片、一些
Html
文件，打包成
Zip
文件并将后缀改为
.gadget
即可。当然，详细的情况有待验证，大家去下载实例自己琢磨吧；

ü


微软开发了新的视频编码方式
VC-1
，感觉太酷了，讲师在无线环境下浏览神奇四侠都绰绰有余，技术带给我们的惊喜实在过瘾，以后在线看片：不卡了。

相关的资源及下载地址为：

Silverlight
：
<http://silverlight.net/Default.aspx>

Blend
：
<http://www.microsoft.com/Expression/default.aspx?ag=false>

**2.3 SaaS/SOA**
**架构相关**

讲师饶有兴致的讲解了：业界动态、
Web2.0
、
SaaS
和
SOA/OBA
等技术，关于
SaaS
从
Experience/Delivery/Federation/Composition/Monetization
几个方面。值得一提的是，在
Experience
时，微软将
Office
作为重要的客户端，足见对
Office2007
作为新技术平台的重视，其他的客户端还有：
Brower
、
Rich Client
。

有趣的是会场上表决，大家没有一个人举手表示了解微软的
SaaS
和
SOA
，挺无趣的，我听得也快睡着了，正赶上我每天的睡点，不多说了。

**2.4 Office Business Application**
**相关**

第四部分，以一段对口相声开场，微软讲师还是有想象力的，对活跃气氛很有经验，马上让我从睡意中醒来，但是又马上陷入另一个朦胧，那就技术在何方？

上面我强调，微软将
Office
作为新的开发平台，整合到系统架构中，这就是
Office Business Application
，简称
OBA
，相信这种力度会越来越大。讲师以实际的在线示例为例，讲解了关于一个大型公司的零件查误到维修的全过程，在讲解中包括了眼花缭乱的技术热点，主要包括：

ü

SharePoint 3.0

ü

WPF/Siverlight

ü

Visual Earth

ü

Office Communication Server

ü

InfoPath

ü

OutLook/WOA

等等，不尽而然。一个及其负责的业务系统，设计到数据库、工作流和业务定制等多方面的问题，都迎刃而解，希望给大家以启示。

大家可以上网查询关于
Contoso
的相关信息来了解。

**2.5 SQL Server 2008**
**相关**

这一部分讲解较短，因为快下班了。重点是了解关于
Reporting Service
的相关内容，讲师以
dundas
控件为例分析了
SQL Server2008
的强大功能，如欲了解，请访问：

<http://www.dundas.com/>

参会的情况就是这些，能分享的也就是这些，希望对大家的信息库有点作用。

**3.**
**思考**

参加微软技术大会，每年都有好几次，每年都有好多场，但是几乎每次的技术都有所不同，微软是一个技术创新型的公司，领导了全球的技术方向和技术眼球，日新月异，作为技术开发人员，我们好像追的太累了。就连微软讲师也坦言，还有人在用
.NET1.1
，现在都
.NET3.5beta
了，我看着讲台上一个一个粉墨登场的技术名字，不仅问了自己？

**Where is your way?**

想了想，给了自己答案和安慰：

**Pick one, not all.**

那么，大家的答案呢？肯定每个人都有自己的汉姆雷特，但是对技术我只能说，眼光和兴趣很重要。

就分享这些吧，今年的
TechEd
可能会在
11
月召开，那又是一场更大的盛会，希望有时间去参与。
