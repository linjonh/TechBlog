    <h3>
     <strong>
      首先wx.showModal()的content是不支持html代码块的。那么怎么在content中换行呢？
     </strong>
    </h3>
    <h4>
     解决方案：使用“\r\n”换行
    </h4>
    <pre><code>wx.showModal({
   title: 'showModal换行',
   content: '这是第一行\r\n这是第二行\r\n这是第三行',
   success(res) {
      if (res.confirm) {
        console.log('用户点击确定')
      } else if (res.cancel) {
        console.log('用户点击取消')
      }
   }
})</code></pre>
    <h4>
     注意：微信开发者工具上并不会换行只是会有空格，但是实际运行手机上是有换行效果的！
    </h4>
    <p>
     <strong>
      这是微信开发者工具的效果：
     </strong>
    </p>
    <p>
     <img alt="" height="252" src="https://i-blog.csdnimg.cn/blog_migrate/dd6ab79301549d025644bd8994914a19.png" width="362"/>
    </p>
    <p>
     <strong>
      这是手机上显示的效果:
     </strong>
    </p>
    <p>
     <img alt="" height="241" src="https://i-blog.csdnimg.cn/blog_migrate/93e494e378e5e69432a9bb78531cdfc2.jpeg" width="344"/>
    </p>
    <h3>
     如果想要在content里面使用变量，就要使用模板字符串。
    </h3>
    <pre><code class="hljs">	const errMes = res.retErr.map(item =&gt; item.mes + `\r\n`).join('').slice(0, -4);
    wx.showModal({
       title: 'showModal换行',
       content: `${errMes}`,
       success(res) {
          if (res.confirm) {
            console.log('用户点击确定')
          } else if (res.cancel) {
            console.log('用户点击取消')
          }
       }
    })
				</code></pre>
    <p>
     <strong>
      .join('') 将数组用""连接成为一个字符串
     </strong>
    </p>
    <p>
     <strong>
      .slice(0, -4) 截取掉数组最后一项的换行，最后一项不需要换行。
     </strong>
    </p>
    <p>
    </p>
    <p>
     <strong>
      这样就完成了微信小程序wx.showModal模态对话框中content的换行。
     </strong>
