    <h2>
     <strong>
      下载并安装 NVM
     </strong>
    </h2>
    <p>
     运行以下命令下载并安装 NVM：
    </p>
    <pre><code>curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.4/install.sh | bash</code></pre>
    <h2>
     配置环境变量
    </h2>
    <pre><code class="hljs">vim ~/.zshrc</code></pre>
    <p>
     按 i 将如下代码复制进去，control+c  ，再按 :wq完成编辑
    </p>
    <pre><code class="hljs">export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] &amp;&amp; \. "$NVM_DIR/nvm.sh"  # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] &amp;&amp; \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion</code></pre>
    <h2>
     更新配置文件
    </h2>
    <pre><code class="hljs">source ~/.zshrc</code></pre>
    <h2>
     查看安装版本
    </h2>
    <pre><code class="hljs">nvm --version</code></pre>
    <h2>
     使用 NVM 安装 Node.js
    </h2>
    <h3>
     查看可用的 Node.js 版本：
    </h3>
    <pre><code>nvm ls-remote</code></pre>
    <h3>
     安装特定版本的 Node.js：
    </h3>
    <pre><code>nvm install &lt;version&gt;  # 替换为具体版本号，例如 `nvm install 20`</code></pre>
    <p>
     也可以安装最新版本：
    </p>
    <pre><code>nvm install node</code></pre>
    <h3>
     切换 Node.js 版本：
    </h3>
    <pre><code>nvm use &lt;version&gt;</code></pre>
    <h3>
     设置默认版本：
    </h3>
    <pre><code>nvm alias default &lt;version&gt;</code></pre>
    <h2>
     验证 Node.js 和 npm 安装
    </h2>
    <p>
     安装完成后，运行以下命令验证 Node.js 和 npm 是否安装成功：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>node -v
npm -v</code></pre>
    <h2>
     安装其他工具（可选）
    </h2>
    <p>
     如果需要，可以安装 Yarn 或 pnpm：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>npm install --global yarn
npm install --global pnpm</code></pre>
    <p>
     通过以上步骤，你可以在 macOS 上使用 NVM 管理 Node.js 版本，并根据项目需求轻松切换不同版本。
    </p>
    <h2>
     安装
     <code>
      nrm
     </code>
    </h2>
    <p>
     <code>
      nrm
     </code>
     是一个用于快速切换 npm 镜像源的工具，可以帮助你提高 npm 包的下载速度。在安装
     <code>
      nrm
     </code>
     之前，请确保你已经安装了 Node.js 和 npm。
    </p>
    <h3 style="background-color:transparent">
     1. 安装命令
    </h3>
    <p>
     使用 npm 全局安装
     <code>
      nrm
     </code>
     ：
    </p>
    <pre><code>npm install -g nrm</code></pre>
    <p>
     如果安装过程中遇到网络问题，可以先切换 npm 源到国内镜像（如淘宝源），然后再安装。
    </p>
    <h3>
     2. 验证安装
    </h3>
    <p>
     安装完成后，运行以下命令查看
     <code>
      nrm
     </code>
     版本：
    </p>
    <pre><code>nrm --version</code></pre>
    <p>
     如果显示版本号，则说明安装成功。
    </p>
    <h3>
     3. 使用
     <code>
      nrm
     </code>
    </h3>
    <h4>
     查看可用的 npm 源
    </h4>
    <p>
     运行以下命令查看当前可用的 npm 镜像源：
    </p>
    <pre><code>nrm ls</code></pre>
    <p>
     输出示例：
    </p>
    <pre><code>* npm ---- https://registry.npmjs.org/
  taobao - https://registry.npm.taobao.org/
  cnpm --- https://r.cnpmjs.org/</code></pre>
    <p>
     带
     <code>
      *
     </code>
     的表示当前正在使用的源。
    </p>
    <h4>
     切换 npm 源
    </h4>
    <p>
     使用以下命令切换到指定的 npm 源，例如切换到淘宝源：
    </p>
    <pre><code>nrm use taobao</code></pre>
    <p>
     切换成功后，可以运行以下命令确认当前使用的源：
    </p>
    <pre><code>nrm current</code></pre>
    <h4>
     测试源的速度
    </h4>
    <p>
     运行以下命令测试所有源的响应速度，选择速度最快的源：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>nrm test</code></pre>
    <h4>
     添加自定义源
    </h4>
    <p>
     如果需要添加自定义的 npm 源，可以使用以下命令：
    </p>
    <pre><code>nrm add &lt;registry&gt; &lt;url&gt;</code></pre>
    <p>
     例如：
    </p>
    <pre><code>nrm add my-registry https://my.custom.registry/</code></pre>
    <h4>
     删除自定义源
    </h4>
    <p>
     使用以下命令删除自定义源：
    </p>
    <pre><code>nrm del &lt;registry&gt;</code></pre>
    <p>
     例如：
    </p>
    <pre><code>nrm del my-registry</code></pre>
    <h3>
     4. 常见问题
    </h3>
    <ul>
     <li>
      <p>
       如果安装后无法使用
       <code>
        nrm
       </code>
       命令，可能是环境变量未正确加载。尝试重新打开终端窗口或运行
       <code>
        source ~/.zshrc
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       如果
       <code>
        nrm ls
       </code>
       未显示星号或无法切换源，可以尝试卸载后重新安装：
      </p>
      <pre><code>npm uninstall nrm -g
npm install nrm -g</code></pre>
     </li>
    </ul>
    <p>
     通过以上步骤，你可以在 macOS 上快速安装和使用
     <code>
      nrm
     </code>
     来管理 npm 镜像源，从而提高 npm 包的下载速度。
    </p>
