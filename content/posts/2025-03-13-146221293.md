    <h2>
     1：返回限定行数，并且每次查询返回的值是随机的
     <br/>
    </h2>
    <pre><code class="language-sql">SELECT * FROM 
(
SELECT ename ,job FROM EMPLOYEE
order by dbms_random.value()
)
WHERE rownum &lt;= 5 ;</code></pre>
    <h2>
     2:把空值替换成现实值
     <br/>
    </h2>
    <pre><code class="language-sql">SELECT COALESCE(comm,0) FROM EMPLOYEE;</code></pre>
    <h2>
     3:排序
    </h2>
    <h3>
     <br/>
     3.1：order by 3 desc/asc
    </h3>
    <p>
     按照第三列排序，从1开始从左往右计数
    </p>
    <h3>
     <br/>
     3.2：如果需要按照员工工号升序，薪资降序
    </h3>
    <pre><code class="language-sql">SELECT * FROM employee 
ORDER BY empno ,sal desc;</code></pre>
    <p>
     两列用，隔开,如果查询里面有distinct、group by 不能使用查询列以外的列排序
     <br/>
    </p>
    <h3>
     3.3：按照员工姓名最后三位排序
     <br/>
    </h3>
    <pre><code class="language-sql">SELECT ename ,job FROM EMPLOYEE
ORDER BY substr(ename,length(ename)-2);</code></pre>
    <h3>
     3.4:按照薪资升序排序把空值放在最后
    </h3>
    <pre><code class="language-sql">SELECT ename,sal,comm FROM
(SELECT ename,sal,comm case when comm is null then 0 else 1 end as is_null
FROM EMPLOYEE)
ORDER BY is_null desc ,comm;</code></pre>
    <p>
     或者
     <br/>
    </p>
    <pre><code class="language-sql">SELECT * FROM EMPLOYEE ORDER BY comm nulls last ;</code></pre>
    <p>
     如果是 nulls first 是把空值放在前面
     <br/>
     <br/>
     如果响应降序排列，就在comm后面加上desc
     <br/>
    </p>
    <h3>
     3.5:如果职业是CLERK那就按照奖金排序，如果是其他职业就按照工资排序
     <br/>
    </h3>
    <pre><code class="language-sql">SELECT * FROM EMPLOYEE ORDER BY CASE WHEN job ='CLERK' then comm else sal end ;l</code></pre>
    <h2>
     4:合并两个表的信息
    </h2>
    <p>
    </p>
    <p>
     union过滤重复union all包含重复，如果没有特殊要求不能直接用union替换union all
    </p>
    <h3>
     <br/>
     <br/>
     4.1两个表的相同行
     <br/>
    </h3>
    <pre><code class="language-sql">SELECT ename ,empno,job,sal,deptno
FROM emp
WHERE (ename,job,sal) in 
(select ename,job,sal from emp intersect select ename,job,sal from V);</code></pre>
    <h3 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     intersect 交集
     <br/>
     <br/>
     4.2:源表和另一个表中不一样的数据
     <br/>
    </h3>
    <pre><code class="language-sql">SELECT EMPNO FROM EMP
minus
SELECT EMPNO FROM USER;</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     数据类型必须一样，minus也可以用not in ，如果混入null就都是null 要结合not exists
     <br/>
    </p>
    <pre><code class="language-sql">SELECT u.empno FROM USER u
WHERE not exists(SELECT NULL FROM EMPLOYEE e WHERE u.empno = e.empno);</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     NOT EXISTS 通常用于在主查询中排除与子查询结果匹配的行。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     它与 EXISTS 操作符相反，EXISTS 用于检查子查询是否返回了至少一行。
    </p>

    <h2>
     <br/>
     7：创建分隔列表
    </h2>
    <pre><code class="language-sql">select deptno ,
ltrim(sys_connect_by_path(ename,','),',')emps
from (select deptno,ename,row_number()over(partition by deptno order by empno) rn,  
count(*) over (partition by depnno)cnt from emp)
where level =cnt
start with rn =1
connect by prior deptno = deptno and prior rn =rn-1</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     PARTITION BY 是一个用于分析函数的关键字，它将查询返回的数据集按照指定的列进行分区，以便在每个分区内进行独立的计算。这种分区操作能够让分析函数在每个分区内执行，而不是在整个结果集上执行，从而实现更灵活的数据分析。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     sys_connect_by_path内置函数创建分隔列表
     <br/>
     ROW_NUMBER()这是一个窗口函数，它的作用是为每一行数据分配一个唯一的序号。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     PARTITION BY deptno。这部分的作用是将数据按照 deptno 列的值分成多个逻辑分区。每个分区独立处理，相当于把数据按部门分组。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     level是Oracle中用于递归查询的伪列，表示递归的层级。这里确保只取到每个部门的最后一行，因为此时level等于员工总数cnt
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     start with rn = 1：递归查询的起点是每个部门的第一行（行号为1的员工）。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     connect by prior deptno = deptno and prior rn = rn - 1：递归条件，表示基于前一行的部门编号相同，且前一行的行号比当前行的行号少1。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     sys_connect_by_path函数将递归路径中的员工名字用逗号连接起来，ltrim用于去掉最前面多余的逗号。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     sys_connect_by_path(ename, ',')：这个函数会将递归路径中的ename值用逗号连接起来，但会在最前面多加一个逗号。例如，如果路径是'Alice,Bob,Charlie'，sys_connect_by_path会返回',Alice,Bob,Charlie'。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ltrim(..., ',')：去掉最前面的逗号，使结果变为'Alice,Bob,Charlie'
    </p>
