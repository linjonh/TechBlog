    <p>
     数据库安装步骤：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#ff0000">
      sudo apt-get update
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      sudo apt install mysql-server mysql-client
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      sudo systemctl start mysql
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      sudo systemctl status mysql
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （1）在命令行登录 MySQL 数据库，并使用 mysql 数据库
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （必须使用这个库，因为 root 在这里面）
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      sudo mysql -u root -p
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      use mysql
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （2）修改连接权限，执行：
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      update user set host='%' where user ='root';
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （3）更改加密方式：
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      update user set plugin='mysql_native_password' where user ='root';
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       修改密码数据库密码
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      use mysql
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （4）
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY '123456';
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （5）授权远程连接：
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      grant all on *.* to 'root'@'%';
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       （6）执行刷新权限：
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#ff0000">
      flush privileges;
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       修改配置文件
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <br/>
     <span style="color:#ff0000">
      sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <span style="color:#222222">
       <strong>
        将
       </strong>
      </span>
     </strong>
     <span style="color:#ff0000">
      bind-address = 127.0.0.1改为0.0.0.0
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#ff0000">
      sudo service mysql restart
     </span>
    </p>
