    <h2>
     <a id="namespace_21">
     </a>
     一，namespace
    </h2>
    <h3>
     <a id="1_namespace_22">
     </a>
     1. namespace的定义
    </h3>
    <h4>
     <a id="11__23">
     </a>
     1.1. 作用域
    </h4>
    <p>
     在学习namespace前，我们先了解域的概念。
     <br/>
     在C语言中，我们学习过全局变量和局部变量，看下面这段代码：
    </p>
    <pre><code class="prism language-c"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span><span class="token string">&lt;stdio.h&gt;</span></span>
<span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
<span class="token keyword">int</span> b <span class="token operator">=</span> <span class="token number">30</span><span class="token punctuation">;</span>
<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	<span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%d\n"</span><span class="token punctuation">,</span> a<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">//输出 20</span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%d"</span><span class="token punctuation">,</span> b<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">//输出 30</span>
	<span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     全局变量的作用域是全局，局部变量的作用域是函数内，且局部变量的优先级高于全局变量。除了这两个域，C++中还有命名空间域，类域。
     <strong>
      域影响的是编译时语法查找一个变量/函数/类型的出处的逻辑
     </strong>
     ，在同一个域中出现同名，就会冲突，如：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
<span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">30</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/78c45ac905374cd49a63ea164ab4162d.png">
      <br/>
      为了解决这个问题，C++引入了
      <code>
       namespace
      </code>
      ，目的是对标识符的名称进行本地化，以避免冲突。
      <br/>
      全局域和局部域还会影响变量的生命周期，命名空间域和类域不影响变量生命周期。
     </img>
    </p>
    <h4>
     <a id="12_namespace_48">
     </a>
     1.2. namespace定义
    </h4>
    <p>
     基本语法：
     <br/>
     <strong>
      <code>
       namespace
      </code>
      关键字 + 空间名称 +
      <code>
       {}
      </code>
     </strong>
     （注意{}后不需要跟
     <code>
      ;
     </code>
     ），如：
    </p>
    <pre><code class="prism language-c">namespace tr
<span class="token punctuation">{<!-- --></span>
	<span class="token comment">// 命名课件内可定义 变量/函数/类</span>

	<span class="token keyword">int</span> <span class="token function">sub</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">,</span> <span class="token keyword">int</span> b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
		<span class="token keyword">return</span> a <span class="token operator">-</span> b<span class="token punctuation">;</span>
	<span class="token punctuation">}</span>

	<span class="token punctuation">}</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="2_namespace_68">
     </a>
     2. namespace的使用
    </h3>
    <h4>
     <a id="21_namespace_69">
     </a>
     2.1. 指定namespace
    </h4>
    <p>
     语法：
     <code>
      名称::变量名
     </code>
     <br/>
     <code>
      ::
     </code>
     是作用域解析符，用来指定要使用的明明空间中的成员
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
namespace tr <span class="token punctuation">{<!-- --></span>

<span class="token punctuation">}</span>

    <h4>
     <a id="22_namespace_83">
     </a>
     2.2. 展开namespace
    </h4>
    <h5>
     <a id="221_namespace_84">
     </a>
     2.2.1. 展开整个namespace
    </h5>
    <p>
     语法：
     <code>
      using namespace 命名空间名称
     </code>
     <br/>
     <code>
      using
     </code>
     指令会将命名空间中的所有名称引入到当前作用域（注意若有与当前作用域同名的变量，就会冲突），如：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
namespace tr <span class="token punctuation">{<!-- --></span>
	<span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
using namespace tr<span class="token punctuation">;</span> <span class="token comment">//因为全局已经有一个a，会冲突，正确可以使用tr::a来访问a</span>
</code></pre>
    <p>
     因此，展开全部成员的这种方法并不推荐，冲突风险很大
     <br/>
     正确用法：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
namespace tr <span class="token punctuation">{<!-- --></span>
	<span class="token keyword">int</span> b <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
	<span class="token keyword">int</span> c <span class="token operator">=</span> <span class="token number">30</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
using namespace tr<span class="token punctuation">;</span>
<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%d"</span><span class="token punctuation">,</span> c<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 输出30</span>
	<span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>

     <a id="222__114">
     </a>
     2.2.2. 展开单个变量

    <p>
     语法：
     <code>
      using 命名空间名称::变量名称
     </code>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>

	<span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>

</code></pre>
    <h4>
     <a id="23__131">
     </a>
     2.3. 使用注意事项
    </h4>
    <ul>
     <li>
      <code>
       namespace
      </code>
      的本质是一个域，与全局域和局部域各自独立，不同的域可以定义同名变量
     </li>
     <li>
      <code>
       namespace
      </code>
      只能定义在全局，可以嵌套定义。
     </li>
     <li>
      一个文件中通常不可以有同名的
      <code>
       namespace
      </code>
      ，如果有，编译器会合并，视作一个
     </li>
     <li>
      不同文件可以有同名的
      <code>
       namespace
      </code>
      ，链接时，编译器会合并同名的
     </li>
     <li>
      <code>
       namespace
      </code>
      未被展开或者指定时，默认在局部和全局里面找变量
     </li>
     <li>
      将
      <code>
       namespace
      </code>
      用
      <code>
       using
      </code>
      展开后，查找顺序为：局部域→命名空间域→全局域
     </li>
    </ul>

    <h2>
     <a id="C_141">
     </a>
     二，C++的输入和输出
    </h2>
    <p>
     在C++中，输入和输出（I/O）主要通过标准库中的
     <code>
      &lt;iostream&gt;
     </code>
     头文件提供的流（stream）机制来实现。C++的I/O流库提供了灵活且类型安全的方式来处理输入和输出操作。
    </p>
    <h3>
     <a id="1__144">
     </a>
     1. 核心概念
    </h3>
    <h4>
     <a id="11_Stream_146">
     </a>
     1.1. 流（Stream）
    </h4>
    <ul>
     <li>
      流是数据在源和目标之间流动的抽象。
     </li>
     <li>
      输入流（Input Stream）：从外部设备（如键盘、文件）读取数据。
     </li>
     <li>
      输出流（Output Stream）：向外部设备（如屏幕、文件）写入数据。
     </li>
    </ul>
    <h4>
     <a id="12__151">
     </a>
     1.2. 标准流对象
    </h4>
    <p>
     C++预定义了以下标准流对象：
    </p>
    <ul>
     <li>
      <strong>
       <code>
        std::cin
       </code>
      </strong>
      ：标准输入流，通常与键盘输入关联。
     </li>
     <li>
      <strong>
       <code>
        std::cout
       </code>
      </strong>
      ：标准输出流，通常与屏幕输出关联。
     </li>
     <li>
      <strong>
       <code>
        std::cerr
       </code>
      </strong>
      ：标准错误流，用于输出错误信息（无缓冲）。
     </li>
     <li>
      <strong>
       <code>
        std::clog
       </code>
      </strong>
      ：标准日志流，用于输出日志信息（有缓冲）。
     </li>
    </ul>
    <p>
     主要了解前两个
    </p>
    <h3>
     <a id="2__159">
     </a>
     2. 基本输入和输出操作
    </h3>
    <h4>
     <a id="21__161">
     </a>
     2.1. 输出
    </h4>
    <ul>
     <li>
      <p>
       使用
       <code>
        std::cout
       </code>
       和插入运算符
       <code>
        &lt;&lt;
       </code>
       将数据输出到屏幕。
      </p>
     </li>
     <li>
      <p>
       示例：
      </p>
      <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">int</span> num <span class="token operator">=</span> <span class="token number">42</span><span class="token punctuation">;</span>
    std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Hello, World!"</span> <span class="token operator">&lt;&lt;</span> std<span class="token double-colon punctuation">::</span>endl<span class="token punctuation">;</span>  <span class="token comment">// 输出字符串并换行</span>
    std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"The number is: "</span> <span class="token operator">&lt;&lt;</span> num <span class="token operator">&lt;&lt;</span> std<span class="token double-colon punctuation">::</span>endl<span class="token punctuation">;</span>  <span class="token comment">// 输出变量</span>
    <span class="token comment">// 都是以字符串的形式输出，相当于把 &lt;&lt; 中间的字符串合并后输出，一个&lt;&lt;   &lt;&lt;之间只能放一个变量</span>
    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <code>
        std::endl
       </code>
       ：插入一个换行符并刷新输出缓冲区
      </p>
     </li>
    </ul>
    <h4>
     <a id="22__179">
     </a>
     2.2. 输入
    </h4>
    <ul>
     <li>
      使用
      <code>
       std::cin
      </code>
      和提取运算符
      <code>
       &gt;&gt;
      </code>
      从键盘读取数据。
     </li>
     <li>
      示例：
      <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Enter your age: "</span><span class="token punctuation">;</span>
    std<span class="token double-colon punctuation">::</span>cin <span class="token operator">&gt;&gt;</span> age<span class="token punctuation">;</span>  <span class="token comment">// 从键盘读取输入并存储到变量 age，多个变量输入时，可用空格，换行符等符号间隔</span>
    std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"You are "</span> <span class="token operator">&lt;&lt;</span> age <span class="token operator">&lt;&lt;</span> <span class="token string">" years old."</span> <span class="token operator">&lt;&lt;</span> std<span class="token double-colon punctuation">::</span>endl<span class="token punctuation">;</span>
    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <strong>
       注意
      </strong>
      ：
      <code>
       std::cin
      </code>
      会跳过空白字符（如空格、制表符、换行符）。
     </li>
    </ul>
    <p>
     <strong>
      <code>
       cin
      </code>
      和
      <code>
       cout
      </code>
      <mark>
       自动判断类型
      </mark>
     </strong>
    </p>
    <h3>
     <a id="3__197">
     </a>
     3. 格式化输出
    </h3>
    <p>
     虽然C++提供了多种方式格式化输出，但是我们任然可以继续使用C语言的格式化输出方式，因为C++兼容C语言
     <br/>
     下面介绍C++的格式化输出：
    </p>
    <h4>
     <a id="31__200">
     </a>
     3.1. 使用流操纵符
    </h4>
    <ul>
     <li>

     </a>
     三，缺省参数
    </h2>
    <p>
     <strong>
      缺省参数（Default Arguments）允许函数在定义时为某些参数指定默认值
     </strong>
     。如果调用函数时没有为这些参数提供实参，则使用默认值。
    </p>
    <h3>
     <a id="1__289">
     </a>
     1. 缺省参数的基本用法
    </h3>
    <h4>
     <a id="11__290">
     </a>
     1.1. 语法
    </h4>
    <pre><code class="prism language-cpp">返回值类型 函数名<span class="token punctuation">(</span>参数类型 参数名 <span class="token operator">=</span> 默认值<span class="token punctuation">)</span><span class="token punctuation">;</span>

     <a id="22__294">
     </a>

    </h4>

    <h3>
     <a id="2__315">
     </a>
     2. 缺省参数的规则
    </h3>
    <h4>
     <a id="21__317">
     </a>
     2.1. 从右向左设置缺省参数
    </h4>
    <ul>
     <li>
      缺省参数必须
      <strong>
       从右向左依次设置，不能跳过中间的参数
      </strong>
      。
     </li>
     <li>
      示例：
      <pre><code class="prism language-cpp"><span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">,</span> <span class="token keyword">int</span> b <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">,</span> <span class="token keyword">int</span> c <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 正确</span>
<span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">5</span><span class="token punctuation">,</span> <span class="token keyword">int</span> b<span class="token punctuation">,</span> <span class="token keyword">int</span> c <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">)</span><span class="token punctuation">;</span>   <span class="token comment">// 错误：b 没有默认值</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="22__325">
     </a>
     2.2. 函数声明和定义
    </h4>
    <ul>
     <li>
      如果函数在声明时指定了缺省参数，定义时不能再重复指定。
     </li>
     <li>
      示例：
      <pre><code class="prism language-cpp"><span class="token comment">// 声明时指定缺省参数</span>

    <h2>
     <a id="_368">
     </a>
     四，函数重载
    </h2>
    <p>
     <strong>
      函数重载（Function Overloading）
     </strong>
     是C++中的一种特性，允许在
     <strong>
      同一个作用域内定义多个同名函数
     </strong>
     ，但这些函数的参数列表必须不同（参数的类型、数量或顺序不同）。
    </p>
    <h3>
     <a id="1__371">
     </a>
     1. 函数重载的基本规则
    </h3>
    <h4>
     <a id="11__373">
     </a>
     1.1. 参数列表必须不同
    </h4>
    <ul>
     <li>
      函数重载的核心是参数列表的不同。参数列表的不同可以体现在：
      <ul>
       <li>
        参数的
        <strong>
         类型不同
        </strong>
        。
       </li>
       <li>
        参数的
        <strong>
         数量不同
        </strong>
        。
       </li>
       <li>
        参数的
        <strong>
         顺序不同
        </strong>
        （如果类型不同）。
       </li>
      </ul>
     </li>
     <li>
      注意：返回值类型不同
      <strong>
       不能
      </strong>
      作为函数重载的依据。
     </li>
    </ul>
    <h4>
     <a id="12__380">
     </a>
     1.2. 示例
    </h4>
    <p>
     下面共五个重载
    </p>
    <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>

<span class="token keyword">void</span> <span class="token function">print</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
 std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Printing int: "</span> <span class="token operator">&lt;&lt;</span> a <span class="token operator">&lt;&lt;</span> std<span class="token double-colon punctuation">::</span>endl<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token keyword">void</span> <span class="token function">print</span><span class="token punctuation">(</span><span class="token keyword">double</span> a<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token comment">// 类型不同</span>

     <a id="13__425">
     </a>
     1.3. 避免歧义调用
    </h4>
    <ul>
     <li>
      <p>
       示例一：
      </p>
      <pre><code class="prism language-cpp"><span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">,</span> <span class="token keyword">double</span> b <span class="token operator">=</span> <span class="token number">3.14</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">)</span><span class="token punctuation">;</span>

</code></pre>
     </li>
     <li>
      <p>
       示例二：
      </p>
     </li>
    </ul>
    <pre><code class="prism language-c"><span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a <span class="token operator">=</span> <span class="token number">3</span><span class="token punctuation">,</span> <span class="token keyword">double</span> b <span class="token operator">=</span> <span class="token number">3.14</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token keyword">void</span> <span class="token function">func</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">,</span> <span class="token keyword">double</span> b<span class="token punctuation">,</span> <span class="token keyword">int</span> c<span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token function">func</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 会调用第一个，因为没参数，编译器可判断</span>
