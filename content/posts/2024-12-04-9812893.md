      一、安装环境
     </span>
    </div>
    <div class="article_content" id="article_content" style="margin:20px 0px 0px; font-size:14px; line-height:26px; font-family:Arial">
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
     </p>
     <div class="item-content" id="main-content" style="line-height:25px; margin:0px auto; padding:0px; border-top-width:0px; border-right-width:0px; border-left-width:0px; border-bottom-style:none; color:rgb(102,102,102); font-size:14px; overflow:hidden; font-family:Verdana,Arial,Helvetica,宋体,sans-serif">
      <div style="line-height:22px; margin:0px auto; padding:0px; border:0px">
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        硬件要求：
        <br/>
        大约 55 GB 的磁盘空间
        <br/>
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        操作系统要求：
        <br/>
        Windows XP Service Pack 3
        <br/>
        Windows Server 2003 Service Pack 1
        <br/>
        Windows Server 2003 R2
        <br/>
        Windows Vista
        <br/>
        Windows Server 2008
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        windows 7
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        软件要求：
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        Visual Studio 2008
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        MSDN Library for Visual Studio 2008
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        Visual Studio 2008 SP1
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        Expression Blend 3
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        Windows Embedded Compact 7
        <br/>
        <br/>
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        安装前提：
        <br/>
        系统上从未安装过 Windows CE, Windows Embedded CE 或 Windows Embedded Compact
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        三、安装步骤
        <br/>
        1、安装操作系统（如XP、win7）
        <br/>
        2、安装 VS2008
        <br/>
        3、安装 MSDN Library for Visual Studio 2008
        <br/>
        4、安装 VS2008 SP1 (带有SP1的版本可省略此步骤.装完切记要打上SP1补丁，不然的话在安装wince7.0的时候会没有PB插件，没法建立PB工程；)
        <br/>
        5、安装 Expression Blend 3 （不使用 Silverlight 技术可省略此步骤）
        <br/>
        6、安装 Windows Embedded Compact 7
        <br/>
        7、安装 Virtual PC 2007 （此步骤可选,一般没用）
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        <span style="font-family:Arial; line-height:26px; text-align:right">
         备注：
        </span>
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        <span style="font-family:Arial; line-height:26px; text-align:right">
         1.vs2008安装文件，我安装的破解版大概有大约7G，包含SP1；
        </span>
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
        <span style="font-family:Arial; line-height:26px">
         2.wince7.0安装文件，共分为两个盘，DISK2是DISK1的一部分，开始安装前，将DISK2拷到DISK1的OS目录下，构成完整的wince7.0安装文件，不然的话，会在安装过程中从网上下载相应的安装文件，这个过程是很漫长的，而且下载的东西安装完成还不一定能用。
        </span>
       </p>
       <p style="line-height:25px; margin-top:10px; margin-bottom:10px">
       </p>
       <div class="article_content" id="article_content" style="margin:20px 0px 0px; font-size:14px; line-height:26px; font-family:Arial">
        <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
         开始安装：
        </p>
        <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
         <span style="white-space:pre">
         </span>
         1.先安装VS2008，MSDN可以选择安装，装完切记要打上SP1补丁，不然的话在安装wince7.0的时候会没有PB插件，没法建立PB工程；
        </p>
        <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px">
         <span style="white-space:pre">
         </span>
         2.安装wince7.0开发环境，安装时选择platform build，语言，以及相应的CPU类型：如：ARMv5,ARMv6,ARMv7，X86等，选择安装就OK了。这个安装过程需要一会儿，等安装完成后就可以在vs2008中新建PB工程了，接下来就是定制wince系统了，主要是选择相应的wince组件，具体根据你的CPU类型，以及你的主板硬件配置来选择，后面再用专门的文章来记录。
        </p>
       </div>
