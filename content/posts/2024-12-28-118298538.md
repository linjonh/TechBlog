        Unity

     <a id="Unity_88">
     </a>
     Unity常用组件学习
    </h5>
    <ul>
     <li>
      <strong>
       组件
      </strong>
      是Unity中很重要的一部分，在我之前的文章中也已经介绍过了
     </li>
     <li>
      一个游戏对象正是因为自身挂载的
      <strong>
       组件
      </strong>
      不同，才致使他们的功能有所不同！
     </li>
    </ul>

      Shader

      </p>
     </li>
     <li>
      <p>

    <h5>
     <a id="Unity_172">
     </a>
     Unity热更新学习
    </h5>
    <p>
     <strong>
      热更新
     </strong>
     ，可能有小伙伴不理解
     <strong>
      热更新
     </strong>
     是啥
    </p>
    <ul>
     <li>
      <p>
       这里举个例子介绍一些，比如王者荣耀经常会更新版本
      </p>
     </li>
     <li>
      <p>
       小版本更新的时候，我们不需要去应用商店重新下载更新，而是在游戏中会跳出一个更新包，直接点击下载完就可以继续游戏了
      </p>
     </li>
     <li>
      <p>
       都不要重新打开游戏就行。
      </p>
     </li>
     <li>
      <p>
       当然在大版本更新的时候，还是需要下载完重新安装，这也是没办法啦~
      </p>
     </li>
     <li>
      <p>
       <strong>
        热更新
       </strong>
       这块是使用
       <strong>
        Lua语言
       </strong>
       编写的
      </p>
     </li>
     <li>
      <p>
       要使用
       <strong>
        热更新
       </strong>
       就需要再学习一门语言，就是
       <strong>
        Lua
       </strong>
       了
      </p>
     </li>
     <li>
      <p>
       网上也是有几种常用的Lua框架可以使用，感兴趣的小伙伴也可以去学习一下
      </p>
     </li>
     <li>
      <p>
       有些游戏甚至是
       <strong>
        全部使用Lua语言
       </strong>
       做出来的，所以
       <strong>
        Lua
       </strong>
       也是一个重点哦~
      </p>
     </li>
    </ul>
