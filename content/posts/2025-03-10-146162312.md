     <span style="color:red">
      】

    <p style="margin-left:0; margin-right:0; text-align:justify">

     </span>
     <span style="color:#00b050">
      来合并，不会去除两个结果中重复的记录
     </span>
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      intersect 合并查询结果的交集
     </span>
    </h4>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     如果MYSQL 不支持intersect关键字 ，可以使用连接查询来替换实现 ( INNER JOIN)
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      except 差集
     </span>
    </h4>
    <h3 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      子查询
     </span>
    </h3>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     一个查询的条件需要用到另一个查询的结果
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     比如：想查询等级大于平均等级的玩家
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      SELECT *FROM player WHERE LEVEL &gt; (SELECT AVG(LEVEL)FROM player);
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      SELECT LEVEL, ROUND((SELECT AVG(LEVEL) FROM player)) AS average,
     </span>
     <span style="color:black">
      round
     </span>
     <span style="color:black">
      是四舍五入
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      LEVEL - ROUND((SELECT AVG(LEVEL) FROM player)) AS diff
     </span>
     <span style="color:black">
      AS xx
     </span>
     <span style="color:black">
      是重命名列名
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      FROM player
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      子查询要用括号包裹
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      ROUND(SELECT AVG(LEVEL) FROM player) --
     </span>
     <span style="color:#00b050">
      错误
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      ROUND((SELECT AVG(LEVEL) FROM player)) --
     </span>
     <span style="color:#00b050">
      正确
     </span>
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      使用子查询创建一个新的表
     </span>
    </h4>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      CREATE TABLE new_player SELECT * FROM player WHERE LEVEL&lt;5
     </span>
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      使用子查询插入数据
     </span>
    </h4>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     insert into
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      INSERT INTO new_player SELECT * FROM player WHERE LEVEL BETWEEN 6 AND 10;
     </span>
    </p>
    <h4 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      判断是否存在
     </span>
    </h4>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      SELECT EXISTS (SELECT * FROM player WHERE LEVEL &gt; 10)
     </span>
    </p>
    <h3 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      表关联
     </span>
    </h3>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     INNER JOIN
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     LEFT JOIN 左连接
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     RIGHT JOIN 右连接
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      SELECT *FROM student
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      INNER JOIN course
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      ON student.Sno = course.Cno;
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      用
     </span>
     <span style="color:#00b050">
      where
     </span>
     <span style="color:#00b050">
      一样
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      表连接
     </span>
     <span style="color:#00b050">
      =
     </span>
     <span style="color:#00b050">
      笛卡尔积
     </span>
     <span style="color:#00b050">
      +
     </span>
     <span style="color:#00b050">
      条件过滤
     </span>
    </p>
    <h3 style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b0f0">
      索引
     </span>
    </h3>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      CREATE [UNIQUE|FULLTEXT|SPATIAL] INDEX index_name
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#00b050">
      ON table_name(index_col_name,……)
