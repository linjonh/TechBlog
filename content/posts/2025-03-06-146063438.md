    <h2>
     <a id="_0">
     </a>
     现状
    </h2>
    <ol>
     <li>
      当从网上下载开源软件的appImage之后； 可以右键创建桌面快捷方式
     </li>
     <li>
      但想用这个软件打开对应类型的文件, 文件-&gt;右键-&gt;找不到下载的开源软件
     </li>
     <li>
      因此，需要把这个开源软件添加到Applications列表中。这样，文件-&gt;右键-&gt;就能找到开源软件
     </li>
    </ol>
    <h2>
     <a id="_4">
     </a>
     创建快捷方式流程
    </h2>
    <ol>
     <li>
      在桌面创建desktop类型的文件MarkText.desktop [以MarkText软件为例]
     </li>
     <li>
      文件内容如下：
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token punctuation">[</span>Desktop Entry<span class="token punctuation">]</span>
<span class="token assign-left variable">Name</span><span class="token operator">=</span>MarkText
<span class="token assign-left variable">Comment</span><span class="token operator">=</span>Markdown Type Utility
<span class="token assign-left variable">Exec</span><span class="token operator">=</span>/usr/local/marktext-x86_64.AppImage <span class="token comment">#软件存放位置</span>
<span class="token assign-left variable">Icon</span><span class="token operator">=</span>/home/WUYING_zswb_197_19853712520/DevSoft/MarkText.png <span class="token comment">#软件图标存放的位置</span>
<span class="token assign-left variable">Type</span><span class="token operator">=</span>Application <span class="token comment">#类型为Application，这样在软件列表中就能找到</span>
<span class="token assign-left variable">StartupNotify</span><span class="token operator">=</span>false 
<span class="token assign-left variable">Terminal</span><span class="token operator">=</span>false <span class="token comment"># 是否终端开发</span>
<span class="token assign-left variable">Categories</span><span class="token operator">=</span>TextTools<span class="token punctuation">;</span>Utility <span class="token comment">#软件所属目录</span>
</code></pre>
    <ol start="3">
     <li>
      创建之后，利用sudo mv命令把文件放在/usr/share/applications文件夹下面
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token function">sudo</span> <span class="token function">mv</span> ./MarkText.desktop /usr/share/applications
</code></pre>
    <ol start="4">
     <li>
      执行sudo mv命令执行之后，可以在软件列表搜到相应的软件
     </li>
     <li>
      同时，右键-&gt;打开方式-&gt;也能找到相应的软件
     </li>
    </ol>
