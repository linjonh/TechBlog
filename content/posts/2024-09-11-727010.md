# 六款微软产品基于Windows工作流平台

有六种微软产品基于Windows工作流平台。

**微软Office SharePoint Server 2007**
，现在为beta2状态，计划10月份正式发布：目标为下一个版本的Office提供工作流功能。

一个新的产品叫作
**System Center “Service Desk”**
将在2007年下半年交付。这个产品将使用WWF和Office2007作为基础，实现自动人机交互。过程、技术和一套核心的MOF流程将被作为解决方案的一部分加入近来。

**微软Speech Server 2007**
现在仍为beta阶段，它基于WWF技术。

**Biztalk Server v.Next**
:提供下一代商业过程管理平台。

**微软Identity Integration Server**
:为MIIS3.5提供用户供应管理流程。

**微软Dynamics**
:以后的产品将基于WWF。
