         <span style="color:#333333">
          <strong>
           一、按核心参数匹配需求
          </strong>
         </span>
        </span>
       </strong>
      </strong>
     </span>
    </h5>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         光谱范围
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         农业监测
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌：需覆盖可见光至近红外（400-1000nm），以捕捉作物叶绿素、水分等特征‌。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         地质勘探
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌：需宽光谱（350-2500nm）及高分辨率（3-10nm），可识别矿物吸收峰‌。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         工业检测
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌：优先可见光波段（400-700nm）的高分辨率线扫描相机
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         光谱分辨率
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         精细化学分析
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌（如医学、药品检测）：需2.5nm以下高分辨率
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         广域分类
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌（如农业、环境监测）：10nm左右分辨率即可满足‌。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         信噪比（SNR）与速度
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         动态监测
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌（如无人机航拍）：需高信噪比（SNR≥500:1）及快速采样，避免图像模糊‌。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌
      </span>
     </span>
     <strong>
      <span style="background-color:#ffffff">
       <span style="color:#333333">
        <strong>
         实验室静态分析
        </strong>
       </span>
      </span>
     </strong>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       ‌：可降低速度要求，优先辐射测量精度‌。
      </span>
     </span>
    </p>
    <h5 style="margin-left:0.0000pt; margin-right:0.0000pt; text-align:left">
     <span style="background-color:#ffffff">

      </span>
     </span>
    </p>

     <span style="background-color:#ffffff">
      <span style="color:#333333">
       1、
      </span>
     </span>

     </span>
    </p>

     <span style="background-color:#ffffff">
      <span style="color:#333333">
       2、
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       体积小：与普通COMS传感器大小一致，工艺成熟度高；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       3、使用方便：单次拍摄、无需时间扫描，信号实时处理，无延迟；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#333333">
       4、性价比高：在市面上，汇能感知的光谱相机/模块产品具有更高性价比。

     </span>
