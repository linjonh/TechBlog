     二、传染病模型的基本概念
    </h3>
    <h4>
     <a id="11_SIR_12">
     </a>
     1.1 SIR模型
    </h4>
    <p>
     首先登场的是我们的主角——
     <strong>
      SIR模型
     </strong>
     。SIR代表易感者(Susceptible)、感染者(Infected)和康复者(Recovered)。
    </p>
    <h5>
     <a id="111__15">
     </a>
     1.1.1 模型假设
    </h5>
    <ul>
     <li>
      <strong>
       人口固定
      </strong>
      ：没有出生和死亡。
     </li>
     <li>
      <strong>
       均匀混合
      </strong>
      ：每个人都有可能接触到其他人。
     </li>
     <li>
      <strong>
       立即感染
      </strong>
      ：易感者接触感染者后立即成为感染者。
     </li>
    </ul>

    <h3>
     <a id="_229">
     </a>
     六、模型的局限性与改进方向
    </h3>
    <h4>
     <a id="61__231">
     </a>
     6.1 模型假设的局限
    </h4>
    <ul>
     <li>
      <strong>
       人口固定
      </strong>
      ：现实中存在出生、死亡和迁移。
     </li>
     <li>
      <strong>
       均匀混合
      </strong>
      ：忽略了社交网络结构。
     </li>
     <li>
      <strong>
       参数恒定
      </strong>
      ：传染率和康复率可能随时间变化。
     </li>
    </ul>
    <h4>
     <a id="62__237">
     </a>
     6.2 改进方向
    </h4>
    <ul>
     <li>
      <strong>
       引入年龄结构
      </strong>
      ：不同年龄段的传染率不同。
     </li>
     <li>
      <strong>
       考虑地理因素
      </strong>
      ：使用空间模型。
     </li>
     <li>
      <strong>
       参数动态调整
      </strong>
      ：结合实时数据，使用机器学习方法。
     </li>
    </ul>
