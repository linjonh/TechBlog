     <li>
      方式1：JAVA中我们可以使用java.util.Collections类的sort(List list)方法对list集合中的元素排序。
     </li>
     <li>
      方式2：JDK8之后特别是lambda表达式的盛行，而且Collections的sort方法其实是调用了List接口自己的sort方法；所以可以使用List接口自己的sort方法排序
     </li>
     <li>
      方式3：Stream流的sort方法写法
     </li>
    </ol>
    <h4>
     一：
     <strong>
      集合元素是基本类型包装类型
     </strong>
    </h4>
    <pre><code class="language-java">public static void main(String[] args)  {
        List&lt;Integer&gt; numList=new ArrayList&lt;&gt;();
        numList.add(999);
        numList.add(123);
        numList.add(456);
        numList.add(66);
        numList.add(9);
 
        Collections.sort(numList);  //使用Collections类的方法排序
 
        numList.sort(new Comparator&lt;Integer&gt;() {//使用List接口的方法排序
            @Override
            public int compare(Integer o1, Integer o2) {
                return o1.compareTo(o2);
            }
        });
       
        //lambda表达式实现List接口sort方法排序
        numList.sort((num1,num2)-&gt;{return num1.compareTo(num2);});
 
        System.out.println(numList);
    }</code></pre>
    <h4>
     二：
     <strong>
      对象的集合根据某个属性排序
     </strong>
    </h4>
    <blockquote>
     <p>
      例如：对User对象集合根据User类的age属性排序，默认是升序。
      <span style="color:#fe2c24;">
       <strong>
        不推荐，不推荐，不推荐
       </strong>
      </span>
     </p>
    </blockquote>
    <pre><code class="language-java">
public static void main(String[] args)  {
	List&lt;User&gt; numList=new ArrayList&lt;&gt;();
	User u=new User();
	u.setAge(12);
	numList.add(u);

	User u1=new User();
	u1.setAge(34);
	numList.add(u1);

	User u2=new User();
	u2.setAge(6);
	numList.add(u2);

	User u3=new User();
	u3.setAge(99);
	numList.add(u3);

	//Collections类的sort方法对对象集合排序，要传集合和Comparator接口两个参数
	Collections.sort(numList, new Comparator&lt;User&gt;() {
		@Override
		public int compare(User o1, User o2) {
			Integer age1= o1.getAge();
			Integer age2= o2.getAge();
			return  age1.compareTo(age2);
		}
	});
	//List接口自身的sort方法对对象集合排序，重写Comparator接口方法即可
	numList.sort(new Comparator&lt;User&gt;() {
		@Override
		public int compare(User u1, User u2) {
			Integer age1= u1.getAge();
			Integer age2= u2.getAge();
			return  age1.compareTo(age2);
		}
	});
	
	//List接口的sort方法，lambda表达式写法
   numList.sort((u4,u5)-&gt;{
		Integer age1= u4.getAge();
		Integer age2= u5.getAge();
		return  age1.compareTo(age2);
	});
	System.out.println(numList);
}</code></pre>
    <h4>
     三：
     <strong>
      使用Stream流排序
     </strong>
    </h4>
    <blockquote>
     <p>
      1.首先你需要
      <code>
       list.parallelStream().sorted
      </code>
      进行流处理，使用
      <code>
       parallelStream
      </code>
      可以充分调度多核CPU。
      <br/>
      2.使用
      <code>
       Comparator.comparing
      </code>
      进行排序，
      <code>
       reversed()
      </code>
      进行倒序排列，
      <code>
       thenComparing
      </code>
      进行下一个排序。
      <br/>
      3.
      <code>
       Comparator.comparing()
      </code>
      里面的内容，也是就是
      <code>
       Object::getter
      </code>
      ，例如
      <code>
       KeywordCounterDTO::getKeyword
      </code>
      <br/>
      4.最后格式化为需要的格式
      <code>
       List
      </code>
      是
      <code>
       .collect(Collectors.toList())
      </code>
      ， Map 是
      <code>
       .collect(Collectors.toMap(KeywordCounterDTO::getKey, KeywordCounterDTO::getValue))
      </code>
     </p>
    </blockquote>
    <pre><code class="language-java">根据年龄倒序排

public static void main(String[] args)  {
	List&lt;User&gt; numList=new ArrayList&lt;&gt;();
	User u=new User();
	u.setAge(12);
	numList.add(u);

	User u1=new User();
	u1.setAge(34);
	numList.add(u1);

	User u2=new User();
	u2.setAge(6);
	numList.add(u2);

	User u3=new User();
	u3.setAge(99);
	numList.add(u3);

	numList = numList.stream().sorted(Comparator.comparing(User::getAge).reversed()).collect(Collectors.toList());
}
</code></pre>
    <pre><code class="language-java">先对学生的班级做排序，然后同班级做倒序
public static void main(String[] args)  {
	List&lt;User&gt; numList=new ArrayList&lt;&gt;();
	User u=new User();
	u.setClass(1);
	u.setAge(12);
	numList.add(u);

	User u1=new User();
	u.setClass(1);
	u1.setAge(34);
	numList.add(u1);

	User u2=new User();
	u.setClass(2);
	u2.setAge(6);
	numList.add(u2);

	User u3=new User();
	u.setClass(2);
	u3.setAge(99);
	numList.add(u3);

	numList = numList.parallelStream().sorted(
	  Comparator.comparing(User::getClass).reversed().thenComparing(User::getAge)
	).collect(Collectors.toList());
}</code></pre>
    <blockquote>
     <p>
      当集合中存在null元素时，可以使用针对null友好的比较器，null元素排在集合的最前面：
      <span style="color:#fe2c24;">
       <strong>
        nullsFirst
       </strong>
      </span>
     </p>
    </blockquote>
    <pre><code class="language-java">public static void main(String[] args)  {
	List&lt;User&gt; numList=new ArrayList&lt;&gt;();
	User u=new User();
	u.setClass(1);
	u.setAge(12);
	numList.add(u);

	User u1=new User();
	u.setClass(1);
	u1.setAge(34);
	numList.add(u1);

	User u2=new User();
	u.setClass(2);
	u2.setAge(6);
	numList.add(u2);

	User u3=new User();
	u.setClass(2);
	u3.setAge(99);
	numList.add(u3);
	
	numList = numList.parallelStream().sorted(
	  Comparator.nullsFirst(User::getClass).reversed().thenComparing(User::getAge)
	).collect(Collectors.toList());
}</code></pre>
