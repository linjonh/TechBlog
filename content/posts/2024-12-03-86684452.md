# 微信小程序开发必看，《使用 .NET Core + DevOps 开发微信跨平台应用》，苏州.NET俱乐部课程分享...

**![640?wx_fmt=jpeg](https://i-blog.csdnimg.cn/blog_migrate/a173be836942194dd026087776f6e9b7.jpeg)**

**【课程名称】**

《使用 .NET Core + DevOps 开发微信跨平台应用》

**【老师介绍】**

苏震巍，苏州盛派网络
CEO，微软最有价值专家（MVP）、微软 Ignite 技术大会讲师，Senparc.Weixin作者，《微信开发深度解析》等书作者。苏州市人工智能学会理事、“中国HTML5研究小组”成员，姑苏优秀创业人物。

**【课程简介】**

使用 .NET Core 快速开发跨平台的公众号及小程序应用，并使用 DevOps 方式实现开发和运维过程中的一些环节的自动化。

**【会场花絮】**

苏总也是微软技术俱乐部（苏州）成立大会暨微软技术交流会活动发起人之一，是苏州.NET俱乐部的现任主席和现任的微软最有价值专家（MVP）。在 MVP 大家庭中，不乏有技术拔尖，同时又投身创业的人士，苏总就是其中的代表之一。他创办的盛派网络，围绕移动化开发的解决方案，取得了不错的业绩。

![640?wx_fmt=jpeg](https://i-blog.csdnimg.cn/blog_migrate/2581bf578ba731960b2af3b470b7aee1.jpeg)

图示：苏主席与陈主席握手广州

盛派网络广为外界所知的是他们开源的一套微信开发SDK，可以极大地提高微信生态开发人员的工作效率，并且减少错误的发生。在苏总的分享中，他给大家展示了他们是如何基于.NET Core进行开发，并且以Azure DevOps实现了CI/CD等最佳实践，让与会者都大呼过瘾。在结束环节，他用这套SDK开发出来的一个产品（微嗨嗨）进行现场互动，气氛燃到爆。
