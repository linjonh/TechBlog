    <h2>
     <strong>
      第一部分：颜色与纹理特征（Part 1）
     </strong>
    </h2>
    <h3>
     <strong>
      1. 颜色特征
     </strong>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        颜色直方图（Color Histogram）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          定义
         </strong>
         ：统计图像中各颜色通道（R/G/B）的像素分布，形成直方图。
        </p>
       </li>
       <li>
        <p>
         <strong>
          计算步骤
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           将每个颜色通道的像素值分为 256 个区间（0-255）。
          </p>
         </li>
         <li>
          <p>
           统计每个区间内的像素数量，生成三个通道的直方图。
          </p>
         </li>
         <li>
          <p>
           将三个直方图拼接成一个 768 维的向量（256×3）。
          </p>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          示例
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           若红色通道的像素集中在高值区域（如 200-255），说明图像偏红色调。
          </p>
         </li>
         <li>
          <p>
           绿色通道的直方图平坦，说明颜色分布均匀。
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          应用场景
         </strong>
         ：图像检索（按颜色搜索相似图片）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        颜色矩（Color Moments）
       </strong>
      </p>
     </li>
    </ol>
    <p>
     <img alt="" height="387" src="https://i-blog.csdnimg.cn/direct/9bbd9a93bc804a8c804f5b417b3a0acb.png" width="491"/>
    </p>
    <p>
    </p>
    <h3>
     <strong>
      2. 纹理特征
     </strong>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        Haralick 纹理特征
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          原理
         </strong>
         ：基于灰度共生矩阵（Gray-Level Co-occurrence Matrix, GLCM），统计相邻像素的灰度值对出现的频率。
        </p>
       </li>
       <li>
        <p>
         <strong>
          GLCM 构建步骤
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           选择距离 d 和方向 a（如水平、垂直、对角线）。
          </p>
         </li>
         <li>
          <p>
           统计所有满足条件的像素对 (i,j)，其中 i 和 j 分别为当前像素和邻域像素的灰度值。
          </p>
         </li>
         <li>
          <p>
           生成 L×L 的矩阵（L 为灰度级数）。
          </p>
         </li>
        </ol>
       </li>
      </ul>
     </li>
    </ol>
    <p class="img-center">
     <img alt="" height="302" src="https://i-blog.csdnimg.cn/direct/1ec99e7170254612827bbfaa129f3fc4.png" width="522"/>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        局部二值模式（Local Binary Pattern, LBP）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          原理
         </strong>
         ：将局部纹理编码为二进制模式，生成纹理描述子。
        </p>
       </li>
       <li>
        <p>
         <strong>
          计算步骤
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           对每个像素，比较其与周围 8 邻域像素的灰度值。
          </p>
         </li>
         <li>
          <p>
           若邻域像素值 ≥ 中心像素值，标记为 1；否则标记为 0。
          </p>
         </li>
         <li>
          <p>
           将 8 位二进制码转换为十进制（0-255），形成 LBP 值。
          </p>
         </li>
         <li>
          <p>
           统计图像中每个 LBP 值的频率，生成 256 维直方图。
          </p>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          改进版本
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           <strong>
            旋转不变 LBP
           </strong>
           ：通过循环移位二进制码，取最小值作为最终编码，减少旋转影响。
          </p>
         </li>
         <li>
          <p>
           <strong>
            多分辨率 LBP
           </strong>
           ：通过不同距离和邻域大小捕捉多尺度纹理。
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          应用
         </strong>
         ：人脸识别（对光照变化鲁棒）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        尺度不变特征变换（Scale-Invariant Feature Transform, SIFT）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          核心思想
         </strong>
         ：提取图像中的关键点，并生成对尺度、旋转、光照不变的描述子。
        </p>
       </li>
       <li>
        <p>
         <strong>
          四步流程
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           <strong>
            尺度空间极值检测
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             构建高斯金字塔（不同尺度的高斯模糊图像）。
            </p>
           </li>
           <li>
            <p>
             计算高斯差分（DoG）金字塔，检测局部极值点（潜在关键点）。
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           <strong>
            关键点定位
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             剔除低对比度点：通过泰勒展开拟合极值位置，计算精确对比度。
            </p>
           </li>
           <li>
            <p>
             剔除边缘响应点：利用 Hessian 矩阵的特征值比值（如主曲率比），过滤边缘点。
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           <strong>
            方向分配
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             计算关键点邻域内像素的梯度方向和幅值。
            </p>
           </li>
           <li>
            <p>
             构建方向直方图（36 个区间），取主方向作为关键点方向。
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           <strong>
            描述子生成
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             将关键点邻域划分为 4×4 的子区域。
            </p>
           </li>
           <li>
            <p>
             对每个子区域计算 8 方向的梯度直方图，形成 128 维向量（4×4×8）。
            </p>
           </li>
          </ul>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          应用
         </strong>
         ：图像拼接（全景图生成）、物体识别。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h2>
     <strong>
      第二部分：形状特征与高级编码（Part 2）
     </strong>
    </h2>
    <h3 style="background-color:transparent">
     <strong>
      1. 形状特征
     </strong>
    </h3>
    <p>
     <img alt="" height="215" src="https://i-blog.csdnimg.cn/direct/10812d2e60a048bc94a0ad84f8cb79d3.png" width="447"/>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        凸性（Convexity）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          凸包（Convex Hull）
         </strong>
         ：包含形状的最小凸集。
        </p>
       </li>
       <li>
        <p>
         <strong>
          凸缺陷（Convex Defect）
         </strong>
         ：凸包与原始形状的差异区域。
        </p>
       </li>
       <li>
        <p>
         凸性 = 原始面积凸包面积凸包面积原始面积​，值越接近 1 越凸。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        长宽比（Aspect Ratio）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         边界框的长宽比，用于区分细长物体（如铅笔）与紧凑物体（如球）。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      2. 形状上下文（Shape Context）
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        原理
       </strong>
       ：通过采样边界点，构建局部极坐标直方图描述形状。
      </p>
     </li>
     <li>
      <p>
       <strong>
        步骤
       </strong>
       ：
      </p>
      <ol>
       <li>
        <p>
         <strong>
          采样边界点
         </strong>
         ：使用 Canny 边缘检测提取形状轮廓点。
        </p>
       </li>
       <li>
        <p>
         <strong>
          构建形状上下文
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           对每个采样点，以该点为中心，将周围区域分为对数极坐标区间（如 5 距离区间 × 12 角度区间）。
          </p>
         </li>
         <li>
          <p>
           统计其他点落在各区间内的数量，形成 60 维直方图。
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          形状匹配
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           计算两个形状的匹配成本矩阵 C(pi,qj)C(pi​,qj​)。
          </p>
         </li>
         <li>
          <p>
           使用匈牙利算法寻找最优一对一匹配，最小化总成本。
          </p>
         </li>
        </ul>
       </li>
      </ol>
     </li>
     <li>
      <p>
       <strong>
        应用
       </strong>
       ：手写数字识别、形状检索。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      3.方向梯度直方图（Histogram of Oriented Gradients, HOG）
     </strong>
    </p>
    <p>
     <img alt="" height="239" src="https://i-blog.csdnimg.cn/direct/4a3b5ae165ed4f53a554e8b6f8919cf8.png" width="498"/>
    </p>
    <p>
     <strong>
      2.单元格划分
     </strong>
     ：
    </p>
    <p>
     将图像划分为 8×8 像素的单元格（Cell）。
    </p>
    <p>
     对每个单元格，统计 9 个方向（0°-180°，每 20°一区间）的梯度幅值直方图。
    </p>
    <p>
     <strong>
      3
     </strong>
     .
     <strong>
      块归一化
     </strong>
     ：
    </p>
    <p>
     将相邻的 2×2 单元格组合为块（Block）。
    </p>
    <p>
     对块内所有单元格直方图进行 L2 归一化，增强光照鲁棒性。
    </p>
    <p>
     <strong>
      4.特征向量生成
     </strong>
     ：
    </p>
    <p>
     将块特征拼接为最终向量。例如，64×128 图像生成 7×15×36 = 3,780 维特征。
    </p>
    <ul>
     <li>
      <p>
       <strong>
        应用
       </strong>
       ：行人检测（HOG + SVM）、车辆识别。
      </p>
     </li>
    </ul>
    <h3 style="background-color:transparent">
     <strong>
      2. 特征编码与词袋模型（Bag-of-Words, BoW）
     </strong>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        词袋模型（BoW）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          核心思想
         </strong>
         ：将局部特征（如 SIFT）聚类为“视觉单词”，生成全局直方图。
        </p>
       </li>
       <li>
        <p>
         <strong>
          步骤
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           <strong>
            构建视觉词典
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             提取所有训练图像的 SIFT 特征（假设每图提取 1,000 个 128 维描述子）。
            </p>
           </li>
           <li>
            <p>
             使用 k-means 聚类将所有描述子分为 kk 类（如 k=1,000），每个聚类中心即为一个视觉单词。
            </p>
           </li>
          </ul>
         </li>
         <li>
          <p>
           <strong>
            特征编码
           </strong>
           ：
          </p>
          <ul>
           <li>
            <p>
             对每张图像，将其 SIFT 描述子分配到最近的视觉单词。
            </p>
           </li>
           <li>
            <p>
             统计每个视觉单词的出现频率，生成 kk 维直方图。
            </p>
           </li>
          </ul>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          优点
         </strong>
         ：将变长局部特征转换为定长全局特征，适合分类任务。
        </p>
       </li>
       <li>
        <p>
         <strong>
          应用
         </strong>
         ：图像分类（如 Caltech-101 数据集）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        k-means 聚类
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          算法流程
         </strong>
         ：
        </p>
        <ol>
         <li>
          <p>
           <strong>
            初始化
           </strong>
           ：随机选择 k 个初始聚类中心。
          </p>
         </li>
         <li>
          <p>
           <strong>
            分配数据点
           </strong>
           ：将每个数据点分配到最近的聚类中心。
          </p>
         </li>
         <li>
          <p>
           <strong>
            更新中心
           </strong>
           ：计算每个簇的均值作为新中心。
          </p>
         </li>
         <li>
          <p>
           <strong>
            迭代
           </strong>
           ：重复步骤 2-3，直至中心不再变化或达到最大迭代次数。
          </p>
         </li>
        </ol>
       </li>
       <li>
        <p>
         <strong>
          关键问题
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           初始中心敏感：可能陷入局部最优，需多次随机初始化。
          </p>
         </li>
         <li>
          <p>
           计算复杂度：O(n⋅k⋅d⋅t)，其中 n 为数据量，d 为维度，t 为迭代次数。
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     <img alt="" height="848" src="https://i-blog.csdnimg.cn/direct/edf6daf6a0744287a6204d9b2ff3d065.png" width="935">

    <h2>
     第三部分：例题讲解
    </h2>
    <p>
     <img alt="" height="453" src="https://i-blog.csdnimg.cn/direct/94cbf15079e54862bb6810f339d02dc0.png" width="1735"/>
    </p>
    <blockquote>
     <p>
      <strong>
       题目：
      </strong>
      <br/>
      以下关于特征描述子的说法中，哪一个是
      <strong>
       错误
      </strong>
      的？
     </p>
     <p>
      A. Haralick 特征是从灰度共生矩阵（GLCM）中提取的。
      <br/>
      B. SIFT 通过计算多个尺度上的梯度直方图来实现旋转不变性。
      <br/>
      C. LBP 描述局部图像纹理，并且可以具有多分辨率和旋转不变性。
      <br/>
      D. 颜色矩比颜色直方图的表示能力更强。
     </p>
     <p>
      <strong>
       解析：
      </strong>
      <br/>
      该题考察的是
      <strong>
       不同特征描述子
      </strong>
      的基本概念，我们逐项分析：
     </p>
     <ul>
      <li>
       <strong>
        A 选项：正确
       </strong>
       。Haralick 特征的计算基于
       <strong>
        灰度共生矩阵（GLCM）
       </strong>
       ，用于描述纹理特征。
      </li>
      <li>
       <strong>
        B 选项：正确
       </strong>
       。SIFT 通过
       <strong>
        计算不同尺度上的梯度直方图
       </strong>
       ，确保旋转、尺度不变性。
      </li>
      <li>
       <strong>
        C 选项：正确
       </strong>
       。局部二值模式（LBP）可以扩展为
       <strong>
        旋转不变（Rotation Invariant, RI-LBP）
       </strong>
       和
       <strong>
        多分辨率 LBP（Multi-resolution LBP）
       </strong>
       ，所以该选项无误。
      </li>
      <li>
       <strong>
        D 选项：错误
       </strong>
       。颜色矩的特征向量仅由
       <strong>
        9 个元素（RGB 的均值、标准差、偏斜度）
       </strong>
       组成，而颜色直方图保留了
       <strong>
        更详细的颜色信息
       </strong>
       ，因此颜色直方图的表示能力更强。
      </li>
     </ul>
     <p>
      ✅
      <strong>
       答案：D
      </strong>
     </p>
