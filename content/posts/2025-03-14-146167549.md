    <p>
     直方图（Histogram）是数据可视化中最基础的图表之一，用于直观展示连续数据的分布特征。在MATLAB中，直方图不仅支持快速生成，还具备强大的数学运算和高度可定制的视觉效果，能够满足科研、工程和商业分析的多样化需求。
    </p>
    <h3>
     <strong>
      一、直方图的核心概念与数学原理
     </strong>
    </h3>
    <h4>
     1.
     <strong>
      直方图的结构与定义
     </strong>
    </h4>
    <p>
     直方图通过等宽区间（Bin）将数据划分为若干子集，统计每个区间内的观测频数（或频率），形成一系列相邻的矩形条。其核心参数包括：
    </p>
    <ul>
     <li>
      <strong>
       区间数（Number of Bins）
      </strong>
      ：决定直方图的分辨率和统计误差。
     </li>
     <li>
      <strong>
       区间宽度（Bin Width）
      </strong>
      ：与区间数互相关联，直接影响分布形态的表达。
     </li>
     <li>
      <strong>
       归一化方式
      </strong>
      ：可选择显示频数（Count）或概率密度（Probability）。
     </li>
    </ul>

     2.
     <strong>
      数学模型

    </h4>

      与相关图表的对比

    <h3>
     <strong>
      二、MATLAB基础直方图绘制
     </strong>
    </h3>
    <h4>
     1.
     <strong>
      使用
      <code>
       histogram
      </code>
      函数
     </strong>
    </h4>
    <h5>
     (1) 基础语法
    </h5>
    <pre><code>data = randn(1000,1);  % 生成正态分布数据
figure;
histogram(data);
title('基础直方图（默认参数）');
xlabel('数值');
ylabel('频数');
grid on;
</code></pre>
    <p>
     <img alt="" height="628" src="https://i-blog.csdnimg.cn/direct/dccc5fc1c6a742c0a49d51449a1db962.png" width="702">
     </img>
    </p>
    <h5>
     (2) 关键参数设置
    </h5>
    <pre><code>data = randn(1000,1);  % 生成正态分布数据
figure;
histogram(data);
title('基础直方图（默认参数）');
xlabel('数值');
ylabel('频数');
grid on;
% 自定义区间数与样式
histogram(data, ...
    'NumBins', 20, ...          % 设置区间数
    'FaceColor', [0.2 0.6 0.9], ...  % 填充颜色
    'EdgeColor', 'black', ...     % 边线颜色
    'FaceAlpha', 0.7, ...         % 透明度
    'BinWidth', 0.5 ...          % 指定区间宽度（与NumBins互斥）
);
</code></pre>
    <p>
     <img alt="" height="628" src="https://i-blog.csdnimg.cn/direct/bebf26bbfadd4db196993cac4caee141.png" width="702">
     </img>
    </p>
    <h4>
     2.
     <strong>
      归一化与概率密度
     </strong>
    </h4>
    <pre><code>data = randn(1000,1);  % 生成正态分布数据
figure;
histogram(data);
title('基础直方图（默认参数）');
xlabel('数值');
ylabel('频数');
grid on;
% 自定义区间数与样式
histogram(data, ...
    'NumBins', 20, ...          % 设置区间数
    'FaceColor', [0.2 0.6 0.9], ...  % 填充颜色
    'EdgeColor', 'black', ...     % 边线颜色
    'FaceAlpha', 0.7, ...         % 透明度
    'BinWidth', 0.5 ...          % 指定区间宽度（与NumBins互斥）
);
% 转换为概率密度（总面积=1）
histogram(data, 'Normalization', 'pdf');
hold on;

% 叠加理论正态分布曲线
x = linspace(min(data), max(data), 100);
mu = mean(data);
sigma = std(data);
pdf_normal = normpdf(x, mu, sigma);
plot(x, pdf_normal, 'r-', 'LineWidth', 2);
legend('直方图', '理论正态分布');
</code></pre>

      三、实战案例：多维数据与高级应用

    <h4>
     1.
     <strong>
      多维数据分布对比

    <h5>
     (1) 并排直方图
    </h5>
    <pre><code>% 生成三组不同分布的数据
data1 = randn(1000,1);
data2 = rand(1000,1)*6 - 3;
data3 = trnd(5,1000,1);  % t分布（自由度为5）

figure;
subplot(1,3,1);
histogram(data1, 'FaceColor', 'blue');
title('正态分布');
xlim([-5 5]);

subplot(1,3,2);
histogram(data2, 'FaceColor', 'red');
title('均匀分布');
xlim([-5 5]);

subplot(1,3,3);
histogram(data3, 'FaceColor', 'green');
title('t分布');
xlim([-5 5]);
</code></pre>

     (2) 重叠直方图

hold on;
histogram(data1, 'BinWidth', 0.5, 'FaceAlpha', 0.5, 'FaceColor', 'b');
histogram(data2, 'BinWidth', 0.5, 'FaceAlpha', 0.5, 'FaceColor', 'r');
histogram(data3, 'BinWidth', 0.5, 'FaceAlpha', 0.5, 'FaceColor', 'g');
xlabel('数值');
ylabel('频数');
legend('正态分布', '均匀分布', 't分布');

    <h3>
     <strong>
      四、进阶技巧与性能优化
     </strong>
    </h3>
    <h4>
     1.
     <strong>
      自动最佳区间数选择
     </strong>
    </h4>
    <p>
     MATLAB内置多种区间数计算规则，可通过
     <code>
      'BinMethod'
     </code>
     参数调用：
    </p>
    <pre><code>% 使用不同算法
methods = {'auto', 'sqrt', 'sturges', 'scott', 'fd'};
figure;
for i = 1:5
    subplot(2,3,i);
    histogram(data, 'BinMethod', methods{i});
    title(['BinMethod = ', methods{i}]);
end
</code></pre>
    <p>
     <img alt="" height="771" src="https://i-blog.csdnimg.cn/direct/32c737b0824748ec96b249799bcc3915.png" width="901"/>

    <h4>
     2.
     <strong>
      二维直方图（Histogram2）
     </strong>
    </h4>
    <pre><code>% 生成二维正态分布数据
x = randn(10000,1);
y = x*0.5 + randn(10000,1)*0.5;

% 绘制二维直方图
figure;
histogram2(x, y, ...
    'BinWidth', [0.3 0.3], ...
    'FaceColor', 'flat', ...  % 根据高度染色
    'ShowEmptyBins', 'on', ...
    'EdgeColor', 'none');
colormap hot;
colorbar;
xlabel('X');
ylabel('Y');
title('二维直方图（热力图风格）');
</code></pre>
