    <p>
     下面我将指导您如何安装 Django 库以及基本的使用方法。Django 是一个高级的 Python Web 框架，它鼓励快速开发和干净、实用的设计。以下是详细的步骤：
    </p>
    <h4>
     1. 安装 Django
    </h4>
    <p>
     首先，确保您的系统上已经安装了 Python 和 pip（Python 的包管理工具）。然后，您可以使用 pip 来安装 Django。
    </p>
    <pre>pip install django</pre>
    <p>
     如果您想安装特定版本的 Django，可以指定版本号，例如：
    </p>
    <pre>pip install django==3.2.15</pre>
    <h4>
     2. 创建 Django 项目
    </h4>
    <p>
     安装完成后，您可以创建一个新的 Django 项目。打开命令行工具并执行以下命令：
    </p>
    <pre>django-admin startproject myproject</pre>
    <p>
     这会在当前目录下创建一个名为
     <code>
      myproject
     </code>
     的文件夹，其中包含一些默认的配置文件。
    </p>
    <h4>
     3. 运行开发服务器
    </h4>
    <p>
     进入项目目录并启动 Django 开发服务器：
    </p>
    <pre>cd myproject
python manage.py runserver</pre>
    <p>
     默认情况下，服务器将在
     <code>
      http://127.0.0.1:8000/
     </code>
     上运行。打开浏览器访问该地址，您应该能看到 "It worked!" 的欢迎页面。
    </p>
    <h4>
     4. 创建应用
    </h4>
    <p>
     在 Django 中，项目由多个应用组成。每个应用负责处理特定的功能。您可以使用以下命令来创建一个新的应用：
    </p>
    <pre>python manage.py startapp myapp</pre>
    <p>
     这会创建一个名为
     <code>
      myapp
     </code>
     的新文件夹，并生成一些初始文件。
    </p>
    <h4>
     5. 配置应用
    </h4>
    <p>
     为了让 Django 知道您的新应用，请编辑
     <code>
      myproject/settings.py
     </code>
     文件，在
     <code>
      INSTALLED_APPS
     </code>
     列表中添加
     <code>
      'myapp'
     </code>
     。
    </p>
    <pre>INSTALLED_APPS = [
    ...
    'myapp',
]
</pre>
    <h4>
     6. 创建视图
    </h4>
    <p>
     在
     <code>
      myapp/views.py
     </code>
     文件中定义一个简单的视图函数：
    </p>
    <pre>from django.http import HttpResponse


def hello_world(request):
    return HttpResponse("Hello, world!")
</pre>
    <h4>
     7. 配置 URL
    </h4>
    <p>
     接下来，我们需要为这个视图配置 URL。首先，在
     <code>
      myapp
     </code>
     目录下创建一个
     <code>
      urls.py
     </code>
     文件，并添加如下内容：
    </p>
    <pre>from django.urls import path
from . import views

urlpatterns = [
    path('hello/', views.hello_world, name='hello_world'),
]
</pre>
    <p>
     然后，在
     <code>
      myproject/urls.py
     </code>
     文件中包含
     <code>
      myapp
     </code>
     的 URL 配置：
    </p>
    <pre>from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]
</pre>
    <h4>
     8. 访问视图
    </h4>
    <p>
     现在，您可以在浏览器中访问 `
     <a href="http://127.0.0.1:8000/hello/%60%EF%BC%8C%E5%BA%94%E8%AF%A5%E8%83%BD%E7%9C%8B%E5%88%B0" rel="nofollow" title="http://127.0.0.1:8000/hello/`，应该能看到 ">
      http://127.0.0.1:8000/hello/`，应该能看到
     </a>
     "Hello, world!" 的消息。
    </p>
    <h4>
     9. 数据库迁移
    </h4>
    <p>
     如果您的应用需要使用数据库，可以创建模型并在
     <code>
      models.py
     </code>
     文件中定义它们。之后，运行以下命令来创建和应用数据库迁移：
    </p>
    <pre>python manage.py makemigrations
python manage.py migrate</pre>
    <h4>
     10. 创建超级用户
    </h4>
    <p>
     为了能够访问 Django 的管理界面，您需要创建一个超级用户：
    </p>
    <pre>python manage.py createsuperuser</pre>
    <p>
     按照提示输入用户名、邮箱和密码。完成后，您可以访问
     <code>
      http://127.0.0.1:8000/admin/
     </code>
     并使用刚刚创建的账户登录。
    </p>
    <p>
     以上就是 Django 的基本安装与使用流程。希望这对您有所帮助！如果有更多问题或需要进一步的帮助，请随时告诉我。
    </p>
