    <blockquote>
     <p>
      本文档介绍在Widows端如何使用navicat或DataStudio应用远程连接openGauss，进行可视化的数据库操作。
     </p>
    </blockquote>
    <h4>
     <a id="_1">
     </a>
     远程连接工具
    </h4>
    <p>
     可使用
     <code>
      navicat
     </code>
     或
     <code>
      DataStudio
     </code>
     远程连接openGauss。
    </p>
    <h5>
     <a id="1__5">
     </a>
     1 连接前准备
    </h5>
    <p>
     1、连接工具所在机器ip要添加到openGauss数据库的远程访问白名单中，可通过gs_guc工具添加，步骤如下：
    </p>
    <ol>
     <li>
      <p>
       使用数据库管理员用户（默认为omm）登录任意服务器节点
      </p>
     </li>
     <li>
      <p>
       执行以下命令
      </p>
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token comment"># 使用reload可直接生效，无需重启数据库</span>
gs_guc reload -N all -I all -c <span class="token string">"listen_addresses = '*'"</span> 

<span class="token comment"># 将xx.xx.xx.xx设置为你机器的IP</span>
gs_guc reload -N all -I all -h <span class="token string">"host all all xx.xx.xx.xx/32 sha256"</span> 
</code></pre>
    <p>
     2、默认创建的数据库用户omm为管理员，无法进行远程登录，需要创建其他用户进行远程登录。
    </p>
    <p>
     需要注意的是，如果使用navicat进行连接，在创建用户前需先修改openGauss的密码加密规则，使其兼容PostgresSQL，而使用DataStudio进行连接则无需该步骤，具体见以下教程。
    </p>
    <h5>
     <a id="2_navicat_25">
     </a>
     2 使用navicat连接
    </h5>
    <p>
     创建用于远程连接的openGauss数据库用户
    </p>
    <blockquote>
     <p>
      <strong>
       注：
      </strong>
      本质上，navicat是将openGauss识别为PostgreSQL数据库进行的连接，所以为了使openGauss兼容PostgreSQL，需要先修改加密规则，再创建远程连接使用的用户
     </p>
    </blockquote>
    <p>
     使用omm管理员用户登录数据库主节点，修改密码加密规则后再创建远程连接的用户
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 修改密码加密规则，兼容PostgreSQL的MD5规则，-D 后衔接数据库的数据目录，请自行替换</span>
gs_guc reload -D /opt/huawei/install/data/dn -c <span class="token string">"password_encryption_type = 1"</span>

<span class="token comment"># 登录openGauss</span>
gsql -d postgres
<span class="token comment"># 创建用于远程连接的用户并设置密码</span>
CREATE <span class="token environment constant">USER</span> user_pgloader WITH PASSWORD <span class="token string">'gauss_123'</span><span class="token punctuation">;</span> 
</code></pre>
    <p>
     创建完成后即可在navicat使用该用户进行远程登录，navicat连接的数据库类型选择：PostgreSQL
    </p>
    <h5>
     <a id="3_DataStudio_44">
     </a>
     3 使用DataStudio连接
    </h5>
    <p>
     DataStudio是openGauss官方开发的工具，使用DataStudio进行远程连接，可直接创建远程连接的数据库用户，无需先修改加密规则。
    </p>
    <pre><code class="prism language-sql"><span class="token comment">-- 创建用于远程连接的用户并设置密码</span>
<span class="token keyword">CREATE</span> <span class="token keyword">USER</span> user_pgloader <span class="token keyword">WITH</span> PASSWORD <span class="token string">'gauss_123'</span><span class="token punctuation">;</span> 
</code></pre>
    <p>
     DataStudio下载地址：https://opengauss.org/zh/download.html
    </p>
    <img alt="image-20220617155716932" src="https://cdn.jsdelivr.net/gh/FireLightGuo/picgo_img@main/img/image-20220617155716932.png">
     <p>
      DataStudio工具：
     </p>
     <ul>
      <li>
       要求安装jdk11及以上版本
      </li>
      <li>
       以管理员模式启动
      </li>
     </ul>
