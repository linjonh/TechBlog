# 前端学习之HTML 下拉框 文本框

注：注释是对下列代码中标签的解释

## 下拉框

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>下拉框</title>
</head>
<body>
    <!--
        使用：
        select标签
        option标签
        就可以了
        select默认选项
        multiple:这个属性可以将下拉框带上下拉的那个东西
     -->
    <select name="books" multiple>
        <option value="">请选择你喜欢看的书：</option>
        <option value="books1 selected">西游记</option>
        <option value="books2">水浒传</option>
        <option value="books3">三国演义</option>
        <option value="books4">红楼梦</option>

    </select>
</body>
</html>
```

## 文本域

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>文本域</title>
</head>
<body>
    <label >个人介绍：<br></label>
    <textarea name="introduce" id="" cols="30" rows="10">请使用100个字赞美我</textarea>
</body>
</html>
```
