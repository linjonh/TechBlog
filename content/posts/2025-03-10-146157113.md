    <h4>
     题目：
    </h4>
    <blockquote>
     <p>
      给定一个数组
      <code>
       nums
      </code>
      ，编写一个函数将所有
      <code>
       0
      </code>
      移动到数组的末尾，同时保持非零元素的相对顺序。
     </p>
     <p>
      请注意 ，必须在不复制数组的情况下原地对数组进行操作。
     </p>
    </blockquote>
    <h4>
     示例 1:
    </h4>
    <blockquote>
     <p>
      输入: nums =[0,1,0,3,12]
     </p>
     <p>
      输出:[1,3,12,0,0]
     </p>
    </blockquote>
    <h4>
     示例 2:
    </h4>
    <blockquote>
     <p>
      输入: nums =[0]
     </p>
     <p>
      输出:[0]
     </p>
    </blockquote>
    <h4>
     提示:
    </h4>
    <blockquote>
     <p>
      <span style="color:#0d0016">
       1 &lt;= nums.length &lt;= 104
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       <code>
        -231 &lt;= nums[i] &lt;= 231 - 1
       </code>
      </span>
     </p>
    </blockquote>
    <h4>
     代码：
    </h4>
    <pre><code class="language-java">class Solution {
    public void moveZeroes(int[] nums) {
        int right = 0;
        int n = nums.length;
        for (int left = 0;left&lt;nums.length;left++){
            if(nums[left]!=0){
                nums[right]=nums[left];
                right++;
            }
        }
        for(int i = right;i&lt;n;i++){
            nums[i]=0;
        }
    }
}</code></pre>
    <h4>
     解释：
    </h4>
    <p>
     通过双指针先将不是0的进行赋值，然后剩余位置补0即可。
    </p>
