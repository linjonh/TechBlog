   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      1题目描述
     </strong>
     <br/>
     具体题目描述如下：
    </p>
    <p>
     Linux 操作系统有多个发行版，distrowatch.com 提供了各个发行版的资料。这些发行版互相存在关联，例如 Ubuntu 基于 Debian 开发，而 Mint 又基于 Ubuntu 开发，那么我们认为 Mint 同 Debian 也存在关联。
     <br/>
     发行版集是一个或多个相关存在关联的操作系统发行版，集合内不包含没有关联的发行版。
     <br/>
     给你一个 n x n的矩阵 isConnected，其中 isConnected[i][j] = 1 表示第 i 个发行版和第 j 个发行版直接关联，而 isConnectedlI = 0 表示二者不直接相连。
     <br/>
     返回最大的发行版集中发行版的数量。
    </p>
    <p>
     <strong>
      2. 输入描述
     </strong>
     <br/>
     第一行输入发行版的总数量N，之后每行表示各发行版间是否直接相关。
    </p>
    <p>
     <strong>
      3. 输出描述
     </strong>
     <br/>
     输出最大的发行版集中发行版的数量。
    </p>
    <p>
     说明
     <br/>
     1 &lt;= N &lt;= 200
    </p>
    <p>
     <strong>
      4. 示例演示
     </strong>
     <br/>
     4.1 示例1
     <br/>
     输入：
    </p>
    <pre><code class="hljs">4
1 1 0 0
1 1 1 0
0 1 1 0
0 0 0 1</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="hljs">3</code></pre>
    <p>
     <strong>
      示例说明：例一表明有四个发行版，横轴i与竖轴j分别对应这四个版本。对角线全为1，因为都属于自己相关自己，所以全为1，我们可以只按行或者列来判断即可。
     </strong>
    </p>
    <p>
     <strong>
      Java代码
     </strong>
    </p>
    <pre><code class="language-java">package odTest;

import java.util.Arrays;
import java.util.Scanner;

public class LinuxLunch {
	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		
		int num = Integer.parseInt(scanner.nextLine());
		int maxCount = 0;
		for(int i = 0;i&lt;num;i++) {
			int[] input = Arrays.stream(scanner.nextLine().split(" ")).mapToInt(Integer::parseInt).toArray();
			int count = 0;
			for(int j=0;j&lt;num;j++) {
			   if(input[j] == 1) {
				   count++;
			   }
			}
			if(count&gt;maxCount) {
				maxCount = count;
			}
		}
		System.out.println(maxCount);
	}

}
</code></pre>
