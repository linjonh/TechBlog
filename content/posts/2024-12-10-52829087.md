    <p>
     <strong>
      一面是技术面
     </strong>
     ，面了大约40多分钟，有两个面试官。
     <br/>
     1、自我介绍。
    </p>
    <p>
     2、项目介绍：比较详细的问了些问题
    </p>
    <p>
     3、会C语言吗？来看段程序。功能是实现字符串的反转的代码，要我写测试程序及测试案例，当场在电脑上编写与调试。
    </p>
    <p>
     4、网络问题：
     <br/>
     1）子网掩码是什么，作用；
     <br/>
     2）给了一道题目，已知IP地址172.168.160.25，求子网掩码和最大主机数；
     <br/>
     3）对TCP/IP协议的了解；
     <br/>
     4）ping命令是基于什么协议的，属于哪一层？
     <br/>
     5）长连接与短连接的区别
    </p>
    <p>
     5、闲聊：
     <br/>
     1）手上有offer吗？
     <br/>
     2）对于这次的中兴招聘，你和同学有什么建议吗？
     <br/>
     3）与你同学相比，你评价一下自己的学习能力。
     <br/>
     4）确定要留南京吗？
    </p>
    <p>
     6、有什么问题要问我们吗？
     <br/>
     ​
    </p>
    <hr/>
    <p>
     <strong>
      二面是综合面试
     </strong>
     ，大约20分钟，两个面试官。
     <br/>
     1、英语测试：
     <br/>
     1）英文自我介绍
     <br/>
     2）英文介绍本科大学所在的城市
    </p>
    <p>
     2、闲聊：
     <br/>
     1）兴趣爱好
     <br/>
     2）对加班的看法
     <br/>
     3）情景模拟：假设不是你的错，但是领导却公开批评了你，你会怎么办？
     <br/>
     4）你对重复做一件事情有什么看法？
     <br/>
     5）能过来参加实习吗？
     <br/>
     6）社会实践活动（简历上写的做青协志愿者，详细询问了活动的内容）
     <br/>
     7）家庭及感情问题
     <br/>
     8）对于软件测试这一岗位，你觉得该具备什么样的能力？
     <br/>
     9）期望的薪资
    </p>
