### 十四届省赛客观题

![](https://i-blog.csdnimg.cn/direct/b76f7773b4e94d88916d8859f1b1c008.png)

解析部分

第一题

时序逻辑电路的定义：在特定的时钟脉冲信号的控制下进行操作的电路，在提供的选项中，计数器、分频器、触

发器属于时序逻辑电路

详细说明：

|  |  |
| --- | --- |
| 计数器（Counter） | 计数器是一种能够在时钟信号的作用下进行计数的电路，可以用来记录输入信号的脉冲数量或者生成特定的数字序列 |
| 分频器（Divider） | 分频器是一种将输入信号的频率分频成较低频率输出的电路，它可以用来降低时钟频率，以匹配外部设备的要求 |
| 触发器（Flip-Flop） | 触发器是一种存储器件，用于在时钟信号的作用下存储输入信号的状态。它们在数字电路中用于存储数据或实现状态机等功能 |
| 编码器（Encounter） | 虽然也是一种数字电路，但通常更多地用于将一种形式的数据转换成另一种形式，而不是存储或处理时序信息。因此，编码器通常不被认为是时序罗hi电路的一部分 |

第二题

关于其他进制转十进制：按权展开

十进制转其他进制：连除法，最先得到的放右边

减法计数器的计数值到0时，会产生一个重装载值，此处重载后就会变成1111 1111(256)。

至于题中假设再经过300个脉冲，那么就会再产生两次重装载，即一次完整的计数周期 加 不完整的周期。因此，可得 256*2-300 = 212，再将其转换成2进制数得到1101 010

第三题

概念题，背答案即可：晶体管的穿透电流Iced能够体现晶体管的温度稳定性

第四题

STM32系列微控制器的程序可以在ROM、RAM上运行，但不可以在EEPROM和寄存器上运行

|  |  |
| --- | --- |
| 寄存器： | 微控制器内部的存储器，用于存储控制寄存器、状态寄存器等重要信息，控制芯片的各种功能。程序通过访问寄存器来控制芯片的各种外设，实现各种功能 |
| EEPROM | 可擦写可编程只读存储器，一般用于存储配置信息、校准数据、唯一序列号等。程序不会在E2PROM上运行，而是通过读写E2PROM的API来读写其中的数据 |

LSB解释：全称Least Significant Bit，是二进制数字中最低有效位，也称为最低位

LSB所在的位对整个数的大小影响最小，通常在进行数字表示和处理时，LSB处的变换引起的数值变化最小

所以在这个题目中DAC为八位，参考电压2.4V，即1LSB输出电压增量是2.4/256
