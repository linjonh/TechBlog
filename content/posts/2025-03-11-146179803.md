    <h2>
     <a id="nginx_2">
     </a>
     nginx的安装【无网络】
    </h2>
    <ol>
     <li>
      准备安装包：在有网络的机器上，从
      <a href="https://nginx.org/en/download.html" rel="nofollow">
       Nginx 官方网站
      </a>
      下载 Nginx 的稳定版本源码包，如nginx-1.26.2.tar.gz。
     </li>
     <li>
      根据系统和 Nginx 的依赖需求，下载 PCRE、zlib、openssl 等相关依赖的源码包，并上传到无网络的 Linux 服务器上。
     </li>
     <li>
      安装依赖
      <br/>
      安装 PCRE：进入 PCRE 解压后的目录，
      <br/>
      执行./configure配置安装路径等选项，
     </li>
    </ol>
    <pre><code>./configure  --prefix=/usr/local/pcre
</code></pre>
    <p>
     然后执行make和make install进行编译和安装。
    </p>
    <pre><code>make &amp;&amp; make install
</code></pre>
    <p>
     安装 zlib：进入 zlib 解压后的目录，
     <br/>
     执行./configure，
    </p>
    <pre><code>./configure  --prefix=/usr/local/zlib
</code></pre>
    <p>
     接着执行make和make install。
    </p>
    <pre><code>make &amp;&amp; make install
</code></pre>
    <p>
     安装 openssl：进入 openssl 解压后的目录，
    </p>
    <p>
     执行./config配置，
    </p>
    <pre><code>./configure  --prefix=/usr/local/openssl
</code></pre>
    <p>
     再执行make和make install。
    </p>
    <pre><code>make &amp;&amp; make install
</code></pre>
    <ol start="4">
     <li>
      安装 Nginx
      <br/>
      解压 Nginx 源码：在无网络的 Linux 服务器上，使用命令解压 Nginx 源码包。
     </li>
    </ol>
    <pre><code>tar -zxvf nginx-1.26.2.tar.gz
</code></pre>
    <p>
     配置 Nginx：进入解压后的 Nginx 目录，执行./configure命令，
    </p>
    <pre><code>./configure --prefix=/usr/local/nginx --with-http_ssl_module --with-http_gzip_static_module --with-http_sub_module
</code></pre>
    <p>
     并通过参数指定 PCRE、zlib、openssl 等依赖库的路径。
     <br/>
     编译安装：执行make命令进行编译，编译完成后执行make install将 Nginx 安装到指定目录。
    </p>
    <pre><code>make &amp;&amp; make install
</code></pre>
    <ol start="5">
     <li>
      运行nginx
      <br/>
      进入 /usr/local/nginx/sbin/ 目录
     </li>
    </ol>
    <pre><code> ./nginx 
</code></pre>
    <h2>
     <a id="nginxsbin_55">
     </a>
     nginx常用命令【在sbin目录中使用命令】
    </h2>
    <p>
     查看版本：
     <br/>
     <code>
      nginx -v
     </code>
     <br/>
     查看nginx安装位置
     <br/>
     <code>
      which nginx
     </code>
     <br/>
     nginx配置文件位置：
     <br/>
     <code>
      whereis nginx
     </code>
     <br/>
     查看加载nginx配置文件有无错误
     <br/>
     <code>
      nginx -t
     </code>
     <br/>
     重新加载nginx配置文件
     <br/>
     <code>
      nginx -s reload
     </code>
     <br/>
     停止nginx服务
     <br/>
     <code>
      nginx -s quit
     </code>
    </p>
