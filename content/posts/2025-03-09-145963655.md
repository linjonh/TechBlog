    Dim API As String
    Dim SendTxt As String
    Dim Http As Object
    Dim status_code As Integer
    Dim response As String

    API = "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions"
    SendTxt = "{""model"": ""qwen-max-0125"", ""messages"": [{""role"":""system"", ""content"":""请根据用户的问题给出详细的解答。""}, {""role"":""user"", ""content"":""" &amp; inputText &amp; """}]}"
    
    Set Http = CreateObject("MSXML2.XMLHTTP")
    With Http
        .Open "POST", API, False
        .setRequestHeader "Content-Type", "application/json"
        .setRequestHeader "Authorization", "Bearer " &amp; api_key
        .send SendTxt
        status_code = .Status
        response = .responseText
    End With
    
    ' 弹出窗口显示 API 响应（调试用）
    ' MsgBox "API Response: " &amp; response, vbInformation, "Debug Info"
    
    If status_code = 200 Then
        CallDeepSeekAPI = response
    Else
        CallDeepSeekAPI = "Error: " &amp; status_code &amp; " - " &amp; response
    End If
    
    Set Http = Nothing
End Function

    <h3 id="4.1%C2%A0OfficeAI%C2%A0%E7%AE%80%E4%BB%8B" name="4.1%C2%A0OfficeAI%C2%A0%E7%AE%80%E4%BB%8B">
     5.1
     <span style="color:#19191c">
      <span style="background-color:#ffffff">
       OfficeAI
      </span>
     </span>
     简介
    </h3>
    <p>
     OfficeAI 助手 是一款免费的智能AI办公工具软件，专为 Microsoft Office 和 WPS 用户打造。 无论你是在寻找如何输入“打勾（√）符号”的方法，还是想知道“怎么在插入表格前添加文字”，或者“该用哪个公式”， AI办公助手都能为你提供快速、准确的解决方案。 通过简单的指令，ExcelAI 插件可以帮你自动完成复杂的公式计算、函数选择。 WordAI 插件还具备整理周报、撰写会议纪要、总结内容、以及文案润色的强大功能。 总之，OfficeAI 助手将大大提升你的办公效率，让日常工作变得更加轻松便捷。
    </p>
    <p>
     <img alt="" height="749" src="https://i-blog.csdnimg.cn/direct/746e2bb6e4b04e2d80cef87bec21e67d.png" width="1569"/>

     下载到本地之后，傻瓜式安装即可

    <h3 id="4.4%20%E9%85%8D%E7%BD%AE%E5%A4%A7%E6%A8%A1%E5%9E%8B" name="4.4%20%E9%85%8D%E7%BD%AE%E5%A4%A7%E6%A8%A1%E5%9E%8B" style="background-color:transparent">
     5.4 配置大模型
    </h3>
    <p>
     安装完上面的Office AI并重启word之后，再次进入时，在右侧就出现了一个对话框，简单来说，以后想要使用这个AI助手来辅助操作word文档，就可以在对话框中进行问题的输入即可
    </p>
    <p>
     <img alt="" height="816" src="https://i-blog.csdnimg.cn/direct/1cada8f6835145208c13c1a51106ab15.png" width="1881"/>
    </p>
    <p>
     在【OfficeAI助手】插件选项卡下点击【右侧面板】，在面板中点击【设置】，在【大模型设置】选项卡内勾选打开【本地模型/KPI-KEY】,最后进入【KPIKEY标签】。
    </p>
    <p>
    </p>
    <p>
     默认进来之后，内置的是豆包模型
    </p>
    <p>
     <img alt="" height="571" src="https://i-blog.csdnimg.cn/direct/9a1d8af5db854a568f3a51719518232d.png" width="656"/>
    </p>
    <p>
     可以切换到APiKey位置，打开之后，可以看到下拉框中支持很多种类型的大模型，比如在这里我选择硅基流动对接的DeepSeek模型
    </p>
    <p>
     <img alt="" height="532" src="https://i-blog.csdnimg.cn/direct/b7bce3f739df4ff08042dcaebaf3dfe1.png" width="621"/>
    </p>
    <p>
    </p>
    <p>
     选择之后，将你在硅基流动平台上的ApiKey的信息粘贴到此处即可，硅基流动地址：
     <a href="https://cloud.siliconflow.cn/models" rel="nofollow" title="Models">
      Models
     </a>
    </p>
    <p>
     <img alt="" height="515" src="https://i-blog.csdnimg.cn/direct/d1f8f8804e184da89439800625b88479.png" width="625"/>
    </p>
    <p>
     保存成功后关掉窗口
    </p>
    <p>
     <img alt="" height="512" src="https://i-blog.csdnimg.cn/direct/1a754b5aae4a427a96e8486c6572c026.png" width="625"/>
    </p>

     <p>
      如果要设置其他的大模型，也是上述类似的方式操作即可
     </p>
    </blockquote>
