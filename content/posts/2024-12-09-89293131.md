    <h3>
     <a id="Java__0">
     </a>
     Java 获取两个日期之间的所有日期
    </h3>
    <p>
     java 8 引入新的Time api使得无需引入第三方包处理日期和时间成为可能。这篇短文看下如何获取两个日期之间的所有日期，通过不同方式实现对比其差异。
    </p>
    <h3>
     <a id="Java_7_api_4">
     </a>
     Java 7 api
    </h3>
    <p>
     在java 7 之前，使用Calendar实例可以计算，循环给迭代中的每个日期加1，使用add方法，直到Calendar.Date属性到达结束日期。
    </p>
    <p>
     下面请看Calendar实例的代码：
    </p>
    <pre><code>public static List&lt;Date&gt; getDatesBetweenUsingJava7(Date startDate, Date endDate) {
    List&lt;Date&gt; datesInRange = new ArrayList&lt;&gt;();
    Calendar calendar = new GregorianCalendar();
    calendar.setTime(startDate);
     
    Calendar endCalendar = new GregorianCalendar();
    endCalendar.setTime(endDate);
 
    while (calendar.before(endCalendar)) {
        Date result = calendar.getTime();
        datesInRange.add(result);
        calendar.add(Calendar.DATE, 1);
    }
    return datesInRange;
}
</code></pre>
    <h3>
     <a id="java_8_api_28">
     </a>
     java 8 api
    </h3>
    <p>
     在java8 中，我们能创建连续无限日期流，然后返回相关的部分。因为需要终止无限流，所以需要先计算连个日期之间的天数，然后通过limit方法终止流：
    </p>
    <pre><code>public static List&lt;LocalDate&gt; getDatesBetweenUsingJava8(LocalDate startDate, LocalDate endDate) { 
    long numOfDaysBetween = ChronoUnit.DAYS.between(startDate, endDate); 
    return IntStream.iterate(0, i -&gt; i + 1)
      .limit(numOfDaysBetween)
      .mapToObj(i -&gt; startDate.plusDays(i))
      .collect(Collectors.toList()); 
}
</code></pre>
    <p>
     首先ChronoUnit.DAYS.between方法获取连个日期之间的差。然后我们创建IntStream表示从开始日期天数，下一步通过plusDays方法转换整数位LocalDate对象，最终转成list。
    </p>
    <h3>
     <a id="_java_9_api_43">
     </a>
     使用 java 9 api
    </h3>
    <p>
     最后使用java 9引入的专门计算方法：
    </p>
    <pre><code>public static List&lt;LocalDate&gt; getDatesBetweenUsingJava9(
  LocalDate startDate, LocalDate endDate) {
  
    return startDate.datesUntil(endDate)
      .collect(Collectors.toList());
}
</code></pre>
