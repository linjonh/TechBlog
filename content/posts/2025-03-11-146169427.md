    <h2>
     <a id="SSHgithub_2">
     </a>
     配置本地SSH，添加到github
    </h2>
    <h3>
     <a id="SSH_3">
     </a>
     首先查看本地是否有SSH密钥
    </h3>
    <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-al</span> ~/.ssh
</code></pre>
    <p>
     可以看到一开始我是没有的，只有一个known_hosts
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d2cdf82f5f3a4977926e27718ffa9293.png"/>
    </p>
    <h3>
     <a id="SSH_10">
     </a>
     生成SSH密钥，和邮箱绑定
    </h3>
    <pre><code class="prism language-bash">ssh-keygen <span class="token parameter variable">-t</span> ed25519 <span class="token parameter variable">-C</span> <span class="token string">"你的邮箱@example.com"</span>
</code></pre>
    <p>
     建议：
     <mark>
      如果懒得话，接下来都可以回车
     </mark>
     <br/>
     <em>
      首先输入要存储到的文件名
     </em>
     <br/>
     <em>
      设置密码
     </em>
     <br/>
     <em>
      重复密码
     </em>
     <br/>
     <mark>
      设置成功！
     </mark>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d10f8f503f4c453b983b57f86523d918.png">
      <br/>
      可以在本地用户路径看到.ssh文件夹
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1ff0ce695b9e4093b5c8dd485ebc5c1b.png"/>
     </img>
    </p>
    <h3>
     <a id="_SSH__sshagent_23">
     </a>
     将 SSH 密钥添加到 ssh-agent：
    </h3>
    <p>
     <em>
      启动ssh-agent
     </em>
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">eval</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span>ssh-agent <span class="token parameter variable">-s</span><span class="token variable">)</span></span>"</span>
</code></pre>
    <p>
     <em>
      添加私钥
     </em>
    </p>
    <pre><code class="prism language-bash">ssh-add ~/.ssh/id_ed25519
</code></pre>
    <h3>
     <a id="_32">
     </a>
     显示本地公钥
    </h3>
    <pre><code class="prism language-bash"><span class="token function">cat</span> ~/.ssh/id_ed25519.pub
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4dc7eaa795a141f28bb06c1936cac10b.png"/>
    </p>
    <h4>
     <a id="github_38">
     </a>
     <em>
      把下面这一串生成的公钥存到github上
     </em>
    </h4>
    <p>
     <em>
      选择github setting （用户设置）
     </em>
     <br/>
     如图所示选择新建SSH密钥
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/de12d12a10b44ae5b2ac887249f0c7fa.png">
      <br/>
      <mark>
       这里title建议写My_PC
      </mark>
      <br/>
      <mark>
       key写刚刚复制的公钥
      </mark>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/100fc413641e4cf7aa03456029f3fb7f.png"/>
     </img>
    </p>
    <p>
     点击添加即可
    </p>
    <h2>
     <a id="SSH_48">
     </a>
     验证SSH配置是否成功
    </h2>
    <h3>
     <a id="_49">
     </a>
     终端跳转到本地仓库
    </h3>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> yzj-
</code></pre>
    <h3>
     <a id="httpSSH_53">
     </a>
     把http协议改为SSH（如果之前是的话）
    </h3>
    <pre><code class="prism language-bash"><span class="token function">git</span> remote set-url origin git@github.com:yzjer/yzj-.git
<span class="token comment"># 示例：</span>
<span class="token comment"># git remote set-url origin git@github.com:yzjer/yzj-.git</span>
</code></pre>
    <h3>
     <a id="_59">
     </a>
     运行验证命令
    </h3>
    <pre><code class="prism language-bash">$ <span class="token function">ssh</span> <span class="token parameter variable">-T</span> git@github.com
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/93c10ed6c27a48c5a721eaed408b8829.png"/>
    </p>
    <p>
     成功！
     <br/>
     <em>
      这之后，以后push和pull都会通过SSH，但是记得每次都把协议先改成SSH
     </em>
    </p>
