    <h2>
     移动游戏类型详解
    </h2>
    <h3>
     ①角色扮演游戏（RPG）
    </h3>
    <div>
     <br/>
    </div>
    <div>
     <span style="font-size:18px">
      角色扮演游戏（Role Playing Game）:
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <span style="font-size:18px">
       <br/>
      </span>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <span style="font-size:18px">
       玩
      </span>
      家扮演一个或数个角色，有完整的故事情节，根据情节需求获取道具、装备来完成任务。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《斗破苍穹》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ②动作游戏（ACT）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      动作游戏是由
      <span style="font-size:18px">
       家
      </span>
      玩控制游戏人物，通过跳跃、攀爬、跑动以及各种武器、道具消灭敌人的过关游戏，有时也有简单的故事情节。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《波斯王子》《神庙逃亡》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ③冒险游戏（AVG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      冒险游戏是由玩家控制游戏人物进行虚拟冒险的游戏。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《生化危机4》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ④策略类游戏（SLG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      策略类游戏是由玩家运用策略与电脑或其他玩家较量，已取得胜利的游戏，或统一全国，或开拓外星殖民地。可分为回合制和即时制两种。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《植物大战僵尸》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      塔防类手机游戏《小小指挥官》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑤即时战略游戏（RTS）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      本来属于策略游戏SLG的一个分支，但由于其在世界上迅速风靡，RTS慢慢发展成了一个单独的类型，知名度甚至超过了SLG。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《帝国时代》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑥格斗游戏（FGT）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      格斗游戏是由玩家操作各种角色与电脑或另一玩家所控制的角色进行格斗的游戏。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《拳皇98》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑦射击类游戏（STG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      如：手机游戏《飞机大战》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑧第一人称视角射击类游戏（FPS）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      严格来说，第一人称视角射击类游戏属于动作游戏的一个分支，但与RTS一样，从DOOM以来，FPS就以令人难以置信的速度向前飞速发展。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《黑帮枪神》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑨休闲益智游戏（PZL）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      如：手机游戏《捕鱼达人》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑩竞速游戏（RCG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      在移动设备上模拟各类赛车运动的游戏，通常是在比赛场景下进行。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《极品飞车》
     </span>
    </div>
    <div>
     <br/>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑪体育游戏（SPT）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      在移动设备上模拟各类竞技体育运动的游戏。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《桌球大师》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑫养成类游戏（TCG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      玩家模拟培养的游戏。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《美少女计划》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑬卡牌类游戏（CAG）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      玩家操纵角色通过卡片战斗模式来进行游戏。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《我叫MT》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑭桌面游戏（TAB）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      各类强手棋、棋牌类游戏
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《德州扑克》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <h3>
     <span style="font-size:18px">
      ⑮音乐游戏（MSC）
     </span>
    </h3>
    <div>
     <span style="font-size:18px">
      培养玩家音乐敏感性，增强音乐感的游戏，伴随美妙的音乐，要求玩家做手指体操。
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      如：手机游戏《音乐英雄》
     </span>
    </div>
    <div>
     <span style="font-size:18px">
      <br/>
     </span>
