# Linux查询及kill所有python进程

当python进程异常中断时，需要查看所有的python进程。命令如下：

```
ps -ef | grep python
```

当采用并行处理的时候，会有非常多的进程存在，一个个kill非常麻烦，可以直接kill所有python进程。命令如下：

```
killall python
```
