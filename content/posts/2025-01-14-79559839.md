      <span style="font-size:18px;">
       二:
       <span style="color:#FF0000;">
        打开
        <span style="font-size:18px;">
         <span style="font-size:18px;">
          YDMHTTPD3
         </span>
        </span>
        进行配置
       </span>
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <span style="color:#FF0000;">
        目的
       </span>
       : 为了以后解决一些网站的验证码更好的使用此文件
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <span style="color:#FF0000;">
        注意
       </span>
       : 普通用户是通过开发者的id和key来进行接口调试的
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       普通用户没有权限调用接口, 只有开发者才有权限调用接口, 所以普通用户想要使用云打码进行在线识别, 必须借助于开发者的id和key, 而开发者也是依据id和key来进行分成的
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#FF0000;">
      <span style="font-size:18px;">
       1. 先获取id和key
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <img alt="" src="//img-blog.csdn.net/20180314204650421?watermark/2/text/Ly9ibG9nLmNzZG4ubmV0L3FxXzQxNjY0NTI2/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70">
        <br/>
       </img>
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#FF0000;">
      <span style="font-size:18px;">
       2. 对文件代码进行一些修改
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       1&gt; 因为你的用户名, 密码, appid, appkey都是固定, 为了以后的使用中不用频繁的使用这些信息, 可以将这些信息写入代码中, 之后运行自己的爬虫只需调用即可
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       2&gt; 为了在你的爬虫文件中引入此文件调用才能调用而不是直接调用.加入:
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#33CCFF;">
      <span style="font-size:18px;">
       if __name == '__main__'
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <span style="color:#FF0000;">
        加入的位置
       </span>
       : 在末端函数之后
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#FF0000;">
      <span style="font-size:18px;">
       3. 实际使用
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#33CCFF;">
      <span style="font-size:18px;">
       from (
       <span style="color:#000000;">
        文件的文件名
       </span>
       ) import YDMHttp
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#33CCFF;">
      <span style="font-size:18px;">
       ydm = YDMHttp( )
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#33CCFF;">
      <span style="font-size:18px;">
       ydm.login( )
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;color:#33CCFF;">
      <span style="font-size:18px;">
       cid, result = ydm.decode('captcha.png', 5000, 20)
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       注:
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <span style="color:#33CCFF;">
        result

       : 经过云打码后得到的验证码的结果
      </span>
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-size:18px;">
       <span style="color:#33CCFF;">
        captcha.png
       </span>
       : 验证码图片
      </span>
     </span>

     <span style="font-size:18px;">
      <span style="font-size:18px;">
