       索引与数据筛选
      </span>
     </strong>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <strong>
      <span style="color:red">
       为什么要筛选数据？
      </span>
     </strong>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       因为在处理数据的实际应用中，数据的筛选很重要。Like：要选择数据集里的某几行某几列，或者将符合某种条件的数据筛选出来单独处理和分析，此时就要进行特定处理。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       处理方案：
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       在pandas模块中，有很多种
      </span>
      <u>
       <span style="color:red">
        索引
       </span>
      </u>
      <span style="color:red">
       方式
      </span>
      <span style="color:#707070">
       来定位和选择DataFrame中的数据。
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#2f5df2">
       学习目标1
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       今天的课程里，我们将依此学习以下4种索引方式：
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       1.
      </span>
      <span style="color:#707070">
       列索引 ：访问列数据
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       2.
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        .loc
       </span>
      </span>
      <span style="color:#707070">
       属性：通过索引的值访问行数据和元素
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       3.
      </span>
      <span style="background-color:#fff5f5">
       <span style="color:#ff502c">
        .iloc
       </span>
      </span>
      <span style="color:#707070">
       属性：通过
      </span>
      <span style="color:#ffb033">
       索引的
      </span>
      <span style="color:#707070">
       位置访问行数据和元素
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       4.
      </span>
      <span style="color:#707070">
       行列索引的结合：访问元素
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:white">
      <span style="color:#707070">
       5.
      </span>
      <span style="color:#707070">
       布尔索引：一个判断条件或多个判断条件
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">

     <span style="background-color:white">

      <span style="color:#707070">

      </span>
     </span>

     </span>
     依次
     <span style="background-color:white">
      <span style="color:#707070">
