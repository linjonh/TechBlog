    <h3>
     <a id="_0">
     </a>
     问题与处理策略
    </h3>
    <h5>
     <a id="_2">
     </a>
     问题描述
    </h5>
    <ul>
     <li>
      在微信小程序项目中，配置并使用 tabBar 报如下错误
     </li>
    </ul>
    <pre><code>[ miniprogram/app.json 文件内容错误] miniprogram/app.json: 
["tabBar"][3]["pagePath"]: 
"pages/mine/mine" need in ["pages"](env: Windows,mp,1.06.2306020; lib: 3.7.8)
</code></pre>
    <h5>
     <a id="_12">
     </a>
     问题原因
    </h5>
    <ol>
     <li>
      <p>
       错误提示在
       <code>
        miniprogram/app.json
       </code>
       文件中，tabBar 配置的某个页面的 pagePath（
       <code>
        pages/mine/mine
       </code>
       ）没有在 pages 数组中注册
      </p>
     </li>
     <li>
      <p>
       微信小程序要求所有在 tabBar 中配置的页面路径，都必须先在 pages 数组中注册
      </p>
     </li>
    </ol>
    <h5>
     <a id="_18">
     </a>
     处理策略
    </h5>
    <ul>
     <li>
      在
      <code>
       miniprogram/app.json
      </code>
      文件中，注册好相应的页面后，再在 tabBar 中配置
     </li>
    </ul>
    <pre><code class="prism language-json"><span class="token punctuation">{<!-- --></span>
 <span class="token string-property property">"pages"</span><span class="token operator">:</span> <span class="token punctuation">[</span>

 <span class="token punctuation">]</span><span class="token punctuation">,</span>

   <span class="token punctuation">]</span>
 <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
