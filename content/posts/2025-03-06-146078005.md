    <h2>
     <a id="_6">
     </a>
     题目解析
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/afd3eea096db4f4aae00a9650ef42c61.png"/>
    </p>
    <blockquote>
     <ol>
      <li>
       搞懂定义
      </li>
     </ol>
     <ul>
      <li>
       <p>
        对于一个正整数，每一次替换为它每个位置上的平方和。
       </p>
      </li>
      <li>
       <p>
        举例：
       </p>
      </li>
      <li>
       <p>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7770beffb78f48c99491916d65a18289.png"/>
       </p>
       <ul>
        <li>
         19这个数字经过处理可以变成1，2这个数字变成了无限循环，
        </li>
        <li>
         所以19是快乐数字，2就不是
        </li>
        <li>
         判断最后的那一个环是否都是一
        </li>
       </ul>
      </li>
     </ul>
    </blockquote>
    <h2>
     <a id="_18">
     </a>
     算法原理解析
    </h2>
    <p>
     我们仔细观察，在最后的结尾，
    </p>
    <ul>
     <li>
      <p>
       为快乐数的数字最后的结尾都是1，我们可以理解成一个园环。
      </p>
     </li>
     <li>
      <p>
       非快乐数的数字最后结尾我们知道，肯定不是1，但是因为鸽巢原理我们会得出结论肯定成环。（不知道也没关系，下面有详细解析）
      </p>
     </li>
     <li>
      <p>
       原理如图所示，上面的是快乐数，下面的数是非快乐数：
       <br/>
       <img alt="原理图" src="https://i-blog.csdnimg.cn/direct/406d580a3b484fc48820a57399aee649.png"/>
      </p>
     </li>
     <li>
      <p>
       判断最后的环的数字是否都是1.
      </p>
     </li>
    </ul>
    <h3>
     <a id="_26">
     </a>
     具体解法
    </h3>
    <blockquote>
     <ul>
      <li>
       解法 快慢双指针。
       <ul>
        <li>
         定义快慢指针
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/77c7f69bb674437e9f61f2cb871d2003.png">
          &gt; - 慢指针每次后移一步，快指针每次后移两步 让慢指针一次进行一次操作，让快指针进行2次快乐数操作
          <br/>
          - 判断相遇的值
          <br/>
          - 直接判断相遇的值
         </img>
        </li>
       </ul>
      </li>
     </ul>
    </blockquote>
    <ul>
     <li>
      鸽巢原理详解：
      <ul>
       <li>
        我们可是让慢指针执行一次，然后对于快指针每一次后移进行执行快乐数的两次操作
        <ul>
         <li>
          <p>
           为什么这些数字不会一直铺开，为什么一定要成环？
           <br/>
           证明原理：鸽巢原理
          </p>
         </li>
         <li>
          <p>
           如果有n个巢，n+1个鸽子可以推论
          </p>
         </li>
         <li>
          <p>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9e4a606d04af4402a86cb7aa1284a104.png"/>
          </p>
          <p>
           证明这道题：
          </p>
         </li>
         <li>
          <p>
           一个数字2.1 * 10^9
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c243bd5a4ddc4199b429d173b240d262.png"/>
          </p>
          <ul>
           <li>
            <p>
             <strong>
              我们已经知道n个巢穴，n+1个鸽子，如果鸽子全部归位，至少有一个巢穴里面的鸽子大于1.
             </strong>
            </p>
           </li>
           <li>
            <p>
             我们拿出9999999999这个数字，进行快乐数操作，
            </p>
           </li>
           <li>
            <p>
             范围将会锁定在【1，810（9^2 * 10）】
             <strong>
              （这个就是巢穴）
             </strong>
             ，我们进行811次快乐数字操作（
             <strong>
              这个就是鸽子）
             </strong>
            </p>
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
