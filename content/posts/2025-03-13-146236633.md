    <p>
     在OpenFOAM中，
     <code>
      lduMatrix
     </code>
     是用于存储稀疏矩阵的类，支持并行计算。并行分区后，
     <code>
      lduMatrix
     </code>
     的矩阵向量乘法通过以下步骤实现：
    </p>
    <h4>
     <a id="1__2">
     </a>
     1. 矩阵分区
    </h4>
    <ul>
     <li>
      <strong>
       分区
      </strong>
      ：将矩阵和向量分配到多个处理器上，每个处理器负责一部分。
     </li>
     <li>
      <strong>
       接口
      </strong>
      ：分区后，处理器间存在共享的边界单元（interface cells），需通过通信同步这些单元的数据。
     </li>
    </ul>
    <h4>
     <a id="2__6">
     </a>
     2. 矩阵向量乘法
    </h4>
    <ul>
     <li>
      <strong>
       局部计算
      </strong>
      ：每个处理器独立计算其负责的矩阵部分与向量的乘积。
     </li>
     <li>
      <strong>
       边界同步
      </strong>
      ：由于边界单元涉及多个处理器，需通过MPI等通信机制交换数据，确保边界单元的值一致。
     </li>
    </ul>
    <h4>
     <a id="3__10">
     </a>
     3. 通信与同步
    </h4>
    <ul>
     <li>
      <strong>
       非阻塞通信
      </strong>
      ：使用非阻塞通信（如MPI_Isend和MPI_Irecv）提高效率。
     </li>
     <li>
      <strong>
       等待完成
      </strong>
      ：在所有通信完成后，继续后续计算。
     </li>
    </ul>
    <h4>
     <a id="4__14">
     </a>
     4. 结果合并
    </h4>
    <ul>
     <li>
      <strong>
       局部结果
      </strong>
      ：每个处理器计算局部结果。
     </li>
     <li>
      <strong>
       全局结果
      </strong>
      ：通过通信将局部结果合并为全局结果。
     </li>
    </ul>
