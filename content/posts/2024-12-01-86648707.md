    <p>
     查看音视频文件信息命令
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 3.mp4
ffmpeg -i 3.aac
ffmpeg -i 3.m4a
</code></pre>
    <p>
     抽取音频命令
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 3.mp4 -vn -y -acodec copy 3.aac
ffmpeg -i 3.mp4 -vn -y -acodec copy 3.m4a
</code></pre>
    <p>
     提取视频 （Extract Video）
    </p>
    <pre><code class="prism language-linux">ffmpeg -i input_file -vcodec copy -an output_file_video
</code></pre>
    <p>
     合并音视频:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 1.mp4 -i 1.m4a -vcodec copy -acodec copy output.mp4
</code></pre>
    <p>
     先转换成ts,然后快速合并:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 1.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 1.ts
ffmpeg -i 2.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 2.ts
ffmpeg -i "concat:1.ts|2.ts" -acodec copy -vcodec copy -absf aac_adtstoasc output.mp4
</code></pre>
    <p>
     参考:
     <a href="https://blog.csdn.net/east196/article/details/79821361">
      https://blog.csdn.net/east196/article/details/79821361
     </a>
    </p>
    <p>
     视频压缩命令:
    </p>
    <p>
     <a href="https://blog.csdn.net/zhezhebie/article/details/79263492">
      https://blog.csdn.net/zhezhebie/article/details/79263492
     </a>
     <br/>
     参考:
     <br/>
     <a href="https://www.jianshu.com/p/cf1e61eb6fc8" rel="nofollow">
      https://www.jianshu.com/p/cf1e61eb6fc8
     </a>
     <br/>
     <a href="https://blog.csdn.net/east196">
      https://blog.csdn.net/east196
     </a>
     <br/>
     <a href="https://blog.csdn.net/u012700515/article/details/80053785?utm_source=blogxgwz6">
      https://blog.csdn.net/u012700515/article/details/80053785?utm_source=blogxgwz6
     </a>
    </p>
    <p>
     只要视频:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 1.ts -vcodec copy -an 1_nomusic.ts
</code></pre>
    <p>
     合并视频:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i 1.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 1.ts
ffmpeg -i "concat:1_nomusic.ts|2.ts" -acodec copy -vcodec copy -absf aac_adtstoasc output.mp4
</code></pre>
    <p>
     减小音频:为原来2/3:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i music.mp3 -af volume=2/3 quiet_music.mp3
</code></pre>
    <p>
     合并音视频:
    </p>
    <pre><code class="prism language-linux">ffmpeg -i output_file.mp4 -i good.m4a -vcodec copy -acodec copy output.mp4
</code></pre>
    <p>
     增加字幕流
    </p>
    <pre><code class="prism language-linux">ffmpeg -i video.avi -i sub.ass -map 0:0 -map 0:1 -map 1 -c:a copy -c:v copy -c:s copy video.mkv
</code></pre>
    <p>
     提取字幕流
     <br/>
     1）原始文本输出
    </p>
    <pre><code class="prism language-linux">ffmpeg -i output.mkv -an -vn -bsf:s mov2textsub -scodec copy -f rawvideo sub.txt
ffmpeg -i output.mkv -an -vn -c:s copy -f rawvideo -map 0:s sub2.txt
</code></pre>
    <p>
     2）ass格式输出
    </p>
    <pre><code class="prism language-linux">ffmpeg -i output.mkv -an -vn -scodec copy sub3.ass
</code></pre>
