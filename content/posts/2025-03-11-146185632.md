    <p>
     路由器常见接口的详细介绍及其应用示例：
    </p>
    <h4>
     <a id="1_CONAUX__4">
     </a>
     <strong>
      1. CON/AUX 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全称
      </strong>
      ：Console/Auxiliary（控制台/辅助接口）
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：
      <ul>
       <li>
        <strong>
         CON（Console）
        </strong>
        ：通过命令行界面（CLI）直接配置路由器，常用于初始设置或故障排查。
       </li>
       <li>
        <strong>
         AUX（Auxiliary）
        </strong>
        ：辅助接口，早期用于连接调制解调器实现远程管理，现较少使用。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        使用Console线（如RJ45转USB）连接电脑，通过PuTTY等终端软件配置路由器。
       </li>
       <li>
        AUX接口在旧设备中可能通过调制解调器拨号进行远程维护。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="2_GE_Combo__13">
     </a>
     <strong>
      2. GE Combo 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全称
      </strong>
      ：Gigabit Ethernet Combo（千兆光电复用接口）
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：同一物理位置提供两种接口（如RJ45电口和SFP光口），但同一时间仅能启用其一。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        根据网络环境选择介质：短距离用RJ45（如连接交换机），长距离用SFP光纤模块（如连接机房核心设备）。
       </li>
       <li>
        华为AR系列路由器的GE Combo口常见于企业级设备。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="3_Mini_USBUSB__20">
     </a>
     <strong>
      3. Mini USB/USB 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Mini USB
      </strong>
      ：
      <ul>
       <li>
        <strong>
         作用
        </strong>
        ：旧款路由器用于固件升级或配置（现多被标准USB取代）。
       </li>
       <li>
        <strong>
         示例
        </strong>
        ：Cisco某些旧型号通过Mini USB连接电脑管理。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       USB 接口
      </strong>
      ：
      <ul>
       <li>
        <strong>
         作用
        </strong>
        ：
        <ul>
         <li>
          外接存储（如U盘/硬盘）实现文件共享或日志存储。
         </li>
         <li>
          插入4G USB调制解调器作为备用网络。
         </li>
         <li>
          固件升级（如上传新系统文件）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         示例
        </strong>
        ：
        <ul>
         <li>
          华硕路由器插入USB 4G网卡实现双WAN冗余。
         </li>
         <li>
          TP-Link路由器外接硬盘作为简易NAS。
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="4_WAN__Uplink__33">
     </a>
     <strong>
      4. WAN 侧 Uplink 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全称
      </strong>
      ：Wide Area Network Uplink（广域网上行接口）
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：连接上级网络设备（如光猫、骨干路由器），接入互联网或广域网。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        家庭路由器WAN口连接光猫的LAN口获取公网IP。
       </li>
       <li>
        企业级路由器通过WAN口接入专线（如MPLS VPN）。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="5_FE__40">
     </a>
     <strong>
      5. FE 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全称
      </strong>
      ：Fast Ethernet（快速以太网接口）
     </li>
     <li>
      <strong>
       速率
      </strong>
      ：100Mbps，最大传输距离100米（基于Cat5e线缆）。
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：连接低带宽需求的设备。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        连接IP电话、监控摄像头或旧款打印机。
       </li>
       <li>
        小型办公室中用于接入次要办公设备。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="6_GE__49">
     </a>
     <strong>
      6. GE 接口
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       全称
      </strong>
      ：Gigabit Ethernet（千兆以太网接口）
     </li>
     <li>
      <strong>
       速率
      </strong>
      ：1Gbps，支持更高速率的数据传输。
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：连接高带宽设备或作为局域网骨干。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        连接NAS设备实现高速文件传输。
       </li>
       <li>
        企业内网中交换机与路由器之间的千兆互联。
       </li>
      </ul>
     </li>
    </ul>

    <h4>
     <a id="_69">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Combo接口冲突
      </strong>
      ：启用光口时，对应的电口会自动禁用。
     </li>
     <li>
      <strong>
       WAN/LAN自适应
      </strong>
      ：部分路由器支持端口角色切换（如将LAN口设为WAN）。
     </li>
     <li>
      <strong>
       USB兼容性
      </strong>
      ：4G网卡需匹配路由器支持的型号（如华为E3372）。
     </li>
    </ul>
    <p>
     通过合理利用这些接口，可以优化网络拓扑结构并提升设备功能扩展性。例如，企业路由器通过GE Combo口光纤上行至核心交换机，同时USB接口接入4G模块实现网络冗余。
