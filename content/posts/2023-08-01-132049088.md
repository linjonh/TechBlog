# Python输出Hello，World！

Python是一种
**解释型、面向对象、动态数据类型**
的高级程序设计语言。

她由 Guido van Rossum 于 1989 年底发明，第一个公开发行版发行于 1991 年。

像 Perl 语言一样, Python 源代码同样遵循 GPL(GNU General Public License) 协议。

编写一个简单程序，输出“Hello,World!”：

```python
print("Hello, World!")
```

如要输出其他内容，只需用输出内容替换“Hello,World!”。

在Python中，你可以使用print语句或print()函数来输出内容。下面是几个例子：

使用print语句：
  
```python
  
print("Hello, world!")
  
```

使用print()函数：
  
```python
  
print("Hello, world!")
  
```

你也可以输出变量的值：
  
```python
  
name = "Alice"
  
print("My name is", name)
  
```

如果需要格式化输出，可以使用字符串的.format()方法或者f-string：
  
```python
  
age = 25
  
print("I am {} years old.".format(age))

or

print(f"I am {age} years old.")
  
```
