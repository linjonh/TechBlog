   <div class="htmledit_views" id="content_views">
    <p>
     在
     <a href="https://cloud.tencent.com/product/cdb?from_column=20065&amp;from=20065" rel="nofollow" title="MySQL">
      MySQL
     </a>
     5.7.2 新增了 log_timestamps 这个参数，该参数主要是控制 error log、genera log，等等记录日志的显示时间参数。 其值默认 为UTC 修改为 SYSTEM 就能解决问题。
    </p>
    <p>
     两种修改方式
    </p>
    <p>
     1：修改/etc/my.cnf
    </p>
    <p>
     <strong>
      log_timestamps = SYSTEM
     </strong>
    </p>
    <p>
     重启服务
    </p>
    <p>
     2：连接到数据库
    </p>
    <pre><code>SHOW GLOBAL VARIABLES LIKE 'log_timestamps';
+----------------+--------+
| Variable_name  | Value  |
+----------------+--------+
| log_timestamps | UTC    |
+----------------+--------+
 
SET GLOBAL log_timestamps = SYSTEM;
Query OK, 0 rows affected (0.00 sec)
 
SHOW GLOBAL VARIABLES LIKE 'log_timestamps';
+----------------+--------+
| Variable_name  | Value  |
+----------------+--------+
| log_timestamps | SYSTEM |
+----------------+--------+</code></pre>
