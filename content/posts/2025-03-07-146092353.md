    <h2>
     一、软件定位与核心功能
    </h2>
    <p>
     GIMP（GNU Image Manipulation Program）是
     <strong>
      开源跨平台图像编辑工具
     </strong>
     ，支持图层管理、高级修图、色彩校正等功能，被广泛应用于平面设计、照片修复等领域。相比商业软件，其优势包括：
    </p>
    <ul>
     <li>
      <strong>
       全功能免费
      </strong>
      ：支持PSD文件编辑、CMYK色彩模式
     </li>
     <li>
      <strong>
       插件生态丰富
      </strong>
      ：600+扩展插件增强功能
     </li>
     <li>
      <strong>
       硬件要求低
      </strong>
      ：1GB内存即可流畅运行
     </li>
    </ul>
    <hr/>

    <h4>
     2. 运行库检测
    </h4>
    <p>
     需安装最新版
     <strong>
      Microsoft Visual C++ Redistributable
     </strong>
     （安装包已集成，无需单独下载）
    </p>
    <hr/>
    <h3>
     三、安装流程详解
    </h3>
    <h4>
     步骤1：获取安装包
    </h4>
    <p>
     访问
     <span contenteditable="false" tabindex="-1">
      <a href="https://pan.quark.cn/s/5de67e6f6176" rel="nofollow noopener noreferrer" target="_blank" title="GIMP官方安装包下载页">
       GIMP官方安装包下载页
      </a>
     </span>
     ，选择 gimp-2.10.38-setup-1.zip
    </p>
    <h4>
     步骤2：启动安装程序
    </h4>
    <p>
     解压压缩包， 右键以管理员身份运行
     <code>
      gimp-2.10.38-setup-1.exe
     </code>
     文件，选择简体中文安装语言：
    </p>
    <p>
     <img alt="" height="211" src="https://i-blog.csdnimg.cn/direct/4f0a36f9ddf14437870fa4fb0ca417a0.png" width="349"/>
    </p>
    <h4>
     步骤3：左下角点击自定义，设置安装路径
    </h4>
    <p>
     建议修改默认路径（示例）：
    </p>
    <pre><code>D:\DesignTools\GIMP 2.10 
</code></pre>
    <h4>
     <img alt="" height="482" src="https://i-blog.csdnimg.cn/direct/dd24190d04104b7c8c406c138d797ebc.png" width="582"/>
    </h4>
    <h4>
     步骤4：选择组件安装
    </h4>
    <p>
     勾选以下核心组件：
     <br/>
     ☑️ 核心程序文件
     <br/>
     ☑️ 附加图标集
     <br/>
     ☑️ Python脚本支持（开发扩展必备）
     <img alt="" height="482" src="https://i-blog.csdnimg.cn/direct/737abd271e7242989ec5cbdd1ae58fb9.png" width="582"/>
    </p>
    <h4>
     步骤5：完成安装
    </h4>
    <p>
     勾选
     <strong>
      创建桌面快捷方式
     </strong>
     ，等待进度条完成后点击完成
    </p>
    <hr/>
    <h3>
     四、首次运行配置
    </h3>
    <h4>
     1. 初始化设置
    </h4>
    <p>
     首次启动时选择工作区布局（推荐
     <strong>
      Default
     </strong>
     或
     <strong>
      Dark
     </strong>
     主题）
    </p>
    <h4>
     2. 中文语言包安装
    </h4>
    <p>
     通过
     <strong>
      Edit &gt; Preferences &gt; Interface &gt; Language
     </strong>
     选择「简体中文」，重启生效
    </p>
    <hr/>
    <h3>
     五、功能验证
    </h3>
    <h4>
     测试1：基础图像处理
    </h4>
    <p>
     打开示例图片，使用
     <strong>
      自由选区工具
     </strong>
     进行抠图操作
    </p>
    <h4>
     测试2：插件扩展安装
    </h4>
    <p>
     通过
     <strong>
      菜单 &gt; 帮助 &gt; 显示插件浏览器
     </strong>
     安装第三方插件（如G'MIC图像特效库）
    </p>
    <hr/>
    <h3>
     六、常见问题解答
    </h3>
    <h4>
     Q1：启动时报错"MSVCP140.dll 丢失"？
    </h4>
    <ul>
     <li>
      安装Visual Studio 2015+运行库
     </li>
     <li>
      或重新运行安装包勾选
      <strong>
       Install runtime
      </strong>
      选项
     </li>
    </ul>
    <h4>
     Q2：如何导出PSD文件？
    </h4>
    <p>
     通过
     <strong>
      文件 &gt; 导出为
     </strong>
     选择Photoshop格式，保留图层信息
    </p>
    <h4>
     Q3：软件界面显示异常？
    </h4>
    <p>
     重置用户配置：删除
     <code>
      C:\Users\用户名\.gimp-2.10
     </code>
     文件夹
    </p>
