    <h3>
     <a id="Stop_Drawing__1">
     </a>
     Stop Drawing 按钮
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a43f7d387483454f9c43947053369d02.png#pic_center"/>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        暂停波形更新
       </strong>
       ：
      </p>
      <ul>
       <li>
        点击“Stop Drawing”按钮时，ModelSim 会暂停将仿真结果实时绘制到波形窗口中。仿真继续运行，波形窗口不会实时更新显示新的仿真数据。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        提高仿真速度
       </strong>
       ：
      </p>
      <ul>
       <li>
        在大规模设计或长时间仿真的情况下，实时更新波形可能会显著减慢仿真速度。通过暂停波形更新，可以减少图形处理的开销，从而加快仿真的执行速度。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        节省资源
       </strong>
       ：
      </p>
      <ul>
       <li>
        暂停波形更新可以减少内存和 CPU 的使用，特别是在处理大量信号或复杂波形时。
        <strong>
         波形数据依然保存
        </strong>
        ，暂停波形更新并不会丢失任何仿真数据。所有仿真数据都会被保存，并且在恢复波形更新时会正确显示。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        手动控制波形刷新
       </strong>
       ：
      </p>
      <ul>
       <li>
        使用“Stop Drawing”后，您可以选择在特定的时间点或仿真事件发生后再恢复波形更新，以便更精确地控制波形窗口中的信息量和内容。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_18">

     加速波形绘制
