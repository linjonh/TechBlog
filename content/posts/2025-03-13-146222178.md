    <h5 style="text-align:justify">
     <a name="_Toc7392">
     </a>
     安装及配置
    </h5>
    <p style="margin-left:20.2500pt">
     1、下载安装包mysql-version-
     <a href="http://lib.csdn.net/base/linux" rel="nofollow" title="linux">
      linux
     </a>
     -glibc2.5-x86_64.tar（可前往官网自行下载：
     <a href="http://dev.mysql.com/downloads/mysql/%EF%BC%89" rel="nofollow" title="http://dev.mysql.com/downloads/mysql/）">
      http://dev.mysql.com/downloads/mysql/）
     </a>
    </p>
    <p style="margin-left:20.2500pt">
     2、卸载系统自带的MariaDB
     <br/>
     打开Terminal终端：
    </p>
    <p style="margin-left:20.2500pt">
     # rpm -qa|grep mariadb  //查询出来已安装的mariadb
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # rpm -e --nodeps 文件名  //卸载mariadb，文件名为上述命令查询出来的文件
     <br/>
     3、删除etc目录下的my.cnf
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # rm /etc/my.cnf
    </p>
    <p style="margin-left:20.2500pt">
     4、执行以下命令来创建mysql用户组
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # groupadd mysql
    </p>
    <p style="margin-left:20.2500pt">
     5、执行以下命令来创建一个用户名为mysql的用户并加入mysql用户组
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # useradd -g mysql mysql
    </p>
    <p style="margin-left:20.2500pt">
     6、将下载的压缩包放到 /usr/local/ 目录下（通过mv 要移动的文件  /usr/local/）
    </p>
    <p style="margin-left:20.2500pt">
     7、解压安装包
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # tar -zxvf mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz
    </p>
    <p style="margin-left:20.2500pt; text-align:justify">
     8、将解压好的文件夹重命名为mysql
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # mv 解压出来的文件夹名 mysql
    </p>
    <p>
     9、进入安装mysql软件目录
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # cd /usr/local/mysql
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     修改当前目录拥有着为mysql用户
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # chown -R mysql:mysql ./
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     安装autoconf库
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     命令: yum -y install autoconf   //此包安装时会安装Data:Dumper模块
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # chown -R mysql:mysql data 修改当前data目录的拥有者为mysql用户
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     安装数据库
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # cd /usr/local/mysql/bin/
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # ./mysql_install_db --user=mysql --basedir=/usr/local/mysql/ --datadir=/usr/local/mysql/data/
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     # /usr/local/mysql/bin/mysqld initialize  --user=mysql --basedir=/usr/local/mysql/ --datadir=/usr/local/mysql/data/
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
     到此数据库安装完毕！
    </p>
