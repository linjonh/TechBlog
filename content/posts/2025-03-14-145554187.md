      <strong>
       位运算内容总结
      </strong>
     </span>
    </h3>
    <blockquote>
     <p>
      <strong>
       1.基础位运算
      </strong>
     </p>
     <p>
      &lt;&lt;  &gt;&gt; ~
     </p>
     <p>
      &amp;:有0就是0
     </p>
     <p>
      | :有1就是1
     </p>
     <p>
      ^:相同为0，相异为1 / 无进位相加
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       2.给一个数n，确定它的二进制表示中的第x位是0还是1
      </strong>
     </p>
     <p>
      &amp;上一个1，要么让第x位右移，要么让1左移，一般用右移x位，  (n&gt;&gt;x)&amp;1
     </p>
    </blockquote>
    <blockquote>
     <p>
      3.将一个数n的二进制表示的第x位修改成1
     </p>
     <p>
      n |= (1&lt;&lt;x)     等价于  n = n | (1 &lt;&lt; x)
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       4.将一个数n的二进制表示的第x位修改为0
      </strong>
     </p>
     <p>
      n &amp;= ~(1&lt;&lt;x)     1左移之后按位取反
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       5.位图的思想
      </strong>
     </p>
     <p>
      2，3，4其实就是一类问题，本质就是一个哈希表，很多情况下就是一个数组。位图就是用一个整形变量的二进制比特位来记录信息。
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       6.提取一个数(n)二进制表示中最右侧的1
      </strong>
      (lowbit)
     </p>
     <p>
      n &amp; -n
     </p>
     <p>
      如何表示-n：n的按位取反再加1，本质就是将最右侧的1的左边区域全部变成相反。
     </p>
     <p>
      <img alt="" height="156" src="https://i-blog.csdnimg.cn/direct/dd204d0eb2524f9d82edf12b85fe5cd6.png" width="261"/>
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       7.去掉一个数(n)二进制表示中最右侧的1
      </strong>
     </p>
     <p>
      n &amp; (n - 1)
     </p>
     <p>
      n-1:将最右侧的1的右边的区域(包含1)全部取反
     </p>
     <p>
      <img alt="" height="151" src="https://i-blog.csdnimg.cn/direct/a8fd19b5bd984dcbbb8afcc1097dace6.png" width="255"/>

    </blockquote>

       9.异或(^)运算的运算律

    <h3>
     <span style="color:#38d8f0">
      <strong>
       面试题01.01.判断字符是否唯一【简单】
      </strong>
     </span>
    </h3>
    <p>
     <a class="link-info" href="https://leetcode.cn/problems/is-unique-lcci/" rel="nofollow" title="力扣链接">
      力扣链接
     </a>
    </p>
    <blockquote>
     <p>
      实现一个算法，确定一个字符串
      <code>
       s
      </code>
      的所有字符是否全都不同。
     </p>
     <p>
      <strong>
       示例 1：输入:
      </strong>
      <code>
       s
      </code>
      = "leetcode"
      <strong>
       输出:
      </strong>
      false
     </p>
     <p>
      <strong>
       示例 2：输入:
      </strong>
      <code>
       s
      </code>
      = "abc"
      <strong>
       输出:
      </strong>
      true
     </p>
     <p>
      <strong>
       限制：
      </strong>
     </p>
     <ul>
      <li>
       <code>
        0 &lt;= len(s) &lt;= 100
       </code>
      </li>
      <li>
       <code>
        s[i]
       </code>
       仅包含小写字母
      </li>
      <li>
       如果
       <span style="color:#fe2c24">
        你不使用额外的数据结构，会很加分
       </span>
       。
      </li>
     </ul>
    </blockquote>
    <p>
     <span style="color:#0d0016">
      <strong>
       解法一：哈希表
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       解法二：位图
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       优化点：鸽巢原理(抽屉原理)
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      如果字符串长度&gt;26个字母，必定有一个字符是重复的。
     </span>
    </p>
    <pre><code class="language-cpp">class Solution {
public:
    bool isUnique(string astr) {
        if(astr.size() &gt; 26) return false;//鸽巢原理做优化

        int bitMap = 0;//设置位图
        for (auto ch : astr)
        {
            int i = ch - 'a';//取得比特位之差
            if (((bitMap &gt;&gt; i) &amp; 1 == 1)) return false;
            else bitMap |=(1&lt;&lt;i);//此时位图中没有这个字符，所以将i位的位图设置位1
        }
        return true;
    }
};</code></pre>

      268.消失的数字【简单】

      <code>
       [0, n]

       nums

      <code>
       [0,2]
      </code>
      内。2 是丢失的数字，因为它没有出现在

     </p>
     <p>
      <strong>

     <span style="color:#0d0016">
      <strong>
       解法三：位运算(异或)：补齐数组再异或

    <pre><code class="language-cpp">class Solution {
public:
    int missingNumber(vector&lt;int&gt;&amp; nums) {
        int ret = 0;
        for (auto x : nums) ret ^= x;
        for (int i = 0; i &lt;= nums.size(); i++) ret ^= i;
        return ret;
    }
};</code></pre>
    <h3>
     <span style="color:#38d8f0">
      <strong>
       137.只出现一次的数字Ⅱ
      </strong>
     </span>
    </h3>
    <p>
     <a class="link-info" href="http://137.%20只出现一次的数字%20II" rel="nofollow" title="力扣链接">
      力扣链接
     </a>
    </p>
    <blockquote>
     <p>
      给你一个整数数组
      <code>
       nums
      </code>
      ，除某个元素仅出现
      <strong>
       一次
      </strong>
      外，其余每个元素都恰出现
      <strong>
       三次 。
      </strong>
      请你找出并返回那个只出现了一次的元素。
     </p>
     <p>
      你必须设计并实现线性时间复杂度的算法且使用常数级空间来解决此问题。
     </p>
     <p>
     </p>
     <p>
      <strong>
       示例 1：输入：
      </strong>
      nums = [2,2,3,2]
      <strong>
       输出：
      </strong>
      3
     </p>
     <p>
      <strong>
       示例 2：输入：
      </strong>
      nums = [0,1,0,1,0,1,99]
      <strong>
       输出：
      </strong>
      99
     </p>
    </blockquote>
    <p>
     <strong>
      思路：
     </strong>
    </p>
    <blockquote>
     <ol>
      <li>
       <p>
        <strong>
         逐位处理
        </strong>
        ：遍历整数的每一位（共32位，因为
        <code>
         int
        </code>
        类型占32位）。
       </p>
      </li>
      <li>
       <p>
        <strong>
         统计每位1的次数
        </strong>
        ：对于每一位，统计数组中所有数字在该位上1出现的总次数。
       </p>
      </li>
      <li>
       <p>
        <strong>
         取模确定结果位
        </strong>
        ：由于其他数字出现三次，该位的总次数对3取余即为唯一数字在该位的值（0或1）。
       </p>
      </li>
      <li>
       <p>
        <strong>
         组合结果
        </strong>
        ：将各二进制位的值组合，得到最终结果。
       </p>
      </li>
     </ol>
    </blockquote>
    <pre><code class="language-cpp">int singleNumber(vector&lt;int&gt;&amp; nums) {
    int res = 0;
    for (int i = 0; i &lt; 32; ++i) { // 遍历每一位（0到31）
        int cnt = 0;
        for (auto x : nums) { // 统计所有数字在第i位的1的总数
            cnt += (x &gt;&gt; i) &amp; 1; // 取出x的第i位，累加到cnt
        }
        res |= (cnt % 3) &lt;&lt; i; // 取余得到结果位，合并到res
    }
    return res;
}</code></pre>

     <span style="color:#38d8f0">
      371.两整数之和
     </span>
    </h3>
    <p>
     <a class="link-info" href="https://leetcode.cn/problems/sum-of-two-integers/" rel="nofollow" title="力扣链接">
      力扣链接
     </a>
    </p>
    <blockquote>
     <p>
      给你两个整数
      <code>
       a
      </code>
      和
      <code>
       b
      </code>
      ，
      <strong>
       不使用
      </strong>
      运算符
      <code>
       +
      </code>
      和
      <code>
       -
      </code>
      ，计算并返回两整数之和。
     </p>
     <p>
     </p>
     <p>
      <strong>
       示例 1：输入：
      </strong>
      a = 1, b = 2
      <strong>
       输出：
      </strong>
      3
     </p>
     <p>
      <strong>
       示例 2：输入：
      </strong>
      a = 2, b = 3
      <strong>
       输出：
      </strong>
      5
     </p>
    </blockquote>
    <pre><code class="language-cpp">class Solution {
public:
    int getSum(int a, int b) {
        while(b != 0)
        {
            int x = a ^ b;
            unsigned int carry = (unsigned int)(a &amp; b) &lt;&lt; 1;
            a = x;
            b = (int)carry;
        }
        return a;
    }
};</code></pre>
    <h3>
     <span style="color:#38d8f0">
      面试题17.19 消失的两个数字
     </span>
    </h3>
    <p>
     <span style="color:#38d8f0">
      <a class="link-info" href="https://leetcode.cn/problems/missing-two-lcci/" rel="nofollow" title="力扣链接">
       力扣链接
      </a>
     </span>
    </p>
    <blockquote>
     <p>
      给定一个数组，包含从 1 到 N 所有的整数，但其中缺了两个数字。你能在 O(N) 时间内只用 O(1) 的空间找到它们吗？
     </p>
     <p>
      以任意顺序返回这两个数字均可。
     </p>
     <p>
      <strong>
       示例 1：输入：
      </strong>
      <code>
       [1]
      </code>
      <strong>
       输出：
      </strong>
      [2,3]
     </p>
     <p>
      <strong>
       示例 2：输入：
      </strong>
      <code>
       [2,3]
      </code>
      <strong>
       输出：
      </strong>
      [1,4]
     </p>
    </blockquote>
    <pre><code class="language-cpp">class Solution {
public:
    vector&lt;int&gt; missingTwo(vector&lt;int&gt;&amp; nums) {
        //1.将所有的数异或在一起
        int tmp = 0;
        for (auto x : nums) tmp ^= x;
        for (int i = 0; i &lt;= nums.size() + 2; i++) tmp ^= i;

        //2.找出a,b中比特位不同的那一位
        int diff = 0;
        while(1)
        {
            if (((tmp &gt;&gt; diff) &amp; 1) == 1) break;
            else diff++;
        }
        //3.根据diff位的不同，将所有的数划分为两类来异或
        int a = 0, b = 0;
        for (int x : nums)
            if (((x &gt;&gt; diff) &amp; 1) == 1) b ^= x;
            else a ^= x;
        for (int i = 1; i &lt;= nums.size() + 2; i++)
            if (((i &gt;&gt; diff) &amp; 1) == 1) b ^= i;
            else a ^= i;
        return {a, b};
    }
};</code></pre>
