    <h2>
     <a id="_0">
     </a>
     概述
    </h2>
    <p>
     这部分基本是引言introduction下面的一个模块，our work
     <br/>
     你可以描述一遍你们的工作；或者说直接上流程图一目了然，也可以两者结合
     <br/>
     但是美赛还是需要好看的图的，如果你是论文手/建模手可以参考以下做法
    </p>
    <h2>
     <a id="Visio_4">
     </a>
     使用Visio画出基本框图
    </h2>
    <p>
     我的基本构架当然是自己想出来的，然后使用Visio把框图的基本位置定好
     <br/>
     Visio可以有效
     <mark>
      对齐、分配方框位置
     </mark>
     ；配色也还可以（画框图的软件太多了，会什么用什么）
     <br/>
     （对自己审美没有信心的可以找帮忙配色的网站）
    </p>
    <h2>
     <a id="PS_8">
     </a>
     使用PS细化
    </h2>
    <p>
     1.在PS中标好箭头和文字
     <br/>
     注意：只使用
     <mark>
      黑白
     </mark>
     两种颜色的字体
     <br/>
     在PS而不是直接在Visio,为了方便移动文字位置/改变大小
     <br/>
     2.使用PS抠图
     <br/>
     将和文章有关系的一些小图，放置在流程图中
     <br/>
     比如我在此题流程图中放置了：潜水器submersible;网络；海浪；象征资料和数据的图
     <br/>
     3.虚线的添加
     <br/>
     虚线我也是直接抠图改变大小，位置和颜色来确保和谐
     <br/>
     虚线可以使流程图更加有序，一个模型在一个虚线内
    </p>
    <h2>
     <a id="_18">
     </a>
     注意整体
    </h2>
    <p>
     文字放到这步的原因就是图和文字要和谐好看，注意位置
     <br/>
     然后在PS中直接利用颜色覆盖和透明度调色小图片，务必和谐
    </p>
