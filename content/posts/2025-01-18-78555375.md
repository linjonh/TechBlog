    <h3 id="前言">
     前言
    </h3>
    <p>
     在
     <code>
      Python
     </code>
     中，我们通常使用
     <code>
      urllib
     </code>
     中的
     <code>
      urlencode
     </code>
     方法将字典编码，用于提交数据给
     <code>
      url
     </code>
     等操作，但是在
     <code>
      Python2
     </code>
     和
     <code>
      Python3
     </code>
     中
     <code>
      urllib
     </code>
     模块中所提供的
     <code>
      urlencode
     </code>
     的包位置有些不同。
    </p>
    <hr/>
    <h3 id="对于python2">
     对于Python2
    </h3>
    <p>
     <code>
      Python2
     </code>
     中提供了
     <code>
      urllib
     </code>
     和
     <code>
      urllib2
     </code>
     两个模块。
    </p>
    <ul>
     <li>
      <code>
       urlencode
      </code>
      方法所在位置为：
     </li>
    </ul>
    <pre class="prettyprint"><code class="language-python hljs">urllib.urlencode(values)    <span class="hljs-comment"># 其中values为所需要编码的数据，并且只能为字典</span></code></pre>
    <ul>
     <li>
      例如模拟登陆CSDN网站，示例程序如下
     </li>
    </ul>
    <pre class="prettyprint"><code class="language-python hljs"><span class="hljs-keyword">import</span> urllib2

values = {<!-- --><span class="hljs-string">"username"</span>:<span class="hljs-string">"962457839@qq.com"</span>,<span class="hljs-string">"password"</span>:<span class="hljs-string">"XXXX"</span>}
data = urllib.urlencode(values) 
url = <span class="hljs-string">"https://passport.csdn.net/account/login?from=http://my.csdn.net/my/mycsdn"</span>
request = urllib2.Request(url,data)
response = urllib2.urlopen(request)
<span class="hljs-keyword">print</span> response.read()</code></pre>
    <hr/>
    <h3 id="对于python3">
     对于Python3
    </h3>
    <p>
     <code>
      Python3
     </code>
     中也有
     <code>
      urllib
     </code>
     和
     <code>
      urllib3
     </code>
     两个库，其中
     <code>
      urllib
     </code>
     几乎是
     <code>
      Python2
     </code>
     中
     <code>
      urllib
     </code>
     和
     <code>
      urllib2
     </code>
     两个模块的集合，所以我们最常用的
     <code>
      urllib
     </code>
     模块，而
     <code>
      urllib3
     </code>
     则作为一个拓展模块使用。
    </p>
    <ul>
     <li>
      <code>
       urlencode
      </code>
      方法所在位置
     </li>
    </ul>
    <pre class="prettyprint"><code class="language-python hljs">urllib.parse.urlencode(values)</code></pre>
    <ul>
     <li>
      例如模拟登陆CSDN网站，示例程序如下:
     </li>
    </ul>
    <pre class="prettyprint"><code class="language-python hljs"><span class="hljs-keyword">from</span> urllib <span class="hljs-keyword">import</span> request
<span class="hljs-keyword">from</span> urllib <span class="hljs-keyword">import</span> parse
<span class="hljs-keyword">from</span> urllib.request <span class="hljs-keyword">import</span> urlopen

values = {<!-- --><span class="hljs-string">'username'</span>: <span class="hljs-string">'962457839@qq.com'</span>, <span class="hljs-string">'password'</span>: <span class="hljs-string">'XXXX'</span>}
data = parse.urlencode(values).encode(<span class="hljs-string">'utf-8'</span>)  <span class="hljs-comment"># 提交类型不能为str，需要为byte类型</span>
url = <span class="hljs-string">'https://passport.csdn.net/account/login?from=http://my.csdn.net/my/mycsdn'</span>
request = request.Request(url, data)
response = urlopen(request)
print(response.read().decode())</code></pre>
