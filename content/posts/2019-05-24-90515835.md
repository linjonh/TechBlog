    <p>
     所以我们在 Windows系统上运行docker命令时首先要启动
     <code>
      docker-machine
     </code>
    </p>
    <pre><code class="prism language-bash">docker-machine start default
</code></pre>
    <p>
     而
     <code>
      docker-machine
     </code>
     可以创建多个,默认执行docker命令是在名称为 default 的 docker-machine 下执行的,如果想要修改的话可以到环境变量中修改名为
     <code>
      DOCKER_MACHINE
     </code>
     的环境变量即可 .
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># git bash 中执行打开资源管理器</span>
start <span class="token variable">$DOCKER_CERT_PATH</span>

<span class="token comment"># cmd 中执行 start 打开资源管理器</span>
start %DOCKER_CERT_PATH%
</code></pre>
    <p>
     如果我们想要修改 docker 的默认镜像源的话,可以去上面命令打开的资源管理器中找
     <code>
      config.json
     </code>
     这个配置文件,这里假定默认的docker-mahcine虚拟机的名称为default.修改
     <code>
      config.json
     </code>
     文件即可.
     <br/>
     这里我们修改的镜像加速源的位置在
     <code>
      $.HostOptions.EngineOptions.RegistryMirror
     </code>
     下:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/130c944f3ceafa6f6d644666f06e4e15.png">
      <br/>
      修改好配置文件后保存.
     </img>
    </p>
    <pre><code class="prism language-bash">docker-machine <span class="token function">ssh</span> default
<span class="token comment"># 这里--registry-mirror 后面要替换为你想要的替换的镜像源</span>
<span class="token function">sed</span> <span class="token parameter variable">-e</span> <span class="token punctuation">\</span>
<span class="token string">"/--label provider/a\--registry-mirror https://xxxxxxxx.mirror.aliyuncs.com"</span> <span class="token punctuation">\</span>
/var/lib/boot2docker/profile
<span class="token function">sudo</span> /etc/init.d/docker restart
<span class="token builtin class-name">exit</span>
<span class="token comment"># 重启 docker-machine</span>
docker-machine restart default
<span class="token comment"># 检查 修改是否完成</span>
docker-machine inspect default
</code></pre>
    <p>
     在Windows命令行执行docker-machine ssh [machine-name]进入VM bash
     <br/>
     sudo vi /var/lib/boot2docker/profile
     <br/>
     在–label provider=virtualbox的下一行添加
     <br/>
     <code>
      --registry-mirror https://xxxxxxxx.mirror.aliyuncs.com
     </code>
