    <h2>
     <a id="DHCP_0">
     </a>
     下载DHCP服务器
    </h2>
    <pre><code class="prism language-bash">yum <span class="token function">install</span> -y dhcp
</code></pre>
    <p>
     我已经下载好了所以显示已经安装
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/425f5544be58f80ca52a40a5c0d88210.png"/>
    </p>
    <h2>
     <a id="_7">
     </a>
     配置
    </h2>
    <pre><code class="prism language-bash">vim /etc/dhcp/dhcpd.conf
</code></pre>
    <p>
     这里说明一下，# 代表注释 我所在的网段是192.168.11.0，配置文件里我只配置了地址池、广播地址和网关
    </p>
    <pre><code class="prism language-bash">subnet 192.168.11.0 netmask 255.255.255.0 <span class="token punctuation">{<!-- --></span>
                  range 192.168.11.160 192.168.11.200<span class="token punctuation">;</span> <span class="token comment">#地址池的范围     </span>
<span class="token comment">#                  option domain-name-servers test.cn;    //为客户端指明DNS服务器IP地址</span>
<span class="token comment">#                  option domain-name "ristar.ristar.com";     //  为客户端指明DNS名字。 </span>
                  option routers 192.168.11.1<span class="token punctuation">;</span>        <span class="token comment">#//路由器ip，可以写网关ip</span>
                  option broadcast-address 192.168.11.255<span class="token punctuation">;</span> <span class="token comment">#  广播地址</span>
                 <span class="token comment"># default-lease-time 1600;    #指定确省租赁时间的长度，单位是秒。</span>
                 <span class="token comment"># max-lease-time 7200;   #指定最大租赁时间长度，单位是秒。</span>
                 <span class="token comment"># filename "pxelinux.0";  # //开始启动文件的名称. 应用于无盘工作站。 </span>
             <span class="token comment">#     next-server 192.168.20.156; #   //设置服务器从引导文件中装如主机名，应用于无盘工作站。</span>
        <span class="token punctuation">}</span>
</code></pre>
    <p>
     保存配置文件后重启DHCP服务器
    </p>
    <pre><code class="prism language-bash">systemctl restart dhcpd
</code></pre>
    <p>
     查看DHCP服务器状态的命令
    </p>
    <pre><code class="prism language-bash">systemctl status dhcpd
</code></pre>
    <h2>
     <a id="DHCPip_36">
     </a>
     客户端获得DHCP服务器获得的ip
    </h2>
    <p>
     我用来测试的客户端也是一台Linux系统的虚拟机
    </p>
    <ol>
     <li>
      指定DHCP服务器的ip地址 (注意:如果不指定DHCP服务器的话，可能获得ip的地方就不是你这台虚拟机，而是你所在局域网中其他的DHCP服务器，指定的话有利于实验的成功)
      <br/>
      dhclient -s [DHCP服务器ip]
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c18e6d0aca861c9bfeeb8328334bef43.png"/>
     </li>
     <li>
      释放原来的ip
     </li>
    </ol>
    <pre><code class="prism language-bash">dhclient -r 
</code></pre>
    <ol start="3">
     <li>
      重新获得ip
     </li>
    </ol>
    <pre><code class="prism language-bash">dhclient
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b6ba6d6ee34fc616c8e96e7175c2af5.png">
      获得的ip是192.168.11.160，刚好是我们设置ip地址池的第一个ip
