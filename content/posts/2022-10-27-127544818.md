    <p>
     <strong>
      【问题描述】
     </strong>
     <br/>
     根据畅捷通T+Cloud专属云18.0的最新安装文档可以看到，
     <br/>
     支持软件的操作系统，已经没有win10和win11了；
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cba3adde73158f5bc4bb39213cf25d58.png">
      <br/>
      并且在安装过程中提示，
      <br/>
      T+18.0仅支持windows server2016及以上版本安装，当前系统不支持。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/87a0e3508b49c26ff7ec5089b24e4cb3.png">
       <br/>
       <strong>
        【解决方案】
       </strong>
       <br/>
       通过系统提示，系统不支持，
       <br/>
       相当于是软件安装过程中先要进行【环境检测】，当检测通过后才可以顺利的安装；
       <br/>
       所以可以使用跳过环境检测的方案，进行安装测试。
      </img>
     </img>
    </p>
    <p>
     以
     <strong>
      管理员身份
     </strong>
     运行命令窗口，
     <br/>
     然后根据软件安装程序所在的路径，
     <br/>
     输入D:\TPlus\setup_std.exe /nocheck=true回车后，软件开始运行程序的安装。
     <br/>
     其中：std是标准版，pro是专业版；根据自己的需要进行修改。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c33e39875018a4bd1fbfbb463bd51a4c.jpeg">
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/002fa846e7a800e2f1fd17dd8d02d7a1.png">
       <br/>
       <strong>
        【备注】
       </strong>
       <br/>
       此方法仅适用于自己测试使用；
       <br/>
       如果商业使用请按软件标准环境进行安装，至于后续厂家是否针对操作系统环境做出优化，在此不做任何评判。
       <br/>
       PS：现在从官网下载的最新安装程序，已经不需要跳过环境就可以安装了。
      </img>
     </img>
    </p>
