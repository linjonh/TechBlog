    <p>
     在HTML中，元素可以分为块级元素（Block-level elements）和内联元素（Inline elements）。这两类元素在页面布局和样式应用上有不同的特点和用途。
    </p>
    <h4>
     <a id="Blocklevel_elements_2">
     </a>
     一、块级元素（Block-level elements）
    </h4>
    <h5>
     <a id="1__4">
     </a>
     1. 定义
    </h5>
    <p>
     块级元素在页面上以块的形式显示，通常会独占一行，其宽度默认为父容器的宽度。
    </p>
    <h5>
     <a id="2__7">
     </a>
     2. 特点
    </h5>
    <ul>
     <li>
      <strong>
       独占一行
      </strong>
      ：块级元素会在新行开始，并且其后的元素也会在新行开始。
     </li>
     <li>
      <strong>
       宽度和高度
      </strong>
      ：可以设置宽度（
      <code>
       width
      </code>
      ）和高度（
      <code>
       height
      </code>
      ）。
     </li>
     <li>
      <strong>
       内边距和外边距
      </strong>
      ：可以设置内边距（
      <code>
       padding
      </code>
      ）和外边距（
      <code>
       margin
      </code>
      ）。
     </li>
    </ul>
    <h5>
     <a id="3__12">
     </a>
     3. 常见的块级元素
    </h5>
    <ul>
     <li>
      <code>
       &lt;div&gt;
      </code>
      ：通用的块级容器，用于布局。
     </li>
     <li>
      <code>
       &lt;p&gt;
      </code>
      ：段落。
     </li>
     <li>
      <code>
       &lt;h1&gt;
      </code>
      到
      <code>
       &lt;h6&gt;
      </code>
      ：标题。
     </li>
     <li>
      <code>
       &lt;ul&gt;
      </code>
      、
      <code>
       &lt;ol&gt;
      </code>
      、
      <code>
       &lt;li&gt;
      </code>
      ：列表。
     </li>
     <li>
      <code>
       &lt;table&gt;
      </code>
      ：表格。
     </li>
     <li>
      <code>
       &lt;form&gt;
      </code>
      ：表单。
     </li>
    </ul>

    <h4>
     <a id="Inline_elements_28">
     </a>
     二、内联元素（Inline elements）
    </h4>
    <h5>
     <a id="1__30">
     </a>
     1. 定义
    </h5>
    <p>
     内联元素在页面上以行内形式显示，不会独占一行，其宽度由内容决定。
    </p>
    <h5>
     <a id="2__33">
     </a>
     2. 特点
    </h5>
    <ul>
     <li>
      <strong>
       行内显示
      </strong>
      ：内联元素不会独占一行，与前后元素在同一行显示。
     </li>
     <li>
      <strong>
       宽度和高度
      </strong>
      ：宽度由内容决定，不能直接设置宽度和高度。
     </li>
     <li>
      <strong>
       内边距和外边距
      </strong>
      ：可以设置垂直方向的内边距（
      <code>
       padding-top
      </code>
      、
      <code>
       padding-bottom
      </code>
      ）和外边距（
      <code>
       margin-top
      </code>
      、
      <code>
       margin-bottom
      </code>
      ），但水平方向的内边距和外边距可能不会按预期工作。
     </li>
    </ul>
    <h5>
     <a id="3__38">
     </a>
     3. 常见的内联元素
    </h5>
    <ul>
     <li>
      <code>
       &lt;span&gt;
      </code>
      ：通用的内联容器，用于样式控制。
     </li>
     <li>
      <code>
       &lt;a&gt;
      </code>
      ：超链接。
     </li>
     <li>
      <code>
       &lt;img&gt;
      </code>
      ：图像。
     </li>
     <li>
      <code>
       &lt;strong&gt;
      </code>
      ：加粗文本。
     </li>
     <li>
      <code>
       &lt;em&gt;
      </code>
      ：强调文本。
     </li>
     <li>
      <code>
       &lt;input&gt;
      </code>
      ：表单输入框。
     </li>
    </ul>

    <h4>
     <a id="_52">
     </a>
     三、块级元素与内联元素的转换
    </h4>
    <h5>
     <a id="1_CSS_54">
     </a>
     1. 使用CSS控制显示方式
    </h5>
    <p>
     可以通过CSS的
     <code>
      display
     </code>
     属性来改变元素的显示方式。
    </p>
    <ul>
     <li>
      <strong>
       <code>
        display: block;
       </code>
      </strong>
      ：将内联元素转换为块级元素。
     </li>
     <li>
      <strong>
       <code>
        display: inline;
       </code>
      </strong>
      ：将块级元素转换为内联元素。
     </li>
     <li>
      <strong>
       <code>
        display: inline-block;
       </code>
      </strong>
      ：将内联元素转换为块级元素，但允许与其他元素在同一行显示。
     </li>
    </ul>
    <h5>
     <a id="2__61">

    <h4>
     <a id="_67">
     </a>
     四、总结
    </h4>
    <ul>
     <li>
      <strong>
       块级元素
      </strong>
      ：独占一行，可以设置宽度和高度，适用于布局和需要独占空间的元素。
     </li>
     <li>
      <strong>
       内联元素
      </strong>
      ：与前后元素在同一行显示，宽度由内容决定，适用于文本和小部件。
     </li>
     <li>
      <strong>
       转换显示方式
      </strong>
      ：通过CSS的
      <code>
       display
      </code>
      属性可以灵活地改变元素的显示方式，实现更复杂的布局需求。
     </li>
    </ul>
