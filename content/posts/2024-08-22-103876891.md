    <h3>
     <a id="_4">
     </a>
     邮件搜索：
    </h3>
    <h4>
     <a id="1_5">
     </a>
     1、发件人搜索，一般在收件箱搜索
    </h4>
    <p>
     格式 ：from:“XXXX” 例如：from:“liba@qq.com”，如果是这种邮件格式，你可以直接from:“liba”
     <br/>
     XXXX可以是详细邮箱地址，也可以是人名，这和邮箱格式有关
    </p>
    <h4>
     <a id="2_8">
     </a>
     2、收件人搜做，一般在发件箱搜索
    </h4>
    <p>
     格式: to:“XXXX”
    </p>
    <h4>
     <a id="3_10">
     </a>
     3、按时间搜索
    </h4>
    <p>
     格式：received:today
     <br/>
     received:this week
     <br/>
     received:last month
     <br/>
     received:2020//1/1
     <br/>
     received:2020/1/1…2010/1/10 //时间段搜索
    </p>
    <h4>
     <a id="4_16">
     </a>
     4、邮件主题搜索
    </h4>
    <p>
     格式：subject:“XXXX” //来往邮件的标题进行搜索
    </p>
    <h4>
     <a id="5_18">
     </a>
     5、内容关键字搜索
    </h4>
    <p>
     格式：直接在搜索框里键入关键字就可以了
    </p>
    <h3>
     <a id="_20">
     </a>
     邮件内容搜索
    </h3>
    <p>
     这种一般当一个邮件来来回回很长的时候用到，直接按F4，弹出的如下搜索框，键入需要搜索的内容即可。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/984440c4f2c154e12fefcc339fbf2b52.png"/>
    </p>
    <h3>
     <a id="_23">
     </a>
     正则表达式
    </h3>
    <p>
     这个都支持，比较高级了，例如
     <br/>
     格式：from:“kai” and to:“li baobao”
     <br/>
     from:“kai” and subject:“1-7580589”
     <br/>
     received:2020/1/1…2010/1/10 //这其实也是一个正则表达式
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6eff410f76bc3306fcfe994024a6b275.png"/>
    </p>
    <h3>
     <a id="_29">
     </a>
     高级搜索
    </h3>
    <p>
     search&gt;&gt;search tools&gt;&gt;advance find
