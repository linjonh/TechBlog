    <h2>
     <a id="_5">
     </a>
     结构体
    </h2>
    <p>
     在
     <code>
      golang
     </code>
     语言中，结构体是一种用户自定义的数据类型，用于将多个不同类型的数据组合在一起。
    </p>
    <h3>
     <a id="_9">
     </a>
     定义
    </h3>
    <p>
     结构体通过关键字
     <code>
      struct
     </code>
     定义，包含一组成员变量。
     <strong>
      成员变量由名称和类型组成
     </strong>
     。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
    Name <span class="token builtin">string</span>
    Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>
</code></pre>
    <ul>
     <li>
      <strong>
       字段名称
      </strong>
      ：字段名称是可选的，如果省略字段名称，则表示该字段是匿名字段。
     </li>
     <li>
      <strong>
       字段类型
      </strong>
      ：字段类型可以是任意类型，包括基本类型、自定义类型、指针类型等。
     </li>
    </ul>
    <h3>
     <a id="_23">
     </a>
     初始化
    </h3>
    <p>
     结构体可以通过多种方式初始化：
     <strong>
      使用字段名初始化
     </strong>
     、
     <strong>
      按字段顺序初始化
     </strong>
     、
     <strong>
      使用
      <code>
       new
      </code>
      函数初始化
     </strong>
     。
    </p>
    <h4>
     <a id="_27">
     </a>
     使用字段名初始化
    </h4>
    <p>
     指定字段名进行初始化，字段顺序可以任意。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span>Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span> Age<span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// {ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     或者写成如下格式，但是需要注意的是，最后一个字段的结尾需要有一个逗号分隔。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span>
		Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span>
		Age<span class="token punctuation">:</span>  <span class="token number">18</span><span class="token punctuation">,</span>
	<span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// {ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     在
     <code>
      golang
     </code>
     中支持对个别字段进行初始化（未指定的字段则为零值），此时必须指定字段名称，否则会存在编译报错，如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span>
		Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span>
	<span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// {ycz 0}</span>

	<span class="token comment">// p1 := Person{"ycz"} // too few values in struct literal of type Person</span>
	<span class="token comment">// fmt.Println(p1)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_91">
     </a>
     按字段顺序初始化
    </h4>
    <p>
     如果不指定字段名，则需要按照结构体定义中字段的顺序进行初始化。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span><span class="token string">"ycz"</span><span class="token punctuation">,</span> <span class="token number">18</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// {ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_new__111">
     </a>
     使用
     <code>
      new
     </code>
     函数初始化
    </h4>
    <p>
     使用
     <code>
      new
     </code>
     函数会分配内存并返回一个指向结构体的指针，但不会初始化结构体的字段。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> <span class="token function">new</span><span class="token punctuation">(</span>Person<span class="token punctuation">)</span>
	p<span class="token punctuation">.</span>Name <span class="token operator">=</span> <span class="token string">"ycz"</span>
	p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">18</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// &amp;{ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     特别的，除了可以使用
     <code>
      new
     </code>
     函数获取指向结构体的指针，也可以结合
     <strong>
      使用字段名初始化
     </strong>
     或者
     <strong>
      按字段顺序初始化
     </strong>
     这两种方式获取指向结构体的指针。如下所示：
    </p>
    <ul>
     <li>
      使用字段名初始化获取指向结构体的指针
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> <span class="token operator">&amp;</span>Person<span class="token punctuation">{<!-- --></span>Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span> Age<span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// &amp;{ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <ul>
     <li>
      按字段顺序初始化获取指向结构体的指针
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> <span class="token operator">&amp;</span>Person<span class="token punctuation">{<!-- --></span><span class="token string">"ycz"</span><span class="token punctuation">,</span> <span class="token number">18</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// &amp;{ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     这两种方式本质上是先在栈上初始化一个结构体对象（注意：不包含内存逃逸的情况），然后通过
     <code>
      &amp;
     </code>
     取地址操作将这个结构体的地址赋值给指针变量。这种方式的核心是
     <strong>
      先创建结构体实例，再取地址
     </strong>
     。而
     <code>
      new
     </code>
     会为
     <code>
      Person
     </code>
     结构体直接在堆空间分配内存，并将所有字段初始化为零值，然后直接返回一个指向这块内存的指针。
    </p>
    <h4>
     <a id="_173">
     </a>
     结构体零值
    </h4>
    <p>
     结构体的零值是其所有字段的零值的组合。其中每个字段的零值取决于该字段的类型。假设有一个结构体定义如下：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
    Name    <span class="token builtin">string</span>
    Age     <span class="token builtin">int</span>
    Married <span class="token builtin">bool</span>
    Address <span class="token operator">*</span><span class="token builtin">string</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     创建一个
     <code>
      Person
     </code>
     对象，但是不进行字段的初始化，查看各个字段的零值：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name    <span class="token builtin">string</span>
	Age     <span class="token builtin">int</span>
	Married <span class="token builtin">bool</span>
	Address <span class="token operator">*</span><span class="token builtin">string</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"%+v\n"</span><span class="token punctuation">,</span> p<span class="token punctuation">)</span> <span class="token comment">// {Name: Age:0 Married:false Address:&lt;nil&gt;}</span>
<span class="token punctuation">}</span>
</code></pre>
    <ul>
     <li>
      <code>
       Name
      </code>
      是
      <code>
       string
      </code>
      类型，零值是空字符串
      <code>
       ""
      </code>
      。
     </li>
     <li>
      <code>
       Age
      </code>
      是
      <code>
       int
      </code>
      类型，零值是
      <code>
       0
      </code>
      。
     </li>
     <li>
      <code>
       Married
      </code>
      是
      <code>
       bool
      </code>
      类型，零值是
      <code>
       false
      </code>
      。
     </li>
     <li>
      <code>
       Address
      </code>
      是指针类型，零值是
      <code>
       nil
      </code>
      。
     </li>
    </ul>
    <blockquote>
     <p>
      其他类型的零值这里不再赘述，感兴趣的同学可以查看文章《
      <code>
       Golang
      </code>
      变量和常量》。
     </p>
    </blockquote>
    <h3>
     <a id="_213">
     </a>
     访问结构体
    </h3>
    <h4>
     <a id="_215">
     </a>
     可访问性
    </h4>
    <p>
     结构体字段的可访问性取决于字段名的首字母。
    </p>
    <ul>
     <li>
      <strong>
       大写字母开头
      </strong>
      ：字段是导出的（public），可以在包外访问。
     </li>
     <li>
      <strong>
       小写字母开头
      </strong>
      ：字段是非导出的（private），只能在定义它的包内访问。
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
    Name <span class="token builtin">string</span> <span class="token comment">// 可以在包外访问</span>
    age  <span class="token builtin">int</span>    <span class="token comment">// 只能在包内访问</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_229">
     </a>
     访问方式
    </h4>
    <p>
     结构体的字段可以通过点号（
     <code>
      .
     </code>
     ）操作符访问。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span><span class="token string">"ycz"</span><span class="token punctuation">,</span> <span class="token number">18</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Name<span class="token punctuation">)</span> <span class="token comment">// ycz</span>
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span>  <span class="token comment">// 18</span>
    
    p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">20</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span>  <span class="token comment">// 20</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     如果结构体变量是通过指针初始化的，可以直接通过指针访问字段，无需解引用。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> <span class="token function">new</span><span class="token punctuation">(</span>Person<span class="token punctuation">)</span>
	p<span class="token punctuation">.</span>Name <span class="token operator">=</span> <span class="token string">"ycz"</span>
	p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">18</span>
	
    fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Name<span class="token punctuation">)</span> <span class="token comment">// ycz</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span>  <span class="token comment">// 18</span>
	
    p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">20</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span> <span class="token comment">// 20</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_278">
     </a>
     结构体方法
    </h3>
    <p>
     结构体方法是一种将函数与结构体类型关联的方式。通过定义方法，可以为结构体类型添加行为，使得结构体不仅能够存储数据，还能执行操作。方法的定义与普通函数类似，但需要在函数名之前指定一个接收者。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">func</span> <span class="token punctuation">(</span>接收者<span class="token punctuation">)</span> 方法名<span class="token punctuation">(</span>参数列表<span class="token punctuation">)</span> 返回值列表 <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// 方法体</span>
<span class="token punctuation">}</span>
</code></pre>
    <ul>
     <li>
      <strong>
       接收者
      </strong>
      ：可以是结构体类型（
      <code>
       T
      </code>
      ）或结构体指针（
      <code>
       *T
      </code>
      ）。
     </li>
     <li>
      <strong>
       方法名
      </strong>
      ：方法的名称，遵循
      <code>
       golang
      </code>
      的命名规则。
     </li>
     <li>
      <strong>
       参数列表
      </strong>
      ：方法的输入参数。
     </li>
     <li>
      <strong>
       返回值列表
      </strong>
      ：方法的返回值。
     </li>
    </ul>
    <blockquote>
     <p>
      关于函数的更多知识点可以参考文章《
      <code>
       Golang
      </code>
      函数》。
     </p>
    </blockquote>
    <h4>
     <a id="_295">
     </a>
     方法的调用
    </h4>
    <h5>
     <a id="_297">
     </a>
     可访问性
    </h5>
    <p>
     方法的可访问性也同样取决于方法名称的大小写规则，这与字段的可访问性类似。
    </p>
    <ul>
     <li>
      <strong>
       大写字母开头
      </strong>
      ：方法是导出的（public），可以在包外访问。
     </li>
     <li>
      <strong>
       小写字母开头
      </strong>
      ：方法是非导出的（private），只能在定义它的包内访问。
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
    Name <span class="token builtin">string</span>
    Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token comment">// getName 是一个非导出的方法，只能在包内访问</span>
<span class="token keyword">func</span> <span class="token punctuation">(</span>p Person<span class="token punctuation">)</span> <span class="token function">getName</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token builtin">int</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> p<span class="token punctuation">.</span>Name
<span class="token punctuation">}</span>

<span class="token comment">// GetAge 是一个导出的方法，可以在包外访问</span>
<span class="token keyword">func</span> <span class="token punctuation">(</span>p Person<span class="token punctuation">)</span> <span class="token function">GetAge</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token builtin">int</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> p<span class="token punctuation">.</span>Age
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="_321">
     </a>
     访问方式
    </h5>
    <p>
     方法的调用方式与字段访问类似，也是通过点号（
     <code>

     </code>
     ）操作符访问，如下所示：

    <h4>
     <a id="_344">
     </a>
     方法接收者
    </h4>
    <p>
     方法的接收者决定了方法如何与结构体实例交互。
     <strong>
      接收者可以是值类型或指针类型
     </strong>
     。
    </p>
    <h5>
     <a id="_348">
     </a>
     值接收者
    </h5>
    <p>
     当方法的接收者是值类型（
     <code>
      func (p Person) ...
     </code>
     ）时，调用方法时会传递结构体的一个副本。
    </p>
    <ul>
     <li>
      <strong>
       传递副本
      </strong>
      ：调用方法时，结构体的值会被复制一份传递给方法。
     </li>
     <li>
      <strong>
       不可修改原始结构体
      </strong>
      ：在方法内部对结构体的修改不会影响原始结构体，因为修改的是副本。
     </li>
    </ul>
    <p>
     如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Person <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Name <span class="token builtin">string</span>
	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token punctuation">(</span>p Person<span class="token punctuation">)</span> <span class="token function">SayHello</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"Hello, my name is %s and I'm %d years old.\n"</span><span class="token punctuation">,</span> p<span class="token punctuation">.</span>Name<span class="token punctuation">,</span> p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span>
	p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">20</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span> <span class="token comment">// 20</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> Person<span class="token punctuation">{<!-- --></span>Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span> Age<span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
	p<span class="token punctuation">.</span><span class="token function">SayHello</span><span class="token punctuation">(</span><span class="token punctuation">)</span>       <span class="token comment">// Hello, my name is ycz and I'm 18 years old.</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span> <span class="token comment">// 18</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="_380">
     </a>
     指针接收者
    </h5>
    <p>
     当方法的接收者是指针类型（
     <code>
      func (p *Person) ...
     </code>
     ）时，调用方法时会传递结构体的地址。
    </p>
    <ul>
     <li>
      <strong>
       传递地址
      </strong>
      ：调用方法时，传递的是结构体的指针，而不是副本。
     </li>
     <li>
      <strong>
       可以修改原始结构体
      </strong>
      ：在方法内部对结构体的修改会直接影响原始结构体。
     </li>
    </ul>
    <p>
     如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

	Age  <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token punctuation">(</span>p <span class="token operator">*</span>Person<span class="token punctuation">)</span> <span class="token function">SayHello</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"Hello, my name is %s and I'm %d years old.\n"</span><span class="token punctuation">,</span> p<span class="token punctuation">.</span>Name<span class="token punctuation">,</span> p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span>
	p<span class="token punctuation">.</span>Age <span class="token operator">=</span> <span class="token number">20</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span> <span class="token comment">// 20</span>
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p <span class="token operator">:=</span> <span class="token operator">&amp;</span>Person<span class="token punctuation">{<!-- --></span>Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span> Age<span class="token punctuation">:</span> <span class="token number">18</span><span class="token punctuation">}</span>
	p<span class="token punctuation">.</span><span class="token function">SayHello</span><span class="token punctuation">(</span><span class="token punctuation">)</span>       <span class="token comment">// Hello, my name is ycz and I'm 18 years old.</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">.</span>Age<span class="token punctuation">)</span> <span class="token comment">// 20</span>

    <p>
     这里需要特别强调的一点是：
     <strong>
      如果结构体被多个协程并发访问，使用指针接收者时需要特别小心，因为指针方法可能会修改共享状态，从而导致竞争问题。在这种情况下，可能需要引入锁或其他同步机制
     </strong>
     。
    </p>

    <h3>
     <a id="_468">
     </a>
     结构体的比较
    </h3>
    <p>
     结构体的比较主要用于判断两个结构体实例是否相等，使用过程中需要遵循以下基本规则：
    </p>
    <ul>
     <li>
      <strong>
       字段逐个比较
      </strong>
      ：结构体的比较是通过逐个比较其字段的值来完成的。如果所有字段的值都相等，则两个结构体相等。
     </li>
     <li>
      <strong>
       字段顺序无关
      </strong>
      ：字段的顺序不影响比较结果，只要字段的值相同即可。
     </li>
     <li>
      <strong>
       字段类型必须可比较
      </strong>
      ：结构体的所有字段类型都必须是可比较的，否则结构体不能进行比较。
     </li>
    </ul>
    <p>
     需要注意的是，并非所有类型都可以进行比较。结构体的字段类型必须满足以下条件，才能进行比较：
    </p>
    <ul>
     <li>
      <strong>
       基本类型
      </strong>
      ：
      <code>
       int
      </code>
      、
      <code>
       float64
      </code>
      、
      <code>
       string
      </code>
      、
      <code>
       bool
      </code>
      等基本类型是可比较的。
     </li>
     <li>
      <strong>
       复合类型
      </strong>
      ：
      <ul>
       <li>
        <strong>
         指针
        </strong>
        ：指针类型是可比较的，比较的是指针的地址。
       </li>
       <li>
        <strong>
         数组
        </strong>
        ：固定长度的数组是可比较的，比较的是数组中每个元素的值。
       </li>
       <li>
        <strong>
         结构体
        </strong>
        ：结构体是可比较的，比较的是结构体中每个字段的值。
       </li>
       <li>
        <strong>
         切片、
         <code>
          map
         </code>
         、通道
        </strong>
        ：这些类型是不可比较的，因此不能作为结构体字段参与比较，如果尝试比较会导致编译错误。
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">type</span> Point <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	X<span class="token punctuation">,</span> Y <span class="token builtin">int</span>
<span class="token punctuation">}</span>

<span class="token keyword">type</span> Line <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Start<span class="token punctuation">,</span> End Point
<span class="token punctuation">}</span>

<span class="token keyword">type</span> LineP <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
	Start<span class="token punctuation">,</span> End <span class="token operator">*</span>Point
<span class="token punctuation">}</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	p1 <span class="token operator">:=</span> Point<span class="token punctuation">{<!-- --></span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">}</span>
	p2 <span class="token operator">:=</span> Point<span class="token punctuation">{<!-- --></span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p1 <span class="token operator">==</span> p2<span class="token punctuation">)</span> <span class="token comment">// true</span>

	l1 <span class="token operator">:=</span> Line<span class="token punctuation">{<!-- --></span>Start<span class="token punctuation">:</span> p1<span class="token punctuation">,</span> End<span class="token punctuation">:</span> p2<span class="token punctuation">}</span>
	l2 <span class="token operator">:=</span> Line<span class="token punctuation">{<!-- --></span>Start<span class="token punctuation">:</span> p2<span class="token punctuation">,</span> End<span class="token punctuation">:</span> p1<span class="token punctuation">}</span>
	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>l1 <span class="token operator">==</span> l2<span class="token punctuation">)</span> <span class="token comment">// true</span>

	lp1 <span class="token operator">:=</span> LineP<span class="token punctuation">{<!-- --></span><span class="token operator">&amp;</span>p1<span class="token punctuation">,</span> <span class="token operator">&amp;</span>p2<span class="token punctuation">}</span>

     </strong>
     。
    </p>
    <h3>
     <a id="_559">
     </a>
     匿名结构体
    </h3>
    <p>
     匿名结构体在
     <code>
      golang
     </code>
     中是一种特殊的结构体类型，它没有显式的类型定义，而是直接定义并使用。而且，它的定义方式与普通结构体类似，但没有类型名称。可以直接在代码中定义并使用。如下所示：
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> main

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token keyword">func</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

	<span class="token comment">// 定义并初始化一个匿名结构体</span>
	p <span class="token operator">:=</span> <span class="token keyword">struct</span> <span class="token punctuation">{<!-- --></span>
		Name <span class="token builtin">string</span>
		Age  <span class="token builtin">int</span>
	<span class="token punctuation">}</span><span class="token punctuation">{<!-- --></span>
		Name<span class="token punctuation">:</span> <span class="token string">"ycz"</span><span class="token punctuation">,</span>
		Age<span class="token punctuation">:</span>  <span class="token number">18</span><span class="token punctuation">,</span>
	<span class="token punctuation">}</span>

	fmt<span class="token punctuation">.</span><span class="token function">Println</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token comment">// {ycz 18}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     由于匿名函数使用非常的灵活，但是在实际使用过程中需要注意一些问题：
    </p>
    <ul>
     <li>
      匿名结构体的类型是唯一的，即使两个匿名结构体的字段完全相同，它们的类型也不同。
     </li>
     <li>
      匿名结构体的字段如果过多或嵌套过深，可能会降低代码的可读性。
     </li>
     <li>
      在初始化时，匿名结构体必须显式指定字段名（除非所有字段都按顺序初始化）。如果字段较多，初始化代码可能会显得冗长。
     </li>
     <li>
      匿名结构体没有类型名称，因此无法在其他地方复用。如果需要复用结构体类型，必须使用普通结构体。
     </li>
     <li>
      在测试和调试时会带来一些不便，因为它们的类型没有名称，调试信息可能不够直观。
     </li>
    </ul>
    <h3>
     <a id="_591">
     </a>

    </h3>
    <p>

<span class="token punctuation">}</span>

     <code>

     <code>
      Person
     </code>

    <h3>
     <a id="_733">
     </a>
     序列化与反序列化
    </h3>
    <p>
     序列化是指将数据结构或对象的状态转换为可存储或可传输的格式的过程。序列化后的数据可以保存到文件、数据库或通过网络传输。
     <strong>
      序列化的目标是将复杂的数据结构转换为一种线性的、可读或可传输的格式
     </strong>
     。
    </p>
    <p>
     反序列化是序列化的逆过程，即将序列化后的数据还原为原始的数据结构或对象。
     <strong>
      反序列化的目标是从存储或传输的格式中恢复数据的原始状态
     </strong>
     。
    </p>
    <h4>
     <a id="_739">
     </a>
     常见的序列化格式
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        JSON
       </code>
      </strong>
      ：轻量级的文本格式，易于阅读和解析，广泛用于 Web 开发和跨语言通信。
     </li>
     <li>
      <strong>
       <code>
        XML
       </code>
      </strong>
      ：标记语言，适合复杂的数据结构，但比
      <code>
       JSON
      </code>
      更冗长。
     </li>
     <li>
      <strong>
       <code>
        Protocol Buffers
       </code>
      </strong>
      ：
      <code>
       Google
      </code>
      开发的二进制格式，高效且跨语言支持。
     </li>
     <li>
      <strong>
       <code>
        gob
       </code>
      </strong>
      ：
      <code>
       golang
      </code>
      语言原生的二进制格式，仅适用于
      <code>
       golang
      </code>
      语言。
     </li>

    <h4>
     <a id="_750">
     </a>
     字段标签
    </h4>
    <p>
     字段标签是一种附加到结构体字段上的元数据，用于为字段提供额外的信息。字段标签在序列化、反序列化、反射等场景中非常有用，可以帮助库或框架正确地处理结构体字段。如果一个字段存在多个标签，则中间采用空格分隔。
    </p>
    <p>
     字段标签的语法如下：
    </p>
    <pre><code class="prism language-go">FieldName FieldType <span class="token string">`tagKey:"tagValue" anotherKey:"anotherValue"`</span>
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        FieldName
       </code>
      </strong>
      ：字段名称。
     </li>
     <li>
      <strong>
       <code>
        FieldType
       </code>
      </strong>
      ：字段类型。
     </li>
     <li>
      <strong>
       <code>
        tagKey
       </code>
      </strong>
      ：标签的键。
     </li>
     <li>
      <strong>
       <code>
        tagValue
       </code>
      </strong>
      ：标签的值。
     </li>
    </ul>
