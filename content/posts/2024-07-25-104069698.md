    <h2>
     <strong>
      概述
     </strong>
    </h2>
    <p>
     <strong>
      一、为什么要学习服务器端开发
     </strong>
    </p>
    <ul>
     <li>
      能够和后端程序员开发更加紧密的配合
     </li>
     <li>
      网站业务逻辑前置，学习前端技术需要后端技术的支持
     </li>
     <li>
      拓展知识视野，能站在更高的角度审视整个项目
     </li>
    </ul>
    <p>
     <strong>
      二、服务器端开发需要做什么事情呢
     </strong>
    </p>
    <ul>
     <li>
      实现网站页面逻辑
     </li>
     <li>
      数据的增删改查
     </li>
    </ul>
    <p>
     <strong>
      三、为什么选择Node
     </strong>
    </p>
    <ul>
     <li>
      使用js语法开发后端应用
     </li>
     <li>
      生态系统活跃，有大量开发库可以使用
     </li>
     <li>
      前端开发工具多是基于Node开发
     </li>
    </ul>
    <p>
     <strong>
      四、Node是什么
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/c45d329b51f3cfaa4c81dfe46ea77411.jpeg"/>
    </p>
    <ul>
     <li>
      简单的说 Node.js 就是运行在服务端的 JavaScript
     </li>
     <li>
      Node.js 是一个基于Chrome JavaScript 运行时建立的一个平台
     </li>
     <li>
      <p>
       Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好
      </p>
     </li>
    </ul>
    <p>
     <strong>
      五、运行环境
     </strong>
    </p>
    <ul>
     <li style="margin-left:.19in;">
      <span style="color:#262626;">
       浏览器（软件）能够运行JavaScript代码，浏览器就是JavaScript代码的运行环境
      </span>
     </li>
     <li style="margin-left:.19in;">
      <span style="color:#262626;">
       Node（软件）能够运行JavaScript代码，Node就是JavaScript代码的运行环境
      </span>
     </li>
    </ul>
