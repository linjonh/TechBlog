    <h2>
     <a id="CSS1_0">
     </a>
     CSS整理学习合集（1）
    </h2>
    <p>
     既然要学习css，那么首先就要先了解css是什么
    </p>
    <h4>
     <a id="CSS_4">
     </a>
     <strong>
      一、CSS简介
     </strong>
    </h4>
    <h5>
     <a id="1CSS_6">
     </a>
     1.什么是CSS?
    </h5>
    <p>
     CSS(Cascading Style Sheet):翻译过来就是层叠样式表,是一组样式设置的规则，用于控制页面的外观样式
    </p>
    <h5>
     <a id="2CSS_10">
     </a>
     2.为什么使用CSS
    </h5>
    <p>
     1.实现内容与样式的分离，便于团队开发
    </p>
    <p>
     2.样式复用，便于网站的后期维护
    </p>
    <p>
     3.页面的精确控制，让页面更精美
    </p>
    <h5>
     <a id="3CSS_18">
     </a>
     3.CSS作用
    </h5>
    <p>
     1.让页面外观美化
    </p>
    <p>
     2.对页面进行布局和元素定位
    </p>
    <h4>
     <a id="CSS_24">
     </a>
     二、CSS的基本用法
    </h4>
    <h5>
     <a id="1CSS_26">
     </a>
     1.CSS的基本语法
    </h5>
    <pre><code>选择器{
			属性名：属性值;
			属性名：属性值;
		}
</code></pre>
    <ul>
     <li>
      选择器：要修饰的对象（标签元素）
     </li>
     <li>
      属性名：修饰对象的哪一个属性（样式）
     </li>
     <li>
      属性值：样式的取值
     </li>
    </ul>
    <p>
     比如：
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;Document&lt;/title&gt;
	&lt;style&gt;
		p{
			color:green;
			background: #cccccc;
		}
		h2{
			color:red;
		}
	&lt;/style&gt;
&lt;/head
&lt;body&gt;
	&lt;p style="color:red"&gt;学会CSS&lt;/p&gt;
	&lt;h2&gt;快要学会啦！11&lt;/h2&gt;
&lt;/body&gt;
&lt;/html&gt;

</code></pre>
    <h5>
     <a id="2CSS_67">
     </a>
     2.CSS引用方式
    </h5>
    <p>
     内部样式、行内样式、外部样式
    </p>
    <h6>
     <a id="21__71">
     </a>
     2.1 内部样式
    </h6>
    <p>
     也称为内嵌样式，在页面头部通过style标签定义
    </p>
    <p>
     对当前页面中所有符合样式选择器的标签都起作用
    </p>
    <p>
     像这样
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;Document&lt;/title&gt;
	&lt;style&gt;
		p{
			color:green;
			background: #cccccc;
		}
		h2{
			color:red;
		}
	&lt;/style&gt;
&lt;/head&gt;
</code></pre>
    <h6>
     <a id="22__99">
     </a>
     2.2 行内样式
    </h6>
    <p>
     也称为嵌入样式，使用HTML标签的style属性定义，只对设置style属性的标签起作用
    </p>
    <pre><code>&lt;p style="color:red"&gt;学会CSS&lt;/p&gt;
</code></pre>
    <h6>
     <a id="23__109">
     </a>
     2.3 外部样式
    </h6>
    <p>
     使用单独的 .CSS 文件定义，然后在页面中使用 link标签 或 @import指令 引入
    </p>
    <ul>
     <li>
      <p>
       使用 link标签 链接外部样式文件
      </p>
      <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
  &lt;head&gt;
    &lt;meta charset="UTF-8" /&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0" /&gt;
    &lt;link rel="stylesheet" href="./css/style.css" /&gt;
    &lt;title&gt;Document&lt;/title&gt;
  &lt;/head&gt;
  &lt;body&gt;&lt;/body&gt;
&lt;/html&gt;
</code></pre>
     </li>
     <li>
      <p>
       @import 指令 导入外部样式文件
      </p>
     </li>
    </ul>
    <pre><code>&lt;style&gt;
	@import "CSS样式文件路径";
	@import url(CSS样式文件路径);
&lt;/style&gt;
</code></pre>
    <h4>
     <a id="_139">
     </a>
     三、选择器
    </h4>
    <h5>
     <a id="1_141">
     </a>
     1、基础选择器
    </h5>
    <h6>
     <a id="11__143">
     </a>
     1.1 标签选择器
    </h6>
    <p>
     也叫元素选择器，使用HTML标签作为选择器的名称以标签名作为样式应用的依据
    </p>
    <h6>
     <a id="12__147">
     </a>
     1.2 类选择器
    </h6>
    <p>
     使用自定义的名称，以 . 号作为前缀，然后再通过HTML标签的class属性调用类选择器,以标签的class属性作为样式应用的依据
    </p>
    <p>
     <strong>
      注意事项
     </strong>
     ：
    </p>
    <p>
     调用时不能添加 . 号
     <br/>
     同时调用多个类选择器时，以 空格 分隔
     <br/>
     类选择器名称不能以 数字 开头
    </p>
    <h6>
     <a id="13_ID_157">
     </a>
     1.3 ID选择器
    </h6>
    <p>
     使用自定义名称，以 # 作为前缀，然后通过HTML标签的id属性进行名称匹配
    </p>
    <p>
     以标签的id属性作为样式应用的依据，一对一的关系
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;Document&lt;/title&gt;
	&lt;style&gt;
		p{   //标签选择器
			color:red;
			font-size:20px;  
		}
		h2{  //标签选择器
			color:yellow;
		}
		.hello{ //类选择器
			background: #cccccc;
		}
		#haha{ //ID选择器
			color:blue;
		}
	&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
	&lt;p&gt;welcome to css!&lt;/p&gt;
	&lt;h2&gt;WEB前端&lt;/h2&gt;
	&lt;hr&gt;
	&lt;p class="hello"&gt;welcome to css!&lt;/p&gt;
	&lt;h2&gt;WEB前端&lt;/h2&gt;
	&lt;hr&gt;
	&lt;h1 id="haha"&gt;哈哈&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <h5>
     <a id="2_197">
     </a>
     2、复杂选择器
    </h5>
    <h6>
     <a id="21__199">
     </a>
     2.1 复合选择器
    </h6>
    <p>
     要求：标签选择器和类选择器、标签选择器和ID选择器，一起使用
    </p>
    <p>
     必须同时满足两个条件才能应用样式
    </p>
    <h6>
     <a id="22_205">
     </a>
     2.2组合选择器
    </h6>
    <p>
     也称为集体声明
    </p>
    <p>
     将多个具有相同样式的选择器放在一起声明，使用逗号隔开
    </p>
    <h6>
     <a id="23__211">
     </a>
     2.3 嵌套选择器
    </h6>
    <p>
     在某个选择器内部再设置选择器，通过空格隔开
    </p>
    <p>
     只有满足层次关系最里层的选择器所对应的标签才会应用样式
    </p>
    <p>
     <strong>
      注意
     </strong>
     ：使用 空格 时不区分父子还是后代，使用CSS3中新增的
     <strong>
      &gt;
     </strong>
     时必须是父子关系才行
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;Document&lt;/title&gt;
	&lt;style&gt;
		/* 标签选择器和类选择器合起来使用----复合选择器 */
		h1.aaa{
			color:red;
		}
		/* 标签选择器和ID选择器合起来使用----复合选择器 */
		p#bbb{
			color:blue;
		}
		/* 2.组合选择器 */
		h1,p,div,span,.ccc{
			font-size:30px;
		}
		div{
			background:violet;
		}
		.ccc{
			font-weight:bold;
		}
		/* 3.嵌套选择器 */
		/* div p{
			color:green;
			text-decoration:underline;
		} */
		div&gt;p{
			color:green;
			text-decoration:underline;
		}
		div h3.ddd{
			color:red;
		}
	&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
	&lt;!-- 需求：只想修饰class属性为aaa的h1标签 --&gt;
	&lt;h1 class="aaa"&gt;标签选择器和类选择器合起来使用----复合选择器&lt;/h1&gt;
	&lt;h4 class="aaa"&gt;css&lt;/h4&gt;
	&lt;h1&gt;hello&lt;/h1&gt;
	&lt;hr&gt;
	&lt;!-- 我要修饰ID属性为bbb的p标签 --&gt;
	&lt;p id="bbb"&gt;标签选择器和ID选择器合起来使用----复合选择器&lt;/p &gt;
	&lt;p&gt;html&lt;/p&gt;
	&lt;h1 id="bbb"&gt;不修饰h1&lt;/h1&gt;
	&lt;hr&gt;
	&lt;!-- 给h1、p、div、span标签中的内容设置字号为30px --&gt;
	&lt;h1&gt;hello&lt;/h1&gt;
	&lt;p&gt;CSS&lt;/p&gt;
	&lt;div&gt;WEB开发&lt;/div&gt;
	&lt;span class="ccc"&gt;JAVA开发&lt;/span&gt;
	&lt;hr&gt;
	&lt;!-- 需求：修饰div内部的p标签 --&gt;
	&lt;div&gt;
		&lt;p&gt;div内部的p标签&lt;/p&gt;
		&lt;h3&gt;div内部的h3标签&lt;/h3&gt;
	&lt;/div&gt;
	&lt;hr&gt;
	&lt;div&gt;
		&lt;h3&gt;
			&lt;p&gt;div内部的h3内部的p标签&lt;/p&gt;
		&lt;/h3&gt;
	&lt;/div&gt;
	&lt;hr&gt;
	&lt;!-- 需求：修饰div内部的class为ddd的标签 --&gt;
	&lt;div&gt;
		&lt;p&gt;div内部的p&lt;/p&gt;
		&lt;h3 class="ddd"&gt;div内部的h3&lt;/h3&gt;
		&lt;p class="ddd"&gt;PPPP&lt;/p&gt;
	&lt;/div&gt;
	&lt;h3 class="ddd"&gt;h3h3h3&lt;/h3&gt;
&lt;/body&gt;
&lt;/html&gt;

</code></pre>
    <h6>
     <a id="24__299">
     </a>
     2.4 伪类选择器
    </h6>
    <p>
     根据不同的状态显示不同的样式，一般多用于 标签
    </p>
    <p>
     四种状态：
    </p>
    <ul>
     <li>
      :link 未访问的链接
     </li>
     <li>
      :visited 已访问的链接
     </li>
     <li>
      :hover 鼠标悬浮到连接上，即移动在连接上
     </li>
     <li>
      :active 选定的链接，被激活
     </li>
    </ul>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;伪类选择器&lt;/title&gt;
	&lt;style&gt;
		a:link,a:visited{
			color:#666666;
			font-size: 13px;
			text-decoration: none;
		}
		a:hover,a:active{
			color:#ff7300;
			text-decoration: underline;
		}
		/*普通的标签也可以使用伪类选择器*/
		p:hover{
			color:red;
		}
		p:active{
			color:blue;
		}
	&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
	&lt;a href="www.baidu.com"&gt;百度搜索CSS&lt;/a&gt;
	&lt;p&gt;CSS从入门到精通！&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;

</code></pre>
    <h6>
     <a id="25__343">
     </a>
     2.5 伪元素选择器
    </h6>
    <ul>
     <li>
      :first-letter 为第一个字符的样式
     </li>
     <li>
      :first-line 为第一行添加样式
     </li>
     <li>
      :before 在元素内容的最前面添加的内容，需要配合content属性使用
     </li>
     <li>
      :after 在元素内容的最后面添加的内容，需要配合content属性使用
     </li>
    </ul>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
	&lt;meta charset="UTF-8"&gt;
	&lt;title&gt;Document&lt;/title&gt;
	&lt;style&gt;
		p:first-letter{
			color:red;
			font-size:30px;
		}
		p:first-line{
			background:pink;
		}
		p:before{
			content:"嘿嘿";
		}
		p:after{
			content:"哈哈";
		}
	&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
	&lt;p&gt;hello world!&lt;/p&gt;
	&lt;hr&gt;
	&lt;p&gt;
		hello world! &lt;br&gt;
		welcome to css!
	&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;

</code></pre>

    <h5>
     <a id="3_389">
     </a>
     3、选择器优先级
    </h5>
    <h6>
     <a id="31__391">
     </a>
     3.1 优先级
    </h6>
    <p>
     行内样式&gt;ID选择器&gt;类选择器&gt;标签选择器
    </p>
    <p>
     原因：首先加载标签选择器,再加载类选择器，然后加载ID选择器，最后加载行内样式
    </p>
    <p>
     后加载会覆盖先加载的同名样式
    </p>
    <h6>
     <a id="32__399">
     </a>
     3.2 内外部样式加载顺
    </h6>
    <p>
     就近原则
    </p>
    <p>
     原因：按照书写顺序依次加载，在同优先级的前提下，后加载的会覆盖先加载的同名样式，所以离的越近
    </p>
    <p>
     越优先
    </p>
    <h6>
     <a id="33_important_407">
     </a>
     3.3 !important
    </h6>
    <p>
     可以使用!important使某个样式有最高的优先级
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
