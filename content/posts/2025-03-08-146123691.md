    <h3>
     <a id="Samba_2">
     </a>
     方案一：使用Samba共享服务
    </h3>
    <p>
     Samba 是在 Linux 和 Unix 系统上实现 SMB/CIFS 协议的一个免费软件，可让 Windows 系统方便地访问 Linux 共享资源。
    </p>
    <h4>
     <a id="Ubuntu_18__5">
     </a>
     Ubuntu 18 端配置
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        安装 Samba 服务
       </strong>
       <br/>
       打开终端，输入以下命令进行安装：
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> samba
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        创建共享目录
       </strong>
       <br/>
       例如，创建一个名为
       <code>
        shared
       </code>
       的共享目录：
      </p>
      <pre><code class="prism language-bash"><span class="token function">mkdir</span> /home/user/shared
</code></pre>
      <p>
       这里的
       <code>
        /home/user
       </code>
       请替换为你实际的用户主目录。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置 Samba
       </strong>
       <br/>
       使用文本编辑器打开 Samba 配置文件：
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">nano</span> /etc/samba/smb.conf
</code></pre>
      <p>
       在文件末尾添加以下内容：
      </p>
      <pre><code class="prism language-plaintext">[shared]
   comment = Shared Folder
   path = /home/user/shared
   read only = no
   browsable = yes
   guest ok = yes
</code></pre>
      <ul>
       <li>
        <code>
         [shared]
        </code>
        ：共享名称，可自定义。
       </li>
       <li>
        <code>
         path
        </code>
        ：共享目录的实际路径。
       </li>
       <li>
        <code>
         read only
        </code>
        ：是否只读，
        <code>
         no
        </code>
        表示可读写。
       </li>
       <li>
        <code>
         browsable
        </code>
        ：是否可浏览。
       </li>
       <li>
        <code>
         guest ok
        </code>
        ：是否允许访客访问。
       </li>
      </ul>
      <p>
       按
       <code>
        Ctrl + X
       </code>
       ，然后按
       <code>
        Y
       </code>
       保存并退出。
      </p>
     </li>
     <li>
      <p>
       <strong>
        重启 Samba 服务
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl restart smbd
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        设置共享目录权限
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">777</span> /home/user/shared
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="Windows_10__48">
     </a>
     Windows 10 端访问
    </h4>
    <ol>
     <li>
      打开“此电脑”，在左侧导航栏中右键点击“网络”，选择“映射网络驱动器”。
     </li>
     <li>
      在“文件夹”输入框中，输入 Ubuntu 电脑的 IP 地址和共享名称，格式为
      <code>
       \\Ubuntu_IP\shared
      </code>
      ，然后点击“完成”。
     </li>
     <li>
      若设置了访问权限，可能需要输入用户名和密码，输入正确信息后即可访问共享目录。
     </li>
    </ol>
    <h3>
     <a id="_SSHFS_Windows__SSH__53">
     </a>
     方案二：使用 SSHFS（需在 Windows 上安装 SSH 客户端）
    </h3>
    <p>
     SSHFS 允许你通过 SSH 协议挂载远程文件系统。
    </p>
    <h4>
     <a id="Ubuntu_18__56">
     </a>
     Ubuntu 18 端配置
    </h4>
    <ol>
     <li>
      确保 SSH 服务已安装并运行：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> openssh-server
<span class="token function">sudo</span> systemctl start <span class="token function">ssh</span>
<span class="token function">sudo</span> systemctl <span class="token builtin class-name">enable</span> <span class="token function">ssh</span>
</code></pre>
     </li>
     <li>
      记录 Ubuntu 电脑的 IP 地址：
      <pre><code class="prism language-bash"><span class="token function">ifconfig</span>
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="Windows_10__69">
     </a>
     Windows 10 端配置
    </h4>
    <ol>
     <li>
      安装 WinFsp 和 SSHFS-Win：
      <ul>
       <li>
        访问
        <a href="https://winfsp.dev/rel/" rel="nofollow">
         WinFsp 官方网站
        </a>
        下载并安装 WinFsp。
       </li>
       <li>
        访问
        <a href="https://github.com/billziss-gh/sshfs-win/releases">
         SSHFS-Win 官方网站
        </a>
        下载并安装 SSHFS-Win。
       </li>
      </ul>
     </li>
     <li>
      打开“命令提示符”或“PowerShell”，输入以下命令挂载远程目录：
      <pre><code class="prism language-plaintext">net use Z: \\sshfs\Ubuntu_Username@Ubuntu_IP\home\user\shared /user:Ubuntu_Username Ubuntu_Password
</code></pre>
      <ul>
       <li>
        <code>
         Z:
        </code>
        ：指定本地挂载的驱动器号。
       </li>
       <li>
        <code>
         Ubuntu_Username
        </code>
        ：Ubuntu 系统的用户名。
       </li>
       <li>
        <code>
         Ubuntu_IP
        </code>
        ：Ubuntu 电脑的 IP 地址。
       </li>
       <li>
        <code>
         Ubuntu_Password
        </code>
        ：Ubuntu 系统用户的密码。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_FTP__82">
     </a>
     方案三：使用 FTP 服务
    </h3>
    <p>
     在 Ubuntu 上搭建 FTP 服务器，让 Windows 10 通过 FTP 客户端访问。
    </p>
    <h4>
     <a id="Ubuntu_18__85">
     </a>
     Ubuntu 18 端配置
    </h4>
    <ol>
     <li>
      安装 vsftpd 服务：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> vsftpd
</code></pre>
     </li>
     <li>
      配置 vsftpd：
      <br/>
      打开配置文件：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">nano</span> /etc/vsftpd.conf
</code></pre>
      确保以下配置项正确：
      <pre><code class="prism language-plaintext">anonymous_enable=NO
local_enable=YES
write_enable=YES
local_umask=022
</code></pre>
      保存并退出。
     </li>
     <li>
      重启 vsftpd 服务：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl restart vsftpd
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="Windows_10__109">
     </a>
     Windows 10 端访问
    </h4>
    <ol>
     <li>
      打开“文件资源管理器”，在地址栏输入
      <code>
       ftp://Ubuntu_IP
      </code>
      ，按回车键。
     </li>
     <li>
      输入 Ubuntu 系统的用户名和密码，即可访问 FTP 共享目录。
     </li>
    </ol>
