    <h2>
     <a id="root_3">
     </a>
     <code>
      :root
     </code>
    </h2>
    <p>
     <strong>
      <code>
       :root
      </code>
     </strong>
     这个 CSS
     <font color="“skyblue”">
      伪类匹配文档树的根元素
     </font>
     。对于 HTML 来说，:root 表示
     <code>
      &lt;html&gt;
     </code>
     元素，除了优先级更高之外，与
     <code>
      html
     </code>
     选择器相同。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择文档的根元素（HTML 中的 &lt;html&gt;） */</span>
<span class="token selector">:root</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background</span><span class="token punctuation">:</span> yellow<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_CSS__14">
     </a>
     声明全局 CSS 变量
    </h4>
    <p>
     在声明全局 CSS 变量时
     <strong>
      <code>
       :root
      </code>
     </strong>
     会很有用：
    </p>
    <pre><code class="prism language-css"><span class="token selector">:root</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">--main-color</span><span class="token punctuation">:</span> hotpink<span class="token punctuation">;</span>
  <span class="token property">--pane-padding</span><span class="token punctuation">:</span> 5px 42px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h2>
     <a id="_CSS__25">
     </a>
     使用 CSS 自定义属性
    </h2>
    <ul>
     <li>
      自定义属性是由 CSS 作者定义的，它包含的值可以在
      <font color="“skyblue”">
       整个文档中重复使用
      </font>
     </li>
     <li>
      由自定义属性标记设定值（比如：
      <strong>
       <code>
        --main-color: black;
       </code>
      </strong>
      ）
     </li>
     <li>
      由
      <font color="“skyblue”">
       var() 函数
      </font>
      来获取值（比如：
      <code>
       color: var(--main-color);
      </code>
      ）
     </li>
     <li>
      自定义属性在某个地方存储一个值，然后在其他许多地方引用它
     </li>
     <li>
      另一个好处是
      <font color="“skyblue”">
       语义化的标识
      </font>
      。比如，
      <code>
       --main-text-color
      </code>
      会比
      <code>
       #00ff00
      </code>
      更易理解
     </li>
     <li>
      自定义属性受级联的约束，并从其父级继承其值
     </li>
    </ul>
    <h4>
     <a id="_34">
     </a>
     基本用法
    </h4>
    <p>
     声明一个自定义属性，
     <font color="“skyblue”">
      属性名需要以两个减号（
      <code>
       --
      </code>
      ）开始
     </font>
     ，属性值则可以是任何有效的 CSS 值
    </p>
    <p>
     和其他属性一样，自定义属性也是写在规则集之内的，如下：
    </p>
    <pre><code class="prism language-css"><span class="token selector">element</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">--main-bg-color</span><span class="token punctuation">:</span> brown<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     注意，规则集所指定的选择器定义了自定义属性的可见作用域。
    </p>
    <p>
     通常的最佳实践是定义在根伪类
     <code>
      :root
     </code>
     下，这样就可以在 HTML 文档的任何地方访问到它了：
    </p>
    <pre><code class="prism language-css"><span class="token selector">:root</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">--main-bg-color</span><span class="token punctuation">:</span> brown<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_56">
     </a>
     注意
    </h4>
    <p>
     <strong>
      <font color="“skyblue”">
       自定义属性名是大小写敏感的
      </font>
     </strong>
     ，
     <code>
      --my-color
     </code>
     和
     <code>
      --My-color
     </code>
     会被认为是两个不同的自定义属性
    </p>
    <h4>
     <a id="Var_60">
     </a>
     Var()函数
    </h4>
    <p>
     如前所述，使用一个局部变量时用
     <font color="“skyblue”">
      <strong>
       <code>
        var()
       </code>
       函数包裹以表示一个合法的属性值
      </strong>
     </font>
     ：
    </p>
    <pre><code class="prism language-css"><span class="token selector">element</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> <span class="token function">var</span><span class="token punctuation">(</span>--main-bg-color<span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_70">
     </a>
     自定义属性的继承性
    </h4>
    <p>
     <font color="“skyblue”">
      自定义属性会继承
     </font>
     。这意味着如果在一个给定的元素上，没有为这个自定义属性设置值，在其父元素上的值会被使用。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>one<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
  <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>two<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>three<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>four<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
  <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <pre><code class="prism language-css"><span class="token selector">.two</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">--test</span><span class="token punctuation">:</span> 10px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token selector">.three</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">--test</span><span class="token punctuation">:</span> 2em<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     在这个情况下，
     <code>
      var(--test)
     </code>
     的结果分别是：
    </p>
    <ul>
     <li>
      对于元素
      <code>
       class="two"
      </code>
      ：
      <code>
       10px
      </code>
     </li>
     <li>
      对于元素
      <code>
       class="three"
      </code>
      ：
      <code>
       2em
      </code>
     </li>
     <li>
      对于元素
      <code>
       class="four"
      </code>
      ：
      <code>
       10px
      </code>
      （继承自父属性）
     </li>
     <li>
      对于元素
      <code>
       class="one"
      </code>
      ：
      <em>
       非法值
      </em>
      ，会变成自定义属性的默认值
     </li>
    </ul>
    <p>
     注意，这些是自定义属性，并不是你在其他编程语言中遇到的实际的变量。这些值仅当需要的时候才会计算，而并不会按其他规则进行保存。
    </p>
    <h4>
     <a id="_102">
     </a>
     自定义属性备用值
    </h4>
    <p>
     用
     <font color="“skyblue”">
      <strong>
       <code>
        var()
       </code>
       函数
      </strong>
     </font>
     可以定义多个
     <strong>
      备用值
     </strong>
     （fallback value），当给定值未定义时将会用备用值替换。
    </p>
    <p>
     <strong>
      备注：
     </strong>
     备用值并不是用于实现浏览器兼容性的。如果浏览器不支持 CSS 自定义属性，备用值也没什么用。它仅对支持 CSS 自定义属性的浏览器提供了一个备份机制，该机制仅当给定值未定义或是无效值的时候生效。
    </p>
    <ul>
     <li>
      函数的第一个参数是自定义属性的名称。
     </li>
     <li>
      如果提供了第二个参数，则表示备用值
     </li>
     <li>
      当自定义属性值无效时生效。第二个参数可以嵌套，但是不能继续平铺展开下去了
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">.two</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> <span class="token function">var</span><span class="token punctuation">(</span>--my-var<span class="token punctuation">,</span> red<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">/* Red if --my-var is not defined */</span>
<span class="token punctuation">}</span>

<span class="token selector">.three</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> <span class="token function">var</span><span class="token punctuation">(</span>
    --my-var<span class="token punctuation">,</span>
    <span class="token function">var</span><span class="token punctuation">(</span>--my-background<span class="token punctuation">,</span> pink<span class="token punctuation">)</span>
  <span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">/* pink if --my-var and --my-background are not defined */</span>
<span class="token punctuation">}</span>

<span class="token selector">.three</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> <span class="token function">var</span><span class="token punctuation">(</span>
    --my-var<span class="token punctuation">,</span>
    --my-background<span class="token punctuation">,</span>
    pink
  <span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">/* Invalid: "--my-background, pink" */</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     第二个例子展示了如何处理一个以上的 fallback。该技术可能会导致性能问题，因为它花了更多的时间在处理这些变量上。
    </p>
    <h4>
     <a id="JavaScript__135">
     </a>
     JavaScript 中的值
    </h4>
    <p>
     在 JavaScript 中获取或者修改 CSS 变量和操作普通 CSS 属性是一样的：
    </p>
    <pre><code class="prism language-css">// 获取一个 Dom 节点上的 CSS 变量
element.style.<span class="token function">getPropertyValue</span><span class="token punctuation">(</span><span class="token string">"--my-var"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token function">getComputedStyle</span><span class="token punctuation">(</span>element<span class="token punctuation">)</span>.<span class="token function">getPropertyValue</span><span class="token punctuation">(</span><span class="token string">"--my-var"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

// 修改一个 Dom 节点上的 CSS 变量
element.style.<span class="token function">setProperty</span><span class="token punctuation">(</span><span class="token string">"--my-var"</span><span class="token punctuation">,</span> jsVar + 4<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <h2>
     <a id="_150">
     </a>
     最佳单位实践
    </h2>
    <h4>
     <a id="1_px_152">
     </a>
     1.
     <code>
      px
     </code>
     （像素）
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：绝对单位，基于显示设备的物理像素点。尽管称为“像素”，但在高分辨率屏幕上可能会有所不同。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        提供了精确的尺寸控制。
       </li>
       <li>
        不随浏览器设置或父元素字体大小的变化而变化。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：当你需要固定的、不可缩放的尺寸时使用，例如细线边框、图标大小等。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">width</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span> <span class="token comment">/* 固定宽度 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="2_em_166">
     </a>
     2.
     <code>
      em
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：相对单位，相对于当前元素的字体大小。如果父元素设置了字体大小，则
      <code>
       em
      </code>
      的值基于该字体大小计算。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        允许根据文本大小调整布局。
       </li>
       <li>
        如果嵌套元素较多，可能导致复杂度增加，因为每个层级的字体大小都会影响下一层级。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：适用于希望随着父元素的字体大小变化而变化的设计部分，如段落、标题等。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-size</span><span class="token punctuation">:</span> 1.5em<span class="token punctuation">;</span> <span class="token comment">/* 相对于父元素字体大小放大1.5倍 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="3_remroot_em_180">
     </a>
     3.
     <code>
      rem
     </code>
     （root em）
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：类似于
      <code>
       em
      </code>
      ，但它是相对于根元素（即
      <code>
       &lt;html&gt;
      </code>
      标签）的字体大小来计算。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        简化了响应式设计中的尺寸管理，因为它只依赖于根元素的字体大小。
       </li>
       <li>
        避免了
        <code>
         em
        </code>
        可能导致的嵌套层级问题。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：非常适合用于整个页面的基础布局和排版，因为它提供了更一致的尺寸关系。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">html</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-size</span><span class="token punctuation">:</span> 16px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-size</span><span class="token punctuation">:</span> 1.2rem<span class="token punctuation">;</span> <span class="token comment">/* 相对于根元素字体大小放大1.2倍 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="4_vw__vh_198">
     </a>
     4.
     <code>
      vw
     </code>
     和
     <code>
      vh
     </code>
     （视窗宽度和高度百分比）
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：
      <code>
       1vw
      </code>
      等于视口宽度的 1%，
      <code>
       1vh
      </code>
      等于视口高度的 1%。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        动态地适应用户的屏幕尺寸。
       </li>
       <li>
        特别适合创建全屏或基于视口比例的设计。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：当需要设计能够自适应不同屏幕尺寸的内容时非常有用，比如全屏背景图、响应式布局等。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">width</span><span class="token punctuation">:</span> 50vw<span class="token punctuation">;</span> <span class="token comment">/* 占据视口宽度的一半 */</span>
  <span class="token property">height</span><span class="token punctuation">:</span> 30vh<span class="token punctuation">;</span> <span class="token comment">/* 占据视口高度的30% */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="5__213">
     </a>
     5.
     <code>
      %
     </code>
     （百分比）
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：相对单位，相对于包含块（通常是父元素）的宽度或高度。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        提供了一种灵活的方式来定义元素大小，使其可以根据其容器大小自动调整。
       </li>
       <li>
        可以应用于宽度、高度、内边距、外边距等多个属性。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：流体布局中，尤其是那些需要根据父容器大小动态调整自身大小的元素。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">div</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">width</span><span class="token punctuation">:</span> 80%<span class="token punctuation">;</span> <span class="token comment">/* 占据父元素宽度的80% */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="6_fr_227">
     </a>
     6.
     <code>
      fr
     </code>
     （网格分数单位）
    </h4>
    <ul>
     <li>
      <strong>
       描述
      </strong>
      ：仅在 CSS Grid 布局中使用的单位，表示可用空间的一部分。
     </li>
     <li>
      <strong>
       特点
      </strong>
      ：
      <ul>
       <li>
        提供了一种简单的方法来分配网格轨道的空间。
       </li>
       <li>
        可以轻松创建复杂的网格布局。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：在构建基于网格的布局时，用于定义行和列的大小。
     </li>
    </ul>
    <pre><code class="prism language-css"><span class="token selector">.container</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">display</span><span class="token punctuation">:</span> grid<span class="token punctuation">;</span>
  <span class="token property">grid-template-columns</span><span class="token punctuation">:</span> 1fr 2fr<span class="token punctuation">;</span> <span class="token comment">/* 第二列是第一列宽度的两倍 */</span>
<span class="token punctuation">}</span>
</code></pre>
    <h2>
     <a id="media_242">
     </a>
     @media
    </h2>
    <p>
     <strong>
      <code>
       @media
      </code>
     </strong>
     CSS@规则可用于基于一个或多个媒体查询的结果来应用样式表的一部分。使用它，你可以指定一个媒体查询和一个 CSS 块，当且仅当该媒体查询与正在使用其内容的设备匹配时，该 CSS 块才能应用于该文档。
    </p>
    <p>
     <strong>
      备注：
     </strong>
     在 JavaScript 中，可以使用
     <code>
      CSSMediaRule
     </code>
     CSS 对象模型接口访问使用
     <code>
      @media
     </code>
     创建的规则。
    </p>
    <h4>
     <a id="_248">
     </a>
     语法
    </h4>
    <p>
     <code>
      @media
     </code>
     at 规则可置于你代码的顶层或嵌套至其他任何的 at 条件规则组中。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 在你的代码的顶层 */</span>
<span class="token atrule"><span class="token rule">@media</span> screen <span class="token keyword">and</span> <span class="token punctuation">(</span><span class="token property">min-width</span><span class="token punctuation">:</span> 900px<span class="token punctuation">)</span></span> <span class="token punctuation">{<!-- --></span>
  <span class="token selector">article</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">padding</span><span class="token punctuation">:</span> 1rem 3rem<span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 嵌套至其他的 at 条件规则中 */</span>
<span class="token atrule"><span class="token rule">@supports</span> <span class="token punctuation">(</span><span class="token property">display</span><span class="token punctuation">:</span> flex<span class="token punctuation">)</span></span> <span class="token punctuation">{<!-- --></span>
  <span class="token atrule"><span class="token rule">@media</span> screen <span class="token keyword">and</span> <span class="token punctuation">(</span><span class="token property">min-width</span><span class="token punctuation">:</span> 900px<span class="token punctuation">)</span></span> <span class="token punctuation">{<!-- --></span>
    <span class="token selector">article</span> <span class="token punctuation">{<!-- --></span>

    <span class="token punctuation">}</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_270">
     </a>
     媒体类型
    </h4>
    <p>
     <em>
      媒体类型
     </em>
     （media type）描述设备的一般类别。除非使用
     <code>
      not
     </code>
     或
     <code>
      only
     </code>
     逻辑运算符，否则媒体类型是可选的，并且会（隐式地）应用
     <code>
      all
     </code>
     类型。
    </p>
    <ul>
     <li>
      <p>
       <code>
        all
       </code>
      </p>
      <p>
       适用于所有设备
      </p>
     </li>
     <li>
      <p>
       <code>
        print
       </code>
      </p>
      <p>
       适用于在打印预览模式下在屏幕上查看的分页材料和文档
      </p>
     </li>
     <li>
      <p>
       <code>
        screen
       </code>
      </p>
      <p>
       主要用于屏幕
      </p>
     </li>
    </ul>
