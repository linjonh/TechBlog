    <h2>
     <a id="_2__0">
     </a>
     深度学习 第2章 线性代数
    </h2>
    <p>
     线性代数是深度学习的语言。 张量操作是神经网络计算的基石，矩阵乘法是前向传播的核心，范数约束模型复杂度，而生成空间理论揭示模型表达能力的本质。 本章介绍线性代数的基本内容，为进一步学习深度学习做准备。
    </p>
    <h3>
     <a id="_2">
     </a>
     主要内容
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fa29b0126b544881874dc44105ba9e96.png"/>
    </p>
    <h4>
     <a id="21__5">
     </a>
     2.1 标量、向量、矩阵和张量
    </h4>
    <ul>
     <li>
      <strong>
       标量
      </strong>
      ：单个数字，用斜体表示，通常赋予小写字母变量名。
     </li>
     <li>
      <strong>
       向量
      </strong>
      ：数字数组，按顺序排列，用粗体小写字母表示，元素通过下标访问。
     </li>
     <li>
      <strong>
       矩阵
      </strong>
      ：二维数字数组，用粗体大写字母表示，元素通过两个索引访问。
     </li>
     <li>
      <strong>
       张量
      </strong>
      ：多维数字数组，用特殊字体表示，元素通过多个索引访问。
     </li>
    </ul>
    <h4>
     <a id="22__11">
     </a>
     2.2 矩阵和向量的乘法
    </h4>
    <ul>
     <li>
      <strong>
       矩阵乘法
      </strong>
      ：两个矩阵相乘得到第三个矩阵，要求第一个矩阵的列数等于第二个矩阵的行数。
     </li>
     <li>
      <strong>
       点积
      </strong>
      ：两个向量的点积是它们对应元素乘积的和，结果是一个标量。
     </li>
     <li>
      <strong>
       矩阵乘法的性质
      </strong>
      ：矩阵乘法是可分配的和结合的，但不具有交换性。
     </li>
    </ul>
    <h4>
     <a id="23__16">
     </a>
     2.3 单位矩阵和逆矩阵
    </h4>
    <ul>
     <li>
      <strong>
       单位矩阵
      </strong>
      ：与任何向量相乘都不会改变该向量的矩阵，主对角线元素为1，其余为0。
     </li>
     <li>
      <strong>
       逆矩阵
      </strong>
      ：矩阵A的逆矩阵A⁻¹满足A⁻¹A = I，用于求解线性方程组。
     </li>
    </ul>
    <h4>
     <a id="24__20">
     </a>
     2.4 线性相关性和生成空间
    </h4>
    <ul>
     <li>
      <strong>
       线性相关性
      </strong>
      ：一组向量中至少有一个向量可以表示为其他向量的线性组合。
     </li>
     <li>
      <strong>
       生成空间
      </strong>
      ：一组向量的所有线性组合构成的集合。
     </li>
    </ul>
    <h4>
     <a id="25__24">
     </a>
     2.5 范数
    </h4>
    <ul>
     <li>
      <strong>
       Lᵖ范数
      </strong>
      ：衡量向量大小的函数，L¹范数（欧几里得范数）和L²范数常用。
     </li>
     <li>
      <strong>
       矩阵范数
      </strong>
      ：衡量矩阵大小的函数，如弗罗贝尼乌斯范数。
      <br/>
      “范数衡量向量大小，L¹鼓励稀疏性，L²平滑但易微分。”
     </li>
    </ul>
    <h4>
     <a id="26__28">
     </a>
     2.6 特殊类型的矩阵和向量
    </h4>
    <ul>
     <li>
      <strong>
       对角矩阵
      </strong>
      ：只有主对角线有非零元素的矩阵。
     </li>
     <li>
      <strong>
       对称矩阵
      </strong>
      ：等于其转置的矩阵。
     </li>
     <li>
      <strong>
       正交矩阵
      </strong>
      ：行和列都是标准正交的矩阵。
     </li>
    </ul>
    <h4>
     <a id="27__33">
     </a>
     2.7 特征分解
    </h4>
    <ul>
     <li>
      <strong>
       特征向量和特征值
      </strong>
      ：矩阵A的特征向量v满足Av = λv，λ是对应的特征值。
     </li>
     <li>
      <strong>
       特征分解
      </strong>
      ：将矩阵分解为特征向量和特征值的组合。
     </li>
    </ul>
    <h3>
     <a id="_37">
     </a>
     总结
    </h3>
    <p>
     线性代数是深度学习的“语法”：
    </p>
    <ul>
     <li>
      张量作为数据容器，支撑图像、语音等高维数据表示。
     </li>
     <li>
      矩阵乘法是神经网络前向传播的核心运算（如全连接层）。
     </li>
     <li>
      范数约束模型复杂度（如正则化项）
     </li>
     <li>
      生成空间理论解释模型表达能力
     </li>
     <li>
      线性相关性直接影响参数优化稳定性。
      <br/>
      掌握这些概念，可深入理解神经网络的计算本质与设计逻辑。
     </li>
    </ul>
    <h3>
     <a id="_46">
     </a>
     延伸思考
    </h3>
    <p>
     深度学习视角：张量（高维数组）是神经网络中数据的通用表示形式（如图像=3D张量）。
     <br/>
     实践意义：广播机制（broadcasting）允许不同维度张量运算，是框架（如PyTorch）高效实现的关键。掌握这些内容，就握住了理解深度学习模型的钥匙！
    </p>

     精彩语句

        英文
       </strong>

       </strong>
       ：矩阵乘法是行与列的线性组合，而非逐元素乘积。
       <br/>
       <strong>
        解释
       </strong>
       ：揭示神经网络中权重与输入交互的本质（如全连接层计算）。
      </p>
     </li>
     <li>
      <p>
       <strong>

       </strong>

       <br/>
       <strong>
        解释
       </strong>
       ：正交性在梯度稳定性和参数初始化中的关键作用（如防止梯度爆炸）。

       <br/>
       <strong>
        解释
       </strong>
       ：指导损失函数设计（如Lasso回归与岭回归的正则化选择）。
      </p>
