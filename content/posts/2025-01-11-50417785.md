    <h4>
     <span style="font-family:宋体; font-size:18px; line-height:150%">
      一、背景
     </span>
    </h4>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
    </p>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      很荣幸接手了成绩系统项目并当了组长，也正直搬家时期，同时也换了新的服务器和数据库，上一版本的系统我们得修改服务器地址和数据库的链接，然后重新发布到新的服务器上，才能让之前的系统跑起来，让我有机会在服务器上发布了一次系统。
     </span>
    </p>
    <p>
     <span style="font-family:宋体; font-size:14px">
      <br/>
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <h4>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      二、在服务器上使用
      <span lang="EN-US">
       IIS
      </span>
      部署
     </span>
    </h4>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
    </p>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
      <a href="http://blog.csdn.net/u010927640/article/details/50408713" rel="noopener noreferrer" target="_blank">
       <span style="color:#ff0000">
       </span>
      </a>
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      都是
      <span lang="EN-US">
       IIS
      </span>
      部署网站，个人觉得应该和本地发布是一样的，但这次是在服务器上部署，而且都是前人的成果，不敢乱来。
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      在部署之前先“浏览”一下以前服务器上是否能够成功，确保在新的服务器上发布出现问题时可以排除是原来项目的问题。下面以部署
      <span lang="EN-US">
       ITOO.ResultTeacherMark.Client
      </span>
      为例在服务器上使用
      <span lang="EN-US">
       IIS
      </span>
      部署网站。
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <h5>
     <span style="color:#ff0000">
      <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
       1.
      </span>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
       把原来服务器上的文件直接拷到新的服务器上，搜索“IIS”→打开“信息服务（
       <span lang="EN-US">
        IIS）
       </span>
       管理器”，如下：
      </span>
     </span>
    </h5>
    <p>
     <img alt="" height="553" src="https://img-blog.csdn.net/20151228152915289?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" style="width:318px; height:485px" width="322">
      <span style="font-family:宋体; line-height:150%; font-size:12pt">

        <br/>
       </span>
      </span>
     </img>
    </p>
    <h5>
     <span style="color:#ff0000">
      <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
       2.
      </span>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
       右击“网站”→“添加网站”：
      </span>
     </span>
    </h5>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      <img alt="" height="493" src="https://img-blog.csdn.net/20151228115510307?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" style="width:376px; height:400px" width="459">
      </img>
     </span>
    </p>
    <p>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      <span style="color:#ff0000">
      </span>

       <span style="color:#ff0000">
       </span>
      </a>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
      </span>
      注意端口是有一定的范围，这时在左侧已经可以看到我们新建的网站了，名称为
      <span lang="EN-US">
       ITOO.ResultTeacherMark.Client
      </span>
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <h5>
     <span style="color:#ff0000">
      <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
       3.
      </span>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
       修改网站的权限为
       <span lang="EN-US">
        everyone
       </span>
       ，右击网站名称
       <span lang="EN-US">
        →
       </span>
       编辑权限
       <span lang="EN-US">
        →
       </span>
       安全
       <span lang="EN-US">
        →
       </span>
       编辑
       <span lang="EN-US">
        →
       </span>
       添加：
      </span>
     </span>
    </h5>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
      <img alt="" height="479" src="https://img-blog.csdn.net/20151228114313781?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" style="width:546px; height:353px" width="702"/>

    </span>
    <p>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      <a href="http://blog.csdn.net/u010927640/article/details/50408713" rel="noopener noreferrer" target="_blank">
       <span style="color:#ff0000">
       </span>
      </a>
      在“输入对象名称来选择”框里输入
      <span lang="EN-US">
       E
      </span>
      ，点击“检查名称”，就可以把网站权限定为
      <span lang="EN-US">
       Everyone，在Everyone的权限中“允许”列全都勾上。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      <span lang="EN-US">

     <span style="color:#ff0000">
      <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
       4.
      </span>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">

    </h5>
    <span style="font-family:宋体; font-size:14px">

     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
      IIS
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      默认用的
      <span lang="EN-US">
       .NET Framework
      </span>
      是
      <span lang="EN-US">
       2.0
      </span>
      版本，这里选用的是所发布网站的框架版本，一般如果用的是
      <span lang="EN-US">
       VS2010
      </span>
      以上版本开发的话选择的是
      <span lang="EN-US">
       V4.0
      </span>
      ，托管道模式要选择“集成”，如果选择“经典”的话，运行时可能会出错。
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <h5>
     <span style="color:#ff0000">
      <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
       5.
      </span>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
       浏览网站
      </span>
     </span>
    </h5>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      找到刚发布好的网站→选择内容视图→找到网站的起始页面
      <span lang="EN-US">
       →
      </span>
      浏览，就可以看到咱们发布的网站了O(∩_∩)O~~
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
      <img alt="" height="405" src="https://img-blog.csdn.net/20151228114535916?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" style="width:595px; height:335px" width="646"/>
     </span>

    <span style="font-family:宋体; font-size:14px">

    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      在发布的过程中遇到的一个比较难的问题：
      <span lang="EN-US">
       windows7
      </span>
      中
      <span lang="EN-US">
       IIS
      </span>
      运行
      <span lang="EN-US">
       ASP
      </span>
      时出现
      <span lang="EN-US">
       ADODB.Connection
      </span>
      错误
      <span lang="EN-US">
       “800a0e7a”
      </span>
     </span>
     <span style="font-family:宋体; font-size:14px">
     </span>
    </p>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
     </span>
     <strong>
      <span style="font-family:宋体; line-height:150%; font-size:12pt">
       原因：
      </span>
     </strong>
    </p>
    <span style="font-family:宋体; font-size:14px">
    </span>
    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
      <a href="http://blog.csdn.net/u010927640/article/details/50408713" rel="noopener noreferrer" target="_blank">
       <span style="color:#ff0000">
       </span>
      </a>
      64
     </span>
     <span style="font-family:宋体; line-height:150%; font-size:12pt">
      位系统下
      <span lang="EN-US">
       IIS
      </span>
      无法运行
      <span lang="EN-US">
       32
      </span>
      位程序，由于
      <span lang="EN-US">
       64
      </span>
      位操作系统不支持
      <span lang="EN-US">
       MicrosoftOLE DB Provider for Jet
      </span>
      驱动程序，也不支持更多的
      <span lang="EN-US">
       Microsofte AccessDriver (*.mdf)
      </span>
      方式连接。
     </span>
    </p>
    <span style="font-family:宋体; font-size:14px">

    <p>
     <span lang="EN-US" style="font-family:宋体; line-height:150%; font-size:12pt">
