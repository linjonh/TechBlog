    <h2 id="ofFX-1741799085378" name="ofFX-1741799085378" style="text-align:left">
     <strong>
      1.静态函数
     </strong>
    </h2>
    <div style="text-align:left">
     静态函数就是被static修饰的函数；
    </div>
    <h3 id="O6Ad-1741799085382" name="O6Ad-1741799085382" style="text-align:left">
     <strong>
      为什么要有静态函数呢？
     </strong>
    </h3>
    <div style="text-align:left">
     <strong>
      背景知识
     </strong>
     ：函数是可以跨文件使用的，也就是说函数a在文件1.c下被定义，也可以在2.c文件下使用。
    </div>
    <div style="text-align:left">
     所谓的静态函数就是使其函数只能在本文件内使用，在其前面加上static修饰。用staic修饰的函数可以缩小可见范围。
    </div>
    <h4 id="EiE4-1741799085388" name="EiE4-1741799085388" style="text-align:left">
     <strong>
      问:用static修饰的函数这样做有什么好处呢？
     </strong>
    </h4>
    <div style="text-align:left">
     这样可以避免在其他文件中出现同名称的情况，避免互相影响。
    </div>
    <h2 id="CDI1-1741799085392" name="CDI1-1741799085392" style="text-align:left">
     <strong>
      2.递归函数
     </strong>
    </h2>
    <div style="text-align:left">
     <strong>
      定义
     </strong>
     ：一个函数函数不断调用自己
    </div>
    <div style="text-align:left">
     <strong>
      特征：
     </strong>
    </div>
    <div>
     <ul style="margin-left:0; margin-right:0">
      <li style="text-align:left">
       递归函数必须要有退出的条件，否则就会进入无线循环，导致栈内存满了而出现段错误；
      </li>
      <li style="text-align:left">
       递归函数包含两个过程，逐渐递进和逐渐回归过程。
      </li>
     </ul>
    </div>
    <div style="text-align:left">
     图解：
    </div>
    <div style="text-align:left">
     板书代码：
    </div>
    <div style="text-align:left">
     <img alt="" height="693" src="https://i-blog.csdnimg.cn/direct/85da5f6d45c941e99cfa0c218a629ca3.png" width="1214">
     </img>
    </div>
    <div style="text-align:left">
    </div>
    <div style="text-align:left">
     递归函数的return返回值必须要退出有条件，在返回值中调用自己，使其不断进入函数内存去接受另外一个返回值，在从最里面退出
    </div>
    <div style="text-align:left">
    </div>
    <div>
     <pre><code>第一次返回值：3*jiecheng(2)      //进入到jiecheng(2)里面
第二次返回值:   2*jiecheng(1)    //jiecheng(2) 返回2*jiecheng(1)
第三次返回值：      return 1     //jiecheng(1)  返回1

也可以理解为
          3*jiecheng(2)
          3*2*jiecheng(1)
          3*2*1</code></pre>
     <p>
     </p>
    </div>
    <div style="text-align:left">
    </div>
    <div style="text-align:left">
    </div>
    <h3 id="%E5%9B%BE%E8%A7%A3%3A%E2%80%8B%E7%BC%96%E8%BE%91" name="%E5%9B%BE%E8%A7%A3%3A%E2%80%8B%E7%BC%96%E8%BE%91" style="text-align:left">
     图解:
     <img alt="" height="646" src="https://i-blog.csdnimg.cn/direct/2ba784df551b4e3a9da362d69404152e.png" width="1129"/>
    </h3>
    <div style="text-align:left">
    </div>
    <div style="text-align:left">
    </div>
    <div style="text-align:left">
     每一次调用函数，都会在栈内存分配一块空间给他使用,可以理解为第一次return返回的值是一个浓缩的值，它内含着很多次的调用。
    </div>
    <div style="text-align:left">
    </div>
    <h3 id="%E4%BB%A3%E7%A0%81%E9%AA%8C%E8%AF%81%3A" name="%E4%BB%A3%E7%A0%81%E9%AA%8C%E8%AF%81%3A" style="text-align:left">
     代码验证:
    </h3>
    <pre><code>#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;

int mi(int a, int n)
{
    if (n  == 0)
    {
        return 1;
    }

    return mi(a, n - 1) * a;
}
int main(void) 
{
    int a = 7, n = 4;
    // 7的4次幂   7*7*7*7；
    int he = mi(a, n);
    printf("%d\n", he);

    return 0;
}</code></pre>
