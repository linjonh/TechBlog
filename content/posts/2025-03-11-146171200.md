    <p>
     Houdini Labs插件是用于游戏中程序化内容生成的工具，可在github上下载：
     <br/>
     <a href="https://github.com/sideeffects/SideFXLabs">
      https://github.com/sideeffects/SideFXLabs
     </a>
    </p>
    <p>
     Building Generator是生成建筑相关的工具集，官方Demo中的生成案例如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/887b5a64c50e46ac93245eecb03707ea.jpeg#pic_center">
      <br/>
      该案例下载地址：
      <br/>
      <a href="https://www.sidefx.com/tutorials/building-generator/" rel="nofollow">
       https://www.sidefx.com/tutorials/building-generator/
      </a>
     </img>
    </p>
    <p>
     本文来入门学习该工具。
    </p>
    <hr/>
    <p>
     1.Building Generator工具有许多组织方式，例如可以通过BuildingGeneratorUtility(标记每一块组件)和BuildingGenerator(整合生成)节点组织并生成房屋：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8ee94454281d4536b74eadc4ec0b185b.jpeg#pic_center">
      <br/>
      2.接下来开始制作，首先制作第一个BuildingGeneratorUtility。先新建Geometry，并添加Box和Building_generator_utility节点，
      <strong>
       将ModuleName命名为Item1
      </strong>
      。
      <br/>
      <strong>
       在building_generator_utility节点中，MoudleName指最终生成房屋时指定的模块名称，另外此处需勾选Auto fill Dimensions，自动填充维度。
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5a087566416542bfb1e5ae28246f9c26.jpeg#pic_center">
       <br/>
       3.添加节点building_generator，第一个输入处链接房屋外观mesh，第二个输入链接所有utility信息。清除building_generator节点所有无关属性，将Wall Face的Facade Module Pattern的名称设置为Item1。
      </img>
     </img>
    </p>
    <p>
     连接好后调节building_generator参数，可以看见只有一个组件的房屋装配完成，
     <strong>
      若不显示可检查房屋外观大小尺寸是否足够
     </strong>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/614ffd8b82d44f48a3f39e6b1419bf59.jpeg#pic_center">
      <br/>
      4.再在utility节点处添加color节点设置颜色，并添加merge节点代替第二个输入位置，方便后续操作。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5ff438657904424abe1a09c2ad6cc76b.jpeg#pic_center">
       <br/>
       5.复制一套Utility节点，设置不同颜色，将两个Utility的MoudleName分别设置为Item1*A和Item1*B
       <br/>
       在这里*号代表变体。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a82e6fd659c04b84ad2999c502f7b6b6.png#pic_center">
        <br/>
        6.修改building_generator的module节点参数为变体名前缀，可以看见显示出了不同效果：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d6775a79b45d4d298b4da6310f9b8b9f.jpeg#pic_center"/>
       </img>
      </img>
     </img>
    </p>
    <p>
     在此基础上继续添加ledge、corner等module即可。
    </p>
