    <p>
     项目名称为Evidential Detection and Tracking Collaboration，主要用于强大的反无人机系统，涉及新问题、基准和算法研究。下面介绍项目的复现步骤：
    </p>
    <ol>
     <li>
      <strong>
       安装环境
      </strong>
      ：使用Anaconda创建并激活名为
      <code>
       edtc
      </code>
      的虚拟环境，Python版本为3.6，然后执行
      <code>
       bash install_pytorch17.sh
      </code>
      脚本安装相关依赖。
     </li>
    </ol>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> edtc <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.6</span>
conda activate edtc
<span class="token function">bash</span> install_pytorch17.sh
</code></pre>
    <ol start="2">
     <li>
      <strong>
       训练YOLO检测器
      </strong>
      <ul>
       <li>
        编辑数据集设置文件
        <code>
         /path/to/EDTC/yolov5/data/antiuav.yaml
        </code>
        ，配置数据集路径等信息。
       </li>
       <li>
        进入
        <code>
         /path/to/EDTC/yolov5
        </code>
        目录，执行
        <code>
         python train.py
        </code>
        开始训练YOLO检测器。
       </li>
      </ul>
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /path/to/EDTC/yolov5
python train.py
</code></pre>
    <ol start="3">
     <li>
      <strong>
       训练跟踪分支
      </strong>
      <ul>
       <li>
        运行
        <code>
         python tracking/create_default_local_file.py --workspace_dir . --data_dir ./data --save_dir .
        </code>
        命令设置项目路径，之后可通过编辑
        <code>
         lib/train/admin/local.py
        </code>
        （训练路径相关）、
        <code>
         experiments/uavtrack/baseline.yaml
        </code>
        （阶段1训练路径相关）、
        <code>
         experiments/uavtrack_eh/baseline.yaml
        </code>
        （阶段2训练路径相关）文件修改路径。
       </li>
       <li>
        阶段1训练：执行
        <code>
         python tracking/train.py --script uavtrack --config baseline --save_dir . --mode multiple --nproc_per_node 8
        </code>
        命令进行训练。
       </li>
       <li>
        阶段2训练：执行
        <code>
         python tracking/train.py --script uavtrack_eh --config baseline --save_dir /PATH/TO/SAVE/UAVTRACK_EH --mode multiple --nproc_per_node 8 --stage1_model /STAGE1/MODEL/PATH
        </code>
        命令进行训练，需指定阶段1模型的路径。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       在AntiUAV600数据集上评估
      </strong>
      <ul>
       <li>
        从[Models and Raw results](Google Driver)下载预训练模型。
       </li>
       <li>
        编辑
        <code>
         lib/test/evaluation/local.py
        </code>
        （测试路径相关）和
        <code>
         experiments/uavtrack_eh/baseline.yaml
        </code>
        （YOLO预训练模型路径，修改其中第133 - 134行）文件，设置相关路径。
       </li>
       <li>
        设置环境变量：
       </li>
      </ul>
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">PYTHONPATH</span><span class="token operator">=</span><span class="token variable">$PYTHONPATH</span>:/path/to/EDTC
<span class="token builtin class-name">export</span> <span class="token assign-left variable">PYTHONPATH</span><span class="token operator">=</span><span class="token variable">$PYTHONPATH</span>:/path/to/EDTC/yolov5
</code></pre>
    <pre><code>- 运行评估命令：`python tracking/test.py uavtrack_eh baseline --dataset antiuav --threads 32 --num_gpus 8 --params__model /path/to/UAVTrackEH.pth.tar --params__search_area_scale 4.55`。若要可视化跟踪结果，需编辑`/path/to/EDTC/lib/test/evaluation/environment.py`文件的第27行，将`self.show_result`设置为`True`。
- 执行`python tracking/evaluate_antiuav_performance.py`命令获取评估指标。
</code></pre>
    <p>
     在复现过程中，若数据集尚未发布，需等待数据集发布后获取并按要求放置在指定路径。同时，确保各路径设置正确，若涉及多GPU训练，需保证GPU环境正常且符合代码要求。
    </p>
