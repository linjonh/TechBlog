# vue项目配置后端地址

在Vue项目中配置后端地址可以通过修改
`config/index.js`
文件来完成。

1. 打开
   `config/index.js`
   文件；
2. 查找到
   `proxyTable`
   属性，如果没有则手动添加该属性；
3. 将需要访问的接口路由设置为对应的后端地址，示例如下所示
   ：

   ```javascript
   module.exports = {
     // ...其他配置...
     
     dev: {
       proxyTable: {
         '/api': {
           target: 'http://localhost:8080', // 这里写上后端服务器的地址及端口号
           changeOrigin: true,
           pathRewrite: {
             '^/api': '' // 若后端接口不包含'/api'前缀，则此处可省略
           }
         },
         
         // 还可以根据需求配置更多的代理表达式
       }
     }
   }
   ```
4. 保存并关闭文件；
5. 重新运行Vue项目时，就会使用配置好的后端地址进行数据交互了。
