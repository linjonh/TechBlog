    <h4 id="NVNQR" name="NVNQR">
     1.1 万能头文件（快捷方式）
    </h4>
    <pre id="O56tJ"><code>#include &lt;bits/stdc++.h&gt;  // GCC 扩展头文件（非标准）
using namespace std;      // 启用标准命名空间（包含 cin/cout）</code></pre>
    <p id="u9eb7c965">
     <strong>
      注意事项
     </strong>
     ：
    </p>
    <ul>
     <li id="ub88106f0">
      仅适用于小型程序/竞赛场景，正式项目推荐显式包含所需头文件（如
      <code>
       &lt;iostream&gt;
      </code>
      <code>
       &lt;vector&gt;
      </code>
      ）
     </li>
     <li id="u6d52cebc">
      使用
      <code>
       using namespace std
      </code>
      可能造成命名污染，大型项目建议显式调用（如
      <code>
       std::cout
      </code>
      ）
     </li>
    </ul>

     3.1 字符串操作（

    <p class="img-center">

    </p>
    <pre id="s5xLc"><code>string s = "Hello";

s.push_back('!');       // 尾部插入字符

s.erase(2, 3);          // 删除从索引2开始的3个字符

    <h3 id="Lm79b" name="Lm79b">
     四、实践建议
    </h3>
    <ol>
     <li id="uc250010c">
      <strong>
       头文件管理
      </strong>
      ：大型项目使用
      <code>
       #pragma once
      </code>
      防止重复包含
     </li>
     <li id="u46d521f0">
      <strong>
       内存安全
      </strong>
      ：vector 替代原始数组，避免越界访问
     </li>
     <li id="u981a3c66">
      <strong>
       性能优化
      </strong>
      ：
      <span style="background-color:#f9eda6">
       优先使用
      </span>
      <code>
       <span style="background-color:#f9eda6">
        emplace_back()
       </span>
      </code>
      <span style="background-color:#f9eda6">
       替代
      </span>
      <code>
       <span style="background-color:#f9eda6">
        push_back()
       </span>
      </code>
      <span style="background-color:#f9eda6">
       减少拷贝
      </span>
     </li>
     <li id="u6e2624c6">
      <strong>
       代码规范
      </strong>
      ：复杂排序规则 建议使用
      <span style="background-color:#f9eda6">
       lambda 表达式保持代码简洁
      </span>
     </li>
    </ol>
