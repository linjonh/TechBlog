    <h3>
     前言
    </h3>
    <p>
     <strong>
      <a class="link-info" href="https://uniapp.dcloud.net.cn/" rel="nofollow" title="uni-app">
       uni-app
      </a>
      的生命周期主要分为：应用生命周期、页面生命周期、组件生命周期三种
     </strong>
    </p>
    <hr/>
    <h3>
     一、
     <a class="link-info" href="https://uniapp.dcloud.net.cn/collocation/App.html#applifecycle" rel="nofollow" title="应用">
      应用
     </a>
     生命周期（App.vue中声明）
    </h3>
    <p>
     <strong>
      <span style="color:#fe2c24">
       注意
      </span>
      ：仅在
      <span style="color:#fe2c24">
       App.vue
      </span>
      中有效，在其他页面监听无效
     </strong>
    </p>
    <p>
     <strong>
      触发场景
     </strong>
     ：整个应用的
     <span style="color:#fe2c24">
      初始化
     </span>
     、前后台切换等
     <span style="color:#fe2c24">
      全局
     </span>
     行为
    </p>
    <p>
     <img alt="" height="937" src="https://i-blog.csdnimg.cn/direct/d4a6885173d842648badd32bf93eaecb.png" width="1537"/>
    </p>
    <h4>
     <strong>
      常用钩子：
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       ​
       <strong>
        onLaunch
       </strong>
       <br/>
       应用初始化完成时触发（仅触发一次）
       <br/>
       典型用途：全局配置（如登录状态检测、系统信息获取）
      </p>
     </li>
     <li>
      <p>
       ​
       <strong>
        onShow
       </strong>
       <br/>
       应用启动或从后台进入前台时触发
       <br/>
       典型用途：恢复定时器、更新全局数据
      </p>
     </li>
     <li>
      <p>
       ​
       <strong>
        onHide
       </strong>
       <br/>
       应用从前台进入后台时触发
       <br/>
       典型用途：暂停动画、保存临时数据
      </p>
     </li>
     <li>
      <p>
       ​
       <strong>
        onError
       </strong>
       <br/>
       全局错误捕获（如脚本错误、API调用失败）
      </p>
     </li>
     <li>
      <p>
       ​
       <strong>
        onPageNotFound
       </strong>
       <br/>
       页面路径不存在时触发（用于自定义404跳转）
      </p>
     </li>
    </ul>
    <h3>
     二、
     <a class="link-info" href="https://uniapp.dcloud.net.cn/tutorial/page.html#lifecycle" rel="nofollow" title="页面">
      页面
     </a>
     生命周期（pages目录下的.vue文件）
    </h3>
    <p>
     页面生命周期函数主要在
     <code>
      pages
     </code>
     目录下的
     <code>
      .vue
     </code>
     文件中使用
    </p>
    <p>
     这些函数涵盖了页面从
     <strong>
      创建
     </strong>
     到
     <strong>
      销毁
     </strong>
     的各个阶段
    </p>
    <p>
     页面文件
     <span style="color:#fe2c24">
      <strong>
       同时拥有两种生命周期
      </strong>
     </span>
     ：
    </p>
    <ol>
     <li>
      ​
      <strong>
       页面特有生命周期
      </strong>
      ​（如onLoad、onShow）
     </li>
     <li>
      ​
      <strong>
       Vue组件标准生命周期
      </strong>
      ​（如created、mounted）
     </li>
    </ol>
    <p>
     <img alt="" height="1123" src="https://i-blog.csdnimg.cn/direct/2ecaf292431e4bb29390a3f5954014dc.png" width="1312">
      <img alt="" height="1123" src="https://i-blog.csdnimg.cn/direct/fc0b952ed43d42069cc71baa654302ca.png" width="1312"/>
     </img>
    </p>
    <h4>
     <strong>
      常用钩子 ：
     </strong>
    </h4>
    <p>
     <strong>
      <code>
       1、onLoad
      </code>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       触发时机
      </strong>
      ：页面
      <span style="color:#fe2c24">
       第一次
      </span>
      加载时触发
     </li>
     <li>
      <strong>
       场景
      </strong>
      ：
      <ul>
       <li>
        <strong>
         接收
        </strong>
        页面传递过来的
        <strong>
         参数
        </strong>
       </li>
       <li>
        发起接口
        <strong>
         请求
        </strong>
        ，获取数据
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="language-javascript">export default {
    onLoad(options) {
        console.log('页面加载时接收到的参数:', options);
        this.fetchData();
    },
    methods: {
        fetchData() {
            // 请求接口数据
        }
    }
};
</code></pre>
    <p>
     <strong>
      <code>
       2、onShow
      </code>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       触发时机
      </strong>
      ：页面
      <span style="color:#fe2c24">
       每次显示
      </span>
      时触发，包括从其他页面返回当前页面
     </li>
     <li>
      <strong>
       场景
      </strong>
      ：
      <ul>
       <li>
        刷新页面数据
       </li>
       <li>
        检查登录状态等
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="language-javascript">export default {
    onShow() {
        console.log('页面显示时执行');
        this.checkLoginStatus();
    },
    methods: {
        checkLoginStatus() {
            // 检查登录状态
        }
    }
};
</code></pre>
    <p>
     <strong>
      <code>
       3、onReady
      </code>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       触发时机
      </strong>
      ：页面
      <span style="color:#fe2c24">
       初次渲染完成
      </span>
      时触发，此时页面已经准备好可以与用户进行
      <span style="color:#fe2c24">
       交互
      </span>
      。
     </li>
     <li>
      <strong>
       用途
      </strong>
      ：
      <ul>
       <li>
        进行DOM操作。
       </li>
       <li>
        初始化页面中的某些组件或插件。
       </li>
      </ul>
     </li>
    </ul>
    <table>
     <thead>
      <tr>
       <th>
        钩子函数
       </th>
       <th>
        触发时机
       </th>
       <th>
        典型场景
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        ​
        <strong>
         onLoad
        </strong>
       </td>
       <td>
        页面首次加载时触发（仅一次）
       </td>
       <td>
        接收参数（如
        <code>
         options.id
        </code>
        ）、初始化页面数据、发起首次接口请求
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         onShow
        </strong>
       </td>
       <td>
        页面每次显示时触发（包括tab切换/返回）
       </td>
       <td>
        更新动态数据（如刷新列表）、启动页面动画
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         onReady
        </strong>
       </td>
       <td>
        页面初次渲染完成（可能早于进场动画结束）
       </td>
       <td>
        操作DOM元素（需注意异步渲染问题，可用
        <code>
         $nextTick
        </code>
        ）
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         onUnload
        </strong>
       </td>
       <td>
        页面卸载（如redirectTo或关闭页面）
       </td>
       <td>
        清除定时器、取消未完成的网络请求
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         onPullDownRefresh
        </strong>
       </td>
       <td>
        用户下拉页面（需在json中开启
        <code>
         enablePullDownRefresh
        </code>
        ）
       </td>
       <td>
        实现下拉刷新功能
       </td>
      </tr>
     </tbody>
    </table>
    <h3>
     三、
     <a class="link-info" href="https://uniapp.dcloud.net.cn/tutorial/vue-api.html#%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F" rel="nofollow" title="组件">
      组件
     </a>
     生命周期（components目录下的.vue文件）
    </h3>
    <p>
     <span style="color:#fe2c24">
      注意
     </span>
     ：无页面特有钩子，组件不包含onLoad、onShow等页面级生命周期
    </p>
    <p>
     <strong>
      遵循Vue标准生命周期：
     </strong>
    </p>
    <p>
     <img alt="" height="974" src="https://i-blog.csdnimg.cn/direct/ddab9a29410647b9b4271ee7013da3ad.png" width="1390"/>
    </p>
    <h4>
     常用钩子：
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        钩子函数
       </th>
       <th>
        触发时机
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        ​
        <strong>
         created
        </strong>
       </td>
       <td>
        实例创建完成（可访问data但未挂载DOM）
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         mounted
        </strong>
       </td>
       <td>
        DOM挂载完成（可操作DOM元素）
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         beforeDestroy
        </strong>
       </td>
       <td>
        实例销毁前（适合清除全局事件监听）
       </td>
      </tr>
     </tbody>
    </table>
    <h3>
     四、生命周期执行顺序
    </h3>
    <pre><code class="language-javascript">应用启动时：
App.onLaunch → App.onShow → Page.onLoad → Page.created → Page.onShow → Page.mounted → Page.onReady

页面跳转时（A→B）：
A.onHide → B.onLoad → B.created → B.onShow → B.mounted → B.onReady</code></pre>
