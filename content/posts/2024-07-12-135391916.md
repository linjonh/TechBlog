    <p>
     HarmonyOS是华为自主开发的操作系统，它在2020年9月正式发布。它最初被称为鸿蒙OS，后来更名为HarmonyOS。HarmonyOS旨在提供一种可在各种设备上无缝运行的统一操作系统，包括智能手机、平板电脑、智能穿戴设备、智能音箱、车载系统、智能家居设备等等。相比于其他操作系统，HarmonyOS具有更高的安全性、更快的响应速度、更好的用户体验和更强的可扩展性。同时，HarmonyOS还支持多种编程语言和多种开发框架，使开发者更容易地开发应用程序。
    </p>
    <p>
     HarmonyOS官网：https://developer.harmonyos.com/
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f6b0684aa37d32964ed329d45345a775.png">
      <br/>
      为什么要学习鸿蒙，因为9月底的发布会上，华为终端业务CEO余承东宣布，鸿蒙原生应用全面启动，HarmonyOS NEXT开发者预览版将在2024年第一季度面向开发者开放。针对HarmonyOS NEXT，华为全新开发了独创的鸿蒙内核、大模型加持的原生智能、方舟引擎颠覆性的性能和功耗等。
     </img>
    </p>
    <p>
     HarmonyOS NEXT是不在兼容安卓的，随着鸿蒙原生应用全面启动，近期多家互联网公司也发布了多个和鸿蒙有关的岗位。比如，网易更新了高级/资深Android开发工程师岗位，职位要求参与云音乐多端多OS的产品（Android、鸿蒙等）研发迭代。美团发布了鸿蒙高级工程师、鸿蒙基建工程师等多个鸿蒙开发相关岗位。根据脉脉平台，美团鸿蒙基建工程师岗位月薪在4万元到6万元之间。同时，钉钉也发布了鸿蒙开发工程师岗位，月薪在1.5万元到3万元之间。
    </p>

    <h3 id="H24">
     🔎3.配置开发环境
    </h3>
    <p>
     双击启动，进入首次运行的欢迎页面：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7566c0e43fd3a8ec09f93a550a1126ff.png"/>
     <br/>
     选择Agree，进入配置导入页面，选择不导入任何配置：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c42524816a2dc3a5333d6097db4ccaa8.png"/>
    </p>
    <p>
     选择要安装的Node.js和Ohpm位置，其中ohpm是Open Harmony Package Management的缩写，也就是类似npm的包管理工具。这里有几点注意事项：
    </p>
    <ul>
     <li>
      如果电脑上已经有Node.js，但是版本不一致，建议选择让工具重新安装
     </li>
     <li>
      如果电脑上已经有Node.js，并且版本一致，可以选择Local，指定node目录即可
     </li>
     <li>
      如果电脑上对Node.js做了一些特殊的options配置，建议先移除
     </li>
     <li>
      配置目录中不要出现中文、特殊字符，建议用默认路径
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8701325da7567f8146f94da594f78af5.png"/>
     <br/>
     选择Next后，进入HarmonyOS的SDK安装目录选择页面，同样注意不要出现中文和特殊字符，建议走默认路径：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fd699e12465c1065f0bff46fa8919855.png"/>
     <br/>
     选择Next后，进入协议页面，全部选择Accept：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/da3055b7ba69a77fe273cfd3ca3dabde.png"/>
     选择Next后，进入SDK确认信息页面：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cd2565c9bf623c9984ac829c9d57dd75.png"/>
     <br/>
     直接点击Next安装即可。
    </p>
    <p>
     具体如果有错误官方文档有更详细说明：https://developer.harmonyos.com/cn/docs/documentation/doc-guides-V3/environment_config-0000001052902427-V3#section551917522319
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b340d30890cbf99e714af39ee1b7dc8.png"/>
    </p>

    <h3 id="H28">
     🔎3.安装模拟器
    </h3>
    <p>
     我们也可以利用设备模拟器来查看更真实的效果。不过需要先配置模拟器。
     <br/>
     首先，选择主菜单中的Tools，找到其中的Device Manager，即设备管理：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/19717422932167b6d69ec3e98e4433d1.png"/>
     <br/>
     设备可以是远端设备，也可以是本地设备，我们以本地设备为例。
     <br/>
     默认本地没有任何设备，选择install来安装一个：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/19f82982f26dc24b0cbe05d8876f2b8b.png"/>
     <br/>
     首次点击时，会弹出一个窗口，下载必要的SDK依赖：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/39c64269d97f3d91c625210ef8d98d10.png"/>
     <br/>
     等待一段时间的下载之后，选择Finish。
     <br/>
     进入创建模拟器页面，选择New Emulator：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/05920f21b23ca3561c2baa36b68c699e.png"/>
     <br/>
     添加手机模拟器：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e4be54cccc1fe8d73e3dbf5e664576f9.png"/>
     <br/>
     选择api9版本，不过需要注意，首次进入此页面，需要下载手机设备需要的系统，大概2.2G，需要耐心等待：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5435e6d3fde520fbae79e52efc8e89a5.png"/>
     <br/>
     下载完成后，才可以选中，然后点击Next，进入下一步：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/48bb0b48c50393b287f69638ca1b850a.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/61103d3696085b4a944b0e65166cfe84.png"/>
     <br/>
     创建完成后，在设备列表中会出现一个本地设备，点击后面的运行按钮即可启动设备模拟器：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9a6ce8466e217073bde08ed86045abdb.png"/>
     <br/>
     启动后如图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17cf3603fdf0fb7aef7de60ef50d1b2b.png"/>
     <br/>
     然后，在应用启动位置选择刚刚添加的模拟器：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ddb9cc3dd41541b5c3ea73eeee64f517.png"/>
     <br/>
     点击启动，可以将应用部署到模拟器：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2ef65663c85eb3ec433271d90f9479c8.png"/>
