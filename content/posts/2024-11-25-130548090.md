    <p>
     在Java中，构造函数是类似于方法的代码块。它在创建类的实例时被调用。在调用构造函数时，对象的内存是在内存中分配的。
    </p>
    <p>
    </p>
    <p>
     它是一种特殊类型的方法，用于初始化对象。
    </p>
    <p>
    </p>
    <p>
     每次使用 new() 关键字创建对象时，至少会调用一个构造函数。
    </p>
    <p>
    </p>
    <p>
     如果类中没有可用的构造函数，它会调用默认构造函数。在这种情况下，Java 编译器默认提供默认构造函数。
    </p>
    <p>
    </p>
    <p>
     Java 中有两种类型的构造函数：无参数构造函数和参数化构造函数。
    </p>
    <p>
    </p>
    <p>
     注意：它被称为构造函数，因为它在对象创建时构造值。没有必要为类编写构造函数。这是因为如果您的类没有默认构造函数，java 编译器会创建一个默认构造函数。
    </p>
    <p>
    </p>
    <h4>
     <strong>
      # 创建 Java 构造函数的规则
     </strong>
    </h4>
    <p>
    </p>
    <p>
     为构造函数定义了两个规则。
    </p>
    <p>
    </p>
    <ol>
     <li>
      <p>
       构造函数名必须与其类名相同
      </p>
     </li>
     <li>
      <p>
       构造函数必须没有明确的返回类型
      </p>
     </li>
     <li>
      <p>
       Java 构造函数不能是抽象的、静态的、最终的和同步的
      </p>
     </li>
    </ol>
    <p>
    </p>
    <p>
     注意：我们可以在声明构造函数时使用访问修饰符。它控制对象的创建。换句话说，我们可以在 Java 中拥有 private、protected、public 或 default 构造函数。
    </p>
    <p>
    </p>
    <h3>
     <strong>
      # Java构造函数的类型
     </strong>
    </h3>
    <p>
    </p>
    <p>
     Java 中有两种类型的构造函数：
    </p>
    <p>
    </p>
    <ol>
     <li>
      <p>
       默认构造函数（无参数构造函数）
      </p>
     </li>
     <li>
      <p>
       参数化构造函数
      </p>
     </li>
    </ol>
    <h4>
    </h4>
    <h3>
     <strong>
      # Java 默认构造函数
     </strong>
    </h3>
    <p>
    </p>
    <p>
     当构造函数没有任何参数时，它被称为“默认构造函数”。
    </p>
    <h4>
    </h4>
    <h4>
     默认构造函数的语法：
    </h4>
    <pre><code class="language-java">
&lt;类名&gt;(){}</code></pre>
    <h3>
     默认构造函数示例
    </h3>
    <p>
    </p>
    <p>
     在此示例中，我们在 Bike 类中创建无参数构造函数。它将在对象创建时被调用。
    </p>
    <pre><code class="language-java">
//Java 程序创建并调用默认构造函数  
class Bike1{  
//创建默认构造函数 
Bike1(){System.out.println("自行车已创建");}  
//主要方法 
public static void main(String args[]){  
//调用默认构造函数 
Bike1 b=new Bike1();  
}  
}</code></pre>
    <blockquote>
     <p>
      输出：
     </p>
     <pre><code class="hljs">
自行车已创建</code></pre>
    </blockquote>
    <p>
     规则：如果类中没有构造函数，编译器会自动创建一个默认构造函数。
    </p>
    <p>
    </p>
    <h4>
     <strong>
      Q) 默认构造函数的目的是什么？
     </strong>
    </h4>
    <p>
    </p>
    <p>
     默认构造函数用于为对象提供默认值，如 0、null 等，具体取决于类型。
    </p>
    <h4>
    </h4>
    <h4>
     显示默认值的默认构造函数示例
    </h4>
    <pre><code class="language-java">
//让我们看另一个默认构造函数的例子 
//显示默认值 
class Student3{  
int id;  
String name;  
//显示id和name值的方法 
void display(){System.out.println(id+" "+name);}  
  
public static void main(String args[]){  
//创建对象 
Student3 s1=new Student3();  
Student3 s2=new Student3();  
//显示对象的值 
s1.display();  
s2.display();  
}  
}</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-java">
0 null
0 null</code></pre>
    <p>
     说明：在上面的类中，您没有创建任何构造函数，因此编译器为您提供了一个默认构造函数。这里 0 和 null 值由默认构造函数提供。
    </p>
    <p>
    </p>
    <h4>
     Java参数化构造函数
    </h4>
    <p>
    </p>
    <p>
     具有特定数量参数的构造函数称为参数化构造函数。
    </p>
    <p>
    </p>
    <h4>
     为什么要使用参数化构造函数？
    </h4>
    <p>
    </p>
    <p>
     参数化构造函数用于为不同的对象提供不同的值。但是，您也可以提供相同的值。
    </p>
    <h4>
    </h4>
    <h4>
     参数化构造函数示例
    </h4>
    <p>
    </p>
    <p>
     在此示例中，我们创建了具有两个参数的 Student 类的构造函数。我们可以在构造函数中有任意数量的参数。
    </p>
    <pre><code class="language-java">
//Java程序演示参数化构造函数的使用。
class Student4{  
    int id;  
    String name;  
    //创建参数化构造函数 
    Student4(int i,String n){  
    id = i;  
    name = n;  
    }  
    //显示值的方法 
    void display(){System.out.println(id+" "+name);}  
   
    public static void main(String args[]){  
    //创建对象并传递值
    Student4 s1 = new Student4(111,"小卡");  
    Student4 s2 = new Student4(222,"小安");  
    //调用方法显示对象的值
    s1.display();  
    s2.display();  
   }  
}</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="hljs">
111 小卡
222 小安</code></pre>
    <h3>
     <strong>
      # Java 中的构造函数重载
     </strong>
    </h3>
    <p>
    </p>
    <p>
     在 Java 中，构造函数就像一个方法，但没有返回类型。它也可以像 Java 方法一样被重载。
    </p>
    <p>
    </p>
    <p>
     Java 中的构造函数重载是一种拥有多个具有不同参数列表的构造函数的技术。它们的排列方式使每个构造函数执行不同的任务。编译器根据列表中参数的数量及其类型来区分它们。
    </p>
    <p>
    </p>
    <h4>
     构造函数重载的例子
    </h4>
    <pre><code class="language-java">//Java程序重载构造函数 
class Student5{  
    int id;  
    String name;  
    int age;  
    //创建两个arg构造函
    Student5(int i,String n){  
    id = i;  
    name = n;  
    }  
    //创建三参数构造函数 
    Student5(int i,String n,int a){  
    id = i;  
    name = n;  
    age=a;  
    }  
    void display(){System.out.println(id+" "+name+" "+age);}  
   
    public static void main(String args[]){  
    Student5 s1 = new Student5(111,"小卡");  
    Student5 s2 = new Student5(222,"小安",25);  
    s1.display();  
    s2.display();  
   }  
}</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-java">
111 小卡 null
222 小安 25</code></pre>
    <h3>
     <strong>
      # Java中构造函数和方法的区别
     </strong>
    </h3>
    <p>
    </p>
    <p>
     构造函数和方法之间存在许多差异。它们在下面给出。
    </p>
    <p>
    </p>
    <table style="width:596px;">
     <thead>
      <tr>
       <th>
        Java构造函数
       </th>
       <th>
        方法
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        构造函数用于初始化对象的状态。
       </td>
       <td>
        方法用于暴露对象的行为。
       </td>
      </tr>
      <tr>
       <td>
        构造函数不能有返回类型。
       </td>
       <td>
        方法必须有返回类型。
       </td>
      </tr>
      <tr>
       <td>
        构造函数被隐式调用。
       </td>
       <td>
        该方法被显式调用。
       </td>
      </tr>
      <tr>
       <td>
        如果类中没有任何构造函数，Java 编译器会提供默认构造函数。
       </td>
       <td>
        在任何情况下，编译器都不会提供该方法。
       </td>
      </tr>
      <tr>
       <td>
        构造函数名必须与类名相同。
       </td>
       <td>
        方法名可能与类名相同，也可能不同。
       </td>
      </tr>
     </tbody>
    </table>
    <h4>
    </h4>
    <h3>
     <strong>
      # Java 复制构造函数
     </strong>
    </h3>
    <p>
    </p>
    <p>
     Java 中没有复制构造函数。但是，我们可以将值从一个对象复制到另一个对象，就像 C++ 中的复制构造函数一样。
    </p>
    <p>
    </p>
    <p>
     在 Java 中有多种方法可以将一个对象的值复制到另一个对象中。他们是：
    </p>
    <p>
    </p>
    <ul>
     <li>
      <p>
       通过构造函数
      </p>
     </li>
     <li>
      <p>
       通过将一个对象的值分配给另一个对象
      </p>
     </li>
     <li>
      <p>
       通过 Object 类的 clone() 方法
      </p>
     </li>
    </ul>
    <p>
    </p>
    <p>
     在此示例中，我们将使用 Java 构造函数将一个对象的值复制到另一个对象中。
    </p>
    <pre><code class="language-java">
//Java 程序将值从一个对象初始化为另一个对象。
class Student6{  
    int id;  
    String name;  
    //初始化整数和字符串的构造函数 
    Student6(int i,String n){  
    id = i;  
    name = n;  
    }  
    //构造函数来初始化另一个对象 
    Student6(Student6 s){  
    id = s.id;  
    name =s.name;  
    }  
    void display(){System.out.println(id+" "+name);}  
   
    public static void main(String args[]){  
    Student6 s1 = new Student6(111,"小卡");  
    Student6 s2 = new Student6(s1);  
    s1.display();  
    s2.display();  
   }  
}</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-java">
111小卡
111小卡</code></pre>
    <h4>
     <strong>
      问）构造函数是否返回任何值？
     </strong>
    </h4>
    <p>
    </p>
    <p>
     是的，它是当前的类实例（你不能使用返回类型但它返回一个值）。
    </p>
    <p>
    </p>
    <h4>
     <strong>
      构造函数可以执行其他任务而不是初始化吗？
     </strong>
    </h4>
    <p>
    </p>
    <p>
     是的，像创建对象、启动线程、调用方法等。您可以像在方法中执行操作一样在构造函数中执行任何操作。
    </p>
    <p>
    </p>
    <h4>
     <strong>
      Java中有构造函数类吗？
     </strong>
    </h4>
    <p>
    </p>
    <p>
     有的。
    </p>
    <h4>
    </h4>
    <h4>
     <strong>
      Constructor类的目的是什么？
     </strong>
    </h4>
    <p>
    </p>
    <p>
     Java提供了一个Constructor类，可以用来获取类中构造函数的内部信息。它位于 java.lang.reflect 包中。
