# jdk-21\_linux-x64\_bin.tar.gz Linux jdk21压缩包安装保姆级（详细安装教程）

---

官网下载地址：

```
https://www.oracle.com/java/technologies/downloads/#java21

```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/c1ac05c5016f452eb40acb0643d1375c.png)

## 一、简洁版（需要对 Linux 操作有一定基础）

1. 下载
   `jdk-21_linux-x64_bin.tar.gz`
   包
2. 在安装前需要先卸载 Linux 虚拟机自带的 java 版本
     
   输入以下指令查看 Java 版本

   ```shell
   rpm -qa | grep java

   ```

   卸载以上列出的所有的 Java 包

   ```shell
   rpm -e --nodeps 安装包1
   rpm -e --nodeps 安装包2

   ```
3. 在/usr/local目录下新建 java 文件夹，并将文件拖拽到
   `/usr/local/java`
   文件夹下

   ```shell
   cd /usr/local
   mkdir java
   cd java

   ```
4. 使用以下指令解压安装

   ```shell
   tar -zxvf jdk-21_linux-x64_bin.tar.gz

   ```
5. 配置 Java 环境变量

   ```shell
   export JAVA_HOME=/usr/local/java/jdk-21.0.6
   export PATH=$JAVA_HOME/bin:$PATH

   ```
6. 让环境变量生效

   ```shell
   source /etc/profile

   ```
7. 验证版本

   ```shell
   java -version

   ```

   可以自己写一个 java 程序试一下 Hello.java

   ```shell
   javac Hello.java
   java Hello

   ```
