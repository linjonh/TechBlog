    <h4>
     <a id="_95">
     </a>
     代码说明：
    </h4>
    <ol>
     <li>
      <strong>
       Base类
      </strong>
      ：这是一个抽象基类，定义了接口
      <code>
       print()
      </code>
      。
     </li>
     <li>
      <strong>
       DerivedA和DerivedB类
      </strong>
      ：这两个类继承自
      <code>
       Base
      </code>
      ，并实现了
      <code>
       print()
      </code>
      方法。
     </li>
     <li>
      <strong>
       Registry类
      </strong>
      ：这是一个注册表类，用于管理对象的创建。它使用一个
      <code>
       std::unordered_map
      </code>
      来存储对象名称和对应的创建函数。
      <code>
       registerCreator
      </code>
      方法用于注册创建函数，
      <code>
       create
      </code>
      方法用于根据名称创建对象。
     </li>
     <li>
      <strong>
       registerCreators函数
      </strong>
      ：这个函数用于注册具体的创建函数。
     </li>
     <li>
      <strong>
       main函数
      </strong>
      ：在
      <code>
       main
      </code>
      函数中，首先注册了
      <code>
       DerivedA
      </code>
      和
      <code>
       DerivedB
      </code>
      的创建函数，然后通过注册表创建并使用了这些对象。
     </li>
    </ol>
