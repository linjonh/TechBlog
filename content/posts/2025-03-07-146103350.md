    <h3>
     <a id="_2">
     </a>
     解题思路：
    </h3>
    <ol>
     <li>
      <strong>
       定义三个指针：
      </strong>
      prev（前驱节点）、current（当前节点）、nextNode（临时保存下一个节点）
     </li>
     <li>
      <strong>
       遍历链表：
      </strong>
      每次将 current.next 指向 prev，移动指针直到 current 为 null。
     </li>
    </ol>

     复杂度分析：

    <ul>
     <li>
      <strong>
       时间复杂度：
      </strong>

     <li>
      <strong>
       空间复杂度：

    <h3>
     <a id="_28">
     </a>
     解题思路：
    </h3>
    <ol>
     <li>
      <strong>
       找中点：
      </strong>
      使用快慢指针，快指针每次移动两步，慢指针每次移动一步，直到快指针到达末尾。此时慢指针位于链表中点。
     </li>
     <li>
      <strong>
       反转后半部分：
      </strong>
      将中点之后的链表部分反转。
     </li>
     <li>
      <strong>
       对称比较：
      </strong>
      从头节点和中点开始，逐个比较对应节点的值是否相等。
     </li>
    </ol>
