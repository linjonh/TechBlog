    记录一下在ubuntu中nginx+php部署tp项目
    <p>
    </p>
    <h2>
     <a id="nginx_2">
     </a>
     nginx
    </h2>
    <p>
     nginx就是正常下载
    </p>
    <h3>
     <a id="nginx_4">
     </a>
     下载nginx
    </h3>
    <p>
     <code>
      sudo apt-get install nginx
     </code>
     <br/>
     tp项目版本是3.2，通过设置路由，以域名/api.php/控制器/xxx的格式进行api的调用，文件后缀为.class.php
     <br/>
     需要在conf中进行一些处理，如下
    </p>
    <h3>
     <a id="nginxconf_8">
     </a>
     配置nginx.conf
    </h3>
    <pre><code class="prism language-nginx">server {
		listen 80 ;
		listen 443 ssl;# 有ssl证书的话
		server_name www.xxx.com;  # 你的域名
		# SSL证书配置
		ssl_certificate  /home/ssl/nginx/xxx.pem;#你的pem文件，在各大平台的ssl上下载
		ssl_certificate_key /home/ssl/nginx/xxx.key;#你的key文件，在各大平台的ssl上下载
		
		root /var/www/xxx;#你的项目根目录
		index index.php index.html index.htm;
		location / {
			try_files $uri $uri/ /api.php/$uri;
		}
		location ~ ^/api\.php(.*)$ {
			fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;  # 根据你的PHP版本调整
			fastcgi_index api.php;
			fastcgi_split_path_info ^(.+\.php)(.*)$;
			fastcgi_param PATH_INFO $fastcgi_path_info;
			fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
			include fastcgi_params;
		}
		location ~ /\.ht {
			deny all;
		}
	}
</code></pre>
    <h2>
     <a id="php_37">
     </a>
     php
    </h2>
    <p>
     使用的是php7.4
     <br/>
     //安装php本体
     <br/>
     <code>
      sudo apt-get install php7.4
     </code>
     安装项目所需要用到的扩展，比如我需要的是mysql相关的扩展
     <br/>
     <code>
      sudo apt-get install php7.4-mysqli php7.4-pdo-mysql
     </code>
     <br/>
     安装后可以通过
     <code>
      php -m|grep mysql
     </code>
     指令查看相关的扩展是否已启用
     <br/>
     如果没有的话可以通过下面的指令启用
     <br/>
     <code>
      sudo a2enmod mysqld
     </code>
     以及在php.ini中注释掉相关扩展前的
     <code>
      ;
     </code>
    </p>
    <p>
     通过以上配置之后，然后执行
     <code>
      systemctl restart nginx
     </code>
     重启nginx即可
    </p>
    <h2>
     <a id="_51">
     </a>
     其他
    </h2>
    <p>
     切换php版本
     <br/>
     <code>
      sudo update-alternatives --config php
     </code>
    </p>
