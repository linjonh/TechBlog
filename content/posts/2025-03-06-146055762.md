     <h2>
      <a id="_42">
      </a>
      二叉树的遍历
     </h2>
     <p>
      对于二叉树这种结构，最简单的操作就是遍历，
      <strong>
       所谓二叉树的遍历就是按照某种特定的规则，依次访问二叉树中的结点
      </strong>
      。只有学会了遍历操作，才能对其它的操作更快的学会。而二叉树的遍历总共有四种，分别是：
      <code>
       前序遍历，中序遍历，后序遍历以及层序遍历
      </code>
      ，最重要的就是前面三个。最基本的要求是对于一棵二叉树，要把这四种情况都能写出来。
      <br/>
      由于被访问的结点是某子树的根，所以N(Node)、L(Left subtree)、R(Right subtree)又可以解释为根、根的左子树和根的右子树。所以NLR、LNR、LRN分别又称为先根遍历，中根遍历和后根遍历。
     </p>
     <h3>
      <a id="_45">
      </a>
      前序遍历
     </h3>
     <p>
      <strong>
       前序遍历，也称先根遍历（NLR）
      </strong>
      ，即访问根节点的操作发生在其遍历左右子树之前，即
      <code>
       遍历顺序为：根、左子树、右子树。
      </code>
      <br/>
      因为二叉树是递归式的，所以遍历也用递归实现即可，代码如下：
     </p>

     <h3>
      <a id="_99">
      </a>
      层序遍历
     </h3>
     <p>
      层序遍历与先，中，后序遍历有所不同。所谓层序遍历就是从二叉树的根节点出发，首先访问第一层的树根节点，然后从左到右访问第二层上的节点，接着是第三层的节点，以此类推，
      <code>
       自上而下，从左至右逐层访问树的节点的过程就是层序遍历。
      </code>
      <br/>
      过程如下图所示：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5dcbc4537229497b8820115179e2c940.png">
       <br/>
       对于层序遍历，就需要借助一个队列。层序遍历的思想如下：
      </img>
     </p>
     <ol>
      <li>
       首先将根节点入队。
      </li>
      <li>
       <code>
        若队列非空，则队头节点出队，并访问该节点。若它有左孩子或者右孩子，则将它的孩子依次入队。
       </code>
      </li>
      <li>
       重复步骤2，直至队列为空。
      </li>
     </ol>
     <p>
      具体代码如下：

     <h2>
      <a id="_135">
      </a>
      二叉树的结点数相关问题
     </h2>
     <p>
      不仅是二叉树的遍历操作要用到递归，关于二叉树节点数相关问题也用的递归，如下面几个操作。
     </p>
     <h3>
      <a id="_137">
      </a>
      二叉树的结点个数
     </h3>
     <p>
      由前面我们知道，一棵二叉树可以分为根，左子树和右子树三个部分，那么想要求二叉树的节点个数时，只要知道左子树节点个数和右子树节点个数，再加上根结点就是整棵二叉树的节点个数。
      <br/>
      即
      <code>
       二叉树的节点个数=左子树节点个数+右子树节点个数+1。
      </code>
      <br/>
      代码如下：
     </p>
     <pre><code class="prism language-cpp"><span class="token keyword">int</span> <span class="token function">TreeSize</span><span class="token punctuation">(</span>BTNode<span class="token operator">*</span> root<span class="token punctuation">)</span>

     <h3>
      <a id="_148">
      </a>
      二叉树的叶子结点个数
     </h3>
     <p>
      所谓叶子节点，就是没有左右孩子，当叶子节点作为根节点时，就是左右子树都为空，按照这个性质，我们就可以求出叶子节点个数。也就是：
     </p>
     <ul>
      <li>
       当该节点没有左右孩子时，是叶子节点，则加一。
      </li>
      <li>
       当该节点有孩子时，将该节点作为根节点加上它下面的孩子为一棵树，也就是求左子树的叶子节点个数+右子树的叶子节点个数。
      </li>
     </ul>
     <p>
      代码如下：
     </p>
     <pre><code class="prism language-cpp"><span class="token keyword">int</span> <span class="token function">TreeLeafSize</span><span class="token punctuation">(</span>BTNode<span class="token operator">*</span> root<span class="token punctuation">)</span>

	<span class="token keyword">if</span> <span class="token punctuation">(</span>root<span class="token operator">-&gt;</span>left <span class="token operator">==</span> <span class="token constant">NULL</span> <span class="token operator">&amp;&amp;</span> root<span class="token operator">-&gt;</span>right <span class="token operator">==</span> <span class="token constant">NULL</span><span class="token punctuation">)</span>
		<span class="token keyword">return</span> <span class="token number">1</span><span class="token punctuation">;</span>
	<span class="token keyword">return</span> <span class="token function">TreeLeafSize</span><span class="token punctuation">(</span>root<span class="token operator">-&gt;</span>left<span class="token punctuation">)</span> <span class="token operator">+</span> <span class="token function">TreeLeafSize</span><span class="token punctuation">(</span>root<span class="token operator">-&gt;</span>right<span class="token punctuation">)</span><span class="token punctuation">;</span>

     <h3>
      <a id="k_165">
      </a>
      二叉树第k层结点个数
     </h3>
     <p>
      和上面一样的方式，当我想求第k层的节点个数时，只要知道
      <code>
       左子树的第k-1层节点数+右子树的k-1层节点数即可。
      </code>
      直到递推到
      <code>
       k==1时，若还有节点则返回1，没有节点则返回0。
      </code>
      <br/>
      如对于我想求第3层的节点个数，就等于左子树第2层的节点数+右子树第2层的节点数，直到左右子树的第k=1层结束。

      代码如下：

      <a id="x_180">

     </h3>
     <p>
      想要查找值为x的节点，其实也就是要遍历各个节点，找到相同的节点。我们可以直接将前序遍历的代码进行改造，
      <code>
       先遍历根结点，看是否为x若不是，则去左子树找，左子树找完去右子树找。
      </code>
      <br/>
      递归结束的两个条件：一个是如果访问的节点为空时，就返回空，另一个是如果节点与x对应，则直接返回该节点。
