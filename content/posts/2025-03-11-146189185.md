    <h2>
     <a id="Mac_OS__19">

     Mac OS 基础知识点总览
    </h2>
    <h3>
     <a id="1_Shell__21">
     </a>
     1. Shell 脚本基础
    </h3>
    <h4>
     <a id="11__23">
     </a>
     1.1 基本命令
    </h4>
    <ul>
     <li>
      <strong>
       echo
      </strong>
      ：输出文本到终端。
     </li>
     <li>
      <strong>
       ls
      </strong>
      ：列出目录内容。
     </li>
     <li>
      <strong>
       cd
      </strong>
      ：切换目录。
     </li>
     <li>
      <strong>
       pwd
      </strong>
      ：显示当前工作目录。
     </li>
     <li>
      <strong>
       mkdir
      </strong>
      ：创建新目录。
     </li>
     <li>
      <strong>
       rm
      </strong>
      ：删除文件或目录。
     </li>
     <li>
      <strong>
       cp
      </strong>
      ：复制文件或目录。
     </li>
     <li>
      <strong>
       mv
      </strong>
      ：移动或重命名文件或目录。
     </li>
    </ul>
    <h4>
     <a id="12__34">
     </a>
     1.2 示例代码块
    </h4>
    <pre><code class="prism language-sh"><span class="token comment"># 输出 Hello, World!</span>
<span class="token builtin class-name">echo</span> <span class="token string">"Hello, World!"</span>

<span class="token comment"># 列出当前目录内容</span>
<span class="token function">ls</span> <span class="token parameter variable">-l</span>

<span class="token comment"># 切换到根目录</span>
<span class="token builtin class-name">cd</span> /

<span class="token comment"># 显示当前工作目录</span>
<span class="token builtin class-name">pwd</span>

<span class="token comment"># 创建一个名为 "example" 的目录</span>
<span class="token function">mkdir</span> example

<span class="token comment"># 删除名为 "file.txt" 的文件</span>
<span class="token function">rm</span> file.txt

<span class="token comment"># 复制 "source.txt" 到 "destination.txt"</span>
<span class="token function">cp</span> source.txt destination.txt

<span class="token comment"># 将 "oldname.txt" 重命名为 "newname.txt"</span>

    <h3>
     <a id="2__62">
     </a>
     2. 文本处理
    </h3>
    <h4>
     <a id="21__64">
     </a>
     2.1 常用工具
    </h4>
    <ul>
     <li>
      <strong>
       cat
      </strong>
      ：连接文件并打印到标准输出。
     </li>
     <li>
      <strong>
       grep
      </strong>
      ：搜索文件中匹配指定模式的行。
     </li>
     <li>
      <strong>
       awk
      </strong>
      ：一个强大的文本处理工具，用于模式扫描和处理语言。
     </li>
     <li>
      <strong>
       sed
      </strong>
      ：流编辑器，用于对文本进行过滤和转换。
     </li>
    </ul>
    <h4>
     <a id="22__71">
     </a>
     2.2 示例代码块
    </h4>
    <pre><code class="prism language-sh"><span class="token comment"># 使用 cat 查看文件内容</span>
<span class="token function">cat</span> file.txt

<span class="token comment"># 使用 grep 搜索包含 "example" 的行</span>
<span class="token function">grep</span> <span class="token string">"example"</span> file.txt

<span class="token comment"># 使用 awk 打印文件的第二列</span>
<span class="token function">awk</span> <span class="token string">'{print $2}'</span> file.txt

<span class="token comment"># 使用 sed 将 "oldword" 替换为 "newword"</span>
<span class="token function">sed</span> <span class="token string">'s/oldword/newword/g'</span> file.txt
</code></pre>
    <h3>
     <a id="3__87">
     </a>
     3. 系统管理
    </h3>
    <h4>
     <a id="31__89">
     </a>
     3.1 进程管理
    </h4>
    <ul>
     <li>
      <strong>
       ps
      </strong>
      ：显示当前进程的状态。
     </li>
     <li>
      <strong>
       top
      </strong>
      ：实时显示系统中各个进程的资源占用情况。
     </li>
     <li>
      <strong>
       kill
      </strong>
      ：终止进程。
     </li>
    </ul>
    <h4>
     <a id="32__95">
     </a>
     3.2 示例代码块
    </h4>
    <pre><code class="prism language-sh"><span class="token comment"># 显示当前所有进程</span>
<span class="token function">ps</span> aux

<span class="token comment"># 实时查看系统资源占用情况</span>
<span class="token function">top</span>

<span class="token comment"># 终止进程 ID 为 1234 的进程</span>
<span class="token function">kill</span> <span class="token number">1234</span>
</code></pre>
    <h3>
     <a id="4__108">
     </a>
     4. 网络命令
    </h3>
    <h4>
     <a id="41__110">
     </a>
     4.1 常用工具
    </h4>
    <ul>
     <li>
      <strong>
       ping
      </strong>
      ：测试网络连通性。
     </li>
     <li>
      <strong>
       ifconfig
      </strong>
      或
      <strong>
       ip
      </strong>
      ：查看和配置网络接口。
     </li>
     <li>
      <strong>
       curl
      </strong>
      ：传输数据，支持多种协议，如 HTTP、FTP 等。
     </li>
    </ul>
    <h4>
     <a id="42__116">
     </a>
     4.2 示例代码块
    </h4>
    <pre><code class="prism language-sh"><span class="token comment"># 测试与 google.com 的连通性</span>
<span class="token function">ping</span> google.com

<span class="token comment"># 查看网络接口配置</span>
<span class="token function">ifconfig</span>
<span class="token comment"># 或者使用 ip 命令</span>
<span class="token function">ip</span> addr show

<span class="token comment"># 使用 curl 获取网页内容</span>
<span class="token function">curl</span> http://www.example.com
</code></pre>
