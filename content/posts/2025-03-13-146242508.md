    <h2>
     <a id="Leetcode236__1">
     </a>
     题目：Leetcode236. 二叉树的最近公共祖先
    </h2>
    <p>
     给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。
    </p>
    <p>
     百度百科中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”
    </p>
    <h2>
     <a id="_11">
     </a>
     原题链接
    </h2>
    <blockquote>
     <p>
      <a href="https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/?envType=problem-list-v2&amp;envId=2cktkvj" rel="nofollow">
       二叉树的最近公共祖先
      </a>
     </p>
    </blockquote>
    <h2>
     <a id="_13">
     </a>
     思路
    </h2>
    <ul>
     <li>
      最近公共祖先
      <code>
       两个节点都位于最近公共祖先的左右子树或者一个节点自己本身就是公共祖先
      </code>
     </li>
     <li>
      首先本题一定是自下而上的找，理所当然的我们使用后续遍历
     </li>
    </ul>
    <hr/>
    <ul>
     <li>
      如果
      <strong>
       left
      </strong>
      和
      <strong>
       right
      </strong>
      都不为空，说明
      <strong>
       p
      </strong>
      和
      <strong>
       q
      </strong>
      分别位于当前节点的左右子树中，因此当前节点
      <strong>
       root
      </strong>
      就是它们的最低公共祖先
     </li>
     <li>
      如果只有
      <strong>
       left
      </strong>
      不为空，说明
      <strong>
       p
      </strong>
      和
      <strong>
       q
      </strong>
      都在左子树中，返回
      <strong>
       left
      </strong>
     </li>
     <li>
      如果只有
      <strong>
       right
      </strong>
      不为空，说明
      <strong>
       p
      </strong>
      和
      <strong>
       q
      </strong>
      都在右子树中，返回
      <strong>
       right
      </strong>
     </li>
    </ul>
