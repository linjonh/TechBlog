    <h3>
     <a id="1Pythoncsv_4">
     </a>
     1.通过标准Python库导入csv文件
    </h3>
    <p>
     Python提供了一个标准的类库CSV文件。这个类库中的reader()函数用来导入CSV文件。当CSV文件被读入后，可以利用这些数据生成一个NumPy数组，用来训练算法模型。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> csv <span class="token keyword">import</span> reader
<span class="token keyword">import</span> numpy <span class="token keyword">as</span> np

<span class="token keyword">with</span> <span class="token builtin">open</span><span class="token punctuation">(</span>filename<span class="token punctuation">,</span> <span class="token string">'rt'</span><span class="token punctuation">,</span> encoding<span class="token operator">=</span><span class="token string">'UTF-8'</span><span class="token punctuation">)</span> <span class="token keyword">as</span> raw_data<span class="token punctuation">:</span>
    readers <span class="token operator">=</span> reader<span class="token punctuation">(</span>raw_data<span class="token punctuation">,</span> delimiter<span class="token operator">=</span><span class="token string">','</span><span class="token punctuation">)</span>
    x <span class="token operator">=</span> <span class="token builtin">list</span><span class="token punctuation">(</span>readers<span class="token punctuation">)</span>
    data <span class="token operator">=</span> np<span class="token punctuation">.</span>array<span class="token punctuation">(</span>x<span class="token punctuation">)</span>

    <span class="token keyword">print</span><span class="token punctuation">(</span>data<span class="token punctuation">.</span>shape<span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h3>
     <a id="2NumPycsv_24">
     </a>
     2.通过NumPy导入csv文件
    </h3>
    <p>
     也可以使用NumPy的loadtxt()函数导入数据。使用这个函数处理的数据没有文件头，并且所有的数据结构都是一样的，也就是说，数据类型是一样的。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> numpy <span class="token keyword">import</span> loadtxt
filename <span class="token operator">=</span> <span class="token string">'文件名'</span>
<span class="token keyword">with</span> <span class="token builtin">open</span><span class="token punctuation">(</span>filename<span class="token punctuation">,</span> <span class="token string">'rt'</span><span class="token punctuation">,</span> encoding<span class="token operator">=</span><span class="token string">'UTF-8'</span><span class="token punctuation">)</span> <span class="token keyword">as</span> raw_data<span class="token punctuation">:</span>
    data <span class="token operator">=</span> loadtxt<span class="token punctuation">(</span>raw_data<span class="token punctuation">,</span> delimiter<span class="token operator">=</span><span class="token string">','</span><span class="token punctuation">)</span>  <span class="token comment"># 数据类型必须一样，如果有列名，会报错</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span>data<span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h3>
     <a id="3Pandascsv_40">
     </a>
     3.通过Pandas导入csv文件
    </h3>
    <p>
     通过Pandas来导入CSV文件要使用
     <code>
      pandas.read_csv()
     </code>
     函数。这个函数的返回值是DataFrame，可以很方便的进行下一步的处理，实际操作过程中推荐使用这种方法。
    </p>
    <p>
     在机器学习的项目中，经常利用Pandas来做数据清洗与数据准备工作。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> pandas <span class="token keyword">import</span> read_csv
filename <span class="token operator">=</span> <span class="token string">"文件名"</span>

<span class="token keyword">print</span><span class="token punctuation">(</span>data<span class="token punctuation">)</span>
</code></pre>
