    <p>
     使用
     <strong>
      贪心算法
     </strong>
     解决这个问题，只需要维护两个最小值
     <code>
      first
     </code>
     和
     <code>
      second
     </code>
     ，如果找到了比
     <code>
      second
     </code>
     还大的数，就说明存在递增的三元组。
    </p>
    <h4>
     <strong>
      代码实现
     </strong>
    </h4>
    <pre><code class="language-python">def increasingTriplet(nums):
    first = second = float('inf')

    for num in nums:
        if num &lt;= first:  
            first = num  # 更新最小值
        elif num &lt;= second:  
            second = num  # 更新次小值
        else:  
            return True  # 找到了 third 使得 first &lt; second &lt; third

    return False

# 测试示例
print(increasingTriplet([1, 2, 3, 4, 5]))  # 输出: True
print(increasingTriplet([5, 4, 3, 2, 1]))  # 输出: False
print(increasingTriplet([2, 1, 5, 0, 4, 6]))  # 输出: True
</code></pre>
    <hr/>
    <h4>
     <strong>
      思路解析
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       初始化
      </strong>
      <code>
       first
      </code>
      和
      <code>
       second
      </code>
      为正无穷大，表示目前找到的最小和次小元素。
     </li>
     <li>
      <strong>
       遍历数组
      </strong>
      ：
      <ul>
       <li>
        若
        <code>
         num &lt;= first
        </code>
        ，更新
        <code>
         first
        </code>
        ，表示找到了更小的数。
       </li>
       <li>
        若
        <code>
         num &lt;= second
        </code>
        ，更新
        <code>
         second
        </code>
        ，表示找到了更小的次小数。
       </li>
       <li>
        若
        <code>
         num &gt; second
        </code>
        ，说明已经找到了
        <code>
         third
        </code>
        ，满足
        <code>
         first &lt; second &lt; third
        </code>
        ，返回
        <code>
         True
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       如果遍历结束还没返回
       <code>
        True
       </code>
      </strong>
      ，则返回
      <code>
       False
      </code>
      。
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      时间 &amp; 空间复杂度
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(n)（遍历一次数组）
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1)（只用了两个额外变量）
     </li>
    </ul>
    <p>
     这种方法高效且不需要额外的存储空间，是最优解！
    </p>
