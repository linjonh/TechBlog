    <h3>
     <a id="11__2">
     </a>
     1.1 应用场景
    </h3>
    <p>
     脚本是一种功能扩展代码，用于提供一些编辑器通用功能提供不了的功能，帮助用户在特定工作场景下提高工作效率，几乎所有主流的编辑器、IDE都支持脚本。
     <br/>
     EverEdit的脚本支持
     <code>
      js
     </code>
     (语法与
     <code>
      javascript
     </code>
     类似)、
     <code>
      VBScript
     </code>
     两种编程语言(注：也可以支持其他语言，但较复杂)，EverEdit本身提供了大量对编辑器自身操作的
     <code>
      API
     </code>
     ，通过脚本语言和
     <code>
      API
     </code>
     ，用户可以极大的扩展编辑器的原有功能，这对提高一些特定场景的编辑效率具有非常大的帮助。
     <br/>
     参考：
     <a href="https://www.cnblogs.com/tyysoft/articles/18574808" rel="nofollow">
      EverEdit常用资源汇总
     </a>
    </p>
    <p>
     <strong>
      总体来说
     </strong>
     ：
    </p>
    <ul>
     <li>
      <u>
       <font color="red">
        没有
        <b>
         <code>
          脚本
         </code>
        </b>
        功能的文本编辑器，不算高级文本编辑器
       </font>
      </u>
     </li>
     <li>
      <u>
       <font color="red">
        不会写脚本的用户，永远停留在初级水平。
       </font>
      </u>
     </li>
    </ul>
    <h3>
     <a id="12__12">
     </a>
     1.2 使用方法
    </h3>
    <p>
     安装脚本后，在扩展菜单下会出现脚本菜单，如下图所示：
     <br/>
     <img alt="外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传" src="https://img-home.csdnimg.cn/images/20230724024159.png?origin_url=vx_images%2F431939711814141.png%20%3D200x&amp;pos_id=img-Tlqqhooq-1741603705882"/>
    </p>
    <p>
     脚本的使用方法与普通菜单相同，选择对应的功能菜单即可。
     <br/>
     脚本菜单中，每个菜单项都对应一个“.ejs”脚本文件
     <br/>
     脚本子菜单对应子文件夹，上图中
     <font color="red">
      红框
     </font>
     中的可展开菜单都对应
     <code>
      macro
     </code>
     文件夹下的子文件夹，macro目录结构：
