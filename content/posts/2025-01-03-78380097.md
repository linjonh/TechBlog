     <span style="margin:0px; padding:0px">
      一、 什么是软件性能
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      定义
     </span>
     ：软件的性能是软件的一种
     <span style="margin:0px; padding:0px">
      非功能特性
     </span>
     ，它关注的不是软件是否能够完成特定的功能，而是在完成该功能时展示出来的
     <span style="margin:0px; padding:0px">
      及时性
     </span>
     。
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     由定义可知性能关注的是软件的非功能特性，所以一般来说性能测试介入的时机是在功能测试完成之后。另外，由定义中的及时性可知性能也是一种指标，可以用时间或其它指标来衡量，通常我们会使用某些工具或手段来检测软件的某些指标是否达到了要求，这就是性能测试。
     <span style="margin:0px; padding:0px; line-height:1.5">
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      <span style="margin:0px; padding:0px">
       性能测试定义
      </span>
      ：
     </span>
     指通过自动化的测试工具模拟多种正常、峰值以及异常负载条件来对系统的各项性能指标进行测试。
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">

     2、测试人员在做性能测试时除开要关注表面的现象如响应时间，也需要关注本质，比如用户看不到的服务器资料利用率，架构设计是否合理？代码是否合理等言方方面面。
    </p>

     <span style="margin:0px; padding:0px">
      三、
      <span style="margin:0px; padding:0px">
       性能测试类型
      </span>
     </span>
    </p>
    <ol style="margin:0px; padding:0px 0px 0px 40px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <li style="margin:0px 0px 1em; padding:0px; list-style:decimal">
      基准测试：在给系统施加较低压力时，查看系统的运行状况并记录相关数做为基础参考
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:decimal">
      负载测试：是指对系统不断地增加压力或增加一定压力下的持续时间，直到系统的某项或多项性能指标达到安全临界值，例如某种资源已经达到饱和状态等 。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:decimal">
      压力测试：
      <span style="margin:0px; padding:0px; line-height:1.5">
       压力测试是评估系统处于或超过预期负载时系统的运行情况，关注点在于系统在峰值负载或超出最大载荷情况下的处理能力。
      </span>
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:decimal">
      稳定性测试：在给系统加载一定业务压力的情况下，使系统运行一段时间，以此检测系统是否稳定。
     </li>
     <li style="margin:0px 0px 1em; padding:0px; list-style:decimal">
      并发测试：测试多个用户同时访问同一个应用、同一个模块或者数据记录时是否存在死锁或者其他性能问题，
     </li>
    </ol>

     a）定义：从用户发送一个请求到用户接收到服务器返回的响应数据这段时间就是响应时间
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">

    </p>

     c) 计算方法：Response time = (N1+N2+N3+N4)+ (A1+A2+a3)，即：（网络时间 + 应用程序处理时间）

    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      2、吞吐量
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      a）定义：
     </span>
     单位时间内系统处理的客户端请求的数量
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      b）计算单位：一般使用
     </span>
     <span style="margin:0px; padding:0px; color:rgb(255,0,0)">
      请求数／秒
      <span style="margin:0px; padding:0px; color:rgb(0,0,0)">
       做为吞吐量的单位，出可以使用
      </span>
     </span>
     页面数／秒表表示。
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      另外，从业务角度来说也可以使用
     </span>
     访问人数 /天 或 页面访问量／天 做为单位。

     <span style="margin:0px; padding:0px; line-height:1.5">
      c）计算方法：
     </span>
     Throughput = (number of requests) / (total time).

    </p>

     3、通常可以利用拐点来进行性能测试分析与定位
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      3、并发数：
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      并发用户数
     </span>
     ：某一物理时刻同时向系统提交请求的用户数，提交的请求可能是同一个场景或功能，也可以是不同场景或功能。
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      在线用户数
     </span>
     ：某段时间内访问系统的用户数，这些用户并不一定同时向系统提交请求
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      系统用户数
     </span>
     ：系统注册的总用户数据
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      三者之间的关系：
     </span>
     系统用户数 &gt;= 在线用户数 &gt;= 并发用户数
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      4、资源利用率
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      a) 定义：
     </span>
     指的是对不同系统资源的使用程度，通常以占用最大值的百分比来衡量
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      b) 通常需要关注的服务器资源如下：
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     1、CPU：就像人的大脑，主要负责相关事情的判断以及实际处理的机制
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      2、内存：大脑中的记忆块区，将眼睛，皮肤等收集到的信息记录起来的地方，以供cpu进行判断，但是是临时的，访问速度快，如果关机或断电这里的数据会消失。
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      3、磁盘IO：大脑中的记忆区块，将重要的数据保存起来（永久保存，关机或断电不会丢失，速度慢），以便将来再次使用这些数据。
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      4、网络：
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
      c）资源利用-负载对应关系：
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px; line-height:1.5">
     </span>
     <span style="margin:0px; padding:0px; line-height:1.5">
      <img alt="" height="234" src="https://i-blog.csdnimg.cn/blog_migrate/ff2f2e0c83ae21cca19ec1f7246b426e.png" style="margin:0px; padding:0px; border:0px; max-width:900px" width="356"/>
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     图中拐点说明：
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     1、服务器某荐资源使用逐渐达到饱和
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     2、通常可以利用拐点来进行性能测试分析与定位
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      5、其它常用概念：
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     a) TPS：Transactions Per Second，每秒事务数
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     b)
     <span style="margin:0px; padding:0px">
      思考时间：
     </span>
     用户每个操作后的暂停时间，或者叫操作之间的间隔时间，此时间内是不对服务器产生压力的
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     c)
     <span style="margin:0px; padding:0px">
      点击数
     </span>
     ：每秒钟用户向WEB服务器提交的HTTP请求数。这个指标是WEB应用特有的一个指标:WEB应用是"请求-响应"模式,用户发出一次申请,服务器就要处理一次，所以点击是WEB应用能够处理的交易的最小单位。如果把每次点击定义为一个交易，点击率和TPS就是一个概念。容易看出，点击率越大，对服务器的压力越大。点击率只是一个性能参考指标，重要的是分析点击时产生的影响。需要注意的是，这里的点击并非指鼠标的一次单击操作，因为在一次单击操作中，客户端可能向服务器发出多个HTTP请求.
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     d）PV：访问一个URL，产生一个PV（Page View，页面访问量），每日每个网站的总PV量是形容一个 网站规模的重要指标。
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     UV：作为一个独立的用户，访问站点的所有页面均算作一个UV（Unique Visitor，用户访问）
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
      六、理发店模型和曲线拐点模型
     </span>
    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; font-family:Verdana,Arial,Helvetica,sans-serif; font-size:14px; background-color:rgb(238,238,238)">
     <span style="margin:0px; padding:0px">
     </span>
     上面介绍了很多性能测试中的基本概念，比较抽象，可以通过
     <span style="margin:0px; padding:0px; background-color:rgb(255,204,0)">
