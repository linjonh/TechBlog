    <p>
     在Spring Boot中，
     <code>
      MultipartFile
     </code>
     是处理文件上传的核心类，通常用于接收客户端上传的文件数据。以下是详细的使用说明：
    </p>
    <hr/>
    <h4>
     <a id="1__4">
     </a>
     <strong>
      1. 基本概念
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       MultipartFile
      </code>
      是Spring框架提供的一个接口，封装了上传文件的所有信息（如文件内容、文件名、类型等）
     </li>
     <li>
      通常与
      <code>
       @RequestParam
      </code>
      或
      <code>
       @ModelAttribute
      </code>
      结合使用，在Controller中接收文件
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2__10">
     </a>
     <strong>
      2. 前置配置
     </strong>
    </h4>
    <p>
     在
     <code>
      application.properties
     </code>
     中配置文件上传参数：
    </p>
    <pre><code class="prism language-properties"># 单个文件最大大小（默认1MB）
spring.servlet.multipart.max-file-size=10MB
# 单次请求总文件最大大小（默认10MB）
spring.servlet.multipart.max-request-size=50MB
# 文件存储临时目录（可选）
spring.servlet.multipart.location=/tmp
</code></pre>

      9. 常见问题解决
     </strong>

    <ol>
     <li>
      <p>
       <strong>
        文件上传后大小为0
       </strong>
      </p>
      <ul>
       <li>
        检查表单是否设置
        <code>
         enctype="multipart/form-data"
        </code>

       <li>
        检查是否使用了正确的
        <code>
         @RequestParam
        </code>
        名称
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        中文文件名乱码
       </strong>
       <br/>
       在
       <code>
        application.properties
       </code>
       添加：
      </p>
      <pre><code class="prism language-properties">spring.servlet.multipart.resolve-lazily=true
