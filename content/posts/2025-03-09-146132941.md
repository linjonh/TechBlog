    <h2>
     <a id="conda_0">
     </a>
     conda常用指令整理
    </h2>
    <p>
     此笔记涵盖conda常用指令，pip、命令行等常用指令。
    </p>
    <h3>
     <a id="1conda_2">
     </a>
     1.查看conda版本
    </h3>
    <p>
     在
     <code>
      Anaconda PowerShell Prompt
     </code>
     中输入
    </p>
    <pre><code class="prism language-bash">conda <span class="token parameter variable">--version</span>
</code></pre>
    <h3>
     <a id="2python_7">
     </a>
     2.创建新python虚拟环境
    </h3>
    <p>
     此python虚拟环境会存储在conda的
     <code>
      envs
     </code>
     中名为
     <code>
      project_name
     </code>
     的文件夹下
    </p>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">--name</span> project_name <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.11</span>
</code></pre>
    <p>
     3.11可替换为其他制定版本
    </p>
    <h3>
     <a id="3python_13">
     </a>
     3.激活python虚拟环境
    </h3>
    <pre><code class="prism language-bash">conda activate project_name
</code></pre>
    <h3>
     <a id="4_17">
     </a>
     4.安装项目制定的依赖包
    </h3>
    <p>
     从GitHub上clone下来的项目往往会有
     <code>
      requirements.txt
     </code>
     文件，制定该项目需要哪些包，我们可以通过此文件一键安装所有包：
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token parameter variable">-r</span> requirements.txt
</code></pre>
    <h3>
     <a id="5pip_22">
     </a>
     5.为pip工具配置源
    </h3>
    <p>
     在没有魔法时，为了加快python包的下载速度，我们通常会替换pip的下载源：
    </p>
    <pre><code class="prism language-bash">pip config <span class="token builtin class-name">set</span> global.index-url 镜像源地址
</code></pre>
    <pre><code class="prism language-bash">常用的 PyPI 镜像源
以下是一些常用的 PyPI 镜像源地址，你可以根据需要选择：
阿里云
https://mirrors.aliyun.com/pypi/simple/
清华
https://pypi.tuna.tsinghua.edu.cn/simple/
中国科学技术大学
https://pypi.mirrors.ustc.edu.cn/simple/
豆瓣
https://pypi.doubanio.com/simple/
官方源（默认）
https://pypi.org/simple/
</code></pre>
    <h3>
     <a id="6python_41">
     </a>
     6.使用临时源安装python包
    </h3>
    <p>
     有时候，某一个镜像源仓库中没有某一个特定的包，这时我们可以尝试临时在其他源中安装此包，如：
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token assign-left variable">tb_nightly</span><span class="token operator">==</span><span class="token number">2.9</span>.0a20220309 <span class="token parameter variable">-i</span> https://mirrors.aliyun.com/pypi/simple
</code></pre>
    <h3>
     <a id="7_47">
     </a>
     7.停用虚拟环境
    </h3>
    <p>
     关闭虚拟环境及项目相关服务：
    </p>
    <pre><code class="prism language-bash">conda deactivate
</code></pre>
    <h3>
     <a id="8_52">
     </a>
     8.查看项目"文件树"
    </h3>
    <p>
     查看项目的结构及包含的所有文件
    </p>
    <h4>
     <a id="Linux__macOS_54">
     </a>
     <strong>
      Linux / macOS（终端）
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        显示当前目录下的所有文件和文件夹（包括隐藏文件）
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-la</span>
</code></pre>
      <ul>
       <li>
        <code>
         ls
        </code>
        ：列出文件
       </li>
       <li>
        <code>
         -l
        </code>
        ：显示详细信息（如权限、所有者、大小等）
       </li>
       <li>
        <code>
         -a
        </code>
        ：显示所有文件，包括隐藏文件（以
        <code>
         .
        </code>
        开头）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        以树状结构显示文件夹及文件（需安装
        <code>
         tree
        </code>
        命令）
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash">tree
</code></pre>
      <p>
       如果
       <code>
        tree
       </code>
       未安装，可以使用：
      </p>
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> tree  <span class="token comment"># Ubuntu/Debian</span>
brew <span class="token function">install</span> tree      <span class="token comment"># macOS（需 Homebrew）</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        递归显示所有子目录及文件
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-R</span>
</code></pre>
      <p>
       这个命令会列出当前目录下的所有文件和子目录，并递归显示它们的内容。
      </p>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="WindowsCMD__81">
     </a>
     <strong>
      Windows（CMD ）
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        显示当前目录下的所有文件和文件夹
       </strong>
       ：
      </p>
      <pre><code class="prism language-cmd">dir /a
</code></pre>
      <ul>
       <li>
        <code>
         /a
        </code>
        ：显示包括隐藏文件在内的所有文件
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        以树状结构显示文件夹及文件
       </strong>
       ：
      </p>
      <pre><code class="prism language-cmd">tree /F /A
</code></pre>
      <ul>
       <li>
        <code>
         /F
        </code>
        ：显示文件
       </li>
       <li>
        <code>
         /A
        </code>
        ：使用 ASCII 字符（适用于不支持 Unicode 的终端）
       </li>
      </ul>
     </li>
    </ol>
