    <h6>
     1、计算英文句子的单词重量
    </h6>
    <p>
     每个句子由多个单词组成，句子中的每个单词的长度都可能不一样，我们假设每个单词的长度Ni为该单词的重量，你需要做的就是给出整个句子的平均重量V。
    </p>
    <h6>
     解答要求
    </h6>
    <p>
     时间限制：2000ms
    </p>
    <p>
     内存限制：2000MB
    </p>
    <h6>
     输入
    </h6>
    <p>
     输入只有一行，包含一个字符串S（长度不会超过100），代表整个句子，句子中只包含大小写的英文字母，每个单词之间有一个空格。
    </p>
    <h6>
     输出
    </h6>
    <p>
     输出句子S的平均重量V（四舍五入保留两位小数）。
    </p>
    <h6>
     样例
    </h6>
    <p>
     输入：Who Love Solo
    </p>
    <p>
     输出：3.67
    </p>
    <h6>
     ArkTs代码
    </h6>
    <pre><code class="language-javascript">process.stdin.resume();
process.stdin.setEncoding('utf-8');
let input = '';
process.stdin.on('data', (data) =&gt; {
  input += data;
});
process.stdin.on('end', () =&gt; {
  let inputArray = input.split('\n');
  /**
   * 待实现函数，在此函数中填入答案代码
   * doFunc()
   */
  function doFunc() {
    let wordArr = inputArray[0].split(' ');
    let sumLength = 0;
    for(let i = 0; i &lt; wordArr.length; i++) {
      sumLength += wordArr[i].length;
    }
    console.log((sumLength / wordArr).toFixed(2));
  }
  doFunc();
  process.exit();
})
</code></pre>
