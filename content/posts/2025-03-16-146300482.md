# VMware安装Centos

> centos7 下载地址：https://mirrors.aliyun.com/centos/7/isos/x86_64/

## 1\. centos7 网络配置

设置网络适配器的时候，设置是NAT模式

### 1.1 固定IP设置

  * 确保Centos 为 NAT 模式
  * 在VMware界面(管理员方式启动) 点击**编辑** 里的**虚拟网络编辑器** ，按如下设置  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/6f95c073963241609a8342d20834ad54.png)

### 1.2 启动虚拟机

进入网络配置文件目录

    
    
    cd /etc/sysconfig/network-scripts/
    

编辑ifcfg-ens33 文件:

    
    
    vim ifcfg-ens33
    

内容如下

    
    
    TYPE="Ethernet"
    PROXY_METHOD="none"
    BROWSER_ONLY="no"
    BOOTPROTO=static  # 启用静态IP地址
    DEFROUTE="yes"
    IPV4_FAILURE_FATAL="no"
    IPV6INIT="yes"
    IPV6_AUTOCONF="yes"
    IPV6_DEFROUTE="yes"
    IPV6_FAILURE_FATAL="no"
    IPV6_ADDR_GEN_MODE="stable-privacy"
    NAME="ens33"
    UUID="bb703380-c8c8-4fd9-9aa1-34dd59a1e014"
    DEVICE="ens33"
    ONBOOT="yes"  # 开启自动启用网络连接
    IPADDR=192.168.159.129  # 设置固定IP地址
    NETMASK=255.255.255.0   # 子网掩码
    GATEWAY=192.168.159.2  # 设置网关
    DNS1=8.8.8.8
    DNS2=114.114.114.114
    DNS3=223.5.5.5
    PREFIX=24
    
    

保存退出

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/922679de81bb4fb3a03e5729bd0cd8e3.png)

### 1.3 重启网卡让网络设置生效

    
    
    service network restart
    

### 检查 IP

    
    
    ip addr
    

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/d7f76e59849e4d708fb94674750fd7f0.png)

### 1.4 验证是否可以访问外网

    
    
    ping www.baidu.com
    

### 设置DNS

    
    
    国内移动、电信和联通通用的DNS：114.114.114.114。
    阿里：首选：223.5.5.5 备用：223.6.6.6
    百度 ：180.76.76.76
    腾讯：首选：119.29.29.29，备用：119.28.28.28
    谷歌 8.8.8.8
