    <h2>
     一、路由器的作用
    </h2>
    <p>
     路由器（Router）是网络层（OSI第三层）的核心设备，承担着跨网络通信的关键任务，主要功能包括：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        跨网络数据转发
       </strong>
       <br/>
       基于IP地址，在不同网络间选择最优路径传输数据包（如连接局域网与互联网）922。例如，家庭路由器将内网设备的数据转发至互联网服务提供商（ISP）网络。
      </p>
     </li>
     <li>
      <p>
       <strong>
        网络地址转换（NAT）
       </strong>
       <br/>
       将内网私有IP（如
       <code>
        192.168.1.10
       </code>
       ）转换为公网IP，实现多设备共享单一公网IP访问互联网719。
      </p>
     </li>
     <li>
      <p>
       <strong>
        子网划分与广播域隔离
       </strong>
       <br/>
       通过划分不同子网（如财务部与市场部），隔离广播域，避免广播风暴影响全网性能1622。
      </p>
     </li>
     <li>
      <p>
       <strong>
        安全防护
       </strong>
       <br/>
       提供防火墙、访问控制列表（ACL）等功能，过滤恶意流量并限制非法访问1119。
      </p>
     </li>
     <li>
      <p>
       <strong>
        动态路由管理
       </strong>
       <br/>
       支持RIP、OSPF、BGP等协议，动态更新路由表以适应网络拓扑变化16。
      </p>
     </li>
    </ol>
    <hr/>
    <h2>
     二、路由器的分类
    </h2>
    <p>
     根据功能、场景和性能，路由器可分为以下几类：
    </p>
    <h4>
     1.
     <strong>
      按功能划分
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        基础路由器
       </strong>
       ：仅支持基本路由功能，适用于小型网络19。
      </p>
     </li>
     <li>
      <p>
       <strong>
        智能路由器
       </strong>
       ：集成QoS（服务质量控制）、VPN、家长控制等高级功能，如家用Wi-Fi 6路由器19。
      </p>
     </li>
    </ul>
    <h4>
     2.
     <strong>
      按应用场景划分
     </strong>
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        类型
       </th>
       <th>
        特点
       </th>
       <th>
        典型场景
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         家用路由器
        </strong>
       </td>
       <td>
        集成无线AP和交换机功能，支持NAT和简单防火墙
       </td>
       <td>
        家庭宽带共享、智能家居
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         企业路由器
        </strong>
       </td>
       <td>
        支持多WAN口负载均衡、VLAN间路由、高并发连接
       </td>
       <td>
        中型企业核心网络
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         核心路由器
        </strong>
       </td>
       <td>
        处理海量数据流量，支持BGP等复杂协议，用于ISP主干网
       </td>
       <td>
        互联网骨干节点
       </td>
      </tr>
     </tbody>
    </table>

     <strong>
      按接口类型划分
     </strong>

    <ul>

      <p>
       <strong>
        有线路由器
       </strong>
       ：提供稳定有线连接，适合对延迟敏感的场景（如企业服务器）19。
      </p>
     </li>
     <li>
      <p>
       <strong>
        无线路由器
       </strong>
       ：集成Wi-Fi功能，支持手机、平板等无线设备接入19。

    <h2>
     三、路由器与交换机的区别
    </h2>
    <p>
     路由器和交换机是网络中的“黄金搭档”，但二者在多个维度存在本质差异：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        <strong>
         对比维度
        </strong>
       </th>
       <th>
        <strong>
         路由器
        </strong>
       </th>
       <th>
        <strong>
         交换机
        </strong>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         工作层级
        </strong>
       </td>
       <td>
        网络层（OSI第三层），处理IP地址
       </td>
       <td>
        数据链路层（OSI第二层），处理MAC地址911
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         核心功能
        </strong>
       </td>
       <td>
        跨网络通信、NAT、防火墙、动态路由
       </td>
       <td>
        局域网内高速数据转发、MAC地址学习、冲突域隔离1222
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         数据转发依据
        </strong>
       </td>
       <td>
        基于IP地址和路由表
       </td>
       <td>
        基于MAC地址表1116
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         广播域处理
        </strong>
       </td>
       <td>
        隔离广播域，每个接口对应一个子网
       </td>
       <td>
        不隔离广播域，所有端口属于同一广播域（除非支持VLAN）1216
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         典型应用场景
        </strong>
       </td>
       <td>
        连接不同网络（如家庭网络与互联网）
       </td>
       <td>
        构建局域网（如办公室多设备互联）22
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         安全性
        </strong>
       </td>
       <td>
        支持高级安全策略（如ACL、VPN）
       </td>
       <td>
        仅支持基础MAC过滤和端口安全916
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         设备成本
        </strong>
       </td>
       <td>
        功能复杂，价格较高
       </td>
       <td>
        功能单一，价格较低1119
       </td>
      </tr>
     </tbody>
    </table>
    <h4>
     补充说明：
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        三层交换机
       </strong>
       ：融合路由器与交换机的特性，支持基于IP的VLAN间路由，适合大型企业内网16。
      </p>
     </li>
     <li>
      <p>
       <strong>
        网络扩展
       </strong>
       ：路由器通过WAN口连接广域网，交换机通过LAN口扩展局域网设备数量1819。
      </p>
     </li>
    </ul>
