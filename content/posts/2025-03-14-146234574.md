    <p>
     算术运算符：
    </p>
    <p>
     加法运算符（+）
     <br/>
     用于计算两个数值的和。
     <br/>
     示例：
    </p>
    <pre><code class="language-sql">SELECT 100+'1'
FROM dual;
/*结果为101*/
SELECT 100+'A'
FROM dual;
/*结果为100，A当作0*/
SELECT 100+NULL
FROM dual;
/*跟NULL参与运算结果都为null*/</code></pre>
    <p>
     <br/>
     2. 减法运算符（-）
     <br/>
     用于计算两个数值的差。
     <br/>
     示例：
    </p>
    <pre><code class="language-sql">SELECT price - 2 AS discounted_price
FROM products;
</code></pre>
    <p>
     <br/>
     此代码将 products 表中 price 列的值都减去 2，并将结果以 discounted_price 列名返回。
     <br/>
     3. 乘法运算符（*）
     <br/>
     用于计算两个数值的乘积。
     <br/>
     示例：
     <br/>
     sql
    </p>
    <pre><code class="language-sql">SELECT price * 1.1 AS price_with_tax
FROM products;</code></pre>
    <p>
     <br/>
     该代码将 products 表中 price 列的值都乘以 1.1，模拟加上 10% 的税，并将结果以 price_with_tax 列名返回。
     <br/>
     4. 除法运算符（/）
     <br/>
     用于计算两个数值的商。
     <br/>
     示例：
     <br/>
    </p>
    <pre><code class="language-sql">SELECT total_sales / num_customers AS average_sale
FROM sales_summary;</code></pre>
    <p>
     <br/>
     此代码将 sales_summary 表中 total_sales 列的值除以 num_customers 列的值，计算平均销售额，并将结果以 average_sale 列名返回。
     <br/>
     5. 取模运算符（% 或 MOD() 函数）
     <br/>
     用于计算两个数值相除的余数。不同数据库系统对取模运算符的支持可能有所不同，有些数据库使用 %，有些则使用 MOD() 函数。
     <br/>
     示例（使用 %）：
    </p>
    <pre><code class="language-sql">SELECT quantity % 3 AS remainder
FROM inventory;</code></pre>
    <p>
     示例（使用 MOD() 函数）：
     <br/>
    </p>
    <pre><code class="language-sql">SELECT MOD(quantity, 3) AS remainder
FROM inventory;</code></pre>
    <p>
     上述代码将 inventory 表中 quantity 列的值除以 3，取其余数，并将结果以 remainder 列名返回。
    </p>
