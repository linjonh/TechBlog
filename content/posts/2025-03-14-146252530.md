    <h2>
     <a id="Linux_updatealternatives__GCC__1">
     </a>
     Linux
     <code>
      update-alternatives
     </code>
     管理 GCC 版本指南
    </h2>
    <p>
     在 Linux 系统（如 Ubuntu 或 Debian）中，
     <code>
      update-alternatives
     </code>
     是一个用于管理同一功能的不同版本的工具。本文将详细讲解如何使用
     <code>
      update-alternatives
     </code>
     管理
     <code>
      gcc
     </code>
     （GNU C Compiler）的不同版本。
    </p>
    <h3>
     <a id="1__5">
     </a>
     1. 基本概念
    </h3>
    <p>
     <code>
      update-alternatives
     </code>
     允许用户在多个版本的同一程序之间进行选择。例如，在安装了
     <code>
      gcc-8
     </code>
     、
     <code>
      gcc-9
     </code>
     和
     <code>
      gcc-10
     </code>
     的情况下，可以通过该工具自由切换默认版本。
    </p>
    <h3>
     <a id="2__GCC__updatealternatives_8">
     </a>
     2. 添加 GCC 版本到
     <code>
      update-alternatives
     </code>
    </h3>
    <h4>
     <a id="_10">
     </a>
     命令格式：
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> <span class="token operator">&lt;</span>主路径<span class="token operator">&gt;</span> <span class="token operator">&lt;</span>名称<span class="token operator">&gt;</span> <span class="token operator">&lt;</span>具体版本路径<span class="token operator">&gt;</span> <span class="token operator">&lt;</span>优先级<span class="token operator">&gt;</span>
</code></pre>
    <h4>
     <a id="_15">
     </a>
     示例：
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-8 <span class="token number">100</span>
</code></pre>
    <ul>
     <li>
      <code>
       /usr/bin/gcc
      </code>
      ：主路径，表示
      <code>
       gcc
      </code>
      命令的默认路径。
     </li>
     <li>
      <code>
       gcc
      </code>
      ：替代项的名称。
     </li>
     <li>
      <code>
       /usr/bin/gcc-8
      </code>
      ：具体的
      <code>
       gcc-8
      </code>
      路径。
     </li>
     <li>
      <code>
       100
      </code>
      ：优先级（值越高，优先级越高）。
     </li>
    </ul>
    <h4>
     <a id="_gcc9__gcc10_24">
     </a>
     如果要添加
     <code>
      gcc-9
     </code>
     或
     <code>
      gcc-10
     </code>
     ：
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-9 <span class="token number">150</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-10 <span class="token number">200</span>
</code></pre>
    <h3>
     <a id="3__GCC__30">
     </a>
     3. 查看当前 GCC 版本候选项
    </h3>
    <p>
     使用以下命令查看已添加的
     <code>
      gcc
     </code>
     版本：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> gcc
</code></pre>
    <p>
     示例输出：
    </p>
    <pre><code>有 3 个候选项用于 gcc (提供 /usr/bin/gcc)。

  选择      路径            优先级   状态
------------------------------------------------------------
* 1          /usr/bin/gcc-8   100     自动模式
  2          /usr/bin/gcc-9   150
  3          /usr/bin/gcc-10  200
</code></pre>
    <h3>
     <a id="4__GCC__47">
     </a>
     4. 手动切换 GCC 版本
    </h3>
    <p>
     在上述选项中输入所需版本的编号。例如，选择
     <code>
      gcc-10
     </code>
     ：
    </p>
    <pre><code class="prism language-bash"><span class="token number">3</span>
</code></pre>
    <h3>
     <a id="5__54">
     </a>
     5. 直接设置默认版本
    </h3>
    <p>
     如果希望直接将某个版本设置为默认版本，而不进入选择界面，可以使用
     <code>
      --set
     </code>
     命令：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--set</span> gcc /usr/bin/gcc-9
</code></pre>
    <h3>
     <a id="6__GCC__61">
     </a>
     6. 验证当前 GCC 版本
    </h3>
    <p>
     使用以下命令检查当前使用的
     <code>
      gcc
     </code>
     版本：
    </p>
    <pre><code class="prism language-bash">gcc <span class="token parameter variable">--version</span>
</code></pre>
    <h3>
     <a id="7__G__68">
     </a>
     7. 同时管理 G++ 版本
    </h3>
    <p>
     如果还需要管理
     <code>
      g++
     </code>
     （C++ 编译器）的版本，可以按照同样的方法：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/g++ g++ /usr/bin/g++-9 <span class="token number">150</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> g++
</code></pre>
    <h3>
     <a id="8__76">
     </a>
     8. 移除某个版本
    </h3>
    <p>
     如果希望移除某个
     <code>
      gcc
     </code>
     版本，可以使用以下命令：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--remove</span> gcc /usr/bin/gcc-8
</code></pre>
    <h3>
     <a id="9__83">
     </a>
     9. 总结
    </h3>
    <p>
     <code>
      update-alternatives
     </code>
     是一个强大且灵活的工具，适用于管理多个版本的
     <code>
      gcc
     </code>
     ，方便开发人员在不同版本之间切换，满足不同项目对编译器版本的需求。
    </p>
