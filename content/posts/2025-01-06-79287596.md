# 前端框架三巨头年度走势对比：Vue 增长率最高

Javascriptreport 近日发布了一篇
[文章](http://javascriptreport.com/javascript-frameworks-by-the-numbers-winter-2018/)
来比较流行前端框架的走势。走势图是通过统计最新的数据，并与前一年的数据进行比较而生成。数据来源于
[npm trends](http://www.npmtrends.com/)
的每日下载量。

前端框架三巨头 React、Angular 和 Vue ，虽然都很受欢迎，且保持着上升趋势，但 Vue 爆发力最强，但在使用率上，仍低于 React 和 Angular 。

![](https://i-blog.csdnimg.cn/blog_migrate/9947691f3390fa7e69bed1ceebbcccdb.png)

如果按目前的趋势继续下去的话，Vue 或将会在 2018 年底或 2019 年初取代 Angular ，成为使用率第二大的框架。

![](https://i-blog.csdnimg.cn/blog_migrate/b3e76a079761792d49cebaa37a6376aa.png)

另外三个老牌但较有影响力的框架，AngularJS、Backbone 和 Ember ，虽然如今在知名度上弱于前面的三巨头，但过去一年仍保持增长趋势。

![](https://i-blog.csdnimg.cn/blog_migrate/beda367197ce624012e3b0746bcb4256.png)

有意思的是，你会发现 AngularJS 实际上比 Vue 的下载量更高。Vue 在去年年底超过了 AngularJS  ，但在今年 1 月的最新数据中，AngularJS 出现了较大的波动。当然，这不会影响 Vue 的长期趋势。

![](https://i-blog.csdnimg.cn/blog_migrate/a105a837334f18688c80b62979e86a74.png)

下面同样是非常优秀的一组框架对比：Preact、Inferno、Aurelia、Polymer 和 Svelte 。

![](https://i-blog.csdnimg.cn/blog_migrate/596aee3f234a73f8d36b3e7bcf3fe2de.png)

其中，Inferno、Preact 和 Svelte 的增长率仅低于 Vue 。Preact 是该梯队的领导者，有望成为最受欢迎的 React 替代品。

![](https://i-blog.csdnimg.cn/blog_migrate/9b60706b630eb89ef025360d5e844f49.png)
