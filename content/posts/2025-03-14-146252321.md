    <h2>
     <a id="_ohosarkuidrawableDescriptor_DrawableDescriptor_0">
     </a>
     鸿蒙 @ohos.arkui.drawableDescriptor (DrawableDescriptor)
    </h2>
    <p>
     在鸿蒙开发中，
     <code>
      @ohos.arkui.drawableDescriptor
     </code>
     模块提供了一种强大的方式来处理图片资源，包括普通图片和分层图片（LayeredDrawableDescriptor）。通过这个模块，开发者可以获取图片的
     <code>
      PixelMap
     </code>
     ，并对其进行裁剪、融合等操作。
    </p>
    <hr/>
    <h3>
     <a id="_6">
     </a>
     一、模块功能概述
    </h3>
    <p>
     <code>
      @ohos.arkui.drawableDescriptor
     </code>
     模块提供了以下功能：
    </p>
    <ul>
     <li>
      <strong>
       获取 PixelMap
      </strong>
      ：支持从普通图片或分层图片中获取
      <code>
       PixelMap
      </code>
      。
     </li>
     <li>
      <strong>
       分层图片处理
      </strong>
      ：支持从 JSON 文件中加载分层图片，包括前景、背景和蒙版。
     </li>
     <li>
      <strong>
       裁剪路径
      </strong>
      ：提供系统内置的裁剪路径参数，用于图片裁剪。
     </li>
    </ul>
