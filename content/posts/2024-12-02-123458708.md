# 数据库\_Redis\_详解\_00

### 背景

Redis是一个基于键值对（Key-Value）的数据结构，可以用来作为数据库、缓存和消息队列。

### 常用数据结构

* string 字符串
* list 列表
* hash 哈希
* set 集合
* zset 有序集合
* bitmap 位图
