    <h2>
     <a id="1_2">
     </a>
     1、阿里云平台的操作流程
    </h2>
    <h3>
     <a id="_3">
     </a>
     账户注册登录
    </h3>
    <p>
     登录阿里云物联网平台
     <code>
      https://iot.console.aliyun.com/product
     </code>
     ，有账户的直接登录，没有账户的点击登录按钮下发的“免费注册”按钮注册账号，填写用户名、用户密码、有效手机号码，点击获取验证码，查看短信验证码，并完成注册；完成后登录，登录后，进入即可
     <code>
      物联网平台/设备管理/产品
     </code>
    </p>
    <h3>
     <a id="_6">
     </a>
     创建产品和设备
    </h3>
    <p>
     参考文档：
     <code>
      https://help.aliyun.com/document_detail/73705.html?spm=a2c4g.11186623.6.579.55423091pFAcCB
     </code>
     <br/>
     在物联网平台（
     <code>
      https://iot.console.aliyun.com/lk/summary/new
     </code>
     ）实例概况页，点击公共实例—&gt;创建产品—&gt;新建产品，填写产品名称、所属品类、节点类型、联网方式、数据格式等，具体填写方式参考文档：
     <code>
      https://help.aliyun.com/document_detail/73705.html?spm=a2c4g.11186623.6.579.55423091pFAcCB
     </code>
     ，点击确定后会跳转到添加设备页面，点击添加设备，选择产品、DeviceName、备注名称，创建设备成功后，在弹出的添加完成对话框，单击前往查看或一键复制设备证书，获取设备证书。
    </p>
    <blockquote>
     <p>
      设备证书包含ProductKey、DeviceName和DeviceSecret。设备证书是设备后续与物联网平台交流的重要凭证，请妥善保管
     </p>
    </blockquote>
    <h2>
     <a id="2MQTTfx_12">
     </a>
     2、MQTT.fx连接测试
    </h2>
    <p>
     MQTT客户端下载链接：
     <code>
      http://www.jensd.de/apps/mqttfx/1.7.1/
     </code>
    </p>
    <h3>
     <a id="_15">
     </a>
     客户端配置
    </h3>
    <p>
     打开MQTT.fx客户端，进入客户端配置页面，参考链接：
     <code>
      https://help.aliyun.com/document_detail/140507.html?spm=a2c4g.11186623.6.584.408215daZBUNSa
     </code>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/343ab553e75d9d1cea780267df5ab7fa.png">
      <br/>
      配置内容说明
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f3858275cfb1b679196ad966dfb4fc3b.png"/>
     </img>
    </p>
    <p>
     图片标注解析：
     <br/>
     1、新建一个配置文件
     <br/>
     2、配置文件名称自定义
     <br/>
     3、Broker Address（公共实例接入域名）：格式：
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c4c07a537778c81e037a4bac58c1dfbb.png">
      <br/>
      ProductKey是a1NbN0LOpNm，地域为华东2，因此Broker Address为：
      <code>
       a1NbN0LOpNm.iot-as-mqtt.cn-shanghai.aliyuncs.com
      </code>
      <br/>
      4、Broker Port：
      <code>
       1883
      </code>
      <br/>
      5、Client ID格式：
      <img alt="必须为设备名称" src="https://i-blog.csdnimg.cn/blog_migrate/4902f774538ef4ee20dccd1b3cc0fbfd.png">
       <br/>
       自定义设备的ID信息为12345，使用TCP直连安全模式为
       <code>
        securemode=3
       </code>
       算法类型
       <code>
        sha1
       </code>
       因此Client ID：
       <code>
        12345|c,signmethod=hmacsha1|
       </code>
      </img>
     </img>
    </p>
    <p>
     6、User Name格式：
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6df7bf8b2d986cd9784d0be6ed853e2d.png">
      <br/>
      ProductKey是
      <code>
       a1NbN0LOpNm
      </code>
      ，DeviceName是
      <code>
       hr_dev_001
      </code>
      ，因此User Name为：
      <code>
       hr_dev_001&amp;a1NbN0LOpNm
      </code>
      ，
      <br/>
      7、Password：
      <br/>
      需要使用手动拼接方式完成，格式说明：
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7c2aff070f04a1d1a56e18c55cc828f2.png">
       <br/>
       加密工具链接：
       <code>
        [https://1024tools.com/hmac](https://1024tools.com/hmac)
       </code>
       <br/>
       clientId值为
       <code>
        12345
       </code>
       ，deviceName值为
       <code>
        hr_dev_001
       </code>
       ，productKey值为
       <code>
        a1NbN0LOpNm
       </code>
       ，因此拼接后结果：clientId12345deviceNamehr_dev_001productKeya1NbN0LOpNm
       <br/>
       将该结果填入工具的消息中，秘钥使用设备的DeviceSecret，这个就打码了哈
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/811cb8b71b3e464f508c9135a43e7e7d.png">
        <br/>
        点击计算，将结果A作为Password填入配置文件
        <br/>
        注意问题点：
       </img>
      </img>
     </img>
    </p>
    <blockquote>
     <p>
      <strong>
       不能使用
       <code>
        Password生成小工具
       </code>
       这种方式不可用，生成的Password不能连接到阿里云平台，需要使用拼接方式实现
      </strong>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8050937cc0ad5303216c3beba40b39a8.png"/>
     </p>
    </blockquote>
