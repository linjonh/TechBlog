      </span>
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">

      </span>
     </p>

      <br/>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       程序员更看重使用并熟练运用语言的技能，而非关注语言本身的内涵。
      </span>
     </p>
     <p>
      <br/>
     </p>

       如果你对自己的选择很坚定，你成功的机会也会更高。选择哪种语言并不重要。但是一旦你做出决定，就必须坚持下去。

      </span>

       如果你遇到了困难，请在向另一个方向出发前，先尝试解决问题。互联网上很可能已经有人记录过该问题，大概率有人和你遇到过同样的问题。尝试在互联网上搜索解决方法。同时，Python 社区也随时欢迎你。你可以随时向他人求助。
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       关于 Python 要学的太多了，它被应用于许多领域。不要跳过基础知识直接到特定领域的专业知识。
       <strong>
        先学习调包再学习 PyTorch，先学习虚拟环境再学习 Django。
       </strong>
       从长远来看，Python 值得我们深入研究。当你使用了合适的库时，它可以减少调试程序所花的时间。
      </span>
     </p>
     <p>
      <br/>
     </p>
     <p style="text-align:center;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/a1b2089bfe65ab320f17264142df637c.jpeg"/>
     </p>
     <p style="line-height:1.75em;text-align:center;">
      <span style="color:rgb(70,118,217);font-size:18px;">
       <strong>
        <span style="font-size:18px;color:rgb(70,118,217);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;letter-spacing:1px;">
         要花多久？
        </span>
       </strong>
      </span>
     </p>
     <p style="line-height:1.75em;">
      <br/>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       学好一样东西需要时间，比大多数人告诉你的要更长。你无法在几个月内成为专家程序员。建议你重点关注 Python 至少一年时间。尽量避免分心和注意力的转移。
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       不要害怕学习新事物。要成为一个全面的开发人员，只会 Python 是不够的。学习 Markdown 和 HTML 都会有很大收益。你可能还需要在此过程中获取 JS 或统计信息。
      </span>
     </p>

      <br/>
     </p>

      <p style="line-height:1.75em;">
       <span style="color:rgb(136,136,136);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
        永远做
       </span>
       <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;color:rgb(136,136,136);">
        一名学者，但不要只做学者！
       </span>
      </p>

     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">

     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       任何创造或科学探索都有两个阶段不断重复：
      </span>
     </p>
     <p>
      <br/>
     </p>
     <ul class="list-paddingleft-2" style="list-style-type:disc;">
      <li>
       <p style="line-height:1.75em;">
        <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
         在学习阶段，你应该专注于积累知识。阅读书籍，完成每个教程，并探索已经完成的工作。
        </span>
       </p>
      </li>
     </ul>
     <p>
      <br/>
     </p>
     <ul class="list-paddingleft-2" style="list-style-type:disc;">
      <li>
       <p style="line-height:1.75em;">
        <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
         在夯实阶段，你可以通过应用上一阶段得到的知识来创造自己的东西。不要怀疑自己的技能，不要听那些质疑的声音，完成一个你自己的应用。一旦完成了，回去再学习一遍。
        </span>
       </p>
      </li>
     </ul>
     <p>
      <br/>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">
       平衡这两个阶段。学习阶段大约 1 到 2 个月，接下来是一个月的夯实阶段。
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">

        <span style="font-size:18px;color:rgb(70,118,217);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;letter-spacing:1px;">
         寓学于乐
        </span>
       </strong>
      </span>
     </p>
     <p style="line-height:1.75em;">
      <span style="font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:15px;letter-spacing:1px;">

       为了激发自我的学习兴趣，你还需要了解语言的一切，了解所在领域的最新进展，了解其设计理念，寓学于乐！
