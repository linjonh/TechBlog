      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       “我国工业互联网发展在国际竞争中起步较早，但仍处于起步阶段。”中国信息通信研究院的报告认为，“因此，国家鼓励工业互联网首先在产业集聚区落地，通过财税支持、政府购买服务等方式鼓励中小企业业务系统向云端迁移。”
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       工业互联网不可能一蹴而就，全国遍地开花。目前，在浙江、江苏、山东、广东等地，工业互联网的抓手——工业云建设正在如火如荼的进行中。12月12日，作为中国重要的工业产业聚集地的西安迎来了一个好日子，西安工业云平台正式上线，在工业互联网建设中先下一城。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <br/>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;">
      <img class="" src="https://img-blog.csdnimg.cn/img_convert/d0eb841cc75b09d7acc405152ddb6ce9.png;wx_lazy=1" style="width: auto !important; height: auto !important; visibility: visible !important;"/>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: center;line-height: 1.75em;">
      <span style="font-size: 14px;letter-spacing: 1px;color: rgb(136, 136, 136);">
       西安工业云平台上线
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <span style="color: rgb(89, 89, 89);font-size: 14px;letter-spacing: 1px;text-align: justify;">
        西安市副市长强晓安认为，西安工业云平台的发布，标志着西安市在工业与互联网相互融合、云资源共享方面，迈出了坚实的一步。
       </span>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       而作为西安工业云平台的研制和技术服务支持单位，中服软件总经理安贵则认为，消费领域以平台为核心的生态化发展，正在工业互联网领域复制。西安工业云平台就是一个“安全、高效、开放、合作”的专业化工业云平台，是企业两化融合和智能制造服务的专业平台，必将会加快先进制造业发展，加速制造业向智能化转型。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: center;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;font-size: 16px;color: rgb(0, 122, 170);">
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: center;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;font-size: 16px;color: rgb(0, 122, 170);">
        互联网从消费领域迈向生产领域
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       在消费互联网时代，中国后来居上，在技术和产业发展上取得突出的成绩，并引领了移动互联网的发展。目前，中国互联网上市公司高达780多家，市值达到了20多万元。在工业互联网领域，中国能否再现这一辉煌呢？
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: center;">
      <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/4f2a49a994661bfe8760f069d0f34de1.jpeg" style="width: 654px !important; height: 445.9657142857143px !important;">
       <span style="font-size: 14px;letter-spacing: 1px;text-align: justify;color: rgb(136, 136, 136);">
        中服软件总经理安贵在做主题演讲
       </span>
      </img>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
        安贵认为，工业互联网有三个显著的特征：
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       第一，它是连接工业全系统、全产业链、全价值链，支撑工业智能化发展的关键基础设施。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       第二，它是新一代信息技术与制造业深度融合所形成的新兴业态与应用模式。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       第三，它是互联网从消费领域向生产领域、从虚拟经济向实体经济拓展的核心载体。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       当前，全球工业互联网正处在格局未定的关键期和规模化扩张的窗口期，发展工业互联网已经成为主要国家抢占全球产业竞争新制高点、重塑工业体系的一个共同选择。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       那么在工业互联网方面，中国处于一个什么位置呢？中国信息通信研究院的报告认为，我国工业互联网发展起步较早，在框架、标准、测试、安全、国际合作等方面都取得初步进展。但仍存在产业支撑能力不足、标准体系不完整、引领发展的国际型龙头企业缺乏、安全保障能力薄弱等问题，一定程度上制约了我国工业互联网迈向更高发展水平。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       对于我国，大力发展工业互联网，对推动互联网和实体经济深度融合，促进大众创业万众创新和一二三产业、大中小企业融通发展，建设制造强国都具有重大而深远的意义。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       安贵说，工业互联网包括网络、平台、安全三大功能体系，其中网络体系是基础，平台体系是核心，安全体系是保障。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       国务院日前印发的《关于深化“互联网+先进制造业”发展工业互联网的指导意见》就重点突出三大体系构建。在网络基础方面，重点推动企业内外网改造升级，构建标识解析与标准体系，建设低时延、高可靠、广覆盖的网络基础设施，为工业全要素互联互通提供有力支撑。平台体系方面，着力夯实平台发展基础，提升平台运营能力，推动企业上云和工业APP培育，形成“建平台”与“用平台”有机结合、互促共进的良好发展格局。在安全保障方面，着力提升安全防护能力，建立数据安全保护体系，推动安全技术手段建设，全面强化工业互联网安全保障能力。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <img class="img_loading" src="https://i-blog.csdnimg.cn/blog_migrate/ada1713dad3e5ed06bdf24d1ce44c515.jpeg" style="white-space: normal; width: 654px !important; height: 434.80781249999995px !important;"/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: center;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;font-size: 16px;color: rgb(0, 122, 170);">
        工业云平台是一个重要抓手
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       安贵认为，西安工业云平台建设首先应该解决两大痛点：
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
        第一个痛点就是企业上云，继而实现数字化转型和智能制造升级面临的诸多困难。
       </span>
      </strong>
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       未来，上云是常态，不上云是例外。进入数字经济时代，传统产业面临改造升级，新技术、新产业、新业态、新模式不断出现。以云计算为代表的IT技术成为企业数字化转型和实现新旧动能转换的重要驱动力。工信部发布《云计算发展三年行动计划》后，浙江、江苏、山东等多省市纷纷启动企业上云行动，通过企业上云，推动企业转型升级。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       另外，通过企业上云和数字化转型，可以实现制造业的升级，发展个性化订制、制造业服务化等新业态。
      </span>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
        <br/>
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <strong>
       <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
        第二个痛点就是政府主管部门如何准确及时掌握本地区的工业经济运行情况，企业如何了解本行业的经济运行情况。
       </span>
      </strong>
     </p>
     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
      <span style="letter-spacing: 1px;color: rgb(89, 89, 89);font-size: 14px;">
       <br/>
      </span>

     <p style="margin-left: 8px;margin-right: 8px;text-align: justify;line-height: 1.75em;">
