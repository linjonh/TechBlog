    <h2>
     一：创建一个新的数据库（companydb),并查看数据库。
    </h2>
    <p>
     <img alt="" height="289" src="https://i-blog.csdnimg.cn/direct/2594c84f6d7e4b2fbf4c5353a3162a23.png" width="888"/>

    <h2>
     二：使用该数据库，并创建表worker。
    </h2>
    <pre><code class="language-sql">mysql&gt; use companydb;
</code></pre>
    <pre><code class="language-sql">mysql&gt; CREATE TABLE worker(
    -&gt; 部门号 INT(11) NOT NULL,
    -&gt; 职工号 INT(11) NOT NULL,
    -&gt; 工作时间 DATE NOT NULL,
    -&gt; 工资 FLOAT(8,2) NOT NULL,
    -&gt; 政治面貌 VARCHAR(10) NOT NULL DEFAULT '群众',
    -&gt; 姓名 VARCHAR(20) NOT NULL,
    -&gt; 出生日期 DATE NOT NULL,
    -&gt; PRIMARY KEY(职工号）
    -&gt; ）ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 ROW_FORMAT=DYNAMIC;</code></pre>

    <h2>
     四：按要求完成。
    </h2>
    <h3>
     1.显示所有职工的基本信息。
    </h3>
    <pre><code class="language-sql"> SELECT * FROM worker;</code></pre>
    <p>
     <img alt="" height="294" src="https://i-blog.csdnimg.cn/direct/afc2f32b35564fdd885fcbf06cfee132.png" width="1469"/>
    </p>
    <h3>
     2.查询所有职工所属部门的部门号，不显示重复的部门号。
    </h3>
    <pre><code class="language-sql">SELECT DISTINCT 部门号 FROM worker;</code></pre>
    <p>
     <img alt="" height="204" src="https://i-blog.csdnimg.cn/direct/0fe3a2bf72374cf2986c3d914132567d.png" width="962"/>
    </p>
    <h3>
     3.求出所有职工的人数。
    </h3>
    <pre><code class="language-sql">SELECT COUNT(*) AS 总人数 FROM worker;</code></pre>
    <p>
     <img alt="" height="202" src="https://i-blog.csdnimg.cn/direct/95abb349ee4d473bb8ba768486aabb9c.png" width="1036"/>
    </p>
    <h3>
     4.列出最高工资和最低工资。
    </h3>
    <pre><code class="language-sql"> SELECT MAX(工资) AS 最高工资, MIN(工资) AS 最低工资 FROM worker;</code></pre>
    <p>
     <img alt="" height="182" src="https://i-blog.csdnimg.cn/direct/c1354452081e446baf90f45e1830b7cf.png" width="1248"/>
    </p>
    <h3>
     5.列出职工的平均工资和总工资。
    </h3>
    <pre><code class="language-sql">SELECT AVG(工资) AS 平均工资, SUM(工资) AS 总工资 FROM worker;</code></pre>
    <p>
     <img alt="" height="187" src="https://i-blog.csdnimg.cn/direct/95b466ce422444b198cefeb67a8c6aee.png" width="1081"/>
    </p>
    <h3>
     6.创建一个只有职工号、姓名和工作时间的新表，名为工作日期表。
    </h3>
    <pre><code class="language-sql">mysql&gt; CREATE TABLE 工作日期表 AS
    -&gt; SELECT 职工号, 姓名, 工作时间 FROM worker;</code></pre>
    <p>
     <img alt="" height="121" src="https://i-blog.csdnimg.cn/direct/87f6d6ed00e34670b5293a8bbd0c98fe.png" width="1255"/>
    </p>
    <h3>
     7.显示所有女职工的年龄。
    </h3>
    <pre><code class="language-sql">SELECT 职工号, 姓名, CASE WHEN YEAR(出生日期) % 2 = 0 THEN '女' ELSE '男' END AS 性别 FROM worker;</code></pre>
    <h3>
     <img alt="" height="284" src="https://i-blog.csdnimg.cn/direct/9d434f6e87a04f8493b9523e5173b8fb.png" width="1380"/>
     <br/>
     8、列出所有姓刘的职工的职工号、姓名和出生日期。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 职工号, 姓名, 出生日期
    -&gt; FROM worker
    -&gt; WHERE 姓名 LIKE '刘%';</code></pre>
    <h3>
     <img alt="" height="116" src="https://i-blog.csdnimg.cn/direct/5ee42896244a462b993d1c2e4a27fe53.png" width="1229"/>
     <br/>
     9、列出1960年以前出生的职工的姓名、参加工作日期。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 姓名, 工作时间
    -&gt; FROM worker
    -&gt; WHERE YEAR(出生日期) &lt; 1960;
</code></pre>
    <h3>
     <img alt="" height="117" src="https://i-blog.csdnimg.cn/direct/3d7bb02098e6452c8fbf138590ef97ca.png" width="1260"/>
     <br/>
     10、列出工资在1000-2000之间的所有职工姓名。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 姓名
    -&gt; FROM worker
    -&gt; WHERE 工资 BETWEEN 1000 AND 2000;</code></pre>
    <h3>
     <img alt="" height="108" src="https://i-blog.csdnimg.cn/direct/01df3387cbc5446885171172e45422e8.png" width="1084"/>
     <br/>
     11、列出所有陈姓和李姓的职工姓名。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 姓名
    -&gt; FROM worker
    -&gt; WHERE 姓名 LIKE '陈%' OR 姓名 LIKE '李%';</code></pre>
    <h3>
     <img alt="" height="225" src="https://i-blog.csdnimg.cn/direct/0d4916d701404c47883d10ff9ecf049e.png" width="998"/>
     <br/>
     12、列出所有部门号为2和3的职工号、姓名、党员否。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 职工号, 姓名, 政治面貌 AS 党员否
    -&gt; FROM worker
    -&gt; WHERE 部门号 IN (2, 3);</code></pre>
    <h3>
     <img alt="" height="116" src="https://i-blog.csdnimg.cn/direct/6addee0a266c4197a6e32a25576665f8.png" width="1124"/>
     <br/>
     13、将职工表worker中的职工按出生的先后顺序排序。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT *
    -&gt; FROM worker
    -&gt; ORDER BY 出生日期 ASC;</code></pre>
    <h3>
     <img alt="" height="335" src="https://i-blog.csdnimg.cn/direct/fc5b5f2d810f4a81aca5515055286688.png" width="1352"/>
     <br/>
     14、显示工资最高的前3名职工的职工号和姓名。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 职工号, 姓名
    -&gt; FROM worker
    -&gt; ORDER BY 工资 DESC
    -&gt; LIMIT 3;</code></pre>
    <h3>
     <img alt="" height="297" src="https://i-blog.csdnimg.cn/direct/19619f49c2df4811bc7eb083b6a87584.png" width="1112"/>
     <br/>
     15、求出各部门党员的人数。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 部门号, COUNT(*) AS 党员人数
    -&gt; FROM worker
    -&gt; WHERE 政治面貌 = '党员'
    -&gt; GROUP BY 部门号;</code></pre>
    <h3>
     <img alt="" height="263" src="https://i-blog.csdnimg.cn/direct/c2975836385c43b7b882822731580f52.png" width="1180"/>
     <br/>
     16、统计各部门的工资和平均工资。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 部门号, SUM(工资) AS 总工资, AVG(工资) AS 平均工资
    -&gt; FROM worker
    -&gt; GROUP BY 部门号;</code></pre>
    <h3>
     <img alt="" height="269" src="https://i-blog.csdnimg.cn/direct/c743d833653e41908ecf491556d014ae.png" width="1426"/>
     <br/>
     17、列出总人数大于4的部门号和总人数。
    </h3>
    <pre><code class="language-sql">mysql&gt; SELECT 部门号, COUNT(*) AS 总人数
    -&gt; FROM worker
    -&gt; GROUP BY 部门号
    -&gt; HAVING COUNT(*) &gt; 4;</code></pre>
