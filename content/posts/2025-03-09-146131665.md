    <h2>
     <a id="export_multiline_0">
     </a>
     @export_multiline
    </h2>
    <p>
     在 Godot 中，@export_multiline 是一个用于脚本变量的注解，允许你在编辑器中为字符串变量提供多行输入支持。
    </p>
    <h3>
     <a id="_2">
     </a>
     作用
    </h3>
    <p>
     多行文本编辑：通常，@export 注解的字符串变量在编辑器中只能单行输入。使用 @export_multiline 后，编辑器会提供一个多行文本框，方便输入和编辑多行文本。
    </p>
    <h3>
     <a id="_5">
     </a>
     用法
    </h3>
    <p>
     将 @export_multiline 注解应用于脚本中的字符串变量，例如：
    </p>
    <pre><code class="prism language-python"><span class="token decorator annotation punctuation">@export_multiline</span>
var description<span class="token punctuation">:</span> String <span class="token operator">=</span> <span class="token string">""</span>
</code></pre>
    <h3>
     <a id="_11">
     </a>
     示例
    </h3>
    <p>
     假设你有一个 NPC 脚本，需要为角色添加多行描述：
    </p>
    <pre><code class="prism language-python">extends Node

<span class="token decorator annotation punctuation">@export_multiline</span>
var npc_description<span class="token punctuation">:</span> String <span class="token operator">=</span> <span class="token string">"这是一个NPC的描述。\n你可以在这里写多行文本。"</span>
</code></pre>
    <p>
     在编辑器中，npc_description 变量会显示为一个多行文本框，便于输入和编辑长文本。
    </p>
    <h3>
     <a id="_21">
     </a>
     总结
    </h3>
    <p>
     @export_multiline 简化了多行文本的编辑，适用于需要处理大量文本的场景，如对话、描述等。
    </p>
