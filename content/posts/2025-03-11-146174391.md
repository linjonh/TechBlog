## 2、安装OpenStack

Tips：这里选择的网络模式Self-Service

### 1）安装Controller

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh install controller 192.168.3.201 192.168.3.202 selfservice

[![](https://i-blog.csdnimg.cn/img_convert/091953d21b086dfdf33a6dd8647cf7dd.png)](https://www.yangxingzhen.com/wp-content/uploads/2025/01/011825_0856_CentOS79Ope7.png)

### 2）安装Compute1

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh install compute1 192.168.3.202 192.168.3.199 selfservice

[![](https://i-blog.csdnimg.cn/img_convert/ab96296c5a20e3530c14c66f283fcc18.png)](https://www.yangxingzhen.com/wp-content/uploads/2025/01/011825_0856_CentOS79Ope8.png)

### 3）安装Compute2

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh install compute2 192.168.3.201 192.168.3.199 selfservice

[![](https://i-blog.csdnimg.cn/img_convert/d6a5316fb803e7e023cad4e070aa9aee.png)](https://www.yangxingzhen.com/wp-content/uploads/2025/01/011825_0856_CentOS79Ope9.png)

### 4）安装Cinder-Node存储节点

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh install cinder 192.168.3.199 /dev/sdb

### 7）创建实例

# 创建实例

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh create m1.nano cirros provider-net selfservice-net selfservice-group ssh-key selfservice-instance-vm1

# 查看实例列表

[root@localhost ~]# openstack server list

[![](https://i-blog.csdnimg.cn/img_convert/59fa63c9cb67e0246e9cbd618ca05edb.png)](https://www.yangxingzhen.com/wp-content/uploads/2025/01/011825_0856_CentOS79Ope14.png)

### 8）添加实例

# 创建实例(Provider网络)

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh add m1.nano cirros provider-net provider-group ssh-key provider-instance-vm1

# 创建实例指定分配IP(Provider网络)

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh add m1.nano cirros provider-net,192.168.3.188 provider-group ssh-key provider-instance-vm2

# 查看实例列表

[root@localhost ~]# openstack server list

[![](https://i-blog.csdnimg.cn/img_convert/2824832953a3a8ade6094caa433b3e1f.png)](https://www.yangxingzhen.com/wp-content/uploads/2025/01/011825_0856_CentOS79Ope16.png)

# 创建实例(Self-Service网络)

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh add m1.nano cirros selfservice-net selfservice-group ssh-key selfservice-instance-vm2

# 创建实例指定分配IP(Self-Service网络)

[root@localhost ~]# bash manage\_openstack\_rocky\_for\_centos7.sh add m1.nano cirros selfservice-net,172.16.1.188 selfservice-group ssh-key selfservice-instance-vm3

# 查看实例列表
