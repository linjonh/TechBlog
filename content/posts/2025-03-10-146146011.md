    <p>
     在 macOS 上，你可以通过以下步骤查看自己的真实 IP 地址：
    </p>
    <h4>
     1. 使用终端命令
    </h4>
    <p>
     打开“终端”应用程序（可以在“应用程序” -&gt; “实用工具”中找到），然后输入以下命令：
    </p>
    <pre>ifconfig | grep "inet " | grep -v 127.0.0.1</pre>
    <h4>
     2. 解释
    </h4>
    <ul>
     <li>
      <p>
       <code>
        ifconfig
       </code>
       ：显示网络接口的详细信息。
      </p>
     </li>
     <li>
      <p>
       <code>
        grep "inet "
       </code>
       ：过滤出包含 IPv4 地址的行。
      </p>
     </li>
     <li>
      <p>
       <code>
        grep -v 127.0.0.1
       </code>
       ：排除本地回环地址（127.0.0.1）。
      </p>
     </li>
    </ul>
    <h4>
     3. 结果
    </h4>
    <p>
     你会看到类似以下的输出：
    </p>
    <pre>inet 192.168.1.100 netmask 0xffffff00 broadcast 192.168.1.255</pre>
    <p>
     其中
     <code>
      192.168.1.100
     </code>
     就是你的本地 IP 地址。
    </p>
    <h4>
     4. 查看公网 IP
    </h4>
    <p>
     如果你想查看公网 IP 地址，可以使用以下命令：
    </p>
    <pre>curl ifconfig.me</pre>
    <p>
     或者：
    </p>
    <pre>curl ipinfo.io/ip</pre>
    <p>
     这些命令会返回你的公网 IP 地址。
    </p>
    <h4>
     总结
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        本地 IP
       </strong>
       ：使用
       <code>
        ifconfig
       </code>
       命令。
      </p>
     </li>
     <li>
      <p>
       <strong>
        公网 IP
       </strong>
       ：使用
       <code>
        curl ifconfig.me
       </code>
       或
       <code>
        curl ipinfo.io/ip
       </code>
       命令。
      </p>
     </li>
    </ul>
