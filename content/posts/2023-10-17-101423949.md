     东南大学网安预推免经验帖
    </h2>
    <p>
     东大夏令营入营，但和深先院冲突。8月底参加参加东大网安预推免，希望通过这个帖子可以帮助到后面的学弟学妹。
    </p>
    <h3>
     <a id="_4">
     </a>
     个人基本条件
    </h3>
    <ol>
     <li>
      某中游211物联网工程专业，专业排名1/107；
     </li>
     <li>
      国家奖学金，国家级软件类大赛二等奖两项；
     </li>
     <li>
      发表会议论文一篇（一作）；
     </li>
     <li>
      专利受理2项；
     </li>
     <li>
      四六级500+
     </li>
    </ol>
    <h3>
     <a id="_11">
     </a>
     报道：报道地点在计院楼旁边，会检查你的四六级原件
    </h3>
    <p>
     <img alt="院照" src="https://i-blog.csdnimg.cn/blog_migrate/5ea5d8085be78022a9972278a96262b1.jpeg"/>
    </p>
    <h3>
     <a id="_13">
     </a>
     笔试
    </h3>
    <h3>
     <a id="501270_14">
     </a>
     一共四道大题，四选二，一题50分。分为计算机和通信方向，第一题是拥塞控制，第二题是物联网信息安全画出架构图，第三题是纯密码学。。。第四题是通信滤波类的知识。我当时选择1、2小题，估计拿了70多分吧。所以注意复习专业课基础，否则会像我一样惨。
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6c505a0016695f4f25897a39ab7555d9.jpeg"/>
    </p>
    <h3>
     <a id="_16">
     </a>
     面试
    </h3>
    <h3>
     <a id="28RELUsigmoid_17">
     </a>
     当时分为三个组，一个组28人左右。首先
     <mark>
      中文
     </mark>
     自我介绍，接着老师问项目（记得带
     <mark>
      简历
     </mark>
     ），当时被怼的很惨，印象深刻的是RELU函数为啥比sigmoid函数更适合于神经网络，我只答出可以防止过拟合。。。最后英文老师冷不丁来一句英文，考察你的口语。
    </h3>
    <h2>
     <a id="_19">
     </a>
     结果
    </h2>
    <h3>
     <a id="3070_21">
     </a>
     最后按照30%笔试+70%面试成绩进行排名，我成功获得东南网安学硕资格。
    </h3>
    <h3>
     <a id="1003131_22">
     </a>
     今年东大网安预推免100人过审，31人学硕，31人专硕拟录取，名额还是很多的，可是明年第一届网安自己的本科生升学，竞争可能会激烈很多。
    </h3>
