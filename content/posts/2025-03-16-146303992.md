    <h3>
     <a id="__1">
     </a>
     一、题目描述 ⭐️
    </h3>
    <h4>
     <a id="44_a_2">
     </a>
     习题4-4 特殊a串数列求和
    </h4>
    <p>
     <strong>
      给定两个均不超过9的正整数a和n，要求编写程序求a+aa+aaa+⋯+aa⋯a（n个a）之和。
     </strong>
    </p>
    <p>
     <strong>
      输入格式:
     </strong>
     <br/>
     <strong>
      输入在一行中给出不超过9的正整数a和n。
     </strong>
    </p>
    <p>
     <strong>
      输出格式:
     </strong>
     <br/>
     <strong>
      在一行中按照“s = 对应的和”的格式输出。
     </strong>
    </p>
    <p>
     <strong>
      输入样例:
     </strong>
    </p>
    <pre><code class="prism language-c"><span class="token number">2</span> <span class="token number">3</span>
</code></pre>
    <p>
     <strong>
      输出样例:
     </strong>
    </p>
    <pre><code class="prism language-c">s <span class="token operator">=</span> <span class="token number">246</span>
</code></pre>
    <h3>
     <a id="__20">
     </a>
     二、解题思路 ⭐️
    </h3>
    <blockquote>
     <p>
      代码中的
      <code>
       getNum()
      </code>
      函数解析：👇🏻
     </p>
    </blockquote>
    <blockquote>
     <p>
      以
      <code>
       a=2
      </code>
      为例：🦋
      <br/>
      <code>
       n=1
      </code>
      时：
      <code>
       2
      </code>
      =
      <code>
       2 * 10^0
      </code>
      ；
      <br/>
      <code>
       n=2
      </code>
      时：
      <code>
       22
      </code>
      =
      <code>
       2 * 10^1 + 2 * 10^0
      </code>
      ；
      <br/>
      <code>
       n=3
      </code>
      时：
      <code>
       222
      </code>
      =
      <code>
       2 * 10^2 + 2 * 10^1 + =2 * 10^0
      </code>
      ；
      <br/>
      。。。
      <br/>
      <code>
       n=n
      </code>
      时：
      <code>
       n个2
      </code>
      =
      <code>
       2 * 10^(n - 1) + ... + 2 * 10^2 + 2 * 10^1 + =2 * 10^0
      </code>
     </p>
    </blockquote>
