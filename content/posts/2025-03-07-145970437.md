    <h3>
     <a id="Maven_303">
     </a>
     Maven的常用命令
    </h3>
    <p>
     通过阅读上文的「
     <code>
      Effevtive POM
     </code>
     」，就
     <em>
      <strong>
       容易理解
      </strong>
     </em>
     我们经常使用的Maven命令，其基础配置是对应上面的默认插件。
     <br/>
     比如：我们使用
     <code>
      mvn clean
     </code>
     命令，实际起作用的是Maven的默认插件
     <code>
      maven-clean-plugin
     </code>
     。
    </p>
    <p>
     Maven的常用命令：
    </p>
    <ul>
     <li>
      <code>
       mvn clean
      </code>
      ：清理项目构建生成的文件和目录
     </li>
     <li>
      <code>
       mvn compile
      </code>
      ：编译项目的源代码
     </li>
     <li>
      <code>
       mvn test
      </code>
      ：执行项目的测试
     </li>
     <li>
      <code>
       mvn package
      </code>
      ：将项目编译后的代码打包成可分发的格式，如JAR包
     </li>
     <li>
      <code>
       mvn install
      </code>
      ：将项目打包后的产物安装到本地Maven仓库
     </li>
     <li>
      <code>
       mvn deploy
      </code>
      ：将项目打包后的产物部署到远程Maven仓库
     </li>
    </ul>
