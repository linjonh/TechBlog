    <h3>
     <a id="_15">
     </a>
     <strong>
      环境准备
     </strong>
    </h3>
    <p>
     在部署 Camel AI Owl 和集成阿里云百炼 DeepSeek-R1 API 之前，我们需要确保环境满足以下要求：
    </p>
    <h4>
     <a id="_19">
     </a>
     <strong>
      硬件和软件要求
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       操作系统
      </strong>
      ：Windows 10/11
     </li>
     <li>
      <strong>
       Python 版本
      </strong>
      ：3.10+
     </li>
     <li>
      <strong>
       工具
      </strong>
      ：Git、pip、curl（或 Postman 测试 API）
     </li>
     <li>
      <strong>
       阿里云百炼 DeepSeek-R1 API
      </strong>
      ：需要阿里云账户和 API 访问密钥（API-KEY）。
     </li>
    </ul>
    <h4>
     <a id="_25">
     </a>
     <strong>
      安装必要工具
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       Python
      </strong>
      ：确保已安装 Python 3.10+，并将其添加到系统的环境变量中。
     </li>
     <li>
      <strong>
       Git
      </strong>
      ：从
      <a href="https://git-scm.com/" rel="nofollow">
       Git官网
      </a>
      下载并安装 Git。
     </li>
     <li>
      <strong>
       pip
      </strong>
      ：Python 包管理工具，随 Python 一起安装。如果未安装，运行以下命令更新或安装 pip：
      <pre><code class="prism language-bash">python <span class="token parameter variable">-m</span> ensurepip <span class="token parameter variable">--upgrade</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="Windows__35">
     </a>
     <strong>
      Windows 下创建虚拟环境并安装依赖
     </strong>
    </h3>
    <p>
     为了确保项目的依赖库不与系统环境发生冲突，我们将使用
     <code>
      venv
     </code>
     创建 Python 虚拟环境。
    </p>
    <h4>
     <a id="1__39">
     </a>
     <strong>
      1. 创建虚拟环境
     </strong>
    </h4>
    <ol>
     <li>
      打开命令提示符（或 PowerShell），导航到你希望项目存放的目录，例如
      <code>
       D:\python_project\owl_project
      </code>
      ：
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> D:<span class="token punctuation">\</span>python_project<span class="token punctuation">\</span>owl_project
</code></pre>
     </li>
     <li>
      克隆 Camel AI Owl 项目到本地：
      <pre><code class="prism language-bash"><span class="token function">git</span> clone https://github.com/camel-ai/owl.git
<span class="token builtin class-name">cd</span> owl
</code></pre>
     </li>
     <li>
      使用 Python 的
      <code>
       venv
      </code>
      模块创建虚拟环境：
      <pre><code class="prism language-bash">python <span class="token parameter variable">-m</span> venv venv
</code></pre>
      这将在项目目录中生成一个名为
      <code>
       venv
      </code>
      的文件夹。
     </li>
    </ol>
    <p>
     如果你还没有安装 uv，请先安装
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> uv
</code></pre>
    <p>
     创建虚拟环境并安装依赖(支持使用 Python 3.10、3.11、3.12)
    </p>
    <pre><code class="prism language-bash">uv venv .venv <span class="token parameter variable">--python</span><span class="token operator">=</span><span class="token number">3.10</span>
</code></pre>
    <p>
     激活虚拟环境
    </p>
    <p>
     对于 macOS/Linux
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> .venv/bin/activate
</code></pre>
    <p>
     对于 Windows
    </p>
    <pre><code class="prism language-bash">.venv<span class="token punctuation">\</span>Scripts<span class="token punctuation">\</span>activate
</code></pre>
    <p>
     安装 CAMEL 及其所有依赖。
    </p>
    <pre><code class="prism language-bash">uv pip <span class="token function">install</span> <span class="token parameter variable">-e</span> <span class="token builtin class-name">.</span>
</code></pre>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/edd9d4b1e6a2463d8d1a278c4838ad47.jpeg"/>
    </p>
    <p>
     安装过程较长，耐心等待即可。
    </p>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/45bdf7f71ace41efaa9ae3d84abf1102.jpeg">
      完成后退出虚拟环境
     </img>
    </p>
    <pre><code class="prism language-bash">deactivate
</code></pre>
