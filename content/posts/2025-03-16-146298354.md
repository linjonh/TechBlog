    <h2>
     <a id="Ubuntu2404ssh_1">
     </a>
     Ubuntu24.04下管理自己的ssh连接
    </h2>
    <h3>
     <a id="_3">
     </a>
     一、远程主机信息
    </h3>
    <ul>
     <li>
      假设有一个远程主机：
      <pre><code class="prism language-shell"><span class="token comment"># ip：8.142.20.10</span>
<span class="token comment"># user: root</span>
<span class="token comment"># password: 12345</span>
<span class="token comment"># 输入以下命令即可登录</span>
<span class="token function">ssh</span> <span class="token parameter variable">-p</span> <span class="token number">22</span> root@8.142.20.10
</code></pre>
     </li>
     <li>
      但是我们不可能记住所有主机的ip和密码，所以我们可以通过配置秘钥认证的方式登录，这样我们可以简化登录流程。
     </li>
    </ul>
    <h3>
     <a id="_16">
     </a>
     二、配置秘钥认证登录
    </h3>
    <ul>
     <li>
      <p>
       客户端生成密钥对
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 生成公钥秘钥对</span>
ssh-keygen <span class="token parameter variable">-t</span> rsa <span class="token parameter variable">-b</span> <span class="token number">4096</span> <span class="token parameter variable">-C</span> <span class="token string">"ali_cloud_8.142.20.10"</span> <span class="token parameter variable">-f</span> ~/.ssh/ali_cloud_8_142_20_10_key
<span class="token comment"># 可以通过以下命令查看公钥的指纹（fingerprint）</span>
ssh-keygen <span class="token parameter variable">-l</span> <span class="token parameter variable">-f</span> ~/.ssh/ali_cloud_8_142_20_10_key.pub
</code></pre>
      <p>
       <img alt="生成密钥对" src="https://i-blog.csdnimg.cn/direct/0a608dfd22244afcb6c500fb74240ec7.png#pic_center">
        <br/>
        <img alt="查看公钥指纹" src="https://i-blog.csdnimg.cn/direct/a17a9a561a4c4a6d92fee7f242996fac.png#pic_center"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       客户端将公钥拷贝到远程主机
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 将客户端公钥拷贝到远程主机</span>
ssh-copy-id root@8.142.20.10
<span class="token comment"># 客户端直接登录，不在需要输入远程主机的密码</span>
<span class="token function">ssh</span> root@8.142.20.10
</code></pre>
      <p>
       <img alt="客户端将公钥拷贝到远程主机" src="https://i-blog.csdnimg.cn/direct/40d2a7b4df2742d6a6f5c298b4029101.png#pic_center">
        <br/>
        <img alt="免密登录" src="https://i-blog.csdnimg.cn/direct/25c79eed0249491ebf79868fe7cd8230.png#pic_center"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       简化登录命令
      </p>
      <pre><code class="prism language-shell"><span class="token comment"># 执行上诉步骤之后，已经不需要密码认证了，但是还是需要记住ip。为了更加简化，我们可以把远程主机的信息记录到客户端ssh配置文件中</span>
<span class="token function">vim</span> ~/.ssh/config
</code></pre>
      <pre><code class="prism language-text"># 相当于执行
# ssh -p 22 -i /home/guoke/.ssh/ali_cloud_8_142_20_10_key root@8.142.20.10
Host ali_cloud
	HostName 8.142.20.10
	User root
	Port 22
	IdentityFile /home/guoke/.ssh/ali_cloud_8_142_20_10_key
</code></pre>
     </li>
    </ul>
    <h3>
     <a id="_55">
     </a>
     三、使用
    </h3>
    <ul>
     <li>
      需要登录远程主机的时候，可以执行以下命令查看自己配置中记录的远程主机信息
      <pre><code class="prism language-shell"><span class="token comment"># 如果写入了配置/etc/ssh/ssh_config中，将以下配置文件路径替换即可</span>
<span class="token comment"># 查看自己配置了那些远程主机</span>
<span class="token function">cat</span> ~/.ssh/config <span class="token operator">|</span> <span class="token function">grep</span> <span class="token string">'^Host '</span> <span class="token operator">|</span> <span class="token function">awk</span> <span class="token string">'{print $2}'</span>
<span class="token comment"># 更加详细的信息直接cat即可</span>
<span class="token function">cat</span> ~/.ssh/config
</code></pre>
     </li>
     <li>
      登录远程主机
      <pre><code class="prism language-shell"><span class="token comment"># ssh [ssh_host]</span>
<span class="token function">ssh</span> ali_cloud
</code></pre>
      <img alt="ssh登录" src="https://i-blog.csdnimg.cn/direct/7e4cafbdbd9a48378883bc4fc5c30a18.png#pic_center"/>
     </li>
    </ul>
