    <p>
     移动管家手机遥控开关，超小体积4G手机遥控开关支持APP控制，不限距离，适用于需长距离或跨区域控制
     <br/>
     的场景‌。
     <br/>
     ‌GSM短信控制‌
     <br/>
     通过GSM控制器驱动继电器，以短信指令远程开关水泵、电饭煲等设备，支持状态查
     <br/>
     询与多手机号绑定‌。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/6eaa6d09ee684d3aa7f56798bdb5b062.jpeg"/>
    </p>
    <p>
     <br/>
     二、核心原理
     <br/>
     ‌继电器驱动‌：WiFi/4G模块或GSM控制器通过接收手机指令驱动继电器，
     <br/>
     控制电路通断‌。
     <br/>
     ‌红外信号匹配‌：手机红外功能模拟传统遥控器信号，需与目标设备品牌型号逐一匹配‌。
     <br/>
     ‌多
     <br/>
     协议兼容‌：部分方案支持自定义短信指令或中文指令，增强灵活性‌。
     <br/>
     三、应用场景
     <br/>
     ‌家居场景‌：控制电视、
     <br/>
     空调、电饭煲、智能门锁等家电设备‌。
     <br/>
     ‌工业场景‌：远程启停水泵、阀门、工控机等设备，支持状态反馈‌。
     <br/>
     ‌
     <br/>
     安防改造‌：老式门禁系统加装WiFi模块实现手机开门‌。
     <br/>
     四、操作步骤示例（以手机APP为例）
     <br/>
     ‌添加设备‌
     <br/>
     打开
     <br/>
     移动管家手机遥控APP，选择“添加设备”并指定品牌类型（如电视、空调）‌。
     <br/>
     ‌信号匹配‌
     <br/>
     对准设备点击“开
     <br/>
     关”按钮，根据响应选择正确的遥控器型号‌。
     <br/>
     ‌线路连接‌
     <br/>
     若为硬件改装，按颜色区分线路（如蓝色接电源、
     <br/>
     红色接检测端），并联或串联按钮接口‌。
     <br/>
     五、注意事项
     <br/>
     ‌兼容性‌：红外控制依赖手机硬件支持，需确认设备
     <br/>
     品牌是否在APP匹配库中‌。
     <br/>
     ‌安全性‌：GSM控制需设置唯一密码，避免未授权操作‌。
     <br/>
     ‌电源配置‌：改装时确保模
     <br/>
     块电压（如5V/24V）与原有电路匹配‌。
     <br/>
     通过上述方案，手机可替代传统遥控器或直接控制各类开关设备，
     <br/>
     覆盖家居、工业等多场景需求
    </p>
