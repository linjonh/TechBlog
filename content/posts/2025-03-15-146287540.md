       <code>
        cur-&gt;next = la;
       </code>
       <br/>
       //将 p指针所指向的链表节点的
       <code>
        next
       </code>
       指针（也就是
       <code>
        p
       </code>
       节点的下一个节点的指针）指向
       <code>
        l1
       </code>
       所指向的链表节点。简单来说，就是把
       <code>
        la
       </code>
       节点连接到
       <code>
        p
       </code>
       节点的后面，更新了链表的连接关系。
      </p>
     </li>
     <li>
      <p>
       <code>
        p = la;
       </code>
       <br/>
       //将p指针的值更新为
       <code>
        la
       </code>
       指针的值，也就是让 p指针指向
       <code>
        la
       </code>
       所指向的那个链表节点,为了后续对这个新连接上的节点进行操作，或继续在链表上向后移动 p指针，以便处理链表的后续部分。
      </p>
     </li>
     <li>
      <p>
       <code>
        la = la-&gt;next;
       </code>
       <br/>
       //将
       <code>
        la
       </code>
       指针指向 la当前所指向节点的下一个节点。也就是，
       <strong>
        <code>
         la
        </code>
        指针向后移动了一位
       </strong>
       ，指向链表中的下一个节点。这在遍历链表或者对链表节点进行逐个处理时是很常见的操作，用于按顺序访问链表中的每个节点。
      </p>
     </li>

       ListNode* merge(ListNode* la, ListNode* lb) {
       <!-- -->
       <br/>
       ListNode *lc = new ListNode(0);
       <br/>
       ListNode *p = lc;
       <br/>
       while (la &amp;&amp; lb ) {
       <!-- -->
       <br/>
       if (la -&gt; val &lt; lb -&gt; val) {
       <!-- -->
       <br/>
       p -&gt; next = la;
       <br/>
       la= la -&gt; next;
       <br/>
       }
       <br/>
       else {
       <!-- -->
       <br/>
       p -&gt; next = lb;
       <br/>
       lb= lb -&gt; next;
       <br/>
       }
       <br/>
       p = p -&gt; next;
       <br/>
       }
       <br/>
       p -&gt; next = (la ? la : lb);
       <br/>
       return lc -&gt; next;
      </p>
