    <h3>
     <a id="_7">
     </a>
     最接近的三数之和
    </h3>
    <blockquote>
     <p>
      题目地址：
      <a href="https://leetcode.cn/problems/3sum-closest/" rel="nofollow">
       最接近的三数之和
      </a>
     </p>
    </blockquote>
    <h4>
     <a id="_11">
     </a>
     题目描述
    </h4>
    <p>
     给你一个长度为
     <code>
      n
     </code>
     的整数数组
     <code>
      nums
     </code>
     和 一个目标值
     <code>
      target
     </code>
     。请你从
     <code>
      nums
     </code>
     中选出三个整数，使它们的和与
     <code>
      target
     </code>
     最接近。
    </p>
    <p>
     返回这三个数的和。
    </p>
    <p>
     假定每组输入只存在恰好一个解。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <pre><code>输入：nums = [-1,2,1,-4], target = 1
输出：2
解释：与 target 最接近的和是 2 (-1 + 2 + 1 = 2)。
</code></pre>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <pre><code>输入：nums = [0,0,0], target = 1
输出：0
解释：与 target 最接近的和是 0（0 + 0 + 0 = 0）。
</code></pre>
    <p>
     <strong>
      提示：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       3 &lt;= nums.length &lt;= 1000
      </code>
     </li>
     <li>
      <code>
       -1000 &lt;= nums[i] &lt;= 1000
      </code>
     </li>
     <li>
      <code>
       -104 &lt;= target &lt;= 104
      </code>
     </li>
    </ul>
    <h4>
     <a id="_40">
     </a>
     解题思路
    </h4>
    <ul>
     <li>
      首先根据题意处理边界条件，当数组长度等于 3 时，不需要任何处理，直接返回和即可。
     </li>
     <li>
      然后，对目标数组进行排序，根据集合有序的特点再结合双指针
      <code>
       l
      </code>
      和
      <code>
       r
      </code>
      ，与目标
      <code>
       target
      </code>
      进行比较：
      <ul>
       <li>
        当前三元和小于目标值：移动左指针
        <code>
         l
        </code>
        增加。
       </li>
       <li>
        当前三元和大于目标值：移动右指针
        <code>
         r
        </code>
        减少。
       </li>
       <li>
        当前三元和等于目标值：由于题目答案唯一，则直接返回当前三元和即可。
       </li>
      </ul>
     </li>
     <li>
      当每次移动指针之后，重新计算当前三元和，然后更新当前已经遍历到元素的三元和
      <code>
       min_sum
      </code>
      和最小三元和与目标
      <code>
       target
      </code>
      的差值
      <code>
       min_sum_df
      </code>
      。
     </li>
     <li>
      更新时，按照规则：
      <strong>
       当差值
       <code>
        min_sum_df
       </code>
       最小时，一定有
       <code>
        min_sum
       </code>
       与
       <code>
        target
       </code>
       最为接近
      </strong>
      。
     </li>
     <li>
      最后返回三元和
      <code>
       min_sum
      </code>
      即可。
     </li>
    </ul>
