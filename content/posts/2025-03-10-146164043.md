    <h4>
     <a id="_HTML__0">
     </a>
     <strong>
      第一章：欢迎来到 HTML 星球！
     </strong>
    </h4>
    <h5>
     <a id="11_HTML__1">
     </a>
     1.1 宇宙的基石：HTML 是什么？
    </h5>
    <ul>
     <li>
      🌍 比喻：HTML 是网页世界的「乐高积木」，用标签搭建一切可见内容
     </li>
     <li>
      🎯 目标：理解 HTML 的作用，掌握第一个
      <code>
       &lt;!DOCTYPE html&gt;
      </code>
      声明
     </li>
    </ul>
    <h5>
     <a id="12__HTML__5">
     </a>
     1.2 认识 HTML 文档的「骨骼结构」
    </h5>
    <ul>
     <li>
      🧩 趣味比喻：HTML 文档像汉堡包
      <ul>
       <li>
        <code>
         &lt;html&gt;
        </code>
        是包装盒（根标签）
       </li>
       <li>
        <code>
         &lt;head&gt;
        </code>
        是配料表（元信息）
       </li>
       <li>
        <code>
         &lt;body&gt;
        </code>
        是汉堡本体（可见内容）
       </li>
      </ul>
     </li>
     <li>
      👀 示例：打印「Hello, HTML 星球！」
     </li>
    </ul>
    <h4>
     <a id="_13">
     </a>
     <strong>
      第二章：标签大狂欢：从基础到变形
     </strong>
    </h4>
    <h5>
     <a id="21__14">
     </a>
     2.1 文本标签的魔法咒语
    </h5>
    <ul>
     <li>
      ✨ 幽默教学：
      <ul>
       <li>
        <code>
         &lt;p&gt;
        </code>
        ：专治「文字乱跑症」的段落标签
       </li>
       <li>
        <code>
         &lt;h1&gt;
        </code>
        -
        <code>
         &lt;h6&gt;
        </code>
        ：标题标签的「身高排行榜」
       </li>
       <li>
        <code>
         &lt;em&gt;
        </code>
        &amp;
        <code>
         &lt;strong&gt;
        </code>
        ：斜体和加粗的「抢戏二人组」
       </li>
      </ul>
     </li>
     <li>
      🎨 练习：用标签写一个搞笑段子
     </li>
    </ul>
    <h5>
     <a id="22__21">
     </a>
     2.2 图片与链接的「穿越门」
    </h5>
    <ul>
     <li>
      🔗 梗：链接标签
      <code>
       &lt;a&gt;
      </code>
      是「网页传送门」，
      <code>
       href
      </code>
      是目的地坐标
     </li>
     <li>
      📸 趣味示例：用
      <code>
       &lt;img&gt;
      </code>
      插入一张「会说话的猫」表情包
     </li>
     <li>
      ❗ 注意事项：
      <code>
       alt
      </code>
      属性的重要性（比如图片挂掉时显示「此处应有猫片」）
     </li>
    </ul>
    <h4>
     <a id="_27">
     </a>
     <strong>
      第三章：布局大师：用标签建造梦幻家园
     </strong>
    </h4>
    <h5>
     <a id="31__28">
     </a>
     3.1 列表的「强迫症治疗术」
    </h5>
    <ul>
     <li>
      📝 比喻：
      <ul>
       <li>
        <code>
         &lt;ul&gt;
        </code>
        ：无序列表是「叛逆少年」（默认带圆点）
       </li>
       <li>
        <code>
         &lt;ol&gt;
        </code>
        ：有序列表是「乖乖学生」（自动编号）
       </li>
       <li>
        <code>
         &lt;dl&gt;
        </code>
        ：定义列表是「百科全书」（术语+解释）
       </li>
      </ul>
     </li>
     <li>
      🤖 挑战：用列表做一个「程序员早餐清单」
     </li>
    </ul>
    <h5>
     <a id="32__35">
     </a>
     3.2 表格的「数据棋盘」
    </h5>
    <ul>
     <li>
      🎲 幽默类比：表格标签
      <code>
       &lt;table&gt;
      </code>
      是「数据麻将桌」
      <ul>
       <li>
        <code>
         &lt;tr&gt;
        </code>
        是麻将行，
        <code>
         &lt;td&gt;
        </code>
        是麻将格子
       </li>
       <li>
        用
        <code>
         &lt;caption&gt;
        </code>
        给表格起个风骚标题
       </li>
      </ul>
     </li>
     <li>
      🚀 示例：用表格做「星球大战 vs 星际迷航」对比表
     </li>
    </ul>
    <h4>
     <a id="_42">
     </a>
     <strong>
      第四章：表单与交互：打造你的「数据捕手」
     </strong>
    </h4>
    <h5>
     <a id="41__43">
     </a>
     4.1 表单元素的「奇葩收集器」
    </h5>
    <ul>
     <li>
      🕹️ 趣味比喻：
      <ul>
       <li>
        文本框
        <code>
         &lt;input type="text"&gt;
        </code>
        ：人类输入的「吐字器」
       </li>
       <li>
        密码框
        <code>
         &lt;input type="password"&gt;
        </code>
        ：隐藏秘密的「黑匣子」
       </li>
       <li>
        单选按钮
        <code>
         &lt;input type="radio"&gt;
        </code>
        ：「非此即彼」的选择题
       </li>
      </ul>
     </li>
     <li>
      💡 搞笑案例：设计一个「你是哪种奇怪人类」测试表单
     </li>
    </ul>
    <h5>
     <a id="42__50">
     </a>
     4.2 多媒体：给网页加点「蹦迪特效」
    </h5>
    <ul>
     <li>
      🎵 梗：用
      <code>
       &lt;audio&gt;
      </code>
      标签插入「好运来」BGM
     </li>
     <li>
      🎥 挑战：用
      <code>
       &lt;video&gt;
      </code>
      标签播放「土拨鼠尖叫」视频
     </li>
     <li>
      🚨 警告：别滥用自动播放（否则会被用户追杀）
     </li>
    </ul>
    <h4>
     <a id="HTML5__56">
     </a>
     <strong>
      第五章：HTML5 大升级：解锁未来技能
     </strong>
    </h4>
    <h5>
     <a id="51__57">
     </a>
     5.1 语义化标签的「身份认证」
    </h5>
    <ul>
     <li>
      🕵️ 比喻：
      <ul>
       <li>
        <code>
         &lt;header&gt;
        </code>
        是「网页大脑门」
       </li>
       <li>
        <code>
         &lt;nav&gt;
        </code>
        是「导航机器人」
       </li>
       <li>
        <code>
         &lt;article&gt;
        </code>
        是「文章小宇宙」
       </li>
      </ul>
     </li>
     <li>
      🤔 趣味对比：用旧标签和新标签分别写「朋友圈动态」
     </li>
    </ul>
    <h5>
     <a id="52__64">
     </a>
     5.2 本地存储：给网页装个「记忆芯片」
    </h5>
    <ul>
     <li>
      💾 幽默解释：
      <code>
       localStorage
      </code>
      是「网页的硬盘」，能记住用户喜好
     </li>
     <li>
      🍪 示例：用存储功能实现「自动记住用户名」的登录框
     </li>
    </ul>
    <h4>
     <a id="_69">
     </a>
     <strong>
      第六章：响应式设计：让网页「见风使舵」
     </strong>
    </h4>
    <h5>
     <a id="61__70">
     </a>
     6.1 媒体查询：网页的「变形金刚」
    </h5>
    <ul>
     <li>
      🦸 比喻：媒体查询是「网页的变形指令」，根据屏幕大小切换形态
     </li>
     <li>
      📱 练习：让一个按钮在手机上变大，在电脑上变小
     </li>
    </ul>
    <h5>
     <a id="62__74">
     </a>
     6.2 视口元标签：专治「手机屏幕晕车症」
    </h5>
    <ul>
     <li>
      🚗 梗：
      <code>
       viewport
      </code>
      是「网页的安全带」，防止页面缩放变形
     </li>
     <li>
      🌐 案例：用视口设置让网页适配所有移动设备
     </li>
    </ul>
    <h4>
     <a id="HTML__79">
     </a>
     <strong>
      第七章：实战项目：建造你的「HTML 太空站」
     </strong>
    </h4>
    <h5>
     <a id="71__1_0__1_80">
     </a>
     7.1 项目 1：个人博客（从 0 到 1）
    </h5>
    <ul>
     <li>
      🚀 步骤：
      <ol>
       <li>
        搭建骨架（
        <code>
         header
        </code>
        /
        <code>
         nav
        </code>
        /
        <code>
         article
        </code>
        /
        <code>
         footer
        </code>
        ）
       </li>
       <li>
        添加文章内容和评论表单
       </li>
       <li>
        用媒体查询实现手机适配
       </li>
      </ol>
     </li>
     <li>
      💡 彩蛋：给博客加个「愚人节模式」按钮
     </li>
    </ul>
    <h5>
     <a id="72__2_87">
     </a>
     7.2 项目 2：电商产品页（进阶挑战）
    </h5>
    <ul>
     <li>
      🛒 任务：
      <ul>
       <li>
        用表格展示产品参数
       </li>
       <li>
        用图片和链接实现「立即抢购」功能
       </li>
       <li>
        加入「库存不足」的本地存储提醒
       </li>
      </ul>
     </li>
     <li>
      🎉 成就：完成后可自称「网页包工头」
     </li>
    </ul>
