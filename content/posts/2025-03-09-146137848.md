    <p>
     Python环境的搭建，我们选择了Anaconda环境，有Python环境，接下来要搭建当然是AI辅助编程环境了。在搭建这个之前，或者说大家在打算安装VSCode之前，需要先检查自已电脑是否已经安装了VSCode，因为这个软件是Windows自带的轻量级的软件，当然如果不习惯用VSCode，用PyCharm也是一样的，原理基本相同。我之所选择VSCode的原因，就是因为其轻量级，平时运行不怎么影响我们的电脑运行。
    </p>
    <h5>
     <a id="1_VSCodePython_1">
     </a>
     <strong>
      1. VSCode核心配置：Python开发环境搭建
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        基础环境配置
       </strong>
      </p>
      <ul>
       <li>
        在安装VSCode前，一定要检查自己电脑是否已经安装有VSCode，如果已经安装，当然就不用再安装了。
       </li>
       <li>
        安装
        <a href="https://code.visualstudio.com/" rel="nofollow">
         VSCode官方版本
        </a>
       </li>
       <li>
        首次启动后安装
        <strong>
         Python扩展
        </strong>
        （搜索
        <code>
         Python
        </code>
        并安装Microsoft官方插件）
        <br/>
        <img alt="Python扩展安装界面  " src="https://i-blog.csdnimg.cn/direct/b2c9ccde65c84833b2a9360fd980b675.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        解释器绑定
       </strong>
      </p>
      <ul>
       <li>
        按
        <code>
         Ctrl+Shift+P
        </code>
        → 输入
        <code>
         Python: Select Interpreter
        </code>
        → 选择Anaconda环境路径（如
        <code>
         Anaconda3/envs/ai-project/python.exe
        </code>
        ） ，一般VSCode会自动识别到相应目录，此处有两种方式选择我们Python选择器。
       </li>
       <li>
       </li>
       <li>
        <img alt="大家可以在此处选择" src="https://i-blog.csdnimg.cn/direct/1fdc5f9fafd6453cbe6651eb8112d03e.png"/>
       </li>
       <li>
        <img alt="解释器选择第二种方法" src="https://i-blog.csdnimg.cn/direct/450f40a125dc47d48080c87e3c123d42.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        个性化设置
       </strong>
      </p>
      <ul>
       <li>
        修改
        <code>
         settings.json
        </code>
        ：
        <pre><code class="prism language-json"><span class="token punctuation">{<!-- --></span>  
  <span class="token string-property property">"python.linting.pylintEnabled"</span><span class="token operator">:</span> <span class="token boolean">false</span><span class="token punctuation">,</span>  
  <span class="token string-property property">"python.formatting.provider"</span><span class="token operator">:</span> <span class="token string">"black"</span><span class="token punctuation">,</span>  
  <span class="token string-property property">"editor.fontFamily"</span><span class="token operator">:</span> <span class="token string">"Consolas, 'Courier New', monospace"</span>  
<span class="token punctuation">}</span>  
</code></pre>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     具体设置方法如下：
     <br/>
     1.settings.json文件在哪（打开settings.json方式）
     <br/>
     （1）C:\Users\【你电脑名字username】\AppData\Roaming\Code\User
     <br/>
     （2）左下角设置按钮
     <br/>
     （3）文件 -&gt; 首选项 -&gt; 设置
     <br/>
     在这设置完属性后，settings.json也会自动更新
     <br/>
     （4）注意：上面是全局的，如果是针对项目的设置，则在项目下面找
     <br/>
     2.设置UI界面和settings.json界面切换（任意切换），可图形化界面设置，也可以在settings.json设置，根据操作习惯来选择。
     <br/>
     <img alt="建议采用这种图形的方法进行设置，这个更有利于我喜欢打代码者" src="https://i-blog.csdnimg.cn/direct/abc16be6f7f84deebf4deb189f805d2c.png"/>
    </p>

     <li>
      <p>
       <strong>
        Jupyter
       </strong>

    <h5>
     <a id="3_Git_43">
     </a>
     <strong>
      3. Git集成优化：版本控制全流程
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        Git安装与注册
       </strong>
      </p>
      <ul>
       <li>
        下载
        <a href="https://git-scm.com/" rel="nofollow">
         Git for Windows
        </a>
        → 安装时勾选
        <code>
         Use VSCode as default editor
        </code>
       </li>
       <li>
        终端配置用户信息：
        <pre><code class="prism language-bash"><span class="token function">git</span> config <span class="token parameter variable">--global</span> user.name <span class="token string">"YourName"</span>  
<span class="token function">git</span> config <span class="token parameter variable">--global</span> user.email <span class="token string">"your@email.com"</span>  
</code></pre>
        这个也可以选择性安装，只有后面大家要做项目了，这个安装才有意义。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        VSCode集成实战
       </strong>
      </p>
      <ul>
       <li>
        初始化仓库：
        <code>
         Ctrl+Shift+G
        </code>
        打开源代码管理 → 初始化仓库
       </li>
       <li>
        提交代码：输入提交信息 → 点击√图标
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        高级优化
       </strong>
      </p>
      <ul>
       <li>
        设置自动拉取：
        <code>
         git config --global pull.rebase true
        </code>
       </li>
       <li>
        启用提交签名：
        <pre><code class="prism language-bash"><span class="token function">git</span> config <span class="token parameter variable">--global</span> commit.gpgsign <span class="token boolean">true</span>  
<span class="token function">git</span> config <span class="token parameter variable">--global</span> gpg.program <span class="token string">"C:\Program Files\Git\usr<span class="token entity" title="\b">\b</span>in\gpg.exe"</span>  
</code></pre>
       </li>
      </ul>
     </li>
    </ol>
