    <h4>
     <strong>
      一、核心功能定位
     </strong>
    </h4>
    <p>
     Prosys OPC UA Gateway 是一款 高性能的协议转换与通信桥接工具，专门用于解决工业自动化领域中 OPC Classic（基于COM/DCOM） 与 OPC UA（统一架构） 设备间的互操作难题。其核心定位为：
     <br/>
     <strong>
      协议兼容性桥梁：
     </strong>
     实现传统OPC DA/AE/HDA服务器与OPC UA客户端/服务器的双向数据交互，支持跨代设备
     <a href="https://www.evget.com/product/4398" rel="nofollow" title="无缝集成 ">
      无缝集成
     </a>
     <br/>
     <strong>
      安全通信隧道：
     </strong>
     通过UA加密通道（如AES256、Basic128Rsa15）代理COM/DCOM通信，解决防火墙穿透问题，保障数据在复杂网络环境中的安全传输
    </p>
    <p>
    </p>

    </p>

    </p>
    <h4>
     <strong>
      二、技术特性与架构优势
     </strong>
    </h4>
    <p>
     <strong>
      多协议支持与双向通信
     </strong>
     <br/>
     <strong>
      协议覆盖：
     </strong>
     兼容OPC UA 1.04及以下版本，同时支持OPC Classic全系列协议（DA 2/3、AE、HDA），实现历史数据、实时数据与事件告警的跨协议同步
     <br/>
     <strong>
      双向模式：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       UA服务器模式：
      </strong>
      将传统OPC DA/AE服务器包装为UA服务器，供UA客户端访问；
     </li>
     <li>
      <strong>
       反向代理模式：
      </strong>
      允许COM/DCOM客户端通过UA通道连接远程UA服务器，突破DCOM网络限制
     </li>
    </ul>
    <p>
     <strong>
      灵活部署与高效扩展
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       跨平台适配
      </strong>
      ：支持Windows全系列（XP至Win11），32/64位系统兼容，支持运行时动态添加连接节点，无需重启服务
     </li>
     <li>
      <strong>
       高并发处理：
      </strong>
      单节点可同时管理50个UA通道、25个DA/AE通道，满足中小型工业场景的通信需求
     </li>
    </ul>
    <p>
     <strong>
      工业级安全机制
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       端到端加密：
      </strong>
      支持AES256-Sha256-RsaPss等安全策略，结合证书双向认证（X.509标准），符合IEC 62443工业安全规范
     </li>
     <li>
      <strong>
       访问控制：
      </strong>
      基于角色的权限管理，支持匿名访问与用户凭证认证，确保数据操作合规性
     </li>
    </ul>
