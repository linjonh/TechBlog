# python安装本地包

[Python](http://lib.csdn.net/base/python "Python知识库")
包（pip)离线安装方法 ：

1、到
<https://pypi.python.org/pypi>
下载相应包并解压
  
2、找到所需要的所有依赖，放进解压后的文件夹里，终端输入：sudo python setup.py install （看是否有setup.py文件）与make cofing相似。
