    </svg>
    <p>
     <a href="https://nodejs.org/zh-cn" rel="nofollow">
      https://nodejs.org/zh-cn
     </a>
     <br/>
     这个错误表明你的系统无法识别
     <code>
      npm
     </code>
     命令。这通常是因为 Node.js 没有正确安装，或者 Node.js 的路径没有添加到系统的环境变量中。以下是解决这个问题的步骤：
    </p>
    <hr/>
    <h4>
     <a id="1__Nodejs__5">
     </a>
     <strong>
      1. 检查 Node.js 是否已安装
     </strong>
    </h4>
    <ol>
     <li>
      打开终端（命令提示符或 PowerShell）。
     </li>
     <li>
      运行以下命令检查 Node.js 是否已安装：
      <pre><code class="prism language-bash"><span class="token function">node</span> <span class="token parameter variable">-v</span>
</code></pre>
      <ul>
       <li>
        如果显示版本号（例如
        <code>
         v18.12.1
        </code>
        ），说明 Node.js 已安装。
       </li>
       <li>
        如果没有显示版本号，说明 Node.js 未安装。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="2__Nodejs_16">
     </a>
     <strong>
      2. 安装 Node.js
     </strong>
    </h4>
    <p>
     如果 Node.js 未安装，请按照以下步骤安装：
    </p>
    <ol>
     <li>
      访问
      <a href="https://nodejs.org/" rel="nofollow">
       Node.js 官网
      </a>
      。
     </li>
     <li>
      下载并安装
      <strong>
       LTS 版本
      </strong>
      （长期支持版本）。
     </li>
     <li>
      安装完成后，重新打开终端，再次运行以下命令检查是否安装成功：
      <pre><code class="prism language-bash"><span class="token function">node</span> <span class="token parameter variable">-v</span>
<span class="token function">npm</span> <span class="token parameter variable">-v</span>
</code></pre>
      <ul>
       <li>
        如果两个命令都显示版本号，说明安装成功。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="3__29">
     </a>
     <strong>
      3. 检查环境变量
     </strong>
    </h4>
    <p>
     如果 Node.js 已安装但
     <code>
      npm
     </code>
     命令仍然无法识别，可能是环境变量未正确配置。
    </p>
    <h5>
     <a id="Windows__32">
     </a>
     <strong>
      Windows 系统
     </strong>
    </h5>
    <ol>
     <li>
      打开“控制面板” &gt; “系统和安全” &gt; “系统” &gt; “高级系统设置”。
     </li>
     <li>
      点击“环境变量”。
     </li>
     <li>
      在“系统变量”部分，找到
      <code>
       Path
      </code>
      变量，点击“编辑”。
     </li>
     <li>
      确保以下路径已添加到
      <code>
       Path
      </code>
      中：
      <ul>
       <li>
        <code>
         C:\Program Files\nodejs\
        </code>
       </li>
       <li>
        如果 Node.js 安装在其他目录，请添加对应的路径。
       </li>
      </ul>
     </li>
     <li>
      保存并关闭所有窗口，然后重新打开终端。
     </li>
    </ol>
    <h5>
     <a id="macOSLinux__41">
     </a>
     <strong>
      macOS/Linux 系统
     </strong>
    </h5>
    <ol>
     <li>
      打开终端，运行以下命令检查 Node.js 的安装路径：
      <pre><code class="prism language-bash"><span class="token function">which</span> <span class="token function">node</span>
<span class="token function">which</span> <span class="token function">npm</span>
</code></pre>
      <ul>
       <li>
        如果路径未显示，说明 Node.js 未正确安装或路径未配置。
       </li>
      </ul>
     </li>
     <li>
      如果路径显示，但仍然无法使用
      <code>
       npm
      </code>
      ，可以手动将路径添加到环境变量：
      <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token environment constant">$PATH</span>:/usr/local/bin/node
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token environment constant">$PATH</span>:/usr/local/bin/npm
</code></pre>
      <ul>
       <li>
        将
        <code>
         /usr/local/bin
        </code>
        替换为
        <code>
         which node
        </code>
        和
        <code>
         which npm
        </code>
        显示的路径。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="4__57">
     </a>
     <strong>
      4. 验证安装
     </strong>
    </h4>
    <ol>
     <li>
      重新打开终端，运行以下命令：
      <pre><code class="prism language-bash"><span class="token function">node</span> <span class="token parameter variable">-v</span>
<span class="token function">npm</span> <span class="token parameter variable">-v</span>
</code></pre>
      <ul>
       <li>
        如果两个命令都显示版本号，说明问题已解决。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="5__npm_init_y_67">
     </a>
     <strong>
      5. 重新运行
      <code>
       npm init -y
      </code>
     </strong>
    </h4>
    <ol>
     <li>
      在终端中导航到你的项目目录：
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> path/to/your/project
</code></pre>
     </li>
     <li>
      运行以下命令初始化项目：
      <pre><code class="prism language-bash"><span class="token function">npm</span> init <span class="token parameter variable">-y</span>
</code></pre>
      <ul>
       <li>
        如果成功，会生成一个
        <code>
         package.json
        </code>
        文件。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="6__80">
     </a>
     <strong>
      6. 如果问题仍然存在
     </strong>
    </h4>
    <p>
     如果按照以上步骤操作后问题仍然存在，可以尝试以下方法：
    </p>
    <ol>
     <li>
      <strong>
       重新安装 Node.js
      </strong>
      ：
      <ul>
       <li>
        卸载现有的 Node.js，然后重新安装。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用 Node 版本管理工具
      </strong>
      ：
      <ul>
       <li>
        对于 Windows，可以使用
        <a href="https://github.com/coreybutler/nvm-windows">
         nvm-windows
        </a>
        。
       </li>
       <li>
        对于 macOS/Linux，可以使用
        <a href="https://github.com/nvm-sh/nvm">
         nvm
        </a>
        。
       </li>
       <li>
        安装后，运行以下命令安装 Node.js：
        <pre><code class="prism language-bash">nvm <span class="token function">install</span> <span class="token parameter variable">--lts</span>
nvm use <span class="token parameter variable">--lts</span>
</code></pre>
       </li>
      </ul>
     </li>
    </ol>
