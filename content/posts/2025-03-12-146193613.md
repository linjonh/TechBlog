    <h1 class="title-article" id="articleContentId">
     PHP 在 if 判断时由于运算符优先级导致 false 的问题
    </h1>
   </div>
  </div>
 </div>

  <div class="article_content clearfix" id="article_content">

    <p>
     首先来看一段代码：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$price</span> <span class="token operator">=</span> <span class="token string single-quoted-string">'187.50'</span><span class="token punctuation">;</span>

<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token operator">!</span><span class="token function">is_numeric</span><span class="token punctuation">(</span><span class="token variable">$price</span><span class="token punctuation">)</span> <span class="token operator">||</span> <span class="token variable">$price</span> <span class="token operator">&lt;=</span> <span class="token number">0</span> <span class="token operator">||</span> <span class="token punctuation">(</span><span class="token variable">$price</span> <span class="token operator">*=</span> <span class="token number">100</span> <span class="token operator">&gt;</span> <span class="token number">1000000</span><span class="token punctuation">)</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">echo</span> <span class="token string double-quoted-string">"价格错误：<span class="token interpolation"><span class="token variable">$price</span></span>\n"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">echo</span> <span class="token string double-quoted-string">"价格正确：<span class="token interpolation"><span class="token variable">$price</span></span>\n"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     乍一看是不是认为并没有什么问题，
     <code>
      $price
     </code>
     会正常乘以
     <code>
      100
     </code>
     转换为“
     <strong>
      分
     </strong>
     ”并输出
     <code>
      价格正确
     </code>
    </p>
    <p>
     实则不然
    </p>
    <pre><code class="prism language-php"><span class="token punctuation">(</span><span class="token variable">$price</span> <span class="token operator">*=</span> <span class="token number">100</span> <span class="token operator">&gt;</span> <span class="token number">1000000</span><span class="token punctuation">)</span>

<span class="token comment">// 由于 *= 100 的优先级低于 &gt;，所以代码被解析为下方代码</span>

<span class="token punctuation">(</span><span class="token variable">$price</span> <span class="token operator">=</span> <span class="token variable">$price</span> <span class="token operator">*</span> <span class="token punctuation">(</span><span class="token number">100</span> <span class="token operator">&gt;</span> <span class="token number">1000000</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     然后就简单了，因为
     <code>
      100 &gt; 1000000
     </code>
     返回了
     <code>
      false
     </code>
     导致被转换为
     <code>
      0
     </code>
     ，最终也就输出了
     <strong>
      <code>
       价格错误：0
      </code>
     </strong>
    </p>
    <p>
     <strong>
      而需要注意的点是
     </strong>
     将
     <code>
      if
     </code>
     中倒数第二个反括号
     <code>
      )
     </code>
     转移到
     <code>
      *= 100
     </code>
     后：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$price</span> <span class="token operator">=</span> <span class="token string single-quoted-string">'187.50'</span><span class="token punctuation">;</span>

<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token operator">!</span><span class="token function">is_numeric</span><span class="token punctuation">(</span><span class="token variable">$price</span><span class="token punctuation">)</span> <span class="token operator">||</span> <span class="token variable">$price</span> <span class="token operator">&lt;=</span> <span class="token number">0</span> <span class="token operator">||</span> <span class="token punctuation">(</span><span class="token variable">$price</span> <span class="token operator">*=</span> <span class="token number">100</span><span class="token punctuation">)</span> <span class="token operator">&gt;</span> <span class="token number">1000000</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">echo</span> <span class="token string double-quoted-string">"价格错误：<span class="token interpolation"><span class="token variable">$price</span></span>\n"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">echo</span> <span class="token string double-quoted-string">"价格正确：<span class="token interpolation"><span class="token variable">$price</span></span>\n"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     保证了
     <code>
      *= 100
     </code>
     先于
     <code>
      &gt;
     </code>
     运算，就不会发生
     <code>
      false
     </code>
     的问题
    </p>
