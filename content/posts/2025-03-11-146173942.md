    <h4>
     <a id="_2">
     </a>
     什么是伪类？
    </h4>
    <p>
     伪类（Pseudo-class）用于选择元素的特定状态或位置。它以冒号（
     <code>
      :
     </code>
     ）作为前缀标识符，常见的伪类包括动态伪类、结构伪类等。
    </p>
    <h5>
     <a id="1__6">
     </a>
     1. 动态伪类
    </h5>
    <p>
     动态伪类用于根据用户行为或元素状态应用样式。
    </p>
    <ul>
     <li>
      <p>
       <strong>
        锚点伪类
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         :link
        </code>
        ：选择未访问的链接。
       </li>
       <li>
        <code>
         :visited
        </code>
        ：选择已访问的链接。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        行为伪类
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         :hover
        </code>
        ：鼠标悬停在元素上时的样式。
       </li>
       <li>
        <code>
         :active
        </code>
        ：用户点击元素时的样式。
       </li>
       <li>
        <code>
         :focus
        </code>
        ：元素获取焦点时的样式。
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="2__18">
     </a>
     2. 结构伪类
    </h5>
    <p>
     结构伪类用于选择元素的特定位置或结构。
    </p>
    <ul>
     <li>
      <code>
       :first-child
      </code>
      ：选择第一个子元素。
     </li>
     <li>
      <code>
       :last-child
      </code>
      ：选择最后一个子元素。
     </li>
     <li>
      <code>
       :nth-child(n)
      </code>
      ：选择第 n 个子元素。
     </li>
     <li>
      <code>
       :lang(language)
      </code>
      ：选择指定语言的元素。
     </li>
    </ul>
    <p>
     <strong>
      注意
     </strong>
     ：伪类以冒号开头，选择器和冒号之间不能有空格，伪类名中间也不能有空格。
    </p>
    <hr/>
    <h4>
     <a id="_30">
     </a>
     什么是伪元素？
    </h4>
    <p>
     伪元素（Pseudo-element）用于选择元素的特定部分或生成内容。它与伪类的区别在于，伪元素表示元素内部的内容，逻辑上存在但在文档树中并不存在。CSS3 规定伪元素用双冒号（
     <code>
      ::
     </code>
     ）表示。
    </p>
    <h5>
     <a id="_34">
     </a>
     常见伪元素
    </h5>
    <ul>
     <li>
      <code>
       ::after
      </code>
      ：在元素内容之后插入内容。
     </li>
     <li>
      <code>
       ::before
      </code>
      ：在元素内容之前插入内容。
     </li>
     <li>
      <code>
       ::first-letter
      </code>
      ：选择元素的第一个字母。
     </li>
     <li>
      <code>
       ::first-line
      </code>
      ：选择元素的第一行。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_42">
     </a>
     实战案例
    </h4>

     <a id="_1_44">
     </a>

     </strong>

     <strong>

    <h5>
     <a id="_2_77">
     </a>
     案例 2：结构伪类

     <strong>

    <h5>
     <a id="_3_109">
     </a>
     案例 3：伪元素
    </h5>
    <p>
     <strong>
      HTML
     </strong>
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>这是一个段落。<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>

     </strong>

    </p>

     </li>

     </li>

     </li>
    </ul>

    <h4>
     <a id="_145">
     </a>
     伪类与伪元素的区别
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        特性
       </th>
       <th>
        伪类
       </th>
       <th>
        伪元素
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        标识符
       </td>
       <td>
        单冒号（
        <code>
         :
        </code>
        ）
       </td>
       <td>
        双冒号（
        <code>
         ::
        </code>
        ）
       </td>
      </tr>
      <tr>
       <td>
        作用
       </td>
       <td>
        选择元素的特定状态或位置
       </td>
       <td>
        选择元素的特定部分或生成内容
       </td>
      </tr>
      <tr>
       <td>
        示例
       </td>
       <td>
        <code>
         :hover
        </code>
        、
        <code>
         :first-child
        </code>
       </td>
       <td>
        <code>
         ::after
        </code>
        、
        <code>
         ::first-letter
        </code>
       </td>
      </tr>
      <tr>
       <td>
        文档树
       </td>
       <td>
        逻辑上存在，文档树中存在
       </td>
       <td>
        逻辑上存在，文档树中不存在
       </td>
      </tr>
     </tbody>
    </table>
    <hr/>
    <h4>
     <a id="_156">
     </a>
     注意事项
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        兼容性
       </strong>
       ：
      </p>
      <ul>
       <li>
        CSS3 规定伪元素用双冒号（
        <code>
         ::
        </code>
        ），但为了兼容旧版浏览器，单冒号（
        <code>
         :
        </code>
        ）也可以使用。
       </li>
       <li>
        例如，
        <code>
         ::after
        </code>
        和
        <code>
         :after
        </code>
        在大多数浏览器中效果相同。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        内容生成
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         ::before
        </code>
        和
        <code>
         ::after
        </code>
        必须与
        <code>
         content
        </code>
        属性一起使用，否则不会生效。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        性能
       </strong>
       ：
      </p>
      <ul>
       <li>
        过度使用伪类和伪元素可能会影响页面性能，尤其是在复杂的页面中。
       </li>
      </ul>
     </li>
    </ol>
