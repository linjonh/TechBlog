#### 将一个字段的值拼接

```bash
string_agg(字段a,'分隔符')

```

#### 将多个字段的值连接

```bash
concat_ws('分隔符',字段a,字段b)

```

#### 将多个字段的值拼接成一条

```bash
string_agg(concat_ws('分隔符',字段a,字段b)) as 新名字

```

#### 测试：

```bash
test=# select * from bb;
```
