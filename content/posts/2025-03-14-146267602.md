     响应（Response）
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <p>
     在 Flask 中，视图函数可以返回多种类型的响应，例如字符串、HTML、JSON、文件等。Flask 提供了
     <code>
      make_response
     </code>
     函数，用于生成和自定义 HTTP 响应。
    </p>
    <h4>
     2.1
     <strong>
      默认响应
     </strong>
    </h4>
    <p>
     默认情况下，视图函数返回的字符串会被 Flask 包装成一个 HTTP 响应，状态码为
     <code>
      200 OK
     </code>
     ，内容类型为
     <code>
      text/html
     </code>
     。
    </p>
    <pre>@app.route('/')
def home():
    return "Hello, World!"  # 默认状态码为 200，内容类型为 text/html</pre>
    <h4>
     2.2
     <strong>
      使用
      <code>
       make_response
      </code>
      自定义响应
     </strong>
    </h4>
    <p>
     <code>
      make_response
     </code>
     函数允许你自定义响应的状态码、头部、内容类型等。
    </p>
    <h5>
     2.2.1
     <strong>
      基本用法
     </strong>
    </h5>
    <pre>from flask import Flask, make_response

app = Flask(__name__)

@app.route('/custom')
def custom_response():
    response = make_response("This is a custom response", 201)  # 状态码为 201
    response.headers['Content-Type'] = 'text/plain'
    return response</pre>
    <ul>
     <li>
      <p>
       <code>
        make_response
       </code>
       的第一个参数是响应内容，第二个参数是状态码。
      </p>
     </li>
     <li>
      <p>
       可以通过
       <code>
        response.headers
       </code>
       设置响应头部。
      </p>
     </li>
    </ul>
    <h5>
     2.2.2
     <strong>
      设置 Cookies
     </strong>
    </h5>
    <pre>@app.route('/set-cookie')
def set_cookie():
    response = make_response("Cookie set!")
    response.set_cookie('username', 'john')
    return response</pre>
    <ul>
     <li>
      <p>
       <code>
        set_cookie
       </code>
       方法用于设置 Cookie。
      </p>
     </li>
    </ul>
    <h5>
     2.2.3
     <strong>
      返回 JSON 数据
     </strong>
    </h5>
    <p>
     python
    </p>
    <p>
     复制
    </p>
    <pre>from flask import jsonify

@app.route('/json')
def json_response():
    data = {'name': 'John', 'age': 30}
    return jsonify(data)  # 返回 JSON 格式的响应</pre>
    <ul>
     <li>
      <p>
       <code>
        jsonify
       </code>
       函数会将 Python 字典转换为 JSON 格式的响应。
      </p>
     </li>
    </ul>
    <h5>
     2.2.4
     <strong>
      返回文件
     </strong>
    </h5>
    <pre>from flask import send_file

@app.route('/download')
def download_file():
    return send_file('path/to/file.txt', as_attachment=True)</pre>
    <ul>
     <li>
      <p>
       <code>
        send_file
       </code>
       函数用于发送文件，
       <code>
        as_attachment=True
       </code>
       表示文件作为附件下载。
      </p>
     </li>
    </ul>
    <hr/>
    <h3>
     3.
     <strong>
      请求和响应的完整示例
     </strong>
    </h3>
    <p>
     以下是一个完整的示例，展示如何处理请求和生成响应：
    </p>
    <p>
     python
    </p>
    <p>
     复制
    </p>
    <pre>from flask import Flask, request, make_response, jsonify

app = Flask(__name__)

@app.route('/greet', methods=['GET', 'POST'])
def greet():
    if request.method == 'GET':
        name = request.args.get('name', 'Guest')
    elif request.method == 'POST':
        name = request.form.get('name', 'Guest')

    response = make_response(f"Hello, {name}!", 200)
    response.set_cookie('username', name)
    return response

@app.route('/api/data')
def api_data():
    data = {'status': 'success', 'message': 'Hello, World!'}
    return jsonify(data)

if __name__ == '__main__':
    app.run(debug=True)</pre>
    <ul>
     <li>
      <p>
       访问
       <code>
        /greet?name=John
       </code>
       会返回
       <code>
        "Hello, John!"
       </code>
       ，并设置一个名为
       <code>
        username
       </code>
       的 Cookie。
      </p>
     </li>
     <li>
      <p>
       访问
       <code>
        /api/data
       </code>
       会返回 JSON 格式的响应。
      </p>
     </li>
    </ul>
