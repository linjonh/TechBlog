    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      <strong>
       1. 安装第三方库
      </strong>
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      这里安装一个第三方图片处理库PIL为例。
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      cmd下执行：pip install Pillow
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     注：
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="color:rgb(102,102,102);background-color:rgb(255,255,255);">
      1）PIL：Python Imaging Library，已经是Python平台事实上的图像处理标准库。由于PIL仅支持到Python 2.7，加上年久失修，于是一群志愿者在PIL的基础上创建了兼容的版本，名字叫
     </span>
     <a href="https://github.com/python-pillow/Pillow">
      <span style="color:rgb(5,147,211);">
       Pillow
      </span>
     </a>
     <span style="color:rgb(102,102,102);background-color:rgb(255,255,255);">
      ，支持最新Python 3.x，又加入了许多新特性，因此，我们可以直接安装使用Pillow。
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="color:rgb(102,102,102);background-color:rgb(255,255,255);">
      2）PIL官方只有32位的安装文件，在64位系统下如果如果安装的python是64位的，安装时会提示找不到python的安装路径。网上有非官方的64位库（官方源码编译版），叫做pillow。Pillow是PIL的替代版本。PIL 软件包提供了基本的图像处理功能，如：改变图像大小，旋转图像，图像格式转换，色场空间转换，图像增强，直方图处理，插值和滤波等等。
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="color:rgb(102,102,102);background-color:rgb(255,255,255);">
      Pillow 为了解决 PIL 的两个问题：
     </span>
    </div>
    <ol>
     <li style="text-align:left;line-height:1.75;font-size:14px;list-style-position:inside;white-space:pre-wrap;font-family:'Microsoft YaHei', STXihei;color:rgb(102,102,102);">
      <span style="background-color:rgb(255,255,255);">
       不兼容 setuptools
      </span>
     </li>
     <li style="text-align:left;line-height:1.75;font-size:14px;list-style-position:inside;white-space:pre-wrap;font-family:'Microsoft YaHei', STXihei;color:rgb(102,102,102);">
      <span style="background-color:rgb(255,255,255);">
       报告的问题太多，而更新太慢
      </span>
     </li>
    </ol>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="color:rgb(102,102,102);background-color:rgb(255,255,255);">
      安装了pillow就要用from PIL import Image替换import Image
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <br/>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      <strong>
       2. 测试
      </strong>
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      首先 cmd下输入python，进入解释器。
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <span style="font-size:16px;">
      我们将一张图片进行进行压缩后再保存，需要依次执行下列命令：
     </span>
    </div>
    <div style="white-space:pre-wrap;text-align:left;line-height:1.75;font-size:14px;">
     <pre><code class="language-cpp">from PIL import Image
im = Image.open('D:\\baoli\\Pictures\\图片\\test.JPG')
print(im.format, im.size, im.mode)
im.thumbnail((200, 100))
im.save('D:\\baoli\\Pictures\\图片\\new.JPG', 'JPEG')</code></pre>
