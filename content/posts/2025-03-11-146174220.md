    <h4>
     <strong>
      一、搭建XAMPP环境
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        安装Apache
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         创建目录结构（如
         <code>
          C:/web/apache2.4
         </code>
         ）。
        </p>
       </li>
       <li>
        <p>
         修改
         <code>
          httpd.conf
         </code>
         配置文件，设置服务器域名、监听端口等。
        </p>
       </li>
       <li>
        <p>
         启动Apache服务，通过
         <code>
          Apache Service Monitor
         </code>
         控制服务的启动、停止和重启。
        </p>
       </li>
       <li>
        <p>
         配置Apache索引页，将
         <code>
          index.php
         </code>
         设置为默认索引页。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装MySQL
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         运行命令行工具安装MySQL服务。
        </p>
       </li>
       <li>
        <p>
         创建配置文件
         <code>
          my.ini
         </code>
         ，指定安装目录、数据目录和端口号。
        </p>
       </li>
       <li>
        <p>
         初始化MySQL并设置root用户密码。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装PHP
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         解压PHP安装包到指定目录。
        </p>
       </li>
       <li>
        <p>
         配置
         <code>
          php.ini
         </code>
         文件，设置时区、扩展目录等。
        </p>
       </li>
       <li>
        <p>
         在Apache中引入PHP模块，确保PHP能够正常工作。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      二、Web服务器配置
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        配置虚拟主机
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         在
         <code>
          hosts
         </code>
         文件中配置域名映射（如
         <code>
          127.0.0.1 www.php.test
         </code>
         ）。
        </p>
       </li>
       <li>
        <p>
         修改
         <code>
          httpd.conf
         </code>
         文件启用虚拟主机配置文件
         <code>
          httpd-vhosts.conf
         </code>
         。
        </p>
       </li>
       <li>
        <p>
         在
         <code>
          httpd-vhosts.conf
         </code>
         中添加虚拟主机配置，指定文档根目录和域名。
        </p>
       </li>
       <li>
        <p>
         示例配置：
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <blockquote>
     <p>
      &lt;VirtualHost *:80&gt;
      <br/>
      DocumentRoot "C:/web/apache2.4/htdocs/www.php.test"
      <br/>
      ServerName www.php.test
      <br/>
      &lt;/VirtualHost&gt;
     </p>
    </blockquote>
    <h4>
     三、
     <strong>
      访问权限控制
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        Apache配置文件中的访问控制
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <code>
          &lt;Directory&gt;
         </code>
         指令用于配置目录访问权限。
        </p>
       </li>
       <li>
        <p>
         <code>
          AllowOverride none
         </code>
         禁止
         <code>
          .htaccess
         </code>
         文件覆盖主配置文件的设置，增强安全性。
        </p>
       </li>
       <li>
        <p>
         示例配置：
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <blockquote>
     <p>
      &lt;Directory /&gt;
      <br/>
      AllowOverride none
      <br/>
      Require all denied
      <br/>
      &lt;/Directory&gt;
     </p>
    </blockquote>
    <ul>
     <li>
      针对
      <code>
       htdocs
      </code>
      目录的配置：
     </li>
    </ul>
    <blockquote>
     <p>
      &lt;Directory "${SRVROOT}/htdocs"&gt;
      <br/>
      Options Indexes FollowSymLinks
      <br/>
      AllowOverride None
      <br/>
      Require all granted
      <br/>
      &lt;/Directory&gt;
     </p>
    </blockquote>
    <ul>
     <li>
      <p>
       <strong>
        禁止目录浏览功能
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         使用
         <code>
          Options -Indexes
         </code>
         禁止显示目录列表。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        分布式配置文件（.htaccess）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         允许在目录中使用
         <code>
          .htaccess
         </code>
         文件覆盖主配置文件的设置。
        </p>
       </li>
       <li>
        <p>
         示例：
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <blockquote>
     <p>
      &lt;Directory "/path/to/directory"&gt;
      <br/>
      AllowOverride All
      <br/>
      &lt;/Directory&gt;
     </p>
    </blockquote>
