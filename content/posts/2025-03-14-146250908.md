    <h3>
     1.加载 Docker 镜像文件
    </h3>
    <p>
     首先，你需要将 `.tar` 格式的镜像文件加载回 Docker 中。
    </p>
    <blockquote>
     <p>
      docker load -i &lt;镜像文件路径&gt;
     </p>
    </blockquote>
    <p>
     加载完成后，你可以使用 `docker images` 命令查看所有可用的镜像，确认加载是否成功。
    </p>
    <p>
     若出现
    </p>
    <p>
     Got permission denied while trying to connect to the Docker daemon socket at unix:///var/run/docker.sock: Post "http://%2Fvar%2Frun%2Fdocker.sock/v1.24/images/load?quiet=0": dial unix /var/run/docker.sock: connect: permission denied
    </p>
    <p>
     则
    </p>
    <blockquote>
     <p>
      sudo docker load -i &lt;镜像文件路径&gt;
     </p>
    </blockquote>
    <blockquote>
     <p>
      例： sudo docker load -i test.tar
     </p>
    </blockquote>
    <p class="img-center">
     <img alt="" height="257" src="https://i-blog.csdnimg.cn/direct/7ab6a3ae25f04f118e38ba719dfd648d.png" width="434"/>
    </p>
    <h3>
     2. 基于镜像启动个Docker 容器
    </h3>
    <blockquote>
     <p>
      sudo docker run -it --name &lt;容器名称&gt; &lt;镜像名称&gt; /bin/bash
     </p>
    </blockquote>
    <p>
     容器名称自定义，镜像名称即使上图最下面的名字
    </p>
    <blockquote>
     <p>
      例：sudo docker run -it --name pytorch test1_latest2 /bin/bash
     </p>
    </blockquote>
    <h3>
     3. 将容器内的文件复制到宿主机上
    </h3>
    <blockquote>
     <p>
      sudo docker cp &lt;容器名称&gt;:&lt;容器内文件路径&gt; &lt;宿主机目标路径&gt;
     </p>
    </blockquote>
    <p>
     首先查找文件路径 ，
    </p>
    <blockquote>
     <p>
      例：find / -name "*pytorch37*" 2&gt;/dev/null
     </p>
    </blockquote>
    <p class="img-center">
     <img alt="" height="35" src="https://i-blog.csdnimg.cn/direct/0a5df7470a6746c984e32961efcbd208.png" width="660"/>
    </p>
    <p>
     再复制
    </p>
    <blockquote>
     <p>
      例：sudo docker cp pytorch:/opt/conda/envs/pytorch37 /home/kaikaixinxin/下载/extracted_files
     </p>
    </blockquote>
    <p class="img-center">
     <img alt="" height="225" src="https://i-blog.csdnimg.cn/direct/79bb74ab29e749afb538e2fbe2bbf2be.png" width="363"/>
    </p>
    <h3>
     4. 停止并删除容器（可选）
    </h3>
    <p>
     如果你不再需要这个容器，可以使用以下命令停止并删除它
    </p>
    <blockquote>
     <p>
      sudo docker stop my_container
     </p>
     <p>
      sudo docker rm my_container
     </p>
     <p>
      例：
     </p>
     <p>
      sudo docker stop pytorch
     </p>
     <p>
      sudo docker rm pytorch
     </p>
    </blockquote>
    <h3>
     5.解除文件夹权限
    </h3>
    <blockquote>
     <p>
      sudo chmod -R 777 文件夹路径
     </p>
     <p>
      例：sudo chmod -R 777 pytorch37
     </p>
    </blockquote>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="207" src="https://i-blog.csdnimg.cn/direct/a9bacca4aca04b4395476550c780ca57.png" width="243"/>
    </p>
    <p>
     通过以上步骤，你就可以将 Docker 打包的镜像文件恢复为原始文件，并将其提取到宿主机上。
    </p>
