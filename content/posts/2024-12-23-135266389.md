    <p>
     Java程序的Hello World代码是一个简单的示例，用于展示如何在Java中编写和运行一个基本的"Hello, World!"程序。以下是一个简单的Java程序Hello World代码：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">HelloWorld</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Hello, World!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     这段代码包含以下几个部分：
    </p>
    <ol>
     <li>
      <p>
       <code>
        public class HelloWorld
       </code>
       : 这是定义一个名为
       <code>
        HelloWorld
       </code>
       的公共类。在Java中，每个程序必须包含一个主类，通常命名为
       <code>
        HelloWorld
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       <code>
        public static void main(String[] args)
       </code>
       : 这是主方法，它是Java程序的入口点。当程序运行时，它会首先执行这个方法。
      </p>
     </li>
     <li>
      <p>
       <code>
        System.out.println("Hello, World!");
       </code>
       : 这是Java中的输出语句，用于在控制台打印文本。在这个例子中，它将打印出"Hello, World!"。
      </p>
     </li>
    </ol>
    <p>
     要运行这个程序，你需要将其保存为一个名为
     <code>
      HelloWorld.java
     </code>
     的文件，然后使用Java编译器（如
     <code>
      javac
     </code>
     ）编译它，最后使用Java运行时环境（如
     <code>
      java
     </code>
     ）运行编译后的程序。
    </p>
