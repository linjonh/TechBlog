    <br/>
    <br/>
    1月19日消息，过去两个月内，微软在美国
    <p>
     专利 
 与商标局（U.S.Patentand Trademark Office）公布申请的专利超过500项。这也反映出微软在2006年中如何积极，因为 
 
   专利申请 
 通常会在提出的一年半后才公开。
     <br/>
     <br/>
     但最近引人注目的一项专利，涵盖了电脑可根据一些 
 
   数据 
 ，如人体的心跳、血压和面部表情等，采取适当行动的 
 
   技术 
 。英国泰晤士报本周刊出的一篇报导，暗指这类科技可能扮演老大哥的角色，例如在员工表现出压力过大，或没有发挥生产力时，通知雇主。
     <br/>
     <br/>
     通常不愿评论 
 
   产品 
 之个别应用的微软，罕见地由主管智慧财产与授权事务的副总Horacio Gutierrez发表声明：
     <br/>
     <br/>
     “这一项特定的专利申请，大致上，描述一种改善移动监视系统的创新，并以监督使用者的心跳为例，说明哪一种生理状态被侦测到时，显示使用者需要帮助，并让他们联络到期他可能提供帮助的使用者。大家必须谨记，和大多数从事创新的组织一样，我们若干的专利申请，是目前已经存在我们产品中的发明，其他的申请则代表尚在开发未来潜在应用的创新。”
     <br/>
     <br/>
     微软加速专利申请有几项原因，其中之一是竞争，和确保微软的敌手不会取得重要的创新。不过，该 
 
   公司 
 也在几年前展开大规模的知识产权授权，开放其技术给许多大、小公司使用。微软在此之后签署了一连串的专利交叉授权协议，最近一起是在15日与日本JVC的协议。
     <br/>
     <br/>
     微软最近公布的专利申请有部分涵盖了搜索与 
 
   广告 
 这个落后于Google的领域，如创造一个影音广告的现货市场，和创造一种结合影片和横幅广告的方式。
     <br/>
     <br/>
     其他值得注意的专利申请，包括可洗式键盘和可洗式滑鼠等硬件设计，另一项是所谓的受控拷贝（managedcopy），也就是让影片档或DVD等内容在数位权利管理（DRM）的保护下，允许其他人复制并用在不同的数位装置里，但不允许无限制的复制。
    </p>
