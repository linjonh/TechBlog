    <h4>
     <a id="032_1">
     </a>
     032-寻亲管理系统功能整理
    </h4>
    <h5>
     <a id="_3">
     </a>
     一、技术栈
    </h5>
    <ul>
     <li>
      <strong>
       后端
      </strong>
      ：Spring Boot框架
     </li>
     <li>
      <strong>
       前端（用户端）
      </strong>
      ：Uniapp（微信小程序）
     </li>
     <li>
      <strong>
       前端（管理端）
      </strong>
      ：Vue.js
     </li>
     <li>
      <strong>
       数据库
      </strong>
      ：MySQL
     </li>
    </ul>
    <h5>
     <a id="_9">
     </a>
     二、用户端主要功能
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        首页
       </strong>
      </p>
      <ul>
       <li>
        展示寻亲公告、最新失踪人员信息、热门话题等。
       </li>
       <li>
        提供快速导航链接到其他功能模块。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        失踪人员
       </strong>
      </p>
      <ul>
       <li>
        列出所有失踪人员的信息，包括姓名、年龄、失踪时间、失踪地点、照片等。
       </li>
       <li>
        提供筛选和排序功能，方便用户查找特定失踪人员。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        留言感谢
       </strong>
      </p>
      <ul>
       <li>
        用户可以对失踪人员信息发布者或帮助过的人进行留言感谢。
       </li>
       <li>
        显示感谢留言的列表，用户可查看和回复。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        交流论坛
       </strong>
      </p>
      <ul>
       <li>
        用户可以在论坛中发布帖子，讨论寻亲相关的话题。
       </li>
       <li>
        提供帖子浏览、回复、点赞等功能。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        我的模块
       </strong>
      </p>
      <ul>
       <li>
        用户个人信息管理，包括头像、昵称、联系方式等。
       </li>
       <li>
        显示用户的发帖记录、收藏记录、认领记录等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        认领操作
       </strong>
      </p>
      <ul>
       <li>
        用户若发现失踪人员信息与自己认识的失踪人员匹配，可进行认领操作。
       </li>
       <li>
        提交认领信息，包括联系方式、确认信息等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        认领信息
       </strong>
      </p>
      <ul>
       <li>
        显示用户提交的认领信息列表，包括已处理、待处理等状态。
       </li>
       <li>
        用户可查看认领详情，并与管理员进行沟通。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        我的收藏管理
       </strong>
      </p>
      <ul>
       <li>
        用户可以收藏感兴趣的失踪人员信息或帖子。
       </li>
       <li>
        提供收藏列表的查看和管理功能。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        我要发帖
       </strong>
      </p>
      <ul>
       <li>
        用户可以在此模块发布新的帖子到交流论坛。
       </li>
       <li>
        提供标题、内容、图片等输入项。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        我的发帖
       </strong>
      </p>
      <ul>
       <li>
        显示用户发布的所有帖子列表，包括帖子标题、内容、发布时间等。
       </li>
       <li>
        用户可查看帖子详情、编辑或删除帖子。
       </li>
      </ul>
     </li>
    </ol>
    <h5>
     <a id="_51">
     </a>
     三、管理端主要功能
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        首页
       </strong>
      </p>
      <ul>
       <li>
        展示系统概况，包括失踪人员数量、认领信息数量、用户数量等。
       </li>
       <li>
        提供快速导航链接到其他管理模块。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        个人中心
       </strong>
      </p>
      <ul>
       <li>
        管理员个人信息管理，包括头像、昵称、联系方式等。
       </li>
       <li>
        显示管理员的登录日志、操作记录等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        用户管理
       </strong>
      </p>
      <ul>
       <li>
        列出所有用户的信息，包括昵称、注册时间、登录次数等。
       </li>
       <li>
        提供用户信息的查看、编辑、删除等功能。
       </li>
       <li>
        可对用户进行权限管理，设置不同的角色和权限。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        失踪人员管理
       </strong>
      </p>
      <ul>
       <li>
        列出所有失踪人员的信息，包括姓名、年龄、失踪时间、失踪地点等。
       </li>
       <li>
        提供失踪人员信息的添加、编辑、删除等功能。
       </li>
       <li>
        可对失踪人员进行状态管理，如设置为已找到、已处理等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        认领信息管理
       </strong>
      </p>
      <ul>
       <li>
        列出所有用户提交的认领信息，包括认领人、认领对象、联系方式等。
       </li>
       <li>
        提供认领信息的查看、审核、处理等功能。
       </li>
       <li>
        可对认领信息进行状态管理，如设置为已处理、待处理等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        交流论坛
       </strong>
      </p>
      <ul>
       <li>
        列出所有用户发布的帖子，包括帖子标题、内容、发布时间等。
       </li>
       <li>
        提供帖子的查看、回复、删除等功能。
       </li>
       <li>
        可对帖子进行状态管理，如设置为精华帖、置顶帖等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        留言感谢
       </strong>
      </p>
      <ul>
       <li>
        列出所有用户的留言感谢信息，包括留言人、留言对象、留言内容等。
       </li>
       <li>
        提供留言感谢的查看、回复等功能。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        系统管理
       </strong>
      </p>
     </li>
    </ol>
