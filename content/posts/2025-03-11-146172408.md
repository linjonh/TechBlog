1）滚动条滑倒最底部

```
"""
滚动条：selenium中没有滚动条方法，需要js代码实现
1、准备js代码："window.scrollBy(0, 1000)"
2、执行js代码：driver.execute_script(js的变量)
模拟滚动条拉到最下面
"""

from time import sleep
from selenium import webdriver

driver = webdriver.Chrome()

driver.get("https://news.baidu.com/")
driver.maximize_window()
driver.implicitly_wait(30)

temp_height = 0
while True:
    # 1、准备js代码
    js_down = "window.scrollBy(0, 2000)"
    # 2、执行js代码
    # 循环将滚动条下拉
    driver.execute_script(js_down)
    # sleep一下让滚动条反应一下
    sleep(2)
    # 获取当前滚动条距离顶部的距离
    check_height = driver.execute_script(
        "return document.documentElement.scrollTop || window.pageYOffset || document.body.scrollTop;")
    print("yxf")
    print(check_height)
    # 如果两者相等说明到底了
    if check_height == temp_height:
        print("到底了")
        break
    temp_height = check_height


# 等待20秒
sleep(5)
# 清理资源
driver.quit()  # 关闭浏览器驱动实例，释放资源



```

操作步骤

Selenium框架中没有专门处理滚动条的方法，需要通过调用 Js 代码实现操作；

​ 1）、第一步：设置操作滚动条操作语句：js\_down="window.scollBy（0,1000)"

​ > 0:为左边距-----》水平滚动条

​ > 1000: 为垂直滚动条

​ 2）、第二步：调用执行js方法，将设置js语句传入方法中

​ > 方法：driver.execute\_script(js\_down)
