    <h2>
     <a id="MySQL_0">
     </a>
     远程连接MySQL数据库
    </h2>
    <h3>
     <a id="_2">
     </a>
     <strong>
      一、连接远程数据库：
     </strong>
    </h3>
    <p>
     <strong>
      1、显示密码
     </strong>
    </p>
    <p>
     如：MySQL 连接远程数据库（192.168.5.116），端口“3306”，用户名为“root”，密码“123456”
    </p>
    <pre><code class="prism language-shell">mysql <span class="token parameter variable">-h192.168.5.116</span> <span class="token parameter variable">-P3306</span> <span class="token parameter variable">-uroot</span> <span class="token parameter variable">-p123456</span>
</code></pre>
    <p>
     <strong>
      2、隐藏密码
     </strong>
    </p>
    <p>
     如：MySQL 连接本地数据库，用户名为“root”，
    </p>
    <pre><code class="prism language-shell">C:/<span class="token operator">&gt;</span>mysql <span class="token parameter variable">-hlocalhost</span> <span class="token parameter variable">-uroot</span> <span class="token parameter variable">-p</span>
</code></pre>
    <h3>
     <a id="mysql_22">
     </a>
     <strong>
      二、配置mysql允许远程链接：
     </strong>
    </h3>
    <p>
     默认情况下，**mysql帐号不允许从远程登陆，只能在localhost登录。**本文提供了二种方法设置mysql可以通过远程主机进行连接。
    </p>
    <p>
     <strong>
      (一)改表法
     </strong>
    </p>
    <p>
     在localhost登入mysql后，更改 “mysql” 数据库里的 “user” 表里的 “host” 项，将"localhost"改称"%"
    </p>
    <p>
     例如：
    </p>
    <pre><code class="prism language-mysql">update user set host = '%' where user = 'root';

select host, user from user;
</code></pre>
    <p>
     <strong>
      (二)授权法
     </strong>
    </p>
    <p>
     例如： 你想myuser使用mypassword（密码）从任何主机连接到mysql服务器的话。
    </p>
    <pre><code class="prism language-shell">mysql<span class="token operator">&gt;</span> GRANT ALL PRIVILEGES ON *.* TO <span class="token string">'myuser'</span>@<span class="token string">'%'</span> IDENTIFIED BY <span class="token string">'mypassword'</span> WITH GRANT OPTION<span class="token punctuation">;</span>
mysql<span class="token operator">&gt;</span>FLUSH PRIVILEGES<span class="token punctuation">;</span>
</code></pre>
    <p>
     如果你想允许用户myuser从ip为192.168.1.6的主机连接到mysql服务器，并使用mypassword作为密码
    </p>
    <pre><code class="prism language-shell">mysql<span class="token operator">&gt;</span>GRANT ALL PRIVILEGES ON *.* TO <span class="token string">'myuser'</span>@<span class="token string">'192.168.1.3'</span> IDENTIFIED BY *** WITH GRANT OPTION<span class="token punctuation">;</span>

mysql<span class="token operator">&gt;</span>FLUSH PRIVILEGES<span class="token punctuation">;</span>
</code></pre>
    <p>
     使修改生效，就可以了
    </p>
    <p>
     例子：
    </p>
    <pre><code class="prism language-mysql">grant all PRIVILEGES on *.* to 'root'@'%' identified by '123456' WITH GRANT OPTION;
</code></pre>
    <h3>
     <a id="_67">
     </a>
     <strong>
      常见问题：
     </strong>
    </h3>
    <p>
     1、在采用法二授权法之后，无法在本地登录mysql（如：
    </p>
    <p>
     #mysql -uroot -p -h192.168.5.116
    </p>
    <p>
     Enter password:ERROR 1045 (28000): Access denied for user ‘root’@‘loadb116’ (using password: YES) 上例中loadb116是主机名.
    </p>
    <h3>
     <a id="_75">
     </a>
     <em>
      <strong>
       解决方法
      </strong>
      ：
     </em>
    </h3>
    <p>
     1、这时可以使用：mysql -uroot -p 登录，进入到mysql后。
    </p>
    <pre><code class="prism language-shell">mysql<span class="token operator">&gt;</span> grant all privileges on *.* to <span class="token string">'root'</span>@<span class="token string">'loadb116'</span> identified by <span class="token string">'123456'</span> with grant option<span class="token punctuation">;</span>
Query OK, <span class="token number">0</span> rows affected <span class="token punctuation">(</span><span class="token number">0.00</span> sec<span class="token punctuation">)</span>

mysql<span class="token operator">&gt;</span> flush privileges<span class="token punctuation">;</span>
Query OK, <span class="token number">0</span> rows affected <span class="token punctuation">(</span><span class="token number">0.00</span> sec<span class="token punctuation">)</span>
</code></pre>
    <p>
     2、在本地使用ip地址登录
    </p>
    <pre><code class="prism language-mysql"> mysql -uroot -p -h192.168.5.116
</code></pre>
