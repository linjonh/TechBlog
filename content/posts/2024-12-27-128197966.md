    <p>
     在企业IT数字化转型趋势下，为支撑业务敏捷、稳定、创新发展，IT运维的重要性已日益凸显。伴随着IT运维管理目标、管理范围、管理对象及管理深度的转变，IT运维平台建设呈现管理一体化、规模扩大化、业务可视化、运维自动化、运维智能化的关键趋势转变，也就对IT运维软件供应商提出了打造统一运维平台软件的需求。
    </p>
    <h4>
     <a id="IT_1">
     </a>
     什么是IT统一运维软件？
    </h4>
    <p>
     如果用一句话来总结其定义：
     <br/>
     IT统一运维软件即拥有IT基础设施管理、IT服务管理、网络管理、网络性能分析管理等能力的IT运维软件解决方案。
     <br/>
     其强调的是“统一”的设计理念，解决的是IT运维人员长久以来最大痛点（经常迷失在各种IT运维工具/系统里，学习运维成本高，运维效率低），其目标是打造统一的IT运维平台，消除运维数据孤岛，端到端拉通运维业务需求，高效解决运维问题，最终通过IT运维创造价值。
    </p>
    <p>
     <em>
      IDC China ：
      <br/>
      The ITUO - IT Unified Operation market includes four functional markets:ITIM - IT Infrastructure Management; ITSM - IT Service Management; NMS - Network Management Software; NPAM - Network Performance Analysis and Management
     </em>
    </p>

     <a id="_8">
     </a>

     当前比较典型的ITUO软件架构如下，供参考：

    <p>
     基于开放的K8S+Docker容器底座，实现“云、网、端”异构全栈资源统一纳管监控，以IT资源配置管理（CMDB）为平台底层核心能力，通过构建运维数据中台，萃取运维数据价值，实现高效统一运维。ITSM开放灵活易用的服务流程编排引擎，实现业务敏捷及服务高效合规交付。以业务为中心，构建三层立体式数字化服务管理模型，实现端到端业务服务质量分析、实时感知用户体验。
     <br/>
     另外，也可在搭建的统一运维平台能力基础上，以行业化客户需求为导向，联合优质生态伙伴共同构建整体运维解决方案。
    </p>
    <h4>
     <a id="_15">
     </a>
     核心业务领域
    </h4>
    <h5>
     <a id="ITIMIT_Infrastructure_Management_16">
     </a>
     ITIM：IT Infrastructure Management
    </h5>
    <p>
     IT基础设施管理，提供最为关键的比如服务器、存储、操作系统、虚拟化、容器、数据库、中间件、应用资源等IT基础设施的监控管理能力；其中，核心业务领域除了硬件监控之外，重点关注APM（应用性能管理），为IT运维人员提供关键的应用性能、用户体验、故障分析诊断、业务调优等的基础支撑数据，大大提升运维效率。
     <br/>
     另外，一般来讲，统一运维需要纳管所有的IT资源，所以除监控资源外，还需搭建CMDB，通过CMDB可以实现对IT资源的统一管理，同时南向适配同步各类资源，并提供CI类型定义、关系定义、资源调和、变更审计等丰富的资源管理能力。CMDB可视化也是CMDB重要能力，需提供作业平台，支持拓扑编排、关系编排及展示、重要告警状态展示等功能。
    </p>
    <h5>
     <a id="ITSM_IT_Service_Management_22">
     </a>
     ITSM: IT Service Management
    </h5>
    <p>
     IT服务管理，是帮助IT运维人员对IT系统的运维进行有效管理的服务流程，是基于 ITIL（IT Infrastructure Library，IT基础架构标准库）标准的实践；ITSM与传统的IT管理不同，它是一种以服务为中心的IT管理；是以流程为中心，从复杂的IT管理活动中梳理出那些核心的流程，比如事故管理、问题管理和配置管理，将这些流程规范化、标准化，明确定义各个流程的目标和范围、成本和效益、运营步骤、关键成功因素和绩效指标、有关人员的责权利，以及各个流程之间的关系。
     <br/>
     一般来讲，ITSM核心能力是提供灵活、开放的流程设计&amp;编排以及业务表单设计能力，用以满足实际IT业务需求进行灵活的服务流程落地实施。
     <br/>
     实施ITSM的根本目标有三个：
     <br/>
     (1)以客户为中心提供IT服务；
     <br/>
     (2)提供高质量、低成本的服务；
     <br/>
     (3)提供的服务是可准确度量的。
    </p>
    <p>
     值得强调的是，ITSM适用于IT管理，而非企业的业务管理。对应ITUO软件来讲，ITSM解决的就是在IT统一运维各业务场景下，通过IT服务流程的拉通能力去承接和串联运维业务逻辑，以促成IT运维的端到端全流程闭环。
    </p>
    <h5>
     <a id="NMS__Network_Management_Software_34">
     </a>
     NMS: Network Management Software
    </h5>
    <p>
     网络管理软件，俗称“网管”，因为网管软件的发展在业界已经有长年的的积累和沉淀，相对来讲已经趋于成熟稳定。
     <br/>
     目前市场上，很多要求比较高的客户尤其是金融行业客户，对网络的运维需求，除了基础网管能力需求外，更多的是追求网络自动化/智能化运维方向的探索落地，比如网络自动驾驶、网络调优、网络故障自愈等方向。
    </p>
    <h5>
     <a id="NPAM_Network_Performance_Analysis_and_Management_40">
     </a>
     NPAM: Network Performance Analysis and Management
    </h5>
    <p>
     网络性能分析与管理，简单来讲就是对网络的性能进行分析和评价，帮助客户解决更深层次的网络问题。
     <br/>
     当今企业的核心网络设备上可能承载了成百上千个应用，传统网络运维手段只能通过分析设备运行指标来监控网络运行状态，无法精准地监控到每一条链路的网络性能及其承载应用的网络运行质量。当终端用户反馈到网络较差或应用访问体验不友好时，传统运维手段很难有效的快速定位问题原因，网络一旦产生故障，需要多部门配合、人工排查，往往处理问题进度慢、界定责任难。
     <br/>
     因此，NPAM主要提供全面监测网络流量访问路径各环节服务质量、并提供关键网络性能异常发现和根因定位的网络性能分析能力。
     <br/>
     例如，可以基于网络流量全数据源的网络性能分析，通过对网络中全量包的实时采集、分析，对网络和应用性能指标进行深度解析，加以精细化、图形化的手段呈现网络中的性能数据、状态及预测趋势；另外实时收集分布在网络各关键节点采集探针分析数据，快速理清网络、链路、应用访问路径各环节运行的异常，高效地定位网络故障发生的根源，对已发生的故障支持回溯取证以及责任界定；智能主动发现网络运行的异常，帮助用户定位分析网络运行过程中的问题。
    </p>

    <h2>
     <a id="_55">
     </a>
     市场空间如何？
    </h2>
    <p>
     据IDC中国发布的《IDC China Semiannual IT Unified Operation Software Tracker - Forecast, 2021H2》报告预测数据分析可以发现：
     <br/>
     1、国内ITUO软件市场继2020年受疫情影响有所降低，在2021年回暖以后，基本持续保持每年10%的增长幅度稳步提升，预计2026年整体市场规格将达到60亿人民币左右。
    </p>
    <p>

     2、按业务领域分，ITIM和NPAM市场空间增长迅速，同期ITSM市场有一定的增长，NMS市场基本趋于稳定。

    <h2>
     <a id="IT_67">
     </a>
     IT统一运维能给客户带来什么价值？
    </h2>
    <p>
     综合来说，统一运维，让新IT运维更简单。
     <br/>
     数字化转型趋势下，业务驱动运维向运营转型、云驱动运维向自动化转型、数据驱动运维向智能化转型。IT即服务，运维模式亟待创新升级。
     <br/>
     新华三发布统一运维服务2021业务全景，依托U-Center统一运维平台，实现基础交付价值化升级、管理模式多元化创新转型、服务管理智慧化运营发展，全方位诠释智能时代运维变革新理念。
     <br/>
     新华三统一运维解决方案已为3000+行业高端用户构建高效、全面的IT运维管理体系，为用户数字化转型，业务永续、创新发展保驾护航。
    </p>
