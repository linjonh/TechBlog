# 【已解决】linux装了jdk，却无法使用java命令

今天在用虚拟机linux的时候，需要kafka服务，却显示java命令确实，输入java -v却得不到结果，后来我这样解决的

## 解决方案：cat查看etc 的profile文件

看看java\_home在不在,在的话直接使用下面命令就行了

```go
source /etc/profile

```

完整命令如下

```go
cat /etc/profile
source /etc/profile
java -version

```

这样一般就行了，主要是环境变量没有生效产生的问题.
