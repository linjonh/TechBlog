     </a>
     Ubuntu安装虚拟环境-bash: /usr/bin/python: No such file or directory
    </h2>
    <p>
     出现报错
    </p>
    <pre><code class="prism language-sh">root@dev1:~# source .bashrc
-bash: /usr/bin/python: No such file or directory
virtualenvwrapper.sh: There was a problem running the initialization hooks.

If Python could not import the module virtualenvwrapper.hook_loader,
check that virtualenvwrapper has been installed for
VIRTUALENVWRAPPER_PYTHON=/usr/bin/python and that PATH is
set properly.
root@dev1:~# cd /usr/lib/python3
root@dev1:/usr/lib/python3# ls

</code></pre>
    <p>
     加上这两句话
    </p>
    <pre><code class="prism language-sh">export VIRTUALENVWRAPPER_PYTHON=/usr/bin/python3
export VIRTUALENVWRAPPER_VIRTUALENV=/usr/local/bin/virtualenv
</code></pre>

    </p>
    <p>
     重新source .bashrc即可
    </p>
