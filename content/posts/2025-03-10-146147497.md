    <p id="20250310103727-jbdgike">
     autoMate 是一款由开源开发的本地自动化工具，以 AI+RPA（人工智能+机器人流程自动化）为核心特色。它将大型语言模型的智能理解与 RPA 的流程执行能力结合，用户只需用自然语言描述任务，如“整理桌面文件”或“生成周报”，即可让电脑自动完成复杂操作。工具支持本地部署，保障数据安全，特别适合需要高效处理重复性任务的个人或团队。autoMate 打破传统 RPA 的复杂规则设置，通过 AI 驱动的直观交互，让自动化更智能、更易用。
    </p>

    </p>

    <h4 id="主要功能">
     主要功能
    </h4>
    <ul id="20250310104037-lmhfi0s">
     <li id="20250310104037-m9v8ncy">
      AI+RPA 任务执行：结合 AI 理解与 RPA 操作，自动完成多步骤工作流程。
     </li>
     <li id="20250310104037-dd4kzol">
      自然语言输入：用日常语言描述任务，无需编程或规则配置。
     </li>
     <li id="20250310104037-dqcx4ws">
      屏幕内容解析：智能识别界面元素，执行点击、输入等操作。
     </li>
     <li id="20250310104037-ucbtp9j">
      本地运行模式：全程离线运行，确保数据隐私。
     </li>
     <li id="20250310104037-5cpokhe">
      全界面适配：支持任意软件或网页的操作。
     </li>
     <li id="20250310104037-bctyyo6">
      多模型支持：兼容主流大语言模型，灵活配置。
     </li>
     <li id="20250310104037-bb1lfup">
      一键部署：简化安装流程，适配中文环境。
     </li>
    </ul>

     安装
    </h4>
    <p id="20250310104102-dsdxg0q">
     autoMate 的安装过程针对 AI+RPA 功能优化，确保用户快速上手。以下是详细步骤：
    </p>
    <ol id="20250310104102-nqgz26k">
     <li id="20250310104102-szyk90p">
      <p id="20250310104102-6aodr1s">

      </p>
      <ul id="20250310104102-d7oznno">
       <li id="20250310104102-g6yp9ze">

       </li>
       <li id="20250310104102-yvmcm71">
        安装 Miniconda：访问 Conda 官网。
       </li>
       <li id="20250310104102-02ys9iz">
        确认系统支持 Python 3.12。
       </li>
      </ul>
     </li>
     <li id="20250310104102-cdqeq0w">
      <p id="20250310104102-b7wo3lk">
       克隆项目：

      <ul id="20250310104102-efh9nei">
       <li id="20250310104102-dbm1fdk">
        <p id="20250310104102-au4chkv">
         在终端运行：
        </p>
        <pre><code>git clone https://github.com/yuruotong1/autoMate.git
</code></pre>
       </li>
       <li id="20250310104102-qc88r1k">
        <p id="20250310104102-zb3995t">
         进入目录：
        </p>
        <pre><code>cd autoMate
</code></pre>
       </li>
      </ul>
     </li>
     <li id="20250310104102-vkgwmdc">
      <p id="20250310104102-5nf3334">

         创建并激活 Conda 环境：
        </p>
        <pre><code>conda create -n "automate" python=3.12
conda activate automate
</code></pre>
       </li>
      </ul>
     </li>
     <li id="20250310104102-joolddt">

       <li id="20250310104102-9b5m0k9">

        </p>
        <pre><code>pip install -r requirements.txt
</code></pre>
       </li>
       <li id="20250310104102-pazsg7i">
        <p id="20250310104102-qx497nx">
         若提示版本冲突，可更新 pip：
        </p>
        <pre><code>pip install --upgrade pip
</code></pre>

      </ul>
     </li>

      <p id="20250310104102-r4mo4s5">
       启动工具：

      <ul id="20250310104102-0lxgdh3">
       <li id="20250310104102-26yztac">

</code></pre>

      </ul>
     </li>
