    <h2>
     <a id="R_21">

     R语言基础知识点总览
    </h2>
    <h3>
     <a id="1_R_23">
     </a>
     1. R语言简介
    </h3>
    <p>
     R是一种用于统计计算和图形的编程语言和软件环境。R语言由Ross Ihaka和Robert Gentleman在1993年在新西兰奥克兰大学统计系创建。R是GNU项目的一部分，并且其源代码在GNU通用公共许可证下发布。
    </p>
    <h3>
     <a id="2_R_26">
     </a>
     2. R语言安装与环境配置
    </h3>
    <ul>
     <li>
      <strong>
       安装R
      </strong>
      ：可以从
      <a href="https://www.r-project.org/" rel="nofollow">
       R官方网站
      </a>
      下载并安装适用于您操作系统的R版本。
     </li>
     <li>
      <strong>
       RStudio
      </strong>
      ：RStudio是一个强大的集成开发环境（IDE），建议安装以提高R编程效率。
     </li>
    </ul>
    <h3>
     <a id="3_R_30">
     </a>
     3. R语言基础语法
    </h3>
    <h4>
     <a id="31__31">
     </a>
     3.1 数据类型
    </h4>
    <p>
     R语言中的数据类型包括：
    </p>
    <ul>
     <li>
      数值型（Numeric）
     </li>
     <li>
      字符型（Character）
     </li>
     <li>
      逻辑型（Logical）
     </li>
     <li>
      因子型（Factor，用于分类数据）
     </li>
    </ul>
    <pre><code class="prism language-R"># 示例代码
num &lt;- 123  # 数值型
char &lt;- "Hello, R!"  # 字符型
log &lt;- TRUE  # 逻辑型
factor_var &lt;- factor(c("Male", "Female", "Male"))  # 因子型
</code></pre>
    <h4>
     <a id="32__46">
     </a>
     3.2 向量与矩阵
    </h4>
    <ul>
     <li>
      向量是R中的一维数组。
     </li>
     <li>
      矩阵是R中的二维数组。
     </li>
    </ul>
    <pre><code class="prism language-R"># 向量示例
vec &lt;- c(1, 2, 3, 4, 5)

# 矩阵示例
mat &lt;- matrix(1:9, nrow = 3, ncol = 3)
</code></pre>
    <h4>
     <a id="33__58">
     </a>
     3.3 数据框与列表
    </h4>
    <ul>
     <li>
      数据框（data.frame）是R中用于存储表格数据的一种结构，类似于Excel中的表格。
     </li>
     <li>
      列表（list）是R中最复杂的数据结构，可以包含不同类型的元素。
     </li>
    </ul>
    <pre><code class="prism language-R"># 数据框示例
df &lt;- data.frame(Name = c("Alice", "Bob", "Charlie"), Age = c(25, 30, 35))

# 列表示例
lst &lt;- list(num = 1:5, char = c("a", "b", "c"), log = TRUE)
</code></pre>
    <h3>
     <a id="4__70">
     </a>
     4. 控制结构
    </h3>
    <h4>
     <a id="41__71">
     </a>
     4.1 条件语句
    </h4>
    <pre><code class="prism language-R"># 条件语句示例
if (2 &gt; 1) {
  print("2 is greater than 1")
} else {
  print("2 is not greater than 1")
}
</code></pre>
    <h4>
     <a id="42__81">
     </a>
     4.2 循环结构
    </h4>
    <pre><code class="prism language-R"># for循环示例
for (i in 1:5) {
  print(i)
}

# while循环示例
count &lt;- 1
while (count &lt;= 5) {
  print(count)
  count &lt;- count + 1
}
</code></pre>
    <h3>
     <a id="5__96">
     </a>
     5. 函数
    </h3>
    <p>
     在R中，函数是组织代码的基本单元。
    </p>
    <pre><code class="prism language-R"># 自定义函数示例
my_function &lt;- function(x) {
  return(x^2)
}

# 调用函数
result &lt;- my_function(4)
print(result)
</code></pre>
    <h3>
     <a id="6__110">
     </a>
     6. 数据可视化
    </h3>
    <p>
     R语言拥有强大的绘图功能，常用的包包括
     <code>
      ggplot2
     </code>
     等。
    </p>
    <pre><code class="prism language-R"># 基础绘图示例
plot(1:10, rnorm(10), type = "o", col = "blue", xlab = "X轴", ylab = "Y轴", main = "基础绘图示例")

# 使用ggplot2绘图示例（需先安装并加载ggplot2包）
# install.packages("ggplot2")
library(ggplot2)
data &lt;- data.frame(x = rnorm(100), y = rnorm(100))
ggplot(data, aes(x = x, y = y)) + geom_point() + theme_minimal()
</code></pre>
