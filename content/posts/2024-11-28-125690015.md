    <h3>
     <a id="1python_0">
     </a>
     1、安装python
    </h3>
    <p>
     官网下载地址：
     <a href="https://www.python.org/ftp/python/3.8.0/python-3.8.0-amd64.exe" rel="nofollow">
      https://www.python.org/ftp/python/3.8.0/python-3.8.0-amd64.exe
     </a>
     <br/>
     双击打开.exe文件
     <br/>
     勾选 Add Python 3.8 to Path选项，然后点击install now即可安装。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/775fe37534ab3247ce37620fedec25ed.png#pic_center"/>
    </p>
    <p>
     安装中：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2df0217216fea1160029c0a8e57ea339.png#pic_center"/>
    </p>
    <p>
     安装完毕后点击close即可。
    </p>
    <h3>
     <a id="2_12">
     </a>
     2、测试
    </h3>
    <p>
     按键盘win+r，在左下角运行窗口里输入cmd，回车。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ff97b81b5a52ed116a33238a3e46c6ad.png#pic_center">
      <br/>
      在弹出的窗口里输入python，回车。得到如下显示。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/000cacc23a823f42d447a004ad8e2c2c.png#pic_center"/>
     </img>
    </p>
    <p>
     在提示符 &gt;&gt;&gt; 后面输入1+1，回车后显示2。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/69be56021f2cca687576356b773230dd.png#pic_center"/>
    </p>
    <p>
     安装成功。
    </p>
    <h3>
     <a id="3VScode_24">
     </a>
     3、安装VScode
    </h3>
    <p>
     官网下载地址：
     <a href="https://code.visualstudio.com" rel="nofollow">
      https://code.visualstudio.com
     </a>
     <br/>
     找到下载好的安装包，双击打开，一直点下一步即可。
    </p>
    <h3>
     <a id="4VScode_29">
     </a>
     4、设置VScode
    </h3>
    <p>
     打开安装好的VScode，可以先设置好中文界面和解决中文注释乱码问题，详细步骤见上一篇博客：
     <a href="https://blog.csdn.net/weixin_43737995/article/details/104214032?spm=1001.2014.3001.5502">
      https://blog.csdn.net/weixin_43737995/article/details/104214032?spm=1001.2014.3001.5502
     </a>
     <br/>
     按照下图步骤搜索并安装。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7a552ae9b9846061e648c5fd728e7a41.png#pic_center">
      <br/>
      添加一个准备存放代码的python文件夹。
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/05bc135e8337992624b1708e75b263f0.png#pic_center">
      <br/>
      打开设置。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/39acda2dab69e7aaecdf3a89069e4b38.png#pic_center"/>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b97aed9ca30d25a07afa20f5eab87396.png#pic_center">
      <br/>
      在settings.json中输入下列代码，用来配置flake8和yapf并关闭pylint工具。
     </img>
    </p>
    <pre><code>{
    "python.linting.flake8Enabled": true,
    "python.formatting.provider": "yapf",
    "python.linting.flake8Args": ["--max-line-length=248"],
    "python.linting.pylintEnabled": false
}
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b48b43afd0087435edf7078b023a2abd.png#pic_center"/>
     <br/>
     保存并关闭。
    </p>
    <h3>
     <a id="5_56">
     </a>
     5、测试
    </h3>
    <p>
     在所选择的文件夹下新建一个python文件，以.py结束。
     <br/>
     在文件中输入要测试的代码：
    </p>
    <pre><code>print('hello world')
</code></pre>
    <p>
     运行并查看结果。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/97d96713034f973b736aee3fbfcf2817.png#pic_center"/>
     <br/>
     结果输出正常则安装完毕。
    </p>
