    <h2>
     引言
    </h2>
    <p>
     <strong>
      启航篇，理解什么是数据结构和算法
     </strong>
    </p>
    <p>
     <strong>
      <span style="background-color:#f3f3f4">
       在 C 语言编程领域，
       <span style="color:#fe2c24">
        数据结构
       </span>
       和
       <span style="color:#fe2c24">
        算法
       </span>
       是两个核心且紧密相关的概念
      </span>
     </strong>
    </p>
    <h2>
     一、数据结构
    </h2>
    <h3>
     定义
    </h3>
    <blockquote>
     <p>
      <strong>
       数据结构是指相互之间存在一种或多种特定关系的
       <span style="color:#fe2c24">
        数据元素的集合
       </span>
       <span style="color:#ff9900">
        （比如数组）
       </span>
       ，它是组织和存储数据的方式，
       <span style="color:#fe2c24">
        目的是
       </span>
       <span style="background-color:#fefcd8">
        为了更高效地对数据进行访问、插入、删除、修改等操作。
       </span>
      </strong>
     </p>
     <p>
      <strong>
       不同的数据结构适用于不同的应用场景，合理选择数据结构可以显著提高程序的性能。
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      下面以数组来理解一下
     </strong>
    </p>
    <h3>
     例子：数组（Array）
    </h3>
    <blockquote>
     <p>
      <strong>
       <span style="background-color:#fefcd8">
        数组就是一种很简单的数据结构
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="color:#fe2c24">
        数组是一种线性数据结构
       </span>
       ，它由
       <span style="color:#ff9900">
        相同类型的元素组成
       </span>
       ，这些元素在内存中连续存储。可以通过下标快速访问数组中的任意元素。
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;

int main() {
    // 定义一个包含5个整数的数组
    int arr[5] = {1, 2, 3, 4, 5};

    // 访问数组元素
    for (int i = 0; i &lt; 5; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}
</code></pre>
    <blockquote>
     <h6>
      当然数据结构还有很多很多，这里是以数组作为一个例子，带你理解数据结构这个定义，如果还是不理解数据结构的话，没关系，当你
      <span style="color:null">
       <span style="background-color:#fff5e6">
        学了许多数据结构后
       </span>
      </span>
      ，你肯定会有理解，这时再看这个定义，就可以联想到那些数据结构，肯定是可以理解的
     </h6>
    </blockquote>
    <p>
     <strong>
      数据结合和算法是相互配合使用的，下面来看什么是算法
     </strong>
    </p>
    <h2>
     二、算法
    </h2>
    <h3>
     定义
    </h3>
    <blockquote>
     <p>
      <strong>
       算法是指解决特定问题的一系列明确的、有限的
       <span style="color:#fe2c24">
        操作步骤
       </span>
       。它是
       <span style="color:#fe2c24">
        对特定问题求解步骤的一种描述
       </span>
       ，是指令的有限序列，其中每一条指令表示一个或多个操作。算法具有有穷性、确定性、可行性、输入和输出等特性。
      </strong>
     </p>
    </blockquote>
    <h3>
     例子1. 排序算法 - 冒泡排序（Bubble Sort）
    </h3>
    <blockquote>
     <p>
      <strong>
       <span style="color:#fe2c24">
        冒泡排序是一种简单的排序
       </span>
       <span style="color:#511b78">
        算法
       </span>
       ，它重复地走访过要排序的数列，一次比较两个元素，如果它们的顺序错误就把它们交换过来。
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;

// 冒泡排序
void bubbleSort(int arr[], int n) {
    for (int i = 0; i &lt; n - 1; i++) {
        for (int j = 0; j &lt; n - i - 1; j++) {
            if (arr[j] &gt; arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    // 调用冒泡排序函数
    bubbleSort(arr, n);

    // 输出排序后的数组
    for (int i = 0; i &lt; n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}
</code></pre>
    <blockquote>
     <p>
      <strong>
       这里的冒泡排序，就是一种算法，说白了，
       <span style="color:#511b78">
        <span style="background-color:#38d8f0">
         算法就是解决某个特定类型的问题而写的代码程序
        </span>
       </span>
      </strong>
     </p>
    </blockquote>
    <h3>
     例子2. 搜索算法 - 线性搜索（Linear Search）
    </h3>
    <p>
     <strong>
      线性搜索是一种简单的搜索算法，它从数组的第一个元素开始，逐个比较元素，直到找到目标元素或遍历完整个数组。
     </strong>
    </p>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;

// 线性搜索
int linearSearch(int arr[], int n, int target) {
    for (int i = 0; i &lt; n; i++) {
        if (arr[i] == target) {
            return i; // 找到目标元素，返回其下标
        }
    }
    return -1; // 未找到目标元素，返回 -1
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = sizeof(arr) / sizeof(arr[0]);
    int target = 30;

    // 调用线性搜索函数
    int result = linearSearch(arr, n, target);

    if (result != -1) {
        printf("目标元素 %d 的下标是 %d\n", target, result);
    } else {
        printf("未找到目标元素 %d\n", target);
    }

    return 0;
}
</code></pre>
    <blockquote>
     <p>
      <strong>
       这里的搜索算法，就是一种算法，是为了解决搜索问题而有的代码
      </strong>
     </p>
    </blockquote>
    <h2>
     三、数据结构与算法的关系
    </h2>
    <blockquote>
     <p>
      <strong>
       <span style="background-color:#ffd900">
        数据结构和算法是相辅相成的。
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="background-color:#e7fafa">
        数据结构是算法实现的基础，不同的数据结构会影响算法的实现方式和效率；
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="background-color:#e7fafa">
        算法则是对数据结构进行操作的具体步骤，合理的算法可以充分发挥数据结构的优势，提高程序的性能。
       </span>
      </strong>
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       <span style="background-color:#f3f3f4">
        例如，在一个大规模的数据集合中进行查找操作，如果使用数组存储数据，可能使用线性搜索算法，时间复杂度为 O(n)；如果使用哈希表这种数据结构，结合哈希查找算法，平均时间复杂度可以达到 O(1)。（看不懂没关系，以后学到了肯定就明白了）。
       </span>
      </strong>
     </p>
    </blockquote>
