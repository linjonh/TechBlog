    <h4>
     <a id="Inline_CSS_0">
     </a>
     一、行内样式（Inline CSS）
    </h4>
    <h5>
     <a id="1__2">
     </a>
     1. 定义与语法
    </h5>
    <p>
     行内样式是直接在HTML标签中使用
     <code>
      style
     </code>
     属性来定义样式。这种方式只对当前标签生效。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>tagname</span> <span class="token special-attr"><span class="token attr-name">style</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span><span class="token value css language-css">css 样式</span><span class="token punctuation">"</span></span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="2__9">
     </a>
     2. 示例
    </h5>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span> <span class="token special-attr"><span class="token attr-name">style</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span><span class="token value css language-css"><span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span> <span class="token property">font-size</span><span class="token punctuation">:</span> 14px<span class="token punctuation">;</span></span><span class="token punctuation">"</span></span></span><span class="token punctuation">&gt;</span></span>这是一个红色的段落文字。<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>div</span> <span class="token special-attr"><span class="token attr-name">style</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span><span class="token value css language-css"><span class="token property">width</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span> <span class="token property">height</span><span class="token punctuation">:</span> 100px<span class="token punctuation">;</span> <span class="token property">background-color</span><span class="token punctuation">:</span> lightblue<span class="token punctuation">;</span></span><span class="token punctuation">"</span></span></span><span class="token punctuation">&gt;</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>div</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="3__15">
     </a>
     3. 优点
    </h5>
    <ul>
     <li>
      <strong>
       精准控制
      </strong>
      ：可以对单个HTML元素进行精确的样式设置。
     </li>
     <li>
      <strong>
       简单直接
      </strong>
      ：无需额外的CSS文件或
      <code>
       &lt;style&gt;
      </code>
      标签，直接在标签中定义样式。
     </li>
    </ul>
    <h5>
     <a id="4__19">
     </a>
     4. 缺点
    </h5>
    <ul>
     <li>
      <strong>
       可维护性差
      </strong>
      ：每个元素都需要单独设置样式，修改时需要逐个修改。
     </li>
     <li>
      <strong>
       代码冗余
      </strong>
      ：如果多个元素有相同的样式，会导致代码重复。
     </li>
    </ul>
    <h5>
     <a id="5__23">
     </a>
     5. 适用场景
    </h5>
    <p>
     适用于需要对单个元素进行临时样式调整的情况，或者在开发过程中进行快速样式测试。
    </p>
    <h4>
     <a id="Internal_CSS_26">
     </a>
     二、内部样式表（Internal CSS）
    </h4>
    <h5>
     <a id="1__28">
     </a>
     1. 定义与语法
    </h5>
    <p>
     内部样式表是将CSS样式定义在HTML文件的
     <code>
      &lt;style&gt;
     </code>
     标签中，通常放在
     <code>
      &lt;head&gt;
     </code>
     部分。这种方式对整个HTML文档生效。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>head</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
        <span class="token selector">选择器</span> <span class="token punctuation">{<!-- --></span>
            css 样式
        <span class="token punctuation">}</span>
    </span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>head</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="2__41">
     </a>
     2. 示例
    </h5>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>head</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css">
        <span class="token selector">body</span> <span class="token punctuation">{<!-- --></span>
            <span class="token property">font-family</span><span class="token punctuation">:</span> Arial<span class="token punctuation">,</span> sans-serif<span class="token punctuation">;</span>
            <span class="token property">background-color</span><span class="token punctuation">:</span> #f0f0f0<span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
        <span class="token selector">h1</span> <span class="token punctuation">{<!-- --></span>
            <span class="token property">color</span><span class="token punctuation">:</span> blue<span class="token punctuation">;</span>
            <span class="token property">text-align</span><span class="token punctuation">:</span> center<span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
        <span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
            <span class="token property">color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
            <span class="token property">line-height</span><span class="token punctuation">:</span> 1.6<span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
    </span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>head</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>body</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>h1</span><span class="token punctuation">&gt;</span></span>欢迎来到我的网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>h1</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>这是一个段落文字。<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>body</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="3__65">
     </a>
     3. 优点
    </h5>
    <ul>
     <li>
      <strong>
       集中管理
      </strong>
      ：所有样式都在一个地方定义，便于维护和修改。
     </li>
     <li>
      <strong>
       减少代码冗余
      </strong>
      ：可以为多个元素定义相同的样式，避免重复代码。
     </li>
    </ul>
    <h5>
     <a id="4__69">
     </a>
     4. 缺点
    </h5>
    <ul>
     <li>
      <strong>
       文件内嵌
      </strong>
      ：样式代码嵌入在HTML文件中，如果HTML文件较大，可能会增加文件体积。
     </li>
     <li>
      <strong>
       无法复用
      </strong>
      ：样式仅在当前HTML文件中有效，无法在其他页面中复用。
     </li>
    </ul>
    <h5>
     <a id="5__73">
     </a>
     5. 适用场景
    </h5>
    <p>
     适用于单个HTML文件的样式定义，或者在开发过程中进行样式测试和调试。
    </p>
    <h4>
     <a id="External_CSS_76">
     </a>
     三、外部样式表（External CSS）
    </h4>
    <h5>
     <a id="1__78">
     </a>
     1. 定义与语法
    </h5>
    <p>
     外部样式表是将CSS样式定义在一个单独的
     <code>
      .css
     </code>
     文件中，然后在HTML文件中通过
     <code>
      &lt;link&gt;
     </code>
     标签引入。这种方式适用于多个HTML文件共享相同的样式。
    </p>
    <h5>
     <a id="2__81">
     </a>
     2. 步骤
    </h5>
    <ol>
     <li>
      创建一个CSS文件（例如
      <code>
       styles.css
      </code>
      ）。
     </li>
     <li>
      在HTML文件的
      <code>
       &lt;head&gt;
      </code>
      部分使用
      <code>
       &lt;link&gt;
      </code>
      标签引入CSS文件。
     </li>
    </ol>
    <h5>
     <a id="3_CSSstylescss_85">
     </a>
     3. CSS文件（styles.css）示例
    </h5>
    <pre><code class="prism language-css"><span class="token selector">body</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">font-family</span><span class="token punctuation">:</span> Arial<span class="token punctuation">,</span> sans-serif<span class="token punctuation">;</span>
    <span class="token property">background-color</span><span class="token punctuation">:</span> #f0f0f0<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token selector">h1</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> blue<span class="token punctuation">;</span>
    <span class="token property">text-align</span><span class="token punctuation">:</span> center<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
    <span class="token property">line-height</span><span class="token punctuation">:</span> 1.6<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="4_HTML_101">
     </a>
     4. HTML文件示例
    </h5>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>head</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>link</span> <span class="token attr-name">rel</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>stylesheet<span class="token punctuation">"</span></span> <span class="token attr-name">type</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>text/css<span class="token punctuation">"</span></span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>styles.css<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>head</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>body</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>h1</span><span class="token punctuation">&gt;</span></span>欢迎来到我的网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>h1</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>p</span><span class="token punctuation">&gt;</span></span>这是一个段落文字。<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>p</span><span class="token punctuation">&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>body</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h5>
     <a id="5__112">
     </a>
     5. 优点
    </h5>
    <ul>
     <li>
      <strong>
       高度复用
      </strong>
      ：一个CSS文件可以被多个HTML文件引用，便于统一管理样式。
     </li>
     <li>
      <strong>
       提高可维护性
      </strong>
      ：修改样式时只需修改CSS文件，所有引用该文件的HTML页面都会自动更新。
     </li>
     <li>
      <strong>
       分离结构与表现
      </strong>
      ：HTML文件专注于内容结构，CSS文件专注于样式表现，符合网页设计的最佳实践。
     </li>
    </ul>
    <h5>
     <a id="6__117">
     </a>
     6. 缺点
    </h5>
    <ul>
     <li>
      <strong>
       额外文件
      </strong>
      ：需要创建和维护额外的CSS文件。
     </li>
     <li>
      <strong>
       加载时间
      </strong>
      ：浏览器需要额外请求CSS文件，可能稍微增加页面加载时间（但可以通过缓存优化）。
     </li>
    </ul>
    <h5>
     <a id="7__121">
     </a>
     7. 适用场景
    </h5>
    <p>
     适用于大型网站或项目，尤其是多个页面需要共享相同样式的情况。
    </p>
    <h4>
     <a id="_124">
     </a>
     四、选择器的使用
    </h4>
    <p>
     在CSS中，选择器用于选择要应用样式的HTML元素。常见的选择器包括：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        元素选择器
       </strong>
       ：选择特定的HTML元素。
      </p>
      <pre><code class="prism language-css"><span class="token selector">h1</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> blue<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        类选择器
       </strong>
       ：选择具有特定类属性的元素。
      </p>
      <pre><code class="prism language-css"><span class="token selector">.my-class</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        ID选择器
       </strong>
       ：选择具有特定ID属性的元素。
      </p>
      <pre><code class="prism language-css"><span class="token selector">#my-id</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">width</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        组合选择器
       </strong>
       ：可以组合使用多种选择器，精确选择元素。
      </p>
      <pre><code class="prism language-css"><span class="token selector">div p</span> <span class="token punctuation">{<!-- --></span>
    <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="CSS_156">
     </a>
     五、CSS样式的作用范围
    </h4>
    <ol>
     <li>
      <strong>
       全局样式
      </strong>
      ：在
      <code>
       &lt;style&gt;
      </code>
      标签或外部CSS文件中定义的样式，对整个文档或所有引用的HTML文件生效。
     </li>
     <li>
      <strong>
       局部样式
      </strong>
      ：在特定的HTML标签中使用
      <code>
       style
      </code>
      属性定义的样式，仅对该标签生效。
     </li>
    </ol>
    <h4>
     <a id="CSS_161">
     </a>
     六、CSS样式的优先级
    </h4>
    <p>
     当多种样式规则同时作用于一个HTML元素时，优先级规则如下（从高到低）：
    </p>
    <ol>
     <li>
      <strong>
       行内样式
      </strong>
      ：直接在HTML标签中定义的样式。
     </li>
     <li>
      <strong>
       内部样式表
      </strong>
      ：在
      <code>
       &lt;style&gt;
      </code>
      标签中定义的样式。
     </li>
     <li>
      <strong>
       外部样式表
      </strong>
      ：在单独的CSS文件中定义的样式。
     </li>
    </ol>
    <p>
     如果优先级相同，则后定义的样式会覆盖先定义的样式。
    </p>
    <h4>
     <a id="_171">
     </a>
     七、总结
    </h4>
    <ul>
     <li>
      <strong>
       行内样式
      </strong>
      ：适用于单个元素的临时样式调整。
     </li>
     <li>
      <strong>
       内部样式表
      </strong>
      ：适用于单个HTML文件的样式定义。
     </li>
     <li>
      <strong>
       外部样式表
      </strong>
      ：适用于多个HTML文件共享的样式定义，便于维护和复用。
     </li>
    </ul>
