# 微软全面拥抱 AI！

11月4日，一年一度的微软IT大会Ignite在美国奥兰多举行，来自全球的2.6万名IT从业者、开发者、数据专家出席会议。在本次会议中，微软发布了多项最新的技术、产品、服务与解决方案，涵盖从Azure到经典的办公服务等多个领域。CSDN记者应邀出席了本次大会，并从现场发回了最新报道。

作为专业的技术社区，本篇文章除了向大家介绍微软本次发布的重要服务和产品之外，也特别为国内的开发者整理了新发布的开发工具。接下来就让我们一起来看看本次Ignite大会的那些亮点吧~

美国东部时间11月4日~8日，一年一度的微软IT大会Ignite在美国奥兰多举行。作为微软最重要的技术会议之一，每年微软都会在Iginite大会上宣布一系列的产品及服务的升级，也会发布一些新的技术解决方案。今年也是如此。

## **萨提亚：** **让所有公司都成为软件技术公司**

## 

在万众瞩目中，微软CEO萨提亚·纳德拉（Satya Nadella）以「技术密集度」（Tech Intensity）为主题的演讲拉开了大会的序幕，并以一种「行为艺术」的方式来阐释选择这个主题的原因。

![640?wx_fmt=jpeg](https://i-blog.csdnimg.cn/blog_migrate/fdb79b70f0c03c0caae392f3b5a0eb04.png)

*微软CEO萨提亚·纳德拉*

在过去的110天里，一个临时组成的开发团队在奥兰多环球影城实现了一个新的展厅。利用Azure Kinect的AI传感器所提供的计算机视觉能力和语音模型，萨提亚（或其他参观者）可以亲自在英雄拯救人质的展览情节中与机器进行交互，并最终解救出人质。

![640?wx_fmt=png](https://i-blog.csdnimg.cn/blog_migrate/9e86b087f5aec3563866cc382e931b37.png)

*萨提亚体验用Azure Kinect实现的环球影城新展厅*

技术不再仅仅是少数互联网巨头的专利，而是被更多像环球影城这样的组织应用到自身的场景之中。

为了实现这一愿景，萨提亚按照从底层到应用层技术栈的顺序依次宣布了微软的一系列新动作。包括：Azure、信任、开发者工具、Power Platform、Dynamics 365、Microsoft 365。

## **「云+终端」打造「世界计算网络」**

## 

在今年的大会中，Azure是发布了最多新东西的产品。在揭晓这些新的服务之前，萨提亚首先回顾了Azure最初的愿景。他谈到，提供云服务并不是微软的最终目标，2030年，全球终端设备的数量将达到500亿，2025年的数据产生量将达到175ZB。

这样的终端和数据量让云成为了链接世界数据的网络，而这种链接将会把每一个终端变成一个计算节点，这些节点将编织成一个全球的计算网络。

而Azure今天发布的新服务，都是围绕这一初衷展开的。

**1. 发布Azure Arc实现混合云部署**

Azure Arc是一套技术解决方案，它所实现的功能主要包括以下三个：

* 跨环境运行，使Azure的服务和管理功能拓展到任何系统或设备上。包括Linux、Windows Server，以及运行在本地、多云环境或者是边缘设备上的任何Kubernetes集群上。用户在本地部署、多云环境和边缘设备都能获得连贯、一致的体验；
* 规模化管理Kubernetes应用，用DevOps部署和管理Kubernetes应用，并可确保应用的部署和配置与最初的一致，并且有一致的操作体验；
* 数据服务多点并行，不同终端和系统中数据的状态实时更新，如修正、升级和安全维护，保持数据的一致性和处于最新的状态。

![640?wx_fmt=png](https://i-blog.csdnimg.cn/blog_migrate/dca2e8f26a3148167ff888823653e358.png)

**2. 发布Azure Stack Edge，提供边缘计算能力**

Azure Stack Edge是一种支持人工智能的边缘托管主机，能够将计算、存储和智能带到任何边缘环境中。

相比于Azure Stack与Azure构建的混合云解决方案，Azure Stack Edge可以提供更加灵活的混合云部署选项，包括虚拟机支持、基于GPU的架构形态、多节点的高可用性，以及多接入边缘计算（MEC）。

**3. 发布Azure Synapse Analytics提供数据分析服务**

Azure Synapse Analytics是一种不设限的分析服务，它在Azure SQL Data Warehouse的基础上加入了增强功能，可以从各类数据源、数据仓库和大数据分析系统中获取和汇总数据洞察，从而实现用户数据的协同工作。

Azure Synapse Analytics主要是针对数据湖（data lake）和数据仓库（data warehouse）这两套独立的数据分析系统的信息不对等的问题而设计的。Azure Synapse Analytics主要包含2个主要特点：

* 支持大规模数据检索，Azure Synapse Analytics可以实现通过简单的SQL语言对关系型数据和非关系型数据的检索，并可以对大规模的并行检索进行优化；
* 分析能力强大，Azure Synapse Analytics将Power BI和Azure机器学习进行了整合，可以对你的数据进行分析，并且在App上使用机器学习模型。

## **Microsoft 365新增多项人工智能能力**

作为微软经典的办公服务，Microsoft 365本次也带来很多更新，并且发布了全新的「Project Cortex」。本次更新主要集中在效率、知识、工作流、安全、管理几个方面。

**1. Microsoft Team加入多窗口聊天功能**

作为微软的办公写作软件，能够在Microsoft Team中实现多窗口聊天功能的呼声一直很高。这次Microsoft Team的升级不仅加入这一功能，还增加了如隐私频道、会议和语音等功能，除此之外，微软这次还把「海外版钉钉」Yammer直接加入到了Microsoft Team中，用户可以直接把Yammer拖拽到左边的功能栏中。

![640?wx_fmt=gif](https://i-blog.csdnimg.cn/blog_migrate/2e45d156ca7ef54188a0b17e7058ca24.png)

**2. Edge支持企业用户标签页、Bing与Microsoft搜索整合**

在这次大会上，微软发布了一款基于Chromium内核的全新Edge浏览器。这款浏览器支持为企业用户提供新的标签页，让企业用户可以直接访问
**Microsoft 365文档、站点或者进行内部网络搜索**
，把每个标签页变成了新的办公入口。此外，这款新的Edge还适用于所有设备平台，包括移动终端，用户在手机上就可以检索公司的文件。

![640?wx_fmt=gif](https://i-blog.csdnimg.cn/blog_migrate/aaa4390fa798cfa737ad75970397f58e.png)

除了Edge之外，Bing搜索也做出了很大的改变。首先是Bing搜索与Microsoft搜索进行了整合，打通了内网和外网，在搜索框中就能对自己的文件、联系人、办公室平面图、公司属于缩写等信息进行搜索。

其二，Bing搜索正在朝着计算知识引擎的方向迈进。与目前常规的搜索引擎只能给出链接的反馈方式不同，随着搜索引擎对语义理解的深入，下一代搜索引擎可以根据语义分析直接给出搜索的答案，这一点与WolframAlpha很像，当你问「什么是G大调」的时候，搜索结果不是一堆简单的链接，而是直接会演奏出来；或者当你输入「抛10次，4次正面向上」的时候，引擎可以很快理解你想问抛硬币的概率问题。

在Ignite大会上，当讲者在Bing里输入一个人名的时候，搜索出来的结果是根据他的联系人信息给出的，而不是简单地抓取所有叫这个名字的人作为搜索结果。

**3. 发布办公应用跨平台解决方案Fluid Framwork**

Fluid Framework是一款跨平台协作的解决方案，它的主要作用有以下3点：

* 实现用户基于Web和文档的同时编辑；
* 实现了文档的拆分、多应用内转发和内容整合；
* 加入了智能办公辅助功能，包括文字翻译、编辑校对等。

**4. 小娜（Cortana）成为Outlook的智能语音助手**

小娜作为语音助手可以实现邮件搜索、处理能功能，并且可以朗读邮件。有趣的是，在邮件朗读的声音选项中，不仅支持女声，也支持男声的朗读，不知道听小娜用男声读邮件是怎样的体验？

此外，小娜还可以根据邮件的内容对会议安排作出整理，归纳相关的文档，以及对当天需要的安排作出提醒。

![640?wx_fmt=png](https://i-blog.csdnimg.cn/blog_migrate/4b20f8731c5e324079e1787918c5ff1d.png)

不过在现场演示中，当讲者语音输入的时候，小娜并未能识别，在后来的采访中，官方给出了回应称是因现场网络状态所导致。但不管怎样，自然语言处理是一个庞大而复杂的技术体系，小娜作为语音助手的体验到底如何，只能等时间来验证了。

**5. Office系列软件的人工智能**

语音识别不仅在邮箱中进行应用，在Office系列软件中也有所体现。比如升级后的Excel支持语音检索，并且实现语音转文字。此外，新的Excel还加入了图片文字识别功能，对图片上的信息进行拍照，就可以自动生成表格，甚至在输入框中输入参数，Excel也会根据参数自动生成相应的表格。

## **开发者工具升级**

## 

**1. Microsoft Graph增加推送提醒**

Microsoft Graph是Microsoft 365中通往数据和智能的网关。它提供统一的可编程模型，可用于访问Office、Windows 10和企业移动性+安全性中的海量数据。

在本次大会中，Microsoft Graph新增了移动应用的推送提醒功能，可以对包括Windows、iOS、Android在内移动应用进行推送提醒。目前升级版的Microsoft Graph已经可以下载到测试版。

**2. MSIX支持虚拟环境下的动态交付**

MSIX是微软去年发布的一种安装包格式，是对MSI的升级。MSIX汇集了App和MSI文件封装格式的所有功能，支持所有Windows文件格式，包括Win32、WPF、WinForm和UWP。开发者只需单独一种封包即可完成安装容器，而不需要考虑不同的格式，也方便通过企业网络或者微软商店来集中部署。

在本次升级中，MSIX支持了虚拟环境下的动态交付，这一升级让开发者不需要为没使用过的应用扩展VM映像。同时也提升了网络效率。

**3. Excel新增操作录制功能**

在这次的Excel的升级中，新增了一个新的工具Office Script。这个工具是为哪些Excel的初学者打造的。老手可以把自己在Excel表格中的操作录下来发给新手。录制的脚本保存之后可以直接在Excel表格中被执行。

![640?wx_fmt=gif](https://i-blog.csdnimg.cn/blog_migrate/6b640d1b8a3ad040ca746f3e0f512350.png)

**4. 支持Windows系统的React Native可正式使用**

支持Windows v0.60版本的React Native目前已可以正式使用，这次发布支持日夜模式、键盘处理模式和签名模式。

同时，为了吸引更多的Web开发者，Windows版本的React Native还支持了JavaScript语言。想要使用这一版本的React Native的同学，可以参考官方的学习文档进行上手：

https://github.com/microsoft/react-native-windows/blob/master/vnext/docs/GettingStarted.md

## **当技术改变世界**

在萨提亚的演讲中，他多次提到「赋能」（empower）这个词：为「为企业拥有技术能力赋能」、「为用户赋能」等，这种「赋能」打破了传统互联网的技术垄断，降低了企业信息化转型的门槛；这种「赋能」使微软从5年前市值不足3000亿美元，到今天突破万亿；这种「赋能」，让微软今天有了「让所有公司都成技术公司」的远见。

正如E=mc^2可以揭示宇宙的终极真理一样，在技术改赋能的进程中，萨提亚也用一个公式来揭示微软技术赋能的本质：

技术密集度=（技术接受度×技术能力）^Trust

的确，这个世界正在被技术改变。
