    <h2>
     <a id="_0">
     </a>
     二叉树的层序遍历
    </h2>
    <p>
     使用队列的数据结构实现
    </p>
    <p>
     <strong>
      思路：
     </strong>
    </p>
    <p>
     先将根节点入队列，然后不断从队列中取出节点并访问，然后将该节点的左右孩子依次放入队列，重复过程直到队列为空
    </p>
    <p>
     <strong>
      实现：
     </strong>
    </p>
    <p>
     用队列q1来遍历，队列的类型为treenode*，
    </p>
    <p>
     用二维向量v1，报错输出的结果，因为题目中的返回类型就是二维向量。
    </p>
    <p>
     按层来存储节点的值，也就是每一层的节点值都要存储在一个
     <code>
      vector&lt;int&gt;
     </code>
     里，然后再把这个一维向量添加到二维向量
     <code>
      v1
     </code>
     中。可以借助一个内层循环来处理每一层的节点。
    </p>
    <p>
     记录当前层的节点数量
     <code>
      int size = q1.size()
     </code>
     ；
    </p>
    <p>
     遍历for循环，循环次数就是当前层次数size，for循环内部：
    </p>
    <p>
     获取队首元素，返回给一个current的树节点
    </p>
    <p>
     队首元素出队
    </p>
    <p>
     队首元素已经保存到了current中，所以把值加入到当前层中。
    </p>
    <p>
     判断左右孩子，如果存在，那么入队。
    </p>
    <p>
     把每一层用1个1维的向量存储：vector currentLevel;
    </p>
    <p>
     for循环结束，就是一层结束，把当前层加入到结果的二维向量中。即：v1.push_back( currentLevel );
    </p>
    <p>
     下面是完整的代码：
    </p>
    <pre><code class="prism language-C++">class Solution {
public:
    vector&lt;vector&lt;int&gt;&gt; levelOrder(TreeNode* root) {
        // 定义队列,类型要设置为节点treenode类型
        queue&lt;TreeNode*&gt; q1;
        //用向量存储这个输出
        vector&lt;vector&lt;int&gt;&gt; v1;

        //树为空
        if(root == nullptr){
            return v1;
        }
        
        //如果queue是int，则这里要-&gt;val,  如果是treenode类型，那么current即可。
        q1.push(root);
        
        //遍历队列，依次出队，然后当前节点左右孩子加入队列
        while(!q1.empty()){
            int size = q1.size();
            vector&lt;int&gt; currentLevel;
            
            //遍历for循环，循环次数就是当前层次数size
            for(int i=0; i&lt;size; i++){
                //取出队首节点
                TreeNode* current = q1.front();
                q1.pop();   //队首节点出队列
                currentLevel.push_back(current-&gt;val);   //这里必须是val，因为容器是int

                if(current-&gt;left){
                    q1.push(current-&gt;left);
                }
                if(current-&gt;right){
                    q1.push(current-&gt;right);
                }
            }
            v1.push_back(currentLevel);
        }
        return v1;
    }
};  
</code></pre>
    <h4>
     <a id="queue_77">
     </a>
     队列queue
    </h4>
    <p>
     queue的定义方式
    </p>
    <p>
     方式一： 使用默认的适配器定义队列。
    </p>
    <pre><code class="prism language-c++">queue&lt;int&gt; q1;
</code></pre>
    <p>
     方式二： 使用特定的适配器定义队列。
    </p>
    <pre><code class="prism language-c++">queue&lt;int, vector&lt;int&gt;&gt; q2;
queue&lt;int, list&lt;int&gt;&gt; q3;
</code></pre>
    <p>
     注意： 如果没有为queue指定特定的底层容器，默认情况下使用deque。
    </p>
    <p>
     queue当中常用的成员函数如下：
    </p>
    <p>
     成员函数 功能
    </p>
    <blockquote>
     <p>
      empty 判断队列是否为空
     </p>
     <p>
      size 获取队列中有效元素个数
     </p>
     <p>
      front 获取队头元素
     </p>
     <p>
      back 获取队尾元素
     </p>
     <p>
      push 队尾入队列
     </p>
     <p>
      pop 队头出队列
     </p>
     <p>
      swap 交换两个队列中的数据
     </p>
    </blockquote>
    <p>
     示例：
    </p>
    <pre><code class="prism language-c++">#include &lt;iostream&gt;
#include &lt;list&gt;
#include &lt;queue&gt;
using namespace std;
int main()
{
        queue&lt;int, list&lt;int&gt;&gt; q;
        q.push(1);
        q.push(2);
        q.push(3);
        q.push(4);
        cout &lt;&lt; q.size() &lt;&lt; endl; //4
        while (!q.empty())
        {
                cout &lt;&lt; q.front() &lt;&lt; " ";
                q.pop();
        }
        cout &lt;&lt; endl; //1 2 3 4
        return 0;
}
</code></pre>
