# 我的2013——微软技术之旅新起点

由于今年进了一家新的公司，新的项目要求用微软的几个核心产品来做信息化云平台，于是从零
开始
学习微软windows server服务器和网络的基础知识，如AD域、DNS、CA证书等等，接着又开始学习Sharepoint Server 2013、Exchange Server 2013、Lync Server 2013、Office Web APPS等几个核心产品，并且进行了规划、部署、配置和二次开发的学习。结合之前所学的ASP.NET，我们组在11月份完成了信息化云平台的初步建设，从此我便踏上了微软技术之旅。

在学习部署开发的过程中，才真正发现微软的强大，同时，我也与CSDN建立了密不可分的关系。因为在学习的过程中，发现太多的问题，于是不断地在网上寻找答案，或者加技术群请教牛人等，于是我的问题营垒一个个被攻破，而这些大部分的功劳要功于各个
牛人前辈的技术博客。然而，我也发现了许多的问题是网上找不到的，原因有很多：1、有的问题太细节，很多
牛人没空写，2、我学的是微软最新的产品，与之前的版本会有差别，研究的人还不多，当然网上的教程出来的也少。针对以上的问题，我的解决方法如下：首先，找
各大

牛人的博客关于之前版本的教程，接着结合
微软官方
技术论坛最新版本的帮助文档
，然后配上自己屡次不断的实验，最后问题迎刃而解。其中有一捷径，就是在QQ技术群中缠住
牛人追问到底，快速获得解决方法，但捷径归捷径，不是每次都有用的。这样学习的过程中，积累了一些的知识以后，于是我就开始在CSDN上写博客，把那些在前人的经验上结合自己的实践
积累
的一些知识和问题的解决方法做一个记录。现在我已养成了一个习惯，就是每做一个部署，或每解决一个问题，都要截图，以便写博客来记录。

总之，这一年是我技术之旅新的起点，我受益匪浅，我对微软产生了浓厚的兴趣，这条微软技术之路我会一直走下去的！同时，我也与CSDN结下了不解之缘，这个博客我会一直写下去的！
