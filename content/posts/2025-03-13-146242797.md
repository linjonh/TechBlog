    <p>
     打印一个菱形图案。程序分为两部分：上半部分和下半部分。上半部分打印一个逐渐增大的星号图案，下半部分打印一个逐渐缩小的星号图案。
    </p>
    <pre><code class="language-cpp">#include&lt;stdio.h&gt;
int main() {
	int row = 5;//定义行数
	int t = 2;
	for (int i = row; 0 &lt; i; i--)
	{
		for (int k = i - 1; k &gt; 0; k--) 
		{putchar(' ');}
		for (int g = t - 1; g &gt; 0; g--) 
		{putchar('*');}
		t = t + 2;
		printf("\n");
	}
	t = t - 4;
	for (int i2 = 1; i2 &lt;row; i2++) 
	{
		for (int k2 = i2; k2 &gt; 0; k2--)
		{putchar(' ');}
		for (int g2 = t - 1; g2 &gt; 0; g2--)
		{putchar('*');}
		t = t - 2;
		printf("\n");
	}
}</code></pre>
    <p>
    </p>
    <p>
     简化版：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        单层循环
       </strong>
       ：使用一个外层循环遍历所有行，一个内层循环遍历所有列。
      </p>
     </li>
     <li>
      <p>
       <strong>
        数学逻辑
       </strong>
       ：通过
       <code>
        abs(i) + abs(j) &lt; row
       </code>
       判断是否打印星号，直接确定菱形边界。
      </p>
     </li>
     <li>
      <p>
       <strong>
        减少变量
       </strong>
       ：不再需要单独处理上下半部分，统一用数学逻辑处理。
      </p>
     </li>
    </ol>
    <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include&lt;stdlib.h&gt;
int main() {
    int row = 5; // 定义行数
    int i, j;
    // 打印整个菱形
    for (i = -row + 1; i &lt; row; i++) {
        for (j = -row + 1; j &lt; row; j++) {
            if (abs(i) + abs(j) &lt; row) putchar('*'); // 打印星号
            else putchar(' '); // 打印空格
        }
        printf("\n");
    }
    return 0;
}</code></pre>
