    <p>
    </p>
    <br/>
    <code>
     @ConfigurationProperties
    </code>
    是 Spring 在
    <code>
     org.springframework.boot.context.properties
    </code>
    包中提供的一个注解。它的作用是根据注解中配置的前缀对 SpringBoot 配置文件（即
    <code>
     application.xxx
    </code>
    ）中前缀相同的配置进行属性绑定。
    <p>
    </p>
    <p>
     对于
     <code>
      @ConfigurationProperties
     </code>
     注解，通常有如下应用场景：
    </p>
    <ul>
     <li>
      场景一：如果需要让 Spring 当中的 Bean 与
      <code>
       application.xxx
      </code>
      中相关的配置进行绑定，在单个配置绑定的情况下，使用
      <code>
       @Value
      </code>
      注解十分方便，但是如果有多个配置都需要绑定的情况下，虽然仍然可以使用
      <code>
       @Value
      </code>
      对其进行配置绑定，但一个个绑定太过麻烦。
     </li>
     <li>
      场景二：如果需要让一个外部类（如第三方 Jar 包）中的配置，在无法修改外部类中的代码的情况下，需要将外部类与
      <code>
       application.xxx
      </code>
      中的相关配置进行绑定，需要在配置类中声明外部 Bean 并进行配置绑定。
     </li>
    </ul>
    <p>
     <code>
      @ConfigurationProperties
     </code>
     注解就可以解决上述两个问题。
    </p>
    <h4>
     <a id="_Bean__11">
     </a>
     内部 Bean 配置绑定
    </h4>
    <p>
     使用
     <code>
      @ConfigurationProperties
     </code>
     可以对 IOC 容器内部的 Bean 进行配置绑定。例如：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">import</span> <span class="token import"><span class="token namespace">lombok<span class="token punctuation">.</span></span><span class="token class-name">Data</span></span><span class="token punctuation">;</span>
<span class="token keyword">import</span> <span class="token import"><span class="token namespace">org<span class="token punctuation">.</span>springframework<span class="token punctuation">.</span>boot<span class="token punctuation">.</span>context<span class="token punctuation">.</span>properties<span class="token punctuation">.</span></span><span class="token class-name">ConfigurationProperties</span></span><span class="token punctuation">;</span>
<span class="token keyword">import</span> <span class="token import"><span class="token namespace">org<span class="token punctuation">.</span>springframework<span class="token punctuation">.</span>stereotype<span class="token punctuation">.</span></span><span class="token class-name">Component</span></span><span class="token punctuation">;</span>

<span class="token annotation punctuation">@Data</span>
<span class="token annotation punctuation">@Component</span>
<span class="token annotation punctuation">@ConfigurationProperties</span><span class="token punctuation">(</span>prefix <span class="token operator">=</span> <span class="token string">"school"</span><span class="token punctuation">)</span>
<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">SchoolProperties</span> <span class="token punctuation">{<!-- --></span>

    <span class="token keyword">private</span> <span class="token class-name">String</span> name<span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">String</span> address<span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">String</span> type<span class="token punctuation">;</span>

<span class="token punctuation">}</span>
</code></pre>
    <p>
     对于以上案例的
     <code>
      application.yml
     </code>
     配置：
    </p>
    <pre><code class="prism language-yaml"><span class="token key atrule">school</span><span class="token punctuation">:</span>
  <span class="token key atrule">name</span><span class="token punctuation">:</span> 人才大学
  <span class="token key atrule">address</span><span class="token punctuation">:</span> 长沙市
  <span class="token key atrule">type</span><span class="token punctuation">:</span> A_School
</code></pre>
    <p>
     上述案例中，声明了一个名为
     <code>
      SchoolProperties
     </code>
     的 Spring Bean，通过
     <code>
      @ConfigurationProperties(prefix = "school")
     </code>
     绑定
     <code>
      application.yml
     </code>
     中以
     <code>
      school
     </code>
     为前缀的三个配置
     <code>
      name
     </code>
     、
     <code>
      address
     </code>
     和
     <code>
      type
     </code>
     。
    </p>
    <h4>
     <a id="_Bean__45">
     </a>
     第三方 Bean 配置绑定
    </h4>
    <p>
     使用
     <code>
      @ConfigurationProperties
     </code>
     也可以对 IOC 容器外部（即第三方包中提供的 Bean）进行配置绑定。例如：
    </p>
    <pre><code class="prism language-java"><span class="token annotation punctuation">@Configuration</span>
<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">SchoolConfig</span> <span class="token punctuation">{<!-- --></span>
    
    <span class="token annotation punctuation">@Bean</span>
    <span class="token annotation punctuation">@ConfigurationProperties</span><span class="token punctuation">(</span>prefix <span class="token operator">=</span> <span class="token string">"school"</span><span class="token punctuation">)</span>
    <span class="token keyword">public</span> <span class="token class-name">SchoolProperties</span> <span class="token function">schoolProperties</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
        <span class="token keyword">return</span> <span class="token keyword">new</span> <span class="token class-name">SchoolProperties</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
    
<span class="token punctuation">}</span>
</code></pre>
    <p>
     上述案例中，假设
     <code>
      SchoolProperties
     </code>
     是一个外部第三方 Jar 包中的类。因为无法修改
     <code>
      SchoolProperties
     </code>
     的源码，所以通过在一个
     <code>
      SchoolConfig
     </code>
     的配置类中手动使用
     <code>
      @Bean
     </code>
     向 Spring IOC 容器中注入了一个
     <code>
      SchoolProperties
     </code>
     类型的 Bean，并且通过
     <code>
      @ConfigurationProperties(prefix = "school")
     </code>
     将配置文件中以
     <code>
      school
     </code>
     为前缀的配置进行属性绑定。
    </p>
