    <p>
     <strong>
      （对比）
     </strong>
     <br/>
     前后端是指代码上的，前后台是指用户视觉上的。
     <br/>
     <strong>
      （前台）
     </strong>
     <br/>
     普通用户以及游客所能看到的界面，如淘宝网站里的各个界面内容。
     <br/>
     <strong>
      （后台）
     </strong>
     <br/>
     系统管理员所能浏览到的界面，用于后台数据的调动，普通用户看不到，如后台管理系统。
     <br/>
     <strong>
      （前端）
     </strong>
     <br/>
     指界面展示效果所用到的html、css、javascript等语言，其作用是显示效果，如网站里的各个页面内容，又比如界面的UI设计。
     <br/>
     <strong>
      （后端）
     </strong>
     <br/>
     指在项目中的controller、model等业务逻辑方面的代码，以及mysql。
    </p>
