    <h2>
     搜索引擎
    </h2>
    <h3>
     一.zoomeye
    </h3>
    <p>
     <a href="https://www.zoomeye.org/" rel="nofollow" title="ZoomEye - Cyberspace Search Engine">
      ZoomEye - Cyberspace Search Engine
     </a>
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/e03d4fde39b245f38cd8e7f44c7408aa.png"/>
    </p>
    <p>
     ZoomEye是一个专为网络安全专业人员和研究人员设计的搜索引擎。它允许用户通过扫描IP地址和识别漏洞来搜索连接到互联网的设备和服务。
    </p>
    <h4>
     关于zoomeye的一些简单使用方法：
    </h4>
    <h5>
     基本搜索指令：
    </h5>
    <p>
     直接在搜索栏中使用关键字(例如，apache来查找apache服务器)。
     <br/>
     结合关键字以获得更具体的结果(例如，端口:80国家:美国)。
    </p>
    <h5>
     <br/>
     过滤器:
    </h5>
    <p>
     应用位置过滤器(如中国为country:CN)或服务类型过滤器(如service:http)。
    </p>
    <h5>
     <br/>
     搜索操作符:
    </h5>
    <p>
     对确切的短语使用引号:“IoT设备”。
     <br/>
     使用AND/OR之类的逻辑运算符(例如，service:http或service:https)。
    </p>
    <h5>
     <br/>
     输出结果:
    </h5>
    <p>
     寻找以CSV等格式下载搜索结果的选项，以便进行进一步分析。
     <br/>
     API命令(如果使用API):
    </p>
    <p>
     参考API文档中以编程方式获取数据的命令(例如，GET /data?query=YOUR_QUERY)。
    </p>
    <h3>
     二.fofa
    </h3>
    <p>
     <a href="https://fofa.info/" rel="nofollow" title="网络空间测绘，网络空间安全搜索引擎，网络空间搜索引擎，安全态势感知 - FOFA网络空间测绘系统">
      网络空间测绘，网络空间安全搜索引擎，网络空间搜索引擎，安全态势感知 - FOFA网络空间测绘系统
     </a>
    </p>
    <p style="text-align:center;">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/b250e0cf49424bd398233692876ac1d6.png"/>
    </p>
    <h4>
     fofa能做什么
    </h4>
    <p>
     1.搜索各种物联网设备、服务器和其他联网设备。
     <br/>
     2.分析从设备收集的数据，发现漏洞或错误配置。
     <br/>
     3.使用复杂查询，可以根据设备类型、协议和其他参数对结果进行过滤。
     <br/>
     4.访问世界各地联网设备的庞大数据库。
    </p>
    <h4>
     关于fofa的一些使用小贴士
    </h4>
    <h5>
     搜索特定服务:
    </h5>
    <p>
     title:"Router" -查找标题中有"Router"的设备。
    </p>
    <h5>
     <br/>
     识别开放的端口:
    </h5>
    <p>
     port:22 -列出SSH(端口22)打开的设备。
    </p>
    <h5>
     <br/>
     查找特定的设备类型:
    </h5>
    <p>
     设备:"webcam" -搜索连接互联网的网络摄像头。
    </p>
    <h5>
     <br/>
     通过软件定位设备:
    </h5>
    <p>
     app:"nginx" -搜索运行nginx web服务器的设备。
    </p>
    <h5>
     <br/>
     特定国家的搜索:
    </h5>
    <p>
     国家:"US"和标题:"surveillance" -在美国找到监控摄像头。
    </p>
    <h5>
     <br/>
     按SSL证书查询:
    </h5>
    <p>
     SSL .cert.organization:"Example Corp" -查找具有颁发给特定组织的SSL证书的设备。
    </p>
    <h5>
     <br/>
     基于位置的查询:
    </h5>
    <p>
     城市:“北京”-检索位于北京的设备。
    </p>

    <h2>
     拓展：子域名收集
    </h2>
    <h3>
     一.应用工具：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        Sublist3r
       </strong>
       ：Python工具，专门用于快速收集子域名。它通过搜索引擎和一些公共API（如VirusTotal）进行查询，输出结果为一个可读的列表。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Amass
       </strong>
       ：开源工具，支持多种子域名收集技术，包括DNS查询、网络爬虫和证书透明度日志。它还具有图形可视化功能，可以帮助你分析子域名之间的关系。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Assetfinder
       </strong>
       ：可以通过Google、Github、Common Crawl等多个来源快速发现子域名。它适合快速收集信息。
      </p>
     </li>
    </ul>
    <h3>
     二.搜索引擎
    </h3>
    <p>
     例如zoomeye
    </p>
    <ul>
     <li>
      <strong>
       查找子域名
      </strong>
      ：
      <code>
       host:example.com
      </code>
     </li>
     <li>
      <strong>
       查找特定服务的子域名
      </strong>
      ：
      <code>
       host:example.com app:"Apache"
      </code>
     </li>
     <li>
      <strong>
       查找特定端口的子域名
      </strong>
      ：
      <code>
       host:example.com port:80
      </code>
     </li>
     <li>
      <strong>
       结合国家代码
      </strong>
      ：
      <code>
       host:example.com country:CN
      </code>
     </li>
    </ul>
