       ②.数据可靠性
       <br/>
       1.数据包完整性 通过报文的数据校验位 来保证数据包的完整性
       <br/>
       2.防丢包 每次通过反向的ACK来保证不丢包
      </p>
     </li>
     <li>
      <p>
       传输层协议-UDP
       <br/>
       ·不面向链接(不需要向TCP那样建立链接才可进行通讯)
       <br/>
       ·不提供数据可靠性校验
       <br/>
       UDP 相比 TCP 传输速度要快
      </p>
     </li>
     <li>
      <p>
       如何选择TCP 与 UDP?
       <br/>
       · 容错性
       <code>
        高 --&gt; UDP 低 --&gt; TCP
       </code>
       <br/>
       · 数据包大小
       <code>
        小 --&gt; UDP 大 --&gt; TCP
       </code>
       <br/>
       · 承受丟包风险
       <code>
        是 --&gt; UDP 否 --&gt; TCP
       </code>
      </p>
