     C#中除了Dictionary，List，HashSet，HashTable 还有哪些可以保存列表的数据类型？
    </h1>
   </div>
  </div>
 </div>

    <p>
     在 C# 中，除了
     <code>
      Dictionary
     </code>
     、
     <code>
      List
     </code>
     、
     <code>
      HashSet
     </code>
     和
     <code>
      Hashtable
     </code>
     之外，还有许多其他可以保存列表或集合类型的数据结构，具体包括以下几类：
    </p>
    <hr/>
    <h3>
     <strong>
      📌 数组类
     </strong>
    </h3>
    <h4>
     1.
     <strong>
      Array
     </strong>
     （数组）
    </h4>
    <ul>
     <li>
      固定长度，性能高，适用于已知大小的数据集。
     </li>
     <li>
      适合频繁访问但不需要修改大小的场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">int[] arr = new int[5] { 1, 2, 3, 4, 5 };</code></pre>
    <hr/>
    <h3>
     <strong>
      📌 列表类
     </strong>
    </h3>
    <h4>
     2.
     <strong>
      Queue&lt;T&gt;
     </strong>
     （队列）
    </h4>
    <ul>
     <li>
      先进先出（FIFO）结构。
     </li>
     <li>
      适用于按顺序处理任务的场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">Queue&lt;int&gt; queue = new Queue&lt;int&gt;(); queue.Enqueue(1); int item = queue.Dequeue(); // 取出第一个元素</code></pre>
    <hr/>
    <h4>
     3.
     <strong>
      Stack&lt;T&gt;
     </strong>
     （栈）
    </h4>
    <ul>
     <li>
      后进先出（LIFO）结构。
     </li>
     <li>
      适用于逆序或撤销操作的场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">Stack&lt;int&gt; stack = new Stack&lt;int&gt;(); stack.Push(1); int item = stack.Pop(); // 取出最后一个元素</code></pre>
    <hr/>
    <h4>
     4.
     <strong>
      LinkedList&lt;T&gt;
     </strong>
     （链表）
    </h4>
    <ul>
     <li>
      双向链表，支持在头尾或中间快速插入和删除操作。
     </li>
     <li>
      适用于频繁插入和删除的场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">LinkedList&lt;int&gt; list = new LinkedList&lt;int&gt;(); list.AddFirst(1); list.AddLast(2);</code></pre>
    <hr/>
    <h3>
     <strong>
      📌 集合类
     </strong>
    </h3>
    <h4>
     5.
     <strong>
      SortedSet&lt;T&gt;
     </strong>
     （有序集合）
    </h4>
    <ul>
     <li>
      保证元素唯一性，并且自动按升序排序。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">SortedSet&lt;int&gt; set = new SortedSet&lt;int&gt;(); set.Add(3); set.Add(1); set.Add(2); // 结果：1, 2, 3（自动排序）</code></pre>
    <hr/>
    <h4>
     6.
     <strong>
      ConcurrentBag&lt;T&gt;
     </strong>
     （线程安全集合）
    </h4>
    <ul>
     <li>
      允许并发访问的集合，适用于多线程场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">ConcurrentBag&lt;int&gt; bag = new ConcurrentBag&lt;int&gt;(); bag.Add(1); bag.Add(2);</code></pre>
    <hr/>
    <h4>
     7.
     <strong>
      BlockingCollection&lt;T&gt;
     </strong>
     （阻塞集合）
    </h4>
    <ul>
     <li>
      提供线程安全的生产者/消费者模式。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">BlockingCollection&lt;int&gt; collection = new BlockingCollection&lt;int&gt;(); collection.Add(1); int item = collection.Take(); // 阻塞直到有数据</code></pre>
    <hr/>
    <h4>
     8.
     <strong>
      ObservableCollection&lt;T&gt;
     </strong>
     （可观察集合）
    </h4>
    <ul>
     <li>
      当集合发生变化时会触发通知（通常用于数据绑定）。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">ObservableCollection&lt;int&gt; observableList = new ObservableCollection&lt;int&gt;();
observableList.CollectionChanged += (sender, e) =&gt; 
{ 
    Console.WriteLine("Collection changed"); 
}; 
observableList.Add(1);</code></pre>
    <hr/>
    <h3>
     <strong>
      📌 映射类
     </strong>
    </h3>
    <h4>
     9.
     <strong>
      SortedDictionary&lt;TKey, TValue&gt;
     </strong>
     （有序字典）
    </h4>
    <ul>
     <li>
      按键的升序排序，基于
      <code>
       Red-Black Tree
      </code>
      实现。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">SortedDictionary&lt;int, string&gt; dict = new SortedDictionary&lt;int, string&gt;(); 
dict.Add(2, "B"); 
dict.Add(1, "A"); 
dict.Add(3, "C"); // 输出顺序为：1 -&gt; 2 -&gt; 3</code></pre>
    <hr/>
    <h4>
     10.
     <strong>
      SortedList&lt;TKey, TValue&gt;
     </strong>
     （有序列表）
    </h4>
    <ul>
     <li>
      基于数组，按键排序，插入和删除速度较慢。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">SortedList&lt;int, string&gt; sortedList = new SortedList&lt;int, string&gt;(); 
sortedList.Add(1, "A"); 
sortedList.Add(2, "B");</code></pre>
    <hr/>
    <h4>
     11.
     <strong>
      ConcurrentDictionary&lt;TKey, TValue&gt;
     </strong>
     （线程安全字典）
    </h4>
    <ul>
     <li>
      线程安全的键值对集合，适合在多线程场景下使用。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">ConcurrentDictionary&lt;int, string&gt; dict = new ConcurrentDictionary&lt;int, string&gt;();
dict.TryAdd(1, "A");</code></pre>
    <hr/>
    <h3>
     <strong>
      📌 特殊集合类
     </strong>
    </h3>
    <h4>
     12.
     <strong>
      BitArray
     </strong>
     （位数组）
    </h4>
    <ul>
     <li>
      用于高效存储和操作位（布尔值）。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">BitArray bits = new BitArray(8); 
bits[0] = true;</code></pre>
    <hr/>
    <h4>
     13.
     <strong>
      NameValueCollection
     </strong>
     （键值对集合，允许重复键）
    </h4>
    <ul>
     <li>
      允许键重复，存储字符串键值对。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">NameValueCollection collection = new NameValueCollection(); 
collection.Add("key", "value1"); 
collection.Add("key", "value2");</code></pre>
    <hr/>
    <h4>
     14.
     <strong>
      HybridDictionary
     </strong>
     （小规模时使用
     <code>
      ListDictionary
     </code>
     ，大规模时自动切换到
     <code>
      Hashtable
     </code>
     ）
    </h4>
    <ul>
     <li>
      在数据量少时使用
      <code>
       ListDictionary
      </code>
      ，大时切换为
      <code>
       Hashtable
      </code>
      。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">HybridDictionary hybridDict = new HybridDictionary(); 
hybridDict.Add("key", "value");</code></pre>
    <hr/>
    <h4>
     15.
     <strong>
      ImmutableArray&lt;T&gt;, ImmutableList&lt;T&gt;, ImmutableDictionary&lt;TKey, TValue&gt;
     </strong>
     （不可变集合）
    </h4>
    <ul>
     <li>
      定义后不可修改，适用于线程安全场景。
     </li>
    </ul>
    <pre></pre>
    <p>
     csharp
    </p>
    <p>
    </p>
    <pre><code class="language-cs">var immutableList = ImmutableList.Create(1, 2, 3); 
immutableList = immutableList.Add(4);</code></pre>

    <p>
     如果你要在
     <strong>
      多线程环境下操作
     </strong>
     ，建议用
     <code>
      ConcurrentDictionary
     </code>
     、
     <code>
      ConcurrentBag
     </code>
     或
     <code>
      BlockingCollection
     </code>
     。
     <br/>
     如果需要
     <strong>
      有序性
     </strong>
     ，用
     <code>
      SortedList
     </code>
     或
     <code>
      SortedDictionary
     </code>
     。
     <br/>
     如果要
     <strong>
      去重
     </strong>
     ，用
     <code>
      HashSet
     </code>
     或
     <code>
      SortedSet
     </code>
     。
