    <h2 id="1%EF%BC%8CCreate" name="1%EF%BC%8CCreate">
     1，Create
    </h2>
    <p>
     <strong>
      语法:
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       insert into 表名  (指定列) values  (values1,values2,...)
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      示例：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       创建一张学生表
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; create table students(
       <br/>
       -&gt; id int unsigned primary key auto_increment,
       <br/>
       -&gt; sn int not null unique comment '学号',
       <br/>
       -&gt; name varchar(20) not null,
       <br/>
       -&gt; qq varchar(20) unique);
      </strong>
     </p>
    </blockquote>
    <h4 id="1.1%EF%BC%8C%E5%8D%95%E8%A1%8C%E6%95%B0%E6%8D%AE%2B%E5%85%A8%E5%88%97%E6%8F%92%E5%85%A5%C2%A0" name="1.1%EF%BC%8C%E5%8D%95%E8%A1%8C%E6%95%B0%E6%8D%AE%2B%E5%85%A8%E5%88%97%E6%8F%92%E5%85%A5%C2%A0">
     1.1，单行数据+全列插入
    </h4>
    <ul>
     <li>
      <strong>
       插入数据
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      注意在这里插入的时候也可以不指明id，因为我们定义了auto_increment属性，mysql会使用默认的值自增。
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into students values (1,101,'张三',1111);
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt; insert into students values (2,102,'李四',11222);
      </strong>
     </p>
    </blockquote>

     <img alt="" height="262" src="https://i-blog.csdnimg.cn/direct/3dc3af7d964d4448a56b7fb9365bd00c.png" width="601"/>
    </p>
    <ul>
     <li>
      <strong>
       查看插入结果
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       select * from students;

      <strong>
       指定列插入数据

       mysql&gt; insert into students  (sn,name,qq) values (103,'李四',11333);

    <h4 id="1.5%EF%BC%8C%E6%8F%92%E5%85%A5%E5%90%A6%E5%88%99%E6%9B%B4%E6%96%B0%C2%A0" name="1.5%EF%BC%8C%E6%8F%92%E5%85%A5%E5%90%A6%E5%88%99%E6%9B%B4%E6%96%B0%C2%A0">
     1.5，插入否则更新
    </h4>
    <p>
     在插入数据的时候，由于
     <strong>
      主键和唯一键
     </strong>
     的存在，可能导致插入的数据已经存在而插入失败。
    </p>
    <p style="text-align:center">
     <img alt="" height="257" src="https://i-blog.csdnimg.cn/direct/387b9acdd54a449bb759781aae604752.png" width="525"/>
    </p>
    <p>
     <strong>
      如果需要更新，可以进行
      <span style="color:#fe2c24">
       更新
      </span>
      操作。也就是在插入数据时，如果不存在就直接插入，如果存在，发生冲突了，就将原来的数据更新为插入的数据。
      <span style="color:#fe2c24">
       这个和STL中的unordered_map类似，在使用[ ]的时候，如果存在就更新，如果不存在就插入。
      </span>
     </strong>
    </p>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       insert  ......
       <span style="color:#fe2c24">
        on duplicate key update
       </span>
       column=value[......]
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      示例：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into students values (1,101,'张三',55555) on duplicate key update id=1,sn=101,name='张三',qq=55555;
      </strong>
     </p>
    </blockquote>
    <p>
     <img alt="" height="215" src="https://i-blog.csdnimg.cn/direct/22956649f03449a1aeacf2e270a84a76.png" width="636"/>
    </p>
    <p>
     <strong>
      在更新的时候也要保证不和其他值冲突。
     </strong>
    </p>
    <p style="text-align:center">
     <strong>
      <img alt="" height="223" src="https://i-blog.csdnimg.cn/direct/68d3a38e58884b3486cb8922d427d673.png" width="668"/>
     </strong>
    </p>
    <h4 id="1.6%EF%BC%8C%E6%9B%BF%E6%8D%A2" name="1.6%EF%BC%8C%E6%9B%BF%E6%8D%A2">
     1.6，替换
    </h4>
    <p>
     替换replace与上面的插入更新类似，如果数据存在就替换，如果数据不存在就直接插入。
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt;
       <span style="color:#fe2c24">
        replace into
       </span>
       students (sn,name) values (120,'广州');//不存在直接插入
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt;
       <span style="color:#fe2c24">
        replace into
       </span>
       students (id,sn,name,qq)  values (1,101,'苏州',99999);//数据已经存在，则替换掉原数据
      </strong>
     </p>
    </blockquote>

    <h2 id="2%EF%BC%8CRetrieve%20%EF%BC%88%E8%AF%BB%E5%8F%96%EF%BC%89" name="2%EF%BC%8CRetrieve%20%EF%BC%88%E8%AF%BB%E5%8F%96%EF%BC%89">
     2，Retrieve （读取）
    </h2>
    <p>
     <strong>
      首先创建一张学生成绩表，作为示例
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; create table exam_result(
       <br/>
       -&gt; id int unsigned primary key auto_increment,
       <br/>
       -&gt; name varchar(20),
       <br/>
       -&gt; chinese int,
       <br/>
       -&gt; math int,
       <br/>
       -&gt; english int);
      </strong>
     </p>
    </blockquote>

      向表中插入一些数据：
     </strong>
    </p>

      <br/>
     </p>

      mysql&gt; insert into exam_result (name,chinese,math,english) values ('猪悟能',88,98,90);

    <h4 id="2.1%EF%BC%8C%E5%85%A8%E5%88%97%E6%9F%A5%E8%AF%A2" name="2.1%EF%BC%8C%E5%85%A8%E5%88%97%E6%9F%A5%E8%AF%A2">
     2.1，全列查询
    </h4>
    <blockquote>
     <p>
      <strong>
       mysql&gt;
       <span style="color:#fe2c24">
        select *
       </span>
       <span style="color:#fe2c24">
        from exam_result;
       </span>
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="239" src="https://i-blog.csdnimg.cn/direct/a8b1e8bd10134f3f87f60412aa3fb09a.png" width="592"/>
    </p>
    <p>
     <strong>
      通常情况 下不建议使用 * 进行全列查询
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       查询的列越多意味着需要传输的数据量越大；
      </strong>
     </li>
     <li>
      <strong>
       可能会影响到索引的作用。（索引在后面更新）
      </strong>
     </li>
    </ul>
    <p>
    </p>
    <h4 id="2.2%EF%BC%8C%E6%8C%87%E5%AE%9A%E5%88%97%E6%9F%A5%E8%AF%A2" name="2.2%EF%BC%8C%E6%8C%87%E5%AE%9A%E5%88%97%E6%9F%A5%E8%AF%A2">
     2.2，指定列查询
    </h4>
    <blockquote>
     <p>
      <strong>
       select 筛选的列名称 from 表名
      </strong>
     </p>
    </blockquote>
    <p>
     示例：
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select id,name,math from exam_result;
       <br/>
       mysql&gt; select id,name,chinese  from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="487" src="https://i-blog.csdnimg.cn/direct/0f224c753a07433a9b4f01ab770d6e48.png" width="558"/>

    <h4 id="2.3%EF%BC%8C%E6%9F%A5%E8%AF%A2%E5%AD%97%E6%AE%B5%E4%B8%BA%E8%A1%A8%E8%BE%BE%E5%BC%8F" name="2.3%EF%BC%8C%E6%9F%A5%E8%AF%A2%E5%AD%97%E6%AE%B5%E4%B8%BA%E8%A1%A8%E8%BE%BE%E5%BC%8F">
     2.3，查询字段为表达式

       表达式不包含字段

    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select id,name,math,10 from exam_result;

      <strong>
       mysql&gt; select id,name,math,10+10 from exam_result;

     </p>
    </blockquote>

     <img alt="" height="570" src="https://i-blog.csdnimg.cn/direct/c6f84fffef4a4ed4ae2312d9f249c142.png" width="496"/>

     <li>
      <strong>
       表达式中包含一个字段
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select id,name,math+10 from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="258" src="https://i-blog.csdnimg.cn/direct/1460866aff44410a971bd5339b08a0ca.png" width="573"/>

     <li>
      <strong>
       表达式中包含多个字段

    <h4 id="%C2%A02.4%EF%BC%8C%E4%B8%BA%E6%9F%A5%E8%AF%A2%E7%BB%93%E6%9E%9C%E6%8C%87%E5%AE%9A%E5%88%AB%E5%90%8D" name="%C2%A02.4%EF%BC%8C%E4%B8%BA%E6%9F%A5%E8%AF%A2%E7%BB%93%E6%9E%9C%E6%8C%87%E5%AE%9A%E5%88%AB%E5%90%8D">

       select  column  [as] 别名 from 表名;

     </p>

    <p>
     <strong>
      示例：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select id,name,math+chinese+english as total from exam_result;
      </strong>
      <br/>
      <strong>
       mysql&gt; select id,name,math+chinese+english  '总分'  from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">

    <p>
     <strong>
      去重，需要使用
      <span style="color:#fe2c24">
       distinct
      </span>
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select
       <span style="color:#fe2c24">
        distinct
       </span>
       math from exam_result;
      </strong>
     </p>
    </blockquote>

     2.6，where条件
    </h4>
    <p>
     前面讲的select 用法，是用来筛选出哪些列，而接下来的这部分内容where条件，是筛选出满足条件的行。
    </p>
    <p>
     <strong>
      比较运算符
     </strong>
    </p>
    <blockquote>
     <ul>
      <li>
       <strong>
        &gt;,&gt;=,&lt;,&lt;=：大于，大于等于，小于，小于等于
       </strong>
      </li>
      <li>
       <strong>
        =：等于，比较null值时不安全，比如null=null返回的结果是null
       </strong>
      </li>
      <li>
       <strong>
        &lt;=&gt;：等于，比较null值时是安全的，比如null=null返回的结果是1
       </strong>
      </li>
      <li>
       <strong>
        !=,&lt;&gt;：不等于
       </strong>
      </li>
      <li>
       <strong>
        between a0  and  a1：范围匹配，[a0,a1]，如果a0&lt;=values&lt;=a1，返回true(1)
       </strong>
      </li>
      <li>
       <strong>
        in(option...)：如果是option中的任意一个，返回true(1)
       </strong>
      </li>
      <li>
       <strong>
        is null：是空
       </strong>
      </li>
      <li>
       <strong>
        is not null：不是空
       </strong>
      </li>
      <li>
       <strong>
        like：模糊匹配。%表示任意多个字符（包括0个），_表示任意一个字符
       </strong>
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      逻辑运算符
     </strong>
    </p>
    <blockquote>
     <ul>
      <li>
       <strong>
        and：多个条件必须都为true，结果才为true
       </strong>
      </li>
      <li>
       <strong>
        or：只要有一个条件为true，结果就为true
       </strong>
      </li>
      <li>
       <strong>
        not：如果条件为true(1)，结果就为false(0)
       </strong>
      </li>
     </ul>
    </blockquote>

     2.7，结果排序
    </h4>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       ---ASC 升序（从小到大）
      </strong>
     </p>
     <p>
      <strong>
       ---DESC 降序（从大到小）
      </strong>
     </p>
     <p>
      <strong>
       ---默认为ASC
      </strong>
     </p>
     <p>
      <strong>
       select ...... from table_name [where...]
       <span style="color:#fe2c24">
        order by
       </span>
       column [ASC|DESC]；
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      注意：null视为比任何值都小
     </strong>
    </p>
    <p>
     <strong>
      案例：
     </strong>
    </p>

     <li>

      </strong>

      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select name,math,english,chinese from exam_result

       </span>
      </strong>
     </p>
    </blockquote>

     在这里为什么可以使用别名了呢？同样这和mysql的子句执行顺序有关。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/0b2bfb1f037b4a65a4e5ce424f8cf5e6.png"/>

    <blockquote>
     <p>
      <strong>
       总结：首先我们需要确定操作哪张表，如果第二步就执行排序，其实表中有一部分数据是不需要进行排序的， 这样做太浪费时间和空间了。所以先确定操作哪张表，再根据where子句条件筛选出来我们要的数据，这时的数据一定是被筛选出来的有价值的数据，这时我们只是选出来数据了，不给显示出来，最后经过排序，再将数据显示出来。
      </strong>
     </p>
     <p>
      <strong>
       因此执行ordere by时，别名已经定义了，所以可以使用。
      </strong>
     </p>
    </blockquote>

     </strong>

     <li>
      <strong>
       删除整张表 ，查看删除结果
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; delete from for_delete;

      <strong>

     </p>
    </blockquote>

    <ul>
     <li>
      <strong>
       删除表之前id自增到3，现在插入一条新数据。
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into for_delete (name) values ('D');

    <h4 id="%C2%A04.3%EF%BC%8C%E6%88%AA%E6%96%AD%E8%A1%A8" name="%C2%A04.3%EF%BC%8C%E6%88%AA%E6%96%AD%E8%A1%A8">
     <strong>
      4.3，截断表
     </strong>
    </h4>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <blockquote>
     <p>
      <strong>
       truncate [table] table_name
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      注意：这个 操作慎用
     </strong>
    </p>
    <p>
     <strong>
      1，这个操作只能对整张表使用，不能向delete一样针对部分数据操作
     </strong>
    </p>
    <p>
     <strong>
      2，不对数据操作，所以比delete快，但是truncate 在删除数据的时候，
      <span style="color:#fe2c24">
       并不经过正真的事务，所以无法回滚
      </span>
     </strong>
    </p>
    <p>
     <strong>
      3，该操作与delete相比，它会
      <span style="color:#fe2c24">
       重置auto_increment项
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       案例：
      </strong>
     </span>
    </p>
    <ul>
     <li>
      <strong>
       准备测试表（和for_delete表属性一样）
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; create table for_truncate like for_delete;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="173" src="https://i-blog.csdnimg.cn/direct/20d76e8d4c5746b4a2315e656ce90b3d.png" width="525"/>
    </p>
    <ul>
     <li>
      <strong>
       插入测试数据 ，查看测试数据
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into for_truncate values (1,'A'),(2,'B'),(3,'C');
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt; select * from for_truncate;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="243" src="https://i-blog.csdnimg.cn/direct/5890f65a9f724e5ea2d7e6081396f851.png" width="581"/>
    </p>
    <ul>
     <li>
      <strong>
       截断整表数据，注意影响行数是0，所以实际上没有对表真正操作
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; truncate table for_truncate;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <strong>
      <img alt="" height="142" src="https://i-blog.csdnimg.cn/direct/606c4ab22c2548fe8b9b4d7962dccc11.png" width="605"/>
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       查看删除结果
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select * from for_truncate;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="99" src="https://i-blog.csdnimg.cn/direct/b0b4edee1b9249ca9657a5a6e85373ff.png" width="576"/>
    </p>
    <ul>
     <li>
      <strong>
       再插入 一条数据，自增id重新开始增长
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into for_truncate (name) values ('D');
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="185" src="https://i-blog.csdnimg.cn/direct/3b195f3ca2084e59952d356e5ea544f8.png" width="628"/>
    </p>
    <ul>
     <li>
      <strong>
       查看表结构
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; show create table for_truncate \G
      </strong>
     </p>
    </blockquote>

    <h2 id="5%EF%BC%8C%20%E6%8F%92%E5%85%A5%E6%9F%A5%E8%AF%A2%E7%BB%93%E6%9E%9C" name="5%EF%BC%8C%20%E6%8F%92%E5%85%A5%E6%9F%A5%E8%AF%A2%E7%BB%93%E6%9E%9C">
     5， 插入查询结果
    </h2>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <blockquote>
     <p>
      <span style="color:#fe2c24">
       <strong>
        insert into table_name [column...]  select ...
       </strong>
      </span>
     </p>
    </blockquote>
    <p>
     <strong>
      案例：删除表中的重复记录，重复数据只能有一份
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       创建原数据表
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; create table duplicate_table(
       <br/>
       -&gt; id int,
       <br/>
       -&gt; name varchar(20));
      </strong>
     </p>
    </blockquote>
    <ul>
     <li>
      <strong>
       插入测试数据
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; insert into duplicate_table values  (100,'aaa'), (100,'aaa'), (200,'bbb'), (200,'bbb'), (200,'bbb'), (300,'ccc');
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      思路：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       创建一张空表no_duplicate_table，结构和duplicate_table一样
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; create table no_duplicate_table like duplicate_table;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/0c7a84cf11e6464a8c5ff280c63f1bb3.png"/>
    </p>
    <ul>
     <li>
      <strong>
       将 duplicate_table数据去重插入到no_duolicate_table表中
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt;
       <span style="color:#fe2c24">
        insert into
       </span>
       no_duplicate_table
       <span style="color:#fe2c24">
        select distinct
       </span>
       * from duplicate_table;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="200" src="https://i-blog.csdnimg.cn/direct/77220bae5372487b8640ca23d3af80a3.png" width="594"/>
    </p>
    <ul>
     <li>
      <strong>
       通过重命名，做到原子的去重操作
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; rename table duplicate_table to old_duplicate_table;
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt; rename table no_duplicate_table to duplicate_table;
      </strong>
     </p>
    </blockquote>
    <ul>
     <li>
      <strong>
       查看最终结果
      </strong>
     </li>
    </ul>
    <p style="text-align:center">
     <strong>
      <img alt="" height="246" src="https://i-blog.csdnimg.cn/direct/535e918d9d07485182283afae5863f23.png" width="563"/>
     </strong>
    </p>
    <p>
    </p>
    <h2 id="6%EF%BC%8C%E8%81%9A%E5%90%88%E5%87%BD%E6%95%B0%C2%A0" name="6%EF%BC%8C%E8%81%9A%E5%90%88%E5%87%BD%E6%95%B0%C2%A0">
     <strong>
      6，聚合函数
     </strong>
    </h2>
    <blockquote>
     <p>
      <strong>
       count()：返回查询到的数据的数量
      </strong>
     </p>
     <p>
      <strong>
       sum()：返回查询到的数据的总和，不是数字没有意义
      </strong>
     </p>
     <p>
      <strong>
       avg()：返回查询到的数据的平均值，不是数字没有意义
      </strong>
     </p>
     <p>
      <strong>
       max()：返回查询到的数据的最大值，不是数字没有意义
      </strong>
     </p>
     <p>
      <strong>
       min()：返回查询到的数据的最小值，不是数字没有意义
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      案例：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       统计班级共有多少学生
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       //使用*做统计
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt; select count(*) from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="150" src="https://i-blog.csdnimg.cn/direct/977541a4958c4cd9962fef85c0e8113b.png" width="620"/>
    </p>
    <blockquote>
     <p>
      <strong>
       //使用表达式统计
      </strong>
     </p>
     <p>
      <strong>
       mysql&gt; select count(1) from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="140" src="https://i-blog.csdnimg.cn/direct/6e63e763930c48b89afdd694235eaec5.png" width="624"/>
    </p>
    <ul>
     <li>
      <strong>
       统计数学成绩总分
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select sum(math) from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/0f116e143545429b91958696e435679f.png"/>
    </p>
    <ul>
     <li>
      <strong>
       统计平均总分
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select avg(chinese+math+english) 平均总分 from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="142" src="https://i-blog.csdnimg.cn/direct/0bc4cab9c0e34663995b68023094f45c.png" width="617"/>
    </p>
    <ul>
     <li>
      <strong>
       统计数学成绩中不重复的成绩个数
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select count(distinct math) 不重复的个数 from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/3b27609ca93347ae96f7cd7b8886f481.png"/>
    </p>
    <ul>
     <li>
      <strong>
       返回英语最高分
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select max(english) from exam_result;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="132" src="https://i-blog.csdnimg.cn/direct/459d46a0088d4ee8b2f65595f343255e.png" width="616"/>
    </p>
    <ul>
     <li>
      <strong>
       返回&gt;70分以上的最低分
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select min(math) from exam_result where math&gt;70;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="134" src="https://i-blog.csdnimg.cn/direct/0e3455d5402b4b27b8ea8c90e10208d9.png" width="638"/>
    </p>
    <h2 id="7%EF%BC%8Cgroup%20by%E5%AD%90%E5%8F%A5%E7%9A%84%E4%BD%BF%E7%94%A8%C2%A0" name="7%EF%BC%8Cgroup%20by%E5%AD%90%E5%8F%A5%E7%9A%84%E4%BD%BF%E7%94%A8%C2%A0">
     7，group by子句的使用
    </h2>
    <p>
     在select 中使用
     <strong>
      <span style="color:#fe2c24">
       group by
      </span>
     </strong>
     <span style="color:#0d0016">
      子句
     </span>
     可以对指定列进行
     <strong>
      分组查询
     </strong>
     。
    </p>
    <blockquote>
     <p>
      <strong>
       select column1 column2... from table group by column
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      案例：
     </strong>
    </p>
    <p>
     <strong>
      准备工作，创建一个雇员信息数据库:
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       emp员工表
      </strong>
     </li>
     <li>
      <strong>
       dept部门表
      </strong>
     </li>
     <li>
      <strong>
       salgrade工作等级表
      </strong>
     </li>
    </ul>
    <p style="text-align:center">
     <img alt="" height="252" src="https://i-blog.csdnimg.cn/direct/86e150ad99654b1caa9f1b730847e570.png" width="613"/>
    </p>
    <p style="text-align:center">
     <img alt="" height="438" src="https://i-blog.csdnimg.cn/direct/fa3be8524faa4c4896d898a29bc7f9d8.png" width="587"/>
    </p>
    <ul>
     <li>
      <strong>
       如何显示每个部门的平均工资和最高工资
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select deptno,avg(sal),max(sal) from emp
       <span style="color:#fe2c24">
        group by deptno;
       </span>
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="301" src="https://i-blog.csdnimg.cn/direct/b38255a7b147402fa40acc19f5145d62.png" width="626"/>
    </p>
    <blockquote>
     <p>
      <strong>
       总结：
      </strong>
     </p>
     <p>
      <span style="color:#fe2c24">
       <strong>
        分组，可以理解为把一张表按照条件在逻辑上分成了很多子表，然后分别对各自的子表进行聚合统计。
       </strong>
      </span>
     </p>
    </blockquote>
    <p>
     <strong>
      分组聚合结果：
     </strong>
    </p>
    <p style="text-align:center">
     <img alt="" height="171" src="https://i-blog.csdnimg.cn/direct/c337dd5447584b4bb840a898dccc821f.png" width="659"/>
    </p>
    <ul>
     <li>
      <strong>
       显示每个部门的每种岗位的 平均工资和最低工资
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select job,deptno,avg(sal),min(sal) from emp  group by job,deptno;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="266" src="https://i-blog.csdnimg.cn/direct/32dd9db569f14681b6bf8727522176fa.png" width="656"/>
    </p>
    <ul>
     <li>
      <strong>
       显示平均工资低于2000的部门和它的平均工资
      </strong>
     </li>
    </ul>
    <p>
     首先统计各个部门的平均工资
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select deptno,avg(sal) from emp group by deptno;
      </strong>
     </p>
    </blockquote>
    <p style="text-align:center">
     <img alt="" height="157" src="https://i-blog.csdnimg.cn/direct/593d5ad0ccab4937b87003ffb947d7f2.png" width="595"/>
    </p>
    <p>
     having和group by配合使用，对group by的结果进行过滤
    </p>
    <blockquote>
     <p>
      <strong>
       mysql&gt; select deptno,avg(sal) 平均工资 from emp group by deptno
       <span style="color:#fe2c24">
        having
       </span>
       平均工资&lt;2000;
      </strong>
     </p>
    </blockquote>
    <p>
     <strong>
      having经常和group by搭配使用，作用是对分组进行筛选，作用有些像where
     </strong>
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/265866f2e32742be90dc7c2d8b511b35.png"/>
    </p>
