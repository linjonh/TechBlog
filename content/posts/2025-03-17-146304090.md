    <p>
     <a href="https://leetcode.cn/problems/dui-cheng-de-er-cha-shu-lcof/description/" rel="nofollow">
      力扣LCR 145. 判断对称二叉树
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7b300ce3359d4c9b8ac487233f17f46c.png#pic_center"/>
    </p>
    <p>
     1.一棵树何时镜像对称？
     <br/>
     —左子树与右子树镜像对称，那么这个树是对称的。
    </p>
    <p>
     2.如何判断左右子树镜像对称？（下面 的 == 是镜像对称的意思）
     <br/>
     —左右子树的根节点相同
     <br/>
     —左子树的左子树 == 右子树的右子树
     <br/>
     —左子树的右子树==右子树的左子树
    </p>
    <p>
     3.用两个指针p q对称的递归遍历树，进行比较即可。
     <br/>
     初始化：p=root-&gt;l q=root-&gt;r
     <br/>
     递归出口：p q都为空，return 1
     <br/>
     p q有一个为空 return 0
     <br/>
     递归条件：p==q
     <br/>
     p-&gt;l ==q-&gt;r
     <br/>
     p-&gt;r ==q-&gt;l
     <br/>
     4.特殊情况：空树 也满足条件
    </p>

    <p>
     <a href="https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/" rel="nofollow">
      力扣236. 二叉树的最近公共祖先
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/53e3bb6ae1624033ab347765b9a33e58.png#pic_center">
      <br/>
      分析：根据p,q的分布情况判断答案
      <br/>
      根节点root。
      <br/>
      1.如果p，q分别出现在root的左右子树中，则root是答案
      <br/>
      2.若p ，q同时出现在root的某一个子树x中，则问题转化为在x树中找公共祖先（递归）
     </img>
    </p>
    <p>
     得到解题思路：递归，去找p,q的出现位置，并判断答案。
     <br/>
     递归函数f(root，p，q) ：在以root为根的树中找p,q。
     <br/>
     1.roo == NULL,空树，返回NULL
     <br/>
     2.roo == p或者root==q，找到了一个，直接返回root。若另一个在root的子树中，root是答案。若不在，则返回找到的这个结点。
     <br/>
     3.root不为空，也不是p,q,，说明p,q在root的左右子树中，则递归调用，分别去左右子树中找。
     <br/>
     l=f(root-&gt;left，p，q) r=f(root-&gt;right，p，q)
     <br/>
     (1)若l,r全为空，返回空
     <br/>
     (2)若l,r有一个为空，返回另一个。说明在另一个子树中找到了p,q或者答案
     <br/>
     (3)若l,r都不为空，说明p,q一边一个，则root是答案，返回root
    </p>
