    <h5>
     <a id="_3">
     </a>
     前言
    </h5>
    <ol>
     <li>
      每次打开书签前，都需要创建新标签页。
     </li>
     <li>
      每次搜索前，都需要创建新标签页。
     </li>
    </ol>
    <p>
     这个真滴很麻烦。下面介绍的方法非常简单，不需要任何插件，通过修改浏览器参数即可。
    </p>
    <h5>
     <a id="_9">
     </a>
     名词
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14bd759a9cf504ed6acbca702efb411d.jpeg"/>
    </p>
    <h5>
     <a id="_11">
     </a>
     设置书签在新标签页打开
    </h5>
    <ol>
     <li>
      打开新标签页
     </li>
     <li>
      在
      <b>
       地址栏
      </b>
      输入
     </li>
    </ol>
    <pre><code class="prism language-html">about:config
</code></pre>
    <ol start="3">
     <li>
      回车
     </li>
     <li>
      点击
      <b>
       我了解此风险
      </b>
     </li>
     <li>
      输入（不用回车）
     </li>
    </ol>
    <pre><code class="prism language-html">browser.tabs.loadBookmarksIntabs
</code></pre>
    <img src="https://i-blog.csdnimg.cn/blog_migrate/0c8c55685423ba61ef98c6e34d3dfe76.jpeg">
     <ol start="6">
      <li>
       双击搜索到的记录，将值改为
       <b>
        true
       </b>
      </li>
     </ol>
     <img src="https://i-blog.csdnimg.cn/blog_migrate/91eb5f17f1023936a986fcd1ed6e6673.jpeg">
      <ol start="7">
       <li>
        设置完成
       </li>
      </ol>
      <h5>
       <a id="_35">
       </a>
       设置搜索栏在新标签页打开
      </h5>
      <ol>
       <li>
        打开新标签页
       </li>
       <li>
        在
        <b>
         地址栏
        </b>
        输入
       </li>
      </ol>
      <pre><code class="prism language-html">about:config
</code></pre>
      <ol start="3">
       <li>
        回车
       </li>
       <li>
        点击
        <b>
         我了解此风险
        </b>
       </li>
       <li>
        输入（不用回车）
       </li>
      </ol>
      <pre><code class="prism language-html">browser.search.openintab
</code></pre>
      <img src="https://i-blog.csdnimg.cn/blog_migrate/f404339296f2908b6765f353018b5c0c.jpeg">
       <ol start="6">
        <li>
         双击搜索到的记录，将值改为
         <b>
          true
         </b>
        </li>
       </ol>
       <img src="https://i-blog.csdnimg.cn/blog_migrate/6beb557e3d9e1b7c546deaa1e9ca3ffc.jpeg">
        <ol start="7">
         <li>
          设置完成
         </li>
         <li>
          以上设置在搜索栏搜索的时候，
          <b>
           回车搜索
          </b>
          （新页面打开），
          <b>
           箭头搜索
          </b>
          （覆盖当前页面）
          <br/>
          <img src="https://i-blog.csdnimg.cn/blog_migrate/73cbd914095a3c208159093037862249.jpeg"/>
         </li>
        </ol>
