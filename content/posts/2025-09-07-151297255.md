# 当没办法实现从win复制东西到Linux虚拟机时的解决办法

① 先确认是否已安装

bash

复制

```

sudo apt list --installed | grep open-vm-tools
```

* 如果 **没有任何回显** → 没装，跳到 ③
* 如果看到 `open-vm-tools` 已安装 → 继续 ②

---

② 启动正确的服务（单词别打错）

bash

复制

```

systemctl status vmtoolsd         # 查看状态
sudo systemctl restart vmtoolsd   # 重启服务
```

正常会显示 `active (running)`，再试复制-粘贴即可恢复。

---

③ 若提示未安装 → 一次性装上

bash

复制

```

sudo apt update
sudo apt install open-vm-tools open-vm-tools-desktop -y
sudo reboot
```

重启后主机 ↔ 虚拟机之间的剪贴板、拖放就能用了。

---

一句话总结  
**单词打对 `vmtoolsd`，没装就装 `open-vm-tools-desktop`，重启即可解决。**
