    <h2>
     <a id="Reactjs__0">
     </a>
     React.js 基础与进阶教程
    </h2>
    <p>
     React.js 是由 Facebook 开发的流行前端 JavaScript 库，专为构建用户界面（UI）设计，尤其适用于单页面应用（SPA）。它采用组件化开发模式，使 UI 结构更加清晰、可维护性更强。本文将带你快速入门 React，并深入了解其核心概念、关键特性及最佳实践。
    </p>
    <h3>
     <a id="1_React__4">
     </a>
     1. React 基础概念
    </h3>
    <h4>
     <a id="11_React__6">
     </a>
     1.1 React 组件
    </h4>
    <p>
     React 采用组件化开发，一个组件代表 UI 的一个独立部分。组件主要分为
     <strong>
      函数组件
     </strong>
     和
     <strong>
      类组件
     </strong>
     。
    </p>
    <p>
     <strong>
      函数组件示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-jsx">function Welcome(props) {
  return &lt;h1&gt;Hello, {props.name}!&lt;/h1&gt;;
}
</code></pre>
    <p>
     <strong>
      类组件示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-jsx">class Welcome extends React.Component {
  render() {
    return &lt;h1&gt;Hello, {this.props.name}!&lt;/h1&gt;;
  }
}
</code></pre>
    <h4>
     <a id="12_JSX__25">
     </a>
     1.2 JSX 语法
    </h4>
    <p>
     JSX 是 React 提供的一种 JavaScript 语法扩展，允许在 JavaScript 代码中直接编写 HTML 结构。
    </p>
    <pre><code class="prism language-jsx">const element = &lt;h1&gt;Hello, World!&lt;/h1&gt;;
</code></pre>
    <p>
     JSX 语法需要 Babel 进行编译，最终转换成
     <code>
      React.createElement
     </code>
     形式。
    </p>
    <h4>
     <a id="13_State__Props_32">
     </a>
     1.3 State 与 Props
    </h4>
    <ul>
     <li>
      <strong>
       Props（属性）
      </strong>
      ：用于组件间传递数据。
     </li>
     <li>
      <strong>
       State（状态）
      </strong>
      ：组件内部的可变数据，影响组件的渲染。
     </li>
    </ul>
    <p>
     示例：
    </p>
    <pre><code class="prism language-jsx">class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }
  
  increment = () =&gt; {
    this.setState({ count: this.state.count + 1 });
  };
  
  render() {
    return (
      &lt;div&gt;
        &lt;p&gt;Count: {this.state.count}&lt;/p&gt;
        &lt;button onClick={this.increment}&gt;Increment&lt;/button&gt;
      &lt;/div&gt;
    );
  }
}
</code></pre>
    <h3>
     <a id="2_React__59">
     </a>
     2. React 进阶
    </h3>
    <h4>
     <a id="21__61">
     </a>
     2.1 组件生命周期
    </h4>
    <p>
     React 组件有三个主要生命周期阶段：
    </p>
    <ul>
     <li>
      <strong>
       挂载（Mounting）
      </strong>
      ：组件创建并插入 DOM。
     </li>
     <li>
      <strong>
       更新（Updating）
      </strong>
      ：组件状态或属性更新时触发。
     </li>
     <li>
      <strong>
       卸载（Unmounting）
      </strong>
      ：组件从 DOM 中移除。
     </li>
    </ul>
    <p>
     常见生命周期方法：
    </p>
    <pre><code class="prism language-jsx">class Example extends React.Component {
  componentDidMount() {
    console.log('组件已挂载');
  }
  componentDidUpdate() {
    console.log('组件更新');
  }
  componentWillUnmount() {
    console.log('组件即将卸载');
  }
}
</code></pre>
    <h4>
     <a id="22__82">
     </a>
     2.2 事件处理
    </h4>
    <p>
     在 React 中，事件处理类似于 HTML，但使用驼峰命名：
    </p>
    <pre><code class="prism language-jsx">&lt;button onClick={this.handleClick}&gt;Click Me&lt;/button&gt;
</code></pre>
    <h4>
     <a id="23__88">
     </a>
     2.3 条件渲染
    </h4>
    <pre><code class="prism language-jsx">{isLoggedIn ? &lt;UserDashboard /&gt; : &lt;LoginPage /&gt;}
</code></pre>
    <h4>
     <a id="24__93">
     </a>
     2.4 列表渲染
    </h4>
    <pre><code class="prism language-jsx">const items = ['Apple', 'Banana', 'Cherry'];
const listItems = items.map(item =&gt; &lt;li key={item}&gt;{item}&lt;/li&gt;);
</code></pre>
    <h3>
     <a id="3_React_HooksReact_168_99">
     </a>
     3. React Hooks（React 16.8+）
    </h3>
    <p>
     React Hooks 允许在函数组件中使用 state 和生命周期。
    </p>
    <h4>
     <a id="31_useState_102">
     </a>
     3.1 useState
    </h4>
    <pre><code class="prism language-jsx">import { useState } from 'react';
function Counter() {
  const [count, setCount] = useState(0);
  return (
    &lt;div&gt;
      &lt;p&gt;Count: {count}&lt;/p&gt;
      &lt;button onClick={() =&gt; setCount(count + 1)}&gt;Increase&lt;/button&gt;
    &lt;/div&gt;
  );
}
</code></pre>
    <h4>
     <a id="32_useEffect_116">
     </a>
     3.2 useEffect
    </h4>
    <pre><code class="prism language-jsx">import { useEffect } from 'react';
function Example() {
  useEffect(() =&gt; {
    console.log('组件渲染');
  }, []);
}
</code></pre>
    <h4>
     <a id="33_useContext_126">
     </a>
     3.3 useContext
    </h4>
    <pre><code class="prism language-jsx">const ThemeContext = React.createContext('light');
function ThemedButton() {
  const theme = useContext(ThemeContext);
  return &lt;button className={theme}&gt;Click Me&lt;/button&gt;;
}
</code></pre>
    <h3>
     <a id="4_React_Router_135">
     </a>
     4. React Router（前端路由）
    </h3>
    <p>
     React Router 允许实现单页面应用路由。
    </p>
    <pre><code class="prism language-jsx">import { BrowserRouter as Router, Route, Routes } from 'react-router-dom';
function App() {
  return (
    &lt;Router&gt;
      &lt;Routes&gt;
        &lt;Route path="/" element={&lt;Home /&gt;} /&gt;
        &lt;Route path="/about" element={&lt;About /&gt;} /&gt;
      &lt;/Routes&gt;
    &lt;/Router&gt;
  );
}
</code></pre>
    <h3>
     <a id="5_Redux_151">
     </a>
     5. Redux（状态管理）
    </h3>
    <p>
     Redux 解决组件间状态共享问题。
    </p>
    <pre><code class="prism language-jsx">import { createStore } from 'redux';
const reducer = (state = { count: 0 }, action) =&gt; {
  switch (action.type) {
    case 'INCREMENT': return { count: state.count + 1 };
    default: return state;
  }
};
const store = createStore(reducer);
</code></pre>
    <h3>
     <a id="6_React__164">
     </a>
     6. React 性能优化
    </h3>
    <h4>
     <a id="61__Reactmemo__165">
     </a>
     6.1 使用 React.memo 进行组件优化
    </h4>
    <pre><code class="prism language-jsx">const MemoizedComponent = React.memo(MyComponent);
</code></pre>
    <h4>
     <a id="62__useCallback__170">
     </a>
     6.2 使用 useCallback 记忆化函数
    </h4>
    <pre><code class="prism language-jsx">const memoizedCallback = useCallback(() =&gt; doSomething(a, b), [a, b]);
</code></pre>
    <h4>
     <a id="63__useMemo__175">
     </a>
     6.3 使用 useMemo 记忆化计算结果
    </h4>
    <pre><code class="prism language-jsx">const memoizedValue = useMemo(() =&gt; computeExpensiveValue(a, b), [a, b]);
</code></pre>
