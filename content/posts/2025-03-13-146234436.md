    <h2>
     <strong>
      1. 添加依赖（Maven项目）
     </strong>
    </h2>
    <p>
     <strong>
      在
      <code>
       pom.xml
      </code>
      中添加 MyBatis 和数据库驱动的依赖（以 MySQL 为例）：
     </strong>
    </p>
    <pre><code class="language-java">&lt;dependencies&gt;
    &lt;!-- MyBatis 核心库 --&gt;
    &lt;dependency&gt;
        &lt;groupId&gt;org.mybatis&lt;/groupId&gt;
        &lt;artifactId&gt;mybatis&lt;/artifactId&gt;
        &lt;version&gt;3.5.13&lt;/version&gt; &lt;!-- 使用最新版本 --&gt;
    &lt;/dependency&gt;
    &lt;!-- MySQL 驱动 --&gt;
    &lt;dependency&gt;
        &lt;groupId&gt;mysql&lt;/groupId&gt;
        &lt;artifactId&gt;mysql-connector-java&lt;/artifactId&gt;
        &lt;version&gt;8.0.33&lt;/version&gt;
    &lt;/dependency&gt;
&lt;/dependencies&gt;</code></pre>
    <h2>
     <strong>
      2. 配置 MyBatis 核心文件
     </strong>
    </h2>
    <p>
     <strong>
      在
      <code>
       resources
      </code>
      目录下创建
      <code>
       mybatis-config.xml
      </code>
      ，配置数据库连接和全局设置：
     </strong>
    </p>
    <pre><code class="language-java">&lt;?xml version="1.0" encoding="UTF-8" ?&gt;
&lt;!DOCTYPE configuration PUBLIC "-//mybatis.org//DTD Config 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-config.dtd"&gt;
&lt;configuration&gt;
    &lt;environments default="development"&gt;
        &lt;environment id="development"&gt;
            &lt;transactionManager type="JDBC"/&gt;
            &lt;dataSource type="POOLED"&gt;
                &lt;!-- 修改为你的数据库信息 --&gt;
                &lt;property name="driver" value="com.mysql.cj.jdbc.Driver"/&gt;
                &lt;property name="url" value="jdbc:mysql://localhost:3306/your_database?useSSL=false&amp;amp;serverTimezone=UTC"/&gt;
                &lt;property name="username" value="root"/&gt;
                &lt;property name="password" value="123456"/&gt;
            &lt;/dataSource&gt;
        &lt;/environment&gt;
    &lt;/environments&gt;
    &lt;!-- 指定 Mapper 文件的位置 --&gt;
    &lt;mappers&gt;
        &lt;mapper resource="mapper/UserMapper.xml"/&gt;
    &lt;/mappers&gt;
&lt;/configuration&gt;</code></pre>
    <h2>
     <strong>
      3. 创建实体类
     </strong>
    </h2>
    <p>
     <strong>
      定义与数据库表对应的实体类，例如
      <code>
       User
      </code>
      ：
     </strong>
    </p>
    <pre><code class="language-java">public class User {
    private Integer id;
    private String name;
    private String email;
    // 省略构造方法、getter/setter 和 toString()
}</code></pre>
    <h2>
     <strong>
      4. 创建 Mapper 接口
     </strong>
    </h2>
    <pre><code class="language-java">public interface UserMapper {
    User selectUserById(int id);
}</code></pre>
    <h2>
     <strong>
      5. 编写 Mapper XML 映射文件
     </strong>
    </h2>
    <p>
     <strong>
      在
      <code>
       resources/mapper
      </code>
      目录下创建
      <code>
       UserMapper.xml
      </code>
      ，编写 SQL 映射：
     </strong>
    </p>
    <pre><code class="language-java">&lt;?xml version="1.0" encoding="UTF-8" ?&gt;
&lt;!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd"&gt;
&lt;mapper namespace="com.example.mapper.UserMapper"&gt;
    &lt;select id="selectUserById" resultType="com.example.entity.User"&gt;
        SELECT * FROM user WHERE id = #{id}
    &lt;/select&gt;
&lt;/mapper&gt;</code></pre>
    <h2>
     <strong>
      6. 使用 MyBatis 执行操作
     </strong>
    </h2>
    <p>
     <strong>
      通过
      <code>
       SqlSessionFactory
      </code>
      获取
      <code>
       SqlSession
      </code>
      并操作数据库：
     </strong>
    </p>
    <pre><code class="language-java">import org.apache.ibatis.io.Resources;
import org.apache.ibatis.session.SqlSession;
import org.apache.ibatis.session.SqlSessionFactory;
import org.apache.ibatis.session.SqlSessionFactoryBuilder;

public class MyBatisDemo {
    public static void main(String[] args) {
        try {
            // 加载 MyBatis 配置文件
            String resource = "mybatis-config.xml";
            InputStream inputStream = Resources.getResourceAsStream(resource);
            
            // 创建 SqlSessionFactory
            SqlSessionFactory sqlSessionFactory = new SqlSessionFactoryBuilder().build(inputStream);
            
            // 获取 SqlSession
            try (SqlSession sqlSession = sqlSessionFactory.openSession()) {
                // 获取 Mapper 接口实例
                UserMapper userMapper = sqlSession.getMapper(UserMapper.class);
                
                // 执行查询
                User user = userMapper.selectUserById(1);
                System.out.println(user);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}</code></pre>
    <h2>
     <strong>
      关键注意事项
     </strong>
    </h2>
    <h3>
     1.
     <strong>
      驱动和 URL 格式
     </strong>
     ：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        MySQL 8+ 驱动类名为
        <code>
         com.mysql.cj.jdbc.Driver
        </code>
        ，URL 需指定时区（如
        <code>
         serverTimezone=UTC.
        </code>
       </strong>
      </p>
     </li>
    </ul>
    <h3>
     2.
     <strong>
      XML 文件位置
     </strong>
     ：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        确保
        <code>
         mybatis-config.xml
        </code>
        和
        <code>
         UserMapper.xml
        </code>
        放在项目的类路径下（如
        <code>
         src/main/resources
        </code>
        ）。
       </strong>
      </p>
     </li>
    </ul>
    <h3>
     3.
     <strong>
      事务管理
     </strong>
     ：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        默认使用
        <code>
         JDBC
        </code>
        事务管理器，执行写操作后需调用
        <code>
         sqlSession.commit()
        </code>
        提交事务。
       </strong>
      </p>
     </li>
    </ul>
    <h3>
     4.
     <strong>
      日志配置
     </strong>
     ：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        如需查看 SQL 日志，可在
        <code>
         mybatis-config.xml
        </code>
        中添加日志实现（如 SLF4J）。
       </strong>
      </p>
     </li>
    </ul>
    <h2>
     <strong>
      常见问题排查
     </strong>
    </h2>
    <ul>
     <li>
      <p>
       <strong>
        配置文件未找到：检查文件是否在
        <code>
         resources
        </code>
        目录下，且路径正确。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        驱动类异常：确认驱动依赖已正确添加，版本兼容。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        Mapper 未绑定：检查
        <code>
         mybatis-config.xml
        </code>
        中
        <code>
         &lt;mappers&gt;
        </code>
        的配置是否指向正确的 XML 文件
       </strong>
      </p>
     </li>
     <li>
      <p>
       通过以上步骤，即可完成 MyBatis 与数据库的基本连接和操作。
      </p>
     </li>
    </ul>
