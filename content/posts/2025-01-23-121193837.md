     无法获得下列许可 solidworks standard 无法连接到服务器
    </h3>
    <p>
     当启动SolidWorks时，出现无法获得下列许可 solidworks standard 无法连接到服务器提示。这里给出解决办法。
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d73f014188b5fc19c4340851b27d4049.png">
      <br/>
      第一步：打开任务管理器
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/85a95369a86c454315bfc07afe135398.png">
       <br/>
       第二步：找到最后一个“服务”选项
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f66a0c4d0a5777f8f3ab066f13a14811.png">
        <br/>
        第三步：找到SolidWorks Flexnet Server 和 SolidWorks Licensing Server 选项，这里你可以发现这两个服务是停止状态，所以把这两个服务启动就可以了。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/954c156bfa655f8325fc1c985adb3cbf.png">
         <br/>
         第四步：点击启动启动此服务，
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/14d466acee1e9a3b97120d20fe66c601.png">
          <br/>
          此时，在重新打开SolidWorks程序完美运行。
