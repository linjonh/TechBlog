     📌 4. 回退与撤销
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        操作
       </th>
       <th>
        命令
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        撤销

         git add
        </code>
        操作

       <td>
        <code>
         git reset HEAD &lt;file&gt;
        </code>
       </td>
      </tr>
      <tr>
       <td>
        撤销最后一次提交
       </td>
       <td>
        <code>
         git reset --soft HEAD~1
        </code>

      </tr>
      <tr>
       <td>
        强制回退到某个提交
       </td>
       <td>
        <code>
         git reset --hard &lt;commit-hash&gt;
        </code>
       </td>
      </tr>
      <tr>
       <td>
        撤销对文件的修改
       </td>
       <td>
        <code>
         git checkout -- &lt;file&gt;
        </code>
       </td>
      </tr>
     </tbody>
    </table>

       <th>

       </th>
