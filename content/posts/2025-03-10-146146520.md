    <h2>
     <a id="_74">
     </a>
     总结
    </h2>
    <h4>
     <a id="1__77">
     </a>
     1.
     <strong>
      内联样式（行内样式）
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       优点
      </strong>
      ：快速、直接针对单个元素生效。
     </li>
     <li>
      <strong>
       缺点
      </strong>
      ：难以维护，复用性差，容易导致代码冗余。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2__84">
     </a>
     2.
     <strong>
      内部样式表
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       优点
      </strong>
      ：可集中管理当前页面的样式，复用性较好。
     </li>
     <li>
      <strong>
       缺点
      </strong>
      ：仅对当前页面生效，多页面需重复编写。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3__91">
     </a>
     3.
     <strong>
      外部样式表
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       优点
      </strong>
      ：最佳实践，样式可跨页面复用，便于维护和扩展。
     </li>
     <li>
      <strong>
       缺点
      </strong>
      ：需额外管理CSS文件。
     </li>
    </ul>
    <hr/>
    <h5>
     <a id="_98">
     </a>
     <strong>
      优先级说明
     </strong>
    </h5>
    <p>
     当多种方式同时存在时，优先级为：
     <strong>
      内联样式 &gt; 内部样式表 &gt; 外部样式表
     </strong>
     （若选择器权重相同）。
     <br/>
     建议优先使用外部样式表，保持结构与样式分离，提高代码可维护性。
    </p>
