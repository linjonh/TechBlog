    <span style="color:#cc00; font-size:16px">
     <strong>
      <u>
       2013年7月微软MVP申请开始了！
      </u>
     </strong>
    </span>
    <a href="http://blog.csdn.net/blogdevteam/article/details/8754544" rel="noopener noreferrer" target="_blank">
     （2013年4月MVP当选名单公布）
    </a>
    <br/>
    <br/>
    CSDN与微软合作，长期为用户提供申请“微软最有价值专家”的平台，希望有兴趣、资历的朋友以及正在朝这个方向努力的朋友能够积极参与。
    <br/>
    <br/>
    <p>
     申请微软MVP的朋友请下载
     <span style="color:#ff0000">
      此新版申请表格
     </span>
     ：
     <a href="http://bbs.csdn.net/download/2013-7MVPApplicationForm%28China%29.zip" rel="noopener noreferrer" target="_blank">
      http://bbs.csdn.net/download/2013-7MVPApplicationForm(China).zip
     </a>
    </p>
    <br/>
    <span style="color:#ff0000">
     填写好相关内容后把申请表格发送到邮箱
    </span>
    <strong>
     admin(at)csdn.net
    </strong>
    <div class="msgfont">
     <br/>
    </div>
    <div class="msgfont">
     我们会在每次MVP申请开始时把用户提交的申请表格推荐到微软（中国）相关部门！
    </div>
    <div class="msgfont">
     <br/>
     <br/>
     2013年7月微软MVP将由CSDN社区进行统一推荐。
     <br/>
     <br/>
     进行Renew的MVP可以将您的申请资料发送到CSDN社区，由社区统一转交，或将您的资料直接发给项目组，并CC给社区一份。如不在CSDN社区统一范围内申请的MVP，将 不再默认为CSDN推荐申请。
     <br/>
     <br/>
     为了防止邮件丢失，请所有发送完邮件的人员，访问：
     <a href="http://surveies.csdn.net/survey/comein/559" rel="nofollow noopener noreferrer" target="_blank">
      http://surveies.csdn.net/survey/comein/559
     </a>
     进行登记。
     <br/>
     <br/>
     <strong>
      申请截止时间：2013年4月10日16点。
      <br/>
      <br/>
      <br/>
      填写注意事项
     </strong>
     ：
     <br/>
     <br/>
     1、 CSDN推荐者必须填写项为：参与第三方技术社区。请务必标明您的个人社区ID。
     <br/>
     <br/>
     2、请详细填写您的自我介绍，最好使用英文，如果没有，微软将提供人员给您翻译，但难以保证翻译质量。
     <br/>
     <br/>
     3、请尽量多的填写您在过去一年中对微软技术推广做出的贡献，但是此行动必须是无偿的。
     <br/>
     <br/>
     4、如您提交的资料不够详细或者存在问题，CSDN或微软工作人员会第一时间联络您进行资料补充。
     <br/>
     <br/>
     <br/>
     注意：所有申请成功者将根据主要技术分享形式，获得CSDN论坛或者CSDN博客颁发的【微软最有价值专家】勋章，如勋章有误，请发邮件到
     <strong>
      admin@csdn.net
     </strong>
     进行说明。

     <br/>

    <strong>
     其他问题补充:
    </strong>
    <br/>
    <br/>
    1 我已经申请过4月的MVP，现在当任务名单未出来，我是否需要再申请？
    <br/>
    <br/>
    答：您可以等待4月MVP名单出来后，再确认是否重新申请。
    <br/>
    <br/>
    2 我是Java,Oracle的领域的专家，是否可以申请？
    <br/>
    <br/>
    答：本申请针对微软技术领域专家，但如果您对微软技术领域有贡献，亦可进行申请。
    <br/>
    <br/>
    3 申请有对职业和年龄的限制吗？
    <br/>
    <br/>
    答：年龄必须满18岁。职业无限制。但不得从事非法职业。
    <br/>
    <br/>
    4 有哪些贡献算MVP贡献
    <br/>
    <br/>
    答：a 线上： 传播微软相关技术，如：社区发帖、回复。 写作BLOG。
    <br/>
    <br/>
    b 线下： 写作、 组织微软技术相关活动 、微软课程培训。
    <br/>
    <br/>
    但所有一切不能为付费行为，均为免费行为。
    <br/>
    <br/>
    5 能给一个申请表填写的示例吗？
    <br/>
    <br/>
    答：请参考：
    <a href="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx" rel="noopener noreferrer" target="_blank" title="http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx">
     http://blog.csdn.net/jinjazz/archive/2008/07/09/2630190.aspx
    </a>
    （此为旧版申请表事例，供参考。）
    <br/>
    <br/>
    <strong>
     再次提醒：本次申请截止时间为：2013年4月10日16点。
