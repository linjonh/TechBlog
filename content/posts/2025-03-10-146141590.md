     SELECT语句-查询
    </h3>
    <h4 id="1.%E6%A0%BC%E5%BC%8F" name="1.%E6%A0%BC%E5%BC%8F">
     1.格式
    </h4>
    <pre><code>📖简单查询:
格式: select 字段1,字段n from 表名；
起别名: 通过在字段后添加 as 别名 as可以省略 改变表头
eg: select username "用户名",password as "密码" from user
</code></pre>
    <h4 id="2.%E6%93%8D%E4%BD%9C" name="2.%E6%93%8D%E4%BD%9C">
     2.操作
    </h4>
    <pre><code>1.创建表
mysql&gt; create table user(

-&gt; username varchar(255),

     3.算数表达式
    </h4>
    <pre><code>1. 两个整数运算，做算数
mysql&gt; select 100+200; 
+---------+
| 100+200 |
+---------+
| 300 |
+---------+
1 row in set (0.00 sec)

2.字符串和整数的运算，先转化字符串，如果成功则算数，如果不成功则字符串变0
mysql&gt; select '100'+200; 
+-----------+
| '100'+200 |
+-----------+
| 300 |
+-----------+
1 row in set (0.00 sec)

3.字符串的转换，可以包含字母，直接省略---一般数字开头的字符串
mysql&gt; select '100q'+200;
+------------+
| '100q'+200 |
+------------+
| 300 |
+------------+
1 row in set, 1 warning (0.00 sec)

4.转化不成功则使用0算数
mysql&gt; select 'q100q'+200;
+-------------+
| 'q100q'+200 |
+-------------+
| 200 |
+-------------+
1 row in set, 1 warning (0.00 sec)

mysql&gt; select 'q100q'+200 + 'q100q';
+-----------------------+
| 'q100q'+200 + 'q100q' |
+-----------------------+
| 200 |
+-----------------------+
1 row in set, 2 warnings (0.00 sec)

5.只要是运算，都需要转化字符串
mysql&gt; select 'q100q'+ 'q100q'; 
+------------------+
| 'q100q'+ 'q100q' |
+------------------+
| 0 |
+------------------+
1 row in set, 2 warnings (0.00 sec)

6.NULL值参与的运算结果直接为null
mysql&gt; select NULL + 200;
+------------+
| NULL + 200 |
+------------+
| NULL |
+------------+
1 row in set (0.00 sec)

mysql&gt; select 200+ 300 + NULL + 200;
+-----------------------+
| 200+ 300 + NULL + 200 |
+-----------------------+
| NULL |
+-----------------------+
1 row in set (0.00 sec)

mysql&gt; select * from user;
+---------+----------+----------+---------+------------+
| user_id | username | password | name | hire_date |
+---------+----------+----------+---------+------------+
| 1 | zhang | 111 | 张三 | 1996-05-04 |
| 2 | zhang | 111 | 张三 | 1996-05-04 |
| 3 | zhang1 | 111 | 张三1 | 1997-05-04 |
| 4 | zhang2 | 111 | 张三2 | 1998-05-04 |
| 5 | zhang3 | 111 | 张三3 | 1999-05-04 |
| 6 | zhang4 | 111 | 张三4 | 1990-05-04 |
| 7 | lijian | 222 | 李剑 | 1990-08-26 |
| 8 | lijian1 | 2221 | 李剑1 | 1990-10-30 |
+---------+----------+----------+---------+------------+
8 rows in set (0.00 sec)

7. 查询字段可以运算
mysql&gt; select username,password * 6,name from user;

| username | password * 6 | name |

| zhang | 666 | 张三 |
