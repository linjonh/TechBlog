#### 无条件多组输入

[奇偶统计改](https://dashoj.com/p/9)
  
删去了以 0 作为循环结束的条件
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/5f2c5292daa84b508e3fe7102a5dcdb6.png)

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n,num_ou=0,sum_ji=0;
	while(cin<<n){
	//少了结束条件
		if(n%2==0) num_ou++;
		else sum_ji+=n;
	}
	cout<<num_ou<<endl<<sum_ji;
}
```
