    <h2>
     <a id="Vim__1">
     </a>
     Vim 编辑器复制文件所有内容
    </h2>
    <p>
     在 Vim 的可视化模式下复制所有内容，可以通过以下步骤完成：
    </p>
    <hr/>
    <h3>
     <a id="_1_7">
     </a>
     <strong>
      方法 1：可视化模式全选复制
     </strong>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        进入可视化模式
       </strong>
       <br/>
       按下
       <code>
        V
       </code>
       （大写 V）进入
       <strong>
        行可视化模式
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        全选内容
       </strong>
       <br/>
       依次按下
       <code>
        gg
       </code>
       （跳转到文件开头） →
       <code>
        G
       </code>
       （跳转到文件末尾），此时会选中整个文件内容。
      </p>
     </li>
     <li>
      <p>
       <strong>
        复制内容
       </strong>
       <br/>
       按下
       <code>
        y
       </code>
       （yank，即复制），选中的内容会被复制到 Vim 的默认寄存器中。
      </p>
     </li>
     <li>
      <p>
       <strong>
        退出可视化模式
       </strong>
       <br/>
       按下
       <code>
        &lt;Esc&gt;
       </code>
       退出。
      </p>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_2_23">
     </a>
     <strong>
      方法 2：直接命令复制（无需可视化模式）
     </strong>
    </h3>
    <p>
     如果不需要可视化操作，更快捷的方式是：
    </p>
    <ol>
     <li>
      在普通模式下输入
      <code>
       ggyG
      </code>
      ：
      <ul>
       <li>
        <code>
         gg
        </code>
        ：跳转到文件开头
       </li>
       <li>
        <code>
         yG
        </code>
        ：从当前位置复制到文件末尾
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_3_33">
     </a>
     <strong>
      方法 3：使用系统剪贴板（跨应用粘贴）
     </strong>
    </h3>
    <p>
     如果希望复制的内容可以直接粘贴到其他程序（如浏览器、编辑器）：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        确保 Vim 支持系统剪贴板
       </strong>
       <br/>
       终端输入
       <code>
        vim --version | grep clipboard
       </code>
       ，如果显示
       <code>
        +clipboard
       </code>
       表示支持。
      </p>
     </li>
     <li>
      <p>
       <strong>
        复制到系统剪贴板
       </strong>
       <br/>
       在普通模式下输入：
      </p>
      <pre><code class="prism language-bash">:%y+
</code></pre>
      <ul>
       <li>
        <code>
         %
        </code>
        ：全选文件内容
       </li>
       <li>
        <code>
         y+
        </code>
        ：复制到系统剪贴板
       </li>
      </ul>
     </li>
    </ol>
    <hr/>

     <a id="_60">

     <strong>
      注意事项
     </strong>
    </h4>
    <ol>
     <li>
      如果 Vim 不支持系统剪贴板（显示
      <code>
       -clipboard
      </code>
      ），需重新安装支持剪贴板的版本：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> vim-gtk  <span class="token comment"># Ubuntu/Debian</span>

</code></pre>
     </li>
     <li>
      粘贴内容时，在普通模式下按
      <code>
       p
      </code>
      （粘贴到光标后）或
      <code>
       P
      </code>
      （粘贴到光标前）。
     </li>
    </ol>
