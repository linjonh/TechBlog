    <p>
     前端向后端传LocalDateTime会报格式错误的异常。此时需要我们在后端配置日期格式
    </p>
    <p>
    </p>
    <p>
     Converter引用的包为：
    </p>
    <pre>import org.springframework.core.convert.converter.Converter;</pre>
    <pre><code class="language-java">    @Bean
    Converter&lt;String,LocalDateTime&gt; dateTimeConverter(){
        return new Converter&lt;String, LocalDateTime&gt;() {
            @Override
            public LocalDateTime convert(String s) {
                return LocalDateTime.parse(s, DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
            }
        };</code></pre>
