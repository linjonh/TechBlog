    <p>
     本文起到数据结构学习前的C语言知识复习补充作用

    <h4>
     <a id="_1">
     </a>
     结构体：
    </h4>
    <h5>
     <a id="1__4">
     </a>
     <strong>
      1. 结构体的定义与作用
     </strong>
    </h5>
    <p>
     <strong>
      核心概念
     </strong>
     ：结构体（
     <code>
      struct
     </code>
     ）是C语言中一种
     <strong>
      自定义数据类型
     </strong>
     ，用于将多个不同类型的变量组合成一个整体。
     <br/>
     <strong>
      解决的问题
     </strong>
     ：当需要同时管理一组逻辑相关但类型不同的数据时（如学生信息、商品属性），结构体可以简化数据管理。
    </p>
    <p>
     <strong>
      示例场景
     </strong>
     ：
     <br/>
     示例一：存储一个学生的信息：姓名（字符串）、年龄（整数）、成绩（浮点数）；
     <br/>
     示例二：存储一个二维平面的点 例如a：（ x ， y ）。
    </p>
    <hr/>
    <h5>
     <a id="2__14">
     </a>
     <strong>
      2. 结构体的声明
     </strong>
    </h5>
    <p>
     <strong>
      语法
     </strong>
     ：
    </p>
    <pre><code class="prism language-c"><span class="token comment">//对应示例一：</span>
<span class="token keyword">struct</span> 结构体名
 <span class="token punctuation">{<!-- --></span>
    数据类型 成员<span class="token number">1</span><span class="token punctuation">;</span>
    数据类型 成员<span class="token number">2</span><span class="token punctuation">;</span>
    <span class="token comment">// ...</span>
 <span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <pre><code class="prism language-c"><span class="token comment">//对应示例二：</span>
<span class="token keyword">struct</span> （结构体名）Point 
<span class="token punctuation">{<!-- --></span>
    数值类型 x<span class="token punctuation">;</span>  <span class="token comment">// 横坐标</span>
    数值类型 y<span class="token punctuation">;</span>  <span class="token comment">// 纵坐标</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      代码示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-c"><span class="token comment">//对应示例一：</span>
<span class="token comment">// 声明一个结构体类型 Student</span>
<span class="token keyword">struct</span> <span class="token class-name">Student</span> 
<span class="token punctuation">{<!-- --></span>
    <span class="token keyword">char</span> name<span class="token punctuation">[</span><span class="token number">20</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
    <span class="token keyword">int</span> age<span class="token punctuation">;</span>
    <span class="token keyword">float</span> score<span class="token punctuation">;</span>
 <span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <pre><code class="prism language-c"><span class="token comment">//对应示例二：</span>
<span class="token keyword">struct</span> <span class="token class-name">Point</span> 
<span class="token punctuation">{<!-- --></span>
    <span class="token keyword">float</span> x<span class="token punctuation">;</span>  <span class="token comment">// x坐标</span>
    <span class="token keyword">float</span> y<span class="token punctuation">;</span>  <span class="token comment">// y坐标</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <hr/>
    <h5>
     <a id="3__57">
     </a>
     <strong>
      3. 结构体变量的定义与初始化
     </strong>
    </h5>
    <h6>
     <a id="1__58">
     </a>
     (1) 直接定义变量
    </h6>
    <pre><code class="prism language-c"><span class="token keyword">struct</span> <span class="token class-name">Student</span>
 <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">char</span> name<span class="token punctuation">[</span><span class="token number">20</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
    <span class="token keyword">int</span> age<span class="token punctuation">;</span>
 <span class="token punctuation">}</span> stu1<span class="token punctuation">,</span> stu2<span class="token punctuation">;</span> <span class="token comment">// 直接定义两个结构体变量 stu1 和 stu2</span>
</code></pre>
    <h6>
     <a id="2__67">
     </a>
     (2) 先声明类型，再定义变量
    </h6>
    <pre><code class="prism language-c"><span class="token keyword">struct</span> <span class="token class-name">Student</span> stu3<span class="token punctuation">;</span> <span class="token comment">// 通过类型名定义变量</span>
</code></pre>
    <h6>
     <a id="3__72">
     </a>
     (3) 初始化赋值
    </h6>
    <pre><code class="prism language-c"><span class="token keyword">struct</span> <span class="token class-name">Student</span> stu4 <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token string">"张三"</span><span class="token punctuation">,</span> <span class="token number">18</span><span class="token punctuation">,</span> <span class="token number">90.5</span><span class="token punctuation">}</span><span class="token punctuation">;</span> <span class="token comment">// 按顺序初始化成员</span>
</code></pre>
    <hr/>
    <h2>
     <a id="typedef_77">
     </a>
     关于结构体括号后是否加名字的问题（
     <code>
      typedef
     </code>
     ）
    </h2>
    <h4>
     <a id="1_typedef_78">
     </a>
     <strong>
      1. 未使用
      <code>
       typedef
      </code>
      的结构体
     </strong>
    </h4>
    <h5>
     <a id="_79">
     </a>
     <strong>
      代码示例
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token comment">// 定义结构体类型</span>
<span class="token keyword">struct</span> <span class="token class-name">Point</span> 
<span class="token punctuation">{<!-- --></span>
    <span class="token keyword">double</span> x<span class="token punctuation">;</span>
    <span class="token keyword">double</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token comment">// 声明结构体变量时必须带"struct"关键字</span>
<span class="token keyword">struct</span> <span class="token class-name">Point</span> a <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token number">3.0</span><span class="token punctuation">,</span> <span class="token number">4.0</span><span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_92">
     </a>
     <strong>
      缺点
     </strong>
    </h5>
    <ul>
     <li>
      每次声明变量时都需要写
      <code>
       struct Point
      </code>
      ，冗余且繁琐。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2_typedef_97">
     </a>
     <strong>
      2. 使用
      <code>
       typedef
      </code>
      后的结构体
     </strong>
    </h4>
    <h5>
     <a id="_98">
     </a>
     <strong>
      代码示例
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token comment">// 通过typedef为结构体定义别名"Point"</span>
<span class="token keyword">typedef</span> <span class="token keyword">struct</span> 
<span class="token punctuation">{<!-- --></span>
    <span class="token keyword">double</span> x<span class="token punctuation">;</span>
    <span class="token keyword">double</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span> Point<span class="token punctuation">;</span>

<span class="token comment">// 声明变量时直接使用别名</span>
Point a <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span><span class="token number">3.0</span><span class="token punctuation">,</span> <span class="token number">4.0</span><span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_111">
     </a>
     <strong>
      优点
     </strong>
    </h5>
    <ul>
     <li>
      简化变量声明，省略
      <code>
       struct
      </code>
      关键字。
     </li>
     <li>
      提升代码可读性，更接近自然语言（如
      <code>
       Point
      </code>
      直接表示点的类型）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3_typedef_117">
     </a>
     <strong>
      3.
      <code>
       typedef
      </code>
      的两种常见写法
     </strong>
    </h4>
    <h5>
     <a id="1__typedef_118">
     </a>
     <strong>
      写法1：匿名结构体 + typedef
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token keyword">typedef</span> <span class="token keyword">struct</span> 
<span class="token punctuation">{<!-- --></span>  
<span class="token comment">// 结构体本身没有名字（匿名）</span>
    <span class="token keyword">double</span> x<span class="token punctuation">;</span>
    <span class="token keyword">double</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span> Point<span class="token punctuation">;</span>  
<span class="token comment">// 别名是Point</span>
</code></pre>
    <h5>
     <a id="2__typedef_129">
     </a>
     <strong>
      写法2：有名字的结构体 + typedef
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token keyword">typedef</span> <span class="token keyword">struct</span> <span class="token class-name">_Point</span> 
<span class="token punctuation">{<!-- --></span>  
<span class="token comment">// 结构体本身有名字"_Point"</span>
    <span class="token keyword">double</span> x<span class="token punctuation">;</span>
    <span class="token keyword">double</span> y<span class="token punctuation">;</span>
<span class="token punctuation">}</span> Point<span class="token punctuation">;</span>  
<span class="token comment">// 别名是Point</span>
</code></pre>
    <h5>
     <a id="_140">
     </a>
     <strong>
      两种写法的区别
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       匿名结构体
      </strong>
      ：适用于不需要在结构体内部引用自身的情况（如普通坐标点）。
     </li>
     <li>
      <strong>
       有名字的结构体
      </strong>
      ：适用于需要
      <strong>
       自引用
      </strong>
      的场景（如链表节点）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4__146">
     </a>
     <strong>
      4. 自引用场景示例（链表节点）
     </strong>
    </h4>
    <h5>
     <a id="_147">
     </a>
     <strong>
      必须使用有名字的结构体
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token keyword">typedef</span> <span class="token keyword">struct</span> <span class="token class-name">Node</span> 
<span class="token punctuation">{<!-- --></span>  
<span class="token comment">// 结构体名为Node</span>
    <span class="token keyword">int</span> data<span class="token punctuation">;</span>
    <span class="token keyword">struct</span> <span class="token class-name">Node</span><span class="token operator">*</span> next<span class="token punctuation">;</span>  
    <span class="token comment">// 内部需要引用自身类型</span>
<span class="token punctuation">}</span> Node<span class="token punctuation">;</span>  
<span class="token comment">// 别名为Node</span>

<span class="token comment">// 使用示例</span>
Node head<span class="token punctuation">;</span>
head<span class="token punctuation">.</span>data <span class="token operator">=</span> <span class="token number">1</span><span class="token punctuation">;</span>
head<span class="token punctuation">.</span>next <span class="token operator">=</span> <span class="token constant">NULL</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_164">
     </a>
     <strong>
      错误写法（匿名结构体）
     </strong>
    </h5>
    <pre><code class="prism language-c"><span class="token keyword">typedef</span> <span class="token keyword">struct</span> 
<span class="token punctuation">{<!-- --></span>  <span class="token comment">// 匿名结构体</span>
    <span class="token keyword">int</span> data<span class="token punctuation">;</span>
    <span class="token keyword">struct</span> <span class="token class-name">Node</span><span class="token operator">*</span> next<span class="token punctuation">;</span>  <span class="token comment">// 错误！此时"Node"还未定义</span>
<span class="token punctuation">}</span> Node<span class="token punctuation">;</span>
</code></pre>
    <hr/>
    <h4>
     <a id="5__175">
     </a>
     <strong>
      5. 伪代码对比
     </strong>
    </h4>
    <h5>
     <a id="typedef_176">
     </a>
     <strong>
      未用
      <code>
       typedef
      </code>
     </strong>
    </h5>
    <pre><code class="prism language-plaintext">定义结构体 Point {
    x: 数值类型
    y: 数值类型
}

声明变量: struct Point a = {x:3, y:4}
</code></pre>
    <h5>
     <a id="typedef_186">
     </a>
     <strong>
      使用
      <code>
       typedef
      </code>
      后
     </strong>
    </h5>
    <pre><code class="prism language-plaintext">定义类型 Point 为结构体 {
    x: 数值类型
    y: 数值类型
}

声明变量: Point a = {x:3, y:4}
</code></pre>
    <hr/>
    <h4>
     <a id="6__198">
     </a>
     <strong>
      6. 总结
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        typedef
       </code>
       的作用
      </strong>
      ：为复杂类型（如结构体）定义简洁的别名。
     </li>
     <li>
      <strong>
       适用场景
      </strong>
      ：
      <ul>
       <li>
        简化代码，减少
        <code>
         struct
        </code>
        重复书写。
       </li>
       <li>
        定义自引用结构体（如链表、树节点）。
       </li>
       <li>
        提高代码可移植性（例如统一修改类型名称）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       注意事项
      </strong>
      ：自引用时必须使用有名字的结构体定义。
     </li>
    </ul>
