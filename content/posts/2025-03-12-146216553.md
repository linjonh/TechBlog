# 洛谷 P5534 【XR-3】等差数列 python

这题不用向下取整//就会错，不太能理解为什么...感觉对结果好像没什么影响啊

![](https://i-blog.csdnimg.cn/direct/7d0f9cb81d1c4c9c95cebb984caea025.png)

```python
a1, a2, n = map(int,input().split())
d = a2 - a1
an = a1 + d * (n-1)
s = (a1+an)*n//2
print(s)
```
