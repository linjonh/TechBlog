    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     一、FusionCompute架构
    </p>
    <ol>
     <li style="text-align:justify;">
      架构CNA作为虚拟化操作系统，VRM作为虚拟化管理平台
      <img alt="" height="312" src="https://i-blog.csdnimg.cn/blog_migrate/de3ddbd4aee70b10126eee738cf4e06e.png" width="244"/>
     </li>
    </ol>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <ol>
     <li style="text-align:justify;">
      正常主机都安装CNA，单独建立VRM集群作为管理集群，我测试环境就一台主机，所以CNA和VRM装在同一台主机上，并且用这台主机分配虚拟机进行测试。
     </li>
    </ol>
    <p style="margin-left:.0001pt;text-align:justify;">
     二、安装VRM
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     1、VRM安装过程基本跟正常装操作系统类似，将硬盘做好raid，镜像刻好光盘，以光盘启动。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     2、设置好每一项，此次安装没有应用vlan模式，配置好主机地址，带*的必填。填好后即可开始安装。
     <img alt="" height="264" src="https://i-blog.csdnimg.cn/blog_migrate/5d13b25e6ea1afb1966521f680516800.png" width="535"/>
    </p>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="text-align:justify;">
     主机安装重启后要进行初始化设置，ssh不允许root用户登陆。初始化执行命令cnaInit，
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     ①选择密码算法模式
     <img alt="" height="75" src="https://i-blog.csdnimg.cn/blog_migrate/7b1009c53f6c1c0f03f7ca81197f99ad.png" width="554"/>
    </p>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     ②输入gandalf密码（作为远程登陆使用）
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
     <img alt="" height="84" src="https://i-blog.csdnimg.cn/blog_migrate/7dfb8ea0175f95d2836b8b83209c042e.png" width="315"/>
    </p>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     ③设置redis密码
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <img alt="" height="53" src="https://i-blog.csdnimg.cn/blog_migrate/4f16c5a673161523e6f04d407407f393.png" width="251"/>
    </p>
    <p>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     注意密码要符合复杂性规则
    </p>
    <ol>
     <li style="text-align:justify;">
      安装完毕，可使用gandalf远程ssh登陆。
     </li>
    </ol>
    <ul>
     <li style="text-align:justify;">
      安装VRM
     </li>
    </ul>
    <ol>
     <li style="text-align:justify;">
      单独安装VRM可以使用iso镜像直接安装，因为是跟CNA共用一台主机，所以使用安装工具进行安装。
     </li>
     <li style="text-align:justify;">
      安装FusionCompute安装工具
     </li>
    </ol>
    <ul>
     <li style="text-align:justify;">
      获取到安装工具FusionCompute-LinuxInstaller-8.3.0-ARM_64.zip
     </li>
     <li style="text-align:justify;">
      使用远程传输工具WinSCP将FusionCompute安装工具的zip包传输至“/home/GalaX8800”目录下.
     </li>
     <li style="text-align:justify;">
      使用gandalf远程ssh登陆到刚安装的CNA主机
     </li>
     <li style="text-align:justify;">
      切换到root用户 su - root
     </li>
     <li style="text-align:justify;">
      执行以下命令解压安装包unzip FusionCompute-LinuxInstaller-8.3.0-ARM_64.zip.
     </li>
     <li style="text-align:justify;">
      执行以下命令访问解压包路径,cd 解压包文件夹名称
     </li>
     <li style="text-align:justify;">
      sh bin/webInstaller.sh install
     </li>
    </ul>
    <ol>
     <li style="text-align:justify;">
      安装VRM
     </li>
    </ol>
    <ul>
     <li style="text-align:justify;">
      在浏览器地址栏输入https://主机ip:8080打开工具界面。输入用户名和密码之后，单击“登录”，初始帐号信息：admin，初始密码：IaaS@PORTAL-CLOUD8!
     </li>
     <li style="text-align:left;">
      在界面左侧选择自定义安装，组件选择VRM
      <img alt="" height="232" src="https://i-blog.csdnimg.cn/blog_migrate/af776e0e31fc6e5b1984fddf94e7cbd5.png" width="553">
      </img>
     </li>
     <li style="text-align:left;">
      第1步为介绍
      <img alt="" height="251" src="https://i-blog.csdnimg.cn/blog_migrate/df47819ab84c0e4c2ce990ce3a2f0c8a.png" width="508">
      </img>
     </li>
     <li style="text-align:left;">
      将下载好的VRM包上传
     </li>
    </ul>
    <p>
     <img alt="" height="228" src="https://i-blog.csdnimg.cn/blog_migrate/034223513c374aa78ba0a1ce53d8f8c6.png" width="490"/>
    </p>
    <p style="margin-left:.0001pt;text-align:left;">
    </p>
    <ul>
     <li style="text-align:left;">
      选择安装模式，一般典型安装即可
     </li>
    </ul>
    <p style="margin-left:.0001pt;text-align:left;">
     <img alt="" height="270" src="https://i-blog.csdnimg.cn/blog_migrate/ccc68904765114d108ce8d04e3b8aec6.png" width="553"/>
    </p>
    <ul>
     <li style="text-align:left;">
      安装模式选择单节点，按照要求填好每一项
      <img alt="" height="444" src="https://i-blog.csdnimg.cn/blog_migrate/2340b6f6f1d5efa3160dfc8e34054d99.png" width="554"/>
     </li>
     <li style="text-align:left;">
      等待安装完成
      <img alt="" height="268" src="https://i-blog.csdnimg.cn/blog_migrate/370184415a019fb1c5b1c516e3fe6787.png" width="553"/>
     </li>
     <li style="text-align:left;">
      在浏览器输入FusionCompute地址跳转到VRM登录页。
      <img alt="" height="239" src="https://i-blog.csdnimg.cn/blog_migrate/ea55d1f1136b262712eeba9d683a7e99.png" width="553"/>
     </li>
    </ul>
    <p style="margin-left:.0001pt;text-align:justify;">
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     至此，CNA和VRM都已经安装完成，可以去创建虚拟机使用了
    </p>
