    <h3>
     <a id="_1">
     </a>
     项目简介
    </h3>
    <blockquote>
     <p>
      该项目是基于Python+Vue开发的电影订票管理系统（前后端分离），这是一项为大学生课程设计作业而开发的项目。该系统旨在帮助大学生学习并掌握Python编程技能，同时锻炼他们的项目设计与开发能力。通过学习基于Python的电影订票管理系统项目，大学生可以在实践中学习和提升自己的能力，为以后的职业发展打下坚实基础。技术学习之路
     </p>
    </blockquote>
    <h3>
     <a id="_7">
     </a>
     主要功能
    </h3>
    <ul>
     <li>
      影片管理：管理系统可以录入、修改和查询影片的基本信息，如名称、价格、语言、地区、简介等。
     </li>
     <li>
      类型管理：系统可以管理影片的类型信息，包括类型的名称等。
     </li>
     <li>
      评论管理：管理和浏览整个网站的评论信息。
     </li>
     <li>
      用户管理：管理和浏览网站的用户信息，可以新增、编辑和删除用户。
     </li>
     <li>
      统计分析：系统可以根据影片的活动数据和用户参与度进行统计和分析，帮助管理员了解整个系统的状况。
     </li>
     <li>
      消息管理：影片管理员可以在系统上发布消息，整个网站的用户都能收到。
     </li>
     <li>
      广告管理：影片管理员可以在系统上发布广告消息，然后在详情页面右侧展示。
     </li>
     <li>
      意见反馈：影片管理员可以在后台查看浏览用户提交的意见反馈信息。
     </li>
     <li>
      系统信息：管理员可以查看系统的基本信息，包括系统名称、服务器信息、内存信息、cpu信息、软件信息等。
     </li>
     <li>
      注册登录：用户通过注册和登录后，才能使用网站。
     </li>
     <li>
      门户浏览：用户进入首页后，可以浏览影片列表信息，包括最新、最热。
     </li>
     <li>
      影片推荐：基于协同过滤推荐算法的热门推荐。
     </li>
     <li>
      用户中心：包括用户基本资料修改、用户基本信息、密码、收藏点赞等。
     </li>
     <li>
      我的订单：包括我购买的影片的门票信息。
     </li>
     <li>
      意见反馈：包括用户提交意见反馈的入口页面。
     </li>
     <li>
      模糊搜索：顶部搜索功能，支持模糊搜索影片信息。
     </li>
     <li>
      影片评论：详情页下侧用户可以评论影片。
     </li>
    </ul>

    <h4>
     <a id="_56">
     </a>
     后端运行步骤
    </h4>
    <p>
     (1) 安装依赖，cd进入server目录下，执行
    </p>
    <pre><code>pip install -r requirements.txt
</code></pre>
    <p>
     (2) 创建数据库，创建SQL如下：
    </p>
    <pre><code>CREATE DATABASE IF NOT EXISTS python_db[your dbname] DEFAULT CHARSET utf8 COLLATE utf8_general_ci
</code></pre>
    <p>
     (3) 恢复数据库数据。在mysql下依次执行如下命令：
    </p>
    <pre><code>mysql&gt; use xxx(数据库名);
mysql&gt; source D:/xxx/xxx/xxx.sql;
</code></pre>
    <p>
     (4) 配置数据库。在server目录下的server下的settings.py中配置您的数据库账号密码
    </p>
    <pre><code>DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'python_db',   # 您的数据库
        'USER': 'root',        # 您的用户名
        'PASSWORD': '4643830', # 您的密码
        'HOST': '127.0.0.1',
        'PORT': '3306',
        'OPTIONS': {
            "init_command": "SET foreign_key_checks = 0;",
        }
    }
}
</code></pre>
    <p>
     (5) 启动django服务。在server目录下执行：
    </p>
    <pre><code>python manage.py runserver
</code></pre>
    <h4>
     <a id="_97">

     前端运行步骤

    </p>
    <pre><code>npm run dev
</code></pre>
