    <p>
     做TopCoder
     <a href="http://community.topcoder.com/stat?c=problem_statement&amp;pm=12506&amp;rd=15496" rel="nofollow noopener noreferrer" target="_blank">
      SRM 576 D2 L3
     </a>
     题目时，程序有个地方需要对一个数大量求幂并取余，导致程序运行时间很长，看了
     <a href="http://apps.topcoder.com/wiki/display/tc/SRM+576" rel="nofollow noopener noreferrer" target="_blank">
      Editoral
     </a>
     之后，发现一个超级高效的求幂并取余的算法，之前做System test时，程序运行时间（最慢的测试用例）为500ms左右，使用此方法之后，运行时间直接减为20ms，快了20多倍，所以将此方法记录下来。
    </p>
    <p>
     算法时间复杂度为 log(n)。
    </p>
    <p>
     这个算法其实就是
     <span style="font-family:Arial,Helvetica,sans-serif; font-size:12px">
      数据结构与算法分析 （Weiss 著）
     </span>
     一书中开头的那个递归求幂算法的非递归版，简洁明了。
    </p>
    <p>
     代码如下：
    </p>
    <p>
    </p>
    <pre><code class="language-cpp">	// Finds x raised to the y-th exponent modulo MOD （即求 x^y % MOD）
    const int MOD = 1e9 + 9;
    int modPow(int x, int y)
	{
		long long res = 1, a = x;
		while (y &gt; 0) {
			if (y &amp; 1) {
				res = (res * a) % MOD;
			}
			a = (a * a) % MOD;
			y &gt;&gt;= 1;
		}
		return static_cast&lt;int&gt; (res);
	}</code></pre>
