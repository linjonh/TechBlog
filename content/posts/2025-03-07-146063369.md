    <h2>
     <a id="refreactive_0">
     </a>
     <code>
      ref
     </code>
     和
     <code>
      reactive
     </code>
     的区别
    </h2>
    <h4>
     <a id="ref_1">
     </a>
     <code>
      ref
     </code>
    </h4>
    <p>
     用途：用于创建一个响应式的基本类型（string、number、boolean）或者引用类型（object、array）的数据。
     <br/>
     返回值：返回一个带有
     <code>
      .value
     </code>
     属性的对象，访问或者修改数据时需要通过
     <code>
      .value
     </code>
     进行。
     <br/>
     使用场景：适合处理单个值或需要明确访问
     <code>
      .value
     </code>
     的场景。
    </p>
    <h4>
     <a id="reactive_5">
     </a>
     <code>
      reactive
     </code>
    </h4>
    <p>
     用途：用于创建一个响应式的对象或者数组。
     <br/>
     返回值：返回一个响应式的
     <code>
      Proxy对象
     </code>
     ，可以直接访问或修改属性，无需
     <code>
      .value
     </code>
     。
     <br/>
     使用场景：适合处理复杂对象或者嵌套数据结构。
    </p>
    <blockquote>
     <p>
      注意：在一个对象或者数组中，如果想要访问更深层的属性或者元素，推荐使用
      <code>
       reactive
      </code>
      <br/>
      使用
      <code>
       reactive
      </code>
      只需要通过
      <code>
       obj.a.b.c
      </code>
      或
      <code>
       arr[0].name
      </code>
      的方式访问深层属性或元素，不需要额外的
      <code>
       .value
      </code>
      <br/>
      使用
      <code>
       ref
      </code>
      则需要通过
      <code>
       obj.value.a.b.c
      </code>
      或
      <code>
       arr[0].value.name
      </code>
      的方式访问深层属性或元素
     </p>
    </blockquote>

    <h2>
     <a id="vif__vshow_62">
     </a>
     <code>
      v-if
     </code>
     和
     <code>
      v-show
     </code>
    </h2>
    <h4>
     <a id="vif_63">
     </a>
     <code>
      v-if
     </code>
    </h4>
    <p>
     条件性地添加或移除
     <code>
      DOM 元素
     </code>
     。
    </p>
    <h4>
     <a id="vshow_65">
     </a>
     <code>
      v-show
     </code>
    </h4>
    <p>
     通过
     <code>
      CSS
     </code>
     的
     <code>
      display
     </code>
     控制显示和隐藏（
     <code>
      display: none
     </code>
     ），元素始终会被渲染到
     <code>
      DOM
     </code>
     中
    </p>
    <blockquote>
     <p>
      注意：当
      <code>
       v-if
      </code>
      和
      <code>
       v-for
      </code>
      同时存在于一个元素上的时候，
      <code>
       v-if
      </code>
      会首先被执行。
     </p>
    </blockquote>
