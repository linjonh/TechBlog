    <h4>
     <a id="4_RANSAC_112">
     </a>
     <strong>
      4. 鲁棒平面拟合（RANSAC）
     </strong>
    </h4>
    <p>
     当数据中存在噪声或离群点时，RANSAC（随机采样一致性）算法可提高拟合鲁棒性。
    </p>
    <h5>
     <a id="_115">
     </a>
     <strong>
      步骤
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       随机采样
      </strong>
      ：
      <br/>
      随机选择3个点计算初始平面参数。
     </li>
     <li>
      <strong>
       内点检测
      </strong>
      ：
      <br/>
      计算所有点到该平面的距离，设定阈值
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         ( 
         
        
          ϵ 
         
        
          ) 
         
        
       
         ( \epsilon )
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 1em; vertical-align: -0.25em;">
          </span>
          <span class="mopen">
           (
          </span>
          <span class="mord mathnormal">
           ϵ
          </span>
          <span class="mclose">
           )
          </span>
         </span>
        </span>
       </span>
      </span>
      ，统计距离小于
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         ( 
         
        
          ϵ 
         
        
          ) 
         
        
       
         ( \epsilon )
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 1em; vertical-align: -0.25em;">
          </span>
          <span class="mopen">
           (
          </span>
          <span class="mord mathnormal">
           ϵ
          </span>
          <span class="mclose">
           )
          </span>
         </span>
        </span>
       </span>
      </span>
      的内点数量。
     </li>
     <li>
      <strong>
       迭代优化
      </strong>
      ：
      <br/>
      重复上述过程，保留内点最多的平面模型。
     </li>
     <li>
      <strong>
       精确拟合
      </strong>
      ：
      <br/>
      使用所有内点通过最小二乘法重新拟合最终平面。
     </li>
    </ol>
