    <h2>
     一、动态规划
    </h2>
    <p>
     <strong>
      动态规划
     </strong>
     是一种用于解决优化问题的算法设计技术，尤其适用于具有重叠子问题和最优子结构性质的问题。它通过将复杂问题分解为更简单的子问题，并保存这些子问题的解以避免重复计算，从而提高效率。
    </p>
    <h4>
     动态规划的核心思想
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        最优子结构（Optimal Substructure）
       </strong>
       ：
      </p>
      <ul>
       <li>
        一个问题的最优解可以通过其子问题的最优解来构造。
       </li>
       <li>
        比如在最短路径问题中，从起点到终点的最短路径可以由起点到某个中间点的最短路径和该中间点到终点的最短路径组合而成。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        重叠子问题（Overlapping Subproblems）
       </strong>
       ：
      </p>
      <ul>
       <li>
        在递归求解过程中，相同的子问题会被多次求解。
       </li>
       <li>
        动态规划通过存储子问题的解来避免重复计算，通常使用数组或哈希表等数据结构来存储这些解。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <span style="color:#4da8ee">
      1.算法原理
     </span>
    </h3>
    <p>
     <span style="color:#1a439c">
      1)状态表示
     </span>
    </p>
    <p>
     dp表(一维数组  填满该表其中某一个值就是结果 数组中某个数据值的意义就是状态表示)
    </p>
    <p>
     通过题目要求 经验  分析问题发现重复子问题 来创建dp表
    </p>
    <p>
     <span style="color:#1a439c">
      2)状态转移方程
     </span>
    </p>
    <p>
     dp[i]等于什么?
    </p>
    <p>
     <span style="color:#1a439c">
      3)初始化
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      根据状态转移方程填表,
     </span>
     <span style="color:#0d0016">
      保证填表的时候不越界
     </span>
    </p>
    <p>
     <span style="color:#1a439c">
      4)填表顺序
     </span>
    </p>
    <p>
     为了填写当前状态的时候,所需要的状态已经计算过了
    </p>
    <p>
     <span style="color:#1a439c">
      5)返回值
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      题目要求+状态表示
     </span>
    </p>
    <h3>
     <span style="color:#0d0016">
      2.例题
     </span>
    </h3>
    <p>
     泰波那契序列 Tn 定义如下：
    </p>
    <p>
     T0 = 0, T1 = 1, T2 = 1, 且在 n &gt;= 0 的条件下 Tn+3 = Tn + Tn+1 + Tn+2
    </p>
    <p>
     给你整数
     <code>
      n
     </code>
     ，请返回第 n 个泰波那契数 Tn 的值。
    </p>
    <p>
     <strong>
      示例 1：输入：
     </strong>
     n = 4
     <strong>
      输出：
     </strong>
     4
     <strong>
      解释：
     </strong>
     T_3 = 0 + 1 + 1 = 2 T_4 = 1 + 1 + 2 = 4
    </p>
    <p>
     状态表示:dp[i]第i个泰波那契数的值
    </p>
    <p>
     <span style="color:#1a439c">
      状态转移方程:dp[i]=dp[i-1]+dp[i-2]+dp[i-3];
     </span>
    </p>
    <p>
     初始化:dp[0]=0;dp[1]=1;dp[2]=1
    </p>
    <p>
     填表顺序:从左向右
    </p>
    <p>
     <span style="color:#1a439c">
      返回值:dp[n]
     </span>
    </p>
    <p>
     JAVA代码
    </p>
    <pre><code class="language-java">class Solution {
    public int tribonacci(int n) {
if(n==0) return 0;
if(n==1||n==2) return 1;
int[] dp=new int[n+1];
dp[0]=0;dp[1]=1;dp[2]=1;
for(int i=3;i&lt;=n;i++){
    dp[i]=dp[i-1]+dp[i-2]+dp[i-3];
}
return dp[n];
    }
}</code></pre>
    <h3>
     3.空间优化
    </h3>
    <p>
     求解某个状态时仅需要其的前几个状态
    </p>
    <p>
     一定要确定好赋序
    </p>
    <pre><code class="language-cs">int tribonacci(int n){
//空间优化 滚动数组
if(n==0) return 0;
if(n==1||n==2) return 1;
int a=0;int b=1,c=1;int d=0;
for(int i=3;i&lt;=n;i++){
    d=a+b+c;
    a=b;
    b=c;
    c=d;

}
return d;
}
</code></pre>

    <h2>
     二、贪心算法
    </h2>
    <h3>
     <span style="color:#6eaad7">
      1.算法原理:
     </span>
    </h3>
    <p>
     局部最优-&gt;全局最优
    </p>
    <p>
     把解决问题的过程分为若干步,
     <span style="color:#1a439c">
      解决每一步的时候都选择当前看起来最优的解
     </span>
     法
    </p>
    <p>
     希望得到全局最优解
    </p>
    <p>
     但贪心算法并不总是保证全局最优解
    </p>
    <h3>
     2.例题
    </h3>
    <p>
     给定一个数组
     <code>
      prices
     </code>
     ，它的第
     <code>
      i
     </code>
     个元素
     <code>
      prices[i]
     </code>
     表示一支给定股票第
     <code>
      i
     </code>
     天的价格。
    </p>
    <p>
     你只能选择
     <strong>
      某一天
     </strong>
     买入这只股票，并选择在
     <strong>
      未来的某一个不同的日子
     </strong>
     卖出该股票。设计一个算法来计算你所能获取的最大利润。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <pre>[7,1,5,3,6,4]
<strong>输出：</strong>5
<strong>解释：</strong>在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。
</pre>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <pre><strong>输入：</strong>prices = [7,6,4,3,1]
<strong>输出：</strong>0
<strong>解释：</strong>在这种情况下, 没有交易完成, 所以最大利润为 0。</pre>
    <p>
     <span style="color:#1a439c">
      在数组中挑出两个数,使其差值最大
     </span>
    </p>
    <p>
     1)暴力解法(两层for循环)
     <span style="color:#fe2c24">
      超出时间限制时间复杂度O(n^2)
     </span>
    </p>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

class Solution {
    public int maxProfit(int[] prices) {
        int ret = 0;
        if (prices == null || prices.length &lt; 2) {
            return 0;
        }
        for (int i = 0; i &lt; prices.length - 1; i++) {
            for (int j = i+1; j &lt; prices.length; j++) {
                ret = Math.max(ret, prices[j] - prices[i]);
            }
        }
        return ret;
    }


        public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        Solution solution=new Solution();
        List&lt;Integer&gt; pricesList = new ArrayList&lt;&gt;();
        while (sc.hasNextInt()) {
            pricesList.add(sc.nextInt());
        }
        int[] prices = new int[pricesList.size()];
        for (int i = 0; i &lt; pricesList.size(); i++) {
            prices[i] = pricesList.get(i);
        }
        int ret=solution.maxProfit(prices);
        System.out.println(ret);
        sc.close();
    }
}//在输入结束后输入一个非数字字符（如字母 'q'），这样 hasNextInt() 会返回 false，从而结束循环,或 Ctrl + Z（Windows）来发送 EOF 信号。</code></pre>
    <p>
     2)贪心
    </p>
    <p>
     分为两段,prevmin表示前一段中最小值(买入),prices[i]卖出去的价钱
    </p>
    <pre><code class="language-java">class Solution {
    public int maxProfit(int[] prices) {
        int ret=0;
        for(int i=0,prevmin=Integer.MAX_VALUE;i&lt;prices.length;i++){
            ret=Math.max(ret,prices[i]-prevmin);
            prevmin=Math.min(prevmin,prices[i]);
        }
        return ret;
    }
}</code></pre>

    <h2 style="background-color:transparent">
     三、 (双指针算法)
    </h2>
    <p>
     给定一个数组
     <code>
      nums
     </code>
     ，编写一个函数将所有
     <code>
      0
     </code>
     移动到数组的末尾，同时保持非零元素的相对顺序。
    </p>
    <p>
     <strong>
      请注意
     </strong>
     ，必须在不复制数组的情况下原地对数组进行操作。
    </p>
    <p>
     <strong>
      示例 1:
     </strong>
    </p>
    <pre><strong>输入:</strong> nums = <code>[0,1,0,3,12]</code>
<strong>输出:</strong> <code>[1,3,12,0,0]</code></pre>
    <p>
     数组划分
    </p>
    <p>
     <span style="color:#1a439c">
      利用数组下标充当指针
     </span>
    </p>
    <p>
     两个指针的作用:
    </p>
    <p>
     cur:从左往右扫描,遍历数组
    </p>
    <p>
     dest:已处理的区间内,非0元素的最后一个位置
    </p>
    <p>
     三个区间:
    </p>
    <p>
     [0,dest]   [dest+1,cur-1]  [cur,n-1]
    </p>
    <p>
     非0         0元素                 待处理的区间
    </p>
    <p>
     解法:
    </p>
    <p>
     初始dest=-1 cur=0
    </p>
    <p>
     遇到0元素,cur++;
    </p>
    <p>
     遇到非0元素 swap(dest+1,cur)交换位置 dest++ cur++
    </p>
    <p>
     <img alt="" height="403" src="https://i-blog.csdnimg.cn/direct/711852ace5474419b54ba005df60fd83.png" width="781"/>
    </p>
    <pre><code class="hljs">class Solution {
    public void moveZeroes(int[] nums) {
int dest=-1;
int cur=0;
int k=0;
int n=nums.length;
while(cur&lt;n){
if(nums[cur]==0)
cur++;
else
{dest++;
k=nums[dest];
nums[dest]=nums[cur];
nums[cur]=k;
cur++;
}

}

}
}</code></pre>
    <p>
    </p>
    <p>
     给你一个整数数组
     <code>
      prices
     </code>
     ，其中
     <code>
      prices[i]
     </code>
     表示某支股票第
     <code>
      i
     </code>
     天的价格。
    </p>
    <p>
     在每一天，你可以决定是否购买和/或出售股票。你在任何时候
     <strong>
      最多
     </strong>
     只能持有
     <strong>
      一股
     </strong>
     股票。你也可以先购买，然后在
     <strong>
      同一天
     </strong>
     出售。
    </p>
    <p>
     返回
     <em>
      你能获得的
      <strong>
       最大
      </strong>
      利润
     </em>
     。
    </p>
    <pre><strong>输入：</strong>prices = [7,1,5,3,6,4]
<strong>输出：</strong>7
<strong>解释：</strong>在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4。
随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3。
最大总利润为 4 + 3 = 7 。</pre>
    <p>
     <span style="color:#1a439c">
      利用数组下标充当指针(i,j)
     </span>
    </p>
    <pre><code class="language-java">class Solution {
    public int maxProfit(int[] prices) {
        //双指针算法
        int ret=0;
        int i=0,j=0;
        for(i=0;i&lt;prices.length;i++){
            j=i;
            while(j+1&lt;prices.length &amp;&amp; prices[j+1]-prices[j]&gt;0){
                j++;
            }
            ret+=prices[j]-prices[i];
            i=j;
        }
        return ret;
           }
}</code></pre>
    <h2>
     四、递归
    </h2>
    <p>
     1.什么是递归
    </p>
    <p>
     函数自己调用自己(主问题-&gt;
     <span style="color:#1a439c">
      相同的子问题
     </span>
     )
    </p>
    <p>
     <span style="color:#1a439c">
      出口(一个问题不能在分割了)
     </span>
    </p>
    <h3>
     1.算法思想
    </h3>
    <p>
     函数头 函数体 递归出口
    </p>
    <p>
     把递归的函数当成一个黑盒,不在意递归的细节
    </p>
    <p>
     深度优先遍历(后序遍历)
    </p>
    <h3>
     2.例题
    </h3>
    <p>
     <a class="link-info" href="https://leetcode.cn/problems/evaluate-boolean-binary-tree/description/" rel="nofollow" title="计算布尔二叉树的值">
      计算布尔二叉树的值
     </a>
    </p>
    <p>
     给你一棵
     <strong>
      完整二叉树
     </strong>
     的根，这棵树有以下特征：
    </p>
    <ul>
     <li>
      <strong>
       叶子节点
      </strong>
      要么值为
      <code>
       0
      </code>
      要么值为
      <code>
       1
      </code>
      ，其中
      <code>
       0
      </code>
      表示
      <code>
       False
      </code>
      ，
      <code>
       1
      </code>
      表示
      <code>
       True
      </code>
      。
     </li>
     <li>
      <strong>
       非叶子节点
      </strong>
      要么值为
      <code>
       2
      </code>
      要么值为
      <code>
       3
      </code>
      ，其中
      <code>
       2
      </code>
      表示逻辑或
      <code>
       OR
      </code>
      ，
      <code>
       3
      </code>
      表示逻辑与
      <code>
       AND
      </code>
      。
     </li>
    </ul>
    <p>
     <strong>
      计算
     </strong>
     一个节点的值方式如下：
    </p>
    <ul>
     <li>
      如果节点是个叶子节点，那么节点的
      <strong>
       值
      </strong>
      为它本身，即
      <code>
       True
      </code>
      或者
      <code>
       False
      </code>
      。
     </li>
     <li>
      否则，
      <strong>
       计算
      </strong>
      两个孩子的节点值，然后将该节点的运算符对两个孩子值进行
      <strong>
       运算
      </strong>
      。
     </li>
    </ul>
    <p>
     返回根节点
     <em>
     </em>
     <code>
      root
     </code>
     的布尔运算值。
    </p>
    <p>
     <strong>
      完整二叉树
     </strong>
     是每个节点有
     <code>
      0
     </code>
     个或者
     <code>
      2
     </code>
     个孩子的二叉树。
    </p>
    <p>
     <strong>
      叶子节点
     </strong>
     是没有孩子的节点。
    </p>
    <pre><code class="language-java">class Solution {
    public boolean evaluateTree(TreeNode root) {
        if(root.left==null) return root.val==0 ? false: true;
        boolean left=evaluateTree(root.left);
        boolean right=evaluateTree(root.right);
        return root.val==2 ?left | right : left &amp; right;
    }
}</code></pre>
