    <h2>
     <a id="JAVA_5">
     </a>
     JAVA环境变量配置
    </h2>
    <ul>
     <li>
      新建系统变量。
      <strong>
       变量名
      </strong>
      ：JAVA_HOME，
      <strong>
       变量值
      </strong>
      （可以选择浏览目录，JAVA的根目录，本处为D:\Java\jdk1.8.0_251）
     </li>
     <li>
      在
      <strong>
       Path
      </strong>
      中新建一行：%JAVA_HOME%\bin
     </li>
    </ul>
    <h2>
     <a id="GIT_10">
     </a>
     GIT环境变量配置
    </h2>
    <ul>
     <li>
      在
      <strong>
       Path
      </strong>
      中新建一行（安装目录）：D:\Java\Git\cmd
     </li>
     <li>
      使用
      <code>
       git -v
      </code>
      可以看到版本号
     </li>
     <li>
      使用
      <code>
       git config --global user.name "自己的用户名"
      </code>
      增加用户
     </li>
     <li>
      使用
      <code>
       git config --global user.email 自己的邮箱
      </code>
      增加邮箱
     </li>
     <li>
      使用
      <code>
       git config --global --list
      </code>
      查看结果
     </li>
     <li>
      若是使用原安装后的目录，需要在旧的代码目录下加载一下
     </li>
    </ul>
    <blockquote>
     <p>
      git config --global --add safe.directory ‘项目路径’
     </p>
    </blockquote>
    <ul>
     <li>
      需要运行两次，一次是带引号的（在IDEA终端中运行一次即可），一次是不带引号且反斜杠（使用TortoiseGit时报错后需要在目录下打开终端运行）。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">git</span> config <span class="token parameter variable">--global</span> <span class="token parameter variable">--add</span> safe.directory <span class="token string">'E:\_work\pps-web'</span>
<span class="token function">git</span> config <span class="token parameter variable">--global</span> <span class="token parameter variable">--add</span> safe.directory E:/_work/pps-web
</code></pre>
    <h2>
     <a id="NODEJS_23">
     </a>
     NODEJS环境变量配置
    </h2>
    <ul>
     <li>
      使用zip版，解压后，先在目录下增加两个文件夹
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/241a674fe0474f4b9aa0e99881397128.png"/>
     </li>
     <li>
      新建系统变量。
      <strong>
       变量名
      </strong>
      ：NODE_HOME，
      <strong>
       变量值
      </strong>
      为安装根目录（本例使用的是ZIP版，不是安装版。D:\Java\node-v22.14.0-win-x64）
     </li>
     <li>
      在
      <strong>
       Path
      </strong>
      中新建三行：
     </li>
    </ul>
    <pre><code class="prism language-bash">%NODE_HOME%
%NODE_HOME%<span class="token punctuation">\</span>node_global
%NODE_HOME%<span class="token punctuation">\</span>node_cache
</code></pre>
    <ul>
     <li>
      在用户变量中的
      <strong>
       Path
      </strong>
      增加node_global路径
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/95e67e234dba422fb7da45ab156a43a1.png"/>
     </li>
    </ul>
