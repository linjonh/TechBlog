    <h4>
     <a id="_89">
     </a>
     关键点说明：
    </h4>
    <ul>
     <li>
      <strong>
       获取DOM对象
      </strong>
      ：使用
      <code>
       document.getElementById("mydiv")
      </code>
      获取页面中id为"mydiv"的元素。
     </li>
     <li>
      <strong>
       获取jQuery对象
      </strong>
      ：使用
      <code>
       $("#mydiv")
      </code>
      选取相同的元素，但这次是作为jQuery对象返回。
     </li>
     <li>
      <strong>
       DOM对象转jQuery对象
      </strong>
      ：使用
      <code>
       $()
      </code>
      把
      <code>
       DOM
      </code>
      对象括起来：
      <code>
       $(DOM对象)
      </code>
     </li>
     <li>
      <strong>
       jQuery对象转DOM对象
      </strong>
      ：可以使用
      <code>
       .get(0)
      </code>
      方法从jQuery对象中提取出对应的DOM对象。
     </li>
    </ul>

     关键点说明
    </h4>
    <ul>
     <li>
      <p>
       正确示例：
       <code>
        $("#id属性值")
       </code>

       <code>
        var clas = $(".blue");
       </code>

       </strong>

        </strong>

        <code>
         #

       </li>
       <li>
        <strong>
         类选择器
        </strong>
        ：通过
        <code>
         .
        </code>
        加上类名来选取所有具有该类的元素。
       </li>
       <li>
        <strong>
         元素选择器
        </strong>
        ：直接使用标签名来选取页面上所有的该类型的元素。
       </li>
       <li>
        <strong>
         通用选择器
        </strong>
        ：使用
        <code>
         *
        </code>
        来选取页面上的所有元素。
       </li>
       <li>
        <strong>
         组合选择器
        </strong>
        ：通过逗号分隔不同的选择器，可以同时选取多个不同条件的元素。
       </li>
      </ul>
     </li>
    </ul>

       </code>

     <li>

     </li>
     <li>
      <p>
       <strong>
        自定义属性
       </strong>
       ：指用户通过
       <code>
        data-*
       </code>

       <code>
        attr()
       </code>
       方法来获取或设置。

     </li>

      <p>

        input
       </code>

       <code>
        type
       </code>

        value
       </code>

       <code>
        checked
       </code>
       ,
       <code>
        selected
       </code>
       ,
       <code>
        disabled
       </code>
       ），推荐使用
       <code>
        prop()
       </code>
       方法。
      </p>
     </li>
     <li>
      <p>
       <strong>
        布尔类型属性
       </strong>
       ：反映状态（真或假）。复选框是否被选中应使用
       <code>
        prop('checked')
       </code>

     </strong>
