# 使用bat命令设置JAVA\_HOME

## 在bat 文件开头指定java\_home环境

set JAVA\_HOME=D:\soft\jdk1.8
  
set PATH=%JAVA\_HOME%/bin;%JAVA\_HOME%/jre/bin
  
set CLASSPATH=.;%JAVA\_HOME%\lib\dt.jar;%JAVA\_HOME%\lib\tools.jar
  
注意:临时有效,过程结束则不再启作用
