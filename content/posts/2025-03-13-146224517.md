    <h4>
     <a id="_0">
     </a>
     前言
    </h4>
    <blockquote>
     <p>
      <strong>
       在我们的工作中，可能有以下场景需要用到合并多个PDF：
      </strong>
     </p>
    </blockquote>
    <blockquote>
     <p>
      <strong>
       文档归档
      </strong>
      ：在企业或组织中，常常需要将相关的文档（如合同、报告、发票等）合并为一个PDF文件，以便于归档和管理。
      <br/>
      <strong>
       报告生成
      </strong>
      ：在学术研究或项目管理中，研究人员或项目经理可能需要将多个研究报告、数据分析或项目进展文档合并为一个完整的报告，以便于分享和展示。
      <br/>
      <strong>
       电子书制作
      </strong>
      ：作者或出版商在制作电子书时，可能需要将多个章节或部分合并为一个PDF文件，以便于读者下载和阅读。
      <br/>
      <strong>
       课程材料整合
      </strong>
      ：教育工作者可以将多个课程材料（如讲义、练习题、参考资料等）合并为一个PDF文件，方便学生获取和学习。
      <br/>
      <strong>
       法律文件整合
      </strong>
      ：律师在处理案件时，可能需要将多个法律文件（如证据、声明、合同等）合并为一个文件，以便于提交和审阅。
      <br/>
      <strong>
       会议资料准备
      </strong>
      ：在会议或研讨会中，组织者可以将多个演讲者的演示文稿或相关资料合并为一个PDF文件，方便与会者查看和下载。
      <br/>
      <strong>
       个人文档整理
      </strong>
      ：个人用户可以将多个PDF文件（如旅行计划、账单、收据等）合并为一个文件，以便于整理和查找。
      <br/>
      <strong>
       数据报告
      </strong>
      ：在数据分析和商业智能领域，分析师可能需要将多个数据报告合并为一个PDF，以便于向管理层或客户展示结果。
     </p>
    </blockquote>
    <hr/>
    <h4>
     <a id="_PyPDF2_15">
     </a>
     安装 PyPDF2
    </h4>
    <p>
     首先，确保你已经安装了
     <code>
      PyPDF2
     </code>
     库。如果还没有安装，可以使用以下命令进行安装：
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> PyPDF2
</code></pre>
    <hr/>
    <h4>
     <a id="_PDF__22">
     </a>
     合并 PDF 文件的代码示例
    </h4>
    <pre><code class="prism language-python"><span class="token keyword">from</span> PyPDF2 <span class="token keyword">import</span> PdfMerger

<span class="token comment"># 创建一个 PdfMerger 对象</span>
merger <span class="token operator">=</span> PdfMerger<span class="token punctuation">(</span><span class="token punctuation">)</span>

<span class="token comment"># 定义要合并的 PDF 文件列表</span>
pdf_files <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token string">'file1.pdf'</span><span class="token punctuation">,</span> <span class="token string">'file2.pdf'</span><span class="token punctuation">,</span> <span class="token string">'file3.pdf'</span><span class="token punctuation">]</span>  <span class="token comment"># 替换为你的 PDF 文件名</span>

<span class="token comment"># 遍历每个 PDF 文件并将其添加到合并对象中</span>
<span class="token keyword">for</span> pdf <span class="token keyword">in</span> pdf_files<span class="token punctuation">:</span>
    <span class="token comment"># 使用 append 方法添加 PDF 文件</span>
    merger<span class="token punctuation">.</span>append<span class="token punctuation">(</span>pdf<span class="token punctuation">)</span>

<span class="token comment"># 指定合并后的输出文件名</span>
output_file <span class="token operator">=</span> <span class="token string">'merged.pdf'</span>

<span class="token comment"># 将合并后的内容写入到新的 PDF 文件中</span>
merger<span class="token punctuation">.</span>write<span class="token punctuation">(</span>output_file<span class="token punctuation">)</span>

<span class="token comment"># 关闭合并对象</span>
merger<span class="token punctuation">.</span>close<span class="token punctuation">(</span><span class="token punctuation">)</span>

<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"成功合并 PDF 文件到 </span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>output_file<span class="token punctuation">}</span></span><span class="token string">"</span></span><span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_51">
     </a>
     代码说明
    </h4>
    <ol>
     <li>
      <strong>
       导入库
      </strong>
      ：首先导入
      <code>
       PdfMerger
      </code>
      类。
     </li>
     <li>
      <strong>
       创建合并对象
      </strong>
      ：使用
      <code>
       PdfMerger()
      </code>
      创建一个合并对象。
     </li>
     <li>
      <strong>
       定义文件列表
      </strong>
      ：创建一个包含要合并的 PDF 文件名的列表。请确保这些文件存在于当前工作目录中，或者提供完整路径。
     </li>
     <li>
      <strong>
       遍历文件
      </strong>
      ：使用
      <code>
       for
      </code>
      循环遍历文件列表，调用
      <code>
       merger.append(pdf)
      </code>
      方法将每个 PDF 文件添加到合并对象中。
     </li>
     <li>
      <strong>
       写入输出文件
      </strong>
      ：使用
      <code>
       merger.write(output_file)
      </code>
      将合并后的内容写入到指定的输出文件中。
     </li>
     <li>
      <strong>
       关闭合并对象
      </strong>
      ：调用
      <code>
       merger.close()
      </code>
      以释放资源。
     </li>
     <li>
      <strong>
       打印成功消息
      </strong>
      ：最后，打印一条消息以确认合并成功。
     </li>
    </ol>
    <h4>
     <a id="_61">
     </a>
     注意事项
    </h4>
    <ul>
     <li>
      确保在合并之前，所有的 PDF 文件都是有效的，并且没有损坏。
     </li>
     <li>
      如果需要合并的 PDF 文件较多，可以考虑使用动态生成文件列表的方式，例如从目录中读取文件名
     </li>
    </ul>
