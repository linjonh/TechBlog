    <br/>
    GIMP是GNU Image Manipulation Program（GNU图像处理程序）的缩写，它是Peter Mattis和Spencer Kimhall开发的免费开源照片和图像处理和创作工具，功能十分强大，完全可以与Windows平台下著名的图像处理软件Photoshop媲美。
    <p>
    </p>
    <h2>
     <a id="GIMP_2">
     </a>
     一、GIMP特点
    </h2>
    <ul>
     <li>
      开源、免费持续更新的正版软件，比起正版PS ￥:2975元，太有吸引力了。
     </li>
     <li>
      全套的绘图工具，包括笔刷（Brush）、铅笔（Pencil）、喷枪（AirBrush）等。
     </li>
     <li>
      平板式内存管理，能够处理的图像尺寸大小只受磁盘自由空间限制。
     </li>
     <li>
      支持各种常见图像格式，包括：GIF、JPG、PNG、XPM、TIFF、TGA、MPEG、PS、PDF、PCX、BMP等。
     </li>
     <li>
      过程数据库允许从外部程序调用内部的GIMP函数。
     </li>
     <li>
      无限次的Undo/Redo（仅受磁盘空间的限制）。
     </li>
     <li>
      支持旋转、缩放、裁剪及反转等变形工具。
     </li>
     <li>
      具有多种选择工具，如矩形、椭圆、自由、模糊、曲线及智能。
     </li>
     <li>
      插件功能允许用户任意增加新的文件格式及新的效果滤镜，现已提供了100多个插件。
     </li>
     <li>
      GIMP对Web图片设计也有不错的支持，同时还提供了一个屏幕抓图工具——屏幕快照
     </li>
    </ul>

     <a id="51__77">

     5.1 图片尺寸及容量的精确调整
    </h3>
    <ol>
     <li>
      打开图片，可以看到原始的像素数。图像 - 缩放图像 - 输入要求的像素数 - 点击缩放
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ada513a16f51b28c48121bb1adcafdc9.png"/>
     </li>
     <li>
      文件 - 导出为 - 输入文件名/目录 - 点击导出
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4bb5c49486d771e0d52992d8d8b8e43f.png"/>
     </li>
     <li>
      勾选预览 ，调整质量滑块，会显示最终文件大小，图像区会显示最终效果，非常直观。
      <br/>
      另外，GIMP的各级窗口都可以独立调整大小和位置，非常人性化，好于很多软件。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d34693803baa2f50c2fb821072f677a7.png"/>
     </li>
     <li>
      对比看一下，左边是原图2102K，右边是压缩的图片41K，容量是原图的1/51，质量依然Nice。
      <br/>
      注意：预览导出时提示29.2K，实际导出后41K，放大了1.4倍左右，所以就要心中有数，也可以多试几次。
      <br/>
