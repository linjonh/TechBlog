     </span>
     <span style="color:#000000;">
      网络
     </span>
     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">
      商品
     </span>
     <span style="color:#000000;">
      、搜索
     </span>
     <span style="color:#000000;">
      商品
     </span>
     <span style="color:#000000;">
      、
     </span>
     <span style="color:#000000;">
      查看商品并进行购买，对购买的商品进行确认收货、退款退货、
     </span>
     <span style="color:#000000;">
      查看个人信息、修改密码
     </span>
     <span style="color:#000000;">
      ，卖家注册、卖家登录，添加商品，对商品进行发货，
     </span>
     <span style="color:#000000;">
      管理员对用户、卖家、二手商品、通知公告、提醒用户、订单
     </span>
     <span style="color:#000000;">
      信息
     </span>
     <span style="color:#000000;">
      的管理等功能。网站界面进行了人性化的设计，操作简单具有一定的使用价值。本文主要从该网站的设计思想、功能的架构，运用的技术、数据库的设计、具体的实现等方面来进行论述。在详细设计部分主要论述了系统功能设计和数据库的设计。在本文的最后对网站的
     </span>
     <span style="color:#000000;">
      测试
     </span>
     <span style="color:#000000;">
      结果进行了说明和总结。

      随着 Internet 技术的发展，网络已经渗透到人们日常生活的方方面面，悄悄地改变着人们的生活方式。在国内，
     </span>
     <span style="color:#000000;">
      利用网站进行“
     </span>
     <span style="color:#000000;">
      二手商品
     </span>
     <span style="color:#000000;">
      ”的交易渐渐变成一种常见的
     </span>
     <span style="color:#000000;">
      交易方式。 比如我们常见的二手车、二手房交易市场
     </span>
     <span style="color:#000000;">
      、咸鱼
     </span>
     <span style="color:#000000;">
      等类似的网站
     </span>
     <span style="color:#000000;">
      近
     </span>
     <span style="color:#000000;">
      几年的发展趋势就非常的好
     </span>
     <span style="color:#000000;">
      。
     </span>
     <span style="color:#000000;">
      这类二手交易网站的发展的最大的优势在于
     </span>
     <span style="color:#000000;">
      信息传播快
     </span>
     <span style="color:#000000;">
      、时效性高
     </span>
     <span style="color:#000000;">
      。随着计算机和Internet技术的发展,
     </span>
     <span style="color:#000000;">
      校园二手交易平台
     </span>
     <span style="color:#000000;">
      给
     </span>
     <span style="color:#000000;">
      传统的
     </span>
     <span style="color:#000000;">
      二手商品交易
     </span>
     <span style="color:#000000;">
      提供了生机。
     </span>
     <span style="color:#000000;">
      但是
     </span>
     <span style="color:#000000;">
      ,消费观念、投资成本、技术人才、支付以及
     </span>
     <span style="color:#000000;">
      二手商品
     </span>
     <span style="color:#000000;">
      存在破损等因素阻碍了它的发展。
     </span>
     <span style="color:#000000;">
      所以
     </span>
     <span style="color:#000000;">
      解决这些问题是促进
     </span>
     <span style="color:#000000;">
      二手商品交易
     </span>
     <span style="color:#000000;">
      发展的首要任务。
     </span>
     <span style="color:#000000;">
      校园二手交易平台
     </span>
     <span style="color:#000000;">
      还是
     </span>
     <span style="color:#000000;">
      存在
     </span>
     <span style="color:#000000;">
      着
     </span>
     <span style="color:#000000;">
      巨大
     </span>
     <span style="color:#000000;">
      的
     </span>
     <span style="color:#000000;">
      潜力
     </span>
     <span style="color:#000000;">
      的
     </span>
     <span style="color:#000000;">
      ,
     </span>
     <span style="color:#000000;">
      大部分用户还是愿意去网上淘一些二手商品的，物美价廉
     </span>
     <span style="color:#000000;">
      。
     </span>
     <span style="color:#000000;">
      随着社会经济不断发展，用户的消费水平也在不断提高，大部分人都会在淘宝或者实体店等购买很多的商品，如：衣服、鞋子、玩具、书籍等等
     </span>
     <span style="color:#000000;">
      ，
     </span>
     <span style="color:#000000;">
      但很多商品买了之后，有些用了一两次或者有些还没使用，就一直摆放在家里，成了二手商品，不仅占用了家里的空间，还让商品没有得到合理的利用，这样还是比较浪费空间和金钱的。
     </span>
     <span style="color:#000000;">
      所以如果我们能够建立一个
     </span>
     <span style="color:#000000;">
      校园二手交易平台
     </span>
     <span style="color:#000000;">
      ，使得这些
     </span>
     <span style="color:#000000;">
      二手商品
     </span>

      能够以较便宜的价格卖给所需的
     </span>
     <span style="color:#000000;">
      用户
     </span>
     <span style="color:#000000;">
      ，这样无论是对不需要这些
     </span>
     <span style="color:#000000;">

     </span>

     </span>

     <span style="color:#000000;">

     </span>
     <span style="color:#000000;">
      校园二手交易平台
     </span>
     <span style="color:#000000;">
