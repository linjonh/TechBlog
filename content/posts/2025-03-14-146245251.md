    <blockquote>
     <h4>
      【题目描述】
     </h4>
     <p>
      给定一个非负整数数组，统计里面每一个数的出现次数。我们只统计到数组里最大的数。假设 Fmax（Fmax&lt;10000）是数组里最大的数，那么我们只统计{0,1,2.....Fmax}里每个数出现的次数。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      第一行n是数组的大小。1≤n≤10000。紧接着一行是数组的n个元素。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      按顺序输出每个数的出现次数，一行一个数。如果没有出现过，则输出0。对于例子中的数组，最大的数是3，因此我们只统计{0,1,2,3}的出现频数。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      5
      <br/>
      1 1 2 3 1
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      0
      <br/>
      3
      <br/>
      1
      <br/>
      1
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e4 + 10;
int nums[N];

int main()
{
	int n; cin &gt;&gt; n;
	int max_v = nums[1], cnt = 0;
	for (int i = 1; i &lt;= n; i++)
	{
		cin &gt;&gt; nums[i];
		if (nums[i] &gt; max_v) max_v = nums[i];
	}
	for (int i = 0; i &lt;= max_v; i++)
	{
		for (int j = 1; j &lt;= n; j++)
		{
			if (nums[j] == i) cnt++;
		}
		cout &lt;&lt; cnt &lt;&lt; endl;
		cnt = 0;
	}

	return 0;
}</code></pre>
