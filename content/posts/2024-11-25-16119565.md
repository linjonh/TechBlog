      <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
       <span class="sentence" id="mt9">
        链接服务器让用户可以对 OLE DB 数据源进行分布式异类查询。
       </span>
       <span class="sentence" id="mt10">
        在创建某一链接服务器后，可对该服务器运行分布式查询，并且查询可以联接来自多个数据源的表。
       </span>
       <span class="sentence" id="mt11">
        如果链接服务器定义为 SQL Server 实例，则可执行远程存储过程。
       </span>
      </p>
      <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
       <span class="sentence" id="mt12">
        链接服务器的功能和必需的参数可能会有很大差异。
       </span>
       <span class="sentence" id="mt13">
        本主题中的示例是典型示例，但并未描述所有选项。
       </span>

      <h4 class="subHeading" style="color:rgb(42,42,42); font-family:'Segoe UI','Lucida Grande',Verdana,Arial,Helvetica,sans-serif; font-size:1.386em; font-weight:normal; margin:0px; padding-bottom:5px; padding-top:5px">
       使用 SQL Server Management Studio
      </h4>
      <div class="subsection">
       <h4 class="procedureSubHeading" style="color:rgb(42,42,42); font-family:'Segoe UI','Lucida Grande',Verdana,Arial,Helvetica,sans-serif; font-size:1.386em; font-weight:normal; margin:0px; padding-bottom:5px; padding-top:5px">
        使用 SQL Server Management Studio 创建与其他 SQL Server 实例的链接服务器
       </h4>
       <div class="subSection">
        <ol>
         <li style="color:rgb(42,42,42)">
          <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
           <span class="sentence" id="mt19">
            在 SQL Server Management Studio 中，打开对象资源管理器，展开
            <span class="label" style="font-weight:700">
             “服务器对象”
            </span>
            ，右键单击
            <span class="label" style="font-weight:700">
             “链接服务器”
            </span>
            ，然后单击
            <span class="label" style="font-weight:700">
             “新建链接服务器”
            </span>
            。
           </span>
          </p>
         </li>
         <li style="color:rgb(42,42,42)">
          <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
           <span class="sentence" id="mt20">
            在
            <span class="label" style="font-weight:700">
             “常规”
            </span>
            页上的
            <span class="label" style="font-weight:700">
             “链接服务器”
            </span>
            框中，键入您链接到的
            <span class="label" style="font-weight:700">
             SQL Server
            </span>
            实例的名称。
           </span>
          </p>
          <dl class="authored">
           <span class="label" style="font-weight:700">
            SQL Server
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt21">
              将链接服务器标识为 Microsoft SQL Server 的实例。
             </span>
             <span class="sentence" id="mt22">
              如果您使用此方法来定义某个 SQL Server 链接服务器，则在
              <span class="label" style="font-weight:700">
               “链接服务器”
              </span>
              中指定的名称必须是该服务器的网络名称。
             </span>
             <span class="sentence" id="mt23">
              另外，从该服务器上检索的所有表都来自该链接服务器上为相应登录名所定义的默认数据库。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            其他数据源
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt24">
              指定 SQL Server 以外的 OLE DB 服务器类型。
             </span>
             <span class="sentence" id="mt25">
              单击此选项将激活其下面的选项。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            提供程序
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt26">
              从列表框中选择 OLE DB 数据源。
             </span>
             <span class="sentence" id="mt27">
              OLE DB 访问接口是使用注册表中给定的 PROGID 注册的。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            产品名称
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt28">
              键入要作为链接服务器添加的 OLE DB 数据源的产品名称。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            数据源
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt29">
              根据 OLE DB 访问接口的说明，键入数据源名称。
             </span>
             <span class="sentence" id="mt30">
              如果要连接到 SQL Server 的实例，请提供实例名称。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            访问接口字符串
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt31">
              键入与数据源相对应的 OLE DB 访问接口的唯一编程标识符 (PROGID)。
             </span>
             <span class="sentence" id="mt32">
              有关有效访问接口字符串的示例，请参阅
              <a href="http://technet.microsoft.com/zh-CN/library/ms190479" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(0,102,221)" target="_blank">
               sp_addlinkedserver (Transact-SQL)
              </a>
              。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            位置
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt33">
              根据 OLE DB 访问接口的说明，键入数据库的位置。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            目录
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt34">
              键入在连接 OLE DB 访问接口时要使用的目录的名称。
             </span>
            </p>
           </dd>
          </dl>
          <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
           <span class="sentence" id="mt35">
            若要测试能否连接到链接服务器，请在对象资源管理器中，右键单击链接服务器，然后单击
            <span class="label" style="font-weight:700">
             “测试连接”
            </span>
            。
           </span>
          </p>

        查看提供程序选项
       </h4>
       <div class="subSection" style="color:rgb(42,42,42)">
        <ul>
         <li style="">
          <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
           <span class="sentence" id="mt88">
            若要查看提供程序提供的选项，请单击
            <span class="label" style="font-weight:700">
             “提供程序选项”
            </span>
            页。
           </span>
          </p>
          <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
           <span class="sentence" id="mt89">
            每个访问接口的选项都各不相同。
           </span>
           <span class="sentence" id="mt90">
            例如，某些类型的数据提供索引，有些则没有提供。
           </span>
           <span class="sentence" id="mt91">
            使用此对话框可以帮助 SQL Server 理解访问接口的功能。
           </span>
           <span class="sentence" id="mt92">
            SQL Server 安装某些常用的数据访问接口，但在提供数据的产品发生更改时，SQL Server 安装的访问接口可能不支持所有最新的功能。
           </span>
           <span class="sentence" id="mt93">
            提供数据的产品功能的有关信息的最佳来源是针对该产品的文档。
           </span>
          </p>
          <dl class="authored">
           <span class="label" style="font-weight:700">
            动态参数
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt94">
              表明访问接口允许对参数化查询使用“?”参数标记语法。
             </span>
             <span class="sentence" id="mt95">
              仅当该访问接口支持
              <strong>
               ICommandWithParameters
              </strong>
              接口并支持“?”作为参数标记时，才应设置此选项。
             </span>
             <span class="sentence" id="mt96">
              如果设置了此选项，则允许 SQL Server 针对该访问接口执行参数化查询。
             </span>
             <span class="sentence" id="mt97">
              这种对访问接口执行参数化查询的能力会提高某些查询的性能。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            嵌套查询
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt98">
              指示访问接口允许在 FROM 子句中使用嵌套的SELECT 语句。
             </span>
             <span class="sentence" id="mt99">
              如果设置了此选项，则允许 SQL Server 将某些查询委托给需要在 FROM 子句中嵌套 SELECT 语句的访问接口。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            仅零级
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt100">
              只对访问接口调用 0 级的 OLE DB 接口。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            允许进程内
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt101">
              SQL Server 允许将访问接口实例化为进程内服务器。
             </span>
             <span class="sentence" id="mt102">
              如果未设置此选项，则默认行为是在 SQL Server 进程外实例化访问接口。
             </span>
             <span class="sentence" id="mt103">
              在 SQL Server 进程外实例化访问接口，可防止 SQL Server 进程在访问接口中出错。
             </span>
             <span class="sentence" id="mt104">
              在 SQL Server 进程外实例化访问接口时，不允许更新或插入长的引用列（
              <span class="input" style="font-weight:700">
               text
              </span>
              、
              <span class="input" style="font-weight:700">
               ntext
              </span>
              或
              <span class="input" style="font-weight:700">
               image
              </span>
              ）。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            非事务更新
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt105">
              SQL Server 允许更新，即使
              <strong>
               ITransactionLocal
              </strong>
              不可用时也是如此。
             </span>
             <span class="sentence" id="mt106">
              如果启用此选项，对访问接口的更新将不可恢复，因为该访问接口不支持事务。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            作为访问路径的索引
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt107">
              SQL Server 尝试使用访问接口的索引来提取数据。
             </span>
             <span class="sentence" id="mt108">
              默认情况下，索引只能用于元数据而且从不打开。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            禁止即席访问
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt109">
              SQL Server 不允许通过 OPENROWSET 和 OPENDATASOURCE 函数对 OLE DB 访问接口进行即席访问。
             </span>
             <span class="sentence" id="mt110">
              如果未设置此选项，则 SQL Server 同样不允许进行即席访问。
             </span>
            </p>
           </dd>
           <span class="label" style="font-weight:700">
            支持 "Like" 运算符
           </span>
           <dd>
            <p style="margin-top:0px; margin-bottom:0px; padding-bottom:15px; line-height:18px">
             <span class="sentence" id="mt111">
              指示访问接口支持使用 LIKE 关键字的查询。
             </span>
            </p>
           </dd>
          </dl>
         </li>
        </ul>
