# 天梯赛-前世档案 二进制的巧妙使用

看到题目的这个图片，我们就想到了二进制表示，用y当0，n当1，计算出相应的二进制后还要+1才是答案,
~~这个图片神似哈夫曼编码树~~

![](https://i-blog.csdnimg.cn/direct/1a557c9bd814443ba69c1a0afe655b79.png)

注意因为是从高位向低位读的，所以要ans += pow(2,n - 1 - i);

```cpp

int n,m;
string s;
void solve()
{
	cin >> n >> m;
	for (int i = 1;i <= m;i ++)
	{
		cin >> s;
		LL ans = 0;
		for (int i = 0;i < n;i ++)
			if (s[i] == 'n') ans += pow(2,n - 1 - i);
		
		cout << ans + 1 << endl;
	}
}
```
