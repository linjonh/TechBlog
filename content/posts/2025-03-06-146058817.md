    <h2>
     一、ReactRouter-路由导航
    </h2>
    <h3>
     1.1、什么是路由导航
    </h3>
    <p>
     路由系统中的多个路由之间需要进行路由跳转，并且在跳转的同时有可能需要传递参数进行通信，同Vue中的路由导航
    </p>
    <p>
     以一个case为例：点击button跳转至文章页
    </p>
    <p class="img-center">
     <img alt="" height="196" src="https://i-blog.csdnimg.cn/direct/471ed60c7a8340f2b92f18d3794902de.png" width="514"/>
    </p>
    <h3>
     1.2、声明式导航
    </h3>
    <p>
     声明式导航是指通过在模板中
     <span style="color:#ed7976">
      通过‘&lt;Link/&gt;’组件描述出要跳转到哪里
     </span>
     去，例如后台管理系统的左侧菜单常用这种方式进行
    </p>
    <pre><code class="language-html">&lt;Link to="/article"&gt;文章&lt;/Link&gt;</code></pre>
    <p>
     语法说明：通过给组件的
     <span style="color:#ed7976">
      to属性指定要跳转到路由path
     </span>
     ，组件会被渲染为浏览器支持的a链接，如果需要传参直接
     <span style="color:#ed7976">
      通过字符串拼接
     </span>
     的方式拼接参数即可
    </p>
    <pre><code class="language-javascript">import { Link } from "react-router-dom"

const Login=()=&gt;{
    return (
    &lt;div&gt;
        登录页
        &lt;Link to="/article"&gt;跳转到文章页&lt;/Link&gt;
    &lt;/div&gt;
    )
}

export default Login</code></pre>
    <p style="text-align:center">
     Login&gt;index.js
    </p>
    <p>
     浏览器解析后的结果如图
    </p>
    <p class="img-center">
     <img alt="" height="17" src="https://i-blog.csdnimg.cn/direct/25e694d2f1cc495ca5f93277446c71d0.png" width="438"/>
    </p>
    <h3>
     1.3、编程式导航
    </h3>
    <p>
     通过‘useNavigate’钩子得到导航方法，然后通过调用方法以命令式的形式进行路由跳转，这种方式更为灵活
    </p>
    <pre><code class="language-javascript">import { Link, useNavigate } from "react-router-dom"

const Login = () =&gt; {
    const Navigate = useNavigate()
    return (
        &lt;div&gt;
            登录页
            &lt;Link to="/article"&gt;跳转到文章页&lt;/Link&gt;
            &lt;button onClick={() =&gt; Navigate('/article')}&gt;&lt;/button&gt;
        &lt;/div&gt;
    )
}

export default Login</code></pre>
    <p style="text-align:center">
     Login&gt;index.js
    </p>
    <p>
     编程式导航（命令式）的写法与Vue的Router除了命名之外基本一样，这里放出Vue作为对比
    </p>
    <pre><code class="language-javascript">import { RouterLink, RouterView, useRouter } from 'vue-router'

const router = useRouter()

const jump=()=&gt;{
    router.push('/article')
}</code></pre>
    <p style="text-align:center">
     vue&gt;router&gt;index.js
    </p>
    <h2>
     二、ReactRouter-导航路由传参
    </h2>
    <h3>
     2.1、searchParams传参
    </h3>
    <p>
     传参：
    </p>
    <pre><code class="language-javascript">import { useNavigate } from "react-router-dom"

const Login = () =&gt; {
    const Navigate = useNavigate()
    return (
        &lt;div&gt;
            登录页
            &lt;button onClick={() =&gt; Navigate('/article?id=1001&amp;name=jim')}&gt;传参&lt;/button&gt;
        &lt;/div&gt;
    )
}

export default Login</code></pre>
    <p style="text-align:center">
     Login&gt;index.js
    </p>
    <p>
     接收：
    </p>
    <pre><code class="language-javascript">import { useSearchParams } from "react-router-dom"

const Article = () =&gt; {
    const [params] = useSearchParams()
    const id = params.get(id)
    return &lt;div&gt;文章页{id}&lt;/div&gt;
}

export default Article</code></pre>
    <p style="text-align:center">
     Article&gt;index.js
    </p>
    <h3>
     2.2、params传参
    </h3>
    <p>
     <span style="color:#ed7976">
      定义传的参数名（重点）：
     </span>
    </p>
    <pre><code class="language-javascript">import Login from '../page/Login'
import Article from '../page/Article'
import { createBrowserRouter } from 'react-router-dom'

const router=createBrowserRouter([
    {
        path:'/login',
        element:&lt;Login&gt;&lt;/Login&gt;
    },
    {
        path:'/article/:id',
        element:&lt;Article&gt;&lt;/Article&gt;
    },
])

export default router</code></pre>
    <p style="text-align:center">
     router&gt;index.js
    </p>
    <p>
     传参：
    </p>
    <pre><code class="language-javascript">import { useNavigate } from "react-router-dom"

const Login = () =&gt; {
    const Navigate = useNavigate()
    return (
        &lt;div&gt;
            登录页
            &lt;button onClick={() =&gt; Navigate('/article/1001')}&gt;传参&lt;/button&gt;
        &lt;/div&gt;
    )
}

export default Login</code></pre>
    <p style="text-align:center">
     Login&gt;index.js
    </p>
    <p>
     接收参数：
    </p>
    <pre><code class="language-javascript">import { useParams } from "react-router-dom"

const Article = () =&gt; {
    const params = useParams()
    return &lt;div&gt;文章页{params.id}&lt;/div&gt;
}

export default Article</code></pre>
    <p style="text-align:center">
     Article&gt;index.js
    </p>
    <p>
     这和Vue中的传参几乎一样，记忆方式可以直接参考Vue路由传参
    </p>
