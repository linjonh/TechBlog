    <h3>
     题目
    </h3>
    <p>
     给你一个有序数组
     <code>
      nums
     </code>
     ，请你
     <strong>
      <strong>
       <a href="http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95" rel="nofollow" title=" 原地">
        原地
       </a>
      </strong>
     </strong>
     删除重复出现的元素，使得出现次数超过两次的元素
     <strong>
      只出现两次
     </strong>
     ，返回删除后数组的新长度。
    </p>
    <p>
     不要使用额外的数组空间，你必须在
     <strong>
      <strong>
       <a href="https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95" rel="nofollow" title="原地 ">
        原地
       </a>
      </strong>
      修改输入数组
     </strong>
     并在使用 O(1) 额外空间的条件下完成。
    </p>
    <p>
    </p>
    <p>
     <strong>
      说明：
     </strong>
    </p>
    <p>
     为什么返回数值是整数，但输出的答案是数组呢？
    </p>
    <p>
     请注意，输入数组是以
     <strong>
      「引用」
     </strong>
     方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。
    </p>
    <p>
     你可以想象内部操作如下:
    </p>
    <pre>// <strong>nums</strong> 是以“引用”方式传递的。也就是说，不对实参做任何拷贝
int len = removeDuplicates(nums);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中<strong> 该长度范围内</strong> 的所有元素。
for (int i = 0; i &lt; len; i++) {
    print(nums[i]);
}
</pre>
    <p>
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <pre><strong>输入：</strong>nums = [1,1,1,2,2,3]
<strong>输出：</strong>5, nums = [1,1,2,2,3]
<strong>解释：</strong>函数应返回新长度 length = 5, 并且原数组的前五个元素被修改为 1, 1, 2, 2, 3。 不需要考虑数组中超出新长度后面的元素。
</pre>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <pre><strong>输入：</strong>nums = [0,0,1,1,1,1,2,3,3]
<strong>输出：</strong>7, nums = [0,0,1,1,2,3,3]
<strong>解释：</strong>函数应返回新长度 length = 7, 并且原数组的前七个元素被修改为 0, 0, 1, 1, 2, 3, 3。不需要考虑数组中超出新长度后面的元素。
</pre>
    <p>
    </p>
    <p>
     <strong>
      提示：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       1 &lt;= nums.length &lt;= 3 * 104
      </code>
     </li>
     <li>
      <code>
       -104 &lt;= nums[i] &lt;= 104
      </code>
     </li>
     <li>
      <p>
       <code>
        nums
       </code>
       已按升序排列
      </p>
     </li>
    </ul>
    <h3>
     题解
    </h3>
    <pre><code class="language-python">class Solution(object):
    def removeDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if len(nums)&lt;=2:
            return len(nums)
        p1=2
        for i in range(2,len(nums)):
            if nums[i]!=nums[p1-2]:
                nums[p1]=nums[i]
                p1+=1
        return p1</code></pre>
    <h3>
     代码说明
    </h3>
    <p>
     <strong>
      1.边界检查
     </strong>
     ：
    </p>
    <p>
     如果数组的长度小于等于
     <code>
      2
     </code>
     ，直接返回数组的长度。
    </p>
    <p>
     <strong>
      2.初始化指针
     </strong>
     ：
    </p>
    <p>
     p1 指针初始化为
     <code>
      2
     </code>
     ，表示前两个元素一定是满足条件的。
    </p>
    <p>
     <strong>
      3.遍历数组
     </strong>
     ：
    </p>
    <p>
     <code>
      i
     </code>
     指针从
     <code>
      2
     </code>
     开始遍历数组。
    </p>
    <p>
     如果
     <code>
      nums[i] != nums[
     </code>
     p1
     <code>
      - 2]
     </code>
     <span style="color:#4d4d4d">
      ，说明
     </span>
     <code>
      nums[i]
     </code>
     <span style="color:#4d4d4d">
      是一个新的元素，或者该元素还没有出现两次，将其复制到
     </span>
     <code>
      nums[
     </code>
     p1
     <code>
      ]
     </code>
     <span style="color:#4d4d4d">
      ，然后
     </span>
     p1
     <span style="color:#4d4d4d">
      指针向前移动。
     </span>
    </p>
    <p>
     如果
     <code>
      nums[i] == nums[
     </code>
     p1
     <code>
      - 2]
     </code>
     <span style="color:#4d4d4d">
      ，说明该元素已经出现了两次，跳过。
     </span>
    </p>
    <p>
     <span style="color:#4d4d4d">
      4.
     </span>
     <strong>
      返回结果
     </strong>
     ：
    </p>
    <p>
     p1 就是新数组的长度。
    </p>
