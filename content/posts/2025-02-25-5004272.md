    <h1 class="title-article" id="articleContentId">
     微软Windows操作系统发展史
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <div>
     <strong>
      <span style="font-size: medium;">
       Windows发展史
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy055.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      古人云：名满天下，谤亦随之。微软Windows操作系统获得巨大成功的同时，批判声总不绝于耳。下面，笔者将和大家一道回顾Windows的发展历程。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       MS-DOS 1.0
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy056.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1981年8月，IBM公司推出了运行微软16位操作系统MS-DOS 1.0的个人电脑，这款系统的发明人正是比尔·盖茨。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows 3.0
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy057.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1990年5月22日，微软正式发布具备图形用户界面、支持VGA标准及配置与目前Windows 系统相似3D功能的Windows 3.0。该操作系统还拥有非常出色的文件和内存管理功能。Windows 3.0因此成为微软历史上首款成功的操作系统。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows NT 3.1
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy058.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1993年10月24日，微软正式向局域网服务器市场推出Windows NT Advanced Server 3.1。同时，微软还推出了Win32 API（windows所提供的应用程序接口）。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      比尔·盖茨对Windows NT的它评价很高：它从根本上解决了企业用户对计算机运算的需求。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows for Workgroups 3.11
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy059.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      Windows for Workgroups 3.11集成了一组对等网络服务和网络应用功能。它的到来让基于Windows系统的PC，第一次与其它的windows系统和软件连接起来。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows 95
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy060.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1995年8月，Windows 95伴随着滚石乐队的《Start Me Up》强势登陆，它彻底地取代了3.1版和DOS版Windows。Windows 95新的桌面、任务栏及开始菜单依然存在于今天的Windows系统中。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      在市场上，Windows 95绝对是有史以来最成功的操作系统。据传，当时很多没有电脑的顾客受到宣传的影响而排队购买软件，但他们甚至根本不知道Windows 95是什么。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows NT 4.0
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy061.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1996年7月，微软推出了Windows NT 4.0。Windows NT 4.0一共有四个版本：工作站版（Workstation）、终端服务器版及两个服务器版。它首次加入了Internet Explorer浏览器，并与通信服务紧密集成，提供文件和打印服务，能运行客户机／服务器应用程序，内置了Internet／Intranet功能。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows CE 1.0
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy062.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1996年11月，微软推出32位嵌入式操作系统Windows CE 1.0。其中CE中的C代表袖珍（Compact）、消费（Consumer）、通信能力（Connectivit)和伴侣（Companion）；E代表电子产品（Electronics）。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      Windows CE 1.0是一款基于Windows95的操作系统，它是微软公司嵌入式、移动计算平台的基础。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows 98
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy063.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      1998年6月，Windows 98正式发布。人们普遍认为，Windows 98并非一款新的操作系统，它只是提高了Windows 95的稳定性。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      值得一提的是，Windows 98 捆绑Internet Explorer 浏览器到Windows GUI和Explorer的做法，成为了美国政府对微软公司反垄断诉讼的导火索。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      Windows 98 SE（第二版）发行于1999年5月5日。它包括了一系列的改进，例如Internet Explorer 5、Internet Connection Sharing、NetMeeting 3.0和DirectX API 6.1。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows 2000
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy064.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      2000年2月，Windows 2000发布。Windows 2000包括一个用户版和一个服务器版。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      Windows 2000是一个preemptive、可中断的、图形化的及面向商业环境的操作系统，为单一处理器或对称多处理器的32位Intel x86电脑而设计。Windows 2000最为重要的功能是Active Directory（Windows 2000网络中的目录服务）。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
     </span>
    </div>
    <div>
     <strong>
      <span style="font-size: medium;">
       Windows ME
      </span>
     </strong>
    </div>
    <p align="center">
     <span style="font-size: medium;">
      <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy065.jpg"/>
     </span>
    </p>
    <div>
     <span style="font-size: medium;">
      2000年12月，被公认为微软最为失败的操作系统Windows ME（Millennium Edition）发布。相对其它Windows系统，短暂的Windows ME只延续了1年，即被Windows XP取代。
     </span>
    </div>
    <div>
     <span style="font-size: medium;">
      Windows Me是最后一个基于实时DOS的Windows 9X系统，其版本号为4.9。其名字有2个意思，一是纪念2000年，Me是英文中千禧年（Millennium）的意思。另外也是指自己，Me在英文中是我的意思。
     </span>
    </div>
    <div>
     <div>
      <strong>
       <span style="font-size: medium;">
        Windows XP
       </span>
      </strong>
     </div>
     <p align="center">
      <span style="font-size: medium;">
       <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy066.jpg"/>
      </span>
     </p>
     <div>
      <span style="font-size: medium;">
       2001年10月25日，微软公司当时的副总裁Jim Allchin首次展示了Windows XP。微软最初发行了两个版本：专业版(Windows XP Professional)和家庭版(Windows XP Home Edition)。家庭版的消费对象是家庭用户，专业版则在家庭版的基础上添加了新的为面向商业的设计的网络认证、双处理器等特性。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
       Windows XP原来的代号是Whistler。字母XP表示英文单词的“体验”（experience）。Windows XP的外部版本是2002，内部版本是5.1（即Windows NT 5.1），正式版的Build是5.1.2600。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
       Windows XP一经推出，便大获成功。著名的市场调研机构Forrester统计的数据显示，Windows XP发布7年后的2009年2月份，Windows XP仍占据71%的企业用户市场。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
      </span>
     </div>
     <div>
      <strong>
       <span style="font-size: medium;">
        Windows Server 2003
       </span>
      </strong>
     </div>
     <p align="center">
      <span style="font-size: medium;">
       <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy067.jpg"/>
      </span>
     </p>
     <div>
      <span style="font-size: medium;">
       2003年4月24日，微软正式发布服务器操作系统Windows Server 2003。它增加了新的安全和配置功能。Windows Server 2003有多种版本，包括Web版、标准版、企业版及数据中心版。Windows Server 2003 R2于2005年12月发布。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
      </span>
     </div>
     <div>
      <strong>
       <span style="font-size: medium;">
        Windows Vista
       </span>
      </strong>
     </div>
     <p align="center">
      <span style="font-size: medium;">
       <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy068.jpg"/>
      </span>
     </p>
     <div>
      <span style="font-size: medium;">
       2006年11月30日，Windows Vista开发完成并正式进入批量生产。此后的两个月仅向MSDN用户、电脑软硬件制造商和企业客户提供。在2007年1月30日，Windows Vista正式对普通用户出售。此后便爆出该系统兼容性存在很大的问题。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
       微软CEO史蒂芬.鲍尔默也公开承认，Vista是一款失败的操作系统产品。而即将到来的Windows 7，预示着Vista的寿命将被缩短。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
      </span>
     </div>
     <div>
      <strong>
       <span style="font-size: medium;">
        Windows Server 2008
       </span>
      </strong>
     </div>
     <p align="center">
      <span style="font-size: medium;">
       <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy069.jpg"/>
      </span>
     </p>
     <div>
      <span style="font-size: medium;">
       2008年2月27日，微软发布新一代服务器操作系统Windows Server 2008。Windows Server 2008 是迄今为止最灵活、最稳定的 Windows Server 操作系统，它加入了包括Server Core、PowerShell和Windows Deployment Services等新功能，并加强了网络和群集技术。Windows Server 2008 R2版也于2009年1月份进入Beta测试阶段。
      </span>
     </div>
     <div>
      <span style="font-size: medium;">
      </span>
     </div>
     <div>
      <strong>
       <span style="font-size: medium;">
        Windows 7
       </span>
      </strong>
     </div>
     <p align="center">
      <span style="font-size: medium;">
       <img alt="" src="http://image.onlinedown.net/2009/huajunpc/01/20090210zy070.jpg"/>
      </span>
     </p>
     <div>
      <span style="font-size: medium;">
       下一代Windows操作系统Windows 7即将结束Beta测试阶段，迎来RC（候选发行）版。Windows 7的正式发布时间很可能在今年下半年。
      </span>
     </div>
    </div>
