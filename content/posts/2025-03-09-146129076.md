    <blockquote>
     <p>
      Tailwind CSS 是一个功能强大实用优先的 CSS 原子性的库，可以帮助你快速构建现代化的用户界面。
     </p>
     <p>
      它支持的种类框架也非常的多！ 例如：
      <code>
       Vite
      </code>
      、
      <code>
       Next.js
      </code>
      等…
     </p>
     <p>
      这个库给我们带来的好处就是可以给我们衍生出其他增强的体验！
     </p>
     <p>
      比如说
      <code>
       shadcn/ui
      </code>
      、
      <code>
       daisyUI
      </code>
      无样式 UI 库！这些 UI 库就是基于
      <code>
       Tailwind CSS
      </code>
      适配开发的！
     </p>
     <p>
      参考：
      <a href="https://www.tailwindcss.cn/docs/guides/vite" rel="nofollow">
       tailwindcss 官网
      </a>
     </p>
     <p>
      以下是安装和使用 Tailwind CSS 的步骤：
     </p>
    </blockquote>
    <h3>
     <a id="1_React__12">
     </a>
     1.创建一个 React 项目
    </h3>
    <pre><code class="prism language-bash"><span class="token function">npm</span> create vite@latest my-project -- <span class="token parameter variable">--template</span> react

<span class="token builtin class-name">cd</span> my-project
</code></pre>
    <h3>
     <a id="2_Tailwind_CSS_20">
     </a>
     2.安装 Tailwind CSS
    </h3>
    <p>
     2.1 首先，你需要在项目中安装
     <code>
      tailwindcss
     </code>
     及其对等依赖 !
    </p>
    <pre><code class="prism language-bash">// <span class="token number">1</span>.安装 Tailwind CSS
<span class="token function">npm</span> <span class="token function">install</span> <span class="token parameter variable">-D</span> tailwindcss@3.4.17

// <span class="token number">2</span>.安装对等依赖 
<span class="token function">npm</span> <span class="token function">install</span> <span class="token parameter variable">-D</span> postcss autoprefixer
</code></pre>
    <blockquote>
     <p>
      <strong>
       ps： 安装时我们要指定版本安装 与官网文档版本保持一致即可！ 如果不指定版本、默认安装是就是
       <code>
        v4
       </code>
       版本的
       <code>
        tailwindcss
       </code>
       。
      </strong>
     </p>
     <p>
      当你继续执行
      <code>
       npx tailwindcss init
      </code>
      就会报错， 没有可执行文件的信息！
     </p>
     <pre><code class="prism language-bash">PS D:<span class="token punctuation">\</span>lx<span class="token punctuation">\</span>project<span class="token punctuation">\</span>emoji-picker<span class="token operator">&gt;</span> npx tailwindcss init
<span class="token function">npm</span> ERR<span class="token operator">!</span> could not determine executable to run

<span class="token function">npm</span> ERR<span class="token operator">!</span> A complete log of this run can be found in: C:<span class="token punctuation">\</span>Users<span class="token punctuation">\</span>AppData<span class="token punctuation">\</span>Local<span class="token punctuation">\</span>npm-cache<span class="token punctuation">\</span>_logs<span class="token punctuation">\</span><span class="token number">2025</span>-03-08T04_37_23_510Z-debug-0.log
</code></pre>
     <p>
      解决方式： 我们安装切换官网的版本下载安装即可解决。
     </p>
     <pre><code class="prism language-bash"><span class="token function">npm</span> <span class="token function">install</span> <span class="token parameter variable">-D</span> tailwindcss@3.4.17
</code></pre>
    </blockquote>
    <h4>
     <a id="3__Tailwind_CSS__51">
     </a>
     3. 初始化 Tailwind CSS 配置文件
    </h4>
    <p>
     3.1 安装完成后，你可以通过以下命令生成
     <code>
      tailwind.config.js
     </code>
     和
     <code>
      postcss.config.js
     </code>
     配置文件：
    </p>
    <blockquote>
     <p>
      因为要分析我们的代码， 把
      <code>
       Tailwind CSS
      </code>
      的代码指令， 编译成浏览器可以识别的 CSS。
     </p>
    </blockquote>
    <pre><code class="prism language-bash">npx tailwindcss init <span class="token parameter variable">-p</span>
</code></pre>
    <p>
     3.2 将这些路径添加到
     <code>
      tailwind.config.js
     </code>
     文件中的所有模板文件中。
    </p>
    <blockquote>
     <p>
      tailwind.config.js 文件允许你自定义 Tailwind 的默认配置，例如颜色、字体、间距等。
     </p>
    </blockquote>
    <pre><code class="prism language-js"><span class="token comment">/** @type {import('tailwindcss').Config} */</span>
<span class="token keyword">export</span> <span class="token keyword">default</span> <span class="token punctuation">{<!-- --></span>
 <span class="token comment">// 指定项目目录下那些文件需要处理！ </span>
  <span class="token literal-property property">content</span><span class="token operator">:</span> <span class="token punctuation">[</span>
    <span class="token string">"./index.html"</span><span class="token punctuation">,</span>
    <span class="token string">"./src/**/*.{js,ts,jsx,tsx}"</span><span class="token punctuation">,</span>
  <span class="token punctuation">]</span><span class="token punctuation">,</span>
  <span class="token literal-property property">theme</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token literal-property property">extend</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span><span class="token punctuation">,</span>
  <span class="token punctuation">}</span><span class="token punctuation">,</span>
  <span class="token literal-property property">plugins</span><span class="token operator">:</span> <span class="token punctuation">[</span><span class="token punctuation">]</span><span class="token punctuation">,</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="4__Tailwind_CSS__79">
     </a>
     4. 创建 Tailwind CSS 入口文件
    </h4>
    <p>
     在你的项目中入口 CSS 文中引入 Tailwind 的基础样式、组件和工具类：
     <br/>
     <strong>
      例如：
      <code>
       src/index.css
      </code>
     </strong>
    </p>
    <pre><code class="prism language-css"><span class="token atrule"><span class="token rule">@tailwind</span> base<span class="token punctuation">;</span></span>
<span class="token atrule"><span class="token rule">@tailwind</span> components<span class="token punctuation">;</span></span>
<span class="token atrule"><span class="token rule">@tailwind</span> utilities<span class="token punctuation">;</span></span>
</code></pre>
    <blockquote>
     <p>
      ps： 且必须在你的项目入口 main.js 文件中将
      <code>
       index.css
      </code>
      文件引入到全局。
     </p>
    </blockquote>
    <h3>
     <a id="5__Tailwind_CSS_91">
     </a>
     5. 在你的页面代码中使用 Tailwind CSS
    </h3>
    <pre><code class="prism language-tsx">function App() {
  const [count, setCount] = useState(0)

  return (
    &lt;&gt;
      &lt;button className='text-pink-400 px-2 py-2 bg-slate-600 rounded'&gt; Hello World &lt;/button&gt;
    &lt;/&gt;
  )
}

export default App
</code></pre>
