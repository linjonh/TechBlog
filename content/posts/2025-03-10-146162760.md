    <p>
     一、mysql下载：
     <br/>
     MySQL获取：
     <br/>
     官网：www.mysql.com
    </p>
    <p>
     也可以从Oracle官方进入：https://www.oracle.com/
    </p>
    <p>
     下载地址：https://downloads.mysql.com/archives/community/
    </p>

     选择对应的版本和对应的操作系统：
    </p>
    <p>
     二、MySQL的安装与配置：
     <br/>
     window操作系统的安装：
     <br/>
     压缩包方式：
     <br/>
     下载位置：mysql下载位置
    </p>
    <p>
     解压缩后位置：D:\Program Files\MySQL\MySQL Server 8.4
    </p>
    <p>
     在主目录下（D:\Program Files\MySQL\MySQL Server 8.4）复制、创建一个xx.ini，修改为my.ini（内容在MySQL的配置文件中），修改其中的basedir和datadir
    </p>
    <p>
     basedir=mysql主目录 ​ datadir=mysql主目录/data
    </p>

      <p>
       mysqld --initialize --console

      <p>
       然后先给mysql服务创建名称(方便到时候建立多个mysql服务时不冲突)
      </p>
      <p>
       mysqld --install mysql8
       <br/>
       启动服务：
      </p>
      <p>
       net start mysql
       <br/>
       登陆旧密码登陆（第3步中的密码）：
      </p>
      <p>
       mysql -u root -p
       <br/>
       修改密码：
      </p>
      <p>
       ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码';
       <br/>
       <br/>
       # 之后键入修改密码的命令：
       <br/>
       ALTER USER root@localhost IDENTIFIED  BY 'root'
       <br/>
       10.使用新密码登陆：
      </p>

     <li>
      <p>
       mysql -u root -p 新密码
      </p>

     <li>
      <p>
       卸载MySQL：
      </p>
      <p>
       停止服务：net stop mysql
      </p>
      <p>
       删除服务：mysqld --remove
      </p>
      <p>
       登录成功后，执行命令：show databases;
      </p>

     <li>
      <p>
       MySQL的配置文件（window版）：
      </p>
     </li>
     <li>
      <p>
       [mysqld]
       <br/>
       # skip_grant_tables
       <br/>
       # 设置3306端口
       <br/>
       port=3306
       <br/>
       # 设置mysql的安装目录
       <br/>
       basedir=D:\Program Files\MySQL\MySQL Server 8.4
       <br/>
       # 设置mysql数据库的数据的存放目录
       <br/>
       datadir=D:\dev_soft\mysql-8.0.20-winx64\data
       <br/>
       # 允许最大连接数
       <br/>
       max_connections=200
       <br/>
       # 允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统
       <br/>
       max_connect_errors=10
       <br/>
       # 服务端使用的字符集默认为UTF8
       <br/>
       character-set-server=utf8mb4
       <br/>
       # 创建新表时将使用的默认存储引擎
       <br/>
       default-storage-engine=INNODB
       <br/>
       <br/>
       [client]
       <br/>
       #password = your_password
       <br/>
       # 设置mysql客户端连接服务端时默认使用的端口
       <br/>
       port=3306
       <br/>
       default-character-set=utf8mb4
       <br/>
       <br/>
       [mysql]
       <br/>
       # 设置mysql客户端默认字符集
       <br/>
       default-character-set=utf8mb4
      </p>
