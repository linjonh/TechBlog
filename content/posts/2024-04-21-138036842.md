    <h2>
     <a id="3_WWW_32">
     </a>
     3 WWW服务器配置
    </h2>
    <h3>
     <a id="_34">
     </a>
     编程要求
    </h3>
    <p>
     本关任务是学会如何配置Apache2服务器。
    </p>
    <p>
     注意：在执行本关任务前，首先执行如下命令来初始化任务环境。
    </p>
    <p>
     mkdir /var/www/html/test
     <br/>
     cp /var/www/html/index.html /var/www/html/test
    </p>
    <p>
     具体编程要求如下：
    </p>
    <p>
     将默认访问端口从80修改为8011;
     <br/>
     添加一个新的监听端口为8012，将根目录设置为
     <code>
      /var/www/html/test
     </code>
     ；
    </p>
    <h3>
     <a id="_47">
     </a>
     输出
    </h3>
    <p>
     实操，通过实验
    </p>
    <h3>
     <a id="_50">
     </a>
     输入
    </h3>
    <p>
     初始化环境：
     <br/>
     sudo apt-get install apache2
     <br/>
     mkdir /var/www/html/test
     <br/>
     cp /var/www/html/index.html /var/www/html/test
     <br/>
     sudo service apache2 start
    </p>
    <p>
     一定要初始化环境！！！
     <br/>
     修改监听窗口：80 -&gt; 8011:
     <br/>
     sudo vim /etc/apache2/ports.conf
    </p>
    <pre><code class="prism language-bash">Listen <span class="token number">8011</span>
Listen <span class="token number">8012</span>
</code></pre>
    <p>
     在行末尾添加以下代码 并把第一行80变为8011:
     <br/>
     sudo vim /etc/apache2/sites-enabled/000-default.conf
    </p>
    <pre><code class="prism language-go"><span class="token operator">&lt;</span>VirtualHost <span class="token operator">*</span><span class="token punctuation">:</span><span class="token number">80</span><span class="token operator">&gt;</span>变成<span class="token operator">&lt;</span>VirtualHost <span class="token operator">*</span><span class="token punctuation">:</span><span class="token number">8011</span><span class="token operator">&gt;</span>

行末添加以下代码：
<span class="token operator">&lt;</span>VirtualHost <span class="token operator">\*</span><span class="token punctuation">:</span><span class="token number">8012</span><span class="token operator">&gt;</span>
ServerAdmin webmaster@localhost
DocumentRoot <span class="token operator">/</span><span class="token keyword">var</span><span class="token operator">/</span>www<span class="token operator">/</span>html<span class="token operator">/</span>test
ErrorLog $<span class="token punctuation">{<!-- --></span>APACHE_LOG_DIR<span class="token punctuation">}</span><span class="token operator">/</span><span class="token builtin">error</span><span class="token punctuation">.</span>log
CustomLog $<span class="token punctuation">{<!-- --></span>APACHE_LOG_DIR<span class="token punctuation">}</span><span class="token operator">/</span>access<span class="token punctuation">.</span>log combined
<span class="token operator">&lt;</span><span class="token operator">/</span>VirtualHost<span class="token operator">&gt;</span>
</code></pre>
<p>
重启 apache：
<br/>
sudo service apache2 start
</p>
