    <h3>
     <a id="1_djangocoreexceptionsImproperlyConfigured_mysqlclient_1xxx_or_newer_is_required_you_have_0xx_40">
     </a>
     报错1: django.core.exceptions.ImproperlyConfigured: mysqlclient 1.x.xx or newer is required; you have 0.x.x.
    </h3>
    <pre><code class="prism language-python"><span class="token keyword">raise</span> ImproperlyConfigured<span class="token punctuation">(</span><span class="token string">'mysqlclient 1.x.xx or newer is required; you have %s.'</span> <span class="token operator">%</span> Database<span class="token punctuation">.</span>__version__<span class="token punctuation">)</span>

django<span class="token punctuation">.</span>core<span class="token punctuation">.</span>exceptions<span class="token punctuation">.</span>ImproperlyConfigured<span class="token punctuation">:</span> mysqlclient <span class="token number">1.</span>x<span class="token punctuation">.</span>xx <span class="token keyword">or</span> newer <span class="token keyword">is</span> required<span class="token punctuation">;</span> you have <span class="token number">0.</span>x<span class="token punctuation">.</span>x<span class="token punctuation">.</span>
</code></pre>
    <p>
     这里 xx 表示版本，报错版本可能不同但解决方法时一样的
    </p>
    <p>
     解决方法：
    </p>
    <p>
     <code>
      /Python37（python安装目录）/Lib/site-packages/django/db/backends/mysql/base.py，
     </code>
     注释掉以下内容：
    </p>
    <pre><code class="prism language-python"><span class="token comment">#if version &lt; (1, 3, 13):</span>
<span class="token comment">#		raise ImproperlyConfigured('mysqlclient 1.3.13 or newer is required; you have %s.' % Database.__version__)</span>
</code></pre>
    <h3>
     <a id="2AttributeError_str_object_has_no_attribute_decode_60">
     </a>
     报错2：AttributeError: ‘str’ object has no attribute ‘decode’
    </h3>
    <p>
     报错内容：
    </p>
    <pre><code class="prism language-xml">File "xx\Python37\lib\site-packages\django\db\backends\mysql\operations.py", line 146, in last_executed_query
query = query.decode(errors='replace')
AttributeError: 'str' object has no attribute 'decode'
</code></pre>
    <p>
     解决方法：
    </p>
    <blockquote>
     <p>
      打开
      <br/>
      xx\Python37\lib\site-packages\django\db\backends\mysql\operations.py
      <br/>
      把146行的 decode 修改为 encode 即可
     </p>
    </blockquote>
