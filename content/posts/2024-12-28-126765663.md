# 前端框架和后端框架总结

**前端框架**

```css
 1-HTML, CSS 和 JavaScript 框架，用于开发响应式，移动端先行的 web 项目
 2-web端开发：Vue.js和 nvue   ，Bootstrap，Layui，Element-UI，Mint UI，Angular，React
   app前端开发：uni-app框架
 
 开发工具：
 Visual studio Code
 hbuilder

```

**后端框架**

```css
 1-php语言开发：ThinkPHP框架  CakePHP
 2-java语言开发：Spring Boot框架，SpringMVC框架
 
 开发工具：
 IDEA
 Eclipse

```

**测试框架**

```css
 1-app自动化测试框架：appium
 2-web自动化测试框架：JMeter + Ant+ Jenkins，junit，selenium，requests + unittest，pytest单元测试框架，TestNG （测试报告：html，allure报告）
 
 测试工具：
 1.数据库管理  Navicat
 2.接口测试：postman，jemeter，Python+Requests，SoupUI
 3.性能测试：loadrunner，kylinTOP测试与监控平台（商用版）
 4.安全测试： AppScan，Burp Suite，OWASP Zed攻击代理（ZAP）
 5.自动化测试： pycharm+selenium
 6.代码管理： git hub
 7.bug管理： 禅道
 8.持续集成工具： jenkins
 9.app测试工具：Monkey 安卓自带的测试工具

```
