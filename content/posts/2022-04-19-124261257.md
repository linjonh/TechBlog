    <h3>
     <a id="Java_0">
     </a>
     Java之格式化字符串及格式化输出详解：
    </h3>
    <hr/>
    <h5>
     <a id="_3">
     </a>
     格式化输出：
    </h5>
    <blockquote>
     <p>
      <code>
       Java 5
      </code>
      引入了与C语言的
      <code>
       printf函数
      </code>
      风格类似的
      <code>
       format方法
      </code>
      和
      <code>
       printf方法
      </code>
      ，这两个方法可用于
      <code>
       java.io.PrintStream
      </code>
      和
      <code>
       java.io.PrintWriter
      </code>
      对象，System.out对象的类型是PrintStream,于是对应C语言的printf函数的Java版本就有了：
      <br/>
      <mark>
       System.out.printf(“书名：%s，定价：%f%n” , title , price);
      </mark>
      <br/>
      双括号括起来的字符串就是格式化字符串。
      <code>
       %s
      </code>
      ,
      <code>
       %f
      </code>
      以及
      <code>
       %n
      </code>
      是占位符，称为
      <code>
       格式说明符
      </code>
      ，它们不但说明了插入数据的位置，还说明了将插入什么类型的数据，以及如何对其格式化。
     </p>
    </blockquote>
    <br/>
    <h5>
     <a id="_10">
     </a>
     格式说明符：
    </h5>
    <blockquote>
     <p>
      语法为：
      <br/>
      <mark>
       %[argument_index$][flags][width][.precision]conversion
      </mark>
      <br/>
      方括号表示改部分是可选的：
      <br/>
      <code>
       argument_index
      </code>
      是参数索引;
      <br/>
      <code>
       flags
      </code>
      是一组修改输出格式的字符；
      <br/>
      <code>
       width
      </code>
      是一个正十进制整数，表示要写入输出的最小字符数；
      <br/>
      <code>
       precision
      </code>
      代表精度，通常用于限制字符数。精度在点号（.）后面给出；
      <br/>
      <code>
       conversion
      </code>
      是格式说明字符.
     </p>
    </blockquote>
    <hr/>
    <h5>
     <a id="_22">
     </a>
     格式化字符串：
    </h5>
    <p>
     String类的静态format方法用于
     <mark>
      创建格式化的字符串
     </mark>
     以及连接多个字符串对象。这里和C语言中的sprintf()方法区别比较，两者有相似之处。format()方法有两种重载方法。具体可以查看Java的API文档，如下图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c61b4ada2816ea6b0059ff00626b39df.png">
      <br/>
      <strong>
       （1）、format(String format , Object…args)
      </strong>
     </img>
    </p>
    <p>
     该方法使用指定的格式字符串和参数返回一个格式化字符串， 格式化后的新字符串使用本地默认的语言环境。
     <br/>
     语法如下：
    </p>
    <pre><code class="prism language-java">str<span class="token punctuation">.</span><span class="token function">format</span><span class="token punctuation">(</span><span class="token class-name">String</span> format<span class="token punctuation">,</span><span class="token class-name">Object</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span>args<span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      <mark>
       format
      </mark>
     </strong>
     ：格式字符串。
    </p>
    <p>
     <strong>
      <mark>
       args
      </mark>
     </strong>
     ：格式字符串中由格式说明符引用的参数。如果还有格式说明符以外的参数，则忽略这些额外的参数。此参数的数目是可变的，可以为0。
    </p>
    <br/>
    <p>
     <strong>
      （2）、format(Local l,String format,Object…args)
     </strong>
    </p>
    <p>
     <strong>
      <mark>
       l
      </mark>
      ：
     </strong>
     格式化过程中要应用的语言环境。如果l为null，则不进行本地化。
    </p>
    <p>
     <strong>
      <mark>
       format
      </mark>
      ：
     </strong>
     格式化字符串。
    </p>
    <p>
     <strong>
      <mark>
       args
      </mark>
      ：
     </strong>
     格式化字符串中由格式说明说明符引用的参数。如果还有格式说明符以外的参数，则忽略这些额外的参数。此参数的数目是可变的，可以为0。
    </p>
