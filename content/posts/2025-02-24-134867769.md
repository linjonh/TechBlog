    <h2>
     <a id="_1__10">
     </a>
     🕒 1. 概述
    </h2>
    <p>
     移动通信网络
    </p>
    <ul>
     <li>
      2.5G：GPRS/EDGE（Enhanced Data GSM Evolution）
     </li>
     <li>
      3G：WCDMA/CDMA2000/TD-SCDMA
     </li>
     <li>
      3.5G：HSDPA（High-Speed Downlink Packet Access）/EV-DO（Evolution-Data Only）
     </li>
     <li>
      4G：LTE（Long Term Evolution）
     </li>
     <li>
      5G
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cfddb9004b539d6b9fbb2ce408234bd3.png"/>
    </p>
    <ul>
     <li>
      蜂窝技术的速率都不高，无法提供类似于无线个域网、无线局域网和无线城域网的
      <strong>
       宽带
      </strong>
      接入技术，无线满足多媒体等应用的需求。
     </li>
     <li>
      多适用于手机、PDA这样的处理能力较低的
      <strong>
       弱终端
      </strong>
      ，对于具有高强处理能力的笔记本电脑是不太适宜的。
     </li>
     <li>
      专门从事无线广域网移动宽带无线接入技术标准制定的工作组是802.20
     </li>
    </ul>
    <h2>
     <a id="_2_3G4G5G_23">
     </a>
     🕒 2. 3G/4G/5G技术概述
    </h2>
    <p>
     3G技术：W-CDMA、CDMA2000、TD-SCDMA
     <br/>
     4G技术：LTE、LTE-Advanced、WiMAX 、HSPA+ 、WirelessMAN-Advanced 、VoLTE
    </p>
    <p>
     国际电信联盟（ITU）定义的4G标准
    </p>
    <ul>
     <li>
      静态传输速率达到1Gbps，用户在高速移动状态下可以达到100Mbps，就可以作为4G的技术之一
     </li>
     <li>
      为了达到上述峰值速率必须使用大于20MHz的带宽
     </li>
    </ul>
    <p>
     LTE-TDD，LTE-FDD，WiMAX，以及HSPA+
     <br/>
     三大运营商制式
    </p>
    <ul>
     <li>
      移动TDD（Time Division Duplex）
     </li>
     <li>
      联通FDD （Frequency Division Duplex）
     </li>
     <li>
      电信FDD（Frequency Division Duplex）
     </li>
    </ul>
    <p>
     五模十三频：
    </p>
    <p>
     2G的GSM，3G的TD-SCDMA、WCDMA，4G的TD-LTE、LTE-FDD五种模式，而十三频则是TD-LTE Band38/39/40，LTE-FDD Band7/3，TD-SCDMA Band34/39，WCDMA Band1/2/5，GSM Band2/3/8共十三个频段。
    </p>
    <p>
     频谱重整（Refarming）：对于已经分配给运营商使用的频段，可统筹规划，重新定义频段用途
    </p>
    <p>
     5G技术：
    </p>
    <ul>
     <li>
      使用频谱28GHz及60GHz，属极高频（EHF），比一般电讯业现行使用的频谱（如2.6GHz）高许多。
     </li>
     <li>
      超高的传输速度，能达到4G网络的40倍以上，延迟极低也比4G低不少。
     </li>
    </ul>
    <blockquote>
     <p>
      NSA（Non-Standalone）：4G和5G混合组网
      <br/>
      SA（Standalone）：5G独立组网
     </p>
    </blockquote>

     🕒 3. 卫星通信系统
    </h2>
    <h3>
     <a id="_31__63">
     </a>
     🕘 3.1 卫星通信系统的概念
    </h3>
    <ul>
     <li>
      卫星通信系统实际上也是一种微波通信，它以卫星作为中继站转发微波信号，在多个地面站之间通信。
     </li>
     <li>
      卫星通信系统由卫星端、地面端、用户端三部分组成。
      <ul>
       <li>
        卫星端在空中起中继站的作用，把地面站发上来的电磁波放大后再返送回另一地面站。
       </li>
       <li>
        地面站则是卫星系统与地面公众网的接口。
       </li>
       <li>
        用户端即是各种用户终端。
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <a id="_32__70">
     </a>
     🕘 3.2 卫星通信系统的分类
    </h3>
    <p>
     按照工作轨道区分，卫星通信系统一般分为以下3类：
     <br/>
     （1）低轨道卫星通信系统(LEO)
     <br/>
     （2）中轨道卫星通信系统(MEO)
     <br/>
     （3）高轨道卫星通信系统(GEO)
     <br/>
     按照通信范围区分，卫星通信系统可以分为国际通信卫星、区域性通信卫星、国内通信卫星。
     <br/>
     按照用途区分，卫星通信系统可以分为综合业务通信卫星、军事通信卫星、海事通信卫星、电视直播卫星等。
     <br/>
     按照转发能力区分，卫星通信系统可以分为无星上处理能力卫星、有星上处理能力卫星。
    </p>
    <h3>
     <a id="_33__79">
     </a>
     🕘 3.3 卫星通信系统的特点
    </h3>
    <p>
     （1）下行广播，覆盖范围广
     <br/>
     （2）工作频带宽：可用频段从150MHz～30GHz；
     <br/>
     （3）通信质量好
     <br/>
     （4）网络建设速度快、成本低
     <br/>
     （5）信号传输时延大
     <br/>
     （6）控制复杂
    </p>

    <h2>
     <a id="_4_80220_96">
     </a>
     🕒 4. 802.20技术
    </h2>
    <h3>
     <a id="_41_80220_97">
     </a>
     🕘 4.1 802.20技术特性
    </h3>
    <p>
     802.20协议族
    </p>
    <ul>
     <li>
      移动宽带无线接入（Mobile Broadband Wireless Access），又称Mobile-Fi
     </li>
     <li>
      支持最高时速为250KM/h的高速移动
     </li>
     <li>
      纯IP架构：基于分组，可确保通道的最大利用率，但是时效性没有电路交换好
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/48a07fcea8b45f5da16dff42f7f754d6.png"/>
    </p>
    <ul>
     <li>
      在物理层技术上，以OFDM和MIMO为核心，充分挖掘
      <strong>
       时域、频域和空间域
      </strong>
      的资源，大大提高了系统的频谱效率
     </li>
     <li>
      在设计理念上，基于分组数据的
      <strong>
       纯IP架构
      </strong>
      应对突发性数据业务的性能也优于现有的3G技术，与3.5G(HSDPA、EV-DO)性能相当
     </li>
     <li>
      另外，在实现、部署成本上也具有较大的优势。
     </li>
    </ul>
    <p>
     IEEE 802.20秉承了IEEE 802协议族的
     <strong>
      纯IP架构
     </strong>
     。
    </p>
    <p>
     纯IP架构，与3GPP和3GPP2所提出的全IP概念有所不同——前者是
     <strong>
      核心网和无线接入网
     </strong>
     都基于IP传输，而后者仅仅实现了核心网的IP化。设计架构的差异使802.20与其它3G技术相比具有明显的优势。
    </p>
    <h3>
     <a id="_42_80220_114">
     </a>
     🕘 4.2 802.20与其他技术间的关系
    </h3>
    <p>
     802.20 vs 3G
    </p>
    <ul>
     <li>
      802.20能够实现所有3G功能
     </li>
     <li>
      物理层更先进、部署价格更低
     </li>
     <li>
      3G上行、下行速率不同
     </li>
    </ul>
    <table>
     <thead>
      <tr>
       <th>
        1
       </th>
       <th>
        IEEE 802.20
       </th>
       <th>
        3G
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        目标市场
       </td>
       <td>
        1．高移动性、高吞吐量数据应用
        <p>
         2．对称数据服务
        </p>
        <p>
         3．对数据服务时延敏感度要求高
        </p>
        <p>
         4．全球移动和漫游
        </p>
       </td>
       <td>
        1．高移动性、语音业务和低速率数据应用
        <p>
         2．非对称数据服务
        </p>
        <p>
         3．对数据服务时延敏感度要求低
        </p>
        <p>
         4．全球移动和漫游
        </p>
       </td>
      </tr>
      <tr>
       <td>
        技术特点
       </td>
       <td>
        1．全新的空中接口(物理层和MAC层)
        <p>
         2．属于广域网技术
        </p>
        <p>
         3．以OFDM、MIMO为物理层核心技术
        </p>
        <p>
         4．工作于3.5GHz以下的许可频段
        </p>
        <p>
         5．典型信道带宽小于5MHz
        </p>
        <p>
         6．纯IP架构
        </p>
        <p>
         7．主要针对移动多媒体应用
        </p>
        <p>
         8．高效的上下行数据传输效率
        </p>
        <p>
         9．低时延架构
        </p>
       </td>
       <td>
        1．基于GSM或IS-41的演进，已有较成熟的空中接口(WCDMA、cdma2000和TD-SCDMA)
        <p>
         2．属于广域网技术
        </p>
        <p>
         3．以CDMA为物理层核心技术
        </p>
        <p>
         4．工作于2.7GHz以下的许可频段
        </p>
        <p>
         5．典型信道带宽小于5MHz
        </p>
        <p>
         6．以基于电路交换的架构为主
        </p>
        <p>
         7．主要针对移动语音业务
        </p>
        <p>
         8．数据传输效率下行一般，上行较低
        </p>
        <p>
         9．高时延架构
        </p>
       </td>
      </tr>
     </tbody>
    </table>
    <ol>
     <li>
      两者的
      <strong>
       目标市场重叠较大
      </strong>
      。
     </li>
    </ol>
    <ul>
     <li>
      首先，都是广域网技术。
     </li>
     <li>
      其次，802.20具有低时延架构，可以基于VoIP技术来提供高质量的语音业务，也就是说可以支持3G所能提供的全部业务。
     </li>
    </ul>
    <ol start="2">
     <li>
      在
      <strong>
       物理层核心技术
      </strong>
      上802.20更为先进，因而拥有更具吸引力的性能优势。
     </li>
     <li>
      802.20的纯IP架构使它在
      <strong>
       组网成本
      </strong>
      上具有较明显的价格优势。
     </li>
    </ol>
    <p>
     802.20 vs LTE
    </p>
    <ul>
     <li>
      竞争关系
     </li>
     <li>
      物理层技术都是基于OFDM和MIMO
     </li>
     <li>
      都是基于IP的架构
     </li>
    </ul>
    <p>
     LTE(Long Term Evolution)是国际标准化组织3GPP在2004年底提出的研究计划，旨在提高3G技术在宽带无线接入市场的竞争力。
    </p>
    <p>
     LTE的市场定位是弥补3G技术在分组接入方面的不足，技术特性与802.20极为相似
    </p>
    <ul>
     <li>
      都是针对
      <strong>
       广域网的移动通信技术
      </strong>
      ，LTE支持最高的移动速率为350 km/h，并且能在15~120 km/h下提供高性能的服务
     </li>
     <li>
      物理层技术都是基于
      <strong>
       OFDM和MIMO
      </strong>
      ，频谱效率都很高，并且摆脱了高通公司的CDMA专利制约
     </li>
     <li>
      都支持
      <strong>
       低时延
      </strong>
      ，LTE的接入网时延在10 ms以内，控制平面时延小于100 ms
     </li>
     <li>
      都是基于
      <strong>
       IP的架构
      </strong>
      ，LTE的目标是建立一个无线接入网与固网融合的纯IP的核心网，以满足宽带无线接入的需求。
     </li>
    </ul>
    <p>
     与802.11、802.16间的关系
    </p>
    <ul>
     <li>
      11和16主要是针对
      <strong>
       牧游式
      </strong>
      的无线接入，提供
      <strong>
       步行速率
      </strong>
      的移动性。802.20的目标市场定位于无线广域网，强调它对
      <strong>
       高速移动性
      </strong>
      的支持。
     </li>
     <li>
      三种技术存在很强的
      <strong>
       互补性
      </strong>
      。若将它们混合组网，取长补短，将是一种非常好的全网覆盖解决方案。
     </li>
    </ul>
    <h3>

     </a>
     🕘 4.3 802.20展望
    </h3>
    <ul>
     <li>
      高移动性和高吞吐量必然是未来无线通信市场的重要需求。
     </li>
     <li>
      IEEE 802.20正是为满足这一需求而专门设计的宽带无线接入技术，并具有性能好、效率高、成本低和部署灵活等特点。
     </li>
     <li>
      802.20在移动性上优于802.16和802.11，在数据吞吐量上强于3G技术，其设计理念也符合下一代技术的发展方向，因而确实是一种非常有前景的无线技术。
     </li>
    </ul>
