   <div class="htmledit_views" id="content_views">
    <p>
     在uni-app中获取小程序openid，可以通过uni.getUserInfo方法来实现。具体步骤如下：
    </p>
    <ol>
     <li>
      在uni-app的入口页面中，引入uni.getUserInfo方法和uni.login方法。
     </li>
    </ol>
    <pre><code class="language-javascript">import uni from 'uni-app';
</code></pre>
    <p>
    </p>
    <ol>
     <li>
      在需要获取openid的地方，调用uni.login方法获取登录凭证code。
     </li>
    </ol>
    <pre><code class="language-javascript">uni.login({
  provider: 'weixin',
  success: res =&gt; {
    if (res.code) {
      // 获取到登录凭证code
      let code = res.code;
      // 调用后端接口，传递code进行获取openid操作
      // ...
    }
  }
});
</code></pre>
    <p>
    </p>
    <ol>
     <li>
      <p>
       在后端接口中，根据传递的code，使用微信提供的API获取openid。具体操作可以参考微信小程序的官方文档。
      </p>
     </li>
     <li>
      <p>
       后端接口返回openid后，可以将openid存储在本地或者传递给前端进行业务逻辑处理。
      </p>
     </li>
    </ol>
    <p>
     注意：获取openid需要在小程序端进行，所以需要将uni-app打包成小程序后，在小程序中使用以上方法获取openid。
    </p>
