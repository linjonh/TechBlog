    <p>
    </p>
    <p>
     <strong>
      小程序用的wepy框架，1.6.1-alpha4 版本开始支持小程序分包
     </strong>
    </p>
    <pre class="has"><code>npm install wepy-cli@1.6.1-alpha4 -g</code></pre>
    <p>
     <strong>
      根据小程序官方文档进行配置，开发工具页面加载出来显示是空白。应该是正常的，分包没有问题，可以试试手机预览看效果，坑！（注意：只改配置就可以了，不用动代码）
     </strong>
    </p>
    <p>
    </p>
    <p>
     <strong>
      刚开始用最笨的方法压缩代码，写了个批处理文件压缩js
     </strong>
    </p>
    <p>
     @echo off
     <br/>
     :: 设置压缩JS文件的根目录，脚本会自动按树层次查找和压缩所有的JS
     <br/>
     SET JSFOLDER=C:\Users\Administrator\Desktop\formini
     <br/>
     echo 正在查找JS文件
     <br/>
     chdir /d %JSFOLDER%
     <br/>
     for /r . %%a in (*.js) do (
     <br/>
     @echo 正在压缩 %%~a ...
     <br/>
     uglifyjs %%~fa  -m -o %%~fa
     <br/>
     )
     <br/>
     echo 完成!
     <br/>
     pause &amp; exit
    </p>
    <p>
     <strong>
      然后发现wepy插件wepy-plugin-uglifyjs
     </strong>
    </p>
    <p>
     安装：
    </p>
    <pre class="has"><code>npm install wepy-plugin-uglifyjs --save-dev</code></pre>
    <p>
     配置：
    </p>
    <pre class="has"><code>module.exports.plugins = {
    'uglifyjs': {
        filter: /\.js$/,
        config: {
        }
    },
};</code></pre>
