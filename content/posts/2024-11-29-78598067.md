    <p>
     Linux是一种应用于PC和工作站的操作系统，而Ubuntu是Linux的一个分支发行套件。
     <br/>
     在Linux诞生之前，有一个相当稳定且成熟的操作系统Unix，可以说Unix是Linux的老
     <br/>
     前辈。
     <br/>
     1973年，Thompson和Ritchie用C语言重写Unix。
     <br/>
     Linux是参考Unix开发并发展起来的一个Unix操作系统。
     <br/>
     Linux是指一种计算机操作系统：Linux也可指操作系统的内核，实际应用上人们
     <br/>
     已经习惯了用Linux来形容整个基于Linux内核，并且搭载了各种人机界面，应用和服
     <br/>
     务软件的操作系统。基于这些组件的Linux软件被称为Linux发行版。
    </p>
    <p>
     在Linux内核中，主要内核模块包括：
     <br/>
     进程管理
     <br/>
     定时器
     <br/>
     中断管理
     <br/>
     内存管理
     <br/>
     模块管理
     <br/>
     虚拟文件系统接口
     <br/>
     文件系统
     <br/>
     设备驱动程序
     <br/>
     网络管理
     <br/>
     系统启动等
    </p>
    <p>
     Ubuntu是一个南非的民族，着眼于人们之间的忠诚和联系，该词来自于祖鲁语和科萨
     <br/>
     语。Ubuntu发音乌邦图，被视为非洲人的传统理念。
    </p>
