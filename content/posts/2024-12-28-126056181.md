    <h3>
     <a id="1_2">
     </a>
     1.相关指令
    </h3>
    <p>
     <strong>
      1.按下键盘上的windows+R键输入cmd，或在搜索框直接输入cmd
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/61c749789ee0ee21321dfda97aff95bf.png">
      <br/>
      <strong>
       2.进入mysql：
      </strong>
      <br/>
      在命令行输入mysql -uroot -p，然后输入你的mysql密码，成功后进入mysql数据库
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/62571d318922c56a530aeed53ee0779a.png">
       <br/>
       <strong>
        3.查看mysql中的数据库
       </strong>
       <br/>
       show databases;
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3de4c9ea1141ac89d48a1fb580ad8a63.png">
        <br/>
        <strong>
         4.使用某一个数据库
        </strong>
        <br/>
        use 数据库名;
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/205c2f9fa4ab7b9d8e34b48c99cbc05d.png">
         <br/>
         当我们发出了使用该数据库的命令后，才能访问该数据库
         <br/>
         <strong>
          5.查看数据库中的表
         </strong>
         <br/>
         记住只有我们开始使用这个数据库时，我们才能使用针对你使用的数据库进行操作
         <br/>
         show tables;
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ceaf9b011efafc58d425de24f448c038.png">
          <br/>
          <strong>
           6.查看表的设计结构
          </strong>
          <br/>
          describe 表名;
          <br/>
          或者
          <br/>
          desc 表名;
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/acaa6c4a138561270915d2bcf2cf5237.png"/>
