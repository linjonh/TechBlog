    <p>
     <strong>
      摘要
     </strong>
     ：卡尔曼滤波（Kalman Filter）是传感器数据融合领域的经典算法，在姿态解算、导航定位等嵌入式场景中广泛应用。本文将从公式推导、代码实现、参数调试三个维度深入解析卡尔曼滤波，并给出基于STM32硬件的完整工程案例。
    </p>
    <hr/>
    <h3>
     一、卡尔曼滤波核心思想
    </h3>
    <h4>
     1.1 什么是卡尔曼滤波？
    </h4>
    <p>
     卡尔曼滤波是一种
     <strong>
      最优递归估计算法
     </strong>
     ，通过融合
     <strong>
      预测值
     </strong>
     （系统模型）与
     <strong>
      观测值
     </strong>
     （传感器数据），在噪声干扰环境下实现对系统状态的动态估计。其核心优势在于
     <strong>
      实时性
     </strong>
     和
     <strong>
      自适应性
     </strong>
     。
    </p>
    <h4>
     1.2 适用场景
    </h4>
    <ul>
     <li>
      <p>
       存在高斯白噪声的线性系统
      </p>
     </li>
     <li>
      <p>
       需要多传感器数据融合的场景
      </p>
     </li>
     <li>
      <p>
       实时性要求高的嵌入式系统（如无人机、平衡车）
      </p>
     </li>
    </ul>

    <h3>
     四、一维卡尔曼滤波代码实现
    </h3>
    <pre><code class="hljs">// 卡尔曼结构体定义
typedef struct {
    float q;    // 过程噪声方差
    float r;    // 测量噪声方差
    float x;    // 状态估计值
    float p;    // 估计误差协方差
    float k;    // 卡尔曼增益
} KalmanFilter;

// 初始化滤波器
void Kalman_Init(KalmanFilter *kf, float q, float r) {
    kf-&gt;q = q;
    kf-&gt;r = r;
    kf-&gt;p = 1.0f;
    kf-&gt;x = 0;
}

// 卡尔曼迭代
float Kalman_Update(KalmanFilter *kf, float measurement) {
    // 预测阶段
    kf-&gt;p += kf-&gt;q;

    // 更新阶段
    kf-&gt;k = kf-&gt;p / (kf-&gt;p + kf-&gt;r);
    kf-&gt;x += kf-&gt;k * (measurement - kf-&gt;x);
    kf-&gt;p *= (1 - kf-&gt;k);
    
    return kf-&gt;x;
}</code></pre>

     5.2 关键代码片段

float Gyro[3], Accel[3];
KalmanFilter kf_x, kf_y, kf_z;

int main(void) {
    // 初始化
    MPU6050_Init();
    Kalman_Init(&amp;kf_x, 0.001, 0.5);
    // 类似初始化kf_y, kf_z

    while(1) {
        // 读取原始数据
        MPU6050_ReadData(Gyro, Accel);
        
        // 执行滤波
        float roll = Kalman_Update(&amp;kf_x, Accel[0]);
        // 同样处理pitch/yaw
        
        // 通过串口输出
        printf("Roll:%.2f\tPitch:%.2f\r\n", roll, pitch);
        
        HAL_Delay(10); // 10ms采样周期
    }
}</code></pre>

    <h3>
     六、参数调试经验
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        Q值调整
       </strong>
       ：增大Q会使滤波器更信任新测量值，响应更快但噪声增大
      </p>
     </li>
     <li>
      <p>
       <strong>
        R值调整
       </strong>
       ：增大R会使滤波器更信任预测值，曲线平滑但滞后明显
      </p>
     </li>
     <li>
      <p>
       <strong>
        典型参数范围
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         加速度计：Q=0.001, R=0.5
        </p>
       </li>
       <li>
        <p>
         陀螺仪：Q=0.003, R=0.1
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       调试工具：使用串口波形工具（如VOFA+）实时观察数据曲线
      </p>
     </li>
    </ol>

    <h3>
     八、常见问题解答
    </h3>
    <p>
     <strong>
      Q1：如何处理非线性系统？
     </strong>
     <br/>
     A：改用扩展卡尔曼滤波（EKF）或无迹卡尔曼滤波（UKF）
    </p>
    <p>
     <strong>
      Q2：滤波器发散怎么办？
     </strong>
     <br/>
     A：检查系统模型是否准确，适当增大Q值
    </p>
    <p>
     <strong>
      Q3：如何验证滤波效果？
     </strong>
     <br/>
     A：通过静态测试（方差分析）和动态测试（阶跃响应）结合验证
    </p>
