    </p>
    <h2>
     <a id="1__1">
     </a>
     1 简介
    </h2>
    <p>
     lua语言与C语言使用还是很紧密,以前我只是学习lua语言比较多,C语言api部分了解比较少,最近在学习tcc编译器的使用进一步学习一下lua C语言api的使用。
    </p>
    <h2>
     <a id="2__3">
     </a>
     2 配置编译环境
    </h2>
    <p>
     首先需配置好tcc编译器环境
     <a href="https://blog.csdn.net/sinat_41629921/article/details/145969790">
      [参考]
     </a>
     ,再配置好lua源码路径
     <a href="https://blog.csdn.net/sinat_41629921/article/details/146031852">
      [参考]
     </a>
     ,新建一个C:\run\test\lua-Capi目录,后续主要在该路径下编程,复制之前的tcc编程环境.bat到当前目录,对其进行修改,更改如下,主要增加一个lua_src变量设置为lua源码路径。
    </p>
    <pre><code class="prism language-c"><span class="token operator">::</span>将编辑器路径设置到path变量
set path<span class="token operator">=</span>C<span class="token operator">:</span>\run\tcc<span class="token punctuation">;</span><span class="token operator">%</span>path<span class="token operator">%</span>
<span class="token operator">::</span>设置lua源码路径
set lua_src<span class="token operator">=</span>C<span class="token operator">:</span>\run\test\lua<span class="token operator">-</span><span class="token number">5.4</span><span class="token number">.7</span>\src
<span class="token operator">::</span>启动命令行
cmd
</code></pre>
    <p>
     其次,需复制之前lua源码路径下之前编译好的动态链接库文件lua.dll和导出文件lua.def到当前目录,方便后续使用。
    </p>
    <h2>
     <a id="3__15">
     </a>
     3 编写第一个程序
    </h2>
    <p>
     第一个程序01hello_lua.c如下
    </p>
    <pre><code class="prism language-c"><span class="token comment">//01hello_lua.c</span>
<span class="token comment">//第一个基本程序</span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;stdio.h&gt;</span></span>

<span class="token comment">//载入lua相关头文件</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span>

     相对还是比较简单.
    </p>
    <h2>
     <a id="3__43">
     </a>
     3 编译第一个程序
    </h2>
    <p>
     打开tcc编程环境.bat,输入以下命令。
    </p>
    <pre><code class="prism language-c"><span class="token operator">::</span>编译程序
tcc <span class="token operator">-</span>I <span class="token operator">%</span>lua_src<span class="token operator">%</span> lua<span class="token punctuation">.</span>def <span class="token number">01</span>hello_lua<span class="token punctuation">.</span>c <span class="token operator">-</span>o <span class="token number">01</span>hello_lua<span class="token punctuation">.</span>exe
<span class="token operator">::</span>执行程序
<span class="token number">01</span>hello_lua
</code></pre>
    <p>
     运行情况如下:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9c6e8e039b664f5eb1752949056eb0fa.png">
      <br/>
      本次采用链接到动态链接库方式编译,这样生成的程序比较小,若要复制程序,需连同lua.dll一起复制。
     </img>
    </p>
    <h2>
     <a id="4_Makefile_55">
     </a>
     4 编写Makefile文件
    </h2>
    <p>
     从上面编译命令看,还是比较复杂,容易输错,我将上面编译指令编写为Makefile文件,如下。
    </p>
    <pre><code class="prism language-c"># 学习lua C语言api的Makefile文件
<span class="token macro property"><span class="token directive-hash">#</span> <span class="token directive keyword">lua</span><span class="token expression">源码路径</span></span>
lua_src<span class="token operator">=</span>C<span class="token operator">:</span>\run\test\lua<span class="token operator">-</span><span class="token number">5.4</span><span class="token number">.7</span>\src
# 显示帮助
help<span class="token operator">:</span>
	echo 用法为make <span class="token punctuation">[</span>源码名称<span class="token punctuation">]</span>


# 编写通用编译指令
# <span class="token operator">%</span>代表匹配任何输入参数<span class="token punctuation">,</span>这里主要为源码名称
# $@代表匹配表示生成的目标文件<span class="token punctuation">,</span>这里为源码名称
<span class="token operator">%</span><span class="token operator">:</span>
	@tcc <span class="token operator">-</span>I $<span class="token punctuation">(</span>lua_src<span class="token punctuation">)</span> lua<span class="token punctuation">.</span>def $@<span class="token punctuation">.</span>c <span class="token operator">-</span>o $@<span class="token punctuation">.</span>exe
	@echo <span class="token operator">--</span><span class="token operator">--</span><span class="token operator">-</span>$@<span class="token punctuation">.</span>exe生成成功<span class="token operator">--</span><span class="token operator">--</span>
	@echo <span class="token operator">--</span><span class="token operator">--</span><span class="token operator">-</span>运行$@<span class="token punctuation">.</span>exe程序<span class="token operator">--</span><span class="token operator">--</span>
	@<span class="token punctuation">.</span><span class="token operator">/</span>$@
</code></pre>
    <p>
     输入下面指令即可编译并运行程序。
    </p>
    <pre><code class="prism language-c">make <span class="token number">01</span>hello_lua
</code></pre>
