    <h2>
     <a id="_6">
     </a>
     前言
    </h2>
    <p>
     中国互联网近年来一直都在蓬勃发展，代表公司有华为、阿里巴巴、腾讯、百度等。其中软件当中最核心的数据库，例如mysql、oracle、sqlserver，一直掌握在外国公司手中。使用外国数据库，不仅会产生昂贵的费用，另外还可能出现隐私泄露，这是政企公司决不能容忍的。所以国产化数据库势在必行，同时国家也大力扶持。
    </p>

    <h2>
     <a id="_24">
     </a>
     二、达梦公司简介
    </h2>
    <p>
     武汉达梦数据库有限公司成立于2000年，为国有控股的基础软件企业，专业从事数据库管理系统研发、销售和服务。其前身是华中科技大学数据库与多媒体研究所，是国内最早从事数据库管理系统研发的科研机构。达梦数据库为中国数据库标准委员会组长单位，得到了国家各级政府的强力支持。
    </p>
    <p>
     公司是国家规划布局内重点软件企业，首批获得国家“双软”认证的高新技术企业，“AAA”资信企业，并通过ISO9001：2000版国际质量管理体系认证、CMMI3级认证、武器装备科研生产单位二级保密资格认证、国家保密局“涉及国家秘密的计算机信息系统集成资质”认证、工业和信息化部“计算机信息系统集成企业资质” 二级认证。
    </p>
    <p>
     公司先后完成了近60项国家级、省部级的科研开发项目，取得了50多项研究成果，皆为国际先进、国内领先水平，其中30多项获国家、省部级科技进步奖，在国内同行中处于领先地位。2005年，达梦数据库被评为“国家高技术产业化示范工程”，连续多次荣获“中国国际软件博览会金奖”。2010年，达梦数据库被评为2009-2010年度中国软件和信息服务业最具潜力企业，并荣获“中国软件明星奖”。达梦数据库管理系统多次被评为中国优秀软件产品。

    <h2>
     <a id="DM8_102">
     </a>
     五、达梦数据库DM8安装
    </h2>
    <ol>
     <li>
      创建用户组
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">groupadd</span> dinstall
</code></pre>
    <ol start="2">
     <li>
      创建用户
     </li>
    </ol>
    <pre><code class="prism language-bash"> <span class="token function">useradd</span> -g dinstall -m -d /home/dmdba -s /bin/bash dmdba
</code></pre>
    <ol start="3">
     <li>
      更改用户 dmdba 的密码
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">passwd</span> dmdba
</code></pre>
    <ol start="4">
     <li>
      创建数据库安装目录
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">mkdir</span> /dm8
</code></pre>
    <ol start="5">
     <li>
      dmdba用户赋予文件权限
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">chown</span> dmdba:dinstall /dm8
</code></pre>
    <ol start="6">
     <li>
      创建镜像文件挂载目录
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">mkdir</span> /mnt/dm
</code></pre>
    <ol start="7">
     <li>
      挂载文件
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">mount</span> /opt/dm8_20210818_x86_rh6_64_ent_8.4.2.18_pack14.iso /mnt/dm
</code></pre>
    <ol start="8">
     <li>
      切换到dmdba用户
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">su</span> - dmdba
</code></pre>
    <ol start="9">
     <li>
      进入挂载目录
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /mnt/dm
</code></pre>
    <ol start="10">
     <li>
      <p>
       以上流程终端显示
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5edae0c73e6fb02cc314747adfd49a79.png#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       执行安装命令
      </p>
     </li>
    </ol>
    <pre><code class="prism language-bash">./DMInstall.bin -i
</code></pre>
    <ol start="12">
     <li>
      临时空间不足报错处理
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f1f77b5099ce1b443799afeab6a6c132.png#pic_center"/>
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">su</span> root
<span class="token function">mkdir</span> -p /tem1
<span class="token function">chown</span> dmdba:dinstall /tem1
<span class="token function">su</span> dmdba
<span class="token builtin class-name">export</span> <span class="token assign-left variable">DM_INSTALL_TMPDIR</span><span class="token operator">=</span>/tem1
./DMInstall.bin -i
</code></pre>
    <ol start="13">
     <li>
      “可打开文件数过少，建议至少设置为65536或更多”警告处理
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">su</span> root
<span class="token function">vi</span> /etc/security/limits.conf
<span class="token function">su</span> dmdba
</code></pre>
    <p>
     末尾添加
     <br/>
     <code>
      dmdba soft nofile 65535
     </code>
     <br/>
     <code>
      dmdba hard nofile 65535
     </code>
     <br/>
     <code>
      root soft nofile 65535
     </code>
     <br/>
     <code>
      root hard nofile 65535
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a0dd536b1543055dc171998a9b18e615.png#pic_center"/>
    </p>
    <p>
     修改以后保存，注销当前用户，重新登录，执行
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">ulimit</span> -a
</code></pre>

    </p>
