# num++ 和 ++num 的区别

这个矛盾在C语言中体现的比较明显,在之后的一些编程语言中都有针对地对这个问题进行了回避或者改正.

比如Python中就不含++ 直接用+=

下面主要介绍一下两者的区别,用人类的语言不管怎么描述都感觉怪怪的,所以 直接来C 吧

++num与以下程序等价

num+=1;

return num;

int tmp=num;

num += 1;

return tmp;
