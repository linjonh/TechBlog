    <h2>
     <a id="_11">
     </a>
     代码实现
    </h2>
    <p>
     在 Vue 项目中使用 Element UI 的
     <code>
      el-tree
     </code>
     组件时，树形数据的懒加载和默认勾选功能是常见的需求。以下是如何实现这两种功能的详细步骤：
    </p>
    <h3>
     <a id="1__13">
     </a>
     1. 基本结构
    </h3>
    <p>
     <code>
      el-tree
     </code>
     是 Element UI 提供的树形组件，支持懒加载、节点勾选等功能。以下是组件的基本结构：
    </p>
    <pre><code class="prism language-vue">&lt;el-drawer v-if="drawerVisible" v-model="drawerVisible" title="角色权限" size="50%"&gt;
  &lt;el-tree
    ref="tree"
    :props="treeProps"
    :load="loadNode"
    lazy
    show-checkbox
    node-key="id"
    :default-checked-keys="defaultCheckedKeys"
    @check="onCheckChange"
  /&gt;
&lt;/el-drawer&gt;
</code></pre>
    <p>
     关键属性说明：
    </p>
    <ul>
     <li>
      <code>
       lazy
      </code>
      ：启用懒加载模式。
     </li>
     <li>
      <code>
       load
      </code>
      ：懒加载数据的方法。
     </li>
     <li>
      <code>
       show-checkbox
      </code>
      ：显示节点复选框。
     </li>
     <li>
      <code>
       node-key
      </code>
      ：节点的唯一标识字段。
     </li>
     <li>
      <code>
       default-checked-keys
      </code>
      ：默认勾选的节点 ID 数组。
     </li>
     <li>
      <code>
       @check
      </code>
      ：节点勾选状态变化时触发的事件。
     </li>
    </ul>
    <h3>
     <a id="2__40">
     </a>
     2. 懒加载实现
    </h3>
    <p>
     懒加载的核心是通过
     <code>
      load
     </code>
     方法动态加载节点数据。以下是一个示例：
    </p>
    <pre><code class="prism language-javascript"><span class="token literal-property property">methods</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
  <span class="token function">loadNode</span><span class="token punctuation">(</span><span class="token parameter">node<span class="token punctuation">,</span> resolve</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

       <span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token comment">//数据列表</span>
      <span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token keyword">if</span> <span class="token punctuation">(</span>node<span class="token punctuation">.</span>level <span class="token operator">===</span> <span class="token number">1</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
          <span class="token function">resolve</span><span class="token punctuation">(</span><span class="token punctuation">[</span>
            <span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token comment">//数据列表</span>
        <span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token punctuation">{<!-- --></span>
      <span class="token comment">// 其他层级返回空数组</span>
      <span class="token function">resolve</span><span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     实现逻辑：
    </p>
    <ul>
     <li>
      <strong>
       根节点加载
      </strong>
      ：当
      <code>
       node.level === 0
      </code>
      时，返回根节点数据。
     </li>
     <li>
      <strong>
       子节点加载
      </strong>
      ：根据父节点的
      <code>
       id
      </code>
      返回对应的子节点数据。
     </li>
     <li>
      <strong>
       叶子节点标记
      </strong>
      ：通过
      <code>
       leaf: true
      </code>
      标记节点为叶子节点，避免进一步加载。
     </li>
    </ul>
    <h3>
     <a id="3__69">
     </a>
     3. 默认勾选功能
    </h3>
    <p>
     通过
     <code>
      default-checked-keys
     </code>
     属性，可以设置默认勾选的节点。该属性接收一个数组，数组中的值为需要勾选的节点 ID。
    </p>
    <pre><code class="prism language-javascript"><span class="token function">data</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token keyword">return</span> <span class="token punctuation">{<!-- --></span>
    <span class="token literal-property property">defaultCheckedKeys</span><span class="token operator">:</span> <span class="token punctuation">[</span><span class="token string">'role'</span><span class="token punctuation">,</span> <span class="token string">'user-list'</span><span class="token punctuation">]</span> <span class="token comment">// 默认勾选的节点 ID</span>
  <span class="token punctuation">}</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     在树加载完成后，
     <code>
      el-tree
     </code>
     会自动勾选
     <code>
      defaultCheckedKeys
     </code>
     中指定的节点。
    </p>
    <h3>
     <a id="4__83">
     </a>
     4. 动态加载初始节点
    </h3>
    <p>

     <code>
      ref
     </code>
     获取树实例，并调用
     <code>
      expand
     </code>
     方法实现：
    </p>
    <pre><code class="prism language-javascript"><span class="token literal-property property">methods</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
  <span class="token function">openDrawer</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    <span class="token comment">//第一种实现方法</span>
    <span class="token keyword">this</span><span class="token punctuation">.</span><span class="token function">$nextTick</span><span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token operator">=&gt;</span> <span class="token punctuation">{<!-- --></span>
      <span class="token keyword">this</span><span class="token punctuation">.</span><span class="token function">loadInitialNodes</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

        <span class="token keyword">this</span><span class="token punctuation">.</span><span class="token function">loadInitialNodes</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
      <span class="token punctuation">}</span><span class="token punctuation">,</span> <span class="token number">500</span><span class="token punctuation">)</span>
  <span class="token punctuation">}</span><span class="token punctuation">,</span>
  <span class="token function">loadInitialNodes</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token keyword">this</span><span class="token punctuation">.</span>$refs<span class="token punctuation">.</span>tree<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
      <span class="token comment">// 展开根节点的所有子节点</span>
      <span class="token keyword">this</span><span class="token punctuation">.</span>$refs<span class="token punctuation">.</span>tree<span class="token punctuation">.</span>root<span class="token punctuation">.</span>childNodes<span class="token punctuation">.</span><span class="token function">forEach</span><span class="token punctuation">(</span><span class="token parameter">node</span> <span class="token operator">=&gt;</span> <span class="token punctuation">{<!-- --></span>
        node<span class="token punctuation">.</span><span class="token function">expand</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
      <span class="token punctuation">}</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     实现逻辑：
    </p>
    <ul>
     <li>
      在
      <code>
       openDrawer
      </code>
      方法中，打开抽屉并等待 DOM 更新完成后调用
      <code>
       loadInitialNodes
      </code>
      。
     </li>
     <li>
      在
      <code>
       loadInitialNodes
      </code>
      中，通过
      <code>
       this.$refs.tree
      </code>
      获取树实例，并遍历根节点的子节点，调用
      <code>
       expand
      </code>
      方法展开节点。
     </li>
    </ul>
    <h3>
     <a id="5__115">
     </a>
     5. 节点勾选事件监听
    </h3>
    <p>
     通过
     <code>
      @check
     </code>
     事件可以监听节点勾选状态的变化。事件回调函数会返回当前勾选的节点数据和节点对象。
    </p>
    <pre><code class="prism language-javascript"><span class="token literal-property property">methods</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
  <span class="token function">onCheckChange</span><span class="token punctuation">(</span><span class="token parameter">checkedData<span class="token punctuation">,</span> checkedStatus</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">'当前勾选状态:'</span><span class="token punctuation">,</span> checkedStatus<span class="token punctuation">)</span><span class="token punctuation">;</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     参数说明：
    </p>
    <ul>
     <li>
      <code>
       checkedData
      </code>
      ：当前勾选的节点数据。
     </li>
     <li>
      <code>
       checkedStatus
      </code>
      ：包含勾选状态信息的对象，如
      <code>
       checkedKeys
      </code>
      （勾选的节点 ID 数组）、
      <code>
       halfCheckedKeys
      </code>
      （半勾选的节点 ID 数组）等。
     </li>
    </ul>
