    <h3>
     一、Linux背景与发展
    </h3>
    <h4>
     1. 发展史
    </h4>
    <ul>
     <li>
      <p>
       1968年，研究人员开发了Multics操作系统，为后续发展奠定了基础。
      </p>
     </li>
     <li>
      <p>
       1969−1970年，Ken Thompson和Dennis Ritchie在Multics基础上开发了UNIX系统。
      </p>
     </li>
     <li>
      <p>
       1991年，Linus Torvalds发布了Linux操作系统，最初版本为0.02，逐步发展至今。
      </p>
     </li>
    </ul>
    <h4>
     2. 开源
    </h4>
    <ul>
     <li>
      <p>
       Linux是自由和开放源代码的类UNIX操作系统，遵循GNU通用公共许可协议（GPL）。
      </p>
     </li>
     <li>
      <p>
       任何个人和机构都可以自由使用、修改和再发布Linux源代码，推动了Linux的广泛应用和发展。
      </p>
     </li>
    </ul>
    <h3>
     二、企业应用现状
    </h3>
    <h4>
     1. 服务器领域
    </h4>
    <ul>
     <li>
      <p>
       Linux在服务器市场占据重要地位，据权威部门统计，目前Linux在服务器领域已经占据75%的市场份额。
      </p>
     </li>
     <li>
      <p>
       广泛应用于政府、金融、农业、交通、电信等关键领域。
      </p>
     </li>
    </ul>
    <h4>
     2. 桌面领域
    </h4>
    <ul>
     <li>
      <p>
       近年国内Linux桌面操作系统发展迅猛，如中标麒麟Linux、红旗Linux、深度Linux等系统软件厂商推出了多款Linux桌面操作系统。
      </p>
     </li>
     <li>
      <p>
       虽然在系统易用性、软件丰富程度等方面与Windows还有一定差距，但随着技术进步，Linux桌面系统的竞争力在逐步提升。
      </p>
     </li>
    </ul>
    <h4>
     3. 移动嵌入式领域
    </h4>
    <ul>
     <li>
      <p>
       Linux的低成本、强定制功能和良好移植性，使其在嵌入式系统中广泛应用，如手机、平板电脑、路由器、电视和电子游戏机等。
      </p>
     </li>
     <li>
      <p>
       安卓系统就是基于Linux内核开发的，目前在全球智能手机市场占据较大份额。
      </p>
     </li>
    </ul>
    <h4>
     4. 云计算/大数据领域
    </h4>
    <ul>
     <li>
      <p>
       随着互联网产业的发展，云计算、大数据产业形成并快速发展，Linux凭借其开源优势，在这些领域占据核心地位。
      </p>
     </li>
     <li>
      <p>
       据Linux基金会研究，86%的企业使用Linux操作系统构建云计算、大数据平台。
      </p>
     </li>
    </ul>
    <h3>
     三、发行版本
    </h3>
    <h4>
     1. Debian
    </h4>
    <ul>
     <li>
      <p>
       运行稳定，适合服务器使用，但对新手用户不太友好，更适合系统管理员和高级用户。
      </p>
     </li>
    </ul>
    <h4>
     2. Ubuntu
    </h4>
    <ul>
     <li>
      <p>
       Debian的衍生版，最受欢迎的免费操作系统之一，在服务器、云计算、移动设备上都很常见，对新手用户较为友好。
      </p>
     </li>
    </ul>
    <h4>
     3. 红帽企业级Linux
    </h4>
    <ul>
     <li>
      <p>
       面向商业市场的Linux发行版，有服务器版本，支持众多处理器架构，通过课程和认证培养系统管理员。
      </p>
     </li>
    </ul>
    <h4>
     4. CentOS
    </h4>
    <ul>
     <li>
      <p>
       企业级Linux发行版，基于红帽企业级Linux的免费源代码构建，外观和行为与红帽企业级Linux相似，适合不想支付高额费用又想体验红帽企业级Linux的用户。
      </p>
     </li>
    </ul>
    <h4>
     5. Fedora
    </h4>
    <ul>
     <li>
      <p>
       红帽公司的测试平台，适合想尝试最新技术的用户，有庞大的用户论坛和丰富的软件包。
      </p>
     </li>
    </ul>
    <h4>
     6. Kali Linux
    </h4>
    <ul>
     <li>
      <p>
       Debian的衍生版，主要用于渗透测试，随带许多渗透测试工具，是文明黑客青睐的操作系统。
      </p>
     </li>
    </ul>
    <h3>
     四、系统概念与操作
    </h3>
    <h4>
     1. 操作系统根本要义
    </h4>
    <ul>
     <li>
      <p>
       使计算机更好用，这是操作系统的根本要义，Linux系统也不例外，旨在提供高效、稳定、安全的计算环境。
      </p>
     </li>
    </ul>
    <h4>
     2. 使用XShell远程登录Linux
    </h4>
    <ul>
     <li>
      <p>
       XShell是一个远程终端软件，可用于远程登录Linux主机。
      </p>
     </li>
     <li>
      <p>
       下载安装时选择"home/school"为免费版本。
      </p>
     </li>
     <li>
      <p>
       查看Linux主机IP地址可使用
       <code>
        ifconfig
       </code>
       指令。
      </p>
     </li>
     <li>
      <p>
       使用
       <code>
        ssh [ip]
       </code>
       指令登录主机，网络畅通后会提示输入用户名和密码。
      </p>
     </li>
    </ul>
    <p>
     在Linux中我们经常要写一些指令
    </p>
    <p>
     下面这是一些常用的指令，今天先说一下这些指令：
    </p>
    <h3>
     一、文件和目录操作类命令
    </h3>
    <h4>
     1.ls
    </h4>
    <p>
     用于列出目录中的文件和文件夹。使用
     <code>
      ls -l
     </code>
     可以以长格式列出文件和文件夹的详细信息，包括权限、所有者、大小等。
    </p>
    <h4>
     2.cd
    </h4>
    <p>
     用于切换当前目录。例如，
     <code>
      cd Documents
     </code>
     可以进入Documents目录。
    </p>
    <h4>
     3.pwd
    </h4>
    <p>
     用于显示当前工作目录的路径。
    </p>
    <h4>
     4.cp
    </h4>
    <p>
     用于复制文件或目录。例如，
     <code>
      cp file.txt backup.txt
     </code>
     将file.txt复制为backup.txt。
    </p>
    <h4>
     5.mv
    </h4>
    <p>
     用于移动或重命名文件或目录。例如，
     <code>
      mv oldname.txt newname.txt
     </code>
     将文件重命名为newname.txt。
    </p>
    <h4>
     6.rm
    </h4>
    <p>
     用于删除文件或目录。例如，
     <code>
      rm file.txt
     </code>
     删除file.txt文件。
    </p>
    <h4>
     7.mkdir
    </h4>
    <p>
     用于创建新目录。例如，
     <code>
      mkdir NewFolder
     </code>
     创建名为NewFolder的目录。
    </p>
    <h4>
     8.rmdir
    </h4>
    <p>
     用于删除空目录。例如，
     <code>
      rmdir EmptyFolder
     </code>
     删除名为EmptyFolder的空目录。
    </p>
    <h3>
     二、文件查看类命令
    </h3>
    <h4>
     1.cat
    </h4>
    <p>
     用于显示文件内容。例如，
     <code>
      cat file.txt
     </code>
     显示file.txt的内容。
    </p>
    <h4>
     2.more
    </h4>
    <p>
     用于分页显示文件内容。例如，
     <code>
      more file.txt
     </code>
     逐页显示file.txt的内容。
    </p>
    <h4>
     3.less
    </h4>
    <p>
     类似于more，但可以向前和向后滚动。例如，
     <code>
      less file.txt
     </code>
     。
    </p>
    <h4>
     4.head
    </h4>
    <p>
     用于显示文件的前几行（默认10行）。例如，
     <code>
      head file.txt
     </code>
     。
    </p>
    <h4>
     5.tail
    </h4>
    <p>
     用于显示文件的后几行（默认10行）。例如，
     <code>
      tail file.txt
     </code>
     。
    </p>
    <h3>
     三、文本处理类命令
    </h3>
    <h4>
     1.grep
    </h4>
    <p>
     在文件中搜索特定的模式或字符串。例如，
     <code>
      grep "error" log.txt
     </code>
     在log.txt中搜索包含"error"的行。
    </p>
    <h4>
     2.wc
    </h4>
    <p>
     统计文件的行数、单词数和字节数。例如，
     <code>
      wc -l file.txt
     </code>
     显示file.txt的行数。
    </p>
    <h3>
     四、系统信息查看类命令
    </h3>
    <h4>
     1.uname
    </h4>
    <p>
     显示系统信息。例如，
     <code>
      uname -a
     </code>
     显示所有系统信息。
    </p>
    <h4>
     2.df
    </h4>
    <p>
     显示磁盘空间使用情况。例如，
     <code>
      df -h
     </code>
     以人类可读的格式显示磁盘使用情况。
    </p>
    <h4>
     3.free
    </h4>
    <p>
     显示系统的内存使用情况。例如，
     <code>
      free -m
     </code>
     以MB为单位显示内存使用情况。
    </p>
    <h3>
     五、文件权限和用户管理类命令
    </h3>
    <h4>
     1.chmod
    </h4>
    <p>
     更改文件或目录的权限。例如，
     <code>
      chmod 755 file.txt
     </code>
     将file.txt的权限设置为rwxr-xr-x。
    </p>
    <h4>
     2.chown
    </h4>
    <p>
     更改文件或目录的所有者。例如，
     <code>
      chown user:group file.txt
     </code>
     将file.txt的所有者更改为user，所属组更改为group。
    </p>
    <h4>
     3.useradd
    </h4>
    <p>
     创建新用户。例如，
     <code>
      useradd username
     </code>
     创建名为username的新用户。
    </p>
