    <p>
     这道题，每个柜子都有很多个格子，但是具体多少格子是不确定的，我们知道的是格子的总数一共不超过10的7次方
    </p>
    <p>
     这道题，如果我们用数组的话，10的五次方乘10的五次方是一定会超时的
    </p>
    <p>
     所以我们只能用vector来解决这道问题，一共才会开不到10的7次方个格子
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;vector&gt;
using namespace std;
const int N = 1e5+10;
//int a[N][N]//空间太大，程序运行不了
vector&lt;int&gt; a[N];//正确√
int n,q;

int main()
{
	cin &gt;&gt; n &gt;&gt; q;
	while(q--)
	{
		int op,i,j;cin &gt;&gt; op &gt;&gt; i &gt;&gt; j;
		if(op == 1)
		{
			int x;cin &gt;&gt; x;
			if(a[i].size() &lt; j)
			{
				a[i].resize(j+1);
			}
			a[i][j] = x;
			
		}
		else if(op == 2)
		{
			cout &lt;&lt; a[i][j] &lt;&lt; endl;
		}
	}
	
	
	
	return 0; 
 } </code></pre>
