    <h3>
     <a id="sfrsbit_5">
     </a>
     sfr与sbit？
    </h3>
    <p>
     这第一课咱们主要来先理解一下sfr与sbit，以下可能是咱们这些新手朋友常见的点灯代码：
    </p>
    <pre><code class="prism language-c"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span><span class="token string">&lt;regx52.h&gt;</span><span class="token comment">//reg52.h只定义sfr而没定义sbit可寻址空间的位变量</span></span>
<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
	<span class="token keyword">while</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">)</span>
	<span class="token punctuation">{<!-- --></span>
		P2_0 <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span><span class="token comment">//P几取决于宁板子上led的对应的gpio</span>
	<span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <p>
     或者是更底层更难理解一点的：
    </p>
    <pre><code class="prism language-c">sfr P2<span class="token operator">=</span><span class="token number">0xA0</span><span class="token punctuation">;</span>
sbit P2_0<span class="token operator">=</span>P2<span class="token operator">^</span><span class="token number">0</span><span class="token punctuation">;</span>
<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
	<span class="token keyword">while</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">)</span>
	<span class="token punctuation">{<!-- --></span>
		P2_0<span class="token operator">=</span><span class="token number">0</span><span class="token punctuation">;</span><span class="token comment">//赋值也有可能是1，这主要取决于led的二极管的连接电路</span>
	<span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <p>
     可能有一些只有c语言基础的朋友就已经开始迷惑了，sfr和sbit是什么？
     <br/>
     先来打个比方：想象单片机是一个储存有许多不同魔法的魔法书，每张书页都对应一个特殊功能：有的控制灯光（GPIO），有的管理时钟（Timer）。
     <strong>
      而sfr（特殊功能寄存器）​就是这些这本书的页码
     </strong>
     。在单片机的sfr寄存器中有着许许多多的空间，空间里充斥着二进制数，这些二进制数控制着单片机接口输出高低电平，当然空间里的二进制数也会因为接口接收的高低电平改变；而
     <strong>
      sfr，一款由keil编译器开发的扩展关键字
     </strong>
     ，
     <mark>
      它的作用类似于c语言中的取别名define
     </mark>
     ，
     <strong>
      只不过它取别名的对象是储存八位二进制数的寄存器中的地址
     </strong>
     ，比如咱们点灯用到的P2接口(GPIO，General Purpose Input Output ，通用功能输出输出 )所对应的sfr的地址就是0xA0，这个地址储存了一个八位二进制数，对应了八个led（对于一般的51单片机板子而言，咱手中这块是40脚的stc89c52rc）；但是，目前咱们只想点一个led怎么办呢？所以咱们需要
     <strong>
      另一款keil扩展关键字sbit
     </strong>
     ，它的操作与^(规范的位寻址运算符)绑定，
     <mark>
      功能是也是取别名
     </mark>
     ，
     <strong>
      但对象是sfr特殊寄存器地址中的某一位
     </strong>
     ，这样咱们就可以直接操控确定的一位二进制数。
    </p>
    <p>
     知道了sfr和sbit的作用与简单gpio的功能引脚定义，咱们现在就可以写出一些花活来：
    </p>
    <pre><code class="prism language-c">sfr LED_GPIO<span class="token operator">=</span><span class="token number">0xA0</span><span class="token punctuation">;</span>
sbit LED0<span class="token operator">=</span>P2<span class="token operator">^</span><span class="token number">0</span><span class="token punctuation">;</span>
<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
	<span class="token keyword">while</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">)</span>
	<span class="token punctuation">{<!-- --></span>
		LED0<span class="token operator">=</span><span class="token number">0</span><span class="token punctuation">;</span>
	<span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     当然这是显然不规范的，蛤蛤蛤，如果是咱们自己兴趣的项目这样写写无伤大雅，但是在企业化的规范化开发中这显然是不能通过检查的。
    </p>
    <p>
     但其实，像c中的头文件引用一样，单片机的开发也有自己的头文件，在51单片机系列中，通用的头文件&lt;regx52.h&gt;或者是&lt;reg52.h&gt;,前者既定义了sfr有定义了sbit；后者仅定义了sfr；这里藏着单片机开发的第一个隐藏关卡：
    </p>
    <blockquote>
     <p>
      ​reg52.h：标准AT89C52头文件 仅定义SFR基地址，如sfr P1 = 0x90; 需要手动位定义：sbit P1_0 =
      <br/>
      P1^0;（很多新手漏掉这一步导致编译报错）
     </p>
    </blockquote>
    <h3>
     <a id="delay_ms_56">
     </a>
     不靠定时器的delay_ms延时函数
    </h3>
    <p>
     在单片机编程中，延时是最基础的需求。比如让LED闪烁时，需要点亮后保持一段时间再熄灭。但单片机没有现成的delay_ms()函数，需要手动实现。
     <strong>
      咱们这里使用的方式是利用空循环消耗CPU时间
     </strong>
     。
    </p>
    <p>
     示例代码分析：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">void</span> <span class="token function">delay_ms</span><span class="token punctuation">(</span><span class="token keyword">unsigned</span> <span class="token keyword">int</span> ms<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">unsigned</span> <span class="token keyword">int</span> i<span class="token punctuation">,</span> j<span class="token punctuation">;</span><span class="token comment">//unsigned关键字占用内存更少，符合单片机资源紧张需要空间高效利用性</span>
    <span class="token keyword">for</span><span class="token punctuation">(</span>i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> ms<span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span>
        <span class="token keyword">for</span><span class="token punctuation">(</span>j <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> j <span class="token operator">&lt;</span> <span class="token number">680</span><span class="token punctuation">;</span> j<span class="token operator">++</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 580次空操作循环约为1ms</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_69">
     </a>
     所谓寄存器
    </h3>
    <p>
     举个不恰当的例子，寄存器就相当于一个api，咱们只需要知道参数(功能引脚初始化定义和sfr何如初始化设置)就可以玩转，在以后的学习中咱会以“api参数”为起始然后再阐述编程的框架！
    </p>
