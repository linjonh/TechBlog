    <p>
     前段时间在BOSS直聘上发了个前端工程师的招聘岗位，想要招一个前端，JD发出来24小时，未读消息
     <strong>
      99+

    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/da979111af33ada3438165bab74a6183.png">
      <br/>
      <strong>
       前端行业面临着前所未有的惨烈竞争
      </strong>
      ，但尽管如此，我还是看到了很多前端工程师，夜以继日的继续研究八股文，学习性能优化，了解算法。
     </img>
    </p>
    <h3>
     <a id="_5">
     </a>
     有用吗？有用，但用处不大。
    </h3>
    <p>
     三大框架Angular、React、Vue已经统治了前端领域接近10年，
     <strong>
      前端发展到今天，已经很久没有出新东西了
     </strong>
     。
    </p>
    <h3>
     <a id="_9">
     </a>
     没有创新，这个领域会变成红海。
    </h3>
    <p>
     我非常幸运。
     <strong>
      12年前
     </strong>
     学习编程的时候，选择了前端，学习了JS。
     <strong>
      8年前
     </strong>
     Nodejs兴起的时候，借着对JS的熟悉，很快学会了后端开发。
     <strong>
      7年前
     </strong>
     参加杭州D2前端大会，回来之后用Electron做了几款桌面应用。
     <strong>
      6年前
     </strong>
     React Native大放异彩的时候，我使用RN将一款APP从日活100做到了日活5w，熟悉了移动开发。
     <strong>
      5年前
     </strong>
     ，辞职创办了一家公司。
     <strong>
      4年前
     </strong>
     Vue3发布，借机用上了TS（TypeScript）。
    </p>
    <h3>
     <a id="HarmonyOSArkTS_13">
     </a>
     如今HarmonyOS大热，使用的是ArkTS。
    </h3>
    <p>
     <strong>
      ArkTS继承了TS的所有特性，是TS的超集
     </strong>
     。前端圈子非常流行的一个定律叫
     <strong>
      Atwood定律，Atwood定律
     </strong>
     ：“任何可以使用JavaScript来编写的应用，最终会由JavaScript编写。”
    </p>
    <h3>
     <a id="HarmonyOS_17">
     </a>
     为什么说如今HarmonyOS应用开发就是前端转行的一个绝佳的机会呢？
    </h3>
    <p>
     <strong>
      主要原因有以下三点：
     </strong>
    </p>
    <p>
     <strong>
      1. 同样是转岗
     </strong>
    </p>
    <p>
     iOS开发和Android开发需要再学习TS；学习JS；学习Promise；学习什么是状态机；学习什么是声明式UI。
     <strong>
      而对于前端来说，这一切都是轻车熟路。
     </strong>
    </p>
    <h3>
     <a id="2__25">
     </a>
     2. 鸿蒙应用开发岗位现在特别缺人
    </h3>
    <p>
     属于蓝海市场，
     <strong>
      稍微学习2-3个月就可以转岗去做HarmonyOS应用开发了
     </strong>
     ，先上车，后面再转做性能、底层、业务等领域。机会很多。
    </p>
    <h3>
     <a id="3_HarmonyOS_29">
     </a>
     3. 在HarmonyOS没有出现之前
    </h3>
    <p>
     移动端操作系统是Android和iOS的天下，对于国家来说有极度的不安全感。急需要一个甚至几个国产的操作系统作为替代品，现在华为的鸿蒙，小米的澎湃属于第一梯队。但华为显然跑的更快一些，华为手机现在从底层的制造到芯片到上层的方舟编译器、ArkTS以及花瓣支付。还剩最后一环操作系统如果打通，将实现完全100%自研，这是大势所趋、国家意志。
    </p>
