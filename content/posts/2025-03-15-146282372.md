    <h3>
     <a id="C_0f__0">
     </a>
     <code>
      C
     </code>
     语言
     <code>
      %.0f
     </code>
     的使用 ⭐️
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3ada6fca9284410c96d0b63f036398a5.jpeg"/>
    </p>
    <blockquote>
     <p>
      <code>
       %.0f
      </code>
      在格式化输出时会 对
      <strong>
       浮点数
      </strong>
      进行
      <mark>
       四舍五入
      </mark>
      ，并
      <strong>
       保留
       <code>
        0
       </code>
       位小数
      </strong>
      。
     </p>
     <ul>
      <li>
       <code>
        %f
       </code>
       是格式化输出浮点数的占位符。
      </li>
      <li>
       <code>
        .0
       </code>
       表示保留
       <code>
        0
       </code>
       位小数。
      </li>
      <li>
       在保留
       <code>
        0
       </code>
       位小数时，
       <code>
        printf
       </code>
       会对小数部分进行四舍五入，然后只输出整数部分。
      </li>
     </ul>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/af9097072f892cb793d19e98fdc0c81c.png">
      <br/>
      <strong>
       示例 1：
      </strong>
      🌰
     </img>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token number">12.49</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 12</span>
</code></pre>
    <blockquote>
     <p>
      12.49 的小数部分是 0.49，
      <code>
       小于 0.5
      </code>
      ，因此四舍五入后输出
      <code>
       12
      </code>
      。
     </p>
    </blockquote>
    <p>
     <strong>
      示例 2：
     </strong>
     🌰
    </p>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token number">12.50</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 13</span>
</code></pre>
    <blockquote>
     <p>
      12.50 的小数部分是 0.50，
      <code>
       等于 0.5
      </code>
      ，因此四舍五入后输出
      <code>
       13
      </code>
      。
     </p>
    </blockquote>
    <p>
     <strong>
      示例 3：
     </strong>
     🌰
    </p>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token number">12.51</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 13</span>
</code></pre>
    <blockquote>
     <p>
      12.51 的小数部分是 0.51，
      <code>
       大于 0.5
      </code>
      ，因此四舍五入后输出
      <code>
       13
      </code>
      。
     </p>
    </blockquote>
    <h3>
     <a id="__33">
     </a>
     注意事项 ⭐️
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/25add7306b6d4bf8ae0640c1341f289a.jpeg"/>
    </p>
    <h4>
     <a id="1_35">
     </a>
     1、四舍五入规则：🍭
    </h4>
    <blockquote>
     <p>
      小数部分
      <code>
       小于 0.5
      </code>
      时，
      <code>
       向下舍入
      </code>
      。
      <br/>
      小数部分
      <code>
       大于或等于 0.5
      </code>
      时，
      <code>
       向上舍入
      </code>
      。
     </p>
    </blockquote>
    <h4>
     <a id="2_40">
     </a>
     2、负数的情况：🍭
    </h4>
    <blockquote>
     <p>
      对于负数，四舍五入规则同样适用。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/af9097072f892cb793d19e98fdc0c81c.png"/>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token operator">-</span><span class="token number">12.49</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 -12</span>
</code></pre>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token operator">-</span><span class="token number">12.50</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 -13</span>
</code></pre>
    <h4>
     <a id="3_55">
     </a>
     3、精度问题：🍭
    </h4>
    <blockquote>
     <p>
      浮点数在计算机中是以二进制形式存储的，可能存在精度问题。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/af9097072f892cb793d19e98fdc0c81c.png"/>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token number">0.5</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> x<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 0（可能因为精度问题导致四舍五入错误）</span>
</code></pre>
    <p>
     <mark>
      如果需要更精确的四舍五入，可以使用
      <code>
       round
      </code>
      函数：
     </mark>
    </p>
    <pre><code class="prism language-c"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;math.h&gt;</span></span>
<span class="token keyword">double</span> x <span class="token operator">=</span> <span class="token number">0.5</span><span class="token punctuation">;</span>
<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"%.0f\n"</span><span class="token punctuation">,</span> <span class="token function">round</span><span class="token punctuation">(</span>x<span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 1</span>
</code></pre>
