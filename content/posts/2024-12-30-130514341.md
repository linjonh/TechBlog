    <p>
     问题：升级 Android Studio Flamingo 出现如下报错信息
    </p>
    <pre><code class="prism language-bash">Unsupported Java. 
Your build is currently configured to use Java <span class="token number">17.0</span>.2 and Gradle <span class="token number">7.0</span>.2.

Possible solution:
 - Open Gradle wrapper settings, change <span class="token variable"><span class="token variable">`</span>distributionUrl<span class="token variable">`</span></span> property to use compatible Gradle version and reload the project
</code></pre>
    <p>
     解决方法：File -&gt; Settings -&gt; Build, Execution, Deployment -&gt; Build Tools -&gt; Gradle
    </p>
    <p>
     将 Gradle JDK 修改为本机安装的 JDK 版本即可，如: JDK-11.0.15
    </p>
