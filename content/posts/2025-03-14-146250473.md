     基于UniApp + Vue3开发的智能汉字转拼音工具
    </h2>
    <h3>
     <a id="_3">
     </a>
     项目简介
    </h3>
    <p>
     这是一个基于 UniApp + Vue3 开发的智能汉字转拼音工具，前端使用 Vue3 构建界面，后端采用 Classic ASP 提供接口支持，通过
     <code>
      pinyin-pro
     </code>
     库实现精准的中文转拼音功能。本工具支持以下特性：
    </p>
    <ul>
     <li>
      支持单个汉字、词组、句子的智能转换
     </li>
     <li>
      支持带声调和不带声调的转换模式
     </li>
     <li>
      支持多音字的智能识别处理
     </li>
     <li>
      支持繁体字自动转换
     </li>
     <li>
      支持多种转换选项自由组合
     </li>
    </ul>

    <h3>
     <a id="_24">
     </a>
     使用指南
    </h3>
    <ol>
     <li>
      在输入框中输入需要转换的中文文本
     </li>
     <li>
      根据需求选择转换选项:
      <ul>
       <li>
        空格间隔：拼音之间是否添加空格
       </li>
       <li>
        包括声调：是否显示声调标记
       </li>
       <li>
        方格显示：每个字单独显示
       </li>
       <li>
        清除标点：去除文本中的标点符号
       </li>
       <li>
        只提取汉字：仅转换汉字部分
       </li>
      </ul>
     </li>
     <li>
      点击"转换"按钮即可查看转换结果
     </li>
    </ol>
