    <h4>
     <a id="_SDK_0">
     </a>
     一.
     <strong>
      大华SDK协议详解
     </strong>
    </h4>
    <p>
     一、大华SDK协议概述
     <br/>
     大华SDK（Software Development Kit）协议是大华股份为开发者提供的一套软件开发工具包，旨在帮助开发者快速集成大华设备（如摄像头、NVR、DVR等）的功能，实现视频监控、设备管理、数据采集等操作。该协议基于TCP/IP协议栈，支持多种通信方式（如HTTP、RTSP、ONVIF等），并提供了丰富的API接口，适用于多种开发语言（如C++、Java、Python等）。
    </p>
    <p>
     二、大华SDK协议的原理
    </p>
    <ol>
     <li>
      通信架构
     </li>
    </ol>
    <ul>
     <li>
      大华SDK协议采用客户端-服务器架构，设备作为服务器端，开发者通过SDK作为客户端与设备进行交互。
     </li>
     <li>
      通信协议包括设备发现、设备登录、视频流传输、控制指令下发等模块，确保数据的可靠传输和实时交互。
     </li>
    </ul>
    <ol start="2">
     <li>
      数据流与协议栈
     </li>
    </ol>
    <ul>
     <li>
      视频流传输：支持RTSP、RTP等协议，实现实时视频流的传输与解码。
     </li>
     <li>
      控制指令：通过HTTP或私有协议下发控制指令（如PTZ控制、报警配置等）。
     </li>
     <li>
      数据加密：支持SSL/TLS加密，确保通信安全。
     </li>
    </ul>
    <ol start="3">
     <li>
      API接口设计
     </li>
    </ol>
    <ul>
     <li>
      提供设备管理、视频播放、报警处理、日志查询等模块的API接口。
     </li>
     <li>
      接口设计遵循模块化原则，便于开发者按需调用。
     </li>
    </ul>
    <p>
     三、大华SDK协议的优势
    </p>
    <ol>
     <li>
      兼容性强
      <br/>
      -支持多种操作系统（如Windows、Linux、Android、iOS）和开发语言，满足不同开发环境的需求。
     </li>
     <li>
      功能丰富
      <br/>
      提供视频监控、设备管理、报警处理、数据分析等一站式解决方案，减少开发者的工作量。
     </li>
     <li>
      性能高效
      <br/>
      优化的视频流传输机制，支持高帧率、低延迟的视频播放。
     </li>
     <li>
      安全性高
      <br/>
      支持多种加密方式，确保数据传输和设备访问的安全性。
      <br/>
      四、大华SDK协议的劣势
     </li>
     <li>
      学习成本较高
      <br/>
      由于功能丰富，开发者需要花费一定时间熟悉API接口和协议细节。
     </li>
     <li>
      私有协议限制
      <br/>
      部分功能依赖大华私有协议，与其他厂商设备的兼容性较差。
     </li>
     <li>
      更新维护依赖厂商
      <br/>
      SDK的更新和维护完全依赖大华股份，开发者无法自主优化协议。
     </li>
    </ol>
    <h4>
     <a id="__39">
     </a>
     二.
     <strong>
      智联视频超融合平台介绍
     </strong>
    </h4>
    <ul>
     <li>
      智联视频超融合平台通过GB/T28181-2011/2016/2022、国网B接口、RTSP、RTMP、ONVIF、GB/T35114、GA1400、海康大华SDK/API等方式，接入海康、大华、宇视等各个厂家的摄像机、录像机，接入直播手机，接入下级平台，提供web客户端进行视频预览、录像回放、配置管理和智慧运维。
     </li>
     <li>
      提供标准RESTful格式的HTTP API，提供web播放器demo，满足二次开发需求。同时支持将音视频数据等转发成各种通用标准的流媒体协议，方便第三方客户端调用展示。
     </li>
     <li>
      支持按照GB/T28181-2011/2016/2022、国网B接口、GB/T35114、GA1400等协议对接到上级平台，包括智联视频云平台和第三方平台。支持普通级联，多级级联，混合级联等多种级联方式。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6ae2cfd7475046af8042d395195555e4.png#pic_center">
       <br/>
       智联视频超融合平台目前已经在公安、电力、交通、教育、物联网、智慧城市、智慧园区等多个行业得到了广泛应用。
      </img>
     </li>
    </ul>
    <h4>
     <a id="_SDK_46">
     </a>
     三.
     <strong>
      智联视频超融合平台接入大华SDK协议步骤
     </strong>
    </h4>
    <ol>
     <li>
      在智联视频超融合平台配置管理页面，选择添加设备，接入协议选择“大华私有协议”：
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5e58f2f01b7b4ee98da0224f5f13815c.jpeg#pic_center"/>
     </li>
     <li>
      填写20位设备ID，输入设备的用户名与密码，输入设备IP加端口号（大华SDK协议默认端口号为37777），设置设备名称（如不填写设备名称默认为设备ID）。
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/404abd08483f4e4fafd4350d7a7c506b.jpeg#pic_center"/>
     </li>
     <li>
      在智联视频超融合平台配置设备管理页面，查看添加的设备是否为在线状态
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e0de049b026a41618ea17a2bc9d469b1.jpeg#pic_center"/>
     </li>
     <li>
      这样就能在智联视频超融合平台WEB客户端看到大华SDK图像信息了
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c63f35eae57f4d438af5d79dfb49dc17.jpeg#pic_center"/>
     </li>
     <li>
      第三方平台可以通过FLV、HLS、RTMP、RTSP、WebRTC等标准协议获取RTMP音视频信息，同时也能通过国标28181、国网B接口等标准协议，将RTMP音视频转发到上级平台中。
     </li>
    </ol>
