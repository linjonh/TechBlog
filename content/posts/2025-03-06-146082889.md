     Scala：解构声明（用例子通俗易懂）
    </h1>

  </div>

  <div class="article_content clearfix" id="article_content">
   <link href="./../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      一句话解释
     </strong>
     ：
     <br/>
     解构声明就是“拆包裹”——把打包好的数据（比如元组、列表）拆开，把里面的值直接赋给变量。
    </p>
    <p>
     ——
    </p>
    <h4 id="举个栗子 🌰（用你代码中的例子）">
     举个栗子 🌰
    </h4>
    <p>
     假设你的代码里有这样一行：
    </p>
    <p>
     scala复制代码
    </p>
    <pre><code>val (math, english, physics) = scores.unzip3</code></pre>
    <p>
     这里发生了什么？
    </p>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         scores
        </code>
        的结构
       </strong>
       ：
      </p>
      <p>
       scala复制代码
      </p>
      <pre><code>// 假设scores是一个包含多个三元组的列表，比如：
List( (80,64,78), (65,87,58), (44,71,77) )</code></pre>
      <ul>
       <li>
        每个三元组代表一个学生的三门课成绩，比如
        <code>
         (80,64,78)
        </code>
        表示数学80，英语64，物理78
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         unzip3
        </code>
        的作用
       </strong>
       ：
      </p>
      <ul>
       <li>
        把“粘在一起”的三门课成绩拆分成三个独立的列表：
        <p>
         scala复制代码
        </p>
        <pre><code>( 
  List(80, 65, 44),    // 所有数学成绩
  List(64, 87, 71),    // 所有英语成绩
  List(78, 58, 77)     // 所有物理成绩
)</code></pre>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        解构赋值
       </strong>
       ：
      </p>
      <p>
       scala复制代码
      </p>
      <pre><code>val (math, english, physics) = 拆出来的三个列表</code></pre>
      <ul>
       <li>
        <code>
         math
        </code>
        =
        <code>
         List(80, 65, 44)
        </code>
        （所有数学成绩）
       </li>
       <li>
        <code>
         english
        </code>
        =
        <code>
         List(64, 87, 71)
        </code>
        （所有英语成绩）
       </li>
       <li>
        <code>
         physics
        </code>
        =
        <code>
         List(78, 58, 77)
        </code>
        （所有物理成绩）
       </li>
      </ul>
     </li>
    </ol>
    <p>
     相当于把三个包裹📦一次性拆开，把里面的东西分别放进三个盒子！
    </p>
    <p>
     ——
    </p>
    <h4 id="为什么要用解构？">
     为什么要用解构？
    </h4>
    <ol>
     <li>
      <strong>
       代码更短
      </strong>
      ：不用写三行分别取每个列表
     </li>
     <li>
      <strong>
       更直观
      </strong>
      ：变量名直接对应科目（math, english, physics）
     </li>
    </ol>
    <p>
     ——
    </p>
    <h4 id="其他常见场景（举个超简单例子）">
     其他常见场景（举个超简单例子）
    </h4>
    <p>
     scala复制代码
    </p>
    <pre><code>// 场景1：元组解构
val 学生信息 = ("张三", 18, 90)
val (姓名, 年龄, 分数) = 学生信息  // 直接拆开
println(姓名) // 输出"张三"

// 场景2：列表解构
val List(第一科, 第二科, _*) = List("数学", "英语", "物理", "化学")
println(第一科) // "数学"
println(第二科) // "英语"
// _* 表示剩下的科目不管了</code></pre>
    <p>
     ——
    </p>
    <h4 id="注意事项 ⚠️">
     注意事项 ⚠️
    </h4>
    <ul>
     <li>
      <strong>
       形状要对得上
      </strong>
      ：左边变量个数要和右边数据结构的元素个数一致
      <p>
       scala复制代码
      </p>
      <pre><code>// 错误示范：元组只有3个元素，左边却要拆成4个变量
val (a,b,c,d) = (1,2,3) // 会报错！</code></pre>
     </li>
    </ul>
    <p>
     这下应该明白啦！其实就是把打包的数据拆开用的快捷方式~ 😄
    </p>
