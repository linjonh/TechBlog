# java Operator ‘/‘ cannot be applied to ‘java.math.BigInteger‘, ‘int‘

Operator '/' cannot be applied to 'java.math.BigInteger', 'int'

这是因为BigInteger不能直接做数学运算，需要先转为int

BigInter a = 1;

int b = a.intValue();
