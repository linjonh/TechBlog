     准备
    </h3>
    <p>
     雷赛MC508CS及LeadSysStudio【
     <a href="https://www.leisai.com/cn/cpzx/info_34_itemid_3324_lcid_1_lcids_127.html" rel="nofollow">
      LeadSysStudioV3.0&amp;0318升级包
     </a>
     】
     <br/>
     LabVIEW2024 32位
     <br/>
     Uaexpert客户端测试工具【
     <a href="https://www.unified-automation.com/downloads.html" rel="nofollow">
      Uaexpert下载
     </a>
     】
     <br/>
     LabVIEW OPC UA工具包 2022Q4【
     <a href="https://www.ni.com/zh-cn/support/downloads/software-products/download.labview-opc-ua-toolkit.html?srsltid=AfmBOopVcu5e1ALcxoDa5SYIBAwJzl89We6ssJxb3aCV3S2sUUDWqWQP#467615" rel="nofollow">
      LabVIEW OPC UA工具包下载
     </a>
     】
    </p>
    <blockquote>
     <p>
      OPC UA 工具包包含 OPC UA API，该 API 以前是 LabVIEW 数据记录和监控 (DSC) 模块和 LabVIEW 实时模块的一部分。从 2017 版本开始，OPC UA Toolkit 成为独立产品。
     </p>
     <p>
      安装路径：C:\Program Files\NI\LVAddons\niopcua
     </p>
    </blockquote>
    <h3>
     <a id="_11">
     </a>
     步骤
    </h3>
    <h4>
     <a id="LeadSysStudio_12">
     </a>
     LeadSysStudio
    </h4>
    <p>
     将需要进行 OPC 通信的变量定义在全局变量中
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7b613d9d074b48af966ec61c4db81ba2.png">
      <br/>
      在程序中使用全局变量（因为未被使用的变量， 不会出现在通信列表中）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d46f3272ac6046eda0a2ac61b65cbae0.png">
       <br/>
       【Application】→【添加对象】→【符号配置】→在弹出的界面中勾选【支持
       <br/>
       OPC UA 特征】→【打开】
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/02ee608da21c447aa7614c2f6046592d.png">
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fb0bcd8bae48452086f2d69c7e1aecbe.png"/>
       </img>
      </img>
     </img>
    </p>
    <p>
     在符号配置表中点击“编译”
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6ebfde9cec384a91a9d54ac0dda2f16b.png">
      <br/>
      选择用于通讯的变量。 记得勾选
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/131d1137f2174b91a4d4b459cf7a1241.png"/>
     </img>
    </p>
    <h4>
     <a id="UaExpert_26">
     </a>
     UaExpert客户端测试
    </h4>
    <p>
     打开【 UaExpert】 后，右键【 Server】，选择【 Add】，添加服务器
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/afe21ef7eb7440cf91efebb224264712.png">
      <br/>
      填写以下参数（根据PLC的IP更换）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/42787022bbee4bc08985131e84186069.png"/>
     </img>
    </p>
    <p>
     添加变量监控；
     <br/>
     【Root】→【Objects】→【DeviceSet】→【 Leadshine-ARM-Linus-SM-CNC-Resources】
     <br/>
     →【Application】
     <br/>

      找到对应变量后直接拖拽到右侧
