    <h2>
     <a id="IP5305T_1">
     </a>
     一.IP5305T芯片手册
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c25c8a84cff44fbbad85a54a9bfed7b2.png">
      <br/>
      注意该芯片低功耗特性，为防止进入待机，每隔一段时间发送一个电平。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fdc2b994fb164e53ac262e8946ffec7e.png">
       <br/>
       官方提供的芯片外围电路设计图：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5a5e6e51f2f74cd488084944cb00d5a6.png">
        <br/>
        电气特性：
        <br/>
        当负载电流持续小于45mA，最小值24s，典型值32s，最大值40s，会自动关机。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/2e62023007414fa59db707e6ea58b316.png">
         <br/>
         如果已经关机或者防止关机，要给予一个短按，给30ms低电平就认为短按了，手册有误差经过实验至少80ms才实现短按。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7b79bcd5267747e0850922079033728e.png">
          <br/>
          按键方式：
          <br/>
          不需要时将按键悬空，连续短按两次可以关闭芯片紧急关机。
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fd1655ada109457480b13faae4063f97.png"/>
         </img>
