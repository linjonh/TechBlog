    <h3>
     <a id="3_NaN___12">
     </a>
     3. NaN 到底等于二进制的多少？ 即它是怎样表示的?
    </h3>
    <p>
     看一个简单代码：
    </p>
    <pre><code>$ cat main.cpp 
#include &lt;math.h&gt;
#include &lt;stdio.h&gt;

typedef union
{
	double dbl;
	long int lint;
}U_DATA;

int main() {
	U_DATA data;
    double x = NAN;  // 直接创建NaN
    printf("x = %f\n", x);  // 输出：x = nan
	data.dbl=x;
	printf("double = %f\n", data.dbl);
	printf("long = 0x%lx\n", data.lint);
    return 0;
}

</code></pre>
    <p>
     执行结果
     <br/>
     $ ./temp
     <br/>
     x = nan
     <br/>
     double = nan
     <br/>
     long = 0x7ff8000000000000
     <br/>
     <strong>
      很惊讶, %f 竟能输出 nan 字符串!
      <br/>
      这是因为进化的printf 针对这个浮点数进行了特殊处理
      <br/>
      当它遇到0x7ff8000000000000, 就打印nan
     </strong>
    </p>
    <h3>
     <a id="4_1000___inf_46">
     </a>
     4. 1.0/0.0 生成的是什么？ 是一个无穷大 inf
    </h3>
    <p>
     我本来想让它生成一个无效数，实际它生成的是无穷大数, 顺便也了解一下无穷大数.
     <br/>
     代码:
    </p>
    <pre><code>$ cat main.cpp 
#include &lt;math.h&gt;
#include &lt;stdio.h&gt;

typedef union
{
	double dbl;
	long int lint;
}U_DATA;

int main() {
	U_DATA data;
	double a=0.0;
	double x=1.0/a;
    printf("x = %f\n", x);  // 输出：x = nan
	data.dbl=x;
	printf("double = %f\n", data.dbl);
	printf("long = 0x%lx\n", data.lint);
    return 0;
}
</code></pre>
    <p>
     运行结果:
     <br/>
     ./temp
     <br/>
     x = inf
     <br/>
     double = inf
     <br/>
     long = 0x7ff0000000000000 //
     <strong>
      重点, 0x7ff0 与 那个0x7ff8是不一样的
     </strong>
    </p>
    <p>
     printf(“%f”) 对无穷大数也要做特殊处理了.
    </p>
    <h3>
     <a id="5_nan_1___80">
     </a>
     5. 运算时生成一个nan, 用-1开平方 , 竟然生成了负不存在
    </h3>
    <p>
     代码:
    </p>
    <pre><code>cat main.cpp 
#include &lt;math.h&gt;
#include &lt;stdio.h&gt;

typedef union
{
	double dbl;
	long int lint;
}U_DATA;

int main() {
	U_DATA data;
//    double x = NAN;  // 直接创建NaN
	double a=-1;
	double x=sqrt(a);
    printf("x = %f\n", x);  // 输出：x = -nan, 其值0xfff8, 与正nan(0x7ff8)不同
	data.dbl=x;
	printf("double = %f\n", data.dbl);
	printf("long = 0x%lx\n", data.lint);
    return 0;
}
</code></pre>
    <p>
     执行结果:
     <br/>
     ./temp
     <br/>
     x = -nan
     <br/>
     double = -nan
     <br/>
     long = 0xfff8000000000000
    </p>
    <p>
     <strong>
      负不存在0xfff8跟12个0 与 正不存在0x7ff8跟12个0 还是不一样的.
     </strong>
    </p>
    <h3>
     <a id="6_nan_isnan_113">
     </a>
     6. nan 的运算和判断isnan()
    </h3>
    <p>
     代码:
    </p>
    <pre><code>$ cat main.cpp
#include &lt;math.h&gt;
#include &lt;stdio.h&gt;

typedef union
{
	double dbl;
	long int lint;
}U_DATA;

int main() {
	U_DATA data;
    double x = NAN;  // 直接创建NaN
//这里得到结论, nan 与数据运算还是nan,
//可见浮点数运算对错误进行了特殊处理
	double y=x+1;
    printf("x+1 = %f\n", y);  // 输出：x+1 = nan
	data.dbl=y;
	printf("double = %f\n", data.dbl);
	printf("long = 0x%lx\n", data.lint); //nan还是用0x7ff8开始后跟12个0表示
	if(isnan(y)) //判断y 是否是一个正常的数据
	{
		printf("y is not a number!\n");
	}
	else
	{
		printf("y is a number!\n");
	}
    return 0;
}
</code></pre>
    <p>
     执行结果:
     <br/>
     $ ./temp
     <br/>
     x+1 = nan
     <br/>
     double = nan
     <br/>
     long = 0x7ff8000000000000
     <br/>
     y is not a number!
    </p>
    <p>
     这里得到结论:
     <br/>
     <strong>
      nan 与数据运算还是nan,
      <br/>
      可见浮点数运算对错误进行了特殊处理,
     </strong>
     <br/>
     它不会傻傻的都按浮点数去运算，而是先判断一下数据是否合法.
     <br/>
     <strong>
      你也可以用isnan()判断数据是否合法
     </strong>
    </p>
