    <h2>
     allWebPlugin简介
    </h2>
    <p>
     <strong>
      allWebPlugin
     </strong>
     <strong>
      中间件是一款为用户提供安全、可靠、便捷的浏览器插件服务的中间件产品
     </strong>
     <strong>
      ，
     </strong>
     <strong>
      致力于将浏览器插件重新应用到所有浏览器。
     </strong>
     它将现有ActiveX控件直接嵌入浏览器，实现插件加载、界面显示、接口调用、事件回调等。支持Chrome、Firefox、Edge、360等浏览器，接口调用友好、集成方便。为用户提供“信息化系统 + allWebPlugin + 插件 + 浏览器”的解决方案。
    </p>
    <h2>
     alWebPlugin中间件核心优势
    </h2>
    <p>
     1、自创定位算法，实现插件与页面布
     <strong>
      局无缝衔接，极致用户体验
     </strong>
     。
     <br/>
     2、自创插件接口调用方法，保持原汁原味的接口调用及事件响应方式，
     <strong>
      轻松实现OA系统升级改造
     </strong>
     。
     <br/>
     3、支持同一页面或不同页面同时加载多个插件，
     <strong>
      适应各种复杂应用场景
     </strong>
     。
     <br/>
     4、自主创新、安全可控，
     <strong>
      不受浏览器插件技术限制，支持更多浏览器
     </strong>
     （谷歌、火狐、Edge、360浏览器等）。
     <br/>
     5、创新沙箱机制,隔离插件与浏览器运行环境，
     <strong>
      增强插件与浏览器稳定性和安全性
     </strong>
     。
     <br/>
     6、攻克诸多技术难点，实现
     <strong>
      插件不修改、不注册也能使用
     </strong>
     ，极大的降低插件开发量和维护量。
    </p>
