     </path>
    </svg>
    <p>
     在 Ubuntu 22.04（代号
     <strong>
      Jammy Jellyfish
     </strong>
     ）中，可以通过以下方法查看系统版本：
    </p>
    <hr/>
    <h4>
     <a id="_1__lsb_release__4">
     </a>
     方法 1: 使用
     <code>
      lsb_release
     </code>
     命令
    </h4>
    <p>
     这是最直接和通用的方法：
    </p>
    <pre><code class="prism language-bash">lsb_release <span class="token parameter variable">-a</span>
</code></pre>
    <p>
     <strong>
      输出示例
     </strong>
     ：
    </p>
    <pre><code>No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 22.04.3 LTS
Release:        22.04
Codename:       jammy
</code></pre>
    <ul>
     <li>
      <strong>
       关键信息
      </strong>
      ：
      <code>
       Description
      </code>
      显示完整版本号（如
      <code>
       22.04.3 LTS
      </code>
      ），
      <code>
       Codename
      </code>
      显示代号（如
      <code>
       jammy
      </code>
      ）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_2__etcosrelease__21">
     </a>
     方法 2: 查看
     <code>
      /etc/os-release
     </code>
     文件
    </h4>
    <p>
     该文件包含系统的详细版本信息：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> /etc/os-release
</code></pre>
    <p>
     <strong>
      输出示例
     </strong>
     ：
    </p>
    <pre><code>PRETTY_NAME="Ubuntu 22.04.3 LTS"
NAME="Ubuntu"
VERSION_ID="22.04"
VERSION="22.04.3 LTS (Jammy Jellyfish)"
VERSION_CODENAME=jammy
ID=ubuntu
ID_LIKE=debian
HOME_URL="https://www.ubuntu.com/"
SUPPORT_URL="https://help.ubuntu.com/"
BUG_REPORT_URL="https://bugs.launchpad.net/ubuntu/"
PRIVACY_POLICY_URL="https://www.ubuntu.com/legal/terms-and-policies/privacy-policy"
UBUNTU_CODENAME=jammy
</code></pre>
    <hr/>
    <h4>
     <a id="_3__hostnamectl__44">
     </a>
     方法 3: 使用
     <code>
      hostnamectl
     </code>
     命令
    </h4>
    <p>
     适用于查看系统整体信息（包括版本）：
    </p>
    <pre><code class="prism language-bash">hostnamectl
</code></pre>
    <p>
     <strong>
      输出示例
     </strong>
     ：
    </p>
    <pre><code>   Static hostname: ubuntu
         Icon name: computer-vm
           Chassis: vm
        Machine ID: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
           Boot ID: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
    Virtualization: kvm
  Operating System: Ubuntu 22.04.3 LTS
            Kernel: Linux 5.15.0-83-generic
      Architecture: x86-64
</code></pre>
    <hr/>
    <h4>
     <a id="_4__64">
     </a>
     方法 4: 查看内核版本（补充信息）
    </h4>
    <p>
     如果需要同时查看内核版本：
    </p>
    <pre><code class="prism language-bash"><span class="token function">uname</span> <span class="token parameter variable">-r</span>
</code></pre>
    <p>
     <strong>
      输出示例
     </strong>
     ：
    </p>
    <pre><code>5.15.0-83-generic
</code></pre>
    <hr/>
    <h4>
     <a id="_5__76">
     </a>
     方法 5: 图形界面查看
    </h4>
    <ol>
     <li>
      打开
      <strong>
       Settings
      </strong>
      （设置）。
     </li>
     <li>
      进入
      <strong>
       About
      </strong>
      （关于）。
     </li>
     <li>
      在
      <strong>
       OS Name
      </strong>
      或
      <strong>
       Version
      </strong>
      中查看系统版本。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_83">
     </a>
     总结
    </h4>
    <ul>
     <li>
      <strong>
       快速查看版本
      </strong>
      ：
      <code>
       lsb_release -a
      </code>
      或
      <code>
       cat /etc/os-release
      </code>
      。
     </li>
     <li>
      <strong>
       查看内核版本
      </strong>
      ：
      <code>
       uname -r
      </code>
      。
     </li>
     <li>
      <strong>
       图形界面
      </strong>
      ：适合不熟悉命令行的用户。
     </li>
    </ul>
