    <h4>
     1. ​
     <strong>
      插入排序 &amp; 计数排序
     </strong>
    </h4>
    <h5>
     插入排序：
    </h5>
    <pre><code class="language-cpp">void insertionSort(int arr[], int n) {
    for (int i = 1; i &lt; n; i++) {
        int key = arr[i];
        int j = i-1;
        while (j &gt;= 0 &amp;&amp; arr[j] &gt; key) {
            arr[j+1] = arr[j];
            j--;
        }
        arr[j+1] = key;
    }
}</code></pre>
    <ul>
     <li>
      ​
      <strong>
       时间复杂度
      </strong>
      ：O(n²)
     </li>
     <li>
      ​
      <strong>
       空间复杂度
      </strong>
      ：O(1)
     </li>
     <li>
      ​
      <strong>
       稳定性
      </strong>
      ：稳定
     </li>
    </ul>
    <h5>
     计数排序：
    </h5>
    <pre><code class="language-cpp">void countingSort(int arr[], int n) {
    int maxVal = *std::max_element(arr, arr+n);
    std::vector&lt;int&gt; count(maxVal+1, 0);
    for (int i = 0; i &lt; n; i++) count[arr[i]]++;
    int idx = 0;
    for (int i = 0; i &lt;= maxVal; i++) {
        while (count[i]--) arr[idx++] = i;
    }
}</code></pre>
    <ul>
     <li>
      ​
      <strong>
       时间复杂度
      </strong>
      ：O(n+k)
     </li>
     <li>
      ​
      <strong>
       空间复杂度
      </strong>
      ：O(k)
     </li>
     <li>
      ​
      <strong>
       稳定性
      </strong>
      ：稳定
     </li>
    </ul>

    <h4>
     2. ​
     <strong>
      数据库的三范式
     </strong>
    </h4>
    <p>
    </p>
    <p>
     数据库的三范式是关系数据库设计的规范化原则，包括：
    </p>
    <ol>
     <li>
      ​
      <strong>
       第一范式（1NF）​
      </strong>
      ：每个字段都是原子的，不可再分。
     </li>
     <li>
      ​
      <strong>
       第二范式（2NF）​
      </strong>
      ：在1NF基础上，非主键字段完全依赖于主键。
     </li>
     <li>
      ​
      <strong>
       第三范式（3NF）​
      </strong>
      ：在2NF基础上，消除传递依赖。
     </li>
    </ol>
    <h5>
     通俗解释：
    </h5>
    <p>
     三范式就像整理房间的规则，确保数据存储的合理性和高效性。
    </p>
    <p>
     ​
     <strong>
      第一范式（1NF）​
     </strong>
     ：每个物品都放在固定的位置，不能乱放。
    </p>
    <p>
     <strong>
      第二范式（2NF）​
     </strong>
     ：每个物品都放在它专属的抽屉里。
    </p>
    <p>
     <strong>
      第三范式（3NF）​
     </strong>
     ：抽屉里不能放其他抽屉的钥匙。
    </p>
