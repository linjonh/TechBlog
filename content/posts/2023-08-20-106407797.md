# java 获取临时目录

### 1.获取临时目录的方法

```
System.getProperty("java.io.tmpdir") 

```

1. 在windows环境中获取的目录是
     
   C:\Users\登录用户~1\AppData\Local\Temp\
2. 在linux中获取的目录是
     
   /tmp

### 2.差异：

显然在windows环境下获取的临时目录比在linux环境下后获取的临时目录多以斜杠，在编码时需要注意。
