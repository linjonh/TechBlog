     二、Shell脚本基础认知
    </h3>
    <h4>
     2.1 什么是Shell脚本？
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        解释型语言
       </strong>
       ：通过
       <code>
        /bin/bash
       </code>
       逐行解析执行
      </p>
     </li>
     <li>
      <p>
       <strong>
        核心价值
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         自动化重复性系统操作（80%运维工作可脚本化）
        </p>
       </li>
       <li>
        <p>
         快速实现任务调度（结合cron）
        </p>
       </li>
       <li>
        <p>
         提升操作可重复性和精确度
        </p>
       </li>
      </ul>
     </li>
    </ul>

    <h3>
     四、核心编程语法精要
    </h3>
    <h4>
     4.1 变量操作
    </h4>
    <pre><code class="language-bash">company="CSDN社区"          # 定义变量（等号无空格）
readonly version="1.0"     # 只读变量（不可修改）
today=$(date +%Y-%m-%d)    # 命令执行结果赋值
echo "${company} ${today}" # 建议变量用花括号包裹</code></pre>
    <h4>
     4.2 条件分支结构
    </h4>
    <pre><code class="language-bash">if [ -f "/etc/passwd" ]; then     # 判断文件存在
  echo "系统密码文件存在"
elif [ -d "/tmp" ]; then          # 判断目录存在
  echo "/tmp目录存在"
else
  echo "条件未满足"
fi</code></pre>
    <h4>
     4.3 循环结构
    </h4>
    <p>
     <strong>
      for循环处理文件列表
     </strong>
     :
    </p>
    <pre><code class="language-bash">for file in *.log; do
  echo "正在处理日志文件：${file}"
done</code></pre>
    <p>
     <strong>
      while按行读取文件
     </strong>
     :
    </p>
    <pre><code class="language-bash">while read line; do
  echo "当前行：$line"
done &lt; input.txt</code></pre>
    <h4>
     4.4 函数定义与调用
    </h4>
    <pre><code class="language-bash"># 带返回值函数
file_exists() {
  [ -f "$1" ] &amp;&amp; return 0 || return 1
}
​
# 调用函数
file_exists /etc/hosts
if [ $? -eq 0 ]; then    # $?获取上一条命令返回值
  echo "文件存在"
fi</code></pre>

    <h3>
     六、自动化实战：系统备份脚本
    </h3>
    <h4>
     6.1 需求说明
    </h4>
    <ul>
     <li>
      <p>
       每日凌晨压缩备份
       <code>
        /data
       </code>
       目录
      </p>
     </li>
     <li>
      <p>
       自动删除30天前的备份文件
      </p>
     </li>
     <li>
      <p>
       记录操作日志
      </p>
     </li>
    </ul>
    <h4>
     6.2 完整代码实现
    </h4>
    <pre><code class="language-bash">#!/bin/bash
# 自动化备份脚本
​
# 配置参数
backup_dir="/data"
dest_path="/backup"
keep_days=30
log_file="/var/log/backup.log"
​
# 目录检查
[ ! -d "$dest_path" ] &amp;&amp; mkdir -p "$dest_path"
[ ! -f "$log_file" ] &amp;&amp; touch "$log_file"
​
# 生成压缩包
timestamp=$(date +%Y%m%d%H%M)
tar -czf "${dest_path}/backup_${timestamp}.tar.gz" "$backup_dir" 2&gt;&amp;1 | tee -a "$log_file"
​
# 清理旧备份
find "$dest_path" -name "*.tar.gz" -mtime +$keep_days -delete 2&gt;&amp;1 | tee -a "$log_file"
​
# 日志记录
echo "[$(date)] 备份完成，保留最近${keep_days}天文件" | tee -a "$log_file"</code></pre>
