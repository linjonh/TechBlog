    <p>
     对于数据库的恢复非常建议采用BAK文件的方法进行备份和还原，因为该种方法相对于SQL脚本文件不仅简单而且高效。具体步骤如下：
     <br/>
     <strong>
      1.备份Bak文件
     </strong>
     <br/>
     1.右键需要备份的数据库(这里以 MyDB 为例)–&gt;任务–&gt;备份
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fa91554dc78810d71367a9954da2a7bb.png">
      <br/>
      2.记住备份目录：C:\Program Files\Microsoft SQL Server\MSSQL11.MSSQLSERVER\MSSQL\Backup\MyDB.bak
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0033fe3e31c3582502ed97bb3e35a933.png">
       <br/>
       3.选项，备份介质选择 “ 覆盖所有现有备份集 ” ，确定。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de1597a3cae32574f314ea62df2a42c4.png">
        <br/>
        4.备份完成！
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a837894aeb08a1737285187a1c556abb.png">

         经过以上4步，数据库已经备份为BAK文件了。还原BAK文件同样也有两种方法，手动恢复和利用SQL语句进行恢复。下边分别进行说明。
         <br/>
         <strong>
          2.还原Bak文件

              3.Sql语句还原Bak文件

             有时还原BAK文件时会出现错误，可以考虑使用代码进行恢复：
