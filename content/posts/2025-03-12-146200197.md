    <h5>
     <a id="CSS__9">
     </a>
     CSS 的优先级是什么？
    </h5>
    <p>
     !important &gt; 内联样式(style=“”) &gt; ID 选择器 &gt; 类选择器 = 属性选择器 = 伪类 &gt; 标签选择器 = 伪元素选择器。
    </p>
    <h4>
     <a id="_12">
     </a>
     层级选择器
    </h4>
    <ul>
     <li>
      foo bar 后代选择器，元素的后代元素
     </li>
     <li>
      foo &gt; bar 子代选择器，元素的子代元素
     </li>
     <li>
      foo + bar 相邻同胞选择器
     </li>
     <li>
      foo ~ bar 通用同胞选择器
     </li>
     <li>
      foo, bar 并集选择器
     </li>
     <li>
      foo.class 交集选择器
     </li>
    </ul>
    <h4>
     <a id="_20">
     </a>
     条件选择器
    </h4>
    <p>
     :lang 具体语言标记的元素
     <br/>
     :dir() 指定编写方向的元素
     <br/>
     :has 包括指定元素的元素
     <br/>
     :is 指定条件的元素
     <br/>
     :not 非指定条件的元素
     <br/>
     :where 指定条件的元素
    </p>
    <h4>
     <a id="_28">
     </a>
     状态选择器
    </h4>
    <p>
     :active
     <br/>
     :hover
     <br/>
     :link
     <br/>
     :visited
     <br/>
     :focus
    </p>
    <h5>
     <a id="_34">
     </a>
     表单
    </h5>
    <p>
     :valid
     <br/>
     :invalid
     <br/>
     :checked switch
    </p>
    <h4>
     <a id="_39">
     </a>
     结构选择器
    </h4>
    <p>
     :root 文档的根元素
     <br/>
     :empty 无子元素的元素
     <br/>
     :nth-child(n) 元素中指定顺序索引的元素。odd/even
     <br/>
     :nth-last-child(n) 倒序索引
     <br/>
     :first-child 元素中为首的元素
     <br/>
     :last-child 元素中为尾的元素
     <br/>
     :only-child 仅有该元素的元素
    </p>
    <h4>
     <a id="_48">
     </a>
     属性选择器
    </h4>
    <p>
     [attr] 指定属性的元素
     <br/>
     [attr=val] 属性等于指定值的元素
     <br/>
     [attr*=val] 包括指定元素的元素
     <br/>
     [attr^=val] 包括开头
     <br/>
     [attr$=val] 包括结尾
    </p>
    <h4>
     <a id="_55">
     </a>
     伪元素
    </h4>

     ::before 在元素前加入
     <br/>
     ::after 在元素后面加入
     <br/>
     ::first-letter
     <br/>
     ::first-line
     <br/>
     ::backdrop 全屏模式的元素 （Document.fullscreen）
     <br/>
     ::placeholder
    </p>

    <p>

    <h4>
     <a id="_81">
     </a>
     容器属性
    </h4>
    <ul>
     <li>
      flex-direction 主轴的方向
      <ul>
       <li>
        row – default
       </li>
       <li>
        column
       </li>
      </ul>
     </li>
     <li>
      flex-wrap 如果一个轴线排不下，如何换行
      <ul>
       <li>
        nowrap – default
       </li>
       <li>
        wrap
       </li>
       <li>
        wrap-reverse
       </li>
      </ul>
     </li>
     <li>
      flex-flow 是以上两者的简写
     </li>
     <li>
      justify-content
      <ul>
       <li>
        flex-start
       </li>
       <li>
        flex-end
       </li>
       <li>
        center
       </li>
       <li>
        space-between
       </li>
       <li>
        space-around
       </li>
       <li>
        space-evenly
       </li>
      </ul>
     </li>
     <li>
      align-items
      <ul>
       <li>
        flex-start
       </li>
       <li>
        flex-end
       </li>
       <li>
        center
       </li>

       </li>
      </ul>

    </ul>
    <h4>
     <a id="_104">
     </a>
     项目属性
    </h4>
    <ul>
     <li>
      <p>
       order
      </p>
      <ul>
       <li>
        属性定义项目的排列顺序，数值越小，排列越靠前，默认为0.
       </li>
      </ul>
     </li>
     <li>
      <p>
       flex-grow
      </p>
      <ul>
       <li>
        放大比例，剩余空间。默认为0；
       </li>
      </ul>
     </li>
     <li>
      <p>
       flex-shrink
      </p>
      <ul>
       <li>
        缩小比例，剩余空间，默认为1；
       </li>
      </ul>
     </li>
     <li>
      <p>
       flex-basis
      </p>
      <ul>
       <li>
        定义在分配多余空间之前，项目占据的空间。
       </li>
      </ul>
     </li>
     <li>
      <p>
       flex
      </p>
      <ul>
       <li>
        flex: 1 是什么意思
        <ul>
         <li>
          如果直接赋值数字，flex:1 – flex: 1 1 0%
         </li>
         <li>
          如果给 auto, flex: auto – flex: 1 1 auto
         </li>
         <li>
          flex:none flex: 0 0 none
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
