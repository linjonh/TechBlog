    <h3>
     <a id="1_Axios_3">
     </a>
     1. Axios简介
    </h3>
    <p>
     Axios是一个基于Promise的HTTP客户端，可以在浏览器和Node.js环境中使用。它提供了一套简单且一致的API，使得发送HTTP请求变得异常便捷。自发布以来，Axios已成为前端开发中处理网络请求的首选工具之一。
    </p>
    <h4>
     <a id="11__7">
     </a>
     1.1 核心特性
    </h4>
    <h5>
     <a id="_Promise_based__Promise_API_9">
     </a>
     -
     <strong>
      Promise based
     </strong>
     - 支持Promise API，便于处理异步操作
    </h5>
    <h5>
     <a id="___Nodejs_10">
     </a>
     -
     <strong>
      跨平台
     </strong>
     - 同时支持浏览器和Node.js环境
    </h5>
    <h5>
     <a id="____11">
     </a>
     -
     <strong>
      拦截请求和响应
     </strong>
     - 通过拦截器机制轻松实现请求/响应转换
    </h5>
    <h6>
     <a id="1_JSON__JSON_12">
     </a>
     1.
     <strong>
      自动转换JSON数据
     </strong>
     - 自动将请求数据转换为JSON格式
    </h6>
    <h6>
     <a id="2____13">
     </a>
     2.
     <strong>
      取消请求
     </strong>
     - 支持取消已发出但尚未完成的请求
    </h6>
    <h6>
     <a id="3_XSRF___14">
     </a>
     3.
     <strong>
      防御XSRF
     </strong>
     - 自带防御跨站请求伪造攻击机制
    </h6>
