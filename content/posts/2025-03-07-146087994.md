    <p>
     <strong>
      摘要
     </strong>
     ：Uniapp作为一款基于Vue.js的跨平台开发框架，支持“一次开发，多端部署”。本文将手把手教你如何将Uniapp项目运行到微信小程序、H5、APP等多个平台，并解析常见问题。
    </p>
    <hr/>
    <h3>
     <a id="_4">
     </a>
     一、环境准备
    </h3>
    <p>
     在开始前，请确保已安装以下工具🔧：
    </p>
    <h4>
     <a id="_6">
     </a>
     代码编辑器
    </h4>
    <ul>
     <li>
      <strong>
       HBuilderX
      </strong>
      （官方IDE，推荐使用）
      <br/>
      创建项目教程：
      <a href="https://uniapp.dcloud.net.cn/quickstart-hx.html" rel="nofollow">
       官方HBuilderX创建项目教程
      </a>
      <br/>
      下载地址：
      <a href="https://www.dcloud.io/hbuilderx.html" rel="nofollow">
       HbuilderX下载地址
      </a>
     </li>
     <li>
      <strong>
       Vscode
      </strong>
      （博主推荐使用🌹🌹）
      <br/>
      但是需要自己通过vue-cli创建项目，并配置运营打包命令，我们可以直接下载官方的模版使用，具体的cli创建项目教程可以按照官方的步骤来：
      <a href="https://uniapp.dcloud.net.cn/quickstart-cli.html" rel="nofollow">
       Vue-cli创建项目教程
      </a>
      <br/>
      下载地址：
      <a href="https://code.visualstudio.com/" rel="nofollow">
       Vscode下载地址
      </a>
     </li>
    </ul>
    <h4>
     <a id="_13">
     </a>
     项目环境
    </h4>
    <ul>
     <li>
      <strong>
       Node.js
      </strong>
      （需≥14版本）
     </li>
    </ul>
    <h4>
     <a id="_15">
     </a>
     其他平台模拟器
    </h4>
    <ul>
     <li>
      各平台开发工具：
      <ul>
       <li>
        微信开发者工具（小程序）
       </li>
       <li>
        Android Studio（Android App）
       </li>
       <li>
        Xcode（iOS App）
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="UniappHBuilderX__23">
     </a>
     二、创建Uniapp项目（我们以HBuilderX 创建项目举例子）
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        新建项目
       </strong>
       <br/>
       打开HBuilderX → 文件 → 新建 → 项目 → 选择
       <code>
        uni-app
       </code>
       模板 → 输入项目名称。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4ac2fa6921d34fc5930a335232b8032e.png">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6cd95622aad540d6acbcb2afd5e403d3.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       <strong>
        项目结构解析
       </strong>
      </p>
      <pre><code class="prism language-markdown">├── pages        // 页面目录
├── static       // 静态资源
├── App.vue      // 根组件
├── main.js      // 入口文件
├── manifest.json// 应用配置（如App图标）
└── pages.json   // 页面路由与样式
</code></pre>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="_41">
     </a>
     三、运行到不同平台
    </h3>
    <h4>
     <a id="31__43">
     </a>
     3.1 微信小程序
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        配置
        <code>
         微信开发者工具路径
        </code>
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b91bd1f85a69442794253d9266591a67.png">
        <br/>
        配置自己的微信开发者工具路径
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a81b9e1cf6774e04984af60545438fcf.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置
        <code>
         manifest.json
        </code>
       </strong>
       <br/>
       打开
       <code>
        manifest.json
       </code>
       → 微信小程序配置 → 输入微信AppID（若无，可跳过但无法真机调试）。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/52c3e9e422d046c69b46d3fd9d7a3957.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        运行项目
       </strong>
      </p>
      <ul>
       <li>
        <p>
         顶部菜单 → 运行 → 运行到小程序模拟器 → 微信开发者工具
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cbd087cfa4444496bbe504d5eed45a1e.png"/>
        </p>
       </li>
       <li>
        <p>
         <em>
          首次运行需在微信开发者工具中设置：设置 → 安全 → 开启服务端口
         </em>
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/238806ec1fe844e88a750a0a9c8e459e.png">
          <br/>
          然后就运行成功了
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/88be9122e4ab4bcdadaeed5c4427507d.png"/>
         </img>
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     <strong>
      常见问题
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       错误提示
       <code>
        [error] 项目未配置appid
       </code>
      </strong>
      <br/>
      在
      <code>
       manifest.json
      </code>
      中补充微信小程序的AppID，或点击“试用”跳过。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="32_H5_69">
     </a>
     3.2 H5网页
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        浏览器运行
       </strong>
       <br/>
       直接点击HBuilderX工具栏的“运行” → 运行到浏览器 → 选择Chrome。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/08c22119811c475592821c646fe59022.png"/>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/08656405ea8642f0a223e2ab248f81c1.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        自定义配置
       </strong>
       <br/>
       在
       <code>
        manifest.json
       </code>
       的H5配置中可修改：
      </p>
      <ul>
       <li>
        路由模式（hash/history）
       </li>
       <li>
        跨域代理（解决本地开发接口跨域问题）
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="33_AndroidiOS_App_83">
     </a>
     3.3 Android/iOS App
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        基础配置
       </strong>
       <br/>
       打开
       <code>
        manifest.json
       </code>
       → App SDK配置 → 选择需要的模块（如地图、支付）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        真机调试
       </strong>
      </p>
      <ul>
       <li>
        连接手机 → 开启USB调试
       </li>
       <li>
        点击“运行” → 运行到手机或模拟器
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/29bb6eb63b9a4944b5ab499662dc3847.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        云打包
       </strong>
       <br/>
       菜单 → 发行 → 原生App-云打包 → 选择平台并提交。
      </p>
     </li>
    </ol>
    <p>
     <strong>
      注意
     </strong>
     ：iOS打包需Apple开发者账号（年费$99），测试阶段可使用免费证书。
    </p>
    <hr/>
    <h4>
     <a id="34__98">
     </a>
     3.4 其他平台（快应用/支付宝小程序等）
    </h4>
    <p>
     在
     <code>
      manifest.json
     </code>
     中配置对应平台参数，运行方式与微信小程序类似，需提前安装各平台开发者工具。
    </p>
    <hr/>
    <h3>
     <a id="_103">
     </a>
     四、条件编译技巧
    </h3>
    <p>
     通过注释实现
     <strong>
      多平台差异化代码
     </strong>
     ：
     <br/>
     可以参考我之前写的文章了解这部分知识：
     <a href="https://blog.csdn.net/zhw1984974676/article/details/145988979">
      UniApp如何判断平台的多种方法（2025最新指南）
     </a>
    </p>
    <pre><code class="prism language-html"><span class="token comment">&lt;!-- #ifdef H5 --&gt;</span>

    <h3>
     <a id="_118">
     </a>
     五、常见问题汇总
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        问题描述
       </th>
       <th>
        解决方案
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        微信开发者工具无法自动打开
       </td>
       <td>
        检查端口是否开启，或手动导入项目目录
       </td>
      </tr>
      <tr>
       <td>
        H5页面路由白屏
       </td>
       <td>
        修改路由模式为
        <code>
         history
        </code>
       </td>
      </tr>
      <tr>
       <td>
        App启动闪退
       </td>
       <td>
        检查是否缺少必要模块权限
       </td>
      </tr>
     </tbody>
    </table>
    <hr/>
    <h3>
     <a id="_127">
     </a>
     六、总结
    </h3>
    <p>
     Uniapp通过统一的技术栈大幅降低多端开发成本，但需注意：
    </p>
    <ul>
     <li>
      各平台API存在差异，建议使用条件编译
     </li>
     <li>
      复杂功能（如原生插件）需单独适配
     </li>
     <li>
      调试时优先使用真机测试
     </li>
    </ul>
