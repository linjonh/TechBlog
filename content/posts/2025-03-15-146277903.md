    <h3>
     <span style="color:#1c7892">
      2.1 静态分配VS动态分配
     </span>
    </h3>
    <h4>
     <span style="color:#9c8ec1">
      2.1.1 静态分配
     </span>
    </h4>
    <p>
     静态分配数组的大小和空间事先已经固定，顺序表会出现空间已满无法加入的情况。
    </p>
    <pre><code class="language-cpp">//线性表最大长度
#define MaxSize 50
typedef struct {
	ElemType data[MaxSize];
	int length;	//顺序表当前长度
}SqList;</code></pre>
    <h4>
     <span style="color:#9c8ec1">
      2.1.2 动态分配
     </span>
    </h4>
    <pre><code class="language-cpp">//表长度初始定义，满了动态扩容
#define MaxSize 50
typedef struct {
	ElemType data[MaxSize];
	//顺序表的容量和顺序表的长度
	int MaxSize,length;
}SqList;</code></pre>
    <p>
     如果满了就扩容：
    </p>
    <pre><code class="language-cpp">//C，最后可以不* InitSize，*2（二倍扩）...都是可以的
L.data = (ElemType*)malloc(sizeof(ElemType) * InitSize);

//C++
L.data = new ElemType[InitSize];</code></pre>

    <h3>
     <span style="color:#1c7892">
      2.3 插入操作
     </span>
    </h3>
    <p>
     <strong>
      在顺序表L的
      <span style="color:#1c7331">
       第i个位置（下标为i- 1）
      </span>
      插入新元素e。
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       时间复杂度：O(N)
      </strong>
     </li>
     <li>
      最好O(1)，最坏O(N),平均O(N/2)
     </li>
    </ul>
    <pre><code class="language-cpp">bool ListInsert(SqList&amp; L, int i, ElemType e) {
	//i的范围无效，插入失败
	if (i &lt; 1 || i &gt;L.length + 1)
		return false;
	//存储空间满，插入失败
	if (L.length &gt;= MaxSize)
		return flase;
	//将第i个元素及之后的元素后移
	for (int j = L.length; j &gt;= i; i--)
		L.data[j] = L.data[j - 1];
	//插入
	L.data[i - 1] = e;
    //顺序表长度+1
	L.length++;
	return true;
}</code></pre>
    <hr/>
    <h3>
     <span style="color:#1c7892">
      2.4 删除操作
     </span>
    </h3>
    <p>
     <strong>
      删除表中
      <span style="color:#be191c">
       第i个位置（下标为i - 1）
      </span>
      的元素，用
      <span style="color:#be191c">
       引用变量e
      </span>
      返回。
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       时间复杂度：O(N)
      </strong>
     </li>
     <li>
      最好O(1)，最坏O(N),平均O((N-1)/2)
     </li>
    </ul>
    <pre><code class="language-cpp">//e要改变，传入实参
bool ListDelete(SqList&amp; L, int i, ElemType&amp; e) {
	if (i &lt; 1 || i &gt;L.length)
		return false;
	//被删除元素的元素赋给e
    //第i个位置元素的下标是i - 1
	e = L.data[i - 1];
	//元素前移
	for (int j = i; j &lt; Length; j++)
		L.data[j - 1] = L.data[j];
	//元素删除，顺序表长度-1
	L.length--;
	return true;
}</code></pre>
    <hr/>
    <h3>
     <span style="color:#1c7892">
      2.5 按值查找（顺序查找）
     </span>
    </h3>
    <p>
     <strong>
      在顺序表中查找第一个元素值等于e的元素，并返回其
      <span style="color:#be191c">
       位序
      </span>
      （数组下标+1）。
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       时间复杂度：O(N)
      </strong>
     </li>
     <li>
      最好O(1)，最坏O(N),平均O((N+1)/2)
     </li>
    </ul>
    <p>
     顺序表还可以
     <u>
      按位查找
     </u>
     ，就是利用数组下标访问，
     <u>
      时间复杂度为O(1).
     </u>
    </p>
    <pre><code class="language-cpp">int LocateElem(SeqList L, ElemType e) {
	int i;
	for (i = 0; i &lt; L.length; i++)
		if (L.data[i] == e)
			//返回的是位序
			return i + 1;
	return 0;
}</code></pre>
    <hr/>
    <p>
     <span style="background-color:#e6b223">
      1.错 对
     </span>
    </p>
    <p>
     顺序表中所有元素的类型必须相同，且必须连续存放，一维数组中的元素可以不连续存放。
    </p>
    <p>
     栈，队列，树也可以利用一维数组表示，它们的逻辑结构就完全不同。
    </p>
    <p>
     顺序表和一维数组的物理结构相似，基本一样（都连续存放，但顺序表除存取外还有其他操作。）
    </p>
    <p>
     <span style="background-color:#e6b223">
      2.A
     </span>
    </p>
    <p>
     存/取方式是指读/写方式，顺序表是一种支持随机存取的存储结构，根据起始地址加上元素的序号，可以很方便地访问任意一个元素，这就是随机存取的概念。
    </p>
    <p>
     顺序存取是按照数据存储的物理位置顺序依次进行访问。就像听磁带，要先听完前面的内容才能听到后面的内容，数据的读取必须从起始位置开始，按顺序逐个读取。
    </p>
