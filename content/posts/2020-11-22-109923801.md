## 开发工具概述

IDEA 是一个专门针对 Java 的集成开发工具 ( IDE ). 由于 Java 语言编写. 所以, 需要有 JRE 运行环境并配置好环境变量. IDEA 可以极大地提升我们的开发效率. 可以自动编译, 检查错误. 很多公司使用 IDEA 进行开发.
