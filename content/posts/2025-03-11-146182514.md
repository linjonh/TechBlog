# linux和windows之间的复制

### 第一步

```bash
sudo apt-get autoremove open-vm-tools


```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/d702e17b903f44acb85c2a0e5881aab0.png)

### 第二步

```bash
sudo apt-get update


```

### 第三步

```bash
sudo apt-get install open-vm-tools-desktop


```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/69f0a4c1028342e9826824cc0af9edbb.png)
  
按y

### 第四步

重启虚拟机，终端下输入

```bash
reboot


```

Linux下 按“ Ctrl+Shift+C V ”复制粘贴
  
Windows下按“ Ctrl + C V ”复制粘贴
