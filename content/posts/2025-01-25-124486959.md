    <h2>
     <a id="_0">
     </a>
     本文由来
    </h2>
    <p>
     本来我并不想写文章的，但是我发现，对于一个0基础的小白vscode用户而言，想完整的下载一个第三方库还是存在一定的问题，并且我在搜索文章的时候发现，完全没有小白教程，太难了，所以说我就写了这篇博客，仅供使用vscode的小白选手安装python的第三方库
    </p>
    <h2>
     <a id="vscode_3">
     </a>
     vscode配置
    </h2>
    <ul>
     <li>
      python配置：没有下载python，直接使用的是Anaconda中的python
     </li>
     <li>
      已经下载过vscode中自带的插件
     </li>
    </ul>
    <h2>
     <a id="1python_6">
     </a>
     1、找自己的python是什么版本
    </h2>
    <p>
     这个真的很重要，因为我不知道我的vscode是什么python，导致我直接pip严重报错
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cb7f81d97446ff24aa649b33b225b0a4.png"/>
    </p>
    <h2>
     <a id="2_10">
     </a>
     2、插件下载
    </h2>
    <p>
     1、如果路径是用的自己下载的python，那么可以使用pip install 语句进行下载
    </p>
    <ul>
     <li>
      win+R（打开控制命令行）
     </li>
     <li>
      <code>
       pip install jieba
      </code>
     </li>
     <li>
      回车结束
     </li>
    </ul>
    <p>
     2、如果路径是用的Anaconda，那么就用conda install下载
    </p>
    <ul>
     <li>
      打开Anaconda prompt
     </li>
     <li>
      输入：
      <code>
       conda install -c conda-forge jieba
      </code>
     </li>
     <li>
      回车等待下载结束
     </li>
    </ul>
