# Jquery AJAX取出后端传出的list

直接list[index]就行

比如我这里后端传进来的是List<List>格式
  
Jquery AJAX

```
$.ajax({
    data :{"name":name,"statue":statue},
    type : "GET",
    dataType : "json",// 预期服务器返回的数据类型
    url : "conditionalQuery",
    success : function(result) {
     alert(result);
     //循环取出外层list的值
     $.each(result,function(i,list){
      alert(list[0]);
     });
    },
   });

```
