    <p>
     官方对npm的使用介绍
    </p>
    <p>
     <a id="httpsdevelopersweixinqqcomminiprogramdevdevtoolsnpmhtml_2">
     </a>
     <a href="https://developers.weixin.qq.com/miniprogram/dev/devtools/npm.html" rel="nofollow">
      https://developers.weixin.qq.com/miniprogram/dev/devtools/npm.html
     </a>
    </p>
    <p>
     <a id="node_modulesUI_4">
     </a>
     可能会出现错误，没有node_modules,导致无法引入第三方UI等
    </p>
    <p>
     <img alt="在这里插入图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/136e4d4d18efb1339c43bcfd8280d5d6.png"/>
    </p>
    <p>
     <a id="_6">
     </a>
     具体正确操作应该是：
    </p>
    <h4>
     <a id="_8">
     </a>
     一：先新建一个小程序项目
    </h4>
    <pre class="has"><code>#  在小程序项目的根目录下，按住Shift键点击鼠标右键，选择在在此次打开命令窗口，弹出cmd操作窗口，然后输入下面指令安装组件
npm init</code></pre>
    <p>
     <img alt="在这里插入图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/51b619b5e848300fd93eeb8e1e6400af.png">
      <br/>
      这样就初始化的npm项目。
     </img>
    </p>
    <h4>
     二：生成package-lock.json，记录使用的第三方插件
    </h4>
    <pre class="has"><code class="language-html"># 接下来生成package-lock.json，记录使用的第三方插件
npm install --production
</code></pre>
    <p>
     <a id="_22">
     </a>
     内容如下
    </p>
    <pre class="has"><code class="language-javascript">{
  "name": "weixin_coder",
  "version": "1.0.0",
  "lockfileVersion": 1,
  "requires": true,
  "dependencies": {
    "vant-weapp": {
      "version": "0.4.8",
      "resolved": "http://registry.npm.taobao.org/vant-weapp/download/vant-weapp-0.4.8.tgz",
      "integrity": "sha1-CxF2lWnm8jrP8Qpl+9n4R5BJZS0="
    }
  }
}
</code></pre>
    <h4>
     <a id="_40">
     </a>
     三：接着引入第三方组件，如slide-view的引用
    </h4>
    <p>
     <a href="https://github.com/wechat-miniprogram/slide-view">
      https://github.com/wechat-miniprogram/slide-view
     </a>
    </p>
    <p>
     安装 slide-view
    </p>
    <pre class="has"><code class="language-javascript">npm install --save miniprogram-slide-view
</code></pre>
    <p>
    </p>
    <p>
     <a id="___npm_50">
     </a>
     执行完后，在项目的根目录下，会出现node_modules的包
    </p>
    <p>
     <img alt="" class="has" height="76" src="https://i-blog.csdnimg.cn/blog_migrate/7fd2a5801ea53a5f9f794ed0ab918b52.png" width="266"/>
    </p>
    <h4>
     四： 开启“使用npm模块”
    </h4>
    <p>
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/d7757e849e6d22de2b7f1089ed9ebfc3.png"/>
    </p>
    <h4>
    </h4>
    <h4>
     五：然后点击工具 =&gt; 构建npm
    </h4>
    <p>
     <img alt="在这里插入图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/e7ea32be42342b30d69d3ef162e9a4ef.png"/>
    </p>
    <p>
     <a id="_52">
     </a>
     最后会在项目目录中生成这样的目录
    </p>
    <p>
     <img alt="在这里插入图片描述" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/5055fb5f339d713edd9c5c49fa3e12c7.png"/>
    </p>
    <p>
     <a id="UI_54">
     </a>
     node_modules文件在小程序开发工具中是不显示里面具体的详细信息，只有在硬盘中才看的到。
    </p>
    <h4>
     六：引用组件
    </h4>
    <p>
     <a id="_56">
     </a>
    </p>
    <ol>
     <li>
      在需要使用 slide-view 的页面 page.json 中添加 slide-view 自定义组件配置
     </li>
    </ol>
    <pre class="has"><code class="language-javascript">{
  "usingComponents": {
    "slide-view": "miniprogram-slide-view"
  }
}</code></pre>
    <p>
     <a id="_66">
     </a>
     对于我们的项目我们改成自己的目录
    </p>
    <pre class="has"><code class="language-javascript">{
    "usingComponents": {
      "slide-view": "/miniprogram_npm/miniprogram-slide-view/index"
    }
}</code></pre>
    <p>
     然后在WXML 文件中引用 slide-view就可以了。
    </p>
