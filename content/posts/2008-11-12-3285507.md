    <div>
     EntLib.com Forum 开源ASP.NET 论坛系统是EntLib.com 小组基于YAF开源论坛，定制化开发及汉化的一款基于ASP.net的论坛软件，分别满足拥有.net Framework 2.0/3.5以及MS SQL Server 2005/2008等不同配置环境的站长用户，采用utf8/Unicode完美支持世界各国语言文字的同时显示，支持虚拟主机用户、更换或自定义模板Theme、FCKEditor编辑器、个性化论坛图标、Ajax无刷新回复等众多优秀的论坛功能。
    </div>
    <div>
    </div>
    <div>
     <strong>
      本次更新功能列表：
     </strong>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      Readme v3.5
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      ============
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      1.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       论坛源代码从VS 2005 升级到VS 2008
      </span>
     </strong>
     <span style="FONT-SIZE: 9pt">
      。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      2.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       增加论坛设置广告或通告的功能-- 显示在论坛顶端
      </span>
     </strong>
     <span style="FONT-SIZE: 9pt">
      。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      3.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       针对论坛帖子增加属性：精华（1、2、3个级别），推荐标识
      </span>
     </strong>
     <span style="FONT-SIZE: 9pt">
      。需要修改yaf_Topic 表结构，增加个字段，同时更新SP - [dbo].[yaf_topic_list]。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      4.
     </span>
     <span style="FONT-SIZE: 9pt">
      帖子/论坛多页显示- 修改显示效果。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      5.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       针对论坛帖子增加是否有附件的标识
      </span>
     </strong>
     <span style="FONT-SIZE: 9pt">
      。需要修改yaf_Topic 表结构，增加个字段，同时更新SP - [dbo].[yaf_topic_list]。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      6.
     </span>
     <span style="FONT-SIZE: 9pt">
      用户注册部分：用户名和密码长度必须大于等于位。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      7.
     </span>
     <span style="FONT-SIZE: 9pt">
      显示版主名单，需要同时修改存储过程yaf_forum_moderators。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      8.
     </span>
     <span style="FONT-SIZE: 9pt">
      调整改进Forum 页面部分显示细节。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      9.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       每个论坛主题设置论坛图标
      </span>
     </strong>
     <span style="FONT-SIZE: 9pt">
      ，需要同时修改表yaf_Forum，增加字段ForumIcon-nvarchar(100)。另外，还需要修改存储过程[yaf_forum_listread]，增加ForumIcon 字段的输出。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      10.
     </span>
     <span style="FONT-SIZE: 9pt">
      论坛顶部导航栏增加到首页、博客的链接。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      11.
     </span>
     <span style="FONT-SIZE: 9pt">
      配置文件web.config 增加首页、博客的配置URL，默认值分别为
      <a href="http://www.entlib.com/">
       <font color="#800080">
        http://www.EntLib.com
       </font>
      </a>
      和
      <a href="http://blog.entlib.com/">
       <font color="#800080">
        http://blog.EntLib.com
       </font>
      </a>
      。
     </span>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      12.
     </span>
     <strong>
      <span style="FONT-SIZE: 9pt; COLOR: #c00000">
       汉化论坛后台管理界面。
      </span>
     </strong>
    </div>
    <div align="left">
     <span style="FONT-SIZE: 9pt">
      13.
     </span>
     <span style="FONT-SIZE: 9pt">
      论坛帖子内容（Message）必须大于等于个字符，避免简单的灌水内容。
     </span>
    </div>
    <div>
     <span style="FONT-SIZE: 9pt">
      14.
     </span>
     <span style="FONT-SIZE: 9pt">
      清理网络新闻组NNTP相关页面文件、后台配置等等，没有用途，保持项目代码的简洁。
     </span>
    </div>
