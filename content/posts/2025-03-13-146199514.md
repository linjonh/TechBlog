    <p>
     1.基本思想：
    </p>
    <p>
     <span style="color:#777777">
      直接插入排序是一种简单的插入排序法，其基本思想是：
     </span>
     <span style="color:#777777">
      <strong>
       把待排序的记录按其关键码值的大小逐个插入到一个已经排好序的有序序列中，直到所有的记录插入完为
      </strong>
     </span>
     <span style="color:#777777">
      <strong>
       止，得到一个新的有序序列
      </strong>
     </span>
     <span style="color:#777777">
      。
     </span>
     <img alt="" height="293" src="https://i-blog.csdnimg.cn/direct/a4fdb2941aba4be5829393af5e330633.png" width="333"/>
    </p>
    <p>
     2直接插入排序：
    </p>
    <div>
     <span style="color:#777777">
      当插入第i(i&gt;=1)个元素时，前面的array[0],array[1],…,array[i-1]已经排好序，此时用array[i]的排序码与
     </span>
     <span style="color:#777777">
      array[i-1],array[i-2],…的排序码顺序进行比较，找到插入位置即将array[i]插入，原来位置上的元素顺序后移
     </span>
    </div>
    <div>
     <img alt="" height="505" src="https://i-blog.csdnimg.cn/direct/4cbe43e1c8784c1bb9f89b8c4f9fb62d.png" width="811">
      注：基础不好的同学在写排序的时候建议先写单趟再写整体
     </img>
    </div>
    <div>
     <p>
      单趟思想：将一个数字插入有序区间
     </p>
     <pre><code>// 插入排序---升序
void InsertSort(int* a, int n)
{
       for (int i = 1; i &lt; n; i++)//第一个数不需要排所以从1开始排
       {
              int end=i-1;
              int temp=a[i];//将temp插入0~end区间中，保持有序
              while (end &gt;= 0)
              {
                      if (temp &lt; a[end])
                      {
                             a[end + 1] = a[end];//比end大那么把此时比temp大的数往后挪一位
                             end--;
                      }
                      else
                      {
                             break;//break出来以后有两种情况，第一种情况是数组元素比temp都大end一直减最后走到-1的位置；第二种情况是end走到一个比temp小的位置，不论是什么情况，end后面都为空并需要将temp插入进去
                      }
              }
              a[end + 1] = temp;
       }
}</code></pre>
     <p>
      break出来以后得两种情况

     </p>

    <div>
     <span style="color:#777777">
      直接插入排序的特性总结：
     </span>
    </div>
    <div>
     <span style="color:#777777">
      1.
     </span>
     <span style="color:#777777">
      元素集合越接近有序，直接插入排序算法的时间效率越高
     </span>
    </div>
    <div>
     <span style="color:#777777">
      2.
     </span>
     <span style="color:#777777">
      时间复杂度：
     </span>
     <span style="color:#777777">
      O(N^2)  最坏情况
     </span>
     <span style="color:#777777">
      O(N^2)  最好情况
     </span>
     <span style="color:#777777">
      O(N)
     </span>
    </div>
    <div>
     <span style="color:#777777">
      3.
     </span>
     <span style="color:#777777">
      空间复杂度：
     </span>
     <span style="color:#777777">
      O(1)
     </span>
     <span style="color:#777777">
      ，它是一种稳定的排序算法
     </span>
    </div>

     <span style="color:#777777">
      4.
     </span>
     <span style="color:#777777">
