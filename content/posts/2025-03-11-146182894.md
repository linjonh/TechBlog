    <h2>
     <a id="TDenginehttpsdocstaosdatacom_0">
     </a>
     了解时序数据库TDengine，可以点击
     <a href="https://docs.taosdata.com/" rel="nofollow">
      官方文档
     </a>
     进行详细查阅
    </h2>
    <h2>
     <a id="_2">
     </a>
     安装步骤
    </h2>
    <h3>
     <a id="windows_3">
     </a>
     首先找到自己需要下载的版本，这边我暂时只写windows版本的安装
    </h3>
    <ul>
     <li>
      首先我们需要点开官网，找到发布历史，目前TDengine的windows版本只更新到3.0.7.1，我们下最新的就好，旧版需要另下taos配置，但是从 3.0.6.0 开始，taosTools 集成到 TDengine 的安装包中，我们可以下新版本，就不需要单独配置了
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/2aab87568d774a2280287f68f0b35992.png"/>
     </li>
    </ul>
    <h3>
     <a id="_6">
     </a>
     下载好后点击安装包自行进行傻瓜式安装
    </h3>
    <h2>
     <a id="_8">
     </a>
     启动方式
    </h2>
    <h3>
     <a id="_9">
     </a>
     第一种：命令行
    </h3>
    <ul>
     <li>
      <p>
       以管理员身份启动cmd窗口
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d3201c03f9c649d5905e020172b40cf9.png"/>
      </p>
     </li>
     <li>
      <p>
       启动taosd服务，输入命令
       <code>
        sc start taosd
       </code>
      </p>
     </li>
     <li>
      <p>
       启动taosadapter服务，输入命令
       <code>
        sc start taosadapter
       </code>
      </p>
     </li>
     <li>
      <p>
       点击桌面生成的Taos Shell，输入命令
       <code>
        taos
       </code>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/60e58319016d451aa994fa0048ce0aeb.png">
        <br/>
        启动成功
       </img>
      </p>
     </li>
    </ul>
    <h3>
     <a id="exe_20">
     </a>
     第二种：直接点击exe
    </h3>
    <ul>
     <li>
      找到安装的TDengine目录，按序从下到上点击执行
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fb17a0ab2aeb48fab858fb2da553df17.png"/>
     </li>
    </ul>
    <h2>
     <a id="_23">
     </a>
     配置开机自启动
    </h2>
    <p>
     直接全局搜服务，找到要启动的两个服务
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d2f85b1ac8e54a59927ff931e972651b.png">
      <br/>
      右击服务，先停止运行，修改属性的启动类型为自动，再启动并应用
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/379c5bee5b734e40939191e540ebdc0b.png"/>
     </img>
    </p>
    <h2>
     <a id="_28">
     </a>
     可视化工具
    </h2>
    <p>
     目前官方文档里写的数据库管理是DBeaver和qStudio，我目前是用DBeaver的，这些在官网里都有教程，可以看看
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7d238b6b6dc24dc3891a50f2103a5c6a.png">
      <br/>
      也可以点击
      <a href="https://blog.csdn.net/qq_52879387/article/details/146183727?sharetype=blogdetail&amp;sharerId=146183727&amp;sharerefer=PC&amp;sharesource=qq_52879387&amp;sharefrom=mp_from_link">
       DBeaver安装教程+连接TDengine数据库
      </a>
      进行安装下载，操作好后可以进行基础使用
     </img>
    </p>
    <h2>
     <a id="TDengine_33">
     </a>
     对TDengine细节的一些自我工作记录
    </h2>
    <ul>
     <li>
      超表不可进行直接操作，只接受自动记录数据或代码记录
     </li>
     <li>
      记录设备的历史数据可以使用mqttx进行模拟连接，发布一些命令，比如已发布的aaa设备发出的累计总电量是44.4，发布成功后查看数据库才可以看到被记录的数值
     </li>
     <li>
      语句基础操作可以右击表进行生成SQL
     </li>
     <li>
      可以用数值查看器进行查看十六进制传输具体数值
     </li>
     <li>
      暂时这么多
     </li>
    </ul>
