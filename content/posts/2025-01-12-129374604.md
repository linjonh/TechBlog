    <h3>
     <a id="nextcloud163_0">
     </a>
     nextcloud配置邮件163服务器
    </h3>
    <h4>
     <a id="_2">
     </a>
     先配置个人信息里的邮箱
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1edc6622bab1ece0f52eab7d330e768e.png"/>
    </p>
    <h4>
     <a id="_4">
     </a>
     获取邮箱的授权码
    </h4>
    <ul>
     <li>
      登录163邮箱
     </li>
     <li>
      进入设置-&gt;POP3/SMTP/IMAP
     </li>
     <li>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1263bb2f3586958f318b7bc59dd75e1b.png"/>
     </li>
     <li>
      在POP3/SMTP/IMAP 界面开启SMTP服务
     </li>
     <li>
      点击新增授权码 ，根据提示发送短信获得授权密码
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d7e5857ab6075c6afbb0a2919812aad6.png"/>
     </li>
    </ul>
    <h4>
     <a id="_11">
     </a>
     配置电子邮件服务器
    </h4>
    <ul>
     <li>
      发送模式选 SMTP； 加密选 SSL/TLS；
     </li>
     <li>
      来自地址填你要发送邮件的地址，@ 163.com；
     </li>
     <li>
      人证方法选 登录 ，将需要人证打钩；
     </li>
     <li>
      服务器地址填 smtp.163.com ; 端口配465；
     </li>
     <li>
      证书就填163邮箱的地址 如: xxxxxx@163.com;
     </li>
     <li>
      将163的授权密码填入密码框
     </li>
     <li>
      点击测试发送，就可以看到邮箱里收到一条测试邮件
     </li>
    </ul>
