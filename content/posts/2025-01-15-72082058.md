    <h3 id="快速体验">
     快速体验
    </h3>
    <p>
     如果你已对oneNET有一定的了解，且能够使用 ESP-IDF 编译 hello-world，则可以按照下面的步骤快速体验。
    </p>
    <ul>
     <li>
      登录oneNET，依次创建产品，添加设备，设置鉴权信息。记录下产品ID、设备ID和鉴权信息。
     </li>
     <li>
      创建一个数据流，并记录下该数据流的名称。
     </li>
     <li>
      修改本仓库源代码目录下的
      <code>
       config.h
      </code>
      文件，主要包括：
      <br/>
      <ul>
       <li>
        <code>
         WIFI_SSID
        </code>
        - esp32需要连接到的AP的ssid。
       </li>
       <li>
        <code>
         WIFI_PASS
        </code>
        - esp32需要连接到的AP的密码。
       </li>
       <li>
        <code>
         ONENET_DEVICE_ID
        </code>
        - 云平台所创建设备的设备ID。
       </li>
       <li>
        <code>
         ONENET_PROJECT_ID
        </code>
        - 云平台所创建的产品的产品ID。
       </li>
       <li>
        <code>
         ONENET_AUTH_INFO
        </code>
        - 自己设置的鉴权信息。
       </li>
       <li>
        <code>
         ONENET_DATA_STREAM
        </code>
        - 自己所创建的数据流的名称。
       </li>
      </ul>
     </li>
     <li>
      编译工程：
      <br/>
      <ul>
       <li>
        指定 ESP-IDF 所在路径：
        <code>
         export IDF_PATH=/你的/ESP/IDF/所在的/路径
        </code>
        。
       </li>
       <li>
        编译&amp;烧写：
        <code>
         make &amp; make flash
        </code>
       </li>
      </ul>
     </li>
    </ul>
