    </p>
    <h2>
     <a id="_1">
     </a>
     一、引言：
    </h2>
    <p>
     jdbc接口的写法:
     <br/>
     是一种用于执行sql语句的JAVA API
     <br/>
     通过这个api可以直接执行sql脚本
    </p>
    <p>
     使用jmeter压测接口的时候，有时候需要批量造数据，需使用jmeter连数据库造对应的测试数据。
     <br/>
     或者测试结束后，对测试的数据还原，删掉一些垃圾数据，都会用到连接数据库执行sql的操作。
    </p>
    <h2>
     <a id="_8">
     </a>
     二、数据库驱动包下载与安装
    </h2>
    <p>
     jmeter 连mysql 数据库需安装对应的驱动包，
     <br/>
     <strong>
      1.下载
     </strong>
     <br/>
     下载地址：
     <br/>
     https://dev.mysql.com/downloads/connector/j/
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/582f1e3d4c48b3f2863ccbbce5015208.png">
      <br/>
      <strong>
       2.安装
      </strong>
      <br/>
      下载完成后解压放到jmeter的lib/ext下
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d3773102718c69fe5a6bed15a7d25229.png">
       <br/>
       重启jmeter
       <br/>
       没有要求jar包和数据库版本一致，即不用考虑
      </img>
     </img>
    </p>
    <h2>
     <a id="JDBC_20">
     </a>
     三、JDBC的连接配置
    </h2>
    <p>
     添加配置元件-JDBC Connection Configuration
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4398121a4019a700770fb1bf7f61ed18.png">
      <br/>
      先弄清楚自己的mysql数据库相关信息：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c0cfdb5e85b7a48177844931eda2e17e.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ab780d83dfa372c6ef62abc2f60f3044.png">
        <br/>
        <mark>
         配置连接数据库mysql相关的配置内容
        </mark>
       </img>
      </img>
     </img>
    </p>
    <ul>
     <li>
      Variable Name for created pool:
      <br/>
      自己定义连接数据库的名称，自己知道是连的哪个数据库就行，如：jdbc_web
     </li>
     <li>
      Database URL：数据库url，jdbc:mysql://主机ip或者机器名称:mysql监听的端口号/数据库名称，
      <br/>
      如：jdbc:mysql://localhost:3306/kyj
      <br/>
      url后面的参数（可以不写）characterEncoding=utf8&amp;useSSL=true&amp;serverTimezone=UTC
      <br/>
      设置时区和编码格式
     </li>
     <li>
      JDBC Driver class：JDBC驱动,我用的mysql选com.mysql.jdbc.Driver
     </li>
     <li>
      username：登陆数据库的用户名
     </li>
     <li>
      passwrod：登陆数据库的密码
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a092eddc04e0f164d9f2a3da17c2ef4.png">
       <br/>
       <strong>
        JDBC Driver
       </strong>
       <br/>
       不同的数据库和驱动连接方式参考下表
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/340d0396fb54e2d5a62691913b02f2f6.png"/>
      </img>
     </li>
    </ul>
    <h2>
     <a id="JDBC_request_41">
     </a>
     四、JDBC request请求
    </h2>
    <p>
     线程组-添加-取样器-JDBC Request
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f555d95b155fdcee2eb7ed1bcac238cd.png">
      <br/>
      <mark>
       JDBC Request 请求里面写个简单的查询SQL验证下是否能查到数据
      </mark>
     </img>
    </p>
    <ul>
     <li>
      Variable Name for created pool 前面自己定义的连接数据库的名称
     </li>
     <li>
      Query Type: 查询的sql选择 Select Statement选项
     </li>
     <li>
      Query: 写个查询的SQL select * from cb_account limit 10
      <br/>
      jmeter中写sql语句，末尾不建议用;
      <br/>
      jdbc request请求中，不支持写多条语句
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/44a655f2d01f915c61cebc1bea818590.png"/>
      <br/>
      2.query type
      <br/>
      select statement 查询脚本，不带变量参数
      <br/>
      updata statement 增删改脚本，不带变量参数
      <br/>
      Prepared select statement 查询脚本，带变量参数
      <br/>
      Prepared statement 增删改脚本，带变量参数
      <br/>
      带变量参数场景，如下图所示，按照下图进行操作
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/74820add6cc1925823e55d44c3032568.png"/>

     </li>
    </ul>
