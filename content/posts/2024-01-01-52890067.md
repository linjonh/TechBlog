     <h2 id="运维平台:8e6e8878ca7d415631798f7cc4fcd04e" style="margin:20px 0px 10px; font-size:36px; font-weight:500; line-height:1.1; color:inherit">
      运维平台
     </h2>
     <p style="margin-top:0px; margin-bottom:10px">
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       elastic-job
      </code>
      运维平台以
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       war
      </code>
      包形式提供，可自行部署到
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       tomcat
      </code>
      或
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       jetty
      </code>
      等支持
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       servlet
      </code>
      的
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       web
      </code>
      容器中。
     </p>
     <p style="margin-top:0px; margin-bottom:10px">
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       elastic-job-console.war
      </code>
      可以通过编译源码或从
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       maven
      </code>
      中央仓库获取。
     </p>
     <h3 id="登录:8e6e8878ca7d415631798f7cc4fcd04e" style="font-weight:500; line-height:1.1; color:inherit; margin-top:20px; margin-bottom:10px; font-size:30px">
      登录
     </h3>
     <p style="margin-top:0px; margin-bottom:10px">
      默认用户名和密码是
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       root/root
      </code>
      ，可以通过修改
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       conf\auth.properties
      </code>
      文件修改默认登录用户名和密码。
     </p>
     <h3 id="主要功能:8e6e8878ca7d415631798f7cc4fcd04e" style="font-weight:500; line-height:1.1; color:inherit; margin-top:20px; margin-bottom:10px; font-size:30px">
      主要功能
     </h3>
     <ul style="margin-top:0px; margin-bottom:10px; line-height:1.6em">
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        登录安全控制
       </p>
      </li>
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        注册中心管理
       </p>
      </li>
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        作业维度状态查看
       </p>
      </li>
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        服务器维度状态查看
       </p>
      </li>
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        快捷修改作业设置
       </p>
      </li>
      <li style="">
       <p style="margin-top:0px; margin-bottom:10px">
        控制作业暂停，恢复运行，停止和删除
       </p>
      </li>
     </ul>
     <h3 id="设计理念:8e6e8878ca7d415631798f7cc4fcd04e" style="font-weight:500; line-height:1.1; color:inherit; margin-top:20px; margin-bottom:10px; font-size:30px">
      设计理念
     </h3>
     <p style="margin-top:0px; margin-bottom:10px">
      运维平台和
      <code style="font-family:Menlo,Monaco,Consolas,'Courier New',monospace; font-size:14px; padding:2px 4px; color:rgb(199,37,78); background-color:rgb(249,242,244)">
       elastic-job
      </code>
      并无直接关系，是通过读取作业注册中心数据展现作业状态，或更新注册中心数据修改全局配置。
     </p>
     <p style="margin-top:0px; margin-bottom:10px">
      控制台只能控制作业本身是否运行，但不能控制作业进程的启停，因为控制台和作业本身服务器是完全分布式的，控制台并不能控制作业服务器。
     </p>
     <h3 id="不支持项:8e6e8878ca7d415631798f7cc4fcd04e" style="font-weight:500; line-height:1.1; color:inherit; margin-top:20px; margin-bottom:10px; font-size:30px">
      不支持项
     </h3>
     <ul style="margin-top:0px; margin-bottom:10px; line-height:1.6em">
      <li style="">
       添加作业。因为作业都是在首次运行时自动添加，使用运维平台添加作业并无必要。
      </li>
     </ul>
     <h3 id="主要界面:8e6e8878ca7d415631798f7cc4fcd04e" style="font-weight:500; line-height:1.1; color:inherit; margin-top:20px; margin-bottom:10px; font-size:30px">
