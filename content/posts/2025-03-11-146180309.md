    <h3>
     一、学习阶段与目标设定
    </h3>
    <h4>
     1. 基础认知时期（1 - 2 个月）
    </h4>
    <p>
     <strong>
      目标
     </strong>
     ：搭建起 Linux 基础认知架构
    </p>
    <ul>
     <li>
      <strong>
       学习要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         系统安装
        </strong>
        ：在虚拟机里安装 Ubuntu 22.04 或者 CentOS Stream 8。
       </li>
       <li>
        <strong>
         命令行操作
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           ls
          </code>
          、
          <code>
           cd
          </code>
          、
          <code>
           pwd
          </code>
          、
          <code>
           mkdir
          </code>
          等 30 个基础命令。
         </li>
         <li>
          熟悉
          <code>
           grep
          </code>
          、
          <code>
           awk
          </code>
          、
          <code>
           sed
          </code>
          等文本处理工具。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         文件系统
        </strong>
        ：
        <ul>
         <li>
          了解根目录下各主要目录的功能，像
          <code>
           /etc
          </code>
          、
          <code>
           /var
          </code>
          、
          <code>
           /usr
          </code>
          。
         </li>
         <li>
          学会使用
          <code>
           df
          </code>
          、
          <code>
           du
          </code>
          、
          <code>
           find
          </code>
          等命令进行磁盘管理。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         用户权限
        </strong>
        ：
        <ul>
         <li>
          能够进行用户和组的创建、删除操作。
         </li>
         <li>
          掌握
          <code>
           chmod
          </code>
          、
          <code>
           chown
          </code>
          、
          <code>
           sudo
          </code>
          的使用方法。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         编辑器使用
        </strong>
        ：
        <ul>
         <li>
          熟练运用
          <code>
           nano
          </code>
          进行简单文本编辑。
         </li>
         <li>
          了解
          <code>
           vim
          </code>
          的基本模式。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       学习资源
      </strong>
      ：
      <ul>
       <li>
        书籍：《鸟哥的 Linux 私房菜》基础篇。
       </li>
       <li>
        网站：
        <a href="https://www.linuxcommand.org/" rel="nofollow" title="Linux Command">
         Linux Command
        </a>
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     2. 系统管理时期（2 - 3 个月）
    </h4>
    <p>
     <strong>
      目标
     </strong>
     ：达成服务器级别的管理能力
    </p>
    <ul>
     <li>
      <strong>
       学习要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         软件包管理
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           apt
          </code>
          /
          <code>
           apt - get
          </code>
          （适用于 Debian 系）、
          <code>
           yum
           <font face="-apple-system, SF UI Text, Arial, PingFang SC, Hiragino Sans GB, Microsoft YaHei, WenQuanYi Micro Hei, sans-serif, SimHei, SimSun">
            /
           </font>
          </code>
          <code>
           dnf
          </code>
          （适用于 RedHat 系）的使用。
         </li>
         <li>
          了解
          <code>
           dpkg
          </code>
          、
          <code>
           rpm
          </code>
          底层原理。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         进程管理
        </strong>
        ：
        <ul>
         <li>
          能够使用
          <code>
           ps
          </code>
          、
          <code>
           top
          </code>
          、
          <code>
           htop
          </code>
          、
          <code>
           kill
          </code>
          等命令。
         </li>
         <li>
          熟悉
          <code>
           systemd
          </code>
          的服务管理，如
          <code>
           systemctl
          </code>
          。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         网络配置
        </strong>
        ：
        <ul>
         <li>
          掌握静态 IP 配置方法。
         </li>
         <li>
          学会使用
          <code>
           netstat
          </code>
          、
          <code>
           ss
          </code>
          、
          <code>
           ping
          </code>
          、
          <code>
           traceroute
          </code>
          等工具。
         </li>
         <li>
          了解 SSH 服务的配置与使用。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         日志分析
        </strong>
        ：
        <ul>
         <li>
          能够查看
          <code>
           /var/log
          </code>
          下的常见日志。
         </li>
         <li>
          掌握
          <code>
           journalctl
          </code>
          的使用。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         Shell 脚本
        </strong>
        ：
        <ul>
         <li>
          熟悉变量、循环、条件判断等语法。
         </li>
         <li>
          能够编写简单的自动化脚本。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       学习资源
      </strong>
      ：
      <ul>
       <li>
        实践：通过
        <code>
         Docker
        </code>
        部署
        <code>
         Nginx
        </code>
        服务。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     3. 服务搭建时期（3 - 4 个月）
    </h4>
    <p>
     <strong>
      目标
     </strong>
     ：实现企业级服务的部署与维护
    </p>
    <ul>
     <li>
      <strong>
       学习要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         Web 服务
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           Nginx
          </code>
          、
          <code>
           Apache
          </code>
          的配置与优化。
         </li>
         <li>
          熟悉
          <code>
           PHP
          </code>
          /
          <code>
           Python
          </code>
          环境的搭建（如
          <code>
           FastCGI
          </code>
          ）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         数据库管理
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           MySQL
          </code>
          /
          <code>
           MariaDB
          </code>
          的安装与配置。
         </li>
         <li>
          熟悉基本的 SQL 操作和用户权限管理。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         存储管理
        </strong>
        ：
        <ul>
         <li>
          了解 LVM 逻辑卷管理。
         </li>
         <li>
          掌握
          <code>
           RAID
          </code>
          配置（软 RAID）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         安全加固
        </strong>
        ：
        <ul>
         <li>
          熟悉
          <code>
           iptables
          </code>
          防火墙规则设置。
         </li>
         <li>
          了解
          <code>
           SELinux
          </code>
          /
          <code>
           AppArmor
          </code>
          的基本策略。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         高可用性
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           Keepalived
          </code>
          实现双机热备。
         </li>
         <li>
          了解
          <code>
           NFS
          </code>
          /
          <code>
           CIFS
          </code>
          共享存储。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       学习资源
      </strong>
      ：
      <ul>
       <li>
        项目：
        <a href="https://www.linuxserver.io/" rel="nofollow" title="LinuxServer.io">
         LinuxServer.io
        </a>
        的容器部署方案。
       </li>
       <li>
        文档：
        <code>
         Nginx
        </code>
        官方文档（
        <a href="https://nginx.org/" rel="nofollow" title="nginx.org">
         nginx.org
        </a>
        ）。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     4. 云原生与自动化时期（4 - 6 个月）
    </h4>
    <p>
     <strong>
      目标
     </strong>
     ：紧跟 DevOps 技术趋势
    </p>
    <ul>
     <li>
      <strong>
       学习要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         容器化技术
        </strong>
        ：
        <ul>
         <li>
          熟练使用
          <code>
           Docker
          </code>
          进行镜像构建和容器编排。
         </li>
         <li>
          掌握
          <code>
           Docker Compose
          </code>
          多容器部署。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         Kubernetes
        </strong>
        ：
        <ul>
         <li>
          熟悉集群搭建（使用
          <code>
           kubeadm
          </code>
          ）。
         </li>
         <li>
          掌握
          <code>
           Pod
          </code>
          、
          <code>
           Service
          </code>
          、
          <code>
           Deployment
          </code>
          等资源管理。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         自动化工具
        </strong>
        ：
        <ul>
         <li>
          了解
          <code>
           Ansible
          </code>
          剧本编写和模块使用。
         </li>
         <li>
          熟悉
          <code>
           GitLab CI/CD
          </code>
          流水线配置。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         监控系统
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           Prometheus
          </code>
          +
          <code>
           Grafana
          </code>
          监控方案。
         </li>
         <li>
          了解
          <code>
           Elasticsearch
          </code>
          +
          <code>
           Logstash
          </code>
          +
          <code>
           Kibana
          </code>
          日志分析。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         云平台实践
        </strong>
        ：
        <ul>
         <li>
          在
          <code>
           AWS EC2
          </code>
          或
          <code>
           阿里云 ECS
          </code>
          上进行实例管理。
         </li>
         <li>
          了解云存储（如
          <code>
           S3
          </code>
          、
          <code>
           OSS
          </code>
          ）和负载均衡。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       学习资源
      </strong>
      ：
      <ul>
       <li>
        认证：
        <code>
         CKA
        </code>
        （Certified Kubernetes Administrator）备考资料。
       </li>
       <li>
        社区：
        <code>
         Kubernetes
        </code>
        官方论坛。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     5. 架构设计时期（6 个月以上）
    </h4>
    <p>
     <strong>
      目标
     </strong>
     ：培养复杂系统的设计与优化能力
    </p>
    <ul>
     <li>
      <strong>
       学习要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         微服务架构
        </strong>
        ：
        <ul>
         <li>
          了解
          <code>
           Service Mesh
          </code>
          （如
          <code>
           Istio
          </code>
          ）。
         </li>
         <li>
          掌握服务发现与注册（如
          <code>
           Consul
          </code>
          ）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         存储优化
        </strong>
        ：
        <ul>
         <li>
          熟悉分布式文件系统（如
          <code>
           Ceph
          </code>
          ）。
         </li>
         <li>
          了解
          <code>
           NoSQL
          </code>
          数据库（如
          <code>
           MongoDB
          </code>
          、
          <code>
           Redis
          </code>
          ）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         性能调优
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           perf
          </code>
          、
          <code>
           sysstat
          </code>
          等性能分析工具。
         </li>
         <li>
          了解内核参数优化（如
          <code>
           sysctl
          </code>
          ）。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         灾难恢复
        </strong>
        ：
        <ul>
         <li>
          掌握
          <code>
           BackupPC
          </code>
          、
          <code>
           borgbackup
          </code>
          等备份方案。
         </li>
         <li>
          熟悉容灾演练流程。
         </li>
        </ul>
       </li>
       <li>
        <strong>
         开源贡献
        </strong>
        ：
        <ul>
         <li>
          参与
          <code>
           Linux
          </code>
          内核或
          <code>
           Kubernetes
          </code>
          社区项目。
         </li>
         <li>
          学习开源协作规范（如
          <code>
           Git
          </code>
          工作流）。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       学习资源
      </strong>
      ：
      <ul>
       <li>
        书籍：《设计数据密集型应用》。
       </li>
       <li>
        案例：
        <code>
         Netflix
        </code>
        或
        <code>
         Google
        </code>
        的云架构实践分享。
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     二、技能成长路径图
    </h3>
    <pre><code>基础命令 → 文件系统 → 用户权限 → 服务管理 →
网络配置 → Shell 脚本 → 容器化 → 云原生 →
微服务架构 → 系统优化 → 开源贡献
</code></pre>
    <h3>
     三、学习方法建议
    </h3>
    <ol>
     <li>
      <strong>
       分阶段实践
      </strong>
      ：
      <ul>
       <li>
        每个阶段设定 1 - 2 个实战项目，例如搭建博客系统、部署容器化电商平台。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       对比学习
      </strong>
      ：
      <ul>
       <li>
        对比不同发行版（如 Ubuntu 与 CentOS）的差异。
       </li>
       <li>
        对比传统部署与云原生架构的优劣。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       社区参与
      </strong>
      ：
      <ul>
       <li>
        加入
        <code>
         Linux
        </code>
        或
        <code>
         Kubernetes
        </code>
        技术社群。
       </li>
       <li>
        定期参加线上技术沙龙或 Meetup。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       认证考试
      </strong>
      ：
      <ul>
       <li>
        考取
        <code>
         LPIC - 1
        </code>
        、
        <code>
         RHCSA
        </code>
        、
        <code>
         CKA
        </code>
        等证书。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     通过这种递进式的学习路径，可逐步从基础操作层面提升至架构设计层面，最终成为具备云原生思维的 Linux 技术专家。
    </p>
