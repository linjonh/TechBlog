    <p>
     【RDO一体化部署OpenStack】**
     <br/>
     0. 环境预备(CentOS7已经安装并配置静态IP)**
     <br/>
     0.1 指定DNS服务器的IP(Specifies the IP address of the DNS server)
     <br/>
     <strong>
      如果出现下面图片则说明已经默认配置好了
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4173bed3a17eb955255e16d84a7243e2.png"/>
    </p>
    <p>
     hostname 查看域名（主机名）：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/38494b8fd1eacd56a653e8bf5a37fdce.png">
      <br/>
      ip addr 查看虚拟机ip:
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f71a5500df82ae91178817a523f482d8.png">
       <br/>
       <strong>
        0.2 IP与域名(或主机名)映射
       </strong>
       <br/>
       vim /etc/hosts
       <br/>
       192.168.253.100 tsw01
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bce543c47a79e76b8f41a3a900bdc2bb.png">
        <br/>
        0.3 关闭防火墙
        <br/>
        systemctl stop firewalld
        <br/>
        systemctl disable firewalld
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fdc79672eab5fdf9ebbda6e4496f742c.png">
         <br/>
         0.4 关闭SELinux
         <br/>
         vim /etc/selinux/config #SELINUX=enforcing 改成 SELINUX=disabled
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/90bb2776bf836c9b7b162dc332ca33c0.png">
          <br/>
          注意，要重启系统后(reboot)，SELinux的修改才能生效
          <br/>
          另，查看SELinux的状态：getenforce sestatus /usr/sbin/sestatus -v
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ecc9b4a7fff29c2f3910db2300e0553.png">
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0b4d2edee0e8bf14caec732b07855b41.png">
            <br/>
            <strong>
             0.5 网络管理由NetworkManager切换成network
            </strong>
            <br/>
            CentOS7默认使用NetworkManager进行网络管理(不支持OpenStack)
            <br/>
            –查看网络管理状态
            <br/>
            systemctl status NetworkManager
            <br/>
            systemctl status network
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c99d97bd87c1e5d501dc78b154788bdf.png"/>
            <br/>
            -
            <strong>
             -停止禁用NetworkManager
            </strong>
            <br/>
            systemctl stop NetworkManager
            <br/>
            systemctl disable NetworkManager
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b630ac99695f41a1a013e33eb4bec61.png"/>
    </p>
    <p>
     –启用启动network
     <br/>
     systemctl enable network
     <br/>
     systemctl restart network
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c791f782a4b124ade0890c3e904aee41.png"/>
     <br/>
     0.6 更新系统
     <br/>
     yum update -y
     <br/>
     看到Complete！就成功了
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a8a8d75d1e0ceeec1ff660898d2f4892.png"/>
     <br/>
     <strong>
      1. 准备所需软件库
     </strong>
     <br/>
     1.1 安装Open Stack Queens
     <br/>
     yum install -y centos-release-openstack-queens
    </p>
    <p>
     看到Complete！就成功了
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cbaf8132d6811e4242852c3c9d875faf.png"/>
     <br/>
     1.2 启用Open Stack Queens资源库
     <br/>
     yum-config-manager --enable centos-openstack-queens
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dc5cf2dfd4a0eb0a14c0e350428645e9.png"/>
     <br/>
     可以看见上图没有找到这个命令，我们只需要安装这个i命令
     <br/>
     注：yum-config-manager在yum-utils包里，系统默认没有该命令，需通过如下命令安装
     <br/>
     yum install -y yum-utils
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2bf8415a05db613dedc9ebf823cee93e.png"/>
    </p>
    <p>
     <strong>
      yum repolist enabled 查询已启用的资源库
      <br/>
      yum repolist disabled 查询已禁用的资源库
      <br/>
      yum repolist all 查询所有资源库
      <br/>
      上面的三个命令需要的自己输入查看即可：
     </strong>
    </p>
    <p>
     安装命令后重新启用Open Stack Queens资源库
     <br/>
     yum-config-manager --enable centos-openstack-queens
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0da78f31f3e91eab8dea71555dd06e95.png"/>
     <br/>
     1.3切换到软件源定义目录
     <br/>
     cd /etc/yum.repos.d/
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/da72b37ab0a8cf8b290bf94592eadf03.png"/>
     <br/>
     <strong>
      1.4 下载软件源定义文件(RDO官网针对CentOS7的)
     </strong>
     <br/>
     curl -O https://trunk.rdoproject.org/centos7/delorean-deps.repo
     <br/>
     curl -O https://trunk.rdoproject.org/centos7/current-passed-ci/delorean.repo
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6f87f3e0f7d4f60a7bc23080f206b7e6.png"/>
    </p>
    <p>
     1.6 升级软件包及系统内核(改变软件及系统设置)
     <br/>
     yum update -y
    </p>
    <p>
     记住（每次升级后要重启虚拟机 命令： reboot）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5f030fb4c658fec54581bd8edd2e22f4.png"/>
     <br/>
     <strong>
      2. 安装Packstack
     </strong>
     <br/>
     执行如下命令安装openstack-packstack及其依赖包
     <br/>
     yum install -y openstack-packstack
     <br/>
     注，过程中会安装许多依赖包，如openstack-packstack-puppet
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9c773fb7e6848e932525b4e1aa447213.png"/>
     <br/>
     3. Packstack安装OpenStack
     <br/>
     packstack --allinone
     <br/>
     出现了上面这个界面就成功了一大半了，如果出现了红色字体（也就是报错了），那就反复输入packstack --allinone进行安装
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3da47125741fc82a7ad64e5aa266b124.png"/>
     <br/>
     注意这一步安装十分缓慢，请耐心等待（一般半小时到两个半小时）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8faf91a521c3de6f8a356cb234561873.png"/>
     <br/>
     如下图所示报了一部分错，报错了重新安装后半部分即可，安装命令（packstack --answer-file=/root/packstack-answers-20211124-114133.txt），后面的packstack-answers-20211124-114133.txt可以用当前界面下ls命令查看。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/debc452c8af0e8d8d11a31987dc66cb2.png"/>
     <br/>
     下图是安装成功界面，用网站访问界面，看看是否成功（http://192.168.253.100/dashboard）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7c05e77cf255806e1a2d33f37ed3bf62.png"/>
     <br/>
     进入的密码通过ls 查看目录 cat ./keystonerc_admin 查看账号密码

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17bfa45cf36d3487b3befb10f68ec8d5.png"/>
