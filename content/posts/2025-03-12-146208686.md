    <h2>
     <a id="centos7yumredis_0">
     </a>
     centos7通过yum安装redis
    </h2>
    <h3>
     <a id="1redis_1">
     </a>
     1.安装redis数据库
    </h3>
    <pre><code>yum install -y redis
</code></pre>
    <h3>
     <a id="2redis_7">
     </a>
     2.启动redis服务
    </h3>
    <pre><code>systemctl start redis
</code></pre>
    <h3>
     <a id="3redis_12">
     </a>
     3.查看redis状态
    </h3>
    <pre><code>systemctl status redis
</code></pre>
    <h3>
     <a id="4_17">
     </a>
     4、停止服务
    </h3>
    <pre><code>systemctl stop redis
</code></pre>
    <h3>
     <a id="5_22">
     </a>
     5、重启服务
    </h3>
    <pre><code>systemctl restart redis
</code></pre>
    <h3>
     <a id="6redis_27">
     </a>
     6、查看redis进程
    </h3>
    <pre><code>ps -ef | grep redis
</code></pre>
    <h3>
     <a id="7_33">
     </a>
     7、开放端口号
    </h3>
    <pre><code>firewall-cmd --zone=public --add-port=6379/tcp --permanent
</code></pre>
    <h3>
     <a id="8_38">
     </a>
     8、重启防火墙
    </h3>
    <pre><code>systemctl restart firewalld
</code></pre>
    <h3>
     <a id="9_43">
     </a>
     9、查看端口
    </h3>
    <pre><code>netstat -lpn | grep 6379
</code></pre>
    <h3>
     <a id="10redis_48">
     </a>
     10、设置redis远程连接和密码
    </h3>
    <pre><code>/etc/redis.conf
bin 127.0.0.1 注释掉，否则只有本地才能访问
#bin 127.0.0.1
保护模式protected-mode yes 
改为protected-mode no
要想修改端口号修改port即可
修改密码
requirepass 123456
</code></pre>
    <h3>
     <a id="11redisredis_60">
     </a>
     11、重启redis，进入redis
    </h3>
    <pre><code>redis-cli -h 127.0.0.1 -p 6379
然后输入info，提示必须验证
输入用户名和密码
auth 12345
</code></pre>
