    <h3>
     <a id="__1">
     </a>
     📝 问题背景
    </h3>
    <p>
     在Vue3项目中，当我们需要将包含图片的数据导出到Excel时，常用的
     <code>
      sheetjs/xlsx
     </code>
     库存在局限性：
     <strong>
      无法直接导出图片到单元格
     </strong>
     。本文将提供完整的解决方案，封装可直接复用的工具函数。
    </p>
    <hr/>
    <h3>
     <a id="__7">
     </a>
     🎯 解决方案
    </h3>
    <h4>
     <a id="_9">
     </a>
     技术选型
    </h4>
    <p>
     使用
     <code>
      exceljs
     </code>
     +
     <code>
      file-saver
     </code>
     组合：
    </p>
    <ul>
     <li>
      <code>
       exceljs
      </code>
      ：支持图片插入的Excel操作库
     </li>
     <li>
      <code>
       file-saver
      </code>
      ：前端文件保存工具
     </li>
    </ul>
    <h4>
     <a id="_14">
     </a>
     功能特性
    </h4>
    <p>
     ✅ 支持多图片列导出
     <br/>
     ✅ 自动识别Base64和DataURL格式
     <br/>
     ✅ 精确控制图片位置
     <br/>
     ✅ 异常处理与性能优化
    </p>
