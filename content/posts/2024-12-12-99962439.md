      运维工作有没有意思？运维有没有前途？
     </p>
     <p style="letter-spacing:.544px;font-size:16px;color:rgb(79,79,79);line-height:26px;text-align:center;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      运维是不是要被各种技术取代？
     </p>
     <p style="text-align:center;">
      <img alt="640" src="https://i-blog.csdnimg.cn/blog_migrate/71f53171c59a572a6ea9f7785fba84e9.jpeg"/>
     </p>
     <p>
     </p>
     <p style="color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;text-indent:2em;">
      <span style="color:rgb(89,89,89);font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:15px;letter-spacing:1px;">
       希望能通过本文向各位描述清楚运维到底是干什么的，至于有没有前途、发展以及会不会失业等，请读者自行判断。
      </span>
     </p>
     <p style="color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;">
      <span style="color:rgb(89,89,89);font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:15px;letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:26px;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <strong>
       运维是干什么的？
      </strong>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       「运维」二字可能有几层意思，分别可以指代运维工程师、运维团队或者是整个运维服务体系。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       我们可以看出，这三层是从狭义到广义的递进。相信绝大部分人问的都是运维工程师，只有极少数人能意识到还有运维服务体系这一层含义。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,0,0);">
       我们经常会听到一些言论，比如：
      </span>
     </p>
     <ul class="list-paddingleft-2" style="list-style-type:square;">
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-weight:700;font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         云服务普及了，运维工程师就要失业了。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-weight:700;font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         等 DevOps 或者 SRE 落地了，运维工程师也要失业了。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-weight:700;">
         <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         </span>
         <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
          容器技术普及了，运维工程师也该失业了……
         </span>
        </span>
       </p>
      </li>
     </ul>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       也记不清运维工程师到底
       <strong>
        被失业
       </strong>
       了多少遍，但我认为就算运维工程师被取代了，运维服务也不会消亡，它将伴随并支撑着业务发展的整个生命周期。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       为何这样说？我们还是用业务的诞生过程来分析。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,0,0);">
       一个站点或者 App，大致经历着这样的诞生过程：
      </span>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       PM 设计出产品原型，交给 Dev 开发实现、QA 测试，然后交付给 Ops 部署到线上运行，最后供用户使用。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       在这几个简单步骤中涉及了众多的人、角色、交付过程等对象，这是一个完整、复杂的系统工程，而任意一个环节的失误都可能影响最终呈现给用户的体验以及效果。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       我们重点考虑从 Dev 把业务产品完成后交付给 Ops 到线上运行的这个阶段，Dev 同事主要负责业务产品的功能完整、逻辑正确等业务指标，而 Ops 同事主要负责业务产品的运行质量、稳定性、可用性等系统指标。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       无论后面的交付步骤是用 DevOps 还是 SRE 的实现方式，都离不开一个广义的运维服务的执行环节。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       所以说， Dev 还是 Dev，Ops 还是 Ops，没有谁被取代，只是运维服务的执行方式升级为更加软件工程化的手段，减少人肉操作，DevOps 强调自动化、拉动式来提高团队交付效率与质量。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       而
      </span>
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
        传统的运维
       </span>
      </strong>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       需要谋求技术转型，从原来只关注操作系统层面的技术已经不够了，还要增加对程序代码的性能调优、持续交付、容器化等软件基础架构方面的技能提升，也需要持续关注整个业务、应用、服务的生命周期管理。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       简单来说，就是把过去传统的黑盒运维的思维方式抛弃，进入白盒运维的时代，我们必须更加深入代码、深入业务运营，让整个线上服务运行于更优质高效的状态。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       至于运维是否会被取代，取决于你属于
      </span>
      <span style="font-size:15px;letter-spacing:1px;text-decoration:underline;color:rgb(255,76,0);">
       <strong>
        哪种运维
       </strong>
      </span>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:26px;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <strong>
       运维工程师和运维开发工程师
      </strong>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       要建设运维自动化或者实践 DevOps 离不开运维开发工程师的参与，但要怎样才能更好地发挥运维开发的作用呢？
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       我曾作为运维产品经理的角色和各种类型的运维开发一起协作过，团队中有本来就做运维开发的，也有本来做其他业务（电商、平台）的开发转来协助运维团队的。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,0,0);">
       和他们协作一段日子后，总体感觉如下：
      </span>
     </p>
     <ul class="list-paddingleft-2" style="list-style-type:square;margin-left:8px;">
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         运维开发首先是一个程序员，不是运维工程师。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         一个好的运维开发需要具备
         <span class="Apple-converted-space">
         </span>
        </span>
        <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
         <strong>
          「运维理解」+「开发能力」
         </strong>
        </span>
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         对「开发能力」的技术要求低于其他业务形态（如游戏、电商、搜索等）。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         对运维业务的理解难度会低于电商、游戏等业务形态，即对「运维理解」的要求不高。
        </span>
       </p>
      </li>
      <li>
       <p style="font-size:16px;color:rgb(79,79,79);line-height:1.75em;">
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         对运维相关技术栈的掌握程度要求高，
        </span>
        <strong>
         <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
          如 Linux、Git、Nginx、Zabbix、Docker、K8S
          <span class="Apple-converted-space">
          </span>
         </span>
        </strong>
        <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
         等。
        </span>
       </p>
      </li>
     </ul>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       综上所述，运维开发是一个深度不算太深的职业分支，而现在之所以对运维开发需求量热起来了，主要由于老一辈的资深运维普遍研发能力有限，而这是有历史原因的。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       对于从业 8 年以上的资深运维来说，他们刚开始做运维的时候更多的是接触机房、机架、主机、交换机、防火墙等硬件设备。然后对接业务运维后，一般通过 Shell、Python 等脚本来辅助工作。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       等到业界提出 DevOps 的时候，他们往往已经专注于团队管理、容量规划、架构调优、运维服务质量等高级范畴，所以基本不太可能抽出大块的时间来重新学习编码并开发自动化系统。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       所以，当我们有自动化系统的建设需求时，需要更
      </span>
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
        专业
       </span>
      </strong>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       的程序员来协助。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       但一般的非专职运维开发的程序员做出来的系统对于运维来说往往不太好使，这时候有部分年轻的运维工程师升级了研发技能，转型运维开发，把好使的运维系统做出来了，赢得了运维团队的好评，大家都为「运维开发」点赞。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       所以，大家将 「好使的运维系统」 和 「运维开发」 等价起来，以为我们只要招来一个运维开发，那么一套完美的运维平台就能自动诞生出来，这是个很大的误区。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       「好使的运维系统」真正等价于
      </span>
      <span style="color:rgb(255,76,0);">
       <strong>
        <span style="font-size:15px;letter-spacing:1px;">
         「运维理解
        </span>
       </strong>
      </span>
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
        」+「开发能力」
       </span>
      </strong>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       ，这两种能力也是可以分离的，不一定要强加在运维开发工程师一个人的身上。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       类似其他业务形态的开发过程，需要产品经理和程序员两种角色分离，企业也不会说要招聘既会写代码、又会出需求的程序员。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       所以，当资深运维能把运维自动化的需求细致地文档化下来，把自动化系统的设计、架构等关键环节确立下来，这就是最好的「运维理解」。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       这时把这份靠谱、好使、细致的需求文档交给具备强「开发能力」的程序员，最终就可以得到「好使的运维系统」。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       再往高级发展的话，运维开发也可以替代资深运维出需求，升级为运维产品经理，以
      </span>
      <strong>
       <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
        程序员的思维角度
       </span>
      </strong>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       来解决运维服务的工程效率和质量问题，我认为这也是类似 Google 所提倡的 SRE 文化。
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       最后，很多运维可能考虑要不要转运维开发，当你觉得编码的乐趣远远大于其他运维技能的时候，尽管争取努力去转！
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:normal;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       把自己当成一个真正的程序员，以程序员的评价标准来要求自己，不要觉得运维能力和编码能力各自半桶水是好事，正如我前面的那句话：
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       “运维开发首先是一个程序员，不是运维工程师 。”
      </span>
     </p>
     <p style="color:rgb(51,51,51);font-family:'-apple-system-font', BlinkMacSystemFont, 'Helvetica Neue', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei', Arial, sans-serif;font-size:17px;letter-spacing:.544px;line-height:27.2px;text-align:justify;text-indent:2em;">
      <span style="color:rgb(89,89,89);font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:15px;letter-spacing:1px;">
       <br/>
      </span>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:26px;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-weight:700;color:rgb(255,76,0);">
       最低要求
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       运维是一种需要技能面比较广的工种，大家普遍都是处于技术面广但不深的状态，我把 2 级定义为科普级，意思是达到该级就可以满足各种日常工作要求。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       路人 A，最好尽快争取把还在1级水平的
      </span>
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
       <span class="Apple-converted-space">
       </span>
       Shell
      </span>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       <span class="Apple-converted-space">
       </span>
       和
       <span class="Apple-converted-space">
       </span>
      </span>
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
       MySQL
       <span class="Apple-converted-space">
       </span>
      </span>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       都提升到 2 级，就可以满足日常工作要求，这也是我们对运维工程师的最低要求。
      </span>
     </p>
     <p style="text-align:center;">
      <img alt="640" src="https://i-blog.csdnimg.cn/blog_migrate/d84e529f218b4aa4e33a88b9f18a3cd3.jpeg"/>
     </p>
     <p style="letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(79,79,79);line-height:26px;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;">
      <span style="font-weight:700;color:rgb(255,76,0);">
       进阶要求
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       除了满足最低要求之外，培养自己的核心竞争力，为日后的发展打下基础，推荐大家对 1~2 项深入学习，达到 4、5 级甚至更高的水平。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       随着互联网运维行业的各种 PaaS、IaaS 普及后，自动化程度越来越高，现在已经不像以前那样需要那么多「操作员」。
      </span>
     </p>
     <p style="margin-left:8px;letter-spacing:.544px;text-align:justify;font-size:16px;color:rgb(62,62,62);line-height:1.75em;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;text-indent:2em;">
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       也就是说，技能水平偏低的运维急需技能升级或者技能转型，能支撑你走多远的不是那些 1、2 级的技能，而是
       <span class="Apple-converted-space">
       </span>
      </span>
      <span style="font-size:15px;letter-spacing:1px;color:rgb(255,76,0);">
       4、5 级
      </span>
      <span style="font-size:15px;color:rgb(89,89,89);letter-spacing:1px;">
       以上的技能。
      </span>
     </p>
