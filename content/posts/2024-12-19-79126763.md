     python安装
    </h2>
    <ul>
     <li>
      <p>
       我安装的是

       </span>
       <span class="MathJax" id="MathJax-Element-1-Frame" style="position: relative;" tabindex="0">

           <span class="mrow" id="MathJax-Span-2">
            <span class="mstyle" id="MathJax-Span-3" style="color: blue;">
             <span class="mrow" id="MathJax-Span-4" style="color: blue;">
              <span class="mtext" id="MathJax-Span-5" style="font-family: MathJax_Main; color: blue;">
               python3.5.2
              </span>
             </span>
            </span>
           </span>
           <span style="display: inline-block; width: 0px; height: 2.212em;">
           </span>
          </span>
         </span>
         <span style="display: inline-block; overflow: hidden; vertical-align: -0.328em; border-left-width: 0px; border-left-style: solid; width: 0px; height: 1.292em;">
         </span>
        </span>
        <span class="MJX_Assistive_MathML">
         python3.5.2
        </span>
       </span>
       (以后描述的相关操作， 均是基于此版本)，具体网址:
       <br/>
       <a href="https://www.python.org/downloads/release/python-352/" rel="nofollow">
        https://www.python.org/downloads/release/python-352/
       </a>
      </p>
     </li>
     <li>
      <p>
       安装的时候会自动添加
       <strong>
        以下功能
       </strong>
      </p>
      <ol>
       <li>
        自动添加环境变量（看了下，貌似是添加到了用户变量中，并非系统变量， 建议还是自己添加一下到系统变量，具体路径从用户变量中复制即可）;s
       </li>
       <li>
        自动下载安装
        <strong>
         pip
        </strong>
        (
        <strong>
         pip
        </strong>
        是对python的附加包进行管理，比如安装、删除等)，另外，无需再安装
        <strong>
         easy_install
        </strong>
        ，好像是
        <strong>
         3.x
        </strong>
        版本都不需要， 不太清楚
       </li>
      </ol>
     </li>
     <li>
      <p>
       最麻烦的是不同版本之间的操作不同,2.x和3.x之间的操作就是完全不同的，比如：
      </p>
      <ul>
       <li>
        2.x为了管理安装包就先需要安装
        <strong>
         easy_install
        </strong>
        ， 然后再安装
        <strong>
         pip
        </strong>
        ；
       </li>
       <li>
        <strong>
         3.x
        </strong>
        ~
        <strong>
         3.y
        </strong>
        还需要手动安装
        <strong>
         pip
        </strong>
        ；
       </li>
       <li>
        但是
        <strong>
         3.y
        </strong>
        以后又会自动安装
        <strong>
         pip
        </strong>
        ；
       </li>
      </ul>
     </li>
    </ul>
    <p>
     综上所述， 安装时一定要弄清楚自己的版本号， 看教程/博客时， 也一定要弄清楚其描述的版本。
    </p>
