    <h3>
     <a id="Vue_0">
     </a>
     <strong>
      Vue脚手架基础
     </strong>
    </h3>
    <hr/>
    <p>
     <strong>
      NodeJS（轻量高效）
     </strong>
     –》相当于Java的jdk
    </p>
    <p>
     Node.js是基于Chrome浏览器引擎的JavaScript运行环境，所以它的作用就相当于是一个浏览器，解析HTML，css，JS代码
    </p>
    <p>
     <strong>
      随着 Node.js 的出现，JavaScript 在服务器端的应⽤逐渐增多，为了让 Node.js 的代码更好维护，就必须要制定⼀种 Node.js 环境下的模块化规范
     </strong>
    </p>
    <p>
     导出数据有
     <code>
      两种⽅式
     </code>
     ：
    </p>
    <ul>
     <li>
      第⼀种⽅式： module.exports = value
     </li>
     <li>
      第⼆种⽅式： exports.name = value
      <pre><code class="prism language-javascript"><span class="token comment">//导出函数</span>
exports<span class="token punctuation">.</span><span class="token function-variable function">add</span><span class="token operator">=</span><span class="token keyword">function</span><span class="token punctuation">(</span><span class="token parameter">a<span class="token punctuation">,</span>b</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> a<span class="token operator">+</span>b<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
      使⽤内置的
      <code>
       require
      </code>
      函数进⾏导⼊数据
     </li>
    </ul>
    <pre><code class="prism language-javascript"><span class="token comment">//导入函数</span>
<span class="token keyword">let</span> test<span class="token operator">=</span><span class="token function">require</span><span class="token punctuation">(</span><span class="token string">'./demo2_01.js'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>test<span class="token punctuation">.</span><span class="token function">add</span><span class="token punctuation">(</span><span class="token number">10</span><span class="token punctuation">,</span><span class="token number">20</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     <strong>
      Node JS的包管理器
     </strong>
     ：
     <strong>
      <code>
       npm
      </code>
      （全球最大的开源生态系统）
     </strong>
    </p>
    <p>
     <strong>
      1、通过npm init进行初始化操作
     </strong>
    </p>
    <p>
     初始化出来的
     <code>
      package.json
     </code>
     先相当于是
     <code>
      maven
     </code>
     的
     <code>
      pom
     </code>
     文件
    </p>
    <p>
     <strong>
      用于存放工程的依赖文件
     </strong>
    </p>
    <p>
     2、通过
     <code>
      npm install 要安装的框架名称
     </code>
     进行本地安装相关模块
    </p>
    <p>
     3、通过**
     <code>
      npm root -g
     </code>
     **查看本地电脑存放依赖的全局位置
    </p>
    <p>
     4、通过
     <code>
      npm install 要安装的框架名称 -g
     </code>
     进行全局安装相关模块
    </p>
    <p>
     <strong>
      5、通过
      <code>
       npm run dev
      </code>
      来运行工程
     </strong>
    </p>
    <p>
     <strong>
      6、通过
      <code>
       npm run build
      </code>
      来编译工程
     </strong>
    </p>
    <hr/>
    <h5>
     <a id="Webpack_48">
     </a>
     <strong>
      Webpack（静态模块打包器）–&gt;前端打包
     </strong>
    </h5>
    <hr/>
    <p>
     1、进行语法转换
    </p>
    <p>
     2、HTML，CSS，JS代码进行打包压缩合并
    </p>
    <p>
     3、webpack可以在开发期间提供一个开发服务器
    </p>
    <hr/>
    <p>
     –》相当于maven中打成war包类似（前端项目也是打包在上线）
    </p>
    <hr/>
    <h5>
     <a id="npm_64">
     </a>
     <strong>
      基于npm方式进行打包
     </strong>
    </h5>
    <p>
     npm run dev
    </p>
    <p>
     可以将dev中的webpack改为
     <strong>
      webpack-dev_server（开发服务器）
     </strong>
    </p>
    <p>
     这样子进行配置了之后那么对应的前端的打包就直接在内存中，而不是每次都需要去硬盘中进行修改
    </p>
    <hr/>
    <hr/>
    <h4>
     <a id="Vuecli_76">
     </a>
     <strong>
      Vue-cli脚手架
     </strong>
    </h4>
    <hr/>
    <p>
     <strong>
      重点：
     </strong>
    </p>
    <p>
     <strong>
      1、启动项目：
     </strong>
    </p>
    <p>
     <strong>
      <code>
       npm run dev
      </code>
     </strong>
    </p>
    <p>
     <strong>
      2、安装当前项目所需要的所有依赖：
     </strong>
    </p>
    <p>
     <strong>
      <code>
       npm install（npm i)
      </code>
     </strong>
    </p>
    <hr/>
    <p>
     vue-cli是Vue的一个基础框架，帮助我们快速搭建一个Vue的项目基础架子
    </p>
    <p>
     ~开箱即用
    </p>
    <p>
     ~零配置
    </p>
    <p>
     ~基于webpack，webpack-dev-server
    </p>
    <hr/>
    <p>
     安装脚手架到全局位置
    </p>
    <p>
     npm install vue-cli -g
    </p>
    <hr/>
    <h4>
     <a id="vuevueAPPvue_108">
     </a>
     <strong>
      项目目录（vue脚手架中后缀名为.vue的称为组件，APP.vue为根组件）
     </strong>
    </h4>
    <p>
     主要：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bd903c0211644c42b5cd0dd5d6381fca.png"/>
    </p>
    <p>
     <strong>
      <code>
       src--项目核心文件（我们所写的代码都放在这个文件夹下）
      </code>
     </strong>
    </p>
    <p>
     <strong>
      <code>
       assets-- 静态资源(样式类文件，如css,less,sass，以及一些外部js文件)
      </code>
     </strong>
    </p>
    <p>
     <strong>
      <code>
       components-- 公共组件
      </code>
     </strong>
    </p>
    <p>
     <strong>
      <code>
       router--路由（配置项目路由）
      </code>
     </strong>
    </p>
    <p>
     <strong>
      <code>
       App.vue --根组件
      </code>
     </strong>
    </p>
    <p>
     <strong>
      <code>
       mainjs-- 入口文件
      </code>
     </strong>
    </p>
    <hr/>
    <p>
     <strong>
      vue 项目的创建
     </strong>
    </p>
    <p>
     官方样例：
    </p>
    <p>
     <a href="https://cn.vuejs.org/guide/quick-start.html" rel="nofollow">
      快速上手 | Vue.js
     </a>
    </p>
    <p>
     创建一个项目：
    </p>
    <p>
     <code>
      npm create vue@latest
     </code>
    </p>
    <pre><code class="prism language-vue">&lt;template&gt;
    &lt;div&gt;
      书写HTML代码的只能书写一个根标签
    &lt;/div&gt;
&lt;/template&gt;
&lt;script&gt;
	书写js代码的
&lt;/script&gt;
&lt;style&gt;
	书写CSS代码的
&lt;/style&gt;
</code></pre>
    <hr/>

      </strong>
     </em>
    </h6>
