    <h4 id="3.1.%20%E5%88%9B%E5%BB%BA%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83">
     1 创建虚拟环境
    </h4>
    <pre><code class="language-bash">conda create -n env_name python=3.8</code></pre>
    <h4 id="3.2.%20%E5%88%9B%E5%BB%BA%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83%E7%9A%84%E5%90%8C%E6%97%B6%E5%AE%89%E8%A3%85%E5%BF%85%E8%A6%81%E7%9A%84%E5%8C%85">
     2 创建虚拟环境的同时安装必要的包
    </h4>
    <pre><code class="language-bash">conda create -n env_name numpy matplotlib python=3.8</code></pre>
    <h4 id="3.3%20%E6%9F%A5%E7%9C%8B%E6%9C%89%E5%93%AA%E4%BA%9B%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83">
     3 查看有哪些虚拟环境
    </h4>
    <p>
     以下三条命令都可以。注意最后一个是”--”，而不是“-”.
    </p>
    <pre><code class="language-bash">conda env list
conda info -e
conda info --envs</code></pre>
    <p>
     所显示的列表中，前面带星号“*“的表示当前活动环境。比如说当前我的环境列表：
    </p>
    <p>
     <img alt="" height="129" src="https://i-blog.csdnimg.cn/direct/e8e9b3001b8d4184b37661a2685daaa8.png" width="667"/>
    </p>
    <h4 id="3.4%20%E6%BF%80%E6%B4%BB%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83">
     4 激活虚拟环境
    </h4>
    <pre><code class="language-bash">conda activate env_name</code></pre>
    <h4 id="3.5%C2%A0%E9%80%80%E5%87%BA%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83">
     5 退出虚拟环境
    </h4>
    <pre><code class="language-bash">conda activate
conda deactivate</code></pre>
    <h4 id="3.5%20%E5%88%A0%E9%99%A4%E8%99%9A%E6%8B%9F%E7%8E%AF%E5%A2%83">
     5 删除虚拟环境
    </h4>
    <p>
     执行以下命令可以将该指定虚拟环境及其中所安装的包都删除。
    </p>
    <pre><code class="language-bash">conda remove --name env_name --all</code></pre>
    <p>
     如果只删除虚拟环境中的某个或者某些包则是：
    </p>
    <pre><code class="language-bash">conda remove --name env_name  package_name</code></pre>
    <h4 id="3.6%20%E5%AF%BC%E5%87%BA%E7%8E%AF%E5%A2%83%C2%A0">
     6 导出环境
    </h4>
    <p>
     很多的软件依赖特定的环境，我们可以导出环境，这样方便自己在需要时恢复环境，也可以提供给别人用于创建完全相同的环境。
    </p>
    <pre><code class="language-bash">#获得环境中的所有配置
conda env export --name myenv &gt; myenv.yml
#重新还原环境
conda env create -f  myenv.yml</code></pre>
    <p>
    </p>
    <h4 id="4.1%20%E6%9F%A5%E8%AF%A2%E5%8C%85%E7%9A%84%E5%AE%89%E8%A3%85%E6%83%85%E5%86%B5">
     1 查询包的安装情况
    </h4>
    <p>
     查询看当前环境中安装了哪些包
    </p>
    <pre><code class="language-bash">conda list</code></pre>
    <p>
     查询当前Anaconda repository中是否有你想要安装的包
    </p>
    <pre><code class="language-bash">conda search package_name</code></pre>
    <h4 id="4.2%20%E6%9F%A5%E8%AF%A2%E6%98%AF%E5%90%A6%E6%9C%89%E5%AE%89%E8%A3%85%E6%9F%90%E4%B8%AA%E5%8C%85">
     2 查询是否有安装某个包
    </h4>
    <p>
     用conda list后跟package名来查找某个指定的包是否已安装，而且支持*通配模糊查找。
    </p>
    <pre><code class="language-bash">conda list pkgname        

conda list pkgname*  </code></pre>
    <h4 id="4.2%C2%A0%E5%8C%85%E7%9A%84%E5%AE%89%E8%A3%85%E5%92%8C%E6%9B%B4%E6%96%B0">
     3 包的安装和更新
    </h4>
    <p>
     在当前（虚拟）环境中安装一个包：
    </p>
    <pre><code class="language-bash">conda install package_name

conda install numpy=0.20.3</code></pre>
    <h4 id="4.3%C2%A0conda%E5%8D%B8%E8%BD%BD%E5%8C%85">
     4 conda卸载包
    </h4>
    <pre><code class="language-bash">conda uninstall package_name

conda uninstall package_name --force</code></pre>
    <h4 id="4.4%C2%A0%E6%B8%85%E7%90%86anaconda%E7%BC%93%E5%AD%98">
     5 清理anaconda缓存
    </h4>
    <pre><code class="language-bash">conda clean -p      # 删除没有用的包 --packages
conda clean -t      # 删除tar打包 --tarballs
conda clean -y -all # 删除所有的安装包及cache(索引缓存、锁定文件、未使用过的包和tar包)</code></pre>
    <h3 id="5.%20Python%E7%89%88%E6%9C%AC%E7%9A%84%E7%AE%A1%E7%90%86">
     5. Python版本的管理
    </h3>
    <h4 id="5.1%20%E5%B0%86%E7%89%88%E6%9C%AC%E5%8F%98%E6%9B%B4%E5%88%B0%E6%8C%87%E5%AE%9A%E7%89%88%E6%9C%AC">
     5.1 将版本变更到指定版本
    </h4>
    <pre><code class="language-bash">conda install python=3.5</code></pre>
    <h4 id="5.2%20%E5%B0%86python%E7%89%88%E6%9C%AC%E6%9B%B4%E6%96%B0%E5%88%B0%E6%9C%80%E6%96%B0%E7%89%88%E6%9C%AC">
     5.2 将python版本更新到最新版本
    </h4>
    <pre><code class="language-bash">conda update python</code></pre>
