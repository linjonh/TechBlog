    <h3>
     <a id="_0">
     </a>
     登录多个私有仓库的方法
    </h3>
    <ol>
     <li>
      使用以下命令登录第一个私有仓库：
      <pre><code class="prism language-bash"><span class="token function">docker</span> login registry1.example.com <span class="token parameter variable">--username</span> your-username
</code></pre>
     </li>
     <li>
      再使用以下命令登录第二个私有仓库：
      <pre><code class="prism language-bash"><span class="token function">docker</span> login registry2.example.com <span class="token parameter variable">--username</span> your-username
</code></pre>
     </li>
    </ol>
    <h3>
     <a id="_10">
     </a>
     查看已登录的仓库
    </h3>
    <p>
     执行以下命令可以查看当前已登录的仓库：
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> ~/.docker/config.json
</code></pre>
    <h4>
     <a id="configjson__16">
     </a>
     <code>
      config.json
     </code>
     模样
    </h4>
    <pre><code class="prism language-json"><span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"auths"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-property property">"registry1.example.com"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
      <span class="token string-property property">"auth"</span><span class="token operator">:</span> <span class="token string">"dXNlcm5hbWU6cGFzc3dvcmQ="</span>
    <span class="token punctuation">}</span><span class="token punctuation">,</span>
    <span class="token string-property property">"registry2.example.com"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
      <span class="token string-property property">"auth"</span><span class="token operator">:</span> <span class="token string">"dXNlcm5hbWU6YW5vdGhlcnBhc3N3b3Jk"</span>
    <span class="token punctuation">}</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_30">
     </a>
     推送镜像到指定仓库
    </h3>
    <ul>
     <li>
      推送到
      <code>
       registry1
      </code>
      ：
      <pre><code class="prism language-bash"><span class="token function">docker</span> tag my-image registry1.example.com/my-image
<span class="token function">docker</span> push registry1.example.com/my-image
</code></pre>
     </li>
     <li>
      推送到
      <code>
       registry2
      </code>
      ：
      <pre><code class="prism language-bash"><span class="token function">docker</span> tag my-image registry2.example.com/my-image
<span class="token function">docker</span> push registry2.example.com/my-image
</code></pre>
     </li>
    </ul>
    <h3>
     <a id="_42">
     </a>
     注意事项
    </h3>
    <ul>
     <li>
      <code>
       docker login
      </code>
      仅针对指定的仓库域名，不会覆盖其他已登录的仓库。
     </li>
     <li>
      若某个仓库登录凭据已过期，可能需要重新执行
      <code>
       docker login
      </code>
      。
     </li>
     <li>
      使用
      <code>
       docker logout &lt;registry_url&gt;
      </code>
      可以登出指定仓库。
     </li>
    </ul>
