   <div class="htmledit_views" id="content_views">
    <p>
     要查看和管理Conda虚拟环境，可以通过以下命令和步骤实现：
    </p>
    <h4>
     一、查看所有虚拟环境列表
    </h4>
    <ol>
     <li>
      <p>
       ​
       <strong>
        基础查看命令
       </strong>
       ​
       <br/>
       使用
       <code>
        conda env list
       </code>
       或
       <code>
        conda info --envs
       </code>
       ，这两个命令会列出所有已创建的虚拟环境，包括环境名称、路径和当前活跃环境（标记为
       <code>
        *
       </code>
       ）
      </p>
      <pre><code>conda env list
# 或
conda info --envs</code></pre>
     </li>
     <li>
      <p>
       ​
       <strong>
        查看环境详细信息
       </strong>
       ​
       <br/>
       若需查看某个环境的元数据（如Python版本、依赖包等），可先激活该环境，再运行
       <code>
        conda list
       </code>
       查看所有已安装的包及其版本
      </p>
      <pre><code>conda activate &lt;env_name&gt;
conda list</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     二、虚拟环境的存储位置
    </h4>
    <ol>
     <li>
      <p>
       ​
       <strong>
        默认路径
       </strong>
       ​
       <br/>
       Conda虚拟环境默认存储在以下位置：
      </p>
      <ul>
       <li>
        Linux/macOS:
        <code>
         ~/anaconda3/envs/
        </code>
        或用户目录下的
        <code>
         .conda/envs
        </code>
        ​
       </li>
      </ul>
     </li>
     <li>
      <p>
       ​
       <strong>
        删除环境
       </strong>
       ​
       <br/>
       使用
       <code>
        conda env remove -n &lt;env_name&gt;
       </code>
       或
       <code>
        conda remove --name &lt;env_name&gt; --all
       </code>
      </p>
     </li>
     <li>
      <p>
       ​
       <strong>
        导出与迁移环境
       </strong>
       ​
      </p>
      <ul>
       <li>
        导出配置：
        <code>
         conda env export --name &lt;env_name&gt; &gt; environment.yml
        </code>
       </li>
       <li>
        重新创建：
        <code>
         conda env create -f environment.yml -p &lt;new_path&gt;
        </code>
       </li>
      </ul>
     </li>
    </ol>
