    <h3 id="ionic-在mac上的环境搭建以及在ios模拟器上测试">
     ionic 在mac上的环境搭建以及在iOS模拟器上测试
    </h3>
    <p>
     <strong>
      1.下载并安装nodejs ,这个不用多说，直接装就行了
     </strong>
    </p>
    <p>
     <strong>
      2. npm install -g cordova ionic //(安装 ionic以及cordova, 不成功就用 sudo npm install -g cordova ionic ；sudo 用于提升权限）时间可能较长，请耐心等待
     </strong>
    </p>
    <p>
     <strong>
      3 直接 命令行：ionic -v 或是 cordova -v 显示版本即为成功，不成功的话就重复步骤（2）
     </strong>
    </p>
    <p>
     <strong>
      4 安装模拟器 :
     </strong>
    </p>
    <pre class="prettyprint"><code class="hljs lasso">     npm install <span class="hljs-attribute">-g</span>  ios<span class="hljs-attribute">-sim</span>       <span class="hljs-comment">//代表模拟器运行环境,</span>
     npm install <span class="hljs-attribute">-g</span>  ios<span class="hljs-attribute">-deploy</span>    <span class="hljs-comment">//代表真机运行环境。</span>

     若不成功可能是权限问题，之后遇到问题先加上sudo试试。
     sudo npm install <span class="hljs-attribute">-g</span>  ios<span class="hljs-attribute">-sim</span>    <span class="hljs-comment">//代表模拟器运行环境,</span>
     sudo npm install <span class="hljs-attribute">-g</span>  ios<span class="hljs-attribute">-deploy</span>  <span class="hljs-comment">//代表真机运行环境。</span></code></pre>
    <p>
     <strong>
      5.创建项目
     </strong>
    </p>
    <pre class="prettyprint"><code class="hljs sql"> ionic <span class="hljs-operator"><span class="hljs-keyword">start</span> myApp         (默认为tab )
 ionic <span class="hljs-keyword">start</span> myApp blank   (空项目)
 ionic <span class="hljs-keyword">start</span> myApp tabs    （有tab栏的项目，类似与手机qq主界面）
 ionic <span class="hljs-keyword">start</span> myApp sidemenu（有侧边栏的项目，类似与手机qq侧边栏）</span></code></pre>
    <p>
     <strong>
      6 生成应用
     </strong>
    </p>
    <pre class="prettyprint"><code class="hljs livecodeserver">
iOS：

ionic <span class="hljs-built_in">platform</span> <span class="hljs-built_in">add</span> ios (可能会需要sudo)
ionic build ios  （注意：此处生成的ios并不是最终的应用，是中间品，需要用Xcode继续编译才能生成ipa文件）

附  
    安卓：
    ionic <span class="hljs-built_in">platform</span> <span class="hljs-built_in">add</span> android (可能会需要sudo)
    ionic build android

</code></pre>
    <p>
     <strong>
      7 在ios模拟器运行
     </strong>
    </p>
    <p>
     ionic emulate –list //检查支持的设备，会有一个列表(注意是两个-)
     <br/>
     ionic emulate ios //模拟器运行，我用的版本默认为iPhone-SE ,10.0.0
     <br/>
     ionic emulate ios –target=”iPhone-6, 10.0” //以iPhone6的形式在模拟器中运行，注意iPhone-6, 10.0 是上列表中支持的 ，所以使用此命令前，检查模拟器支持的设备（注意target前是两个-）
    </p>
