## 统计内存条数量

```bash
dmidecode -t memory | grep -A5 "Memory Device" | grep "^[[:space:]]*Size:" | grep -v "No Module Installed" | awk '{size=$2; unit=$3} END {print size, unit, "x", NR}'

```

