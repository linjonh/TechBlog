    </p>
    <ul>
     <li style="text-align:justify">
      <strong>
       <strong>
        产品简介
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     3DWeld是一款智能化的焊接工艺设计与仿真软件。在三维交互式环境下，自动识别焊接特征，基于焊接工艺知识库，快速定义焊接工序内容，形成结构化的焊接工艺信息，辅助焊接工艺仿真模拟，并可通过可视化的方式对焊枪姿态进行调整与规划；同时可为焊接机器人系统传输焊接参数，为其它生产系统提供焊接工艺数据信息，从而提高工艺设计与制造的协同效率，减少差错、缩短产品研发周期。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/4340717bd24f447fa7c7c395637941e7.jpeg"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:center">
    </p>
    <ul>
     <li style="text-align:justify">
      <strong>
       <strong>
        业务现状
       </strong>
      </strong>
     </li>
    </ul>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       1.
      </strong>
     </strong>
     <strong>
      <strong>
       焊接工艺设计效率低
      </strong>
     </strong>
     <strong>
      <strong>
       ：
      </strong>
     </strong>
     焊接工艺中的各参数设置复杂，面对复杂产品的设计，传统的二维图纸不便理解，焊接工艺编制尤为困难，效率低且易出错。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       2.
      </strong>
     </strong>
     <strong>
      <strong>
       二维工艺卡片指导性差：
      </strong>
     </strong>
     传统焊接工艺运用二维工艺软件进行编制，对焊接顺序、焊缝表达不清晰易导致出现漏焊、错焊，输出的工艺卡片不直观，对生产操作指导性不强。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       3.
      </strong>
     </strong>
     <strong>
      <strong>
       工艺设计质量参差不齐：
      </strong>
     </strong>
     焊接工艺专业性要求高，依赖个人能力和经验，编制的焊接工艺文件参差不齐，难以实现焊缝状态与焊枪姿态可视化，焊接产品质量也难以标准化。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       4.
      </strong>
     </strong>
     <strong>
      <strong>
       专业人才培养难度高：
      </strong>
     </strong>
     焊接机器人需依据适配的工艺规程及操作程序进行离线编程方能进行操作，这对工艺人员能力提出了更高阶挑战，培养难，招聘更难。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       三、
      </strong>
     </strong>
     <strong>
      <strong>
       核心功能
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       1.
      </strong>
     </strong>
     <strong>
      <strong>
       焊缝特征自动识别分析
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     对三维CAD模型数据进行分析，自动识别工件形状，板材、型材、管材，自动识别焊缝、坡口特征，分析工件接头形式。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/b9c51d2daed64b59a1f6921c0a6d2bf8.jpeg"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       2.
      </strong>
     </strong>
     <strong>
      <strong>
       焊接工艺参数推理
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     采用交互方式，多策略编排，快速定义工序的焊接内容，也支持用户自定义工序内容，模型拼装可支持选择新增件和继承件，适用场景广。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/1573c8a219df48f2a368ccfd5f495dd7.jpeg"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="text-align:justify">
     <strong>
      <strong>
       3、焊接工艺参数自动推理
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     结合焊缝特征识别结果以及读取的零件外形尺寸、剖口尺寸、材质、标准名称等数据，运用正向推理技术，逐级求解，快速生成焊接工艺参数。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       4.
      </strong>
     </strong>
     <strong>
      <strong>
       焊接顺序规划
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在焊缝编排的基础上，通过控制线设计对焊枪姿态进行优化。通过引入焊枪三维模型，在三维可视化环境下，可直观的对焊枪姿态进行编辑与调整。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/286141c5180e489396de8576aa5eb214.jpeg"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:center">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       5.
      </strong>
     </strong>
     <strong>
      <strong>
       离线编程与工艺仿真
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     将焊接顺序、焊缝轨迹、焊枪姿态等参数输出为xml仿真文件，与离线编程软件集成，自动生成程序；与仿真软件集成，实现虚拟仿真与干涉、碰撞检查
     <strong>
      <strong>
       。
      </strong>
     </strong>
     <strong>
     </strong>
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/44b02e7953fe49c8bb3ff005de7a2d61.jpeg"/>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="text-align:justify">
     <strong>
      <strong>
       6、焊接工艺文件及发布
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     支持多种工艺文件，如3DPDF焊接工艺卡片、焊接工艺仿真数据等，可直观查看焊接工艺动画，及结构化的数据，实现机器人自动化焊接。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       四、
      </strong>
     </strong>
     <strong>
      <strong>
       应用价值
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       高效率：
      </strong>
     </strong>
     基于三维模型的焊接工艺设计，通过“一键式”焊缝识别、自动化参数推理、可视化工艺编制，缩短研发周期，焊接工艺设计效率可大幅提升。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       高质量：
      </strong>
     </strong>
     通过自动推理技术，输出优化的焊接工艺参数，实现焊接工艺知识利用。结合焊接机器人离线编 程及仿真技术，验证工艺可行性，快速提升焊接质量。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       知识积累：
      </strong>
     </strong>
     工艺知识的持续积累和有效复用，使得企业整体工艺水平得到稳定和提高；同时，可增强企业焊接工艺创新能力，提高市场竞争力。
    </p>
