# Linux常用命令——查看python安装路径，与默认路径

查看哪些路径安装了python用whereis命令，当然whereis也可以查看某个文件夹所在的路径

```
whereis python
```

![](https://i-blog.csdnimg.cn/blog_migrate/f27c14764f7b27fba18cd7ba043bc2b3.png)

查看默认调用的python是哪个，which 用来查看当前要执行的命令所在的路径

```
which python
```
