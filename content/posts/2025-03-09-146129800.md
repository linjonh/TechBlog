     <a id="_11">

     题目

    <blockquote>
     <p>
      N皇后问题
     </p>
     <p>
      <strong>
       题目描述
      </strong>
      *
     </p>
     <p>
      在 N×N的方格棋盘放置了 N 个皇后，使得它们不相互攻击（即任意 2个皇后不允许处在同一排，同一列，也不允许处在与棋盘边框成 45 角的斜线上。你的任务是，对于给定的 N，求出有多少种合法的放置方法。

      </strong>

      </strong>
     </p>

      </p>

</code></pre>

      </p>

</code></pre>

      </strong>
     </p>
     <ul>

      </li>

    <h3>
     <a id="_48">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using namespace std;
using ll=long long;
const int N=11;
int vis[N][N];
int n,ans=0;
void dfs(int dep){
  if(dep==n+1){
    ans++;
    return;
  }

  for(int i=1;i&lt;=n;i++){
    if(vis[dep][i])continue;

    for(int _i=1;_i&lt;=n;_i++)vis[_i][i]++;
    for(int _i=dep,_j=i;_i&gt;=1&amp;&amp;_j&gt;=1;_i--,_j--)vis[_i][_j]++;
    for(int _i=dep,_j=i;_i&lt;=n&amp;&amp;_j&gt;=1;_i++,_j--)vis[_i][_j]++;
    for(int _i=dep,_j=i;_i&gt;=1&amp;&amp;_j&lt;=n;--_i,++_j)vis[_i][_j]++;
    for(int _i=dep,_j=i;_i&lt;=n&amp;&amp;_j&lt;=n;++_i,++_j)vis[_i][_j]++;

    dfs(dep+1);
    for(int _i=1;_i&lt;=n;_i++)vis[_i][i]--;
    for(int _i=dep,_j=i;_i&gt;=1&amp;&amp;_j&gt;=1;_i--,_j--)vis[_i][_j]--;
    for(int _i=dep,_j=i;_i&lt;=n&amp;&amp;_j&gt;=1;_i++,_j--)vis[_i][_j]--;
    for(int _i=dep,_j=i;_i&gt;=1&amp;&amp;_j&lt;=n;--_i,++_j)vis[_i][_j]--;
    for(int _i=dep,_j=i;_i&lt;=n&amp;&amp;_j&lt;=n;++_i,++_j)vis[_i][_j]--;
  }
}
int main()
{
  cin&gt;&gt;n;
  dfs(1);
  cout&lt;&lt;ans;
  return 0;
}
</code></pre>
    <h3>
     <a id="_89">
     </a>
     题解分析
    </h3>
    <p>
     本题为经典的DFS题，使用回溯法可以解决。
    </p>
    <ul>
     <li>
      首先可以肯定的是，
      <mark>
       每一行必然有且仅有一个皇后
      </mark>
      （因为不可能出现两个皇后在同一行），于是就通过枚举每一层皇后的位置来搜索所有可能解即可。
     </li>
     <li>
      每放置一个皇后就将对应的米字型区域设置为“禁区”，后面的皇后就不能放在“禁区”，
      <mark>
       回溯的时候将禁区取消掉
      </mark>
      。同时，为了正确维护“禁区”，不能使用
      <code>
       bool
      </code>
      数组来表示禁区，需要使用
      <code>
       int
      </code>
      数组，表示这个位置被“
      <mark>
       多少个皇后占用了
      </mark>
      ”，当占用数为
      <code>
       0
      </code>
      时表示“禁区解除”。
     </li>
     <li>
      层数到
      <code>
       n+1
      </code>
      时表示找到了一个解，不可行的解都到不了第
      <code>
       n+1
      </code>
      行
     </li>
    </ul>

    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using namespace std;
int gcd(int a,int b){
  return b==0?a:gcd(b,a%b);

  int t;cin&gt;&gt;t;

    <h3>
     <a id="_160">
     </a>
     题解分析
    </h3>
    <p>
     本题考察最大公约数。
    </p>
    <ul>
     <li>
      通过
      <mark>
       辗转相除法
      </mark>
      和
      <mark>
       三目运算符
      </mark>
      求最大公约数。
      <code>
       b==0?a:gcd(b,a%b);
      </code>
     </li>
     <li>
      我们知道两数相乘等于两数最大公约数和最小公倍数，所以可以通过最大公约数求最小公倍数。
      <code>
       return a/gcd(a,b)*b;
      </code>
      这里先除再乘是为了减少溢出的发生。
     </li>
    </ul>

    <h3>
     <a id="_173">
     </a>
     题目
    </h3>
    <blockquote>
     <p>
      <strong>
       问题描述
      </strong>
      <br/>
      给定一个正整数 n，请你计算 1∼n 中有多少对不同的素数，满足它们的差也是素数。
     </p>
     <p>
      <strong>
       输入格式
      </strong>
      <br/>
      共一行，包含一个正整数 n(2≤n≤10
      <sup>
       5
      </sup>
      )。
     </p>
     <p>
      <strong>
       输出格式
      </strong>
      <br/>
      共一行，包含一个正整数，表示答案。
     </p>
     <p>
      样例输入
      <br/>
      5
      <br/>
      样例输出
      <br/>
      2
      <br/>
      样例输入
      <br/>
      20
      <br/>
      样例输出
      <br/>
      8
     </p>
    </blockquote>
    <h3>
     <a id="_193">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using ll=long long;
const int N=1e5+10;
using namespace std;
vector&lt;int&gt;primes;
bool vis[N];

void init(int n){
  vis[0]=vis[1]=true;
  for(ll i=2;i&lt;=n;i++){
    if(!vis[i]){
      primes.push_back(i);
      for(int j=2*i;j&lt;=n;j+=i)vis[j]=true;
      }
  }
}
int main()
{
  int n,ans=0;cin&gt;&gt;n;
  init(n);

  for(int i=0;i&lt;primes.size();i++){
    for(int j=0;j&lt;i;j++){
      if(!vis[primes[i]-primes[j]])ans++;
    }
  }
  cout&lt;&lt;ans&lt;&lt;endl;
  return 0;
}
</code></pre>
    <h3>
     <a id="_227">
     </a>
     题解分析
    </h3>
    <p>
     本题使用埃式筛法求素数。
    </p>
    <ul>
     <li>
      <p>
       埃式筛法，即埃拉托斯特尼筛法，
       <mark>
        是一种用于求一定范围内所有素数的高效算法
       </mark>
       。其原理是：从
       <code>
        2
       </code>
       开始，将每个素数的倍数都标记为合数并筛去，剩下的未被筛去的数就是素数。
      </p>
     </li>
     <li>
      <p>
       例如求
       <code>
        100
       </code>
       以内的素数，先将
       <code>
        2
       </code>
       标记为素数，筛去其倍数
       <code>
        4、6、8
       </code>
       等；接着
       <code>
        3
       </code>
       未被筛去，标记为素数，筛去
       <code>
        6、9、12
       </code>
       等；依此进行。该算法简单直观，时间复杂度为
       <code>
        O(n\log\log n)
       </code>
       ，相比
       <mark>
        暴力枚举效率更高
       </mark>
       ，能快速筛选出大量素数，在数论和密码学等领域应用广泛。
      </p>
     </li>
     <li>
      <p>
       通过
       <code>
        bool
       </code>
       类型的数组来记录当前值是否为素数，
       <code>
        false
       </code>
       表示为素数，
       <code>
        true
       </code>
       表示不为素数。
      </p>
     </li>
    </ul>

    <h3>

     </p>
     <p>

     </p>

     </p>

      第二行，包括 n个整数，是数字序列中具体的每个数字。
     </p>

      <strong>

      </strong>
     </p>
     <p>

     </p>
     <p>
      <strong>

      </strong>
     </p>

3 3 3 1 13 15 
</code></pre>
     <p>
      <strong>
       样例输出
      </strong>
     </p>
     <pre><code class="prism language-text">2
</code></pre>
     <p>
      <strong>
       评测数据规模
      </strong>
     </p>
     <p>
      对于所有评测数据，1≤n≤10
      <sup>
       5
      </sup>
      ,ai≤10
      <sup>
       9
      </sup>
      。
     </p>
    </blockquote>
    <h3>
     <a id="_278">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using namespace std;
using ll=long long;
const int N=1e5+10;
int main()
{
   int n,x;cin&gt;&gt;n;
   map&lt;int ,int&gt;mp;
   while(n--){
     cin&gt;&gt;x;
     mp[x]++;
   }
   int ans=0;
   for(auto pair:mp){
     int key=pair.first;
     int value=pair.second;
     if(value&gt;=key)ans+=value-key;
     else ans+=value;
   }
  cout&lt;&lt;ans;
  return 0;
}
</code></pre>
    <h3>
     <a id="_305">
     </a>
     题解分析
    </h3>
    <p>
     本题使用
     <code>
      map
     </code>
     容器解题。
    </p>
    <ul>
     <li>
      <p>
       代码通过读取整数数量
       <code>
        n
       </code>
       及
       <code>
        n
       </code>
       个整数，用
       <code>
        map
       </code>
       统计每个整数出现次数。之后遍历
       <code>
        map
       </code>
       中的键值对，键为整数，值为该整数出现次数。根据值与键的大小关系计算累加结果：
       <mark>
        若值大于等于键，累加差值
       </mark>
       ；
       <mark>
        若值小于键，累加值本身
       </mark>
       。最终输出累加结果。整体时间复杂度为
       <code>
        O(nlogn)
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       <code>
        pair.first
       </code>
       是
       <code>
        pair
       </code>
       的键，
       <code>
        pair.second
       </code>
       是
       <code>
        pair
       </code>
       的值。
      </p>
     </li>
     <li>
      <p>
       <code>
        mp[x]++
       </code>
       表示
       <code>
        mp
       </code>
       中
       <code>
        x
       </code>
       对应的值加1。
      </p>
     </li>
    </ul>

     本题核心思路是枚举所有可能的目标值（从1到60），并对每个目标值模拟操作过程，计算所需的操作次数，最终取最小值。
    </p>
    <ul>
     <li>
      <mark>
       枚举目标值
      </mark>
      ：遍历所有可能的目标值
      <code>
       i
      </code>
      （1到60），假设最终数组的所有元素都变为
      <code>
       i
      </code>
      。
     </li>
     <li>
      <mark>
       模拟操作
      </mark>
      ：对于每个目标值
      <code>
       i
      </code>
      ，复制原数组到临时数组
      <code>
       b
      </code>
      ，然后遍历
      <code>
       b
      </code>
      数组。当遇到不等于i的元素时，执行一次操作，将从该位置开始的连续
      <code>
       k
      </code>
      个元素变为
      <code>
       i
      </code>
      ，并增加操作次数。同时，跳过接下来的
      <code>
       k-1
      </code>
      个元素，避免重复操作。
     </li>
     <li>
      <mark>
       更新最小值
      </mark>
      ：记录每个目标值
      <code>
       i
      </code>
      所需的操作次数，并更新全局最小值。
     </li>
    </ul>

ll qmi(ll a,ll b)
{
  ll res=1;
  while(b)
  {
    if(b&amp;1)res=res*a%p;
    a=a*a%p,b&gt;&gt;=1;
  }
  return res;
}
ll inv(ll x)
{return qmi(x,p-2);}
int main()
{
  ll n,k,x;
  cin&gt;&gt;n&gt;&gt;k&gt;&gt;x;
  for(ll i=1;i&lt;=n;i++)cin&gt;&gt;a[i];
  ll hh=1,tt=0;
  for(ll i=1;i&lt;=n;i++)
  {

    while(hh&lt;=tt&amp;&amp;a[q[tt]]&gt;a[i])tt--;

    while(hh&lt;=tt&amp;&amp;a[q[tt]]&lt;a[i])tt--;
    q[++tt]=i;
   ma[i]=a[q[hh]];
  }
  int ans=0;
  for(int i=k;i&lt;=n;i++)if(ma[i]-mi[i]&lt;=x)ans++;
