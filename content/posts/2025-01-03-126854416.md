    <p>
     1 变量
     <br/>
     原生小程序的变量都是{
     <!-- -->
     {}}包裹的内容，包括属性变量。
     <br/>
     Uniapp的属性变量是:class
     <br/>
     2 点击事件
     <br/>
     原生小程序的点击事件是bindtap。
     <br/>
     Uniapp的点击事件是@tap.
     <br/>
     3 input的value值绑定与监听
     <br/>
     原生小程序绑定是value, 监听是bindinput。
     <br/>
     Uniapp是v-model。
     <br/>
     4 更新视图的方法
     <br/>
     原生小程序是this.setData({})。
     <br/>
     Uniapp是this.a = xxx
     <br/>
     5 方法的传参方式
     <br/>
     原生小程序的方法和参数分开写，参数放在data-属性名。
     <br/>
     Uniapp是方法参数列表中。
    </p>
