    <p>
     在不同的操作系统上，查看自己电脑的IP地址的方法可能会有所不同。以下是在常见的操作系统中查看IP地址的方法：
    </p>
    <h2 id="Windows%EF%BC%9A">
     Windows
    </h2>
    <hr/>
    <p>
     1. 使用命令提示符 (Command Prompt)：
     <br/>
     - 打开命令提示符（按下`Win + R`，输入 `cmd`，然后按回车）。
     <br/>
     - 在命令提示符中，输入 `ipconfig` 并按回车。
     <br/>
     - 查找并记下以太网适配器或无线局域网适配器的 IPv4 地址。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 在任务栏搜索框中键入 "网络和共享中心" 并打开。
     <br/>
     - 在左侧选择你正在使用的网络连接。
     <br/>
     - 在弹出的窗口中，点击 "详细信息"。
     <br/>
     - 查找 IPv4 地址。
    </p>
    <h2 id="macOS%EF%BC%9A">
     macOS
    </h2>
    <hr/>
    <p>
     1. 使用终端：
     <br/>
     - 打开终端（在"应用程序" -&gt; "实用工具" 中找到）。
     <br/>
     - 输入 `ifconfig` 并按回车。
     <br/>
     - 查找并记下你正在使用的网络接口的 IPv4 地址。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 点击苹果图标，选择 "系统偏好设置"。
     <br/>
     - 选择 "网络"。
     <br/>
     - 在左侧选择你正在使用的网络连接。
     <br/>
     - 查找 IPv4 地址。
    </p>
    <h2 id="Linux%EF%BC%9A">
     Linux
    </h2>
    <hr/>
    <p>
     1. **使用终端：
     <br/>
     - 打开终端。
     <br/>
     - 输入 `ifconfig` 或 `ip addr` 并按回车。
     <br/>
     - 查找并记下你正在使用的网络接口的 IPv4 地址。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 不同的 Linux 发行版可能有不同的网络管理工具，但通常可以在系统设置或网络设置中找到。
    </p>
    <h2 id="%E6%89%8B%E6%9C%BA%EF%BC%9A">
     手机
    </h2>
    <hr/>
    <p>
     <strong>
      IOS
     </strong>
     <br/>
     - 进入 "设置"。
     <br/>
     - 选择 "Wi-Fi"。
     <br/>
     - 点击连接的 Wi-Fi 网络，查找 IP 地址。
    </p>
    <p>
     <strong>
      Android
     </strong>
     <br/>
     - 进入 "设置"。
     <br/>
     - 选择 "网络和互联网" 或类似的选项。
     <br/>
     - 点击 "Wi-Fi"。
     <br/>
     - 长按连接的 Wi-Fi 网络，选择 "修改网络" 或 "网络详情"。
     <br/>
     - 查找 IP 地址。
    </p>
    <p>
     这些方法应该能够帮助你在不同的操作系统中找到你电脑或设备的 IP 地址。
    </p>
    <hr/>
    <hr/>
    <hr/>
    <p>
     <strong>
      物理地址，也称为MAC地址（Media Access Control地址），是网络接口卡（NIC）的唯一标识符。在不同的操作系统中，查看自己电脑的物理地址的方法可能会有所不同。
     </strong>
    </p>
    <p>
     <strong>
      Windows：
     </strong>
    </p>
    <p>
     1. 使用命令提示符 (Command Prompt)：
     <br/>
     - 打开命令提示符（按下`Win + R`，输入 `cmd`，然后按回车）。
     <br/>
     - 在命令提示符中，输入 `ipconfig /all` 并按回车。
     <br/>
     - 查找并记下以太网适配器或无线局域网适配器的 "物理地址"。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 在任务栏搜索框中键入 "网络和共享中心" 并打开。
     <br/>
     - 在左侧选择你正在使用的网络连接。
     <br/>
     - 点击 "详细信息"。
     <br/>
     - 查找 "物理地址"。
    </p>
    <p>
     <strong>
      macOS：
     </strong>
    </p>
    <p>
     1. 使用终端：
     <br/>
     - 打开终端（在"应用程序" -&gt; "实用工具" 中找到）。
     <br/>
     - 输入 `ifconfig` 并按回车。
     <br/>
     - 查找你正在使用的网络接口的 "ether" 字段，后面的一串数字和字母就是物理地址（MAC地址）。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 点击苹果图标，选择 "关于本机"。
     <br/>
     - 点击 "系统报告"。
     <br/>
     - 在左侧选择 "网络"。
     <br/>
     - 在右侧找到你正在使用的网络接口，并查找 "硬件地址"。
    </p>
    <p>
     <strong>
      Linux：
     </strong>
    </p>
    <p>
     1. 使用终端：
     <br/>
     - 打开终端。
     <br/>
     - 输入 `ifconfig` 或 `ip addr` 并按回车。
     <br/>
     - 查找你正在使用的网络接口的 "ether" 字段，后面的一串数字和字母就是物理地址（MAC地址）。
    </p>
    <p>
     2. 使用图形界面：
     <br/>
     - 不同的 Linux 发行版可能有不同的网络管理工具，但通常可以在系统设置或网络设置中找到。
    </p>
    <p>
     <strong>
      手机：
     </strong>
    </p>
    <p>
     - iOS：
     <br/>
     - 进入 "设置"。
     <br/>
     - 选择 "Wi-Fi"。
     <br/>
     - 点击连接的 Wi-Fi 网络，查找 "物理地址" 或 "MAC地址"。
    </p>
    <p>
     - Android：
     <br/>
     - 进入 "设置"。
     <br/>
     - 选择 "关于手机" 或 "关于设备"。
     <br/>
     - 查找 "状态" 或类似的选项。
     <br/>
     - 查找 "Wi-Fi MAC地址" 或 "物理地址"。
    </p>
    <p>
     <strong>
      这些方法应该能够帮助你在不同的操作系统中找到你电脑或设备的物理地址（MAC地址）。物理地址通常以十六进制表示，由六个字节组成。
     </strong>
    </p>
