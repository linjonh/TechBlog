# 小程序：uniapp解决主包体积过大的问题

已经分包但还是体积过大

1. 运行时勾选“运行时是否压缩代码”进行压缩
     
   ![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/a782178b9aadad029fdb2cb57710b0ce.png)
2. 在
   `manifest.json`
   配置（开启分包优化）

```
"mp-weixin" : {
	"optimization" : {
	    "subPackages" : true
	}
	//....
},

```

3. 在
   `app.json`
   配置（设置组件按需注入）

```
{
  "lazyCodeLoading": "requiredComponents"
}

```
