    <h3>
     一、字符串的访问
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        Objective-C
       </strong>
       : 使用
       <code>
        characterAtIndex:
       </code>
       方法访问字符。
      </p>
     </li>
    </ul>
    <pre><code class="language-objectivec">NSString *str = @"Hello, World!";
unichar character = [str characterAtIndex:0];  // 访问第一个字符 'H'
NSLog(@"%C", character);  // 输出: H</code></pre>
    <p>
     <code>
      NSString
     </code>
     内部存储的是 UTF-16 编码的字符序列，而不是简单的字符数组。由于 Unicode 字符的长度可能不同（例如，某些字符由多个 UTF-16 码点组成），直接通过索引访问字符可能会导致错误。所以只能通过 [str characterAtIndex:0];  访问
    </p>
    <ul>
     <li>
      <p>
       <strong>
        Swift
       </strong>
       : 使用
       <code>
        String.Index
       </code>
       访问字符。
      </p>
     </li>
    </ul>
    <p>
     在 Swift 中，字符串的字符访问需要通过
     <code>
      String.Index
     </code>
     来实现，因为 Swift 的字符串是由 Unicode 标量组成的，字符的长度可能不一致（例如，某些字符由多个 Unicode 标量组成）。因此，直接使用整数索引（如
     <code>
      str[1]
     </code>
     ）是不允许的
    </p>
    <p>
     要访问第二个字符，可以按照以下步骤操作：
    </p>
    <pre><code class="language-javascript">let str = "Hello, World!"

// 获取字符串的起始索引
let startIndex = str.startIndex

// 计算第二个字符的索引
let secondCharIndex = str.index(startIndex, offsetBy: 1)

// 访问第二个字符
let secondChar = str[secondCharIndex]

print(secondChar)  // 输出: e</code></pre>
    <p>
     Swift 的字符串索引是
     <code>
      String.Index
     </code>
     类型，而不是简单的整数。
    </p>
    <p>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        Python
       </strong>
       : 通过索引直接访问字符。
      </p>
     </li>
    </ul>
    <p>
     Python 中的字符串是不可变的序列，可以通过索引直接访问字符。
    </p>
    <pre><code class="language-python">s = "Hello, World!"
char = s[0]  # 访问第一个字符 'H'
print(char)  # 输出: H</code></pre>
    <ul>
    </ul>
    <p>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        Kotlin
       </strong>
       : 通过索引直接访问字符。
      </p>
     </li>
    </ul>
    <p>
     Kotlin 中的字符串是
     <code>
      String
     </code>
     类型，可以通过索引直接访问字符。
    </p>
    <pre><code class="language-Kotlin">val str = "Hello, World!"
val char = str[0]  // 访问第一个字符 'H'
println(char)  // 输出: H</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        JavaScript
       </strong>
       : 通过索引直接访问字符。
      </p>
     </li>
    </ul>
    <p>
     JavaScript 中的字符串可以通过索引直接访问字符。
    </p>
    <pre><code class="language-javascript">let str = "Hello, World!";
let char = str[0];  // 访问第一个字符 'H'
console.log(char);  // 输出: H</code></pre>
    <p>
    </p>
    <h3>
     二、字符串的遍历
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        Objective-C
       </strong>
       : 使用
       <code>
        characterAtIndex:
       </code>
       和
       <code>
        for
       </code>
       循环。
      </p>
     </li>
    </ul>
    <pre><code class="language-objectivec"> NSString *string = @"中国人民解放军";
 for (int i = 0 ; i &lt; string.length; i++) {
     NSLog(@"遍历的结果char==%hu",[string characterAtIndex:i]);
 }
    </code></pre>
    <ul>
     <li>
      <p>
       <strong>
        Swift
       </strong>
       : 使用
       <code>
        String.Index
       </code>
       或转换为字符数组后遍历。
      </p>
     </li>
    </ul>
    <pre><code class="language-Swift">let str = "Hello"
var index = str.startIndex
while index &lt; str.endIndex {
    let char = str[index]
    print(char)  // 逐个输出字符
    index = str.index(index, offsetBy: 1)
}</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        Kotlin
       </strong>
       : 直接使用
       <code>
        for
       </code>
       循环遍历字符串。
      </p>
     </li>
    </ul>
    <pre><code class="language-Kotlin">val str = "Hello"
for (char in str) {
    println(char)  // 逐个输出字符
}</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        JavaScript
       </strong>
       : 使用
       <code>
        for
       </code>
       循环或
       <code>
        for...of
       </code>
       遍历字符串。
      </p>
     </li>
    </ul>
    <pre><code class="language-javascript">let str = "Hello";
for (let char of str) {
    console.log(char);  // 逐个输出字符
}</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        Python
       </strong>
       : 直接使用
       <code>
        for
       </code>
       循环遍历字符串。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">s = "Hello"
for char in s:
    print(char)  # 逐个输出字符</code></pre>
    <p>
    </p>
    <h3 id="3%E5%AD%97%E7%AC%A6%E4%B8%B2%E7%9A%84%E6%8B%BC%E6%8E%A5">
     三、字符串的拼接
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        Objective-C
       </strong>
       : 使用
       <code>
        stringByAppendingString:
       </code>
       或
       <code>
        stringWithFormat:
       </code>
       。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">NSString *str1 = @"Hello";
NSString *str2 = @" World!";
NSString *result = [str1 stringByAppendingString:str2];
NSLog(@"%@", result);  // 输出: Hello World!

NSString *str1 = @"Hello";
NSString *str2 = @"World";
NSString *result = [NSString stringWithFormat:@"%@ %@!", str1, str2];
NSLog(@"%@", result);  // 输出: Hello World!</code></pre>
    <p>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        Python
       </strong>
       : 使用
       <code>
        +
       </code>
       运算符或
       <code>
        join
       </code>
       方法。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">#可以通过 + 连接
#模版语法
str11 = "中国"
str22 = "人民"
str33 = f'{str11}{str22}'
</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        Swift
       </strong>
       : 使用
       <code>
        +
       </code>
       运算符或字符串插值。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">#可以通过 + 连接
#模版语法
str11 = "中国"
str22 = "人民"
str33 = "\(str11)\(str22)"</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        Kotlin
       </strong>
       : 使用
       <code>
        +
       </code>
       运算符或字符串模板。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">#可以通过 + 连接
#模版语法
str11 = "中国"
str22 = "人民"
str33 = "$str11 $str22"</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        JavaScript
       </strong>
       : 使用
       <code>
        +
       </code>
       运算符或模板字符串。
      </p>
     </li>
    </ul>
    <pre><code class="language-python">#可以通过 + 连接
#模版语法
str11 = "中国"
str22 = "人民"
str33 = `${str11} ${str22}`</code></pre>
    <p>
