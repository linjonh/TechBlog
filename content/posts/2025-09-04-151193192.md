# 迈威通信从送快递角度教你分清网络二层和三层

还在为网络里的二层、三层概念头大?其实就像送快递那么简单!今天迈威通信用最接地气的方式给你讲明白～

## **网络传输 = 送快递?没错!**

![](https://i-blog.csdnimg.cn/direct/b5159573f99041fca2827e1bfe27c2f2.png)

### **二层网络：本地送货员**

负责同小区的包裹配送(局域网传输)，就像小区里的快递站(对应设备：交换机)，认的是“门牌号”(MAC地址)，但不知道其他小区怎么走(没法跨局域网送件)。

### **三层网络：跨城快递车**

能把包裹送到外地(跨网络传输)，还会智能选不堵车的路线，好比快递分拣中心(对应设备：路由器)，靠着“省市地址”(IP地址)精准送达。

## **关键指标对比，一眼看穿区别**

![](https://i-blog.csdnimg.cn/direct/a219e0229a3c4cd3a3a2106d22159f3b.png)

|  |  |  |  |
| --- | --- | --- | --- |
| **绰号** | **认路能力** | **工作范围** | **对应设备** |
| 小区送货员（二层） | 看门牌号（MAC） | 同一小区楼栋 | 二层交换机 |
| 跨省快递车（三层） | 看省市地址（IP） | 不同国家城市 | 三层交换机、路由器 |

## **技术本质原来这么简单**

### **二层：像对讲机呼叫**

交换机就靠“设备身份证”(MAC地址)直接喊人。同一栋楼里广播“203的小王，快递到了”，只有小王会回应，其他人自动忽略，精准又高效。

![](https://i-blog.csdnimg.cn/direct/4c37f49cd8944a77a3d3918f201fe52e.png)

### **三层：像查导航送快递**

路由器按着“城市地图”(IP协议)找路线。先看大方向，比如从湖北武汉到广东深圳(跨网络寻址)，再看具体街道，像南山区科技园A栋(子网定位)，一步步精准送达。

![](https://i-blog.csdnimg.cn/direct/ad98de6c1cfe49fdabf7fc10f6b994df.png)

## **实际应用场景看这里**

![](https://i-blog.csdnimg.cn/direct/7cab7f5465fb4f79b2388a721c23a638.png)

|  |  |  |
| --- | --- | --- |
| **场景** | **二层网络行动** | **三层网络行动** |
| 车间PLC控制机械臂 | 交换机0.1秒内传输指令，通过MAC直连，让机械臂同步动作 | 不插手，避免延迟干扰生产 |
| 总部查看分厂摄像头 | 无法跨区域传输 | 路由器将视频流切分，通过 IP 地址穿越城市送达总部 |
| 工程师远程修设备 | 车间交换机识别维修平板 MAC，直连故障设备 | 建立加密 VPN 隧道，让外地专家安全接入车间网络 |
| 智能电表抄表 | 电表通过二层广播电量，由本地网关收集 | 网关经路由器上传数据到云平台，生成报表（IP远程传输） |

|
|  |

## **迈威小Tips：**

三层设备还有这些俗称 —— 能让多个网络通信的设备、跨网段的设备。简单说，二层认 “人脸”(MAC)，三层看 “地图”(IP)，两者配合，网络才能畅通无阻～
