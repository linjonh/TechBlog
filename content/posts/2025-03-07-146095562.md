    <h4>
    </h4>
    <p>
     在
     <strong>
      A5:SQL Mk-2
     </strong>
     中，拖拽
     <code>
      .a5er
     </code>
     文件后，你可以查询某个表的名称以及生成数据表的 SQL 脚本。以下是具体操作方法：
    </p>
    <p>
     <img alt="" height="653" src="https://i-blog.csdnimg.cn/direct/c7d44e36c63a433280338a817e72406b.png" width="1740"/>
    </p>
    <hr/>
    <h3>
     <strong>
      1. 查找某个表名
     </strong>
    </h3>
    <p>
     如果你已经打开
     <code>
      .a5er
     </code>
     文件，并且看到了 ER 图，你可以用以下方法查找特定表：
    </p>
    <p>
     <img alt="" height="914" src="https://i-blog.csdnimg.cn/direct/b171b160dc0d4c75818c17dd893d4a47.png" width="1451"/>
    </p>
    <h4>
     <strong>
      方法 1：在 ER 图中查找
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       拖拽
       <code>
        .a5er
       </code>
       文件
      </strong>
      进入
      <strong>
       A5:SQL Mk-2
      </strong>
      ，等待 ER 图加载完成。
     </li>
     <li>
      在
      <strong>
       ER 图界面
      </strong>
      ，使用  （
      <strong>
       右键+L
      </strong>
      ），输入表名或部分表名进行查找。
     </li>
     <li>
      选中表后，可以查看字段、索引等详细信息。
      <img alt="" height="914" src="https://i-blog.csdnimg.cn/direct/ef5d8da353f14e0c805859a9b2a6622f.png" width="1451">
       <img alt="" height="590" src="https://i-blog.csdnimg.cn/direct/112554950f464451b4706869f4cc0559.png" width="1126">
        <img alt="" height="599" src="https://i-blog.csdnimg.cn/direct/1793925ee46f42a5a1c582eec6ed8cf1.png" width="1064">
         <img alt="" height="263" src="https://i-blog.csdnimg.cn/direct/3b0604ee6d6c46139f9fb1a84566e4ee.png" width="296"/>
        </img>
       </img>
      </img>
     </li>
    </ol>
    <h4>
    </h4>
    <hr/>
    <h3>
     <strong>
      2. 生成数据表的 SQL 脚本
     </strong>
    </h3>
    <p>
     如果你想获取某张表的
     <code>
      CREATE TABLE
     </code>
     SQL 语句，使用以下方法：
    </p>
    <h4>
     <strong>
      方法 1：在 ER 图中导出 SQL
     </strong>
    </h4>
    <ol>
     <li>
      在
      <strong>
       ER 图界面
      </strong>
      ，
      <strong>
       右键表名
      </strong>
      ，选择
      <strong>
       [生成 SQL (Generate SQL)]
      </strong>
      。
     </li>
     <li>
      选择
      <strong>
       [生成 CREATE TABLE 语句]
      </strong>
      （Generate CREATE TABLE）。
     </li>
     <li>
      SQL 语句会出现在 SQL 编辑器中，你可以复制或执行。
      <img alt="" height="263" src="https://i-blog.csdnimg.cn/direct/3b0604ee6d6c46139f9fb1a84566e4ee.png" width="296">
       <img alt="" height="1012" src="https://i-blog.csdnimg.cn/direct/ea0cf45b4e0e455589950315a6de9520.png" width="898">
        <img alt="" height="777" src="https://i-blog.csdnimg.cn/direct/9ac0ac556f074c12bb526ee52df42234.png" width="933"/>
       </img>
      </img>
     </li>
    </ol>
    <h4>
     <strong>
      方法 2：使用数据库对象导出 SQL
     </strong>
    </h4>
    <p>
     如果你连接了数据库：
    </p>
    <ol>
     <li>
      在左侧
      <strong>
       数据库对象列表
      </strong>
      （Database Explorer）中，找到你的表。
     </li>
     <li>
      <strong>
       右键表名
      </strong>
      → 选择
      <strong>
       [DDL (Definition)]
      </strong>
      或
      <strong>
       [Export as SQL]
      </strong>
      。
     </li>
     <li>
      在弹出的窗口中，你可以选择
      <strong>
       生成 SQL 脚本
      </strong>
      并复制。
     </li>
    </ol>
    <h4>
     <strong>
      方法 3：导出整个数据库的 SQL
     </strong>
    </h4>
    <p>
     如果想导出所有表：
    </p>
    <ol>
     <li>
      点击
      <strong>
       [工具 (Tools)]
      </strong>
      →
      <strong>
       [输出 SQL (Output SQL)]
      </strong>
      。
     </li>
     <li>
      选择
      <strong>
       [导出整个数据库结构]
      </strong>
      （Export database structure）。
     </li>
     <li>
      选择要导出的表，并生成 SQL 脚本。
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      示例：A5:SQL Mk-2 生成的
      <code>
       CREATE TABLE
      </code>
      语句
     </strong>
    </h4>
    <pre><code class="language-sql">CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
</code></pre>
    <p>
     你可以用这个 SQL 在数据库中创建相同的表。
    </p>
