     <a id="2_25">

     2解题思路
    </h2>
    <p>
     当然小学奥数知识只是一个前置条件，你有了也未必能想到下面的思路，下面开始下面的解题思路
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/02c89443fe364023b3f3cf312f456c57.png">
      <br/>
      首先我们可以将Ha,Hb,Hc全部拆解质因数（为什么会想到这样，主要还是为了化简这个复杂的式子。）
      <br/>
      这里我们把HaHbHc表示成他们所有的质因数的幂的乘积（这里有的幂是0）
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/00348e5b127747fd8c183de4fa456ba5.png">
       <br/>
       然后们再看旁边的式子
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7514c8ec6d2543138193d23d588840f8.png">
        <br/>
        LCM其实就是我们三个所有的质因数的里面去取那个幂最大的玩，那么既然我们要依次对所有的质因数去带入这个式子，那么我们的式子就变成了
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/15bcccb3a3a74b1183d4441ce7b32ddc.png">
         <br/>
         这样看着就舒服很多，然后我们再进行化简
         <br/>
         这里我们把上面三个字母设成X&gt;=Y&gt;=Z
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/07c43d9b77a44b47b10a7c850617f04f.png">
          <br/>
          那么这个Z其实就是他们的质因数的最小的那个，那他们所有质因数的最小的幂的乘积其实就是最大公约数也就是我们上面说的
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/374bc41aff4d4bd694116e196381d2cf.png"/>
          <br/>
          那么这道题目的难度瞬间少了一沓，就是求三个数的最大公约数的最大值。

     这里我们的代码实现将三个数的最大公约数直接转化成了一个数的倍数有三个（其实就是一个转化问题，问三个数的最大公约数，并给定你范围，那么其实就是要你在这个范围内找到最大的能满足三个倍数的数字），可以说还是很巧妙的，然后我们从小的数字向大的数字输出，刚好满足了题目要求的字典序最小。
