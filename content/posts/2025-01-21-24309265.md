    <p>
     新建一个Excel然后另存为启用宏的Excel工作簿（后缀为.xlsm），如果保存时候提示 “文档检查器”警告，需要在Excel选项的 信任中心 设置，去掉 “文档特性设置”里面的勾。
    </p>
    <p>
     Excel -》开发工具-》Visual Basic  增加以下代码：
    </p>
    <p>
     <br/>
    </p>
    <p>
     Function code128b(Tar As Range)   '128B码:ChrW(204).
     <br/>
     Dim s$, i%, ss$, j%, curR%, checkB%
     <br/>
     curR = Tar.Row
     <br/>
     s = Tar.Value
     <br/>
     checkB = 1  '开始位的码值为104 mod 103 =1
     <br/>
     For i = 1 To Len(s)
     <br/>
     ss = Mid(s, i, 1)
     <br/>
     j = Asc(ss) '不过滤无效字符,比如汉字.
     <br/>
     If j &lt; 135 Then
     <br/>
     j = j - 32
     <br/>
     ElseIf j &gt; 134 Then
     <br/>
     j = j - 100
     <br/>
     End If
     <br/>
     checkB = (checkB + i * j) Mod 103   '计算校验位
     <br/>
     Next
     <br/>
     If checkB &lt; 95 And checkB &gt; 0 Then  '有的资料直接求103的模,解说不充分,因为有的校验位超过127时,系统会"吃"掉它们(连带休止符).
     <br/>
     checkB = checkB + 32
     <br/>
     ElseIf checkB &gt; 94 Then '字体设置时,字模被定义了2个值.观察字体文件时能发现.
     <br/>
     checkB = checkB + 100
     <br/>
     End If
     <br/>
     code128b = ChrW(204) &amp; s &amp; IIf(checkB, ChrW(checkB), Chr(32)) &amp; ChrW(206)
     <br/>
     End Function
     <br/>
