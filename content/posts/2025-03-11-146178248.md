# 树莓派4B使用Ubuntu20.04连接不上热点

### 以下是正常步骤，如果不行，看最后一步

1.第一步创建configuration file，用于配置连接网络的具体情况

```
sudo nano /etc/wpa_supplicant.conf
```

2.打开后输入如下

```
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=wheel
update_config=1
network={
ssid="WIFI名称"        //你的热点名称
scan_ssid=1            //表示通过隐藏网络寻找并链接你的热点
psk="WIFI 密码"        //你的热点密码
key_mgmt=WPA-PSK       //这个设置表示使用 WPA2 或 WPA3 安全协议，通常是家用路由器的默认配置。
}
```

如果你的WiFi名字是

```
Laww
```

密码是

```
Sambes
```

#### 那就是

```
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=wheel
update_config=1
network={
ssid="Laww"
scan_ssid=1
psk="Sambes"
key_mgmt=WPA-PSK
}
```

3.使用这个来连接网络

```
sudo wpa_supplicant -B iwlan0 -c/etc/wpa_supplicant.conf
```

### **4.如果此时发现连接不上，可能是你的网络是不隐藏的（也就是别人打开WLAN可以发现你的网络），那**

#### `scan_ssid=1错了，要改成`

```
scan_ssid=0
```

如果你的 Wi-Fi 路由器配置为隐藏 SSID（即它不广播网络名称），你需要在
`wpa_supplicant.conf`
中添加
`scan_ssid=1`
，让设备能够扫描并连接到这个隐藏网络，此刻你的网络是可见的，那scan\_ssid=0
`=0即可`
