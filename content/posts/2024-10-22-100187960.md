    <h2>
     <a id="Human_Resource_Machine_0">
     </a>
     Human Resource Machine
    </h2>
    <h4>
     <a id="_1">
     </a>
     简介
    </h4>
    <p>
     Human Resource Machine，其中文名为程序员升职记（或人力资源机器），是由Tomorrow Corporation游戏开发商开发的。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6fd2f0975bbbaf41807fc7f7c24682e3.png">
      <br/>
      这款游戏不同于其他大众的游戏，不是所有人都玩得下来，说上手吧挺容易，但是说通关嘛就比较难了。正如其官方译名“程序员升职记”一样，是一款特别适合想学编程的人玩的。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ddb9ce97e26dc2b31f6faa51482d549.png"/>
     </img>
    </p>
    <h3>
     <a id="_7">
     </a>
     游戏元素
    </h3>
    <p>
     这款游戏的元素比较简单，开局选择关卡，每个关卡代表一个编程任务，只有通过了才可以进入下一个关卡
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/818d8d95fe4351c472574bc0b81fc87f.png">
      <br/>
      选择关卡进入后，就会在右边看到这个关卡的任务，然后拖动给定的代码来完成这个任务
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0624d460f863bce4b64bfa41e12be46e.png">
       <br/>
       若你写的程序有BUG，则通不过任务，需要重写，直到没BUG为止
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5e458463b0b5ce77ae1d21dbe17d9036.png">
        <br/>
        当任务顺利完成了，会给出评价，包括代码数量的评价以及效率的评价，若这两项都这其给定的范围内，则可点亮两颗星星，注意这两项并不要求同时点亮，可以分开点亮。其实这也说明了编程序的特点：效率和空间很多时候都不可能同时兼得。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/789737bd386a19b7e643b98d217594e0.png">
         <br/>
         这款游戏大概有50个关卡左右，难度逐渐上升，博主断断续续玩了几次，玩到了入职第16年（本人比较菜）
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b7aae930a0898c43f6d5956732fafa6.png">
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0240037ee6e9cf5d8bdaa0b010ee0417.png"/>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <h3>
     <a id="_20">
     </a>
     社会价值与贡献
    </h3>
    <p>
     这款游戏不仅说可以消磨时间，还可以锻炼一个人的编程思想。而且我觉得这款程序做得最好的一点是，无需掌握任何一门具体的编程语言，只需要拖动它给定的执行语句去完成编程即可，这样也大大提高了可玩性。游戏中有一个核心理念，即把逻辑思维碎片化。不会让玩家直接面对某种“数学问题”，而是用一种更加感性的方式展现游戏。这家公司还有其他几款游戏也相当不错，像《70亿人》，大家有空可以去体验一下，特别是对编程感兴趣的同学。
    </p>
