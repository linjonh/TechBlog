# Ollama|轻松使用大型语言模型（Llama/deepseek）

设备：Mac pro，Apple M2

下载Ollama

[Download Ollama on macOS](https://ollama.com/download "Download Ollama on macOS")

下载完毕，解压并安装，打开终端

安装llama/deepseek

```
ollama run llama3.2
ollama run deepseek-r1
```

安装成功直接使用
