# 数据库逻辑名称和数据库物理名称

个人理解:

数据库的物理名称:就在存在磁盘上的名字.

逻辑名称:数据库的内部名称.

```
改变数据库的物理名称,并不会改变数据库的逻辑名称.  

  

  

逻辑名称可以更改（SQL SERVER 2005）  

ALTER DATABASE 数据库名  

MODIFY NAME  

（  

    NAME = Logic_name，  

    NEWNAME = NEW_Logic_name  

）
  


```
