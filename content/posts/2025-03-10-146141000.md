    <h3>
     <a id="_5">
     </a>
     一、启动安装程序
    </h3>
    <p>
     点击开启此虚拟机
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/04e7e133dd85424d8bda9cffa00f0d93.jpeg">
      <br/>
      进入到系统选择界面，默认选择第一个（Install openEuler 24.03-LTS）即可，此步忘记截图了，截取一个类似的
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bc5c0582570d43759d7b24b31f6fa3eb.png">
       <br/>
       点击后，默认进入系统各种检查阶段，没有报错就稍微等待即可。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8ed9f52b3dd143d4a1b8473b93e06032.jpeg">
        <br/>
        <strong>
         语言与时区设置
        </strong>
        <br/>
        系统初始化完成后会出现以下界面，选择中文然后点击继续即可
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6606b0a7276d4076b9c88b542d6f54cd.png">
         <br/>
         <strong>
          磁盘分区：
         </strong>
         <br/>
         可以根据自己需要调整磁盘分区，分区方案，新手可选择自动配置分区，后续可以详细讲解Linux 的lvm管理。
         <br/>
         <strong>
          网络与主机名
         </strong>
         <br/>
         设置主机名（如openeuler）
         <br/>
         <strong>
          软件选择
         </strong>
         <br/>
         根据自己需要选择安装
         <br/>
         <strong>
          用户配置
         </strong>
         <br/>
         设置启用root账户，并设置复杂密码（需满足复杂度要求）
         <br/>
         可选创建普通用户（建议勾选将此用户设为管理员）
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/22eb372993224fab9b2fc851f1f1dbb4.png">
          <br/>
          都设置完毕以后，点击开始安装
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4bda2eab545644119eeb3d8b6e795f68.png">
           <br/>
           根据上一步选择的安装类型，安装时间将有所不同。
