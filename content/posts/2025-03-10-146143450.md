    <h5>
     什么是CSS？
    </h5>
    <p>
     <strong>
      CSS
     </strong>
     （Cascading Style Sheets，层叠样式表）是用于控制网页外观和布局的代码语言，它与 HTML 配合使用，可以让网页从“纯内容”变成“美观的界面”。CSS是一种样式语言，告诉网页浏览器如何呈现HTML元素。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="448" src="https://i-blog.csdnimg.cn/img_convert/b0923c6d88e4e514a8ac22d3ea48aac4.png" width="760"/>
    </p>
    <h5>
     CSS的类型
    </h5>
    <p>
     有三种主要的方式将CSS添加到你的HTML中，这三种方式分别是：
    </p>
    <ol>
     <li>
      <p>
       内联CSS：直接在HTML标签中
      </p>
     </li>
     <li>
      <p>
       内部CSS：在HTML的
       <code>
        &lt;head&gt;
       </code>
       中
      </p>
     </li>
     <li>
      <p>
       外部CSS：单独的.css文件
      </p>
     </li>
    </ol>
    <p>
     接着我们就来看看这三种CSS的实际应用。
    </p>
    <h5>
     内联&amp;内部CSS
    </h5>
    <p>
     代码示例：
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;

&lt;head&gt;
  &lt;style&gt;
    /* 这是内部CSS */
    body {
      background-color: #f0f0f0;
      font-family: Arial, sans-serif;
    }

    h1 {
      color: #333333;
      text-align: center;
    }
  &lt;/style&gt;
&lt;/head&gt;

&lt;body&gt;
  &lt;h1&gt;欢迎来到CSS！&lt;/h1&gt;
  &lt;p style="color: blue;"&gt;这是一个带有内联CSS的段落。&lt;/p&gt;
&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     在这个例子中，我们同时使用了内部和内联CSS。内部的CSS在
     <code>
      &lt;style&gt;
     </code>
     标签中设置了整个页面的背景颜色和字体，并对
     <code>
      &lt;h1&gt;
     </code>
     元素进行了样式设置。
     <code>
      &lt;p&gt;
     </code>
     标签上的内联CSS使这个特定的段落变为蓝色。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="169" src="https://i-blog.csdnimg.cn/img_convert/207d027690eea32abc4443de219bd492.png" width="1080"/>
    </p>
    <h5>
     外部CSS
    </h5>
    <p>
     现在从进行以下步骤：
    </p>
    <ol>
     <li>
      <p>
       为你的项目创建一个新文件夹。
      </p>
     </li>
     <li>
      <p>
       在该文件夹中，创建一个HTML文件（例如，
       <code>
        index.html
       </code>
       ）和一个CSS文件（例如，
       <code>
        styles.css
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       打开你的HTML文件并添加以下基本结构：
      </p>
     </li>
    </ol>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;link rel="stylesheet" href="style.css"&gt;
&lt;body&gt;
  &lt;h1&gt;欢迎来到CSS！&lt;/h1&gt;
  &lt;p &gt;这是一个带有内联CSS的段落。&lt;/p&gt;
&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     创建并打开你的
     <code>
      .css
     </code>
     文件，随意命名，文章中的文件名为，让我们写下我们的第一个CSS规则：
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;body&gt;
  &lt;h1&gt;欢迎来到CSS！&lt;/h1&gt;
  &lt;p &gt;这是一个带有内联CSS的段落。&lt;/p&gt;
&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     我们在html文件中使用
     <code>
      &lt;link rel="stylesheet" href="style.css"&gt;
     </code>
     链接外部
     <code>
      .css
     </code>
     文件，得到如下效果：
    </p>
    <ul>
     <li>
      <p>
       我们使用浅蓝色背景和字体设置了
       <code>
        &lt;body&gt;
       </code>
       元素的样式。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;h1&gt;
       </code>
       居中，深灰色，并有轻微的文字阴影。保存两个文件并在浏览器中打开的HTML：
      </p>
     </li>
    </ul>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="223" src="https://i-blog.csdnimg.cn/img_convert/472bfcea5480ff86330b1c68a94356be.png" width="1080"/>
    </p>
    <h5>
     CSS属性
    </h5>
    <p>
     CSS有大量的属性可以玩转。以下是一些常见的：
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="264" src="https://i-blog.csdnimg.cn/img_convert/78fda226e4f71497f6dc20928d75e058.png" width="1080"/>
    </p>
    <p>
     这个时候，让我们添加一个段落并对其进行样式设置：
    </p>
    <pre><code class="language-html">&lt;link rel="stylesheet" href="styles.css"&gt;
&lt;p class="intro"&gt;CSS太神奇了！它让我们以无数的方式设置网页样式。&lt;/p&gt;</code></pre>
    <p>
     在你的css文件中例子中的css文件为
     <code>
      styles.css
     </code>
     ：
    </p>
    <pre><code class="language-css">.intro {
font-size: 18px;
color: #4a4a4a;
line-height: 1.6;
margin: 20px;
padding: 15px;
border: 2px dashed #7caed6;
border-radius: 10px;
}</code></pre>
    <p>
     这创建了一个带有虚线边框和圆角的样式段落。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="71" src="https://i-blog.csdnimg.cn/img_convert/eb94606dcaf51789fe39df2f4bc767db.png" width="1080"/>
    </p>
    <h6>
     为什么要麻烦用CSS？我们不是可以直接在HTML中设置样式吗？
    </h6>
    <ol>
     <li>
      <p>
       <strong>
        关注点分离
       </strong>
       ：CSS允许我们将表示（东西看起来如何）与结构（内容及其组织）分开。这使得我们的代码更干净，更容易维护。
      </p>
     </li>
     <li>
      <p>
       <strong>
        一致性
       </strong>
       ：使用CSS，你可以一次定义样式，然后跨多个页面应用。需要改变所有标题的颜色？在CSS文件中更改一次，然后就可以啦！
      </p>
     </li>
     <li>
      <p>
       <strong>
        灵活性
       </strong>
       ：CSS为你提供了对网页布局和外观的精细控制。从简单的颜色更改到复杂的动画，CSS都能做到。
      </p>
     </li>
     <li>
      <p>
       <strong>
        响应性
       </strong>
       ：CSS允许你创建适应不同屏幕尺寸的设计，确保你的网站在从智能手机到大型桌面显示器上都看起来很棒。
      </p>
     </li>
     <li>
      <p>
       <strong>
        带宽效率
       </strong>
       ：通过将样式信息移到单独的CSS文件，你可以减少HTML中的冗余，从而减小文件大小，加快加载时间。
      </p>
     </li>
    </ol>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="58" src="https://i-blog.csdnimg.cn/img_convert/4e9331643676ecd949b9db10e330cc1c.png" width="473"/>
    </p>
    <h5>
     CSS - 语法
    </h5>
    <h6>
     类型选择器
    </h6>
    <p>
     类型选择器是CSS选择器中最简单的形式。它们针对特定HTML标签类型的所有元素。
    </p>
    <pre><code class="language-css">p {
color: blue;
}</code></pre>
    <p>
     在这个例子中，网页上的所有
     <code>
      &lt;p&gt;
     </code>
     （段落）元素都会被染成蓝色。
    </p>
    <h6>
     通用选择器
    </h6>
    <p>
     它针对页面上的每一个元素。它用一个星号（*）表示。
    </p>
    <pre><code class="language-css">* {
margin: 0;
padding: 0;
}</code></pre>
    <p>
     这个片段重置了所有元素的边距和内边距。
    </p>
    <h6>
     后代选择器
    </h6>
    <p>
     后代选择器允许你针对嵌套在其他元素内的元素。
    </p>
    <pre><code class="language-css">p a {
text-decoration: none;
}</code></pre>
    <p>
     这个规则会移除所有位于段落内的链接（
     <code>
      &lt;a&gt;
     </code>
     标签）的下划线。
    </p>
    <h6>
     类选择器
    </h6>
    <p>
     类选择器非常灵活。它们允许你将样式应用于具有特定类属性的元素。
    </p>
    <pre><code class="language-css">.highlight {
background-color: yellow;
}</code></pre>
    <p>
     任何带有
     <code>
      class="highlight"
     </code>
     的元素都将有黄色背景。
    </p>
    <h6>
     ID选择器
    </h6>
    <p>
     ID选择器与类选择器类似，但它们是为唯一元素设计的。每个ID在页面上只能使用一次
    </p>
    <pre><code class="language-css">#header {
font-size: 24px;
font-weight: bold;
}</code></pre>
    <p>
     这个样式将应用于带有
     <code>
      id="header"
     </code>
     的元素，它非常适合页面上独一无二的单个元素。
    </p>
    <h6>
     子选择器
    </h6>
    <p>
     子选择器比后代选择器更具体。它们只针对元素的直接子元素。
    </p>
    <pre><code class="language-css">ul &gt; li {
list-style-type: square;
}</code></pre>
    <p>
     这个规则将项目符号更改为正方形，但仅针对作为
     <code>
      &lt;ul&gt;
     </code>
     元素直接子元素的
     <code>
      &lt;li&gt;
     </code>
     元素。
    </p>
    <h6>
     属性选择器
    </h6>
    <p>
     属性选择器允许你根据元素的属性或属性值来定位元素。
    </p>
    <pre><code class="language-css">input[type="text"] {
border: 1px solid blue;
}</code></pre>
    <p>
     这个规则将蓝色边框应用于所有文本输入字段。
    </p>
    <h6>
     多个样式规则
    </h6>
    <p>
     将多个样式规则应用于同一个选择器。每个规则应该单独一行，以提高可读性。
    </p>
    <pre><code class="language-css">h1 {
color: navy;
font-size: 20px;
text-align: center;
}</code></pre>
    <p>
     在这里我们给
     <code>
      &lt;h1&gt;
     </code>
     元素赋予了海军蓝色，大小为20像素，并居中对齐。
    </p>
    <h6>
     选择器分组
    </h6>
    <p>
     有时，我们可能会希望将相同的样式应用于多个选择器。这个时候可以将选择器分组在一起。
    </p>
    <pre><code class="language-css">h1, h2, h3 {
font-family: Arial, sans-serif;
}</code></pre>
    <p>
     这个规则将Arial字体（或者如果Arial不可用，则使用任何无衬线字体）应用于所有
     <code>
      &lt;h1&gt;
     </code>
     、
     <code>
      &lt;h2&gt;
     </code>
     和
     <code>
      &lt;h3&gt;
     </code>
     元素。
    </p>
    <h6>
     CSS 伪类选择器
    </h6>
    <p>
     伪类选择器目标处于特定状态的元素。
    </p>
    <pre><code class="language-css">a:hover {
  text-decoration: underline;
}

input:focus {
  outline: 2px solid blue;
}

li:nth-child(odd) {
  background-color: #f0f0f0;
}</code></pre>
    <p>
     这些样式会在鼠标悬停时给链接添加下划线，给聚焦的输入框添加蓝色轮廓，以及给每个奇数列表项添加浅灰色背景。
    </p>
    <h6>
     CSS 伪元素选择器
    </h6>
    <p>
     伪元素选择器允许你样式化元素的具体部分。
    </p>
    <pre><code class="language-css">p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

h1::before {
  content: "? ";
}

div::selection {
  background-color: yellow;
}</code></pre>
    <p>
     这段代码放大并加粗了段落的第一个字母，给 h1 元素之前添加了一个指向手指表情符号，并将 div 中选中的文本背景设置为黄色。
    </p>
    <p>
     现在，让我们将这些选择器类型总结在一个方便的表格中：
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="349" src="https://i-blog.csdnimg.cn/img_convert/4e8b342b6525829d33721ef9e34511b1.png" width="1080"/>
    </p>

    <h5>
     CSS - 溢出：掌握内容控制
    </h5>
    <h6>
     什么是CSS溢出？
    </h6>
    <p>
     CSS溢出就像你往一个小盒子里塞了太多东西，盒子装不下了，这时候内容就会“溢出来”。用专业的话说，
     <strong>
      当元素的内容（比如文字、图片）超过了它自己设定的大小（比如固定宽高），多出来的部分就会溢出到盒子外面
     </strong>
     。
    </p>
    <p>
     举个生活中的例子🌰：假设你有一个高10厘米的玻璃杯（相当于网页中的一个div容器），如果倒进去15厘米高的水（相当于内容），多出来的5厘米水就会溢出杯口，这就是“溢出”。
    </p>
    <h6>
     CSS溢出 - 使用可见和隐藏值
    </h6>
    <p>
     CSS溢出属性有两个基本值：
     <code>
      visible
     </code>
     和
     <code>
      hidden
     </code>
     。
    </p>
    <h6>
     visible（默认）
    </h6>
    <p>
     这是默认行为。就像说：“让它都露出来！”
    </p>
    <pre><code class="language-css">&lt;div class="overflow-visible"&gt;
&lt;p&gt;这是一个很长的段落，它将溢出其容器。&lt;/p&gt;
&lt;/div&gt;</code></pre>
    <p>
     <code>
      .css
     </code>
     文件
    </p>
    <pre><code class="language-css">.overflow-visible {
width: 100px;
height: 100px;
border: 1px solid black;
overflow: visible;
}</code></pre>
    <p>
     就像给你的盒子装上轮子，以防你需要移动东西。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="162" src="https://i-blog.csdnimg.cn/img_convert/96dd2a593792b79c4a5f90104e853519.png" width="149"/>
    </p>
    <h4>
     auto
    </h4>
    <p>
     这是一个智能选项。它只在必要时添加滚动条。
    </p>
    <pre><code class="language-css">&lt;div class="overflow-auto"&gt;
&lt;p&gt;这个内容只有溢出时才会出现滚动条。&lt;/p&gt;
&lt;/div&gt;
.overflow-auto {
width: 100px;
height: 100px;
border: 1px solid black;
overflow: auto;
}</code></pre>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="69" src="https://i-blog.csdnimg.cn/img_convert/3d57d1d94ceec32cb6390898a0e94082.png" width="378"/>
    </p>
    <h6>
     CSS溢出 - 相关属性
    </h6>
    <p>
     更多控制的相关属性：
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="图片" height="221" src="https://i-blog.csdnimg.cn/img_convert/5942744be58ef1f95aea71c123697e84.png" width="1080"/>

    <h6>
     CSS溢出 - 相关属性
    </h6>
    <p>
     更多控制的相关属性：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        属性
       </th>
       <th>
        描述
       </th>
       <th>
        示例
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        overflow-x
       </td>
       <td>
        控制水平溢出
       </td>
       <td>
        <code>
         overflow-x: scroll;
        </code>
       </td>
      </tr>
      <tr>
       <td>
        overflow-y
       </td>
       <td>
        控制垂直溢出
       </td>
       <td>

        </code>
       </td>
      </tr>
      <tr>

       </td>

       </td>

        </code>

      </tr>
      <tr>

    <p>
     让我们看看这些属性的实际应用：
    </p>
    <pre><code class="language-css">&lt;div class="overflow-fancy"&gt;
