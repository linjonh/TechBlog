     </a>
     <strong>
      附：常用工作流程示例
     </strong>
    </h4>
    <p>
     <strong>
      制作角色循环跑动动画
     </strong>
    </p>
    <ol>
     <li>
      <code>
       Alt + 拖动视口
      </code>
      调整摄像机追踪角度 →
      <code>
       Shift + S
      </code>
      对齐关键帧到节奏点。
     </li>
     <li>
      在曲线编辑器用
      <code>
       F
      </code>
      平滑运动曲线 →
      <code>
       Ctrl + C/V
      </code>
      复制脚部关键帧到对称骨骼。
     </li>
     <li>
      按
      <code>
       Ctrl + L
      </code>
      设置循环 →
      <code>
       Space
      </code>
      预览并
      <code>
       Alt + 滚轮
      </code>
      调整时间轴精度。
     </li>
     <li>
      按
      <code>
       Ctrl + Alt + S
      </code>
      切换至Sequencer → 导出为动画片段（
      <code>
       右键导出
      </code>
      ）。
     </li>
    </ol>
