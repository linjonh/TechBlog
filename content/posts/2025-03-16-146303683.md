# Unreal Engine 5.4 (虚幻引擎5)中动画制作快捷键大全

**Unreal Engine 5.4**
中动画制作（重点围绕
**Sequencer**
与
**动画编辑器**
）的核心快捷键大全，按功能分类整理：

---

#### **一、Sequencer（动画序列控制）**

| **功能目标** | **快捷键** | **说明** |
| --- | --- | --- |
| 新建Sequencer | `Ctrl + Alt + S` | 快速创建或打开Sequencer时间轴 |
| 添加关键帧 | `S` | 为选中属性添加关键帧（需选中属性） |
| 删除关键帧 | `Delete` | 移除选中关键帧 |
| 播放/暂停动画 | `Space` | 从当前播放头位置开始播放 |
| 逐帧前进/后退 | `→` / `←` | 逐帧调整播放头位置 |
| 跳转到片段起始/结束 | `Home` / `End` | 快速定位到时间轴端点 |
| 缩放时间轴显示范围 | `Alt + 鼠标滚轮` | 水平缩放时间刻度 |
| 多选关键帧 | `Ctrl + 点击` | 按住Ctrl选择多个关键帧或对象 |
| 复制/粘贴关键帧 | `Ctrl + C` / `Ctrl + V` | 支持跨轨道复制关键帧数据 |
| 移动关键帧 | 选中后直接拖拽 | 按住 `Alt` 拖拽可复制并移动 |
| 对齐关键帧到播放头 | `Shift + S` | 将选中关键帧吸附至播放头位置 |

---

#### **二、骨骼与动画蓝图**

| **功能目标** | **快捷键** | **说明** |
| --- | --- | --- |
| 进入骨骼编辑模式 | `L` | 切换至骨骼层次结构视图 |
| 添加骨骼键 | `Insert键` | 在动画蓝图中插入新骨骼节点 |
| 快速预览动画 | `Ctrl + P` | 在动画蓝图中实时测试动画逻辑 |
| 创建混合空间 | `右键菜单 → Blend Space` | 生成动画混合空间资源 |

---

#### **三、视口控制与摄像机**

| **功能目标** | **快捷键** | **说明** |
| --- | --- | --- |
| **WASD移动视角** | `按住右键 + WASD` | 场景视角自由移动（默认飞行模式） |
| **切换透视/正交视图** | `Alt + G` | 循环切换摄像机投影模式 |
| **聚焦选中对象** | `F` | 摄像机视角快速聚焦选中资产或角色 |
| **创建摄像机Cut** | `Ctrl + 拖动视口` | 在Sequencer中直接录制摄像机路径 |
| **重置摄像机位置** | `Backspace` | 恢复默认视角 |
| **垂直升降视角** | `Q` （降） / `E` （升） | 配合WASD调整高度 |

---

#### **四、关键帧曲线与插值**

| **功能目标** | **快捷键** | **说明** |
| --- | --- | --- |
| 打开曲线编辑器 | `双击轨迹或属性名` | 精细化调整动画曲线形态 |
| 切换插值类型 | `右键关键帧 → 类型` | 切换为Constant/Linear/Cubic等插值 |
| 快速设置循环动画 | `Ctrl + L` | 在曲线编辑器中设置循环区间与模式 |
| 自动平滑曲线 | `F` （曲线编辑器中） | 自动平滑选中关键帧的切线 |

---

#### **五、效率操作与工具**

| **功能目标** | **快捷键** | **说明** |
| --- | --- | --- |
| 多选轨道/对象 | `Shift + 点击` | 连续多选轨道或资产 |
| 展开/收起所有轨道 | `Shift + 点击轨道折叠图标` | 批量展开或缩进层级 |
| 设置标记 | `M` | 在时间轴上添加标注（便于对齐事件） |
| 快速保存 | `Ctrl + S` | 即时保存当前工程文件 |
| 撤销/重做 | `Ctrl + Z` / `Ctrl + Y` | 支持无限步骤撤销（根据设置） |
| 显示/隐藏轨道组 | `H` | 快速隐藏非选中轨道以简化视图 |

#### **附：常用工作流程示例**

**制作角色循环跑动动画**

1. `Alt + 拖动视口`
   调整摄像机追踪角度 →
   `Shift + S`
   对齐关键帧到节奏点。
2. 在曲线编辑器用
   `F`
   平滑运动曲线 →
   `Ctrl + C/V`
   复制脚部关键帧到对称骨骼。
3. 按
   `Ctrl + L`
   设置循环 →
   `Space`
   预览并
   `Alt + 滚轮`
   调整时间轴精度。
4. 按
   `Ctrl + Alt + S`
   切换至Sequencer → 导出为动画片段（
   `右键导出`
   ）。

#### **其他设置建议**

* **自定义快捷键**
  ：点击
  `Edit → Editor Preferences → Keyboard Shortcuts`
  根据习惯调整键位。
* **动画通知工具**
  ：按
  `N`
  添加动画事件标记（如脚步声触发点）。
* **快速回退**
  ：若操作失误，按
  `Ctrl + Z`
  可撤销多步历史操作。
