    <p>
     主要是为了梳理一下整个过程，其实步骤很简单，但容易出错。
     <br/>
     注意，实际只有两个文件需要写入，一个是
     <strong>
      jupyter_lab_config.py
     </strong>
     ，一个是
     <strong>
      jupyter.service
     </strong>
     。
     <br/>
     配置文件的名字要写对，如果总是copy网上的代码，可能会出现不同的config，前前后后很混乱。
    </p>
    <h3>
     <a id="1pip3jupyter_lab_3">
     </a>
     1.首先更新pip3安装jupyter lab
    </h3>
    <pre><code class="prism language-bash">pip3 <span class="token function">install</span> <span class="token parameter variable">--upgrade</span> pip  <span class="token comment"># 更新 pip3 到最新版本</span>
pip3 <span class="token function">install</span> jupyter jupyterlab  <span class="token comment"># 安装或更新 Jupyter Lab</span>
</code></pre>
    <h3>
     <a id="2_9">
     </a>
     2.生成配置文件
    </h3>
    <pre><code class="prism language-bash">jupyter lab --generate-config
</code></pre>
    <p>
     会生成一个jupyter_lab_config.py文件，位于.jupyter目录下。
    </p>
    <h3>
     <a id="3_15">
     </a>
     3.配置文件
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">vim</span> /home/lrhan/.jupyter/jupyter_lab_config.py
</code></pre>
    <p>
     写入
    </p>
    <pre><code class="prism language-bash">c.NotebookApp.ip <span class="token operator">=</span> <span class="token string">'0.0.0.0'</span>  <span class="token comment"># 允许远程访问</span>
c.NotebookApp.port <span class="token operator">=</span> <span class="token number">8888</span>  <span class="token comment"># 端口号</span>
c.NotebookApp.open_browser <span class="token operator">=</span> False  <span class="token comment"># 不自动打开浏览器</span>
c.NotebookApp.token <span class="token operator">=</span> <span class="token string">''</span>  <span class="token comment"># 关闭 Token 认证</span>
c.NotebookApp.password <span class="token operator">=</span> <span class="token string">''</span>  <span class="token comment"># 关闭密码认证</span>
c.NotebookApp.allow_remote_access <span class="token operator">=</span> True  <span class="token comment"># 允许远程访问</span>
</code></pre>
    <h3>
     <a id="4_28">
     </a>
     4.设置开机自启动
    </h3>
    <p>
     先查看一下jupyter-lab的安装位置
    </p>
    <pre><code class="prism language-bash"><span class="token function">which</span> jupyter-lab
</code></pre>
    <p>
     创建jupyter.service文件
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">vim</span> /etc/systemd/system/jupyter.service
</code></pre>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>Unit<span class="token punctuation">]</span>
<span class="token assign-left variable">Description</span><span class="token operator">=</span>Jupyter Lab
<span class="token assign-left variable">After</span><span class="token operator">=</span>network.target

<span class="token punctuation">[</span>Service<span class="token punctuation">]</span>
<span class="token assign-left variable">Type</span><span class="token operator">=</span>simple
<span class="token assign-left variable">User</span><span class="token operator">=</span>lrhan
<span class="token assign-left variable">Group</span><span class="token operator">=</span>lrhan
<span class="token assign-left variable">WorkingDirectory</span><span class="token operator">=</span>/home/lrhan
<span class="token assign-left variable">ExecStart</span><span class="token operator">=</span>/home/lrhan/.local/bin/jupyter-lab <span class="token parameter variable">--ip</span><span class="token operator">=</span><span class="token number">0.0</span>.0.0 <span class="token parameter variable">--port</span><span class="token operator">=</span><span class="token number">8888</span> --no-browser
<span class="token assign-left variable">Restart</span><span class="token operator">=</span>always
<span class="token assign-left variable">Environment</span><span class="token operator">=</span><span class="token string">"PATH=/home/lrhan/.local/bin:/usr/bin:/bin"</span>

<span class="token punctuation">[</span>Install<span class="token punctuation">]</span>
<span class="token assign-left variable">WantedBy</span><span class="token operator">=</span>multi-user.target
</code></pre>
    <h3>
     <a id="_57">
     </a>
     注意
    </h3>
    <p>
     如果中间多次配置，请注意杀死进程，重启jupyter！！！
     <br/>
     如果配置文件设置正确，并且你想要确保新的配置生效，执行以下命令来重启 Jupyter 服务：
    </p>
    <pre><code class="prism language-bash"><span class="token function">pkill</span> <span class="token parameter variable">-9</span> <span class="token parameter variable">-f</span> jupyter  <span class="token comment"># 杀死所有 Jupyter 进程</span>
<span class="token function">sudo</span> systemctl restart jupyter  <span class="token comment"># 重新启动 Jupyter</span>
</code></pre>
    <p>
     然后可以使用下面的命令检查 Jupyter 是否在运行，并且没有 Token：
    </p>
    <pre><code class="prism language-bash">jupyter lab list
</code></pre>
    <blockquote>
     <p>
      http://lrhan-desktop:8888/lab
     </p>
    </blockquote>
    <p>
     现在你可以通过浏览器直接访问：
    </p>
    <pre><code class="prism language-bash">http://<span class="token operator">&lt;</span>Jetson_IP<span class="token operator">&gt;</span>:8888/lab
</code></pre>
