    <h2>
     基于TCP通信
    </h2>
    <p>
     在之前的博客中发布和订阅段有对应的bind和connect方法进行绑定和连接：
    </p>
    <pre><code class="language-cpp">bool ZmqPublisher::bind(quint16 port) {
    QString arg = QString("tcp://*:%1").arg(port);
    int rc = zmq_bind(socket, arg.toUtf8().constData());
    return rc == 0;
}

bool ZmqSubscriber::connectTo(quint16 port) {
    QString arg = QString("tcp://localhost:%1").arg(port);
    int rc = zmq_connect(socket, arg.toUtf8().constData());
    return rc == 0;
}</code></pre>
    <p>
     这两个方法内部使用tcp进行网络通信。bind和connect也可以绑定具体的ip和端口，以下是扩充方法：
    </p>
    <pre><code class="language-cpp">bool ZmqPublisher::bind(const QString &amp;ip, quint16 port)
{
    QString arg = QString("tcp://%1:%2").arg(ip).arg(port);
    int rc = zmq_bind(socket, arg.toUtf8().constData());
    return rc == 0;
}

bool ZmqSubscriber::connectTo(const QString &amp;ip, quint16 port)
{
    QString arg = QString("tcp://%1:%2").arg(ip).arg(port);
    int rc = zmq_connect(socket, arg.toUtf8().constData());
    return rc == 0;
}
</code></pre>

     基于本地文件通信
    </h2>
    <p>
     zmq除了借助TCP网络进行进程间通信还可以使用 文件进行进程间通信，这里封装对应的方法：
    </p>
    <pre><code class="language-cpp">bool ZmqPublisher::bind(const QString &amp;path)
{
    int rc = zmq_bind(socket, path.toUtf8().constData());
    return rc == 0;
}

bool ZmqSubscriber::connectTo(const QString &amp;path)
{
    int rc = zmq_connect(socket, path.toUtf8().constData());

    <h2>
     代码优化
    </h2>
    <p>
     之前写的代码调用比较麻烦，其中要在调用处新建线程类，然后订阅部分每connect一个都要新建sub对象，所以我在pub和sub类之上在新建一个类ZmqPubSub，对二者进行管理，调用者只需要新建ZmqPubSub这一个类就可以进行发布订阅消息处理相关操作，而且方便处理多个订阅端，以下是代码：
    </p>
    <p>
     头文件：
    </p>
    <pre><code class="language-cpp">#ifndef ZMQPUBSUB_H
#define ZMQPUBSUB_H

#include &lt;QObject&gt;

class QThread;
class ZmqSubscriber;
class ZmqPublisher;
class ZmqPubSub : public QObject
{
    Q_OBJECT
public:
    explicit ZmqPubSub(QObject *parent = nullptr);
    void init(const QString &amp;pubPath,const QStringList &amp;subPaths);

signals:
    void dataReceived(const QByteArray &amp;data);

public slots:
    void publishData(const QByteArray &amp;data);

private:
    bool m_Inited;
    ZmqPublisher *m_Pub;
    QList&lt;ZmqSubscriber *&gt;m_Subs;
    QList&lt;QThread *&gt;m_Ths;

};

#endif // ZMQPUBSUB_H
</code></pre>
    <p>
     源文件：
    </p>
    <pre><code class="language-cpp">#include "zmqpubsub.h"
#include "zmqpublisher.h"
#include "zmqsubscriber.h"
#include &lt;QThread&gt;
#include &lt;QDataStream&gt;

ZmqPubSub::ZmqPubSub(QObject *parent) : QObject(parent),m_Inited(false),m_Pub(nullptr)
{

}

void ZmqPubSub::init(const QString &amp;pubPath, const QStringList &amp;subPaths)
{
    if(m_Inited)return;
    QThread *pubTh=new QThread;
    m_Pub=new ZmqPublisher;
    m_Pub-&gt;bind(pubPath);
    m_Pub-&gt;moveToThread(pubTh);
    pubTh-&gt;start();
    m_Ths.append(pubTh);
    for(const QString &amp;subPath:subPaths)
    {
        QThread *subTh=new QThread;
        ZmqSubscriber *sub=new ZmqSubscriber;
        sub-&gt;connectTo(subPath);
        sub-&gt;moveToThread(subTh);
        subTh-&gt;start();
        m_Subs.append(sub);
        m_Ths.append(subTh);
        connect(sub, &amp;ZmqSubscriber::dataReceived,this,&amp;ZmqPubSub::dataReceived);
        QMetaObject::invokeMethod(sub, &amp;ZmqSubscriber::procesMessage);
    }

    m_Inited=true;
}

void ZmqPubSub::publishData(const QByteArray &amp;data)
{
    if(m_Inited)
    {
       m_Pub-&gt;publishData(data);
    }
}
</code></pre>
    <p>
     调用的时候只需要新建ZmqPubSub类即可，线程相关在内部自行处理 ，init方法初始化可以处理多个订阅端的问题。一个类实现发布订阅功能。
    </p>
