    <p>
     Vue (读音 /vjuː/，类似于
     <strong>
      view
     </strong>
     ) 是一套用于构建用户界面的
     <strong>
      渐进式框架
     </strong>
     。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与
     <a href="https://v2.cn.vuejs.org/v2/guide/single-file-components.html" rel="nofollow" title="现代化的工具链">
      现代化的工具链
     </a>
     以及各种
     <a href="https://github.com/vuejs/awesome-vue#libraries--plugins" title="支持类库">
      支持类库
     </a>
     结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。
    </p>
    <p>
     那什么是框架呢？
    </p>
    <p>
     他其实就是在基础语言之上，对各种基础功能进行封装，方便开发者，提高开发效率。JavaScript就属于前端的原生框架，而Vue.js也是一个js框架，并不是代替js的，只是对js的封装。
    </p>

     1.创建Vue

    <p>
     首先我们创建一个普通的Vue项目
    </p>
    <p>
     <img alt="" height="970" src="https://i-blog.csdnimg.cn/blog_migrate/65c3c7e68d5b883299efedb6bb47725b.png" width="1169"/>
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       删除js文件夹里的v3.2.8文件夹
      </strong>
     </span>
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="219" src="https://i-blog.csdnimg.cn/blog_migrate/954a5387ae06c634eaddd50a66c3e01c.png" width="265"/>
    </p>
    <p>
     点击index.html文件（新手推荐在html文件中通过如下方式引入Vue），删除其&lt;head&gt;和&lt;body&gt;标签中所有多余的内容，随后在head标签中添加一下&lt;script&gt;标签内容即可
     <strong>
      （以下两种标签二选一即可）
     </strong>
    </p>
    <blockquote>
     <p>
      &lt;script src="js/v2.6.10/vue.js"&gt;&lt;/script&gt;
     </p>
     <p>
      &lt;script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"&gt;&lt;/script&gt;
     </p>
    </blockquote>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="339" src="https://i-blog.csdnimg.cn/blog_migrate/5ae8aa478162c3e8b10229d7da1dd0b7.png" width="694"/>
    </p>
    <p>
     随后我们进入上面提供的Vue官网，复制以下模版代码到&lt;body&gt;标签中
    </p>
    <blockquote>
     <p>
      &lt;div id="app"&gt;
      <br/>
      {
      <!-- -->
      { message }}
      <br/>
      &lt;/div&gt;
     </p>
     <p>
      &lt;script&gt;
      <br/>
      var app = new Vue({
      <!-- -->
      <br/>
      el: '#app',
      <br/>
      data: {
      <!-- -->
      <br/>
      message: 'Hello Vue!'
      <br/>
      }
      <br/>
      })
      <br/>
      &lt;/script&gt;
     </p>
    </blockquote>
    <p>
     这样我们就完成了第一个Vue应用！
    </p>
    <h4 id="2.%E8%AF%A6%E7%BB%86%E8%A7%A3%E8%AF%BB">
     2.详细解读
    </h4>
    <p style="margin-left:.0001pt;text-align:left;">
     <span style="color:#000000;">
      Vue.js 的核心是一个允许采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统：
     </span>
    </p>
    <pre><code class="language-javascript">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8" /&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"&gt;&lt;/script&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;!-- 创建一个标签 --&gt;
		&lt;div id="app"&gt;
			&lt;!-- {<!-- -->{变量}} ——&gt;插值表达式 获取下面data中定义的数据--&gt;
		  {<!-- -->{ message }}
		&lt;/div&gt;
		
		&lt;script&gt;
			var app = new Vue({		// 创建了一个vue对象
				//#---&gt;id选择器
			  el: '#app',//绑定挂载点,建议使用id选择器，不能使用body，HTML与vue对象绑定
			  data: {//定义数据，可以定义多个，使用“ , ”分隔，Vue用到的数据定义在data中
			    message: 'Hello Vue!',
				message1: 'jim'
			  }
			})
		&lt;/script&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     注：data中也可以写复杂的数据，如数组、对象等。
    </p>
    <h2 id="%E4%BA%8C%E3%80%81Vue%E6%8C%87%E4%BB%A4">
     二、Vue指令
    </h2>
    <h4 id="1.v-text%E3%80%81v-html">
     1.v-text、v-html
    </h4>
    <p>
     我们都知道在标签中通过 {
     <!-- -->
     {message}} 插入一个值，不会影响标签中其他内容。而
    </p>
    <p>
     <span style="color:#fe2c24;">
      <strong>
       v-html = "message",v-text = "message" 会覆盖标签中其他内容
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     {
     <!-- -->
     {message}}，v-text = "message" 不能解析内容中html标签
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     v-html = "message" 可以解析内容中html标签
    </p>
    <pre><code class="language-javascript">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"&gt;&lt;/script&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="app"&gt;
		  &lt;p&gt;{<!-- -->{ message }} aaa&lt;/p&gt;
		  &lt;p v-html="message"&gt;aaa&lt;/p&gt;
		  &lt;p v-text="message"&gt;aaa&lt;/p&gt;
		&lt;/div&gt;
		
		&lt;script&gt;
			var app = new Vue({
			  el: '#app',
			  data: {
			    message: 'Hello Vue!',
			  }
			})
		&lt;/script&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>

     得到的效果如下：

    <h4 id="2.v-on">
     2.v-on
    </h4>
    <p>
     <strong>
      其作用是为元素绑定事件，有两种写法
     </strong>
    </p>
    <blockquote>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       &lt;input type="button" value="按钮" v-on:click="test(1,2)" /&gt;
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       &lt;input type="button" value="按钮" @click="test" /&gt;
      </span>
      <span style="color:#ff9900;">
       （用@符号简化）
      </span>

			  }

    <h4 id="4.v-show%E3%80%81v-if">
     4.v-show、v-if
    </h4>
    <blockquote>
     <p style="margin-left:.0001pt;text-align:justify;">
      v-show "布尔值"（效率高） true=显示 false=隐藏 隐藏时控制标签dispaly属性——隐藏显示标签的属性
     </p>
     <p style="margin-left:.0001pt;text-align:justify;">
      v-if "布尔值"（效率低，可搭配v-else使用） true=显示 false=隐藏 隐藏时直接删除标签，显示要重新创建
     </p>
    </blockquote>
    <p>
     这里我们演示一个通过布尔值变换实现图片切换的效果：
    </p>
    <pre><code class="language-javascript">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script src="js/v2.6.10/vue.js"&gt;&lt;/script&gt;
		&lt;style&gt;
			img{
				width: 100px;
			}
		&lt;/style&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="app"&gt;
			&lt;img src="img/1.jpg" v-show="isShow" /&gt;
			&lt;img src="img/2.jpg" v-show="age&gt;18" /&gt;
			
			&lt;img src="img/3.jpg" v-if="isShow" /&gt;
			&lt;img src="img/4.jpg" v-if="age&gt;18" /&gt;
			&lt;img src="img/5.jpg" v-else /&gt;
			&lt;input type="button" value="操作" @click="oper()"/&gt;
		&lt;/div&gt;
		
		&lt;script&gt;
			var app = new Vue({
				el:'#app',
				data:{
					isShow:true,
					age:15
				},
				methods:{
					oper(){
						this.isShow = !this.isShow;
					}
				}
			})
		&lt;/script&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>

    <h4 id="5.v-bind">
     5.v-bind
    </h4>
    <blockquote>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       作用是为元素绑定属性
      </span>
      <span style="color:#000000;">
       ，标签的属性值可以动态修改
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       完整写法是 v-bind：属性名="变量名"(一旦为属性添加 v-bind 值就是一个在data中定义的变量了)
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       简写
      </span>
      <span style="color:#000000;">
       省略 v-bind
      </span>
      <span style="color:#000000;">
       直接 ：属性名
      </span>
     </p>
    </blockquote>
    <pre><code class="language-javascript">&lt;!DOCTYPE html&gt;
&lt;html&gt;
	&lt;head&gt;
		&lt;meta charset="utf-8"&gt;
		&lt;title&gt;&lt;/title&gt;
		&lt;script src="js/v2.6.10/vue.js"&gt;&lt;/script&gt;
		&lt;style&gt;
			img{
				width: 100px;
			}
		&lt;/style&gt;
	&lt;/head&gt;
	&lt;body&gt;
		&lt;div id="app"&gt;
			&lt;img v-bind:src="imgurl[index]" :title="array[index]" /&gt;
			&lt;input type="button" @click="oper()" value="操作"/&gt;
		&lt;/div&gt;
		
		&lt;script&gt;
			var app = new Vue({
				el:'#app',
				data:{
					imgurl:['img/1.jpg','img/2.jpg','img/3.jpg'],
					array:['苹果手机','华为手机','小米手机'],
					index:0
				},
				methods:{
					oper(){
						this.index = (this.index+1)%3;
					}
				}
			})
		&lt;/script&gt;
	&lt;/body&gt;
&lt;/html&gt;</code></pre>

    <h4 id="6.v-for">
     6.v-for
    </h4>
    <blockquote>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       作用是根据数据生成列表结构
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       数组经常和 v-for 结合
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       使用语法是（item，index）in 数据
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       item 和 index 可以结合其他指令一起使用
      </span>
     </p>
     <p style="margin-left:.0001pt;text-align:left;">
      <span style="color:#000000;">
       数组长度的更新会同步到页面上是响应式的
