# css中实现border距离视图左右两侧有距离

首先看效果图

![](https://i-blog.csdnimg.cn/direct/e87086f09def4c1389460205e92dee48.png)

再看css是如何实现

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style>
			.main {
				background-color: aqua;
				display: block;
				width: 300px;
				padding: 0px 32px;
				box-sizing: border-box;
			}
			/* 新增的子元素样式 */
			.main-inner {
				/* padding: 0 16px; */
				border-bottom: 1px solid black;
				height: 100%;
				height: 300px;
			}
		</style>
	</head>
	<body>
		<div>
			<div class="main">
				<!-- 新增子元素 -->
				<div class="main-inner">
					1111
				</div>
			</div>
		</div>
	</body>
</html>
```
