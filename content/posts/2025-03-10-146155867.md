    <p>
     在Eclipse 中使用 MyBatis 进行开发，通常需要以下步骤：
    </p>
    <h4>
     1. 创建 Maven 项目
    </h4>
    <p>
     首先，在 Eclipse 中创建一个 Maven 项目。如果你还没有安装 Maven 插件，可以通过 Eclipse Marketplace 安装 Maven 插件。
    </p>
    <ol>
     <li>
      <p>
       打开 Eclipse，选择
       <code>
        File -&gt; New -&gt; Maven Project
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       在向导中选择
       <code>
        Create a simple project
       </code>
       ，然后点击
       <code>
        Next
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       输入
       <code>
        Group Id
       </code>
       和
       <code>
        Artifact Id
       </code>
       ，然后点击
       <code>
        Finish
       </code>
       。
      </p>
     </li>
    </ol>
    <h4>
     2. 添加 MyBatis 依赖
    </h4>
    <p>
     在
     <code>
      pom.xml
     </code>
     文件中添加 MyBatis 的依赖。
    </p>
    <p>
     xml
    </p>
    <p>
     复制
    </p>
    <pre>&lt;dependencies&gt;
    &lt;!-- MyBatis 核心依赖 --&gt;
    &lt;dependency&gt;
        &lt;groupId&gt;org.mybatis&lt;/groupId&gt;
        &lt;artifactId&gt;mybatis&lt;/artifactId&gt;
        &lt;version&gt;3.5.7&lt;/version&gt;
    &lt;/dependency&gt;

    &lt;!-- 数据库驱动，例如 MySQL --&gt;
    &lt;dependency&gt;
        &lt;groupId&gt;mysql&lt;/groupId&gt;
        &lt;artifactId&gt;mysql-connector-java&lt;/artifactId&gt;
        &lt;version&gt;8.0.26&lt;/version&gt;
    &lt;/dependency&gt;

    &lt;!-- 其他依赖，如日志框架 --&gt;
    &lt;dependency&gt;
        &lt;groupId&gt;org.slf4j&lt;/groupId&gt;
        &lt;artifactId&gt;slf4j-log4j12&lt;/artifactId&gt;
        &lt;version&gt;1.7.32&lt;/version&gt;
    &lt;/dependency&gt;
&lt;/dependencies&gt;</pre>
    <p>
     运行 HTML
    </p>
    <h4>
     3. 配置 MyBatis
    </h4>
    <p>
     在
     <code>
      src/main/resources
     </code>
     目录下创建 MyBatis 的配置文件
     <code>
      mybatis-config.xml
     </code>
     。
    </p>
    <p>
     xml
    </p>
    <p>
     复制
    </p>
    <pre>&lt;?xml version="1.0" encoding="UTF-8" ?&gt;
&lt;!DOCTYPE configuration
  PUBLIC "-//mybatis.org//DTD Config 3.0//EN"
  "http://mybatis.org/dtd/mybatis-3-config.dtd"&gt;
&lt;configuration&gt;
    &lt;environments default="development"&gt;
        &lt;environment id="development"&gt;
            &lt;transactionManager type="JDBC"/&gt;
            &lt;dataSource type="POOLED"&gt;
                &lt;property name="driver" value="com.mysql.cj.jdbc.Driver"/&gt;
                &lt;property name="url" value="jdbc:mysql://localhost:3306/your_database"/&gt;
                &lt;property name="username" value="your_username"/&gt;
                &lt;property name="password" value="your_password"/&gt;
            &lt;/dataSource&gt;
        &lt;/environment&gt;
    &lt;/environments&gt;
    &lt;mappers&gt;
        &lt;mapper resource="com/example/mapper/YourMapper.xml"/&gt;
    &lt;/mappers&gt;
&lt;/configuration&gt;</pre>
    <p>
     运行 HTML
    </p>
    <h4>
     4. 创建实体类和 Mapper 接口
    </h4>
    <p>
     在
     <code>
      src/main/java
     </code>
     目录下创建实体类和 Mapper 接口。
    </p>
    <p>
     java
    </p>
    <p>
     复制
    </p>
    <pre>package com.example.model;

public class User {
    private int id;
    private String name;
    private String email;

    // Getters and Setters
}</pre>
    <p>
     java
    </p>
    <p>
     复制
    </p>
    <pre>package com.example.mapper;

import com.example.model.User;

public interface UserMapper {
    User selectUser(int id);
}</pre>
    <h4>
     5. 创建 Mapper XML 文件
    </h4>
    <p>
     在
     <code>
      src/main/resources/com/example/mapper
     </code>
     目录下创建
     <code>
      UserMapper.xml
     </code>
     文件。
    </p>
    <p>
     xml
    </p>
    <p>
     复制
    </p>
    <pre>&lt;?xml version="1.0" encoding="UTF-8" ?&gt;
&lt;!DOCTYPE mapper
  PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
  "http://mybatis.org/dtd/mybatis-3-mapper.dtd"&gt;
&lt;mapper namespace="com.example.mapper.UserMapper"&gt;
    &lt;select id="selectUser" resultType="com.example.model.User"&gt;
        SELECT * FROM users WHERE id = #{id}
    &lt;/select&gt;
&lt;/mapper&gt;</pre>
    <p>
     运行 HTML
    </p>
    <h4>
     6. 使用 MyBatis
    </h4>
    <p>
     在 Java 代码中使用 MyBatis 进行数据库操作。
    </p>
    <p>
     java
    </p>
    <p>
     复制
    </p>
    <pre>package com.example;

import java.io.InputStream;
import org.apache.ibatis.session.SqlSession;
import org.apache.ibatis.session.SqlSessionFactory;
import org.apache.ibatis.session.SqlSessionFactoryBuilder;
import com.example.mapper.UserMapper;
import com.example.model.User;

public class MyBatisExample {
    public static void main(String[] args) {
        String resource = "mybatis-config.xml";
        InputStream inputStream = MyBatisExample.class.getClassLoader().getResourceAsStream(resource);
        SqlSessionFactory sqlSessionFactory = new SqlSessionFactoryBuilder().build(inputStream);

        try (SqlSession session = sqlSessionFactory.openSession()) {
            UserMapper mapper = session.getMapper(UserMapper.class);
            User user = mapper.selectUser(1);
            System.out.println(user.getName());
        }
    }
}</pre>
    <h4>
     7. 运行项目
    </h4>
    <p>
     右键点击
     <code>
      MyBatisExample.java
     </code>
     ，选择
     <code>
      Run As -&gt; Java Application
     </code>
     ，运行项目。
    </p>
    <h4>
     8. 调试和优化
    </h4>
    <p>
     根据项目需求，继续添加更多的 Mapper 接口和 XML 文件，优化 SQL 语句，配置日志等。
    </p>
    <h4>
     9. 其他注意事项
    </h4>
    <ul>
     <li>
      <p>
       确保数据库连接信息正确。
      </p>
     </li>
     <li>
      <p>
       确保 Mapper XML 文件路径和命名空间正确。
      </p>
     </li>
     <li>
      <p>
       可以使用 MyBatis Generator 自动生成实体类、Mapper 接口和 XML 文件。
      </p>
     </li>
    </ul>
    <p>
     通过以上步骤，你应该能够在 Eclipse 中成功使用 MyBatis 进行开发。
    </p>
