     安装 IntelliJ IDEA 2023.3.1 非常简单，以下是详细的安装步骤，适用于 Windows、macOS 和 Linux 系统。
    </p>
    <hr/>
    <h4>
     <strong>
      1. 下载 IntelliJ IDEA
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        IntelliJ IDEA下载链接：
        <a class="link-info" href="https://pan.quark.cn/s/3ad975664934" rel="nofollow" title="https://pan.quark.cn/s/3ad975664934">
         https://pan.quark.cn/s/3ad975664934
        </a>
       </strong>
      </p>
     </li>
     <li>
      <p>
       选择适合你的操作系统的版本：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          Ultimate 版
         </strong>
         ：功能全面，支持所有开发语言和框架（需付费）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          Community 版
         </strong>
         ：免费版，适合 Java 和 Kotlin 开发。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       点击下载按钮，等待下载完成。
      </p>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      2. 安装 IntelliJ IDEA
     </strong>
    </h4>
    <h5>
     <strong>
      Windows 系统
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       找到下载的安装文件（通常是
       <code>
        ideaIU-2023.3.1.exe
       </code>
       或
       <code>
        ideaIC-2023.3.1.exe
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       双击运行安装程序。
      </p>
     </li>
     <li>
      <p>
       按照安装向导的提示进行操作：
      </p>
      <ul>
       <li>
        <p>
         选择安装路径（默认路径通常是
         <code>
          C:\Program Files\JetBrains\IntelliJ IDEA 2023.3.1
         </code>
         ）。
        </p>
       </li>
       <li>
        <p>
         选择是否创建桌面快捷方式。
        </p>
       </li>
       <li>
        <p>
         选择是否将 IntelliJ IDEA 添加到系统 PATH 环境变量（可选）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       点击
       <strong>
        Install
       </strong>
       开始安装。
      </p>
     </li>
     <li>
      <p>
       安装完成后，点击
       <strong>
        Finish
       </strong>
       并启动 IntelliJ IDEA。
      </p>
     </li>
    </ol>
