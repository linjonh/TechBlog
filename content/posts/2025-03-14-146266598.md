    <h3>
     题目描述：
    </h3>
    <p>
     给你一个仅由数字 0 - 9 组成的字符串
     <code>
      num
     </code>
     。如果偶数下标处的数字之和等于奇数下标处的数字之和，则认为该数字字符串是一个
     <strong>
      平衡字符串
     </strong>
     。
    </p>
    <p>
     如果
     <code>
      num
     </code>
     是一个
     <strong>
      平衡字符串
     </strong>
     ，则返回
     <code>
      true
     </code>
     ；否则，返回
     <code>
      false
     </code>
     。
    </p>
    <h3>
     代码思路：
    </h3>
    <ol>
     <li>
      <strong>
       初始化变量
      </strong>
      ：
      <ul>
       <li>
        <code>
         n
        </code>
        ：字符串
        <code>
         num
        </code>
        的长度。
       </li>
       <li>
        <code>
         num1
        </code>
        和
        <code>
         num2
        </code>
        ：分别用于存储偶数索引和奇数索引上数字的和，初始值都为 0。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       遍历字符串
      </strong>
      ：
      <ul>
       <li>
        使用一个
        <code>
         for
        </code>
        循环遍历字符串
        <code>
         num
        </code>
        的每个字符。
       </li>
       <li>
        在每次迭代中，通过
        <code>
         i % 2
        </code>
        判断当前索引
        <code>
         i
        </code>
        是偶数还是奇数。
        <ul>
         <li>
          如果
          <code>
           i % 2 == 0
          </code>
          ，说明当前索引是偶数，将当前字符（转换为整数）加到
          <code>
           num1
          </code>
          上。
         </li>
         <li>
          如果
          <code>
           i % 2 != 0
          </code>
          （即
          <code>
           i
          </code>
          是奇数），将当前字符（转换为整数）加到
          <code>
           num2
          </code>
          上。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       比较和返回结果
      </strong>
      ：
      <ul>
       <li>
        在遍历完所有字符后，比较
        <code>
         num1
        </code>
        和
        <code>
         num2
        </code>
        的值。
       </li>
       <li>
        如果它们相等，说明偶数索引和奇数索引上的数字之和相等，返回
        <code>
         True
        </code>
        ，表示字符串是平衡的。
       </li>
       <li>
        如果它们不相等，返回
        <code>
         False
        </code>
        ，表示字符串不是平衡的。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     代码实现：
    </h3>
    <pre><code class="hljs">class Solution:
    def isBalanced(self, num: str) -&gt; bool:
        n = len(num)
        num1 = num2 = 0
        for i in range(n):
            if i % 2 == 0:
                num1 += int(num[i])
            else:
                num2 += int(num[i])
        if num1 == num2:
            return True
        else:
            return False</code></pre>
