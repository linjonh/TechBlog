    <h2>
     NFS
    </h2>
    <h3>
     NFS服务
    </h3>
    <p>
     nfs，最早是Sun这家公司所发展出来的，它最大的功能就是可以透过网络，让不同的机器，不同的操作系统，进行实现文档的共享。所以你可以简单的将他看做是文件服务器。
    </p>
    <h3>
     实验准备
    </h3>
    <p>
     ①先准备一个服务器端的操作系统和客户端的操作系统(Red Hat)。
    </p>
    <p>
     ②选择NAT模式，让这两个操作系统能够ping通
    </p>
    <p>
     ③连接镜像
    </p>
    <h3>
     实验步骤
    </h3>
    <h4>
     服务器端
    </h4>
    <ol>
     <li>
      <p>
       配置ip地址，并与客户端ping通
      </p>
      <p class="img-center">
       <img alt="alt" height="792" src="https://i-blog.csdnimg.cn/img_convert/30b4a48ce8477fbf298901c0975947d6.png" width="1143"/>
      </p>
     </li>
     <li>
      <p>
       挂载镜像 创建挂载点：
       <code>
        mkdir /dvd
       </code>
       挂载镜像：
       <code>
        mount /dev/sr0 /dvd
       </code>
       查看镜像是否挂载成功：
       <code>
        df -hT
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="359" src="https://i-blog.csdnimg.cn/img_convert/ffab56716127f940bd2e53f89dce94c8.png" width="1029"/>
      </p>
     </li>
     <li>
      <p>
       配置yum源
       <code>
        vim /etc/yum.repos.d/app.repo
       </code>
       <code>
        [app] name=app baseurl=file:///dvd gpgcheck=0
       </code>
      </p>
      <p>
       <code>
        vim /etc/yum.repos.d/base.repo
       </code>
       <code>
        [base] name=base baseurl=file:///dvd gpgcheck=0
       </code>
       gpgcheck={1|0}# gpgcheck=1表示检查；可以不检查gpgcheck=0 清理缓存：
       <code>
        yum clean all
       </code>
       查看yum源配置是否成功：
       <code>
        yum repolist
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="448" src="https://i-blog.csdnimg.cn/img_convert/ef30c83bc851a0c2766d903729a9fcbf.png" width="787"/>
      </p>
      <p>
      </p>
      <p class="img-center">
       <img alt="alt" height="610" src="https://i-blog.csdnimg.cn/img_convert/2030e9673377a10078a1a52c77f263f4.png" width="1196"/>
      </p>
     </li>
     <li>
      <p>
       安装NFS
       <code>
        yum install nfs-utils
       </code>
       启动：
       <code>
        systemctl start nfs-server.service
       </code>
       <code>
        systemctl enable nfs-server.service
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="1073" src="https://i-blog.csdnimg.cn/img_convert/bf7e8df2d3a575637c564ae3ab669c28.png" width="1205"/>
      </p>
      <p class="img-center">
       <img alt="alt" height="108" src="https://i-blog.csdnimg.cn/img_convert/a4aff927f742a4ce8d83def482ede9b3.png" width="857"/>
      </p>
     </li>
     <li>
      <p>
       创建挂载目录 创建挂载目录：
       <code>
        mkdir /aa
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="33" src="https://i-blog.csdnimg.cn/img_convert/fecc7e072f9b108ded898169dc992f66.png" width="453"/>
      </p>
     </li>
     <li>
      <p>
       给目录设置权限 给目录设置权限：
       <code>
        chmod 777 /aa
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="33" src="https://i-blog.csdnimg.cn/img_convert/8002170dea5897a87a1ddccb986656fd.png" width="514"/>
      </p>
     </li>
     <li>
      <p>
       编写配置文件 编写配置文件：
       <code>
        vim /etc/exports
       </code>
      </p>
      <pre> /aa  另一个操作系统的地址/24(rw,sync)
      //客户端的ip地址 （读写权限，资料同步写入到内存与硬盘中）</pre>
     </li>
    </ol>
    <p>
    </p>
    <p class="img-center">
     <img alt="alt" height="180" src="https://i-blog.csdnimg.cn/img_convert/628234acee12e7be8ee09ebec46bd34f.png" width="562"/>
    </p>
    <ol>
     <li>
      <p>
       启动 启动：
       <code>
        systemctl restart nfs-server.service
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="37" src="https://i-blog.csdnimg.cn/img_convert/359e23d05efdf1ff6ea0cc79953455c6.png" width="859"/>
      </p>
     </li>
     <li>
      <p>
       关闭防火墙 关闭防火墙：
       <code>
        systemctl stop firewalld
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="30" src="https://i-blog.csdnimg.cn/img_convert/6b8f662e512e42bee9ee80af75a3f81e.png" width="683"/>
      </p>
     </li>
    </ol>
    <p>
    </p>
    <h4>
     客户端
    </h4>
    <ol>
     <li>
      <p>
       关闭防火墙 关闭防火墙：
       <code>
        systemctl stop firewalld
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="47" src="https://i-blog.csdnimg.cn/img_convert/9de717e4cfded111e2be3579cb0c9403.png" width="678"/>
      </p>
     </li>
     <li>
      <p>
       验证 验证：
       <code>
        showmount -e （服务器端的ip地址）
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="96" src="https://i-blog.csdnimg.cn/img_convert/754b725c8cf278b2f1de68ec63dfdb70.png" width="793"/>
      </p>
     </li>
     <li>
      <p>
       创建挂载点 创建挂载点：
       <code>
        mkdir /bb
       </code>
      </p>
     </li>
     <li>
      <p>
       设置权限
       <code>
        Chmod 777 /bb
       </code>
      </p>
     </li>
     <li>
      <p>
       挂载 临时挂载:
       <code>
        mount -t nfs 服务器端的ip地址：/aa /bb
       </code>
      </p>
     </li>
    </ol>
    <p>
     永久挂载：
     <code>
      vim /etc/fstab
     </code>
    </p>
    <p>
     编辑：
     <code>
      服务器端的ip地址：/aa /bb nfs defaults 0 0
     </code>
    </p>
    <p class="img-center">
     <img alt="alt" height="519" src="https://i-blog.csdnimg.cn/img_convert/86053f62038b431a69d52eb5d5af0c9a.png" width="1224"/>
    </p>
    <ol>
     <li>
      <p>
       查看是否挂载上 查看是否挂载上：
       <code>
        df -hT
       </code>
      </p>
      <p class="img-center">
       <img alt="alt" height="321" src="https://i-blog.csdnimg.cn/img_convert/ff00486d449444018a1f5e7cf4dc7567.png" width="1203"/>
      </p>
     </li>
    </ol>
