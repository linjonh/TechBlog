# 设计师必备的用户界面设计工具，工具包和资源

原文：
<http://blog.aotaostudio.com/919.html>

---

都是一些UI界面与设计使用的元素，软件，网站等。内容很丰富，适合网页设计师，用户体验设计师，界面设计师，产品设计师，JS前段开发，手机产品设计，ipad产品设计等使用

Lumzy

Lumzy是一个网站应用和原型界面制作工具。使用Lumzy，您可以轻松创建UI模型并即时发送到客户电脑中。 Lumzy还具有团队协作编辑工具。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141034_1.jpg)

[Mockingbird](https://gomockingbird.com/)

[百舌鸟是一个在线工具，它使您轻松地创建UI界面模型，预览UI，并可以共享你的版面编排效果图。](https://gomockingbird.com/)

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141052_1.jpg)

The Pencil Project

为设计图表和用户界面图形原型开发的一个自由和开源工具。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141105_1.jpg)

Dojo

Dojo提供了强大的性能并节省您的时间。这是适合经验丰富前端开发者的JS开发工具包。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141257_1.jpg)

Mocklinkr

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141317_1.jpg)

MockFlow

MockFlow是一个在线工具，用户界面设计和协作版面编排工具。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141346_1.jpg)

Serena Prototype Composer

Serena Prototype Composer是制作流程图，模拟程序流程和用户界面设计程序。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141400_1.jpg)

Cacoo

Cocoo是一个用户友好的在线绘图工具，允许你创建一个如网站地图，线框，UML和网络图图等。 Cacoo可以免费使用。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141418_1.jpg)

DENIM

一个早期非正式的网站和用户界面设计工具

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141431_1.jpg)

Hot Gloo

网上应用线框

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141456_1.jpg)

User Interface Design Framework

一个免费提供网页设计师使用的用户界面设计工具。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141507_1.jpg)

Pattern Tap

可以寻找一些设计资源与模式等。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141535_1.jpg)

User Interface Design Patterns

提供一些热门的用户UI界面设计模式参考图，分类很详细，如果设计网站或UI界面可以上去找一些资源，很不错。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141603_1.jpg)

Patternry

Patternry是用户界面设计模式库的目的是帮助解决常见的接口设计问题。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141639_1.jpg)

Mephobox

UI设计界面资源欣赏参考网站。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141655_1.jpg)

Android Asset Studio

可以在线生成图表，只需要输入参数即可

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141709_1.jpg)

Mobility

移动用户界面设计使用，可以输入参数进行测试，很方便

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141721_1.jpg)

Android UI Elements Set

可以自由设定Android操作系统的用户界面元素。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141735_1.jpg)

OSX Leopard GUI Set

该软件包含一些常用的Photoshop原文件和PNG文件，这些元素都是用来设计GUI界面的素材，通过该软件你可以快速制作出一些出色的界面

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141747_1.jpg)

ALL In One Web Elements Kit

网页布局元素大集合,PSD文件。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141817_1.jpg)

iPad GUI Set

iPad平板电脑上设计布局用的元素集合。

![](http://www.cocoachina.com/cms/uploads/allimg/110822/3292_110822141829_1.jpg)
