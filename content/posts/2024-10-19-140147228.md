    <h3 id="%E5%89%8D%E8%A8%80">
     <a id="_7">
     </a>
     前言
    </h3>
    <p>
     本文解释了如何用操纵杆或游戏手柄控制你的飞行器，使用任务计划器向飞行器发送"
     <span style="color:#98c091;">
      <strong>
       RC Override
      </strong>
     </span>
     "消息。
    </p>
    <hr/>
    <p style="margin-left:.0001pt;text-align:justify;">
     其他
     <span style="color:#4da8ee;">
      GCS
     </span>
     也可能支持"
     <span style="color:#98c091;">
      <strong>
       RC Override
      </strong>
     </span>
     "，但本文不涉及这些。
    </p>
    <p style="margin-left:.0001pt;text-align:center;">
     <img alt="" height="393" src="https://i-blog.csdnimg.cn/blog_migrate/e99b4493abab0d556d7b24b1a4756a01.jpeg" width="437"/>
    </p>
    <p style="margin-left:.0001pt;text-align:center;">
     罗技
     <span style="color:#fe2c24;">
      F310
     </span>
     操纵杆
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#28ae7b;">
      <strong>
       <span style="color:#ffffff;">
        ！
       </span>
      </strong>
      <strong>
       <span style="color:#ffffff;">
        Tip
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#e2feeb;">
      即使使用操纵杆飞行，你也应该保持一个常规的发射器/接收器连接，并准备作为备份使用。在
      <span style="color:#956fe7;">
       ArduPilot
      </span>
      的未来版本中，在完成了更多的失控保护测试后，这个建议可能会改变。
     </span>
    </p>
    <h3 id="1%20%E4%BD%A0%E5%B0%86%E9%9C%80%E8%A6%81%E4%BB%80%E4%B9%88" style="margin-left:.0001pt;text-align:justify;">
     1 你将需要什么
    </h3>
    <p style="margin-left:.0001pt;text-align:justify;">
     你将需要以下设备：
    </p>
    <ul>
     <li style="text-align:justify;">
      <span style="color:#4da8ee;">
       USB
      </span>
      <span style="color:#494949;">
       操纵杆或游戏手柄，如罗技
      </span>
      <span style="color:#4da8ee;">
       F310
      </span>
      <span style="color:#494949;">
       (
       <a href="https://www.logitechg.com/en-us/products/gamepads/f310-gamepad.html" rel="nofollow" title="Logitech F310">
        Logitech F310
       </a>
       )（如上图）；
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="color:#494949;">
       你的地面站和飞行器之间的遥测连接(
       <a href="https://ardupilot.org/copter/docs/common-telemetry-landingpage.html#common-telemetry-landingpage" rel="nofollow" title="telemetry connection">
        telemetry connection
       </a>
       )；
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="color:#494949;">
       一台运行任务规划器的笔记本电脑。
      </span>
     </li>
    </ul>
    <h3 id="2%20%E6%A0%A1%E5%87%86" style="margin-left:.0001pt;text-align:justify;">
     2 校准
    </h3>
    <p style="margin-left:.0001pt;text-align:justify;">
     首先，你必须使用
     <span style="color:#4da8ee;">
      Windows
     </span>
     操纵杆/游戏控制器向导来校准操纵杆。
    </p>
    <p style="margin-left:.0001pt;text-align:center;">
     <img alt="" height="942" src="https://i-blog.csdnimg.cn/blog_migrate/289a019a354b9cef1d871effe0425747.jpeg" width="814"/>
    </p>
    <h3 id="3%20%E7%94%A8%E4%BB%BB%E5%8A%A1%E8%A7%84%E5%88%92%E5%99%A8%E8%BF%9B%E8%A1%8C%E8%AE%BE%E7%BD%AE" style="margin-left:.0001pt;text-align:justify;">
     3 用任务规划器进行设置
    </h3>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       1.
      </strong>
     </span>
     将你的
     <span style="color:#4da8ee;">
      USB
     </span>
     操纵杆/游戏手柄连接到笔记本电脑上。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       2.
      </strong>
     </span>
     打开任务计划器飞行数据屏幕。在"
     <span style="color:#98c091;">
      <strong>
       行动
      </strong>
     </span>
     "选项卡上按下"
     <span style="color:#98c091;">
      <strong>
       操纵杆
      </strong>
     </span>
     "按钮。
    </p>
    <p style="margin-left:.0001pt;text-align:center;">
     <img alt="" height="803" src="https://i-blog.csdnimg.cn/blog_migrate/d3e79912ab9044afde6e6f0b6bcd844a.png" width="1189"/>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       3.
      </strong>
     </span>
     确保罗技操纵杆出现在下拉菜单中。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       4.
      </strong>
     </span>
     一旦按下"
     <span style="color:#98c091;">
      <strong>
       启用
      </strong>
     </span>
     "按钮，将告诉任务规划器开始向飞行器发送指令，因此在最初的设置中，不要按下该按钮。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       5.
      </strong>
     </span>
     在"
     <span style="color:#98c091;">
      <strong>
       横滚
      </strong>
     </span>
     "行，点击自动检测按钮，然后摇动你想用于横滚的控制器，向左和向右。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       6.
      </strong>
     </span>
     该通道可以通过反转复选框进行反转。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       7.
      </strong>
     </span>
     <span style="color:#4da8ee;">
      Expo
     </span>
     的数字应该在
     <span style="color:#4da8ee;">
      -100
     </span>
     到
     <span style="color:#4da8ee;">
      100
     </span>
     的范围内。
    </p>
    <ul>
     <li style="text-align:justify;">
      <span style="color:#fe2c24;">
       0
      </span>
      <span style="color:#494949;">
       =
      </span>
      <span style="color:#4da8ee;">
       no expo
      </span>
      <span style="color:#494949;">
       ；
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="color:#fe2c24;">
       100
      </span>
      <span style="color:#494949;">
       = 中间周围的反应低，边缘的反应非常快；
      </span>
     </li>
     <li style="text-align:justify;">
      <span style="color:#fe2c24;">
       -100
      </span>
      <span style="color:#494949;">
       = 中间周围的反应非常快，边缘的反应较小（很少有人使用负
      </span>
      <span style="color:#4da8ee;">
       expo
      </span>
      <span style="color:#494949;">
       ）。
      </span>
     </li>
    </ul>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       8.
      </strong>
     </span>
     重复俯仰、油门和方向舵的操作，然后按下保存按钮。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       9.
      </strong>
     </span>
     你可能想设置其他的按钮，用于解锁和加锁。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       10.
      </strong>
     </span>
     与其为
     <span style="color:#4da8ee;">
      ch5
     </span>
     （飞行模式通道）设置一个杆或按钮，不如通过选择"
     <span style="color:#98c091;">
      <strong>
       改变模式
      </strong>
     </span>
     "来设置个别按钮启动飞行模式，然后按下"
     <span style="color:#98c091;">
      <strong>
       设置
      </strong>
     </span>
     "按钮，从下拉菜单中选择飞行模式（见上图）。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       11.
      </strong>
     </span>
     完成后，按下保存按钮。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="color:#fe2c24;">
      <strong>
       12.
      </strong>
     </span>
     在
     <span style="color:#4da8ee;">
      CONFIG/TUNING
     </span>
     | 全部参数列表中，检查
     <span style="color:#0070c0;">
      <a href="https://ardupilot.org/copter/docs/parameters.html#sysid-mygcs" rel="nofollow" title="SYSID_MYGCS">
       SYSID_MYGCS
      </a>
     </span>
     参数是否与你的
     <span style="color:#4da8ee;">
      GCS
     </span>
     的系统
     <span style="color:#4da8ee;">
      ID
     </span>
     相匹配。这个参数限制了哪个
     <span style="color:#4da8ee;">
      GCS
     </span>
     可以向飞行器发送超控信号。
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#28ae7b;">
      <strong>
       <span style="color:#ffffff;">
        ！
       </span>
      </strong>
      <strong>
       <span style="color:#ffffff;">
        Tip
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#e2feeb;">
      对于任务规划器，默认的
      <span style="color:#4da8ee;">
       GCS
      </span>
      系统
      <span style="color:#4da8ee;">
       ID
      </span>
      是
      <span style="color:#fe2c24;">
       255
      </span>
      。
      <span style="color:#4da8ee;">
       APM
      </span>
      任务规划器
      <span style="color:#4da8ee;">
       2
      </span>
      的默认系统
      <span style="color:#4da8ee;">
       ID
      </span>
      是
      <span style="color:#fe2c24;">
       252
      </span>
      （但
      <span style="color:#4da8ee;">
       GCS
      </span>
      将自动设置其
      <span style="color:#4da8ee;">
       ID
      </span>
      以匹配
      <span style="color:#4da8ee;">
       SYSID_MYGCS
      </span>
      的
      <span style="color:#4da8ee;">
       RC
      </span>
      覆盖命令）。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#53a8dd;">
      <strong>
       <span style="color:#ffffff;">
        ！
       </span>
      </strong>
      <strong>
       <span style="color:#ffffff;">
        Note
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#d3eafd;">
      如果你得到一个错误
      <span style="color:#fe2c24;">
       PreArm
      </span>
      ：
      <span style="color:#4da8ee;">
       RC
      </span>
      未校准（校准后），你还需要手动将
      <span style="background-color:#d9d9d9;">
       <span style="color:#f44528;">
        RC1_MIN
       </span>
      </span>
      改为
      <span style="color:#fe2c24;">
       1101
      </span>
      ，
      <span style="background-color:#d9d9d9;">
       <span style="color:#f44528;">
        RC1_MAX
       </span>
      </span>
      改为
      <span style="color:#fe2c24;">
       1901
      </span>
      （然后对
      <span style="color:#4da8ee;">
       RC2_
      </span>
      、
      <span style="color:#4da8ee;">
       RC3_
      </span>
      和
      <span style="color:#4da8ee;">
       RC4_
      </span>
      最大/最小参数进行重复）。
     </span>
    </p>
    <p style="margin-left:.0001pt;text-align:justify;">
     <span style="background-color:#d3eafd;">
      造成这个错误的原因是
      <span style="color:#956fe7;">
       Mission Planner
      </span>
      将操纵杆精确地映射到
      <span style="color:#4da8ee;">
       RC
      </span>
      最小和最大范围，但预解锁检查认为，如果这些值没有至少偏离默认值
      <span style="color:#fe2c24;">
       1
      </span>
      个
      <span style="color:#4da8ee;">
       PWM
      </span>
      ，则说明没有进行校准。如果你已经对你的发射机/接收机系统进行了遥控校准，这就没有必要了。不要对
      <span style="color:#4da8ee;">
       JOYSTICK
      </span>
      进行遥控校准，它与遥控系统使用的控制机制不同。它使用
      <span style="color:#4da8ee;">
       MAVLink
      </span>
      的覆盖消息。
     </span>
    </p>
