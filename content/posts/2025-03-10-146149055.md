    <h2>
     <a id="_3">
     </a>
     简介
    </h2>
    <p>
     <strong>
      什么是 AIGC ？
     </strong>
    </p>
    <blockquote>
     <p>
      <code>
       AIGC
      </code>
      （
      <code>
       Artificial Intelligence Generated Content
      </code>
      ）是“人工智能生成内容”的简称，指的是利用人工智能技术，通过算法和模型自动生成各种形式的内容，包括但不限于文本、图像、音频、视频等。
     </p>
    </blockquote>
    <p>
     <strong>
      什么是 Chat Nio ？
     </strong>
    </p>
    <blockquote>
     <p>
      <code>
       Chat Nio
      </code>
      是一个下一代
      <code>
       AIGC
      </code>
      一站式商业解决方案，旨在为
      <code>
       B/C
      </code>
      端用户提供丰富的功能和良好的用户体验。它支持多种
      <code>
       AI
      </code>
      模型，包括
      <code>
       OpenAI
      </code>
      、
      <code>
       Midjourney
      </code>
      、
      <code>
       Claude
      </code>
      等。
     </p>
    </blockquote>
    <h3>
     <a id="_13">
     </a>
     主要特点
    </h3>
    <ol>
     <li>
      <strong>
       多模型支持
      </strong>
      : 集成多个
      <code>
       AI
      </code>
      模型，满足不同需求。
     </li>
     <li>
      <strong>
       优雅的用户界面
      </strong>
      : 采用现代设计标准，兼容
      <code>
       PC
      </code>
      、平板和移动设备。
     </li>
     <li>
      <strong>
       国际化支持
      </strong>
      : 支持多语言切换，欢迎贡献翻译。
     </li>
     <li>
      <strong>
       强大的对话同步
      </strong>
      : 实现跨设备的对话同步和分享功能。
     </li>
     <li>
      <strong>
       模型市场和自定义预设
      </strong>
      : 提供模型介绍和自定义功能，支持云端同步。
     </li>
     <li>
      <strong>
       丰富的文件解析
      </strong>
      : 支持多种文件格式的解析和
      <code>
       OCR
      </code>
      图像识别。
     </li>
     <li>
      <strong>
       灵活的计费方式
      </strong>
      : 提供订阅和弹性计费选项，满足不同用户的需求。
     </li>
     <li>
      <strong>
       综合后台管理
      </strong>
      : 包括用户管理、通知管理、价格设置等多种功能。
     </li>
    </ol>
    <p>
     <code>
      Chat Nio
     </code>
     的目标是结合强大的
     <code>
      API
     </code>
     分发系统与美观的用户界面设计，提升用户体验和粘性，同时支持企业级的管理与功能。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/292e321c98e7a781de2ea18ed71fb3c3.png"/>
    </p>
    <p>
     所以从功能上看，很像是
     <code>
      ChatGPT Next Web
     </code>
     和
     <code>
      One API
     </code>
     的合体
    </p>
    <blockquote>
     <p>
      文章传送门：
     </p>
     <ul>
      <li>
       <a href="https://laosu.tech/2024/04/08/%E8%B7%A8%E5%B9%B3%E5%8F%B0%E7%A7%81%E4%BA%BAChatGPT%E5%BA%94%E7%94%A8ChatGPT-Next-Web" rel="nofollow">
        跨平台私人ChatGPT应用ChatGPT-Next-Web
       </a>
      </li>
      <li>
       <a href="https://laosu.tech/2024/04/22/%E5%A4%A7%E6%A8%A1%E5%9E%8B%E6%8E%A5%E5%8F%A3%E7%AE%A1%E7%90%86%E5%92%8C%E5%88%86%E5%8F%91%E7%B3%BB%E7%BB%9FOne%20API" rel="nofollow">
        大模型接口管理和分发系统One API
       </a>
      </li>
     </ul>

     就能看到主界面

    <p>
     部署成功后, 管理员账号为
     <code>
      root
     </code>
     , 密码默认为
     <code>
      chatnio123456
     </code>

     <code>
      kimi-free-api
     </code>
     为例，以下几项为必填
    </p>
    <ul>
     <li>
      名称：随意
     </li>
     <li>
      类型：选择
      <code>
       OpenAI
      </code>
     </li>
     <li>
      模型：设置模型名称，例如
      <code>
       kimi
      </code>
     </li>
     <li>
      密钥：在
      <code>
       https://kimi.moonshot.cn
      </code>
      获取的
      <code>
       refresh_token
      </code>
     </li>
     <li>
      接入点：
      <code>
       kimi-free-api
      </code>
      的访问地址
     </li>
