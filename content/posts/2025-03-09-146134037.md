       <p style="margin-left:0; margin-right:0">
        使用Docker，只需要配置一次Docker容器上面的应用，就可以跨平台，跨服务器，实现应用程序跨平台的无缝衔接
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        docker的基本组成
       </p>
       <p style="margin-left:0; margin-right:0">
        <img alt="" height="389" src="https://i-blog.csdnimg.cn/direct/371199fce9414e6ab0389ef4a41d266f.jpeg" width="709"/>
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        镜像(image):
       </p>
       <p style="margin-left:0; margin-right:0">
        docker镜像就好比一个模板，可用通过这个模板来创建容器服务，通过这个镜像可以创建多个容器,最终服务允许或者项目运行就是在这个容器之中
       </p>
       <p style="margin-left:0; margin-right:0">
        tomcat镜像=》run=》tomcker01
       </p>
       <p style="margin-left:0; margin-right:0">
        tomcat镜像=》run=》tomcker02
       </p>
       <p style="margin-left:0; margin-right:0">
        容器(container):
       </p>
       <p style="margin-left:0; margin-right:0">
        docker利用利用容器技术，独立运行一个或者一个组应用，通过镜像来创建，
       </p>
       <p style="margin-left:0; margin-right:0">
        容器就好比一个微小型的Linux系统，这个微小系统里面有启动、停止、删除这些基本命令
       </p>
       <p style="margin-left:0; margin-right:0">
        仓库(repository):
       </p>
       <p style="margin-left:0; margin-right:0">
        存放镜像的地方，分为公用仓库与私有仓库
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        安装Docker
       </p>
       <p style="margin-left:0; margin-right:0">
        需要先安装一个yum install -y yum-utils
       </p>
       <p style="margin-left:0; margin-right:0">
        配置docker的yum源
       </p>
       <p style="margin-left:0; margin-right:0">
        yum-config-manager --add-repo
        <a href="https://download.docker.com/linux/centos/docker-ce.repo" rel="nofollow" title="https://download.docker.com/linux/centos/docker-ce.repo">
         https://download.docker.com/linux/centos/docker-ce.repo
        </a>
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        安装docker命令
       </p>
       <p style="margin-left:0; margin-right:0">
        yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        查看docker版本 docker -v
       </p>
       <p style="margin-left:0; margin-right:0">
        启动docker：systemctl start docker
       </p>
       <p style="margin-left:0; margin-right:0">
        停止docker：systemctl stop docker
       </p>
       <p style="margin-left:0; margin-right:0">
        重启：systemctl restart docker
       </p>
       <p style="margin-left:0; margin-right:0">
        开机制动启动：systemctl enable docker
       </p>
       <p style="margin-left:0; margin-right:0">
       </p>
       <p style="margin-left:0; margin-right:0">
        配置镜像加速
       </p>
       <p style="margin-left:0; margin-right:0">
        <img alt="" height="303" src="https://i-blog.csdnimg.cn/direct/07a9bb27f476438bbfa50da7502035a0.jpeg" width="579"/>
       </p>
       <p>
        <span style="color:#2f2f2f">
        </span>
       </p>
       <p style="margin-left:0; margin-right:0">
        sudo mkdir -p /etc/docker sudo tee /etc/docker/daemon.json &lt;&lt;-'EOF' { "registry-mirrors": ["
        <a href="https://cenb7bp7.mirror.aliyuncs.com" rel="nofollow" title="https://cenb7bp7.mirror.aliyuncs.com">
         https://cenb7bp7.mirror.aliyuncs.com
        </a>
        "] } EOF sudo systemctl daemon-reload sudo systemctl restart docker
