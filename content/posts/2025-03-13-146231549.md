# idea cpu干到100%的解决方法?

一、环境信息

idea版本: IntelliJ IDEA 2024.1.7 (Ultimate Edition)

jdk版本: 1.8

操作系统版本: win10

二、解决办法

Help >> Change Memory Settings设置成2048M后重启idea

三、说明

idea将cpu打满后电脑会相当卡顿，Change Memory Settings后idea内存占用也会很高(16G内存环境下保证至少有3G可用空间)，但不至于让电脑卡顿，cpu消耗降下去电脑即可流畅运行。
