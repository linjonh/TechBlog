    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:yellow">
      <span style="color:black">
       考点25 基于ARM内核的嵌入式芯片的硬件组成
      </span>
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      1.
     </span>
     <span style="color:black">
      按照AMBA总线规范，以ARM内核为基础的嵌入式处理芯片，采用系统总线（AHB）与外围总线（APB）两层结构的方式构建片上系统，系统总线主要用于连接高带宽快速组件，而外围总线支持的内置外设不断增加，性能不断增强。它主要连接低带宽组件以及与外部相连的硬件组件。示意图如下：
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <img alt="" height="288" src="https://i-blog.csdnimg.cn/direct/2805df8ed30347099349dd394af8a317.png" width="710"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      2.
     </span>
     <span style="color:black">
      存储器及控制器
     </span>
    </p>
    <ol>
     <li style="text-align:left">
      <span style="color:black">
       ARM
      </span>
      <span style="color:black">
       处理芯片内部硬件中，除了ARM内核外，最重要的组件就是存储器及其管理组件，用于管理和控制片内的SRAM、ROM和Flash ROM。通过外部存储器控制器对外部扩展存储器，如Flash存储器及DRAM等进行管理与控制。
      </span>
     </li>
     <li style="text-align:left">
      <span style="color:red">
       片内程序存储器通常用的是Flash ROM
      </span>
      <span style="color:black">
       ，
      </span>
      <span style="color:red">
       片内数据存储器通常使用SRAM
      </span>
      <span style="color:black">
       。
      </span>
     </li>
     <li style="text-align:left">
      <span style="color:black">
       目前，程序存储器大都采用Flash存储器，数据存储器可采用SRAM和DDR或DDR2或普通的DRAM。
      </span>
     </li>
     <li style="text-align:left">
      <span style="color:black">
       高带宽外部存储器控制接口为外部存储器扩展提供了接口，可扩展
      </span>
      <span style="color:red">
       程序存储器及数据存储器
      </span>
      <span style="color:black">
       。高带宽外部存储器控制接口与AMBA的系统总线（AHB）部分相连。
      </span>
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      3.
     </span>
     <span style="color:black">
      中断控制器：
     </span>
    </p>
    <ol>
     <li style="text-align:left">
      <span style="color:black">
       中断控制器负责对其他硬件组件的中断请求进行管理和控制，一般采用向量中断（VIC）或嵌套向量中断（NVIC）的方式管理中断。处理中断有两种形式，分别是标准的中断控制器和向量中断控制器（VIC）。
      </span>
     </li>
     <li style="text-align:left">
      <span style="color:red">
       标准中断控制器
      </span>
      <span style="color:black">
       在一个外设的设备需要服务时，发送一个中断请求信号给处理器核。中断处理程序通过读取中断控制器中各设备对应表示中断请求的寄存器内容，从而判断哪个设备需要服务。
      </span>
     </li>
     <li style="text-align:left">
      <span style="color:red">
       向量中断控制器（VIC）
      </span>
      比标准中断控制器的功能更强大，它区分中断的
      <span style="color:red">
       优先级
      </span>
      ，简化了判断中断源的过程。只有当一个新的中断，其优先级高于当前正在执行的中断处理优先级时，VIC才向内核提出中断请求。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       嵌套向量中断（NVIC）
      </span>
      比VIC更进一步，可以进行中断的嵌套，即高优先级的中断可以进入低优先级中断的处理过程中。待高优先级中断处理完成之后，才去继续执行低优先级中断，也有人称为
      <span style="color:red">
       抢占式优先级中断
      </span>
      。Cortex-m系列就支持嵌套的向量诊断。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      4.
     </span>
     DMA控制器
    </p>
    <ol>
     <li style="text-align:left">
      ARM处理芯片内部的DMA控制器（即直接存储器访问控制器）是一种硬件组件，
      <span style="color:red">
       可将数据块从外设传输至内存，从内存传输至外设或从内存传输至内存
      </span>
      。
     </li>
     <li style="text-align:left">
      数据传输过程中不需要CPU参与，因此可降低处理器的负荷，通过将CPU设为低功率状态，并使用DMA控制器传输数据，降低了系统的功耗。
     </li>
     <li style="text-align:left">
      当ARM与串行通信端、USB接口、CAN接口、以太网接口等通道进行交流时，
      <span style="color:red">
       他们既可以由ARM内核控制其数据传输，也可以通过DMA控制器控制数据传输
      </span>
      。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      5.
     </span>
     电源管理与时钟控制器
    </p>
    <ol>
     <li style="text-align:left">
      ARM处理芯片内部的
      <span style="color:red">
       电源管理
      </span>
      主要有正常工作模式，慢时钟模式，空闲模式，掉电模式，休眠模式，深度休眠模式等，以控制不同的组件的功耗。
     </li>
     <li style="text-align:left">
      时钟信号是ARM芯片定时的关键，
      <span style="color:red">
       时钟控制器
      </span>
      负责对时钟的分配，产生不同频率的定时时钟供片内各组件作为同步时钟使用，例如。有供快速通道的存储器时钟、供DMA控制器及中断控制器的时钟，也有经过桥接器之后，经若干分频得到了慢速时钟，供APB总线上的各个接口作为同步信号。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      6.
     </span>
     GPIO：即通用输入输出端口。
     <span style="color:red">
      作为输入时具有缓冲功能，作为输出时具有锁存功能
     </span>
     ，也可以作为双向I/O使用。在ARM处理芯片中，GPIO引脚通常是多功能使用的，目的是为了减少芯片引脚数，缩小PCB面积，以减少功耗。有的引脚是双功能的，也有三功能甚至四功能的引脚。不同厂家的ARM处理芯片及具体引脚的定义不同。
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      7.
     </span>
     定时计数组件：
    </p>
    <ol>
     <li style="text-align:left">
      <span style="color:red">
       看门狗定时器（WDT）
      </span>
      是能使系统定时复位的硬件或软件。它的主要功能是当处理器在进入错误状态后的一定时间内复位，保证系统的稳定运行。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       Timer
      </span>
      <span style="color:red">
       通用定时器
      </span>
      ，可用于一般的定时。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       RTC
      </span>
      <span style="color:red">
       可直接提供年月日时分秒
      </span>
      ，使应用系统具有自己独立的时日期和时间。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       脉冲宽度调制器（PWM）
      </span>
      用于脉冲宽度的调制，比如电机控制、用于变频调整等多种场合。
     </li>
     <li style="text-align:left">
      以上所有与定时有关的组件，有一个共同特点，就是对特定输入的始终通过分频后接入计数器进行加1或减1计数，计数达到预定的数值后，将引发一个中断并置一定的标志位。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      8.
     </span>
     模拟通道组件：
    </p>
    <ol>
     <li style="text-align:left">
      <span style="color:red">
       ADC
      </span>
      <span style="color:red">
       是模拟到数字的转换器，可完成由模拟信号到数字信号的转换
      </span>
      ，对于嵌入式系统而言，它是一个模拟输入的组件，ARM芯片内部只有一个AD变换单元，分辨率一般为10位和12位居多。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       DAC
      </span>
      <span style="color:red">
       是数字到模拟的转换器，可完成由数字信号到模拟信号的转换。
      </span>
      对于嵌入式系统而言，是一个模拟输出组件，一般内置的DAC有10位，12位，14位等分辨率。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       比较器：
      </span>
      可方便地
      <span style="color:red">
       对模拟电压信号等与基准信号相比较
      </span>
      ，可监测电源电压，芯片温度等，通过比较采样值与基准值就可以判断是否欠压，温度是否超高等。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      9.
     </span>
     互联通信组件
    </p>
    <ol>
     <li style="text-align:left">
      <span style="color:red">
       .通用异步收发器（UART）
      </span>
      ：标准的
      <span style="color:red">
       串行通信接口
      </span>
      ，字符格式按照低位在前，高位在后的次序进行传输，1位起始位，5-8位数据位，1位奇偶校验位，1-2位停止位。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       .I
      </span>
      <span style="color:red">
       ²C
      </span>
      是集成电路互联的一种总线标准，只有
      <span style="color:red">
       两个信号线
      </span>
      ，一根是
      <span style="color:red">
       时钟线SCL
      </span>
      ，一根是
      <span style="color:red">
       数据线SDA（双向三态）
      </span>
      ，即可完成数据的传输操作。具有特定的起始位和终止位，可完成
      <span style="background-color:yellow">
       <span style="color:red">
        同步半双工串行通信技术
       </span>
      </span>
      。常用于板级芯片之间的短距离低速通信。
     </li>
    </ol>
    <ol>
     <li style="text-align:left">
      总线上所有器件都不使用总线时（
      <span style="color:red">
       总线空闲
      </span>
      ），SCL线和SDA线各自的
      <span style="color:red">
       上拉电阻
      </span>
      ，把电平拉高，使他们
      <span style="color:red">
       均处于高电平
      </span>
      。
     </li>
     <li style="text-align:left">
      数据传送格式，数据传送是高位在前，低位在后，每次传送的
      <span style="color:red">
       字节数目没有限制
      </span>
      ，传输操作启动后，主控器件的传输的第一字节是地址。
     </li>
     <li style="text-align:left">
      应答（ACK）信号传送。为了完成一字节的传送，接收方应该发送一个确认信号ACK给发送，ACK信号出现在SCL线的第九个时钟脉冲上，有效应答ACK在SDA上呈现低电平。
     </li>
     <li style="text-align:left">
      读写操作：在发送模式下，数据被发送出去后，I²C接口处于等待状态，SCL线保持低电平，直到有新的数据写入I²C数据发送寄存器后，SCL线才被释放，继续发送数据；在接收模式下，I²C接口接收到数据后，处于等待状态，直到数据接收寄存器内容被读取后，SCL线才被释放，继续继续传输数据。
     </li>
     <li style="text-align:left">
      总线制裁：I²C总线属于
      <span style="color:red">
       多主总线
      </span>
      ，即允许总线上有一个或多个主控器件或若干从器件同时进行操作，总线上连接的器件有时同时竞争总线的控制权，这就需要仲裁。
     </li>
     <li style="text-align:left">
      异常中断条件，如果没有一个从器件对主控器件发出的地址进行确认，那么SDA线保持高电平。如果主控器件涉入异常中断，则从器件接收到最后一个数据字节后，主器件将通过取消一个ACK信号的产生来通知从器件传送操作结束。
     </li>
     <li style="text-align:left">
      总线接口的连接：ARM芯片内部集成了I²C总线接口，因此可以将基于I²C总线的主控器件或被控器件挂接到I²C总线上，每个器件的I²C总线信号SCL和SDA与其他具有I²C总线的处理器或设备
      <span style="color:red">
       同名端相连
      </span>
      ，在SCL和SDA线上要接上拉电阻。在ARM芯片中内置I²C总线控制器，I²C总线在主器件和从器件之间进行数据传输之前，必须根据要求设置相应的I²C有关功能寄存器，包括
      <span style="color:red">
       I
      </span>
      <span style="color:red">
       ²C总线控制寄存器，I²C总线状态寄存器，I²C总线地址寄存器，I²C总线接收/发送数据移位寄存器
      </span>
      等。
     </li>
     <li style="text-align:left">
      <span style="color:red">
       I
      </span>
      <span style="color:red">
       ²S
      </span>
      是一种面向多媒体应用的
      <span style="color:red">
       音频串行总线
      </span>
      ，是SONY、PHILIPS等公司共同推出的接口标准。主要针对数字音频设备如便携CD机、数字音频处理器等，专用于这些音频设备之间的数据传输。
     </li>
    </ol>
    <p>
     3..
     <span style="color:red">
      I²S
     </span>
     是一种面向多媒体应用的
     <span style="color:red">
      音频串行总线
     </span>
     ，是SONY、PHILIPS等公司共同推出的接口标准。主要针对数字音频设备如便携CD机、数字音频处理器等，专用于这些音频设备之间的数据传输。     4..
     <span style="color:red">
      SPI
     </span>
     是串行外设接口总线系统，是一种
     <span style="color:red">
      同步串行外设接口
     </span>
     ，可以使MCU与各种外围设备以串行方式进行通信。通常用四线制，包括
     <span style="color:red">
      MISO
     </span>
     <span style="color:red">
      （主输入从输出）、MOSI（主输出从输入）、SSL（芯片选择）和SCK（时钟）
     </span>
     。可完成
     <span style="background-color:yellow">
      <span style="color:red">
       全双工的同步串行通信
      </span>
     </span>
     ，用于
     <span style="color:red">
      板级芯片之间的短距离通信
     </span>
     。
    </p>
    <p style="text-align:left">
     <span style="color:red">
      5..CAN
     </span>
     是控制器局域网，仅有
     <span style="color:red">
      CANH
     </span>
     <span style="color:red">
      和CANL两根信号线
     </span>
     ，采用
     <span style="color:red">
      差分传输的方式
     </span>
     ，可以进行远距离（1200m）多机通信。主要用于要求抗干扰能力强的工业控制领域，可组成多主多从系统。
    </p>
    <p style="text-align:left">
     <span style="color:red">
      6..USB
     </span>
     是一种
     <span style="color:red">
      通用串行总线
     </span>
     ，主要用于与外设设备的短距离通信，采用
     <span style="color:red">
      差分方式
     </span>
     传输数据，速度快，效率高，是目前
     <span style="color:red">
      运用最广的串行总线接口形式
     </span>
     。
    </p>
    <p style="text-align:left">
     <span style="color:red">
      7..Ethernet
     </span>
     是以太网通信接口，在许多新型ARM芯片（如Cortex-m3系列）中均集成了这些接口，使连接以太网变得非常容易，外部仅需要连接一个RJ-45连接器即可。
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
    </p>
    <p>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="background-color:yellow">
      考点26存储器分类
     </span>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      1.
     </span>
     分类：
    </p>
    <ol>
     <li style="text-align:left">
      按照其存取特性，可分为
      <span style="color:red">
       随机存取存储器RAM
      </span>
      和
      <span style="color:red">
       只读存储器ROM
      </span>
      。
     </li>
     <li style="text-align:left">
      按照所处的物理位置，可以分为
      <span style="color:red">
       片内存储器
      </span>
      （芯片内置的存储器）和
      <span style="color:red">
       片外存储器
      </span>
      （外部扩展的存储器以及外部存储设备）
     </li>
     <li style="text-align:left">
      在嵌入式系统中，把
      <span style="color:red">
       片内存储器及片外扩展的存储器简称为主存储器或主存
      </span>
      ，把
      <span style="color:red">
       外部存储器设备简称为辅助存储器
      </span>
      。
     </li>
     <li style="text-align:left">
      按照存储信息的不同，可以分为
      <span style="color:red">
       程序存储器和数据存储器
      </span>
      。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      2.
     </span>
     主存种类有：半导体存储器、新型存储器。
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <img alt="" height="229" src="https://i-blog.csdnimg.cn/direct/d72b50400bf64ce7bbc9978a14eb8d53.png" width="321">
      <img alt="" height="232" src="https://i-blog.csdnimg.cn/direct/211372787eaa49d2b1612a2fb5b7449f.png" width="249"/>
     </img>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      3.
     </span>
     随机存取存储器RAM
    </p>
    <ol>
     <li style="text-align:left">
      随机存取存储器有两种形式，分为静态SRAM和动态DRAM，它们均为
      <span style="color:red">
       易失性存储器
      </span>
      。现在多数嵌入式处理器
      <span style="color:red">
       内嵌的数据存储器都采用SRAM
      </span>
      ，而
      <span style="color:red">
       外部扩展的存储器都采用DRAM及其改进型
      </span>
      。
     </li>
     <li style="text-align:left">
      DRAM开发了许多新品种，主要包括如下几种:
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <img alt="" height="261" src="https://i-blog.csdnimg.cn/direct/a9e7b39c2ede4896b78b90970cd22a4b.png" width="630"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      4.
     </span>
     只读存储器ROM
    </p>
    <ol>
     <li style="text-align:left">
      MROM是主要用于
      <span style="color:red">
       不可升级的成熟产品存储程序或不变的参数
      </span>
      等信息的只读存储器。
     </li>
     <li style="text-align:left">
      PROM是一次可编程的只读存储器，
      <span style="color:red">
       只能一次编程，一次编程完就无法修改
      </span>
      。
     </li>
     <li style="text-align:left">
      EPROM是
      <span style="color:red">
       紫外线可擦除可编程的只读存储器
      </span>
      ，擦除编程次数十万次以内，编程速度慢，擦除时间长。
     </li>
     <li style="text-align:left">
      EEPROM是一种
      <span style="color:red">
       电可擦除可编程的只读存储器
      </span>
      ，可以在线改写和擦除信息，无需紫外线照射，EEPROM通常写为E²PROM。
     </li>
     <li style="text-align:left">
      Flash ROM是近些年
      <span style="color:red">
       应用最广，速度最快的只读存储器
      </span>
      ，原理是从E²PROM技术上改进发展而来，特点是
      <span style="color:red">
       擦除和编写速度快
      </span>
      ，因此得名为闪存（闪速存储器）。
     </li>
     <li style="text-align:left">
      NOR Flash和NAND Flash是现在市场上主要的闪存技术。前者特点是
      <span style="color:red">
       以字节为单位随机存取
      </span>
      ，所以应用程序可以直接在Flash ROM上执行，不必把程序预先读入到RAM中；后者
      <span style="color:red">
       以页（行）为单位随机存取
      </span>
      ，对比前者在容量、使用寿命和成本上有优势，但读取速度慢，因此大多作为数据存储器使用，如数码相机、MP3随身听记忆卡、U盘等。
     </li>
    </ol>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#fe2c24">
      5.
     </span>
     铁电随机存取存储器（FRAM）具有只读存储器的
     <span style="color:red">
      非易失性
     </span>
     ，又具有随机存储器的
     <span style="color:red">
      可快速随机读写
     </span>
     的特点，而且速度快，功耗低。
    </p>
    <p style="margin-left:0; margin-right:0; text-align:left">
     6.磁性随机存取存储器（MRAM）是一种非易失性的磁性随机存储器，具有静态随机存储器SRAM的
     <span style="color:red">
      高速存取特点
     </span>
     以及动态随机存储器DRAM的
     <span style="color:red">
      高集成度
     </span>
     ，因此可以基本上无限次的重复写入。
    </p>
