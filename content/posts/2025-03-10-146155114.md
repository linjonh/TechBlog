    <h3>
     <a id="1__EPEL__0">
     </a>
     1. 安装 EPEL 仓库（如果尚未安装）
    </h3>
    <p>
     EPEL（Extra Packages for Enterprise Linux）仓库提供了许多额外的软件包，Redis 就在其中。执行以下命令安装 EPEL 仓库：
    </p>
    <pre><code class="prism language-java">sudo yum install <span class="token operator">-</span>y epel<span class="token operator">-</span>release
</code></pre>
    <h3>
     <a id="2__Redis_6">
     </a>
     2. 安装 Redis
    </h3>
    <p>
     启用 EPEL 仓库后，使用 yum 命令来安装 Redis：
    </p>
    <pre><code class="prism language-java">sudo yum install <span class="token operator">-</span>y redis
</code></pre>
    <h3>
     <a id="3__Redis__12">
     </a>
     3. 启动 Redis 服务
    </h3>
    <pre><code class="prism language-java">sudo systemctl start redis
</code></pre>
    <h3>
     <a id="4__Redis__17">
     </a>
     4. 设置 Redis 开机自启
    </h3>
    <pre><code class="prism language-java">sudo systemctl enable redis
</code></pre>
    <h3>
     <a id="_22">
     </a>
     手动建立软链接（可选）
    </h3>
    <p>
     如果你希望通过 redis 命令来启动 Redis 服务，可以手动建立软链接。假设使用 yum 安装后 Redis 的可执行文件路径为 /usr/bin/redis-server，可以执行以下命令建立软链接：
    </p>
    <pre><code class="prism language-java">sudo ln <span class="token operator">-</span>s <span class="token operator">/</span>usr<span class="token operator">/</span>bin<span class="token operator">/</span>redis<span class="token operator">-</span>server <span class="token operator">/</span>usr<span class="token operator">/</span>local<span class="token operator">/</span>bin<span class="token operator">/</span>redis
</code></pre>
    <p>
     这样，在任意目录下输入 redis 命令就可以启动 Redis 服务。
    </p>
    <h3>
     <a id="_Redis__30">
     </a>
     验证 Redis 安装和启动
    </h3>
    <p>
     使用 redis-cli 工具连接到 Redis 服务器进行验证。如果没有设置密码，直接执行：
    </p>
    <pre><code class="prism language-java">redis<span class="token operator">-</span>cli ping
</code></pre>
    <p>
     若输出 PONG，则表示 Redis 服务已成功启动并正常工作。如果设置了密码，需要使用 -a 选项指定密码：
    </p>
    <pre><code class="prism language-java">redis<span class="token operator">-</span>cli <span class="token operator">-</span>a yourpassword ping
</code></pre>
