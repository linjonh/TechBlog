    <p>
     好的！我用一个简单的数值例子来说明开环和闭环控制在LQR中的应用，数据尽量简化。
    </p>
    <hr/>
    <h4>
     <a id="_4">
     </a>
     <strong>
      场景
     </strong>
    </h4>
    <p>
     假设一辆小车的
     <strong>
      目标速度
     </strong>
     是
     <strong>
      60 km/h
     </strong>
     ，初始速度是
     <strong>
      0 km/h
     </strong>
     ，控制周期为
     <strong>
      1秒/次
     </strong>
     （每秒调整一次油门）。
    </p>
    <hr/>
    <h4>
     <a id="1__9">
     </a>
     <strong>
      1. 开环控制（无反馈）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        控制策略
       </strong>
       ：提前计划好油门的固定操作，不根据实际速度调整。
      </p>
     </li>
     <li>
      <p>
       <strong>
        数据例子
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         计划
        </strong>
        ：前3秒油门踩
        <strong>
         50%
        </strong>
        ，后3秒油门踩
        <strong>
         30%
        </strong>
        （总时间6秒）。
       </li>
       <li>
        <strong>
         实际效果
        </strong>
        （假设每1秒油门50%能加速20 km/h，油门30%加速12 km/h）：
        <ul>
         <li>
          第1秒后：速度
          <strong>
           0 + 20 = 20 km/h
          </strong>
         </li>
         <li>
          第2秒后：速度
          <strong>
           20 + 20 = 40 km/h
          </strong>
         </li>
         <li>
          第3秒后：速度
          <strong>
           40 + 20 = 60 km/h
          </strong>
          （目标达成！）
         </li>
         <li>
          第4秒后：油门降到30%，速度
          <strong>
           60 + 12 = 72 km/h
          </strong>
          （超速！）
         </li>
         <li>
          第5秒后：速度
          <strong>
           72 + 12 = 84 km/h
          </strong>
         </li>
         <li>
          第6秒后：速度
          <strong>
           84 + 12 = 96 km/h
          </strong>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        问题
       </strong>
       ：
      </p>
      <ul>
       <li>
        实际中可能有风阻或坡度干扰，比如每1秒实际加速比理论少
        <strong>
         5 km/h
        </strong>
        ：
        <ul>
         <li>
          第3秒实际速度：
          <strong>
           20 + 15 + 15 = 50 km/h
          </strong>
          （未达到目标60 km/h）。
         </li>
        </ul>
       </li>
       <li>
        但开环控制仍按原计划执行，导致结果偏离目标。
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2_LQR_28">
     </a>
     <strong>
      2. 闭环控制（LQR反馈）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        控制策略
       </strong>
       ：根据当前速度与目标的误差，实时计算油门。
      </p>
     </li>
     <li>
      <p>
       <strong>
        LQR设计
       </strong>
       ：假设反馈矩阵
       <strong>
        K = 0.4
       </strong>
       ，控制规则为：
       <br/>
       <strong>
        油门增量 = K × (目标速度 - 当前速度)
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        数据例子
       </strong>
       （考虑风阻，每秒实际加速比理论少5 km/h）：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          目标速度
         </strong>
         ：60 km/h
        </p>
       </li>
       <li>
        <p>
         <strong>
          初始速度
         </strong>
         ：0 km/h
        </p>
       </li>
       <li>
        <p>
         <strong>
          第1秒
         </strong>
         ：
        </p>
        <ul>
         <li>
          误差 = 60 - 0 = 60 km/h
         </li>
         <li>
          油门增量 = 0.4 × 60 =
          <strong>
           24%
          </strong>
         </li>
         <li>
          理论加速 = 24% × 0.5（系数） = 12 km/h
         </li>
         <li>
          实际加速 = 12 - 5 = 7 km/h
         </li>
         <li>
          当前速度 = 0 + 7 =
          <strong>
           7 km/h
          </strong>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          第2秒
         </strong>
         ：
        </p>
        <ul>
         <li>
          误差 = 60 - 7 = 53 km/h
         </li>
         <li>
          油门增量 = 0.4 × 53 =
          <strong>
           21.2%
          </strong>
         </li>
         <li>
          理论加速 = 21.2% × 0.5 = 10.6 km/h
         </li>
         <li>
          实际加速 = 10.6 - 5 = 5.6 km/h
         </li>
         <li>
          当前速度 = 7 + 5.6 =
          <strong>
           12.6 km/h
          </strong>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          第3秒
         </strong>
         ：
        </p>
        <ul>
         <li>
          误差 = 60 - 12.6 = 47.4 km/h
         </li>
         <li>
          油门增量 = 0.4 × 47.4 ≈
          <strong>
           19%
          </strong>
         </li>
         <li>
          理论加速 = 19% × 0.5 ≈ 9.5 km/h
         </li>
         <li>
          实际加速 = 9.5 - 5 = 4.5 km/h
         </li>
         <li>
          当前速度 = 12.6 + 4.5 ≈
          <strong>
           17.1 km/h
          </strong>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          持续调整
         </strong>
         ：误差逐渐减小，油门百分比也逐步降低，最终逼近目标速度60 km/h。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_62">
     </a>
     <strong>
      对比总结
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       开环控制
      </strong>
      ：油门固定，
      <strong>
       理论完美但实际翻车
      </strong>
      （要么超速，要么未达标）。
     </li>
     <li>
      <strong>
       闭环控制（LQR）
      </strong>
      ：实时根据误差调整油门，
      <strong>
       动态逼近目标
      </strong>
      ，抗干扰能力强。
     </li>
    </ul>
    <p>
     闭环控制就像“边开车边看车速表，随时踩或松油门”；开环则是“蒙眼踩油门，赌它能达到目标”。
    </p>
