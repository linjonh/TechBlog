    <h4>
     <a id="_0">
     </a>
     <strong>
      一、镜像管理
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        拉取镜像
       </strong>
       <br/>
       <code>
        docker pull [镜像名]:[标签]
       </code>
       <br/>
       示例：
       <code>
        docker pull nginx:latest
       </code>
       <br/>
       （记忆：pull=拉取，类似git拉取代码）
      </p>
     </li>
     <li>
      <p>
       <strong>
        构建镜像
       </strong>
       <br/>
       <code>
        docker build -t [镜像名]:[标签] .
       </code>
       <br/>
       示例：
       <code>
        docker build -t myapp:v1 .
       </code>
       <br/>
       （-t指定标签，
       <code>
        .
       </code>
       表示当前目录的Dockerfile）
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看本地镜像
       </strong>
       <br/>
       <code>
        docker images
       </code>
       <br/>
       （直接列出所有镜像，可加
       <code>
        --filter
       </code>
       筛选）
      </p>
     </li>
     <li>
      <p>
       <strong>
        删除镜像
       </strong>
       <br/>
       <code>
        docker rmi [镜像ID/名称]
       </code>
       <br/>
       示例：
       <code>
        docker rmi myapp:v1
       </code>
       <br/>
       （rmi=remove image）
      </p>
     </li>
     <li>
      <p>
       <strong>
        标记镜像
       </strong>
       <br/>
       <code>
        docker tag [原镜像] [新镜像名]:[标签]
       </code>
       <br/>
       示例：
       <code>
        docker tag ubuntu:latest myubuntu:1.0
       </code>
       <br/>
       （类似给镜像重命名）
      </p>
     </li>
    </ol>
    <h4>
     <a id="_26">
     </a>
     <strong>
      二、容器管理
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        运行容器
       </strong>
       <br/>
       <code>
        docker run [选项] [镜像]
       </code>
       <br/>
       示例：
       <code>
        docker run -d -p 80:80 --name web nginx
       </code>
       <br/>
       （-d后台运行，-p映射端口，–name指定容器名）
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看容器状态
       </strong>
       <br/>
       <code>
        docker ps
       </code>
       查看运行中的容器，
       <code>
        docker ps -a
       </code>
       查看所有容器。
       <br/>
       （类似Linux的ps命令）
      </p>
     </li>
     <li>
      <p>
       <strong>
        启动/停止/重启容器
       </strong>
       <br/>
       <code>
        docker start/stop/restart [容器ID/名称]
       </code>
       <br/>
       示例：
       <code>
        docker restart web
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        进入容器
       </strong>
       <br/>
       <code>
        docker exec -it [容器ID] [命令]
       </code>
       <br/>
       示例：
       <code>
        docker exec -it web bash
       </code>
       <br/>
       （-it开启交互模式，可直接操作容器内系统）
      </p>
     </li>
     <li>
      <p>
       <strong>
        删除容器
       </strong>
       <br/>
       <code>
        docker rm [容器ID/名称]
       </code>
       <br/>
       示例：
       <code>
        docker rm web
       </code>
       <br/>
       （需先停止容器，加
       <code>
        -f
       </code>
       强制删除）
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看日志
       </strong>
       <br/>
       <code>
        docker logs [容器ID]
       </code>
       <br/>
       示例：
       <code>
        docker logs -f web
       </code>
       <br/>
       （-f实时跟踪日志输出）
      </p>
     </li>
    </ol>
    <h4>
     <a id="_56">
     </a>
     <strong>
      三、网络配置
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        创建自定义网络
       </strong>
       <br/>
       <code>
        docker network create [网络名]
       </code>
       <br/>
       示例：
       <code>
        docker network create mynet
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看网络列表
       </strong>
       <br/>
       <code>
        docker network ls
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        连接容器到网络
       </strong>
       <br/>
       <code>
        docker network connect [网络名] [容器ID]
       </code>
       <br/>
       示例：
       <code>
        docker network connect mynet web
       </code>
      </p>
     </li>
    </ol>
    <h4>
     <a id="_69">
     </a>
     <strong>
      四、数据卷管理
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        创建数据卷
       </strong>
       <br/>
       <code>
        docker volume create [卷名]
       </code>
       <br/>
       示例：
       <code>
        docker volume create myvol
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>
        挂载数据卷
       </strong>
       <br/>
       <code>
        docker run -v [卷名]:[容器路径]
       </code>
       <br/>
       示例：
       <code>
        docker run -v myvol:/app nginx
       </code>
       <br/>
       （-v指定卷挂载，容器内数据持久化）
      </p>
     </li>
    </ol>
    <h4>
     <a id="_80">
     </a>
     <strong>
      五、系统管理
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        查看系统信息
       </strong>
       <br/>
       <code>
        docker info
       </code>
       <br/>
       （显示Docker版本、存储驱动、内核等）
      </p>
     </li>
     <li>
      <p>
       <strong>
        清理无用资源
       </strong>
       <br/>
       <code>
        docker system prune
       </code>
       <br/>
       示例：
       <code>
        docker system prune -a
       </code>
       <br/>
       （-a清理所有未使用的镜像和卷）
      </p>
     </li>
    </ol>
    <h4>
     <a id="_91">
     </a>
     <strong>
      六、进阶命令
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Docker Compose
      </strong>
      <br/>
      <code>
       docker-compose up -d
      </code>
      启动多容器服务，
      <code>
       docker-compose down
      </code>
      停止并清理。
      <br/>
      （用于复杂项目编排）
     </li>
    </ul>
