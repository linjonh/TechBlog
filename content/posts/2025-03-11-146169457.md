     </span>
    </p>
    <h4>
     2.MySQL压缩包
    </h4>
    <p>
     压缩包的安装方式也相对简单。
    </p>
    <div>
     <span style="color:#000000">
      1.
     </span>
     <span style="color:#000000">
      解压缩后位置：
     </span>
     <span style="color:#000000">
      D:\mysql-8.0.15-winx64
     </span>
    </div>
    <div>
     <span style="color:#000000">
      2.
     </span>
     <span style="color:#000000">
      在主目录下复制、创建一个
     </span>
     <span style="color:#000000">
      xx.ini
     </span>
     <span style="color:#000000">
      ，修改为
     </span>
     <span style="color:#c9ae75">
      my.ini
     </span>
     <span style="color:#000000">
      ，修改其中的
     </span>
     <span style="color:#c9ae75">
      basedir
     </span>
     <span style="color:#000000">
      和
     </span>
     <span style="color:#c9ae75">
      datadir
     </span>
    </div>
    <div>
     配置文件xx.ini的主要内容（注意：其中的
     <span style="color:#c9ae75">
      basedir
     </span>
     和
     <span style="color:#c9ae75">
      datadir
     </span>
     修改成自己的路径即可，如果压缩包里没有data文件夹，自己创建，并保证初始化时，该文件夹为空）
    </div>
    <div>
    </div>
    <div>
     <pre><code class="hljs">[mysqld]
# skip_grant_tables
# 设置3306端口
port=3306
# 设置mysql的安装目录
basedir=D:\dev_soft\mysql-8.0.20-winx64
# 设置mysql数据库的数据的存放目录
datadir=D:\dev_soft\mysql-8.0.20-winx64\data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统max_connect_errors=10
# 服务端使用的字符集默认为UTF8
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎default-storage-engine=INNODB
[client]
#password = your_password
# 设置mysql客户端连接服务端时默认使用的端口port=3306
default-character-set=utf8mb4
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4</code></pre>
     <div>
      <span style="color:#000000">
       配置
      </span>
      <span style="color:#000000">
       bin目录为环境变量（方便在终端情况下，查找
      </span>
      <span style="color:#000000">
       MySQL
      </span>
      <span style="color:#000000">
       的相关命令）
      </span>
     </div>
     <p>
      鼠标右键windows图标--------&gt;系统---------&gt;高级系统---------&gt;设置环境变量--------&gt;path
     </p>
     <p>
      新建--------&gt;编辑-------&gt;找到MySQLbin路径，粘贴点确认即可。
     </p>
     <p>
      <img alt="" height="1020" src="https://i-blog.csdnimg.cn/direct/c4c8a58ecf2b4891b47f52fb2c3f378b.png" width="1920"/>
     </p>
    </div>
    <p>
     <span style="color:#000000">
      初始化命令（记录控制台给出的随机密码，一定记住）：
     </span>
    </p>
    <div>
     <span style="color:#000000">
      注意：打开
     </span>
     <span style="color:#000000">
      cmd
     </span>
     <span style="color:#000000">
      时，必须使用管理员身份！！！
     </span>
     <span style="color:#000000">
      cmd
     </span>
     <span style="color:#000000">
      打开后，切换到
     </span>
     <span style="color:#000000">
      bin
     </span>
     <span style="color:#000000">
      目录再执行如下命令
     </span>
    </div>
    <pre><code class="hljs"> mysqld --initialize --console</code></pre>
    <div>
     <span style="color:#000000">
      然后先给
     </span>
     <span style="color:#000000">
      mysql
     </span>
     <span style="color:#000000">
      服务创建名称
     </span>
     <span style="color:#000000">
      (
     </span>
     <span style="color:#000000">
      方便到时候建立多个
     </span>
     <span style="color:#000000">
      mysql
     </span>
     <span style="color:#000000">
      服务时不冲突
     </span>
     <span style="color:#000000">
      )
     </span>
    </div>
    <div>
     <span style="color:#000000">
      mysqld --install mysql84
     </span>
    </div>
    <div>
    </div>
    <div>
     <span style="color:#000000">
      之后启动服务
     </span>
    </div>
    <div>
     <pre><code class="hljs">net start mysql


#登陆旧密码登陆（第3步中的密码）
mysql -u root -p </code></pre>
     <p>
      修改密码
     </p>
     <pre><code class="hljs">ALTER USER root@localhost IDENTIFIED  BY '新密码'</code></pre>
     <div>
      <span style="color:#000000">
       卸载
      </span>
      <span style="color:#000000">
       MySQL
      </span>
      <span style="color:#000000">
       ：
      </span>
     </div>
     <div>
      <pre><code class="hljs">#停止服务
net stop mysql
#删除服务
mysqld --remove </code></pre>
      <div>
       <span style="color:#000000">
        登录成功后，执行命令：
       </span>
      </div>
     </div>
     <div>
      <pre><code class="hljs">show databases;</code></pre>
      <p>
       在电脑上如何找到该服务，可以鼠标----&gt;右键windows-------&gt;任务管理器--------&gt;服务------&gt;搜索MySQL就找到该服务，可以进行启动，停止，重启等操作。
      </p>
     </div>
