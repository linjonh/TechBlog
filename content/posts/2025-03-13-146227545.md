# ROS2基础——Linux

Ctrl+Alt+T : 打开一个新终端

1.查看终端目录命令

```bash
$ pwd #查看终端当前目录
---
/home/lvvx
```

2.切换终端目录到根目录

```bash
$ cd / #从当前进入根目录
$ pwd
---
    /
```

3.查看当前目录下文件

```bash
$ ls #查看当前目录下文件
---
bin  dev home lib   lib64  lost+found mnt proc run  snap sys usr
boot ect init lib32 libx32 media      opt root sbin srv  tmp var  
```

4.查看主目录内容

```bash
$ cd ~
$ ls
---
公共的 模板 视频 图片 文档 下载 音乐 桌面 snap
```

5.创建文件夹和文件

```bash
$ cd ~                  #进入主目录
$ mkdir chapt1          # 在主目录下创建chapt1
$ cd chapt1             #从主目录进入chapt1
$ touch hello_world.txt #创建空白文件
$ pwd                   #查看当前路径
--- 
/home/lvvx/chapt1
$ ls                    #查看chapt1目录下所有文件
---
hello_world.txt
```

6.使用nano编辑文件

```bash
$ nano hello_world.txt
```

nano是安装Ubuntu时自带的文本编辑工具，输入 Hello ROS2 ! ,按Ctrl+O键，然后按回车键将内容写入文件，再按Ctrl+X键退出编辑

7.查看文件内容和删除文件

cat查看文件内容，rm命令删除文件

```bash
$ cat hello_world.txt
---
Hello ROS2 !
$rm hello_world.txt
```

8.帮助查看命令使用的命令

```bash
$ ros2 run --help
---
usage:....
```
