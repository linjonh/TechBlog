    <h4 id="1. case class 是什么？">
     1.
     <strong>
      <code>
       case class
      </code>
      是什么？
     </strong>
    </h4>
    <p>
     想象你要做一个表格，比如学生信息表，每一行需要填：
     <strong>
      姓名、年龄、成绩
     </strong>
     。
     <br/>
     在代码里，这种“表格的一行”就是一个数据对象，
     <code>
      case class
     </code>
     就是帮你快速创建这种“表格行”的工具。
    </p>
    <p>
     <strong>
      普通方法（麻烦版）：
     </strong>
     <br/>
     你要自己写很多代码定义“学生”的类，还要写各种方法（比如打印信息、比较是否相同等），麻烦！
    </p>
    <p>
     <strong>
      用
      <code>
       case class
      </code>
      （省事版）：
     </strong>
     <br/>
     你只需要写一行代码，剩下的（打印信息、比较等）自动生成！
    </p>
    <p>
     scala复制代码
    </p>
    <pre><code>// 定义一个“学生”模板（包含姓名、年龄、成绩）
case class Student(name: String, age: Int, score: Double)

// 创建具体的学生对象（不用写 new！）
val alice = Student("Alice", 20, 95.5)  // 自动生成！
val bob = Student("Bob", 22, 88.0)</code></pre>
    <hr/>
    <h4 id="2. case class 有啥好处？">
     2.
     <strong>
      <code>
       case class
      </code>
      有啥好处？
     </strong>
    </h4>
    <h5 id="① 直接拿数据，不用写 new">
     ①
     <strong>
      直接拿数据，不用写
      <code>
       new
      </code>
     </strong>
    </h5>
    <p>
     scala复制代码
    </p>
    <pre><code>// 普通类必须写 new
class Animal(name: String)
val cat = new Animal("Tom")

// case class 不用写 new！自动创建对象
case class Animal(name: String)
val dog = Animal("Buddy")  // 自动帮你 new！</code></pre>
    <h5 id="② 自动生成“打印信息”">
     ②
     <strong>
      自动生成“打印信息”
     </strong>
    </h5>
    <p>
     scala复制代码
    </p>
    <pre><code>// 普通类打印是看不懂的地址
println(cat)  // 输出：Animal@4e50df2e

// case class 打印具体数据
println(dog)  // 输出：Animal(Buddy)</code></pre>
    <h5 id="③ 自动比较内容是否相同">
     ③
     <strong>
      自动比较内容是否相同
     </strong>
    </h5>
    <p>
     scala复制代码
    </p>
    <pre><code>val dog1 = Animal("Buddy")
val dog2 = Animal("Buddy")

// 普通类比较地址，不同
println(cat == new Animal("Tom"))  // false

// case class 比较内容，相同！
println(dog1 == dog2)  // true</code></pre>
    <h5 id="④ 模式匹配超方便（拆开数据用）">
     ④
     <strong>
      模式匹配超方便
     </strong>
     （拆开数据用）
    </h5>
    <p>
     scala复制代码
    </p>
    <pre><code>def checkStudent(student: Student): String = student match {
    case Student("Alice", 20, _) =&gt; "这是Alice，年龄20！"
    case Student(_, age, score) if score &lt; 60 =&gt; "不及格！"
    case _ =&gt; "其他学生"
}

println(checkStudent(alice))  // 输出：这是Alice，年龄20！</code></pre>
    <hr/>
    <h4 id="3. 什么时候用 case class？">
     3.
     <strong>
      什么时候用
      <code>
       case class
      </code>
      ？
     </strong>
    </h4>
    <ul>
     <li>
      当你的类
      <strong>
       主要是用来装数据
      </strong>
      的时候（比如学生信息、订单信息）。
     </li>
     <li>
      当你需要
      <strong>
       快速比较、打印、复制数据
      </strong>
      的时候。
     </li>
     <li>
      当你想用
      <strong>
       模式匹配
      </strong>
      拆解数据的时候。
     </li>
    </ul>
    <hr/>
    <h4 id="4. 超简单总结">
     4.
     <strong>
      超简单总结
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       case class
      </code>
      =
      <strong>
       自动生成的数据盒子
      </strong>
      <br/>
      （帮你省掉写一堆代码的麻烦，还能方便拆开用）
     </li>
     <li>
      常用在：数据存储、模式匹配、传输消息（比如聊天消息）。
     </li>
    </ul>
    <hr/>
    <p>
     <strong>
      动手试试！
     </strong>
     你可以在代码里写一个
     <code>
      case class
     </code>
     ，体验它的方便之处。例如：
    </p>
    <p>
     scala复制代码
    </p>
    <pre><code>case class Book(title: String, price: Double)
val myBook = Book("Scala入门", 39.9)
println(myBook)  // 试试打印结果！</code></pre>
