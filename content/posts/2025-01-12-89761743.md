      <h5>
       如何使用WeUI样式库
      </h5>
      <p>
       WeUI是在遵守微信视觉设计规范下，由官方设计团队为小程序量身定制的基础样式库。
      </p>
      <p>
       WeUI是官方出品，并且在10月28号IDE更新后，IDE不能识别和引用css，同时也屏蔽了从网络获取样式文件，和css划清界限。
      </p>
      <p>
       现在到了重点：
       <strong>
        如何引入WeUI？
       </strong>
      </p>
      <h6>
       1. 下载WeUI
      </h6>
      <p>
       <a href="https://github.com/Tencent/weui-wxss/tree/master/dist/style" rel="nofollow noopener noreferrer" target="_blank">
        weui.wxss
       </a>
      </p>
      <p>
       注意是在【weui-wxss/dist/style/】目录下，不要【weui-wxss/src】目录里面下载
      </p>
      <ul>
       <li>
        组件的wxml结构请看
        <code>
         dist/example/
        </code>
        下的组件
       </li>
       <li>
        样式文件可直接引用
        <code>
         dist/style/weui.wxss
        </code>
        ，或者单独引用
        <code>
         dist/style/widget
        </code>
        下的组件的
        <code>
         wxss
        </code>
        文档
       </li>
      </ul>
      <h6>
       2. 将weui.wxss拷贝到小程序根目录中
      </h6>
      <div class="image-package">
       <div class="image-container">
        <div class="image-container-fill">
        </div>
        <div class="image-view">
         <img class="" src="//upload-images.jianshu.io/upload_images/3409462-19a3cae52784712a.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/602/format/webp"/>
        </div>
       </div>
       <div class="image-caption">
       </div>
      </div>
      <h6>
       3. 在app.wxss或页面wxss导入weui.wxss
      </h6>
      <pre><code class="language-css"><code class="css"><span class="hljs-comment">/**app.wxss**/</span>
@<span class="hljs-keyword">import</span> <span class="hljs-string">'weui.wxss'</span>;
</code></code></pre>
      <h6>
       4. 以上就成功引入了weui.wxss，weui也提供了单个组件的样式引入，流程同上。
      </h6>
      <p>
       地址为：
       <a href="https://github.com/Tencent/weui-wxss/tree/master/dist/style/widget" rel="nofollow noopener noreferrer" target="_blank">
        单个Widget的样式文件
       </a>
      </p>
      <h5>
       如何使用WeUI
      </h5>
      <ul>
       <li>
        根组件使用class="page"
       </li>
      </ul>
      <pre><code class="language-javascript"><code class="javascript">&lt;view <span class="hljs-class"><span class="hljs-keyword">class</span></span>=<span class="hljs-string">"page"</span>&gt;
<span class="xml"><span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span></span>
</code></code></pre>
      <ul>
       <li>
        页面骨架组件使用class="page__xxx"(注意是两个下划线)
       </li>
      </ul>
      <pre><code class="language-javascript"><code class="javascript">&lt;view <span class="hljs-class"><span class="hljs-keyword">class</span></span>=<span class="hljs-string">"page"</span>&gt;
<span class="xml"><span class="hljs-comment">&lt;!--页头--&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"page__hd"</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
<span class="hljs-comment">&lt;!--主体--&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"page__bd"</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
<span class="hljs-comment">&lt;!--没有页脚--&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span></span>
</code></code></pre>
      <ul>
       <li>
        其他组件都已weui-开头后接组件名称，例如class="weui-footer"
       </li>
      </ul>
      <pre class="hljs xml"><code class="xml"><span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"weui-footer"</span>&gt;</span>我是页脚<span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
</code></pre>
      <ul>
       <li>
        组件的子组件样式，例如view.weui-footer组件还有链接和版权信息。
       </li>
      </ul>
      <pre class="hljs xml"><code class="xml"><span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"weui-footer"</span>&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"weui-footer__links"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-name">navigator</span> <span class="hljs-attr">url</span>=<span class="hljs-string">""</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"weui-footer__link"</span>&gt;</span>小黄象<span class="hljs-tag">&lt;/<span class="hljs-name">navigator</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-name">view</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"weui-footer__text"</span>&gt;</span>Copyright © 精品专栏组<span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-name">view</span>&gt;</span>
</code></pre>
      <p>
       组件和子组件使用两个下划线衔接，所以要区分什么时候用"-",什么时候用"__"
      </p>
