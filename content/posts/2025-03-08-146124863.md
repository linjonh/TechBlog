    <h3>
     <a id="4_XML__SQL_79">
     </a>
     4. XML 中使用动态 SQL
    </h3>
    <p>
     在 XML 中，使用
     <code>
      where
     </code>

     <code>

    <h3>
     <a id="_94">
     </a>
     说明
    </h3>
    <ul>
     <li>
      <strong>
       <code>
        @Param("ew")
       </code>
      </strong>
      : 在 Mapper 方法中使用
      <code>
       @Param("ew")
      </code>
      注解来传递
      <code>
       QueryWrapper
      </code>
      ，
      <code>
       ew
      </code>
      是 MyBatis-Plus 约定的参数名。
     </li>
     <li>
      <strong>
       动态 SQL
      </strong>
      : 在 XML 中使用
      <code>
       ${ew.sqlSegment}
      </code>
      来插入
      <code>
       QueryWrapper
      </code>
      构建的 SQL 片段。
     </li>
     <li>
      <strong>
       灵活性
      </strong>
      : 这种方式结合了 XML 的灵活性和
      <code>
       QueryWrapper
      </code>
      的动态条件构建能力。
     </li>
    </ul>
    <p>
     通过这种方式，您可以在 XML 中定义基本的查询逻辑，并在 Java 代码中使用
     <code>
      QueryWrapper
     </code>
     来动态添加查询条件和排序。
    </p>
