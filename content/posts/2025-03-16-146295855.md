    <h2>
     <a id="Kafka_Tool_0">
     </a>
     Kafka Tool工具
    </h2>
    <h3>
     <a id="_2">
     </a>
     介绍
    </h3>
    <p>
     使用Kafka的小伙伴，有没有为无法直观地查看 Kafka 的 Topic 里的内容而发过愁呢？下面推荐给大家一款带有可视化页面的Kafka工具：Kafka Tool （目前最新版本是 3.0.2）
     <br/>
     <mark>
      注意：以前叫Kafka Tool现在官方已更名为Offset Explorer,为了使用习惯本文章仍然使用老叫法。
     </mark>
    </p>
    <h3>
     <a id="Kafka_Toolhttpswwwkafkatoolcomdownloadhtml_6">
     </a>
     <a href="https://www.kafkatool.com/download.html" rel="nofollow">
      Kafka Tool下载地址
     </a>
    </h3>
    <p>
     <mark>
      根据你自己的系统选择对应的软件，博主是W64位所以选择的是Windows 64-Bit版本，下载后双击exe文件一键安装。
     </mark>
     <img alt="下载软件" src="https://i-blog.csdnimg.cn/direct/7e3815e47ee44d72983540a2e6bde0ea.png">
      <br/>
      <img alt="点击安装" src="https://i-blog.csdnimg.cn/direct/306e1da19df546daabe368214161ad37.png"/>
     </img>
    </p>
    <h3>
     <a id="Kafka_Tool_11">
     </a>
     Kafka Tool工具的使用
    </h3>
    <h4>
     <a id="exeKafka_Tool_12">
     </a>
     在安装目录下点击exe启动Kafka Tool
    </h4>
    <p>
     <img alt="启动软件" src="https://i-blog.csdnimg.cn/direct/cbb2b815a1234cf9819ed23ecdc42eee.png"/>
    </p>
    <h4>
     <a id="Kafka_14">
     </a>
     配置Kafka信息
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/882f94aa329a487ea3db0a583f23a10d.png">
      <br/>
      <mark>
       配置你的zk和kafka链接信息，add之前Test一下，确保链接信息没有填写错误
      </mark>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4301b156d097480885ac26ac068be9d9.png"/>
     </img>
    </p>
    <h4>
     <a id="Kafka_18">
     </a>
     查看Kafka消息
    </h4>
    <p>
     选择一个Topic —&gt; Data —&gt;Retrieve Messages，Key/Message设置展示格式，点查询即可看到数据
    </p>
    <p>
     Messages选择Newest，查看最新的Kafka数据；列表每列都可以通过点击表头，进行升序或者降序排序。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/944e2c69b33d43bd8f2004db9bc61f96.png"/>
    </p>

    <h4>
     <a id="Topic_38">
     </a>
     创建Topic
    </h4>
    <p>
     <mark>
      1、右键“Topics” 文件夹 --&gt; 选择 “Create Topic”，或点击“Topics”，单击右侧页面中的“+”，会弹出“Add Topic”页面；
      <br/>
      2、填写需要创建的Topic名称，分区数和备份数，点击“Add”添加，即可创建 topic。
     </mark>
     <br/>
