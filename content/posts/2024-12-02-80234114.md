    <h3 id="1安装">
     1.安装
    </h3>
    <h4 id="方式1全局安装">
     方式1：全局安装
    </h4>
    <p>
     通过composer安装 laravel 安装器
    </p>
    <pre class="prettyprint"><code class="hljs php">composer <span class="hljs-keyword">global</span> <span class="hljs-keyword">require</span> <span class="hljs-string">"laravel/installer"</span></code></pre>
    <p>
     确保将 composer vender bin 目录放置在你的系统环境变量 $PATH 中，以便系统可以找到 Laravel 的可执行文件。该目录根据您的操作系统存在不同的位置中；一些常见的配置包括：
    </p>
    <ul>
     <li>
      macOS: $HOME/.composer/vendor/bin
     </li>
     <li>
      GNU / Linux 发行版: $HOME/.config/composer/vendor/bin
      <br/>
      安装完后可通过
      <code>
       laravel new 项目名
      </code>
      安装laravel
     </li>
    </ul>
    <h4 id="方式二命令行安装">
     方式二：命令行安装
    </h4>
    <pre class="prettyprint"><code class="hljs livecodeserver">composer <span class="hljs-built_in">create</span>-project <span class="hljs-comment">--prefer -dist laravel/laravel 项目名</span></code></pre>
    <h3 id="2配置公共目录">
     2.配置公共目录
    </h3>
    <p>
     安装完laravel后必须将web服务器根目录指向
     <code>
      public
     </code>
     文件夹。如果是本地环境可忽略此操作。
    </p>
    <h3 id="3配置读写权限">
     3.配置读写权限
    </h3>
    <p>
     <code>
      storage
     </code>
     和
     <code>
      bootstrap/cache
     </code>
     两个文件夹需要写入权限。如果未配置laravel程序将无法运行。如果是本地环境可忽略此操作。
    </p>
    <h3 id="4创建env文件">
     4.创建.env文件
    </h3>
    <p>
     laravel安装完成，如果只有.env.example文件，需要创建.evn文件并将.env.example中的内容复制到.env中。
    </p>
    <h3 id="5生成应用秘钥">
     5.生成应用秘钥
    </h3>
    <p>
     laravel安装完成后，需要在命令行下运行
     <code>
      php artisan key:generate
     </code>
     ，该命令会生成一个32位的随机数，并写入.env文件中
    </p>
    <h3 id="6修改configappphp下的timezone">
     6.修改config/app.php下的timezone
    </h3>
    <p>
     app.php中的timezone默认是UTC,UTC是世界统一时间，需要将时区改成中国时区
     <code>
      Asia/Shanghai
     </code>
    </p>
    <h3 id="7配置数据库">
     7.配置数据库
    </h3>
    <p>
     创建数据库并将数据库配置信息写入.env文件
    </p>
    <pre class="prettyprint"><code class="hljs ini"><span class="hljs-setting">DB_CONNECTION=<span class="hljs-value">mysql</span></span>
<span class="hljs-setting">DB_HOST=<span class="hljs-value"><span class="hljs-number">127.0</span>.<span class="hljs-number">0.1</span></span></span>
<span class="hljs-setting">DB_PORT=<span class="hljs-value"><span class="hljs-number">3306</span></span></span>
<span class="hljs-setting">DB_DATABASE=<span class="hljs-value">homestead</span></span>
<span class="hljs-setting">DB_USERNAME=<span class="hljs-value">root</span></span>
<span class="hljs-setting">DB_PASSWORD=<span class="hljs-value">root</span></span></code></pre>
    <h3 id="8运行本地开发环境">
     8.运行本地开发环境
    </h3>
    <p>
     在命令行窗口运行
     <code>
      php artisan server
     </code>
     完成。
    </p>
