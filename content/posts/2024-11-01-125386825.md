    <h3>
     <a id="_3">
     </a>
     环境说明
    </h3>
    <ul>
     <li>
      Windows Server 2008（域控制器）
     </li>
     <li>
      Windows 2012（域内主机）
     </li>
    </ul>
    <h3>
     <a id="_6">
     </a>
     搭建过程
    </h3>
    <ol>
     <li>
      打开 “服务器管理器”，点击左侧的 “角色”，再点击 “添加角色”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bdf2966ee77aab09b6247d545843d9c6.png"/>
     </li>
     <li>
      勾选 “Active Directory 域服务”，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a6ce1ee9a56e336c2ffad3737eefdb8.png"/>
     </li>
     <li>
      点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0f1aa83dc8006bbe77afdf51936b4548.png"/>
     </li>
     <li>
      点击 “安装”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b6e8073850893963039f1d4c5bd83f8c.png"/>
     </li>
     <li>
      安装完成后点击 “关闭”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/254e5406ab6f7ab5825779c98a9ed0f7.png"/>
     </li>
     <li>
      打开 Win+R，输入dcpromo.exe。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/55bc414014989ae9fe7654d1a2baff81.png"/>
     </li>
     <li>
      弹出Active Directory域服务安装向导，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/72ca1488dbad5b1b1eb5751b609f2809.png"/>
     </li>
     <li>
      点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ddc448647fdb62ad4cffee401f18fdb.png"/>
     </li>
     <li>
      选择 “新林中新建域”，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/610a2fad5046e81610552eac47f8c4bb.png"/>
     </li>
     <li>
      输入自定义的域名，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6175e690062caa293a51ceb51a922c92.png"/>
     </li>
     <li>
      林功能级别选择Windows Server 2008，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/06900777028bd8fcf69e6b2623133337.png"/>
     </li>
     <li>
      勾选 “DNS服务器”，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7a7541f5e79048e09609212d38a67869.png"/>
     </li>
     <li>
      点击 “是”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8f83c611e9f702349742a66f444a31ea.png"/>
     </li>
     <li>
      选择文件存放路径，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/494ca7be1f67da7fa05323efff8b607e.png"/>
     </li>
     <li>
      设置目录服务还原模式Administrator的密码，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1cbcc2400df4cb7a054c72a293102e9a.png"/>
     </li>
     <li>
      查看配置信息，点击 “下一步”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9da652fd232913446519505479febc1d.png"/>
     </li>
     <li>
      完成安装后选择 “不立即重启”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f40676f84245b03d97aa33ee4abb9b78.png"/>
     </li>
     <li>
      打开网卡的IP配置信息，将DNS服务器设置为本机ip。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7e55e4bce249e5b01e64c25d92d6af3f.png"/>
     </li>
     <li>
      重启服务器即可。
     </li>
    </ol>
    <h3>
     <a id="_46">
     </a>
     检测域是否建立成功
    </h3>
    <ol>
     <li>
      开启第二台机器Windows2012，修改主机的DNS为域服务器ip。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/97defb68cc4790219c7e10599357bb59.png"/>
     </li>
     <li>
      右击 “计算机”，选择 “属性”，点击右方的 “更改设置”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/518df22349f866d7adde4fd439ba86d8.png"/>
     </li>
     <li>
      选择 “更改”。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/88cc8902655cd8e30f6bd2e8634de1d2.png"/>
     </li>
     <li>
      填入定义好的域名。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dcf9215f50b46227e8782ebbbe57c516.png"/>
     </li>
     <li>
      输入域管理员账号和密码，等待加域成功，重启后生效。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/18020f3e257a903e9f764fbd912a24d7.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2edc552ba5275358339b88a76fa08bb9.png"/>
     </li>
    </ol>
