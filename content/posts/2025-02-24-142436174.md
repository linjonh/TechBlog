      团队共由三名成员组成：A，B，C，均来自XXXX大学，是计算机与通信工程学院本科生与在读研究生。在日常学习之余，通过帆软官网提供的FineReport的学习视频和帮助文档进行学习，并顺利通过帆软认证资深报表工程师考试，获得帆软发送的项目交付能力认证证书。希望能够通过参加此次比赛，提升自己的能力与技术，向他人学习。这次的作品由三人分工合作，共同完成。

    <h3>
     <a id="2__6">
     </a>
     2. 作品背景
    </h3>
    <blockquote>
     <p>
      2021年我国数字经济规模达到45.5万亿元，同比增长16.2%，数字经济占GDP比重达到39.8%，已成为驱动我国经济高质量发展的关键动力。2021年12月，国务院印发 《“十四五”数字经济发展规划》，对数字经济发展提出新的发展原则和发展目标，并在“打造智慧共享的新型数字生活” 部分提出要发展云游戏新业态，而千兆宽带用户数、软件和信息技术服务业等规模预期，对于云游戏产业的发展和普及同样有着重要的推动作用。考虑到数字经济对国民经济的巨大贡献,探究中国云游戏行业的发展态势具有极为重要的现实意义。
      <br/>
      本作品从云游戏行业发展现状、云游戏用户特征分析、云游戏产业链分析、云游戏面临挑战、云游戏未来展望共五方面入手，对中国云游戏行业近年来的发展数据进行可视化分析，致力于回答以下问题：云游戏的现状和增长空间如何？云游戏产业链如何进一步完善型发展？云游戏面临什么新挑战以及采取何种解决途径？云游戏的发展趋势以及未来如何演绎？
     </p>
    </blockquote>

    <h4>
     <a id="41__28">
     </a>
     4.1 中国云游戏行业发展现状
    </h4>
    <blockquote>
     <p>
      2021年中国云游戏市场规模约34亿元，同比增长54.5%，行业将在未来几年保持高速增长，预计在2022年至2025年，云游戏市场整体复合增长率达到76.6%。2022年，云游戏市场受到游戏版号发放趋缓、商业变现突破困难等影响，行业增速 有所回落；随着5G建设的加速和元宇宙的持续发展，云游戏市场增长的主要驱动力，一方面来源于优质云游戏产品的推出 及云原生游戏的持续研发；另一方面来源于云游戏体验升级带来的用户的付费意愿和ARPU值的提升。且2021年，中国云游戏月活用户规模约为7000万人，同比增长32.7%，保持着高速的增长，预计在2025年月活用户规模将 超过1.8亿人。随着云游戏相关技术的突破，云游戏体验的持续升级是用户规模增长的主要推动力；同时，更多的互联网平台如抖音、快手、哔哩哔哩等纷纷布局云游戏业务，有望将平台用户往云游戏用户转化，进一步扩大云游戏用户规模。中国云游戏市场份额与用户渗透率以及云游戏在整体游戏市场中占比快速提升 ，在游戏市场增速和用户增速逐步放缓的背景下，云游戏将成为游戏市场越来越重要的组成部分，并推动游戏市场进一步增 长，预计在2025年云游戏市场规模占比将达到7.2%，云游戏用户规模占比将达到27.0%。中国云游戏行业产业链也持续完善，在我们的日常生活中已经非常热本。中国云游戏行业发展现状良好，且通过研究分析不难得到在未来云游戏行业也必将更加飞速发展的结论。
      <br/>
      中国云游戏行业发展现状如下图2所示：
      <br/>
      <img alt="图2 中国云游戏行业发展现状" src="https://i-blog.csdnimg.cn/direct/619a4d38d06e4a1191a8f1604dee3bde.png">
       <br/>
       该模块分为四个部分：1.在网页左侧上方，通过轮播多重柱状图的方式展示了中国云游戏行业两种渗透率的相关分析，通过分析2020-2025年中国云游戏市场规模渗透率与中国云游戏用户规模渗透率，可以发现云游戏市场在游戏市场中所占规模与云游戏用户在所有用户的比例正逐年增加，云游戏玩家用户数量正不断增加，在游戏市场增速和用户增速逐步放缓的背景下，云游戏将成为游戏市场越来越重要的组成部分，并推动游戏市场进一步增 长，预计在2025年云游戏市场规模占比将达到7.2%，云游戏用户规模占比将达到27.0%。2.在网页左侧下方，通过轮播图表展示2022年中国云游戏热门产品以及2022年中国云游戏热门题材分析，都选出了榜单排名前10的内容进行展示，可以发现受国内游戏市场的大环境和游戏用户偏好影响，在云游戏市场中，移动游戏同样也是最热门的产品。根据咪咕快游等云游戏平台的数据显示，《原神》、《迷你世界》、《崩坏3》是用户玩得最多的云游戏产品；从游戏品类看，云游戏用户偏好的游戏题材与传统游戏用户存在一定差异，角色扮演类、模拟经营类和策略类是最受云游戏用户欢迎。 3.右侧上方通过五个指标卡来对中国云游戏行业的发展前景以及痛点进行概括分析，1-3为发展前景，4-5为目前云游戏行业还存在的问题，可以发现云游戏给行业带来新的增量，但发展过程中仍存在较多痛点，元宇宙的火热使得云游戏迅速进入主流游戏市场，并给游戏行业带来新的想象空间：云游戏有望给行业带来新的市场增量 和用户增量，通过无需下载、点击即玩、跨屏互动等特性，提升用户体验和用户粘性。但在近三年的发展过程中，行业仍在着挑战和痛点，在游戏内容、用户体验和商业模式等方面仍需要市场进一步探索及验证。4.右侧下方通过组合图的形式对2022-2025年云游戏月均活跃用户规模进行分析，2021年，中国云游戏月活用户规模约为7000万人，同比增长32.7%，保持着高速的增长，预计在2025年月活用户规模将 超过1.8亿人。随着云游戏相关技术的突破，云游戏体验的持续升级是用户规模增长的主要推动力；同时，更多的互联网 平台如抖音、快手、哔哩哔哩等纷纷布局云游戏业务，有望将平台用户往云游戏用户转化，进一步扩大云游戏用户规模，可以得出云游戏用户规模有望快速爆发的结论。

    <h4>
     <a id="42__35">
     </a>
     4.2 中国云游戏用户特征分析
    </h4>
    <blockquote>
     <p>
      随着互联网的发展，越来的越多的人们加入游戏的行列，用户所玩的游戏设备也有所增多，从最初的游戏机，到现在的手机、电脑和平板都可以支持打游戏，这也为云游戏的发展增添了一份力。在最初的游戏只有手柄游戏，比较单一，但是现在的云游戏有很多的操作方式，可以通过触屏或者是键盘鼠标。随着用户水平的提高，用户在游戏方面的个人消费水平也有一定的提高。
      <br/>
      中国云游戏用户特征分析模块如下图所示：
      <br/>
      <img alt="图3 中国云游戏用户特征分析" src="https://i-blog.csdnimg.cn/direct/ce61716f66764454858f55a079332424.png">
       <br/>
       该模块细分为八部分：1.采用横向柱状图来显示用户在玩云游戏使用设备轻情况；2.采用横向柱状图展示用户玩云游戏操作方式的偏好；3.采用饼状图展示中国云游戏用户性别情况；4.采用柱状图展示中国云游戏用户年龄的分布；5.采用折线图来展示云游戏用户在玩游戏时遇到的突出问题。6.用横向柱状图来展示云游戏用户的个人消费水平；7.采用面积图来展示玩游戏的主要原因所占的百分比；8.采用柱状图来展示用户所玩云游戏的品类。
       <br/>
       通过图表我们可以得到如下信息：男性是主要云游戏用户群体，用户年龄段分布相对均衡中国云游戏用户中，男性用户占比在六成以上，处于发展初期的云游戏吸引了更多男性用户尝试。从年龄结构看，云游戏 用户整体相对年轻，调研数据显示，30岁以下的云游戏用户占比接近7成，达到68.2%。用户在云游戏的个人月消费集中在3001-5000元，占比达到28.9%。从云游戏用户偏好内容看，移动游戏仍是用户玩云游戏时最受欢迎的产品；而从游戏品类看，角色扮演类、FPS射击类和 动作格斗类是用户最喜爱的游戏品类；无需下载和安装，点击即玩的便捷属性，是云游戏吸引用户的最主要原因。手机是用户主要接入设备，触摸操作最受用户认可。基于智能手机在国内的广泛普及，手机以超过8成的渗透率成为用户玩云游戏的最主要使用设备；同时，移动游戏流行带 来的触摸屏操作，也对国内游戏用户的操作习惯带来的很深的影响，超过65%的用户在玩云游戏时同样偏好触摸屏操作。云游戏仍存在一些突出的用户痛点。据调研数据显示，43.6%的云游戏用户在体验过程中遇到延迟过高、画面不流畅和卡顿等情况，是对用户游戏体验影响最 大的问题；同时，免费时间、游戏内容、排队时间长等也是影响用户体验的突出问题。如何持续提升用户体验，是云游戏 平台企业面临的核心问题。
      </img>
     </p>
    </blockquote>
    <h4>
     <a id="43__43">
     </a>
     4.3 中国云游戏产业链分析
    </h4>
    <blockquote>
     <p>
      互联网的兴起带动了云游戏的发展，同时云游戏的发展也带动了其他行业，5G时代的带来，给云游戏的很好的利益。云游戏行业产业链上游以游戏开发商和软硬件厂商为主，游戏开发商将研发完成的游戏授权给中游；中游是云计算服务和云游戏平台进行游戏发行，为开发者提供更专业的游戏推广资源和营销方案，使游戏产品达到一个更高的高度；下游主要为终端设备与游戏用户，5G技术带动智能手机出货量增长，云游戏存在跨平台操作优势，因此移动端和固定端设备出货量的增加有利于云游戏推广。
      <br/>
      中国云游戏产业链分析模块如下图所示：
      <br/>

    <h4>
     <a id="44__50">
     </a>
     4.4 中国云游戏面临新挑战
    </h4>
    <blockquote>
     <p>
      云游戏在5G和边缘计算的加持下展现了迅猛的发展态势，但由于我国云游戏发展起步较晚，目前仍有许多短板亟待解决。云游戏技术革新将面临云游戏画质、时延等用户体验问题的考验；随着中国游戏产品审核趋严，进口游戏版号比例出现下降，或影响云游戏内容填充速度；目前已上线运营的云游戏平台均采取订阅付费模式，或将面临用户消费习惯考验；支持云游戏的大部分软硬件技术仍被海外国家掌控，外围不确定性或带来技术供给影响。
      <br/>
      中国云游戏面临新挑战模块如下图所示：
      <br/>
      <img alt="图5 中国云游戏面临新挑战" src="https://i-blog.csdnimg.cn/direct/8428e39dd33e41a6887120e20a1cd711.png">
       <br/>
       该模块共分为四部分：1.订阅付费模式VS用户消费习惯。该部分细分为两部分，其中，游戏用户月均游戏充值意愿情况采用饼状图展示，以柱状图的形式展示游戏用户移动网络流量月均费用情况；2.游戏环境约束，用户体验欠佳。采用横向柱状图将游戏用户放弃一款游戏的原因占比进行展示；3.游戏审核趋严，引进速率减缓。该部分细分为两部分，其中，采用柱状图和折线图相结合展示过审游戏类型及占比，采用横向柱状图展示过审游戏本土/进口情况；4.核心技术未掌控，技术供给受限。以滚动轮播的形式展示部分云游戏重要支持技术开发商及其所属国。
       <br/>
       通过图表的展示我们可以得到如下信息：目前已上线运营的云游戏平台，其付费模式均采取了订阅付费的形式。但在国内过去市场环境下沉淀了大量不付费和少量付费的游戏用户，月付费在50元以下的用户占比接近6成，对于这部分用户来说，适应云游戏平台的订阅费用仍存在一定困难；游戏用户在移动网络流量上的消费也将面临5G网络冲击而提高，这对于低付费群体同样不够友好；外挂、付费与平衡性、技术漏洞、画质音效等方面存在的问题会大概率导致用户放弃游戏，此外，在付费上云游戏相比于传统端会产生额外的费用，这也会一定程度影响用户的体验；2018年以来，中国游戏产业从政策端发起了精品化改革，产品审批趋严。从版号数量来看，能够过审进入市场的新产品大幅度减少，或将降低云游戏平台的内容填充速率；海外可供进口的优质产品较多，但研发标准与我国存在差异，内容流入也存在一定困难。从过审游戏进口情况来看，过审游戏进口占比在不断下降；2019年以来，部分海外硬件、软件技术在国内的使用频频受到不确定性影响，而支持云游戏的大部分软件及硬件技术仍被海外国家掌控，若海外限制落地，将对国内云游戏市场带来较大负面影响。
       <br/>
       对于云游戏目前遇到的问题，既是挑战，也是机遇。云游戏商家要多做市场调研，解决云游戏画质、时延等用户体验问题，采取过渡的形式使得用户接受订阅付费模式，提供给游戏用户良好的游戏环境；对于中国游戏产业政策端制定者，对待游戏产品精品化改革的同时，增强国内云游戏平台的内容填充速率，同时适当地引进海外优质产品；要加强云游戏重要支持技术的研发，尽早摆脱海外国家技术主导，实现游戏技术独立。

     </p>

      <br/>
      中国云游戏未来展望模块如下图所示：
      <br/>
      <img alt="图6 中国云游戏未来展望" src="https://i-blog.csdnimg.cn/direct/adce161215cf45adbdb1c4a737ce7618.png"/>
      <br/>
      该模块共分为五部分：1.采用面积图展示了云游戏专利申请趋势分析情况；2.用柱形图和折线图的组合展示了全球互联网用户人数，其中柱形图展示用户人数，折线图展示用户的增长率；3.利用气泡树展示云游戏的多元化应用场景；4.采用柱形图展示了全球云游戏收入的市场预测；5.采用扇形图展示了云游戏的受众的占比。
