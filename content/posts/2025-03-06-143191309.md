    <h2>
     <a id="_3">
     </a>
     跳转命令
    </h2>
    <h3>
     <a id="_4">
     </a>
     跳转命令
    </h3>
    <ul>
     <li>
      <code>
       #
      </code>
      ：向前查找光标当前所在单词，并跳转到该单词的上一个出现位置。
     </li>
     <li>
      <code>
       *
      </code>
      ：向后查找光标当前所在单词，并跳转到该单词的下一个出现位置。
     </li>
    </ul>
    <h4>
     <a id="_9">
     </a>
     行内跳转
    </h4>
    <ul>
     <li>
      <code>
       0
      </code>
      ：跳转到当前行的行首。
      <strong>
       [Home]
      </strong>
     </li>
     <li>
      <code>
       $
      </code>
      ：跳转到当前行的行尾。
      <strong>
       [End]
      </strong>
     </li>
     <li>
      <code>
       ^
      </code>
      ：跳转到当前行的第一个非空字符处。
     </li>
     <li>
      <code>
       g_
      </code>
      ：跳转到行尾最后一个非空白字符。
     </li>
     <li>
      <code>
       |n
      </code>
      ：跳转到当前行的第 n 列（例如：
      <code>
       |10
      </code>
      跳转到第 10 列）。
     </li>
    </ul>
    <h4>
     <a id="_17">
     </a>
     文件内跳转
    </h4>
    <ul>
     <li>
      <code>
       gg
      </code>
      ：跳转到文件第一行的行首。
     </li>
     <li>
      <code>
       G
      </code>
      ：跳转到文件最后一行的行首。
     </li>
     <li>
      <code>
       G$
      </code>
      ：跳转到文件最后一行的行尾。
      <strong>
       [Ctrl + End]
      </strong>
     </li>
     <li>
      <code>
       :line-number
      </code>
      或
      <code>
       :n
      </code>
      ：跳转到指定行号的行（例如：
      <code>
       :100
      </code>
      跳转到第 100 行）。
     </li>
     <li>
      <code>
       nG
      </code>
      ：跳转到文件的第 n 行（例如：
      <code>
       20G
      </code>
      跳转到文件的第 20 行）。
     </li>
     <li>
      <code>
       /{pattern}
      </code>
      ：跳转到第一个与
      <code>
       {pattern}
      </code>
      匹配的位置。
     </li>
    </ul>
    <h4>
     <a id="_26">
     </a>
     屏幕跳转
    </h4>
    <ul>
     <li>
      <code>
       H
      </code>
      ：跳转到屏幕的最上方一行的第一个字符。
     </li>
     <li>
      <code>
       M
      </code>
      ：跳转到屏幕的中央那一行的第一个字符。
     </li>
     <li>
      <code>
       L
      </code>
      ：跳转到屏幕的最下方一行的第一个字符。
     </li>
     <li>
      <code>
       Ctrl + b
      </code>
      ：向上滚动一屏。
     </li>
     <li>
      <code>
       Ctrl + f
      </code>
      ：向下滚动一屏。
     </li>
    </ul>
    <h4>
     <a id="_34">
     </a>
     单词跳转
    </h4>
    <ul>
     <li>
      <code>
       b
      </code>
      ：跳转到前 1 个单词的开头（例如：
      <code>
       5b
      </code>
      向前跳转到第 5 个单词的开头）。
     </li>
     <li>
      <code>
       e
      </code>
      ：跳转到当前单词的末尾。
     </li>
     <li>
      <code>
       w
      </code>
      ：跳转到下 1 个单词的开头（例如：
      <code>
       5w
      </code>
      向后跳转到第 5 个单词的开头）。
     </li>
     <li>
      <code>
       ge
      </code>
      ：向前跳转到当前单词的结尾（例如：
      <code>
       3ge
      </code>
      向前跳转到第 3 个单词的结尾）。
     </li>
    </ul>
    <h4>
     <a id="_41">
     </a>
     字符跳转
    </h4>
    <ul>
     <li>
      <code>
       f{char}
      </code>
      ：跳转到当前行下一个字符
      <code>
       {char}
      </code>
      的位置（例如：
      <code>
       fa
      </code>
      跳转到下一个字母 a）。
     </li>
     <li>
      <code>
       F{char}
      </code>
      ：跳转到当前行上一个字符
      <code>
       {char}
      </code>
      的位置。
     </li>
     <li>
      <code>
       h
      </code>
      或
      <code>
       ←
      </code>
      ：向左跳转一个字符（例如：
      <code>
       10h
      </code>
      向后跳转 10 个字符）。
     </li>
     <li>
      <code>
       l
      </code>
      或
      <code>
       →
      </code>
      ：向右跳转一个字符（例如：
      <code>
       10l
      </code>
      向前跳转 10 个字符）。
     </li>
    </ul>
    <h4>
     <a id="_48">
     </a>
     行间跳转
    </h4>
    <ul>
     <li>
      <code>
       j
      </code>
      或
      <code>
       ↓
      </code>
      ：向下跳转一行（例如：
      <code>
       5j
      </code>
      向下跳转 5 行）。
     </li>
     <li>
      <code>
       k
      </code>
      或
      <code>
       ↑
      </code>
      ：向上跳转一行（例如：
      <code>
       5k
      </code>
      向上跳转 5 行）。
     </li>
     <li>
      <code>
       n&lt;Enter&gt;
      </code>
      ：光标向下跳转 n 行
     </li>
    </ul>
    <h3>
     <a id="_54">
     </a>
     删除命令
    </h3>
    <h4>
     <a id="_56">
     </a>
     字符删除
    </h4>
    <ul>
     <li>
      <code>
       x
      </code>
      ：删除当前光标所在位置的字符，并将光标自动向后跳转一个字符（例如：
      <code>
       3x
      </code>
      删除光标后的三个字符，
      <code>
       3X
      </code>
      删除光标前的三个字符）。
     </li>
     <li>
      <code>
       dl
      </code>
      ：同
      <code>
       x
      </code>
      ，删除光标所在位置的字符（例如：
      <code>
       5dl
      </code>
      删除光标后的 5 个字符）。
     </li>
     <li>
      <code>
       dh
      </code>
      ：删除光标所在位置的前一个字符。
     </li>
    </ul>
    <h4>
     <a id="_62">
     </a>
     行删除
    </h4>
    <ul>
     <li>
      <code>
       dd
      </code>
      ：删除当前行（例如：
      <code>
       5dd
      </code>
      向下删除 5 行，共 6 行）。
     </li>
     <li>
      <code>
       D
      </code>
      ：删除从光标当前位置一直到该行的末尾（包括光标所在的字符）。
     </li>
     <li>
      <code>
       d$
      </code>
      ：删除至行尾。
     </li>
     <li>
      <code>
       d0
      </code>
      ：删除至行首。
     </li>
    </ul>
    <h4>
     <a id="_69">
     </a>
     单词删除
    </h4>
    <ul>
     <li>
      <code>
       dw
      </code>
      ：删除光标所在位置到下 1 个单词开头之间的内容（不保留空格，例如：
      <code>
       5dw
      </code>
      删除连续的 5 个单词）。
     </li>
     <li>
      <code>
       de
      </code>
      ：删除从光标位置到当前单词末尾的内容（不会删除空格）。
     </li>
     <li>
      <code>
       dex
      </code>
      ：删除从光标所在位置到当前单词末尾之间的所有字符，包括当前单词末尾的空格。
     </li>
     <li>
      <code>
       db
      </code>
      ：删除光标所在单词及其前面的空格。
     </li>
     <li>
      <code>
       daw
      </code>
      ：删除光标所在的单词及其后面的空格。
     </li>
     <li>
      <code>
       diw
      </code>
      ：删除光标所在的单词，并保留其前后的空格。
     </li>
     <li>
      <code>
       bdw
      </code>
      ：删除光标所在位置到前 1 个单词开头之间的内容，并且不保留空格。
     </li>
    </ul>
    <h4>
     <a id="_79">
     </a>
     范围删除
    </h4>
    <ul>
     <li>
      <code>
       :3,5d
      </code>
      ：删除从第 3 行到第 5 行。
     </li>
     <li>
      <code>
       d1G
      </code>
      ：删除从当前行到第一行之间的文本。
     </li>
     <li>
      <code>
       dG
      </code>
      ：删除到文件尾，也就是删除整个文档的后半部分（包括光标所在的字符）。
     </li>
     <li>
      <code>
       dgg
      </code>
      ：删除到文件首。
     </li>
     <li>
      <code>
       dj
      </code>
      ：删除当前行和下一行（例如：
      <code>
       5dj
      </code>
      删除当前行及其下面的 5 行，即向下删除 6 行）。
     </li>
     <li>
      <code>
       dk
      </code>
      ：删除当前行和上一行（例如：
      <code>
       5dk
      </code>
      删除当前行及其上面的 5 行，即向上删除 6 行）。
     </li>
     <li>
      <code>
       d2w
      </code>
      ：删除光标后面的两个单词。
     </li>
     <li>
      <code>
       dX
      </code>
      ：删除当前单词前面的空格，但不包括当前单词本身
     </li>
     <li>
      <code>
       wde
      </code>
      ：删除从光标位置到下 1 个单词开头的内容，并且保留空格。其中数字可以指定要删除的单词数
     </li>
    </ul>
    <h4>
     <a id="_91">
     </a>
     其他
    </h4>
    <ul>
     <li>
      <code>
       d{motion}
      </code>
      :
      <code>
       motion
      </code>
      可以是任何移动光标的命令,例如
      <code>
       dw
      </code>
      ,
      <code>
       d2w
      </code>
      ,
      <code>
       d$
      </code>
      等.
     </li>
    </ul>
    <p>
     <strong>
      注意:
     </strong>
     <code>
      d
     </code>
     删除的字符存放到 Vim 的剪贴板中，可以使用
     <code>
      p
     </code>
     命令将其粘贴到另一个位置。
    </p>
    <h3>
     <a id="_97">
     </a>
     复制和粘贴
    </h3>
    <h4>
     <a id="_99">
     </a>
     复制
    </h4>
    <ul>
     <li>
      <code>
       y
      </code>
      ：复制（默认是复制到"寄存器）。
     </li>
     <li>
      <code>
       "+y
      </code>
      ：复制到系统剪贴板（也就是 vim 的 + 寄存器）。
     </li>
     <li>
      <code>
       yw
      </code>
      ：复制 1 个单词（例如：
      <code>
       y3w
      </code>
      复制 3 个单词）。
     </li>
     <li>
      <code>
       yy
      </code>
      ：复制游标所在的那一行(常用)。
     </li>
     <li>
      <code>
       nyy
      </code>
      ：n 为数字，复制光标所在的向下 n 行（例如：
      <code>
       20yy
      </code>
      则是复制 20 行）。
     </li>
     <li>
      <code>
       y1G
      </code>
      ：复制游标所在行到第一行的所有数据。
     </li>
     <li>
      <code>
       yG
      </code>
      ：复制游标所在行到最后一行的所有数据。
     </li>
     <li>
      <code>
       y0
      </code>
      ：复制光标所在字符到该行行首的所有数据。
     </li>
     <li>
      <code>
       y$
      </code>
      ：复制光标所在字符到该行行尾的所有数据。
     </li>
     <li>
      <code>
       5bgyw
      </code>
      ：复制光标所在单词向前 5 个单词
     </li>
     <li>
      <code>
       5egyw
      </code>
      ：复制光标所在单词向后 5 个单词
     </li>
     <li>
      <code>
       5b5egyw
      </code>
      ：复制光标所在单词向前向后 3 个单词
     </li>
    </ul>
    <h4>
     <a id="_114">
     </a>
     粘贴
    </h4>
    <ul>
     <li>
      <code>
       p
      </code>
      ：粘贴（默认从"寄存器取出内容粘贴，在光标后）。
     </li>
     <li>
      <code>
       P
      </code>
      ：粘贴到光标前。
     </li>
     <li>
      <code>
       "+p
      </code>
      ：从系统剪贴板粘贴。
     </li>
     <li>
      <code>
       5p
      </code>
      ：粘贴五份。
     </li>
    </ul>
    <h4>
     <a id="_121">
     </a>
     插入新行
    </h4>
    <ul>
     <li>
      <code>
       o
      </code>
      ：在当前行下面插入 1 个新行，并将光标放置到新行的起始位置。
     </li>
     <li>
      <code>
       O
      </code>
      ：在当前行上面插入 1 个新行，并将光标放置到新行的起始位置。
     </li>
    </ul>
    <h3>
     <a id="_126">
     </a>
     插入模式
    </h3>
    <ul>
     <li>
      <code>
       i
      </code>
      ：在当前光标位置处插入文本。
     </li>
     <li>
      <code>
       I
      </code>
      ：在当前行的开头插入文本。
     </li>
     <li>
      <code>
       a
      </code>
      ：在当前光标位置后插入文本。
     </li>
     <li>
      <code>
       A
      </code>
      ：在当前行的结尾插入文本。
     </li>
     <li>
      <code>
       o
      </code>
      ：在当前行的下一行插入文本。
     </li>
     <li>
      <code>
       O
      </code>
      ：在当前行的上一行插入文本。
     </li>
    </ul>
    <h3>
     <a id="_135">
     </a>
     撤销和重做
    </h3>
    <ul>
     <li>
      <code>
       u
      </code>
      ：撤销，或输入
      <code>
       :undo
      </code>
      命令。
     </li>
     <li>
      <code>
       Ctrl + r
      </code>
      ：重做或输入
      <code>
       :redo
      </code>
      命令。
     </li>
    </ul>
    <h3>
     <a id="_140">
     </a>
     替换和搜索
    </h3>
    <ul>
     <li>
      <code>
       :%s/old/new/g
      </code>
      ：快速替换文本中所有的匹配项。
     </li>
     <li>
      <code>
       :%s/old/new/gi
      </code>
      ：快速替换文本中所有的匹配项，忽略大小写。
     </li>
     <li>
      <code>
       :%s/[\\u4e00-\\u9fa5]//g
      </code>
      ：替换所有中文字符。
     </li>
     <li>
      <code>
       :g/^$/d
      </code>
      ：删除所有空行。
     </li>
     <li>
      <code>
       :%s/\\s\\+/ /g
      </code>
      ：将连续空格替换为单个空格。
     </li>
     <li>
      <code>
       :g/^\\s*$/d
      </code>
      ：删除包含空格的空行。
     </li>
     <li>
      <code>
       :%s/^\\s*\\n//
      </code>
      ：删除文档中的连续空格，只保留单个空格，并且删除包含空格的空行。
     </li>
    </ul>
    <h3>
     <a id="_150">
     </a>
     代码块操作
    </h3>
    <h4>
     <a id="_152">
     </a>
     代码块缩进
    </h4>
    <ul>
     <li>
      <code>
       &lt;
      </code>
      ：代码块左移（减少缩进）。
     </li>
     <li>
      <code>
       &gt;
      </code>
      ：代码块右移（增加缩进）。
     </li>
     <li>
      <code>
       &lt;&lt;
      </code>
      ：减少选中行的缩进，块状可视模式下使用。
     </li>
     <li>
      <code>
       &gt;&gt;
      </code>
      ：增加选中行的缩进，块状可视模式下使用。
     </li>
    </ul>
    <h4>
     <a id="_159">
     </a>
     代码块排序
    </h4>
    <ul>
     <li>
      <code>
       :sort
      </code>
      ：按字典顺序排序选中的行。
     </li>
     <li>
      <code>
       :sort!
      </code>
      ：按逆字典顺序排序选中的行。
     </li>
     <li>
      <code>
       :sort n
      </code>
      ：按数字顺序排序选中的行。
     </li>
    </ul>
    <h4>
     <a id="_165">
     </a>
     代码块注释
    </h4>
    <ul>
     <li>
      <code>
       I
      </code>
      ：在当前行的开头插入注释（需要配合可视块模式）。
     </li>
     <li>
      <code>
       A
      </code>
      ：在当前行的末尾添加注释（需要配合可视块模式）。
     </li>
     <li>
      <code>
       :%s/^/# /
      </code>
      ：在每一行的行首插入 “#” 作为注释符号。
     </li>
    </ul>
    <h4>
     <a id="_171">
     </a>
     代码块格式化
    </h4>
    <ul>
     <li>
      <code>
       =
      </code>
      ：重新格式化选中的代码块（自动缩进）。
     </li>
    </ul>
    <h3>
     <a id="_175">
     </a>
     其他命令
    </h3>
    <ul>
     <li>
      <code>
       nohl
      </code>
      ：取消高亮。
     </li>
     <li>
      <code>
       :set whichwrap+=&lt;,&gt;
      </code>
      ：允许光标在行首和行尾跳转到下一行或上一行。
     </li>
     <li>
      <code>
       :set nofixeol
      </code>
      ：禁用文件末尾自动添加新行。
     </li>
     <li>
      <code>
       :set mouse=a
      </code>
      ：启用鼠标支持。
     </li>
     <li>
      <code>
       :set mousemodel=extend
      </code>
      ：设置鼠标滚动时光标的移动方式为扩展模式
     </li>
     <li>
      <code>
       :set number
      </code>
      或者
      <code>
       :set nu
      </code>
      ：显示行号
     </li>
     <li>
      <code>
       :set nonumber
      </code>
      或者
      <code>
       :set nonu
      </code>
      ：不显示行号
     </li>
    </ul>
    <h3>
     <a id="_185">
     </a>
     文件操作
    </h3>
    <h4>
     <a id="_187">
     </a>
     保存并退出
    </h4>
    <ul>
     <li>
      <code>
       :wq
      </code>
      ：保存文件并退出编辑器。
     </li>
     <li>
      <code>
       :x
      </code>
      ：保存文件并退出编辑器，仅当文件自上次保存以来有修改时。
     </li>
     <li>
      <code>
       :xit
      </code>
      ：保存文件并退出编辑器，无论文件是否被修改。
     </li>
     <li>
      <code>
       :ZZ
      </code>
      ：保存文件并退出编辑器
     </li>
    </ul>
    <h4>
     <a id="_194">
     </a>
     仅保存
    </h4>
    <ul>
     <li>
      <code>
       :w
      </code>
      ：保存更改但不退出 Vim。
     </li>
     <li>
      <code>
       :w!
      </code>
      ：强制保存更改，忽略某些文件类型插件的警告。
     </li>
    </ul>
    <h4>
     <a id="_199">
     </a>
     保存到其他文件
    </h4>
    <ul>
     <li>
      <code>
       :w &lt;filename&gt;
      </code>
      ：将更改保存到指定的文件名中，不退出 Vim。
     </li>
     <li>
      <code>
       :w! &lt;filename&gt;
      </code>
      ：强制将更改保存到指定的文件名中，忽略警告。
     </li>
    </ul>
    <h4>
     <a id="_204">
     </a>
     保存所有打开的文件
    </h4>
    <ul>
     <li>
      <code>
       :wqa
      </code>
      ：保存所有打开的文件并退出 Vim。
     </li>
    </ul>
    <h4>
     <a id="_208">
     </a>
     撤销保存
    </h4>
    <ul>
     <li>
      <code>
       :undo
      </code>
      ：撤销最近的一次保存操作。
     </li>
    </ul>
    <h4>
     <a id="_212">
     </a>
     设置自动保存
    </h4>
    <ul>
     <li>
      <code>
       :set autowrite
      </code>
      ：自动保存更改，每次执行命令时自动写入文件。
     </li>
     <li>
      <code>
       :set noautowrite
      </code>
      ：取消自动保存。
     </li>
    </ul>
    <h4>
     <a id="_217">
     </a>
     保存并退出所有打开的标签页
    </h4>
    <ul>
     <li>
      <code>
       :wqa
      </code>
      ：在有多个标签页的情况下，保存所有更改并退出 Vim。
     </li>
    </ul>
    <h4>
     <a id="_221">
     </a>
     保存并关闭当前窗口
    </h4>
    <ul>
     <li>
      <code>
       :close
      </code>
      ：关闭当前窗口
     </li>
     <li>
      <code>
       :winc
      </code>
      ：保存当前窗口的更改并关闭窗口，不退出 Vim。
     </li>
    </ul>
    <h4>
     <a id="_226">
     </a>
     保存并关闭所有窗口
    </h4>
    <ul>
     <li>
      <code>
       :wqa
      </code>
      ：保存所有更改并关闭所有窗口，退出 Vim。
     </li>
    </ul>
    <h3>
     <a id="_230">
     </a>
     自定义快捷键
    </h3>
    <ul>
     <li>
      <code>
       :map &lt;C-f&gt; &lt;PageDown&gt;
      </code>
      ：将 Ctrl+f 映射为 PageDown（向下滚动一页）。
     </li>
     <li>
      <code>
       :map &lt;C-b&gt; &lt;PageUp&gt;
      </code>
      ：将 Ctrl+b 映射为 PageUp（向上滚动一页）。
     </li>
     <li>
      <code>
       :map &lt;C-s&gt; :w&lt;CR&gt;
      </code>
      : 将 Ctrl + s 映射为保存。
     </li>
    </ul>
    <h3>
     <a id="_236">
     </a>
     正则表达式和替换命令
    </h3>
    <h4>
     <a id="_238">
     </a>
     插入特定字符
    </h4>
    <ul>
     <li>
      <code>
       :%s/^/#/
      </code>
      ：在整个文件中的每一行的行首插入 “#” 字符
     </li>
     <li>
      <code>
       :1,5s/^/# /
      </code>
      ：在第 1 行到第 5 行之间的每一行的行首插入 “#” 字符
     </li>
    </ul>
    <h4>
     <a id="_243">
     </a>
     删除特定字符
    </h4>
    <ul>
     <li>
      <code>
       :%s/^#//
      </code>
      ：删除整个文件中的每一行的行首的 “#” 字符
     </li>
     <li>
      <code>
       :1,5s/^# //
      </code>
      ：删除第 1 行到第 5 行之间的每一行的行首插入的 “#” 字符
     </li>
    </ul>
    <h4>
     <a id="_248">
     </a>
     插入行首和行尾字符
    </h4>
    <ul>
     <li>
      <code>
       :%s/^/\\/\\/
      </code>
      ：在每一行的行首插入 “//”
     </li>
     <li>
      <code>
       :1,5s/^/# /
      </code>
      ：在第 1 行到第 5 行的行首插入 "# "
     </li>
    </ul>
    <h4>
     <a id="_253">
     </a>
     删除行首和行尾字符
    </h4>
    <ul>
     <li>
      <code>
       :%s/^\\s\\+//
      </code>
      ：删除每行行首的空格
     </li>
     <li>
      <code>
       :%s/\\s\\+$//
      </code>
      ：删除每行行尾的空格
     </li>
    </ul>
    <h4>
     <a id="_258">
     </a>
     替换文本
    </h4>
    <ul>
     <li>
      <code>
       :%s/old/new/g
      </code>
      ：将文件中所有的 “old” 替换为 “new”
     </li>
     <li>
      <code>
       :%s/old/new/gi
      </code>
      ：将文件中所有的 “old” 替换为 “new”，忽略大小写
     </li>
     <li>
      <code>
       :%s/\\n//g
      </code>
      ：删除所有的换行符
     </li>
     <li>
      <code>
       :%s/\\r//g
      </code>
      ：删除所有的回车符
     </li>
     <li>
      <code>
       :%s/\\t//g
      </code>
      ：删除所有的制表符
     </li>
    </ul>
    <h4>
     <a id="_266">
     </a>
     特定范围替换
    </h4>
    <ul>
     <li>
      <code>
       :&lt;起始行号&gt;,&lt;结束行号&gt;s/^/&lt;插入的字符&gt;/
      </code>
      ：在特定范围的行上执行插入操作
     </li>
     <li>
      <code>
       :&lt;起始行号&gt;,&lt;结束行号&gt;s/^&lt;要删除的字符&gt;//
      </code>
      ：在特定范围的行上执行删除操作
     </li>
    </ul>
