    <h3>
     <a id="_0">
     </a>
     使用命令
    </h3>
    <pre><code class="prism language-shell"><span class="token function">sed</span> -i <span class="token string">'s/&lt;原字符串&gt;/&lt;替换字符串&gt;/g'</span> <span class="token operator">&lt;</span>文件<span class="token operator">&gt;</span>
<span class="token function">grep</span> -rl <span class="token string">"&lt;搜索字符串&gt;"</span>
<span class="token function">find</span> -name <span class="token string">"*&lt;搜索字符串&gt;*"</span>
<span class="token function">rename</span> <span class="token operator">&lt;</span>原字符串<span class="token operator">&gt;</span>  <span class="token operator">&lt;</span>替换字符串<span class="token operator">&gt;</span>  <span class="token operator">&lt;</span>文件<span class="token operator">&gt;</span>
</code></pre>
    <h3>
     <a id="_7">
     </a>
     使用方法
    </h3>
    <h6>
     <a id="1_8">
     </a>
     1.替换文件内容
    </h6>
    <p>
     搜索当前目录下所有.v文件,并将文件中字符串"string1",替换为"string2"
    </p>
    <pre><code class="prism language-shell"><span class="token function">sed</span> -i <span class="token string">'s/string1/string2/g'</span> *.v
</code></pre>
    <h6>
     <a id="2_13">
     </a>
     2.替换文件名
    </h6>
    <p>
     搜索当前目录下全部.v文件，并将文件名中字符串"string1"替换为"string2".
    </p>
    <pre><code class="prism language-shell"><span class="token function">rename</span> string1  string2  *.v
</code></pre>
    <h6>
     <a id="3_18">
     </a>
     3.批量替换文件内容
    </h6>
    <p>
     搜索该目录及子目录下全部文件包含字符串"string1"的文件，并将文件中全部的字符串"string2"替换为"string3".
    </p>
    <pre><code class="prism language-shell"><span class="token function">sed</span> -i <span class="token string">'s/string2/string3/g'</span> <span class="token variable"><span class="token variable">`</span><span class="token function">grep</span> -rl <span class="token string">"string1"</span><span class="token variable">`</span></span>
</code></pre>
    <p>
     其中
     <code>
      's/string2/string3/g'
     </code>
     的三个
     <code>
      ‘/’
     </code>
     可替换为
     <code>
      ‘=’ 、‘+’
     </code>
     等，字符串可写为正则表达式。需要注意`grep -rl “string1”`中的"`"为英文输入法时的"Tab"上面的键。
    </p>
    <h6>
     <a id="4_24">
     </a>
     4.批量替换文件名
    </h6>
    <p>
     搜索该目录及子目录下全部文件名包含字符串"string1"的文件，并将文件名中字符串"string2"替换为"string3".
    </p>
    <pre><code class="prism language-shell"><span class="token function">rename</span> string2  string3  <span class="token variable"><span class="token variable">`</span><span class="token function">find</span> -name <span class="token string">"*string1*"</span><span class="token variable">`</span></span>
</code></pre>
