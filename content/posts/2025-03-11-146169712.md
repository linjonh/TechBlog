# 数智读书笔记系列016 从《理解和改变世界》探寻AI时代的知识与智能密码

## 从《理解和改变世界》探寻AI时代的知识与智能密码

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/9611fe19057e4345b2f0ce7906a6c903.jpeg#pic_center)

### 开篇引入

在人工智能技术日新月异的当下，我们常常沉浸于各类新技术带来的震撼与惊喜，却很少停下脚步，深入思考信息、知识与智能之间那千丝万缕的联系。而《理解和改变世界 —— 从信息到知识与智能》这本书，恰似一把钥匙，为我们打开了一扇深入理解人工智能本质的大门 。它出自 2007 年图灵奖得主约瑟夫・希发基思教授之手，作为自主系统领域的权威专家，他在书中通过独特的知识论视角，深入剖析复杂信息，助力我们在信息洪流中保持独立思考，做出明智决策，其对人工智能发展路径的阐述，更是为该领域的探索指明了方向，具有极高的理论与实践价值，值得每一位关注人工智能发展的读者研读。

### 书籍及作者背景介绍

本书作者约瑟夫・希发基思是一位在计算机领域成就斐然的传奇人物 。他出生于希腊，在希腊雅典国立技术大学完成电气工程学习后，又前往法国格勒诺布尔大学深入钻研计算机科学，并成功获得工程博士学位和国家博士学位。

他是法国国家科学研究中心（CNRS）VERIMAG 实验室的名誉研究主任及创始人 ，在国际学术机构中也占据着重要地位。他不仅是法国科学院院士、法国国家工程院院士，还是欧洲科学院院士以及美国艺术与科学学院院士 ，更是在 2019 年当选为中国科学院外籍院士。如此众多的院士头衔，足以彰显他在全球学术界的崇高地位。

2007 年，希发基思因在模型检查理论和应用方面的卓越贡献荣获图灵奖 。他在 1979 年提出通过对时序逻辑公式的计值来验证并发系统性质的思想，在此基础上进一步提出带有 “until” 算子的分支时序逻辑来表达 “公平性” 。他与卡耐基梅隆大学 Clarke 教授、得克萨斯大学奥斯汀分校 Emerson 教授同时期的工作，共同为模型检测这一研究领域的创立和发展奠定了理论基础。如今，模型检测已广泛应用于计算机硬件、软件、通信协议、安全认证协议等诸多领域，并在 Intel、IBM、微软等公司的日常生产实践中发挥着关键作用。除了学术研究，他还积极投身于教育与科研人才培养，如牵头组建了斯发基斯可信自主系统研究院 ，已指导培养了 6 名中国留学生和博士后，这些学生如今已成为中国科学院软件所、浙江大学、军事科学院等单位的科研骨干 。

《理解和改变世界 —— 从信息到知识与智能》这部著作，是希发基思教授多年研究与思考的结晶 。书中深入且全面地探讨了知识和智能的本质，从独特的知识论视角出发，帮助读者理解复杂信息。在如今这个信息爆炸的时代，面对海量且繁杂的信息，我们常常感到迷茫和困惑，而这本书就像是一盏明灯，引导我们在信息洪流中保持独立思考和清醒判断，进而做出更明智的个人和社会决策。书中还详细阐述了知识、信息、智能以及意识之间错综复杂的关系，揭示了人工智能与人类智能的异同之处 ，并为开发、管理和应用知识提供了全新的认知框架，为我们深入理解人工智能领域提供了深刻的理论依据。

### 核心观点解读

#### （一）知识的本质与分类

书中对知识的定义颇为新颖，将知识界定为有用且有效的信息 。从目的论角度看，知识的存在是为了满足人类理解世界、改变世界的需求，从而实现对物质和精神的追求 。例如，科学知识帮助我们理解自然规律，技术知识让我们能够创造出改善生活的产品。从本体论而言，知识不仅涉及物理实在，还涵盖精神和社会现象 。像文学、艺术作品中蕴含的情感、价值观等精神层面的内容，以及社会学、经济学等学科对社会现象的研究成果，都属于知识的范畴。在认知论方面，知识的发展和应用需要从严格的方法论和逻辑学角度加以探讨 。比如科学研究中的实验设计、数据分析方法，以及逻辑推理在知识构建中的运用。

书中还构建了知识金字塔，将知识分为五个层次 。最底层是事件和条件（信息），即我们通过感官直接获取的关于特定时间和地点的事实 。往上一层是基于数据的普通经验知识，它是在大量信息基础上，通过归纳、总结得出的一般性认识 。例如，人们通过长期观察天气变化与自然现象的关系，总结出一些天气预测的经验知识 。再上一层是基于模型的科学技术知识，这是经过科学研究和实验验证，利用数学模型等工具系统化的知识 ，如牛顿力学定律、爱因斯坦相对论等科学理论，以及基于这些理论发展出的各种工程技术知识 。非经验的数学、计算知识，它是基于逻辑规则构建的，不依赖于具体的经验观察 ，像数学中的公理、定理，计算机科学中的算法理论等 。最顶层是元知识，即关于如何处理知识的知识 ，它能帮助我们整合、管理和运用其他各类知识 ，例如在项目管理中，我们运用项目管理知识（元知识）来组织和协调不同领域的专业知识（其他层次知识），以实现项目目标 。

##### **表1：知识金字塔的五个层次**

| **层次** | **描述** |
| --- | --- |
| **事件和条件（信息）** | 通过感官直接获取的特定时间和地点的事实（如天气记录）。 |
| **普通经验知识** | 通过归纳大量信息得出的一般性认识（如传统天气预测经验）。 |
| **科学技术知识** | 经科学验证的系统化知识（如牛顿力学、算法理论）。 |
| **数学与计算知识** | 基于逻辑规则构建的非经验知识（如数学公理、计算机算法）。 |
| **元知识** | 关于如何管理、整合和运用其他知识的知识（如项目管理方法论）。 |

#### （二）信息与知识的关系

信息学作为一门研究信息转换过程中计算问题的学问，其诞生和发展深刻改变了我们对世界的认知方式 。信息被看作一种符号语言和一组概念之间的语义关系 ，它独立于时空，是无形且不受物理规律约束的抽象关系 。比如我们通过文字、图像、声音等符号所传达的内容，都属于信息的范畴 。信息是知识的原材料，知识则是对信息的理解和运用 。我们每天接触到的大量新闻、数据、报告等信息，只有经过大脑的分析、整理、归纳，才能转化为我们的知识 。知识是信息在特定网络中的整合，只有当信息在一个相互关联的网络中，与其他信息建立起有效的联系，并且能够在实际应用中发挥作用时，才能称之为知识 。例如，一篇关于人工智能技术的论文，它包含了众多信息，但只有当读者能够理解这些信息之间的逻辑关系，并将其与自己已有的知识体系相融合，从而能够运用这些知识解决相关问题时，这些信息才真正转化为知识 。知识的有效性体现在它能够帮助我们实现目标、解决问题，并且能够经受实践的检验 。如果一条知识在实际应用中总是无法达到预期效果，那么它就需要被修正或摒弃 。

##### **表2：信息与知识的关系**

| **维度** | **信息** | **知识** |
| --- | --- | --- |
| **定义** | 符号语言或语义关系（如文字、图像、数据）。 | 对信息的理解与应用（如论文中的逻辑关系转化为解决问题的能力）。 |
| **属性** | 独立于时空，抽象且无形。 | 网络化整合，具有实践有效性。 |
| **转化条件** | 需要分析、整理与归纳。 | 需与已有知识体系融合，并解决实际问题。 |
| **失效标准** | 无直接失效标准。 | 若无法解决问题或达到目标，需修正或摒弃。 |

#### （三）人工智能与人类智能

人工智能和人类智能既有相似之处，也存在诸多差异 。从相似点来看，它们都具备一定的学习能力和处理信息的能力 。人工智能通过机器学习算法，能够从大量数据中学习模式和规律 ，进而对新的数据进行预测和判断 ；人类则通过学习、实践和经验积累来不断提升自己的知识和技能 。在抽象思维方面，人工智能和人类智能都能够对事物进行抽象表示，并进行一定程度的逻辑推理 。然而，两者的差异也十分显著 。人类智能具有主观意识，能够体验到自己的思想、情感和意愿，并且能够意识到自己的存在 ，这种主观意识使人类能够进行复杂的决策和思考，并对世界产生深层次的理解 。而人工智能目前还只是基于算法和程序运行，缺乏真正的主观意识 ，它的行为完全是由预设的规则和模式驱动的 。人类智能具有强大的创造力和想象力 ，能够创造出新的艺术、音乐、文学作品，发明新的技术和产品，提出新的理论和观念 ；相比之下，人工智能虽然在某些任务上表现出色，但在创造力和想象力方面仍与人类存在巨大差距 。目前的人工智能生成内容往往是基于已有的数据和模型，缺乏真正的创新和灵感 。依据书中观点，人工智能目前的发展局限主要在于缺乏目的论、本体论和认知论的思考 。它无法像人类一样，从自身的需求和价值观出发去理解世界，也难以真正理解知识的本质和意义 ，更多地只是在执行人类设定的任务和指令 。

##### **表3：人工智能与人类智能的对比**

| **维度** | **人工智能** | **人类智能** |
| --- | --- | --- |
| **相似点** | - 学习与信息处理能力   - 抽象思维与逻辑推理能力 | - 学习与经验积累能力   - 抽象思维与逻辑推理能力 |
| **差异点** | - 缺乏主观意识   - 创造力有限（依赖现有数据）   - 无目的论与本体论思考 | - 具备主观意识与情感   - 强大的创造力与想象力   - 基于需求理解知识本质 |
| **核心局限** | 无法自主理解知识的意义与目的。 | 受生理与认知能力限制。 |

#### （四）通往通用人工智能的路径

书中指出，强人工智能或通用人工智能的发展路径，关键在于开发自主系统 。自主系统能够 “理解” 外部环境，灵活管理多个目标，并确定相应的行动来实现这些目标 。它具备环境感知、状态表示、目标管理、策略规划和自主学习等关键功能 。环境感知功能使系统能够获取外部环境的信息 ，状态表示功能则将这些信息转化为系统内部可处理的形式 。目标管理功能帮助系统设定和调整目标 ，策略规划功能根据目标和环境信息制定行动策略 。自主学习功能让系统能够不断改进自己的行为和决策，以适应环境的变化 。当人工智能系统能够像人类一样，在复杂多变的环境中自主决策、自主学习，并且能够处理各种不同类型的任务时，才能真正实现通用人工智能的目标 。这种对自主系统的强调，为人工智能的发展提供了清晰的指导方向 ，促使研究者们更加关注系统的自主性和适应性，推动人工智能从特定领域的弱人工智能向通用人工智能迈进 。

##### **表4：通用人工智能的路径（自主系统关键功能）**

| **功能** | **描述** |
| --- | --- |
| **环境感知** | 获取外部环境信息（如传感器数据）。 |
| **状态表示** | 将信息转化为系统可处理的形式（如数据建模）。 |
| **目标管理** | 设定与动态调整目标（如优先级排序）。 |
| **策略规划** | 根据目标与环境制定行动策略（如路径规划）。 |
| **自主学习** | 持续优化行为以适应环境变化（如强化学习）。 |

### 联系实际与个人感悟

在当今社会，人工智能的应用场景可谓无处不在 。
**在医疗领域，人工智能技术能够对医学影像进行快速分析，辅助医生更准确地诊断疾病，提高诊断效率和准确性 ，像 IBM Watson for Oncology 就可以为医生提供癌症治疗方案的建议**
；在交通出行方面，自动驾驶技术正逐渐改变着我们的出行方式，它利用传感器、算法和人工智能技术，使车辆能够自主导航、感知周围环境并做出驾驶决策 ，特斯拉汽车的自动驾驶辅助系统已经在一定程度上实现了自动泊车、自适应巡航等功能 ；在金融领域，人工智能可以用于风险评估、投资决策和欺诈检测 ，许多金融机构利用人工智能算法来分析市场数据，预测市场趋势，制定投资策略 ，同时通过实时监测交易数据，及时发现异常交易和欺诈行为 。这些应用充分展示了人工智能的强大能力和广阔前景 ，也让我们深刻感受到它对我们生活和工作方式的巨大影响 。

##### **表5：人工智能的实际应用场景**

| **领域** | **应用案例** |
| --- | --- |
| **医疗** | IBM Watson辅助癌症诊断、医学影像分析。 |
| **交通** | 特斯拉自动驾驶（自动泊车、自适应巡航）。 |
| **金融** | 风险评估（市场趋势预测）、欺诈检测（实时交易监测）。 |
| **教育** | 个性化学习推荐系统、智能教学助手。 |

阅读完这本书后，我深受启发 。在这个信息爆炸的时代，信息如潮水般涌来，我们每天都会接触到海量的信息 。然而，并非所有的信息都是有用的，甚至有些信息可能会误导我们 。这本书让我明白，我们需要具备辨别信息的能力，学会从纷繁复杂的信息中筛选出有价值的部分 ，将其转化为知识 。例如，在面对网络上众说纷纭的观点和信息时，我们不能盲目相信，而应该运用批判性思维，对信息进行分析、判断，去伪存真 。同时，我们也要不断学习新知识，拓宽自己的知识面，提高自己的认知水平 ，只有这样，才能更好地理解和应对这个快速变化的世界 。

对于我们个人而言，在人工智能不断发展的背景下，保持独立思考和学习能力至关重要 。独立思考能力使我们能够在面对人工智能提供的信息和建议时，做出理性的判断和决策 ，不被其左右 。例如，当我们使用智能推荐系统时，不能仅仅依赖系统推荐的内容，而应该思考这些推荐是否真正符合自己的需求和价值观 。学习能力则是我们不断适应时代发展的关键 ，随着人工智能技术的不断更新换代，新的知识和技能层出不穷 ，我们只有持续学习，才能跟上时代的步伐 ，不被淘汰 。我们可以通过阅读书籍、参加培训课程、参与在线学习社区等方式，不断学习新的知识和技能 ，提升自己的竞争力 。

### 总结与推荐

《理解和改变世界 —— 从信息到知识与智能》是一本极具深度和价值的著作 。它通过独特的知识论视角，深入探讨了知识和智能的本质，为我们揭示了信息、知识、智能以及意识之间的复杂关系 。书中对人工智能与人类智能的剖析，让我们对这两种智能形式有了更为清晰的认识 。同时，关于通往通用人工智能路径的阐述，也为人工智能领域的研究和发展提供了重要的理论指导 。

无论是人工智能爱好者、研究者，还是相关从业者，都能从这本书中汲取到丰富的知识和深刻的启示 。对于爱好者而言，它能够帮助我们更好地理解人工智能的内涵和发展方向，激发我们对这一领域的探索热情 ；对于研究者来说，书中的理论和观点为他们的研究提供了新的思路和方法，有助于推动人工智能研究的深入发展 ；对于从业者而言，这本书能帮助他们更好地把握人工智能技术的本质和应用场景，从而在实际工作中做出更明智的决策 。希望更多的人能够阅读这本书，一同探索信息、知识与智能的奇妙世界 。
