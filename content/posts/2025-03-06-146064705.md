# Excel中COUNTIF用法解析

`COUNTIF`
是 Excel 中一个非常实用的函数，用于统计满足某个条件的单元格数量。它的基本语法如下：

#### 基本语法

```
=COUNTIF(范围, 条件)
```

* **范围**
  ：需要统计的单元格区域，例如
  `A1:A10`
  或整列
  `A:A`
  。
* **条件**
  ：用于判断哪些单元格需要被统计的条件，可以是数字、文本或表达式。

#### 常见用法

##### 1. **统计特定文本出现的次数**

如果要统计某列中某个特定文本出现的次数，可以直接写条件。例如：

```
=COUNTIF(D:D, "观塘")
```

* **功能**
  ：统计列
  `D`
  中“观塘”出现的次数。
* **注意**
  ：条件中的文本需要用双引号括起来。

##### 2. **统计满足某个数字条件的单元格**

如果条件是数字或数字范围，可以直接写条件。例如：

```
=COUNTIF(E:E, ">500")
```

* **功能**
  ：统计列
  `E`
  中大于 500 的单元格数量。
* **注意**
  ：条件中的数字范围需要用双引号括起来，例如
  `">500"`
  或
  `"<100"`
  。

##### 3. **统计满足某个日期条件的单元格**

日期也可以作为条件。例如：

```
=COUNTIF(F:F, ">=2024-01-01")
```

* **功能**
  ：统计列
  `F`
  中日期大于或等于 2024 年 1 月 1 日的单元格数量。

##### 4. **使用通配符**

`COUNTIF`
支持使用通配符（
`*`
或
`?`
）来统计满足模糊条件的单元格。例如：

```
=COUNTIF(D:D, "观塘*")
```

* **功能**
  ：统计列
  `D`
  中以“观塘”开头的文本数量。
* **注意**
  ：

  + `*`
    表示任意字符序列。
  + `?`
    表示任意单个字符。

##### 5. **统计非空或空单元格**

可以使用
`*`
或
`""`
来统计非空或空单元格。例如：

```
=COUNTIF(A:A, "*")  // 统计列A中非空单元格的数量
=COUNTIF(A:A, "")   // 统计列A中空单元格的数量
```

#### 注意事项

1. **范围和条件的匹配**
   ：
   `COUNTIF`
   的范围和条件必须匹配。如果范围是数字，条件也应该是数字或数字表达式；如果范围是文本，条件也应该是文本。
2. **大小写不敏感**
   ：
   `COUNTIF`
   对文本条件是大小写不敏感的。例如，“观塘”和“观塘”会被视为相同。
3. **多个条件的统计**
   ：如果需要统计多个条件同时满足的情况，可以使用
   `COUNTIFS`
   函数（带“S”），它支持多个范围和条件。

#### 示例总结

假设你的数据中“地区”在列
`D`
，要统计“观塘”出现的次数：

```
=COUNTIF(D:D, "观塘")
```
