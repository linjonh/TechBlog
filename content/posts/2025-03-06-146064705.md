    <h1 class="title-article" id="articleContentId">
     Excel中COUNTIF用法解析
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <code>
      COUNTIF
     </code>
     是 Excel 中一个非常实用的函数，用于统计满足某个条件的单元格数量。它的基本语法如下：
    </p>
    <h4>
     基本语法
    </h4>
    <p>
    </p>
    <pre><code>=COUNTIF(范围, 条件)</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        范围
       </strong>
       ：需要统计的单元格区域，例如
       <code>
        A1:A10
       </code>
       或整列
       <code>
        A:A
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        条件
       </strong>
       ：用于判断哪些单元格需要被统计的条件，可以是数字、文本或表达式。
      </p>
     </li>
    </ul>
    <h4>
     常见用法
    </h4>
    <h5>
     1.
     <strong>
      统计特定文本出现的次数
     </strong>
    </h5>
    <p>
     如果要统计某列中某个特定文本出现的次数，可以直接写条件。例如：
    </p>
    <p>
    </p>
    <pre><code>=COUNTIF(D:D, "观塘")</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：统计列
       <code>
        D
       </code>
       中“观塘”出现的次数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        注意
       </strong>
       ：条件中的文本需要用双引号括起来。
      </p>
     </li>
    </ul>
    <h5>
     2.
     <strong>
      统计满足某个数字条件的单元格
     </strong>
    </h5>
    <p>
     如果条件是数字或数字范围，可以直接写条件。例如：
    </p>
    <p>
    </p>
    <pre><code>=COUNTIF(E:E, "&gt;500")</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：统计列
       <code>
        E
       </code>
       中大于 500 的单元格数量。
      </p>
     </li>
     <li>
      <p>
       <strong>
        注意
       </strong>
       ：条件中的数字范围需要用双引号括起来，例如
       <code>
        "&gt;500"
       </code>
       或
       <code>
        "&lt;100"
       </code>
       。
      </p>
     </li>
    </ul>
    <h5>
     3.
     <strong>
      统计满足某个日期条件的单元格
     </strong>
    </h5>
    <p>
     日期也可以作为条件。例如：
    </p>
    <p>
    </p>
    <pre><code>=COUNTIF(F:F, "&gt;=2024-01-01")</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：统计列
       <code>
        F
       </code>
       中日期大于或等于 2024 年 1 月 1 日的单元格数量。
      </p>
     </li>
    </ul>
    <h5>
     4.
     <strong>
      使用通配符
     </strong>
    </h5>
    <p>
     <code>
      COUNTIF
     </code>
     支持使用通配符（
     <code>
      *
     </code>
     或
     <code>
      ?
     </code>
     ）来统计满足模糊条件的单元格。例如：
    </p>
    <pre><code>=COUNTIF(D:D, "观塘*")</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：统计列
       <code>
        D
       </code>
       中以“观塘”开头的文本数量。
      </p>
     </li>
     <li>
      <p>
       <strong>
        注意
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <code>
          *
         </code>
         表示任意字符序列。
        </p>
       </li>
       <li>
        <p>
         <code>
          ?
         </code>
         表示任意单个字符。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     5.
     <strong>
      统计非空或空单元格
     </strong>
    </h5>
    <p>
     可以使用
     <code>
      *
     </code>
     或
     <code>
      ""
     </code>
     来统计非空或空单元格。例如：
    </p>
    <pre><code>=COUNTIF(A:A, "*")  // 统计列A中非空单元格的数量
=COUNTIF(A:A, "")   // 统计列A中空单元格的数量</code></pre>
    <h4>
     注意事项
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        范围和条件的匹配
       </strong>
       ：
       <code>
        COUNTIF
       </code>
       的范围和条件必须匹配。如果范围是数字，条件也应该是数字或数字表达式；如果范围是文本，条件也应该是文本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        大小写不敏感
       </strong>
       ：
       <code>
        COUNTIF
       </code>
       对文本条件是大小写不敏感的。例如，“观塘”和“观塘”会被视为相同。
      </p>
     </li>
     <li>
      <p>
       <strong>
        多个条件的统计
       </strong>
       ：如果需要统计多个条件同时满足的情况，可以使用
       <code>
        COUNTIFS
       </code>
       函数（带“S”），它支持多个范围和条件。
      </p>
     </li>
    </ol>
    <h4>
     示例总结
    </h4>
    <p>
     假设你的数据中“地区”在列
     <code>
      D
     </code>
     ，要统计“观塘”出现的次数：
    </p>
    <pre><code>=COUNTIF(D:D, "观塘")</code></pre>
