    <p>
     使用 OpenSSL 生成的 RSA 私钥文件（如
     <code>
      prikey.pem
     </code>
     ）可以用于加密和解密数据。以下是一个完整的示例，展示如何使用 OpenSSL 命令行工具和私钥文件对数据进行加密和解密。
    </p>
    <p>
     1.生成私钥和公钥
    </p>
    <p>
     首先，生成一个 RSA 私钥文件
     <code>
      prikey.pem
     </code>
     ，并从中提取公钥文件
     <code>
      pubkey.pem
     </code>
     。
    </p>
    <p>
     生成私钥
    </p>
    <pre><code class="prism language-bash">openssl genrsa <span class="token parameter variable">-out</span> prikey.pem <span class="token number">1024</span>
</code></pre>
    <p>
     提取公钥
    </p>
    <pre><code class="prism language-bash">openssl rsa <span class="token parameter variable">-in</span> prikey.pem <span class="token parameter variable">-pubout</span> <span class="token parameter variable">-out</span> pubkey.pem
</code></pre>
    <p>
     2.使用公钥加密数据
    </p>
    <p>
     假设你有一个需要加密的文件
     <code>
      plaintext.txt
     </code>
     ，你可以使用公钥对其进行加密。
    </p>
    <p>
     创建一个明文文件
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token string">"This is a secret message"</span> <span class="token operator">&gt;</span> plaintext.txt
</code></pre>
    <p>
     使用公钥加密
    </p>
    <pre><code class="prism language-bash">openssl rsautl <span class="token parameter variable">-encrypt</span> <span class="token parameter variable">-pubin</span> <span class="token parameter variable">-inkey</span> pubkey.pem <span class="token parameter variable">-in</span> plaintext.txt <span class="token parameter variable">-out</span> encrypted.bin
</code></pre>
    <p>
     这将生成一个加密后的二进制文件
     <code>
      encrypted.bin
     </code>
     。
    </p>
    <p>
     3.使用私钥解密数据
    </p>
    <p>
     使用私钥对加密后的数据进行解密。
    </p>
    <p>
     使用私钥解密
    </p>
    <pre><code class="prism language-bash">openssl rsautl <span class="token parameter variable">-decrypt</span> <span class="token parameter variable">-inkey</span> prikey.pem <span class="token parameter variable">-in</span> encrypted.bin <span class="token parameter variable">-out</span> decrypted.txt
</code></pre>
    <p>
     这将生成一个解密后的文件
     <code>
      decrypted.txt
     </code>
     。
    </p>
    <p>
     4.验证解密结果
    </p>
    <p>
     检查解密后的文件内容是否与原始明文相同。
    </p>
    <pre><code class="prism language-bash"><span class="token function">cat</span> decrypted.txt
</code></pre>
    <p>
     输出应为：
    </p>
    <pre><code>This is a secret message
</code></pre>
    <hr/>
    <p>
     完整示例代码
    </p>
    <p>
     以下是一个完整的示例脚本，展示如何生成密钥对、加密和解密数据：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 生成私钥</span>
openssl genrsa <span class="token parameter variable">-out</span> prikey.pem <span class="token number">1024</span>

<span class="token comment"># 提取公钥</span>
openssl rsa <span class="token parameter variable">-in</span> prikey.pem <span class="token parameter variable">-pubout</span> <span class="token parameter variable">-out</span> pubkey.pem

<span class="token comment"># 创建明文文件</span>
<span class="token builtin class-name">echo</span> <span class="token string">"This is a secret message"</span> <span class="token operator">&gt;</span> plaintext.txt

<span class="token comment"># 使用公钥加密</span>
openssl rsautl <span class="token parameter variable">-encrypt</span> <span class="token parameter variable">-pubin</span> <span class="token parameter variable">-inkey</span> pubkey.pem <span class="token parameter variable">-in</span> plaintext.txt <span class="token parameter variable">-out</span> encrypted.bin

<span class="token comment"># 使用私钥解密</span>
openssl rsautl <span class="token parameter variable">-decrypt</span> <span class="token parameter variable">-inkey</span> prikey.pem <span class="token parameter variable">-in</span> encrypted.bin <span class="token parameter variable">-out</span> decrypted.txt

<span class="token comment"># 验证解密结果</span>
<span class="token function">cat</span> decrypted.txt
</code></pre>
    <p>
     运行上述脚本后，你将看到
     <code>
      decrypted.txt
     </code>
     文件的内容与原始明文相同。
    </p>
    <hr/>
    <p>
     注意事项
    </p>
    <p>
     • 密钥长度：建议使用更高的密钥长度（如 2048 或 4096 位）以提高安全性。
    </p>
    <p>
     • 数据大小限制：RSA 加密对数据大小有限制，通常不超过密钥长度。对于大文件，建议使用对称加密（如 AES）结合 RSA 加密。
    </p>
    <p>
     • 安全性：私钥文件（
     <code>
      prikey.pem
     </code>
     ）应妥善保管，避免泄露。
    </p>
    <p>
     通过上述步骤，你可以使用 OpenSSL 和 RSA 密钥对数据进行加密和解密。
    </p>
