    <h3 style="background-color:transparent">
     <strong>
      1. 图像形成的基本原理
     </strong>
    </h3>
    <p>
     图像形成是物理世界与传感器（如胶片、CCD/CMOS）交互的过程，核心是光线的传播与记录。
    </p>
    <h6 style="background-color:transparent">
     <strong>
      1.1 直接放置胶片模型
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        物理原理
       </strong>
       ：物体表面反射的光线直接照射到胶片上，但无任何遮挡或聚焦机制。
      </p>
     </li>
     <li>
      <p>
       <strong>
        问题
       </strong>
       ：所有物体点的光线会覆盖整个胶片，导致完全模糊（如图1）。
      </p>
     </li>
     <li>
      <img alt="" height="208" src="https://i-blog.csdnimg.cn/direct/5b16327e27ef4c22ae069f55fc073c8b.png" width="912"/>
     </li>
    </ul>
    <h6>
     <strong>
      1.2 针孔相机模型
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        改进
       </strong>
       ：在物体与胶片间加入针孔屏障，仅允许通过小孔的光线到达胶片（图2）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        关键参数
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          孔径大小（d）
         </strong>
         ：孔径越小，成像越清晰但亮度越低（需权衡信噪比）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          焦距（f）
         </strong>
         ：针孔到胶片的距离，决定成像比例。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <img alt="" height="347" src="https://i-blog.csdnimg.cn/direct/b083fa028e5c4546b87a943b4d1146f5.png" width="925">
     </img>
    </p>
    <h6>
     <strong>
      1.3 透镜模型
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        原理
       </strong>
       ：透镜通过折射集中光线，替代针孔以提高亮度（图3）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        薄透镜公式
       </strong>
       ：
      </p>
      1/f=1/u+1/v​
      <ul>
       <li>
        <p>
         u：物距（物体到透镜的距离）
        </p>
       </li>
       <li>
        <p>
         v：像距（透镜到成像平面的距离）
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        景深（Depth of Field）
       </strong>
       ：透镜允许一定范围内的物体清晰成像，与光圈大小成反比。
      </p>
     </li>
    </ul>

     <strong>
      3. 颜色空间与模型
     </strong>
    </h3>
    <h6>
     <strong>
      3.1 RGB颜色空间
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        定义
       </strong>
       ：通过红、绿、蓝三原色的叠加表示颜色（图6）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        缺点
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          通道强相关
         </strong>
         ：调整亮度需同时改变所有通道。
        </p>
       </li>
       <li>
        <p>
         <strong>
          非线性感知
         </strong>
         ：人眼对亮度的响应接近对数尺度，而RGB是线性的。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <h6>
     <strong>
      3.2 HSV/HSL颜色空间
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        组成
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          H（色相）
         </strong>
         ：颜色类型（0°-360°）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          S（饱和度）
         </strong>
         ：颜色纯度（0%-100%）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          V（明度）/L（亮度）
         </strong>
         ：颜色明暗程度。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        应用
       </strong>
       ：图像编辑软件（如Photoshop）中的颜色选择器。
      </p>
     </li>
    </ul>
    <h6>
     <strong>
      3.3 YCbCr颜色空间
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        分离亮度与色度
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          Y（亮度）
         </strong>
         ：Y=0.299R+0.587G+0.114B
        </p>
       </li>
       <li>
        <p>
         <strong>
          Cb（蓝色色差）
         </strong>
         ：Cb=0.564(B−Y)+128
        </p>
       </li>
       <li>
        <p>
         <strong>
          Cr（红色色差）
         </strong>
         ：Cr=0.713(R−Y)+128
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        压缩优势
       </strong>
       ：人眼对亮度更敏感，可对Cb/Cr进行子采样（如4:2:0）。
      </p>
     </li>
    </ul>

    <h3 style="background-color:transparent">
     <strong>
      4. 数字图像形成技术
     </strong>
    </h3>
    <h6>
     <strong>
      4.1 空间采样（Spatial Sampling）
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        奈奎斯特采样定理
       </strong>
       ：采样频率需大于信号最高频率的2倍，避免混叠（图7）。
      </p>
      <ul>
       <li>
        <p>
         <strong>
          示例
         </strong>
         ：若图像最高空间频率为100 cycles/mm，则采样间隔需小于0.005 mm。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        像素阵列
       </strong>
       ：
      </p>
      x=j⋅Δx,y=k⋅Δy(j,k 为整数)
     </li>
    </ul>
    <h6>
     <strong>
      4.2 量化（Quantization）
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        位深度与灰度级
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          8位
         </strong>
         ：256级（JPEG标准）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          12位
         </strong>
         ：4096级（医学影像）。
        </p>
       </li>
       <li>
        <p>
         <strong>
          16位
         </strong>
         ：65536级（专业摄影）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        量化误差
       </strong>
       ：
      </p>
      误差=动态范围2n(n=位深度)
     </li>
    </ul>
    <h6>
     <strong>
      4.3 分辨率与存储权衡
     </strong>
    </h6>
    <ul>
     <li>
      <p>
       <strong>
        空间分辨率
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          人脸识别
         </strong>
         ：64×64像素（约0.1MP）可满足基本需求。
        </p>
       </li>
       <li>
        <p>
         <strong>
          高清显示
         </strong>
         ：1920×1080像素（2MP）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        存储计算
       </strong>
       ：
      </p>
      单张图像大小=宽×高×通道数×位深度单张图像大小=宽×高×通道数×位深度
      <ul>
       <li>
        <p>
         <strong>
          示例
         </strong>
         ：24位彩色图（3通道×8位），1024×768像素：
        </p>
        1024×768×3×8=18,874,368 bits=2.36 MB
       </li>
      </ul>
     </li>
    </ul>
    <h5>
    </h5>
    <h3>
     <strong>
      5. 空间域与变换域操作
     </strong>
    </h3>
    <ul>
     <li>
      <ul>
       <li>
        <p>
         <strong>
          定义
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           <strong>
            图像处理
           </strong>
           ：输入图像 → 输出图像，目标是抑制失真、增强有用信息（如对比度调整、去噪）。
          </p>
         </li>
         <li>
          <p>
           <strong>
            图像分析
           </strong>
           ：输入图像 → 提取特征/测量（如边缘检测、目标计数）。
          </p>
         </li>
         <li>
          <p>
           <strong>
            计算机视觉
           </strong>
           ：输入图像 → 语义理解（如目标识别、场景重建）。
          </p>
         </li>
        </ul>
       </li>
      </ul>
      <ul>
       <li>
        <p>
         <strong>
          空间域操作
         </strong>
         ：直接在像素值上进行处理，分为两类：
        </p>
        <ul>
         <li>
          <p>
           <strong>
            点操作
           </strong>
           ：单个像素独立变换，公式：
          </p>
          g(x,y)=T(f(x,y))（如对比度拉伸、反转）g(x,y)=T(f(x,y))（如对比度拉伸、反转）
         </li>
         <li>
          <p>
           <strong>
            邻域操作
           </strong>
           ：基于像素周围区域计算，公式：
          </p>
          g(x,y)=T(f(x,y),f(x+1,y),f(x−1,y),… )（如均值滤波）g(x,y)=T(f(x,y),f(x+1,y),f(x−1,y),…)（如均值滤波）
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>

     10.
     <strong>
      总结
     </strong>

    <blockquote>
     <ul>
      <li>
       <p>
        <strong>
         图像形成
        </strong>
        ：从物理模型（针孔、透镜）到数学建模（透视/仿射投影）。

       </p>
      </li>
      <li>
       <p>
        <strong>
         数字化技术
        </strong>
        ：采样与量化需平衡质量与效率，避免信息损失。
       </p>
      </li>
      <li>
       <p>
        <strong>
         点操作
        </strong>
        ：直接修改像素值，适合对比度调整和二值化。
       </p>
      </li>
      <li>
       <p>
        <strong>
         直方图处理
        </strong>
        ：通过分布调整优化全局或局部对比度。
       </p>
      </li>
      <li>
       <p>
        <strong>
         算术操作
        </strong>
        ：多用于图像融合与噪声抑制。
       </p>
      </li>
      <li>
       <p>
        <strong>
         核心公式与算法
        </strong>
        ：如Otsu阈值、直方图均衡化需熟练掌握推导与应用。
       </p>
      </li>
     </ul>
