     我们来新建一个的蓝图模块-班级模块，后面可以和学生模块，实现一对多的数据库操作。

    <p>
     blueprint下新建grade.py
    </p>
    <p>
     <img alt="" height="144" src="https://i-blog.csdnimg.cn/direct/b36e5d31041b41c8b90354cba91549ca.png" width="307"/>

    <p>
     创建蓝图对象，以及定义班级模型，和班级类。
    </p>
    <pre><code class="language-python">from flask import Blueprint, render_template

from exts import db

# 创建用户蓝图对象
grade_bp = Blueprint("grade", __name__, url_prefix='/grade')


class GradeModel(db.Model):
    """
    班级模型
    """
    __tablename__ = "t_grade"
    id = db.Column(db.Integer, primary_key=True, autoincrement=True)  # 主键id
    name = db.Column(db.String(50), nullable=False)  # 班级名称
    remark = db.Column(db.Text)  # 备注


class Grade:
    def __init__(self, name, remark):
        self.name = name
        self.remark = remark</code></pre>
    <p>
     app.py里面注册蓝图
    </p>
    <pre><code class="language-python">app.register_blueprint(grade_bp)</code></pre>
    <p>
     SQLAlchemy模型提供query对象的all()方法能够查询出所有数据，定义index视图方法，获取数据，转发到模版页面。
    </p>
    <pre><code class="language-python">@grade_bp.route('/list')
def index():
    gradeList = GradeModel.query.all()
    return render_template('grade/list.html', gradeList=gradeList)</code></pre>
    <p>
     templates下新建grade目录，grade目录下新建list.html页面。
