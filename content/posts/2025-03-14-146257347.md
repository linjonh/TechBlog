    <p>
     在 Java 中，泛型通配符（
     <code>
      ?
     </code>
     ）用于表示未知类型，通常用于增强泛型的灵活性。通配符可以与上下限结合使用，以限制泛型的范围。以下是通配符及上下限的使用示例：

     <code>

     </code>
     或其子类。

     3.
     <strong>
      下界通配符 (
      <code>
       &lt;? super T&gt;
      </code>
      )
     </strong>
    </h4>
    <p>
     下界通配符表示泛型类型必须是
     <code>
      T
     </code>
     或其父类。

    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">addNumbers</span><span class="token punctuation">(</span><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token operator">?</span> <span class="token keyword">super</span> <span class="token class-name">Integer</span><span class="token punctuation">&gt;</span></span> list<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

      示例调用：
      <pre><code class="prism language-java"><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token class-name">Integer</span><span class="token punctuation">&gt;</span></span> srcList <span class="token operator">=</span> <span class="token class-name">Arrays</span><span class="token punctuation">.</span><span class="token function">asList</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <hr/>
    <h4>
     <a id="_97">
     </a>
     总结：
    </h4>
    <ol>
     <li>
      <strong>
       <code>
        ?
       </code>
      </strong>
      ：无界通配符，接受任意类型。
     </li>
     <li>
      <strong>
       <code>
        &lt;? extends T&gt;
       </code>
      </strong>
      ：上界通配符，限制为
      <code>
       T
      </code>
      或其子类。
     </li>
     <li>
      <strong>
       <code>
        &lt;? super T&gt;
       </code>
      </strong>
      ：下界通配符，限制为
      <code>
       T
      </code>
      或其父类。
     </li>
    </ol>
    <p>
     通过合理使用通配符和上下限，可以增强泛型的灵活性，同时保证类型安全。
    </p>
