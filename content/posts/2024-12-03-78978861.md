    <h1 class="title-article" id="articleContentId">
     ubuntu16.04升级python2.7x版本到python3.5版本
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <a href="https://www.cnblogs.com/wmr95/p/7637077.html" rel="nofollow noopener noreferrer" target="_blank">
      <u>
       <span style="color:#80080">
        参考:https://www.cnblogs.com/wmr95/p/7637077.html
       </span>
      </u>
     </a>
    </p>
    <p align="justify">
     一般情况下
     <span style="font-family:Calibri">
      ubuntu16.04
     </span>
     <span style="font-family:宋体">
      都会自动安装
     </span>
     <span style="font-family:Calibri">
      python2.7
     </span>
     <span style="font-family:宋体">
      的版本，在此尝试将其升级为
     </span>
     <span style="font-family:Calibri">
      python3.5
     </span>
     <span style="font-family:宋体">
      版本
     </span>
    </p>
    <p align="justify">
     一：打开
     <span style="font-family:Calibri">
      ubuntu
     </span>
     <span style="font-family:宋体">
      的终端
     </span>
     在命令行输入命令：
    </p>
    <p align="justify">
     python
    </p>
    <p align="justify">
     显示如下，说明系统自带的
     <span style="font-family:Calibri">
      python
     </span>
     <span style="font-family:宋体">
      版本为
     </span>
     <span style="font-family:Calibri">
      python2.7.12
     </span>
    </p>
    <p align="justify">
     <img alt="" src="https://img-blog.csdn.net/20180105110734226?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvd2VpeGluXzQxMjQ3NzYw/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
      二：在命令行输入命令：
     </img>
    </p>
    <p align="justify">
     sudo apt-get install python3
     <span style="font-family:宋体">
      （在此安装
     </span>
     <span style="font-family:Calibri">
      python3.5,
     </span>
     <span style="font-family:宋体">
      命令为：
     </span>
     <span style="font-family:Calibri">
      sudo apt-get install python3.5
     </span>
     <span style="font-family:宋体">
      ）
     </span>
    </p>
    <p align="justify">
     输入你的密码后会下载，刚才下载的
     <span style="font-family:Calibri">
      Python
     </span>
     <span style="font-family:宋体">
      程序被安装在
     </span>
     <span style="font-family:Calibri">
      usr/localb/python3.5
     </span>
     <span style="font-family:宋体">
      中。
     </span>
    </p>
    <p align="justify">
     <img alt="" src="https://img-blog.csdn.net/20180105110712626?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvd2VpeGluXzQxMjQ3NzYw/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
      三：指定默认打开的是
      <span style="font-family:Calibri">
       python3.5
      </span>
      <span style="font-family:宋体">
       版本（你新安装的
      </span>
      <span style="font-family:Calibri">
       python
      </span>
      <span style="font-family:宋体">
       版本）。
      </span>
     </img>
    </p>
    <p align="justify">
     安装完成之后，你在终端中输入
     <span style="font-family:Calibri">
      python
     </span>
     <span style="font-family:宋体">
      ，输出的信息里面会提示是
     </span>
     <span style="font-family:Calibri">
      2.7
     </span>
     <span style="font-family:宋体">
      版本的，也就是说默认打开的并不是刚才安装好的
     </span>
     <span style="font-family:Calibri">
      3.5
     </span>
     <span style="font-family:宋体">
      ，所以还需要我们重新修改一下链接。方法如下：
     </span>
    </p>
    <p align="justify">
     第一步：先备份原来的链接（在对系统执行删除之前进行备份是个好的习惯）。在
     <span style="font-family:Calibri">
      ternimal
     </span>
     <span style="font-family:宋体">
      下输入命令：
     </span>
    </p>
    <p align="justify">
     <span style="font-family:Calibri">
      sudo cp /usr/bin/python /usr/bin/python_bak
     </span>
    </p>
    <p align="justify">
     第二步：删除原来默认指向
     <span style="font-family:Calibri">
      python2.7
     </span>
     <span style="font-family:宋体">
      版本的链接。在
     </span>
     <span style="font-family:Calibri">
      ternimal
     </span>
     <span style="font-family:宋体">
      下输入命令：
     </span>
    </p>
    <p align="justify">
     <span style="font-family:Calibri">
      sudo rm /usr/bin/python
     </span>
    </p>
    <p align="justify">
     第三步：重新指定新的链接给
     <span style="font-family:Calibri">
      python3.5
     </span>
     <span style="font-family:宋体">
      版本。输入命令：
     </span>
    </p>
    <p align="justify">
     <span style="font-family:Calibri">
      sudo ln -s /usr/bin/python3.5 /usr/bin/python
     </span>
    </p>
    <p align="justify">
     至此，
     <span style="font-family:Calibri">
      python
     </span>
     <span style="font-family:宋体">
      版本更新已经完成。
     </span>
    </p>
    <p align="justify">
     <img alt="" src="https://img-blog.csdn.net/20180105110748425?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvd2VpeGluXzQxMjQ3NzYw/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
      注：
      <span style="font-family:Calibri">
       python2.7
      </span>
      <span style="font-family:宋体">
       和
      </span>
      <span style="font-family:Calibri">
       3.5
      </span>
      <span style="font-family:宋体">
       版本之间随意切换（这里
      </span>
      <span style="font-family:Calibri">
       3.5
      </span>
      <span style="font-family:宋体">
       切换回
      </span>
      <span style="font-family:Calibri">
       2.7
      </span>
      <span style="font-family:宋体">
       版本）：
      </span>
     </img>
    </p>
    <p align="justify">
     <span style="font-family:Calibri">
      sudo rm /usr/bin/python
     </span>
    </p>
    <p align="justify">
     <span style="font-family:Calibri">
      sudo ln -s /usr/bin/python2.7 /usr/bin/python
     </span>
    </p>
