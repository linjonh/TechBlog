    <br/>
    mac下的alfred是非常强大的搜索工具
    <p>
    </p>
    <h2>
     <a id="_2">
     </a>
     设置偏好设置
    </h2>
    <p>
     alfred默认双击 command，在弹出框输入文字，默认是google搜索，设置百度搜索方法如下：
     <br/>
     首先双击command，呼出alfred，在图标上选择右键，选择Preferences偏好设置,或者直接command + , 快捷键呼出偏好设置，如下图
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d5149aa6dff2bd4d38816713e39c750d.png"/>
    </p>
    <h2>
     <a id="Add_Custom_Search_8">
     </a>
     设置Add Custom Search
    </h2>
    <p>
     在窗口里面选择Web Search 再点击右下角的 Add Custom Search，如下图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4561d4cd3433e0a5ccd512d54d57efc.png">
      <br/>
      在弹出框里面的 Search URL:里面填写上：https://www.baidu.com/s?wd={query}
      <br/>
      这样就可以在之后的搜索的时候把你输入的关键字直接填入到baidu里面
      <br/>
      Title可以随便起名，用作标题
      <br/>
      keyword尽量填写短一点，这里我填写的是：b，这样之后我在alfred里面直接输入b空格+关键字也能使用，
      <br/>
      其他的就默认就行，然后点击右下角的save如下图
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d587583bb51b31d540263e17ca4beb43.png">
       <br/>
       选择 左上角目录 ：Default Results ,再在右下角点击 Setup fallback results，如下图
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7f29b62c33431ae81ef7279c3c1be9a1.png">
        <br/>
        点击加号
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ea97e2fe8ccd20229c1ff5994a59fd38.png">
         <br/>
         选择Custom Search 选择刚才的Title标题，我设置的是百度搜索，这里选中
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7f6d462dfb56f99a4f8032a39841bd45.png"/>
    </p>
    <p>
     选中以后,默认百度搜索排在最后面，拖拽到最上面，然后删除掉多余的，只留下百度和google,这样，以后默认搜索百度搜索就排在最上面，点击save如图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dee072a70ce53cd51bd79d08997be0a4.png"/>
    </p>
    <p>
     然后我们双击command 输入一个比较长的文字，比如 :小手琴师的博客 ，按下回车，就会打开浏览器自动在百度搜索：小手琴师的博客，
     <br/>
     如果不能，注意看之前的步骤（在Search URL:里面填写上：https://www.baidu.com/s?wd={query}）
     <br/>
     如果按下command+2，就会默认搜索谷歌
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1de8ea00efdd08a1920ea847ffcc7f0f.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6d92f5c717fc0a9353352ecf9790a1b6.png"/>
     </img>
    </p>
    <h2>
     <a id="_36">
     </a>
     输入文字太短，使用快捷键呼出百度搜索
    </h2>
    <p>
     有时候输入文字太短，比如输入123,那么就会弹出计算器，和其他文件搜索，的123的结果，如下图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f9fd1b572b97051d3e42a8b9d3352e86.png">
      <br/>
      这时候使用快捷键,刚才我设置的是快捷键keyword是b
      <br/>
      输入：b 空格 + 123，或者直接b123，就会在百度搜索123，
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c1f09f485cc3ace48fd0e8648ac54d19.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/027f7176396e82f620c344f1b9291bb2.png"/>
      <br/>
      如果alfred区分不开字符，比如连续输入bb，就中间加空格，如下图
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d3c681b8e143b1f24472fb39da8f74f6.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/91535506fb0c7369d0bdd57b20f434f6.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5dcf41d7d1c7aeedfb3e69e14c1583e7.png"/>
     </img>
    </p>
