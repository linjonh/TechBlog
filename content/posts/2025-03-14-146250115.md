    <h4>
     <a id="_2">
     </a>
     <strong>
      预训练与微调
     </strong>
    </h4>
    <h5>
     <a id="1__4">
     </a>
     <strong>
      1. 预训练
     </strong>
    </h5>
    <p>
     •
     <strong>
      目标
     </strong>
     ：通过大规模数据训练模型，学习通用表示能力。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 自监督学习（如BERT、GPT）
     <br/>
     • 多模态预训练（如CLIP、DALL·E）
    </p>
    <hr/>
    <h5>
     <a id="2__12">
     </a>
     <strong>
      2. 微调
     </strong>
    </h5>
    <p>
     微调是在预训练模型基础上，针对特定任务或领域进行优化。主要分为以下几类：
    </p>
    <h6>
     <a id="21__15">
     </a>
     <strong>
      2.1 多模态模型微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：适配多模态任务（如图文生成、视觉问答）。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 跨模态对齐微调
     <br/>
     • 多模态联合训练
    </p>
    <h6>
     <a id="22__21">
     </a>
     <strong>
      2.2 文本模型微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：优化文本相关任务（如文本生成、分类）。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 监督微调
     <br/>
     • 无监督/自监督微调
    </p>
    <h6>
     <a id="23__27">
     </a>
     <strong>
      2.3 监督微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：使用标注数据优化特定任务。
     <br/>
     •
     <strong>
      细分
     </strong>
     ：
     <br/>
     •
     <strong>
      指令微调
     </strong>
     ：通过指令-输出对增强泛化能力。
     <br/>
     •
     <strong>
      对话微调
     </strong>
     ：优化对话生成和上下文理解。
     <br/>
     •
     <strong>
      领域适配
     </strong>
     ：将模型适配到特定领域（如医疗、法律）。
     <br/>
     •
     <strong>
      文本分类
     </strong>
     ：优化文本分类任务（如情感分析、主题分类）。
    </p>
    <h6>
     <a id="24__35">
     </a>
     <strong>
      2.4 无监督/自监督微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：利用未标注数据提升模型性能。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 掩码语言模型（如BERT）
     <br/>
     • 对比学习（如SimCSE）
    </p>
    <h6>
     <a id="25__41">
     </a>
     <strong>
      2.5 强化学习微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：通过奖励机制优化模型输出。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 人类反馈强化学习（RLHF）
     <br/>
     • 直接偏好优化（DPO）
    </p>
    <h6>
     <a id="26__47">
     </a>
     <strong>
      2.6 特殊的微调
     </strong>
    </h6>
    <p>
     •
     <strong>
      目标
     </strong>
     ：针对特定需求或场景进行优化。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 知识蒸馏：将大模型知识迁移到小模型。
     <br/>
     • 增量学习：逐步适配新任务或数据。
    </p>
    <hr/>
    <h5>
     <a id="3__55">
     </a>
     <strong>
      3. 知识终端
     </strong>
    </h5>
    <p>
     •
     <strong>
      目标
     </strong>
     ：将外部知识整合到模型中。
     <br/>
     •
     <strong>
      方法
     </strong>
     ：
     <br/>
     • 知识图谱嵌入
     <br/>
     • 检索增强生成（RAG）
    </p>
