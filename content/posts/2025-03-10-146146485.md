    <h3>
     <a id="1ubuntu24mysql_37">
     </a>
     1.ubuntu24在线安装mysql
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> upgrade <span class="token parameter variable">-y</span>
</code></pre>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> mysql-server <span class="token parameter variable">-y</span>
</code></pre>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl start mysql
<span class="token function">sudo</span> systemctl <span class="token builtin class-name">enable</span> mysql
</code></pre>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> systemctl status mysql
</code></pre>
    <h3>
     <a id="2_57">
     </a>
     2.运行安全配置脚本
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> mysql_secure_installation
</code></pre>
    <p>
     <em>
      根据提示操作：
      <br/>
      选择密码强度验证策略（默认按回车跳过）。
      <br/>
      设置root用户的密码。
      <br/>
      移除匿名用户（Y）。
      <br/>
      禁止远程root登录（Y）。
      <br/>
      删除测试数据库（Y）。
      <br/>
      立即应用更改（Y）。
     </em>
    </p>
    <h3>
     <a id="3MySQL_70">
     </a>
     3.验证MySQL安装
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> mysql <span class="token parameter variable">-u</span> root <span class="token parameter variable">-p</span>
</code></pre>
    <h3>
     <a id="4_76">
     </a>
     4.配置远程访问（可选）
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">vi</span> /etc/mysql/mysql.conf.d/mysqld.cnf
</code></pre>
    <p>
     <em>
      修改：bind-address = 0.0.0.0 # 允许所有IP访问
     </em>
    </p>
    <pre><code class="prism language-bash"><span class="token comment">#创建远程用户并授权：</span>
