    <h2 id="%E5%B9%BF%E5%BA%A6%E4%BC%98%E5%85%88%E9%81%8D%E5%8E%86%EF%BC%88%E5%B1%82%E5%BA%8F%E9%81%8D%E5%8E%86%EF%BC%89">
     广度优先遍历（层序遍历）
    </h2>
    <h3 id="%E4%B8%80%E3%80%81%E4%BB%80%E4%B9%88%E6%98%AF%E5%B9%BF%E5%BA%A6%E4%BC%98%E5%85%88%E9%81%8D%E5%8E%86%EF%BC%88BFS%EF%BC%89">
     一、什么是广度优先遍历（BFS）
    </h3>
    <p>
     广度优先遍历（Breadth-First Search，BFS）是一种图形搜索算法，从图的某一特定顶点出发，首先访问其所有邻接顶点，然后再依次访问这些邻接顶点的邻接顶点，如此一层一层地向外扩展，直到访问完所有顶点。在二叉树等数据结构中，BFS 通常按照从根节点开始，一层一层地横向遍历节点。
    </p>
    <h3 id="%E4%BA%8C%E3%80%81%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E5%B9%BF%E5%BA%A6%E4%BC%98%E5%85%88%E9%81%8D%E5%8E%86">
     二、为什么要广度优先遍历
    </h3>
    <ol>
     <li>
      探索图或树的结构：可以全面地了解图或树的整体结构，对于需要对整个结构进行分析的问题非常有用。
     </li>
     <li>
      寻找最短路径：在某些情况下，BFS 可以有效地找到从一个起始点到目标点的最短路径。因为 BFS 在访问每个节点时，都是先访问距离起始点最近的节点，再逐渐扩展到更远的节点。
     </li>
     <li>
      分层处理问题：对于具有层次结构的数据，如二叉树的层序遍历，可以清晰地按照层次进行处理，便于理解和分析问题。
     </li>
    </ol>
    <h3 id="%E4%B8%89%E3%80%81%E4%BB%80%E4%B9%88%E6%97%B6%E5%80%99%E7%94%A8BFS">
     三、什么时候用BFS
    </h3>
    <ol>
     <li>
      当需要找到从一个起始点到目标点的最短路径时。
     </li>
     <li>
      对图或树进行分层处理，例如二叉树的层序遍历、图的按层遍历等。
     </li>
     <li>
      当问题可以通过逐层探索来解决时，比如在迷宫问题中，找到从起点到终点的最短路径。
     </li>
    </ol>
    <h3 id="%E5%9B%9B%E3%80%81%E7%AE%80%E5%8D%95%E7%9A%84BFS%E6%BC%94%E7%A4%BA%20%E4%BA%8C%E5%8F%89%E6%A0%91%E7%9A%84%E5%B1%82%E5%BA%8F%E9%81%8D%E5%8E%86">
     四、简单的BFS演示 二叉树的层序遍历
    </h3>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;queue&gt;
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

void levelOrderTraversal(TreeNode* root) {
    if (root == nullptr) return;
    queue&lt;TreeNode*&gt; q;
    q.push(root);
    while (!q.empty()) {
        TreeNode* node = q.front();
        q.pop();
        cout &lt;&lt; node-&gt;val &lt;&lt; " ";
        if (node-&gt;left) q.push(node-&gt;left);
        if (node-&gt;right) q.push(node-&gt;right);
    }
}</code></pre>
    <h3 id="%E4%BA%94%E3%80%81%E9%80%9A%E8%BF%87%E9%98%9F%E5%88%97%20%E5%92%8C%20%E8%AE%B0%E5%BD%95%E5%B1%82%E6%95%B0%20%E6%9D%A5%E8%BF%9B%E8%A1%8C%E5%B1%82%E5%BA%8F%E9%81%8D%E5%8E%86">
     五、通过队列 和 记录层数 来进行层序遍历
    </h3>
    <p>
     这个题目真的很有意义，一定要耐心看完，我觉得肯定会对你有所提升。
    </p>
    <p>
     对于普通的层序遍历，二叉树只是队列入节点，在增加节点，但是这里又要考虑层数的控制，每层数值的比较，是思维更加跳跃灵活的体现。
    </p>
    <p>
     <img alt="" height="1200" src="https://i-blog.csdnimg.cn/direct/e583c6ee25c8432a8818c86e460e05fc.png" width="1058"/>
    </p>
    <p>
     <strong>
      一定要自己先了解题目，尝试怎么根据层序遍历，来控制层数和每一层的数的奇偶比较。
     </strong>
    </p>
    <p>
     解析真的写的很详细了：
    </p>
    <h4 id="%E5%B9%BF%E5%BA%A6%E4%BC%98%E5%85%88%E6%90%9C%E7%B4%A2%EF%BC%9A">
     <strong>
      广度优先搜索：
     </strong>
    </h4>
    <p>
     由于判断一棵二叉树是否为奇偶树的条件是针对同一层的节点，因此可以使用广度优先搜索，每一轮搜索访问同一层的全部节点，且只会访问这一层的节点。
    </p>
    <p>
     使用队列存储节点。初始时，将根节点加入队列。每一轮搜索之前，队列中的节点是同一层的全部节点，记队列的大小为 size，该轮搜索只访问 size 个节点，即可保证该轮搜索访问的恰好是同一层的全部节点。搜索过程中，将当前层的节点的非空子节点依次加入队列，用于下一层的搜索。
    </p>
    <p>
     判断一棵二叉树是否为奇偶树，需要考虑两个条件
     <strong>
      ，一是节点值的奇偶性，二是节点值的单调性，
     </strong>
     这两个条件都由层下标的奇偶性决定。因此，需要维护搜索到的层下标，以及对于
     <strong>
      每一层搜索都需要维护上一个节点值
     </strong>
     。
    </p>
    <p>
     如果当前层下标是偶数，则要求当前层的所有节点的值都是奇数，且节点值从左到右严格递增。如果遇到节点值是偶数，或者当前节点值小于等于上一个节点值，则二叉树一定不是奇偶树。
    </p>
    <p>
     如果当前层下标是奇数，则要求当前层的所有节点的值都是偶数，且节点值从左到右严格递减。如果遇到节点值是奇数，或者当前节点值大于等于上一个节点值，则二叉树一定不是奇偶树。
    </p>
    <p>
     如果二叉树的所有节点都满足奇偶树的条件，则二叉树是奇偶树。
    </p>
    <pre><code class="language-cpp">/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    queue&lt;TreeNode*&gt; q;
    bool isEvenOddTree(TreeNode* root) {
        q.push(root);
        int level=0;//根 0层

        while(!q.empty())
        {
            int size=q.size();
            int prev=level%2==0?INT_MIN:INT_MAX;//偶数层递增 奇数层递减  (同一层的上一个节点值进行比较)
            for(int i=0;i&lt;size;i++)
            {
                TreeNode* node=q.front();
                int value=q.front()-&gt;val;
                q.pop();
                if(level%2==value%2) return false; //层数跟数的奇偶相反
                if((level%2==0&amp;&amp;value&lt;=prev)||(level%2==1&amp;&amp;value&gt;=prev)) return false;
                prev=value;
                if(node-&gt;left) q.push(node-&gt;left);
                if(node-&gt;right) q.push(node-&gt;right);
            }
            level++;
        }
        return true;
    }
};
</code></pre>
