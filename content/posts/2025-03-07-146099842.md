    <h1 class="title-article" id="articleContentId">
     在 Ubuntu 上安装和切换多个 GCC 版本
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="markdown_views prism-atom-one-light" id="content_views">
    <svg style="display: none;" xmlns="http://www.w3.org/2000/svg">
     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     在软件开发过程中，有时需要在不同的 GCC 版本之间进行切换。本文将介绍如何在 Ubuntu 上安装多个 GCC 版本，并使用
     <code>
      update-alternatives
     </code>
     工具进行版本切换。
    </p>
    <h5>
     <a id="1__GCC__3">
     </a>
     1. 安装所需的 GCC 版本
    </h5>
    <p>
     首先，更新包列表并安装所需的 GCC 版本。本文将安装
     <code>
      gcc-7
     </code>
     ,
     <code>
      gcc-9
     </code>
     和
     <code>
      gcc-11
     </code>
     。
    </p>
    <pre><code class="prism language-sh"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> gcc-7 g++-7 gcc-9 g++-9 gcc-11 g++-11
</code></pre>
    <h5>
     <a id="2__updatealternatives__GCC__G_12">
     </a>
     2. 使用
     <code>
      update-alternatives
     </code>
     工具配置 GCC 和 G++
    </h5>
    <p>
     使用
     <code>
      update-alternatives
     </code>
     工具来管理和切换 GCC 和 G++ 版本。
    </p>
    <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-7 <span class="token number">70</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-9 <span class="token number">90</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/gcc gcc /usr/bin/gcc-11 <span class="token number">100</span>

<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/g++ g++ /usr/bin/g++-7 <span class="token number">70</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/g++ g++ /usr/bin/g++-9 <span class="token number">90</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/g++ g++ /usr/bin/g++-11 <span class="token number">100</span>
</code></pre>
    <h5>
     <a id="3__GCC__G__26">
     </a>
     3. 切换 GCC 和 G++ 版本
    </h5>
    <p>
     运行以下命令来选择默认的 GCC 和 G++ 版本：
    </p>
    <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> gcc
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> g++
</code></pre>
    <p>
     系统会显示一个可用 GCC 和 G++ 版本的列表，输入相应的数字选择所需的版本（例如
     <code>
      gcc-7
     </code>
     和
     <code>
      g++-7
     </code>
     ）。
    </p>
    <h5>
     <a id="4__GCC__G__37">
     </a>
     4. 验证 GCC 和 G++ 版本
    </h5>
    <p>
     切换完成后，可以通过以下命令验证当前使用的 GCC 和 G++ 版本：
    </p>
    <pre><code class="prism language-sh">gcc <span class="token parameter variable">--version</span>
g++ <span class="token parameter variable">--version</span>
</code></pre>
    <h5>
     <a id="5__46">
     </a>
     5. 手动设置环境变量（如果需要）
    </h5>
    <p>
     如果
     <code>
      update-alternatives
     </code>
     设置正确，但仍然显示旧版本，可以尝试手动设置环境变量：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        编辑
        <code>
         .bashrc
        </code>
        或
        <code>
         .zshrc
        </code>
        文件
       </strong>
       ：
      </p>
      <pre><code class="prism language-sh"><span class="token function">nano</span> ~/.bashrc
</code></pre>
      <p>
       或者：
      </p>
      <pre><code class="prism language-sh"><span class="token function">nano</span> ~/.zshrc
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        添加以下行来设置 GCC 和 G++ 的路径
       </strong>
       ：
      </p>
      <pre><code class="prism language-sh"><span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span>/usr/bin/gcc-7:<span class="token environment constant">$PATH</span>
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span>/usr/bin/g++-7:<span class="token environment constant">$PATH</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        保存并退出编辑器
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        重新加载配置文件
       </strong>
       ：
      </p>
      <pre><code class="prism language-sh"><span class="token builtin class-name">source</span> ~/.bashrc
</code></pre>
      <p>
       或者：
      </p>
      <pre><code class="prism language-sh"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
     </li>
    </ol>
    <h5>
     <a id="6__83">
     </a>
     6. 验证版本
    </h5>
    <p>
     再次验证 GCC 和 G++ 版本：
    </p>
    <pre><code class="prism language-sh">gcc <span class="token parameter variable">--version</span>
g++ <span class="token parameter variable">--version</span>
</code></pre>
    <h4>
     <a id="_92">
     </a>
     总结
    </h4>
    <p>
     通过上述步骤，你可以在 Ubuntu 上安装和切换多个 GCC 版本。
    </p>
