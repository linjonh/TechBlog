    <h2>
     <a id="RNN_0">
     </a>
     RNN（循环神经网络）
    </h2>
    <blockquote>
     <p>
      RNN, Recurrent Neural Network
     </p>
     <p>
      RNN的特点就在于不同时刻之间的网络是有关联的，隐藏层会将其前一时刻的输出值作为输入参数。
     </p>
    </blockquote>
    <h3>
     <a id="_6">
     </a>
     一、类型：
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        一对一
       </strong>
       ：固定的输入到输出，如
       <strong>
        图像分类
       </strong>
      </p>
      <p>
       一对一的
       <code>
        RNN
       </code>
       和
       <code>
        CNN
       </code>
       差不多。
      </p>
     </li>
     <li>
      <p>
       <strong>
        一对多
       </strong>
       ：固定的输入到序列输出，如
       <strong>
        图像的文字描述、音乐生成
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        多对一
       </strong>
       ：序列输入到输出，如 **情感分析，分类正面负面情绪；单步序列预测，比如根据过去几天数据，预测未来第一天的值 **
      </p>
     </li>
     <li>
      <p>
       <strong>
        多对多
       </strong>
       ：序列输入到序列的输出，如
       <strong>
        机器翻译，称之为编解码网络
       </strong>
       （例如下方图片中的第四个模型，左下角的三红三绿叫做编码器，右上角的三蓝三绿叫做解码器）（有时候蓝色和红色不会共用一个绿色）
      </p>
     </li>
     <li>
      <p>
       <strong>
        同步多对多
       </strong>
       ：同步序列输入到同步输出，如
       <strong>
        文本生成、语言建模，视频每一帧的分类，也称之为序列生成
       </strong>
       ，或者
       <strong>
        命名实体识别（NER）任务中，输入是一个句子，输出是该句子中每个词的实体类别（如人名、地点、组织等）
       </strong>
       。
      </p>
     </li>
    </ul>

    <p>
     答：序列数据前后之间有很强的关联性、并且序列数据的输入输出长度不固定。

     有什么缺点吗？
    </h4>
    <p>
     <span class="katex--inline">

       <span class="katex-mathml">

    <h4>
     <a id="3_41">
     </a>
     3）题外话：文字在计算机中存在的形式：
    </h4>
    <p>
     文字在计算机中以
     <strong>
      编码-存储-渲染
     </strong>
     的链路存在：
    </p>
    <ol>
     <li>
      <strong>
       编码层
      </strong>
      ：ASCII、Unicode等将符号映射为二进制；
     </li>
     <li>
      <strong>
       存储层
      </strong>
      ：二进制数据通过文件系统或数据库管理（纯文本文件（.txt）无格式信息，而格式文本（如.docx）嵌入字体、颜色等元数据）；
     </li>
     <li>
      <strong>
       显示层
      </strong>
      ：字体渲染与图形硬件协同生成可视化结果，系统根据编码查找对应字形，结合字号、颜色等参数生成像素矩阵，最后通过显卡等硬件将其转变换位显示器信号。
     </li>
    </ol>
