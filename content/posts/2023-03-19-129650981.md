    <h3>
     <a id="Anaconda_2">
     </a>
     一、Anaconda介绍
    </h3>
    <p>
     <strong>
      Anaconda
     </strong>
     是一个用于学习和开发Python编程语言的软件包一个由Python编写的集成开发环境（IDE）。它提供了多个命令行工具，如shell、condal等，以及配套的依赖管理器pip。通过使用这些工具和库文件，你可以快速地搭建起自己的Python环境，并进行各种类型的编码实验。与其他 Unix/Linux 命令行界面相比，Anaconda具有更高的可定制性和灵活性，因为它提供了类似于Windows桌面应用程序的图形用户界面（GUI）。
    </p>
    <p>
     <strong>
      anaconda3
     </strong>
     软件包是Anaconda套件的一部分，该套件包含许多实用的工具和库，如
     <u>
      pip_packager、condal-install、jupyterhub
     </u>
     等，这些工具可以帮助你轻松地构建、测试和部署python代码。通过使用Anaconda3，你可以更加高效地进行python学习和开发，并且可以在较短的时间内掌握各种主流框架和技术。
    </p>
    <h4>
     <a id="Anacondapython_8">
     </a>
     Anaconda与python的关系
    </h4>
    <p>
     Anaconda3与python的关系非常密切。作为一个集成开发环境（IDE），Anaconda3可以将许多不同的框架和库连接起来，从而方便地开发和调试Python应用程序。
    </p>
    <p>
     <strong>
      Anaconda包括
     </strong>
     Conda、Python以及一大堆安装好的工具包，比如：
     <a href="https://baike.baidu.com/item/numpy/5678437?fromModule=lemma_inlink" rel="nofollow">
      numpy
     </a>
     、
     <a href="https://baike.baidu.com/item/pandas/17209606?fromModule=lemma_inlink" rel="nofollow">
      pandas
     </a>
     等
    </p>
    <p>
     <strong>
      Miniconda
     </strong>
     包括Conda、
     <a href="https://baike.baidu.com/item/Python/407313?fromModule=lemma_inlink" rel="nofollow">
      Python
     </a>
     ,节约磁盘空间优选
    </p>
    <p>
     <strong>
      conda
     </strong>
     是一个开源的包、环境管理器，可以用于在同一个机器上安装不同版本的软件包及其依赖，并能够在不同的环境之间切换
    </p>
    <h4>
     <a id="Anaconda_18">
     </a>
     Anaconda的优点
    </h4>
    <p>
     通过使用Anaconda，你可以更加高效地进行python学习和开发，并且可以在较短的时间内掌握各种主流框架和技术。
    </p>
    <p>
     首先，
     <strong>
      anaconda具有更高的可定制性和灵活性
     </strong>
     ，因为它提供了类似于Windows桌面应用程序的图形用户界面（GUI）。这意味着你可以更加个性化地设置工具栏、菜单、按钮等元素，并且可以更加方便地进行快捷键操作。同时，Anaconda还提供了丰富的插件和扩展程序，这些可以帮助你更好地构建和测试复杂的 Python 代码。
    </p>
    <p>
     其次，
     <strong>
      anaconda具有更好的语言支持
     </strong>
     。与其他命令行界面相比，Anaconda提供了更加完善的自动化库和示例代码文件，这使得你可以更加轻松地理解和掌握所使用的编程语言和开发环境。同时，Anaconda还提供了许多预装的函数和模块，这些可以帮助你更好地开发和部署项目。
    </p>
    <p>
     最后,
     <strong>
      Anaconda的社区比较活跃
     </strong>
     ，有很多热心的志愿者提供技术支持和答疑解惑。
    </p>
    <p>
     <strong>
      conda将几乎所有的工具、第三方包都当做package对待，甚至包括python和conda自身
     </strong>
     ！因此，conda打破了包管理与环境管理的约束，能非常方便地安装各种版本python、各种
     <a href="https://so.csdn.net/so/search?q=package&amp;spm=1001.2101.3001.7020">
      package
     </a>
     并方便地切换。
    </p>
    <p>
     参考：
     <a href="https://blog.csdn.net/weixin_56197703/article/details/124630222">
      还是搞不懂Anaconda是什么?读这一篇文章就够了
     </a>
    </p>
    <h3>
     <a id="Anaconda3_32">
     </a>
     二、Anaconda3的安装
    </h3>
    <p>
     在Windows环境下安装Anaconda3可以按照以下步骤进行：
    </p>
    <ol>
     <li>
      <p>
       下载Anaconda3软件包。
      </p>
      <p>
       官网下载地址:
       <a href="https://www.anaconda.com/products/distribution" rel="nofollow">
        https://www.anaconda.com/products/distribution
       </a>
      </p>
      <p>
       清华镜像源下载:
       <a href="https://repo.anaconda.com/archive/" rel="nofollow">
        https://repo.anaconda.com/archive/
       </a>
       (注意选择自己对应的版本)
      </p>
     </li>
     <li>
      <p>
       右键下载好的安装包以管理员身份运行，进入安装界面。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0ec6b4b7d16bf84e83b195ef14dd1bd8.png">
        <br/>
        点击 Next &gt; I agree (选择 just me 即可)&gt; Next &gt;
       </img>
      </p>
     </li>
     <li>
      <p>
       选择安装位置，然后点击安装。(安装路径建议使用英文,尽量不要包含空格及其它字符)
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c87ded3e80ea8da488fb8284e65afe70.png">
        <br/>
        ps: 在“Advanced Installation Options”中
        <strong>
         不建议
        </strong>
        勾选“Add Anaconda to my PATH environment variable.”（“添加Anaconda至我的环境变量。”）。因为如果勾选，可能会影响其他程序的使用。
       </img>
      </p>
     </li>
     <li>
      <p>
       等待安装完成即可
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0796b2e4c704328b7e05607816193f69.png">
        <br/>
        装成功后，你会看到“Thanks for installing Anaconda”对话框:
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8a9440c59b67d71aa7edcbe79687413.png">
         <br/>
         ps: 注意安装后系统默认的 Python 会变为随 Anaconda 所安装的 Python 版本。Anaconda 安装后会在系统中创建一个默认的 Python 环境，在命令行下输入
         <code>
          conda list
         </code>
         就可以列出默认环境中已安装的所有包。
        </img>
       </img>
      </p>
     </li>
    </ol>
    <p>
     官方安装教程：
     <a href="https://docs.anaconda.com/anaconda/install/windows/" rel="nofollow">
      https://docs.anaconda.com/anaconda/install/windows/
     </a>
    </p>
    <p>
     命令行输入
     <code>
      conda --version
     </code>
     后出现以下内容说明安装成功
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f129c7e1b296805a0ae2078a284e9ee8.png"/>
    </p>
    <p>
     ‘conda’ 不是内部或外部命令，也不是可运行的程序或批处理文件。解决参考：
    </p>
    <p>
     <a href="https://blog.csdn.net/m0_68744965/article/details/125700817">
      https://blog.csdn.net/m0_68744965/article/details/125700817
     </a>
    </p>
    <h4>
     <a id="_65">
     </a>
     设置国内镜像源
    </h4>
    <p>
     如果需要安装很多packages，你会发现conda下载的速度经常很慢，因为Anaconda.org的服务器在国外,这里我们可以使用清华镜像源加快下载速度,配置如下：
    </p>
    <pre><code class="prism language-shell"><span class="token comment"># 添加Anaconda的清华镜像</span>
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

<span class="token comment"># 设置搜索时显示通道地址</span>
conda config --set show_channel_urls <span class="token function">yes</span>
</code></pre>
    <h3>
     <a id="_77">
     </a>
     三、基本用法
    </h3>
    <p>
     命令基本用法：
    </p>
    <pre><code class="prism language-shell">usage: conda-script.py <span class="token punctuation">[</span>-h<span class="token punctuation">]</span> <span class="token punctuation">[</span>-V<span class="token punctuation">]</span> <span class="token builtin class-name">command</span> <span class="token punctuation">..</span>.
</code></pre>
    <p>
     使用帮助：
    </p>
    <pre><code>conda --help 或者 conda 
</code></pre>
    <p>
     常见使用命令：
    </p>
    <pre><code class="prism language-shell">conda clean <span class="token comment"># 删除未使用的包和缓存</span>

conda config <span class="token comment"># 修改.condarc中的配置值</span>

conda create <span class="token comment"># 从指定包的列表中创建一个新的conda环境</span>
<span class="token comment">#eg：conda create --name &lt;env_name&gt; &lt;package_names&gt;</span>

conda info <span class="token comment"># 显示有关当前conda安装的信息</span>

conda <span class="token function">install</span> <span class="token comment"># 将程序包列表安装到指定的conda环境中</span>

conda list <span class="token comment"># 列出所有已安装的科学包及其依赖项</span>

conda package <span class="token comment"># 低级别conda软件包实用程序。（实验）</span>

conda remove <span class="token comment"># 从指定的conda中删除程序包列表环境</span>

conda run <span class="token comment"># 在conda环境中运行可执行文件</span>

conda search <span class="token comment"># 搜索包并显示相关的包信息。</span>

conda uninstall <span class="token comment"># conda remove的别名</span>

conda update <span class="token comment"># 将conda软件包更新到最新的兼容版本</span>
<span class="token comment">#eg: 更新conda至最新版本 conda update conda</span>

conda upgrade <span class="token comment"># conda update 的别名</span>
</code></pre>
    <p>
     <strong>
      环境管理举例:
     </strong>
    </p>
    <ol>
     <li>
      创建包含
      <em>
       numpy
      </em>
      和
      <em>
       pandas
      </em>
      包的环境 NN1
     </li>
    </ol>
    <pre><code class="prism language-python">conda create <span class="token operator">-</span>n NN1 numpy pandas
</code></pre>
    <p>
     ps：如果创建环境后安装Python时没有指定Python的版本，那么将会安装与Anaconda版本相同的Python版本
    </p>
    <ol start="2">
     <li>
      切换环境到NN1
     </li>
    </ol>
    <pre><code class="prism language-shell">conda activate NN1
</code></pre>
    <ol start="3">
     <li>
      退出环境
     </li>
    </ol>
    <pre><code class="prism language-shell">conda deactivate
</code></pre>
    <ol start="4">
     <li>
      显示已创建环境
     </li>
    </ol>
    <pre><code class="prism language-shell">conda info --envs 或者 conda info -e 或者 conda <span class="token function">env</span> list
</code></pre>
    <ol start="5">
     <li>
      复制环境NN1到 NN2
     </li>
    </ol>
    <pre><code class="prism language-shell">conda create --name NN1 --clone NN2
</code></pre>
    <p>
     <em>
      ps：克隆后两环境存在且配置相同
     </em>
    </p>
    <ol start="6">
     <li>
      删除环境NN1
     </li>
    </ol>
    <pre><code class="prism language-shell">conda remove --name NN1 --all
</code></pre>
    <p>
     <strong>
      包管理举例：
     </strong>
    </p>
    <ol>
     <li>
      更新所有包
     </li>
    </ol>
    <pre><code class="prism language-shell">conda update --all
</code></pre>
    <ol start="2">
     <li>
      在当前环境中安装numpy包
     </li>
    </ol>
    <pre><code class="prism language-shell">conda <span class="token function">install</span> numpy
</code></pre>
    <ol start="3">
     <li>
      卸载当前环境中的numpy包
     </li>
    </ol>
    <pre><code class="prism language-shell">conda remove numpy
</code></pre>
    <p>
     <em>
      ps：conda的使用同python的pip，pip不带环境管理功能,同样的，pip也适用于conda
     </em>
    </p>
    <p>
     具体可参考：
     <a href="https://zhuanlan.zhihu.com/p/32925500" rel="nofollow">
      Anaconda介绍、安装及使用教程
     </a>

    <p>
     四、解决冲突的问题
    </p>
    <p>
     前置确认：
    </p>
    <ol>
     <li>
      电脑中同时安装了Python和Anaconda并配置了环境变量
     </li>
     <li>
      两者均需要使用（否则卸载其中之一即可）
     </li>
    </ol>
    <h5>
     <a id="_196">
     </a>
     解决方法一：
    </h5>
    <p>
     取消其中一个的环境变量。
    </p>
    <p>
     例如想默认使用原生python的话在环境变量中删除Anaconda的根目录即可。
    </p>
    <h5>
     <a id="_202">
     </a>
     解决方法二（推荐）：
    </h5>
    <p>
     在安装目录中更改python.exe文件的名字区分两者,(先复制一份)
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c6e13762f32c245e666fcaf38d101ff0.png"/>
    </p>
    <p>
     这样在使用时输入
     <code>
      python
     </code>
     默认是用原生的Python，而如果使用
     <code>
      python-a
     </code>
     则使用Anaconda
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/49b984412e7f9936e079be1f5e499276.png#pic_center">
      <br/>
      注意：这样更改后pip命令也要进行区分，使用anaconda则需要在pip命令前加上
      <code>
       python-a -m
      </code>
     </img>
    </p>
    <pre><code class="prism language-shell">Python-ana –m pip –-version <span class="token comment">#查看当前Python对应的pip版本；</span>

Python-ana –m pip list <span class="token comment">#查看当前Python对应的pip安装的第三方库；</span>

Python-ana –m pip <span class="token function">install</span> <span class="token comment">#库名即可安装对应的扩展库；</span>

Python-ana –m pip uninstall <span class="token comment">#库名即可卸载对应的扩展库</span>
</code></pre>
    <p>
     问题：
    </p>
    <p>
     每次用命令行打开
     <code>
      python-a
     </code>
     都出现同样的warning, 每次都需要先activate environment.
    </p>
    <p>
     另外，默认的conda environment叫base, 所以运行：
     <code>
      conda activate base
     </code>
     即可。
    </p>
