#### 二、PDF转Word转换器概括介绍

一个基于Flet框架开发的图形界面PDF转Word转换工具。

#### 三、功能特性

* 🖱️ 1. 图形界面操作，简单易用
* 📄 2. 支持批量转换多个PDF文件
* 📁 3. 可选择自定义输出目录
* 📊 4. 实时显示转换进度
* ⚡ 5. 自动安装所需依赖

#### 四、安装依赖

```bash
pip install -r requirements.txt

```

或手动安装：

```bash
pip install flet pdf2docx

```

#### 五、运行程序

```bash
python pdf_to_word_converter.py

```

#### 六、使用说明

1. **选择PDF文件**：点击"选择PDF文件"按钮，选择要转换的PDF文件（支持多选）
2. **选择输出目录**：（可选）点击"选择输出目录"按钮选择转换后文件的保存位置，如果不选择则默认保存在PDF文件所在目录
3. **开始转换**：点击"开始转换"按钮开始转换过程
4. **查看进度**：转换过程中会显示实时进度和状态信息
5. **完成转换**：转换完成后会显示结果，转换后的Word文件(.docx)将保存在指定目录

#### 七、注意事项

* 首次运行时程序会自动安装pdf2docx依赖库
* 转换质量取决于原PDF文件的格式和内容
* 复杂的PDF文件（如包含大量图片、表格）可能需要较长转换时间
* 确保有足够的磁盘空间存储转换后的文件

#### 八、技术栈

* **GUI框架**：Flet
* **文件选择**：FilePicker
* **PDF转换**：pdf2docx
* **编程语言**：Python 3.7+

#### 九、系统要求

* Python 3.7 或更高版本
* Windows/macOS/Linux
* 至少500MB可用磁盘空间
