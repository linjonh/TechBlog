    <h2>
     <strong>
      一、 基础篇
     </strong>
    </h2>
    <p>
     <strong>
      1.1 数据结构基础知识
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        什么是数据结构？
       </strong>
       :
      </p>
      <ul>
       <li>
        <p>
         定义：数据结构是用来存储和组织数据的方式，它定义了数据在内存中的布局和访问方式。
        </p>
       </li>
       <li>
        <p>
         目的：高效地存储、组织和访问数据，提高程序的运行效率。
        </p>
       </li>
       <li>
        <p>
         分类：线性结构 (数组、链表、栈、队列)、非线性结构 (树、图、哈希表)。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        基本概念
       </strong>
       :
      </p>
      <ul>
       <li>
        <p>
         元素 (Element)：数据结构的基本单位。
        </p>
       </li>
       <li>
        <p>
         节点 (Node)：包含数据元素和指向其他节点的指针。
        </p>
       </li>
       <li>
        <p>
         指针 (Pointer)：存储内存地址的变量，用于连接节点。
        </p>
       </li>
       <li>
        <p>
         数组 (Array)：连续存储相同类型元素的线性结构。
        </p>
       </li>
       <li>
        <p>
         链表 (Linked List)：通过指针连接节点的线性结构。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        时间复杂度和空间复杂度
       </strong>
       :
      </p>
      <ul>
       <li>
        <p>
         时间复杂度：衡量算法执行时间随输入规模变化的趋势。
        </p>
       </li>
       <li>
        <p>
         空间复杂度：衡量算法使用的内存空间随输入规模变化的趋势。
        </p>
       </li>
       <li>
        <p>
         大O表示法：描述算法复杂度的常用方法。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      1.2 基本数据结构
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        数组 (Array):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：连续存储相同类型元素的线性结构。
        </p>
       </li>
       <li>
        <p>
         特点：随机访问快，插入和删除效率低。
        </p>
       </li>
       <li>
        <p>
         操作：访问、插入、删除、查找、排序。
        </p>
       </li>
       <li>
        <p>
         应用场景：存储有序数据、实现队列和栈等。
        </p>
       </li>
       <li>
        <p>
         练习：实现数组的动态扩容、查找最大/最小值、反转数组等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        链表 (Linked List):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         单链表：每个节点包含数据和指向下一个节点的指针。
        </p>
       </li>
       <li>
        <p>
         双链表：每个节点包含数据和指向前后节点的指针。
        </p>
       </li>
       <li>
        <p>
         循环链表：最后一个节点指向第一个节点。
        </p>
       </li>
       <li>
        <p>
         特点：插入和删除效率高，随机访问慢。
        </p>
       </li>
       <li>
        <p>
         操作：插入、删除、遍历、查找。
        </p>
       </li>
       <li>
        <p>
         应用场景：实现栈、队列、哈希表等。
        </p>
       </li>
       <li>
        <p>
         练习：实现链表的反转、合并两个有序链表、检测链表是否有环等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        栈 (Stack):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：后进先出 (LIFO) 的线性结构。
        </p>
       </li>
       <li>
        <p>
         操作：入栈 (push)、出栈 (pop)、查看栈顶元素 (peek)、判断栈空 (isEmpty)。
        </p>
       </li>
       <li>
        <p>
         应用场景：函数调用、表达式求值、回溯算法等。
        </p>
       </li>
       <li>
        <p>
         练习：实现栈的逆序、括号匹配、中缀表达式转后缀表达式等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        队列 (Queue):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：先进先出 (FIFO) 的线性结构。
        </p>
       </li>
       <li>
        <p>
         操作：入队 (enqueue)、出队 (dequeue)、查看队首元素 (front)、判断队列空 (isEmpty)。
        </p>
       </li>
       <li>
        <p>
         应用场景：任务调度、缓冲区、广度优先搜索等。
        </p>
       </li>
       <li>
        <p>
         练习：实现队列的循环结构、优先队列等。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      1.3 练习与项目
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        LeetCode:
       </strong>
       选择简单和中等难度的数组、链表、栈、队列相关的题目进行练习。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Codewars:
       </strong>
       选择与数据结构相关的 Kata 进行练习。
      </p>
     </li>
     <li>
      <p>
       <strong>
        个人项目:
       </strong>
       设计一个简单的应用程序，例如 TODO 列表、计算器等，并使用学到的数据结构实现。
      </p>
     </li>
    </ul>
    <h2>
     <strong>
      二、 进阶篇
     </strong>
    </h2>
    <p>
     <strong>
      2.1 树形结构
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        二叉树:
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：每个节点最多有两个子节点的树形结构。
        </p>
       </li>
       <li>
        <p>
         性质：高度、深度、叶节点、父节点、子节点等。
        </p>
       </li>
       <li>
        <p>
         遍历：前序遍历、中序遍历、后序遍历、层序遍历。
        </p>
       </li>
       <li>
        <p>
         应用场景：表达式树、二叉搜索树等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        二叉搜索树 (BST):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：左子树的所有节点值小于根节点值，右子树的所有节点值大于根节点值。
        </p>
       </li>
       <li>
        <p>
         特点：查找、插入、删除效率高。
        </p>
       </li>
       <li>
        <p>
         操作：插入、删除、查找、最小值、最大值。
        </p>
       </li>
       <li>
        <p>
         应用场景：数据库索引、排序等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        平衡二叉树 (AVL树、红黑树):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         定义：高度平衡的二叉搜索树，保证查找、插入、删除操作的时间复杂度为 O(log n)。
        </p>
       </li>
       <li>
        <p>
         应用场景：需要高效率查找、插入、删除操作的场景。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        堆 (Heap):
       </strong>
      </p>
      <ul>
       <li>
        <p>
         最大堆：根节点的值最大。
        </p>
       </li>
       <li>
        <p>
         最小堆：根节点的值最小。
        </p>
       </li>
       <li>
        <p>
         操作：插入、删除、查找最大/最小值。
        </p>
       </li>
       <li>
        <p>
         应用场景：堆排序、优先队列等。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      2.2 图 (Graph)
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        定义:
       </strong>
       由节点 (Vertex) 和边 (Edge) 组成，用于表示事物之间的关系。
      </p>
     </li>
     <li>
      <p>
       <strong>
        表示方法:
       </strong>
       邻接矩阵、邻接表。
      </p>
     </li>
     <li>
      <p>
       <strong>
        图的遍历:
       </strong>
       深度优先搜索 (DFS)、广度优先搜索 (BFS)。
      </p>
     </li>
     <li>
      <p>
       <strong>
        最短路径算法:
       </strong>
       Dijkstra算法、Bellman-Ford算法。
      </p>
     </li>
     <li>
      <p>
       <strong>
        最小生成树算法:
       </strong>
       Prim算法、Kruskal算法。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景:
       </strong>
       社交网络、地图导航、推荐系统等。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      2.3 哈希表 (Hash Table)
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        定义:
       </strong>
       使用哈希函数将键映射到数组中的索引，实现快速查找。
      </p>
     </li>
     <li>
      <p>
       <strong>
        哈希函数:
       </strong>
       用于将键映射到索引的函数，需要满足均匀分布和冲突最小化。
      </p>
     </li>
     <li>
      <p>
       <strong>
        冲突解决方法:
       </strong>
       开链法、开放寻址法。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景:
       </strong>
       数据库索引、缓存、符号表等。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      2.4 排序算法
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        时间复杂度:
       </strong>
       O(n log n) 的排序算法 (归并排序、快速排序、堆排序)
      </p>
     </li>
     <li>
      <p>
       <strong>
        时间复杂度:
       </strong>
       O(n^2) 的排序算法 (冒泡排序、插入排序、选择排序)
      </p>
     </li>
     <li>
      <p>
       <strong>
        稳定性:
       </strong>
       指排序算法是否保持相同元素的相对顺序。
      </p>
     </li>
     <li>
      <p>
       <strong>
        应用场景:
       </strong>
       数据排序、查找等。
      </p>
     </li>
    </ul>
    <p>
     <strong>
      2.5 练习与项目
     </strong>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        LeetCode:
       </strong>
       选择中等和困难难度的树形结构、图、哈希表、排序相关的题目进行练习。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Codeforces:
       </strong>
       参加算法竞赛，提高算法设计和实现能力。
      </p>
     </li>
     <li>
      <p>
       <strong>
        个人项目:
       </strong>
       设计一个更复杂的应用程序，例如社交网络、地图导航等，并使用学到的数据结构和算法实现。
      </p>
     </li>
    </ul>
