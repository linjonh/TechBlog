# ubuntu安装java8

1. 去oracle官网下载jar包，我选择下载
**jdk-8u201-linux-x64.tar.gz**

> <https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html>

2. 创建目录

sudo mkdir /usr/local/java

3. 将下载的文件复制到我们创建的文件夹下

sudo cp /home/lsk/Downloads/jdk-8u201-linux-x64.tar.gz /usr/local/java (注意lsk是我的用户名，jdk版本也许注意)

4. 解压缩文件

sudo tar -zxvf jdk-8u201-linux-x64.tar.gz

5. 配置环境变量

vim /etc/profile

> 末尾添加
>
> #Jdk Config
>   
> JAVA\_HOME=/usr/local/java/jdk1.8.0\_201
>   
> JAVA\_BIN=/usr/local/java/jdk1.8.0\_201/bin
>   
> PATH=$PATH:$JAVA\_BIN
>   
> CLASSPATH=$JAVA\_HOME/lib/dt.jar:$JAVA\_HOME/lib/tools.jar
>   
> export JAVA\_HOME JAVA\_BIN PATH CLASSPATH

6. 重启环境变量

source /etc/profile

7. 验证

java -version

> root@ubuntu:/usr/local/java/jdk1.8.0\_201/bin# java -version
>   
> java version "1.8.0\_201"
>   
> Java(TM) SE Runtime Environment (build 1.8.0\_201-b09)
>   
> Java HotSpot(TM) 64-Bit Server VM (build 25.201-b09, mixed mode)

完成。
