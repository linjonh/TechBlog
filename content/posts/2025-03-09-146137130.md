     <br/>
     2、点击个人头像旁边的加号，选择新建仓库：

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cd68db00559e4f2b8304e60b10c76fd2.png">

      3、填写仓库相关信息
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a6a0dd461ede4d44bc8014ac781099ff.png">
       <br/>

       <br/>
       4、复制Gitee仓库的地址
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b61b4a906a8c4482ae9ff61025771c37.png">
        <br/>
        <br/>
        <br/>
        5、绑定我们的本地仓库与远程仓库
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e893db5cbde74fc3a432a9f0cf6c6192.png">
         <br/>
         <br/>
         <br/>
         6、将本地仓库发布（推送）到远程仓库：
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/75f14221339c42309182420ef909c083.png">
          <br/>
          注意到此处报错，有关于大文件LFS。
          <br/>
          我们打开GitBash命令行，输入
          <code>
           rm .git/hooks/pre-push
          </code>
          删除掉pre-push文件，当然你也可以选择手动去.git/hooks/删除pre-push。
          <br/>
          <br/>
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/24a4e1a69c524dd5b2e65aa1e2745cd0.png">
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7d06cf0a811742ab8942dcd1999edba8.png">
            <br/>
            <br/>
            <br/>
            再输入
            <code>
             git push -u origin master
            </code>
            推送：
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9079d61da4d740c793bce202a13a9ea7.png">
             <br/>
             <br/>
             <br/>
             7、至此，大功告成。
             <br/>
             <br/>
             <br/>
             补充：
             <br/>
             在我们推送或者拉取时也有可能遇到以下报错：
             <br/>
             <strong>
              fatal: refusing to merge unrelated histories
             </strong>
             <br/>
             问题产生原因：本地库和远程库没有相关性， 然后本地要去推送到远端， 远端觉得这个本地库跟自己不相干， 所以告知无法合并。
             <br/>
             解决方法：打开GitBash命令行，输入
             <code>
              git pull origin master --allow-unrelated-histories
             </code>
