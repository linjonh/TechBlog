    <h2>

     离散数学第一章
    </h2>
    <h3>
     <a id="1_1">
     </a>
     1.公式类型
    </h3>
    <h5>
     <a id="1_2">
     </a>
     1）重言式
    </h5>
    <p>
     也是永真式，公式真值恒为1。
    </p>
    <h5>
     <a id="2_4">
     </a>
     2）矛盾式
    </h5>
    <p>
     永假式，真值恒为0。
    </p>
    <h5>
     <a id="3_6">
     </a>
     3）可满足式
    </h5>
    <p>
     不是矛盾式的就都是可满足式。重言式一定是可满足式。
    </p>
    <h3>
     <a id="2_8">
     </a>
     2.成真赋值与成假赋值
    </h3>
    <p>
     也叫成真指派与成假指派。
     <br/>
     一组原子的取值（真值指派）使得公式为真：成真指派（赋值）
     <br/>
     一组原子的取值（真值指派）使得公式为假：成假指派（赋值）
    </p>
    <h3>
     <a id="3_12">
     </a>
     3.等价式
    </h3>
    <p>
     也叫等值式，即任意的分量指派情况下，公式的值都相同，则等价。
     <br/>
     可用真值表、等值演算法判断。
    </p>
    <h3>
     <a id="4_17">
     </a>
     4.析取范式与合取范式
    </h3>
    <p>
     有限个命题变量（及其非命题）的析取称为析取式。
     <br/>
     有限个命题变量（及其非命题）的合取称为合取式。
    </p>
    <p>
     有限个合取式的析取称为析取范式。
     <br/>
     有限个析取式的合取称为合取范式。
     <br/>
     合取范式与析取范式仍不唯一。
    </p>
    <p>
     析取范式与合取范式中只能包含非、且、或。
     <br/>
     任意公式都存在等值的析取范式与合取范式。
    </p>
    <p>
     极小项：（小项、布尔合取）包含n个命题变元的
     <strong>
      合取式
     </strong>
     ，且每个命题变元或其非命题变元出现且仅出现一次（p出现则非p不出现、非p出现则p不出现），且出现的次序与下标次序（或字母的字典序）一致。
    </p>
    <p>
     eg：三个命题变元，对应的小项就有八个小项。
     <br/>
     <img alt="小项举例" src="https://i-blog.csdnimg.cn/blog_migrate/266179d93cb5b412d5a3be5c97528094.png"/>
    </p>
    <p>
     极大项：（大项、布尔析取）包含n个命题变元的
     <strong>
      析取式
     </strong>
     ，且每个命题变元或其非命题变元出现且仅出现一次（p出现则非p不出现、非p出现则p不出现），且出现的次序与下标次序（或字母的字典序）一致。
    </p>
    <p>
     eg：三个命题变元，对应也有八个大项，需要注意的是用的是成假赋值。
     <br/>
     <img alt="大项举例" src="https://i-blog.csdnimg.cn/blog_migrate/557936064701bdef81166325a1ed808f.png">
      <br/>
      <strong>
       主合取范式
      </strong>
      ：合取范式中每个析取式都是大项，则称为主合取范式。
      <br/>
      <strong>
       主析取范式
      </strong>
      ：析取范式中每个合取项都是小项，则称为主析取范式
     </img>
    </p>
    <p>
     求主析取范式与主合取范式：真值表法与等值演算法。
    </p>
    <p>
     eg：真值表举例：
     <br/>
     <img alt="主析取范式与主合取范式举例" src="https://i-blog.csdnimg.cn/blog_migrate/cf98eccfc3e4939b8b7339436d7e0a83.png">
      <br/>
      要注意例子中求主析取范式中用的是
      <strong>
       小项
      </strong>
      ，而求主合取范式用的是求主析取范式所剩下的项所对应的
      <strong>
       成假赋值的大项
      </strong>
      。
     </img>
    </p>
