# 【新版发布】Apache DolphinScheduler 3.3.1 正式上线：更稳、更快、更安全！



![1](https://i-blog.csdnimg.cn/img_convert/9cce5279fa40eb6fb047903ca2e71c76.png)

亲爱的社区小伙伴们，大家好！

我们迎来了 Apache DolphinScheduler 的又一次重要更新——**3.3.1 版本正式发布**！🎉

作为一款稳定、易用、功能强大的分布式工作流调度平台，DolphinScheduler 始终致力于为用户提供极致的调度体验。本次发布的 3.3.1 是一个以 **Bug修复** 和 **体验优化** 为核心的迭代版本，旨在为您带来更稳定、更高效、更安全的调度服务。

快来看看本次更新有哪些不容错过的亮点吧！

##### ✨ **新增功能 (Features)**

虽然是修复版本，我们依然带来了一些实用的新功能，让您的工作流管理更加得心应手！

1. **本地任务支持 SSH 密钥**：现在，您可以在执行本地任务时使用 SSH 密钥进行认证，大幅提升了操作的便捷性和安全性。
2. **支持 LDAP TLS/SSL**：增强了对 LDAP 的安全支持，企业用户在进行用户认证时可以启用加密传输，保障账户信息安全。
3. **ClickHouse JDBC URL 支持自定义参数**：为 ClickHouse 数据源提供了更高的灵活性，方便用户根据特定场景配置连接参数。

##### 🛡️ **重要修复与安全更新 (BugFix & Security)**

稳定性和安全性是我们永远的追求。在此版本中，我们修复了社区反馈的数十个问题，并解决了一个重要的安全漏洞。

* **修复关键安全漏洞**：解决了 `CVE-2024-38686` 漏洞，增强了系统的安全性。**强烈建议所有用户升级！**
* **任务停止逻辑优化**：修复了在某些特定情况下任务无法被正常停止的问题，确保工作流的控制更加精确可靠。
* **Worker 分组管理优化**：修复了 Worker 分组在重启后可能丢失的问题，提升了集群管理的稳定性。
* **修复补数功能问题**：解决了补数功能在处理依赖任务时可能出现的逻辑错误。
* **前端体验修复**：修复了多个前端UI的显示和操作问题，让您的视觉和操作体验更加流畅。

（更多详细修复列表，请参考文末的 Release Note 链接）

##### 🛠️ **体验优化与改进 (Improvements)**

细节决定成败。我们对多个模块进行了深度优化，只为给您带来更好的使用体验。

* **注册中心目录结构优化**：优化了服务在 Zookeeper/Etcd 中的注册目录结构，使服务发现更加清晰高效。
* **任务日志与错误信息优化**：优化了任务日志的输出格式和 Python Gateway 的日志记录，同时为常见的错误增加了更详细的提示信息，帮助您更快地定位和解决问题。
* **依赖组件升级**：升级了部分依赖库，提升了系统整体的性能和稳定性。

##### 🔗 **立即体验**

我们强烈建议所有正在使用 DolphinScheduler 的用户升级至 3.3.1 版本，以获得最佳的稳定性和安全性。
