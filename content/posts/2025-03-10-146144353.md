    <h3>
     <a id="_0">
     </a>
     引言
    </h3>
    <p>
     Kibana 是 Elasticsearch 的可视化工具，提供了一个用户友好的界面来管理和操作 Elasticsearch 中的数据。通过 Kibana，你可以轻松地执行数据搜索、创建可视化图表、构建仪表盘等操作。本文将带你从零开始学习如何通过 Kibana 操作 Elasticsearch，掌握其基本功能和进阶操作。
    </p>
    <h3>
     <a id="1_Kibana__4">
     </a>
     1. Kibana 简介
    </h3>
    <h4>
     <a id="11__Kibana_6">
     </a>
     1.1 什么是 Kibana？
    </h4>
    <p>
     Kibana 是一个开源的数据可视化工具，专为 Elasticsearch 设计。它提供了丰富的功能，包括：
    </p>
    <ul>
     <li>
      <strong>
       数据探索
      </strong>
      ：通过 Discover 功能，可以轻松搜索和查看 Elasticsearch 中的数据。
     </li>
     <li>
      <strong>
       可视化
      </strong>
      ：支持多种图表类型，如柱状图、饼图、折线图等。
     </li>
     <li>
      <strong>
       仪表盘
      </strong>
      ：可以将多个可视化图表组合成一个仪表盘，方便监控和分析数据。
     </li>
     <li>
      <strong>
       开发工具
      </strong>
      ：提供了 Dev Tools，方便开发者通过控制台直接与 Elasticsearch 交互。
     </li>
    </ul>
    <h4>
     <a id="12_Kibana__Elasticsearch__15">
     </a>
     1.2 Kibana 与 Elasticsearch 的关系
    </h4>
    <p>
     Kibana 是 Elastic Stack（也称为 ELK Stack）的一部分，与 Elasticsearch 紧密集成。Kibana 通过 Elasticsearch 的 RESTful API 与 Elasticsearch 进行通信，用户可以通过 Kibana 的界面来操作 Elasticsearch 中的数据。
    </p>
    <h3>
     <a id="2__19">
     </a>
     2. 安装与配置
    </h3>
    <h4>
     <a id="21__Kibana_21">
     </a>
     2.1 安装 Kibana
    </h4>
    <p>
     Kibana 可以在多种操作系统上运行，以下以 Linux 为例介绍安装步骤。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        下载 Kibana
       </strong>
      </p>
      <p>
       访问
       <a href="https://www.elastic.co/cn/downloads/kibana" rel="nofollow">
        Kibana 官方下载页面
       </a>
       ，选择适合的版本进行下载。
      </p>
      <pre><code class="prism language-bash"><span class="token function">wget</span> https://artifacts.elastic.co/downloads/kibana/kibana-7.10.1-linux-x86_64.tar.gz
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        解压并安装
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token function">tar</span> <span class="token parameter variable">-xzf</span> kibana-7.10.1-linux-x86_64.tar.gz
<span class="token builtin class-name">cd</span> kibana-7.10.1-linux-x86_64/
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        启动 Kibana
       </strong>
      </p>
      <pre><code class="prism language-bash">./bin/kibana
</code></pre>
      <p>
       默认情况下，Kibana 会在
       <code>
        localhost:5601
       </code>
       上启动。
      </p>
     </li>
    </ol>
    <h4>
     <a id="22__Kibana_48">
     </a>
     2.2 配置 Kibana
    </h4>
    <p>
     Kibana 的配置文件位于
     <code>
      config/kibana.yml
     </code>
     ，常见的配置项包括：
    </p>
    <ul>
     <li>
      <strong>
       Elasticsearch 地址
      </strong>
      ：
      <code>
       elasticsearch.hosts
      </code>
     </li>
     <li>
      <strong>
       Kibana 端口
      </strong>
      ：
      <code>
       server.port
      </code>
     </li>
     <li>
      <strong>
       Kibana 绑定地址
      </strong>
      ：
      <code>
       server.host
      </code>
     </li>
    </ul>
    <p>
     例如，修改 Elasticsearch 地址：
    </p>
    <pre><code class="prism language-yaml"><span class="token key atrule">elasticsearch.hosts</span><span class="token punctuation">:</span> <span class="token punctuation">[</span><span class="token string">"http://localhost:9200"</span><span class="token punctuation">]</span>
</code></pre>
    <h3>
     <a id="3__62">
     </a>
     3. 基本操作
    </h3>
    <h4>
     <a id="31__Dev_Tools__Elasticsearch_64">
     </a>
     3.1 使用 Dev Tools 操作 Elasticsearch
    </h4>
    <p>
     Kibana 提供了 Dev Tools，方便开发者通过控制台直接与 Elasticsearch 交互。以下是一些常用的操作示例。
    </p>
    <h5>
     <a id="311__68">
     </a>
     3.1.1 创建索引
    </h5>
    <pre><code class="prism language-json"><span class="token constant">PUT</span> <span class="token operator">/</span>my_index
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"settings"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-property property">"number_of_shards"</span><span class="token operator">:</span> <span class="token number">1</span><span class="token punctuation">,</span>
    <span class="token string-property property">"number_of_replicas"</span><span class="token operator">:</span> <span class="token number">1</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="312__80">
     </a>
     3.1.2 插入文档
    </h5>
    <pre><code class="prism language-json"><span class="token constant">POST</span> <span class="token operator">/</span>my_index<span class="token operator">/</span>_doc<span class="token operator">/</span><span class="token number">1</span>
<span class="token punctuation">{<!-- --></span>

<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="313__91">
     </a>
     3.1.3 查询文档
    </h5>
    <pre><code class="prism language-json"><span class="token constant">GET</span> <span class="token operator">/</span>my_index<span class="token operator">/</span>_doc<span class="token operator">/</span><span class="token number">1</span>
</code></pre>
    <h5>
     <a id="314__97">
     </a>
     3.1.4 搜索文档
    </h5>
    <pre><code class="prism language-json"><span class="token constant">GET</span> <span class="token operator">/</span>my_index<span class="token operator">/</span>_search
<span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"query"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-property property">"match"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
      <span class="token string-property property">"content"</span><span class="token operator">:</span> <span class="token string">"入门"</span>
    <span class="token punctuation">}</span>
  <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="32__Discover__110">
     </a>
     3.2 使用 Discover 探索数据
    </h4>
    <p>
     Kibana 的 Discover 功能允许你轻松搜索和查看 Elasticsearch 中的数据。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Discover
       </strong>
      </p>
      <p>
       在 Kibana 左侧导航栏中，点击
       <strong>
        Discover
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        选择索引模式
       </strong>
      </p>
      <p>
       在 Discover 页面中，选择你想要探索的索引模式。如果没有索引模式，可以先创建一个。
      </p>
     </li>
     <li>
      <p>
       <strong>
        搜索数据
       </strong>
      </p>
      <p>
       在搜索栏中输入查询条件，Kibana 会实时显示匹配的文档。
      </p>
     </li>
    </ol>
    <h4>
     <a id="33__126">
     </a>
     3.3 创建可视化图表
    </h4>
    <p>
     Kibana 提供了多种可视化图表类型，如柱状图、饼图、折线图等。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Visualize Library
       </strong>
      </p>
      <p>
       在 Kibana 左侧导航栏中，点击
       <strong>
        Visualize Library
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        创建可视化图表
       </strong>
      </p>
      <p>
       点击
       <strong>
        Create visualization
       </strong>
       ，选择图表类型（如柱状图）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置数据源
       </strong>
      </p>
      <p>
       选择索引模式，并配置 X 轴和 Y 轴的数据字段。
      </p>
     </li>
     <li>
      <p>
       <strong>
        保存可视化图表
       </strong>
      </p>
      <p>
       配置完成后，点击
       <strong>
        Save
       </strong>
       保存图表。
      </p>
     </li>
    </ol>
    <h4>
     <a id="34__146">
     </a>
     3.4 构建仪表盘
    </h4>
    <p>
     Kibana 的仪表盘功能允许你将多个可视化图表组合在一起，方便监控和分析数据。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Dashboard
       </strong>
      </p>
      <p>
       在 Kibana 左侧导航栏中，点击
       <strong>
        Dashboard
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        创建仪表盘
       </strong>
      </p>
      <p>
       点击
       <strong>
        Create dashboard
       </strong>
       ，然后点击
       <strong>
        Add from library
       </strong>
       ，选择之前创建的可视化图表。
      </p>
     </li>
     <li>
      <p>
       <strong>
        调整布局
       </strong>
      </p>
      <p>
       拖动图表调整布局，点击
       <strong>
        Save
       </strong>
       保存仪表盘。
      </p>
     </li>
    </ol>
    <h3>
     <a id="4__162">
     </a>
     4. 进阶操作
    </h3>
    <h4>
     <a id="41__Canvas__164">
     </a>
     4.1 使用 Canvas 创建动态报告
    </h4>
    <p>
     Kibana 的 Canvas 功能允许你创建动态的、交互式的报告。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Canvas
       </strong>
      </p>
      <p>
       在 Kibana 左侧导航栏中，点击
       <strong>
        Canvas
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        创建工作区
       </strong>
      </p>
      <p>
       点击
       <strong>
        Create workpad
       </strong>
       ，选择数据源并添加元素（如文本、图表、图片等）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置元素
       </strong>
      </p>
      <p>
       配置每个元素的数据和样式，点击
       <strong>
        Save
       </strong>
       保存工作区。
      </p>
     </li>
    </ol>
    <h4>
     <a id="42__Machine_Learning__180">
     </a>
     4.2 使用 Machine Learning 进行数据分析
    </h4>
    <p>
     Kibana 集成了 Elasticsearch 的机器学习功能，可以进行异常检测、预测等操作。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Machine Learning
       </strong>
      </p>
      <p>
       在 Kibana 左侧导航栏中，点击
       <strong>
        Machine Learning
       </strong>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        创建作业
       </strong>
      </p>
      <p>
       点击
       <strong>
        Create job
       </strong>
       ，选择数据源并配置分析任务。
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看结果
       </strong>
      </p>
      <p>
       作业完成后，可以在 Kibana 中查看分析结果和可视化图表。
      </p>
     </li>
    </ol>
    <h3>
     <a id="5__196">
     </a>
     5. 总结
    </h3>
    <p>
     通过本文，你已经了解了如何通过 Kibana 操作 Elasticsearch，掌握了 Kibana 的基本功能和进阶操作。Kibana 是一个功能强大的可视化工具，能够极大地提升 Elasticsearch 的使用体验。接下来，你可以进一步学习 Kibana 的高级功能，如 Canvas、Machine Learning 等。
