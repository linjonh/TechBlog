    <h2>
     一、直流电机
    </h2>
    <h3>
     1.介绍
    </h3>
    <ul>
     <li>
      直流电机是一种将电能转换为机械能的装置。一般的直流电机有两个电极，当电极正接时，电机正转，当电极反接时，电机反转
     </li>
     <li>
      直流电机主要由永磁体（定子）、线圈（转子）和换向器组成
     </li>
     <li>
      除直流电机外，常见的电机还有步进电机（机壳装载了一圈电磁铁，对相对两个电磁铁进行一圈脉冲式通电，优点是严格控速精密）、舵机（跟船舵一样，方便控制转向）、无刷电机（转速特别快）、空心杯电机（体积很小）等
     </li>
     <li>
      左1就是我们开发板里的电机，有两个电极；左1是N20，一个直流电机+减速箱；右1比左2还多带一个编码器（测速），PID算法（常用）控制电机速度
     </li>
     <li>
      <img alt="" height="179" src="https://i-blog.csdnimg.cn/direct/eb0aae3494b64c9da63774fe40e1fbaf.png" width="709"/>
     </li>
    </ul>
    <h3>
     2.驱动电路
    </h3>
    <p>
     有两种方式
    </p>
    <ul>
     <li>
      <p>
       大功率器件直接驱动
      </p>
     </li>
    </ul>
    <p>
     不能正反转
     <img alt="" height="265" src="https://i-blog.csdnimg.cn/direct/e411e28b872848bfaa449e8e4dea7dfd.png" width="255">
      上面是一个三极管控制开关，下面是一个蓄流二极管，用于保护电路；因为电机（继电器）是感性负载元件，驱动的时候会出现很高的电压（电感的特效）
     </img>
    </p>
    <p>
    </p>
    <p>
     这个ULN2003里面也是一个大功率元件
    </p>
    <p>
    </p>
    <p>
     <img alt="" height="152" src="https://i-blog.csdnimg.cn/direct/bb2fbd96b68246d58ef536ba61d478f4.png" width="229"/>
    </p>
    <ul>
     <li>
      <p>
       H桥驱动
      </p>
     </li>
    </ul>
    <p>
     能正反转
     <img alt="" height="265" src="https://i-blog.csdnimg.cn/direct/d825f5b28a094cc890a86f1d3d4018e7.png" width="254">
      因为我们使第二、四象限的二极管导通其余的截止，就会令电流向右走，如果反过来就可以令电流向左走；因为没有蓄流二极管保护，所以对mos管和晶体管的耐压特效要求比较高
     </img>
    </p>
    <h2>
     二、PWM
    </h2>
    <p>
     PWM是控制直流电机调速的
    </p>
    <h3>
     <br/>
     1.介绍
    </h3>
    <ul>
     <li>
      PWM（Pulse Width Modulation）即脉冲宽度调制，在具有惯性的系统中，可以通过对一系列脉冲的宽度进行调制，来等效地获得所需要的模拟参量，常应用于电机控速、开关电源等领域
     </li>
     <li>
      PWM重要参数：
     </li>
     <li>
      频率=1/Ts  占空比=Ton/Ts  精度=占空比变化步距
     </li>
     <li>
      <img alt="" height="198" src="https://i-blog.csdnimg.cn/direct/b59f74f9493d45c4a82ad3cd6ce710e9.png" width="761"/>
     </li>
    </ul>
    <p>
    </p>
    <h2>
     三、代码实战
    </h2>
    <h3>
     1.LED呼吸灯
    </h3>
    <p>
     先定义一下引脚
    </p>
    <p>
     然后再定义一个比较快的延时函数，让这个LED灯先亮一段时间，再灭一段时间，这就是PWM的工作原理，通常为了保证周期是稳定的，不随占空比变化而变化，所以亮和灭加起来的时间也要是个固定值
    </p>
    <pre><code class="hljs">#include &lt;REGX52.H&gt;

sbit LED=P2^0;

void Delay(unsigned int t)
{
	while(t--);
}

void main()
{
	
	while(1)
	{
 		LED=0;
		Delay(5);
		LED=1;
		Delay(95);
	}
}</code></pre>
    <p>
     接下来就是让亮和灭的值不断变化，从而实现呼吸灯的效果
    </p>
    <p>
     我们就可以定义一个变量Time，让亮的时间为Time，灭的时间为100-Time，这样就可以实现两个数加起来是固定值，然后我们再对Time进行for循环，不断++
    </p>
    <p>
     但是这个速度还是比较快，我们让他每次亮灭变换的时间再久一点，定义一个i，在for循环里嵌套再进行一次for循环，这样就可以实现LED一直从暗到亮
    </p>
    <p>
     但是我们还想实现从暗到亮，再从亮到暗又怎么做呢，就是把之前的外层对Time的for循环反过来再依次--
    </p>
    <pre><code class="hljs">#include &lt;REGX52.H&gt;

sbit LED=P2^0;

void Delay(unsigned int t)
{
	while(t--);
}

void main()
{
	unsigned char Time,i;
	while(1)
	{
		for(Time=0;Time&lt;100;Time++)
		{
			for(i=0;i&lt;20;i++)
			{
				LED=0;
				Delay(Time);
				LED=1;
				Delay(100-Time);				
			}
		}
		for(Time=100;Time&gt;0;Time--)
		{
			for(i=0;i&lt;20;i++)
			{
				LED=0;
				Delay(Time);
				LED=1;
				Delay(100-Time);				
			}
		}
	}
}</code></pre>
    <p>
     这样就可以实现呼吸灯从暗到亮，再从亮到暗
    </p>
    <p>
     但是会有一个缺点，就是占用了我们的主循环不断翻转IO口，还比较占用CPU
    </p>
    <p>
     一般都会写到定时器上，或者从硬件上解决
    </p>
    <p>
     所以在下面的代码中我们用定时器来实现
    </p>
    <h3>
     2.直流电机调速
    </h3>
    <p>
     在这个代码我们就用定时器来实现LED呼吸灯和直流电机调速功能
    </p>
    <p>
     <strong>
      产生PWM方法
     </strong>
    </p>
    <p>
     <img alt="" height="466" src="https://i-blog.csdnimg.cn/direct/de385f3fd4624109a37f3606534afe75.png" width="862"/>
    </p>
    <p>
     我们通过改变比较值就可以改变占空比
    </p>
    <p>
     由于之前的定时器设置有点慢，我们让他快一点
    </p>
    <p>
     <img alt="" height="307" src="https://i-blog.csdnimg.cn/direct/027745685fb441a4bd7987d48a523a81.png" width="451"/>
    </p>
    <p>
     这个PWM驱动电机在一定范围内是越快越好，一般设置在10K-20K范围里，如果频率比较低，电流就会抖动，如果在1K左右，就会产生一些蜂鸣器的声音
    </p>
    <p>
     这个中断就是每隔100us中断一次
    </p>
    <pre><code class="hljs">//timer0.c

#include &lt;REGX52.H&gt;

/**
  * @brief  定时器0初始化，100us@12.000MHz
  * @param  无
  * @retval 无
  */
void Timer0_Init(void)
{
	TMOD &amp;= 0xF0;		//设置定时器模式
	TMOD |= 0x01;		//设置定时器模式
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	TF0 = 0;		//清除TF0标志
	TR0 = 1;		//定时器0开始计时
	ET0=1;
	EA=1;
	PT0=0;
}

/*定时器中断函数模板
void Timer0_Routine() interrupt 1
{
	static unsigned int T0Count;
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	T0Count++;
	if(T0Count&gt;=1000)
	{
		T0Count=0;
		
	}
}
*/
</code></pre>
    <p>
     由上面的图可以知道要定义两个值，计数值和比较值，
    </p>
    <p>
     我们让计数值从0-100然后清零，实现定时自增
    </p>
    <p>
     Counter++;
     <br/>
     Counter%=100;
    </p>
    <p>
     这样写和if（counter==100）{清零}是一个效果
    </p>
    <p>
     接下来再让计数值和比较值进行比较
    </p>
    <p>
     可以观察到如果比较值越大，低电平时间越久，也就是越暗
    </p>
    <p>
     这样我们通过控制比较值Compare就可以控制LED的亮度，还剩下下了主循环，接下来我们就实现按键功能
    </p>
    <pre><code class="hljs">#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " Key.h"
#include " Nixie.h"
#include " Timer0.h"

sbit LED=P2^0;

unsigned char Counter,Compare;

void main()
{
	Timer0_Init();
	Compare=5;
	while(1)
	{
 		
	
	}

}

void Timer0_Routine() interrupt 1
{
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	Counter++;
	Counter%=100;
	if(Counter&lt;Compare)
	{
		LED=0;
	}
	else
	{
		LED=1;
	}
}</code></pre>
    <p>
     注意后面取的Key函数，如果是拿的定时器扫描按键的那章写的Key函数，要在定时器中写上这个Key_Loop()
    </p>
    <pre><code class="hljs">#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " Key.h"
#include " Nixie.h"
#include " Timer0.h"

sbit LED=P2^0;

unsigned char Counter,Compare;
unsigned char KeyNum,Speed;

void main()
{

	Timer0_Init();
	Compare=5;
	while(1)
	{
		KeyNum=Key();
		if(KeyNum==1)
		{
			Speed++;
			Speed%=4;
		}
		Nixie(1,Speed);
	}

}

void Timer0_Routine() interrupt 1
{
	Key_Loop();
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	Counter++;
	Counter%=100;
	if(Counter&lt;Compare)
	{
		LED=0;
	}
	else
	{
		LED=1;
	}
}</code></pre>
    <p>
     这样就可以实现按下按键1数码管显示++并且过3就清零
    </p>
    <p>
     接下来将Speed跟Compare比较值对应起来，就可以实现按键调速并显示
    </p>
    <p>
     这样就可以实现LED的亮度控制
    </p>
    <p>
     if(Speed==0){Compare=5;}
     <br/>
     if(Speed==1){Compare=5;}
     <br/>
     if(Speed==2){Compare=50;}
     <br/>
     if(Speed==3){Compare=100;}
    </p>
    <p>
     那我们如果想控制电机，就直接把LED的端口改成电机的端口，而又因为我们的LED是给0亮给1灭，而电机是给1转给0灭，所以需要把极性反过来
    </p>
    <p>
     这里有个小问题，在刚上电的时候，电机会转动，这是硬件的问题，因为单片机在复位的时候，IO口默认高电平，所以电机会转
    </p>
    <pre><code class="hljs">#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " Key.h"
#include " Nixie.h"
#include " Timer0.h"

sbit Motor=P1^0;

unsigned char Counter,Compare;
unsigned char KeyNum,Speed;

void main()
{

	Timer0_Init();
	Compare=5;
	while(1)
	{
		KeyNum=Key();
		if(KeyNum==1)
		{
			Speed++;
			Speed%=4;
			if(Speed==0){Compare=0;}
			if(Speed==1){Compare=50;}
			if(Speed==2){Compare=75;}
			if(Speed==3){Compare=100;}	
		}
		Nixie(1,Speed);
	}

}

void Timer0_Routine() interrupt 1
{
	Key_Loop();
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	Counter++;
	Counter%=100;
	if(Counter&lt;Compare)
	{
		Motor=1;
	}
	else
	{
		Motor=0;
	}
}</code></pre>
    <p>
     这样就实现本节课的代码啦，按下按键调节电机的速度并且显示出来。
    </p>
