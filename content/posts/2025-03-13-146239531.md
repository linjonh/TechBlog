     4. 求1000以内水仙花数（3位数）。
    </p>
    <pre><code class="language-python">num = 0
for i in range(100, 1000+1):
    num_str = str(i)
    g = int(num_str[2])
    s = int(num_str[1])
    b = int(num_str[0])
    if g**3 + s**3 + b**3 == i:
        print(i)
        num+=1
print("总数：", num)</code></pre>
