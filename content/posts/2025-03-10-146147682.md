    <h3>
     <a id="_1_JetBrains_Toolbox__0">
     </a>
     <strong>
      📌 1. JetBrains Toolbox 介绍
     </strong>
    </h3>
    <p>
     JetBrains Toolbox 是 JetBrains 开发的工具管理器，可用于安装、更新和管理 IntelliJ IDEA、PyCharm、WebStorm、CLion 等。本指南记录了 JetBrains Toolbox 在 Ubuntu 24.04 上的
     <strong>
      安装、路径调整、权限管理
     </strong>
     及
     <strong>
      遇到的问题与解决方案
     </strong>
     。
    </p>
    <hr/>
    <h3>
     <a id="_2__JetBrains_Toolbox_6">
     </a>
     <strong>
      🛠 2. 下载 JetBrains Toolbox
     </strong>
    </h3>
    <p>
     JetBrains Toolbox 官方提供
     <code>
      .tar.gz
     </code>
     版本，可以在
     <a href="https://www.jetbrains.com/toolbox-app/" rel="nofollow">
      JetBrains 官网
     </a>
     下载。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ea772741b397411183bc78fdf05662c2.png"/>
    </p>
    <h4>
     <a id="_21__12">
     </a>
     <strong>
      ✅ 2.1 下载安装包
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~/下载
<span class="token function">wget</span> https://download.jetbrains.com/toolbox/jetbrains-toolbox-2.5.4.38621.tar.gz
</code></pre>
    <h4>
     <a id="_22__19">
     </a>
     <strong>
      ✅ 2.2 解压安装包
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">tar</span> <span class="token parameter variable">-xzf</span> jetbrains-toolbox-2.5.4.38621.tar.gz
</code></pre>
    <h4>
     <a id="_23__Toolbox_25">
     </a>
     <strong>
      ✅ 2.3 运行 Toolbox
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> jetbrains-toolbox-2.5.4.38621
./jetbrains-toolbox
</code></pre>
    <p>
     <strong>
      ⚠️ 可能的问题：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       如果无法运行
      </strong>
      ， 请先安装依赖以下软件包：
      <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> <span class="token parameter variable">-y</span> libfuse2 libxi6 libxrender1 libxtst6 mesa-utils libfontconfig libgtk-3-bin <span class="token function">tar</span> dbus-user-session
</code></pre>
     </li>
     <li>
      <strong>
       如果 Toolbox 没有 UI 反应
      </strong>
      ，尝试使用 X11 模式启动：
      <pre><code class="prism language-bash"><span class="token function">env</span> <span class="token assign-left variable"><span class="token environment constant">XDG_SESSION_TYPE</span></span><span class="token operator">=</span>x11 ./jetbrains-toolbox
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_3__44">
     </a>
     <strong>
      🚀 3. 设置全局路径（所有用户可用）
     </strong>
    </h3>
    <p>
     默认情况下，Toolbox 只适用于当前用户。我们可以将它移动到
     <code>
      /opt/jetbrains-toolbox
     </code>
     ，让所有用户都可以访问。
    </p>
    <h4>
     <a id="_31__Toolbox__48">
     </a>
     <strong>
      ✅ 3.1 移动 Toolbox 到全局路径
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mkdir</span> <span class="token parameter variable">-p</span> /opt/jetbrains-toolbox
<span class="token function">sudo</span> <span class="token function">mv</span> ~/下载/jetbrains-toolbox-2.5.4.38621 /opt/jetbrains-toolbox
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">755</span> /opt/jetbrains-toolbox
</code></pre>
    <h4>
     <a id="_32__56">
     </a>
     <strong>
      ✅ 3.2 创建终端命令快捷方式
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">ln</span> <span class="token parameter variable">-s</span> /opt/jetbrains-toolbox/jetbrains-toolbox /usr/local/bin/jetbrains-toolbox
</code></pre>
    <p>
     现在，任何用户都可以运行：
    </p>
    <pre><code class="prism language-bash">jetbrains-toolbox
</code></pre>
    <hr/>
    <h3>
     <a id="_4__Toolbox__70">
     </a>
     <strong>
      📂 4. 配置 Toolbox 的工具安装目录
     </strong>
    </h3>
    <p>
     默认情况下，Toolbox 会将 IDE 安装到
     <code>
      ~/.local/share/JetBrains/Toolbox/apps
     </code>
     ，但如果我们想要全局安装，可以改为
     <code>
      /opt/jetbrains-toolbox/apps
     </code>
     。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/11bf4ad703cd465aaba80d47876fce5e.png"/>
    </p>
    <h4>
     <a id="_41__76">
     </a>
     <strong>
      ✅ 4.1 创建全局应用安装目录
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">mkdir</span> <span class="token parameter variable">-p</span> /opt/jetbrains-toolbox/apps
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">777</span> /opt/jetbrains-toolbox/apps
</code></pre>
    <h4>
     <a id="_42__Toolbox__83">
     </a>
     <strong>
      ✅ 4.2 修改 Toolbox 配置
     </strong>
    </h4>
    <ol>
     <li>
      打开 Toolbox
     </li>
     <li>
      <strong>
       进入“工具”设置
      </strong>
     </li>
     <li>
      <strong>
       修改安装路径：
      </strong>
      <code>
       /opt/jetbrains-toolbox/apps
      </code>
     </li>
     <li>
      <strong>
       修改 shell 脚本路径：
      </strong>
      <code>
       /opt/jetbrains-toolbox/scripts
      </code>
     </li>
    </ol>
    <h4>
     <a id="_43__PATH__Toolbox__90">
     </a>
     <strong>
      ✅ 4.3 确保
      <code>
       $PATH
      </code>
      包含 Toolbox 脚本目录
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">nano</span> /etc/profile
</code></pre>
    <p>
     在文件末尾添加：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token string">"/opt/jetbrains-toolbox/scripts:<span class="token environment constant">$PATH</span>"</span>
</code></pre>
    <p>
     使更改生效：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> /etc/profile
</code></pre>
    <hr/>
    <h3>
     <a id="_5__110">
     </a>
     <strong>
      ⚠️ 5. 遇到的问题及解决方案
     </strong>
    </h3>
    <h4>
     <a id="_51_Toolbox__112">
     </a>
     <strong>
      🛑 5.1 Toolbox 运行后无法点击（无响应）
     </strong>
    </h4>
    <p>
     <strong>
      问题描述：
     </strong>
    </p>
    <ul>
     <li>
      Toolbox 启动后，窗口显示正常，但鼠标点击没有反应。
     </li>
    </ul>
    <p>
     <strong>
      解决方案：
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/.local/share/JetBrains/Toolbox
<span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/.config/JetBrains/Toolbox
<span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/.cache/JetBrains/Toolbox
</code></pre>
    <p>
     然后重新运行 Toolbox。
    </p>
    <hr/>
    <h4>
     <a id="_52_Toolbox__129">
     </a>
     <strong>
      🛑 5.2 Toolbox 没有出现在应用菜单
     </strong>
    </h4>
    <p>
     <strong>
      解决方案：
     </strong>
    </p>
    <p>
     手动创建桌面快捷方式：
    </p>
    <pre><code class="prism language-bash"><span class="token function">nano</span> ~/.local/share/applications/jetbrains-toolbox.desktop
</code></pre>
    <p>
     添加以下内容：
    </p>
    <pre><code class="prism language-ini">[Desktop Entry]
Name=JetBrains Toolbox
Exec=/opt/jetbrains-toolbox/jetbrains-toolbox
Icon=/opt/jetbrains-toolbox/toolbox.svg
Type=Application
Categories=Development;
StartupWMClass=jetbrains-toolbox
</code></pre>
    <p>
     保存并退出后：
    </p>
    <pre><code class="prism language-bash"><span class="token function">chmod</span> +x ~/.local/share/applications/jetbrains-toolbox.desktop
update-desktop-database ~/.local/share/applications/
</code></pre>
    <hr/>
    <h3>
     <a id="_6__JetBrains_Toolbox_160">
     </a>
     <strong>
      🔄 6. 更新 JetBrains Toolbox
     </strong>
    </h3>
    <p>
     当 Toolbox 有新版本时，可以手动更新：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /opt/jetbrains-toolbox
<span class="token function">wget</span> https://download.jetbrains.com/toolbox/jetbrains-toolbox-latest.tar.gz
<span class="token function">sudo</span> <span class="token function">tar</span> <span class="token parameter variable">-xzf</span> jetbrains-toolbox-latest.tar.gz --strip-components<span class="token operator">=</span><span class="token number">1</span>
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">755</span> /opt/jetbrains-toolbox
</code></pre>
    <hr/>
