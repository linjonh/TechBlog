# 星辰诞愿——生日快乐

## 前言

> 今天这篇博客并非技术文章，而是庆祝我可爱的妹妹18岁生日以及介绍我半年以来的学习经历

> **`祝生网站：`[星辰诞愿](http://120.25.176.226:2007/login.html)**(用户列表里第一位就是我妹妹，希望大家能献上自己的祝福，能分享转发更好，我在此感谢大家。如果使用手机，复制**120.25.176.226:2007/login.html**到浏览器**横屏浏览**)![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/6eee1b1c801a4b4d96eee7be81410bfe.png#pic_center)

> **- 一个承诺，半年坚持，无数个深夜的代码——这一切都始于她的一声"哥哥"**

## 1.缘起：在游戏世界中相遇

> 那是一个深秋的夜晚，2024年11月的某个凌晨，窗外的梧桐叶在寒风中簌簌作响。作为游戏群的管理员，照例在荧荧蓝光前守到深夜。突然在凌晨2点时，有个小姑娘(那时还不是我妹)在群里哭诉，她(当时高二)和群里另一位小学生有经济纠纷(具体原因不展开细讲)。虽然萍水相逢，但还是莫名地心疼(各位看官这是真的)，再说了我身为管理员，不可能置身事外呀。于是在凌晨三点多的时候，我找到那个小伙子进行询问(当然不是威胁人家，我拿捏小朋友还是有一手)。中午12点多的时候，我建了一个群，把他们两个拉进来调和矛盾，毫无意外地把问题解决了![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/f5cdd0a65c17455194de47e7c3e41633.jpeg#pic_center)

`......`

> 这个小丫头经常熬夜，打游戏、听小说等等(其实我也是)，这难免让我产生了一些误判。直到某个寻常的冬夜(大概在她期末考试结束后的几天)，我就突然问了她期末考试的情况。各位看官猜怎么着，她成绩稳居年级前三，保底是C9名校那个档次，她家里几个哥哥也都是清华毕业/在校清华(她家在深圳定居)。如果非要在我所有亲戚朋友中进行遍历，或许真能找到这个档次的，只不过总感觉距离我很遥远，和我没啥关系。但那次的交流令我彻夜未眠，想到自己读的双非本科，自己的出身。即使是现在正在写博客的我，心里还有很复杂的情绪，无法用文字来表达

`......`

> 翌日破晓时分，我稳了稳心神，感觉自己不该如此。于是立刻联系指导老师，了解到制作一个网站的所有技术栈，从此开始了我漫长的学习历程。我深刻地记得刚学完Java SE语法时很高兴，就和我的指导老师说：**我要给一个重要的人做一个网站**![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/5c977cc1922043139be0a710584fecab.jpeg#pic_center)

## 2.立志：君子一诺千金

```java
// 那时候我只会写这样的代码
public class Demo {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

```

> 开学后，宿舍的灯总是亮到最晚的那一盏。键盘敲击声在夜里格外清晰，偶尔夹杂着室友的调侃，但只有我自己知道前路的困难。  
>  大一下那学期，我学完了多线程，JVM，文件IO，计算机网络，数据库，前端三剑客。学校的课则是能不去就不去，读过计算机专业的都知道，学校的课没什么作用。到了暑假，刚回家没两天电脑就有点毛病，了解机械革命的人都知道，这来回寄修浪费了将近两周的时间。算算时间，距离妹妹过生日还有不到两个月的时间，我SpringBoot还没开始学，心里有点着急，所以暑假也是天天熬夜赶进度。终于在8月中旬的时候成功将“星辰诞愿”开发完毕。我曾经也跟妹妹说过，要给她做个网站，好像她当时就不在意。时隔半年，现在肯定不知道这回事，但**君子一诺千金**

## 3.回望：永恒的心意

> > 如今，项目完成，静心回望这半年的学习历程，我的所作所为，归根结底是为了自己
>
> > 当然，妹妹的存在是一个极其重要且美好的契机和催化剂。或许她至今都未曾意识到，自己当初随口的一句话，竟然在我这里产生了如此巨大的能量，间接推动我完成了第一次质的飞跃。这份因她而起的动力，成就了最终这份送给她的、独一无二的礼物
>
> > 结语：代码会迭代，但心意不会
>
> > 如果你也在学习编程的路上挣扎，记住：每一个大神都曾经是菜鸟，每一个复杂的系统都从简单的“Hello World”开始。最重要的是开始行动，然后坚持走下去——就像半前那个只会Java语法的我，鼓起勇气许下承诺时一样
