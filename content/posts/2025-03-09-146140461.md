         <code>
          /bin
         </code>
        </strong>
       </td>
       <td>

       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          /sbin
         </code>
        </strong>
       </td>
       <td>

       </td>
      </tr>

       <td>
        <strong>
         <code>

       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          /dev
         </code>
        </strong>
       </td>
       <td>

       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          /etc
         </code>
        </strong>
       </td>
       <td>

       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          /home
         </code>
        </strong>
       </td>
       <td>

         <code>
          /lib
         </code>

          /lib64

    <h2>
     <a id="_50">
     </a>
     <strong>
      二、各目录递归详解
     </strong>
    </h2>
    <h5>
     <a id="1_bin_52">
     </a>
     <strong>
      1. /bin
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：基础用户命令（如
      <code>
       ls
      </code>
      ,
      <code>
       cp
      </code>
      ,
      <code>
       mv
      </code>
      ）
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：无
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         bash
        </code>
        ：Bash shell
       </li>
       <li>
        <code>
         sh
        </code>
        ：系统默认shell（通常是Bash的符号链接）
       </li>
       <li>
        <code>
         date
        </code>
        ,
        <code>
         echo
        </code>
        ,
        <code>
         kill
        </code>
        ,
        <code>
         ps
        </code>
        ,
        <code>
         tar
        </code>
        等
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="2_boot_60">
     </a>
     <strong>
      2. /boot
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：系统引导文件
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /boot/grub/
        </code>
        ：GRUB引导程序配置
        <ul>
         <li>
          <code>
           grub.cfg
          </code>
          ：GRUB菜单配置
         </li>
        </ul>
       </li>
       <li>
        <code>
         /boot/efi/
        </code>
        ：EFI固件引导分区（UEFI系统）
       </li>
      </ul>
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         vmlinuz-&lt;version&gt;
        </code>
        ：内核镜像
       </li>
       <li>
        <code>
         initrd.img-&lt;version&gt;
        </code>
        ：初始内存盘
       </li>
       <li>
        <code>
         config-&lt;version&gt;
        </code>
        ：内核配置文件
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="3_dev_71">
     </a>
     <strong>
      3. /dev
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：设备文件与特殊文件
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /dev/disk/
        </code>
        ：磁盘设备映射
       </li>
       <li>
        <code>
         /dev/pts/
        </code>
        ：伪终端（pty）
       </li>
       <li>
        <code>
         /dev/shm/
        </code>
        ：共享内存（tmpfs）
       </li>
      </ul>
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         sda
        </code>
        ,
        <code>
         sda1
        </code>
        ：硬盘及分区
       </li>
       <li>
        <code>
         tty
        </code>
        ,
        <code>
         console
        </code>
        ：终端设备
       </li>
       <li>
        <code>
         null
        </code>
        ,
        <code>
         zero
        </code>
        ,
        <code>
         random
        </code>
        ,
        <code>
         urandom
        </code>
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="4_etc_82">
     </a>
     <strong>
      4. /etc
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：系统配置文件
     </li>
     <li>
      <strong>
       重要子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /etc/apt/
        </code>
        ：Debian/Ubuntu包管理配置
        <ul>
         <li>
          <code>
           sources.list
          </code>
          ：软件源列表
         </li>
        </ul>
       </li>
       <li>
        <code>
         /etc/ssh/
        </code>
        ：SSH配置
        <ul>
         <li>
          <code>
           sshd_config
          </code>
          ：SSH服务配置
         </li>
        </ul>
       </li>
       <li>
        <code>
         /etc/sysconfig/
        </code>
        ：系统服务配置（CentOS等）
       </li>
       <li>
        <code>
         /etc/network/
        </code>
        ：网络配置（Ubuntu）
       </li>
       <li>
        <code>
         /etc/fstab
        </code>
        ：文件系统挂载表
       </li>
       <li>
        <code>
         /etc/passwd
        </code>
        ：用户账户信息
       </li>
       <li>
        <code>
         /etc/group
        </code>
        ：用户组信息
       </li>
       <li>
        <code>
         /etc/shadow
        </code>
        ：加密密码存储（root权限）
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="5_home_96">
     </a>
     <strong>
      5. /home
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：普通用户家目录
     </li>
     <li>
      <strong>
       结构
      </strong>
      ：
      <ul>
       <li>
        <code>
         /home/username/
        </code>
        ：用户主目录
        <ul>
         <li>
          <code>
           .bashrc
          </code>
          ：用户bash配置
         </li>
         <li>
          <code>
           .ssh/
          </code>
          ：SSH密钥（
          <code>
           id_rsa
          </code>
          ,
          <code>
           id_rsa.pub
          </code>
          ）
         </li>
         <li>
          <code>
           .cache/
          </code>
          ：应用缓存
         </li>
         <li>
          <code>
           .config/
          </code>
          ：用户应用配置
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="6_lib_105">
     </a>
     <strong>
      6. /lib
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：系统库文件
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /lib/modules/
        </code>
        ：内核模块
       </li>
       <li>
        <code>
         /lib64/
        </code>
        ：64位系统库
       </li>
      </ul>
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         ld-linux.so.2
        </code>
        ：动态链接器
       </li>
       <li>
        <code>
         libc.so.6
        </code>
        ：C标准库
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="7_media_114">
     </a>
     <strong>
      7. /media
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：可移动媒体自动挂载点
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        <code>
         /media/usb-drive/
        </code>
        ：U盘挂载点
       </li>
       <li>
        <code>
         /media/cdrom/
        </code>
        ：光驱挂载点
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="8_mnt_120">
     </a>
     <strong>
      8. /mnt
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：手动挂载点
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-bash"><span class="token function">mount</span> /dev/sda1 /mnt  <span class="token comment"># 临时挂载硬盘分区</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="9_opt_127">
     </a>
     <strong>
      9. /opt
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：第三方软件安装目录
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        <code>
         /opt/nginx/
        </code>
        ：Nginx服务器
       </li>
       <li>
        <code>
         /opt/java/
        </code>
        ：Java运行环境
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="10_proc_133">
     </a>
     <strong>
      10. /proc
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：内核与进程信息（伪文件系统）
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         /proc/cpuinfo
        </code>
        ：CPU信息
       </li>
       <li>
        <code>
         /proc/meminfo
        </code>
        ：内存信息
       </li>
       <li>
        <code>
         /proc/loadavg
        </code>
        ：系统负载
       </li>
       <li>
        <code>
         /proc/[pid]/
        </code>
        ：进程详细信息（如
        <code>
         /proc/1/
        </code>
        为init进程）
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="11_root_141">
     </a>
     <strong>
      11. /root
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：root用户家目录
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         .bashrc
        </code>
        ,
        <code>
         .ssh/
        </code>
        ：与普通用户类似
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="12_run_146">
     </a>
     <strong>
      12. /run
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：运行时数据（系统启动后生成）
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         /run/nginx.pid
        </code>
        ：Nginx进程ID
       </li>
       <li>
        <code>
         /run/user/1000/
        </code>
        ：普通用户运行时数据
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="13_sbin_152">
     </a>
     <strong>
      13. /sbin
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：系统管理命令（root权限）
     </li>
     <li>
      <strong>
       关键文件
      </strong>
      ：
      <ul>
       <li>
        <code>
         init
        </code>
        ：系统初始化进程
       </li>
       <li>
        <code>
         shutdown
        </code>
        ,
        <code>
         reboot
        </code>
        ：关机/重启命令
       </li>
       <li>
        <code>
         ifconfig
        </code>
        ,
        <code>
         ip
        </code>
        ：网络配置工具
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="14_srv_159">
     </a>
     <strong>
      14. /srv
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：服务数据存储
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        <code>
         /srv/www/
        </code>
        ：Web服务器内容
       </li>
       <li>
        <code>
         /srv/svn/
        </code>
        ：Subversion仓库
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="15_sys_165">
     </a>
     <strong>
      15. /sys
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：内核设备信息（伪文件系统）
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <ul>
       <li>
        <code>
         /sys/class/net/eth0/
        </code>
        ：网络接口信息
       </li>
       <li>
        <code>
         /sys/devices/
        </code>
        ：硬件设备树
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="16_tmp_171">
     </a>
     <strong>
      16. /tmp
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：临时文件存储（重启后清空）
     </li>
     <li>
      <strong>
       安全特性
      </strong>
      ：
      <ul>
       <li>
        <code>
         sticky
        </code>
        位：仅文件所有者可删除
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="17_usr_176">
     </a>
     <strong>
      17. /usr
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：用户程序与资源（类似Windows的
      <code>
       C:\Program Files
      </code>
      ）
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /usr/bin/
        </code>
        ：普通用户命令（如
        <code>
         firefox
        </code>
        ,
        <code>
         git
        </code>
        ）
       </li>
       <li>
        <code>
         /usr/sbin/
        </code>
        ：系统管理命令（如
        <code>
         apache2ctl
        </code>
        ）
       </li>
       <li>
        <code>
         /usr/share/
        </code>
        ：共享资源（文档、图标）
       </li>
       <li>
        <code>
         /usr/include/
        </code>
        ：C语言头文件
       </li>
       <li>
        <code>
         /usr/lib/
        </code>
        ：程序库文件
       </li>
       <li>
        <code>
         /usr/local/
        </code>
        ：本地编译安装的软件
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="18_var_186">
     </a>
     <strong>
      18. /var
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：可变数据（日志、邮件、缓存）
     </li>
     <li>
      <strong>
       子目录
      </strong>
      ：
      <ul>
       <li>
        <code>
         /var/log/
        </code>
        ：系统日志
        <ul>
         <li>
          <code>
           syslog
          </code>
          ：通用系统日志
         </li>
         <li>
          <code>
           auth.log
          </code>
          ：认证日志
         </li>
         <li>
          <code>
           kern.log
          </code>
          ：内核日志
         </li>
        </ul>
       </li>
       <li>
        <code>
         /var/lib/
        </code>
        ：程序状态数据（如MySQL数据库）
       </li>
       <li>
        <code>
         /var/spool/
        </code>
        ：队列文件（如邮件队列）
       </li>
       <li>
        <code>
         /var/cache/
        </code>
        ：应用缓存（如
        <code>
         /var/cache/apt/
        </code>
        ）
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="_198">
     </a>
     <strong>
      三、特殊目录与文件补充
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        /dev/null
       </strong>
      </p>
      <ul>
       <li>
        黑洞设备，写入的数据会被丢弃，常用于重定向错误输出。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        /dev/zero
       </strong>
      </p>
      <ul>
       <li>
        生成空数据，用于填充文件或内存。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        /lost+found
       </strong>
      </p>
      <ul>
       <li>
        文件系统检查（
        <code>
         fsck
        </code>
        ）时恢复的文件存储。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        /etc/init.d/
       </strong>
      </p>
      <ul>
       <li>
        系统服务启动脚本（System V初始化）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        /etc/systemd/
       </strong>
      </p>
      <ul>
       <li>
        systemd服务配置（现代Linux发行版）。
       </li>
      </ul>
     </li>
    </ol>
