    <h4>
     <a id="buildroot_0">
     </a>
     在buildroot中打开配置选项
    </h4>
    <pre><code class="prism language-bash">$ ./build.sh buildrootconfig
</code></pre>
    <p>
     进入buildroot的配置界面，搜索libwebsockets，选中。
    </p>
    <h4>
     <a id="libwebsockets_7">
     </a>
     更改libwebsockets的配置文件
    </h4>
    <p>
     打开 luckfox-pico/sysdrv/source/buildroot/buildroot-2023.02.6/package/libwebsockets/libwebsockets.mk 文件，添加-DLWS_WITH_MINIMAL_EXAMPLES=ON
    </p>
    <pre><code class="prism language-bash">LIBWEBSOCKETS_CONF_OPTS <span class="token operator">=</span> <span class="token punctuation">\</span>
	<span class="token parameter variable">-DDISABLE_WERROR</span><span class="token operator">=</span>ON <span class="token punctuation">\</span>
	<span class="token parameter variable">-DLWS_WITHOUT_TESTAPPS</span><span class="token operator">=</span>ON <span class="token punctuation">\</span>
	<span class="token parameter variable">-DLWS_IPV6</span><span class="token operator">=</span>ON <span class="token punctuation">\</span>
	<span class="token parameter variable">-DLWS_UNIX_SOCK</span><span class="token operator">=</span>ON <span class="token punctuation">\</span>
	<span class="token parameter variable">-DLWS_WITHOUT_EXTENSIONS</span><span class="token operator">=</span>OFF<span class="token punctuation">\</span>
	<span class="token parameter variable">-DLWS_WITH_MINIMAL_EXAMPLES</span><span class="token operator">=</span>ON
</code></pre>
    <p>
     这样会编译例程文件
    </p>
    <h4>
     <a id="_21">
     </a>
     编译
    </h4>
    <pre><code class="prism language-bash">./build.sh
</code></pre>
    <p>
     重新编译
    </p>
    <h4>
     <a id="_28">
     </a>
     测试
    </h4>
    <p>
     将从新编译好的rootfs镜像刷进去，重启rv1106摄像头，把例程scp上去
     <br/>
     例如，我想测试lws-minimal-ws-server这个程序，可以
    </p>
    <pre><code class="prism language-bash">luckfox-pico/sysdrv/source/buildroot/buildroot-2023.02.6/output/build/libwebsockets-4.3.2/bin$ <span class="token function">scp</span> ./lws-minimal-ws-server root@192.168.9.163:/oem/usr/bin
luckfox-pico/sysdrv/source/buildroot/buildroot-2023.02.6/output/build/libwebsockets-4.3.2/bin$ <span class="token function">scp</span> <span class="token parameter variable">-r</span> ./mount-origin/ root@192.168.9.163:/oem/usr/bin
</code></pre>
    <p>
     进入rv1106摄像头
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>root@luckfox bin<span class="token punctuation">]</span><span class="token comment"># ./lws-minimal-ws-server</span>
</code></pre>
    <p>
     在手机浏览器用两个页面打开
     <br/>
     192.168.9.163:7681
     <br/>
     可以见到一个简易聊天室，能正常对话。
    </p>
