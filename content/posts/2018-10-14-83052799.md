     <blockquote>
      <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
       <span style="font-size:15px;color:rgb(136,136,136);">
        多年后，想必沈向洋的父亲，也没有想到，当年自己从报纸上看到的关于计算机前景的文字，竟可以影响沈向洋的一生，甚至让华人世界多了一位科技领军人物！
       </span>
      </p>
     </blockquote>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/be5885dc9aa8d87f02b4f0484b57af78.jpeg">
       <br/>
      </img>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       微软全球执行副总裁、微软人工智能及微软研究事业部负责人沈向洋
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       沈向洋（Harry Shum），微软目前唯一的华人EVP（Executive Vice President，执行副总裁），负责微软中长期战略制定，亲自带领5000人以上的人工智能团队，他也是目前微软领导结构中位置最高的华人。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       在学术上，沈向洋是计算机视觉和图形学研究的世界级专家，是美国电气电子工程协会院士（IEEE Fellow）、国际计算机协会院士（ACM Fellow），曾任国际计算视觉期刊编委会成员。2017年2月，沈向洋当选美国工程院院士。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       如此功成名就，他却说，自己的人生是被一张报纸改变的，这是怎么回事？一起往下看！
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <br/>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/8902df2bdffaccbf1c9b314d89e22746.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        沈向洋在中国
       </strong>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       沈向洋习惯于，将自己的人生分成，在中国、在美国、在科技行业三部分。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       1966年10月，沈向洋出生在南京溧水区，儿时梦想着成为“伟大的数学家”的他，在13岁就考入了南京工学院（现东南大学）。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而在报专业的时候，他的
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        父亲拿了一份《参考消息》走进家门，兴冲冲地对对他说，“
       </span>
       你应该报计算机专业，报纸上说计算机这个东西不得了！”
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       于是，他在别人还上初中的年纪，就触了机。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       百度总裁张亚勤，和他同年出生，两人中，一个13岁进大学，一个12岁上中科大。人们称他为“神童”，但他始终不这么认为，他将这些都归咎于自己的兴趣。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        从东南大学毕业后，
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         沈向洋又获得了
        </span>
        香港大学电机电子工程系硕士学位。
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         他认为，人到了一定阶段，就能知道自己的数学潜能究竟如何。
        </span>
       </span>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        于是，决定“转行”——他选择了与数学，有着密不可分之关系的计算机领域。
       </span>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <br/>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/35681423b96fcd60bcc7a569defc40bd.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        沈向洋在美国
       </strong>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       1991年，沈向洋进入全美计算机专业排名第一的卡内基梅隆大学计算机学院，成为图灵奖获得者、著名计算机专家拉吉·瑞迪（
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        Raj Reddy
       </span>
       ）教授的学生
      </span>
      <span style="color:rgb(136,136,136);font-size:14px;">
       （拉吉·瑞迪与爱德华·费根鲍姆，因为在人工智能方面的贡献，而分享了1994年的图灵奖）
      </span>
      <span style="font-size:15px;">
       。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       同门师兄，还有李开复和洪小文，后者则是微软亚洲研究院的现任院长。
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/4cad19346f25d1279774dfbd961c57f5.jpeg"/>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="color:rgb(136,136,136);font-size:14px;">
       人工智能先驱、图灵奖得主Raj Reddy
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       沈向洋那一代的学生，对人工智能非常着迷，他那时就开始研究计算机视觉、语音识别和机器人科学等领域了。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       但在90年代，沈向洋从卡内基梅隆大学，获得博士学位时，人工智能相关专业的毕业生，却找不到好工作。因为，那时人工智能，还只是一个愿景。
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       不过他的导师，非常支持他从事计算机视觉方面的研究，后来的
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        沈向洋
       </span>
       研究方向，就成了机器人视觉。
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/1c7c7ee72a12de9134ea5ead2a5ace5c.jpeg"/>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="text-align:center;color:rgb(136,136,136);font-size:14px;">
       年轻时的沈向洋
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       除了世界顶尖的导师，卡内基梅隆
       <span style="font-size:15px;letter-spacing:1px;text-align:center;">
        大学
       </span>
       注重
       <span style="font-size:15px;letter-spacing:1px;text-align:center;">
        团队协作
       </span>
       的科研风格，也一定程度上影响了后来沈向洋的leader气质。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;text-align:center;">
       而同样优秀的麻省理工大学，和卡内基梅隆大学
      </span>
      <span style="font-size:15px;text-align:center;">
       的
      </span>
      <span style="font-size:15px;text-align:center;">
       最大区别就是：麻省理工大学的每个人，都是一头虎，而卡内基梅隆大学出来的，都是一群狼。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;text-align:center;">
       所以，他和小伙伴们，很少单打独斗，都是一个团队一起，做一个大项目。也正因此，很多世界IT公司CTO级别的人物，都是该校出来的。
       <br/>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;text-align:center;">
       <br/>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        <img alt="640?wx_fmt=png" src="https://i-blog.csdnimg.cn/blog_migrate/71d6090e79cf441419041a7759a6a6b8.png"/>
       </strong>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:18px;">
       <strong>
        沈向洋在科技行业
       </strong>
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <br/>
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/11f723b0384cb34931ecbafdf880c9b2.jpeg" style="letter-spacing:1px;text-align:justify;width:468px;"/>
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       1994和1995年暑期，沈向洋曾在美国DEC公司剑桥实验室、及苹果公司交互媒体实验室实习，并在完成博士论文后，加盟位于硅谷的“真实空间”（Real Space）创业公司。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <strong>
       <span style="font-size:15px;">
        加入微软，
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         与世界顶尖天才为伍
        </span>
       </span>
      </strong>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       1995年，比尔·盖茨在西雅图创立了微软研究院，当时他的愿望，是让计算机能听会讲、能看会想。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       因此，微软研究院最早成立的其中三个研究组，是计算机语音、计算机视觉和自然语言处理。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       1996年，沈向洋作为研究员，加入微软研究院，从事计算机视觉研究。对他而言，加入微软研究院，无疑是一种幸运。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       从此，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        沈向洋有
       </span>
       机会，与世界顶尖的天才为伍。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       直到今天，他仍清楚地记得，刚到微软研究院的那个星期，他发现自己离计算机图形学领域的传奇人物吉姆·布林（Jim Blinn）、美国计算机科学家、美国宇航局的喷气推进实验室（JPL）只隔着4个办公室。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       当时他难以抑制心中的兴奋，抓起电话，语无伦次地把这个发现，骄傲地分享给太太。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <strong>
       <span style="font-size:15px;">
        领导微软亚洲研究院
       </span>
      </strong>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        1998年，
       </span>
       加入微软不久后，在盖茨的授意下，微软在北京，成立了亚洲研究院。带着自己的专利与技术，沈向洋旋即飞回国内，开始领导亚洲研究院的相关研究。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       1999年1月15日，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        沈向洋
       </span>
       正式转到中国，成为了第一个研究员，到现在差不多快20年了。他依旧记得，当时的第一份工作，是来选地毯的颜色。
      </span>
     </p>
     <p style="text-align:center;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/48816e57882ab4ef069757909e7e1c6a.jpeg"/>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       在和李开复、张亚勤一同创建微软亚洲研究院的岁月里，他所领导的图形图像组一战成名。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而在全球计算机科研领域最富盛名的国际图形学年会SIGGRAPH上，发表的4篇论文，也让微软亚洲研究院，在学术界站稳了脚跟。
      </span>
      <br/>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       2004年，成为微软亚洲研究院第三任院长之后的沈向洋，将更多微软的先进技术，应用到中国市场，以技术创新为导向，赋能中国、促进中国IT生态圈发展。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
       他在微软的成绩，也一直都有口皆碑。
      </span>
     </p>
     <blockquote>
      <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
       <span style="font-size:15px;color:rgb(136,136,136);">
        微软全球资深副总裁、微软研究院创始人、美国国家工程院院士里克·雷斯（
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
         Rick Rashid
        </span>
        ）博士表示，沈向洋让微软亚洲研究院
        <strong>
         “发展到了难以置信的高度”、“不仅为微软产品和未来战略提供了可信赖的研究保障，还在亚太地区学术界和教育界扮演着越来越重要的角色”。
        </strong>
       </span>
      </p>
      <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
       <span style="color:rgb(136,136,136);font-size:15px;letter-spacing:1px;text-align:justify;">
        美国《麻省理工学院技术评论》曾这样评价他：
        <strong>
         “如果你在感情上难以支持微软，那是你还未曾遇见沈向洋。”
        </strong>
       </span>
      </p>
      <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
       <span style="color:rgb(136,136,136);font-size:15px;letter-spacing:1px;text-align:justify;">
        新华社曾报道：
       </span>
       <strong>
        <span style="font-size:15px;letter-spacing:1px;text-align:justify;color:rgb(136,136,136);">
         “在微软19年，沈向洋是如今微软领导结构中，位置最高的华人，也被外界誉为‘微软在中国的形象大使’。”
        </span>
       </strong>
      </p>
      <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
       <span style="color:rgb(136,136,136);font-size:15px;letter-spacing:1px;text-align:justify;">
        还曾有媒体形容，沈向洋在微软技术研发上，所做的努力，说明了他要
        <strong>
         “
         <span style="color:rgb(136,136,136);font-size:15px;letter-spacing:1px;text-align:justify;">
          让全世界都知道，最先进的技术一定来自中国
         </span>
         ”
        </strong>
        。
       </span>
      </p>
     </blockquote>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       而微软亚洲研究院，也成为中国IT界的“黄埔军校”，聚合了一大批国内的高端人才。
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       后来，李开复、张亚勤等人在离开微软之后，成为了各自领域的领袖。作为留守者，沈向洋于2007年重回美国，并在2014年升任执行副总裁至今。
      </span>
     </p>
     <p style="margin-left:8px;text-align:justify;line-height:1.75em;letter-spacing:1px;">
      <strong>
       <span style="font-size:15px;">
        为
        <strong>
         Bing
        </strong>
        付出7年，为此长满白发
       </span>
      </strong>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       十年前，沈向洋开始负责Bing搜索，这一干就是近7年的时间。作为Bing的奠基人，沈向洋说，自己的满头白发，就是那个时候起来的。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       关于
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        Bing
       </span>
       ，
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        沈向洋曾说，
       </span>
       “这么多年下来以后，我很感激这个团队......现在Bing整个产品线在公司已经是一个盈利非常显著的部门了。”
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <strong>
       <span style="font-size:15px;">
        沈向洋给微软的人工智能带来诸多突破
       </span>
      </strong>
     </p>
     <p style="text-align:center;letter-spacing:1px;margin-left:8px;line-height:1.75em;">
      <img alt="640?wx_fmt=jpeg" src="https://i-blog.csdnimg.cn/blog_migrate/474692b3619aec7aece44b796cb8ab3a.jpeg" style="width:488px;"/>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        沈向洋曾毫不避讳地指出微软的问题，
       </span>
       “从PC时代，微软用Windows横扫世界；但是到互联网时代，微软不再是领头羊；再到移动互联网时代，微软掉队了。”
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       但他同时表示，“
       <span style="font-size:15px;letter-spacing:1px;text-align:justify;">
        在AI时代，微软要定义在这个时代自己的份量，微软完全有能力和决心，在AI时代引领潮流。
       </span>
       ”
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       于是，在沈向洋的领导下，近几年，微软在人工智能、以及相关领域如机器学习、自然语言理解、图像识别等，取得了诸多突破。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       并且衍生出了一系列如小娜、小冰、Skype Translator（Skype实时翻译），以及去年4月公布的Project Oxford等项目。
      </span>
      <br/>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <strong>
       <span style="font-size:15px;">
        后记
       </span>
      </strong>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
      <span style="font-size:15px;">
       可以说，在中国成长、加入微软、又回国助力微软亚洲研究院、到现在领导微软技术与研发部门的沈向洋，正在成为沟通微软与中国创新的桥梁。
      </span>
     </p>
     <p style="text-align:justify;margin-left:8px;line-height:1.75em;letter-spacing:1px;">
