     机器学习中说的正向传递和反向传递是什么意思
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     在机器学习，尤其是深度学习领域，​
     <strong>
      正向传递（Forward Pass）​
     </strong>
     和
     <strong>
      反向传递（Backward Pass）​
     </strong>
     是神经网络训练过程中的两个核心步骤。它们共同构成了训练神经网络的基础框架，通常与
     <strong>
      梯度下降
     </strong>
     算法结合使用。
    </p>
    <h4>
     1. ​
     <strong>
      正向传递（Forward Pass）​
     </strong>
    </h4>
    <p>
     正向传递是指输入数据通过神经网络的前向传播过程，从输入层经过隐藏层最终到达输出层，计算网络的预测结果。具体步骤如下：
    </p>
    <ul>
     <li>
      ​
      <strong>
       输入数据
      </strong>
      ：将训练数据（例如图像、文本等）输入到神经网络的第一层（输入层）。
     </li>
     <li>
      ​
      <strong>
       逐层计算
      </strong>
      ：数据通过每一层的神经元，经过加权求和和激活函数的处理，逐层传递。
     </li>
     <li>
      ​
      <strong>
       输出结果
      </strong>
      ：最终在输出层得到网络的预测值（例如分类标签、回归值等）。
     </li>
     <li>
      ​
      <strong>
       计算损失
      </strong>
      ：将网络的预测值与真实标签进行比较，使用损失函数（如均方误差、交叉熵等）计算误差。
     </li>
    </ul>
    <p>
     正向传递的目的是计算网络的输出和损失，这是后续反向传递的基础。
    </p>
    <h4>
     2. ​
     <strong>
      反向传递（Backward Pass / Backpropagation）​
     </strong>
    </h4>
    <p>
     反向传递是指通过反向传播算法计算损失函数对网络参数的梯度，以便更新网络参数。具体步骤如下：
    </p>
    <ul>
     <li>
      ​
      <strong>
       计算梯度
      </strong>
      ：从输出层开始，逐层计算损失函数对每一层参数的偏导数（梯度）。这涉及到链式法则的应用。
     </li>
     <li>
      ​
      <strong>
       反向传播梯度
      </strong>
      ：将梯度从输出层传递回输入层，计算每一层的权重和偏置的梯度。
     </li>
     <li>
      ​
      <strong>
       更新参数
      </strong>
      ：使用梯度下降（或其变体，如Adam、RMSProp等）根据计算出的梯度更新网络的参数（权重和偏置）。
     </li>
    </ul>
    <p>
     反向传递的目的是优化网络参数，使得损失函数最小化，从而提高网络的预测性能。
    </p>
    <h4>
     3. ​
     <strong>
      正向传递和反向传递的关系
     </strong>
    </h4>
    <ul>
     <li>
      ​
      <strong>
       正向传递
      </strong>
      是计算网络的输出和损失，为反向传递提供基础。
     </li>
     <li>
      ​
      <strong>
       反向传递
      </strong>
      是计算梯度并更新参数，以优化网络性能。
     </li>
     <li>
      两者交替进行：在训练过程中，每一轮迭代（epoch）通常包括一次正向传递和一次反向传递。
     </li>
    </ul>
    <h4>
     4. ​
     <strong>
      简单示例
     </strong>
    </h4>
    <p>
     假设有一个简单的神经网络，输入为 x，输出为 y，损失函数为 L。
    </p>
    <ul>
     <li>
      <p>
       ​
       <strong>
        正向传递
       </strong>
       ：
      </p>
      <ol>
       <li>
        计算网络输出：y=f(x;θ)，其中 θ 是网络参数。
       </li>
       <li>
        计算损失：L=loss(y,ytrue​)。
       </li>
      </ol>
     </li>
     <li>
      <p>
       ​
       <strong>
        反向传递
       </strong>
       ：
      </p>
      <ol>
       <li>
        计算损失对参数的梯度：∂θ∂L​。
       </li>
       <li>
        更新参数：θ=θ−η⋅∂θ∂L​，其中 η 是学习率。
       </li>
      </ol>
     </li>
    </ul>
    <h4>
     总结
    </h4>
    <ul>
     <li>
      ​
      <strong>
       正向传递
      </strong>
      ：数据从输入到输出，计算预测值和损失。
     </li>
     <li>
      ​
      <strong>
       反向传递
      </strong>
      ：从损失反向传播梯度，更新网络参数。
      <br/>
      两者共同构成了神经网络训练的核心过程。
     </li>
    </ul>
