    <h2>
     <a id="docker_2">
     </a>
     安装docker
    </h2>
    <p>
     瑞芯微开发板自带docker环境，可跳过
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 删除老版本的docker</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> remove <span class="token function">docker</span> docker-engine

<span class="token comment"># 开始安装</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> update
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> docker.io

<span class="token comment"># 查看docker版本号</span>
<span class="token function">sudo</span> <span class="token function">docker</span> <span class="token parameter variable">-v</span>
</code></pre>
    <h2>
     <a id="_17">
     </a>
     更改镜像源
    </h2>
    <p>
     为了加快下载速度，更换成国内的源
     <br/>
     编辑文件
     <br/>
     <code>
      vim /etc/docker/daemon.json
     </code>
     <br/>
     文件内容
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">{<!-- --></span>
    <span class="token string">"registry-mirrors"</span><span class="token builtin class-name">:</span> <span class="token punctuation">[</span>
        <span class="token string">"https://docker-0.unsee.tech"</span>,
        <span class="token string">"https://docker-cf.registry.cyou"</span>,
        <span class="token string">"https://docker.1panel.live"</span>
    <span class="token punctuation">]</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     重启docker
     <br/>
     <code>
      service docker restart
     </code>
    </p>
    <h2>
     <a id="dockerfile_36">
     </a>
     编写dockerfile
    </h2>
    <p>
     在
     <a href="https://hub.docker.com/" rel="nofollow">
      docker hub
     </a>
     中选择一个合适的镜像作为基类镜像，然后根据自己的需求，添加一些库。
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 继承Pytorch官方的cuda镜像</span>
FROM loopeai/aibox-rknn-single:v1.3.1

LABEL maintainer <span class="token operator">=</span> <span class="token string">"xxxxxxx@qq.com"</span>
LABEL version <span class="token operator">=</span> <span class="token string">"0.2"</span>
LABEL description <span class="token operator">=</span> <span class="token string">"rknn environment"</span>


RUN <span class="token function">apt-get</span> update <span class="token punctuation">\</span>
	<span class="token operator">&amp;&amp;</span> <span class="token function">apt-get</span> <span class="token function">install</span> nlohmann-json3-dev 

</code></pre>
    <h2>
     <a id="docker_53">
     </a>
     构建docker镜像
    </h2>
    <p>
     <code>
      docker build -t rknn:v1 .
     </code>
     <br/>
     -t：指定镜像的标签（名称和版本）。
     <br/>
     .：表示使用当前目录下的 Dockerfile 进行构建。
     <br/>
     PS:dockerfile要在当前路径下，否则找不到
    </p>
    <h2>
     <a id="docker_59">
     </a>
     构建docker容器
    </h2>
    <p>
     在终端中键入下面的命令
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">docker</span> run <span class="token parameter variable">-it</span> <span class="token punctuation">\</span>
<span class="token parameter variable">--privileged</span><span class="token operator">=</span>true <span class="token punctuation">\</span>
<span class="token parameter variable">--network</span> <span class="token function">host</span> <span class="token punctuation">\</span>
 <span class="token parameter variable">--env</span><span class="token operator">=</span><span class="token string">"DISPLAY"</span>  <span class="token punctuation">\</span>
 <span class="token parameter variable">--env</span><span class="token operator">=</span><span class="token string">"QT_X11_NO_MITSHM=1"</span> <span class="token punctuation">\</span>
 <span class="token parameter variable">--volume</span><span class="token operator">=</span><span class="token string">"/tmp/.X11-unix:/tmp/.X11-unix:rw"</span> <span class="token punctuation">\</span>
 <span class="token parameter variable">--volume</span><span class="token operator">=</span><span class="token string">"/dev:/dev"</span> <span class="token punctuation">\</span>
 <span class="token parameter variable">-v</span> /dev:/dev <span class="token punctuation">\</span>
 <span class="token parameter variable">-v</span> /mnt/sdcard/yp:/yp <span class="token punctuation">\</span>
 <span class="token parameter variable">--name</span> rknn <span class="token punctuation">\</span>
 rknn:v1  /bin/bash
</code></pre>
    <p>
     PS：参数的含义自己去查表，不能完全照搬。
    </p>
    <p>
     下次进入时，查询下镜像
     <br/>
     <code>
      sudo docker ps -a
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6e5cf7c8b65a4ee6853268a8e9191bc1.png"/>
    </p>
    <p>
     模板命令
     <br/>
     <code>
      docker exec -it &lt;容器 ID 或容器名称&gt; &lt;要执行的命令&gt;
     </code>
    </p>
    <ul>
     <li>
      <p>
       -i：表示保持标准输入打开，用于交互式操作。
      </p>
     </li>
     <li>
      <p>
       -t：为命令分配一个伪终端，模拟一个交互式的终端环境。
      </p>
     </li>
     <li>
      <p>
       &lt;容器 ID 或容器名称&gt;&lt;/容器 ID 或容器名称&gt;``：可以通过 docker ps` 命令查看正在运行的容器的 ID 或名称。
      </p>
     </li>
     <li>
      <p>
       &lt;要执行的命令&gt;&lt;/要执行的命令&gt;``：通常是 /bin/bash或/bin/sh`，用于启动一个 shell 会话。
      </p>
     </li>
    </ul>
    <p>
     在这里就是
     <br/>
     <code>
      sudo docker exec -it f286105c362d /bin/bash
     </code>
    </p>
