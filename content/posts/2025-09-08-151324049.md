# linux播放视频出现需要MPEG-4 AAC解码器，H.265(Main Profile)解码器，但是没有安装



1.打开终端

输入

```

sudo apt update
```

更新你的软件包列表，确保安装的是最新版本

2.

```

sudo apt install ubuntu-restricted-extras
```

1. 安装过程中会弹出一个蓝色窗口，显示软件许可协议。使用 `Tab` 键将光标移动到 `<Ok>` 上，然后按回车。在下一个界面，选择 `<Yes>` 并回车表示你接受许可条款。
2. 等待安装完成。这个过程会自动下载并安装包括 MPEG-4 AAC、H.265/HEVC 在内的几乎所有常用媒体解码器。

安装完成后，完全关闭你的视频播放器（如果正在运行的话），然后重新打开并播放视频，应该就可以正常播放了。
