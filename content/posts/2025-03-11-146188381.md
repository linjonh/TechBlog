# ubuntu软件

视频软件，大部分的编码都能适应

```bash
sudo apt install vlc

```

图片软件

```bash
sudo apt install gwenview

```

截图软件

```bash
sudo apt install flameshot

```

```bash
sudo apt remove nautilus-extension-gnome-terminal
sudo apt-get install filemanager-actions
sudo apt-get install nautilus-actions

```
