        SEO优化排名靠前2个最重要问题
        <br/>
        1、锚文本
        <br/>
        <br/>
        当a标签文本子啊psd图当中已经书写好静态文字{即 给前段工程师的文字时确定的}
        <br/>
        我们要去尝试书写title
        <br/>
        <br/>
        对于img要添加的链接标签  一定要书写标签alt、title
        <br/>
        <br/>
        2、网页内容 即代码
        <br/>
        1、标签语义性和重要作用
        <br/>
        <br/>
        title
        <br/>
        meta(当前description还是有用的 author等为结构化滑稽，但是爬虫未必会去进行搜索，keywords已经被废除 即seo已经不再将关键字作为收录标准了)
        <br/>
        <br/>
        标签语义化（h系列 ul ol dl table的合理使用；注意：是否使用strong和em）
        <br/>
        <br/>
        h1-h3（较少使用到h4）
        <br/>
        <br/>
        form的legend（不是很重要）
        <br/>
        <br/>
        table的caption（不是很重要）
        <br/>
        <br/>
        2、代码书写质量
        <br/>
        <br/>
        ①可扩展性——审查标准：代码的大小相对较小 服务器加载速度快
        <br/>
        ②缩进良好 格式规范——乱的话 爬虫会认为是机器生成的代码 从而影响SEo排名
        <br/>
        ③合理嵌套
        <br/>
        <br/>
        3、不要用js去加载处理重要的部分，书写为纯静态【爬虫在进行搜索时 直接查找生成 的静态页面 对于js控制的dom部分 并不能检查】
        <br/>
        <br/>
        4、不要用图片区替换掉重要的文本内容
        <br/>
        <br/>
        5、 关键词（2%-8%） 相对于整篇文章
        <br/>
        <br/>
        6、白帽优化：
        <br/>
        <br/>
        7、黑帽优化：博客作弊 链接养殖场 挂黑链

     </h3>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      SEO全称：Search Engine Optimization，即
      <a href="http://baike.baidu.com/view/7147.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎优化
      </a>
      ，是指为了从搜索引擎中获得更多的免费
      <a href="http://baike.baidu.com/view/66422.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       流量
      </a>
      ，从
      <a href="http://baike.baidu.com/view/3504707.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网站结构
      </a>
      、内容建设方案、用户互动传播、
      <a href="http://baike.baidu.com/view/305736.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       页面
      </a>
      等角度进行合理规划，使网站更适合
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      的检索原则的行为。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      使网站更适合
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      的检索原则又被称为对搜索引擎友好，对搜索引擎友好不仅能够提高SEO的效果，还能提高
      <a href="http://baike.baidu.com/view/274884.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       用户体验
      </a>
      ，因为这样会使搜索引擎中显示的网站相关信息对用户来说更具有吸引力。为了说明什么是网站对搜索引擎友好，我们不妨看看对搜索引擎不友好的网站有哪些特征：
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      1、网页中大量采用图片或者
      <a href="http://baike.baidu.com/view/7641.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Flash
      </a>
      等
      <a href="http://baike.baidu.com/view/184217.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       富媒体
      </a>
      （Rich Media）形式，没有可以检索的文本信息，而SEO最基本的就是文章SEO和图片SEO；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      2、网页没有
      <a href="http://baike.baidu.com/view/182735.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       标题
      </a>
      ，或者标题中没有包含有效的
      <a href="http://baike.baidu.com/view/10083.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       关键词
      </a>
      ；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      3、网页正文中有效
      <a href="http://baike.baidu.com/view/10083.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       关键词
      </a>
      比较少（一般小于整文章2%~8%）；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      4、网站导航系统让搜索引擎“看不懂”；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      5、大量
      <a href="http://baike.baidu.com/view/348756.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       动态网页
      </a>
      让搜索引擎无法检索；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      6、没有被其他已经被搜索引擎收录的网站提供的
      <a href="http://baike.baidu.com/view/147669.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       链接
      </a>
      ；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      7、网站中充斥大量欺骗搜索引擎的垃圾信息，如“
      <a href="http://baike.baidu.com/view/3491570.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       过渡页
      </a>
      ”、“
      <a href="http://baike.baidu.com/view/667579.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       桥页
      </a>
      ”、
      <a href="http://baike.baidu.com/view/19878.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       颜色
      </a>
      与
      <a href="http://baike.baidu.com/view/1174910.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       背景色
      </a>
      相同的
      <a href="http://baike.baidu.com/view/32894.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       文字
      </a>
      ；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      8、网站中缺少
      <a href="http://baike.baidu.com/view/563.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       原创
      </a>
      的内容，完全照搬硬抄别人的内容等。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      关于seo，严谨的定义如下：seo是指在了解
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      <a href="http://baike.baidu.com/view/2914101.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       自然排名
      </a>
      机制的基础上，对网站进行内部及外部的调整优化，改进网站在搜索引擎中
      <a href="http://baike.baidu.com/view/10083.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       关键词
      </a>
      的
      <a href="http://baike.baidu.com/view/2914101.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       自然排名
      </a>
      ，获得更多流量，吸引更多目标客户，从而达到
      <a href="http://baike.baidu.com/view/5422.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网络营销
      </a>
      及
      <a href="http://baike.baidu.com/view/544578.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       品牌建设
      </a>
      的目标。搜索引擎检索
      <a href="http://baike.baidu.com/view/616305.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       原则
      </a>
      是不断更改的，检索原则的更改会直接导致网站在搜索引擎上排名的变化，所以搜索引擎优化并非一劳永逸。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在意识层面，我们对
      <a href="http://baike.baidu.com/view/1047.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SEO
      </a>
      可以形象理解为: 当用户进入
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      ，就好像进入了一个偌大的
      <a href="http://baike.baidu.com/view/17618.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       图书馆
      </a>
      。那么在这个图书馆里面有非常多的分类,分类分成大分类（可以理解为书架）小分类（可以理解为书架上面的格子）以及具体到最小的分类（书本身的内容）——
      <a href="http://baike.baidu.com/view/1138930.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       长尾关键词
      </a>
      。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在
      <a href="http://baike.baidu.com/view/17618.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       图书馆
      </a>
      里面每天都在增加新的内容，那么一个新的网站出现就等于一个分类里面的一本新书出现。那么书里面的书名以及顺序就相对应到网站的
      <a href="http://baike.baidu.com/view/706499.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       TITLE
      </a>
      以及
      <a href="http://baike.baidu.com/view/538066.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Description
      </a>
      。可是如果“别人”（可以指搜索引擎）进入图书馆发现这本书，虽然是这个分类或者一个系列里的书，但是这本书的书名跟之前的一些书的书名完全一样，那么肯定会让读者误会你是在
      <a href="http://baike.baidu.com/view/257319.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       抄袭
      </a>
      别人的书，所以原创将影响到你的排名。那么虽然你的网站是新的，但是其实在刚开始你就不可能有一个好的排名了。这严重影响到日后的
      <a href="http://baike.baidu.com/view/548.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       优化
      </a>
      。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <a href="http://baike.baidu.com/view/7200.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网站优化
      </a>
      能够帮助你提高网页的
      <a href="http://baike.baidu.com/view/714323.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       综合指数
      </a>
      ，如果你的链接已经得到提升，继续保持高质量反向链接数的增加和内容的维护，你的左侧排名会继续保持或提高。除非后期应用了作弊的方式而受到惩罚或停止后期的维护。而竞价广告如果停止了，网站
      <a href="http://baike.baidu.com/view/147669.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       链接
      </a>
      也就会即刻消失。超过95%的搜索引擎使用者会优先考虑
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      给出的常规结果，这其中绝大部分人只有在左侧无法得到满意结果时才会去浏览右侧的广告。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      据调查显示，有87%的网民会利用
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      服务查找需要的信息，而这之中有近70%的搜索者会直接在搜索结果的自然排名的第一页查找自己所需要的信息。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <a href="http://baike.baidu.com/view/1093881.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       竞价广告
      </a>
      的广告展现位置有限，竞价的结果使大量的客户因价格的原因无法排在首页，使得这部分客户很难通过竞价广告获得良好的使用效果。
     </div>

       发展历史
      </span>
      <span class="text_edit editable-title" style="display:block; float:right; padding:6px 0px 0px 8px; margin-left:10px">
       <a class="nslog:1019" href="http://baike.baidu.com/link?url=EoVjGv9zcQoIjYUq3AIGhn4u3Cb_Ko6VVVWhY_rLTfEA8eaVYVEDil-8Y-OVZfh2#" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(136,136,136); height:15px; line-height:16px; padding-left:18px; background-color:transparent; font-size:12px; font-family:宋体; display:block" target="_blank" title="编辑本段">

     </h3>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2000年，
      </strong>
      出现按点击付费(Pay-per-click)的搜索引擎广告模式。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2001年，
      </strong>
      部分
      <a href="http://baike.baidu.com/view/601953.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       分类目录
      </a>
      (中文)开始收费登录，每个网站每年要交纳数百元到数千元不等的费用。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2002年，
      </strong>
      国内陆陆续续有人涉足SEO这一领域。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2003年，
      </strong>
      出现基于内容定位的搜索引擎广告(
      <a href="http://baike.baidu.com/view/105.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Google
      </a>
      AdSense)。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2004年，
      </strong>
      国内潜伏的SEO开始浮出水面，SEO队伍逐步壮大。SEO市场处于混乱无序、违规操作、
      <a href="http://baike.baidu.com/view/470208.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       恶性竞争
      </a>
      的状态。大多数SEO采取个人作坊式经营，公司性运作规模小。SEO培训市场诞生。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2006年，
      </strong>
      随着网络
      <a href="http://baike.baidu.com/view/61624.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       市场竞争
      </a>
      白热化，企业对
      <a href="http://baike.baidu.com/view/44976.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网络公司
      </a>
      的所为和网络产品有了新的认识，企业开始理智对待
      <a href="http://baike.baidu.com/view/5422.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网络营销
      </a>
      市场，随着百度竞价的盛行，企业也认识到了搜索引擎的重要性，同时也伴随着诞生了很多SEO服务公司。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2007年，
      </strong>
      随着SEO信息的普及，网络公司技术的上升，部分公司推出了
      <a href="http://baike.baidu.com/view/361844.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       按效果付费
      </a>
      的SEO服务项目，从网站建设，到关键词定位，到搜索引擎优化全方位服务，整体
      <a href="http://baike.baidu.com/view/157335.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SEM
      </a>
      网络营销方案的推出和实施。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2008年，
      </strong>
      随着SEO服务公司的技术和理念逐渐成熟，部分公司推出了
      <a href="http://baike.baidu.com/view/288270.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       网站策划
      </a>
      服务，服务以效益型网站建设(更加注重网站用户体验)和网站用户
      <a href="http://baike.baidu.com/view/834105.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       转化率
      </a>

     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       1、如何加速蜘蛛更新我的网站？
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      答：那就要勤快一点，天天更新网站，而且要多点原创，否则有点难度，新站百度前一段时间可能天天来，老站只能依靠更新和大量的高质量连接来实现。需要说明的是，
      <a href="http://baike.baidu.com/view/262.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       百度
      </a>
      对新闻源更新很快，一般新内容20分钟内收录。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       2、为什么我们的原创信息被别人转载后，别人的信息排在前面？
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      答：自然排名的衡量因素比较多，常见的有：网页评价或权重（PR\SR）、网页地址的
      <a href="http://baike.baidu.com/view/549611.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       深度
      </a>
      、指向该网页的外部链接、网站在行内的
      <a href="http://baike.baidu.com/view/580675.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       权重
      </a>
      、网站在
      <a href="http://baike.baidu.com/view/1154.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎
      </a>
      的影响、网页的内部链接、网页指向其他网站的链接、标题的合理性、描述的恰当、
      <a href="http://baike.baidu.com/view/10083.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       关键词

      <a href="http://baike.baidu.com/subview/13930/5063889.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       质量
      </a>
      的可读性、
      <a href="http://baike.baidu.com/view/274884.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       用户体验
      </a>
      的体现度等等。这些因素综合起来，如果每样工作都能做好，我们自然能超越其他站点。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

       </a>

      下降，或突然消失。这就不正常了，需要检测查看站内是否有
      <a href="http://baike.baidu.com/view/18723.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">

       4、多个域名对网站有什么影响？
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      答：同一站点使用多个域名属于作弊行为，多域名短期内可以提高收录量，但是会造成权重、补充材料、重复页面等问题，建议只保留一个，其他的可以做301
      <a href="http://baike.baidu.com/view/1245190.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       重定向
      </a>
      。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       5、是不是同一IP上的其他站被百度删除，我也可能受到牵连？
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      答：同一
      <a href="http://baike.baidu.com/view/8370.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       IP
      </a>
      的站点，其他的站受到
      <a href="http://baike.baidu.com/subview/558781/5110214.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">

      </a>
      ，但是自己的站也不一定受到
      <a href="http://baike.baidu.com/view/558781.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       惩罚
      </a>
      ，如果这个IP下的其他站作弊厉害，或者内容很不符合搜索引擎收录标准的话，那么这个IP可能受到了搜索引擎的禁止。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       6、如何快速增加外部链接？
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

      </a>
      。外部链接分数量和质量两个参数，不要忽略链接质量。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

      <a href="http://baike.baidu.com/view/1231085.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       黑帽
      </a>
      做法是
      <a href="http://baike.baidu.com/view/1190337.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Google炸弹
      </a>
      ，
      <a href="http://baike.baidu.com/view/1126860.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       白帽
      </a>
      做法是相关性。Google炸弹的作用至今在引擎都无法杜绝，站点相关性的提高有点难度，需要大量的高质量内容做支撑，在权威站点和知名站点出现类似现象比较多。
     </div>

     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      1、选择稳定的服务器或者
      <a href="http://baike.baidu.com/view/7383.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       虚拟主机

      2、对网站的程序进行综合分析，去掉烦琐的代码，使用简单的
      <a href="http://baike.baidu.com/view/399288.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">

      布局，杜绝漏洞隐患
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      3、定期对服务器系统进行更新，保持安全稳定的
      <a href="http://baike.baidu.com/view/880.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">

      4、网站后台管理员账号和密码尽量复杂

     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <a href="http://baike.baidu.com/view/2755932.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       搜索引擎蜘蛛
      </a>
      是通过链接地址来寻找网页的。搜索引擎蜘蛛名称随搜索引擎的不同而不同。那它的原理是由一个起始链接开始抓取网页内容，同时也采集网页上的链接，并将这些链接作为它下一步抓取的链接地址，如此循环，直到达到某个停止条件后才会停止。停止条件的设定通常是以时间或是数量为依据，可以通过链接的层数来限制网络蜘蛛的爬取。同时页面信息的重要性为客观因素决定了蜘蛛对该网站页面的检索。站长工具中的搜索引擎蜘蛛模拟器其实它就是这个原理。基于这蜘蛛工作原理，站长都会不自然的增加页面关键词出现次数，虽然对密度产生量的变化，但对蜘蛛而言并没达到一定的变化。这在搜索引擎优化过程中应该要避免的。

     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：质量分：是不是页面内容书写的完整，有意义，并且有很实在有用的高质量内容。这个项如果做好就有正3分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Cr：Research – have you researched the keywords people may use to find your content?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：关键词研究分：是否尝试研究那些用户会搜索的关键词了呢。本项为正3分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Cw：Words – do pages use words &amp; phrases you hope they’ll be found for?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：文字：是否使用了用户想要搜索的文字或是短语在内容中了。本项为正2分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Ce：Engage – do visitors spend time reading or bounce away quickly?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：活跃度：是否用户愿意花时间阅读你的内容或是直接就跳走了。 本项为正2分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Cf：Fresh – are pages fresh &amp; about hot topics?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：新鲜度：页面内容是否足够新颖或是关于当下热门的话题。本项为正2分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Vt：Thin – is content thin or shallow &amp; lacking substance?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：内容单薄：页面是否过于单薄或是肤浅而缺少实质有用的内容。本项为负2分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      Va：Ads – is your content ad-heavy especially above the fold?
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      注解：广告多：页面内容是否充斥着较多广告内容特别在页面收关注度最高的上方?本项为负1分。
     </div>

      <span class="headline-content">
       常见术语
      </span>
     </h4>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <a href="http://baike.baidu.com/view/928180.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        SOM
       </strong>
      </a>
      <strong>
       (搜索优化营销)
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <div class="text_pic layoutright" style="margin:0px 0px 10px; padding-bottom:3px; position:relative; float:right; clear:right; text-indent:0px; width:250px; padding-left:20px">

       </p>
      </div>
      <a href="http://baike.baidu.com/view/157335.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SEM
      </a>
      搜索引擎营销和
      <a href="http://baike.baidu.com/view/1047.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SEO
      </a>
      搜索引擎优化。是SEM 和SEO的统称。长期以来，SEM在很多从业者或客户认知为搜索引擎竞价系统（参见百度SEM的官方定义），SEO视为搜索引擎优化即百度自然排名。为了更加规范专业术语，故提出了SOM的概念。即：SOM=SEM+SEO，优化公司给客户提出了SOM的整体解决方案。其中一般都涉及到SEM和SEO两个部分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       链接场（Link Farm）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在 SEO 术语中，链接场是指一个充满链接的页面，这些链接其实没有实际作用，它们只作为链接存在，而没有任何实际的上下文。那些采用运用黑帽 SEO 方法的人
      <a href="http://baike.baidu.com/view/48413.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       利用
      </a>
      链接场，在一个页面中增加大量链接，希望能
      <a href="http://baike.baidu.com/view/276627.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       通过
      </a>
      这种方式使 Google 误认为这个页面很有链接的价值。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       交互链接（Reciprocal Link）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      也称作双向链接，是对一个网页不光有超文本链接，同时对应有和原始网页的链接。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       有机列表（Organic Listing）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      有机列表是 SERP 中的免费列表。有机列表的 SEO 通常涉及改进 Web 站点的实际内容，这往往是在页面或基础架构级别进行的。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       PageRank
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <a href="http://baike.baidu.com/view/1518.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       PageRank
      </a>
      是迷恋
      <a href="http://baike.baidu.com/view/105.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Google
      </a>
      的人们用来测试其站点在 Google 中的排名的一种度量标准。SEO 和搜索引擎营销（SEM）专家也使用这个术语描述网页在 SERP 中的排名以及 Google 根据排名算法给予站点的分数。无论如何定义，PageRank 都是 SEO 的重要部分。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       付费列表（Paid Listing）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      顾名思义，付费列表就是只有在付费后才能列入搜索引擎的服务。根据搜索引擎的不同，付费列表可能意味着：为包含于索引之中、每次点击（PPC）、赞助商链接（Sponsored Link）或者在搜索目标关键词和短语时让站点出现在 SERP 中的其他方式而付费。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       永久重定向（Permanent Redirect）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      也被称为301 重定向，是一条对网站浏览器的指令来显示浏览器被要求显示的不同的URL，当一个网页经历过其URL 的最后一次变化以后时使用。一个永久定向是一种服务器端的重定向，能够被搜索引擎蜘蛛适当地处理。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       排名（Ranking）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      排名是页面在目标关键词的
      <a href="http://baike.baidu.com/view/983960.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SERP
      </a>
      中列出的位置。SEO 的目标是提高 Web 页面针对目标关键词的排名。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       排名算法（Ranking Algorithm）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      排名算法是搜索引擎用来对其索引中的列表进行评估和排名的规则。排名算法决定哪些结果是与特定查询相关的。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       搜索引擎营销（Search Engine Marketing，SEM）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      SEM 这个术语可以与 SEO 互换使用，但 SEM 常常是指通过付费和广告向搜索引擎推销 Web 站点，同时应用 SEO 技术。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       搜索引擎优化（Search Engine Optimization，SEO）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      SEO 就是根据对搜索引擎的吸引力和可见性来优化内容，从而使 Web 页面能够被搜索引擎选中。SEO 主要用来提高有机列表的排名。我使用SEO这个术语描述我推荐的技术，但是其中许多技术也可以归入 SEM 的范畴。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       搜索引擎结果页面（Search Engine Results Page，SERP）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      SERP 是为特定搜索显示的列表或结果。SERP 有时候定义为搜索引擎结果的安排（placement）。根据本系列的目的，我将其称为页面而不是安排。在SEO领域中，在SERP中取得良好的表现就是一切。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       垃圾技术（Spamming）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      垃圾技术是一种欺诈性的SEO手段，它尝试欺骗
      <a href="http://baike.baidu.com/view/2144110.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       爬行器
      </a>
      （Spider），并利用排名算法中的漏洞来影响针对目标关键词的排名。垃圾技术可以表现为多种形式，但是 “垃圾技术” 最简单的定义是 Web 站点用来伪装自己并影响排名的任何技术。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       关键字隐密字 （Hidden Text With Keyword Stuffing）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      是另外一欺骗搜索引擎的做法。通常是指设置关键字的颜色和网页背景颜色一样，或通过 CSS Hidden Attribute （隐密特性） 来达到优化效果。这种做法一旦被Google发现，遭遇也会是该站点从Google的数据库中除名。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       爬行器（Spider）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      爬行器在 Web 上漫游，寻找要添加进搜索引擎索引中的列表。爬行器有时也称为 Web 爬行榜（Webcrawler）或机器人。针对有机列表优化页面也就是为了吸引爬行器的注意。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在
      <a href="http://baike.baidu.com/view/1047.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       SEO
      </a>
      中过多使用flash和
      <a href="http://baike.baidu.com/view/1641.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Ajax
      </a>
      技术，会给搜索引擎带来很多麻烦,搜索引擎还不能很好地索引由flash、ajax表现的网页,原因就是搜索引擎只索引页面,而不会索引
      <a href="http://baike.baidu.com/view/330120.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       应用程序
      </a>
      。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      还有即使搜索引擎做到了完全解释
      <a href="http://baike.baidu.com/view/7641.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       Flash
      </a>
      文件或者ajax应用程序，并能分析和索引它们的相关内容，也还是没有任何办法使用url导航应用程序，对排名没有一点帮助。搜索引擎的主要目标的是将搜索结果提交给用户，但当处理这些媒体信息时，搜索引擎将无法排列这些信息的顺序。总之，Flash和Ajax将会带来更多有创意的设计，也会带来更难侦测的垃圾网站形式。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      谷歌索引的Flash，给的建议就是，网站设计者应该仅在需要使用Flash和 Ajax的地方使用它，应该主要还是多使用
      <a href="http://baike.baidu.com/view/115411.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       HTML语言
      </a>
      设计网站，而将Flash和Ajax作为辅助技术，为用户提供切实的好处。通常，HTML和 JavaScript的混合也基本可以达到使用这些技术的效果。也就是说，在使用Flash和Ajax时应把它们当成页面上的单元，而不是页面本身。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Link Popularity 网站的
      </strong>
      <a href="http://baike.baidu.com/view/3809812.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        链接广泛度
       </strong>
      </a>
      <strong>
       。
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在搜索引擎排名中的作用已得到广泛的认同和重视。实际上，即使你没有在GOOGLE上提交你的站点，但与其它网站作了链接，Google也可能收录你的网站。搜索引擎还可能完全依据你的链接广泛度决定你的网站排名。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Internal Links 内部链接：
      </strong>
      本网站内部网页之间的链接
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       External Links外部链接：
      </strong>
      本网站外部的链接，一般是指其他网站连到本网站的链接
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Backward Links或Incoming/Inbound Links反向/导入
      </strong>
      <a href="http://baike.baidu.com/view/147669.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        链接
       </strong>
      </a>
      <strong>
       ：
      </strong>
      意义与“外部链接”接近，指其他网站连到本网站的链接
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Outgoing/Outbound Links导出链接：
      </strong>
      指从本网站连到其他网站的链接
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Search Engine Positioning 搜索引擎定位
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Search Engine Ranking 搜索引擎排名
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       External Files外部文件存储
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      把
      <a href="http://baike.baidu.com/view/16168.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       JavaScript
      </a>
      文件和
      <a href="http://baike.baidu.com/view/15916.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       CSS
      </a>
      文件分别放在JS和CSS外部文件中。这样做的好处是把重要的页面内容放到页面顶部，同时能缩小文件大小。有利于搜索引擎快速准确地抓取页面重要内容。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Keyword Density 关键词密度
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       ROBOTS 搜索机器人
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Affiliate Marketing 联属网络营销（会员制营销、网站联盟等）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      联属网络营销 指的是一种网站A为网站B放置广告按钮，然后从为网站B带来的销售额中获得回佣的一种广告系统。某些广告主通过这种方式获得市场信息而不是现金销售。这种方式被营销费用比较紧张的新网站采用得比较普遍。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Cost-Per-Click （简称
      </strong>
      <a href="http://baike.baidu.com/view/259728.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        CPC
       </strong>
      </a>
      <strong>
       ）按点击数付费
      </strong>
      <div class="text_pic layoutright" style="margin:0px 0px 10px; padding-bottom:3px; position:relative; float:right; clear:right; text-indent:0px; width:220px; padding-left:20px">
       <a class="nslog:1199" href="http://baike.baidu.com/picview/1047/1047/0/faedab64034f78f0bf58016e79310a55b2191cdf.html" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194); height:177px; width:220px; display:block; position:relative" target="_blank">
        <span class="imgBorder" style="border:1px solid rgb(224,224,224); position:absolute; top:0px; left:0px; display:block; height:175px; width:218px">
        </span>
        <img alt="SEO" class="editorImg log-set-param" height="177" src="https://i-blog.csdnimg.cn/blog_migrate/7f486e256a80b7bbd2718aeb09da32ac.jpeg" style="border:0px; display:block" title="SEO" width="220"/>
       </a>
       <p class="pic-info" style="margin-top:0px; margin-bottom:0px; padding:8px 7px; color:rgb(85,85,85); font-size:12px; font-family:宋体; word-wrap:break-word; word-break:break-all; line-height:15px; min-height:12px; border-width:0px 1px 1px; border-right-style:solid; border-left-style:solid; border-right-color:rgb(224,224,224); border-left-color:rgb(224,224,224); border-bottom-style:solid; border-bottom-color:rgb(204,204,204); zoom:1">
        SEO

      </div>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      联属网络营销管理系统记录每个客人在联属会员网站上点击到商家网站的文字的或者图片的链接（或者Email链接）次数，商家（Merchant）按每个点击多少钱的方式支付广告费。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Cost-Per-Lead （简称
      </strong>
      <a href="http://baike.baidu.com/view/56177.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        CPL
       </strong>
      </a>
      <strong>
       ）或Cost-Per-Acquisition （简称
      </strong>
      <a href="http://baike.baidu.com/view/1946.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        CPA
       </strong>
      </a>
      <strong>
       ）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      按引导数付费或访问者通过联属会员的链接进入商家网站后，如果填写并提交了某个表单，管理系统就会产生一个对应给这个联属会员的引导（Lead）记录，商家按引导记录数给会员付费。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Cost-Per-Sale (简称
      </strong>
      <a href="http://baike.baidu.com/view/204275.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        CPS
       </strong>
      </a>
      <strong>
       ) 按销售额付费
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      商家只在联属会员的链接介绍的客人在商家网站上产生了实际的购买行为后（大多数是在线支付）才给联属会员付费，一般是设定一个佣金比例（销售额的10%到50%不等）。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Pay For Performance （按效果付费）
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      联属营销按效果付费的营销方式无论对于商家还是联属会员都是比较容易接受的。由于网站的自动化流程越来越完善，
      <a href="http://baike.baidu.com/view/1603565.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       在线支付系统
      </a>
      也越来越成熟，越来越多的联属网络营销系统采用按销售额付费的方法。由于这种方法对商家来说是一种零风险的广告分销方式，商家也愿意设定比较高的佣金比例，这样就使得这种方式的营销系统被越来越多地采用。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Open Directory Project (简称ODP) 目录索引
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Full Text Search Engine 全文搜索引擎
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Search Index/Directory 目录索引类搜索引擎
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       META Search Engine 元搜索引擎
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Spider 俗称“蜘蛛”
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Indexer 检索程序
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Free For All Links （简称
      </strong>
      <a href="http://baike.baidu.com/view/1472508.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        FFA
       </strong>
      </a>
      <strong>
       ）免费链接列表
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Stop Words/Filter Words 停用词/过滤词
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      这两者意义一样，都是指一些太常用以至没有任何检索价值的单词，搜索引擎碰到这些词时一般都会过滤掉。因此为节省空间，应尽量避免使用这一类的词，尤其是在对文字数量有严格限制的地方。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Spam：搜索引擎垃圾技术的统称
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Frame Sets 框架结构
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      有些搜索引擎（如FAST）是不支持框架结构的，他们的“蜘蛛”程序无法阅读这样的网页。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Image Maps 图象区块
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      当“蜘蛛”程序遇到这种结构时，往往会感到茫然不知所措。因此尽量不要设置Image Map链接。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Dynamic Pages 动态网页
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      通过程序和数据库连接制作的网页，任何地址中带“？”号、“&amp;”号（及其他类似符号）的网页都会把“蜘蛛”程序挡在门外。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Invisable/hidden text 隐藏文本内容
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      意欲在不影响网站美观的前提下通过包含大量关键词的网页提高关键词相关性得分，从而达到改善搜索引擎排名的目的。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Invisable/hidden links 隐藏链接
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      意欲在不影响网站美观的前提下通过在其它页面添加指向目标优化页的隐形链接，通过提升链接得分而改善搜索引擎排名。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Misleading Words 误导性关键词
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在页面中使用与该网页毫不相干的误导性关键词来吸引查询该主题的访问者访问网站。这种做法严重影响了搜索引擎所提供结果的相关性和客观性，为搜索引擎所深恶痛绝。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Repeated Words 重复性关键词
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      这种作弊技术也被称为“关键词堆砌欺骗(Keyword Stuffing)”，它利用搜索引擎对网页正文和标题中出现的关键词的高度关注来对关键词进行不合理的(过度)重复。类似的其它做法还包括在HTML元标识中大量堆砌关键字或使用多个关键字元标识来提高关键词的相关性。这种技术很容易被搜索引擎察觉并受到相应惩罚。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Keyword Stuffing 关键词堆砌
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Cloaked Page 隐形页面
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      对实际访问者或搜索引擎任一方隐藏真实网站内容，以向搜索引擎提供非真实的搜索引擎友好的内容提升排名。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Deceptive redirects 欺骗性重定向
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      指把用户访问的第一个页面(着陆页)迅速重定向至一个内容完全不同的页面。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Shadow Domain 鬼域
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      这是最常见的欺骗性重定向技术，通过欺骗性重定向使用户访问另外一个网站或页面。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Meta Refresh刷新标识
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Doorway Page 门页
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      也叫“Bridge/Portal/Jump/Entry Page”。是为某些关键字特别制作的页面，专为搜索引擎设计，目的是提高特定关键词在搜索引擎中的排名所设计的富含目标关键词的域名，且重定向至另一域名的真实网站。搜索引擎的Spiders往往忽略对那些自动重定向到其它页的页面的检索。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Mirror Sites
      </strong>
      <a href="http://baike.baidu.com/view/275023.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        镜象
       </strong>
      </a>
      <strong>
       站点
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      通过复制网站或网页的内容并分配以不同域名和服务器，以此欺骗搜索引擎对同一站点或同一页面进行多次索引。大多数搜索引擎都提供有能够检测镜象站点的适当的过滤系统，一旦发觉镜象站点，则源站点和镜象站点都会被从索引数据库中删除。
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Link Spamming 作弊链接/恶意链接
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       link farms 链接工厂
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Bulk Link Exchange Programs大宗链接交换程序
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Cross Link
      </strong>
      <a href="http://baike.baidu.com/view/2194688.htm" rel="nofollow noopener noreferrer" style="text-decoration:none; color:rgb(19,110,194)" target="_blank">
       <strong>
        交叉链接
       </strong>
      </a>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      <strong>
       Doorway Domain 门域
      </strong>
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      专为提高特定关键词在搜索引擎中的排名所设计的富含目标关键词的域名，然后重定向至其它域名的主页。由于搜索引擎一般忽略自动重定向至其它页的页面的检索，所以不提倡使用这种技术。
     </div>

      、关

     </h4>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">
      在一个网址前加"site:"，可以限制只搜索某个具体网站、网站频道、或某域名内的网页；
     </div>
     <div class="para" style="color:rgb(51,51,51); margin-bottom:5px; text-indent:2em">

     </div>
