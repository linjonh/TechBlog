    <h4>
     <a id="__5">
     </a>
     一. 配置问题
    </h4>
    <p>
     我的系统是Ubuntu18.04
    </p>
    <ol>
     <li>
      <p>
       在做实验时需要打开Windows中的docx文档
       <br/>
       关于Ubuntu安装WPS教程
       <br/>
       <a href="https://blog.csdn.net/ErMiaoYY/article/details/82782433">
        https://blog.csdn.net/ErMiaoYY/article/details/82782433
       </a>
      </p>
     </li>
     <li>
      <p>
       我的Windows一直用搜狗输入法，登录账号可以关联，而且自带的输入法不好用
       <br/>
       搜狗输入法下载安装教程
       <br/>
       <a href="https://blog.csdn.net/areigninhell/article/details/79696751#commentBox">
        https://blog.csdn.net/areigninhell/article/details/79696751#commentBox
       </a>
      </p>
     </li>
     <li>
      <p>
       操作系统实验需要在linux环境下采用C/C++编程
       <br/>
       关于C/C++的配置
       <br/>
       主要的是：sudo apt-get install build-essential
      </p>
     </li>
     <li>
      <p>
       采用的IDE是CodeBlocks，关于CodeBlocks的使用教程
       <br/>
       <a href="https://blog.csdn.net/wushirenfeig/article/details/72487280#commentBox">
        https://blog.csdn.net/wushirenfeig/article/details/72487280#commentBox
       </a>
       <br/>
       也使用了Geany，安装：Ubuntu软件-&gt;开发工具-&gt;Geany，点击安装
      </p>
     </li>
     <li>
      <p>
       实验中的运行结果需要截屏保存，关于截屏设置，我习惯QQ的 ctr l+ alt + a
       <br/>
       在Ubuntu中可以自己设置
       <br/>
       设置 -&gt; 设备 -&gt; 键盘 -&gt; 截图，找到对应的截屏方案，输入快捷键即可
      </p>
     </li>
    </ol>
