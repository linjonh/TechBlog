    <h4>
     1.
     <a href="http://jquerymobile.com/" rel="nofollow noopener noreferrer" target="_blank">
      jQuery Mobile
     </a>
     :
     <a href="http://jquerymobile.com/" rel="nofollow">
      http://jquerymobile.com/
     </a>
    </h4>
    <p>
     假如你不熟悉
     <a href="http://www.open-open.com/ajax/jquery.htm" rel="nofollow" title="jquery插件">
      jQuery
     </a>
     ，可以选择
     <a href="http://cpojer.net/MooTools/mobile/Demos/" rel="nofollow">
      MooTools Mobile
     </a>
     , 一个由MooTools核心开发人员Christoph Pojer开发的一个MooTools插件。还有一个是
     <a href="http://dojotoolkit.org/features/mobile" rel="nofollow">
      Dojo Mobile
     </a>
     , 这个利用了Dojo API来帮助开发人员快速构建一个用于移动平台的应用。
    </p>
    <h4>
     2.
     <a href="http://www.appcelerator.com/products/titanium-mobile-application-development/" rel="nofollow">
      Titanium Mobile
     </a>
     :
     <a href="http://www.appcelerator.com/products/titanium-mobile-application-development/" rel="nofollow">
      http://www.appcelerator.com/products/titanium-mobile-application-development/
     </a>
    </h4>
    <p>
     Titanium Mobile支持原生的iOS和Android UI元素如Table views, tabs, switches和popovers。它提供能够与移动设备的摄像头和本地文件存储系统相交互的特性。
    </p>
    <h4>
     3.
     <a href="http://the-m-project.net/" rel="nofollow noopener noreferrer" target="_blank">
      The-M-Project
     </a>
     :
     <a href="http://the-m-project.net/" rel="nofollow">
      http://the-m-project.net/
     </a>
    </h4>
    <p>
     The-M-Project是另外一个强大的JavaScript框架，它利用HTML5新的特性来更好和更简便地开发移动应用。
    </p>
    <p>
     这个框架遵循著名的MVC软件架构模式。
     <br/>
     它还支持离线，所以你的用户可以在没有连接网络的情况下继续操作（当下次有连线的时候，再将数据同步到服务器中）。提供优秀的文档(这个项目拥有一个引导新用户入门的
     <a href="http://the-m-project.net/getStarted.html" rel="nofollow noopener noreferrer" target="_blank">
      开发指南
     </a>
     ).
    </p>
    <p>
     可以查看The-M-Project提供的
     <a href="http://the-m-project.net/showcase.html" rel="nofollow noopener noreferrer" target="_blank">
      示例
     </a>
     来对该项目有一个初目的了解。这些示例包括：
     <a href="http://the-m-project.net/apps/todos2/index.html" rel="nofollow">
      ToDo App
     </a>
     （待办事项目应用） 和
     <a href="http://the-m-project.net/apps/kitchensink/index.html" rel="nofollow">
      KitchenSink
     </a>
     (这个示例包括这个框架提供的所有UI元素)。
    </p>
    <p>
    </p>
    <h4>
     4.
     <a href="http://joapp.com/" rel="nofollow">
      Jo
     </a>
     ：
     <a href="http://joapp.com/demos.html" rel="nofollow">
      http://joapp.com/demos.html
     </a>
    </h4>
    <p>
     Jo这个框架可用于开发那支持HTML5的移动设备，如iOS, webOS, Android和Chrome OS等平台。
    </p>
    <p>
     它拥有标准，类原生的UI元素比如用于屏幕登录的Web表单控件，还有弹出小部件可用于在用户点击界面时提供一些额外的信息。
    </p>
    <p>
     可以查看其网站提供的
     <a href="http://joapp.com/demos.html" rel="nofollow noopener noreferrer" target="_blank">
      示例页面
     </a>
     ，它例子展示了在多种移动设备平上的Jo应用情况。
    </p>
    <h4>
     5.
     <a href="http://xuijs.com/" rel="nofollow noopener noreferrer" target="_blank">
      xui.js
     </a>
     ：
     <a href="http://xuijs.com/" rel="nofollow">
      http://xuijs.com/
     </a>
    </h4>
    <p>
     有时候你可能需要一个超轻量级的框架，只要拥有用于开发标准客户端所需要的功能如：DOM操作，事件处理，
     <a href="http://sixrevisions.com/category/ajax/" rel="nofollow">
      Ajax
     </a>
     和一些动画效果。如果是这样的话，这个框架刚好非常适合你。整个框架采用GZIP压缩完之后只有4.2 KB。
    </p>
    <p>
     6.
     <a href="http://uxebu.github.com/embedjs/" rel="nofollow">
      EmbedJS
     </a>
     ：
     <a href="http://uxebu.github.com/embedjs/" rel="nofollow">
      http://uxebu.github.com/embedjs/
     </a>
    </p>
    <p>
     EmbedJS是一个用于嵌入式设备的JavaScript框架如：移动电话，TVs、tablets和so forth。
    </p>
    <p>
     EmbedJS强大之处在于，它拥有专门为特定平台和浏览器如iOS, Firefox, Android等提供相应的开发版本。这样就能够以最少的代码，为用户提供最佳的体验。而且假如你喜欢自己定制，可以利用其提供的
     <a href="https://github.com/uxebu/embedjs-tools#readme">
      EmbedJS Build tool
     </a>
     工具实现。
    </p>
    <p>
     EmbedJS基于Dojo实现，所以你如果熟悉Dojo API语法，那EmbedJS将是你最佳的选择。
    </p>
    <h4>
     7.
     <a href="http://zeptojs.com/" rel="nofollow noopener noreferrer" target="_blank">
      zepto.js
     </a>
     ：
     <a href="http://zeptojs.com/" rel="nofollow">
      http://zeptojs.com/
     </a>
    </h4>
    <p>
     zepto.js是一个专为mobile WebKit浏览器(如：Safari和Chrome)而开发的一个JavaScript框架。
     <br/>
     它标榜自己在其简约的开发理念，能够帮助开发人员简单、快速地完成开发交付任务。更重要的是这个JS框架，是超轻量级的，只有5KB。
    </p>
    <p>
     zepto.js的语法借鉴并且兼容
     <a href="http://www.open-open.com/ajax/jquery.htm" rel="nofollow noopener noreferrer" target="_blank">
      jQuery
     </a>
     。
    </p>
    <h4>
     8.
     <a href="http://dhtmlx.com/touch/" rel="nofollow noopener noreferrer" target="_blank">
      DHTMLX Touch
     </a>
     ：
     <a href="http://dhtmlx.com/touch/" rel="nofollow">
      http://dhtmlx.com/touch/
     </a>
    </h4>
    <p>
     DHTMLX Touch是一个免费的HTML5/JavaScript框架，专门为触摸屏设备而优化。为您带来快速开发工艺精美的移动Web应用程序的能力。
    </p>
    <p>
     <a href="http://dhtmlx.com/touch/designer.shtml" rel="nofollow">
      DHTMLX Touch UI Designer
     </a>
     是一个可视化的编辑器用于构建移动用户界面。它能够帮您以最少的编码构建一流的用户界面。
    </p>
    <p>
     在其主页上提供一些示例可以展示DHTMLX Touch强大的用户界面。包括一个
     <a href="http://dhtmlx.com/touch/samples/apps/menu/index.html" rel="nofollow">
      menu app for the iPad
     </a>
     (适用于开发餐厅应用) 和Book Shop (一个电子书店应用)。
    </p>
    <h4>
     9.
     <a href="http://mobilizejs.com/" rel="nofollow">
      Mobilize.js
     </a>
     ：
     <p>
      <strong>
       <a href="http://mobilizejs.com/" rel="nofollow">
        http://mobilizejs.com/
       </a>
       <a href="http://mobilizejs.com/" rel="nofollow noopener noreferrer" target="_blank">
        Mobilize.js
       </a>
      </strong>
      是一个开源的
      <strong>
       HTML5-JavaScript框架
      </strong>
      用于从任何现在有标准网站快速，简便地构建移动网站。
     </p>
     <p>
      这个框架其实就是将需要在移动设备上显示的部分页面以jQuery Mobile的默认主题显示，而不是实现一个全新完整的移动页面。
     </p>
     <p>
      <strong>
       Mobilize.js
      </strong>
      可用于任意网站，但对于基于WordPress和Sphinx的网站不需要使用该框架转换，因为已经有非常多的插件可以使用。
     </p>
     <p>
      对于
      <a href="http://sixrevisions.com/category/wordpress/" rel="nofollow" title="WordPress category on Six Revisions">
       WordPress
      </a>
      用户可以使用其提供的
      <a href="http://cdn.mobilizejs.com/docs/trunk/manual/integrations/wordpress.html#installing-mobilize-js-for-wordpress-plug-in" rel="nofollow">
       Mobilize.js WordPress plugin
      </a>
      插件。
     </p>
    </h4>
    <h4>
     10.
     <a href="http://chocolatechipmobile.wordpress.com/" rel="nofollow">
      ChocolateChip Mobile
     </a>
    </h4>
    <p>
     ChocolateChip Mobile是一个移动Web应用开发JavaScript框架。它的语法类似于jQuery和Prototype。
    </p>
    <p>
     与jQuery相似并不仅停留在语法上。比如可以像jQuery一样通过绑定和定义事件处理。并拥有像
     <code>
      .css
     </code>
     和
     <code>
      .toggleClass
     </code>
     这样的方法。
    </p>
    <p>
     你可以从
     <a href="https://github.com/rbiggs/chocolatechip">
      source code off GitHub
     </a>
     获得ChocolateChip Mobile的源码。
    </p>
    <p>
     <br/>
     原文：
     <a href="http://sixrevisions.com/javascript/mobile%C2%A0web-development-frameworks/" rel="nofollow">
      http://sixrevisions.com/javascript/mobile%C2%A0web-development-frameworks/
     </a>
    </p>
