    <h3>
     <a id="_root__0">
     </a>
     允许 root 登录桌面
    </h3>
    <p>
     修改 /etc/pam.d/gdm-autologin , /etc/pam.d/gdm-password
     <br/>
     加
     <code>
      #
     </code>
     以注释掉
     <code>
      auth required pam_succeed_if.so user != root quiet_success
     </code>
    </p>
    <h3>
     <a id="_root__ssh__3">
     </a>
     允许 root 通过 ssh 登录
    </h3>
    <p>
     修改 /etc/ssh/sshd_config
    </p>
    <pre><code>...
#PermitRootLogin prohibit-password
PermitRootLogin yes
...
</code></pre>
    <h3>
     <a id="_root_11">
     </a>
     自动登录 root
    </h3>
    <p>
     修改 /etc/gdm3/custom.conf
    </p>
    <pre><code>[daemon]
AutomaticLoginEnable=true
AutomaticLogin=root
</code></pre>
    <h3>
     <a id="_Ubuntu__18">
     </a>
     允许 Ubuntu 客户机与主机拖拽传递文件
    </h3>
    <ol>
     <li>
      <code>
       sudo apt install open-vm-tools
      </code>
     </li>
     <li>
      <code>
       sudo apt install open-vm-tools-desktop
      </code>
     </li>
     <li>
      修改 /etc/gdm3/custom.conf
      <br/>
      <code>
       WaylandEnable=false
      </code>
     </li>
    </ol>
    <h3>
     <a id="_ssh_24">
     </a>
     安装 ssh
    </h3>
    <ol>
     <li>
      <code>
       sudo apt install openssh-server
      </code>
     </li>
     <li>
      进入 /etc/ssh/sshd_config 改 PermitRootLogin yes
     </li>
     <li>
      <code>
       sudo systemctl restart sshd
      </code>
     </li>
    </ol>
