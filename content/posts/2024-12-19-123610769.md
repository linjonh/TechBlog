# java 序列化 alt+enter生成序列号

### java 一键生成序列

类继承了Serializable接口之后，使用alt+enter快捷键自动创建序列化id

idea默认设置没有开启：alt+enter (不同系统和自定义的配置可能会快捷键可能略有不同)发现并没有提供
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/a22acdcae6c4903f79ba71fc019da921.png)
  
在Idea左上角点击
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/c45d8b751bde89e454393767cfd80a42.png)
  
然后输入serializations issues，点击without serialVersionUid。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/3aec7e33f328e8a57c9c41b33ece5830.png)
  
点击即可

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/c4bb7f7f5c8a2478d25b333f4948045c.png)
