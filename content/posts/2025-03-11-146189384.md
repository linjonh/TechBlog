      1.缺失的环节【算法赛】 - 蓝桥云课
     </a>
    </p>
    <p>
     水题.
    </p>
    <p>
     先考虑朴素想法,枚举len从1到n,然后内部循环枚举.设枚举开始时值为x
    </p>
    <p>
     这个时候发现,
    </p>
    <p>
     只需要对x去掉最高位(如果为1),然后左移一位,再加上右面新增位就可以获得下一位值
    </p>
    <p>
     显然有pow(2,len) &gt; n - len + 1时必然有解.那么右边看成n放缩下,差不多就是logn时
    </p>
    <p>
     时间复杂度O(n * logn)显然可以过.
    </p>
    <p>
     代码
    </p>
    <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
string s;
int n;
const int N = 1e5+10;
bool vis[N];
int main()
{
    scanf("%d",&amp;n);
    cin&gt;&gt;s;
    s = " " + s;
    for(int len = 1; len&lt;=n; len++)
    {
        int cur =0;
        for(int i =1; i&lt;= len; i++)
        {
            cur = (cur&lt;&lt;1) + s[i] - '0';
        }
        vis[cur] = true;
        
        for(int i = len + 1; i&lt;= n; i++)
        {

            if((cur&gt;&gt;(len-1)) &amp; 1)
                cur -= 1 &lt;&lt; (len-1);
            cur&lt;&lt;=1;
            cur += s[i] - '0';
            vis[cur] = true;
        }
        if(len == 1 &amp;&amp; vis[0] == false)
        {
            cout&lt;&lt;0&lt;&lt;endl;
            return 0;
        }
        for(int i = 1&lt;&lt;(len-1); i&lt; 1&lt;&lt;len; i++)
        {
            if(vis[i])
                continue;
            cout&lt;&lt;i&lt;&lt;endl;
            return 0;
        }

    }
    return 0;
}</code></pre>
