# python中安装excel使用的模块

### 1. 在官网下载模块安装包

* xlrd：
  <https://pypi.python.org/pypi/xlrd>
* xlwt：
  <https://pypi.python.org/pypi/xlwt>

### 2. 选择whl文件下载

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/e68c54d5c406d3c271800cd97a414b05.png)

### 3. 在命令行窗口用pip3命令安装这两个模块

> pip3 install xlrd-1.2.0-py2.py3-none-any.whl
>   
> pip3 install xlwt-1.2.0-py2.py3-none-any.whl
