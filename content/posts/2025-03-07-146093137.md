    <h2>
     <a id="ffmpeg_0">
     </a>
     一、ffmpeg提取音频数据
    </h2>
    <h3>
     <a id="11__2">
     </a>
     1.1 保留封装格式
    </h3>
    <p>
     这里保留原有的封装格式，如mp4、mkv、ts等等，但只提取容器中的音频或视频
    </p>
    <p>
     提取音频使用-v 忽略视频即可：
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-acodec</span> copy <span class="token parameter variable">-vn</span> 1_audio.mp4
</code></pre>
    <p>
     提取生成的mp4容器只有音频：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/87b2a9280f4f49d2b7f808e091cece26.png"/>
    </p>
    <p>
     提取视频则使用-an忽略音频即可
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-vcodec</span> copy <span class="token parameter variable">-an</span> 1_video.mp4
</code></pre>
    <p>
     同理，mp4容器中只有视频没有音频
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5f1eefa1e7ad41148d1597ce9480ec3a.png"/>
    </p>
    <h3>
     <a id="12__26">
     </a>
     1.2 提取视频，不保留封装格式
    </h3>
    <p>
     与保留封装格式类似，只需要把后缀名改为.h264、.h265等视频编码格式即可
    </p>
    <pre><code class="prism language-shell"> ffmpeg <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-vcodec</span> copy <span class="token parameter variable">-an</span> 1_video.h264
</code></pre>
    <p>
     只有视频编码数据，没有容器数据
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/910edf377d5f488183e5dea7fb2f9483.png"/>
    </p>
    <p>
     或者指定编码器转换数据格式，比如libx265
    </p>
    <pre><code class="prism language-shell"> ffmpeg <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-vcodec</span> libx265 <span class="token parameter variable">-an</span> <span class="token parameter variable">-t</span> <span class="token string">"10"</span>  1_video.h265
</code></pre>
    <p>
     这里的视频编码格式就hevc即h265
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4ac470b5da5740778535f3ab206af2b9.png"/>
    </p>
    <h3>
     <a id="13__46">
     </a>
     1.3 提取音频数据
    </h3>
    <p>
     保留原来的编码格式
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span> <span class="token number">1</span>.mp4 <span class="token parameter variable">-acodec</span> copy <span class="token parameter variable">-vn</span> 1_audio.aac
</code></pre>
    <p>
     使用ffprobe查看：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fdeaf26faf734a8f917a436ec81763bf.png"/>
    </p>
    <p>
     也可以使用音频编码器转换格式，如libmp3lame等等
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-i</span>  <span class="token number">1</span>.mp4 <span class="token parameter variable">-acodec</span> libmp3lame <span class="token parameter variable">-vn</span>  1_audio.mp3
</code></pre>
    <p>
     格式成功转换为mp3
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4e1c0e523e104b90a0d2c6bc70b877a5.png"/>
    </p>
