    <p>
     在MATLAB中实现PID控制仿真可以通过代码编程或Simulink图形化建模两种方式完成。以下是两种方法的详细操作步骤和示例：
    </p>
    <hr/>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3d8b3bb58c8d41edabb91375884f67e9.png"/>
    </p>
    <h4>
     <a id="1MATLAB_5">
     </a>
     <strong>
      方法1：使用MATLAB脚本编程（基于控制系统工具箱）
     </strong>
    </h4>
    <h5>
     <a id="1_7">
     </a>
     <strong>
      步骤1：定义被控对象的数学模型
     </strong>
    </h5>
    <p>
     假设被控对象是一个一阶惯性环节，传递函数为：
     <br/>
     [ G(s) = \frac{1}{s+1} ]
    </p>
    <pre><code class="prism language-matlab">sys <span class="token operator">=</span> <span class="token function">tf</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token punctuation">[</span><span class="token number">1</span> <span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 创建传递函数对象</span>
</code></pre>
    <h5>
     <a id="2PID_14">
     </a>
     <strong>
      步骤2：设计PID控制器
     </strong>
    </h5>
    <p>
     使用
     <code>
      pid
     </code>
     函数创建PID控制器，初始参数设置为 ( K_p = 1 ), ( K_i = 0.5 ), ( K_d = 0.1 )：
    </p>
    <pre><code class="prism language-matlab">Kp <span class="token operator">=</span> <span class="token number">1</span><span class="token punctuation">;</span>
Ki <span class="token operator">=</span> <span class="token number">0.5</span><span class="token punctuation">;</span>
Kd <span class="token operator">=</span> <span class="token number">0.1</span><span class="token punctuation">;</span>
C <span class="token operator">=</span> <span class="token function">pid</span><span class="token punctuation">(</span>Kp<span class="token punctuation">,</span> Ki<span class="token punctuation">,</span> Kd<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 生成PID控制器传递函数</span>
</code></pre>
    <h5>
     <a id="3_23">
     </a>
     <strong>
      步骤3：构建闭环系统
     </strong>
    </h5>
    <p>
     将控制器与被控对象串联，形成闭环系统：
    </p>
    <pre><code class="prism language-matlab">closed_loop_sys <span class="token operator">=</span> <span class="token function">feedback</span><span class="token punctuation">(</span>C <span class="token operator">*</span> sys<span class="token punctuation">,</span> <span class="token number">1</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 反馈连接</span>
</code></pre>
    <h5>
     <a id="4_29">
     </a>
     <strong>
      步骤4：仿真并绘制响应曲线
     </strong>
    </h5>
    <p>
     模拟阶跃响应（设定值从0跳变到1）：
    </p>
    <pre><code class="prism language-matlab">t <span class="token operator">=</span> <span class="token number">0</span><span class="token operator">:</span><span class="token number">0.01</span><span class="token operator">:</span><span class="token number">10</span><span class="token punctuation">;</span>        <span class="token comment">% 时间范围0~10秒，步长0.01秒</span>
<span class="token punctuation">[</span>y<span class="token punctuation">,</span> t<span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token function">step</span><span class="token punctuation">(</span>closed_loop_sys<span class="token punctuation">,</span> t<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 计算阶跃响应</span>
<span class="token function">plot</span><span class="token punctuation">(</span>t<span class="token punctuation">,</span> y<span class="token punctuation">)</span><span class="token punctuation">;</span>
grid on<span class="token punctuation">;</span>
<span class="token function">xlabel</span><span class="token punctuation">(</span><span class="token string">'Time (s)'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">ylabel</span><span class="token punctuation">(</span><span class="token string">'Output'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">title</span><span class="token punctuation">(</span><span class="token string">'PID Control Step Response'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="5_41">
     </a>
     <strong>
      步骤5：调整参数优化性能
     </strong>
    </h5>
    <p>
     手动调整 ( K_p ), ( K_i ), ( K_d )，或使用自动整定工具（见下文进阶方法）。
    </p>
    <hr/>
    <h4>
     <a id="2Simulink_46">
     </a>
     <strong>
      方法2：使用Simulink图形化建模
     </strong>
    </h4>
    <h5>
     <a id="1Simulink_48">
     </a>
     <strong>
      步骤1：新建Simulink模型
     </strong>
    </h5>
    <ul>
     <li>
      在MATLAB命令窗口输入
      <code>
       simulink
      </code>
      ，选择“Blank Model”。
     </li>
     <li>
      从库浏览器（Library Browser）拖拽以下模块：
      <ul>
       <li>
        <strong>
         PID Controller
        </strong>
        （控制系统工具箱）
       </li>
       <li>
        <strong>
         Transfer Fcn
        </strong>
        （被控对象模型）
       </li>
       <li>
        <strong>
         Step
        </strong>
        （阶跃输入信号）
       </li>
       <li>
        <strong>
         Scope
        </strong>
        （显示输出结果）
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="2_56">
     </a>
     <strong>
      步骤2：配置模块参数
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       PID Controller
      </strong>
      ：双击设置 ( K_p ), ( K_i ), ( K_d ) 初始值。
     </li>
     <li>
      <strong>
       Transfer Fcn
      </strong>
      ：设置分子（Numerator）为
      <code>
       [1]
      </code>
      ，分母（Denominator）为
      <code>
       [1 1]
      </code>
      。
     </li>
    </ul>
    <h5>
     <a id="3_60">
     </a>
     <strong>
      步骤3：连接模块
     </strong>
    </h5>
    <p>
     将模块按闭环控制结构连接：
    </p>
    <pre><code>Step → PID Controller → Transfer Fcn → Scope
              ↑                        |
              |________________________|
</code></pre>
    <h5>
     <a id="4_68">
     </a>
     <strong>
      步骤4：运行仿真
     </strong>
    </h5>
    <ul>
     <li>
      点击“Run”按钮，双击Scope查看输出波形。
     </li>
     <li>
      调整PID参数并重新仿真，观察响应变化。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="PID_74">
     </a>
     <strong>
      进阶技巧：自动整定PID参数
     </strong>
    </h4>
    <h5>
     <a id="1_pidtune_76">
     </a>
     <strong>
      1. 使用
      <code>
       pidtune
      </code>
      函数（代码方式）
     </strong>
    </h5>
    <pre><code class="prism language-matlab"><span class="token comment">% 定义被控对象</span>
sys <span class="token operator">=</span> <span class="token function">tf</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token punctuation">[</span><span class="token number">1</span> <span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token comment">% 自动整定PID参数（目标相位裕度默认60°）</span>
<span class="token punctuation">[</span>C<span class="token punctuation">,</span> info<span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token function">pidtune</span><span class="token punctuation">(</span>sys<span class="token punctuation">,</span> <span class="token string">'pid'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 可指定'p'、'pi'、'pid'</span>

<span class="token comment">% 显示整定结果</span>
<span class="token function">disp</span><span class="token punctuation">(</span><span class="token string">'整定后的PID参数：'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">disp</span><span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token string">'Kp = '</span><span class="token punctuation">,</span> <span class="token function">num2str</span><span class="token punctuation">(</span>info<span class="token punctuation">.</span>Kp<span class="token punctuation">)</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">disp</span><span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token string">'Ki = '</span><span class="token punctuation">,</span> <span class="token function">num2str</span><span class="token punctuation">(</span>info<span class="token punctuation">.</span>Ki<span class="token punctuation">)</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">disp</span><span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token string">'Kd = '</span><span class="token punctuation">,</span> <span class="token function">num2str</span><span class="token punctuation">(</span>info<span class="token punctuation">.</span>Kd<span class="token punctuation">)</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

<span class="token comment">% 仿真并对比原始参数</span>
closed_loop_auto <span class="token operator">=</span> <span class="token function">feedback</span><span class="token punctuation">(</span>C <span class="token operator">*</span> sys<span class="token punctuation">,</span> <span class="token number">1</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">step</span><span class="token punctuation">(</span>closed_loop_auto<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">% 绘制自动整定后的阶跃响应</span>
</code></pre>
    <h5>
     <a id="2_Simulink_PID_Tuner_95">
     </a>
     <strong>
      2. 使用Simulink PID Tuner工具
     </strong>
    </h5>
    <ol>
     <li>
      在Simulink模型中双击PID Controller模块。
     </li>
     <li>
      点击“Tune”按钮启动PID Tuner。
     </li>
     <li>
      根据响应曲线调整性能目标（如响应时间、鲁棒性）。
     </li>
     <li>
      点击“Update Block”应用新参数。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="PID_103">
     </a>
     <strong>
      示例代码：完整PID仿真脚本
     </strong>
    </h4>
    <pre><code class="prism language-matlab"><span class="token comment">% 定义被控对象</span>

    <hr/>
    <h4>
     <a id="_130">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       模型准确性
      </strong>
      ：被控对象的传递函数需尽可能接近真实系统，否则仿真结果可能误导实际调试。
     </li>
     <li>
      <strong>
       抗饱和处理
      </strong>
      ：实际应用中需添加积分抗饱和逻辑（Simulink中可使用PID Controller的Anti-Windup选项）。
     </li>
     <li>
      <strong>
       噪声影响
      </strong>
      ：若需模拟传感器噪声，可在闭环中加入
      <code>
       Band-Limited White Noise
      </code>
      模块。
     </li>
    </ol>
    <p>
     通过以上方法，你可以快速在MATLAB中验证PID控制效果，并通过参数调整优化系统性能。
    </p>
