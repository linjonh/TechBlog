    <p>
     CUDA数学库
     <br/>
     高性能数学例程
     <br/>
     CUDA数学库是经过行业验证的，高度准确的标准数学函数的集合。只需在源代码中添加“ #include math.h”，即可用于任何CUDA C或CUDA C ++应用程序，CUDA Math库可确保应用程序受益于针对每种NVIDIA GPU架构进行了优化的高性能数学例程。
     <br/>
     数字密集型GPU加速的应用程序和库，包括NVIDIA提供的所有CUDA库，都依靠CUDA Math库来提供突破性的结果。
     <br/>
     主要特点
     <br/>
     • 完全支持所有C99标准浮点和双数学功能
     <br/>
     • IEEE-754对浮点，双精度和所有舍入模式均准确
     <br/>
     • 扩展三角函数和指数函数
     <br/>
     o cospi，sincos，sinpi，exp10
     <br/>
     • 附加的反误差函数
     <br/>
     o erfinv，erfcinv
     <br/>
     • 优化的对等函数
     <br/>
     o rsqrt，rcbrt
     <br/>
     • 浮点数据属性
     <br/>
     o signbit，isfinite，isinf，isnan
     <br/>
     • 贝塞尔函数
     <br/>
     o j0，j1，jn，y0，y1，yn
     <br/>
     • 统计数据
     <br/>
     o normcdf，normcdfinv
