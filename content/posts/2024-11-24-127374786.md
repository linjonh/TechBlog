# C语言的含义及基本框架

1 C语言的含义

C语言是一门通用计算机编程语言，广泛应用于底层开发。C语言的设计目标是提供一种能以简易的方式编译、处理低级存储器、产生少量的机器码以及不需要任何运行环境支持便能运行的编程语言。(基本为数据处理及计算机语言基础)，

1 输出

printf表示为print function表示打印功能，printf是表示输出的基本，printf(＂hello world\n＂)；其中＂＂中的内容被叫做“字符串”，printf会将括号中的内容打印出来，\n所表示的是所输出的结果换一行，

2#inculde<stdio.h>

include是一个计算机专业术语，一指C/C++中包含头文件命令，用于将指定头文件嵌入源文件中。二指include 指令，在JSP中包含一个静态的文件，同时解析这个文件中的JSP语句。三指PHP语句

3 int main

1 int main()是C语言main函数的一种声明方式；

2 int表示函数的返回值类型，表示该主函数的返回值是一个int类型的值；

4大括号

**{}**
*函数名后 { 和 } 之间的部分称之为函数体。*
  
这一左一右两个花括号，左边的开始花括号{标志着函数的开始，而右边的结束花括号}则标志着函数的结束。

5return 0
