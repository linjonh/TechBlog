    <h2>
     <a id="VM_1">
     </a>
     添加VM选项
    </h2>
    <p>
     首先命令行输入chcp查看系统默认编码格式
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7ad0480c72ad68c8ea492bcd57bef2fe.png">
      <br/>
      那么我电脑的默认编码格式就是cp936
      <br/>
      运行配置中加上VM选项
      <code>
       -Dfile.encoding="cp936"
      </code>
      <br/>
      就可以解决问题
      <br/>
      cp936也可以换成GBK（cp936其实就是GBK）
     </img>
    </p>
    <h2>
     <a id="_8">
     </a>
     转码
    </h2>
    <p>
     查看系统默认编码格式和上面一样
     <br/>
     接下来把抓到的数据编码格式转换成cp936
    </p>
    <pre><code class="prism language-java"><span class="token class-name">String</span> html <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">String</span><span class="token punctuation">(</span>driver<span class="token punctuation">.</span><span class="token function">getPageSource</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">.</span><span class="token function">getBytes</span><span class="token punctuation">(</span><span class="token class-name">StandardCharsets</span><span class="token punctuation">.</span>UTF_8<span class="token punctuation">)</span><span class="token punctuation">,</span> <span class="token string">"cp936"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
