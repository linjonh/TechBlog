    <h4>
     <a id="_0">
     </a>
     问题描述
    </h4>
    <p>
     配置mongodb-org.8.0.repo后执行
     <code>
      yum install mongodb-org
     </code>
     后报错信息如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/174f7772cc2d4b32a28a31dfbec3128f.png"/>
    </p>
    <h4>
     <a id="_4">
     </a>
     解决办法
    </h4>
    <p>
     在
     <code>
      /etc/hosts
     </code>
     文件中添加
     <code>
      54.192.143.33 repo.mongodb.org
     </code>
     行，指定下载地址的host解析ip。保存后再执行，问题得以解决。
    </p>
    <h4>
     <a id="_7">
     </a>
     备注
    </h4>
    <p>
     怎么获取域名所对应的ip地址：
     <br/>
     执行命令
     <code>
      54.192.143.33
     </code>
     或者命令
     <code>
      dig repo.mongodb.org
     </code>
     。
    </p>
