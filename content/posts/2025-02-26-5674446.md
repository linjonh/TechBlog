# 工作在微软

前一段时间应邀写一篇关于

Windows

内核研究方面的文章，发表到

MSRA

在

sina

的

blog

上

(
<http://blog.sina.com.cn/s/blog_4caedc7a0100k8jt.html>


)

。正好借这篇文章的名义介绍了

Windows

内核原理一书。实际上，我最初的版本中引用到了下面一幅图，体现了微软在各个领域中与其他软件厂商或组织的竞争关系。

![](http://hi.csdn.net/attachment/201006/16/0_1276707064ca5s.gif)

这幅图来自
<http://mshiltonj.com/software_wars>


，可惜

2006

年以后没有再更新，因而

2006

年以后发布的新软件（像

Windows Azure

）未能列于图上（但

Vista

和

Office 2007

已经呼之欲出了，所以也列上了）。这也正是一位研究院领导建议从文章中撤出这幅图的原因之一。

我个人比较喜欢这幅图，它直观地展示了微软在多个领域中面临的竞争和挑战。尽管我自己所涉及的技术领域只是很窄的一部分，也只接触到很少的产品部门，但是，在工作中仍然能耳濡目染微软在各个领域中的先进技术，譬如，在每年的微软技术节

(TechFest)

大会上，能看到在各个重要领域中最先进的一些研究成果，同时也有机会接待来自不同产品部门的同事。

在微软工作，总有机会发现自己的兴趣点，如果恰好从事与兴趣相符的软件项目，那又何尝不是一种幸福。
