    <h3>
     <a id="1__SSH__0">
     </a>
     1. 打开 SSH 配置文件
    </h3>
    <p>
     按 Ctrl+Shift+P（或 F1）打开命令面板。
     <br/>
     输入并选择 Remote-SSH: Open Configuration File。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/31c22023d28d4050b19877233dbfe920.png">
      <br/>
      选择对应的文件，比如 ~/.ssh/config。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/489e1f1c4dc64751857c5ce70b4b5c74.png"/>
     </img>
    </p>
    <h3>
     <a id="2__8">
     </a>
     2. 查找并删除对应主机条目
    </h3>
    <p>
     删除整个对应条目（从 Host 行开始到该条目的最后一行），格式如下所示：
    </p>
    <pre><code class="prism language-bash">Host myserver
    HostName <span class="token number">192.168</span>.1.100
    User root
    Port <span class="token number">22</span>
</code></pre>
    <h3>
     <a id="3_CtrlS_16">
     </a>
     3. 保存文件（快捷键：Ctrl+S）
    </h3>
