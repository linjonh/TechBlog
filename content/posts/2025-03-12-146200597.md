    <h2>
     <a id="_0">
     </a>
     一、概述
    </h2>
    <p>
     <strong>
      FileZilla
     </strong>
     服务器是一个免费的开源FTP和FTPS服务器，是根据GNU通用公共许可证条款免费发布的开源软件。FileZilla支持FTP、FTPS、SFTP等文件传输协议，相比其他FTP服务器，最大的优势是FileZilla自由(免费)。
     <br/>
     FileZilla的官网地址是：
     <strong>
      https://filezilla-project.org/
     </strong>
    </p>
    <h2>
     <a id="_4">
     </a>
     二、安装服务器软件
    </h2>
    <h4>
     <a id="21__5">
     </a>
     2.1 下载
    </h4>
    <p>
     登录 FileZilla 官网，点击 Download FileZilla Server 按钮进入下载界面，点击下载按钮进行服务器端软件下载。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c6e9d770dd984756b83f60913c7f99bb.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f489221186d6452f9161fedebcd07208.png"/>
    </p>
    <h4>
     <a id="22__12">
     </a>
     2.2 安装
    </h4>
    <p>
     除了下列几个地方需要改动，其他步骤都可以按默认选项：
    </p>
    <ul>
     <li>
      <strong>
       Choose Install Location
      </strong>
      ：安装路径，有需要可以更改。
     </li>
     <li>
      <strong>
       Server Settings
      </strong>
      ：服务端口号和管理密码配置。
     </li>
     <li>
      <strong>
       Admin Settings
      </strong>
      ：服务启动设置，选择
      <strong>
       Start manually
      </strong>
      选项。
     </li>
    </ul>
    <h2>
     <a id="FileZilla__18">
     </a>
     三、FileZilla 服务器的使用
    </h2>
    <h4>
     <a id="31__19">
     </a>
     3.1 连接服务器
    </h4>
    <p>
     FileZilla 服务器安装运行起来后，点击界面
     <strong>
      Connect to server…
     </strong>
     按钮连接服务器
     <br/>
     Host：填
     <strong>
      127.0.0.1
     </strong>
     表示本机IP
     <br/>
     Port：填的是安装时Server settings的端口
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/da35c33c70aa45e99387de950d5b8236.png"/>
    </p>
    <p>
     出现下面信息提示连接成功
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fd389b1d95f14cc4a72fe9d51bb2839d.png"/>
    </p>
    <h4>
     <a id="32__30">
     </a>
     3.2 配置服务器
    </h4>
    <p>
     在软件菜单栏中依次点击：
     <strong>
      Server -&gt; Configure…
     </strong>
     进入设置页面，选择
     <strong>
      Server listeners
     </strong>
     选项卡，将其中的
     <strong>
      Protocol
     </strong>
     更改掉，否则会导致连接不到 FTP 服务器。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/589ed5ef33a141eaa2ca859ccd920aa7.png"/>
    </p>
    <p>
     添加一个用户
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5a71dce6a0f946d4bd5de98b43d82287.png"/>
    </p>
    <h2>
     <a id="_Window__FTP__40">
     </a>
     四、 Window 中访问 FTP 服务器
    </h2>
    <p>
     先打开 cmd 命令窗口查看本机的 ip 地址，再随便打开一个文件夹，输入
     <strong>
      ftp://
     </strong>
     加上服务器IP地址，如：“ftp://192.168.1.105” ，回车后会弹出登录身份窗口，输入之前配置时添加的用户名和密码进行登录。
    </p>

    <h2>
     <a id="_FileZilla__52">
     </a>
     五、安装 FileZilla 客户端软件
    </h2>
    <p>
     打开官网：
     <strong>
      https://filezilla-project.org/
     </strong>
     <br/>
     点击按钮 “
     <strong>
      Download FileZilla Client
     </strong>
     ” 进行下载安装软件，下载后按默认步骤进行安装即可。
