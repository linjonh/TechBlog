    <h1 class="title-article" id="articleContentId">
     MS-DOS 6.22 下建立 FTP 服务器
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     在 MS-DOS 6.22 下建立 FTP 服务器需要使用第三方软件。以下是一个基本的步骤指南：
    </p>
    <h4>
     1. 下载并安装 mTCP 套件
    </h4>
    <p>
     mTCP 是一个适用于 MS-DOS 的网络工具包，其中包含一个 FTP 服务器程序。你可以从
     <a href="https://www.brutman.com/mTCP/mTCP_FTPSrv.html" rel="nofollow" title="mTCP 官方网站">
      mTCP 官方网站
     </a>
     下载该工具包。
    </p>
    <h4>
     2. 配置网络
    </h4>
    <p>
     确保你的 MS-DOS 系统已经配置了网络连接。你需要安装并配置适用于 MS-DOS 的网络驱动程序，如
     <strong>
      Microsoft Network Client 3.0 for MS-DOS
     </strong>
    </p>
    <h4>
     3. 配置 FTP 服务器
    </h4>
    <p>
     解压下载的 mTCP 套件，并找到
     <code>
      FTPSRV.EXE
     </code>
     文件。你需要创建一个配置文件（如
     <code>
      FTPSRV.CFG
     </code>
     ），其中包含以下内容：
    </p>
    <pre><code># FTP Server Configuration
# Example configuration file for mTCP FTP Server

# Network configuration
IP=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1

# FTP server settings
PORT=21
ROOT=C:\FTP
USER=yourusername
PASS=yourpassword
</code></pre>
    <p>
     将
     <code>
      IP
     </code>
     、
     <code>
      NETMASK
     </code>
     和
     <code>
      GATEWAY
     </code>
     替换为你的网络设置，将
     <code>
      ROOT
     </code>
     设置为你想要共享的目录。
    </p>
    <p>
     如果需要允许匿名访问，配置文件：
    </p>
    <pre><code># FTP Server Configuration
# Example configuration file for mTCP FTP Server

# Network configuration
IP=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1

# FTP server settings
PORT=21
ROOT=C:\FTP
ANONYMOUS=YES
</code></pre>
    <p>
     确保
     <code>
      ANONYMOUS=YES
     </code>
     这一行存在
    </p>
    <h4>
     4. 启动 FTP 服务器
    </h4>
    <p>
     在命令行中运行以下命令启动 FTP 服务器：
    </p>
    <pre><code>FTPSRV.EXE -c FTPSRV.CFG
</code></pre>
    <h4>
     5. 测试连接
    </h4>
    <p>
     使用 FTP 客户端（如 Windows 自带的 FTP 命令行工具）连接到你的 MS-DOS FTP 服务器，测试文件上传和下载功能。
    </p>
    <pre><code>ftp 192.168.1.100
</code></pre>
    <p>
     输入配置文件中设置的用户名和密码，进行文件传输操作。
    </p>
    <h4>
     添加开机启动 FTP 服务器的命令
    </h4>
    <p>
     在
     <code>
      AUTOEXEC.BAT
     </code>
     文件的末尾添加启动 FTP 服务器的命令。例如，如果你使用的是 mTCP 套件中的
     <code>
      FTPSRV.EXE
     </code>
     ，并且配置文件为
     <code>
      FTPSRV.CFG
     </code>
     ，你可以添加以下行：
    </p>
    <pre><code>C:\mTCP\FTPSRV.EXE -c C:\mTCP\FTPSRV.CFG
</code></pre>
    <p>
     确保路径和文件名与实际情况相符。
    </p>
    <p>
