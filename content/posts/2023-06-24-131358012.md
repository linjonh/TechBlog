### 二、安装visual studio

#### （一）更改安装路径

首先，我们要设置一下安装路径。因为visual studio占用的空间十分大，我们千万千万不要把它一股脑地安装在C盘，这会让我们本来就贫乏的C盘空间，雪上加霜。
  
在这里，切换到安装位置，将下述三项的路径改成D盘或其他非系统盘，这里图片是我已经改完了的。
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/e315d3ca59eb65650632e74dd522637e.png)

#### （二）自定义安装组件

这时我们就可以自定义选择组件进行下载
  
因为我是需要使用C++，所以这里选择了
**使用C++的桌面开发和visual studio扩展开发**
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/d60c68460aea60402c494e6cbc303674.png)
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/c267917784596ca8c93fdd4c0c95cdeb.png)
  
下面会显示下载所占的空间，供参考（我这里因为已经下载过了）
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/dd3eedac8176944b6314c625fdf1e090.png)

大家可以根据具体需求，选择自己需要下载的组件
  
选择好后，在右下角选择安装，就会出现下载界面，等下载完毕后如图所示，点击启动就可以使用啦~
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/ac0939d675e77fdbd29c54a1a047d89d.png)

> D盘占用6.8G

### （五）万能的HelloWorld

在界面输入hello world代码

```

using namespace std;
int main() {
	cout << "Hello World" << endl;
	return 0;
}


```

点击上方的执行按钮
