    <h2>
     一、引入
    </h2>
    <hr/>
    <p>
     线性结构是数据结构中最为重要的一种数据存储结构之一，同时也是其他数据结构的基础。今天我们就来学学数据结构中的一种基础类型——线性表。
    </p>
    <h2>
     二、线性表的定义和特点
    </h2>
    <hr/>
    <p>
     线性表的定义：用数据元素的有限序列表示。具体情况如下图所示：
    </p>
    <p>
     <img alt="" height="711" src="https://i-blog.csdnimg.cn/direct/a16f8ede211440f3929924497a347ae4.png" width="1708">
      在日常生活中 ，线性表的例子比比皆是，例如，26个英文字母的字母表：
     </img>
    </p>
    <p style="text-align:center">
     <strong>
      （A,B,C,D.......,Z)
     </strong>
    </p>
    <p>
     线性表的几个特点如下：
    </p>
    <p>
     ①只有一个首结点和尾结点；
    </p>
    <p>
     ② 除首尾结点外，其他结点只有一个直接前驱和一个 直接后继。
    </p>
    <p>
     Tips：线性结构反映结点间的逻辑关系是 一对一 的。线性结构包括线性表、堆栈、队列、字符串、数组等。
    </p>
    <h2>
     三、线性表的顺序表示和实现
    </h2>
    <hr/>
    <p>
     线性表的顺序表示又称为顺序存储结构，在这里，我们就要先了解以下两个概念了：
    </p>
    <p>
     <strong>
      顺序存储定义
     </strong>
     ：把逻辑上相邻的数据元素存储在物理上相邻的存储单元中的存储结构。（逻辑上相邻，物理上也相邻）。
    </p>
    <p>
     <strong>
      顺序存储方法
     </strong>
     ：用一组地址连续的存储单元依次存储 线性表的元素，可通过数组V[n]来实现。
    </p>
    <p class="img-center">
     <img alt="" height="500" src="https://i-blog.csdnimg.cn/direct/c242e6baaa994067b09100095ec56a8c.png" width="506"/>
    </p>
    <p>
     顺序表的类型定义 ：
    </p>
    <p>
     基本类型的定义：构建出顺序表的基础架构，分别有最大长度、指向首地址的指针、记录长度的变量等组成元素。
    </p>
    <pre><code class="language-cpp">#define  MAXSIZE 100     //最大长度
typedef  struct {
     ElemType *elem;     //指向数据元素的首地址
     int  length;         //线性表的当前长度
}SqList；


 
</code></pre>
    <p>
     下面用一个图书表的顺序存储结构类型定义来作为例子展示顺序表定义的步骤与效果：
    </p>
    <pre><code class="language-cpp">#define MAXSIZE 10000     //图书表可能达到的最大长度
typedefstruct     //图书信息定义
{ 
    char no[20];     //图书ISBN
    char name[50];     //图书名字
    float price;     //图书价格
}Book;         //数据元素的类型定义


typedefstruct
 { 
    Book*elem;     //存储空间的基地址
    int length;     //图书表中当前图书个数
}SqList;     //图书表的顺序存储结构类型为SqList</code></pre>
    <h2>
     四、线性表的重要基本操作与算法实现
    </h2>
    <hr/>
    <h3>
     4.1、 初始化线性表L
    </h3>
    <hr/>
    <p>
     线性表的初始化和一般数据类型的初始化相似，在构建该数据结构的时候得先进行空间分配、赋值等操作。在这其中我们要先了解这样一个自定义函数类型——Status。
    </p>
    <p>
     Status是用户自定义的一种函数类型，其基础架构和功能与bool类型相似，只是在返回值上做出了些改变。例如：
    </p>
    <p>
     <img alt="" height="87" src="https://i-blog.csdnimg.cn/direct/f664a2d44fb642ddb8481e33bb1b22b8.png" width="1609">
      为了方便直观的表示算法，常用Status类型来封装。例如下面对线性表L的初始化操作：
     </img>
    </p>
    <pre><code class="language-cpp">Status InitList_Sq (SqList &amp;L)
{    //构造一个空的顺序表L
     L.elem=new ElemType[MAXSIZE];   //为顺序表分配空间
    if(!L.elem){
         exit(OVERFLOW);      //存储分配失败
    }
    L.length=0;    //空表长度为0   
     return OK;
}</code></pre>
    <h3>
     4.2、取值（根据位置i获取相应位置数据元素的内容）
    </h3>
    <hr/>
    <p>
     取值操作主要用来获取线性表L中的某个数据元素的内容：
    </p>
    <pre><code class="language-cpp">int GetElem(SqList L, int i, ElemType &amp;e)
{
    if (i&lt;1||i&gt;L.length){
        return ERROR;     //判断i值是否合理，若不合理，返回ERROR
    }
    e=L.elem[i-1];     //第i-1的单元存储着第i个数据
    return OK;
}</code></pre>
    <h3>
     4.3、查找（根据指定数据获取数据所在的位置）
    </h3>
    <hr/>
    <p>
     由于涉及到指针操作，所以下面给出一个示意图来帮助大家理解：
    </p>
    <p class="img-center">
     <img alt="" height="500" src="https://i-blog.csdnimg.cn/direct/d4fa375723c049c1b37f66747e566918.png" width="795"/>
    </p>
    <p>
     如果想要在线性表L中查找值为e的数据元素，用代码该如何表示呢？
    </p>
    <p>
     代码描述：
    </p>
    <pre><code class="language-cpp">int LocateELem(SqListL, ElemTypee)
 {
     for (i=0;i&lt; L.length;i++){
         if (L.elem[i]==e){ 
            return i+1;                
            }
         }
     return 0;
 }</code></pre>
    <h3>
     4.4、插入（在第i个位置上插入）
    </h3>
    <hr/>
    <p>
     算法示意图表示：
    </p>
    <p class="img-center">
     <img alt="" height="500" src="https://i-blog.csdnimg.cn/direct/d799ecf61dd748c892a09c9431055cd0.png" width="839"/>
    </p>
    <p>
     【算法步骤】：
    </p>
    <ol>
     <li>
      判断顺序表的存储空间是否已满，满了就不能插入新元素了。
     </li>
     <li>
      判断插入位置i 是否合法。[1, L.length+1]
     </li>
     <li>
      将第n至第i 位的元素依次向后移动一个位置，空出第i个位置。
     </li>
     <li>
      将要插入的新元素e放入第i个位置。
     </li>
     <li>
      表长加1，插入成功返回OK。
     </li>
    </ol>
    <p>
     若想在线性表L中第i个数据元素之前插入数据元素e ，也就是成 为第i个元素，用代码做如下操作：
    </p>
    <pre><code class="language-cpp">Status ListInsert_Sq(SqList &amp;L, int i , ElemType e)
{
    if(i&lt;1 || i&gt;L.length+1){
       return ERROR;      //i值不合法  
    }         
    if(L.length==MAXSIZE){     
       return ERROR;    //当前存储空间已满
    }
    for(j=L.length-1; j&gt;=i-1; j--){   //L.length-1是最后一个元素， i-1是第i个元素
        L.elem[j+1]=L.elem[j];    //插入位置及之后的元素后移
    }
    L.elem[i-1]=e; //将新元素e放入第i个位置             
    ++L.length;//表长增1
    return OK;
}
       
</code></pre>
    <p>
     Tips：线性表的长度为L.length，j=L.length-1 因为数组的下标从0开始 。
    </p>
    <h3>
     4.5、删除（删除第i个结点）
    </h3>
    <hr/>
    <p>
     算法示意图表述：
    </p>
    <p class="img-center">
     <img alt="" height="370" src="https://i-blog.csdnimg.cn/direct/3b9b1564e16d445b836804b29a871f16.png" width="500"/>
    </p>
    <p>
     【算法步骤】：
    </p>
    <ol>
     <li>
      判断表是不是空（L.length=0 ）。
     </li>
     <li>
      判断删除位置i 是否合法（合法值为1≤i≤n）。
     </li>
     <li>
      将欲删除的元素保留在e中。
     </li>
     <li>
      将第i+1至第n 位的元素依次向前移动一个位置。
     </li>
     <li>
      表长减1，删除成功返回OK。
     </li>
    </ol>
    <p>
     代码描述将线性表L中第i个数据元素删除：
    </p>
    <pre><code class="language-cpp">Status ListDelete_Sq (SqList &amp;L, int i)
{
    if（L.length==0）{
        return ERROR; //表为空
    }
    if((i&lt;1)||(i&gt;L.length)) 
        return ERROR; //i值不合法
    }    
    for (j=i; j&lt;=L.length-1;j++){                   
        L.elem[j-1]=L.elem[j];     //被删除元素之后的元素前移
        --L.length;         //表长减1 
    }     
return OK;
}
 
</code></pre>
    <h3>
     4.6、其他操作：
    </h3>
    <hr/>
    <p>
     清空线性表：
    </p>
    <pre><code class="language-cpp">void ClearList(SqList&amp;L) 
{
 L.length=0;                //将线性表的长度置为0
}
</code></pre>
    <p>
     求线性表L的长度：
    </p>
    <pre><code class="language-cpp">intGetLength(SqListL)
 {
 return (L.length);             
}</code></pre>
    <p>
     求线性表是否满了：
    </p>
    <pre><code class="language-cpp">intGetLength(SqListL)
 {
     if(L.length==MAXSIZE){     
        return 1；
     }
     else{ 
        return 0;
     }
 }</code></pre>
    <p>
     判断线性表L是否为空：
    </p>
    <pre><code class="language-cpp">intIsEmpty(SqListL)
 {
    if (L.length==0){ 
        return 1;    
    }  
    else{ 
        return 0;
    }
 }</code></pre>
