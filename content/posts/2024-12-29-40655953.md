# 服务器硬件配置清单

**服务器配置清单**

1.testServer 服务器

|  |  |
| --- | --- |
| 机架式服务器 | 1u 或者 2u |
| CPU | 8 核 |
| 内存 | 32 G |
| 硬盘 | 2块 300G 硬盘 |
| 网卡 | 百兆网卡 |
| 操作系统 | CentOS6.5 |
| 带宽 | 10 M |

300G 硬盘1 ：操作系统

300G 硬盘2 ：appServer 站点

2.testweb 服务器

|  |  |
| --- | --- |
| 机架式服务器 | 1u 或者 2u |
| CPU | 8 核 |
| 内存 | 32 G |
| 硬盘 | 2块 300G 硬盘 |
| 网卡 | 百兆网卡 |
| 操作系统 | CentOS6.5 |
| 带宽 | 10 M |

300G 硬盘1 ：操作系统

300G 硬盘2 ：testweb 站点

3.数据库、ftp服务器

|  |  |
| --- | --- |
| 机架式服务器 | 1u 或者 2u |
| CPU | 8 核 |
| 内存 | 64G |
| 硬盘 | 2块 300G硬盘   2 块 1 T 硬盘 |
| 网卡 | 百兆网卡 |
| 操作系统 | CentOS6.5 |
| 带宽 | 30 M |

300G 硬盘1 ：操作系统

300G 硬盘2 ：数据库、ftp 应用程序

1T 硬盘1：数据库文件

1T 硬盘2：ftp 图片文件
