    <h3>
     一、系统功能列表
    </h3>
    <p>
     设备管理系统旨在提供全面的设备监控、管理、维护和分析功能。以下是该系统的主要功能模块：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        设备管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          设备录入
         </strong>
         ：支持手动或批量导入设备信息，包括设备名称、型号、规格、生产厂家、采购日期、价格等。
        </p>
       </li>
       <li>
        <p>
         <strong>
          设备分类
         </strong>
         ：根据设备类型进行分类管理，如办公设备、生产设备、IT设备等。
        </p>
       </li>
       <li>
        <p>
         <strong>
          设备档案
         </strong>
         ：为每个设备建立详细的档案，记录设备的基本信息、使用记录、维护记录等。
        </p>
       </li>
       <li>
        <p>
         <strong>
          设备状态管理
         </strong>
         ：实时监控设备的状态，如正常、维修中、报废等。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        告警管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         实时告警监控
        </p>
       </li>
       <li>
        <p>
         历史告警记录查询
        </p>
       </li>
       <li>
        <p>
         告警分类与统计分析
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        版本管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         软件版本控制
        </p>
       </li>
       <li>
        <p>
         软件升级与回滚
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        用户管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         用户账户创建与管理
        </p>
       </li>
       <li>
        <p>
         权限分配与角色管理
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        系统管理
       </strong>
      </p>
      <ul>
       <li>
        <p>
         系统设置与配置
        </p>
       </li>
       <li>
        <p>
         日志记录与审计
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        地图集成
       </strong>
      </p>
      <ul>
       <li>
        <p>
         设备地理位置显示
        </p>

        <p>
         地图操作（缩放、拖动、标记）
        </p>
       </li>

     </li>
     <li>
      <p>

      </p>
      <ul>

    <h3>
     二、技术实现
    </h3>
    <h4>
     前端技术栈
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        VUE.js
       </strong>
       ：一个渐进式JavaScript框架，用于构建用户界面。VUE.js因其易用性、灵活性和高效的数据绑定而受到广泛欢迎。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Vue Router
       </strong>
       ：用于实现前端路由，管理页面之间的跳转和导航。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Vuex
       </strong>
       ：用于全局状态管理，管理设备信息、用户信息等共享状态。
      </p>
     </li>
     <li>
      <p>
       <strong>
        IVIEW
       </strong>
       ：一个基于VUE.js的UI组件库，提供了丰富的组件，如按钮、表单、表格、图表等，帮助快速构建美观且响应式的界面。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Axios
       </strong>
       ：一个基于Promise的HTTP客户端，用于浏览器和node.js，简化了与后端API的通信。
      </p>
     </li>
     <li>
      <p>
       <strong>
        ECharts
       </strong>
       ：一个强大的图表库，用于在系统中展示各种统计图表，如电量统计图、告警统计图等。
      </p>
     </li>
    </ul>
    <h4>
     后端技术栈
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        .NET Core
       </strong>
       ：一个跨平台的高性能框架，用于构建后端服务。.NET Core支持多种编程语言，如C#，提供了丰富的库和工具，简化了开发过程。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Entity Framework Core
       </strong>
       ：一个对象关系映射（ORM）框架，用于简化数据库操作。Entity Framework Core支持多种数据库，如SQL Server、MySQL等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        ASP.NET Core Identity
       </strong>
       ：用于实现用户认证和授权。
      </p>
     </li>
     <li>
      <p>
       <strong>
        JWT（JSON Web Token）
       </strong>
       ：用于用户身份验证和授权，确保系统的安全性。
      </p>
     </li>
     <li>
      <p>
       <strong>
        Swagger
       </strong>
       ：用于生成API文档，方便前端开发人员查看和调试API。
      </p>
     </li>
     <li>
      <p>
       <strong>
        AutoMapper
       </strong>
       ：用于对象之间的映射，简化数据传输对象（DTO）与实体类之间的转换。
      </p>
     </li>
    </ul>
    <h4>
     <strong>
      前后端交互
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        RESTful API
       </strong>
       ：前后端通过RESTful API进行数据交互，后端提供设备信息、用户信息等数据的增删改查接口。
      </p>
     </li>
     <li>
      <p>
       <strong>
        跨域处理
       </strong>
       ：通过CORS（跨域资源共享）解决前后端分离开发中的跨域问题。
      </p>
     </li>
     <li>
      <p>
       <strong>
        JWT身份验证
       </strong>
       ：前端通过JWT进行用户身份验证，每次请求时携带Token，后端验证Token的合法性。
      </p>
     </li>
    </ul>
    <h4>
     数据库
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        MySql
       </strong>
       ：一个关系型数据库管理系统，用于存储系统数据，如设备信息、用户信息、告警记录等。
      </p>
     </li>
    </ul>
    <h4>
     地图服务
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        高德地图API
       </strong>
       或
       <strong>
        百度地图API
       </strong>
       ：用于在系统中集成地图服务，显示设备的地理位置。
      </p>
     </li>
    </ul>
