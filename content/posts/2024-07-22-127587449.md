    <h2>
     <a id="NPMpython_0">
     </a>
     NPM运行项目时切换不同版本的python？
    </h2>
    <h3>
     <a id="_2">
     </a>
     临时使用：
    </h3>
    <pre><code>npm install --python=python3.2
</code></pre>
    <h3>
     <a id="_7">
     </a>
     设置为长期使用：
    </h3>
    <pre><code>npm config set python python3.2
</code></pre>
    <h3>
     <a id="windows_12">
     </a>
     windows系统还可以使用以下方式：
    </h3>
    <pre><code>npm install --python=C:\Python27\python.exe
</code></pre>
    <h3>
     <a id="Linux_17">
     </a>
     Linux系统：
    </h3>
    <pre><code>export PYTHON=python3.2
或者
set PYTHON=python3.2
</code></pre>
