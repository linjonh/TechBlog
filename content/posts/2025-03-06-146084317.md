### 已知快捷键冲突

GitHub Copilot 与 InterSystems 插件存在快捷键冲突。经过本地测试，以下快捷键可以正常使用。

分类| 功能| 德语键盘快捷键| 英语键盘快捷键  
---|---|---|---  
**导航**|  跳转到大纲中的元素(类、方法等)| `Strg + Shift + O`| `Ctrl + Shift + O`  
| 打开文件快速切换面板| `Strg + P`| `Ctrl + P`  
| 跳转到指定行| `Strg + G`| `Ctrl + G`  
| 跳转到匹配的括号| `Strg + Shift + ^^`| `Ctrl + Shift + \`  
| 转到行首/行尾| `Pos1 / Ende`| `Home / End`  
| 转到文件开头/结尾| `Strg + Pos1 / Ende`| `Ctrl + Home / End`  
| 向左/向右切换标签页| `Strg + Bild / Bild`| `Ctrl + PageUp / PageDown`  
| 在资源管理器中显示活动文件| `Strg + K`, `R`| `Ctrl + K`, `R`  
| 显示所有符号| `Strg + T`| `Ctrl + T`  
**通用**|  命令面板(排序 / 重启扩展)| `F1` 或 `Strg + Shift + P`| `F1` 或 `Ctrl + Shift + P`  
| 打开设置| `Strg + ,`| `Ctrl + ,`  
| 全屏切换| `F11`| `F11`  
| 切换侧边栏可见性| `Strg + B`| `Ctrl + B`  
| 新窗口/实例| `Strg + Shift + N`| `Ctrl + Shift + N`  
| 放大/缩小| `Strg + = / -`| `Ctrl + = / -`  
**集成终端**|  打开终端| `Strg + Ö`| `Ctrl + ~`  
| 新建终端| `Strg + Shift + Ö`| `Strg + Shift + ^^ `  
| 切换终端| `Strg + Bild↑ / Bild↓`| `Ctrl + PageUp / PageDown`  
| 打开新命令提示符/终端| `Strg + Shift + C`| `Ctrl + Shift + C`  
**代码编辑**|  上下移动当前行| `Alt + ↑ / ↓`| `Alt + ↑ / ↓`  
| 重命名| `F2`| `F2`  
| 显示代码提示| `Strg + Leertaste`| `Ctrl + Space`  
| 代码操作(快速修复等)| `Strg + .`| `Ctrl + .`  
| 跳转到定义| `Strg + Klick`| `Ctrl + Click`  
| 注释 / 取消注释| `Strg + #`| `Ctrl + /`  
| 多行注释| `Shift + Alt + A`| `Shift + Alt + A`  
| 复制当前行| `Alt + Shift + ↓`| `Alt + Shift + ↓`  
| 删除当前行| `Strg + Shift + K`| `Ctrl + Shift + K`  
| 选中当前行| `Strg + L`| `Ctrl + L`  
| 折叠/展开区域| `Strg + Shift + ß / Strg + →`| `Ctrl + Shift + [ / ]`  
| 插入光标| `Alt + Klick`| `Alt + Click`  
| 往上方/下方选中区域| `Strg + Alt + ↑ / ↓`| `Ctrl + Alt + ↑ / ↓`  
| 折叠所有区域| `Strg + K`, `Strg + 0`| `Ctrl + K`, `Ctrl + 0`  
| 展开所有区域| `Strg + K`, `Strg + J`| `Ctrl + K`, `Ctrl + J`  
| 选择所有匹配项| `Strg + F2`| `Ctrl + F2`  
| 选中下一个匹配| `Strg + D`| `Ctrl + D`  
| 添加行注释| `Strg + K`, `Strg + C`| `Ctrl + K`, `Ctrl + C`  
| 删除行注释| `Strg + K`, `Strg + U`| `Ctrl + K`, `Ctrl + U`  
| 展开选择| `Shift + Alt + →`| `Shift + Alt + →`  
**搜索与替换**|  在文件中搜索(过滤)| `Strg + Shift + F`| `Ctrl + Shift + F`  
| 在文件中替换| `Strg + Shift + H`| `Ctrl + Shift + H`  
**版本控制**|  打开版本控制视图| `Strg + Shift + G`| `Ctrl + Shift + G`  
**调试**|  显示调试| `Strg + Shift + D`| `Ctrl + Shift + D`  
**扩展**|  显示扩展| `Strg + Shift + X`| `Ctrl + Shift + X`  
**Markdown**|  打开 Markdown 预览| `Strg + K`, `V`| `Ctrl + K`, `V`  
| 切换 Markdown 预览| `Strg + Shift + V`| `Ctrl + Shift + V`  
**文件管理**|  关闭所有文件| `Strg + K`, `Strg + W`| `Ctrl + K`, `Ctrl + W`  
| 重新打开关闭的文件| `Strg + Shift + T`| `Ctrl + Shift + T`  
| 复制活动文件的路径| `Strg + K`, `P`| `Ctrl + K`, `P`  
**任务与输出**|  运行任务| `Strg + Shift + B`| `Ctrl + Shift + B`  
| 显示输出面板| `Strg + Shift + U`| `Ctrl + Shift + U`
