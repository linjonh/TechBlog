    <p>
     <code>
      update-alternatives
     </code>
     是一个用于管理符号链接的工具，它允许你在系统中安装多个版本的同一个程序，并方便地在它们之间切换。以下是详细的原理介绍：
    </p>
    <h4>
     <a id="_2">
     </a>
     原理
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        符号链接管理
       </strong>
       ：
       <br/>
       <code>
        update-alternatives
       </code>
       通过创建和管理符号链接来实现不同版本程序的切换。符号链接是指向另一个文件或目录的指针。
      </p>
     </li>
     <li>
      <p>
       <strong>
        优先级机制
       </strong>
       ：
       <br/>
       每个可选项都有一个优先级，优先级数值越高，优先级越高。当你使用
       <code>
        auto mode
       </code>
       时，系统会自动选择优先级最高的版本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        手动模式和自动模式
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         手动模式
        </strong>
        ：用户明确选择某个版本，系统将一直使用该版本，直到用户再次更改。
       </li>
       <li>
        <strong>
         自动模式
        </strong>
        ：系统根据优先级自动选择版本，当有新的版本添加且优先级更高时，系统会自动切换到新版本。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_14">
     </a>
     操作步骤
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        安装新版本
       </strong>
       ：
       <br/>
       使用
       <code>
        update-alternatives --install
       </code>
       命令将新版本添加到管理系统中。例如：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.10 <span class="token number">1</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.8 <span class="token number">2</span>
</code></pre>
      <p>
       这两条命令分别将 Python 3.10 和 Python 3.8 添加到
       <code>
        update-alternatives
       </code>
       系统中，并设置它们的优先级。
      </p>
     </li>
     <li>
      <p>
       <strong>
        配置版本
       </strong>
       ：
       <br/>
       使用
       <code>
        update-alternatives --config
       </code>
       命令来选择你想要的版本。例如：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> python3
</code></pre>
      <p>
       运行该命令后，系统会列出所有可用的版本及其优先级，并提示你选择一个版本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        查看当前配置
       </strong>
       ：
       <br/>
       你可以使用
       <code>
        update-alternatives --display
       </code>
       命令查看当前的配置。例如：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--display</span> python3
</code></pre>
      <p>
       这条命令会显示当前
       <code>
        python3
       </code>
       的所有可选项及其状态。
      </p>
     </li>
    </ol>
    <h4>
     <a id="_38">
     </a>
     示例
    </h4>
    <p>
     假设你已经安装了 Python 3.8 和 Python 3.10，并希望在它们之间切换：
    </p>
    <ol>
     <li>
      <p>
       添加 Python 3.8 和 Python 3.10 到
       <code>
        update-alternatives
       </code>
       系统中：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.10 <span class="token number">1</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.8 <span class="token number">2</span>
</code></pre>
     </li>
     <li>
      <p>
       配置 Python 版本：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> python3
</code></pre>
      <p>
       选择你想要的版本对应的数字，然后按回车键。
      </p>
     </li>
     <li>
      <p>
       查看当前配置：
      </p>
      <pre><code class="prism language-sh"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--display</span> python3
</code></pre>
     </li>
    </ol>
    <p>
     通过这些步骤，你可以方便地在不同版本的 Python 之间切换，而无需手动更改符号链接。
    </p>
