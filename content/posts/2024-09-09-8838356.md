    <p>
     <span style="font-size:14px">
      pip:强大的Python包管理工具(A tool for installing and managing Python packages.)
     </span>
    </p>
    Project Page:
    <a href="https://github.com/pypa/pip" rel="noopener noreferrer" target="_blank">
     https://github.com/pypa/pip
    </a>
    <br/>
    <p>
     <span style="font-size:14px">
      安装包(Install a package):
     </span>
    </p>
    <p>
    </p>
    <pre><code class="language-plain">$ pip install SomePackage==1.0
  [...]
  Successfully installed SomePackage</code></pre>
    <span style="font-size:14px">
     显示已安装的包(Show what files were installed):
    </span>
    <p>
    </p>
    <p>
     <br/>
    </p>
    <pre><code class="language-plain">$ pip show --files SomePackage
  Name: SomePackage
  Version: 1.0
  Location: /my/env/lib/pythonx.x/site-packages
  Files:
   ../somepackage/__init__.py
   [...]</code></pre>
    <br/>
    <p>
     <span style="font-size:14px">
      列出所有过时的包(List what packages are outdated):
     </span>
    </p>
    <p>
     <span style="font-size:14px">
     </span>
    </p>
    <pre><code class="language-plain">$ pip list --outdated
  SomePackage (Current: 1.0 Latest: 2.0)</code></pre>
    <br/>
    升级一个包(Upgrade a package):
    <p>
    </p>
    <p>
    </p>
    <pre><code class="language-plain">$ pip install --upgrade SomePackage
  [...]
  Found existing installation: SomePackage 1.0
  Uninstalling SomePackage:
    Successfully uninstalled SomePackage
  Running setup.py install for SomePackage
  Successfully installed SomePackage</code></pre>
    <br/>
    卸载一个包(Uninstall a package:)
    <p>
    </p>
    <p>
    </p>
    <pre><code class="language-plain">$ pip uninstall SomePackage
  Uninstalling SomePackage:
    /my/env/lib/pythonx.x/site-packages/somepackage
  Proceed (y/n)? y
  Successfully uninstalled SomePackage</code></pre>
    <br/>
    <p>
     上传一个自己做的包:
    </p>
    <pre><code class="language-plain">python setup.py register
#python setup.py bdist_egg
python setup.py sdist upload
python setup.py  bdist_wininst upload</code></pre>
