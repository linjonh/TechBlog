    <h2>
     <a id="_0">
     </a>
     注意
    </h2>
    <p>
     The Excel JavaScript API 没有“Cell”对象或类。 相反，Excel JavaScript API 将所有 Excel 单元格定义为 Range 对象。 Excel UI 中的单个单元格转换为 Excel JavaScript API 中包含一个单元格的 Range 对象。 单个 Range 对象也可以包含多个连续的单元格。 若要了解详细信息，请参阅
     <a href="https://learn.microsoft.com/zh-cn/office/dev/add-ins/excel/excel-add-ins-cells" rel="nofollow">
      使用 Excel JavaScript API 处理单元格
     </a>
     。
    </p>

    <h2>
     <a id="_Excel_JavaScript_API__14">
     </a>
     使用 Excel JavaScript API 设置和获取范围值、文本或公式
    </h2>
    <p>
     <a href="https://learn.microsoft.com/zh-cn/office/dev/add-ins/excel/excel-add-ins-ranges-set-get-values" rel="nofollow">
      https://learn.microsoft.com/zh-cn/office/dev/add-ins/excel/excel-add-ins-ranges-set-get-values
     </a>
    </p>
    <p>
     简单总结下
    </p>
    <p>
     获得某个、某些连续的单元格的range对象
    </p>
    <pre><code class="prism language-javascript"><span class="token keyword">let</span> range <span class="token operator">=</span> sheet<span class="token punctuation">.</span><span class="token function">getRange</span><span class="token punctuation">(</span><span class="token string">"MyRange"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <p>
     读取值
     <br/>
     需要先调用
     <code>
      load(prop)
     </code>
     方法，用于加载对应的内容，所传的
     <code>
      prop
     </code>
     参数有三种：
     <br/>
     // values如果单元格里面是公式，则可能拿不到值；text：取得与表格中展示出来的内容，没公式的是原值，有公式的就是公式得到的值； formulas:没公式的就是原值，有公式的就是公式
    </p>
    <pre><code class="prism language-javascript"><span class="token comment">// 先调用方法，显式加载相应的值</span>

     // 设置值，由于是个区域对象，会包含多行多列的数据，因此需要提供一个二维数组。一层数组中的每个对象表示一行数据，二层数组中的每个数据为该行、该列，即具体单元格的值。

    <pre><code class="prism language-javascript"><span class="token comment">// 设置单值</span>
