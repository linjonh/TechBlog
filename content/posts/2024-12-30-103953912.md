# python\_dockerfile 部署 flask项目

**环境: linux+docker
  
开发工具: pycharm
  
项目： flask项目**

1. pycharm编辑器执行命令，打印所需依赖清单 requirements.txt

```python
pip freeze > requirements.txt

```

2. 编写dockerfile文件

```python
FROM python:3.8.0

COPY . /app
WORKDIR ./app
RUN pip install -r requirements.txt
EXPOSE 8687
ENTRYPOINT ["python"]
CMD ["app.py"]

```

3. 创建一个app文件夹，文件夹结构
     
   ![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/89464187e341108274e0f6b7fea415f3.png)
     
   app.py为flask项目

4.执行Dockerfile 文件打成镜像包：

```python
docker build -t arcgis_proxy:latest.

```

查看镜像

```python
docker images

```

5.运行镜像:

```python
docker run -p 8687:8687 arcgis_proxy:latest

```

查看运行中的容器

```python
docker ps

```
