    <blockquote>
     <p>
      随着无人机在工业检测、仓储物流、应急救援等室内场景的广泛应用，高精度室内定位技术成为关键支撑。超宽带（UWB）技术凭借其高时间分辨率、强抗多径能力等优势，成为室内定位的主流方案。然而，不同的定位方法（如TOA、TDOA、AOA）在复杂室内环境中的性能差异显著，如何优化算法以提升定位精度和鲁棒性成为研究热点。本课题通过构建仿真系统，对比分析典型定位算法的性能特性，并针对TDOA方法的缺陷提出改进方案，为实际工程应用提供理论依据。
     </p>
    </blockquote>
    <p>
    </p>
    <p>
    </p>
    <h2>
     <a id="_12">
     </a>
     研究目标
    </h2>
    <ol>
     <li>
      <strong>
       多算法性能对比
      </strong>
      ：建立TOA、TDOA、AOA定位方法的数学模型，对比其在噪声干扰下的定位误差特性
     </li>
     <li>
      <strong>
       TDOA算法优化
      </strong>
      ：解决传统线性TDOA模型的误差累积问题，提出基于非线性优化的改进方法
     </li>
     <li>
      <strong>
       仿真验证
      </strong>
      ：设计动态无人机轨迹与多基站场景，验证算法改进效果及适用性
     </li>
    </ol>
    <hr/>
    <h2>
     <a id="_19">
     </a>
     研究内容
    </h2>
    <h3>
     <a id="_20">
     </a>
     系统建模与仿真环境构建
    </h3>
    <ul>
     <li>
      <strong>
       基站部署
      </strong>
      ：4基站矩形布局模拟典型室内环境（10m×10m）
     </li>
     <li>
      <strong>
       动态轨迹生成
      </strong>
      ：无人机沿圆形路径运动（半径4m，中心[5,5]）
     </li>
     <li>
      <strong>
       噪声模型
      </strong>
      ：
      <ul>
       <li>
        TOA测量噪声：标准差1ns（对应约0.3m）
       </li>
       <li>
        TDOA测量噪声：标准差0.5ns
       </li>
       <li>
        AOA角度噪声：标准差3度
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <a id="_28">
     </a>
     定位算法实现与改进
    </h3>
    <ul>
     <li>
      <strong>
       TOA定位
      </strong>
      ：基于最小二乘的非线性优化（
      <code>
       fminunc
      </code>
      ）
     </li>
     <li>
      <strong>
       TDOA优化
      </strong>
      ：重构目标函数，最小化时差测量误差平方和
      <pre><code class="prism language-matlab">fun <span class="token operator">=</span> <span class="token operator">@</span><span class="token punctuation">(</span>p<span class="token punctuation">)</span> <span class="token function">sum</span><span class="token punctuation">(</span> <span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token function">vecnorm</span><span class="token punctuation">(</span>p<span class="token operator">-</span>other_stations<span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">)</span> <span class="token operator">-</span> <span class="token function">vecnorm</span><span class="token punctuation">(</span>p<span class="token operator">-</span>ref_station<span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token operator">/</span>c <span class="token operator">-</span> tdoa<span class="token operator">'</span><span class="token punctuation">)</span><span class="token operator">.^</span><span class="token number">2</span> <span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       AOA定位
      </strong>
      ：多基站角度交汇的最小二乘解算
     </li>
    </ul>
    <h3>
     <a id="_36">
     </a>
     性能评估体系
    </h3>
    <ul>
     <li>
      <strong>
       误差指标
      </strong>
      ：欧氏距离误差（
      <code>
       vecnorm
      </code>
      ）
     </li>
     <li>
      <strong>
       可视化分析

       <li>

       </li>
      </ul>
     </li>
    </ul>
