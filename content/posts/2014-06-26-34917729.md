     <span style="font-weight: normal;">
      当前主流搜索引擎在解决互联网技术创意这个垂直领域的搜索上存在如下几个问题：
      <br/>
      1. 实时性问题：收录延迟，信息过时，竞价维度干扰度高；
      <br/>
      2. 语言问题：国外一线资源获取不到；
      <br/>
      3. 交互性问题：搜索结果平面，不具社交属性，缺乏互动；
      <br/>
      4. 数据冗余：不同来源的内容高度重复。
      <br/>
      踏得网致力于解决上述问题，成为一个具有创新形态的（垂直+社交+即时）、领先的技术资源搜索引擎。
      <br/>
      踏得网专注于互联网动画、特效、交互动效和3D视觉领域，实时收录国内外产品创意、技术作品代码演示、数据模型和在线教程，
      <br/>
      并以众包的方式对内容索引进行再加工、编译。
      <br/>
      踏得网同时还是一个简单易用的、免费的分享和推广平台：踏得网为Web技术人员提供云开发测试平台，为互联网产品提供免费推广渠道。
      <br/>
      <br/>
      我们和用户一起"轻量"运营内容索引，让信息获取变简单；
      <br/>
      我们和用户一起分享全球视野的产品创意、设计、技术，让新技术得到更好的普及，让技术成为创新的第一推手。
      <br/>
      我们相信十年世事变迁，信息搜索服务本身会演变出新的形态。
