    <h3>
     <a id="_234">
     </a>
     创建用户
    </h3>
    <h4>
     <a id="_235">
     </a>
     语法
    </h4>
    <p>
     <code>
      create user '用户名'@'登陆主机/ip' identified by '密码';
     </code>
     <br/>
     如果登录主机被设置为
     <code>
      %
     </code>
     ，则表示该用户可以在任何地方登陆user；
     <br/>
     <strong>
      此设置方法需要谨慎使用
     </strong>
    </p>

    </h3>

    </h4>

    </p>

     <strong>
      说明：
     </strong>
    </p>
    <ul>
     <li>
      权限列表，多个权限用逗号分开
     </li>
    </ul>
    <pre><code class="prism language-sql"><span class="token keyword">grant</span> <span class="token keyword">select</span> <span class="token keyword">on</span> <span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span>
