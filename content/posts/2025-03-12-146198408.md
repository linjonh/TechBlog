    <p>
     首先，需要说明的是，本方案于2025年3月12日配置成功，若多年后vscode或若干插件改版，则可能部分步骤不支持哦
    </p>
    <p>
     为了通过VS Code快速连接到远程服务器的Docker容器（假设名为
     <code>
      my_fancy_docker
     </code>
     ），并实现本地修改代码、使用容器内Python环境的需求，请按照以下步骤操作：
    </p>
    <hr/>
    <h4>
     <a id="_RemoteSSH__RemoteContainers__6">
     </a>
     <strong>
      通过 Remote-SSH + Remote-Containers 扩展
     </strong>
    </h4>
    <h5>
     <a id="_1_7">
     </a>
     <strong>
      步骤 1：安装必要扩展
     </strong>
    </h5>
    <ol>
     <li>
      在本地VS Code中安装以下扩展：
      <ul>
       <li>
        <strong>
         Remote - SSH
        </strong>
       </li>
       <li>
        <strong>
         Remote - Containers
        </strong>
       </li>
      </ul>
     </li>
    </ol>
    <h5>
     <a id="_2_SSH__12">
     </a>
     <strong>
      步骤 2：配置 SSH 连接到远程服务器
     </strong>
    </h5>
    <ol>
     <li>
      打开VS Code的命令面板（
      <code>
       Ctrl+Shift+P
      </code>
      ），输入
      <strong>
       <code>
        Remote-SSH: Connect to Host
       </code>
      </strong>
      。
     </li>
     <li>
      选择
      <strong>
       <code>
        Configure SSH Hosts...
       </code>
      </strong>
      ，编辑SSH配置文件（如
      <code>
       ~/.ssh/config
      </code>
      ），添加服务器信息：
      <pre><code class="prism language-plaintext">Host remote-server
  HostName &lt;服务器IP或域名&gt;
  User &lt;用户名&gt;
  IdentityFile ~/.ssh/私钥路径
</code></pre>
     </li>
     <li>
      保存后，选择
      <strong>
       <code>
        Remote-SSH: Connect to Host
       </code>
      </strong>
      &gt;
      <strong>
       <code>
        remote-server
       </code>
      </strong>
      ，输入密码（如需要）连接到服务器。
     </li>
    </ol>
    <h5>
     <a id="_3_Docker__23">
     </a>
     <strong>
      步骤 3：附加到运行中的 Docker 容器
     </strong>
    </h5>
    <ol>
     <li>
      成功连接远程服务器后，在VS Code中打开命令面板（
      <code>
       Ctrl+Shift+P
      </code>
      ）。
     </li>
     <li>
      输入并选择
      <strong>
       <code>
        Remote-Containers: Attach to Running Container...
       </code>
      </strong>
      。
     </li>
     <li>
      从列表中选择容器
      <strong>
       <code>
        my_fancy_docker
       </code>
      </strong>
      ，VS Code将重启窗口并进入容器内部。
     </li>
    </ol>
    <h5>
     <a id="_4_28">
     </a>
     <strong>
      步骤 4：配置开发环境
     </strong>
    </h5>
    <ol>
     <li>
      在容器内打开项目文件夹（如
      <code>
       /app
      </code>
      ）。
     </li>
     <li>
      安装
      <strong>
       Python 扩展
      </strong>
      （首次进入容器时，VS Code会提示安装推荐扩展）。
     </li>
     <li>
      选择Python解释器：
      <ul>
       <li>
        打开命令面板（
        <code>
         Ctrl+Shift+P
        </code>
        ），输入
        <strong>
         <code>
          Python: Select Interpreter
         </code>
        </strong>
        ，选择容器内的Python路径（如
        <code>
         /usr/local/bin/python3
        </code>
        ）。
       </li>
      </ul>
     </li>
    </ol>
