#### 1.布尔类型

布尔类型有 2 个值

* True
* False

```Python
# 演示进行内容的相等比较
num1 = 10
num2 = 10
print(f"10 == 10的结果是：{num1 == num2}")

```

`布尔取反`

在 Python 中，可以使用 not 关键字来对布尔值进行取反操作。not 操作符将 True 变为 False，将 False 变为 True。例如：

```
x = True
y = not x
print(y) # 输出False

z = False
w = not z
print(w) # 输出True

```

在上面的代码中，变量 y 被赋值为 not x，因此它的值为 False。同样，变量 w 被赋值为 not z，因此它的值为 True。

#### 3.成年人判断

```python
# 获取键盘输入
age = int(input("请输入你的年龄："))
# 通过if判断是否是成年人
if age >= 18:
    print("您已成年，游玩需要买票，10元.")
print("祝您游玩愉快")

```

#### 4.if else

```py
# 定义键盘输入获取身高数据
height = int(input("请输入你的身高（cm）："))
# 通过if进行判断
if height > 120:
    print("您的身高超出120CM，需要买票，10元。")
else:
    print("您的身高低于120CM，可以免费游玩。")
print("祝您游玩愉快")

```

#### 5.elif

```py
# 通过if判断，可以使用多条件判断的语法
# 第一个条件就是if
if int(input("请输入你的身高（cm）：")) < 120:
    print("身高小于120cm，可以免费。")
elif int(input("请输入你的VIP等级（1-5）：")) > 3:
    print("vip级别大于3，可以免费。")
elif int(input("请告诉我今天几号：")) == 1:
    print("今天是1号免费日，可以免费")
else:
    print("不好意思，条件都不满足，需要买票10元。")

```
