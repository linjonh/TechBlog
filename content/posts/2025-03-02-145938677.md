    <h4>
     <a id="_30">
     </a>
     定义属性
    </h4>
    <h5>
     <a id="_32">
     </a>
     容器相关属性
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         display
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：指定元素是否以弹性容器显示，虽然默认是弹性布局，但可以提供一个属性让用户选择是否开启。
       </li>
       <li>
        类型：
        <code>
         Boolean
        </code>
       </li>
       <li>
        默认值：
        <code>
         true
        </code>
       </li>
       <li>
        说明：当设置为
        <code>
         false
        </code>
        时，组件不使用弹性布局。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         flexDirection
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义主轴的方向，即子元素的排列方向。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        可选值：
        <code>
         'row'
        </code>
        、
        <code>
         'row-reverse'
        </code>
        、
        <code>
         'column'
        </code>
        、
        <code>
         'column-reverse'
        </code>
       </li>
       <li>
        默认值：
        <code>
         'row'
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         flexWrap
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义子元素是否换行。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        可选值：
        <code>
         'nowrap'
        </code>
        、
        <code>
         'wrap'
        </code>
        、
        <code>
         'wrap-reverse'
        </code>
       </li>
       <li>
        默认值：
        <code>
         'nowrap'
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         flexFlow
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：是
        <code>
         flexDirection
        </code>
        和
        <code>
         flexWrap
        </code>
        的缩写属性。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        默认值：
        <code>
         undefined
        </code>
       </li>
       <li>
        说明：如果设置了
        <code>
         flexFlow
        </code>
        ，则
        <code>
         flexDirection
        </code>
        和
        <code>
         flexWrap
        </code>
        的设置将被忽略。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         justifyContent
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义子元素在主轴上的对齐方式。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        可选值：
        <code>
         'flex-start'
        </code>
        、
        <code>
         'flex-end'
        </code>
        、
        <code>
         'center'
        </code>
        、
        <code>
         'space-between'
        </code>
        、
        <code>
         'space-around'
        </code>
        、
        <code>
         'space-evenly'
        </code>
       </li>
       <li>
        默认值：
        <code>
         'flex-start'
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         alignItems
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义子元素在交叉轴上的对齐方式。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        可选值：
        <code>
         'flex-start'
        </code>
        、
        <code>
         'flex-end'
        </code>
        、
        <code>
         'center'
        </code>
        、
        <code>
         'baseline'
        </code>
        、
        <code>
         'stretch'
        </code>
       </li>
       <li>
        默认值：
        <code>
         'stretch'
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         alignContent
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义多根轴线的对齐方式，如果子元素只有一根轴线，该属性不起作用。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        可选值：
        <code>
         'flex-start'
        </code>
        、
        <code>
         'flex-end'
        </code>
        、
        <code>
         'center'
        </code>
        、
        <code>
         'space-between'
        </code>
        、
        <code>
         'space-around'
        </code>
        、
        <code>
         'stretch'
        </code>
       </li>
       <li>
        默认值：
        <code>
         'stretch'
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <h5>
     <a id="_76">
     </a>
     子元素相关属性
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         gap
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：定义子元素之间的间距。
       </li>
       <li>
        类型：
        <code>
         String
        </code>
       </li>
       <li>
        默认值：
        <code>
         '0'
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         inline
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：指定弹性容器是否为行内元素。
       </li>
       <li>
        类型：
        <code>
         Boolean
        </code>
       </li>
       <li>
        默认值：
        <code>
         false
        </code>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_88">
     </a>
     定义事件
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         resize
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：当弹性容器大小改变时触发，可用于在容器大小变化时进行一些额外的处理，比如重新计算子元素的布局。
       </li>
       <li>
        参数：
        <code>
         { width: number, height: number }
        </code>
        ，表示容器的新宽度和高度。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         childAdded
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：当有新的子元素添加到弹性容器时触发，可用于在子元素添加后进行一些初始化操作。
       </li>
       <li>
        参数：
        <code>
         HTMLElement
        </code>
        ，表示新添加的子元素。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         childRemoved
        </code>
       </strong>
      </p>
      <ul>
       <li>
        作用：当有子元素从弹性容器中移除时触发，可用于在子元素移除后进行一些清理操作。
       </li>
       <li>
        参数：
        <code>
         HTMLElement
        </code>
        ，表示被移除的子元素。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="_103">
     </a>
     其他
    </h4>
    <ol>
     <li>
      <strong>
       提供插槽
      </strong>
      ：使用 Vue 的插槽机制，允许用户将任意内容插入到弹性容器中，增加组件的灵活性。
     </li>
     <li>
      <strong>
       样式隔离
      </strong>
      ：确保组件的样式不会影响到外部的其他元素，同时也不会被外部样式影响，可以使用 CSS 作用域或者 CSS Modules。
     </li>
     <li>
      <strong>
       响应式设计
      </strong>
      ：考虑结合媒体查询或者 Vue 的响应式数据，让弹性布局在不同的屏幕尺寸下有不同的表现。
     </li>
     <li>
      <strong>
       文档和示例
      </strong>
      ：编写详细的文档，说明每个属性和事件的作用，并提供一些使用示例，方便其他开发者使用该组件。
     </li>
    </ol>
