# oracle配置文件在哪里在哪里,在文件中配置oracle数据库服务

我们不需要每次配置Oracle数据库都需要通过Net Configuration Assistant工具去配置，可以通过手动修改一个文件就可以达到配置oracle数据库德效果。

首先找到oracle的安装目录下/NETWORK/ADMIN.

1.添加一个名为listener.ora的文件.listener.ora文件内容

#ORACLE\_HOME 为oracle安装的目录 SID\_LIST\_LISTENER = (SID\_LIST = (SID\_DESC = (SID\_NAME = PLSExtProc) (ORACLE\_HOME = C:/oracle\_client) (PROGRAM = extproc) ) ) #HOST为一台已经起了ORACLE监听机器的IP地址 LISTENER = (DESCRIPTION\_LIST = (DESCRIPTION = (ADDRESS\_LIST = (ADDRESS = (PROTOCOL = TCP)(HOST = 127.0.0.1)(PORT = 1521)) ) (ADDRESS\_LIST = (ADDRESS = (PROTOCOL = IPC)(KEY = EXTPROC)) ) ) )

2.添加一个sqlnet.ora文件，sqlnet.ora文件内容

SQLNET.AUTHENTICATION\_SERVICES= (NTS) NAMES.DIRECTORY\_PATH= (TNSNAMES)

3.添加一个tnsnames.ora文件，tnsnames.ora文件内容

#METARDB是SQLplus显示的数据库 #HOST为安装数据库的服务器IP #SERVICE\_NAME为oracle数据库服务名 METARDB = (DESCRIPTION = (ADDRESS\_LIST = (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.121.40)(PORT = 1521)) ) (CONNECT\_DATA = (SERVICE\_NAME = metardb) ) ) jx = (DESCRIPTION = (ADDRESS\_LIST = (ADDRESS = (PROTOCOL = TCP)(HOST = 134.235.44.36)(PORT = 1521)) ) (CONNECT\_DATA = (SERVICE\_NAME = jxdcnmstc) ) )

这样一个完整的数据库配置就完了。
