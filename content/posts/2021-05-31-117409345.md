    <h2>
     <a id="Java_5">
     </a>
     什么是Java？
    </h2>
    <p>
     Java是一种OOP（面向对象的编程语言）、基于类的、具有VM（虚拟机）平台的多平台编程语言。OOP 是一种基于包含代码和数据的对象概念的编程范式。虚拟机可帮助您创建可在任何平台、任何地方灵活运行的编译程序。Java 将此概念称为
     <strong>
      一次编写，随处运行。
     </strong>
    </p>
    <h3>
     <a id="_Java__8">
     </a>
     如何开始用 Java 编写代码
    </h3>
    <p>
     您需要遵循两个简单的设置步骤：
    </p>
    <p>
     1.下载并设置 Java SE 开发工具包
     <br/>
     2.选择您喜欢的任何编辑器
    </p>
    <p>
     作为一个初学者我建议你选择记事本，因为它是最基本的。其他的编辑器或 IDE 也可以工作，只是尽量不要太花哨，因为您现在想专注于基本要素。
    </p>
    <h3>
     <a id="Java_16">
     </a>
     用Java编写一个简单的代码
    </h3>
    <p>
     让我们看看如何用 Java 简单地打印一个
     <code>
      hello world
     </code>
     程序。
    </p>
    <pre><code class="prism language-java"><span class="token comment">// 一个用于打印简单字符串（字符输入类型，例如 HelloWorld）的 Java 程序，</span>

<span class="token keyword">class</span> <span class="token class-name">HelloWorld</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Hello, World!"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_29">
     </a>
     代码说明
    </h3>
    <ul>
     <li>
      您看到的第一行以描述我们程序的功能的通用注释语句开头。您可以看到它以
      <code>
       //XXX
      </code>
      开头，它是单行注释（仅限于单行的注释）的符号表示。
     </li>
     <li>
      在我们进入下一行之前，让我们探索大括号。大括号
      <code>
       {
       <!-- -->
      </code>
      是指相应类或函数的开头，而大括号
      <code>
       }
      </code>
      是指它结束的地方。
     </li>
     <li>
      <code>
       class
      </code>
      指的是一个对象构造函数，或者简单地说，是我们从中创建对象的蓝图。每个类都有一个名称，在我们这种情况下，我们的类名称是
      <code>
       HelloWorld
      </code>
      。
     </li>
     <li>
      现在进入我们程序的主要方法，即
     </li>
    </ul>
    <pre><code class="prism language-java">  <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span><span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span>
</code></pre>
    <p>
     现在让我们把它分解成几块：
    </p>
    <p>
     1.
     <code>
      public
     </code>
     指任何人都可以从任何地方访问此方法，这意味着它在全局范围内可用。
     <br/>
     2.
     <code>
      static
     </code>
     是一个关键字，它简单地定义此方法是静态的且无法更改。
     <br/>
     3.
     <code>
      void
     </code>
     也是一个关键字，指的是不返回任何内容的方法。
     <br/>
     4.
     <code>
      main
     </code>
     指的是 Java main 方法的名称。
     <br/>
     5.
     <code>
      String args[]
     </code>
     指的是传递给 main 方法的参数。
     <code>
      args
     </code>
     指的是字符串的名称。您可以
     <code>
      args
     </code>
     使用任何其他字符串名称命名，这只是用于它的标准命名。
     <br/>
     6.
     <code>
      System.out.println("Hello, World!");
     </code>
     这是我们
     <code>
      Hello, World!
     </code>
     打印的实际输出行。单个语句和变量声明以分号结尾。
    </p>
    <p>
     对于只在屏幕上打印单个字符串的程序来说，这听起来很复杂吗？确实如此，这也是为什么学习 JavaScript 是初学者开发者更好的选择的原因之一。
    </p>
    <h2>
     <a id="_JavaScript_51">
     </a>
     什么是 JavaScript？
    </h2>
    <p>
     JavaScript 是一种轻量级的高级脚本编程语言，通常用于使网页具有动态性和交互性。它可以将动态文本添加到 HTML 中，并以浏览器语言而闻名。我们将在与 Java 的比较中讨论它的详细特性。
    </p>
    <h3>
     <a id="_JavaScript__54">
     </a>
     如何开始使用 JavaScript 编写代码
    </h3>
    <p>
     您只需要具备以下 2 件事即可开始使用 JavaScript：
    </p>
    <p>
     1.浏览器（谷歌浏览器、Safari、Firefox、Internet Explorer等等）
     <br/>
     2.编辑器（记事本、VS Code、Atom等）
    </p>
    <h3>
     <a id="JavaScript__59">
     </a>
     如何编写你的第一个JavaScript 程序
    </h3>
    <p>
     让我们看看如何用 JavaScript 编写第一个
     <code>
      hello world
     </code>
     程序。
    </p>
    <pre><code class="prism language-javascript"><span class="token comment">// 一个用于打印简单字符串（字符输入类型，例如 HelloWorld）的 JavaScript 程序，</span>

console<span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">'Hello World'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <h3>
     <a id="_68">
     </a>
     代码说明
    </h3>
    <ul>
     <li>
      第一行也是同一个单行注释，就像我们前面在 Java 程序中看到的那样。
     </li>
     <li>
      在 JavaScript 中，我们简单地使用语句
     </li>
    </ul>
    <pre><code class="prism language-javascript">  console<span class="token punctuation">.</span>log
</code></pre>
    <p>
     在控制台上打印任何内容（一个提供对浏览器调试控制台的访问的对象）。简而言之，您可以在以下情况下查看它：
    </p>
    <p>
     1.你右击鼠标
     <br/>
     2.点击检查
     <br/>
     3.选择控制台选项卡
     <br/>
     4.在控制台查看记录的语句
    </p>
    <p>
     现在告诉我，和Java相比，它要容易多少？
    </p>
    <p>
     你是对的，它们甚至一点都不相似。
    </p>
    <h2>
     <a id="Java__JavaScript__88">
     </a>
     Java 和 JavaScript 之间的主要区别
    </h2>
    <p>
     我们可以注意到两种语言之间存在以下主要差异：
    </p>
    <ul>
     <li>
      <strong>
       编程范式
      </strong>
      ： Java 严格遵循面向对象编程范式，而 JavaScript 遵循多范式，包括面向对象编程、过程化和脚本编程语言。
     </li>
     <li>
      <strong>
       代码执行
      </strong>
      ： Java 应用程序具有在 JVM（Java 的虚拟运行时环境）上运行的灵活性，而 JavaScript仅在特定于浏览器的特定于应用程序的环境中运行。
     </li>
     <li>
      <strong>
       对象
      </strong>
      ： Java 对象纯粹是基于类的，而 JavaScript 对象是基于原型的。
     </li>
     <li>
      <strong>
       类型检查
      </strong>
      ： Java 确保在编译前对其变量和函数进行强类型检查，这使得运行或编译时运行的概率非常低。然而，JavaScript是弱类型的，变量的类型在编译之前是未知的，因此，运行或编译时错误的机会增加。
     </li>
     <li>
      <strong>
       文件扩展名
      </strong>
      ： Java 具有.java文件扩展名，而 JavaScript 具有.js文件扩展名。（顺带提一下，我们常说的Js等同于JavaScript ，只不过一个是全称，一个是缩写）
     </li>
     <li>
      <strong>
       多线程
      </strong>
      ： Java 支持多线程（同时执行两个或多个线程以最大限度地利用 CPU），而 JavaScript 不支持。
     </li>
     <li>
      <strong>
       内存使用
      </strong>
      ：由于 Java 有很多内容，所以它占用更多的空间，而 JavaScript 占用的空间更少。
     </li>
     <li>
      <strong>
       语言依赖
      </strong>
      ：两种语言都可以独立工作，也可以与其他语言配对。
     </li>
     <li>
      <strong>
       并发方法
      </strong>
      ： Java 利用其多线程能力并具有基于线程的方法，而 JavaScript 则遵循事件驱动的方法。
     </li>
     <li>
      <strong>
       性能
      </strong>
      ：由于其参与的性质，脚本语言总是比纯编程语言更有效，因此，与 JavaScript 相比，Java 效率较低且速度较慢。
     </li>
    </ul>
    <h2>
     <a id="Java__JavaScript__102">
     </a>
     Java 和 JavaScript 之间的相似之处
    </h2>
    <p>
     刚才探讨了有什么不同对吧？现在让我们也探讨一下这两者有什么共同点。
    </p>
    <ul>
     <li>
      <strong>
       浏览器兼容性
      </strong>
      ：两种语言都可以在浏览器上运行。
     </li>
     <li>
      <strong>
       支持
      </strong>
      ：两种语言都有很多在线支持社区。
     </li>
     <li>
      <strong>
       语法和编程概念
      </strong>
      ：
     </li>
    </ul>
    <p>
     虽然两者都是两种不同的语言，但都共享相同的核心编程概念和一些语法概念，例如：
    </p>
    <p>
     1.使用编程循环，例如 for 循环、while 循环。
     <br/>
     2.使用条件语句，例如 if 和 else if。
     <br/>
     3.使用 Math.pow 等数学库。
     <br/>
     4.常见的共享语法符号，例如代码块定界符{}，以分号结尾的代码语句。
    </p>
    <ul>
     <li>
      <strong>
       相似名称
      </strong>
      ：这个很奇怪，Java 和JavaScript 都有“Java”的共同点，尽管这两种语言完全不同。
     </li>
    </ul>
    <h2>
     <a id="_118">
     </a>
     结论
    </h2>
    <p>
     综上所述，Java和JavaScript是两种不同的不同语言。它们之间的一些相似之处主要来自核心编程原则，但除此之外，它们都是两个不同的世界，它们具有相似的名称，但主要区别在于彼此。
    </p>
