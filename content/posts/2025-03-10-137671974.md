    <h2 id="B%EF%BC%9A%E9%A1%BA%E5%AD%90%E6%97%A5%E6%9C%9F%EF%BC%88%E5%AD%98%E5%9C%A8%E4%BA%89%E8%AE%AE%EF%BC%89">
     2B：年号字符（填空5分_进制）
    </h2>
    <blockquote>
     <p>
      <strong>
       题目描述：
      </strong>
     </p>
     <p>
      <strong>
      </strong>
      小明用字母 A对应数字 1，B对应 2，以此类推，用 Z对应 26。对于 27以上的数字，小明用两位或更长位的字符串来对应，例如 AA对应27，AB对应28，AZ对应52，LQ对应329。
     </p>
     <p>
      请问2019对应的字符串是什么？
     </p>
    </blockquote>
    <p>
     <strong>
      题目分析：
     </strong>
    </p>
    <p>
     <strong>
     </strong>
     这题就是类似于一个进制转换，你可以回想一下十进制转二进制如何转换，然后再想想将十进制转化成26进制，口诀就是除p取余
    </p>
    <p>
     答案：BYQ
    </p>
    <pre><code class="language-cpp">
<code class="language-cpp">//小明用字母 A对应数字 1，B对应 2，以此类推，用 Z对应 26。对于 27以上的数字，
//小明用两位或更长位的字符串来对应，例如 AA对应27，AB对应28，AZ对应52，LQ对应329。
//请问2019对应的字符串是什么？		
// AZ = 1 * 26 + 26 = 52。LQ = 12 * 26 + 17
// ZZ = 26 * 26 + 26
#include &lt;iostream&gt;
using namespace std;

int main()
{
	int i = 0;
	for (char ch = 'A'; ch &lt;= 'Z'; ++ch)
	{
		i++;
		cout &lt;&lt; i &lt;&lt; " " &lt;&lt; ch &lt;&lt; endl;
	}
	cout &lt;&lt; 12 * 26 + 17 &lt;&lt; endl;
	cout &lt;&lt; 329 / 26 &lt;&lt; endl;
	cout &lt;&lt; 26 * 26  &lt;&lt; endl; // 676
	cout &lt;&lt; 2019 / 676 &lt;&lt; endl; // 2 -&gt; B
	cout &lt;&lt; 2019 - 2 * 676 &lt;&lt; endl; // 667
	cout &lt;&lt; 667 / 26 &lt;&lt; endl; // 25 -&gt; Y
	cout &lt;&lt; 667 - 25 * 26 &lt;&lt; endl; // 17 -&gt; Q 
	// BYQ
	return 0;
}</code></code></pre>

    <h2 id="C%EF%BC%9A%E5%88%B7%E9%A2%98%E7%BB%9F%E8%AE%A1">
     3C：数列求值（填空10分_枚举）
    </h2>
    <blockquote>
     <p>
      <strong>
       题目描述：
      </strong>
      <br/>
      给定数列1,1,1,3,5,9,17,…，从第4项开始，每项都是前3项的和。求
      <br/>
      第20190324项的最后4位数字。
     </p>
    </blockquote>
    <p>
     <strong>
      题目解析：
     </strong>
    </p>
    <p>
     暴力循环+位数判断。答案：4659
    </p>
    <pre><code class="language-cpp">
<code class="language-cpp">#include&lt;iostream&gt;
using namespace std;

int main()
{
	int a = 1, b = 1, c = 1, res = 0;
	for(int i = 4; i &lt;= 20190324; ++i)
	{
		res = (a + b + c) % 10000;
		a = b;
		b = c;
		c = res;
	}
	cout &lt;&lt; res &lt;&lt; endl; // 答案4659
	return 0;
}</code></code></pre>

    <h2 id="D%EF%BC%9A%E4%BF%AE%E5%89%AA%E7%81%8C%E6%9C%A8">
     4D：数的分解（填空10分）
    </h2>
    <blockquote>
     <p>
      <strong>
       题目描述：
      </strong>
      <br/>
      把 2019分解成 3个各不相同的正整数之和，并且要求每个正整数都不包
      <br/>
      含数字2和4，一共有多少种不同的分解方法？
      <br/>
      注意交换 3个整数的顺序被视为同一种方法，例如 1000+1001+18和
      <br/>
      1001+1000+18被视为同一种。
     </p>
    </blockquote>
    <p>
     答案40785
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;

bool have_24(int x)
{
	while (x)
	{
		if (x % 10 == 2 || x % 10 == 4)
			return true;
		x /= 10;
	}
	return false;
}

int main()
{
	int res = 0;
	for (int i = 1; i &lt; 2019; ++i)
	{
		if (!have_24(i))
		for (int j = i + 1; j &lt; 2019; ++j)
		{
			if (have_24(j))
				continue;
			int k = 2019 - i - j;
			if (!have_24(k) &amp;&amp; k &gt; j) // i j k 从小到大
				++res;
		}
	}
	cout &lt;&lt; res &lt;&lt; endl; // 答案40785
	return 0;
}</code></pre>
    <hr/>
    <p>
     思路：先获得2021041820210418所有质因数（所以质因数也就一百多个），再通过质因数去组合从而获得所有的正约数，最后只需在所有的正约数找3个乘积为2021041820210418就行。
    </p>
    <p>
     答案：2430
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;vector&gt;

using namespace std;
#define int long long
#define endl '\n'
// n比较大，会爆因子

signed main()
{
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout.tie(0);

	int n = 2021041820210418;
	vector&lt;int&gt; v;
	for (int i = 1; i * i &lt;= n; i++) // 得到n的所有约数
	{
		if (n % i == 0)
		{
			v.push_back(i);
			if (n / i != i)
				v.push_back(n / i);
		}
	}
	//cout &lt;&lt; v.size() &lt;&lt; endl;
	int res = 0;
	for (auto&amp; a : v) //枚举一下a b c
	{
		for (auto&amp; b : v)
		{
			for (auto&amp; c : v)
			{
				if (a * b * c == n)
					++res;
			}
		}
	}
	cout &lt;&lt; res &lt;&lt; endl;
	return 0;
}
// 答案：2430</code></pre>

     一道bfs类型的题目，与平常的bfs不同的是，本题要求的走的路径，所以可以先用bfs计算出相邻格子距离差值为1的路径，然后遍历该路径，并且一定是要求题目所说的字典序来排序（在bfs中无所谓），在main函数中需要讲究先后顺序，所以dir数组的顺序是‘D’,'U','L','R'，如果满足相邻的各自距离为1，那么就将该字母字母加入string ans，然后break跳出，为的是只走一次，因为是一条完整的最短路径。
    </p>
    <p>
     答案：
    </p>
    <p>
     UUUULLRLLLLLLLULLLUUUDUULUUUUUUUUUUUULULULLRLLRRLLUUUULULLLLLLRLLULLUUULLLLRRLRRRRRRRDRDDRRRRLLLLRRDDDRRRRDDRRRDRRLLRLLRLRLLLULULLLLULULUUDDDUULLUULUUDUUUDDUUDDDUDUUUDUULLLLLLLLLUUUUUULL

    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;queue&gt;
using namespace std;
typedef pair&lt;int, int&gt; PII;
char g[30][50];
int dist[30][50];

char dir[] = { 'D', 'L', 'R', 'U' };
int dx[4] = { 0, 0, -1, 1 };
int dy[4] = { -1, 1, 0, 0 };
void bfs()
{
	memset(dist, -1, sizeof(dist));
	queue&lt;PII&gt; q;
	q.push({ 29, 49 });
	dist[29][49] = 0;

	while (!q.empty())
	{
		PII t = q.front();
		q.pop();

		for (int i = 0; i &lt; 4; i++)
		{
			int a = t.first + dx[i], b = t.second + dy[i];
			if (a &lt; 0 || a &gt;= 30 || b &lt; 0 || b &gt;= 50 || g[a][b] == '1' || dist[a][b] != -1)
				continue;
			q.push({ a, b });
			dist[a][b] = dist[t.first][t.second] + 1;
		}
	}
}

int main()
{
	for (int i = 0; i &lt; 30; i++)
	{
		cin &gt;&gt; g[i];
	}
	bfs();
	string ans;
	int x = 0, y = 0;
	while (x != 29 || y != 49)
	{
		for (int i = 0; i &lt; 4; i++)
		{
			int a = x + dx[i], b = y + dy[i];
			if (a &lt; 0 || a &gt;= 30 || b &lt; 0 || b &gt;= 50 || g[a][b] == '1')
				continue;
			if (dist[x][y] == dist[a][b] + 1)
			{
				ans += dir[i];
				x = a;
				y = b;
				break;
			}
		}
	}
	cout &lt;&lt; ans &lt;&lt; endl;
	return 0;
}</code></pre>

     解析代码（数论_最大公约数）
    </h3>
    <blockquote>
     <p>
      <strong>
       题目概述：给出一段序列，求出满足该段序列的最短等差数列的长度
      </strong>
     </p>
     <ol>
      <li>
       既然要求的是最短的等差数列的长度，那么就要要求公差比较大，那么这个序列就短
      </li>
      <li>
       那么问题来了：公差怎么取？
      </li>
     </ol>
     <p>
      如果公差取得比较大，那么很有可能就不满足等差序列这个条件了：比如下面这个例子
     </p>
     <p>
      2  4   8  （如果公差取4，那么2-&gt;4，就不满足公差为4，这个性质了-&gt;不满足等差数列）
     </p>
     <p>
      综上所述：我们要求的是一个数列中满足题意的最小的公差即可，那么现在的问题就转为了：如何求上述的特殊公差？
     </p>
     <p>
      根据数据范围我们可以直到这个算法不能写O(N^2)。2 ≤ N ≤ 100000（两层循环会超时）0 ≤ Ai≤ 10^9
     </p>
     <p>
      <strong>
       优化：
      </strong>
      首先遍历一遍数组是跑不了了，要做的优化就是求最小的公差那部分，有没有一种可能，可以用另外一种方式求公差，那就是欧几里得算法求公约数，为什么可以这样写？为说明会联想到公约数？
     </p>
     <p>
      因为在等差数列中，
      <strong>
       首项为a1，剩下的数均可表示为a1+nd
      </strong>
      ，那么是不是只需要同时减去a[0]，也就是首项，那么剩下的数均可表示为nd，那么n不同，d一定相同，那么就完美满足了最大公约数这个性质了。
     </p>
    </blockquote>
    <pre><code class="language-cpp">#include &lt;cstdio&gt;
#include &lt;cstring&gt;
#include &lt;iostream&gt;
#include &lt;algorithm&gt;
using namespace std;
const int N = 100010;
int a[N];
int gcd(int a, int b)
{
	return b ? gcd(b, a % b) : a;
}
int main()
{
	int n;
	cin &gt;&gt; n;
	for (int i = 0; i &lt; n; i++)
	{
		cin &gt;&gt; a[i];
	}
	sort(a, a + n); // 排序是为了让首项为a[0]
	int d = 0; // 0与任何数的最大公约数都是它的本身
	for (int i = 1; i &lt; n; i++)
	{
		//d = gcd(d, a[i] - a[0]); // 减去首项a1
		d = __gcd(d, a[i] - a[0]); // 减去首项a1，Linux或另一些编译器能用，蓝桥杯也能，VS2022不能
	}
	if (!d)
		printf("%d\n", n); // 如果公约数为0.那么就证明这时一个常数数列
	else
		printf("%d\n", (a[n - 1] - a[0]) / d + 1); // 公式
	return 0;
}
</code></pre>

     而一个曲线符合下列情况时符合要求：
     <br/>
     1.左端点小于右端点 即要求s[0]&lt;s[n]
    </p>
    <pre><code class="language-cpp">        LL s0 = s[0], sn = s[n];
        if (s0 &gt; sn)
            swap(s0, sn);</code></pre>
    <p>
     2.极小值在极大值左边
     <br/>
     该点要求我们在后续选点的时
     <br/>
     应s[0]向左取 s[n]向右取 因为只有这样才能取得两边的极值
    </p>
    <pre><code class="language-cpp">        int l = 0, r = n; // 构造重叠部分最小的序列
        for (int i = s0; i &gt;= 0; i -= 2)
        {
            f[l++] = s[i], st[i] = true;
        }
        for (int i = sn; i &lt;= n; i += 2)
        {
            f[r--] = s[i], st[i] = true;
        }
        for (int i = 0; i &lt;= n; ++i)
        {
            if (st[i] == false)
                f[l++] = s[i];
        }</code></pre>
    <p>
     这样以后就可以保证序列为f为重叠部分最小的前缀和序列
    </p>
    <pre><code class="language-cpp">        LL res = 0;
        for (int i = 1;i &lt;= n;i++)
        {
            res = max(res, abs(f[i] - f[i - 1]));
        }
        cout &lt;&lt; res &lt;&lt; endl;</code></pre>
    <p>
     res即为所求结果，以下为完整代码。
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;algorithm&gt;
#include &lt;cstdio&gt;
#include &lt;cstring&gt;
using namespace std;
const int N = 3e5 + 10;
#define endl '\n'
//由于a[]可能达到1e9所以需要使用到LL
typedef long long LL;

LL a[N];    //用于存放初始灵能值
LL s[N];    //用于存放前缀和
LL f[N];    //用于存放最终的有序序列
bool st[N];
signed main()
{
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout.tie(0);
    int T;
    cin &gt;&gt; T;
    while (T--)
    {
        int n;
        cin &gt;&gt; n;
        s[0] = 0;// 注意这一步不要忘了
        for (int i = 1;i &lt;= n;i++)
        {
            cin &gt;&gt; a[i];
            s[i] = s[i - 1] + a[i];
        }
        LL s0 = s[0], sn = s[n];
        if (s0 &gt; sn)
            swap(s0, sn);
        sort(s, s + 1 + n);
        for (int i = 0; i &lt;= n; ++i) // 找到排序后 s0,sn的位置
        {
            if (s0 == s[i])
            {
                s0 = i;
                break;
            }
        }
        for (int i = 0; i &lt;= n; ++i)
        {
            if (sn == s[i])
            {
                sn = i;
                break;
            }
        }
        memset(st, false, sizeof st);

        int l = 0, r = n; // 构造重叠部分最小的序列
        for (int i = s0; i &gt;= 0; i -= 2)
        {
            f[l++] = s[i], st[i] = true;
        }
        for (int i = sn; i &lt;= n; i += 2)
        {
            f[r--] = s[i], st[i] = true;
        }
        for (int i = 0; i &lt;= n; ++i)
        {
            if (st[i] == false)
                f[l++] = s[i];
        }
        LL res = 0;
        for (int i = 1;i &lt;= n;i++)
        {
            res = max(res, abs(f[i] - f[i - 1]));
        }
        cout &lt;&lt; res &lt;&lt; endl;
    }
    return 0;
}
