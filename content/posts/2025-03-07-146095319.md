    <ol>
     <li>
      <strong>
       硬件抽象层（HAL）
      </strong>
      <br/>
      ST官方提供的固件库（如
      <code>
       stm32f10x_i2c.c
      </code>
      ），直接操作寄存器实现基础功能。
     </li>
     <li>
      <strong>
       总线驱动层
      </strong>
      <br/>
      封装I2C协议的核心操作（发送/接收数据），提供
      <code>
       I2C_Configuration()
      </code>
      等接口。
     </li>
     <li>
      <strong>
       器件驱动层
      </strong>
      <br/>
      针对具体外设（如LM75A温度传感器）的驱动实现。
     </li>
     <li>
      <strong>
       用户应用层
      </strong>
      <br/>
      调用驱动函数实现业务逻辑，如温度显示。
     </li>
    </ol>
