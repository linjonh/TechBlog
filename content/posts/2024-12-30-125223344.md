     <h2 id="1%20%E3%80%81%E5%A4%A7%E6%95%B0%E6%8D%AE%E6%A6%82%E5%BF%B5" style="margin-left:.0001pt;text-align:justify;">
      1 、大数据概念
     </h2>
     <p style="margin-left:.0001pt;text-align:justify;">
      <br/>
      大数据(BigData) :指
      <span style="color:#fe2c24;">
       无法在一定时间范围内
      </span>
      用常规软件工具进行捕捉、管理和
      <br/>
      处理的数据集合.是需要新处理模式才能具有更强的决策力、洞察发现力和流程优化
      <br/>
      能力的
      <span style="color:#fe2c24;">
       海量
      </span>
      、
      <span style="color:#fe2c24;">
       高增长率
      </span>
      和
      <span style="color:#fe2c24;">
       多样化
      </span>
      的
      <span style="color:#fe2c24;">
       信息资产
      </span>
      。
      <br/>
      大数据主要解决，
      <span style="color:#fe2c24;">
       海量数据
      </span>
      的
      <span style="color:#fe2c24;">
       采集
      </span>
      、
      <span style="color:#fe2c24;">
       存储
      </span>
      和
      <span style="color:#fe2c24;">
       分析计算
      </span>
      问题。
     </p>
     <p style="margin-left:.0001pt;text-align:justify;">
      按照顺序给出数据存储单位：bit . Byte  KB MB GB TB PB EB ZB YB BB NB DB
     </p>
     <p style="margin-left:.0001pt;text-align:justify;">
      1Byte = 8bit         1KB = 1024Byte        1MB = 1024KB         1GB = 1024MB
     </p>
     <p style="margin-left:.0001pt;text-align:justify;">
      1G = 1024MB       1TB = 1024GB                1PB = 1024TB
     </p>
    </blockquote>

     <h2 id="%C2%A02%E3%80%81%E5%A4%A7%E6%95%B0%E6%8D%AE%E7%89%B9%E7%82%B9%EF%BC%885V%EF%BC%89" style="margin-left:.0001pt;text-align:justify;">
      2、大数据特点（5V）
     </h2>
     <p>
      <span style="color:#000000;">
       <strong>
        1
       </strong>
      </span>
      <span style="color:#000000;">
       <strong>
        、
       </strong>
      </span>
      <span style="color:#000000;">
       <strong>
        Volume
       </strong>
      </span>
      <span style="color:#000000;">
       <strong>
        （大量）
       </strong>
      </span>
     </p>
     <div>
      <span style="color:#000000;">
       截至目前，人类生产的所有
      </span>
      <span style="color:#ff0000;">
       印刷材料的数据量是
      </span>
      <span style="color:#ff0000;">
       200PB
      </span>
      <span style="color:#000000;">
       ，而历史上全人类总共
      </span>
     </div>
     <div>
      <span style="color:#ff0000;">
       说过的话的数据量大约是
      </span>
      <span style="color:#ff0000;">
       5EB
      </span>
      <span style="color:#000000;">
       。当前，典型个人计算机硬盘的容量为
      </span>
      <span style="color:#000000;">
       TB
      </span>
      <span style="color:#000000;">
       量级，而
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       一些大
      </span>
      <span style="color:#ff0000;">
       企业的数据量已经接近
      </span>
      <span style="color:#ff0000;">
       EB
      </span>
      <span style="color:#000000;">
       量级。
      </span>
     </div>
     <div>
      <div>
       <span style="color:#000000;">
        <strong>
         2
        </strong>
       </span>
       <span style="color:#000000;">
        <strong>
         、
        </strong>
       </span>
       <span style="color:#000000;">
        <strong>
         Velocity
        </strong>
       </span>
       <span style="color:#000000;">
        <strong>
         （高速）
        </strong>
       </span>
      </div>
      <div>
       <div>
        <span style="color:#000000;">
         这是大数据区分于传统数据挖掘的最显著特征。根据
        </span>
        <span style="color:#000000;">
         IDC
        </span>
        <span style="color:#000000;">
         的“数字宇宙”的报
        </span>
       </div>
       <div>
        <span style="color:#000000;">
         告，
        </span>
        <span style="color:#548235;">
         预计到
        </span>
        <span style="color:#548235;">
         2025
        </span>
        <span style="color:#548235;">
         年，全球数据使用量将达到
        </span>
        <span style="color:#548235;">
         163ZB
        </span>
        <span style="color:#000000;">
         。在如此海量的数据面前，处
        </span>
       </div>
       <div>
        <span style="color:#000000;">
         理数据的效率就是企业的生命。
        </span>
       </div>
       <div>
        <span style="color:#ff0000;">
         天猫双十一：
        </span>
        <span style="color:#ff0000;">
         2017
        </span>
        <span style="color:#ff0000;">
         年
        </span>
        <span style="color:#ff0000;">
         3
        </span>
        <span style="color:#ff0000;">
         分
        </span>
        <span style="color:#ff0000;">
         01
        </span>
        <span style="color:#ff0000;">
         秒，天猫交易额超过
        </span>
        <span style="color:#ff0000;">
         100
        </span>
        <span style="color:#ff0000;">
         亿
        </span>
       </div>
       <div>
        <span style="color:#ff0000;">
         2020
        </span>
        <span style="color:#ff0000;">
         年
        </span>
        <span style="color:#ff0000;">
         96
        </span>
        <span style="color:#ff0000;">
         秒，天猫交易额超过
        </span>
        <span style="color:#ff0000;">
         100
        </span>
        <span style="color:#ff0000;">
         亿
        </span>
       </div>
       <div>
        <div>
         <span style="color:#000000;">
          <strong>
           3
          </strong>
         </span>
         <span style="color:#000000;">
          <strong>
           、
          </strong>
         </span>
         <span style="color:#000000;">
          <strong>
           Variety
          </strong>
         </span>
         <span style="color:#000000;">
          <strong>
           （多样）
          </strong>
         </span>
        </div>
        <div>
         <div>
          <span style="color:#000000;">
           这种类型的多样性也让数据被分为结构化数据和非结构化数据。相对于以往便于存储的
          </span>
         </div>
         <div>
          <span style="color:#ff0000;">
           以数据库
          </span>
          <span style="color:#ff0000;">
           /
          </span>
          <span style="color:#ff0000;">
           文本为主的结构化数据
          </span>
          <span style="color:#000000;">
           ，
          </span>
          <span style="color:#ff0000;">
           非结构化数据
          </span>
          <span style="color:#000000;">
           越来越多，包括
          </span>
          <span style="color:#ff0000;">
           网络日志、音频、视频、图
          </span>
         </div>
         <div>
          <span style="color:#ff0000;">
           片、地理位置信息
          </span>
          <span style="color:#000000;">
           等，这些多类型的数据对数据的处理能力提出了更高要求。
          </span>
         </div>
         <div>
          <div>
           <span style="color:#000000;">
            <strong>
             4
            </strong>
           </span>
           <span style="color:#000000;">
            <strong>
             、
            </strong>
           </span>
           <span style="color:#000000;">
            <strong>
             Value
            </strong>
           </span>
           <span style="color:#000000;">
            <strong>
             （低价值密度）
            </strong>
           </span>
          </div>
          <div>
           <div>
            <span style="color:#000000;">
             价值密度的高低与数据总量的大小成反比。
            </span>
           </div>
           <div>
            <div>
             <span style="color:#000000;">
              如何
             </span>
             <span style="color:#ff0000;">
              快速对有价值数据“提纯”成为目前大数据背景下待解决的难题。
             </span>
            </div>
            <div>
             <span style="color:#000000;">
              <strong>
               5、Veracity
              </strong>
             </span>
             <span style="color:#000000;">
              <strong>
               （真实性）
              </strong>
             </span>
            </div>
            <div>
             <span style="color:#000000;">
              <strong>
              </strong>
              可信性，真伪性 来源&amp;信誉 有效性 可审计性
             </span>
            </div>
           </div>
          </div>
         </div>
        </div>
       </div>
      </div>
