    <h3>
     <a id="1REM_4">
     </a>
     1.什么是REM
    </h3>
    <p>
     先来认识几个常见单位：
    </p>
    <ul>
     <li>
      px：像素，这个大家都知道。
     </li>
     <li>
      vh：viewport height，视窗高度，是指占整个浏览视窗高度的百分数。
     </li>
     <li>
      vw：viewport width，视窗宽度，是指占整个浏览视窗宽度的百分数。
     </li>
     <li>
      em：一个大写字母M的宽度（有时候也说一个汉字的宽度，但其实是错的），多少em是指大写字母M的宽度的倍数
     </li>
    </ul>
    <p>
     而rem（Root em） ，理解过来是指根元素（
     <code>
      &lt;html&gt;
     </code>
     标签）中的
     <code>
      font-size
     </code>
     ，虽然
     <strong>
      rem中有em但是这里两个单位没有任何关系
     </strong>
     ，这里需要特别注意。
     <br/>
     举个例子，chrome默认字体大小为16px，如果不设置
     <code>
      html{font-size:XXpx}
     </code>
     ,所以在这时
     <strong>
      1rem就是16px
     </strong>
     ，
    </p>
    <h3>
     <a id="2REMEM_13">
     </a>
     2.REM和EM的区别
    </h3>
    <p>
     <strong>
      首先两者什么关系都没有！
     </strong>
     <br/>
     em是指自身的
     <code>
      font-size
     </code>
     的倍数，这里举个例子
    </p>
    <pre><code class="prism language-css"><span class="token selector">html</span> <span class="token punctuation">{<!-- --></span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>16px<span class="token selector">;
｝
p</span> <span class="token punctuation">{<!-- --></span>
	<span class="token property">font-size</span><span class="token punctuation">:</span>12px<span class="token punctuation">;</span>
	<span class="token property">height</span><span class="token punctuation">:</span>2em<span class="token punctuation">;</span>	<span class="token comment">/*2 x 12 = 24px*/</span>
	<span class="token property">width</span><span class="token punctuation">:</span>2rem<span class="token punctuation">;</span> <span class="token comment">/*2 x 16 = 32px*/</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="3_26">
     </a>
     3.手机端方案的特点
    </h3>
    <p>
     由于分辨率不统一，不同品牌不同设备的屏幕显示比例不同，移动端开发需要做适配的话只能使用等比例放大缩小来进行（这里不考虑响应式，只是在接近的分辨率中进行比较）。
    </p>
    <h3>
     <a id="4JSREM_28">
     </a>
     4.使用JS动态调整REM
    </h3>
    <p>
     如果想通过缩放来显示内容，就必须与显示的宽度（
     <code>
      innerWidth
     </code>
     ）有关系。但是像上面的说的，这几个单位除了
     <code>
      vw
     </code>
     之外都和显示宽度没有关系。然而，
     <code>
      vd
     </code>
     目前浏览器兼容性很差，别说手机，就是电脑浏览器都有些不兼容。因此，就有人想出使用JS来获取显示宽度进行设 置。只需要在html页面中，加载css文件之前添加以下代码
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>script</span><span class="token punctuation">&gt;</span></span><span class="token script language-javascript">
document<span class="token punctuation">.</span><span class="token function">write</span><span class="token punctuation">(</span>`<span class="token operator">&lt;</span>style<span class="token operator">&gt;</span>html<span class="token punctuation">{<!-- --></span>font<span class="token operator">-</span>size<span class="token punctuation">:</span>$<span class="token punctuation">{<!-- --></span>document<span class="token punctuation">.</span>innerWidth<span class="token punctuation">}</span>px<span class="token punctuation">;</span><span class="token punctuation">}</span><span class="token operator">&lt;</span><span class="token operator">/</span>style<span class="token operator">&gt;</span><span class="token punctuation">)</span>； 
</span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>script</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     在此之后可以利用
     <strong>
      1rem=innerWidth
     </strong>
     来进行布局，这个时候
     <strong>
      1rem=100vd
     </strong>
     。如果觉得小数太小了写着不方便也可以取
     <code>
      font-size==｛doucment.innerWidth/10｝
     </code>
     ,
     <font color="red">
      注意，这里不要用100的，因为有的浏览器会设置最小字体，比如chrome浏览器默认最小为12px，当设置小于这个值的时候会不生效
     </font>
    </p>
    <h3>
     <a id="5REM_36">
     </a>
     5.REM与其他单位同时存在
    </h3>
    <p>
     当使用rem的时候不影响其他的单位使用，有时候考虑到以rem为单位的换算成px可能不到一像素，这个时候还是使用px为单位。
    </p>
