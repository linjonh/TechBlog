## 一、文件与目录操作

### (1) 查看目录，切换目录

#### pwd

pwd就是显示当前所造位置的指令。![](https://i-blog.csdnimg.cn/direct/af7452a853bb45f497d568830ed2d2e3.png)

#### ls

ls是展开当前路径下存在的文件。

```
ls -l   # 长格式显示（权限/所有者/大小）
ls -a   # 显示隐藏文件（以.开头）
ls -t   # 按修改时间排序
ls -lh  # 人类可读的文件大小（KB/MB）

```

![](https://i-blog.csdnimg.cn/direct/aeb711b92f7d4e639e5bb104b879f347.png)

#### cd

cd则是切换位置

```
cd .. #返回上级
cd /  #绝对路径定位⽬录或者⽂件
cd ~ # 快速进⼊⾃⼰的家⽬录
cd - # 直接可以回退到最近⼀次所处的⽬录
```

### （2）创建、 删除

#### mkdir

创建目录

```
mkdir #创建目录
mkdir -p #递归创建目录
```

![](https://i-blog.csdnimg.cn/direct/89f8a9188e9c4d3f9f519784b2c8a613.png)

#### touch

touch命令参数可更改⽂档或⽬录的⽇期时间，包括存取时间和更改时间，或者新建⼀个不存在 的⽂件。

此次只介绍简单的创建文件

![](https://i-blog.csdnimg.cn/direct/54ef9613eade4d2e91c33500e4369e98.png)

#### rmdir

删除空目录

```
rmdir #只能删除一个

```

#### rm

删除目录或者文件

```
rm -i *.tmp   # 删除时需确认
rm -rf /path  # 强制递归删除（慎用）

```

rmdir仅仅只能删除空目录因此我们常常使用rm而非rmdir![](https://i-blog.csdnimg.cn/direct/921b30d6727a49d58cbf45002a3549b6.png)

#### cp

复制⽂件或⽬录

```
cp -v src/* dest/  # 显示复制详情
cp -r dir1 dir2    # 递归复制目录
cp -u *.js dist/   # 仅复制更新过的文件

```

![](https://i-blog.csdnimg.cn/direct/3bba977ab17a48f897dca31e7e4b8b03.png)

#### mv

mv命令是move的缩写，可以⽤来移动⽂件或者将⽂件改名（move(rename)files,经常⽤来备份⽂件 或者⽬录

```
mv *.log /archive/  # 批量移动日志
mv oldname.md newname.md  # 重命名文件

```

> 1. 视mv命令中第⼆个参数类型的不同（是⽬标⽂件还是⽬标⽬录），mv命令将⽂件重命名或将其 移⾄⼀个新的⽬录中。
>
> 2. 当第⼆个参数类型是⽂件时，mv命令完成⽂件重命名，此时，源⽂件只能有⼀个（也可以是源⽬ 录名），它将所给的源⽂件或⽬录重命名为给定的⽬标⽂件名。
>
> 3. 当第⼆个参数是已存在的⽬录名称时，源⽂件或⽬录参数可以有多个，mv命令将各参数指定的源 ⽂件均移⾄⽬标⽬录中。

![](https://i-blog.csdnimg.cn/direct/a01b6c61975146a1a056c8e576e97676.png)

第二个参数为文件名，更改名字![](https://i-blog.csdnimg.cn/direct/28e92c25524e46079e0caeee8d5016b4.png)

第二个参数为补录，将其挪动。

### （2）系统帮助

#### man

man 指令（Manual的缩写）是 Linux 系统中一个重要的工具，用于查看各种命令、系统调用、库函数等的帮助手册。

![](https://i-blog.csdnimg.cn/direct/dde4544ef1e84195ac58c840e04bb764.png)![](https://i-blog.csdnimg.cn/direct/825acdf289cf4917a3e5f6495f3ac83a.png)我们就可以看到man的使用方法

> • 1是普通的命令  
>  • 2是系统调⽤,如open,write之类的(通过这个，⾄少可以很⽅便的查到调⽤这个函数，需要加什么头⽂件)  
>  • 3是库函数,如printf,fread4是特殊⽂件,也就是/dev下的各种设备⽂件  
>  • 5是指⽂件的格式,⽐如passwd,就会说明这个⽂件中各个字段的含义  
>  • 6是给游戏留的,由各个游戏⾃⼰定义  
>  • 7附件还有⼀些变量,⽐如像environ这种全局变量在这⾥就有说明  
>  • 8是系统管理⽤的命令,这些命令只能由root使⽤,如ifconfig

![](https://i-blog.csdnimg.cn/direct/41d0772dcc6c4e2d8c719f774da47787.png)![](https://i-blog.csdnimg.cn/direct/2f48b8fa23aa42cd9f755f2d8c4e6884.png)

#### top

任务管理器

| **快捷键** | **功能** | **应用场景** |
| --- | --- | --- |
| `P` | 按 CPU 使用率排序 | 定位高 CPU 进程 |
| `M` | 按内存占用排序 | 检测内存泄漏 |
| `T` | 按运行时间排序 | 查找长期运行进程 |
| `k` | 杀死进程（需输入 PID） | 终止异常进程 |
| `1` | 展开多核 CPU 详情 | 分析各核心负载均衡 |
| `z` | 切换颜色高亮 | 增强可读性 |
| `E`/`e` | 切换内存单位（KiB/MiB/GiB） | 精确查看大内存占用 |

### (2)tar格式

#### **tar**

`tar` 是 Linux 系统中功能强大的文件归档和压缩命令，常用于文件的打包、备份、传输等操作

> c ：建⽴⼀个压缩⽂件的参数指令(create 的意思)  
>  -x ：解开⼀个压缩⽂件的参数指令  
>  -t ：查看 tarfile ⾥⾯的⽂件  
>  -z ：是否同时具有 gzip 的属性？亦即是否需要⽤ gzip 压缩  
>  -j ：是否同时具有 bzip2 的属性？亦即是否需要⽤ bzip2 压缩  
>  -v ：压缩的过程中显⽰⽂件！这个常⽤，但不建议⽤在背景执⾏过程  
>  -f ：使⽤档名，请留意，在 f 之后要⽴即接档名喔！不要再加参数  
>  -C ： 解压到指定⽬录
