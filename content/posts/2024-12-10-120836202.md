    <h2>
     <a id="_6">
     </a>
     接口说明
    </h2>
    <p>
     采用AD9364接收的差分时钟[DATA_CLK_P/ DATA_CLK_N]作为时钟源，衍生出同频单端时钟iddr_clk，用于接收差分信号。衍生出同频单端时钟oddr_clk，作为发射差分信号时钟源，衍生出半频sample_clk信号用于信号处理
     <br/>
     本工程中Iddr_clk 和oddr_clk 为40MHz；Sample_clk为20MHz
    </p>

    <h2>
     <a id="ADCDAC_FPGA_25">
     </a>
     ADC/DAC FPGA实现框图
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/31d371564ae6c7f8d2594bda016a6ca0.png"/>
     <br/>
     注意：iddr_clk或者oddr_clk 的时钟相位要配合着 SPI指令来调准
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/67dc6f1bd1dacf86b67fa46185ec3cf5.png"/>
     <br/>
     首先要通过AD9364模块的自检指令确定ADC/DAC接口没有时序问题，再进行信号测试。
    </p>
    <p>
     参考AD9361 BIST FAQ.pdf文档 总结了如下可以设置的测试项（指令），对ADC/DAC接口进行测试：
     <br/>
     发射9361内部正弦波
     <br/>
     发射9361内部伪随机码
     <br/>
     接收9361内部正弦波
     <br/>
     接收9361内部伪随机码（可观测误码发生次数）
     <br/>
     发射FPGA生成的直流
     <br/>
     发射FPGA生成的线性增加数
     <br/>
     发射FPGA生成伪随机码
     <br/>
     发射FPGA生成的正弦波
     <br/>
     回环测试_回环直流
     <br/>
     回环测试_回环线性增加数
     <br/>
     回环测试_回环伪随机码（可观测误码发生次数）
     <br/>
     回环测试_回环正弦波
    </p>

     这些测试项后面再详细列出实现指令
    </p>
