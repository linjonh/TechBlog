    <h2>
     <a id="1_PythonEasyGUI_codebox_12">
     </a>
     1 Python图形编程之EasyGUI: codebox的用法
    </h2>
    <h3>
     <a id="11__14">
     </a>
     1.1 基本用法
    </h3>
    <p>
     <code>
      codebox
     </code>
     是
     <code>
      textbox
     </code>
     的一个简化封装，函数内部调用了
     <code>
      textbox
     </code>
     ，且完全可以由
     <code>
      textbox
     </code>
     窗口来代替，仅需要将
     <code>
      textbox
     </code>
     函数的
     <code>
      codebox
     </code>
     参数设置为
     <code>
      True
     </code>
     即可。另外，
     <code>
      codebox
     </code>
     功能上还缺少回调函数机制。
    </p>
    <ul>
     <li>
      <strong>
       示例代码：
      </strong>
     </li>
    </ul>
    <pre><code class="prism language-py"><span class="token keyword">import</span> easygui <span class="token keyword">as</span> eg

ret  <span class="token operator">=</span> eg<span class="token punctuation">.</span>codebox<span class="token punctuation">(</span>msg<span class="token operator">=</span><span class="token string">"请输入代码"</span><span class="token punctuation">,</span> title<span class="token operator">=</span><span class="token string">"代码输入"</span><span class="token punctuation">,</span>text<span class="token operator">=</span><span class="token string">"#include &lt;stdio.h&gt;"</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>ret<span class="token punctuation">)</span>
</code></pre>
    <ul>
     <li>
      <strong>
       效果：
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3b808c6328024d7ebe69fd59ef90540e.png" width="600"/>
     </li>
    </ul>
    <h3>
     <a id="12__31">
     </a>
     1.2 参数及返回值
    </h3>
    <ul>
     <li>
      <strong>
       函数原型：
      </strong>
     </li>
    </ul>
    <pre><code class="prism language-py"><span class="token keyword">def</span> <span class="token function">codebox</span><span class="token punctuation">(</span>msg<span class="token operator">=</span><span class="token string">""</span><span class="token punctuation">,</span> title<span class="token operator">=</span><span class="token string">" "</span><span class="token punctuation">,</span> text<span class="token operator">=</span><span class="token string">""</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        参数：
       </strong>
      </p>
      <ul>
       <li>
        <code>
         msg
        </code>
        ：提示信息
       </li>
       <li>
        <code>
         title
        </code>
        ：窗口标题
       </li>
       <li>
        <code>
         text
        </code>
        ：要显示的大段文本
       </li>
      </ul>
     </li>
     <li>
      <p>
       **返回值：**编辑框中的文本
      </p>
     </li>
    </ul>

     🕮说明
    </b>
    ：
    <br/>
    这里的代码编辑窗口中的编辑控件，并不能支持代码的语法着色，仅仅是在换行上做了点设置(不换行)。
