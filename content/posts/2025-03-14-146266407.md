    <h2>
     <a id="Mac__Xcode__Clang__1">
     </a>
     Mac 系统下安装 Xcode 或 Clang 详细指南
    </h2>
    <h3>
     <a id="_3">
     </a>
     一、安装前的准备
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        确认系统版本
       </strong>
       <br/>
       macOS 10.9 及以上版本支持 Xcode 和 Clang。若版本过低，需先升级系统。
      </p>
     </li>
     <li>
      <p>
       <strong>
        了解工具区别
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         Xcode
        </strong>
        ：苹果官方 IDE，包含完整开发环境、模拟器、调试工具等，适合 iOS/macOS 应用开发。
       </li>
       <li>
        <strong>
         Clang
        </strong>
        ：轻量级 C/C++ 编译器，可通过终端直接使用，适合简单项目或命令行开发。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_Clang_12">
     </a>
     二、安装 Clang（命令行工具）
    </h3>
    <h4>
     <a id="_1_13">
     </a>
     步骤 1：打开终端
    </h4>
    <ul>
     <li>
      <strong>
       方法 1
      </strong>
      ：点击屏幕右上角「Spotlight 搜索」（放大镜图标），输入
      <code>
       终端
      </code>
      并回车。
     </li>
     <li>
      <strong>
       方法 2
      </strong>
      ：使用快捷键
      <code>
       Command + 空格
      </code>
      调出搜索，输入
      <code>
       终端
      </code>
      后回车。
     </li>
    </ul>
    <h4>
     <a id="_2_17">
     </a>
     步骤 2：检查是否已安装
    </h4>
    <p>
     在终端输入以下命令并回车：
    </p>
    <pre><code class="prism language-bash">xcode-select <span class="token parameter variable">--version</span>
</code></pre>
    <ul>
     <li>
      <strong>
       若显示版本号
      </strong>
      ：说明已安装，可跳过后续步骤。
     </li>
     <li>
      <strong>
       若提示未安装
      </strong>
      ：继续执行安装命令。
     </li>
    </ul>
    <h4>
     <a id="_3_25">
     </a>
     步骤 3：安装命令行工具
    </h4>
    <p>
     在终端输入以下命令并回车：
    </p>
    <pre><code class="prism language-bash">xcode-select <span class="token parameter variable">--install</span>
</code></pre>
    <ul>
     <li>
      弹出窗口后点击「安装」，阅读并同意许可协议。
     </li>
     <li>
      等待安装完成（约 5-10 分钟），点击「完成」。
     </li>
    </ul>
    <h4>
     <a id="_4_33">
     </a>
     步骤 4：验证安装
    </h4>
    <p>
     输入以下命令检查 Clang 版本：
    </p>
    <pre><code class="prism language-bash">clang <span class="token parameter variable">--version</span>
</code></pre>
    <ul>
     <li>
      预期输出类似：
      <code>
       Apple clang version 14.0.0
      </code>
      。
     </li>
    </ul>
    <h3>
     <a id="_Xcode_41">
     </a>
     三、安装 Xcode（完整开发环境）
    </h3>
    <h4>
     <a id="_1_App_Store__42">
     </a>
     步骤 1：从 App Store 下载
    </h4>
    <ol>
     <li>
      打开「App Store」，搜索「Xcode」。
     </li>
     <li>
      点击「获取」并安装（文件较大，需耐心等待）。
     </li>
    </ol>
    <h4>
     <a id="_2_46">
     </a>
     步骤 2：启动并配置
    </h4>
    <ol>
     <li>
      安装完成后，打开「Applications」文件夹，双击运行 Xcode。
     </li>
     <li>
      首次启动需同意许可协议，点击「同意」。
     </li>
     <li>
      等待 Xcode 配置组件（约 10-30 分钟）。
     </li>
    </ol>
    <h4>
     <a id="_3_51">
     </a>
     步骤 3：配置命令行工具
    </h4>
    <ol>
     <li>
      打开终端，输入以下命令并回车：
      <pre><code class="prism language-bash">xcode-select <span class="token parameter variable">--install</span>
</code></pre>
     </li>
     <li>
      弹出窗口后点击「安装」，按提示完成安装。
     </li>
    </ol>
    <h4>
     <a id="_4_58">
     </a>
     步骤 4：验证安装
    </h4>
    <p>
     输入以下命令检查 Xcode 版本：
    </p>
    <pre><code class="prism language-bash">xcodebuild <span class="token parameter variable">-version</span>
</code></pre>
    <ul>
     <li>
      预期输出类似：
      <code>
       Xcode 15.0
      </code>
      。
     </li>
    </ul>
    <h3>
     <a id="_66">
     </a>
     四、常见问题处理
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        提示“已安装但需更新”
       </strong>
      </p>
      <ul>
       <li>
        打开「App Store」，进入「更新」标签页，查找 Xcode 更新并安装。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        权限问题
       </strong>
      </p>
      <ul>
       <li>
        部分操作需管理员权限，按提示输入密码即可。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安装失败
       </strong>
      </p>
      <ul>
       <li>
        尝试重启电脑后重新执行安装命令。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_77">
     </a>
     五、总结
    </h3>
    <ul>
     <li>
      <strong>
       仅需 Clang
      </strong>
      ：通过
      <code>
       xcode-select --install
      </code>
      快速安装。
     </li>
     <li>
      <strong>
       需要完整开发环境
      </strong>
      ：从 App Store 下载 Xcode，包含 Clang 及其他工具。
     </li>
    </ul>
    <p>
     通过以上步骤，您可以轻松在 Mac 上搭建 C/C++ 开发环境。若需进一步配置 IDE（如 VS Code），可参考官方文档或第三方教程。
    </p>
