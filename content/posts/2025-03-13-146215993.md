    <h2>
     一. 优先级队列
    </h2>
    <h3>
     1.1  优先级队列的概念
    </h3>
    <p>
     优先级队列是一种特殊的队列，它在入队时会根据元素的优先级进行排序，优先级最高的元素排在队列的前面，出队时会优先出队优先级最高的元素。
    </p>
    <h3>
     1.2 优先级队列的区别
    </h3>
    <h3>
     （1）与普通队列的区别
    </h3>
    <ol>
     <li>
      普通队列是先进先出的，元素按照入队的顺序依次出队。
     </li>
     <li>
      优先级队列不考虑入队的先后顺序，只根据元素的优先级来决定出队顺序
     </li>
    </ol>
    <h3>
     （2）与栈的区别
    </h3>
    <ol>
     <li>
      栈是后进先出的，最后入栈的元素最先出栈。
     </li>
     <li>
      优先级队列则根据优先级决定出队顺序，与入队顺序无关。
     </li>
    </ol>
    <p>
     <span style="color:#fe2c24">
      <strong>
       核心特点
      </strong>
     </span>
     ：
     <strong>
      元素按优先级排序，每次取出优先级最高（或最低）的元素
     </strong>
    </p>
    <h2>
     二. 堆
    </h2>
    <p>
     PriorityQueue类底层使用堆这种数据结构，堆适用于完全二叉树
    </p>
    <h3>
     2.1 堆的概念
    </h3>
    <p>
     <strong>
      堆
     </strong>
     是一种特殊的
     <strong>
      完全二叉树
     </strong>
     数据结构
    </p>
    <p>
     完全二叉树：除了最后一层外，每一层的节点数都是满的，并且最后一层的节点都靠左排列。
    </p>
    <p>
     堆的分类：
    </p>
    <p>
     <img alt="" height="412" src="https://i-blog.csdnimg.cn/direct/ff6bf92d922d4a20a2a5f558f127d3db.png" width="1006"/>
    </p>
    <p>
     <strong>
      （1）最大堆：
     </strong>
     父节点 ≥ 子节点，根节点为全局最大值。
    </p>
    <p>
     <strong>
      （2）最小堆：
     </strong>
     父节点 ≤ 子节点，根节点为全局最小值。
    </p>
    <h3>
     2.2 堆的性质
    </h3>
    <p>
     <strong>
      （1）堆序性：
     </strong>
    </p>
    <ol>
     <li>
      根节点是堆中的最大元素（大根堆）或最小元素（小根堆）
     </li>
    </ol>
    <ul>
     <li>
      比如：在大根堆中，根节点（第一层）值大于等于第二层子节点的值，第二层子节点的值又大于等于第三层子节点的值，以此类推。
     </li>
    </ul>
    <p>
     <strong>
      （2）完全二叉树：
     </strong>
    </p>
    <ol>
     <li>
      <p>
       所有层（除最后一层）必须完全填满
      </p>
     </li>
     <li>
      <p>
       最后一层的节点必须从左到右连续，不能出现中间空缺
      </p>
     </li>
    </ol>

    <h3>
     2.3 堆的操作
    </h3>
    <h4>
     （1）向下调整
    </h4>
    <p>
     常用于
     <strong>
      创建堆和删除操作
     </strong>
    </p>
    <p class="img-center">
     <img alt="" height="302" src="https://i-blog.csdnimg.cn/direct/64b3a813b2e2493dbd73549ec23bdce3.png" width="518"/>
    </p>
    <p>
     主要步骤：
    </p>
    <p>
     循环条件：子节点的下标小于总数
    </p>
    <ol>
     <li>
      根据父亲节点找到字节点
     </li>
     <li>
      比较得出左右节点的最大节点
     </li>
     <li>
      比较父节点和最大节点，如果左右最大节点大于父节点，则交换
     </li>
     <li>
      父亲节点等于字节点
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java">    //向下调整
    public void shiftDown(int i ,int usedSize){
        int child = (i*2)+1;//左孩子
        while (child&lt;usedSize) {
            if (child + 1 &lt; usedSize &amp;&amp; elem[child] &lt; elem[child + 1]) {
                child++;
            }
            if (elem[child] &gt; elem[i]) {
                swap(child,i);
                i = child;
                child = (i*2)+1;
            }else {
                break;
            }
        }
    }</code></pre>
    <h4>
     （2）向上调整
    </h4>
    <p>
     常用于
     <strong>
      增加操作
     </strong>
    </p>
    <p>
     <img alt="" height="426" src="https://i-blog.csdnimg.cn/direct/cfed2910b6a348cab7484a9f9d9378f7.png" width="1246"/>
    </p>
    <p>
     主要步骤：
    </p>
    <p>
     循环条件：子节点的下标大于0
    </p>
    <ol>
     <li>
      找出子节点的父亲节点
     </li>
     <li>
      与父亲节点进行大小比较，大于则交换
     </li>
     <li>
      让子节点等于父亲
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java">    //向上调整
    public void shiftUp(int child){
        int parent = (child-1)/2;
        while (child&gt;0) {
            if (elem[parent] &lt; elem[child]) {
                swap(parent, child);
                child = parent;
                parent = (child - 1) / 2;
            } else {
                break;
            }
        }
    }</code></pre>
    <h4>
     （3）堆的创建（最大堆）
    </h4>
    <p>
     <strong>
      从最后一个非叶子节点开始，自底向上逐个向下调整
     </strong>
    </p>
    <p>
     <img alt="" height="972" src="https://i-blog.csdnimg.cn/direct/456a76a268974daaa058311f967b3211.png" width="1639"/>
    </p>
    <p>
     主要步骤：
    </p>
    <ol>
     <li>
      得到最大下标的父亲节点（最后一个非叶子节点）
     </li>
     <li>
      向下调整
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java"> public void createHeap(){
        //找出最大的根节点
        for (int i = (usedSize-1-1)/2; i &gt;=0 ; i--) {
            shiftDown(i,usedSize);
        }
    }
    //向下调整
    public void shiftDown(int i ,int usedSize){
        int child = (i*2)+1;//左孩子
        while (child&lt;usedSize) {
            if (child + 1 &lt; usedSize &amp;&amp; elem[child] &lt; elem[child + 1]) {
                child++;
            }
            if (elem[child] &gt; elem[i]) {
                swap(child,i);
                i = child;
                child = (i*2)+1;
            }else {
                break;
            }
        }
    }</code></pre>
    <h4>
     （4）堆的插入
    </h4>
    <p>
     <img alt="" height="389" src="https://i-blog.csdnimg.cn/direct/2f0f6fefb67d49849d74036104ba3979.png" width="1138"/>
    </p>
    <p>
     主要步骤：
    </p>
    <ol>
     <li>
      将插入的元素放入最后一位
     </li>
     <li>
      向上调整
     </li>
     <li>
      堆的元素个数加一个
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java">    public void offer(int val){
        if(isFull()){
            this.elem = Arrays.copyOf(elem,elem.length+1);
        }else {
            elem[usedSize] = val;
            shiftUp(usedSize);
            usedSize++;
        }
    }</code></pre>
    <p>
     注意：必须是在已经有序的基础上插入
    </p>
    <h4>
     （5）堆的删除
    </h4>
    <p>
     <strong>
      堆的删除是删除栈顶元素
     </strong>
    </p>
    <p>
     <img alt="" height="338" src="https://i-blog.csdnimg.cn/direct/be1aca0a1a8c4455a3b939824744d799.png" width="1367"/>
    </p>
    <p>
     主要步骤：
    </p>
    <ol>
     <li>
      将栈顶元素和最后一个元素进行交换
     </li>
     <li>
      堆中的元素总数减少一个
     </li>
     <li>
      栈顶元素向下调整
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java">    public int poll(){
        int tmp = elem[0];
        swap(0,usedSize-1);
        usedSize--;
        shiftDown(0,usedSize);
        return tmp;
    }</code></pre>
    <h4>
     <strong>
      （6）堆的排序
     </strong>
    </h4>
    <p>
     最大堆——由小到大
    </p>
    <p class="img-center">
     <img alt="" height="425" src="https://i-blog.csdnimg.cn/direct/83957e0a5d5f45f8af0a5babfd618828.png" width="475"/>
    </p>
    <p>
     主要步骤：
    </p>
    <ol>
     <li>
      交换堆顶与末尾元素
     </li>
     <li>
      调整剩余堆
     </li>
     <li>
      重复直至有序
     </li>
    </ol>
    <p>
     代码实现：
    </p>
    <pre><code class="language-java">    //排序从小 到大
    public void heapSort(){
        int end = usedSize-1;
        while(end&gt;0){
            swap(0,end);
            shiftDown(0,end);
            end--;
        }
    }</code></pre>

    <h3>
     3.3 练习
    </h3>
    <p>
     top-k问 题：最大或者最小的前k个数据
    </p>
    <p>
     <strong>
      求前k最大元素--使用最小堆
     </strong>
    </p>
    <p>
     <strong>
      求前k最小元素--使用最大堆
     </strong>
    </p>
    <h5>
     <strong>
      （1）求前k个最大元素
     </strong>
    </h5>
    <p>
     <strong>
      方法1：使用最大堆
     </strong>
     （不推荐，如果数据太多，时间复杂度会很大，很占用内存）
    </p>
    <p>
     主要步骤：
    </p>
    <ol>
     <li>
      将数组内所有元素放入优先级队列中
     </li>
     <li>
      每次删除队列中的最大值
     </li>
     <li>
      将最大值赋值给新的数组内
     </li>
    </ol>
    <pre><code class="language-java">    public int[] maxKK(int[] arr, int k) {
        int[] ret = new int[k];
        if(arr==null||k&lt;=0){
            return ret;
        }
        //默认最小堆，通过使用比较器，变成默认最大堆
        PriorityQueue&lt;Integer&gt; queue = new PriorityQueue&lt;&gt;(arr.length);
        for (int i = 0; i &lt; arr.length; i++) {
            queue.offer(arr[i]);
        }
        //每次弹出的就是最大的
        for (int i = 0; i &lt; k; i++) {
            ret[i] = queue.poll();
        }
        return ret;
    }

    </p>
