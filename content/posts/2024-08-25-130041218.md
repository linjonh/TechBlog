    <p>
     获取时间的方式有很多：
     <span style="background-color:#ffd900;">
      GPS授时
     </span>
     、
     <span style="background-color:#ffd900;">
      无卡CDMA授时

     和
     <span style="background-color:#ffd900;">
      网络授时
     </span>
     等，NTP是通过Internet或局域网，从NTP服务器获取时间。
    </p>
    <blockquote>
     <p>
      <a href="https://so.csdn.net/so/search?q=NTP&amp;spm=1001.2101.3001.7020" title="NTP">
       NTP
      </a>
      时间服务器是用于局域网服务器时间同步使用的，可以保证局域网所有的服务器与时间服务器的时间保持一致，SP-SS06NTP授时服务器具有抗干扰性强、低功耗、多平台多系统支持、故障报警等功能，为网络设备（用户）提供精确、标准、安全、可靠和多功能的同步时间服务。
     </p>
    </blockquote>

    <h2 id="%E4%B8%80%E3%80%81NTP">
     一、NTP
    </h2>
    <p>
     详细的NTP协议移步更专业的文章：
     <a class="link-info" href="http://t.csdn.cn/z7uFW" rel="nofollow" title="NTP服务器授时原理">
      NTP服务器授时原理
     </a>
    </p>
    <blockquote>
     <p>
      客户端首先向服务端发送一个NTP 包，其中包含了该包离开客户端的时间戳T1，当服务端接收到该包时，依次填入包到达的时间戳T2、包离开的时间戳T3，然后立即把包返回给客户端。客户端在接收到响应包时，记录包返回的时间戳T4。
     </p>
    </blockquote>
    <h2 id="%C2%A0%E4%BA%8C%E3%80%81%E6%8A%A5%E6%96%87%E8%A7%A3%E6%9E%90">
     二、报文解析
    </h2>

     如果只是获取一个简单的时间，大可不必发送复杂的报文，Client发送的报文只需要这样：

     总共48个字节，23表示是Client，也可以是1b
    </p>

    </p>
    <p>

    <p>
     简单点，第41到48个字节取出来，转换成十进制数就是一个时间戳，这是相对于1970年的时间戳。
    </p>
    <h2 id="%E4%B8%89%E3%80%81%E4%BB%A3%E7%A0%81">
     三、代码
    </h2>
    <p>
     1.发送请求报文，这个根据连接方式不同，代码不同，不做笔记
    </p>
    <p>
     2.接收报文分析：
    </p>
    <p>
     一些定义：
    </p>
    <pre><code class="language-cpp">#define SEC_1900_1970         (2208988800UL)  //(1900-1970年的秒数)

typedef struct
{
  int year;
  uint8_t month;
  uint8_t day;
  uint8_t hour;
  uint8_t minute;
  uint8_t second;
}Time;
</code></pre>
    <pre><code class="language-cpp">//从NTP报文中获取时间，返回0表示获取成功，时间正确性高，返回1表示获取失败
//@data: 报文指针，48个字节
//@time：用于存放时间的Time指针变量
uint8_t NTP_GetTime(uint8_t *data , Time *t)
{
    int time_stamp;        
    struct tm *tm;
    time_t tick;

    time_stamp = (data[40]&lt;&lt;24)|(data[41]&lt;&lt;16)|(data[42]&lt;&lt;8)|(data[43]&lt;&lt;0);     //相对于1970
    time_stamp -= SEC_1900_1970;                                                //得到相对于1900的时间戳
    tick = (time_t)time_stamp;
    tm = localtime(&amp;tick);

    t-&gt;year     = tm-&gt;tm_year+1900;
    t-&gt;month    = tm-&gt;tm_mon+1;
    t-&gt;day      = tm-&gt;tm_mday;
    t-&gt;hour     = tm-&gt;tm_hour;
    t-&gt;minute   = tm-&gt;tm_min;
    t-&gt;second   = tm-&gt;tm_sec;
    UTCToBeijing(t);    //UTC时间转为北京时间
    if((t-&gt;year&gt;0)&amp;&amp;(t-&gt;month&gt;0)&amp;&amp;(t-&gt;month&lt;13)&amp;&amp;(t-&gt;day&gt;0)&amp;&amp;(t-&gt;day&lt;32)&amp;&amp;(t-&gt;hour&gt;0)&amp;&amp;(t-&gt;hour&lt;24)&amp;&amp;(t-&gt;minute&lt;60)&amp;&amp;(t-&gt;second&lt;60))   //判断范围是否正确
    {
        return 1;
    }else
    {
        return 0;
    }
}

void UTCToBeijing(Time *UTC)
{
	int year=0,month=0,day=0,hour=0;
    int lastday = 0;// 月的最后一天日期
	
	year=UTC-&gt;year;
	month=UTC-&gt;month;
	day=UTC-&gt;day;
	hour=UTC-&gt;hour+8;//UTC+8转换为北京时间
	if(month==1 || month==3 || month==5 || month==7 || month==8 || month==10 || month==12)
	{
      	lastday = 31;
    }
    else if(month == 4 || month == 6 || month == 9 || month == 11)
	{
    	lastday = 30;
    }
    else
	{
    	if((year%400 == 0)||(year%4 == 0 &amp;&amp; year%100 != 0))//闰年的2月为29天，平年为28天
    	    lastday = 29;
    	else
    	    lastday = 28;
    }
	if(hour &gt;= 24)//当算出的时大于或等于24：00时，应减去24：00，日期加一天
	{
		hour -= 24;
		day += 1; 
		if(day &gt; lastday)//当算出的日期大于该月最后一天时，应减去该月最后一天的日期，月份加上一个月
		{ 
			day -= lastday;
			month += 1;
			if(month &gt; 12)//当算出的月份大于12，应减去12，年份加上1年
			{
				month -= 12;
				year += 1;
			}
		}
	}
	UTC-&gt;year = year;
    UTC-&gt;month = month;
    UTC-&gt;day = day;
    UTC-&gt;hour = hour;
}</code></pre>
    <p>
     NTP的时间戳是相对于1970年，而localtime函数是相对于1900年的时间戳转换，所以NTP的时间戳要减去1970-1900的秒数，转换出来之后是UTC时间，再转换为北京时间。
    </p>
