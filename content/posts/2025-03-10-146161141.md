    <p>
     在 C++ 中，构造函数（Constructor）是一种特殊的成员函数，用于初始化对象。构造函数在创建对象时自动调用，且名称必须与类名相同。以下是 C++ 中所有常见的构造函数类型及其实现方法：
    </p>
    <hr/>
    <h4>
     1.
     <strong>
      默认构造函数（Default Constructor）
     </strong>
    </h4>
    <p>
     默认构造函数是没有参数的构造函数。如果用户没有定义任何构造函数，编译器会自动生成一个默认构造函数。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName() {
    // 初始化代码
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    MyClass() {
        cout &lt;&lt; "Default Constructor Called!" &lt;&lt; endl;
    }
};

int main() {
    MyClass obj; // 调用默认构造函数
    return 0;
}</code></pre>
    <p>
    </p>
    <h4>
     2.
     <strong>
      参数化构造函数（Parameterized Constructor）
     </strong>
    </h4>
    <p>
     参数化构造函数接受一个或多个参数，用于初始化对象的成员变量。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName(parameters) {
    // 初始化代码
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    int value;
    MyClass(int v) {
        value = v;
        cout &lt;&lt; "Parameterized Constructor Called! Value: " &lt;&lt; value &lt;&lt; endl;
    }
};

int main() {
    MyClass obj(10); // 调用参数化构造函数
    return 0;
}</code></pre>
    <h4>
     3.
     <strong>
      拷贝构造函数（Copy Constructor）
     </strong>
    </h4>
    <p>
     拷贝构造函数用于通过另一个同类型的对象初始化新对象。它通常以引用方式接受一个同类型的对象作为参数。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName(const ClassName&amp; other) {
    // 拷贝初始化代码
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    int value;
    MyClass(int v) : value(v) {} // 参数化构造函数
    MyClass(const MyClass&amp; other) {
        value = other.value;
        cout &lt;&lt; "Copy Constructor Called! Value: " &lt;&lt; value &lt;&lt; endl;
    }
};

int main() {
    MyClass obj1(10); // 调用参数化构造函数
    MyClass obj2 = obj1; // 调用拷贝构造函数
    return 0;
}</code></pre>
    <p>
    </p>
    <h4>
     4.
     <strong>
      移动构造函数（Move Constructor）
     </strong>
    </h4>
    <p>
     移动构造函数用于通过“移动”另一个对象的资源来初始化新对象。它通常以右值引用方式接受一个同类型的对象作为参数。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName(ClassName&amp;&amp; other) {
    // 移动资源代码
}</code></pre>
    <h5>
     示例：
    </h5>
    <pre><code class="language-cpp">class MyClass {
public:
    int* data;
    MyClass(int size) {
        data = new int[size];
        cout &lt;&lt; "Parameterized Constructor Called!" &lt;&lt; endl;
    }
    MyClass(MyClass&amp;&amp; other) noexcept {
        data = other.data; // 移动资源
        other.data = nullptr; // 置空原对象的指针
        cout &lt;&lt; "Move Constructor Called!" &lt;&lt; endl;
    }
    ~MyClass() {
        delete[] data;
    }
};

int main() {
    MyClass obj1(10); // 调用参数化构造函数
    MyClass obj2 = std::move(obj1); // 调用移动构造函数
    return 0;
}</code></pre>
    <h4>
     5.
     <strong>
      委托构造函数（Delegating Constructor）
     </strong>
    </h4>
    <p>
     委托构造函数允许一个构造函数调用同一个类中的另一个构造函数，以避免代码重复。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName() : ClassName(default_value) {
    // 委托构造函数
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    int value;
    MyClass() : MyClass(0) { // 委托构造函数
        cout &lt;&lt; "Delegating Constructor Called!" &lt;&lt; endl;
    }
    MyClass(int v) : value(v) { // 参数化构造函数
        cout &lt;&lt; "Parameterized Constructor Called! Value: " &lt;&lt; value &lt;&lt; endl;
    }
};

int main() {
    MyClass obj; // 调用委托构造函数
    return 0;
}</code></pre>
    <h4>
     6.
     <strong>
      继承中的构造函数（Constructor in Inheritance）
     </strong>
    </h4>
    <p>
     在继承中，派生类的构造函数可以调用基类的构造函数来初始化基类部分。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">DerivedClass() : BaseClass() {
    // 派生类初始化代码
}</code></pre>
    <h5>
     示例：
    </h5>
    <pre><code class="language-cpp">class Base {
public:
    Base() {
        cout &lt;&lt; "Base Constructor Called!" &lt;&lt; endl;
    }
};

class Derived : public Base {
public:
    Derived() {
        cout &lt;&lt; "Derived Constructor Called!" &lt;&lt; endl;
    }
};

int main() {
    Derived obj; // 先调用基类构造函数，再调用派生类构造函数
    return 0;
}</code></pre>
    <h4>
     7.
     <strong>
      初始化列表构造函数（Constructor with Initializer List）
     </strong>
    </h4>
    <p>
     初始化列表构造函数用于在构造函数中直接初始化成员变量，而不是在构造函数体内赋值。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">ClassName() : member1(value1), member2(value2) {
    // 构造函数体
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    int a, b;
    MyClass(int x, int y) : a(x), b(y) { // 使用初始化列表
        cout &lt;&lt; "Initializer List Constructor Called! a: " &lt;&lt; a &lt;&lt; ", b: " &lt;&lt; b &lt;&lt; endl;
    }
};

int main() {
    MyClass obj(10, 20); // 调用初始化列表构造函数
    return 0;
}</code></pre>
    <h4>
     8.
     <strong>
      显式构造函数（Explicit Constructor）
     </strong>
    </h4>
    <p>
     显式构造函数用于防止隐式类型转换。通过
     <code>
      explicit
     </code>
     关键字修饰的构造函数只能显式调用。
    </p>
    <h5>
     语法：
    </h5>
    <pre><code class="language-cpp">explicit ClassName(parameters) {
    // 初始化代码
}</code></pre>
    <p>
     示例：
    </p>
    <pre><code class="language-cpp">class MyClass {
public:
    int value;
    explicit MyClass(int v) : value(v) {
        cout &lt;&lt; "Explicit Constructor Called! Value: " &lt;&lt; value &lt;&lt; endl;
    }
};

int main() {
    // MyClass obj = 10; // 错误：不能隐式转换
    MyClass obj(10); // 正确：显式调用构造函数
    return 0;
}</code></pre>
