# 全局引用scss文件定义的变量

在vite.config.ts文件中写如下代码

```javascript
css: {
   preprocessorOptions: {
        scss: {
            additionalData: `@use "@/assets/styles/index.scss";`,//这一句
            javascriptEnabled: true
        }
    }
}

```

在scss文件中定义变量

```javascript
$color:#000;

```

在其他文件引用变量

```javascript
<style lang="scss" scoped>

.text{
    color:index.$color;
}
</style>

```
