    <h2>
     <a id="_11">
     </a>
     前言
    </h2>
    <p>
     为了提升虚拟机性能以及增强虚拟机与宿主机之间交互，简化操作。
    </p>
    <ul>
     <li>
      VMware Tools 和 open-vm-tools 工具集应运而生。
     </li>
     <li>
      对于于大多数现代 Linux 发行版来说，直接通过包管理器安装
      <strong>
       open-vm-tools
      </strong>
      已成为推荐的做法。
     </li>
    </ul>
    <hr/>
    <h2>
     <a id="VMware_Tools__openvmtools__22">
     </a>
     一、VMware Tools 和 open-vm-tools 是什么
    </h2>
    <h3>
     <a id="1VMware_Tools_24">
     </a>
     1、VMware Tools
    </h3>
    <ul>
     <li>
      这是由 VMware
      <strong>
       官方
      </strong>
      提供的
      <strong>
       工具包
      </strong>
      ，旨在提高虚拟机的性能并改进虚拟机与宿主机之间的交互。
     </li>
     <li>
      包括了
      <strong>
       驱动程序 和 实用程序
      </strong>
      ，比如可以实现更高效的图形处理、时间同步、拖放文件传输等功能。
     </li>
    </ul>
    <h3>
     <a id="2openvmtools_28">
     </a>
     2、open-vm-tools
    </h3>
    <ul>
     <li>
      开源版本的 VMware Tools，由 VMware 贡献给开源社区的项目。
     </li>
     <li>
      包含了与 VMware Tools 类似的功能，但作为开源软件，它可以被更多的Linux发行版包含在它们的软件仓库中，方便用户安装和更新。
     </li>
     <li>
      自 VMware vSphere 5.5 开始，VMware
      <strong>
       官方推荐
      </strong>
      使用
      <strong>
       open-vm-tools
      </strong>
      版本，特别是对于 Linux 虚拟机而言。这代表了一个趋势，即推动更加开放和兼容的解决方案。
     </li>
    </ul>
    <h2>
     <a id="openvmtools_34">
     </a>
     二、推荐使用open-vm-tools（简单）
    </h2>
    <h3>
     <a id="1_36">
     </a>
     1、虚拟机安装
    </h3>
    <p>
     如果还没有安装虚拟机，推荐观看以下文章：
    </p>
    <ul>
     <li>
      <a href="https://blog.csdn.net/weixin_55599565/article/details/145824260">
       虚拟机 | Ubuntu 安装流程以及界面太小问题解决
      </a>
     </li>
     <li>
      <a href="https://blog.csdn.net/weixin_55599565/article/details/143690812">
       虚拟机 | centos Stream 9 安装流程以及finalshell连接
      </a>
     </li>
    </ul>
    <h3>
     <a id="2openvmtools_41">
     </a>
     2、open-vm-tools安装
    </h3>
    <p>
     下面，是基于Ubuntu操作系统进行演示：
    </p>
    <h4>
     <a id="1_sudo_su__44">
     </a>
     1、首先打开终端，输入 sudo su ，并输入超级用户密码。
    </h4>
    <pre><code class="prism language-shell"><span class="token function">sudo</span> <span class="token function">su</span> 
</code></pre>
    <ul>
     <li>
      进入超级用户（root用户）模式。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/df3fb7bab06f48c6ad34c53b35f5458a.png#pic_center"/>
    </p>
    <h4>
     <a id="2_Ubuntu_54">
     </a>
     2、 确保Ubuntu系统软件包为最新（重要）
    </h4>
    <ul>
     <li>
      安装open-vm-tools出现依赖安装失败问题，可能是apt-get不是最新版。
     </li>
     <li>
      在终端并执行以下命令：
     </li>
    </ul>
    <pre><code class="prism language-shell"><span class="token function">apt-get</span> update
</code></pre>
    <h4>
     <a id="3aptget_install_openvmtools_openvmtoolsdesktop_62">
     </a>
     3、执行：apt-get install open-vm-tools open-vm-tools-desktop
    </h4>
    <pre><code class="prism language-shell"><span class="token function">apt-get</span> <span class="token function">install</span> open-vm-tools open-vm-tools-desktop
</code></pre>
    <h4>
     <a id="_68">
     </a>
     介绍：
    </h4>
    <h5>
     <a id="openvmtools__69">
     </a>
     open-vm-tools 安装原因
    </h5>
    <ul>
     <li>
      尽管，大多数现代 Linux 发行版都直接在官方软件仓库中提供了 open-vm-tools 包，但是执行以下也无伤大雅。
     </li>
    </ul>
    <h5>
     <a id="openvmtoolsdesktop_72">
     </a>
     open-vm-tools-desktop安装原因
    </h5>
    <ul>
     <li>
      open-vm-tools-desktop 是在 open-vm-tools 基础上的扩展，专注于提供图形用户界面相关的增强功能。
     </li>
     <li>
      作用
      <ul>
       <li>
        剪贴板共享
       </li>
       <li>
        <strong>
         拖放文件传输
        </strong>
        （本文章使用的功能）
       </li>
       <li>
        显示器分辨率自动调整
       </li>
       <li>
        多监视器支持：支持在多个显示器间扩展虚拟机桌面。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9a5578ead50c4e8bac07c6817dd008de.png#pic_center"/>
    </p>
    <h4>
     <a id="_3__84">
     </a>
     步骤 3: 重启虚拟机（如果需要）
    </h4>
    <p>
     某些情况下，可能需要重启你的Ubuntu虚拟机以确保所有组件都能正确加载和运行。
    </p>
    <pre><code class="prism language-shell"> <span class="token function">reboot</span>
</code></pre>
    <h2>
     <a id="_91">
     </a>
     总结
    </h2>
    <p>
     在某些情况下，特别是对于服务器环境或无图形界面的虚拟机，可能并不需要这些额外的功能。因此，是否安装取决于你的具体需求。
