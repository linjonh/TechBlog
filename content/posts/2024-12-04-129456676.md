# vue前端对一列对象集合通过某一字段删除某一个元素对象

Vue.js 是一个前端框架，在处理对象集合时可以使用 JavaScript 原生的方法来进行删除元素。

假设你有一个对象数组
`data`
，需要删除其中一个元素的字段
`field`
的值等于
`value`
的对象。可以使用 JavaScript 的
`filter()`
方法来进行过滤，得到一个新的数组，再赋值给
`data`
即可。

在 Vue 中可以这么写:

```
this.data = this.data.filter(item => item.field !== value);

```

这样你就可以删除了一个符合条件的元素对象。
