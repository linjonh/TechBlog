  <div class="article_content clearfix" id="article_content">

    <h3>
     1、安装【微信开发者工具】
    </h3>
    <p>
     使用的开发平台为——微信开发者工具（
     <a class="link-info" href="https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html" rel="nofollow" title="点击此处链接">
      点击此处链接
     </a>
     ）
    </p>
    <p>
     依次点击【工具】-&gt;【下载】-&gt;【稳定版更新日志】，找到适合的版本，进行下载安装。
    </p>
    <p class="img-center">
     <img alt="" height="463" src="https://i-blog.csdnimg.cn/blog_migrate/82881ec13f6d8aab3be41c016fdf67ca.png" width="800"/>
    </p>
    <h3>
     2、注册微信小程序
    </h3>
    <p>
     注册微信小程序（
     <a class="link-info" href="https://mp.weixin.qq.com/" rel="nofollow" title="点击注册链接">
      点击注册链接
     </a>
     ），注意注册的是”小程序“。
    </p>
    <p>
     根据注册提示完成后，进入以下界面：
    </p>
    <p class="img-center">
     <img alt="" height="416" src="https://i-blog.csdnimg.cn/blog_migrate/593cbc534552afcb3287efa2ca5f751f.png" width="800"/>
    </p>
    <p>
     在侧边栏点击【开发】--&gt;【开发管理】--&gt;【开发设置】
    </p>
    <p>
     <span style="color:#fe2c24;">
      获取AppID（小程序ID）
     </span>
     ，后期创建 小程序时会用到。
    </p>
    <p class="img-center">
     <img alt="" height="410" src="https://i-blog.csdnimg.cn/blog_migrate/b301e6e59e4b00e8c5ddb21603955217.png" width="800"/>
    </p>
    <h3>
     3、创建【小程序】
    </h3>
    <p>
     打开安装好的微信开发者工具，点击【小程序】，进入【创建小程序】界面。
    </p>
    <p>
     此时将步骤2，所得到的AppID,复制到对应的位置即可。默认选择微信云开发。
    </p>
    <p class="img-center">
     <img alt="" height="446" src="https://i-blog.csdnimg.cn/blog_migrate/8604db54c08db8f76593478b02a444cb.png" width="600"/>
    </p>
    <h3>
     4、搭建node.js环境
    </h3>
    <ul>
     <li>
      <h4>
       下载node.js地址:
       <a href="https://nodejs.org/en" rel="nofollow" title="节点.js (nodejs.org)">
        节点.js (nodejs.org)
       </a>
      </h4>
     </li>
    </ul>
    <p>
     点击【Other Downloads】--&gt;【Windows Binary(.zip)】--&gt;【64-bit】
    </p>
    <p class="img-center">
     <img alt="" height="271" src="https://i-blog.csdnimg.cn/blog_migrate/7664c3e49f972b509259c3a1b0129296.png" width="600"/>
    </p>
    <p class="img-center">
     <img alt="" height="328" src="https://i-blog.csdnimg.cn/blog_migrate/e0e1cbcf9c7770811031c7641a863f13.png" width="600"/>
    </p>
    <p>
     解压后，放在路径 D:\SoftSetup\nodejs 下
    </p>
    <ul>
     <li>
      <h4>
       配置环境变量
      </h4>
     </li>
    </ul>
    <p>
     由【此电脑】--&gt;【高级系统设置】--&gt;【系统属性】--&gt;【环境变量】
    </p>
    <p>
     在
     <span style="color:#ff9900;">
      用户变量
     </span>
     处，新建 变量名
     <span style="background-color:#a2e043;">
      NODE_PATH
     </span>
     , 变量值为上面
     <span style="background-color:#a2e043;">
      nodejs的路径
     </span>
     。（建议直接点击【浏览目录】，找到对应文件夹，即可获取路径）
    </p>
    <p class="img-center">
     <img alt="" height="104" src="https://i-blog.csdnimg.cn/blog_migrate/f59c447ddcc30468bcba3200da163158.png" width="400"/>
    </p>
    <p>
     点击用户变量中的变量【PATH】，新建 %NODE_PATH%
    </p>
    <p class="img-center">
     <img alt="" height="157" src="https://i-blog.csdnimg.cn/blog_migrate/d0139220574d15c71857369423c5ec8b.png" width="400"/>
    </p>
    <p>
     在【系统变量】处，点击变量【PATH】，新建 %NODE_PATH%
    </p>
    <p class="img-center">
     <img alt="" height="187" src="https://i-blog.csdnimg.cn/blog_migrate/d5953af4ebc32455aacec4f9040372a8.png" width="400"/>
    </p>
    <ul>
     <li style="margin-left:0px;">
      <h4>
       <strong>
        检查是否安装成功
       </strong>
      </h4>
     </li>
     <li style="margin-left:0px;">
      <span style="color:#4d4d4d;">
       <span style="background-color:#ffffff;">
        win+r ,输入cmd,回车
       </span>
      </span>
     </li>
     <li style="margin-left:0px;">
      <p class="img-center">
       <img alt="" height="106" src="https://i-blog.csdnimg.cn/blog_migrate/84c972751285ce9aadbb8bf9fe2f9b11.png" width="200"/>
      </p>
     </li>
     <li style="margin-left:0px;">
      <span style="color:#4d4d4d;">
       <span style="background-color:#ffffff;">
        进入在命令提示符中输入：node -v
       </span>
      </span>
      <span style="color:#4d4d4d;">
       <span style="background-color:#ffffff;">
        ,如图所示node环境安装成功
       </span>
      </span>
     </li>
     <li style="margin-left:0px;">
      <p class="img-center">
       <img alt="" height="218" src="https://i-blog.csdnimg.cn/blog_migrate/2738c1b6b0012d6aa9d2875bf9774330.png" width="350"/>
      </p>
      <h3 style="background-color:transparent;">
       5、npm配置
      </h3>
     </li>
     <li style="margin-left:0px;">
      npm 是Nodejs下的包管理器，安装完node.js后npm的本地仓库默认会在C盘。如图
      <ul>
       <li style="margin-left:0px;">
        <p class="img-center">
         <img alt="" height="432" src="https://i-blog.csdnimg.cn/blog_migrate/1b16bb035ffc1310498bcbc13b0c13cb.png" width="600"/>
        </p>
        我们配置npm的全局模块的存放路径以及cache的路径。我是放在了node.js 的安装路径。在D:\SoftSetup\nodejs 文件夹内新建两个文件夹，分别为node_cache和node_global
       </li>
      </ul>
     </li>
     <li style="margin-left:0px;">
      <p class="img-center">
       <img alt="" height="193" src="https://i-blog.csdnimg.cn/blog_migrate/9005dc01ba2d96cc32bccc40a49ca290.png" width="600"/>
      </p>
      修改npm的全局目录路径和缓存目录，将对应的模块目录改到D盘的nodejs的安装目录。
     </li>
     <li style="margin-left:0px;">
      <pre><code class="language-cs">#全局安装目录
npm config set prefix "D:\SoftSetup\nodejs\node_global"</code></pre>
      <pre><code class="language-cs">#缓存目录
npm config set cache "D:\SoftSetup\nodejs\node_cache"</code></pre>
      <p>
       配置环境变量：
      </p>
     </li>
     <li style="margin-left:0px;">
      <p>
       由【此电脑】--&gt;【高级系统设置】--&gt;【系统属性】--&gt;【环境变量】
      </p>
      <ul>
       <li style="margin-left:0px;">
        <p>
         在系统环境变量中，新建变量名： NODE_PATH ,变量值：D:\SoftSetup\nodejs\node_global\node_modules
        </p>
       </li>
       <li style="margin-left:0px;">
        <p class="img-center">
         <img alt="" height="155" src="https://i-blog.csdnimg.cn/blog_migrate/4e20bd0b941938ac304adc31f9ced71c.png" width="600"/>
        </p>
        <p>
         在用户环境变量中新建或修改，默认的
         <code>
          C
         </code>
         盘下的  C:\Users\Paterson\AppData\Local\npm为D:\SoftSetup\nodejs\node_global
        </p>
       </li>
       <li style="margin-left:0px;">
        <p class="img-center">
         <img alt="" height="282" src="https://i-blog.csdnimg.cn/blog_migrate/2a718bdccd364653f9fe9a6200f4844b.png" width="600"/>
        </p>
        <p>
         更换镜像源：
        </p>
       </li>
       <li style="margin-left:0px;">
        <pre><code class="language-cs">#全局使用淘宝镜像源
npm config set registry https://registry.npm.taobao.org

#查看当前镜像源
npm config get registry</code></pre>
        <p class="img-center">
         <img alt="" height="244" src="https://i-blog.csdnimg.cn/blog_migrate/0b3d4e365366cdc7227e6dd9afef7eb1.png" width="500"/>
        </p>
        <p>
         6、npm配置vant weapp框架
        </p>
       </li>
       <li style="margin-left:0px;">
        <p>
         打开步骤3，新建的项目，从小程序文件的根目录打开终端窗口
        </p>
       </li>
       <li style="margin-left:0px;">
        <p>
         <img alt="" height="493" src="https://i-blog.csdnimg.cn/blog_migrate/12eb1bd18947ed0ed456360f2a63cc3c.png" width="800"/>
        </p>
        <p>
         通过 npm 安装，一定要在项目的根目录下进行。
        </p>
       </li>
       <li style="margin-left:0px;">
        <p>
         比如我新建的小程序项目1就存放在SmaProgram文件夹内，所以Smagram就是我的根目录
        </p>
       </li>
       <li style="margin-left:0px;">
        <pre><code class="language-cs">#安装 Vant Weapp 组件库
npm i @vant/weapp -S --production

#初始化项目，
npm init -y</code></pre>
        <p>
         生成了node_modules文件夹，package-lock.json和package.json文件。
        </p>
       </li>
       <li style="margin-left:0px;">
        <p>
        </p>
        <p>
         <img alt="" height="493" src="https://i-blog.csdnimg.cn/blog_migrate/2767554c2fd2f861b5568774227d9a42.png" width="800"/>
        </p>
        <p>
         以下是我终端运行过程：
        </p>
       </li>
      </ul>
     </li>

     <li style="margin-left:0px;">
      <p>

      <p>
       构建npm,【工具】-&gt;【构建npm】
