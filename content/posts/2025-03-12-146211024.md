     Git使用（一）--如何在 Windows 上安装 Git：详细步骤指南
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>

    <p>
     如果你想在 Windows 机器上安装 Git，可以按照以下详细指南进行操作。
    </p>
    <h4 id="ktPB7">
     第一步：下载 Git
    </h4>
    <p id="u5c041197">
     可通过官网下载 适用于 Windows 的 Git 最新版本。 如果下载速度较慢，可以通过下面提供的
     <strong>
      百度网盘
     </strong>
     链接下载安装包，
     <br/>
     <a href="https://git-scm.com/downloads" rel="nofollow" title="https://git-scm.com/download">
      https://git-scm.com/download
     </a>
    </p>
    <p>
     <img alt="" class="left" height="836" id="ue8f4b5ad" src="https://i-blog.csdnimg.cn/img_convert/4b1625a4553481433604acd46c9a1dd7.png" width="1028"/>
    </p>
    <p id="u47fb3dd2">
     下载最新版
    </p>
    <p id="ufdaa3cf1">
    </p>
    <p>
     <img alt="" class="left" height="612" id="u0f650d90" src="https://i-blog.csdnimg.cn/img_convert/ba57eb91caa130c8ddd43d60d0ea04d8.png" width="975"/>
    </p>
    <p id="u4acab381">
     链接:
     <a href="https://pan.baidu.com/s/18NgWZS0ypq3lH-uQkawxPg?pwd=tx8a" rel="nofollow" title="百度网盘 请输入提取码">
      百度网盘 请输入提取码
     </a>
     提取码: tx8a
    </p>
    <h4 id="lURNz">
     第二步：开始安装
    </h4>
    <p id="uaddf1f27">
     下载完 Git 安装包后，打开可执行文件，开始安装过程。
    </p>
    <p id="u1b5b5931">
    </p>
    <p>
     <img alt="" class="left" height="387" id="u4c884acf" src="https://i-blog.csdnimg.cn/img_convert/9ab96b865114659e89e09bcbb25e090a.png" width="496"/>
    </p>
    <p id="ue051ae3e">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u7d65b7ac" src="https://i-blog.csdnimg.cn/img_convert/289cb18b7f268f9b3193aee9003d5847.png" width="509"/>
    </p>
    <h4 id="dFqXo">
     第三步：选择安装组件
    </h4>
    <p id="u3a40692e">
     <strong>
      选择默认 Git 编辑器
     </strong>
     ：
    </p>
    <ul>
     <li id="u2f247818">
      安装向导会要求你选择默认的 Git 编辑器。你可以选择
      <strong>
       Vim
      </strong>
      （功能强大但不太容易使用），
     </li>
    </ul>
    <p id="ud95b96b8">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u5be209b9" src="https://i-blog.csdnimg.cn/img_convert/90757e7c514d4456dc94bf98472839ae.png" width="509"/>
    </p>
    <p id="u4c6f1f87">
     <strong>
      也可以选择其他编辑器，比如
     </strong>
     <strong>
      Notepad++
     </strong>
     <strong>
      或
     </strong>
     <strong>
      VSCode
     </strong>
     <strong>
      。你可以点击
     </strong>
     <code>
      <strong>
       Browse
      </strong>
     </code>
     <strong>
      按钮，选择自己喜欢的编辑器的
     </strong>
     <code>
      <strong>
       .exe
      </strong>
     </code>
     <strong>
      文件。
     </strong>
    </p>
    <p id="u47444240">
    </p>
    <p>
     <img alt="" class="left" height="397" id="udef4741a" src="https://i-blog.csdnimg.cn/img_convert/3acfd06176fe68cd724412cae92d4116.png" width="509"/>
    </p>
    <ul>
     <li id="ud98c7e6e">
      我选择了
      <strong>
       VSCode
      </strong>
      作为默认编辑器。
     </li>
    </ul>
    <p id="u2787fd95">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u60b04e79" src="https://i-blog.csdnimg.cn/img_convert/2310f93ee150dc97168c3686a60cfa00.png" width="509"/>
    </p>
    <h4 id="nBXxb">
     第四步：调整初始化项目的名字
    </h4>
    <p id="u59c28876">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u2ebc5e81" src="https://i-blog.csdnimg.cn/img_convert/4c798511fe9ff61927c6c38003645626.png" width="509"/>
    </p>
    <p id="u8a21a138">
     默认是“master”，如果想修改分支名称，点击第二个选项，在“main”的位置修改即可
    </p>
    <h4 id="L4Q1N">
     第五步：配置 PATH 环境变量
    </h4>
    <p id="u9913e8a6">
     你可以配置如何通过命令行访问 Git。
    </p>
    <p id="ufb9274d8">
    </p>
    <p>
     <img alt="" class="left" height="397" id="ua08b88e9" src="https://i-blog.csdnimg.cn/img_convert/14492dd086a9b635a0449f5c6ba60e1e.png" width="509"/>
    </p>
    <ul>
     <li id="u6b93e23d">
      <strong>
       选项一
      </strong>
      ： "仅通过 Git Bash 使用 Git" — 这种方式不会更改你的 PATH 环境变量，意味着你只能通过 Git Bash 终端来访问 Git。
     </li>
     <li id="u07a86f17">
      <strong>
       选项二
      </strong>
      ： "通过命令提示符和第三方软件使用 Git" — 这是推荐的选项。它允许你通过 Git Bash、命令提示符、PowerShell，以及任何可能需要 Git 的第三方软件来访问 Git。
     </li>
     <li id="u96b1b2db">
      <strong>
       选项三
      </strong>
      ： "通过命令提示符使用 Git 和可选的 Unix 工具" — 这个选项会覆盖 Windows 的某些命令，如
      <code>
       find
      </code>
      和
      <code>
       sort
      </code>
      ，仅当你了解其影响时再使用。
     </li>
    </ul>
    <p id="u86f76e07">
     大多数情况下，选择
     <strong>
      第二个选项
     </strong>
     是最合适的。
    </p>
    <h4 id="iB5n9">
     第六步：选择 SSH 执行文件
    </h4>
    <p id="ueb38acb5">
     这一页面让你选择 SSH 相关的配置，通常使用默认选项即可。
    </p>
    <p id="u5ec821ac">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u1c15b6ec" src="https://i-blog.csdnimg.cn/img_convert/fe10c39bd401172a40eca654392c82a6.png" width="509"/>
    </p>
    <h4 id="LnHgB">
     第七步：选择 HTTPS 后端传输
    </h4>
    <p id="ub1012188">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u9dfea604" src="https://i-blog.csdnimg.cn/img_convert/1918b783d733f97bd68fe326dcb2bc70.png" width="509"/>
    </p>
    <p id="ue82ef369">
     在这一界面，你可以选择
     <strong>
      OpenSSL 库
     </strong>
     或
     <strong>
      Windows 安全通道库
     </strong>
     。推荐选择
     <strong>
      OpenSSL 库
     </strong>
     ，它使用
     <code>
      ca-bundle.crt
     </code>
     文件来验证服务器证书。
    </p>
    <h4 id="hOCij">
     第八步：配置行尾符号转换
    </h4>
    <p id="uac23da83">
     Git 会根据你的操作系统自动处理行尾符号转换。一般情况下，选择默认选项即可。
    </p>
    <p id="u486341e5">
    </p>
    <p>
     <img alt="" class="left" height="397" id="ufb283009" src="https://i-blog.csdnimg.cn/img_convert/f2a2ef27607c4f9c49d97a364122c113.png" width="509"/>
    </p>
    <h4 id="U6LJO">
     第九步：配置终端模拟器
    </h4>
    <p id="u47719dd3">
     在这一部分，你可以配置 Git Bash 使用的终端模拟器。默认选项通常是合适的，建议保留默认设置。
    </p>
    <p id="u8fae674e">
    </p>
    <p>
     <img alt="" class="left" height="397" id="uf0eb3e14" src="https://i-blog.csdnimg.cn/img_convert/fef9c26af31c74197c8b55b7363e110b.png" width="509"/>
    </p>
    <h4 id="PKrNj">
     第十步：选择默认的
     <code>
      git pull
     </code>
     行为
    </h4>
    <p id="u09b2e31b">
     此界面让你配置默认的
     <code>
      git pull
     </code>
     行为，默认设置通常是合适的，可以直接点击
     <strong>
      Next
     </strong>
     继续。
    </p>
    <p id="u12db75ef">
    </p>
    <p>
     <img alt="" class="left" height="585" id="u6977b6a9" src="https://i-blog.csdnimg.cn/img_convert/9253923ecf5091e305c1c88e2f07f45a.png" width="759"/>
    </p>
    <h4 id="1edbe9ff">
     第十一步：配置额外的选项
    </h4>
    <p id="uf918fc65">
     在这一页面，你可以选择是否启用额外的 Git 配置项。通常情况下，选择默认选项即可。
    </p>
    <p id="uc50503bd">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u8ce6a579" src="https://i-blog.csdnimg.cn/img_convert/3581c6e3783829b558d72fd47a59a274.png" width="509"/>
    </p>
    <h4 id="552cd15a">
     第十二步：配置实验性选项
    </h4>
    <p id="uaed7b763">
     这里的实验性选项不需要启用，直接选择
     <strong>
      Install
     </strong>
     按钮开始安装即可。
    </p>
    <p id="u1bc9fea0">
    </p>
    <p>
     <img alt="" class="left" height="397" id="u3f2db223" src="https://i-blog.csdnimg.cn/img_convert/c9e432117d75fd2af62a9d05f0ce70c3.png" width="509"/>
    </p>
    <h4 id="8bd39e4e">
     安装完成
    </h4>
    <p id="u62953c39">
     最后，点击
     <strong>
      Install
     </strong>
     按钮开始安装。安装过程将自动进行，完成后你可以选择
     <strong>
      Finish
     </strong>
     退出安装向导。
    </p>
    <p id="ua9359b08">
    </p>
    <p class="img-center">
     <img alt="" height="616" id="u81bd901e" src="https://i-blog.csdnimg.cn/img_convert/59fa0299f72368fa48cade8b0b7ce9c4.png" width="802"/>
    </p>
    <h4 id="SNsSd">
     验证是否安装成功
    </h4>
    <p id="u73a75299">
     cmd输入“git --version”，出现版本信息即安装成功。
    </p>
    <p id="ue1f885fc">
    </p>
    <p>
     <img alt="" class="left" height="65" id="uec69185f" src="https://i-blog.csdnimg.cn/img_convert/21dd3abc26a8fbf719071e7f6655408f.png" width="355"/>
    </p>
