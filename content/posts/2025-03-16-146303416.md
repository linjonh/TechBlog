# 【AWS入门】AWS云计算简介

## 【AWS入门】AWS云计算简介

#### A Brief Introduction to AWS Cloud Computing

By Jackson@ML

什么是云计算？云计算能干什么？我们如何利用云计算？云计算如何实现？

带着一系列问题，我将做一个普通布道者，引领广大读者，针对AWS各项云技术和服务，逐步进入云计算赋能千行百业的神奇世界。

本文简要介绍AWS提供的云计算，希望对读者有所帮助。

#### 1\. 云计算

云计算通过互联网**按需提供 IT 资源** ，并且采用**按使用量付费** 的定价方式。

您可以根据需要从诸如 Amazon Web Services (AWS)
之类的云提供商那里获得技术服务，例如计算能力、存储和数据库，而无需购买、拥有和维护物理数据中心及服务器。

#### 2\. AWS简介

**Amazon Web Service** ，即**AWS** ，中文称为“**亚马逊云科技**
”；AWS是亚马逊在全球范围内开发、运营和推广云计算及其服务的子公司。以下是AWS官网页面。

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/e381acb8bff04f65b1402b579e3f416c.png)

AWS云，于2006年正式推出，是市场领先的云服务提供商，为用户提供各种基础设施服务，例如：
_**计算、存储、网络互联、AI、数据库、机器学习、无服务器部署**_
等，这些广泛的服务基于强大的平台能力。这个AWS平台具有分布式部署、移动业务、数据分析和机器学习等。

AWS的每项服务都遵循按需可用、**按需付费(pay-as-you-go)**的原则。

AWS可以被看作是可编程的数据中心。尽管无需打电话或者发Email到特定服务器或其它资源，但用户可以通过应用程序编程接口 **(Application
Programming Interface , 即API)** 以编程来实现资源管控。

AWS在全世界范围内运营很多数据中心，因此，用户可以无限制地、不间断地访问这些数据中心。

#### 3\. 云服务提供商

**Cloud Service Provider（CSP）** ，即**云服务提供商** 。云服务提供商是通过远程网络连接为用户提供按需计算资源的公司。

计算资源可以是数据存储、计算能力、应用程序、基础设施和平台等。一些 CSP
通过集成专有和开源工具以及定制服务来扩展其产品，以满足现代初创公司、SMB(小型商业企业） 和企业不断增长的需求。

可用选项的广度确实令人印象深刻。然而，浏览这些不同的选择以找到定价、性能、服务、灵活性、可扩展性等方面的理想选择是具有挑战性的。

#### 4\. 全球云平台服务商

云服务提供商通常都需要构建庞大的、分布在全球范围的数据中心以及相应的很多基础设施，各项服务运营在统一平台上。因此，云服务提供商也叫做云平台服务提供商。

据权威技术媒体geeksforgeeks报道，截止2024年11月，全球TOP 10 Cloud Platform Service
Providers(即全球云平台服务提供商前10名)分别为：

  1. Amazon Web Services (AWS)
  2. Microsoft Azure
  3. Kamatera
  4. Alibaba Cloud
  5. Oracle Cloud
  6. IBM Cloud (Kyndryl)
  7. Tencent Cloud
  8. OVHcloud
  9. DigitalOcean
  10. Linode (owned by Akamai)

AWS凭借雄厚的技术实力，以及全球最广泛的计算部署，连续成为市场的“领头羊”，也是全球最佳的云服务提供商。

AWS应用程序具有可扩展性、灵活性、可靠性、安全性和值得信赖的特点。

#### 5\. 使用AWS云服务的企业

全球范围内有很多知名公司持续使用AWS云服务来为企业赋能和高效运营，典型的有以下公司：

  * Netflix
  * Spotify
  * Airbnb
  * Uber
  * Peloton
  * Expedia
  * Pinterest
  * Samsung
  * Sony
  * Novartis

#### 6\. AWS用户注册

新用户可以是个人，也可以是企业。

AWS用户注册可以在官网链接： <https://signin.aws.amazon.com/> 完成创建用户的工作。

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/d7be0108a2a746b2ae40725fb4005e49.png)

由于AWS提供的云服务有国内和国际两种账户，因此，用户可根据需要，注册AWS中国区域账户或者全球账户。

> * **注册AWS账户** ，请参看文章：  
>  [【AWS入门】2025
> AWS亚马逊云科技账户注册指南](https://jackson.blog.csdn.net/article/details/146252853?spm=1001.2014.3001.5502)，此处不再赘述。

AWS中国区域在创建时有两个**可用区（Availability Zone, 简称AZ）** ，即**北京** 和**宁夏**
可用区，分别由光环新网和西云科技运营。

2021年6月8日，AWS亚马逊云科技在北京新增第三个可用区。

AWS每个可用区都由一个或者多个独立的数据中心组成。每个数据中心都配备冗余的电力、网络设备和连接，确保给用户提供稳定可靠的云服务。
