    <h2>
     <a id="2962__K__0">
     </a>
     2962. 统计最大元素出现至少 K 次的子数组
    </h2>
    <blockquote>
     <p>
      给你一个整数数组 nums 和一个 正整数 k 。
      <br/>
      请你统计有多少满足 「 nums 中的 最大 元素」至少出现 k 次的子数组，并返回满足这一条件的子数组的数目。
      <br/>
      子数组是数组中的一个连续元素序列。
      <br/>
      示例 1：
      <br/>
      输入：nums = [1,3,2,3,3], k = 2
      <br/>
      输出：6
      <br/>
      解释：包含元素 3 至少 2 次的子数组为：[1,3,2,3]、[1,3,2,3,3]、[3,2,3]、[3,2,3,3]、[2,3,3] 和 [3,3] 。
      <br/>
      示例 2：
      <br/>
      输入：nums = [1,4,2,1], k = 3
      <br/>
      输出：0
      <br/>
      解释：没有子数组包含元素 4 至少 3 次。
      <br/>
      提示：
      <br/>
      1 &lt;= nums.length &lt;= 105
      <br/>
      1 &lt;= nums[i] &lt;= 106
      <br/>
      1 &lt;= k &lt;= 105
     </p>
    </blockquote>
    <h3>
     <a id="_17">
     </a>
     题解
    </h3>
    <p>
     如标题所示，本题采用
     <strong>
      滑动数组
     </strong>
     进行解题
    </p>
    <p>
     题目要求
     <strong>
      所有
     </strong>
     满足条件的子数组
     <br/>
     那么我们自然需要考虑所有的子数组
     <br/>
     我们该如何做到呢
    </p>
    <p>
     首先看看如何枚举所有的子数组
     <br/>
     我们可以用一个循环枚举出所有子数组可能的开头，然后内层再写一个循环枚举所有可能的结尾，这样就枚举了所有的子数组
    </p>
    <p>
     那么
     <strong>
      滑动数组
     </strong>
     又该如何考虑到所有的
     <strong>
      子数组
     </strong>
     呢？
    </p>
    <hr/>
    <p>
     类似的，我们可以写一个循环枚举出所有
     <strong>
      子数组的结尾
     </strong>
     i
     <br/>
     然后使用指针 l=0 作为
     <strong>
      子数组的开头
     </strong>
     ，那么
     <strong>
      l 与 i
     </strong>
     就是滑动窗口的区间
     <br/>
     我们使用变量 count 记录滑动窗口中的最大值的个数，res=0 记录返回值
     <br/>
     此时 i 作为窗口的右边不断右移
     <br/>
     当 count == k 时，此时的滑动窗口满足条件，我们找到了一个答案
    </p>
    <p>
     但是问题是，接下来我们要如移动滑动窗口呢？
     <br/>
     将 i 向右移，还是将 l 向右移
    </p>
    <p>
     注意到，我们第一层循环是枚举所有子数组的结尾
     <br/>
     只要对于每一种结尾，我们都找到所有可能的子数组就能解决问题
     <br/>
     但是我们显然不能枚举开头 l ，否则与枚举就一样了，时间复杂高
     <br/>
     当 count == k 时，我们可以将 窗口左边 l 右移，直到 count != k
     <br/>
     那么对于此时的 i ，此时所有以 l 的左边为开头的子数组 [ l, i ] 都是满足条件的
     <br/>
     也就是我们找到了以 i 结尾的所有满足条件的子数组
     <br/>
     所以滑动窗口 [ l, i ] 的含义就是以 i 结尾的，第一个不满足条件的子数组
     <br/>
     res+=l ，枚举下一个 i
     <br/>
     如果 count != k，那么 l 的位置不变，res+=l
     <br/>
     如果count==k，那么接着移动 l 直到滑动窗口不满足条件
    </p>
    <p>
     总计滑动窗口划过一次数组，时间复杂度为 O(n)
    </p>
