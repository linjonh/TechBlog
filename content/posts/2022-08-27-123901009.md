    <h2>
     <strong>
      〖猜数字游戏〗
     </strong>
    </h2>
    <p>
     <strong>
      使程序随机生成一个1~100之间的数字，并让用户输入一个数字（10次输入机会），判断这两个数的大小，如果用户输入的数字大于随机数，那么提示“你猜大了”，如果用户输入的数字小于随机数，则提示“你猜小了”，如果两个数字相等，就提示“恭喜你，猜对了”，结束程序。
     </strong>
    </p>
    <p>
     <strong>
      代码如下:
     </strong>
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
 &lt;head&gt;
  &lt;meta charset="utf-8"&gt;
  &lt;title&gt;猜数字随机数1-100&lt;/title&gt;
 &lt;/head&gt;
 &lt;body&gt;
  &lt;script type="text/javascript"&gt;
   function getRandom(min,max){
    return Math.floor(Math.random() * (max-min+1))+min;
   }
   var num = getRandom(1,100);
   var count = 10;
   var finish = false;
   while(!finish){
	if(count == 0){alert('次数已经用完了');break;}
	var postnum = prompt('猜一个1~100的数字');
	count--;
    if(postnum &gt; num){
     alert('你猜大了，还有'+ count +'次机会');
     
    }
    else if(postnum &lt; num){
     alert('你猜小了，还有'+ count +'次机会');
    }
	else{
		alert('恭喜你，你猜对了');
		finish = true;
	}
   }
  &lt;/script&gt;
 &lt;/body&gt;
&lt;/html&gt;</code></pre>
