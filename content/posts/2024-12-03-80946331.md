## Linux(Ubuntu)不好用吗？是因为你没有安装这些软件

编程之路在何方

2018-07-05 18:00:00

本文介绍了Ubuntu常用的一些软件，包含办公、视频、音乐、图像处理、游戏、编程语言、数据库、开发工具、编辑器等，建议收藏，如果你身边有正在学Linux的朋友建议转发给他。

## 办公

**WPS Office**

可以直接到官网下载然后使用下面的命令安装

```
sudo dpkg -i 你下载的deb文件的路径
```

**LibreOffce**

一款开源的Office套件，支持Windows，Linux和MacOSX，在Ubuntu下一般内置该软件

**Xmind**

一款非常好用的思维导图工具，支持Windows，Linux和MacOSX, 可以直接在官网下载

## 视频和音乐

**网易云音乐**

这应该是国内唯一一款Linux下的音乐软件了，直接在官网下载，使用dpkg命令安装

**VLC**

网易云音乐也是用的vlc的播放能力，如果网易云音乐安装成功以后vlc会自动安装。VLC是博主一直在使用的本地视频播放软件，支持Windows，Linux，Mac OSX，Android和IOS，在Ubuntu下可以使用下面的命令安装

```
sudo apt install vlc
```

**Pitivi**

Pitivi是一款Linux下的视频剪辑软件，在Ubuntu下的安装命令如下

```
sudo apt install pitivi
```

## 游戏

**我的世界**

我的世界网易代理以后只开发了Windows，Android和ios版本，正版是支持Linux，Mac OSX，Windows，Android和ios的。在Linux下可以下载Hello Minecraft! Launcher

**Steam**

Steam是一个游戏和软件分发平台，近两年比较火的绝地求生就是在这里分发的，不过只支持Windows，Steam支持Windows，Linux，和Mac OSX，在Ubuntu下可以使用下面的命令安装

```
sudo apt install steam
```

**Dota2**

安装Steam后在Steam中安装

**CS:GO**

安装Steam后在Steam中安装

## 图像处理

**GIMP**

Linux下没有Photoshop，不过可以安装GIMP替代Photoshop，虽然功能没有Photoshop强大，但是GIMP免费，且支持Windows，Linux和Mac OSX，在ubuntu下使用下面的命令安装

```
sudo apt install gimp
```

**Shutter**

shutter是一款非常强大的截图功能，在Ubuntu下使用下面的命令安装

```
sudo apt install shutter
```

**OBS**

obs是一个跨平如的录屏，直播软件，支持Windows，Linux和Mac OSX，在Ubuntu下的安装方法请参考我之前的文章

**Inkscape**

Inkscape是一款矢量图形编辑工具，可以生成svg图形，支持Windows，Linux和Mac OSX，在Ubuntu下使用下面的命令安装

```
sudo apt install inkscape
```

**Blender**

Blender是一款跨平如的3D建模软件支持Windows，Linux和Mac OSX，在Ubuntu下使用下面的命令安装

```
sudo apt install blender
```

## 编程语言

编辑语言我就不用介绍了，只说安装方法

**PHP**

php-后面接php的扩展名可以安装对应的扩展

```
sudo apt install php php-mysqli php-json php-gd php-xml php-cli php-fpm
```

**OpenJDK**

```
sudo apt install openjdk-8-jdk
```

**OracleJDK**

请参考我之前的文章

**Kotlin**

kotlin最终还是在jvm上运行，安装jdk后用idea可以创建Kotlin项目，用Android Studio 可以创建Kotlin语言的Andoird软件项目，更多信息请移步官网

**Python**

Ubuntu自带， 不用安装

**NodeJS**

可以到官网上下载， 不过官网比较慢，可以使用淘宝Node镜像

**GCC**

Ubuntu自带， 不用安装

**G++**

Ubuntu自带， 不用安装

**Swift**

到官网下载安装

**.Net**

```
sudo apt install mono-devel
```

**Ruby**

```
sudo apt install ruby
```
