    <h2>
     3、scanf函数
    </h2>
    <blockquote>
     <pre><code class="hljs">#include &lt;stdio.h&gt;


int main()

{
  int score = 0;
  printf("请输入文字");
  scanf("%d", &amp;score);
  printf("成绩为:%d\n", score);
  return 0;

}</code></pre>
     <p>
      <code>
       &amp;
      </code>
      符号是
      <strong>
       取地址运算符
      </strong>
      ，它的作用是获取变量在内存中的存储地址。
     </p>
     <p>
      <code>
       scanf
      </code>
      函数需要知道变量在内存中的具体位置，才能将用户输入的值写入对应的内存空间
     </p>
     <h4>
      <strong>
       1. 变量名的本质
      </strong>
     </h4>
     <p>
      当你声明
      <code>
       int score = 0;
      </code>
      时，编译器会做两件事：
     </p>
     <ol>
      <li>
       ​
       <strong>
        分配内存
       </strong>
       ：为
       <code>
        score
       </code>
       分配一块内存空间（假设地址是
       <code>
        0x1000
       </code>
       ）。
      </li>
      <li>
       ​
       <strong>
        绑定名称
       </strong>
       ：将变量名
       <code>
        score
       </code>
       与地址
       <code>
        0x1000
       </code>
       关联起来，后续所有对
       <code>
        score
       </code>
       的操作（读取或写入）都通过这个地址完成。
      </li>
     </ol>
     <hr/>
     <h4>
      ​
      <strong>
       2. 直接赋值时发生了什么？
      </strong>
     </h4>
     <p>
      当执行
      <code>
       score = 10;
      </code>
      时：
     </p>
     <ul>
      <li>
       ​
       <strong>
        编译器自动翻译
       </strong>
       ：编译器知道
       <code>
        score
       </code>
       的内存地址是
       <code>
        0x1000
       </code>
       ，所以直接生成机器指令：​
       <strong>
        将值
        <code>
         10
        </code>
        写入地址
        <code>
         0x1000
        </code>
       </strong>
       。
      </li>
      <li>
       ​
       <strong>
        无需手动操作地址
       </strong>
       ：变量名
       <code>
        score
       </code>
       已经隐含了地址信息，因此不需要用
       <code>
        &amp;
       </code>
       去取地址。
      </li>
     </ul>
     <hr/>
     <h4>
      ​
      <strong>
       3. 为什么
       <code>
        scanf
       </code>
       需要
       <code>
        &amp;
       </code>
       ？
      </strong>
     </h4>
     <p>
      <code>
       scanf
      </code>
      是一个函数，它的工作机制不同：
     </p>
     <ul>
      <li>
       ​
       <strong>
        函数参数传递
       </strong>
       ：C语言默认是“值传递”，即函数接收的是参数的
       <strong>
        副本
       </strong>
       ，而非原始数据。
      </li>
      <li>
       ​
       <strong>
        必须显式传递地址
       </strong>
       ：要让
       <code>
        scanf
       </code>
       修改外部变量
       <code>
        score
       </code>
       ，必须告诉它
       <code>
        score
       </code>
       的地址（
       <code>
        &amp;score
       </code>
       ），否则它只能操作一个临时的副本。
      </li>
     </ul>
     <p>
      简单理解：形参要想改变实参的值，必须加上&amp;
     </p>
    </blockquote>

    <h2>
     6、指针
    </h2>
    <pre><code class="hljs">#include &lt;stdio.h&gt;


void main()

{
  int a = 10;
  int *ptr = &amp;a; // 这里的*ptr表明声明一个指针变量，存储a的地址
  // a（变量）  → 内存地址：0x1000，存储的值：10
  // ptr（指针） → 内存地址：0x2000，存储的值：0x1000（即 a 的地址）
  printf("打印指针里面存储的地址%d\n", ptr);
  printf("打印指针里面存储的地址里面的值%d", *ptr);

}</code></pre>
