    </h3>
    <ul>
     <li>
      使用npm全局安装koa-generator生成器，代码及结果如下：
     </li>
    </ul>
    <pre><code class="prism language-bash">$ <span class="token function">npm</span> <span class="token function">install</span> -g koa-generator
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8413e1974701839d51fb6eedcb7ecd4b.png"/>
    </p>
    <ul>
     <li>
      <code>
       koa2 back-end
      </code>
      生成名为back-end 的koa项目
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5decec3feb9525abf57a6392232c7e3f.png"/>
     </li>
     <li>
      进入end-back目录，
      <code>
       npm install
      </code>
      安装依赖
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/48909a892132ccba25d68f1934e01c55.png"/>
     </li>
     <li>
      安装完成后使用npm start启动后端就可以在3000端口跑起来了
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/23871c2527c62fb30dd129deb9520320.png"/>
