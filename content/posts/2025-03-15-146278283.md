    <h2>
     <a id="1__0">
     </a>
     1 背景说明
    </h2>
    <p>
     在使用 Python 进行数据可视化时，matplotlib 是一个非常强大的工具。它提供了丰富的绘图功能，并且可以通过调整配置文件来自定义图表样式，以满足个性化需求。
    </p>
    <h2>
     <a id="2__2">
     </a>

    </h2>
    <p>

    </p>

</code></pre>
    <h2>
     <a id="3__11">
     </a>
     3 配置文件内容
    </h2>
    <pre><code class="prism language-text">Matplotlib配置目前分为以下几个部分：
  - BACKENDS 后端
  - LINES 线条
  - PATCHES 图形填充
  - HATCHES 填充图案
  - BOXPLOT 箱线图
  - FONT 字体
  - TEXT 文本
  - LaTeX LaTeX渲染
  - AXES 坐标轴
  - DATES 日期
  - TICKS 刻度
  - GRIDS 网格
  - LEGEND 图例
  - FIGURE 图表
  - IMAGES 图像
  - CONTOUR PLOTS 等高线图
  - ERRORBAR PLOTS 误差线图
  - HISTOGRAM PLOTS 直方图
  - SCATTER PLOTS 散点图
  - AGG RENDERING AGG渲染
  - PATHS 路径
  - SAVING FIGURES 保存图表
  - INTERACTIVE KEYMAPS 交互快捷键
  - ANIMATION 动画
</code></pre>
