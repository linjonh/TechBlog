     今天我们来制作一个Python编译的猜拳小游戏，如果去掉注释，仅仅13行呦！需要的朋友可以
     <strong>
      <span style="color:#fe2c24;">

    <p>
     <span style="color:#0d0016;">
      其实实现这个游戏也挺简单的，只需要导入“random”（随机）模块（旧版本编辑器叫函数），再加上“randint”或者是“choice”就可以辣！当然，这一段代码与其他代码不同的是：
      <strong>
       增添了
      </strong>
     </span>
     <strong>
      <span style="color:#fe2c24;">
       判断玩家输入信息是否与“剪刀”“石头”“布”匹配的信息
      </span>
      <span style="color:#0d0016;">
       ！
      </span>
     </strong>
     <span style="color:#0d0016;">
      如果输入的信息不匹配，就输出
     </span>
     <strong>
      <span style="color:#0d0016;">
       “
      </span>
     </strong>
     The option you entered is not recognized!
     <strong>
      <span style="color:#0d0016;">
       ”
      </span>
     </strong>
     <span style="color:#0d0016;">
      （输入信息不合法）。好嘞，话不多说，开始咯！
     </span>
    </p>
    <pre><code class="language-python">#猜拳小游戏开始啦！
from random import *
list1=['剪刀','石头','布']
player=str(input())
if player in list1:
    computer=choice(list1)
    #computer=randint(list1)
    if computer==player:
        print('tie!')
    elif (computer=='剪刀' and player=='石头') or (computer=='石头' and player=='布') or (computer=='布' and player=='剪刀'):
        print('player win!')
    else:
        print('computer win!')
else:
    print('The option you entered is not recognized!')</code></pre>
