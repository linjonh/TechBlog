       </span>
      </strong>
     </a>
     <span style="font-size:large;">
      <strong>
       什么是
       <span lang="en-us">
        WebService
       </span>
      </strong>
      <strong>
      </strong>
     </span>
    </p>
    <p class="MsoNormal" style="text-indent:21pt;">
     <span style="font-size:large;">
      <span lang="en-us">
       Web Service
      </span>
      是有企业发布的完成其特定商务需求的在线应用服务，其他公司或者应用软件可以能够通过
      <span lang="en-us">
       Internet
      </span>
      来访问这项在线服务。
     </span>
    </p>
    <p class="MsoNormal" style="text-indent:21pt;">
     <span style="font-size:large;">
      更多资源访问：
      <span lang="en-us">
       <a href="http://baike.baidu.com/view/837392.htm" rel="nofollow">
        http://baike.baidu.com/view/837392.htm
       </a>
      </span>
     </span>
    </p>
    <p class="MsoNormal">
     <span lang="en-us">
      <span style="font-size:large;">
      </span>
     </span>
    </p>
    <p class="MsoNormal">
     <a name="_Toc306194910">
      <strong>
       <span lang="en-us">
        <span style="font-size:large;">
         2.
        </span>
       </span>
      </strong>
     </a>
     <span style="font-size:large;">
      <strong>
       为什么需要
       <span lang="en-us">
        WebService
       </span>
      </strong>
      <strong>
      </strong>
     </span>
    </p>
    <p class="MsoNormal" style="text-indent:21pt;">
     <span style="font-size:large;">
      我们知道如果我们要把一个桌面程序发布成
      <span lang="en-us">
       web
      </span>
      程序的话，在以前并不是一件很简单的事情，当然在最早的时候有一项分布式开发技术
      <span lang="en-us">
       RMI
      </span>
      技术就实现了桌面程序
      <span lang="en-us">
       web
      </span>
      化。现在使用
      <span lang="en-us">
       WebService
      </span>
      就能够很简单的解决这个问题。其次使用
      <span lang="en-us">
       WebService
      </span>
      能够让两门语言进行通信，也就是说，你可以服务端是
      <span lang="en-us">
       Java
      </span>
      语言，而客户端是其他语言，比如
      <span lang="en-us">
       Rubby
      </span>
      语言。
     </span>
    </p>
    <p class="MsoNormal" style="text-indent:21pt;">
     <span style="font-size:large;">
      当然以上只是简单的说明，如果要连接更多的
      <span lang="en-us">
       webService
      </span>
      还需要您自己细心的体会。
     </span>
    </p>
    <p class="MsoNormal" style="text-indent:21pt;">
     <span lang="en-us">
      <span style="font-size:large;">
      </span>
     </span>
    </p>
    <p class="MsoNormal">
     <a name="_Toc306194911">
      <strong>
       <span lang="en-us">
        <span style="font-size:large;">
         3.
        </span>
       </span>
      </strong>
     </a>
     <span style="font-size:large;">
      <strong>
       主流
       <span lang="en-us">
        WebService
       </span>
       框架
      </strong>
      <strong>
      </strong>
     </span>
    </p>
    <p class="MsoNormal">
     <span style="font-size:large;">
      <span lang="en-us">
       Axis2
      </span>
      ——全称
      <span lang="en-us">
       Apache EXtensible Interaction System
      </span>
      即阿帕奇可扩展交互系统。
     </span>
    </p>
    <p class="MsoNormal">
     <span style="font-size:large;">
      <span lang="en-us">
       JAX-WS
      </span>
      ——
      <span lang="en-us">
       Java API for XML Web Services (JAX-WS)
      </span>
      是
      <span lang="en-us">
       Java
      </span>
      程序设计语言一个用来创建
      <span lang="en-us">
       Web
      </span>
      服务的
      <span lang="en-us">
       API
      </span>
      。
     </span>
    </p>
    <p class="MsoNormal">
     <span style="font-size:large;">
      等等。
     </span>
    </p>

    <p align="left" class="MsoNormal" style="text-align:left;">
     <span style="font-size:large;">
      <span lang="en-us">
      </span>
      接下来将以一个实例讲解
      <span lang="en-us">
       JAX-WS
      </span>
      。
     </span>
    </p>
    <p align="left" class="MsoNormal" style="text-align:left;">
     <span style="font-size:large;">
      在讲解之前我们大体了解下要开发一个
      <span lang="en-us">
       JAX-WS
      </span>
      的步骤：
     </span>
    </p>
    <p align="left" class="MsoNormal">
     <span style="font-size:large;">
      <span lang="en-us">
       （1）
      </span>

      <span lang="en-us">
       webservice
      </span>
      的接口
     </span>
    </p>

     <span style="font-size:large;">
      <span lang="en-us">

      </span>

      <span lang="en-us">

      </span>

      <span lang="en-us">

      </span>
