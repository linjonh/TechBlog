    </ul>
    <h5>
     <a id="2public_13">
     </a>
     （2）
     <code>
      public
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>

     </li>
     <li>
      <strong>
       示例
      </strong>

    </ul>

     <a id="3abstract_22">

     （3）
     <code>
      abstract
     </code>
    </h5>
    <ul>
     <li>
      <strong>

    </h4>
    <h5>
     <a id="1public_79">
     </a>
     （1）
     <code>
      public
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>

     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>

    </ul>
    <h5>
     <a id="2protected_86">
     </a>
     （2）
     <code>
      protected
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量可以在同一个类、同一个包以及子类中访问。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">protected</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="3private_93">
     </a>
     （3）
     <code>
      private
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量只能在同一个类中访问。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">private</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">30</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="4static_100">
     </a>
     （4）
     <code>
      static
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量属于类级别，而不是实例级别。所有实例共享同一个静态变量。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">40</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="5final_107">
     </a>
     （5）
     <code>
      final
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量不能被重新赋值（常量）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">final</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">50</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="6volatile_114">
     </a>
     （6）
     <code>
      volatile
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量在多线程环境下可见，确保线程安全。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">volatile</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">60</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="7transient_121">
     </a>
     （7）
     <code>
      transient
     </code>
    </h5>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：变量不会被序列化。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">transient</span> <span class="token keyword">int</span> myVar <span class="token operator">=</span> <span class="token number">70</span><span class="token punctuation">;</span>
