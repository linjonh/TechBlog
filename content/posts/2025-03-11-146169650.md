    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     格式：tar [选项] 归档及压缩包文件名(带.tar.gz或.tar.bz2) 需要归档的源文件或目录
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     tar命令的常用选项如下：
    </p>
    <ul>
     <li style="text-align:justify">
      -c：创建.tar格式的包文件。
     </li>
     <li style="text-align:justify">
      -C：解压时指定释放的目标文件夹。
     </li>
     <li style="text-align:justify">
      -f：表示使用归档文件。
     </li>
     <li style="text-align:justify">
      -j：调用bzip2程序进行压缩或解压缩。
     </li>
     <li style="text-align:justify">
      -p：打包时保留文件及目录的权限。
     </li>
     <li style="text-align:justify">
      -P：打包时保留文件及目录的绝对路径。
     </li>
     <li style="text-align:justify">
      -t：列表查看包内的文件。
     </li>
     <li style="text-align:justify">
      -v：输出详细信息。
     </li>
     <li style="text-align:justify">
      -x：解开.tar格式的包文件。
     </li>
     <li style="text-align:justify">
      -z：调用gzip程序进行压缩或解压缩。
     </li>
    </ul>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     从归档文件中恢复数据
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     格式：tar [选项] 归档及压缩包文件名 [-C 目标目录]
    </p>
    <h2 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     vim 编辑器练习基础操作
    </h2>
    <p>
     <strong>
      vimtutor
     </strong>
     是一个用于学习 Vim 编辑器基本命令的教程工具。它是 Vim 自带的一个教程，旨在帮助用户快速掌握 Vim 的基本操作和功能。通过这个教程，用户可以在大约 30 分钟内学习到大部分 Vim 的基本命令。
    </p>
    <p>
     如何使用 vimtutor
    </p>
    <p>
     在 Unix 系统中，如果 Vim 安装正常，可以通过命令行运行以下命令来启动
    </p>
    <p>
     <span style="color:#fe2c24">
      vimtutor
     </span>
    </p>

      <p>

      </p>
