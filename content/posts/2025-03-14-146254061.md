  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     板卡1、
    </p>
    <p>
     采用标准3U VPX架构，板上集成1片深圳国微电子SMQ7V690TFFG1761（pin-to-pin兼容Xilinx的XC7VX690T-2FFG1761I）FPGA，1片国防科大FT-M6678（功能兼容TI的TMS320C6678），对外接口RS422、网口、JTAG、TTL等接口。
    </p>
    <p>
     板卡2、
    </p>
    <p>
     板卡资源：
     <br/>
      DSP
     <br/>
     1) 1片多核DSP FT-M6678；
     <br/>
      FPGA
     <br/>
     1) 板卡采用1片复旦微的JFM7VX690T36 FPGA；
     <br/>
      存储能力
     <br/>
     1) DSP外挂4GB DDR3；
     <br/>
     2) DSP外挂256Mb SPI FLASH；
     <br/>
     3) FPGA芯片外挂2组2GB DDR3（每组2片），位宽32bit，总容量4GB。
     <br/>
     4) FPGA外挂256Mb BPI FLASH；
     <br/>
      板内互联能力
     <br/>
     1) DSP和FPGA之间通过4x SRIO实现互联；
     <br/>
     2) DSP和FPGA之间通过SPI、GPIO和EMIF实现互联；
     <br/>
      对外接口：
     <br/>
     a) 接口板的FPGA通过背板对外提供1组4x SRIO@5Gbps，2组1x SRIO@5Gbps；
     <br/>
     b) 接口板的FPGA通过背板对外提供1组SGMII；
     <br/>
     c) 接口板的DSP通过RJ45对外提供1个SGMII转1000BASE-T接口；
     <br/>
     d) 接口板的FPGA通过背板提供1路同步RS422，2路异步RS422，1路隔离TTL(自检信号)，1个TTL信号（备份）；
     <br/>
     e) 接口板通过背板提供槽位定义接口、IIC管理网口及外部时钟；
     <br/>
     f) 接口板通过背板对外提供1路MT接口的光纤信号，支持四收四发；
     <br/>
     g) 前面板提供1路FPGA的JTAG调试接口，1路FT-M6678的JTAG调试接口，1路MCU的JTAG调试接口，若干LED状态指示灯以及1个复位按键。
    </p>
    <p>
     板卡3、
    </p>
    <p>
     FT-M6678+复旦微电子FMQL45T900N全国产化匹配板
    </p>
    <p>
     1） FT-M6678N外挂4片DDR3，为深圳国微的4GB的DDR3颗粒；
     <br/>
     2） FT-M6678N外挂1片256MBIT NOR FLASH；
     <br/>
     3） FMQL45T900-N PS端外挂1组DDR3，数据位宽为32BIT，每组容量为1GB；
     <br/>
     4） FMQL45T900-N PL端外挂1组DDR3；
     <br/>
     5） FMQL45T900-N外接1片256MB FLASH。
    </p>
