    <h2>
     一.CSS：
    </h2>
    <h3>
     1.定义：
    </h3>
    <p>
     层叠样式表（Casecading Style Sheets），是一种样式表语言，用来描述HTML文档的呈现（美化内容）
    </p>
    <p>
     书写位置：title标签下方添加style双标签，style标签里面书写CSS代码
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        p{
            /* 文字颜色 */
            color: aqua;
            /* 字号 */
            font-size: 30px;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;p&gt;CSS&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
    </p>
    <h3>
     2.CSS引入方式
    </h3>
    <h4>
     （1）.内部样式表（学习使用）
    </h4>
    <p>
     CSS代码一般写在style标签里面
    </p>
    <p>
    </p>
    <h4>
     （2）.外部样式表（开发使用）
    </h4>
    <p>
     CSS代码写在单独的CSS文件中，在HTML中使用link标签引入
    </p>
    <pre><code class="language-html">&lt;link rel="stylesheet" href="./test.css"&gt;</code></pre>
    <p>
     其中link表示引入外部样式表，rel表示关系，样式表
    </p>
    <p>
    </p>
    <h4>
     （3）.行内样式（配合JavaScript使用）
    </h4>
    <p>
     CSS写在标签的style属性值里
    </p>
    <pre><code class="language-html">&lt;div style="color: red; font-size: 20px;"&gt;div标签&lt;/div&gt;</code></pre>
    <p>
    </p>
    <h2>
     二.选择器
    </h2>
    <h3>
     1.作用：
    </h3>
    <p>
     查找标签，设置样式
    </p>
    <p>
    </p>
    <h3>
     2.基础选择器
    </h3>
    <h4>
     （1）.标签选择器
    </h4>
    <p>
     使用标签名作为选择器，可以选中同名标签设置相同的样式
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        /* 只要是p标签就会是现在这个基础选择器中的样式 */
        p {
            color: red;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;p&gt;p标签&lt;/p&gt;
&lt;p&gt;文本内容&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
    </p>
    <h4>
     （2）.类选择器
    </h4>
    <p>
     查找标签，差异化设置标签的显示效果
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        /* 定义一个名字为red的类选择器 */
        .red {
            color: red;
        }

        .size {
            font-size: 50px;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;!-- 使用类选择器 --&gt;
&lt;div class="red"&gt;div标签&lt;/div&gt;
&lt;!-- 一个类选择器可以给多个标签使用 --&gt;
 &lt;p class="red"&gt;p标签&lt;/p&gt;
 &lt;!-- 一个标签可以使用多个选择器 --&gt;
  &lt;p class="red size"&gt;测试文本&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     一个类选择器可以给多个标签使用，一个标签也可以使用多个标签
    </p>
    <p>
    </p>
    <h4>
     （3）.id选择器
    </h4>
    <p>
     查找标签，差异化设置标签的显示效果
    </p>
    <p>
     使用场景：id选择器一般配合JavaScript使用，很少用来设置CSS样式
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        #red {
            color:red;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;p id="red"&gt;id选择器测试&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     同一个id选择器只能在一个页面使用一次
    </p>
    <p>
    </p>
    <h4>
     （4）.通配符选择器
    </h4>
    <p>
     查找页面所有标签，设置相同样式
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        * {
            color: red;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;p&gt;p标签&lt;/p&gt;
&lt;div&gt;div标签&lt;/div&gt;
&lt;h1&gt;h1标签&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     通配符选择器不需要调用，浏览器会自动查找页面所有标签，将其全部设为相同的样式
    </p>
    <p>
    </p>
    <h2>
     三.盒子
    </h2>
    <p>
     目标，使用合适的选择器画盒子
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        .red {
            width: 100px;
            height: 100px;
            background-color: red;
        }

        .orange {
            width: 200px;
            height: 200px;
            background-color: orange;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div class="red"&gt;div_test1&lt;/div&gt;
    &lt;div class="orange"&gt;div_test2&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     效果图：
    </p>

    <h2>
     四.文字控制属性
    </h2>
    <h3>
     font-size：
    </h3>
    <p>
     字体大小（常用单位：px）
    </p>
    <p>
    </p>
    <h3>
     font-weight：
    </h3>
    <p>
     字体粗细（开发时可以直接使用数字作为属性值（不需要加单位），也可以使用关键字如normal，bold（加粗））
    </p>
    <p>
    </p>
    <h3>
     font-style：
    </h3>
    <p>
     字体倾斜（属性值有normal和italic（倾斜），可以使用normal属性清除字体倾斜）
    </p>
    <p>
    </p>
    <h3>
     line-height：
    </h3>
    <p>
     行高（属性值常用单位：px，或直接使用数字，表示为当前font-size的倍数）
    </p>
    <p>
     作用：设置多行文本的间距
    </p>
    <pre><code class="language-html">    &lt;style&gt;
        p {
            font-size: 20px;
            /* px单位 */
            line-height: 20px;
            /* 纯数字，表示为当前font-size的倍数 */
            line-height: 2;
        }
    &lt;/style&gt;</code></pre>
    <p>
     行高的测量方法：从一行文字的最顶端（最底端）量到下一行文字的最顶端（最底端）
    </p>
    <p>
    </p>
    <h3>
     font-family：
    </h3>
    <p>
     字体族（改变当前文字的字体，如楷体等等）
    </p>
    <p>
     font-family属性值可以书写多个字体名，各个字体名之间用逗号隔开，执行顺序是从左向右依次查找，一旦找到在当前环境中存在该字体就会使用该字体
    </p>
    <p>
    </p>
    <h3>
     font：
    </h3>
    <p>
     字体复合属性
    </p>
    <p>
     能够将多种属性同时写到本属性中，是属性的简写方式，一个属性对应多个值的写法，各个属性值之间用空格隔开
    </p>
    <pre><code class="language-html">    &lt;style&gt;
        p {
            /* 是否倾斜 加粗 字号 行高 字体 */
            font:italic 700 30px/2 楷体;
        }
    &lt;/style&gt;</code></pre>
    <p>
     使用场景：设置网页文字公共样式
    </p>
    <p>
     注意font中的属性必须按顺序书写，分别为：是否倾斜 是否加粗 字号/行高 字体，而在这之中字号和字体值必须书写，否则font属性不生效
    </p>
    <p>
    </p>
    <h3>
     text-indent：
    </h3>
    <p>
     文本缩进（属性值常用单位为：px和em，而1em等于当前标签的字号大小）
    </p>
    <pre><code class="language-html">    &lt;style&gt;
        p {
            text-indent: 2em;
        }
    &lt;/style&gt;</code></pre>
    <p>
    </p>
    <p>
    </p>
    <h3>
     text-align：
    </h3>
    <p>
     文本对齐（left：左对齐，center：居中对齐，right：右对齐）
    </p>
    <pre><code class="language-html">    &lt;style&gt;
        p {
            text-align: center;
        }
    &lt;/style&gt;</code></pre>
    <p>
     我们可以使用本属性完成文本对齐（left，center，right）
    </p>
    <p>
     text-align还可以设置图片的水平对齐，但是因为text-align本质是控制内容的对齐方式，所以属性要设置给内容的父级才能够生效
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        p {
            text-align: center;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;div&gt;
    &lt;img src="" alt=""&gt;
&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
    </p>
    <h3>
     text-decoration：
    </h3>
    <p>
     修饰线（属性值分别为：none：无，underline：下划线，line-through：删除线，overline：上划线）
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;title&gt;Document&lt;/title&gt;
    &lt;style&gt;
        .none {
            text-decoration: none;
        }
        .underline {
            text-decoration: underline;
        }
        .line-through {
            text-decoration: line-through;
        }
        .overline {
            text-decoration: overline;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div&gt;
        &lt;p class="none"&gt;none&lt;/p&gt;
        &lt;p class="underline"&gt;underline&lt;/p&gt;
        &lt;p class="line-through"&gt;line-through&lt;/p&gt;
        &lt;p class="overline"&gt;overline&lt;/p&gt;
    &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     效果图：
    </p>
    <p>
     <img alt="" height="186" src="https://i-blog.csdnimg.cn/direct/f91d70beb37a4907b68f9b06af41d0f0.png" width="135"/>
    </p>
    <p>
    </p>
    <h3>
     color：
    </h3>
    <p>
     颜色
    </p>
    <p>
     可以用四种方式来表示颜色：
    </p>
    <p>
     1.颜色关键字（red，green等等）
    </p>
    <p>
     2.rgb表示法（rgb（r,g,b）其中r，g，b表示红绿蓝三原色，取值为0到255）
    </p>
    <p>
     3.rgba表示法（其中多出来的a表示透明度，取值为0到1）
    </p>
    <p>
     4.十六进制表示法（#RRGGBB，这是在开发时使用的，并且我们只需要从设计稿中复制参数即可）
    </p>
