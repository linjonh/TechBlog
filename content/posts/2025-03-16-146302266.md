    <h3>
     默认成员函数
    </h3>
    <p>
     构造函数，用于初始化string对象
     <br/>
     参数str默认为空字符串""，这样可以在不传入参数时创建一个空的string对象。
     <br/>
     如果传入了有效的C风格字符串，则根据该字符串来初始化string对象。
    </p>
    <pre><code> string(const char* str = "") {
            // 如果str为空指针，创建一个空字符串。
            // 此时_size设为0表示字符串长度为0，_capacity也设为0表示当前容量为0，
            // 分配一个大小为1的字符数组用于存储结束符'\0'，并将其赋值给_str。
            if (str == nullptr) {
                _size = 0;
                _capacity = 0;
                _str = new char[1];
                _str[0] = '\0';
            }
            // 如果str不为空，计算其长度并赋值给_size，_capacity初始化为与_size相同。
            // 分配一个大小为_strlen + 1的字符数组（包含结束符'\0'），并将str的内容复制到该数组中。
            else {
                _size = strlen(str);
                _capacity = _size;
                _str = new char[_capacity + 1];
                strcpy(_str, str);
            }
        }</code></pre>
    <h3>
     拷贝构造函数
    </h3>
    <ul>
     <li>
      用于从另一个string对象创建新的string对象
     </li>
     <li>
      采用深拷贝，目的是确保新对象和原对象相互独立，修改新对象不会影响原对象，反之亦然。
     </li>
     <li>
      传统写法：先为新对象分配与原对象容量相同加1（包含结束符'\0'）的内存空间，再将原对象的字符串内容复制到新分配的内存中，最后更新新对象的_size和_capacity。
     </li>
    </ul>
    <p>
    </p>
    <pre><code> string(const string&amp; s)
            : _str(new char[s._capacity + 1])
            , _size(0)
            , _capacity(0) {
            strcpy(_str, s._str);
            _size = s._size;
            _capacity = s._capacity;
        }</code></pre>
    <h3>
     赋值运算符重载函数
    </h3>
    <p>
     用于将一个string对象赋值给另一个string对象
    </p>
    <pre><code> // 传统写法：先检查是否为自赋值（即当前对象和要赋值的对象是否为同一个对象），
        // 如果不是自赋值，先释放原对象的内存，然后分配与赋值对象容量相同加1的内存空间，
        // 再将赋值对象的字符串内容复制到新分配的内存中，最后更新当前对象的_size和_capacity。
        string&amp; operator=(const string&amp; s) {
            if (this != &amp;s) {
                delete[] _str;
                _str = new char[s._capacity + 1];
                strcpy(_str, s._str);
                _size = s._size;
                _capacity = s._capacity;
            }
            return *this;
        }

        // 现代写法：利用值传递的特性，让编译器调用拷贝构造函数创建临时对象
        // 然后通过交换函数将临时对象和当前对象的内容交换，实现赋值操作。
        // 这样写的好处是代码更简洁，并且避免了传统写法中复杂的内存管理操作。
        string&amp; operator=(string s) {
            swap(s);
            return *this;
        }</code></pre>

    <h4>
     reserve()
    </h4>
    <ul>
     <li>
      调整字符串的容量为至少n个字符的空间
     </li>
     <li>
      如果n大于当前容量，则重新分配内存并复制原字符串内容，以满足新的容量需求。
     </li>
     <li>
      如果n小于等于当前容量，则不进行任何操作，因为当前容量已经足够。
     </li>
    </ul>
    <pre><code class="hljs"> void reserve(size_t n) {
            if (n &gt; _capacity) {
                char* new_str = new char[n + 1];
                strcpy(new_str, _str);
                delete[] _str;
                _str = new_str;
                _capacity = n;
            }
        }</code></pre>
    <h4>
     resize()
    </h4>
    <ol>
     <li>
      调整字符串的大小为n个字符
     </li>
     <li>
      如果n小于当前大小，则截断字符串，将_size设置为n，并在新的末尾位置添加结束符'\0'。
     </li>
     <li>
      如果n大于当前大小，且n大于当前容量，则先调整容量，再用指定字符ch填充新增的空间。
     </li>
     <li>
      如果n大于当前大小，但n小于等于当前容量，则直接用指定字符ch填充新增的空间。
     </li>
    </ol>
    <pre><code class="hljs"> void resize(size_t n, char ch = '\0') {
            if (n &lt; _size) {
                _size = n;
                _str[_size] = '\0';
            }
            else {
                if (n &gt; _capacity) {
                    reserve(n);
                }
                for (size_t i = _size; i &lt; n; ++i) {
                    _str[i] = ch;
                }
                _size = n;
                _str[_size] = '\0';
            }
        }
</code></pre>

    <h4>
     push_back
    </h4>
    <p>
     第一种
    </p>
    <pre><code class="hljs"> // 在字符串末尾添加一个字符ch
        // 首先检查当前容量是否足够，如果当前_size等于_capacity，说明容量已满，
        // 此时进行扩容操作，扩容策略为：若当前容量为0则初始化为4，否则扩容为原来的2倍。
        // 然后将字符ch添加到字符串末尾，并更新_size和添加结束符'\0'。
        void push_back(char ch) {
            if (_size == _capacity) {
                reserve(_capacity == 0? 4 : _capacity * 2);
            }
            _str[_size] = ch;
            _str[_size + 1] = '\0';
            ++_size;
        }</code></pre>
    <p>
     第二种
    </p>
    <pre><code class="hljs"> // 复用insert函数在字符串末尾插入字符，效果与push_back相同
        // 这样做的好处是可以减少代码重复，提高代码的复用性。
        void push_back(char ch) {
            insert(_size, ch);
        }
</code></pre>
    <h4>
     append
    </h4>
    <ul>
     <li>
      在字符串末尾添加一个C风格字符串str
     </li>
     <li>
      先计算添加后的总长度，如果总长度超过当前容量，则进行扩容，
     </li>
     <li>
      然后将str复制到字符串末尾，并更新_size，以反映新的字符串长度。
     </li>
    </ul>
    <pre><code class="hljs"> void append(const char* str) {
            size_t new_size = _size + strlen(str);
            if (new_size &gt; _capacity) {
                reserve(new_size);
            }
            strcpy(_str + _size, str);
            _size = new_size;
        }
</code></pre>

     <strong>
      复用insert函数在字符串末尾插入字符串，效果与append相同

    <h4>
     insert
    </h4>
    <ul>
     <li>
      在字符串的指定位置pos插入一个字符ch
     </li>
     <li>
      先检查位置pos的合法性，如果pos大于_size则抛出异常，因为位置超出了字符串的范围。
     </li>
     <li>
      再检查当前容量是否足够，不够则进行扩容，扩容策略与push_back中相同。
     </li>
     <li>
      然后将pos及之后的字符向后移动一位，为插入字符ch腾出位置，插入字符ch，并更新_size。
     </li>
    </ul>
    <pre><code class="hljs">  string&amp; insert(size_t pos, char ch) {
            if (pos &gt; _size) {
                throw std::out_of_range("Invalid position");
            }
            if (_size == _capacity) {
                reserve(_capacity == 0? 4 : _capacity * 2);
            }
            for (size_t i = _size; i &gt; pos; --i) {
                _str[i] = _str[i - 1];
            }
            _str[pos] = ch;
            ++_size;
            return *this;
        }</code></pre>
    <p>
     在字符串的指定位置pos插入一个C风格字符串str
     <br/>
     先检查位置pos的合法性，如果pos大于_size则抛出异常，因为位置超出了字符串的范围。
     <br/>
     计算插入后的总长度，若超过当前容量则进行扩容，
     <br/>
     将pos及之后的字符向后移动str的长度位，为插入字符串str腾出位置，插入字符串str，并更新_size。
    </p>
    <pre><code class="hljs"> string&amp; insert(size_t pos, const char* str) {
            if (pos &gt; _size) {
                throw std::out_of_range("Invalid position");
            }
            size_t len = strlen(str);
            if (_size + len &gt; _capacity) {
                reserve(_size + len);
            }
            for (size_t i = _size; i &gt;= pos; --i) {
                _str[i + len] = _str[i];
            }
            strncpy(_str + pos, str, len);
            _size += len;
            return *this;
        }</code></pre>
    <h4>
     erase
    </h4>
    <p>
     删除从位置pos开始的len个字符
     <br/>
     先检查位置pos的合法性，如果pos大于等于_size则抛出异常，因为位置超出了字符串的范围。
     <br/>
     如果len大于等于剩余字符数，则直接截断字符串，将_size设置为pos，并在新的末尾位置添加结束符'\0'。
     <br/>
     否则将pos + len之后的字符复制到pos位置，并更新_size，以反映删除后的字符串长度。
    </p>
    <pre><code class="hljs">    string&amp; erase(size_t pos, size_t len) {
            if (pos &gt;= _size) {
                throw std::out_of_range("Invalid position");
            }
            if (len &gt;= _size - pos) {
                _size = pos;
                _str[_size] = '\0';
            }
            else {
                strcpy(_str + pos, _str + pos + len);
                _size -= len;
            }
            return *this;
        }</code></pre>

    </h4>
    <p>
     重载[]运算符，用于通过下标访问字符串中的字符
     <br/>
     <strong>
      非const版本
     </strong>
     ，返回字符的引用，允许对字符进行修改，
     <br/>
     先检查下标i的合法性，如果i大于等于_size则抛出异常，因为下标超出了字符串的范围，
     <br/>
     否则返回_str[i]的引用。
    </p>
    <pre><code class="hljs"> char&amp; operator[](size_t i) {
            if (i &gt;= _size) {
                throw std::out_of_range("Invalid index");
            }
            return _str[i];
        }
</code></pre>
    <h4>

     <br/>

    <h4>
     <a class="link-info" href="https://legacy.cplusplus.com/reference/string/string/find/" rel="nofollow" title="find">
      find
     </a>
    </h4>
    <p>
     从位置pos开始查找字符ch在字符串中第一次出现的位置
     <br/>
     如果找到则返回该位置的下标，否则返回npos（定义为-1），表示未找到。
     <br/>
     先检查位置pos的合法性，如果pos大于等于_size则抛出异常，因为位置超出了字符串的范围，
     <br/>
     然后从pos位置开始遍历字符串，找到字符ch则返回其位置，遍历结束未找到则返回npos。
    </p>
    <pre><code class="hljs">        size_t find(char ch, size_t pos = 0) const {
            if (pos &gt;= _size) {
                throw std::out_of_range("Invalid position");
            }
            for (size_t i = pos; i &lt; _size; ++i) {
                if (_str[i] == ch) {
                    return i;
                }
            }
            return npos;
        }</code></pre>
    <pre><code class="hljs"> size_t find(const char* str, size_t pos = 0) const {
            if (pos &gt;= _size) {
                throw std::out_of_range("Invalid position");
            }
            const char* result = strstr(_str + pos, str);
            if (result) {
                return result - _str;
            }
            return npos;
        }</code></pre>
    <h4>
     <a class="link-info" href="https://legacy.cplusplus.com/reference/string/string/rfind/" rel="nofollow" title="rfind">
      rfind
     </a>
    </h4>
    <p>
     从位置pos开始反向查找字符ch在字符串中最后一次出现的位置
     <br/>
     如果找到则返回该位置的下标，否则返回npos（定义为-1），表示未找到。
     <br/>
     先检查位置pos的合法性，如果pos大于等于_size则将pos设为_size - 1，
     <br/>
     然后将字符串逆置，调用正向查找函数，再将结果转换为原字符串的位置。
     <br/>
    </p>
    <pre><code class="hljs">        size_t rfind(char ch, size_t pos = npos) const {
            if (pos &gt;= _size) {
                pos = _size - 1;
            }
            string tmp(*this);
            std::reverse(tmp.begin(), tmp.end());
            size_t rev_pos = _size - 1 - pos;
            size_t ret = tmp.find(ch, rev_pos);
            if (ret != npos) {
                return _size - 1 - ret;
            }
            return npos;
        }
</code></pre>

     <a class="link-info" href="http://重载输入运算符&gt;&gt;" rel="nofollow" title="重载输入运算符&gt;&gt;">
      重载输入运算符&gt;&gt;
     </a>
    </h4>
    <p>
     解释：重载的输入运算符&gt;&gt;从输入流in中读取字符，直到遇到空格或换行符，
     <br/>
     将读取的字符逐个添加到字符串s中，实现字符串的输入功能。
    </p>
    <pre><code>std::istream&amp; operator&gt;&gt;(std::istream&amp; in, string&amp; s) {
        s.clear();
        char ch;
        while (in.get(ch) &amp;&amp; ch != ' ' &amp;&amp; ch != '\n') {
            s.push_back(ch);
        }
        return in;
    }
</code></pre>

    <h4 style="background-color:transparent">
     getline
    </h4>
    <p>
     getline函数用于读取一行含有空格的字符串。实现时于&gt;&gt;运算符的重载基本相同，只是当读取到’\n’的时候才停止读取字符。
    </p>
    <pre><code class="hljs">//读取一行含有空格的字符串
istream&amp; getline(istream&amp; in, string&amp; s)
{
	s.clear(); //清空字符串
	char ch = in.get(); //读取一个字符
	while (ch != '\n') //当读取到的字符不是'\n'的时候继续读取
	{
		s += ch; //将读取到的字符尾插到字符串后面
		ch = in.get(); //继续读取字符
	}
	return in;
}
</code></pre>
