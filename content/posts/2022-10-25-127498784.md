    <h2 id="%E4%B8%80%E3%80%81%E5%85%A8%E5%B1%80%E9%85%8D%E7%BD%AE">
     一、全局配置
    </h2>
    <h4 id="requiredPrivateInfos">
     requiredPrivateInfos：
     <a href="https://developers.weixin.qq.com/miniprogram/dev/reference/configuration/app.html#requiredPrivateInfos" rel="nofollow" title="全局配置 | 微信开放文档">
      全局配置 | 微信开放文档
     </a>
    </h4>
    <p>
     申明需要使用的地理位置相关接口，类型为数组。目前支持以下项目：
    </p>
    <ul>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.getFuzzyLocation.html" rel="nofollow" title="getFuzzyLocation">
       getFuzzyLocation
      </a>
      : 获取模糊地理位置
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.getLocation.html" rel="nofollow" title="getLocation">
       getLocation
      </a>
      : 获取精确地理位置
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.onLocationChange.html" rel="nofollow" title="onLocationChange">
       onLocationChange
      </a>
      : 监听试试地理位置变化事件
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.startLocationUpdate.html" rel="nofollow" title="startLocationUpdate">
       startLocationUpdate
      </a>
      : 接收位置消息（前台）
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.startLocationUpdateBackground.html" rel="nofollow" title="startLocationUpdateBackground">
       startLocationUpdateBackground
      </a>
      : 接收位置消息（前后台）
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.chooseLocation.html" rel="nofollow" title="chooseLocation">
       chooseLocation
      </a>
      : 打开地图选择位置
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/location/wx.choosePoi.html" rel="nofollow" title="choosePoi">
       choosePoi
      </a>
      : 打开 POI 列表选择位置
     </li>
     <li>
      <a href="https://developers.weixin.qq.com/miniprogram/dev/api/open-api/address/wx.chooseAddress.html" rel="nofollow" title="chooseAddress">
       chooseAddress
      </a>
      : 获取用户地址信息
     </li>
    </ul>
    <p>
     如：app.json
    </p>
    <pre><code>{
  "pages": ["pages/index/index"],
  "requiredPrivateInfos": [
      "getLocation",
      "onLocationChange",
      "startLocationUpdateBackground",
      "chooseAddress",
      "chooseLocation",
      "choosePoi"
  ]
}
</code></pre>
    <p>
     注：若使用以上接口，均需在小程序管理后台，
     <a href="https://mp.weixin.qq.com/wxamp/categoryapi/index?token=1033339147&amp;lang=zh_CN" rel="nofollow" title="「开发」-「开发管理」-「接口设置」">
      「开发」-「开发管理」-「接口设置」
     </a>
     中自助开通该接口权限。
    </p>
