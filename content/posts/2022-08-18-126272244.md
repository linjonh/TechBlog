    <p>
     String数据类型的split()方法是根据
     <span style="color:#fe2c24;">
      正则表达式
     </span>
     <span style="color:#0d0016;">
      来分割字符串
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      方法中提供两个参数split(String regex,int limit)
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      1.String regex
     </span>
    </p>
    <p>
     <span style="color:#0d0016;">
      2.int limit
     </span>
    </p>
    <p>
     我们平时使用比较多的是方法中的参数regex
    </p>
    <p>
     接下来记录下limit参数的用法
    </p>
    <p>
    </p>
    <pre><code class="language-java">//常用的分割方式
String str = "123;456;789;;";
String[] strList = str.split(";");
for(String s : strList){
    System.out.println(s);
}

输出结果为：
123
456
789

String str = "123;456;789;;";
String[] strList = str.split(";"，2);
for(String s : strList){
System.out.println(s);
}

输出结果为：
123
456;789;;

String str = "123;456;789;;";
String[] strList = str.split(";"，5);
for(String s : strList){
System.out.println(s+"_"); //"_"是为了打印效果，结果可以看出空字符也被分割出来
}

输出结果为：
123*
456*
789\*

-
-

String str = "123;456;789;;";
String[] strList = str.split(";"，-1);
for(String s : strList){
System.out.println(s+"_"); //"_"是为了打印效果，结果可以看出空字符也被分割出来
}

输出结果为：
123*
456*
789\*

- *</code></pre>
  <p>
  注意：
  </p>
  <p>
  .和$、|、*等的转义字符，需要使用
  <span style="color:#fe2c24;">
  双斜杠
  </span>
  \\进行转义
  </p>
  <p>
  如：
  </p>
  <pre><code>split("\\.")</code></pre>
