    <h4>
     <a id="1_lower_bound__4">
     </a>
     1.
     <strong>
      <code>
       lower_bound
      </code>
      函数
     </strong>
    </h4>
    <h5>
     <a id="_5">
     </a>
     功能：
    </h5>
    <p>
     在有序数组
     <code>
      a
     </code>
     中查找第一个
     <strong>
      大于或等于
     </strong>
     <code>
      target
     </code>
     的元素的位置。
    </p>
    <h5>
     <a id="_8">
     </a>
     参数：
    </h5>
    <ul>
     <li>
      <code>
       a[]
      </code>
      ：有序数组。
     </li>
     <li>
      <code>
       n
      </code>
      ：数组的长度。
     </li>
     <li>
      <code>
       target
      </code>
      ：目标值。
     </li>
    </ul>
    <h5>
     <a id="_13">
     </a>
     实现逻辑：
    </h5>
    <ol>
     <li>
      初始化左边界
      <code>
       l = 0
      </code>
      ，右边界
      <code>
       r = n - 1
      </code>
      。
     </li>
     <li>
      使用二分查找：
      <ul>
       <li>
        计算中间位置
        <code>
         mid = l + (r - l) / 2
        </code>
        。
       </li>
       <li>
        如果
        <code>
         a[mid] &gt;= target
        </code>
        ，说明目标值在左半部分，更新右边界
        <code>
         r = mid
        </code>
        。
       </li>
       <li>
        否则，目标值在右半部分，更新左边界
        <code>
         l = mid + 1
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      最终返回
      <code>
       l
      </code>
      ，即第一个大于或等于
      <code>
       target
      </code>
      的位置。
     </li>
    </ol>
