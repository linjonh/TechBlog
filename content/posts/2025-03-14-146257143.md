     CSS3 选择器语法知识点及案例代码
    </h2>
    <h3>
     <a id="_1">
     </a>
     一、基础选择器
    </h3>
    <h4>
     <a id="1__3">
     </a>
     1. 元素选择器
    </h4>
    <p>
     根据 HTML 元素的标签名来选择元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择所有的 p 元素 */</span>
<span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="2__14">
     </a>
     2. 类选择器
    </h4>
    <p>
     根据 HTML 元素的 class 属性来选择元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 class 为 "myClass" 的元素 */</span>
<span class="token selector">.myClass</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-size</span><span class="token punctuation">:</span> 16px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="3_ID__25">
     </a>
     3. ID 选择器
    </h4>
    <p>
     根据 HTML 元素的 id 属性来选择元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 id 为 "myId" 的元素 */</span>
<span class="token selector">#myId</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> blue<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_36">
     </a>
     二、组合选择器
    </h3>
    <h4>
     <a id="1__38">
     </a>
     1. 后代选择器
    </h4>
    <p>
     选择作为某个元素后代的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 div 元素内部的所有 p 元素 */</span>
<span class="token selector">div p</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-weight</span><span class="token punctuation">:</span> bold<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="2__49">
     </a>
     2. 子代选择器
    </h4>
    <p>
     选择作为某个元素直接子代的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 div 元素的直接子代 p 元素 */</span>
<span class="token selector">div &gt; p</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> green<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="3__60">
     </a>
     3. 相邻兄弟选择器
    </h4>
    <p>
     选择紧接在另一个元素后的元素，且二者具有相同的父元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择紧接在 p 元素后的 span 元素 */</span>
<span class="token selector">p + span</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">text-decoration</span><span class="token punctuation">:</span> underline<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="4__71">
     </a>
     4. 通用兄弟选择器
    </h4>
    <p>
     选择与某个元素具有相同父元素的所有后续兄弟元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 div 元素内部所有在 p 元素之后的 span 元素 */</span>
<span class="token selector">p ~ span</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> purple<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_82">
     </a>
     三、属性选择器
    </h3>
    <h4>
     <a id="1__84">
     </a>
     1. 基本属性选择器
    </h4>
    <p>
     根据元素的属性名来选择元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择具有 title 属性的元素 */</span>
<span class="token selector">[title]</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">border</span><span class="token punctuation">:</span> 1px solid black<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="2__95">
     </a>
     2. 属性值选择器
    </h4>
    <p>
     根据元素的属性值来选择元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 type 属性值为 "text" 的元素 */</span>
<span class="token selector">[type="text"]</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">width</span><span class="token punctuation">:</span> 200px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="3__106">
     </a>
     3. 属性值包含选择器
    </h4>
    <p>
     选择属性值包含指定字符串的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 class 属性值包含 "menu" 的元素 */</span>
<span class="token selector">[class*="menu"]</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">display</span><span class="token punctuation">:</span> inline-block<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="4__117">
     </a>
     4. 属性值以特定字符串开头选择器
    </h4>
    <p>
     选择属性值以指定字符串开头的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 data-id 属性值以 "item" 开头的元素 */</span>
<span class="token selector">[data-id^="item"]</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">margin</span><span class="token punctuation">:</span> 5px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="5__128">
     </a>
     5. 属性值以特定字符串结尾选择器
    </h4>
    <p>
     选择属性值以指定字符串结尾的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择 lang 属性值以 "CN" 结尾的元素 */</span>
<span class="token selector">[lang$="CN"]</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">padding</span><span class="token punctuation">:</span> 10px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="_139">
     </a>
     四、伪类选择器
    </h3>
    <h4>
     <a id="1__141">
     </a>
     1. 链接伪类
    </h4>
    <p>
     用于选择不同状态的链接。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 未访问的链接 */</span>
<span class="token selector">a:link</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> blue<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 已访问的链接 */</span>
<span class="token selector">a:visited</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> purple<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 鼠标悬停在链接上 */</span>
<span class="token selector">a:hover</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 鼠标点击链接时 */</span>
<span class="token selector">a:active</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> yellow<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="2__167">
     </a>
     2. 目标伪类
    </h4>
    <p>
     用于选择当前活动的元素（如被锚点定位的元素）。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 被锚点定位的元素 */</span>
<span class="token selector">:target</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">border</span><span class="token punctuation">:</span> 2px solid green<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="3_UI__178">
     </a>
     3. UI 状态伪类
    </h4>
    <p>
     用于选择处于特定状态的 UI 元素（如按钮）。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 被禁用的按钮 */</span>
<span class="token selector">button:disabled</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">opacity</span><span class="token punctuation">:</span> 0.6<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 被选中的选项 */</span>
<span class="token selector">option:checked</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> lightblue<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="4__194">
     </a>
     4. 结构伪类
    </h4>
    <p>
     用于选择基于文档结构的元素。
    </p>
    <pre><code class="prism language-css"><span class="token comment">/* 选择父元素的第一个子元素 */</span>
<span class="token selector">:first-child</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-weight</span><span class="token punctuation">:</span> bold<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的最后一个子元素 */</span>
<span class="token selector">:last-child</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> gray<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的第 n 个子元素 */</span>
<span class="token selector">:nth-child(n)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">background-color</span><span class="token punctuation">:</span> lightgray<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的第奇数个子元素 */</span>
<span class="token selector">:nth-child(odd)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">border</span><span class="token punctuation">:</span> 1px solid black<span class="token punctuation">;</span>

<span class="token comment">/* 选择父元素的第偶数个子元素 */</span>
<span class="token selector">:nth-child(even)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">padding</span><span class="token punctuation">:</span> 5px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的倒数第 n 个子元素 */</span>
<span class="token selector">:nth-last-child(n)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">margin</span><span class="token punctuation">:</span> 10px<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的第 n 个类型元素 */</span>
<span class="token selector">:nth-of-type(n)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">font-style</span><span class="token punctuation">:</span> italic<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素的倒数第 n 个类型元素 */</span>
<span class="token selector">:nth-last-of-type(n)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">text-transform</span><span class="token punctuation">:</span> uppercase<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素中不是第一个子元素的元素 */</span>
<span class="token selector">:not(:first-child)</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">color</span><span class="token punctuation">:</span> brown<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token comment">/* 选择父元素中只有该类型的元素 */</span>
<span class="token selector">:only-of-type</span> <span class="token punctuation">{<!-- --></span>
  <span class="token property">border</span><span class="token punctuation">:</span> 2px solid red<span class="token punctuation">;</span>

<span class="token comment">/* 选择父元素中只有该类型的元素 */</span>

<span class="token comment">/* 选择父元素中空的元素（没有子元素） */</span>

</code></pre>
