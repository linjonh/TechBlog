    <h4 id="cjwzG">
     1.
     <span style="color:#fe2c24">
      <strong>
       确保npm能用（安装Node.js）
      </strong>
     </span>
    </h4>
    <p id="u6a147bce">
     Node.js的下载地址:
    </p>
    <p id="uf31254ca">
     <a href="https://nodejs.org/zh-cn/download/" rel="nofollow" title="Node.js — Download Node.js®">
      Node.js — Download Node.js®
     </a>
    </p>
    <p id="ued339dbe">
     直接下一步下一步安装就行
    </p>
    <p id="u72f4d99c">
     安装结束后，打开终端，输入npm命令，注意配置环境变量
    </p>
    <p>
     2.
    </p>
    <h4 id="oiNyn">
     Vue CLI（脚手架安装）
    </h4>
    <h5 id="R26JS">
     1、 Vue的脚手架简介
    </h5>
    <p id="u2abf37b1">
     Vue的脚手架（Vue CLI: Command Line Interface）是Vue官方提供的标准化开发平台。它可以将我们.vue的代码进行编译生成html css js代码，并且可以将这些代码自动发布到它自带的服务器上，为我们Vue的开发提供了一条龙服务。脚手架官网地址：
    </p>
    <p id="ucf3f279b">
     <a href="https://cli.vuejs.org/zh" rel="nofollow" title="Vue CLI">
      Vue CLI
     </a>
    </p>
    <p id="u5dd6a3ab">
     注意：Vue CLI 4.x需要Node.js v8.9及以上版本，推荐v10以上。
    </p>
    <h5 id="xI7UP">
     2、脚手架安装步骤：
    </h5>
    <p id="u6c03f9cc">
     <strong>
      ① 建议先配置一下npm镜像：(可装可不装)
     </strong>
    </p>
    <p id="u633639f1">
     1) 豆瓣镜像：npm config set registry
     <a href="http://pypi.douban.com/simple/" rel="nofollow" title="http://pypi.douban.com/simple/">
      http://pypi.douban.com/simple/
     </a>
    </p>
    <p id="u5777d9ef">
     清华镜像：npm config set registry
     <a href="https://pypi.tuna.tsinghua.edu.cn/simple" rel="nofollow" title="Simple Index">
      Simple Index
     </a>
    </p>
    <p id="u7fd31dc2">
     切回原来的npm包：npm config set registry
     <a href="https://registry.npmjs.org/" rel="nofollow" title="https://registry.npmjs.org">
      https://registry.npmjs.org
     </a>
    </p>
    <p id="uad3c3ede">
     2)终端输入： npm config get registry
    </p>
    <p id="u91fab3bc">
     返回成功对应进行地址，表示设置成功
    </p>
    <p id="u6b87f9c1">
     <strong>
      ② 第一步：安装脚手架（全局方式：表示只需要做一次即可）
     </strong>
    </p>
    <p id="u12928118">
     1) npm install -g @vue/cli
    </p>
    <p id="ua3c5fc6d">
     2) 安装完成后，重新打开DOS命令窗口，输入vue --version命令查看脚手架版本
    </p>
    <p id="u92bf8577">
     有版本信息，表示成功
    </p>
    <p id="ud3fab39c">
     注意：环境变量的配置
    </p>
    <p id="u7fc1ecbc">
     npm config get prefix 获取node的位置
    </p>
    <p id="u3a089b53">
     编辑系统变量---&gt;高级---&gt;环境变量---&gt;系统变量----&gt;path--&gt;node的位置
    </p>
    <p id="uef0beec1">
     <strong>
      ③ 第二步：创建项目（项目中自带脚手架环境，自带一个HelloWorld案例）
     </strong>
    </p>
    <h6 id="noual">
     第一种方式
    </h6>
    <p id="uc16a7047">
     1) 切换到要创建项目的目录，然后使用 vue create vue_pro
    </p>
    <p id="u790478ad">
    </p>
    <p class="img-center">
     <img alt="" height="282" id="ua74c6d41" src="https://i-blog.csdnimg.cn/img_convert/c2bc054fdc93ff3b1d2c2c7e43ab26b4.png" width="790"/>
    </p>
    <p id="u8eace69d">
     这里选择Vue2，
    </p>
    <p id="uc4e3aa57">
     babel：负责ES6语法转换成ES5。
    </p>
    <p id="uce5db1ac">
     eslint：负责语法检查的。
    </p>
    <p id="ue18f5333">
     回车之后，就开始创建项目，创建脚手架环境（内置了webpack loader），自动生成HelloWorld案例。
    </p>
    <p id="uaa7b6a2c">
    </p>
    <p class="img-center">
     <img alt="" height="331" id="u4b3cce84" src="https://i-blog.csdnimg.cn/img_convert/cc1e367680f81717ff54dcc4876b2ba8.png" width="1280"/>
    </p>
    <p id="ucc786bec">
     <strong>
      ③ 第三步：编译Vue程序，自动将生成html css js放入内置服务器，自动启动服务。
     </strong>
    </p>
    <p id="u479d9485">
     1) dos命令窗口中切换到项目根：cd vue_pro
    </p>
    <p id="u717fb146">
     2) 执行：npm run serve，这一步会编译HelloWorld案例
    </p>
    <p id="u7b282dc6">
    </p>
    <p class="img-center">
     <img alt="" height="463" id="u45e06550" src="https://i-blog.csdnimg.cn/img_convert/1570a35920c486c5cd1cc3e2639e21b8.png" width="1280"/>
    </p>
    <p id="u34260234">
     ctrl + c停止服务。
    </p>
    <p id="uca8a9298">
     3) 打开浏览器，访问：
     <a href="http://localhost:8080" rel="nofollow" title="http://localhost:8080">
      http://localhost:8080
     </a>
    </p>

     第二种方式

    <p id="u40860bf2">
     点击左上角wordvue选择
     <a href="https://so.csdn.net/so/search?q=Vue%E9%A1%B9%E7%9B%AE&amp;spm=1001.2101.3001.7020" title="Vue项目">
      Vue项目

     管理器，打开之后可以创建项目 也可以打开之前已有的项目
