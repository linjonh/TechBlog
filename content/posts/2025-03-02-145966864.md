    <h3>
     前言
    </h3>
    <p>
     随着人工智能技术的迅猛发展，越来越多的开发者希望在本地环境中部署和调用 AI 模型，以满足特定的业务需求。本文将详细介绍如何在本地环境中使用 Ollama 配置 DeepSeek 模型，并在 IntelliJ IDEA 中创建一个 Spring Boot 项目来调用该模型。通过这些步骤，您将能够在本地环境中高效地运行和测试 AI 模型，提升开发效率。
    </p>
    <h3>
     详细步骤
    </h3>
    <h4>
     一、本地配置DeepSeek
    </h4>
    <h5>
     1.安装Ollama
    </h5>
    <p>
     Ollama 是一个开源平台，旨在简化大型语言模型的本地部署和管理。您可以从
     <a class="link-info" href="https://ollama.com/" rel="nofollow" title="Ollama官网">
      Ollama官网
     </a>
     下载适用于 Windows、Linux 或 macOS 的安装包。
    </p>
    <p>
     <img alt="" height="991" src="https://i-blog.csdnimg.cn/direct/06917c30fa284e85ac0eafbb6be38cda.png" width="1920"/>
    </p>
    <p>
     安装成功后可以使用win+R并输入cmd打开终端，接着输入ollama检查是否安装成功，如果有输出对应信息则说明安装成功了
    </p>
    <p>
     <img alt="" height="590" src="https://i-blog.csdnimg.cn/direct/40ef87de31bb4a0da0b540c007d95f65.png" width="782"/>
    </p>
    <h5>
     2.下载对应DeepSeek模型并运行
    </h5>
    <p>
     首先选中想要配置的DeepSeek版本并复制右侧指令
    </p>
    <p>
     <img alt="" height="987" src="https://i-blog.csdnimg.cn/direct/611cdcfe926b4e29a9eb3bfb7b1bfbc6.png" width="1920"/>
    </p>
    <p>
     将指令复制到终端并进行下载对应版本的DeepSeek，下载完成之后到这个界面，就是下载成功了，可以输入一些信息进行测试
    </p>
    <p>
     <img alt="" height="633" src="https://i-blog.csdnimg.cn/direct/db3fd05e5fe84b13aa64663c0ddc7cbe.png" width="1221"/>
    </p>
    <h4>
     二、SpringBoot项目调用本地DeepSeek
    </h4>
    <h5>
     1.创建springboot项目
    </h5>
    <p>
     具体操作如下：
    </p>
    <ul>
     <li>
      打开 IntelliJ IDEA，选择 "New Project"。
     </li>
     <li>
      选择 "Spring Initializr" 作为项目类型。
     </li>
     <li>
      填写项目的基本信息，如 Group、Artifact 等。
     </li>
     <li>
      在 "Dependencies" 中添加所需的依赖，例如 "Spring Web"。
     </li>
     <li>
      点击 "Create" 创建项目。
     </li>
    </ul>
    <p>
     这里我添加了Web和Ollama的依赖，创建成功界面如下
    </p>
    <p>
     <img alt="" height="1080" src="https://i-blog.csdnimg.cn/direct/8c86b5accde8483487b7f7ff71268efb.png" width="1920"/>
    </p>
    <h5>
     2.添加deepseek对应配置信息
    </h5>
    <p>
     在application.properties中添加下面配置信息
    </p>
    <pre><code class="language-XML">spring.ai.ollama.chat.options.model=deepseek-r1:1.5b
spring.ai.ollama.base-url=http://127.0.0.1:11434
spring.ai.ollama.chat.enabled=true
server.port=9099</code></pre>
    <p>
     <img alt="" height="564" src="https://i-blog.csdnimg.cn/direct/8f433fc629d5403296548a01372d4af7.png" width="1229"/>
    </p>
    <h5>
     3.编码调用deepseek
    </h5>
    <p>
     接着创建简单controller类和service实现类，结构如图：
    </p>
    <p>
     <img alt="" height="282" src="https://i-blog.csdnimg.cn/direct/a293e18ad902464b804d5cb77f4045c4.png" width="508"/>
    </p>
    <p>
     对应代码如下：
     <br/>
     controller:
    </p>
    <pre><code class="language-java">@RestController
public class testController {

    @Autowired
    private DeepSeekTestService deepSeekTestService;

    @RequestMapping("/ask1")
    public String speak(@RequestParam String msg){
        return deepSeekTestService.getResponse(msg);
    }
}</code></pre>
    <p>
     service接口：
    </p>
    <pre><code class="language-java">public interface DeepSeekTestService {
    String getResponse(String message);
}
</code></pre>
    <p>
     service实现类
    </p>
    <pre><code class="language-java">@Service
public class DeepSeekServiceImpl implements DeepSeekTestService {

    private final OllamaChatModel ollamaChatModel;

    public DeepSeekServiceImpl(OllamaChatModel ollamaChatModel) {
        this.ollamaChatModel = ollamaChatModel;
    }

    @Override
    public String getResponse(String message) {
        String response = ollamaChatModel.call(message);
        return response;
    }
}</code></pre>
    <h5>
     4.测试
    </h5>
    <p>
     <br/>
     启动项目，在浏览器的url路径中输入对应信息进行测试,这样一个简单的springboot对接deepseek项目就完成了！
    </p>
