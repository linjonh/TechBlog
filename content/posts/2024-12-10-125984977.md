# mac如何通过VScode连接远程服务器

## 首先本地生成ssh的公钥

```bash
ssh-keygen -t rsa -C  'your email@xxx.com'

```

## 查看公钥

```bash
cat ~/.ssh/.ssh/id_rsa.pub

```

## 在远程服务器上添加公钥

###### 在authorized\_keys文件添加本地的公钥

```bash
vim ~/.ssh/authorized_keys

```

###### 如不存在，则要先创建，在添加

```bash
mkdir .ssh
touch .ssh/authorized_keys
vim .ssh/authorized_keys

```
