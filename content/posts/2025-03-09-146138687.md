     <h2>
      <a id="_Erlang__Rabbitmq__3">
      </a>
      下载 Erlang 和 Rabbitmq 安装包
     </h2>
     <p>
      下载前，需要查看 RabbitMQ 和 Erlang 的对应版本，版本不对应会有问题，可打开此链接：
      <a href="https://www.rabbitmq.com/docs/which-erlang" rel="nofollow">
       Erlang Version Requirements | RabbitMQ
      </a>
      查看版本对应关系。
     </p>
     <p>
      比如我下载的rabbitmq 是 3.10.11 版本，那它适配的 erlang 最低是24.2，最高时25.2，所以我下载的 erlang 是 25.1.1 的：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6beac53203a54d128a6c6e07756dfa74.png"/>
     </p>
     <p>
      erlang 下载地址：
      <a href="https://www.erlang.org/" rel="nofollow">
       https://www.erlang.org/
      </a>
      <br/>
      Rabbitmq 下载地址：
      <a href="https://github.com/rabbitmq/rabbitmq-server/releases/tag/v3.10.11">
       https://github.com/rabbitmq/rabbitmq-server/releases/tag/v3.10.11
      </a>
      ，需要哪个版本，就在地址最后修改就可以了。
     </p>
     <p>
      下载压缩包保存到本地：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/193e3bc9dc86422fbf9f863bc1a3b50d.png"/>
     </p>
     <h2>
      <a id="CentOS_7__14">
      </a>
      CentOS 7 中安装
     </h2>
     <p>
      通过 “rz” 命令将两个压缩包上传到 CentOS 7 虚拟机上，对其进行解压安装
     </p>
     <h3>
      <a id="1_Erlang_16">
      </a>
      （1）安装 Erlang
     </h3>
     <ul>
      <li>
       安装 Linux 依赖库：
       <pre><code class="prism language-bash">yum <span class="token parameter variable">-y</span> <span class="token function">install</span> <span class="token function">make</span> gcc gcc-c++ kernel-devel m4 ncurses-devel openssl-devel
</code></pre>
      </li>
      <li>
       解压 erlang 压缩包文件：
       <pre><code class="prism language-bash"><span class="token function">tar</span> <span class="token parameter variable">-zxvf</span> otp_src_25.1.1.tar.gz
</code></pre>
      </li>
      <li>
       配置：
       <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> otp_src_25.1.1
./configure
</code></pre>
      </li>
      <li>
       编译
       <pre><code class="prism language-bash"><span class="token function">make</span>
</code></pre>
      </li>
      <li>
       安装
       <pre><code class="prism language-bash"><span class="token function">make</span> <span class="token function">install</span>
</code></pre>
      </li>
      <li>
       删除 erlang 的解压的文件夹：
       <pre><code class="prism language-bash"><span class="token function">rm</span> <span class="token parameter variable">-rf</span> otp_src_25.1.1
</code></pre>
      </li>
     </ul>
     <p>
      输入 erl 命令，出现下面形式，说明 erlang 安装成功：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/76b72e8381f84e0386eaaebf6cac7616.png"/>
     </p>
     <h3>
      <a id="2_Rabbitmq_45">
      </a>
      （2）安装 Rabbitmq
     </h3>
     <p>
      解压 Rabbitmq 压缩包，指定解压到 /usr/local 目录下：
     </p>
     <pre><code class="prism language-bash"><span class="token function">tar</span> <span class="token parameter variable">-xvf</span> rabbitmq-server-generic-unix-3.10.11.tar.xz <span class="token parameter variable">-C</span> /usr/local
</code></pre>
     <p>
      进入解压后的 rabbitmq 文件夹中，进入 sbin 目录，
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8345a097269e4457bcd01bc17015e41d.png">
       <br/>
       执行 rabbitmq-server，就可以启动 Rabbitmq。
      </img>
     </p>
     <h2>
      <a id="_54">
      </a>
      配置环境变量
     </h2>
     <p>
      打开 /etc/profile 文件，并将指标指到最后一行：
     </p>
     <pre><code class="prism language-bash"><span class="token function">vi</span> +$ /etc/profile
</code></pre>
     <p>
      后面加入：
     </p>
     <pre><code class="prism language-bash"><span class="token assign-left variable">RABBIT_HOME</span><span class="token operator">=</span>/usr/local/rabbitmq_server-3.10.11
<span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token environment constant">$PATH</span><span class="token builtin class-name">:</span><span class="token variable">$RABBIT_HOME</span>/sbin

<span class="token builtin class-name">export</span> RABBIT_HOME <span class="token environment constant">PATH</span>
</code></pre>
     <p>
      然后 esc，:wq 保存退出。
     </p>
     <p>
      刷新：
     </p>
     <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> /etc/profile
</code></pre>
     <p>
      此时，配置过后，在任何一个目录下都可以启动：
     </p>
     <pre><code class="prism language-bash">rabbitmq-server <span class="token parameter variable">-detached</span>
</code></pre>
     <p>
      查看 rabbitmq 状态：
     </p>
     <pre><code class="prism language-bash">rabbitmqctl status
</code></pre>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e371d181bc4e48e6958e53ba968fd7dd.png">
       <br/>
       这样就是启动成功了。
      </img>
     </p>
     <h2>
      <a id="_86">
      </a>
      用户管理
     </h2>
     <p>
      用户管理：
     </p>
     <pre><code class="prism language-bash"><span class="token comment"># 查看帮助</span>
rabbitmqctl add_user <span class="token parameter variable">--help</span>
<span class="token comment"># 查看当前用户列表</span>
rabbitmqctl list_users
<span class="token comment"># 新增一个用户</span>
rabbitmqctl add_user Username Password
</code></pre>
     <p>
      设置用户角色：
     </p>
     <pre><code class="prism language-bash">rabbitmqctl set_user_tags User Tag
<span class="token comment"># rabbitmqctl set_user_tags admin administrator</span>
</code></pre>
     <p>
      设置用户权限：
     </p>
     <pre><code class="prism language-bash">rabbitmqctl set_permissions <span class="token parameter variable">-p</span> / admin <span class="token string">".*"</span> <span class="token string">".*"</span> <span class="token string">".*"</span>
</code></pre>
     <h2>
      <a id="web__106">
      </a>
      web 后台管理
     </h2>
     <p>
      Rabbitmq 有一个 Web 管理后台，是以插件的方式提供的。
      <br/>
      启用管理后台，切换到 sbin 目录下：
     </p>
     <pre><code class="prism language-bash"><span class="token comment"># 查看 rabbitmq 插件列表</span>
./rabbitmq-plugins list
<span class="token comment"># 启用</span>
./rabbitmq-plugins <span class="token builtin class-name">enable</span> rabbitmq_management
<span class="token comment"># 禁用</span>
./rabbitmq-plugins disable rabbitmq_management
</code></pre>
     <p>
      启用管理后台后， rabbitmqctl stauts 查看状态：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/753b81df5b1b4b95bd6e2a72f73f4395.png"/>
     </p>
     <p>
      防火墙操作：
     </p>
     <pre><code class="prism language-bash"><span class="token comment"># 检查防火墙状态</span>
systemctl status firewalld
<span class="token comment"># 关闭防火墙</span>
systemctl stop firewalld
</code></pre>
     <p>
      访问：http://虚拟机地址:15672
      <br/>
      会出现一下页面：
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6e7fc4c93aa74a1aa01c5b7af7ef68f2.png">
       <br/>
       填写前面设置的用户名和密码，即可登录：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a2c24548ec084540b1a9864657915151.png"/>
      </img>
     </p>
