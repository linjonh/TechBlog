    <p>
    </p>
    <p>
     <span id="cke_bm_3568C" style="display:none">
     </span>
     <span id="cke_bm_3797C" style="display:none">
     </span>
     主机号（Host ID）
    </p>
    <p>
    </p>
    <ul>
     <li>
      <p>
       <strong>
        IP地址简介
       </strong>
       ：IP地址是每台接入互联网的设备所拥有的唯一标识符，类似于电话号码的分层结构，由
       <strong>
        网络号
       </strong>
       和
       <strong>
        主机号
       </strong>
       组成。为了便于记忆，32位二进制的IP地址通常以点分十进制表示。
      </p>
     </li>
     <li>
      <p>
       网络号（Network I有
      </p>
     </li>
     <li>
      <p>
       <strong>
        定义
       </strong>
       ：网络号是IP地址中用于标识设备所属网络的那一部分。它告诉人们设备位于哪个网络中，就像家庭住址中的街道名称。
      </p>
     </li>
     <li>
      <p>
       <strong>
        作用
       </strong>
       ：在网络通信中，网络号用于将互联网上的主机分配到不同的网络中，避免地址冲突，确保数据包能够正确地传输到目标网络。
      </p>
     </li>
     <li>
      <p>
       <strong>
        定义
       </strong>
       ：主机号是IP地址中用于区分同一网络中不同设备的那一部分，就像家庭住址中的门牌号，用于标识网络中的具体设备。
      </p>
     </li>
     <li>
      <p>
       <strong>
        作用
       </strong>
       ：在同一网络中，主机号用于唯一标识一台计算机或设备，使得网络中的设备能够相互识别和通信。
      </p>
     </li>
     <li>
      <p>
       <strong>
        示例
       </strong>
       ：在IP地址192.168.1.10中，如果子网掩码是255.255.255.0，那么主机号是0.0.0.10。
      </p>
      <ul>
       <li>
        <p>
         <strong>
          示例
         </strong>
         ：在IP地址192.168.1.10中，如果子网掩码是255.255.255.0，那么网络号是192.168.1.0。
        </p>
       </li>
       <li>
        <p>
         <strong>
          IP地址分类
         </strong>
         ：
        </p>
        <ul>
         <li>
          <strong>
           A类
          </strong>
          ：首位为“0”，网络号占8位（1-126可用），主机号占24位，支持16777216个主机。
         </li>
         <li>
          <strong>
           B类
          </strong>
          ：前两位为“10”，网络号占16位（128.0-191.255可用），主机号占16位，支持65536个主机。
         </li>
         <li>
          <strong>
           C类
          </strong>
          ：前三位为“110”，网络号占24位（192.0.0-223.255.255可用），主机号占8位，支持256个主机。
          <ul>
           <li>
            还有D类和E类地址，分别用于多播和保留用途。
            <img alt="" height="661" src="https://i-blog.csdnimg.cn/direct/0535a8b9a89b4016a674384e00dea6a9.png" width="674"/>
           </li>
          </ul>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          公有与私有地址
         </strong>
         ：
        </p>
        <ul>
         <li>
          <strong>
           公有地址
          </strong>
          由Inter NIC分配，全球唯一，直接访问互联网。
         </li>
         <li>
          <strong>
           私有地址
          </strong>
          为非注册地址，仅限内部使用，包括A类（10.0.0.0-10.255.255.255）、B类（172.16.0.0-172.31.255.255）和C类（192.168.0.0-192.168.255.255）。
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ul>
