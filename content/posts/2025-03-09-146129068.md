    <h3>
     <a id="_2">
     </a>
     简介
    </h3>
    <p>
     记得大概在2019年的时候吧，我写过一篇关于如何在ubtuntu下安装opencv的文章，反响还是挺好的很多人都看过。
     <a href="https://blog.csdn.net/public669/article/details/99044895?spm=1001.2014.3001.5502">
      ubuntu安装opencv的正确方式
     </a>
    </p>
    <p>
     那个是安装在全局的环境中的，现在呢考虑到项目的迁移性，所以希望可以将opencv当一个三方库在项目中使用，这样的话更加的灵活。
     <br/>
     那么下面呢就介绍如何操作吧！
    </p>
    <h3>
     <a id="opencv_source_7">
     </a>
     下载opencv source
    </h3>
    <p>
     如题所示就是到opencv的官网下载你需要的版本
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0a38e94983484d80a66877eb275cc58b.png">
      <br/>
      我这里下载的是4.5.4的版本，然后在本地进行解压
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/077a40e5b0dd443fa400f996369c7a37.png">
       <br/>
       解压完成以后，就需要给解压出来的文件放到对应项目的对应文件夹中
       <br/>
       那么我这里是给它放到了third_party文件中了
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c3ba55d30f0648809276e9ce2a21a324.png"/>
      </img>
     </img>
    </p>
    <h3>
     <a id="_15">
     </a>
     编译
    </h3>
    <p>
     完成了上面的操作以后，就需要开始进行编译了。编译的时候呢也需要有一些前期的准备工作。
     <br/>
     首先就是需要在opencv-4.5.4的目录下新建一个build文件夹，这里我建的是opencv_build这个目录
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/77c634532f6d4594a4799b688b379be9.png">
      <br/>
      完成了这个操作以后呢，就需要进入到这个build目录下
     </img>
    </p>
    <pre><code class="prism language-bash">opencv_build
</code></pre>
    <p>
     接着就是执行下面的命令进行如下的操作
    </p>
    <pre><code class="prism language-bash">cmake <span class="token punctuation">..</span>/ <span class="token parameter variable">-DCMAKE_INSTALL_PREFIX</span><span class="token operator">=</span><span class="token punctuation">..</span>/opencv_install <span class="token parameter variable">-DBUILD_SHARED_LIBS</span><span class="token operator">=</span>ON
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/41a1c0c9138e4d998ff7ed784a0f1cd7.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c7b5258159ca439eab06bd6131ecd69c.png">
       <br/>
       然后就是需要进行编译了，需要根据自己的设备性能进行编译：
      </img>
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">make</span> <span class="token parameter variable">-j8</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/879ab21425bc436795852161f981a47b.png">
      <br/>
      这个过程可能需要一段时间，大概10分钟以内可以完成。
      <br/>
      看到如下的结果表示成功
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4a31715bb11145c29410a00f06679d77.png"/>
     </img>
    </p>
