    <h4>
     <a id="Tomcat__0">
     </a>
     Tomcat 新手入门指南
    </h4>
    <p>
     Apache Tomcat 是一个开源的 Java Servlet 容器和 Web 服务器，广泛用于部署和运行 Java Web 应用程序。以下是 Tomcat 的入门指南，帮助你快速上手。
    </p>
    <h5>
     <a id="1__Tomcat_4">
     </a>
     1. 安装 Tomcat
    </h5>
    <p>
     <strong>
      步骤 1: 下载 Tomcat
     </strong>
    </p>
    <ul>
     <li>
      访问
      <a href="https://tomcat.apache.org/" rel="nofollow">
       Apache Tomcat 官网
      </a>
      。
     </li>
     <li>
      选择适合的版本（如 Tomcat 9.x）并下载压缩包（zip 或 tar.gz）。
     </li>
    </ul>
    <p>
     <strong>
      步骤 2: 解压文件
     </strong>
    </p>
    <ul>
     <li>
      将下载的压缩包解压到指定目录，如
      <code>
       /opt/tomcat
      </code>
      或
      <code>
       C:\tomcat
      </code>
      。
     </li>
    </ul>
    <p>
     <strong>
      步骤 3: 设置环境变量
     </strong>
    </p>
    <ul>
     <li>
      在 Linux/macOS 上，编辑
      <code>
       ~/.bashrc
      </code>
      或
      <code>
       ~/.zshrc
      </code>
      文件，添加：
      <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">CATALINA_HOME</span><span class="token operator">=</span>/opt/tomcat
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token environment constant">$PATH</span><span class="token builtin class-name">:</span><span class="token variable">$CATALINA_HOME</span>/bin
</code></pre>
     </li>
     <li>
      在 Windows 上，通过系统属性设置
      <code>
       CATALINA_HOME
      </code>
      环境变量。
     </li>
    </ul>
    <p>
     <strong>
      步骤 4: 启动 Tomcat
     </strong>
    </p>
    <ul>
     <li>
      在 Linux/macOS 上，运行：
      <pre><code class="prism language-bash"><span class="token variable">$CATALINA_HOME</span>/bin/startup.sh
</code></pre>
     </li>
     <li>
      在 Windows 上，运行：
      <pre><code class="prism language-bash">%CATALINA_HOME%<span class="token punctuation">\</span>bin<span class="token punctuation">\</span>startup.bat
</code></pre>
     </li>
    </ul>
    <p>
     <strong>
      步骤 5: 验证安装
     </strong>
    </p>
    <ul>
     <li>
      打开浏览器，访问
      <code>
       http://localhost:8080
      </code>
      ，看到 Tomcat 欢迎页面即表示安装成功。
     </li>
    </ul>
    <h5>
     <a id="2__Web__34">
     </a>
     2. 部署 Web 应用
    </h5>
    <p>
     <strong>
      步骤 1: 准备 WAR 文件
     </strong>
    </p>
    <ul>
     <li>
      将 Java Web 应用打包为 WAR 文件，如
      <code>
       myapp.war
      </code>
      。
     </li>
    </ul>
    <p>
     <strong>
      步骤 2: 部署 WAR 文件
     </strong>
    </p>
    <ul>
     <li>
      将 WAR 文件复制到
      <code>
       $CATALINA_HOME/webapps
      </code>
      目录。
     </li>
     <li>
      Tomcat 会自动解压并部署应用。
     </li>
    </ul>
    <p>
     <strong>
      步骤 3: 访问应用
     </strong>
    </p>
    <ul>
     <li>
      打开浏览器，访问
      <code>
       http://localhost:8080/myapp
      </code>
      。
     </li>
    </ul>
    <h5>
     <a id="3__Tomcat_46">
     </a>
     3. 配置 Tomcat
    </h5>
    <p>
     <strong>
      步骤 1: 修改端口
     </strong>
    </p>
    <ul>
     <li>
      编辑
      <code>
       $CATALINA_HOME/conf/server.xml
      </code>
      ，找到
      <code>
       &lt;Connector&gt;
      </code>
      标签，修改
      <code>
       port
      </code>
      属性：
      <pre><code class="prism language-xml"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>Connector</span> <span class="token attr-name">port</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>8080<span class="token punctuation">"</span></span> <span class="token attr-name">protocol</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>HTTP/1.1<span class="token punctuation">"</span></span> <span class="token attr-name">...</span> <span class="token punctuation">/&gt;</span></span>
</code></pre>
     </li>
    </ul>
    <p>
     <strong>
      步骤 2: 配置用户角色
     </strong>
    </p>
    <ul>
     <li>
      编辑
      <code>
       $CATALINA_HOME/conf/tomcat-users.xml
      </code>
      ，添加管理员用户：
      <pre><code class="prism language-xml"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>role</span> <span class="token attr-name">rolename</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>manager-gui<span class="token punctuation">"</span></span><span class="token punctuation">/&gt;</span></span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>user</span> <span class="token attr-name">username</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>admin<span class="token punctuation">"</span></span> <span class="token attr-name">password</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>password<span class="token punctuation">"</span></span> <span class="token attr-name">roles</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>manager-gui<span class="token punctuation">"</span></span><span class="token punctuation">/&gt;</span></span>
</code></pre>
     </li>
    </ul>
    <p>
     <strong>
      步骤 3: 重启 Tomcat
     </strong>
    </p>
    <ul>
     <li>
      修改配置后，重启 Tomcat 使更改生效。
     </li>
    </ul>
    <h5>
     <a id="4__Tomcat_64">
     </a>
     4. 管理 Tomcat
    </h5>
    <p>
     <strong>
      步骤 1: 使用管理界面
     </strong>
    </p>
    <ul>
     <li>
      访问
      <code>
       http://localhost:8080/manager/html
      </code>
      ，使用配置的用户名和密码登录，管理应用。
     </li>
    </ul>
    <p>
     <strong>
      步骤 2: 查看日志
     </strong>
    </p>
    <ul>
     <li>
      日志文件位于
      <code>
       $CATALINA_HOME/logs
      </code>
      目录，如
      <code>
       catalina.out
      </code>
      。
     </li>
    </ul>
    <p>
     <strong>
      步骤 3: 停止 Tomcat
     </strong>
    </p>
    <ul>
     <li>
      在 Linux/macOS 上，运行：
      <pre><code class="prism language-bash"><span class="token variable">$CATALINA_HOME</span>/bin/shutdown.sh
</code></pre>
     </li>
     <li>
      在 Windows 上，运行：
      <pre><code class="prism language-bash">%CATALINA_HOME%<span class="token punctuation">\</span>bin<span class="token punctuation">\</span>shutdown.bat
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="5__82">
     </a>
     5. 常见问题
    </h5>
    <p>
     <strong>
      问题 1: 端口冲突
     </strong>
    </p>
    <ul>
     <li>
      确保 8080 端口未被占用，或修改
      <code>
       server.xml
      </code>
      中的端口号。
     </li>
    </ul>
    <p>
     <strong>
      问题 2: 权限不足
     </strong>
    </p>
    <ul>
     <li>
      在 Linux/macOS 上，确保 Tomcat 目录有适当的读写权限。
     </li>
    </ul>
    <p>
     <strong>
      问题 3: 应用无法访问
     </strong>
    </p>
    <ul>
     <li>
      检查 WAR 文件是否正确部署，或查看日志文件排查问题。
     </li>
    </ul>
    <h4>
     <a id="_93">
     </a>
     总结
    </h4>
    <p>
     通过以上步骤，你应该能够成功安装、配置和管理 Tomcat，并部署 Java Web 应用。随着经验的积累，你可以进一步探索 Tomcat 的高级功能。
    </p>
