    <p>
     在 Python 中，
     <code>
      CPR
     </code>
     （
     <code>
      py-cpr
     </code>
     ）库用于与 HTTP 代理进行配合，编写爬虫程序是一个常见的任务。你可以通过
     <code>
      CPR
     </code>
     库来发送 HTTP 请求并通过代理服务器来抓取数据。以下是如何使用
     <code>
      CPR
     </code>
     库和 HTTP 代理一起编写爬虫程序的示例。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f80025f121a047a7bee04d0e2b494946.png#pic_center"/>
    </p>
    <p>
     <strong>
      1、安装
      <code>
       py-cpr
      </code>
      和
      <code>
       requests
      </code>
      库
     </strong>
    </p>
    <p>
     首先，确保你已安装了
     <code>
      py-cpr
     </code>
     和
     <code>
      requests
     </code>
     库（
     <code>
      requests
     </code>
     用于发送 HTTP 请求）。你可以使用以下命令来安装：
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> py-cpr requests
</code></pre>
    <p>
     <strong>
      2、编写爬虫程序
     </strong>
    </p>
    <p>
     <strong>
      (1) 导入所需模块
     </strong>
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> requests
<span class="token keyword">from</span> cpr <span class="token keyword">import</span> CPR
</code></pre>
    <p>
     <strong>
      (2) 设置 HTTP 代理
     </strong>
    </p>
    <p>
     使用代理时，我们需要设置代理的地址和端口。例如，假设你有一个 HTTP 代理服务，地址为
     <code>
      http://localhost:8080
     </code>
     ，你需要通过代理来抓取网页。
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 设置代理</span>
proxy <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">"http"</span><span class="token punctuation">:</span> <span class="token string">"http://localhost:8080"</span><span class="token punctuation">,</span>
    <span class="token string">"https"</span><span class="token punctuation">:</span> <span class="token string">"http://localhost:8080"</span>
<span class="token punctuation">}</span>

<span class="token comment"># 创建 CPR 对象并配置代理</span>
cpr <span class="token operator">=</span> CPR<span class="token punctuation">(</span>proxies<span class="token operator">=</span>proxy<span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      (3) 发送 HTTP 请求
     </strong>
    </p>
    <p>
     你可以使用
     <code>
      requests
     </code>
     或
     <code>
      CPR
     </code>
     来发送请求。如果你使用
     <code>
      requests
     </code>
     发送请求时通过代理，则请求会通过代理服务器发送。
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 使用 requests 库直接发送请求</span>
response <span class="token operator">=</span> requests<span class="token punctuation">.</span>get<span class="token punctuation">(</span><span class="token string">'https://httpbin.org/ip'</span><span class="token punctuation">,</span> proxies<span class="token operator">=</span>proxy<span class="token punctuation">)</span>

<span class="token comment"># 打印响应内容</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>response<span class="token punctuation">.</span>json<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     或者，你也可以使用
     <code>
      CPR
     </code>
     库来发送请求，
     <code>
      CPR
     </code>
     本质上是对
     <code>
      requests
     </code>
     的封装，它将支持更多代理相关的功能。
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 使用 CPR 发送请求</span>
response <span class="token operator">=</span> cpr<span class="token punctuation">.</span>get<span class="token punctuation">(</span><span class="token string">'https://httpbin.org/ip'</span><span class="token punctuation">)</span>

<span class="token comment"># 打印响应内容</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>response<span class="token punctuation">.</span>json<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     在这个例子中，
     <code>
      https://httpbin.org/ip
     </code>
     会返回你当前请求的 IP 地址。当使用代理时，返回的 IP 地址应该是代理服务器的地址，而不是你的真实 IP 地址。
    </p>
    <p>
     <strong>
      (4) 处理 HTTP 响应
     </strong>
    </p>
    <p>
     在获取响应后，你可以根据需要解析响应数据。例如，如果响应是 JSON 格式，你可以使用
     <code>
      response.json()
     </code>
     来解析。
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 解析并打印 JSON 响应</span>
data <span class="token operator">=</span> response<span class="token punctuation">.</span>json<span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"Your IP via Proxy: "</span><span class="token punctuation">,</span> data<span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      3、完整示例：使用 HTTP 代理抓取网页
     </strong>
    </p>
    <p>
     以下是一个完整的 Python 程序，使用
     <code>
      CPR
     </code>
     和
     <code>
      requests
     </code>
     库，通过 HTTP 代理抓取网页内容并显示网页的标题。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> requests
<span class="token keyword">from</span> cpr <span class="token keyword">import</span> CPR

<span class="token comment"># 设置 HTTP 代理</span>
proxy <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">"http"</span><span class="token punctuation">:</span> <span class="token string">"http://localhost:8080"</span><span class="token punctuation">,</span>
    <span class="token string">"https"</span><span class="token punctuation">:</span> <span class="token string">"http://localhost:8080"</span>
<span class="token punctuation">}</span>

<span class="token comment"># 创建 CPR 对象并配置代理</span>
cpr <span class="token operator">=</span> CPR<span class="token punctuation">(</span>proxies<span class="token operator">=</span>proxy<span class="token punctuation">)</span>

<span class="token comment"># 使用 requests 通过代理发送 GET 请求</span>
response <span class="token operator">=</span> requests<span class="token punctuation">.</span>get<span class="token punctuation">(</span><span class="token string">'https://www.example.com'</span><span class="token punctuation">,</span> proxies<span class="token operator">=</span>proxy<span class="token punctuation">)</span>

<span class="token comment"># 打印响应的 HTML 内容</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>response<span class="token punctuation">.</span>text<span class="token punctuation">)</span>

<span class="token comment"># 或者使用 CPR 发送请求并抓取页面内容</span>
cpr_response <span class="token operator">=</span> cpr<span class="token punctuation">.</span>get<span class="token punctuation">(</span><span class="token string">'https://www.example.com'</span><span class="token punctuation">)</span>

<span class="token comment"># 打印网页内容</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>cpr_response<span class="token punctuation">.</span>text<span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      4、总结
     </strong>
    </p>
    <p>
     通过上述代码示例，你可以看到如何结合
     <code>
      CPR
     </code>
     和 HTTP 代理来编写爬虫程序。
     <code>
      CPR
     </code>
     库封装了 HTTP 请求，提供了方便的接口来通过代理发送请求。你可以根据需求修改代理设置，或者通过
     <code>
      requests
     </code>
     库实现更加复杂的代理配置。
    </p>
    <p>
     <strong>
      主要步骤：
     </strong>
    </p>
    <ol>
     <li>
      设置代理。
     </li>
     <li>
      使用
      <code>
       requests
      </code>
      或
      <code>
       CPR
      </code>
      发送请求。
     </li>
     <li>
      处理 HTTP 响应，解析数据。
     </li>
    </ol>
    <p>
     这种方法适用于通过代理访问受限资源、绕过防火墙或进行匿名爬取等任务。
    </p>
