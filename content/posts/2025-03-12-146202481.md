    <h2>
     <a id="_1">
     </a>
     前言
    </h2>
    <ul>
     <li>
      近期人工智能领域迎来了一位备受瞩目的新星——Manus。Manus 能够独立执行复杂的现实任务，无需人工干预。由于限制原因大部分人无法体验实际效果，但仅过一天 OpenManus 横空出世，让大家能够本地部署并体验 Manus 相同的能力。
     </li>
    </ul>
    <h2>
     <a id="_OpenManus_4">
     </a>
     什么是 OpenManus
    </h2>
    <ul>
     <li>
      OpenManus 是一个开源项目，旨在帮助用户通过简单配置在本地运行智能体，实现各种创意想法。它由 MetaGPT 社区的成员在短短 3 小时内开发完成，展示了团队的高效协作和技术实力。 与需要邀请码的 Manus 不同，OpenManus 无需任何准入门槛，用户只需克隆代码、配置 LLM API 即可快速上手体验。
     </li>
     <li>
      项目地址：https://github.com/mannaandpoem/OpenManus/tree/main
     </li>
    </ul>
    <h2>
     <a id="_8">
     </a>
     构建方式
    </h2>
    <ul>
     <li>
      下面是快速部署 OpenManus 的步骤，实现本地快速部署并体验其强大能力。
     </li>
    </ul>
    <h3>
     <a id="_11">
     </a>
     环境准备
    </h3>
    <ul>
     <li>
      使用 conda 创建独立的 Python 环境，以避免依赖冲突。
     </li>
    </ul>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> open_manus <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.12</span>
conda activate open_manus
</code></pre>
    <h3>
     <a id="_20">
     </a>
     克隆代码仓库
    </h3>
    <ul>
     <li>
      从 GitHub 获取 OpenManus 项目代码。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone https://github.com/mannaandpoem/OpenManus.git
<span class="token builtin class-name">cd</span> OpenManus
</code></pre>
    <h3>
     <a id="_27">
     </a>
     安装依赖
    </h3>
    <ul>
     <li>
      使用 pip 安装所需的 Python 包。
     </li>
    </ul>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token parameter variable">-r</span> requirements.txt
</code></pre>
    <h3>
     <a id="_LLM_API_34">
     </a>
     配置 LLM API
    </h3>
    <ul>
     <li>
      在 config 目录下创建 config.toml 文件，填写所需的 LLM API 信息。
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token function">cp</span> config/config.example.toml config/config.toml
</code></pre>
    <ul>
     <li>
      然后编辑 config.toml，添加 API 密钥和自定义设置：
     </li>
    </ul>
    <pre><code class="prism language-txt"># 全局 LLM 配置
[llm]
model = "gpt-4o"
base_url = "https://api.openai.com/v1"
api_key = "sk-..."  # 替换为真实 API 密钥
max_tokens = 4096
temperature = 0.0
</code></pre>
    <h3>
     <a id="_OpenManus_53">
     </a>
     运行 OpenManus
    </h3>
    <ul>
     <li>
      执行以下命令启动 OpenManus。
     </li>
    </ul>
    <pre><code class="prism language-bash">python main.py
</code></pre>
