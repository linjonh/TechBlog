      <h3>
       <a id="_0">
       </a>
       泛读精读，建立框架
      </h3>
      <ul>
       <li>
        区块链的创新在于老技术的组合创新，也正是区块链技术的系统之美，包括：分布式系统、拜占庭问题、密码学、数据结构、P2P网络等技术，以及博弈论、经济学等思想。针对这个特点要
        <strong>
         系统化学习
        </strong>
        ，具体是
        <strong>
         以泛读精读的方式，先建立知识框架，对区块链有个大致的认识
        </strong>
        。
       </li>
       <li>
        区块链是伴随比特币产生，因此要搞明白区块链，先弄懂比特币
        <ul>
         <li>
          <a href="https://www.zhihu.com/search?q=%E6%99%AE%E6%9E%97%E6%96%AF%E9%A1%BF%E5%A4%A7%E5%AD%A6&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1077996076%7D" rel="nofollow">
           普林斯顿大学
          </a>
          课程
          <a href="https://link.zhihu.com/?target=https%3A//www.coursera.org/learn/cryptocurrency/home/welcome" rel="nofollow">
           Bitcoin and Cryptocurrency Technologies
          </a>
         </li>
         <li>
          <a href="https://link.zhihu.com/?target=http%3A//v1.8btc.com/books/834/masterbitcoin2cn/_book/appdx-bitcoinwhitepaper.html" rel="nofollow">
           《精通比特币》
          </a>
         </li>
         <li>
          <a href="https://link.zhihu.com/?target=https%3A//github.com/xiaolai/bitcoin-whitepaper-chinese-translation" rel="nofollow">
           比特币白皮书
          </a>
         </li>
        </ul>
       </li>
       <li>
        了解作为区块链2.0代表的以太坊
        <ul>
         <li>
          <a href="https://link.zhihu.com/?target=https%3A//github.com/ethereum/wiki/wiki/%5B%E4%B8%AD%E6%96%87%5D-%E4%BB%A5%E5%A4%AA%E5%9D%8A%E7%99%BD%E7%9A%AE%E4%B9%A6" rel="nofollow">
           以太坊白皮书
          </a>
         </li>

           以太坊开发入门指南

         </li>
         <li>
          <a href="https://link.zhihu.com/?target=https%3A//github.com/inoutcode/ethereum_book" rel="nofollow">
           精通以太坊 （中文版）
          </a>
         </li>
         <li>
          区块链技术指南
         </li>
        </ul>
       </li>
       <li>
        这个过程中建立宏观整体的认识，并在日后学习中不断加深、修正和完善框架。同时思考这些问题：
        <ul>
         <li>
          区块链、比特币和以太坊是什么？它们的工作原理大概是如何的？比特币和以太坊的区别？
         </li>
         <li>
          它们具备什么 性质？包含哪些关键的技术点？
         </li>
         <li>
          区块链的发明目的是解决什么问题？除此以外，还能解决什么问题？
         </li>
         <li>
          区块链具备什么优缺点？
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h3>
       <a id="_18">
       </a>
       从外到内，逐一突破
      </h3>
      <ul>
       <li>
        此阶段要：**发挥主观能动性—快速定位你存在疑惑的概念和知识点，用一切方法来攻克它。**在脑海中形成清晰的概念，两个区块链中文社区
        <ul>
         <li>
          <a href="https://link.zhihu.com/?target=https%3A//learnblockchain.cn/" rel="nofollow">
           深入浅出区块链
          </a>
         </li>
         <li>
          <a href="https://link.zhihu.com/?target=https%3A//ethfans.org/" rel="nofollow">
           以太坊爱好者
          </a>
         </li>
        </ul>
       </li>
       <li>
        攻克某一概念时难免遇到其他新的知识点，要注意向下递归的深度
       </li>
       <li>
        重点关注的范围应该是区块链工作原理相关的概念，不必过分追求技术实现细节（存疑）。
        <ul>
         <li>
          去中心化
         </li>
         <li>
          共识机制
         </li>
         <li>
          工作量证明
         </li>
         <li>
          非对称加密
         </li>
         <li>
          硬分叉
         </li>
         <li>
          双花
         </li>
         <li>
          智能合约
         </li>
         <li>
          Merkle Tree
         </li>
         <li>
          51%攻击
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <h3>
       <a id="_35">
       </a>
       从点到面，构建知识网络
      </h3>
      <ul>
       <li>
        <p>
         在前面建立框架的基础上，扩大区块链技术的广度和深度，包括但不限于
        </p>
        <ul>
         <li>
          其他区块链项目，如：Filecoin、Fabric、EOS等
         </li>
         <li>
          不同类型的共识算法
         </li>
         <li>
          零知识证明
         </li>
         <li>
          区块链的可扩展性方案
         </li>
         <li>
          智能合约的编写
         </li>
         <li>
          ……
         </li>
        </ul>
       </li>
       <li>
        <p>
         学习这些东西过程中，不断思考和构建知识之间的联系—这个知识和别的知识有什么关系？是如何关联一起的？
        </p>
        <blockquote>
         <p>
          知识的本质永远不是信息本身，而是信息之间的联系。正是这种联系，涌现出了超越单个信息点总和的「系统性」。
         </p>
        </blockquote>
        <p>
         区块链技术创新的本身恰恰就是
         <strong>
          系统性
         </strong>
        </p>
       </li>
       <li>
        <p>
         所使用的方法有
        </p>
        <ul>
         <li>
          <p>
           对比 如：PoW与PoS之间的区别？

           <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2efe90ecaf901d0c922f0dea8d9007c7.jpeg"/>

         </li>
         <li>
          <p>
           分类 如：目前解决区块链的可行性方案有哪些潜在的研发方向？具体有哪些技术手段？
          </p>
          <p>
           <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/88fcbd345ce492882193dd8c08825c9b.jpeg"/>

         </li>
         <li>
          <p>
           提炼 尝试用最精炼的语言貌似一类相关的知识点，比如比特币的核心原理：
          </p>
          <blockquote>
           <p>
            中本聪使用非对称加密解决电子货币的所有权问题；用区块时间戳解决交易的存在性问题；用分布式账本解决剔除第三方结构后交易的验证问题；用工作量证明和最长链约定来保证节点状态的一致性，已解决「双花」问题。
           </p>
          </blockquote>
         </li>
         <li>
          <p>
           架构 尝试对系统中的关键模块和模块间的关系进行抽象，并绘制成架构图，如：区块链的分层架构。
