    </p>
    <h2>
     <a id="1_C_1">
     </a>
     1 C函数要求
    </h2>
    <p>
     Lua可以调用C语言函数，但这并不意味着Lua可以调用所有的C函数。当Lua调用C函数时，这个C函数必须遵循某种规则来获取参数和返回结果。此外，当Lua调用C函数时，我们必须注册该函数，即必须以一种恰当的方式为Lua提供该C 函数的地址。所有在lua中注册的函数必须遵循下面原型。
    </p>
    <pre><code class="prism language-c"><span class="token keyword">typedef</span> <span class="token keyword">int</span> <span class="token punctuation">(</span><span class="token operator">*</span>lua_CFunction<span class="token punctuation">)</span> <span class="token punctuation">(</span>lua_State <span class="token operator">*</span> L<span class="token punctuation">)</span><span class="token punctuation">;</span>

</code></pre>
    <p>
     举例如下,
    </p>
    <pre><code class="prism language-c"><span class="token comment">// 定义C函数,参数必须为lua_State* L状态,</span>
<span class="token comment">// 返回值为lua函数test的返回值数量,为0.</span>
<span class="token keyword">static</span> <span class="token keyword">int</span> <span class="token function">test</span><span class="token punctuation">(</span> lua_State<span class="token operator">*</span> L<span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"使用lua调用C函数test成功\n"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h2>
     <a id="2_C_19">
     </a>
     2 注册一个C函数
    </h2>
    <p>
     定义好一个C函数后,需使用lua_register函数进行注册后,才能在lua语言中使用。lua_register格式要求如下。
    </p>
    <pre><code class="prism language-c"><span class="token keyword">void</span> <span class="token function">lua_register</span> <span class="token punctuation">(</span>lua_State <span class="token operator">*</span>L<span class="token punctuation">,</span> <span class="token keyword">const</span> <span class="token keyword">char</span> <span class="token operator">*</span>name<span class="token punctuation">,</span> lua_CFunction f<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     举例如下,
    </p>
    <pre><code class="prism language-c"><span class="token comment">//04注册一个C函数.c</span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;stdio.h&gt;</span></span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;stdlib.h&gt;</span></span>

<span class="token comment">// Lua数据类型与函数接口</span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">"lua.h"</span> </span>
<span class="token comment">// Lua与C交互辅助函数接口    </span>

<span class="token comment">// Lua标准库打开接口</span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">"lualib.h"</span>  </span>


<span class="token comment">// 定义C函数,参数必须为lua_State* L状态,</span>
<span class="token comment">// 返回值为lua函数test的返回值数量,为0.</span>
<span class="token keyword">static</span> <span class="token keyword">int</span> <span class="token function">C_test</span><span class="token punctuation">(</span> lua_State<span class="token operator">*</span> L<span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"使用lua调用C函数C_test成功\n"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
    <span class="token comment">// 创建Lua状态</span>
    lua_State<span class="token operator">*</span> L <span class="token operator">=</span> <span class="token function">luaL_newstate</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>           
    <span class="token comment">// 打开标准库</span>
    <span class="token function">luaL_openlibs</span><span class="token punctuation">(</span>L<span class="token punctuation">)</span><span class="token punctuation">;</span>  
	<span class="token comment">// 注册C函数test为lua函数test</span>
	<span class="token function">lua_register</span><span class="token punctuation">(</span>L<span class="token punctuation">,</span><span class="token string">"test"</span><span class="token punctuation">,</span>C_test<span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token comment">// 解析并执行Lua脚本字符串</span>
	<span class="token function">luaL_dostring</span><span class="token punctuation">(</span>L<span class="token punctuation">,</span><span class="token string">"test()"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
	
    <span class="token comment">// 关闭Lua线程</span>
    <span class="token function">lua_close</span><span class="token punctuation">(</span>L<span class="token punctuation">)</span><span class="token punctuation">;</span>

    <h2>
     <a id="3_Clua_68">
     </a>
     3 C语言与lua语言参数传递
    </h2>
    <p>
     上面例子中lua语言调用C函数test,中间没包含参数传递,即函数test既没有参数输入也没有返回值,比较简单。C语言与lua语言参数传递需使用到虚拟栈 lua_State* L进行交换。举例如下,
     <br/>
     lua调用
    </p>
    <pre><code class="prism language-lua"><span class="token comment">-- add为一个用C函数C_add定义的函数</span>
<span class="token comment">-- 当在lua执行下面函数add时,lua语言会将参数7,2,5按顺序压入虚拟栈L,再调用C函数C_add,</span>
<span class="token comment">-- 后根据C函数C_add的返回值,将栈顶弹出传递给lua语言,赋值给变量a</span>
a<span class="token operator">=</span><span class="token function">add</span><span class="token punctuation">(</span><span class="token number">7</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">,</span><span class="token number">5</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     C函数C_add定义如下
    </p>
    <pre><code class="prism language-c"><span class="token comment">// 定义C函数,参数必须为lua_State* L状态</span>

	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"使用lua调用C函数C_add成功\n"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token comment">//使用lua_tointeger按索引获取栈L的元素</span>
	<span class="token comment">//索引1为函数参数1,2为函数参数2,以此类推.</span>
	<span class="token comment">//lua传递参数给C语言需进行数据转换,</span>
	<span class="token comment">//因为这个函数参数均为整数,所以使用lua_tointeger</span>
	<span class="token comment">//若其他数据类型,需使用类似的lua_to*函数</span>
	<span class="token keyword">int</span> a1 <span class="token operator">=</span> <span class="token function">lua_tointeger</span><span class="token punctuation">(</span>L<span class="token punctuation">,</span><span class="token number">1</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

	<span class="token keyword">int</span> a3 <span class="token operator">=</span> <span class="token function">lua_tointeger</span><span class="token punctuation">(</span>L<span class="token punctuation">,</span><span class="token number">3</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token comment">//打印各个参数的值</span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"第1个参数=%d\n"</span><span class="token punctuation">,</span>a1<span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"第2个参数=%d\n"</span><span class="token punctuation">,</span>a2<span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token function">printf</span><span class="token punctuation">(</span><span class="token string">"第3个参数=%d\n"</span><span class="token punctuation">,</span>a3<span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token comment">//将三个参数值相加并压入栈</span>
	<span class="token comment">//使用lua_pushinteger函数将整数值压入,其他数据类型需使用类似lua_push*函数</span>
	<span class="token comment">//若有多个返回值,按顺序压入栈</span>
	<span class="token function">lua_pushinteger</span><span class="token punctuation">(</span>L<span class="token punctuation">,</span>a1<span class="token operator">+</span>a2<span class="token operator">+</span>a3<span class="token punctuation">)</span><span class="token punctuation">;</span>
	<span class="token comment">// 返回值为lua函数add的返回值数量,为1.</span>
	<span class="token keyword">return</span> <span class="token number">1</span><span class="token punctuation">;</span>
