      <span style="font-size: small;">
       <span style="font-family: 宋体;">
        随着计算机应用的迅速普及，其应用已经深入到各行各业。操作系统作为计算机系统的核心与基石，负责支撑应用程序运行环境以及用户操作环境的系统软件。操作系统作为用户与计算机之间的接口肩负着重要使命，它要能够方便用户对计算机硬件的操作来完成各种各样的任务。计算机硬件的不断更新要求操作系统不断改进以更好的利用硬件资源。　随着信息技术（
        <span lang="EN-US">
         IT
        </span>
        ）的飞速发展，特别是互联网的迅速普及，
        <span lang="EN-US">
         3C
        </span>
        （计算机、通讯、消费电子）合一的加速，数字化时代已经来临。嵌入式接入设备是数字化时代的一大主流产品，嵌入式操作系统领域有着巨大的利益，世界各国在此领域开始了激烈竞争，以争取获得主导地位。随着互联网技术的迅猛发展，计算机系统的安全性越来越受到人们的关注。不法分子很可能通过系统漏洞在互联网上窃取个人隐私、商业机密，甚至是国家机密，这会给我们造成重大的损失。“十一五”期间，国家有关研究计划的设立也是我们进一步开展高水平核心软件技术深入研究应用的良好机遇。我们要把握机遇，积极开展自主知识产权的新型操作系统体系结构、构件
        <span lang="EN-US">
         /
        </span>
        中间件技术、跨平台技术等前沿性、前瞻性研究；开展跨学科、跨行业的国内和国际交流与合作，促进新技术的产业化推广。同时我们将发挥本中心在操作系统平台技术上的优势，积极为信息技术研究院的其他研究中心提供技术支撑，在无线通信、数字电视、高速轨道交通等国家重大应用项目的研究与开发方面，使新型操作系统和中间件技术成为自主知识产权的完整解决方案的组成部分。综上所述操作系统的研究于国于民有着重要的意义。

        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         3.
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         国内操作系统
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt 36pt;">
     <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
      <span style="font-size: small;">
       <span style="font-family: 宋体;">
        <span style="font-size: small;">
         说句实在话，从某种意义上来说，中国的操作系统还不能列入到主流的操作系统中去。众所周知目前有四大操作系统平台，
         <span lang="EN-US">
          WINDOWS
         </span>
         是微软一家独占市场，
         <span lang="EN-US">
          UNIX
         </span>
         体系下除了
         <span lang="EN-US">
          IBM
         </span>
         、
         <span lang="EN-US">
          HP
         </span>
         等企业外，其他的几家商场也略有一点份额，
         <span lang="EN-US">
          Solaris
         </span>
         则是
         <span lang="EN-US">
          SUN
         </span>
         公司的独家产品（现在应该是甲骨文了，只有
         <span lang="EN-US">
          Linux
         </span>
         平台上，还能略微看到中国企业的身影如红旗
         <span lang="EN-US">
          Linux
         </span>
         ，麒麟等，可市场上的红帽，
         <span lang="EN-US">
          Novell
         </span>
         等企业早就对这个领域虎视眈眈，准备侵蚀；而桌面领域非
         <span lang="EN-US">
          WINDOWS
         </span>
         的平台只能说极少吧！还有苹果的
         <span lang="EN-US">
          MAC
         </span>
         操作系统，当然因为它是专属于苹果的所以就不能算在主流操作系统之内，通过上述所说我们就可想而知，中国在操作系统层面上处于非常弱势的地位。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <strong>
        </strong>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
      <span style="font-size: small;">
       <span style="font-family: 宋体;">
        <span style="font-size: small;">
         <strong>
          三、操作系统未来发展趋势
         </strong>
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         1
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         、
        </span>
       </span>
      </span>
     </span>
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         网络化趋势加大和安全性增强
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         <span style="mso-spacerun: yes;">
         </span>
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         随着互联网的普及和用户对互联网依赖的加深，一方面各个厂商加快了对网络用户的争夺，另一方面安全性的要求越来越高。操作系统老大微软面对谷歌等互联网公司的挑战，在操作系统层面增加了很多特性以满足用户的需要，如
         <span lang="EN-US">
          Vista
         </span>
         就增加了
         <span lang="EN-US">
          Windows
          <span style="mso-spacerun: yes;">
          </span>
          Rally
         </span>
         等新技术提高用户的网络体验；安全性方面也有较大提升，如
         <span lang="EN-US">
          Windows 7
         </span>
         相比
         <span lang="EN-US">
          Windows XP
         </span>
         增加了过滤平台帐户控制等功能。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         2
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         、　更加绚丽的桌面和
         <span lang="EN-US">
          3D
         </span>
         视觉效果
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         <span style="mso-spacerun: yes;">
         </span>
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         在摩尔定律的支配下，硬件产品的快速发展而价格的大幅下降，软件行业有了更大的发展空间。在新的硬件技术和应用需求的驱动之下，现代的操作系统功能更加强大，界面更绚丽。例如，
         <span lang="EN-US">
          Vista
         </span>
         内置了强大的可以代替
         <span lang="EN-US">
          ghost
         </span>
         的备份工具
         <span lang="EN-US">
          ,
         </span>
         取代系统还原的
         <span lang="EN-US">
          SafeDoc
         </span>
         功能等。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         3
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         、各平台操作系统的竞争变得更加激烈
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         <span style="mso-spacerun: yes;">
         </span>
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         在个人桌面领域，
         <span lang="EN-US">
          Windows
         </span>
         仍然一家独大。一个很有趣的例子就是，
         <span lang="EN-US">
          Windows
         </span>
         跌破
         <span lang="EN-US">
          90%
         </span>
         的占有率就引起了业界的震动，足见
         <span lang="EN-US">
          Windows
         </span>
         对个人桌面领域的垄断地位。出色的产品，成功的营销策略，大量受欢迎的第三方软件造就了今天
         <span lang="EN-US">
          Windows
         </span>
         的地位。但是随着微软反盗版力度加大，各国政府对
         <span lang="EN-US">
          Windows
         </span>
         的不信任以及开源社区的流行，一批优秀
         <span lang="EN-US">
          Linux
         </span>
         厂商如
         <span lang="EN-US">
          Ubuntu
         </span>
         、
         <span lang="EN-US">
          RedHat
         </span>
         的不断创新和进步，
         <span lang="EN-US">
          Linux
         </span>
         系统在桌面大有可为。比如在上网本市场，出于价格上的考虑，上网本中
         <span lang="EN-US">
          30%
         </span>
         预装了
         <span lang="EN-US">
          Linux
         </span>
         。不过，由于巨大
         <span lang="EN-US">
          Windows
         </span>
         用户群和开发人员的存在，
         <span lang="EN-US">
          Windows
         </span>
         已经成为事实上的标准，
         <span lang="EN-US">
          Windows
         </span>
         地位仍然难以撼动。在服务器领域，
         <span lang="EN-US">
          Unix/Linux
         </span>
         系操作系统仍占据着优势的地位，但是随着
         <span lang="EN-US">
          Windows 2008 Server
         </span>
         的发行，服务器领域的争夺将更加白热化。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         4
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         、传统操作系统也许会被边缘化
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         <span style="mso-spacerun: yes;">
         </span>
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         如果现在把
         <span lang="EN-US">
          intel
         </span>
         的
         <span lang="EN-US">
          CPU
         </span>
         更换成
         <span lang="EN-US">
          AMD
         </span>
         的，也许很多人都不能马上发觉，也许不远的未来，同样的情况也会发生在操作系统上。这绝不是危言耸听，操作系统对很多应用和用户来说越来越底层了，很多应用也许只需要一个浏览器就可以了，上网本概念的提出和应用正是有鉴于此。某些互联网公司如谷歌、
         <span lang="EN-US">
          FaceBook
         </span>
         等也提供了开放的
         <span lang="EN-US">
          API
         </span>
         和个性化服务试图成为未来意义上的操作系统。甚至谷歌专为手机设计的
         <span lang="EN-US">
          Android
         </span>
         操作系统也将拓展到上网本领域。微软也即将发布
         <span lang="EN-US">
          Windows Azure
         </span>
         云端操作系统，希望在云计算平台大展身手。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         5
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         、嵌入式操作系统发展状况
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt;">
     <span style="font-size: small;">
      <span style="font-family: 宋体;">
       <span style="font-size: small;">
        <span lang="EN-US" style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         <span style="mso-spacerun: yes;">
         </span>
        </span>
        <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
         随着智能移动终端的发展和人们对随时随地计算能力的迫切需要，嵌入式操作系统大战愈演愈烈。由于没有统一的标准，开发者叫苦不迭，用户也无从适从。诺基亚在完全收购
         <span lang="EN-US">
          Symbian
         </span>
         系统之后以基金会的方式将之开源，希望将
         <span lang="EN-US">
          Symbian
         </span>
         系统成为手机操作系统的标准。谷歌为涉足手机行业提出了
         <span lang="EN-US">
          Android
         </span>
         平台，而基于
         <span lang="EN-US">
          Android
         </span>
         平台的智能手机
         <span lang="EN-US">
          HTCG1
         </span>
         已经在美国正式发布。微软的
         <span lang="EN-US">
          Windows Mobile
         </span>
         平台以及
         <span lang="EN-US">
          Linux
         </span>
         平台经过多年的发展，已经颇为成熟。未来嵌入式操作系统的发展，越来越依赖出色的第三方应用软件的开发，苹果
         <span lang="EN-US">
          iphone
         </span>
         的成功很大一部分就依赖于此。各大平台也尽量努力吸引优秀的开发者，
         <span lang="EN-US">
          Android
         </span>
         平台天生能吸引
         <span lang="EN-US">
          Java
         </span>
         开发者的加盟，
         <span lang="EN-US">
          Windows Mobile
         </span>
         平台下可以方便的用
         <span lang="EN-US">
          c#
         </span>
         开发，诺基亚公司也向开源社区提供资金帮助开发者在
         <span lang="EN-US">
          Symbian
         </span>
         平台的开发。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt; text-indent: 42pt; mso-char-indent-count: 4.0;">
     <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
      <span style="font-size: small;">
       <span style="font-family: 宋体;">
        <span style="font-size: small;">
         当然，实现以上的功能，对于操作系统而言，也是要付出一些代价的，这个代价就是需要主频更高的
         <span lang="EN-US">
          CPU
         </span>
         、更大的内存和 硬盘空间做支持，以上功能才能得到更好的发挥。另外，在未来几年的发展中，
         <span lang="EN-US">
          Windows
         </span>
         和
         <span lang="EN-US">
          Linux
         </span>
         操作系统功能会更加的相似，
         <span lang="EN-US">
          Linux
         </span>
         桌面操作系统的市场份额将会有更大的突破。为适应时代要求和用户需要，操作系统加强了对网络和安全的支持，功能更加强大，各平台操作系统之间的竞争也变得更加激烈。
        </span>
       </span>
      </span>
     </span>
    </p>
    <p class="MsoPlainText" style="margin: 0cm 0cm 0pt; text-indent: 36.75pt; mso-char-indent-count: 3.5;">
     <span style="mso-hansi-font-family: 宋体; mso-bidi-font-family: 宋体;">
      <span style="font-size: small;">
       <span style="font-family: 宋体;">
        <span style="font-size: small;">
         发展更加强大的操作系统，需要全人类的共同努；力。引用屈原名作《离骚》中的一句话“路漫漫其修远兮，吾将上下而求索”。
        </span>
       </span>

    </p>
