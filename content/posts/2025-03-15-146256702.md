    <ol>
     <li>
      <a href="https://www.enterprisedb.com/download-postgresql-binaries" rel="nofollow">
       官网下载 Binaries
      </a>
     </li>
     <li>
      <a href="https://download.osgeo.org/postgis/windows/" rel="nofollow">
       下载 PostGIS 相应版本
      </a>
     </li>
     <li>
      根据说明文本解压缩合并
     </li>
     <li>
      将 C:\Program Files\pgsql-16.8\bin 路径加入环境变量
     </li>
     <li>
      <code>
       initdb -D "C:\Program Files\pgsql-16.8\data" -U postgres
      </code>
      初始化数据库，超级用户
     </li>
     <li>
      要修改该连接方式则修改 data 下的 pg_hba.conf，postgresql.conf
     </li>
     <li>
      <code>
       psql -U postgres -p 12345
      </code>
      用户，端口号连接
     </li>
     <li>
      <code>
       alter user postgres with password '123456';
      </code>
      <br/>
      创建个数据库看看
      <br/>
      <code>
       create database test_db;
      </code>
      <br/>
      \c test_db 连接 test_db
      <br/>
      \q 退出
     </li>
     <li>
      <code>
       pg_ctl start -D "C:\Program Files\pgsql-16.8\data" -l "C:\Program Files\pgsql-16.8\logs\postgresql.log"
      </code>
      启动
     </li>
     <li>
      <code>
       pg_ctl restart -D "C:\Program Files\pgsql-16.8\data" -l "C:\Program Files\pgsql-16.8\logs\postgresql.log"
      </code>
      重启
     </li>
     <li>
      <code>
       pg_ctl stop -D "C:\Program Files\pgsql-16.8\data"
      </code>
      关闭
     </li>
     <li>
      <code>
       pg_ctl status -D "C:\Program Files\pgsql-16.8\data"
      </code>
      查看状态
     </li>
    </ol>
