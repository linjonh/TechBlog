    <blockquote>
     <p>
      模拟轮盘抽奖游戏
      <br/>
      轮盘分为三部分: 一等奖, 二等奖和三等奖;
      <br/>
      轮盘转的时候是随机的,
      <br/>
      如果范围在[0,0.08)之间,代表一等奖,
      <br/>
      如果范围在[0.08,0.3)之间,代表2等奖,
      <br/>
      如果范围在[0, 1.0)之间,代表3等奖,
     </p>
     <p>
      模拟本次活动1000人参加, 模拟游戏时需要准备各等级奖品的个数.
      <br/>
      import random
      <br/>
      print(random.random()) 可以生成0～1之间的数
     </p>
    </blockquote>
    <pre><code>import random

cjdict = {'一等奖': (0, 0.08),
          '二等奖': (0.08, 0.3),
          '三等奖': (0.3, 1.0)
          }

def cjfun():
    n = random.random()
    for k, v in cjdict.items():
        if v[0] &lt;= n &lt; v[1]:
            return k

dict = {}
num = int(input("请输入抽奖次数："))
for i in range(num):
    a = cjfun()
    if a in dict:
        dict[a] += 1
    else:
        dict[a] = 1

for k,v in dict.items():
    print(k,"的数量是",v)
</code></pre>
