    <h2>
     <a id="21__0">
     </a>
     <strong>
      2.1 线性代数（机器学习的核心）
     </strong>
    </h2>
    <p>
     线性代数是机器学习的基础之一，许多核心算法都依赖矩阵运算。本章将介绍线性代数中的基本概念，包括标量、向量、矩阵、矩阵运算、特征值与特征向量，以及奇异值分解（SVD）。
    </p>
    <hr/>
    <h3>
     <a id="211__6">
     </a>
     <strong>
      2.1.1 标量、向量、矩阵
     </strong>
    </h3>
    <h4>
     <a id="1_Scalar_8">
     </a>
     <strong>
      1. 标量（Scalar）
     </strong>
    </h4>
    <p>
     标量是一个单独的数，例如：
    </p>
    <pre><code class="prism language-plaintext">a = 5
</code></pre>
    <p>
     在 Python 中：
    </p>
    <pre><code class="prism language-python">a <span class="token operator">=</span> <span class="token number">5</span>  <span class="token comment"># 标量</span>
</code></pre>
    <h4>
     <a id="2_Vector_18">
     </a>
     <strong>
      2. 向量（Vector）
     </strong>
    </h4>
    <p>
     向量是由多个数值组成的一维数组，例如：
    </p>
    <pre><code class="prism language-plaintext">v = [2, 3, 5]
</code></pre>
    <p>
     Python 实现：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> numpy <span class="token keyword">as</span> np
v <span class="token operator">=</span> np<span class="token punctuation">.</span>array<span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">,</span> <span class="token number">5</span><span class="token punctuation">]</span><span class="token punctuation">)</span>  <span class="token comment"># 一维数组表示向量</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>v<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="3_Matrix_30">
     </a>
     <strong>
      3. 矩阵（Matrix）
     </strong>
    </h4>
    <p>
     矩阵是一个二维数组，例如：
    </p>
    <pre><code class="prism language-plaintext">A = [[1, 2],
     [3, 4]]
</code></pre>
    <p>
     Python 实现：
    </p>
    <pre><code class="prism language-python">A <span class="token operator">=</span> np<span class="token punctuation">.</span>array<span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">]</span><span class="token punctuation">,</span> <span class="token punctuation">[</span><span class="token number">3</span><span class="token punctuation">,</span> <span class="token number">4</span><span class="token punctuation">]</span><span class="token punctuation">]</span><span class="token punctuation">)</span>  <span class="token comment"># 二维数组表示矩阵</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>A<span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h3>
     <a id="212__44">
     </a>
     <strong>
      2.1.2 矩阵运算
     </strong>
    </h3>
    <h4>
     <a id="1__46">
     </a>
     <strong>
      1. 矩阵加法
     </strong>
    </h4>
    <p>
     两个相同形状的矩阵可以相加：
    </p>
    <pre><code class="prism language-plaintext">A + B = [[1, 2],    +   [[5, 6],    =   [[6,  8],
         [3, 4]]         [7, 8]]         [10, 12]]
</code></pre>
    <p>
     Python 计算：
    </p>
    <pre><code class="prism language-python">A <span class="token operator">=</span> np<span class="token punctuation">.</span>array<span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">]</span><span class="token punctuation">,</span> <span class="token punctuation">[</span><span class="token number">3</span><span class="token punctuation">,</span> <span class="token number">4</span><span class="token punctuation">]</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
B <span class="token operator">=</span> np<span class="token punctuation">.</span>array<span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token punctuation">[</span><span class="token number">5</span><span class="token punctuation">,</span> <span class="token number">6</span><span class="token punctuation">]</span><span class="token punctuation">,</span> <span class="token punctuation">[</span><span class="token number">7</span><span class="token punctuation">,</span> <span class="token number">8</span><span class="token punctuation">]</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
C <span class="token operator">=</span> A <span class="token operator">+</span> B
<span class="token keyword">print</span><span class="token punctuation">(</span>C<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="2__60">
     </a>
     <strong>
      2. 矩阵乘法
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       逐元素相乘（Hadamard 乘积）
      </strong>
     </li>
    </ul>
    <pre><code class="prism language-plaintext">A ⊙ B = [[1×5,  2×6],
         [3×7,  4×8]]
       = [[5, 12],
          [21, 32]]
</code></pre>
    <p>
     Python 实现：
    </p>
    <pre><code class="prism language-python">C <span class="token operator">=</span> A <span class="token operator">*</span> B  <span class="token comment"># 逐元素相乘</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>C<span class="token punctuation">)</span>
</code></pre>
    <ul>
     <li>
      <strong>
       矩阵乘法（点积）
      </strong>
     </li>
    </ul>
    <pre><code class="prism language-plaintext">A × B = [[1×5 + 2×7,  1×6 + 2×8],
         [3×5 + 4×7,  3×6 + 4×8]]
       = [[19, 22],
          [43, 50]]
</code></pre>
    <p>
     Python 实现：
    </p>
    <pre><code class="prism language-python">C <span class="token operator">=</span> np<span class="token punctuation">.</span>dot<span class="token punctuation">(</span>A<span class="token punctuation">,</span> B<span class="token punctuation">)</span>  <span class="token comment"># 矩阵乘法</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>C<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="3__87">
     </a>
     <strong>
      3. 矩阵转置
     </strong>
    </h4>
    <p>
     矩阵转置是将行变成列：
    </p>
    <pre><code class="prism language-plaintext">A^T = [[1, 3],
       [2, 4]]
</code></pre>
    <p>
     Python 计算：
    </p>
    <pre><code class="prism language-python">A_T <span class="token operator">=</span> A<span class="token punctuation">.</span>T  <span class="token comment"># 计算转置</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>A_T<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="4__99">
     </a>
     <strong>
      4. 逆矩阵
     </strong>
    </h4>
    <p>
     如果矩阵
     <code>
      A
     </code>
     是可逆的（即
     <code>
      det(A) ≠ 0
     </code>
     ），那么存在一个矩阵
     <code>
      A^(-1)
     </code>
     使得：
    </p>
    <pre><code class="prism language-plaintext">A × A^(-1) = I  (单位矩阵)
</code></pre>
    <p>
     Python 计算：
    </p>
    <pre><code class="prism language-python">A_inv <span class="token operator">=</span> np<span class="token punctuation">.</span>linalg<span class="token punctuation">.</span>inv<span class="token punctuation">(</span>A<span class="token punctuation">)</span>  <span class="token comment"># 计算逆矩阵</span>
<span class="token keyword">print</span><span class="token punctuation">(</span>A_inv<span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h3>
     <a id="213__112">
     </a>
     <strong>
      2.1.3 特征值与特征向量
     </strong>
    </h3>
    <p>
     特征值（Eigenvalue）和特征向量（Eigenvector）在机器学习中用于主成分分析（PCA）等算法。
    </p>
    <h4>
     <a id="1__116">
     </a>
     <strong>
      1. 定义
     </strong>
    </h4>
    <p>
     对于矩阵
     <code>
      A
     </code>
     ，如果存在一个向量
     <code>
      v
     </code>
     和一个数
     <code>
      λ
     </code>
     使得：
    </p>
    <pre><code class="prism language-plaintext">A × v = λ × v
</code></pre>
    <p>
     那么
     <code>
      v
     </code>
     是
     <code>
      A
     </code>
     的特征向量，
     <code>
      λ
     </code>
     是对应的特征值。
    </p>
    <h4>
     <a id="2_Python__123">
     </a>
     <strong>
      2. Python 计算特征值和特征向量
     </strong>
    </h4>
    <pre><code class="prism language-python">eigenvalues<span class="token punctuation">,</span> eigenvectors <span class="token operator">=</span> np<span class="token punctuation">.</span>linalg<span class="token punctuation">.</span>eig<span class="token punctuation">(</span>A<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"特征值:"</span><span class="token punctuation">,</span> eigenvalues<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"特征向量:"</span><span class="token punctuation">,</span> eigenvectors<span class="token punctuation">)</span>
</code></pre>
    <hr/>
    <h3>
     <a id="214_SVD_132">
     </a>
     <strong>
      2.1.4 SVD（奇异值分解）
     </strong>
    </h3>
    <p>
     奇异值分解（Singular Value Decomposition, SVD）是矩阵分解的一种重要方法，它可以表示为：
    </p>
    <pre><code class="prism language-plaintext">A = U × Σ × V^T
</code></pre>
    <p>
     其中：
    </p>
    <ul>
     <li>
      <code>
       U
      </code>
      是左奇异向量矩阵
     </li>
     <li>
      <code>
       Σ
      </code>
      是对角矩阵，对角线上的元素称为奇异值
     </li>
     <li>
      <code>
       V^T
      </code>
      是右奇异向量矩阵的转置
     </li>
    </ul>
    <h4>
     <a id="Python__SVD_143">
     </a>
     <strong>
      Python 计算 SVD
     </strong>
    </h4>
    <pre><code class="prism language-python">U<span class="token punctuation">,</span> S<span class="token punctuation">,</span> V_T <span class="token operator">=</span> np<span class="token punctuation">.</span>linalg<span class="token punctuation">.</span>svd<span class="token punctuation">(</span>A<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"U 矩阵:"</span><span class="token punctuation">,</span> U<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"Σ 矩阵:"</span><span class="token punctuation">,</span> S<span class="token punctuation">)</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"V^T 矩阵:"</span><span class="token punctuation">,</span> V_T<span class="token punctuation">)</span>
</code></pre>
    <p>
     SVD 在降维（如 PCA）中有重要应用，后续章节将深入介绍。
    </p>
