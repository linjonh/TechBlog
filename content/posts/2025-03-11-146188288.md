    <h3>
     <a id="_0">
     </a>
     简介
    </h3>
    <p>
     通过配置使用 ODBC 连接器，Excel 可以快速访问 TDengine 的数据。用户可以将标签数据、原始时序数据或按时间聚合后的时序数据从 TDengine 导入到 Excel，用以制作报表整个过程不需要任何代码编写过程。
    </p>
    <h3>
     <a id="_4">
     </a>
     前置条件
    </h3>
    <p>
     准备以下环境：
    </p>
    <ul>
     <li>
      TDengine 3.3.5.8 以上版本集群已部署并正常运行（企业及社区版均可）。
     </li>
     <li>
      taosAdapter 能够正常运行，详细参考
      <a href="https://docs.taosdata.com/reference/components/taosadapter/" rel="nofollow">
       taosAdapter 参考手册
      </a>
      。
     </li>
     <li>
      Excel 安装并运行, 如未安装，请下载并安装, 具体操作请参考 Microsoft 官方文档。
     </li>
     <li>
      从 TDengine 官网下载最新的 Windows 操作系统 X64 客户端驱动程序并进行安装，详细参考
      <a href="https://blog.csdn.net/ticktick999/article/details/146188531">
       配置ODBC 数据源
      </a>
      。
     </li>
    </ul>
    <h3>
     <a id="_12">
     </a>
     配置数据源
    </h3>
    <p>
     <strong>
      第 1 步
     </strong>
     ，在 Windows 操作系统的开始菜单中搜索并打开【ODBC数据源（64位）】管理工具并进行配置。详细参考
     <a href="https://docs.taosdata.com/reference/connector/odbc/#%E9%85%8D%E7%BD%AE%E6%95%B0%E6%8D%AE%E6%BA%90" rel="nofollow">
      配置ODBC数据源
     </a>
     。
    </p>
    <p>
     <strong>
      第 2 步
     </strong>
     ，在 Windows 系统环境下启动 Excel，之后选择【数据】-&gt;【获取数据】-&gt;【自其他源】-&gt;【从ODBC】。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f8ed78545871474da74877d1d6e15cf9.png"/>
    </p>
    <p>
     <strong>
      第 3 步
     </strong>
     ，在弹出窗口的【数据源名称(DSN)】下拉列表中选择需要连接的数据源后，点击【确定】按钮。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0aaa8a7e49d74a7bb1ce6d8cd4bf4c29.png"/>
    </p>
    <p>
     <strong>
      第 4 步
     </strong>
     ，输入 TDengine 的用户名密码。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/baa10e5c56fe4c55a0d2e37b7e8ad136.png"/>
    </p>
    <p>
     <strong>
      第 5 步
     </strong>
     ，在弹出的【导航器】对话框中，选择要加载的库表, 并点击【加载】完成数据加载。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d209c87316b748e3bb80a3f61839adc3.png"/>
    </p>
    <h3>
     <a id="_34">
     </a>
     数据分析
    </h3>
    <p>
     选中导入的数据，在【插入】选项卡中选择柱状图，并且在右侧的【数据透视图】中配置数据字段。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b0117478002a4d6f9985a94768647ebd.png"/>
    </p>
