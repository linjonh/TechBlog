    <h2>
     <a id="data_1">
     </a>
     数据（data）
    </h2>
    <ul>
     <li>
      定义：数据是
      <strong>
       描述事物的符号记录
      </strong>
      。
     </li>
    </ul>
    <p>
     数据是数据库中存储的基本对象。描述事物的符号可以为数字、文本、图像、音频、视频等。
    </p>
    <h2>
     <a id="DataBase_DB_6">
     </a>
     数据库（DataBase, DB）
    </h2>
    <ul>
     <li>
      定义：数据库是
      <strong>
       长期存储在计算机内、有组织的、可共享的大量数据的集合
      </strong>
      。
     </li>
    </ul>
    <p>
     数据库，顾名思义，是存放数据的仓库。只不过这个仓库是在计算机存储设备上，而且数据是按照一定的格式存放的。总的来讲，数据库数据具有
     <strong>
      永久存储
     </strong>
     、
     <strong>
      有组织
     </strong>
     和
     <strong>
      可共享
     </strong>
     三个基本特点。
    </p>
    <h2>
     <a id="DataBase_Management_System_DBMS_11">
     </a>
     数据库管理系统（DataBase Management System, DBMS）
    </h2>
    <ul>
     <li>
      定义：数据库管理系统是
      <strong>
       位于用户和操作系统之间的一层数据管理软件
      </strong>
      。
     </li>
    </ul>
    <p>
     数据库管理软件的主要任务是如何
     <strong>
      科学地组织和存储数据
     </strong>
     ，如何
     <strong>
      高效地获取和维护数据
     </strong>
     。它的主要功能包括一下几个方面：
    </p>
    <ol>
     <li>
      <strong>
       数据定义功能
      </strong>
      <br/>
      数据库管理系统提供数据定义语言（Data Definition Language,
      <strong>
       DDL
      </strong>
      ），用户通过它可以方便地对数据库中的
      <strong>
       数据对象的组成和结构
      </strong>
      进行定义。
     </li>
     <li>
      <strong>
       数据组织、存储和管理
      </strong>
      <br/>
      数据库管理系统要分类组织、存储和管理各种数据。要确定以何种文件结构和存取方式在存储级上组织这些数据，如何实现数据之间的联系。数据组织和存储的基本目标是提高存储空间利用率和方便存取，提供多种存取方法（如索引查找、hash查找、顺序查找等）来提高存取效率。
     </li>
     <li>
      <strong>
       数据操纵功能
      </strong>
      <br/>
      数据库管理系统提供数据操纵语言（Data Manipulation Language,
      <strong>
       DML
      </strong>
      ），用户可以使用它操纵数据，实现对数据库的基本操作，如查询、插入、删除和修改等。
     </li>
     <li>
      <strong>
       数据库的事务管理和运行管理
      </strong>
      <br/>
      数据库在建立、运用和维护时由数据库管理系统统一管理和控制，以保证事务的正确运行，保证数据的安全性、完整性、多用户对数据的并发使用以及发生故障后的系统恢复。
     </li>
     <li>
      <strong>
       数据库的建立和维护功能
      </strong>
      <br/>
      数据库的建立和维护功能包括数据库初始数据的输入、转换功能，数据库的转储、恢复功能，数据库的重组织功能和性能监视、分析功能等。这些功能通常由一些实用程序或管理工具完成。
     </li>
     <li>
      <strong>
       其他功能
      </strong>
      <br/>
      其他功能包括数据库管理系统与网络中其他软件系统的通信功能，一个数据库管理系统与另一个数据库管理系统或文件系统的数据转换功能，异构数据库之间的互访和互操作功能等。
     </li>
    </ol>
    <h2>
     <a id="DataBase_System_DBS_29">
     </a>
     数据库系统（DataBase System, DBS）
    </h2>
    <ul>
     <li>
      定义：数据库系统是由
      <strong>
       数据库
      </strong>
      、
      <strong>
       数据库管理系统
      </strong>
      （及其应用开发工具）、
      <strong>
       应用程序
      </strong>
      和
      <strong>
       数据库管理员
      </strong>
      （DataBase Administrator，DBA）组成的
      <strong>
       存储、管理、处理和维护数据
      </strong>
      的系统。
     </li>
    </ul>
