    <blockquote>
     <p>
      如何看torch 和torchvision版本，如何看CUDA的版本，我是ubuntu的系统，建立的环境是py38。
     </p>
    </blockquote>
    <p>
     在 Ubuntu 下，你可以使用以下命令来检查
     <strong>
      Torch（PyTorch）
     </strong>
     、
     <strong>
      Torchvision
     </strong>
     以及
     <strong>
      CUDA
     </strong>
     版本信息。
    </p>
    <hr/>
    <h4>
     <strong>
      ✅ 1. 查看 PyTorch 版本
     </strong>
    </h4>
    <p>
     先激活
     <code>
      labelme
     </code>
     环境：
    </p>
    <pre><code>conda activate py38
</code></pre>
    <p>
     然后，运行：
    </p>
    <pre><code>python -c "import torch; print(torch.__version__)"
</code></pre>
    <p>
     如果 PyTorch 已安装，它会返回类似：
    </p>
    <pre><code>2.1.0+cu121
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        2.1.0
       </code>
      </strong>
      代表 PyTorch 版本
     </li>
     <li>
      <strong>
       <code>
        cu121
       </code>
      </strong>
      代表 CUDA 12.1 版本
     </li>
     <li>
      如果返回
      <code>
       cpu
      </code>
      ，说明 PyTorch 不是 GPU 版本（没有安装 CUDA 相关依赖）
     </li>
    </ul>
    <hr/>
    <h4>
     <strong>
      ✅ 2. 查看 Torchvision 版本
     </strong>
    </h4>
    <pre><code>python -c "import torchvision; print(torchvision.__version__)"
</code></pre>
    <p>
     示例返回：
    </p>
    <pre><code>0.16.0
</code></pre>
    <p>
     代表你的
     <code>
      torchvision
     </code>
     版本是
     <code>
      0.16.0
     </code>
     。
    </p>
    <hr/>
    <h4>
     <strong>
      ✅ 3. 查看 CUDA 版本
     </strong>
    </h4>
    <p>
     CUDA 版本有两种：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        PyTorch 内部 CUDA 版本
       </strong>
      </p>
      <pre><code>python -c "import torch; print(torch.version.cuda)"
</code></pre>
      <ul>
       <li>
        返回示例：
        <pre><code>12.1
</code></pre>
       </li>
       <li>
        说明 PyTorch 运行在
        <strong>
         CUDA 12.1
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        系统安装的 CUDA 版本
       </strong>
       如果你在系统上安装了 CUDA，你可以运行：
      </p>
      <pre><code>nvcc --version
</code></pre>
      <ul>
       <li>
        可能返回：
        <pre><code>nvcc: NVIDIA (R) Cuda compiler
release 11.8, V11.8.89
</code></pre>
       </li>
       <li>
        说明系统安装了
        <strong>
         CUDA 11.8
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        检查 GPU 是否支持 CUDA
       </strong>
      </p>
      <pre><code>nvidia-smi
</code></pre>
      <ul>
       <li>
        输出示例：
        <pre><code>+-----------------------------------------------------------------------------+
| NVIDIA-SMI 525.85.12    Driver Version: 525.85.12    CUDA Version: 12.0     |
+-----------------------------------------------------------------------------+
</code></pre>
       </li>
       <li>
        这里
        <code>
         CUDA Version: 12.0
        </code>
        代表
        <strong>
         NVIDIA 驱动支持 CUDA 12.0
        </strong>
        ，但不一定是 PyTorch 使用的版本。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      ✅ 4. 确保 PyTorch 正确使用 GPU
     </strong>
    </h4>
    <p>
     如果你希望 PyTorch 使用 GPU，请运行：
    </p>
    <pre><code>python -c "import torch; print(torch.cuda.is_available())"
</code></pre>
    <ul>
     <li>
      <strong>
       如果返回
       <code>
        True
       </code>
      </strong>
      ，说明 PyTorch
      <strong>
       可以使用 GPU
      </strong>
      。
     </li>
     <li>
      <strong>
       如果返回
       <code>
        False
       </code>
      </strong>
      ，可能：
      <ul>
       <li>
        你安装的 PyTorch 版本不支持 CUDA（只支持 CPU）。
       </li>
       <li>
        你的 NVIDIA 驱动或者 CUDA 版本不匹配。
       </li>
      </ul>
     </li>
    </ul>

      <pre><code>pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
</code></pre>
      这里
      <code>
       cu121
