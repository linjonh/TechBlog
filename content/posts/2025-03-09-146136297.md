    </p>
    <pre><code class="hljs">tar -zxvf prometheus-3.0.1.linux-amd64.tar.gz</code></pre>

     运行

    <h2 style="margin-left:0px; margin-right:0px; text-align:justify">
     Grafana安装

    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     下载地址：

      <span class="link-card-box" contenteditable="false">

        Download Grafana | Grafana Labs

        Overview of how to download and install different versions of Grafana on different operating systems.

       <span class="link-link">
        <img class="link-link-icon" src="https://csdnimg.cn/release/blog_editor_html/release2.3.8/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=P1C7"/>
        https://grafana.com/grafana/download

    <h2 style="background-color:transparent; margin-left:0px; margin-right:0px; text-align:justify">
     Prometheus被监控端安装插件
    </h2>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     下载插件（此处下载一个node 和 mysql插件）
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <a class="has-card" href="https://prometheus.io/download/" rel="nofollow" title="Download | Prometheus">
      <span class="link-card-box" contenteditable="false">
       <span class="link-title">
        Download | Prometheus
       </span>
       <span class="link-desc">
        An open-source monitoring system with a dimensional data model, flexible query language, efficient time series database and modern alerting approach.
       </span>
       <span class="link-link">
        <img class="link-link-icon" src="https://csdnimg.cn/release/blog_editor_html/release2.3.8/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=P1C7"/>
        https://prometheus.io/download/
       </span>
      </span>
     </a>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="483" src="https://i-blog.csdnimg.cn/direct/3a7b1c33130f474999896014aa2733d8.png" width="692"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     在被监控端主机解压这两个文件
    </p>
    <pre><code class="hljs">tar -zxvf node_exporter-1.8.2.linux-amd64.tar.gz </code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <pre><code class="hljs">node_exporter插件可直接运行

cd node_exporter-1.8.2.linux-amd64</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     node_exporter插件可直接运行
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     cd node_exporter-1.8.2.linux-amd64
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="180" src="https://i-blog.csdnimg.cn/direct/cff19bc270514cae8009e42f393d432f.png" width="692"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     nohup ./node_exporter &amp;
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="591" src="https://i-blog.csdnimg.cn/direct/42c51d41aeff43a29cb0c0afdee706bb.png" width="692"/>
    </p>
    <blockquote>
     <p style="margin-left:0; margin-right:0; text-align:justify">
      mysqld_exporter需要连接到MySQL,需要授权
     </p>
     <p style="margin-left:0; margin-right:0; text-align:justify">
     </p>
     <p style="margin-left:0; margin-right:0; text-align:justify">
      mysqld_exporter插件需要创建一个用户+密码的文件
     </p>
    </blockquote>
    <pre><code class="hljs">tar -zxvf mysqld_exporter-0.16.0.linux-amd64.tar.gz</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="82" src="https://i-blog.csdnimg.cn/direct/0b594e793ff84c5682e7a2872d339318.png" width="692"/>
    </p>
    <pre><code class="hljs">cd mysqld_exporter-0.16.0.linux-amd64</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="132" src="https://i-blog.csdnimg.cn/direct/9fdfadd445bc471c8cedd324b009db0a.png" width="692"/>
    </p>
    <pre><code class="hljs">mysql&gt; use mysql;

mysql&gt; create user dbuser;</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="408" src="https://i-blog.csdnimg.cn/direct/4cab73d790b846f98dc7deb3205f50ca.png" width="691"/>
    </p>
    <pre><code class="hljs">mysql&gt; GRANT REPLICATION CLIENT, PROCESS ON *.* TO 'dbuser'@'localhost' identified by '123456';</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="291" src="https://i-blog.csdnimg.cn/direct/ae6cb4f8315b41bf84399e6b6bb3bd99.png" width="692"/>
    </p>
    <pre><code class="hljs">mysql&gt; GRANT SELECT ON performance_schema.* TO 'dbuser'@'localhost';</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="121" src="https://i-blog.csdnimg.cn/direct/00e09861898b4e13a51bf9002f227c6b.png" width="691"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     创建 .my.cnf 加入以下内容
    </p>
    <pre><code class="hljs">[client]

user=dbuser

password=123456</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="95" src="https://i-blog.csdnimg.cn/direct/f5cf451ad7a14e3eb6091e7b9123f801.png" width="351"/>
    </p>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     启动mysqld_exporter
    </p>
    <pre><code class="hljs">nohup ./mysqld_exporter --config.my-cnf=.my.cnf &amp;</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <img alt="" height="240" src="https://i-blog.csdnimg.cn/direct/45d5cf00f3854ab4be9a9fe146f47032.png" width="692"/>
    </p>
    <h2 style="margin-left:0px; margin-right:0px; text-align:justify">
     Prometheus服务端添加被监控端
    </h2>
    <p style="margin-left:0; margin-right:0; text-align:justify">

    <pre><code class="hljs">cd prometheus-3.0.1.linux-amd64</code></pre>
    <pre><code class="hljs">vi prometheus.yml  加入

  - job_name: system-status

    static_configs:

    - targets: ['192.168.126.50:9100']

      labels:

        instance: 7-7



  - job_name: mysql-status

    static_configs:

    - targets: ['192.168.126.40:9104']

      labels:

        instance: 7-7-mysql</code></pre>
