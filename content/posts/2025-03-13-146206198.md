    <h3>
     3.思路
    </h3>
    <ul>
     <li>
      统计每一个字符最后出现的位置
     </li>
     <li>
      从头遍历字符，并更新字符的最远出现下标，如果找到字符最远出现位置下标和当前下标相等了，则找到了分割点
     </li>
    </ul>
    <h2>
     <a class="link-info" href="https://leetcode.cn/problems/merge-intervals/description/" rel="nofollow" title="二、56.合并区间">
      二、56.合并区间
     </a>
    </h2>
    <h3>
     1.题目描述
    </h3>
    <p>
     <img alt="" height="90" src="https://i-blog.csdnimg.cn/direct/991290c6ccd14b4ea4abc5f1c40ce429.png" width="825"/>
    </p>
    <h3>
     2.代码
    </h3>
    <p>
     <img alt="" height="531" src="https://i-blog.csdnimg.cn/direct/274785d8cdb54f368f99f4e1b3ef1240.png" width="829"/>
    </p>
    <h3>
     3.思路
    </h3>
    <p>
     先排序，让所有的相邻区间尽可能的重叠在一起。
    </p>
    <ol>
     <li>
      使用
      <code>
       Arrays.sort
      </code>
      方法对
      <code>
       intervals
      </code>
      数组按照区间的起始位置进行排序。通过
      <code>
       (x, y) -&gt; Integer.compare(x[0], y[0])
      </code>
      作为比较器，确保区间按起始位置从小到大排列。
     </li>
     <li>
      <strong>
       初始化合并区间的起始位置和右边界
      </strong>
      ：将第一个区间的起始位置
      <code>
       intervals[0][0]
      </code>
      赋值给
      <code>
       start
      </code>
      ，作为当前合并区间的起始位置；将第一个区间的结束位置
      <code>
       intervals[0][1]
      </code>
      赋值给
      <code>
       rightmostRightBound
      </code>
      ，作为当前合并区间的右边界。
     </li>
     <li>
      <strong>
       遍历区间数组
      </strong>
      ：从第二个区间开始遍历，对于每个区间
      <code>
       intervals[i]
      </code>
      ：
      <ul>
       <li>
        若当前区间的起始位置
        <code>
         intervals[i][0]
        </code>
        大于当前合并区间的右边界
        <code>
         rightmostRightBound
        </code>
        ，说明当前区间与之前的合并区间不重叠。此时，将之前的合并区间
        <code>
         [start, rightmostRightBound]
        </code>
        添加到结果列表
        <code>
         res
        </code>
        中，并更新
        <code>
         start
        </code>
        为当前区间的起始位置
        <code>
         intervals[i][0]
        </code>
        ，
        <code>
         rightmostRightBound
        </code>
        为当前区间的结束位置
        <code>
         intervals[i][1]
        </code>
        。
       </li>
       <li>
        若当前区间的起始位置
        <code>
         intervals[i][0]
        </code>
        小于或等于当前合并区间的右边界
        <code>
         rightmostRightBound
        </code>
        ，说明当前区间与之前的合并区间重叠。此时，更新
        <code>
         rightmostRightBound
        </code>
        为当前合并区间右边界和当前区间结束位置的较大值，即
        <code>
         rightmostRightBound = Math.max(rightmostRightBound, intervals[i][1])
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       添加最后一个合并区间
      </strong>
      ：遍历结束后，将最后一个合并区间
      <code>
       [start, rightmostRightBound]
      </code>
      添加到结果列表
      <code>
       res
      </code>
      中。
     </li>
     <li>
      <strong>
       返回结果
      </strong>
      ：将结果列表
      <code>
       res
      </code>
      转换为二维数组并返回，该数组即为合并后的区间数组。
     </li>
    </ol>
