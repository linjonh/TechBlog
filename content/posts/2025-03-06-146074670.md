    <h4>
     AI提示词（Prompt）的理解和学习指南
    </h4>
    <h5>
     一、什么是AI提示词？
    </h5>
    <p>
     AI提示词（Prompt）是用户输入给人工智能模型的指令或问题，用于引导模型生成特定类型的回答或内容。它如同与AI沟通的“钥匙”，设计得当的提示词能显著提升输出质量。
    </p>
    <h5>
     二、提示词的核心要素
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        明确目标
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         模糊示例
        </strong>
        ：“写一篇关于环保的文章。”
       </li>
       <li>
        <strong>
         优化示例
        </strong>
        ：“以‘垃圾分类’为主题，撰写一篇面向社区居民的科普文章，要求包含实施步骤和常见误区，字数约800字。”
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        提供上下文
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         不足示例
        </strong>
        ：“解释量子计算。”
       </li>
       <li>
        <strong>
         优化示例
        </strong>
        ：“假设读者是高中生，用通俗易懂的语言解释量子计算的基本原理，并举例说明其潜在应用。”
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        指定格式
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         通用请求
        </strong>
        ：“列出学习Python的建议。”
       </li>
       <li>
        <strong>
         结构化请求
        </strong>
        ：
        <br/>
        <code>
         请按以下格式提供Python学习路径建议：
        </code>
        <code>
         1. 基础阶段（1-2周）：包含3个重点知识点 2. 进阶阶段（3-4周）：推荐2个实战项目 3. 资源推荐：书籍、网站、视频各1个
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        控制风格与语气
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         无指定
        </strong>
        ：“写一封投诉信。”
       </li>
       <li>
        <strong>
         细化要求
        </strong>
        ：“以专业但礼貌的语气，撰写一封针对快递延迟的投诉邮件，需包含订单号、问题时间线和具体诉求。
       </li>
      </ul>
     </li>
    </ol>
    <h5>
     三、提示词设计进阶技巧
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        角色扮演法
       </strong>
       <br/>
       赋予AI特定角色以聚焦回答方向：
      </p>
      <code>
       你是一位资深的网络安全工程师，请用技术文档风格编写一份关于HTTPS协议加密原理的说明，包含以下章节： - 对称加密与非对称加密对比 - TLS握手流程详解 - 常见攻击防护建议
      </code>
     </li>
     <li>
      <p>
       <strong>
        分步引导法
       </strong>
       <br/>
       将复杂任务拆解为步骤：
      </p>
      <code>
       请按顺序完成以下任务： 1. 分析用户提供的销售数据（见附件） 2. 识别过去三个月销量下降最明显的前3个产品 3. 对每个产品提出3条改进建议
      </code>
     </li>
     <li>
      <p>
       <strong>
        示例示范法
       </strong>
       <br/>
       提供输入-输出样例规范结果：
      </p>
      <code>
       示例： 输入：帮我将会议时间从英文转换成中文 输出：好的，请提供具体时间内容（如"2 PM EST on March 5th"） 现在请处理：Convert "10
      </code>
     </li>
    </ol>
