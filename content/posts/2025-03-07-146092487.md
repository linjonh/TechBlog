   <div class="htmledit_views" id="content_views">
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       修改表名
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     命令：rename table 原表名 to 新表名;
     <br/>
     <br/>
     例如：在表MyClass名字更改为YouClass
     <br/>
     mysql&gt; rename table MyClass to YouClass;
     <br/>
     <br/>
     当你执行 RENAME 时，你不能有任何锁定的表或活动的事务。你同样也必须有对原初表的 ALTER 和 DROP 权限，以及对新表的 CREATE 和 INSERT 权限。
     <br/>
     <br/>
     如果在多表更名中，MySQL 遭遇到任何错误，它将对所有被更名的表进行倒退更名，将每件事物退回到最初状态。
     <br/>
     <br/>
     RENAME TABLE 在 MySQL 3.23.23 中被加入。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      <strong>
       备份数据库
      </strong>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     命令在DOS的[url=\\mysql\\bin]\\mysql\\bin[/url]目录下执行
     <br/>
     <br/>
     <strong>
      <strong>
       1.导出整个数据库
      </strong>
     </strong>
     <br/>
     导出文件默认是存在mysql\bin目录下
     <br/>
     mysqldump -u 用户名 -p 数据库名 &gt; 导出的文件名
     <br/>
     mysqldump -u user_name -p123456 database_name &gt; out
     <br/>
     <br/>
     <strong>
      <strong>
       2.导出一个表
      </strong>
     </strong>
     <br/>
     mysqldump -u 用户名 -p 数据库名 表名&gt; 导出的文件名
     <br/>
     mysqldump -u user_name -p database_name table_name &gt; out
     <br/>
     <br/>
     <strong>
      <strong>
       3.导出一个数据库结构
      </strong>
     </strong>
     <br/>
     mysqldump -u user_name -p -d –add-drop-table database_name &gt; out
     <br/>
     -d 没有数据 –add-drop-table 在每个create语句之前增加一个drop table
     <br/>
     <br/>
     <strong>
      <strong>
       4.带语言参数导出
      </strong>
     </strong>
     <br/>
     mysqldump -uroot -p –default-character-set=latin1 –set-charset=gbk –skip-opt database_name &gt; out
     <br/>
     <br/>
     例如，将aaa库备份到文件back_aaa中：
     <br/>
     [root@test1 root]# cd　/home/data/mysql
     <br/>
     [root@test1 mysql]# mysqldump -u root -p --opt aaa &gt; back_aaa
    </p>
