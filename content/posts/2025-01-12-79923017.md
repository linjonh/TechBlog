# Python版本区别说明

Python版本区别说明

Python 3.7.0b3
  
发布日期：2018-03-29
  
这是python 3.7的beta版预览版python 3.7仍在开发中。
  
此发行版3.7.0b3是四个计划的测试版发布预览中的第三个。
  
  
  
Python 3.6.5
  
发布日期：2018-03-28
  
python 3.6.5是python 3.6的第五个维护版本。
  
python 3.6系列包含许多新功能和优化。
  
  
  
python 3.6.5rc1
  
发布日期：2018-03-13
  
python 3.6.5rc1是python 3.6的第五个维护版本的发布候选预览版。
  
python 3.6系列包含许多新功能和优化。
  
  
  
注：
  
带b的是beta版预览版。

带rc的是发布候选预览版。
