    <h2 id="java输出windows中的全部汉字">
     Java输出windows中的全部汉字
    </h2>
    <h3 id="1简介">
     1、简介
    </h3>
    <p>
     本篇博客向大家展示一个小程序，即通过Java编程输出Windows中的全部汉字，并将这些汉字存入程序所在路径的txt文件中。
    </p>
    <hr/>
    <h3 id="2实现过程">
     2、实现过程
    </h3>
    <p>
     1）对于Java中文件的操作可以参考工程训练中的任务2.1进行学习，本部分需要使用到文件的写入，在java中可以通过BufferedWriter或FileWriter对文件进行写入操作，在使用BufferedWriter时也一定会使用到FileWriter，本程序选择直接使用FileWriter类进行文件的写入；
    </p>
    <p>
     2）由于不能保证指定路径下的文件存在故可以使用if语句的判断条件配合createNewFile（）创建指定文件，但是需要使用try和catch代码块，这些知识在工程训练中都有接触，故不一一赘述了；
    </p>
    <p>
     3）由于可以通过汉字unicode码做循环来实现这一程序，所以就必须知道汉字unicode码的范围，通过查阅资料可知汉字该范围为4e00-9fa5，故可以使用for循环来实现汉字的输出；
    </p>
    <p>
     4）程序代码如下：
    </p>
    <pre class="prettyprint"><code class="hljs java"><span class="hljs-keyword">import</span> java.io.File;<span class="hljs-comment">//导入java.io包下的File类</span>
<span class="hljs-keyword">import</span> java.io.FileWriter;<span class="hljs-comment">//导入java.io包下的FileWriter类</span>
<span class="hljs-keyword">import</span> java.io.IOException;<span class="hljs-comment">//导入java.io包下的IOException类</span>
<span class="hljs-keyword">public</span> <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">Test</span> {<!-- --></span><span class="hljs-comment">//类名</span>
    <span class="hljs-keyword">public</span> <span class="hljs-keyword">static</span> <span class="hljs-keyword">void</span> <span class="hljs-title">main</span>( String[] args ){ <span class="hljs-comment">//程序主入口函数</span>
        <span class="hljs-keyword">try</span>{<!-- --><span class="hljs-comment">//try代码块，当发生异常时会转到catch代码块中</span>
            File file =<span class="hljs-keyword">new</span> File(<span class="hljs-string">"Test666.txt"</span>);<span class="hljs-comment">//创建文件对象</span>
            <span class="hljs-keyword">if</span>(!file.exists()){<!-- --><span class="hljs-comment">//if语句的条件，若指定路径下该文件不存在</span>
            file.createNewFile();<span class="hljs-comment">//则在指定路径下新建该文件</span>
            }
            FileWriter fw=<span class="hljs-keyword">new</span> FileWriter(<span class="hljs-string">"Test666.txt"</span>);<span class="hljs-comment">//创建FileWiter类对象</span>
            String start=<span class="hljs-string">"4e00"</span>;<span class="hljs-comment">//定义一个字符串变量为4e00</span>
            String end=<span class="hljs-string">"9fa5"</span>;<span class="hljs-comment">//定义一个字符串变量为9fa5</span>
            <span class="hljs-keyword">int</span> s=Integer.parseInt(start, <span class="hljs-number">16</span>);<span class="hljs-comment">//将16进制字符start转换为10进制整数</span>

    }
