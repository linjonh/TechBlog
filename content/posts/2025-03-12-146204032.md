    <h2>
     <span style="color:#0d0016">
      详细步骤指导如何使用DBeaver来连接到数据库：
     </span>
    </h2>
    <h4>
     <span style="color:#0d0016">
      <strong>
       步骤 1: 下载并安装 DBeaver
      </strong>
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      如果还没有安装DBeaver，请访问
      <a href="https://dbeaver.io/" rel="nofollow" title="DBeaver官网">
       DBeaver官网
      </a>
      下载适合操作系统的版本，并按照指示完成安装。
     </span>
    </p>
    <h4>
     <span style="color:#0d0016">
      <strong>
       步骤 2: 启动 DBeaver
      </strong>
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      安装完成后，启动DBeaver应用程序。
     </span>
    </p>
    <h4>
     <span style="color:#0d0016">
      <strong>
       步骤 3: 创建一个新的数据库连接
      </strong>
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      <strong>
       打开“新建连接”窗口
      </strong>
      ：在DBeaver主界面中，点击左上角的“数据库”菜单，然后选择“新建连接”。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       选择数据库类型
      </strong>
      ：在弹出的窗口中，从列表中选择想要连接的数据库类型（例如MySQL、PostgreSQL、Oracle等）。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       配置连接参数
      </strong>
      ：
     </span>
    </p>
    <ul>
     <li>
      <span style="color:#0d0016">
       <strong>
        主机名或IP地址
       </strong>
       ：输入数据库服务器的主机名或IP地址。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       <strong>
        端口
       </strong>
       ：输入数据库服务监听的端口号（通常是默认值，如MySQL是3306，PostgreSQL是5432）。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       <strong>
        数据库名称
       </strong>
       ：输入要连接的具体数据库名称。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       <strong>
        用户名和密码
       </strong>
       ：输入用于连接数据库的用户名和密码。
      </span>
     </li>
    </ul>
    <p>
     <span style="color:#0d0016">
      <strong>
       测试连接
      </strong>
      ：填写完信息后，点击“测试连接”按钮以确保所有信息正确无误。如果一切正常，将看到一个成功消息。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <strong>
       完成连接设置
      </strong>
      ：如果测试成功，点击“完成”按钮以保存连接设置。
     </span>
    </p>
    <h4>
     <span style="color:#0d0016">
      <strong>
       步骤 4: 使用已建立的连接
      </strong>
     </span>
    </h4>
    <ul>
     <li>
      <span style="color:#0d0016">
       连接成功后，可以在左侧的“数据库导航器”中看到新创建的连接。展开该连接可以查看数据库中的表、视图和其他对象。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       右键点击数据库或其下的任意对象，可以选择执行各种操作，比如运行SQL查询、查看数据、编辑表结构等。
      </span>
     </li>
    </ul>
    <h2>
     <span style="color:#0d0016">
      使用DBeaver构造ERD图的步骤：
     </span>
    </h2>
    <h4>
     <span style="color:#0d0016">
      自动生成ERD图
     </span>
    </h4>
    <ol>
     <li>
      <span style="color:#0d0016">
       连接数据库：打开DBeaver并连接到目标数据库。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       选择表：在数据库导航器中，找到要生成ER图的数据库表或视图，并展开它们。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       右键点击表：右键点击要生成ER图的表或视图，选择“ER图”选项。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       选择其他表：在弹出的窗口中，选择要添加到ER图中的其他表或视图，并设置它们之间的关系。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       生成ERD图：点击“确定”按钮，DBeaver将自动生成ER图。
      </span>
     </li>
    </ol>
    <h4>
     <span style="color:#0d0016">
      自定义ERD图
     </span>
    </h4>
    <ol>
     <li>
      <span style="color:#0d0016">
       创建新ER图：在左侧导航栏找到“ER Diagrams”节点，右键点击选择“创建新的ER图”。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       配置参数：在弹出的对话框中，为新图表指定一个名字，挑选要可视化的特定数据库连接以及所关注的数据对象列表。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       完成创建：确认无误后点击“完成”，生成初步的ER图。
      </span>
     </li>
     <li>
      <span style="color:#0d0016">
       调整显示效果：通过多种方式调整ER图的外观，如设置表间关系线的参照约束方向、更改线条风格等。
      </span>
     </li>
    </ol>
    <h4>
     <span style="color:#0d0016">
      导出ERD图
     </span>
    </h4>
    <ul>
     <li>
      <span style="color:#0d0016">
       完成ER图的编辑后，可以将其导出为图片或其他格式。在DBeaver中，可以通过右键点击ER图并选择“导出”来实现这一功能。
      </span>
     </li>
    </ul>
    <h2>
     <span style="color:#0d0016">
      DBeaver格式化SQL
     </span>
    </h2>
    <h4>
     <span style="color:#0d0016">
      在DBeaver中，可以使用以下
     </span>
     <span style="color:#fe2c24">
      快捷键
     </span>
     <span style="color:#0d0016">
      来格式化SQL语句：
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      <strong>
       Windows/Linux
      </strong>
      ：
      <strong>
       <span style="background-color:#ffd900">
        Ctrl + Shift + F
       </span>
      </strong>
      <br/>
      <strong>
       Mac
      </strong>
      ：
      <strong>
       <span style="background-color:#ff9900">
        Command + Shift + F
       </span>
      </strong>
     </span>
    </p>
    <h4>
     <span style="color:#0d0016">
      如果上述快捷键在DBeaver版本中不起作用，可以通过以下步骤
     </span>
     <span style="color:#fe2c24">
      手动
     </span>
     <span style="color:#0d0016">
      格式化SQL语句：
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      1. 打开SQL编辑器。
      <br/>
      2. 编写或粘贴需要格式化的SQL代码。
      <br/>
      3. 选中需要格式化的SQL代码。
      <br/>
      4. 点击菜单栏的“编辑”-&gt;“格式化SQL”。
      <br/>
      5. 或者在SQL编辑器的右键菜单中选择“格式化”。
     </span>
    </p>
    <h4>
     <span style="color:#0d0016">
      此外，还可以通过DBeaver的设置来
     </span>
     <span style="color:#fe2c24">
      自定义
     </span>
     <span style="color:#0d0016">
      格式化选项：
     </span>
    </h4>
    <p>
     <span style="color:#0d0016">
      1. 点击菜单栏的“窗口”-&gt;“首选项”。
      <br/>
      2. 在左侧导航栏中选择“编辑器”-&gt;“SQL编辑器”-&gt;“格式化”。
      <br/>
      3. 在这里可以调整缩进、空格、换行等格式化选项。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      如果格式化结果不符合预期，可以尝试更新DBeaver到最新版本，或者检查是否有插件影响了格式化功能。
     </span>
    </p>
