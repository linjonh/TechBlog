# 谷歌学术高级搜索技巧

> 对科研工作者来说，文献调研是一项几乎每天都要面对的事情，高效地进行文献调研是必备的能力之一。当接触一个新的领域时，搜索综述文章（Review）可以帮助我们对该领域有一个基础性的认识，包括发展历史、面临的挑战、发展趋势等；当需要研究某个特定问题时，通过精确搜索可以帮助我们寻找到相关的文献；想要阅读某领域大牛的文章时，通过文献搜索可以帮助我们寻找高质量、前沿的文献资源。
>
> 谷歌学术作为学术界最受欢迎的学术搜索引擎，其重要性不言而喻。然而，有了好的工具，会用，才能发挥它的最大价值，服务于我们的科研工作。本文重点介绍谷歌学术的高级搜索功能。高级搜索功能能够帮助我们精确高效地搜索到我们想要的东西。

谷歌学术有普通搜索和高级搜索两种模式，界面分别如下：

对于经常使用Google Scholar的人来说，细心一点便能发现，不论时普通搜索还是高级搜索，最终都是将我们键入的字段转换成一串通过逻辑字符（如
**`AND`**

**、**

**`OR`**
）连接的表达式。于是，这篇文章将直接介绍如何通过表达式来实现高级搜索，这种方式是最本质也是最高效的方式。

谷歌学术的默认逻辑符号

从简单的说起，当我们打开谷歌学术，假如直接在搜索框里输入下面几个关键词：

```
surface acoustic wave microfluidics

```

那么，谷歌学术是如何处理上面输入的关键词的呢？默认，在搜索框中输入的不同关键词是
**并**
的关系，也就是
**`AND`**
的关系。于是，上面的搜索等效于：

```
surface AND acoustic AND wave AND microfluidics

```

注意这里的
`AND`
必须大写。大家可以亲自试一下，上面两种方式搜索到的文献数量是一样的。那么，有一个问题来了，搜索到的文献和上面的几个关键词又是什么关系呢？既然已经说了谷歌学术将几个关键词作
`AND`
处理，那么也就是说搜索到的每一篇文献中都出现了我们输入的所有关键词。

关键词和短语的区别

分清楚
**关键词**
和
**短语**
的区别可以说是文献搜索最重要的一点，也是精确搜索最关键的地方。如果我们想要搜索包含关键词
**`surface`**


、
**`acoustic`**
、
**`wave`**
的文献，那么直接在搜索框键入：

```
surface acoustic wave

```

便可以完成搜索。注意，这里说的包含指
**分散包含**
（三个关键词可以出现在文章中的不同位置），而不一定是
**连续包含**
（直接包含
**`surface`**


**`acoustic`**


**`wave`**
）。那么，如果我们希望搜索包含短语
**`surface`**


**`acoustic`**


**`wave`**
的文献，那么只需要在搜索框输入：

```
"surface acoustic wave"

```

也就是说用一对
**英文双引号**
包含需要搜索的短语。大家亲自试一试就可以发现第二种方式搜索到的文献数量要比第一种方式搜索到的要少。所以，以短语的方式可以进行更加精确的搜索，帮助我们快速找到需要的文献。

逻辑符号
**AND**
和
**OR**

**`AND`**
和
**`OR`**
分别是
**并**
和
**或**
的意思。直接举例说明。

```
"surface acoustic wave" AND "pressure sensor"

```

这串表达式的含义是搜索同时包含短语
**`surfac`**


**`acoustic`**


**`wave`**
和
**`pressure`**


**`sensor`**
的文献。

再看下面的表达式：

```
"surface acoustic wave" OR "pressure sensor"

```

这串表达式的含义是搜索包含短语
**`surfac`**


**`acoustic`**


**`wave`**
或者
**`pressure`**


**`sensor`**
的文献。

`AND`
和
`OR`
还能发挥更大的作用，后面会得到体现。

搜索标题

有点时候我们需要搜索标题中包含特定关键词或者短语的文献，谷歌学术提供了直接对标题进行搜索的功能。如果我们需要搜索标题中包含短语
**`surface`**


**`acoustic`**


**`wave`**
的文献，使用下面的表达式即可。

```
intitle:"surface acoustic wave"

```

需要注意的是，上面的表达式中，
`:`
为
**英文冒号**
，且紧跟在
**`intitle`**
后面，冒号后面
**没有空格**
。

再看下面的表达式：

```
intitle:("surface acoustic wave" AND "pressure sensor")

```

这串表达式的含义是搜索标题中同时包含短语
`surface acoustic wave`
和
`pressure sensor`
的文献。注意，要用英文括号
**`()`**
括住所有短语。

搜索作者

很多时候，我们会对某个领域大牛发表过的文献很感兴趣，这个时候就需要根据作者搜索文献。如果我们需要搜索
`Richard Fu`
发表过的文章，使用下面的表达式即可。

```
author:"Richard Fu"

```

注意这里需要用双引号包含，因为人名是一个整体，也就是说是一个短语。

另外一点需要关注的是，由于不同期刊对作者姓名的缩写方式不同，因此搜索时就面临使用哪个形式的名字的问题。比如某个作者全名叫做
`Congcong Li`
，那么不同在不同期刊可能出现如
`CC Li`
、
`C Li`
、
`Congcong Li`
等形式，那么当我们想要搜索该作者以不同姓名形式发表的所有文章时，可以使用下面的方式：

```
author:("CC Li" OR "C Li" OR "Congcong Li")

```

搜索文献来源期刊

每个研究领域都有一些比较权威或顶尖的期刊，这些期刊上发表的文献的质量往往非常高，阅读这些文献能够让我们学到诸多知识，启发我们思考。此时，我们便需要搜索发表在特定期刊上的文献。假如我们需要搜索发表在期刊
`APPLIED PHYSICS LETTERS`
上有关
`surface acoustic wave`
的文献，可以使用下面的表达式：

```
"surface acoustic wave" source:"APPLIED PHYSICS LETTERS"

```

也就是说谷歌学术用
**`source`**
字段来表示文献的来源期刊。

限定搜索年限

当我们需要跟踪某个领域的最新进展时，我们需要知道世界上的科研工作者都在重点研究哪个问题，从而帮助我们紧跟前沿。这时，搜索最新发表的文献就很有必要。谷歌学术没有直接提供限定搜索时间的字段，而是在搜索界面的左侧提供了限定年限的工具。如下图所示。
