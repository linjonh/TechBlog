    <h2>
     <a id="_0">
     </a>
     功能描述
    </h2>
    <ol>
     <li>
      打开微信扫一扫，扫描产品上的二维码，
      <strong>
       弹出小程序，跳到“邀请用户”页面
      </strong>
      。
     </li>
     <li>
      <strong>
       解析二维码中的参数
      </strong>
      ，自动填充到页面中的“邀请码”输入框。
     </li>
    </ol>
    <h2>
     <a id="_4">
     </a>
     操作步骤
    </h2>
    <ol>
     <li>
      <p>
       首先，要到微信公众平台对
       <code>
        扫普通链接二维码打开小程序
       </code>
       功能进行配置。
      </p>
      <ul>
       <li>
        找到"开发管理"-“开发设置”-“扫普通链接二维码打开小程序”
        <br/>
        <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/f7c1220fb807e5d600d16ef9b1090f0d.png">

        </img>
       </li>
       <li>
        填写配置项
        <ul>
         <li>
          <code>
           线上版本
          </code>
          的二维码配置中，
          <strong>
           二维码规则
          </strong>
          和
          <strong>
           校验文件
          </strong>
          两项的配置需要后端配合。（不用填写“测试链接”）
         </li>
         <li>
          <code>
           测试
          </code>
          时，二维码规则可以自定义，校验文件不用管，但是
          <strong>
           测试链接
          </strong>
          一定要填写，测试范围选择
          <strong>
           体验版
          </strong>
          ，这样的话，用微信扫描测试链接的二维码，就可以跳转到体验版小程序。
          <br/>
          <img alt="image.png" src="https://i-blog.csdnimg.cn/blog_migrate/a8b408589059058833dc3914557792cd.png"/>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       测试时，可以去
       <a href="https://cli.im/" rel="nofollow">
        草料网
       </a>
       根据上一步中填写的测试链接生成二维码。
