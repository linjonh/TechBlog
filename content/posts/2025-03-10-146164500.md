# ngx\_openssl\_create\_conf

#### ngx\_openssl\_create\_conf

#### 声明在 src\event\ngx\_event\_openssl.c

```
static void *ngx_openssl_create_conf(ngx_cycle_t *cycle);
```

#### 定义在 src\event\ngx\_event\_openssl.c

```
static void *
ngx_openssl_create_conf(ngx_cycle_t *cycle)
{
    ngx_openssl_conf_t  *oscf;

    oscf = ngx_pcalloc(cycle->pool, sizeof(ngx_openssl_conf_t));
    if (oscf == NULL) {
        return NULL;
    }

    /*
     * set by ngx_pcalloc():
     *
     *     oscf->engine = 0;
     */

    return oscf;
}
```

#### 就是从内存池中分配一个 ngx\_openssl\_conf\_t 结构体大小的内存，然后返回内存地址
