# python怎么加载库_用python怎么安装库

python安装库的方法：1、在【运行】窗口，输入命令“cmd”，回车，打开cmd命令行窗口。2、在命令行中输入命令“pip install 库名称”即可安装指定库；若是python3.0以上的版本，则使用“pip3 install 库名称”命令进行安装。

![5ce3ee2b1edd4791.jpg](https://i-blog.csdnimg.cn/blog_migrate/725415afc8d38724d9e3ae3d54cdd33f.jpeg)

今天教大家如何安装安装python第三方库。

目前python主流版本是python2和python3,以下示例在python3中演示。

python3.0以上的版本自带pip。pip是python的一种包管理工具，它可以支持包的安装、卸载，升级等操作。

安装库之前确保计算机成功安装了python!!!

安装步骤：

使用【win+r】打开【运行】窗口，输入命令“cmd”，回车，打开cmd命令行窗口

![1600314195996304.png](https://i-blog.csdnimg.cn/blog_migrate/ea0991f1f30701ae885858d820a3024b.png)

命令行中输入pip3 install xxx ，xxx代表包的名字。如安装requests库
