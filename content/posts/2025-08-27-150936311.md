### 五、AI协作进阶技巧

#### 5.1 上下文管理策略

* **黄金300token原则**：当对话超过15条时，新建会话并传递关键上下文
* **文档索引法**：将需求文档、接口规范存入Cursor Docs，生成时自动引用

#### 5.2 错误处理范式

```typescript
// AI生成的错误处理代码
try {
  await submitMessage();
} catch (error: any) {
  const analysis = await Cursor.analyzeError(error);
  if (analysis.code === 'NETWORK_ERROR') {
    showToast('网络异常，请重试', 'error');
  }
}

```

#### 5.3 性能优化实践

* **AI驱动的懒加载**：通过`// @vite-lazy`注释自动生成懒加载代码
* **代码分割建议**：Cursor自动识别大组件并建议拆分
