    <h2>
     <a id="_0">
     </a>
     什么是“网络空间安全”？
    </h2>
    <p>
     网络空间安全指的是保护网络系统、设备、程序和数据免受未经授权的访问、破坏、篡改、泄露和滥用等各种威胁和风险的能力。它是保障国家安全、社会稳定、经济发展和个人隐私安全的重要方面。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/da110cab424c3aeeef4b3bd53777bad4.png#pic_center" width="600"/>
    </p>
    <p>
     网络空间安全主要涉及以下几个方面：
    </p>
    <ol>
     <li>
      保护网络系统和设备的安全：网络系统和设备是网络空间安全的基础，保护其安全对于维护网络空间安全至关重要。
     </li>
     <li>
      保护网络通信的安全：包括对网络通信进行加密、认证和防窃听等，确保网络通信的机密性、完整性和可用性。
     </li>
     <li>
      保护网络应用程序和数据的安全：网络应用程序和数据是网络空间安全的核心，需要采取多种技术手段保护其安全，如防火墙、反病毒软件、入侵检测和数据加密等。
     </li>
     <li>
      管理网络安全风险：网络安全风险是不可避免的，需要进行风险评估和管理，制定相应的应急响应计划，以应对网络安全事件和威胁。
     </li>
    </ol>
    <p>
     网络空间安全是一个复杂的领域，需要政府、企业和个人共同努力，采取综合措施，建立完善的法律法规和管理体系，保护网络空间安全。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/226003b4027f37236811fb595005ca91.png"/>
    </p>
    <h2>
     <a id="_13">
     </a>
     网络空间安全行业好就业么？
    </h2>
    <p>
     网络空间安全是一个不断发展壮大的行业，随着互联网技术的普及和发展，网络安全行业的前景也越来越好。可以预见，未来网络安全行业的就业需求将会不断增加。
    </p>
    <p>
     目前，网络空间安全行业正处于高速发展期，这个行业中的职业种类很多，包括安全工程师、网络安全研究员、网络安全顾问、安全运营工程师、渗透测试工程师等等。随着信息化程度的提高，网络安全的需求也会越来越大。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/de8433303435935a046175ac3fd46423.png"/>
    </p>
    <p>
     从整个行业来看，网络安全行业的就业前景良好。一方面，国家对网络安全的重视程度不断加大，相关政策不断推出，这使得网络安全行业的就业机会不断增加。另一方面，网络安全威胁不断增多，企业和个人对网络安全的重视程度也在不断提高，这也为网络安全行业提供了更多的就业机会。
    </p>
    <p>
     总体来说，网络空间安全行业具有较好的就业前景，未来也将继续保持良好发展态势。然而，行业竞争也相当激烈，需要具备较强的技能和知识储备。
    </p>
    <h2>
     <a id="_22">
     </a>
     如何入门学习网络安全呢？
    </h2>
    <p>
     在了解完网络空间安全的前景之美好，一份
     <strong>
      为期三个月
     </strong>
     的网络安全初、中级学习计划将呈现在大家面前：
    </p>
    <h3>
     <a id="_24">
     </a>
     初级阶段
    </h3>
    <p>
     <strong>
      1、网络安全理论知识（2天）
     </strong>
     <br/>
     ①了解行业相关背景，前景，确定发展方向。
     <br/>
     ②学习网络安全相关法律法规。
     <br/>
     ③网络安全运营的概念。
     <br/>
     ④等保简介、等保规定、流程和规范。（非常重要）
    </p>
    <p>
     <strong>
      2、渗透测试基础（7天）
     </strong>
     <br/>
     ①渗透测试的流程、分类、标准
     <br/>
     ②信息收集技术：主动/被动信息搜集、Nmap工具、Google Hacking
     <br/>
     ③漏洞扫描、漏洞利用、原理，利用方法、工具（MSF）、绕过IDS和反病毒侦察
     <br/>
     ④主机攻防演练：MS17-010、MS08-067、MS10-046、MS12-20等
    </p>
    <p>
     <strong>
      3、操作系统基础（7天）
     </strong>
     <br/>
     ①Windows系统常见功能和命令
     <br/>
     ②Kali Linux系统常见功能和命令
     <br/>
     ③操作系统安全（系统入侵排查/系统加固基础）
    </p>
    <p>
     <strong>
      4、计算机网络基础（7天）
     </strong>
     <br/>
     ①计算机网络基础、协议和架构
     <br/>
     ②网络通信原理、OSI模型、数据转发流程
     <br/>
     ③常见协议解析（HTTP、TCP/IP、ARP等）
     <br/>
     ④网络攻击技术与网络安全防御技术
     <br/>
     ⑤Web漏洞原理与防御：主动/被动攻击、DDOS攻击、CVE漏洞复现
    </p>
    <p>
     <strong>
      5、数据库基础操作（2天）
     </strong>
     <br/>
     ①数据库基础
     <br/>
     ②SQL语言基础
     <br/>
     ③数据库安全加固
    </p>
    <p>
     <strong>
      6、Web渗透（7天）
     </strong>
     <br/>
     ①HTML、CSS和JavaScript简介
     <br/>
     ②OWASP Top10
     <br/>
     ③Web漏洞扫描工具
     <br/>
     ④Web渗透工具：Nmap、BurpSuite、SQLMap、其他（菜刀、漏扫等）
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2dd81e8daffd010e493be732fbe566e7.png">
      <br/>
      恭喜你，当你学到这里时，基本可以从事一份网络安全相关的工作，比如渗透测试、Web渗透、安全服务、安全分析等岗位；其中如果等保模块学的好，还可以从事等保工程师。
     </img>
    </p>
    <p>
     薪资区间：
     <font color="red">
      <strong>
       6k~15k
      </strong>
     </font>
    </p>
    <p>
     到此为止，耗时约1个月左右，你已经成为了一名
     <strong>
      “脚本小子”
     </strong>
     ，那么你还想往下探索吗？
    </p>
    <h3>
     <a id="_67">
     </a>
     中级阶段
    </h3>
    <p>
     <strong>
      脚本编程学习（4周）
     </strong>
    </p>
    <p>
     在网络安全领域。是否具备编程能力是“脚本小子”和真正网络安全工程师的本质区别。在实际的渗透测试过程中，面对复杂多变的网络环境，当常用工具不能满足实际需求的时候，往往需要对现有工具进行扩展，或者编写符合我们要求的工具、自动化脚本，这个时候就需要具备一定的编程能力。在分秒必争的CTF竞赛中，想要高效地使用自制的脚本工具来实现各种目的，更是需要拥有编程能力。
    </p>
    <ul>
     <li>
      <p>
       零基础入门的同学，我建议选择脚本语言Python/PHP/Go/Java中的一种，对常用库进行编程学习；
      </p>
     </li>
     <li>
      <p>
       搭建开发环境和选择IDE，PHP环境推荐Wamp和XAMPP，IDE强烈推荐Sublime；
      </p>
     </li>
     <li>
      <p>
       Python编程学习，学习内容包含：语法、正则、文件、 网络、多线程等常用库，推荐《Python核心编程》，没必要看完；
      </p>
     </li>
     <li>
      <p>
       用Python编写漏洞的exp,然后写一个简单的网络爬虫；
      </p>
     </li>
     <li>
      <p>
       PHP基本语法学习并书写一个简单的博客系统；
      </p>
     </li>
     <li>
      <p>
       熟悉MVC架构，并试着学习一个PHP框架或者Python框架 (可选)；
      </p>
     </li>
     <li>
      <p>
       了解Bootstrap的布局或者CSS。
      </p>
     </li>
    </ul>
    <p>
     薪资区间
     <font color="red">
      <strong>
       20k~25k
      </strong>
     </font>
    </p>
