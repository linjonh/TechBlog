      要求：
     </img>
    </p>
    <ol>
     <li>
      在AR1配置Easy IP策略实现内网可以访问Internet
     </li>
     <li>
      主机IP如图所示，这里不做展示
     </li>
    </ol>
    <p>
     AR1接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">192.168</span>.0.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface GigabitEthernet0/0/1
 <span class="token function">ip</span> address <span class="token number">10.0</span>.1.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
<span class="token function">ip</span> route-static <span class="token number">0.0</span>.0.0 <span class="token number">0.0</span>.0.0 <span class="token number">10.0</span>.1.2
</code></pre>
    <p>
     AR2接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">20.0</span>.0.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface GigabitEthernet0/0/1
 <span class="token function">ip</span> address <span class="token number">10.0</span>.1.2 <span class="token number">255.255</span>.255.0 
</code></pre>
    <p>
     下面开始配置Easy IP，在AR2路由器的GE0/0/1端口的公网地址做地址转换以便内网可以访问Internet
     <br/>
     创建ACL定义允许通过NAPT访问Internet的内网网段
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>acl <span class="token number">2000</span>
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>rule <span class="token number">5</span> permit <span class="token builtin class-name">source</span> <span class="token number">192.168</span>.0.0 <span class="token number">0.0</span>.0.255
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>rule <span class="token number">10</span> deny
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     然后在AR1上连接Internet的端口GE0/0/1上配置NAPT
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>interface GigabitEthernet <span class="token number">0</span>/0/1
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/1<span class="token punctuation">]</span>nat outbound <span class="token number">2000</span>
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/1<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     至此Easy IP配置完成，下面测试内网地址访问Internet
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/79bb5349644a4d48954bea12ac4d1ed6.png">
      <br/>
      小结：
     </img>
    </p>
    <ol>
     <li>
      Easy IP就是一种简化了的NAPT，不需要配置公网地址池，直接使用外网接口的地址做地址转换。
     </li>
     <li>
      Easy IP适用于中小企业拨号上网，没有固定公网地址的情况，Easy IP路由器连接公网的接口获得什么地址就用什么地址做地址转换
     </li>
    </ol>
