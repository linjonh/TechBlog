      标题所提到的大坑：
      <br/>
      访问服务器的时候用的是http://公网ip：端口。于此同时，安全组端口开放ok，jdk安装和版本和环接配置ok，防火墙特定端口开放ok，springboot项目打成jar包。然后浏览器仍然无法访问，下面是解决办法：
     </li>
    </ol>
    <pre><code>1、查看httpd服务
	chkconfig --list httpd
提示服务不存在
2、安装httpd 服务
	yum -y install httpd
3、启动httpd服务
	service httpd start
</code></pre>
    <ol start="2">
     <li>

      <br/>
      测试java -version时还是无法使用。
     </li>
