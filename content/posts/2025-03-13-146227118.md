    <p>
     ONLYOFFICE 宏功能强大且用途广泛，不仅可用于自动执行文档、电子表格和演示文稿中的任务，还可用于 PDF 表单中的任务。在这篇博文中，我们将向您展示如何创建和运行一个独特的宏，该宏对表单中的特定字段执行计算。在此示例中，宏有助于计算税款，使流程更快、更准确。

     <img alt="如何使用 ONLYOFFICE 宏对 PDF 表单中的特定字段执行计算" height="742" src="https://i-blog.csdnimg.cn/img_convert/e31ef499f9f14937b4d1119e70eb71a8.png" width="1470"/>

    <h3>
     <strong>
      构建宏
     </strong>
    </h3>
    <h4>
     <strong>
      从文档中检索所有表格
     </strong>
    </h4>
    <p>
     首先我们从文档中检索所有字段：
    </p>
    <pre><code class="hljs">// Get all forms from the document
    const document = Api.GetDocument();
    let forms = document.GetAllForms();</code></pre>
    <ul>
     <li>
      <strong>
       Api.GetDocument()
      </strong>
      获取活动文档。
     </li>
     <li>
      <strong>
       GetAllForms()
      </strong>
      检索文档中的所有表单字段
     </li>
    </ul>
    <h4>
     <strong>
      定义
     </strong>
     <strong>
      表单
     </strong>
     <strong>
      键
     </strong>
    </h4>
    <p>
     然后，我们定义表单键：
    </p>
    <pre><code class="hljs">  // Formkeys of input forms
    var formkey1 = "Form1";
    var formkey2 = "Form2";
    var taxFormkey = "TaxForm";

    // Formkeys of result forms
    var sumResultKey = "SumResult";
    var taxResultKey = "TaxResult";</code></pre>
    <ul>
     <li>
      <strong>
       formkey1
      </strong>
      <strong>
       、
      </strong>
      <strong>
       formkey2
      </strong>
      和
      <strong>
       taxFormkey
      </strong>
      存储输入字段的标识符。
     </li>
     <li>
      <strong>
       sumResultKey
      </strong>
      和
      <strong>
       taxResultKey
      </strong>
      存储将插入结果的输出字段的标识符。
     </li>
    </ul>
    <h4>
     <strong>
      检索
     </strong>
     <strong>
      表单
     </strong>
     <strong>
      值
     </strong>
     <strong>
     </strong>
    </h4>
    <p>
     <strong>
      getFormValue
     </strong>
     函数遍历表单以查找与 formKey 匹配的
     <strong>
      表单
     </strong>
     ：
     <strong>
     </strong>
    </p>
    <pre><code class="hljs">  // Function to get the value of a form by its key
    function getFormValue(formKey) {
        for (let form of forms) {
            if (form.GetFormKey() === formKey) {
                return parseFloat(form.GetText()) || 0;
            }
        }
    }</code></pre>
    <ul>
     <li>
      <strong>
       GetText()
      </strong>
      检索表单的文本值。
     </li>
     <li>
      <strong>
       parseFloat()
      </strong>
      <strong>
      </strong>
      将文本转换为数字。
     </li>
    </ul>
    <p>
     如果转换失败，则默认为0，以防止计算错误。
    </p>
    <h4>
     <strong>
      向
     </strong>
     <strong>
      表单
     </strong>
     <strong>
      中
     </strong>
     <strong>
      插入
     </strong>
     <strong>
      值
     </strong>
     <strong>
     </strong>
    </h4>
    <p>
     <strong>
      setFormValue
     </strong>
     函数搜索与
     <strong>
      formKey
     </strong>
     匹配的表单字段。
     <strong>
      SetText()
     </strong>
     方法使用格式化的数值（2 位小数）更新该字段。
    </p>
    <pre><code class="hljs">  // Function to set the value of a result form
    function setFormValue(formKey, value) {
        for (let form of forms) {
            if (form.GetFormKey() === formKey) {
                form.SetText(value.toFixed(2));
            }
        }
    }</code></pre>
    <h4>
     <strong>
      主要
     </strong>
     <strong>
      计算
     </strong>
    </h4>
    <p>
     在主计算函数中我们：
    </p>
    <ul>
     <li>
      从表单字段检索用户输入的值。
     </li>
     <li>
      <strong>
       计算输入1
      </strong>
      <strong>
      </strong>
      和
      <strong>
       输入2
      </strong>
      的总和。
     </li>
     <li>
      按总额的百分比计算税额。
     </li>
     <li>
      调用
      <strong>
       setFormValue()
      </strong>
      将结果插入到各自的字段中。
     </li>
    </ul>
    <pre><code class="hljs"> // Main calculation function
    function calculateAndInsert() {
        let input1 = getFormValue(formkey1);
        let input2 = getFormValue(formkey2);
        let taxInput = getFormValue(taxFormkey);
        // Perform calculations
        var sum = parseFloat(input1) + parseFloat(input2);
        var tax = sum * taxInput / 100; // % tax

        // Insert results
        setFormValue(sumResultKey, sum);
        setFormValue(taxResultKey, tax);
    }</code></pre>
    <p>
     如下：
    </p>
    <pre><code class="hljs">(function () {
    // Get all forms from the document
    const document = Api.GetDocument();
    let forms = document.GetAllForms();
    // Formkeys of input forms
    var formkey1 = "Form1";
    var formkey2 = "Form2";
    var taxFormkey = "TaxForm";

    // Formkeys of result forms
    var sumResultKey = "SumResult";
    var taxResultKey = "TaxResult";

    // Function to get the value of a form by its key
    function getFormValue(formKey) {
        for (let form of forms) {
            if (form.GetFormKey() === formKey) {
                return parseFloat(form.GetText()) || 0;
            }
        }
    }

    // Function to set the value of a result form
    function setFormValue(formKey, value) {
        for (let form of forms) {
            if (form.GetFormKey() === formKey) {
                form.SetText(value.toFixed(2));
            }
        }
    }

    // Main calculation function
    function calculateAndInsert() {
        let input1 = getFormValue(formkey1);
        let input2 = getFormValue(formkey2);
        let taxInput = getFormValue(taxFormkey);
        // Perform calculations
        var sum = parseFloat(input1) + parseFloat(input2);
        var tax = sum * taxInput / 100; // % tax

        // Insert results
        setFormValue(sumResultKey, sum);
        setFormValue(taxResultKey, tax);
    }

    // Call the calculation function
    calculateAndInsert();
})();</code></pre>
