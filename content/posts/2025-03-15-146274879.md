    <p>
    </p>
    <p>
    </p>
    <p>
     在 Windows 系统上使用 Docker 镜像运行 PaddlePaddle 或其他应用程序非常简单。以下是详细步骤：
    </p>
    <h4>
     <a id="1__Docker_Desktop_4">
     </a>
     1. 安装 Docker Desktop
    </h4>
    <p>
     Docker Desktop 是 Windows 上运行 Docker 的官方工具。以下是安装步骤：
    </p>
    <h5>
     <a id="11__Docker_Desktop_7">
     </a>
     1.1 下载 Docker Desktop
    </h5>
    <p>
     访问
     <a href="https://www.docker.com/products/docker-desktop/" rel="nofollow">
      Docker 官方网站
     </a>
     。
    </p>
    <p>
     下载适用于 Windows 的 Docker Desktop 安装程序。
    </p>
    <h5>
     <a id="12__Docker_Desktop_12">
     </a>
     1.2 安装 Docker Desktop
    </h5>
    <p>
     1、运行下载的安装程序。
    </p>
    <p>
     2、按照提示完成安装。
    </p>
    <p>
     3、安装完成后，重启电脑。
    </p>
    <h5>
     <a id="13__WSL_2__HyperV_19">
     </a>
     1.3 启用 WSL 2 或 Hyper-V
    </h5>
    <p>
     Docker Desktop 需要 WSL 2（Windows Subsystem for Linux 2）或 Hyper-V 支持。
    </p>
    <p>
     #WSL 2（推荐）：
    </p>
    <p>
     1、打开 PowerShell（管理员权限）。
    </p>
    <p>
     2、运行以下命令启用 WSL 2：
    </p>
    <pre><code class="prism language-python">wsl <span class="token operator">-</span><span class="token operator">-</span>install
</code></pre>
    <p>
     3、安装完成后，重启电脑。
    </p>
    <p>
     #Hyper-V：
    </p>
    <p>
     1、打开“控制面板” -&gt; “程序” -&gt; “启用或关闭 Windows 功能”。
    </p>
    <p>
     2、勾选“Hyper-V”并点击“确定”。
    </p>
    <p>
     3、重启电脑。
    </p>
    <h5>
     <a id="14__Docker_Desktop_41">
     </a>
     1.4 启动 Docker Desktop
    </h5>
    <p>
     1、在开始菜单中搜索并启动 Docker Desktop。
    </p>
    <p>
     2、等待 Docker 启动（右下角系统托盘中的 Docker 图标变为绿色）。
    </p>
    <h5>
     <a id="15_Docker_46">
     </a>
     1.5 Docker不运行解决方法
    </h5>
    <p>
     <mark>
      如果界面显示Engine stopped，则卸载Docker Desktop，在安装 Docker Desktop 之前，先安装 WSL 2 并配置至少一个 Linux 发行版（如 Ubuntu)。
     </mark>
     参考以下两篇文章
     <br/>
     <a href="https://blog.csdn.net/Natsuago/article/details/145594631?spm=1001.2014.3001.5501">
      全网最全Win10/11系统下WSL2+Ubuntu20.04的全流程安装指南（两种支持安装至 D 盘方式）
     </a>
     <br/>
     <a href="https://blog.csdn.net/Natsuago/article/details/145588600">
      一篇就够！Windows上Docker Desktop安装 + 汉化完整指南（包含解决wsl更新失败方案）
     </a>
    </p>
    <h4>
     <a id="2__PaddlePaddle_Docker__51">
     </a>
     2. 拉取 PaddlePaddle Docker 镜像
    </h4>
    <p>
     PaddlePaddle 提供了预配置的 Docker 镜像，支持 GPU 和 CPU。
    </p>
    <h5>
     <a id="21__54">
     </a>
     2.1 打开终端
    </h5>
    <p>
     打开 PowerShell 或 Windows 终端。
    </p>
    <h5>
     <a id="22__57">
     </a>
     2.2 拉取镜像
    </h5>
    <p>
     运行以下命令拉取 PaddlePaddle 镜像（以 GPU 版本为例）：
    </p>
    <pre><code class="prism language-python">docker pull paddlepaddle<span class="token operator">/</span>paddle<span class="token punctuation">:</span><span class="token number">2.5</span><span class="token number">.2</span><span class="token operator">-</span>gpu<span class="token operator">-</span>cuda11<span class="token punctuation">.</span><span class="token number">7</span><span class="token operator">-</span>cudnn8
</code></pre>
    <h4>
     <a id="3__Docker__63">
     </a>
     3. 运行 Docker 容器
    </h4>
    <p>
     拉取镜像后，可以通过以下命令运行容器。
    </p>
    <h5>
     <a id="31__GPU__66">
     </a>
     3.1 运行 GPU 容器
    </h5>
    <p>
     如果你的系统支持 GPU，可以使用以下命令运行容器：
    </p>
    <pre><code class="prism language-python">docker run <span class="token operator">-</span><span class="token operator">-</span>gpus <span class="token builtin">all</span> <span class="token operator">-</span>it paddlepaddle<span class="token operator">/</span>paddle<span class="token punctuation">:</span><span class="token number">2.5</span><span class="token number">.2</span><span class="token operator">-</span>gpu<span class="token operator">-</span>cuda11<span class="token punctuation">.</span><span class="token number">7</span><span class="token operator">-</span>cudnn8 <span class="token operator">/</span><span class="token builtin">bin</span><span class="token operator">/</span>bash
</code></pre>
    <h5>
     <a id="32__CPU__72">
     </a>
     3.2 运行 CPU 容器
    </h5>
    <p>
     如果你的系统不支持 GPU，可以使用以下命令运行 CPU 容器：
    </p>
    <pre><code class="prism language-python">docker run <span class="token operator">-</span>it paddlepaddle<span class="token operator">/</span>paddle<span class="token punctuation">:</span><span class="token number">2.5</span><span class="token number">.2</span> <span class="token operator">/</span><span class="token builtin">bin</span><span class="token operator">/</span>bash
</code></pre>
    <h4>
     <a id="4__PaddlePaddle__79">
     </a>
     4. 验证 PaddlePaddle 安装
    </h4>
    <p>
     在容器中运行以下命令，验证 PaddlePaddle 是否正确安装：
    </p>
    <pre><code class="prism language-python">python <span class="token operator">-</span>c <span class="token string">"import paddle; print(paddle.device.is_compiled_with_cuda())"</span>
</code></pre>
    <p>
     如果输出 True，说明 GPU 支持已启用。
    </p>
    <p>
     如果输出 False，说明当前运行的是 CPU 版本。
    </p>
    <h4>
     <a id="5__Docker__89">
     </a>
     5. 使用 Docker 容器
    </h4>
    <p>
     在容器中，你可以像在本地环境中一样运行 Python 脚本或 Jupyter Notebook。
    </p>
    <h5>
     <a id="51__Python__92">
     </a>
     5.1 运行 Python 脚本
    </h5>
    <p>
     将本地脚本挂载到容器中并运行：
    </p>
    <pre><code class="prism language-python">docker run <span class="token operator">-</span><span class="token operator">-</span>gpus <span class="token builtin">all</span> <span class="token operator">-</span>v C<span class="token punctuation">:</span><span class="token operator">/</span>path<span class="token operator">/</span>to<span class="token operator">/</span>your<span class="token operator">/</span>script<span class="token punctuation">:</span><span class="token operator">/</span>workspace <span class="token operator">-</span>it paddlepaddle<span class="token operator">/</span>paddle<span class="token punctuation">:</span><span class="token number">2.5</span><span class="token number">.2</span><span class="token operator">-</span>gpu<span class="token operator">-</span>cuda11<span class="token punctuation">.</span><span class="token number">7</span><span class="token operator">-</span>cudnn8 <span class="token operator">/</span><span class="token builtin">bin</span><span class="token operator">/</span>bash
cd <span class="token operator">/</span>workspace
python your_script<span class="token punctuation">.</span>py
</code></pre>
    <p>
     5.2 运行 Jupyter Notebook
     <br/>
     启动 Jupyter Notebook 服务：
    </p>
    <pre><code class="prism language-python">docker run <span class="token operator">-</span><span class="token operator">-</span>gpus <span class="token builtin">all</span> <span class="token operator">-</span>p <span class="token number">8888</span><span class="token punctuation">:</span><span class="token number">8888</span> <span class="token operator">-</span>v C<span class="token punctuation">:</span><span class="token operator">/</span>path<span class="token operator">/</span>to<span class="token operator">/</span>your<span class="token operator">/</span>notebooks<span class="token punctuation">:</span><span class="token operator">/</span>workspace paddlepaddle<span class="token operator">/</span>paddle<span class="token punctuation">:</span><span class="token number">2.5</span><span class="token number">.2</span><span class="token operator">-</span>gpu<span class="token operator">-</span>cuda11<span class="token punctuation">.</span><span class="token number">7</span><span class="token operator">-</span>cudnn8 jupyter notebook <span class="token operator">-</span><span class="token operator">-</span>ip<span class="token operator">=</span><span class="token number">0.0</span><span class="token number">.0</span><span class="token number">.0</span> <span class="token operator">-</span><span class="token operator">-</span>allow<span class="token operator">-</span>root
</code></pre>
    <p>
     然后在浏览器中访问
     <a href="http://localhost:8888" rel="nofollow">
      http://localhost:8888
     </a>
     。
    </p>
    <h4>
     <a id="6__109">
     </a>
     6. 退出和删除容器
    </h4>
    <p>
     退出容器：在容器中运行 exit。
    </p>
    <p>
     删除容器：使用以下命令删除已停止的容器：
    </p>
    <pre><code class="prism language-python">docker rm <span class="token operator">&lt;</span>container_id<span class="token operator">&gt;</span>
</code></pre>
    <h4>
     <a id="7__117">
     </a>
     7. 常见问题
    </h4>
    <h5>
     <a id="71_Docker__118">
     </a>
     7.1 Docker 无法启动
    </h5>
    <p>
     确保 WSL 2 或 Hyper-V 已启用。
    </p>
    <p>
     确保 BIOS 中启用了虚拟化技术（VT-x/AMD-V）。
    </p>
    <h5>
     <a id="72_GPU__123">
     </a>
     7.2 GPU 不支持
    </h5>
    <p>
     确保已安装 NVIDIA 驱动程序。
    </p>
    <p>
     确保已安装
     <a href="https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/install-guide.html" rel="nofollow">
      NVIDIA Container Toolkit
     </a>
     。
    </p>
