    <p>
     在 HTML 中，插入图片是通过
     <code>
      &lt;img&gt;
     </code>
     标签实现的。
     <code>
      &lt;img&gt;
     </code>
     标签是一个空标签，意味着它不需要闭合标签。以下是插入图片的基本语法和常用属性的详细讲解。
    </p>
    <h3>
     <a id="_4">
     </a>
     一、基本语法
    </h3>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>图片路径<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>替代文本<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        src
       </code>
      </strong>
      ：指定图片的路径，可以是相对路径或绝对路径。
     </li>
     <li>
      <strong>
       <code>
        alt
       </code>
      </strong>
      ：为图片提供替代文本，用于无障碍访问和 SEO。
     </li>
    </ul>
    <h3>
     <a id="_13">
     </a>
     二、常用属性
    </h3>
    <h4>
     <a id="1_src_15">
     </a>
     1.
     <code>
      src
     </code>
    </h4>
    <p>
     <code>
      src
     </code>
     属性用于指定图片的路径。路径可以是相对路径（相对于当前页面的路径）或绝对路径（完整的 URL）。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span> <span class="token comment">&lt;!-- 相对路径 --&gt;</span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com/images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span> <span class="token comment">&lt;!-- 绝对路径 --&gt;</span>
</code></pre>
    <h4>
     <a id="2_alt_24">
     </a>
     2.
     <code>
      alt
     </code>
    </h4>
    <p>
     <code>
      alt
     </code>
     属性为图片提供替代文本，用于无障碍访问和 SEO。如果图片无法加载，浏览器会显示
     <code>
      alt
     </code>
     属性中的文本。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="3_width__height_32">
     </a>
     3.
     <code>
      width
     </code>
     和
     <code>
      height
     </code>
    </h4>
    <p>
     <code>
      width
     </code>
     和
     <code>
      height
     </code>
     属性用于指定图片的宽度和高度。这些属性可以以像素为单位设置。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span> <span class="token attr-name">width</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>200<span class="token punctuation">"</span></span> <span class="token attr-name">height</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>100<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="4_title_40">
     </a>
     4.
     <code>
      title
     </code>
    </h4>
    <p>
     <code>
      title
     </code>
     属性为图片添加额外的信息，通常作为工具提示显示。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span> <span class="token attr-name">title</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>这是一个示例图片<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="5_style_48">
     </a>
     5.
     <code>
      style
     </code>
    </h4>
    <p>
     <code>
      style
     </code>
     属性用于为图片定义内联样式。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span> <span class="token special-attr"><span class="token attr-name">style</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span><span class="token value css language-css"><span class="token property">border</span><span class="token punctuation">:</span> 1px solid black<span class="token punctuation">;</span> <span class="token property">border-radius</span><span class="token punctuation">:</span> 5px<span class="token punctuation">;</span></span><span class="token punctuation">"</span></span></span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="6_class__id_56">
     </a>
     6.
     <code>
      class
     </code>
     和
     <code>
      id
     </code>
    </h4>
    <p>
     <code>
      class
     </code>
     和
     <code>
      id
     </code>

    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>img</span> <span class="token attr-name">src</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>images/example.jpg<span class="token punctuation">"</span></span> <span class="token attr-name">alt</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>示例图片<span class="token punctuation">"</span></span> <span class="token attr-name">class</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>example-image<span class="token punctuation">"</span></span> <span class="token attr-name">id</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>unique-id<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>

     四、图片懒加载
    </h3>
    <p>
     为了优化网页性能，可以使用
     <code>
