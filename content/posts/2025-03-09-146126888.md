    <h2>
     PHP MySQL 创建数据库
    </h2>
    <h3>
     引言
    </h3>
    <p>
     在网站开发中，数据库是存储和管理数据的核心部分。PHP 和 MySQL 是最常用的网页开发语言和数据库管理系统之一。本文将详细介绍如何在 PHP 中使用 MySQL 创建数据库，并对其操作进行详细讲解。
    </p>
    <h3>
     前提条件
    </h3>
    <p>
     在开始创建数据库之前，请确保以下条件已经满足：
    </p>
    <ol>
     <li>
      已安装 PHP 和 MySQL。
     </li>
     <li>
      已配置好 PHP 与 MySQL 的连接。
     </li>
     <li>
      熟悉 PHP 和 MySQL 的基本语法。
     </li>
    </ol>
    <h3>
     1. 创建数据库
    </h3>
    <h4>
     1.1 连接到 MySQL
    </h4>
    <p>
     在 PHP 中，我们可以使用
     <code>
      mysqli
     </code>
     扩展或
     <code>
      PDO
     </code>
     扩展来连接 MySQL 数据库。以下是使用
     <code>
      mysqli
     </code>
     扩展连接数据库的示例代码：
    </p>
    <pre><code class="language-php">&lt;?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "myDB";

// 创建连接
$conn = new mysqli($servername, $username, $password, $dbname);

// 检查连接
if ($conn-&gt;connect_error) {
    die("连接失败: " . $conn-&gt;connect_error);
}
?&gt;
</code></pre>
    <h4>
     1.2 创建数据库
    </h4>
    <p>
     在连接到 MySQL 后，我们可以使用
     <code>
      CREATE DATABASE
     </code>
     语句创建一个新数据库。以下是一个示例：
    </p>
    <pre><code class="language-php">&lt;?php
// 创建数据库
$sql = "CREATE DATABASE myDB";

if ($conn-&gt;query($sql) === TRUE) {
    echo "数据库创建成功";
} else {
    echo "Error: " . $sql . "&lt;br&gt;" . $conn-&gt;error;
}
?&gt;
</code></pre>
    <h3>
     2. 创建表
    </h3>
    <p>
     在创建完数据库后，接下来我们需要创建一个或多个表来存储数据。
    </p>
    <h4>
     2.1 创建表
    </h4>
    <p>
     我们可以使用
     <code>
      CREATE TABLE
     </code>
     语句来创建一个新表。以下是一个示例：
    </p>
    <pre><code class="language-php">&lt;?php
// 创建表
$sql = "CREATE TABLE myTable (
    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    firstname VARCHAR(30) NOT NULL,
    lastname VARCHAR(30) NOT NULL,
    email VARCHAR(50),
    reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)";

if ($conn-&gt;query($sql) === TRUE) {
    echo "表创建成功";
} else {
    echo "Error: " . $sql . "&lt;br&gt;" . $conn-&gt;error;
}
?&gt;
</code></pre>
    <h4>
     2.2 查询表结构
    </h4>
    <p>
     创建完表后，我们可以使用
     <code>
      DESCRIBE
     </code>
     语句来查询表结构。以下是一个示例：
    </p>
    <pre><code class="language-php">&lt;?php
// 查询表结构
$sql = "DESCRIBE myTable";

$result = $conn-&gt;query($sql);

if ($result-&gt;num_rows &gt; 0) {
    while($row = $result-&gt;fetch_assoc()) {
        echo "id: " . $row["Field"]. " - Type: " . $row["Type"]. "&lt;br&gt;";
    }
} else {
    echo "0 results";
}
?&gt;
</code></pre>
    <h3>
     3. 总结
    </h3>
    <p>
     本文详细介绍了如何在 PHP 中使用 MySQL 创建数据库和表。通过以上步骤，您可以轻松地在 PHP 中创建数据库和表，以便存储和管理数据。
