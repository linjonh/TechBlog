    <h2>
     <a id="2__68">
     </a>
     2. 使用方式:
    </h2>
    <p>
     <strong>
      useSyncQueue返回两个方法:
     </strong>
    </p>
    <ul>
     <li>
      addTask添加任务；
      <br/>
      可单个添加，可数组添加；
      <br/>
      任务对象QueueTask由唯一ID和获取异步Promise的execute方法组成；
      <br/>
      添加任务后，会自动执行队列任务，直到队列为空
     </li>
     <li>
      awaitExeOver等待任务全部执行完；
      <br/>
      要在添加过任务之后调用awaitExeOver方法；
      <br/>
      执行awaitExeOver方法，才会创建回调方法overCallback；
      <br/>
      队列为空时，执行overCallback，awaitExeOver方法等待结束；
     </li>
    </ul>
