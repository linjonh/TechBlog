    <p>
     拓图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/62bcb650d0d7456889c38e99c79ed8ac.png"/>
    </p>
    <p>
     要求：
    </p>
    <ol>
     <li>
      允许 10.0.0.0 网段 telent 登录AR1，不允许其他主机telnet登录路由器
     </li>
     <li>
      设置接口如图所示
     </li>
    </ol>
    <p>
     AR1接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">30.0</span>.0.1 <span class="token number">255.255</span>.255.0
<span class="token comment">#</span>
<span class="token function">ip</span> route-static <span class="token number">0.0</span>.0.0 <span class="token number">0.0</span>.0.0 <span class="token number">30.0</span>.0.2
</code></pre>
    <p>
     AR2接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">10.0</span>.0.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
<span class="token function">ip</span> route-static <span class="token number">0.0</span>.0.0 <span class="token number">0.0</span>.0.0 <span class="token number">10.0</span>.0.2
</code></pre>
    <p>
     AR3接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">20.0</span>.0.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
<span class="token function">ip</span> route-static <span class="token number">0.0</span>.0.0 <span class="token number">0.0</span>.0.0 <span class="token number">20.0</span>.0.2
</code></pre>
    <p>
     AR4接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">30.0</span>.0.2 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface GigabitEthernet0/0/1
 <span class="token function">ip</span> address <span class="token number">10.0</span>.0.2 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface GigabitEthernet1/0/0
 <span class="token function">ip</span> address <span class="token number">20.0</span>.0.2 <span class="token number">255.255</span>.255.0 
</code></pre>
    <p>
     下面开始配置ACL
     <br/>
     在AR1路由器上配置策略ACL
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>acl <span class="token number">2000</span>
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>rule <span class="token number">5</span> permit <span class="token builtin class-name">source</span> <span class="token number">10.0</span>.0.0 <span class="token number">0.0</span>.0.255
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>rule <span class="token number">10</span> deny
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>quit
<span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>user-interface vty <span class="token number">0</span> <span class="token number">4</span>
<span class="token punctuation">[</span>AR1-ui-vty0-4<span class="token punctuation">]</span>authentication-mode password 
Please configure the login password <span class="token punctuation">(</span>maximum length <span class="token number">16</span><span class="token punctuation">)</span>:abc123,
<span class="token punctuation">[</span>AR1-ui-vty0-4<span class="token punctuation">]</span>acl <span class="token number">2000</span> inbound 
<span class="token punctuation">[</span>AR1-ui-vty0-4<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     或者，在AR4路由器上配置策略ACL，然后把策略应用到出接口上
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR4<span class="token punctuation">]</span>acl <span class="token number">3000</span>
<span class="token punctuation">[</span>AR4-acl-adv-3000<span class="token punctuation">]</span>rule <span class="token number">5</span> deny tcp <span class="token builtin class-name">source</span> <span class="token number">20.0</span>.0.0 <span class="token number">0.0</span>.0.255 destination-port eq telnet 
<span class="token punctuation">[</span>AR4-acl-adv-3000<span class="token punctuation">]</span>quit
<span class="token punctuation">[</span>AR4<span class="token punctuation">]</span>interface GigabitEthernet <span class="token number">0</span>/0/0
<span class="token punctuation">[</span>AR4-GigabitEthernet0/0/0<span class="token punctuation">]</span>traffic-filter outbound acl <span class="token number">3000</span>
</code></pre>
