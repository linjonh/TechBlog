   <link href="./../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <span style="font-family:Courier New; font-size:18px; color:#000099">
     Java编程语言为8个原始基本类型:
     <br/>
     boolean,char,byte,short,int,long,double,flout
     <br/>
     <br/>
     <br/>
     逻辑类:boolean
     <br/>
     boolean truth = true;
     <br/>
     <br/>
    </span>
    <p>
     <span style="font-family:Courier New; font-size:18px; color:#000099">
      注意:在整数类型和boolean之间无转换计算.有些语言(特别强调的是C和C++)允许将数字值,转换成逻辑值,这在Java编程语言中是不允许的;boolean类型只允许使用boolean值.
     </span>
    </p>
    <span style="font-family:Courier New; font-size:18px; color:#000099">
     if(a==2)条件表达式返回的是boolean.
     <span style="white-space:pre">
     </span>
     <br/>
     <br/>
     文本类:char和String
     <br/>
     char代表一个16-bit的Unicode字符.
     <br/>
     必须包含用单引号(')引用的文字.
     <br/>
     '\t' 一个制表符;
     <br/>
     '\u????' 一个特殊的Unicode字符,????应严格使用四个16进制数进行替换.
     <br/>
     <br/>
     <br/>
     String-不是原始类型,而是一个类,它被用来表示字符序列.(与C和C++不同String不能用\0作为结束);
     <br/>
     -具有双引号引用的文字.
     <br/>
     注意:String s = "Good, morning \n";                //"\n"表示一个换行
     <br/>
     System.out.println(s);
     <span style="white-space:pre">
     </span>
     //这里打印后会换2行
     <br/>
     <br/>
     <br/>
     整数类:byte,short,int,long(注意java中整数类型都是带符号的)
     <br/>
     <span style="white-space:pre">
     </span>
     1,采用3种进制十,八,十六进制
     <br/>
     <span style="white-space:pre">
     </span>
     2 表示十进制的2
     <br/>
     <span style="white-space:pre">
     </span>
     017 首位的0,表示这是一个八进制数
     <br/>
     <span style="white-space:pre">
     </span>
     0x1A 前2位的0x,表示这是一个16进制数
     <br/>
     <br/>
     <br/>
     <span style="white-space:pre">
     </span>
     2,具有缺省int(整数类文字属int类型,除非在后面直接跟一个字母"L")."L"表示一个long值
     <br/>
     <br/>
     <br/>
     浮点(float和double):
     <br/>
     <span style="white-space:pre">
     </span>
     浮点变量可用float或double来声明.如果一个数字包括小数点或指数部分,或者在数字后面带有字母F或f,D或d,则该
     <br/>
     文字为浮点.
     <br/>
     <span style="white-space:pre">
     </span>
     -缺省为double
     <br/>
     <span style="white-space:pre">
     </span>
     -浮点文字包括小数点或下列各项之一
     <br/>
     <span style="white-space:pre">
     </span>
     -E或e 增加指数值
     <br/>
     <span style="white-space:pre">
     </span>
     -F或f(float) 32bits
     <br/>
     <span style="white-space:pre">
     </span>
     -D或d(double) 64bits
     <br/>
     <br/>
     <br/>
     <span style="white-space:pre">
     </span>
    </span>
    <p>
     <span style="font-family:Courier New; font-size:18px; color:#000099">
      System.out.println(123.4E+306D);//=1.234E308???
     </span>
    </p>
    <p>
     <span style="font-family:Courier New; font-size:18px; color:#000099">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:'Courier New'; font-size:18px">
      <span style="font-family:'Courier New'; font-size:18px; background-color:rgb(255,255,255)">
       <span style="color:#cc0000">
        类型自动转换
       </span>
      </span>
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:Courier New; font-size:18px; color:#000099">
      ...................char
      <br/>
      ........↓
      <br/>
      byte→short→int→long
      <br/>
      ..... ... ↘
      <br/>
      .......float →  double
      <br/>
      <br/>
      另外long int 和float double有可能损失精度
      <br/>
      <br/>
      <br/>
      System.out.println((12.0==12));
      <br/>
      12.0浮点型double 12数值型int
      <br/>
      int→double自动转换 所以结果为真 返还布尔值TRUE
      <br/>
      <br/>
      <br/>
      System.out.println(('a'==97.0));
      <br/>
      "a"字符char 97.0为浮点值double
      <br/>
      char→int→double自动转换
      <br/>
      a的进制表示就是97
      <br/>
      97==97.0所以结果为真 返还布尔值TRUE
      <br/>
     </span>
    </p>
