    <h2>
     简介
    </h2>
    <p>
     <code>
      sed
     </code>
     （Stream Editor）是 Linux 中强大的
     <strong>
      流式文本处理工具
     </strong>
     ，支持正则表达式，常用于文本替换、删除、插入等操作。以下是
     <code>
      sed
     </code>
     的核心用法和实用示例：
    </p>
    <h2>
     <strong>
      基础语法
     </strong>
    </h2>
    <p>
     sed [选项] '指令' 文件
     <br/>
     sed [选项] -e '指令1' -e '指令2' 文件  # 多个指令
     <br/>
     sed [选项] -f 脚本文件 文件            # 从文件读取指令
    </p>
    <h2>
     <strong>
      常用选项
     </strong>
    </h2>
    <table>
     <thead>
      <tr>
       <th>
        选项
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         -n
        </code>
       </td>
       <td>
        仅输出处理后的行（需配合
        <code>
         p
        </code>
        指令）
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -i
        </code>
       </td>
       <td>
        直接修改原文件（谨慎使用）
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -e
        </code>
       </td>
       <td>
        指定多个编辑指令
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -r
        </code>
        或
        <code>
         -E
        </code>
       </td>
       <td>
        启用扩展正则表达式（简化语法）
       </td>
      </tr>
     </tbody>
    </table>
    <h2>
     <strong>
      核心操作指令
     </strong>
    </h2>
    <h3>
     <strong>
      基本替换
     </strong>
     ：
    </h3>
    <p>
     sed 's/apple/orange/' file.txt     # 每行第一个 apple → orange
     <br/>
     sed 's/apple/orange/g' file.txt    # 全局替换（所有 apple → orange）
    </p>
    <h3>
     <strong>
      删除行：
      <code>
       d
      </code>
     </strong>
    </h3>
    <p>
     sed '/^$/d' file.txt
    </p>
    <p>
     sed '/error/d' file.txt            # 删除包含 error 的行
     <br/>
     sed '5,10d' file.txt               # 删除第 5-10 行
    </p>
    <h3>
     <strong>
      插入/追加行：
      <code>
       i\
      </code>
      和
      <code>
       a\
      </code>
     </strong>
    </h3>
    <p>
     <strong>
      <code>
       sed '3i\插入内容' file.txt         # 在第3行前插入
      </code>
     </strong>
    </p>
    <h3>
     <strong>
      在行后追加
     </strong>
    </h3>
    <p>
     <strong>
      <code>
       sed '/pattern/a\追加内容' file.txt  # 在匹配行后追加
      </code>
     </strong>
    </p>
    <p>
    </p>
    <h2>
     <strong>
      注意事项
     </strong>
    </h2>
    <ol>
     <li>
      <p>
       <strong>
        转义特殊字符
       </strong>
       ：若替换内容含
       <code>
        /
       </code>
       ,
       <code>
        &amp;
       </code>
       ,
       <code>
        \
       </code>
       等，需用
       <code>
        \
       </code>
       转义。
      </p>
     </li>
     <li>
      <p>
       <strong>
        贪婪匹配
       </strong>
       ：
       <code>
        sed
       </code>
       默认贪婪匹配，使用
       <code>
        .*?
       </code>
       需谨慎（可用
       <code>
        [^ ]*
       </code>
       限制）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        跨平台差异
       </strong>
       ：macOS 的
       <code>
        sed
       </code>
       与 GNU
       <code>
        sed
       </code>
       存在差异（如
       <code>
        -i
       </code>
       必须指定备份后缀）。
      </p>
     </li>
    </ol>
