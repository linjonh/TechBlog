    <h3>
     <a id="1idea_0">
     </a>
     1、激活idea
    </h3>
    <p>
     使用idea eval reset 激活
     <br/>
     （1）安装方式：Preferences -&gt; plugins
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9477df6efacc44c43ee977005f5ed4b7.png">
      <br/>
      点击右侧的Manage Plugin Repositories，点击后跳出弹框点击+号，添加图中的网址：https://plugins.zhile.io
     </img>
    </p>
    <p>
     然后搜索 IDE Eval Reset安装即可
    </p>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7e5b99ffd3221ceccb974e42a17d8039.png">
      <br/>
      （2）使用方法
      <br/>
      安装插件后，IDE在 Help 下会多一个 Eval Reset 按钮。
      <br/>
      点击eval reset按钮后会出现如下图：
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ce0bf1fad56c5794ff7a41150212c44c.png"/>
     </img>
    </p>
    <p>
     按钮：Reload 用来刷新界面上的显示信息。
     <br/>
     按钮：Reset 点击会询问是否重置试用信息并重启 IDE。选择 Yes 则执行重置操作并重启 IDE 生效，选择 No 则什么也不做。（此为手动重置方式）
     <br/>
     勾选项：Auto reset before per restart 如果勾选了，则自勾选后每次重启/退出 IDE 时会自动重置试用信息，你无需做额外的事情。（此为自动重置方式）
    </p>
    <h3>
     <a id="3ideajava_17">
     </a>
     3、idea配置java
    </h3>
    <p>
     （1）首先打开IDEA点击File，在File下选择Project Structure。
    </p>
    <p>
     把里面Modules中的改成8和1.8。如下图所示
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/151e1d1532c2a0e4c6c3ceb4af6bb4c6.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eabf86cb0dd97ac9f9cf208ccea9f2dd.png">
       <br/>
       选择完成之后点击Apply 然后再点击OK。
      </img>
     </img>
    </p>
    <p>
     （2）然后点击IntelliJ IDEA，选择Preferences
    </p>
    <p>
     再点进去找到Java Compiler改成8
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e0757f9b83af10f724cf9c612c6f7317.png">
      <br/>
      选择完成之后点击Apply 然后再点击OK。
     </img>
    </p>
    <h3>
     <a id="4ideamaven_31">
     </a>
     4、idea配置maven
    </h3>
    <p>
     点击IntelliJ IDEA，选择Preferences
     <br/>
     Preferences -&gt; Build,Execution,Deploy -&gt;Build Tools -&gt; Maven如下图
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/048b1168529275dc1aff738dd7a633b9.png">
      <br/>
      设置Maven home path：放maven的settings.xml文件路径。
      <br/>
      设置User settings file：放maven的repository文件路径。
      <br/>
      最后点击Apply之后点OK即完成配置。
     </img>
    </p>
