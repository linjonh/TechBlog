    <p>
     今天，TIOBE 在更新了官网的设计之后，重磅带来了新一个月的编程语言榜单。
    </p>
    <p>
    </p>
    <h3>
     <strong>
      C++ 即将超越 Java
     </strong>
    </h3>
    <p>
    </p>
    <p>
     在最新的 6 月榜单中，TIOBE 官方用“C++ 即将超越 Java”为标题，凸显出了最大的变化。
    </p>
    <p>
     几年前我们还在谈论 Java 时往往以常青藤来形容，只是偶尔多一些“Java 老矣，尚能饭否”的疑虑，然而，随着时间的推移，这样的疑虑慢慢变成了肯定。
    </p>
    <p>
     根据 TIOBE 历年的榜单显示，2020 年 4 月，Java 仍然是 TIOBE 指数的第一名。然而同月，Java 第一的宝座就被 C 占领了。后来，在 2021 年，Python 在人工智能这条大船下势不可挡，也超越了 Java。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/74b8d4031987db389219b06ed3e536ef.png"/>
    </p>
    <p>
     上个月了，TIOBE CEO Paul Jansen 做出预判，C++ 将会是下一个超越 Java 的语言。相比 5 月 TIOBE 榜单中 Java 以 -0.74% 的趋势下滑，这个月 Java 使用率下降了 1.07%，仅占比 10.47%。与之形成强烈对比的是，占比 9.63% 的 C++ 正以 2.26% 的增速迅速俘获了大批开发者，它与 Java 之间如今只有 0.84% 的差距。
    </p>
    <p>
     而对于 C++ 如此受欢迎的原因，Paul Jansen 也进行了初步分析，其表示，“
     <strong>
      具有高性能的高级编程是 C++ 表现出色的关键原因
     </strong>
     。C++ 语言每 3 年修订一次，此类修订包含开创性的新功能，使语言与 C# 和 Java 相提并论，但没有垃圾收集器的性能损失。让我们看看未来几个月 C++ 将走向何方。”
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/214b59f158d30737d6b2e4c28b433c03.png"/>
    </p>
