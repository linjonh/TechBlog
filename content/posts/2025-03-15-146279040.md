# conda的基本使用及pycharm里设置conda环境

#### 创建conda环境

conda create --name your_env_name python=3.8

把your_env_name换成实际的conda环境名称，python=后边的根据自己的需要，选择python的版本。

![](https://i-blog.csdnimg.cn/direct/ca19e8056d464b7f969164f121516545.png)

#### 激活conda环境

conda activate your_env_name

![](https://i-blog.csdnimg.cn/direct/23cc2b6286894d3ab5a980a10b93c2e0.png)

#### 安装相关的包、库

conda install package_name

或者

pip install package_name

如果下载的太慢可以用清华镜像：

-i https://pypi.tuna.tsinghua.edu.cn/simple 直接加载后面就可以了。

比如：

conda install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple

#### 退出conda环境

conda deactivate

#### 删除conda环境

conda env remove --name your_env_name

或者

conda remove --name your_env_name \--all

#### 查看所有环境

conda env list

![](https://i-blog.csdnimg.cn/direct/4b20a50ebb7b47b1923b7d3095ac3320.png)

### 如何在pycharm中用conda环境

首先打开pycharm，看向右下角：

![](https://i-blog.csdnimg.cn/direct/66e716cf24f34d05981c39e6444e75af.png)![](https://i-blog.csdnimg.cn/direct/daf50bc78cf14ebdbea9e2c384263e02.png)![](https://i-blog.csdnimg.cn/direct/05eb290d375649ac9123e6287fc4406a.png)

比如我选的是py37这个环境

![](https://i-blog.csdnimg.cn/direct/8a7f106049b244668a694d7d59340edb.png)
