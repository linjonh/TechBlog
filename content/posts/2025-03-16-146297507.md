    <h3>
     <a id="__21">
     </a>
     🔍
     <strong>
      文明特质对照｜同步期的异与同
     </strong>
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        前5世纪·轴心时代
       </strong>
      </p>
      <ul>
       <li>
        希腊：苏格拉底街头论辩，雅典民主达峰
       </li>
       <li>
        中国：孔子周游列国，《孙子兵法》成书
       </li>
       <li>
        共性：都在战乱中催生思想突破（希波战争vs战国纷争）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        公元1世纪·帝国治理
       </strong>
      </p>
      <ul>
       <li>
        罗马：万神殿建成（125年），罗马法体系成型
       </li>
       <li>
        中国：汉武帝推"推恩令"，盐铁官营强化中央
       </li>
       <li>
        差异：罗马依赖行省制+军事殖民，汉朝依托郡县制+文官体系
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        公元13世纪·东西方碰撞
       </strong>
      </p>
      <ul>
       <li>
        拜占庭：第四次十字军东征（1204），元气大伤
       </li>
       <li>
        中国：元朝统一（1271），马可·波罗到访
       </li>
       <li>
        纽带：丝绸之路重启，郭守敬《授时历》与格里高利历精度相近
       </li>
      </ul>
     </li>
    </ol>
