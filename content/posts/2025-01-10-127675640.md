     2.登录策略（Windows，linux）

    </p>

     Windows:

    <h3 style="margin-left:0px;">
     A-5任务五 防火墙策略（Linux）
    </h3>
    <h4 style="margin-left:0px;">
     11.为防止Nmap扫描软件探测到关键信息，设置iptables防火墙策略对3306号端口进行流量处理；
    </h4>
    <pre><code class="language-bash">iptables -A INPUT -p tcp --tcp-flags ALL FIN,URG,PSH -j DROP

iptables -A INPUT -p tcp --tcp-flags SYN,RST SYN,RST -j DROP

iptables -A INPUT -p tcp --tcp-flags SYN,FIN SYN,FIN -j DROP

iptables -A INPUT -p tcp --tcp-flags SYN,SYN --dport 3306 -j DROP</code></pre>
