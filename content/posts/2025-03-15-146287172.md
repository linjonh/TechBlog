  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     在使用 Element UI 的 Tabs 组件时，动态添加 Vue 组件或页面可以通过操作
     <code>
      tabs
     </code>
     数组来实现。假设你要根据参数值来动态添加一个 Vue 页面（这里假设是一个 Vue 组件），你可以按照以下步骤操作：
    </p>
    <p>
     首先，确保你已经安装并引入了 Element UI。
    </p>
    <p>
     然后，在你的 Vue 组件中，定义一个数组来存储 Tabs 的数据，每个 Tab 对象可以包含一个
     <code>
      name
     </code>
     属性和
     <code>
      content
     </code>
     属性等，
     <code>
      content
     </code>
     属性可以用来存储对应的 Vue 组件。
    </p>
    <p>
     &lt;template&gt;
     <br/>
     &lt;div&gt;
     <br/>
     &lt;el-tabs v-model="activeTab" type="card"&gt;
     <br/>
     &lt;el-tab-pane
     <br/>
     v-for="(tab, index) in tabs"
     <br/>
     :key="index"
     <br/>
     :label="tab.name"
     <br/>
     :name="tab.name"
     <br/>
     &gt;
     <br/>
     &lt;component :is="tab.content"&gt;&lt;/component&gt;
     <br/>
     &lt;/el-tab-pane&gt;
     <br/>
     &lt;/el-tabs&gt;
     <br/>
     &lt;/div&gt;
     <br/>
     &lt;/template&gt;
    </p>
    <p>
     &lt;script&gt;
     <br/>
     import ComponentA from './ComponentA.vue';
     <br/>
     import ComponentB from './ComponentB.vue';
    </p>
    <p>
     export default {
     <!-- -->
     <br/>
     data() {
     <!-- -->
     <br/>
     return {
     <!-- -->
     <br/>
     activeTab: 'tab1',
     <br/>
     tabs: [
     <br/>
     {
     <!-- -->
     <br/>
     name: 'Tab 1',
     <br/>
     content: ComponentA
     <br/>
     }
     <br/>
     ]
     <br/>
     };
     <br/>
     },
     <br/>
     methods: {
     <!-- -->
     <br/>
     addTab(param) {
     <!-- -->
     <br/>
     // 假设 param 决定了要添加哪个组件
     <br/>
     let newComponent;
     <br/>
     if (param === 'a') {
     <!-- -->
     <br/>
     newComponent = ComponentA;
     <br/>
     } else if (param === 'b') {
     <!-- -->
     <br/>
     newComponent = ComponentB;
     <br/>
     }
    </p>
    <p>
     if (newComponent) {
     <!-- -->
     <br/>
     const newTab = {
     <!-- -->
     <br/>
     name: `Tab ${this.tabs.length + 1}`,
     <br/>
     content: newComponent
     <br/>
     };
     <br/>
     this.tabs.push(newTab);
     <br/>
     this.activeTab = newTab.name; // 将新添加的 Tab 设置为激活状态
     <br/>
     }
     <br/>
     }
     <br/>
     }
     <br/>
     };
     <br/>
     &lt;/script&gt;
    </p>
    <p>
    </p>
    <p>
     在这个例子中，
     <code>
      addTab
     </code>
     方法根据传入的
     <code>
      param
     </code>
     参数来决定添加哪个组件到 Tabs 中。你可以根据实际需要扩展
     <code>
      addTab
     </code>
     方法，比如从服务器获取组件名称或根据其他条件动态决定添加的组件。
    </p>
