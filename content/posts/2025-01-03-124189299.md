    <h2>
     <a id="_1">
     </a>
     前言
    </h2>
    <p>
     在做电路设计的过程中经常需要用到电路仿真软件对设计的电路进行仿真，以确定电路工作特性或者元件的参数取值。使用电路仿真软件可以缩短电路开发时间、降低成本、提高效率。
    </p>
    <p>
     接触过的大多数电路仿真软件都是收费的，可以免费使用的电路仿真软件主要有下面两款：
     <br/>
     TINA-TI：
     <a href="https://www.ti.com.cn/tool/cn/TINA-TI" rel="nofollow">
      https://www.ti.com.cn/tool/cn/TINA-TI
     </a>
     <br/>
     LTspice：
     <a href="https://www.analog.com/cn/design-center/design-tools-and-calculators/ltspice-simulator.html" rel="nofollow">
      https://www.analog.com/cn/design-center/design-tools-and-calculators/ltspice-simulator.html
     </a>
    </p>
    <p>
     上面两款软件在功能上基本能够满足日常需求，不过在UED/UXD（用户体验设计）方面以现在的标准来说实在是有些落伍了。前段时间发现个开源的电路仿真软件
     <strong>
      CircuitJS1
     </strong>
     。使用下来感觉功能上虽然简单些，但也基本够用；用户体验上来说比上面两个软件有比较大的提升。这篇文章将
     <br/>
     介绍使用该软件。
    </p>
    <p>
     <em>
      <code>
       本文中所有的动图因为我使用的制作软件原因线条和颜色上可能存在较大问题
      </code>
     </em>
    </p>
    <h2>
     <a id="_13">
     </a>
     基础介绍
    </h2>
    <p>
     项目地址：
     <a href="https://github.com/sharpie7/circuitjs1">
      https://github.com/sharpie7/circuitjs1
     </a>
    </p>
    <blockquote>
     <p>
      <strong>
       CircuitJS1 is an electronic circuit simulator that runs in the browser.
      </strong>
      <br/>
      It was originally written by Paul Falstad as a Java Applet.
      <br/>
      It was adapted by Iain Sharp to run in the browser using GWT.
     </p>
    </blockquote>
    <p>
     CircuitJS1是一个Java开发的程序，使用GWT将它在浏览器中运行：
     <br/>
     在线使用地址：
     <a href="http://www.falstad.com/circuit/" rel="nofollow">
      http://www.falstad.com/circuit/
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/16fccf358f51f5d07feb0fa6e43c2d57.gif#pic_center"/>
    </p>
    <p>
     另外也有使用Electron封装成本地应用的版本，支持Windows、macOS、Linux 。
     <br/>
     独立（离线）版本下载地址：
     <a href="http://www.falstad.com/circuit/offline/" rel="nofollow">
      http://www.falstad.com/circuit/offline/
     </a>
     <br/>
     Windows版本的下载后解压就可以使用：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4447a2442b2bf58f550ebc9b89e24232.gif#pic_center"/>
    </p>
    <p>
     除了上面的版本，CircuitJS1还有一些变种版本的探索，比如支持Arduino的版本：
     <br/>
     <a href="http://www.falstad.com/circuit/avr8js/" rel="nofollow">
      http://www.falstad.com/circuit/avr8js/
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/59a3f19804602bbdf587347d62038981.gif#pic_center"/>
    </p>
    <h2>
     <a id="_32">
     </a>
     使用入门
    </h2>
    <p>
     CircuitJS1使用比较简单，网站上也有分布给出说明与使用手册。另外从第一个动图可以看到软件内置了很多示例，这些电路示例也都有详细的介绍。
    </p>
    <p>
     简单使用说明：
     <a href="http://www.falstad.com/circuit/directions.html" rel="nofollow">
      http://www.falstad.com/circuit/directions.html
     </a>
     <br/>
     完整使用手册：
     <a href="https://www.bait-consulting.com/publications/circuit_simulator_manual.pdf" rel="nofollow">
      https://www.bait-consulting.com/publications/circuit_simulator_manual.pdf
     </a>
     <br/>
     预置电路说明：
     <a href="http://www.falstad.com/circuit/e-index.html" rel="nofollow">
      http://www.falstad.com/circuit/e-index.html
     </a>
    </p>
    <p>
     <em>
      下面以离线的Windows版本进行说明，当前版本为2.6.9
     </em>
    </p>
    <h3>
     <a id="_41">
     </a>
     界面与显示
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c6a3463aebed308bf147a80adecc2087.png#pic_center"/>
    </p>
    <ul>
     <li>
      界面上主要就是上图中几块内容，大部分主要的操作都可以在电路设计区域中完成；
     </li>
     <li>
      左侧
      <code>
       运行/停止运行
      </code>
      按钮红色表示电路停止运行；
     </li>
     <li>
      当电路运行时，电路中红色为负电压、绿色为正电压、灰色为接地、移动的黄点指示电流；
     </li>
     <li>
      示波器中绿色表示电压、黄色表示电流；
     </li>
     <li>
      当光标移动到元件上时元件会变蓝色，并且在右下角信息区域显示该元件的状态信息；
     </li>
    </ul>
    <h3>
     <a id="_50">
     </a>
     绘制与操作
    </h3>
    <ul>
     <li>
      <p>
       要绘制电路可以在顶部菜单
       <code>
        文件
       </code>
       中创建空白电路图，或者在
       <code>
        电路
       </code>
       中选择一个预置的电路上进行加工；
      </p>
     </li>
     <li>
      <p>
       电路设计区域中可以使用鼠标右键打开
       <code>
        绘制
       </code>
       菜单，可以从中选择需要绘制的元件；选择之后光标将变为十字形状，在空白区域按住鼠标左键拖动创建元件；可以按下
       <code>
        Esc
       </code>
       或
       <code>
        空格
       </code>
       退出元件绘制模式；
      </p>
     </li>
     <li>
      <p>
       元件引脚都有端点，如果端点连接异常则会以红色点形式显示；
      </p>
     </li>
     <li>
      <p>
       光标在元件上时可以使用滚轮滚动或是鼠标左键双击来修改元件属性，也可以使用鼠标右键打开更详细菜单；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9451970c9a819f8c82429db350c03099.gif#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       电路设计区域中鼠标滚轮滚动可以进行视图缩放，按住
       <code>
        ATL + 鼠标左键
       </code>
       可以移动视图；
      </p>
     </li>
     <li>
      <p>
       电路设计区域中可以使用
       <code>
        Ctrl + C
       </code>
       <code>
        Ctrl + V
       </code>
       <code>
        Delete
       </code>
       进行复制、粘贴、删除等操作；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9826e50723b6186689f3cbf06c65d30b.gif#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       光标在元件上时可以在鼠标右键菜单中打开示波器，可以选择自由的示波器或是固定在下方的示波器；
      </p>
     </li>
     <li>
      <p>
       可以在示波器窗口中鼠标右键对其进行详细设置；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0234c7cc63336f8b5359b5d7a977efe4.gif#pic_center"/>
      </p>
     </li>
    </ul>
    <h3>
     <a id="_65">
     </a>
     保存为文件
    </h3>
    <p>
     CircuitJS1软件上画的所有内容对于软件本身而言都是以文本形式描述的，你可以保存成文本文件（比如xxx.txt），方便之后二次使用：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d84fa6afe8285e276c3162848e00077b.gif#pic_center"/>
    </p>
    <h3>
     <a id="_69">
     </a>
     子电路绘制与使用
    </h3>
    <p>
     CircuitJS1中可以把常用的电路单元封装成模块（子电路）以供之后重复使用。要把电路封装成模块需要在电路上添加输入输出标签，然后从菜单
     <code>
      文件
     </code>
     中选择创建分支电路（子电路）：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ca3df11e68688d6b155c7643edc87076.gif#pic_center"/>
    </p>
    <p>
     创建之后就可以在绘制时使用已创建的模块了：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fb9c1a736e40b4d9e43c418cdace22d1.gif#pic_center"/>
    </p>
    <p>
     CircuitJS1是基于浏览器的应用，创建的这些全局的模块会保存在浏览器缓存中，可以通过删除缓存来删除创建的模块，删除后重新打开软件就看不到模块了：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/477e7862f13c22bb87157a76864df815.gif#pic_center"/>
    </p>
    <h2>
     <a id="_79">
     </a>
     总结
    </h2>
    <p>
     CircuitJS1使用起来非常简单，对于一般的用用来说功能和性能也都还行，可以满足一些基本的使用需求。当然在更高性能或是更加精密化的场合下还是不够用的。
    </p>
