# 【专属福利】 倒计时4天！全球云原生顶会即将盛大开幕！

![](https://i-blog.csdnimg.cn/blog_migrate/cfdb374ec7392f486a09e5803042e4c6.gif)

2023年 9月26日-28日，由Linux基金会、云原生计算基金会（CNCF）主办的KubeCon + CloudNativeCon + Open Source Summit China 2023 将于上海跨国采购会展中心盛大开幕。

**文末为CSDN社区用户准备了专属福利**

**扫码申请即有机会获得价值2400元大会门票，一定不要错过！**

**01**

**20+技术分论坛 共探云原生技术前沿**

![](https://i-blog.csdnimg.cn/blog_migrate/0de4b3e595e34d07cd55c71bf1dd5066.png)

本次 KubeCon + CloudNativeCon + Open Source Summit China 2023 涵盖了云计算、云原生领域的多个重点方向，包括云原生新手 、SDLC(软件开发生命周期)、平台工程、运维与性能、安全性、数据+处理+存储、服务网格、新兴和先进技术、云原生体验等议题进行主题分享，分享最新的前沿技术与经验，与行业大咖共话开源与云原生新发展！

![](https://i-blog.csdnimg.cn/blog_migrate/5e20fbe7d87af524d50db19aa1f8795c.jpeg)

**02**

**150+技术专家 满满干货分享**

本次大会有多位云原生技术领域崭露头角的领导者、贡献者，包含负责 Linux 生态和基金会管理的灵魂人物 Jim Zemlin、Hugging Face 机器学习工程师 Loubna Ben-Allal、Upbound 开发者倡导者 Viktor Farcic，还有本次大会的联合主席 Fog Dong 和 Kevin Wang 等等。

上百位来自全球技术专家及行业领袖将齐聚于此，走近创新前沿，聚焦云原生、开源等技术如何促进数字经济迅速发展， 分享云原生和开源的前沿洞察，深入解析热门技术在行业中的实践和落地，一同期待这场优质的技术盛会！

![](https://i-blog.csdnimg.cn/blog_migrate/bfbbe2b070c0e16cdb677495b30e95fb.jpeg)

**03**

**云原生领域人才开放交流 行业大咖深度互动**

为了吸引云原生领域人才，KubeCon特意举办了走廊交流会、欢迎酒会等活动，为开源技术爱好者们提供交流平台，共谋开源技术行业生态创新发展。

超多精彩活动、精美礼品

期待与全球开源技术同好

共赴云原生技术新浪潮！
