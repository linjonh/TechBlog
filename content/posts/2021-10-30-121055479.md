    <h4>
     <a id="CSI_5">
     </a>
     一、无线感知：从菲涅尔区模型到CSI商模型
    </h4>
    <p>
     <strong>
      1、背景
     </strong>
     ：
    </p>
    <ol>
     <li>
      2030年60岁老年人口比例比例达到23.29%
     </li>
     <li>
      老年人健康问题：心血管疾病、中风、老年痴呆、帕金森综合征
     </li>
     <li>
      共性需求：行为姿态、跌到、步态、心率、呼吸、体动、体温、
     </li>
    </ol>
    <p>
     <strong>
      2. 解决之道
     </strong>
     ：长期持续的感知
    </p>
    <ol start="4">
     <li>
      监测生命体征：呼吸心跳
     </li>
     <li>
      识别日常行为：跌到、步态、外人入侵
     </li>
     <li>
      定位和跟踪
     </li>
    </ol>
    <h4>
     <a id="WiFi4G5G_18">
     </a>
     二、基于WiFi或者4G、5G信号感知
    </h4>
    <p>
     <strong>
      1、优势：
     </strong>
     <br/>
     信号无处不在、现存设施很多、无需穿戴、隐私担忧很低
     <br/>
     <strong>
      2、无线感知原理:
     </strong>
     <br/>
     信号的传输受到空间物理的阻碍，造成多路径的反射
     <br/>
     通过信号的变化与人的行为建立映射关系
     <br/>
     <strong>
      3、无线感知的困扰：
     </strong>
     <br/>
     设备位置、人的位置、动作幅度、环境变了，映射关系就变了
     <br/>
     <strong>
      4、两个重要问题：
     </strong>
     <br/>
     WiFi的感知基础是什么？
     <br/>
     WiFi的感知边界在哪里？
     <br/>
     <strong>
      5、无线感知研究目标：
     </strong>
     <br/>
     提出一套通用的无线感知模型、理论和技术
     <br/>
     感知的机理和感知极限
     <br/>
     在室内构建一套实时、连续监测、识别系统
    </p>
    <h4>
     <a id="Fresnel_ZoneWiFi_33">
     </a>
     三、Fresnel Zone模型：WiFi无线非接触感知的一个新的新理论基础
    </h4>
    <p>
     <strong>
      1、模型简介
     </strong>
     ：
     <br/>
     将无线信号的传输路径长度分为1-6个区，每个区的路径波长相差半米波长。
     <br/>
     <strong>
      2、移动的物体影响信号的接收
     </strong>
     ：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d6df637d4155f820e11d950fe0cdccbd.png#pic_center">
      <br/>
      <strong>
       3、 物体沿着不同的轨迹走
      </strong>
      ，会穿过不同数量的Fresnel Zone，振幅上产生相应数量的波峰/波谷
      <br/>
      <strong>
       4、物体穿越FZ的速率
      </strong>
      （单位时间穿越的菲涅尔区数量）决定信号的周期与频率
      <br/>
      <strong>
       5、 特质：
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dcf430d779347f06eb649adb8e74e890.png#pic_center">
       <br/>
       <strong>
        6. 菲涅尔区模型性质推论小结：
       </strong>
       <br/>
       不同位置，不同方向，同一个行为，信号变化模式不一样
       <br/>
       不同位置、不同行为、信号变化模式可能一样
       <br/>
       少量采样的机器学习，无法实现稳定的行为识别
       <br/>
       <strong>
        7. 人体呼吸引起的信号变化是sine周期的一个片段
       </strong>
       <br/>
       <strong>
        8. WiFi感知极限：
       </strong>
       <br/>
       路径长度变化与向量旋转角度满足下面的公式
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e944cec37e6ae2dd081a00b228c09980.png#pic_center">
        <br/>
        <strong>
         9. 如何解决呼吸检测盲区、检测距离太近的问题？
        </strong>
        <br/>
        （1）用CSI商模型提高信噪比、构建正交基信号
        <br/>
        （2）WiFi CSI噪声问题：
        <br/>
        WiFi设备本身的噪声(PAU)
        <br/>
        WiFi设备的CSI相位包含随机位移的偏移（CFO,SFO,PDD）
        <br/>
        商用的WiFi设备获得CSI的相位和振幅不再正交互补
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ec38161e5dd9da116a43d22be86f98b4.png#pic_center">

         （3）CSI商模型：它可以恢复正交性，同时消除大部分CSI噪音
