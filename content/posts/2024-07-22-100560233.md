# 达梦数据库 开发版试用时间限制

达梦数据库开发版，是指用于业务场景开发时所用的数据库。它和企业版在

**功能、性能上没有任何区别**

，仅有的区别是开发版只能

**试用1年**

，1年后如何想继续用该库，就需要增加dm.key。
  
开发版，查询试用时间：

```sql
SELECT SVR_VERSION FROM V$INSTANCE;
LINEID     SVR_VERSION                                                    
---------- ---------------------------------------------------------------
1          DM Database Server x64 V8.1.0.137-Build(2019.02.14-102842)ENT 

```

其中2019.02.14即该开发版的编译时间，试用期限为2019.02.14 - 2020.02.13，2020.02.13后还想继续用该库，就需要增加dm.key文件了。
  
该试用期限不跟你什么时候开始用有关，而是所用版本编译日期+1年，如果你用是的2017.02.14的版本，而现在是2019年，这个版本肯定是无法试用了。

另外还可以通过安装包的名称来判断试用时间，如：

dm8_setup_rh7_64_ent_8.1.0.157_20190507.iso

表示20190507为该开发版的编译时间，试用期限为2019.05.17 - 2020.05.16， 2020.05.16后还想继续用该库，就需要增加dm.key文件了。
