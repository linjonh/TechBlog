    <h3>
     前言:
    </h3>
    <blockquote>
     <p>
      在看黑马javaweb的时候，发现视频中的版本是老版本,而我的是新版本 为了记录新版本的数据库界面图形化操作我打算写下这篇博客
     </p>
    </blockquote>
    <h3>
     案例
    </h3>
    <p>
     创建tb_user表  对应的结构如下
     <img alt="" height="333" src="https://i-blog.csdnimg.cn/direct/373c30c41b5a4c2dbdc378885f8e6be4.png" width="1126"/>
    </p>
    <h4>
     要求
    </h4>
    <p>
     1.id 是一行数据的唯一标识
    </p>
    <p>
     2.username 用户名字段是非空且唯一的
    </p>
    <p>
     3.name 姓名字段是不允许存储空值的
    </p>
    <p>
     4.gender 性别字段是有默认值，默认为男
    </p>
    <h4>
     操作
    </h4>
    <p>
     这里我们只创建表头部分，这里是采用SQL语句来创建表头但是我们要用图形化界面的方式来创建，可以简化很多
    </p>
    <pre><code class="hljs">create table tb_user (
    id int primary key comment 'ID,唯一标识', 
    username varchar(20) not null unique comment '用户名',
    name varchar(10) not null comment '姓名',
    age int comment '年龄',
    gender char(1) default '男' comment '性别'
) comment '用户表';</code></pre>
    <p>
     打开idea对应的界面(这里我已经提早创建好了)
    </p>

     在db01这个数据库中新建一个表叫tb_user

     然后我们先起一个表名叫tb_user
