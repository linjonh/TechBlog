    <h4>
     <a id="find__56">
     </a>
     find 查找文件或目录
    </h4>
    <p>
     <strong>
      命令参数
     </strong>
    </p>
    <ul>
     <li>
      -name 根据文件或目录名查找
     </li>
     <li>
      -type 根据文件类型查找
      <ul>
       <li>
        f 文件
       </li>
       <li>
        d 目录
       </li>
      </ul>
     </li>
     <li>
      -perm 根据权限查找
     </li>
     <li>
      -size 根据文件大小查找
     </li>
     <li>
      -uid 根据用户id查找
     </li>
     <li>
      -user 根据所属用户查找
     </li>
     <li>
      -gid 根据组id查找
     </li>
     <li>
      -group 根据所属组查找
     </li>
     <li>
      -amin 根据访问时间查找( 单位分钟)
     </li>
     <li>
      -cmin 根据文件状态修改时间查找( 文件权限,用户等信息的修改时间)
     </li>
     <li>
      -mmiin 根据文件内容修改时间查找
     </li>
     <li>
      -atime 根据访问时间查找( 单位天)
     </li>
     <li>
      -ctime 根据文件状态修改时间查找( 文件权限,用户等信息的修改时间)
     </li>
     <li>
      -mtime 根据文件内容修改时间查找
     </li>
    </ul>
    <p>
     <strong>
      命令实例
     </strong>
    </p>
    <p>
     find [搜索目录] [-参数] [内容]
     <br/>
     根据文件类型查找, 查找文件或者目录
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/df380ea45d7b8fad3eb8be4380ebf001.png">
      <br/>
      根据文件名, uid, 所属用户, 文件大小查找文件
      <br/>
      需要注意的是 -size根据大小查找文件时, 单位是K, 正数代表大于多少K的文件, 负数代表小于多少K的文件
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2c5fcf4733c48cb271a3785aaafdc27c.png">
       <br/>
       根据文件的访问/修改时间查找文件
       <br/>
       需要注意的是, *min命令的时间单位是分钟, *time命令的时间单位是天
       <br/>
       无论时间单位是什么, 整数代表多长时间内, 负数代表多长时间以外
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/83d4d32ea88e752548d21e0673512ae8.png"/>
      </img>

     <li>
      -i 忽略大小写
     </li>
     <li>
      -n 显示行号( 内容位于文件中的哪一行)
     </li>
     <li>
      -v 反向查找
     </li>
     <li>
      -w 只显示全字符都匹配的行
     </li>
     <li>
      –color 高亮显示
     </li>
    </ul>
    <p>
     <strong>
      命令实例
     </strong>
    </p>
    <p>
     grep [-参数] [内容] [文件路径]
    </p>
    <pre><code>	grep 'aa' a.txt	-- 指定文件查找内容
	grep '^a' a.txt	-- 查找以a开头的内容
	grep 'a$' a.txt	-- 查找以a结尾的内容
	grep 'aa' a.txt b.txt	-- 多文件同时查找内容
	grep -n 'aaa' a.txt	-- 显示查询结果再源文件中位于第几行
	grep -i 'aaa' a.txt	-- 忽略大小写
	grep -c 'aaa' a.txt	-- 查询匹配的内容共有几行
	grep -v 'aaa' a.txt	-- 查询不包含指定内容的行
	grep -w 'aaa' a.txt	-- 查询全字符都匹配的行,而不是部分匹配
	grep --color 'aa' a.txt	-- 匹配的字符高亮显示
	ls -l | grep passwd	-- 配合管道符过滤内容

</code></pre>
