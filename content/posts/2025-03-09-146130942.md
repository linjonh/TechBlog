    <h2>
     <a id="MyBatisPlus__7">
     </a>
     MyBatis-Plus 注解大全
    </h2>
    <p>
     MyBatis-Plus 是基于 MyBatis 的增强工具，通过注解简化了单表 CRUD 操作和复杂查询的配置。以下是常用注解的分类及详细说明：
    </p>
    <hr/>
    <h3>
     <a id="_13">
     </a>
     一、核心注解
    </h3>
    <h4>
     <a id="1_TableName_15">
     </a>
     1.
     <code>
      @TableName
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识实体类对应的数据库表名（当类名与表名不一致时使用）。
     </li>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         value
        </code>
        ：数据库表名（默认空，使用类名驼峰转下划线形式）。
       </li>
       <li>
        <code>
         schema
        </code>
        ：数据库 schema（可选）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token annotation punctuation">@TableName</span><span class="token punctuation">(</span><span class="token string">"sys_user"</span><span class="token punctuation">)</span>
<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">User</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// ...</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="2_TableId_28">
     </a>
     2.
     <code>
      @TableId
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识实体类的主键字段。
     </li>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         value
        </code>
        ：数据库主键字段名（默认空，使用字段名驼峰转下划线形式）。
       </li>
       <li>
        <code>
         type
        </code>
        ：主键生成策略（默认为
        <code>
         IdType.NONE
        </code>
        ），可选值：
        <ul>
         <li>
          <code>
           AUTO
          </code>
          ：数据库自增。
         </li>
         <li>
          <code>
           NONE
          </code>
          ：无策略（手动输入）。
         </li>
         <li>
          <code>
           INPUT
          </code>
          ：用户输入。
         </li>
         <li>
          <code>
           ASSIGN_ID
          </code>
          ：雪花算法生成 Long 类型 ID。
         </li>
         <li>
          <code>
           ASSIGN_UUID
          </code>
          ：生成 UUID 字符串。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">User</span> <span class="token punctuation">{<!-- --></span>
    <span class="token annotation punctuation">@TableId</span><span class="token punctuation">(</span>value <span class="token operator">=</span> <span class="token string">"id"</span><span class="token punctuation">,</span> type <span class="token operator">=</span> <span class="token class-name">IdType</span><span class="token punctuation">.</span><span class="token constant">ASSIGN_ID</span><span class="token punctuation">)</span>
    <span class="token keyword">private</span> <span class="token class-name">Long</span> userId<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="3_TableField_46">
     </a>
     3.
     <code>
      @TableField
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识非主键字段与数据库列的映射关系。
     </li>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         value
        </code>
        ：数据库字段名（默认空，使用字段名驼峰转下划线形式）。
       </li>
       <li>
        <code>
         exist
        </code>
        ：是否为数据库表字段（默认
        <code>
         true
        </code>
        ，若为
        <code>
         false
        </code>
        表示不参与 SQL 操作）。
       </li>
       <li>
        <code>
         fill
        </code>
        ：字段自动填充策略（如插入或更新时自动填充值），可选值：
        <ul>
         <li>
          <code>
           FieldFill.DEFAULT
          </code>
          ：不处理。
         </li>
         <li>
          <code>
           FieldFill.INSERT
          </code>
          ：插入时填充。
         </li>
         <li>
          <code>
           FieldFill.UPDATE
          </code>
          ：更新时填充。
         </li>
         <li>
          <code>
           FieldFill.INSERT_UPDATE
          </code>
          ：插入和更新时填充。
         </li>
        </ul>
       </li>
       <li>
        <code>
         select
        </code>
        ：是否参与查询（默认
        <code>
         true
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">User</span> <span class="token punctuation">{<!-- --></span>
    <span class="token annotation punctuation">@TableField</span><span class="token punctuation">(</span>value <span class="token operator">=</span> <span class="token string">"create_time"</span><span class="token punctuation">,</span> fill <span class="token operator">=</span> <span class="token class-name">FieldFill</span><span class="token punctuation">.</span><span class="token constant">INSERT</span><span class="token punctuation">)</span>
    <span class="token keyword">private</span> <span class="token class-name">LocalDateTime</span> createTime<span class="token punctuation">;</span>
    
    <span class="token annotation punctuation">@TableField</span><span class="token punctuation">(</span>exist <span class="token operator">=</span> <span class="token boolean">false</span><span class="token punctuation">)</span>
    <span class="token keyword">private</span> <span class="token class-name">String</span> remark<span class="token punctuation">;</span> <span class="token comment">// 非数据库字段</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>

    <h3>

     </a>
     二、条件构造注解
    </h3>
    <h4>
     <a id="4_Param_72">
     </a>
     4.
     <code>
      @Param
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：在 Mapper 接口方法中定义参数别名，用于 XML 或注解中的 SQL 绑定（非 MyBatis-Plus 特有，但常用）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token class-name">List</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token class-name">User</span><span class="token punctuation">&gt;</span></span> <span class="token function">selectByName</span><span class="token punctuation">(</span><span class="token annotation punctuation">@Param</span><span class="token punctuation">(</span><span class="token string">"name"</span><span class="token punctuation">)</span> <span class="token class-name">String</span> name<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_81">
     </a>
     三、结果映射注解
    </h3>
    <h4>
     <a id="5_TableLogic_83">
     </a>
     5.
     <code>
      @TableLogic
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识逻辑删除字段（标记记录是否被删除）。
     </li>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         value
        </code>
        ：未删除时的值（默认
        <code>
         0
        </code>
        ）。
       </li>
       <li>
        <code>
         delval
        </code>
        ：删除后的值（默认
        <code>
         1
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token annotation punctuation">@TableLogic</span><span class="token punctuation">(</span>value <span class="token operator">=</span> <span class="token string">"0"</span><span class="token punctuation">,</span> delval <span class="token operator">=</span> <span class="token string">"1"</span><span class="token punctuation">)</span>
<span class="token keyword">private</span> <span class="token class-name">Integer</span> isDeleted<span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>

    <h3>
     <a id="_96">
     </a>
     四、其他功能注解
    </h3>
    <h4>
     <a id="6_Version_98">
     </a>
     6.
     <code>
      @Version
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识乐观锁版本号字段（用于并发控制）。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token annotation punctuation">@Version</span>
<span class="token keyword">private</span> <span class="token class-name">Integer</span> version<span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="7_EnumValue_106">
     </a>
     7.
     <code>
      @EnumValue
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：标识枚举类中与数据库存储值对应的字段。
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">enum</span> <span class="token class-name">GenderEnum</span> <span class="token punctuation">{<!-- --></span>
    <span class="token function">MALE</span><span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token string">"男"</span><span class="token punctuation">)</span><span class="token punctuation">,</span>
    <span class="token function">FEMALE</span><span class="token punctuation">(</span><span class="token number">2</span><span class="token punctuation">,</span> <span class="token string">"女"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    
    <span class="token annotation punctuation">@EnumValue</span>
    <span class="token keyword">private</span> <span class="token keyword">final</span> <span class="token class-name">Integer</span> code<span class="token punctuation">;</span>
    <span class="token keyword">private</span> <span class="token keyword">final</span> <span class="token class-name">String</span> desc<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>
    <h4>
     <a id="8_KeySequence_120">
     </a>
     8.
     <code>
      @KeySequence
     </code>
    </h4>
    <ul>
     <li>
      <strong>
       作用
      </strong>
      ：指定主键序列名称（适用于 Oracle、PostgreSQL 等数据库的序列主键）。
     </li>
     <li>
      <strong>
       属性
      </strong>
      ：
      <ul>
       <li>
        <code>
         value
        </code>
        ：序列名。
       </li>
       <li>
        <code>
         clazz
        </code>
        ：主键类型（如
        <code>
         Long.class
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token annotation punctuation">@KeySequence</span><span class="token punctuation">(</span>value <span class="token operator">=</span> <span class="token string">"SEQ_USER"</span><span class="token punctuation">,</span> clazz <span class="token operator">=</span> <span class="token class-name">Long</span><span class="token punctuation">.</span><span class="token keyword">class</span><span class="token punctuation">)</span>
<span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">User</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// ...</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>
    <h4>

        ：是否忽略多租户拦截（默认
        <code>
         "false"
        </code>
        ）。
       </li>
       <li>
        <code>
         dynamicTableName
        </code>
        ：是否忽略动态表名拦截。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-java"><span class="token annotation punctuation">@InterceptorIgnore</span><span class="token punctuation">(</span>tenantLine <span class="token operator">=</span> <span class="token string">"true"</span><span class="token punctuation">)</span>

    </ul>

    <h3>
     <a id="_179">
     </a>
     六、常见问题
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        表名或字段名驼峰转下划线
       </strong>
       <br/>
       MyBatis-Plus 默认开启驼峰转下划线映射（如
       <code>
        userName
       </code>
       →
       <code>
        user_name
       </code>
       ）。若需关闭，配置：
      </p>
      <pre><code class="prism language-yaml"><span class="token key atrule">mybatis-plus</span><span class="token punctuation">:</span>
  <span class="token key atrule">configuration</span><span class="token punctuation">:</span>
    <span class="token key atrule">map-underscore-to-camel-case</span><span class="token punctuation">:</span> <span class="token boolean important">false</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        逻辑删除全局配置
       </strong>
       <br/>
       在
       <code>
        application.yml
       </code>
       中配置逻辑删除的默认值：
      </p>
