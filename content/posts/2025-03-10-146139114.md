    <p>
     按照网上很多教程安装，结果都失败了，后来才发现是路径问题，其实安装过程很简单（无奈）
     <br/>
     <strong>
      第一步：
     </strong>
     <br/>
     在
     <a href="https://www.ffmpeg.org/download.html#build-mac" rel="nofollow">
      官网下载
     </a>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/75a0679b5e534dbebe010ff5ece10645.png">
      <br/>
      打开页面后，可以看到FFmpeg、FFprobe、FFplay和FFserver的下载图标
      <br/>
      <strong>
       第二步：
      </strong>
      <br/>
      解压下载的文件夹，可以得到四个可执行的文件，打开可执行文件，当然MacOS会显示无法验证，可以百度一下如何打开，很简单大概在设置里搞一下就行~
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/db873fe7057348b099cb9cda9e2a5f2c.png">
       <br/>
       双击执行后的样子：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d4aaf89d1fc6405eb605e35bcfa7d290.png"/>
      </img>
     </img>
    </p>
    <p>
     <strong>
      第三步：
     </strong>
     <br/>
     很关键的一步
     <br/>
     建立一个名为FFmpeg的文件夹，在FFmpeg的文件夹之下再建立bin文件夹，然后将可执行文件都放进去（一定要建立
     <strong>
      bin文件夹
     </strong>
     ！！！）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bab25b1581404f468032fe512d01a36b.png"/>
    </p>
    <p>
     <strong>
      第四步：
     </strong>
     <br/>
     打开终端，输入指令：
    </p>
    <pre><code class="prism language-shell"><span class="token function">vi</span> ~/.zshrc
</code></pre>
    <p>
     打开的文件是无法插入的，输入
     <code>
      i
     </code>
     就能插入了，然后增加下面的内容：
    </p>
    <pre><code class="prism language-shell"><span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token string">"<span class="token environment constant">$PATH</span>:/Users/{XXX}/FFmpeg/bin"</span>
</code></pre>
    <p>
     点击
     <code>
      esc
     </code>
     ，输入
     <code>
      :wq!
     </code>
     ，按回车即可修改成功。最后再输入指令保存配置：
    </p>
    <pre><code class="prism language-powershell">source ~<span class="token operator">/</span><span class="token punctuation">.</span>zshrc
</code></pre>
    <p>
     <strong>
      第五步：
     </strong>
     <br/>
     验证是否安装成功，这里用FFmpeg举例，输入指令：
    </p>
    <pre><code class="prism language-shell">ffmpeg <span class="token parameter variable">-version</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b9bc0fc03a424199977351ebdac358b3.png"/>
    </p>
    <p>
     即安装成功
    </p>
