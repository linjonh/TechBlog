    <h2>
     一.定位
    </h2>
    <h3>
     1.作用：
    </h3>
    <p>
     灵活地改变盒子在网页中的位置
    </p>
    <p>
    </p>
    <h3>
     2.实现：
    </h3>
    <h4>
     （1）.定位模式：
    </h4>
    <p>
     position
    </p>
    <h4>
     （2）.边偏移：
    </h4>
    <p>
     设置盒子的位置（left，right，top，bottom）
    </p>
    <p>
    </p>
    <h3>
     3.相对定位：
    </h3>
    <h4>
     （1）.特点：
    </h4>
    <p>
     ①：改变位置的参照物是自己原来的位置
    </p>
    <p>
     ②：不脱离标准流的控制，仍然是占位的
    </p>
    <p>
     ③：标签显示模式的特点是不变的（原来是什么样的显示模式，之后仍然是这样的显示模式）
    </p>
    <p>
    </p>
    <h4>
     （2）.产生作用的方式
    </h4>
    <p>
     ①：设置position:relative
    </p>
    <p>
     ②：设置边偏移
    </p>
    <p>
     两者必须都进行设置，否则不会生效
    </p>
    <p>
    </p>
    <h3>
     4.绝对定位：
    </h3>
    <h4>
     （1）.定位的使用场景：
    </h4>
    <p>
     子级设置绝对定位，父级设置相对定位
    </p>
    <h4>
     （2）.特点：
    </h4>
    <p>
     ①：脱离标准流的控制，不占位
    </p>
    <p>
     ②：参照物：先找最近的已经定位的祖先元素，如果所有的祖先元素都没有定位，那么参照浏览器可视区域进行定位
    </p>
    <p>
     ③：绝对定位会使显示模式的特点改变为宽高生效的模式（具备了行内块的特点）
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
&lt;meta charset="UTF-8"&gt;
&lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
&lt;style&gt;
    img {
        width: 400px;
    }

    .news {
        /* 父级中设置相对定位 */
        position: relative;
        margin: 100px auto;
        width: 400px;
        height: 350px;
        background-color: #fff;
    }

    .news span {
        /* 在子级中设置绝对定位 */
        position: absolute;
        /* 不仅要设置position属性，还要设置边偏移 */
        right: 0;
        top: 0;
        display: block;
        width: 92px;
        height: 32px;
        background-color: rgba(0, 0, 0, 0.5);
        text-align: center;
        line-height: 32px;
        color: #fff;
    }

&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
  &lt;div class="news"&gt;
    &lt;img src="./images/news.jpg" alt=""&gt;
    &lt;span&gt;展会活动&lt;/span&gt;
    &lt;h4&gt;世界移动大会&lt;/h4&gt;
  &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <p>
    </p>
    <h3>
     5.定位居中
    </h3>
    <h4>
     （1）.实现步骤：
    </h4>
    <p>
     ①：绝对定位
    </p>
    <p>
     ②：水平和垂直的边偏移为50%
    </p>
    <p>
     ③：子级向左，上移动自身尺寸的一半（可以把左，上的外边距设置为尺寸的一半，还可以设置transform属性为translate(-50%,-50%)）
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
&lt;meta charset="UTF-8"&gt;
&lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
&lt;style&gt;
    img {
        position: absolute;
        left: 50%;
        top: 50%;

        /* 移动图片尺寸的一半 */
        /* margin-left: -149px;
        margin-top: -220px; */
        /* 两者皆可 */
        /* transform: translate(-50%, -50%); */
    }
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;img src="./images/4.jpeg" alt=""&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <p>
    </p>
    <h3>
     6.固定定位
    </h3>
    <h4>
     （1）.使用场景：
    </h4>
    <p>
     适用于元素的位置在网页滚动时不会改变的场景下
    </p>
    <h4>
     （2）.使用方式：
    </h4>
    <p>
     ①：设置position属性为fixed
    </p>
    <p>
     ②：设置边偏移定位到我们想要的位置上
    </p>
    <h4>
     （3）.特点：
    </h4>
    <p>
     ①：脱离标准流的控制，不占位
    </p>
    <p>
     ②：始终固定在相同的位置
    </p>
    <p>
     ③：参照物是浏览器窗口
    </p>
    <p>
     ④：显示模式特点会改变为行内块的特点
    </p>
    <p>
    </p>
    <p>
     7.堆叠层级z-index
    </p>
    <p>
     默认效果：按照标签书写顺序，后来者居上
    </p>
    <p>
     作用：设置定位元素的层级顺序，改变定位元素的显示顺序
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
&lt;meta charset="UTF-8"&gt;
&lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
&lt;style&gt;
    div {
        /* 设置了定位之后两者之间的堆叠层级就出现了 */
        position: absolute;
        width: 200px;
        height: 200px;
    }

    .box1 {
        background-color: blue;
        /* z-index的取值是整数，默认是0，取值越大层级越靠上（可以参考权重的概念，取值越大
        权重越大） */
        z-index: 1;
    }

    .box2 {
        background-color: blueviolet;
        left: 100px;
        top: 100px;
    }
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div class="box1"&gt;1&lt;/div&gt;
    &lt;div class="box2"&gt;2&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>

    <h2>
     二.CSS Sprites（精灵）
    </h2>
    <h3>
     1.作用：
    </h3>
    <p>
     CSS Sprites是一种网页图片应用处理方式，把网页中一些背景图片整合到一张图片文件中，再通过background-position精确地定位出背景图片的位置
    </p>
    <p>
    </p>
    <h3>
     2.使用CSS Sprites的优点：
    </h3>
    <p>
     减少服务器被请求的次数，减轻服务器的压力，提高页面加载速度
    </p>
    <p>
    </p>
    <h3>
     3.实现步骤
    </h3>
    <p>
     （1）.创建盒子，盒子的尺寸要设计得与小图的尺寸相同
    </p>
    <p>
     （2）.设置盒子背景图为Sprites图
    </p>
    <p>
     （3）.添加background-position属性改变背景图的位置
    </p>
    <p>
     ①：使用PxCook测量小图片左上角的坐标
    </p>
    <p>
     ②：取负数坐标为background-position属性值（向左上移动图片的位置）
    </p>
