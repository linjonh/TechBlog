    <h3>
     <a id="python3119_0">
     </a>
     安装python（3.11.9）
    </h3>
    <p>
     根据系统环境自行安装。
    </p>
    <h3>
     <a id="_3">
     </a>
     创建虚拟环境并激活环境
    </h3>
    <h4>
     <a id="Win_4">
     </a>
     Win环境
    </h4>
    <h5>
     <a id="_5">
     </a>
     生成虚拟环境
    </h5>
    <pre><code class="prism language-bash"><span class="token comment"># 创建虚拟环境（默认名称为 venv，可自定义）</span>
python <span class="token parameter variable">-m</span> venv myenv
</code></pre>
    <p>
     这会生成一个 myenv 文件夹，包含独立的 Python 解释器和依赖库。
    </p>
    <h5>
     <a id="_13">
     </a>
     激活虚拟环境
    </h5>
    <pre><code class="prism language-bash"><span class="token comment"># 在 CMD 中运行</span>
myenv<span class="token punctuation">\</span>Scripts<span class="token punctuation">\</span>activate.bat

<span class="token comment"># 在 PowerShell 中运行</span>
myenv<span class="token punctuation">\</span>Scripts<span class="token punctuation">\</span>Activate.ps1
</code></pre>
    <p>
     激活后，命令行提示符会显示虚拟环境名称（如 (myenv) C:…）。
    </p>
    <h5>
     <a id="_24">
     </a>
     退出虚拟环境
    </h5>
    <pre><code class="prism language-bash">deactivate
</code></pre>
    <h4>
     <a id="Mac_29">
     </a>
     Mac环境
    </h4>
    <h5>
     <a id="_30">
     </a>
     生成虚拟环境
    </h5>
    <pre><code class="prism language-bash">python3 <span class="token parameter variable">-m</span> venv myenv
</code></pre>
    <p>
     这会生成一个 myenv 文件夹，包含独立的 Python 解释器和依赖库。
    </p>
    <h5>
     <a id="font_stylecolorrgba0_0_0_09backgroundcolorrgb252_252_252font_37">
     </a>
     <strong>
      激活虚拟环境
     </strong>
    </h5>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> myenv/bin/activate
</code></pre>
    <p>
     激活后，终端提示符会显示虚拟环境名称（如 (myenv) $）。
    </p>
    <h5>
     <a id="font_stylecolorrgba0_0_0_09backgroundcolorrgb252_252_252font_44">
     </a>
     <strong>
      退出虚拟环境
     </strong>
    </h5>
    <pre><code class="prism language-bash">deactivate
</code></pre>
    <h3>
     <a id="github_49">
     </a>
     github仓库
    </h3>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone https://github.com/mannaandpoem/OpenManus.git
</code></pre>
    <h3>
     <a id="_54">
     </a>
     进入仓库
    </h3>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> OpenManus
</code></pre>
    <h3>
     <a id="_59">
     </a>
     安装依赖
    </h3>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token parameter variable">-r</span> requirements.txt
</code></pre>
    <h3>
     <a id="_64">
     </a>
     配置
    </h3>
    <p>
     进入仓库根目录下的
     <code>
      config
     </code>
     ，复制
     <code>
      config.example.toml
     </code>
     ，重名为
     <code>
      config.toml
     </code>
     。
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># Global LLM configuration</span>
<span class="token punctuation">[</span>llm<span class="token punctuation">]</span>
model <span class="token operator">=</span> <span class="token string">"deepseek-chat"</span>
base_url <span class="token operator">=</span> <span class="token string">"https://api.deepseek.com/v1"</span>
api_key <span class="token operator">=</span> <span class="token string">"你的API key"</span>
max_tokens <span class="token operator">=</span> <span class="token number">4096</span>
temperature <span class="token operator">=</span> <span class="token number">0.0</span>

<span class="token comment"># [llm] #AZURE OPENAI:</span>
<span class="token comment"># api_type= 'azure'</span>
<span class="token comment"># model = "YOUR_MODEL_NAME" #"gpt-4o-mini"</span>
<span class="token comment"># base_url = "{YOUR_AZURE_ENDPOINT.rstrip('/')}/openai/deployments/{AZURE_DEPOLYMENT_ID}"</span>
<span class="token comment"># api_key = "AZURE API KEY"</span>
<span class="token comment"># max_tokens = 8096</span>
<span class="token comment"># temperature = 0.0</span>
<span class="token comment"># api_version="AZURE API VERSION" #"2024-08-01-preview"</span>

<span class="token comment"># Optional configuration for specific LLM models</span>
<span class="token punctuation">[</span>llm.vision<span class="token punctuation">]</span>

base_url <span class="token operator">=</span> <span class="token string">"https://api.deepseek.com/v1"</span>
api_key <span class="token operator">=</span> <span class="token string">"你的API key"</span>
</code></pre>
    <h3>
     <a id="_92">
     </a>
     运行命令
    </h3>
    <pre><code class="prism language-bash">python main.py
</code></pre>
