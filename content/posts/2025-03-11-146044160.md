     Centos 安装docker,docker-compose
    </h1>
   </div>

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     如果之前有装过，请先删除。用以下命令
    </p>
    <p>
     yum remove docker-ce
    </p>
    <p>
     rm -rf /var/lib/docker
    </p>
    <p>
     <br/>
     sudo dnf remove docker \
     <br/>
     docker-client \
     <br/>
     docker-client-latest \
     <br/>
     docker-common \
     <br/>
     docker-latest \
     <br/>
     docker-latest-logrotate \
     <br/>
     docker-logrotate \
     <br/>
     docker-engine
    </p>
    <p>
     通过下列命令安装：
    </p>
    <p>
     <strong>
      安装docker
     </strong>
    </p>
    <p>
     1.设置仓库
    </p>
    <pre><code>sudo dnf -y install dnf-plugins-core</code></pre>
    <p>
     2.设置镜像源
    </p>
    <pre><code>sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
//切换清华镜像
sed -i 's+https://download.docker.com+https://mirrors.tuna.tsinghua.edu.cn/docker-ce+' /etc/yum.repos.d/docker-ce.repo</code></pre>
    <p>
     3.安装
    </p>
    <pre><code>sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin</code></pre>
    <p>
     4.启动
    </p>
    <pre><code>sudo systemctl start docker
#设置开机启动
sudo systemctl enable --now docker</code></pre>
    <p>
    </p>
    <p>
     docker配置daemon.json
    </p>
    <p>
     # 创建目录
     <br/>
     mkdir -p /etc/docker
    </p>
    <p>
     # 复制内容
     <br/>
     tee /etc/docker/daemon.json &lt;&lt;-'EOF'
     <br/>
     {
     <!-- -->
     <br/>
     "registry-mirrors": [
     <br/>
     "http://hub-mirror.c.163.com",
     <br/>
     "https://mirrors.tuna.tsinghua.edu.cn",
     <br/>
     "http://mirrors.sohu.com",
     <br/>
     "https://ustc-edu-cn.mirror.aliyuncs.com",
     <br/>
     "https://ccr.ccs.tencentyun.com",
     <br/>
     "https://docker.m.daocloud.io",
     <br/>
     "https://docker.awsl9527.cn"
     <br/>
     ]
     <br/>
     }
     <br/>
     EOF
    </p>
    <p>
     # 重新加载配置
     <br/>
     systemctl daemon-reload# 重启Docker
     <br/>
     systemctl restart docker
    </p>
    <p>
     <strong>
      安装docker-compose
     </strong>
    </p>
    <p>
     <br/>
    </p>
    <pre><code class="hljs"># 下载并重命名
sudo curl -L https://github.com/docker/compose/releases/download/v2.21.0/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose
# 设置可执行权限
chmod +x /usr/local/bin/docker-compose

安装后查看版本
docker-compose -v</code></pre>
    <p>
