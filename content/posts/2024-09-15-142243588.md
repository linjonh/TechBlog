### 项目介绍

懒盘搜索是一个综合性的短剧网盘搜索引擎源码项目，它支持对多个知名网盘如百度网盘、阿里云盘、夸克网盘、迅雷网盘的资源进行索引和搜索。此项目基于MIT许可证开源，提供了一个简易的方式来搭建个性化的网盘搜索引擎网站，适合想要自建网盘搜索服务的开发者或爱好者。

#### 主要特点

* **多网盘支持**
  ：集成百度、阿里、夸克及迅雷网盘资源。
* **后台管理**
  ：拥有简单的后台管理系统，可管理资源、分类、用户等。
* **响应式设计**
  ：支持手机和电脑端访问。
* **自定义配置**
  ：允许用户调整搜索引擎的关键数据和前端展示。

### 项目快速启动

#### 环境准备

确保您的开发环境已具备以下条件：

1. **Node.js**
   : 至少安装Node.js的一个稳定版本。
2. **Git**
   : 安装用于克隆项目代码。
3. **npm**
   或
   **pnpm**
   : 建议使用
   `pnpm`
   作为包管理器，也可以选择npm。

#### 实施步骤

1. **克隆项目**

   ```bash
   git clone https://github.com/Xwudao/lzpan_search.git

   ```
2. **安装依赖**
     
   进入项目目录并使用pnpm安装所有必要的依赖：

   ```bash
   cd lzpan_search
   pnpm i

   ```
3. **配置修改**
     
   根据需要，您可以编辑
   `src/data/data.json`
   来定制搜索引擎的数据来源和
   `public/index.html`
   、
   `public/mobile.html`
   中的页面信息。
4. **构建应用**
     
   执行构建命令来准备生产环境部署：

   ```bash
   npm run build

   ```
5. **部署**
     
   构建完成后，将
   `dist`
   目录下的文件上传至您的网站服务器根目录。
