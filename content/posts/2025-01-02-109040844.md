# 小程序 如何调整视频帧速率\_如何在NVIDIA驱动程序中设置最大帧速率

![小程序 如何调整视频帧速率](https://i-blog.csdnimg.cn/blog_migrate/2825da367220ea9f83f20fdb58b25406.png)

小程序 如何调整视频帧速率

![A NVIDIA RTX-20 series gaming laptop](https://i-blog.csdnimg.cn/blog_migrate/688b074ddde1e60e27755a8dc6024a87.png)


[NVIDIA](https://nvidianews.nvidia.com/multimedia)
[英伟达](https://nvidianews.nvidia.com/multimedia)

At the beginning of 2020, NVIDIA released new GeForce drivers with a highly requested feature. You can now cap your PC’s framerates—either for all games on your PC or just for specific games. Here’s how.

2020年初，NVIDIA发布了新的具有强大功能的GeForce驱动程序。 现在，您可以为PC上的所有游戏或特定游戏设置PC的帧速率。 这是如何做。

### 为什么要限制FPS？ ( Why Would You Want to Cap Your FPS? )

This feature is particularly useful on Windows gaming laptops with NVIDIA hardware, as you can stop your GPU from running as fast as it can. This will save battery power and reduce heat consumption, letting you game for longer when you’re away from a power outlet.

在具有NVIDIA硬件的Windows游戏笔记本电脑上，此功能特别有用，因为您可以阻止GPU尽快运行。 这样可以节省电池电量并减少热量消耗，使您在远离电源插座时可以玩得更长久。

You might also find the cap useful to reduce screen tearing in games. You can try setting the maximum FPS to your display’s
[refresh rate](https://www.howtogeek.com/359691/what-is-a-monitors-refresh-rate-and-how-do-i-change-it/)
if you don’t have a
[G-Sync](https://www.howtogeek.com/270672/how-to-enable-optimize-and-tweak-nvidia-g-sync/)
or
[FreeSync](https://www.howtogeek.com/401792/what-is-g-sync-compatible-and-how-to-enable-g-sync-on-freesync-monitors/)
-enabled monitor with
[variable refresh rates](https://www.howtogeek.com/228735/g-sync-and-freesync-explained-variable-refresh-rates-for-gaming/)
. The NVIDIA Control Panel also says this feature can “[reduce] system latency in certain scenarios.”

您可能还会发现顶盖有助于减少游戏中的屏幕撕裂。 如果您没有启用
[G-Sync](https://www.howtogeek.com/270672/how-to-enable-optimize-and-tweak-nvidia-g-sync/)
或
[FreeSync](https://www.howtogeek.com/401792/what-is-g-sync-compatible-and-how-to-enable-g-sync-on-freesync-monitors/)
且
[刷新率可变的](https://www.howtogeek.com/228735/g-sync-and-freesync-explained-variable-refresh-rates-for-gaming/)
显示器，可以尝试将最大FPS设置为显示器的
[刷新率](https://www.howtogeek.com/359691/what-is-a-monitors-refresh-rate-and-how-do-i-change-it/)
。 NVIDIA控制面板还表示，此功能可以“减少某些情况下的系统延迟”。

### 如何为所有游戏设置最大FPS ( How to Set a Maximum FPS for All Games )

This feature is built into the NVIDIA Control Panel. To open it, right-click your Windows desktop and select “NVIDIA Control Panel.”

此功能内置在NVIDIA控制面板中。 要打开它，请右键单击Windows桌面，然后选择“ NVIDIA控制面板”。

(If you don’t see this menu option, you likely don’t have NVIDIA’s drivers installed.)

(如果没有看到此菜单选项，则可能未安装NVIDIA的驱动程序。)

![Launching the NVIDIA Control Panel from the Windows desktop](https://i-blog.csdnimg.cn/blog_migrate/226eca4caca59a7dbf4b324adc5201da.png)

Select “Manage 3D Settings” under 3D Settings at the left side of the NVIDIA Control Panel window.

在“ NVIDIA控制面板”窗口左侧的3D设置下选择“管理3D设置”。

To control the maximum frame rate for all applications on your PC, ensure the “Global Settings” tab is selected.

要控制PC上所有应用程序的最大帧速率，请确保已选择“全局设置”选项卡。

![Managing global 3D settings in the NVIDIA Control Panel](https://i-blog.csdnimg.cn/blog_migrate/029a81c361d84e73473dd81a6fb28fb3.png)

In the list of settings, click the box to the right of “Max Frame Rate.” By default, this option is disabled, and there is no maximum frame rate.

在设置列表中，单击“最大帧速率”右侧的框。 默认情况下，此选项是禁用的，并且没有最大帧速率。

To set a maximum frame rate, select “On” and choose your maximum frames per second (FPS.)

要设置最大帧速率，请选择“开”，然后选择每秒最大帧数(FPS)。

![Enabling a max frame rate globally in NVIDIA's drivers](https://i-blog.csdnimg.cn/blog_migrate/fe5dbb60ac3bd1e600a632b19b3cdd0c.png)

Click “Apply” at the bottom of the window to save your settings.

单击窗口底部的“应用”以保存您的设置。

![Applying NVIDIA Control Panel settings on Windows 10](https://i-blog.csdnimg.cn/blog_migrate/c6ba2d29eb0df924ba305af7895a263a.png)

### 如何控制特定游戏的最大帧速率 ( How to Control Max Frame Rate for Specific Games )

You can also control the maximum frame rate setting for individual applications. Click the “Program Settings” tab at the top of the list of settings. Under “Select a program to customize,” choose the app you want to control.

您还可以控制各个应用程序的最大帧速率设置。 单击设置列表顶部的“程序设置”选项卡。 在“选择要自定义的程序”下，选择要控制的应用。

If a game doesn’t appear in the list, you can click “Add” and specify its .exe file.

如果游戏没有出现在列表中，则可以单击“添加”并指定其.exe文件。

![Managing 3D settings for an individual game in the NVIDIA Control Panel.](https://i-blog.csdnimg.cn/blog_migrate/9938f4d7af76c996626421add0619b67.png)

Look for the “Max Frame Rate” option, click it, and select your desired setting. By default, each game is set to “Use global setting”—it’ll use whatever setting you chose on the Global Settings tab.

查找“最大帧速率”选项，单击它，然后选择所需的设置。 默认情况下，每个游戏都设置为“使用全局设置”，它将使用您在“全局设置”选项卡上选择的任何设置。

However, you can select different settings here. For example, you might leave the maximum frame rate option disabled on the “Global Settings” tab and configure a different maximum frame rate for each game you want to cap. Or, you might set a maximum frame rate globally and exempt individual games from the cap. It’s up to you.

但是，您可以在此处选择其他设置。 例如，您可以在“全局设置”选项卡上禁用“最大帧速率”选项，并为要设置的每个游戏配置不同的最大帧速率。 或者，您可以全局设置最大帧速率，并从上限中排除单个游戏。 由你决定。

![Setting a frame rate cap for a specific game in NVIDIA's GeForce drivers.](https://i-blog.csdnimg.cn/blog_migrate/1d1025217d5651f4fd4f653157d33760.png)

Be sure to click “Apply” to save your settings.

确保单击“应用”以保存您的设置。
