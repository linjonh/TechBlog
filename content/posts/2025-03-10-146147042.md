    <h3 id="4.2%20%E5%AF%B9%E8%B1%A1%E7%89%B9%E5%BE%81" name="4.2%20%E5%AF%B9%E8%B1%A1%E7%89%B9%E5%BE%81">
     4.2 对象特征
    </h3>
    <p>
     对象的初始化和清理：C++中，每个对象都有初始设置和对象销毁前的清理数据的设置。
    </p>
    <p>
    </p>
    <h4 id="4.2.1%20%E6%9E%84%E9%80%A0%E5%87%BD%E6%95%B0%E5%92%8C%E6%9E%90%E6%9E%84%E5%87%BD%E6%95%B0" name="4.2.1%20%E6%9E%84%E9%80%A0%E5%87%BD%E6%95%B0%E5%92%8C%E6%9E%90%E6%9E%84%E5%87%BD%E6%95%B0">
     4.2.1 构造函数和析构函数
    </h4>
    <p>
     C++中利用构造函数和析构函数对对象进行初始化和清理。这两个函数会被编译器自动调用，完成对象的初始化和清理。如果程序员不提供构造和析构，编译器会提供构造函数和析构函数，但是是空的。
    </p>
    <p>
     <strong>
      构造函数：
     </strong>
     创建对象时为对象的成员属性赋值，构造函数由编译器自动调用，无须手动调用。
    </p>
    <p>
     语法：类名（）{}
    </p>
    <p>
     1.构造函数，没有返回值也不写void；
    </p>
    <p>
     2.函数名与类名相同；
    </p>
    <p>
     3.构造函数可以有参数，因此可以发生重载；
    </p>
    <p>
     4.程序在调用对象时会自动调用构造，无须手动调用，且只调用一次。
    </p>
    <p>
     <strong>
      析构函数：
     </strong>
     对象销毁前系统自动调用，执行一些清理工作。
    </p>
    <p>
     语法：~类名（）{}
    </p>
    <p>
     1.析构函数，没有返回值也不写void；
    </p>
    <p>
     2.函数名与类名相同，在函数名前加上符号~；
    </p>
    <p>
     3.析构函数不可以有函数，因此不可以发生重载；
    </p>
    <p>
     4.程序在对象销毁前会自动调用析构，无须手动调用，且只调用一次。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;

class Person
{
public:
    //构造函数 初始化对象
    Person()
    {
        cout&lt;&lt;"Person构造函数的调用"&lt;&lt;endl;
    }

    //析构函数 销毁/清理对象
    ~Person()
    {
        cout&lt;&lt;"Person析构函数的调用"&lt;&lt;endl;
    }
};

void test01()
{
    Person p;//栈上的数据，该函数执行完后，p这个对象会被释放
}

int main()
{
    //对象的初始化和清理

    test01();
    Person p;

    system("pause");
    return 0;
}</code></pre>

     4.2.2 构造函数的分类
    </h4>
    <p>
     按参数分：有参构造、无参构造；
    </p>
    <p>
     按类型分：普通构造、拷贝构造。
    </p>
    <p>
     调用方式：括号法、显示法、隐式转换法。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;

{

    //无参(默认)构造

        age=a;

    Person(const Person &amp;p)

        age=p.age;

    }

    Person p2(21);//有参构造函数调用
    Person p3(p2);//拷贝构造函数调用

    //调用：显示法
    cout&lt;&lt;"显示法调用构造函数："&lt;&lt;endl;

    Person P6=Person(p5);//拷贝构造

    Person(21);//表示一个匿名对象，在等式左边的P2就是给他取的名字，匿名对象执行后会立即回收。
    cout&lt;&lt;"匿名对象清理后执行了这句代码"&lt;&lt;endl;

    //PS:不要用拷贝构造 初始化匿名对象,编译器会认为Person(p3);是一个对象声明Person p3;
    //Person(p3);


    //调用：隐式转换法
    cout&lt;&lt;"显示法调用构造函数："&lt;&lt;endl;
    Person p7=21;//有参构造
    Person p8=p7;//拷贝构造
}

int main()
{
    test01();
    return 0;

    </p>

      2.默认析构函数（无参，函数体为空）
     </li>
     <li>

     </li>
    </ol>

    <h4 id="4.2.5%20%E6%B7%B1%E6%8B%B7%E8%B4%9D%E4%B8%8E%E6%B5%85%E6%8B%B7%E8%B4%9D" name="4.2.5%20%E6%B7%B1%E6%8B%B7%E8%B4%9D%E4%B8%8E%E6%B5%85%E6%8B%B7%E8%B4%9D">
     4.2.5 深拷贝与浅拷贝
    </h4>
    <p>
     浅拷贝：编译器提供的拷贝函数，简单的赋值拷贝操作；
    </p>
    <p>
     缺点：容易导致堆区的重复释放，利用深拷贝解决。
    </p>
    <p>
     深拷贝：在堆区重新申请空间，进行拷贝操作，而不是与被拷贝的指针指向相同的空间。
    </p>
    <p>
     PS：如果属性有在堆区开辟的，一定要自己定义拷贝构造函数，防止浅拷贝中出现的问题。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;

class Person
{
public:
    Person()
    {
        cout&lt;&lt;"person的默认构造函数"&lt;&lt;endl;
    }
    Person(int a,int h)
    {
        age=a;
        height=new int(h);
        cout&lt;&lt;"Person的有参构造函数的调用"&lt;&lt;endl;
    }
    //自己实现拷贝构造函数，解决浅拷贝的问题
    Person(const Person &amp;p)
    {
        age=p.age;
        height=p.height;//编译器写的（浅拷贝）
        height= new int(*p.height);//深拷贝操作,另外开辟空间
        cout&lt;&lt;"Person的拷贝构造函数的调用"&lt;&lt;endl;
    }

    ~Person()
    {
        //析构的作用，将堆区new的数据手动释放
        if(height!=NULL)//若指针不为空，则需要释放
        {
            delete height;//P2先释放，完了之后P也需要释放，但两个对象的指针操作的是同一个堆区中的地址，造成重复释放的非法操作，因此会报错
            height=NULL;//防止野指针出现，将指针置空
        }
        cout&lt;&lt;"Person析构函数的调用"&lt;&lt;endl;
    }

    int age;
    int * height;
};

void test01()
{
    Person p(28,160);

    Person p2(p);
    cout&lt;&lt;"p2的年龄为："&lt;&lt;p2.age&lt;&lt;" 身高为："&lt;&lt;*p2.height&lt;&lt;endl;
}

int main()
{
    test01();

    return 0;
}</code></pre>

    </h4>
    <p>

    <h4 id="4.2.7%20%E7%B1%BB%E5%AF%B9%E8%B1%A1%E4%BD%9C%E4%B8%BA%E7%B1%BB%E6%88%90%E5%91%98" name="4.2.7%20%E7%B1%BB%E5%AF%B9%E8%B1%A1%E4%BD%9C%E4%B8%BA%E7%B1%BB%E6%88%90%E5%91%98">
     4.2.7 类对象作为类成员
    </h4>
    <p>
     C++中类的成员可以是另一个类的对象，称为对象成员。
    </p>
    <p>
     注意对象作为成员时，两种对象的构造和析构函数的顺序。（先构造其他类，再构造本类，先析构本类，再析构其他类）
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
#include &lt;string&gt;

//对象成员
class Phone
{
public:
    //手机品牌
    string PName;
    Phone(string pname)
    {
        cout&lt;&lt;"Phone的构造函数的调用"&lt;&lt;endl;
        PName=pname;
    }

    ~Phone()
    {
        cout&lt;&lt;"Phone析构函数的调用"&lt;&lt;endl;
    }
};

class Person
{
public:
    //P(pname)相当于Phone P=pname; 隐式转换法
    Person(string name,string pname):Name(name),P(pname)
    {
        cout&lt;&lt;"Person的构造函数的调用"&lt;&lt;endl;
    }


    ~Person()
    {
        cout&lt;&lt;"Person析构函数的调用"&lt;&lt;endl;
    }

    string Name;
    Phone P;
};

void test01()
{
    Person p("张三","iPhone18");

    cout&lt;&lt;p.Name&lt;&lt;"拿着"&lt;&lt;p.P.PName&lt;&lt;endl;
}



int main()
{
    test01();

    return 0;
}</code></pre>

    <h4 id="4.2.8%20%E9%9D%99%E6%80%81%E6%88%90%E5%91%98" name="4.2.8%20%E9%9D%99%E6%80%81%E6%88%90%E5%91%98">
     4.2.8 静态成员
    </h4>
    <p>
     静态成员是指在成员变量和成员函数卡加static关键字，静态成员都有三种访问权限。
    </p>
    <p>
     静态成员变量：
    </p>
    <ul>
     <li>
      所有对象共享同一份数据
     </li>
     <li>
      在编译阶段分配内存（程序运行前）
     </li>
     <li>
      类内声明，类外初始化
     </li>
    </ul>
    <p>
     静态成员函数
    </p>
    <ul>
     <li>
      所有对象共享同一个函数
     </li>
     <li>
      静态成员函数只能访问静态成员变量
     </li>
    </ul>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
#include &lt;string&gt;

//静态成员
class Person
{
public:
    //静态成员变量
    static int A;//类内声明
    int B;

    //静态成员函数
    static void func()
    {
        A=44;
        //B=22;//静态成员函数访问非静态成员变量，报错,无法区分是哪个对象的B
        cout&lt;&lt;"静态成员函数调用"&lt;&lt;endl;
    }
};

//类外初始化
int Person::A=100;

void test01()
{
    Person p;
    cout&lt;&lt;p.A&lt;&lt;endl;

    Person p2;
    p2.A=200;
    
    //所有对象共享同一份数据,因此有两种访问方式：通过对象访问；通过类名访问
    cout&lt;&lt;p.A&lt;&lt;endl;
    cout&lt;&lt;Person::A&lt;&lt;endl;
}

void test02()
{
    //两种访问方式：通过对象访问；通过类名访问
    Person p;
    p.func();

    Person::func();
    cout&lt;&lt;p.A&lt;&lt;endl;
}

int main()
{
    test01();
    test02();
    return 0;
}</code></pre>

    <h4 id="4.2.9%20%E6%88%90%E5%91%98%E5%8F%98%E9%87%8F%E5%92%8C%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0%E7%9A%84%E5%AD%98%E5%82%A8" name="4.2.9%20%E6%88%90%E5%91%98%E5%8F%98%E9%87%8F%E5%92%8C%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0%E7%9A%84%E5%AD%98%E5%82%A8">
     4.2.9 成员变量和成员函数的存储
    </h4>
    <p>
     类内的成员变量和成员函数分开存储，只有非静态成员变量才属于类的对象上的。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
#include &lt;string&gt;

//静态成员
class Person1
{

};

class Person2
{
    int A;//非静态成员变量 
};

class Person3
{
    int A;
    static int B;//静态成员变量
};

int Person3::B=9;

class Person4
{
    int A;
    static int B;
    void func(){}//非静态成员函数
};

class Person5
{
    int A;
    static int B;
    void func(){}//非静态成员函数
    static void func2(){};
};

void test01()
{
    Person1 p1;
    //空对象占用内存为1，为了区分空对象占内存的位置，每个空对象有一个唯一的地址
    cout&lt;&lt;"size of p1="&lt;&lt;sizeof(p1)&lt;&lt;endl;

    Person2 p2;
    //有非静态成员变量，占4字节  属于类的对象上的数据
    cout&lt;&lt;"size of p2="&lt;&lt;sizeof(p2)&lt;&lt;endl;

    Person3 p3;
    //有静态成员变量  不属于类的对象上的数据
    cout&lt;&lt;"size of p3="&lt;&lt;sizeof(p3)&lt;&lt;endl;

    Person4 p4;
    //非静态成员函数  不属于类的对象上的数据
    cout&lt;&lt;"size of p4="&lt;&lt;sizeof(p4)&lt;&lt;endl;

    Person5 p5;
    //静态成员函数  不属于类的对象上的数据
    cout&lt;&lt;"size of p5="&lt;&lt;sizeof(p5)&lt;&lt;endl;
}

int main()
{
    test01();

    return 0;
}</code></pre>

    <h4 id="4.2.10%20this%E6%8C%87%E9%92%88" name="4.2.10%20this%E6%8C%87%E9%92%88">
     4.2.10 this指针
    </h4>
    <p>
     每一个非静态成员函数只会产生一个函数实例，所有同类中的多个对象会公用一块代码。
    </p>
    <p>
     C++提供this指针来指向被调用的成员函数所属的对象。this指针是隐含每一个非静态成员函数内的一种指针，不需要定义，直接使用即可。
    </p>
    <p>
     用途：
    </p>
    <ul>
     <li>
      当形参和成员变量同名时，用this指针来区分
     </li>
     <li>
      在类的非静态成员函数中返回对象本身，可使用return *this。
     </li>
    </ul>
    <p>
     PS：用Person&amp;定义返回值类型，是因为可以一直对同一个空间操作，用Person定义返回值类型表示值返回，会复制一份新的数据（按照本体p2创建了新的数据，而不是返回的p2本体），调用了拷贝构造函数。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
#include &lt;string&gt;

class Person
{
public:
    int age;
    Person(int age)
    {
        //age=age;//报错
        //this指针指向被调用的成员函数所属的对象p1
        this-&gt;age=age;
    }
//用Person&amp;定义返回值类型，是因为可以一直对同一个空间操作，用Person定义返回值类型表示值返回，会复制一份新的数据（按照本体p2创建了新的数据，而不是返回的p2本体），调用了拷贝构造函数
     Person&amp; PersonAddAge(Person &amp;p)
     {
        this-&gt;age+=p.age;
        //this指向p2的指针，*p2指向p2本体
        return *this;
     }
};

//解决名称冲突
void test01()
{
    Person p1(18);
    cout&lt;&lt;p1.age&lt;&lt;endl;
}

//用*this 返回对象本身
void test02()
{
    Person p1(31);

    Person p2(31);
    p2.PersonAddAge(p1);
    cout&lt;&lt;p2.age&lt;&lt;endl;
    p2.PersonAddAge(p1).PersonAddAge(p1);//用this*返回才能链式追加
    cout&lt;&lt;p2.age&lt;&lt;endl;

}

int main()
{
    test01();
    test02();
    return 0;
}</code></pre>

    <h4 id="4.2.11%20%E7%A9%BA%E6%8C%87%E9%92%88%E8%AE%BF%E9%97%AE%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0" name="4.2.11%20%E7%A9%BA%E6%8C%87%E9%92%88%E8%AE%BF%E9%97%AE%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0">
     4.2.11 空指针访问成员函数
    </h4>
    <p>
     C++中空指针可以调用成员函数，但需要注意有没有用this指针。如果用到this指针，需要加以判断保证代码的健壮性。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;

//空指针调用成员函数
class Person
{
public:
    void showClassName()
    {
        cout&lt;&lt;"this is person class"&lt;&lt;endl;
    }

    void showPersonAge()
    {
        if(this==NULL)
        {
            return;
        }
        //传入指针为空，报错  在前面加一个空指针的判断
        cout&lt;&lt;"age="&lt;&lt;this-&gt;age&lt;&lt;endl;
    }

    int age;
};

void test01()
{
    Person *p=NULL;
    p-&gt;showClassName();
    p-&gt;showPersonAge();
}

int main()
{
    test01();
    return 0;
}</code></pre>

    <h4 id="4.2.12%20const%E4%BF%AE%E9%A5%B0%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0" name="4.2.12%20const%E4%BF%AE%E9%A5%B0%E6%88%90%E5%91%98%E5%87%BD%E6%95%B0">
     4.2.12 const修饰成员函数
    </h4>
    <p>
     常函数：
    </p>
    <ul>
     <li>
      成员函数后加const后称为常函数；
     </li>
     <li>
      常函数内不可用修改成员属性；
     </li>
     <li>
      成员属性声明时加关键词mutable后，在常函数中依然可以修改
     </li>
    </ul>
    <p>
     常对象：
    </p>
    <ul>
     <li>
      声明对象前加const称该对象为常对象；
     </li>
     <li>
      <p>
       常对象不允许修改指针指向的值；
      </p>
     </li>
     <li>
      常对象只能调用常函数
     </li>
    </ul>
    <p>
     代码如下：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;

//常函数
class Person
{
public:
    //this指针的本质是一个指针常量Person * const this 指针的指向是不可修改的
    //后面加的const相当于const Person * const this，使this指向的值也不可修改
    void showPerson() const
    {
        this-&gt;b=99;
        //this=NULL;//this的指针指向不能修改
        cout&lt;&lt;"this is person class"&lt;&lt;endl;
    }
    Person(){}//不写默认构造函数会报错实例化的常对象没有初始化

    void func(){}
    int age;
    mutable int b;
};

void test01()
{
    Person p;
    p.showPerson();
}

void test02()
{
    const Person p;
    //p.age=99;//报错 常对象不允许修改指针指向的值
    p.b=88;
    p.showPerson();
    //p.func();//报错 常对象不能调用非常函数
}

int main()
{
    test01();
    test02();
    return 0;
}</code></pre>
