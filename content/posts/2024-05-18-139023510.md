    <h2>
     <a id="Docker_23">
     </a>
     Docker部署（推荐）
    </h2>
    <p>
     开始计时，一分钟搭建好。
    </p>
    <p>
     首先，复制这行docker运行命令
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token parameter variable">-p</span> <span class="token number">3210</span>:3210 <span class="token punctuation">\</span>
  <span class="token parameter variable">-e</span> <span class="token assign-left variable">OPENAI_API_KEY</span><span class="token operator">=</span>sk-xxxx <span class="token punctuation">\</span>
  <span class="token parameter variable">-e</span> <span class="token assign-left variable">ACCESS_CODE</span><span class="token operator">=</span>lobe66 <span class="token punctuation">\</span>
  <span class="token parameter variable">--name</span> lobe-chat <span class="token punctuation">\</span>
  lobehub/lobe-chat
</code></pre>
    <p>
     简单说一下参数：
    </p>
    <pre><code class="prism language-bash">run：运行docker

-d：后台运行

-p：暴露端口号为3210（如果端口占用，可以修改第一个3210。）

 -e：配置环境变量参数：OPENAI_API_KEY为openai的apikey，要改成你自己的。没有的话可以私信我。

--name：给这个容器起个名字（可自定义）

lobehub/lobe-chat：是镜像名称，理解为带运行环境和代码的软件。
</code></pre>
    <p>
     其次，使用远程连接工具连接你的服务器。（以finalshell为例）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/abb95892fedc15b8ef7e66dbb9cd3525.png">
      <br/>
      粘贴上面复制的docker命令。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9ffea02ad61b77dec444e093aa7537f0.png">
       <br/>
       因为我已经部署过了，所以我的端口改成了3220。
      </img>
     </img>
    </p>
    <p>
     注意：
     <br/>
     环境变量OPENAI_API_KEY，要改成你自己的key。
    </p>
    <p>
     我这里用的是中转，所以还要加一个环境变量：OPENAI_PROXY_URL
    </p>
    <p>
     如果你是官方的key，就不需要加这个变量。
    </p>
    <p>
     修改完后直接回车。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f5084439035c4db902d409357d7ce1c3.png">
      <br/>
      出现一串id，就说明部署成功了。
     </img>
    </p>
    <p>
     我们用ip:端口的方式来查看项目是否运行成功。

    <h2>
     <a id="_81">
     </a>
     如何通过域名访问（扩展）
    </h2>
    <p>
     没有域名的需要先去购买一个域名，阿里或者腾讯都可以购买。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3dd414daf894a88b33fa9eb7b678784.png"/>
     <br/>
     点击解析设置，添加记录。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6389623f8d553dd325ea6b6e95233997.png"/>
     <br/>
     只需要填写我箭头标注的地方：主机记录、记录值。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/88ce3621709ae2dd8cd498ca4d2abc26.png"/>
     <br/>
     主机记录：这里建议用二级域名
    </p>
    <p>
     什么是二级域名？
    </p>
    <p>
     以qq.com域名举例，是一级域名，weixin.qq.com这是二级域名。
    </p>
    <p>
     可以随便写一个自己喜欢的，比如chat。
    </p>
    <p>
     记录值：你服务器ip
    </p>
    <p>
     新增记录是立马生效的，接下来要配置nginx。
    </p>
    <p>
     找到你nginx.conf文件
    </p>
    <p>
     我的这个文件路径在：/www/server/nginx/conf/
    </p>
    <p>
     查找命令：find / -name nginx.conf
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a6e5e821ee81f8881205a08c133aaf4a.png"/>
     <br/>
     找到以后进行修改，我这里用的是vi命令。
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /www/server/nginx/conf  <span class="token operator">&amp;&amp;</span> <span class="token function">vi</span> nginx.conf
</code></pre>
    <p>
     nginx配置参考：
    </p>
    <pre><code class="prism language-bash">
server<span class="token punctuation">{<!-- --></span>
   server_name 你解析的域名<span class="token punctuation">;</span>  <span class="token comment"># 请根据实际情况修改你的域名</span>
   
   location / <span class="token punctuation">{<!-- --></span>
          client_max_body_size  64m<span class="token punctuation">;</span> <span class="token comment"># 上传文件大小限制</span>
          proxy_http_version <span class="token number">1.1</span><span class="token punctuation">;</span> <span class="token comment"># http版本</span>
          proxy_pass http://localhost:3210<span class="token punctuation">;</span>  <span class="token comment"># 请根据实际情况修改你的端口</span>
          proxy_set_header Host <span class="token variable">$host</span><span class="token punctuation">;</span>
          proxy_set_header X-Forwarded-For <span class="token variable">$remote_addr</span><span class="token punctuation">;</span>
          proxy_cache_bypass <span class="token variable">$http_upgrade</span><span class="token punctuation">;</span>
          proxy_set_header Accept-Encoding <span class="token function">gzip</span><span class="token punctuation">;</span>
          proxy_read_timeout 300s<span class="token punctuation">;</span>  <span class="token comment"># GPT-4 需要较长的超时时间，请自行调整</span>
   <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     配置好以后就可以通过域名直接访问了。
