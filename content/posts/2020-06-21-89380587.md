# datatables提交json参数到后端

datatables post请求以json的格式提交请求到后端服务器

```
	ajax: {
            url: '请求地址',
            method: 'POST',
            data: function (data) {
                data.approveStatus = $("#approveStatus").val();
                data.readType = $("#readType").val();
                data.town = $("#town").find("option:selected").data("value")
                return JSON.stringify(data)
            }
        }

```
