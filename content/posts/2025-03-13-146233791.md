    <h4 id="创建-docker-网络">
     创建 Docker 网络
    </h4>
    <p>
     假设你想要创建一个名为
     <code>
      runtime_default
     </code>
     的 Docker 网络，并设置标签
     <code>
      com.docker.compose.network=default
     </code>
     ，你可以使用以下 PowerShell 脚本：
    </p>
    <pre><code># 定义网络名称和标签
$networkName = "runtime_default"
$labelKey = "com.docker.compose.network"
$labelValue = "default"

# 检查网络是否已经存在
$existingNetwork = docker network ls --filter name=$networkName --format "{<!-- -->{.Name}}"

if ($existingNetwork -eq $networkName) {
    Write-Output "网络 $networkName 已经存在。"
} else {
    # 创建新的 Docker 网络并设置标签
    docker network create --label "$labelKey=$labelValue" $networkName

    if ($?) {
        Write-Output "网络 $networkName 创建成功。"
    } else {
        Write-Output "创建网络 $networkName 时发生错误。"
    }
}
</code></pre>
    <p>
    </p>
    <h4 id="解释">
     解释
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        定义网络名称和标签
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         $networkName
        </code>
        是你想要创建的网络名称。
       </li>
       <li>
        <code>
         $labelKey
        </code>
        和
        <code>
         $labelValue
        </code>
        是你想要设置的标签键和值。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        检查网络是否已经存在
       </strong>
       ：
      </p>
      <ul>
       <li>
        使用
        <code>
         docker network ls
        </code>
        命令列出所有网络，并通过
        <code>
         --filter
        </code>
        选项过滤出名称为
        <code>
         $networkName
        </code>
        的网络。
       </li>
       <li>
        如果网络已经存在，脚本会输出一条消息并停止执行。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        创建新的 Docker 网络并设置标签
       </strong>
       ：
      </p>
      <ul>
       <li>
        使用
        <code>
         docker network create
        </code>
        命令创建新的网络，并通过
        <code>
         --label
        </code>
        选项设置标签。
       </li>
       <li>
        检查命令是否成功执行，并输出相应的消息。
       </li>
      </ul>
     </li>
    </ol>
    <h4 id="运行脚本">
     运行脚本
    </h4>
    <p>
     将上述脚本保存为一个
     <code>
      .ps1
     </code>
     文件，例如
     <code>
      CreateDockerNetwork.ps1
     </code>
     ，然后在 PowerShell 中运行它：
    </p>
    <pre><code>.\CreateDockerNetwork.ps1
</code></pre>
    <p>
     确保你有足够的权限来运行 Docker 命令。如果需要，你可能需要以管理员身份运行 PowerShell。
    </p>
    <h4 id="验证网络">
     验证网络
    </h4>
    <p>
     创建网络后，你可以通过以下命令验证网络是否正确创建：
    </p>
    <pre><code>docker network inspect runtime_default
</code></pre>
    <p>
     这将显示网络的详细信息，包括标签。
    </p>
    <p>
     希望这对你有帮助！如果有任何问题，请随时提问。
    </p>
