    <h2>
     <span style="color:#fe2c24">
      <strong>
       指针
      </strong>
     </span>
    </h2>
    <h3>
     指针概念
    </h3>
    <p>
     指针保存地址，地址是字节的编号
    </p>
    <p>
     指针类型和保存的地址类型要一直
    </p>
    <p>
     使用时注意，把地址转换为&amp;变量的格式来看        int a[3];  a转为&amp;a[0]
    </p>
    <h3>
     指针的大小
    </h3>
    <p>
     64bit        固定8字节，        32bit        固定4字节
    </p>
    <h3>
     指针的定义
    </h3>
    <p>
     格式：数据类型 变量名
    </p>
    <p>
     指针的标识：
    </p>
    <p>
     int a;
    </p>
    <p>
     int a[3];  先写单个变量，变量名后加[]和元素个数
    </p>
    <p>
     int *a;  先写单个变量，在变量名后面加*，表示指向该空间类型的指针
    </p>
    <p>
     空间类型==地址类型==数据类型
    </p>
    <h3>
     多级指针
    </h3>
    <p>
     <img alt="" height="401" src="https://i-blog.csdnimg.cn/direct/09651a49b34f40f89f481107e69fd339.png" width="762"/>
    </p>
    <h3>
     指针的初始化
    </h3>
    <p>
     指针不初始化--野指针，应避免野指针的出现
    </p>
    <p>
     int *p=NULL;
    </p>
    <pre><code>int a=5;
int *p=&amp;a;//初始化直接指向变量地址
char *p= “hello”;
char a[10]= “hello”;</code></pre>
    <h3>
     指针的使用
    </h3>
    <p>
     <span style="color:null">
      * 乘法 双目 1*2
     </span>
    </p>
    <p>
     <span style="color:null">
      * 解指针/解地址 单目 *地址
     </span>
    </p>
    <p>
     <span style="color:null">
      &amp; 取地址 同空间变量获取空间的地址
     </span>
    </p>
    <p>
     <span style="color:null">
      * 解地址 通过空间地址找到空间变量
     </span>
    </p>
    <p>
     <span style="color:null">
      所以*&amp;可以抵消
     </span>
    </p>
    <pre><code>int a=5;
int *p=&amp;a;
int **q=&amp;p;
int ***k=&amp;q;
printf(“%d”, a);//5
printf(“%d”, *p);//5
printf(“%d”, **q);//5
printf(“%d”, ***k);//5</code></pre>
    <h3>
     类型转换
    </h3>
    <p>
     数据转换：自动转换，自动向高精度转换
    </p>
    <p>
     强制转换：float a=3.14; printf("%d", (int)a);
    </p>
    <p>
     地址转换/空间类型转换：强制转换
    </p>
    <pre><code>int a=65;
char *p=(char *)&amp;a;// &amp;a 代表 4 字节空间地址 (char *)强制转换为 1 字节空间地址
if(*p == 65)
{
printf(“小端\n”);
}</code></pre>
    <h3>
     大小端
    </h3>
    <p>
     低位地址存高位数据        大端存储
    </p>
    <p>
     低位地址存低位数据        小端存储
    </p>
    <p>
     <span style="color:null">
      数据在进行网络传输时，尤其是跨平台和设备，会出现存储方式不同而引起的数据错误，所以在发送数据之前需要进行大小端测试
     </span>
    </p>
    <h3>
     <span style="color:null">
      地址偏移
     </span>
    </h3>
    <p>
     <span style="color:null">
      地址加减，加减的是空间个数，空间的大小由空间类型决定
     </span>
    </p>
    <pre><code>int a[5][20];
int *p=&amp;a[0][0];
p+1 == &amp;a[0][0]+1 == &amp;a[0][1]
(int (*)[20])p+1 == &amp;a[0]+1 == &amp;a[1]
int (*q)[20]=&amp;a[0]; // sizeof(q)==8
变量 q 先与*结合，所以 q 是指针
再与[20]结合，所以指针指向数组，数组有 20 个元素
最后与 int 结合，所以数组元素是整型变量
q+1 == &amp;a[0]+1 == &amp;a[1]
(int *)q+1 == &amp;a[0][0]+1 == &amp;a[0][1]</code></pre>
    <h3>
     指针常量&amp;&amp;常量指针
    </h3>
    <pre><code>关键字：const
功能：修饰变量为常量
int *p; // 整型指针
float *p; // 浮点型指针
char *p; // 字符指针
常量指针：指向常量的指针
数字常量 字符常量 字符串常量
指针常量：指针自己是常量
const char *p;//常量指针 const 修饰*p，指针指向的空间是常量
char const *p;//常量指针
char * const p;//指针常量 const 修饰 p，指针自身是常量
const char * const p;//常量指针常量</code></pre>
    <h3>
     指针数组&amp;&amp;数组指针
    </h3>
    <pre><code>int a[5];// 整型数组
float a[5];// 浮点型数组
char a[5];// 字符数组
指针数组：由指针组成的数组
int *a[5];// 整型指针数组
char *a[5];// 字符指针数组
数组指针：指向整个数组的指针
int (*p)[5];// 整型数组指针
int a[2][3];
int (*p[2])[3]; // 整型数组指针数组
p[0]=&amp;a[0];
p[1]=&amp;a[1];</code></pre>
    <p>
     二维字符数组&amp;&amp;字符指针数组
    </p>
    <pre><code>char a[3][5]={“123”, “456”, “789”}; // 二维字符数组
    a 字节大小：15
char *a[3]={“123789”, “456789”, “7896666666”}; // 字符指针数组
    a 字节大小：24
    printf(“%s\n”, a[1]+1); // 56789</code></pre>
