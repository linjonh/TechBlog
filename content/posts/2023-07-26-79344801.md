    <p>
     2、
     <strong>
      画面
     </strong>
    </p>
    <p>
     <span style="font-family:'宋体';">
      我想每一个玩过《阴阳师》的玩家就算不夸赞游戏画风，也不会找到任何不满之处。其整体使用
     </span>
     “和风”，每一个人物、每一个场景都刻画得极其精美。而且在人设上的设计，采用了“阴阳师”的概念，还有声优用的也是日本知名的，音乐也做的极其到位。再配上其充实新颖的剧情，并且使用的是视频播放方法。一开始我还疑惑为什么游戏界面一直推荐佩戴耳机进行游戏，直至我入坑，发现这一切都是一场视听盛宴。没有声音，这只是画面精美剧情好看的游戏。而有了声音，这就是一个可以把玩家带到游戏里面，身临其境的游戏。对二次元，动漫宅，喜爱和风的人来说，这些都是最具吸引力的，而且也是我最夸赞该游戏的。在手游界，我觉得《阴阳师》画面独一档，其他手游一档。
    </p>
    <p>
    </p>
    <p align="justify">
     3.
     <strong>
      玩法
     </strong>
    </p>
    <p align="justify">
     PVP
     <span style="font-family:'宋体';">
      ：这需要一定的基础，需要游戏入坑达到一定时间，并经常去肝才有资格
     </span>
     <span style="font-family:Calibri;">
      PVP
     </span>
     <span style="font-family:'宋体';">
      ，要不然纯粹是找虐。形式分为
     </span>
     <span style="font-family:Calibri;">
      3V3
     </span>
     <span style="font-family:'宋体';">
      ，
     </span>
     <span style="font-family:Calibri;">
      5V5
     </span>
     <span style="font-family:'宋体';">
      ，这对于平民玩家和入坑没多久的我来说并不适合，因为我的式神都太菜了，还没能培养起来。
     </span>
    </p>
    <p align="justify">
     PVE
     <span style="font-family:'宋体';">
      ：前期玩家需要的就是不停的
     </span>
     <span style="font-family:Calibri;">
      PVE
     </span>
     <span style="font-family:'宋体';">
      ，跟着任务做，跟着剧情走，在我看来这才是这游戏的核心玩法。而且在做这些的时候玩家不会厌倦，只会越来越吸引。因为画风、声乐、剧情都做得太好了，而且在不停的培养角色时候还会攒一些勾玉或者抽奖券，可以拿去玄学画符，抽
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      ，抽到了晒朋友圈那才是这游戏的最大成就。在我《阴阳师》朋友圈里，大多走的
     </span>
     <span style="font-family:Calibri;">
      PVE
     </span>
     <span style="font-family:'宋体';">
      路线，大家关注的都是你培养的式神如何，强不强力，有多少个式神，有没有
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      游戏的战斗采用回合制，技能效果也是一流的，很炫。我觉得游戏挖掘的不只是战斗的玩法、复杂性，还有美术、人设、
     </span>
     UI
     <span style="font-family:'宋体';">
      、音乐、配音这些。这在手游界是一种创新，它跟别的游戏不一样。首先它就吸引了玩家的眼球，然后就是式神的培养方式，充分利用了玩家的碎片时间。在玩法上给人感觉是一种唯美的体验，而不是游戏的重度沉迷。
     </span>
    </p>
    <p align="justify">
     4.
     <strong>
      付费
     </strong>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      游戏内流通的货币主要是勾玉，然后商店里面有皮肤、密卷、密魂等等。整个商城系统做得很完善，一切的画面和加成物品都走的是精美路线。光是点开商店，画面内的种类之多，之复杂，都可以感受到其背后的工作量有多巨大。所以也做得很贴心，玩家想买什么都能买到，更利于促使玩家付费。毕竟是网易，大公司制作的精品游戏，在付费系统方面是做得很完善的。而且根据一些新闻报道，其已经长期霸占
     </span>
     IOS
     <span style="font-family:'宋体';">
      畅销榜了，甚至有猜测其收入可跻身全球手游
     </span>
     <span style="font-family:Calibri;">
      TOP5
     </span>
     <span style="font-family:'宋体';">
      。这从另一方面说明了该游戏的付费用户之多，盈利之强。
     </span>
    </p>
    <p align="justify">
     5.
     <strong>
      <span style="font-family:'宋体';">
       活动
      </span>
     </strong>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      其活动也是很多，比如《阴阳师》动漫视听、收集积分换
     </span>
     SSR
     <span style="font-family:'宋体';">
      、平安庙祈福、收集拼图换皮肤、一寮同心、网易应用专属免流等等。活动内容覆盖方方面面，这都显示这个游戏是大制作，诚意游戏，不是那些换个壳就来圈钱的。以上活动我觉得最吸引人的就是收集积分换
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      了，毕竟
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      是这游戏的追求，就跟
     </span>
     <span style="font-family:Calibri;">
      DNF
     </span>
     <span style="font-family:'宋体';">
      有
     </span>
     <span style="font-family:Calibri;">
      SS
     </span>
     <span style="font-family:'宋体';">
      就能毕业是一种信仰一样。而且这个活动积分获取也是多种多样的，可以探索关卡、觉醒副本、御魂副本等等，很亲民，像我这种平民玩家不停的肝去做这些活动有一天也能获得自己喜欢的
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      。我觉得每一个活动深深挖掘下去，都是一种很有趣吸引人的玩法。对玩家的吸引力，一直都很强劲。
     </span>
    </p>
    <p align="justify">
     <strong>
      6.
      <span style="font-family:'宋体';">
       体验
      </span>
     </strong>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      首先是选择游戏区域，整个游戏已经在全球发布了，有简体中文、繁体中文、日文区、国际英文区，然后就是角色所属平台、服务器，国际区、双平台、
     </span>
     IOS
     <span style="font-family:'宋体';">
      、安卓都有。就是只要你想玩这个游戏，不管你所属哪个地区，系统用哪个，都可以玩。这从另一方面说明了这游戏做得有多强大，有多火，有巨量的玩家，就是一种玩这个游戏绝对不会错的感觉。然后进入游戏了是真真实实感受到那种和风，日本声优形象的配音，跟着剧情发展还会有小视频观看，很形象具体。因为一直都听说玩这游戏会有
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      ，所以我玩的时候，直接勾玉什么的都用来抽奖了，就搏那一点点的几率。结果肯定是没有抽到
     </span>
     <span style="font-family:Calibri;">
      SSR
     </span>
     <span style="font-family:'宋体';">
      了，但得了几个
     </span>
     <span style="font-family:Calibri;">
      SR
     </span>
     <span style="font-family:'宋体';">
      ，至少也比原来拿那些
     </span>
     <span style="font-family:Calibri;">
      R
     </span>
     <span style="font-family:'宋体';">
      级式神来得强。虽然此举在长远看来是毁号行为，老玩家都是不推荐的，但像我这种新手肯定是不在乎的，目前玩得开心就好。然后因为抽奖把能用的资源都用了，发现好穷，没有办法去培养式神了，所以战斗力肯定也是弱鸡的了。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      另外游戏里的探险模式一开始我还不知道小怪身边围绕的卷轴、金币是什么意思，后来玩多了才发现想升级快就要打那些身边挂着卷轴的，想要钱就要打挂着金币的，至于身边挂着企鹅，就是落御魂等物品几率增加。偶尔在探险模式里面会碰到妖怪，大
     </span>
     BOSS
     <span style="font-family:'宋体';">
      出现，那就是意外之喜了，把它打赢，战利品就比较多了，简直美滋滋。还有战队场景的动画效果也是很炫，可以自动战斗也是很贴心了。
     </span>
    </p>
    <p align="justify">
     <span style="font-family:'宋体';">
      这游戏最吸引我的是美术、人设、
     </span>
     UI
     <span style="font-family:'宋体';">
      、音乐、配音、剧情这些，因为真的做得很好，而一些战斗的细节我倒不在意了。我觉得可以改进的一点是在播放剧情小短片的时候，里面的人物嘴巴动起来就好了。现在就是有声音了人物也动了，可是嘴巴是闭着的，有点不协调，美工还要加把劲啊。
     </span>
    </p>
    <p align="justify">
    </p>
    <p align="justify">
     <strong>
      7
      <span style="font-family:'宋体';">
       、与《小冰冰传奇》的对比
      </span>
     </strong>
    </p>
    <p>
     <span style="font-family:'宋体';">
      知名度方面：《小冰冰传奇》（原版名为《刀塔传奇》），是上海莉莉丝科技股份有限公司研发、北京中清龙图网络技术有限公司运营的一款卡牌类手机游戏。其在
     </span>
     2014
     <span style="font-family:'宋体';">
      年也曾登顶
     </span>
     <span style="font-family:Calibri;">
      App Store
     </span>
     <span style="font-family:'宋体';">
      畅销榜，获封
     </span>
     <span style="font-family:Calibri;">
      2014
     </span>
     <span style="font-family:'宋体';">
      年度最大黑马及国民级手游。但我觉得可能是因为版权改名问题和
     </span>
     <span style="font-family:Calibri;">
      2014
     </span>
     <span style="font-family:'宋体';">
      年到现在已经过了几年的问题，所以现在已经没看到有人玩了，可能算是过气巨星？而《阴阳师》那肯定就是当好炸子鸡了，长期霸榜
     </span>
     <span style="font-family:Calibri;">
      App Store
     </span>
     <span style="font-family:'宋体';">
      。《阴阳师》在社交平台上，如朋友圈和微博上的刷屏似乎已成为一种常态，比如微信平台上其官方公众号上的文章几乎每一篇都达到了
     </span>
     <span style="font-family:Calibri;">
      100000+
     </span>
     <span style="font-family:'宋体';">
      的阅读量，官方微博已收揽了超过
     </span>
     <span style="font-family:Calibri;">
      40
     </span>
     <span style="font-family:'宋体';">
      万粉丝，
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      阴阳师手游
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      话题席卷了
     </span>
     <span style="font-family:Calibri;">
      13.1
     </span>
     <span style="font-family:'宋体';">
      亿阅读量，另一个话题
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      网易阴阳师手游
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      也收获了
     </span>
     <span style="font-family:Calibri;">
      3.1
     </span>
     <span style="font-family:'宋体';">
      亿的阅读量，甚至连衍生的话题
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      脱非入欧
     </span>
     <span style="font-family:Calibri;">
      #
     </span>
     <span style="font-family:'宋体';">
      也达到了
     </span>
     <span style="font-family:Calibri;">
      1
     </span>
     <span style="font-family:'宋体';">
      亿的阅读量。可能是因为我错过了《小冰冰传奇》的巅峰期，我现在感受到的是《阴阳师》对手游圈的支配。
     </span>
    </p>
    <p>
     <span style="font-family:'宋体';">
      画面方面：《小冰冰传奇》是比较
     </span>
     Q
     <span style="font-family:'宋体';">
      版的，刚开始玩的时候我甚至觉得这游戏的受众应该是小学生与初中生，让我想起了
     </span>
     <span style="font-family:Calibri;">
      2008
     </span>
     <span style="font-family:'宋体';">
      年的《
     </span>
     <span style="font-family:Calibri;">
      QQ
     </span>
     <span style="font-family:'宋体';">
      三国》。其画面上的建筑都比较小巧，感觉画面不是很大，但又完美的包容了整个游戏的各种功能图标、界面，而且整体上也比较仿真那种刀塔世界的感觉。而《阴阳师》和风唯美的画面配上水墨渲染，给人非常独特的游戏画面感受。整体人设很细腻唯美，场景用色有种淡雅的感觉。更别说还有知名声音的配音，有趣的短视频。虽然《小冰冰传奇》画面已经做得很好了，可是相比《阴阳师》，那还是差了许多。
     </span>
    </p>
    <p>
     剧情方面：《小冰冰传奇》只能是无功无过吧，讲述的是萨格拉斯率领燃烧军团入侵世界的故事，在手游界我觉得是很一般的，毕竟在《阴阳师》之前也没有哪个游戏在剧情下那么多功夫。而《阴阳师》，剧情是阴阳师封妖、找回记忆的故事。重点是其在展示剧情的时候配带有视频，而且这个类型的剧情也是较为吸引人的，对于二次元、死宅来说更为致命。
    </p>
    <p>
     <span style="font-family:'宋体';">
      战斗方面：刷副本时候《小冰冰传奇》给人感觉就很迟缓，而且可能因为是
     </span>
     Q
     <span style="font-family:'宋体';">
      版的原因，一点都不炫，也没有自动攻击，每次打完一回合就要鼠标点才能进入下一回合。而且技能方面也较为固定，只要等能量条集满点一下就可以放了，是比较简单的。在现在看来我觉得这种简单普通而且还不能自动的战斗是非常吃亏的。相比《阴阳师》，战斗画面就非常炫了，各种妖法，而且每个式神还可以控制释放不同的技能，式神之间的搭配也比《小冰冰传奇》多种多样。自动战斗还可以
     </span>
     <span style="font-family:Calibri;">
      X2
     </span>
     <span style="font-family:'宋体';">
      的速度，其在利用玩家的碎片时间方面是首屈一指的。
     </span>
    </p>
    <p>
     副本方面：《小冰冰传奇》的副本也是一贯的朴实画风，每打完一个副本就开启下一个，这些都是很普遍的了。而《阴阳师》副本就比较多种多样了，因为其式神也很多，育成方式也不同，还配有觉醒功能，所以就需要大量不同的副本去支撑这些。另外要说的是其副本小怪分为了三种类型，所以这可能是其没有开通副本扫荡这一功能的原因？想要经验什么的还是需要去手动，这有点不够好，而《小冰冰传奇》可以直接扫荡就好。
    </p>
    <p>
     商城、活动方面：《小冰冰传奇》也是中规中矩吧，可能是因为以我现在的眼光看来，导致对它的评分偏低了。而《阴阳师》这类由大公司发行的游戏，关于这些切身关乎到网易利益的东西，自然是做得非常好了，也就只有大公司才能做到玩法这么多样，虚拟物品这么具体了。
    </p>
