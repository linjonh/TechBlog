    <h2 style="background-color:transparent">
     一、CSS 简介
    </h2>
    <p>
     CSS 是
     <span style="color:#fe2c24">
      <strong>
       层叠样式表 ( Cascading Style Sheets )
      </strong>
     </span>
     的简称. 有时我们也会称之为
     <span style="color:#fe2c24">
      CSS 样式表
     </span>
     或
     <span style="color:#fe2c24">
      级联样式表｡
     </span>
    </p>
    <p>
     CSS 是也是一种标记语言
    </p>
    <p>
     CSS 主要用于
     <span style="color:#fe2c24">
      <strong>
       设置 HTML 页面中的文本内容(字体､大小､对齐方式等)､图片的外形(宽高､边框样式､ 边距等)以及版面的布局和外观显示样式｡
      </strong>
     </span>
    </p>
    <p>
     CSS 让我们的网页更加丰富多彩,布局更加灵活自如｡简单理解:
     <span style="color:#fe2c24">
      CSS 可以美化 HTML , 让 HTML 更漂亮,
     </span>
     让页面布局更简单｡
    </p>
    <p>
     <strong>
      总结:
     </strong>
    </p>
    <blockquote>
     <p>
      1. HTML 主要做结构,显示元素内容.
     </p>
     <p>
      2. CSS 美化 HTML ,布局网页.
     </p>
     <p>
      <strong>
       3. CSS 最大价值: 由 HTML 专注去做结构呈现,样式交给 CSS,即 结构 ( HTML ) 与样式( CSS ) 相分离｡
      </strong>
     </p>
     <p>
     </p>
    </blockquote>
    <h3>
     1.CSS 语法规范
    </h3>
    <p>
     使用 HTML 时,需要遵从一定的规范,CSS 也是如此｡要想熟练地使用 CSS 对网页进行修饰,首先需要了解 CSS 样式规则｡
    </p>
    <p>
     <span style="color:#fe2c24">
      CSS 规则由两个主要的部分构成:选择器以及一条或多条声明｡
     </span>
    </p>
    <p>
     <img alt="" height="142" src="https://i-blog.csdnimg.cn/direct/d4f1a7e894b64e1982092df6b24dd9a9.png" width="451"/>
    </p>
    <blockquote>
     <p>
      <span style="color:#fe2c24">
       选择器
      </span>
      是用于指定 CSS 样式的
      <span style="color:#fe2c24">
       HTML 标签
      </span>
      ,花括号内是对该对象设置的具体样式
     </p>
     <p>
      属性和属性值以“键值对”的形式出现
     </p>
     <p>
      属性是对指定的对象设置的样式属性,例如字体大小､文本颜色等
     </p>
     <p>
      属性和属性值之间用英文“:”分开
     </p>
     <p>
      多个“键值对”之间用英文“;”进行区分
     </p>
     <p>
     </p>
    </blockquote>
    <p>
     所有的样式,都包含在 &lt;style&gt; 标签内,表示是样式表｡
     <span style="color:#fe2c24">
      <strong>
       &lt;style&gt; 一般写到 &lt;/head&gt; 上方｡
      </strong>
     </span>
    </p>
    <pre><code>&lt;head&gt; 
&lt;style&gt; 
h4 { 
color: blue; 
font-size: 100px; 
}
&lt;/style&gt; 
&lt;/head&gt;</code></pre>
    <h2>
     二、CSS 基础选择器
    </h2>
    <h3>
     1.选择器分类
    </h3>
    <p>
     选择器分为
     <span style="color:#fe2c24">
      <strong>
       基础选择器
      </strong>
     </span>
     和
     <span style="color:#fe2c24">
      <strong>
       复合选择器
      </strong>
     </span>
     两个大类,我们这里先讲解一下基础选择器｡
    </p>
    <blockquote>
     <p>
      基础选择器是由
      <span style="color:#fe2c24">
       <strong>
        单个选择器
       </strong>
      </span>
      组成的
     </p>
     <p>
      基础选择器又包括:
      <span style="color:#fe2c24">
       标签选择器､类选择器､id 选择器
      </span>
      和
      <span style="color:#fe2c24">
       通配符选择器
      </span>
     </p>
    </blockquote>
    <h3>
     2.标签选择器
    </h3>
    <p>
     <span style="color:#fe2c24">
      <strong>
       标签选择器(元素选择器)
      </strong>
     </span>
     是指用
     <span style="color:#fe2c24">
      <strong>
       <span style="background-color:#ffd900">
        HTML 标签名称
       </span>
       作为选择器,
      </strong>
     </span>
     按标签名称分类,为页面中某一类标签指定 统一的 CSS 样式｡
    </p>
    <p>
     语法
    </p>
    <pre><code>标签名{ 
属性1: 属性值1; 
属性2: 属性值2; 
属性3: 属性值3; 
广</code></pre>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;基础选择器之标签选择器&lt;/title&gt;
    &lt;style&gt;
        /* 标签选择器 : 写上标签名 */
        p {
            color: green;
        }

        div {
            color: pink;
        }
    &lt;/style&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;p&gt;男生&lt;/p&gt;
    &lt;p&gt;男生&lt;/p&gt;
    &lt;p&gt;男生&lt;/p&gt;
    &lt;div&gt;女生&lt;/div&gt;
    &lt;div&gt;女生&lt;/div&gt;
    &lt;div&gt;女生&lt;/div&gt;
&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>
    <p>
     <span style="color:null">
      <span style="background-color:#ffd900">
       会把所有的某一类的标签全部设置css
      </span>
     </span>
    </p>
    <p>
     <img alt="" height="880" src="https://i-blog.csdnimg.cn/direct/f8e9c5227ce1427aa12f3ca60b75b59e.png" width="1648"/>
    </p>
    <blockquote>
     <p>
      <strong>
       作用
      </strong>
     </p>
     <p>
      标签选择器可以把某一类标签全部选择出来,比如所有的 &lt;div&gt; 标签和所有的 &lt;span&gt; 标签｡
     </p>
     <p>
      <strong>
       优点
      </strong>
     </p>
     <p>
      能快速为页面中同类型的标签统一设置样式｡
     </p>
     <p>
      <strong>
       缺点
      </strong>
     </p>
     <p>
      不能设计差异化样式,只能选择全部的当前标签｡
     </p>
    </blockquote>
    <h3>
     3.类选择器
    </h3>
    <h4>
     ⑴.类选择器
    </h4>
    <p>
     如果想要差异化选择不同的标签,单独选一个或者某几个标签,可以使用
     <span style="color:#fe2c24">
      <strong>
       类选择器.
      </strong>
     </span>
    </p>
    <p>
     语法
    </p>
    <pre><code>.类名 { 
属性1: 属性值1;
  ...
}</code></pre>
    <p>
     例如,将所有拥有 red 类的 HTML 元素均为红色｡
    </p>
    <pre><code>.red { 
color: red;
}</code></pre>
    <p>
     结构需要用class属性来调用 class 类的意思
    </p>
    <pre><code>&lt;div class=‘red’&gt; 变红色 &lt;/div&gt;</code></pre>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;基础选择器之类选择器&lt;/title&gt;
    &lt;style&gt;
        /* 类选择器口诀: 样式点定义  结构类(class)调用  一个或多个 开发最常用*/
      .red {
          color: red;
      }
      .star-sing {
        color: green;
      }
      .memeda {  
         color: pink;
      }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;ul&gt;
        &lt;li class="red"&gt;冰雨&lt;/li&gt;
        &lt;li class="red"&gt;来生缘&lt;/li&gt;
        &lt;li&gt;李香兰&lt;/li&gt;
        &lt;li class="memeda"&gt;生僻字&lt;/li&gt;
        &lt;li class="star-sing"&gt;江南style&lt;/li&gt;
    &lt;/ul&gt;
    &lt;div class="red"&gt;我也想变红色&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>
    <p>
     <img alt="" height="927" src="https://i-blog.csdnimg.cn/direct/964295a841db43d69a22f19fbc5ededa.png" width="1468"/>
    </p>
    <p>
     类选择器在 HTML 中以
     <span style="color:#fe2c24">
      class
     </span>
     属性表示,在 CSS 中,类选择器以一个点“.”号显示｡
    </p>
    <p>
     <strong>
      注意
     </strong>
    </p>
    <blockquote>
     <p>
      ①类选择器使用“.”(英文点号)进行标识,后面紧跟类名(自定义,我们自己命名的)｡
     </p>
     <p>
      ②可以理解为给这个标签起了一个名字,来表示｡
     </p>
     <p>
      ③长名称或词组可以使用中横线来为选择器命名｡
     </p>
     <p>
      ④不要使用纯数字､中文等命名,尽量使用英文字母来表示｡
     </p>
     <p>
      ⑤命名要有意义,尽量使别人一眼就知道这个类名的目的｡
     </p>
     <p>
      ⑥命名规范:见附件( Web 前端开发规范手册.doc)
     </p>
    </blockquote>
    <h4>
     ⑵.类选择器-多类名
    </h4>
    <p>
     我们可以给一个标签指定
     <span style="color:#fe2c24">
      多个类名,
     </span>
     从而达到更多的选择目的｡ 这些类名都可以选出这个标签. 简单理解就是一个标签有多个名字.
    </p>
    <p>
     <img alt="" height="423" src="https://i-blog.csdnimg.cn/direct/b84470b6a22b411f8df66c9539526a2b.png" width="909"/>
    </p>
    <h5>
     ①. 多类名使用方式
    </h5>
    <p>
     <strong>
      注意点：
     </strong>
    </p>
    <blockquote>
     <p>
      (1) 在标签class 属性中写 多个类名
     </p>
     <p>
      (2) 多个类名中间必须用空格分开
     </p>
     <p>
      (3) 这个标签就可以分别具有这些类名的样式
     </p>
    </blockquote>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;利用类选择器画三个盒子&lt;/title&gt;
    &lt;style&gt;
        .red {
            width: 100px;
            height: 100px;
            /* 背景颜色 */
            background-color: red;
        }
        .green {
            width: 100px;
            height: 100px;
            background-color: green;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div class="red"&gt;红色&lt;/div&gt;
    &lt;div class="green"&gt;绿色&lt;/div&gt;
    &lt;div class="red"&gt;红色&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>
    <p>
     <img alt="" height="883" src="https://i-blog.csdnimg.cn/direct/e4de2e0dd6c04f508f28e063112dcdd3.png" width="1716"/>
    </p>
    <h5>
     ②.多类名开发中使用场景
    </h5>
    <blockquote>
     <p>
      (1) 可以把一些标签元素相同的样式(共同的部分)放到一个类里面.
     </p>
     <p>
      (2) 这些标签都可以调用这个公共的类,然后再调用自己独有的类.
     </p>
     <p>
      (3) 从而节省CSS代码,
      <span style="color:#fe2c24">
       <strong>
        统一修改也非常方便
       </strong>
      </span>
      .
     </p>
    </blockquote>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;利用类选择器画三个盒子&lt;/title&gt;
    &lt;style&gt;
        .box {
            width: 150px;
            height: 100px;
            font-size: 30px;
        }
        .red {
        
            /* 背景颜色 */
            background-color: red;
        }
        .green {
           
            background-color: green;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div class="box red"&gt;红色&lt;/div&gt;
    &lt;div class="box green"&gt;绿色&lt;/div&gt;
    &lt;div class="box red"&gt;红色&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>

    <h3>
     4.id 选择器
    </h3>
    <p>
     id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式｡
    </p>
    <p>
     HTML 元素以
     <span style="color:#fe2c24">
      <strong>
       id 属性
      </strong>
     </span>
     来设置 id 选择器,CSS 中 id 选择器以
     <span style="color:#fe2c24">
      “#"
     </span>
     来定义｡
    </p>
    <p>
     语法
    </p>
    <pre><code>#id名 { 
属性1: 属性值1; 
 ...
}</code></pre>
    <p>
     例如,将 id 为 nav 元素中的内容设置为红色｡
    </p>
    <pre><code>#nav { 
color:red;
}</code></pre>
    <blockquote>
     <p>
      ① 类选择器(class)好比人的名字,一个人可以有多个名字,同时一个名字也可以被多个人使用｡
     </p>
     <p>
      ② id 选择器好比人的身份证号码,全中国是唯一的,不得重复｡
     </p>
     <p>
      ③ id 选择器和类选择器最大的不同在于使用次数上｡
     </p>
     <p>
      ④ 类选择器在修改样式中用的最多,id 选择器一般用于页面唯一性的元素上,经常和 JavaScript 搭配使用｡
     </p>
    </blockquote>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;基础选择器之id选择器&lt;/title&gt;
    &lt;style&gt;
        /* id选择器的口诀: 样式#定义, 结构id调用, 只能调用一次, 别人切勿使用 */
      #pink {
          color: pink;
      }
    
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div id="pink"&gt;迈克尔·杰克逊&lt;/div&gt;
    &lt;div&gt;pink老师&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>

    <h3>
     5.通配符选择器
    </h3>
    <p>
     在 CSS 中,通配符选择器使用
     <span style="color:#fe2c24">
      “*
     </span>
     ”定义,它表示选取页面中所有元素(标签)｡
    </p>
    <p>
     语法
    </p>
    <pre><code>* {
属性1: 属性值1;
...
}
</code></pre>
    <p>
     <span style="color:#fe2c24">
      <strong>
       通配符选择器不需要调用, 自动就给所有的元素使用样式
      </strong>
     </span>
    </p>
    <p>
     特殊情况才使用,后面讲解使用场景(以下是
     <span style="color:#fe2c24">
      <strong>
       清除所有的元素标签的内外边距
      </strong>
     </span>
     ,后期讲)
    </p>
    <pre><code>* {
margin: 0;
padding: 0;
}</code></pre>
    <p>
     <span style="color:#fe2c24">
      <strong>
       通配符选择器不需要调用
      </strong>
     </span>
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;基础选择器之通配符选择器&lt;/title&gt;
    &lt;style&gt;
     * {
         color: red;
     }
     /* * 这里把 html  body  div  span  li 等等的标签都改为了红色 */
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div&gt;我的&lt;/div&gt;
    &lt;span&gt;我的&lt;/span&gt;
    &lt;ul&gt;
        &lt;li&gt;还是我的&lt;/li&gt;
    &lt;/ul&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     运行结果：
    </p>

     5.字体复合属性
    </h3>
    <p>
     字体属性可以把以上文字样式综合来写, 这样可以更节约代码:
    </p>
    <pre><code class="hljs">body {

}

</code></pre>
    <p>
     使用 font 属性时,必须按上面语法格式中的顺序书写,
     <span style="color:#fe2c24">
      不能更换顺序
     </span>
     ,并且各个属性间以
     <span style="color:#fe2c24">
      空格
     </span>
     隔开
    </p>
    <p>
     不需要设置的属性可以省略(取默认值),但
     <span style="color:#fe2c24">
      必须保留 font-size 和 font-family 属性
     </span>
     ,否则 font 属性将不起作用
    </p>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;CSS字体属性之复合属性&lt;/title&gt;
    &lt;style&gt;
        /* 想要div文字变倾斜 加粗 字号设置为16像素 并且 是微软雅黑 */
        div {
            /* font-style: italic;
           font-weight: 700;
           font-size: 16px;
           font-family: 'Microsoft yahei'; */


            /* 复合属性: 简写的方式  节约代码 */
            /* font: font-style  font-weight  font-size/line-height  font-family; */
            /* font: italic 700 16px 'Microsoft yahei'; */
            font: 20px '黑体';
        }
    &lt;/style&gt;
&lt;/head&gt;

    <h2>
     五、CSS 引入方式
    </h2>
    <h3>
     1. CSS 的三种样式表
    </h3>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;内部样式表&lt;/title&gt;
    &lt;style&gt;
        div {
            color: pink;
        }
    &lt;/style&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;div&gt;所谓内部样式表,就是在html页面内部写样式,但是是单独写到style标签内部.&lt;/div&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     按照 CSS 样式书写的位置(或者引入的方式),CSS 样式表可以分为三大类:
    </p>
    <p>
     1. 行内样式表(行内式)
    </p>
    <p>
     2. 内部样式表(嵌入式)
    </p>
    <p>
     3. 外部样式表(链接式)
    </p>
    <h3>
     2.内部样式表
    </h3>
    <p>
     内部样式表(内嵌样式表)是写到html页面内部. 是将所有的 CSS 代码抽取出来,单独放到一个 &lt;style&gt; 标签中｡
    </p>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;内部样式表&lt;/title&gt;
    &lt;style&gt;
        div {
            color: pink;
        }
    &lt;/style&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;div&gt;所谓内部样式表,就是在html页面内部写样式,但是是单独写到style标签内部.&lt;/div&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     <img alt="" height="396" src="https://i-blog.csdnimg.cn/direct/989338487586452b87f8c9365de9abb1.png" width="1399"/>
    </p>
    <p>
     &lt;style&gt; 标签理论上可以放在 HTML 文档的任何地方,但一般会放在文档的&lt;head&gt;标签中
    </p>
    <p>
      通过此种方式,可以方便控制当前整个页面中的元素样式设置
    </p>
    <p>
      代码结构清晰,但是并没有实现结构与样式完全分离
    </p>
    <p>
      使用内部样式表设定 CSS,通常也被称为
     <span style="color:#fe2c24">
      嵌入式引入
     </span>
     ,这种方式是我们练习时常用的方式
    </p>
    <h3>
     3.行内样式表
    </h3>
    <p>
     行内样式表(内联样式表)是
     <span style="color:#fe2c24">
      在元素标签内部的 style 属性中设定 CSS 样式
     </span>
     ｡适合于修改简单样式.
    </p>
    <p>
     style 其实就是标签的属性
    </p>
    <p>
      在双引号中间，写法要符合 CSS 规范
    </p>
    <p>
      可以控制当前的标签设置样式
    </p>
    <p>
      由于书写繁琐，并且没有体现出结构与样式相分离的思想，所以不推荐大量使用，只有对当前元素添加简 单样式的时候，可以考虑使用
    </p>
    <p>
      使用行内样式表设定 CSS，通常也被称为
     <span style="color:#fe2c24">
      行内式引入
     </span>
    </p>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;行内样式表&lt;/title&gt;
&lt;/head&gt;

&lt;body&gt;

    &lt;p&gt; 晚风吹过温暖我心底我又想起你&lt;/p&gt;

    &lt;p&gt; 多甜蜜多甜蜜怎能忘记&lt;/p&gt;

    &lt;p&gt;不能忘记你把你写在日记里&lt;/p&gt;

    &lt;p&gt;不能忘记你心里想的还是你&lt;/p&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p>
     <img alt="" height="685" src="https://i-blog.csdnimg.cn/direct/bd030cd8e08649aba43465a8b24d2e21.png" width="699"/>
    </p>
    <h3>
     4.外部样式表
    </h3>
    <p>
     实际开发都是外部样式表. 适合于样式比较多的情况. 核心是:样式单独写到CSS 文件中,之后把CSS文件引入 到 HTML 页面中使用.
    </p>
    <p>
     引入外部样式表分为两步:
    </p>
    <blockquote>
     <p>
      1. 新建一个后缀名为 .css 的样式文件,把所有 CSS 代码都放入此文件中｡
     </p>
     <p>
      2. 在 HTML 页面中,使用&lt;link&gt; 标签引入这个文件｡
     </p>
    </blockquote>
    <pre><code class="hljs">&lt;link rel="stylesheet" href="css文件路径"&gt;</code></pre>
    <p>
     <img alt="" height="183" src="https://i-blog.csdnimg.cn/direct/b5a586438e244c41923633fb7ef0c25e.png" width="972"/>
    </p>
    <p>
     使用外部样式表设定 CSS,通常也被称为
     <span style="color:#fe2c24">
      外链式或链接式引入,
     </span>
     这种方式是开发中常用的方式
    </p>
    <blockquote>
     <p>
      在&lt;link rel="stylesheet" href="css文件路径"&gt;中，rel属性用于定义当前文档与被链接文档之间的关系。
      <span style="color:#fe2c24">
       <strong>
        当rel的值为"stylesheet"时，表示被链接的文档是一个样式表文件
       </strong>
      </span>
      ，它告知浏览器被链接的文件是用于为当前 HTML 页面提供样式信息的，浏览器会根据这个信息来正确加载和应用该样式表，将其中的样式规则应用到对应的 HTML 元素上，从而对页面进行样式渲染。
     </p>
    </blockquote>
    <p>
     html
    </p>
    <pre><code class="hljs">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
    &lt;meta http-equiv="X-UA-Compatible" content="ie=edge"&gt;
    &lt;title&gt;外部样式表&lt;/title&gt;
    &lt;link rel="stylesheet" href="style.css"&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;div&gt;来呀~快活呀,反正有大把时间...&lt;/div&gt;
&lt;/body&gt;
