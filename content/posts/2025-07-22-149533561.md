# Oracle 12c 创建数据库初级教程

##### 1. 连接到Oracle

`sqlplus /@ as sysdba` Oracle数据库名称默认为ORCL

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/6d87d5f5b541494194bff5d3d07a9742.png)

##### 2. 创建表空间（数据库）

`create user YOUR_USERNAME identified by "YOUR_PASSWORD";`  
YOUR_USERNAME为数据库名称和登录用户名，YOUR_PASSWORD为数据库登录密码

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/962d7900cda446198ca425dd148d5f21.png)

##### 如果提示`ORA-65096: 公用用户名或角色名无效`。

###### 2.1 执行`show con_name`;

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/e4a161ff9eff429297fbcdac5215e71b.png)

###### 2.2 执行`select name from v$pdbs;`

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/1f85a69ade554f5ba00303ee2fd4c4d7.png)

###### 2.3 执行`alter pluggable database ORCLPDB open;`

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/e415dd4794b34ac48c880dd5a35fd4a7.png)

###### 2.4 执行`alter session set container=ORCLPDB;`

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/5f23123a032a4b51aeed7fedbb518229.png)

###### 2.5 重新执行步骤2

##### 3. 修改权限

connect：连接权限  
resource：增删改查权限(DML)  
dba：建表、修改表、删除表权限(DDL)  
unlimited tablespace：无限表空间权限

* 授予权限  
  `grant connect,resource,dba to YOUR_USERNAME;`  
  ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/c55e60c2fbb142e9ae7564d25f6ab2fc.png)
* 创建表之后，撤销DBA权限  
  `revoke dba from YOUR_USERNAME;`  
  ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/094485d549ea42ed85c7beca3bc42557.png)
* 授予UNLIMITED TABLESPACE权限，不然无法向表中添加首条数据  
  `grant unlimited tablespace to test;`  
  ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/b65b8ab7b26247ea9aea7eacd826c4b0.png)

##### 4. 可以使用数据库了
