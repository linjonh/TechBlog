    <h4>
     <a id="_stdthread__0">
     </a>
     <strong>
      如何让一个类作为可调用对象，被
      <code>
       std::thread
      </code>
      调用
     </strong>
    </h4>
    <p>
     在 C++ 中，可以让一个类对象作为
     <strong>
      可调用对象（Callable Object）
     </strong>
     ，然后用
     <code>
      std::thread
     </code>
     进行调用。要实现这一点，主要有三种方法：
    </p>
    <ol>
     <li>
      <strong>
       重载
       <code>
        operator()
       </code>
       （仿函数）
      </strong>
     </li>
     <li>
      <strong>
       使用成员函数
      </strong>
     </li>
     <li>
      <strong>
       使用
       <code>
        std::bind
       </code>
       或
       <code>
        std::function
       </code>
      </strong>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="1__operator_9">
     </a>
     <strong>
      1. 方法一：重载
      <code>
       operator()
      </code>
      （仿函数）
     </strong>
    </h3>
    <p>
     <strong>
      最常见的方式
     </strong>
     是定义一个
     <strong>
      仿函数（functor）
     </strong>
     ，即重载
     <code>
      operator()
     </code>
     使类的实例可直接调用。
    </p>
    <h4>
     <a id="_12">
     </a>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;thread&gt;</span></span>

<span class="token keyword">class</span> <span class="token class-name">CallableObject</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token keyword">void</span> <span class="token keyword">operator</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>  <span class="token comment">// 让对象像函数一样调用</span>
        std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Thread running with CallableObject\n"</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    CallableObject obj<span class="token punctuation">;</span>
    std<span class="token double-colon punctuation">::</span>thread <span class="token function">t</span><span class="token punctuation">(</span>obj<span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 直接将对象传给 std::thread</span>
    t<span class="token punctuation">.</span><span class="token function">join</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_31">
     </a>
     <strong>
      解析
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       CallableObject
      </code>
      <strong>
       重载了
       <code>
        operator()
       </code>
      </strong>
      ，因此它的实例
      <code>
       obj
      </code>
      可以像函数一样被调用。
     </li>
     <li>
      <code>
       std::thread
      </code>
      可以直接接受
      <code>
       obj
      </code>
      作为参数，调用
      <code>
       operator()
      </code>
      。
     </li>
     <li>
      线程执行
      <code>
       obj.operator()()
      </code>
      ，输出：
      <pre><code>Thread running with CallableObject
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="2__41">
     </a>
     <strong>
      2. 方法二：使用类的成员函数
     </strong>
    </h3>
    <p>
     可以直接用类的
     <strong>
      成员函数
     </strong>
     作为
     <code>
      std::thread
     </code>
     的目标，但要注意
     <strong>
      非静态成员函数需要绑定对象
     </strong>
     。
    </p>
    <h4>
     <a id="_44">
     </a>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;thread&gt;</span></span>

<span class="token keyword">class</span> <span class="token class-name">Worker</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>

    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>

    <h4>
     <a id="_63">
     </a>
     <strong>
      解析
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       &amp;Worker::run
      </code>

      <code>
       Worker
      </code>
      的
      <strong>
       成员函数指针
      </strong>
      。
     </li>
     <li>
      <code>
       &amp;worker
      </code>
      传入对象指针，
      <strong>
       std::thread 需要对象来调用成员函数
      </strong>
      。
     </li>
     <li>
      线程执行
      <code>
       worker.run()
      </code>
      ，输出：
      <pre><code>Thread running with member function
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="3__stdbind__73">
     </a>
     <strong>
      3. 方法三：使用
      <code>
       std::bind
      </code>
      绑定成员函数
     </strong>
    </h3>
    <p>
     如果
     <code>
      std::thread
     </code>
     需要额外的参数，可以用
     <code>
      std::bind
     </code>
     或
     <code>
      std::function
     </code>
     绑定成员函数和对象。
    </p>
    <h4>
     <a id="_76">
     </a>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;thread&gt;</span></span>
<span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;functional&gt;</span></span>

<span class="token keyword">class</span> <span class="token class-name">Worker</span> <span class="token punctuation">{<!-- --></span>

    <span class="token keyword">void</span> <span class="token function">run</span><span class="token punctuation">(</span><span class="token keyword">int</span> x<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    Worker worker<span class="token punctuation">;</span>
    std<span class="token double-colon punctuation">::</span>thread <span class="token function">t</span><span class="token punctuation">(</span>std<span class="token double-colon punctuation">::</span><span class="token function">bind</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>Worker<span class="token double-colon punctuation">::</span>run<span class="token punctuation">,</span> <span class="token operator">&amp;</span>worker<span class="token punctuation">,</span> <span class="token number">42</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 绑定成员函数和参数</span>

    <h4>
     <a id="_96">
     </a>
     <strong>
      解析
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       std::bind(&amp;Worker::run, &amp;worker, 42)
      </code>
      绑定
      <code>
       worker.run(42)
      </code>
      。
     </li>
     <li>
      <code>
       std::thread
      </code>
      线程启动后会执行
      <code>
       worker.run(42)
      </code>
      。
     </li>
     <li>
      输出：
      <pre><code>Thread running with parameter: 42
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="4__stdfunction__Lambda_106">
     </a>
     <strong>
      4. 方法四：使用
      <code>
       std::function
      </code>
      结合 Lambda
     </strong>
    </h3>
    <p>
     另一种方式是用
     <code>
      std::function
     </code>
     存储可调用对象（比如 Lambda 或仿函数），然后传给
     <code>
      std::thread
     </code>
     。
    </p>
    <h4>
     <a id="_109">
     </a>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code class="prism language-cpp"><span class="token macro property"><span class="token directive-hash">#</span><span class="token directive keyword">include</span> <span class="token string">&lt;iostream&gt;</span></span>

    Worker worker<span class="token punctuation">;</span>
    std<span class="token double-colon punctuation">::</span>function<span class="token operator">&lt;</span><span class="token keyword">void</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token operator">&gt;</span> func <span class="token operator">=</span> std<span class="token double-colon punctuation">::</span><span class="token function">bind</span><span class="token punctuation">(</span><span class="token operator">&amp;</span>Worker<span class="token double-colon punctuation">::</span>run<span class="token punctuation">,</span> <span class="token operator">&amp;</span>worker<span class="token punctuation">,</span> <span class="token number">42</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

     </a>
     <strong>
      解析
     </strong>
    </h4>

      <code>

      </code>

      <code>
       worker.run(42)
      </code>
      。
     </li>
     <li>
      <code>
       std::thread
      </code>
      线程执行
      <code>
       func()
      </code>
      。
     </li>
     <li>
      输出：
      <pre><code>Thread running with parameter: 42
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="5__140">
     </a>
     <strong>
      5. 选择哪种方式？
     </strong>
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        <strong>
         方法
        </strong>
       </th>
       <th>
        <strong>
         特点
        </strong>
       </th>
       <th>
        <strong>
         适用场景
        </strong>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         仿函数（operator()）
        </strong>
       </td>
       <td>
        直接使用对象，无需额外绑定
       </td>
       <td>
        线程对象无需额外参数
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         成员函数
        </strong>
       </td>
       <td>
        需要对象指针才能调用
       </td>
       <td>
        适用于普通类成员函数
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          std::bind
         </code>
         绑定成员函数
        </strong>
       </td>
       <td>
        允许传递额外参数
       </td>
       <td>
        需要动态传递参数时
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         <code>
          std::function
         </code>
        </strong>
       </td>
       <td>
        结合
        <code>
         std::bind
        </code>
        ，适用于通用接口
       </td>
       <td>
        适用于异步任务调度
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     对于
     <strong>
      简单任务
     </strong>
     ，建议使用
     <strong>
      重载
      <code>
       operator()
      </code>
     </strong>
     。对于
     <strong>
      需要参数的任务
     </strong>
     ，建议使用
     <strong>
      <code>
       std::bind
      </code>
      或
      <code>
       std::function
      </code>
     </strong>
     。
    </p>

    <p>
     <strong>
      输出
     </strong>
    </p>
    <pre><code>Thread 0 is running
Thread 1 is running
Thread 2 is running
Thread 3 is running
Thread 4 is running
</code></pre>
    <hr/>
    <h3>
     <a id="_192">
     </a>
     <strong>
      总结
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       让类成为可调用对象
      </strong>
      的常见方式：
      <ol>
       <li>
        <strong>
         仿函数（重载
         <code>
          operator()
         </code>
         ）
        </strong>
       </li>
       <li>
        <strong>
         成员函数（使用
         <code>
          &amp;Class::method, &amp;object
         </code>
         绑定）
        </strong>
       </li>
       <li>
        <strong>
         使用
         <code>
          std::bind()
         </code>
         绑定成员函数
        </strong>
       </li>
       <li>
        <strong>
         使用
         <code>
          std::function
         </code>
         存储可调用对象
        </strong>
       </li>
      </ol>
     </li>
     <li>
      <strong>
       推荐用法
      </strong>
      <ul>
       <li>
        <strong>
         简单的可调用对象
        </strong>
        ➝
        <strong>
         仿函数
        </strong>
       </li>
       <li>
        <strong>
         普通成员函数
        </strong>
        ➝
        <strong>
         <code>
          std::thread t(&amp;Class::method, &amp;object)
         </code>
        </strong>
       </li>
       <li>
        <strong>
         需要传参的成员函数
        </strong>
        ➝
        <strong>
         <code>
          std::bind
         </code>
         或
         <code>
          std::function
         </code>
        </strong>
       </li>
       <li>
        <strong>
         多个线程调用同一个对象
        </strong>
        ➝
        <strong>
         <code>
          std::vector&lt;std::thread&gt;
         </code>
        </strong>
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="__stdvectorstdthread_206">
     </a>
     <strong>
      多个线程调用同一个对象 ➝
      <code>
       std::vector&lt;std::thread&gt;
      </code>
     </strong>
    </h4>
    <p>
     在 C++ 多线程编程中，我们经常希望
     <strong>
      多个线程同时调用同一个对象的成员函数
     </strong>
     。这通常用于：
    </p>
    <ul>
     <li>
      <strong>
       并行计算
      </strong>
      （多个线程操作同一个数据对象）
     </li>
     <li>
      <strong>
       任务调度
      </strong>
      （多个线程调用同一个处理对象）
     </li>
     <li>
      <strong>
       服务器/多客户端模型
      </strong>
      （多个线程访问同一个服务器对象）
     </li>
    </ul>
    <p>
     使用
     <code>
      std::vector&lt;std::thread&gt;
     </code>
     可以创建多个线程，并让它们调用
     <strong>
      同一个对象的成员函数
     </strong>
     。
    </p>
    <hr/>
    <h3>

        std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Thread "</span> <span class="token operator">&lt;&lt;</span> id <span class="token operator">&lt;&lt;</span> <span class="token string">" is working\n"</span><span class="token punctuation">;</span>

</code></pre>

     <a id="_291">

    <h3>
     <a id="5__387">
     </a>
     <strong>
      5. 结论
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       多个线程可以调用同一个对象的成员函数
      </strong>
      ，可以使用
      <code>
       std::vector&lt;std::thread&gt;
      </code>
      统一管理线程。
     </li>
     <li>
      <strong>
       如果成员函数只读，不修改成员变量，线程是安全的。
      </strong>
     </li>
     <li>
      <strong>
       如果成员函数修改成员变量，必须使用
       <code>
        std::mutex
       </code>
       保护数据，防止数据竞争（Race Condition）。
      </strong>
     </li>
     <li>
      <strong>
       <code>
        std::vector&lt;std::thread&gt;
       </code>
       让多个线程的管理更加方便，避免单独管理多个
       <code>
        std::thread
       </code>
       变量。
      </strong>
     </li>
    </ul>
    <p>
     这种模式在
     <strong>
      多线程服务器、任务分发、并行计算
     </strong>
     等场景非常常见。
    </p>
