    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        嵌入式系统无疑是当前最热门最有发展前途的
       </span>
       <span style="color:#333333">
        IT
       </span>
       <span style="color:#333333">
        应用领域之一。
       </span>
      </strong>
      <span style="color:#333333">
       嵌入式系统用在一些特定专用设备上，通常这些设备的硬件资源（如处理器、存储器等）非常有限，并且对成本很敏感，有时对实时响应要求很高等。特别是随着消费家电的智能化，嵌入式更显重要。像我们平常常见到的
       <strong>
        手机、
       </strong>
      </span>
      <strong>
       <span style="color:#333333">
        PDA
       </span>
       <span style="color:#333333">
        、电子字典、可视电话、
       </span>
       <span style="color:#333333">
        VCD/DVD/MP3 Player
       </span>
       <span style="color:#333333">
        、数字相机（
       </span>
       <span style="color:#333333">
        DC
       </span>
       <span style="color:#333333">
        ）、数字摄像机
       </span>
       <span style="color:#333333">
        (DV)
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        U-Disk
       </span>
       <span style="color:#333333">
        、机顶盒
       </span>
       <span style="color:#333333">
        (Set Top Box)
       </span>
       <span style="color:#333333">
        、高清电视
       </span>
       <span style="color:#333333">
        (HDTV)
       </span>
       <span style="color:#333333">
        、游戏机、智能玩具、交换机、路由器、数控设备或仪表、汽车电子、家电控制系统、医疗仪器、航天航空设备等等都是典型的嵌入式系统。
       </span>
      </strong>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
      </span>
      <span style="color:#333333">
       嵌入式系统是软硬结合的东西，搞嵌入式开发的人有两类。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
      </span>
      <span style="color:#333333">
       一类是学电子工程、通信工程等偏硬件专业出身的人，他们主要是搞硬件设计，有时要开发一些与硬件关系最密切的最底层软件，如
      </span>
      <span style="color:#333333">
       BootLoader
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       Board Support Package(
      </span>
      <span style="color:#333333">
       像
      </span>
      <span style="color:#333333">
       PC
      </span>
      <span style="color:#333333">
       的
      </span>
      <span style="color:#333333">
       BIOS
      </span>
      <span style="color:#333333">
       一样，往下驱动硬件，往上支持操作系统），最初级的硬件驱动程序等。他们的优势是对硬件原理非常清楚，不足是他们更擅长定义各种硬件接口，但对复杂软件系统往往力不从心（例如嵌入式操作系统原理和复杂应用软件等）。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        另一类是学软件、计算机专业出身的人，主要从事嵌入式操作系统和应用软件的开发。
       </span>
      </strong>
      <span style="color:#333333">
       如果我们学软件的人对硬件原理和接口有较好的掌握，我们完全也可写
      </span>
      <span style="color:#333333">
       BSP
      </span>
      <span style="color:#333333">
       和硬件驱动程序。嵌入式硬件设计完后，各种功能就全靠软件来实现了，嵌入式设备的增值很大程度上取决于嵌入式软件，这占了嵌入式系统的最主要工作（目前有很多公司将硬件设计包给了专门的硬件公司，稍复杂的硬件都交给台湾或国外公司设计，国内的硬件设计力量很弱，
       <strong>
        很多嵌入式公司自己只负责开发软件，因为公司都知道，嵌入式产品的差异很大程度在软件上，在软件方面是最有
       </strong>
      </span>
      <strong>
       <span style="color:#333333">
        “
       </span>
       <span style="color:#333333">
        花头
       </span>
       <span style="color:#333333">
        “
       </span>
       <span style="color:#333333">
        可做的），所以我们搞软件的人完全不用担心我们在嵌入式市场上的用武之地，越是智能设备越是复杂系统，软件越起关键作用，而且这是目前的趋势。
       </span>
      </strong>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
       </span>
       <span style="color:#333333">
        从事嵌入式软件开发的好处是：
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        1
       </span>
       <span style="color:#333333">
        ）目前国内外这方面的人都很稀缺。
       </span>
      </strong>
      <span style="color:#333333">
       一方面，是因为这一领域入门门槛较高，不仅要懂较底层软件（例如操作系统级、驱动程序级软件），对软件专业水平要求较高（嵌入式系统对软件设计的时间和空间效率要求较高），而且必须懂得硬件的工作原理，所以非专业
      </span>
      <span style="color:#333333">
       IT
      </span>
      <span style="color:#333333">
       人员很难切入这一领域；另一方面，是因为这一领域较新，目前发展太快，很多软硬件技术出现时间不长或正在出现（如
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       处理器、嵌入式操作系统、
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       技术、无线通信协议等），掌握这些新技术的人当然很找。
       <strong>
        嵌入式人才稀缺，身价自然就高，越有经验价格就越高。其实嵌入式人才稀少
       </strong>
      </span>
      <strong>
       <span style="color:#333333">
        ,
       </span>
       <span style="color:#333333">
        根本原因可能是大多数人无条件接触
       </span>
       <span style="color:#333333">
        ,
       </span>
       <span style="color:#333333">
        这需要相应的嵌入式开发板和软件
       </span>
       <span style="color:#333333">
        ,
       </span>
       <span style="color:#333333">
        另外需要有经验的人进行指导开发流程。
       </span>
      </strong>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        2
       </span>
       <span style="color:#333333">
        ）与企业计算等应用软件不同，嵌入式领域人才的工作强度通常低一些（但收入不低）。
       </span>
      </strong>
      <span style="color:#333333">
       搞企业应用软件的
      </span>
      <span style="color:#333333">
       IT
      </span>
      <span style="color:#333333">
       企业，这个用户的系统搞完了，又得去搞下一个用户的，而且每个用户的需求和完成时间都得按客户要求改变，往往疲于奔命，重复劳动。
       <strong>
        相比而言，搞嵌入式系统的公司，都有自己的产品计划，按自己的节奏行事。所开发的产品通常是通用的，不会因客户的不同而修改。一个产品型号开发完了，往往有较长一段空闲时间（或只是对软件进行一些小修补），有时间进行充电和休整。
       </strong>
       另外，从事嵌入式软件的每个人工作范围相对狭窄，所涉及的专业技术范围就是那些（
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       RTOS
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       802.11
      </span>
      <span style="color:#333333">
       等），时间长了这些东西会越搞越有经验，卖卖老本，几句指导也够让那些初入道者琢磨半年的。若搞应用软件，可能下一个客户要换成一个完全不同的软件开发平台，那就苦了。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       （
      </span>
      <span style="color:#333333">
       3
      </span>
      <span style="color:#333333">
       ）
       <strong>
        哪天若想创业，搞自已的产品，那么嵌入式是一个不错的主意，这可不像应用软件那样容易被盗版。
       </strong>
       土木学院有一个叫启明星的公司开发出一个好象叫
      </span>
      <span style="color:#333333">
       “
      </span>
      <span style="color:#333333">
       工程
      </span>
      <span style="color:#333333">
       e”
      </span>
      <span style="color:#333333">
       的掌上
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       （南校区门口有广告），施工技术人员用该
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       可当场进行土木概预算和其它土木计算，据说销路特好。我认识的某大学老师，他开发的饭馆用的点菜
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       （
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       平台，可无线连网和上网），据他说销路不错，饭馆点点
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       让客户点菜，多显派头档次。我记得
      </span>
      <span style="color:#333333">
       00
      </span>
      <span style="color:#333333">
       级
      </span>
      <span style="color:#333333">
       2+2
      </span>
      <span style="color:#333333">
       班当年有一组同学在学
      </span>
      <span style="color:#333333">
       Windows
      </span>
      <span style="color:#333333">
       程序设计课程时用
      </span>
      <span style="color:#333333">
       VC++
      </span>
      <span style="color:#333333">
       设计了一个功能很强的点菜系统做为课程项目，当时真想建议他们将这个软件做成
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       ，估计会有些销路（上海火车站南广场的
      </span>
      <span style="color:#333333">
       Macdonald
      </span>
      <span style="color:#333333">
       便使用很漂亮的
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       给用户点食品，像摸像样的）。这些
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       的硬件设计一般都是请其它公司给订做（这叫
      </span>
      <span style="color:#333333">
       “
      </span>
      <span style="color:#333333">
       贴牌
      </span>
      <span style="color:#333333">
       ”
      </span>
      <span style="color:#333333">
       ：
      </span>
      <span style="color:#333333">
       OEM
      </span>
      <span style="color:#333333">
       ），都是通用的硬件，我们只管设计软件就变成自己的产品了。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
      </span>
      <span style="color:#333333">
       从事嵌入式软件开发的缺点是：
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        1
       </span>
       <span style="color:#333333">
        ）入门起点较高，所用到的技术往往都有一定难度，若软硬件基础不好，特别是操作系统级软件功底不深，则可能不适于此行。
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        2
       </span>
       <span style="color:#333333">
        ）这方面的企业数量要远少于企业计算类企业。特别是从事嵌入式的小企业数量较多（小企业要搞自己的产品创业），知名大公司较少（搞嵌入式的大公司主要有
       </span>
       <span style="color:#333333">
        Intel
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Motorola
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        TI
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Philip
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Samsung
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Sony
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Futjtum
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        Bell-Alcatel
       </span>
       <span style="color:#333333">
        、意法半导体、
       </span>
       <span style="color:#333333">
        Microtek
       </span>
       <span style="color:#333333">
        、研华、华为、中兴通信、上广电等制造类企业）。这些企业的习惯思维方式是到电子、通信等偏硬专业找人。
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        3
       </span>
       <span style="color:#333333">
        ）有少数公司经常要硕士以上的人搞嵌入式，主要是基于嵌入式的难度。但大多数公司也并无此要求，只要有经验即可。
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        同学若学习嵌入式，显然应偏重于嵌入式软件，特别是嵌入式操作系统方面。对于搞嵌入式软件的人，
       </span>
      </strong>
      <span style="color:#333333">
       最重要的技术显然是（实际上很多公司的招聘广告上就是这样写的）：
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        1
       </span>
       <span style="color:#333333">
        ）
       </span>
       <span style="color:#333333">
        掌握主流嵌入式微处理器的结构与原理
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        2
       </span>
       <span style="color:#333333">
        ）
       </span>
       <span style="color:#333333">
        必须掌握一个嵌入式操作系统
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        3
       </span>
       <span style="color:#333333">
        ）
       </span>
       <span style="color:#333333">
        必须熟悉嵌入式软件开发流程并至少做过一个嵌入式软件项目。
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
      </span>
      <span style="color:#333333">
       嵌入式软件方面最重要的课程包括：
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        1
       </span>
       <span style="color:#333333">
        ）嵌入式微处理器结构与应用
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       这是一门嵌入式硬件基础课程，我院用这门课取代了传统的
      </span>
      <span style="color:#333333">
       “
      </span>
      <span style="color:#333333">
       微机原理与接口
      </span>
      <span style="color:#333333">
       ”
      </span>
      <span style="color:#333333">
       课程（目前国内已有少部分高校
      </span>
      <span style="color:#333333">
       IT
      </span>
      <span style="color:#333333">
       专业这样做了，因为讲
      </span>
      <span style="color:#333333">
       x86
      </span>
      <span style="color:#333333">
       微机原理与接口很难找到实际用处，只为教学而已）。我们说过，
       <strong>
        嵌入式是软硬件结合的技术，
       </strong>
       搞嵌入式软件的人应对
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       处理器工作原理和接口技术有充分了解，包括
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       的汇编指令系统。若不了解处理器原理，怎么能控制硬件工作，怎么能写出节省内存又运行高速的最优代码（嵌入式软件设计特别讲究时空效率），怎么能写出驱动程序（驱动程序都是与硬件打交道的）？很多公司招聘嵌入式软件人员时都要求熟悉
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       处理器，将来若同学到公司中从事嵌入式软件开发，公司都会给你一本该设备的硬件规格说明书
      </span>
      <span style="color:#333333">
       (xxx Specification)
      </span>
      <span style="color:#333333">
       ，您必须能看懂其中的内存分布和端口使用等最基本的说明（就像
      </span>
      <span style="color:#333333">
       x86
      </span>
      <span style="color:#333333">
       汇编一样），否则怎么设计软件。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       有些同学觉得嵌入式处理器课程较枯燥，这主要是硬件课程都较抽象的原因，等我们的嵌入式实验室
      </span>
      <span style="color:#333333">
       10
      </span>
      <span style="color:#333333">
       月份建好后，您做了一些实验后就会觉得看得见摸得着。还有同学对
      </span>
      <span style="color:#333333">
       ARM
      </span>
      <span style="color:#333333">
       汇编不感兴趣，以为嵌入式开发用
      </span>
      <span style="color:#333333">
       C
      </span>
      <span style="color:#333333">
       语言就足够了。其实不应仅是将汇编语言当成一个程序设计语言，
       <strong>
        学汇编主要是为了掌握处理器工作原理的。
       </strong>
       一个不熟悉汇编语言的人，怎么能在该处理器写出最优的
      </span>
      <span style="color:#333333">
       C
      </span>
      <span style="color:#333333">
       语言代码。在嵌入式开发的一些关键部分，有时还必须写汇编，如
      </span>
      <span style="color:#333333">
       Bootloader
      </span>
      <span style="color:#333333">
       等（可能还包括
      </span>
      <span style="color:#333333">
       BSP
      </span>
      <span style="color:#333333">
       ）。特别是在对速度有极高要求的场合（如
      </span>
      <span style="color:#333333">
       DSP
      </span>
      <span style="color:#333333">
       处理器的高速图像采集和图像解压缩），目前主要还要靠汇编写程序（我看到过很多公司是这样做的）。
       <strong>
        当您在一个嵌入式公司工作时，在查看描述原理的手册时，可能很多都是用汇编描述的（我就遇到过），这是因为很多硬件设计人员只会写或者喜欢用汇编描述，此时您就必须看懂汇编程序，否则软硬件人员可能就无法交流。很多嵌入式职位招聘时都要求熟悉汇编。
       </strong>
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        (2)
       </span>
       <span style="color:#333333">
        嵌入式操作系统类课程
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       除了
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       的实时性稍差外，大多数嵌入式操作系统的实时性都很强
      </span>
      <span style="color:#333333">
       ,
      </span>
      <span style="color:#333333">
       所以也可称为实时操作系统
      </span>
      <span style="color:#333333">
       Real Time Operating System.
      </span>
      <span style="color:#333333">
       从事嵌入式的人至少须掌握一个嵌入式操作系统
      </span>
      <span style="color:#333333">
       (
      </span>
      <span style="color:#333333">
       当然掌握两个更好
      </span>
      <span style="color:#333333">
       ),
      </span>
      <span style="color:#333333">
       这在嵌入式的所有技术中是最为关键的了。目前最重要的
      </span>
      <span style="color:#333333">
       RTOS
      </span>
      <span style="color:#333333">
       主要包括：
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        第一类、传统的经典
       </span>
       <span style="color:#333333">
        RTOS
       </span>
       <span style="color:#333333">
        ：
       </span>
      </strong>
      <span style="color:#333333">
       最主要的便是
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       操作系统，以及其
      </span>
      <span style="color:#333333">
       Tornado
      </span>
      <span style="color:#333333">
       开发平台。
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       因出现稍早，实时性很强（据说可在
      </span>
      <span style="color:#333333">
       1ms
      </span>
      <span style="color:#333333">
       内响应外部事件请求），并且内核可极微（据说最小可
      </span>
      <span style="color:#333333">
       8K
      </span>
      <span style="color:#333333">
       ），可靠性较高等，所以在北美，
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       占据了嵌入式系统的多半疆山。特别是在通信设备等实时性要求较高的系统中，几乎非
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       莫属。
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       的很多概念和技术都和
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       很类似，主要是
      </span>
      <span style="color:#333333">
       C
      </span>
      <span style="color:#333333">
       语言开发。像
      </span>
      <span style="color:#333333">
       Bell-alcatel
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       Lucent
      </span>
      <span style="color:#333333">
       、华为等通信企业在开发产品时，
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       用得很多。但
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       因价格很高，所以一些小公司或小产品中往往用不起。目前很多公司都在往嵌入式
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       转（华为、中兴以及
      </span>
      <span style="color:#333333">
       UT
      </span>
      <span style="color:#333333">
       目前正在这样转）。但无论如何，
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       在一段长时间内仍是不可动摇的。与
      </span>
      <span style="color:#333333">
       Vxworks
      </span>
      <span style="color:#333333">
       类似的稍有名的实时操作系统还有
      </span>
      <span style="color:#333333">
       pSOS
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       QNX
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       Nucleus
      </span>
      <span style="color:#333333">
       等
      </span>
      <span style="color:#333333">
       RTOS
      </span>
      <span style="color:#333333">
       。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        第二类、嵌入式
       </span>
       <span style="color:#333333">
        Linux
       </span>
       <span style="color:#333333">
        操作系统
       </span>
      </strong>
      <span style="color:#333333">
       ：
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       的前途除作为服务器操作系统外，
       <strong>
        最成功的便是在嵌入式领域的应用，原因当然是免费、开源、支持软件多、呼拥者众，这样嵌入式产品成本会低。
       </strong>
      </span>
      <strong>
       <span style="color:#333333">
        Linux
       </span>
       <span style="color:#333333">
        本身不是一个为嵌入式设计的操作系统，不是微内核的，并且实时性不强。目前应用在嵌入式领域的
       </span>
       <span style="color:#333333">
        Linux
       </span>
       <span style="color:#333333">
        系统主要有两类：一类是专为嵌入式设计的已被裁减过的
       </span>
       <span style="color:#333333">
        Linux
       </span>
       <span style="color:#333333">
        系统，
       </span>
      </strong>
      <span style="color:#333333">
       最常用的是
      </span>
      <span style="color:#333333">
       uClinux
      </span>
      <span style="color:#333333">
       （不带
      </span>
      <span style="color:#333333">
       MMU
      </span>
      <span style="color:#333333">
       功能），目前占较大应用份额，可在
      </span>
      <span style="color:#333333">
       ARM7
      </span>
      <span style="color:#333333">
       上跑；
       <strong>
        另一类是跑在
       </strong>
      </span>
      <strong>
       <span style="color:#333333">
        ARM 9
       </span>
       <span style="color:#333333">
        上的
       </span>
      </strong>
      <span style="color:#333333">
       ，一般是将
      </span>
      <span style="color:#333333">
       Linux 2.4.18
      </span>
      <span style="color:#333333">
       内核移植在其上，可使用更多的
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       功能（当然
      </span>
      <span style="color:#333333">
       uClinux
      </span>
      <span style="color:#333333">
       更可跑在
      </span>
      <span style="color:#333333">
       ARM 9
      </span>
      <span style="color:#333333">
       上）。很多人预测，嵌入式
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       预计将占嵌入式操作系统的
      </span>
      <span style="color:#333333">
       50%
      </span>
      <span style="color:#333333">
       以上份额，非常重要。缺点是熟悉
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       的人太少，开发难度稍大。另外，目前我们能发现很多教材和很多大学都以
      </span>
      <span style="color:#333333">
       ucOS/II
      </span>
      <span style="color:#333333">
       为教学用实时操作系统，这主要是由于
      </span>
      <span style="color:#333333">
       ucOS/II
      </span>
      <span style="color:#333333">
       较简单，且开源，非常适合入门者学习实时操作系统原理，但由于
      </span>
      <span style="color:#333333">
       ucOS/II
      </span>
      <span style="color:#333333">
       功能有限，实用用得较少，所以我院不将其作为教学重点，要学习就应学直接实用的，比如
      </span>
      <span style="color:#333333">
       uClinux
      </span>
      <span style="color:#333333">
       就很实用。况且熟悉了
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       开发，不仅在嵌入式领域有用，对开发
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       应用软件，对加深操作系统的认识也有帮助，可谓一举多得。据我所知，目前
      </span>
      <span style="color:#333333">
       Intel
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       Philip
      </span>
      <span style="color:#333333">
       都在大搞
      </span>
      <span style="color:#333333">
       ARM+LINUX
      </span>
      <span style="color:#333333">
       的嵌入式开发，
      </span>
      <span style="color:#333333">
       Fujitum
      </span>
      <span style="color:#333333">
       则是在自己的处理器上大搞
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       开发。目前在嵌入式
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       领域，以下几个方面的人特别难找，一是能将
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       移植到某个新型号的开发版上；二是能写
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       驱动程序的人；三是熟悉
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       内核裁减和优化的人。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        第三类、
       </span>
       <span style="color:#333333">
        Windows CE
       </span>
       <span style="color:#333333">
        嵌入式操作系统
       </span>
      </strong>
      <span style="color:#333333">
       ：
      </span>
      <span style="color:#333333">
       Microsoft
      </span>
      <span style="color:#333333">
       也看准了嵌入式的巨大市场，
      </span>
      <span style="color:#333333">
       MS
      </span>
      <span style="color:#333333">
       永远是最厉害的，
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       出来只有几年时间，但目前已占据了很大市场份额，特别是在
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       、手机、显示仪表等界面要求较高或者要求快速开发的场合，
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       目前已很流行（据说有一家卖工控机的公司板子卖得太好，以至来不及为客户裁减
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       ）。
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       目前主要为
      </span>
      <span style="color:#333333">
       4.2
      </span>
      <span style="color:#333333">
       版（
      </span>
      <span style="color:#333333">
       .NET
      </span>
      <span style="color:#333333">
       ），开发平台主要为
      </span>
      <span style="color:#333333">
       WinCE Platform Builder
      </span>
      <span style="color:#333333">
       ，有时也用
      </span>
      <span style="color:#333333">
       EVC
      </span>
      <span style="color:#333333">
       环境开发一些较上层的应用，对于急于完成，不想拿嵌入式
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       冒险的开发场合，
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       是最合适了（找嵌入式
      </span>
      <span style="color:#333333">
       Linux
      </span>
      <span style="color:#333333">
       的人可没那么好找的），毕竟公司不能像学生学习那样试试看，保证开发成功更重要。根据不同的侧重点
      </span>
      <span style="color:#333333">
       ，
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       还有两个特殊版本，一个是
      </span>
      <span style="color:#333333">
       MS PocketPC
      </span>
      <span style="color:#333333">
       操作系统专用于
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       上（掌上电脑），另一个是
      </span>
      <span style="color:#333333">
       MS SmartPhone
      </span>
      <span style="color:#333333">
       操作系统用于智能手机上（带
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       功能的手机），两者也都属于
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       平台。在
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       和手机市场上，除
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       外，著名的
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       嵌入式操作系统还有
      </span>
      <span style="color:#333333">
       Palm OS
      </span>
      <span style="color:#333333">
       （因出现很早，很有名）、
      </span>
      <span style="color:#333333">
       Symbian
      </span>
      <span style="color:#333333">
       等，但在
      </span>
      <span style="color:#333333">
       WinCE
      </span>
      <span style="color:#333333">
       的强劲冲击下，
      </span>
      <span style="color:#333333">
       Palm
      </span>
      <span style="color:#333333">
       和
      </span>
      <span style="color:#333333">

      </span>
      <span style="color:#333333">
       来日还能有多长？
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        总结关于嵌入式操作系统类课程，若您觉得自己功底较深且能钻研下去，则可去学嵌入式
       </span>
       <span style="color:#333333">
        Linux
       </span>

       </span>
       <span style="color:#333333">

     <span style="font-size:18px">
      <span style="color:#333333">
       (3)
      </span>
      <span style="color:#333333">
       嵌入式开发的其它相关软件课程
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px; color:#333333">
      搞嵌入式若能熟悉嵌入式应用的一些主要领域，这样的人更受企业欢迎。主要的相关领域包括：
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        A
       </span>
       <span style="color:#333333">
        、数字图像压缩技术
       </span>
      </strong>
      <span style="color:#333333">
       ：这是嵌入式最重要最热门的应用领域之一，主要是应掌握
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       编解码算法和技术，如
      </span>
      <span style="color:#333333">
       DVD
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       MP3
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       PDA
      </span>
      <span style="color:#333333">
       、高精电视、机顶盒等都涉及
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       高速解码问题。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        B
       </span>
       <span style="color:#333333">
        、通信协议及编程技术
       </span>
      </strong>
      <span style="color:#333333">
       ：这包括传统的
      </span>
      <span style="color:#333333">
       TCP/IP
      </span>
      <span style="color:#333333">
       协议和热门的无线通信协议。首先，大多数嵌入式设备都要连入局域网或
      </span>
      <span style="color:#333333">
       Internet
      </span>
      <span style="color:#333333">
       ，所以首先应掌握
      </span>
      <span style="color:#333333">
       TCP/IP
      </span>
      <span style="color:#333333">
       协议及其编程，这是需首要掌握的基本技术；其次，无线通信是目前的大趋势，所以掌握无线通信协议及编程也是是很重要的。无结通信协议包括无线局域网通信协议
      </span>
      <span style="color:#333333">
       802.11
      </span>
      <span style="color:#333333">
       系列，
      </span>
      <span style="color:#333333">
       Bluetooth
      </span>
      <span style="color:#333333">
       ，以及移动通信（如
      </span>
      <span style="color:#333333">
       GPRS
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       GSM
      </span>
      <span style="color:#333333">
       、
      </span>
      <span style="color:#333333">
       CDMA
      </span>
      <span style="color:#333333">
       等）。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        C
       </span>
       <span style="color:#333333">
        、网络与信息安全技术
       </span>
      </strong>
      <span style="color:#333333">
       ：如加密技术，数字证书
      </span>
      <span style="color:#333333">
       CA
      </span>
      <span style="color:#333333">
       等。我院有这方面的选修课。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <strong>
       <span style="color:#333333">
        D
       </span>
       <span style="color:#333333">
        、
       </span>
       <span style="color:#333333">
        DSP
       </span>
       <span style="color:#333333">
        技术
       </span>
      </strong>
      <span style="color:#333333">
       ：
      </span>
      <span style="color:#333333">
       DSP
      </span>
      <span style="color:#333333">
       是
      </span>
      <span style="color:#333333">
       Digital Signal
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       Process
      </span>
      <span style="color:#333333">
       数字信号处理的意思，
      </span>
      <span style="color:#333333">
       DSP
      </span>
      <span style="color:#333333">
       处理器通过硬件实现数字信号处理算法，如高速数据采集、压缩、解压缩、通信等。数字信号处理是电子、通信等硬件专业的课程，对于搞软件的人若能了解一下最好。目前
      </span>
      <span style="color:#333333">
       DSP
      </span>
      <span style="color:#333333">
       人才较缺。如果有信号与系统、数字信号处理等课程基础，对于学习
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       编解码原理会有很大帮助。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        （
       </span>
       <span style="color:#333333">
        4
       </span>
       <span style="color:#333333">
        ）嵌入式开发的相关硬件基础
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <strong>
      <span style="font-size:18px">
       <span style="color:#333333">
        对于软件工程专业的学生，从事嵌入式软件开发，像数字电路、计算机组成原理、嵌入式微处理器结构等硬件课程是较重要的。另外，汇编语言、
       </span>
       <span style="color:#333333">
        C/C++
       </span>
       <span style="color:#333333">
        、数据结构和算法、特别是操作系统等软件基础课也是十分重要的。主要目地是能看懂硬件工作原理，但重点应是在嵌入式软件，特别操作系统级软件。
       </span>
      </span>
     </strong>
    </p>
    <p align="left" style="background:white">
     <span style="font-size:18px">
      <span style="color:#333333">
       研究生里有些是学电子、通信类专业过来的，有较好的模拟电路和单片机基础，学嵌入式非常合适。嵌入式本身就是从单片机发展过来的，只是单片机不带
      </span>
      <span style="color:#333333">
       OS
      </span>
      <span style="color:#333333">
       ，而现在很多嵌入式应用越来越复杂，以至不得不引入嵌入式操作系统。另外，为追求更高速的信号处理速度，现在在一些速度要求较高的场合，有不少公司是将一些
      </span>
      <span style="color:#333333">
       DSP
      </span>
      <span style="color:#333333">
       算法，如
      </span>
      <span style="color:#333333">
       MPEG
      </span>
      <span style="color:#333333">
       压缩解压缩算法等用硬件来实现，这就涉及到
      </span>
      <span style="color:#333333">
       HDL
      </span>
      <span style="color:#333333">
       数字电路设计技术及其
      </span>
      <span style="color:#333333">
       FPGA/IP
      </span>
      <span style="color:#333333">
       核实现技术，这方面的人目前市场上也很缺。
      </span>
     </span>
    </p>
    <p align="left" style="background:white">
