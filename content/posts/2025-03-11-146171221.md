    <h2>
     <a id="Dockerjar_0">
     </a>
     Docker构建启动jar包
    </h2>
    <h3>
     <a id="1javajar_1">
     </a>
     1、首先是把java服务打包成jar包
    </h3>
    <pre><code class="prism language-java">mvn clean install '<span class="token operator">-</span><span class="token class-name">Dmaven</span><span class="token punctuation">.</span>skip<span class="token punctuation">.</span>test<span class="token operator">=</span><span class="token boolean">true</span>' <span class="token keyword">package</span> <span class="token operator">-</span><span class="token class-name">Pprod</span>
</code></pre>
    <p>
     这个命令的意思是，跳过测试，打包prod环境。
    </p>
    <h3>
     <a id="2Dockerfile_6">
     </a>
     2、编写Dockerfile文件
    </h3>
    <pre><code class="prism language-java"># 拉取jdk8作为基础镜像
<span class="token constant">FROM</span> registry<span class="token punctuation">.</span>supos<span class="token punctuation">.</span>ai<span class="token operator">/</span>library<span class="token operator">/</span>openjdk<span class="token operator">:</span><span class="token number">8</span><span class="token operator">-</span>jdk<span class="token operator">-</span>stretch
# 添加jar到镜像并命名
<span class="token constant">ADD</span> sup<span class="token operator">-</span>gateway<span class="token operator">-</span><span class="token number">0.0</span><span class="token number">.1</span><span class="token operator">-</span><span class="token constant">SNAPSHOT</span><span class="token punctuation">.</span>jar sup<span class="token operator">-</span>gateway<span class="token punctuation">.</span>jar
# 镜像启动后暴露的端口
<span class="token constant">EXPOSE</span> <span class="token number">18000</span>
# jar运行命令，参数使用逗号隔开
<span class="token constant">ENTRYPOINT</span> <span class="token punctuation">[</span><span class="token string">"java"</span><span class="token punctuation">,</span><span class="token string">"-jar"</span><span class="token punctuation">,</span><span class="token string">"sup-gateway.jar"</span><span class="token punctuation">]</span>

</code></pre>
    <p>
     FROM registry.supos.ai/library/openjdk:8-jdk-stretch 这个命令是要使用服务器上的jdk镜像，要改成自己服务器的jdk镜像。
     <br/>
     先查一下，自己的镜像是什么。命令如下
    </p>
    <pre><code class="prism language-language">docker images | grep jdk
</code></pre>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/9eafc357b5fe332d2e92a122893e4ca6.png"/>
    </p>
    <p>
     这是我的镜像。
     <br/>
     其他的命令就是关于jar包的一些，也要修改成自己的。
     <br/>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/714dcd0f2d68542243f13965d0ee863f.png"/>
    </p>
    <h3>
     <a id="3_28">
     </a>
     3、把这两个文件上传到服务器上
    </h3>
    <p>
     怎么上传就略了，但是要都上传到一个文件夹下，像我这样。
     <br/>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/d654eea2b51f624c535d993579555b96.png"/>
    </p>
    <h3>
     <a id="4_31">
     </a>
     4、构建镜像启动服务
    </h3>
    <p>
     cd 到上传的路径下
     <br/>
     <strong>
      然后使用docker命令构建镜像：
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> build <span class="token parameter variable">-t</span> sup-gateway <span class="token builtin class-name">.</span>
</code></pre>
    <p>
     <strong>
      命令中有个 . 不要落下
     </strong>
     <br/>
     sup-gateway表示镜像名称,最后的.表示Dockerfile在当前目录。
    </p>
    <p>
     <strong>
      打包完成后查看镜像
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> images
</code></pre>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/c8ebc79ff65b2fa18bf4da23dc3ea6f4.png"/>
    </p>
    <p>
     <strong>
      使用docker run命令创建并运行容器：
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token parameter variable">--name</span> sup-gateway <span class="token parameter variable">-p</span> <span class="token number">18000</span>:18000 <span class="token parameter variable">-v</span> /home/lims/images/face:/home/lims/images/face  sup-gateway
</code></pre>
    <p>
     命令解释如下：
     <br/>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/2b2572321de2d879a3efdcd33695411b.png"/>
    </p>
    <p>
     如果有容器被占用可以先删除。
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> <span class="token function">rm</span> <span class="token parameter variable">-f</span> sup-sys
</code></pre>
    <p>
     <strong>
      启动完成,查看启动日志：
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> logs <span class="token parameter variable">-f</span> <span class="token parameter variable">-t</span> sup-gateway
</code></pre>
    <p>
     <strong>
      jar日志启动成功，查看容器：
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token function">docker</span> <span class="token function">ps</span>
</code></pre>
    <p>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/1687de63c08329e9a6b37de45e251581.png"/>
    </p>
    <p>
     <strong>
      访问，验证效果：
     </strong>
     <br/>
     <img alt="image.png" src="https://i-blog.csdnimg.cn/img_convert/d330d998e4e111805370351d6893ce40.png"/>
    </p>
