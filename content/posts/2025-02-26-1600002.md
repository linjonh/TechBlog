# 移动及嵌入式Ubuntu

5

月


6

～


11

日，在西班牙的


Seville

市举行“


Ubuntu

开发者峰会”，规划订于今年


10

月份发布的


Ubuntu 7.10

新版本的具体目标和开发计划。届时，一项称为“移动及嵌入式


Ubuntu

”的开发计划提交大会讨论通过，“移动


Ubuntu

”的第一个正式版本包括在


Ubuntu 7.10

版本的系列里面。此举将把


Ubuntu

开发者的活动范围扩大（实为引导）到互联网移动设备的软件设计、开发和制造领域。

不久之前，


Intel

宣布了一项新的低功耗处理器和芯片集架构，用于整个互联网移动设备的产品系列。为实现


Ubuntu

的宏伟目标（使命），迎接互联网移动设备新浪潮的技术挑战，


Ubuntu

国际社区决定进入“移动


Ubuntu

”这一新的领域。

很明显，近几年来互联网的“移动设备”层出不穷，迅速地改变着现代人们的生活节奏和工作“模式”。这些“移动设备”（也可以成为“互联网设备”）向开源软件技术提出新的技术挑战，要求有全新的图形接口、能源管理和快速响应机制。当今，在这一技术领域，被许多私有软件技术所控制，不利于全球互联网的进一步完善和发展。

大家知道，多年以来，“桌面


Linux

”是一块“硬骨头”。有人主张跟着


Windows

跑，全面模仿、兼容


Windows

，但是，收效甚微。但是，主张“自主创新”（不是我们经常讲的那种单个民族的“自主创新”）的


Ubuntu

计划，用了不到


3

年时间，就敢于把


Ubuntu 7.04

桌面版本拿出来与


Vista

桌面进行公开比试，结果打个了“平手”，让不少国际知名人士“大跌眼镜”。

Ubuntu

开发者（蜂拥着）进入互联网移动领域，对于某些笃信私有软件发财致富的“移动老板”，（确实）不是一个好消息。
