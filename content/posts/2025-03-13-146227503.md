      扩写指令
     </strong>

    <pre><code>  - 儿童故事
   - 待扩写梗概：在一片生机盎然的森林中，动物们过着和谐的生活。最近，森林之王决定举办一场盛大的运动会，希望通过比赛激发动物们的团结精神和竞技热情。小鹿、小熊、小兔子、小象、小猩猩得知消息后，都跃跃欲试，期待能在比赛中大展身手。最后他们在森林接力赛中充分发挥自身特长及团队合作精神，打败了其他小动物组获得了冠军。
     - 扩写指令：请扩写以上内容形成一篇儿童故事，需要简单易懂、生动活泼、细节丰富，大于600字。
 - 情书
     - 待扩写梗概：亲爱的，自从那次在雨中与你邂逅，你灿烂的笑容就刻在了我心间。和你一起去过法国、巴黎、看过大海、搭讪，我真的很幸福，谢谢你的出现。
     - 任务：扩写成200字的情书，但输出情节不可以超出梗概
     - 扩写指令：动手试试你来写一个🫵
</code></pre>

    <h4>
     <a id="_40">
     </a>
     问答指令
    </h4>
    <p>
     通过一系列具体实例和步骤，指导您如何设计并应用不同类型的问答Prompt，以提升信息检索、理解和交流的效率。我们将覆盖直接问答、材料问答（含无答案与原文作答两种情况）、以及时间逻辑问答的设计与实操方法。
    </p>
    <h5>
     <a id="_44">
     </a>
     直接问答
    </h5>
    <p>
     直接问答是指用户直接提出问题，系统或人工智能直接给出答案的交互方式。这种方式简洁高效，适用于快速获取明确信息。
    </p>
    <ul>
     <li>
      <p>
       Prompt设计：
      </p>
      <ul>
       <li>
        问题：地球离太阳有多远？
       </li>
       <li>
        回答：地球距离太阳的平均距离约为1.496亿公里（或1个天文单位）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        实操步骤
       </strong>
       ：
      </p>
     </li>
    </ul>
    <ol>
     <li>
      明确问题焦点：确保问题是具体且可回答的。
     </li>
     <li>
      设计直接且清晰的Prompt格式，如“问题：… 回答：…”。
     </li>
    </ol>
    <h5>
     <a id="_58">
     </a>
     材料问答
    </h5>
    <p>
     材料问答是在给定一段或多段文本材料的基础上，用户提出问题，系统需从材料中寻找答案。这要求系统具备文本理解和信息抽取的能力。
    </p>
    <ul>
     <li>
      <p>
       无答案情况：
      </p>
      <ul>
       <li>
        <p>
         Prompt设计：
        </p>
        <ul>
         <li>
          材料：在探索世界的旅途中，我走访了许多令人惊叹的城市。首先，我来到了法国，被浪漫氛围深深吸引，尤其是埃菲尔铁塔的壮丽景色。随后，我前往了日本，那里的科技氛围和繁忙的街头景象让我印象深刻。接着，我飞越了大西洋，来到了美国，自由女神像作为这座城市的象征，矗立在哈德逊河口，仿佛在诉说着自由与梦想的故事。
         </li>
         <li>
          问题：文本中提到了哪些国家的首都？
         </li>
         <li>
          回答：
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <ul>
       <li>
        <p>
         实操步骤：
        </p>
        <ul>
         <li>
          准备文本材料，确保其内容丰富且具代表性。
         </li>
         <li>
          设计问题，考虑材料可能包含或不包含答案的情况。
         </li>
         <li>
          看看在无答案的情况下，大模型是否可以告诉你材料中无答案
         </li>
        </ul>
        <p>
         <img alt="image-20250313120448272" src="https://i-blog.csdnimg.cn/img_convert/b573c8aeaab30833e555a8c1a5674f8b.png"/>
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       原文作答情况：
      </p>
      <ul>
       <li>
        <p>
         Prompt设计：
        </p>
        <ul>
         <li>
          材料：中国，这个拥有悠久历史和灿烂文化的国家，其政治、经济、文化中心都集中在首都——北京。北京，作为中国的首都，不仅承载着深厚的文化底蕴，还展现着现代都市的繁荣与活力。故宫的红墙金瓦见证了明清两代的辉煌，天安门广场的庄严升旗仪式则是国家尊严与荣耀的象征。此外，北京还是科技创新的前沿阵地，众多高科技企业和研究机构汇聚于此，推动着中国乃至世界的发展。在这样的背景下，北京不仅是中国人民心中的骄傲，也是国际社会关注的焦点。
         </li>
         <li>
          问题：中国的首都是哪里？
         </li>
         <li>
          回答：
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <ul>
       <li>
        <strong>
         实操步骤
        </strong>
        ：
       </li>
      </ul>
     </li>
    </ul>
    <ol>
     <li>
      确保材料中包含问题的答案。
     </li>
     <li>
      设计Prompt时，引导系统直接引用材料中的原句或关键词作为答案。
     </li>
     <li>
      看看大模型能否准确识别并引用材料内容。
     </li>
    </ol>
    <p>
     <img alt="image-20250313120516064" src="https://i-blog.csdnimg.cn/img_convert/2cd8a1a564363a063ed57e79cc3b56e6.png"/>
    </p>
    <h5>
     <a id="_94">
     </a>
     时间逻辑问答
    </h5>
    <p>
     时间逻辑问答涉及对事件或信息在时间轴上的理解和推理。这类问题要求系统能够解析时间关系，如先后顺序、时间跨度等。
    </p>
    <ul>
     <li>
      <p>
       Prompt设计：
      </p>
      <ul>
       <li>
        问题：请按照时间顺序排列以下事件： 1) 互联网诞生 2) 人类登月 3) 蒸汽机的发明
       </li>
      </ul>
     </li>
     <li>
      <p>
       回答：
      </p>
     </li>
     <li>
      <p>
       <strong>
        实操步骤
       </strong>
       ：
      </p>
     </li>
    </ul>
    <ol>
     <li>
      <ol>
       <li>
        准备一系列有时间关联的事件或信息。
       </li>
       <li>
        设计问题，明确要求按时间顺序或其他时间逻辑进行排序或解释。
       </li>
       <li>
        看看大模型时间关系的理解和推理能力，确保答案准确无误。
       </li>
      </ol>
     </li>
    </ol>
    <p>
     <img alt="image-20250313120651812" src="https://i-blog.csdnimg.cn/img_convert/71b61f23fb6773d3f3551269f328afe9.png"/>
    </p>

     总结、摘要、翻译指令
    </h3>
    <h5>
     <a id="_113">
     </a>
     总结
    </h5>
    <p>
     文字总结是将长文或复杂信息精炼成简短、易于理解的形式，有助于快速把握核心内容。它适用于多种场景，如新闻报道、学术论文、产品说明等。
    </p>
    <p>
     <strong>
      实操
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       TLDR总结：
      </p>
      <ul>
       <li>
        <strong>
         原文：
        </strong>
        “本文深入探讨了人工智能在医疗领域的应用，包括疾病诊断、个性化治疗方案设计以及远程医疗服务的发展。通过多个案例分析，展示了AI如何显著提高医疗效率和患者满意度。”
       </li>
       <li>
        <strong>
         总结：
        </strong>
        试着在末尾加个TLDR看看？🫵
       </li>
      </ul>
     </li>
     <li>
      <p>
       有字数要求的总结：
      </p>
      <ul>
       <li>
        <strong>
         要求
        </strong>
        ：将上述内容总结为不超过50字。
       </li>
       <li>
        <strong>
         总结
        </strong>
        ：试试看吧🫵
       </li>
      </ul>
      <blockquote>
       <p>
        **什么是 TLDR？**TLDR，全称 “Too Long; Didn’t Read”，在互联网上常用来表示对过长内容的总结。而 TLDR 项目则将这一理念应用到了命令行工具的帮助文档上。它以一种更简洁、易懂的方式为常用的命令行工具提供使用示例和说明，旨在帮助用户快速掌握命令的使用方法，提高工作效率。
       </p>
      </blockquote>

     </li>
    </ul>
    <h5>
     <a id="_133">
     </a>
     信息抽取
    </h5>
    <p>
     信息抽取是从大量文本数据中自动提取出结构化信息的过程，如关键词、实体、关系等。这在舆情监测、市场分析等领域尤为重要。
    </p>
    <p>
     <strong>
      实操：舆情监测
     </strong>
    </p>
    <ul>
     <li>
      <p>
       提取关键词：
      </p>
      <ul>
       <li>
        <strong>
         例子
        </strong>
        ：文本：“用户对最新发布的智能手机普遍给予好评，称赞其摄像头性能卓越，电池续航能力强，但价格略高。”
       </li>
       <li>
        <strong>
         关键词提取
        </strong>
        ：智能手机、好评、摄像头性能、电池续航、价格略高。
       </li>
      </ul>
     </li>
     <li>
      <p>
       情感分析：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          任务
         </strong>
         ：判断文本表达的情感倾向（正面、负面、中性）。
        </p>
       </li>
       <li>
        <p>
         例子：同上文本。
        </p>
        <ul>
         <li>
          <strong>
           情感分析
          </strong>
          ：看看大模型能不能分辨出其中的情感吧
         </li>
        </ul>

       </li>
       <li>
        <p>
         <strong>
          待分析评论：
         </strong>
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <ol>
     <li>
      “自从发现了这款泡面，家里的零食柜就再也没缺过货。口味多样，每一种都让人回味无穷，特别是那个海鲜味，鲜得眉毛都要掉下来了！”

     翻译

    </p>
    <p>
     <strong>
      实操：语言翻译
     </strong>
    </p>
    <ul>
     <li>
      <p>
       例子：
      </p>
      <ul>
       <li>
        <strong>
         原文
        </strong>
        （中文）：“北京是中国的首都，拥有丰富的历史文化遗产和现代化的城市风貌。”

        <strong>
