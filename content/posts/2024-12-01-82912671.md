# 解决python报错ImportError: No module named '\_tkinter'

## 解决python报错ImportError: No module named ‘\_tkinter’

报错信息：ImportError: No module named ‘\_tkinter’, please install the python3-tk package

* 尝试一

  sudo pip3 install python3-tk

  反馈：Could not find a version that satisfies the requirement python3-tk (from versions: )
    
  No matching distribution found for python3-tk
* 尝试二

  sudo apt install python3-tk

  问题解决！
