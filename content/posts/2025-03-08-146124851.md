    <h1 class="title-article" id="articleContentId">
     three.js 在 webGL 添加纹理
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
    </p>
    <p>
     在我们生成了3D设计之后，我们可以添加纹理使其更加吸引人。在 webGL 和 p5.js中，可以使用 gl.texImage2D() 和 texture()  API来为形状应用纹理。
    </p>
    <h4>
     使用 webGL
    </h4>
    <p>
     在 webGL 中，gl.texImage2D() 函数用于从图像文件生成2D纹理。该函数接受许多参数，包括目标，细节级别，内部格式，图像的宽度和高度，以及图像数据的格式和类型。
    </p>
    <p>
     为了方便，我将使用
     <a href="https://vitejs.dev/" rel="nofollow" title="vite">
      vite
     </a>
     搭建一个原生 js 项目。
    </p>
    <h5 id="创建项目">
     1.创建项目
    </h5>
    <p>
     npm create
     <a href="https://so.csdn.net/so/search?q=vite&amp;spm=1001.2101.3001.7020" title="vite">
      vite
     </a>
     @latest p5-demo
     <br/>
     选：Vanilla
     <br/>
     选：JavaScript
    </p>
    <p>
     <strong>
      2.初始化项目
     </strong>
     <br/>
     cd p5-demo
     <br/>
     cnpm install
    </p>
    <p>
     <strong>
      3.安装 p5.js
     </strong>
     <br/>
     cnpm install p5 --save
    </p>
    <p>
     cd p5-demo
     <br/>
     curl -O https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.min.js
    </p>
    <p>
     <strong>
      编写 three_texture.html
     </strong>
     如下
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
   &lt;meta charset="UTF-8"&gt;
   &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
   &lt;script src="three.min.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&lt;body&gt;
   &lt;script&gt;
      // curl -O https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.min.js
      // Set up the scene
      var scene = new THREE.Scene();
      var camera = new THREE.PerspectiveCamera(75,
            window.innerWidth / window.innerHeight,
            0.1, 1000
        );
      var renderer = new THREE.WebGLRenderer();
      renderer.setSize(window.innerWidth, window.innerHeight);
      document.body.appendChild(renderer.domElement);

      // Create a cube
      var geometry = new THREE.BoxGeometry(3, 3, 3);//
      // 随意选一个.jpg 文件作为纹理图像
      var texture = new THREE.TextureLoader().load("./海边.jpg");
      var material = new THREE.MeshBasicMaterial({ map: texture });
      var cube = new THREE.Mesh(geometry, material);
      scene.add(cube);

      // Position the camera
      camera.position.z = 5; 
      // Render the scene
      function render() {
         requestAnimationFrame(render);
         cube.rotation.x += 0.01;
         cube.rotation.y += 0.01;
         renderer.render(scene, camera);
      }
      render();
   &lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <p>
     <strong>
      5.运行
     </strong>
     npm run dev
     <br/>
     访问 http://localhost:5173/three_texture.html
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/230d9118f1b74d3fbe41a6e37a03574a.png"/>
    </p>
    <hr/>
    <p>
     在 p5.js 中使用 texture() 函数可以将纹理应用到一个对象上。texture() 函数接受一个参数：纹理图像文件。
    </p>
    <p>
     <strong>
      编写 p5_texture.html
     </strong>
     如下
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
   &lt;meta charset="UTF-8"&gt;
   &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
   &lt;title&gt;p5.js Texture Example&lt;/title&gt;
   &lt;script src="p5.min.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&lt;body&gt;
   &lt;script&gt;
      let img;
      function preload() { 
         img = loadImage("./海边.jpg");
      }
      function setup() {
         createCanvas(650, 400, WEBGL);
         noStroke();
      }
      function draw() {
         background(200);
         texture(img);
         rotateX(frameCount * 0.01);
         rotateY(frameCount * 0.01);
         box(100);
      }
   &lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
    <p>
     访问 http://localhost:5173/p5_texture.html
    </p>
    <p>
     可见同样的程序 p5.js 的代码量比 three.js 和 python 都要少。
    </p>
