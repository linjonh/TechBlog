# 如何ping通后端的地址？

当你在联调的时候，如何查看是否后端ip地址是否通?

```
// window + r --> cmd -->  telnet 后端地址   端口号
例如： telnet 192.168.xxx.xxx xxx

```

后端给你链接，如何查看链接是否可行？
  
把链接复制到浏览器的网址上，或者用手机打开该网址
