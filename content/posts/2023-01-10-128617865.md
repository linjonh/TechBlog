# python环境免费

## 前言

之前发布了C/C++的开发环境搭建教程，除了 C/C++ 之外，Python 在部分高校中也很受青睐，同时我也十分建议初学者从 Python 开始入门。所以今天就来教大家如何搭建 Python 的开发环境。

Python 是一门对新手十分友好的语言，这点也体现在了 Python 的环境搭建中。该文章介绍三种搭建环境的方式，操作都非常简单，各位可以按需选择。

### 安装 Python，使用自带的 IDLE

Python 在官网提供了安装程序，我们只需要在 Python 的官网下载并运行安装程序就可获得 Python 的开发环境和自带的代码编辑器 IDLE。IDLE 对于初学者来说基本够用，但是个人感觉编辑器界面不够美观，不适合长期使用。

安装教程：

打开Python官网：www.python.org

点击Downloads

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/2c524d13ed0228f72c8fbafefa006dc0.png#pic_center)
  
若没有特定的版本要求，可以直接点击 Download Python 3.xx.x 下载（这里默认会下载最新版本的 Python，下载速度可能会很慢，建议多尝试几次）

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/0f364a33a790ecc69bf27f514110f064.png#pic_center)
  
下载完成后运行，勾选 Add python to PATH 后，点击 Install Now 即可开始安装。

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/bea902c69ef3cd8b6948c5159212b151.png#pic_center)
  
安装完成后在开始菜单中可找到Python文件夹，其中就包含了 IDLE。

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/127d8ef4befca3721e9e5b47a333cc3a.png#pic_center)

### 2. 安装 PyCharm

PyCharm 是著名的 IDE 开发公司 JetBrains 针对 Python 语言开发的 IDE，拥有十分丰富且强大的功能，并且社区版免费。

安装教程（安装 PyCharm 前需先安装 Python）：

打开 PyCharm 的官网：www.jetbrains.com/pycharm/

点击 Download

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/ffd07e81b1322efb1823e35115e4c105.png#pic_center)
  
下载社区版

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/b3f3aab95ce30c081063dc8e6d2ee345.png#pic_center)
  
下载完成后运行，一路点击 Next ，到如下界面时可以按需勾选。

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/411f19dede68f5feb10bf4d16135eb44.png#pic_center)
  
选择完成后继续按照提示操作完成安装。

### 3. 安装 VS code

VS code 是一个多功能的代码编辑器，之前在 C/C++ 开发环境搭建教程中也提到过，VS code 可以通过安装不同的插件来兼容不同的编程语言，所以只需要安装 Python 之后再安装 VS code 的 Python 插件即可使用 VS code 来编写Python代码。

VS code 的安装方法在之前的 C/C++ 的环境搭建教程中已经提及，此处不再赘述，如果想了解详细的安装方法可以查看之前的推送：手把手教你搭建开发环境之 C/C++

这里附上 VS code 的官网，以免各位找错（因为之前确实有见到一些同学因为下载 VS code 找错官网被骗的）：code.visualstudio.com

安装好 VS code 之后，在扩展商店搜索 Python，安装第一个 Python 插件。

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/33c2a00fe1d125b55c4753326ee2e2a0.png#pic_center)
  
安装完成插件后就可以使用 VS code 来编写和运行 Python 代码啦。
