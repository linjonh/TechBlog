    <p>
     做基于uni-app的微信小程序的时候，遇到了一点问题，uni-app（小程序）背景图片的处理一直报错。
    </p>
    <h3>
     <a id="_1">
     </a>
     背景怎么引用本地图片
    </h3>
    <p>
     想要为
     <code>
      &lt;view&gt;
     </code>
     标签添加一个背景，常规做法：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c77b1700095dd8ab1f34a2ec98618a55.png">
      然后就报错啦：本地资源图片无法通过WXSS获取，可以使用网络图片，或者base64,或者使用
      <code>
       &lt;image/&gt;
      </code>
      标签
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e420291eeae263e6f21c589064dad260.png">
       解决方案按照报错信息也给出来了：
      </img>
     </img>
    </p>
    <ol>
     <li>
      使用网络图片，找个图床把你要用的图片传上去，然后以网络图片的形式引入到你的项目里面。
     </li>
     <li>
      使用使用
      <code>
       &lt;image/&gt;
      </code>
      标签，这个就不多说了，image作为背景，用绝对定位分隔图层出来就可以实现，可行但是我初衷还是想要用background的形式引入背景图片。
     </li>
     <li>
      将图片转为base64，我用的是这个方法，网上有免费的在线转换器，比如：
      <a href="http://imgbase64.duoshitong.com/" rel="nofollow">
       http://imgbase64.duoshitong.com/
      </a>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/61d0e1d90d9ecb9873be59d655a03fc8.png">
       上传图片，转换，复制得到的base64编码，然后粘贴到原来的
       <code>
        background:url(base64编码)
       </code>
       就可以了。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/57bccfb28e82b514f95e52c039ef168b.png"/>
      </img>
     </li>
    </ol>
    <h3>
     <a id="base64_10">
     </a>
     集中处理base64编码的图片并引入
    </h3>
    <p>
     上一步用图片转成base64的编码很长，放在vue文件里非常不美观，还有就是我的Hbuilder编辑器滚动时会因此卡顿，所以为了可持续发展，早早的把这些base64编码变量抽离出来。
    </p>
    <ol>
     <li>
      新建一个scss文件集中放这些base64编码图片
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9827b1989119cae688d258edcca125d5.png"/>
     </li>
     <li>
      在uni.scss文件中导入这个自定义文件，作为预处理变量
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dcc03d068dc295ede1a4100543fc9873.png"/>
     </li>
     <li>
      在vue文件里用相应的变量来引入图片
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/27a6235097994487f6e3c1d8dc389875.png">
       <br/>
       这样就大功告成啦~
       <br/>
       效果：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/427aebf7ff723b540a132c16efa407fd.png"/>
      </img>
     </li>
    </ol>
