### 三、Ubuntu操作系统设置静态IP地址

1、备份网卡配置文件
  
注意：网络配置文件名称不一定是
`01-network-manager-all.yaml`
，进到
`/etc/netplan/`
目录
`ls`
出来的就是网络的配置文件！！

```bash
cd /etc/netplan/
cp 01-network-manager-all.yaml 01-network-manager-all.yaml.default

```

2、修改网卡配置文件
  
注意：网卡配置文件是yaml格式，修改时注意缩进，内容要遵循yaml格式！！

```yaml
cat 01-network-manager-all.yaml
network:
    ethernets:
        ens33:
            dhcp4: no
            addresses:
                    - 16.32.15.123/24
            gateway4: 16.32.15.254 
            nameservers:
                    addresses:
                            - 8.8.8.8
    version: 2

```

3、使用 netplan apply 命令立即生效

```bash
netplan apply 

```

4、验证

```bash
ip addr

```

### 四、Ubuntu设置国内阿里镜像源

国内镜像源有很多，这里使用的是阿里云镜像源
  
1、备份sources.list文件

```bash
cp /etc/apt/sources.list /etc/apt/sources.list.back

```

2、更新sources.list文件内阿里云地址
  
首先清空sources.list文件内容

```bash
cd /etc/apt/
> sources.list

```

将一些内容粘贴到sources.list文件

```bash
deb https://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
deb-src https://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
deb https://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
deb-src https://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
deb https://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
deb-src https://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
# deb https://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
# deb-src https://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
deb https://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse
deb-src https://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse

```

3、更新软件列表和软件源

```bash
sudo apt-get update
sudo apt-get upgrade

```

4、安装
`vim`
命令进行测试

```bash
apt-get purge vim-common
apt install vim

```

### 五、安装sshd服务并Xshell远程连接测试

1、安装sshd服务

```bash
sudo apt-get install openssh-server

```

2、启动sshd服务

```bash
/etc/init.d/ssh start

```

3、验证22端口是否启动

```bash
ss -anput |grep 22
```
