    <h1 class="title-article" id="articleContentId">
     Adobe Photoshop CC 2025配置要求
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <ul>
     <li>
      <strong>
       操作系统
      </strong>
      <ul>
       <li>
        <strong>
         Windows
        </strong>
        ：Windows 10（版本 22H2）或 Windows 11（版本 21H2、22H2、23H2）。
       </li>
       <li>
        <strong>
         macOS
        </strong>
        ：macOS Monterey（12.0）或更高版本。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       处理器
      </strong>
      ：支持 AVX2 和 SSE 4.2 的多核 Intel、AMD 或 WinARM 处理器。推荐使用最新一代的 Intel Core 或 AMD Ryzen 处理器。
     </li>
     <li>
      <strong>
       内存
      </strong>
      <ul>
       <li>
        <strong>
         最低要求
        </strong>
        ：8GB RAM。
       </li>
       <li>
        <strong>
         推荐配置
        </strong>
        ：16GB 或更高，特别是对于处理高分辨率图像或进行复杂设计的用户。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       显卡
      </strong>
      <ul>
       <li>
        <strong>
         最低要求
        </strong>
        ：支持 DirectX 12 的 GPU，至少 1.5GB GPU 内存。推荐 NVIDIA GeForce GTX 10 系列或更高，或 AMD Radeon RX 500 系列或更高。
       </li>
       <li>
        <strong>
         macOS
        </strong>
        ：支持 Metal 的 GPU。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       硬盘空间
      </strong>
      <ul>
       <li>
        <strong>
         最低要求
        </strong>
        ：至少 10GB 的可用空间。
       </li>
       <li>
        <strong>
         推荐配置
        </strong>
        ：SSD 固态硬盘，以提高软件启动速度和文件读写效率。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       显示器
      </strong>
      ：支持 OpenGL 4.x 的显示器，分辨率建议为 1920x1080 或更高。
     </li>
     <li>
      <strong>
       其他
      </strong>
      ：用于 Adobe Creative Cloud 的网络连接。
     </li>
    </ul>
