    <h1 class="title-article" id="articleContentId">
     H3C交换机中 LI、SI、EI、HI 各型号的区别
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <div class="artical-content-bak main-content editor-side-new">
     <div class="con editor-preview-side" id="result">
      <p>
       <span style="font-family:Simsun;">
        LI(Lite software Image)表示设备为弱特性版本。
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        SI (Standard software Image)表示设备为标准版本，包含基础特性。
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        EI(Enhanced software Image)表示设备为增强版本，包含某些高级特性。
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        HI(Hyper software Image)表示设备为高级版本，包含某些更高级特性
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        Z，表示没有上行接口；（新产品不允许此位）
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        G，表示上行GBIC接口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        P，表示上行SFP接口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        T，表示上行RJ45接口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        V，表示上行VDSL接口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        W，表示上行可配置WAN接口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        C，表示上行接口可选配；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        M，表示上行接口为多模光口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        S，表示上行接口为单模光口；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        F，表示下行接口为模板板，可插光接口板或电接口板。主要为兼容3526F，3526EF，3552F等老产品的命名。
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        当同时存在时，表示上行接口为多种接口类型复合
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        注：Combo端口不在命名中显示。
       </span>
       <br style="font-family:Simsun;"/>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        EI的增强特性有：
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        1、支持基于协议的VLAN；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        2、支持Voice VLAN；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        3、支持配置用户IP+MAC地址+端口的组合绑定；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        4、支持DLDP（Device Link Detection Protocol，设备链路探测协议）；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        5、支持配置禁止VLAN学习MAC地址；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        6、支持EAD（Endpoint Admission Defense，端点准入防御）快速部署；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        7、支持在 DHCP Snooping 功能中应用 Option82；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        8、支持基本/高级 ACL（S3100-SI 系列交换机上定义的基本/高级ACL 只能用于被上层软件引用的情况，不支持下发到硬件；S3100-EI 系列交换机上定义的 ACL 支持被上层软件引用和下发到硬件）；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        9、支持二层ACL；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        10、支持QoS Profile ；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        11、支持SSH1版本；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        12、支持VLAN Mapping；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        13、支持BPDU Tunnel；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        14、支持灵活QinQ；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        15、支持IPv6地址配置；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        16、支持基于 IPv6 的 Ping、Traceroute、TFTP、Telnet 应用；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        17、支持动态域名解析；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        18、支持Smart Link；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        19、支持Monitor Link；
       </span>
       <br style="font-family:Simsun;"/>
       <span style="font-family:Simsun;">
        20、支持多业务VLAN规划
       </span>
      </p>
