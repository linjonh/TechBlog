     1、查询数据导出csv数据
    </p>
    <pre><code class="prism language-sql"> \COPY <span class="token punctuation">(</span><span class="token keyword">SELECT</span> w<span class="token punctuation">.</span><span class="token operator">*</span> <span class="token keyword">from</span> t_sys_warn w  <span class="token punctuation">)</span> <span class="token keyword">TO</span> <span class="token string">'/home/cuadmin/warn_output.csv'</span> <span class="token keyword">WITH</span> CSV HEADER<span class="token punctuation">;</span>

    <p>
     2、导出sql Insert语句
    </p>
    <pre><code class="prism language-bash">pg_dump <span class="token parameter variable">-U</span> 用户名 <span class="token parameter variable">-h</span> 主机名 <span class="token parameter variable">-p</span> 端口号 <span class="token parameter variable">-d</span> 数据库名 <span class="token parameter variable">--inserts</span> <span class="token parameter variable">-t</span> 表名 <span class="token operator">&gt;</span> 导出文件.sql

pg_dump <span class="token parameter variable">-U</span> username <span class="token parameter variable">-d</span> dbname <span class="token parameter variable">-t</span> tablename --data-only --column-inserts <span class="token operator">&gt;</span> filename.sql
</code></pre>
    <p>
     -U username：指定数据库的用户名。
    </p>
    <p>
     -d dbname：指定要导出的数据库名称。
    </p>
    <p>
     -t tablename：指定要导出的表名。如果你需要导出多个表，可以多次使用 -t 选项，或者省略此选项以导出整个数据库。
    </p>
    <p>
     –data-only：只导出数据，不包括表结构。
    </p>
    <p>
     –column-inserts：生成格式化的 INSERT 语句，每列一个值。
    </p>
    <blockquote>
     <p>
      filename.sql：将输出重定向到文件。
     </p>
    </blockquote>
