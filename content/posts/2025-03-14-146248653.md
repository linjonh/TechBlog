    <h2>
     Java EE Web环境安装
    </h2>
    <p>
     一、JDK安装与测试（Windows环境）
    </p>
    <h5>
     1. 安装JDK
    </h5>
    <ol>
     <li>
      官网下载：
      <a href="https://www.oracle.com/java/technologies/downloads/" rel="nofollow">
       Oracle JDK
      </a>
      （选择Windows x64 Installer）
     </li>
     <li>
      双击安装包，按向导完成安装
     </li>
    </ol>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/direct/b0aeeda2a10849708d0f275465a97286.png">
      ​
     </img>
    </p>
    <h5>
     2. 环境变量配置
    </h5>
    <ol>
     <li>
      <p>
       右键【此电脑】→【属性】→【高级系统设置】→【环境变量】
      </p>
     </li>
     <li>
      <p>
       新建系统变量：
       <br/>
       ​
       <img alt="image" src="https://i-blog.csdnimg.cn/direct/2e8d847ef7a6458ba2a63d38ac0bac2a.png"/>
      </p>
     </li>
     <li>
      <p>
       编辑系统变量
       <code>
        Path
       </code>
       ​：
      </p>
      <ul>
       <li>
        新增条目：
        <code>
         %JAVA_HOME%\bin
        </code>
        ​
        <br/>
        ​
        <img alt="image" src="https://i-blog.csdnimg.cn/direct/d3899f78df3c4eec8fa3ec9baf2f4f28.png"/>
       </li>
      </ul>
     </li>
     <li>
      <p>
       验证配置（CMD执行）：
      </p>
      <pre><code class="language-bash prism"><span class="token function">java</span> <span class="token parameter variable">-version</span>
javac <span class="token parameter variable">-version</span>
</code></pre>
      <p>
       <img alt="image" src="https://i-blog.csdnimg.cn/direct/7d83f5d9f8c046fca345885302d21702.png"/>
      </p>
     </li>
    </ol>
    <p>
     ‍

    <hr/>
    <h4>
     二、Tomcat安装与测试（绿色版）
    </h4>
    <h5>
     1. 下载解压
    </h5>
    <ol>
     <li>
      官网下载：
      <a href="https://tomcat.apache.org/" rel="nofollow">
       Apache Tomcat
      </a>
      （选择
      <code>
       zip
      </code>
      ​格式的Core版本）
     </li>
     <li>
      解压
     </li>
    </ol>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/direct/730302f2aa934feebb5754852b4f8223.png"/>
    </p>
    <h5>
     2. 启动与测试
    </h5>
    <ol>
     <li>
      进入
      <code>
       bin
      </code>
      ​ 目录，双击
      <code>
       startup.bat
      </code>
      ​ 启动服务
      <br/>
      ​
      <img alt="image" src="https://i-blog.csdnimg.cn/direct/ab6cf1d54fbb45f2a3662f88cb94ff68.png"/>
     </li>
     <li>
      浏览器访问
      <code>
       http://localhost:8080
      </code>
      ​，显示Tomcat欢迎页面即成功
      <br/>
      ​
      <img alt="image" src="https://i-blog.csdnimg.cn/direct/0215bf8d3bdd4efca09d5aeeb234a8c2.png"/>
     </li>
    </ol>
    <hr/>
    <h4>
     三、Eclipse安装与配置
    </h4>
    <h5>
     1. 下载解压
    </h5>
    <ol>
     <li>
      官网下载：
      <a href="https://www.eclipse.org/downloads/" rel="nofollow">
       Eclipse IDE
      </a>
      （选择"Eclipse IDE for Enterprise Java and Web Developers"）
     </li>
     <li>
      解压
     </li>
    </ol>
    <p>
     <img alt="image" src="https://i-blog.csdnimg.cn/direct/7a5bd43a73fe40bea757bfffc8a2a97c.png"/>
    </p>
    <h5>
     2. 配置Tomcat服务器
    </h5>
    <ol>
     <li>
      启动Eclipse → 菜单栏【Window】→【Preferences】→【Server】→【Runtime Environments】
     </li>
     <li>
      点击【Add】→ 选择"Apache Tomcat"版本 → 指定Tomcat解压目录 → 完成
      <br/>
      ​
      <img alt="image" src="https://i-blog.csdnimg.cn/direct/b011e99e3bd946439b15a73c50bd0f06.png"/>
      <br/>
      ​
      <img alt="image" src="https://i-blog.csdnimg.cn/direct/9bcb2aacdfe743188ce6205c9a44fa6f.png"/>
     </li>
    </ol>
    <h5>
     3. 创建Web工程
    </h5>
    <ol>
     <li>
      <p>
       【File】→【New】→【Dynamic Web Project】
       <br/>
       ​
       <img alt="image" src="https://i-blog.csdnimg.cn/direct/69472d3d363f442e8c3d05fe2abf9479.png"/>
      </p>
     </li>
     <li>
      <p>
       输入项目名 → 选择Tomcat运行时 → 完成
      </p>
     </li>
     <li>
      <p>
       在
       <code>
        src
       </code>
       ​目录新建Java类（示例代码）：
      </p>
      <pre><code class="language-java prism"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">HelloWorld</span> <span class="token punctuation">{<!-- --></span>
