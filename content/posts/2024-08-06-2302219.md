# Eclipse的安装和Java项目创建

**Eclipse的安装和Java项目创建**

1.

Eclipse

的安装

Eclipse

是开放源码的项目，因此可以到

www.eclipse.org

下载

Eclipse SDK

。它包括

Eclipse

开发环境、

Java

开发环境、

Plug-in

开发环境及其所有的源代码和文档。但如果只是进行

Java

应用，而不是插件开发或者研究

Eclipse

代码，则只需要下载

Platform Runtime Binary+JDT Runtime Binary

。

下载之后，直接解压到指定目录即可，无需安装。但这之前，请确定你已安装好了

JDK

，以及配置好了

JDK

的环境变量。第一次使用，双击运行，会提示你设定工作空间

(Workspace)

，它将作为文件存储的位置。此外，中文用户还可以安装

Eclipse

多国语言包插件，该插件可以根据操作系统的语言环境选择对应的语言包，对

Eclipse

进行本地区域化。可以到

www.eclipse.org

下载对应版本的语言包。下载之后，需要在关闭

Eclipse

的前提下：

·

先对压缩包解压。

·

然后，将获得的文件保存到

Eclipse

安装目录的

Language

子目录下。

·

接着，还需要在

Eclipse

的安装目录下新创建一个目录

links

，并在其下新建一个文本文件，再修改其文件名及扩展名为

language.strat

（文件名可以任意，不一定要是

language

）。

·

最后，打开这个文件键入

path=d://eclipse//language(

即多国语言包

language

的目录路径

)

。

至此，重新启动

Eclipse

，就进入了中文环境。如果，在这之前曾使用过

Eclipse

，还可能会有部分英文。可以删除

configuration

子目录下面的

org.eclipse.update

目录，以删除原有的记录信息后，再启动即可。

为了使编写的程序文件都能正常的现实中文信息，需要设置文本文件编码类型为

UTF8

。方法是选择



窗口

->

首选项

，在弹出对话框中选择



常规

->

编辑器

，在右侧文本文件编码设定中的“其它”中，从下拉列表选项中选择

UTF-8

。

2

.Eclipse

集成开发环境介绍

Eclipse

开发环境在启动后，就进入了

Eclipse

的默认视图——

Java

透视图。

可以

浏览一下帮助中的《工作台用户指南》，熟悉一下概念

。

透视图包括以下

4

个部分

：

·

编辑器窗格：代码编写和调试的核心区域。开发人员可以方便地进行编写和查看，并获得

Eclipse

提供的提示和帮助。

·

包资源管理器窗格：显示在

Eclipse

中进行开发的项目的资源，随着开发的项目类型的不同而改变，例如开发

Java

项目时，就显示包资源管理器。

·

大纲窗格：以层次结构清晰地显示出程序源代码的基本结构，有利于程序的迅速定位和查看。

·

任务窗格：在项目开发过程中会有不同的选项卡，例如：控制台、

JavaDoc

文档等。通过这个窗格的信息，开发者可以在编写过程中或运行过程中，查看到程序的错误提示或者控制台输出等信息。

3.

创建

Java

项目

■

直接创建：

在

Eclipse

中使用

Java

开发工具进行

Java

项目开发，简称为

JDT

的开发。

·

首先，要确认

Eclipse

中与

JDK

相关的环境变量是否正确：打开



窗口

->

首选项



，选择树形窗格中的



Java->

已安装的

JRE



，检查对应路径的

JRE

安装。

·

然后，再在菜单中选择



文件

->

新建

->

项目



，在弹出对话框向导中，选择向导中的



Java->Java

项目



。

·

接着，进入“下一步”，输入项目名称，其他可以默认，也可以设置项目创建的内容位置或来源、

JDK

一直性、项目布局等细节。

·

再进入“下一步”，确认与项目相关的各个构建的设置，要注意检查以一下“

库

”选项卡中与

JDK

相关的制作。

·

最后，确定。如果原先的视图不是

Java

透视图，屏幕会弹出对话框，提示你切换到

Java

透视图，单击“是”。

创建完成后，会发现由于是

Java

项目，在包资源管理器中有相关的包资源的树状结构，即

JDK

相关的包。同时，在

Eclipse

的工作空间所在的硬盘目录中，

Eclipse

创建了项目对应的文件夹，生成了与项目相关的辅助文件。

■

打开已有

Java

项目

如果要把已有的

Java

项目导入到

Eclipse

开发环境。

·

在



包资源管理器



窗格空白处单击右键，在快捷菜单中选择

导入

。

·

在弹出的对话框向导中选择



现有项目到工作空间



。

·

点击下一步，浏览选择已有项目文件夹目录位置。

·

确定，即导入完成。

注意的是，如果该项目位置原本不在

Eclipse

工作空间下，那么导入后该项目的位置仍处于原位置。

■

导入外部类

此外，还可以进行类的导入，将已存在的外部资源导入到当前项目中。

·

在包资源管理器的某个项目名称上右击，从快捷菜单中选择

导入

。

·

然后，在弹出的对话框中浏览目录，选择

Java

外部类资源文件的位置。再勾选择需要的

.java

类文件即可。

·

确定完成，该类即被导入当前项目。
