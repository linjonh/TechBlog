      1. RESTful接口测试全流程
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Postman自动化测试脚本
      </strong>
      ：
      <pre><code class="language-javascript">// 在Postman的Tests标签中编写断言  
pm.test("Status code is 200", () =&gt; pm.response.to.have.status(200));  
pm.test("Response data is valid", () =&gt; {  
    const data = pm.response.json();  
    pm.expect(data.userId).to.eql(1);  
});</code></pre>
     </li>

      2. WebSocket实时通信测试

    </h4>

      <strong>
       浏览器控制台测试
      </strong>
      ：
      <pre><code class="language-javascript">const socket = new WebSocket("ws://echo.websocket.org");  
socket.onmessage = (event) =&gt; console.log("Received: " + event.data);  
socket.send("Hello Server!");</code></pre>
     </li>
     <li>
      <strong>
       断线重连测试
      </strong>
      ：
      <pre><code class="language-javascript">socket.onclose = () =&gt; setTimeout(() =&gt; {  
    socket = new WebSocket(socket.url);  // 自动重连  
}, 3000);</code></pre>
