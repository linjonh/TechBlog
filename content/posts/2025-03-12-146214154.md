     49. 字母异位词分组
    </h2>
    <ul>
     <li>
      <p>
       给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。
      </p>
     </li>
     <li>
      <p>
       字母异位词 是由重新排列源单词的所有字母得到的一个新单词。
      </p>
     </li>
     <li>
      <p>
       示例 1:
      </p>
      <ul>
       <li>
        输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]
       </li>
       <li>
        输出: [[“bat”],[“nat”,“tan”],[“ate”,“eat”,“tea”]]
       </li>
      </ul>
     </li>
     <li>
      <p>
       示例 2:
      </p>
      <ul>
       <li>
        输入: strs = [“”]
       </li>
       <li>
        输出: [[“”]]
       </li>
      </ul>
     </li>
     <li>
      <p>
       示例 3:
      </p>
      <ul>
       <li>
        输入: strs = [“a”]
       </li>
       <li>
        输出: [[“a”]]
       </li>
      </ul>
     </li>
     <li>
      <p>
       思路：
      </p>
      <ul>
       <li>
        <ol>
         <li>
          字母异位词的特点是它们包含的字母种类和数量完全相同，只是排列顺序不同。因此，可以利用排序法来识别哪些单词是字母异位词（26字母表顺序）。
         </li>
        </ol>
       </li>
       <li>
        <ol start="2">
         <li>
          创建一个哈希表，键是排序后的字符串，值是对应的字母异位词列表。遍历字符串数组，将每个单词排序后作为键，并将单词添加到对应的字母异位词列表中。
         </li>
        </ol>
       </li>
      </ul>
     </li>
    </ul>
