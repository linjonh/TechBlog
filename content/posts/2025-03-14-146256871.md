# Ubuntu 安装Mujoco3.3.0

### 1.打开终端，使用以下命令将下载的 MuJoCo 压缩包解压到~/.mujoco目录

> mkdir -p ~/.mujoco
>
> tar -xvzf mujoco-3.3.0-linux-x86\_64.tar.gz -C ~/.mujoco

### 2.配置环境变量

打开终端，编辑~/.bashrc文件

> nano ~/.bashrc

在文件末尾添加以下内容，ctrl+s保存

> export LD\_LIBRARY\_PATH=$LD\_LIBRARY\_PATH:~/.mujoco/mujoco3-3.0/bin

![](https://i-blog.csdnimg.cn/direct/16db6fef3f3b4bef9a75332d08474f21.png)

保存并退出文件，然后执行以下命令使配置生效

> source ~/.bashrc

### 3.验证安装

> pip install mujoco

创建一个 Python 脚本，例如tes.py，内容如下：

> import mujoco # 加载一个示例模型 model = mujoco.MjModel.from\_xml\_path('/home/user\_name/.mujoco/mujoco-3.3.0/model/humanoid/humanoid.xml') data = mujoco.MjData(model) # 模拟一段时间 for \_ in range(1000): mujoco.mj\_step(model, data) print("MuJoCo安装成功！")

测试 ：

> python test.py

输出 “MuJoCo 安装成功！”，则说明 MuJoCo 已经成功安装。
