    <h3>
     <a id="4_OpenManus__58">
     </a>
     4、配置 OpenManus 环境
    </h3>
    <ul>
     <li>
      如果是 PyCharm 启动，则直接手动复制
      <code>
       /config/config.example.toml
      </code>
      将复制出来的文件名改为
      <code>
       config.toml
      </code>
      即可
     </li>
    </ul>
    <pre><code class="prism language-toml"># Global LLM configuration
[llm]
model = "qwen-plus"
base_url = "https://dashscope.aliyuncs.com/compatible-mode/v1"
api_key = "sk-.................................."
max_tokens = 4096
temperature = 0.0
</code></pre>
    <blockquote>
     <p>
      model ：一定要是支持 function calling 的模型
      <br/>
      base_url：百炼云平台的接口地址，只要是百炼的模型，接口地址几乎是相同的，不用修改
      <br/>
      api_key：这个就是之前注册的 api-key，直接复制接口
      <br/>
      其他的都不用修改
     </p>
    </blockquote>
    <ul>
     <li>
      如果是通过 CMD 启动的，则使用命令复制文件，然后将正确的内容填写进去即可
     </li>
    </ul>
    <pre><code class="prism language-shell"><span class="token function">cp</span> config/config.example.toml config/config.toml
</code></pre>
