     <h4>
      【题目描述】
     </h4>
     <p>
      将一个数组中的值按逆序重新存放。例如，原来的顺序为8,6,5,4,1。要求改为1,4,5,6,8。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      两行：第一行数组中元素的个数n（1&lt;n&lt;100)，第二行是n个整数，每两个整数之间用空格分隔。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      一行：输出逆序后数组的整数，每两个整数之间用空格分隔。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      5
      <br/>
      8 6 5 4 1
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      1 4 5 6 8
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e2 + 10;
int nums[N];

int main()
{
	int n; cin &gt;&gt; n;
	for (int i = 1; i &lt;= n; i++)
	{
		cin &gt;&gt; nums[i];
	}
	int i = 1, j = n;
	while (i &lt; j)
	{
		swap(nums[i], nums[j]);
		i++; j--;
	}
	for (int i = 1; i &lt;= n; i++)
	{
		cout &lt;&lt; nums[i] &lt;&lt; ' ';
	}

	return 0;
}</code></pre>
