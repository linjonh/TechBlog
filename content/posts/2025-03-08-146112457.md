    <p>
     在 HTML 中，超链接是通过
     <code>
      &lt;a&gt;
     </code>
     标签（anchor tag）创建的。超链接允许用户通过点击文本、图像或其他元素跳转到另一个网页、文件或页面的特定部分。本文将详细介绍 HTML 超链接的语法、属性和应用场景。
    </p>
    <h3>
     <a id="_4">
     </a>
     一、基本语法
    </h3>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>URL<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>链接文本<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <ul>
     <li>
      <strong>
       <code>
        &lt;a&gt;
       </code>
      </strong>
      ：表示超链接的开始。
     </li>
     <li>
      <strong>
       <code>
        href
       </code>
      </strong>
      ：属性用于指定链接的目标 URL。
     </li>
     <li>
      <strong>
       <code>
        URL
       </code>
      </strong>
      ：可以是绝对 URL（完整地址）或相对 URL（相对于当前页面的地址）。
     </li>
     <li>
      <strong>
       <code>
        链接文本
       </code>
      </strong>
      ：用户在浏览器中看到的可点击部分。
     </li>
    </ul>
    <h3>
     <a id="_15">
     </a>
     二、超链接的类型
    </h3>
    <h4>
     <a id="1__17">
     </a>
     1. 外部链接
    </h4>
    <p>
     外部链接用于链接到其他网站或资源。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>访问示例网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="2__25">
     </a>
     2. 内部链接
    </h4>
    <p>
     内部链接用于在同一个网站的不同页面之间跳转。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>/about.html<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>关于我们<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="3__33">
     </a>
     3. 锚点链接
    </h4>
    <p>
     锚点链接用于在页面内跳转到特定的位置。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>#section1<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>跳转到第一部分<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>

     三、常用属性
    </h3>
    <h4>
     <a id="1_href_46">
     </a>
     1.
     <code>
      href
     </code>
    </h4>
    <p>
     <code>
      href
     </code>
     属性用于指定链接的目标 URL。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>访问示例网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="2_target_54">
     </a>
     2.
     <code>
      target
     </code>
    </h4>
    <p>
     <code>
      target
     </code>
     属性用于指定链接在何处打开。
    </p>
    <ul>
     <li>
      <strong>
       <code>
        _self
       </code>
      </strong>
      ：在当前窗口中打开链接（默认行为）。
     </li>
     <li>
      <strong>
       <code>
        _blank
       </code>
      </strong>
      ：在新窗口中打开链接。
     </li>
     <li>
      <strong>
       <code>
        _parent
       </code>
      </strong>
      ：在父框架中打开链接。
     </li>
     <li>
      <strong>
       <code>
        _top
       </code>
      </strong>
      ：在整个窗口中打开链接。
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com<span class="token punctuation">"</span></span> <span class="token attr-name">target</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>_blank<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>在新窗口中打开<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>

     </a>
     3.
     <code>
      download
     </code>
    </h4>
    <p>
     <code>
      download
     </code>
     属性用于指定下载链接。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>/path/to/file.pdf<span class="token punctuation">"</span></span> <span class="token attr-name">download</span><span class="token punctuation">&gt;</span></span>下载文件<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>

     4.
     <code>
      title
     </code>
    </h4>
    <p>
     <code>
      title
     </code>
     属性为链接提供额外的信息，通常作为工具提示显示。
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com<span class="token punctuation">"</span></span> <span class="token attr-name">title</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>这是一个示例网站<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>访问示例网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <h4>
     <a id="5_rel_83">
     </a>
     5.
     <code>
      rel
     </code>
    </h4>
    <p>
     <code>
      rel
     </code>
     属性用于指定链接与当前文档的关系。
    </p>
    <ul>
     <li>
      <strong>
       <code>
        nofollow
       </code>
      </strong>
      ：告诉搜索引擎不要跟随此链接。
     </li>
     <li>
      <strong>
       <code>
        noopener
       </code>
      </strong>
      ：防止新窗口中的页面与原始页面建立关系。
     </li>
     <li>
      <strong>
       <code>
        noreferrer
       </code>
      </strong>
      ：防止新窗口中的页面访问原始页面的引用信息。
     </li>
    </ul>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>a</span> <span class="token attr-name">href</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>https://www.example.com<span class="token punctuation">"</span></span> <span class="token attr-name">rel</span><span class="token attr-value"><span class="token punctuation attr-equals">=</span><span class="token punctuation">"</span>nofollow<span class="token punctuation">"</span></span><span class="token punctuation">&gt;</span></span>访问示例网站<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>a</span><span class="token punctuation">&gt;</span></span>
