### 实际使用中出现的问题

1.获取服务端时间戳和时间

```
    void GetServerTime()
    {
        //BmobWindows bmobWindows = new BmobWindows();
		//需要实例化一个bmob（可继承）的对象才可以正常使用方法
        BmobUnity.inst.Timestamp((resp, exception) =>
        {
            if (exception != null)
            {
                print("请求失败, 失败原因为： " + exception.Message);
                return;
            }
            //返回服务器时间（单位：秒）
            print("返回时间戳为： " + resp.timestamp);
            print("返回格式化的日期为： " + resp.datetime);
        }
        );
    }

```
