    <h3>
     <a id="_0">
     </a>
     移动光标及选择文本
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       ←
      </code>
      /
      <code>
       →
      </code>
      ：以单词为单位移动游标
     </li>
     <li>
      <code>
       Home
      </code>
      /
      <code>
       End
      </code>
      ：光标移到行首/行位
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Home
      </code>
      /
      <code>
       End
      </code>
      ：光标移到文件首和文件尾
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       \
      </code>
      ：在匹配的分隔符之间跳转
     </li>
    </ul>
    <blockquote>
     <p>
      <code>
       配对的分隔符
      </code>
      是指分隔代码元素的字符，比如字符串的双引号、代码段的大括号 (
      <code>
       {}
      </code>
      ) 等。
     </p>
    </blockquote>
    <blockquote>
     <p>
      <code>
       文本选择
      </code>
      需要多加一个
      <code>
       Shift
      </code>
      键。
     </p>
    </blockquote>
    <h3>
     <a id="_10">
     </a>
     缩进代码
    </h3>
    <ul>
     <li>
      <code>
       Shift
      </code>
      +
      <code>
       Alt
      </code>
      +
      <code>
       F
      </code>
      ：对选中代码片段或整个文件格式化
     </li>
    </ul>
    <h3>
     <a id="_13">
     </a>
     注释和取消注释
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       /
      </code>
      ：切换
      <code>
       行注释
      </code>
      (
      <code>
       //
      </code>
      ，按一次注释，再按一次取消注释)
     </li>
     <li>
      <code>
       Shift
      </code>
      +
      <code>
       Alt
      </code>
      +
      <code>
       A
      </code>
      ：切换
      <code>
       块注释
      </code>
      (
      <code>
       /* */
      </code>
      )
     </li>
    </ul>
    <h3>
     <a id="_17">
     </a>
     撤销与重做
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Z
      </code>
      ：撤销 (Undo)
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Y
      </code>
      ：重做 (Redo)
     </li>
    </ul>
    <h3>
     <a id="_21">
     </a>
     分割窗口
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       \
      </code>
      ：将文件水平切割，当前文件被复制到右侧面板中
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       1
      </code>
      、
      <code>
       Ctrl
      </code>
      +
      <code>
       2
      </code>
      ：分别跳转到第 1 个面板和第 2 个面板
     </li>
    </ul>
    <h3>
     <a id="_25">
     </a>
     跳转
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       P
      </code>
      ：跳转到指定文件
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       G
      </code>
      ：跳转到当前文件的指定行
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       T
      </code>
      ：跳转到工作区中的符号
     </li>
    </ul>
    <h3>
     <a id="_30">
     </a>
     搜索与替换
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       F
      </code>
      ：在当前文件中搜素字符串。在搜索框中按
      <code>
       Enter
      </code>
      键跳转到下一个搜索项，按
      <code>
       Shift
      </code>
      +
      <code>
       Enter
      </code>
      键跳转到上一个搜索项。
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       H
      </code>
      ：在当前文件中替换。
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       F
      </code>
      ：在整个工作区中搜索字符串
     </li>
    </ul>
    <h3>
     <a id="_35">
     </a>
     框选
    </h3>
    <p>
     临时创建多个光标，并行地在多个光标处编辑文本。
    </p>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       Alt
      </code>
      + (方向键)：列 (框) 选择
     </li>
    </ul>
    <h3>
     <a id="_39">
     </a>
     编译与调试
    </h3>
    <p>
     使用 CMake 创建的工程：
    </p>
    <ul>
     <li>
      <code>
       Shift
      </code>
      +
      <code>
       F5
      </code>
      ：编译并执行
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       F5
      </code>
      ：调试
     </li>
    </ul>
    <h3>
     <a id="_44">
     </a>
     显示或隐藏资源管理器
    </h3>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       B
      </code>
     </li>
    </ul>
    <h3>
     <a id="_46">
     </a>
     书签
    </h3>
    <p>
     需要先安装
     <code>
      Bookmarks
     </code>
     插件：
    </p>
    <ul>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Alt
      </code>
      +
      <code>
       K
      </code>
      ：创建或取消书签
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Alt
      </code>
      +
      <code>
       L
      </code>
      ：跳转到下一个书签
     </li>
     <li>
      <code>
       Ctrl
      </code>
      +
      <code>
       Alt
      </code>
      +
      <code>
       P
      </code>
      ：跳转到上一个书签
     </li>
    </ul>
