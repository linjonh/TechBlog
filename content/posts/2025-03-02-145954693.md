    <h2>
     <a id="_0">
     </a>
     一、单片机概述
    </h2>
    <h3>
     <a id="_1">
     </a>
     单片机的组成及其特点
    </h3>
    <p>
     单片机是指在一块芯片上集成了
     <code>
      CPU
     </code>
     、
     <code>
      ROM
     </code>
     、
     <code>
      RAM
     </code>
     、
     <code>
      定时器/计数器
     </code>
     和
     <code>
      多种I/O接口电路
     </code>
     等，具有一定规模的微型计算机。
     <br/>
     <code>
      特点：
     </code>
     <br/>
     1、单片机的存储器以ROM、RAM严格分工。
     <br/>
     2、采用面向控制的指令系统。
     <br/>
     3、单片机的I/O口引脚通常是多功能的。
     <br/>
     4、功能齐全，功能扩展性强。
     <br/>
     5、单片机的应用是通用的。
     <br/>
     6、性价比高，电路板小，接插件少。
     <br/>
     7、体积尺寸小。
    </p>
    <h3>
     <a id="_12">
     </a>
     单片机的分类
    </h3>
    <p>
     <code>
      通用型/专用型
     </code>
     <br/>
     这是单片机适用范围、使用场合来区分的。
     <br/>
     <code>
      总线型/非总线型
     </code>
     <br/>
     这是按单片机是否提供并行总线来区分的。
     <br/>
     <code>
      工控型/家电型
     </code>
     <br/>
     这是按照单片机的应用领域来区分的。
     <br/>
     <code>
      8位、16位、32位
     </code>
     <br/>
     8位主要分为51系列及非51系列单片机。
    </p>
    <h3>
     <a id="_22">
     </a>
     单片机的应用
    </h3>
    <p>
     软件开发涉及的四部分内容：
     <code>
      C语言编译器
     </code>
     、
     <code>
      汇编器
     </code>
     、
     <code>
      调试器
     </code>
     、
     <code>
      烧录软件
     </code>
     。
    </p>
