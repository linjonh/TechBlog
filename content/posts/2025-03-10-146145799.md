    <h2>

    <h3>
     <a id="WebUI_38">
     </a>
     WebUI搭建
    </h3>
    <p>
     完成部署后，看着简陋的命令行背景，感觉还是不如chatgpt、Deepseek那些网页版体验好，于是找了一下如何部署一个Web UI。主要参考ollama-webui-lite项目，尝试如下：
    </p>
    <p>
     首先需要clone项目：
    </p>
    <pre><code>git clone https://github.com/ollama-webui/ollama-webui-lite.git
cd ollama-webui-lite
</code></pre>
    <p>
     然后运行
    </p>
    <pre><code>yarn
</code></pre>
    <p>
     结果我这里报错
    </p>
    <pre><code>zsh command not found yarn
</code></pre>
    <p>
     于是我使用brew来安装yarn，直接
     <code>
      brew install yarn
     </code>
     是无法成功的，经过搜索发现需要更新
     <code>
      brew update
     </code>
     ，然后顺利安装yarn。
    </p>
    <p>
     在运行之前，还需要确保本机的node&gt;=16，可以直接使用
     <br/>
     <code>
      brew install node
     </code>
     来安装。
    </p>
    <h3>
     <a id="_60">
     </a>
     启动界面
    </h3>
    <p>
     在项目目录下，运行
     <code>
      yarn dev
     </code>
     ，即可成功设置WebUI，点击
     <a href="http://localhost:3000/" rel="nofollow">
      http://localhost:3000/
     </a>
     即可打开本地的WebUI。
    </p>
