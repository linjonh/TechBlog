    <h6>
     <strong>
      全新的产品特性
     </strong>
    </h6>
    <p>
     LightningChart JS v.7.0引入了新的产品特性，以下是概述：
    </p>
    <h6>
     <strong>
      对内置交互的更改
     </strong>
    </h6>
    <p>
     最新版本的LightningChart JS，它具有全新的直观交互，可以实现无缝的开发体验：
    </p>
    <ul>
     <li>
      双击缩放匹配交互
     </li>
     <li>
      根据图表结构自动内置智能交互方案选择
     </li>
     <li>
      更好的触摸屏交互
     </li>
     <li>
      在滚动轴应用程序上更方便的交互
     </li>
    </ul>
    <h6>
     <strong>
      引入setUserInteractions
     </strong>
    </h6>
    <p>
     官方正在引入一个新的
     <a href="https://lightningchart.com/js-charts/docs/migration-guides/v7.0.0/#interactions-rework" rel="nofollow" title="API">
      API
     </a>
     ，允许更详细的交互操作配置：
    </p>
    <ul>
     <li>
      选择受交互作用影响的轴（X、Y或两者）
     </li>
     <li>
      启用基于Ctrl、Shift或Alt组合键的交互
     </li>
     <li>
      根据您的应用要求调整平移和变焦灵敏度
     </li>
     <li>
      探索新的交互，如分页、缩放还原、键盘控制和3D轴修改
     </li>
    </ul>
    <pre><code class="language-javascript">// Right mouse click on Y axis title

})
// dataPoint.x, dataPoint.y, dataPoint.id</code></pre>
    <h6>
     <strong>
      自定义交互的更改
     </strong>
    </h6>
    <ul>
     <li>
      <strong>
       简化的API
      </strong>
      - 现在遵循熟悉的addEventListener/removeEventListener语法，以便更容易实现。
     </li>
     <li>
      <strong>
       一致的跨设备交互
      </strong>
      - 以前的鼠标、触摸和拖动交互现在在所有设备上都是一致的。
     </li>
     <li>
      <strong>
       右键支持
      </strong>
      - 添加上下文菜单事件处理，来改进交互选项。
     </li>
     <li>
      <strong>
       扩展交互跟踪
      </strong>
      - 现在包括图表标题、轴标题、地图区域等。
     </li>
     <li>
      <strong>
       拖放支持
      </strong>
      - 支持在图表中移动元素或集成外部HTML元素，如交换轴或添加数据源。
     </li>
    </ul>
    <pre><code class="language-javascript">// Right mouse click on Y axis title

pointSeries.addEventListener('pointermove', (event, dataPoint) =&gt; {
})
// dataPoint.x, dataPoint.y, dataPoint.id</code></pre>
    <h6>
     <strong>
      文本系列
     </strong>
    </h6>
    <p>
    </p>
    <p class="img-center">
     <img alt="LightningChart JS 7.0产品图集" height="453" src="https://i-blog.csdnimg.cn/img_convert/87ba2507595becab012a3ff310f57372.jpeg" width="1251"/>
    </p>
    <p>
     在ChartXY中显示文本的一个新的、更有效的API提高了性能和可用性，它支持viewport剪切，并提供了调整文本大小和边界的方便方法，使其在许多情况下成为chart.addUIElement()的合适替代品。
    </p>
    <h6>
     <strong>
      ChartXY布局更改事件
     </strong>
    </h6>
    <p>
    </p>
    <p class="img-center">
     <img alt="LightningChart JS 7.0产品图集" height="420" src="https://i-blog.csdnimg.cn/img_convert/619d02fea700b17c6d862e345c172430.jpeg" width="1280"/>
    </p>
    <p>
     一个新的便利API提供了对ChartXY元素的结构、布局和位置的详细了解，它对于静态长宽比应用程序或将UI元素与图表组件（如轴）对齐特别有用。
    </p>
    <h6>
     <strong>
      提高视觉质量
     </strong>
    </h6>
    <p>
     <strong>
      UI元素中的圆角
     </strong>
     ——图例、光标、矩形系列、条形图和树状图现在都以圆角为特征，以实现更现代、更有视觉吸引力的设计。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="LightningChart JS 7.0产品图集" height="1101" src="https://i-blog.csdnimg.cn/img_convert/ced76dce1a3d38019b6e2dad7b9b2615.jpeg" width="1280"/>
    </p>
    <p>
     其他改进包括：
    </p>
    <ul>
     <li>
      <strong>
       改进的游标可读性
      </strong>
      - 为游标添加了单独的标题背景，使内置的游标信息更清晰，更易于阅读。
     </li>
     <li>
      <strong>
       光标颜色适应数据点
      </strong>
      - 当使用每个数据点的颜色时，光标现在将显示其悬停在特定数据点上的颜色，从而改善视觉环境。
     </li>
    </ul>
    <h6>
     <strong>
      轴刻度布局和显示改进
     </strong>
    </h6>
    <p>
    </p>
    <p class="img-center">
     <img alt="LightningChart JS 7.0产品图集" height="591" src="https://i-blog.csdnimg.cn/img_convert/7d8493e8a631d8872dbafa7ab4bb373a.jpeg" width="1571"/>
    </p>
    <ul>
     <li>
      <strong>
       更少的刻度显示清晰
      </strong>
      - 刻度的数量已经减少，来尽量减少视觉混乱，同时保持可读性。
     </li>
     <li>
      <strong>
       改进的小轴刻度显示
      </strong>
      - 以前非常小的轴有时只显示一个不可读的刻度，现在系统优化tick placement来获得更好的可见性。
     </li>
     <li>
      <strong>
       刻度标签保持在图表边界内
      </strong>
      - 默认情况下，刻度标签现在被限制在图表容器内，以防止它们在紧凑的图表布局中被切断。
     </li>
     <li>
      <strong>
       简化的刻度标签颜色
      </strong>
      - 内置主题现在使用单一的统一颜色的刻度标签，而不是单独的暗淡/明亮的变化，创造一个更干净的外观。
     </li>
    </ul>
    <h6>
     <strong>
      改进条形图标签拟合和添加文本阴影
     </strong>
    </h6>
    <p>
    </p>
    <p class="img-center">
     <img alt="LightningChart JS 7.0产品图集" height="688" src="https://i-blog.csdnimg.cn/img_convert/6b7a3fefea629bbe7f321d1b22ecfd1f.jpeg" width="1477"/>
    </p>
    <ul>
     <li>
      <strong>
       改进的标签拟合
      </strong>
      - 以前的算法过于保守，限制了显示标签的数量；现在可以显示更多的标签而不会造成重叠。
     </li>
     <li>
      <strong>
       增强的文本可读性
      </strong>
      - 微妙的文本阴影已添加到各种图表类型，使标签更清晰，更容易阅读。
     </li>
    </ul>
