    <h4>
     1. ​
     <code>
      std::list
     </code>
     基本概念
    </h4>
    <ul>
     <li>
      ​
      <strong>
       定义
      </strong>
      ：
      <code>
       std::list
      </code>
      是 C++ 标准库提供的带头（哨兵位）双向循环链表容器，支持高效的元素插入和删除。
     </li>
     <li>
      ​
      <strong>
       头文件
      </strong>
      ：
      <code>
       #include &lt;list&gt;
      </code>
     </li>
    </ul>
    <hr/>
    <h4>
     2. ​
     <strong>
      构造函数
     </strong>
    </h4>
    <h5>
     (1) ​
     <strong>
      默认构造函数
     </strong>
    </h5>
    <pre><code class="language-cpp">list&lt;int&gt; list1;  // 创建一个空list，size=0</code></pre>
    <h5>
     (2) ​
     <strong>
      指定初始大小和默认值
     </strong>
    </h5>
    <pre><code class="language-cpp">list&lt;int&gt; list2(5);       // 5个元素，默认初始化（int为0）
list&lt;double&gt; list3(5, 3.14);  // 5个元素，每个值为3.14</code></pre>
    <h5>
     (3) ​
     <strong>
      通过迭代器范围构造
     </strong>
    </h5>
    <pre><code class="language-cpp">int arr[] = {1, 2, 3};
list&lt;int&gt; list4(arr, arr + 3);  // 复制数组的[arr, arr+3)范围元素</code></pre>
    <h5>
     (4) ​
     <strong>
      拷贝构造函数
     </strong>
    </h5>
    <pre><code class="language-cpp">list&lt;int&gt; list5(list4);  // 深拷贝list4的内容</code></pre>
    <hr/>
    <h4>
     3. ​
     <strong>
      修改元素的成员函数
     </strong>
    </h4>
    <h5>
     (1) ​
     <code>
      push_back()
     </code>
     和
     <code>
      push_front()
     </code>
    </h5>
    <p>
     在末尾或头部添加元素：
    </p>
    <pre><code class="language-cpp">list&lt;int&gt; lst;
lst.push_back(10);   // 末尾添加10 → {10}
lst.push_front(5);   // 头部添加5 → {5, 10}</code></pre>
    <h5>
     (2) ​
     <code>
      insert()
     </code>
    </h5>
    <p>
     插入元素到指定位置，支持以下重载：
    </p>
    <ul>
     <li>
      ​
      <strong>
       插入单个值
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst = {1, 3};
list&lt;int&gt;::iterator it = lst.begin();
++it;
lst.insert(it, 2);  // 在位置1插入2 → {1, 2, 3}</code></pre>
     </li>
     <li>
      ​
      <strong>
       插入多个相同值
      </strong>
      ：
      <pre><code class="language-cpp">lst.insert(lst.end(), 2, 4);  // 在末尾插入2个4 → {1, 2, 3, 4, 4}</code></pre>
     </li>
     <li>
      ​
      <strong>
       通过迭代器范围插入
      </strong>
      ：
      <pre><code class="language-cpp">int arr[] = {5, 6};
lst.insert(lst.end(), arr, arr + 2);  // 插入数组内容 → {1, 2, 3, 4, 4, 5, 6}</code></pre>
     </li>
    </ul>
    <h5>
     (3) ​
     <code>
      erase()
     </code>
    </h5>
    <p>
     删除指定位置或范围的元素：
    </p>
    <ul>
     <li>
      ​
      <strong>
       删除单个元素
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt;::iterator it = lst.begin();
++it;
lst.erase(it);  // 删除位置1的元素 → {1, 3, 4, 4, 5, 6}</code></pre>
     </li>
     <li>
      ​
      <strong>
       删除范围元素
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt;::iterator first = lst.begin();
advance(first, 3);  // 移动到位置3
lst.erase(first, lst.end());  // 删除位置3到末尾 → {1, 3, 4}</code></pre>
     </li>
    </ul>
    <h5>
     (4) ​
     <code>
      clear()
     </code>
    </h5>
    <p>
     清空所有元素：
    </p>
    <pre><code class="language-cpp">lst.clear();  // size=0</code></pre>
    <h5>
     (5) ​
     <code>
      assign()
     </code>
    </h5>
    <p>
     重新分配元素（覆盖原有内容）：
    </p>
    <ul>
     <li>
      ​
      <strong>
       指定数量和值
      </strong>
      ：
      <pre><code class="language-cpp">lst.assign(3, 10);  // 3个10 → {10, 10, 10}</code></pre>
     </li>
     <li>
      ​
      <strong>
       通过迭代器范围
      </strong>
      ：
      <pre><code class="language-cpp">int arr[] = {20, 30, 40};
lst.assign(arr, arr + 2);  // 复制前2个元素 → {20, 30}</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     4. ​
     <strong>
      访问元素的成员函数
     </strong>
    </h4>
    <h5>
     (1) ​
     <code>
      front()
     </code>
     和
     <code>
      back()
     </code>
    </h5>
    <p>
     访问首尾元素：
    </p>
    <pre><code class="language-cpp">int first = lst.front();  // 第一个元素
int last = lst.back();    // 最后一个元素</code></pre>
    <hr/>
    <h4>
     5. ​
     <strong>
      容量管理的成员函数
     </strong>
    </h4>
    <h5>
     (1) ​
     <code>
      size()
     </code>
    </h5>
    <p>
     获取元素数量：
    </p>
    <pre><code class="language-cpp">size_t s = lst.size();  // 当前元素数量</code></pre>
    <h5>
     (2) ​
     <code>
      resize()
     </code>
    </h5>
    <p>
     调整元素数量：
    </p>
    <pre><code class="language-cpp">lst.resize(5);      // 调整size=5，多出的元素默认初始化（int为0）
lst.resize(3, 100); // 调整size=3，若需要新增元素则初始化为100</code></pre>
    <hr/>
    <h4>
     6. ​
     <strong>
      链表特有操作
     </strong>
    </h4>
    <h5>
     (1) ​
     <code>
      splice()
     </code>
    </h5>
    <p>
     将另一个链表的元素移动（拼接）到当前链表，支持以下重载：
    </p>
    <ul>
     <li>
      ​
      <strong>
       移动整个链表
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst1 = {1, 2};
list&lt;int&gt; lst2 = {3, 4};
lst1.splice(lst1.end(), lst2);  // lst1 → {1, 2, 3, 4}, lst2变为空</code></pre>
     </li>
     <li>
      ​
      <strong>
       移动单个元素
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst3 = {5, 6};
list&lt;int&gt;::iterator it = lst3.begin();
lst1.splice(lst1.begin(), lst3, it);  // lst1 → {5, 1, 2, 3, 4}, lst3 → {6}</code></pre>
     </li>
     <li>
      ​
      <strong>
       移动范围元素
      </strong>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst4 = {7, 8, 9};
list&lt;int&gt;::iterator first = lst4.begin();
list&lt;int&gt;::iterator last = lst4.end();
--last;
lst1.splice(lst1.end(), lst4, first, last);  // lst1 → {5, 1, 2, 3, 4, 7, 8}, lst4 → {9}</code></pre>
     </li>
    </ul>
    <h5>
     (2) ​
     <code>
      remove()
     </code>
    </h5>
    <p>
     删除特定元素：
    </p>
    <ul>
     <li>
      ​
      <code>
       remove()
      </code>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst = {1, 2, 3, 2, 4};
lst.remove(2);  // 删除所有值为2的元素 → {1, 3, 4}</code></pre>
     </li>
    </ul>
    <h5>
     (3) ​
     <code>
      sort()
     </code>
     和
     <code>
      merge()
     </code>
    </h5>
    <p>
     排序和合并链表：
    </p>
    <ul>
     <li>
      ​
      <code>
       sort()
      </code>
      ：
      <pre><code class="language-cpp">list&lt;int&gt; lst = {3, 1, 4, 2};
lst.sort();  // 默认升序 → {1, 2, 3, 4}</code></pre>
     </li>
     <li>
      ​
      <code>
       merge()
      </code>
      （要求两个链表已排序）：
      <pre><code class="language-cpp">list&lt;int&gt; lst1 = {1, 3};
list&lt;int&gt; lst2 = {2, 4};
lst1.merge(lst2);  // lst1 → {1, 2, 3, 4}, lst2变为空</code></pre>
     </li>
    </ul>
    <h5>
     (4) ​
     <code>
      unique()
     </code>
    </h5>
    <p>
     删除连续重复元素（需先排序）：
    </p>
    <pre><code class="language-cpp">list&lt;int&gt; lst = {1, 1, 2, 3, 3};
lst.unique();  // 删除连续重复 → {1, 2, 3}</code></pre>
    <h5>
     (5) ​
     <code>
      reverse()
     </code>
    </h5>
    <p>
     反转链表元素顺序：
    </p>
    <pre><code class="language-cpp">list&lt;int&gt; lst = {1, 2, 3};
lst.reverse();  // → {3, 2, 1}</code></pre>
    <hr/>
    <h4>
     7. ​
     <strong>
      迭代器操作
     </strong>
    </h4>
    <ul>
     <li>
      ​
      <code>
       begin()
      </code>
      和
      <code>
       end()
      </code>
      ：
      <pre><code class="language-cpp">for (list&lt;int&gt;::iterator it = lst.begin(); it != lst.end(); ++it) {
    cout &lt;&lt; *it &lt;&lt; " ";
}</code></pre>
     </li>
     <li>
      ​
      <code>
       rbegin()
      </code>
      和
      <code>
       rend()
      </code>
      （反向迭代器）​：
      <pre><code class="language-cpp">for (list&lt;int&gt;::reverse_iterator rit = lst.rbegin(); rit != lst.rend(); ++rit) {
    cout &lt;&lt; *rit &lt;&lt; " ";
}</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     8. ​
     <strong>
      注意事项
     </strong>
    </h4>
    <ul>
     <li>
      ​
      <strong>
       迭代器失效
      </strong>
      ：删除操作仅使被删除元素的迭代器失效，其他迭代器仍然有效（后面我会专门整理这个问题）。
     </li>
     <li>
      ​
      <strong>
       性能特点
      </strong>
      ：插入和删除操作高效，但随机访问效率低（需遍历链表）。
     </li>
     <li>
      ​
      <strong>
       排序与合并
      </strong>
      ：
      <code>
       sort()
      </code>
      和
      <code>
       merge()
      </code>
      是成员函数，不同于
      <code>
       &lt;algorithm&gt;
      </code>
      中的全局函数。
     </li>
    </ul>
