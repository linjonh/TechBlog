    <h3 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       一、JSP 内置对象：开发者的“工具箱”
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0">
     JSP 内置对象[1]，说白了就是 JSP 提供给开发者的“工具箱”，里头装着各种实用的工具，比如
     <strong>
      <strong>
       request
      </strong>
     </strong>
     、
     <strong>
      <strong>
       response
      </strong>
     </strong>
     、
     <strong>
      <strong>
       session
      </strong>
     </strong>
     这些。这些内置对象不用声明，拿来就能用，简直是“懒人福音”。
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
    </h4>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       1. request：请求的“搬运工”
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     <strong>
      <strong>
       request
      </strong>
     </strong>
     对象是 JSP 里最常用的内置对象之一，它负责“搬运” HTTP 请求的数据。无论是表单数据、URL 参数，还是请求头信息，
     <strong>
      <strong>
       request
      </strong>
     </strong>
     都能“一网打尽”。
    </p>

     <tbody>

         <span style="color:#ffffff">
          &lt;%
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          String name = request.getParameter("name");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          out.println("你好，" + name);
         </span>
        </p>

     <strong>
      <strong>
       request
      </strong>
     </strong>
     对象的主要方法有：
    </p>
    <ul>
     <li>
      <strong>
       <strong>
        getParameter(String name)
       </strong>
      </strong>
      ：获取请求参数的值。
     </li>
     <li>
      <strong>
       <strong>
        getAttribute(String name)
       </strong>
      </strong>
      ：获取请求属性的值。
     </li>
     <li>
      <strong>
       <strong>
        setAttribute(String name, Object value)
       </strong>
      </strong>
      ：设置请求属性的值。
     </li>
    </ul>

     <strong>
      <strong>
       response
      </strong>
     </strong>

     <strong>
      <strong>
       response
      </strong>
     </strong>
     都能“搞定”。
    </p>

    </p>

      </strong>

     <li>
      <strong>
       <strong>
        sendRedirect(String location)
       </strong>
      </strong>
      ：重定向到指定的 URL。

    <h3 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       三、JSP 内置对象的“历史”：从 JSP 1.0 到 JSP 2.3
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0">
     JSP 内置对象从 JSP 1.0 开始就有了，虽然现在 JSP 已经有点“过气”了，但它的内置对象依然是开发者的“好帮手”。
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       1. JSP 1.0：初代“工具箱”
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     JSP 1.0 在 1999 年发布，内置对象是它的“初代工具箱”。虽然功能简单，但已经能满足大部分开发需求了。
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       2. JSP 2.0：引入 EL 表达式
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     JSP 2.0 在 2003 年发布，引入了 EL 表达式[2]，让开发者能更方便地访问内置对象。EL 表达式的引入，让 JSP 的“玩法”更加丰富。
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       3. JSP 2.3：最后的“辉煌”
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     JSP 2.3 在 2013 年发布，是 JSP 的“最后版本”。虽然内置对象的功能已经很强大了，但 JSP 的“辉煌”已经过去了，因为大家都开始用 Spring Boot、Vue.js 这些新技术了。
    </p>
    <p style="margin-left:.0001pt; margin-right:0">
    </p>
    <h3 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       四、JSP 内置对象的“骚操作”
      </strong>
     </strong>
    </h3>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       1. request 的“骚操作”
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     <strong>
      <strong>
       request
      </strong>
     </strong>
     对象不仅能“搬运”数据，还能“搬运”文件。比如，用户上传了一个文件，
     <strong>
      <strong>
       request
      </strong>
     </strong>
     就能把文件“搬”到服务器上。
    </p>
    <table border="1" cellspacing="0">
     <tbody>
      <tr>
       <td style="background-color:#0a3041; width:522.8000pt">
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          Jsp Code
         </span>
        </p>
       </td>
      </tr>
      <tr>
       <td style="background-color:#0a3041; width:522.8000pt">
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          &lt;%
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          Part filePart = request.getPart("file");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          String fileName = filePart.getSubmittedFileName();
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          filePart.write("/uploads/" + fileName);
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          out.println("文件上传成功！");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          %&gt;
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
        </p>
       </td>
      </tr>
     </tbody>
    </table>
    <p style="margin-left:.0001pt; margin-right:0">
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       2. response 的“骚操作”
      </strong>
     </strong>
    </h4>
    <p style="margin-left:.0001pt; margin-right:0">
     <strong>
      <strong>
       response
      </strong>
     </strong>
     对象不仅能“快递”数据，还能“快递”文件。比如，用户下载了一个文件，
     <strong>
      <strong>
       response
      </strong>
     </strong>
     就能把文件“快递”给用户。
    </p>
    <table border="1" cellspacing="0">
     <tbody>
      <tr>
       <td style="background-color:#0a3041; width:522.8000pt">
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          Jsp Code
         </span>
        </p>
       </td>
      </tr>
      <tr>
       <td style="background-color:#0a3041; width:522.8000pt">
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          &lt;%
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          response.setContentType("application/octet-stream");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          response.setHeader("Content-Disposition", "attachment; filename=\"file.txt\"");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          InputStream in = new FileInputStream("/path/to/file.txt");
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          OutputStream out = response.getOutputStream();
         </span>

        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          byte[] buffer = new byte[4096];
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          int length;
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          while ((length = in.read(buffer)) &gt; 0) {
          <!-- -->
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          out.write(buffer, 0, length);
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          }
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          in.close();
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          out.flush();
         </span>
        </p>
        <p style="margin-left:.0001pt; margin-right:0">
         <span style="color:#ffffff">
          %&gt;
         </span>
        </p>

    <h4 style="margin-left:0.0001pt; margin-right:0px">
     <strong>
      <strong>
       3. session 的“骚操作”
      </strong>

    <p style="margin-left:.0001pt; margin-right:0">
