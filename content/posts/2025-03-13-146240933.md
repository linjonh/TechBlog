    <h2>
     二、核心概念与术语
    </h2>
    <h5>
     1.
     <strong>
      基础术语
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       DB（Database）
      </strong>
      ：存储数据的集合（如：用户表、订单表）。
     </li>
     <li>
      <strong>
       DBMS（Database Management System）
      </strong>
      ：管理数据库的软件（如：MySQL、Oracle）。
     </li>
     <li>
      <strong>
       MIS（Management Information System）
      </strong>
      ：集成数据库的管理信息系统（如：ERP系统）。
     </li>
     <li>
      <strong>
       OA（Office Automation）
      </strong>
      ：基于数据库的办公自动化系统（如：审批流程系统）。
     </li>
    </ul>
    <h5>
     2.
     <strong>
      数据库结构层次
     </strong>
    </h5>
    <pre><code>数据库（DB）
├── 表（Table）
│   ├── 字段（Field，最小单位，如：姓名、年龄）
│   └── 记录（Record，由字段组成的一行数据）
└── 其他表...
</code></pre>
    <ul>
     <li>
      <strong>
       字段
      </strong>
      ：表的列，定义数据类型（如：
      <code>
       VARCHAR
      </code>
      、
      <code>
       INT
      </code>
      ）。
     </li>
     <li>
      <strong>
       记录
      </strong>
      ：表的行，表示一条完整数据。
     </li>
    </ul>

    <h2>
     五、数据表操作 (DDL)
    </h2>
    <h5>
     1. 创建表
    </h5>
    <pre><code>-- 基础表结构
CREATE TABLE users (
    id INTEGER PRIMARY KEY,   -- 主键约束
    name TEXT NOT NULL,       -- 非空约束
    age INTEGER,
    reg_date DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 带自增主键
CREATE TABLE user3 (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name CHAR(20),
    age INT
);
</code></pre>
    <p>
     <em>
      SQLite 是弱类型数据库，字段类型仅为建议类型
     </em>
    </p>
    <h5>
     2. 删除表
    </h5>
    <pre><code>DROP TABLE users;  -- 慎用！会删除表结构和所有数据
</code></pre>
    <hr/>
    <h2>
     六、数据操作 (DML)
    </h2>
    <h5>
     1. 插入数据
    </h5>
    <pre><code>-- 全字段插入
INSERT INTO users VALUES (1, '张三', 25, '2023-08-20 10:00:00');

-- 指定字段插入
INSERT INTO users (name, age) VALUES ('李四', 30);

-- 插入当前时间
INSERT INTO user1 VALUES (2, '王五', 28, datetime('now','+8 hours'));
</code></pre>
    <h5>
     2. 更新数据
    </h5>
    <pre><code>-- 修改指定记录
UPDATE users SET age = 26 WHERE name = '张三';

-- 多条件更新
UPDATE users SET age = 27 WHERE name = '李四' AND id = 2;
</code></pre>
    <h5>
     3. 删除数据
    </h5>
    <pre><code>-- 清空表 (保留结构)
DELETE FROM users;

-- 条件删除
DELETE FROM users WHERE age &lt; 18;
DELETE FROM users WHERE id = 3 OR name = 'test';
</code></pre>

     七、数据查询 (DQL)
    </h2>
    <h5>

    </h5>

-- 查询指定字段

</code></pre>

    <h2>
     十、SQLite C 语言接口编程
    </h2>
    <hr/>
    <h4>
     （一）核心操作流程
    </h4>
    <pre><code>graph TD
    A[打开数据库 sqlite3_open] --&gt; B[执行操作]
    B --&gt; C{操作类型}
    C --&gt;|查询| D[使用 sqlite3_get_table]
    C --&gt;|增删改| E[使用 sqlite3_exec]
    B --&gt; F[关闭数据库 sqlite3_close]
</code></pre>
    <hr/>
    <h4>
     （二）核心函数详解
    </h4>
    <h5>
     1.
     <strong>
      打开数据库
      <code>
       sqlite3_open
      </code>
     </strong>
    </h5>
    <pre><code>int sqlite3_open(const char *filename, sqlite3 **ppDb);
</code></pre>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         filename
        </code>
       </td>
       <td>
        数据库文件路径 (若不存在则自动创建)
       </td>
      </tr>
      <tr>
       <td>
        <code>
         ppDb
        </code>
       </td>
       <td>
        输出参数，用于接收数据库句柄指针
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      返回值
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       SQLITE_OK
      </code>
      (0)：成功
     </li>
     <li>
      其他值：错误代码 (需用
      <code>
       sqlite3_errmsg()
      </code>
      获取错误信息)
     </li>
    </ul>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code>sqlite3 *db = NULL;
int rc = sqlite3_open("test.db", &amp;db);
if (rc != SQLITE_OK) {
    fprintf(stderr, "无法打开数据库: %s\n", sqlite3_errmsg(db));
    exit(1);
}
</code></pre>
    <hr/>
    <h5>
     2.
     <strong>
      执行查询操作
      <code>
       sqlite3_get_table
      </code>
     </strong>
    </h5>
    <pre><code>int sqlite3_get_table(
    sqlite3 *db,          // 数据库句柄
    const char *zSql,     // SQL 语句
    char ***pazResult,    // 结果集指针
    int *pnRow,           // 返回行数
    int *pnColumn,        // 返回列数
    char **pzErrmsg       // 错误信息指针
);
</code></pre>
    <p>
     <strong>
      结果集结构
     </strong>
     ：
    </p>
    <pre><code>// 假设查询返回 2 列 (name, age) 和 3 行数据
pazResult[0] = "name"
pazResult[1] = "age"
pazResult[2] = "Alice"
pazResult[3] = "25"
pazResult[4] = "Bob"
pazResult[5] = "30"
</code></pre>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code>char **result;
int rows, cols;
char *errmsg = NULL;

rc = sqlite3_get_table(db, "SELECT name, age FROM users", 
                      &amp;result, &amp;rows, &amp;cols, &amp;errmsg);

if (rc == SQLITE_OK) {
    // 处理结果集
    for (int i = 0; i &lt;= rows; i++) { // 第0行是列名
        for (int j = 0; j &lt; cols; j++) {
            printf("%s\t", result[i*cols + j]);
        }
        printf("\n");
    }
    sqlite3_free_table(result); // 必须释放内存！
} else {
    fprintf(stderr, "SQL错误: %s\n", errmsg);
    sqlite3_free(errmsg);
}
</code></pre>
    <hr/>
    <h5>
     3.
     <strong>
      执行非查询操作
      <code>
       sqlite3_exec
      </code>
     </strong>
    </h5>
    <pre><code>int sqlite3_exec(
    sqlite3 *db,               // 数据库句柄
    const char *sql,           // SQL 语句
    int (*callback)(void*,int,char**,char**), // 回调函数
    void *arg,                 // 回调函数参数
    char **errmsg              // 错误信息指针
);
</code></pre>
    <p>
     <strong>
      回调函数原型
     </strong>
     ：
    </p>
    <pre><code>int callback(
    void *arg,        // 用户传递的参数
    int num_cols,     // 结果列数
    char **col_values, // 当前行的列值
    char **col_names   // 列名数组
);
</code></pre>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code>// 插入数据
rc = sqlite3_exec(db, "INSERT INTO users VALUES(3, 'Charlie', 28)", 
                  NULL, NULL, &amp;errmsg);

// 带回调的查询
int print_callback(void *arg, int cols, char **values, char **names) {
    for (int i = 0; i &lt; cols; i++) {
        printf("%s: %s\t", names[i], values[i]);
    }
    printf("\n");
    return 0; // 必须返回0
}

rc = sqlite3_exec(db, "SELECT * FROM users", 
                 print_callback, NULL, &amp;errmsg);
</code></pre>
    <hr/>
    <h5>
     4.
     <strong>
      关闭数据库
      <code>
       sqlite3_close
      </code>
     </strong>
    </h5>
    <pre><code>int sqlite3_close(sqlite3 *db);
</code></pre>
    <p>
     <strong>
      注意事项
     </strong>
     ：
    </p>
    <ul>
     <li>
      关闭前必须完成所有事务
     </li>
     <li>

      <code>
       SQLITE_BUSY
      </code>
      表示仍有未完成的查询
     </li>
    </ul>
    <hr/>
    <h4>
     （三）内存管理与错误处理
    </h4>
    <h5>
     1.
     <strong>
      释放资源
     </strong>
    </h5>
    <pre><code>sqlite3_free_table(char **result);  // 释放 sqlite3_get_table 的结果
sqlite3_free(void *ptr);           // 释放错误信息内存
</code></pre>
    <h5>
     2.
     <strong>
      错误信息获取
     </strong>
    </h5>
    <pre><code>const char *errmsg = sqlite3_errmsg(db);
printf("错误信息: %s\n", errmsg);</code></pre>
    <p>
    </p>
    <h2>
     实例：
    </h2>
    <p>
     程序用于将一个文本格式的字典文件（每行包含一个单词及其解释）转换为 SQLite 数据库格式，便于后续的查询和管理。
    </p>
    <pre><code class="language-bash">#include &lt;stdio.h&gt;
#include &lt;sqlite3.h&gt;
#include &lt;string.h&gt;
#include &lt;unistd.h&gt;

// 主函数，参数为字典文本文件和目标数据库文件
