    <p>
     #设计模式 #Qt
    </p>
    <p>
     在原生 Qt 框架的设计中，原型模式（Prototype Pattern）通过对象克隆机制实现高效的对象创建与复制，典型体现在隐式共享（Copy-On-Write）和深拷贝设计中。以下是 Qt 中原型模式的具体应用场景：
    </p>
    <h4>
     <a id="1%C2%A0QImage%C2%A0_5">
     </a>
     1.
     <strong>
      <code>
       QImage
      </code>
      的隐式共享与深拷贝
     </strong>
    </h4>
    <h5>
     <a id="_7">
     </a>
     应用方式
    </h5>
    <ul>
     <li>
      <strong>
       抽象原型类
      </strong>
      ：
      <code>
       QImage
      </code>
      作为基础图像类，提供
      <code>
       copy()
      </code>
      方法实现深拷贝。
     </li>
     <li>
      <strong>
       隐式共享机制
      </strong>
      ：默认拷贝操作（如赋值
      <code>
       =
      </code>
      ）为浅拷贝，共享同一份像素数据；修改时触发深拷贝
      <code>
       detach()
      </code>
      。
     </li>
    </ul>

    <h4>
     <a id="2%C2%A0QString%C2%A0_44">
     </a>
     2.
     <strong>
      <code>
       QString
      </code>
      的隐式共享
     </strong>
    </h4>
    <h5>
     <a id="_46">
     </a>
     应用方式
    </h5>
    <ul>
     <li>
      <strong>
       抽象原型类
      </strong>
      ：
      <code>
       QString
      </code>

      <strong>

      ：字符串数据在未修改时共享，修改时深拷贝。

     <a id="3%C2%A0QBrush%C2%A0QPen_81">

     3.
     <strong>

      <code>
       QBrush
      </code>
      ,
      <code>
       QPen
      </code>
      ）
     </strong>
    </h4>
    <h5>
     <a id="_83">
     </a>
     应用方式
    </h5>
    <ul>
     <li>
      <strong>
       抽象原型类

      ：
      <code>
       QBrush

      <code>
       QPen
      </code>
      等图形类默认支持隐式共享。
     </li>
     <li>
      <strong>
       隐式共享
      </strong>
      ：默认拷贝操作共享资源，修改时分离数据。
     </li>
    </ul>

    <h4>
     <a id="4%C2%A0QList%C2%A0QVector_109">
     </a>
     4.
     <strong>
      容器类的隐式共享（
      <code>
       QList
      </code>
      ,
      <code>
       QVector
      </code>
      ）
     </strong>
    </h4>
    <h5>
     <a id="_111">

     应用方式
    </h5>
    <ul>
     <li>
      <strong>

       QList

    <h4>
     <a id="5%C2%A0QSharedData_137">
     </a>
     5.
     <strong>
      自定义对象的深拷贝（
      <code>
       QSharedData
      </code>
      ）
     </strong>
    </h4>
    <h5>
     <a id="_139">
     </a>
     应用方式
    </h5>
    <ul>

       QSharedData
