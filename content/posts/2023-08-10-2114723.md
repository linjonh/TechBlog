# 微软公布四大战略转变 对技术及商业活动做重大调整

微软周四宣布，为了更好的和包括开源社区在内的其它软件提供商合作，微软将改变其商业惯例。
  
  
微软公布了四条旨在提高批量企业用户软件支持标准及更好处理来自非微软软件数据的原则。
  
  
微软明确表示，将在其“批量产品”中公布“应用程序接口”（API）以及通讯协议。开发者无需购买授权许可或者支付版税即可访问这些信息。
  
  
微软将首次公布与Windows Server通讯的协议，原先，仅有获得商业秘密授权许可的人可能获得这些协议。微软表示，与Office 2007互通的协议将在未来几个月内公布。
  
  
微软透露，这一政策最终将延伸至Windows Vista，.NET Framework，Windows Server 2008，SQL Server 2008，Office 2007，Exchange Server 2007，以及Office SharePoint Server 2007。
  
  
微软表示，不会起诉那些在微软协议基础上开发非商业软件的开发人员。
  
  
微软还表示，将提供新的“应用程序接口”给开发人员，以便Office 2007能够与其它文件格式兼容。微软Office 2007目前不支持ODF文件格式。
  
  
虽然不支持ODF格式，但微软提出了“文件互通计划”，以便在范围广阔的文件格式之间实现数据交换。
  
  
微软表示，“开源互通计划”将加快形成开源项目与微软之间的良好工作关系，并为其提供技术协助，象互通性测试等。
  
  
尽管以前创办了“互通性主管客户委员会”，但微软将寻求建立一个与合作伙伴，客户，开发者更好的对话与沟通渠道。
  
  
在一份声明中，微软总经理Brad Smith表示，这些措施是微软满足欧盟一审法院提出的反垄断要求的组成部分。
  
  
Smith说：“在去年9月欧盟一审法院做出决定后，微软就表示，将采取所有必要的措施确保我们符合欧洲法律的规定。”
