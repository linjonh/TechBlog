     3.源码下载(系统环境：WSL2、ubuntu 20.04)
    </h3>
    <h4>
     <a id="31_pythonpip_16">
     </a>
     3.1 安装python、pip工具
    </h4>
    <pre><code>sudo apt -y upgrade
ln -s /usr/bin/python3 /usr/bin/python // 建立一个软链接
sudo apt install python3-pip  // 注意安装完成后是 pip3，因此 3.2 中的pip命令=&gt;pip3
</code></pre>
    <h4>
     <a id="32_repo_22">
     </a>
     3.2 安装repo
    </h4>
    <pre><code>sudo curl https://gitee.com/oschina/repo/raw/fork_flow/repo-py3 &gt; /usr/local/bin/repo
sudo chmod a+x /usr/local/bin/repo
sudo pip install -i https://pypi.tuna.tsinghua.edu.cn/simple requests
</code></pre>
    <h4>
     <a id="33_repo_28">
     </a>
     3.3 初始化repo仓库
    </h4>
    <pre><code>cd /what/you/want/to/develop/directory
repo init -u https://gitee.com/openharmony/manifest.git -b master
repo sync -c // 开始拉取远程代码，大概需要十几分钟的事件，我们可以先去做其他事情了，嘻嘻
</code></pre>

     4.3 修改系统文件把工具添加到环境变量
    </h4>
    <pre><code>文件路径：~/.profile
增加下面一行

修改完，记得 source ~/.profile 让配置文件生效
</code></pre>
