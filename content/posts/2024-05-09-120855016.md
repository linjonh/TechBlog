    <h3>
     <a id="_0">
     </a>
     前言
    </h3>
    <p>
     随着时代的进步，科技不断更迭，各行各业都正在或即将发生翻天覆地的变化。程序员这行就更不用说了，开发技术的迭代更是日新月异。于是从几年前开始，一搜Android开发，画风就是这样的：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b413c8f079efb418f16aa53463493703.png#pic_center"/>
    </p>
    <p>
     这画风朋友们是不是都非常熟悉？伴随着Android岗位趋于饱和，市场对于Android开发技能要求的日益提高，Android变得逐渐卷了起来，每年都有人唱衰Android，还有人总是在问是不是该谋求转行。
    </p>
    <p>
     咱们要先承认一个事实：此时的Android肯定没有2012—2015年发展那么火热了，但远没有到凉凉的地步，无论现在还是将来，只要市场没有出现能替代手机系统的产品，移动互联网的市场就仍旧是刚需。很多人一通分析：随着“5G时代”的来临，从技术层面讲对Android来说会有一定排斥，Android马上就要被彻底放弃。
     <strong>
      在这个音视频、物联网正呈现大趋势的的时代，说5G会让Android凉凉，真是让人笑掉大牙。真正要凉的不是Android，而是那些不愿提升自己技术的开发者。
     </strong>
    </p>
    <hr/>
    <h3>
     <a id="5G_10">
     </a>
     <strong>
      5G时代下的音视频开发
     </strong>
    </h3>
    <p>
     回看2020，在疫情的重创之下，各个行业都苦不堪言，实体经济更是苦不堪言。但与此同时，互联网则显示出了自己的优势逆流而上，资金依靠网络在不同的人、不同的地区畅通无阻。也是基于网络直播、短视频等业务发展迅猛，远程会议、学习、工作、线上教育……甚嚣尘上，进入前所未有的空前盛世。抖音、快手等短视频APP更是为音视频添砖加瓦。
     <br/>
     以高速度、泛在网、低功耗（设备端）、低时延、万物互联、重构安全为特点的5G技术，为音视频的传播和发展提供了更多的可能——这也导致现在的互联网公司越来越看重音视频技术。
     <br/>
     BAT三大元老不必多说，新兴的抖音，快手、斗鱼等依靠短视频、直播起家的公司更是看重这方面的人才。
     <strong>
      音视频技术已经成为Android开发必不可少的一个技能，虽说不是绝对，但也是占据了举足轻重的地位。掌握了这方面的技术，在面试中会拥有很大的加分项，也为拥抱高薪提供了更多的可能。
     </strong>
     <img alt="感受下音视频开发的优厚薪酬" src="https://i-blog.csdnimg.cn/blog_migrate/1fd7909db3016498be4094d5ed4f8b14.png#pic_center"/>
    </p>
    <hr/>
    <h3>
     <a id="_19">
     </a>
     如何学习音视频开发
    </h3>
    <p>
     <strong>
      首先要掌握好基础知识：
     </strong>
    </p>
    <ul>
     <li>
      对音视频的基础知识要有所了解，不能对PCM、H264等不知所云
     </li>
     <li>
      C/C++的基本知识，例如指针这个是必须要会的，不然你可能连别人的代码都看不懂，但不必在这耗费过多精力，与音视频学习同步进行即可。
     </li>
     <li>
      JNI/NDK的基本知识：崩溃定位，Cmake语法等。
     </li>
     <li>
      OpenGL这个主要是在视频渲染或对视频进行逐帧加工处理会用到，属于高阶音视频开发技巧了。
     </li>
     <li>
      还有像shell脚本等知识我先暂且不表，再说下去担心把大家劝退……
     </li>
    </ul>
    <hr/>
    <p>
     <strong>
      掌握扎实的基础知识之后，音视频学习路径可以这样规划：
     </strong>
     <br/>
     <strong>
      初级
     </strong>
     <br/>
     Java层面的一些API调用：使用AudioRecord进行录音得到PCM，使用AudioTrack播放PCM音频、使用Camera2采集摄像头数据、使用 MediaExtractor 和 MediaMuxer API 解析和封装 mp4 文件、使用MediaCodec进行硬件编解码等。
    </p>
    <p>
     <strong>
      中级
     </strong>
     <br/>
     中级更多涉及到一些音视频开源库的编译和使用,
     <strong>
      这里就需要NDK的相关知识了
     </strong>
     ：FFmpeg交叉编译，FFmpeg进行软件编解码、FFmpeg集成xh264，FFmpeg集成Rtmp推流等。
    </p>
    <p>
     <strong>
      高级进阶
     </strong>
     <br/>
     深入研究音视频相关的网络协议，如 rtmp，hls，以及封包格式，如：flv，mp4。深入学习一些音视频领域的开源项目，如 webrtc，ffmpeg，ijkplayer，librtmp 等等。尝试造一个ijkplayer的轮子、尝试使用OpenGL模仿各种抖音效果等等。
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/842b7a5980611e73d06d3b88d1079bd3.png"/>
    </p>
