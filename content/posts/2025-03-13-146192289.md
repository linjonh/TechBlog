    <p>
     <a href="https://leetcode.cn/problems/string-compression/description/?envType=study-plan-v2&amp;envId=leetcode-75" rel="nofollow" title="443. 压缩字符串 - 力扣（LeetCode）">
      443. 压缩字符串 - 力扣（LeetCode）
     </a>
    </p>
    <p>
     <img alt="" height="734" src="https://i-blog.csdnimg.cn/direct/758dd615e85a46208c8440104a5cd9b6.png" width="1392"/>
    </p>
    <p>
     <img alt="" height="966" src="https://i-blog.csdnimg.cn/direct/9a2e62ede54e4a838864cf593c44aaca.png" width="1380"/>
    </p>
    <p>
     可以使用双指针法（即一个写指针和一个读指针）来进行
     <strong>
      原地压缩
     </strong>
     。算法的核心思想是：
    </p>
    <ol>
     <li>
      <strong>
       遍历
       <code>
        chars
       </code>
       数组
      </strong>
      ，找到连续相同的字符组。
     </li>
     <li>
      <strong>
       写入字符
      </strong>
      到
      <code>
       chars
      </code>
      中。
     </li>
     <li>
      <strong>
       如果字符重复次数大于 1
      </strong>
      ，则将计数转换为字符串并写入
      <code>
       chars
      </code>
      。
     </li>
     <li>
      <strong>
       返回最终写入的长度
      </strong>
      。
     </li>
    </ol>
    <p>
     代码实现如下：
    </p>
    <pre><code class="language-python">def compress(chars):
    write = 0  # 记录写入的位置
    read = 0   # 记录读取的位置
    
    while read &lt; len(chars):
        char = chars[read]  # 当前字符
        count = 0  # 记录当前字符的出现次数
        
        # 统计当前字符的连续出现次数
        while read &lt; len(chars) and chars[read] == char:
            read += 1
            count += 1
        
        # 写入字符
        chars[write] = char
        write += 1
        
        # 如果 count &gt; 1，则写入数字（拆分成单个字符）
        if count &gt; 1:
            for c in str(count):
                chars[write] = c
                write += 1
    
    return write  # 返回新长度
</code></pre>
    <h4>
     <strong>
      示例
     </strong>
    </h4>
    <pre><code class="language-python">chars = ["a", "a", "b", "b", "c", "c", "c"]
new_length = compress(chars)
print(chars[:new_length])  # 输出 ['a', '2', 'b', '2', 'c', '3']
</code></pre>
    <h4>
     <strong>
      时间 &amp; 空间复杂度
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(n)，因为我们遍历了
      <code>
       chars
      </code>
      一次。
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1)，仅使用了常量额外空间（
      <code>
       write
      </code>
      和
      <code>
       read
      </code>
      指针）。
     </li>
    </ul>
    <p>
     这个方法可以在
     <strong>
      原地
     </strong>
     进行字符串压缩，并返回正确的新长度！🚀
    </p>
