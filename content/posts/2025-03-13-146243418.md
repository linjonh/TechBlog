    <p>
     Mermaid 是一款用于生成流程图、时序图、甘特图等图表的 JavaScript 库。它可以将简单的文本描述转化为美观的图表，方便开发者进行可视化展示。
    </p>
    <h3>
     <a id="_9">
     </a>
     安装
    </h3>
    <p>
     Mermaid 可以直接在浏览器中使用，也可以在 Node.js 环境中使用。在浏览器中使用可以通过 CDN 引入，例如：
    </p>
    <pre><code class="prism language-HTML">&lt;script src="https://cdn.jsdelivr.net/npm/mermaid@8.14.0/dist/mermaid.min.js"&gt;&lt;/script&gt;
</code></pre>
    <p>
     在 Node.js 中使用可以通过 npm 安装：
    </p>
    <pre><code class="prism language-Shell">npm install mermaid
</code></pre>
    <h3>
     <a id="_25">
     </a>
     基本语法
    </h3>
    <p>
     Mermaid 的语法非常简单，只需要在代码块中使用
     <code>
      mermaid
     </code>
     关键字，并在代码块中编写 Mermaid 语法即可。例如：
    </p>
    <pre><code class="prism language-Mermaid">graph LR;
    A--&gt;B;
    A--&gt;C;
    B--&gt;D;
    C--&gt;D;
</code></pre>

    <p>
     这段代码会生成一个简单的流程图，其中
     <code>
      graph
     </code>
     是关键字，
     <code>
      LR
     </code>
     表示从左到右布局，
     <code>
      A--&gt;B
     </code>
     表示从节点 A 到节点 B 有一条箭头。
    </p>
