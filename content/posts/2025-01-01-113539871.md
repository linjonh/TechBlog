    <p>
     大家好，我是为广大程序员兄弟操碎了心的小编，每天推荐一个小工具/源码，装满你的收藏夹，每天分享一个小技巧，让你轻松节省开发效率，实现不加班不熬夜不掉头发，是我的目标！
    </p>
    <p>
     今天小编推荐一款PHP开发框架
     <a href="https://gitee.com/celaraze/chemex" rel="nofollow">
      咖啡壶（
      <strong>
       Chemex
      </strong>
      ）
     </a>
     ，Chemex是一个轻量的、现代设计风格的 ICT 资产管理系统。得益于 Laravel框架以及Dcat Admin开发平台，使其具备了优雅、简洁的优秀体验。 咖啡壶（Chemex） 是完全免费且开源的，任何人都可以无限制的修改代码以及部署服务，这对于很多想要对ICT资产做信息化管理的中小型企业来说，是一个很好的选择：低廉的成本换回的是高效的管理方案，同时又有健康的生态提供支持。
    </p>
    <h3>
     <a id="_4">
     </a>
     特点
    </h3>
    <ul>
     <li>
      经典的 LNMP 环境即可运行。
     </li>
     <li>
      使用先进的 Web 框架进行开发。
     </li>
     <li>
      简洁优雅的使用体验。
     </li>
    </ul>
    <h3>
     <a id="_10">
     </a>
     环境要求
    </h3>
    <ul>
     <li>
      <code>
       git
      </code>
      ，用于管理版本，部署和升级必要工具。
     </li>
     <li>
      <code>
       composer
      </code>
      ，用于安装 PHP 依赖，这是 PHP 的包管理工具。
     </li>
     <li>
      <code>
       PHP 7.3 +
      </code>
      ，暂时不要使用 PHP8，目前有一些小问题需要解决。
     </li>
     <li>
      <code>
       MariaDB 10.2 +
      </code>
      ，数据库引擎，理论上
      <code>
       MySQL 5.6+
      </code>
      兼容支持。
     </li>
     <li>
      <code>
       ext-zip
      </code>
      扩展，注意和 PHP 版本相同。
     </li>
     <li>
      <code>
       ext-json
      </code>
      扩展，注意和 PHP 版本相同。
     </li>
     <li>
      <code>
       ext-fileinfo
      </code>
      扩展，注意和 PHP 版本相同。
     </li>
     <li>
      <code>
       ext-ldap
      </code>
      扩展，注意和 PHP 版本相同。
     </li>
     <li>
      <code>
       ext-bcmath
      </code>
      扩展，注意和 PHP 版本相同。
     </li>
    </ul>
