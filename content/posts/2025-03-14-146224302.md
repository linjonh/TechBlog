    <h2>
     <a id="3_Modbus_RTU_1">
     </a>
     3. Modbus RTU
    </h2>
    <h3>
     <a id="31_Modbus_RTUModbus_TCP_2">
     </a>
     3.1. Modbus RTU和Modbus TCP的区别
    </h3>
    <ol>
     <li>
      报文头：Modbus RTU只有从机id一位，Modbus TCP有7位
     </li>
     <li>
      CRC校验：Modbus RTU应用于串口通信，需要校验码；Modbus TCP应用于网络通信，网络协议中自带校验
     </li>
     <li>
      通信：Modbus TCP通过socket网络通信，需要创建套接字；Modbus RTU需要写入串口信息
     </li>
    </ol>
    <h3>
     <a id="32_Modbus_RTU__6">
     </a>
     3.2. Modbus RTU 特点
    </h3>
    <ol>
     <li>
      主从问答通信
     </li>
     <li>
      串口参数：波特率9600；数据位8；停止位1；无流控
     </li>
    </ol>
    <h3>
     <a id="33_Modbus_RTU_9">
     </a>
     3.3. Modbus RTU协议格式
    </h3>
    <p>
     地址码+功能码+数据+校验码
     <br/>
     <strong>
      地址码：
     </strong>
     1字节，从机ID，1~247
     <br/>
     <strong>
      功能码：
     </strong>
     1字节，同Modbus TCP
     <br/>
     <strong>
      数据：
     </strong>
     起始地址，数量，数据……同Modbus TCP
     <br/>
     <strong>
      校验码：
     </strong>
     2字节，校验范围：对前面三部分，可以通过函数生成
    </p>
