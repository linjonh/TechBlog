    <p>
     python内置函数bytes返回一个新的bytes类型的对象，bytes类型对象是不可变序列，包含范围为 0 &lt;= x &lt; 256 的整数。bytes可以看做是bytearray的不可变版本，它同样支持索引和切片操作 bytes语法 class bytes([source[, encoding[, errors]]])
    </p>
    <p>
     语法结构：
    </p>
    <pre><code class="language-python">class bytes([source[, encoding[, errors]]])</code></pre>
    <p>
     参数解释：
    </p>
    <ul>
     <li>
      可选形参source可以传入字符串，int,iterable 可迭代对象, 遵循 缓冲区接口 的对象, 不同的类型将有不同的效果
     </li>
     <li>
      string ，如果source是字符串，则必须指定encoding参数，bytearray() 会使用 str.encode() 方法来将 string 转变成 bytes
     </li>
     <li>
      int ,如果source是int，会初始化大小为该数字的数组，并使用 null 字节填充
     </li>
     <li>
      如果是一个遵循 缓冲区接口 的对象，该对象的只读缓冲区将被用来初始化字节数组
     </li>
     <li>
      iterable 可迭代对象， 要求元素的范围必须是 0 &lt;= x &lt; 256 的整数，它会被用作数组的初始内容
     </li>
     <li>
      如果没有传入source参数，则返回一个长度为0的bytes数组
     </li>
    </ul>
    <p>
     示例代码1：
    </p>
    <pre><code class="language-python">print(bytes())
print(bytes("I love python", encoding='utf-8'))
print(bytes(6))
print(bytes([11, 22, 33]))
</code></pre>
