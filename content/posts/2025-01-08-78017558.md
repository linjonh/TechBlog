# python 指定运行版本，python 版本

具体代码：

```
//查看python位置
which python

//先备份
1、sudo cp /usr/bin/python /usr/bin/python_cp
//删除
2、sudo rm /usr/bin/python
//默认设置成python3.5，创建链接
3、sudo ln -s /usr/bin/python3.5 /usr/bin/python
```

**注意：我本地查看的位置是再 local 下面的，请根据位置不一样修改上面的代码，要不然不会起效～**

```

```
