     Docker save命令怎么用
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="./../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">

    <p>
     <code>
      docker save
     </code>
     命令用于将一个 Docker 镜像保存为一个 tar 文件。这在将镜像传输到其他系统或进行备份时非常有用。通过此命令，你可以将镜像导出到本地存储设备或传输给其他人。
    </p>
    <h4>
     <a id="_2">
     </a>
     基本语法
    </h4>
    <pre><code class="prism language-shell"><span class="token function">docker</span> save <span class="token parameter variable">-o</span> <span class="token operator">&lt;</span>保存文件路径<span class="token operator">&gt;</span> <span class="token operator">&lt;</span>镜像名<span class="token operator">&gt;</span>:<span class="token operator">&lt;</span>标签<span class="token operator">&gt;</span>
</code></pre>
    <ul>
     <li>
      <code>
       -o &lt;保存文件路径&gt;
      </code>
      ：指定导出 tar 文件的保存路径和文件名。
     </li>
     <li>
      <code>
       &lt;镜像名&gt;:&lt;标签&gt;
      </code>
      ：指定要导出的镜像名称和标签。
     </li>
    </ul>
    <h4>
     <a id="_11">
     </a>
     示例
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        将镜像保存为 tar 文件
       </strong>
       假设你有一个名为
       <code>
        mysql
       </code>
       的镜像，并且希望将其保存为
       <code>
        mysql.tar
       </code>
       文件：
      </p>
      <pre><code class="prism language-shell"><span class="token function">docker</span> save <span class="token parameter variable">-o</span> mysql.tar mysql
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        指定标签的镜像
       </strong>
       如果你想保存一个带有标签的镜像，例如
       <code>
        nginx:latest
       </code>
       ：
      </p>
      <pre><code class="prism language-shell"><span class="token function">docker</span> save <span class="token parameter variable">-o</span> nginx_image.tar nginx:latest
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        保存多个镜像
       </strong>
       如果你想保存多个镜像，可以在命令中列出多个镜像名称：
      </p>
      <pre><code class="prism language-shell"><span class="token function">docker</span> save <span class="token parameter variable">-o</span> my_images.tar nginx:latest nginx:latest
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="_31">
     </a>
     导入镜像
    </h4>
    <p>
     一旦你使用
     <code>
      docker save
     </code>
     保存了镜像文件，你可以使用
     <code>
      docker load
     </code>
     命令将镜像加载回 Docker：
    </p>
    <pre><code>docker load -i mysql.tar
</code></pre>
    <p>
     这样，你就能够在其他机器上恢复或共享该镜像了。
    </p>
    <h4>
     <a id="_41">
     </a>
     总结
    </h4>
    <p>
     <code>
      docker save
     </code>
     命令非常适用于备份、迁移和分享镜像，它允许你将 Docker 镜像导出为文件。
    </p>
