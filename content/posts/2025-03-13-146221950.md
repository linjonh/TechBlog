    <h2 id="%E9%A2%98%E7%9B%AE" name="%E9%A2%98%E7%9B%AE" style="background-color:transparent">
     题目
    </h2>
    <p>
     一个机器人位于一个
     <code>
      m x n
     </code>
     <em>
     </em>
     网格的左上角 （起始点在下图中标记为 “Start” ）。
    </p>
    <p>
     机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。
    </p>
    <p>
     问总共有多少条不同的路径？
    </p>
    <p>
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
    </p>

    </p>
    <pre><strong>输入：</strong>m = 3, n = 7
<strong>输出：</strong>28</pre>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <pre><strong>输入：</strong>m = 3, n = 2
<strong>输出：</strong>3
<strong>解释：</strong>
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -&gt; 向下 -&gt; 向下
2. 向下 -&gt; 向下 -&gt; 向右
3. 向下 -&gt; 向右 -&gt; 向下
</pre>

    <h2 id="%E6%80%9D%E8%B7%AF" name="%E6%80%9D%E8%B7%AF">
     思路
    </h2>
    <p>
     机器人从(0 , 0) 位置出发，到(m - 1, n - 1)终点。
    </p>
    <ul>
     <li>
      确定dp数组以及下标的含义
     </li>
    </ul>
    <p>
     dp[i][j] ：表示从（0 ，0）出发，到(i, j) 有dp[i][j]条不同的路径。
    </p>
    <ul>
     <li>
      确定递推公式
     </li>
    </ul>
    <p>
     题目中说只能向右或者向下走，
    </p>
    <p>
     想要求dp[i][j]，只能有两个方向来推导出来，即dp[i - 1][j] 和 dp[i][j - 1]。
    </p>
    <p>
     dp[i - 1][j] 表示从(0, 0)的位置到(i - 1, j)有几条路径，dp[i][j - 1]同理。
    </p>
    <p>
     那么很自然，dp[i][j] = dp[i - 1][j] + dp[i][j - 1]，因为dp[i][j]只有这两个方向过来。
    </p>
    <p>
     <span style="background-color:#ffd900">
      有的可能会觉得为什么不是dp[i][j] = dp[i - 1][j] +1+ dp[i][j - 1]+1呢？如果+1就是再走一步到达终点，而题目中要求的是路径有几种，所以不要加。
     </span>
    </p>
    <ul>
     <li>
      dp数组的初始化
     </li>
    </ul>
    <p>
     如何初始化呢，首先dp[i][0]一定都是1，因为从(0, 0)的位置到(i, 0)的路径只有一条，那么dp[0][j]也同理。因为他只能往右或者下走
    </p>
    <ul>
     <li>
      确定遍历顺序
     </li>
    </ul>
    <p>
     这里要看一下递推公式dp[i][j] = dp[i - 1][j] + dp[i][j - 1]，dp[i][j]都是从其上方和左方推导而来，那么从左到右一层一层遍历就可以了。
    </p>
    <p>
     这样就可以保证推导dp[i][j]的时候，dp[i - 1][j] 和 dp[i][j - 1]一定是有数值的。
    </p>
    <ul>
     <li>
      举例推导dp数组
     </li>
    </ul>
    <p>
     m=3，n=2
    </p>

    <h2 id="%E4%BB%A3%E7%A0%81" name="%E4%BB%A3%E7%A0%81">
     代码
    </h2>
    <pre><code class="language-java">class Solution {
    public int uniquePaths(int m, int n) {
        int dp[][]=new int[m][n];//定义一个二维数组
        for(int i=0;i&lt;m;i++){//初始化
            dp[i][0]=1;
        }
        for(int j=0;j&lt;n;j++){
            dp[0][j]=1;
        }
        for(int i=1;i&lt;m;i++){
            for(int j=1;j&lt;n;j++){
                dp[i][j]=dp[i-1][j]+dp[i][j-1];
            }
        }
        return dp[m-1][n-1];
    }
}</code></pre>
