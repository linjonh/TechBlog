    <h3 id="instr-方法-参数说明及使用">
     instr() 方法 参数说明及使用
    </h3>
    <p>
     –instr(str1,str2[,num1,num2]) “[,num1,num2]” 为可选参数
     <br/>
     –str1 ：被搜索的目标
     <br/>
     –str2 ：希望搜索的目标
     <br/>
     –num1 ：str1 的检索开始位置 ，默认为 1 ，（从左往右检索），若为 负数，则从右往左检索
     <br/>
     –num2 ：str2 在 str1 出现的第 num2 次的位置，默认为 1
     <br/>
     1. 简单例子
     <br/>
     –第一次出现的位置
    </p>
    <pre class="prettyprint"><code class="hljs sql"><span class="hljs-operator"><span class="hljs-keyword">select</span> instr(<span class="hljs-string">'sfadfzhaisdaio'</span>,<span class="hljs-string">'ai'</span>,<span class="hljs-number">1</span>,<span class="hljs-number">1</span>) A <span class="hljs-keyword">from</span> dual;</span></code></pre>
    <p>
     –结果
     <br/>
     —A
     <br/>
     – 8
    </p>
    <p>
     –第二次出现的位置
    </p>
    <pre class="prettyprint"><code class="hljs sql"><span class="hljs-operator"><span class="hljs-keyword">select</span> instr(<span class="hljs-string">'sfadfzhaisdaio'</span>,<span class="hljs-string">'ai'</span>,<span class="hljs-number">1</span>,<span class="hljs-number">2</span>) A <span class="hljs-keyword">from</span> dual;</span></code></pre>
    <p>
     –结果
     <br/>
     —-A
     <br/>
     – 12
    </p>
    <p>
     –从右往左检索
    </p>
    <pre class="prettyprint"><code class="hljs sql"><span class="hljs-operator"><span class="hljs-keyword">select</span> instr(<span class="hljs-string">'sfadfzhaisdaio'</span>,<span class="hljs-string">'ai'</span>,-<span class="hljs-number">1</span>,<span class="hljs-number">1</span>) A <span class="hljs-keyword">from</span> dual;</span></code></pre>
    <p>
     –结果
     <br/>
     —A
     <br/>
     —12
    </p>
    <p>
     2.应用小技巧
     <br/>
     可应用于查询结果集排序：
     <br/>
     自定义将某字段 组合成 str – (bs、ee、33、s) 想 将 按照 s 、ee、 33、bs 排序
    </p>
    <pre class="prettyprint"><code class="hljs sql"><span class="hljs-operator"><span class="hljs-keyword">select</span> forder <span class="hljs-keyword">from</span> (
<span class="hljs-keyword">select</span>  <span class="hljs-string">'bs'</span>  forder <span class="hljs-keyword">from</span> dual 
<span class="hljs-keyword">union</span>
<span class="hljs-keyword">select</span>  <span class="hljs-string">'ee'</span>  <span class="hljs-keyword">from</span> dual
<span class="hljs-keyword">union</span>
<span class="hljs-keyword">select</span>  <span class="hljs-string">'33'</span>  <span class="hljs-keyword">from</span> dual
<span class="hljs-keyword">union</span>
<span class="hljs-keyword">select</span>  <span class="hljs-string">'s'</span>  <span class="hljs-keyword">from</span> dual
) <span class="hljs-keyword">order</span> <span class="hljs-keyword">by</span>  instr(<span class="hljs-string">'s 、ee、 33、bs'</span>, forder)   ;</span></code></pre>
