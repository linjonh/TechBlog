    <h2>
     一、代码解释与实现功能：
    </h2>
    <p>
     基数排序是一种非比较型整数排序算法，它通过将整数按位数切割成不同的数字，然后按每个位数分别进行比较和排序。
    </p>
    <ol>
     <li>
      <h3>
       <strong>
        输入数据
       </strong>
       ：
      </h3>
      <ul>
       <li>
        <p>
         用户输入一个整数
         <code>
          n
         </code>
         ，表示数组的长度。
        </p>
       </li>
       <li>
        <p>
         然后输入
         <code>
          n
         </code>
         个整数，存储在数组
         <code>
          a
         </code>
         中。
        </p>
       </li>
       <li>
        <p>
         同时，代码会找到数组中的最大值
         <code>
          max
         </code>
         ，并计算最大值的位数
         <code>
          num
         </code>
         （即需要排序的轮数）。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <h3>
       <strong>
        基数排序
       </strong>
       ：
      </h3>
      <ul>
       <li>
        <p>
         代码通过
         <code>
          rs
         </code>
         函数实现基数排序。
        </p>
       </li>
       <li>
        <p>
         每一轮排序根据当前位数（个位、十位、百位等）对数组进行排序。
        </p>
       </li>
       <li>
        <p>
         排序过程中使用计数排序（Counting Sort）作为子程序，确保每一轮排序的稳定性。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <h3>
       <strong>
        输出结果
       </strong>
       ：
      </h3>
      <ul>
       <li>
        <p>
         每一轮排序后，代码会输出当前排序的结果。
        </p>
       </li>
       <li>
        <p>
         最终，数组
         <code>
          a
         </code>
         会被完全排序。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <h2>
     二、具体代码展示：
    </h2>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;
void rs(int a[],int n,int num)
{
    int b=1;
    for(int i=0;i&lt;num;i++)
    {
        int past[1000];
        int s[10]={0};
        for(int i=0;i&lt;n;i++)
            s[(a[i]/b)%10]++;
        for(int i=1;i&lt;n;i++)
            s[i]+=s[i-1];
        for(int i=n-1;i&gt;=0;i--)
            past[--s[(a[i]/b)%10]]=a[i];
        for(int i=0;i&lt;n;i++)
            cout&lt;&lt;past[i]&lt;&lt;' ';
        cout&lt;&lt;endl;
        for(int i=0;i&lt;n;i++)
            a[i]=past[i];
        b*=10;
    }
}
int main()

    cin&gt;&gt;n;
    int a[n];
    for(int i=0;i&lt;n;i++)
    {

        if(max&lt;a[i])
            max=a[i];

    {

    <p>
     具体行解释：
    </p>
    <pre><code class="language-cpp">int main()
{
    int n, max = 0, num = 0;
    cin &gt;&gt; n; // 输入数组长度
    int a[n]; // 定义数组
    for (int i = 0; i &lt; n; i++)
    {
        cin &gt;&gt; a[i]; // 输入数组元素
        if (max &lt; a[i])
            max = a[i]; // 找到最大值
    }
    while (max &gt; 0)
    {
        max /= 10;
        num++; // 计算最大值的位数
    }
    rs(a, n, num); // 调用基数排序函数
}</code></pre>
    <ul>
     <li>
      <p>
       输入数组并找到最大值。
      </p>
     </li>
     <li>
      <p>
       计算最大值的位数
       <code>
        num
       </code>
       ，决定需要多少轮排序。
      </p>
     </li>
     <li>
      <p>
       调用
       <code>
        rs
       </code>
       函数进行排序。
      </p>
     </li>
    </ul>
    <pre><code class="language-cpp">void rs(int a[], int n, int num)
{
    int b = 1; // b 表示当前位数（1, 10, 100, ...）
    for (int i = 0; i &lt; num; i++) // 按位数进行排序
    {
        int past[1000]; // 临时数组，用于存储排序结果
        int s[10] = {0}; // 计数数组，用于计数每个数字（0-9）的出现次数

        // 统计当前位数的数字出现次数
        for (int i = 0; i &lt; n; i++)
            s[(a[i] / b) % 10]++;

        // 计算前缀和，确定每个数字的最终位置
        for (int i = 1; i &lt; 10; i++)
            s[i] += s[i - 1];

        // 将元素按当前位数排序到临时数组 past 中
        for (int i = n - 1; i &gt;= 0; i--)
            past[--s[(a[i] / b) % 10]] = a[i];

        // 输出当前排序结果
        for (int i = 0; i &lt; n; i++)
            cout &lt;&lt; past[i] &lt;&lt; ' ';
        cout &lt;&lt; endl;

        // 将排序结果复制回原数组 a
        for (int i = 0; i &lt; n; i++)
            a[i] = past[i];

        b *= 10; // 处理下一位
    }
}</code></pre>
    <ul>
     <li>
      <p>
       每一轮排序：
      </p>
      <ol>
       <li>
        <p>
         统计当前位数的数字（0-9）出现次数。
        </p>
       </li>
       <li>
        <p>
         计算前缀和，确定每个数字的最终位置。
        </p>
       </li>
       <li>
        <p>
         将数组元素按当前位数排序到临时数组
         <code>
          past
         </code>
         中。
        </p>
       </li>
       <li>
        <p>
         输出当前排序结果。
        </p>
       </li>
       <li>
        <p>
         将排序结果复制回原数组
         <code>
          a
         </code>
         。
        </p>
       </li>
      </ol>
     </li>
     <li>
      <p>
       重复上述过程，直到所有位数处理完毕。
      </p>
     </li>
    </ul>
