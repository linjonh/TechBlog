    <h2>
     <a id="_4">
     </a>
     一、相对路径符号及含义
    </h2>
    <p>
     <code>
      ./
     </code>
     表示当前目录。例如，./data.txt表示当前目录下的data.txt文件。若省略./，直接写data.txt也等效
    </p>
    <p>
     <code>
      ../
     </code>
     表示上一级目录（父目录）。例如，…/config.yml表示当前目录的父级目录下的config.yml文件
    </p>
    <p>
     <code>
      ../../
     </code>
     表示上上级目录（祖父目录），每多一层…/则向上回溯一级目录。例如，…/…/images/cat.jpg表示向上回溯两级目录后的images文件夹下的文件
    </p>
    <h2>
     <a id="_11">
     </a>
     二、相对路径的写法示例
    </h2>
    <p>
     假设当前工作目录为：C:\project\src，文件结构如下：
    </p>
    <pre><code class="prism language-c">project<span class="token operator">/</span>
├─ src<span class="token operator">/</span>
│  ├─ main<span class="token punctuation">.</span>py
│  └─ data<span class="token operator">/</span>
│     └─ input<span class="token punctuation">.</span>csv
└─ config<span class="token operator">/</span>
   └─ settings<span class="token punctuation">.</span>ini
</code></pre>
    <h3>
     <a id="1__23">
     </a>
     1. 访问同级目录
    </h3>
    <pre><code class="prism language-c">file_path <span class="token operator">=</span> <span class="token string">"./data/input.csv"</span>  # 当前目录下的data子目录中的文件
</code></pre>
    <h3>
     <a id="2__28">
     </a>
     2. 访问父级目录
    </h3>
    <pre><code class="prism language-c">file_path <span class="token operator">=</span> <span class="token string">"../config/settings.ini"</span>  # 上一级目录中的config子目录中的文件
</code></pre>
    <h3>
     <a id="3__33">
     </a>
     3. 访问更高级目录
    </h3>
    <pre><code class="prism language-c">file_path <span class="token operator">=</span> <span class="token string">"../../external/tools.py"</span>  # 上上级目录中的external子目录中的文件
</code></pre>
