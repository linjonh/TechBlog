# 音视频终端 SDK（腾讯云视立方）项目管理

音视频终端 SDK（腾讯云视立方）控制台提供项目管理功能，您可参照以下步骤为您的应用快速添加音视频通话能力和多人音视频互动能力。

### 前提条件

在使用音视频通话 SDK 和多人音视频房间 SDK 前，请先
[注册腾讯云](https://partner.cloud.tencent.com/invitation/99377079157835da208bd0/100036043966?inviteType=2 "注册腾讯云")
账号并完成
[实名认证](https://cloud.tencent.com/document/product/378/3629 "实名认证")
。

### 创建项目

登录
[腾讯云视立方控制台 > 项目管理](https://console.cloud.tencent.com/vcube/project/manage "腾讯云视立方控制台 > 项目管理")
，我们提供创建项目的完整体验流程和快速开始流程：

**注意：**

**新建项目并绑定/创建应用后，可领取体验版免费体验音视频通话能力**
，有效期7天且支持续期1次（7天），每个应用可领取1次体验资格，一个账号下所有应用的体验总次数为10次。

完整体验

快速开始

1.
**创建项目，选择类型及场景，开通相关服务。**

1.1

单击
**创建项目**
开始创建。

![](https://img-blog.csdnimg.cn/img_convert/d7d0d605b320552ad81c46df2b7fb388.png)

1.2
**选择项目类型**
。可选择
**创建新项目**
并输入项目名称，
**或关联已有项目**
。

![](https://img-blog.csdnimg.cn/img_convert/4d72f7c1970f541766bc934c83ba60cf.png)

1.3

**选择接入场景**
。选择快速接入的音视频场景，分为
**音视频通话**
和
**多人音视频互动**
。

![](https://img-blog.csdnimg.cn/img_convert/da37d1485d74a3c968a17a220337221a.png)

1.4
**选择集成方式**
。可选择
**含 UI 快速集成**
或
**不含 UI 集成**
。推荐选择
**含 UI 快速集成**
，可购买音视频通话能力套餐/多人音视频房间套餐并快速接入。

![](https://img-blog.csdnimg.cn/img_convert/b4b163a800a8580c33a142b3720a1ae9.png)

1.5
**开通相关服务**
。使用音视频通话 SDK 需
**提前开通即时通信 IM 和实时音视频 TRTC 服务**
，仅开通服务，不涉及付费购买产品。

![](https://img-blog.csdnimg.cn/img_convert/c4c908939ca15451078cf7b1786470d6.png)

1.6单击
**创建项目并下一步**
，完成项目创建，进入准备开发阶段。

![](https://img-blog.csdnimg.cn/img_convert/b16c7eebd0c7f9a074e7de08616ad309.png)

2.
**准备开发。**

2.1
**体验 DEMO**
。您可提前体验 DEMO 了解 SDK 应用效果，包括 Android、iOS 和 Web 端。

2.2
**关联/创建 IM 应用**
。项目创建需要使用 IM 服务，请创建新的应用或者选择一个 IM 应用进行关联。

音视频通话

多人音视频互动

![](https://img-blog.csdnimg.cn/img_convert/7d53d1b240ba3930c0a9f3643020184c.png)

![](https://img-blog.csdnimg.cn/img_convert/e5a06b8870a2fd5022d1ea028a107479.png)

![](https://i-blog.csdnimg.cn/direct/1515cc2da8c5402f82d3b07d2a2035b6.jpeg)

2.3
**领取资源**
。每个应用可免费体验音视频通话 SDK /多人音视频房间 SDK 功能1次，有效期7天，支持续期1次，账号下所有应用可体验总次数为10次。

体验版详细能力支持请参见
[音视频通话 SDK 功能概览](https://cloud.tencent.com/document/product/1640/82856 "音视频通话 SDK 功能概览")
或
[多人音视频房间 SDK 功能概览](https://cloud.tencent.com/document/product/1690/89367 "多人音视频房间 SDK 功能概览")
。业务上线请升级购买正式版本套餐，享受完整功能。

音视频通话

多人音视频互动

![](https://img-blog.csdnimg.cn/img_convert/ca98df03b786eea10744b31e0abafec3.png)

![](https://img-blog.csdnimg.cn/img_convert/3f5458c65417001d1f1ecca5b0e937b0.png)

2.4 单击
**前往集成**
。

1.
**领取体验资源并开始集成。**

1.1

点击 领取体验资源，快速开始。

![](https://img-blog.csdnimg.cn/img_convert/d8825f7a199455488c7c4689c32dc9d0.png)

1.2 选择场景并点击立即领取，耐心等待即可完成。若未开通过实时音视频 TRTC 和即时通信 IM 云产品服务，需要勾选同意相关协议。

![](https://img-blog.csdnimg.cn/img_convert/730ea6f5646a9e0a4983405f1b7bd82b.png)

1.3 领取成功后，跳转至体验项目-集成指南页面，开始集成音视频通话 SDK。

![](https://img-blog.csdnimg.cn/img_convert/a015b9cee0467ff3087ebc27edb91a29.png)

### 集成指南

1.
**选择集成环境并下载 TUIKit 开发包**
。集成环境包括 Android、iOS 和 Web 端。

2.
**集成测试**
。下载 TUIKit 开发包后，参照集成测试的步骤查看相关集成文档完成集成测试。

3.

**正式开发**

。完成 SDK 接入测试后，若需正式开发并上线音视频应用，可购买音视频通话套餐包，包括1V1通话版和群组通话版，升级当前项目业务，享受完整功能。

![](https://img-blog.csdnimg.cn/img_convert/1592c52b4079b5a2bd50f68e73041966.png)

### 查看项目配置

1. 您可在项目管理页面查看所有的项目，单击右侧
**项目配置**
即可进入查看项目配置。

![](https://img-blog.csdnimg.cn/img_convert/8eae06b5b5a7a36e73c22158500b6959.png)

2.
**项目配置**
：您可在此查看项目的名称、接入场景、使用的服务版本和应用的创建时间及到期时间，同时可单击自动续费，前往
**费用中心**
>
[续费管理](https://console.cloud.tencent.com/account/renewal "续费管理 ")
对套餐进行续费。

![](https://img-blog.csdnimg.cn/img_convert/f8345696924229df4f4cd80b0847d0dd.png)

3.
**集成指南**
：快速选择并下载 SDK 开发包，按照步骤指引简单快速集成。

![](https://img-blog.csdnimg.cn/img_convert/3c5dac73718e1da2e638579a692779a7.png)
