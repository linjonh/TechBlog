    <h2>
     <a id="_10">
     </a>
     前言
    </h2>
    <p>
     图像预处理是
     <strong>
      机器学习和计算机视觉任务
     </strong>
     中至关重要的一环，其核心目的是通过对
     <strong>
      原始图像进行优化和调整
     </strong>
     ，
     <strong>
      提升模型性能、降低噪声干扰、增强关键特征，并适应模型的输入要求
     </strong>
     。以下是分领域的详细说明：
    </p>
    <hr/>
    <h2>
     <a id="_19">
     </a>
     一、通用目的
    </h2>
    <h3>
     <a id="11__20">
     </a>
     1.1 数据标准化
    </h3>
    <h4>
     <a id="_21">
     </a>
     目的
    </h4>
    <p>
     目的：统一输入数据的分布，避免因
     <strong>
      像素值范围差异
     </strong>
     （如0-255或0-1）导致模型训练不稳定。
    </p>
    <h4>
     <a id="_25">
     </a>
     实现
    </h4>
    <pre><code class="prism language-python"><span class="token comment">#归一化到 [0, 1]</span>
image_normalized <span class="token operator">=</span> image <span class="token operator">/</span> <span class="token number">255.0</span>
<span class="token comment">#标准化（均值中心化 + 方差归一化）</span>
image_standardized <span class="token operator">=</span> <span class="token punctuation">(</span>image <span class="token operator">-</span> mean<span class="token punctuation">)</span> <span class="token operator">/</span> std
</code></pre>
    <h3>
     <a id="12__34">
     </a>
     1.2 噪声抑制
    </h3>
    <h4>
     <a id="_36">
     </a>
     目的
    </h4>
    <p>
     目的：消除图像中的
     <strong>
      随机噪声（如高斯噪声、椒盐噪声）
     </strong>
     ，提升
     <strong>
      特征提取的鲁棒性
     </strong>
     。
    </p>
    <h4>
     <a id="_39">
     </a>
     实现
    </h4>
    <h5>
     <a id="_40">
     </a>
     高斯滤波
    </h5>
    <blockquote>
     <p>
      高斯滤波：cv2.GaussianBlur()
     </p>
    </blockquote>
    <h5>
     <a id="_42">
     </a>
     中值滤波
    </h5>
    <blockquote>
     <p>
      中值滤波：cv2.medianBlur()
     </p>
    </blockquote>
    <h5>
     <a id="_44">
     </a>
     双边滤波
    </h5>
    <blockquote>
     <p>
      双边滤波：保留边缘的同时去噪，cv2.bilateralFilter()
     </p>
    </blockquote>
    <h3>
     <a id="13__47">
     </a>
     1.3 尺寸统一化
    </h3>
    <h4>
     <a id="_49">
     </a>
     目的
    </h4>
    <p>
     目的：确保所有
     <strong>
      输入图像尺寸一致
     </strong>
     ，适配
     <strong>
      模型输入层
     </strong>
     （如CNN要求固定尺寸）。
    </p>
    <h4>
     <a id="_52">
     </a>
     实现
    </h4>
    <pre><code class="prism language-python">resized_image <span class="token operator">=</span> cv2<span class="token punctuation">.</span>resize<span class="token punctuation">(</span>image<span class="token punctuation">,</span> <span class="token punctuation">(</span>width<span class="token punctuation">,</span> height<span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="14__58">
     </a>
     1.4 数据增强
    </h3>
    <h4>
     <a id="_60">
     </a>
     目的
    </h4>
    <p>
     目的：通过**几何变换（旋转、翻转、裁剪）
     <strong>
      或
     </strong>
     颜色扰动（亮度、对比度调整）**扩充数据集，防止过拟合。
    </p>
    <h4>
     <a id="_63">
     </a>
     实现
    </h4>
    <pre><code class="prism language-python"><span class="token comment">#使用OpenCV或深度学习框架（如TensorFlow的ImageDataGenerator）</span>
flipped <span class="token operator">=</span> cv2<span class="token punctuation">.</span>flip<span class="token punctuation">(</span>image<span class="token punctuation">,</span> <span class="token number">1</span><span class="token punctuation">)</span>  <span class="token comment"># 水平翻转</span>
rotated <span class="token operator">=</span> cv2<span class="token punctuation">.</span>rotate<span class="token punctuation">(</span>image<span class="token punctuation">,</span> cv2<span class="token punctuation">.</span>ROTATE_90_CLOCKWISE<span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="15__71">
     </a>
     1.5 特征增强
    </h3>
    <h4>
     <a id="_72">
     </a>
     目的
    </h4>
    <p>
     目的：突出
     <strong>
      目标区域的关键特征（如边缘、纹理）
     </strong>
     。
    </p>
    <h4>
     <a id="_75">
     </a>
     实现：
    </h4>
    <h5>
     <a id="_77">
     </a>
     边缘检测
    </h5>
    <blockquote>
     <p>
      边缘检测：Sobel算子、Canny算子
     </p>
    </blockquote>
    <h5>
     <a id="_79">
     </a>
     直方图均衡化
    </h5>
    <blockquote>
     <p>
      直方图均衡化：cv2.equalizeHist()（增强对比度）
     </p>
    </blockquote>
    <h5>
     <a id="_81">
     </a>
     锐化
    </h5>
    <blockquote>
     <p>
      锐化：拉普拉斯算子或自定义卷积核
     </p>
    </blockquote>
    <h2>
     <a id="_84">
     </a>
     二、分领域预处理
    </h2>
    <h3>
     <a id="21_SVM_85">
     </a>
     2.1 传统机器学习（如SVM、随机森林）
    </h3>
    <h4>
     <a id="211__86">
     </a>
     2.1.1 特点
    </h4>
    <p>
     特点：依赖
     <strong>
      手工提取特征
     </strong>
     （如HOG、LBP、颜色直方图）。
    </p>
    <h4>
     <a id="212__89">
     </a>
     2.1.2 预处理重点
    </h4>
    <h5>
     <a id="_90">
     </a>
     灰度化
    </h5>
    <p>
     灰度化：
     <strong>
      减少计算量
     </strong>
     ，cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    </p>
    <h5>
     <a id="_92">
     </a>
     二值化
    </h5>
    <p>
     二值化：cv2.threshold()（如
     <strong>
      文档OCR任务
     </strong>
     ）
    </p>
    <h5>
     <a id="_94">
     </a>
     形态学操作
    </h5>
    <p>
     形态学操作：腐蚀、膨胀（
     <strong>
      去除小斑点或连接断裂区域
     </strong>
     ）
    </p>
    <h5>
     <a id="_96">
     </a>
     特征工程
    </h5>
    <p>
     特征工程：提取
     <strong>
      统计特征
     </strong>
     （均值、方差）或结构特征。
    </p>
    <h3>
     <a id="22_CNNTransformer_99">
     </a>
     2.2 深度学习（如CNN、Transformer）
    </h3>
    <h4>
     <a id="221__100">
     </a>
     2.2.1 特点
    </h4>
    <p>
     特点：端到端
     <strong>
      学习特征
     </strong>
     ，但依赖大量数据和标准化输入。
    </p>
    <h4>
     <a id="222__103">
     </a>
     2.2.2 预处理重点
    </h4>
    <h5>
     <a id="_104">
     </a>
     通道顺序调整
    </h5>
    <p>
     通道顺序调整：
     <strong>
      适配框架要求
     </strong>
     （如TensorFlow的HWC格式或PyTorch的CHW格式）。
    </p>
    <h5>
     <a id="_106">
     </a>
     批量归一化
    </h5>
    <p>
     批量归一化：在
     <strong>
      数据加载阶段统一处理
     </strong>
     （如torchvision.transforms.Normalize）。
    </p>
    <h5>
     <a id="_108">
     </a>
     高级增强
    </h5>
    <p>
     高级增强：MixUp、CutMix等基于混合样本的策略。
    </p>
    <h5>
     <a id="_110">
     </a>
     预训练模型适配
    </h5>
    <p>
     预训练模型适配：
     <strong>
      输入需与预训练数据分布一致
     </strong>
     （如ImageNet的均值方差）。
    </p>
    <h3>
     <a id="23__113">
     </a>
     2.3 其他领域
    </h3>
    <h4>
     <a id="231MRICT_114">
     </a>
     2.3.1医学影像（如MRI、CT）
    </h4>
    <h5>
     <a id="_115">
     </a>
     去伪影
    </h5>
    <blockquote>
     <p>
      去伪影：消除扫描过程中的
      <strong>
       运动伪影或设备噪声
      </strong>
      。
     </p>
    </blockquote>
    <h5>
     <a id="_117">
     </a>
     标准化
    </h5>
    <p>
     标准化：使用Z-score或直方图匹配**（不同设备间的数据一致性**）。
    </p>
    <h4>
     <a id="232__120">
     </a>
     2.3.2 卫星遥感
    </h4>
    <h5>
     <a id="_121">
     </a>
     多光谱融合
    </h5>
    <blockquote>
     <p>
      多光谱融合：合并
      <strong>
       不同波段的图像
      </strong>
      （如红外与可见光）。
     </p>
    </blockquote>
    <h5>
     <a id="_123">
     </a>
     辐射校正
    </h5>
    <blockquote>
     <p>
      辐射校正：消除大气干扰。
     </p>
    </blockquote>
    <h4>
     <a id="233__126">
     </a>
     2.3.3 工业检测
    </h4>
    <h5>
     <a id="_127">
     </a>
     背景分割
    </h5>
    <p>
     背景分割：
     <strong>
      提取ROI
     </strong>
     （如产品表面缺陷检测）。
    </p>
    <h5>
     <a id="_129">
     </a>
     高动态范围
    </h5>
    <p>
     高动态范围（HDR）：
     <strong>
      处理过曝或欠曝区域
     </strong>
     。
    </p>
    <h2>
     <a id="_132">
     </a>
     三、工具与库
    </h2>
    <h3>
     <a id="31_OpenCV_133">
     </a>
     3.1 OpenCV
    </h3>
    <blockquote>
     <p>
      OpenCV：基础操作（
      <strong>
       滤波、几何变换、颜色空间转换
      </strong>
      ）。
     </p>
    </blockquote>
    <h3>
     <a id="32_PILPillow_135">
     </a>
     3.2 PIL/Pillow
    </h3>
    <blockquote>
     <p>
      PIL/Pillow：简单的图像增强（缩放、裁剪）。
     </p>
    </blockquote>
    <h3>
     <a id="33_scikitimage_137">
     </a>
     3.3 scikit-image
    </h3>
    <blockquote>
     <p>
      scikit-image：高级滤波和特征提取（如局部二值模式）。
     </p>
    </blockquote>
    <h3>
     <a id="34_TensorFlow_139">
     </a>
     3.4 TensorFlow
    </h3>
    <blockquote>
     <p>
      TensorFlow：tf.image模块
     </p>
    </blockquote>
    <h3>
     <a id="35_PyTorch_142">
     </a>
     3.5 PyTorch
    </h3>
    <blockquote>
     <p>
      PyTorch：torchvision.transforms
     </p>
    </blockquote>
    <h2>
     <a id="_145">
     </a>
     四、总结
    </h2>
    <p>
     图像预处理的选择需结合具体任务：
    </p>
    <h3>
     <a id="41__147">
     </a>
     4.1 目标检测
    </h3>
    <p>
     目标检测：可能更关注
     <strong>
      多尺度输入
     </strong>
     和
     <strong>
      颜色增强
     </strong>
     。
    </p>
    <h3>
     <a id="42__149">
     </a>
     4.2 语义分割
    </h3>
    <p>
     语义分割：需保留
     <strong>
      空间信息
     </strong>
     ，避免
     <strong>
      过度下采样
     </strong>
     。
    </p>
    <h3>
     <a id="43__151">
     </a>
     4.3 低光照场景
    </h3>
    <p>
     低光照场景：需Retinex算法或
     <strong>
      深度学习去噪
     </strong>
     （如AutoEncoder）。
    </p>
    <p>
     通过合理设计预处理流程，可显著提升
     <strong>
      模型泛化能力
     </strong>
     和
     <strong>
      收敛速度
     </strong>
     。
    </p>
