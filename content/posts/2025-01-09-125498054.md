# 5 个最好的免费开源反病毒工具（PC版）

## 5 个最好的免费开源反病毒工具 PC版

防病毒软件使人们能够轻松使用手机、平板电脑和计算机，能够在网络上更安全地浏览，而不用担心将私人信息传播给其他人（或任何病毒）

![AntiVirus](https://i-blog.csdnimg.cn/blog_migrate/233a831368c8d7f0d10779f62d77b83c.png)

一些好的防病毒软件还包含对来自其他计算机的威胁（如垃圾邮件，网上银行攻击，恶意URL，骗局和钓鱼攻击，在线身份（隐私），社交工程技术，高级持续威胁（APT）和僵尸网络DDoS攻击 的保护措施。

以下是免费开源的、且轻量级的前 5 个
[AntiVirus](https://iyouport.wixsite.com/iyouport/blog/search/posts?query=%23AntiVirus)
工具列表。无论您是初学者还是高级用户，这些都可以帮助您保持计算机的安全：

### 1.Open AntiVirus Project

![Open AntiVirus Project](https://i-blog.csdnimg.cn/blog_migrate/b87bbfa13d394db4cce8bba03712f9a5.png)
  
Open AntiVirus Project：其功能包括 VirusHammer，一款独立的病毒扫描程序，包含ScannerDaemon 和PatternFinder 的项目，提供防病毒保护和Squid-vscan

### 2.ClamAV

![ClamAV](https://i-blog.csdnimg.cn/blog_migrate/cc3e587f6c16c5cb4e7917dd2ecbefbb.png)

ClamAV Clam AntiVirus 或
[ClamAV](https://www.clamav.net/)
是一款免费的，跨平台开源防病毒软件工具包，用C 和C ++编写，并在 GNU 通用公共许可证下授权。4 月9 日发布，不仅用于 Unix，还适用AIX，BSD，HP-UX，Linux，macOS，OpenVMS，OSF（Tru64）和Solaris

它可以检测许多类型的恶意软件和病毒，并且可以在邮件服务器上将其用作服务器端电子邮件病毒扫描程序。从版本 0.97.5 开始，ClamAV 在 Microsoft Windows 上构建并运行，并且可以免费下载。

ClamAV 包含许多实用程序，如命令行扫描程序，自动数据库更新程序和可扩展的多线程守护程序，它们在共享库的反病毒引擎上运行。

### 3.ClamWin

![ClamWin](https://i-blog.csdnimg.cn/blog_migrate/e3d59bcc1523ca9f5bfd50efbeb4093d.png)

[ClamWin](http://www.clamwin.com/)
用于 Windows 的免费和开源防病毒工具，用 C ++和 Python编写，为 Clam AntiVirus 引擎提供图形用户界面。它是根据 GNU 通用公共许可证授权的。

### 4.Armadito Antivirus

![Armadito Antivirus](https://i-blog.csdnimg.cn/blog_migrate/6aabc9dc887821247174f56601482b42.png)

[Armadito Antivirus](https://github.com/armadito)
一款适用于服务器和 PC 的开源防病毒工具。它保护您的系统免受任何病毒和恶意软件的侵害，并为 Windows 机器提供可用的解决方案。它是通过基于 Web 的中央管理控制台创建的，可以通过直观的界面从任何位置远程使用，可以访问大量功能。其仪表板可以访问按需扫描、实时防护、威胁检测日志、技术支持等。

### 5.Moon Secure AV

![Moon Secure AV](https://i-blog.csdnimg.cn/blog_migrate/dccd582413daad580d85923292df6b7b.png)

Moon Secure AV 一款基于 GNU/GPL 协议的免费软件，使用 clamav engine 引擎，功能比较全，查、杀、防病毒，反木马和间谍软件，有防火墙，其功能还包括按需扫描恶意软件，实时保护，实时更新，资源占用率低，可在XP + 的所有 Windows 上运行。
