    <h4 style="margin-left:0px; margin-right:0px; text-align:justify">
     <strong>
      Ajax：异步的JavaScript和XML
     </strong>
    </h4>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     作用：
    </p>
    <ol>
     <li style="text-align:justify">
      数据交换
     </li>
     <li style="text-align:justify">
      异步交互
     </li>
    </ol>
    <h4>
     <span style="color:#1a439c">
      <strong>
       Axios：就是对原生Ajax进行封装，简化书写，快速开发
      </strong>
     </span>
    </h4>
    <p>
     使用逻辑：
    </p>
    <p>
     <img alt="" height="564" src="https://i-blog.csdnimg.cn/direct/3031ba1801fb4a58b8750eb90418cf0b.png" width="1166"/>
    </p>
    <p>
     首先要安装Axios，可以通过npm在项目中安装：
    </p>
    <ol>
     <li>
      打开命令行工具，进入项目根目录
     </li>
     <li>
      运行以下命令来安装Axios:
     </li>
    </ol>
    <pre><code>npm install axios
</code></pre>
    <ol>
     <li>
      安装完成后，你可以在项目中引入Axios来发送HTTP请求。在JavaScript文件中引入Axios的方法如下:
     </li>
    </ol>
    <pre><code class="language-javascript">const axios = require('axios');
</code></pre>
    <p>
     或者如果你是在浏览器环境中使用Axios，你可以通过
     <code>
      &lt;script&gt;
     </code>
     标签引入CDN：
    </p>
    <pre><code class="language-html">&lt;script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"&gt;&lt;/script&gt;
</code></pre>
    <p>
     这样就可以开始使用Axios来发送网络请求了。
    </p>
    <pre><code class="language-javascript">  axios({
    method:'GET',
    url:"http://localhost:3000/api/data"
  }).then((result)=&gt;{           //成功回调函数
    console.log(result.data);
  }).catch((err)=&gt;{             //失败回调函数
    alert(err);
  });</code></pre>
    <blockquote>
     <p>
      <span style="color:#be191c">
       <strong>
        method
       </strong>
      </span>
      :请求方式，GET/POST....
      <br/>
      <span style="color:#be191c">
       <strong>
        url
       </strong>
      </span>
      :请求路径
      <br/>
      <span style="color:#be191c">
       <strong>
        data
       </strong>
      </span>
      :请求数据(POST)
      <br/>
      <span style="color:#be191c">
       <strong>
        params
       </strong>
      </span>
      :发送请求时携带的url参数 如:...?key=val
     </p>
    </blockquote>
    <h3>
     <span style="color:#1a439c">
      <strong>
       Axios-请求方式别名
      </strong>
     </span>
    </h3>
    <p>
     为了方便起见，Axios已经为所有支持的请求方法提供了别名
    </p>
    <p>
     在Axios中，可以通过提供不同的方法来发送不同类型的HTTP请求。下面是一些常用的Axios请求别名方法：
    </p>
    <p>
     <code>
      axios.request(config)
     </code>
     ：发送自定义请求，可以定义请求的方法、URL、请求头、请求体等配置。
    </p>
    <pre><code class="language-javascript">axios.request({
  method: 'post',
  url: 'https://api.example.com/custom_request',
  data: {
    key: 'value'
  }
})
.then(response =&gt; {
  console.log(response.data);
})
.catch(error =&gt; {
  console.error(error);
});
</code></pre>
    <p>
     <code>
      axios.get(url[, config])
     </code>
     ：发送GET请求，配置可以包括params、headers等。
    </p>
    <pre><code class="language-javascript">axios.get('https://api.example.com/get_data', { params: { id: 123 } })
.then(response =&gt; {
  console.log(response.data);
})
.catch(error =&gt; {
  console.error(error);
});
</code></pre>
    <p>
     <code>
      axios.delete(url[, config])
     </code>
     ：发送DELETE请求，可以设置URL和请求配置。
    </p>
    <pre><code class="language-javascript">axios.delete('https://api.example.com/delete_data', { params: { id: 456 } })
.then(response =&gt; {
  console.log(response.data);
})
.catch(error =&gt; {
  console.error(error);
});
</code></pre>
    <p>
     <code>
      axios.post(url[, data[, config]])
     </code>
     ：发送POST请求，同时可以传递请求体数据和配置。
    </p>
    <pre><code class="language-javascript">axios.post('https://api.example.com/post_data', { key: 'value' }, { headers: { 'Content-Type': 'application/json' } })
.then(response =&gt; {
  console.log(response.data);
})
.catch(error =&gt; {
  console.error(error);
});
</code></pre>
    <p>
     <code>
      axios.put(url[, data[, config]])
     </code>
     ：发送PUT请求，可以传递URL、请求体数据和配置。
    </p>
    <pre><code class="language-javascript">axios.put('https://api.example.com/put_data', { key: 'value' })
.then(response =&gt; {
  console.log(response.data);
})
.catch(error =&gt; {
  console.error(error);
});
</code></pre>
    <p>
     通过使用这些请求别名方法，你可以更加方便地发送不同类型的HTTP请求。
    </p>
    <p>
    </p>
    <h3 style="margin-left:0px; margin-right:0px; text-align:justify">
     输出先后顺序：
    </h3>
    <p style="margin-left:0; margin-right:0; text-align:justify">
     <span style="color:#fe2c24">
      原因：Ajax是异步请求
     </span>
    </p>
    <p>
     <img alt="" height="222" src="https://i-blog.csdnimg.cn/direct/3eb5c603419644f6aae7ae0d01cf3d87.png" width="876"/>
    </p>
    <h4>
     解决方法：
    </h4>
    <p>
     <span style="color:#be191c">
      <strong>
       异步同步化（让代码从上往下按顺序执行）
      </strong>
     </span>
    </p>
    <p>
     可以通过async、await可以让异步变为同步操作。async就是来声明一个异步方法，await是用来等待异步任务执行
    </p>
