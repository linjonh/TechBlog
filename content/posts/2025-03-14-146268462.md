     1 OpenManus
    </h3>
    <h4>
     <a id="11__7">
     </a>
     1.1 简介
    </h4>
    <p>
     <strong>
      开发团队
     </strong>
     : MetaGPT 核心贡献者（5 人团队，3 小时完成开发）
     <br/>
     <strong>
      特点
     </strong>
     :
    </p>
    <ul>
     <li>
      <p>
       <strong>
        无需邀请码
       </strong>
       ，支持本地部署与实时反馈
      </p>
     </li>
     <li>
      <p>
       <strong>
        模块化 Agent 系统
       </strong>
       ：
      </p>
      <ul>
       <li>
        主代理（项目经理）
       </li>
       <li>
        规划代理（任务分解）
       </li>
       <li>
        工具调用代理（执行操作）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        多模型支持
       </strong>
       ：Claude 3.5、Qwen VL Plus 等，可扩展其他模型
      </p>
     </li>
     <li>
      <p>
       <strong>
        工具链
       </strong>
       ：浏览器自动化、Python 执行器、文件处理系统
       <br/>
       <strong>
        GitHub
       </strong>
       :
       <a href="https://github.com/mannaandpoem/OpenManus">
        mannaandpoem/OpenManus
       </a>
      </p>
     </li>
    </ul>
    <h4>
     <a id="12___24">
     </a>
     1.2 安装教程
    </h4>
    <p>
     1、 使用Conda安装
    </p>
    <pre><code>conda create -n open_manus python=3.12
conda activate open_manus
git clone https://github.com/mannaandpoem/OpenManus.git
cd OpenManus
pip install -r requirements.txt
</code></pre>
    <p>
     2、环境配置
     <br/>
     OpenManus 需要配置 LLM APIs
     <br/>
     （1）创建一个config.toml文件，也可以用以下命令创建文件：
    </p>
    <pre><code>cp config/config.example.toml config/config.toml
</code></pre>
    <p>
     （2）编辑
     <code>
      config/config.toml
     </code>
     ，添加t API keys 和用户配置:
    </p>
    <pre><code># Global LLM configuration
[llm]
model = "gpt-4o"
base_url = "https://api.openai.com/v1"
api_key = "sk-..."  # Replace with your actual API key
max_tokens = 4096
temperature = 0.0

# Optional configuration for specific LLM models
[llm.vision]
model = "gpt-4o"
base_url = "https://api.openai.com/v1"
api_key = "sk-..."  # Replace with your actual API key
</code></pre>
    <h4>
     <a id="13__62">
     </a>
     1.3 运行
    </h4>
    <p>
     运行命令：
    </p>
    <pre><code>python main.py
</code></pre>
    <p>
     也可以尝试运行非稳定版本，运行命令：
    </p>
    <pre><code>python run_flow.py
</code></pre>

    <h4>
     <a id="22___100">
     </a>
     2.2 安装教程
    </h4>
    <p>
     1、 使用Conda安装
    </p>
    <pre><code># Clone github repo
git clone https://github.com/camel-ai/owl.git

# Change directory into project directory
cd owl

# Create a conda environment
conda create -n owl python=3.10

# Activate the conda environment
conda activate owl

# Option 1: Install as a package (recommended)
pip install -e .

# Option 2: Install from requirements.txt
pip install -r requirements.txt

# Exit the conda environment when done
conda deactivate
</code></pre>
    <p>
     2、设置环境变量
    </p>
    <p>
     OWL需要将变量 API keys 与不同的服务进行交互. 找到owl/.env_template文件，这个文件中包含了所有的 API keys接口。
    </p>
    <p>
     （1）复制并重命名文件：
    </p>
    <pre><code>cd owl
cp .env_template .env
</code></pre>
    <p>
     （2）配置API Keys
    </p>
    <p>
     打开
     <code>
      .env
     </code>
     文件，插入你的 API keys 。
    </p>
    <h4>
     <a id="23__142">
     </a>
     2.3 运行
    </h4>
    <p>
     尝试MCP（Model Context Protocal）交互
    </p>
    <pre><code># Set up MCP servers (one-time setup)
npx -y @smithery/cli install @wonderwhy-er/desktop-commander --client claude
npx @wonderwhy-er/desktop-commander setup

# Run the MCP example
python owl/run_mcp.py
</code></pre>
    <p>
     可以使用基础的命令运行：
    </p>
    <pre><code>python owl/run.py
</code></pre>

     3.2 安装教程和运行
    </h4>
    <p>
     在Docker中运行OpenHands：
    </p>
    <pre><code>docker pull docker.all-hands.dev/all-hands-ai/runtime:0.28-nikolaik

docker run -it --rm --pull=always \
    -e SANDBOX_RUNTIME_CONTAINER_IMAGE=docker.all-hands.dev/all-hands-ai/runtime:0.28-nikolaik \
    -e LOG_ALL_EVENTS=true \
    -v /var/run/docker.sock:/var/run/docker.sock \
    -v ~/.openhands-state:/.openhands-state \
    -p 3000:3000 \
    --add-host host.docker.internal:host-gateway \
    --name openhands-app \
    docker.all-hands.dev/all-hands-ai/openhands:0.28
</code></pre>
    <p>
     然后打开浏览器，输入
     <a href="http://localhost:3000/" rel="nofollow">
      http://localhost:3000
