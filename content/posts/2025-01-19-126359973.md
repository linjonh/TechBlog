    <blockquote>
     <p>
      最近微软的开发工具VSCode频繁更新Java支持，又是支持SpringBoot，又是支持Lombok，让我不禁好奇VSCode是不是也能胜任Java开发了？于是抽空体验了一把，确实完全可以胜任，Java开发者又有了新选择，不仅好用而且开源！今天带大家体验下VSCode，体验超越Eclipse，不输IDEA！
     </p>
    </blockquote>
    <p>
     SpringBoot实战电商项目mall（50k+star）地址：
     <a href="https://github.com/macrozheng/mall">
      https://github.com/macrozheng/mall
     </a>
    </p>
    <h3>
     <a id="VSCode_4">
     </a>
     VSCode简介
    </h3>
    <p>
     VSCode全称Visual Studio Code，是微软开源的一款集成开发工具，目前在Github上已有
     <code>
      135K+Star
     </code>
     。VSCode号称能重新定义代码编辑器，不仅免费而且开源，能在多个平台上运行使用。VSCode对比IDEA不仅更轻量级，而且构建与响应速度也更快！
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/10e2109db70c961eed4b1e51e61b0b01.png"/>
    </p>
    <p>
     再来看下使用VSCode开发Java项目的效果图，界面还是挺炫酷的！
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/164b17d2825f2568b4a116fbe600855f.png"/>
    </p>
    <h3>
     <a id="_14">
     </a>
     下载及安装
    </h3>
    <blockquote>
     <p>
      接下来我们将下载并安装VSCode，让它具有开发Java应用的能力。
     </p>
    </blockquote>
    <ul>
     <li>
      首先我们需要下载VSCode的安装包，下载压缩包版本简单又快捷，下载地址：https://code.visualstudio.com/Download
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/adc867c4508c110bb56214ab3968855d.png"/>
    </p>
    <ul>
     <li>
      下载完成后解压到指定目录，双击
      <code>
       .exe
      </code>
      文件即可启动；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9e327239c1dc86c765608b93fd0d4f8c.png"/>
    </p>
    <ul>
     <li>
      如果我们启动的系统为中文环境，VSCode会提示你下载中文插件，果断下载；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6cbaad2e4d78e15d375abc1229770a61.png"/>
    </p>
    <ul>
     <li>
      接下来就是安装各种插件来方便我们开发了，首先安装微软提供的Java全家桶插件
      <code>
       Extension Pack for Java
      </code>
      ，此插件包含6个基本的Java扩展插件，可以满足我们开发Java程序的基本需求；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2f012dc37e79acd844d67696f65bcecc.png"/>
    </p>
    <ul>
     <li>
      这个Java全家桶插件目前已有
      <code>
       1390w
      </code>
      下载量，看样子很多Java开发者都已经开始使用VSCode了；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/169eec5b7e96f1c0cde356e401d15e83.png"/>
    </p>
    <ul>
     <li>
      由于我们平时使用SpringBoot进行开发，还需要安装SpringBoot全家桶插件，此插件包含3个SpringBoot扩展插件；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1a23eae76b2d665d4dc3a6283954496a.png"/>
    </p>
    <ul>
     <li>
      本想再安装下Lombok插件，后来发现VSCode已经内置了Lombok支持，可以在设置中进行开启，安装完这些插件后就可以开始开发Java程序了。
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f9c1143f17dee5dcd48630197f77c1b4.png"/>
    </p>
    <h3>
     <a id="_46">
     </a>
     使用
    </h3>
    <blockquote>
     <p>
      接下来将以我的
      <a href="https://github.com/macrozheng/mall">
       开源电商项目mall
      </a>
      为例，来聊聊如何使用VSCode开发SpringBoot应用。
     </p>
    </blockquote>
    <h4>
     <a id="_50">
     </a>
     配置字体大小
    </h4>
    <p>
     在使用代码编辑器之前，很多朋友都会有配置字体大小的习惯，VSCode配置字体大小的方式十分简单，直接在
     <code>
      外观
     </code>
     设置中点击
     <code>
      放大
     </code>
     或
     <code>
      缩小
     </code>
     按钮即可。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/2bbe26584d337857c57c03ac09bba57a.png"/>
    </p>
    <h4>
     <a id="Maven_56">
     </a>
     导入Maven项目
    </h4>
    <ul>
     <li>
      由于VSCode只是个开发工具，它不会像IDEA那样直接集成Maven，这里我们需要安装好Maven并配置环境变量，首先下载Maven的安装包，下载地址：https://maven.apache.org/download.cgi
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ac5b1b5db666df763b728e714d911a23.png"/>
    </p>
    <ul>
     <li>
      下载完成后解压到指定目录，然后配置系统环境变量；
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token assign-left variable">MAVEN_HOME</span><span class="token operator">=</span>D:<span class="token punctuation">\</span>developer<span class="token punctuation">\</span>env<span class="token punctuation">\</span>apache-maven-3.8.6
<span class="token assign-left variable">Path</span><span class="token operator">+=</span>%MAVEN_HOME%<span class="token punctuation">\</span>bin
</code></pre>
    <ul>
     <li>
      接下来导入mall项目的代码，下载地址：https://github.com/macrozheng/mall
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b88eec1df247fc8ea364aa19c91cc611.png"/>
    </p>
    <ul>
     <li>
      解压到指定目录，然后通过
      <code>
       打开文件夹
      </code>
      按钮打开目录，接下来项目就会自动构建了，个人感觉构建速度比IDEA快多了；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/78407525eb2dab7671fc6bb8eb37bccf.png"/>
    </p>
    <ul>
     <li>
      我们来看下代码编辑界面，还是很炫酷的！
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/164b17d2825f2568b4a116fbe600855f.png"/>
    </p>
    <h4>
     <a id="_81">
     </a>
     运行调试项目
    </h4>
    <ul>
     <li>
      运行和调试代码的方式非常简单，直接点击右上角的运行按钮即可；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/506a85dcb3a2d529bf4b07476bf871b1.png"/>
    </p>
    <ul>
     <li>
      由于我的mall项目使用的SpringBoot，还可以使用
      <code>
       Spring Boot Dashboard
      </code>
      来运行，是不是和在IDEA中运行项目有点类似；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/841261c177399d54db8211d759feac42.png"/>
    </p>
    <ul>
     <li>
      我们还可以通过打断点的方式进行调试，左侧变量区将显示所有局部变量的值，可以通过顶部按钮进行跳过、跳出断点操作，鼠标悬浮在变量上时会显示变量的值，IDEA里有的功能VSCode基本也有了！
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/71d0fb9d64a08ecbfa69ab75e2a2cb88.png"/>
    </p>
    <h4>
     <a id="_95">
     </a>
     浏览项目代码
    </h4>
    <ul>
     <li>
      我们平时看代码时经常需要按类名进行搜索，在VSCode中使用
      <code>
       Ctrl+T
      </code>
      快捷键就能实现；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1901bc7193aee791d768c14ad3e96bf4.png"/>
    </p>
    <ul>
     <li>
      如果想直接按照文件名称进行搜索，使用
      <code>
       Ctrl+P
      </code>
      快捷键即可实现；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/27b6b1f6223fab4ada2b12e02fda1bfc.png"/>
    </p>
    <ul>
     <li>
      使用
      <code>
       Ctrl+鼠标左键
      </code>
      可以直接跳转查看类的源码；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/8ed1bd256bb7b1d2065480c77fa057ab.png"/>
    </p>
    <ul>
     <li>
      有时候我们查看第三方库的源码时会没有注释，而在IDEA中我们点击类名会自动下载源码，此时只要在项目的根目录下通过如下命令下载源码即可；
     </li>
    </ul>
    <pre><code class="prism language-bash">mvn dependency:sources -DdownloadSources<span class="token operator">=</span>true -DdownloadJavadocs<span class="token operator">=</span>true
</code></pre>
    <ul>
     <li>
      Maven默认只下载
      <code>
       xxx.jar
      </code>
      这种Jar包，像
      <code>
       xxx-sources.jar
      </code>
      这种带注释的源码包需要自行下载；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4a8381870174f19a2f8a927df0d61c13.png"/>
    </p>
    <ul>
     <li>
      由于默认安装了微软的
      <code>
       IntelliCode
      </code>
      插件，VSCode的代码提示还是很智能的；
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6bf082ceacb6e6ccc7e814e2f25a9f19.png"/>
    </p>
    <ul>
     <li>
      我们还可以通过左侧大纲视图快速找到类中的变量和方法，IDEA中大部分常用功能VSCode基本都能支持，感兴趣的小伙伴可以自行体验下。
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/8b894df83195bbce6b612e2f80b9dcf0.png"/>
    </p>
    <h3>
     <a id="_127">
     </a>
     快捷键
    </h3>
    <blockquote>
     <p>
      VSCode的功能还是很强大的，用来进行Java应用开发完全够用了。这里我根据平时使用IDEA的快捷键，整理了一套VSCode的常用快捷键，大家可以尝试下。
     </p>
    </blockquote>
    <ul>
     <li>
      查看类源码：
      <code>
       Ctrl+鼠标左键
      </code>
     </li>
     <li>
      快捷键命令搜索：
      <code>
       Ctrl+Shift+P+命令名称
      </code>
     </li>
     <li>
      按文件名搜索文件：
      <code>
       Ctrl+P
      </code>
     </li>
     <li>
      根据类名查找类：
      <code>
       Ctrl+T
      </code>
     </li>
     <li>
      注释代码：
      <code>
       Ctrl+/
      </code>
     </li>
     <li>
      开启/关闭侧边栏：
      <code>
       Ctrl+B
      </code>
     </li>
     <li>
      全局查找并替换：
      <code>
       Ctrl+Shift+F
      </code>
     </li>
     <li>
      文件内查找：
      <code>
       Ctrl+F
      </code>
     </li>
     <li>
      修改变量或类名：
      <code>
       Fn
      </code>
     </li>
     <li>
      查找变量或类的引用：
      <code>
       Shift+F12
      </code>
     </li>
     <li>
      查找接口或方法的实现：
      <code>
       Ctrl+F12
      </code>
     </li>
     <li>
      格式化代码：
      <code>
       Shift+Alt+F
      </code>
     </li>
     <li>
      重构抽取变量或方法：
      <code>
       Ctrl+Shift+R
      </code>
     </li>
     <li>
      回退到上一个操作：
      <code>
       Alt+左箭头
      </code>
     </li>
     <li>
      前进到下一个操作：
      <code>
       Alt+右箭头
      </code>
     </li>
     <li>
      提示可能的操作，如变量生成：
      <code>
       Alt+.
      </code>
     </li>
     <li>
      移动一行代码：
      <code>
       Alt+上箭头
      </code>
      或
      <code>
       Alt+下箭头
      </code>
     </li>
     <li>
      删除当前行代码：
      <code>
       Ctrl+Shift+K
      </code>
     </li>
    </ul>
    <h3>
     <a id="_150">
     </a>
     总结
    </h3>
    <p>
     今天尝试了下使用VSCode开发SpringBoot应用，确实能够胜任！VSCode不仅界面炫酷、功能强大，而且插件市场的插件也非常丰富。对比IDEA来说，它的构建速度和响应速度更快。虽然也有些小细节做的不如IDEA，比如控制台不能输出彩色的日志、需要手动下载带注释的第三方库源码。不过VSCode功能更新非常快，相信很快就能成为更加出色的Java开发工具！
    </p>
