    <p>
     <strong>

       <span style="background-color:#efedf6">
        字符分类和转换函数
       </span>

    <h2>
     一、字符测试函数
    </h2>
    <h3>
     <br/>
     1.  isalnum(int c)
    </h3>
    <h3>
     <strong>
      <span style="background-color:#e7fafa">
       <span style="color:#fe2c24">
        判断
       </span>
       字符 c 是否为字母或数字（ a-z、A-Z、0-9 ）
      </span>
     </strong>
    </h3>
    <blockquote>
     <p>
      <span style="color:#956fe7">
       <strong>
        isalnum(int c)
       </strong>
      </span>
      <br/>
      <span style="color:#fe2c24">
       <span style="background-color:#e7fafa">
       </span>
      </span>
      <strong>
       <span style="background-color:#e7fafa">
        <span style="color:#fe2c24">
         判断
        </span>
        字符 c 是否为字母或数字（ a-z、A-Z、0-9 ）
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="background-color:#e7fafa">
        是 则返回非0值，否 则返回0；
       </span>
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-cpp">char a;
if(isalnum(a)) //判断字符a是否为字母或数字（ a-z、A-Z、0-9 ）。
{
    printf("a是字母或数字");
}
else
{
    printf("a不是字母或数字");
}
</code></pre>
    <blockquote>
     <p>
      <strong>
       验证用户名是否合法时常用此函数。
      </strong>
     </p>
    </blockquote>

char a;

    <h2>
     二、字符转换函数
    </h2>
    <h3>
     <br/>
     1.  tolower(int c)
    </h3>
    <h3>
     <strong>
      <span style="background-color:#e7fafa">
       <span style="color:#fe2c24">
        将大写字母转换为小写字母
       </span>
       ，若`c`
       <span style="color:#ff9900">
        不是大写字母则返回原值。
       </span>
      </span>
     </strong>
    </h3>
    <blockquote>
     <p>
      <br/>
      <strong>
       <span style="background-color:#e7fafa">
        <span style="color:#fe2c24">
         将大写字母转换为小写字母
        </span>
        ，若`c`
        <span style="color:#ff9900">
         不是大写字母则返回原值。
        </span>
       </span>
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-cpp">char c = A;
char c = tolower(c); // 将 A 转换成 a</code></pre>
    <blockquote>
     <p>
      <strong>
       统一用户名格式时使用。
      </strong>
     </p>
    </blockquote>
    <h3>
     2. toupper(int c)
    </h3>
    <h3>
     <strong>
      <span style="background-color:#edf6e8">
      </span>
      <span style="color:#fe2c24">
       <span style="background-color:#edf6e8">
        将小写字母转换为大写字母
       </span>
      </span>
      <span style="background-color:#edf6e8">
       ，若 'c'
      </span>
      <span style="color:#ff9900">
       <span style="background-color:#edf6e8">
        非小写字母时返回原值。
       </span>
      </span>
     </strong>
    </h3>
    <blockquote>
     <p>
      <br/>
      <strong>
       <span style="background-color:#edf6e8">
       </span>
       <span style="color:#fe2c24">
        <span style="background-color:#edf6e8">
         将小写字母转换为大写字母
        </span>
       </span>
       <span style="background-color:#edf6e8">
        ，若 'c'
       </span>
       <span style="color:#ff9900">
        <span style="background-color:#edf6e8">
         非小写字母时返回原值。
        </span>
       </span>
      </strong>
     </p>
    </blockquote>
    <pre><code class="language-cpp">char c = a;
char c = toupper(c); // 将 a 转换成 A</code></pre>
    <blockquote>
     <p>
      <strong>
       适用于生成统一格式的字符串。
      </strong>
     </p>
    </blockquote>

    <h2 style="background-color:transparent">
     三、应用场景
    </h2>
    <p>
     <br/>
     <strong>
      <span style="color:#fe2c24">
       - 输入验证：
      </span>
      如使用 isdigit 确保用户输入的是数字，或 isalpha 限制只能输入字母。
      <br/>
      <span style="color:#fe2c24">
       - 字符串处理：
      </span>
      通过 tolower / toupper 统一字符串大小写，或利用 isspace 分割单词。
      <br/>
      <span style="color:#fe2c24">
       - 数据解析：
      </span>
      例如用 isxdigit 解析十六进制颜色代码，或 ispunct 过滤标点符号。
     </strong>
    </p>
    <p>
    </p>
    <h2>
     四、注意事项
    </h2>
    <p>
     <br/>
     <span style="color:null">
      <strong>
       - 参数范围：
      </strong>
     </span>
     <strong>
      <span style="color:#fe2c24">
       传入的`int`值需在`0-255`或`EOF`范围内
      </span>
      <span style="color:null">
       ，否则可能导致未定义行为。
       <br/>
       - 本地化影响：
      </span>
      <span style="color:#fe2c24">
       部分函数（如 isspace）的行为可能受本地化设置影响，需注意跨平台兼容性。
      </span>
     </strong>
    </p>
    <p>
     <strong>
      通过合理使用这些函数，可以避免手动编写字符判断逻辑，提升代码的可读性和可维护性。
     </strong>
    </p>
