### 5.值得注意

一个文件夹下不能写多个main函数，只能有一个，不然多了会报错，这是跟vs是一样的基本语法。

### 6.案例测试

判断一个数字是偶数还是奇数：

```c
#include<stdio.h>
int main()
{
    int num;

    printf("输入一个数字 : ");
    scanf("%d",&num);

    (num%2==0)?printf("偶数"):printf("奇数");

}

```

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/6195e03ec2ec69d12bf6f800da151088.png)
  
我也不清楚这个多了的 format 啥意思，但是把代码复制粘贴出来并没有 format,因此我认为这个 format 是 CLion 是输入输出自带的，但是又实际不存在的。
