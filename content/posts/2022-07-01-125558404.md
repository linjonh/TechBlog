### （1）用起泡法对10个数由小到大排序. 即将相邻两个数比较,将小的调到前头.

建立qipao函数：

```python
function y = qipao(x)
n = length(x);%求数组个数
%气泡排序
for a = 1:n
    for b = 1:n-a
    if(x(b)>x(b+1))
        c = x(b);
        x(b) = x(b+1);
        x(b+1) = c;
    end
    end
end
y = x
        

```

### （3）编程求 ∑ n = 1 20 n ! \displaystyle\sum\_{n=1}^{20} n! n = 1 ∑ 2 0 ​ n !

建立jiec函数：

```python
function f = jiec(n)
sum = 0;

for i = 1:n
    p = 1;
    j = 1;
    for j = 1:i
        p = p * j;
        j = j - 1;
    end
    sum = sum + p;
    i = i + 1;
end
f = sum;
end


```

建立fantanh函数：

```python
function [s,h] = fantanh(n)
z = 100;
s = 100;
for i = 1:n
    z = z/2;
    s = s + 2*z;
end
h = z/2
```

## 7. 在同一平面的两个窗口中分别画出心形线和马鞍面

要求：在图形上加格栅、图例和标注；定制坐标；以不同的角度观察马鞍面。

```python
%心形
a=0.01
t=0:0.01:2*pi;
x1=a.*(2.*cos(t)-cos(2*t));
y1=a.*(2.*sin(t)-sin(2*t));
%马鞍面
x = linspace(-10,10);
y = linspace(-10,10);
[X,Y] = meshgrid(x,y);%生成网格
Z = X.^2 - Y.^2;
subplot(1,2,1);plot(x1,y1);title('心型图');
subplot(1,2,2);mesh(X,Y,Z);title('马鞍面');

```
