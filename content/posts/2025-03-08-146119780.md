### 摘要

随着旅游业的快速发展，度假村作为旅游目的地的核心，其管理效率和客户体验成为影响竞争力的关键因素。基于Asp.net的度假村管理系统采用先进的信息技术，为度假村提供了全面的数字化管理解决方案。该系统涵盖了预订管理、客户信息管理、住宿与餐饮服务等多个方面，通过在线预订和实时数据分析，实现了对度假村运营的全面优化。利用Asp.net框架的强大功能，系统不仅提高了度假村的管理效率，还显著提升了客户体验，为度假村在激烈的市场竞争中赢得了优势。

### 研究意义

基于Asp.net的度假村管理系统的研究具有重要意义。首先，它解决了传统度假村管理方式中存在的诸多挑战，如预订管理混乱、客户服务响应慢、资源调度不合理等问题。通过数字化手段，系统实现了对度假村各项业务的精准管理，提高了运营效率。其次，该系统提升了客户体验。游客可以通过在线预订、移动端访问等功能，更加便捷地规划和享受度假村的服务。系统还提供了实时信息和个性化的推荐，增强了游客的满意度和忠诚度。此外，该系统的研究还有助于推动旅游业的信息化进程，为其他旅游企业提供了可借鉴的范例。通过不断优化和完善，该系统有望为旅游业的繁荣发展注入新的活力。

### 研究目的

本研究的主要目的是开发一套基于Asp.net的度假村管理系统，以解决度假村管理面临的诸多挑战。具体而言，研究旨在通过数字化手段实现对度假村运营的全面优化，提高管理效率和客户体验。系统需要涵盖预订管理、客户信息管理、住宿与餐饮服务等多个方面，以满足度假村日常运营的需求。同时，研究还注重系统的可扩展性和易用性，以便在未来的发展中能够持续适应度假村业务的变化和升级。通过本研究，我们期望能够为度假村提供一个高效、智能、便捷的管理工具，助力度假村在激烈的市场竞争中脱颖而出。

[HttpPost]
public IActionResult Login(LoginModel model)
{
    // 验证用户信息
    var user = _userService.ValidateUser(model.Username, model.Password);
    if (user == null)
    {
        return Json(new { success = false, message = "用户名或密码错误" });
    }

    // 设置用户会话信息
    HttpContext.Session.SetString("UserId", user.Id.ToString());
    HttpContext.Session.SetString("UserName", user.Username);

    return Json(new { success = true, message = "登录成功" });
}
