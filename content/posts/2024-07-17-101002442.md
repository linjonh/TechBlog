# 阿里大学开放 11 门免费 Python 视频课程

Python 现在是世界编程语言排名第 3，学起来非常容易上手，很多运营、产品的小伙伴都在自学。

阿里大学开放了 11 门免费 Python 视频课程，从入门语法到爬虫实战，可以说是面面俱到了。

Python 语言近几年越来越火，语言使用率占比节节攀升。

我们知道
**Python 现在稳居世界编程语言前三名，在 PYPL 语言流行指数上更是稳居第一**
，可见 python 的适用范围、受众基础、影响力都非常强大，是非常适合深入研究学习的一门语言。

因为 Python 的普及程度如此高，广大的技术小伙伴都有这方面的学习需求，因此阿里云大学开发了 python 系列教学，并免费开放给所有阿里用户。

这个系列教学，特点是：
**免费、视频教学、附带测试**
。

11 门教学由浅入深，从环境搭建、语法入门、到网络编程、框架开发、爬虫实战等，适合不同层次的学习需求。

**介绍一下教学内容：**

《Python 环境搭建与语法入门》：Python 介绍、语法入门、数据类型、流程控制语句等入门操作。

《Python 语言基础：列表、元组、字典、集合》：Python 的 4 种基本数据结构列表（List）、元组（Tuple）、字典（Dictionary）和集合（Set）。

《Python 函数、面向对象、异常处理》：Python 的函数、面向对象编程、模块、异常处理、文件操作等高级编程方法。

《Python 网络编程》：Socket、TCP 开发实战。

《Python Web 开发基础》：网站 B/S 开发基础学习。

《Python Web 框架 Flask 快速入门》：学习轻量级的 Flask 开发框架，也被称为“微框架”。

《Python Web 框架 Django 快速入门》：Django 是开源的 Web 开发框架，能够简便、快速的开发数据库驱动的网站。

《Python 爬虫实战》：Python 爬虫协议、数据爬取、常用库学习。

《Python 数据可视化库 Matplotlib 快速入门》：Matplotlib 是一个 Python 的 2D 绘图库，简单易用。

《Python 科学计算库 NumPy 快速入门》：NumPy（Numeric Python）是一个用 Python 实现的科学计算库。

《Python 数据分析库 Panda 快速入门》：Pandas 是 Python 的一个数据分析包，Pandas 纳入了大量库和一些标准的数据模型，提供了高效地操作大型数据集所需的工具。

**这十一门教学从入门语法开始，循序渐进，到最后掌握强大的实战技巧，非常适合自学 Python 的小伙伴们。**
