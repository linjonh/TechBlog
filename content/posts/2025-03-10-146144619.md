    <h3>
     <a id="Homebrew_0">
     </a>
     一、安装Homebrew

    <h3>
     <a id="Homebrewminiconda_3">
     </a>
     二、使用Homebrew安装miniconda
    </h3>
    <pre><code class="prism language-shell">brew search conda
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bcf3878dfd0c47028935b5f66bb7fa43.png"/>
    </p>
    <pre><code class="prism language-shell">brew <span class="token function">install</span> miniconda
</code></pre>
    <p>
     安装完成后的截图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e33623cbfc8247399cfbafcf8e8e19a9.png"/>
    </p>
    <pre><code class="prism language-shell"><span class="token comment"># 查看是否安装成功</span>
brew list
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/157d3b94a6ea4c75bcdbda47b7873b45.png"/>
    </p>
    <h4>
     <a id="_19">
     </a>
     环境变量（无需手动配置）
    </h4>
    <p>
     先执行命令看能不能正常返回，如果不能正常返回结果，需要配置环境变量
    </p>
    <pre><code class="prism language-shell">conda info
conda info <span class="token parameter variable">-e</span>
</code></pre>
    <p>
     正常返回的截图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/48ec9a55dd44474f997e6548e176d9ac.png">
      <br/>
      如果不能正产返回结果，按照以下步骤配置环境变量
     </img>
    </p>
    <pre><code class="prism language-shell"><span class="token function">vim</span> ~/.bash_profile
<span class="token comment"># 添加到配置文件，并保存退出</span>
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token string">"/opt/homebrew/Caskroom/miniconda/base/bin:<span class="token environment constant">$PATH</span>"</span>
<span class="token comment"># 使配置生效</span>
<span class="token builtin class-name">source</span> ~/.bash_profile
</code></pre>
    <h3>
     <a id="IDEAminiconda_36">
     </a>
     三、IDEA集成miniconda环境
    </h3>
    <pre><code class="prism language-shell"><span class="token comment"># 创建虚拟环境 命名为myenv2 python版本为3.8</span>
conda create <span class="token parameter variable">--name</span> myenv2 <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.8</span>
</code></pre>
