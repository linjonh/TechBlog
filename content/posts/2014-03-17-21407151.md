     [Phonegap+Sencha Touch] 移动开发10 调用focus方法使输入框获得焦点，不弹出软键盘的解决方法
    </h1>

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <span style="font-size:13px">
     在phonegap打包的web应用中，调用输入框的.focus()方法的时候，输入框是获得焦点了，但是不会弹出软键盘。
    </span>
    <div style="word-wrap:break-word; font-size:13px">
     1、对于安卓，使用phonegap插件：
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <a href="https://github.com/cjpearson/cordova-plugin-keyboard" rel="noopener noreferrer" style="word-wrap:break-word; color:rgb(0,153,0); text-decoration:none" target="_blank">
      https://github.com/cjpearson/cordova-plugin-keyboard
     </a>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     使用的时候，只要在输入框调用focus()方法之后，再调用Keyboard.show()即可。
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     <br style="word-wrap:break-word"/>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     2、对于IOS，更改
     <span style="color:#ff0000; word-wrap:break-word">
      phonegap\platforms\ios\{项目名称}\config.xml
     </span>
     文件，
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     找到
     <span style="color:#8b0000; word-wrap:break-word">
      &lt;preference name
     </span>
     =
     <span style="color:#008080; word-wrap:break-word">
      "KeyboardDisplayRequiresUserAction"
     </span>
     <span style="color:#8b0000; word-wrap:break-word">
      value
     </span>
     =
     <span style="color:#008080; word-wrap:break-word">
      "true"
     </span>
     <span style="color:#8b0000; word-wrap:break-word">
      /&gt;
     </span>
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     把true改为
     <span style="color:#ff0000; word-wrap:break-word">
      false
     </span>
     即可。
    </div>
    <div style="word-wrap:break-word; font-size:13px">
     这个配置项的意思是：
     <span style="word-wrap:break-word; font-family:Helvetica,Arial,sans-serif">
      如果设置为true，用户必须手动的点击页面上的元素才能显示键盘；如果设置为false，一个元素的焦点事件就能弹出软键盘。
     </span>
     <br/>
    </div>
