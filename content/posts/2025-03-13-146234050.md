     书籍：《Visual C++ 2017从入门到精通》的2.3.8 Win32控件编程

     环境：visual studio 2022

    <h3>
     ​
     <strong>
      1. 函数功能与用途
     </strong>
    </h3>
    <p>
     <code>
      SetWindowPos()
     </code>
     是 Windows API 中用于
     <span style="color:#fe2c24">
      动态调整窗口属性的核心函数
     </span>
     ，支持以下操作：
    </p>
    <ul>
     <li>
      ​
      <strong>
       改变窗口位置
      </strong>
      ：通过坐标参数
      <code>
       X
      </code>
      和
      <code>
       Y
      </code>
      定义窗口左上角的新位置。
     </li>
     <li>
      ​
      <strong>
       调整窗口尺寸
      </strong>
      ：通过
      <code>
       cx
      </code>
      和
      <code>
       cy
      </code>
      参数设置窗口的宽度和高度。
     </li>
     <li>
      ​
      <strong>
       控制窗口 Z 顺序
      </strong>
      ：通过
      <code>
       hWndInsertAfter
      </code>
      参数或
      <code>
       SWP_NOZORDER
      </code>
      标志调整窗口在屏幕上的
      <span style="color:#fe2c24">
       叠放层次
      </span>
      。
     </li>
     <li>
      ​
      <strong>
       修改窗口状态
      </strong>
      ：如隐藏/显示窗口（
      <code>
       SWP_HIDEWINDOW
      </code>
      ）、禁止激活（
      <code>
       SWP_NOACTIVATE
      </code>
      ）等。
     </li>
    </ul>
    <h3 style="background-color:transparent">
     ​
     <strong>
      2. 函数原型
     </strong>
    </h3>
    <pre><code class="language-cpp">BOOL SetWindowPos(
    HWND hWnd,         // 窗口句柄
    HWND hWndInsertAfter, // Z 顺序参考窗口
    int X,              // 新位置 X 坐标（客户坐标）
    int Y,              // 新位置 Y 坐标（客户坐标）
    int cx,             // 新宽度（像素）
    int cy,             // 新高度（像素）
    UINT uFlags        // 控制标志组合
);</code></pre>
    <h3>
     ​
     <strong>
      3. 关键参数解析
     </strong>
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        参数
       </th>
       <th>
        说明
       </th>
       <th>
        来源
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        ​
        <strong>
         hWndInsertAfter
        </strong>
       </td>
       <td>
        Z 顺序参考窗口，可选值：
        <br/>
        -
        <code>
         HWND_TOPMOST
        </code>
        （置顶）
        <br/>
        -
        <code>
         HWND_BOTTOM
        </code>
        （置底）
       </td>
       <td>
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         uFlags
        </strong>
       </td>
       <td>
        控制行为标志，常用组合：
        <br/>
        -
        <code>
         SWP_NOZORDER
        </code>
        ：忽略
        <code>
         hWndInsertAfter
        </code>
        ，保持当前 Z 顺序
        <p>
         -
         <code>
          SWP_SHOWWINDOW
         </code>
         ：显示窗口
        </p>
        <p>
         -
         <code>
          SWP_NOREDRAW
         </code>
         ：禁止自动重绘
        </p>
       </td>
       <td>
       </td>
      </tr>
     </tbody>
    </table>
    <h3>
     ​
     <strong>
      4. 使用注意事项
     </strong>
    </h3>
    <ul>
     <li>
      ​
      <strong>
       权限限制
      </strong>
      ：调用进程需拥有窗口句柄的权限，否则可能失败。
     </li>
     <li>
      ​
      <strong>
       Z 顺序冲突
      </strong>
      ：若窗口已为
      <code>
       TOPMOST
      </code>
      ，设置
      <code>
       HWND_NOTOPMOST
      </code>
      会失效。
     </li>
     <li>
      ​
      <strong>
       多显示器适配
      </strong>
      ：需结合屏幕坐标系使用，例如通过
      <code>
       ScreenWidth/2
      </code>
      定位副显示器。
     </li>
     <li>
      ​
      <strong>
       消息触发差异
      </strong>
      ：与
      <code>
       MoveWindow()
      </code>
      不同，
      <code>
       SetWindowPos()
      </code>
      仅发送
      <code>
       WM_WINDOWPOSCHANGED
      </code>
      消息，需手动处理其他事件（如重绘）。
     </li>
    </ul>
    <h3>
     ​
     <strong>
      5. 典型应用场景
     </strong>
    </h3>
    <ol>
     <li>
      ​
      <strong>
       窗口置顶/取消置顶
      </strong>
      ：
      <pre><code class="language-cpp">SetWindowPos(hWnd, HWND_TOPMOST, 0, 0, 0, 0, SWP_NOMOVE | SWP_NOSIZE); // 置顶
SetWindowPos(hWnd, HWND_NOTOPMOST, 0, 0, 0, 0, SWP_NOMOVE | SWP_NOSIZE); // 取消置顶</code></pre>
      <p>
       ​
       <strong>
        多窗口布局控制
       </strong>
       ：通过调整
       <code>
        X/Y
       </code>
       和
       <code>
        cx/cy
       </code>
       实现窗口并排或重叠显示。
      </p>
     </li>
     <li>
      ​
      <strong>
       动态隐藏/显示控件
      </strong>
      ：结合
      <code>
       SWP_HIDEWINDOW
      </code>
      和
      <code>
       SWP_SHOWWINDOW
      </code>
      实现界面元素动态切换。
     </li>
    </ol>
    <h3>
     ​
     <strong>
      6. 错误处理
     </strong>
    </h3>
    <ul>
     <li>
      ​
      <strong>
       返回值
      </strong>
      ：成功返回
      <code>
       TRUE
      </code>
      ，失败返回
      <code>
       FALSE
      </code>
      ，可通过
      <code>
       GetLastError()
      </code>
      获取错误代码（如
      <code>
       ERROR_ACCESS_DENIED
      </code>
      ）。
     </li>
     <li>
      ​
      <strong>
       常见错误
      </strong>
      ：
      <ul>
       <li>
        无效窗口句柄（
        <code>
         hWnd
        </code>
        为
        <code>
         NULL
        </code>
        或未创建）。
       </li>
       <li>
        跨进程操作受限（需使用
        <code>
         SetWindowPos
        </code>
        的扩展版本或
        <code>
         WM_WINDOWPOSCHANGING
        </code>
        消息）。
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     ​
     <strong>
      7. 与
      <code>
       MoveWindow()
      </code>
      的对比
     </strong>
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        特性
       </th>
       <th>
        <code>
         SetWindowPos()
        </code>
       </th>
       <th>
        <code>
         MoveWindow()
        </code>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        ​
        <strong>
         消息触发
        </strong>
       </td>
       <td>
        仅发送
        <code>
         WM_WINDOWPOSCHANGED
        </code>
       </td>
       <td>
        发送
        <code>
         WM_MOVE
        </code>
        、
        <code>
         WM_SIZE
        </code>
        等系列消息
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         Z 顺序控制
        </strong>
       </td>
       <td>
        支持通过
        <code>
         hWndInsertAfter
        </code>
        或标志调整
       </td>
       <td>
        仅移动位置，不改变 Z 顺序
       </td>
      </tr>
      <tr>
       <td>
        ​
        <strong>
         参数灵活性
        </strong>
       </td>
       <td>
        <h3>
         支持更多标志（如
         <code>
          SWP_NOREDRAW
         </code>
         ）
        </h3>
       </td>
       <td>
        功能单一，仅移动/缩放窗口
       </td>
      </tr>
     </tbody>
    </table>
    <hr/>
    <h4>
     ​
     <strong>
      总结
     </strong>
    </h4>
    <p>
     <code>
      SetWindowPos()
     </code>
     是 Windows 窗口管理的核心工具，适用于动态调整窗口属性、控制界面布局等场景。使用时需注意权限、Z 顺序逻辑及消息处理差异，合理组合标志参数可优化性能（如
     <code>
      SWP_NOREDRAW
     </code>
     避免重绘开销）。在多显示器或复杂界面设计中，结合屏幕坐标系和标志参数可实现精确控制。
    </p>
