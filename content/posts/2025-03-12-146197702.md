    <h2>
     <a id="_5">
     </a>
     简介
    </h2>
    <p>
     <strong>
      什么是 KOReader ?
     </strong>
    </p>
    <blockquote>
     <p>
      <code>
       KOReader
      </code>
      是一款维护了十多年的开源项目，他是
      <strong>
       主要针对电子墨水阅读器的文档查看器
      </strong>
      ，它支持几乎所有主流的电子书格式。 包括
      <code>
       EPUB
      </code>
      、
      <code>
       MOBI
      </code>
      、
      <code>
       PDF
      </code>
      ，
      <code>
       TXT
      </code>
      、
      <code>
       DJVU
      </code>
      ，
      <code>
       KOReader
      </code>
      都能轻松应对。
     </p>
    </blockquote>
    <p>
     <strong>
      什么是 KOmpanion ？
     </strong>
    </p>
    <blockquote>
     <p>
      <code>
       KOmpanion
      </code>
      是一个为
      <code>
       KOReader
      </code>
      用户设计的极简自托管
      <code>
       Web
      </code>
      应用程序，旨在方便用户管理和同步阅读数据。它主要提供书籍上传和查看、
      <code>
       OPDS
      </code>
      下载、
      <code>
       KOReader
      </code>
      同步进度
      <code>
       API
      </code>
      以及通过
      <code>
       WebDAV
      </code>
      获取
      <code>
       KOReader
      </code>
      书籍统计等功能，从而增强
      <code>
       KOReader
      </code>
      的使用体验。
     </p>
    </blockquote>
    <h3>
     <a id="_15">
     </a>
     主要功能
    </h3>
    <ul>
     <li>
      <strong>
       书架管理
      </strong>
      ：用户可以上传和查看自己的书籍列表。
     </li>
     <li>
      <strong>
       OPDS 支持
      </strong>
      ：允许用户下载书籍。
     </li>
     <li>
      <strong>
       同步进度 API
      </strong>
      ：支持设备间的阅读进度同步。
     </li>
     <li>
      <strong>
       书籍统计
      </strong>
      ：通过
      <code>
       WebDAV
      </code>
      提供书籍阅读统计信息。
     </li>
    </ul>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/img_convert/2e8cd0b4685e7b81debc3cdead17ba7d.png"/>
    </p>
    <p>
     该项目并非用于网页端阅读或格式转换，而是专注于与
     <code>
      KOReader
     </code>
     的紧密集成，并提供了多种安装方式，包括
     <code>
      Railway
     </code>
     、
     <code>
      Docker
     </code>
     和预编译的二进制文件。
    </p>
