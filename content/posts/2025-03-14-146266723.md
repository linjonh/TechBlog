    <h2>
     <a id="1_RO_3">
     </a>
     1 RO介绍
    </h2>
    <p>
     在ARM架构中，R0寄存器是一个通用寄存器；是16 个通用寄存器（R0 到 R15）中的第一个，通常用于存储函数的参数和返回值。
    </p>
    <h3>
     <a id="11_R0__5">
     </a>
     1.1 R0 的主要作用
    </h3>
    <h4>
     <a id="1__6">
     </a>
     1 函数返回值
    </h4>
    <ul>
     <li>
      在 ARM 架构中，R0 通常用于存储函数的返回值。
     </li>
     <li>
      当函数执行完毕后，返回值会被放入 R0，调用者可以从 R0 中读取返回值。
     </li>
    </ul>
    <pre><code class="prism language-s">func:
    MOV r0, #123  @ 将返回值 123 存入 r0
    BX lr        @ 返回到调用者
</code></pre>
    <h4>
     <a id="2__15">
     </a>
     2 函数参数
    </h4>
    <ul>
     <li>
      在函数调用时，R0 到 R3 通常用于传递前 4 个参数（如果参数多于 4 个，额外的参数会通过栈传递）。
      <br/>
      例如，如果一个函数接受两个参数，第一个参数会放在 R0 中，第二个参数会放在 R1 中；
     </li>
    </ul>
    <pre><code class="prism language-c"><span class="token keyword">int</span> <span class="token function">add</span><span class="token punctuation">(</span><span class="token keyword">int</span> a<span class="token punctuation">,</span> <span class="token keyword">int</span> b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> a <span class="token operator">+</span> b<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

add<span class="token operator">:</span>
    ADD r0<span class="token punctuation">,</span> r0<span class="token punctuation">,</span> r1  @ r0 <span class="token operator">=</span> r0 <span class="token operator">+</span> r1
    BX lr           @ 返回
</code></pre>
    <h4>
     <a id="3__28">
     </a>
     3 通用寄存器
    </h4>
    <ul>
     <li>
      R0 是一个通用寄存器，可以用于存储临时数据或中间计算结果
     </li>
    </ul>
    <pre><code class="prism language-c">MOV r0<span class="token punctuation">,</span> #<span class="token number">10</span>   @ 将 <span class="token number">10</span> 存入 r0
ADD r0<span class="token punctuation">,</span> r0<span class="token punctuation">,</span> #<span class="token number">5</span> @ r0 <span class="token operator">=</span> r0 <span class="token operator">+</span> <span class="token number">5</span>
</code></pre>
    <h4>
     <a id="4__35">
     </a>
     4 与其他寄存器的区别
    </h4>
    <ul>
     <li>
      R0 到 R12：通用寄存器，可以自由使用。
     </li>
     <li>
      R13（SP）：栈指针寄存器，用于指向当前栈顶。
     </li>
     <li>
      R14（LR）：链接寄存器，用于存储函数返回地址。
     </li>
     <li>
      R15（PC）：程序计数器，用于存储下一条指令的地址。
     </li>
    </ul>
    <h3>
     <a id="12__41">
     </a>
     1.2 示例
    </h3>
    <p>
     使用R0作为返回值和参数的用法；
    </p>
    <pre><code class="prism language-s">.global main

main:
    MOV r0, #10   @ 将 10 存入 r0（第一个参数）
    MOV r1, #20   @ 将 20 存入 r1（第二个参数）
    BL add        @ 调用 add 函数
    BX lr         @ 返回到调用者（例如操作系统）

add:
    ADD r0, r0, r1 @ r0 = r0 + r1
    BX lr          @ 返回到调用者

</code></pre>
    <ul>
     <li>
      在 main 函数中，R0 和 R1 分别存储参数 10 和 20。
     </li>
     <li>
      调用 add 函数后，R0 存储计算结果 30，并作为返回值返回。
      <br/>
      【总结】
     </li>
     <li>
      R0 是 ARM 架构中非常重要的寄存器，主要用于存储函数返回值和前几个参数。
     </li>
     <li>
      它是通用的，可以用于存储临时数据或中间计算结果。
     </li>
     <li>
      在编写 ARM 汇编代码时，理解 R0 的作用非常重要。
     </li>
    </ul>
