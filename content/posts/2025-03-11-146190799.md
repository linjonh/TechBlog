    <p>
     在 C++20 Concepts 中，使用简写形式的
     <code>
      template&lt;ConceptName T&gt;
     </code>
     与使用完整形式的
     <code>
      template&lt;typename T&gt; requires ConceptName&lt;T&gt;
     </code>
     是等价的。以下是两种写法的具体转换：
    </p>
    <h4>
     <a id="_4">
     </a>
     原始写法（简写形式）
    </h4>
    <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">concept</span> <span class="token class-name">Arithmetic</span> <span class="token operator">=</span> std<span class="token double-colon punctuation">::</span>is_arithmetic_v<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span><span class="token punctuation">;</span>

<span class="token keyword">template</span><span class="token operator">&lt;</span>Arithmetic T<span class="token operator">&gt;</span>  <span class="token comment">// 简写形式</span>
T <span class="token function">add</span><span class="token punctuation">(</span>T a<span class="token punctuation">,</span> T b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">return</span> a <span class="token operator">+</span> b<span class="token punctuation">;</span> <span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="requires__13">
     </a>
     等效写法（requires 从句形式）
    </h4>
    <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">concept</span> <span class="token class-name">Arithmetic</span> <span class="token operator">=</span> std<span class="token double-colon punctuation">::</span>is_arithmetic_v<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span><span class="token punctuation">;</span>

<span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>    <span class="token comment">// 完整形式</span>
<span class="token keyword">requires</span> Arithmetic<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span>  <span class="token comment">// requires 从句</span>
T <span class="token function">add</span><span class="token punctuation">(</span>T a<span class="token punctuation">,</span> T b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">return</span> a <span class="token operator">+</span> b<span class="token punctuation">;</span> <span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_23">
     </a>
     关键区别说明：
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        特征
       </th>
       <th>
        简写形式
       </th>
       <th>
        requires 从句形式
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        语法结构
       </td>
       <td>
        <code>
         template&lt;ConceptName T&gt;
        </code>
       </td>
       <td>
        <code>
         template&lt;typename T&gt; requires ...
        </code>
       </td>
      </tr>
      <tr>
       <td>
        可组合性
       </td>
       <td>
        只能指定单个 Concept
       </td>
       <td>
        可通过逻辑运算符组合多个约束
       </td>
      </tr>
      <tr>
       <td>
        约束表达式位置
       </td>
       <td>
        模板参数声明中
       </td>
       <td>
        模板参数列表后的 requires 从句中
       </td>
      </tr>
      <tr>
       <td>
        适用场景
       </td>
       <td>
        简单约束
       </td>
       <td>
        复杂约束（需要组合多个条件时）
       </td>
      </tr>
     </tbody>
    </table>
    <h4>
     <a id="_31">
     </a>
     组合多个约束的示例：
    </h4>
    <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">concept</span> <span class="token class-name">Signed</span> <span class="token operator">=</span> std<span class="token double-colon punctuation">::</span>is_signed_v<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span><span class="token punctuation">;</span>

<span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">requires</span> Arithmetic<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span> <span class="token operator">&amp;&amp;</span> Signed<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span>  <span class="token comment">// 组合两个约束</span>
T <span class="token function">negative</span><span class="token punctuation">(</span>T value<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> <span class="token operator">-</span>value<span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token function">negative</span><span class="token punctuation">(</span><span class="token number">5</span><span class="token punctuation">)</span><span class="token punctuation">;</span>     <span class="token comment">// OK</span>
<span class="token function">negative</span><span class="token punctuation">(</span><span class="token number">3.14</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// OK</span>
<span class="token function">negative</span><span class="token punctuation">(</span><span class="token string">"123"</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 错误：不满足 Arithmetic 约束</span>
<span class="token function">negative</span><span class="token punctuation">(</span><span class="token number">2u</span><span class="token punctuation">)</span><span class="token punctuation">;</span>    <span class="token comment">// 错误：unsigned 不满足 Signed 约束</span>
</code></pre>
    <h4>
     <a id="_48">
     </a>
     两种形式的编译结果：
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        错误信息对比
       </strong>
       ：
      </p>
      <pre><code class="prism language-cpp"><span class="token function">add</span><span class="token punctuation">(</span><span class="token string">"hello"</span><span class="token punctuation">,</span> <span class="token string">"world"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 错误信息中会明确显示：</span>
<span class="token comment">// 简写形式："constraints not satisfied for 'T'"</span>
<span class="token comment">// requires 从句形式："constraint 'Arithmetic&lt;T&gt;' was not satisfied"</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        元编程特性保留
       </strong>
       ：
      </p>
      <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span>Arithmetic T<span class="token operator">&gt;</span>      <span class="token comment">// 两种形式均可配合</span>
<span class="token keyword">struct</span> <span class="token class-name">Calculator</span> <span class="token punctuation">{<!-- --></span>         <span class="token comment">// SFINAE、if constexpr 等特性</span>
    <span class="token keyword">static_assert</span><span class="token punctuation">(</span>Signed<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="_64">
     </a>
     更复杂的约束示例：
    </h4>
    <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">concept</span> <span class="token class-name">Printable</span> <span class="token operator">=</span> <span class="token keyword">requires</span><span class="token punctuation">(</span>std<span class="token double-colon punctuation">::</span>ostream<span class="token operator">&amp;</span> os<span class="token punctuation">,</span> T val<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token punctuation">{<!-- --></span> os <span class="token operator">&lt;&lt;</span> val <span class="token punctuation">}</span> <span class="token operator">-&gt;</span> std<span class="token double-colon punctuation">::</span>same_as<span class="token operator">&lt;</span>std<span class="token double-colon punctuation">::</span>ostream<span class="token operator">&amp;</span><span class="token operator">&gt;</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">requires</span> Arithmetic<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span> <span class="token operator">&amp;&amp;</span> Printable<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span>
<span class="token keyword">void</span> <span class="token function">print_sum</span><span class="token punctuation">(</span>T a<span class="token punctuation">,</span> T b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    std<span class="token double-colon punctuation">::</span>cout <span class="token operator">&lt;&lt;</span> <span class="token punctuation">(</span>a <span class="token operator">+</span> b<span class="token punctuation">)</span> <span class="token operator">&lt;&lt;</span> <span class="token string">"\n"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>

<span class="token function">print_sum</span><span class="token punctuation">(</span><span class="token number">3</span><span class="token punctuation">,</span> <span class="token number">5</span><span class="token punctuation">)</span><span class="token punctuation">;</span>    <span class="token comment">// OK：int 是算术类型且可打印</span>
<span class="token function">print_sum</span><span class="token punctuation">(</span><span class="token number">2.71</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// OK：double 符合条件</span>
<span class="token function">print_sum</span><span class="token punctuation">(</span><span class="token boolean">true</span><span class="token punctuation">,</span> <span class="token boolean">false</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 错误：bool 是算术类型但不符合 Printable（需要重载&lt;&lt;）</span>
</code></pre>
    <h4>
     <a id="_82">
     </a>
     标准库风格的约束：
    </h4>
    <pre><code class="prism language-cpp"><span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">concept</span> <span class="token class-name">Addable</span> <span class="token operator">=</span> <span class="token keyword">requires</span><span class="token punctuation">(</span>T a<span class="token punctuation">,</span> T b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token punctuation">{<!-- --></span> a <span class="token operator">+</span> b <span class="token punctuation">}</span> <span class="token operator">-&gt;</span> std<span class="token double-colon punctuation">::</span>convertible_to<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">template</span><span class="token operator">&lt;</span><span class="token keyword">typename</span> <span class="token class-name">T</span><span class="token operator">&gt;</span>
<span class="token keyword">requires</span> Addable<span class="token operator">&lt;</span>T<span class="token operator">&gt;</span>  <span class="token comment">// 替代 C++17 的 std::void_t 技巧</span>
T <span class="token function">sum</span><span class="token punctuation">(</span>T a<span class="token punctuation">,</span> T b<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">return</span> a <span class="token operator">+</span> b<span class="token punctuation">;</span> <span class="token punctuation">}</span>
</code></pre>
    <p>
     建议：当约束条件简单时使用简写形式，需要组合多个约束或需要更清晰的错误信息时使用
     <code>
      requires
     </code>
     从句形式。
    </p>
