     1、CSS
    </h2>
    <p>

    <h2>
     <a id="3_67">
     </a>
     3、选择器
    </h2>
    <p>
     <strong>
      要美化HTML，就要先找到对应的标签，选择器就是用来查找对应的标签
     </strong>
     ，并给它设置样式，
     <strong>
      基础的
     </strong>
     选择器有四种：
    </p>
    <ul>
     <li>
      标签选择器
     </li>
     <li>
      类选择器
     </li>
     <li>
      id选择器
     </li>
     <li>
      通配符选择器
     </li>
    </ul>
    <h3>
     <a id="31__77">
     </a>
     3.1 标签选择器
    </h3>
    <p>
     使用标签名作为选择器 →
     <strong>
      选中同名标签
     </strong>
     设置相同的样式，如p, h1, div, a, img…
    </p>
    <pre><code class="prism language-html"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>style</span><span class="token punctuation">&gt;</span></span><span class="token style"><span class="token language-css"> 
	<span class="token selector">p</span> <span class="token punctuation">{<!-- --></span>
		<span class="token property">color</span><span class="token punctuation">:</span> red<span class="token punctuation">;</span> 
	<span class="token punctuation">}</span>
</span></span><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>style</span><span class="token punctuation">&gt;</span></span>
</code></pre>
    <p>
     缺点：同名标签的显示效果只能一样，即不能差异化
    </p>
    <h3>
     <a id="32__90">
     </a>
     3.2 类选择器
    </h3>
    <p>
     用于查找标签，差异化设置标签的显示效果：
    </p>
    <ul>
     <li>
      定义类选择器 →
      <code>
       .类名
      </code>
     </li>
     <li>
      使用类选择器 → 标签添加
      <code>
       class="类名"
      </code>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8325faf73c774cf39f371f718845d218.png"/>
    </p>
    <p>
     注意：
    </p>
    <ul>
     <li>
      类名自定义，不要用纯数字或中文，尽量用英文命名，见名知意，多个单词用-连接，如news-hd
     </li>
     <li>
      <code>
       一个
      </code>
      类选择器可以供
      <code>
       多个
      </code>
      标签使用
     </li>
     <li>
      一个标签可以
      <code>
       使用多个类名
      </code>
      ，类名之间用空格隔开
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bd37256ab6134650a3f00ae6a0020869.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fb7e6618abab40b196290b29c944b0c0.png"/>
    </p>
    <h3>
     <a id="33_id_108">
     </a>
     3.3 id选择器
    </h3>
    <p>
     用于查找标签，差异化设置标签的显示效果，
     <strong>
      这和类选择器一样，但id选择器一般配合 JavaScript 使用，很少用来设置 CSS 样式
     </strong>
     ：
    </p>
    <ul>
     <li>
      定义id选择器→
      <code>
       #id名
      </code>
     </li>
     <li>
      使用 id 选择器 → 标签添加
      <code>
       id= "id名"
      </code>
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/10e9fe9459164bb5b4353a923cce6b2c.png"/>
    </p>
    <p>
     另外，和类选择器不同的是，同一个id选择器，在一个页面只能使用一次
    </p>
    <h3>
     <a id="34__118">
     </a>
     3.4 通配符选择器
    </h3>
    <p>
     用于查找页面所有标签，
     <strong>
      设置相同样式
     </strong>
     ，使用上：
    </p>
    <ul>
     <li>
      <code>
       *，不需要调用
      </code>
      ，浏览器自动查找页面所有标签，设置相同的样式
     </li>
    </ul>
    <pre><code class="prism language-html">*{
	color: red; 
}
</code></pre>
    <ul>
     <li>
      可以用于清除标签的默认样式，比如，h和p标签之间，或者两个p标签之间自带的默认边距
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f980455c960a4b20a59722ed6978f8b0.png"/>
    </p>
