# 全息电话会议开源项目LiveScan3D-Hololens

|  |
| --- |
| LiveScan3D：该开源程序可以传输点云数据流到微软的 [hololens](http://www.52vr.com/armr/Hololens/) [AR](http://www.52vr.com/armr/) [眼镜](http://www.52vr.com/forum/118/) ，并在hololens中以渲染的方式还原点云图像。这种技术可以被用来实现——远距离实时全息电视会议。  [开源全息Hololens电话会议开源项目LiveScan3D-Hololens](https://i-blog.csdnimg.cn/blog_migrate/798bea1a968f15ff94a576a6a6537d76.png)  LiveScan3D如何运行 首先在本文后面有项目工程 [源码](http://www.52vr.com/forum/codes/) 链接. 下载这份工程源码，然后打开LiveScan, 打开客户端实时画面(该画面就是要传输到hololens上的画面). 下一步打开该项目中的 [unity](http://www.52vr.com/forum/unity/) 工程，该Unity工程是作为服务器存在的，如果不出问题的话，就应该准备好了，你就能在hololens中看到传输过来的摄像头预览画面了。  如果你觉得该工程比较easy, 你完全可以自己在此基础上开发一个 xaml [uwp](http://www.52vr.com/armr/Hololens/) 应用，一旦你在输入了IP地址后，启动了Hololens，你同样能够看到传输过来的点云数据。  [开源全息Hololens电话会议开源项目LiveScan3D-Hololens](https://i-blog.csdnimg.cn/blog_migrate/c964f41f8e2882e3675ad8f823737fb8.png)  [开源全息Hololens电话会议开源项目LiveScan3D-Hololens](https://i-blog.csdnimg.cn/blog_migrate/545bdcb9a63e1e0576824dffc0c704b1.png)  如何在Hololens中控制场景 这个hololens程序可以进行拖放、放大缩小和绕着Y轴旋转, 可以通过语音进行如下控制功能:   * Move - 可以通过点击和捏住手势控制场景移动the scene can be moved using the tap and hold gesture. * Rotate - 可以通过点击和捏住手势水平移动来表示旋转。 * Zoom - 可以通过点击和捏住手势垂直移动来表示旋转。 * Reset - 可以通过reset返回到之前的位置，缩放倍数和旋转角度.   [开源全息Hololens电话会议开源项目LiveScan3D-Hololens](https://i-blog.csdnimg.cn/blog_migrate/b9e8b1266d99fbccf760bad6e7d225f4.png)  Citations 如果你在你的项目中使用了该程序，请附带以下信息。  Kowalski, M.; Naruniec, J.; Daniluk, M.: "LiveScan3D: A Fast and Inexpensive 3D Data Acquisition System for Multiple [Kinect](http://www.52vr.com/hardware/local/) v2 Sensors". in 3D Vision (3DV), 2015 International Conference on, Lyon, France, 2015   该全息会议项目下载地址： github：  https://github.com/MarekKowalski/LiveScan3D-Hololens |
