    <h2>
     <a id="Map_0">
     </a>
     Map集合
    </h2>
    <h3>
     <a id="_2">
     </a>
     概述
    </h3>
    <p>
     Map 是一种无序的键值对的集合。
    </p>
    <p>
     Map 最重要的一点是通过 key 来快速检索数据，key 类似于索引，指向数据的值。
    </p>
    <p>
     Map 是一种集合，所以我们可以像迭代数组和切片那样迭代它。不过，Map 是无序的，遍历 Map 时返回的键值对的顺序是不确定的。
    </p>
    <p>
     在获取 Map 的值时，如果键不存在，返回该类型的零值，例如 int 类型的零值是 0，string 类型的零值是 “”。
    </p>
    <p>
     Map 是引用类型，如果将一个 Map 传递给一个函数或赋值给另一个变量，它们都指向同一个底层数据结构，因此对 Map 的修改会影响到所有引用它的变量。
    </p>
    <h3>
     <a id="map_14">
     </a>
     定义map
    </h3>
    <h4>
     <a id="map_16">
     </a>
     map关键字定义
    </h4>
    <p>
     <strong>
      基本语法
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token keyword">var</span> 集合名 <span class="token keyword">map</span><span class="token punctuation">[</span>键类型<span class="token punctuation">]</span>值类型<span class="token punctuation">;</span>
</code></pre>
    <p>
     <strong>
      举例
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> map_knowledge

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token comment">//通过关键字定义map</span>
<span class="token keyword">func</span> <span class="token function">CreateMap</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"map1的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      结果
     </strong>
    </p>
    <pre><code class="prism language-sh">map1的值为map<span class="token punctuation">[</span>string<span class="token punctuation">]</span>string<span class="token punctuation">(</span>nil<span class="token punctuation">)</span>
</code></pre>
    <p>
     <strong>
      注意
     </strong>
    </p>
    <p>
     用关键字定义的map初始值为
     <code>
      nil
     </code>
     ,我们不能对其直接操作。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">package</span> map_knowledge

<span class="token keyword">import</span> <span class="token string">"fmt"</span>

<span class="token comment">//通过关键字定义map</span>
<span class="token keyword">func</span> <span class="token function">CreateMap</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
	<span class="token keyword">var</span> map1  <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">;</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"map1的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">)</span>


	<span class="token comment">//用关键字定义的map为nil，不能直接操作</span>
	<span class="token comment">/*
		报错:panic: assignment to entry in nil map
	*/</span>
	<span class="token comment">// map1["name"] = "zhansan"</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      小结
     </strong>
    </p>
    <p>
     这种用关键字定义的map就是一个空集合，如果要能够操作只有两种等价方案。
    </p>
    <pre><code class="prism language-go">function <span class="token function">CreateMap</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
    <span class="token comment">//方案1:定义完后用make开辟空间</span>
    <span class="token keyword">var</span> map1 <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span>
    map1 <span class="token operator">=</span> <span class="token function">make</span><span class="token punctuation">(</span><span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">)</span>
    
    <span class="token comment">//方案2:定义的时候赋值初始化</span>
    <span class="token comment">//这样golang也会为map开辟空间</span>
    <span class="token keyword">var</span> map2 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">{<!-- --></span><span class="token string">"name"</span><span class="token punctuation">:</span><span class="token string">"lisa"</span><span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="make_83">
     </a>
     make定义
    </h4>
    <p>
     <code>
      make
     </code>
     既可以用于帮空集合开辟空间，也可以直接定义集合，golang官方也推荐这种形式。
    </p>
    <p>
     <strong>
      基本语法
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token comment">/* 使用 make 函数 */</span>
map_variable <span class="token operator">:=</span> <span class="token function">make</span><span class="token punctuation">(</span><span class="token keyword">map</span><span class="token punctuation">[</span>KeyType<span class="token punctuation">]</span>ValueType<span class="token punctuation">,</span> initialCapacity<span class="token punctuation">)</span>
</code></pre>
    <p>
     其中
     <code>
      KeyType
     </code>
     是键的类型，
     <code>
      ValueType
     </code>
     是值的类型，
     <code>
      initialCapacity
     </code>
     是可选的参数，用于指定 Map 的初始容量。
    </p>
    <p>
     Map 的容量是指 Map 中可以保存的键值对的数量，当 Map 中的键值对数量达到容量时，Map 会自动扩容。
    </p>
    <p>
     如果不指定 initialCapacity，Go 语言会根据实际情况选择一个合适的值。
    </p>
    <blockquote>
     <p>
      一般来说我们不用特别指定容量，因为即使我们指定了，如果不够也会自己扩容。
     </p>
    </blockquote>
    <p>
     <strong>
      举例
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token comment">//通过make定义map</span>
<span class="token keyword">func</span> <span class="token function">CreateMapByMake</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token function">make</span><span class="token punctuation">(</span><span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">)</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"map1的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      结果
     </strong>
    </p>
    <pre><code class="prism language-go">map1的值为<span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      注意
     </strong>
    </p>
    <p>
     由于make定义开辟了空间，所以map可以直接操作。(这也是推荐用make定义的原因。)
    </p>
    <pre><code class="prism language-go"><span class="token keyword">func</span> <span class="token function">CreateMapByMake</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token function">make</span><span class="token punctuation">(</span><span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">,</span><span class="token number">2</span><span class="token punctuation">)</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"map1的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">)</span>

	map1<span class="token punctuation">[</span><span class="token string">"name"</span><span class="token punctuation">]</span><span class="token operator">=</span><span class="token string">"lisa"</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"赋值后,map1的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     结果为
    </p>
    <pre><code class="prism language-go">map1的值为<span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>
赋值后<span class="token punctuation">,</span>map1的值为<span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">string</span><span class="token punctuation">{<!-- --></span><span class="token string">"name"</span><span class="token punctuation">:</span><span class="token string">"lisa"</span><span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="map_139">
     </a>
     map初始化
    </h3>
    <h4>
     <a id="_141">
     </a>
     <strong>
      字面量初始化
     </strong>
    </h4>
    <p>
     <strong>
      基本语法
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token keyword">var</span> <span class="token keyword">map</span>变量 <span class="token keyword">map</span><span class="token punctuation">[</span>键类型<span class="token punctuation">]</span>值类型 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span>键类型<span class="token punctuation">]</span>值类型<span class="token punctuation">{<!-- --></span>初始值<span class="token punctuation">}</span>
<span class="token comment">//可以省略左边的类型</span>
<span class="token keyword">var</span> <span class="token keyword">map</span>变量 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span>键类型<span class="token punctuation">]</span>值类型<span class="token punctuation">{<!-- --></span>初始值<span class="token punctuation">}</span>
</code></pre>
    <p>
     map可以通过赋值直接初始化，例如:
    </p>
    <pre><code class="prism language-go"><span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">int</span><span class="token punctuation">{<!-- --></span>
    <span class="token comment">//最后一项也要有逗号</span>
    <span class="token string">"name"</span><span class="token punctuation">:</span><span class="token number">10</span><span class="token punctuation">,</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      注意事项
     </strong>
    </p>
    <p>
     以字面量初始化
     <code>
      map
     </code>
     ,内部的每一项用
     <code>
      ,
     </code>
     隔开，且最后一项也必须要有
     <code>
      ,
     </code>
    </p>
    <h4>
     <a id="make_164">
     </a>
     make初始化
    </h4>
    <p>
     我们先查看
     <code>
      make
     </code>

    </p>
    <pre><code class="prism language-go"><span class="token keyword">func</span> <span class="token function">make</span><span class="token punctuation">(</span>t Type<span class="token punctuation">,</span> size <span class="token operator">...</span>IntegerType<span class="token punctuation">)</span> Type

    <p>
     所以
     <code>
      make
     </code>

    </p>
    <pre><code class="prism language-go"><span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token function">make</span><span class="token punctuation">(</span><span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">int</span><span class="token punctuation">)</span>
map1<span class="token punctuation">[</span><span class="token string">"age"</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token number">19</span>
</code></pre>
    <h3>
     <a id="map_179">
     </a>
     map访问
    </h3>
    <p>
     <strong>
      注意事项
     </strong>
    </p>
    <p>
     我们只能操作开辟了内存空间的map，用关键字定义但是没有初始化的map是空map，无法操作.(见上文)
    </p>
    <h4>
     <a id="map_185">
     </a>
     <strong>
      访问map
     </strong>
    </h4>
    <p>
     <strong>
      基本语法
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token keyword">map</span><span class="token punctuation">[</span>keyValue<span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     访问不存在的键名不会报错，只会返回值类型的零值。
    </p>
    <blockquote>
     <p>
      因为
      <code>
       map[键类型]值类型
      </code>
      ，起初就指定了值的类型
     </p>
    </blockquote>
    <p>
     <strong>
      举例
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token comment">//访问map</span>
<span class="token keyword">func</span> <span class="token function">VisitMap</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">int</span><span class="token punctuation">{<!-- --></span>
		<span class="token string">"age"</span><span class="token punctuation">:</span><span class="token number">19</span><span class="token punctuation">,</span>
	<span class="token punctuation">}</span>

	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"age的值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">[</span><span class="token string">"age"</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"访问不存在的项值为%#v\n"</span><span class="token punctuation">,</span>map1<span class="token punctuation">[</span><span class="token string">"other"</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      结果
     </strong>
    </p>
    <pre><code class="prism language-sh">age的值为19
访问不存在的项值为0
</code></pre>
    <h4>

     </a>

    </h4>

     <code>

     </code>
     ，来得知键是否存在。
    </p>
    <p>
     <strong>

</code></pre>
    <p>
     <strong>
      举例
     </strong>
    </p>
    <pre><code class="prism language-go"><span class="token keyword">func</span> <span class="token function">VisitMapIsOk</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token keyword">var</span> map1 <span class="token operator">=</span> <span class="token keyword">map</span><span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">]</span><span class="token builtin">int</span><span class="token punctuation">{<!-- --></span>

	<span class="token punctuation">}</span>
	ageVal<span class="token punctuation">,</span>isAgeOk <span class="token operator">:=</span> map1<span class="token punctuation">[</span><span class="token string">"age"</span><span class="token punctuation">]</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"age存在为%v,age的值为%#v\n"</span><span class="token punctuation">,</span>isAgeOk<span class="token punctuation">,</span>ageVal<span class="token punctuation">)</span>

	nameVal<span class="token punctuation">,</span>isNameOk <span class="token operator">:=</span> map1<span class="token punctuation">[</span><span class="token string">"name"</span><span class="token punctuation">]</span>
	fmt<span class="token punctuation">.</span><span class="token function">Printf</span><span class="token punctuation">(</span><span class="token string">"name存在为%v,name的值为%#v\n"</span><span class="token punctuation">,</span>isNameOk<span class="token punctuation">,</span>nameVal<span class="token punctuation">)</span>

     <a id="_254">
     </a>
     任意类型的值
    </h4>
    <p>
     我们可以用
     <code>
      interface{}
     </code>
     空接口类型来接收任意类型的值。
    </p>
    <pre><code class="prism language-go"><span class="token keyword">func</span> <span class="token function">InterfaceMap</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
    <span class="token comment">//interface{}是值类型</span>
