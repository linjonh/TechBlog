    <ul>
     <li>
      <p>
       在 MySQL 中，BETWEEN
       <font face="monospace">
        ...AND
       </font>
       语句是一个用于筛选数据的条件运算符，它可以帮助你从表中选取指定范围内的数据。
      </p>
      <pre><code class="language-sql">SELECT column1, column2, ...
FROM table_name
WHERE column_name BETWEEN value1 AND value2;</code></pre>
      <p>
       <strong>
        如果你想选取不在指定范围内的数据，可以使用
        <code>
         NOT BETWEEN...AND
        </code>
        ：
        <code>
         BETWEEN...AND
        </code>
        是包含边界值的，即选取的数据包含
        <code>
         value1
        </code>
        和
        <code>
         value2
        </code>
        。
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        在使用日期型数据时，要确保日期格式与数据库中存储的格式一致。
       </strong>
      </p>
     </li>
    </ul>
    <p>
     <strong>
      <span style="background-color:#ffd900">
       运算符：in
      </span>
     </strong>
    </p>
    <p>
     <strong>
      <font face="monospace">
       IN
      </font>
      操作符会检查表中列中的每个值是否等于值列表中的任何一个值。如果相等，则该行会被包含在查询结果中。
     </strong>
    </p>
    <pre><code class="language-sql">CREATE TABLE students (
    id INT,
    name VARCHAR(50),
    class VARCHAR(10)
);

INSERT INTO students (id, name, class) VALUES
(1, 'Alice', 'Class A'),
(2, 'Bob', 'Class B'),
(3, 'Charlie', 'Class A'),
(4, 'David', 'Class C');
SELECT *
FROM students
WHERE class IN ('Class A', 'Class C');</code></pre>
    <p>
     这个查询会返回
     <code>
      class
     </code>
     列的值为 'Class A' 或 'Class C' 的所有行。
    </p>
    <p>
     LIKE:模糊查询
    </p>
    <p>
     百分号（%）：代表任意数量（包括零个）的任意字符。
     <br/>
     下划线（_）：代表单个任意字符。
    </p>
    <p>
     _:比如你要查询第2个字符是a的员工信息
    </p>
    <pre><code class="language-sql">SELECT *
FROM employees
WHERE name LIKE '_a%';</code></pre>
    <p>
     例如：可以查询包含字符‘a’的员工信息
    </p>
    <pre><code class="language-sql">SELECT *
FROM employees
WHERE name LIKE '%a%';</code></pre>
