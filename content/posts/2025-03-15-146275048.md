# 智慧加油站小程序数据库设计文档

## 智慧加油站系统 - 数据库与API设计文档

### 1\. 数据库设计

#### 1.1 ER模型

系统的核心实体关系如下：

    
    
    用户(User) ---< 订单(Order) ---< 加油记录(RefuelRecord)
       |               |                |
       |               |                v
       v               v            油枪(OilGun)
    角色(Role)      油品(Oil)          |
       |               |                |
       v               v                v
    权限(Permission)  价格历史(PriceHistory)  站点(Station)
