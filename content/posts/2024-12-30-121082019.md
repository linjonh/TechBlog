    <p>
     <strong>
      我们在进行前端页面开发布局的时候，为了给用户呈现不同的效果，经常需要设置透明度，那么CSS设置透明度的方法是什么呢？
     </strong>
    </p>
    <p>
     本文就来给大家介绍用
     <strong>
      CSS设置透明度
     </strong>
     的两种方法（代码实例）。我们在未来使用CSS设置页面透明度的时候有一定的参考价值，能够起到一点帮助。
    </p>
    <h4>
     一、css rgba()设置颜色透明度
    </h4>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <p>
     1rgba（R,G,B,A）;
     <br/>
     RGBA 是代表Red（红色） Green（绿色） Blue（蓝色）和 Alpha（不透明度）三个单词的缩写。RGBA 颜色值是 RGB 颜色值的扩展，带有一个 alpha 通道 - 它规定了对象的不透明度。
     <br/>
     rgba（)里的值的介绍：
    </p>
    <ul>
     <li>
      R：红色值。正整数 （0~255）
     </li>
     <li>
      G：绿色值。正整数 （0~255）
     </li>
     <li>
      B：蓝色值。正整数（0~255）
     </li>
     <li>
      A：透明度。取值0~1之间
     </li>
    </ul>
    <p>
     rgba()只是单纯的可以设置颜色透明度，这样在页面的布局中有很多应用。比如说：让背景出现透明效果，但上面的文字不透明。
    </p>
    <p>
     <strong>
      代码实例：
     </strong>
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
&lt;meta charset="UTF-8"&gt;
&lt;title&gt;rgba()&lt;/title&gt;
&lt;style&gt;
.demo{
width: 350px;
height: 300px;
margin: 50pxauto;
}
.demo *{
width: 120px;
height: 120px;
margin: 10px;
float: left;
}
.demo1{
background:rgba(255,0,0,1);
}
.demo2{
background:rgba(255,0,0,0.5);
}
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;div class="demo"&gt;
&lt;div class="demo1"&gt;背景色不透明，文字不透明！&lt;/div&gt;
&lt;div class="demo2"&gt;背景色半透明，文字不透明！&lt;/div&gt;
&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <strong>
      效果图：
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" height="168" src="https://i-blog.csdnimg.cn/blog_migrate/c4514ce4bd29dbbddc6785fe41838bb1.png" width="450"/>
    </p>
    <p>
     上例中，设置的颜色值是一样的，只是透明度不同。这样看不出来有除了颜色之外的不同，我们在父容器demo上也设置一个背景色，这样的效果图：
    </p>
    <p style="text-align:center;">
     <img alt="" height="203" src="https://i-blog.csdnimg.cn/blog_migrate/be2c0fadc14815bee353ad295481999c.png" width="509"/>
    </p>
    <p>
     这样就可以看出：第一个盒子（demo1）没有设置透明度，红色完全把下面盒子（demo）的颜色给覆盖住了；第二个盒子（demo2）设置了透明度，使得颜色半透明，没有把下面盒子（demo）的颜色给完全覆盖，而是混合显示了。
    </p>
    <h4>
     二、css opacity属性设置背景透明度
    </h4>
    <p>
     <strong>
      语法：
     </strong>
    </p>
    <p>
     1opacity: value ;
    </p>
    <p>
     value ：指定不透明度，从0.0（完全透明）到1.0（完全不透明）。
    </p>
    <p>
     opacity属性具有继承性，会使容器中的所有元素都具有透明度；
    </p>
    <p>
     <strong>
      代码实例：
     </strong>
    </p>
    <pre><code class="language-css">&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
&lt;meta charset="UTF-8"&gt;
&lt;title&gt;opacity属性&lt;/title&gt;
&lt;style&gt;
.demo{
width: 280px;
height: 140px;
margin: 50pxauto;
}
.demo1,.demo2{
width: 120px;
height: 120px;
margin: 10px;
float: left;
background:#2DC4CB;
}
.demo1{
opacity:1;
}
.demo2{
opacity:0.5;
}
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;div class="demo"&gt;
&lt;div class="demo1"&gt;
&lt;p&gt;背景色不透明，文字不透明！&lt;/p&gt;
&lt;/div&gt;
&lt;div class="demo2"&gt;
&lt;p&gt;背景色透明，文字也透明！&lt;/p&gt;
&lt;/div&gt;
&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <strong>
      效果图：
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" height="169" src="https://i-blog.csdnimg.cn/blog_migrate/6890b0096a891c87fdc6d3d255dc82ca.png" width="458"/>
    </p>
    <p>
     opacity:0.5;使得demo2容器的所以元素都呈现半透明。
    </p>
    <h4>
     总结：
    </h4>
    <p>
     rgba()方法与opacity方法虽然都可以实现透明度效果，但rgba()只作用于元素的颜色或其背景色（设置了rgb(）透明度元素的子元素不会继承其透明效果）；而opacity具有继承性，既作用于元素本身，也会使元素内的所有子元素具有透明度。至于在什么场景下使用那种方法就要看实际需要了。
    </p>
    <p>
     以上就是css设置透明度的两种方式，当然，我们在实际的前端页面开发中，如果要设置页面的透明度还需要考虑到其他的复杂因素，限制条件越多，CSS的设置也就越复杂。
    </p>
