    <p>
     Image.network的具体用法
    </p>
    <p>
     <code>
      Image.network
     </code>
     是 Flutter 中用于从网络加载图片的便捷方法。它基于
     <code>
      NetworkImage
     </code>
     ，可以快速加载并显示网络图片。以下是
     <code>
      Image.network
     </code>
     的具体用法和常见参数说明。
    </p>
    <hr/>
    <h4>
     基本用法
    </h4>
    <p>
     最简单的用法是提供一个图片的 URL：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network('https://example.com/image.png')</pre>
    <hr/>
    <h4>
     常用参数
    </h4>
    <p>
     <code>
      Image.network
     </code>
     提供了许多参数来定制图片的加载和显示行为。以下是一些常用的参数：
    </p>
    <h5>
     1.
     <code>
      src
     </code>
     (必填)
    </h5>
    <p>
     图片的 URL 地址。
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network('https://example.com/image.png')</pre>
    <h5>
     2.
     <code>
      headers
     </code>
    </h5>
    <p>
     设置 HTTP 请求头，例如禁用缓存：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  headers: {
    'Cache-Control': 'no-cache',
  },
)</pre>
    <h5>
     3.
     <code>
      width
     </code>
     和
     <code>
      height
     </code>
    </h5>
    <p>
     设置图片的宽度和高度：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  width: 200,
  height: 200,
)</pre>
    <h5>
     4.
     <code>
      fit
     </code>
    </h5>
    <p>
     设置图片的填充方式，常见的值有：
    </p>
    <ul>
     <li>
      <p>
       <code>
        BoxFit.cover
       </code>
       ：保持宽高比，填满整个空间。
      </p>
     </li>
     <li>
      <p>
       <code>
        BoxFit.contain
       </code>
       ：保持宽高比，完整显示图片。
      </p>
     </li>
     <li>
      <p>
       <code>
        BoxFit.fill
       </code>
       ：拉伸图片以填满整个空间。
      </p>
     </li>
    </ul>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  fit: BoxFit.cover,
)</pre>
    <h5>
     5.
     <code>
      alignment
     </code>
    </h5>
    <p>
     设置图片的对齐方式，例如居中：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  alignment: Alignment.center,
)</pre>
    <h5>
     6.
     <code>
      color
     </code>
     和
     <code>
      colorBlendMode
     </code>
    </h5>
    <p>
     为图片添加颜色滤镜：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  color: Colors.blue,
  colorBlendMode: BlendMode.colorBurn,
)</pre>
    <h5>
     7.
     <code>
      loadingBuilder
     </code>
    </h5>
    <p>
     自定义图片加载过程中的显示内容：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  loadingBuilder: (BuildContext context, Widget child, ImageChunkEvent? loadingProgress) {
    if (loadingProgress == null) {
      return child; // 加载完成，显示图片
    }
    return Center(
      child: CircularProgressIndicator(
        value: loadingProgress.expectedTotalBytes != null
            ? loadingProgress.cumulativeBytesLoaded / loadingProgress.expectedTotalBytes!
            : null,
      ),
    );
  },
)</pre>
    <h5>
     8.
     <code>
      errorBuilder
     </code>
    </h5>
    <p>
     自定义图片加载失败时的显示内容：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/invalid-url.png',
  errorBuilder: (BuildContext context, Object error, StackTrace? stackTrace) {
    return Text('加载失败');
  },
)</pre>
    <h5>
     9.
     <code>
      frameBuilder
     </code>
    </h5>
    <p>
     自定义图片加载过程中的动画效果：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  frameBuilder: (BuildContext context, Widget child, int? frame, bool wasSynchronouslyLoaded) {
    if (wasSynchronouslyLoaded) {
      return child; // 图片已同步加载完成
    }
    return AnimatedOpacity(
      child: child,
      opacity: frame == null ? 0 : 1,
      duration: const Duration(seconds: 1),
      curve: Curves.easeOut,
    );
  },
)</pre>
    <h5>
     10.
     <code>
      cacheWidth
     </code>
     和
     <code>
      cacheHeight
     </code>
    </h5>
    <p>
     设置图片的缓存分辨率（以像素为单位），用于优化内存占用：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  cacheWidth: 100,
  cacheHeight: 100,
)</pre>
    <h5>
     11.
     <code>
      repeat
     </code>
    </h5>
    <p>
     设置图片的重复方式，例如平铺：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  repeat: ImageRepeat.repeat,
)</pre>
    <h5>
     12.
     <code>
      filterQuality
     </code>
    </h5>
    <p>
     设置图片的过滤质量，例如
     <code>
      FilterQuality.low
     </code>
     或
     <code>
      FilterQuality.high
     </code>
     ：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>Image.network(
  'https://example.com/image.png',
  filterQuality: FilterQuality.low,
)</pre>
    <hr/>
    <h4>
     完整示例
    </h4>
    <p>
     以下是一个完整的示例，展示了
     <code>
      Image.network
     </code>
     的常见用法：
    </p>
    <p>
     dart
    </p>
    <p>
     复制
    </p>
    <pre>import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Image.network 示例')),
        body: Center(
          child: Image.network(
            'https://example.com/image.png',
            width: 300,
            height: 200,
            fit: BoxFit.cover,
            headers: {
              'Cache-Control': 'no-cache',
            },
            loadingBuilder: (BuildContext context, Widget child, ImageChunkEvent? loadingProgress) {
              if (loadingProgress == null) {
                return child;
              }
              return CircularProgressIndicator(
                value: loadingProgress.expectedTotalBytes != null
                    ? loadingProgress.cumulativeBytesLoaded / loadingProgress.expectedTotalBytes!
                    : null,
              );
            },
            errorBuilder: (BuildContext context, Object error, StackTrace? stackTrace) {
              return Text('加载失败');
            },
          ),
        ),
      ),
    );
  }
}</pre>

     </code>
