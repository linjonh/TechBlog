    <h2>
     一、区块链概念
    </h2>
    <div>
     <span style="color:#000000;">
      （1）
     </span>
     <span style="color:#ff0000;">
      <strong>
       它作为一个去
      </strong>
     </span>
     <span style="color:#ffc000;">
      <strong>
       中心化
      </strong>
     </span>
     <span style="color:#ff0000;">
      <strong>
       的分布式账本，
      </strong>
     </span>
     <strong>
      其本身是一系列使用密码学而产生的具有互相关了的数据块。
     </strong>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      2）区块链实质是一个不断增长的分布式结算数据库，使用算法证明机制来解决信息系统中的信任危机。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      3）
     </span>
     <span style="color:#ffc000;">
      <strong>
       区块链的运行及分布式数据存储和点对点传输，且需要共识机制、加密算法（哈希，非对称加密）
      </strong>
     </span>
     <span style="color:#ffc000;">
      <strong>
       等计算机技术来辅助记录数。
      </strong>
     </span>
    </div>
    <h4>
     <img alt="" height="170" src="https://i-blog.csdnimg.cn/blog_migrate/bcbbab7f6b4dd37d4857f2eed1a9a5e9.png" width="155">
      <span style="color:#0d0016;">
       <strong>
        现代世界需要区块链这个去中心化的分布式账本的原因：
       </strong>
      </span>
      <span style="color:#be191c;">
       <strong>
        价值转移
       </strong>
      </span>
      <span style="color:#1a439c;">
       <strong>
        和
       </strong>
      </span>
      <span style="color:#be191c;">
       <strong>
        信用建设
       </strong>
      </span>
      <span style="color:#1a439c;">
       <strong>
        。
       </strong>
      </span>
      <img alt="" height="60" src="https://i-blog.csdnimg.cn/blog_migrate/79e922e0e3394761e48a181ba5bae2f3.png" width="530"/>
     </img>
    </h4>
    <p>
     <img alt="" height="51" src="https://i-blog.csdnimg.cn/blog_migrate/b60215d1f1cc1b0462f51377a9e7b4d0.png" width="526"/>

    <h3>
     技术基础：
     <span style="color:#1c7331;">
      去中心化
     </span>
     和
     <span style="color:#1c7331;">
      信用共识
     </span>
     。
     <br/>
     区块链的实质：
     <span style="color:#511b78;">
      一个不断增长的分布式结算数据库。
     </span>
    </h3>
    <h3>
     公信新格局：在区块链技术支持下，信任建立在区块链上，而不由单个组织掌控，着将有利于公信力被多方交叉验证与监督，
     <span style="color:#511b78;">
      形成更加透明公正的公信格局。
     </span>
    </h3>
    <h3>
     公信力的特点：
     <span style="color:#1c7331;">
      分布式
     </span>
     和
     <span style="color:#1c7331;">
      公证人
     </span>
    </h3>
    <h3>
     <span style="color:#0d0016;">
      总结：
     </span>
     <span style="color:#fe2c24;">
      区块链是一台创造信任的机器，最核心的作用就是解决信用共识的问题。
     </span>
    </h3>
    <hr/>
    <h2>
     二、区块链技术特点
    </h2>
    <p>
     <img alt="" height="244" src="https://i-blog.csdnimg.cn/blog_migrate/2c6935fd153512da9635ea3e9661e256.png" width="482"/>
    </p>
    <div>
     <span style="color:#000000;">
      （1）
     </span>
     <span style="color:#ffc000;">
      <strong>
       网络分散
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       区块链网络，完全没有中央服务器和控制中心。由网络自己做出决定，从而控制自己。
      </strong>
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      2）
     </span>
     <span style="color:#ffc000;">
      <strong>
       可追溯性
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       区块链是一个分散的数据库，分散数据库记录了区块链每笔交易的输入输出，从而可以
      </strong>
     </span>
     <span style="color:#ff0000;">
      <strong>
       轻松的追踪资产数量变化和交易活动。
      </strong>
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      3）
     </span>
     <span style="color:#ffc000;">
      <strong>
       共识机制
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       是区块链事务达成分散式共识的算法。区块链的节点是分散的，没有做出决策的中心化
      </strong>
     </span>
     <span style="color:#ff0000;">
      <strong>
       权威机构。网络中保存数据的所有节点，必须汇集在一起共同决策，共识机制，决定了区块链数据库的
      </strong>
     </span>
     <span style="color:#ff0000;">
      <strong>
       真实性。
      </strong>
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      4）
     </span>
     <span style="color:#ffc000;">
      <strong>
       安全性
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       区块链采用分散式数据库，利用散列和算法保障数据安全，安全性是在网络中集体创建的，
      </strong>
     </span>
     <span style="color:#ff0000;">
      <strong>
       没有任何一方需要负责安全，安全性由参与者共同授予。
      </strong>
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      5）
     </span>
     <span style="color:#ffc000;">
      <strong>
       高可用性
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       24*7 全天运行，无权限限制，内容无法篡改，何人何时何地均可使用。
      </strong>
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      6）
     </span>
     <span style="color:#ffc000;">
      <strong>
       不变化
      </strong>
     </span>
     <span style="color:#000000;">
      ：
     </span>
     <span style="color:#ff0000;">
      <strong>
       任何放入区块链的信息，都会永远存在区块链上，不会消失。
      </strong>
     </span>
    </div>
    <h3>
     <span style="color:#511b78;">
      比特币：

    <h2>
     三、区块链的技术价值
    </h2>
    <div>
     <span style="color:#000000;">
      （1）
     </span>
     <span style="color:#ff0000;">
      <strong>
       安全
      </strong>
     </span>
     <span style="color:#000000;">
      :基于去中心化 P2P 网络技术，区块链系统中所有参与节点将共同完成数 据的存储、维护，可
     </span>
     <span style="color:#000000;">
      有效避免中心化数据系统遭到攻击时数据泄露的风险。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      2）
     </span>
     <span style="color:#ff0000;">
      <strong>
       确权
      </strong>
     </span>
     <span style="color:#000000;">
      ：区块链的另一个价值体现在数据资产所有权的确认上。数据一经上链， 即可通过区块链网络
     </span>
     <span style="color:#000000;">
      确定与用户间的锚定关系，且后续的每一次数据操作都会被准 确记录，不可篡改。该特点可对数据资产
     </span>
     <span style="color:#000000;">
      进行有效保护，防止他人恶意篡改、窃取 与利用；
     </span>
    </div>
    <div>
     （
     <span style="color:#000000;">
      3）
     </span>
     <span style="color:#ff0000;">
      <strong>
       信任
      </strong>
     </span>
     <span style="color:#000000;">
      ：去中心化区块链系统中，数据账本存于各节点，信息公开透明，为数据 可信提供了基础保证。
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      4）
     </span>
     <span style="color:#ff0000;">
      <strong>
       自动化
      </strong>
     </span>
     <span style="color:#000000;">
      ：智能合约是一种可编程化的数字协议，当合约参与方满足触发条件后， 合约条款将自动执
     </span>
     <span style="color:#000000;">
      行。智能合约的加入，为区块链带来了新的契机，保障部署在区块链上的合同条款可以实现自动、去中
     </span>
     <span style="color:#000000;">
      心化的计算，从而为诸多领域（尤其针对流程复杂繁琐、效率低下的领域）的效率提升、成本控制起到
     </span>
     <span style="color:#000000;">
      不可忽视的作用；
     </span>
    </div>
    <div>
     <span style="color:#000000;">
      （
     </span>
     <span style="color:#000000;">
      5）
     </span>
     <span style="color:#ff0000;">
      <strong>
       价值共享
      </strong>
     </span>
     <span style="color:#000000;">
      :传统中心化系统常涉及较多环节，参与中介多、流程复杂，耗费成本较高。而区块链系统
     </span>
     <span style="color:#000000;">
      则实现了各参与节点的去中心化连接，且各节点数据资产可 进行点对点自由流通，以降低中间成本。
     </span>
     <hr/>
     <h2>
      四、区块链分类
     </h2>
     <div>
      <span style="color:#000000;">
       （
      </span>
      <span style="color:#000000;">
       1）
      </span>
      <span style="color:#ff0000;">
       <strong>
        按开放程度划分：公有链、联盟链、私有链
       </strong>
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       ①、
      </span>
      <span style="color:#ff0000;">
       <strong>
        公有链
       </strong>
      </span>
      <span style="color:#000000;">
       ：系统全开放，人人可参与（
      </span>
      <span style="color:#000000;">
       BTC：比特币,ETH：以太坊）
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       ②、
      </span>
      <span style="color:#ff0000;">
       <strong>
        联盟链
       </strong>
      </span>
      <span style="color:#000000;">
       ：系统 半开放，仅限联盟成员参与（
      </span>
      <span style="color:#000000;">
       R3 联盟：国际银行和金融机构组成、原本链：横跨全球
      </span>
      <span style="color:#000000;">
       五大洲部署节点）
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       ③、
      </span>
      <span style="color:#ff0000;">
       <strong>
        私有链
       </strong>
      </span>
      <span style="color:#000000;">
       ：系统为封闭，仅限个人或公司内部参与（Multichain：平台，创建和部署私链）
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       （
      </span>
      <span style="color:#000000;">
       2）
      </span>
      <span style="color:#ff0000;">
       <strong>
        按应用范围划分：基础链、行业链
       </strong>
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       ①、
      </span>
      <span style="color:#ff0000;">
       <strong>
        基础链
       </strong>
      </span>
      <span style="color:#000000;">
       ：一般以公有链有主，提供接口快速开发相关应用。（
      </span>
      <span style="color:#000000;">
       ETH, EOS:企业操作系统）
      </span>
     </div>
     <div>
      <span style="color:#000000;">
       ②、
      </span>
      <span style="color:#ff0000;">
       <strong>
        行业链
       </strong>
      </span>
      <span style="color:#000000;">
       ：为行业提供相关基础工具和协议。（
      </span>
      <span style="color:#000000;">
       BTM：比原链）
      </span>
     </div>
