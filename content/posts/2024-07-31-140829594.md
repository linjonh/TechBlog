    <h3 id="小程序开发详细教程">
     小程序开发详细教程
    </h3>
    <h4 id="1--环境准备">
     1. 环境准备
    </h4>
    <h5 id="1-1-注册账号">
     1.1 注册账号
    </h5>
    <ul>
     <li>
      <strong>
       微信小程序
      </strong>
      ：
      <ol>
       <li>
        前往
        <a href="https://mp.weixin.qq.com/" rel="nofollow" title="微信公众平台">
         微信公众平台
        </a>
        。
       </li>
       <li>
        点击“注册”，选择“小程序”。
       </li>
       <li>
        按照提示填写相关信息，完成注册。
       </li>
       <li>
        注册完成后，你将获得一个AppID，用于后续开发。
       </li>
      </ol>
     </li>
    </ul>
    <h5 id="1-2-安装开发工具">
     1.2 安装开发工具
    </h5>
    <ul>
     <li>
      下载并安装
      <a href="https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html" rel="nofollow" title="微信开发者工具">
       微信开发者工具
      </a>
      。
     </li>
     <li>
      安装完成后，打开开发者工具。
     </li>
    </ul>
    <h4 id="2--创建第一个小程序">
     2. 创建第一个小程序
    </h4>
    <h5 id="2-1-创建项目">
     2.1 创建项目
    </h5>
    <ol>
     <li>
      打开微信开发者工具，选择“新建小程序”。
     </li>
     <li>
      输入你的AppID（如果没有，可以选择“无AppID”进行测试）。
     </li>
     <li>
      填写项目名称和项目路径，点击“创建”。
     </li>
     <li>
      选择“使用默认模板”以快速开始。
     </li>
    </ol>
    <h5 id="2-2-项目结构">
     2.2 项目结构
    </h5>
    <p>
     小程序的基本项目结构如下：
    </p>
    <pre><code>project
│
├── miniprogram
│   ├── pages
│   │   ├── index
│   │   │   ├── index.js
│   │   │   ├── index.json
│   │   │   ├── index.wxml
│   │   │   └── index.wxss
│   │   └── ...
│   ├── app.js
│   ├── app.json
│   └── app.wxss
└── ...
</code>​</pre>
    <h4 id="3--编写代码">
     3. 编写代码
    </h4>
    <h5 id="3-1-app-json">
     3.1 app.json
    </h5>
    <p>
     在
     <code>
      app.json
     </code>
     中配置小程序的页面路径和窗口样式：
    </p>
    <pre><code>{
  "pages": [
    "pages/index/index"
  ],
  "window": {
    "navigationBarTitleText": "我的小程序",
    "navigationBarBackgroundColor": "#ffffff",
    "navigationBarTextStyle": "black"
  }
}
</code>​</pre>
    <ul>
     <li>
      <strong>
       pages
      </strong>
      ：定义小程序的页面路径。
     </li>
     <li>
      <strong>
       window
      </strong>
      ：设置小程序的窗口样式。
     </li>
    </ul>
    <h5 id="3-2-app-js">
     3.2 app.js
    </h5>
    <p>
     在
     <code>
      app.js
     </code>
     中编写小程序的逻辑代码（通常可以保持为空）：
    </p>
    <pre><code>App({
  onLaunch: function () {
    // 小程序初始化
    console.log("小程序启动");
  }
});
</code>​</pre>
    <ul>
     <li>
      <code>
       onLaunch
      </code>
      ：小程序初始化时触发的事件。
     </li>
    </ul>
    <h5 id="3-3-index-wxml">
     3.3 index.wxml
    </h5>
    <p>
     在
     <code>
      index.wxml
     </code>
     中编写页面的结构：
    </p>
    <pre><code>&lt;view class="container"&gt;
  &lt;text class="title"&gt;欢迎来到我的小程序！&lt;/text&gt;
  &lt;button bindtap="onTap"&gt;点击我&lt;/button&gt;
&lt;/view&gt;
</code>​</pre>
    <ul>
     <li>
      <code>
       &lt;view&gt;
      </code>
      ：相当于HTML中的
      <code>
       &lt;div&gt;
      </code>
      ，用于布局。
     </li>
     <li>
      <code>
       &lt;text&gt;
      </code>
      ：用于显示文本。
     </li>
     <li>
      <code>
       &lt;button&gt;
      </code>
      ：按钮组件，
      <code>
       bindtap
      </code>
      用于绑定点击事件。
     </li>
    </ul>
    <h5 id="3-4-index-wxss">
     3.4 index.wxss
    </h5>
    <p>
     在
     <code>
      index.wxss
     </code>
     中编写页面的样式：
    </p>
    <pre><code>.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}

.title {
font-size: 24px;
color: #333;
margin-bottom: 20px;
}
</code>​</pre>
<ul>
<li>
使用 CSS 样式来美化页面。
</li>
</ul>
<h5 id="3-5-index-js">
3.5 index.js
</h5>
<p>
在
<code>
index.js
</code>
中编写页面的逻辑：
</p>
<pre><code>Page({
onTap: function() {
wx.showToast({
title: '按钮被点击了！',
icon: 'success'
});
}
});
</code>​</pre>
<ul>
<li>
<code>
Page
</code>
：定义一个页面，
<code>
onTap
</code>
是按钮点击事件的处理函数。
</li>
</ul>
<h4 id="4--运行和调试"> 4. 运行和调试
</h4>
<ul>
<li>
在微信开发者工具中，点击“编译”按钮，查看效果。
</li>
<li>
使用“调试”功能，可以查看控制台输出和网络请求。
</li>
<li>
可以在控制台中使用
<code>
console.log
</code>
输出调试信息。
</li>
</ul>
<h4 id="5--添加更多页面"> 5. 添加更多页面
</h4>
<p>
如果你想添加更多页面，可以按照以下步骤进行：
</p>
<h5 id="5-1-创建新页面">
5.1 创建新页面
</h5>
<ol>
<li>
在
<code>
pages
</code>
文件夹中创建新的页面文件夹，例如
<code>
about
</code>
。
</li>
<li>
在
<code>
about
</code>
文件夹中创建
<code>
about.js
</code>
、
<code>
about.json
</code>
、
<code>
about.wxml
</code>
和
<code>
about.wxss
</code>
文件。
</li>
</ol>
<h5 id="5-2-配置新页面">
5.2 配置新页面
</h5>
<p>
在
<code>
app.json
</code>
中添加新页面的路径：
</p>
<pre><code>{
"pages": [
"pages/index/index",
"pages/about/about"
]
}
</code>​</pre>
<h5 id="5-3-编写新页面代码">
5.3 编写新页面代码
</h5>
<p>
在
<code>
about.wxml
</code>
中编写页面结构：
</p>
<pre><code>&lt;view class="about-container"&gt;
&lt;text&gt;这是关于页面&lt;/text&gt;
&lt;button bindtap="goBack"&gt;返回首页&lt;/button&gt;
&lt;/view&gt;
</code>​</pre>
<p>
在
<code>
about.wxss
</code>
中编写样式：
</p>
<pre><code>.about-container {
display: flex;
flex-direction: column;
align-items: center;
justify-content: center;
height: 100vh;
}
</code>​</pre>
<p>
在
<code>
about.js
</code>
中编写逻辑：
</p>
<pre><code>Page({
goBack: function() {
wx.navigateBack(); // 返回上一页
}
});
</code>​</pre>
<h4 id="6--路由导航"> 6. 路由导航
</h4>
<p>
在
<code>
index.wxml
</code>
中添加一个按钮，跳转到关于页面：
</p>
<pre><code>&lt;button bindtap="goToAbout"&gt;关于&lt;/button&gt;
</code>​</pre>
<p>
在
<code>
index.js
</code>
中添加跳转逻辑：
</p>
<pre><code>Page({
onTap: function() {
wx.showToast({
title: '按钮被点击了！',
icon: 'success'
});
},
goToAbout: function() {
wx.navigateTo({
url: '/pages/about/about'
});
}
});
</code>​</pre>
