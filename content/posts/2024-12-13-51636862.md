    <h2 id="css预处理器">
     CSS预处理器
    </h2>
    <pre><code>css预处理器基本思想就是用一种专门的编程语言，为css增加了一些编程的特性，将css作为目标生成文件，然后开发者就只要使用这种语言进行编码的工作
</code></pre>
    <p>
     通俗来讲：“css预处理器用一种专门的编程语言，进行web页面样式的设计，然后再编译成正常的css文件，以供项目使用。css预处理器为css增加一些编程的特性，无需考虑浏览器的兼容性问题”
    </p>
    <p>
     <strong>
      下面介绍了常用的两种css预处理器
     </strong>
    </p>
    <h3 id="sass">
     Sass
    </h3>
    <blockquote>
     <p>
      Sass语法是Sass最初的语法格式，通过tab键控制缩进的一种语法规则，不带任何分号和大括号，以.sass为扩展名
     </p>
    </blockquote>
    <p>
     <strong>
      Scss:
     </strong>
    </p>
    <blockquote>
     <p>
      Scss是Sass的新语法格式，不过他的语法更符合前端人员的书写习惯，即代码包裹在大括号里，并且末尾借书处有分号，以.scss为扩展名
     </p>
    </blockquote>
    <h4 id="sass的编译">
     Sass的编译
    </h4>
    <ul>
     <li>
      命令编译
     </li>
     <li>
      GUI工具编译
     </li>
     <li>
      自动化编译
     </li>
    </ul>
    <h4 id="变量">
     变量
    </h4>
    <p>
     sass的变量必须$开头，后面紧跟变量名，变量值和变量名之间用冒号隔开（同css），如果值后面加上!default则表示默认值
    </p>
    <h4 id="嵌套">
     嵌套
    </h4>
    <p>
     sass嵌套分为：
     <br/>
     1.选择器嵌套
     <br/>
     2.属性的嵌套
     <br/>
     一般用到的都是选择器嵌套
    </p>
    <h5 id="选择器嵌套">
     选择器嵌套
    </h5>
    <blockquote>
     <p>
      选择器嵌套指的是在一个选择器中嵌套另一个选择器来实现继承，从而增强了sass文件的结构性和可读性。
      <br/>
      在选择器嵌套中，可以使用
      <strong>
       &amp;表示父元素选择器
      </strong>
     </p>
    </blockquote>
    <p>
     例如：
    </p>
    <pre class="prettyprint"><code class="hljs scss"><span class="hljs-id">#top_nav</span>{
  <span class="hljs-attribute">line-height</span><span class="hljs-value">: <span class="hljs-number">40</span>px;</span>
  <span class="hljs-attribute">text-transform</span><span class="hljs-value">: capitalize;</span>
  <span class="hljs-attribute">background-color</span><span class="hljs-value">:<span class="hljs-hexcolor">#333</span>;</span>
  <span class="hljs-tag">li</span>{
    <span class="hljs-attribute">float</span><span class="hljs-pseudo">:left</span>;
  }
  <span class="hljs-tag">a</span>{
    <span class="hljs-attribute">display</span><span class="hljs-value">: block;</span>
    <span class="hljs-attribute">padding</span><span class="hljs-value">: <span class="hljs-number">0</span> <span class="hljs-number">10</span>px;</span>
    <span class="hljs-attribute">color</span><span class="hljs-value">: <span class="hljs-hexcolor">#fff</span>;</span>

    &amp;<span class="hljs-pseudo">:hover</span>{
      <span class="hljs-attribute">color</span><span class="hljs-value">:<span class="hljs-hexcolor">#ddd</span>;</span>
    }
  }
}</code></pre>
    <p>
     编译成功后的css为：
    </p>
    <pre class="prettyprint"><code class="hljs css"><span class="hljs-id">#top_nav</span><span class="hljs-rules">{

  <span class="hljs-rule"><span class="hljs-attribute">text-transform</span>:<span class="hljs-value"> capitalize</span></span>;
  <span class="hljs-rule"><span class="hljs-attribute">background-color</span>:<span class="hljs-value"><span class="hljs-hexcolor">#333</span></span></span>;

<span class="hljs-id">#top_nav</span> <span class="hljs-tag">li</span><span class="hljs-rules">{

<span class="hljs-rule">}</span></span>
<span class="hljs-id">#top_nav</span> <span class="hljs-tag">a</span><span class="hljs-rules">{

<span class="hljs-rule">}</span></span></code></pre>
    <h5 id="属性嵌套">
     属性嵌套
    </h5>
    <blockquote>
     <p>
      属性嵌套指的是有些属性拥有同一个开始单词，如border-width，border-color都是以border开头,可以在sass编写中省去相同开头
     </p>
    </blockquote>
    <p>
     例如：
    </p>
    <pre class="prettyprint"><code class="hljs css"><span class="hljs-class">.fakeshadow</span> <span class="hljs-rules">{
  <span class="hljs-rule"><span class="hljs-attribute">border</span>:<span class="hljs-value"> {
    style: solid</span></span>;
    <span class="hljs-rule"><span class="hljs-attribute">left</span>:<span class="hljs-value"> {
      width: <span class="hljs-number">4</span>px</span></span>;
      <span class="hljs-rule"><span class="hljs-attribute">color</span>:<span class="hljs-value"> <span class="hljs-hexcolor">#888</span></span></span>;
    <span class="hljs-rule">}</span></span>
    <span class="hljs-tag">right</span>: <span class="hljs-rules">{
      <span class="hljs-rule"><span class="hljs-attribute">width</span>:<span class="hljs-value"> <span class="hljs-number">2</span>px</span></span>;
      <span class="hljs-rule"><span class="hljs-attribute">color</span>:<span class="hljs-value"> <span class="hljs-hexcolor">#ccc</span></span></span>;
    <span class="hljs-rule">}</span></span>
  }
}</code></pre>
    <p>
     编译成功：
    </p>
    <pre class="prettyprint"><code class="hljs css"><span class="hljs-class">.fakeshadow</span> <span class="hljs-rules">{
  <span class="hljs-rule"><span class="hljs-attribute">border-style</span>:<span class="hljs-value"> solid</span></span>;
  <span class="hljs-rule"><span class="hljs-attribute">border-left-width</span>:<span class="hljs-value"> <span class="hljs-number">4</span>px</span></span>;
  <span class="hljs-rule"><span class="hljs-attribute">border-left-color</span>:<span class="hljs-value"> <span class="hljs-hexcolor">#888</span></span></span>;

  <span class="hljs-rule"><span class="hljs-attribute">border-right-color</span>:<span class="hljs-value"> <span class="hljs-hexcolor">#ccc</span></span></span>; 
<span class="hljs-rule">}</span></span></code></pre>
    <h4 id="at-root跳出选择器嵌套">
     @at-root跳出选择器嵌套
    </h4>
    <h4 id="混合mixin">
     混合(mixin)
    </h4>
    <blockquote>
     <p>
      sass中使用@mixin声明混合，可以传递参数，参数名以$符号开始，多个参数以逗号分开，也可以给参数设置默认值。声明的@mixin通过@include来调用。
     </p>
    </blockquote>
    <pre class="prettyprint"><code class="hljs scss"><span class="hljs-comment">//sass style</span>
<span class="hljs-comment">//-------------------------------   </span>
<span class="hljs-at_rule">@<span class="hljs-keyword">mixin</span><span class="hljs-preprocessor"> opacity</span>($opacity:<span class="hljs-number">50</span>) {<!-- --></span>
  <span class="hljs-attribute">opacity</span><span class="hljs-value">: $opacity / <span class="hljs-number">100</span>;</span>
  <span class="hljs-attribute">filter</span><span class="hljs-value">: alpha(opacity=$opacity);</span>
}

<span class="hljs-comment">//css style</span>
<span class="hljs-comment">//-------------------------------</span>
<span class="hljs-class">.opacity</span>{
  <span class="hljs-at_rule">@<span class="hljs-keyword">include</span><span class="hljs-preprocessor"> opacity</span>;</span> <span class="hljs-comment">//参数使用默认值</span>
}
<span class="hljs-class">.opacity-80</span>{
  <span class="hljs-at_rule">@<span class="hljs-keyword">include</span><span class="hljs-preprocessor"> opacity</span>(<span class="hljs-number">80</span>);</span> <span class="hljs-comment">//传递参数</span>
}</code></pre>
    <p>
     参数可以使用无参，也可以多个参数
    </p>
    <h4 id="继承">
     继承
    </h4>
    <blockquote>
     <p>
      sass中，选择器继承可以让选择器继承另一个选择器的所有样式，并联合声明。使用选择器的继承，要使用关键词@extend，后面紧跟需要继承的选择器
     </p>
    </blockquote>
    <pre class="prettyprint"><code class="hljs scss"><span class="hljs-comment">//sass style</span>
<span class="hljs-comment">//-------------------------------</span>
<span class="hljs-tag">h1</span>{
  <span class="hljs-attribute">border</span><span class="hljs-value">: <span class="hljs-number">4</span>px solid <span class="hljs-hexcolor">#ff9aa9</span>;</span>
}
<span class="hljs-class">.speaker</span>{
  <span class="hljs-at_rule">@<span class="hljs-keyword">extend</span><span class="hljs-preprocessor"> h1</span>;</span>
  <span class="hljs-attribute">border-width</span><span class="hljs-value">: <span class="hljs-number">2</span>px;</span>
}

<span class="hljs-comment">//css style</span>
<span class="hljs-comment">//-------------------------------</span>
<span class="hljs-tag">h1</span>,<span class="hljs-class">.speaker</span>{
  <span class="hljs-attribute">border</span><span class="hljs-value">: <span class="hljs-number">4</span>px solid <span class="hljs-hexcolor">#ff9aa9</span>;</span>
}
<span class="hljs-class">.speaker</span>{
  <span class="hljs-attribute">border-width</span><span class="hljs-value">: <span class="hljs-number">2</span>px;</span>
}</code></pre>
    <p>
     关于sass的语法还有很多，例如函数，运算，条件判断，循环等等
     <br/>
     推荐一个学习sass的网站：
     <a href="http://www.w3cplus.com/sassguide/syntax.html" rel="nofollow">
      http://www.w3cplus.com/sassguide/syntax.html
     </a>
    </p>
    <h3 id="less">
     Less
    </h3>
    <p>
     <strong>
      LESS和Sass的主要不同就是他们的实现方式，LESSS是基于JavaScript，所以，是在客户端处理的。Sass是基于Ruby的，是在服务器端处理的
     </strong>
     <br/>
     - 变量
     <br/>
     Less和Sass的语法规则大同小异，不过Less引用变量的形式为@变量名值
     <br/>
     - 混合(mixin)
     <br/>
     在Sass中，要在样式规则前面添加@mixin声明，规定它是个嵌套。然后，通过@include来调用它：
    </p>
    <pre class="prettyprint"><code class="hljs scss"><span class="hljs-at_rule">@<span class="hljs-keyword">mixin</span><span class="hljs-preprocessor"> border</span> {<!-- --></span> <span class="hljs-attribute">border-top</span><span class="hljs-value">: <span class="hljs-number">1</span>px dotted <span class="hljs-hexcolor">#333</span>;</span> } <span class="hljs-tag">article</span><span class="hljs-class">.post</span> { <span class="hljs-attribute">background</span><span class="hljs-value">: <span class="hljs-hexcolor">#eee</span>;</span> <span class="hljs-at_rule">@<span class="hljs-keyword">include</span><span class="hljs-preprocessor"> border</span>;</span> } <span class="hljs-tag">ul</span><span class="hljs-class">.menu</span> { <span class="hljs-attribute">background</span><span class="hljs-value">: <span class="hljs-hexcolor">#ccc</span>;</span> <span class="hljs-at_rule">@<span class="hljs-keyword">include</span><span class="hljs-preprocessor"> border</span>;</span> }</code></pre>
    <p>
     但在Less中：
    </p>
    <pre class="prettyprint"><code class="hljs scss"><span class="hljs-class">.border</span> { <span class="hljs-attribute">border-top</span><span class="hljs-value">: <span class="hljs-number">1</span>px dotted <span class="hljs-hexcolor">#333</span>;</span> } <span class="hljs-tag">article</span><span class="hljs-class">.post</span> { <span class="hljs-attribute">background</span><span class="hljs-value">: <span class="hljs-hexcolor">#eee</span>;</span> <span class="hljs-class">.border</span>; } <span class="hljs-tag">ul</span><span class="hljs-class">.menu</span> { <span class="hljs-attribute">background</span><span class="hljs-value">: <span class="hljs-hexcolor">#ccc</span>;</span> <span class="hljs-class">.border</span>; }即可完成</code></pre>
    <p>
     <strong>
      注释：
     </strong>
     <br/>
     LESS中允许两种注释写法。标准的CSS注释，/* comment */，是有效的，而且能够通过处理并正确输出。当行注释，// comment，同样可以用但是不能够通过处理也不能被输出
    </p>
