     一、知识讲解
    </h3>
    <h4 id="1.%20Hooks%20%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F" name="1.%20Hooks%20%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F">
     1. Hooks 是什么？
    </h4>
    <p>
     React Hooks 是 React 16.8 引入的革命性特性，它允许开发者在函数组件中使用状态（state）和其他 React 特性。Hooks 的诞生解决了以下问题：
    </p>
    <ul>
     <li>
      类组件中生命周期函数带来的逻辑分散
     </li>
     <li>
      高阶组件带来的嵌套地狱（Wrapper Hell）
     </li>
     <li>
      函数组件无法使用状态的局限
     </li>
     <li>
      状态逻辑难以复用的问题
     </li>
    </ul>
    <h4 id="2.%20useState%20%E7%9A%84%E4%BD%9C%E7%94%A8" name="2.%20useState%20%E7%9A%84%E4%BD%9C%E7%94%A8">
     2. useState 的作用
    </h4>
    <p>
     useState 是最基础且重要的 Hook，它让函数组件具备管理局部状态的能力。其核心功能包括：
    </p>
    <ul>
     <li>
      声明组件内部状态
     </li>
     <li>
      提供状态更新方法
     </li>
     <li>
      触发组件重新渲染
     </li>
     <li>
      保持状态在多次渲染间的持久性
     </li>
    </ul>
    <h4 id="3.%20%E5%9F%BA%E6%9C%AC%E8%AF%AD%E6%B3%95%E8%A7%A3%E6%9E%90" name="3.%20%E5%9F%BA%E6%9C%AC%E8%AF%AD%E6%B3%95%E8%A7%A3%E6%9E%90">
     3. 基本语法解析
    </h4>
    <p>
     JAVASCRIPT
    </p>
    <blockquote>
     <pre><code>const [state, setState] = useState(initialState);
</code></pre>
     <p>
     </p>
    </blockquote>
    <ul>
     <li>
      <code>
       initialState
      </code>
      ：状态的初始值，可以是任何数据类型
     </li>
     <li>
      <code>
       state
      </code>
      ：当前状态值
     </li>
     <li>
      <code>
       setState
      </code>
      ：用于更新状态的函数
     </li>
    </ul>
    <h4 id="4.%20%E5%B7%A5%E4%BD%9C%E5%8E%9F%E7%90%86" name="4.%20%E5%B7%A5%E4%BD%9C%E5%8E%9F%E7%90%86">
     4. 工作原理
    </h4>
    <ul>
     <li>
      <strong>
       初始化阶段
      </strong>
      ：组件首次渲染时，useState 会读取初始值并保存
     </li>
     <li>
      <strong>
       更新阶段
      </strong>
      ：调用 setState 后：
      <ol>
       <li>
        将新值加入更新队列
       </li>
       <li>
        触发组件重新渲染
       </li>
       <li>
        在下次渲染时返回更新后的状态
       </li>
      </ol>
     </li>
     <li>
      <strong>
       渲染机制
      </strong>
      ：每次渲染都有独立的状态快照（闭包特性）
     </li>
    </ul>
    <h4 id="5.%20%E5%8F%82%E6%95%B0%E8%AF%A6%E8%A7%A3" name="5.%20%E5%8F%82%E6%95%B0%E8%AF%A6%E8%A7%A3">
     5. 参数详解
    </h4>
    <h5 id="a)%20%E5%88%9D%E5%A7%8B%E5%80%BC%E8%AE%BE%E7%BD%AE%E6%96%B9%E5%BC%8F" name="a)%20%E5%88%9D%E5%A7%8B%E5%80%BC%E8%AE%BE%E7%BD%AE%E6%96%B9%E5%BC%8F">
     a) 初始值设置方式
    </h5>
    <p>
     JAVASCRIPT
    </p>
    <blockquote>
     <pre><code>// 直接值初始化
const [count, setCount] = useState(0);

// 函数式初始化（惰性初始化）
const [count, setCount] = useState(() =&gt; {
  const initialValue = calculateExpensiveValue();
  return initialValue;
});
</code></pre>
    </blockquote>
    <h5 id="b)%20%E6%9B%B4%E6%96%B0%E5%87%BD%E6%95%B0%E7%89%B9%E6%80%A7" name="b)%20%E6%9B%B4%E6%96%B0%E5%87%BD%E6%95%B0%E7%89%B9%E6%80%A7">
     b) 更新函数特性
    </h5>
    <ul>
     <li>
      支持直接赋值更新：
      <p>
       JAVASCRIPT
      </p>
      <blockquote>
       <pre><code>setCount(5);
</code></pre>
      </blockquote>
     </li>
     <li>
      支持函数式更新（推荐用于依赖前值的场景）：
      <p>
       JAVASCRIPT
      </p>
      <blockquote>
       <pre><code>setCount(prevCount =&gt; prevCount + 1);
</code></pre>
      </blockquote>
     </li>
    </ul>
    <h4 id="6.%20%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9" name="6.%20%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9">
     6. 注意事项
    </h4>
    <ul>
     <li>
      <strong>
       异步更新
      </strong>
      ：状态更新是异步的，不能立即获取到新值
     </li>
     <li>
      <strong>
       合并更新
      </strong>
      ：React 会自动合并多个 setState 调用
     </li>
     <li>
      <strong>
       性能优化
      </strong>
      ：使用函数式更新避免闭包陷阱
     </li>
     <li>
      <strong>
       不可变原则
      </strong>
      ：始终返回新值而不是修改原值
     </li>
    </ul>
    <h4 id="7.%20%E7%B1%BB%E7%BB%84%E4%BB%B6%E5%AF%B9%E6%AF%94" name="7.%20%E7%B1%BB%E7%BB%84%E4%BB%B6%E5%AF%B9%E6%AF%94">

    <h4 id="8.%20%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98%E8%A7%A3%E7%AD%94" name="8.%20%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98%E8%A7%A3%E7%AD%94">
     8. 常见问题解答
    </h4>
    <p>
     <strong>
      Q：useState 和类组件的 state 有何不同？
     </strong>
     <br/>
     A：useState 不会自动合并对象类型的状态，需要手动合并
    </p>
    <p>
     <strong>
      Q：为什么 setCount(count + 1) 在异步操作中不可靠？
     </strong>
     <br/>
     A：由于闭包特性，异步回调中的 count 可能不是最新值，应该使用函数式更新
    </p>
    <p>
     <strong>
      Q：可以在条件语句中使用 useState 吗？
     </strong>
     <br/>
     A：不行，所有 Hooks 都必须保证每次渲染的调用顺序一致
    </p>
    <p>
     <strong>
      Q：如何存储复杂对象？
     </strong>
    </p>
    <p>
     JAVASCRIPT
    </p>
    <blockquote>
     <pre><code>const [user, setUser] = useState({
  name: 'John',
  age: 25,
  address: {
    city: 'New York',
    zip: '10001'
  }
});

// 正确更新方式
setUser(prev =&gt; ({
  ...prev,
  age: 26,
  address: {
    ...prev.address,
    zip: '10002'
  }
}));
</code></pre>
    </blockquote>
    <h3 id="%E4%BA%8C%E3%80%81%E6%A0%B8%E5%BF%83%E4%BB%A3%E7%A0%81%E7%A4%BA%E4%BE%8B" name="%E4%BA%8C%E3%80%81%E6%A0%B8%E5%BF%83%E4%BB%A3%E7%A0%81%E7%A4%BA%E4%BE%8B">
     二、核心代码示例
    </h3>
    <p>
     JSX
    </p>
    <blockquote>
     <pre><code>import React, { useState } from 'react';

function Counter() {
  // 声明状态变量 count 及其更新函数 setCount
  // 初始值设为 0
  const [count, setCount] = useState(0);

  // 定义增加计数器的方法
  const increment = () =&gt; {
    // 使用函数式更新确保基于最新状态值
    setCount(prevCount =&gt; prevCount + 1);
  };

  // 定义减少计数器的方法
  const decrement = () =&gt; {
    setCount(prevCount =&gt; prevCount - 1);
  };

  // 定义重置计数器的方法
  const reset = () =&gt; {
    setCount(0);
  };

  return (
    &lt;div style={styles.container}&gt;
      &lt;h2&gt;当前计数: {count}&lt;/h2&gt;
      &lt;div style={styles.buttonGroup}&gt;
        {/* 绑定点击事件 */}
        &lt;button style={styles.button} onClick={increment}&gt;+&lt;/button&gt;
        &lt;button style={styles.button} onClick={decrement}&gt;-&lt;/button&gt;
        &lt;button style={styles.button} onClick={reset}&gt;重置&lt;/button&gt;
      &lt;/div&gt;
      
      {/* 条件渲染提示信息 */}
      {count &gt;= 10 &amp;&amp; &lt;p style={styles.tip}&gt;计数已经超过10啦！&lt;/p&gt;}
    &lt;/div&gt;
  );
}

// 样式对象
const styles = {
  container: {
    textAlign: 'center',
    padding: '2rem',
    backgroundColor: '#f0f0f0',
    borderRadius: '8px',
    maxWidth: '400px',
    margin: '2rem auto'
  },
  buttonGroup: {
    display: 'flex',
    justifyContent: 'center',
    gap: '1rem',
    marginTop: '1rem'
  },
  button: {
    padding: '0.5rem 1rem',
    fontSize: '1.2rem',
    cursor: 'pointer',
    backgroundColor: '#2196f3',
    color: 'white',
    border: 'none',
    borderRadius: '4px',
    transition: 'background-color 0.3s'
  },
  tip: {
    color: '#d32f2f',
    marginTop: '1rem',
    fontWeight: 'bold'
  }
};

export default Counter;
</code></pre>
    </blockquote>

     四、学习要点总结
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        基础概念
       </strong>
      </p>
      <ul>
       <li>
        useState 是用于给函数组件添加状态管理的 Hook
       </li>
       <li>
        遵循
        <code>
         const [state, setState] = useState(initialValue)
        </code>
        语法
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        使用规则
       </strong>
      </p>
      <ul>
       <li>
        只在组件的顶层调用 Hooks
       </li>
       <li>
        不要在循环、条件或嵌套函数中调用 Hooks
       </li>
       <li>
        遵循不可变原则（immutability）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        最佳实践
       </strong>
      </p>
      <ul>
       <li>
        对于复杂对象，使用扩展运算符进行合并更新
       </li>
       <li>
        异步操作时使用函数式更新确保获取最新状态
       </li>
       <li>
        拆分多个 useState 管理不同状态
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        性能优化
       </strong>
      </p>
      <ul>
       <li>
        对于昂贵的初始化，使用函数式初始化
       </li>
       <li>
        当更新依赖前值时，必须使用函数式更新
       </li>
       <li>
        使用 React.memo 防止不必要的重新渲染
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        常见错误
       </strong>
      </p>
      <ul>
       <li>
        直接修改状态对象
       </li>
      </ul>
      <p>
       JAVASCRIPT
      </p>
      <blockquote>
       <pre><code>// 错误示例
const [user, setUser] = useState({name: 'John'});
user.name = 'Mike'; // 直接修改原对象
setUser(user);      // 不会触发更新

// 正确做法
setUser({...user, name: 'Mike'});
</code></pre>
      </blockquote>
     </li>
    </ol>
    <ul>
     <li>
      在渲染函数中执行副作用操作
     </li>
     <li>
      忽视异步更新的特性
     </li>
    </ul>

import { useState } from 'react';

function useCart(initialItems = []) {
  const [cart, setCart] = useState(initialItems);

  const total = cart.reduce((sum, item) =&gt; sum + (item.price * item.quantity), 0);

  const addItem = (newItem) =&gt; {
    setCart([...cart, {
      ...newItem,
      id: Date.now(),
      quantity: 1
    }]);
  };

  const updateQuantity = (itemId, delta) =&gt; {
    setCart(currentCart =&gt;
      currentCart.map(item =&gt;
        item.id === itemId
          ? { ...item, quantity: Math.max(1, item.quantity + delta) }
          : item
      )
    );
  };

  const removeItem = (itemId) =&gt; {
    setCart(currentCart =&gt; currentCart.filter(item =&gt; item.id !== itemId));
  };

  return {
    cart,
    total,
    addItem,
    updateQuantity,
    removeItem
  };
}
