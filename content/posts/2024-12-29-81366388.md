    <p>

    <h3 id="一命令行工具的使用">
     一、命令行工具的使用
    </h3>
    <p>
     在开始菜单中打开
     <code>
      运行
     </code>
     工具，或使用快捷键
     <code>
      win+R
     </code>
     键打开，输入
     <code>
      cmd
     </code>
     即可。
     <br/>
     <img alt="命令行工具" src="https://i-blog.csdnimg.cn/blog_migrate/95e2ac1e3e176a65907d0753dd5a18a4.png" title=""/>
    </p>
    <p>
     输入所要运行的java文件的地址，即文件所在文件夹的位置，如我的java测试文件放在
     <code>
      F:\Software\eclipse\project\上课\src\dailystudy
     </code>
     ，首先输入
     <code>
      F:
     </code>
     ，表示磁盘，然后输入
     <code>
      cd 文件夹位置
     </code>
     ，cd表示进入文件夹内（相反
     <code>
      cd ..
     </code>
     表示返回上一层文件夹），如图所示：
     <br/>
     <img alt="进入磁盘" src="https://i-blog.csdnimg.cn/blog_migrate/cce5161351a061184691c958b3a68b72.png" title="">
      <br/>
      再输入java文件所在的位置：
      <br/>
      <img alt="输入文件夹位置" src="https://i-blog.csdnimg.cn/blog_migrate/e40f3e2f2283a2a2032a748b163bab2a.png" title="">
       <br/>
       到这里第一步就完成了，成功进入了该java文件的目录：
       <br/>
       <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cc59f96f98b2af1bd94e182493fea238.png" title="">
        <br/>
        <a href="https://blog.csdn.net/phmiscro/article/details/54882511">
         更多cmd操作
        </a>
       </img>
      </img>
     </img>
    </p>
    <h3 id="二运行java文件">
     二、运行java文件
    </h3>
    <p>
     一般java的源码里是含有package关键字的，也就是说java文件是在一个包中，那么我们输入
     <code>
      javac -d . 文件名.java
     </code>
     进行编译，注意
     <code>
      -d
     </code>
     前后和
     <code>
      .
     </code>
     的前后都有空格，如图
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c41cead0e4cf0e23609cd6d589613929.png" title="">
      <br/>
      我们发现在目标文件夹内出现了dailystudy的文件夹（Daily的文件的包我设置为dailystudy），这时候，再在命令行中输入
      <code>
       java dailystudy.Daily
      </code>
      ，即可运行该java文件。
      <br/>
      这里的
      <code>
       dailystudy.Daily
      </code>
      表示某个包中的某个类（
      <code>
       包名.类名
      </code>
      ），而类必须是程序的入口点，即主函数所在的类
      <br/>
      <img alt="运行java文件" src="https://i-blog.csdnimg.cn/blog_migrate/72e9c03954556faeb30325d31abd8590.png" title="">
       <br/>
       <strong>
        值得注意的是，java源文件中如果有使用了package语句，才能这样运行，这表示在所定义的package包中生成java文件的字节码文件。而如果没有使用package语句，则表示在该文件夹下生成字节码文件，则应该使用下面这种方式运行
       </strong>
      </img>
     </img>
    </p>
    <p>
     输入
     <code>
      javac 文件名.java
     </code>
     进行编译，输入
     <code>
      java 文件名
     </code>
     运行，则在该文件夹中会生成一个
     <code>
      .class
     </code>
     的字节码文件。
    </p>
