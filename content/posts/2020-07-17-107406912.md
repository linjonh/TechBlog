    <p>
     就小程序而言，其实支付宝小程序与微信小程序的相差并不大，如果你掌握了微信小程序，支付宝小程序基本上手到擒来。
     <br/>
     原生支付宝小程序越写越不对，这个编译器比微信小程序开发者工具还难用！！！！
    </p>
    <p>
     1、image标签会被支付宝小程序编译为background属性，如果不加上background-size或指定mode，会出现即使你设置了图片大小，图片依然不能显示完的效果。
    </p>
    <p>
     2、axml中标签必须先写
     <strong>
      &lt;
     </strong>
     才能自动补全标签。这对coding的我们实在有些头疼
    </p>
    <p>
     3、有可能是我没有设置对，修改过的小程序文件必须一个一个保存，不能
     <strong>
      ctrl+s
     </strong>
     保存所有修改的文件
    </p>
    <h3>
     <a id="_8">
     </a>
     小程序容纳大小
    </h3>
    <p>
     支付宝小程序与微信小程序主包大小都只能容纳2M的内容，但是二者同样都支持分包加载，微信小程序的主包内容不能超过2M，其余分包单个不能超过2M，加主包不能超过20M，但是支付宝小程序分包加主包大小却不能超过4M，开发者要多留意本地的内容大小。
     <br/>
     <a href="https://blog.csdn.net/YourNikee/article/details/103695854">
      微信小程序超过2M怎么办？分包加载的方法
     </a>
    </p>
    <h3>
     <a id="_11">
     </a>
     顶部导航
    </h3>
    <p>
     微信小程序可以通过
     <br/>
     <code>
      {"navigationStyle": "custom"}
     </code>
     去掉单页的头部导航，包括返回按钮，二者都不能隐藏胶囊按钮。
     <br/>
     支付宝小程序缺不能通过这个属性移除头部的返回按钮，且只是将内容进行了透明处理。
     <br/>
     支付宝小程序需要在app.json中添加以下属性控制。
    </p>
    <pre><code class="prism language-clike"><span class="token string">"window"</span><span class="token punctuation">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string">"defaultTitle"</span><span class="token punctuation">:</span> <span class="token string">""</span><span class="token punctuation">,</span><span class="token comment">//默认头部导航标题</span>
    <span class="token string">"transparentTitle"</span><span class="token punctuation">:</span> <span class="token string">"always"</span><span class="token punctuation">,</span> 
    <span class="token string">"titlePenetrate"</span><span class="token punctuation">:</span><span class="token string">"YES"</span>
  <span class="token punctuation">}</span>
</code></pre>
    <h3>
     <a id="api_24">
     </a>
     api调用
    </h3>
    <p>
     首先，支付宝小程序api调用方式是以my.api的形式，而微信则是wx.api。
    </p>
    <p>
     <strong>
      wx.getMenuButtonBoundingClientRect()
     </strong>
     获取胶囊位置：
     <br/>
     支付宝没有**wx.getMenuButtonBoundingClientRect()**获取胶囊位置，但是可以通过获取系统状态栏高度实现同样效果，
    </p>
    <p>
     <strong>
      getSystemInfo
     </strong>
     获取系统信息：
     <br/>
     而获取系统信息
     <strong>
      getSystemInfo
     </strong>
     也有一定区别
     <br/>
     支付宝：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d39ff22659400b2ee6c6cc8055cfc4c.png">
      <br/>
      微信：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c1de7044747f00ff11e5472ef03d434a.png">
       <br/>
       可以通过获取titleBarHeight去实现类似的效果。
      </img>
     </img>
    </p>
    <p>
     <strong>
      my.showToast
     </strong>
     短提示
     <br/>
     微信小程序可以指定icon类型，以下为具体参数
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8ab59fc14a4d91041a8063f6e3810105.png">
      <br/>
      支付宝小程序可以指定提示类型，区分代表不同的图标样式。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b6efe05f12a441317d163227f48c69eb.png"/>
     </img>
    </p>
    <pre><code class="prism language-clike"><span class="token comment">//微信</span>
	wx<span class="token punctuation">.</span><span class="token function">showToast</span><span class="token punctuation">(</span><span class="token punctuation">{<!-- --></span>
      title<span class="token punctuation">:</span> message<span class="token punctuation">,</span>

      type<span class="token punctuation">:</span> <span class="token string">'none'</span><span class="token punctuation">,</span>
      content<span class="token punctuation">:</span> message<span class="token punctuation">,</span>

    <h3>
     <a id="_80">
     </a>
     获取用户授权：
    </h3>
    <p>
     **
     <br/>
     微信小程序需要通过用户触发按钮手动授权，且也可不用授权就可在页面展示用户头像、昵称等信息。
     <br/>
     支付宝小程序：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/528e5c3d82d0f9ac0bc66420cd5e52bb.png">
      <br/>
      登录支付宝开放平台后，需要点击能力管理，去获取更多能力，在下拉列表中找到获取会员信息。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bf3227e2f95a67e94dcf551bdf9d35e9.png">
       <br/>
       注意：手机号与用户信息要单独获取，且手机号信息需要企业账号才有资格获取。
      </img>
     </img>
    </p>
    <h3>
     <a id="_90">
     </a>
     <strong>
      微信运动：
     </strong>
    </h3>
    <p>
     微信小程序：
     <br/>
     需要在公众号中打开该功能，通过调用api
     <code>
      wx.getWeRunData
     </code>
     去过去微信用户运动数据。
     <br/>
     支付宝：
     <br/>
     需要企业账号才能使用该功能，且需要用户手动去
     <strong>
      运动
     </strong>
     中打开，才可调用api去获取用户运动数据。
    </p>
