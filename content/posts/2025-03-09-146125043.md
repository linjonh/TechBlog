    <h3 id="%E4%B8%80.C%2B%2B%E7%9A%84%E7%AC%AC%E4%B8%80%E4%B8%AA%E7%A8%8B%E5%BA%8F" name="%E4%B8%80.C%2B%2B%E7%9A%84%E7%AC%AC%E4%B8%80%E4%B8%AA%E7%A8%8B%E5%BA%8F" style="text-align:center">
     一.C++的第一个程序
    </h3>
    <blockquote>
     <p>
      <strong>
       C++作为C语言的升级版，除了可以无缝兼容C语言的各种语法结构和各种框架外，在很大程度上对C语言的许多代码进行了优化升级，在保持原有代码逻辑结构相同的情况下，
       <span style="color:#ed7976">
        C++的简便性和应用性相比C语言不可同日而语
       </span>
       ，以下我要阐述的就是在学完C语言然后进入C++后需要掌握的一些
       <span style="color:#79c6cd">
        基础知识
       </span>
       ：
       <span style="color:#6eaad7">
        首先来看第一个C++程序
       </span>
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;//C++需要包含的头文件
using namespace std;//展开对应的命名空间
int main()
{
	cout &lt;&lt; "hello world" &lt;&lt; endl;//区别于C语言的输出方式
//cout&lt;&lt;输出内容，后面的endl是换行（endline），但和C语言里面的\n不用，endl是一个函数。

	return 0;
}</code></pre>
     <p>
      <strong>
       基础结构大概就是这样，下面就是对每一部分的具体介绍：
      </strong>
     </p>
    </blockquote>
    <p>
    </p>
    <h3 id="%E4%BA%8C.%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%EF%BC%88namespace%EF%BC%89" name="%E4%BA%8C.%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%EF%BC%88namespace%EF%BC%89" style="text-align:center">
     <strong>
      二.命名空间（namespace）
     </strong>
    </h3>
    <h4 id="1.%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E7%9A%84%E5%AE%9A%E4%B9%89%E4%B8%8E%E4%BD%BF%E7%94%A8%EF%BC%9A" name="1.%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E7%9A%84%E5%AE%9A%E4%B9%89%E4%B8%8E%E4%BD%BF%E7%94%A8%EF%BC%9A">
     <strong>
      1.命名空间的定义与使用：
     </strong>
    </h4>
    <blockquote>
     <p>
      <u>
       <strong>
        书面上的定义简而言之是：namespace的本质是
        <span style="color:#ed7976">
         定义出一个域
        </span>
        ，这个域跟
        <span style="color:#faa572">
         全局域
        </span>
        各自独立，
        <span style="color:#ed7976">
         不同的域可以定义同名变量
        </span>
       </strong>
      </u>
     </p>
     <h5 id="%EF%BC%881%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E9%87%8C%E5%8F%AF%E4%BB%A5%E5%AE%9A%E4%B9%89%E5%8F%98%E9%87%8F%EF%BC%8C%E5%87%BD%E6%95%B0%EF%BC%8C%E7%BB%93%E6%9E%84%E4%BD%93%E7%AD%89%E5%A4%9A%E7%A7%8D%E7%B1%BB%E5%9E%8B" name="%EF%BC%881%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E9%87%8C%E5%8F%AF%E4%BB%A5%E5%AE%9A%E4%B9%89%E5%8F%98%E9%87%8F%EF%BC%8C%E5%87%BD%E6%95%B0%EF%BC%8C%E7%BB%93%E6%9E%84%E4%BD%93%E7%AD%89%E5%A4%9A%E7%A7%8D%E7%B1%BB%E5%9E%8B">
      <strong>
       （1）命名空间里可以定义变量，函数，结构体等多种类型
      </strong>
     </h5>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
namespace space
{
//定义变量
	int rand = 10;
	char time = 20;
//定义函数
	int add(int a, int b)
	{
		return a + b;
	}
//定义结构体
	struct student
	{
		char name[20];
		char sex[6];
		int age;
	};
}
int main()
{
	return 0;
}</code></pre>
     <h5 id="%EF%BC%882%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E8%B0%83%E7%94%A8%EF%BC%88%EF%BC%9A%EF%BC%9A%EF%BC%89%E4%B8%8E%E5%B1%95%E5%BC%80" name="%EF%BC%882%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E8%B0%83%E7%94%A8%EF%BC%88%EF%BC%9A%EF%BC%9A%EF%BC%89%E4%B8%8E%E5%B1%95%E5%BC%80">
      （2）命名空间调用（：：）与展开
     </h5>
     <p>
      <span style="color:#ed7976">
       <strong>
        在调用的时候，只要用空间名：：函数名这样的格式来调用
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
int main()
{
	std::cout &lt;&lt; "hello world" &lt;&lt; std::endl;
 
	return 0;
}</code></pre>
     <p>
      <strong>
       也可以
       <span style="color:#ed7976">
        只展开一个命名空间中的某个变量或函数
       </span>
       ，这样的话访问这个变量或函数就更简便
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
namespace b
{
	int test(int x, int y)
	{
		return x + y;
	}
}
using b::test;
int main()
{
	int m = test(1, 5);

	cout &lt;&lt; m;
	return 0;
}</code></pre>
     <h5 id="%EF%BC%883%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E7%9A%84%E5%B5%8C%E5%A5%97" name="%EF%BC%883%EF%BC%89%E5%91%BD%E5%90%8D%E7%A9%BA%E9%97%B4%E7%9A%84%E5%B5%8C%E5%A5%97">
      <strong>
       （3）命名空间的嵌套
      </strong>
     </h5>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
namespace a
{
	int test(int x, int y)
	{
		return x + y;
	}
	namespace b
	{
		int test(int x, int y)
		{
			return x * y;
		}
	}

}
int main()
{
	int m = a::test(2, 3);
	int n = a::b::test(2, 3);
	cout &lt;&lt; m &lt;&lt; " " &lt;&lt; n;
	return 0;
}</code></pre>
     <h5 id="%EF%BC%884%EF%BC%89%E5%85%B7%E4%BD%93%E4%BD%BF%E7%94%A8%E5%9C%BA%E6%99%AF" name="%EF%BC%884%EF%BC%89%E5%85%B7%E4%BD%93%E4%BD%BF%E7%94%A8%E5%9C%BA%E6%99%AF">
      <strong>
       （4）具体使用场景
      </strong>
     </h5>
     <pre><code class="language-cpp">#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
int rand = 10;
int main()
{
 // 编译报错：error C2365: “rand”: 重定义；以前的定义是“函数” 
 printf("%d\n", rand);
 return 0;
}</code></pre>
     <p>
      <span style="color:#e6b223">
       <strong>
        这样写c语言会报错：报错的原因就是我们自己定的变量rand和C语言自带的库函数冲突了，这时候编译器就无法识别，这其实是C语言的不足之处。而c++中的命名空间就完美的解决这个问题
       </strong>
      </span>
     </p>
    </blockquote>
    <h3 id="" name="" style="text-align:center">
    </h3>
    <h3 id="%E4%B8%89.%E7%BC%BA%E7%9C%81%E5%8F%82%E6%95%B0" name="%E4%B8%89.%E7%BC%BA%E7%9C%81%E5%8F%82%E6%95%B0" style="text-align:center">
     三.缺省参数
    </h3>
    <h4 id="1.%E5%9F%BA%E6%9C%AC%E5%AE%9A%E4%B9%89%EF%BC%9A" name="1.%E5%9F%BA%E6%9C%AC%E5%AE%9A%E4%B9%89%EF%BC%9A">
     1.基本定义：
    </h4>
    <blockquote>
     <p>
      <span style="color:#ed7976">
       <strong>
        在函数定义时为参数指定的默认值，当调用函数时如果没有传递该参数，则使用默认值。这样
       </strong>
      </span>
      <span style="color:#d7d8d9">
       <strong>
        可以提高函数的灵活性和可重用性
       </strong>
      </span>
      <br/>
      <span style="color:#98c091">
       <strong>
        举例例如一个计算面积的函数，参数有长和宽，其中宽可以设为默认值1，这样调用时只传长就可以得到正方形的面积，这样的例子比较直观
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
int test(int x = 2, int y = 6)
{
	return x + y;
}
//如果调用这个函数但是不传递参数，那么返回结果就是8，同样如果掺入参数1和2，那么返回结果就是3</code></pre>
    </blockquote>
    <h4 id="2.%E5%87%A0%E4%B8%AA%E6%B3%A8%E6%84%8F%EF%BC%9A" name="2.%E5%87%A0%E4%B8%AA%E6%B3%A8%E6%84%8F%EF%BC%9A">
     2.几个注意：
    </h4>
    <blockquote>
     <p>
      <strong>
       （1）缺省参数必须从右往左给，比如上面的程序，我们
       <span style="color:#fbd4d0">
        只给x缺省参数但不给y
       </span>
       缺省参数这样其实是不合法的
      </strong>
     </p>
     <p>
      <strong>
       （2）函数实参一定是
       <span style="color:#fbd4d0">
        从左往右
       </span>
       给函数的
      </strong>
     </p>
     <p>
      <strong>
       （3）函数声明和定义分离时，
       <span style="color:#faa572">
        不能在声明和定义中同时
       </span>
       给缺省值
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
 
int test(int x = 1, int y = 2);
int main()
{
	int a = test();
	cout &lt;&lt; a;
	return 0;
}
int test(int x = 1, int y = 2)
{
	return x + y;
}
//比如这样同时个给缺省值的函数是不合法的，一般只在声明中给缺省值</code></pre>
    </blockquote>
    <p>
    </p>
    <h3 id="%E5%9B%9B.%E5%87%BD%E6%95%B0%E9%87%8D%E8%BD%BD" name="%E5%9B%9B.%E5%87%BD%E6%95%B0%E9%87%8D%E8%BD%BD" style="text-align:center">
     四.函数重载
    </h3>
    <h4 id="1.%E5%AE%9A%E4%B9%89%E4%B8%8E%E4%BD%BF%E7%94%A8%EF%BC%9A" name="1.%E5%AE%9A%E4%B9%89%E4%B8%8E%E4%BD%BF%E7%94%A8%EF%BC%9A">
     1.定义与使用：
    </h4>
    <blockquote>
     <p>
      <strong>
       即重名函数可以通过识别传入的参数类型来
       <span style="color:#faa572">
        自动选择相对应的函数
       </span>
       ，例如：
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
 
int test(int x , int y )
{
	return x + y;
}
double test(double x, double y)
{
	return x + y;
}
int main()
{
	double x = 1.5, y = 2.8;
	int a = test(1,2);
	double b = test(x, y);
	cout &lt;&lt; a &lt;&lt;" "&lt;&lt; b ;
	return 0;
}</code></pre>
    </blockquote>
    <h3 id="" name="" style="text-align:center">
    </h3>
    <h3 id="%E4%BA%94.%E5%BC%95%E7%94%A8" name="%E4%BA%94.%E5%BC%95%E7%94%A8" style="text-align:center">
     五.引用
    </h3>
    <h4 id="1.%E5%AE%9A%E4%B9%89%EF%BC%9A" name="1.%E5%AE%9A%E4%B9%89%EF%BC%9A">
     1.定义：
    </h4>
    <blockquote>
     <p>
      <strong>
       引用
       <span style="color:#faa572">
        不是新定义一个变量
       </span>
       ，而是给已存在变量取了一个
       <span style="color:#ffd7b9">
        别名
       </span>
       ，编译器
       <span style="color:#faa572">
        不会为引用变量开辟内存空间
       </span>
       ，它和它引用的变量
       <span style="color:#faa572">
        共用同一块内存空间，
       </span>
       <span style="color:#0d0016">
        但实际上在C++的底层逻辑里引用依旧是以指针的形式存在，但我们在实际使用中并不需要考虑那么多，就当引用是C++给我们带来的一种更加方便写代码的方法就行了，以下是引用使用的基本规则：
       </span>
      </strong>
     </p>
     <p style="text-align:center">
      <span style="color:#fe2c24">
       <strong>
        类型&amp; 引用别名=引用对象
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
int main()
{
	int a = 30;
	//一个变量可以取多个别名
	int&amp; b = a;//给a变量取别名
	int&amp; c = a;//c也是a的别名
	int&amp; d = b;//给b变量取别名，相当于d也是a的别名
	cout &lt;&lt; &amp;a &lt;&lt; endl &lt;&lt; &amp;b &lt;&lt; endl &lt;&lt; &amp;c &lt;&lt; endl &lt;&lt; &amp;d &lt;&lt; endl;
    //从地址中也可以看出来a,b,c，d占用同一块内存空间
	return 0;
 
}</code></pre>
     <p style="text-align:center">
      <img alt="" height="1079" src="https://i-blog.csdnimg.cn/direct/27152566d5fb48f58691223b60d5b3f8.png" width="1919"/>
     </p>
     <p>
      <strong>
       当然，引用相对指针是非常方便好理解的，我举个例子：
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
//传址调用,指针
void swap(int* a, int* b)
{
	int tmp = *a;
	*a = *b;
	*b = tmp;
}
//引用
void swap(int&amp; a, int&amp; b)//函数重载
{
	int tmp = a;
	a = b;
	b = tmp;
}
int main()
{
	int a = 10;
	int b = 20;
	swap(a, b);
	swap(a, b);
	return 0;
}</code></pre>
    </blockquote>
    <h4 id="2.%E5%BC%95%E7%94%A8%E7%9A%84%E7%89%B9%E6%80%A7%EF%BC%9A" name="2.%E5%BC%95%E7%94%A8%E7%9A%84%E7%89%B9%E6%80%A7%EF%BC%9A">
     2.引用的特性：
    </h4>
    <blockquote>
     <p>
      <strong>
       （1）引用必须在定义时初始化。比如int&amp; a=0;这句代码其实是不合法的
      </strong>
     </p>
     <p>
      <em>
       <span style="color:#d7d8d9">
        <strong>
         （在 C++ 中，引用必须在定义时初始化，这意味着当声明一个引用变量时，必须立即将其绑定到一个已存在的对象（变量、常量或表达式的结果），与普通变量不同，普通变量可以在后续赋值时初始化）
        </strong>
       </span>
      </em>
     </p>
     <p>
      <strong>
       （2）一个变量可以有多个引用
      </strong>
     </p>
     <p>
      <strong>
       （3）引用一旦引用一个实体，再不能引用其他实体
      </strong>
     </p>
     <p>
      <span style="color:#ed7976">
       <strong>
        举个栗子：
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
 
int main()
{
	int a = 10;
	int&amp; b = a;
	int c = 20;
	b = c;
	cout &lt;&lt; &amp;a &lt;&lt;" "&lt;&lt; &amp;b &lt;&lt;" "&lt;&lt; &amp;c &lt;&lt; endl;
	cout &lt;&lt; a &lt;&lt;" "&lt;&lt; b &lt;&lt;" "&lt;&lt; c &lt;&lt; endl;
	return 0;
}
//这里进行了b=c的操作，只是把c的值赋值给了b，但并没有改变b的引用实体，因此a和b地址是相同的</code></pre>
    </blockquote>
    <h4 id="3.%E5%BC%95%E7%94%A8%E7%9A%84%E4%BD%BF%E7%94%A8%EF%BC%88%E5%8C%BA%E5%88%AB%E4%BC%A0%E5%80%BC%E8%BF%94%E5%9B%9E%E5%92%8C%E4%BC%A0%E5%BC%95%E7%94%A8%E8%BF%94%E5%9B%9E%EF%BC%89%EF%BC%9A" name="3.%E5%BC%95%E7%94%A8%E7%9A%84%E4%BD%BF%E7%94%A8%EF%BC%88%E5%8C%BA%E5%88%AB%E4%BC%A0%E5%80%BC%E8%BF%94%E5%9B%9E%E5%92%8C%E4%BC%A0%E5%BC%95%E7%94%A8%E8%BF%94%E5%9B%9E%EF%BC%89%EF%BC%9A">
     3.引用的使用（区别传值返回和传引用返回）：
    </h4>
    <blockquote>
     <p>
      <span style="color:#ed7976">
       <strong>
        引用在实践中主要是于引用传参和引用做返回值中减少拷贝提高效率和改变引用对象时同时改变被 引用对象
       </strong>
      </span>
     </p>
     <p>
      <strong>
       引用传参介绍过了，然后我们看一下引用做返回值（即传引用返回）与传值返回区别
      </strong>
     </p>
     <pre><code class="language-cpp">int STTop(ST&amp; rs)//注意这里是传值返回（因为返回类型是int二百元引用类型的int&amp;）
{
 assert(rs.top &gt; 0);
 return rs.a[rs.top];
}
int main()
{
 ST st1;
 STInit(st1);
 STPush(st1, 1);//入栈
 STPush(st1, 2);
 STTop(st1)+=10;
 return 0;
}</code></pre>
     <p>
      <span style="color:#7b7f82">
       <strong>
        截取了一部分之前栈的操作，发现对于STTop(st1)+=10;这个操作无法达到目的，
       </strong>
      </span>
     </p>
     <p>
      <span style="color:#7b7f82">
       <strong>
        我们本意想的是让栈顶的数据加上10，但是实际上栈顶的数据并没有改变
       </strong>
      </span>
     </p>
     <p>
      <span style="color:#7b7f82">
       <strong>
        这是因为这串代码返回的是一个值，相当于我们把原来的数拷贝出了他的值然后返回了，
       </strong>
      </span>
     </p>
     <p>
      <span style="color:#7b7f82">
       <strong>
        因此就算把这份拷贝的值加上10也无法改变原来的栈顶数据
       </strong>
      </span>
     </p>
     <p>
      <span style="color:#0d0016">
       <u>
        <em>
         <strong>
          以上发生的情况就是由
         </strong>
        </em>
       </u>
      </span>
      <span style="color:#ed7976">
       <u>
        <em>
         <strong>
          传值引用
         </strong>
        </em>
       </u>
      </span>
      <span style="color:#0d0016">
       <u>
        <em>
         <strong>
          引发的，传值引用简而言之就是为了保护源数据不被修改的场景二存在，通过返回对栈顶元素的值的拷贝，从而使实际上的栈顶元素不受改变
         </strong>
        </em>
       </u>
      </span>
     </p>
     <p>
      <span style="color:#7b7f82">
       <strong>
        但如果代码这样改，就可以轻易改变栈顶元素的值：
       </strong>
      </span>
     </p>
     <p>
      <span style="color:#7b7f82">
       <strong>
        （即传引用返回）
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">int&amp; STTop(ST&amp; rs)
{
 assert(rs.top &gt; 0);
 return rs.a[rs.top];
}
int main()
{
 ST st1;
 STInit(st1);
 STPush(st1, 1);
 STPush(st1, 2);
 STTop(st1)+=10;
 return 0;
}</code></pre>
    </blockquote>
    <h4 id="4.const%E5%BC%95%E7%94%A8%EF%BC%9A" name="4.const%E5%BC%95%E7%94%A8%EF%BC%9A">
     4.const引用：
    </h4>
    <blockquote>
     <p>
      <strong>
       const引用简而言之就是对
       <span style="color:#ed7976">
        引用范围的缩小
       </span>
       ：即  const对象必须用const引用，普通对象也可以用const引用，但普通引用引用不了const对象
      </strong>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
int main()
{
	const int a = 10;//只可读
	//int&amp; b = a;
	//发生错误，b可读可写，权限变大
	//应加上const 
	const int&amp; b = a;
	int c = 20;
	const int&amp; d = c;//权限缩小，可以
	c++;
	//d++不可以;c可以改变，d不可以
	return 0;
}</code></pre>
     <p>
      <strong>
       这些就涉及到
       <span style="color:#faa572">
        权限的问题
       </span>
       ，普通对象是一个变量，可以被改变的，所以他的权限更低。而const对象不能被改变所以他的权限更高。普通引用引用const对象权限就放大了，所以编译器会报错，反过来const引用可以引用普通对象，因为权限是可以缩小的，总结来说：
       <span style="color:#ed7976">
        访问权限可以缩小，但是不能放大（访问权限只针对引用和指针）
       </span>
      </strong>
     </p>
    </blockquote>
    <p>
    </p>
    <h3 id="%E5%85%AD.inline%E5%86%85%E8%81%94" name="%E5%85%AD.inline%E5%86%85%E8%81%94" style="text-align:center">
     <span style="color:null">
      <strong>
       六.inline内联
      </strong>
     </span>
    </h3>
    <blockquote>
     <p>
      <span style="color:null">
       <strong>
        在 C++ 中，内联函数（Inline Functions）是一种优化机制，旨在通过在编译时将函数体直接替换到调用位置，减少函数调用的开销（如栈帧压入/弹出、参数传递等）
       </strong>
      </span>
     </p>
     <pre><code class="language-cpp">#include&lt;iostream&gt;
using namespace std;
inline int add(int x,int y)
{
	return x + y;
}
int main()
{
	cout &lt;&lt; add(1, 5) &lt;&lt; endl;
 
	return 0;
}

</code></pre>
     <p>
      <strong>
       对于短小的函数，inline操作会使函数
       <span style="color:#ed7976">
        在编译时展开
       </span>
       ，可以提高效率。但是如果这个函数过长就不适合使用内联了，过长的函数内联反而会降低时间，因此，内联函数
       <span style="color:#ed7976">
        也并不是万能的
       </span>
      </strong>
     </p>
     <p>
     </p>
     <p style="text-align:center">
      <u>
       <strong>
        还有一点非常值得注意，就是内联函数的定义和声明
        <span style="color:#faa572">
         必须在同一个文件
        </span>
       </strong>
      </u>
     </p>
    </blockquote>
