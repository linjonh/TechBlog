    </p>
    <h3 id="%E4%B8%80.%E6%B8%B8%E6%88%8F%E8%A7%84%E5%88%99">
     <strong>
      一.游戏规则
     </strong>
    </h3>
    <p>
     系统自动生成一个随机整数
     <strong>
      （1- 100）
     </strong>
     ，然后由用户输入一个猜测的数字，如果输入数字比生成数大，则提示“猜大了”；如果输入数字比生成数小，则提示“猜小了”；如果输入数字与生成数一样，则提示“猜对了”。
    </p>
    <p>
    </p>
    <h3 id="%E4%BA%8C.%E7%BC%96%E7%A8%8B%E6%80%9D%E8%B7%AF">
     <strong>
      二.编程思路
     </strong>
    </h3>
    <p>
     对于这样一个小游戏，首先来理清它的
     <strong>
      编程思路
     </strong>
     。
    </p>
    <p>
     <span style="color:#1c7331;">
      1.
      <strong>
       首先需要生成一个1-100的随机数
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#be191c;">
      2.
      <strong>
       从键盘循环捕捉一个数字，与生成的随机数比较
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#1c7331;">
      <strong>
       3.若输入的数字大于随机数，输出猜大了，若输入的数字小于随机数，输出，猜小了。
      </strong>
     </span>
    </p>
    <h3 id="%E4%B8%89.%E7%BC%96%E7%A8%8B%E8%BF%87%E7%A8%8B">
     <strong>
      三.编程过程
     </strong>
    </h3>
    <h4 id="1.%E7%AC%AC%E4%B8%80%E6%AD%A5">
     <strong>
      1.第一步
     </strong>
    </h4>
    <p>
     生成一个随机数，我们需要首先
     <span style="color:#be191c;">
      <strong>
       导入生成随机数的包
      </strong>
      ：
     </span>
    </p>
    <pre><code class="language-java">import java.util.Random;//生成随机数使用</code></pre>
    <p>
     再生成一个随机数：
    </p>
    <pre><code class="language-java"> Random random=new Random();//为随机数分配内存空间
 int guess=random.nextInt(100);//获取1到100随机数</code></pre>
    <p>
     <span style="color:#1c7331;">
      <strong>
       int guess=random.nextInt(100)
      </strong>
      <strong>
       括号里的100表示生成随机数的范围是1到100；
      </strong>
     </span>
    </p>
    <p>
    </p>
    <h4 id="2.%E7%AC%AC%E4%BA%8C%E6%AD%A5">
     <strong>
      2.第二步
     </strong>
    </h4>
    <p>
     生成随机数以后，用户需要从键盘循环输入一个猜测的数字，与随机数进行比较。这里的输入要进行循环输入，可以用
     <span style="color:#be191c;">
      <strong>
       while循环
      </strong>
     </span>
     来解决；
    </p>
    <pre><code class="language-java">import java.util.Random;//导入生成随机数需要的包
import java.util.Scanner;//导入键盘输入需要的包
public class Game {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        Random random=new Random();//为随机数分配内存空间
        int guess=random.nextInt(100);//获取1到100随机数
        while(true)//输入到猜对为止 使用死循环 条件填入true
        {
            System.out.println("请猜一个1到100的数字：");//屏幕提示输入
            int number=sc.nextInt();//键盘捕捉猜测的数字


        }

    }

}
</code></pre>
<p>
要从键盘输入数字，一样的，也需要导入相应的包：
</p>
<pre><code class="language-java">import java.util.Scanner;</code></pre>
<h4 id="3.%E7%AC%AC%E4%B8%89%E6%AD%A5">
<strong> 3.第三步
</strong>
</h4>
<p>
如果输入的数大于随机数，则输出猜大了，如果输入的数小于随机数，则输出猜小了。
</p>
<p>
这一步可以用
<span style="color:#1c7331;">
<strong>
if 语句
</strong>
</span>
来解决：
</p>
<pre><code class="language-java">if(number&gt;guess) //猜测数大于随机数
{
System.out.println("猜大了！");
}
else if(number&lt;guess)//猜测数小于随机数
{
System.out.println("猜小了！");
}
else if(number==guess)//猜测数等于随机数
{
System.out.println("猜对了！");
sc.close();//关闭键盘输入
return;//猜对退出程序
}
else {
System.out.println("输入有误！");//若输入值不在 1~100 提示输入错误
}</code></pre>
<p>
</p>
<p>
完成以上步骤，就完成了猜数字游戏的全部代码：
</p>
<pre><code class="language-java">import java.util.Random;
import java.util.Scanner;
public class Game {
public static void main(String[] args) {
Scanner sc=new Scanner(System.in);
Random random=new Random();
int guess=random.nextInt(100);//获取 1 到 100 随机数
while(true)//输入到猜对为止 使用死循环
{
System.out.println("请猜一个 1 到 100 的数字：");
int number=sc.nextInt();
if(number&gt;guess)
{
System.out.println("猜大了！");
}
else if(number&lt;guess)
{
System.out.println("猜小了！");
}
else if(number==guess)
{
System.out.println("猜对了！");
sc.close();//关闭键盘输入
return;//猜对退出程序
}
else {
System.out.println("输入有误！");
}

        }

    }

}
</code></pre>
