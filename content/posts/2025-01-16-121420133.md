    <h2>
     <a id="_17">
     </a>
     一、整数的概念、欧几里得除法
    </h2>
    <h3>
     <a id="11_18">
     </a>
     1.1整除的概念
    </h3>
    <p>
     定义：设a、b是任意两个整数，其中b≠0.如果存在一个整数q使得等式a = q·b成立，就称b整除a或者a被b整除，记作b|a,并把b叫做a的因数，把a叫做b的倍数。
    </p>
    <p>
     （1）整数传递性
     <br/>
     定理：设a、b≠0,c≠0是三个整数，若b|a,c|b,则c|a.
    </p>
    <p>
     例：因为7 | 42，42 | 84，所以7| 84
    </p>
    <p>
     (2)加法运算
     <br/>
     定理：设a、b、c≠0是三个整数，若c|a,c|b,则c|a±b
    </p>
    <p>
     (3)整数的线性组合
     <br/>
     定理：设a、b、c≠0是三个整数，若c|a,c|b,则对任意整数s、t,有c|(s·a+t·b)
    </p>
    <p>
     （4）
     <br/>
     定理：设a、b都是非零整数，若a|b,b|a，则a = ±b
    </p>
    <p>
     (5)
     <br/>
     定理:设n是一个正合数，p是n的一个大于1的最小正因数，则p一定是素数且p≤根号n
    </p>
    <h3>
     <a id="12Eratoshenes_38">
     </a>
     1.2Eratoshenes筛法（平凡除法）
    </h3>
    <p>
     定理:设n是一个正合数，如果对所有素数p≤根号n，都有p不能整除根号n，则n一定是素数
    </p>
    <h3>
     <a id="13_40">
     </a>
     1.3欧几里得除法——最小非负余数
    </h3>
    <p>
     定理:设a、b是两个整数，其中b&gt;0，则存在唯一的整数q，r使得a = q·b+r，0≤r&lt;b.
    </p>

    <h3>
     <a id="15_46">
     </a>
     1.5欧几里得除法——一般余数
    </h3>
    <p>
     定理：（欧几里得除法）设a、b是两个整数，其中b&gt;0.则对任意的整数c,存在唯一的整数q，r使得a = q·b+r，c≤r&lt;b+c.
    </p>
    <h2>
     <a id="_49">

     二、最大公因数与广义欧几里得除法
    </h2>
    <h3>
     <a id="21_50">
     </a>
     2.1最大公因数
    </h3>
    <p>
     定理:设b是任一正整数，则（0，b）= b.
    </p>
    <p>
     定理：设a,b,c是三个不全为零的整数.如果a = q·b+c,其中q是整数，则（a,b）=(b,c)
     <br/>
     证明：设d=(a,b),d’=(b,c),则d|a,a|b.
     <br/>
     ∴ d | a+（-q）· b = c
     <br/>
     ∴ d是b,c的公因数
     <br/>
     ∴d≤d’
     <br/>
     同理得到d’≤d
     <br/>
     所以d=d’
     <br/>
     定理成立
    </p>

    <h2>
     <a id="_80">
     </a>
     三、整除的近一步性质及最小公倍数
    </h2>
    <h3>
     <a id="31_81">
     </a>
     3.1整数的进一步性质
    </h3>
    <p>
     定理：设a,b,c是三个整数，且c≠0.如果c|ab,(a,c)=1,则c|b.
    </p>
    <p>
     定理：设p是素数，若p|ab,则p|a或p|b.
    </p>
    <h3>
     <a id="32_86">
     </a>
     3.2最小公倍数
    </h3>
    <p>
     定理：设a,b是两个互素正整数，则（1）若a | D, b | D,则a·b | D;(2)[a,b] = a·b
    </p>
    <h3>
     <a id="33_89">
     </a>
     3.3最小公倍数与最大公因数
    </h3>
    <p>
     定理：设a,b是两个正整数，则（1）若a | D, b | D,则[a,b] | D;
     <br/>
     (2)[a,b]=a·b / （a，b）
    </p>
