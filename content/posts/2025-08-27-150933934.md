### DevEco Studio代码实战

```
@Entry
@Component
struct Index {

  // build 里面要有唯一的容器根组件
  build() {
    Column() {
      Column() {
        Text('大壮')
        Text('中壮')
        Text('小壮')
      }
      Row() {
        Text('大壮')
        Text('中壮')
        Text('小壮')
      }
    }
  }
}
```
