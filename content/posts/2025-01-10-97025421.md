# 微信小程序 - 解决自定义添加遮罩层，底部页面仍滚动

##### 前言

底部的页面有滚动条，在自定义组件创建遮罩层时，底部页面仍然会滑动

##### 解决方法：

为了防止事件冒泡，在遮罩层容器内设置

```
// 小程序
catchtouchmove="true"
// mpvue 
@touchmove.stop.prevent="moveHandle"

methods: {
	moveHandle () {}
}

```
