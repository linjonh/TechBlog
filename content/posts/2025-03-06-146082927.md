    <h1 class="title-article" id="articleContentId">
     Mac同时安装jdk8和jdk17，默认选择jdk8
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p id="">
     在Mac上同时安装JDK 8和JDK 17，并设置默认版本为JDK 8，可以按照以下步骤操作：
    </p>
    <h4>
     一、下载并安装JDK 8和JDK 17
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        下载JDK 8
       </strong>
      </p>
      <ul>
       <li>
        访问
        <a href="https://www.oracle.com/java/technologies/javase-downloads.html" rel="nofollow" title="Oracle JDK下载页面">
         Oracle JDK下载页面
        </a>
        。
       </li>
       <li>
        在“Java SE Archive Downloads”部分，找到JDK 8的下载链接。
       </li>
       <li>
        选择适合您Mac芯片类型的版本（如Arm 64或x64），下载
        <code>
         .dmg
        </code>
        安装包。
       </li>
       <li>
        双击
        <code>
         .dmg
        </code>
        文件，按照安装向导的提示完成JDK 8的安装。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        下载并安装JDK 17
       </strong>
      </p>
      <ul>
       <li>
        仍然在Oracle JDK下载页面，找到JDK 17的下载链接。
       </li>
       <li>
        选择适合您Mac芯片类型的版本，下载
        <code>
         .dmg
        </code>
        安装包。
       </li>
       <li>
        双击
        <code>
         .dmg
        </code>
        文件，按照安装向导的提示完成JDK 17的安装。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     二、验证安装
    </h4>
    <ol>
     <li>
      <p id="">
       打开终端（可以通过Spotlight搜索“终端”来找到）。
      </p>
     </li>
     <li>
      <p id="">
       输入以下命令，查看已安装的JDK版本：
      </p>
      <pre><code class="hljs">/usr/libexec/java_home -V</code></pre>
      <p id="">
       您应该能够看到类似以下的输出，显示JDK 8和JDK 17的安装路径：
      </p>
      <pre><code class="hljs">Matching Java Virtual Machines (2):
	17.0.x (x86_64) "Oracle Corporation" - "JDK 17.0.x" /Library/Java/JavaVirtualMachines/jdk-17.0.x.jdk/Contents/Home
	1.8.0_x (x86_64) "Oracle Corporation" - "Java SE 8" /Library/Java/JavaVirtualMachines/jdk1.8.0_x.jdk/Contents/Home</code></pre>
     </li>
    </ol>
    <h4>
     三、配置环境变量
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        编辑配置文件
       </strong>
      </p>
      <ul>
       <li>
        <p id="">
         根据您的Shell类型（通常是
         <code>
          bash
         </code>
         或
         <code>
          zsh
         </code>
         ），编辑相应的配置文件。
        </p>
        <ul>
         <li>
          对于
          <code>
           bash
          </code>
          ，编辑
          <code>
           ~/.bash_profile
          </code>
          文件。
         </li>
         <li>
          对于
          <code>
           zsh
          </code>
          ，编辑
          <code>
           ~/.zshrc
          </code>
          文件。
         </li>
        </ul>
       </li>
       <li>
        <p id="">
         打开终端，输入以下命令之一来编辑配置文件：
        </p>
        <pre><code class="hljs">vim ~/.bash_profile   # 如果使用bash
vim ~/.zshrc          # 如果使用zsh</code></pre>
       </li>
       <li>
        <p id="">
         或者，您也可以使用其他文本编辑器，如
         <code>
          nano
         </code>
         或
         <code>
          open -e
         </code>
         命令。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        添加环境变量
       </strong>
      </p>
      <ul>
       <li>
        <p id="">
         在配置文件的末尾添加以下内容，以设置JDK 8为默认版本：
        </p>
        <pre><code class="hljs">export JAVA_8_HOME="/Library/Java/JavaVirtualMachines/jdk1.8.0_x.jdk/Contents/Home"
export JAVA_17_HOME="/Library/Java/JavaVirtualMachines/jdk-17.0.x.jdk/Contents/Home"
export JAVA_HOME=$JAVA_8_HOME
export PATH=$JAVA_HOME/bin:$PATH

# 为方便切换JDK版本，添加别名
alias jdk8="export JAVA_HOME=$JAVA_8_HOME"
alias jdk17="export JAVA_HOME=$JAVA_17_HOME"</code></pre>
       </li>
       <li>
        <p id="">
         请确保将
         <code>
          jdk1.8.0_x.jdk
         </code>
         和
         <code>
          jdk-17.0.x.jdk
         </code>
         替换为您实际安装的JDK版本的文件名。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        保存并关闭文件
       </strong>
      </p>
      <ul>
       <li>
        在
        <code>
         vim
        </code>
        编辑器中，按
        <code>
         Esc
        </code>
        键退出编辑模式，然后输入
        <code>
         :wq
        </code>
        保存并退出。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        使配置生效
       </strong>
      </p>
      <ul>
       <li>
        <p id="">
         在终端中输入以下命令，使配置的环境变量立即生效：
        </p>
        <pre><code class="hljs">source ~/.bash_profile   # 如果使用bash
source ~/.zshrc          # 如果使用zsh</code></pre>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     四、验证默认JDK版本
    </h4>
    <ol>
     <li>
      <p id="">
       在终端中输入以下命令，查看当前默认的JDK版本：
      </p>
      <pre><code class="hljs">java -version</code></pre>
      <p id="">
       您应该能够看到JDK 8的版本信息，表明JDK 8已设置为默认版本。
      </p>
     </li>
    </ol>
    <h4>
     五、切换JDK版本
    </h4>
    <ol>
     <li>
      <p id="">
       您可以随时使用之前配置的别名来切换JDK版本。例如，要切换到JDK 17，可以输入以下命令：
      </p>
      <pre><code class="hljs">jdk17</code></pre>
     </li>
     <li>
      <p id="">
       然后再次输入
       <code>
        java -version
       </code>
       ，验证JDK版本是否已切换为JDK 17。
      </p>
     </li>
    </ol>
    <p id="">
     通过以上步骤，您可以在Mac上同时安装JDK 8和JDK 17，并设置默认版本为JDK 8。需要时，您可以轻松切换JDK版本以满足不同的开发需求。
    </p>
