     ISCE参数
    </h2>
    <p>
     在遥感和地理信息领域，合成孔径雷达干涉测量（InSAR） 是研究地表形变、地震活动、冰川运动以及城市沉降的重要技术。Sentinel-1 卫星 提供了高质量的 TOPS SAR 数据，但其处理流程较为复杂。为了解决这一问题，ISCE的stackSentinel.py 提供了一个自动化的处理工具，使得研究人员可以高效地完成数据预处理、影像配准、干涉图计算、相位解缠等工作。
    </p>
    <p>
     本篇文章将深入解析 stackSentinel.py 的用法，帮助你掌握 Sentinel-1 InSAR 数据处理的核心步骤。
    </p>
    <pre><code class="prism language-bash"><span class="token number">1</span>. stackSentinel.py 是什么？
stackSentinel.py 是一个 Python 脚本，用于准备Sentinel-1 时序堆栈（stack） 处理的目录结构和配置文件。它支持：

影像预处理（SLC 影像、轨道数据、DEM）
影像配准（几何配准/NESD 配准）
干涉图计算（相干性、多视处理、滤波）
相位解缠（ICU/Snaphu 方法）
电离层校正
GPU 加速 <span class="token operator">&amp;</span> 并行计算
</code></pre>
    <pre><code class="prism language-bash"><span class="token number">2</span>. stackSentinel.py 关键参数解析
📌 输入数据
为了运行 stackSentinel.py，你需要准备： ✅ SLC 影像数据（-s 参数）
✅ 轨道数据（精轨）（-o 参数）
✅ 辅助文件（-a 参数）
✅ DEM（数字高程模型）（-d 参数）
</code></pre>
    <pre><code class="prism language-bash">工作流			作用
slc				处理原始 SLC 影像
correlation		计算影像相关性
interferogram	生成干涉图（默认）
offset			计算偏移量
</code></pre>
    <pre><code class="prism language-bash">研究区域
可以通过 <span class="token parameter variable">-b</span> 设置研究区域的边界
</code></pre>
    <pre><code class="prism language-bash">时间筛选
可以设定要处理的时间范围：
stackSentinel.py <span class="token parameter variable">--start_date</span> <span class="token number">2015</span>-01-23 <span class="token parameter variable">--stop_date</span> <span class="token number">2017</span>-02-26
如果只想处理特定日期，可以：
stackSentinel.py <span class="token parameter variable">-i</span> <span class="token string">"20141007,20141031"</span>
或排除某些日期：
stackSentinel.py <span class="token parameter variable">-x</span> <span class="token string">"20141007,20141031"</span>
</code></pre>
    <pre><code class="prism language-bash">影像配准
影像配准（coregistration）是 InSAR 处理的关键步骤，stackSentinel.py 提供 两种配准方式：
geometry（几何配准）
NESD（基于噪声等效信号延迟计算，更精确，默认）
</code></pre>
    <pre><code class="prism language-bash">干涉图计算
多视处理（Multi-looking）
方位向（Azimuth）多视因子：-z
距离向（Range）多视因子：-r
默认 3x9，可减少噪声，提高信号质量
stackSentinel.py <span class="token parameter variable">-z</span> <span class="token number">3</span> <span class="token parameter variable">-r</span> <span class="token number">9</span>

干涉图滤波（Filtering）
<span class="token parameter variable">-f</span> 控制滤波强度，默认 <span class="token number">0.5</span>


</code></pre>
    <pre><code class="prism language-bash">相位解缠
相位解缠是从干涉图获取高精度形变信息的关键步骤。stackSentinel.py 提供两种解缠方法：
方法	适用场景
icu	适用于高相干区域
snaphu（默认）	适用于低相干区域
</code></pre>
    <pre><code class="prism language-bash">电离层校正
电离层对 SAR 信号的影响不可忽视，stackSentinel.py 允许进行电离层误差估计：
</code></pre>
    <pre><code class="prism language-bash">GPU 加速 <span class="token operator">&amp;</span> 并行计算
如果你想让 stackSentinel.py 更快，可以启用 GPU 加速 和 多进程：
stackSentinel.py <span class="token parameter variable">-useGPU</span> <span class="token parameter variable">--num_proc</span> <span class="token number">4</span>

其中：

-useGPU：开启 GPU 加速
--num_proc：设置并行进程数（推荐 <span class="token number">4</span>~8）
--num_proc4topo：用于地形计算的并行进程

</code></pre>
    <pre><code class="prism language-bash">完整命令示例如下：
stackSentinel.py <span class="token punctuation">\</span>
  <span class="token parameter variable">-s</span> /data/SLCs <span class="token punctuation">\</span>
  <span class="token parameter variable">-o</span> /data/orbits <span class="token punctuation">\</span>
  <span class="token parameter variable">-a</span> /data/aux <span class="token punctuation">\</span>
  <span class="token parameter variable">-d</span> /data/DEM <span class="token punctuation">\</span>
  <span class="token parameter variable">-b</span> <span class="token string">"19 20 -99.5 -98.5"</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--start_date</span> <span class="token number">2015</span>-01-01 <span class="token parameter variable">--stop_date</span> <span class="token number">2020</span>-12-31 <span class="token punctuation">\</span>
  <span class="token parameter variable">-W</span> interferogram <span class="token punctuation">\</span>
  <span class="token parameter variable">-C</span> NESD <span class="token punctuation">\</span>
  <span class="token parameter variable">-z</span> <span class="token number">3</span> <span class="token parameter variable">-r</span> <span class="token number">9</span> <span class="token parameter variable">-f</span> <span class="token number">0.7</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">-u</span> snaphu <span class="token parameter variable">-rmFilter</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--num_proc</span> <span class="token number">4</span> <span class="token parameter variable">-useGPU</span>


这条命令的作用： ✅ 处理 SLC 影像
✅ 选取 <span class="token number">2015</span>-2020 年数据，限制处理范围
✅ 采用 NESD 配准
✅ 生成 干涉图 并应用滤波
✅ 进行 相位解缠
✅ 开启 GPU 并行计算 🚀
</code></pre>
