    <h2>
     <a id="_Clutter_0">
     </a>
     视觉 Clutter（视觉杂乱）
    </h2>
    <p>
     视觉 Clutter 的本质是 干扰用户目标的“视觉噪声”。好的设计应像导演一样，引导用户的视线和行为路径，而非用杂乱元素让其自行探索。
    </p>
    <h3>
     <a id="_3">
     </a>
     定义
    </h3>
    <p>
     <strong>
      视觉 Clutter
     </strong>
     指在设计、界面或视觉呈现中，因元素过多、布局混乱导致信息识别困难、注意力分散的现象，降低用户体验和信息传达效率。
    </p>
    <hr/>
    <h3>
     <a id="_8">
     </a>
     核心表现
    </h3>
    <h4>
     <a id="1__10">
     </a>
     1. 元素过载
    </h4>
    <ul>
     <li>
      过多图标、按钮、文字、图片堆积
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：网页广告弹窗密集，PPT 页面塞满文字
     </li>
    </ul>
    <h4>
     <a id="2__14">
     </a>
     2. 缺乏层次
    </h4>
    <ul>
     <li>
      颜色/字体/大小对比不足，关键信息被淹没
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：海报中所有文字用同一字号和颜色
     </li>
    </ul>
    <h4>
     <a id="3__18">
     </a>
     3. 无序排版
    </h4>
    <ul>
     <li>
      元素对齐混乱，间距不一致
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：App 界面控件随意摆放
     </li>
    </ul>
    <h4>
     <a id="4__22">
     </a>
     4. 冗余装饰
    </h4>
    <ul>
     <li>
      不必要的装饰性元素干扰内容
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：表格中使用多种颜色高亮
     </li>
    </ul>
    <hr/>

    <h3>
     <a id="_39">
     </a>
     典型案例
    </h3>
    <h4>
     <a id="1__41">
     </a>
     1. 网页设计
    </h4>
    <ul>
     <li>
      弹窗、横幅广告、侧边栏工具同时出现，用户找不到关闭按钮
     </li>
    </ul>
    <h4>
     <a id="2__44">
     </a>
     2. 数据可视化
    </h4>
    <ul>
     <li>
      图表中过多的数据标签、网格线和图例，掩盖核心趋势
     </li>
    </ul>
    <h4>
     <a id="3_PPT__47">
     </a>
     3. PPT 设计
    </h4>
    <ul>
     <li>
      每页文字密集无留白，听众难以抓住重点
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_52">
     </a>
     解决策略
    </h3>
    <h4>
     <a id="1__54">
     </a>
     1. 极简原则
    </h4>
    <ul>
     <li>
      <strong>
       删除
      </strong>
      ：移除非必要元素（如装饰性图标）
     </li>
     <li>
      <strong>
       分组
      </strong>
      ：用卡片/边框归类相关内容
     </li>
     <li>
      <strong>
       留白
      </strong>
      ：增加元素间距（建议留白占比 ≥30%）
     </li>
    </ul>
    <h4>
     <a id="2__59">
     </a>
     2. 强化视觉层次
    </h4>
    <ul>
     <li>
      <strong>
       对比
      </strong>
      ：使用颜色（红 &gt; 灰）、字号（标题 24px &gt; 正文 14px）、粗细（加粗关键数据）
     </li>
     <li>
      <strong>
       对齐
      </strong>
      ：采用网格系统（如 12 列网格布局）
     </li>
    </ul>
    <h4>
     <a id="3__63">
     </a>
     3. 统一设计规范
    </h4>
    <ul>
     <li>
      <strong>
       颜色
      </strong>
      ：主色 + 辅色 ≤5 种（参考 Material Design 色板）
     </li>
     <li>
      <strong>
       字体
      </strong>
      ：不超过 3 种字体系列（如 Roboto + Open Sans）
     </li>
    </ul>
    <h4>
     <a id="4__67">
     </a>
     4. 用户测试优化
    </h4>
    <ul>
     <li>
      <strong>
       眼动追踪
      </strong>
      ：分析用户视线热图
     </li>
     <li>
      <strong>
       A/B 测试
      </strong>
      ：对比不同设计方案点击率
     </li>
    </ul>
