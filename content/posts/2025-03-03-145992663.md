    <h4 id="数据清洗报告">
     生成的数据如下
    </h4>
    <pre><code class="language-sql">WITH SalesData AS (
    SELECT 
        dept_id AS 门店,
        goods_id AS 商品,
        SUM(num * price) AS 累计销售额,
        ROW_NUMBER() OVER (PARTITION BY dept_id ORDER BY SUM(num * price) DESC) AS 排名
    FROM 
        t_order_detail
    WHERE 
        order_time &gt;= '2025-02-01' AND order_time &lt; '2025-03-01'
    GROUP BY 
        dept_id, goods_id
)
SELECT 
    门店,
    商品,
    累计销售额,
    排名
FROM 
    SalesData
WHERE 
    累计销售额 &gt;= (SELECT PERCENTILE_CONT(0.8) WITHIN GROUP (ORDER BY 累计销售额) FROM SalesData AS sd WHERE sd.门店 = SalesData.门店)
ORDER BY 
    门店 DESC, 商品 DESC;</code></pre>
    <h4 id="查询语句的解释">
     查询语句的解释
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        CTE (Common Table Expression)
        <code>
         SalesData
        </code>
       </strong>
       :
      </p>
      <ul>
       <li>
        <strong>
         门店
        </strong>
        : 从
        <code>
         t_order_detail
        </code>
        表中提取
        <code>
         dept_id
        </code>
        字段，表示门店。
       </li>
       <li>
        <strong>
         商品
        </strong>
        : 从
        <code>
         t_order_detail
        </code>
        表中提取
        <code>
         goods_id
        </code>
        字段，表示商品。
       </li>
       <li>
        <strong>
         累计销售额
        </strong>
        : 计算每个门店、每个商品在2025年2月份的累计销售额，即
        <code>
         SUM(num * price)
        </code>
        。
       </li>
       <li>
        <strong>
         排名
        </strong>
        : 使用
        <code>
         ROW_NUMBER()
        </code>
        窗口函数，按门店分组，按累计销售额降序排列，生成每个门店内商品的排名。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        主查询
       </strong>
       :
      </p>
      <ul>
       <li>
        <strong>
         筛选条件
        </strong>
        : 使用子查询计算每个门店的累计销售额的80%分位数，并筛选出累计销售额大于或等于该分位数的商品。
       </li>
       <li>
        <strong>
         排序
        </strong>
        : 按门店ID和商品ID倒序排列结果。
       </li>
      </ul>
     </li>
    </ol>
