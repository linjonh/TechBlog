    <h3>
     <a id="Selenium__0">
     </a>
     <code>
      Selenium
     </code>
     简介
    </h3>
    <p>
     <code>
      Selenium
     </code>
     是支持 Web 浏览器自动化的一系列工具和库的集合。
     <code>
      Selenium
     </code>
     是开源项目。目前最新版本为
     <code>
      Selenium 3
     </code>
     。
    </p>
    <p>
     <code>
      Selenium
     </code>
     支持多平台，如：Linux、Windows和mac OS。
     <br/>
     <code>
      Selenium
     </code>
     支持多种编程语言，如：Java、Python、C#、Ruby、JavaScript、Kotlin
     <br/>
     <code>
      Selenium
     </code>
     支持多种浏览器，如：IE、Firefox、Google Chrome、Edge、Safari、Opera等
    </p>
    <p>
     <code>
      Selenium
     </code>
     项目主要包括以下三个部分：
    </p>
    <ul>
     <li>
      <code>
       Selenium WebDriver
      </code>
      是
      <code>
       Selenium
      </code>
      项目的核心，我们一般提到
      <code>
       Selenium
      </code>
      指的就是
      <code>
       Selenium WebDriver
      </code>
      。它是一个遵循
      <code>
       W3C WebDriver
      </code>
      规范支持所有（主要）浏览器技术的通用编程接口。使用浏览器供应商提供的浏览器自动化 API 来控制浏览器和运行测试。就像真正的用户正在操作浏览器一样。由于 WebDriver 不要求使用应用程序代码编译其 API, 因此它本质上不具有侵入性。
     </li>
     <li>
      <code>
       Selenium IDE
      </code>
      是一个浏览器插件，可用于录制、播放自动化脚本，实现定时任务。
     </li>
     <li>
      <code>
       Selenium Grid
      </code>
      可以在多台机器上或不同平台上执行测试用例。
     </li>
    </ul>
    <h3>
     <a id="Selenium_WebDriver_12">
     </a>
     <code>
      Selenium WebDriver
     </code>
     环境部署
    </h3>
    <p>
     <code>
      Selenium WebDriver
     </code>
     的环境部署与其他项目稍微不同。需要安装三部分内容：
    </p>
    <ul>
     <li>
      <code>
       Selenium webdriver API
      </code>
      ：
      <code>
       Selenium
      </code>
      针对不同语言提供了API，Python语言的API即
      <code>
       Selenium
      </code>
      包，使用pip可快速安装。
      <code>
       pip install selenium
      </code>
     </li>
     <li>
      Web浏览器
     </li>
     <li>
      Web浏览器驱动二进制文件（WebDriver）：下载WebDriver二进制文件并文件所在路径加入系统 环境变量
      <code>
       PATH
      </code>
      中。
      <ul>
       <li>
        常用驱动下载地址如下：
        <ul>
         <li>
          Chromedriver：
          <code>
           http://npm.taobao.org/mirrors/chromedriver/
          </code>
         </li>
         <li>
          Geckodriver：
          <code>
           https://github.com/mozilla/geckodriver/releases
          </code>
         </li>
        </ul>
       </li>
       <li>
        <strong>
         将存放驱动的路径添加到环境变量
         <code>
          PATH
         </code>
         中。
        </strong>
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <a id="Selenium_WebDriver_Python_API_22">
     </a>
     <code>
      Selenium WebDriver Python API
     </code>
     环境测试
    </h3>
    <p>
     确保已经正确安装、配置了
     <code>
      Selenium
     </code>
     包、浏览器、浏览器驱动后，执行如下代码，如果正常打开对应浏览器说明环境正常。
    </p>
