    <h1 class="title-article" id="articleContentId">
     Git使用（二）--如何配置 GitHub 远程仓库及本地 Git 环境
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p id="ubc429c9e">
     在日常的开发过程中，使用版本控制工具 Git 是一个非常重要的技能，特别是对于管理和协作开发。通过 GitHub，我们可以轻松地进行代码版本管理和共享。这篇博客将带您一步步学习如何配置 Git 环境并将本地仓库与 GitHub 远程仓库连接起来。
    </p>
    <h4 id="b6siE">
     一、安装 Git Bash 并创建 GitHub 仓库
    </h4>
    <p id="u8f92eae5">
     未安装Git Bash可参考
     <a href="https://blog.csdn.net/m0_68987050/article/details/146211024?sharetype=blogdetail&amp;sharerId=146211024&amp;sharerefer=PC&amp;sharesource=m0_68987050&amp;spm=1011.2480.3001.8118" title="Git使用（一）--如何在 Windows 上安装 Git：详细步骤指南-CSDN博客">
      Git使用（一）--如何在 Windows 上安装 Git：详细步骤指南-CSDN博客
     </a>
    </p>
    <p id="ue66a964f">
     首先，确保已经安装 Git Bash，打开 Git Bash 后，我们开始配置 GitHub 仓库。
    </p>
    <ul>
     <li id="ucb6f3a86">
      打开 GitHub（https://github.com），登录后点击右上角的
      <strong>
       “New Repository”
      </strong>
      创建一个新的仓库。建议为您的仓库命名，如
      <code>
       personalspace
      </code>
      ，并确保选择公开或私有仓库。
     </li>
     <li id="uc94a6b1f">
      <p class="img-center">
       <img alt="" height="594" id="uae018f55" src="https://i-blog.csdnimg.cn/img_convert/8975d2be87801867e432f4ccfce1c022.png" width="1211"/>
      </p>
      <p class="img-center">
       <img alt="" height="783" id="u67567034" src="https://i-blog.csdnimg.cn/img_convert/6a9915a150c1369b44702969ba1dbad7.png" width="935"/>
      </p>
     </li>
     <li id="u7268c36e">
      创建完仓库后，接下来的任务是在本地创建一个 Git 仓库并将其与 GitHub 远程仓库关联。
     </li>
    </ul>
    <h4 id="WobnP">
     二、在本地创建版本库
    </h4>
    <p id="u0848d914">
     接下来，我们需要在本地创建一个新的 Git 仓库。
    </p>
    <ol>
     <li id="u8bdde437">
      在计算机上，新建一个文件夹，名称与 GitHub 上的仓库名称一致，例如
      <code>
       personalspace
      </code>
      。
     </li>
    </ol>
    <p id="uc70be5f7">
    </p>
    <p class="img-center">
     <img alt="" height="32" id="u670edbb1" src="https://i-blog.csdnimg.cn/img_convert/ae6d6ddd7a97e1f6f817bf38a6749a74.png" width="140"/>
    </p>
    <ol>
     <li id="ua268dfb3">
      右键点击该文件夹并选择
      <strong>
       “Git Bash Here”
      </strong>
      选项，打开 Git Bash。
      <p class="img-center">
       <img alt="" height="334" id="uefd639a7" src="https://i-blog.csdnimg.cn/img_convert/8fe7c727dc41a8e645529b0a172f0506.png" width="483"/>
      </p>
     </li>
     <li id="ufef02e66">
      在 Git Bash 中输入以下命令初始化本地 Git 仓库：
     </li>
    </ol>
    <pre id="PMdb0"><code>git init</code></pre>
    <p id="uf7941d31">
    </p>
    <p class="img-center">
     <img alt="" height="56" id="uc094fe64" src="https://i-blog.csdnimg.cn/img_convert/98f5bbd0094e244315c9c725f92bad56.png" width="428"/>
    </p>
    <ol>
     <li id="u3e094723">
      成功初始化后，您会发现文件夹中多了一个
      <code>
       .git
      </code>
      文件夹，这是 Git 用来管理版本控制的文件夹。
      <p class="img-center">
       <img alt="" height="175" id="u0e11a9d8" src="https://i-blog.csdnimg.cn/img_convert/4946db8331ffc998075efc1bea548b5b.png" width="512"/>
      </p>
     </li>
     <li id="u6b339578">
      将您的项目或源代码复制到此文件夹中（后面进行上传）。
     </li>
    </ol>
    <p id="ufc7c925e">
    </p>
    <p class="img-center">
     <img alt="" height="103" id="ud8417eb9" src="https://i-blog.csdnimg.cn/img_convert/f6b1714b5e4ec29aedbd570de57ff16d.png" width="492"/>
    </p>
    <h4 id="xflZ5">
     三、添加文件并提交Git ,对项目中的文件进行管理和版本控制。
    </h4>
    <p id="u16817162">
     首先，需要添加文件到暂存区。
    </p>
    <p id="uc1d08074">
     1.使用以下命令查看文件状态：
    </p>
    <pre id="Rf7TT"><code>git status</code></pre>
    <p id="u2124ad6c">
    </p>
    <p class="img-center">
     <img alt="" height="171" id="u859968be" src="https://i-blog.csdnimg.cn/img_convert/cb77154960f88578f4c4ab8f751a201a.png" width="544"/>
    </p>
    <p id="u2124ad6c">
    </p>
    <p class="img-center">
     <img alt="" height="240" id="u63e59404" src="https://i-blog.csdnimg.cn/img_convert/bc1e14f992ae65633a0aa13bab6bb993.png" width="510"/>
    </p>
    <p id="u02bb4191">
     红色表示未添加到 Git 仓库，绿色表示已经添加。
    </p>
    <p id="uc804e96d">
     2.使用以下命令将文件添加到暂存区：
    </p>
    <pre id="p6h9G"><code>git add .</code></pre>
    <p id="u97cf78f6">
    </p>
    <p class="img-center">
     <img alt="" height="52" id="u715e804b" src="https://i-blog.csdnimg.cn/img_convert/e437c4c9cdb0a978d88b7c7eacb7f2e6.png" width="423"/>
    </p>
    <p id="ucca31ee8">
     这样就将当前目录下的所有文件添加到了 Git 的暂存区。
    </p>
    <p id="u961ead6b">
     3.使用以下命令提交文件到本地仓库：
    </p>
    <pre id="fjpn5"><code>git commit -m "first commit"</code></pre>
    <p id="ub805f6aa">
    </p>
    <p class="img-center">
     <img alt="" height="95" id="ubdc407db" src="https://i-blog.csdnimg.cn/img_convert/cdc5400ef2058403c0b90f856d3ce428.png" width="559"/>
    </p>
    <h4 id="f5DcQ">
     四、配置 SSH 密钥
    </h4>
    <p id="u1cf28cf3">
     为了将本地仓库与 GitHub 远程仓库安全地连接，我们需要配置 SSH 密钥。
    </p>
    <ol>
     <li id="u4dead829">
      在 Git Bash 中输入以下命令设置您的用户名和邮箱：(这步没什么用)
     </li>
    </ol>
    <pre id="q0Ez7"><code>git config --global user.name "注册名"//自定义用户名
git config --global user.email "注册邮箱"//用户邮箱</code></pre>
    <ol>
     <li id="uc50b0b81">
      创建SSH Key
     </li>
    </ol>
    <p id="u68117d8e">
     在用户主目录下，查询是否存在“.ssh”文件。
    </p>
    <p id="u0076bed4">
    </p>
    <p class="img-center">
     <img alt="" height="609" id="FHZeH" src="https://i-blog.csdnimg.cn/img_convert/cf0f45140aefc6581a37291947dafc5d.png" width="902"/>
    </p>
    <p id="u94cbbcd0">
     再看文件下有没有id_rsa和id_rsa.pub两个文件。SSH Key的秘钥对：
     <strong>
      id_rsa是私钥，不能泄露；id_rsa.pub是公钥，可以公开。
     </strong>
     <br/>
     我们要使用的是其中的公钥id_rsa.pub
    </p>
    <p id="u9f9c9764">
     如果没有的话：
     <strong>
      生成SSH
     </strong>
    </p>
    <p id="u956b04a7">
     <strong>
      git
     </strong>
     bash
     <strong>
      中输入指令：
     </strong>
    </p>
    <pre id="Cs1Ms"><code>ssh-keygen -t rsa -C "自己的邮箱"//邮箱是GitHub注册时使用的邮箱</code></pre>
    <p id="u0675f464">
     SSH文件存放在C:/User/用户/.ssh下，id_rsa为私钥，id_rsa.pub为公钥。
    </p>
    <p id="u538f7619">
     红色三行出现时，直接回车即可。
    </p>
    <p id="ue734cecc">
    </p>
    <p class="img-center">
     <img alt="" height="249" id="Hm94K" src="https://i-blog.csdnimg.cn/img_convert/edf026934dec1fb703eba6eb237015e8.png" width="783"/>
    </p>
    <p id="ud3834830">
     执行完后，会在用户的主目录下生成
     <code>
      .ssh
     </code>
     文件夹，其中包含
     <code>
      id_rsa
     </code>
     （私钥）和
     <code>
      id_rsa.pub
     </code>
     （公钥）。
    </p>
    <p id="u3df6d028">
     3.打开 GitHub，进入
     <strong>
      Settings
     </strong>
     -&gt;
     <strong>
      SSH and GPG Keys
     </strong>
     ，点击
     <strong>
      New SSH Key
     </strong>
     ，将
     <code>
      id_rsa.pub
     </code>
     文件中的内容粘贴到文本框中。
    </p>
    <p id="u523de043">
    </p>
    <p class="img-center">
     <img alt="" height="1032" id="gC0dg" src="https://i-blog.csdnimg.cn/img_convert/bdbdc3e40c07d772e61f11d8e9a6f667.png" width="1244"/>
    </p>
    <p id="u9dbfc767">
    </p>
    <p class="img-center">
     <img alt="" height="1032" id="lkEFd" src="https://i-blog.csdnimg.cn/img_convert/c2dad863c1ed98ef0a2625c9b28b30dd.png" width="1244"/>
    </p>
    <p id="uecebb11a">
     title自定义，key复制id_rsa.pub的全部内容到文本框中
    </p>
    <p id="u3899e166">
    </p>
    <p class="img-center">
     <img alt="" height="538" id="hQqG2" src="https://i-blog.csdnimg.cn/img_convert/87ef30e0520e8246c807fb56930e4e6f.png" width="912"/>
    </p>
    <h4 id="yhjFj">
     五、 将本地仓库与 GitHub 远程仓库关联
    </h4>
    <p id="u359fbc7d">
     1.验证是否成功，在git bash里输入下面的命令
    </p>
    <pre id="hVo91"><code>ssh -T git@github.com</code></pre>
    <p id="u5bb1f127">
     初次设置需要输入yes，出现第二个红框内容表示成功。
    </p>
    <p id="uf1cb3f5f">
    </p>
    <p class="img-center">
     <img alt="" height="370" id="e3N4H" src="https://i-blog.csdnimg.cn/img_convert/e3f396d80a5937be8126f9d0b3292cb4.png" width="581"/>
    </p>
    <p id="ueb0cdb9d">
     2. 回到 Git Bash，输入以下命令将本地仓库与 GitHub 上的远程仓库关联：
    </p>
    <p id="ubcdd41e2">
     在GitHub代码仓库中找到SSH
    </p>
    <p id="u9407b060">
    </p>
    <p class="img-center">
     <img alt="" height="542" id="E2h7H" src="https://i-blog.csdnimg.cn/img_convert/7c71b9c97a9ef7603e147231412de82f.png" width="1167"/>
    </p>
    <p id="u340c26e6">
     git bash执行指令：（换成自己的SSH）
    </p>
    <pre id="UtUjI"><code>git remote set-url origin git@github.com:LiYuxuanDevSpace/personalspace.git</code></pre>
    <p id="u9c03355c">
    </p>
    <p class="img-center">
     <img alt="" height="54" id="QDvLx" src="https://i-blog.csdnimg.cn/img_convert/6a3fb9e2f7ac3ce2ef666eec0a407c33.png" width="552"/>
    </p>
    <p id="u92220d42">
     3. 然后使用以下命令将本地仓库推送到远程仓库：
    </p>
    <p id="u2ba7593a">
     由于新建的远程仓库是空的，所以要加上-u这个参数
    </p>
    <pre id="QkT7i"><code>git push -u origin master</code></pre>
    <p id="u7175721d">
     之后仓库不是空的，就不用加上-u
    </p>
    <pre id="u8hHB"><code>git push origin master</code></pre>
    <p id="ue4e05529">
    </p>
    <p class="img-center">
     <img alt="" height="284" id="u893cbd27" src="https://i-blog.csdnimg.cn/img_convert/76667087616590226c7d6023bdd2946d.png" width="550"/>
    </p>
    <h4 id="c47dc5bc">
     六、总结
    </h4>
    <p id="u67353b25">
     1.初始化本地仓库
    </p>
    <pre id="YXvAk"><code>git init</code></pre>
    <p id="u53df6d22">
     2.使用以下命令将文件添加到暂存区：
    </p>
    <pre id="tz62E"><code>git add .</code></pre>
    <p id="u70c3dae7">
     3.提交到本地仓库：
    </p>
    <pre id="OmJTL"><code>git commit -m "英文注释"</code></pre>
    <p id="uf7dc213e">
     4.与GitHub建立连接
    </p>
    <pre id="UVP7S"><code>git remote set-url origin git@github.com:LiYuxuanDevSpace/personalspace.git</code></pre>
    <p id="u6859f886">
     5. 将本地仓库推送到远程仓库：
    </p>
    <pre id="EIabT"><code>git push -u origin master</code></pre>
    <p id="u16c563b6">
