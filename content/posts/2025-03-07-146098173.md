     创建一个数据库，在数据库种创建一张叫heros的表，在表中插入几个四大名著的角色：
    </p>
    <h2 id="1.%E5%88%9B%E5%BB%BA%E6%95%B0%E6%8D%AE%E5%BA%93%E5%92%8C%E8%A1%A8" name="1.%E5%88%9B%E5%BB%BA%E6%95%B0%E6%8D%AE%E5%BA%93%E5%92%8C%E8%A1%A8">
     1.创建数据库和表
    </h2>
    <pre><code class="language-sql">#创建表
CREATE DATABASE db_test;

#查看创建的数据库
show databases;</code></pre>
    <p>
     <img alt="" height="385" src="https://i-blog.csdnimg.cn/direct/c704761b9cc04a328bc2cc017fbf58cf.png" width="486"/>
    </p>
    <pre><code class="language-sql">#使用db_test数据库
USE db_test;

#查看当前所在的数据库
select database();</code></pre>
    <p>
     <img alt="" height="268" src="https://i-blog.csdnimg.cn/direct/43e76b647bc94b03a9beb92ec256889f.png" width="348"/>
    </p>
    <pre><code class="language-sql">#创建heros表
create table heros(
    id int,
    name varchar(50),
    gender char(5),
    novel varchar(50)
);

#显示数据库中的表
show tables;

#查看表的详细信息
desc heros;</code></pre>
    <p>
     <img alt="" height="545" src="https://i-blog.csdnimg.cn/direct/5b503d729aa04557972203fd481f9255.png" width="564"/>

     2.插入数据
    </h2>
