    <h3>
     <a id="_0">
     </a>
     问题描述
    </h3>
    <p>
     3月6日，在使用Windsuf 时，遇到以下网络连接错误：
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>unavailable<span class="token punctuation">]</span> unavailable: dial tcp <span class="token number">35.223</span>.238.178:443: connectex: 
A connection attempt failed because the connected party did not properly respond after a period of time, 
or established connection failed because connected <span class="token function">host</span> has failed to respond.
</code></pre>
    <p>
     该错误表明程序尝试通过
     <code>
      443
     </code>
     端口连接
     <code>
      35.223.238.178
     </code>
     时失败，常见于以下场景：
    </p>
    <ul>
     <li>
      网络防火墙/代理拦截了连接
     </li>
     <li>
      目标服务器暂时不可用
     </li>
     <li>
      本地网络配置异常
     </li>
    </ul>
    <h3>
     <a id="_13">
     </a>
     逐一排查
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        防火墙限制
       </strong>
       <br/>
       本地防火墙或安全软件可能阻止了出站连接。
       <br/>
       （然而，我并没有对其限制）
      </p>
     </li>
     <li>
      <p>
       <strong>
        DNS解析问题
       </strong>
       <br/>
       域名解析失败可能导致无法连接到正确的服务器。
       <br/>
       （ping 了一下，感觉没啥问题）
       <br/>
       看了眼目标服务器，全绿，感觉没啥问题
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d815417a78524438b6dbf8f400fb44c9.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        代理配置缺失
       </strong>
       <br/>
       许多特殊网络环境需要通过代理服务器访问外部资源。如果应用程序未正确识别系统代理设置，会导致连接失败。
       <br/>
       （我确实开了代理，应该是这个原因）
      </p>
     </li>
    </ol>
    <h3>
     <a id="_29">
     </a>
     解决方案
    </h3>
    <p>
     通过在
     <code>
      settings.json
     </code>
     配置文件中添加代理检测配置，使应用程序自动识别系统代理设置：
    </p>
    <pre><code class="prism language-json"><span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"windsurf.detectProxy"</span><span class="token operator">:</span> <span class="token boolean">true</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <strong>
      保存并重启应用
     </strong>
     ：修改后保存文件，并重启相关应用使配置生效。
    </p>

     </li>
