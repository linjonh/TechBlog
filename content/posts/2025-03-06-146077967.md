# 修改git fetch后引用没更新

通常更新远程仓库：

```
git fetch origin  xx
```

但是有时候会出现分支没有更新到本地的引用，执行下面的这个

```
git fetch origin  xx:refs/remotes/origin/xx
```

当你执行 `git fetch origin xx:refs/remotes/origin/xx ` 时，Git 会从远程仓库 origin 中获取 xx分支的最新提交，并将其更新到本地的远程引用路径 refs/remotes/origin/xx。
