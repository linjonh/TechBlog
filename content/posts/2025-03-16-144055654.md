    <h4>
     <a id="cannot_import_name_Config_from_mmcv_7">
     </a>
     cannot import name ‘Config’ from ‘mmcv’
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6e322b8bae984e418b9e38073ac1e010.png">
      <br/>
      根据 mmcv 的官方文档，在 2.0.0 及更高版本中，许多模块已迁移到新的 mmengine 库中。因此，Config 等功能需要从 mmengine 而非 mmcv 中导入。
      <br/>
      修改 tools/train.py 脚本中的相关导入，将 mmcv 的部分导入更换为 mmengine：
     </img>
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 将以下代码</span>
<span class="token comment"># from mmcv import Config, DictAction</span>

<span class="token comment"># 替换为</span>
<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>config <span class="token keyword">import</span> Config
<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>utils <span class="token keyword">import</span> DictAction

</code></pre>
    <h4>
     <a id="ImportError_cannot_import_name_print_log_from_mmcv_20">
     </a>
     ImportError: cannot import name ‘print_log’ from ‘mmcv’
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/edcca7ecdf1243a68b99eeeaba2594a2.png">
      <br/>
      找到以下导入语句：
     </img>
    </p>
    <pre><code class="prism language-python"><span class="token keyword">from</span> mmcv<span class="token punctuation">.</span>runner <span class="token keyword">import</span> init_dist<span class="token punctuation">,</span> set_random_seed
修改为：

<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>runner <span class="token keyword">import</span> init_dist
<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>runner <span class="token keyword">import</span> set_random_seed
<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>logging <span class="token keyword">import</span> print_log
</code></pre>
    <h4>
     <a id="ImportError_cannot_import_name_init_dist_from_mmenginerunner_33">
     </a>
     ImportError: cannot import name ‘init_dist’ from ‘mmengine.runner’
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b8957458af5b42d0a5735ac5843b73e7.png">
      <br/>
      在 mmengine 的最新版本中，init_dist 函数通常已迁移到 mmengine.distributed。
      <br/>
      将导入语句：
     </img>
    </p>
    <pre><code class="prism language-python"><span class="token comment">#from mmengine.runner import init_dist</span>
<span class="token comment">#替换为：</span>
<span class="token keyword">from</span> mmengine<span class="token punctuation">.</span>dist <span class="token keyword">import</span> init_dist
</code></pre>
