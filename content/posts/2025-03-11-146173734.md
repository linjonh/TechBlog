      【题目描述】

     <p>
      求 S=1!+2!+3!+....+n!

      【输入】
     </h4>
     <p>
      输入正整数n。

      输出s。

     </h4>

      3
     </p>

      【输出样例】
     </h4>
     <p>
      9
     </p>
     <h4>
      【提示】
     </h4>
     <p>
      【数据范围】
     </p>
     <p>
      对于所有数据：1≤n≤10。
     </p>

    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

int main()
{
	int n; cin &gt;&gt; n;

	long long  ans = 1, sum = 1;
	for (int i = 2; i &lt;= n; i++)
	{
		ans *= i;
		sum += ans;
	}
	cout &lt;&lt; sum;

	return 0;
}</code></pre>
