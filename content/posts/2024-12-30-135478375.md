    <h3>
     <a id="1_15">
     </a>
     1.问题描述
    </h3>
    <p>
     在 idea 进行开发时,有时候注释的符号和代码直接会空白很多,如下图所示,对于我这样的强迫症开发者来说,是很难受的,那么有没有什么办法既能达到注释的效果,又能不出现空格呢?可参考以下方法进行设置,根据需要进行合理配置就可以达到想要的效果
    </p>
    <h3>
     <a id="2_19">
     </a>
     2.本身的注释
    </h3>
    <p>
     <img alt="image-20240109114316504" src="https://i-blog.csdnimg.cn/blog_migrate/63386f694941911f4264bc4dbd2902b0.png"/>
    </p>
    <h3>
     <a id="3_23">
     </a>
     3.本身的配置
    </h3>
    <p>
     打开 settings。
     <br/>
     依次点击 Editor ——&gt; Code Style ——&gt; Java ——&gt; Code Generation。
     <br/>
     找到 Comment Code。
    </p>
    <ul>
     <li>
      Line comment at first column（行注释在第 1 列）（取消勾选）
     </li>
     <li>
      Add a space at comment start（在注释开始添加一个空格）（建议勾选）
     </li>
     <li>
      Block comment at first column（注释快在第 1 列）（取消勾选）
     </li>
    </ul>
    <p>
     <img alt="image-20240109114412053" src="https://i-blog.csdnimg.cn/blog_migrate/a2aaf478342df41747d2d4972e4517dc.png"/>
    </p>
    <h3>
     <a id="4_35">
     </a>
     4.配置说明
    </h3>
    <ul>
     <li>
      Line comment at first column
     </li>
     <li>
      Add a space at comment start
     </li>
     <li>
      Block comment at first column
     </li>
    </ul>
    <p>
     以下是对 Comment Code 中的三个选项的中文解释：
    </p>
    <ol>
     <li>
      <p>
       Line comment at first column（行注释在第 1 列）：取消勾选此选项后，行注释将不会在每行代码的第一列开始，而是在代码之后的位置开始。
      </p>
     </li>
     <li>
      <p>
       Add a space at comment start（在注释开始添加一个空格）：勾选此选项后，在行注释开始的位置会添加一个空格。
      </p>
     </li>
     <li>
      <p>
       Block comment at first column（注释块在第 1 列）：取消勾选此选项后，注释块将不会在每行代码的第一列开始，而是在代码之后的位置开始。
      </p>
     </li>
    </ol>
    <p>
     这些选项可以根据个人的编码风格和偏好进行设置，以使代码更易读和易于维护。
    </p>
