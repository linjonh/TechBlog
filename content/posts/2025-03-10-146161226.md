    <h2>
     1.块级标签
    </h2>
    <p>
     <span style="color:#fe2c24">
      <strong>
       块级元素会独占一行，通常用于构建页面的结构。常见的块级元素包括：
      </strong>
     </span>
    </p>
    <ul>
     <li>
      <p>
       <code>
        &lt;div&gt;
       </code>
       ：通用的块级容器。没有任何语意。可以创建网页的不同部分，导航栏侧边栏等。
      </p>
     </li>
    </ul>
    <pre><code class="language-html">&lt;body&gt;
    &lt;div class="nav"&gt;
        &lt;a href="#"&gt;链接 1&lt;/a&gt;
        &lt;a href="#"&gt;链接 2&lt;/a&gt;
        &lt;a href="#"&gt;链接 3&lt;/a&gt;
    &lt;/div&gt;
&lt;/body&gt;</code></pre>
    <p>
     若是想创建一个&lt;div&gt;初始带标签的话：可以直接：
     <span style="color:#fe2c24">
      <strong>
       #nav
      </strong>
     </span>
     或者
     <span style="color:#fe2c24">
      <strong>
       div.nav
      </strong>
     </span>
     或者
     <span style="color:#fe2c24">
      <strong>
       div#nav
      </strong>
     </span>
    </p>
    <p>
     创建第二个div标签，第一个是导航栏，第二个就是主要内容了
     <span style="color:#fe2c24">
      <strong>
       .content
      </strong>
     </span>
    </p>
    <pre><code class="language-html">&lt;div class="content"&gt;
    &lt;h1&gt;文章标题&lt;/h1&gt;
    &lt;p&gt;文章内容&lt;/p&gt;
    &lt;p&gt;文章内容&lt;/p&gt;
    &lt;p&gt;文章内容&lt;/p&gt;
&lt;/div&gt;</code></pre>
    <ul>
     <li>
      <p>
       <code>
        &lt;p&gt;
       </code>
       ：段落。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;h1&gt;
       </code>
       到
       <code>
        &lt;h6&gt;
       </code>
       ：标题。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;ul&gt;
       </code>
       ：无序列表。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;ol&gt;
       </code>
       ：有序列表。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;li&gt;
       </code>
       ：列表项。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;table&gt;
       </code>
       ：表格。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;form&gt;
       </code>
       ：表单。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;header&gt;
       </code>
       ：页眉。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;footer&gt;
       </code>
       ：页脚。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;section&gt;
       </code>
       ：页面的一部分。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;article&gt;
       </code>
       ：文章内容。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;aside&gt;
       </code>
       ：侧边栏。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;nav&gt;
       </code>
       ：导航栏。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;main&gt;
       </code>
       ：页面主要内容。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;blockquote&gt;
       </code>
       ：引用块。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;hr&gt;
       </code>
       ：水平分割线。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;pre&gt;
       </code>
       ：预格式化文本。
      </p>
     </li>
     <li>
      <h2>
       1. 行内元素
      </h2>
     </li>
     <li>
      <p>
       <strong>
        <span style="color:#fe2c24">
         行内元素不会独占一行，它们会与其他行内元素在同一行显示。常见的行内元素包括：
        </span>
       </strong>
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;span&gt;
       </code>
       ：用于对文本的一部分进行样式化或操作。包装起来，以便使用gss或者js行为。
      </p>
     </li>
    </ul>
    <pre><code class="language-html">&lt;span&gt;这是第一个span标签&lt;/span&gt;
&lt;span&gt;这是第二个span标签&lt;/span&gt;
&lt;span&gt;这是第三个span标签&lt;/span&gt;
&lt;span&gt;这是第四个span标签&lt;/span&gt;
&lt;hr&gt;
&lt;span&gt;链接点击这里&lt;a href="#"&gt;链接&lt;/a&gt;&lt;/span&gt;
</code></pre>
    <ul>
     <li>
      <p>
       <code>
        &lt;a&gt;
       </code>
       ：超链接。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;strong&gt;
       </code>
       ：加粗文本，表示重要性。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;em&gt;
       </code>
       ：斜体文本，表示强调。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;img&gt;
       </code>
       ：插入图片。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;input&gt;
       </code>
       ：输入框。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;label&gt;
       </code>
       ：表单标签。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;button&gt;
       </code>
       ：按钮。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;br&gt;
       </code>
       ：换行符。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;i&gt;
       </code>
       ：斜体文本。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;b&gt;
       </code>
       ：加粗文本。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;small&gt;
       </code>
       ：小号文本。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;sup&gt;
       </code>
       ：上标文本。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;sub&gt;
       </code>
       ：下标文本。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;code&gt;
       </code>
       ：代码片段。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;time&gt;
       </code>
       ：表示时间或日期。
      </p>
     </li>
     <li>
      <h2>
       3. 表单元素（Form Elements）
      </h2>
      <p>
       表单用于收集用户输入。常见的表单元素包括：
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;form&gt;
       </code>
       ：表单容器，用于
       <span style="color:#fe2c24">
        <strong>
         包裹所有表单元素
        </strong>
       </span>
       。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;input&gt;
       </code>
       ：输入框，类型包括：
      </p>
      <ul>
       <li>
        <p>
         <code>
          text
         </code>
         ：文本输入。
        </p>
       </li>
       <li>
        <p>
         <code>
          password
         </code>
         ：密码输入。
        </p>
       </li>
       <li>
        <p>
         <code>
          email
         </code>
         ：电子邮件输入。
        </p>
       </li>
       <li>
        <p>
         <code>
          number
         </code>
         ：数字输入。
        </p>
       </li>
       <li>
        <p>
         <code>
          date
         </code>
         ：日期输入。
        </p>
       </li>
       <li>
        <p>
         <code>
          checkbox
         </code>
         ：复选框。
        </p>
       </li>
       <li>
        <p>
         <code>
          radio
         </code>
         ：单选按钮。
        </p>
       </li>
       <li>
        <p>
         <code>
          submit
         </code>
         ：提交按钮。
        </p>
       </li>
       <li>
        <p>
         <code>
          reset
         </code>
         ：重置按钮。
        </p>
       </li>
       <li>
        <p>
         <code>
          file
         </code>
         ：文件上传。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <p>
     最常用的就是type属性：
    </p>
    <pre><code class="language-html">&lt;from&gt;
    &lt;label for="username"&gt;用户名&lt;/label&gt;      //这是在文本框前面显示要填写的信息
    &lt;input type="text" id="username" placeholder="请输入用户名"&gt;&lt;br&gt;&lt;br&gt;
    &lt;label for="pwd"&gt;密码： &lt;/label&gt;     //在文本框前面显示输入的是密码
    &lt;input type="password" id="pwd" placeholder="请输入内容"&gt;

//也可以用单选框：rasio

    &lt;label for=""&gt;性别&lt;/label&gt;
    &lt;input type="rasio" name="gender"&gt; 男     //name属性设置的单选
    &lt;input type="rasio" name="gender"&gt; 女
    &lt;input type="rasio" name="gender"&gt; 其他&lt;br&gt;&lt;br&gt;

//可以设置多选属性：chackbook

    &lt;label&gt; 爱好：&lt;/label&gt;
    &lt;input type="checkbox" name="hobby"&gt;唱歌
    &lt;input type="checkbox" name="hobby"&gt;跳舞
    &lt;input type="checkbox" name="hobby"&gt;rap
    &lt;input type="checkbox" name="hobby"&gt;打篮球

//显示提交

    &lt;input type="submit" value="上传"&gt;
&lt;/from&gt;</code></pre>
    <p>
     以上提交的位置：from标签有很多属性，自带的一个
     <strong>
      action属性，
     </strong>
     表示提交标签的时候向何处发送数据所以action的属性值也就是一个
     <strong>
      URL
     </strong>
     但是，需要后端给我们提供
     <strong>
      API
     </strong>
     。所以可以直接用
     <strong>
      #来表示不跳转
     </strong>
     ，不想提交的化也可以设置成上传。
    </p>
    <p>
    </p>

     总结
    </h4>
    <p>

    <h4>
     示例代码
    </h4>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="zh-CN"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;title&gt;表单示例&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;注册表单&lt;/h1&gt;
    &lt;form action="/submit" method="post"&gt;
        &lt;div&gt;
            &lt;label for="username"&gt;用户名:&lt;/label&gt;
            &lt;input type="text" id="username" name="username" required&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;label for="password"&gt;密码:&lt;/label&gt;
            &lt;input type="password" id="password" name="password" required&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;label for="email"&gt;电子邮件:&lt;/label&gt;
            &lt;input type="email" id="email" name="email" required&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;label for="gender"&gt;性别:&lt;/label&gt;
            &lt;input type="radio" id="male" name="gender" value="male"&gt;
            &lt;label for="male"&gt;男&lt;/label&gt;
            &lt;input type="radio" id="female" name="gender" value="female"&gt;
            &lt;label for="female"&gt;女&lt;/label&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;label for="country"&gt;国家:&lt;/label&gt;
            &lt;select id="country" name="country"&gt;
                &lt;option value="china"&gt;中国&lt;/option&gt;
                &lt;option value="usa"&gt;美国&lt;/option&gt;
                &lt;option value="uk"&gt;英国&lt;/option&gt;
            &lt;/select&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;label for="bio"&gt;个人简介:&lt;/label&gt;
            &lt;textarea id="bio" name="bio" rows="4" cols="50"&gt;&lt;/textarea&gt;
        &lt;/div&gt;
        &lt;div&gt;
            &lt;button type="submit"&gt;提交&lt;/button&gt;
            &lt;button type="reset"&gt;重置&lt;/button&gt;
        &lt;/div&gt;
    &lt;/form&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <ul>
     <li>
      <p>
       <code>
        &lt;textarea&gt;
       </code>
       ：多行文本输入框。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;select&gt;
       </code>
       ：下拉选择框。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;option&gt;
       </code>
       ：下拉选择框的选项。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;label&gt;
       </code>
       ：表单标签，用于描述输入字段。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;button&gt;
       </code>
       ：按钮，可以用于提交表单或触发其他操作。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;fieldset&gt;
       </code>
       ：用于将表单中的相关元素分组。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;legend&gt;
       </code>
       ：用于描述
       <code>
        &lt;fieldset&gt;
       </code>
       的内容。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;datalist&gt;
       </code>
       ：为输入框提供预定义的选项列表。
      </p>
     </li>
     <li>
      <p>
       <code>
        &lt;output&gt;
       </code>
       ：用于显示计算结果或脚本输出。
      </p>
     </li>
     <li>
      <p>
       <strong>
        行内元素
       </strong>
       ：
       <span style="color:#fe2c24">
        不会独占一行，通常用于包裹文本或其他行内元素。
       </span>
      </p>
     </li>
     <li>
      <p>
       <strong>
        块级元素
       </strong>
       ：
       <span style="color:#fe2c24">
        独占一行，通常用于构建页面结构
       </span>
       。
      </p>
     </li>
     <li>
      <p>
       <strong>
        表单元素
       </strong>
       ：
       <span style="color:#fe2c24">
        用于收集用户输入，常见的表单元素包括
        <code>
         &lt;input&gt;
        </code>
        、
        <code>
         &lt;textarea&gt;
        </code>
        、
        <code>
         &lt;select&gt;
        </code>
        等
       </span>
       。
      </p>
     </li>
    </ul>
