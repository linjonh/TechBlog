    <h2>
     <a id="_Ubuntu__0">
     </a>
     如何修改 Ubuntu 软件源（镜像源）
    </h2>
    <h3>
     <a id="_2">
     </a>
     <strong>
      前言
     </strong>
    </h3>
    <p>
     在使用 Ubuntu 时，默认的软件源可能速度较慢，影响软件安装和系统更新的效率。我们可以通过修改
     <code>
      sources.list
     </code>
     文件或使用图形界面更换更快的镜像源，提升软件包管理的速度。
    </p>
    <p>
     本指南适用于大部分 Ubuntu 版本，并提供了适配多个版本的方法，方便以后扩展。
    </p>
    <hr/>
    <h3>
     <a id="1__9">
     </a>
     <strong>
      1. 通过图形界面更换软件源
     </strong>
    </h3>
    <p>
     如果你更喜欢使用 GUI（图形界面）来修改软件源，可以按照以下步骤操作：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开“软件和更新”
       </strong>
      </p>
      <ul>
       <li>
        点击
        <strong>
         “活动”
        </strong>
        ，在搜索框中输入
        <strong>
         “软件和更新”
        </strong>
        （或
        <code>
         Software &amp; Updates
        </code>
        ），然后打开它。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        更改下载服务器
       </strong>
      </p>
      <ul>
       <li>
        在
        <strong>
         Ubuntu 软件
        </strong>
        选项卡中，找到
        <strong>
         下载自
        </strong>
        ，点击下拉菜单，选择
        <strong>
         其他站点（Other…）
        </strong>
        。
       </li>
       <li>
        点击
        <strong>
         选择最佳服务器（Select Best Server）
        </strong>
        ，系统会自动测试速度最快的镜像，选中后点击
        <strong>
         选择服务器（Choose Server）
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        应用更改
       </strong>
      </p>
      <ul>
       <li>
        关闭窗口，系统可能会提示你
        <strong>
         重新加载（Reload）
        </strong>
        软件包信息，点击确认。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="2__24">
     </a>
     <strong>
      2. 备份原始软件源
     </strong>
    </h3>
    <p>
     在修改软件源之前，建议先备份原始的
     <code>
      sources.list
     </code>
     文件，以便在出现问题时可以恢复：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">cp</span> /etc/apt/sources.list /etc/apt/sources.list.bak
</code></pre>
    <p>
     这样，如果新的软件源不可用，可以恢复原始状态：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">cp</span> /etc/apt/sources.list.bak /etc/apt/sources.list
</code></pre>
    <hr/>
    <h3>
     <a id="3__Ubuntu__39">
     </a>
     <strong>
      3. 确定 Ubuntu 版本代号
     </strong>
    </h3>
    <p>
     Ubuntu 使用版本代号来标识其软件源，例如：
    </p>
    <table>
     <thead>
      <tr>
       <th>
        版本
       </th>
       <th>
        代号
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        Ubuntu 24.04
       </td>
       <td>
        noble
       </td>
      </tr>
      <tr>
       <td>
        Ubuntu 22.04
       </td>
       <td>
        jammy
       </td>
      </tr>
      <tr>
       <td>
        Ubuntu 20.04
       </td>
       <td>
        focal
       </td>
      </tr>
      <tr>
       <td>
        Ubuntu 18.04
       </td>
       <td>
        bionic
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     可以使用以下命令查看系统的代号：
    </p>
    <pre><code class="prism language-bash">lsb_release <span class="token parameter variable">-sc</span>
</code></pre>
    <hr/>
    <h3>
     <a id="4__sourceslist__57">
     </a>
     <strong>
      4. 修改
      <code>
       sources.list
      </code>
      文件
     </strong>
    </h3>
    <p>
     使用以下命令打开
     <code>
      sources.list
     </code>
     进行编辑：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">nano</span> /etc/apt/sources.list
</code></pre>
    <h4>
     <a id="_Ubuntu__64">
     </a>
     <strong>
      国内推荐镜像源（适用于不同 Ubuntu 版本）
     </strong>
    </h4>
    <p>
     <strong>
      阿里云镜像源：
     </strong>
    </p>
    <pre><code class="prism language-plaintext">deb http://mirrors.aliyun.com/ubuntu/ &lt;版本代号&gt; main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ &lt;版本代号&gt;-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ &lt;版本代号&gt;-security main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ &lt;版本代号&gt;-backports main restricted universe multiverse
</code></pre>
    <p>
     <strong>
      清华大学镜像源：
     </strong>
    </p>
    <pre><code class="prism language-plaintext">deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ &lt;版本代号&gt; main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ &lt;版本代号&gt;-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ &lt;版本代号&gt;-security main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ &lt;版本代号&gt;-backports main restricted universe multiverse
</code></pre>
    <p>
     <strong>
      中科大镜像源：
     </strong>
    </p>
    <pre><code class="prism language-plaintext">deb https://mirrors.ustc.edu.cn/ubuntu/ &lt;版本代号&gt; main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ &lt;版本代号&gt;-updates main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ &lt;版本代号&gt;-security main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ &lt;版本代号&gt;-backports main restricted universe multiverse
</code></pre>
    <p>
     <strong>
      示例：
     </strong>
     <br/>
     如果你使用的是
     <strong>
      Ubuntu 24.04（noble）
     </strong>
     ，那么
     <code>
      sources.list
     </code>
     应该修改为：
    </p>
    <pre><code class="prism language-plaintext">deb http://mirrors.aliyun.com/ubuntu/ noble main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ noble-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ noble-security main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ noble-backports main restricted universe multiverse
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d0434a1955fd43a5b98d6d472e112d85.png"/>
    </p>
    <hr/>
    <h3>
     <a id="5__102">
     </a>
     <strong>
      5. 保存并更新软件源
     </strong>
    </h3>
    <p>
     完成修改后，按
     <code>
      Ctrl + X
     </code>
     退出 nano 编辑器，按
     <code>
      Y
     </code>
     保存更改。
    </p>
    <p>
     然后运行以下命令更新软件源信息：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update <span class="token operator">&amp;&amp;</span> <span class="token function">sudo</span> <span class="token function">apt</span> upgrade <span class="token parameter variable">-y</span>
</code></pre>
    <hr/>
    <h3>
     <a id="6__113">
     </a>
     <strong>
      6. 自动选择最快的软件源（可选）
     </strong>
    </h3>
    <p>
     如果不确定哪个镜像源最快，可以使用
     <code>
      netselect-apt
     </code>
     或
     <code>
      mirrors
     </code>
     自动选择最佳源：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> netselect-apt <span class="token parameter variable">-y</span>
<span class="token function">sudo</span> netselect-apt
</code></pre>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> apt-mirror
apt-mirror
</code></pre>
    <p>
     系统会自动检测并生成
     <code>
      sources.list
     </code>
     ，然后运行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
</code></pre>
    <hr/>
    <h3>
     <a id="7__Ubuntu__134">
     </a>
     <strong>
      7. 恢复默认 Ubuntu 源（如果遇到问题）
     </strong>
    </h3>
    <p>
     如果新的镜像源不可用或有问题，可以恢复默认源。
    </p>
    <p>
     <strong>
      官方 Ubuntu 源：
     </strong>
    </p>
    <pre><code class="prism language-plaintext">deb http://archive.ubuntu.com/ubuntu &lt;版本代号&gt; main restricted universe multiverse
deb http://archive.ubuntu.com/ubuntu &lt;版本代号&gt;-updates main restricted universe multiverse
deb http://security.ubuntu.com/ubuntu &lt;版本代号&gt;-security main restricted universe multiverse
deb http://archive.ubuntu.com/ubuntu &lt;版本代号&gt;-backports main restricted universe multiverse
</code></pre>
    <p>
     然后执行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
</code></pre>
    <hr/>
    <h3>
     <a id="_152">
     </a>
     <strong>
      总结
     </strong>
    </h3>
    <ol>
     <li>
      使用图形界面（推荐给新手）或手动修改
      <code>
       sources.list
      </code>
      来更换镜像源。
     </li>
     <li>
      备份
      <code>
       sources.list
      </code>
      ，防止出错。
     </li>
     <li>
      确定 Ubuntu 版本代号（如
      <code>
       noble
      </code>
      、
      <code>
       jammy
      </code>
      ）。
     </li>
     <li>
      选择合适的国内镜像源（阿里云、清华、中科大等）。
     </li>
     <li>
      修改
      <code>
       sources.list
      </code>
      并保存。
     </li>
     <li>
      执行
      <code>
       sudo apt update &amp;&amp; sudo apt upgrade -y
      </code>
      确保更新。
     </li>
     <li>
      遇到问题可以恢复默认源。
     </li>
    </ol>
    <p>
     通过这些步骤，你可以有效地优化 Ubuntu 的软件源，提高软件更新和安装的速度。如果你在未来使用其他版本 Ubuntu，也只需替换
     <code>
      &lt;版本代号&gt;
     </code>
     即可轻松适配。
    </p>
