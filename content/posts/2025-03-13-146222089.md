    <h4>
     <a id="2__User_Snippets_32">
     </a>
     2. 使用用户代码片段 (User Snippets)
    </h4>
    <p>
     你可以创建自定义的代码片段，以便快速插入常用的代码模板。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开用户代码片段
       </strong>
       :
      </p>
      <ul>
       <li>
        打开命令面板 (
        <code>
         Ctrl + Shift + P
        </code>
        或
        <code>
         Cmd + Shift + P
        </code>
        )，然后输入 “Preferences: Configure User Snippets”。
       </li>
       <li>
        选择你想要为其创建代码片段的语言（例如 JavaScript、HTML 等）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        创建代码片段
       </strong>
       :
      </p>
      <ul>
       <li>
        <p>
         在打开的
         <code>
          snippets.json
         </code>
         文件中，添加一个新的代码片段。例如：
        </p>
        <pre><code class="prism language-json"><span class="token string-property property">"Print to console"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-property property">"prefix"</span><span class="token operator">:</span> <span class="token string">"log"</span><span class="token punctuation">,</span>
    <span class="token string-property property">"body"</span><span class="token operator">:</span> <span class="token punctuation">[</span>
        <span class="token string">"console.log('$1');"</span><span class="token punctuation">,</span>
        <span class="token string">"$2"</span>
    <span class="token punctuation">]</span><span class="token punctuation">,</span>
    <span class="token string-property property">"description"</span><span class="token operator">:</span> <span class="token string">"Log output to console"</span>
<span class="token punctuation">}</span>
</code></pre>
       </li>
       <li>
        <p>
         在这个例子中，当你输入
         <code>
          log
         </code>
         并按下
         <code>
          Tab
         </code>
         键时，VSCode 会自动插入
         <code>
          console.log('');
         </code>
         ，并且光标会停留在
         <code>
          ''
         </code>
         中。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="3__58">
     </a>
     3. 使用扩展
    </h4>
    <p>
     你也可以安装一些扩展来快速生成代码模板，例如：
    </p>
    <ul>
     <li>
      <strong>
       HTML Snippets
      </strong>
      : 提供了一系列 HTML 代码片段。
     </li>
     <li>
      <strong>
       JavaScript (ES6) code snippets
      </strong>
      : 提供了 ES6 的代码片段。
     </li>
     <li>
      <strong>
       Reactjs code snippets
      </strong>
      : 提供了 React 相关的代码片段。
     </li>
    </ul>
    <h4>
     <a id="4__66">
     </a>
     4. 自定义快捷键
    </h4>
    <p>
     你可以为常用的代码片段或命令绑定自定义快捷键。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开快捷键设置
       </strong>
       :
      </p>
      <ul>
       <li>
        打开命令面板 (
        <code>
         Ctrl + Shift + P
        </code>
        或
        <code>
         Cmd + Shift + P
        </code>
        )，然后输入 “Preferences: Open Keyboard Shortcuts”。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        添加快捷键
       </strong>
       :
      </p>
      <ul>
       <li>
        在快捷键设置中，搜索你想要绑定的命令（例如 “Insert Snippet”），然后点击左侧的加号图标来添加快捷键。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="5__77">
     </a>
     5. 使用文件模板
    </h4>
    <p>
     你可以创建文件模板，以便在创建新文件时自动插入预定义的代码。
    </p>
    <ol>
     <li>
      <p>
       <strong>
        创建模板文件
       </strong>
       :
      </p>
      <ul>
       <li>
        创建一个包含你想要插入的代码的文件，例如
        <code>
         template.html
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        使用模板
       </strong>
       :
      </p>
      <ul>
       <li>
        当你创建新文件时，手动复制模板文件的内容，或者使用扩展（如 “File Templates”）来自动插入模板内容。
       </li>
      </ul>
     </li>
    </ol>
