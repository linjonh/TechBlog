    <h3>
     <a id="_7">
     </a>
     最小路径和
    </h3>
    <blockquote>
     <p>
      题目地址：
      <a href="https://leetcode.cn/problems/minimum-path-sum/" rel="nofollow">
       最小路径和
      </a>
     </p>
    </blockquote>
    <h4>
     <a id="_11">
     </a>
     题目描述
    </h4>
    <p>
     给定一个包含非负整数的
     <code>
      m x n
     </code>
     网格
     <code>
      grid
     </code>
     ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。
    </p>
    <p>
     <strong>
      说明：
     </strong>
     每次只能向下或者向右移动一步。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cf3eeb1475464eca93a386fba9bbafa7.png"/>
    </p>
    <pre><code>输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：因为路径 1→3→1→1→1 的总和最小。
</code></pre>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <pre><code>输入：grid = [[1,2,3],[4,5,6]]
输出：12
</code></pre>
    <p>
     <strong>
      提示：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       m == grid.length
      </code>
     </li>
     <li>
      <code>
       n == grid[i].length
      </code>
     </li>
     <li>
      <code>
       1 &lt;= m, n &lt;= 200
      </code>
     </li>
     <li>
      <code>
       0 &lt;= grid[i][j] &lt;= 200
      </code>
     </li>
    </ul>

      <p>

       <code>

      </p>
     </li>
     <li>

       <code>
        tmp[m-1][n-1]
       </code>
       的值，因为
       <code>
        tmp[m-1][n-1]
       </code>
       存储的是就是到达右下角的最小路径和。
      </p>
     </li>
    </ul>
