# 用户界面草图设计工具-工具包和资源

来自：
<http://www.cnblogs.com/RuiLei/archive/2011/11/01/2231218.html>

#### UI界面与设计使用的元素，软件，网站等。内容很丰富，适合网页设计师，用户体验设计师，界面设计师，产品设计师，JS前段开发，手机产品设计，ipad产品设计等使用

**[Lumzy](http://www.lumzy.com/)**

Lumzy是一个网站应用和原型界面制作工具。使用Lumzy，您可以轻松创建UI模型并即时发送到客户电脑中。 Lumzy还具有团队协作编辑工具。
  
[![0.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040018Amw.jpg "0.jpg")](http://www.lumzy.com/)

**[Mockingbird](https://gomockingbird.com/)**

百舌鸟是一个在线工具，它使您轻松地创建UI界面模型，预览UI，并可以共享你的版面编排效果图。
  
[![1.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040019upM.jpg "1.jpg")](https://gomockingbird.com/)

**[The Pencil Project](http://pencil.evolus.vn/en-US/Home.aspx)**

为设计图表和用户界面图形原型开发的一个自由和开源工具。
  
[![2.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040019dsO.jpg "2.jpg")](http://pencil.evolus.vn/en-US/Home.aspx)
  
**[Dojo](http://dojotoolkit.org/)**

Dojo提供了强大的性能并节省您的时间。这是适合经验丰富前端开发者的JS开发工具包。
  
[![3.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040020MTA.jpg "3.jpg")](http://dojotoolkit.org/)
  
**[Mocklinkr](http://www.mocklinkr.com/)**

[![4.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040021PG9.jpg "4.jpg")](http://www.mocklinkr.com/)
  
**[MockFlow](http://www.mockflow.com/signup/)**

MockFlow是一个在线工具，用户界面设计和协作版面编排工具。
  
[![5.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040021q2L.jpg "5.jpg")](http://www.mockflow.com/signup/)
  
**[Serena Prototype Composer](http://www.serena.com/products/prototype-composer/index.html)**

Serena Prototype Composer是制作流程图，模拟程序流程和用户界面设计程序。
  
[![6.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040022eD2.jpg "6.jpg")](http://www.serena.com/products/prototype-composer/index.html)
  
**[Cacoo](http://cacoo.com/)**

Cocoo是一个用户友好的在线绘图工具，允许你创建一个如网站地图，线框，UML和网络图图等。 Cacoo可以免费使用。
  
[![7.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/0400233fz.jpg "7.jpg")](http://cacoo.com/)
  
**[DENIM](http://dub.washington.edu:2007/denim/)**

一个早期非正式的网站和用户界面设计工具
  
[![8.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040023lx7.jpg "8.jpg")](http://dub.washington.edu:2007/denim/)
  
**[Hot Gloo](http://www.hotgloo.com/)**

网上应用线框
  
[![9.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040024ygk.jpg "9.jpg")](http://www.hotgloo.com/)
  
**[User Interface Design Framework](http://www.webalys.com/design-interface-application-framework.php)**

一个免费提供网页设计师使用的用户界面设计工具。
  
[![10.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040025hhI.jpg "10.jpg")](http://www.webalys.com/design-interface-application-framework.php)
  
**[Pattern Tap](http://patterntap.com/)**

可以寻找一些设计资源与模式等。
  
[![11.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040026sm4.jpg "11.jpg")](http://patterntap.com/)
  
**[User Interface Design Patterns](http://ui-patterns.com/)**

提供一些热门的用户UI界面设计模式参考图，分类很详细，如果设计网站或UI界面可以上去找一些资源，很不错。
  
[![12.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/0400276zn.jpg "12.jpg")](http://ui-patterns.com/)
  
**[Patternry](http://patternry.com/)**

Patternry是用户界面设计模式库的目的是帮助解决常见的接口设计问题。

[![13.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/0400275to.jpg "13.jpg")](http://patternry.com/)
  
**[Mephobox](http://box.mepholio.com/)**

UI设计界面资源欣赏参考网站。
  
[![14.jpg](http://www.iptu.net/wp-content/themes/titan/functions/lazyload/grey.gif "14.jpg")](http://box.mepholio.com/)
  
**[Android Asset Studio](http://android-ui-utils.googlecode.com/hg/asset-studio/dist/icons-launcher.html)**

可以在线生成图表，只需要输入参数即可
  
[![15.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/04002871k.jpg "15.jpg")](http://android-ui-utils.googlecode.com/hg/asset-studio/dist/icons-launcher.html)
  
**[Mobility](http://www.fullcreative.com/2010/10/mobility-a-free-set-of-mobile-ui-design-elements/)**

移动用户界面设计使用，可以输入参数进行测试，很方便
  
[![16.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040029Zvh.jpg "16.jpg")](http://www.fullcreative.com/2010/10/mobility-a-free-set-of-mobile-ui-design-elements/)
  
**[Android UI Elements Set](http://www.webdesignshock.com/freebies/free-photoshop-android-interface-gui/)**

可以自由设定Android操作系统的用户界面元素。
  
[![17.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/0400302f3.jpg "17.jpg")](http://www.webdesignshock.com/freebies/free-photoshop-android-interface-gui/)
  
**[OSX Leopard GUI Set](http://www.tutorialshock.com/freebie/mac-osx-leopard-gui-set/)**

该软件包含一些常用的Photoshop原文件和PNG文件，这些元素都是用来设计GUI界面的素材，通过该软件你可以快速制作出一些出色的界面
  
[![18.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040031ZUf.jpg "18.jpg")](http://www.tutorialshock.com/freebie/mac-osx-leopard-gui-set/)
  
**[ALL In One Web Elements Kit](http://bestblogbox.com/freebies/all-in-one-web-elements-kit/)**

网页布局元素大集合,PSD文件。
  
[![19.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040032Q26.jpg "19.jpg")](http://bestblogbox.com/freebies/all-in-one-web-elements-kit/)
  
**[iPad GUI Set](http://www.teehanlax.com/blog/2010/02/01/ipad-gui-psd/)**

iPad平板电脑上设计布局用的元素集合。
  
[![20.jpg](http://www.iptu.net/wp-content/uploads/auto_save_image/2011/03/040033vc6.jpg "20.jpg")](http://www.teehanlax.com/blog/2010/02/01/ipad-gui-psd/)
