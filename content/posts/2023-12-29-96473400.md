     <h4>
      <a id="pip2_3">
      </a>
      一、安装pip2
     </h4>
     <p>
      (因为我的机子上默认装了python2因此先装pip2)
      <br/>
      # 下载pip2
      <br/>
      》》apt-get install python-pip python-dev build-essential
     </p>
     <p>
      # 更新到最新版本
      <br/>
      》》sudo pip install --upgrade pip
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4ffafa0e5bcf6d3dec8b904cb1000e68.png"/>
     </p>
     <blockquote>
      <p>
       <strong>
        报错Tips
       </strong>
       ：查看pip2版本时报错：【升级pip到10.0.0版本后报错ImportError: cannot import name main】
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a0a26a2c786c0313c5eb6b05557ebbc5.png">
        <br/>
        <strong>
         —————————— 解决办法：——————————
        </strong>
       </img>
      </p>
      <blockquote>
       <p>
        由于pip升级到10以后函数不兼容所致！
       </p>
      </blockquote>
      <p>
       # 编辑/usr/bin/pip文件，将：
       <br/>
       》》vi /usr/bin/pip
      </p>
      <pre><code class="prism language-bash">from pip <span class="token function">import</span> main
<span class="token keyword">if</span> __name__ <span class="token operator">==</span> <span class="token string">'__main__'</span><span class="token keyword">:</span>
	sys.exit<span class="token punctuation">(</span>main<span class="token punctuation">(</span><span class="token punctuation">))</span>
</code></pre>
      <p>
       改为以下即可：
      </p>
      <pre><code class="prism language-bash">from pip <span class="token function">import</span> __main__
<span class="token keyword">if</span> __name__ <span class="token operator">==</span> <span class="token string">'__main__'</span><span class="token keyword">:</span>
	sys.exit<span class="token punctuation">(</span>__main__._main<span class="token punctuation">(</span><span class="token punctuation">))</span>
</code></pre>
     </blockquote>
     <p>
      # python2、pip2安装完成

     </p>
     <h4>
      <a id="Python3_31">
      </a>
      二、安装Python3
     </h4>
     <h5>
      <a id="21_python3_32">
      </a>
      2.1 卸载旧版的python3
     </h5>
     <p>
      # 查看本机上是否有旧版本的python3
      <br/>
      》》sudo apt list python3
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9710ccc29a5dc6c3b3df64f135d172f7.png">
       <br/>
       # 卸载旧版本python3.5.3-1
       <br/>
       》》apt-get autoremove python3
      </img>
     </p>
     <h5>
      <a id="22__38">
      </a>
      2.2 官网选择版本
     </h5>
     <p>
      # 官网选择好版本右击复制链接地址
      <a href="https://www.python.org/ftp/python/" rel="nofollow">
       官网传送门
      </a>
      <br/>
      》》https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tgz
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1216743471c8d712a19478fd000083fb.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cd1c0225c051e2cb6b34930bb1a39774.png"/>
      </img>
     </p>
     <h5>
      <a id="23__43">
      </a>
      2.3 创建安装目录
     </h5>
     <p>
      》》 mkdir /usr/local/python3
     </p>
     <h5>
      <a id="24_python3_46">
      </a>
      2.4 安装解压python3包
     </h5>
     <p>
      》》 wget https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tgz
      <br/>
      》》 tar -zxvf Python-3.7.4.tgz
      <br/>
      》》rm -rf Python-3.7.4.tgz
      <br/>
      # cd到Python源目录
      <br/>
      》》cd Python-3.7.4
     </p>
     <h5>
      <a id="25_python3_52">
      </a>
      2.5 安装python3依赖包
     </h5>
     <p>
      》》apt -y install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev
     </p>
     <blockquote>
      <p>
       yum -y groupinstall “Development tools”
       <br/>
       yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel
       <br/>
       yum install libffi-devel -y
      </p>
     </blockquote>
     <h5>
      <a id="26__58">
      </a>
      2.6 编译安装
     </h5>
     <p>
      # 配置安装的位置
      <br/>
      》》./configure --prefix=/usr/local/python3 --with-ssl
     </p>
     <p>
      # 编译源码
      <br/>
      》》make all
     </p>
     <p>
      # 安装python二进制文件
      <br/>
      》》make altinstall
     </p>
     <p>
      # 创建Python3和pip3软链接，将系统命令python3、pip3指向刚安装的python3、pip3
      <br/>
      》》 ln -s /usr/local/python3/bin/python3 /usr/bin/python3
      <br/>
      》》 ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3
     </p>
