    <h2>
     1. ES6 简介
    </h2>
    <p>
     ES6（ECMAScript 6）是 JavaScript 的一个重要版本，它在 ES5 的基础上进行了扩展和优化。ES6 主要应用于现代 Web 开发，提高了 JavaScript 的编程效率和可读性。
    </p>
    <h3>
     2. ES6 与 JavaScript 的关系
    </h3>
    <p>
     JavaScript 是一种基于 ECMAScript 规范的编程语言，而 ES6 是 ECMAScript 的第六个主要版本（ECMAScript 2015），它带来了许多新的特性，如
     <code>
      let
     </code>
     和
     <code>
      const
     </code>
     关键字、箭头函数、类（Class）、模块化（Module）、模板字符串等，使 JavaScript 变得更加强大和易用。
    </p>
    <h3>
     3. 变量与常量
    </h3>
    <h4>
     3.1 变量（Variable）
    </h4>
    <p>
     在 ES6 中，定义变量可以使用
     <code>
      let
     </code>
     关键字。例如：
    </p>
    <pre><code>let name = "鹏哥";
console.log(name); // 输出：鹏哥
console.log(typeof name); // 输出：string</code></pre>
    <p>
     变量
     <code>
      name
     </code>
     存储了一个字符串，并且可以在后续代码中被重新赋值。
    </p>
    <h4>
     3.2 常量（Constant）
    </h4>
    <p>
     常量使用
     <code>
      const
     </code>
     关键字定义，定义后不能修改。
    </p>
    <pre><code>const pi = 3.14;
console.log(pi); // 输出：3.14
console.log(typeof pi); // 输出：number</code></pre>
    <h3>
     4. 数据类型
    </h3>
    <p>
     ES6 提供了多种数据类型，主要包括：
    </p>
    <h4>
     4.1 字符串（String）
    </h4>
    <p>
     字符串使用双引号
     <code>
      ""
     </code>
     或单引号
     <code>
      ''
     </code>
     进行定义。
    </p>
    <pre><code>let greeting = "你好，ES6";
console.log(greeting);
console.log(typeof greeting); // 输出：string</code></pre>
    <h4>
     4.2 数值（Number）
    </h4>
    <p>
     ES6 的数值类型既可以表示整数，也可以表示浮点数。
    </p>
    <pre><code>let age = 22;
console.log(age);
console.log(typeof age); // 输出：number</code></pre>
    <h4>
     4.3 布尔值（Boolean）
    </h4>
    <p>
     布尔值只有
     <code>
      true
     </code>
     和
     <code>
      false
     </code>
     两种状态。
    </p>
    <pre><code>let isStudent = true;
console.log(isStudent);
console.log(typeof isStudent); // 输出：boolean</code></pre>
    <h4>
     4.4 对象（Object）
    </h4>
    <p>
     对象用于存储键值对。
    </p>
    <pre><code>let boy = {
    name: "theodore",
    age: 22
};
console.log(boy);
console.log(typeof boy); // 输出：object</code></pre>
    <h4>
     4.5 映射（Map）
    </h4>
    <p>
     <code>
      Map
     </code>
     是一种键值对集合，键的类型可以是任何值。
    </p>
    <pre><code>let girl = new Map([
    ["name", "andy"],
    ["age", 20]
]);
console.log(girl);
console.log(typeof girl); // 输出：object</code></pre>
    <h4>
     4.6 集合（Set）
    </h4>
    <p>
     <code>
      Set
     </code>
     是一种不允许重复值的数据结构。
    </p>
    <pre><code>let numbers = new Set([1, 2, 3, 4, 5, 5, 6]);
console.log(numbers);
console.log(typeof numbers); // 输出：object</code></pre>
    <h4>
     4.7 数组（Array）
    </h4>
    <p>
     数组用于存储多个值。
    </p>
    <pre><code>let strArray = ["a", "1", "你好"];
console.log(strArray);
console.log(typeof strArray); // 输出：object</code></pre>
    <h4>
     4.8 函数（Function）
    </h4>
    <p>
     函数用于封装可复用的代码。
    </p>
    <pre><code>function add(a, b) {
    return a + b;
}
console.log(add(2, 3)); // 输出：5
console.log(typeof add); // 输出：function</code></pre>
    <h4>
     4.9 类（Class）
    </h4>
    <p>
     ES6 引入了类的概念，使 JavaScript 具备面向对象编程的特性。
    </p>
    <pre><code>class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    info() {
        console.log(this.name, this.age);
    }
}
let c = new Person("大哥", 23);
c.info(); // 输出：大哥 23</code></pre>
    <h3>
     5. 结语
    </h3>
    <p>
     ES6 作为 JavaScript 的一个重要版本，极大地增强了语言的可读性和可维护性。通过本篇博客，我们了解了 ES6 的基本概念、变量与常量的定义方式，以及各种数据类型的使用方法。掌握这些基础知识，将有助于更深入地学习 JavaScript 及其应用。
    </p>
