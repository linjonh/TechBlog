    <h4>
     1.
     <strong>
      jps（Java Virtual Machine Process Status Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：列出当前用户的所有 Java 进程及其 PID。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：快速定位目标 Java 应用的进程 ID。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jps -l  # 显示主类全名
jps -v  # 显示 JVM 启动参数</code></pre>
     </li>
    </ul>
    <h4>
     2.
     <strong>
      jstat（JVM Statistics Monitoring Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：实时监控 JVM 内存、GC、类加载等状态。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：监控垃圾回收（GC）性能，分析内存使用情况。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jstat -gcutil &lt;pid&gt; 1000 10  # 每 1 秒采样 GC 状态，共 10 次
jstat -class &lt;pid&gt;  # 类加载统计</code></pre>
     </li>
    </ul>
    <h4>
     3.
     <strong>
      jmap（Memory Map Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：生成堆转储文件（Heap Dump）或查看堆内存对象分布。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：分析内存泄漏、排查内存溢出问题。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jmap -heap &lt;pid&gt;  # 查看堆内存使用详情
jmap -histo:live &lt;pid&gt;  # 统计存活对象
jmap -dump:format=b,file=/path/to/dump.hprof &lt;pid&gt;  # 生成堆转储文件</code></pre>
     </li>
    </ul>
    <h4>
     4.
     <strong>
      jstack（Stack Trace Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：生成线程快照，用于分析线程死锁或高 CPU 问题。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：排查线程死锁、分析线程状态。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jstack &lt;pid&gt; &gt; thread_dump.txt  # 输出线程快照到文件</code></pre>
     </li>
    </ul>
    <h4>
     5.
     <strong>
      jcmd（JVM Command Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：多功能工具，支持 GC、类加载、线程等诊断。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：动态调整 JVM 参数、生成堆转储文件、打印线程信息等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jcmd &lt;pid&gt; VM.flags  # 查看 JVM 参数
jcmd &lt;pid&gt; GC.heap_info  # 查看堆内存信息
jcmd &lt;pid&gt; Thread.print  # 打印线程快照</code></pre>
     </li>
    </ul>
    <h4>
     6.
     <strong>
      jinfo（Java Information Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：查看和动态修改 JVM 的运行时参数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：查看 JVM 启动参数、动态调整参数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jinfo -flags &lt;pid&gt;  # 查看 JVM 参数
jinfo -flag +HeapDumpBeforeFullGC &lt;pid&gt;  # 动态启用 HeapDumpBeforeFullGC</code></pre>
     </li>
    </ul>
    <h4>
     7.
     <strong>
      jhat（Heap Analysis Tool）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：分析 jmap 生成的堆转储文件（Heap Dump）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：分析内存泄漏、排查对象占用情况。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用命令
       </strong>
       ：
      </p>
      <p>
       bash复制
      </p>
      <pre><code>jhat /path/to/dump.hprof  # 启动 HTTP 服务器，通过浏览器访问分析结果</code></pre>
     </li>
    </ul>
    <h4>
     8.
     <strong>
      jconsole（Java Monitoring and Management Console）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：图形化工具，用于监控 JVM 的内存、线程、类加载等信息。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：开发环境中的实时监控和调优。
      </p>
     </li>
     <li>
      <p>
       <strong>
        使用方法
       </strong>
       ：直接启动
       <code>
        jconsole
       </code>
       ，选择目标进程进行监控。
      </p>
     </li>
    </ul>
    <h4>
     9.
     <strong>
      jmc（Java Mission Control）
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：高级的图形化监控工具，支持飞行记录器（Flight Recorder）功能。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用场景
       </strong>
       ：生产环境中的性能监控和问题诊断。
      </p>
     </li>
     <li>
      <p>
       <strong>
        使用方法
       </strong>
       ：启动
       <code>
        jmc
       </code>
       ，连接到目标 JVM，进行实时监控或分析飞行记录。
      </p>
     </li>
    </ul>
    <h4>
     总结
    </h4>
    <p>
     这些工具各有用途，可以根据实际需求选择合适的工具：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        开发环境
       </strong>
       ：优先使用图形化工具（如 jconsole、jmc）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        生产环境
       </strong>
       ：推荐使用低开销工具（如 jcmd、jstat）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        深度分析
       </strong>
       ：结合日志分析工具（如 jmap + jhat、MAT）进行优化。
      </p>
     </li>
    </ul>
