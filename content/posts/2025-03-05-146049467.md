    <p>
     <strong>
      nvm（Node Version Manager）是一个用于管理多个 Node.js 版本的工具，支持在同一个系统上轻松切换不同的 Node.js 版本
     </strong>
    </p>
    <h3>
     <a id="1_Windows__2">
     </a>
     1.在 Windows 上安装
    </h3>
    <p>
     官方最新迁移镜像地址 (永久替换 2021-04-18 留言)
    </p>
    <pre><code>npm config set registry http://registry.npmmirror.com
</code></pre>
    <p>
     对于 Windows 用户，可以使用 nvm-windows：
     <br/>
     下载安装程序：
     <br/>
     <a href="https://nvm.uihtm.com/" rel="nofollow">
      nvm官网https://nvm.uihtm.com/
     </a>
     下载最新版本的安装程序（如 nvm-setup.zip）。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dd5ba68cc1664d8386d6a67dd490b14c.png"/>
    </p>
    <p>
     安装：
     <br/>
     解压下载的文件并运行安装程序，按照提示完成安装。（注：新手可以将nvm安装在C盘下，我安装到了D盘）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/235a0abae7904356a8680f6f3edfc98a.png"/>
    </p>
    <p>
     验证安装：
     <br/>
     打开命令提示符并输入以下命令检查是否安装成功
    </p>
    <pre><code>nvm version
</code></pre>
    <p>
     要查看所有可安装的 Node.js 版本，可以使用：
    </p>
    <pre><code>nvm ls-remote
或者
nvm ls available
</code></pre>
    <p>
     要安装特定版本的 Node.js，请使用以下命令：
    </p>
    <pre><code>nvm install &lt;version&gt;
例如：nvm install 16.14.0
</code></pre>
    <p>
     查看当前已安装的所有 Node.js 版本：
    </p>
    <pre><code>nvm ls
</code></pre>
    <p>
     切换 Node.js 版本，要在不同版本之间切换，请使用以下命令：
    </p>
    <pre><code>nvm use &lt;version&gt;
例如：nvm use16.14.0
</code></pre>
    <p>
     查看是否切换完成：
    </p>
    <pre><code>node -v
</code></pre>
    <p>
     要卸载某个已安装的 Node.js 版本：
    </p>
    <pre><code>nvm uninstall &lt;version&gt;
例如：
nvm uninstall 16.14.0
</code></pre>
    <p>
     使用node下载依赖：
    </p>
    <pre><code>npm install
</code></pre>
    <p>
     以下是下了多个node版本文件夹展示，这些都是通过nvm来控制；
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3471a9f9090d498ebb0a015db5cff1c1.png"/>
    </p>
    <h3>
     <a id="2nvmnode_66">
     </a>
     2.如果nvm安装node时候一直失败怎么办？
    </h3>
    <p>
     可以采用手动下载node压缩文件，
     <a href="https://nodejs.org/en/about/previous-releases?spm=5176.28103460.0.0.38405d27e7dMp5" rel="nofollow">
      node下载地址
     </a>
     <br/>
     然后解压到nvm这个文件夹，与其他node版本同一个位置，并且文件夹命名改为node正确版本名比方：v10.24.1
     <br/>
     举例：进入网站点击
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/29bbccb2190a44c08ccf9d230ec64c39.png">
      <br/>
      点击压缩包下载解压到nvm文件夹；
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0f055cbe27214d4ba33fc8ddcc0517c2.png">
       <br/>
       创建文件夹v10.24.1,解压到此文件夹如下：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bd0dfa460c1544a5bb1769572c0b533c.png"/>
      </img>
     </img>
    </p>
    <p>
     这个时候就可以在项目中使用nvm切换到这个版本了，然后npm install 下载依赖了
    </p>
    <h3>
     <a id="3_77">
     </a>
     3、配置环境变量
    </h3>
    <p>
     找到下载nvm的目录，在地址栏复制文件路径
    </p>
    <p>
     Windows环境：计算机——&gt;右键——&gt;属性——&gt;高级属性设置——&gt;环境变量——&gt;系统环境变量——&gt;path——&gt;编辑（一般会自动生成环境变量）——&gt;确定
    </p>
    <p>
     在终端输入：nvm -v 出现如下图所示，说明安装nvm成功，显示版本号以及nvm的一些命令
     <br/>

      https://blog.csdn.net/weixin_43615570/article/details/128740964
     </a>
    </p>
