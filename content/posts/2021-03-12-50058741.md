     <span style="font-size:24px">
      discuz没有直接进入云平台地址怎么办：
      <br/>
      <br/>
      DiscuzX3.2已经改版，后台不能直接进云平台列表，所以不方便操作，用云平台服务，大家可以安照下面操作：
      <br/>
      <br/>
      Discuz X3.2进入云平台地址：
      <br/>
      <br/>
      1、登录后台；
      <br/>
      <br/>
      2、输入：http://你域名/admin.php?frames=yes&amp;action=cloud&amp;operation=applist
      <br/>
      <br/>
      3、即可访问域名进入云平台列表
      <br/>
      <br/>
      Discuz X3.2云平台开通地址：
      <br/>
      <br/>
      1、登录后台；
      <br/>
      <br/>
      2、输入：http://你域名/admin.php?action=cloud&amp;operation=open
      <br/>
      <br/>
      3、进入域名进行开通云平台页面 ：
      <br/>
      <br/>
      最后总结：
      <br/>
      <br/>
      由于要经常进入Discuz X3.2云平台。所以建议大家收藏这文章，或者收藏内容下来，这样也方便很多
