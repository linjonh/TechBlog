    <p>
     以下是
     <strong>
      在 Ubuntu 上安装 Python 3.10
     </strong>
     的详细步骤（兼容 Ubuntu 20.04/22.04）：
    </p>
    <hr/>
    <h4>
     <a id="_PPA__4">
     </a>
     方法一：通过 PPA 仓库安装（推荐）
    </h4>
    <h5>
     <a id="1__deadsnakes_PPA_5">
     </a>
     1. 添加 deadsnakes PPA
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> software-properties-common
<span class="token function">sudo</span> add-apt-repository ppa:deadsnakes/ppa
<span class="token function">sudo</span> <span class="token function">apt</span> update
</code></pre>
    <h5>
     <a id="2__Python_310_13">
     </a>
     2. 安装 Python 3.10
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> python3.10 python3.10-venv python3.10-dev <span class="token parameter variable">-y</span>
</code></pre>
    <h5>
     <a id="3__18">
     </a>
     3. 验证安装
    </h5>
    <pre><code class="prism language-bash">python3.10 <span class="token parameter variable">--version</span>
<span class="token comment"># 应输出：Python 3.10.x</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_26">
     </a>
     方法二：手动编译安装（自定义路径）
    </h4>
    <h5>
     <a id="1__27">
     </a>
     1. 安装编译依赖
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev libsqlite3-dev <span class="token function">wget</span> libbz2-dev <span class="token parameter variable">-y</span>
</code></pre>
    <h5>
     <a id="2__Python_310__33">
     </a>
     2. 下载 Python 3.10 源码
    </h5>
    <pre><code class="prism language-bash"><span class="token function">wget</span> https://www.python.org/ftp/python/3.10.13/Python-3.10.13.tar.xz
<span class="token function">tar</span> <span class="token parameter variable">-xf</span> Python-3.10.13.tar.xz
<span class="token builtin class-name">cd</span> Python-3.10.13
</code></pre>
    <h5>
     <a id="3__usrlocal_40">
     </a>
     3. 编译安装（到
     <code>
      /usr/local
     </code>
     ）
    </h5>
    <pre><code class="prism language-bash">./configure --enable-optimizations
<span class="token function">make</span> <span class="token parameter variable">-j</span> <span class="token variable"><span class="token variable">$(</span>nproc<span class="token variable">)</span></span>
<span class="token function">sudo</span> <span class="token function">make</span> altinstall  <span class="token comment"># 使用 altinstall 避免覆盖系统默认 Python</span>
</code></pre>
    <h5>
     <a id="4__47">
     </a>
     4. 验证安装
    </h5>
    <pre><code class="prism language-bash">python3.10 <span class="token parameter variable">--version</span>
<span class="token comment"># 应输出：Python 3.10.13</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_Python_310__55">
     </a>
     配置 Python 3.10 为默认版本（可选）
    </h4>
    <h5>
     <a id="1__Python__56">
     </a>
     1. 查看当前 Python 版本
    </h5>
    <pre><code class="prism language-bash">python3 <span class="token parameter variable">--version</span>
</code></pre>
    <h5>
     <a id="2__alternatives__61">
     </a>
     2. 更新 alternatives 配置
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.10 <span class="token number">1</span>
</code></pre>
    <h5>
     <a id="3__66">
     </a>
     3. 切换默认版本
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> python3
</code></pre>
    <p>
     按提示选择 Python 3.10 对应的编号
    </p>
    <hr/>
    <h4>
     <a id="_pip_for_Python_310_74">
     </a>
     安装 pip for Python 3.10
    </h4>
    <h5>
     <a id="1__pip_75">
     </a>
     1. 安装 pip
    </h5>
    <pre><code class="prism language-bash"><span class="token function">curl</span> https://bootstrap.pypa.io/get-pip.py <span class="token parameter variable">-o</span> get-pip.py
python3.10 get-pip.py
</code></pre>
    <h5>
     <a id="2__pip_81">
     </a>
     2. 验证 pip
    </h5>
    <pre><code class="prism language-bash">pip3.10 <span class="token parameter variable">--version</span>
<span class="token comment"># 应输出：pip 23.x.x from ... (python 3.10)</span>
</code></pre>
    <hr/>
    <h4>
     <a id="Python_310_89">
     </a>
     创建虚拟环境（Python 3.10）
    </h4>
    <pre><code class="prism language-bash">python3.10 <span class="token parameter variable">-m</span> venv myenv
<span class="token builtin class-name">source</span> myenv/bin/activate
<span class="token comment"># 此时命令行提示符会显示 (myenv)</span>
</code></pre>
    <hr/>
    <h4>
     <a id="_PPA__98">
     </a>
     卸载方法（仅限 PPA 安装）
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> remove python3.10 python3.10-venv python3.10-dev
<span class="token function">sudo</span> add-apt-repository <span class="token parameter variable">--remove</span> ppa:deadsnakes/ppa
</code></pre>
    <hr/>
    <h4>
     <a id="_106">
     </a>
     常见问题解决
    </h4>
    <h5>
     <a id="_1addaptrepository__107">
     </a>
     问题 1：
     <code>
      add-apt-repository 命令不存在
     </code>
    </h5>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> software-properties-common
</code></pre>
    <h5>
     <a id="_2SSL__112">
     </a>
     问题 2：SSL 模块缺失
    </h5>
    <p>
     手动编译时需确保 OpenSSL 开发包已安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> libssl-dev
</code></pre>
    <h5>
     <a id="_3pip__118">
     </a>
     问题 3：pip 安装失败
    </h5>
    <p>
     使用官方脚本重装：
    </p>
    <pre><code class="prism language-bash">python3.10 <span class="token parameter variable">-m</span> ensurepip <span class="token parameter variable">--upgrade</span>
</code></pre>
