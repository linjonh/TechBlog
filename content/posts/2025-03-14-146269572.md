# Ubuntu 服务器安装 Python 环境 的详细指南

以下是
**在 Ubuntu 上安装 Python 3.10**
的详细步骤（兼容 Ubuntu 20.04/22.04）：

---

#### 方法一：通过 PPA 仓库安装（推荐）

##### 1. 添加 deadsnakes PPA

```bash
sudo apt update
sudo apt install software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update

```

##### 2. 安装 Python 3.10

```bash
sudo apt install python3.10 python3.10-venv python3.10-dev -y

```

##### 3. 验证安装

```bash
python3.10 --version
# 应输出：Python 3.10.x

```

---

#### 方法二：手动编译安装（自定义路径）

##### 1. 安装编译依赖

```bash
sudo apt update
sudo apt install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev libsqlite3-dev wget libbz2-dev -y

```

##### 2. 下载 Python 3.10 源码

```bash
wget https://www.python.org/ftp/python/3.10.13/Python-3.10.13.tar.xz
tar -xf Python-3.10.13.tar.xz
cd Python-3.10.13

```

##### 3. 编译安装（到 `/usr/local` ）

```bash
./configure --enable-optimizations
make -j $(nproc)
sudo make altinstall  # 使用 altinstall 避免覆盖系统默认 Python

```

##### 4. 验证安装

```bash
python3.10 --version
# 应输出：Python 3.10.13

```

---

#### 配置 Python 3.10 为默认版本（可选）

##### 1. 查看当前 Python 版本

```bash
python3 --version

```

##### 2. 更新 alternatives 配置

```bash
sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.10 1

```

##### 3. 切换默认版本

```bash
sudo update-alternatives --config python3

```

按提示选择 Python 3.10 对应的编号

---

#### 安装 pip for Python 3.10

##### 1. 安装 pip

```bash
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python3.10 get-pip.py

```

##### 2. 验证 pip

```bash
pip3.10 --version
# 应输出：pip 23.x.x from ... (python 3.10)

```

---

#### 创建虚拟环境（Python 3.10）

```bash
python3.10 -m venv myenv
source myenv/bin/activate
# 此时命令行提示符会显示 (myenv)

```

---

#### 卸载方法（仅限 PPA 安装）

```bash
sudo apt remove python3.10 python3.10-venv python3.10-dev
sudo add-apt-repository --remove ppa:deadsnakes/ppa

```

---

#### 常见问题解决

##### 问题 1： `add-apt-repository 命令不存在`

```bash
sudo apt install software-properties-common

```

##### 问题 2：SSL 模块缺失

手动编译时需确保 OpenSSL 开发包已安装：

```bash
sudo apt install libssl-dev

```

##### 问题 3：pip 安装失败

使用官方脚本重装：

```bash
python3.10 -m ensurepip --upgrade

```
