# 寻找数字c++

### 题目描述

小杨有一个正整数 a，小杨想知道是否存在一个正整数b， 满足
![a=b^{4}](https://latex.csdn.net/eq?a%3Db%5E%7B4%7D)
，如16=2\*2\*2\*2=24

### 输入

第一行包含一个正整数t ，代表测试数据组数。 对于每组测试数据，第一行包含一个正整数代表a 。

### 输出

对于每组测试数据，如果存在满足条件的正整数b ，则输出b ，否则输出 -1。

### 样例输入

```
3
16
81
10
```

### 样例输出

```
2
3
-1
```

### 提示

对于全部数据，保证有1<=t<=105,1<=a<=108 。

示例代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long t,a,b;
    cin>>t; 
    for(int i=1;i<=t;i++){
    	cin>>a;
    	bool l=1;
    	for(b=1;b<=sqrt(a);b++){
    		if(b*b*b*b==a){
    			l=0;
    			break;
			}
		}
		if(l==0) cout<<b<<endl;
		else cout<<-1<<endl;
	}
	return 0;
}
```
