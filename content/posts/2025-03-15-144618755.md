    <h3>
     题目描述
    </h3>
    <p>
     小杨有一个正整数 a，小杨想知道是否存在一个正整数b， 满足
     <img alt="a=b^{4}" class="mathcode" src="https://latex.csdn.net/eq?a%3Db%5E%7B4%7D">
      ，如16=2*2*2*2=24
     </img>
    </p>
    <h3>
     输入
    </h3>
    <p>
     第一行包含一个正整数t ，代表测试数据组数。 对于每组测试数据，第一行包含一个正整数代表a 。
    </p>
    <h3>
     输出
    </h3>
    <p>
     对于每组测试数据，如果存在满足条件的正整数b ，则输出b ，否则输出 -1。
    </p>
    <h3>
     样例输入
    </h3>
    <pre>3
16
81
10</pre>
    <h3>
     样例输出
    </h3>
    <pre>2
3
-1</pre>
    <h3>
     提示
    </h3>
    <p>
     对于全部数据，保证有1&lt;=t&lt;=105,1&lt;=a&lt;=108 。
    </p>
    <p>
    </p>
    <p>
     示例代码：
    </p>
    <pre><code class="language-cpp">#include &lt;bits/stdc++.h&gt;
using namespace std;
int main(){
    long long t,a,b;
    cin&gt;&gt;t; 
    for(int i=1;i&lt;=t;i++){
    	cin&gt;&gt;a;
    	bool l=1;
    	for(b=1;b&lt;=sqrt(a);b++){
    		if(b*b*b*b==a){
    			l=0;
    			break;
			}
		}
		if(l==0) cout&lt;&lt;b&lt;&lt;endl;
		else cout&lt;&lt;-1&lt;&lt;endl;
	}
	return 0;
}</code></pre>
