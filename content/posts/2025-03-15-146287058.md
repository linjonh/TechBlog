    <h4 id="dklds" name="%E5%B1%82%E7%BA%A7">
     层级
    </h4>
    <p>
     地图的配置是一层盖一层，从
     <code>
      底层
     </code>
     到
     <code>
      顶层
     </code>
     的配置依次为：
    </p>
    <p>
     <code>
      geo
     </code>
     =&gt;
     <code>
      series
     </code>
     =&gt;
     <code>
      visualMap
     </code>
    </p>
    <p>
     <code>
      geo
     </code>
     和
     <code>
      series
     </code>
     中都有两个配置项
    </p>
    <ul>
     <li>
      <code>
       itemStyle
      </code>
      未选中的样式
     </li>
     <li>
      <code>
       emphasis
      </code>
      选中的样式
     </li>
    </ul>
    <p>
     <code>
      geo
     </code>
     隐藏掉只有
     <code>
      series
     </code>
     也是能正常显示整个地图的。
    </p>
    <p>
     <code>
      geo
     </code>
     和
     <code>
      series
     </code>
     的搭配可以实现一些阴影，偏移从而实现类似立体的效果。
    </p>
    <h4 id="a3cis" name="visualMap">
     visualMap
    </h4>
    <p>
     其中
     <code>
      visualMap
     </code>
     设置的背景颜色是动态的，会根据数值的大小在两个颜色之间变化。
    </p>
    <pre><code class="hljs">visualMap: {
  show: false,
  seriesIndex: [0],
  inRange: {
    color: ["#cdcaf8", "#026aa9"],
  },
},</code></pre>
    <p>
     需要注意的是：
    </p>
    <blockquote>
     <p>
      当区域有数据的时候会使用visualMap的颜色覆盖，如果没有数据则会使用
      <code>
       series
      </code>
      的
      <code>
       itemStyle
      </code>
      的样式。
     </p>
    </blockquote>
    <p>
     默认最小值和最大值会根据数据自动计算，当然我们也可以指定具体的值。
    </p>
    <pre><code class="hljs">visualMap: {
    min: 0,
    max: 100,
    inRange: {
        color: ['#e0ffff', '#0066ff'] // 设置颜色范围，浅色到深色
    },
    calculable: true,
    show: true
},</code></pre>

    <h4 id="5pk4q" name="%E7%BA%BF%E6%80%A7%E6%B8%90%E5%8F%98">
     线性渐变
    </h4>
    <p>
     沿着一条直线从一种颜色过渡到另一种颜色。
    </p>
    <pre><code class="hljs">{
    type: "linear",
    x: 0,
    y: 0,
    x2: 1,
    y2: 0,
    colorStops: [
      {
        offset: 0,
        color: "#8db1f1", // 起始颜色
      },
      {
        offset: 1,
        color: "#026aa9", // 终止颜色
      },
    ],
}</code></pre>
    <h4 id="fatfq" name="%E5%BE%84%E5%90%91%E6%B8%90%E5%8F%98">
     径向渐变
    </h4>
    <p>
     从一个中心点向四周辐射的渐变色。
    </p>
    <pre><code class="hljs">{
    type: 'radial',
    x: 0.5, 
    y: 0.5,
    r: 0.5,
    colorStops: [{
        offset: 0, color: 'rgba(255, 0, 0, 1)' // 起始颜色
    }, {
        offset: 1, color: 'rgba(0, 0, 255, 1)' // 终止颜色
    }]
}</code></pre>
    <p>
     你可以设置以下属性：
    </p>
    <ol>
     <li>
      type：设置渐变的类型，固定值为 ‘radial’。
     </li>
     <li>
      x、y：设置渐变的中心点坐标，取值范围为 0~1，其中 (0, 0) 表示左上角，(1, 1) 表示右下角。
     </li>
     <li>
      r：设置渐变的半径，也是相对于整个绘图区域的半径，取值范围为 0~1，1 表示半径等于绘图区域的最短边的一半。
     </li>
     <li>
      colorStops：设置渐变的起始颜色、终止颜色以及它们的位置。 它是一个数组，每个元素包含两个属性，分别是 offset 和 color。 其中 offset 表示该位置距离渐变位置的百分比，取值范围为 0~1，color 则是该位置所对应的颜色值。
     </li>
    </ol>
