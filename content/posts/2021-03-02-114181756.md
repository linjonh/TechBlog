    <h2>
     <a id="windows_4">
     </a>
     达梦数据库windows安装
    </h2>
    <p>
     达梦数据库产品下载页面：
     <br/>
     <a href="https://eco.dameng.com/download/" rel="nofollow">
      达梦产品下载页面
     </a>

     下载windows开发版即可
    </p>
    <p>
     我这里下载的是dm8_20201211_x86_win_64_ent_8.1.1.156.iso版本
    </p>
    <p>
     这里再放一个达梦云适配中心链接，里面的文档模块有很多官方教程，社区模块也可以查询或者提出各类使用问题
     <br/>

      达梦云适配中心
     </a>
    </p>
    <h3>
     <a id="_14">
     </a>
     打开安装程序
    </h3>
    <p>
     使用压缩软件解压iso文件，进入到解压的文件夹，可以看到安装执行程序
     <strong>
      setup.exe
     </strong>
     ，右键以管理员身份运行即可进入图形化安装界面。（也可以通过软碟通加载iso镜像文件到虚拟盘的形式进入安装文件目录）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf20f8294af2b4cabdcf5bfa3a74421b.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/217d288eb091e740c14869e302c856a0.png#pic_center"/>
    </p>
    <ol>
     <li>
      <p>
       第一步窗口是选择语言和时区，我们直接默认，点击确定；
      </p>
     </li>
     <li>
      <p>
       安装向导一直点击下一步，勾选接受授权协议；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4ddfa5208e397549943695e3546a695.png#pic_center">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bd36878c9cf5c006be53d17b9695d76a.png#pic_center"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       因为我们是用于测试自用的开发版，这里key选择界面我们无需理会，直接点击下一步跳过；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5384e33cd063cd29d44a3675f91391f5.png"/>
      </p>
     </li>
     <li>
      <p>
       选择组件界面，默认选择典型安装，安装所有,包括服务端、客户端、驱动、手册、注册服务和工具等；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3b5fc5d8416878e5e7db6c77e4cbd43a.png"/>
      </p>
     </li>
     <li>
      <p>
       安装目录，自行选择即可；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/94cb00a1db5b02eb4e4c75cf851abcad.png"/>
      </p>
     </li>
     <li>
      <p>
       最后点击安装；
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a7b06943a85dfdda38f6790325374fd9.png"/>
      </p>
     </li>
     <li>
      <p>
       安装完成后，会弹出初始化数据库实例向导，
      </p>
     </li>
    </ol>
    <h3>
     <a id="_41">
     </a>
     第一次安装需要初始化实例，创建数据库实例
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7895f49d30cf313d7a4b9aa8abbf5fc.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/362b862d95801d42d30c2c014ab33b34.png"/>
    </p>
    <ol start="8">
     <li>
      创建数据库模板、指定数据库目录、数据库标识、数据库文件等可以保持默认配置，一路点击下一步，到达初始化参数界面，
      <strong>
       这里修改一下页大小为32，日志大小磁盘空间充裕的话建议改为最大2048
      </strong>
      ，字符集、大小写是否敏感、兼容 ORACLE结尾空格填充模式、VARCHAR类型以字节还是字符为单位则根据实际使用需要做选择。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1987285e51e2a62a7bc5558e76d75b47.png"/>
     </li>
     <li>
      往后口令管理可以修改一下系统用户密码，实例库可以默认或者需要添加，然后点击完成即可初始化实例。完成会弹出成功提示界面
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/35013a6b28f7f83493d6b1c777bdd194.png"/>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b6b8017b0f969bb02f52df78e50ddf8.png"/>
     </li>
     <li>
      完成之后我们还有一步，就是把数据库实例添加到windows服务中，在成功界面点击继续，到助手界面选择注册数据库服务
     </li>
    </ol>
    <h3>
     <a id="dminit_53">
     </a>
     手动注册数据库实例服务（如果是使用dminit工具初始化数据库实例才需要此步骤）
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a06212c073513b9cfbd95d20bf318d11.png"/>
     <br/>
     12. 弹出配置选择界面，我们点击INI配置文件，浏览按钮，找到刚刚初始化实例的目录下的dm.ini文件，如果前面我们是保持默认的，则文件应该在data\DAMENG\里
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/52690f9d5d70b80915f6f81e77f5c753.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0df7e9a53146eb2ce103014ae6e64198.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9acdafeac10208a39f7d2c645af6f358.png"/>
     <br/>
     13. 选择配置文件后，点击完成
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b63fc899593f1e42516fab3742a31f89.png"/>
     <br/>
     14. 注册完成后，打开开始-达梦数据库-DM服务查看器，可以看到数据库实例服务已经在正常运行。

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/118f2d9744b54734f5a93d127980f3c8.png"/>
