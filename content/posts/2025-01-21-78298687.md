# python制作二维码

随着科技的高速发展，我们现在出门买东西都是用二维码支付的，还有一些网页是通过扫描二维码进入的

我们说下怎么用python制作一个简单的二维码

我也是看其他大佬的讲解才会了，时间太长了，忘了是那个网址

进入正题

我们要用到的模块 qrcode

安装模块

```python
在cmd命令行中  pip install qrcode
```

或者在pycharm中安装

```python
import qrcode   #导入模块
q= qrcode.main.QRCode()
q.add_data('输入内容')
q.make()
m = q.make_image()
m.save('1.png') #1.png是生成的二维码的名字
```
