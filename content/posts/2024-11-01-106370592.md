     <h3>
      <a id="_2">
      </a>
      标题：生命游戏
     </h3>
     <p>
      康威生命游戏是英国数学家约翰·何顿·康威在1970年发明的细胞自动机。
      <br/>
      这个游戏在一个无限大的2D网格上进行。
     </p>
     <p>
      初始时，每个小方格中居住着一个活着或死了的细胞。
      <br/>
      下一时刻每个细胞的状态都由它周围八个格子的细胞状态决定。
     </p>
     <p>
      具体来说：
     </p>
     <ol>
      <li>
       当前细胞为存活状态时，当周围低于2个（不包含2个）存活细胞时， 该细胞变成死亡状态。（模拟生命数量稀少）
      </li>
      <li>
       当前细胞为存活状态时，当周围有2个或3个存活细胞时， 该细胞保持原样。
      </li>
      <li>
       当前细胞为存活状态时，当周围有3个以上的存活细胞时，该细胞变成死亡状态。（模拟生命数量过多）
      </li>
      <li>
       当前细胞为死亡状态时，当周围有3个存活细胞时，该细胞变成存活状态。 （模拟繁殖）
      </li>
     </ol>
     <p>
      当前代所有细胞同时被以上规则处理后, 可以得到下一代细胞图。按规则继续处理这一代的细胞图，可以得到再下一代的细胞图，周而复始。
     </p>
     <p>
      例如假设初始是:(X代表活细胞，.代表死细胞)
      <br/>
      …
      <br/>
      …
      <br/>
      .XXX.
      <br/>
      …
     </p>
     <p>
      下一代会变为:
      <br/>
      …
      <br/>
      …X…
      <br/>
      …X…
      <br/>
      …X…
      <br/>
      …
     </p>
     <p>
      康威生命游戏中会出现一些有趣的模式。例如稳定不变的模式：
     </p>
     <p>
      …
      <br/>
      .XX.
      <br/>
      .XX.
      <br/>
      …
     </p>
     <p>
      还有会循环的模式：
     </p>
     <p>
      … … …
      <br/>
      .XX… .XX… .XX…
      <br/>
      .XX… .X… .XX…
      <br/>
      …XX. -&gt; …X. -&gt; …XX.
      <br/>
      …XX. …XX. …XX.
      <br/>
      … … …
     </p>
     <p>
      本题中我们要讨论的是一个非常特殊的模式，被称作"Gosper glider gun"：
     </p>
     <p>
      …
      <br/>
      …X…
      <br/>
      …X.X…
      <br/>
      …XX…XX…XX.
      <br/>
      …X…X…XX…XX.
      <br/>
      .XX…X…X…XX…
      <br/>
      .XX…X…X.XX…X.X…
      <br/>
      …X…X…X…
      <br/>
      …X…X…
      <br/>
      …XX…
      <br/>
      …
     </p>
     <p>
      假设以上初始状态是第0代，请问第1000000000(十亿)代一共有多少活着的细胞？
     </p>
     <p>
      注意：我们假定细胞机在无限的2D网格上推演，并非只有题目中画出的那点空间。
      <br/>
      当然，对于遥远的位置，其初始状态一概为死细胞。
     </p>
     <p>
      注意：需要提交的是一个整数，不要填写多余内容。
     </p>

     <pre class="has"><code>package 第三次模拟;
 
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

class Point{ //代表细胞

    public int x;    //行
    public int y;    //列

    public Point(int x,int y){
        this.x=x;
        this.y=y;
    }

    @Override
    public boolean equals(Object obj) {
        Point newPoint=(Point)obj;
        return this.x==newPoint.x &amp;&amp; this.y==newPoint.y;
    }

}

    static int row=11;
    static int rank=38;
    static List&lt;Integer&gt; saveCellNum=new ArrayList&lt;Integer&gt;();    //存放每一代的细胞数
    static List&lt;Point&gt; livingCell=new ArrayList&lt;Point&gt;();    //存放活细胞
    static List&lt;Point&gt; nextLivingCell;    //存放下一代的活细胞
    static int dirX[]={-1,-1,-1,0,0,1,1,1};    //八个方向，左上、上、右上、左、右、左下、下、右下
    static int dirY[]={-1,0,1,-1,1,-1,0,1};

    public static void main(String args[]){

        Scanner reader=new Scanner(System.in);
        char tmp[][]=new char[row][rank];
        for(int i=0;i&lt;row;i++){
            String str=reader.nextLine();
            tmp[i]=str.toCharArray();
        }
        //加入活细胞
        for(int i=0;i&lt;row;i++){
            for(int j=0;j&lt;rank;j++){
                if(tmp[i][j]=='X'){
                    livingCell.add(new Point(i,j));
                }
            }
        }
        saveCellNum.add( new Integer(livingCell.size()) );    //加入第0代活细胞数目

        for(int loop=1;loop&lt;=300;loop++){    //循环100次观察规律
            nextLivingCell=new ArrayList&lt;Point&gt;();
            for(int i=0;i&lt;livingCell.size();i++){    //遍历当前代的所有活细胞
                Point curPoint=(Point)livingCell.get(i);
                int nigorCells=0;    //周围八个细胞的存活数目
                for(int j=0;j&lt;8;j++){
                    int dx=curPoint.x+dirX[j];
                    int dy=curPoint.y+dirY[j];
                    if(livingCell.contains(new Point(dx,dy))){
                        nigorCells++;
                    }else{    //只有在活细胞旁边的死亡细胞才存在存活的可能
                        if(!nextLivingCell.contains(new Point(dx,dy))){    //需要注意，不同的活细胞周围可能存在同一个死亡细胞
                            int nigorhPoint=0;
                            for(int k=0;k&lt;8;k++){    //同样判断此白细胞
                                int dxx=dx+dirX[k];
                                int dyy=dy+dirY[k];
                                if(livingCell.contains(new Point(dxx,dyy))){
                                    nigorhPoint++;
                                }
                            }
                            if(nigorhPoint==3){    //死亡细胞复活
                                nextLivingCell.add(new Point(dx,dy));
                            }
                        }
                    }
                }
                if(nigorCells==2 || nigorCells==3){    //活细胞延续
                    nextLivingCell.add(curPoint);
                }
            }
            //至此上一代的已经更新完毕
            saveCellNum.add(new Integer(nextLivingCell.size()));
            livingCell=nextLivingCell;
        }
        for(int i=0;i&lt;saveCellNum.size();i++){
            System.out.print(saveCellNum.get(i)+" ");
        }
        System.out.println();
        for(int i=1;i&lt;saveCellNum.size();i++){
            System.out.print(saveCellNum.get(i)-saveCellNum.get(i-1)+" ");
        }
    }

}
