    <p>
     <strong>
      6.3 精通移动App应用服务程序开发
     </strong>
     <br/>
     熟悉轻开平台针对移动App（json格式）的规则及开发例子
     <br/>
     除了基本开发标签语言及表达式（参见同一下载包中的开发手册）外，轻开平台特别针对移动App最常用的json文本格式开发的对应规则：
     <br/>
      文件扩展名须为json（如：one.json）或用JspEasy扩展的jsp文件名
     <br/>
      文件内容须为闭合的json格式，静态内容则可直接书写json格式，如：
    </p>
    <pre class="prettyprint"><code class="hljs xml">{“item”:
    {
<span class="hljs-tag">&lt;<span class="hljs-title">chtml</span>&gt;</span>
… …
<span class="hljs-tag">&lt;/<span class="hljs-title">chtml</span>&gt;</span>
    },”other”:”other value”
}</code></pre>
    <p>
      value属性参数表示输出不带引号的值，如：
    </p>
    <pre class="prettyprint"><code class="hljs xml"><span class="hljs-tag">&lt;<span class="hljs-title">price</span> <span class="hljs-attribute">value</span>&gt;</span>@{item:price}<span class="hljs-tag">&lt;/<span class="hljs-title">price</span>&gt;</span></code></pre>
    <p>
     输出：”price”:51.24
    </p>
    <p>
      list属性参数表示输出为数组列表格式，end属性参数表示数组结束的位置（跟for的end值相等）
    </p>
    <pre class="prettyprint"><code class="hljs xml"><span class="hljs-tag">&lt;<span class="hljs-title">for</span> <span class="hljs-attribute">bags</span>=<span class="hljs-value">item</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">list</span>=<span class="hljs-value">"@{item:getSuffix}"</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span>&gt;</span>
    <span class="hljs-comment">&lt;!-- list属性参数表示输出为数组列表格式，end属性参数表示数组结束的位置（跟for的end值相等） --&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">name</span>&gt;</span>@{item:name}<span class="hljs-tag">&lt;/<span class="hljs-title">name</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">price</span> <span class="hljs-attribute">value</span>&gt;</span>@{item:price}<span class="hljs-tag">&lt;/<span class="hljs-title">price</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">unit</span>&gt;</span>@{item:unit}<span class="hljs-tag">&lt;/<span class="hljs-title">unit</span>&gt;</span>
    <span class="hljs-tag">&lt;/<span class="hljs-title">item</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-title">for</span>&gt;</span></code></pre>
    <p>
      nolast属性参数表示数组后边输出一个逗号
    </p>
    <pre class="prettyprint"><code class="hljs xml"><span class="hljs-tag">&lt;<span class="hljs-title">for</span> <span class="hljs-attribute">bags</span>=<span class="hljs-value">item</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">list</span>=<span class="hljs-value">"@{item:getSuffix}"</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span> <span class="hljs-attribute">nolast</span>&gt;</span>
    <span class="hljs-comment">&lt;!-- list属性参数表示输出为数组列表格式，end属性参数表示数组结束的位置（跟for的end值相等）, nolast属性参数表示数组后边输出一个逗号--&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">name</span>&gt;</span>@{item:name}<span class="hljs-tag">&lt;/<span class="hljs-title">name</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">price</span> <span class="hljs-attribute">value</span>&gt;</span>@{item:price}<span class="hljs-tag">&lt;/<span class="hljs-title">price</span>&gt;</span><span class="hljs-comment">&lt;!-- value属性参数表示输出不带引号的值 --&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">unit</span>&gt;</span>@{item:unit}<span class="hljs-tag">&lt;/<span class="hljs-title">unit</span>&gt;</span>
    <span class="hljs-tag">&lt;/<span class="hljs-title">item</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-title">for</span>&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-title">tatol</span>&gt;</span>@{item:getLength}<span class="hljs-tag">&lt;/<span class="hljs-title">tatol</span>&gt;</span><span class="hljs-comment">&lt;!-- 前边的nolast就是用于在这组前边输出一个逗号 --&gt;</span></code></pre>
    <p>
      nokey属性参数表示不输出数组名称
    </p>
    <pre class="prettyprint"><code class="hljs xml"><span class="hljs-tag">&lt;<span class="hljs-title">for</span> <span class="hljs-attribute">bags</span>=<span class="hljs-value">item</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-title">item</span> <span class="hljs-attribute">list</span>=<span class="hljs-value">"@{item:getSuffix}"</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span> <span class="hljs-attribute">nokey</span>&gt;</span>
    <span class="hljs-comment">&lt;!-- list属性参数表示输出为数组列表格式，end属性参数表示数组结束的位置（跟for的end值相等）, nokey属性参数表示不输出数组名称--&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">name</span>&gt;</span>@{item:name}<span class="hljs-tag">&lt;/<span class="hljs-title">name</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">price</span> <span class="hljs-attribute">value</span>&gt;</span>@{item:price}<span class="hljs-tag">&lt;/<span class="hljs-title">price</span>&gt;</span><span class="hljs-comment">&lt;!-- value属性参数表示输出不带引号的值 --&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">unit</span>&gt;</span>@{item:unit}<span class="hljs-tag">&lt;/<span class="hljs-title">unit</span>&gt;</span>
    <span class="hljs-tag">&lt;/<span class="hljs-title">item</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-title">for</span>&gt;</span></code></pre>
    <p>
      无名称数组直接累成字符串输出即可
    </p>
    <pre class="prettyprint"><code class="hljs xml"><span class="hljs-tag">&lt;<span class="hljs-title">chtml</span>&gt;</span>
<span class="hljs-comment">&lt;!-- 正式json格式的内容，无名称数组直接累成字符串输出即可 --&gt;</span>
<span class="hljs-tag">&lt;<span class="hljs-title">for</span> <span class="hljs-attribute">bags</span>=<span class="hljs-value">item</span> <span class="hljs-attribute">end</span>=<span class="hljs-value">"@{item:getLength}"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-title">if</span> <span class="hljs-attribute">x</span>=<span class="hljs-value">"@{item:getSuffix}"</span> <span class="hljs-attribute">else</span>=<span class="hljs-value">1</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">bag</span> <span class="hljs-attribute">id</span>=<span class="hljs-value">pPage</span> <span class="hljs-attribute">x</span>=<span class="hljs-value">0</span>&gt;</span><span class="hljs-tag">&lt;<span class="hljs-title">we</span> <span class="hljs-attribute">name</span>=<span class="hljs-value">name</span>&gt;</span>"@{item:name}"<span class="hljs-tag">&lt;/<span class="hljs-title">we</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-title">bag</span>&gt;</span>
        <span class="hljs-tag">&lt;<span class="hljs-title">bag</span> <span class="hljs-attribute">id</span>=<span class="hljs-value">pPage</span>&gt;</span><span class="hljs-tag">&lt;<span class="hljs-title">we</span> <span class="hljs-attribute">name</span>=<span class="hljs-value">name</span>&gt;</span>@{pPage:name},"@{item:name}"<span class="hljs-tag">&lt;/<span class="hljs-title">we</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-title">bag</span>&gt;</span>
    <span class="hljs-tag">&lt;/<span class="hljs-title">if</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-title">for</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-title">chtml</span>&gt;</span>[@{pPage:name}]</code></pre>
    <p>
      空数组直接输出：
    </p>
    <pre class="prettyprint"><code class="hljs json">[]</code></pre>
