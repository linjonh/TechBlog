    <h2>
     <a id="ESP12S_32">
     </a>
     三、ESP12-S配网
    </h2>
    <h3>
     <a id="1GAgent_33">
     </a>
     1、烧写机智云GAgent固件
    </h3>
    <ol>
     <li>
      到机智云平台下载烧写固件、烧写工具、串口助手
     </li>
     <li>
      使用USB-TTL烧写器按要求连接ESP12-S，使用烧写软件烧写
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d07b9d009d1212c3dfc208e6ea7bf71d.png"/>
      <br/>
      烧写时，
      <strong>
       要短接JP1两个引脚
      </strong>
      ，即把GPIO0与地短接，把WiFi芯片切换到
      <mark>
       烧写模式
      </mark>
      <br/>
      烧写成功效果图
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4886d1a225ef53b45ca0077cef3f2178.png"/>
     </li>
    </ol>
    <h3>
     <a id="2WiFi_39">
     </a>
     2、使用机智云串口助手配置WiFi
    </h3>
    <ol>
     <li>
      烧写成功后断开GPIO0和GND，重新上电；
     </li>
     <li>
      打开机智云串口调试助手连接WiFi模块；
     </li>
     <li>
      把WiFi芯片设置为softAP配网模式，使用机智云APP配网
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5874bfb9b94418e856b8e28f6a7c8937.png"/>
     </li>
     <li>
      把机智云平台创建的项目product_key和Product Secret复制过来让本地硬件与机智云平台进行连接
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7fc2c03a5832bacdbf61fe09bff4df8a.png"/>
     </li>
     <li>
      在机智云平台能看到设备在线，说明配网成功
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1d6e1dcd31e23bf132373353626e0b04.png"/>
     </li>
    </ol>
    <h3>
     <a id="3ArduinoESP12S_49">
     </a>
     3、把Arduino与ESP-12S连接
    </h3>
    <ol>
     <li>
      连接原理图：
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d802c66d4cf00699d2ff842a4899446e.png"/>
     </li>
     <li>
      实物效果图
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9e565c24bca93faed72a29822a2f0de1.png"/>
      <br/>
      <strong>
       烧写Arduino程序，把ESP芯片拔掉
      </strong>
      ，因为Arduino只有一个串口可以使用，两个同时使用会出现冲突。
     </li>
    </ol>
