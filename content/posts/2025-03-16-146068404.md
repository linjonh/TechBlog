    <h3 name="1.%20%E5%BC%95%E8%A8%80">
     <span style="color:#ed7976">
      1. 引言
     </span>
    </h3>
    <p>
     在 Flutter 中，
     <code>
      OutlinedButton
     </code>
     是一种带
     <span style="color:#fe2c24">
      有边框但无背景色的按钮
     </span>
     ，适用于
     <span style="color:#fe2c24">
      强调次要操作
     </span>
     。它相比
     <code>
      ElevatedButton
     </code>
     少了背景色，相比
     <code>
      TextButton
     </code>
     多了一个边框，适用于不希望 UI 过于突出的场景，如“取消”按钮或次要操作按钮。
    </p>
    <h3 id="2.%20OutlinedButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95" name="2.%20OutlinedButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95">
     <span style="color:#ed7976">
      2. OutlinedButton 的基本用法
     </span>
    </h3>
    <p>
     <code>
      OutlinedButton
     </code>
     需要
     <code>
      onPressed
     </code>
     事件和
     <code>
      child
     </code>
     组件。
    </p>
    <pre><code class="language-Dart">OutlinedButton(
  onPressed: () {
    print('OutlinedButton 被点击');
  },
  child: Text('取消'),
)
</code></pre>
    <p>
     如果
     <code>
      onPressed
     </code>
     设为
     <code>
      null
     </code>
     ，按钮会变为不可点击状态。
    </p>
    <pre><code class="language-Dart">OutlinedButton(
  onPressed: null,
  child: Text('不可点击'),
)
</code></pre>

      4. 自定义按钮样式
     </span>

    <h4 id="4.1%20%E4%BF%AE%E6%94%B9%E8%BE%B9%E6%A1%86%E9%A2%9C%E8%89%B2%E5%92%8C%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2" name="4.1%20%E4%BF%AE%E6%94%B9%E8%BE%B9%E6%A1%86%E9%A2%9C%E8%89%B2%E5%92%8C%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2">
     <span style="color:#956fe7">
      4.1 修改边框颜色和文本颜色
     </span>
    </h4>
    <pre><code class="language-Dart">OutlinedButton(
  style: OutlinedButton.styleFrom(
    primary: Colors.blue, // 文字颜色
    side: BorderSide(color: Colors.blue, width: 2), // 边框颜色
  ),
  onPressed: () {},
  child: Text('自定义边框颜色'),
)
</code></pre>

    <h3 id="5.%20%E7%BB%93%E8%AE%BA" name="5.%20%E7%BB%93%E8%AE%BA">
     <span style="color:#ed7976">
      5. 结论
     </span>
    </h3>
    <p>
     <code>
      OutlinedButton
     </code>
     适用于需要按钮但不希望其过于突出的场景。通过
     <code>
      style
     </code>
     属性可以自定义颜色、边框、形状等。同时也要遵循一些
     <strong>
      重要设计原则：
     </strong>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        视觉一致性
       </strong>
       ：边框颜色应与文本颜色协调
      </p>
     </li>
     <li>
      <p>
       <strong>
        对比度保障
       </strong>
       ：禁用状态需要保持至少 3:1 的对比度
      </p>
     </li>
     <li>
      <p>
       <strong>
        平台适配
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         Material Design：推荐使用 1px 边框
        </p>
       </li>
       <li>
        <p>
         iOS 风格：推荐使用 0.8px 边框 + 圆角半径 10.0
        </p>
       </li>
      </ul>
     </li>
    </ol>
