    <p>
     卷积神经网络（CNN, Convolutional Neural Networks）是深度学习中最重要的模型之一，广泛应用于
     <strong>
      计算机视觉
     </strong>
     、
     <strong>
      目标检测
     </strong>
     、
     <strong>
      语义分割
     </strong>
     等任务。自 LeNet 诞生以来，CNN 结构经历了多个重要发展阶段，出现了许多
     <strong>
      经典架构
     </strong>
     ，包括 AlexNet、VGG、GoogLeNet（Inception）、ResNet、DenseNet、MobileNet 等。
    </p>
    <p>
     本文将详细介绍 CNN 的主要架构及其核心思想，帮助大家深入理解 CNN 发展历程及各架构的特点。
    </p>

    <h3>
     <a id="5_ResNet2015__102">
     </a>
     <strong>
      5. ResNet（2015）—— 解决深度退化问题
     </strong>
    </h3>
    <p>
     <strong>
      提出者
     </strong>
     ：Kaiming He, Microsoft Research
     <br/>
     <strong>
      贡献
     </strong>
     ：
    </p>
    <ul>
     <li>
      提出
      <strong>
       残差学习（Residual Learning）
      </strong>
      ，通过
      <strong>
       跳跃连接（Shortcut Connection）
      </strong>
      解决梯度消失问题，使 CNN 深度突破 100 层。
     </li>
     <li>
      ResNet-50, ResNet-101, ResNet-152 等版本成为主流。
     </li>
    </ul>
    <p>
     <strong>
      残差块（Residual Block）：
     </strong>
     <br/>
     <span class="katex--display">
      <span class="katex-display">

        <span class="katex-html">
         <span class="base">

          </span>
          <span class="mord mathnormal" style="margin-right: 0.0359em;">

          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>

           =
          </span>
          <span class="mspace" style="margin-right: 0.2778em;">
          </span>
         </span>
         <span class="base">

          <span class="mopen">

          <span class="mord mathnormal">
           x
          </span>
          <span class="mclose">
           )
          </span>
          <span class="mspace" style="margin-right: 0.2222em;">
          </span>
          <span class="mbin">

          </span>
          <span class="mspace" style="margin-right: 0.2222em;">

     </span>

     其中：
    </p>
    <ul>

     </strong>

      。

    <h3>
     <a id="6_MobileNet2017__CNN_121">
     </a>
     <strong>
      6. MobileNet（2017）—— 轻量级 CNN
     </strong>
    </h3>
    <p>
     <strong>
      提出者
     </strong>
     ：Google
     <br/>
     <strong>
      贡献
     </strong>
     ：
    </p>
    <ul>
     <li>
      采用
      <strong>
       深度可分离卷积（Depthwise Separable Convolution）
      </strong>
      ，减少计算量和参数量，使 CNN 可用于移动端。
     </li>
     <li>
      可用于实时应用，如手机端人脸识别。
     </li>
    </ul>
    <p>
     <strong>
      结构：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       标准卷积
      </strong>
      替换为
      <strong>
       深度可分离卷积
      </strong>
     </li>
     <li>
      <strong>
       1×1 卷积降维
      </strong>
      ，减少参数量
     </li>
     <li>
      提供 MobileNetV1, V2, V3 版本，逐步优化
     </li>
    </ul>
    <p>
     <strong>
      特点：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       计算量减少
      </strong>
      90% 以上，适合移动设备。
     </li>
     <li>
      <strong>
       轻量级
      </strong>
      ，适用于嵌入式 AI 任务（如自动驾驶）。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="7_EfficientNet2019__CNN_138">
     </a>
     <strong>
      7. EfficientNet（2019）—— 自动搜索优化 CNN
     </strong>
    </h3>
    <p>
     <strong>
      提出者
     </strong>
     ：Google Brain
     <br/>
     <strong>
      贡献
     </strong>
     ：
    </p>
    <ul>
     <li>
      提出了
      <strong>
       复合缩放（Compound Scaling）
      </strong>
      方法，统一调整 CNN 的
      <strong>
       深度、宽度和分辨率
      </strong>
      ，提高计算效率。
     </li>
     <li>
      采用
      <strong>
       MobileNetV2
      </strong>
      的深度可分离卷积和
      <strong>
       Swish 激活函数
      </strong>
      ，减少计算量。
     </li>
     <li>
      在 ImageNet 上比 ResNet、DenseNet 等模型更高效。
     </li>
    </ul>
    <p>
     <strong>
      结构：
     </strong>
    </p>
    <ul>
     <li>
      使用
      <strong>
       MBConv
      </strong>
      （MobileNetV2 提出的 Inverted Residual Block）。
     </li>
     <li>
      通过
      <strong>
       神经架构搜索（NAS）
      </strong>
      自动优化 CNN 结构。
     </li>
    </ul>
    <p>
     <strong>
      特点：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       参数更少，性能更强
      </strong>
      （EfficientNet-B7 在 ImageNet 上超越 ResNet152）。
     </li>
     <li>
      <strong>
       可扩展
      </strong>
      ：EfficientNet-B0 到 B7 提供不同计算复杂度的版本。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="8_RegNet2020__CNN_155">
     </a>
     <strong>
      8. RegNet（2020）—— 通过规则生成 CNN
     </strong>
    </h3>
    <p>
     <strong>
      提出者
     </strong>
     ：Facebook AI
     <br/>
     <strong>
      贡献
     </strong>
     ：
    </p>
    <ul>
     <li>
      通过系统化搜索找到最佳的 CNN 设计规则，而非单独优化某个模型。
     </li>
     <li>
      提出了
      <strong>
       可预测的 CNN 结构
      </strong>
      ，在计算资源受限的情况下仍然保持高性能。
     </li>
    </ul>
    <p>
     <strong>
      特点：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       比 ResNet 更高效
      </strong>
      ，在 ImageNet 上实现更优性能。
     </li>
     <li>
      <strong>
       提供多个变体
      </strong>
      （RegNetX, RegNetY），可适应不同计算需求。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="9_ConvNeXt2022_CNN__Transformer_167">
     </a>
     <strong>
      9. ConvNeXt（2022）—— CNN 重新对标 Transformer
     </strong>
    </h3>
    <p>
     <strong>
      提出者
     </strong>
     ：Facebook AI
     <br/>
     <strong>
      贡献
     </strong>
     ：
    </p>
    <ul>
     <li>
      受 Vision Transformer（ViT） 启发，对 CNN 进行结构优化，使其在 ImageNet 上与 ViT 竞争。
     </li>
     <li>
      通过简化 ResNet 设计，提升 CNN 在现代硬件上的性能。
     </li>
    </ul>
    <p>
     <strong>
      特点：
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       改进的残差结构
      </strong>
      （ResNet 变体）。
     </li>
     <li>
      <strong>
       大核卷积（7×7）
      </strong>
      提升感受野。
     </li>
     <li>
      <strong>
       层归一化（LayerNorm）
      </strong>
      替代 BatchNorm，提高训练稳定性。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="CNN__180">
     </a>
     <strong>
      CNN 的未来趋势
     </strong>
    </h3>
    <p>
     虽然近年来
     <strong>
      Vision Transformer（ViT）
     </strong>
     和
     <strong>
      MLP 结构
     </strong>
     （如 MLP-Mixer）取得了较大突破，但 CNN 仍然是计算机视觉领域的核心方法。未来 CNN 可能会：
    </p>
    <ol>
     <li>
      <strong>
       与 Transformer 结合
      </strong>
      （如 Swin Transformer + CNN）。
     </li>
     <li>
      <strong>
       进一步优化轻量级架构
      </strong>
      （如 MobileNetV3, EfficientNetLite）。
     </li>
     <li>
      <strong>
       增强可解释性
      </strong>
      ，提升透明度和泛化能力。
     </li>
    </ol>
