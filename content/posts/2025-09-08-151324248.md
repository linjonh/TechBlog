# 【Ubuntu20.04 + VS code 1.103.2 最新版，中文输入法失效】



### 问题：

Ubuntu20.04 + VS code 1.103.2 最新版，出现不能切换中英文输入法，且在vscode编辑时中文输入失效的问题。

### 原因：

这是vs code版本太新，与ubuntu 20.04系统不适配带来的问题。

vs code1.103.2已经加入了Copilot。

### 解决办法：

安装之前的vscode版本

[VS CODE下载](https://code.visualstudio.com/)

我下载的是1.101.0版本，回退后，中文输入法就正常了。

https://update.code.visualstudio.com/1.101.0/linux-snap-x64/stable

回退前先在VScode设置里面关闭自动更新vscode。

使用微软官方下载的vscode安装包安装时需要在安装命令中加入 “–dangerous” 字段，否则会报错。

报错信息：

```bash
错误：cannot find signatures with metadata for snap/component "code-stable-x64-1749655393.snap"

```

这通常是因为你尝试安装的 Snap 包不是从官方 Snap 商店下载的。

安装微软下载的vs code包的命令：

```bash
sudo snap install --dangerous code-stable-x64-1749655393.snap

```
