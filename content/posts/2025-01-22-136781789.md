Git 是一个免费的开源分布式版本控制系统，旨在处理从小到具有速度和效率的超大型项目。

Git 易于学习，占用空间小，性能快如闪电。 它优于 Subversion、CVS、Perforce 和 ClearCase 等 SCM 工具 具有廉价本地分支等功能， 方便的暂存区域和多个工作流程。

### 1  下载Git

1. 首先我们在浏览器中通过下方链接搜索Git，进入到Git的官方下载界面。

[Git](https://git-scm.com/ "Git")
![](https://i-blog.csdnimg.cn/blog_migrate/093de8d5cd700849bd51ed73a22b3c3e.png)

2. 点击Downloads。

![](https://i-blog.csdnimg.cn/blog_migrate/61b2fbb9712bacfa00b851ad76ddde74.png)

3. 进入后选择Windows版本下载。

![](https://i-blog.csdnimg.cn/blog_migrate/cbfc1c8fbad140f54be669aca6ead1fe.png)

4. 提示如何下载，按如图步骤进行。

![](https://i-blog.csdnimg.cn/blog_migrate/bf2eb6b5a1d31f7af8d1069d607239a6.png)

5. 下载完成，文件夹中找到安装程序。

### 

### 2  安装Git

1. 双击文件夹中的Git软件包安装。

![](https://i-blog.csdnimg.cn/blog_migrate/82a4442b82bc03cdbb00f4708de13734.png)

2. 进入安装信息显示界面，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/2043485f36596c587ec39b35d12391c3.png)

3. 选择安装路径，我选择默认安装，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/b3cda1460ad1a6ce728e3a566de6a86a.png)

4. 选择组件，默认即可，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/f1863633c171c401672647436eb05cb0.png)

5. 选中开始菜单文件夹，默认即可，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/dad89af3fb24c1299db3637b25d16e3f.png)

6. 选择Git使用的默认编辑器，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/2fd9bc34c6a55f7bc3e1f52d3dcaddff.png)

7. 进入调整新库中初始分支的名称界面，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/fc2ec8702ec839cb7ce17e791a7c4e6a.png)

8. 调整path环境，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/5feb44e44f12fe7d0adb07c1a11f04d0.png)

9. 选择SSH可执行文件，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/08a3996e860f2940b664aa0da71e2f4e.png)

10. 选择HTTPS传输后端，即你希望Git为HTTPS连接使用哪个SSL/TLS库？默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/8f6abd5f4c851d7f1b3c393f6865d1d5.png)

11. 设置Git应该如何处理文本文件的行尾，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/bab30d5a42bbcf3ca8b5400aade76ca1.png)

12. 配置于Git Bash一起使用的终端仿真器，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/ca4025baa7ef06436b82caccb7409170.png)

13. 选择git pull的默认行为，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/53e9d20fb6a5a973fdcbc446d01fa1d7.png)

14. 配置凭证，即得到管理凭证，比如登录的账号名、密码等，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/166975fbb88382de71c8d4d651b800a2.png)

15. 配置额外选项，默认选择，点击“Next”。

![](https://i-blog.csdnimg.cn/blog_migrate/846ed86f328e0b7e7958776a19738cfc.png)

16. 配置实验选项，默认选择，点击“Install”。

![](https://i-blog.csdnimg.cn/blog_migrate/eb0dea921dcc0d0597f0e67612226c94.png)

17. 等待安装。

![](https://i-blog.csdnimg.cn/blog_migrate/bc36a8daab904818ee27519a99518bde.png)

18. 安装完成，点击“Finish”。

![](https://i-blog.csdnimg.cn/blog_migrate/4f756501113f4104519a81f3740428c5.png)

### 3  查看、启动Git Bash

在电脑桌面上鼠标右击，可以看到Open Git GUI here和Open Git Bash here，即为Git安装成功。

![](https://i-blog.csdnimg.cn/blog_migrate/8c67e7f14dc81f901ff282c43b960531.png)

右击后点击Open Git Bash here打开，如图为Git Bash界面。

![](https://i-blog.csdnimg.cn/blog_migrate/2c7185ca032f4585b1bc811aa940192b.png)
