    <h3>
     <a id="wxshowToast_2">
     </a>
     wx.showToast()
    </h3>
    <p>
     代码:
     <br/>
     wx.showToast({
     <!-- -->
     <br/>
     title: ‘提示信息’,
     <br/>
     })
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5b7bbf94b2387b87380552ed140eac2f.png">
      <br/>
      参数:
      <br/>
      title：显示的提示信息，在没有图标的情况下，文本内容可显示两行
      <br/>
      icon: 显示的图标
      <br/>
      success：成功图标
      <br/>
      loading：加载图标
      <br/>
      none：没有图标
      <br/>
      image：自定义显示的图标，优先级高于icon
      <br/>
      duration：延迟的时间，弹出框弹出后的显示时间
      <br/>
      mask：true/false是否显示遮罩层
      <br/>
      success：接口调用成功的回调函数
      <br/>
      fail：接口调用失败的回调函数
      <br/>
      complete：不管成功还是失败都会执行的函数
     </img>
    </p>
    <h3>
     <a id="wxshowLoading_21">

     wx.showLoading()
    </h3>
    <p>
     代码:
     <br/>
     wx.showLoading({
     <!-- -->
     <br/>
     title: ‘玩命加载中。。。’,
     <br/>
     })
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/553d45dea4ee5573385edf0d6e85e232.png">
      <br/>
      参数:
      <br/>
      title：加载的提示信息eg：玩命加载中…
      <br/>
      mask：是否现思遮罩层
      <br/>
      success：接口调用成功的回调函数
      <br/>
      fail：接口调用失败的回调函数
      <br/>
      complete：无论成功还是失败都会执行的函数
     </img>
    </p>
    <h3>
     <a id="wxshowModal_36">
     </a>
     wx.showModal()
    </h3>
    <p>
     代码:
     <br/>
     wx.showModal({
     <!-- -->
     <br/>
     title: ‘提示’,
     <br/>
     content: ‘这是一个模态弹窗’,
     <br/>
     success: function(res){
     <!-- -->
     <br/>
     if (res.confirm) {
     <!-- -->
     <br/>
     console.log(‘用户点击确定’)
     <br/>
     } else if (res.cancel) {
     <!-- -->
     <br/>
     console.log(‘用户点击取消’)
     <br/>
     }
     <br/>
     }
     <br/>
     })
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5a88d016ebc84ab3ba2f7b7077091bb2.png">
      <br/>
      参数:
      <br/>
      title：提示信息的标题
      <br/>
      content：提示信息的内容
      <br/>
      showCancel：true/false是否显示取消按钮
      <br/>
      cancelText：取消按钮的文本内容，不得超过四个字符
      <br/>
      cancelColor：取消按钮的文本颜色，默认#000000
      <br/>
      confirmText：确认按钮的文本内容，不得超过四个字符
      <br/>
      confirmColor：却惹按钮的文本颜色，默认#000000
      <br/>
      success：接口成功的回调
      <br/>
      fail：接口失败的回调
      <br/>
      complete：无论成功或失败都会调用
     </img>
    </p>
