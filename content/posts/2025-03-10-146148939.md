    <h2>
     <a id="Ubuntu_2204__Gradle__0">
     </a>
     Ubuntu 22.04 官方下载安装 Gradle 记录
    </h2>
    <p>
     Gradle 是一个强大的自动化构建工具，广泛用于 Java、Android 等项目的构建中。下面详细介绍如何在 Ubuntu 22.04 中使用官网下载安装 Gradle。
    </p>
    <hr/>
    <h3>
     <a id="_6">
     </a>
     一、准备工作
    </h3>
    <p>
     首先，确保你的系统已安装 Java JDK（推荐 Java 17 或以上版本）。使用以下命令检查 Java 是否安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
    <p>
     若未安装，可执行以下命令进行安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> update
<span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> openjdk-17-jdk
</code></pre>
    <h3>
     <a id="_Gradle_21">
     </a>
     二、官网下载 Gradle
    </h3>
    <p>
     访问 Gradle 官网的下载页面获取最新版 Gradle 地址：
     <a href="https://gradle.org/releases/" rel="nofollow">
      Gradle 官方下载页
     </a>
     。
    </p>
    <p>
     可以使用
     <code>
      wget
     </code>
     命令直接下载，例如下载 Gradle 8.7（示例版本，请以官网实际版本为准）：
    </p>
    <pre><code class="prism language-bash"><span class="token function">wget</span> https://services.gradle.org/distributions/gradle-8.7-bin.zip
</code></pre>
    <p>
     如果你使用了网络代理，可使用如下命令通过代理下载（以HTTP代理为例）：
    </p>
    <pre><code class="prism language-bash"><span class="token function">wget</span> <span class="token parameter variable">-e</span> <span class="token string">"http_proxy=http://your_proxy_ip:port"</span> https://services.gradle.org/distributions/gradle-8.7-bin.zip
</code></pre>
    <h3>
     <a id="_Gradle_37">
     </a>
     三、安装 Gradle
    </h3>
    <h4>
     <a id="_optgradle_39">
     </a>
     解压到
     <code>
      /opt/gradle
     </code>
    </h4>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">unzip</span> <span class="token parameter variable">-d</span> /opt/gradle gradle-8.7-bin.zip
</code></pre>
    <h4>
     <a id="_45">
     </a>
     创建符号链接（可选）
    </h4>
    <p>
     为便于版本管理，建议建立一个符号链接：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">ln</span> <span class="token parameter variable">-s</span> /opt/gradle/gradle-8.7 /opt/gradle/latest
</code></pre>
    <h3>
     <a id="_53">
     </a>
     四、配置环境变量
    </h3>
    <p>
     编辑用户环境变量文件：
    </p>
    <pre><code class="prism language-bash"><span class="token function">nano</span> ~/.bashrc
</code></pre>
    <p>
     在文件底部添加以下内容：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">GRADLE_HOME</span><span class="token operator">=</span>/opt/gradle/latest
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token variable">${GRADLE_HOME}</span>/bin:<span class="token variable">${<!-- --><span class="token environment constant">PATH</span>}</span>
</code></pre>
    <p>
     保存文件后，使环境变量生效：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.bashrc
</code></pre>
    <h3>
     <a id="_74">
     </a>
     五、验证安装
    </h3>
    <p>
     运行以下命令验证 Gradle 是否成功安装：
    </p>
    <pre><code class="prism language-bash">gradle <span class="token parameter variable">-v</span>
</code></pre>
    <p>
     如果安装成功，将显示类似以下版本信息：
    </p>
    <pre><code>------------------------------------------------------------
Gradle 8.7
------------------------------------------------------------

Build time:   2024-02-09 17:52:22 UTC
Revision:     bbf87bf29dfdbb21c431cb7c9de7349c5226d61c

Kotlin:       1.9.22
Groovy:       3.0.20
Ant:          Apache Ant(TM) version 1.10.13 compiled on January 4 2023
JVM:          17.0.10 (Ubuntu 17.0.10+7-post-Ubuntu-0ubuntu122.04)
OS:           Linux 5.15.0-94-generic amd64
</code></pre>
    <p>
     至此，Gradle 已经成功安装并配置完毕，可用于各类构建任务。
    </p>
