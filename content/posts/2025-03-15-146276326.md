    <h4>
     <a id="1__0">
     </a>
     1.
     <strong>
      安装所有依赖包
     </strong>
    </h4>
    <p>
     在终端运行以下命令（如果使用 GPU，请确保已安装 CUDA 驱动）：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 安装 Flask 和其他依赖</span>
pip <span class="token function">install</span> flask sentence-transformers torch

<span class="token comment"># 如果是 GPU 环境，推荐安装 CUDA 版本的 PyTorch（根据你的 CUDA 版本调整）</span>
pip <span class="token function">install</span> torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
</code></pre>
    <hr/>
    <h4>
     <a id="2__12">
     </a>
     2.
     <strong>
      验证安装
     </strong>
    </h4>
    <pre><code class="prism language-bash">python3 <span class="token parameter variable">-c</span> <span class="token string">"import flask; print('Flask 已安装')"</span>
python3 <span class="token parameter variable">-c</span> <span class="token string">"import torch; print(f'PyTorch 版本: {torch.__version__}, 是否使用GPU: {torch.cuda.is_available()}')"</span>
</code></pre>
    <hr/>
    <h4>
     <a id="3__20">
     </a>
     3.
     <strong>
      使用虚拟环境（推荐）
     </strong>
    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># 创建并激活虚拟环境</span>
python3 <span class="token parameter variable">-m</span> venv myenv
<span class="token builtin class-name">source</span> myenv/bin/activate

<span class="token comment"># 安装依赖</span>
pip <span class="token function">install</span> flask sentence-transformers torch

<span class="token comment"># 运行脚本</span>
python3 javaRunGpu.py <span class="token parameter variable">--api</span>  <span class="token comment"># 或命令行模式</span>
</code></pre>
    <hr/>
    <h4>
     <a id="4__35">

     4.

      重新运行脚本

    </h4>
    <pre><code class="prism language-bash"><span class="token comment"># API 模式</span>
python3 javaRunGpu.py <span class="token parameter variable">--api</span>

<span class="token comment"># CLI 模式</span>
python3 javaRunGpu.py <span class="token string">"需要向量化的文本"</span>
</code></pre>
