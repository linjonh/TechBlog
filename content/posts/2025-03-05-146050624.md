    <h4>
     1. 确保启用了 Universe 仓库
    </h4>
    <p>
     在某些情况下，
     <code>
      python2-minimal
     </code>
     包可能位于 Universe 仓库中。你可以通过以下命令启用 Universe 仓库并更新软件包列表：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>sudo add-apt-repository universe
sudo apt update</code></pre>
    <p>
     然后尝试安装：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>sudo apt install python2-minimal</code></pre>
    <p>
     如果仍然无法安装，可以尝试安装
     <code>
      python2
     </code>
     或
     <code>
      python2.7
     </code>
     。
    </p>
    <h4>
     2. 安装 Python 2.7
    </h4>
    <p>
     在较新的 Ubuntu 版本中（如 Ubuntu 20.04 及以上），
     <code>
      python2-minimal
     </code>
     包可能已被移除，但你可以安装
     <code>
      python2.7
     </code>
     ：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>sudo apt update
sudo apt install python2.7</code></pre>
    <h4>
     3、使用
     <code>
      update-alternatives
     </code>
     配置 Python
    </h4>
    <p>
     如果你需要将 Python 2 设置为默认的 Python 解释器，可以使用
     <code>
      update-alternatives
     </code>
     ：
    </p>
    <pre><code>sudo update-alternatives --install /usr/bin/python python /usr/bin/python2.7 1</code></pre>
    <p>
     这样，当你运行
     <code>
      python
     </code>
     命令时，它将指向 Python 2.7。
    </p>
    <p>
     <strong>
      设置 Python 3 为默认解释器
     </strong>
     ： 假设你的 Python 3 的路径是
     <code>
      /usr/bin/python3
     </code>
     ，运行以下命令：
    </p>
    <p>
     bash复制
    </p>
    <pre><code>sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 1</code></pre>
    <p>
     如果运行没有生效，可以手动替换添加。
    </p>
    <pre><code class="hljs">sudo update-alternatives --config python</code></pre>
