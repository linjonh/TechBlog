    <h2>
     1.排序的概念及其运用
    </h2>
    <h3>
     1.1排序的概念
    </h3>
    <p>
     排序：所谓排序，就是使一串记录，按照其中的某个或某些关键字的大小，递增或递减的排列起来的操作。
    </p>
    <p>
     稳定性：假定在待排序的记录序列中，存在多个具有相同的关键字的记录，若经过排序，这些记录的相对次 序保持不变，即在原序列中，r[i]=r[j]，且r[i]在r[j]之前，而在排序后的序列中，r[i]仍在r[j]之前，则称这种排 序算法是稳定的；否则称为不稳定的。
    </p>
    <p>
     内部排序：数据元素全部放在内存中的排序。
    </p>
    <p>
     外部排序：数据元素太多不能同时放在内存中，根据排序过程的要求不能在内外存之间移动数据的排序。
    </p>

      2、常见排序算法思想

    <h3>
     <strong>
      2.1 插入排序
     </strong>
    </h3>
    <h4>
     2.11 基本思想：
    </h4>
    <p>
     把待排序的记录按其关键码值的大小逐个插入到一个已经排好序的有序序列中，直到所有的   记录插入完为止，得到一个新的有序序列 。当我们排序扑克牌是就是用的这种思想。
    </p>
    <p>
     <img alt="" height="276" src="https://i-blog.csdnimg.cn/direct/f09fdc2673374a809cc85ce5892922e9.png" width="435"/>
    </p>
    <h4>
     2.12 直接插入排序
    </h4>
    <p>
     当插入第i(i&gt;=1)个元素时，前面的array[0],array[1],…,array[i-1]已经排好序，此时用array[i]的排序码与 array[i-1],array[i-2],…的排序码顺序进行比较，找到插入位置即将array[i]插入，原来位置上的元素顺序后移。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/76c6588e1d0c49acafa9d5674ebff89d.png"/>
    </p>
    <p style="text-align:center">
    </p>
    <p>
     代码演示：
    </p>
    <p>
    </p>
    <pre><code class="language-cpp">void InsertSort(int* a, int n)
{
	for (int i = 0;i &lt; n - 1;i++)
	{
		int tmp = a[i + 1];
		int end = i;
		while (end &gt;= 0)
		{
			if (a[end] &gt; tmp)
			{
				a[end + 1] = a[end];
				end--;
			}
			else
			{
				break;
			}
		}
		a[end + 1] = tmp;
	}
}</code></pre>
    <p>
     <span style="color:#fe2c24">
      注：插入排序一般都是在原数组上进行操作，不用再创建一个数组 。可以结合代码和上图一同看。
     </span>
    </p>
    <p>
     直接插入排序的特性总结：
    </p>
    <p>
     1. 元素集合越接近有序，直接插入排序算法的时间效率越高
    </p>
    <p>
     2. 时间复杂度：O(N^2)
    </p>
    <p>
     3. 空间复杂度：O(1)，它是一种稳定的排序算法
    </p>
    <p>
     4. 稳定性：稳定
    </p>
    <h4>
     2.1.3 希尔排序( 缩小增量排序 )
    </h4>
    <p>
     希尔排序法又称缩小增量法。希尔排序法的基本思想是：先选定一个整数，把待排序文件中所有记录分成个 组，所有距离为的记录分在同一组内，并对每一组内的记录进行排序。然后，取，重复上述分组和排序的工 作。当到达=1时，所有记录在统一组内排好序。
    </p>
    <p>
     <img alt="" height="597" src="https://i-blog.csdnimg.cn/direct/dcf2658f6a9b4fa896df48e4ce2143d6.png" width="798"/>
    </p>
    <p>
     简单来说希尔排序就是把一个数组变得比较有序，然后再进行插入排序。因为通过简单的计算我们能发现，插入排序在有序的情况下时间复杂度是O（N）。
    </p>
    <p>
     代码演示：
    </p>
    <pre><code class="language-cpp">void ShellSort(int* a, int n)
{
	int gap = n;
	while (gap &gt; 1)
	{
		gap /= 2;

		for (int i = 0;i &lt; n - gap;i++)
		{
			int tmp = a[i + gap];
			int end = i;
			while (end &gt;= 0)
			{
				if (a[end] &gt; tmp)
				{
					a[end + gap] = a[end];
					end = end - gap;
				}
				else
				{
					break;
				}
			}
			a[end + gap] = tmp;
		}

	}
}</code></pre>
    <p>
     注：如果是一趟一趟看的话建议可以多加一层循环，时间复杂度还是不变的。这里gap就相当于前面插入排序的1，只不过这里跳的步数大。
    </p>
    <p>
     1. 希尔排序是对直接插入排序的优化。
    </p>
    <p>
     2. 当gap &gt; 1时都是预排序，目的是让数组更接近于有序。当gap == 1时，数组已经接近有序的了，这样就 会很快。这样整体而言，可以达到优化的效果。我们实现后可以进行性能测试的对比。
    </p>
    <p>
     3. 希尔排序的时间复杂度不好计算，因为gap的取值方法很多，导致很难去计算，因此在好些树中给出的 希尔排序的时间复杂度都不固定。
    </p>
    <p>
     4. 稳定性：不稳定
    </p>
    <h3>
     2.2  选择排序
    </h3>
    <h4>
     2.2.1基本思想：
    </h4>
    <p>
     每一次从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置，直到全部待排序的 数据元素排完 。
    </p>
    <h4>
     2.2.2 直接选择排序:
    </h4>
    <p>
     在元素集合array[i]--array[n-1]中选择关键码最大(小)的数据元素 若它不是这组元素中的最后一个(第一个)元素，则将它与这组元素中的最后一个（第一个）元素交换 在剩余的array[i]--array[n-2]（array[i+1]--array[n-1]）集合中，重复上述步骤，直到集合剩余1个元素
    </p>
    <p>
     <img alt="" height="620" src="https://i-blog.csdnimg.cn/direct/a31ba8e595c348878f420d1d47d694ca.png" width="1102"/>
    </p>
    <p>
     代码演示：
    </p>
    <pre><code class="language-cpp">void swap(int* p1, int* p2)
{
	int tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}
void SelectSort(int* a, int n)
{
	int left = 0;
	int right = n - 1;
	int max = 0;
	int min = 0;
	while (left &lt; right)
	{
		max = left;
		min = right;
		for (int i = left;i &lt;= right;i++)
		{
			if (a[i] &gt; a[max])
			{
				max = i;
			}
			if (a[i] &lt; a[min])
			{
				min = i;
			}
		}
		swap(&amp;a[left], &amp;a[min]);
		if (left == max)
		{
			max = min;
		}
		swap(&amp;a[right], &amp;a[max]);

		left++;
		right--;
	}
}</code></pre>
    <p>
     注：代码中的swap函数为交换函数 ，当左指针指向的值为max时，我们需要加一个把max的值重新换回来的判断，否则会将最大值与最小值交换，那么最大值的指针指向的就不是最大值了。反之同理。
    </p>
    <p>
    </p>
    <p>
     直接选择排序的特性总结：
    </p>
    <p>
     1. 直接选择排序思考非常好理解，但是效率不是很好。实际中很少使用
    </p>
    <p>
     2. 时间复杂度：O(N^2)
    </p>
    <p>
     3. 空间复杂度：O(1)
    </p>
    <p>
     4. 稳定性：不稳定
    </p>
    <h4>
     2.2.3 堆排序
    </h4>
    <p>
     堆排序(Heapsort)是指利用堆积树（堆）这种数据结构所设计的一种排序算法，它是选择排序的一种。它是 通过堆来进行选择数据。需要注意的是排升序要建大堆，排降序建小堆。
    </p>
    <p>
     <img alt="" height="765" src="https://i-blog.csdnimg.cn/direct/3116b6d750454621b45bedbb58753742.png" width="820"/>
    </p>
    <p>
     代码演示：
    </p>
    <pre><code class="language-cpp">
void swap(int* p1, int* p2)
{
	int tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}
void adjustdwon(int* a, int n, int parent)
{
	int child = (parent * 2) + 1;
	while (child &lt; n)
	{
		if (child + 1 &lt; n &amp;&amp; a[child] &lt; a[child + 1])
		{
			child++;
		}
		if (a[child] &gt; a[parent])
		{
			swap(&amp;a[child], &amp;a[parent]);
			parent = child;
			child = (parent * 2) + 1;
		}
		else
		{
			break;
		}
	}
}
void HeapSort(int* a, int n)
{
	int i = 0;
	for (i = (n - 1 - 1) / 2;i &gt;= 0;i--)
	{
		adjustdwon(a, n, i);
	}
	int end = n - 1;
	while (end &gt; 0)
	{
		swap(&amp;a[end], &amp;a[0]);
		adjustdwon(a, end, 0);
		end--;
	}
}</code></pre>
    <p>
     注： 因为向下调整建大堆时，要左右子树都为大堆因此我们要从最后一个不是叶子节点的 位置开始向下调整建大堆，而这个位置就是最后一个叶子节点的父亲。每次拿到最大值以后与最后位置交换，之后就不要动最大值了。
    </p>
    <p>
     直接选择排序的特性总结：
    </p>
    <p>
     1. 堆排序使用堆来选数，效率就高了很多。
    </p>
    <p>
     2. 时间复杂度：O(N*logN)
    </p>
    <p>
     3. 空间复杂度：O(1)
    </p>
    <p>
     4. 稳定性：不稳定
    </p>
    <h3>
     2.3 交换排序
    </h3>
    <p>
     基本思想：所谓交换，就是根据序列中两个记录键值的比较结果来对换这两个记录在序列中的位置，交换排 序的特点是：将键值较大的记录向序列的尾部移动，键值较小的记录向序列的前部移动。
    </p>
    <h4>
     2.3.1冒泡排序
    </h4>
    <p>
     基本思想：两个数字相互比较大的换到小的前面，如此反复。当一趟走完以后，最大的会在最后面。
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/30a86aca530d46578fd772e97cdc5d9b.gif"/>
    </p>
    <p>
     代码演示：
    </p>
    <pre><code class="language-cpp">void swap(int* p1, int* p2)
{
	int tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}
void BubbleSort(int* a, int n)
{
	for (int j = 0;j &lt; n;j++)
	{
		for (int i = 1;i &lt; n - j;i++)
		{
			if (a[i - 1] &gt; a[i])
			{
				swap(&amp;a[i - 1], &amp;a[i]);
			}
		}
	}

}</code></pre>
    <p>
     冒泡排序的特性总结：
    </p>
    <p>
     1. 冒泡排序是一种非常容易理解的排序
    </p>
    <p>
     2. 时间复杂度：O(N^2)
    </p>
    <p>
     3. 空间复杂度：O(1)
    </p>
    <p>
     4. 稳定性：稳定
    </p>
    <h4>
     2.3.2 快速排序
    </h4>
    <p>
     快速排序是Hoare于1962年提出的一种二叉树结构的交换排序方法，其基本思想为：任取待排序元素序列中 的某元素作为基准值，按照该排序码将待排序集合分割成两子序列，左子序列中所有元素均小于基准值，右 子序列中所有元素均大于基准值，然后最左右子序列重复该过程，直到所有元素都排列在相应位置上为止。
    </p>
    <p>
     1. hoare版本
    </p>
    <p>
     可以先看一个动图：
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/017bf66a496040aeb5dca537b98f6572.gif"/>
    </p>
    <p style="text-align:center">
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/bb45c83e6c2e4f2a910b73857fef9ac7.png"/>
    </p>
    <p>
    </p>
    <p>
     代码演示：
    </p>
    <pre><code class="language-cpp">void swap(int* p1, int* p2)
{
	int tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}

{

	if (a[left] &lt; a[right])
	{
		if (a[midi] &gt; a[left])
		{
			return midi;
		}
		else if (a[midi] &lt; a[left])
		{
			return left;
		}
		else
		{
			return right;
		}
	}
	//(a[left]&gt;a[right])
	else
	{
		if (a[midi] &gt;a[right])
		{
			return midi;
		}
		else if (a[midi] &lt; a[right])
		{
			return right;
		}
		else
		{
			return left;
		}
	}
}
int PartSort1(int* a, int left, int right)
{
	//随机取值
	/*int randi = left + (rand() % (right - left));
	swap(&amp;a[left], &amp;a[randi]);
	int keyi = left;*/
	//三数取中
	int midi = getmidi(a, left, right);
	swap(&amp;a[left], &amp;a[midi]);
	int keyi = left;
	while (left &lt; right)
	{
		while (left &lt; right &amp;&amp; a[right]&gt;=a[keyi])
		{
			right--;
		}
		while (left &lt; right &amp;&amp; a[left] &lt;= a[keyi])
		{
			left++;
		}
		swap(&amp;a[left], &amp;a[right]);
	}
	swap(&amp;a[left], &amp;a[keyi]);
	keyi = left;
	return keyi;
}
void QuickSort(int* a, int left, int right)
{
	if (left &gt;= right)
	{
		return;
	}
	int keyi = PartSort1(a, left, right);
	QuickSort(a, left, keyi - 1);
	QuickSort(a, keyi + 1, right);
}</code></pre>
    <p>
     注：代码里面的随机取值是为了让key值取一个比较随机的值，因为当数组为有序时，如果用常规的Hoare快排会让时间复杂度变成（N^2）下面将用一个图解释一下。
    </p>
    <p>
     <img alt="" height="674" src="https://i-blog.csdnimg.cn/direct/4c4894950cb94557a37cf000abec63c5.png" width="846"/>
    </p>
    <p>
     这里一般是有两种方法第一个就是取随机值像上面代码呈现的一样，第二个就是三数取中在左边中间右边位置，取一个中间值。如果遇到大量重复数据，快排的效率会大大降低，这里就需要三路划分的方法。（ keyi代表key值的下标）
    </p>
    <p>
     2. 挖坑法
    </p>
    <p style="text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/90142653cfa2481b895b5cdd8a671f77.gif"/>
    </p>
    <p>
     挖坑其实和 hoare版本十分相似，唯一不同的就是将key值拿到外面去制造一个坑位（这里注意还是要保持左边做key右边先走的思想，这里是左边的值拿出来右边就先走）然后右边找到一个小于key值的数放到坑里。左边再找小于key的值放到右边坑位，如此反复。
    </p>
    <p>
     代码演示：
     <br/>
    </p>
    <pre><code class="language-cpp">
void swap(int* p1, int* p2)
{
	int tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}
int getmidi(int* a, int left, int right)
{
	int midi = (left + right) / 2;
	if (a[left] &lt; a[right])
	{
		if (a[midi] &gt; a[left])
		{
			return midi;
		}
		else if (a[midi] &lt; a[left])
		{
			return left;
		}
		else
		{
			return right;
		}
	}
	//(a[left]&gt;a[right])
	else
	{
		if (a[midi] &gt;a[right])
		{
			return midi;
		}
		else if (a[midi] &lt; a[right])
		{

	while (left &lt; right)

		while (left &lt; right &amp;&amp; a[right] &gt;= key)

    <h2>
     2.5 非比较排序
    </h2>
    <p>
     思想：计数排序又称为鸽巢原理，是对哈希直接定址法的变形应用。
    </p>
    <p>
     操作步骤： 1. 统计相同元素出现次数
    </p>
    <p>
     2. 根据统计的结果将序列回收到原来的序列中
    </p>
    <p>
     <img alt="" height="343" src="https://i-blog.csdnimg.cn/direct/40177a8bf8c14c0db10068c9e222a426.png" width="840"/>
    </p>
    <pre><code class="language-cpp">void CountSort(int* a, int n)
{
	int max = a[0];
	int min = a[0];
	
	for (int i = 1;i &lt; n;i++)
	{
		if (max &lt; a[i])
		{
			max = a[i];
		}
		if (min &gt; a[i])
		{
			min = a[i];
		}
	}
	int range = max - min+1;
	int* counta = (int*)malloc(sizeof(int) * range);
	if (counta == NULL)
	{
		perror("malloc fail");
		return;
	}
	memset(counta, 0, sizeof(int) * range);
	//计数
	for (int i = 0;i &lt; n;i++)
	{
		counta[a[i] - min]++;
	}
	//排序
	int j = 0;
	for (int i = 0;i&lt; range;i++)
	{
		while (counta[i]--)
		{
			a[j++] = i+ min;
		}
	}
	free(counta);
}</code></pre>
