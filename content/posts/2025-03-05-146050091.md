    <h3>
     <a id="_2">
     </a>
     引言
    </h3>
    <p>
     Qt 是一个强大的跨平台 C++ 开发框架，广泛应用于 GUI 开发、嵌入式系统和工业软件等领域。然而，许多开发者习惯于在 Linux（如 Ubuntu）环境下使用 Qt，而在 Windows 上搭建 Qt 开发环境时可能会遇到许多问题，如编译器兼容性、依赖项管理等。因此，本文将介绍最快速的方法在 Windows 上安装 Qt 5，并避免 Ubuntu 工程移植到 Windows 的额外开销。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9dff12e9392143df974e3d7f62b3b476.png"/>
    </p>
    <h3>
     <a id="_7">
     </a>
     一、准备工作
    </h3>
    <p>
     在正式安装 Qt 之前，需要先准备以下内容：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        Windows 版本要求
       </strong>
      </p>
      <ul>
       <li>
        推荐使用 Windows 10 或更高版本。
       </li>
       <li>
        64 位操作系统（Qt 官方不再支持 32 位 Windows）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        所需工具
       </strong>
      </p>
      <ul>
       <li>
        Qt 官方安装程序
       </li>
       <li>
        C++ 编译器（MinGW 或 MSVC）
       </li>
       <li>
        CMake（可选）
       </li>
       <li>
        Ninja（可选）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        网络环境
       </strong>
      </p>
      <ul>
       <li>
        Qt 安装文件较大（数 GB），建议使用稳定的网络环境下载。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_Qt_5_24">
     </a>
     二、下载安装 Qt 5
    </h3>
    <h4>
     <a id="1__Qt__26">
     </a>
     1. 下载 Qt 安装程序
    </h4>
    <p>
     最简单的方式是使用 Qt 官方提供的安装程序。访问
     <a href="https://www.qt.io/download" rel="nofollow">
      Qt 官方网站
     </a>
     并选择以下选项：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        在线安装器
       </strong>
       ：
      </p>
      <ul>
       <li>
        适用于想要安装最新 Qt 版本的用户。
       </li>
       <li>
        需要 Qt 账户登录（免费注册）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        离线安装包
       </strong>
       ：
      </p>
      <ul>
       <li>
        适用于安装 Qt 旧版本，或想避免网络问题的用户。
       </li>
       <li>
        无需 Qt 账户登录。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     通常建议使用
     <strong>
      在线安装器
     </strong>
     ，它可以自动下载和管理所有依赖项。
    </p>
    <h4>
     <a id="2__Qt_5_40">
     </a>
     2. 安装 Qt 5
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        运行安装程序
       </strong>
      </p>
      <ul>
       <li>
        下载
        <code>
         qt-unified-windows-x64.exe
        </code>
        并运行。
       </li>
       <li>
        选择 “Custom Installation”（自定义安装）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择 Qt 版本
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         Qt 5.15.x（LTS 版，长期支持）
        </strong>
       </li>
       <li>
        <strong>
         Qt 5.12.x（较老但稳定）
        </strong>
       </li>
       <li>
        <strong>
         Qt 5.9.x（更旧但仍在一些项目中使用）
        </strong>
       </li>
      </ul>
      <p>
       选择适合你的 Qt 版本，建议使用
       <strong>
        Qt 5.15.x
       </strong>
       ，因为它是最后一个 LTS（长期支持）版本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        选择组件
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         必须安装
        </strong>
        <ul>
         <li>
          Qt 5.x.x（对应版本）
         </li>
         <li>
          Qt Creator（官方 IDE）
         </li>
        </ul>
       </li>
       <li>
        <strong>
         编译器支持
        </strong>
        <ul>
         <li>
          MinGW（推荐，开箱即用）
         </li>
         <li>
          MSVC（如果使用 Visual Studio）
         </li>
        </ul>
       </li>
       <li>
        <strong>
         额外工具（可选）
        </strong>
        <ul>
         <li>
          Qt Charts（数据可视化）
         </li>
         <li>
          Qt WebEngine（浏览器相关）
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        设置安装路径
       </strong>
      </p>
      <ul>
       <li>
        推荐安装到
        <code>
         C:\Qt\Qt5.x.x
        </code>
        目录，避免路径过长导致的问题。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        开始安装
       </strong>
      </p>
      <ul>
       <li>
        点击 “Next”，等待安装完成。
       </li>
       <li>
        由于 Qt 包体积较大，下载和安装可能需要 10~30 分钟。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <a id="_Qt__71">
     </a>
     三、配置 Qt 开发环境
    </h3>
    <h4>
     <a id="1__Qt_Creator_73">
     </a>
     1. 配置 Qt Creator
    </h4>
    <p>
     Qt Creator 是官方推荐的 IDE，安装完成后可按以下步骤配置：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        打开 Qt Creator
       </strong>
      </p>
     </li>
     <li>
      <p>
       <strong>
        设置 Qt 版本
       </strong>
      </p>
      <ul>
       <li>
        进入
        <code>
         Tools
        </code>
        →
        <code>
         Options
        </code>
        →
        <code>
         Qt Versions
        </code>
       </li>
       <li>
        确保 Qt 5.x.x 版本已正确识别
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择编译器
       </strong>
      </p>
      <ul>
       <li>
        进入
        <code>
         Tools
        </code>
        →
        <code>
         Options
        </code>
        →
        <code>
         Kits
        </code>
       </li>
       <li>
        添加
        <strong>
         MinGW
        </strong>
        或
        <strong>
         MSVC
        </strong>
        编译器。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        配置 CMake（可选）
       </strong>
      </p>
      <ul>
       <li>
        进入
        <code>
         Tools
        </code>
        →
        <code>
         Options
        </code>
        →
        <code>
         CMake
        </code>
       </li>
       <li>
        确保 CMake 已正确识别。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <a id="2__90">
     </a>
     2. 设置环境变量（命令行开发）
    </h4>
    <p>
     如果你打算在命令行使用 Qt，需要设置环境变量：
    </p>
    <ul>
     <li>
      打开
      <strong>
       终端（cmd 或 PowerShell）
      </strong>
     </li>
     <li>
      运行以下命令（适用于 MinGW）：
      <pre><code class="prism language-shell"><span class="token builtin class-name">set</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span>C:<span class="token punctuation">\</span>Qt<span class="token punctuation">\</span>Qt5.x.x<span class="token punctuation">\</span>mingw81_64<span class="token punctuation">\</span>bin<span class="token punctuation">;</span>%<span class="token environment constant">PATH</span>%
</code></pre>
     </li>
    </ul>
    <p>
     如果使用 MSVC，需要打开
     <code>
      x64 Native Tools Command Prompt
     </code>
     并执行：
    </p>
    <pre><code class="prism language-shell"><span class="token builtin class-name">set</span> <span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span>C:<span class="token punctuation">\</span>Qt<span class="token punctuation">\</span>Qt5.x.x<span class="token punctuation">\</span>msvc2019_64<span class="token punctuation">\</span>bin<span class="token punctuation">;</span>%<span class="token environment constant">PATH</span>%
</code></pre>
    <h3>
     <a id="_Qt__105">
     </a>
     四、快速创建 Qt 项目
    </h3>
    <h4>
     <a id="1__Qt_Creator__GUI__107">
     </a>
     1. 使用 Qt Creator 创建 GUI 应用
    </h4>
    <ol>
     <li>
      打开 Qt Creator，点击
      <code>
       File
      </code>
      →
      <code>
       New Project
      </code>
     </li>
     <li>
      选择
      <code>
       Qt Widgets Application
      </code>
      或
      <code>
       Qt Quick Application
      </code>
     </li>
     <li>
      选择 Qt 版本和编译器
     </li>
     <li>
      编写代码，点击
      <code>
       Run
      </code>
      运行
     </li>
    </ol>
    <h4>
     <a id="2__Qt_qmake_114">
     </a>
     2. 在命令行创建 Qt 项目（qmake）
    </h4>
    <p>
     如果想用命令行创建 Qt 项目，可以执行：
    </p>
    <pre><code class="prism language-shell"><span class="token function">mkdir</span> my_project
<span class="token builtin class-name">cd</span> my_project
qmake <span class="token parameter variable">-project</span>
qmake
mingw32-make
</code></pre>
    <h3>
     <a id="_Ubuntu__Windows_126">
     </a>
     五、避免直接移植 Ubuntu 工程到 Windows
    </h3>
    <p>
     许多开发者希望直接将 Ubuntu 的 Qt 项目迁移到 Windows，但这可能会引发兼容性问题。以下是几个常见问题及其解决方案：
    </p>
    <h4>
     <a id="1__130">
     </a>
     1.
     <strong>
      路径问题
     </strong>
    </h4>
    <ul>
     <li>
      Ubuntu 使用
      <code>
       /home/user/project
      </code>
      ，而 Windows 使用
      <code>
       C:\Users\user\project
      </code>
     </li>
     <li>
      解决方案：在代码中使用
      <code>
       QDir::toNativeSeparators()
      </code>
      处理路径。
     </li>
    </ul>
    <h4>
     <a id="2__134">
     </a>
     2.
     <strong>
      库兼容性
     </strong>
    </h4>
    <ul>
     <li>
      Ubuntu 使用
      <code>
       .so
      </code>
      共享库，Windows 使用
      <code>
       .dll
      </code>
     </li>
     <li>
      解决方案：在 CMake 或 qmake 文件中针对不同平台设置不同的库路径。
     </li>
    </ul>
    <h4>
     <a id="3__138">
     </a>
     3.
     <strong>
      终端命令不兼容
     </strong>
    </h4>
    <ul>
     <li>
      Ubuntu 使用
      <code>
       make
      </code>
      ，Windows 可能需要
      <code>
       mingw32-make
      </code>
     </li>
     <li>
      解决方案：在 Windows 上使用
      <code>
       Ninja
      </code>
      或
      <code>
       MSBuild
      </code>
      代替
      <code>
       make
      </code>
      。
     </li>
    </ul>
    <h3>
     <a id="_142">
     </a>
     六、总结
    </h3>
    <p>
     在 Windows 上安装 Qt 5 并不复杂，只需按照以下步骤：
    </p>
    <ol>
     <li>
      下载并安装 Qt 在线安装程序。
     </li>
     <li>
      选择合适的 Qt 版本和编译器（推荐 MinGW）。
     </li>
     <li>
      配置 Qt Creator 和环境变量。
     </li>
     <li>
      使用 Qt Creator 或命令行创建和运行 Qt 项目。
     </li>
    </ol>
    <p>
     避免直接移植 Ubuntu 工程到 Windows，可以通过路径兼容性、库管理和构建工具等方式优化跨平台 Qt 开发。
    </p>
    <p>
     这样，你就可以在 Windows 上顺利进行 Qt 5 开发，而不必依赖 Linux 环境！
    </p>
