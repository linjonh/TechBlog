     自增自减运算符
    </h3>
    <ul>
     <li>
      操作数：2
     </li>
     <li>
      和C语言中的自增自减运算符一样，自增自减运算符可以放在操作元素的前面或者后面，作用也和C语言中的一样，前置则先执行++或–，后使用操作元素的值，后置则先使用操作元素的值，后执行++或–
     </li>
    </ul>
    <h3>
     <a id="_22">
     </a>
     算术混合运算的精度
    </h3>
    <p>
     精度从低到高的排序是：
     <code>
      byte short char int long float double
     </code>
    </p>
    <p>
     在Java中计算算术表达式的值时，使用以下的精度运算规则：
    </p>
    <ol>
     <li>
      如果表达式中有双精度浮点数，按双精度进行计算
     </li>
     <li>
      如果表达式的最高精度是单精度浮点数，则按单精度进行运算
     </li>
     <li>
      如果表达式中的最高精度是long型，则按long精度进行运算
     </li>
     <li>
      如果表达式中的最高精度低于int型，则按int型精度进行运算
     </li>
    </ol>
    <p>
     Java允许把不超出byte、short、char的取值范围的常量算术表达式的值赋给byte、short和char型变量
     <br/>
     <code>
      byte x = (byte)30+'a';
     </code>
     是正确的，
     <code>
      byte x = (byte)30+'b;
     </code>
     是错误的
    </p>
    <p>
     当赋值号右边的表达式中有变量时，编译只检查变量的类型，不检查变量中的值。例如，
     <code>
      byte x = 97 + 1
     </code>
     和
     <code>
      byte y = 1
     </code>
     都是正确的，但是
     <code>
      byte z = 97 + y
     </code>
     是错误的，这是因为表达式右侧按int进行运算，不检查y中的值，所以会提示类型不兼容
    </p>

      <code>
       op1||op2

    <h3>
     <a id="_84">
     </a>
     位运算符
    </h3>
    <p>
     整型数据在内存中以二进制的方式表示，最高位为符号位，位运算符就是对整型数据对每个二进制分别进行位操作
    </p>
    <p>
     位运算符包括：
     <code>
      按位与&amp;
     </code>
     、
     <code>
      按位或|
     </code>
     、
     <code>
      按位非!
     </code>
     、
     <code>
      按位异或^
     </code>
    </p>
    <ul>
     <li>
      按位与就是将左右两个操作元的对应每一位进行与&amp;操作，都为1则结果为1，有一个是0则结果为0
     </li>
     <li>
      按位或就是将左右两个操作元的对应每一位进行或|操作，有一个操作元的该位为1则结果为1，均为零则结果为零
     </li>
     <li>
      按位非就是按位取反，将操作数的每一位0变1，1变0
     </li>
     <li>
      按位异或就是对两个操作元每个二进制位进行异或操作，相同为0，相异为1
     </li>
    </ul>
    <h3>
     <a id="instanceof_94">

     instanceof运算符
    </h3>
    <p>
     instanceof运算符是二目操作符，左边的操作元是一个对象，右边是一个类，当左边的对象是右边类或子类创建的对象时，运算结果为true，否则为false
    </p>

    <h2>
     <a id="_116">
     </a>
     分支语句
    </h2>
    <p>
     语法上看，Java的分支语句和C语言的分支语句几乎完全相同
    </p>
    <h3>
     <a id="if_118">
     </a>
     if条件分支语句
    </h3>
    <h4>
     <a id="if_119">
     </a>
     if语句
    </h4>
    <p>
     if语句是单条件、单分支语句
    </p>
    <pre><code class="prism language-java"><span class="token keyword">if</span><span class="token punctuation">(</span>表达式）
<span class="token punctuation">{<!-- --></span>
语句
<span class="token punctuation">}</span>
</code></pre>
    <p>
     if后的表达式的值必须为boolean型，同样不能像C语言一样使用零或非零表示
     <br/>
     如果表达式为true则执行花括号后的语句，否则不执行
    </p>
    <p>
     如果语句只有一条，可以省略花括号，建议写上，保持良好的代码风格
    </p>
    <h4>
     <a id="ifelse_131">
     </a>
     if-else语句
    </h4>
    <p>
     if-else语句是单条件、双分支语句
    </p>
    <pre><code class="prism language-java"><span class="token keyword">if</span><span class="token punctuation">(</span>表达式<span class="token number">1</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
    语句<span class="token number">1</span>；
<span class="token punctuation">}</span>
<span class="token keyword">else</span>
<span class="token punctuation">{<!-- --></span>
    语句<span class="token number">2</span>；
<span class="token punctuation">}</span>
</code></pre>
    <p>
     if-else语句同样需要表达式为boolean型，如果值为true则执行语句1，否则执行语句2
    </p>
    <h4>
     <a id="ifelse_ifelse_144">
     </a>
     if-else if-else语句
    </h4>
    <p>
     if-else if-else语句是多条件、多分支语句
    </p>
    <pre><code class="prism language-java"><span class="token keyword">if</span><span class="token punctuation">(</span>表达式<span class="token number">1</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
    语句<span class="token number">1</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token keyword">else</span> <span class="token keyword">if</span><span class="token punctuation">(</span>表达式<span class="token number">2</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
    语句<span class="token number">2</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token keyword">else</span>
<span class="token punctuation">{<!-- --></span>
    语句<span class="token number">3</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <ul>
     <li>
      <p>
       若表达式1为true，则执行语句1；
      </p>
     </li>
     <li>
      <p>
       若表达式1为false，判断表达式2，
      </p>
      <ul>
       <li>
        若表达式2为true，执行语句2，
       </li>
       <li>
        若表达式2为false，执行语句3
       </li>
      </ul>
     </li>
    </ul>
    <p>
     如果有更多的if-else语句也是同理，逐次向下判断表达式的值，直到有一个值为true或所有的值均为false，然后执行else
    </p>
    <h3>
     <a id="switch_168">
     </a>
     switch
    </h3>
    <p>
     switch语句是单条件、多分支的开关语句
    </p>
    <pre><code class="prism language-java"><span class="token keyword">switch</span><span class="token punctuation">(</span>表达式<span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>

			若干语句
			<span class="token keyword">break</span><span class="token punctuation">;</span>
	<span class="token keyword">case</span> 常量值<span class="token number">2</span><span class="token operator">:</span>
			若干语句
			<span class="token keyword">break</span><span class="token punctuation">;</span>
	<span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span>
	<span class="token keyword">case</span> 常量值n<span class="token operator">:</span>
			若干语句
			<span class="token keyword">break</span><span class="token punctuation">;</span>
	<span class="token keyword">default</span><span class="token operator">:</span>
			若干语句
<span class="token punctuation">}</span>
</code></pre>
    <p>
     switch语句中，表达式的值可以是byte short int char String类型，但不可以是long型
    </p>
    <p>
     switch语句首先计算表达式的值，如果表达式的值与某个case后的常量值相同，就执行该case后面的语句，直到遇到break退出，如果没有break语句，那么会继续执行后面其他case后的语句。如果表达式的值与任何case都不相同，则执行default后的语句，default是可选的，如果没有default，并且表达式的值不与任何的常量值相同，那么switch语句就不会进行任何处理
    </p>
    <h2>
     <a id="_190">
     </a>
     循环语句
    </h2>
    <p>
     语法上看，Java的循环语句和C语言的循环语句基本相同，但Java中没有
     <code>
      goto
     </code>
     语句
    </p>
    <h3>
     <a id="for_192">
     </a>
     for循环
    </h3>
    <pre><code class="prism language-java"><span class="token keyword">for</span> <span class="token punctuation">(</span> i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> length<span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>    <span class="token comment">//初始化 // 判断   //调整</span>
    语句；<span class="token comment">//循环体</span>
<span class="token punctuation">}</span>   
</code></pre>
    <pre><code>1.for后面的括号中包括三个内容，依次为：

	对变量的初始化  循环条件判断  对某些变量的调整（一般是控制循环退出的变量）

	两两之间用分号间隔

2.上述三项内容中，初始化可省略（不建议），可在初始化处定义新变量，但只能在循环内使用该变量

3.for循环条件判断可由多个变量控制，用&amp;&amp;（与）||（或）分割

4.若省略循环条件判断，会使判断恒成立，进入死循环

5.有两条及以上语句时，需加大括号

6.循环体可为空语句
</code></pre>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> test<span class="token punctuation">{<!-- --></span>
	<span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token class-name">String</span> argv<span class="token punctuation">[</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
		<span class="token keyword">for</span> <span class="token punctuation">(</span><span class="token keyword">int</span> i <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span> i <span class="token operator">&lt;</span> <span class="token number">10</span><span class="token punctuation">;</span> i<span class="token operator">++</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
			<span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span>i<span class="token punctuation">)</span><span class="token punctuation">;</span>
		<span class="token punctuation">}</span>
	<span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b76310e295fe4736bdf469ac1ad5d042.png"/>
    </p>
    <h3>
     <a id="while_228">
     </a>
     while循环
    </h3>
    <pre><code class="prism language-java"><span class="token keyword">while</span><span class="token punctuation">(</span>表达式<span class="token punctuation">)</span>
<span class="token punctuation">{<!-- --></span>
    循环体；
<span class="token punctuation">}</span>
</code></pre>
    <p>
     while语句的执行规则如下：
    </p>
    <ol>
     <li>
      计算表达式的值，为true则进行2，否则进行3
     </li>
     <li>
      执行循环体，再进行1
     </li>
     <li>
      结束while语句
     </li>
    </ol>
    <h3>
     <a id="dowhile_239">
     </a>
     do-while循环
    </h3>
    <pre><code class="prism language-java"><span class="token keyword">do</span>
<span class="token punctuation">{<!-- --></span>
   <span class="token comment">//循环体；         </span>
<span class="token punctuation">}</span> 
<span class="token keyword">while</span> <span class="token punctuation">(</span>表达式<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     do-while循环和while循环的区别是do-while循环至少执行一次，执行第一次后再计算表达式的值，为true则继续执行，否则结束循环。一般在循环体中要加入对表达式值的调整，否则容易进入死循环
    </p>
    <h3>
     <a id="breakcontinue_249">
     </a>
     break和continue
    </h3>
    <p>
     <strong>
      break
     </strong>
     在循环中起到跳出循环的作用，当程序执行到break语句时，无论循环条件是否满足，直接跳出循环
    </p>
    <p>
     <strong>
      continue
     </strong>
     用于跳过本次循环中，位于continue后面的语句，进入下一次循环
    </p>
    <pre><code>1.continue不能直接跳出循环，而是跳过本次循环剩余的语句	
2.continue在for和while中有不同的效果
	（1）在for循环中，continue跳过后，for(表达式1;表达式2;调整)  调整部分仍然会执行，不影响整体循环的退出
	（2）在while/do while循环中，continue会跳过剩余全部代码，若调整部分位于continue后，则不会执行，可能会影响循环的退出
</code></pre>
    <h3>
     <a id="for_259">
     </a>
     for循环和数组
    </h3>
    <p>
     JDK5对for语句的功能给予了扩充、增强，以便更好的遍历数组。。语法如下：
    </p>
    <pre><code class="prism language-java"><span class="token keyword">for</span><span class="token punctuation">(</span>声明循环变量<span class="token operator">:</span>数组的名字<span class="token punctuation">)</span><span class="token punctuation">{<!-- --></span>
	<span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span>

    <p>
     其中，声明的循环变量的类型必须和数组的类型相同。
    </p>
    <p>
     这种类型的for循环可以理解为“每次循环依次取数组中每一个元素的值”
     <br/>
     声明的变量中取出的是数组元素的
     <strong>
      值
     </strong>
     ，而不是索引下标

     这种方式只能访问查看数组元素，不能对数组元素进行修改
