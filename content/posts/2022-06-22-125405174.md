    <p>
     搞前端的，项目需要一个记录日志的功能，决定用python+前端的方式实现
    </p>
    <p>
     记录前端与python数据交互的简单尝试
    </p>
    <p>
     1、引入flask模块（用之前先安装，我已经安装过了）
    </p>
    <pre><code class="language-python"># flask具有轻量、简捷易上手的特点，适合新手使用
from flask import Flask
# request用于接收数据
from flask import request
# 解决跨域问题
from flask_cors import CORS</code></pre>
    <p>
     2、创建FLask对象
    </p>
    <pre><code>#  创建flask服务对象
app = Flask(__name__)
#  动态解决前端跨域问题
CORS(app, supports_credentials=True)</code></pre>
    <p>
     3、写接收数据的接口
    </p>
    <pre><code>#  指定请求路径、方法
@app.route('/log', methods=['GET'])
def reciveLog():
    #get请求数据获取 post请求数据获取request.form.get("log")
    data = request.args.get("log")
    return data</code></pre>
    <p>
     4、运行
    </p>
    <p>
     端口地址指不指定都行
    </p>
    <pre><code>if __name__== "__main__":
    #  指定端口号和地址
    app.run(port=1234)</code></pre>
    <p>
     点击运行，控制台会显示访问地址，前端访问这个地址+接口路径，就可以访问了
    </p>
