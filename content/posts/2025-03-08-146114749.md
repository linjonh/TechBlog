    <p>
     <strong>
      输入描述
     </strong>
     <br/>
     第一行为一个数 N，表示路灯个数：1 &lt;= N &lt;= 100000，
     <br/>
     第二行为 N 个空格分割的数，表示路灯的照明半径，1 &lt;= 照明半径。
    </p>
    <p>
     <strong>
      输出描述
     </strong>
     <br/>
     无法照明的区间的长度和。
    </p>
    <p>
     <strong>
      示例演示
     </strong>
     <br/>
     <strong>
      示例1
     </strong>
     <br/>
     输入：
    </p>
    <pre><code class="hljs">2
50 50</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="hljs">0</code></pre>
    <p>
     <strong>
      示例2
     </strong>
     <br/>
     输入：
    </p>
    <pre><code class="hljs">4
50 70 20 70</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="hljs">20</code></pre>
    <p>
     示例说明： 50+70&gt;100，70+20&lt;100,20+70&lt;100,所以有 两个未要盖的区间，总里程为 10+10=20。
    </p>
    <p>
     注:这里需要注意照明半径大于100时，需要判断当前灯半径是否大于前一个灯半径-100，如果是大于则将当前灯半径传给下一个灯，如果小于则，将前一个灯半径-100传给下一个灯。
    </p>
    <pre><code class="language-java">package odTest;

import java.util.Arrays;
import java.util.Scanner;

public class lightQuestion {
	static int sum = 0;
	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		int num = Integer.parseInt(scanner.nextLine());
		
		int[] lightCoverArea = Arrays.stream(scanner.nextLine().split(" ")).mapToInt(Integer::parseInt).toArray();
		
		judgeUnCover(num,lightCoverArea,lightCoverArea[0],1);
		System.out.println(sum);
	}

	private static void judgeUnCover(int num, int[] lightCoverArea, int preVal,int index) {
		// TODO Auto-generated method stub
		if(index==num) {
			return;
		}
		if(preVal+lightCoverArea[index]&lt;100) {
//			System.out.println(preVal);
			sum = sum+100-(preVal+lightCoverArea[index]);
//			System.out.println(sum);
		}
		
		//判断当前照明半径，传给下一个灯
		if(preVal&lt;100&amp;&amp;lightCoverArea[index]&lt;100) {
//			System.out.println(1);
			preVal=lightCoverArea[index];
		}else if(preVal&gt;100&amp;&amp;lightCoverArea[index]+100&lt;preVal) {
			preVal=preVal-100;
		}else if(preVal&gt;100&amp;&amp;lightCoverArea[index]+100&gt;preVal) {
			preVal=lightCoverArea[index];
		}else if(preVal&lt;100&amp;&amp;lightCoverArea[index]&gt;100) {
			preVal=lightCoverArea[index];
		}
		judgeUnCover(num,lightCoverArea,preVal,index+1);
	}

}
</code></pre>
