# 问题解决：Kali Linux 中配置启用 Vim 复制粘贴功能

在 Kali Linux 系统中，使用 `XShell` 或其他类似终端时，Vim
编辑器的默认设置并不支持直接进行复制和粘贴操作，这对于日常的开发工作或渗透测试人员来说可能会造成不便。幸运的是，通过简单的配置调整，可以让 Vim
轻松支持复制和粘贴功能。

要在 Kali Linux 的 Vim 中启用复制和粘贴功能，只需执行以下几个简单的步骤：

#### **步骤 1：进入 Vim 命令模式**

首先，打开你的终端，使用 `vim` 打开一个文件。例如：

    
    
    vim test.txt
    

进入 Vim 编辑器后，按 `Esc` 键进入命令模式，确保你处于正常编辑模式。

#### **步骤 2：启用鼠标支持**

在 Vim 中启用鼠标支持可以让你使用终端的复制和粘贴功能。你可以通过输入以下命令来启用鼠标：

    
    
    :set mouse=c
    

  * `:set mouse=c` 命令的作用是启用 Vim 的鼠标支持，`c` 表示启用所有模式下的鼠标支持（包括 Normal 模式、Insert 模式和 Visual 模式）。
  * 启用鼠标支持后，你可以在 Vim 中自由地选择文本并进行复制粘贴操作，像在 GUI 环境下使用鼠标一样。

#### **步骤 3：验证配置是否生效**

配置完成后，你应该可以像在普通终端命令行中一样，正常地使用鼠标在 Vim 中进行复制和粘贴操作！
