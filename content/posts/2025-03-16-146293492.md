    <h4>
     <a id="2_Linking_38">
     </a>
     2. 连接阶段（Linking）
    </h4>
    <h5>
     <a id="_Verification_39">
     </a>
     ① 验证（Verification）
    </h5>
    <ul>
     <li>
      文件格式验证
     </li>
     <li>
      元数据验证
     </li>
     <li>
      字节码验证
     </li>
     <li>
      符号引用验证
     </li>
    </ul>
    <h5>
     <a id="_Preparation_45">
     </a>
     ② 准备（Preparation）
    </h5>
    <ul>
     <li>
      为静态变量分配内存
     </li>
     <li>
      设置默认初始值（如 int-&gt;0, boolean-&gt;false）
     </li>
    </ul>
    <h5>
     <a id="_Resolution_49">
     </a>
     ③ 解析（Resolution）
    </h5>
    <ul>
     <li>
      符号引用转直接引用
     </li>
     <li>
      分为静态解析与动态解析
     </li>
    </ul>
    <h4>
     <a id="3_clinit_53">
     </a>
     3. 初始化阶段（）
    </h4>
    <ul>
     <li>
      执行类构造器代码（静态变量赋值、静态代码块）
     </li>
     <li>
      父类优先初始化原则
     </li>
     <li>
      线程安全保证（同步锁）
     </li>
    </ul>
