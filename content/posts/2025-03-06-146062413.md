     今天我们将下载10x官方人肺癌FFPE样本Xenium5k下机数据，使用python的spatialdata库，演示如何进行Xenium单个样本/多样本数据读取，以及简单绘图功能展示。

    <p>
     <strong>
      3. 数据读取
     </strong>
    </p>
    <p>
     由于Xenium下机数据较大，多个样本按顺序读取耗时较长，这里我们使用多线程并行读取，缩短时间。
    </p>
    <p>
     data_dir参数是xenium下机数据文件位置；
    </p>
    <p>
     sample_info参数是样本信息.txt文件，一共三列，第一列是下机数据问价夹名称，第二列是样本名称，第三列是样本分组名称，使用'\t'分隔，有多个样本的，文件中就有多行。
    </p>
    <pre><code>import os</code><code>import threading</code><code>import spatialdata as sd</code><code>from spatialdata_io import xenium</code>
<code># 多线程读取Xenium下机数据读取</code><code>def xenium_data_load_multithreaded(data_dir, sample_info):</code><code>    def sd_read_xenium(sample_data, sample_name, sdata_dict):</code><code>        sdata = xenium(path=sample_data, cells_boundaries=True, n_jobs=6)</code><code>        sdata_dict[sample_name] = sdata</code><code>    threads = []</code><code>    sdata_dict = {}</code><code>    sample_2_group = {}</code><code>    with open(sample_info, 'r') as f:</code><code>        for line in f:</code><code>            raw_name, sample_name, group_name = line.strip().split('\t')[:3]  # 这里根据自己实际情况修改</code><code>            sample_2_group[sample_name] = group_name</code><code>            thread = threading.Thread(target=sd_read_xenium, args=(os.path.join(data_dir, raw_name),sample_name, sdata_dict,))</code><code>            threads.append(thread)</code><code>            thread.start()</code><code>    for thread in threads:</code><code>        thread.join()</code><code>    </code><code>    sdata = sd.concatenate(</code><code>                sdata_dict,</code><code>                concatenate_tables=True, # 这里是将多样本的单细胞数据合并在一起到table中</code><code>                obs_names_make_unique=True</code><code>            )</code><code>    sdata.tables['table'].obs["sample"] = sdata.tables['table'].obs["region"].str.replace('cell_circles-', '')</code><code>    sdata.tables['table'].obs["group"] = sdata.tables['table'].obs["sample"].apply(lambda x: sample_2_group[x])</code><code>    sdata.tables['table'].obs["cell_boundaries"] = sdata.tables['table'].obs["region"].str.replace('cell_circles', 'cell_boundaries')</code><code>    sdata.set_table_annotates_spatialelement(table_name='table', region=[i for i in sdata.shapes.keys() if i.startswith('cell_boundaries-')], region_key='cell_boundaries')
return sdata</code></pre>
