    <h3>
     <a id="_0">
     </a>
     题目：
    </h3>
    <p>
     197.上升的温度
     <br/>
     表： Weather
    </p>
    <p>
     ±--------------±--------+
     <br/>
     | Column Name | Type |
     <br/>
     ±--------------±--------+
     <br/>
     | id | int |
     <br/>
     | recordDate | date |
     <br/>
     | temperature | int |
     <br/>
     ±--------------±--------+
     <br/>
     id 是该表具有唯一值的列。
     <br/>
     没有具有相同 recordDate 的不同行。
     <br/>
     该表包含特定日期的温度信息
    </p>
    <p>
     编写解决方案，找出与之前（昨天的）日期相比温度更高的所有日期的 id 。
    </p>
    <p>
     返回结果 无顺序要求 。
    </p>
    <p>
     结果格式如下例子所示。
    </p>
    <p>
     示例 1：
    </p>
    <p>
     输入：
     <br/>
     Weather 表：
     <br/>
     ±—±-----------±------------+
     <br/>
     | id | recordDate | Temperature |
     <br/>
     ±—±-----------±------------+
     <br/>
     | 1 | 2015-01-01 | 10 |
     <br/>
     | 2 | 2015-01-02 | 25 |
     <br/>
     | 3 | 2015-01-03 | 20 |
     <br/>
     | 4 | 2015-01-04 | 30 |
     <br/>
     ±—±-----------±------------+
     <br/>
     输出：
     <br/>
     ±—+
     <br/>
     | id |
     <br/>
     ±—+
     <br/>
     | 2 |
     <br/>
     | 4 |
     <br/>
     ±—+
     <br/>
     解释：
     <br/>
     2015-01-02 的温度比前一天高（10 -&gt; 25）
     <br/>
     2015-01-04 的温度比前一天高（20 -&gt; 30）
    </p>
    <h3>
     <a id="_48">
     </a>
     思路：
    </h3>
    <p>
     这次需要对比日期大小，所以引入DATEDIFF()函数，该函数的作用为返回两个日期值之间的差值。
     <br/>
     结合题目的查询条件为“与之前（昨天的）日期相比温度更高的所有日期的 id”：
     <br/>
     a)两个日期的时间为今天和昨天
     <br/>
     b)今天的温度要比昨天的温度高
     <br/>
     所以使用join子句，以上述条件作为自连接的查询条件，获取返回结果的id值即可。
    </p>
    <h3>
     <a id="_55">
     </a>
     代码：
    </h3>
    <pre><code class="prism language-sql"><span class="token keyword">SELECT</span> a<span class="token punctuation">.</span>id
<span class="token keyword">FROM</span> weather a
<span class="token keyword">JOIN</span> weather b 
<span class="token keyword">ON</span> DATEDIFF<span class="token punctuation">(</span>a<span class="token punctuation">.</span>recordDate<span class="token punctuation">,</span>b<span class="token punctuation">.</span>recordDate<span class="token punctuation">)</span> <span class="token operator">=</span> <span class="token number">1</span> 
<span class="token operator">AND</span> a<span class="token punctuation">.</span>temperature <span class="token operator">&gt;</span> b<span class="token punctuation">.</span>temperature
</code></pre>
