    <h2>
     一.DS1302时钟
    </h2>
    <h3>
     1.编写DS1302.c文件
    </h3>
    <h4>
     （1）重新对端口定义名字
    </h4>
    <pre><code>sbit DS1302_SCLK=P3^6;
sbit DS1302_IO=P3^4;
sbit DS1302_CE=P3^5;
</code></pre>
    <h4>
     （2）初始化
    </h4>
    <p>
     因为单片机上电默认是1，所以要初始化为0
    </p>
    <pre><code>void DS1302_Init(void)
{
	DS1302_CE=0;
	DS1302_SCLK=0;
}</code></pre>
    <h4>
     （3）写入函数
    </h4>
    <p>
     在给SCLK赋值的时候
    </p>
    <p>
     DS1302_SCLK=1;
     <br/>
     DS1302_SCLK=0;
    </p>
    <p>
     在这中间置1后马上置0需要有一个最小的延时，但实际操作后发现不用加延时可以运行，因为我们的单片机运行没有这么快
    </p>
    <p>
     这里相当于我们已经写入了第0位
    </p>
    <pre><code>//Command是命令字
void DS1302_WriteByte(unsigned char Command,Data)
{
	DS1302_CE=1;
	DS1302_IO=Command&amp;0x01;  //取第0位
	DS1302_SCLK=1;  
	DS1302_SCLK=0;
	
}</code></pre>
    <p>
     同理第1位即    DS1302_IO=Command&amp;0x02;
    </p>
    <p>
     第2位即    DS1302_IO=Command&amp;0x04; .......
    </p>
    <p>
     所以我们可以用一个for循环来实现取8位
    </p>
    <pre><code>//Command是命令字
void DS1302_WriteByte(unsigned char Command,Data)
{
	unsigned char i
	DS1302_CE=1;
	
	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Command&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
	}

}
</code></pre>
    <p>
     此时我们已经完成写入操作的一半，又发现后面的写入数据和前面的写入指令是一样的，所以我们可以复制for循环的代码思路
     <img alt="" height="214" src="https://i-blog.csdnimg.cn/direct/f19a4ffd16e3461783c27323932a64f3.png" width="793"/>
    </p>
    <pre><code>//Command是命令字
void DS1302_WriteByte(unsigned char Command,Data)
{
	unsigned char i
	DS1302_CE=1;
	
	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Command&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
	}

	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Data&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
	}
	DS1302_CE=0;
}</code></pre>
    <p>
     记得在最后把CE置0
    </p>
    <p>
     到这我们就完成时序的写入函数，我们就可以对任何的寄存器进行写入操作
    </p>
    <h4>
     （4）读取函数
    </h4>
    <p>
     虽然该部分的时序跟上面的写入类似，但不能直接照搬，因为在SCLK这条线上只有15个脉冲（写入是16个）因为在最中间的脉冲同时进行上升沿和下降沿的操作
    </p>
    <p>
     所以我们把SCLK赋值的顺序颠倒一下，先给0再给1，这样当for循环8次后，刚好全是上升沿
    </p>
    <p>
     <img alt="" height="201" src="https://i-blog.csdnimg.cn/direct/bb2640471aca498cbebe46e1e9b703a2.png" width="805"/>
    </p>
    <p>
     这里根据代码理解在图上比划一下就很好理解，第一个for循环里先给0再给1，在循环结束后SCLK依旧是1，但是在第二个for循环里还是要先给1再给0，目的就是为了跳过一个周期适配15个脉冲
    </p>
    <pre><code>unsigned char DS1302_ReadByte(unsigned char Command)
{
	unsigned char i,Data=0x00;
	DS1302_CE=1;
	
	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Command&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=0;  
			DS1302_SCLK=1;
	}
	for(i=0;i&lt;8;i++)
	{
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
			if(DS1302_IO){Data |= (0X01&lt;&lt;i);}	
	}
	DS1302_CE=0;
	return Data;
}
</code></pre>
    <p>
     一般来说，与操作是为了清零，或操作是为了置1
    </p>
    <p>
     最后不用忘了return 返回值，因为这是有返回值的函数
    </p>
    <h4>
     （5）测试
    </h4>
    <p>
     写完上面3个函数，我们就已经对时序模拟出来了、

    <h4>
     （6）扩展知识点
    </h4>
    <p>
     BCD码（Binary Coded Decimal）：用4位二进制数来表示1位十进制
    </p>
    <p>
     内部的寄存器不是以二进制来存储的，而是以BCD码来存储
    </p>
    <p>
     例：0001 0011表示13，1000 0101表示85，0001 1010不合法
    </p>
    <p>
     在十六进制中的体现：0x13表示13，0x85表示85,0x1A不合法
    </p>
    <p>
     BCD码转十进制：DEC=BCD/16*10+BCD%16;（2位BCD）
    </p>
    <p>
     十进制转BCD码：BCD=DEC/10*16+DEC%16;（2位BCD）
    </p>
    <pre><code>#include &lt;REGX52.H&gt;
#include " LCD1602.h"
#include " DS1302.h"
#include " Delay.h"

unsigned char Second;

void main()
{
	DS1302_Init();
	DS1302_WriteByte(0x8E,0x00);
	LCD_Init();
	LCD_ShowString(1,1,"RTC");
	
	DS1302_WriteByte(0x80,0x03);

	
	while(1)
	{
 			Second=DS1302_ReadByte(0x81);
			LCD_ShowHexNum(2,1,Second,3);
	}

}</code></pre>
    <p>
     所以我们在这里想让Second自加显示在LCD上，就得使用下面这个函数，而不是ShowNum，否则他会从9突变到16，这是因为ShowNum是以十进制来显示，而时钟寄存器的自加是上面所说的BCD码，而下面这个函数是以十六进制显示的            LCD_ShowHexNum(2,1,Second,3);  十六进制和BCD码有部分兼容
    </p>
    <p>
     <strong>
      下面解释寄存器里的BCD码
     </strong>
    </p>
    <p>
     CH是时钟静止，给1静止给0运行，高3位显示10秒，低4位显示秒，分、时、日、月、年都是同理
    </p>
    <p>
     小时的最高位是选择12/24制，第6位选择AM还是PM
    </p>
    <p>
     <img alt="" height="260" src="https://i-blog.csdnimg.cn/direct/c848a8575b944834aaeb8903283f10ec.png" width="724"/>
    </p>
    <p>
     根据前面BCD码转十进制，我们就可以写出
    </p>
    <p>
     LCD_ShowNum(2,1,Second/16*10+Second%16,3);
    </p>
    <p>
     这样就可以在LCD上正常显示了
    </p>

    <pre><code>#include &lt;REGX52.H&gt;

//引脚定义
sbit DS1302_SCLK=P3^6;
sbit DS1302_IO=P3^4;
sbit DS1302_CE=P3^5;

#define DS1302_DATE    0x86
#define DS1302_MONTH   0x88
#define DS1302_DAY     0x8A
#define DS1302_YEAR    0x8C
#define DS1302_WP      0x8E

unsigned char DS1302_Time[]={25,03,04,12,59,55,2};

/**
  * @brief  DS1302初始化
  * @param  无

   */

	DS1302_CE=0;

/**

  * @param  Command命令字/地址
  * @param  Data要写入的数据
  * @retval 无

   */
void DS1302_WriteByte(unsigned char Command,Data)
{
	unsigned char i;
	DS1302_CE=1;
	
	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Command&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
	}

	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Data&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
	}
	DS1302_CE=0;
}

  * @brief  DS1302读一个字节
  * @param  Command命令字/地址

unsigned char DS1302_ReadByte(unsigned char Command)
{
	unsigned char i,Data=0x00;
	Command |=0x01;
	DS1302_CE=1;
	
	for(i=0;i&lt;8;i++)
	{
			DS1302_IO=Command&amp;(0x01&lt;&lt;i);  
			DS1302_SCLK=0;  
			DS1302_SCLK=1;
	}
	for(i=0;i&lt;8;i++)
	{
			DS1302_SCLK=1;  
			DS1302_SCLK=0;
			if(DS1302_IO){Data |= (0X01&lt;&lt;i);}	
	}
	DS1302_CE=0;
	DS1302_IO=0;
	return Data;
}

/**
  * @brief  DS1302设置时间，调用之后，DS1302_Time数组的数字会被设置到DS1302中
  * @param  无
  * @retval 无
  */
void DS1302_SetTime(void)
{
	DS1302_WriteByte(DS1302_WP,0x00);
	DS1302_WriteByte(DS1302_YEAR,DS1302_Time[0]/10*16+DS1302_Time[0]%10);//鍗佽繘鍒惰浆BCD鐮佸悗鍐欏叆
	DS1302_WriteByte(DS1302_MONTH,DS1302_Time[1]/10*16+DS1302_Time[1]%10);
	DS1302_WriteByte(DS1302_DATE,DS1302_Time[2]/10*16+DS1302_Time[2]%10);
	DS1302_WriteByte(DS1302_HOUR,DS1302_Time[3]/10*16+DS1302_Time[3]%10);
	DS1302_WriteByte(DS1302_MINUTE,DS1302_Time[4]/10*16+DS1302_Time[4]%10);
	DS1302_WriteByte(DS1302_SECOND,DS1302_Time[5]/10*16+DS1302_Time[5]%10);
	DS1302_WriteByte(DS1302_DAY,DS1302_Time[6]/10*16+DS1302_Time[6]%10);
	DS1302_WriteByte(DS1302_WP,0x80);
}

  * @brief  DS1302读取时间，调用之后，DS1302中的数据会被读取到DS1302_Time数组中
  * @param  无
  * @retval 无

void DS1302_ReadTime(void)
{
	unsigned char Temp;
	Temp=DS1302_ReadByte(DS1302_YEAR);
	DS1302_Time[0]=Temp/16*10+Temp%16;
	Temp=DS1302_ReadByte(DS1302_MONTH);
	DS1302_Time[1]=Temp/16*10+Temp%16;
	Temp=DS1302_ReadByte(DS1302_DATE);
	DS1302_Time[2]=Temp/16*10+Temp%16;
	Temp=DS1302_ReadByte(DS1302_HOUR);
	DS1302_Time[3]=Temp/16*10+Temp%16;
	Temp=DS1302_ReadByte(DS1302_MINUTE);
