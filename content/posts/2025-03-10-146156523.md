    <h2>
     一、思路
    </h2>
    <p>
     这题的思路很简单，其实就是一个26叉树，但是这个数不同通过通常的左右节点属性，而是一个数组来存储的，每个数组下标存储下层的数组。其中有以下需要注意的点：
    </p>
    <h3 style="background-color:transparent">
     1.private Trie[] children;
    </h3>
    <p>
     这个说明孩子节点是一个Trie类型的数组，即数组中每个下标存储的就是一个Trie结构类型，每个下标包含两个属性，即可以通过Trie.children[index]访问到子节点，Trie.children[index].isEnd访问到子节点的isEnd属性。
    </p>
    <h3>
     2.递归数据结构VS递归函数
    </h3>
    <p>
     以下是递归数据结构的核心代码，可以明显的看出和普通函数调用式递归的区别
    </p>
    <p>
     还有就是其中通过'this'来获取根类
    </p>
    <pre><code class="hljs">public void insert(String word){
            Trie node = this;//获取当前类，赋值给node
            for (int i =0;i&lt;word.length();i++){
                char letter = word.charAt(i);
                int num = letter - 'a';//获取当前字符相对于a的位置
                //注意下面代码都是对当前层的node进行操作，不要直接引用类属性，不然就是在反复修改根节点了
                if (node.children[num]==null){//当前字母对应的数组为空
                    node.children[num]=new Trie();//使用上面构造的初始化函数，向下赋值；
                }
                node = node.children[num];
            }
            node.isEnd = true;
        }</code></pre>
    <blockquote>
     <h5 style="background-color:transparent">
      <strong>
       1. 递归结构本质
      </strong>
     </h5>
     <ul>
      <li>
       <p>
        <strong>
         数据结构的递归性
        </strong>
        ：
        <br/>
        Trie 树的每个节点（
        <code>
         Trie
        </code>
        对象）包含一个子节点数组
        <code>
         children
        </code>
        ，每个子节点又是一个
        <code>
         Trie
        </code>
        实例。这种结构天然形成一棵多叉树，通过
        <strong>
         对象间的引用关系
        </strong>
        隐式构建递归层级。
       </p>
      </li>
      <li>
       <p>
        <strong>
         操作逻辑的非递归性
        </strong>
        ：
        <br/>
        代码中通过
        <strong>
         循环迭代
        </strong>
        （如
        <code>
         for
        </code>
        循环）遍历树的层级，而非显式调用自身方法。例如，
        <code>
         insert
        </code>
        方法通过移动
        <code>
         node
        </code>
        指针（
        <code>
         node = node.children[num]
        </code>
        ）逐层向下插入节点。
       </p>
      </li>
      <li>
       <p>
        <strong>
         迭代代替递归
        </strong>
        ：
        <br/>
        通过
        <code>
         node
        </code>
        指针的更新模拟递归的层级深入，但未发生函数调用栈的变化。
       </p>
      </li>
     </ul>
     <h5>
      <strong>
       2. 优点
      </strong>
     </h5>
     <ul>
      <li>
       <p>
        <strong>
         内存高效
        </strong>
        ：避免函数递归调用的栈开销，适合处理深度较大的数据结构。
       </p>
      </li>
      <li>
       <p>
        <strong>
         性能稳定
        </strong>
        ：无栈溢出风险，适合处理长字符串或大规模数据。
       </p>
      </li>
      <li>
       <p>
        <strong>
         逻辑直观
        </strong>
        ：通过对象引用直接操作树形结构，代码可读性高
       </p>
      </li>
     </ul>

    <pre><code class="hljs">public class Trie{
        private Trie[] children;
        private boolean isEnd;
        public Trie(){
            children = new Trie[26];
            isEnd = false;
        }
        public void insert(String word){
            Trie node = this;//获取当前类，赋值给node
            for (int i =0;i&lt;word.length();i++){
                char letter = word.charAt(i);
                int num = letter - 'a';//获取当前字符相对于a的位置
                //注意下面代码都是对当前层的node进行操作，不要直接引用类属性，不然就是在反复修改根节点了
                if (node.children[num]==null){//当前字母对应的数组为空
                    node.children[num]=new Trie();//使用上面构造的初始化函数，向下赋值；
                }
                node = node.children[num];
            }
            node.isEnd = true;
        }

        public boolean search(String word){
            Trie trie = searchPrefix(word);
            return trie!=null &amp;&amp; trie.isEnd;//和搜索前缀唯一的不同就是要求当前节点是结束节点
        }

        public boolean startsWith(String prefix){
            return searchPrefix(prefix)!=null;
        }

        private Trie searchPrefix(String word){
            Trie node = this;
            for (int i =0;i&lt;word.length();i++){
                char letter = word.charAt(i);
                int num = letter - 'a';//获取当前字符相对于a的位置
                if (node.children[num]==null){//当前字母对应的数组为空
                    return null;
                }
                node = node.children[num];
            }
            return node;
        }
    }</code></pre>
