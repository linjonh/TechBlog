pip 是一个现代的，通用的 Python 包管理工具 [1]  。提供了对Python 包的查找、下载、安装、卸载的功能。

官方提供的pip 示例$ pip install requests

$ pip search xml

$ pip show beautifulsoup4

$ pip uninstall requests

Python 2.7.9 + 或 Python 3.4+ 以上版本都自带 pip 工具。

可以通过以下命令来判断是否已安装：pip --version

如果还未安装，则可以使用以下方法来安装：$ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py   # 下载安装脚本

$ sudo python get-pip.py    # 运行安装脚本

注意：用哪个版本的 Python 运行安装脚本，pip 就被关联到哪个版本，如果是 Python3 则执行以下命令：$ sudo python3 get-pip.py    # 运行安装脚本。
