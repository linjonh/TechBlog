     之前会有在线的图标链接地址，让我们进行引入即可。

    <p>
     但是现在没找到，应该是得下载到本地 然后进行使用= =
    </p>
    <p>
     那我们只能将项目里的图标，选择 Symbol 并 下载至本地
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/367af236cac7a4770f3b24322a1f8482.png"/>
    </p>
    <p>
     将其放到 src\assets\svg 目录下
    </p>
    <p>
     进行解压，删除不必要的东西，只留下 iconfont.js 文件即可
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d73fc0223b81391700cba2d6fea13280.jpeg"/>
    </p>
    <p>
     之后在 main.ts 中进行全局导入
    </p>
    <pre><code>import './assets/svg/iconfont.js'
</code></pre>
    <p>
     <strong>
      项目中引入 svg-icon
     </strong>
    </p>
    <p>
     在 src 目录下，创建一个用于存放插件的目录 plugin
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b694cc85e58c759244c64a88a58904fd.jpeg"/>
    </p>
    <pre><code>// index.vue
 
&lt;template&gt;
  &lt;svg :class="svgClass" aria-hidden="true"&gt;
    &lt;use :xlink:href="iconClassName" :fill="color" /&gt;
  &lt;/svg&gt;
&lt;/template&gt;
 
&lt;script setup&gt;
import { computed } from 'vue'
const props = defineProps({
  iconName: {
    type: String,
    required: true
  },
  className: {
    type: String,
    default: ''
  },
  color: {
    type: String,
    default: '#409eff'
  }
})
// 图标在 iconfont 中的名字
const iconClassName = computed(() =&gt; {
  return `#${props.iconName}`
})
// 给图标添加上类名
const svgClass = computed(() =&gt; {
  if (props.className) {
    return `svg-icon ${props.className}`
  }
  return 'svg-icon'
})
&lt;/script&gt;
 
&lt;style scoped&gt;
.svg-icon {
  width: 1em;
  height: 1em;
  position: relative;
  fill: currentColor;
  vertical-align: -2px;
}
&lt;/style&gt;
</code></pre>
    <pre><code>// index.ts
 
import { App } from 'vue'
 
import SvgIcon from './index.vue'
 
export function setupSvgIcon(app: App) {
  app.component('SvgIcon', SvgIcon)
}
</code></pre>
