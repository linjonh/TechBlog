    <h2>
     <span style="color:#1a439c">
      1.变量定义
     </span>
    </h2>
    <p>
     对于一个日期的记录我们需要年，月，日来进行表示，所以我们定义_year,month,day，这三个变量来进行表示。
    </p>
    <p>
     我们同时还需要定义三个我文件，一个test.cpp,Date.h,Date.cpp。
    </p>
    <p>
     我们类的定义和函数声明放在Date.h头文件里面。
    </p>
    <p>
     我们函数的定义放在Date.cpp里面。
    </p>
    <p>
     实现一个函数功能后放在test.cpp里面进行测试查看是否正确。
    </p>
    <pre><code class="language-cpp">class Date
{	
private:
	int _year;
	int _month;
	int _day;
};</code></pre>
    <p>
     由于函数的成员变量是不可以直接访问的所以我们设置为私有。
    </p>
    <h2>
     <span style="color:#1a439c">
      2.功能函数（操作符重载）
     </span>
    </h2>
    <p>
     <span style="color:null">
      日期类里面的功能需要设计日期比较，日期推算，操作符+的重载，-的重载，++，--，流插入，流提取，=，-=，+=，&gt;，&lt;，&lt;=，&gt;=，!=，==，对这些操作符的重载。
     </span>
    </p>
    <h3>
     <span style="color:#6eaad7">
      2.1拷贝构造
     </span>
    </h3>
    <p>
     <span style="color:null">
      由于我们日期类的构造都是使用的内置类型没有自定义类型所以只需要使用构造函数就可以满足了
     </span>
    </p>
    <pre><code class="language-cpp">Date::Date(int year, int month, int day)
{
	_year = year;
	_month = month;
	_day = day;
}</code></pre>
    <h3>
     <span style="color:#6eaad7">
      2.2日期比较大小
     </span>
    </h3>
    <p>
     <span style="color:null">
      我们再进行日期比较大小时只需要写两个操作符重载然后进行复用就可以了。我们需要实现一个&lt;的操作符重载和一个==的操作符重载就可以了，其他的操作符重载就进行这两操作符进行复用就可以了。
     </span>
    </p>
    <pre><code class="language-cpp">bool Date::operator&lt;(const Date&amp; d)
{
	if (_year &lt; d._year)
	{
		return true;
	}
	else if (_month &lt; d._month)
	{
		return true;
	}
	else if (_day &lt; d._day)
	{
		return true;
	}
	else
	{
		return false;
	}
}
bool Date::operator&gt;(const Date&amp; d)
{
	return !(*this &lt; d) &amp;&amp; (*this != d);
}
bool Date::operator&lt;=(const Date&amp; d)
{
	return (*this &lt; d) || (*this == d);
}
bool Date::operator&gt;=(const Date&amp; d)
{
	return (*this &gt; d) || (*this == d);
}
bool Date::operator==(const Date&amp; d)
{
	return _year == d._year
		&amp;&amp; _month == d._month
		&amp;&amp; _day == d._day;
}
bool Date::operator!=(const Date&amp; d)
{
	return !(*this == d);
}

Date&amp; Date::operator=(Date&amp; d)
{
	this-&gt;_year = d._year;
	this-&gt;_month = d._month;
	this-&gt;_day = d._day;
	return *this;

}</code></pre>
    <h3>
     <span style="color:#6eaad7">
      2.3日期加减操作
     </span>
    </h3>
    <p>
     <span style="color:null">
      我们要实现一个用日期加上一个天数的方式来获得一个日期，或者减去一个日期的方式来获得一个日期。
     </span>
    </p>
    <p>
     先来实现+=
    </p>
    <pre><code class="language-cpp">Date&amp; Date::operator+=(int day)
{
	int MonthArr[] = { 0,31,28,31,30,31,30,31,31,30,31,30,31 };

	_day += day;

	while (_day &gt; MonthArr[_month])
	{
		MonthArr[2] = 28;
		if (_year % 4 == 0 &amp;&amp; _year % 100 != 0 || _year % 400 == 0)
		{
			MonthArr[2] = 29;
		}

		if (_day &gt; MonthArr[_month])
		{
			_day -= MonthArr[_month];
			_month++;
		}
		else if (_month &gt; 12)
		{
			_year++;
		}
	}

	return *this;
}
</code></pre>
    <p>
     -=：
    </p>
    <pre><code class="language-cpp">Date&amp; Date::operator-=(int day)
{
	int MonthArr[] = { 0,31,28,31,30,31,30,31,31,30,31,30,31 };

	_day -= day;

	while (_day &lt;1)
	{
		MonthArr[2] = 28;
		if (_year % 4 == 0 &amp;&amp; _year % 100 != 0 || _year % 400 == 0)
		{
			MonthArr[2] = 29;
		}
		if (_day &lt; 1)
		{
			_month--;
			_day += MonthArr[_month];			
		}
		else if (_month &lt; 1)
		{
			_year--;
		}
	}

	return *this;
}
</code></pre>
    <p>
     我们有了+=和-=来实现加就太容易了，可以直接套用+=，-=。我们要获得一个日期减后（加后）的一个新日期我们的返回值就不能再使用引用了的方式了，这样的话我们的新变量只是以前的一个别名，会改变以前的那个变量。所以会使用他的拷贝构造。
    </p>
    <p>
     +和-：
    </p>
    <pre><code class="language-cpp">Date Date:: operator+(int day)
{
	Date temp = *this;
	temp += day;
	return temp;
}

Date Date:: operator-(int day)
{
	Date temp = *this;
	temp -= day;
	return temp;
}
</code></pre>
    <h3>
     <span style="background-color:null">
      <span style="color:#6eaad7">
       2.4++和--的操作
      </span>
     </span>
    </h3>
    <p>
     <span style="background-color:null">
      <span style="color:null">
       我们有了+=可以使用+=来实现++，就是对this*+=1，但是我们这样实现出来的是前置++，要想实现后置++就需要在后面函数传参的地方加一个int。这样才可以实现后置++。--是同理的。
      </span>
     </span>
    </p>
    <pre><code class="language-cpp">Date&amp; Date:: operator++()
{
	return (*this) += 1;
}

Date&amp; Date:: operator--()
{
	return (*this) -= 1;
}

Date Date::operator++(int)
{
	Date temp = *this;
	*this += 1;
	return temp;
}

Date Date::operator--(int)
{
	Date temp = *this;
	*this -= 1;
	return temp;
}</code></pre>
    <h3>
     <span style="color:#6eaad7">
      2.5流插入，流提取操作符重载
     </span>
    </h3>
    <p>
     我们需要对类里面的私有变量进行操作的时候我们需要流插入流提取的变量，但是放在类里面会将函数的this指针当作是第一个变量，我们把函数放在类外面的时候有访问不到私有变量。我们就要使用友元函数来对其声明。
    </p>
    <pre><code class="language-cpp">ostream&amp; operator&lt;&lt;(ostream&amp; out , const Date&amp; d)
{
	out &lt;&lt; d._year &lt;&lt; "" &lt;&lt; d._month &lt;&lt; "" &lt;&lt; d._day &lt;&lt; "" &lt;&lt; endl;
	return out;
}

istream&amp; operator&gt;&gt;(istream&amp; in, Date&amp; d)
{
	in &gt;&gt; d._year &gt;&gt; d._month &gt;&gt; d._day ;
	return in;
}</code></pre>
