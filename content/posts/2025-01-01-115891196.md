# 鸿蒙HarmonyOS DevEco Studio的安装教程 和 第一个项目

HarmonyOS DevEco Studio下载地址：

```
https://developer.harmonyos.com/cn/develop/deveco-studio#download

```

![image.png](https://i-blog.csdnimg.cn/blog_migrate/67af4104393cf668abb943121f619316.png)

下载按步骤安装就可。

创建一个project:

首先安装好SDK,比如没有安装好js,那创建不了js的project。

![image.png](https://i-blog.csdnimg.cn/blog_migrate/3b0e9286e374e6336d8da0a4192f8b58.png)

现在选一个手机的Java project

![image.png](https://i-blog.csdnimg.cn/blog_migrate/57acdb28f3532a690cca73912cd5e0b9.png)

创建一个虚拟手机，在菜单Tools->HVD Manager

![image.png](https://i-blog.csdnimg.cn/blog_migrate/1360d995f8195392004c4c27d2734842.png)

需要登录华为账号

![image.png](https://i-blog.csdnimg.cn/blog_migrate/008cc7f98a584dd367b1791b69176916.png)

右上角直接运行

![image.png](https://i-blog.csdnimg.cn/blog_migrate/777f529813a5496623cdf5bdf7fdffcc.png)

发现报了个错，应该是SDK版本不对，可能是旧的SDK没清除的原因,把releaseType删除就可。

```
Session 'com.example.myapplication1': Error Deploying HAP

```

![image.png](https://i-blog.csdnimg.cn/blog_migrate/d3bce9e51f0443127baa63dc6512d6fc.png)

运行成功
