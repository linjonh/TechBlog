     <li>

      <strong>
       轮廓系数

      结合
      <strong>
       肘部法则
      </strong>
      和业务理解
     </li>
     <li>
      尝试
      <strong>
       Gap Statistic
      </strong>
      等高级方法
     </li>
    </ul>
    <h5>
     <a id="Q2_224">
     </a>
     <strong>
      Q2：如何处理非球形数据？
     </strong>
    </h5>
    <ul>
     <li>
      使用
      <strong>
       谱聚类
      </strong>
      或
      <strong>
       DBSCAN
      </strong>
      替代
     </li>
     <li>
      通过
      <strong>
       核方法
      </strong>
      将数据映射到高维空间
      <pre><code class="prism language-python"><span class="token keyword">from</span> sklearn<span class="token punctuation">.</span>cluster <span class="token keyword">import</span> SpectralClustering
