    <h3>
     <a id="_1">
     </a>
     概要
    </h3>
    <p>
     在小程序中，您可以通过调用企业微信的接口来实现客服功能。
    </p>
    <h3>
     <a id="_10">
     </a>
     整体流程
    </h3>
    <h4>
     1. 创建企业微信应用
    </h4>
    <ul>
     <li>
      登录企业微信管理后台：
      <a class="link-info" href="https://work.weixin.qq.com/" rel="nofollow" title="企业微信管理后台">
       企业微信管理后台
      </a>
     </li>
     <li>
      在左侧菜单中选择“应用管理” -&gt; “添加应用”，根据提示创建一个新的应用。
     </li>
     <li>
      在管理后台的首页，可以找到企业的基本信息，其中就包括 corpId
     </li>
    </ul>
    <h4>
     2. 获取应用的相关信息
    </h4>
    <p>
     在创建应用后，记录以下信息：
    </p>
    <ul>
     <li>
      <strong>
       Agent ID
      </strong>
      ：应用的唯一标识。
     </li>
     <li>
      <strong>
       Secret
      </strong>
      ：应用的凭证，用于调用接口时进行身份验证。
     </li>
    </ul>
    <h4>
     3. 配置小程序的服务器域名
    </h4>
    <p>
     <a class="link-info" href="https://developers.weixin.qq.com/miniprogram/dev/api/open-api/service-chat/wx.openCustomerServiceChat.html" rel="nofollow" title="微信小程序官方文档">
      微信小程序官方文档
     </a>
    </p>
    <ul>
     <li>
      在小程序管理后台中，找到“开发” -&gt; “开发设置”，设置合法的请求域名，通常需要配置你的服务器域名。
     </li>
    </ul>
    <h4>
     4. 调用企业微信客服接口
    </h4>
    <p>
     <a class="link-info" href="https://developer.work.weixin.qq.com/document/path/97013" rel="nofollow" title="企业微信开发者官方文档">
      企业微信开发者官方文档
     </a>
    </p>
    <p>
     在小程序中，您可以通过调用企业微信的接口来实现客服功能。以下是一个基本的示例，演示如何发送消息到企业微信客服。
    </p>
    <h3>
     <a id="_16">
     </a>
     调用api代码
    </h3>
    <pre><code class="language-javascript">&lt;template&gt;
  &lt;button @click="contactCustomerService"&gt;联系客服&lt;/button&gt;
&lt;/template&gt;

&lt;script&gt;
export default {
  methods: {
    contactCustomerService() {
      wx.openCustomerServiceChat({
        extInfo: {
          url: `${this.item.path}`,
        },
        //在管理后台的首页，可以找到企业的基本信息，其中就包括 corpId
        corpId: '',
        success: (res) =&gt; {
          if (res.errMsg === 'openCustomerServiceChat:ok') {
            this.$toast('联系客服成功', { type: 'success' });
          }
        },
        fail: (err) =&gt; {
          this.$toast('联系客服失败', { type: 'error', content: `${err}` });
        },
      });
    },
  },
  data() {
    return {
      item: {
        path: 'https://example.com', // 具体的 URL
      },
    };
  },
};
&lt;/script&gt;
</code></pre>
