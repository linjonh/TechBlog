    <p>
     Windows系统最大的特点就是可视化操作，点点鼠标就能操作软件。但是在特殊的情况下，比如docker desktop图标点了之后没反应，但是看后台程序，它又已经运行了，这时候就要使用命令行来操作了。
    </p>
    <p>
     针对这次情况，所做的操作：先尝试在任务管理器里找到docker desktop，结束任务。但是后台任务还在执行。后来是结束了docker desktop backend，才彻底结束了它。然后再点图标启动，终于正常了。
    </p>
    <p>
     尽管现在已经可以用视图操作，还是来学习实践一下命令行的操作。
    </p>

    <h3>
     docker启动
     <strong>
      容器
     </strong>
     docker run
    </h3>
    <p>
     在powershell下，跟Linux下不同，最后的连接符，用的是"`" 而不是"\" 。而在cmd下，使用
     <code>
      ^
     </code>
     符号作为换行符，下面以powershell下为例：
    </p>
    <pre><code>docker run -it --rm --pull=always `
    -e SANDBOX_RUNTIME_CONTAINER_IMAGE=docker.all-hands.dev/all-hands-ai/runtime:0.28-nikolaik `
    -e LOG_ALL_EVENTS=true  `
    -v C:\Users\Admin\.openhands-state:/.openhands-state  `
    -p 3000:3000  `
    --add-host host.docker.internal:host-gateway  `
    --name openhands-app  `
    docker.all-hands.dev/all-hands-ai/openhands:0.28
</code></pre>
    <p>
     比如我们启动一个简单的web服务，就既可以在cmd下，也可以在powershell下使用：
    </p>
    <pre><code>docker run tomcat</code></pre>
    <h3>
     docker启动
     <strong>
      容器
     </strong>
     docker start
    </h3>
    <p>
     start和run的区别是，start之后会退回到终端。
    </p>
    <pre><code>docker start busy_galileo
busy_galileo
</code></pre>
    <h3>
     查看启动的
     <strong>
      容器
     </strong>
     docker ps
    </h3>
    <pre><code>docker ps

    <h3>
     关闭
     <strong>
      容器
     </strong>
     docker stop
    </h3>
    <pre><code>docker stop busy_galileo

2eb731da3231   4328059557e8   "/main"   4 minutes ago   Up 4 minutes             vigorous_goodall
</code></pre>
    <h3>
     <strong>
      进入正在运行的容器
     </strong>
     ：
    </h3>
    <pre><code>docker exec -it &lt;container_id&gt; /bin/bash</code></pre>
    <p>
     比如用docker ps列出当前运行容器，然后可以根据容器id进去。
    </p>

    </p>
    <h3>
     删除容器docker rm
    </h3>

    <h3>
     关于docker服务相关
    </h3>
    <h4>
     查看docker服务的名字
    </h4>
    <p>
     在cmd命令行下：
    </p>
    <pre><code>sc queryex type=service state=all | findstr "docker"
SERVICE_NAME: com.docker.service</code></pre>
    <h4>
     启动和关闭服务
    </h4>
    <p>
     在powershell下
    </p>
    <pre><code>PS C:\Windows\system32&gt; Start-Service -Name com.docker.service
PS C:\Windows\system32&gt; Stop-Service -Name com.docker.service</code></pre>
    <p>
     但是测试下来好像并没有生效。
    </p>
    <p>
     在以管理员启动的cmd下：
    </p>
    <pre><code>C:\Windows\system32&gt;net start com.docker.service
Docker Desktop Service 服务正在启动 .
Docker Desktop Service 服务已经启动成功。


C:\Windows\system32&gt;net stop com.docker.service
Docker Desktop Service 服务正在停止.
Docker Desktop Service 服务已成功停止。</code></pre>
    <p>
     启动和停止服务显示是生效的，好像并没有生效。
