    <h3>
     1. 进制转换
    </h3>
    <p>
     计算机中的数据通常以二进制形式存储，但在编程和调试过程中，我们经常需要与十进制、八进制和十六进制打交道。因此，掌握进制转换是C语言学习中的重要一环。
    </p>
    <hr/>
    <h4>
     1.1 进制的基本概念
    </h4>
    <ul>
     <li>
      <strong>
       二进制（Binary）
      </strong>
      ：
      <span style="background-color:#edf6e8">
       基数为2
      </span>
      ，使用数字0和1。
     </li>
     <li>
      <strong>
       八进制（Octal）
      </strong>
      ：
      <span style="background-color:#edf6e8">
       基数为8
      </span>
      ，使用数字0到7。
     </li>
     <li>
      <strong>
       十进制（Decimal）
      </strong>
      ：
      <span style="background-color:#edf6e8">
       基数为10
      </span>
      ，使用数字0到9。
     </li>
     <li>
      <strong>
       十六进制（Hexadecimal）
      </strong>
      ：
      <span style="background-color:#edf6e8">
       基数为16
      </span>
      ，使用数字0到9和字母A到F。
     </li>
    </ul>
    <hr/>
    <h4>
     1.2 进制转换方法
    </h4>
    <h5>
     1.2.1 二进制转十进制
    </h5>
    <p>
     将二进制数的每一位乘以
     <span style="background-color:#eaf4fc">
      2的幂次方
     </span>
     ，然后相加。
    </p>
    <p>
     将二进制数​
     <code>
      ​1011​
     </code>
     ​​转换为十进制：
    </p>
    <pre><code class="language-cs">1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 1 * 2^0
= 8 + 0 + 2 + 1
= 11</code></pre>
    <h5>
     1.2.2 十进制转二进制
    </h5>
    <p>
     通过
     <span style="background-color:#eaf4fc">
      不断除以2并记录余数
     </span>
     ，直到商为0，然后将余数倒序排列。
    </p>
    <p>
     将十进制数​
     <code>
      ​11​
     </code>
     ​​转换为二进制：
    </p>
    <pre><code class="language-cs">11 / 2 = 5 余 1
5 / 2 = 2 余 1
2 / 2 = 1 余 0
1 / 2 = 0 余 1</code></pre>
    <p>
     倒序排列余数：​
     <code>
      ​1011​
     </code>
     ​。
    </p>
    <hr/>
    <h5>
     1.2.3 二进制转八进制
    </h5>
    <p>
     将二进制数从右到左
     <span style="background-color:#eaf4fc">
      每三位一组
     </span>
     ，不足三位补零，然后将每组转换为对应的八进制数。
    </p>
    <p>
     将二进制数​
     <code>
      ​101110​
     </code>
     ​​转换为八进制：
    </p>
    <pre><code class="language-cs">101 110
5   6</code></pre>
    <p>
     结果为​
     <code>
      ​56​
     </code>
     ​。
    </p>
    <hr/>
    <h5>
     1.2.4 二进制转十六进制
    </h5>
    <p>
     将二进制数从右到左
     <span style="background-color:#eaf4fc">
      每四位一组
     </span>
     ，不足四位补零，然后将每组转换为对应的十六进制数。
    </p>
    <p>
     将二进制数​
     <code>
      ​101110​
     </code>
     ​​转换为十六进制：
    </p>
    <pre><code class="language-cs">0010 1110
2    E</code></pre>
    <p>
     结果为​
     <code>
      ​2E​
     </code>
     ​。
    </p>
    <hr/>
    <h5>
     1.2.5 十进制转十六进制
    </h5>
    <p>
     通过不断
     <span style="background-color:#eaf4fc">
      除以16并记录余数
     </span>
     ，直到商为0，然后将余数倒序排列。
    </p>
    <p>
     将十进制数​
     <code>
      ​255​
     </code>
     ​转换为十六进制：
    </p>
    <pre><code class="language-cs">255 / 16 = 15 余 15 (F)
15 / 16 = 0 余 15 (F)</code></pre>
    <p>
     倒序排列余数：​
     <code>
      ​FF​
     </code>
     ​。
    </p>
    <hr/>
    <h3>
     2. 反码、补码和原码
    </h3>
    <p>
     <span style="background-color:#efedf6">
      计算机可以区分整数的正负，但是程序最终都会被转换为二进制指令，二进制可以分正负，在二进制数的前面添加1bit，作为符号位，并且bit=0，用于表示二进制数为正数，bit=1，用于表示二进制数为负数。
     </span>
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="814" src="https://i-blog.csdnimg.cn/img_convert/c3275036853593d4bfacb95f59628f1a.png" width="796"/>
    </p>
    <p>
     可以知道，计算机内部是以二进制的补码方式来存储数据的，所以需要
     <span style="background-color:#efedf6">
      把二进制数的原码转换为补码
     </span>
     形式。
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="238" src="https://i-blog.csdnimg.cn/img_convert/f7e59749f2d1a73ea1800082380596a8.png" width="864"/>
    </p>
    <p>
     可以知道，正数的原码和补码是相同的，所以
     <span style="background-color:#efedf6">
      负数才需要把原码转换为补码
     </span>
     ，再进行存储！
    </p>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="519" src="https://i-blog.csdnimg.cn/img_convert/6084525693f4dc5e11176e1b1657d09b.png" width="865"/>
    </p>
    <p>
    </p>
    <hr/>
    <h4>
     2.1 原码
    </h4>
    <p>
     原码是二进制数的最基本表示形式，最高位为符号位（0表示正数，1表示负数），其余位表示数值。
    </p>
    <ul>
     <li>
      ​
      <code>
       ​+5​
      </code>
      ​的原码：​
      <code>
       ​00000101​
      </code>
      ​
     </li>
     <li>
      ​
      <code>
       ​-5​
      </code>
      ​的原码：​
      <code>
       ​10000101​
      </code>
      ​
     </li>
    </ul>
    <p>
     原码表示法存在以下问题：
    </p>
    <ul>
     <li>
      0有两种表示形式：​
      <code>
       ​00000000​
      </code>
      ​（+0）和​
      <code>
       ​10000000​
      </code>
      ​（-0）。
     </li>
     <li>
      加减法运算复杂，需要额外的硬件支持。
     </li>
    </ul>
    <hr/>
    <h4>
     2.2 反码
    </h4>
    <p>
     反码是对原码的改进，正数的反码与原码相同，负数的反码是对其原码的符号位不变，其余位取反。
    </p>
    <ul>
     <li>
      ​
      <code>
       ​+5​
      </code>
      ​的反码：​
      <code>
       ​00000101​
      </code>
      ​
     </li>
     <li>
      ​
      <code>
       ​-5​
      </code>
      ​的反码：​
      <code>
       ​11111010​
      </code>
      ​
     </li>
    </ul>
    <p>
     反码表示法仍然存在以下问题：
    </p>
    <ul>
     <li>
      0有两种表示形式：​
      <code>
       ​00000000​
      </code>
      ​（+0）和​
      <code>
       ​11111111​
      </code>
      ​（-0）。
     </li>
     <li>
      加减法运算仍然复杂。
     </li>
    </ul>
    <hr/>
    <h4>
     2.3 补码
    </h4>
    <p>
     补码是计算机中表示有符号整数的标准方式。正数的补码与原码相同，负数的补码是其反码加1。
    </p>
    <ul>
     <li>
      ​
      <code>
       ​+5​
      </code>
      ​的补码：​
      <code>
       ​00000101​
      </code>
      ​
     </li>
     <li>
      ​
      <code>
       ​-5​
      </code>
      ​的补码：
     </li>
     <li>
      原码：​
      <code>
       ​10000101​
      </code>
      ​
     </li>
     <li>
      反码：​
      <code>
       ​11111010​
      </code>
      ​
     </li>
     <li>
      补码：​
      <code>
       ​11111011​
      </code>
      ​
     </li>
     <li>
      0只有一种表示形式：​
      <code>
       ​00000000​
      </code>
      ​。
     </li>
     <li>
      加减法运算统一，可以直接使用加法器实现。
     </li>
    </ul>
    <p>
     <strong>
      补码的运算
     </strong>
     ：
     <br/>
     补码的一个重要特性是，一个数的补码的补码等于其原码。因此，补码可以方便地用于表示负数和进行减法运算。
    </p>
    <p>
     计算​
     <code>
      ​5 - 3​
     </code>
     ​：
    </p>
    <ul>
     <li>
      ​
      <code>
       ​5​
      </code>
      ​的补码：​
      <code>
       ​00000101​
      </code>
      ​
     </li>
     <li>
      ​
      <code>
       ​-3​
      </code>
      ​的补码：​
      <code>
       ​11111101​
      </code>
      ​
     </li>
     <li>
      相加：​
      <code>
       ​00000101 + 11111101 = 00000010​
      </code>
      ​（结果为​
      <code>
       ​2​
      </code>
      ​）。
     </li>
    </ul>
    <p>
     <span style="background-color:#efedf6">
      注意
     </span>
     ：
     <span style="color:#fe2c24">
      设计程序时，定义的变量如果是有符号的，则尽量避免写入超过变量范围的数值!!!!!!
     </span>
    </p>
    <p>
     <span style="color:#fe2c24">
      字符型： char  1字节 -- 有符号 -- 数值范围 -128 ~ 127  --  无符号 -- 数值范围 0 ~ 255
     </span>
    </p>
    <p>
     <span style="color:#fe2c24">
      短整型：short  2字节 -- 有符号 -- 数值范围 -32768 ~ 32767 无符号 --数值范围 0 ~ 65535
     </span>
    </p>
    <hr/>
    <h3>
     3. C语言中的进制表示与位操作
    </h3>
    <h4>
     3.1 进制表示
    </h4>
    <ul>
     <li>
      <strong>
       二进制
      </strong>
      ：C语言不支持直接表示二进制数，但可以通过前缀​
      <code>
       ​0b​
      </code>
      ​或​
      <code>
       ​0B​
      </code>
      ​表示（某些编译器支持）。
     </li>
     <li>
      <strong>
       八进制
      </strong>
      ：以​
      <code>
       ​0​
      </code>
      ​开头，如​
      <code>
       ​012​
      </code>
      ​表示十进制的​
      <code>
       ​10​
      </code>
      ​。
     </li>
     <li>
      <strong>
       十六进制
      </strong>
      ：以​
      <code>
       ​0x​
      </code>
      ​或​
      <code>
       ​0X​
      </code>
      ​开头，如​
      <code>
       ​0x1A​
      </code>
      ​表示十进制的​
      <code>
       ​26​
      </code>
      ​。
     </li>
    </ul>
    <pre><code class="language-cs">#include &lt;stdio.h&gt;

int main() {
    int binary = 0b1010;  // 二进制，值为10
    int octal = 012;      // 八进制，值为10
    int hexadecimal = 0x1A;  // 十六进制，值为26
    printf("Binary: %d\n", binary);
    printf("Octal: %d\n", octal);
    printf("Hexadecimal: %d\n", hexadecimal);
    return 0;
}</code></pre>
    <p>
     <strong>
      输出
     </strong>
     ：
    </p>
    <pre><code class="language-cs">Binary: 10
Octal: 10
Hexadecimal: 26</code></pre>
    <hr/>
    <h4>
     3.2 位操作
    </h4>
    <p>
     C语言提供了以下位操作运算符：
    </p>
    <ul>
     <li>
      <strong>
       按位与（
      </strong>
      ​
      <code>
       ​
       <strong>
        &amp;
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：对应位都为1时结果为1，否则为0。
     </li>
     <li>
      <strong>
       按位或（
      </strong>
      ​
      <code>
       ​
       <strong>
        |
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：对应位有一个为1时结果为1，否则为0。
     </li>
     <li>
      <strong>
       按位异或（
      </strong>
      ​
      <code>
       ​
       <strong>
        ^
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：对应位不同时结果为1，否则为0。
     </li>
     <li>
      <strong>
       按位取反（
      </strong>
      ​
      <code>
       ​
       <strong>
        ~
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：将所有位取反。
     </li>
     <li>
      <strong>
       左移（
      </strong>
      ​
      <code>
       ​
       <strong>
        &lt;&lt;
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：将二进制数向左移动指定位数，低位补0。
     </li>
     <li>
      <strong>
       右移（
      </strong>
      ​
      <code>
       ​
       <strong>
        &gt;&gt;
       </strong>
       ​
      </code>
      ​
      <strong>
       ）
      </strong>
      ：将二进制数向右移动指定位数，高位补符号位（算术右移）或0（逻辑右移）。
     </li>
    </ul>
    <pre><code class="language-cs">#include &lt;stdio.h&gt;

int main() {
    int a = 5;  // 二进制：00000101
    int b = 3;  // 二进制：00000011

    printf("a &amp; b: %d\n", a &amp; b);  // 00000001 (1)
    printf("a | b: %d\n", a | b);  // 00000111 (7)
    printf("a ^ b: %d\n", a ^ b);  // 00000110 (6)
    printf("~a: %d\n", ~a);        // 11111010 (-6，补码表示)
    printf("a &lt;&lt; 1: %d\n", a &lt;&lt; 1);  // 00001010 (10)
    printf("a &gt;&gt; 1: %d\n", a &gt;&gt; 1);  // 00000010 (2)
    return 0;
}</code></pre>
    <p>
     <strong>
      输出
     </strong>
     ：
    </p>
    <pre><code class="language-cs">a &amp; b: 1
a | b: 7
a ^ b: 6
~a: -6
a &lt;&lt; 1: 10
a &gt;&gt; 1: 2</code></pre>
