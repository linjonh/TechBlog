    <h2>
     <a id="Photoshop__0">
     </a>
     Photoshop 中如何快速抠图？
    </h2>
    <p>
     抠图是 Photoshop 中的常见操作，无论是去除背景还是提取特定对象，都需要掌握高效的抠图技巧。本文将介绍几种快速抠图的方法，帮助你轻松完成设计任务。
    </p>
    <hr/>
    <h3>
     <a id="1_Quick_Selection_Tool_6">
     </a>
     1. 快速选择工具（Quick Selection Tool）
    </h3>
    <h4>
     <a id="_8">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      打开图片，选择
      <strong>
       快速选择工具
      </strong>
      （快捷键：W）。
     </li>
     <li>
      在工具选项栏中调整画笔大小。
     </li>
     <li>
      在需要选择的区域点击并拖动，工具会自动识别边缘。
     </li>
     <li>
      按住
      <strong>
       Alt
      </strong>
      键可以减去多余的选择区域。
     </li>
    </ol>
    <h4>
     <a id="_14">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      背景与主体对比明显的图片。
     </li>
     <li>
      简单的几何形状或轮廓清晰的物体。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="2_Magic_Wand_Tool_20">
     </a>
     2. 魔棒工具（Magic Wand Tool）
    </h3>
    <h4>
     <a id="_22">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      选择
      <strong>
       魔棒工具
      </strong>
      （快捷键：W）。
     </li>
     <li>
      在工具选项栏中设置容差值（Tolerance），容差值越高，选择的颜色范围越广。
     </li>
     <li>
      点击需要选择的区域，工具会自动选择相似颜色区域。
     </li>
     <li>
      按住
      <strong>
       Shift
      </strong>
      键可以增加选择区域，按住
      <strong>
       Alt
      </strong>
      键可以减去选择区域。
     </li>
    </ol>
    <h4>
     <a id="_28">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      背景颜色单一或对比明显的图片。
     </li>
     <li>
      快速选择大面积相似颜色区域。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="3_Object_Selection_Tool_34">
     </a>
     3. 对象选择工具（Object Selection Tool）
    </h3>
    <h4>
     <a id="_36">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      选择
      <strong>
       对象选择工具
      </strong>
      （快捷键：W）。
     </li>
     <li>
      在工具选项栏中选择模式（矩形或套索）。
     </li>
     <li>
      拖动鼠标框选需要抠图的对象，工具会自动识别对象边缘。
     </li>
     <li>
      按住
      <strong>
       Shift
      </strong>
      键可以增加选择区域，按住
      <strong>
       Alt
      </strong>
      键可以减去选择区域。
     </li>
    </ol>
    <h4>
     <a id="_42">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      复杂背景中的主体对象。
     </li>
     <li>
      需要快速选择特定对象的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="4_Pen_Tool_48">
     </a>
     4. 钢笔工具（Pen Tool）
    </h3>
    <h4>
     <a id="_50">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      选择
      <strong>
       钢笔工具
      </strong>
      （快捷键：P）。
     </li>
     <li>
      在需要抠图的物体边缘点击创建锚点，逐步绘制路径。
     </li>
     <li>
      闭合路径后，右键选择
      <strong>
       “建立选区”
      </strong>
      。
     </li>
     <li>
      设置羽化半径（Feather Radius），点击
      <strong>
       “确定”
      </strong>
      。
     </li>
    </ol>
    <h4>
     <a id="_56">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      需要高精度抠图的场景。
     </li>
     <li>
      复杂边缘或细节丰富的物体。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="5_Select_and_Mask_62">
     </a>
     5. 选择并遮住（Select and Mask）
    </h3>
    <h4>
     <a id="_64">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      使用任意选择工具（如快速选择工具）初步选择对象。
     </li>
     <li>
      点击工具选项栏中的
      <strong>
       “选择并遮住”
      </strong>
      按钮。
     </li>
     <li>
      在面板中使用
      <strong>
       边缘检测
      </strong>
      、
      <strong>
       平滑
      </strong>
      、
      <strong>
       羽化
      </strong>
      等工具优化选区。
     </li>
     <li>
      点击
      <strong>
       “确定”
      </strong>
      完成抠图。
     </li>
    </ol>
    <h4>
     <a id="_70">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      毛发、透明物体等复杂边缘。
     </li>
     <li>
      需要精细调整选区的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="6_Channel_Masking_76">
     </a>
     6. 通道抠图（Channel Masking）
    </h3>
    <h4>
     <a id="_78">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      打开
      <strong>
       通道面板
      </strong>
      （Window &gt; Channels）。
     </li>
     <li>
      选择一个对比度最高的通道（通常是蓝色或红色通道）。
     </li>
     <li>
      复制该通道，使用
      <strong>
       曲线
      </strong>
      或
      <strong>
       色阶
      </strong>
      增强对比度。
     </li>
     <li>
      使用画笔工具将需要保留的区域涂成白色，背景涂成黑色。
     </li>
     <li>
      按住
      <strong>
       Ctrl
      </strong>
      键点击通道缩略图，生成选区。
     </li>
     <li>
      返回图层面板，复制选区内容。
     </li>
    </ol>
    <h4>
     <a id="_86">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      透明物体（如玻璃、烟雾）。
     </li>
     <li>
      复杂背景中的半透明物体。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="7_Layer_Mask_92">
     </a>
     7. 图层蒙版（Layer Mask）
    </h3>
    <h4>
     <a id="_94">
     </a>
     使用步骤
    </h4>
    <ol>
     <li>
      使用任意选择工具选择需要保留的区域。
     </li>
     <li>
      点击图层面板底部的
      <strong>
       “添加图层蒙版”
      </strong>
      按钮。
     </li>
     <li>
      使用画笔工具在蒙版上涂抹，黑色隐藏，白色显示。
     </li>
     <li>
      调整画笔硬度和不透明度，优化边缘。
     </li>
    </ol>
    <h4>
     <a id="_100">
     </a>
     适用场景
    </h4>
    <ul>
     <li>
      非破坏性编辑，保留原图。
     </li>
     <li>
      需要反复调整选区的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_106">
     </a>
     注意事项
    </h3>
    <ol>
     <li>
      <strong>
       选择合适的工具
      </strong>
      ：根据图片特点选择最合适的抠图工具。
     </li>
     <li>
      <strong>
       优化选区边缘
      </strong>
      ：使用羽化、平滑等功能优化选区边缘。
     </li>
     <li>
      <strong>
       保存原图
      </strong>
      ：抠图前复制图层，避免破坏原图。
     </li>
    </ol>
