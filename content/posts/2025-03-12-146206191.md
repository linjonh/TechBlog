    <h4>
     <a id="font_stylecolorrgba0_0_0_087Homebrew_font_0">
     </a>
     通过Homebrew 安装（推荐）
    </h4>
    <p>
     打开mac终端：
    </p>
    <h5>
     <a id="font_stylecolorrgba0_0_0_0871__Homebrewfont_2">
     </a>
     1. 安装 Homebrew（如果尚未安装）
    </h5>
    <pre><code class="prism language-bash">/bin/bash <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <h5>
     <a id="font_stylecolorrgba0_0_0_0872__Neo4jfont_7">
     </a>
     2. 安装 Neo4j
    </h5>
    <pre><code class="prism language-bash">brew <span class="token function">install</span> neo4j
</code></pre>
    <h5>
     <a id="font_stylecolorrgba0_0_0_0873__Neo4j_font_12">
     </a>
     3. 启动 Neo4j 服务
    </h5>
    <pre><code class="prism language-bash">brew services start neo4j  <span class="token comment"># 后台运行</span>
<span class="token comment"># 或</span>
neo4j console              <span class="token comment"># 前台运行（查看实时日志）</span>
</code></pre>
    <h5>
     <a id="4__Neo4j_font_19">
     </a>
     4. 访问 Neo4j 控制台
    </h5>
    <ul>
     <li>
      打开浏览器访问：http://localhost:7474
     </li>
     <li>
      默认登录凭证：
      <ul>
       <li>
        用户名：neo4j
       </li>
       <li>
        初始密码：neo4j
        <br/>
        （首次登录后会强制修改）
       </li>
      </ul>
     </li>
    </ul>
