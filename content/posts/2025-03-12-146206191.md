# Mac安装Neo4j图数据库

#### 通过Homebrew 安装（推荐）

打开mac终端：

##### 1. 安装 Homebrew（如果尚未安装）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```

##### 2. 安装 Neo4j

```bash
brew install neo4j

```

##### 3. 启动 Neo4j 服务

```bash
brew services start neo4j  # 后台运行
# 或
neo4j console              # 前台运行（查看实时日志）

```

##### 4. 访问 Neo4j 控制台

* 打开浏览器访问：http://localhost:7474
* 默认登录凭证：
  + 用户名：neo4j
  + 初始密码：neo4j
      
    （首次登录后会强制修改）
