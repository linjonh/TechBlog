    <h3>
     <a id="_2">
     </a>
     分类问题的评价
    </h3>
    <p>
     对于1个分类问题，不论其是
     <strong>
      二分类问题
     </strong>
     还是
     <strong>
      多分类问题
     </strong>
     ，或者是
     <strong>
      多标签分类问题
     </strong>
     （1个样本可以有多个类别），都可以对其判定结果进行统计，以类别为单位统计其各个指标，来衡量模型对该类别的预测情况。基于对某个分类的4种分类判别情况，引出了精确率、召回率、准确率和F1 Score的评价指标。
    </p>
    <h3>
     <a id="_4">
     </a>
     分类问题对特定样本类别的判定情况
    </h3>
    <p>
     对特定样本类别A分类判定的4种情况
    </p>
    <ul>
     <li>
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         T 
         
        
          P 
         
        
       
         TP
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 0.6833em;">
          </span>
          <span class="mord mathnormal" style="margin-right: 0.1389em;">
           TP
          </span>
         </span>
        </span>
       </span>
      </span>
      （True Positive） 真阳性，某条样本判定为A类样本，实际也为A类样本
     </li>
     <li>
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         F 
         
        
          P 
         
        
       
         FP
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 0.6833em;">
          </span>
          <span class="mord mathnormal" style="margin-right: 0.1389em;">
           FP
          </span>
         </span>
        </span>
       </span>
      </span>
      （False Positive）假阳性，某条样本判定为A类样本，但实际不为A类样本
     </li>
     <li>
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         T 
         
        
          N 
         
        
       
         TN
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 0.6833em;">
          </span>
          <span class="mord mathnormal" style="margin-right: 0.109em;">
           TN
          </span>
         </span>
        </span>
       </span>
      </span>
      （True Negative）真阴性，某条样本判定不为A类样本，实际也不为A类样本
     </li>
     <li>
      <span class="katex--inline">
       <span class="katex">
        <span class="katex-mathml">
         F 
         
        
          N 
         
        
       
         FN
        </span>
        <span class="katex-html">
         <span class="base">
          <span class="strut" style="height: 0.6833em;">
          </span>
          <span class="mord mathnormal" style="margin-right: 0.109em;">
           FN
          </span>
         </span>
        </span>
       </span>
      </span>
      （False Negative）假阴性，某条样本判定不为A类样本，但实际为A类样本
     </li>
    </ul>
    <p>
     在其中T表示预测结果是正确的，F表示预测结果是错误的，P表示预测样本为A类别，N表示预测样本不为A类别。
    </p>
    <h3>
     <a id="Confusion_Matrix_12">
     </a>
     混淆矩阵（Confusion Matrix）
    </h3>
    <p>
     混淆矩阵以表格形式，展示了模型对A类样本的预测情况，表格为2×2的尺寸，以
     <strong>
      是否预测是否为A类样本
     </strong>
     和
     <strong>
      真实是否为A类样本
     </strong>
     2个维度进行列举。
    </p>
    <table>
     <thead>
      <tr>
       <th>
       </th>
       <th>
        真实为A类
       </th>
       <th>
        真实非A类
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        预测为A类
       </td>
       <td>
        TP
       </td>
       <td>
        FP
       </td>
      </tr>
      <tr>
       <td>
        预测非A类
       </td>
       <td>
        FN
       </td>
       <td>
        TN
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     其中T表示预测正确，F表示预测错误，P表示预测为A类样本，N表示预测为非A类样本。
    </p>
    <p>
     这其中需要注意的是FN类别，这个类别表示预测样本不是A类别（N），但是预测错误了（F），因此这个类别实际上是属于A类别的。
    </p>

    </p>
