     在想要的目录下，执行如下指令（前提当然是需要安装了python）：
     <br/>
     <br/>
    </p>
    <pre><code class="language-python">python -m CGIHTTPServer</code></pre>
    <p>
     <br/>
     默认的端口为 8000
     <br/>
     如果要改变端口，可以这样：
     <br/>
    </p>
    <pre><code class="language-python">python -m CGIHTTPServer 80
</code></pre>
    <p>
     <br/>
     -m 选项的含义：
     <br/>
     -m mod : run library module as a script (terminates option list)
