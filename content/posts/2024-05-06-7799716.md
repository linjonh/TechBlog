# [Python] 用Python建立Web服务器

在想要的目录下，执行如下指令（前提当然是需要安装了python）：

```python
python -m CGIHTTPServer
```

默认的端口为 8000
  
如果要改变端口，可以这样：

```python
python -m CGIHTTPServer 80

```

-m 选项的含义：
  
-m mod : run library module as a script (terminates option list)
