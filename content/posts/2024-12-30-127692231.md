    <p>
     例如我的电脑中同时安装了Python3.7.5和Python3.8.5，平时使用的是Python3.7.5，环境变量中Python3.7.5的位置排在Python3.8.5上面
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ed0c54d00912dd32e8963056e46033c2.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/334a501a2e6ec6a4e427330046f61167.png">
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2258bef1f98bc67bc767ffc4535d83a4.png">
        <br/>
        如果直接输入 pip install XXX，此时安装的是Python3.7.5的依赖库，如果想要安装Python3.8.5的依赖库，输入
       </img>
      </img>
     </img>
    </p>
    <pre><code>py -3.8 -m pip install XXX
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7b6f12531dfbc82ec67b11ebbb76596d.png">
      <br/>
      查看Python安装了哪些依赖库
      <br/>
      输入 pip list 显示的是Python3.7.5安装的依赖库
     </img>
    </p>
    <pre><code>py -3.8 -m pip list
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/17afded8aa696261b42edc7cc0fba0de.png">
      <br/>
      pip更新
     </img>
    </p>
    <pre><code>py -3.8 -m pip install --upgrade pip
</code></pre>
