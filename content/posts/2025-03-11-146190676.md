    <h3 id="h0">
     Win11查看防火墙MySQL的设置
    </h3>
    <p>
     在现代网络安全中，防火墙是一个不可或缺的组件，尤其是在使用数据库，如MySQL时。了解如何在Windows 11中查看和设置防火墙对于确保数据库安全是至关重要的。本文将详细介绍这个过程，并提供相关代码示例。
    </p>
    <h4 id="h1">
     <a id="_4">
     </a>
     一、什么是防火墙？
    </h4>
    <p>
     防火墙是网络安全系统的一部分，通过配置规则来监控和控制网络流量。防火墙可以是硬件设备，也可以是软件。具体到Windows 11，操作系统内置的Windows Defender Firewall可以有效地保护系统免受未授权访问。
    </p>
    <h4 id="h2">
     <a id="MySQL_8">
     </a>
     二、MySQL与防火墙的关系
    </h4>
    <p>
     MySQL是一种流行的开源数据库管理系统，通常用于Web应用程序和其他软件开发项目。在防火墙的设置中，我们需要允许MySQL使用的端口（默认情况下为3306）接受外部连接。否则，MySQL将无法正确接受这些请求。
    </p>
    <h4 id="h3">
     <a id="_12">
     </a>
     三、查看防火墙设置的流程
    </h4>
    <p>
     接下来，我们将了解如何在Windows 11中查看防火墙设置，以确保MySQL可以正常工作。
    </p>
    <pre><code>flowchart TD
    A[开始] --&gt; B[打开Windows设置]
    B --&gt; C[点击“隐私与安全”]
    C --&gt; D[选择“Windows安全”]
    D --&gt; E[点击“防火墙和网络保护”]
    E --&gt; F[选择“允许应用通过防火墙”]
    F --&gt; G[找到MySQL]
    G --&gt; H{是否允许}
    H --&gt;|是| I[完成]
    H --&gt;|否| J[点击“更改设置”]
    J --&gt; K[勾选MySQL]
    K --&gt; I[完成]</code></pre>
    <h5 id="h4">
     <a id="_31">
     </a>
     步骤说明：
    </h5>
    <ol>
     <li>
      <strong>
       打开Windows设置
      </strong>
      ：你可以通过点击开始菜单，然后点击齿轮图标来打开设置。
     </li>
     <li>
      <strong>
       点击“隐私与安全”
      </strong>
      ：在左侧面板中找到并选择这一选项。
     </li>
     <li>
      <strong>
       选择“Windows安全”
      </strong>
      ：此时可以查看各种安全设置。
     </li>
     <li>
      <strong>
       点击“防火墙和网络保护”
      </strong>
      ：在这个部分，你可以找到防火墙相关的信息。
     </li>
     <li>
      <strong>
       选择“允许应用通过防火墙”
      </strong>
      ：在此可以查看哪些应用已被允许通过防火墙。
     </li>
     <li>
      <strong>
       找到MySQL
      </strong>
      ：在应用列表中找到MySQL服务器。
     </li>
     <li>
      <strong>
       确认是否允许
      </strong>
      ：需要检查是否勾选了允许通过防火墙。
     </li>
     <li>
      <strong>
       如果未允许
      </strong>
      ：则需要点击“更改设置”并勾选MySQL。
     </li>
    </ol>
    <h4 id="h5">
     <a id="_42">
     </a>
     四、代码示例
    </h4>
    <p>
     在实际开发中，有时我们需要通过命令行来进行MySQL的设置和查看。以下是一些常用的MySQL命令：
    </p>
    <h5 id="h6">
     <a id="1_MySQL_46">
     </a>
     1. 查看MySQL服务状态
    </h5>
    <p>
     使用以下命令可以查看MySQL服务是否正在运行：
    </p>
    <pre><code id="code_id_0"># 在Windows命令提示符中
sc query mysql
</code></pre>
    <h5 id="h7">
     <a id="2_MySQL_54">
     </a>
     2. 启动MySQL服务
    </h5>
    <p>
     如果MySQL服务未运行，可以使用以下命令来启动它：
    </p>
    <pre><code id="code_id_1"># 在Windows命令提示符中
net start mysql
</code></pre>
    <h5 id="h8">
     <a id="3_MySQL_62">
     </a>
     3. 停止MySQL服务
    </h5>
    <p>
     如果需要停止MySQL服务，可以使用以下命令：
    </p>
    <pre><code id="code_id_2"># 在Windows命令提示符中
net stop mysql
</code></pre>
    <h5 id="h9">
     <a id="4_MySQL_70">
     </a>
     4. 允许MySQL通过防火墙
    </h5>
    <p>
     如果你已经找到了防火墙设置，但是MySQL未被允许，你可以手动添加 MySQL 到 Windows 防火墙的例外列表。执行以下PowerShell命令：
    </p>
    <pre><code id="code_id_3"># 以管理员权限运行PowerShell
New-NetFirewallRule -DisplayName "MySQL" -Direction Inbound -Protocol TCP -Action Allow -LocalPort 3306
</code></pre>

     <a id="MySQL_78">
