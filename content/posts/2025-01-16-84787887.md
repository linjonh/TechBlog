    <h2>
     <a id="_2">
     </a>
     概述
    </h2>
    <p>
     支付宝小程序是手机应用嵌入支付宝客户端的一种方法，有以下特点：
    </p>
    <ul>
     <li>
      基于 Web 技术，学习成本低
     </li>
     <li>
      一套代码，同时支持 iOS 和 Android，接近原生体验
     </li>
     <li>
      提供丰富的组件和 API(比如获取用户信息、本地存储、支付功能等)
     </li>
     <li>
      下面就是一个简单的示例，演示如何快速开发一个小程序。
     </li>
    </ul>
    <h2>
     <a id="_11">
     </a>
     小程序应用
    </h2>
    <p>
     登录
     <a href="https://open.alipay.com/platform/manageHome.htm" rel="nofollow">
      开放平台
     </a>
     ，进入开发者中心频道，点击右侧小程序菜单，可在我的小程序列表页看到对应的小程序应用。在IDE上传版本时需要在关联应用选中该应用:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/90312fdf7a118900239bf382fd5a285e.png"/>
    </p>
    <h2>
     <a id="_15">
     </a>
     下载
    </h2>
    <p>
     首先，请下载
     <a href="https://docs.alipay.com/mini/ide/download" rel="nofollow">
      小程序开发者工具(IDE)
     </a>
     。它是一个辅助开发支付宝小程序的本地应用工具，包含本地调试、代码编辑、真机预览、发布等功能，覆盖了应用开发的完整流程。
    </p>
    <p>
     下载完成之后，打开它，即可使用。部分功能如真机预览和版本上传等需要登录才能使用，在这种情况下开发者可打开手机上的支付宝应用，扫码确定即可登录。
    </p>
    <blockquote>
     <p>
      注意：如果登录没有反应，可尝试完全退出手机上的支付宝应用再重新打开。
     </p>
    </blockquote>
    <p>
     打开IDE，点击新建项目，在支付宝下选择"小程序"，创建项目：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/42f119b8ad99d1550e9f6fafbf93e7f9.png"/>
    </p>
    <p>
     小程序支持多端（支付宝、钉钉、mPaaS）研发和发布，点此
     <a href="https://docs.alipay.com/mini/multi-platform" rel="nofollow">
      查看详情
     </a>
     。
    </p>
    <h2>
     <a id="_27">
     </a>
     代码编辑
    </h2>
    <p>
     打开项目，会默认进入代码编辑模式。从左到右，依次是文件树、代码编辑器和模拟器。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/65ac06e24d80ab0200aa8d4a8873e05b.png"/>
    </p>
    <p>
     代码编辑器可以对当前项目进行代码编写和文件的添加、删除以及重命名等基本操作。
    </p>
    <h4>
     <a id="_33">
     </a>
     实时预览
    </h4>
    <p>
     在代码编辑器修改任何代码都会重新编译，然后自动刷新应用。
    </p>
    <h4>
     <a id="_37">
     </a>
     自动补全
    </h4>
    <p>
     工具针对 my 接口和 axml 提供了大量的自动补全提示，以帮助开发者提高效率。
    </p>
    <h2>
     <a id="_40">
     </a>
     本地调试
    </h2>
    <h4>
     <a id="_41">
     </a>
     模拟器
    </h4>
    <p>
     这里可真实模拟在支付宝应用里的表现，并针对绝大部分的 api 提供了模拟功能。
    </p>
    <h4>
     <a id="_44">
     </a>
     调试器
    </h4>
    <p>
     点击左下角调试器图标，可唤起调试器窗口。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/50ac8e5c30754b8a53edb911c8bfa356.png"/>
    </p>
    <p>
     小程序调试器提供了 axml 和 acss 的支持，支持组件层级、属性回写等功能；同时也包含了 Chrome 调试工具中的网络请求、DOM 元素检查、源码 Debug 等。
    </p>
    <h2>
     <a id="_50">
     </a>
     真机预览
    </h2>
    <p>
     真机预览功能可以让开发者在手机上预览当前代码在真机上运行的效果。
     <br/>
     真机预览功能需要扫码登录并且选择关联应用才能使用。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3731f771ac9cc7f530355c90e2807e73.png"/>
    </p>
    <h2>
     <a id="_54">
     </a>
     上传与发布
    </h2>
    <p>
     开发者扫码登录并选择关联应用后可上传版本到开放平台:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bb91ca40cd3ac4399158ff65ee6bb66b.png">
      <br/>
      上传完毕后，需要登录到开放平台提交审核，审核完毕后，小程序即可进行灰度和发布操作。详细步骤请点击
      <a href="https://docs.alipay.com/mini/introduce/release" rel="nofollow">
       这里
      </a>
      进行查看。
     </img>
    </p>
    <p>
     本教程参考于支付宝-开放平台-小程序。
    </p>
