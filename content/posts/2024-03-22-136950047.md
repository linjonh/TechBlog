# 全是干货！全网最全AI绘画提示词使用技巧

我们来介绍在AI绘画中还有哪些重要的风格提示词写作技巧。

废话不多说，小伙伴们接着往下看。

**艺术媒介/艺术表现形式的提示词：**

在艺术创作中，媒介是指艺术家用来表达创意的工具或材料，可以包括绘画、雕塑、摄影、数字艺术、陶瓷等不同形式。以下是提示词中输入艺术媒介的示例：

![](https://i-blog.csdnimg.cn/blog_migrate/6fbca0b4beca66ab76749a56fac8fce6.png)

![](https://i-blog.csdnimg.cn/blog_migrate/ee825def41cb2f3af5768f519843983a.png)

**艺术家方面的提示词：**

在写提示词的时候，我们如果需要画面具有某个艺术家的作品风格特征，我们可以直接在提示词中输入艺术家的名字，让我们的AI作品具有跟某个艺术家相近的风格。

**肖像艺术家：**

在提示词中输入以肖像画闻名的艺术家名字可以非常有助于创造特定的风格。下面的图片是使用相同的提示词，包括种子，只是艺术家的名字不同所产生的画风变化：

![](https://i-blog.csdnimg.cn/blog_migrate/d50605a8c6fea6b853c493d6d52cb749.png)

**风景画家：**

下面同样是相同的提示词和种子，不同的艺术家名字所产生的不同画面表现。我们在用AI创作风景画时，在提示词中需要输入时间，如上午、中午或晚上，并写明季节。需要注意的是，有些艺术家只会对输出产生微妙的影响，而另一些则会对其进行剧烈的更改。

![](https://i-blog.csdnimg.cn/blog_migrate/62169d5415f50e2802e9bee9c3bfdaf9.png)

**恐怖主义艺术家：**

下面也是一样，提示词除了艺术家名字不同，其它都一样，包括种子数：

![](https://i-blog.csdnimg.cn/blog_migrate/7b51f7d0665905b51b2d16bcb805ad1a.png)

**动漫艺术家：**

在提示词中使用动漫艺术家名字时，要留意他们不同时间段的风格作品，有些艺术家在不同的时间段所创作的作品风格差异是比较大的，如果有这种情况，我们需要在提示词中加上“from…”（从…起）。

![](https://i-blog.csdnimg.cn/blog_migrate/43646c70499fadb799db725cf03040b4.png)

**科幻艺术家：**

在提示词中选择合适的艺术家可以极大地改变输出。科幻小说艺术家往往有非常独特的风格样式。在提示词中输入科幻艺术家名字时，不仅可以在提示词中加入使用传统的艺术媒介，还可以使用电影中的艺术家。

![](https://i-blog.csdnimg.cn/blog_migrate/392e36145229d693fb87cb428b5158ea.png)

**摄影艺术家：**

![](https://i-blog.csdnimg.cn/blog_migrate/911107b47fdad0168770a35d4431b02e.png)

**概念艺术家：**

概念艺术家中的一些人会更擅长场景表现，而另一些人更擅长人物表现。

![](https://i-blog.csdnimg.cn/blog_migrate/42af3c4f83f85c3745f33118d42a9d4d.png)

**混合艺术家风格：**

我们在提示词中输入艺术家名字时，还可以输入多个艺术家的名字，画面会呈现出两种风格相融合的独特艺术风格。

![](https://i-blog.csdnimg.cn/blog_migrate/926b4ef6ddd61dbb5f12786a12feea49.png)

![](https://i-blog.csdnimg.cn/blog_migrate/6a198ff24052d0b7a9469d2c0752daf0.png)

![](https://i-blog.csdnimg.cn/blog_migrate/63ffe2e2075be9e49940ce20cd20bfee.png)

上面我们介绍了在提示词中输入相同的提示词，不同的艺术家名字AI给我们呈现的不同画风。下面介绍几位按风格划分的部分艺术家：

**肖像画：**

1. **Derek Gores：**Derek Gores 是一位当代艺术家，以拼贴艺术闻名。他的作品通常使用回收材料和杂志图片来创作独特的艺术品。
2. **Miles Aldridge**
   ：Miles Aldridge 是一位英国摄影师，以其色彩饱满、充满戏剧性和超现实主义风格的摄影作品而著名。
3. **Jean-Baptiste Carpeaux：**Jean-Baptiste Carpeaux 是19世纪法国雕塑家，他的作品以其精湛的技艺和栩栩如生的表现而闻名。
4. **Anne-Louis Girodet：**Anne-Louis Girodet 是18至19世纪法国画家，他是新古典主义画派的代表人物之一，擅长历史题材和肖像画。

**风景画：**

1. **Alejandro Burdisio：**Alejandro Burdisio 是一位阿根廷艺术家，以其现代艺术作品和独特的创作风格而知名。
2. **Jacques-Laurent Agasse：**Jacques-Laurent Agasse 是18至19世纪瑞士画家，以其动物画作品而著名，尤其擅长描绘马匹和其他动物。
3. **Andreas Achenbach：**Andreas Achenbach 是19世纪德国画家，以其风景画和海景画而闻名，被认为是浪漫主义风格的代表人物之一。
4. **Cuno Amiet：**Cuno Amiet 是瑞士画家和雕塑家，他是20世纪早期表现主义艺术运动的重要人物之一，擅长绘制肖像画和风景画。

**恐怖/超现实主义风格：**

1. **H.R. Giger：**H.R. Giger 是瑞士艺术家，以其超现实主义风格和对未来科技的描绘而著名。他设计了《异形》电影中的异形怪物，因此在科幻艺术领域享有盛誉。
2. **Tim Burton：**Tim Burton 是美国导演、编剧和制片人，以其独特的黑暗幽默风格和奇幻题材的电影而闻名，如《爱丽丝梦游仙境》、《爱伦·坡的黑猫》等。
3. **Andy Fairhurst：**Andy Fairhurst 是英国插画师和艺术家，他以其独特的风格重新诠释了许多经典电影和漫画作品，深受粉丝喜爱。
4. **Zdzislaw Beksinski：**Zdzislaw Beksinski 是波兰画家和雕塑家，以其超现实主义和恐怖风格的作品而著名。他的作品常常描绘荒诞、黑暗和超自然的主题。

**日漫风格：**

1. **Makoto Shinkai：**新海诚（Makoto Shinkai）是一位知名的日本动画电影导演和编剧，他以其细腻的画面和感人的故事情节而著名，代表作品包括《你的名字》（Your Name）和《天气之子》（Weathering With You）。
2. **Katsuhiro Otomo：**大友克洋（Katsuhiro Otomo）是一位著名的日本漫画家和动画导演，他创作了《AKIRA》这部极具影响力和创新性的漫画作品，后来也拍摄了同名的动画电影。
3. **Masashi Kishimoto：**岸本斉史（Masashi Kishimoto）是日本漫画家，他是著名漫画《火影忍者》（Naruto）的作者，这部作品在全球范围内都享有极高的人气和影响力。
4. **Kentaro Miura：**三浦建太郎（Kentaro Miura）是日本漫画家，他是著名漫画《暗黑破坏神》（Berserk）的作者，这部作品以其黑暗、成熟和复杂的故事情节而著称，被认为是漫画史上的经典之一。

**科幻风格：**

1. **Chesley Bonestell：**Chesley Bonestell 是美国的一位知名科幻艺术家，擅长绘制太空和未来科技的插画作品，对科幻艺术领域产生了重大影响。
2. **Karel Thole：**Karel Thole 是荷兰的一位著名科幻艺术家，他以其独特的风格和对未来世界的想象而闻名，为许多科幻小说和杂志创作过插画。
3. **Jim Burns：**Jim Burns 是英国的一位杰出科幻艺术家，他的作品常常描绘未来世界、太空飞船和科幻场景，获得了多项奖项和认可。
4. **Enki Bilal：**Enki Bilal 是法国的一位知名漫画家和电影导演，他擅长创作融合科幻、政治和哲学元素的作品，以其独特的视角和风格而备受赞誉。

**摄影风格：**

1. **Ansel Adams：**Ansel Adams 是美国著名的风景摄影师，他以其对自然风光的精湛拍摄技巧和黑白摄影作品而闻名，被认为是20世纪最伟大的摄影师之一。
2. **Ray Earnes：**Ray Earnes 是一位著名的美国建筑摄影师，他以其对建筑结构和设计的独特视角和精湛摄影技巧而备受赞誉。
3. **Peter Kemp：**Peter Kemp 是一位知名的摄影师，他擅长拍摄人像和纪实摄影作品，作品风格多样且富有表现力。
4. **Ruth Bernhard：**Ruth Bernhard 是一位著名的美国女性摄影师，她以其对人体和静物的细腻拍摄而著称，作品充满了艺术感和情感表达。

**概念艺术风格：**

1. **Emerson Tung：**Emerson Tung 是一位著名的概念艺术家和插画师，他擅长创作各种风格的概念艺术作品，包括角色设计、环境设计等。
2. **Shaddy Safadi：**Shaddy Safadi 是一位知名的概念艺术家和电影概念设计师，他在电影和游戏行业中拥有丰富的经验，作品风格多样且富有创意。
3. **Kentaro Miura：**Kentaro Miura 是日本著名的漫画家，他是《暗黑破坏神》（Berserk）的作者，以其精湛的画风和复杂的故事情节而备受赞誉。
