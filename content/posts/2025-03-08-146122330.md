    <h1 class="title-article" id="articleContentId">
     SQL SELECT DISTINCT 语句
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     在 SQL 中，
     <code>
      SELECT DISTINCT
     </code>
     语句用于从表中查询不重复的值。这对于需要从数据库检索唯一值时非常有用。
     <code>
      DISTINCT
     </code>
     关键字会去除结果集中重复的行，只返回唯一的记录。
    </p>
    <pre><code class="hljs">SELECT DISTINCT column1, column2, ...
FROM table_name;</code></pre>
    <ul>
     <li>
      <code>
       column1
      </code>
      ,
      <code>
       column2
      </code>
      , ... 是你希望在结果集中应用去重的列。
     </li>
     <li>
      <code>
       table_name
      </code>
      是你要查询的表。
     </li>
    </ul>
    <h4>
     多列 DISTINCT
    </h4>
    <p>
     当你指定多个列时，
     <code>
      DISTINCT
     </code>
     会作用于这些列的组合。也就是说，只有当所有指定列的组合相同时才会去除重复。
    </p>
    <pre><code class="hljs">SELECT DISTINCT first_name, last_name
FROM employees;</code></pre>
    <p id="">
     在这个例子中，查询会返回所有不同的
     <code>
      first_name
     </code>
     和
     <code>
      last_name
     </code>
     组合。如果两个员工的名字和姓氏完全相同，则只会显示一条结果。
    </p>
    <h4>
     注意事项
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        性能影响
       </strong>
       ：
       <code>
        DISTINCT
       </code>
       操作可能会增加查询的计算成本，尤其是在大数据集上，因为它需要对数据进行排序和去重。
      </p>
     </li>
     <li>
      <p id="">
       <strong>
        NULL 值处理
       </strong>
       ：
       <code>
        DISTINCT
       </code>
       也会去掉重复的
       <code>
        NULL
       </code>
       值。在 SQL 中，
       <code>
        NULL
       </code>
       被视为一个特殊的值，因此多个
       <code>
        NULL
       </code>
       在结果集中只会被显示一次。
      </p>
     </li>
     <li>
      <p id="">
       <strong>
        结合其他子句
       </strong>
       ：
       <code>
        SELECT DISTINCT
       </code>
       可以与其他 SQL 子句结合使用，如
       <code>
        WHERE
       </code>
       、
       <code>
        ORDER BY
       </code>
       等，以实现更复杂的查询。
      </p>
     </li>
    </ol>
