个值得记录。

认证和各种UNIX认证考试(

)，选择哪个大方向，哪个小门类？在学校接触的都是简单的windows开发，根本没有unix开发，毕业设计还是windows开发，但是找的工作呢，是UNIX环境的，早知道选公司给的题目了了。如此尴尬的境地，但是这两者都是熟悉不熟悉的问题而已，无论是哪个核心都是算法，对于人而言就是思维，当然也要多练习。

微软认证介绍，完全复制自百度百科。

  
  
微软认证是微软公司设立的推广微软技术，培养系统网络管理和应用开发人才的完整技术金字塔证书体系，在全世界90多个国家认可有效，并且可以作为薪资职位变迁的有效证明，公司资质实力证明和移民加分等多项益处！微软认证从1992年设立至今，在业界的影响力也越来越大，全球共计有8万位MCSE2003和3万多位MCSD产生，是具备相当含金量和实用价值的高端证书。
  
  

2012年，微软认证计划进行了全面升级，以涵盖云技术相关的解决方案。新一代微软认证基于产品与技术解决方案共有以下四个认证路径：
  
  
**1、Client ：**

**1) MCSA：Windows 8**

此认证可证明持证者有能力胜任配置、管理及维护企业级Windows 8系统，通过两项考试70687、70688即可获得此认证。

**2) MCSA：Windows 7**

此认证目前将继续留在市场，不会退役，通过两项考试70680、70685即可获得此认证。

**2、Server：**

**1) MCSA： Windows Server 2012**

此认证可证明持证者在商业环境中多个解决方案领域里，具有主要的Windows Server技能，通过三项考试70410、70411、70412即可获得此认证。

**2) MCSA： Windows Server 2008**

此认证目前将继续留在市场，不会退役，通过三项考试70640、70642、70646即可获得此认证。

**3) MCSE：Desktop Infrastructure**

此认证可以证实持证者具有所需技能，能够独立成功搭建桌面基础架构解决方案。在MCSA：Windows Server 2012认证的基础上，通过两项考试70415、70417即可获得此认证。

  
  

4) MCSE：Server Infrastructure

此认证可以充分证明持证者已经具备相关技能，能够独立实施服务器基础架构解决方案，从而为企业提供此种平台系统。在MCSA：Windows Server 2012认证的基础上，通过两项考试70413、70414即可获得此认证。

**5) MCSE：Private Cloud**

考取私有云技术认证证书可以全面证明持证者在微软私有云计算技术的实现与管理领域中已经充分掌握相关知识及专业技能。在MCSA：Windows Server 2012认证的基础上，通过两项考试70246、70247即可获得此认证。
  
  

**6) MCSE：Communications**

此认证可以帮助您在沟通管理解决方案领域中获得相应专业技能，从而全面实现为您的企业成功搭建性能稳定的沟通管理解决方案，并帮助您的同事与位于世界各地的千百万人随时保持联络。在MCSA：Windows Server 2012认证的基础上，通过两项考试70336、70337即可获得此认证。

**7) MCSE：Messaging**

此认证将充分证明，持证者已经具备全面所需技能，可以将您的企业成功迁移至云计算领域，进一步提升用户工作能力及业务灵活性，减少数据丢失，并实现更高层次的企业数据安全保护。在MCSA：Windows Server 2012认证的基础上，通过两项考试70341、70342即可获得此认证。

**8) MCSE：SharePoint**

此认证将充分证明，您已经具备所需专长，可以帮助您的企业在整个组织架构之中全面实现信息管理、同步、协作与共享。在MCSA：Windows Server 2012认证的基础上，通过两项考试70331、70332即可获得此认证
  
**3、Database：**

**1) MCSA：SQL Server 2012**

此认证可展示持证者在开发与维护关键任务环境中的下一个浪潮，具备基本持能和突破性见解，。通过三考试70461、70462、70463即可获得此认证。

**2) MCSA：SQL Server 2008**

此认证目前将继续留在市场，不会退役，通过两项考试70432、70448即可获得此认证。

**3) MCSE：Data Platform**

此认证可以表明持证者拥有全面能力，能够在SQL Server 2012平台上实现高性能解决方案的管理操控。在MCSA：SQL server 2012认证的基础上，通过两项考试70464、70465即可获得此认证。
  

**4) MCSE：Business Intelligence**

此认证可以帮助持证者确保自己在商业情报信息领域中具有最新的专业知识。通过设计、构建以及部署相关解决方案，您所掌握的技术和技能可以帮助整个企业组织中的更多用户及时分享和共享更多数据。在MCSA：SQL server 2012认证的基础上，通过两项考试70466、70467即可获得此认证。
  
**4、Development：**

**1) MCSD：Windows Store Apps Using HTML**

此认证将充分证明持证者已具有相应能力，可使用HTML设计并快速开发出Windows8应用程序。通过三项考试70480、70481、70482即可获得此认证。

**2) MCSD：Windows Store Apps Using C#**

此认证将充分证明持证者已具有相应能力，可使用C#设计并快速开发出Windows8应用程序。通过三项考试70483、70484、70485即可获得此认证。

**3) MCSD：Web Applications**

此认证说明持证者在创建和部署现在Web应用程序和服务上具备必要的技能，并使其专业知识得到认可。通过三项考试70480、70486、70487即可获得此认证。

**4) MCSD：Application Lifecycle Management**

此认证可证明持证者在管理应用程序开发整个生命周期具备必要的专业知识并得到认可，通过三项考试70496、70497、70498即可获得此认证。

### 考试时间

微软认证考试时间不同于国内的考试，是随报随考，考试的时间都由考生在携带身份证和考试费报名时自己决定的，只要考试中心正常营业，考生就可以正常报考！考生头天报名预约，第二天参加考试！
  
  

### 考试内容

微软认证考试的内容非常专业化，主要有以下几个方面：

1、操作系统(WindowsNT4.0的安装，系统配置，账户管理，文件、打印机资源管理，远程访问服务、域的规划建立及管理、企业的域模型设计，对NT的网络分析和优化及故障排除等)；

2、Windows2000(Windows2000平台系列，安装、DNS、活动目录，远程安装服务、组策略，终端服务、文件与磁盘管理、灾难保护，目录服务基础、命名策略、权限委派，活动目录逻辑结构、物结构、实施等)；

3、网络技术课程(网络设计，协议连网的实现，网络的连接、规划、配置、管理、监测等)；

4、数据库(数据库的安装、配置、管理，安全性管理，数据库的备份和恢复，性能调整，用户账户管理，多表查询，事务处理，视图，存储进程实施等)；

5、开发工程师(数据库的开发，COM的开发，MFC的开发，收集、分析商业需求，设计商业方案，设计数据服务和数据模型等)；

6、电子商务(创建网业，创建COM构件及MTS的使用，建立、维护、优化电子商务站点等)。

### 考试语言

微软认证考试语言主要为英语，这是由微软认证是国际认证的定位所决定的，但国内部分科目也有中文考试，但IT认证考试资源网的所有老师都不建议您报考中文考试！现在因为授权的原因，微软的官方教材都还是英文版本，在这样的境况下学习英文教材，参加中文考试，其本身就处于一个比较尴尬的境地，其难度比学英文教材考英文考试要难得多；其次即使是报考微软中文考试，但由于本地化进度没有跟上，中文考试科目不全，如果社会考生报考微软中文考试的话，会面临前面有中文考试考完了，剩下的考试却不得不考英文考试一个更尴尬的局面！

### 考试教材

微软认证的教材除了有微软公司自己组织出版的原版MOC培训教材和Training Kit系列自学教材外，还有多家其他公司出版的系列，比如Sybex公司出版的Study Guide系列，QUE公司出版的ExamCram系列，这些都在易证宝附赠光盘上有详细介绍！原则上IT认证考试资源网都推荐学员采用微软的原版MOC培训教材，原版MOC培训教材由于授权的原因，到目前为止都只有英文原版，没有出版过中文版本！

### 认证选择

建议初学者在选择微软认证或者在刚开始的技术选型时，尽量选择主流的应用软件或系统对应的证书，比如系统工程师尽量考虑MCSE2003证书，软件开发则可以考虑MCPD系列证书等，选择主流的应用一可以获得丰富的技术资源和交流人群，这对学习很有帮助；二可以尽快方便地寻求实习实践的场所和机会，对学员理论联系实际获得经验；三则是市场对这样的人才需求比较大，其证书效力在业界的认可度也要大得多！

具体各种微软证书的区别和适用人群介绍请看 《初学者报考微软认证指南》
