    <h5>
     一、前置准备
    </h5>
    <p id="">
     在开始签名之前，需要准备以下信息：
    </p>
    <ol>
     <li>
      <strong>
       Access Key ID（AK）
      </strong>
      ：请求火山引擎OpenAPI的安全凭证之一。
     </li>
     <li>
      <strong>
       Secret Access Key（SK）
      </strong>
      ：与AK成对使用，用于签名计算。
     </li>
     <li>
      <strong>
       请求参数
      </strong>
      ：包括HTTP请求方法、URI、查询参数、请求头等。
     </li>
    </ol>
    <h5>
     二、签名流程(
     <a class="link-info" href="https://www.volcengine.com/docs/6369/67269" rel="nofollow" title="文档地址)">
      文档地址)
     </a>
    </h5>
    <p id="">
     火山引擎API签名流程大致分为以下几个步骤：
    </p>
    <ol>
     <li>
      <strong>
       创建规范请求
      </strong>
      ：将请求参数进行规范化处理，形成CanonicalRequest。
     </li>
     <li>
      <strong>
       创建待签名字符串
      </strong>
      ：基于CanonicalRequest和其他元数据信息，构建StringToSign。
     </li>
     <li>
      <strong>
       派生签名密钥
      </strong>
      ：从SK派生出签名密钥（signing key）。
     </li>
     <li>
      <strong>
       计算签名
      </strong>
      ：使用HMAC-SHA256算法和签名密钥，计算StringToSign的签名。
     </li>
     <li>
      <strong>
       构建Authorization头
      </strong>
      ：将签名和其他必要信息添加到请求头中。
     </li>
    </ol>
    <h5>
     三、具体实现
    </h5>
    <p id="">
     以下是一个PHP示例代码，火山引擎API请求生成签名：
    </p>
    <pre><code class="language-php">/**
 *
 * Author:刘星麟
 * @param $ak   Access Key ID
 * @param $sk   Secret Access Key
 * @param $region   服务Region
 * @param $service  服务名称
 * @param $body     请求参数 没有给空
 * @param $query    接口请求参数
 * @param $method   请求方法  GET|POST
 * @param $path     服务路径
 * @param $headers  服务请求header  签名必要  https://www.volcengine.com/docs/6369/67268#_2-%E7%AD%BE%E5%90%8D%E5%8F%82%E6%95%B0
 * @return mixed
 */
function signv4($ak, $sk, $region, $service, $body, $query, $method, $path, $headers)
{
    if ($path === '') {
        $path = '/';
    }
    
    $ldt = gmdate('Ymd\THis\Z');
    $sdt = substr($ldt, 0, 8);
    $headers['X-Date'] = $ldt;
    $bodyHash = hash('sha256', $body);
    $headers['X-Content-Sha256'] = $bodyHash;
    
    $credentialScope = "$sdt/$region/$service/request";
    $signedHeaders = [];
    foreach ($headers as $key =&gt; $value) {
        if ($key == "Host" || $key == "Content-Md5" || $key == "Content-Type" || substr($key, 0, 2) == "X-") {
            $key = strtolower($key);
            $signedHeaders[$key] = $value;
        }
    }
    
    
    ksort($signedHeaders);
    
    $signed_str = '';
    foreach ($signedHeaders as $k =&gt; $v) {
        $signed_str .= $k . ':' . $v . "\n";
    }
    
    $signedHeadersString = implode(';', array_keys($signedHeaders));
    $canon = implode("\n", array($method, $path, $query, $signed_str, $signedHeadersString, $bodyHash));
    $hash = hash('sha256', $canon);
    $toSign = implode("\n", array("HMAC-SHA256", $ldt, $credentialScope, $hash));
    $signingKey = getSigningKey($sdt, $region, $service, $sk);
    $signature = hash_hmac('sha256', $toSign, $signingKey);
    $credential = $ak . '/' . $credentialScope;
    $headers['Authorization'] = "HMAC-SHA256 Credential={$credential}, SignedHeaders={$signedHeadersString}, Signature={$signature}";
    return $headers;
}

 function getSigningKey($date, $region, $service, $sk)
{
    $dateKey = hash_hmac('sha256', $date, $sk, true);
    $regionKey = hash_hmac('sha256', $region, $dateKey, true);
    $serviceKey = hash_hmac('sha256', $service, $regionKey, true);
    return hash_hmac('sha256', 'request', $serviceKey, true);
}</code></pre>

    <h5>
     五、注意事项
    </h5>
    <ol>
     <li>
      <strong>
       时间戳
      </strong>
      ：X-Date头必须使用UTC时间，并遵循ISO 8601标准格式。
     </li>
     <li>
      <strong>
       请求头排序
      </strong>
      ：参与签名的请求头必须按照ASCII码顺序排序。
     </li>
     <li>
      <strong>
       密钥安全
      </strong>
      ：SK是敏感信息，必须妥善保管，避免泄露。
     </li>
     <li>
      <strong>
       签名算法
      </strong>
      ：火山引擎目前仅支持HMAC-SHA256算法进行签名。
     </li>
     <li>
      <strong>
       $body
      </strong>
      ：请求参数 没有一定要给空
     </li>
    </ol>
