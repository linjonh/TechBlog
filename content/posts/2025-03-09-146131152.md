# git 提交报错

remote: <CH.00905401> HTTP Basic: Access denied.

remote: The password-based authentication of Git has been removed. Please use your personal access token instead of the password. Request-id is XXXXXX.

fatal: Authentication failed for 'https://gitcode.com/......'

看了下错误信息，提示  请使用您的个人访问令牌。。。

[GitCode - 全球开发者的开源社区,开源代码托管平台](https://gitcode.com/setting/token-classic "GitCode - 全球开发者的开源社区,开源代码托管平台")

‘
[https://gitcode.com/setting/token-classic](https://gitcode.com/setting/token-classic "https://gitcode.com/setting/token-classic")
’

点击右上角的【+新建访问令牌】，然后创建。这个访问令牌只会出现一次，关了网页就无法查询了。
