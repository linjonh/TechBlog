    <h2>
     <a id="_1">
     </a>
     概述
    </h2>
    <h3>
     <a id="_2">
     </a>
     目的
    </h3>
    <p>
     安装Ubuntu系统到一台新的华为服务器，并为服务器配置向日葵远程桌面，实现服务器的远程维护。
    </p>
    <h3>
     <a id="_4">
     </a>
     环境
    </h3>
    <p>
     硬件环境：华为 2288H V5服务器，6Tx5硬盘，RAID5后实际空间约24T
     <br/>
     软件系统：Ubuntu 20.04.2.0 LTS
     <br/>
     向日葵: V11
    </p>
    <h2>
     <a id="_9">
     </a>
     安装准备
    </h2>
    <h3>
     <a id="_10">
     </a>
     下载系统
    </h3>
    <p>
     系统下载地址:https://ubuntu.com/download/desktop
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b1f3860de1fe260f7d66413788727b60.png"/>
    </p>
    <h3>
     <a id="_13">
     </a>
     制作启动盘
    </h3>
    <p>
     Ubuntu启动盘官方制作教程:https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview
     <br/>
     下载启动盘制作工具Rufus: https://rufus.ie/zh/
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d4156b87b446837a7f0fb7f82e9dea46.png"/>
    </p>
    <h3>
     <a id="BIOS_17">
     </a>
     设置BIOS
    </h3>
    <ul>
     <li>
      <p>
       重启计算机，重启时按键盘上的
       <code>
        Delete
       </code>
       键
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ebe7ef9d7dbdf4c3e15de1c40f2fa782.png"/>
      </p>
     </li>
     <li>
      <p>
       进入BIOS需要密码，一般密码在服务器的机壳上贴着
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e5cd7d44b81c692a9a92f0b4c9bd0bae.png">
        <br/>
        <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d12e732fdb4d8b0d9eea8af34522313e.png"/>
       </img>
      </p>
     </li>
     <li>
      <p>
       进入修改BIOS信息
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9cca3ea48f14c8ce57d6d1cbcad82a43.png"/>
      </p>
     </li>
     <li>
      <p>
       先在主界面中修改语言为简体中文,再在“启动”项中启用“特殊启动”的“USB优先”
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/3b93c3c7fc68947e24cf1ffcbe58ab08.png"/>
      </p>
     </li>
     <li>
      <p>
       按F10保存，然后退出重启系统
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/bfedceb509a6eab221f37c70eeefc84a.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        在计算机重启之前，记得将前面制作的启动U盘插到服务器上
       </strong>
      </p>
     </li>
    </ul>
    <h2>
     <a id="_34">
     </a>
     安装系统
    </h2>
    <h3>
     <a id="_35">
     </a>
     进入优盘系统
    </h3>
    <ul>
     <li>
      系统重启之后，自动进入U盘系统，先选择“简体中文”，再选择“试用Ubuntu”
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5969d3f1fcb1327abb136e53af7a22dd.png"/>
     </li>
    </ul>
    <h3>
     <a id="_38">
     </a>
     硬盘分区(对于已经分好区的，此步非必要)
    </h3>
    <ul>
     <li>
      <p>
       进入系统后选择“硬盘”
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6e6431c398c653796c7f6d74fc7f64f1.png"/>
      </p>
     </li>
     <li>
      <p>
       调整硬盘的分区
      </p>
     </li>
    </ul>
    <blockquote>
     <p>
      我这里的服务器硬盘共有24T的空间，分了3个区，两个1T的，一个22T的
     </p>
    </blockquote>
    <p>
     选中硬盘后，调整大小
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/8e495d3dfd56fd68e657a67aec5a978e.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5850ae4b6f951c4a9f7527c381e85063.png"/>
    </p>
    <p>
     先分出1T空间
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b5c31fb0324a26feb950661bcfb69fee.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c710ada9e979527e759d770275102bcf.png"/>
    </p>
    <p>
     再在调整出的23T未分配的空间上创建1T新的分区
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6a3da0384c499ab67bb9ea5e04549e2b.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b2f34866007fe09b6c66caee78a2e79d.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d8f728787e7bf255654f718d773af1c6.png"/>
    </p>
    <p>
     最后将剩余的22T空间创建成一个大分区
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/170683f983c8f498d38b250e48c1612b.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f43ca672906ec3c669731f07d2175888.png"/>
    </p>
    <h3>
     <a id="_61">
     </a>
     开始安装系统
    </h3>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/49c8a2771d5a6d476d5c64e72e719150.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/139ba10e0d3ac26a6839a9720466ea5f.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5a66a71c9d03e88b89e15cc9beabf63f.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/09886cec361e6f73265addbd8942dd69.png"/>
    </p>
    <ul>
     <li>
      <p>
       根据自己的情况选择安装类型，我这里选择的是“其他选项”
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/9f18bcb807b979169a8eac95b644d4b1.png"/>
      </p>
     </li>
     <li>
      <p>
       编辑分区，分别将3个盘挂载到相应的位置
      </p>
     </li>
    </ul>
    <blockquote>
     <p>
      我这里将根目录
      <code>
       /
      </code>
      挂载到了第一个1T的分区，将
      <code>
       /home
      </code>
      挂载到了第二个1T分区，将
      <code>
       /usr
      </code>
      挂载到了最大的分区
      <br/>
      根据自己的情况选择是否格式化，
      <strong>
       如果盘中已有数据，慎选格式化
      </strong>
      。
     </p>
    </blockquote>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/113c40cf99a334a8b4f305707cf5eef4.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b4e6227b04579662d938b1fd3098b7d8.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f3a31e606ad4bed500a4b6b276fd7ff6.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/63536bc2d0ea74e159bf9a3f58b0138c.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0bb917cda393e4d09d8b1f8eb129dd52.png"/>
    </p>
    <ul>
     <li>
      <p>
       选择时区，输入用户名和密码
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0d3457f866ae9be577a8a5aafc8ec7d7.png"/>
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ca2b2c9c7248f9b89b09d2a11a93e388.png"/>
      </p>
     </li>
     <li>
      <p>
       下面的安装过程可以无人值守了，直到安装完成，重启系统
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/abe0bd1ba990c71a9af025750fdfc874.png"/>
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/76cb90886758e4797528f34d63abb266.png"/>
      </p>
     </li>
     <li>
      <p>
       点击重启后，需要先拔掉安装U盘，按回车键继续启动
       <br/>
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/7826805c153e024791b30c8cea9694ef.png"/>
      </p>
     </li>
    </ul>
    <h2>
     <a id="_91">
     </a>
     配置系统
    </h2>
    <blockquote>
     <p>
      此后的各种操作可能需要频繁的输入密码，要求输入的时候输入密码点击“认证”即可。
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4ecbc1f4180d38b759360db8deb7ed77.png"/>
     </p>
    </blockquote>
    <h3>
     <a id="_95">
     </a>
     联网
    </h3>
    <p>
     插上网线，设置网络参数
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d3465fed59a87a7e90a3088b15d71500.png"/>
    </p>
    <h3>
     <a id="_98">
     </a>
     换源
    </h3>
    <p>
     点击桌面左侧的文件夹，然后依次 其他位置-&gt;etc-&gt;apt,在apt文件夹下双击"sources.list",然后在“下载自”选择框中选择合适的站点。我这里选择的是阿里云镜向服务器，不改使用默认的中国服务器也可以。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/31f911f1013aba6b0a694d1eb7e871c1.png"/>
    </p>
    <p>
     修改后，关闭时会让重新载入。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/401e7b639163fb53d3d0a75deaa3250f.png"/>
    </p>
    <h3>
     <a id="_105">
     </a>
     软件更新
    </h3>
    <p>
     刚进入系统时会提示软件更新，可以根据需要选择立即或者稍后（建议在完成换源之后更新软件）。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ec83ae76d0b3234b299c59a13135f4e6.png"/>
    </p>
    <h2>
     <a id="_109">
     </a>
     配置向日葵远程桌面
    </h2>
    <p>
     服务器一般都需要远程维护，可以选择一款远程桌面软件进行远程维护。我这里选择向日葵远程桌面。
    </p>
    <h3>
     <a id="_111">
     </a>
     下载向日葵
    </h3>
    <p>
     向日葵下载地址：https://sunlogin.oray.com/download/
     <br/>
     选择Linux Ubuntu版本
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/5a3529e37da4e328b71680e6e324ffd3.png"/>
    </p>
    <h3>
     <a id="_115">
     </a>
     安装向日葵
    </h3>
    <p>
     找到下载的文件，双击安装（在上述"软件更新"进行时，可能无法正确安装，提示不能安装时，等软件更新完成即可）。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ae59825f28466be82fe9fc8a6d9d7054.png"/>
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/779455cf70b8a0b8e65fb973ebf6aacf.png"/>
    </p>
    <h3>
     <a id="lightdm_120">
     </a>
     安装lightdm
    </h3>
    <p>
     ubuntu下，向日葵的工作依赖
     <code>
      lightdm
     </code>
     ,运行向日葵前先要安装
     <code>
      lightdm
     </code>
     。否则的话虽然能够启动向日葵，但是远程连接上后是黑屏的，或者直接连接不上，如下图。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/23787df2d9310ce7842bca0995f446fd.png"/>
    </p>
    <ul>
     <li>
      安装lightdm
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6631152758fcdf92c7f67d9ff74c0ad5.png"/>
      <br/>
      安装过程中会让选择默认的选项，选择
      <code>
       lightdm
      </code>
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/df1621275a6c62cebf923c67f30cb7cf.png"/>
      <br/>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/6f21c52e096baf9fb44f4892377f5e2f.png"/>
     </li>
     <li>
      使用命令行切换lightdm和gdm3(会调出上面两图)
     </li>
    </ul>
    <pre><code>sudo dpkg-reconfigure lightdm
</code></pre>
    <h3>
     <a id="_134">
     </a>
     启动向日葵
    </h3>
    <p>
     选择账号登陆向日葵或者使用机器识别码，账号登陆的时候建议绑定机器。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/ffd51ed8d42963e2177642174c5bc35e.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4dcae0223f52fd4f0f9c131c7df93ddd.png"/>
    </p>
    <h2>
     <a id="_140">
     </a>
     重启计算机
    </h2>
    <ul>
     <li>
      重启计算机，恢复BIOS设置
     </li>
     <li>
      恢复BIOS之后，可以拔掉鼠标、键盘、显示器，用向日葵远程登陆操作维护了。
     </li>
    </ul>
