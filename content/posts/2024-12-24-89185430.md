# python中两个list去重

## python中两个list去重

python中两个list会出现重复的情况，比如说两个文件夹A和B，A中保存原始的数据，B中保存已处理的数据。

```
A = range(0,10)
B = range(0,20)
C = list(set(A) - set(B))

```

同时，set数据还支持对数据进行交集，并集的运算

```
union = list(set(A)|set(B))
insect = list(set(A)&set(B))

```
