    <h3 id="1.%20%E5%BC%95%E8%A8%80" name="1.%20%E5%BC%95%E8%A8%80">
     <span style="color:#ed7976">
      1. 引言
     </span>
    </h3>
    <p>
     <code>
      Text
     </code>
     组件是 Flutter 中最常用的 UI 组件之一，用于显示文本内容。它支持样式自定义、多行显示、溢出控制等功能，适用于各种文本场景。本文将详细介绍
     <code>
      Text
     </code>
     组件的使用方式及其重要参数。
    </p>
    <h3 id="2.%20%E5%9F%BA%E6%9C%AC%E4%BD%BF%E7%94%A8" name="2.%20%E5%9F%BA%E6%9C%AC%E4%BD%BF%E7%94%A8">
     <span style="color:#ed7976">
      2. 基本使用
     </span>
    </h3>
    <pre><code class="language-Dart">import 'package:flutter/material.dart';

class ScTextPage extends StatefulWidget {
  const ScTextPage({super.key});

  @override
  State&lt;ScTextPage&gt; createState() =&gt; _ScTextPageState();
}

class _ScTextPageState extends State&lt;ScTextPage&gt; {
  @override
  Widget build(BuildContext context) {
    return Text('Hello, Flutter!');
  }
}</code></pre>
    <p class="img-center">
     <img alt="" height="159" src="https://i-blog.csdnimg.cn/direct/6e152e64cb1c4530a5b1eedba29ba5b4.png" width="350"/>
    </p>
    <p>
     <strong>
      解析
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       Text('Hello, Flutter!')
      </code>
      创建了一个最简单的文本组件。
     </li>
     <li>
      默认使用
      <code>
       defaultTextStyle
      </code>
      ，字体大小、颜色取决于应用主题。
     </li>
    </ul>
    <p>
     虽然设置了主题，但是没有设置 Scaffold，依然丑，我们加一下Scaffold ：
    </p>
    <p class="img-center">
     <img alt="" height="159" src="https://i-blog.csdnimg.cn/direct/37229ef7ac9f40acac7c0fea5efeeee2.png" width="500"/>
    </p>
    <p>
     加了  Scaffold 样式好看了很多，但是位置有点问题，咱们可以在下面调整一下
    </p>
    <h3 id="3.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%A0%B7%E5%BC%8F" name="3.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%A0%B7%E5%BC%8F">
     <span style="color:#ed7976">
      3. 自定义样式
     </span>
    </h3>
    <p>
     <code>
      Text
     </code>
     组件可以通过
     <code>
      style
     </code>
     参数自定义字体样式。
    </p>
    <pre><code class="language-Dart">class _ScTextPageState extends State&lt;ScTextPage&gt; {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Text(
          'Hello, Flutter!',
          style: TextStyle(
            fontSize: 24,
            fontWeight: FontWeight.bold,
            color: Colors.blue,
            fontStyle: FontStyle.italic,
            decoration: TextDecoration.underline,
          ),
        ),
      )
    );
  }
}
</code></pre>
    <p>
     <strong>
      常用样式参数：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       fontSize
      </code>
      ：字体大小
     </li>
     <li>
      <code>
       fontWeight
      </code>
      ：字体粗细（
      <code>
       FontWeight.bold
      </code>
      、
      <code>
       FontWeight.w300
      </code>
      等）
     </li>
     <li>
      <code>
       color
      </code>
      ：文本颜色
     </li>
     <li>
      <code>
       fontStyle
      </code>
      ：字体风格（
      <code>
       FontStyle.normal
      </code>
      、
      <code>
       FontStyle.italic
      </code>
      ）
     </li>
     <li>
      <code>
       decoration
      </code>
      ：文本装饰（如下划线、删除线）
     </li>
    </ul>
    <p class="img-center">
     <img alt="" height="273" src="https://i-blog.csdnimg.cn/direct/cbbe46224be2462bb395cc014aa7939f.png" width="500"/>
    </p>
    <h3 id="4.%20%E6%96%87%E6%9C%AC%E5%AF%B9%E9%BD%90%E4%B8%8E%E6%BA%A2%E5%87%BA%E6%8E%A7%E5%88%B6" name="4.%20%E6%96%87%E6%9C%AC%E5%AF%B9%E9%BD%90%E4%B8%8E%E6%BA%A2%E5%87%BA%E6%8E%A7%E5%88%B6">
     <span style="color:#ed7976">
      4. 文本对齐与溢出控制
     </span>
    </h3>
    <p>
     当文本过长时，可以使用
     <code>
      overflow
     </code>
     参数控制显示方式。
    </p>
    <pre><code class="language-Dart">Text(
  'Text 组件是 Flutter 中最常用的 UI 组件之一，用于显示文本内容。',
  maxLines: 1,
  overflow: TextOverflow.ellipsis,
)
</code></pre>
    <p>
     <strong>
      重要参数：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       maxLines
      </code>
      ：限制最大行数
     </li>
     <li>
      <code>
       overflow
      </code>
      ：文本溢出处理方式（
      <code>
       TextOverflow.ellipsis
      </code>
      显示省略号）
     </li>
    </ul>
    <p class="img-center">
     <img alt="" height="165" src="https://i-blog.csdnimg.cn/direct/943bcea21349401399ae61542b3e9f44.png" width="600"/>
    </p>
    <h3 id="5.%20%E5%A4%96%E8%BE%B9%E8%B7%9D" name="5.%20%E5%A4%96%E8%BE%B9%E8%B7%9D">
     <span style="color:#ed7976">
      5. 外边距
     </span>
    </h3>
    <p>
     在 Flutter 中，
     <code>
      Text
     </code>
     组件本身没有直接的
     <code>
      margin
     </code>
     属性，但可以通过以下 4 种方式设置外边距。
    </p>
    <h4 id="5.1%C2%A0%E4%BD%BF%E7%94%A8%20Container%20%E5%8C%85%E8%A3%B9" name="5.1%C2%A0%E4%BD%BF%E7%94%A8%20Container%20%E5%8C%85%E8%A3%B9">
     <span style="color:#956fe7">
      5.1 使用 Container 包裹
     </span>
    </h4>
    <p>
     这是最常用的解决方案，通过
     <code>
      Container
     </code>
     的
     <code>
      margin
     </code>
     属性控制外边距：
    </p>
    <pre><code class="language-Dart">class _ScTextPageState extends State&lt;ScTextPage&gt; {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child:Container(
          margin: EdgeInsets.all(20), // 四周外边距
          child: Text('带外边距的文本'),
          ),
        )
      )
    );
  }
}</code></pre>
    <p class="img-center">
     <img alt="" height="242" src="https://i-blog.csdnimg.cn/direct/1bead17654e3474cafa1d919895139d8.png" width="500"/>
    </p>
    <h4 id="5.2%20%E4%BD%BF%E7%94%A8%20Padding%20%E7%BB%84%E4%BB%B6" name="5.2%20%E4%BD%BF%E7%94%A8%20Padding%20%E7%BB%84%E4%BB%B6">
     <span style="color:#956fe7">
      5.2 使用 Padding 组件
     </span>
    </h4>
    <p>
     如果只需要单边外边距，推荐使用
     <code>
      Padding
     </code>
     组件：
    </p>
    <pre><code class="language-Dart">Padding(
    // padding: EdgeInsets.all(40),//可以all
    padding: EdgeInsets.only(left: 20,right: 40,),//可以上下左右随意设置
    child: Text('左侧缩进20，右侧缩进40'),
)</code></pre>
    <h4 id="5.3%20%E5%9C%A8%20Row%2FColumn%20%E4%B8%AD%E4%BD%BF%E7%94%A8" name="5.3%20%E5%9C%A8%20Row%2FColumn%20%E4%B8%AD%E4%BD%BF%E7%94%A8">
     <span style="color:#956fe7">
      5.3 在 Row/Column 中使用
     </span>
    </h4>
    <p>
     当文本位于行列布局中时，可用
     <code>
      SizedBox
     </code>
     间隔：
    </p>
    <pre><code class="language-Dart">Row(
  children: [
    Text('左边文本'),
    SizedBox(width: 20), // 横向间距
    Text('右边文本')
  ],
)</code></pre>
    <h4 id="5.4%20%E5%8A%A8%E6%80%81%E8%BE%B9%E8%B7%9D%E8%B0%83%E6%95%B4" name="5.4%20%E5%8A%A8%E6%80%81%E8%BE%B9%E8%B7%9D%E8%B0%83%E6%95%B4">
     <span style="color:#956fe7">
      5.4 动态边距调整
     </span>
    </h4>
    <p>
     使用
     <code>
      MediaQuery
     </code>
     响应式布局：
    </p>
    <pre><code class="language-Dart">Container(
  margin: EdgeInsets.symmetric(
    horizontal: MediaQuery.of(context).size.width * 0.1
  ),
  child: Text('自适应边距文本'),
)</code></pre>
    <h4 id="5.5%C2%A0%E5%85%B3%E9%94%AE%E5%8C%BA%E5%88%AB%E8%AF%B4%E6%98%8E" name="5.5%C2%A0%E5%85%B3%E9%94%AE%E5%8C%BA%E5%88%AB%E8%AF%B4%E6%98%8E">
     <span style="color:#956fe7">
      5.5 关键区别说明
     </span>
    </h4>
    <table style="width:500px">
     <thead>
      <tr>
       <th>
        属性/组件
       </th>
       <th>
        作用范围
       </th>
       <th>
        典型使用场景
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         margin
        </code>
       </td>
       <td>
        组件外部空间
       </td>
       <td>
        控制与其他组件的间距
       </td>
      </tr>
      <tr>
       <td>
        <code>
         padding
        </code>
       </td>
       <td>
        组件内部空间
       </td>
       <td>
        控制文本内容与容器的间距
       </td>
      </tr>
      <tr>
       <td>
        <code>
         SizedBox
        </code>
       </td>
       <td>
        布局间隔
       </td>
       <td>
        在行列布局中精确控制元素间距
       </td>
      </tr>
     </tbody>
    </table>
    <h4 id="5.6%C2%A0%E8%AE%BE%E7%BD%AE%20margin%20%E6%97%A0%E6%95%88" name="5.6%C2%A0%E8%AE%BE%E7%BD%AE%20margin%20%E6%97%A0%E6%95%88">
     <span style="color:#956fe7">
      5.6 设置 margin 无效
     </span>
    </h4>
    <ul>
     <li>
      <p>
       检查父容器是否允许子组件扩展（如
       <code>
        ListView
       </code>
       需要设置
       <code>
        shrinkWrap: true
       </code>
       ）
      </p>
     </li>
     <li>
      <p>
       确保外层没有被
       <code>
        Center
       </code>
       或
       <code>
        Align
       </code>
       等布局组件限制
      </p>
     </li>
    </ul>
    <h3 id="5.%20%E7%BB%93%E8%AE%BA" name="5.%20%E7%BB%93%E8%AE%BA">
     <span style="color:#ed7976">
      6. 结论
     </span>
    </h3>
    <p>
     <code>
      Text
     </code>
     组件是 Flutter 开发中不可或缺的一部分，它提供了丰富的自定义选项，适用于不同文本场景。掌握
     <code>
      Text
     </code>
     组件的各种参数和特性，可以帮助开发者构建更加美观和灵活的 UI 界面。
    </p>
