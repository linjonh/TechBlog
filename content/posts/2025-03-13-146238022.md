    <h2>
     <a id="_Qt__1">
     </a>
     用 Qt 动画制作炫酷的界面效果：属性动画教程
    </h2>
    <p>
     Qt 是一个非常强大的图形界面框架，不仅能帮助你制作漂亮的界面，还能让你的应用变得更加生动、交互性更强。在本篇博客中，我们将深入学习如何使用 Qt 动画来让界面元素动起来。通过
     <code>
      QPropertyAnimation
     </code>
     ，你可以让控件的属性（如旋转角度、透明度、大小等）平滑过渡，制作出流畅的动画效果。
    </p>
    <h3>
     <a id="_5">
     </a>
     动画简介：为什么要用动画？
    </h3>
    <p>
     动画能让你的应用更加吸引人，提供更好的用户体验。在以下几个方面，动画都可以为你的应用加分：
    </p>
    <ul>
     <li>
      <strong>
       增强用户体验
      </strong>
      ：动画能够使界面更加生动，提升整体的互动性，避免界面显得死板或者突兀。
     </li>
     <li>
      <strong>
       吸引用户注意力
      </strong>
      ：通过动画，你可以将用户的注意力引导到某些重要的界面元素上，例如按钮、提示信息等。
     </li>
     <li>
      <strong>
       改善界面美感
      </strong>
      ：动画能够使得控件变化过程更加自然，比如按钮的点击效果，控件的显示隐藏，背景的渐变等。
     </li>
    </ul>
    <p>
     Qt 提供了强大的动画框架，允许开发者对控件的属性进行平滑过渡。通过使用动画，用户会感受到应用更具现代感和交互感。
    </p>
    <h3>
     <a id="Qt_QPropertyAnimation_15">
     </a>
     Qt 动画的基本概念：
     <code>
      QPropertyAnimation
     </code>
    </h3>
    <h4>
     <a id="_17">
     </a>
     什么是属性动画？
    </h4>
    <p>
     属性动画（Property Animation）是 Qt 中的一种常用动画类型，它允许你在一定的时间内平滑地修改一个控件的属性值。例如，你可以让一个按钮从屏幕左边移动到右边，或者让一个图片在屏幕上旋转，所有这些都可以通过动画来完成。
    </p>
    <p>
     <code>
      QPropertyAnimation
     </code>
     是 Qt 提供的类，用来对控件的属性进行动画处理。你只需要指定控件、动画持续时间、属性目标值等，Qt 会自动处理属性值的过渡和控件的重绘。
    </p>
    <h4>
     <a id="QPropertyAnimation__23">
     </a>
     <code>
      QPropertyAnimation
     </code>
     的核心方法
    </h4>
    <h5>
     <a id="1_setTargetObjectQObject_object_25">
     </a>
     1.
     <code>
      setTargetObject(QObject *object)
     </code>
    </h5>
    <p>
     这个方法用来指定动画的目标对象，即要操作的控件或对象。
    </p>
    <h5>
     <a id="2_setPropertyNameconst_QByteArray_name_28">
     </a>
     2.
     <code>
      setPropertyName(const QByteArray &amp;name)
     </code>
    </h5>
    <p>
     这个方法指定要执行动画的属性名称，比如你可以为
     <code>
      rotation
     </code>
     属性（旋转角度）或者
     <code>
      opacity
     </code>
     属性（透明度）设置动画。
    </p>
    <h5>
     <a id="3_setDurationint_msec_31">
     </a>
     3.
     <code>
      setDuration(int msec)
     </code>
    </h5>
    <p>
     这个方法设置动画的持续时间，以毫秒为单位。比如设置为 1000 毫秒（即 1 秒），表示动画会在 1 秒内完成。
    </p>
    <h5>
     <a id="4_setEndValueconst_QVariant_value_34">
     </a>
     4.
     <code>
      setEndValue(const QVariant &amp;value)
     </code>
    </h5>
    <p>
     这个方法设置动画的目标值。动画会逐步将属性值从起始值变化到目标值。
    </p>
    <h5>
     <a id="5_start_37">
     </a>
     5.
     <code>
      start()
     </code>
    </h5>
    <p>
     启动动画。调用这个方法后，动画就会开始执行，直到达到目标值。
    </p>
    <h4>
     <a id="_40">
     </a>
     动画的工作流程
    </h4>
    <ol>
     <li>
      <strong>
       设置初始值和目标值
      </strong>
      ：你需要设置动画的起始值（通常是当前属性值）和目标值（动画结束时的属性值）。
     </li>
     <li>
      <strong>
       设置动画时间和曲线
      </strong>
      ：你可以指定动画的持续时间、缓动效果等。
     </li>
     <li>
      <strong>
       启动动画
      </strong>
      ：调用
      <code>
       start()
      </code>
      方法开始动画，动画会根据设定的时长和缓动曲线逐渐改变属性值。
     </li>
     <li>
      <strong>
       重绘控件
      </strong>
      ：动画会逐步修改控件的属性值，并在每次属性更新时触发控件的重绘（通过
      <code>
       paintEvent()
      </code>
      ）。
     </li>
    </ol>

    <h5>
     <a id="1__139">

     1.
     <strong>
      初始化动画
     </strong>
    </h5>
    <p>
     在
     <code>
      init()
     </code>
     方法中，我们首先创建了一个
     <code>
      QPropertyAnimation
     </code>
     对象：
    </p>
    <pre><code class="prism language-cpp">an <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token function">QPropertyAnimation</span><span class="token punctuation">(</span><span class="token keyword">this</span><span class="token punctuation">,</span> <span class="token string">"rotation"</span><span class="token punctuation">,</span> <span class="token keyword">this</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <p>
     这里的
     <code>
      "rotation"
     </code>
     表示我们要对控件的
     <code>
      rotation
     </code>
     属性应用动画。在
     <code>
      QPropertyAnimation
     </code>
     中，动画是通过属性名来识别的。
    </p>
    <p>
     然后，我们设置了动画的持续时间为 30 毫秒，设置了动画的缓动曲线为
     <code>
      QEasingCurve::InQuad
     </code>
     ，即动画开始时较慢，接着加速：
    </p>
    <pre><code class="prism language-cpp">an<span class="token operator">-&gt;</span><span class="token function">setDuration</span><span class="token punctuation">(</span><span class="token number">30</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

     <a id="2__156">
     </a>
     2.
     <strong>
      设置动画目标值
     </strong>
    </h5>
    <p>
     在
     <code>
      setExpanded()
     </code>
     函数中，我们根据是否展开，分别设置了动画的目标值为 90 或 0：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">if</span> <span class="token punctuation">(</span>expand<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    an<span class="token operator">-&gt;</span><span class="token function">setEndValue</span><span class="token punctuation">(</span><span class="token number">90</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 旋转到 90 度</span>

    <p>
     <code>
      setEndValue(90)
     </code>
     表示动画结束时
     <code>
      rotation
     </code>
     属性的值应该是 90 度，
     <code>
      setEndValue(0)
     </code>
     则表示动画结束时
     <code>
      rotation
     </code>
     的值应该回到 0 度。
    </p>

     <a id="3__170">

     3.
     <strong>
      触发动画并重绘
     </strong>
    </h5>
    <p>
     通过调用
     <code>
      start()
     </code>
     ，我们启动了动画，
     <code>
      rotation
     </code>

     <code>
      rotation

     更新时，
     <code>
      setRotation()
     </code>
     会被调用，更新控件的旋转角度，并触发
     <code>
      update()
     </code>
     来进行重绘。

     4.

    <p>
     在
     <code>
      paintEvent()

      QPainter
