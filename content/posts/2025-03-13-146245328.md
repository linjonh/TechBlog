    <p>
     分组用 box和endbox ，颜色用#xxx，标注用"xxx"
    </p>
    <pre><code class="prism language-bash">box <span class="token string">"浏览器"</span> <span class="token comment">#LightGreen</span>
participant <span class="token string">"浏览器1"</span> as Browser
participant <span class="token string">"浏览器2"</span> as Browser2
endbox

participant <span class="token string">"服务端"</span> as Server <span class="token comment">#orange</span>
</code></pre>

    <p>
     效果：可见图中浏览器1和浏览器2放到了一起，并且叫浏览器的title
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/5f9495d9f7aa49b596ba78e615f4660b.png"/>
    </p>
    <pre><code class="prism language-bash">@startuml

autonumber

actor <span class="token string">"用户"</span> as User

box <span class="token string">"浏览器"</span> <span class="token comment">#LightGreen</span>
participant <span class="token string">"浏览器1"</span> as Browser
participant <span class="token string">"浏览器2"</span> as Browser2
endbox

participant <span class="token string">"服务端"</span> as Server <span class="token comment">#orange</span>

activate User

User -<span class="token operator">&gt;</span> Browser: 输入 URL
activate Browser

Browser -<span class="token operator">&gt;</span> Server: 请求服务器
activate Server

Server -<span class="token operator">&gt;</span> Server: 模板渲染
note right of Server: 这是一个注释

Server -<span class="token operator">&gt;</span> Browser: 返回 HTML
deactivate Server

Browser --<span class="token operator">&gt;</span> User

@enduml
</code></pre>
