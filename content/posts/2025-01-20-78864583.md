    <h3 style="font-family:Arial; line-height:26px; color:rgb(69,69,69); margin:0px; font-size:2.15em; padding:0px">
     值得学习的C/C++语言开源项目
    </h3>
    <div style='margin:0px; padding:0px; color:rgb(69,69,69); font-family:"PingFang SC","Microsoft YaHei",SimHei,Arial,SimSun; font-size:16px'>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (1)ACE
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      庞大、复杂，适合大型项目。开源、免费，不依赖第三方库，支持跨平台。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://www.cs.wustl.edu/~schmidt/ACE.html" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://www.cs.wustl.edu/~schmidt/ACE.html
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (2)Asio
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      Asio基于Boost开发的异步IO库，封装了Socket，简化基于socket程序的开发。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      开源、免费，支持跨平台。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://think-async.com/" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://think-async.com/
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (3)POCO
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      POCO C++ Libraries 提供一套 C++ 的类库用以开发基于网络的可移植的应用程序，功能涉及线程、线程同步、文件系统访问、流操作、共享库和类加载、套接字以及网络协议包括：HTTP、 FTP、SMTP 等；其本身还包含一个 HTTP 服务器，提供 XML 的解析和 SQL 数据库的访问接口。POCO库的模块化、高效的设计及实现使得POCO特别适合嵌入式开发。在嵌入式开发领域，由于C++既适合底层（设备I/O、中断处理等）和高层面向对象开发，越来越流行。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://pocoproject.org/" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://pocoproject.org/
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (4)libevent
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      Libevent是一个轻量级的开源高性能网络库，从它的官网标题：libevent-an event notification library就能知道它的机制是采用事件触发，封装了以下三种事件的响应:IO事件,定时器事件,信号事件。select模型来实现跨平台的操作，Windows环境下支持IOCP。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://libevent.org/" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://libevent.org/
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (5)libev
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      libev和libevent很像，按照作者的介绍，可以作为libevent的替代者，能够提供更高的性能。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      libev是一个高性能事件循环，所实现的功能就是一个强大的reactor。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://software.schmorp.de/pkg/libev.html" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://software.schmorp.de/pkg/libev.html
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (6)c++ sockets library
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      封装了sockets C API的C++类库。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      支持SSL, IPv6, tcp 和 udp sockets, sctp sockets, http协议, 高度可定制的错误处理。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://www.alhem.net/Sockets/" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://www.alhem.net/Sockets/
      </a>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <span style="">
       <span style="margin:0px; padding:0px; font-weight:400; color:rgb(51,102,255)">
        (7)libcurl
       </span>
      </span>
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      libcurl是免费的轻量级的客户端网络库，支持DICT, FILE, FTP, FTPS, Gopher, HTTP, HTTPS, IMAP, IMAPS, LDAP, LDAPS, POP3, POP3S, RTMP, RTSP, SCP, SFTP, SMTP, SMTPS, Telnet, TFTP. 支持SSL, HTTP POST, HTTP PUT, FTP上传, HTTP form上传，代理，cookies, 用户名与密码认证。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      如果你开发的是客户端，libcurl是一个不错的选择。
     </p>
     <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal">
      <a href="http://curl.haxx.se/libcurl/" rel="nofollow noopener noreferrer" style="background:transparent; color:rgb(202,12,22); margin:0px; padding:0px; outline:0px" target="_blank">
       http://curl.haxx.se/libcurl/
      </a>

     - 1. Webbench
    </h3>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     Webbench是一个在linux下使用的非常简单的网站压测工具。它使用fork()模拟多个客户端同时访问我们设定的URL，测试网站在压力下工作的性能，最多可以模拟3万个并发连接去测试网站的负载能力。Webbench使用C语言编写, 代码实在太简洁，源码加起来不到600行。
    </p>

     - 3. cJSON
    </h3>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     cJSON是C语言中的一个JSON编解码器，非常轻量级，C文件只有500多行，速度也非常理想。
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     cJSON也存在几个弱点，虽然功能不是非常强大，但cJSON的小身板和速度是最值得赞赏的。其代码被非常好地维护着，结构也简单易懂，可以作为一个非常好的C语言项目进行学习。
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">

     - 4. CMockery
    </h3>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     cmockery是google发布的用于C单元测试的一个轻量级的框架。它很小巧，对其他开源包没有依赖，对被测试代码侵入性小。cmockery的源代码行数不到3K，你阅读一下will_return和mock的源代码就一目了然了。
    </p>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     主要特点：
    </p>
    <ul style="margin:0px; padding:0px; list-style-position:initial; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     <li style="margin:0px 0px 0px 40px; padding:0px; list-style:disc">
      免费且开源，google提供技术支持；
     </li>
     <li style="margin:0px 0px 0px 40px; padding:0px; list-style:disc">
      轻量级的框架，使测试更加快速简单；
     </li>
     <li style="margin:0px 0px 0px 40px; padding:0px; list-style:disc">
      避免使用复杂的编译器特性，对老版本的编译器来讲，兼容性好;
     </li>
     <li style="margin:0px 0px 0px 40px; padding:0px; list-style:disc">
      并不强制要求待测代码必须依赖C99标准，这一特性对许多嵌入式系统的开发很有用
     </li>
    </ul>
    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">

    <p style="margin-top:0px; margin-bottom:0px; padding-top:0px; padding-bottom:0px; word-wrap:break-word; word-break:normal; color:rgb(69,69,69); font-family:Arial; font-size:14px; line-height:26px">
     libev是一个开源的事件驱动库，基于epoll，kqueue等OS提供的基础设施。其以高效出名，它可以将IO事件，定时器，和信号统一起来，统一放在事件处理这一套框架下处理。基于Reactor模式，效率较高，并且代码精简（4.15版本8000多行），是学习事件驱动编程的很好的资源。
    </p>
