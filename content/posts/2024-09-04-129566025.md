# java 连接数据库

Java 连接数据库的常用方法如下：

1. 使用 JDBC (Java Database Connectivity) 连接数据库。这是 Java 语言提供的用于连接数据库的标准接口。通常需要使用相应数据库的 JDBC 驱动程序来连接数据库。
