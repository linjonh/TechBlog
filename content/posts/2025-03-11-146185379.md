    <h2 style="background-color:transparent">
     预处理工具的算法
    </h2>
    <ul>
     <li>
      <h3 style="background-color:transparent">
       图像增强
      </h3>
      <ul>
       <li>
        <strong>
         加减常量
        </strong>
        ：通过为灰度图像中每个像素的灰度值添加正值或负值，使用图像变得更亮或更暗
       </li>
       <li>
        <strong>
         乘以常量：
        </strong>
        将图像的每个像素值乘以一个常数，可调整图像的亮度和对比度
       </li>
       <li>
        <strong>
         均衡：
        </strong>
        通过直方图均衡方法增强图像对比度，使图像的灰度值分布更均匀
       </li>
       <li>
        <strong>
         像素映射：
        </strong>
        将输入图像的像素按照一定规则映射到另一张图像的对应位置上，形成一张新的图像
       </li>
      </ul>
     </li>
     <li>
      <h3 style="background-color:transparent">
       图像滤波（滤波处理）
      </h3>
      <ul>
       <li>
        <strong>
         卷积：
        </strong>
        使用卷积核对图像进行运算，可实现平滑、模糊、去噪、锐化、边缘提取等效果
       </li>
       <li>
        <strong>
         高斯采样器：
        </strong>
        对输入图像进行子采样和平滑操作，可以减少图像中的纹理和噪声
       </li>
       <li>
        <strong>
         中值滤波：
        </strong>
        使用N×M中值滤波器，可有效去除椒盐噪声等
       </li>
       <li>
        <strong>
         高通滤波：
        </strong>
        保留图像中频率较高的部分，突出边缘等细节特征
       </li>
      </ul>
     </li>
     <li>
      <h3>
       图像变换
      </h3>
      <ul>
       <li>
        <strong>
         扩展：
        </strong>
        用指定的放大倍数放大整个图像或图像的一部分
       </li>
       <li>
        <strong>
         旋转：
        </strong>
        对输入图像的全部或部分执行水平翻转或顺时针旋转
       </li>
       <li>
        <strong>
         仿射变换：
        </strong>
        通过仿射变换产生新的图像，可实现平移、旋转、缩放等几何变换
       </li>
      </ul>
     </li>
     <li>
      <h3 style="background-color:transparent">
       形态学处理
      </h3>
      <ul>
       <li>
        <strong>
         腐蚀：
        </strong>
        降低图像的亮点，可消除噪点像素或小的缺陷
       </li>
       <li>
        <strong>
         膨胀：
        </strong>
        增强图像的明亮特征，同时抑制较暗特征
       </li>
       <li>
        <strong>
         打开：
        </strong>
        先腐蚀后膨胀，删除少量明亮特征，然后增强其余的明亮功能
       </li>
       <li>
        <strong>
         关闭：
        </strong>
        先膨胀后腐蚀，减少或完全消除图像的暗区
       </li>
      </ul>
     </li>
     <li>
      <h3>
       数字图像
      </h3>
      <ul>
       <li>
        <strong>
         量化：
        </strong>
        减少输入图像中离散灰度值的数量，有助于分离相似灰度值的特征
       </li>
       <li>
        <strong>
         采样卷积：
        </strong>
        同时执行卷积核采样，常用于高斯平滑进行降采样
       </li>
       <li>
        <strong>
         二次采样：
        </strong>
        生成分辨率和尺寸较小的输出图像，可提高其他视觉工具的运行速度
       </li>
       <li>
        <strong>
         丢失像素：
        </strong>
        将深度图像中缺失的像素替换为基于周围像素分析的像素值
       </li>
      </ul>
     </li>
    </ul>

       箝位：
      </strong>

     </p>

     卷积
    </h3>
    <blockquote>
     <p>
      Visionpro支持卷积运算符，基于相邻像素值进行修改像素值，Visionpro将输入图像中的每个像素，乘以一个核的数值矩阵，然后替换输出图像中的相应像素，不同卷积核可以实现不同的效果，比如
      <strong>
       平滑、模糊、去噪、锐化、边缘提取等
      </strong>
      ，都可以通过卷积操作来完成。
     </p>

    <h3>
     高斯采样器
    </h3>
    <blockquote>
     <p>
      对输入图像进行子采样，使输出图像仅包含原始像素的一部分，并平滑图像
     </p>
     <ol>
      <li>
       当视觉工具在缩小的图像上同样有效地工作并且想要提高应用程序的速度时，可使用采样操作。例如：2*2图像缩小一倍
      </li>
      <li>
       使用平滑操作可减轻图像中纹理，信号噪声带来的影响
      </li>
      <li>
       可调整幅度偏移因子，范围为 -7 至 7 .使用负值作为移动因子可产生较暗的输出图像，而使用正值可产生较亮的输出图像
      </li>
     </ol>

    <h3>
     量化
    </h3>
    <h5>
     具体详细链接：
     <a href="https://blog.csdn.net/2403_88923953/article/details/145900466" title="Halcon 入门篇基础和 数字图像的认识_halcon只留下黑白像素-CSDN博客">
      Halcon 入门篇基础和 数字图像的认识_halcon只留下黑白像素-CSDN博客
     </a>
    </h5>
    <blockquote>
     <p>
      减少输入图像中离散灰度值的数量。执行量化操作可以帮助分离，具有相似灰度值的单个特征，或将所需特征减少为单个灰度值，这可以使它们更易于使用其他视觉工具进行分析。
     </p>
     <ol>
      <li>
       量化等级越多，所得图像层次越丰富，灰度分辨率越高，质量越好，数据量大
      </li>
      <li>
       量化等级越少，图像层次丰富，灰度分辨率低，质量变差，会出现假轮廓现象，数据量小
      </li>
     </ol>

     二次采样
    </h3>
    <blockquote>
     <p>
      生成输出图像，其中分辨率和尺寸会减小。子采样可以使其他视觉工具 在缩小的图像上更快的运行，尽管减小图像尺寸可能会导致精度降低。子采样器操作提供了两种类型的算法来生成输出图像
