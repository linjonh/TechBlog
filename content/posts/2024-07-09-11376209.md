     Linux系统中设置默认的Java版本
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <span style="font-size:18px">
      有时候会根据某些需要或是不经意间（对于初学的我来说是后者了），会在系统中安装多个版本的Java。有些早期的代码需要用早期的Java版来编译运行。可以通过
     </span>
    </p>
    <p>
    </p>
    <pre><code class="language-plain">$ java -version</code></pre>
    <br/>
    <span style="font-size:18px">
     来查看当前使用的Java版本。如果你安装了一个新的Java版本，可以使用以下方法来更改默认的Java版本：
    </span>
    <p>
    </p>
    <p style="text-align:left">
     <span style="font-size:18px">
      0、需要使用update-alternatives命令，通过
     </span>
    </p>
    <p style="text-align:left">
     <span style="font-size:18px">
     </span>
    </p>
    <pre><code class="language-plain">$ man update-alternatives</code></pre>
    <br/>
    <span style="font-size:18px">
     查看使用方法，需要使用 --install 和 --config 两个子命令。
    </span>
    <p>
    </p>
    <p style="text-align:left">
     <span style="font-size:18px">
      1、使用update-alternatives命令告诉系统你新安装的版本：
     </span>
    </p>
    <p style="text-align:left">
     <span style="font-size:18px">
     </span>
    </p>
    <pre><code class="language-plain">$ sudo update-alternatives --install /usr/bin/java java /opt/jdk1.7.0_25/bin/java 2
</code></pre>
    <br/>
    <p>
     <span style="font-size:18px">
      2、再次使用update-alternatives命令选择版本
     </span>
    </p>
    <p style="text-align:left">
    </p>
    <pre><code class="language-plain">$ sudo update-alternatives --config java</code></pre>
    <br/>
    <span style="font-size:18px">
     我的会出现如下信息，我想选择Java 7，输入1就好了。
    </span>
    <p>
    </p>
    <p style="text-align:left">
     <span style="font-size:18px">
     </span>
    </p>
    <pre><code class="language-plain">There are 2 choices for the alternative java (providing /usr/bin/java).

  Selection    Path                                           Priority   Status
------------------------------------------------------------
  0            /usr/lib/jvm/java-6-openjdk-i386/jre/bin/java   1061      auto mode
* 1            /opt/jdk1.7.0_25/bin/java                       2         manual mode
  2            /usr/lib/jvm/java-6-openjdk-i386/jre/bin/java   1061      manual mode

Press enter to keep the current choice[*], or type selection number: 1</code></pre>
    <br/>
    <p>
     <span style="font-size:18px">
      3、验证默认Java版本设置成功
     </span>
    </p>
    <p>
     <span style="font-size:18px">
     </span>
    </p>
    <pre><code class="language-plain">$ java -version</code></pre>
    <p>
     <span style="font-size:18px">
      会出现类似信息：
     </span>
    </p>
    <p>
    </p>
    <p>
    </p>
    <pre><code class="language-plain">java version "1.7.0_25"
Java(TM) SE Runtime Environment (build 1.7.0_25-b15)
Java HotSpot(TM) Server VM (build 23.25-b01, mixed mode)</code></pre>
