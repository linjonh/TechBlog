    <h2>
     <a id="_0">
     </a>
     问题
    </h2>
    <p>
     这是一张区域表，依据业务要求：根据某些条件，来判断使用条件。举例：
     <br/>
     <strong>
      错误SQL语句
     </strong>
     ：
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">SELECT</span> <span class="token operator">*</span>
<span class="token keyword">FROM</span> s_sg_area s
<span class="token keyword">WHERE</span>
	<span class="token keyword">case</span>
	<span class="token keyword">when</span> <span class="token boolean">true</span>
	<span class="token keyword">then</span> s<span class="token punctuation">.</span>area_code <span class="token operator">=</span> <span class="token string">'440111015004'</span>
	<span class="token keyword">else</span> s<span class="token punctuation">.</span>area_type <span class="token operator">=</span> <span class="token string">'03'</span>
	<span class="token keyword">end</span>
</code></pre>
    <p>
     <strong>
      报错信息
     </strong>
     ：
    </p>
    <pre><code class="prism language-sql"><span class="token punctuation">[</span>执行语句<span class="token number">1</span><span class="token punctuation">]</span>:
<span class="token keyword">SELECT</span> <span class="token operator">*</span>
<span class="token keyword">FROM</span> s_sg_area s
<span class="token keyword">WHERE</span>
	<span class="token keyword">case</span>
	<span class="token keyword">when</span> <span class="token boolean">true</span>
	<span class="token keyword">then</span> s<span class="token punctuation">.</span>area_code <span class="token operator">=</span> <span class="token string">'440111015004'</span>
	<span class="token keyword">else</span> s<span class="token punctuation">.</span>area_type <span class="token operator">=</span> <span class="token string">'03'</span>
	<span class="token keyword">end</span>
执行失败<span class="token punctuation">(</span>语句<span class="token number">1</span><span class="token punctuation">)</span>
第 <span class="token number">6</span> 行<span class="token punctuation">,</span> 第 <span class="token number">19</span> 列<span class="token punctuation">[</span><span class="token operator">=</span><span class="token punctuation">]</span>附近出现错误: 
语法分析出错

<span class="token number">1</span>条语句执行失败
</code></pre>
    <h2>
     <a id="_31">
     </a>
     折中解决方案
    </h2>
    <p>
     利用判断条件的逻辑或的特性（
     <strong>
      当第一个条件为true时，不执行第二个条件
     </strong>
     ）。示例如下:
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">select</span> s<span class="token punctuation">.</span><span class="token operator">*</span>
<span class="token keyword">from</span> s_sg_area s
<span class="token keyword">where</span> <span class="token number">1</span><span class="token operator">=</span><span class="token number">1</span>
<span class="token comment">-- xxx 不为空 执行</span>
<span class="token operator">and</span> <span class="token punctuation">(</span><span class="token punctuation">(</span>? <span class="token operator">is</span> <span class="token boolean">null</span> <span class="token punctuation">)</span> <span class="token operator">or</span> s<span class="token punctuation">.</span>AREA_CODE <span class="token operator">=</span> <span class="token string">'440111015004'</span> <span class="token punctuation">)</span>
<span class="token comment">-- xxx 为空 执行</span>
<span class="token operator">and</span> <span class="token punctuation">(</span><span class="token punctuation">(</span>? <span class="token operator">is</span> <span class="token operator">not</span> <span class="token boolean">null</span> <span class="token punctuation">)</span> <span class="token operator">or</span> s<span class="token punctuation">.</span>AREA_TYPE <span class="token operator">=</span> <span class="token string">'03'</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     刚涉及到达梦数据库，不是很了解，如有更好的方案，还请赐教。
    </p>
