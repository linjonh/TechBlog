# 我的目标->MSDBA

三个月过去了,年初订考研究生,放弃了,不是我想要的,后来部门经理问今年有什么考证打算,也没回复,自己认为考证是自己的事,并且对考证不怎么感冒,三个月后的我,看看过往,自己的目标,我自己都回答不出,看到书上的一句话.
向着自己的目标不计代价的努力
,看看现在的我.....似乎少了个明确的目标,我知道这样是没法实现我的梦想的,在此给自己的目标做个存档,开始一段
MCDBA
的考证路途.

目标早在我心中出现过几次,就是没有确定下来,让之前段时间的我感觉有点迷茫,今天也算是我向自己目标的起跑吧,凡事有过程,有阶段性,从微软copy了一下,感觉可以按照这个来取得
MCDBA
认证

微软认证数据库管理员 (MCDBA) 证书是对实施和管理 Microsoft SQL Server™ 数据库的专业人员的最高级认证

|  |  |
| --- | --- |
| 第 1 步 | **确定哪项认证对您适合**   微软提供的认证着重于特定的专业技术领域和工作技能范围。您可以根据自己的经验、技能和兴趣决定哪项认证适合您。如果想了解哪种职业适合哪项认证，请单击下面的标题了解各项认证的说明。微软认证分为以下几个主要专业技术领域：   * **系统工程**       [微软认证系统工程师+Internet](http://www.microsoft.com/china/learning/MSattestation/mcsei/default.aspx)       [微软认证系统工程师 (MCSE)](http://www.microsoft.com/china/learning/MSattestation/mcse/default.aspx)       [微软认证专家+Internet (MCP+I)](http://www.microsoft.com/china/learning/MSattestation/mcpi/default.aspx)       [微软认证专家 (MCP)](http://www.microsoft.com/china/learning/MSattestation/mcp/default.aspx)           **解决方案开发**       [微软认证解决方案开发专家 (MCSD)](http://www.microsoft.com/china/learning/MSattestation/mcsd/default.aspx)       [微软认证专家+网站构造 (MCP+SB)](http://www.microsoft.com/china/learning/MSattestation/mcpsb/default.aspx)       [微软认证专家 (MCP)](http://www.microsoft.com/china/learning/MSattestation/mcp/default.aspx)           **数据库管理**       [微软认证数据库管理员 (MCDBA)](http://www.microsoft.com/china/learning/MSattestation/mcdba/default.aspx)       [微软认证专家 (MCP)](http://www.microsoft.com/china/learning/MSattestation/mcp/default.aspx)           **培训**       [微软认证讲师 (MCT)](http://www.microsoft.com/china/learning/MSattestation/mct/default.aspx) |
| 第 2 步 | **通过微软产品或技术获得实际操作经验**   高级认证（ [MCSE+I](http://www.microsoft.com/china/learning/MSattestation/mcsei/default.aspx) 、 [MCSE](http://www.microsoft.com/china/learning/MSattestation/mcse/default.aspx) 、 [MCSD](http://www.microsoft.com/china/learning/MSattestation/mcsd/default.aspx) 、 [MCDBA](http://www.microsoft.com/china/learning/MSattestation/mcdba/default.aspx) ）需要至少一年使用相关微软产品和技术的经验。其他认证（ [MCP+SB](http://www.microsoft.com/china/learning/MSattestation/mcpsb/default.aspx) 、 [MCP+I](http://www.microsoft.com/china/learning/MSattestation/mcpi/default.aspx) 、 [MCP](http://www.microsoft.com/china/learning/MSattestation/mcp/default.aspx) 、 [MCT](http://www.microsoft.com/china/learning/MSattestation/mct/default.aspx) ）建议有 6 到 12 个月的使用经验。关于建议使用经验的更多信息，请通过选择上面的标题了解这些认证的要求。 |
| 第 3 步 | **阅读考试指南并参加练习测试**   在您选择了想要参加的考试后，请阅读备考指南，了解该考试所考查的技能。请访问 [查找考试指南页](http://www.microsoft.com/traincert/mcpexams/prepare/findexam.asp) 搜索您计划参加的考试的准备指南。您还可以通过模拟考试来评估自己的技能等级。虽然您的模拟考试得分不一定反映出您在实际认证考试中取得的成绩，但是，模拟考试为您提供了评估自己掌握的知识和考前准备情况的机会，并且您可以及时获得反馈。许多公司销售模拟考试产品；微软的模拟考试提供商计划对模拟考试提供商进行评估，以帮助 MCP 应试者选择最好的产品。 [请了解有关模拟考试的更多信息。](http://www.microsoft.com/china/learning/Mcpexams/prepare/practicetests.aspx) |
| 第 4 步 | **通过培训弥补您的经验**   您还可以使用以下培训资源补充自己的技能和经验：   * [微软正式教程 (MOC)](http://www.microsoft.com/china/traincert/training/moc/free.asp) * [微软出版社](http://www.microsoft.com/mspress/china/) * [微软认证高级技术培训中心 (Microsoft CPLS)](http://www.microsoft.com/china/traincert/training/providers.asp) * [微软授权院校培训计划 (Microsoft AATP)](http://www.microsoft.com/education/default.mspx) * [微软技术信息网络 (TechNet)](http://www.microsoft.com/china/technet/default.mspx) * [微软开发人员网络 (MSDN)](http://www.microsoft.com/china/msdn/) |
| 第 5 步 | **将您的技能应用到实际环境中**   使用并了解您计划获得认证的微软产品或技术。继续将您的技能应用到实际环境中并积累自己的经验基础。在培训后立即实际应用自己所学的技能将有助于您记住相关的知识。 |
| 第 6 步 | **考试注册**   当您认为自己已做好考试的充分准备时，您应该了解 [如何进行考试注册](http://www.microsoft.com/learning/mcpexams/register/mcpsteps.asp) 。Prometric 和 VUE 管理所有的微软认证专家考试。     在前往考试中心参加考试时，一定要带上自己的照片 ID 和其他 ID。微软要求认证应试者在参加认证考试前接受保密协议。有关详细信息，请参见 [考试中心有关事项](http://www.microsoft.com/china/learning/mcpexams/faq/procedures.aspx#D) 的常见问题解答。 |
