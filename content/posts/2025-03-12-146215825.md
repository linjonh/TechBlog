    <h2>
     <a id="Tone_Mapping_0">
     </a>
     色调映射（Tone Mapping）参数
    </h2>
    <h3>
     <a id="_1">
     </a>
     色调映射算法类型
    </h3>
    <p>
     作用：决定动态范围压缩的方式和视觉风格。
     <br/>
     常见值：
    </p>
    <ul>
     <li>
      hable（Filmic）：模拟胶片曲线，保留高光细节，适合自然场景
     </li>
     <li>
      reinhard：全局压缩，平衡高光和暗部
     </li>
     <li>
      aces（ACES Filmic）：基于电影工业标准，提供均衡的对比度和色彩表现
     </li>
     <li>
      linear：线性缩放，可能导致高光过曝或暗部丢失细节
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8a1d0b0e4f53480aa9cf4d097886273c.png"/>
    </p>
    <h3>
     <a id="_11">
     </a>
     动态范围拓展参数
    </h3>
    <p>
     作用：控制HDR到LDR的亮度压缩比例。
     <br/>
     常见参数
    </p>
    <ul>
     <li>
      peak：输入HDR的峰值亮度（单位：nit），默认根据元数据自动检测
     </li>
     <li>
      desat：去饱和度强度（0-1），防止高光区域颜色失真（如desat=0表示完全保留颜色）
     </li>
    </ul>
    <h3>
     <a id="_16">
     </a>
     对比度与细节增强
    </h3>
    <p>
     作用：调整图像明暗对比和局部细节。
     <br/>
     常见参数：
    </p>
    <ul>
     <li>
      contrast：全局对比度调整（如contrast=1.0为原始对比度）。
     </li>
     <li>
      detail：局部细节增强系数（如detail=0.5提升纹理）
     </li>
    </ul>
    <h3>
     <a id="_22">
     </a>
     高光/阴影控制
    </h3>
    <p>
     作用：单独调节高光和暗部区域的亮度。
     <br/>
     常见参数：
    </p>
    <ul>
     <li>
      highlight：高光压缩强度（0-1），防止过曝。
     </li>
     <li>
      shadow：暗部提亮强度（0-1），提升可视细节
     </li>
    </ul>
    <h2>
     <a id="Gamut_Mapping_28">
     </a>
     色域映射（Gamut Mapping）参数
    </h2>
    <h3>
     <a id="_29">
     </a>
     输入/输出色域标准
    </h3>
    <p>
     作用：定义源色域和目标色域范围。
     <br/>
     常见值：
    </p>
    <ul>
     <li>
      输入色域：bt.2020（HDR）、bt.709（SDR）。
     </li>
     <li>
      输出色域：display-p3（广色域显示器）、srgb（标准色域）
     </li>
    </ul>
    <h3>
     <a id="_36">
     </a>
     色域裁剪方式
    </h3>
    <p>
     作用：处理超出目标色域的颜色。
     <br/>
     常见方法：
    </p>
    <ul>
     <li>
      硬裁剪：直接截断超界颜色，可能导致色阶断裂。
     </li>
     <li>
      软裁剪：通过渐变压缩，保留平滑过渡
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/df0fef7a6a2a48b7bac9e67d87ac40c8.png"/>
     </li>
    </ul>
    <pre><code class="prism language-shell"><span class="token comment"># 硬剪裁示例：直接截断BT.2020到BT.709</span>
ffmpeg <span class="token parameter variable">-i</span> input.mov <span class="token parameter variable">-vf</span> <span class="token string">"zscale=matrix=bt709:range=limited"</span> output_sdr.mp4

<span class="token comment"># 软剪裁示例：色度压缩+饱和度调整</span>
ffmpeg <span class="token parameter variable">-i</span> input.mov <span class="token parameter variable">-vf</span> <span class="token string">"zscale=transfer=linear,tonemap=desat=0.5"</span> output_soft.mp4
</code></pre>
    <h3>
     <a id="_49">
     </a>
     色彩饱和度调整
    </h3>
    <p>
     作用：补偿色域缩小导致的颜色褪色。
     <br/>
     常见参数：
    </p>
    <ul>
     <li>
      saturation：全局饱和度增益（如saturation=1.2提升鲜艳度）。
     </li>
     <li>
      hue_shift：色相偏移角度（如hue_shift=5微调颜色倾向）
     </li>
    </ul>
    <h3>
     <a id="_55">
     </a>
     颜色空间转换参数
    </h3>
    <p>
     作用：调整中间色彩空间的转换方式。
     <br/>
     常见值：
    </p>
    <ul>
     <li>
      <p>
       ipt：基于感知的线性色彩空间，适合色域映射
      </p>
     </li>
     <li>
      <p>
       lab：分离亮度与色度，减少映射失真
      </p>
     </li>
    </ul>
