    <p>
     <strong>
      1.切换到oracle账号
     </strong>
     <br/>
     su - oracle
     <br/>
     <strong>
      2.查看监听器的状况
     </strong>
     <br/>
     lsnrctl status
     <br/>
     <strong>
      3.停止监听器
     </strong>
     <br/>
     lsnrctl stop
     <br/>
     <strong>
      4.再次查看监听器的状况
     </strong>
     <br/>
     lsnrctl status
     <br/>
     <strong>
      5.进入SQLPlus
     </strong>
     <br/>
     sqlplus /nolog
     <br/>
     <strong>
      6.管理员账号连接数据库
     </strong>
     <br/>
     conn / as sysdba
     <br/>
     <strong>
      7.关闭数据库
     </strong>
     <br/>
     shutdown immediate
     <br/>
     <strong>
      8.退出SQLPlus
     </strong>
     <br/>
     Exit
     <br/>
     <strong>
      9.启动监听器
     </strong>
     <br/>
     lsnrctl start
     <br/>
     <strong>
      10.查看监听器的状况
     </strong>
     <br/>
     lsnrctl status
     <br/>
     <strong>
      11.进入SQLPlus
     </strong>
     <br/>
     sqlplus /nolog
     <br/>
     <strong>
      12.管理员账号连接数据库
     </strong>
     <br/>
     conn / as sysdba
     <br/>
     <strong>
      13.启动数据库
     </strong>
     <br/>
     startup
     <br/>
     <strong>
      14.退出SQLPlus
     </strong>
     <br/>
     Exit
     <br/>
     <strong>
      15.以普通用户身份登录
     </strong>
     <br/>
     sqlplus 账号/密码
    </p>
