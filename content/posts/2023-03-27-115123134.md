| 1.1  绑定  /  解绑物联卡 |

|  |
| --- |
| 用户在蓝蜂科技购买的物联卡，卡身印有该卡的唯一码串号，如下图： |

|  |
| --- |
| 使用管理员账号登录  EMCP  平台，在后台管理  —>  物联卡管理功能中，使用串号将该卡绑定到此管理  员账号，之后就可以在此页面进行此物联卡状态的检验和查询，如果物联卡作废，可以使用解绑按钮对该  卡进行解绑，如下图： |
