    <h4>
     方法 1：使用
     <code>
      Arrays.asList
     </code>
    </h4>
    <pre><code class="language-java">import java.util.Arrays;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        Long[] ids = {1L, 2L, 3L, 4L};
        List&lt;Long&gt; list = Arrays.asList(ids);
        System.out.println(list); // 输出: 
    }
}</code></pre>
    <h5>
     注意：
    </h5>
    <ul>
     <li>
      <code>
       Arrays.asList
      </code>
      返回的是一个固定大小的
      <code>
       List
      </code>
      ，不能添加或删除元素。
     </li>
     <li>
      如果
      <code>
       ids
      </code>
      是
      <code>
       long[]
      </code>
      （基本类型数组），
      <code>
       Arrays.asList
      </code>
      会将其视为单个元素，而不是转换为
      <code>
       List&lt;Long&gt;
      </code>
      。
     </li>
    </ul>
    <h4>
     方法 2：使用
     <code>
      Stream
     </code>
     （Java 8+）
    </h4>
    <pre><code class="language-java">import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class Main {
    public static void main(String[] args) {
        Long[] ids = {1L, 2L, 3L, 4L};
        List&lt;Long&gt; list = Arrays.stream(ids).collect(Collectors.toList());
        System.out.println(list); // 输出: 
    }
}</code></pre>
    <h5>
     优点：
    </h5>
    <ul>
     <li>
      返回的是一个可变的
      <code>
       List
      </code>
      ，支持添加或删除元素。
     </li>
     <li>
      适用于
      <code>
       Long[]
      </code>
      或
      <code>
       long[]
      </code>
      数组。
     </li>
    </ul>
    <h4>
     方法 3：使用
     <code>
      Collections.addAll
     </code>
    </h4>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        Long[] ids = {1L, 2L, 3L, 4L};
        List&lt;Long&gt; list = new ArrayList&lt;&gt;();
        Collections.addAll(list, ids);
        System.out.println(list); // 输出: 
    }
}</code></pre>
    <h5>
     优点：
    </h5>
    <ul>
     <li>
      返回的是一个可变的
      <code>
       List
      </code>
      ，支持添加或删除元素。
     </li>
    </ul>
    <h4>
     方法 4：手动遍历数组
    </h4>
    <pre><code class="language-java">import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        Long[] ids = {1L, 2L, 3L, 4L};
        List&lt;Long&gt; list = new ArrayList&lt;&gt;();
        for (Long id : ids) {
            list.add(id);
        }
        System.out.println(list); // 输出: 
    }
}</code></pre>
    <h5>
     优点：
    </h5>
    <ul>
     <li>
      灵活性高，可以在遍历时进行额外操作。
     </li>
    </ul>
    <h4>
     方法 5：使用
     <code>
      Guava
     </code>
     库（第三方库）

    <pre><code class="language-java">import com.google.common.collect.Lists;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        Long[] ids = {1L, 2L, 3L, 4L};
        List&lt;Long&gt; list = Lists.newArrayList(ids);
        System.out.println(list); // 输出: 
    }
}</code></pre>
    <h5>
     优点：
    </h5>
    <ul>
     <li>
      代码简洁，适用于多种集合操作。
     </li>
    </ul>
