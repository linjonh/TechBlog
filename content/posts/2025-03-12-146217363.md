    <p>
     docker 相关的命令
    </p>
    <h2>
     <a id="_1">
     </a>
     在公共仓库中下载
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> pull bitnami/postgresql:12.8.0
</code></pre>
    <h2>
     <a id="_7">
     </a>
     查看镜像
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> images <span class="token operator">|</span><span class="token function">grep</span> postgresql
</code></pre>
    <h2>
     <a id="tag_13">
     </a>
     打tag推送到本地仓库
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> tag  postgresql:12.8.0 docker.公司域名.com/library/postgresql:12.8.0
</code></pre>
    <h2>
     <a id="_19">
     </a>
     推送到本地仓库
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> push docker.公司域名com/library/postgresql:12.8.0
</code></pre>
    <h2>
     <a id="_25">
     </a>
     镜像导出，在没有网络的情况下转移
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> save postgresql:12.8.0 <span class="token operator">&gt;</span> postgresql-12.8.0.tar
</code></pre>
    <h2>
     <a id="_31">
     </a>
     镜像加载
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> load <span class="token operator">&lt;</span> postgresql-12.8.0.tar
</code></pre>
    <h2>
     <a id="ID_37">
     </a>
     查询服务的ID
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token function">ps</span> <span class="token operator">|</span><span class="token function">grep</span> postgresql 运行中的容器
<span class="token function">docker</span> <span class="token function">ps</span>  <span class="token parameter variable">-a</span> <span class="token operator">|</span><span class="token function">grep</span> postgresql  全部的容器
<span class="token comment">#根据id查 日志</span>
<span class="token function">docker</span> logs <span class="token parameter variable">-f</span> f8911198f0d3 <span class="token parameter variable">--tail</span><span class="token operator">=</span><span class="token number">100</span>
</code></pre>
    <h2>
     <a id="_46">
     </a>
     查看该镜像的元信息
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> inspect f8911198f0d3 
</code></pre>
    <h2>
     <a id="_52">
     </a>
     重启镜像服务
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> restart f8911198f0d3 
</code></pre>
    <h2>
     <a id="_58">
     </a>
     停止镜像服务
    </h2>
    <pre><code class="prism language-bash"><span class="token function">docker</span> stop f8911198f0d3 
</code></pre>
    <p>
     #启动镜像服务
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> start f8911198f0d3 
</code></pre>
    <p>
     #删除镜像服务
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token function">rm</span> f8911198f0d3
</code></pre>
    <p>
     #删除镜像
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> rmi postgresql:12.8.0
</code></pre>
    <p>
     #进入容器
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token builtin class-name">exec</span> <span class="token parameter variable">-it</span> f8911198f0d3 <span class="token function">bash</span>
</code></pre>
    <p>
     容器中的数据库目录
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /bitnami/postgresql/data
</code></pre>
    <p>
     宿主机中的的目录
