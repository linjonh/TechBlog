    <h3>
     <a id="IntelliJ_IDEA__0">
     </a>
     <strong>
      IntelliJ IDEA 插件开发指南
     </strong>
    </h3>
    <p>
     IntelliJ IDEA 是 JetBrains 开发的一款强大的 IDE，支持插件扩展，使其更具灵活性。你可以使用
     <strong>
      Java
     </strong>
     和
     <strong>
      Maven
     </strong>
     构建插件，实现各种自定义功能，比如代码自动生成、重构增强、代码检查等。
    </p>
    <hr/>
    <h3>
     <a id="1_IDEA__5">
     </a>
     <strong>
      1. IDEA 插件开发概述
     </strong>
    </h3>
    <h4>
     <a id="11__IDEA__6">
     </a>
     <strong>
      1.1 什么是 IDEA 插件
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       插件（Plugin）
      </strong>
      是一种扩展
      <strong>
       IDEA
      </strong>
      的方式，允许开发者添加新功能、增强现有功能或集成外部工具。
     </li>
     <li>
      JetBrains 提供了
      <strong>
       IntelliJ Platform SDK
      </strong>
      ，用于开发插件。
     </li>
    </ul>
    <h4>
     <a id="12__IDEA__10">
     </a>
     <strong>
      1.2 为什么开发 IDEA 插件
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       提高开发效率
      </strong>
      ：自动化重复任务，例如 Javadoc 生成、代码格式化等。
     </li>
     <li>
      <strong>
       增强 IDE 功能
      </strong>
      ：添加新工具，如定制代码检查、快捷操作。
     </li>
     <li>
      <strong>
       集成第三方工具
      </strong>
      ：如数据库管理、API 调试、Git 扩展等。
     </li>
     <li>
      <strong>
       商业化插件
      </strong>
      ：在 JetBrains Marketplace 发布插件，甚至可以收费。
     </li>
    </ul>
    <h4>
     <a id="13_IDEA__16">
     </a>
     <strong>
      1.3 IDEA 插件开发者路径
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       掌握 Java 基础
      </strong>
      <ul>
       <li>
        面向对象编程
       </li>
       <li>
        反射、注解、Lambda 表达式
       </li>
      </ul>
     </li>
     <li>
      <strong>
       熟悉 IntelliJ Platform
      </strong>
      <ul>
       <li>
        了解
        <strong>
         IntelliJ Platform SDK
        </strong>
        结构
       </li>
       <li>
        学习
        <strong>
         PSI（Program Structure Interface）
        </strong>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       掌握 IDEA 插件 API
      </strong>
      <ul>
       <li>
        <code>
         AnAction
        </code>
        （操作）
       </li>
       <li>
        <code>
         PsiElement
        </code>
        （代码解析）
       </li>
       <li>
        <code>
         Editor
        </code>
        （文本编辑）
       </li>
       <li>
        <code>
         Notification
        </code>
        （通知）
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用 Maven 构建插件
      </strong>
      <ul>
       <li>
        依赖
        <strong>
         IntelliJ Platform SDK
        </strong>
       </li>
       <li>
        使用
        <strong>
         maven-compiler-plugin
        </strong>
        、
        <strong>
         idea-plugin
        </strong>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       发布到 JetBrains Marketplace
      </strong>
      <ul>
       <li>
        生成插件 ZIP
       </li>
       <li>
        注册
        <strong>
         JetBrains 开发者账号
        </strong>
       </li>
       <li>
        提交审核
       </li>
      </ul>
     </li>
    </ol>
