# EhViewer 安装与使用指南

## EhViewer 安装与使用指南

[EhViewer



项目地址:https://gitcode.com/gh_mirrors/ehvi/EhViewer](https://gitcode.com/gh_mirrors/ehvi/EhViewer/?utm_source=artical_gitcode&index=top&type=card&webUrl "EhViewer")

### 1. 项目介绍

EhViewer 是一个基于 Android 的开源应用程序，旨在为用户提供轻量级且高效的 E-Hentai 网站浏览体验。该项目由多个开发者共同维护，包括 Hippo Seven、NekoInverter 和 Tarsin Norbin。EhViewer 实现了Material Design 3界面风格，并支持动态色彩，同时还提供了画廊的阅读、下载、收藏、评分和评论等特性。该应用遵循 GPL v3 许可协议，可在多种设备上运行，并支持多国语言。

### 2. 项目快速启动

#### 环境要求

确保您的设备满足以下条件：

* Android 8.0+（对于较旧版本的 Android，支持可能有限）
* 允许安装来自第三方来源的应用程序

#### 安装步骤

1. 打开浏览器，前往
   [EhViewer 的 GitHub Release 页面](https://github.com/EhViewer-NekoInverter/EhViewer/releases)
   。
2. 下载适用于您设备的最新 APK 文件。
3. 在您的设备设置中，允许从未知来源安装应用程序（通常在安全设置中找到）。
4. 找到下载的 APK 文件并点击进行安装。

#### 运行应用

1. 安装完成后，打开 EhViewer 应用。
2. 输入 E-Hentai 的用户名和密码以登录您的账户（如果您有的话），或者直接开始浏览公共内容。
3. 探索应用的各种功能，如搜索漫画、下载画廊或管理已收藏的内容。

### 3. 应用案例和最佳实践

* **离线阅读**
  ：通过提前下载画廊，可以在无网络连接时享受阅读体验。
* **收藏夹管理**
  ：定期整理收藏的画廊，保持有序性。
* **设置偏好**
  ：根据自己的喜好调整图像质量、加载速度和夜间模式等设置。
* **隐私保护**
  ：使用私人模式浏览以避免保存浏览记录。

### 4. 典型生态项目

* **Kotlin & KotlinX**
  : EhViewer 使用 Kotlin 作为主要编程语言，并利用 KotlinX 库提高开发效率。
* **Arrow**
  : 提供实用的函数式编程工具集，用于简化代码结构。
* **Material Icons**
  : 应用内使用的图标来源于 Google 的 Material Icons 设计。
* **Ktor**
  : 用来构建服务器端通信部分，处理网络请求。
* **Coil**
  : 图片加载库，负责高效地显示网络上的图片。
* **Compose Destinations**
  : 用于导航管理的 Jetpack Compose 插件。
