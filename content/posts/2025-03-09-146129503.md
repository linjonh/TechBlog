    <h3>
     <a id="Ubuntu__Fcitx5__1">
     </a>
     Ubuntu 安装 Fcitx5 并添加字典教程
    </h3>
    <h4>
     <a id="_3">
     </a>
     前言
    </h4>
    <p>
     书接上回，一时兴起将主力机的 Ubuntu 20.04 LTS 升级至了刚刚发布的 24.04 LTS。从 X 切换到 Wayland、GNOME 从 3.36 升级至 42、Python 默认为 3.10 等等……使用太新的软件包反而暂时带来了麻烦，部分原有的软件和插件都不可用了。尝试安装了搜狗，出现闪烁bug.。故需要寻找新的中文拼音输入法。经简单浏览对比，选择了 Fcitx 5。
    </p>
    <hr/>
    <h4>
     <a id="_Fcitx5_8">
     </a>
     安装 Fcitx5
    </h4>
    <h5>
     <a id="1__10">
     </a>
     1. 检查系统中文环境
    </h5>
    <p>
     在 Ubuntu 设置中打开
     <strong>
      「区域与语言」
     </strong>
     →
     <strong>
      「管理已安装的语言」
     </strong>
     ，然后会自动检查已安装语言是否完整。若不完整，根据提示安装即可。
    </p>
    <h5>
     <a id="2__Fcitx5__13">
     </a>
     2. 安装 Fcitx5 及相关组件
    </h5>
    <p>
     Fcitx 5 由以下三部分组成：
    </p>
    <ul>
     <li>
      <strong>
       Fcitx 5 主程序
      </strong>
     </li>
     <li>
      <strong>
       中文输入法引擎
      </strong>
     </li>
     <li>
      <strong>
       图形界面相关组件
      </strong>
     </li>
    </ul>
    <p>
     使用以下命令安装：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> fcitx5 <span class="token punctuation">\</span>
fcitx5-chinese-addons <span class="token punctuation">\</span>
fcitx5-frontend-gtk3 fcitx5-frontend-gtk2 <span class="token punctuation">\</span>
fcitx5-frontend-qt5 kde-config-fcitx5
</code></pre>
    <hr/>
    <h4>
     <a id="_29">
     </a>
     配置输入法
    </h4>
    <p>
     安装完成后，需要手动设置输入法框架。
    </p>
    <h5>
     <a id="1__33">
     </a>
     1. 设置环境变量
    </h5>
    <p>
     编辑
     <code>
      ~/.profile
     </code>
     或
     <code>
      ~/.bashrc
     </code>
     ，添加以下内容：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">GTK_IM_MODULE</span><span class="token operator">=</span>fcitx5
<span class="token builtin class-name">export</span> <span class="token assign-left variable">QT_IM_MODULE</span><span class="token operator">=</span>fcitx5
<span class="token builtin class-name">export</span> <span class="token assign-left variable"><span class="token environment constant">XMODIFIERS</span></span><span class="token operator">=</span>@im<span class="token operator">=</span>fcitx5
</code></pre>
    <p>
     然后运行以下命令使其生效：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.profile
</code></pre>
    <h5>
     <a id="2__Fcitx5_45">
     </a>
     2. 启动 Fcitx5
    </h5>
    <p>
     在终端执行：
    </p>
    <pre><code class="prism language-bash">fcitx5 <span class="token parameter variable">-r</span>
</code></pre>
    <p>
     或者直接重启系统，使其自动启动。
    </p>
    <hr/>
    <h4>
     <a id="_54">
     </a>
     安装中文词库
    </h4>
    <h5>
     <a id="1__56">
     </a>
     1. 下载维基百科拼音词库
    </h5>
    <p>
     在 GitHub 打开
     <a href="https://github.com/felixonmars/fcitx5-pinyin-zhwiki/releases">
      维基百科中文拼音词库的 Releases
     </a>
     界面，下载最新版的
     <code>
      .dict
     </code>
     文件。
    </p>
    <h5>
     <a id="2__Fcitx5__59">
     </a>
     2. 查找 Fcitx5 词库目录
    </h5>
    <p>
     由于不同系统配置可能导致 Fcitx5 的词库目录不同，我们可以使用
     <code>
      find
     </code>
     命令查找：
    </p>
    <pre><code class="prism language-bash"><span class="token function">find</span> ~ <span class="token parameter variable">-type</span> d <span class="token parameter variable">-name</span> <span class="token string">"fcitx5"</span>
</code></pre>
    <p>
     例如，可能会返回如下路径：
    </p>
    <pre><code>/home/jio/.config/fcitx5#这是配置文件

/home/jio/.local/share/fcitx5#字典一般要放这里边

</code></pre>
    <p>
     通常词库存放在
     <code>
      ~/.local/share/fcitx5/pinyin/dictionaries/
     </code>
     ，但具体路径可能因系统不同而有所变化。
    </p>
    <h5>
     <a id="3__73">
     </a>
     3. 移动词库到正确的目录
    </h5>
    <p>
     根据
     <code>
      find
     </code>
     查找到的目录，调整以下命令中的路径，将词库放入正确的位置：
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 下载词库文件</span>
<span class="token function">wget</span> https://github.com/felixonmars/fcitx5-pinyin-zhwiki/releases/download/0.2.4/zhwiki-20220416.dict

<span class="token comment"># 创建存储目录（如果不存在）</span>
<span class="token function">mkdir</span> <span class="token parameter variable">-p</span> ~/.local/share/fcitx5/pinyin/dictionaries/

<span class="token comment"># 移动词库文件至该目录</span>
<span class="token function">mv</span> zhwiki-20220416.dict ~/.local/share/fcitx5/pinyin/dictionaries/
</code></pre>
    <p>
     如果
     <code>
      ~/.local/share/fcitx5/pinyin/dictionaries/
     </code>
     目录不存在，可以将
     <code>
      find
     </code>
     查找到的
     <code>
      fcitx5
     </code>
     目录下的
     <code>
      pinyin/dictionaries/
     </code>
     作为目标目录。
    </p>
