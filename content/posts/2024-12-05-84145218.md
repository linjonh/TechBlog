    <p>
     针对于某个数据库进行用户的添加（不使用root进行登陆），方法入下：
    </p>
    <ol>
     <li>
      创建用户和密码；
      <br/>
      username : 用于登陆数据库的用户名；
      <br/>
      password ：用于登陆时输入的密码
     </li>
    </ol>
    <pre><code>create user username identified by 'password; 
</code></pre>
    <ol start="2">
     <li>
      给新建的用户设置关于某个数据库的权限;
      <br/>
      databasename : 是数据库的名字；
      <br/>
      .*： 时针对这个数据库的所有表格拥有权限；
      <br/>
      ‘username’@’%’: 是这个用户通过任何ip访问数据库
     </li>
    </ol>
    <pre><code>grant all on databasename.* to 'username'@'%'; 
</code></pre>
    <ol start="3">
     <li>
      刷新上面的操作，使其生效
     </li>
    </ol>
    <pre><code>flush privileges;
</code></pre>
