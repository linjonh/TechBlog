     linux下查找文件及查找包含指定内容的文件常用命令。
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <div style="top:0px">
     ﻿﻿
    </div>
    <p>
     每一种操作系统都是由成千上万个不同种类的文件所组成的。其中有系统本身自带的文件，用户自己的文件，还有共享文件等等。我们有时候经常忘记某份文件放在硬盘中的哪个地方。
    </p>
    <p>
     在微软的Windows操作系统中要查找一份文件是相当简单的事情，只要在桌面上点击“开始”－“搜索”中就能按照各种方式在本地硬盘上，局域网络，甚至在INTERNET上查找各种文件，文档。
    </p>
    <p>
     可是使用Linux的用户就没有那么幸运了，在Linux上查找某个文件确实是一件比较麻烦的事情。毕竟在Linux中需要我们使用专用的“查找”命令来寻找在硬盘上的文件。废话不多说，下面小董就详细为大家介绍一下
     <strong>
      <a href="http://303i.com/article/2013/01/31/763.html" rel="nofollow noopener noreferrer" target="_blank">
       linux下查找文件及查找包含指定内容的文件常用命令
      </a>
     </strong>
     。
    </p>
    <p>
     whereis &lt;程序名称&gt;
     <br/>
     查找软件的安装路径
     <br/>
     -b 只查找二进制文件
     <br/>
     -m 只查找帮助文件
     <br/>
     -s 只查找源代码
     <br/>
     -u 排除指定类型文件
     <br/>
     -f 只显示文件名
     <br/>
     -B &lt;目录&gt; 在指定目录下查找二进制文件
     <br/>
     -M &lt;目录&gt; 在指定目录下查找帮助文件
     <br/>
     -S &lt;目录&gt; 在指定目录下查找源代码
    </p>
    <p>
     locate &lt;文件名称&gt;
     <br/>
     在文件索引数据库中搜索文件
     <br/>
     -d &lt;数据库路径&gt; 搜索指定数据库
     <br/>
     updatedb
     <br/>
     更新文件索引数据库
    </p>
    <p>
     find [路径] &lt;表达式&gt;
     <br/>
     查找文件
     <br/>
     -name &lt;表达式&gt; 根据文件名查找文件
     <br/>
     -iname &lt;表达式&gt; 根据文件名查找文件，忽略大小写
     <br/>
     -path &lt;表达式&gt; 根据路径查找文件
     <br/>
     -ipath &lt;表达式&gt; 根据路径查找文件，忽略大小写
     <br/>
     -amin &lt;分钟&gt; 过去N分钟内访问过的文件
     <br/>
     -atime &lt;天数&gt; 过去N天内访问过的文件
     <br/>
     -cmin &lt;分钟&gt; 过去N分钟内修改过的文件
     <br/>
     -ctime &lt;天数&gt; 过去N天内修改过的文件
     <br/>
     -anewer &lt;参照文件&gt; 比参照文件更晚被读取过的文件
     <br/>
     -cnewer &lt;参照文件&gt; 比参照文件更晚被修改过的文件
     <br/>
     -size &lt;大小&gt; 根据文件大小查找文件，单位b c w k M G
     <br/>
     -type &lt;文件类型&gt; 根据文件类型查找文件。b 块设备 c 字符设备 d 目录 p 管道文件 f 普通文件 l 链接 s 端口文件
     <br/>
     -user &lt;用户名&gt; 按归属用户查找文件
     <br/>
     -uid &lt;uid&gt; 按UID查找文件
     <br/>
     -group &lt;群组名&gt; 按归属群组查找文件
     <br/>
     -gid &lt;gid&gt; 按GID查找文件
     <br/>
     -empty 查找空文件
    </p>
    <p>
     从文件内容查找匹配指定字符串的行：
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">$ grep </span><span class="str">"被查找的字符串"</span><span class="pln"> </span><span class="pun">文件名</span></pre>
    <p>
     从文件内容查找与正则表达式匹配的行：
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">$ grep </span><span class="pun">–</span><span class="pln">e </span><span class="pun">“正则表达式”</span><span class="pln"> </span><span class="pun">文件名</span></pre>
    <p>
     查找时不区分大小写：
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">$ grep </span><span class="pun">–</span><span class="pln">i </span><span class="str">"被查找的字符串"</span><span class="pln"> </span><span class="pun">文件名</span></pre>
    <p>
     查找匹配的行数：
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">$ grep </span><span class="pun">-</span><span class="pln">c </span><span class="str">"被查找的字符串"</span><span class="pln"> </span><span class="pun">文件名</span></pre>
    <p>
     从文件内容查找不匹配指定字符串的行：
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">$ grep </span><span class="pun">–</span><span class="pln">v </span><span class="str">"被查找的字符串"</span><span class="pln"> </span><span class="pun">文件名</span></pre>
    <p>
     从根目录开始查找所有扩展名为.log的文本文件，并找出包含”ERROR”的行
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">find </span><span class="pun">/</span><span class="pln"> </span><span class="pun">-</span><span class="pln">type f </span><span class="pun">-</span><span class="pln">name </span><span class="str">"*.log"</span><span class="pln"> </span><span class="pun">|</span><span class="pln"> xargs grep </span><span class="str">"ERROR"</span></pre>
    <p>
     系统查找到nginx.conf文件后即时在屏幕上显示nginx.conf文件信息。
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">find</span><span class="pun">/-</span><span class="pln">name</span><span class="str">"nginx.conf"</span><span class="pun">-</span><span class="pln">ls</span></pre>
    <p>
     在根目录下查找某个文件
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">find </span><span class="pun">.</span><span class="pln"> </span><span class="pun">-</span><span class="pln">name </span><span class="str">"test"</span></pre>
    <p>
     在某个目录下查找包含某个字符串的文件
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">grep </span><span class="pun">-</span><span class="pln">r </span><span class="str">"zh_CN"</span><span class="pln"> </span><span class="pun">./</span></pre>
    <p>
     循环目录下所有文件内容中包含303i的文件
    </p>
    <pre class="wp-code-highlight prettyprint prettyprinted"><span class="pln">find </span><span class="pun">.</span><span class="pln"> </span><span class="pun">-</span><span class="pln">type f </span><span class="pun">-</span><span class="kwd">exec</span><span class="pln"> grep </span><span class="pun">-</span><span class="pln">l </span><span class="str">"303i"</span><span class="pln"> </span><span class="pun">{}</span><span class="pln"> \;</span></pre>
