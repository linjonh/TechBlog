    <h2>
     <span style="color:#0d0016">
      一、链表的概念与结构
     </span>
    </h2>
    <h3>
     <span style="color:#0d0016">
      1.1 概念：
     </span>
    </h3>
    <p>
     <span style="color:#0d0016">
      通俗的来说，链表是由一个个
      <strong>
       结点连接
      </strong>
      起来的就叫
      <strong>
       链表
      </strong>
      。
     </span>
    </p>
    <h3>
     <span style="color:#0d0016">
      1.2 结构：
     </span>
    </h3>
    <p>
     <span style="color:#0d0016">
      链表存储的数据 在
      <strong>
       物理上是不一定连续
      </strong>
      的，它是由
      <strong>
       前面链接后面
      </strong>
      ，一个个连起来的。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <img alt="" height="289" src="https://i-blog.csdnimg.cn/direct/cff101d72be547d28aa1eaa64cafa680.png" width="1216"/>
     </span>
    </p>
    <h2>
     <span style="color:#0d0016">
      二、Java底层的 LinkedList
     </span>
    </h2>
    <h3>
     <span style="color:#0d0016">
      2.1 Java底层构成：
     </span>
    </h3>
    <p>
     <span style="color:#0d0016">
      LinkedList 的底层是
      <strong>
       双向链表
      </strong>
      结构， 由于链表没有将元素存储在连续的空间中，元素存储在单独的j结点中，然后通过引用将 结点 连接起来了。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      因此在在任意位置
      <strong>
       插入
      </strong>
      或者
      <strong>
       删除
      </strong>
      元素时，不需要搬移元素，
      <strong>
       效率比较高
      </strong>
      。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      <img alt="" height="529" src="https://i-blog.csdnimg.cn/direct/c84cd8ff1c9c4f568c0377888a144aa9.png" width="1115"/>
     </span>
    </p>
    <p>
     <strong>
      <span style="color:#0d0016">
       LinkedList 中包含 LinkNode 类
      </span>
     </strong>
    </p>
    <p>
     <span style="color:#0d0016">
      从上图我们可以看到 每个结点都有三部分构成，
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      分别是
      <strong>
       数据  (elem)
      </strong>
      ，结点的
      <strong>
       前驱 (prev)
      </strong>
      ，结点的
      <strong>
       后继(next)
      </strong>
     </span>
    </p>
    <h3>
     <span style="color:#0d0016">
      2.2 链表的构造方法：

     <span style="color:#0d0016">

    <h3>
     <span style="color:#0d0016">
      3.2 链表的遍历
     </span>
    </h3>
    <pre><code class="language-java">public static void main(String[] args) {
        LinkedList&lt;Integer&gt; linkedList2 = new LinkedList&lt;&gt;();
        linkedList2.add(1);//默认都是尾插
        linkedList2.addLast(3);
        linkedList2.addLast(4);
        linkedList2.addLast(5);
        System.out.println(linkedList2);
        
        //迭代器反向遍历
        System.out.println("===== ListIterator ====");
        ListIterator&lt;Integer&gt; lit2 =  linkedList2.listIterator(linkedList2.size());
        while (lit2.hasPrevious()) {
            System.out.print(lit2.previous()+" ");
        }
        System.out.println();

        //迭代器正向遍历，属于子类
        System.out.println("===== ListIterator ====");
        ListIterator&lt;Integer&gt; lit =  linkedList2.listIterator();
        while (lit.hasNext()) {
            System.out.print(lit.next()+" ");
        }
        System.out.println();
        
        //迭代器正向遍历
        System.out.println("===== Iterator ====");
        Iterator&lt;Integer&gt; it = linkedList2.iterator();
        while (it.hasNext()) {
            System.out.print(it.next()+" ");
        }
        System.out.println();

        //for each遍历
        System.out.println("=====for each====");

        for(Integer x : linkedList2) {
            System.out.print(x +" ");
        }
        System.out.println();
        
        //for循环遍历
        System.out.println("=====for====");
        for (int i = 0; i &lt; linkedList2.size(); i++) {
            System.out.print(linkedList2.get(i) +" ");
        }
        System.out.println();


    }</code></pre>
