    <h2>
     <strong>
      一、时间复杂度
     </strong>
    </h2>
    <p>
     <br/>
     时间复杂度（Time Complexity）表示算法运行时间随输入规模增长的变化趋势。通常用大 O 表示法（Big O Notation）来描述。
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <caption>
      常见时间复杂度

        名称

       <td>
        O(1)
       </td>
       <td>
        常数时间复杂度
       </td>
       <td>
        访问数组中的某个元素。
       </td>
      </tr>
      <tr>
       <td>
        O(log n)
       </td>
       <td>
        对数时间复杂度
       </td>
       <td>
        <span style="background-color:#ff9900">
         二分查找。
        </span>
       </td>
      </tr>
      <tr>
       <td>
        O(n)

    <p>
     例子
    </p>
    <p>
     1. O(1)：常数时间复杂度
    </p>
    <pre><code class="language-java">int getFirstElement(int[] arr) {
    return arr[0]; // 无论数组多大，只需一次操作
}</code></pre>
    <p>
     2. O(n)：线性时间复杂度
    </p>
    <pre><code class="language-java">void printArray(int[] arr) {
    for (int i = 0; i &lt; arr.length; i++) { // 遍历数组，操作次数与数组长度成正比
        System.out.println(arr[i]);
    }
}</code></pre>
    <p>
     3. O(n²)：平方时间复杂度
    </p>
    <pre><code class="language-java">void bubbleSort(int[] arr) {
    for (int i = 0; i &lt; arr.length; i++) {
        for (int j = 0; j &lt; arr.length - 1; j++) { // 双重循环，操作次数与数组长度的平方成正比
            if (arr[j] &gt; arr[j + 1]) {
                swap(arr, j, j + 1);
            }
        }
    }
}
</code></pre>
    <p>
     4. O(log n)：对数时间复杂度
    </p>
    <pre><code class="language-java">int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left &lt;= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        if (arr[mid] &lt; target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}</code></pre>

    <h2>
     二、空间复杂度
    </h2>
    <p>
     <br/>
     空间复杂度（Space Complexity）表示算法运行过程中所需的额外存储空间随输入规模增长的变化趋势。通常也用大 O 表示法描述。
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <caption>
      常见空间复杂度
     </caption>
     <tbody>
      <tr>
       <td>
        复杂度
       </td>
       <td>
        名称
       </td>
       <td>
        例子
       </td>
      </tr>
      <tr>
       <td>
        O(1)
       </td>
       <td>
        常数空间复杂度
       </td>
       <td>
        只使用固定数量的变量。
       </td>
      </tr>
      <tr>
       <td>
        O(n)
       </td>
       <td>
        线性空间复杂度
       </td>
       <td>
        使用一个与输入规模成正比的数组或列表。
       </td>
      </tr>
      <tr>
       <td>
        O(n²)
       </td>
       <td>
        平方空间复杂度
       </td>
       <td>
        使用一个二维数组（如邻接矩阵）。
       </td>
      </tr>
     </tbody>
    </table>
    <p>
    </p>
    <p>
     例子
    </p>
    <p>
     1. O(1)：常数空间复杂度
    </p>
    <pre><code class="language-java">int sum(int a, int b) {
    return a + b; // 只使用了固定数量的变量（a 和 b）
}</code></pre>
    <p>
     <br/>
     2. O(n)：线性空间复杂度
    </p>
    <pre><code class="language-java">int[] copyArray(int[] arr) {
    int[] newArr = new int[arr.length]; // 创建了一个与输入数组大小相同的新数组
    for (int i = 0; i &lt; arr.length; i++) {
        newArr[i] = arr[i];
    }
    return newArr;
}

</code></pre>
    <p>
     3. O(n²)：平方空间复杂度
    </p>
    <pre><code class="language-java">int[][] createMatrix(int n) {
    int[][] matrix = new int[n][n]; // 创建了一个 n x n 的二维数组
    for (int i = 0; i &lt; n; i++) {
        for (int j = 0; j &lt; n; j++) {
            matrix[i][j] = i + j;
        }
    }
    return matrix;
}</code></pre>
    <h2>
     三. 如何分析时间复杂度和空间复杂度
    </h2>
    <h3>
     时间复杂度分析
    </h3>
    <p>
     <br/>
     1. 找出基本操作：通常是循环、递归、条件判断等。
     <br/>
     2. 计算基本操作的执行次数：通常与
     <span style="color:#ff9900">
      <strong>
       输入规模
      </strong>
     </span>
     （如数组长度 `n`）有关。
     <br/>
     3. 用大 O 表示法表示：忽略常数项和低阶项。
    </p>
    <pre><code class="language-java">void printPairs(int[] arr) {
    for (int i = 0; i &lt; arr.length; i++) { // O(n)
        for (int j = 0; j &lt; arr.length; j++) { // O(n)
            System.out.println(arr[i] + ", " + arr[j]); // O(1)
        }
    }
}</code></pre>
    <p>
     总时间复杂度：`O(n)
     <span style="color:#ff9900">
      *
     </span>
     O(n)
     <span style="color:#ff9900">
      *
     </span>
     O(1) = O(n²)`。
    </p>
    <h3>
     空间复杂度分析
    </h3>
    <p>
     <br/>
     1. 找出额外存储空间：通常是变量、数组、递归栈等。
     <br/>
     2. 计算存储空间的大小：通常与
     <span style="color:#ff9900">
      输入规模（如数组长度 `n`）
     </span>
     有关。
     <br/>
     3. 用大 O 表示法表示：忽略常数项和低阶项。
    </p>
    <p>
     例子
    </p>
    <pre><code class="language-java">int[] reverseArray(int[] arr) {
    int[] result = new int[arr.length]; // O(n)
    for (int i = 0; i &lt; arr.length; i++) {
        result[i] = arr[arr.length - 1 - i];
    }
    return result;
}</code></pre>
    <p>
     <br/>
     总空间复杂度：`O(n)`（用于存储 `result` 数组）。
    </p>
    <p>
    </p>
    <p>
     4. 总结
    </p>
    <p>
     时间复杂度：衡量
     <span style="color:#956fe7">
      算法运行时间
     </span>
     随输入规模增长的变化趋势。
     <br/>
     空间复杂度：衡量
     <span style="color:#956fe7">
      算法运行过程中所需的额外存储空间
     </span>
     随输入规模增长的变化趋势。
     <br/>
     大 O 表示法：忽略常数项和低阶项，关注增长趋势。
    </p>
