    <p>
     在人工智能迅猛发展的时代，如何高效、安全地利用本地 AI 进行日常工作和研究？如果你是 macOS 用户，那么
     <strong>
      Sidekick
     </strong>
     可能正是你需要的那款 AI 助手！这是一款专为 macOS 设计的原生应用，让你无需额外安装软件，即可与本地 LLM（大语言模型）对话，并且能够直接利用 Mac 上的文件、文件夹和网页信息进行智能分析。更重要的是，
     <strong>
      所有对话均在本地运行，数据完全保密
     </strong>
     。
    </p>
    <h3>
     <a id="Sidekick__4">
     </a>
     Sidekick 的核心亮点
    </h3>
    <h4>
     <a id="_6">
     </a>
     零配置，开箱即用
    </h4>
    <p>
     Sidekick 采用
     <strong>
      本地优先
     </strong>
     的设计理念，安装后即可使用。无需繁琐的配置，就能立即享受 AI 智能交互体验。无论是对 LLM 了解甚少的新手，还是 AI 技术爱好者，都能轻松上手。
    </p>
    <h4>
     <a id="_10">
     </a>
     智能检索本地资源
    </h4>
    <p>
     Sidekick 具备
     <strong>
      强大的本地信息整合能力
     </strong>
     ，可以直接访问你的
     <strong>
      文件、文件夹和网页
     </strong>
     ，并在回答问题时引用相关内容。例如：
    </p>
    <blockquote>
     <p>
      你正在撰写一篇关于阿兹特克人历史的论文，并想知道他们是否使用了缴获的西班牙武器。只需向 Sidekick 提问，它就能在你的资料库中检索相关文献，并提供页码和简要分析。
     </p>
    </blockquote>
    <p>
     此外，Sidekick 还能让你
     <strong>
      点击引用的文献，立即在 Mac 上打开原文档
     </strong>
     ，大大提高查阅和验证效率。
    </p>
    <h4>
     <a id="RAG__18">
     </a>
     RAG 技术，智能增强搜索
    </h4>
    <p>
     Sidekick 采用
     <strong>
      RAG（检索增强生成）
     </strong>
     技术，使其能够处理
     <strong>
      海量的本地文献和数据源
     </strong>
     。你可以创建不同的“专家”知识库，如
     <strong>
      English Literature、Mathematics、Computer Science、Physics
     </strong>
     等，并在需要时激活特定专家，让 Sidekick 只检索相关领域的信息，提高答案的精准度。
    </p>
    <p>
     此外，你还能通过
     <strong>
      拖拽文件
     </strong>
     让 Sidekick 访问新资料，让 AI 快速学习你手头的内容。
    </p>
    <h4>
     <a id="_24">
     </a>
     在线搜索，获取最新资讯
    </h4>
    <p>
     虽然 Sidekick 主要侧重本地数据处理，但它也提供
     <strong>
      网页搜索功能
     </strong>
     ，让你在需要时获取最新的外部信息，补充本地知识库。
    </p>
    <h3>
     <a id="_AI__28">
     </a>
     强大功能，让 AI 更懂你！
    </h3>
    <h4>
     <a id="_API__AI__30">
     </a>
     自带 API 密钥，支持多种 AI 模型
    </h4>
    <p>
     除了本地推理功能，Sidekick 还支持
     <strong>
      OpenAI 兼容 API
     </strong>
     ，你可以输入自己的 API 密钥，切换到其他 AI 模型，如：
    </p>
    <ul>
     <li>
      <strong>
       阿里云 QwQ-32B
      </strong>
     </li>
     <li>
      <strong>
       DeepSeek-R1
      </strong>
     </li>
     <li>
      以及其他兼容 OpenAI API 的大模型
     </li>
    </ul>
    <p>
     这样，你既能享受本地 AI 的安全性，又可以灵活调用远程 AI 的强大计算能力。
    </p>
    <h4>
     <a id="___40">
     </a>
     代码解释器，提升数学 &amp; 逻辑能力
    </h4>
    <p>
     小模型通常在代码编写上比数学计算更强。Sidekick 采用
     <strong>
      代码解释器
     </strong>
     ，让 AI 生成代码并运行，确保回答数学和逻辑问题时更可靠。
    </p>
    <h4>
     <a id="_44">
     </a>
     内置图像生成
    </h4>
    <p>
     Sidekick 还支持
     <strong>
      文本生成图像
     </strong>
     ，无需额外操作，AI 会自动识别适合图像生成的提示，并直接输出视觉素材。
    </p>
    <p>
     <strong>
      注意
     </strong>
     ：图像生成功能适用于 macOS 15.2 及更高版本，并需要 Apple Intelligence 支持。
    </p>
    <h4>
     <a id="_50">
     </a>
     内联写作助手
    </h4>
    <p>
     如果你在 Word 或其他文档编辑软件中工作，按下
     <code>
      Command + Control + I
     </code>
     组合键，即可调出 Sidekick 的
     <strong>
      写作助手
     </strong>
     ，帮你快速回答问题、整理内容，大幅提升写作效率。
    </p>
    <h4>
     <a id="LaTeX___Markdown__54">
     </a>
     LaTeX 数学公式 &amp; Markdown 渲染
    </h4>
    <p>
     如果你是科研人员或工程师，Sidekick 的
     <strong>
      LaTeX 支持
     </strong>
     能让你流畅输入和展示数学公式。同时，Sidekick 还能以
     <strong>
      美观的 Markdown 格式
     </strong>
     渲染文本，让笔记和文章更加整洁清晰。
    </p>
    <h4>
     <a id="___58">
     </a>
     数据可视化 &amp; 代码高亮
    </h4>
    <p>
     Sidekick 能自动
     <strong>
      生成表格和多种图表
     </strong>
     （如柱状图、折线图、饼图等），并允许你
     <strong>
      直接拖拽到第三方应用
     </strong>
     。同时，它还支持
     <strong>
      代码语法高亮
     </strong>
     ，方便开发者阅读和复制代码。
    </p>
    <h4>
     <a id="Sidekick__62">
     </a>
     Sidekick 工具箱：全面提升你的工作流
    </h4>
    <p>
     Sidekick 还配备了一系列实用工具，包括：
    </p>
    <ul>
     <li>
      <strong>
       AI 文本检测器
      </strong>
      ：判断文本 AI 生成的可能性，并提供改写建议。
     </li>
     <li>
      <strong>
       关系图生成器
      </strong>
      ：只需输入提示，即可自动绘制
      <strong>
       复杂的概念图
      </strong>
      。
     </li>
     <li>
      <strong>
       AI 幻灯片助手
      </strong>
      ：只需写下演示大纲，Sidekick 就能自动生成 PPT，5 分钟搞定 10 分钟的演示内容。支持导出 PDF 和 PowerPoint 格式。
     </li>
    </ul>
    <h3>
     <a id="_Sidekick_70">
     </a>
     为什么选择 Sidekick？
    </h3>
    <ol>
     <li>
      <strong>
       零配置，即装即用
      </strong>
      ：对小白友好，无需复杂设置。
     </li>
     <li>
      <strong>
       本地运行，数据私密
      </strong>
      ：所有信息仅存于你的 Mac，安全有保障。
     </li>
     <li>
      <strong>
       智能检索文件和网页
      </strong>
      ：让 AI 直接引用你的资料，提高信息可靠性。
     </li>
     <li>
      <strong>
       支持多种 LLM 模型
      </strong>
      ：本地推理 + 远程 API，灵活切换最佳方案。
     </li>
     <li>
      <strong>
       超快响应，优化 Mac 运行
      </strong>
      ：Sidekick 采用优化后的
      <strong>
       llama.cpp 引擎
      </strong>
      ，在 Apple Silicon 上实现超快 AI 推理，同时支持
      <strong>
       推测解码
      </strong>
      ，进一步加速处理速度。
     </li>
    </ol>
    <h3>
     <a id="_Sidekick_78">
     </a>
     如何安装 Sidekick？
    </h3>
    <p>
     <strong>
      系统要求
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       Apple Silicon 芯片（M1/M2/M3）
      </strong>
     </li>
     <li>
      <strong>
       至少 8GB 内存
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      安装步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <strong>
       前往 [GitHub Releases]https://github.com/johnbean393/Sidekick/releases 下载最新版本
      </strong>
     </li>
     <li>
      打开
      <code>
       .dmg
      </code>
      安装包，按照提示安装
     </li>
     <li>
      运行 Sidekick，开始你的 AI 之旅！
     </li>
    </ol>

     <strong>
      更多用户能无门槛使用开放、私密、本地 AI
     </strong>
     ，同时提供可选的在线模型支持。它的核心原则包括：
    </p>
    <ul>
     <li>
      <strong>
       无需配置，人人可用
      </strong>
     </li>

      </strong>

      </strong>
     </li>
     <li>
      <strong>
       开源透明，值得信赖
      </strong>
     </li>
    </ul>
    <p>
     在 AI 时代，Sidekick 让你的 Mac
     <strong>
      更智能、更高效、更安全
     </strong>
     ！快来试试吧！
