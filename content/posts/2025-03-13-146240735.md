    <p>
     <code>
      std::list
     </code>
     是 C++ 标准库中的一个双向链表容器，定义在
     <code>
      &lt;list&gt;
     </code>
     头文件中。它支持在任意位置高效地插入和删除元素，但不支持随机访问。以下是
     <code>
      std::list
     </code>
     的基本用法和一些常见操作：
    </p>
    <h4>
     1. 包含头文件
    </h4>
    <pre><code class="language-cpp">#include &lt;list&gt;</code></pre>
    <h4>
     2. 定义和初始化
    </h4>
    <pre><code class="language-cpp">std::list&lt;int&gt; myList; // 定义一个空的 list
std::list&lt;int&gt; myList2 = {1, 2, 3, 4, 5}; // 使用初始化列表初始化</code></pre>
    <h4>
     3. 插入元素
    </h4>
    <pre><code class="language-cpp">myList.push_back(10); // 在末尾插入元素
myList.push_front(5); // 在开头插入元素

auto it = myList.begin();
std::advance(it, 2); // 移动迭代器到第3个位置
myList.insert(it, 15); // 在指定位置插入元素</code></pre>
    <h4>
     4. 删除元素
    </h4>
    <pre><code class="language-cpp">myList.pop_back(); // 删除末尾元素
myList.pop_front(); // 删除开头元素

auto it = myList.begin();
std::advance(it, 2); // 移动迭代器到第3个位置
myList.erase(it); // 删除指定位置的元素

myList.remove(10); // 删除所有值为10的元素</code></pre>
    <h4>
     5. 访问元素
    </h4>
    <pre><code class="language-cpp">int firstElement = myList.front(); // 访问第一个元素
int lastElement = myList.back(); // 访问最后一个元素

// 遍历 list
for (auto it = myList.begin(); it != myList.end(); ++it) {
    std::cout &lt;&lt; *it &lt;&lt; " ";
}</code></pre>
    <h4>
     6. 大小和容量
    </h4>
    <pre><code class="language-cpp">bool isEmpty = myList.empty(); // 检查是否为空
size_t size = myList.size(); // 获取元素个数</code></pre>
    <h4>
     7. 排序和反转
    </h4>
    <pre><code class="language-cpp">myList.sort(); // 对 list 进行排序
myList.reverse(); // 反转 list</code></pre>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;list&gt;

int main() {
    std::list&lt;int&gt; myList = {5, 3, 1, 4, 2};

    // 默认升序排序
    myList.sort();
    std::cout &lt;&lt; "升序排序: ";
    for (int val : myList) {
        std::cout &lt;&lt; val &lt;&lt; " ";
    }
    std::cout &lt;&lt; std::endl;

    // 自定义降序排序
    myList.sort([](int a, int b) {
        return a &gt; b;
    });
    std::cout &lt;&lt; "降序排序: ";
    for (int val : myList) {
        std::cout &lt;&lt; val &lt;&lt; " ";
    }
    std::cout &lt;&lt; std::endl;

    return 0;
}</code></pre>
    <h4>
     8. 合并和拼接
    </h4>
    <pre><code class="language-cpp">std::list&lt;int&gt; anotherList = {20, 30, 40};
myList.merge(anotherList); // 合并两个 list，合并后 anotherList 为空

myList.splice(myList.begin(), anotherList); // 将 anotherList 的元素拼接到 myList 的开头</code></pre>
    <h4>
     9. 其他操作
    </h4>
    <pre><code class="language-cpp">myList.unique(); // 删除连续重复的元素
myList.resize(10); // 调整 list 的大小</code></pre>
    <h4>
     10. 示例代码
    </h4>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;list&gt;

int main() {
    std::list&lt;int&gt; myList = {1, 2, 3, 4, 5};

    myList.push_back(6);
    myList.push_front(0);

    for (auto it = myList.begin(); it != myList.end(); ++it) {
        std::cout &lt;&lt; *it &lt;&lt; " ";
    }
    std::cout &lt;&lt; std::endl;

    myList.remove(3);
    myList.sort();
    myList.reverse();

    for (auto it = myList.begin(); it != myList.end(); ++it) {
        std::cout &lt;&lt; *it &lt;&lt; " ";
    }
    std::cout &lt;&lt; std::endl;

    return 0;
}</code></pre>
    <h4>
     总结
    </h4>
    <p>
     <code>
      std::list
     </code>
     是一个灵活的双向链表容器，适合频繁插入和删除操作的场景。由于它不支持随机访问，因此在需要随机访问时，
     <code>
      std::vector
     </code>
     或
     <code>
      std::deque
     </code>
     可能是更好的选择。
    </p>
