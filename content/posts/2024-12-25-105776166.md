# 有哪些常用的搜索引擎指令？
搜索引擎高级指令作为搜索引擎的扩展使用方法，有指定搜索，筛选搜索，SEO分析等作用，常用的指令如下：
![有哪些常用的搜索引擎指令？](https://i-blog.csdnimg.cn/blog\_migrate/e9f4d0fdcf79abe27dbc761a39e8eaea.jpeg)
\*\*site指令\*\*
用来搜索某个域名在搜索引擎收录的所有页面和文件。Google、百度、搜狗、360搜索、bing、雅虎等各大搜索引擎都支持该指令。
实际作用：查询某个特定网站的收录情况，如果结果有返回数据，则表明该网站已被收录，如果返回数据为空，则该网站未被收录。
指令格式举例：
```bash
site:www.ezyw.com
```
该指令反应出知云网下所有搜索引擎收录的页面
其它用法：
1. site命令可以限制在某一网站内搜索，site语法把搜索范围局限在这些网站内，以提高搜索效率。
例如：需要搜索“搜索引擎”为关键词，在知云网www.ezyw.com中的内容。
那么可以用以下格式输入：搜索引擎 site:www.ezyw.com
2. site指令也可以单独用于子域名搜索
例如：
```bash
site:jz.ezyw.com
```
搜索的就是该子域名下的所有收录页面。
\*\*inurl指令\*\*
用于搜索查询呈现在url 中的页面，baidu和google均支持该指令。
指令格式举例：在百度搜索输入inurl：知云网（这时在返回的结果是所有url中包含“知云网”的页面，支持中英文）
实际作用：由于关键词出现在url中对排名有一定影响，使用inurl:搜索可以更准确地找到竞争对手。在实际网址一般由英文url组成，中文网站的网址使用中文url不多，对优化的意义也有限。
使用技巧：inurl命令可以准确查询个人博客的收录情况，比如：我在百度申请了一个用户名为ezyw\\_com的博客，那么我们准确的查询博客收录呢？正确的查询方法是
```bash
site:hi.baidu.com inurl:ezyw\_com
```
这条命令的意思就是hi.baidu.com（百度空间域名） 包含ezyw\\_com（用户名）的页面。
\*\*allinurl指令\*\*
与inurl相似，区别在于allinurl指令可以同时查询一组多个关键词
指令格式举例：
```bash
allinurl:seo 知云网
```
这时在返回的结果是所有url中同时包含“seo”和“知云网”的页面，支持中英文
\*\*intitle指令\*\*
该指令返回的结果是页面title包含关键词，baidu和google均支持该指令。
指令格式举例：
```bash
intitle:知云网
```
搜索结果返回所有title包含知云网的页面
实际作用：intitle指令可以查询到标题中出现特定的关键词的页面。把关键词放进Title中是SEO重要的组成部分，因此使用intitle指令找到的文字是更准确的竞争对手页面。而没有出现在title中的大部分是并没有针对关键词进行优化，也不是有力的竞争对手
\*\*allintitle指令\*\*
该标签返回的结果是页面标题中包含多组关键词的文件。
指令格式举例：allintitle:SEO 知云网（搜索结果返回title中既包含”SEO”也包含”知云网”的页面）
\*\*filetype指令\*\*
搜索特定后缀格式的文档，如pdf、doc、xls等。
指令格式举例：
```bash
filetype:doc 知云网
```
搜索结果返回包含“知云网”的doc文档
实际作用：相当于在百度文库中寻找指定格式的文档文件。
\*\*（“”）双引号\*\*
把搜索词放在双引号中，代表完整匹配搜索，也就是说搜索结果返回的页面包含双引号中出现的所有词，必需完整匹配，baidu和Google 都支持这个指令。
指令格式举例：
```bash
“seo知云网”
```
搜索结果返回完整匹配关键词为“seo知云网”的搜索结果。
实际作用：可以实现精确化搜索结果
\*\*（ ）空格\*\*
在百度搜索查询时，将多个关键词之间输入空格，相当于百度会在多个以空格隔开的词语之间自动添加”+”，提供符合您全部查询条件的资料，并把最相关的网页排在前列。
指令格式举例：
```bash
“seo 知云网”
```
搜索结果返回关键词为既包含“seo”，也包含“知云网”的搜索结果。
实际作用：可以实现多个关键词筛联合相关的搜索结果
\*\*（-）减号\*\*
减号代表搜索不包含减号后面的词的页面。使用这个指令时减号前面必需是空格，减号后面没有空格，紧跟着需求排除的词，baidu和Google 均支持该指令。
指令格式举例：搜索 -引擎 返回的结果则是包含“搜索”这个词，却不包含“引擎”这个词的结果。
实际作用：可以实现筛选后的搜索结果
\*\*domain指令用 domain:\*\*
```bash
domain:www.ezyw.com
```
这条指令可以查询到知云网的百度相关域，即百度外链，该指令仅适用于百度搜索引擎，在 Google中该指令相当于一个关键词。
除此之外，还有许多高级搜索指令只支持google搜索，一些是百度搜索引擎不支持的高级指令，例如：
\*\*related指令\*\*
查询相关网页指令：可以用它来查询和你的网站内容相关的网站，利用 related指令可以很轻松的找到和你最相关的对手。
\*\*info指令\*\*
info指令：能查询某个特定网站的收录信息、最近的快照情况、相似网页站点链接、内部链接及包含域名的网页。
\*\*allintext/intext指令\*\*
有效的查询某个特定关键词在特定网页文档上出现情况。发现最相关的网页和潜在链接对象。该指令只支持 Google搜索引擎。
\*\*allinanchor/unanchor指令\*\*
可以理解为査找特定关键词的锚文本链接。有点像查找外链情况，所不同的是输入的是文本。该指令只支持 Google搜索引擎。
\*\*define指令\*\*
查找特定的关键词非常的有效。Ggle比百度查询结果相对精准，但百度的范围较广，实践中可根据自己的情况取舍。
\*\*Link指令\*\*
link指令用来搜索某个url的反向链接，既包括内部链接，也包括外部链接。但是现在Google对这个指令只返回其索引库中的一部分，而且是近乎随机的一部分，所以用这个指令查反链几乎没有用。
百度、搜狗、360搜索等不支持该指令。
指令格式举例：
```bash
link:www.ezyw.com
```
搜索www.ezyw.com网站下所有的外部链接页面
实际作用：由于这个指令可以查询某个特定网站的外部链接，这是一个主要获取排名和权重以及网站PR值的主要因素，所以过去是SEO优化的重要分析指标。
