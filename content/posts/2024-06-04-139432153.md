    <h2>
     <a id="_18">
     </a>
     一、前言
    </h2>
    <p>
     通常在嵌入式开发中，很多开发者习惯于使用Windows进行代码的编辑，比如使用Windows的Visual Studio Code进行OpenHarmony代码的开发。但当前阶段，仅Hi3861系列开发板支持在Windows环境下进行编译，大部分的开发板源码还不支持在Windows环境下进行编译，如Hi3516系列开发板。因此，需要使用Ubuntu的编译环境对源码进行编译。
    </p>
    <p>
     在以上的设备开发场景中，可以搭建一套Windows+Ubuntu混合开发的环境，其中使用Windows平台的DevEco Device Tool可视化界面进行相关操作，通过远程连接的方式对接Ubuntu下的DevEco Device Tool（Ubuntu下可以不安装Visual Studio Code），然后对Ubuntu下的源码进行开发、编译、烧录等操作。
    </p>
    <h2>
     <a id="_23">
     </a>
     二、运行环境要求
    </h2>
    <p>
     为保证DevEco Studio正常运行，建议电脑配置满足如下要求：
    </p>
    <pre><code>操作系统：Windows10 64位
内存：8GB及以上
硬盘：100GB及以上
分辨率：1280*800像素及以上
</code></pre>
    <h2>
     <a id="DevEco_Studio_31">
     </a>
     三、下载和安装DevEco Studio
    </h2>
    <p>
     进入
     <a href="https://developer.harmonyos.com/cn/develop/deveco-studio#download" rel="nofollow">
      HUAWEI DevEco Studio产品页
     </a>
     ，单击下载列表右侧的按钮，下载DevEco Studio。
     <br/>
     下载完成后，双击下载的“deveco-studio-xxxx.exe”，进入DevEco Studio安装向导。在如下界面选择安装路径，默认安装于C:\Program Files路径下，也可以单击Browse…指定其他安装路径，然后单击
     <strong>
      Next
     </strong>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/508f4b3330fa418db91f2610329ddb53.png#pic_center">
      <br/>
      在如下安装选项界面勾选
      <strong>
       DevEco Studio
      </strong>
      后，单击
      <strong>
       Next
      </strong>
      ，直至安装完成。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/55a2b5e7351c40d5b86b441d533085c5.png#pic_center">
       <br/>
       安装完成后，单击
       <strong>
        Finish
       </strong>
       完成安装。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1397aaf9813c4d4a9752d79bf9ea5c37.png#pic_center"/>

    <h3>
     <a id="42_HDC_98">
     </a>
     4.2 配置HDC工具环境变量
    </h3>
    <p>
     HDC是为开发者提供HarmonyOS应用/服务的调试工具，为方便使用HDC工具，请为HDC工具及其端口号设置环境变量。
    </p>
    <p>
     在
     <strong>
      此电脑 &gt; 属性 &gt; 高级系统设置 &gt; 高级 &gt; 环境变量
     </strong>
     中，添加HDC端口变量名为：HDC_SERVER_PORT，变量值可设置为任意未被占用的端口，如7035。
    </p>
    <p>

    <h2>
     <a id="_163">
     </a>
     一、前言
    </h2>
    <p>
     通常在嵌入式开发中，很多开发者习惯于使用Windows进行代码的编辑，比如使用Windows的Visual Studio Code进行OpenHarmony代码的开发。但当前阶段，仅Hi3861系列开发板支持在Windows环境下进行编译，大部分的开发板源码还不支持在Windows环境下进行编译，如Hi3516系列开发板。因此，需要使用Ubuntu的编译环境对源码进行编译。
    </p>
    <p>
     在以上的设备开发场景中，可以搭建一套Windows+Ubuntu混合开发的环境，其中使用Windows平台的DevEco Device Tool可视化界面进行相关操作，通过远程连接的方式对接Ubuntu下的DevEco Device Tool（Ubuntu下可以不安装Visual Studio Code），然后对Ubuntu下的源码进行开发、编译、烧录等操作。
    </p>
    <h2>
     <a id="_168">
     </a>
     二、运行环境要求
    </h2>
    <p>
     为保证DevEco Studio正常运行，建议电脑配置满足如下要求：
    </p>
    <pre><code>操作系统：Windows10 64位
内存：8GB及以上
硬盘：100GB及以上
分辨率：1280*800像素及以上
</code></pre>

      运行已安装的DevEco Studio，首次使用，请选择
      <strong>
       Do not import settings
      </strong>
      ，单击OK。
     </li>
     <li>
      进入DevEco Studio操作向导页面，修改
      <strong>
       npm registry
      </strong>
      ，DevEco Studio已预置对应的仓（默认的npm仓，可能出现部分开发者无法访问或访问速度缓慢的情况），直接单击
      <strong>
       Start using DevEco Studio
      </strong>
      进入下一步。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      如果配置向导界面出现的是设置Set up HTTP Proxy界面，说明网络受限，请根据
      <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V3/environment_config-0000001052902427-V3#section11105193853519" rel="nofollow">
       参考信息
      </a>
      配置DevEco Studio代理和NPM代理后，再下载SDK。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/86e2bf6f23b5463683aa68b7475f3452.png#pic_center"/>
     <br/>
     3. 设置Node.js信息，可以指定本地已安装的Node.js（Node.js版本要求为v14.19.1及以上，且低于v15.0.0；对应的npm版本要求为6.14.16及以上，且低于7.0.0版本）；如果本地没有合适的版本，可以选择
     <strong>
      Download
     </strong>
     按钮，在线下载Node.js。本示例以下载Node.js为例，选择下载源和存储路径后，单击
     <strong>
      Next
     </strong>
     进入下一步。
    </p>
    <blockquote>
     <p>
      说明
      <br/>
      若配置过程中检测到在环境变量中，或在C:\Users\userName目录下有符合要求的Node.js版本，则设置Node.js步骤将自动跳过。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f4092348e94245988305b16e1f86cc3c.png#pic_center"/>
     <br/>
     等待Node.js安装完成，然后单击
     <strong>
      Finish
     </strong>
     进入下一步。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7aeee898ef3e4ac8a55ec1d72f964155.png#pic_center"/>
    </p>
    <ol start="4">
     <li>
      在
      <strong>
       SDK Components Setup
      </strong>
      界面，设置OpenHarmony SDK和HarmonyOS SDK存储路径，单击
      <strong>
       Next
      </strong>
      进入下一步。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      OpenHarmony SDK和HarmonyOS SDK不能设置为同一个路径，且路径中不能包含中文字符。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8a86a52eb0654a9a946130182c2b972d.png#pic_center"/>
     <br/>
     5. 在弹出的SDK下载信息页面，单击
     <strong>
      Next
     </strong>
     ，并在弹出的
     <strong>
      License Agreement
     </strong>
     窗口，阅读License协议，需同意License协议后（需同时接受OpenHarmony SDK和HarmonyOS SDK的License协议），单击
     <strong>
      Next
     </strong>
     开始下载SDK。
    </p>
    <blockquote>
     <p>
      说明
      <br/>
      下载SDK过程中，如果出现下载JS SDK失败，提示“Install Js dependencies failed.”，请根据
      <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/faq-development-environment-0000001168829643#section1311117111474" rel="nofollow">
       JS SDK安装失败处理
      </a>
      指导进行处理。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3b5b0cad821644c0ba72a66501d1406b.png#pic_center"/>
     <br/>
     6. 等待SDK下载完成后，单击Finish，完成SDK的下载，界面会进入到DevEco Studio欢迎页。
    </p>
    <h3>
     <a id="42_HDC_243">
     </a>
     4.2 配置HDC工具环境变量
    </h3>
    <p>
     HDC是为开发者提供HarmonyOS应用/服务的调试工具，为方便使用HDC工具，请为HDC工具及其端口号设置环境变量。
    </p>
    <p>
     在
     <strong>
      此电脑 &gt; 属性 &gt; 高级系统设置 &gt; 高级 &gt; 环境变量
     </strong>
     中，添加HDC端口变量名为：HDC_SERVER_PORT，变量值可设置为任意未被占用的端口，如7035。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/103ab83decb94a97ae1a7076095770d6.png#pic_center"/>
    </p>
    <p>
     在系统或者用户的path变量中，添加HDC工具路径，HDC工具路径为：
     <em>
      HarmonyOS SDK安装目录/toolchains/{版本号}
     </em>
     。
     <br/>
     环境变量配置完成后，关闭并重启DevEco Studio。
    </p>
    <h2>
     <a id="5_Hello_World_254">
     </a>
     5. 创建和运行Hello World
    </h2>
    <p>
     DevEco Studio配置开发环境完成后，可以通过运行Hello World工程来验证环境设置是否正确。接下来以创建一个Phone设备的工程为例进行介绍。
    </p>
    <h3>
     <a id="51__258">
     </a>
     5.1 创建一个新工程
    </h3>
    <ol>
     <li>
      打开DevEco Studio，在欢迎页单击Create Project，创建一个新工程。
     </li>
     <li>
      根据工程创建向导，在HarmonyOS页签，选择“Empty Ability”模板，然后单击Next。关于工程模板的介绍和支持的设备类型，请参考
      <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/device_template-0000001053702407" rel="nofollow">
       工程模板和开发语言介绍
      </a>
      。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9254a9403b5043fbba9aeae2cae86f0b.png#pic_center"/>
     </li>
     <li>
      填写工程相关信息，保持默认值即可，单击Finish。关于各个参数的详细介绍，请参考
      <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/create_new_project-0000001053342414" rel="nofollow">
       创建一个新的工程
      </a>
      。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      默认情况下，新建工程的Compile SDK为8，在设备中运行该工程时，请选择API 8及以上的设备才能运行该工程。
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7f528423fcd549a3b8678c6514178e5f.png#pic_center"/>
     <br/>
     工程创建完成后，DevEco Studio会自动进行工程的同步，同步成功如下图所示：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/55d699d72adb41d3b5eae226666322b1.png#pic_center"/>
    </p>
    <h3>
     <a id="52_Hello_World_270">
     </a>
     5.2 运行Hello World
    </h3>
    <p>
     Hello World工程可以运行在模拟器中，或者运行在真机设备设备中。本示例以选择将Hello World工程运行在模拟器中进行说明，如果选择运行在真机设备中，需要先对工程进行签名，然后才能运行在真机设备中，详细指导请参考在
     <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/run_phone_tablat-0000001064774652" rel="nofollow">
      Phone设备中运行应用/服务
     </a>
     。
    </p>
    <p>
     DevEco Studio提供远程模拟器和本地模拟器，本示例以远程模拟器为例进行说明。关于本地模拟器的使用请参考
     <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/run_simulator-0000001053303709#section99703295412" rel="nofollow">
      使用Local Emulator运行应用/服务
     </a>
     。
    </p>
    <ol>
     <li>
      在DevEco Studio菜单栏，单击
      <strong>
       Tools &gt; Device Manager
      </strong>
      。
     </li>
     <li>
      在
      <strong>
       Remote Emulator
      </strong>
      页签中单击
      <strong>
       Sign In
      </strong>
      ，在浏览器中弹出华为开发者联盟帐号登录界面，请输入
      <a href="https://developer.huawei.com/consumer/cn/doc/start/ibca-0000001062388135" rel="nofollow">
       已实名认证
      </a>
      的华为开发者联盟帐号的用户名和密码进行登录（
      <a href="https://developer.harmonyos.com/cn/docs/documentation/doc-guides/faq-remote-emulator-login-0000001087243722" rel="nofollow">
       查看远程模拟器登录常见问题
      </a>
      ）。
     </li>
    </ol>
    <blockquote>
     <p>
      说明
      <br/>
      推荐使用最新版本Chrome浏览器，如果使用Safari、360等其他浏览器，要
      <strong>
       取消阻止跨站跟踪
      </strong>
      和
      <strong>
       阻止所有Cookie
      </strong>
      功能。
     </p>
    </blockquote>
    <ol start="3">
     <li>
      <p>
       登录后，请单击界面的允许按钮进行授权，授权完成后，切换回Device Manager界面。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cc20a3081f5e43ff8e597b50049459ca.png#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       在设备列表中，选择Phone设备P50，并单击
       <strong>
        运行
       </strong>
       按钮，运行模拟器。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/9a1d5f62432f4e0981cb9668074abfd0.png#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       单击DevEco Studio工具栏中的
       <strong>
        运行
       </strong>
       按钮运行工程，或使用默认快捷键
       <strong>
        Shift+F10
       </strong>
       运行工程。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e2621c4ec34f460080807d1d6e3e47b7.png#pic_center"/>
      </p>
     </li>
     <li>
      <p>
       DevEco Studio会启动应用/服务的编译构建，完成后应用/服务即可运行在模拟器上。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/35253be7193049b7b2a8351fdced8015.png#pic_center"/>
      </p>
     </li>
    </ol>
    <p>
     至此，就成功运行了第一个HarmonyOS应用/服务。冲冲冲。
    </p>
