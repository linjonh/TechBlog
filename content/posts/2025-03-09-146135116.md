    <h3>
     现象
    </h3>
    <p>
     Pycharm显示单词存在错误，下面看着有下划波浪线，看着很不舒服。
    </p>
    <p>
     <img alt="" height="249" src="https://i-blog.csdnimg.cn/direct/5a92a60f3028431f824f2f61f5220967.png" width="1358"/>
    </p>
    <p>
     快捷键Alt+Enter，查看提示错误。
    </p>
    <p>
     <img alt="" height="53" src="https://i-blog.csdnimg.cn/direct/95a05747e0c246c7887064e1427bd24d.png" width="269"/>
    </p>
    <p>
     Typo是啥?
    </p>
    <blockquote>
     <p>
      "Typo" 这个词通常用于描述打字或排印过程中的小错误，尤其是拼写错误。它指的是在文本中由于打字或印刷错误而产生的错误。例如，一个常见的typo可能是不小心将“the”打成了“hte”。在编程或写作中，typo可能会导致误解或错误，因此识别和纠正typo是很重要的。
     </p>
    </blockquote>
    <h3>
     解决办法（推荐第三种）
    </h3>
    <ol>
     <li>
      <strong>
       忽略这个错误
      </strong>
      ：如果这个错误不影响代码的运行，你可以选择忽略它。
     </li>
     <li>
      <strong>
       添加到字典
      </strong>
      ：如果你使用的是PyCharm，你可以将
      <code>
       'freqs'
      </code>
      添加到IDE的自定义字典中，这样它就不会被标记为拼写错误了。在PyCharm中，你可以通过以下步骤添加：
      <ul>
       <li>
        打开PyCharm的设置（
        <code>
         File
        </code>
        &gt;
        <code>
         Settings
        </code>
        或使用快捷键
        <code>
         Ctrl+Alt+S
        </code>
        ）。
       </li>
       <li>
        导航到
        <code>
         Editor
        </code>
        &gt;
        <code>
         Spelling
        </code>
        。
       </li>
       <li>
        在
        <code>
         Dictionary
        </code>
        部分，点击
        <code>
         +
        </code>
        按钮添加新单词。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       禁用拼写检查
      </strong>
      ：如果你不希望IDE检查拼写错误，你可以禁用这个功能。在PyCharm中，你可以通过以下步骤禁用：
      <ul>
       <li>
        打开设置（
        <code>
         File
        </code>
        &gt;
        <code>
         Settings
        </code>
        或使用快捷键
        <code>
         Ctrl+Alt+S
        </code>
        ）。
       </li>
       <li>
        导航到
        <code>
         Editor
        </code>
        &gt;
        <code>
         Inspections
        </code>
        。
       </li>
       <li>
        在搜索框中输入
        <code>
         Spelling
        </code>
        ，找到相关的检查项，取消勾选以禁用。
       </li>
      </ul>
     </li>
    </ol>

    </p>
    <p>
     完美解决！
    </p>
