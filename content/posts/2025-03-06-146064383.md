     Ubuntu20.04本地配置IsaacLab 4.5.0的训练环境（一）
    </h1>
   </div>
  </div>
 </div>

     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
    </p>
    <br/>
    <strong>
     写在前面，听说IsaacLab 4.2.0有BUG，所以装个4.5.0试试。基本安装步骤和4.2.0一致。
    </strong>
    <p>
    </p>
    <p>
     <strong>
      写在前面，需要约200G的空间（因为存在资产包压缩包的下载与解压，同时存在可能会要很大的空间，但实际上用上的只是解压后的文件），如果空间不够的可能需要考虑后面在下载资产包的时候少下载一些，但是只下载部分资产包是否有用无法确定。
     </strong>
    </p>
    <p>
     <strong>
      写在前面，后续所有的安装，比如conda虚拟环境名称等，你都可以自己指定，并不一定要跟我一样。但要注意相应路径需要修改。
     </strong>
    </p>
    <p>
     <strong>
      写在前面，下述的安装主要是为了强化学习使用，若只是使用IsaacLab平台，但不一定使用强化学习，不必全部按照下面方式安装。
     </strong>
    </p>
    <h2>
     <a id="condaIsaacLabisaaclabsh_install_8">
     </a>
     配置conda虚拟环境（对于这一步，个人感觉跟在配置IsaacLab那一节的./isaaclab.sh --install同样要执行这一步，建议先不执行）
    </h2>
    <p>
     首先确认python版本，torch版本及其cuda版本。
     <strong>
      对于IsaacLab 4.5.0，需要python3.10，torch2.5.1
     </strong>
     ，而
     <strong>
      torch2.5.1对应的cuda版本有几个
     </strong>
     ，这可通过
     <a href="https://pytorch.org/get-started/previous-versions/" rel="nofollow">
      pytorch官网确认
     </a>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/782e7b2d1dba45bcade1149bbad06b3a.png">
      <br/>
      由于本人之前在安装4.2.0时已经配置了cuda12.1，在此直接安装匹配cuda12.1的torch。
     </img>
    </p>
    <p>
     启动终端，执行：
    </p>
    <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> isaaclab_4_5_0 <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.10</span>
conda activate isaaclab_4_5_0
conda <span class="token function">install</span> <span class="token assign-left variable">pytorch</span><span class="token operator">==</span><span class="token number">2.5</span>.1 <span class="token assign-left variable">torchvision</span><span class="token operator">==</span><span class="token number">0.20</span>.1 <span class="token assign-left variable">torchaudio</span><span class="token operator">==</span><span class="token number">2.5</span>.1 pytorch-cuda<span class="token operator">=</span><span class="token number">12.1</span> <span class="token parameter variable">-c</span> pytorch <span class="token parameter variable">-c</span> nvidia <span class="token comment"># 安装pytorch2.5.1</span>
</code></pre>
    <h2>
     <a id="IsaacSim_21">
     </a>
     配置IsaacSim
    </h2>
    <p>
     从
     <a href="https://docs.isaacsim.omniverse.nvidia.com/latest/installation/download.html" rel="nofollow">
      官网
     </a>
     <strong>
      下载IsaacSim 4.5.0安装包（Latest Release中选择4.5.0版本，Linux版本）
     </strong>
     ：
    </p>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/2d963f5a6e084ebca48e5077ab0a7bcb.png">
      <br/>
      在**./local/share/ov/pkg/目录下创建文件夹isaac-sim-4.5.0**，将安装包
      <strong>
       解压至该文件夹内
      </strong>
      。
     </img>
    </p>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/0ec3d1b4e9864da3a6855fd65e14092d.png">
      <br/>
      <strong>
       在该文件夹下启动终端
      </strong>
      ，执行：
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 若不是在./local/share/ov/pkg文件夹下isaac-sim-4.5.0中进行文件解压，还需执行一步./post_install.sh</span>
./isaac-sim.selector.sh
</code></pre>
    <p>
     出现如下GUI界面（
     <strong>
      第一次运行可能会出现报错，点击确定即可，对后续使用并没有影响
     </strong>
     ）：
    </p>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/5183233e042645bf81763cab45cedfa9.png">
      <br/>
      点击SATRT即启动IsaacSim，第一次启动会比较慢，等候即可。(Streaming表示不显示界面）
     </img>
    </p>
    <p>
     <strong>
      另一种启动方法
     </strong>
     则是直接在终端中运行：
    </p>
    <pre><code class="prism language-bash">./isaac-sim.sh
</code></pre>
    <p>
     设置环境变量，在终端中执行（或直接鼠标打开）：
    </p>
    <pre><code class="prism language-bash">gedit ~/.bashrc
</code></pre>
    <p>
     加入指令：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">ISAACSIM_PATH</span><span class="token operator">=</span><span class="token string">"<span class="token variable">${<!-- --><span class="token environment constant">HOME</span>}</span>/.local/share/ov/pkg/isaac-sim-4.5.0"</span> <span class="token comment"># 这里请跟你解压的路径对应</span>
<span class="token builtin class-name">export</span> <span class="token assign-left variable">ISAACSIM_PYTHON_EXE</span><span class="token operator">=</span><span class="token string">"<span class="token variable">${ISAACSIM_PATH}</span>/python.sh"</span>
</code></pre>
    <h2>
     <a id="IsaacLab_59">
     </a>
     配置IsaacLab
    </h2>
    <p>
     从
     <a href="https://docs.isaacsim.omniverse.nvidia.com/latest/installation/download.html" rel="nofollow">
      官网
     </a>
     <strong>
      下载IsaacSim Assets，版本4.5.0，三个资产包
     </strong>
     ，若空间足够可全部下载。
    </p>
    <p>
     <strong>
      在任意位置创建文件夹
     </strong>
     （比如我则是在home目录下创建了isaaclab文件夹，其中创建了4-5-0子文件夹，后续我都会以这个文件夹来说明，
     <strong>
      读者根据自己需求进行更改
     </strong>
     。）
    </p>
    <p>
     将
     <strong>
      所有下载的资产包解压至同一文件夹内
     </strong>
     。
    </p>
    <p>
     安装依赖：
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt</span> <span class="token function">install</span> cmake build-essential
<span class="token builtin class-name">cd</span> ~/.local/share/ov/pkg/
<span class="token function">git</span> clone https://github.com/isaac-sim/IsaacLab.git <span class="token comment"># 一定要拉去最新版的！旧版IsaacLab不适配！出现网络问题无法git clone的可以试试用镜像，git clone https://kkgithub.com/isaac-sim/IsaacLab.git或git clone https://githubfast.com/isaac-sim/IsaacLab.git</span>
<span class="token builtin class-name">cd</span> IsaacLab
gedit isaaclab.sh
</code></pre>
    <p>
     在sh文件上方，命令“export ISAACLAB_PATH=“ ( c d " ( cd " (cd"( dirname “${BASH_SOURCE[0]}” )” &amp;&gt; /dev/null &amp;&amp; pwd )"”之前添加代码：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.local/share/ov/pkg/isaac-sim-4.5.0/setup_conda_env.sh
</code></pre>
    <p>
     而后，关闭文件，在终端中执行：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> ~/.bashrc
<span class="token function">ln</span> <span class="token parameter variable">-s</span> ~/.local/share/ov/pkg/isaac-sim-4.5.0 _isaac_sim
<span class="token comment"># 对于下述指令，若没有安装虚拟环境，请执行</span>
./isaaclab.sh <span class="token parameter variable">--conda</span> my_env <span class="token comment"># my_env实际上是创建的conda虚拟环境的名称，由读者自行指定，也可不指定，用默认名称isaaclab</span>
<span class="token comment"># 随后执行</span>
./isaaclab.sh <span class="token parameter variable">--install</span> <span class="token comment"># 执行这一步，请确保你已经在conda虚拟环境中</span>
</code></pre>
    <p>
     若出现如下报错：
    </p>
    <pre><code class="prism language-bash">ERROR: Could not <span class="token function">find</span> a version that satisfies the requirement rsl-rl <span class="token punctuation">(</span>unavailable<span class="token punctuation">)</span> <span class="token punctuation">(</span>from omni-isaac-lab-tasks<span class="token punctuation">[</span>all<span class="token punctuation">]</span><span class="token punctuation">)</span> <span class="token punctuation">(</span>from versions: none<span class="token punctuation">)</span>
ERROR: No matching distribution found <span class="token keyword">for</span> rsl-rl <span class="token punctuation">(</span>unavailable<span class="token punctuation">)</span>
</code></pre>
    <p>
     执行（
     <strong>
      确保你在conda虚拟环境中
     </strong>
     ）：
    </p>
    <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token string">"git+https://github.com/leggedrobotics/rsl_rl.git"</span>
</code></pre>
    <p>
     然后，启动isaaclab，第一次启动会比较费时，要几分钟：
    </p>
    <pre><code class="prism language-bash">./isaaclab.sh <span class="token parameter variable">-s</span>
</code></pre>
    <p>
     成功后出现界面，
     <strong>
      顶部显示Isaac Sim Full 4.5.0
     </strong>
     。
    </p>
    <p>
     而后，在./local/share/ov/pkg/isaac-sim-4.5.0/apps/isaacsim.exp.base.kit文件中添加（
     <strong>
      注意把其中的路径都修改为你自己的本地路径
     </strong>
     ）：
    </p>
    <pre><code class="prism language-bash">persistent.isaac.asset_root.default <span class="token operator">=</span> <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5"</span>
exts.<span class="token string">"isaacsim.asset.browser"</span>.folders <span class="token operator">=</span> <span class="token punctuation">[</span>
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Robots"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/People"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/IsaacLab"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Props"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Environments"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Materials"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Samples"</span>,
    <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5/Isaac/Sensors"</span>,
<span class="token punctuation">]</span><span class="token operator">!</span><span class="token punctuation">[</span>请添加图片描述<span class="token punctuation">]</span><span class="token punctuation">(</span>https://i-blog.csdnimg.cn/direct/e0d7260fbd584be0baa3575526184cd8.png<span class="token punctuation">)</span>

</code></pre>
    <p>
     <strong>
      在IsaacSim文件夹
     </strong>
     下执行：
    </p>
    <pre><code class="prism language-bash">./isaac-sim.sh --/persistent/isaac/asset_root/default<span class="token operator">=</span><span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5"</span> <span class="token comment"># 这里的路径是你自己存放资产的本地路径</span>
</code></pre>
    <p>
     若运行过程中除了ROS2报错外无其它错误，点击如图所示的按钮，若
     <strong>
      显示路径为资产包路径
     </strong>
     ，说明设置成功。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/eb498cac2a74483e9f407d094b86fd4d.png">
      随后，将IsaacLab文件夹下的
     </img>
    </p>
    <pre><code class="prism language-bash">NUCLEUS_ASSET_ROOT_DIR <span class="token operator">=</span> carb.settings.get_settings<span class="token punctuation">(</span><span class="token punctuation">)</span>.get<span class="token punctuation">(</span><span class="token string">"/persistent/isaac/asset_root/cloud"</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     修改为你自己的资产包路径：
    </p>
    <pre><code class="prism language-bash">NUCLEUS_ASSET_ROOT_DIR <span class="token operator">=</span> <span class="token string">"/home/&lt;username&gt;/isaacsim_assets/Assets/Isaac/4.5"</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1c0985d319a14a39b5519d42a51765ad.png"/>
    </p>
    <p>
     随后
     <strong>
      在IsaacLab文件夹下运行
     </strong>
     ：
    </p>
    <pre><code class="prism language-bash">./isaaclab.sh <span class="token parameter variable">-p</span> scripts/demos/quadrupeds.py
</code></pre>
    <p>
     出现如下界面：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/67de2844ec5640088acd477d458bc7eb.png"/>
