    <h2>
     问题描述：
    </h2>
    <p>
     java.lang.ArrayIndexOutOfBoundsException: 0
    </p>
    <h2>
     问题分析：
    </h2>
    <p>
     1、数组初始化没有给数组定义大小，给数组添加元素时索引大于数组大小，导致保错。
    </p>
    <pre><code class="language-java">        String[] data = new String[]{};
        for (int i = 0; i &lt; 5; i++) {
            data[i] = i + "";
        }</code></pre>
    <p>
     解决办法：
    </p>
    <p>
     数组初始化时给数组指定大小，并且赋值时判断索引是否小于数组大小。
    </p>
    <pre><code class="language-java">        String[] data = new String[5];
        for (int i = 0; i &lt; data.length; i++) {
            data[i] = i + "";
        }</code></pre>
