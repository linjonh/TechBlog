     <p>
      推荐一个开源项目，它能帮助你沉浸式编程。这是一个基于 Linux 的 VR 桌面开源项目：Simula ，这是一个运行在 Godot 之上的 Linux 虚拟现实窗口管理器.
     </p>
     <p>
      Godot：是一款制作游戏的软件，通过基于节点的架构来设计 2D 和 3D 游戏。3D 渲染器设计可以增强 3D 游戏的画面。具有内置工具的 2D 游戏功能以像素坐标工作，可以掌控 2D 游戏效果。
     </p>
     <p style="text-align:center;">
      <img alt="f53e3ad0b7ee1414fc80902ebd583983.gif" src="https://i-blog.csdnimg.cn/blog_migrate/bd9503747e9058cff5497ea04e126bff.gif"/>
     </p>
     <p>
      如果说戴上 VR 眼镜看不到键盘和鼠标，Simula 也帮你考虑到了，这个虚拟桌面有一个摄像头视图，可让您从 VR 中查看鼠标和键盘。
      <br/>
     </p>
     <p>
      <img alt="bb827605f101b14bd9bb51288a794055.png" src="https://i-blog.csdnimg.cn/blog_migrate/83a027a8cf9c059248090db4a56d1035.png"/>
     </p>
     <p>
      开发者说这个开源项目的使命是：促进基于虚拟现实和增强现实的 Linux  桌面发展。如果你想安装这个 VR 桌面，只需要运行下面的命令和准备其他设备：
     </p>
     <pre class="has"><code class="language-go"># Clone
git clone --depth 1 --recursive https://github.com/SimulaVR/Simula
cd Simula

# Download (or build, if downloading fails)

source ./utils/Helpers.sh &amp;&amp; installSimula

# Launch

./result/bin/simula # Needs SteamVR to be running</code></pre>
<p>
安装时间不到 1 分钟。
</p>
