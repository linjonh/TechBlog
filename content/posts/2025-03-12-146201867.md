    <p>
     今天田辛老师为大家带来一篇关于在VMWare虚拟机上安装Ubuntu系统的详细教程。无论是学习、开发还是测试，虚拟机都是一个非常实用的工具，它允许我们在同一台物理机上运行多个操作系统。Ubuntu作为一款开源、免费且用户友好的Linux发行版，深受广大开发者和用户的喜爱。接下来，我将一步步带大家完成在VMWare上安装Ubuntu的过程。
    </p>
    <h3>
     <a id="1__2">
     </a>
     1 下载安装镜像
    </h3>
    <h4>
     <a id="11__3">
     </a>
     1.1 下载的网站
    </h4>
    <p>
     国内的用户可以直接到Ubuntu的国内站点进行访问
     <a href="https://cn.ubuntu.com/" rel="nofollow">
      https://cn.ubuntu.com/
     </a>
     。这个站点提供了快速且稳定的下载服务，非常适合国内用户。
     <br/>
     <img alt="showdoc_b745ec784dcfe77db16df39d99438544.png" src="https://i-blog.csdnimg.cn/img_convert/5e96f0a22cdcdfac3adc86f60607c6ee.png"/>
    </p>
    <h4>
     <a id="12__6">
     </a>
     1.2 选择相应的版本进行下载。
    </h4>
    <ul>
     <li>
      <p>
       在下载页面上，我们选择“桌面版”进行下载。桌面版提供了完整的图形用户界面，适合大多数用户。
       <br/>
       <img alt="showdoc_662b74170b7445583485cf74f75fecad.png" src="https://i-blog.csdnimg.cn/img_convert/02e24a89d1c051a9ea2f0dd483483f7e.png"/>
      </p>
     </li>
     <li>
      <p>
       选择右侧【下载24.04.02】
       <br/>
       <img alt="QQ_1741747142525.png" src="https://i-blog.csdnimg.cn/img_convert/2204897d576158caa19a62c4d4f24c90.png"/>
      </p>
     </li>
     <li>
      <p>
       下载完成后，我们将得到一个ISO镜像文件，这是安装Ubuntu所必需的。
       <br/>
       <img alt="QQ_1741747353820.png" src="https://i-blog.csdnimg.cn/img_convert/3ee2132561e77643cb0fc0b793757497.png"/>
      </p>
     </li>
    </ul>
    <h3>
     <a id="2__15">
     </a>
     2 新建虚拟机
    </h3>
    <p>
     接下来，我们需要在VMWare中新建一个虚拟机来安装Ubuntu。
    </p>
    <ol>
     <li>
      创建虚拟机：打开VMWare，点击【菜单】&gt; 【文件】&gt; 选择“新建虚拟机”。
      <br/>
      <img alt="QQ_1741749404837.png" src="https://i-blog.csdnimg.cn/img_convert/7d620da567927f0f182ff0cd641f7cb4.png"/>
     </li>
     <li>
      选择自定义（高级）‌：为了更灵活地配置虚拟机，我们选择“自定义（高级）”选项。
      <img alt="QQ_1741749702557.png" src="https://i-blog.csdnimg.cn/img_convert/36500d70200717362aee608ffb9356b4.png"/>
     </li>
     <li>
      设定虚拟硬件兼容‌：根据VMWare的版本和硬件兼容性要求，选择合适的硬件兼容级别。
      <br/>
      <img alt="QQ_1741749791786.png" src="https://i-blog.csdnimg.cn/img_convert/d81b7a78e619a8ba083bc9599171d8a8.png"/>
     </li>
     <li>
      设定安装ISO文件‌：在这一步，我们选择之前下载的Ubuntu ISO镜像文件作为安装源。此时，VMWare可能会触发Ubuntu简易安装机制，简化安装过程。（虽然没什么用:)）
      <br/>
      <img alt="QQ_1741749874550.png" src="https://i-blog.csdnimg.cn/img_convert/86243ae76fc7c78d21fcc3f66e809fc3.png"/>
     </li>
     <li>
      设定建议安装信息‌：此处瞎填就行，只要符合验证即可。 实测这么页面没用
      <br/>
      <img alt="QQ_1741750217839.png" src="https://i-blog.csdnimg.cn/img_convert/9330715c90d3396e3fdeda2b3e1b7ff5.png"/>
     </li>
     <li>
      设定虚拟机名称和存储位置‌：为虚拟机命名并选择存储位置，方便后续管理。
      <br/>
      <img alt="QQ_1741750308572.png" src="https://i-blog.csdnimg.cn/img_convert/e46b4b7084377f9ad56fab5f43e0f554.png"/>
     </li>
     <li>
      设定处理器信息‌：根据实际需求分配处理器核心数和线程数。
      <br/>
      <img alt="QQ_1741750409235.png" src="https://i-blog.csdnimg.cn/img_convert/29c4f6a9e00ec8c621d4841b899872e6.png"/>
     </li>
     <li>
      设定内存‌：根据虚拟机用途和宿主机内存资源，合理分配内存大小。
      <br/>
      <img alt="QQ_1741750533332.png" src="https://i-blog.csdnimg.cn/img_convert/1666ef30d27aef13297e0f847507a546.png"/>
     </li>
     <li>
      设定网络连接方式‌：选择合适的网络连接方式，如NAT或桥接，以满足虚拟机访问网络的需求。
      <br/>
      <img alt="QQ_1741750670529.png" src="https://i-blog.csdnimg.cn/img_convert/6c2bc88e4117eb098de9a6a16621ee54.png"/>
     </li>
     <li>
      设定I/O控制器的连接方式‌：通常保持默认设置即可。
      <br/>
      <img alt="QQ_1741750747791.png" src="https://i-blog.csdnimg.cn/img_convert/d1987ccaac5851902cc36d297b1eb1b9.png"/>
     </li>
     <li>
      设定虚拟磁盘类型‌：选择合适的虚拟磁盘类型，直接选SCSI
      <br/>
      <img alt="QQ_1741750837156.png" src="https://i-blog.csdnimg.cn/img_convert/443bbf945eb717404f94226085b293b0.png"/>
     </li>
     <li>
      设置虚拟磁盘：新建还是使用现有。
      <br/>
      <img alt="QQ_1741750909222.png" src="https://i-blog.csdnimg.cn/img_convert/b51a7877a845991966cd7cd955b23f54.png"/>
     </li>

      <br/>
      <img alt="QQ_1741751247478.png" src="https://i-blog.csdnimg.cn/img_convert/3340b8db00ae736e83e0945d6a0201d6.png"/>

      <img alt="QQ_1741755858256.png" src="https://i-blog.csdnimg.cn/img_convert/d938c931a5c395887f698be74048e64e.png"/>

     </li>
     <li>
      准备就绪
      <br/>
      <img alt="QQ_1741757357640.png" src="https://i-blog.csdnimg.cn/img_convert/1384ff1d2401024a419a4ee1be27a22f.png"/>
     </li>
    </ol>
    <h4>
     <a id="42__96">
     </a>
     4.2 系统升级
    </h4>
    <ol>
     <li>
      用
      <code>
       Ctrl+Alt+T
      </code>
      打开终端
     </li>
     <li>
      执行命令
      <code>
       sudo apt update
      </code>
      : 更新本地软件包列表，从Ubuntu的软件源服务器获取最新的软件包信息。作为升级软件的第一步，确保您获取到的是最新的软件包信息。
      <br/>
      <img alt="QQ_1741757923186.png" src="https://i-blog.csdnimg.cn/img_convert/8fd27a5aa0b2e62ed047b3d8186038bb.png"/>
     </li>
     <li>
      执行命令
      <code>
       sudo apt upgrade
      </code>
      ：升级已安装的软件包到最新版本，基于
      <code>
       sudo apt update
      </code>
      获取的最新软件包列表。
      <br/>
      <img alt="QQ_1741758125966.png" src="https://i-blog.csdnimg.cn/img_convert/d79607815433920a1d2e1c0f1c3af898.png"/>
      <br/>
      <img alt="QQ_1741758203462.png" src="https://i-blog.csdnimg.cn/img_convert/ea87ec608e25d663536adbfefd82970b.png"/>
     </li>
     <li>
      执行命令
      <code>
       sudo apt dist-upgrade
      </code>
      此命令除了升级已安装的软件包外，还可能升级系统内核，并解决依赖关系问题，可能需要安装或卸载一些软件包。
      <br/>
      <img alt="QQ_1741758334290.png" src="https://i-blog.csdnimg.cn/img_convert/e723b3cc783447bbeb81724a62e5eddd.png"/>
     </li>
    </ol>
