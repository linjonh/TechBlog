    <h3>
     <a id="Python_0">
     </a>
     查看系统中已安装的所有Python版本
    </h3>
    <p>
     使用 ls 命令来查看你的系统中都有那些 Python 的二进制文件可供使用。
    </p>
    <pre><code class="prism language-css">xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ ls /usr/bin/python*
/usr/bin/python            /usr/bin/python2-config  /usr/bin/python3m
/usr/bin/python2           /usr/bin/python3         /usr/bin/python-config
/usr/bin/python2.7         /usr/bin/python3.6
/usr/bin/python2.7-config  /usr/bin/python3.6m

</code></pre>
    <p>
     可以看到系统中安装了python2.7和python3.6。
    </p>
    <h3>
     <a id="UbuntuPython_12">
     </a>
     Ubuntu中查看已安装Python的版本
    </h3>
    <p>
     终端命令，查看默认使用python的版本
    </p>
    <pre><code class="prism language-css">python --version
</code></pre>
    <p>
     查看Python2版本
    </p>
    <pre><code class="prism language-css">python2 --version 
</code></pre>
    <p>
     查看Python3版本
    </p>
    <pre><code class="prism language-css">python3 --version
</code></pre>
    <h3>
     <a id="Python_25">
     </a>
     修改默认Python版本
    </h3>
    <p>
     如果系统中安装了多个版本的Python，可以通过命令进行设置。假设当前系统是Python2，现在修改为Python3：
    </p>
    <pre><code class="prism language-css">echo alias python=python3 &gt;&gt; ~/.bashrc  #切换python默认版本类型
</code></pre>
    <p>
     <strong>
      如果系统里安装了多个python3，比如python 3.6 python3.8，可以通过版本号进一步指定对应版本。
     </strong>
    </p>
    <p>
     通过source命令使修改立即生效或者重启系统：
    </p>
    <pre><code class="prism language-css">source ~/.bashrc  #更新配置文件的命令，只更新当前终端上的配置
</code></pre>
    <pre><code class="prism language-css">xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ python --version
Python 2.7.17
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ python2 --version
Python 2.7.17
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ python3 --version
Python 3.6.9
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ echo alias python=python3 &gt;&gt; ~/.bashrc
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ source ~/.bashrc
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ python --version
Python 3.6.9
xiyou@<span class="token property">xiyou-virtual-machine</span><span class="token punctuation">:</span>~$ 

</code></pre>
