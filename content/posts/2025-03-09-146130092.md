    <h2>
     <strong>
      LeetCode 跳跃游戏 IV：二进制字符串的跳跃问题
     </strong>
    </h2>
    <h3>
     <strong>
      题目描述
     </strong>
    </h3>
    <p>
     给定一个下标从 0 开始的二进制字符串
     <code>
      s
     </code>
     和两个整数
     <code>
      minJump
     </code>
     和
     <code>
      maxJump
     </code>
     。初始时，你位于下标 0 处（保证该位置为 '0'）。你需要判断是否能到达字符串的最后一个位置（下标为
     <code>
      s.length - 1
     </code>
     ）。移动规则如下：
    </p>
    <ul>
     <li>
      从位置
      <code>
       i
      </code>
      移动到
      <code>
       j
      </code>
      ，需满足
      <code>
       i + minJump ≤ j ≤ min(i + maxJump, s.length - 1)
      </code>
      。
     </li>
     <li>
      目标位置
      <code>
       j
      </code>
      必须为 '0'。
     </li>
    </ul>
    <h3>
     <strong>
      解题思路分析
     </strong>
    </h3>
    <h4>
     <strong>
      动态规划 + 前缀和优化
     </strong>
    </h4>
    <h5>
     <strong>
      核心思想
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       动态规划数组
       <code>
        dp
       </code>
      </strong>
      ：
      <code>
       dp[i]
      </code>
      表示是否能到达位置
      <code>
       i
      </code>
      。
     </li>
     <li>
      <strong>
       前缀和数组
       <code>
        pre
       </code>
      </strong>
      ：记录从 0 到当前位置的可达状态总和，用于快速计算区间和。
     </li>
     <li>
      <strong>
       有效区间
      </strong>
      ：对于位置
      <code>
       j
      </code>
      ，其可跳跃的起始位置范围为
      <code>
       [j - maxJump, j - minJump]
      </code>
      。通过前缀和数组快速判断该区间内是否有可达的位置。
     </li>
    </ol>
    <h5>
     <strong>
      关键步骤
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        初始化
       </strong>
       ：
      </p>
      <ul>
       <li>
        若最后一个字符为 '1'，直接返回
        <code>
         false
        </code>
        。
       </li>
       <li>
        <code>
         dp[0] = true
        </code>
        （初始位置可达）。
       </li>
       <li>
        前缀和数组
        <code>
         pre
        </code>
        记录可达位置的累计数量。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        遍历每个位置
        <code>
         j
        </code>
       </strong>
       ：
      </p>
      <ul>
       <li>
        若当前位置为 '1'，跳过。
       </li>
       <li>
        计算可跳跃的起始位置范围
        <code>
         [left, right]
        </code>
        。
       </li>
       <li>
        利用前缀和数组快速查询该区间内是否有可达位置。
       </li>
       <li>
        更新
        <code>
         dp[j]
        </code>
        和前缀和数组
        <code>
         pre
        </code>
        。
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     <strong>
      代码实现
     </strong>
    </h3>
    <h4>
     <strong>
      方法一：标准动态规划 + 前缀
     </strong>
    </h4>
    <pre><code class="language-java">class Solution {
    public boolean canReach(String s, int minJump, int maxJump) {
        int n = s.length();
        if (s.charAt(n - 1) != '0') return false;
        
        boolean[] dp = new boolean[n];
        dp[0] = true;
        int[] pre = new int[n + 1]; // pre[i] 表示前i个位置的可达数
        pre[1] = 1;
        
        for (int j = 1; j &lt; n; j++) {
            if (s.charAt(j) != '0') {
                pre[j + 1] = pre[j];
                continue;
            }
            
            int left = j - maxJump;
            int right = j - minJump;
            left = Math.max(left, 0);
            right = Math.min(right, j - 1);
            
            if (left &gt; right) {
                pre[j + 1] = pre[j];
                continue;
            }
            
            int sum = pre[right + 1] - pre[left];
            dp[j] = sum &gt; 0;
            pre[j + 1] = pre[j] + (dp[j] ? 1 : 0);
        }
        return dp[n - 1];
    }
}</code></pre>
    <h4>
     <strong>
      方法二：简化前缀和处理
     </strong>
    </h4>
    <pre><code class="language-java">class Solution {
    public boolean canReach(String s, int minJump, int maxJump) {
        int n = s.length();
        if (s.charAt(n - 1) != '0') return false;
        
        boolean[] dp = new boolean[n];
        dp[0] = true;
        int pre = 0; // 当前窗口内的可达数
        
        for (int i = 1; i &lt; n; i++) {
            if (i &gt;= minJump) pre += dp[i - minJump] ? 1 : 0;
            if (i &gt; maxJump) pre -= dp[i - maxJump - 1] ? 1 : 0;
            dp[i] = s.charAt(i) == '0' &amp;&amp; pre &gt; 0;
        }
        return dp[n - 1];
    }
}</code></pre>
    <h3>
     <strong>
      代码解释
     </strong>
    </h3>
    <h4>
     <strong>
      方法一关键点
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        前缀和数组
        <code>
         pre
        </code>
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         pre[i]
        </code>
        表示前
        <code>
         i
        </code>
        个位置（即索引
        <code>
         0
        </code>
        到
        <code>
         i-1
        </code>
        ）的可达数。
       </li>
       <li>
        通过
        <code>
         pre[right+1] - pre[left]
        </code>
        快速计算区间
        <code>
         [left, right]
        </code>
        的可达数。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        有效区间计算
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         left = j - maxJump
        </code>
        ，
        <code>
         right = j - minJump
        </code>
        ，确保区间不越界。
       </li>
       <li>
        若区间为空（
        <code>
         left &gt; right
        </code>
        ），则当前位置不可达。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     <strong>
      方法二优化
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       滚动窗口优化
      </strong>
      ：
      <ul>
       <li>
        直接维护当前窗口内的可达数
        <code>
         pre
        </code>
        ，无需额外数组。
       </li>
       <li>
        当
        <code>
         i
        </code>
        超过
        <code>
         minJump
        </code>
        时，将
        <code>
         dp[i - minJump]
        </code>
        加入窗口。
       </li>
       <li>
        当
        <code>
         i
        </code>
        超过
        <code>
         maxJump
        </code>
        时，将
        <code>
         dp[i - maxJump - 1]
        </code>
        移出窗口。
       </li>
      </ul>
     </li>
    </ul>
    <h3>
     <strong>
      复杂度分析
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：
      <img alt="O(n)" class="mathcode" src="https://latex.csdn.net/eq?O%28n%29">
       ，每个位置仅遍历一次。
      </img>
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：
      <img alt="O(n)" class="mathcode" src="https://latex.csdn.net/eq?O%28n%29">
       ，需存储
       <code>
        dp
       </code>
       数组和前缀和数组（方法一）或仅
       <code>
        dp
       </code>
       数组（方法二）。
      </img>
     </li>
    </ul>
    <h3>
     <strong>
      总结与优化
     </strong>
    </h3>
    <ol>
     <li>
      <strong>
       动态规划是核心
      </strong>
      ：通过
      <code>
       dp
      </code>
      数组记录状态，避免重复计算。
     </li>
     <li>
      <strong>
       前缀和优化
      </strong>

      <img alt="O(n)" class="mathcode" src="https://latex.csdn.net/eq?O%28n%29">

       <img alt="O(1)" class="mathcode" src="https://latex.csdn.net/eq?O%281%29">
        ，大幅提升效率。
       </img>
