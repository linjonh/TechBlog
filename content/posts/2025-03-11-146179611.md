    <p>
     OWL: 适用于现实任务自动化的多智能体协作框架
    </p>
    <p>
    </p>
    <h5>
     引言
    </h5>
    <p>
    </p>
    <p>
     在人工智能技术不断发展的今天，多智能体协作成为了推动自动化进步的重要方向。OWL（Optimized Workforce Learning）是一个开源框架，基于CAMEL-AI架构，旨在优化多智能体协作能力，提高现实任务自动化的效率和灵活性。
    </p>
    <p>
    </p>
    <h5>
     核心特性
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        实时信息检索
       </strong>
       ：支持从Wikipedia、Google Search等在线资源获取最新信息。
      </p>
     </li>
     <li>
      <p>
       <strong>
        多模态处理
       </strong>
       ：能够处理视频、图片和音频数据，实现跨媒介信息处理。
      </p>
     </li>
     <li>
      <p>
       <strong>
        浏览器自动化
       </strong>
       ：借助Playwright框架，实现页面滚动、点击、输入、文件下载等操作。
      </p>
     </li>
     <li>
      <p>
       <strong>
        文档解析
       </strong>
       ：支持Word、Excel、PDF和PPT等格式的文档内容提取。
      </p>
     </li>
     <li>
      <p>
       <strong>
        代码执行
       </strong>
       ：支持Python代码的编写与执行，增强自动化能力。
      </p>
     </li>
     <li>
      <p>
       <strong>
        内置工具库
       </strong>
       ：包括ArxivToolkit、AudioAnalysisToolkit、GoogleScholarToolkit等，满足不同任务需求。
      </p>
     </li>
    </ol>
    <p>
    </p>
    <h5>
     安装与使用
    </h5>
    <p>
    </p>
    <h6>
     安装方法
    </h6>
    <p>
     OWL支持多种安装方式，包括
     <code>
      uv
     </code>
     、
     <code>
      venv + pip
     </code>
     和
     <code>
      conda
     </code>
     。推荐使用
     <code>
      uv
     </code>
     进行安装：
    </p>
    <pre><code># 克隆GitHub仓库
git clone https://github.com/camel-ai/owl.git
cd owl

# 安装uv并创建虚拟环境
pip install uv
uv venv .venv --python=3.10

# 激活虚拟环境
source .venv/bin/activate  # macOS/Linux
.venv\Scripts\activate    # Windows

# 安装依赖
uv pip install -e .</code></pre>
    <p>
    </p>
    <h6>
     快速开始
    </h6>
    <p>
     运行以下命令启动OWL：
    </p>
    <pre><code>python owl/run.py</code></pre>
    <p>
    </p>
    <h6>
     运行不同模型
    </h6>
    <pre><code>python owl/run_qwen.py      # 使用Qwen模型
python owl/run_deepseek.py  # 使用Deepseek模型
python owl/run_openai_compatiable_model.py  # 兼容OpenAI模型</code></pre>
    <p>
    </p>
    <h5>
     Web界面
    </h5>
    <p>
     OWL提供基于Gradio的Web界面，用户可以方便地选择模型、管理环境变量、查看交互历史等。
    </p>
    <pre><code>python run_app.py</code></pre>
