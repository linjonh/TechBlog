# 微信小程序长按识别图片二维码

## 提示：二维码图片才能显示识别菜单

## 1.第一种方法

添加属性：show-menu-by-longpress

```

添加属性：show-menu-by-longpress
<image src="{{shop.wx_qrcode}}" mode="widthFix" show-menu-by-longpress></image>
```

## 2.第二种方法

先点击图片预览，再长按识别

```

previewImage(e){
		var src='';//图片链接
		wx.previewImage({
			current: src, //链接字符串，默认显示urls第一张
  			urls: ['',''] // 数组，需要预览的图片列表
		})
}

```
