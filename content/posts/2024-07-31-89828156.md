# H3C 查看路由器的硬件信息

display version //查看版本

display diagnostic-information  //查看设备运行的统计信息

display device verbose   //
查看单板的详细信息

display device manuinfo //查看单板的电子标签信息,包括序列号

display cpu-usage //
查看单板

CPU

利用率的统计信息

display memory  //
查看单板内存的使用状况

display device cf-card //
查看

CF

卡的相关信息

display fan //
查看设备风扇的工作状态

display power //
查看电源的工作状态

display environment //
查看单板的温度告警门限
