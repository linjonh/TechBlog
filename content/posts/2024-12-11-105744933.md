    <h4>
     <a id="_1">
     </a>
     前言
    </h4>
    <p>
     Windows搭建FTP服务器后，相应配置登录FTP服务器的用户，方便之后使用这些用户访问FTP资源。
     <br/>
     这篇文章是
     <a href="https://blog.csdn.net/u012815136/article/details/104881301">
      windows上搭建FTP服务器
     </a>
     的续集。
    </p>
    <h4>
     <a id="_5">
     </a>
     环境
    </h4>
    <p>
     Windows 10 专业版
    </p>
    <h4>
     <a id="FTP_8">
     </a>
     FTP用户信息
    </h4>
    <blockquote>
     <p>
      示例添加两个FTP用户，一个有读写权限，一个只有读权限
     </p>
    </blockquote>
    <ol>
     <li>
      ftpadmin，完全控制FTP，有读写权限
     </li>
     <li>
      ftpview，只有读权限
     </li>
    </ol>
    <h4>
     <a id="_14">
     </a>
     操作步骤详解
    </h4>
    <h5>
     <a id="1__15">
     </a>
     1. 添加用户
    </h5>
    <p>
     选择
     <code>
      此电脑
     </code>
     &gt;
     <code>
      管理
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eb11485594fccb4d62455c87c833709b.png">
      <br/>
      在“计算机管理”界面，找到
      <code>
       计算机管理(本地)
      </code>
      &gt;
      <code>
       系统工具
      </code>
      &gt;
      <code>
       本地用户和组
      </code>
      &gt;
      <code>
       用户
      </code>
      ，右击
      <code>
       用户
      </code>
      ，选择
      <code>
       新用户
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1cbbdb8317dbe0ce3fbb410d12228526.png">
       <br/>
       在“新用户”界面添加用户
       <code>
        ftpadmin
       </code>
      </img>
     </img>
    </p>
    <blockquote>
     <p>
      为方便使用，去掉勾选“用户下次登录时须更改密码”，勾选“用户不能更改密码”和“密码永不过期”
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/01df95409e5733fb9a5b345653863412.png">
      <br/>
      按相同方式添加用户
      <code>
       ftpview
      </code>
     </img>
    </p>
    <blockquote>
     <p>
      小提示：点击
      <code>
       创建
      </code>
      添加完一个用户后，Windows系统会立即展示一个全新的“新用户”界面，方便连续添加操作
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de8e9a91c64bcad9db307b0b4426d9da.png">
      <br/>
      用户添加完后，可以点击
      <code>
       系统工具
      </code>
      &gt;
      <code>
       本地用户和组
      </code>
      &gt;
      <code>
       用户
      </code>
      进行查看
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2cb8ab20e0b4c215ceb7ab6d5823cde0.png">
       <br/>
       【可选操作】Windows默认将用户添加到Users组，这里可以根据个人需要，将刚才的两个用户从Users组中删除。方法是选择
       <code>
        系统工具
       </code>
       &gt;
       <code>
        本地用户和组
       </code>
       &gt;
       <code>
        组
       </code>
       &gt;
       <code>
        Users
       </code>
       ，右击
       <code>
        Users
       </code>
       ，点击
       <code>
        属性
       </code>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7e9e3c51e278ca8337d2f94d0056209d.png">
        <br/>
        在“Users属性”界面中，分别选择用户
        <code>
         ftpadmin
        </code>
        、
        <code>
         ftpview
        </code>
        ，点击
        <code>
         删除
        </code>
        ，完成将两个用户从Users组中删除。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c7db00d2f86e36c0eb691c69ac35b956.png"/>
       </img>
      </img>
     </img>
    </p>
    <h5>
     <a id="2__34">
     </a>
     2. 配置用户权限
    </h5>
    <p>
     仍然是在“计算机管理界面”，选择
     <code>
      服务和应用程序
     </code>
     &gt;
     <code>
      Internet Information Service（IIS）管理器
     </code>
     &gt;
     <code>
      网站
     </code>
     ，点击之前自己搭建的FTP服务器，进入FTP服务器配置主页
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/86b0cf4c68151f63a2c5e48801f0671b.png">
      <br/>
      右击
      <code>
       FTP授权规则
      </code>
      ，选择
      <code>
       编辑权限
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/de8b93d6f344391a584e6358982dca3c.png"/>
      <br/>
      选择
      <code>
       安全
      </code>
      &gt;
      <code>
       编辑
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0faa7dea005b4738bb6287bd3cbe7a15.png"/>
      <br/>
      点击
      <code>
       添加
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8ad8657d9cbd0d091beb6a97c964216c.png"/>
      <br/>
      输入
      <code>
       ftpadmin
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/576116c81061135aa4b51485c3fd4714.png"/>
      <br/>
      点击
      <code>
       检查名称
      </code>
      &gt;
      <code>
       确定
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2ecd265f052b5787f1ba32bff5bd1dda.png"/>
      <br/>
      成功给FTP服务器映射的本地目录（这里是
      <code>
       D:\FTP_HOME
      </code>
      ）添加操作用户
      <code>
       ftpadmin
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5f6bc6db64bb3343723e1d593d1faa38.png"/>
      <br/>
      选择
      <code>
       ftpadmin
      </code>
      用户，勾选
      <code>
       完全控制
      </code>
      ，点击
      <code>
       应用
      </code>
      ，配置好
      <code>
       ftpadmin
      </code>
      的可读可写权限
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/341acd39441aa0c9a88e234eeacea0cc.png"/>
      <br/>
      类似
      <code>
       ftpadmin
      </code>
      ，采用同样的步骤添加
      <code>
       ftpview
      </code>
      。由于默认是只有读权限，所以添加
      <code>
       ftpview
      </code>
      后，点击
      <code>
       确定
      </code>
      即可。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0e71e9e395785f0e3dad34eb407f6052.png"/>
      <br/>
      回到
      <code>
       FTP服务器主页
      </code>
      ，双击
      <code>
       FTP授权规则
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9b6114286dc1a39fcf72a7cbfd73af32.png"/>
      <br/>
      <code>
       FTP授权规则
      </code>
      ，默认有一条“授权允许规则”，将其删除掉
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a4a135e956af9acbe60943866aebfd74.png"/>
      <br/>
      右击打开右键菜单，选择
      <code>
       添加允许规则
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4f6c935eeca6f6eee75705a02e097933.png"/>
      <br/>
      在“添加允许授权规则”界面中，按如下图所示配置用户
      <code>
       ftpadmin
      </code>
      ，权限勾选“读取”与“写入”
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3c2a83460709f545087c0ee0e396e00b.png"/>
      <br/>
      类似地，配置用户
      <code>
       ftpview
      </code>
      ，权限仅勾选“读取”
      <br/>
      <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/761205c639ca77ed5bd67995e5199f50.png"/>
      <br/>
      大功告成，FTP用户已配置完成，接下来可愉快地通过FTP客户端工具使用配置好的两个用户访问FTP服务器了。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f48098b5d4cfc609642b383842294bc9.png"/>
     </img>
    </p>
    <h4>
     <a id="_66">
     </a>
     后记
    </h4>
    <ol>
     <li>
      文中提到的读取权限，指浏览、下载这些功能；写入权限，指上传、删除这些功能；
     </li>
     <li>
      配置用户权限时，可以根据实际情况灵活变通，比如添加匿名用户，再比如先将用户添加到一个组，再给组配置权限，等等。
     </li>
    </ol>
