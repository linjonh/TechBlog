    <h3>
     <a id="_11">
     </a>
     题目
    </h3>
    <blockquote>
     <p>
      给定一个数组，其采用如下代码定义：
     </p>
     <pre><code class="prism language-c++">int data[200];
for(i = 0 ; i &lt; 200 ; i ++）data[i] = 4 * i + 6;
</code></pre>
     <p>
      现给定某个数，请你求出它在 data 数组中的位置（下标）。
     </p>
     <p>
      <strong>
       输入描述
      </strong>
      <br/>
      输入一个待查找的整数（该整数一定在数组 data 中）。
     </p>
     <p>
      <strong>
       输出描述
      </strong>
      <br/>
      输出该整数在数组中的指标。
     </p>
     <p>
      <strong>

      </strong>
      <br/>
      示例 1
      <br/>
      输入
     </p>

      262
      <br/>
      输出
     </p>
     <p>

      <br/>

      <br/>

     </p>
     <p>

      <br/>
      输出
     </p>
     <p>
      108
     </p>
     <p>
      <strong>
       运行限制
      </strong>
     </p>
     <ul>
      <li>
       <p>
        最大运行时间：1s
       </p>
      </li>
      <li>
       <p>
        最大运行内存: 128M
       </p>
      </li>
     </ul>
    </blockquote>
    <h3>
     <a id="_52">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;iostream&gt;
#include&lt;vector&gt;
#include&lt;algorithm&gt;
using namespace std;
int main()
{
  int data[200];
  for(int i = 0 ; i &lt; 200 ; i ++)data[i] = 4 * i + 6;
  int n;
  cin&gt;&gt;n;
  cout&lt;&lt;lower_bound(data,data+200,n)-data&lt;&lt;endl;
  return 0;
}
</code></pre>
    <h3>
     <a id="_70">
     </a>
     题解分析
    </h3>
    <p>
     本题可以使用整数二分查找解题，这里采用C++的
     <code>
      lower_bound
     </code>
     库函数，使解题更
     <mark>
      迅速、简洁
     </mark>
     。
    </p>
    <ul>
     <li>
      <code>
       lower_bound(st,ed,x)
      </code>
      库函数需要输入三个参数，前两个参数是
      <mark>
       迭代器
      </mark>
      ，最后一个参数为目标元素。
      <mark>
       其返回第一个大于等于目标元素的迭代器
      </mark>
      。如果没有，就返回最后一个元素的下一个地址。
     </li>
     <li>
      <mark>
       注意
      </mark>
      ：查找区间
      <code>
       [st ,ed)
      </code>
      为左闭右开区间，所以题解中是
      <code>
       [data,data+200)
      </code>
      。
     </li>
     <li>
      该库函数默认在升序序列中使用，如果要在降序序列中使用，则要加上比较函数。
      <code>
       lower_bound(st,ed,x,greater&lt;int&gt;())
      </code>
     </li>
    </ul>

    <h3>
     <a id="_84">
     </a>
     题目
    </h3>
    <blockquote>
     <p>
      <strong>
       题目描述
      </strong>
      <br/>
      给定一个长度为 N的数组 A，请你先从小到大输出它的每个元素，再从大到小输出它的每个元素。
     </p>
     <p>
      <strong>
       输入描述
      </strong>
      <br/>
      第一行包含一个整数 N
     </p>
     <p>
      第二行包含 N 个整数 a1,…,an，表示数组 A 的元素。
     </p>
     <p>
      1≤N≤5×10^5, −10^9≤ai≤ 10^9
     </p>
     <p>
      <strong>
       输出描述
      </strong>
      <br/>
      输出共两行，每行包含 N 个整数，表示答案。
     </p>
     <p>
      <strong>
       输入输出样例
      </strong>
      <br/>
      示例 1
      <br/>
      输入
     </p>
     <p>
      5
      <br/>
      1 3 2 6 5
      <br/>
      输出
     </p>
     <p>
      1 2 3 5 6
      <br/>
      6 5 3 2 1
      <br/>
      <strong>
       运行限制
      </strong>
     </p>
     <ul>
      <li>
       最大运行时间：3s
      </li>
      <li>
       最大运行内存: 128M
      </li>
     </ul>
    </blockquote>
    <h3>
     <a id="_114">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;iostream&gt;
#include&lt;vector&gt;
#include&lt;algorithm&gt;
using namespace std;

{
  vector&lt;int&gt;s;
  int N;
  cin&gt;&gt;N;
  s.resize(N);

    cin&gt;&gt;s[i];
  }
  sort(s.begin(),s.end());
 for(int ele:s){
   cout&lt;&lt;ele&lt;&lt;" ";
 }
 cout&lt;&lt;endl;
  sort(s.begin(),s.end(),greater&lt;int&gt;());

    <h3>
     <a id="_144">
     </a>
     题解分析
    </h3>
    <p>
     本题可以使用数组解题，这里使用了STL中的
     <code>
      vector
     </code>
     容器解题，遍历起来更加
     <mark>
      方便
     </mark>
     。
    </p>
    <ul>
     <li>
      <p>
       题目思路是使用
       <code>
        sort
       </code>
       库函数进行
       <mark>
        升序排序
       </mark>
       ，再
       <mark>
        降序排序
       </mark>
       ，各自遍历输出一遍即可。
      </p>
     </li>
     <li>
      <p>
       <code>
        sort
       </code>
       库函数的参数与
       <code>
        lower_bound
       </code>
       一样为左闭右开区间，加上
       <code>
        greater&lt;int&gt;()
       </code>
       进行降序排序。
      </p>
     </li>
     <li>
      <p>
       当然，这里的降序排序可以使用
       <code>
        reverse
       </code>
       进行反转，代码为
       <code>
        reverse(s.begin(),s.end());
       </code>
       。此时，数据会发生
       <mark>
        反转
       </mark>
       ，再输出即可。
      </p>
     </li>
     <li>
      <p>
       注意：
       <code>
        vector
       </code>
       容器在输入时要先用
       <code>
        resize
       </code>
       分配空间。也可以先录入给临时变量，再用
       <code>
        push_back()
       </code>
       输入到
       <code>
        vector
       </code>
       容器中。
      </p>
      <pre><code class="prism language-c++">for(int i=0;i&lt;N;i++){
    int t;cin&gt;&gt;t;
    s.push_back(t);
}
</code></pre>
     </li>
    </ul>

     代码

    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;

{
  vector&lt;int&gt;s;
  int n;
  cin&gt;&gt;n;
  s.resize(n);
  for(int i=0;i&lt;n;i++){
    cin&gt;&gt;s[i];

     <code>
      bits/stdc++.h

     <code>

    <h3>

    </h3>
    <blockquote>
     <p>
      <strong>
       题目描述
      </strong>
      <br/>
      CLZ 银行只有两个接待窗口，VIP 窗口和普通窗口，VIP 用户进入 VIP 窗口排队，剩下的进入普通窗口排队。现有 M 次操作，操作四种类型，如下：
     </p>
     <ul>
      <li>
       IN name V：表示一名叫 name 的用户到 VIP 窗口排队
      </li>
      <li>
       OUT V：表示 VIP 窗口队头的用户离开排队
      </li>
      <li>
       IN name N：表示一名叫 name 的用户到普通窗口排队
      </li>
      <li>
       OUT N：表示普通窗口队头的用户离开排队
       <br/>
       求M 次操作结束后 VIP 窗口队列和普通窗口队列中的姓名。
      </li>
     </ul>
     <p>
      <strong>
       输入描述
      </strong>
      <br/>
      第一行是一个整数 M（1≤M≤1000），表示一共有 M 次操作。
     </p>
     <p>
      第二行到第 M+1 行输入操作，格式如下：
     </p>
     <p>
      IN name V
     </p>
     <p>
      OUT V
     </p>
     <p>
      IN name N
     </p>
     <p>
      OUT N
     </p>
     <p>
      <strong>
       输出描述
      </strong>
      <br/>
      输出M 次操作后 VIP 窗口队列和普通窗口队列中的姓名（从头到尾），先输出 VIP 窗口队列后输出普通窗口队列。
     </p>
     <p>
      示例 1
     </p>
     <p>
      输入
     </p>
     <p>
      5
      <br/>
      IN xiaoming N
      <br/>
      IN Adel V
      <br/>
      IN laozhao N
      <br/>
      OUT N
      <br/>
      IN CLZ V
     </p>
     <p>
      输出
     </p>
     <p>
      Adel
      <br/>
      CLZ
      <br/>
      laozhao
     </p>
     <p>
      <strong>
       运行限制
      </strong>
     </p>
     <ul>

    <h3>
     <a id="_309">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using namespace std;
int main()
{
  int m;
  cin&gt;&gt;m;
  string a;
  queue&lt;string&gt; v,n;
  for(int i=0;i&lt;m;i++){
    cin&gt;&gt;a;
    if(a=="IN"){
      string name,p;cin&gt;&gt;name&gt;&gt;p;
      if(p=="V")v.push(name);
      else n.push(name);
    }
    else{
      string p;cin&gt;&gt;p;
      if(p=="V")v.pop();
      else n.pop();
    }
  }
  while(v.size()){
    cout&lt;&lt;v.front()&lt;&lt;endl;
    v.pop();
  }
  while(n.size()){
    cout&lt;&lt;n.front()&lt;&lt;endl;
    n.pop();
  }
  return 0;
}
</code></pre>
    <h3>
     <a id="_345">
     </a>
     题解分析
    </h3>
    <p>
     本题情景是
     <mark>
      银行窗口排队
     </mark>
     ，可以想到使用
     <code>
      queue
     </code>
     容器(队列)解决。有两个窗口就使用两个
     <code>
      queue
     </code>
     容器，然后根据题目分情况解题即可。
    </p>
    <ul>
     <li>
      <p>
       <code>
        queue
       </code>
       容器使用
       <code>
        string
       </code>
       类型，进行字符串输入。
      </p>
     </li>
     <li>
      <p>
       <code>
        q.push()
       </code>
       表示添加元素；
       <code>
        q.pop()
       </code>
       表示移除队头元素；
       <code>
        q.front()
       </code>
       表示获取队头元素；
       <code>
        q.size()
       </code>
       表示队列里的元素个数；
      </p>
     </li>
     <li>
      <p>
       队列遍历输出：
      </p>
      <pre><code class="prism language-c++">while(q.size()){
    cout&lt;&lt;q.front()&lt;&lt;endl;
    q.pop();
  }
</code></pre>
     </li>
    </ul>

       我们应该使得每一组内尽可能装两种礼物（最多只能装两件），尽量占满一组的容量
      </mark>
      。
     </li>
     <li>
      先进行升序排序，然后使用对撞指针，对右指针进行判断，如果其与左指针相加或其本身大于
      <code>
       W
      </code>
      ，就独自为一组，否则，两指针为一组。
     </li>
     <li>
      这里的贪心策略为，每一个贵的礼物带一个便宜的成一组。
     </li>
    </ul>

     <p>

    <h3>
     <a id="_1761">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include&lt;bits/stdc++.h&gt;
using namespace std;
const int N=1e5+9;
int a[N],S;

int main(){
  ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
  int n,s;cin&gt;&gt;n&gt;&gt;s;

  for(int i=1;i&lt;=n;i++)cin&gt;&gt;a[i];

  int ans=n+1;

  for(int i=1,j=0,sum=0;i&lt;=n;i++){
    while(i&gt;j||(j+1&lt;=n&amp;&amp;sum&lt;s))sum+=a[++j];

    if(sum&gt;=s)ans=min(ans,j-i+1);
    sum-=a[i];
  }
cout&lt;&lt;(ans&gt;n?0:ans)&lt;&lt;endl;

  return 0;
}

        main
       </code>

       </code>

       <code>
        n
       </code>
       个整数存入数组
       <code>
        a
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       定义变量
       <code>
        ans
       </code>
       初始值为
       <code>
        n + 1
       </code>
       。然后通过两个指针
       <code>
        i

       <code>

       <code>

    <h2>
     <a id="_1861">
     </a>
     题目二十五
    </h2>
    <p>
     原题链接：
     <a href="https://www.lanqiao.cn/problems/3400/learning/?page=1&amp;first_category_id=1&amp;problem_id=3400" rel="nofollow">
      lanqiao3400
     </a>
    </p>
    <h3>
     <a id="_1865">
     </a>
     题目
    </h3>
    <blockquote>
     <p>
      <strong>
       问题描述
      </strong>
      <br/>
      在一个神秘的世界中，存在着一个称为"异或森林"的地方。异或森林中的每个树木都拥有独特的力量。肖恩进入了这片森林，他得到了一个任务：找出数组中满足条件的连续子数组，使得连续子数组中所有元素异或运算结果的因数个数为偶数。完成任务将揭示宝藏的所在地。现在，你能告诉肖恩有多少个连续子数组满足条件吗？
     </p>
     <p>
      注意：0 的因数个数视为奇数。
     </p>
     <p>
      <strong>
       输入描述
      </strong>
      <br/>
      第一行输入一个数字 n 表示数组元素个数。
     </p>
     <p>
      第二行输入 n 个数字，第 i 个数字 a[i] 表示数组的第 i 个元素。
     </p>
     <p>
      <strong>
       数据保证
      </strong>
      <br/>
      1≤n≤10
      <sup>
       4
      </sup>
      1≤a[i]≤n 。
     </p>
     <p>
      <strong>
       输出描述
      </strong>
      <br/>
      输出一个数字表示满足条件的连续子数组的数量。
     </p>
     <p>
      样例输入
      <br/>
      5
      <br/>
      1 2 3 4 5
      <br/>
      样例输出
      <br/>
      7
     </p>
    </blockquote>
    <h3>
     <a id="_1889">
     </a>
     代码
    </h3>
    <pre><code class="prism language-c++">#include &lt;bits/stdc++.h&gt;
using namespace std;
const int N=1e5+9;
int a[N],prexor[N],cnt[N];

int main()
{
 int n;cin&gt;&gt;n;
 for(int i=1;i&lt;=n;i++)cin&gt;&gt;a[i];
 for(int i=1;i&lt;=n;i++)prexor[i]=prexor[i-1]^a[i];
 cnt[0]=1;
 int ans=n*(n+1)/2;
 for(int i=1;i&lt;=n;i++){
   for(int j=0;j&lt;=200;j++){
     int sq=j*j;
     ans-=cnt[prexor[i]^sq];
   }
   cnt[prexor[i]]++;
 }
 cout&lt;&lt;ans&lt;&lt;'\n';
  return 0;
}
</code></pre>
    <h3>
     <a id="_1916">
     </a>
     题解分析
    </h3>
    <ul>
     <li>
      <p>
       <mark>
        首先转化条件
       </mark>
       ：因数个数为偶数个=&gt;不是完全平方数。
      </p>
     </li>
     <li>
      <p>
       <mark>
        正难则反
       </mark>
       ：计算有多少个子数组的异或和为完全平方数，用总区间个数减去即可。
      </p>
     </li>
     <li>
      <p>
       先求前缀异或和
       <code>
        prexor
       </code>
       ，枚举所有平方数
       <code>
        sq
       </code>
       （不超过200个)，然后对于
       <code>
        prexor[i]
       </code>
       ，要有
       <code>
        prexor[i] ^ prexor[i]= sq
       </code>
       ，于是就是统计多少个
       <code>
        i
       </code>
       满足
       <code>
        j&lt;i
       </code>
       且,
       <code>
        prexor[j]=prexor[i] ^ sq
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       总区间个数为
       <code>
        n* (n + 1)/ 2
       </code>
       ，且元素异或和最大值不超过
       <code>
        2e4
       </code>
       。
      </p>
     </li>
    </ul>
