    <h2>
     <a id="StableDiffusionWebui_1">
     </a>
     一、环境搭建及Stable-Diffusion-Webui
    </h2>
    <h3>
     <a id="_2">
     </a>
     系统配置
    </h3>
    <p>
     学校的云计算平台（Linux系统）（CPU 1核，GPU 0.5卡，内存 20G）、Xftp7
    </p>
    <h3>
     <a id="_4">
     </a>
     基础环境配置
    </h3>
    <p>
     配置python3.10环境，
     <code>
      &lt;env_name&gt;
     </code>
     为设定的环境名称
    </p>
    <pre><code class="prism language-bash">conda create -n <span class="token operator">&lt;</span>env_name<span class="token operator">&gt;</span> <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.10</span> ipykernel
</code></pre>
    <p>
     激活环境
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> activate <span class="token operator">&lt;</span>env_name<span class="token operator">&gt;</span>
</code></pre>
    <p>
     将环境写入Notebook的Kernel里
    </p>
    <pre><code class="prism language-bash">python -m ipykernel <span class="token function">install</span> --user --name <span class="token operator">&lt;</span>env_name<span class="token operator">&gt;</span> --display-name <span class="token string">"&lt;env_name&gt;"</span>
</code></pre>
    <p>
     重启实例，可以看到自己创建的py3.10环境，打开Terminal，安装git
    </p>
    <pre><code class="prism language-bash">conda <span class="token function">install</span> <span class="token function">git</span>
</code></pre>
    <h3>
     <a id="Stable_Diffusion_22">
     </a>
     正式部署Stable Diffusion
    </h3>
    <p>
     激活环境
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">source</span> activate <span class="token operator">&lt;</span>env_name<span class="token operator">&gt;</span>
</code></pre>
    <p>
     确认git版本，最好是2.XX
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fc6d4fa46022deec3768c67df1ec6546.png">
      <br/>
      下载代码
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
</code></pre>
    <p>
     进入目录
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> stable-diffusion-webui
</code></pre>
    <p>
     安装依赖
    </p>
    <pre><code class="prism language-bash">pip config <span class="token builtin class-name">set</span> global.index-url https://mirrors.aliyun.com/pypi/simple/ 
pip <span class="token function">install</span> -r requirements_versions.txt 
</code></pre>
    <p>
     打开
     <code>
      stable-diffusion-webui/launch.py
     </code>
     ，找到
     <code>
      prepare_
     </code>
     函数，把里面所有的“
     <code>
      https://github.com
     </code>
     ”替换为“
     <code>
      https://ghproxy.com/https://github.com
     </code>
     ”，利用Ghproxy进行代理，加速国内Git
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fb6e3dd701330c1ce8a0dd1ae3d8ecf4.png">
      <br/>
      打开目录下的
      <code>
       webui-user.bat
      </code>
      ，在图示位置增加
      <code>
       --xformers
      </code>
      ，虽然对于我遇到的
      <code>
       --xformers Cannot import xformers
      </code>
      报错好像没什么用，但谁知道对其他朋友有没有用呢？
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/784c4912cf4c964b6c1e8fa77209b403.png">
       <br/>
       启动程序
      </img>
     </img>
    </p>
    <pre><code class="prism language-bash">./webui.sh
</code></pre>
    <p>
     遇到报错尝试根据Command，直接利用git下载，比如：
    </p>
    <pre><code class="prism language-bash"><span class="token function">git</span> clone https://xx.git
</code></pre>
    <p>
     下载完之后继续重复上述操作
    </p>
    <p>
     如果提示没有主模型checkpoint，去
     <a href="https://huggingface.co/CompVis/stable-diffusion-v-1-4-original" rel="nofollow">
      官网
     </a>
     下载主模型
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d071d306fcdc262d485148625e52783e.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ba0bded994d4ba75bc261ecbb2afb183.png">
       <br/>
       sd-v1-4为半精度，大概4G，sd-v1-4-full-ema为单精度，大概7G，我是先下载到本地，再通过xftp连接到云平台进行上传到下面这个位置：
      </img>
     </img>
    </p>
    <pre><code class="prism language-bash">/home/XXXX/stable-diffusion-webui/models/Stable-diffusion/
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/26cf7a413fec1e15f088407e41c49375.png">
      <br/>
      我装第二次的时候
      <s>
       （谁知道我个憨憨为什么不小心把文件夹删了还找不回来呢）
      </s>
      ，非常顺利，没有提示open_clip的问题，还自动给我安装了v1-5的主模型（咱就是说很玄学）
      <br/>
      至于xformers说没安装的问题，其实也不影响运行，只是个警告⚠而已。。。
     </img>
    </p>
    <p>
     open_clip的问题，我也找到解决方法，可供参考👇
     <br/>
     直接去github把
     <a href="https://gitcode.net/mirrors/mlfoundations/open_clip/-/tree/main/src" rel="nofollow">
      open_clip
     </a>
     下载到stable-diffusion-webui/目录下面，注意是
     <code>
      /src/open_clip
     </code>
     这里的
     <code>
      open_clip
     </code>
    </p>
    <p>
     等到显示了一个
     <code>
      local URL
     </code>
     之后，输入
     <code>
      ./webui.sh --share
     </code>
     再跑一下，就能有个
     <code>
      public URL
     </code>
     ，粘贴到浏览器就可以使用了，手机电脑都能运行（只要云平台不关机）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e226056fb5b23c80489a7a55dbde56cd.png"/>
    </p>
    <h2>
     <a id="StableDiffusionWebui_73">
     </a>
     二、Stable-Diffusion-Webui使用手册
    </h2>
    <ul>
     <li>
      <p>
       模型调整（改变风格、优化人体形态等等），网上有很多公开的ckpt文件，把它们传到上面主模型位置处就行，使用的时候，通过切换Stable
       <br/>
       Diffusion checkpoint即可更换模型
      </p>
     </li>
     <li>
      <p>
       prompt提示词参考网站：
       <a href="https://lexica.art/" rel="nofollow">
        https://lexica.art/
       </a>
      </p>
     </li>
     <li>
      <p>
       推荐大佬博客《
       <a href="https://zhuanlan.zhihu.com/p/570954565" rel="nofollow">
        模型应用—使用Stable Diffusion UI手册
       </a>
       》
      </p>
     </li>
     <li>
      <p>
       推荐大佬博客《
       <a href="https://zhuanlan.zhihu.com/p/574200991" rel="nofollow">
        使用stable-diffusion-webui部署NovelAi/Stable Diffusion 保姆级教程、命令解释、原理讲解(colab、windows、Linux )
       </a>
       》
      </p>
     </li>
     <li>
      <p>
       界面使用（目前我探索的还比较有限，会进一步跟进理论知识学习）
       <br/>
       txt2img — 标准的文字生成图像，具体参考下图所示；
       <br/>
       img2img — 根据图像成文范本、结合文字生成图像，增加要模仿的图像，参数设置输出图片尺寸、与原图一致性程度
       <br/>
       Extras — 优化(清晰、扩展)图像；
       <br/>
       PNG Info — 图像基本信息
       <br/>
       Checkpoint Merger — 模型合并
       <br/>
       Textual inversion — 训练模型对于某种图像风格
       <br/>
       Settings — 默认参数修改
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f1151273a4b20868a8bf173c1f2a3ca3.png">
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e537615dc8e22ded7cfe8157eb378252.png"/>
       </img>
      </p>
     </li>
    </ul>
