    <h2>
     <a id="1_1">
     </a>
     1.电机控制
    </h2>
    <p>
     <strong>
      电机除了转，就只能转了，但它简单的现象背后却是一个典型的
      <font color="red">
       非线性、强耦合、多变量、周期性
      </font>
      的机电能量转换的复杂系统。
     </strong>
    </p>
    <p>
     <strong>
      电机控制无非就是达到三个控制目的：
      <br/>
      ①转矩控制
      <br/>
      ②速度控制
      <br/>
      ③位置控制
     </strong>
     <br/>
     <strong>
      <font color="red">
       其中转矩控制是核心，因为矢量控制的核心目标就是为了让电机以稳定的转矩旋转，在这个基础上再实现速度环、位置环。举例子他们三环内外放置时的关系：在电流-速度环中，不同的电流电机内部就产生不同情况的磁场，面对不同负载，电流大，磁场强，转速快。在电流-速度-位置环中，位置是速度在时间上的积分，我们希望以稳定的速度走一段位置，那么对应速度，也对应着某一种电流情况。显然，电流环是最内环(重要)的
      </font>
     </strong>
     <br/>
     以三相无刷电机举例，它的三相对称正弦电流可以这样表示：
     <strong>
      <font color="red">
       <br/>
       Ia=Im * cos(ωt+φ)
       <br/>
       Ib=Im * cos(ωt+φ-120°)
       <br/>
       Ic=Im * cos(ωt+φ+120°)
      </font>
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e87a630023512bf7901102537310e319.gif#pic_center"/>
    </p>
    <h3>
     <a id="11__17">
     </a>
     1.1 方法论
    </h3>
    <p>
     时域上直接建立的电机转矩动态数学模型，是一组含有时变系数的非线性微分方程，难以直接分析和求解，这方面随便拿一本电机类的书籍都有介绍。为了便于控制，需要简化数学模型，简化的基本方法是坐标变换。
     <br/>
     所谓坐标变换就是将方程中的一组变量用一组新的变量来代替，或者说用新的坐标系去替换原来的坐标系，以便使分析、计算得以简化。若新、旧变量之间为线性关系，则变换为线性变换，电机分析中常用到的坐标变换都是线性变换。
     <br/>
     目的很明确，
     <strong>
      强耦合系统解耦，非线性系统线性化处理。
     </strong>
     <br/>
     我们希望三相电机像直流电机那样简单易控。
     <br/>
     <font color="red">
      <strong>
       克拉克Clarke变换帮我们将三相电机等效成两相电机；
       <br/>
       帕克Park变换帮我们将电流解耦，使我们控制交流电流像控制直流那样简便。
      </strong>
     </font>
    </p>
    <h2>
     <a id="2Clarke3s2s_24">
     </a>
     2.克拉克Clarke变换(3s/2s)
    </h2>
    <h3>
     <a id="21__25">
     </a>
     2.1 变换思想
    </h3>
    <p>
     克拉克变换将三相系统（在 abc 坐标系中）的时域分量转换为正交静止坐标系 (αβ) 中的两个分量。我们要把三相电机等效成两相电机，方便后续分析。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/52ae4c670af0498cb5ac9fdf296a8f3e.png#pic_center"/>
    </p>
    <p>
     设
     <strong>
      Is
     </strong>
     为三相对称正弦电流的合成矢量。
     <br/>
     <strong>
      Is
     </strong>
     =
     <strong>
      Ia
     </strong>
     +
     <strong>
      Ib
     </strong>
     +
     <strong>
      Ic
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/53f07fd9e7b3466892cac3b52d50ba3c.png#pic_center"/>
    </p>
    <p>
     <font color="red">
      <strong>
       Clarke变换旨在利用向量分解，将三相坐标合成为两相坐标
      </strong>
     </font>
     <br/>
     根据投影，我们得到：
     <strong>
      <font color="#0099ff">
       <br/>
       i_alpha=Ia-Ib * cos(π/3)-Ic * cos(π/3)
       <br/>
       i_beta=Ib * cos(π/6)-Ic * cos(π/6)
      </font>
     </strong>
     <br/>
     用矩阵可以表示:
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f4cb9c0f1f4b4df5aa578c6ee5e1d5a6.png#pic_center"/>
    </p>
    <h3>
     <a id="21_k1_39">
     </a>
     2.1 原始变换(k=1)
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/be93435669050018312fa6cc9d168b51.jpeg#pic_center"/>
    </p>
    <p>
     根据三相电机简化模型，由电路基尔霍夫定律：
     <br/>
     <strong>
      <font color="red">
       Ia+Ib+Ic=0
      </font>
     </strong>
     <br/>
     展开后易得：
     <strong>
      <font color="#0099ff">
       <br/>
       i_alpha=3/2 * Ia
       <br/>
       i_beta=√3/2 * Ia + √3Ib
      </font>
     </strong>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0bf41ae8d40f38bba33d52ac7aad0267.gif#pic_center"/>
    </p>
    <p>
     <font color="red">
      <strong>
       由动态图我们可以看到，通过克拉克变换，我们成功地将三相电机等效为了两相电机，这就是克拉克变换的意义。
       <br/>
       通过类比，试想：部分步进电机就是两相电机，无需克拉克变换，直接使用两相的数据代入帕克变换中直接解耦，是否也能做FOC控制？
      </strong>
     </font>
    </p>
    <h3>
     <a id="22_k23_52">
     </a>
     2.2 幅值不变原则的变换(k=2/3)
    </h3>
    <p>
     由
     <strong>
      Is
     </strong>
     易知：
     <br/>
     <strong>
      Is
     </strong>
     =
     <strong>
      Ia
     </strong>
     +
     <strong>
      Ib
     </strong>
     +
     <strong>
      Ic
     </strong>
     =
     <strong>
      i_alpha
     </strong>
     +
     <strong>
      i_beta
     </strong>
     <br/>
     <font color="#0099ff">
      <strong>
       故i_alpha+i_beta的模值依然是3/2 * Im
      </strong>
     </font>
     <br/>
     若我们希望合成前后的模值不变，即依然是Im,那我们就在前面乘以一个系数k，对其缩减，显而易见按，3/2k=1
     <br/>
     那么
     <font color="red">
      <strong>
       k1=2/3
      </strong>
     </font>
     <br/>
     代入易得：
     <strong>
      <font color="#0099ff">
       <br/>
       i_alpha= Ia
       <br/>
       i_beta=1/√3 * Ia + 2/√3Ib
      </font>
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cc132f69e84f4c0f96804246fc066225.png#pic_center"/>
    </p>
    <p>
     <strong>
      为什么要幅值不变？
     </strong>
     <br/>
     电压和电流解耦用的克拉克变换和帕克变换方法必须是同一套，
     <font color="red">
      <strong>
       不能
      </strong>
     </font>
     电流用这套克拉克变换和帕克变换，而电压用另一种。
     <br/>
     仅仅是电流的话，电机正常运行时幅值上没有讲究；但电压就不一样，受到直流母线电压Vdc限制。这样做：
     <br/>
     <font color="red">
      <strong>
       是为了后续SVPWM调制时，确保输出电压在正六边形内接圆里(半径为Um,类比上述推导中的电流Im)，使电压输出波形最大不失真。
      </strong>
     </font>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f8ad58f875b14d37bde3f296e4f179f4.png#pic_center"/>
    </p>
    <h3>
     <a id="23_k23_71">
     </a>
     2.3 功率不变原则的变换(k=√(2/3))
    </h3>
    <p>
     &lt;1&gt;变换前
     <br/>
     P0=Um * Im * 3（三相）
     <br/>
     &lt;2&gt;变换后：
     <br/>
     U=3/2 * k * Um
     <br/>
     I=3/2 * k * Im
     <br/>
     P=UI * 2（两相）
     <br/>
     令P0=P:
     <br/>
     解有等功率变换系数
     <strong>
      <font color="red">
       k2=√(2/3)
      </font>
     </strong>
     <br/>
     代入易得：
     <strong>
      <font color="#0099ff">
       <br/>
       i_alpha= √6/2 * Ia
       <br/>
       i_beta=√2/2 * Ia + √2Ib
      </font>
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/930c8ce525dc4174b94fd640096d840e.png#pic_center"/>
     <br/>
     <font color="red">
      <strong>
       显然等功率变换时K比幅值不变变换情况下更小了，所以电压必定会被限制在电压圆内，这时等功率法下整定的K合理。
      </strong>
     </font>
    </p>
    <h3>
     <a id="24__86">
     </a>
     2.4 它的反变换
    </h3>
    <p>
     通过2.2或2.3的结论可逆推得到。
    </p>
    <h2>
     <a id="3Park2s2r_88">
     </a>
     3.帕克Park变换（2s/2r）
    </h2>
    <h3>
     <a id="31__89">
     </a>
     3.1 变换思想
    </h3>
    <p>
     即使我们已经通过克拉克变化将三相变成了两相，但是从i_alpha和i_beta的展开式中，依旧含有时间t，电流依旧由时间决定，分析起来还是很复杂。
     <br/>
     帕克变换中，在克拉克变换的基础上，试图将表达式中的时间t消去，就是帕克变换的核心思想和目的。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f17b981b3f68bef4b9d6841cf66b527d.gif#pic_center"/>
    </p>
    <p>
     如图，旋转坐标轴以ω的角速度旋转。
    </p>
    <h3>
     <a id="32_95">
     </a>
     3.2.变换公式
    </h3>
    <p>
     根据投影分析，我们得到：
     <strong>
      <font color="#0099ff">
       <br/>
       i_d=i_alpha * cosθ+ i_beta * sinθ
       <br/>
       i_q=-i_alpha * sinθ+i_beta * cosθ
      </font>
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/001762da89d0c22185dd49c7d176746d.png#pic_center"/>
    </p>
    <h3>
     <a id="33__102">
     </a>
     3.3 变换推导
    </h3>
    <p>
     <strong>
      这里使用等幅值原则的结论进行推导。
     </strong>
     <br/>
     我们尝试进一步简化：
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/779903cb1f4bfdfd989170a5dcd4c407.jpeg#pic_center"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dfbc3d0b5c411726f9edced8eb8e0da8.jpeg#pic_center"/>
    </p>
    <p>
     <strong>
      从结论知，只要φ不变，id与iq在时域上，必是直线。
     </strong>
     <br/>
     因为iq垂直与定子磁场，所以力矩只由iq决定；
     <br/>
     而id与定子磁场平行，不作用于力矩控制。
    </p>
    <h3>
     <a id="34__113">
     </a>
     3.4 它的反变换
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/885457f9a05cbc790ea5534d4ce693c1.jpeg#pic_center"/>
    </p>
    <p>
     <strong>
      <font color="#0099ff">
       <br/>
       v_alpha =v_d * cosθ-v_q * sinθ
       <br/>
       v_beta=v_d * sinθ+v_q * cosθ
      </font>
     </strong>
    </p>
    <h2>
     <a id="4_119">
     </a>
     4.坐标变换结论
    </h2>
    <h3>
     <a id="41__120">
     </a>
     4.1 两种变换的总结
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d15d82532a9c40010c05256149d3da2d.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="42__124">
     </a>
     4.2 机械角度与电角度
    </h3>
    <p>
     设p是电机极数
     <br/>
     则P=0.5p
     <br/>
     P是电机的极对数
     <br/>
     <strong>
      <font color="red">
       电角度=P * 机械角度
       <br/>
       我们代入公式中计算的，用的都是电角度
      </font>
     </strong>
     <br/>
     我们Ia，Ib与Ic中的ωt+φ，ωt+φ-120°，ωt+φ+120°指的都是电角度。
    </p>
    <h3>
     <a id="43_tP_131">
     </a>
     4.3 ωt，θ，φ，和P的关系
    </h3>
    <p>

     </font>

     <br/>

      </font>
     </strong>
     <br/>
     ωt，θ，
     <font color="red">
      φ
     </font>
     三者都是电角度
     <br/>
     ωt=θ，是两相旋转坐标轴与两相静止坐标轴的夹角(电角度θe)。
     <br/>
     <font color="red">
      φ
     </font>
     就有意思了，它是定子磁场和转子磁场的角度。

     <br/>
     <strong>
      <font color="red">
       当定子磁场和转子磁场的角度为90°，即φ=90°时，转矩最大
      </font>
     </strong>
    </p>
