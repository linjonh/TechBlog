    <h1 class="title-article" id="articleContentId">
     在windows10系统上搭建npm仓库源
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      安装Verdaccio
     </strong>
     ：
    </p>
    <h4>
     npm install -g verdaccio
    </h4>
    <p>
     启动verdaccio
    </p>
    <p>
     <code>
      <code>
       verdaccio
      </code>
     </code>
    </p>
    <p>
     打开
     <code>
      config.yaml 路径在第一行
     </code>
    </p>
    <p>
     config file  - C:\Users\huyun\AppData\Roaming\verdaccio\config.yaml
    </p>
    <p>
     最下面添加一行
    </p>
    <p>
     <code>
      listen: 0.0.0.0:4873
     </code>
    </p>
    <p>
     另外我们的国内的镜像源一般是使用淘宝镜像去下载东西的，那么可以把我们的镜像源的来源设置为淘宝的镜像源，这样可以提高国内的下载速度。做法也是修改这个配置文件，找到
     <code>
      uplinks
     </code>
     下面两行把url替换一下，具体的如下：
    </p>
    <pre># a list of other known repositories we can talk to
uplinks:
  npmjs:
    url: https://registry.npm.taobao.org/
    # 最好禁用上游仓库,若不想禁用,不用复制下面的配置,禁用了之后发布到本地仓库会很快
    enabled: false</pre>
    <p>
     配置好后就大功告成了，把配置文件保存一下，然后重启
     <code>
      verdaccio
     </code>
     （命令行中
     <code>
      Ctrl + C
     </code>
     ，然后重新运行
     <code>
      verdaccio
     </code>
     命令）。
    </p>
    <p>
     此时可以在浏览器键入
     <code>
      http://你的IP:4873/
     </code>
     或者
     <code>
      http://localhost:4873/
     </code>
     来访问管理页面。
    </p>
    <p>
     切换到自己的镜像源很简单，只要一行代码就搞定：
    </p>
    <pre>npm config set registry http://你的IP:4873/</pre>
    <h3>
     上传一个依赖包
    </h3>
    <p>
     说了这么多我们现在可以开发一个自己的包，然后上传到我们的镜像源上。 首先切换到我们自己的源上，然后添加用户（我们镜像源上的用户不是npm镜像源中的用户）。添加用户使用下面命令，然后依次输入用户名、密码、邮箱。输入密码的时候是看不到字符的，尽管输入就可以了。当操作完了会输出一行日志
     <code>
      Logged in as orange on http://你的IP:4873/.
     </code>
     表示创建用户并登录成功了。
    </p>
    <pre>npm adduser --registry http://localhost:4873/</pre>
    <p>
     此时你也可以查看你是否登录成功了
    </p>
    <pre>npm whoami</pre>
    <p>
     我这里执行后,会显示我的用户名
    </p>
    <p>
     root
    </p>
    <p>
     此时你可以创建一个依赖包，然后上传上去了。我们来定义一个依赖包的名称，比如就叫
     <code>
      my-package
     </code>
     ，找一个目录输入下面命令：
    </p>
    <pre>mkdir my-package</pre>
    <pre>cd my-package</pre>
    <pre>npm init</pre>
    <p>
     <code>
      npm init
     </code>
     后会让你输入其他的配置项，比如包名是
     <code>
      my-package
     </code>
     ，其他的直接按回车，最后输入一个
     <code>
      yes
     </code>
     就可以了。此时你会看到目录下多了一个文件夹叫
     <code>
      my-package
     </code>
     里面有个文件
     <code>
      package.json
     </code>
     。
    </p>
    <p>
     接下来我们在
     <code>
      my-package
     </code>
     文件夹下，创建一个
     <code>
      index.js
     </code>
     的文件。输入下面的代码：
    </p>
    <pre><code class="hljs">module.exports = function (){
  console.log(`Hello World!`);
}</code></pre>
    <p>
     此时我们已经创建好了一个非常简单的包，然后我们可以上传了，使用shell/cmd进入我们的
     <code>
      my-package
     </code>
     目录，然后输入命令：
    </p>
    <p>
     npm publish --registry http://localhost:4873/
    </p>
    <p>
     此时如果看到
     <code>
      + my-package@1.0.0
     </code>
     说明已经上传成功了，你可以用浏览器打开
     <code>
      http://你的IP:4873/
     </code>
     你会发现确实多了一个包。
    </p>
    <p>
     如果你在之后上传的时候一定要修改
     <code>
      package.json
     </code>
     的
     <code>
      version
     </code>
     字段，并且要大于现在，否则不能上传。也可以使用命令
     <code>
      npm version 1.0.1
     </code>
     直接修改版本号。
    </p>
    <h3>
     使用开发好的依赖包
    </h3>
    <p>
     现在我们使用我们的依赖包，首先在与
     <code>
      my-package
     </code>
     同级目录下再建一个文件夹，比如叫
     <code>
      use-package
     </code>
     。同样使用
     <code>
      npm init
     </code>
     去初始化我们的项目。并且创建一个
     <code>
      index.js
     </code>
     的文件。
    </p>
    <p>
     此时你可以安装我们刚才发布的那个包了：
    </p>
    <pre><code>npm install my-package</code></pre>
    <p>
     并且在
     <code>
      index.js
     </code>
     中输入以下代码：
    </p>
    <pre><code>const helloWorld = require('my-package')
helloWorld()</code></pre>
    <p>
     此时大功告成，在shell/cmd中打开
     <code>
      use-package
     </code>
     文件夹，然后输入命令
     <code>
      node index
     </code>
     ，如果看到
     <code>
      Hello World!
     </code>
     说明我们上传的依赖已经可以使用了。
    </p>
    <h3>
     上传npm包到自己搭建的仓库源里
    </h3>
    <p>
     从其他npm源下载包,例如下载webpack:
    </p>
    <pre><code>npm pack webpack --registry=https://registry.npm.taobao.org</code></pre>
    <p>
     上传下载好的webpack包到自己本地搭建的npm仓库:
    </p>
    <p>
     npm publish webpack-5.98.0.tgz --registry=http://localhost:4873
    </p>
    <p>
     如果发布不上去，是包的问题，可以用上面例子测试发布.tgz包
    </p>
    <p>
     npm pack .\my-package\
    </p>
    <p>
     npm unpublish my-package --force
    </p>
    <p>
     npm publish my-package-1.0.0.tgz --registry=http://localhost:4873
    </p>
