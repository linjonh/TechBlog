# Golang开发工具

## 1. Golang开发工具

### 1.1. JSON-to-Go

JSON-to-Go 是一个将 json 数据转换为 Go 结构体的在线服务。

地址：
<https://mholt.github.io/json-to-go/>

### 1.2. TOML-to-Go

TOML-to-Go 是一个将 toml 数据转换为 Go 结构体的在线服务。

地址：
<https://xuri.me/toml-to-go/>

### 1.3. YAML-to-Go

TOML-to-Go 是一个将 yaml 数据转换为 Go 结构体的在线服务。

地址：
[https://zhwt.github.io/yaml-to-go/](https://xuri.me/toml-to-go/)

### 1.4. curl-to-Go

curl-to-Go 是一个将 curl 请求命令和数据格式转换为 Go 相关代码的在线服务。

地址：
[https://mholt.github.io/curl-to-go/](https://xuri.me/toml-to-go/)

### 1.5. sql2struct

sql2struct 是一款根据 sql 语句自动生成 Go 结构体的 chrome 插件。

地址：
<https://github.com/idoubi/sql2struct>
