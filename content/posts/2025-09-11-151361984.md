# 【译】Visual Studio 八月更新已发布 —— 更智能的人工智能、更出色的调试功能以及更多控制权



Visual Studio 2022 （v17.14）2025年8月更新现已发布，此次更新旨在帮助开发者保持专注、提高效率并掌控全局。无论您是在开发游戏、优化性能还是探索人工智能，这个版本都带来了切实的改进，让日常开发更加流畅和智能。

![1](https://i-blog.csdnimg.cn/img_convert/ab3e7e57f89ef3e4d261aadba7be1beb.png)

## 现已支持 GPT-5

　　我们很高兴地宣布，GPT-5 现已在 Visual Studio 中推出，将最新的人工智能进展直接带到您的开发环境中。通过 GPT-5 的集成，您可以利用更强大、更准确且具有上下文感知能力的代码建议和聊天体验。无论您是在编写复杂算法、重构大型代码库，还是构思新功能，GPT-5 都能帮助您更快速、更自信地开展工作。

![2](https://i-blog.csdnimg.cn/img_convert/48223fd809fa5aa8215919746cc93b0e.png)

## 通过 MCP 连接您的整个技术栈

　　我们很高兴地宣布，MCP 支持现已在 Visual Studio 中全面可用。 MCP 是一种强大的协议，它能将人工智能 Agent 连接到外部工具（如数据库、代码搜索和部署系统），而无需为每个工具进行自定义集成。可以把它看作是工具连接领域的 HTTP。

　　通过此次发布，Visual Studio 比以往任何时候都更便于管理和连接到  MCP 服务器：

**- 对任何提供商的 OAuth 支持**：使用新的授权规范，直接从 Visual Studio 通过任何 OAuth 提供商进行身份验证。

**- 一键式从网页安装服务器**：从受支持的存储库中一键添加 MCP 服务器，无需再手动编辑 JSON。

**- 新的服务器添加流程**：使用 Copilot Chat 工具选择器中的绿色加号按钮，快速配置并连接到新服务器。

**- 治理控制**：组织现在可以通过 GitHub 策略设置管理 MCP 访问，以实现更好的合规性和控制力。

![3](https://i-blog.csdnimg.cn/img_convert/a6daba62cd2d959391837f1d72e51c66.png)

　　此次更新让 MCP 成为 Visual Studio 中的一流体验，助力团队在其整个工程技术栈中获取更丰富的实时上下文信息。

## 更智能的 Copilot Chat，具备更优的上下文

　　Copilot Chat 现在能够更可靠地呈现相关的代码片段。它如今运用改进后的语义代码搜索，能更好地识别何时应根据查询触发代码查找。当检测到该上下文时，它会在您的解决方案或工作区中进行搜索，以检索最相关的代码片段，即便通过自然语言描述也能实现，从而减少了手动浏览代码库的需求。

![4](https://i-blog.csdnimg.cn/img_convert/2bc2982191cbbcd0a018803c86ebb7fb.png)

## 使用谷歌账号注册 Copilot

　　开始使用 Copilot 如今比以往任何时候都更加简单。您可以直接在 Visual Studio 中使用谷歌账号注册。这是一种快速、顺畅的方式，能让您立即开始使用人工智能驱动的编程功能 —— 无需额外设置。

![5](https://i-blog.csdnimg.cn/img_convert/0f88ba3095f1b620edf1c7e769ae4af8.png)

## 将您自己的 AI 模型带入 Chat

　　想要对自己的人工智能体验有更多控制权吗？现在，您可以使用来自 OpenAI、谷歌或 Anthropic 的 API 密钥，将自己的语言模型连接到 Visual Studio 聊天功能。这让您能够灵活选择最适合自己工作流程的模型，无论您是在追求性能优化、隐私保障还是进行实验探索。

![6](https://i-blog.csdnimg.cn/img_convert/1c69cf9dcd110749e93c5ecfb9fbc01a.png)

## 虚幻引擎的统一调试

　　如果您正在使用虚幻引擎进行 C++ 开发，调试功能刚刚得到了重大升级。Visual Studio 现在允许您在一个会话中同时调试蓝图和原生代码。您会在调用栈和本地窗口中看到蓝图数据，甚至可以直接在蓝图代码中设置断点。

![7](https://i-blog.csdnimg.cn/img_convert/5347b1da8ef90c11df59e9dcce5cc6bc.png)

　　这使得追踪交互以及修复两个脚本层中的问题变得更加容易。

## 按需提供的 Copilot 建议

　　更喜欢安静的编辑器环境？现在您可以关闭 Copilot 的自动建议功能，通过键盘快捷键手动触发建议。这样您就能完全掌控建议出现的时机，在需要专注时保持专注，在想要帮助时获得帮助。

## 通过折叠建议实现更简洁的编辑

　　下一步编辑建议（NES）现在默认隐藏。它们不再自动弹出，而是在相关时以不显眼的页边缘指示器形式显示。您可以决定何时启用，从而保持编辑器整洁且不受干扰。

![8](https://i-blog.csdnimg.cn/img_convert/80d5512a509fa41a36e05862b93c5a6f.png)

## 部分接受代码补全内容

　　您是否曾经希望只接受 Copilot 代码补全中的前几个词或几行，而不是接受整个补全内容？我们很高兴地宣布，现在您可以逐词或逐行部分接受补全内容了！

## Copilot Chat 中的 Git 上下文

　　Copilot Chat 现在能够理解您的 Git 历史记录。您可以在聊天中直接引用提交记录和未提交的更改，来总结工作、解释更新或生成测试 —— 所有这些都无需离开编辑器。

![9](https://i-blog.csdnimg.cn/img_convert/f005e9a07531289b17c9e4a9467b41ac.png)

　　这使得在检查或优化代码时更容易保持心流状态。

## 基于您的反馈构建

　　此版本中的许多功能和修复直接来自开发者社区。您的建议和错误报告持续影响着 Visual Studio 的发展，我们非常感谢您的投入。您可以在发行说明中查看所有由社区驱动的更新和修复。

　　再次感谢您的反馈和支持。我们很期待继续与您一同打造 Visual Studio —— 一次一个更新。请告诉我们您的想法，一如既往，祝您编码愉快！
