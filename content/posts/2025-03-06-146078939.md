# React基础之插值

React中使用使用插值时

添加花括号与不使用花括号时有区别的

使用花括号时，插入的是一个js表达式

const

divContent

=

"

标签内部

"

const

divTitle

=

'

标签标题

'

function

App

() {

return

(

<

div

title

=

{

divTitle

}

>

{

divContent

}

</

div

>

);

}

export default

App

;
