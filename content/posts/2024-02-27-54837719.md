     <span style="white-space:pre;">
     </span>
     最近在开发一个同步数据的统计任务的时候遇到一个问题：
     <strong>
      要在Java代码中判断数据库中某张表是否存在
     </strong>
     ，查资料后，总结了以下两种方法：
    </p>
    <p>
    </p>
    <div>
     <strong>
      1、使用
      <span style="font-family:Consolas;">
       <span style="font-size:11pt;">
        JdbcTemplate bean
       </span>
      </span>
     </strong>
    </div>
    <div align="left" style="min-height:13pt;">
     <pre><code class="language-java">public boolean validateTableNameExist(String tableName) {
           int tableNum = jdbcTemplate.queryForInt("SELECT COUNT(*) FROM ALL_TABLES WHERE TABLE_NAME=" + tableName);
           if (tableNum &gt; 0) {
                 return true;
           }else {
                 return false;
           }
     }</code></pre>
    </div>
    <div align="left" style="min-height:13pt;">
     <strong>
      <span style="font-size:15px;">
       <span style="font-family:Consolas;">
        2、使用Connection对象
       </span>
      </span>
     </strong>
    </div>
    <div align="left" style="min-height:13pt;">
     <pre><code class="language-java">public boolean validateTableNameExist(String tableName) {
         Connection con = getYourCnnection;
         ResultSet rs = con.getMetaData().getTables(null, null, tableName, null);
         if (rs.next()) {
               return true;
         }else {
               return false;
         }
     }</code></pre>
     <div>
      <span style="font-size:11pt;">
       <span style="font-family:Consolas;">
        附录：
       </span>
      </span>
     </div>
     <div>
      1、
      <strong>
       检查某表中是否存在某个字段，注意大写
      </strong>
     </div>
     <div align="left" style="color:rgb(68,68,68);">
      <pre><code class="language-sql">select count(*) from User_Tab_Columns where  table_name='TABLENAME' and column_name='COLUMNNAME';</code></pre>
     </div>
     <div align="left">
      <span style="color:rgb(68,68,68);">
       <span style="font-size:14px;">
        <span style="font-family:Arial;">
         <strong>
          2、
         </strong>
        </span>
       </span>
      </span>
      <strong>
       检查某

        数据库
       </a>
       内，是否存在某张表，注意表名要大写
      </strong>
     </div>
     <div align="left" style="color:rgb(68,68,68);">
      <pre><code class="language-sql">select count(*) from all_tables where table_name='TABLENAME';</code></pre>
