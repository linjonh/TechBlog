# 微信小程序CSS样式图片闪烁

### 微信小程序CSS样式图片闪烁

微信小程序直接用CSS样式让图片闪烁起来！！！

### 废话不多说！直接上代码

```javascript
// 这是设置小程序WXSS页面
.shanshuo{-webkit-animation: twinkling 2s infinite ease-in-out;}
@-webkit-keyframes twinkling{0% {opacity: 0; }100% {opacity: 1; }
}

```
