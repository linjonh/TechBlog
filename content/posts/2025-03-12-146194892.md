    <h3>
     <a id="_36">
     </a>
     三、表单校验触发方式
    </h3>
    <p>
     ‌1.
     <strong>
      提交时整体校验‌
     </strong>
     <br/>
     绑定表单 ref，通过 validate 方法触发校验‌15：
    </p>
    <pre><code class="prism language-vue">&lt;template&gt;
  &lt;a-form :model="formModel" :rules="rules" ref="formRef"&gt;
    &lt;!-- 表单项 --&gt;
    &lt;a-button @click="handleSubmit"&gt;提交&lt;/a-button&gt;
  &lt;/a-form&gt;
&lt;/template&gt;

&lt;script setup&gt;
const formRef = ref();
const handleSubmit = async () =&gt; {
  try {
    await formRef.value.validate();
    // 校验通过，提交数据
  } catch (error) {
    console.error('校验失败:', error);
  }
};
&lt;/script&gt;
</code></pre>
    <p>
     2‌.
     <strong>
      动态字段校验‌
     </strong>
     <br/>
     针对循环生成的表单项（如多被保人），使用动态 prop 路径‌4：
    </p>
    <pre><code class="prism language-vue">&lt;div v-for="(item, index) in insuredList" :key="index"&gt;
  &lt;a-form-item 
    :name="['insuredList', index, 'name']" 
    :rules="[{ required: true, message: '被保人姓名必填' }]"
  &gt;
    &lt;a-input v-model:value="item.name" /&gt;
  &lt;/a-form-item&gt;
&lt;/div&gt;
</code></pre>
