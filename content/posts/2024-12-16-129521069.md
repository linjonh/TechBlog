# ubuntu22.04安装python

在 Ubuntu 22.04 上安装 Python 可以使用以下步骤：

1. 打开终端：按快捷键
   `Ctrl + Alt + T`
   打开终端。
2. 更新包管理器：在终端中输入
   `sudo apt update`
   并回车。
3. 安装 Python：在终端中输入
   `sudo apt install python3`
   并回车。
4. 验证安装：在终端中输入
   `python3`
   并回车，进入 Python 交互式环境，可以看到 Python 版本信息。

这样就完成了在 Ubuntu 22.04 上安装 Python 的过程。
