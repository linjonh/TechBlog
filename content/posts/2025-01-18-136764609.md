    <h3>
     <a id="Windowjava_4">
     </a>
     Window操作系统安装java流程
    </h3>
    <h4>
     <a id="JDK_6">
     </a>
     下载JDK
    </h4>
    <p>
     首先我们需要下载 java 开发工具包 JDK，下载地址：https://www.oracle.com/cn/java/technologies/downloads/ 在下载页面中根据自己的系统选择对应的版本
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bf49a9a1dd1b2c2d68a4ea63298e2fcf.png#pic_center"/>
    </p>
    <ul>
     <li>
      压缩包地址： https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.zip
     </li>
     <li>
      安装程序：https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.exe
     </li>
     <li>
      msi安装程序：https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.msi
     </li>
    </ul>
    <h4>
     <a id="_14">
     </a>
     安装
    </h4>
    <ul>
     <li>
      如果 下载安装程序 直接根据安装根据提示进行，还有安装 JDK 的时候也会安装 JRE，一并安装就可以了。
      <br/>
      ::: danger
      <br/>
      安装JDK，安装过程中可以自定义安装目录等信息，例如我们选择安装目录为 C:\Program Files (x86)\Java\jdk17-0026。
      <br/>
      :::
     </li>
     <li>
      如果 下载的是压缩包程序，解压到指定的自定义目录即可
     </li>
    </ul>
    <h4>
     <a id="_21">
     </a>
     配置环境变量
    </h4>
    <ol>
     <li>
      安装完成后，右击"我的电脑"，点击"属性"，选择"高级系统设置"；
     </li>
     <li>
      然后就会出现如下图所示的画面：
     </li>
     <li>
      在 “系统变量” 中设置 3 项属性，JAVA_HOME、PATH、CLASSPATH(大小写无所谓),若已存在则点击"编辑"，不存在则点击"新建"。
      <br/>
      ::: tip
      <br/>
      注意：如果使用 1.5 以上版本的 JDK，不用设置 CLASSPATH 环境变量，也可以正常编译和运行 Java 程序。
      <br/>
      :::
     </li>
     <li>
      变量设置参数如下：
     </li>
    </ol>
    <ul>
     <li>
      变量名：JAVA_HOME
     </li>
     <li>
      变量值：C:\Program Files (x86)\Java\jdk17-0026 // 要根据自己的实际路径配置
     </li>
     <li>
      变量名：CLASSPATH
     </li>
     <li>
      变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; //记得前面有个"."
     </li>
     <li>
      变量名：Path
     </li>
     <li>
      变量值：%JAVA_HOME%\bin;%JAVA_HOME%\bin;
     </li>
    </ul>
    <p>
     ::: tip
     <br/>
     注意：在 Windows10 或者 11 中，Path 变量里是分条显示的，我们需要将 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin; 分开添加，否则无法识别：
    </p>
    <p>
     %JAVA_HOME%\bin;
     <br/>
     %JAVA_HOME%\jre\bin;
     <br/>
     :::
    </p>
    <h4>
     <a id="JDK_43">
     </a>
     测试JDK是否安装成功
    </h4>
    <ol>
     <li>
      “开始”-&gt;“运行”，键入"cmd"；
     </li>
     <li>
      键入命令: java -version、java、javac 几个命令，出现以下信息，说明环境变量配置成功；
     </li>
    </ol>
    <pre><code class="prism language-shell">zeng@zmyMacBook-Pro ~ % <span class="token function">java</span> <span class="token parameter variable">-version</span>
<span class="token function">java</span> version <span class="token string">"17.0.10"</span> <span class="token number">2024</span>-01-16 LTS
Java<span class="token punctuation">(</span>TM<span class="token punctuation">)</span> SE Runtime Environment <span class="token punctuation">(</span>build <span class="token number">17.0</span>.10+11-LTS-240<span class="token punctuation">)</span>
Java HotSpot<span class="token punctuation">(</span>TM<span class="token punctuation">)</span> <span class="token number">64</span>-Bit Server VM <span class="token punctuation">(</span>build <span class="token number">17.0</span>.10+11-LTS-240, mixed mode, sharing<span class="token punctuation">)</span>
</code></pre>
    <h3>
     <a id="Macjava_53">
     </a>
     Mac操作系统安装java流程
    </h3>
    <h4>
     <a id="JDK_55">
     </a>
     下载JDK
    </h4>
    <p>
     查看系统架构：
    </p>
    <pre><code class="prism language-shell"> <span class="token function">uname</span> <span class="token parameter variable">-m</span>
<span class="token comment"># 若输出arm64,系统为arm64架构 选择下载 ARM64-xxx</span>
<span class="token comment"># 若输出x86_64,系统为x86-64（X64）架构  选择下载 x64-xxx</span>
</code></pre>
    <p>
     首先我们需要下载 java 开发工具包 JDK，下载地址：https://www.oracle.com/cn/java/technologies/downloads/ 在下载页面中根据自己的系统选择对应的版本
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ad395d8753d92c4cb8c34e91791047f9.png#pic_center"/>
    </p>
    <ul>
     <li>
      压缩包地址（arm64）： https://download.oracle.com/java/17/latest/jdk-17_macos-aarch64_bin.tar.gz
     </li>
     <li>
      压缩包地址（x86）： https://download.oracle.com/java/17/latest/jdk-17_macos-x64_bin.tar.gz
     </li>
     <li>
      安装程序（arm64）：https://download.oracle.com/java/17/latest/jdk-17_macos-aarch64_bin.dmg
     </li>
     <li>
      安装程序（x64）：https://download.oracle.com/java/17/latest/jdk-17_macos-x64_bin.dmg
     </li>
    </ul>
    <h4>
     <a id="_71">
     </a>
     安装
    </h4>
    <ul>
     <li>
      如果 下载安装程序 直接根据安装根据提示进行，还有安装 JDK 的时候也会安装 JRE，一并安装就可以了。
     </li>
     <li>
      如果 下载的是压缩包程序，解压到指定的自定义目录即可
     </li>
    </ul>
    <h4>
     <a id="_75">
     </a>
     配置环境变量
    </h4>
    <ol>
     <li>
      打开终端并输入，需使用sudo要有管理员权限否则无法修改保存文件
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token function">sudo</span> <span class="token function">vim</span> /etc/profile

</code></pre>
<ol start="2">
<li>
输入如下配置信息，要根据自己安装的版本号配置，可在安装路径下自行查看/Library/Java/JavaVirtualMachines）：
</li>
</ol>
<pre><code class="prism language-shell"><span class="token assign-left variable">JAVA_HOME</span><span class="token operator">=</span><span class="token string">"/Library/Java/JavaVirtualMachines/jdk-11.jdk/Contents/Home"</span>
<span class="token builtin class-name">export</span> JAVA_HOME
<span class="token assign-left variable">CLASS_PATH</span><span class="token operator">=</span><span class="token string">"<span class="token variable">$JAVA_HOME</span>/lib"</span>
<span class="token assign-left variable"><span class="token environment constant">PATH</span></span><span class="token operator">=</span><span class="token string">".<span class="token environment constant">$PATH</span>:<span class="token variable">$JAVA_HOME</span>/bin"</span>
</code></pre>
    <p>
     vim的简单操作：
    </p>
    <ul>
     <li>
      键盘按下i，显示INSERT，进入编辑模式
     </li>
     <li>
      esc退出编辑模式
     </li>
     <li>
      输入：wq！保存并退出
     </li>
    </ul>
    <ol start="3">
     <li>
      使环境变量生效
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token builtin class-name">source</span> /etc/profile
</code></pre>
    <p>
     检查环境变量配置是否成功
     <br/>
     终端输入如下命令，能得到JAVA_HOME路径，则JAVA 环境变量配置完毕
    </p>
    <pre><code class="prism language-shell"><span class="token builtin class-name">echo</span> <span class="token variable">$JAVA_HOME</span>
</code></pre>
<ol start="4">
<li>
检验环境
</li>
</ol>
<pre><code class="prism language-shell"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
