    <p>
     Vue-Lottie 是一个基于 Vue.js 的库，用于在 Vue 应用程序中渲染 Adobe After Effects 动画，通过 Bodymovin 插件导出的 JSON 数据来实现。这个项目非常适合那些想要在他们的Vue项目中加入精美动画而又不希望因为视频文件或 GIF 图片而增加应用大小的开发者。Vue-Lottie采用JavaScript编写，并且高度依赖Vue.js 进行工作。
    </p>
    <h5>
     安装Vue-Lottie
    </h5>
    <p>
     npm install vue-lottie --save
    </p>
    <p>
     或是用 CDN 方式引入
    </p>
    <pre><code>&lt;script src="https://cdnjs.cloudflare.com/ajax/libs/bodymovin/5.7.6/lottie.min.js"&gt;&lt;/script&gt;</code></pre>
    <h5>
     引入和使用Vue-Lottie
    </h5>
    <pre><code>import Lottie from 'vue-lottie'

export default {
  components: {
    Lottie,
  },
};</code></pre>
