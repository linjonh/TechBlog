# java中main函数

**java中main函数详解：**

```
class hellojava {

    public static void main (String[] args) {

    }

}
```

---

```
public : 公共的。权限是最大，在任何情况下都可以访问。
为了保证让 JVM 在任何情况下都可以访问 main 方法。

主方法是 java 虚拟机调用的。

static: 静态。
静态可以让 jvm 调用函数的时候更加方便。不需要通过对象调用。

void：没有返回值。
因为返回的数据时给 jvm。jvm 使用这个数据时没有意义的，所以就不要了。

main：函数名。注意 main 不是关键字，只不过是 jvm 识别的一个特殊的函数名而已。

arguments：参数 ... 担心某些程序在 启动时需要参数。
数组
java hellojava aa bb cc

```
