    <h4>
     <a id="linux__updatealternatives__54">
     </a>
     linux 软件版本管理命令 update-alternatives 使用详解
    </h4>
    <p>
     update-alternatives 命令用于处理 linux 系统中软件版本的切换，在各个 linux 发行版中均提供了该命令，命令参数略有区别，但大致是一样的。
    </p>
    <h5>
     <a id="alternatives__56">
     </a>
     alternatives 管理方式
    </h5>
    <pre><code class="prism language-bash">mali@ubuntu:~$ <span class="token function">ls</span> -l /usr/bin/python
lrwxrwxrwx 1 root root 24 6月  28 11:37 /usr/bin/python -<span class="token operator">&gt;</span> /etc/alternatives/python
mali@ubuntu:~$ <span class="token function">ls</span> -l /etc/alternatives/python
lrwxrwxrwx 1 root root 16 6月  28 11:37 /etc/alternatives/python -<span class="token operator">&gt;</span> /usr/bin/python3
</code></pre>
    <p>
     <code>
      python
     </code>
     这个可执行命令实际是一个链接，指向了
     <code>
      /etc/alternatives/python
     </code>
     。而
     <code>
      /etc/alternatives/python
     </code>
     也是一个链接，指向了
     <code>
      /usr/bin/python3
     </code>
     ，这才是最终的可执行文件。
     <code>
      alternatives
     </code>
     实际上是通过软链接的方式对版本进行管理。
    </p>
