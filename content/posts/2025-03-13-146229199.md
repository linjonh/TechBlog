    <p>
     1.首先需要在自己的静态资源目录下，放入两个sass文件。需要注意的是 .d 文件，是用来在开发的时候给予提示用的。
    </p>
    <p>
     <img alt="" height="419" src="https://i-blog.csdnimg.cn/direct/c074572c92824ed1b18304a3b729395d.png" width="438"/>
    </p>
    <p>
     这是里面的内容：
    </p>
    <pre><code class="language-css">// function.scss

@use "sass:math";

@function vw($px) {
    @return math.div($px, 1920) * 100vw;
}

@function vh($px) {
    @return math.div($px, 1080) * 100vh;
}</code></pre>
    <pre><code class="language-css">// function.d.sass

@function vw($px) {}

@function vh($px) {}</code></pre>
    <p>
     2.在构建工具中配置：
    </p>
    <p>
     这里以vite为例：
    </p>
    <pre><code class="language-javascript">export default defineConfig({
    css: {
        preprocessorOptions: {
            scss: {
                additionalData: `@use "@/assets/functions" as *;`
              }
        }
    }
})</code></pre>
    <p>
     到这里，你的任何.vue文件中的style中都能直接使用 vw、vh方法了。
    </p>
