     如何更新 Oh My Zsh
    </h1>
   </div>
  </div>

  <div class="article_content clearfix" id="article_content">

     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">

    <p>
     Oh My Zsh 会定期更新，提供新功能、优化和 Bug 修复。如果你想获取最新版本，可以手动更新。
    </p>
    <hr/>
    <h3>
     <a id="_1__4">
     </a>
     <strong>
      方法 1：使用官方更新命令（推荐 ✅）
     </strong>
    </h3>
    <p>
     Oh My Zsh 提供了内置的更新命令，非常简单高效。
    </p>
    <h4>
     <a id="11__7">
     </a>
     <strong>
      1.1 运行更新命令
     </strong>
    </h4>
    <pre><code class="prism language-sh">omz update
</code></pre>
    <p>
     或者
    </p>
    <pre><code class="prism language-sh">upgrade_oh_my_zsh
</code></pre>
    <h4>
     <a id="12__Zsh__16">
     </a>
     <strong>
      1.2 重启 Zsh 使更新生效
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token builtin class-name">exec</span> <span class="token function">zsh</span>
</code></pre>
    <p>
     ✅
     <strong>
      这个方法最推荐
     </strong>
     ，因为它：
    </p>
    <ul>
     <li>
      自动拉取最新的 Oh My Zsh 版本
     </li>
     <li>
      更新所有插件
     </li>
     <li>
      <strong>
       不会覆盖你的
       <code>
        .zshrc
       </code>
       配置
      </strong>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_2_Oh_My_Zsh_28">
     </a>
     <strong>
      方法 2：手动更新 Oh My Zsh
     </strong>
    </h3>
    <p>
     如果
     <code>
      omz update
     </code>
     命令失败，可以手动使用 Git 更新。
    </p>
    <h4>
     <a id="21__Oh_My_Zsh__31">
     </a>
     <strong>
      2.1 进入 Oh My Zsh 目录
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token builtin class-name">cd</span> ~/.oh-my-zsh
</code></pre>
    <h4>
     <a id="22__Git__36">
     </a>
     <strong>
      2.2 使用 Git 拉取最新版本
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token function">git</span> pull origin master
</code></pre>
    <p>
     ⚠️
     <strong>
      如果你本地修改了 Oh My Zsh
     </strong>
     ，可以先保存修改：
    </p>
    <pre><code class="prism language-sh"><span class="token function">git</span> stash
<span class="token function">git</span> pull origin master
<span class="token function">git</span> stash pop
</code></pre>
    <h4>
     <a id="23__Zsh__48">
     </a>
     <strong>
      2.3 重启 Zsh 使更新生效
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token builtin class-name">exec</span> <span class="token function">zsh</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_3_Oh_My_Zsh_55">
     </a>
     <strong>
      方法 3：自动更新 Oh My Zsh
     </strong>
    </h3>
    <p>
     Oh My Zsh 支持自动更新，但需要手动启用。
    </p>
    <h4>
     <a id="31__58">
     </a>
     <strong>
      3.1 启用自动更新
     </strong>
    </h4>
    <p>
     在
     <code>
      ~/.zshrc
     </code>
     中添加或取消注释以下行：
    </p>
    <pre><code class="prism language-sh">zstyle <span class="token string">':omz:update'</span> frequency <span class="token number">7</span>
</code></pre>
    <p>
     这表示
     <strong>
      Oh My Zsh 每 7 天会自动检查更新
     </strong>
     。
    </p>
    <h4>
     <a id="32__65">
     </a>
     <strong>
      3.2 应用更改
     </strong>
    </h4>
    <p>
     编辑
     <code>
      ~/.zshrc
     </code>
     文件：
    </p>
    <pre><code class="prism language-sh"><span class="token function">nano</span> ~/.zshrc
</code></pre>
    <p>
     找到
     <code>
      zstyle ':omz:update' frequency 7
     </code>
     这行，并确保它
     <strong>
      没有
      <code>
       #
      </code>
      注释符号
     </strong>
     。
    </p>
    <p>
     然后重新加载配置：
    </p>
    <pre><code class="prism language-sh"><span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <h4>
     <a id="33__77">
     </a>
     <strong>
      3.3 更改自动更新频率
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       每 3 天检查一次更新
      </strong>
      ：
      <pre><code class="prism language-sh">zstyle <span class="token string">':omz:update'</span> frequency <span class="token number">3</span>
</code></pre>
     </li>
     <li>
      <strong>
       禁用自动更新
      </strong>
      ：
      <pre><code class="prism language-sh">zstyle <span class="token string">':omz:update'</span> frequency <span class="token number">0</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_4_Oh_My_Zsh_89">
     </a>
     <strong>
      方法 4：重新安装 Oh My Zsh（如果更新失败）
     </strong>
    </h3>
    <p>
     如果你的 Oh My Zsh 版本过旧或更新失败，可以考虑
     <strong>
      重新安装
     </strong>
     。
    </p>
    <h4>
     <a id="41__zshrc__92">
     </a>
     <strong>
      4.1 备份原来的
      <code>
       .zshrc
      </code>
      配置
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token function">cp</span> ~/.zshrc ~/.zshrc.backup
</code></pre>
    <h4>
     <a id="42__Oh_My_Zsh_97">
     </a>
     <strong>
      4.2 卸载旧版 Oh My Zsh
     </strong>
    </h4>
    <pre><code class="prism language-sh">uninstall_oh_my_zsh
</code></pre>
    <p>
     如果
     <code>
      uninstall_oh_my_zsh
     </code>
     命令不可用，可以手动删除：
    </p>
    <pre><code class="prism language-sh"><span class="token function">rm</span> <span class="token parameter variable">-rf</span> ~/.oh-my-zsh
<span class="token function">rm</span> ~/.zshrc
</code></pre>
    <h4>
     <a id="43__Oh_My_Zsh_107">
     </a>
     <strong>
      4.3 重新安装 Oh My Zsh
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token function">sh</span> <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <h4>
     <a id="44__zshrc__112">
     </a>
     <strong>
      4.4 恢复
      <code>
       .zshrc
      </code>
      配置
     </strong>
    </h4>
    <pre><code class="prism language-sh"><span class="token function">mv</span> ~/.zshrc.backup ~/.zshrc
<span class="token builtin class-name">source</span> ~/.zshrc
</code></pre>
    <hr/>
    <h3>
     <a id="_Oh_My_Zsh__120">
     </a>
     <strong>
      如何检查 Oh My Zsh 是否更新成功？
     </strong>
    </h3>
    <p>
     运行以下命令：
    </p>
    <pre><code class="prism language-sh">omz version
</code></pre>
    <p>
     如果能看到最新版本号，说明更新成功 🎉。
    </p>
