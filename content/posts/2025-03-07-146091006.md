    <h2>
     <a id="1__0">
     </a>
     1 设置-文件关联
    </h2>
    <h3>
     <a id="11__2">
     </a>
     1.1 应用场景
    </h3>
    <p>
     文件关联是指在文件管理器中
     <font color="blue">
      双击某类型的文件
     </font>
     ，操作系统自动调用可以打开该文件的应用程序，比如：用户双击
     <code>
      XXXX.txt
     </code>
     文件，系统默认会使用记事本打开该文件。
     <br/>
     由于各行各业都会定义特有的文件类型，即使有些文件类型本质还是文本文件，但由于扩展名的差异，系统并不一定能找到正确的应用程序来打开该文件。并且，不同用户安装的软件差异，有些软件在安装时会默认绑定用于打开某些文件类型，所以，在电脑A上双击可以打开某文件，但在电脑B上双击就不一定可以打开，或者使用了不同的应用程序打开。
    </p>
    <p>
     EverEdit支持大量的文本编码以及语法着色，其优点是打开速度快，避免用户长时间等待，绑定EverEdit打开各类文本文件也是一个不错的选项。
    </p>
    <h3>
     <a id="12__9">
     </a>
     1.2 使用方法
    </h3>
    <p>
     选择主菜单
     <code>
      工具 -&gt; 设置 -&gt; 文件关联
     </code>
     ，会弹出如下的关联窗口，如下图所示：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/29bca9b7855545a09727b8dbb2dd7655.png" width="600"/>
    </p>
    <ul>
     <li>
      方法1：如果文件扩展名已经在EverEdit中定义，则直接勾选对应的项，单击
      <code>
       应用
      </code>
      按钮即可。
     </li>
     <li>
      方法2：单击自定义按钮，在输入框中直接输入扩展名，此时并不能生效，会在列表中添加
      <code>
       用户自定义
      </code>
      项，同样需要勾选后单击
      <code>
       应用
      </code>
      按钮。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6d33c4f5c3b1457cb2c47f316af0fc7f.png" width="350"/>
     </li>
    </ul>
    <h3>
     <a id="13__19">
     </a>
     1.3 异常处理
    </h3>
    <p>
     部分用户由于系统环境的复杂性，可能使用上述方法设置后，在文件管理器上双击文件，仍然不能通过EverEdit打开相关文件，此时可以通过以下方式关联到EverEdit：
    </p>
    <ol>
     <li>
      按下
      <code>
       Win+I
      </code>
      打开控制面板
     </li>
     <li>
      在
      <code>
       控制面板
      </code>
      中选择
      <code>
       应用
      </code>
      如下图所示：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/31a67210d1ed49cd9fc8af81321b28ad.png" width="700"/>
     </li>
    </ol>
    <p>
     注：不同系统可能存在差异，请用户自行在网络上搜索相关解决办法。
    </p>
    <ol start="3">
     <li>
      <p>
       单击
       <code>
        默认应用
       </code>
       ，如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7688dc063a8a422386664b9a4b2f3e0b.png" width="300"/>
      </p>
     </li>
     <li>
      <p>
       在右侧选择
       <code>
        按文件类型指定默认应用
       </code>
       如下图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0dde50bfd4004d55b13262bcc1c7d9b7.png" width="300"/>
      </p>
     </li>
     <li>
      <p>
       在文件类型列表中找到需要绑定的文件类型，并选择默认应用
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7486149fae6740d8ac3f74d8260e09ba.png" width="800"/>
      </p>
     </li>
    </ol>

     🕮说明
