    <h3>
     网格布局grid
    </h3>
    <p>
     提供了带有行和列的基于网格的布局系统，无需使用浮动和定位。
    </p>
    <p>
     当 HTML 元素的
     <code>
      display
     </code>
     属性设置为
     <code>
      grid
     </code>
     或
     <code>
      inline-grid
     </code>
     时，它就会成为网格容器。
    </p>
    <p>
     更多布局模式可以参考之前的博客：
     <br/>
     <a class="link-info" href="https://blog.csdn.net/m0_74556076/article/details/146029752?spm=1001.2014.3001.5501" title="​​​​​​CSS—flex布局、过渡transition属性、2D转换transform属性、3D转换transform属性">
      <br/>
      ​​​​​​CSS—flex布局、过渡transition属性、2D转换transform属性、3D转换transform属性
     </a>
    </p>
    <p>
     有关display属性之前也有写过：
    </p>
    <p>
     <a class="link-info" href="https://blog.csdn.net/m0_74556076/article/details/146000658?spm=1001.2014.3001.5501" title="CSS—显示模式display、float浮动，清除浮动的6种方式、定位position、元素溢出overflow">
      CSS—显示模式display、float浮动，清除浮动的6种方式、定位position、元素溢出overflow
     </a>
    </p>
    <p>
     <img alt="" height="251" src="https://i-blog.csdnimg.cn/direct/0bab6ea041c04d83a99cf762f8a4d142.png" width="343">
      <img alt="" height="260" src="https://i-blog.csdnimg.cn/direct/84dcaad01cb847d4871452f18ab5fe7f.png" width="333"/>
     </img>
    </p>
    <p>
     当把网格项目放在网格容器中时，引用行号：
    </p>
    <p>
     把网格项目放在列线 1，并在列线 3 结束它：
    </p>
    <pre><code>.item1 {
  grid-column-start: 1;
  grid-column-end: 3;
}</code></pre>
    <table>
     <thead>
      <tr>
       <th>
        属性
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         grid-column
        </code>
       </td>
       <td>
        定义将项目放置在哪一列上。是 grid-column-start和grid-column-end缩写&lt;/br&gt;例如：
        <code>
         grid-column: 2/3;
        </code>
        第2列开始第3列结束。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-row
        </code>
       </td>
       <td>
        定义将项目放置在哪一行上，写法与
        <code>
         grid-column
        </code>
        类似
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-area
        </code>
       </td>
       <td>
        <code>
         grid-row-start、grid-column-start、grid-row-end 和 grid-column-end
        </code>
        属性的简写。
        <strong>
         也可用于为项目命名
        </strong>
        。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-column-gap
        </code>
       </td>
       <td>
        调整列间隙
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-row-gap
        </code>
       </td>
       <td>
        调整行间隙
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-gap
        </code>
       </td>
       <td>
        调整间隙
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-template-columns
        </code>
       </td>
       <td>
        定义网格布局中的列数，并可定义每列的宽度。以空格分隔的列表，其中每个值定义相应列的长度。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         grid-template-rows
        </code>
       </td>
       <td>
        定义每列的高度
       </td>
      </tr>
      <tr>
       <td>
        <code>
         justify-content
        </code>
       </td>
       <td>
        在容器内对齐整个网格。网格的总宽度必须小于容器的宽度， justify-content 属性才能生效。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         align-content
        </code>
       </td>
       <td>
        垂直对齐容器内的整个网格。网格的总高度必须小于容器的高度， align-content 属性才能生效。
       </td>
      </tr>
     </tbody>
    </table>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
&lt;style&gt;
.item1 {
  grid-area: myArea;
}

.grid-container {
  display: grid;
  grid-template-areas: 'myArea myArea . . .' 'myArea myArea . . .';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.grid-container &gt; div {
  background-color: rgba(255, 255, 255, 0.8);
  text-align: center;
  padding: 20px 0;
  font-size: 30px;
}
&lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;

&lt;p&gt;Item1 的名为 "myArea"，会占据两列两行&lt;/p&gt;

&lt;div class="grid-container"&gt;
  &lt;div class="item1"&gt;1&lt;/div&gt;
  &lt;div class="item2"&gt;2&lt;/div&gt;
  &lt;div class="item3"&gt;3&lt;/div&gt;  
  &lt;div class="item4"&gt;4&lt;/div&gt;
  &lt;div class="item5"&gt;5&lt;/div&gt;
  &lt;div class="item6"&gt;6&lt;/div&gt;
&lt;/div&gt;

&lt;/body&gt;
&lt;/html&gt;</code></pre>
