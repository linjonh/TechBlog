    <p>
     我在搭建zookeeper集群的时候总是报：
     <code>
      JAVA_HOME is not set and java could not be found in PATH.
     </code>
     的错误，但是我的java环境是没问题的，这个问题困扰我两天了，终于解决了，在此记录一下：
    </p>
    <h6>
     <a id="_1">
     </a>
     解决办法：
    </h6>
    <ol>
     <li>
      打开zookeeper安装文件中
      <code>
       bin
      </code>
      路径下的zkEnv.sh文件。
     </li>
     <li>
      通过
      <code>
       export
      </code>
      查找出自己的jdk的路径，如图：
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a69969ddd6bb91656afd14d3775d2465.png"/>
     </li>
     <li>
      在文件的最前面（注意是最前面，别问为啥，可以自己测试去）添加：
     </li>
    </ol>
    <pre><code class="prism language-Ruby">export JAVA_HOME=/usr/local/src/jdk1.8(换成你自己的jdk路径)
</code></pre>
    <p>
     如图：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a42f0ab193311fe0406df77a4ab4edb7.png"/>
    </p>
