    <h3>
     <a id="_0">
     </a>
     一、文件管理类
    </h3>
    <h4>
     <a id="1_ls___2">
     </a>
     1.
     <code>
      ls
     </code>
     - 列出目录内容
    </h4>
    <p>
     <strong>
      作用
     </strong>
     ：查看文件与目录信息。
    </p>
    <pre><code class="prism language-bash"><span class="token function">ls</span> <span class="token parameter variable">-lh</span> /var/log       <span class="token comment"># 以人性化单位显示文件大小</span>
<span class="token function">ls</span> <span class="token parameter variable">-a</span> ~/.ssh           <span class="token comment"># 显示隐藏文件（如 SSH 密钥）</span>
<span class="token function">ls</span> <span class="token parameter variable">-R</span> /opt             <span class="token comment"># 递归列出子目录内容</span>
</code></pre>
    <h4>
     <a id="2_cp___10">
     </a>
     2.
     <code>
      cp
     </code>
     - 文件复制
    </h4>
    <p>
     <strong>
      作用
     </strong>
     ：复制文件或目录。
    </p>
    <pre><code class="prism language-bash"><span class="token function">cp</span> <span class="token parameter variable">-r</span> /etc/nginx /backup/conf  <span class="token comment"># 递归复制配置目录</span>
<span class="token function">cp</span> <span class="token parameter variable">-v</span> file.txt<span class="token punctuation">{<!-- --></span>,.bak<span class="token punctuation">}</span>          <span class="token comment"># 备份文件（生成 file.txt.bak）</span>
</code></pre>
    <h4>
     <a id="3_rm___17">
     </a>
     3.
     <code>
      rm
     </code>
     - 文件删除
    </h4>
    <p>
     <strong>
      作用
     </strong>
     ：删除文件或目录。
     <br/>
     ** 注意**：
     <code>
      rm -rf
     </code>
     命令慎用，可能导致数据不可恢复！
    </p>
    <pre><code class="prism language-bash"><span class="token function">rm</span> <span class="token parameter variable">-f</span> *.log          <span class="token comment"># 强制删除日志文件</span>
<span class="token function">rm</span> <span class="token parameter variable">-rf</span> /tmp/*        <span class="token comment"># 清空临时目录（危险操作！）</span>
</code></pre>

     </code>

     <code>

     </code>

<span class="token function">htop</span> <span class="token parameter variable">--sort</span><span class="token operator">=</span>MEM        <span class="token comment"># 按内存使用排序（需安装 htop）</span>

     2.
     <code>
      df
     </code>
     /
     <code>
      du
     </code>
     - 磁盘空间分析
    </h4>

     <strong>

<span class="token function">du</span> --max-depth<span class="token operator">=</span><span class="token number">1</span> /home <span class="token operator">|</span> <span class="token function">sort</span> <span class="token parameter variable">-hr</span>  <span class="token comment"># 按层级统计子目录大小</span>

     </a>
     3.
     <code>
      free
     </code>

    </h4>
    <p>
     <strong>
      作用
     </strong>
