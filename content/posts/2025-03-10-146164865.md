# EXCEL IF自动填充功能

## 使用Excel自动填充端口用途：提升工作效率的技巧

在日常工作中，Excel 是一个非常强大的工具，尤其是在处理大量数据时。通过使用 Excel 的自动填充功能，我们可以快速地为数据添加额外的信息，从而提升工作效率。本文将介绍如何根据端口号自动填充其用途，帮助你在处理类似数据时更加高效。

### 场景描述

假设我们有一个包含 IP 地址、端口号、服务类型等信息的 Excel 表格：

| A | B | C | D | E | F | G |
| --- | --- | --- | --- | --- | --- | --- |
| **IP** | **端口** | **服务** | **产品** | **版本** | **CDN/云** | 用途 |
|  | 9001 | https |  |  |  |  |
|  | 9002 | https |  |  |  |  |
|  | 9003 | https |  |  |  |  |
|  | 9101 | https |  |  |  |  |
|  | 9111 | https |  |  |  |  |
|  | 9201 | https |  |  |  |  |
|  | 9299 | https |  |  |  |  |

表格大概内容如上，实际表格内容：

![](https://i-blog.csdnimg.cn/img_convert/6c1202cecdbed2134e5a59aa30be6385.png)

表格有700多行，我们需要根据端口号的不同，自动填充对应的用途信息。具体规则如下：

* 端口号为 9001，用途为“提供客户端入口”。
* 端口号为 9002，用途为“提供逻辑服务入口”。
* 端口号为 9003，用途为“认证auth认证服务入口”。
* 端口号以 91 开头，用途为“提供gateway服务”。
* 端口号以 92 开头，用途为“提供pvp-game服务”。

### 实现步骤

#### 1. 打开Excel文件

首先，打开包含端口信息的 Excel 文件。假设文件名为
`xxx.xlsx`
，并且数据位于名为“端口扫描”的工作表中。

#### 2. 添加用途列

在 Excel 表格的 G 列添加一个新的列，命名为“用途”。这一列将用于存储根据端口号自动填充的用途信息。

#### 3. 使用IF函数进行条件判断

我们可以使用 Excel 的
`IF`
函数来实现根据端口号自动填充用途的功能。具体步骤如下：

在 G2 单元格中输入以下公式：

```plain
=IF(B2=9001,"提供客户端入口",IF(B2=9002,"提供逻辑服务入口",IF(B2=9003,"认证auth认证服务入口",IF(LEFT(B2,2)="91","提供gateway服务",IF(LEFT(B2,2)="92","提供pvp-game服务","")))))

```

这个公式的逻辑如下：

```
- 如果 B2 单元格的值为 9001，则返回“提供客户端入口”。
- 如果 B2 单元格的值为 9002，则返回“提供逻辑服务入口”。
- 如果 B2 单元格的值为 9003，则返回“提供认证auth认证服务入口”。
- 如果 B2 单元格的值以 91 开头，则返回“提供gateway服务”。
- 如果 B2 单元格的值以 92 开头，则返回“提供pvp-game服务”。
- 如果以上条件都不满足，则返回空字符串。

```

1. 按下回车键，G2 单元格将根据 B2 单元格的值自动填充对应的用途信息。
2. 将 G2 单元格的公式向下拖动，应用到 G 列的其他单元格中。这样，所有行的用途信息都会根据端口号自动填充。

#### 4. 检查结果

完成上述步骤后，G 列将根据 B 列的端口号自动填充对应的用途信息

![](https://i-blog.csdnimg.cn/img_convert/e7ab47096aaebeafd24856d8fd7d9060.png)

你可以检查一下结果，确保所有端口号的用途都正确填充。

### 总结

通过使用 Excel 的
`IF`
函数和
`LEFT`
函数，我们可以轻松地根据端口号自动填充对应的用途信息。这种方法不仅节省了手动输入的时间，还减少了出错的可能性。在处理大量数据时，这种自动化操作可以显著提升工作效率。
