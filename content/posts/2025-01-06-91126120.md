    <p>
     最近电脑突然出现
     <code>
      请去 Windows 应用商店查看有关 xxx 的更多信息
     </code>
     的错误提示，无奈百度许多没有解决，去微软官网找答案，搜索到了一篇，再此进行笔记。
     <br/>
     请参考下方本人电脑信息和错误问题，来判断是否与您的错误一致
    </p>
    <pre><code>Windows 规格
版本 Windows 10 专业版
版本号 1809
安装日期 2018/12/25
操作系统版本 17763.503
微软账号状态 未登录
</code></pre>
    <pre><code>错误描述 ：
系统所有的商店应用，如计算机，日历，天气，邮件，Xbox等都无法打开，均有类似下方图片的提示
但是Windows Microsoft Store 可以正常打开，但无法直接重新更新计算机等微软商店应用。  
</code></pre>
    <p>
     <img alt="错误图片" src="https://i-blog.csdnimg.cn/blog_migrate/e570ef09466b9fd5f938cd9d6ff587cf.png">
      <br/>
      搜索到的官方原文回复如下：
     </img>
    </p>
    <pre><code>您好！      
感谢您联系微软技术支持。 
我了解到您遇到了Windows 10应用商店和部分应用无法打开的问题。 
请问您是升级Windows后出现这样的问题还是使用了一段时间后出现的呢？您最近有安装过什么优化软件吗？ 
您可以尝试以下方案： 
1. 右键点击任务栏，选择“属性”，切换到“导航”选项卡，勾选第三项“当我右键点击......PowerShell”，确认； 
2. 右键点击“开始”徽标，选择“Windows PowerShell（管理员）”  
3. 在Windows PowerShell（管理员）下执行命令（可复制后右键黏贴下列命令）：
$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest
如果上述步骤无法解决，请找到以下文件夹，将其本地缓存文件删除试试：
1.进入“这台电脑” - "C盘“
2.找到”User“文件夹
3.打开您当前账户命名的文件夹
4.打开”Appdata“-”Local“-”Packages“-”Microsoft.WindowsStore_8wekyb3d8bbwe“-”LocalCachel“
5.将目录下的所有文件删除即可。
希望以上信息可以帮助到您
</code></pre>
    <p>
     发现好多人按照这个执行，仍未解决，其实不然，这个就是解决办法。
    </p>
    <p>
     首先按照官方回复打开
     <code>
      Windows PowerShelll (管理员)
     </code>
     ，Winodws+X弹出，选择
     <code>
      Windows PowerShelll (管理员)
     </code>
     ，输入下方命令。
    </p>
    <pre><code>$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest  
命令较长，请复制全。注意空格。
</code></pre>
    <p>
     输入后 接着去
     <code>
      C:\Users\DELL\AppData\Local\Packages\Microsoft.WindowsStore_8wekyb3d8bbwe
     </code>
     文件夹删除
     <code>
      LocalCache
     </code>
     文件夹清除缓存。
    </p>
    <p>
     提示：DELL文件夹为我得电脑用户名称，如果您的不同请选择您的用户文件夹。
    </p>
    <p>
     如果您的文件夹中没有AppData文件夹，请在窗口上方打开
     <strong>
      查看
     </strong>
     中的
     <strong>
      选项
     </strong>
     (
     <code>
      文件夹选项
     </code>
     )按钮，找到查看选项卡，在列表中找到
     <strong>
      显示隐藏的文件、文件夹和驱动器
     </strong>
     并选中，然后依次点击
     <strong>
      应用
     </strong>
     ，
     <strong>
      确定
     </strong>
     ，并关闭窗口即可。
    </p>
    <h6>
     <a id="_51">
     </a>
     重点：
    </h6>
    <p>
     官方回复到这里就结束了，但尝试打开计算器等其他应用，仍然出现
     <code>
      请去 Windows 应用商店查看有关 xxx 的更多信息
     </code>
     的错误提示，这时，不用慌张，打开
     <code>
      Windows Microsoft Store
     </code>
     ，过程中请保持网络状态连接和良好，打开后右上角有个
     <code>
      下载和更新
     </code>
     ，点进去，系统会自动加载本地所有已存在的应用，然后点击
     <strong>
      全部更新
     </strong>
     。耐心等待更新完成，Winodws 应用就恢复正常了。
    </p>
    <p>
     更新中页面：
     <br/>
     <img alt="正在安装图片" src="https://i-blog.csdnimg.cn/blog_migrate/6502ea555d079fcff4d645a1c3ce717a.png">
      <br/>
      更新完成后可以正常打开计算机等程序
      <br/>
      <img alt="成功图片" src="https://i-blog.csdnimg.cn/blog_migrate/33e8ab464d23640b8e10da8d04fc7967.png"/>

    </p>
    <h4>
     <a id="_59">
     </a>
     哒哒哒第二种情况
    </h4>
    <p>
     系统Win10，最近偶然发现，所有微软应用都无法打开了，包括微软商店，感觉很奇怪。全部灰色了，经过一番研究，发现，
     <strong>
      So easy
     </strong>
     。微软系列应用有如下特征：需要检测
     <strong>
      防火墙
     </strong>
     和
     <strong>
      Windows Update服务
     </strong>
     ，开启这两个服务后，重新点击灰色图标的微软商店，发现不再弹出错误（错误忘记截图了），右下角通知中出现了
     <code>
      微软商店正在下载
     </code>
     的新通知，然后就提示安装完成。打开微软商店，去下载列表，获取所有最新下载，然后开始更新所有应用，Ojbk，所有应用重新点亮恢复生机，美滋滋。
    </p>
