# 微软与 OpenAI 达成合作，获得 GPT-3 独家使用授权！

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/6196a1372ef3bb2ee4ff852a23617268.jpeg#pic_center)

当地时间 9 月 22 日，微软执行副总裁兼首席技术官 Kevin Scott 在官方博客上正式宣布：微软已与 Open AI 达成合作，获得 GPT-3 语言模型的独家许可。

GPT-3 模型是一种自动生成文本程序，它不仅可以用于写“论文”，也可以应用于“自动生成代码”，自今年 7 月发布后，也被业界视为最强大的人工智能语言模型。

事实上，微软与 OpenAI 此次的强强联手，也属于意料之中。

早在去年，微软为了提升在 AI 上的储备，就已花 10 亿美元投资 OpenAI，同时，为了让 OpenAI 推出高质量的训练模型，微软配置了由 28.5 万个 CPU 核心、1 万个英伟达 GPU 核心的超级计算机，目前这台超算在全球算力榜上排名前五。其中 GPT-3 容量达到了 45TB，参数个数有 1750 亿。

如今，两家公司也正在共同构建新的 Azure AI 超级计算技术，OpenAI 也将其服务移植到 Microsoft Azure 云计算平台上运行，用于创建新的 AI 技术并实现通用人工智能，其中，Azure 是 OpenAI 访问其培训许多模型所需的大量计算资源的平台。

针对此次合作，微软首席技术官 Kevin Scott 表示，"我们认为这是一个令人难以置信的机会，可以扩展我们的 Azure 成为支持 AI 的平台，使得人工智能技术民主化，支持新产品、服务等，并增加 AI 在 Scale 中的积极影响。"同时，“这使得微软能够利用 GPT-3 技术为客户开发和交付高级 AI 解决方案，并利用高级自然语言生成的惊人功能创建新的解决方案。”

另外，OpenAI 也表示，"
**这笔交易不会影响通过 OpenAI 的 API 继续访问 GPT-3 模型，其现有和未来用户将继续像往常一样使用我们的 API 构建应用程序。**
"

那么，此次的合作，微软究竟从中获得了什么？

目前具体的细节部分还不清楚。但是据外媒 The Verge 报道，微软获得 GPT-3 的独家许可允许其独特的访问 GPT-3 的基础代码，其中包含最新的更新等，微软希望将其集成到它的产品和服务中。
