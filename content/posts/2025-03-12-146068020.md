    <h3 name="1.%20%E5%BC%95%E8%A8%80">
     <span style="color:#ed7976">
      1. 引言
     </span>
    </h3>
    <p>
     在 Flutter 中，
     <code>
      TextButton
     </code>
     是一种无背景的按钮，适用于次要或轻量级操作。它的外观更加简洁，仅包含文字，适合用作辅助性操作，如“取消”或“了解更多”。相比
     <code>
      ElevatedButton
     </code>
     ，
     <code>
      TextButton
     </code>
     没有阴影和背景色，更加简约。
    </p>
    <h3 id="2.%20TextButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95" name="2.%20TextButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95">
     <span style="color:#ed7976">
      2. TextButton 的基本用法
     </span>
    </h3>
    <p>
     <code>
      TextButton
     </code>
     需要
     <code>
      onPressed
     </code>
     事件和
     <code>
      child
     </code>
     组件。
    </p>
    <pre><code class="language-Dart">TextButton(
  onPressed: () {
    print('TextButton 被点击');
  },
  child: Text('点击我'),
)
</code></pre>
    <p class="img-center">
     <img alt="" height="282" src="https://i-blog.csdnimg.cn/direct/fa6e8c69c68240c0b457228c21483232.png" width="500"/>
    </p>
    <p>
     如果
     <code>
      onPressed
     </code>
     设为
     <code>
      null
     </code>
     ，按钮会变为不可点击状态。
    </p>
    <pre><code class="language-Dart">TextButton(
  onPressed: null,
  child: Text('不可点击'),
)
</code></pre>
    <h3 id="3.%20%E4%B8%BB%E8%A6%81%E5%B1%9E%E6%80%A7" name="3.%20%E4%B8%BB%E8%A6%81%E5%B1%9E%E6%80%A7">
     <span style="color:#ed7976">
      3. 主要属性
     </span>
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        属性
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         onPressed
        </code>
       </td>
       <td>
        按钮点击时的回调函数
       </td>
      </tr>
      <tr>
       <td>
        <code>
         onLongPress
        </code>
       </td>
       <td>
        长按时触发的回调
       </td>
      </tr>
      <tr>
       <td>
        <code>
         child
        </code>
       </td>
       <td>
        按钮的内容，如
        <code>
         Text
        </code>
        或
        <code>
         Icon
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <code>
         style
        </code>
       </td>
       <td>
        自定义按钮样式
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     示例：
    </p>
    <pre><code class="language-Dart">TextButton(
  onPressed: () {},
  onLongPress: () =&gt; print('长按按钮'),
  child: Text('长按试试'),
)
</code></pre>
    <h3 id="4.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%8C%89%E9%92%AE%E6%A0%B7%E5%BC%8F" name="4.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%8C%89%E9%92%AE%E6%A0%B7%E5%BC%8F">
     <span style="color:#ed7976">
      4. 自定义按钮样式
     </span>
    </h3>
    <h4 id="4.1%20%E4%BF%AE%E6%94%B9%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2" name="4.1%20%E4%BF%AE%E6%94%B9%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2">
     <span style="color:#956fe7">
      4.1 修改文本颜色
     </span>
    </h4>
    <pre><code class="language-Dart">TextButton(
  style: TextButton.styleFrom(
    primary: Colors.blue, // 文字颜色
  ),
  onPressed: () {},
  child: Text('自定义颜色'),
)
</code></pre>
    <h4 id="4.2%20%E6%B7%BB%E5%8A%A0%E8%83%8C%E6%99%AF%E8%89%B2" name="4.2%20%E6%B7%BB%E5%8A%A0%E8%83%8C%E6%99%AF%E8%89%B2">
     <span style="color:#956fe7">
      4.2 添加背景色
     </span>
    </h4>
    <pre><code class="language-Dart">TextButton(
  style: TextButton.styleFrom(
    backgroundColor: Colors.blue,
    primary: Colors.white,
  ),
  onPressed: () {},
  child: Text('带背景色的 TextButton'),
)
</code></pre>
    <h4 id="4.3%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%BD%A2%E7%8A%B6%E5%92%8C%E8%BE%B9%E8%B7%9D" name="4.3%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%BD%A2%E7%8A%B6%E5%92%8C%E8%BE%B9%E8%B7%9D">
     <span style="color:#956fe7">
      4.3 修改按钮形状和边距
     </span>
    </h4>
    <pre><code class="language-Dart">TextButton(
  style: TextButton.styleFrom(
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(20),
    ),
    padding: EdgeInsets.symmetric(horizontal: 20, vertical: 10),
  ),
  onPressed: () {},
  child: Text('圆角按钮'),
)
</code></pre>
    <h4 id="4.4%20%E6%A0%B7%E5%BC%8F%E5%AE%9A%E5%88%B6" name="4.4%20%E6%A0%B7%E5%BC%8F%E5%AE%9A%E5%88%B6">
     <span style="color:#956fe7">
      4.4 样式定制
     </span>
    </h4>
    <pre><code class="language-Dart">TextButton(
            style: ButtonStyle(
              // 文字颜色（包括禁用状态）
              foregroundColor: WidgetStateProperty.resolveWith&lt;Color&gt;(
                    (Set&lt;WidgetState&gt; states) {
                  if (states.contains(WidgetState.disabled)) return Colors.grey;
                  return Colors.blue;
                },
              ),
              // 背景色
              backgroundColor: WidgetStateProperty.all(Colors.transparent),
              // 水波纹颜色
              overlayColor: WidgetStateProperty.all(Colors.blue.withOpacity(0.1)),
              // 内边距
              padding: WidgetStateProperty.all(EdgeInsets.symmetric(horizontal: 16)),
              // 边框形状
              shape: WidgetStateProperty.all(
                RoundedRectangleBorder(
                  borderRadius: BorderRadius.circular(8),
                ),
              ),
            ),
            onPressed: () {},
            child: Text('自定义样式'),
          )</code></pre>
    <h3 id="5.%C2%A0%E9%AB%98%E7%BA%A7%E5%BA%94%E7%94%A8%E6%8A%80%E5%B7%A7" name="5.%C2%A0%E9%AB%98%E7%BA%A7%E5%BA%94%E7%94%A8%E6%8A%80%E5%B7%A7">
     <span style="color:#ed7976">
      5. 高级应用技巧
     </span>
    </h3>
    <h4 id="5.1%C2%A0%E5%9B%BE%E6%A0%87%2B%E6%96%87%E6%9C%AC%E7%BB%84%E5%90%88" name="5.1%C2%A0%E5%9B%BE%E6%A0%87%2B%E6%96%87%E6%9C%AC%E7%BB%84%E5%90%88">
     5.1 图标+文本组合
    </h4>
    <pre><code class="language-Dart">          TextButton.icon(
            icon: Icon(Icons.add_box_rounded, size: 20),
            label: Text('添加好友'),
            onPressed: () {},
            style: ButtonStyle(
              padding: WidgetStateProperty.all(
                EdgeInsets.symmetric(vertical: 12, horizontal: 20),
              ),
            ),</code></pre>
    <p class="img-center">
     <img alt="" height="118" src="https://i-blog.csdnimg.cn/direct/b44496826ff3403c9199f413992d817b.png" width="300"/>
    </p>
    <h4 id="5.2%C2%A0%C2%A0%E4%B8%BB%E9%A2%98%E7%BB%9F%E4%B8%80%E9%85%8D%E7%BD%AE" name="5.2%C2%A0%C2%A0%E4%B8%BB%E9%A2%98%E7%BB%9F%E4%B8%80%E9%85%8D%E7%BD%AE">
     <span style="color:#956fe7">
      5.2  主题统一配置
     </span>
    </h4>
    <pre><code class="language-Dart">MaterialApp(
  theme: ThemeData(
    textButtonTheme: TextButtonThemeData(
      style: ButtonStyle(
        foregroundColor: MaterialStateProperty.all(Colors.purple),
        textStyle: MaterialStateProperty.all(
          TextStyle(fontWeight: FontWeight.bold)),
      ),
    ),
  ),
)</code></pre>
    <h4 id="5.3%20%E5%8A%A8%E6%80%81%E4%BA%A4%E4%BA%92" name="5.3%20%E5%8A%A8%E6%80%81%E4%BA%A4%E4%BA%92">
     <span style="color:#956fe7">
      5.3 动态交互
     </span>
    </h4>
    <pre><code class="language-Dart">// 点击缩放动画
TextButton(
  onPressed: () {},
  child: AnimatedContainer(
    duration: Duration(milliseconds: 200),
    transform: isPressed ? Matrix4.diagonal3Values(0.95, 0.95, 1) : null,
    child: Text('动态按钮'),
  ),
)</code></pre>
    <h3 id="6.%20%E6%80%A7%E8%83%BD%E4%BC%98%E5%8C%96%E4%B8%8E%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9" name="6.%20%E6%80%A7%E8%83%BD%E4%BC%98%E5%8C%96%E4%B8%8E%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9">
     <span style="color:#ed7976">
      6. 性能优化与注意事项
     </span>
    </h3>
    <h4 id="6.1%C2%A0%E7%82%B9%E5%87%BB%E5%8C%BA%E5%9F%9F%E4%BC%98%E5%8C%96" name="6.1%C2%A0%E7%82%B9%E5%87%BB%E5%8C%BA%E5%9F%9F%E4%BC%98%E5%8C%96">
     <span style="color:#956fe7">
      6.1 点击区域优化
     </span>
    </h4>
    <p>
     默认最小尺寸为 48x48（Material规范），可通过
     <code>
      minimumSize
     </code>
     调整：
    </p>
    <pre><code class="language-Dart">style: ButtonStyle(
      minimumSize: MaterialStateProperty.all(Size(100, 50)
    ),</code></pre>
    <h4 id="6.2%C2%A0%E9%81%BF%E5%85%8D%E8%BF%87%E5%BA%A6%E9%87%8D%E5%BB%BA" name="6.2%C2%A0%E9%81%BF%E5%85%8D%E8%BF%87%E5%BA%A6%E9%87%8D%E5%BB%BA">
     <span style="color:#956fe7">
      6.2 避免过度重建
     </span>
    </h4>
    <p>
     对静态按钮使用
     <code>
      const
     </code>
     优化：
    </p>
    <pre><code class="language-Dart">const TextButton(
  onPressed: _handleClick,
  child: Text('静态按钮'),
)</code></pre>
    <h4 id="6.3%C2%A0%E6%97%A0%E9%9A%9C%E7%A2%8D%E6%94%AF%E6%8C%81" name="6.3%C2%A0%E6%97%A0%E9%9A%9C%E7%A2%8D%E6%94%AF%E6%8C%81">
     <span style="color:#956fe7">
      6.3 无障碍支持
     </span>
    </h4>
    <p>
    </p>
    <pre><code class="language-Dart">const TextButton(
  onPressed: _handleClick,
  child: Text('静态按钮'),
)</code></pre>
    <h4 id="%C2%A06.4%20%E7%82%B9%E5%87%BB%E6%97%A0%E5%93%8D%E5%BA%94" name="%C2%A06.4%20%E7%82%B9%E5%87%BB%E6%97%A0%E5%93%8D%E5%BA%94">
     <span style="color:#956fe7">
      6.4 点击无响应
     </span>
    </h4>
    <ul>
     <li>
      <p>
       检查
       <code>
        onPressed
       </code>
       是否为 null
      </p>
     </li>
     <li>
      <p>
       确认父组件未被
       <code>
        IgnorePointer
       </code>
       或
       <code>
        AbsorbPointer
       </code>
       包裹
      </p>
     </li>
     <li>
      <p>
       检测是否被其他组件覆盖（如透明层）
      </p>
     </li>
    </ul>
