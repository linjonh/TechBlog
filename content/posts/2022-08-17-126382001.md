    <h2>
     <a id="_0">
     </a>
     需求
    </h2>
    <p>
     在离线环境下，使用anaconda创建环境的时候，需要指定Python的版本，但是无法自动下载，需要手动下载Python包，然后用conda进行安装。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ffaa52820968235c5a0b6a4056b611b9.png"/>
    </p>
    <h2>
     <a id="_3">
     </a>
     解决
    </h2>
    <p>
     首先下载Python包，在
     <code>
      https://repo.anaconda.com/
     </code>
     中选择
     <code>
      Anaconda Packages
     </code>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8fe169c7d8584b919f524d30459abbc.png">
      <br/>
      可以看到里面有Python包也有R包还有Microsoft R Open等一些包，然后选择Python的相关包：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/49691870ce3a496e543d3a454f1329e4.png">
       <br/>
       最终打开的页面：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bf2fb9679de796b3fdffbb73c00647c4.png"/>
      </img>
     </img>
    </p>
    <p>
     也可以通过
     <code>
      https://repo.anaconda.com/pkgs/main/linux-64/
     </code>
     直接打开。
    </p>
    <blockquote>
     <p>
      有时，这个页面比较大，很难打开，无法获取文件的下载链接，可以通过
      <code>
       wget https://repo.anaconda.com/pkgs/main/linux-64/
      </code>
      来将页面下载到本地，用文本编辑器打开，然后获取到文件名，通过URL：
      <code>
       https://repo.anaconda.com/pkgs/main/linux-64/python-3.6.13-h12debd9_1.tar.bz2
      </code>
      来下载。
     </p>
    </blockquote>
    <p>
     这里我选择了
     <code>
      python-3.6.13-h12debd9_1.tar.bz2
     </code>
     , 同时安装pip,
     <code>
      pip-20.3-py36h06a4308_0.tar.bz2
     </code>
    </p>
    <ol>
     <li>
      在
      <code>
       ~/anaconda3/envs/
      </code>
      目录下，新建一个空文件夹，目录名为环境名，使用
      <code>
       conda env list
      </code>
      查看环境列表时并不会显示该环境名称；这里我创建一个test环境名。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9ea1112db73042cd8f0605268ec6be77.png"/>
     </li>
     <li>
      但是可以进入新建的环境：使用
      <code>
       conda activate +环境名进入新环境
      </code>
      ；
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d2f7f37276cba730a72538afa69596e5.png">
       <br/>
       <strong>
        有时候提示
        <code>
         Use 'conda create' to convert the directory to a conda environment.
        </code>
        这时需要通过
        <code>
         conda create --name test --offline
        </code>
        来创建一个空环境
       </strong>
      </img>
     </li>
     <li>
      然后在当前环境下安装Python：
      <code>
       conda install python-3.6.13-h12debd9_1.tar.bz2
      </code>
     </li>
    </ol>
    <blockquote>
     <p>
      有时候会报错：
      <br/>
      requests.exceptions.ConnectionError: HTTPSConnectionPool(host=‘repo.anaconda.com’, port=443): Max retries exceeded with url: /pkgs/main/notices.json (Caused by NewConnectionError(‘&lt;urllib3.connection.HTTPSConnection object at 0x7fb9a8912dc0&gt;: Failed to establish a new connection: [Errno 101] Network is unreachable’))
      <br/>
      可以使用命令
      <code>
       conda install python-3.6.13-h12debd9_1.tar.bz2 --offline
      </code>
      来解决
     </p>
    </blockquote>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/eeaf9d2f38420f6529461d73a7a914fd.png">
      <br/>
      4. 在新的环境中可以看到安装的Python
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/234dbe0cca9f1136761fbfe3b9592851.png">
       <br/>
       5. 安装pip，
       <code>
        conda install pip-20.3-py36h06a4308_0.tar.bz2
       </code>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e50996b2383c743d73c878633156fe6a.png">
        <br/>
        6. 此时安装的pip位于新建环境中，输入pip --version 查看pip应位于新建环境目录下。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f79b4593e0946f2e4790667ffb66632c.png"/>
       </img>
      </img>
     </img>
    </p>
    <h2>
     <a id="conda_36">
     </a>
     conda查看依赖
    </h2>
    <p>
     我们创建一个离线环境：
     <code>
      conda create --name py3.10_test --offline
     </code>
     ，并且切换到这个环境：
     <code>
      source activate py3.10_test
     </code>
    </p>
    <p>
     安装python，但是我们现在不知道下载python哪个版本，也不知道对应的依赖，应该怎么办？
    </p>
    <p>
     解决办法：
    </p>
    <p>
     使用
     <code>
      conda install --download-only python==3.10
     </code>
     ，可以下载python3.10版本。
    </p>
    <blockquote>
     <p>
      可以将所需要下载的依赖包，保存到requirements.txt文件中 conda install --download-only --no-deps --file ./requirements.txt;
     </p>
    </blockquote>
    <p>
     控制台输出如下：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9ef021d8d7b68a7a12dd59865f25eae2.png"/>
     <br/>
     上面列出了
     <code>
      python 3.10
     </code>
     所依赖的包。依次在
     <code>
      https://repo.anaconda.com/pkgs/main/linux-aarch64/
     </code>
     下载即可。
    </p>
    <p>
     将下载的文件拷贝到离线环境中，执行命令：
     <code>
      conda install --offline ./*.tar.bz2
     </code>
     可以安装所有的依赖。
    </p>
