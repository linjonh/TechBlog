    <h3>
     <a id="Literature_Review_16">
     </a>
     Literature Review
    </h3>
    <p>
     这个demo通过下面的Agent组成的Team对一个领域的论文进行总结并形成文献综述：
    </p>
    <ul>
     <li>
      <code>
       Arxiv Search Agent
      </code>
      ：使用
      <code>
       Arxiv API
      </code>
      搜索指定议题的关联论文并返回；
     </li>
     <li>
      <code>
       Google Search Agent
      </code>
      ：使用
      <code>
       Google Search api
      </code>
      搜索指定议题的关联论文并返回；
     </li>
     <li>
      <code>
       Report Agent
      </code>
      ：根据上面两个Agent搜索到的结果汇总并形成报告；
     </li>
    </ul>
    <p>
     首先要安装依赖：
    </p>
    <pre><code class="prism language-python">$ pip install arxiv
</code></pre>
