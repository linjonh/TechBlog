    <h3>
     <a id="_14">
     </a>
     老司机教你三招保平安
    </h3>
    <h4>
     <a id="5_16">
     </a>
     第一招：关门上锁（5分钟搞定）
    </h4>
    <h5>
     <a id="Windows_18">
     </a>
     Windows用户
    </h5>
    <pre><code class="prism language-python"><span class="token number">1.</span> 右键【此电脑】→【属性】→【高级系统设置】
<span class="token number">2.</span> 新增环境变量：
   <span class="token operator">-</span> 名称：OLLAMA_HOST
   <span class="token operator">-</span> 值：<span class="token number">127.0</span><span class="token number">.0</span><span class="token number">.1</span><span class="token punctuation">:</span><span class="token number">11434</span>
<span class="token number">3.</span> 重启时记得用任务管理器彻底关闭进程
</code></pre>
    <h5>
     <a id="Linux_28">
     </a>
     Linux用户
    </h5>
    <p>
     编辑服务文件（一般在/etc/systemd/system/ollama.service），加一行：
    </p>
    <pre><code class="prism language-python">Environment<span class="token operator">=</span><span class="token string">"OLLAMA_HOST=127.0.0.1:11434"</span>
</code></pre>
    <p>
     然后重启ollama：
    </p>
    <pre><code class="prism language-python">sudo systemctl daemon<span class="token operator">-</span><span class="token builtin">reload</span>
sudo systemctl restart ollama
</code></pre>
    <h4>
     <a id="_43">
     </a>
     第二招：设置防火墙
    </h4>
    <h5>
     <a id="Windows_45">
     </a>
     Windows
    </h5>
    <p>
     打开"Windows Defender 防火墙" &gt; “高级设置” &gt; “入站规则”，新建规则：
    </p>
    <ul>
     <li>
      类型：端口
     </li>
     <li>
      协议：TCP
     </li>
     <li>
      端口：11434
     </li>
     <li>
      操作：阻止
     </li>
    </ul>
    <h5>
     <a id="Linux_54">
     </a>
     Linux
    </h5>
    <p>
     用ufw简单粗暴：
    </p>
    <pre><code class="prism language-python">sudo ufw deny <span class="token number">11434</span>
sudo ufw <span class="token builtin">reload</span>
</code></pre>
