    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <pre>在进行嵌入式开发的过程中，Linux系统是嵌入式操作系统的常用选择。那么什么是Linux呢？Linux是一套免费使用和自由传播的类Unix操作系统，其目的是建立不受任何商品化软件版权制约、可以自由使用的Unix兼容产品。</pre>
     <pre>Linux的出现，最早开始于一位名叫Linus Torvalds的芬兰大学生，其目的是想设计出一个代替Minix的操作系统。他设计的操作系统可用于386、486或奔腾处理器的个人计算机上，并且具有Unix操作系统的全部功能。</pre>
     <pre>Linux的发展离不开GNU。GNU计划又称革奴计划，由Richard Stallman在1983年9月27日公开发起的，其目标是创建一套完全自由的操作系统。GNU计划开发出如GCC、GDB、Bash Shell等高质量软件，为Linux开发创造了基本环境，故Linux严格来说应该被称为GNU/Linux。</pre>
     <pre>早期Linux主要被用于服务器，传统上有以Linux为基础的“LAMP”（Linux、Apache、MySQL，Perl/PHP/Python的组合），提供了包括操作系统、数据库 、网站服务器、动态网页的一整套网站架设支持。如此广泛的应用可以源于它的几个特点：</pre>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（1）自由软件；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（2）真正的多用户、多任务操作系统；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（3）极强的平台可伸缩性；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（4）提供强大的管理功能；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（5）完全符合POSIX标准；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（6）具有丰富的图形用户界面；</pre>
     </blockquote>
     <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
      <pre>（7）具有强大的网路功能。</pre>
     </blockquote>
    </blockquote>
    <blockquote style="margin:0 0 0 40px; border:none; padding:0px">
     <pre>一个典型的Linux发行版本包括：Linux内容、GNU程序库和工具、命令行shell、桌面环境。目前比较受欢迎的发行版有Debian、Redhat、Ubuntu、Suse、Fedora</pre>
    </blockquote>
