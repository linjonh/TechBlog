    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     下面为你详细介绍如何创建一个简单有趣的 Django 项目——博客系统。这个项目允许用户创建、查看、编辑和删除博客文章。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 1：环境准备
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     首先，确保你已经安装了 Python 和 pip。然后，创建一个虚拟环境并激活它，接着安装 Django。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     bash
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 创建虚拟环境
    </p>
    <pre><code class="language-bash">python -m venv myblogenv</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 激活虚拟环境
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 在 Windows 上
    </p>
    <pre><code class="language-bash">myblogenv\Scripts\activate</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 在 macOS/Linux 上
    </p>
    <pre><code class="language-bash">source myblogenv/bin/activate</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 安装 Django
    </p>
    <pre><code class="language-bash">pip install django</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 2：创建 Django 项目和应用
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     bash
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 创建 Django 项目
    </p>
    <pre><code class="language-bash">django-admin startproject myblogproject</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 进入项目目录
    </p>
    <pre><code class="language-bash">cd myblogproject</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     # 创建应用
    </p>
    <pre><code class="language-bash">python manage.py startapp blogapp</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 3：配置项目
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     打开 `myblogproject/settings.py` 文件，将 `blogapp` 添加到 `INSTALLED_APPS` 列表中：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     python
    </p>
    <pre><code class="language-python">INSTALLED_APPS = [

    'django.contrib.admin',

    'django.contrib.auth',

    'django.contrib.contenttypes',

    'django.contrib.sessions',

    'django.contrib.messages',

    'django.contrib.staticfiles',

    'blogapp',  # 添加这一行

]</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 4：定义模型
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     打开 `blogapp/models.py` 文件，定义博客文章的模型：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     python
    </p>
    <pre><code class="language-python">from django.db import models



class BlogPost(models.Model):

    title = models.CharField(max_length=200)

    content = models.TextField()

    pub_date = models.DateTimeField(auto_now_add=True)



    def __str__(self):

        return self.title</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 5：创建数据库迁移文件并应用迁移
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     bash
    </p>
    <pre><code class="language-bash">python manage.py makemigrations

python manage.py migrate</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 6：创建视图
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     打开 `blogapp/views.py` 文件，编写视图函数：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     python
    </p>
    <pre><code class="language-python">from django.shortcuts import render, redirect, get_object_or_404

from .models import BlogPost

from .forms import BlogPostForm



def post_list(request):

    posts = BlogPost.objects.all().order_by('-pub_date')

    return render(request, 'blogapp/post_list.html', {'posts': posts})



def post_detail(request, pk):

    post = get_object_or_404(BlogPost, pk=pk)

    return render(request, 'blogapp/post_detail.html', {'post': post})



def post_create(request):

    if request.method == 'POST':

        form = BlogPostForm(request.POST)

        if form.is_valid():

            form.save()

            return redirect('post_list')

    else:

        form = BlogPostForm()

    return render(request, 'blogapp/post_form.html', {'form': form})



def post_edit(request, pk):

    post = get_object_or_404(BlogPost, pk=pk)

    if request.method == 'POST':

        form = BlogPostForm(request.POST, instance=post)

        if form.is_valid():

            form.save()

            return redirect('post_detail', pk=post.pk)

    else:

        form = BlogPostForm(instance=post)

    return render(request, 'blogapp/post_form.html', {'form': form})



def post_delete(request, pk):

    post = get_object_or_404(BlogPost, pk=pk)

    post.delete()

    return redirect('post_list')</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 7：创建表单
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在 `blogapp` 目录下创建 `forms.py` 文件：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     python
    </p>
    <pre><code class="language-python">from django import forms

from .models import BlogPost



class BlogPostForm(forms.ModelForm):

    class Meta:

        model = BlogPost

        fields = ['title', 'content']</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 8：配置 URL
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     打开 `blogapp/urls.py` 文件（如果不存在则创建）：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Python
    </p>
    <pre><code class="language-python">from django.urls import path

from .views import post_list, post_detail, post_create, post_edit, post_delete



urlpatterns = [

    path('', post_list, name='post_list'),

    path('&lt;int:pk&gt;/', post_detail, name='post_detail'),

    path('new/', post_create, name='post_create'),

    path('&lt;int:pk&gt;/edit/', post_edit, name='post_edit'),

    path('&lt;int:pk&gt;/delete/', post_delete, name='post_delete'),

]</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     然后在 `myblogproject/urls.py` 中包含 `blogapp` 的 URL 配置：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     python
    </p>
    <pre><code class="language-python">from django.contrib import admin

from django.urls import path, include



urlpatterns = [

    path('admin/', admin.site.urls),

    path('', include('blogapp.urls')),

]</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 9：创建模板
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在 `blogapp` 目录下创建 `templates` 目录，再在 `templates` 目录下创建 `blogpost.pk。在 `blogapp/templates/blogapp` 目录下创建以下模板文件：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     #### `post_list.html`
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     html
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;

&lt;html lang="en"&gt;

&lt;head&gt;

    &lt;meta charset="UTF-8"&gt;

    &lt;title&gt;Blog Posts&lt;/title&gt;

&lt;/head&gt;

&lt;body&gt;

    &lt;h1&gt;Blog Posts&lt;/h1&gt;

    &lt;!-- 假设视图名为 post_create，正确创建新文章链接 --&gt;

    &lt;a href="{% url 'post_create' %}"&gt;New Post&lt;/a&gt;

    &lt;!-- 假设 posts 是包含博客文章的列表 --&gt;

    {% for post in posts %}

        &lt;h2&gt;&lt;a href="{% url 'post_detail' post.pk %}"&gt;{<!-- -->{ post.title }}&lt;/a&gt;&lt;/h2&gt;

        &lt;p&gt;{<!-- -->{ post.content|post.pkpost.pkewords:20 }}&lt;/p&gt;

        &lt;p&gt;Published on {<!-- -->{ post.pub_date }}&lt;/p&gt;

        &lt;a href="{% url 'post_edit' post.pk %}"&gt;Edit&lt;/a&gt;

        &lt;a href="{% url 'post_delete' post.pk %}"&gt;Delete&lt;/a&gt;

    {% endfor %}

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     #### `post_detail.html`
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     html
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;

&lt;html lang="en"&gt;

&lt;head&gt;

    &lt;meta charset="UTF-8"&gt;

    &lt;title&gt;{<!-- -->{ post.title }}&lt;/title&gt;

&lt;/head&gt;

&lt;body&gt;

    &lt;h1&gt;{<!-- -->{ post.title }}&lt;/h1&gt;

    &lt;p&gt;{<!-- -->{ post.content }}&lt;/p&gt;

    &lt;p&gt;Published on {<!-- -->{ post.pub_date }}&lt;/p&gt;

    &lt;a href="{% url 'post_edit' post.pk %}"&gt;Edit&lt;/a&gt;

    &lt;a href="{% url 'post_delete' post.pk %}"&gt;Delete&lt;/a&gt;

    &lt;a href="{% url 'post_list' %}"&gt;Back to List&lt;/a&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     #### `post_form.html`
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     html
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;

&lt;html lang="en"&gt;

&lt;head&gt;

    &lt;meta charset="UTF-8"&gt;

    &lt;title&gt;{% if form.instance.pk %}Edit Post{% else %}New Post{% endif %}&lt;/title&gt;

&lt;/head&gt;

&lt;body&gt;

    &lt;h1&gt;{% if form.instance.pk %}Edit Post{% else %}New Post{% endif %}&lt;/h1&gt;

    &lt;form method="post"&gt;

        {% csrf_token %}

        {<!-- -->{ form.as_p }}

        &lt;button type="submit"&gt;Save&lt;/button&gt;

    &lt;/form&gt;

    &lt;a href="{% url 'post_list' %}"&gt;Cancel&lt;/a&gt;

&lt;/body&gt;

&lt;/html&gt;</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     步骤 10：运行项目
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     bash
    </p>
    <pre><code class="language-bash">python manage.py runserver</code></pre>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     现在，你可以在浏览器中访问 `http://127.0.0.1:8000/` 查看博客系统。你可以创建、查看、编辑和删除博客文章。
    </p>
