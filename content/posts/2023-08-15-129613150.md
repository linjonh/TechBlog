# springboot后端用一个实体类和一个map接收前端怎么传值

Spring Boot后端可以使用一个实体类和一个Map来接收前端传递的值。

首先，在实体类中定义好所需要接收的属性。例如：

```java
public class User {
    private String name;
    private int age;
    // 省略其他代码
}
```
