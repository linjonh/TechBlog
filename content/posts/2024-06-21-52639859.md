# Eclipse IDE for Java Developers没有Server功能

eclipse for java，是针对开发纯java应用程序的eclipse。它是没有servers服务器功能的。
  
如果需要支持server，要下载eclipse IDE for jee版本的，这个版本的eclipse可以开发J2EE的java项目，支持服务器功能。
