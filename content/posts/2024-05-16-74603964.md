    <p>
     项目里有需求就是根据用户说的话,打开相应的app.从网上查看了些资料.加以自己的想法.供自己以后学习之用.
    </p>
    <h2 id="有两种情况">
     有两种情况,
    </h2>
    <h2 id="1-手机存在此app">
     (1) 手机存在此app,
    </h2>
    <pre><code> 需要先遍历手机中所有的应用程序包名,通过包名拿到应用的名称.如果和用户说的名称一样.就可以拿到一个属于该应用的intent.然后再开启就可以了.
</code></pre>
    <p>
     public static Intent getIntentByName(Context context, String name){
     <br/>
     //拿到包的管理者
     <br/>
     PackageManager pm = context.getApplicationContext().getPackageManager();
     <br/>
     //得到所有手机中APP的信息
     <br/>
     List allApps = pm.getInstalledPackages(0);
     <br/>
     Intent intent = null;
     <br/>
     //遍历所有的app信息,查看是否有相等的应用名称,有的话,就返回一个intent.
     <br/>
     if (null != allApps &amp;&amp; null != name) {
     <br/>
     for (PackageInfo pi : allApps) {
     <br/>
     if (name.equals(pi.applicationInfo.loadLabel(pm))) {
     <br/>
     intent = pm.getLaunchIntentForPackage(pi.packageName);
     <br/>
     break;
     <br/>
     }
     <br/>
     }
     <br/>
     }
     <br/>
     //通过返回的Intent开启app
     <br/>
     public void startApp(Context context, String name){
     <br/>
     if(null != name ){
     <br/>
     Intent intent = getIntentByName(context,name);
     <br/>
     intent.setFlags(FLAG_ACTIVITY_NEW_TASK);
     <br/>
     context.startActivity(intent);
     <br/>
     }
     <br/>
     }
    </p>
    <h2 id="2手机不存在此app">
     (2)手机不存在此App
    </h2>
    <pre><code>这里面又可以分为两种情况,
第一种,有市面上大部分应用的app的包名的数据库,当遍历完手机中的app没有查找到符合条件的时候,从这个数据库中查找对应应用程序的包名.找到过后可以通过包名直接打开应用宝对应应用的详情界面.
</code></pre>
    <pre class="prettyprint"><code class="hljs cs">    Uri uri = Uri.parse(<span class="hljs-string">"market://details?id="</span> + packageName);
    Intent intent = <span class="hljs-keyword">new</span> Intent(Intent.ACTION_VIEW, uri);
    <span class="hljs-comment">//然后和上面的步骤一样,开启Activity就可以了</span></code></pre>
    <pre><code>但是,这种需要的数据库可能有点大,比较占内存.

第二种.不进入具体的应用详情界面,而是根据用户指定的名称来跳转到应用宝的搜索界面,属于模糊定位.
</code></pre>
    <pre class="prettyprint"><code class="hljs cs">     Uri uri = Uri.parse(<span class="hljs-string">"market://search?q=pub:+"</span> + name);
    Intent intent = <span class="hljs-keyword">new</span> Intent(Intent.ACTION_VIEW, uri);
    <span class="hljs-comment">//也是开启Activity就可以了</span></code></pre>
    <p>
     推荐使用第二种,因为市面上的应用太多了,还是交给应用宝本身去搜索比较好.
    </p>
