# 3ds Max 鼠标与快捷键组合操作指南

以下是
**3ds Max 鼠标与快捷键组合操作指南**
，按功能分类整理，涵盖
**视图控制、对象操作、建模工具**
等核心场景。自用记录，不喜勿喷。

---

#### **一、视图控制（鼠标核心操作）**

| **操作** | **快捷键+鼠标动作** | **功能说明** |
| --- | --- | --- |
| **平移视图** | 鼠标中键拖动 | 按住中键平移视图（无需快捷键） |
| **旋转视图** | Alt + 鼠标中键拖动 | 以当前视角为中心自由旋转（透视图/正交视图） |
| **缩放视图** | 鼠标滚轮滚动 | 快速缩放视图（或按住 Ctrl + Alt + 中键拖动） |
| **最大化视图切换** | Alt + W | 结合鼠标：点击视图右上角最大化按钮（或双击视图标签） |
| **聚焦选中对象** | Z | 选中对象后按 Z，鼠标滚轮可调整视图远近 |

---

#### **二、对象操作**

| **操作** | **快捷键+鼠标动作** | **功能说明** |
| --- | --- | --- |
| **多选对象** | Ctrl + 鼠标左键单击 | 累加选择多个对象 |
| **框选模式切换** | Q（选择工具） + 鼠标拖动 | 默认矩形框选，按住 Ctrl 切换圆形/多边形框选 |
| **快速克隆对象** | Shift + 鼠标拖动（移动/旋转/缩放） | 拖动对象时按住 Shift，松开后选择克隆方式（复制/实例/参考） |
| **对齐工具** | Alt + A（对齐） + 鼠标点击目标对象 | 先选源对象，按 Alt + A 后点击目标对象，设置对齐轴向 |
| **父子级链接** | 鼠标拖动到父对象 | 选中子对象，点击工具栏链接图标（或按 Ctrl + L），拖拽连线到父对象 |

---

#### **三、建模工具（多边形/样条线）**

| **操作** | **快捷键+鼠标动作** | **功能说明** |
| --- | --- | --- |
| **顶点/边/面层级切换** | 1/2/3/4/5 + 鼠标选择 | 按数字键切换层级，鼠标直接点击或框选子对象 |
| **快速切割** | Alt + C + 鼠标点击连线 | 进入切割模式后，单击起点和终点创建新边 |
| **挤出操作** | Ctrl + E + 鼠标拖动方向 | 选中面/边后，按住 Ctrl + E 拖动鼠标控制挤出高度 |
| **环形/循环边选择** | Alt + 鼠标双击边 | 双击一条边可选中环形边（多边形建模）；Alt + 双击选中循环边 |
| **目标焊接顶点** | Alt + Shift + T + 鼠标拖拽 | 拖动一个顶点到另一个顶点上完成焊接 |
| **快速切分** | Ctrl + Shift + E + 鼠标划线 | 在模型表面直接划线切分（支持对称操作） |

---

#### **四、变换操作（移动/旋转/缩放）**

| **操作** | **快捷键+鼠标动作** | **功能说明** |
| --- | --- | --- |
| **轴向锁定** | F5/F6/F7（X/Y/Z 轴锁定） | 按 F5 锁定 X 轴后，鼠标拖动仅沿该轴移动 |
| **百分比缩放** | R（缩放工具） + 鼠标右键拖动 | 右键拖动缩放轴可输入精确百分比（如 50%） |
| **旋转角度捕捉** | A（角度捕捉开关） + 鼠标旋转 | 按 A 启用角度捕捉（默认 5°），旋转时按 Ctrl 微调角度 |
| **坐标系切换** | X + 鼠标点击视图 | 按 X 切换世界/局部/屏幕坐标系，拖动时自动吸附到网格或对象 |

---

#### **五、材质与渲染**

| **操作** | **快捷键+鼠标动作** | **功能说明** |
| --- | --- | --- |
| **材质拖拽赋予** | 材质球拖拽到对象 | 在材质编辑器（M）中，直接拖动材质球到视口中的对象上 |
| **实时渲染区域** | Shift + Q + 鼠标框选区域 | 框选视图区域，仅渲染该部分（节省时间） |
| **环境光调整** | 8（环境面板） + 鼠标拖动参数 | 直接拖动数值栏或色块调整环境光颜色/强度 |

---

#### **六、实用技巧**

1. **右键菜单加速**

   * 按住 Alt + 右键：快速呼出
     **变换工具菜单**
     （移动/旋转/缩放）。
   * 按住 Ctrl + 右键：呼出
     **创建基本体菜单**
     。
2. **专家模式**

   * 按 Ctrl + X 进入全屏模式，鼠标悬停视图边缘可快速切换布局。
3. **自定义快捷键+鼠标**

   * 通过
     **Customize > Customize User Interface**
     ，将高频操作（如布尔运算、对称修改器）绑定到 Shift + 鼠标侧键。
