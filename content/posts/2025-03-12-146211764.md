       <td>

        </strong>
       </td>
       <td>
        Alt + Shift + T + 鼠标拖拽
       </td>
       <td>
        拖动一个顶点到另一个顶点上完成焊接
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         快速切分
        </strong>
       </td>
       <td>
        Ctrl + Shift + E + 鼠标划线

       <td>
        在模型表面直接划线切分（支持对称操作）

    <h4>

     </a>
     <strong>
      六、实用技巧
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        右键菜单加速
       </strong>
      </p>
      <ul>
       <li>
        按住 Alt + 右键：快速呼出
        <strong>
         变换工具菜单
        </strong>
        （移动/旋转/缩放）。
       </li>
       <li>
        按住 Ctrl + 右键：呼出
        <strong>
         创建基本体菜单
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        专家模式
       </strong>
      </p>
      <ul>
       <li>
        按 Ctrl + X 进入全屏模式，鼠标悬停视图边缘可快速切换布局。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        自定义快捷键+鼠标
       </strong>
      </p>

        <strong>
         Customize &gt; Customize User Interface
        </strong>
        ，将高频操作（如布尔运算、对称修改器）绑定到 Shift + 鼠标侧键。
       </li>
      </ul>
     </li>
