    <h3>
     <a id="1__0">
     </a>
     1. 启动控制脚本
    </h3>
    <p>
     远程作到 Raspberry Pi 中，并运行以下脚本：
    </p>
    <pre><code class="prism language-bash">conda activate lerobot

python lerobot/scripts/control_robot.py <span class="token punctuation">\</span>
  <span class="token parameter variable">--robot.type</span><span class="token operator">=</span>lekiwi <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.type</span><span class="token operator">=</span>remote_robot
</code></pre>
    <p>
     登录笔记本电脑上，同时运行以下脚本：
    </p>
    <pre><code class="prism language-bash">conda activate lerobot

python lerobot/scripts/control_robot.py <span class="token punctuation">\</span>
  <span class="token parameter variable">--robot.type</span><span class="token operator">=</span>lekiwi <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.type</span><span class="token operator">=</span>teleoperate <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.fps</span><span class="token operator">=</span><span class="token number">30</span>
</code></pre>
