    <h2>
     <a id="1__0">
     </a>
     题目1 股票买卖
    </h2>
    <p>
     给定一个长度为 N 的数组，数组中的第 i 个数字表示一个给定股票在第 i 天的价格。
    </p>
    <p>
     设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票）。
    </p>
    <p>
     注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。
    </p>
    <h5>
     <a id="_7">
     </a>
     输入格式
    </h5>
    <p>
     第一行包含整数 N，表示数组长度。
    </p>
    <p>
     第二行包含 N 个不大于 10000 的正整数，表示完整的数组。
    </p>
    <h5>
     <a id="_13">
     </a>
     输出格式
    </h5>
    <p>
     输出一个整数，表示最大利润。
    </p>
    <h5>
     <a id="_17">
     </a>
     数据范围
    </h5>
    <p>
     1≤N≤105
    </p>
    <h5>
     <a id="1_21">
     </a>
     输入样例1：
    </h5>
    <pre><code>6
7 1 5 3 6 4
</code></pre>
    <h5>
     <a id="1_28">
     </a>
     输出样例1：
    </h5>
    <pre><code>7
</code></pre>
    <h5>

    </h5>

</code></pre>
    <h5>

    </h5>

</code></pre>
    <h5>

    </h5>

</code></pre>
    <h5>
     <a id="3_54">
     </a>
     输出样例3：
    </h5>
    <pre><code>0
</code></pre>
    <h5>
     <a id="_60">
     </a>
     样例解释
    </h5>
    <p>
     样例1：在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5-1 = 4 。随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6-3 = 3 。共得利润 4+3 = 7。
    </p>
    <p>
     样例2：在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5-1 = 4 。注意你不能在第 1 天和第 2 天接连购买股票，之后再将它们卖出。因为这样属于同时参与了多笔交易，你必须在再次购买前出售掉之前的股票。
    </p>
    <p>
     样例3：在这种情况下, 不进行任何交易, 所以最大利润为 0。
    </p>

     思路

    <p>
     蓝桥杯笔记：
     <a href="https://mbd.pub/o/bread/ZpWbmJ9x" rel="nofollow">
      蓝桥杯备赛笔记

    <p>
     很多贪心题目一下子真的很难想出来，怎么办？找规律
     <br/>
     首先保证分类情况没有重复没有遗漏
    </p>
    <ol>
     <li>
      全是正数
      <br/>
      1 2 3 4 5
     </li>
    </ol>
    <ul>
     <li>
      n=4,m=0:全部相加
     </li>
     <li>
      n=3,m=1:5+4+3+2-1
     </li>
     <li>
      n=2,m=2:5+4+3-(1-2)
     </li>
     <li>
      n=1,m=3:5+4-(1-2-3)
     </li>
     <li>
      n=0,m=4:2-(1-5-4-3)
     </li>
    </ul>
    <ol start="2">
     <li>
      全是负数
      <br/>
      -5 -4 -3 -2 -1
     </li>
    </ol>
    <ul>
     <li>
      n=4,m=0:全部相加
     </li>
     <li>
      n=3,m=1:(-1)-{(-5)+(-4)+(-3)+(-2)}
     </li>
     <li>
      n=2,m=2:(-1)-{(-5)+(-4)+(-3)}-(-2)
     </li>
     <li>
      n=1,m=3:(-1)-{(-5)+(-4)}-(-3)-(-2)
     </li>
     <li>
      n=0,m=4:(-1)-(-5)-(-4)-(-3)-(-2)
     </li>
    </ul>
    <ol start="3">
     <li>
      有正有负
      <br/>
      -5 -4 -3 2 1
     </li>
    </ol>
    <ul>
     <li>
      n=4,m=0:全部相加
     </li>
     <li>
      n=3,m=1:(1)-{(-5)+(-4)+(-3)+(2)}
     </li>
     <li>
      n=2,m=2:(1)-{(-5)+(-4)}-(-3)+(2)
     </li>
     <li>
      n=1,m=3:(1)-{(-5)}-(-4)-(-3)+2
     </li>
     <li>
      n=0,m=4:(1)-(-5)-(-4)-(-3)-(-2)
     </li>
    </ul>
    <p>
     不知道你找到规律没？关键在于 减号的数量
    </p>
    <ul>
     <li>
      减号数量为0，所有元素相加
     </li>
     <li>
      减号数量不为0，列表中最大元素-最小元素+其余每一个元素的绝对值
     </li>
    </ul>
    <pre><code class="prism language-Python3">import os
import sys
n,m=map(int,input().split())
data=list(map(int,input().split()))
data.sort()
ans=0
if m==0:#一个负号都不存在
    ans=sum(data)
else:#至少存在一个负号
    ans=data[-1]-data[0]
    for i in range(1,len(data)-1):
        ans+=abs(data[i])
print(ans)
