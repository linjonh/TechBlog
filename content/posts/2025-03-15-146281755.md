    <p>
     正则表达式（Regular Expression，简称 regex 或 regexp）是一种用于描述文本模式的字符串。它通过特定的语法规则，允许你匹配、搜索和操作字符串中的内容。
    </p>
    <p>
     正则表达式可以用来：
    </p>
    <ol>
     <li>
      <strong>
       匹配文本
      </strong>
      ：可以在字符串中查找符合特定模式的文本，如查找邮件地址、电话号码等。
     </li>
     <li>
      <strong>
       文本替换
      </strong>
      ：能够使用匹配到的模式对文本进行替换操作，比如将所有的逗号替换为句号。
     </li>
     <li>
      <strong>
       数据验证
      </strong>
      ：用于验证输入数据的格式，例如检查用户输入的密码是否符合特定规则。
     </li>
     <li>
      <strong>
       分割字符串
      </strong>
      ：可以根据匹配的模式对字符串进行分割，例如根据空格、逗号等分隔符将词句拆分成单词。
     </li>
    </ol>
    <p>
    </p>
    <h2 style="background-color:transparent">
     正则表达式的通用语法
    </h2>
    <p>
     正则表达式由普通字符（如字母和数字）和特殊字符（用于匹配特定类型的字符或模式）组成。
    </p>
    <blockquote>
     <p>
      <strong>
       -- 匹配单个字符
      </strong>
      <br/>
      "snm":包含 'snm' 即可匹配
      <br/>
      "s|n|m":包含's' or 'n' or 'm'即可匹配
      <br/>
      "[snm]":包含's' or 'n' or 'm'即可匹配
      <br/>
      "[a-z]":包含字母a-z任意一个即可匹配
      <br/>
      "[1-9]":包含数字1-9任意一个即可匹配
      <br/>
      '\\':转义字符
      <br/>
      '.' :匹配任意单个字符
     </p>
     <p>
     </p>
     <p>
      <strong>
       -- 匹配字符类
      </strong>
      <br/>
      [[:alnum:]] : 任意字母和数字=&gt;[a-zA-Z0-9]
      <br/>
      [[:digit:]] : 任意数字=&gt;[0-9]
      <br/>
      [[:lower:]] : 任意小写字母=&gt;[a-z]
      <br/>
      [[:upper:]] : 任意大写字母=&gt;[A-Z]
      <br/>
      [[:alpha:]] : 任意字母 =&gt;[a-zA-Z]
      <br/>
      [[:xdigit:]] : 任意十六进制数字=&gt;[a-fA-F0-9]
      <br/>
      [[:blank:]] : 空格和制表符 =&gt;[\\t]
      <br/>
      [[:space:]] : 包括空格在内的任意的空白字符=&gt;[\\f\\n\\r\\t\\v]
     </p>
     <p>
      [[:cntrl:]] : ASCII控制字符(ASCII 0到31 和 127)
      <br/>
      [[:print:]] : 任意的可打印出来的字符
      <br/>
      [[:graph:]] : 相对于print，不匹配空格
      <br/>
      [[:punct:]] : 非字母、非数字、非ASCII控制字符
     </p>
     <p>
     </p>
     <p>
      <strong>
       -- 匹配多个字符
      </strong>
      <br/>
      '^S' :必须以S开始
      <br/>
      'S$' :必须以S结尾
      <br/>
      '^S$':只能是S
     </p>
     <p>
     </p>
     <p>
      * 任意个匹配"S*"任意个(可以0)S即可匹配
      <br/>
      + 任意个匹配"S*"任意个(至少1)S即可匹配
      <br/>
      ? 0个或1个
     </p>
     <p>
      <br/>
      {n}  指定n个
      <br/>
      {n,} 至少n个
      <br/>
      {n,m} n到m个
     </p>
    </blockquote>

     MySQL数据库中的正则表达式
    </h2>
    <pre><code class="language-sql">-- regexp为MySQL中正则表达式的关键字

regexp "正则表达式";</code></pre>
    <h2>
     C++中的正则表达式
    </h2>
    <h4>
     完全匹配match
    </h4>
    <pre><code class="language-cpp">int main() {
	regex re_num("[0-9]*");//用于匹配任意数字串
	cout &lt;&lt; regex_match("464664841650", re_num) &lt;&lt; endl;
	cout &lt;&lt; regex_match("4646648a41650", re_num) &lt;&lt; endl;
	return 0;
}</code></pre>

    <h4>
     全文搜索search
    </h4>
    <pre><code class="language-cpp">int main() {
	regex re_num("[0-9]*");//用于匹配任意数字串
	bool isfind = regex_search("askdgkasg123asdf",re_num);
	cout &lt;&lt; isfind &lt;&lt; endl;
	return 0;
}</code></pre>

     文本替换replace
    </h4>
    <pre><code class="language-cpp">int main() {
	regex re_num("[0-9]+");//用于匹配任意数字串
	string s = "123abc";
	cout&lt;&lt; regex_replace(s, re_num, "kkk");
	return 0;
}</code></pre>
