    /* 
     * 设置背景颜色为半透明的青色

     3.1 background
    </h6>
    <p>
     简写属性：background:background-color|background-image|background-repeat|background-position

    <p>
     以空格隔开，书写顺序没有要求
    </p>
    <pre><code>/* 选择具有 'bg-example' 类的元素 */
.bg-example {
    /* 
     * 使用 background-color 属性设置背景颜色为淡蓝色 
     * 十六进制颜色代码 #e6f7ff 代表淡蓝色
     */
    background-color: #e6f7ff;

    /* 
     * 使用 background-image 属性设置背景图片 
     * 这里假设 'background.jpg' 是你项目中的图片文件，路径根据实际情况调整
     */
    background-image: url('background.jpg');

    /* 
     * 使用 background-repeat 属性设置背景图片的重复方式 
     * 'no-repeat' 表示背景图片不重复，只显示一次
     */
    background-repeat: no-repeat;

    /* 
     * 使用 background-position 属性设置背景图片的显示位置 
     * 'center center' 表示将背景图片水平和垂直都居中显示
     */
    background-position: center center;

    /* 
     * 使用 background-attachment 属性设置背景图片是否跟随滚动 
     * 'fixed' 表示背景图片固定，不跟随页面滚动
     */
    background-attachment: fixed;

    /* 
     * 使用 background 简写属性一次性设置多个背景相关属性 
     * 顺序依次为：背景颜色、背景图片、背景重复方式、背景位置、背景附着方式
     * 以下代码效果和上面分开设置的效果相同
     */
    /* background: #e6f7ff url('background.jpg') no-repeat center center fixed; */
}
</code></pre>

     <a id="61_border_325">
     </a>
     6.1 border
    </h6>
    <p>
     表示盒子的边框
    </p>
    <p>
     分为四个方向：上top、右right、下bottom、左left
     <br/>
     border-top、border-right、border-bottom、border-left
    </p>
    <p>
     每个边框包含三种样式：
    </p>
    <p>
     border-top-color、border-top-width、border-top-style
     <br/>
     border-right-color、border-right-width、border-right-style
     <br/>
     border-bottom-color、border-bottom-width、border-bottom-style
     <br/>
     border-left-color、border-left-width、border-left-style
     <br/>
     样式style的取值：
    </p>
    <p>
     solid实线、dashed虚线、dotted点线、double双线、inset内嵌的3D线、outset外嵌的3D线
    </p>
    <p>
     简写，三种方式：
    </p>
    <p>
     按方向简写：
    </p>
    <p>
     border-top、border-right、border-bottom、border-left
    </p>
    <p>
     书写顺序：
    </p>
    <p>
     border-顺序：width style color
    </p>
    <p>
     按样式简写：
    </p>
    <p>
     border-color、border-width、border-style
    </p>
    <p>
     书写顺序：
    </p>
    <p>
     border-样式：top right bottom left
    </p>
    <p>
     必须按顺时针方向书写，同时可以缩写：
    </p>
    <p>
     border-width:2px;---------&gt;四个边框的宽度均为2px
     <br/>
     border-width:1px 2px;
     <br/>
     border-width:1px 2px 4px;
     <br/>
     规则：如果省略，则认为上下一样，左右一样
    </p>
    <p>
     终级简写：
    </p>
    <p>
     如果四个边框样式完全相同，border:width style color;
    </p>
    <h6>
     <a id="62_padding_371">
     </a>
     6.2 padding
    </h6>
    <p>
     表示盒子的内边距，即内容与边框之间的距离
    </p>
    <p>
     同样也分为四个方向，也可以简写（按顺时针方向，默认上下一样，左右一样）
    </p>
    <p>
     注意：如果上下冲突，则以上为准，如果左右冲突，则以左为准
    </p>
    <h6>
     <a id="63_margin_379">
     </a>
     6.3 margin
    </h6>
    <p>
     表示盒子的外边距，即盒子与盒子之间的距离
    </p>
    <p>
     同样也分为四个方向，也可以简写（按顺时针方向，默认上下一样，左右一样）
    </p>
    <h6>
     <a id="64__385">
     </a>
     6.4 元素所占空间
    </h6>
    <p>
     页面中的元素实际所占的空间
    </p>
    <p>
     宽度=width+左右padding+左右border+左右margin
     <br/>
     高度=height+上下padding+上下border+上下margin
    </p>
    <pre><code>p{
			width:250px;
			background:#ccc;
		}
		.first{
			/* border-top-color:red;
			border-top-width:1px;
			border-top-style:solid;
			border-right-color:blue;
			border-right-width:2px;
			border-right-style:dotted;
			border-bottom-color:green;
			border-bottom-width:4px;
			border-bottom-style:dashed;
			border-left-color:gray;
			border-left-width:6px;
			border-left-style:double; */

			/* border-top:1px solid red;
			border-bottom:2px dashed blue; */
			
			/* border-color:red yellow green;
			border-width:1px 2px 4px 6px;
			border-style:solid dashed dotted solid; */

			border:1px solid red;
			padding:20px;
			margin:10px;
		}
		.second{
			/* padding-top:5px;
			padding-left:3px;
			padding-bottom:10px;
			padding-right:8px; */

			/* padding:1px 2px 4px 6px; */

			padding:5px;
		}
		.third{
			/* margin-top:50px;
			margin-left:30px; */

			/* margin:10px 20px; */

			/* 元素的居中 */
			/* 1.块级元素水平居中 */
			margin:auto;
			/* 2.文本水平居中 */
			text-align:center;
			/* 3.文本垂直居中 将height与line-height设置为相同 */
			height:100px;
			line-height:100px;
		}
</code></pre>
    <h5>
     <a id="7_449">
     </a>
     7.定位
    </h5>
    <p>
     通过position属性实现对元素的定位
    </p>
    <table>
     <thead>
      <tr>
       <th align="center">
        取值
       </th>
       <th align="center">
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td align="center">
        static（默认）
       </td>
       <td align="center">
        按照常规文档流进行显示
       </td>
      </tr>
      <tr>
       <td align="center">
        relative
       </td>
       <td align="center">
        相对于标签原来的位置进行的定位
       </td>
      </tr>
      <tr>
       <td align="center">
        absolute
       </td>
       <td align="center">
        相对于第一个非static定位的父标签的定位
       </td>
      </tr>
      <tr>
       <td align="center">
        fixed
       </td>
       <td align="center">
        相对于浏览器窗品进行定位
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     设置定位方式后，还要设置偏移量：top、bottom、left、right
    </p>
    <h6>
     <a id="71__462">
     </a>
     7.1 相对定位
    </h6>
    <p>
     先设置元素的position属性为relative,然后再设置偏移量
    </p>
    <h6>
     <a id="72__466">
     </a>
     7.2 绝对定位
    </h6>
    <p>
     先设置父标签为非static定位，然后设置元素的position属性为absolute，最后再设置偏移量
    </p>
    <p>
     <strong>
      注意：
     </strong>
    </p>
    <ul>
     <li>
      一般来说都会将父标签设置为非static定位
     </li>
     <li>
      如果父标签不是非static定位，则会相对于浏览器窗口进行定位
     </li>
     <li>
      设置元素为绝对定位后，元素会浮到页面上方
     </li>
    </ul>
    <h6>
     <a id="73__476">
     </a>
     7.3 固定定位
    </h6>
    <p>
     先设置元素的position属性为fixed，然后再设置偏移量
    </p>
    <p>
     设置元素为固定定位后，元素会浮动在面面上方
    </p>
    <h6>
     <a id="74_zindex_482">
     </a>
     7.4 z-index
    </h6>
    <p>
     设置元素定位方式后，元素会浮在页面上方，此时可以通过z-index属性设置优先级，控制元素的堆叠顺序
    </p>
    <p>
     取值为数字，值越大优先级越高，默认为auto(大多数浏览器默认为0)
    </p>
    <p>
     **注意：**只能给非static定位的元素设置z-index属性
    </p>
    <pre><code>#container{
			width:800px;
			border:1px solid #000000;
			position:relative;
		}
		.div1,.div2,.div3,.div4{
			width:100px;
			height:50px;
		}
		.div1{
			background:red;
			position:relative;/* 相对定位 */
			top:20px;
			left:50px;
			z-index:-5;
		}
		.div2{
			background:blue;
			position:absolute;
			left:100px;
			bottom:50px;
			z-index:-8;
		}
		.div3{
			background:green;
			position:fixed;
			bottom:50px;
			left:100px;
		}
		.div4{
			background:cyan;
		}
</code></pre>
