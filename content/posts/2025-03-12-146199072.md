    <h3>
     <a id="Stream_45">
     </a>
     Stream流的使用步骤
    </h3>
    <p>
     准备数据源（集合、数组、…）→ 过滤 → 排序 → 去重→ … →获取结果
    </p>
    <ul>
     <li>
      1.
      <strong>
       获取Stream流
      </strong>
      ，Stream流代表着一条流水线，并能与数据源建立连接；
     </li>
     <li>
      2.
      <strong>
       调用流水线各种方法
      </strong>
      ，对数据进行处理；
     </li>
     <li>
      3.
      <strong>
       获取处理的数据
      </strong>
      ，遍历、统计、收集到一个新的集合中返回结果。
     </li>
    </ul>

    </h4>

       </li>
       <li>
        Stream流：方便操作集合/数组的手段
       </li>
      </ul>
     </li>
    </ul>

    <h3>
     <a id="_310">
     </a>
     方法中的可变参数（拓展）
    </h3>
    <ul>
     <li>
      就是一种特殊形参，定义在方法、构造器的形参列表里，格式是:数据类型…参数名称；
     </li>
     <li>
      可变参数的特点和好处
      <ul>
       <li>
        特点：可以不传数据给它；可以传一个或者同时传多个数据给它；也可以传一个数组给它。
       </li>
       <li>
        好处：常常用来灵活的接收数据。
       </li>
      </ul>
     </li>
     <li>
      可变参数的注意事项：
      <ul>
       <li>
        可变参数在方法内部就是一个数组
       </li>
       <li>
        一个形参列表中可变参数只能有一个
       </li>
       <li>
        可变参数必须放在形参列表的最后面
       </li>
      </ul>
     </li>
    </ul>
