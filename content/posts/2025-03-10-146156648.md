    <p>
     <strong>
      引言
     </strong>
     <br/>
     DevEco AI辅助编程工具（CodeGenie）为开发者提供高效的应用/元服务AI辅助编程工具，支持HarmonyOS NEXT领域的智能知识问答、ArkTS代码补全/生成和万能卡片生成能力，提高开发者编码效率。
     <br/>
     <strong>
      一、安装使用
     </strong>
     <br/>
     建议使用DevEco Studio 5.0.3.403及以上版本。
     <br/>
     <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/ide-codegenie-V5" rel="nofollow" title="文档中心">
      文档中心
     </a>
    </p>
    <p>
     <strong>
      二、新建项目
     </strong>
     <br/>
     项目构建：
     <br/>
     若首次打开DevEco Studio，请点击Create Project创建工程。如果已经打开了一个工程，请在菜单栏选择File &gt; New &gt; Create Project来创建一个新工程。
     <br/>
    </p>

     <br/>
     进入配置工程界面，Compatible SDK表示兼容的最低API Version，此处以选择“5.0.0(12)”为例，其他参数保持默认设置即可

    </p>

    <p>
     <br/>
     点击Finish，工具会自动生成示例代码和相关资源，等待工程创建完成。
    </p>

     Poetry01相关代码说明
     <br/>
     目录结构
     <br/>
    </p>
    <p class="img-center">
     <img alt="HarmonyOS NEXT开发实战：DevEco AI辅助编程工具（CodeGenie）的使用-鸿蒙开发者社区" height="357" src="https://i-blog.csdnimg.cn/img_convert/83730021063f6ea0c6e36ad06015d03d.png" width="268"/>
    </p>
    <p>
     <br/>
     在module &gt; src &gt; main &gt; ets 路径下， formcommon目录用于存放生成卡片的逻辑代码。
     <br/>
     formsetting：存放用户可配置的文件。
     <br/>
     formsetting &gt; formdbsetting：自定义配置以数据库方式进行卡片刷新的相关参数。
     <br/>
     formdbsetting &gt; formdbinfo：存放包含卡片信息的Info.ets文件，可在Info.ets文件中，添加卡片刷新所需要的具体的数据，后续会读取该文件并将数据存入数据库中。
     <br/>
     UserSettings.ets：可以自定义卡片刷新时从数据库获取数据的规则、数据解析规则、message内容刷新规则。
     <br/>
     formsetting &gt; formhttpsetting：自定义配置以网络请求方式进行卡片刷新的相关参数。
     <br/>
     formhttpsetting &gt; formhttpinfo：存放包含卡片信息的Info.ets文件，可在Info.ets文件中添加获取卡片刷新数据的URL。
     <br/>
     UserSettings.ets：可以自定义卡片刷新时从URL获取数据的规则、数据解析规则、message内容刷新规则。
     <br/>
     说明
     <br/>
     如需使用网络请求方式刷新卡片页面，需在EntryFormAbility.ets文件中将FormDbUpdate的接口注释掉，并将启用FormHttpUpdate接口。
     <br/>
     formsetting &gt; FormAction.ets：配置卡片事件。
     <br/>
     utils：存放工具类的目录，用户不可修改，如果修改再次生成逻辑代码时utils目录会被刷新。
    </p>
