    <h2>
     <a id="1_if_else_9">
     </a>
     1. if else语句
    </h2>
    <p>
     if满足条件会执行相应的代码语句，如果不满足呢？
     <br/>
     有没有不满足的情况下，可供执行的代码呢？
    </p>
    <p>
     <mark>
      if else 语句可以实现
     </mark>
     <br/>
     演示Python中if else的组合判断语句
    </p>
    <pre><code class="prism language-python">age <span class="token operator">=</span> <span class="token builtin">int</span><span class="token punctuation">(</span><span class="token builtin">input</span><span class="token punctuation">(</span><span class="token string">"请输入你的年龄："</span><span class="token punctuation">)</span><span class="token punctuation">)</span>

<span class="token keyword">if</span> age <span class="token operator">&gt;=</span> <span class="token number">18</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"您已成年，需要买票10元。"</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"您未成年，可以免费游玩。"</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     代码结果：
    </p>
    <blockquote>
     <p>
      请输入你的年龄：17
      <br/>
      祝您游玩愉快
      <br/>
      请输入你的年龄：19
      <br/>
      您已成年，游玩需要买票，10元.
      <br/>
      祝您游玩愉快
     </p>
    </blockquote>
    <h2>
     <a id="2_if_else_33">
     </a>
     2. if else语句注意点
    </h2>
    <ol>
     <li>
      else后，不需要判断条件
     </li>
     <li>
      和if的代码块一样，else的代码块同样需要4个空格作为缩进
     </li>
    </ol>
    <h2>
     <a id="3__37">
     </a>
     3. 总结
    </h2>
    <ol>
     <li>
      <p>
       if else 语句，其中
       <br/>
       if和其代码块，条件满足时执行
       <br/>
       else搭配if的判断条件，当不满足的时候执行
      </p>
     </li>
     <li>
      <p>
       if else语句的注意事项：
       <br/>
       else不需要判断条件，当if的条件不满足时，else执行
       <br/>
       else的代码块，同样要4个空格作为缩进
      </p>
     </li>
    </ol>
    <h2>
     <a id="4__46">
     </a>
     4. 练习案例：我要买票吗
    </h2>
    <p>
     通过input语句获取键盘输入的身高
     <br/>
     判断身高是否超过120cm，并通过print给出提示信息。
    </p>
    <p>
     <strong>
      演示if else练习题：我要买票吗
     </strong>
    </p>
    <pre><code class="prism language-python"><span class="token comment"># 定义键盘输入获取身高数据</span>
height <span class="token operator">=</span> <span class="token builtin">int</span><span class="token punctuation">(</span><span class="token builtin">input</span><span class="token punctuation">(</span><span class="token string">"请输入你的身高（cm）："</span><span class="token punctuation">)</span><span class="token punctuation">)</span>

<span class="token comment"># 通过if进行判断</span>
<span class="token keyword">if</span> height <span class="token operator">&gt;</span> <span class="token number">120</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"您的身高超出120CM，需要买票，10元。"</span><span class="token punctuation">)</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"您的身高低于120CM，可以免费游玩。"</span><span class="token punctuation">)</span>

<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"祝您游玩愉快"</span><span class="token punctuation">)</span>

</code></pre>
    <p>
     代码结果：
    </p>
    <blockquote>
     <p>
      请输入你的身高（cm）：90
      <br/>
      您的身高低于120CM，可以免费游玩。
      <br/>
      祝您游玩愉快
      <br/>
      请输入你的身高（cm）：130
      <br/>
      您的身高超出120CM，需要买票，10元。
      <br/>
      祝您游玩愉快
     </p>
