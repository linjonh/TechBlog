    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     FTP服务器（File Transfer Protocol Server）是在互联网上提供文件存储和访问服务的计算机，它们依照FTP协议提供服务。
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     FTP（File Transfer Protocol: 文件传输协议）作用： Internet 上用来传送文件的协议。
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     常见FTP服务器：
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     Windows：Serv-U FTP Server
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     Linux：ProFTPD:（Professional FTP daemon）一个Unix平台上或是类Unix平台上（如Linux, FreeBSD等）的FTP服务器程序。
    </p>

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     /etc/vsftpd/user_list：指定允许使用vsftpd 的用户列表文件。  白名单
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     vim  /etc/vsftpd/user_list

    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     # 如果userlist_deny= YES（默认），绝不允许在这个文件中的用户登录ftp，甚至不提示输入密码

    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      公司技术部准备搭建一台功能简单的FTP 服务器，允许所有员工上传和下载文件，并允许创建用户自己的目录。

    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      分析：
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      允许所有员工上传和下载文件需要设置成允许匿名用户登录并且需要将允许匿名用户上传功能
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

      开启，
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      anon_mkdir_write_enable 字段可以控制是否允许匿名用户创建目录。
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      备份原配置文件：
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      [root@xiaolyu10 vsftpd]# cp vsftpd.conf vsftpd.conf_bak
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      [root@xiaolyu10 ~]# vim /etc/vsftpd/vsftpd.conf
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      允许匿名用户访问
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      anonymous_enable=YES
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      允许匿名用户上传文件并可以创建目录
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      anon_upload_enable=YES
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      anon_mkdir_write_enable=YES

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     [root@xiaolyu10 ~]# service vsftpd restart
     <br/>
     Shutting down vsftpd:                                      [  OK  ]
     <br/>
     Starting vsftpd for vsftpd: 500 OOPS: bad bool value in config file for: anon_other_write_enable
     <br/>
     [FAILED]
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b2ee222eb3952313ee4292fc0837450c.png" style="border:0px;" width="705"/>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     根据提示，说明我刚才添加的那一行匿名写权限的参数有问题
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     [root@xiaolyu10 ~]# vim /etc/vsftpd/vsftpd.conf
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d82bc7d38cd0bcdb510dbe10b1bd28e3.png" style="border:0px;" width="705"/>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     也就是说参数后面不能有多余的空格。
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     重新启动就可以对重新命名和删除文件夹了，
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <span style="color:rgb(255,0,0);">
      <strong>
       匿名用户就可以删除和修改，非常不安全，慎重使用这个参数。
      </strong>
     </span>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      注意，默认匿名用户家目录的权限是755，这个权限是不能改变的。切记！
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     [root@xiaolyu10 ~]# service vsftpd restart
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
    </p>
    <h4 style="color:rgb(51,51,51);font-family:tahoma, '宋体';background-color:rgb(250,250,252);">
     实战2：
    </h4>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      公司内部现在有一台FTP 和WEB 服务器，FTP 的功能主要用于维护公司的网站内容，包括上传文件、创建目录、更新网页等等。公司现有两个部门负责维护任务，他们分别适用team1 和team2帐号进行管理。先要求仅允许team1 和team2 帐号登录FTP 服务器，但不能登录本地系统，并将这两个帐号的根目录限制为/var/www/html，不能进入该目录以外的任何目录。
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      ftp 和www web服务器相结合。
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      www web服务器根目录：  /var/www/html
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      只允许：team1和team2两用户 可以上传。 vsftp禁止匿名。
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      分析：
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      将FTP 和WEB 服务器做在一起是企业经常采用的方法，这样方便实现对网站的维护，为了增强安全性，首先需要使用仅允许本地用户访问，并禁止匿名用户登录。其次使用chroot 功能将team1和team2 锁定在/var/www/html 目录下。如果需要删除文件则还需要注意本地权限
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      解决方案：
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      （1）建立维护网站内容的ftp 帐号team1 和team2 并禁止本地登录，然后设置其密码
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      [root@xiaolyu10 ~]# useradd -s /sbin/nologin team1
      <br/>
      [root@xiaolyu10 ~]# useradd -s /sbin/nologin team2
      <br/>
      [root@xiaolyu10 ~]# echo "123456" | passwd --stdin team1
      <br/>
      Changing password for user team1.
      <br/>
      passwd: all authentication tokens updated successfully.
      <br/>
      [root@xiaolyu10 ~]# echo "123456" | passwd --stdin team2
      <br/>
      Changing password for user team2.
      <br/>
      passwd: all authentication tokens updated successfully.
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      2）配置vsftpd.conf 主配置文件并作相应修改
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      [root@xiaolyu10 vsftpd]# cp vsftpd.conf_bak vsftpd.conf
      <br/>
      cp: overwrite `vsftpd.conf'? y
      <br/>
      [root@xiaolyu10 vsftpd]# vim vsftpd.conf
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      #anonymous_enable=YES
      <br/>
      anonymous_enable=NO    禁止匿名用户登录
      <br/>
      #
      <br/>
      # Uncomment this to allow local users to log in.
      <br/>
      local_enable=YES  允许本地用户登录
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/b6932a86d1f3a26bb832dcd929d13504.png" style="border:0px;" width="705"/>
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     继续将：
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/4707e028bb9e8149830682b91c33d47c.png" style="border:0px;" width="705"/>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     修改为：
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/bea843c2f412457cb58f34dc048e134e.png" style="border:0px;" width="705"/>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      local_root=/var/www/html：设置本地用户的根目录为/var/www/html
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      chroot_list_enable=YES：激chroot 功能
     </strong>
    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
     <strong>
      chroot_list_file=/etc/vsftpd/chroot_list：设置锁定用户在根目录中的列表文件。此文件存放要锁定的用户名
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

     <strong>
      （3）建立/etc/vsftpd/chroot_list 文件，添加team1 和team2 帐号
     </strong>
    </p>
    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

     <strong>
      team1

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

    <p align="left" style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">

      （4）重启vsftpd 服务使配置生效

    </p>
    <p style="color:rgb(51,51,51);font-family:tahoma, '宋体';font-size:14px;background-color:rgb(250,250,252);">
