    <h1 class="title-article" id="articleContentId">
     DeepSeek &amp; Kimi详细生成PPT的步骤
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     以下是使用
     <strong>
      DeepSeek
     </strong>
     和
     <strong>
      Kimi
     </strong>
     协作生成 PPT 的详细步骤，结合了两者的优势实现高效创作：
    </p>
    <hr/>
    <h4>
     <strong>
      第一步：使用 DeepSeek 生成 PPT 大纲或内容
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        明确需求并输入提示词
       </strong>
       <br/>
       在 DeepSeek 的对话界面中，输入具体指令，要求生成 PPT 大纲或内容。建议使用以下模板：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          模板示例
         </strong>
         ：
        </p>
        <ul>
         <li>
          <p>
           <em>
            “请用 Markdown 格式生成一份关于[主题]的 PPT 大纲，需包含封面、目录、3个核心章节（每章至少2个子论点）、总结页，语言简洁专业，避免口语化。”
           </em>
          </p>
         </li>
         <li>
          <p>
           <em>
            “我需要一个主题为‘中国传统文化艺术魅力’的 PPT 大纲，要求包含数据支持和图文结合建议。”
           </em>
          </p>
         </li>
        </ul>
       </li>
       <li>
        <p>
         <strong>
          附加要求
         </strong>
         ：可指定风格（如学术、商务）、数据来源、图文比例等，提升内容严谨性。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        生成与优化内容
       </strong>
      </p>
      <ul>
       <li>
        <p>
         若大纲内容不完整，可进一步要求 DeepSeek 补充案例、图表描述或具体数据。
        </p>
       </li>
       <li>
        <p>
         如需结合现有文档，可上传文件并输入指令：
         <em>
          “根据附件内容生成 PPT 大纲，以 Markdown 格式输出。”
         </em>
         。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      第二步：通过 Kimi 一键生成 PPT
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        导入内容到 Kimi PPT 助手
       </strong>
      </p>
      <ul>
       <li>
        <p>
         登录
         <a href="https://kimi.ai/" rel="nofollow" title="Kimi 官网">
          Kimi 官网
         </a>
         ，进入
         <strong>
          Kimi+
         </strong>
         →
         <strong>
          PPT 助手
         </strong>
         ，粘贴 DeepSeek 生成的 Markdown 或文本内容。
        </p>
       </li>
       <li>
        <p>
         Kimi 会自动精简冗余文本，优化逻辑结构。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择模板并生成 PPT
       </strong>
      </p>
      <ul>
       <li>
        <p>
         根据主题选择模板（如商务、科技、中国风等），点击“一键生成 PPT”。
        </p>
       </li>
       <li>
        <p>
         生成时间通常在 1 分钟内，支持预览和重新选择模板。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      第三步：编辑与优化 PPT
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        调整内容与布局
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          大纲编辑
         </strong>
         ：在 Kimi 编辑界面中，可手动调整章节顺序或补充细节。
        </p>
       </li>
       <li>
        <p>
         <strong>
          模板替换
         </strong>
         ：若对设计不满意，可更换模板或自定义配色、字体。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        优化图文与多媒体
       </strong>
      </p>
      <ul>
       <li>
        <p>
         <strong>
          图片替换
         </strong>
         ：点击图片直接上传本地文件或使用 Kimi 内置素材库。
        </p>
       </li>
       <li>
        <p>
         <strong>
          数据可视化
         </strong>
         ：通过侧边栏插入图表、形状，或使用“PPT 拼图”功能将多页内容合并为长图。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        导出与演示
       </strong>
      </p>
      <ul>
       <li>
        <p>
         支持导出为 PPTX、PDF 格式，或在线演示。
        </p>
       </li>
       <li>
        <p>
         若需进一步美化，可将文件导入 PowerPoint 或 WPS 进行深度调整。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <strong>
      进阶技巧与注意事项
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        多文档整合
       </strong>
       ：上传多个相关文档至 Kimi，提炼综合大纲（如：
       <em>
        “根据提供的两份报告生成电商运营策略 PPT 大纲”
       </em>
       ）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        数据安全
       </strong>
       ：避免在提示词中包含敏感信息，建议使用脱敏数据。
      </p>
     </li>
     <li>
      <p>
       <strong>
        版权合规
       </strong>
       ：AI 生成的图片或素材需替换为开源内容，避免侵权。
      </p>
     </li>
    </ul>
    <hr/>
    <p>
     通过以上步骤，用户可在 10 分钟内完成从内容生成到 PPT 设计的全流程，显著提升效率。如需具体案例或更详细的操作演示，可参考清华大学团队的教程视频。
    </p>
