    <p>
    </p>
    <p>
     <code>
      os.environ
     </code>
     是 Python 中
     <code>
      os
     </code>
     模块的一个字典对象，用于访问和操作系统的环境变量。环境变量是操作系统或用户设置的全局变量，程序可以通过它们获取配置信息或系统状态。
    </p>
    <h4>
     <a id="1__4">
     </a>
     1. 基本用法
    </h4>
    <p>
     <code>
      os.environ
     </code>
     是一个类似字典的对象，存储了当前进程的环境变量。可以通过键值对的方式访问和修改这些变量。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os

<span class="token comment"># 获取环境变量</span>
home_dir <span class="token operator">=</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'HOME'</span><span class="token punctuation">]</span>
<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"Home directory: </span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>home_dir<span class="token punctuation">}</span></span><span class="token string">"</span></span><span class="token punctuation">)</span>

<span class="token comment"># 获取不存在的环境变量（会抛出 KeyError）</span>
<span class="token keyword">try</span><span class="token punctuation">:</span>
    unknown_var <span class="token operator">=</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'UNKNOWN_VAR'</span><span class="token punctuation">]</span>
<span class="token keyword">except</span> KeyError<span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"Environment variable 'UNKNOWN_VAR' does not exist."</span><span class="token punctuation">)</span>

<span class="token comment"># 使用 get 方法避免 KeyError</span>

<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"Unknown variable: </span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>unknown_var<span class="token punctuation">}</span></span><span class="token string">"</span></span><span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="2__27">
     </a>
     2. 修改环境变量
    </h4>
    <p>
     可以通过
     <code>
      os.environ
     </code>
     修改环境变量，但修改仅对当前进程有效，不会影响系统或其他进程。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os

<span class="token comment"># 设置新的环境变量</span>
os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'MY_VAR'</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token string">'my_value'</span>

<span class="token comment"># 修改现有环境变量</span>
os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'PATH'</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token string">'/new/path:'</span> <span class="token operator">+</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'PATH'</span><span class="token punctuation">]</span>

<span class="token comment"># 删除环境变量</span>
<span class="token keyword">del</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'MY_VAR'</span><span class="token punctuation">]</span>
</code></pre>
    <h4>
     <a id="3__45">
     </a>
     3. 遍历环境变量
    </h4>
    <p>
     可以遍历
     <code>
      os.environ
     </code>
     查看所有环境变量及其值。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os

<span class="token keyword">for</span> key<span class="token punctuation">,</span> value <span class="token keyword">in</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">.</span>items<span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string-interpolation"><span class="token string">f"</span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>key<span class="token punctuation">}</span></span><span class="token string">: </span><span class="token interpolation"><span class="token punctuation">{<!-- --></span>value<span class="token punctuation">}</span></span><span class="token string">"</span></span><span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="4__58">
     </a>
     4. 检查环境变量是否存在
    </h4>
    <p>
     使用
     <code>
      in
     </code>
     操作符检查某个环境变量是否存在。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os

<span class="token keyword">if</span> <span class="token string">'HOME'</span> <span class="token keyword">in</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">:</span>
    <span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"HOME environment variable exists."</span><span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="5__70">
     </a>
     5. 环境变量的作用
    </h4>
    <p>
     环境变量常用于以下场景：
    </p>
    <ul>
     <li>
      <strong>
       配置管理
      </strong>
      ：存储数据库连接、API 密钥等配置信息。
     </li>
     <li>
      <strong>
       路径设置
      </strong>
      ：如
      <code>
       PATH
      </code>
      变量指定可执行文件的搜索路径。
     </li>
     <li>
      <strong>
       调试与日志
      </strong>
      ：通过环境变量控制调试模式或日志级别。
     </li>
    </ul>
    <h4>
     <a id="6__78">
     </a>
     6. 子进程中的环境变量
    </h4>
    <p>
     使用
     <code>
      os.environ
     </code>
     修改的环境变量会影响当前进程及其子进程。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os
<span class="token keyword">import</span> subprocess

<span class="token comment"># 设置环境变量</span>
os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'MY_VAR'</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token string">'my_value'</span>

<span class="token comment"># 在子进程中访问环境变量</span>
subprocess<span class="token punctuation">.</span>run<span class="token punctuation">(</span><span class="token punctuation">[</span><span class="token string">'echo'</span><span class="token punctuation">,</span> os<span class="token punctuation">.</span>environ<span class="token punctuation">[</span><span class="token string">'MY_VAR'</span><span class="token punctuation">]</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="7__95">
     </a>
     7. 环境变量的持久性
    </h4>
    <p>
     通过
     <code>
      os.environ
     </code>
     设置的变量仅在当前进程有效，进程结束后变量消失。若需永久修改，需通过操作系统设置。
    </p>
    <h4>
     <a id="8__99">
     </a>
     8. 跨平台注意事项
    </h4>
    <p>
     不同操作系统环境变量名称可能不同，如：
    </p>
    <ul>
     <li>
      <strong>
       Windows
      </strong>
      ：
      <code>
       USERPROFILE
      </code>
      表示用户主目录。
     </li>
     <li>
      <strong>
       Linux/macOS
      </strong>
      ：
      <code>
       HOME
      </code>
      表示用户主目录。
     </li>
    </ul>
    <p>
     编写跨平台代码时需注意这些差异。
    </p>
    <h4>
     <a id="9__108">
     </a>
     9. 示例：使用环境变量配置应用程序
    </h4>
    <p>
     环境变量常用于配置应用程序，如 Flask 通过
     <code>
      FLASK_ENV
     </code>
     控制运行环境。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> os
<span class="token keyword">from</span> flask <span class="token keyword">import</span> Flask

    app<span class="token punctuation">.</span>debug <span class="token operator">=</span> <span class="token boolean">True</span>

<span class="token decorator annotation punctuation">@app<span class="token punctuation">.</span>route</span><span class="token punctuation">(</span><span class="token string">'/'</span><span class="token punctuation">)</span>
<span class="token keyword">def</span> <span class="token function">hello</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
    <span class="token keyword">return</span> <span class="token string">"Hello, World!"</span>

<span class="token keyword">if</span> __name__ <span class="token operator">==</span> <span class="token string">'__main__'</span><span class="token punctuation">:</span>
    app<span class="token punctuation">.</span>run<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="_132">
     </a>
     总结
    </h4>
    <p>
     <code>
      os.environ
     </code>
     提供了访问和操作环境变量的接口，适用于配置管理、路径设置等场景。修改的环境变量仅对当前进程有效，跨平台时需注意变量名的差异。
    </p>
