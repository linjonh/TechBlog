    <h2>
     <a id="_116">
     </a>
     五、电压、电流的基础知识科普
    </h2>
    <h4>
     <a id="_118">
     </a>
     <strong>
      直流电的电流和电压概念详解
     </strong>
    </h4>
    <p>
     在电学中，
     <strong>
      电压（Voltage）
     </strong>
     和
     <strong>
      电流（Current）
     </strong>
     是最基本的概念。它们分别代表着电能的驱动力和电荷的流动。我们先分别解释它们的概念，再分析它们的关系，并解释为什么直流电（DC）有正负极。
    </p>
    <hr/>
    <h3>
     <a id="1__124">
     </a>
     <strong>
      1. 什么是电压？为什么有正负极？
     </strong>
    </h3>
    <h4>
     <a id="1_126">
     </a>
     <strong>
      （1）电压的概念
     </strong>
    </h4>
    <p>
     电压是指
     <strong>
      电荷之间的电势差
     </strong>
     ，它是驱使电流流动的“推动力”，单位是伏特（V）。可以类比成“水压”——水压越高，水就流得越快；同样，电压越高，电流的驱动力就越大。
    </p>
    <p>
     <strong>
      打个比方
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       高处的水流向低处
      </strong>
      ，因为水有“水势差”；
     </li>
     <li>
      <strong>
       电荷从高电势流向低电势
      </strong>
      ，因为它们之间有“电势差”（也就是电压）。
     </li>
    </ul>
    <p>
     所以，
     <strong>
      电压就是推动电荷流动的力量
     </strong>
     ，它决定了电流的方向。
    </p>
    <hr/>
    <h4>
     <a id="2_139">
     </a>
     <strong>
      （2）为什么有正极和负极？
     </strong>
    </h4>
    <p>
     <strong>
      直流电（DC）
     </strong>
     之所以有正极（+）和负极（-），是因为它的电压是恒定的，不随时间变化。一般：
    </p>
    <ul>
     <li>
      <strong>
       正极（+）：电势高
      </strong>
     </li>
     <li>
      <strong>
       负极（-）：电势低
      </strong>
     </li>
    </ul>
    <p>
     电荷在电压的作用下，会从
     <strong>
      负极流向正极
     </strong>
     ，但是一般约定电流方向为
     <strong>
      正极流向负极
     </strong>
     （这个是人为规定的，叫“约定电流方向”）。
    </p>
    <hr/>
    <h3>
     <a id="2__150">
     </a>
     <strong>
      2. 什么是电流？
     </strong>
    </h3>
    <p>
     电流表示
     <strong>
      电荷的流动
     </strong>
     ，单位是安培（A）。可以把它理解为水管中的“水流量”，水流量越大，水就流得越快。同样，电流的大小表示单位时间内通过导体某一点的电荷数量。
    </p>
    <ul>
     <li>
      <strong>
       直流电流（DC Current）：电流的方向恒定不变
      </strong>
      ，比如电池提供的电流；
     </li>
     <li>
      <strong>
       交流电流（AC Current）：电流的方向随时间变化
      </strong>
      ，比如家庭用电。
     </li>
    </ul>
    <p>
     在直流电中，
     <strong>
      电子从负极流向正极
     </strong>
     ，但由于“约定电流方向”，我们一般认为
     <strong>
      电流从正极流向负极
     </strong>
     。
    </p>
    <hr/>
    <h3>
     <a id="3__161">
     </a>
     <strong>
      3. 电压、电流的关系
     </strong>
    </h3>
    <p>
     电压和电流的关系可以用
     <strong>
      欧姆定律（Ohm’s Law）
     </strong>
     表示：
    </p>
    <p>
     U=IRU = IR
    </p>
    <p>
     其中：
    </p>
    <ul>
     <li>
      <strong>
       U 是电压（Volt，V）
      </strong>
     </li>
     <li>
      <strong>
       I 是电流（Ampere，A）
      </strong>
     </li>
     <li>
      <strong>
       R 是电阻（Ohm，Ω）
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      理解方式
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       电压是推动力（类似水压）
      </strong>
     </li>
     <li>
      <strong>
       电流是流动的电荷（类似水流）
      </strong>
     </li>
     <li>
      <strong>
       电阻是阻碍电流的因素（类似水管的狭窄程度）
      </strong>
     </li>
    </ul>
    <p>
     如果电阻固定：
    </p>
    <ul>
     <li>
      <strong>
       电压升高，电流变大
      </strong>
      （水压高，水流快）
     </li>
     <li>
      <strong>
       电压降低，电流变小
      </strong>
      （水压低，水流慢）
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="4__186">
     </a>
     <strong>
      4. 总结
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       电压
      </strong>
      （V）：电势差，推动电流流动的“驱动力”。
     </li>
     <li>
      <strong>
       电流
      </strong>
      （A）：电荷的流动，方向由正极流向负极（约定方向）。
     </li>
     <li>
      <strong>
       电阻
      </strong>
      （Ω）：阻碍电流流动的因素。
     </li>
     <li>
      <strong>
       直流电有正负极
      </strong>
      ，因为它的电压是固定的，电流方向不变。
     </li>
    </ul>
    <p>
     你可以把它想象成一个水循环系统：
    </p>
    <ul>
     <li>
      <strong>
       电池是水泵
      </strong>
      （提供电压）
     </li>
     <li>
      <strong>
       电流是水流
      </strong>
     </li>
     <li>
      <strong>
       电阻是水管的狭窄程度
      </strong>
     </li>
     <li>
      <strong>
       电压越高，水泵推得越猛，水流（电流）就越大！
      </strong>
     </li>
    </ul>
    <h2>
     <a id="_202">
     </a>
     六、电机转动会影响其他模块吗？
    </h2>
    <p>
     在直流电路系统中，
     <strong>
      电机转动时导致其他模块无法正常工作
     </strong>
     ，一般是因为
     <strong>
      供电问题
     </strong>
     。
    </p>
    <hr/>
    <h3>
     <a id="1__208">
     </a>
     <strong>
      1. 电机瞬间启动时，电流需求过大
     </strong>
    </h3>
    <h4>
     <a id="_210">
     </a>
     <strong>
      问题描述
     </strong>
    </h4>
    <p>
     电机在启动时，一般需要较大的
     <strong>
      瞬时电流
     </strong>
     ，远远超过它正常运行时的电流。这叫做
     <strong>
      启动电流冲击
     </strong>
     。
    </p>
    <h4>
     <a id="_214">
     </a>
     <strong>
      为什么会这样？
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       电机启动时
      </strong>
      ，需要克服惯性，从静止状态到转动状态，需要大量能量；
     </li>
     <li>
      <strong>
       电机的电磁线圈
      </strong>
      在初始状态是“低阻抗”的，导致电流猛增；
     </li>
     <li>
      <strong>
       如果电源供电能力不足
      </strong>
      ，瞬间的高电流消耗会导致整个电路系统的电压下降（电压跌落），其他模块因此可能得不到足够的电压，导致无法正常工作。
     </li>
    </ul>
    <h4>
     <a id="_220">
     </a>
     <strong>
      表现现象
     </strong>
    </h4>
    <ul>
     <li>
      其他模块可能
      <strong>
       死机、重启、工作异常
      </strong>
      ；
     </li>
     <li>
      可能听到电机启动时，
      <strong>
       电压降低的“嗡嗡”声
      </strong>
      ；
     </li>
     <li>
      测量电压时，可能发现电压在电机启动时明显下降。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="2__228">
     </a>
     <strong>
      2. 电源功率不足（供电能力不够）
     </strong>
    </h3>
    <h4>
     <a id="_230">
     </a>
     <strong>
      问题描述
     </strong>
    </h4>
    <p>
     如果电源（比如电池或电源适配器）的功率（W）或最大输出电流（A）不足，就会导致电机运转时，其他模块无法获得足够的电能。
    </p>
    <h4>
     <a id="_234">
     </a>
     <strong>
      为什么会这样？
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       电源有最大电流输出能力
      </strong>
      （比如 12V 5A 电源，最大只能提供 5A）。
     </li>
     <li>
      <strong>
       如果总负载电流超过这个数值
      </strong>
      ，电源可能会进入限流状态，甚至触发保护机制，导致电压下降。
     </li>
     <li>
      <strong>
       其他模块电压不稳定，就可能死机或异常
      </strong>
      。
     </li>
    </ul>
    <h4>
     <a id="_240">
     </a>
     <strong>
      如何检查？
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       计算总电流需求
      </strong>
      ：各个模块的电流 + 电机的最大电流，看看是否超过电源的额定输出。
     </li>
     <li>
      <strong>
       更换更大功率的电源
      </strong>
      ，比如从 12V 5A 升级到 12V 10A，看看是否有所改善。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="3__247">
     </a>
     <strong>
      3. 电源线或电路的导线太细，电阻过大
     </strong>
    </h3>
    <h4>
     <a id="_249">
     </a>
     <strong>
      问题描述
     </strong>
    </h4>
    <p>
     电机启动时需要大电流，如果供电线路（电线）过细或者接触不良，会导致供电线路本身有较大
     <strong>
      电阻
     </strong>
     ，形成
     <strong>
      电压降
     </strong>
     。
    </p>
    <h4>
     <a id="_253">
     </a>
     <strong>
      为什么会这样？
     </strong>
    </h4>
    <ul>
     <li>
      细导线的
      <strong>
       电阻大
      </strong>
      ，电流流过时会有电压损耗；
     </li>
     <li>
      远距离供电时，
      <strong>
       线缆本身就会损耗一部分电压
      </strong>
      ；
     </li>
     <li>
      结果就是：
      <strong>
       电机工作时，其他模块供电电压不够，导致故障
      </strong>
      。
     </li>
    </ul>
    <h4>
     <a id="_259">
     </a>
     <strong>
      如何检查？
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       测量电源输出端 vs. 负载端的电压
      </strong>
      ，看看电压是否有明显下降。
     </li>
     <li>
      <strong>
       使用更粗的电源线
      </strong>
      ，减少电阻损耗。
     </li>
     <li>
      <strong>
       缩短电源线长度
      </strong>
      ，减少电压损失。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="4__267">
     </a>
     <strong>
      4. 供电电容不足，导致电压不稳定
     </strong>
    </h3>
    <h4>
     <a id="_269">
     </a>
     <strong>
      问题描述
     </strong>
    </h4>
    <p>
     当电机启动时，它会造成瞬间的电压波动（电压跌落）。如果没有足够的滤波电容，其他模块会受到影响。
    </p>
    <h4>
     <a id="_273">
     </a>
     <strong>
      为什么会这样？
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       电容可以存储电荷，在电机启动时提供额外的电流支持
      </strong>
      ；
     </li>
     <li>
      <strong>
       如果没有大电容，电压波动会直接影响其他模块的工作
      </strong>
      ；
     </li>
     <li>
      低压 MCU（如 3.3V、5V 的单片机）特别容易受影响。
     </li>
    </ul>
    <h4>
     <a id="_279">
     </a>
     <strong>
      解决方案
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       在电机附近并联大电容
      </strong>
      （比如 470μF~1000μF 的电解电容）；
     </li>
     <li>
      <strong>
       在主电源线上加适当的电容
      </strong>
      （比如 1000μF 的电解电容 + 0.1μF 的陶瓷电容）；
     </li>
     <li>
      <strong>
       使用独立的电源给 MCU 等敏感模块供电
      </strong>
      ，避免受到电机干扰。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="5_EMI_287">
     </a>
     <strong>
      5. 电机产生电磁干扰（EMI）影响其他模块
     </strong>
    </h3>
    <h4>
     <a id="_289">
     </a>
     <strong>
      问题描述
     </strong>
    </h4>
    <p>
     电机工作时，特别是
     <strong>
      直流有刷电机
     </strong>
     ，会产生很强的电磁干扰，影响电路的正常工作。
    </p>
    <h4>
     <a id="_293">
     </a>
     <strong>
      如何降低干扰？
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       在电机两端并联滤波电容
      </strong>
      （常用 0.1μF 陶瓷电容）。
     </li>
     <li>
      <strong>
       使用屏蔽电缆或尽量让信号线远离电机电源线
      </strong>
      。
     </li>
     <li>
      <strong>
       给 MCU 或传感器供电加滤波电路
      </strong>
      ，比如
      <strong>
       LC 滤波
      </strong>
      或者
      <strong>
       独立电源
      </strong>
      。
     </li>
    </ol>
