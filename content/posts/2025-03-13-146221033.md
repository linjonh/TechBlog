    <h4>
     <a id="_0">
     </a>
     简介
    </h4>
    <p>
     <code>
      zgrep
     </code>
     用于在压缩 (
     <code>
      .gz
     </code>
     ) 文件中搜索模式，就像
     <code>
      grep
     </code>
     在常规文本文件中所做的那样。它的工作原理是将文件临时解压到内存中，搜索模式并显示匹配的行。
    </p>
    <h4>
     <a id="_4">
     </a>
     基础语法
    </h4>
    <pre><code class="prism language-shell">zgrep <span class="token punctuation">[</span>OPTIONS<span class="token punctuation">]</span> PATTERN FILE.gz

或

<span class="token function">gzip</span> <span class="token parameter variable">-dc</span> FILE.gz <span class="token operator">|</span> <span class="token function">grep</span> <span class="token punctuation">[</span>OPTIONS<span class="token punctuation">]</span> PATTERN
</code></pre>
    <h4>
     <a id="_14">
     </a>
     示例用法
    </h4>
    <h5>
     <a id="_gz__16">
     </a>
     在 .gz 文件中搜索字符串
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token string">"error"</span> logfile.gz

或

<span class="token function">gzip</span> <span class="token parameter variable">-dc</span> logfile.gz <span class="token operator">|</span> <span class="token function">grep</span> <span class="token string">"error"</span>
</code></pre>
    <h5>
     <a id="_26">
     </a>
     大小写不敏感搜索
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-i</span> <span class="token string">"error"</span> logfile.gz
</code></pre>
    <h5>
     <a id="_32">
     </a>
     在多个压缩文件中搜索
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token string">"error"</span> *.gz

或

<span class="token function">gzip</span> <span class="token parameter variable">-dc</span> *.gz <span class="token operator">|</span> <span class="token function">grep</span> <span class="token string">"error"</span>
</code></pre>
    <h5>
     <a id="_42">
     </a>
     显示行号
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-n</span> <span class="token string">"error"</span> logfile.gz

<span class="token comment"># 显示匹配的行以及行号</span>
</code></pre>
    <h5>
     <a id="_50">
     </a>
     统计匹配到的行数
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-c</span> <span class="token string">"error"</span> logfile.gz
</code></pre>
    <h5>
     <a id="_56">
     </a>
     仅显示匹配的文件名
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-l</span> <span class="token string">"error"</span> *.gz

<span class="token comment"># 仅列出包含“error”的 .gz 文件的文件名</span>
</code></pre>
    <h5>
     <a id="_64">
     </a>
     反向匹配
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-v</span> <span class="token string">"error"</span> logfile.gz

<span class="token comment"># 显示除包含“error”的行之外的所有行</span>
</code></pre>
    <h5>
     <a id="_72">
     </a>
     在目录中递归搜索
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-r</span> <span class="token string">"error"</span> /var/log/

<span class="token comment"># 在 /var/log/ 下的 .gz 文件中递归搜索“error”</span>
</code></pre>
    <h5>
     <a id="E__80">
     </a>
     使用正则表达式（-E 表示扩展正则表达式）
    </h5>
    <pre><code class="prism language-shell">zgrep <span class="token parameter variable">-E</span> <span class="token string">"error|warning|failed"</span> logfile.gz

<span class="token comment"># 查找包含“error”、“warning”或“failed”的行</span>
</code></pre>
