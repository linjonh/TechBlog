    <ul>
     <li>
      调整操作系统为性能最佳
      <br/>
      <ul>
       <li>
        计算机 -&gt; 属性 -&gt; 系统 -&gt; 高级系统设置 -&gt; 高级选项卡 -&gt; 设置（性能） -&gt; 选择调整为最佳性能 -&gt; 只勾选平滑屏幕字体边缘
       </li>
      </ul>
     </li>

      <br/>
      <ul>
       <li>
        控制面板 -&gt; 计算机管理 -&gt; 服务和应用程序 -&gt; 服务
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <ul>
     <li>
      设置虚拟内存
      <br/>
      <ul>
       <li>
        计算机 -&gt; 属性 -&gt; 系统 -&gt; 高级 -&gt; 性能设置 -&gt; 性能选项中选择高级 -&gt; 虚拟内存更改 -&gt; 选择C盘 -&gt; 设置值为物理内存的1/2 ~ 1倍之间
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <ul>
     <li>
      启用快捷启动栏
      <br/>
      <ul>
       <li>
        在C盘下搜索Quick，找到如下目录：C:\Users\Administrator\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch
       </li>
       <li>
        在任务栏右键 -&gt; 新建工具栏 -&gt; 打开窗口 -&gt; 粘贴以上的路径 -&gt; 打开文件夹 -&gt; OK
       </li>
      </ul>
     </li>
    </ul>

    <ul>
     <li>
      修改任务栏，使用小图标，不分组
      <br/>
      <ul>
