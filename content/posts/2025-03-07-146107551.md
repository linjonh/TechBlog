    <h3>
     <a id="Reference_Types_49">
     </a>
     二、引用类型（Reference Types）‌
    </h3>
    <p>
     引用类型存储数据地址，分配在堆内存中，可赋值为 null。
    </p>
    <ol>
     <li>
      类（Class）‌
     </li>
    </ol>
    <ul>
     <li>
      基础引用类型（如 object、自定义类）‌
     </li>
    </ul>
    <ol start="2">
     <li>
      数组（Array）‌
     </li>
    </ol>
    <ul>
     <li>
      同类型元素的集合（如 int[] arr = {1, 2, 3};）‌
     </li>
    </ul>
    <ol start="3">
     <li>
      字符串（String）‌
     </li>
    </ol>
    <ul>
     <li>
      不可变的字符序列（如 string s = “Hello”;）‌
     </li>
    </ul>
    <ol start="4">
     <li>
      接口（Interface）‌
     </li>
    </ol>
    <ul>
     <li>
      定义方法契约（如 IDisposable）‌
     </li>
    </ul>
    <ol start="5">
     <li>
      ‌委托（Delegate）‌
     </li>
    </ol>
    <ul>
     <li>
      封装方法引用（类似函数指针）‌
     </li>
    </ul>
    <ol start="6">
     <li>
      ‌动态类型（Dynamic）‌
     </li>
    </ol>
    <ul>
     <li>
      运行时解析类型（如 dynamic obj = 10;）‌
     </li>
    </ul>
    <h3>
     <a id="__76">
     </a>
     三、特殊类型 ‌
    </h3>
    <ol>
     <li>
      可空类型（Nullable）‌
     </li>
    </ol>
    <ul>
     <li>
      允许值类型赋值为 null（如 int? num = null;）。
     </li>
    </ul>
    <ol start="2">
     <li>
      指针类型（Pointer）‌
     </li>
    </ol>
    <ul>
     <li>
      用于非安全代码操作内存地址（需 unsafe 块）。
     </li>
    </ul>
    <h3>
     <a id="_83">
     </a>
     关键注意事项
    </h3>
    <ol>
     <li>
      浮点精度问题‌
     </li>
    </ol>
    <ul>
     <li>
      float/double 可能存在舍入误差，decimal 适用于精确计算‌。
     </li>
    </ul>
    <ol start="2">
     <li>
      ‌可空类型安全访问‌
     </li>
    </ol>
    <ul>
     <li>
      需通过 Nullable.HasValue 和 Nullable.Value 操作‌。
     </li>
    </ul>
    <ol start="3">
     <li>
      ‌字符串不可变性‌
     </li>
    </ol>
    <ul>
     <li>
      频繁修改字符串建议使用 StringBuilder‌。
     </li>
    </ul>
