# python安装

对于第一次接触python的新手来说，软件的安装成为一大问题，此文章针对python安装的步骤详解细化（安装版本python3.10.2）

安装时
**注意**
要安装适合自己笔记本的版本(x86是32位的，x86-64是64位的)

* **双击下载好的安装包，跳出如下图弹窗：**

![](https://i-blog.csdnimg.cn/blog_migrate/37b44725de00c0e4f9670d0b1116fc10.png)

Select Install Now to install Python with default settings,or choose customize to enable or dieable features.

选择“立即安装”以使用默认设置安装python，或者选择“自定义”安装用以启用或者禁用功能。

**Install Now**

立即安装（点击后按照默认路径直接安装）

Include IDLE ,pip and documentation

包括IDLE（python内部编辑器），pip（python的 包管理器，第三方插件，可以安装其他python包），documentation（文档及日志文件随路径安装并写入）

Creates shortcuts and file associations

创建快捷方式和文件关联

**Customize installation**

自定义安装

Choose location and features

选择位置和功能

**Install launvher for all users (recommended)**

为所有用户安装启动器（推荐）

（如果不能勾选，原因如下：你没有管理员权限；你有管理员权限，但不是
`Administrator`
用户。）

**Add Python 3.10 to PATH**

添加python3.10的环境变量（此选项勾上后，就不用去手动添加环境变量了）

如果没有勾选，手动添加环境变量：桌面计算机--右键打开属性--点击高级系统设置--环境变量--系统变量--选择PATH--双击在变量值中添加（添加到最后，即“；”后面，添加的地址为python的安装地址）

* 此处我们选择
  **Customize installation**
  自定义安装

![](https://i-blog.csdnimg.cn/blog_migrate/ced36ef007e7277168bd3b96c0e29848.png)

（立即安装也有此窗口）

**Optional Features**

可选功能

Documentation，pip（务必勾选），IDLE（初学者请勾选）上述都已提到过。

**Python test suite**

python测试套件

py launcher与 for all users

此处简单来理解就是变量及安装至全部用户。

此处可根据自己需要进行勾选，随后next。

* **切记更改**
  **路径**

![](https://i-blog.csdnimg.cn/blog_migrate/1368b0208194f5168a6ab79c3e71af98.png)

**Install for all users**

所有用户可使用

**Associate files with Python**

关联python的相关文件（！）

**Create shortcuts for installed applications**

创建桌面的快捷方式

**Add Python to environment variables**

添加系统变量 （！）

**Precompile standard library**

安装预编译标准库（此处可进行自行选择）

**Download debugging symbols**

安装调试模块（此功能适宜开发者使用）

**Download debug binaries**

安装用于VS的调试符号-二进制

**Customize install location**

！！！安装路径，如果不满意可进行更改。

选择好适合自己的功能后，点击install。

* 等待安装完成

![](https://i-blog.csdnimg.cn/blog_migrate/c4a30aa7050e9113f76adc80ca6f6247.png)

cancel是关闭取消，不要点！

* **安装完成**

![](https://i-blog.csdnimg.cn/blog_migrate/16a429e89cb4b8bd3d791ce6a18e8b5b.png)

点击close

* **在开始菜单可以找到刚安装的pyhon**

![](https://i-blog.csdnimg.cn/blog_migrate/2c35ef01df14be420ae54d095ca231ba.png)

此次安装详解到此结束，如有疑问可以留言。
