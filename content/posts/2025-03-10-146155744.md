### 调用API 完成图片生成部署：

我们也可以自己使用api去调用，下面我将分享一下如何通过简单的代码调用通义万相2.1的API生成图像，看看它是如何通过文本生成的。

#### 1. 安装必备库

在开始之前，我们需要安装
`requests`
和
`Pillow`

```bash
pip install requests
pip install pillow

```

### 2. 使用api进行图像生成

接下来，我将展示如何通过Python调用通义万相2.1的API生成一张图像。在这个示例中，我想生成一张描述“未来城市天际线”的图像。

```python
import requests
from PIL import Image
from io import BytesIO
import json

# 设置API请求的URL
api_url = "https://your-cloud-platform-api.com/tongyi/wanxiang/v2/generate"  # 假设这是API的URL

# 通义万相2.1的API密钥（替换为你自己的API密钥）
api_key = "your_api_key_here"

# 设置请求头，包含API密钥
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

# 设置文本描述，用于生成图像
description = "A futuristic city skyline at sunset, with flying cars and neon lights"

# 设置请求数据
data = {
    "text": description,  # 图像生成的文本描述
    "style": "realistic",  # 图像风格：'realistic', 'cartoon', 'abstract'等
    "width": 1024,  # 图像宽度
    "height": 1024  # 图像高度
}

# 发送POST请求，调用通义万相2.1 API进行图像生成
response = requests.post(api_url, json=data, headers=headers)

# 检查请求是否成功
if response.status_code == 200:
    # 获取返回的图像数据
    image_data = response.content
    # 使用Pillow打开图像
    image = Image.open(BytesIO(image_data))
    
    # 显示生成的图像
    image.show()
    
    # 保存图像到本地
    image.save("generated_image.png")
    print("Image saved as 'generated_image.png'")
else:
    print(f"Error: {response.status_code}, {response.text}")

```

我们来分析一下上面的代码：

* **请求URL和请求头**
  ：

  + `api_url`
    是你要发送请求的API地址。在实际应用中，你需要替换为通义万相2.1的真实API地址。
  + `api_key`
    是通义万相2.1平台提供的API密钥，必须正确配置才能成功调用API。

  **设置文本描述**
  ：

  + `description`
    字段是你想要生成图像的文本描述。在这个示例中，我使用了“未来城市天际线”的描述，你可以根据需要修改成其他描述。

  **请求数据**
  ：

  + `text`
    字段是你输入的图像描述，
    `style`
    字段定义了你希望图像的风格，
    `width`
    和
    `height`
    字段决定了生成图像的尺寸。

  **发送请求并处理响应**
  ：

  + 我通过
    `requests.post()`
    发送POST请求，并从API响应中获取图像数据。
  + 使用
    `Pillow`
    库将返回的二进制图像数据转换为图片对象，并显示出来。如果图像生成成功，还会将其保存到本地文件夹中。
