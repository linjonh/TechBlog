     4. 考试分数是否通过
    </h2>
    <p>
     题目：
     <br/>
     KiKi想知道他的考试分数是否通过，请帮他判断。从键盘任意输入一个整数表示的分数，编程判断该分数是否在及格范围内，如果及格，即：分数大于等于60分，是输出“Pass”，否则，输出“Fail”。

     输入描述：
     <br/>
     多组输入，每行输入包括一个整数表示的分数（0~100）。
     <br/>
     输出描述：
     <br/>

     94

     <br/>
     Pass

     示例2
     <br/>
     输入：
     <br/>
     44
     <br/>

     Fail
