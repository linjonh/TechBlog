     Newer than known valid Java version (21.0.5), gradle (7.6.3). Treating as valid configuration. The configured version of Java detected may conflict with the Gradle version in your new Flutter app. [RECOMMENDED] If so, to keep the default Gradle version 7.6.3, make sure to download a compatible Java version (Java 11 &lt;= compatible Java version &lt; Java 20). You may configure this compatible Java version by running: `flutter config --jdk-dir=&lt;JDK_DIRECTORY&gt;` Note that this is a global configuration for Flutter.

    </p>
    <p>
     解决方法就是：Java 11 &lt;= compatible Java version &lt; Java 20
    </p>
    <p>
     下载安装jdk  命令设置
    </p>
    <p>
     flutter config --jdk-dir="/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home"
