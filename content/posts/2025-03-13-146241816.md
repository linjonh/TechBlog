# Flutter Mac Error
Newer than known valid Java version (21.0.5), gradle (7.6.3). Treating as
valid configuration. The configured version of Java detected may conflict with
the Gradle version in your new Flutter app. [RECOMMENDED] If so, to keep the
default Gradle version 7.6.3, make sure to download a compatible Java version
(Java 11 <= compatible Java version < Java 20). You may configure this
compatible Java version by running: `flutter config --jdk-dir=`
Note that this is a global configuration for Flutter.
Flutter版本3.22.0 AS2024.3.1 低版本的Flutter高版本的AS会出现这个问题
解决方法就是：Java 11 <= compatible Java version < Java 20
下载安装jdk 命令设置
flutter config --jdk-
dir="/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home"
