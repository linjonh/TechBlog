# 怎样成为微软认证讲师（MCP）（一）

凡是在微软相关技术和演讲方面有一定能力和兴趣的人员都有机会成为微软认证讲师（

MCT

）。如果您一直致力于研究或使用微软产品和解决方案、拥有相当的实际经验并具备良好的表达能力和整体形象，您可以按照以下

5

步完成微软认证讲师（以下简称

MCT

）的申请。

**第**
**1**
**步：获得微软认证专家（**
**MCP**
**）或微软商业解决方案（**
**MBS**
**）资格证书**

  

根据微软公司全球统一标准，在申请成为

MCT

之前，您必须获得下列证书之一。此项标准的目的是为了核实

MCT

申请者具备经过证实的技术实力和教学工作能力。一旦获得某项微软认证，即表明您已经在与您的证书相关的领域或工作岗位中展示了您运用微软产品和技术开展工作的熟练程度。

  
· Microsoft Certified Systems Engineer (MCSE) on Windows Server 2003
  
· Microsoft Certified Systems Administrator (MCSA) on Windows Server 2003
  
· Microsoft Certified Systems Engineer (MCSE) on Microsoft Windows 2000 Server
  
· Microsoft Certified Systems Administrator (MCSA) on Microsoft Windows 2000 Server
  
· Microsoft Certified Desktop Support Technician (MCDST)
  
· Microsoft Certified Solution Developer (MCSD) for Microsoft .NET
  
· Microsoft Certified Application Developer (MCAD) for Microsoft .NET
  
· Microsoft Certified Database Administrator (MCDBA) on Microsoft SQL Server 2000
  
· Microsoft Certified Professional: Exam 70-301, Managing, Organizing, and Delivering IT Projects by Using the Microsoft Solutions Framework 3.0
  
· Microsoft Certified Professional Developer (MCPD)
  
· Microsoft Certified IT Professional (MCITP)
  
· Microsoft Certified IT Professional (MCITP): Enterprise Desktop Administrator 7
  
· Microsoft Certified Technology Specialist (MCTS): Managing Projects with Microsoft Office Project 2007
  
· Microsoft Certified Technology Specialist (MCTS): Enterprise Project Management with Microsoft Office Project Server 2007
  
· Microsoft Certified IT Professional (MCITP): Enterprise Project Management with Microsoft Office Project Server 2007
  
· Microsoft Certified Technology Specialist (MCTS): Microsoft Office SharePoint Server 2007 – Configuration
  
· Microsoft Certified Technology Specialist (MCTS): Microsoft Windows SharePoint Services 3.0 – Application Development
  
· Microsoft Certified Technology Specialist (MCTS): BizTalk Server 2006
  
· MCTS: Microsoft System Center Configuration Manager 2007
  
· MCTS: Microsoft System Center Operations Manager 2007, Configuring
  
· MCTS: Microsoft Internet Security and Acceleration Server 2006, Configuring
  
· MCTS: Microsoft Forefront Client and Server, Configuring
