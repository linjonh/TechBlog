    <p>
     小程序发布流程
     <br/>
     <strong>
      注意：我们在发布小程序之前，需要配置服务器域名的白名单
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/61fd71a4b164fca60b365b607bd689a0.png#pic_center"/>
    </p>
    <h3>
     <a id="1_3">
     </a>
     1、首先是上传代码
    </h3>
    <p>
     打开我们小程序开发者工具，右上角一般会有一个上传按钮。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5613cba64d0f48f761cb1e9b001635b2.png#pic_center">
      <br/>
      然后会提示输入版本号，和项目备注，然后点击上传。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6a4f411cdd1b7e7a4c768814741ac7fa.png#pic_center">
       <br/>
       然后上传成功会提示你：上传成功。
      </img>
     </img>
    </p>
    <h3>
     <a id="2_9">
     </a>
     2、提交审核
    </h3>
    <p>
     接着我们去小程序的官网
     <a href="https://mp.weixin.qq.com" rel="nofollow">
      小程序官网
     </a>
     <br/>
     登录之后我们找到左边的版本管理，然后点击提交审核。
     <br/>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/98a6f0d342c7b2d24c52e51712440457.png#pic_center">
      <br/>
      首次提交审核会让你去完成基本资料。前去填写就可
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5e9096b5153e4b251285dc75c963a049.png#pic_center">
       <br/>
       <strong>
        填写小程序信息
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7634cd56f84cf65f121df8d3da25a4a9.png#pic_center">
        <br/>
        然后我们填写一下
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f4620050070b0a3401e0f29b1754c456.png#pic_center">
         <br/>
         在返回去点击 添加开发者，然后我们接着点击编辑，这是编辑下面会出来添加开发者。
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/663e0244f812ddd3f116e23bbc912c68.png#pic_center">
          <br/>
          然后我们根据微信号来添加相应的开发者。
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5a18ce330fc911aa1609f0471047238a.png#pic_center">
           <br/>
           添加成功后我们返回主页，重新点击提交审核。
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/08cd332748380fc06facc708892b270a.png#pic_center">
            <br/>
            然后就弹出一个提示框。我们点击已阅读并了解平台审核规则，然后点击下一步即可。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6b61638fc4f634784f402ebaf1d829c1.png#pic_center"/>
            <br/>
            输入小程序审核信息
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1dfe8a2e12c0227663a2592c0dca0e14.png#pic_center"/>
            <br/>
            提交之后我们可以在首页中的版本管理中看到审核状态。
            <br/>
            成功或者失败后都会给我们返回消息的。
            <br/>
            如果成功了，我们在小程序中搜素我们小程序的名字，就可以使用了。
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
