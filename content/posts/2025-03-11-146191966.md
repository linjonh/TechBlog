    <h2>
     <a id="Excel__0">
     </a>
     Excel 中如何实现数据透视表？
    </h2>
    <p>
     数据透视表（PivotTable）是 Excel 中强大的数据分析工具，能够快速汇总、分析和展示大量数据。本文将详细介绍如何在 Excel 中创建和使用数据透视表。
    </p>
    <hr/>
    <h3>
     <a id="1__6">
     </a>
     1. 数据透视表的基本概念
    </h3>
    <p>
     数据透视表是一种交互式表格，可以对数据进行动态汇总和分析。它可以帮助你：
    </p>
    <ul>
     <li>
      快速汇总数据。
     </li>
     <li>
      分析数据的趋势和模式。
     </li>
     <li>
      生成报表和图表。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="2__15">
     </a>
     2. 创建数据透视表
    </h3>
    <h4>
     <a id="21__17">
     </a>
     2.1 准备数据
    </h4>
    <ul>
     <li>
      确保数据区域是一个连续的表格，且每列都有标题。
     </li>
     <li>
      数据中不应有空行或空列。
     </li>
    </ul>
    <h4>
     <a id="22__21">
     </a>
     2.2 插入数据透视表
    </h4>
    <ol>
     <li>
      选中数据区域。
     </li>
     <li>
      点击
      <strong>
       “插入”
      </strong>
      选项卡，选择
      <strong>
       “数据透视表”
      </strong>
      。
     </li>
     <li>
      在弹出的对话框中：
      <ul>
       <li>
        选择数据区域（通常会自动识别）。
       </li>
       <li>
        选择放置数据透视表的位置（新工作表或现有工作表）。
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       “确定”
      </strong>
      。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="3__31">
     </a>
     3. 配置数据透视表
    </h3>
    <h4>
     <a id="31__33">
     </a>
     3.1 添加字段
    </h4>
    <ul>
     <li>
      在数据透视表字段列表中，将字段拖拽到以下区域：
      <ul>
       <li>
        <strong>
         行
        </strong>
        ：用于分组的字段。
       </li>
       <li>
        <strong>
         列
        </strong>
        ：用于横向分组的字段。
       </li>
       <li>
        <strong>
         值
        </strong>
        ：用于汇总计算的字段。
       </li>
       <li>
        <strong>
         筛选器
        </strong>
        ：用于筛选数据的字段。
       </li>
      </ul>
     </li>
    </ul>
    <h4>
     <a id="32__40">
     </a>
     3.2 调整汇总方式
    </h4>
    <ul>
     <li>
      默认情况下，数值字段会进行
      <strong>
       求和
      </strong>
      计算。
     </li>
     <li>
      可以右键点击值字段，选择
      <strong>
       “值字段设置”
      </strong>
      ，更改汇总方式（如平均值、计数、最大值等）。
     </li>
    </ul>
    <h4>
     <a id="33__44">
     </a>
     3.3 添加筛选器
    </h4>
    <ul>
     <li>
      将字段拖拽到
      <strong>
       筛选器
      </strong>
      区域，可以在数据透视表上方添加筛选器，动态筛选数据。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="4__49">
     </a>
     4. 数据透视表的常用操作
    </h3>
    <h4>
     <a id="41__51">
     </a>
     4.1 刷新数据
    </h4>
    <ul>
     <li>
      如果源数据发生变化，可以右键点击数据透视表，选择
      <strong>
       “刷新”
      </strong>
      。
     </li>
    </ul>
    <h4>
     <a id="42__54">
     </a>
     4.2 更改布局
    </h4>
    <ul>
     <li>
      在
      <strong>
       “设计”
      </strong>
      选项卡中，可以选择不同的布局样式（如压缩、大纲、表格）。
     </li>
    </ul>
    <h4>
     <a id="43__57">
     </a>
     4.3 添加计算字段
    </h4>
    <ol>
     <li>
      点击数据透视表，选择
      <strong>
       “分析”
      </strong>
      选项卡。
     </li>
     <li>
      点击
      <strong>
       “字段、项目和集”
      </strong>
      ，选择
      <strong>
       “计算字段”
      </strong>
      。
     </li>
     <li>
      输入公式并命名新字段。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="5__64">
     </a>
     5. 数据透视表的进阶技巧
    </h3>
    <h4>
     <a id="51__66">
     </a>
     5.1 分组数据
    </h4>
    <ul>
     <li>
      右键点击行或列字段，选择
      <strong>
       “分组”
      </strong>
      ，可以按日期、数字范围等进行分组。
     </li>
    </ul>
    <h4>
     <a id="52__69">
     </a>
     5.2 创建数据透视图
    </h4>
    <ol>
     <li>
      点击数据透视表，选择
      <strong>
       “分析”
      </strong>
      选项卡。
     </li>
     <li>
      点击
      <strong>
       “数据透视图”
      </strong>
      ，选择图表类型。
     </li>
     <li>
      数据透视图会与数据透视表联动，动态展示数据。
     </li>
    </ol>
    <h4>
     <a id="53__74">
     </a>
     5.3 使用切片器
    </h4>
    <ol>
     <li>
      点击数据透视表，选择
      <strong>
       “分析”
      </strong>
      选项卡。
     </li>
     <li>
      点击
      <strong>
       “插入切片器”
      </strong>
      ，选择需要筛选的字段。
     </li>
     <li>
      切片器可以直观地筛选数据。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="6__81">
     </a>
     6. 注意事项
    </h3>
    <ol>
     <li>
      <strong>
       数据格式
      </strong>
      ：确保数据格式一致，避免错误。
     </li>
     <li>
      <strong>
       数据更新
      </strong>
      ：如果源数据变化，记得刷新数据透视表。
     </li>
     <li>
      <strong>
       字段命名
      </strong>
      ：字段名称应清晰易懂，方便后续分析。
     </li>
    </ol>
