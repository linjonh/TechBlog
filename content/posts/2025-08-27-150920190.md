# 如何在阿里云百炼中使用钉钉MCP

本文通过阿里云百炼+钉钉MCP配合，完成钉钉AI表格（多维表）数据管理 ，其他AI开发工具可参照本文完成部署。

### **准备工作**

1. 在正式开始前，需要提前了解什么是钉钉MCP，详情请参考[钉钉服务端API MCP 概述](https://open.dingtalk.com/document/ai-dev/dingtalk-server-api-mcp-overview# "钉钉服务端API MCP 概述")。
2. 已经注册了阿里云账号，如未注册，请前往[阿里云官网](https://www.aliyun.com/benefit?utm_content=se_1021124579 "阿里云官网")注册。
3. [创建一个企业内部应用](https://open.dingtalk.com/document/dingstart/create-application# "创建一个企业内部应用")且拥有**钉钉企业管理员**或**子管理员**权限，并获取[Client ID/Client Secret](https://open.dingtalk.com/document/dingstart/basic-concepts-beta#section-pje-9wf-l7c "Client ID/Client Secret")**。**
4. 已经完成了权限申请，权限点请查看[钉钉 MCP 功能清单](https://open.dingtalk.com/document/ai-dev/dingtalk-mcp-feature-list# "钉钉 MCP 功能清单")。

### **步骤一：开通钉钉MCP服务**

1. 登录并访问[阿里云百炼](https://bailian.console.aliyun.com/?spm=5176.29597918.J_SEsSjsNv72yRuRFS2VknO.2.36c67b08LBjcPC&tab=mcp#/mcp-market "阿里云百炼")平台，在MCP类目下选择**钉钉**。

![](https://i-blog.csdnimg.cn/img_convert/47c0964e590d4e556f5f92afca9ef0af.png)

2 .在钉钉MCP详情页，点击**立刻开通**，开通钉钉MCP服务。

![](https://i-blog.csdnimg.cn/img_convert/5e00694833d9760de4beaa2a0ce73788.png)

在授权界面，点击**同意授权**后，进入参数设置界面。

![](https://i-blog.csdnimg.cn/img_convert/4670e0a0ed44bedd7b7280f52c1e9d55.png)

3. 在参数设置界面，填写参数值，选择部署模式后点击**确认开通**即可。

* **DINGTALK_Client_ID**：钉钉应用的Client ID，详情可参考[Client ID/Client Secret](https://open.dingtalk.com/document/dingstart/basic-concepts-beta#section-pje-9wf-l7c "Client ID/Client Secret")。
* **DINGTALK_Client_Secret**：钉钉应用的Client Secret ，详情可参考[Client ID/Client Secret](https://open.dingtalk.com/document/dingstart/basic-concepts-beta#section-pje-9wf-l7c "Client ID/Client Secret")。
* **ACTIVE_PROFILES**：启用的服务模块，"ALL" 表示全部启动，详情可参考[安装并使用钉钉 MCP](https://open.dingtalk.com/document/ai-dev/second-level-node-1# "安装并使用钉钉 MCP")。
* **部署模式**：本示例中选择“公共FC资源部署”。

  ![](https://i-blog.csdnimg.cn/img_convert/b443870e3c6838dbd4b554cde87282d6.png)

  4. 服务开通成功后，钉钉MCP详情页如下图所示：

1. ![](https://i-blog.csdnimg.cn/img_convert/859f82222a1312b845866dbaabfcccd6.png)

### **步骤二：创建应用**

1. 登录并访问[阿里云百炼](https://bailian.console.aliyun.com/?spm=5176.29597918.J_SEsSjsNv72yRuRFS2VknO.2.36c67b08LBjcPC&tab=mcp#/mcp-market "阿里云百炼")平台，在应用类目下，选择应用管理，并点击创建应用，创建一个应用。

![](https://i-blog.csdnimg.cn/img_convert/1563004d7af089f2285098851b7e84b0.png)

2. 选择**智能体应用**，并单击**立即创建**。

![](https://i-blog.csdnimg.cn/img_convert/4b080b4b252b8a4768491ae1426518ca.png)

3. 在**应用配置**中，设置智能体应用的基本信息，并绑定**MCP服务**。

**说明**

智能体应用提供了长期记忆功能，长期记忆可以提取对话的关键信息并保存至对应的记忆体（Memory ID）中，详情可查看[长期记忆](https://open.dingtalk.com/document/application-user-guide/long-term-memory# "长期记忆")。

![](https://i-blog.csdnimg.cn/img_convert/7aa0863d38a38fe865637cf32417feed.png)

4. 在**已开通**的类目下，选择钉钉MCP服务，最后点击**确认**即可。

![](https://i-blog.csdnimg.cn/img_convert/37194d6d803898d844daf4166f5455ee.png)

5. 绑定成功后，在发布渠道中设置需要发布的基本信息。

**说明**

可以基于已创建的应用 API 进行“发布到平台”或“集成其他能力继续开发”的操作，本教材以网页版体验为主，若需要发布到其他平台，可根据实际情况进行选择。

![](https://i-blog.csdnimg.cn/img_convert/af7916d46ea05d6e6849d08c3a8b52ea.png)

点击创建后，系统会自动生成一个有效时间为7 天临时访问链接，你需要复制该链接，用户后续的体验。

![](https://i-blog.csdnimg.cn/img_convert/b3b935ab59b0529cd6092ff79347f331.png)

6. 绑定成功后，在应用界面点击右上角**发布**按钮，发布当前应用即可。

![](https://i-blog.csdnimg.cn/img_convert/b53959a082257ba69af2eb1ddd579a96.png)

### **步骤三：智能体应用体验**

本教材以查询某人基本信息为示例，通过询问智能体，查询某人的姓名、userId和头像等基础信息。在正式体验前，需要在钉钉应用的权限设置中开通基础权限。

**说明**

在正式体验前，你需要开通以下基础权限：

* 读取用户信息：qyapi_get_member
* 根据手机号获取成员基本信息：qyapi_get_member_by_mobile

1. 通过浏览器打开步骤二中复制的链接，如下图所示：

![](https://i-blog.csdnimg.cn/img_convert/a7386858b149b15a2d8f9fcd5ba84ab6.png)

2. 在输入框中输入需要提问内容，即可获取相关数据结果，例如：查询张三的基本信息。

![](https://i-blog.csdnimg.cn/img_convert/14c8a0ee366f921a6d5e41dc776623d3.png)

3.至此，你已经了解了百炼与钉钉MCP的基本使用，如需使用钉钉其他能力，可参考[安装并使用钉钉 MCP](https://open.dingtalk.com/document/ai-dev/second-level-node-1# "安装并使用钉钉 MCP")文档中说明，添加接口权限后即可体验。
