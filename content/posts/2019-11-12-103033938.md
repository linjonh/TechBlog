    <h3>
     <a id="_0">
     </a>
     如何从零开始拥有自己的网站？
    </h3>
    <h4>
     <a id="_2">
     </a>
     一、购买服务器
    </h4>
    <p>
     阿里云推出了一个云翼计划的产品，通过学生认证（或24岁以下）之后可以享受超低的价格，原价1700多一年，学生只需要114RMB，下面将详细介绍如何购买阿里云服务器。
    </p>
    <h5>
     <a id="1_6">
     </a>
     1.入口
    </h5>
    <p>
     <a href="https://www.aliyun.com/" rel="nofollow">
      云翼计划
     </a>
    </p>
    <h5>
     <a id="2_9">
     </a>
     2.认证
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/63d671e420bfd141e7a78a474106b1bc.png">
      <br/>
      第一步：登陆（可以使用支付宝扫码登陆，方便快捷）
      <br/>
      第二步：完成实名认证
      <br/>
      第三步：完成学生认证。
      <br/>
      我在这一步时输入学籍信息总是提示我信息有误，还好我是24岁以下，完成第二步返回首页刷新一下第三步就自动完成了。然后就可以进行购买了。
     </img>
    </p>
    <h5>
     <a id="3_15">
     </a>
     3.购买。
    </h5>
    <p>
     我选择的是linux系统
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0f69d0e3925912da93f0efdac0043047.png"/>
    </p>
    <h5>
     <a id="4_18">
     </a>
     4.支付
    </h5>
    <p>
     我返回订单好几次，实际上我只支付了114RMB，只要你按照步骤来，这里肯定是114，勾选右下角两个框，去支付
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/01101967773295ba67727dd04df61e29.png"/>
    </p>
    <h5>
     <a id="5_21">
     </a>
     5.支付成功，进入控制台（这两个地方都可以）
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8df3ac9a61b9d35334bf5c694e7d89a2.png"/>
    </p>
    <h4>
     <a id="_23">
     </a>
     二、管理控制台，配置服务器
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5375fa3417f4fc7717d08847f3357c03.png">
      <br/>
      恭喜，现在你已经拥有属于自己的服务器了，下面我们来对其进行一些配置，方便我们部署web项目。
     </img>
    </p>
    <h5>
     <a id="1__26">
     </a>
     1. 设置端口。
    </h5>
    <p>
     添加规则，这里我们把tomcat和mysql的访问端口设置进来。服务器需要端口放行，否则你的端口是无法访问的，因为安全需要，所以云服务器存在安全组这个东西。记住常用的端口放行，tomcat 8080端口，mysql 3306端口 https 443和80端口。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cc2e22bced15b20b2d2961ede7acb9f2.png"/>
    </p>
    <h5>
     <a id="2__29">
     </a>
     2. 第一次远程连接需要设置用户名和密码。
    </h5>
    <p>
     设置密码之后可以远程连接
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/51dc74ae83aad8389682b18c17b94569.png"/>
    </p>
    <h5>
     <a id="3__32">
     </a>
     3. 远程连接,
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cb724468e4548b41fcb4b070a0a5b20c.png">
      <br/>
      可以在命令行输入一些命令对linux系统进行操作（步骤4），也可以通过软件对系统进行操作（步骤6）。
     </img>
    </p>
    <h5>
     <a id="4__35">
     </a>
     4. 首先切换成管理员权限。
    </h5>
    <p>
     输入；sudo su root创建存放jdk,tomcat的文件夹
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8370024f5806193c7b9123247ca504bd.png"/>
    </p>
    <h5>
     <a id="5JDKTOMCAT_38">
     </a>
     5.下载JDK和TOMCAT
    </h5>
    <p>
     （1） JDK(这里选择的是JDK1.8)
     <br/>
     下载地址为
     <a href="http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html" rel="nofollow">
      http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
     </a>
     <br/>
     我选择的是jdk-8u131-linux-x64.tar.gz
     <br/>
     （2） Tomcat 8
     <br/>
     下载地址为
     <a href="http://tomcat.apache.org/download-80.cgi#8.0.46" rel="nofollow">
      http://tomcat.apache.org/download-80.cgi#8.0.46
     </a>
     <br/>
     我选择的是apache-tomcat-8.5.47.tar.gz
    </p>
    <h5>
     <a id="6_XFtp_45">
     </a>
     6. 使用XFtp上传文件到服务器.
    </h5>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4794bf8714ef59a4f195c5184e0cceab.png"/>
     <br/>
     右边创建文件夹存放JDK和TOMCAT。选中左边的文件直接拖过去就可以了。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f8494bfb50a6ea2fc6abeb3dd7064700.png"/>
    </p>
    <h5>
     <a id="7_XShell_50">
     </a>
     7. 使用XShell查看文件是否已经成功上传到对应的文件夹下。
    </h5>
    <p>
     连接XShell
     <br/>
     （1）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c4e8f487e800afa15c292a9cf1678982.png"/>
     <br/>
     （2）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/05b71fd55352880372f9de50a9fe1e86.png"/>
     （3）
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8ed4681f3aac2d2d6a2e11e33c926da2.png"/>
     <br/>
     查看文件是否存在，我当时把mysql也传上去了，所以会有三个文件。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ce24af35825b6c5ef7a320d51ad88cfb.png"/>
     <br/>
     解压tomcat和jdk
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d6ec87422f4f6022ae6b9b4cb8041570.png"/>
     <br/>
     可以给解压后的两个文件改个名字，方便操作。
     <br/>
     mv 文件之前的名字 文件之后的名字
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4f86639a62725042605e236e95a17231.png"/>
    </p>
    <h5>
     <a id="8jdktomcat_64">
     </a>
     8配置jdk和tomcat的环境变量。
    </h5>
    <p>
     有两种配置方法，一种是在vim下编辑，另一种是使用XFTP，复制到本地，修改后再上传覆盖原来的文件即可。这里推荐第二种方法。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cd37221e4aef6263c6bef303efc6c693.png"/>
     <br/>
     在文档末尾输入以下信息
     <br/>
     JAVA_HOME=/存放文件地址/你的jdk文件夹名
     <br/>
     PATH=/存放文件地址/你的jdk文件夹名/bin:$PATH
     <br/>
     CLASSPATH=.:/存放文件地址/你的jdk文件夹名/lib/dt.jar:/存放文件地址/你的jdk文件夹名/lib/tools.jar
     <br/>
     CATALINA_HOME=/存放文件地址/你的tomcat文件夹名export
     <br/>
     JAVA_HOME PATH CLASSPATH CATALINA_HOME
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9cf707a243507944893988ed66af9eba.png"/>
     <br/>
     修改后导入即可
     <br/>
     打开XShell，输入以下内容，查看jdk环境变量是否配置正确
    </p>
    <pre><code class="prism language-powershell"> source <span class="token operator">/</span>etc/profile
</code></pre>
    <pre><code class="prism language-powershell">javac
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cac9da1182e71b0ed64eefcf1f0ee784.png"/>
     <br/>
     启动tomcat，若出现以下信息，说明配置正确。
    </p>
    <pre><code class="prism language-powershell">cd <span class="token operator">/</span>root/software/tomcat8/bin
<span class="token punctuation">.</span><span class="token operator">/</span>satrtup<span class="token punctuation">.</span>sh
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f7ac62b1eb6da4d6a4e224ef1cc9ee5b.png"/>
     <br/>
     恭喜！现在服务器已经搭建完毕！可通过浏览器访问服务器中的tomcat.
     <br/>
     输入http://服务器IP地址：8080
     <br/>
     返回tomcta主页表示成功！
    </p>
    <h3>
     <a id="web_96">
     </a>
     三、上传网站源码，部署web项目
    </h3>
    <p>
     1.使用XFtp找到tomcat下webapps文件夹
     <br/>
     2.将网站的源码整个放入webapps下
     <br/>
     3.通过:服务器IP地址：8080/项目名/放页面的文件夹/index.jsp
     <br/>
     如果页面是index.html，通过：服务器IP地址：8080/项目名/放页面的文件夹/index就可以访问到自己写的网站了。
     <br/>
