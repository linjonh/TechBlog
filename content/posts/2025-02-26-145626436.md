     <h2>
      <a id="_3">
      </a>
      前言
     </h2>
     <p>
      没个像样显示器，就好比揣着三八大盖在战场上突突，虽然略有夸张，但是对于长期从事软件开发岗位的开发者，巴适的显示器对眼睛保护以及工作效率都具有非常大的好处。然而我期待的显示器希望具备：
     </p>
     <ul>
      <li>
       支持升降式+旋转，提高看代码体验
      </li>
      <li>
       高度的色彩还原性
      </li>
      <li>
       硬件级滤蓝光+防眩光
      </li>
      <li>
       有丰富的接口，不需要转接器就可以直接使用
      </li>
     </ul>
     <p>
      目前我所使用的显示器只是一个普通显示器，经常会有下面的现象
     </p>
     <ul>
      <li>
       因分屏不足而频繁切换窗口
      </li>
      <li>
       因色差问题，跟设计人员多次沟通
      </li>
      <li>
       低频闪屏、固定视角，影响开发体验
      </li>
      <li>
       屏幕防蓝光一般，不具备防眩晕功能
      </li>
      <li>
       使用过程中需要转接器
      </li>
     </ul>
     <p>
      我在网上查了很多显示器，最终选择了
      <code>
       明基RD320U
      </code>
      。
      <code>
       RD320U
      </code>
      真的是开发者利器，32英寸4K超高清显示器，具有编码专用色彩模式、独特的编码菜单栏，真的是我的菜！
     </p>
     <blockquote>
      <p>
       本文主要记录下自己购买明基RD320U之后的开箱体验
      </p>
     </blockquote>
     <h2>
      <a id="_25">
      </a>
      开箱
     </h2>
     <p>
      快递盒子很大，整个箱子尺寸为78cm * 52cm * 28cm，还是非常重的。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/987741ed5b97415c8a2eec6ca18d90b0.png">
       <br/>
       打开包装之后已经感受到品牌的用心，一共有两层，上层：电源线、Type-B 线、Type-C 线和 HDMI 线、底座，配件还是非常齐全的。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/222ecc5a736e4074b630b41bae45bc68.png">
        <br/>
        下层：显示器主体、可调节人体工学支架 。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1995c790324248669effbedcff47fae9.png"/>
       </img>
      </img>
     </p>
     <h2>
      <a id="_33">
      </a>
      巴适的显示视觉
     </h2>
     <p>
      <code>
       明基RD320U
      </code>
      的基本配置：31.5英寸，4K超高清，98%DCI-P3色域，16:9屏比，HDR400、抗反射面板、HiDPi，2000:1对比度。
     </p>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ffd9648745e5422abd5e3fddd13153a3.png"/>
     </p>
     <h2>
      <a id="_39">
      </a>
      接口丰富
     </h2>
     <p>
      <code>
       明基RD320U
      </code>
      支持两个HDMI、DP1.4、Usb3.2(TypeC/90W)、Usb3.2(TypeC/下行)、Usb3.2(TypeB)、3.5mm 耳机接口、两个Usb3.2(TypeA/5Gbps)接口，并且还支持KVM功能，可以轻松地在两台机器之间切换。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f2c8903abf744b77aca6cf84f16138ba.png"/>
     </p>
     <h2>
      <a id="_44">
      </a>
      编码专用的色彩模式
     </h2>
     <p>
      <code>
       明基RD320U
      </code>
      有多种色彩模式支持，当你亲自切换模式的时候，会有非常棒的视觉冲击效果，它通过硬件层面加强语法高亮的效果，使代码更容易辨别区分。这里我使用普通显示器与RD320U显示器，可以看出普通显示器的高亮效果对眼睛产生的视觉冲突显然没有RD320U显示器的大，并且RD320U显示器亮色主题和深色主题，让我不论在白天还是夜晚在开发时对眼睛都非常友好。

     <h2>
      <a id="DualView_Plus__61">
      </a>
      DualView Plus 多色彩模式
     </h2>
     <p>
      令人惊喜的
      <code>
       DualView Plus
      </code>
      功能可以自定义应用程序或视窗间的颜色模式，从而简化多屏幕显示。另外RD320U对
      <code>
       PIP和PBP
      </code>
      的支持，可以轻松管理双来源或单一来源的视图，进行分屏显示，屏幕也可设置尺寸，更适合前端工程师和UI设计。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d22779b68e2248c2be514d6b2e5a7b4e.png"/>

     <p>
      并且RD320U提供丰富、生动的色彩表现，
      <code>
       98%DCI-P3色域
      </code>
      具有出色的色彩表现能力，能够覆盖DCI-P3色彩空间中绝大部分的颜色范围，从而为用户提供丰富、生动的视觉体验。我们可以对比一下：

     <h2>
      <a id="_73">
      </a>
      防蓝光&amp;防眩光
     </h2>
     <p>
      OSD菜单中的
      <code>
       EyeCare
      </code>
      模式，可以开启“
      <code>
       智慧蓝光
      </code>
      ”功能，能够过滤一部分蓝光，保护视力。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d282ad884b4540f491552524d3a0b842.png"/>
     </p>
     <p>
      <code>
       明基RD320U
      </code>
      使用莱茵认证的
      <code>
       抗反射面板
      </code>
      ，通过覆盖抗反射涂层，减少屏幕眩光对代码是别的干扰，更专注代码本身。可以看一下下面普通显示器以及RD320U的对比，真的是非常明显，普通显示器屏幕明显可以看到光晕以及人影，而RD320U看不到人影，眩光度非常低且代码清晰可见。

     <h2>
      <a id="MoonHalo_87">
      </a>
      智慧光环MoonHalo
     </h2>
     <p>
      在显示器后面有
      <code>
       智慧光环MoonHalo
      </code>
      ，营造出沉浸式的工作氛围，这个贴心的背光设计，让我可以在夜间敲代码更加专注，也非常的护眼。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c66f14a8f7ef44699bb2fc9572802e94.png"/>
      <br/>
      可以通过编码触键来开启，向左或者向右，就可以触发这个功能，可以设置亮度、色温、灯光模式（270度/360度）
     </p>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/18d393cfcf184ebea013cb456adc3106.png"/>
     </p>
     <p>
      下图是我在晚上编程所拍的照片，完全不需要打开其他的灯光，只要开启
      <code>
       MoonHalo
      </code>
      ，体验感直接拉满，敲代码更加专注了！
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e3f59e2e43ac497ca0f53938d0aac63f.png"/>

     <h2>
      <a id="_103">
      </a>
      人体工学设计
     </h2>
     <p>
      人体工学设计这点非常契合我的买点，谁不想轻轻松松看屏幕敲代码、看设计图呢！显示器底座可以升降式调节，最高可调节110MM，也可以倾斜调整-5˚~20˚，旋转-15˚～15˚。并且后面还有专门的
      <code>
       皮质束线带
      </code>
      ，可以将多余的线长进行固定，防止线混乱在桌面，这点也是我非常喜欢的。
     </p>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cafb8aec658b418086cf0e14a431cbb0.png"/>
     </p>
     <p>
      并且
      <code>
       明基RD320U
      </code>
      这款支架臂比起之前的款式做了升级。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/591c8a33237744cf87ddd282b0eba79a.png"/>

      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/35b5965b225e4e218f7df53d7ea023c6.png"/>

     <h2>
      <a id="Display_Pilot2__117">
      </a>
      Display Pilot2 软件辅助
     </h2>
     <p>
      另外可以使用官方的
      <code>
       Display Pilot2
      </code>
      进行一键配置。同时还支持在不同的应用中，自动切换当前显示器的颜色模式。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/1895be18425d4f3e879a082e6fe78c72.png"/>
      <br/>
      Display Pilot2支持需要功能比如：一键调试屏幕功能模式、提供多个搜索引擎快速搜索、多功能场景切换、键盘快捷方式。通过该软件更快捷的设置自己需要的模式，非常方便
     </p>
     <h2>
      <a id="_121">
      </a>
      最后
     </h2>
     <p>
      <code>
       明基RD320U
      </code>
      对我的帮助非常大，已经成为我非常亲密的小伙伴儿！程序员真的应该配置一台这样的显示器，对自己的眼睛、工作效率等都能起到非常大的作用，在多次使用之后，非常巴适。3:2屏幕比例让我能在单屏上看到更多代码，加上4K分辨率和抗反射面板，不管是白天还是晚上，看着都很舒服，尤其在深色主题下，眼睛也不容易累。还有KVM功能、自定义快捷键功能，以及配套的Display Pilot 2软件，真的让我编程时顺手多了，非常nice!
     </p>
