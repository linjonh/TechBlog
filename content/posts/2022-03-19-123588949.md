    <h2>
     <a id="_10">
     </a>
     前言
    </h2>
    <p>
     WebRTC 实现P2P音视频通话系列记录了从零-&gt;搭建信令服务器-&gt;搭建stun/turn P2P穿透和转发服务器-&gt;WebRTC P2P音视频通话。
     <br/>
     WebRTC 实现P2P音视频通话——搭建stun/turn P2P穿透和转发服务 本文将在公网服务器主机上使用coturn搭建一个穿透和转发服务器。
     <br/>
     Coturn集成了stun+turn协议,实现NAT检测，穿透就需要通过stun协议， NAT检测无法进行穿越时就需要通过turn服务进行流媒体的转发了，而coturn就是将两者协议进行整合并进行择优优化， 在WebRTC端到端音视频传输 stun/turn 穿透和转发是必不可少的 (除非自己实现一个流媒体服务器)。
    </p>
    <h2>
     <a id="_15">
     </a>
     环境：
    </h2>
    <p>
     1.公网服务器主机或内网主机(内网验证，需要在同一内网)，只是临时做验证可以使用阿里云实验室，两小时免费
     <a href="https://developer.aliyun.com/adc/scenario/bd3fac7dfaaf4be3abbc1bb980907f70?spm=a2c6h.13858375.0.0.629079a9BqQXVI" rel="nofollow">
      阿里云实验室
     </a>
     <br/>
     2.Ubuntu Ubuntu Server 20.04 LTS 64bit
    </p>
    <h2>
     <a id="Coturn_18">
     </a>
     一、安装Coturn
    </h2>
    <p>
     通过SSH进入服务器主机
    </p>
    <pre><code class="prism language-powershell">ssh 用户名@IP地址 例如：ssh root@192<span class="token punctuation">.</span>168<span class="token punctuation">.</span>1<span class="token punctuation">.</span>1
</code></pre>
    <p>
     拉取coturn源码 并进入
    </p>
    <pre><code class="prism language-powershell">git clone https:<span class="token operator">/</span><span class="token operator">/</span>github<span class="token punctuation">.</span>com/coturn/coturn 
</code></pre>
    <p>
     提示 -bash: /usr/bin/git: No such file or directory 时就需要安装一下git了
    </p>
    <pre><code class="prism language-powershell">apt install git
</code></pre>
    <p>
     进入coturn 目录下，配置coturn并指定–prefix安装路径
    </p>
    <pre><code class="prism language-powershell">cd coturn
<span class="token punctuation">.</span><span class="token operator">/</span>configure <span class="token operator">--</span>prefix=<span class="token operator">/</span>usr/local/coturn 
</code></pre>
    <p>
     配置过程中碰到的问题
    </p>
    <blockquote>
     <p>
      ERROR: pkg-config not found
      <br/>
      <img alt="示例：pandas 是基于NumPy 的一种工具，该工具是为了解决数据分析任务而创建的。" src="https://i-blog.csdnimg.cn/blog_migrate/937800fb8af66873a8843ddc71b98493.png">
       <br/>
       apt install pkg-config
      </img>
     </p>
    </blockquote>
    <blockquote>
     <p>
      ERROR: OpenSSL Crypto development libraries are not installed properly in required location.
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8ee31dfc844f1a7f6fdf840a570961ec.png">
       <br/>
       apt install libssl-dev
      </img>
     </p>
    </blockquote>
    <blockquote>
     <p>
      ERROR: Libevent2 development libraries are not installed properly in required location.
      <br/>
      ERROR: may be you have just too old libevent tool - then you have to upgrade it.
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d871f11b689cbdcabcb8f38118c03ec2.png">
       <br/>
       apt install libevent-dev
      </img>
     </p>
    </blockquote>
    <p>
     安装好相关依赖库，在重新./configure
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a22db82dc0b72f9ccc9fb0b3e214bfaf.png">
      <br/>
      配置成功后就可以编译，安装了
     </img>
    </p>
    <pre><code class="prism language-powershell">make &amp; make install
</code></pre>
    <h2>
     <a id="Coturn_58">
     </a>
     二、配置环境变量及Coturn
    </h2>
    <h4>
     <a id="1_59">
     </a>
     1.配置环境变量
    </h4>
    <p>
     修改bashrc配置文件
    </p>
    <pre><code class="prism language-powershell">vi ~<span class="token operator">/</span><span class="token punctuation">.</span>bashrc 
</code></pre>
    <p>
     最文件最后添加coturn的安装路径
    </p>
    <pre><code class="prism language-powershell">export PATH=<span class="token operator">/</span>usr/local/coturn/bin:<span class="token variable">$PATH</span>:
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/71fcaa9b4ec5fee5789439dee5623146.png">
      <br/>
      重新应用bashrc配置文件
     </img>
    </p>
    <pre><code class="prism language-powershell">source ~<span class="token operator">/</span><span class="token punctuation">.</span>bashrc 
</code></pre>
    <h4>
     <a id="2Coturn_73">
     </a>
     2.配置Coturn
    </h4>
    <p>
     先创建用户，密码并指定域名
    </p>
    <pre><code class="prism language-powershell">turnadmin <span class="token operator">-</span>a <span class="token operator">-</span>u 用户名 <span class="token operator">-</span>p 密码 <span class="token operator">-</span>r 域名（给自己的域名）
turnadmin <span class="token operator">-</span>a <span class="token operator">-</span>u test <span class="token operator">-</span>p 123456 <span class="token operator">-</span>r wenpipi
</code></pre>
    <p>
     进入到/usr/local/coturn/etc目录，拷贝一份turnserver.conf.default
    </p>
    <pre><code class="prism language-powershell">cd <span class="token operator">/</span>usr/local/coturn/etc
<span class="token function">cp</span> turnserver<span class="token punctuation">.</span>conf<span class="token punctuation">.</span>default turnserver<span class="token punctuation">.</span>conf
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a2187ec72ced3f34cb7a7a24aca46a38.png">
      <br/>
      修改拷贝出来的配置文件turnserver.conf
     </img>
    </p>
    <pre><code class="prism language-powershell">vi turnserver<span class="token punctuation">.</span>conf
</code></pre>
    <p>
     修改下面👇对应的值
    </p>
    <pre><code class="prism language-powershell"><span class="token comment">#监听端口</span>
<span class="token comment">#公网ip</span>
<span class="token comment">#用户名:密码</span>
<span class="token comment">#域名。需要跟创建用户，启动时的相同</span>
listening-port=3478 
external-ip=139<span class="token punctuation">.</span>196<span class="token punctuation">.</span>187<span class="token punctuation">.</span>17 
user=test:123456
realm=wenpipi 
</code></pre>
    <p>
     以上配置完成，就可以启动turnserver了
    </p>
    <pre><code class="prism language-powershell">turnserver <span class="token operator">-</span>o <span class="token operator">-</span>a <span class="token operator">-</span>f <span class="token operator">-</span>v <span class="token operator">-</span>r wenpipi
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/625428a5f71a2b7d447fcc00bda11c41.png">
      <br/>
      查看活跃端口，就能看到启动的turnserver服务了
     </img>
    </p>
    <pre><code class="prism language-powershell">netstat <span class="token operator">-</span>ntpl
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d53738342c2f4b1f52cb5ea993687f69.png"/>
    </p>
