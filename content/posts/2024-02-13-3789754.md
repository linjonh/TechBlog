     什么是微软最有价值专家 (MVP)
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/BrightskyLiu/EntryImages/20090115/mvptitle.gif"/>
    </p>
    <p>
     <a href="http://www.microsoft.com/china/community/Professional/mvp/regnow.aspx" rel="nofollow">
      <img alt="" height="39" src="https://i-blog.csdnimg.cn/blog_migrate/fc5c8a6d055930130d056054159dd05c.png" width="110"/>
     </a>
    </p>
    <p class="text4">
     微软最有价值专家 (MVP) 是指具备一种或多种微软技术专业知识，并且积极参与在线或离线的社群活动，经常与其他专业人士分享知识和专业技能，受人尊敬、信任，而且平易近人的专家。
    </p>
    <p class="text4">
     实际生活中，人们总是信任专家的建议和反馈。MVP正是这样一群拥有丰富知识和实际经验的微软技术专家。他们不是微软的员工，但是非常乐于通过在线或离线社区的方式帮助技术人士。另一方面，微软公司时刻不忘倾听来自用户的意见反馈，不断开发新产品，改进技术，提高用户体验。MVP代表来自社群的广大用户，他们的专家意见更能协助微软公司了解用户的真实需求。
    </p>
    <p class="text4">
     Most Valuable Professionals（最有价值专家）是微软对上述专家在技术社群专业贡献的一种正式认同。该项目主要目的在于鼓励形成一个充满活力的全球性社群，使得微软和用户之间建立良好的相互关系，增进相互了解。目前主要策略为：
    </p>
    <ul class="list1">
     <li class="list2">
      在全球范围内认可 MVP 并建立交流渠道 － 通过跨产品、服务和行业的广大社群，嘉奖有影响力和特殊贡献的专家，并赋予他们特殊的资源和权力
     </li>
     <li class="list2">
      贴近用户、提高体验 － 认可技术专家的特殊贡献，不论他们来自哪个领域、使用何种语言，致力提高使用微软技术的亲身体验
     </li>
     <li class="list2">
      推动项目日臻完美 － 不断提高对技术专家的支持力度，在全球范围建立协调和沟通网络，增进微软和用户的相互了解
     </li>
    </ul>
    <p class="text4">
     该项目已经运作了十一年，在全球81个国家拥有接近3000位最有价值专家。
    </p>
    <p class="text4">
     <strong>
      谁是我们的微软最有价值专家（MVP）？
     </strong>
    </p>
    <ul class="list1">
     <li class="list2">
      以微软技术为主题的作家、讲师、培训师
     </li>
     <li class="list2">
      IT业界的业内知名专业人士，通过印刷媒体、blog或其它形式分享经验和观点
     </li>
     <li class="list2">
      参与和微软技术有关的项目，担任主要角色的技术和管理人员
     </li>
     <li class="list2">
      建立讨论微软技术的技术网站，担任主要角色的技术和管理人员
     </li>
     <li class="list2">
      参与微软中文新闻组，积极地帮助论坛用户解决疑难问题的技术论坛高手
     </li>
     <li class="list2">
      参与其他第三方的微软技术论坛，积极地帮助论坛用户解决疑难问题的技术论坛高手
     </li>
    </ul>
    <p>
    </p>
    <p class="text4">
     <strong>
      为什么要参与微软最有价值专家的评选？
     </strong>
    </p>
    <ul class="list1">
     <li class="list2">
      <strong>
       成为微软全球3000位最有价值专家中的一员
      </strong>
      <br/>
      微软公司视所有的MVP为“最有价值合作伙伴”，目前全球仅有3000位MVP。在MVP颁奖峰会上，您经常能够看见Bill Gates, Steve Ballmer, Eric Rudder, Joe Peterson, Lori Moore 等微软的最高层领导出现在大会上，亲自向MVP致词。来自MVP的反馈、建议，一直都是微软高层和各个产品组极为重视的声音。
     </li>
     <li class="list2">
      <strong>
       扩大您的专家关系网络
      </strong>
      <br/>
      • MVP可以通过微软私有新闻组(private newsgroup)和3000多位世界各地的MVP在线交流。
      <br/>
      • MVP也可以通过中国地区的MVP邮件列表交流分享专业信息和资源。
     </li>
     <li class="list2">
      <strong>
       分享微软最前沿的技术资源
      </strong>
      <br/>
      • 所有MVP将获赠价值人民币32,000元的MSDN光盘宇宙版或TechNet的一年免费订阅。
      <br/>
      • 微软MVP可以在专有站点上下载微软最新的内部技术文档，访问合作伙伴级知识库(Partner Knowledge Base)，并参加专门为MVP制作的在线学院培训。
      <br/>
      • 微软总部的产品研发部门与MVP将保持密切的联系。MVP将参与公司最新技术的审核和产品Beta测试，并可以透过特殊渠道对微软的各项产品、活动提出意见和建议
      <br/>
      • MVP将被邀请参加微软公司在各地的大型市场活动，比如免费参加当年的TechEd。
     </li>
     <li class="list2">
      <strong>
       微软助您不断攀登新的职业高峰
      </strong>
      <br/>
      • 在本人允许的前提下，微软将在官方站点公开宣传MVP的个人信息，并在各类专业印刷媒体上开设MVP专栏，使MVP和MVP所在的企业更有知名度。
      <br/>
      • MVP能得到微软总部授予的全球统一的证书及精美纪念品。
      <br/>
      • MVP各地的项目主管会随时与MVP保持联系，协助MVP更好的获取资源和提供反馈。
     </li>
     <li class="list2">
      <strong>
       参与丰富多彩MVP的聚会
      </strong>
      <br/>
      年度 MVP 全球峰会是微软以最激动人心的方式，答谢最有价值专家做出的特殊贡献而举办的全球盛会。今年的会议将于 9 月 28 日至 10 月 1日在美国华盛顿州 Redmond 召开，届时，首席执行官 Steve Ballmer、销售、营销和服务组副总裁 Kevin Johnson 及Microsoft 平台组副总裁 Jim Allchin 将发表主题演讲。
     </li>
    </ul>
