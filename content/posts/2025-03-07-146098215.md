    <h3>
     1. Unity3D 渲染管线概述
    </h3>
    <p>
     Unity3D 的渲染管线负责将 3D 场景转换为 2D 图像，最终显示在屏幕上。Unity 支持多种渲染管线，包括：
    </p>
    <ul>
     <li>
      <strong>
       <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E5%86%85%E7%BD%AE%E6%B8%B2%E6%9F%93%E7%AE%A1%E7%BA%BF&amp;zhida_source=entity" rel="nofollow" title="内置渲染管线">
        内置渲染管线
       </a>
       （Built-in Render Pipeline）
      </strong>
      ：Unity 默认的渲染管线，适用于大多数项目。
     </li>
     <li>
      <strong>
       <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E9%80%9A%E7%94%A8%E6%B8%B2%E6%9F%93%E7%AE%A1%E7%BA%BF&amp;zhida_source=entity" rel="nofollow" title="通用渲染管线">
        通用渲染管线
       </a>
       （Universal Render Pipeline, URP）
      </strong>
      ：适用于移动端、VR 和 2D 项目的轻量级渲染管线。
     </li>
     <li>
      <strong>
       <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E9%AB%98%E6%B8%85%E6%B8%B2%E6%9F%93%E7%AE%A1%E7%BA%BF&amp;zhida_source=entity" rel="nofollow" title="高清渲染管线">
        高清渲染管线
       </a>
       （High Definition Render Pipeline, HDRP）
      </strong>
      ：适用于高端 PC 和主机的高质量渲染管线。
     </li>
    </ul>
    <h4>
     1.1 渲染管线的主要阶段
    </h4>
    <ol>
     <li>
      <strong>
       几何处理（Geometry Processing）
      </strong>
      ：
     </li>
    </ol>
    <ul>
     <li>
      <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E9%A1%B6%E7%82%B9%E7%9D%80%E8%89%B2%E5%99%A8&amp;zhida_source=entity" rel="nofollow" title="顶点着色器">
       顶点着色器
      </a>
      （Vertex Shader）：处理顶点数据，如位置、法线、纹理坐标等。
     </li>
     <li>
      曲面细分（Tessellation）：可选阶段，用于增加几何细节。
     </li>
     <li>
      <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E5%87%A0%E4%BD%95%E7%9D%80%E8%89%B2%E5%99%A8&amp;zhida_source=entity" rel="nofollow" title="几何着色器">
       几何着色器
      </a>
      （Geometry Shader）：可选阶段，用于生成新的几何图元。
     </li>
    </ul>
    <p>
    </p>
    <ol>
     <li>
      <strong>
       光栅化（Rasterization）
      </strong>
      ：
     </li>
    </ol>
    <ul>
     <li>
      将几何图元转换为像素片段（Fragments）。
     </li>
     <li>
      深度测试（Depth Test）和模板测试（Stencil Test）在此阶段进行。
     </li>
    </ul>
    <p>
    </p>
    <ol>
     <li>
      <strong>
       像素处理（Pixel Processing）
      </strong>
      ：
     </li>
    </ol>
    <ul>
     <li>
      <a href="https://zhida.zhihu.com/search?content_id=254631946&amp;content_type=Article&amp;match_order=1&amp;q=%E7%89%87%E6%AE%B5%E7%9D%80%E8%89%B2%E5%99%A8&amp;zhida_source=entity" rel="nofollow" title="片段着色器">
       片段着色器
      </a>
      （Fragment Shader）：计算每个像素的颜色。
     </li>
     <li>
      透明度处理（Alpha Blending）：处理透明物体的渲染。
     </li>
    </ul>
    <p>
    </p>
    <ol>
     <li>
      <strong>
       后处理（Post-Processing）
      </strong>
      ：
     </li>
    </ol>
    <ul>
     <li>
      对渲染结果进行后期处理，如模糊、色彩校正等。
     </li>
    </ul>

    <h3>
     2. Unity3D 中的渲染技术
    </h3>
    <h4>
     2.1 材质与着色器
    </h4>
    <p>
     材质（Material）是 Unity 中用于定义物体表面外观的资源，它依赖于着色器（Shader）来定义如何渲染物体。Unity 支持多种着色器语言，包括：
    </p>
    <ul>
     <li>
      <strong>
       ShaderLab
      </strong>
      ：Unity 自带的着色器语言，用于编写表面着色器（Surface Shader）。
     </li>
     <li>
      <strong>
       HLSL
      </strong>
      ：用于编写顶点和片段着色器。
     </li>
    </ul>
    <h4>
     2.1.1 编写自定义着色器
    </h4>
    <p>
     以下是一个简单的 HLSL 着色器示例，用于实现基本的漫反射光照：
    </p>
    <pre><code>Shader "Custom/DiffuseShader"
{
    Properties
    {
        _MainTex ("Texture", 2D) = "white" {}
        _Color ("Color", Color) = (1,1,1,1)
    }
    SubShader
    {
        Tags { "RenderType"="Opaque" }
        LOD 200

        Pass
        {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag

            #include "UnityCG.cginc"

            struct appdata
            {
                float4 vertex : POSITION;
                float2 uv : TEXCOORD0;
                float3 normal : NORMAL;
            };

            struct v2f
            {
                float2 uv : TEXCOORD0;
                float4 pos : SV_POSITION;
                float3 worldNormal : NORMAL;
            };

            sampler2D _MainTex;
            float4 _Color;

            v2f vert (appdata v)
            {
                v2f o;
                o.pos = UnityObjectToClipPos(v.vertex);
                o.uv = v.uv;
                o.worldNormal = UnityObjectToWorldNormal(v.normal);
                return o;
            }

            fixed4 frag (v2f i) : SV_Target
            {
                float3 lightDir = normalize(_WorldSpaceLightPos0.xyz);
                float3 normal = normalize(i.worldNormal);
                float diff = max(dot(normal, lightDir), 0.0);
                float4 texColor = tex2D(_MainTex, i.uv);
                return texColor * _Color * diff;
            }
            ENDCG
        }
    }
}</code></pre>

    <h4>
     2.2.1 实现阴影
    </h4>
    <p>
     Unity 使用阴影映射（Shadow Mapping）技术来实现阴影效果。以下代码展示了如何在自定义着色器中接收阴影：
    </p>
    <pre><code>v2f vert (appdata v)
{
    v2f o;
    o.pos = UnityObjectToClipPos(v.vertex);
    o.uv = v.uv;
    o.worldNormal = UnityObjectToWorldNormal(v.normal);
    o.worldPos = mul(unity_ObjectToWorld, v.vertex).xyz;
    TRANSFER_SHADOW(o); // 传递阴影数据
    return o;
}

fixed4 frag (v2f i) : SV_Target
{
    float3 lightDir = normalize(_WorldSpaceLightPos0.xyz);
    float3 normal = normalize(i.worldNormal);
    float diff = max(dot(normal, lightDir), 0.0);
    float4 texColor = tex2D(_MainTex, i.uv);
    float shadow = SHADOW_ATTENUATION(i); // 计算阴影衰减
    return texColor * _Color * diff * shadow;
}</code></pre>
    <h4>
     2.3 后处理效果
    </h4>
    <p>
     后处理效果可以通过 Unity 的
     <code>
      OnRenderImage
     </code>
     方法实现。以下代码展示了如何实现一个简单的灰度效果：
    </p>
    <pre><code>using UnityEngine;

[ExecuteInEditMode]
public class GrayscaleEffect : MonoBehaviour
{
    public Material grayscaleMaterial;

    void OnRenderImage(RenderTexture src, RenderTexture dest)
    {
        if (grayscaleMaterial != null)
        {
            Graphics.Blit(src, dest, grayscaleMaterial);
        }
        else
        {
            Graphics.Blit(src, dest);
        }
    }
}</code></pre>
    <p>
     对应的着色器代码如下：
    </p>
    <pre><code>Shader "Custom/Grayscale"
{
    Properties
    {
        _MainTex ("Texture", 2D) = "white" {}
    }
    SubShader
    {
        Tags { "RenderType"="Opaque" }
        LOD 200

        Pass
        {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag

            #include "UnityCG.cginc"

            struct appdata
            {
                float4 vertex : POSITION;
                float2 uv : TEXCOORD0;
            };

            struct v2f
            {
                float2 uv : TEXCOORD0;
                float4 pos : SV_POSITION;
            };

            sampler2D _MainTex;

            v2f vert (appdata v)
            {
                v2f o;
                o.pos = UnityObjectToClipPos(v.vertex);
                o.uv = v.uv;
                return o;
            }

            fixed4 frag (v2f i) : SV_Target
            {
                fixed4 col = tex2D(_MainTex, i.uv);
                float gray = dot(col.rgb, float3(0.299, 0.587, 0.114));
                return fixed4(gray, gray, gray, col.a);
            }
            ENDCG
        }
    }
}</code></pre>
