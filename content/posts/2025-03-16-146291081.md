    <hr/>
    <h4>
     <a id="1__19">
     </a>
     <strong>
      1. 评价类问题
     </strong>
     （比较）
    </h4>
    <p>
     评价类问题的核心是对事物进行定量或定性的评价、排名、分类等。常见问题包括评价方案优劣、排序或打分。
    </p>
    <h5>
     <a id="_23">
     </a>
     <strong>
      解题步骤：
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        明确评价目标
       </strong>
      </p>
      <ul>
       <li>
        明确评价的对象和
        <mark>
         评价的目标
        </mark>
        （比如评价城市的宜居程度、商品质量等）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        构建评价指标体系
       </strong>
      </p>
      <ul>
       <li>
        <p>
         确定
         <mark>
          评价的指标
         </mark>
         （主观和客观指标）。
        </p>
       </li>
       <li>
        <p>
         判断指标的独立性和覆盖性，必要时做主成分分析、因子分析等降维。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        指标量化与数据获取
       </strong>
      </p>
      <ul>
       <li>
        处理原始数据，必要时进行
        <mark>
         数据标准化
        </mark>
        （归一化、无量纲化）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <mark>
         指标权重
        </mark>
        的确定
       </strong>
      </p>
      <ul>
       <li>
        使用方法：
        <ul>
         <li>
          主观方法：专家评分法、层次分析法（AHP）。
         </li>
         <li>
          客观方法：熵值法、标准差法、极差法。
         </li>
         <li>
          混合方法：组合赋权法。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <mark>
         构建评价模型
        </mark>
       </strong>
      </p>
      <ul>
       <li>
        简单模型：加权平均法、多级评价法等。
       </li>
       <li>
        复杂模型：TOPSIS法、灰色关联分析、模糊综合评价、层次分析模型（AHP）等。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        分析与解释
       </strong>
      </p>
      <ul>
       <li>
        对模型的评价结果进行分析（如排名、分数）。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        得出结论并建议改进方案
       </strong>
      </p>
      <ul>
       <li>
        根据评价结果，得出相应的结论，并提出优化建议。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     个人总结：
    </p>
    <ol>
     <li>
      确定评价的目标
     </li>
     <li>
      选择对应的评价指标
     </li>
     <li>
      确定各指标的权重
     </li>
     <li>
      构建评价模型
     </li>
     <li>
      分析和解释结果
     </li>
     <li>
      得出结果并改进方案
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="2__72">
     </a>
     <strong>
      2. 预测类问题
     </strong>
     （函数）
    </h4>
    <p>
     预测类问题的核心是根据历史数据或规律预测未来的趋势、状态或结果。常见问题包括销量预测、人口预测、气象预测等。
    </p>
    <h5>
     <a id="_76">
     </a>
     <strong>
      解题步骤：
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       明确预测目标
      </strong>
      <ul>
       <li>
        确定需要
        <mark>
         预测的目标
        </mark>
        （时间点、趋势、范围等）。
       </li>
       <li>
        明确
        <mark>
         输入变量和输出变量
        </mark>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       收集并分析数据
      </strong>
      <ul>
       <li>
        获取历史数据（时间序列数据、统计数据等）。
       </li>
       <li>
        检查数据的完整性、异常值，进行数据预处理。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       特征分析
      </strong>
      <ul>
       <li>
        分析数据的
        <strong>
         相关性、周期性、趋势性
        </strong>
        。
       </li>
       <li>
        对特征变量进行筛选和构建。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       选择预测模型
      </strong>
      <ul>
       <li>
        确定建模方法：
        <ul>
         <li>
          统计方法：时间序列分析（ARIMA、指数平滑法等）。
         </li>
         <li>
          回归方法：线性回归、多元回归、Lasso等。
         </li>
         <li>
          机器学习方法：支持向量机（SVM）、随机森林、神经网络、LSTM等。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       模型训练与验证
      </strong>
      <ul>
       <li>
        训练模型：划分训练集和测试集，进行模型拟合。
       </li>
       <li>
        验证模型：通过交叉验证、残差分析、评价指标（如RMSE、MAE等）来评估模型性能。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       预测结果分析
      </strong>
      <ul>
       <li>
        输出预测结果，判断其合理性。
       </li>
       <li>
        必要时结合情景分析（如假设不同条件）进行预测。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       调整模型与优化
      </strong>
      <ul>
       <li>
        结合实际需求和结果，不断调整模型参数或方法，直到结果可靠。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       应用与可视化
      </strong>
      <ul>
       <li>
        将预测结果应用到实际问题中，配合直观图表展示（折线图、散点图等）。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     个人总结：
    </p>
    <ol>
     <li>
      确定预测的目标，以及输入输出变量
     </li>
     <li>
      数据的查找与预处理
     </li>
     <li>
      对数据进行观察，特征分析（周期性，趋势性，相关性）
     </li>
     <li>
      选择预测的模型
     </li>
     <li>
      根据模型，将数据分为训练集和验证集，对模型进行训练和评估
     </li>
     <li>
      对预测目标进行预测，得到结果
     </li>
     <li>
      根据结果调整模型参数，对最终最优的结果进行分析
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="3__117">
     </a>
     <strong>
      3. 优化类问题
     </strong>
     （极值）
    </h4>
    <p>
     优化类问题的核心是找到最优解或接近最优解的方案。常见问题包括资源分配优化、路径优化、排班优化等。
    </p>
    <h5>
     <a id="_121">
     </a>
     <strong>
      解题步骤：
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       明确优化目标
      </strong>
      <ul>
       <li>
        明确优化的
        <strong>
         目标函数
        </strong>
        （如最小成本、最大利润、最短路径等）。
       </li>
       <li>
        确定
        <strong>
         优化的变量和约束条件
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       构建数学模型
      </strong>
      <ul>
       <li>
        目标函数：将优化目标抽象为一个数学表达式。
       </li>
       <li>
        约束条件：列出所有约束条件（等式、不等式）。
       </li>
       <li>
        确保模型可解（线性/非线性、连续/离散等）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       选择求解方法
      </strong>
      <ul>
       <li>
        线性优化问题：单纯形法、内点法等。
       </li>
       <li>
        非线性优化问题：梯度下降法、牛顿法等。
       </li>
       <li>
        离散优化问题：动态规划、整数规划、遗传算法、模拟退火等。
       </li>
       <li>
        多目标优化问题：Pareto最优、目标加权法等。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       模型求解
      </strong>
      <ul>
       <li>
        根据问题的特性，选择合适的优化工具（如Python的Scipy、Gurobi、Matlab的优化工具箱等）。
       </li>
       <li>
        使用迭代算法或精确算法求解。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       分析解的合理性
      </strong>
      <ul>
       <li>
        检查最优解是否满足约束条件。
       </li>
       <li>
        判断结果的实际可行性。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       模型优化与调整
      </strong>
      <ul>
       <li>
        如果结果不合理，重新审视目标函数和约束条件。
       </li>
       <li>
        引入更复杂的求解方法或启发式算法。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       结果分析与建议
      </strong>
      <ul>
       <li>
        分析最优解对实际问题的意义。
       </li>
       <li>
        给出具体实施建议。
       </li>
      </ul>
     </li>
    </ol>
    <p>
     个人总结：
    </p>
    <ol>
     <li>
      确定目标函数和决策变量，以及约束条件
     </li>
     <li>
      构建出数学模型（优化模型）
     </li>
     <li>
      判断所建出的模型使用的求解方法
     </li>
     <li>
      对模型进行求解
     </li>
     <li>
      对结果进行分析（合理性）
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_160">
     </a>
     总结
    </h4>
    <p>
     这三类问题的解题思路可以总结为以下通用步骤：
     <br/>
     <strong>
      明确目标 → 数据处理 → 模型建立 → 求解与验证 → 结果分析与改进
     </strong>
     。
     <br/>
     三类问题的关键在于：
    </p>
    <ul>
     <li>
      <strong>
       评价类问题：注重指标构建和权重分配。
      </strong>
     </li>
     <li>
      <strong>
       预测类问题：注重数据的时间序列性和模型的泛化能力。
      </strong>
      <br/>
      性）
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_172">
     </a>
     总结
    </h4>
    <p>
     这三类问题的解题思路可以总结为以下通用步骤：
     <br/>
     <strong>
      明确目标 → 数据处理 → 模型建立 → 求解与验证 → 结果分析与改进
     </strong>
     。
     <br/>
     三类问题的关键在于：
    </p>
    <ul>
     <li>
      <strong>
       评价类问题：注重指标构建和权重分配。
      </strong>
     </li>
     <li>
      <strong>
       预测类问题：注重数据的时间序列性和模型的泛化能力。
      </strong>
     </li>
     <li>
      <strong>
       优化类问题：注重目标函数和约束条件的合理性。
      </strong>
     </li>
    </ul>
