    <h2>
     一、AD模数转换
    </h2>
    <p>
     先导入之前写好的Delay和LCD1602模块,然后新建XPT2046点c和点h文件
    </p>
    <p>
     接下来就是对照这个时序把数据读出来
    </p>
    <p>
     <img alt="" height="319" src="https://i-blog.csdnimg.cn/direct/06b619dfa47047cca755f165e6b27eea.png" width="824"/>
    </p>
    <h3>
     1.先定义引脚
    </h3>
    <p>
     <img alt="" height="460" src="https://i-blog.csdnimg.cn/direct/d957c6a023b94d13ae6c16f2fa1aadd5.png" width="725"/>
    </p>
    <pre><code>#include &lt;REGX52.H&gt;

sbit XPT2046_CS=P3^5;
sbit XPT2046_DCLK=P3^6;
sbit XPT2046_DIN=P3^4;
sbit XPT2046_DOUT=P3^7;</code></pre>
    <h3>
     2.定义函数
    </h3>
    <p>
     给个参数（控制字），用于选择通道
    </p>
    <p>
     并且定义一个变量ADAvalue，最后返回出来
    </p>
    <p>
     <img alt="" height="347" src="https://i-blog.csdnimg.cn/direct/1cbab0b47ee54d67804d12b635e1a8b4.png" width="886"/>
    </p>
    <p>
     <img alt="" height="260" src="https://i-blog.csdnimg.cn/direct/9135c2d770b34c39acfb9989814fc49a.png" width="265"/>
    </p>
    <p>
     然后是根据上图对每个引脚先后赋值
    </p>
    <p>
     首先把DCLK置0（初始化），CS置0，DIN赋值参数控制字的最高位
     <br/>
     然后再给DCLK置1（上升沿），DIN就把最高位发出去，然后置0（下降沿），查看手册发现上下时间是ns级，所以不用延时
    </p>
    <p>
     接下来就是同理依次把8个位发出去，用for循环实现
    </p>
    <p>
     发出去之后就可以读数据了
    </p>
    <p>
     <img alt="" height="238" src="https://i-blog.csdnimg.cn/direct/e980546c99444418ab62e839da24b6b3.png" width="446"/>
    </p>
    <p>
     接下来就是到图中这个位置，先给DCLK一个上升沿，一个下降沿，数据才会过来
    </p>
    <p>
     进行一个判断，如果Dout为1，就让ADAValue把数据读出来（因为是16位所以或上0x8000）
    </p>
    <p>
     然后对该过程循环16位，用for循环实现，因为只有12位是有数值的，后面都是零填充的，所以我们要在最后返回ADAValue的时候向右移位（如果选择8位读就移8位，12位读就移4位）
    </p>
    <p>
     最后再把CS置高，结束这个时序，到这里就写完这个函数了
    </p>
    <pre><code>//XPT2046.c

#include &lt;REGX52.H&gt;

sbit XPT2046_CS=P3^5;
sbit XPT2046_DCLK=P3^6;
sbit XPT2046_DIN=P3^4;
sbit XPT2046_DOUT=P3^7;

unsigned int XPT2046_ReadAD(unsigned char Command)
{
	unsigned char i;
	unsigned int ADValue=0;
	XPT2046_DCLK=0;
	XPT2046_CS=0;
	
	for(i=0;i&lt;8;i++)
	{
		XPT2046_DIN=Command&amp;(0x80&gt;&gt;i);
		XPT2046_DCLK=1;
		XPT2046_DCLK=0;
	}
	for(i=0;i&lt;16;i++)
	{
		XPT2046_DCLK=0;
		XPT2046_DCLK=1;
		if(XPT2046_DOUT){ADValue|=(0x8000&gt;&gt;i);}
	}
	XPT2046_CS=1;
	if(Command&amp;0x08)
	{
		return ADValue&gt;&gt;8;
	}
	else
	{
		return ADValue&gt;&gt;4;
	}
}
}</code></pre>
    <p>
    </p>
    <p>
     这里再讲一下
     <strong>
      命令字怎么写
     </strong>
    </p>
    <p>
     <img alt="" height="40" src="https://i-blog.csdnimg.cn/direct/3ed7c13984184cbdbff57802f31ca76f.png" width="252"/>
    </p>
    <p>
     S就是起始位，给1；A2A1A0是地址，选择通道；MODE就是模式选择位，=0是12位模式，=1是8位模式；SER/DFR也是选择模式，=1是单端模式，=0是差分模式，我们在这用单端模式；PD0和PD1表示掉电和内部参考电压配置的关系，若为11，处于供电状态，若为00，处于低功率模式（当PD1为0时关闭内部参考电压源，为1时打开，内部参考源作用即假设原本测量值是从0-5V，接到这就是2.5-5V）
    </p>
    <p>
     在这里我们给1 A2 A1 A0 1 1 0 0，A2A1A0的选择要看下图
    </p>
    <p>
     下图只看前面部分，A2A1A0控制测哪的电压
    </p>
    <p>
     <img alt="" height="387" src="https://i-blog.csdnimg.cn/direct/1b4a4dd5e9a34a769987ddbe68788542.png" width="953"/>
    </p>
    <p>
     为了方便写代码，我们把上面这些值给宏定义，这里要根据开发板的原理图对应上图，以及前面教的命令字写出来，因为是要外部调用，所以写在.h文件里
    </p>
    <p>
     <img alt="" height="485" src="https://i-blog.csdnimg.cn/direct/75ad2f959dc44de28270e0ccbea09f82.png" width="501"/>
    </p>
    <p>
     我们定义不同的命令字，一个是8位，一个是12位，方便后面选择
    </p>
    <p>
     8位是0-255,12位是0-4096，更精细
    </p>
    <pre><code>#ifndef __XPT0246_H_
#define __XPT0246_H_

#define XPT2046_XP_8       0X9C
#define XPT2046_YP_8        0XDC
#define XPT2046_VBAT_8        0XAC
#define XPT2046_AUX_8        0XEC

#define XPT2046_XP_12       0X94
#define XPT2046_YP_12       0XD4
#define XPT2046_VBAT_12       0XA4
#define XPT2046_AUX_12       0XE4


unsigned int XPT2046_ReadAD(unsigned char Command);


#endif</code></pre>
    <p>
     因为AUX接到了一个接口上，所以这里我们不读，只读其他三个
    </p>
    <pre><code>//main.c

#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " LCD1602.h"
#include " XPT2046.h"

unsigned int ADValue;

void main()
{
	LCD_Init();
	LCD_ShowString(1,1,"ADJ NTC RG");
	while(1)
	{
 		ADValue=XPT2046_ReadAD(XPT2046_XP_8);
		LCD_ShowNum(2,1,ADValue,3);
 		ADValue=XPT2046_ReadAD(XPT2046_YP_8);
		LCD_ShowNum(2,5,ADValue,3);
 		ADValue=XPT2046_ReadAD(XPT2046_VBAT_8);
		LCD_ShowNum(2,9,ADValue,3);
		Delay(10);
	}
}</code></pre>
    <p>
     这里我需要把DS18B20拔掉才能显示正常，暂时还未找到原因
    </p>
    <pre><code>#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " LCD1602.h"

unsigned int ADValue;

	}

    <p>
     这样就可以调节我们电路板上的电阻，然后会把阻值通过AD转换后输出在LCD上
    </p>
    <p>
     GR是光敏电阻，NTC是热敏电阻，AD1(ADJ)是可调电阻
    </p>

    <h2>
     二、 DA数模转换
    </h2>
    <p>
     接下是PWM，因为跟之前写过的直流电机调速原理是一样的，所以我们复制过来创建一个新的工程
    </p>
    <p>
     首先是更改接口
    </p>
    <p>
     sbit DA=P2^1;
    </p>
    <p>
     <img alt="" height="221" src="https://i-blog.csdnimg.cn/direct/6d47e7fd0fc84dff9d5f64016b46e0f1.png" width="291"/>
    </p>
    <p>
     然后是更改命名，可以点这里一键替代
    </p>
    <p>
     <img alt="" height="336" src="https://i-blog.csdnimg.cn/direct/8f9c599961f4446b9c0d84c14d2ba53a.png" width="371"/>
    </p>
    <p>
     然后把关于按键的部分删掉，在主函数里实现一个呼吸灯的效果
    </p>
    <pre><code>#include &lt;REGX52.H&gt;
#include " Delay.h"
#include " Key.h"
#include " Nixie.h"
#include " Timer0.h"

sbit DA=P2^1;

unsigned char Counter,Compare;
unsigned char i;

void main()
{

	Timer0_Init();
	while(1)
	{
		for(i=0;i&lt;100;i++)
		{
			Compare=i;
			Delay(10);
		}
		for(i=100;i&gt;0;i--)
		{
			Compare=i;
			Delay(10);
		}
  }
}
void Timer0_Routine() interrupt 1
{
	Key_Loop();
	TL0 = 0xA4;		//设置定时初值
	TH0 = 0xFF;		//设置定时初值
	Counter++;
	Counter%=100;
	if(Counter&lt;Compare)
	{
		DA=1;
	}
	else
	{
		DA=0;
	}
}</code></pre>
    <p>
     其他部分都不需要改，这样就实现了呼吸灯的效果。
    </p>
