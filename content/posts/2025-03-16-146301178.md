# Adobe Premiere Pro2023配置要求
#### Windows 系统
* **最低配置**
* **处理器** ：Intel® 第六代或更新版本的 CPU，或 AMD Ryzen™ 1000 系列或更新版本的 CPU，需要支持 Advanced Vector Extensions 2（AVX2）。
* **操作系统** ：Windows 10（64 位）v20H2 或更高版本。
* **内存** ：8GB 的 RAM。
* **GPU** ：2GB 的 GPU 内存。
* **存储** ：8GB 可用硬盘空间用于安装，安装期间所需的额外可用空间，不能安装在可移动闪存存储器上，还需用于媒体的额外高速驱动器。
* **显示器** ：1920x1080 分辨率，声卡需与 ASIO 兼容或 Microsoft Windows Driver Model。
* **推荐配置**
* **处理器** ：具有快速同步功能的 Intel® 第十一代或更新版本的 CPU，或 AMD Ryzen™ 3000 系列 / Threadripper 2000 系列或更新版本的 CPU。
* **操作系统** ：Windows 10（64 位）22H2 版或更高版本，或者 Windows 11。
* **内存** ：16GB 的 RAM 用于 HD 媒体，32GB 或以上适用于 4K 及更高分辨率。
* **GPU** ：4GB 的 GPU 内存适用于 HD 和某些 4K 媒体，6GB 或以上适用于 4K 和更高分辨率。
* **存储** ：用于应用程序安装和缓存的快速内部 SSD，以及用于媒体的额外高速驱动器。
* **显示器** ：1920x1080 或更高分辨率，Display HDR 1000 适用于 HDR 工作流程。
#### macOS 系统
* **最低配置**
* **操作系统** ：macOS 10.15 或更高版本。
* **处理器** ：多核 Intel 处理器。
* **内存** ：8GB RAM。
* **显卡** ：支持 OpenCL 1.2 以上的显卡。
* **硬盘空间** ：6GB 可用硬盘空间用于安装。
