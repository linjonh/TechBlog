    <p>
     用户体验可分为六大类：
     <br/>
     感官体验、交互体验、浏览体验、情感体验、价值体验、信任体验
    </p>
    <p>
     下面对这些分类进行简单的介绍：
    </p>
    <p>
     <strong>
      （1）感官体验
     </strong>
     <br/>
     ——视觉上面的体验
     <br/>
     ●设计风格/网站LoGo
     <br/>
     ●色彩/页面速度
     <br/>
     ●动画效果/图片展示/图标使用
     <br/>
     ●广告位/背景音乐
    </p>
    <p>
     <strong>
      （2）交互体验
     </strong>
     <br/>
     ——呈现给用户操作上的体验，强调易用/可用性。
     <br/>
     ●会员申请、会员注册、表单填写、表单提交
     <br/>
     ●按钮设置、点击提示、错误提示、在线问答
     <br/>
     ●意见反馈、在线调查、在线搜索、页面刷新
     <br/>
     ●资料安全、显示路径
    </p>
    <p>
     <strong>
      （3）浏览体验
     </strong>
     <br/>
     ——呈现给用户浏览上的体验，强调吸引性
     <br/>
     ●栏目的命令、栏目的层级
     <br/>
     ●内容的分类、内容的丰富性、内容的原创性
     <br/>
     ●信息的更新频率、精彩内容的推荐
     <br/>
     ●相关内容的推荐、收藏夹的设置、栏目的订阅、信息的搜索
     <br/>
     ●文字排列、字体、页面底色、页面的长度、分页浏览
    </p>
    <p>
     <strong>
      （4）情感体验
     </strong>
     <br/>
     ——呈现给用户心理上的体验，强调友好性以及与用户关系的维系
     <br/>
     ●友好提示、会员交流、售后与反馈
     <br/>
     ●会员优惠、会员推荐、鼓励用户参与、会员活动
     <br/>
     ●专家答疑、邮件、短信问候
    </p>
    <p>
     <strong>
      （5）价值体验
     </strong>
     <br/>
     ——以陌陌 为例
     <br/>
     ●基础价值：能勾搭妹纸（核心服务）
     <br/>
     ●超于基础价值的期望价值：不但能勾搭妹纸，还发现这里有好多好多妹纸 （高于核心服务的期望）
    </p>
    <p>
     <strong>
      （6）信任体验
     </strong>
     <br/>
     ——呈现给用户的信任体验，强调可靠性
     <br/>
     ●搜索引擎、公司介绍、投资者关系、服务保障
     <br/>
     ●文章来源、文章编辑、联系方式、服务热线
     <br/>
     ●有效的投诉途径、安全及隐私条款
     <br/>
     ●法律声明、网站备案、相关链接、帮助中心
    </p>
