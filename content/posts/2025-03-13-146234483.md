    <h2>
     算法解析
    </h2>
    <p>
     位运算是用于二进制的运算符号。而对于多次出现的数字，其二进制都是一模一样的，这里是3次重复的出现是数字。由此我们可以想到，如果我们
     <span style="background-color:#ffd900">
      由低到高去计算为一个bit位上的和
     </span>
     ，对和取余3。
     <span style="background-color:#ffd900">
      如果为0则代表这个bit位上都是重复出现的数字。如果位1则代表出现的我们要找的数字
     </span>
     。我们将这个bit的
     <span style="background-color:#ffd900">
      结果记录
     </span>
     ，再去计算和判断下一个bit位
    </p>
    <h2>
     代码实现
    </h2>
    <pre><code class="language-cpp">//计算每一个bit位的和
class Solution {
public:
    int singleNumber(vector&lt;int&gt;&amp; nums)
    {
        int ret = 0;//ret负责记录每一个bit位的变化情况

        for (int i = 0; i &lt; 32; i++)//一个整型一共有32个bit位
        {
            int sum = 0;
            for (auto&amp; e : nums)
            {

                if (((e &gt;&gt; i) &amp; 1) == 1)//从低到高计算每个元素在同一个bit位上的和
                    sum++;
            }

            sum %= 3;
            if (sum == 1)//当取余结果为1时，说明出现了我们要找的数字，我们将其记录
                ret |= (sum &lt;&lt; i);
        }
        return ret;
    }
};</code></pre>

    <h2>
     拓展

    <p>
     其实对于这种题：一个元素只出现一次，其余元素出现n次。方法是一样的，只需要
     <span style="background-color:#ffd900">
      将取余3改为取余n即可
