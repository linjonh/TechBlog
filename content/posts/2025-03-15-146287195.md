    <h2>
     <a id="1_RequestParam_0">
     </a>
     1. @RequestParam
    </h2>
    <ul>
     <li>
      @RequestParam：get请求时如果用不到它的3个属性，可以省略；其他请求如果是通过param传送数据，必须使用该注释
     </li>
    </ul>
    <blockquote>
     <ul>
      <li>
       要点：
       <ol>
        <li>
         可用于任何类型的请求（get请求数据在请求行中， post请求数据在请求体中）
        </li>
        <li>
         无论时在请求行还是请求体，格式都是：username=zhangsan&amp;password=1234&amp;email=zhangsan@powernode.com
        </li>
       </ol>
      </li>
     </ul>
    </blockquote>
    <h3>
     <a id="_7">
     </a>
     属性
    </h3>
    <ul>
     <li>
      value： 不解释看代码
     </li>
     <li>
      defaultValue ：设置默认值
     </li>
    </ul>

    <h2>
     <a id="2_RequestBody_35">
     </a>
     2. @RequestBody
    </h2>
    <ul>
     <li>
      @RequestBody： 最主要的用处，post请求时前端发送json格式字符串，后端用类接收
     </li>
    </ul>
    <blockquote>
     <ul>
      <li>
       用法
       <ol>
        <li>
         可用于非get请求的其他请求
        </li>
        <li>
         前端发送的是json字符串，后端用类接收
        </li>
        <li>
         @RequestBody不可以省略
        </li>
        <li>
         required属性：是否必需传
        </li>
       </ol>
      </li>
     </ul>
    </blockquote>
    <pre><code class="prism language-javascript">@<span class="token function">PostMapping</span><span class="token punctuation">(</span><span class="token string">"/user/{id}"</span><span class="token punctuation">)</span>

    <h2>
     <a id="3_PathVariable_77">
     </a>
     3. @PathVariable
    </h2>
    <ul>
     <li>
      @PathVariable：发送Rest风格请求时使用
     </li>
    </ul>
    <blockquote>
     <p>
      作用：
     </p>
     <ol>
      <li>
       Rest风格请求的占位符
      </li>
      <li>
       可用于任何请求中，get、post、put等等
      </li>
      <li>
       也有required 属性
      </li>
     </ol>
    </blockquote>
    <h3>
     <a id="1__84">
     </a>
     1. 基本使用
    </h3>
