    <h3>
     <a id="_50">
     </a>
     服务器配置
    </h3>
    <p>
     这个略复杂，上图你们照着一步步来就ok了。
     <br/>
     第一步登录服务器，并进入mysql
    </p>
    <pre><code class="prism language-css">mysql -u root -p
</code></pre>
    <p>
     下面是进入后的标志
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/75a35cd99310112bc217fb990117b9ad.png"/>
     <br/>
     进入不成功的话，自行百度
    </p>
    <p>
     输入语句
     <code>
      show databases;
     </code>
     ，查看现有的数据库，那我就用我创建好的
     <strong>
      test01数据库
     </strong>
     做演示
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c91112b7d76f489f95c29908ee749ebe.png"/>
     <br/>
     然后，在控制台输入下面语句
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">grant</span> <span class="token keyword">all</span> <span class="token keyword">on</span> test01<span class="token punctuation">.</span><span class="token operator">*</span> <span class="token keyword">to</span> dbadmin @'<span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token string">' identified by '</span><span class="token number">123456</span>'<span class="token punctuation">;</span>
</code></pre>
    <ul>
     <li>
      <p>
       <strong>
        注意：
       </strong>
       这句话意思是： 允许所有IP地址的终端用账号dbadmin密码123456操作test01数据库
      </p>
      <p>
       理解上句话后，大家可根据需求，自行更改命令。
      </p>
     </li>
    </ul>
    <p>
     然后更新权限
     <code>
      flush privileges;
     </code>
     如图
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/60be5eb51ab307a03ee8b4f91fa919d3.png"/>
     <br/>
     这样就好了。
     <br/>
     请回到目录，点pycharm配置，将pycharm连接到mysql
    </p>
    <h3>
     <a id="_78">
     </a>
     问题汇总
    </h3>
    <p>
     1.服务器配置里，语句中英文单引号错误
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">grant</span> <span class="token keyword">all</span> <span class="token keyword">on</span> test01<span class="token punctuation">.</span><span class="token operator">*</span> <span class="token keyword">to</span> dbadmin @'<span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token punctuation">.</span><span class="token operator">%</span><span class="token string">' identified by '</span><span class="token number">123456</span>'<span class="token punctuation">;</span>
</code></pre>
    <p>
     2.服务器配置里，数据库名称输入错误
     <br/>
     3.服务器未开启3306端口
    </p>

     最后，以上若存在描述错误的地方，请提出批评并指正！
