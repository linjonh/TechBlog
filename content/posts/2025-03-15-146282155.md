    <h2>
     <a id="_Qt__1">
     </a>
     深入理解 Qt 系统托盘图标：创建自定义的系统托盘图标类
    </h2>
    <p>
     在现代桌面应用程序中，
     <strong>
      系统托盘图标
     </strong>
     （通常位于任务栏右下角）是一个常见的交互方式。它为用户提供了快捷的操作界面，如显示应用程序、退出程序、查看通知等。在 Qt 中，
     <code>
      QSystemTrayIcon
     </code>
     类允许我们轻松地在系统托盘中添加图标并与用户交互。今天，我们将介绍如何创建一个自定义的系统托盘图标类，使其能够响应用户的点击事件，并显示一个右键菜单。
    </p>
    <h3>
     <a id="1__QSystemTrayIcon_5">
     </a>
     1. 什么是
     <code>
      QSystemTrayIcon
     </code>
     ？
    </h3>
    <p>
     在 Qt 中，
     <code>
      QSystemTrayIcon
     </code>
     是一个用于在系统托盘（操作系统的任务栏右下角）显示图标的类。通过它，应用程序可以在系统托盘中创建图标，用户可以通过点击这个图标来执行一些操作。
    </p>
    <p>
     <strong>
      <code>
       QSystemTrayIcon
      </code>
      的主要功能
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       显示图标
      </strong>
      ：在任务栏右下角显示应用程序的图标。
     </li>
     <li>
      <strong>
       处理用户交互
      </strong>
      ：通过用户点击图标来执行不同的操作，如显示主窗口、退出应用等。
     </li>
     <li>
      <strong>
       右键菜单
      </strong>
      ：右键点击图标时弹出一个上下文菜单，提供更多的操作选项。
     </li>
    </ul>
    <h4>
     <a id="2_SysTraylcon_14">
     </a>
     2. 自定义系统托盘图标类：
     <code>
      SysTraylcon
     </code>
    </h4>
    <p>
     我们创建了一个名为
     <code>
      SysTraylcon
     </code>
     的类，继承自
     <code>
      QSystemTrayIcon
     </code>
     ，并实现了一些附加功能，如响应用户的点击事件、弹出右键菜单等。
    </p>

    <ul>
     <li>
      <p>
       <strong>
        <code>
         QSystemTrayIcon::Trigger
        </code>

       <code>
        reason

    <h4>
     <a id="4__130">
     </a>
     4. 总结
    </h4>
    <p>
     在这篇博客中，我们创建了一个名为
     <code>
      SysTraylcon
     </code>
     的自定义系统托盘图标类，它继承自
     <code>
      QSystemTrayIcon
     </code>
     ，并添加了以下功能：
    </p>
    <ul>
     <li>
      <strong>
       设置托盘图标
      </strong>
      ：显示一个图标，并设置鼠标悬停时的提示信息。
     </li>
     <li>
      <strong>
       响应左键点击
      </strong>
      ：当用户左键点击托盘图标时，显示并激活主窗口。
     </li>
     <li>
      <strong>
       响应右键点击
      </strong>
      ：当用户右键点击托盘图标时，弹出自定义的右键菜单，提供 “显示” 和 “退出” 功能。
     </li>
     <li>
      <strong>
       动态创建菜单项
      </strong>
      ：使用
      <code>
       CustomMenu
      </code>
      类为菜单添加菜单项，并通过信号与槽机制连接菜单项的动作。
     </li>
    </ul>
    <p>
     通过这些功能，我们能够创建一个具有系统托盘图标的应用程序，并使用户能够通过托盘图标快速访问应用程序的功能。
    </p>
