      <strong>
       总结
       <strong>
       </strong>
      </strong>

     <p style="text-align:left;">
      <strong>
       阿里云 IoT
      </strong>
      企业实例包年包月模式下更适合设备24小时在线场景，性价比最高，但设备在线时长不足会造成一部分成本浪费；
      <strong>
       华为云
      </strong>
      对消息量限制较大，企业架构师需要关注对业务影响；
      <strong>
       AWS IoT Core
      </strong>
      为企业提供了按需付费模式，整体上成本最优；
      <strong>
       微软Azure
      </strong>
      <strong>
       IoT Hub
      </strong>
      按规格计费，单设备年成本刚刚突破1元，可以接受；
      <strong>
       腾讯云新
      </strong>
      <strong>
       企业实例
      </strong>
      计费调整后，单设备成本飙升到
      <strong>
       4.56元
      </strong>
      ，不推荐中小企业选择；
      <strong>
       天翼 CTWing
      </strong>
      开始对非电信卡设备收费0.6元，虽然不高，但中途单方面提价难免让企业感觉被刺，商誉收到折损；
      <strong>
       中移 OneNET
      </strong>
      适合成本敏感对数据可靠性要求低的初创企业，但也要提防后续单方面强制收费。
