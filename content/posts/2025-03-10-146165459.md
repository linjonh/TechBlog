    <h4 id="h2">
     <a id="2__13">

     2. 网络扫描工具示例
    </h4>
    <p>
     下面是一个简单的网络扫描脚本，利用
     <code>
      socket
     </code>
     库扫描指定IP范围内的开放端口。
    </p>
    <pre><code id="code_id_0">import socket

def scan_ports(ip, start_port, end_port):
    print(f"Scanning ports on {ip}...")
    for port in range(start_port, end_port + 1):
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(1)
        result = sock.connect_ex((ip, port))
        if result == 0:
            print(f"Port {port} is open")
        sock.close()

if __name__ == "__main__":
    target_ip = "192.168.1.1"
    scan_ports(target_ip, 1, 1024)
</code></pre>
    <ul>
    </ul>
    <h5 id="h3">
     <a id="_35">
     </a>
     代码解释
    </h5>
    <p>
     上面的代码定义了一个
     <code>
      scan_ports
     </code>
     函数，该函数接受一个IP地址和端口范围作为参数，并在指定的端口范围内检查哪些端口是开放的。通过
     <code>
      socket
     </code>
     库建立连接并捕获连接结果，从而判断端口的开放状态。

    <h4 id="h4">
     <a id="3__39">
     </a>
     3. 日志分析示例
    </h4>
    <p>
     日志分析是识别网络攻击和异常活动的重要一环。下面是一个简单的日志分析脚本，能够识别可疑的登录尝试。
    </p>
    <pre><code id="code_id_1">import re

def analyze_logs(log_file):
    with open(log_file, 'r') as file:
        logs = file.readlines()
    
    failed_attempts = {}
    
    for log in logs:
        if "Failed password" in log:
            ip = re.search(r'(\d+\.\d+\.\d+\.\d+)', log)
            if ip:
                ip_address = ip.group(0)
                if ip_address in failed_attempts:
                    failed_attempts[ip_address] += 1
                else:
                    failed_attempts[ip_address] = 1
    
    print("Suspicious login attempts:")
    for ip, count in failed_attempts.items():
        print(f"{ip}: {count} failed attempts")

if __name__ == "__main__":
    log_file = "auth.log"
    analyze_logs(log_file)
</code></pre>
    <ul>
    </ul>
    <h5 id="h5">
     <a id="_71">
     </a>
     代码解释
    </h5>
    <p>
     这个脚本会读取一个日志文件（如
     <code>
      auth.log
     </code>
     ），并查找"Failed password"来确定失败的登录尝试。通过正则表达式提取IP地址，并统计每个IP的失败次数，从而帮助管理员识别潜在的攻击来源。
    </p>
