**二、MySQL图形界面安装**
  
**（一）安装MySQL**
  
1.打开安装文件，进入安装界面，我选择默认安装类型
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/0b996b4f9c3d66dda03f46d47eb5726c.png#pic_center)
  
这里列出了五种安装类型：

1. Developer Default：默认安装类型；
2. Server only：仅作为服务；
3. Client only：仅作为客户端；
4. Full：完全安装；
5. Custom：自定义安装类型。

2.点击execute安装组件
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/ff8f78bda2f6393294e3c52a81da4441.png#pic_center)
  
3.全部安装完成
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/a3792f56c47b5f0fa572ae75d60d617b.png#pic_center)
  
**（二）配置MySQL**
  
下面进行对服务器的配置
  
1.在上一步中点击next进入服务器配置窗口，进行信息确认，确认后点击next，如图：
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/fcdcead3649da5173db391833ecdd9f7.png#pic_center)
  
2.进入网络类型配置窗口，默认设置，点击next
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/c02392eac865ace196a90039d80c9aa2.png#pic_center)
  
Config Type中 3 个选项的具体含义如下：

* Development Machine（开发机器）：安装的 MySQL 服务器作为开发机器的一部分，在三种可选的类型中，占用的内存最少；
* Server Machine（服务器）：安装的 MySQL 服务器作为服务器机器的一部分，占用的内存在三种类型中居中；
* Dedicated MySQL Server Machine（专用服务器）：安装专用 MySQL 数据库服务器，占用机器全部有效的内存。

> 初学者建议选择“Development Machine”选项，这样占用系统的资源比较少。

3.继续点击next
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/bba04482ea442de665dcf7cbcf4a64a7.png#pic_center)
  
4.进入服务器密码设置窗口
  
系统默认用户名为root，点击Add User即可添加新用户
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/522474a60c4c1274bf349528599813b2.png#pic_center)
  
5.进入服务器名称窗口设置服务器名称，这里无特殊需要也不建议修改。继续单击 Next 按钮，如图所示：
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/26e99ef458bcbc7112888aa6a8c6482c.png#pic_center)
  
6.打开确认设置服务器窗口，点击 Execute 按钮完成 MySQL 的各项配置
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/3da937e5cafa0dcb7cbcc499f03b7dc2.png#pic_center)
  
配置完成：
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/b77ec4778ea45a51d80629ce88d1d53b.png#pic_center)
  
7.最后打开 Windows 任务管理器对话框，可以看到 MySQL 服务进程 mysqld.exe 已经启动了，如图：
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/8bebeb319973c24923e51949c5321d8d.png#pic_center)
  
8.接下来跟着提示进行操作就可以了，最后来到如下界面：
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/ce348a52839ef45c5443eec360d7d86e.png#pic_center)
  
**（三）测试MySQL是否安装成功**
  
打开MySQL 8.0 Command Line Client输入密码
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/f8428882f0328ed0ece1406ae8b00364.png#pic_center)

至此，MySQL安装配置就完成了。
