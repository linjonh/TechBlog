    <p>
     Windows下有两种安装MySQL的方式：
    </p>
    <ul>
     <li>
      图形界面安装 （.msi文件）

     <li>

     </li>
    </ul>
    <p>
     MySQL下载官网：
     <a href="http://www.mysql.com/" rel="nofollow">
      http://www.mysql.com
     </a>
    </p>
    <p>
     也可前往百度网盘提取（两种安装方式文件都有）：
     <br/>
     链接：
     <a href="https://pan.baidu.com/s/1NMRUu_E098h4ErzSXTUKgA" rel="nofollow">
      https://pan.baidu.com/s/1NMRUu_E098h4ErzSXTUKgA
     </a>
     <br/>
     提取码：3tfb
    </p>
    <p>
     <strong>
      一、MySQL免安装版配置教程
     </strong>
     <br/>
     <a href="http://c.biancheng.net/view/2412.html" rel="nofollow">
      http://c.biancheng.net/view/2412.html
     </a>
     <br/>
     <strong>
      二、MySQL图形界面安装
     </strong>
     <br/>
     <strong>
      （一）安装MySQL
     </strong>
     <br/>
     1.打开安装文件，进入安装界面，我选择默认安装类型
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0b996b4f9c3d66dda03f46d47eb5726c.png#pic_center">
      <br/>
      这里列出了五种安装类型：
     </img>
    </p>
    <ol>
     <li>
      Developer Default：默认安装类型；
     </li>
     <li>
      Server only：仅作为服务；
     </li>
     <li>
      Client only：仅作为客户端；
     </li>
     <li>
      Full：完全安装；
     </li>
     <li>
      Custom：自定义安装类型。
     </li>
    </ol>
    <p>
     2.点击execute安装组件
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ff8f78bda2f6393294e3c52a81da4441.png#pic_center">
      <br/>
      3.全部安装完成
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a3792f56c47b5f0fa572ae75d60d617b.png#pic_center">
       <br/>
       <strong>
        （二）配置MySQL
       </strong>
       <br/>
       下面进行对服务器的配置
       <br/>
       1.在上一步中点击next进入服务器配置窗口，进行信息确认，确认后点击next，如图：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fcdcead3649da5173db391833ecdd9f7.png#pic_center">
        <br/>
        2.进入网络类型配置窗口，默认设置，点击next
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c02392eac865ace196a90039d80c9aa2.png#pic_center">
         <br/>
         Config Type中 3 个选项的具体含义如下：
        </img>
       </img>
      </img>
     </img>
    </p>
    <ul>
     <li>
      Development Machine（开发机器）：安装的 MySQL 服务器作为开发机器的一部分，在三种可选的类型中，占用的内存最少；
     </li>
     <li>
      Server Machine（服务器）：安装的 MySQL 服务器作为服务器机器的一部分，占用的内存在三种类型中居中；
     </li>
     <li>
      Dedicated MySQL Server Machine（专用服务器）：安装专用 MySQL 数据库服务器，占用机器全部有效的内存。
     </li>
    </ul>
    <blockquote>
     <p>
      初学者建议选择“Development Machine”选项，这样占用系统的资源比较少。
     </p>
    </blockquote>
    <p>
     3.继续点击next
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/bba04482ea442de665dcf7cbcf4a64a7.png#pic_center">
      <br/>
      4.进入服务器密码设置窗口
      <br/>
      系统默认用户名为root，点击Add User即可添加新用户
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/522474a60c4c1274bf349528599813b2.png#pic_center">
       <br/>
       5.进入服务器名称窗口设置服务器名称，这里无特殊需要也不建议修改。继续单击 Next 按钮，如图所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/26e99ef458bcbc7112888aa6a8c6482c.png#pic_center">
        <br/>
        6.打开确认设置服务器窗口，点击 Execute 按钮完成 MySQL 的各项配置
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3da937e5cafa0dcb7cbcc499f03b7dc2.png#pic_center">
         <br/>
         配置完成：
         <br/>
         <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b77ec4778ea45a51d80629ce88d1d53b.png#pic_center">
          <br/>
          7.最后打开 Windows 任务管理器对话框，可以看到 MySQL 服务进程 mysqld.exe 已经启动了，如图：
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8bebeb319973c24923e51949c5321d8d.png#pic_center">
           <br/>
           8.接下来跟着提示进行操作就可以了，最后来到如下界面：
           <br/>
           <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ce348a52839ef45c5443eec360d7d86e.png#pic_center">
            <br/>
            <strong>
             （三）测试MySQL是否安装成功
            </strong>
            <br/>
            打开MySQL 8.0 Command Line Client输入密码
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f8428882f0328ed0ece1406ae8b00364.png#pic_center"/>
            <br/>
            如果显示数据库列表，则说明 MySQL 数据库安装成功。
           </img>
          </img>
         </img>
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     至此，MySQL安装配置就完成了。

    <p>
     6月9日，麦可思研究2023年版就业蓝皮书（包括《2023年中国本科生就业报告》《2023年中国高职生就业报告》）正式发布。
    </p>
    <p>
     <strong>
      2022届大学毕业生月收入较高的前10个专业
     </strong>
    </p>
    <p>
     本科计算机类、高职自动化类专业月收入较高。2022届本科计算机类、高职自动化类专业月收入分别为6863元、5339元。其中，本科计算机类专业起薪与2021届基本持平，高职自动化类月收入增长明显，2022届反超铁道运输类专业（5295元）排在第一位。
    </p>
    <p>
     具体看专业，2022届本科月收入较高的专业是信息安全（7579元）。对比2018届，电子科学与技术、自动化等与人工智能相关的本科专业表现不俗，较五年前起薪涨幅均达到了19%。数据科学与大数据技术虽是近年新增专业但表现亮眼，已跻身2022届本科毕业生毕业半年后月收入较高专业前三。五年前唯一进入本科高薪榜前10的人文社科类专业——法语已退出前10之列。

     <strong>
      网络安全行业特点
     </strong>
    </h4>
    <p>
     1、就业薪资非常高，涨薪快 2021年猎聘网发布网络安全行业就业薪资行业最高人均33.77万！
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/3eee8b056e63572d53f1ce9aa0531723.png"/>
    </p>
    <p>
     2、人才缺口大，就业机会多
    </p>
    <p>
     2019年9月18日《中华人民共和国中央人民政府》官方网站发表：我国网络空间安全人才 需求140万人，而全国各大学校每年培养的人员不到1.5W人。猎聘网《2021年上半年网络安全报告》预测2027年网安人才需求300W，现在从事网络安全行业的从业人员只有10W人。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/026650dbd8291f5e4e7452f5820f7709.png"/>
    </p>
    <p>
     <strong>
      行业发展空间大，岗位非常多
     </strong>
    </p>
    <p>
     网络安全行业产业以来，随即新增加了几十个网络安全行业岗位︰网络安全专家、网络安全分析师、安全咨询师、网络安全工程师、安全架构师、安全运维工程师、渗透工程师、信息安全管理员、数据安全工程师、网络安全运营工程师、网络安全应急响应工程师、数据鉴定师、网络安全产品经理、网络安全服务工程师、网络安全培训师、网络安全审计员、威胁情报分析工程师、灾难恢复专业人员、实战攻防专业人员…
    </p>
    <p>
     <strong>
      职业增值潜力大
     </strong>
    </p>
    <p>
     网络安全专业具有很强的技术特性，尤其是掌握工作中的核心网络架构、安全技术，在职业发展上具有不可替代的竞争优势。
    </p>
    <p>
     随着个人能力的不断提升，所从事工作的职业价值也会随着自身经验的丰富以及项目运作的成熟，升值空间一路看涨，这也是为什么受大家欢迎的主要原因。
    </p>
    <p>
     从某种程度来讲，在网络安全领域，跟医生职业一样，越老越吃香，因为技术愈加成熟，自然工作会受到重视，升职加薪则是水到渠成之事。
