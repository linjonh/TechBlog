    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      上个月，欧盟指责微软把浏览器
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IE
      </span>
     </span>
     <span style="font-size: medium;">
      与
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows
      </span>
     </span>
     <span style="font-size: medium;">
      捆绑是滥用其市场垄断地位。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       2
      </span>
     </span>
     <span style="font-size: medium;">
      月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       9
      </span>
     </span>
     <span style="font-size: medium;">
      日，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IDG
      </span>
     </span>
     <span style="font-size: medium;">
      新闻透露欧盟新近允许
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      参与这项反垄断诉讼，谷歌（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Chrome
      </span>
     </span>
     <span style="font-size: medium;">
      浏览器）和苹果（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Safar
      </span>
     </span>
     <span style="font-size: medium;">
      浏览器）也跟在后面起哄，微软真够倒霉。怎么办呢？
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows
      </span>
     </span>
     <span style="font-size: medium;">
      捆绑浏览器
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IE
      </span>
     </span>
     <span style="font-size: medium;">
      ，十年前，美国政府也想以此整治微软，最后，不了了之。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       2004
      </span>
     </span>
     <span style="font-size: medium;">
      年，欧盟以
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows
      </span>
     </span>
     <span style="font-size: medium;">
      捆绑媒体播放器又狠狠整了微软一下，这次以微软服输、受罚了事。当今，正值
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows 7
      </span>
     </span>
     <span style="font-size: medium;">
      上市之际，（捆绑问题）争执又起，令微软十分恼火。如果
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows 7
      </span>
     </span>
     <span style="font-size: medium;">
      不能捆绑浏览器，那还有什么意思？这一回，微软一定要“挣扎”一番（此处，挣扎不是指“垂死挣扎”的意思），当然，我们要仔细观看是怎么“挣扎”的。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      现今，人们使用
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       XP
      </span>
     </span>
     <span style="font-size: medium;">
      ，用
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IE
      </span>
     </span>
     <span style="font-size: medium;">
      上网冲浪，已成习惯（而且很舒服），当然不用理会欧盟的这一套（胡闹）。但是，法律是法律，欧盟偏偏要在这个问题上较劲。我们该怎么看？我们应当认为，多年前，微软曾经利用捆绑浏览器（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IE
      </span>
     </span>
     <span style="font-size: medium;">
      ）的手段把网景（
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Netscape
      </span>
     </span>
     <span style="font-size: medium;">
      公司）整垮，算是很典型的市场垄断行为，但是，后来（在多年之后）所发生的事情，比如，火狐浏览器的兴起，浏览器市场（谷歌
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Chrome
      </span>
     </span>
     <span style="font-size: medium;">
      、苹果
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Safari
      </span>
     </span>
     <span style="font-size: medium;">
      和
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Opera
      </span>
     </span>
     <span style="font-size: medium;">
      浏览器）竞争激烈，技术创新不断，改变了整个事情的性质。现在，整个情况都表明，微软的市场垄断（地位）正在不断被削弱，或者，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       on its way out
      </span>
     </span>
     <span style="font-size: medium;">
      。现在的真实情况是，微软虽然还是市场“老大”，但是，（对整个
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IT
      </span>
     </span>
     <span style="font-size: medium;">
      产业）已经不能发号施令了。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       2
      </span>
     </span>
     <span style="font-size: medium;">
      月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       9
      </span>
     </span>
     <span style="font-size: medium;">
      日，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       FireFox
      </span>
     </span>
     <span style="font-size: medium;">
      架构师
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mike Connor
      </span>
     </span>
     <span style="font-size: medium;">
      对媒体表示，他不想把
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       FireFox
      </span>
     </span>
     <span style="font-size: medium;">
      与
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Windows
      </span>
     </span>
     <span style="font-size: medium;">
      捆绑（这应当由微软来进行），他认为，浏览器的好坏，应该由用户自己去选择。言下之意是，微软捆绑
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IE
      </span>
     </span>
     <span style="font-size: medium;">
      ，怕什么，随他去，无可指责，欧盟偏袒
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Opera
      </span>
     </span>
     <span style="font-size: medium;">
      浏览器（挪威）没有道理（所谓“护短”）。如果
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      当真参与了这场诉讼，将会使得欧盟与微软的这场官司处境很微妙。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      到底应该不应该参加欧盟对微软的讨伐？对于此事，国际舆论界有不同的声音。
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       2
      </span>
     </span>
     <span style="font-size: medium;">
      月
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       12
      </span>
     </span>
     <span style="font-size: medium;">
      日，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      自己终于想明白了，急忙发表声明，
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      不能参加（这种讨伐），同时声称：
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      对此事只是作为“感兴趣的一方”（“
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       interested party”
      </span>
     </span>
     <span style="font-size: medium;">
      ）。这个词（类似外交词汇）是
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Mozilla
      </span>
     </span>
     <span style="font-size: medium;">
      憋了好几天才想出来的。于是，微软也该松一口气了，欧盟整治微软，没完没了，简直是胡闹。
     </span>
    </p>
    <p style="margin-bottom: 0cm;">
     <span style="font-size: medium;">
      我国软件重大专项要搞操作系统、数据库、办公套件，等等，就是不见我们自己要搞”国产”浏览器的研发。这是为什么？答曰：这玩意儿不赚钱。由此可见，所谓“软件重大专项”都是有商业赢利指标（目的）的。实际上，浏览器直接关系到国家信息安全，为何我们不自己研发？浏览器技术是否很简单？非也。我认为，国家投入资金，确保国家信息安全，不能忽视浏览器这一关键技术。我们国家特别需要一个自己的安全浏览器，而不是什么“国产”操作系统（因为，已经有了
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       GNU
      </span>
     </span>
     <span style="font-size: medium;">
      ／
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       Linux
      </span>
     </span>
     <span style="font-size: medium;">
      操作系统）。有些专家，平时不读书，不看报，净给（主管
     </span>
     <span style="font-family: AR PL UMing CN,serif;">
      <span style="font-size: medium;">
       IT
      </span>
     </span>
     <span style="font-size: medium;">
      技术的）官员出“馊主意”。
     </span>
    </p>
