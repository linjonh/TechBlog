    <h4>
     1.MySQL服务器
    </h4>
    <ul>
     <li>
      是名为
      <strong>
       mysqld
      </strong>
      的数据库服务器程序，
     </li>
     <li>
      和“主机”（host）不一样
     </li>
     <li>
      是一个
      <strong>
       多线程
      </strong>
      的
      <strong>
       单进程
      </strong>
     </li>
     <li>
      <strong>
       管理
      </strong>
      对磁盘和内存中数据库的访问
     </li>
     <li>
      <strong>
       支持
      </strong>
      并发的客户端连接
     </li>
     <li>
      <strong>
       支持
      </strong>
      多个存储引擎，常见的存储引擎包括
      <strong>
       InnoDB、MyISAM、Memory、Archive
      </strong>
     </li>
     <li>
      <strong>
       支持
      </strong>
      事务表（支持事务操作，如提交、回滚）和非事务表
     </li>
     <li>
      通过使用缓存和缓存区优化内存使用
     </li>
    </ul>
    <h4>
     2.简单理解术语：服务器和主机
    </h4>
    <ul>
     <li>
      <strong>
       服务器
      </strong>
      ：是具有版本号和功能列表的软件程序。
     </li>
     <li>
      <strong>
       主机
      </strong>
      ：是指运行服务器的物理或虚拟机器，可以是本地计算机、远程服务器或云服务器。包括硬件配置（cpu，内存，磁盘，网络接口），机器上运行的操作系统，网络地址（主机在网络中的唯一标识）。
     </li>
     <li>
      多个服务器可以在一台主机上运行。
     </li>
    </ul>
