   <div class="htmledit_views" id="content_views">
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       1.
      </span>
      查看是否安装过MySQL，如果不存在，则不显示任何内容
     </span>
    </p>
    <pre><code class="language-bash">rpm -qa | grep -i mysql # -i 忽略大小写</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       2.
      </span>
      查看MySQL服务状态
     </span>
    </p>
    <pre><code class="language-bash">systemctl status mysqld.service</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       3.
      </span>
      关闭 mysql 服务
     </span>
    </p>
    <pre><code class="language-bash">systemctl stop mysqld.service</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       4.
      </span>
      查看当前 mysql 卸载状况
     </span>
    </p>
    <pre><code class="language-bash">rpm -qa | grep -i mysql
# 或
yum list installed | grep mysql</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       5.
      </span>
      卸载上述命令查询出的已安装程序
     </span>
    </p>
    <pre><code class="language-bash">yum remove mysql-xxx mysql-xxx mysql-xxx mysqk-xxxx</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       6.
      </span>
      删除 mysql 相关文件
     </span>
    </p>
    <pre><code class="language-bash"># 查找相关文件
find / -name mysql

# 删除上述命令查找出的相关文件
rm -rf xxx</code></pre>
    <p style="margin-left:0; margin-right:0; text-align:left">
     <span style="color:#000000">
      <span style="color:#3370ff">
       7.
      </span>
      删除 my.cnf
     </span>
    </p>
    <pre><code class="language-bash">rm -rf /etc/my.cnf</code></pre>
    <p>
