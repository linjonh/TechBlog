    <h2>
     一、安装mysql
    </h2>
    <h3>
     1.下载地址
    </h3>
    <p>
     mysql官方网站：
     <a href="www.mysql.com" rel="nofollow">
      www.mysql.com
     </a>
    </p>
    <p>
     也可以从Oracle官网进入：
     <a href="https://www.oracle.com/" rel="nofollow" title="https://www.oracle.com/">
      https://www.oracle.com/
     </a>
    </p>
    <p>
     下载地址：
     <a href="https://dev.mysql.com/downloads/mysql/" rel="nofollow" title="https://dev.mysql.com/downloads/mysql/">
      https://dev.mysql.com/downloads/mysql/
     </a>
    </p>
    <p>
     <img alt="" height="869" src="https://i-blog.csdnimg.cn/direct/f8e1f5cb2adc4083a85fd1fa11caf0d2.png" width="1920"/>
    </p>
    <p>
     <img alt="" height="869" src="https://i-blog.csdnimg.cn/direct/4cd759776a874c12b5421fcd10054753.png" width="1920"/>
    </p>
    <p>
     <img alt="" height="869" src="https://i-blog.csdnimg.cn/direct/5ac5b8895e19408fbe33fea76f53eec0.png" width="1920"/>
    </p>
    <p>
     进入以下页面后，选择对应的版本和操作系统，本文以8.0.26版本的压缩包方式举例。
    </p>
    <p>
     <img alt="" height="869" src="https://i-blog.csdnimg.cn/direct/fd2a93a08fd04469bf68183b54c10a61.png" width="1920"/>
    </p>
    <h3>
     2.解压缩
    </h3>
    <p>
     点击第一个压缩包进行下载，下载完成后将压缩包解压到你想解压的位置，我的解压路径为：D:\Mysql
    </p>
    <p>
     <img alt="" height="869" src="https://i-blog.csdnimg.cn/direct/b3a4d35670ca425eac9dd20190f6937f.png" width="1920"/>
    </p>
    <h3>
     3.修改配置文件
    </h3>
    <p>
     解压完成后，在主目录下复制、创建一个xx.ini，修改为
     <code>
      my.ini
     </code>
    </p>
    <p>
     <img alt="" height="593" src="https://i-blog.csdnimg.cn/direct/2ced3f1f1a074d48a543a890e1e1f158.png" width="1004"/>
    </p>
    <p>
     以下为my.ini的内容，修改其中的
     <code>
      basedir
     </code>
     和
     <code>
      datadir
     </code>
    </p>
    <blockquote>
     <p>
      basedir=mysql主目录 ​
     </p>
     <p>
      datadir=mysql主目录\data
     </p>
    </blockquote>
    <blockquote>
     <p>
      <code>
       [mysqld]
       <br/>
       # 设置3306端口
       <br/>
       port=3306
       <br/>
       # 设置mysql的安装目录   ----------是你的文件路径-------------
       <br/>
       basedir=D:\Mysql\mysql-8.0.26-winx64
       <br/>
       # 设置mysql数据库的数据的存放目录  ---------是你的文件路径,data文件夹自行创建
       <br/>
       #datadir=D:\Mysql\mysql-8.0.26-winx64\data
       <br/>
       # 允许最大连接数
       <br/>
       max_connections=200
       <br/>
       # 允许连接失败的次数。
       <br/>
       max_connect_errors=10
       <br/>
       # 服务端使用的字符集默认为utf8mb4
       <br/>
       character-set-server=utf8mb4
       <br/>
       # 创建新表时将使用的默认存储引擎
       <br/>
       default-storage-engine=INNODB
       <br/>
       # 默认使用“mysql_native_password”插件认证
       <br/>
       #mysql_native_password
       <br/>
       default_authentication_plugin=mysql_native_password
       <br/>
       [mysql]
       <br/>
       # 设置mysql客户端默认字符集
       <br/>
       default-character-set=utf8mb4
       <br/>
       [client]
       <br/>
       # 设置mysql客户端连接服务端时默认使用的端口
       <br/>
       port=3306
       <br/>
       default-character-set=utf8mb4
      </code>
     </p>
    </blockquote>
    <h3>
     4.配置bin目录为环境变量
    </h3>
    <ul>
     <li>
      <p>
       在桌面或文件资源管理器中，右键点击“此电脑”或“计算机”，选择“属性”。
      </p>
     </li>
     <li>
      <p>
       在系统窗口中，点击“高级系统设置”。
      </p>
     </li>
     <li>
      <p>
       在“系统属性”窗口中，点击“高级”选项卡下的“环境变量”按钮。
      </p>
     </li>
     <li>
      <p>
       在“环境变量”窗口中，你可以看到“用户变量”和“系统变量”两部分。（配置“用户变量”中的Path则只有当前用户可以使用，配置”系统变量“则所有用户可以使用）
      </p>
     </li>
     <li>
      <p>
       点击编辑”系统变量“中的Path，点击新建，将你的bin目录路径添加上去，点击确定就ok了
       <img alt="" height="664" src="https://i-blog.csdnimg.cn/direct/94d90276fc04474aa732edf0b52e507a.png" width="677"/>
      </p>
     </li>
    </ul>
    <h3>
     5.初始化命令（记录控制台给出的随机密码 ，一定要记住，这是初次登录的密码）
    </h3>
    <p>
     打开cmd，注意：一定要用管理员身份打开，cmd打开后切换到bin目录下并进行初始化
    </p>
    <blockquote>
     <p>
      C:\Windows\System32&gt;cd D:\Mysql\mysql-8.0.26-winx64\bin
     </p>
     <p>
      C:\Windows\System32&gt;d:
     </p>
     <p>
      D:\Mysql\mysql-8.0.26-winx64\bin&gt;
     </p>
     <p>
      D:\Mysql\mysql-8.0.26-winx64\bin&gt;
      <code>
       mysqld --initialize --console
      </code>
     </p>
    </blockquote>
    <p>
     <img alt="" height="290" src="https://i-blog.csdnimg.cn/direct/3b30552a5da94d7cbb2fb517810135e2.png" width="238"/>
    </p>
    <h3>
     6.然后先给mysql服务创建名称(方便到时候建立多个mysql服务时不冲突)
    </h3>
    <blockquote>
     <p>
      mysqld --install mysql8
     </p>
    </blockquote>
    <p>
     这时候查看一下服务是否注册
    </p>
    <ul>
     <li>
      <p>
       按
       <code>
        Win + R
       </code>
       ，输入
       <code>
        services.msc
       </code>
       ，然后按
       <code>
        Enter
       </code>
       。或者右键任务栏选择”任务管理器“，点击服务，启动类型可以更改，右键服务，点击属性即可更改
       <img alt="" height="582" src="https://i-blog.csdnimg.cn/direct/fd04a9a11ef5451da5e430e06cf14afc.png" width="1005"/>
      </p>
     </li>
    </ul>
    <h3>
     7.启动服务：
    </h3>
    <blockquote>
     <p>
      net start mysql
     </p>
    </blockquote>
    <p>
     <img alt="" height="89" src="https://i-blog.csdnimg.cn/direct/de6e564a58b04053a1402d0426d0c7da.png" width="543"/>
    </p>
    <h3>
     8.密码登陆（第5步中的密码）：
    </h3>
    <blockquote>
     <p>
      mysql -u root -p
     </p>
    </blockquote>
    <p>
     <img alt="" height="373" src="https://i-blog.csdnimg.cn/direct/c22862d9fa7642109eb3f3e1c0e77aa6.png" width="785"/>
    </p>
    <h3>
     9.修改密码：
    </h3>
    <blockquote>
     <p>
      ALTER USER root@localhost IDENTIFIED  BY '新密码'
     </p>
    </blockquote>
    <h3>
     10. 停止服务：
    </h3>
    <blockquote>
     <p>
      net stop mysql
     </p>
    </blockquote>
    <h3>
     11.删除服务：（第6步中的服务就没有mysql8了）
    </h3>
    <blockquote>
     <p>
      mysqld --remove
     </p>
    </blockquote>

    -&gt; deed varchar(255)
