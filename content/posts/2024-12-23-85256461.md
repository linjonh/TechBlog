     详解 Python 中的数字类型

    <p>
     <strong>
      <span style="color:#f33b45;">
       Python 3 ⽀持 3 种不同类型的数字类型。
      </span>
     </strong>
     <br/>
     int 整型数字，⽐如 2015。
     <br/>
     float 浮点型数字，⽐如 3.14。
     <br/>
     complex 复数，⽐如 3+2j。
    </p>
    <h2>
     2.1 查看变量类型
    </h2>
    <p>
     Python 使⽤内置函数 type() 来查看变量的类型。在 Python 中，内置了⼀些⾼效强⼤的对象类型，使得开发⼈员不⽤从零开始进⾏编程。实际上， Python 中的每样东西都是对象。虽然 Python 中没有类型声明，但表达式的语法决定了创建和使⽤的对象的类型。⼀旦创建了⼀个对象，它就和操作集合绑定了，这就是所谓的动态类型和强类型语⾔。即 Python ⾃动根据表达式创建类型，⼀旦创建成功，只能对⼀个对象进⾏适合该类型的有效操作。
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; x = 12
&gt;&gt;&gt; type(x)
&lt;class 'int'&gt;</code></pre>
    <h2>
     2.2 整型变量
    </h2>
    <p>
     整型（int）字⾯量在 Python 中属于 int 类。
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; i = 100
&gt;&gt;&gt; i
100</code></pre>
    <p>
     数字可以进⾏各种运算，如：
    </p>
    <pre class="has"><code class="language-python">123 + 345</code></pre>
    <p>
     还可以使⽤数学模块进⾏更⾼级的运算，如产⽣随机数等等：
    </p>
    <pre class="has"><code class="language-python">import random
print(random.random())</code></pre>
    <h2>
     2.3 浮点类型
    </h2>
    <p>
     浮点数（float）是指有⼩数点的数字。
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; f = 12.3
&gt;&gt;&gt; type(f)
&lt;class 'float'&gt;</code></pre>
    <h2>
     2.4 复数
    </h2>
    <p>
     复数（Complex number）由实数和虚数两部分构成，虚数⽤ j 表⽰。我们可以这样定义⼀个复数：
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; x = 2+3j
&gt;&gt;&gt; type(x)
&lt;class 'complex'&gt;</code></pre>
    <h2>
     2.5 运算符
    </h2>
    <p>
     Python 有各种运算符，我们可以使⽤这些运算符完成计算。运算符见下表
     <br/>
     <img alt="" class="has" height="323" src="https://i-blog.csdnimg.cn/blog_migrate/365c0cf75e497403caf632b7f325781d.png" width="570"/>
    </p>
    <h2>
     2.6 运算符的优先级别
    </h2>
    <p>
     Python 按照运算符的有限级别计算表达式的值，⽐如：
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; 3 * 4 + 1</code></pre>
    <p>
     在上⾯的表达式中，应该先进⾏加运算还是乘运算？为了搞清楚这个问题，我们需要明⽩Python 中运算符的优先级别，表3-2显⽰了运算符的优先级别，依次从⾼到底排列如下：
    </p>
    <p>
     <img alt="" class="has" height="824" src="https://i-blog.csdnimg.cn/blog_migrate/6f9e8b28d6ab9b6499ad11821ff3afb4.png" width="466"/>
    </p>
    <p>
     在上表中我们可以看到，乘法运算的级别⾼于加法，因此，先进⾏乘法运算，再进⾏加法运算，最后的计算结果为 13。
    </p>
    <h2>
     <br/>
     2.7 增强赋值运算符
    </h2>
    <p>
     增强赋值运算符能简化赋值声明语句，例如：
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; count = 1
&gt;&gt;&gt; count = count + 1
&gt;&gt;&gt; count
2</code></pre>
    <p>
     使⽤增强赋值运算符，我们可以将上述代码变为：
     <br/>
    </p>
    <pre class="has"><code class="language-python">&gt;&gt;&gt; count = 1
&gt;&gt;&gt; count += 1
&gt;&gt;&gt; count
2</code></pre>
    <p>
     类似的增强赋值运算符，除了 += 外，还有-=， %=， // = , / = , ∗= , ∗∗=。
     <br/>
    </p>
