    <h3>
     <a id="_14">
     </a>
     摘要
    </h3>
    <p>
     在 Swift 语言中，原生并没有提供基于队列的栈实现。本篇文章将介绍如何
     <strong>
      使用两个队列模拟 LIFO 栈
     </strong>
     ，并提供完整的 Swift 代码实现。我们会分析代码逻辑、测试示例，并探讨时间和空间复杂度，帮助开发者理解该方法的核心原理。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/033fb49516be49f08d7533c3cc0df26d.png"/>
    </p>
    <h3>
     <a id="_19">
     </a>
     描述
    </h3>
    <p>
     请你仅使用两个队列实现一个后入先出（LIFO）的栈，并支持普通栈的全部四种操作（
     <code>
      push
     </code>
     、
     <code>
      top
     </code>
     、
     <code>
      pop
     </code>
     和
     <code>
      empty
     </code>
     ）。
    </p>
    <p>
     实现
     <code>
      MyStack
     </code>
     类：
    </p>
    <ul>
     <li>
      <code>
       void push(int x)
      </code>
      将元素 x 压入栈顶。
     </li>
     <li>
      <code>
       int pop()
      </code>
      移除并返回栈顶元素。
     </li>
     <li>
      <code>
       int top()
      </code>
      返回栈顶元素。
     </li>
     <li>
      <code>
       boolean empty()
      </code>
      如果栈是空的，返回
      <code>
       true
      </code>
      ；否则，返回
      <code>
       false
      </code>
      。
     </li>
    </ul>
    <p>
     <strong>
      注意：
     </strong>
    </p>
    <ul>
     <li>
      你只能使用队列的标准操作 —— 也就是
      <code>
       push to back
      </code>
      、
      <code>
       peek/pop from front
      </code>
      、
      <code>
       size
      </code>
      和
      <code>
       is empty
      </code>
      这些操作。
     </li>
     <li>
      你所使用的语言也许不支持队列。 你可以使用 list （列表）或者 deque（双端队列）来模拟一个队列 , 只要是标准的队列操作即可。
     </li>
    </ul>
    <p>
     <strong>
      示例：
     </strong>
    </p>
    <pre><code>输入：
["MyStack", "push", "push", "top", "pop", "empty"]
[[], [1], [2], [], [], []]
输出：
[null, null, null, 2, 2, false]

解释：
MyStack myStack = new MyStack();
myStack.push(1);
myStack.push(2);
myStack.top(); // 返回 2
myStack.pop(); // 返回 2
myStack.empty(); // 返回 False
</code></pre>
    <p>
     <strong>
      提示：
     </strong>
    </p>
    <ul>
     <li>
      <code>
       1 &lt;= x &lt;= 9
      </code>
     </li>
     <li>
      最多调用
      <code>
       100
      </code>
      次
      <code>
       push
      </code>
      、
      <code>
       pop
      </code>
      、
      <code>
       top
      </code>
      和
      <code>
       empty
      </code>
     </li>
     <li>
      每次调用
      <code>
       pop
      </code>
      和
      <code>
       top
      </code>
      都保证栈不为空
     </li>
    </ul>
    <p>
     <strong>
      进阶：
     </strong>
     你能否仅用一个队列来实现栈。
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/direct/a0d338fa17054066953ec43bb8f178a1.png"/>
    </p>
    <h3>
     <a id="_62">
     </a>
     题解答案
    </h3>
    <p>
     我们可以使用
     <strong>
      两个队列
     </strong>
     <code>
      queue1
     </code>
     和
     <code>
      queue2
     </code>
     来模拟栈的行为。实现方式如下：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         push(x)
        </code>
       </strong>
       :
      </p>
      <ul>
       <li>
        直接将
        <code>
         x
        </code>
        添加到
        <code>
         queue1
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         pop()
        </code>
       </strong>
       :
      </p>
      <ul>
       <li>
        将
        <code>
         queue1
        </code>
        中的
        <strong>
         前 n-1 个元素
        </strong>
        依次移动到
        <code>
         queue2
        </code>
        。
       </li>
       <li>
        <code>
         queue1
        </code>
        中仅剩下最后一个元素，即栈顶元素，弹出它并返回。
       </li>
       <li>
        交换
        <code>
         queue1
        </code>
        和
        <code>
         queue2
        </code>
        ，使
        <code>
         queue1
        </code>
        始终作为主要存储队列。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         top()
        </code>
       </strong>
       :
      </p>
      <ul>
       <li>
        先执行与
        <code>
         pop()
        </code>
        相同的步骤，但不移除栈顶元素，而是返回它。
       </li>
       <li>
        依然交换
        <code>
         queue1
        </code>
        和
        <code>
         queue2
        </code>
        ，保持队列状态不变。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        <code>
         empty()
        </code>
       </strong>
       :
      </p>
      <ul>
       <li>
        直接判断
        <code>
         queue1
        </code>
        是否为空。
       </li>
      </ul>
     </li>
    </ol>

    <h3>
     <a id="_131">
     </a>
     题解代码分析
    </h3>
    <h4>
     <a id="1_pushx_133">
     </a>
     1. push(x)
    </h4>
    <ul>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(1) —— 直接插入队列末尾，时间复杂度为 O(1)。
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1) —— 仅存储额外的
      <code>
       x
      </code>
      ，不会增加额外数据结构的空间。
     </li>
    </ul>
    <h4>
     <a id="2_pop_138">
     </a>
     2. pop()
    </h4>
    <ul>
     <li>
      <strong>
       核心逻辑
      </strong>
      ：
      <ol>
       <li>
        将
        <code>
         queue1
        </code>
        中的
        <code>
         n-1
        </code>
        个元素逐个转移到
        <code>
         queue2
        </code>
        。
       </li>
       <li>
        <code>
         queue1
        </code>
        中最后一个元素即为栈顶元素，将其移除并返回。
       </li>
       <li>
        交换
        <code>
         queue1
        </code>
        和
        <code>
         queue2
        </code>
        角色，确保
        <code>
         queue1
        </code>
        始终是主要存储队列。
       </li>
      </ol>
     </li>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(n) —— 需要移动
      <code>
       n-1
      </code>
      个元素。
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1) —— 只使用
      <code>
       queue1
      </code>
      和
      <code>
       queue2
      </code>
      两个队列。
     </li>
    </ul>
    <h4>
     <a id="3_top_147">
     </a>
     3. top()
    </h4>
    <ul>
     <li>
      <strong>
       核心逻辑
      </strong>
      ：
      <ol>
       <li>
        先执行
        <code>
         pop()
        </code>
        的前半部分，将
        <code>
         n-1
        </code>
        个元素转移到
        <code>
         queue2
        </code>
        。
       </li>
       <li>
        记录
        <code>
         queue1
        </code>
        剩余的最后一个元素（即栈顶）。
       </li>
       <li>
        将该元素移入
        <code>
         queue2
        </code>
        ，再交换
        <code>
         queue1
        </code>
        和
        <code>
         queue2
        </code>
        。
       </li>
      </ol>
     </li>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(n) —— 需要移动
      <code>
       n-1
      </code>
      个元素。
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1) —— 只使用
      <code>
       queue1
      </code>
      和
      <code>
       queue2
      </code>
      。
     </li>
    </ul>
    <h4>
     <a id="4_empty_156">
     </a>
     4. empty()
    </h4>
    <ul>
     <li>
      <strong>
       时间复杂度
      </strong>
      ：O(1) —— 直接判断
      <code>
       queue1
      </code>
      是否为空。
     </li>
     <li>
      <strong>
       空间复杂度
      </strong>
      ：O(1) —— 仅返回布尔值。
     </li>
    </ul>

     总结

    <ol>

      </ul>
     </li>
