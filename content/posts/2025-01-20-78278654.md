     三种方法：
    </p>
    <p>
     1. pip 安装 ：
    </p>
    <p>
    </p>
    <pre><code class="language-html">pip install opencv-python ，最新版为opencv3</code></pre>
    安装后
    <br/>
    <pre><code class="language-python">&gt;&gt;&gt; import cv2
&gt;&gt;&gt; print cv2.__version__</code></pre>
    <br/>
    <br/>
    <p>
    </p>
    <p>
     <span style="font-family:monospace">
      参考：
      <a href="http://www.cnblogs.com/lclblack/p/6377710.html" rel="nofollow noopener noreferrer" target="_blank">

     </span>
    </p>
    <p>
     <span style="font-family:monospace">
      2. anaconda的conda安装 ,可以指定opencv版本，如opencv2
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
     </span>
    </p>
    <pre><code class="language-html">conda install -c https://conda.binstar.org/menpo opencv=2.4.9</code></pre>
    <br/>

    <p>
     <span style="font-family:monospace">
      3. 源码装
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
      <a href="https://docs.opencv.org/3.0-beta/doc/py_tutorials/py_setup/py_setup_in_fedora/py_setup_in_fedora.html#install-opencv-python-in-fedora" rel="nofollow noopener noreferrer" target="_blank">
       https://docs.opencv.org/3.0-beta/doc/py_tutorials/py_setup/py_setup_in_fedora/py_setup_in_fedora.html#install-opencv-python-in-fedora
      </a>
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
      装完后，
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
     </span>
    </p>
    <p style='margin-top:1em; margin-bottom:1em; text-align:justify; line-height:18.2px; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>
     Installation is over. All files are installed in
     <tt class="docutils literal" style='font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; min-width:780px; max-width:1080px; color:rgb(0,48,72); padding:1px'>
      <span class="pre">
       /usr/local/
      </span>
     </tt>
     folder. But to use it, your Python should be able to find OpenCV module. You have two options for that.
    </p>
    <ol class="arabic" style='list-style-position:initial; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>
     <li style="text-align:justify; line-height:18.2px; margin-top:1em; margin-bottom:1em">
      <p class="first" style="margin-top:1em; margin-bottom:1em; line-height:18.2px">
       <strong>
        Move the module to any folder in Python Path
       </strong>
       : Python path can be found out by entering
       <tt class="docutils literal" style='font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; min-width:780px; max-width:1080px; color:rgb(0,48,72); padding:1px'>
        <span class="pre">
         import
        </span>
        <span class="pre">
         sys;print
        </span>
        <span class="pre">
         sys.path
        </span>
       </tt>
       in Python terminal. It will print out many locations. Move
       <tt class="docutils literal" style='font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; min-width:780px; max-width:1080px; color:rgb(0,48,72); padding:1px'>
        <span class="pre">
         /usr/local/lib/python2.7/site-packages/cv2.so
        </span>
       </tt>
       to any of this folder. For example,
      </p>
      <blockquote>
       <div class="highlight-bash">
        <div class="highlight" style="background:rgb(238,255,204)">
         <pre style='overflow-x:auto; overflow-y:hidden; font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; font-size:13px; letter-spacing:0.015em; line-height:15.6px; padding:0.5em; border:1px solid rgb(204,204,204); background-color:rgb(248,248,248)'>su mv /usr/local/lib/python2.7/site-packages/cv2.so /usr/lib/python2.7/site-packages
</pre>
        </div>
       </div>
      </blockquote>
     </li>
    </ol>
    <p style='margin-top:1em; margin-bottom:1em; text-align:justify; line-height:18.2px; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>
     But you will have to do this every time you install OpenCV.
    </p>
    <ol class="arabic" start="2" style='list-style-position:initial; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>
     <li style="text-align:justify; line-height:18.2px; margin-top:1em; margin-bottom:1em">
      <p class="first" style="margin-top:1em; margin-bottom:1em; line-height:18.2px">
       <strong>
        Add ``/usr/local/lib/python2.7/site-packages`` to the PYTHON_PATH
       </strong>
       : It is to be done only once. Just open
       <tt class="docutils literal" style='font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; min-width:780px; max-width:1080px; color:rgb(0,48,72); padding:1px'>
        <span class="pre">
         ~/.bashrc
        </span>
       </tt>
       and add following line to it, then log out and come back.
      </p>
      <blockquote>
       <div class="highlight-bash">
        <div class="highlight" style="background:rgb(238,255,204)">
         <pre style='overflow-x:auto; overflow-y:hidden; font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; font-size:13px; letter-spacing:0.015em; line-height:15.6px; padding:0.5em; border:1px solid rgb(204,204,204); background-color:rgb(248,248,248)'><span class="nb" style="color:rgb(0,112,32)">export </span><span class="nv" style="color:rgb(187,96,213)">PYTHONPATH</span><span class="o" style="color:rgb(102,102,102)">=</span><span class="nv" style="color:rgb(187,96,213)">$PYTHONPATH</span>:/usr/local/lib/python2.7/site-packages
</pre>
        </div>
       </div>
      </blockquote>
     </li>
    </ol>
    <p style='margin-top:1em; margin-bottom:1em; text-align:justify; line-height:18.2px; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>
     Thus OpenCV installation is finished. Open a terminal and try
     <tt class="docutils literal" style='font-family:Consolas,"DejaVu Sans Mono","Bitstream Vera Sans Mono",monospace; min-width:780px; max-width:1080px; color:rgb(0,48,72); padding:1px'>
      <span class="pre">
       import
      </span>
      <span class="pre">
       cv2
      </span>
     </tt>
     .
    </p>
    <p style='margin-top:1em; margin-bottom:1em; text-align:justify; line-height:18.2px; font-family:"Open Sans","Lucida Grande","Lucida Sans Unicode",Geneva,Verdana,sans-serif; font-size:14px'>

    <p>
     5. 覆盖原有opencv
    </p>
    <p>
     如果服务器已经安装了opencv,但是通过
     <span style="font-family:monospace">
      anaconda的conda安装，也安装了python的opencv接口，会发现其中也包含了opencv的lib,include文件，如下：
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
      <img alt="" src="https://img-blog.csdn.net/20171028004459583?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveGlhbWVudGluZ3Rhbw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
      </img>
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
      这时可以使用这个版本的opencv覆盖原装的，方法是修改bashrc,如下：
     </span>
    </p>
    <p>
     <span style="font-family:monospace">
     </span>
    </p>
    <pre><code class="language-plain">## 动态库搜索路径</code></pre>
    <pre><code class="language-plain"> export LD_LIBRARY_PATH=/home/caiyong.wang/anaconda2/lib:$LD_LIBRARY_PATH 
##(静态库搜索路径) 程序编译期间查找动态链接库时指定查找共享库的路径
 export LIBRARY_PATH=/home/caiyong.wang/anaconda2/lib:$LIBRARY_PATH 
 
##c程序头文件搜索路径
 export C_INCLUDE_PATH=/home/caiyong.wang/anaconda2/include:$C_INCLUDE_PATH 
##c++程序头文件搜索路径
 export CPLUS_INCLUDE_PATH=/home/caiyong.wang/anaconda2/include:$CPLUS_INCLUDE_PATH 
##pkg-config 路径
 export PKG_CONFIG_PATH=/home/caiyong.wang/anaconda2/lib/pkgconfig/:$PKG_CONFIG_PATH</code></pre>
    <br/>
    可以使用原方法测试。
