# 浅谈Windows发展史

## 

## 简介

从微软发布Windows1.0开始，到现在已经有快40年历史了，接下来让我们浅浅的谈一下微软的发展史（只记录大家都知道的）

## Windows1.0

![](https://i-blog.csdnimg.cn/blog_migrate/984fe933351a4f302f9110ffbfbfa317.png)

Windows 1.0是
[微软](https://baike.baidu.com/item/%E5%BE%AE%E8%BD%AF/124767?fromModule=lemma_inlink "微软")
于1985年11月20日发布的操作系统，这也是微软第一个图形化操作系统。基本的功能也是有了。

## Windows2.0

![](https://i-blog.csdnimg.cn/blog_migrate/62ebb824dd8df2720e686978eb701e9c.png)

Windows 2.0是微软于1987年12月9日发布的基于
[MS-DOS](https://baike.baidu.com/item/MS-DOS/1120792?fromModule=lemma_inlink "MS-DOS")
的操作系统，和苹果系统很相似，所以苹果把微软告上了法庭，但是微软赢了官司，要不然现在不会有win10,win11了

## Windows3.0

![](https://i-blog.csdnimg.cn/blog_migrate/1b713ee05a4e7b93d9995819929516d7.png)

Windows 3.0是微软于1990年5月22日发布的操作系统，推出了多种语言，销量大增。

## Windows3.1

![](https://i-blog.csdnimg.cn/blog_migrate/91e5ab41da07af7f174dbdaf8bac302b.png)

Windows 3.1是微软于1992年4月6日发布的操作系统，基本就是Windows3.0另一个名字

## Windows3.2

![](https://i-blog.csdnimg.cn/blog_migrate/943be5fa6644178fdc53b20affe71915.png)

Windows 3.2于1994年4月14日发布的，这是第一个支持中文的系统，很多人对它的印象很深（尤其是老人）

## Windows95

![](https://i-blog.csdnimg.cn/blog_migrate/6e9998fe26d4515fd1b97c9fa7369efa.png)

这是微软于1995年发布的操作系统。很多人对这个系统印象很深，因为这是微软最成功的操作系统，为微软奠定的霸主的地位。当时排队购买的人堪比买iPhone的人多。

## Windows98

这是微软于1998推出的操作系统推出的操作系统。Windows98稳固了微软的霸主地位，很多人对他的印象也很深。

Windows me于2000年发布,俗称蓝屏小王子。其稳定性有多差你是知道的，所以这也是Windows的最大的败笔，导致很多人都换了Windows2000。

## Windows xp

![](https://i-blog.csdnimg.cn/blog_migrate/2441f0c547ff44af7f4a29eec3650f5a.png)

Windows xp于2001年发布，凭借着全新的UI和强大的功能，占领了市场，但是由于vista的问题，硬是挺到了2009年的windows7

## Windows vista

![](https://i-blog.csdnimg.cn/blog_migrate/1f9abff9ba3a90d63dd9d116524f727b.png)

Windows vista是微软于2006年推出的系统，是公认的Windows历史上最美的系统，但是由于微软认为人们太有钱了，推出的过于着急，导致了vista成为Windows上的败笔

## Windows7

![](https://i-blog.csdnimg.cn/blog_migrate/1e0cb9165d72a55039f5284c1d5781f7.png)

Windows7于2009年推出，微软吸取了上次的教训，把vista改了改，成了Windows7，获得了大家的一致好评，于2023年（去年）彻底结束支持，目前还有不少“钉子户”

## Windows Thin PC

![](https://i-blog.csdnimg.cn/blog_migrate/7be35ea498a17b805eee1b0c628c4d78.png)

虽然这个系统不常见，但是我提一下，这里是Windows7的精简版，还是官方的，不过只支持英文

## Windows8

![](https://i-blog.csdnimg.cn/blog_migrate/31d56e57bebc8d07bdaed3dc13bd6635.png)

Windows8于2011年发布，对UI进行了大改动，但是设计的不太友好，所以不被人所喜欢

## Windows8.1

![](https://i-blog.csdnimg.cn/blog_migrate/34d5316cff2ce653cdcff7964a6a1723.png)

Windows8.1在8上做了一些改动，不过不大（Windows8.1的主流停止支持时间和Windows7的ESU版本停止支持时间一样，在2023/1/10）

## Windows10

![](https://i-blog.csdnimg.cn/blog_migrate/062661a56810458a439c4e4cb5265713.png)

Windows10于2015年发布，微软保留了win8的磁贴，大受欢迎（用Windows10的小伙伴们要珍惜了，明年，也就是2025 年，微软就要对Windows10的主流版本停止支持了）

## Windows11

![](https://i-blog.csdnimg.cn/blog_migrate/9af78b553ba374b7d2ee2bbba20140bc.png)

Windows11于2021年发布，微软干了很多反人类的事情，如右键菜单强制隐藏，口碑一般
