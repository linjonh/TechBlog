    <h2>
     转换的本质：
    </h2>
    <p>
     对于一个十进制的数字，比如说153.其本质是每一数位上的数字乘上这一位上的权重，即：
    </p>
    <p>
     153=（1*10^2）+(5*10^1)+(3*10^0)
    </p>
    <p>
     对于二进制，只不过把10换成2，任意一个非负整数都有唯一的一个二进制表示：
    </p>
    <p>
     （153）10=（10011001）2
    </p>
    <h2>
     将任意进制转换为十进制
    </h2>
    <p>
     假设给一个数组来表示一个K进制（假设k&gt;10）的整数，我们该如何得到它的十进制数？
    </p>
    <pre><code class="language-cpp">ll x = 0;
for(int i=1;i&lt;=n;i++){
    x=x*k+a[i];
}
cout&lt;&lt;x&lt;&lt;endl;</code></pre>
    <h2>
     把十进制转化为任意进制
    </h2>
    <p>
     假设现在有一个十进制的数X,如何转换为k进制？
     <br/>
    </p>
    <pre><code class="language-cpp">ll x;
cin&gt;&gt;x;
while (x)
    a[++ cnt] = x%k,x/=k;
reverse(a+1,a+1+cnt);</code></pre>
    <h3>
     例题（lanqiao OJ 2489）
    </h3>
    <h4 id="问题描述">
     问题描述
    </h4>
    <p>
     请问十六进制数 2021ABCD 对应的十进制是多少？
    </p>
    <h4 id="答案提交">
     答案提交
    </h4>
    <p>
     这是一道结果填空的题，你只需要算出结果后提交即可。本题的结果为一个整数，在提交答案时只填写这个整数，填写多余的内容将无法得分。
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
typedef long long ll;
const int N=50;
int a[N];

int main()
{
    string s="2021ABCD";
    for(int i=0;i&lt;s.length();++i)
    {
       if('0'&lt;=s[i]&amp;&amp;s[i]&lt;='9')a[i+1]=s[i]-'0';
       else a[i+1]=s[i]-'A'+10;
    }  
     ll x=0;
     for(int i=1;i&lt;=s.length();++i)
      x=x*16+a[i];
      cout&lt;&lt;x&lt;&lt;"\n";
       return 0;
}</code></pre>
