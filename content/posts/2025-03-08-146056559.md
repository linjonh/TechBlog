    <h3>
     <a id="1TypeScript_1">
     </a>
     1、TypeScript简介
    </h3>
    <h4>
     <a id="11TypeScript_2">
     </a>
     1.1、什么是TypeScript？
    </h4>
    <p>
     <strong>
      TypeScript
     </strong>
     是JavaScript的
     <strong>
      超集
     </strong>
     ，它添加了
     <strong>
      可选的静态类型
     </strong>
     和
     <strong>
      基于类的面向对象编程
     </strong>
     等特性
    </p>
    <p>
     TypeScript代码
     <strong>
      最终会被编译成JavaScript代码
     </strong>
     在浏览器中运行
    </p>
    <h4>
     <a id="12TypeScript_7">
     </a>
     1.2、为什么选择TypeScript？
    </h4>
    <ul>
     <li>
      <strong>
       静态类型检查
      </strong>
      ，提前发现错误
     </li>
     <li>
      更好的IDE支持，智能提示更强大
     </li>
     <li>
      <strong>
       更容易维护
      </strong>
      的代码结构
     </li>
     <li>
      支持最新的JavaScript特性
     </li>
    </ul>
    <h3>
     <a id="2_12">
     </a>
     2、环境准备
    </h3>

     <a id="21NodejsTypeScript_13">

    </p>

      </a>
     </p>
    </blockquote>
    <p>
     接下来安装下
     <strong>
      TypeScript编译器
     </strong>
    </p>
    <pre><code class="prism language-sh"><span class="token comment"># 安装TypeScript编译器</span>
<span class="token function">npm</span> <span class="token function">install</span> <span class="token parameter variable">-g</span> typescript

<span class="token comment"># 验证安装</span>

     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/93b148c8f2cc442082db85c2eac8bf23.png"/>

    <h3>
     <a id="3_60">
     </a>
     3、基本类型系统
    </h3>
    <h4>
     <a id="31_61">
     </a>
     3.1、基本类型
    </h4>
    <h5>
     <a id="_62">
     </a>
     数字类型
    </h5>
    <p>
     TypeScript里的
     <strong>
      所有数字都是浮点数
     </strong>
     ，其类型为
     <code>
      number
     </code>
     <br/>
     除了支持十进制和十六进制字面量，TypeScript还支持ECMAScript 2015中引入的二进制和八进制字面量
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> decimal<span class="token operator">:</span> <span class="token builtin">number</span> <span class="token operator">=</span> <span class="token number">6</span><span class="token punctuation">;</span>     <span class="token comment">// 十进制</span>
<span class="token keyword">let</span> hex<span class="token operator">:</span> <span class="token builtin">number</span> <span class="token operator">=</span> <span class="token number">0xf00d</span><span class="token punctuation">;</span>     <span class="token comment">// 十六进制</span>
<span class="token keyword">let</span> binary<span class="token operator">:</span> <span class="token builtin">number</span> <span class="token operator">=</span> <span class="token number">0b1010</span><span class="token punctuation">;</span>    <span class="token comment">// 二进制</span>
</code></pre>
    <h5>
     <a id="_70">
     </a>
     布尔类型
    </h5>
    <p>
     仅仅true和false两个值，用于进行逻辑判断
    </p>
    <pre><code class="prism language-ts"><span class="token comment">// 布尔类型</span>
<span class="token keyword">let</span> isDone<span class="token operator">:</span> <span class="token builtin">boolean</span> <span class="token operator">=</span> <span class="token boolean">false</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_76">
     </a>
     字符串
    </h5>
    <p>
     用于存储文本数据，可以使用
     <code>
      '
     </code>
     或者
     <code>
      "
     </code>
     包裹文本内容，使用反引号包裹后，可以使用
     <code>
      ${}
     </code>
     嵌入表达式
    </p>
    <pre><code class="prism language-ts"><span class="token comment">// 字符串类型</span>
<span class="token keyword">let</span> color<span class="token operator">:</span> <span class="token builtin">string</span> <span class="token operator">=</span> <span class="token string">"blue"</span><span class="token punctuation">;</span>
<span class="token keyword">let</span> sentence<span class="token operator">:</span> <span class="token builtin">string</span> <span class="token operator">=</span> <span class="token template-string"><span class="token template-punctuation string">`</span><span class="token string">The color is </span><span class="token interpolation"><span class="token interpolation-punctuation punctuation">${<!-- --></span>color<span class="token interpolation-punctuation punctuation">}</span></span><span class="token template-punctuation string">`</span></span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="any_83">
     </a>
     any类型
    </h5>
    <p>
     可以用于对在编程阶段不清楚类型的值指定一个类型，通常这种值可能来源于
     <strong>
      动态内容
     </strong>
     ，比如用户的输入、三方代码等
     <br/>
     如果我们
     <strong>
      不希望类型检查器对其进行检查
     </strong>
     ，而是
     <strong>
      直接通过编译检查
     </strong>
     ，可以使用
     <code>
      any
     </code>
     类型对变量进行标记
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> notSure<span class="token operator">:</span> <span class="token builtin">any</span> <span class="token operator">=</span> <span class="token number">4</span><span class="token punctuation">;</span>
notSure <span class="token operator">=</span> <span class="token string">"maybe a string"</span><span class="token punctuation">;</span>
</code></pre>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> list<span class="token operator">:</span> <span class="token builtin">any</span><span class="token punctuation">[</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token boolean">true</span><span class="token punctuation">,</span> <span class="token string">"free"</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
list<span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token number">100</span><span class="token punctuation">;</span>  <span class="token comment">// 可以修改为任意类型</span>
</code></pre>
    <p>
     给了我们更大的灵活性，但同时也
     <strong>
      失去了 TypeScript 的类型安全特性
     </strong>
     ，建议谨慎使用
     <code>
      any
     </code>
     类型
    </p>
    <h5>
     <a id="nullundefined_95">
     </a>
     null和undefined
    </h5>
    <p>
     默认情况下
     <code>
      null
     </code>
     和
     <code>
      undefined
     </code>
     是
     <strong>
      所有类型的子类型
     </strong>
     ，因此他俩可以赋值给其他类型
     <br/>
     但是，如果指定
     <code>
      --strictNullChecks
     </code>
     时，
     <code>
      null
     </code>
     和
     <code>
      undefined
     </code>
     只能赋值给
     <code>
      void
     </code>
     和它们各自
    </p>
    <pre><code class="prism language-ts"><span class="token comment">// null 和 undefined 类型示例</span>
<span class="token keyword">let</span> u<span class="token operator">:</span> <span class="token keyword">undefined</span> <span class="token operator">=</span> <span class="token keyword">undefined</span><span class="token punctuation">;</span>  <span class="token comment">// undefined 类型只能赋值 undefined</span>
<span class="token keyword">let</span> n<span class="token operator">:</span> <span class="token keyword">null</span> <span class="token operator">=</span> <span class="token keyword">null</span><span class="token punctuation">;</span>           <span class="token comment">// null 类型只能赋值 null</span>
</code></pre>
    <p>
     二者也要注意意义上的区别：
    </p>
    <ul>
     <li>
      <code>
       undefined
      </code>
      ：表示一个变量
      <strong>
       未被赋值
      </strong>
     </li>
     <li>
      <code>
       null
      </code>
      ：表示一个变量
      <strong>
       被明确地赋值为空
      </strong>
     </li>
    </ul>
    <p>
     放一个空的文件
     <code>
      tsconfig.json
     </code>
     在当前目录下
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a816f2a860844d3a9de3448bdeddf27a.png"/>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> result<span class="token operator">:</span> <span class="token builtin">number</span><span class="token punctuation">;</span>   <span class="token comment">// 通常情况下null和undefined可以赋值给其他类型</span>
result <span class="token operator">=</span> <span class="token keyword">null</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     此时，再修改
     <code>
      tsconfig.json
     </code>
     中的编译配置
    </p>
    <pre><code class="prism language-json"><span class="token punctuation">{<!-- --></span>
  <span class="token string-property property">"compilerOptions"</span><span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-property property">"strict"</span><span class="token operator">:</span> <span class="token boolean">true</span><span class="token punctuation">,</span>   <span class="token comment">// 开启严格模式</span>
  <span class="token punctuation">}</span><span class="token punctuation">,</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     开启
     <strong>
      严格模式
     </strong>
     后，
     <code>
      null
     </code>
     和
     <code>
      undefined
     </code>
     除了给自身外，只能赋值给
     <code>
      void
     </code>
     (
     <code>
      undefinded
     </code>
     可以，
     <code>
      null
     </code>
     不行)
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7b2ab71607194e6db9bdf2caf0fc5632.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c13f7dcf55644689a23c6dfbd9eb7165.png"/>
     </img>
    </p>
    <blockquote>
     <p>
      鼓励尽可能地使用
      <code>
       --strictNullChecks
      </code>
     </p>
    </blockquote>
    <h5>
     <a id="void_126">
     </a>
     void类型
    </h5>
    <p>
     <code>
      void
     </code>
     类型
     <strong>
      表示没有任何类型
     </strong>
     ，一个函数没有任何返回值时，通常的返回值类型就是
     <code>
      void
     </code>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">function</span> <span class="token function">warnUser</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token operator">:</span> <span class="token keyword">void</span> <span class="token punctuation">{<!-- --></span>
    <span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token string">"This is a warning message"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token comment">// 不需要返回值</span>
<span class="token punctuation">}</span>
</code></pre>
    <pre><code class="prism language-ts"><span class="token comment">// void 类型变量只能赋值 undefined（在非严格模式下也可以赋值 null）</span>
<span class="token keyword">let</span> unusable<span class="token operator">:</span> <span class="token keyword">void</span> <span class="token operator">=</span> <span class="token keyword">undefined</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="unknown_139">
     </a>
     unknown类型
    </h5>
    <p>
     <code>
      unknown
     </code>
     类型是TypeScript中的一个特殊类型，它表示任何值，但比
     <code>
      any
     </code>
     更安全
     <br/>
     使用
     <code>
      unknown
     </code>
     类型时，我们
     <strong>
      必须先进行类型检查或类型断言
     </strong>
     才能对值进行操作
     <br/>
     如果我们有一个
     <code>
      unknown
     </code>
     类型的变量，我们不能直接调用它的方法或访问它的属性，
     <strong>
      必须先通过
      <code>
       typeof
      </code>
      、
      <code>
       instanceof
      </code>
      等类型检查或使用类型断言来确定具体类型
     </strong>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> userName<span class="token operator">:</span> <span class="token builtin">string</span><span class="token punctuation">;</span>
<span class="token keyword">let</span> userInput<span class="token operator">:</span> <span class="token builtin">unknown</span><span class="token punctuation">;</span>
<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token keyword">typeof</span> userInput <span class="token operator">===</span> <span class="token string">"string"</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    userName <span class="token operator">=</span> userInput<span class="token punctuation">;</span> <span class="token comment">// 正确：已经进行类型检查</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="32_151">
     </a>
     3.2、复合类型
    </h4>
    <h5>
     <a id="_152">
     </a>
     数组
    </h5>
    <p>
     数组的特征是
     <strong>
      多个类型相同的元素
     </strong>
     组成的一个列表，成为一个复合变量，有两种方法进行定义：
     <br/>
     第一种，在数组元素类型的后面跟上
     <code>
      []
     </code>
     表示对应类型的数组
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> arr<span class="token operator">:</span> <span class="token builtin">number</span><span class="token punctuation">[</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">]</span><span class="token punctuation">;</span>    <span class="token comment">// number类型的数组</span>
</code></pre>
    <p>
     第二种，使用内置的
     <code>
      Array
     </code>
     接口配合泛型使用
     <code>
      Array&lt;T&gt;
     </code>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> list<span class="token operator">:</span> <span class="token builtin">Array</span><span class="token operator">&lt;</span><span class="token builtin">number</span><span class="token operator">&gt;</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_162">
     </a>
     元组
    </h5>
    <p>
     元组用来表示一个
     <strong>
      已知元素数量和类型的数组
     </strong>
     ，
     <strong>
      各元素类型不必相同
     </strong>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> tuple<span class="token operator">:</span> <span class="token punctuation">[</span><span class="token builtin">string</span><span class="token punctuation">,</span> <span class="token builtin">number</span><span class="token punctuation">]</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token string">"hello"</span><span class="token punctuation">,</span> <span class="token number">10</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="_167">
     </a>
     枚举
    </h5>
    <p>
     枚举类型可以增加一组数值的
     <strong>
      可读性
     </strong>
     ，其中的变体默认从
     <strong>
      0
     </strong>
     开始编号，也可以手动赋值
     <br/>
     可以通过枚举的值获取枚举的名字
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">enum</span> Color <span class="token punctuation">{<!-- --></span>
    Red<span class="token punctuation">,</span>
    Green<span class="token punctuation">,</span>
    Blue
<span class="token punctuation">}</span>
<span class="token keyword">let</span> c<span class="token operator">:</span> Color <span class="token operator">=</span> Color<span class="token punctuation">.</span>Green<span class="token punctuation">;</span>
<span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>c<span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 输出 1</span>
<span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>Color<span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// "Green"</span>
</code></pre>
    <h4>
     <a id="33_180">
     </a>
     3.3、高级类型
    </h4>
    <h5>
     <a id="_181">
     </a>
     联合类型
    </h5>
    <p>
     联合类型指的是
     <strong>
      多个类型组成的一个新类型
     </strong>
     ，使用符号
     <code>
      |
     </code>
     表示
     <br/>
     比如两个类型，A和B，组合成联合类型为
     <code>
      A|B
     </code>
     ，此时既可以赋值为A类型，也可以赋值为B类型
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> value<span class="token operator">:</span> <span class="token builtin">string</span> <span class="token operator">|</span> <span class="token builtin">number</span><span class="token punctuation">;</span>
value <span class="token operator">=</span> <span class="token string">"Hello"</span><span class="token punctuation">;</span>
<span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token keyword">typeof</span> value<span class="token punctuation">)</span><span class="token punctuation">;</span>   <span class="token comment">// 此时是string</span>
value <span class="token operator">=</span> <span class="token number">123</span><span class="token punctuation">;</span>
<span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span><span class="token keyword">typeof</span> value<span class="token punctuation">)</span><span class="token punctuation">;</span>   <span class="token comment">// 此时是number</span>
</code></pre>
    <p>
     常用的用法比如将变量声明为
     <strong>
      可空
     </strong>
     的类型
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> enName<span class="token operator">:</span> <span class="token builtin">string</span> <span class="token operator">|</span> <span class="token keyword">null</span><span class="token punctuation">;</span>
<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token function">getInput</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>   <span class="token comment">// 不确定的状态</span>
    enName <span class="token operator">=</span> <span class="token string">"Tom"</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token keyword">else</span> <span class="token punctuation">{<!-- --></span>
    enName <span class="token operator">=</span> <span class="token keyword">null</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token builtin">console</span><span class="token punctuation">.</span><span class="token function">log</span><span class="token punctuation">(</span>enName<span class="token operator">?.</span><span class="token function">charAt</span><span class="token punctuation">(</span><span class="token number">0</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 通过?判空</span>
</code></pre>
    <h5>
     <a id="_201">
     </a>
     交叉类型
    </h5>
    <p>
     交叉类型指的多个类型组成的一个新类型，使用符号
     <code>
      &amp;
     </code>
     表示
     <br/>
     感觉很奇怪啊，怎么既是A又是B？
     <br/>
     <code>
      let x: string &amp; number;
     </code>
     这是什么？编译器觉得这不是个东西
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/982f18f5de9d4c768028c82086141e50.png">
      <br/>
      其实它主要用于
      <strong>
       对象的合成
      </strong>
     </img>
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">type</span> <span class="token class-name"><span class="token constant">API</span></span> <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>     <span class="token comment">// 网站接口信息</span>
    address<span class="token operator">:</span> <span class="token builtin">string</span><span class="token punctuation">;</span>
    port<span class="token operator">:</span> <span class="token builtin">number</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token keyword">type</span> <span class="token class-name">RegisterInfo</span> <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>    <span class="token comment">// 网站注册信息</span>
    name<span class="token operator">:</span> <span class="token builtin">string</span><span class="token punctuation">;</span>
    createdAt<span class="token operator">:</span> Date<span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token keyword">let</span> webInfo<span class="token operator">:</span> <span class="token constant">API</span> <span class="token operator">&amp;</span> RegisterInfo <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>    <span class="token comment">// 全量信息，两者内容拼接</span>
    address<span class="token operator">:</span> <span class="token string">"192.168.101.76"</span><span class="token punctuation">,</span>
    port<span class="token operator">:</span> <span class="token number">8080</span><span class="token punctuation">,</span>
    name<span class="token operator">:</span> <span class="token string">"Grace Under Pressure"</span><span class="token punctuation">,</span>
    createdAt<span class="token operator">:</span> <span class="token keyword">new</span> <span class="token class-name">Date</span><span class="token punctuation">(</span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h5>
     <a id="_223">
     </a>
     类型别名
    </h5>
    <p>
     使用
     <code>
      type
     </code>
     命令可以为一个类型指定别名
     <br/>
     上一个例子里已经用到了，
     <code>
      API
     </code>
     、
     <code>
      RegisterInfo
     </code>
     所代表的其实就是别名，这是为了增加代码的可读性
     <br/>
     不使用别名，直接使用右边的内容也可以，但是这样明显不够优雅
    </p>
    <pre><code class="prism language-ts"><span class="token keyword">let</span> webInfo<span class="token operator">:</span> <span class="token punctuation">{<!-- --></span>
  address<span class="token operator">:</span> <span class="token builtin">string</span><span class="token punctuation">;</span>
  port<span class="token operator">:</span> <span class="token builtin">number</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token operator">&amp;</span> <span class="token punctuation">{<!-- --></span>
  name<span class="token operator">:</span> <span class="token builtin">string</span><span class="token punctuation">;</span>
  createdAt<span class="token operator">:</span> Date<span class="token punctuation">;</span>
<span class="token punctuation">}</span> <span class="token operator">=</span> <span class="token punctuation">{<!-- --></span>
  address<span class="token operator">:</span> <span class="token string">"192.168.101.76"</span><span class="token punctuation">,</span>
  port<span class="token operator">:</span> <span class="token number">8080</span><span class="token punctuation">,</span>
  name<span class="token operator">:</span> <span class="token string">"Grace Under Pressure"</span><span class="token punctuation">,</span>
  createdAt<span class="token operator">:</span> <span class="token keyword">new</span> <span class="token class-name">Date</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">,</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
