    <h1 class="title-article" id="articleContentId">
     a = b &amp;&amp;c 的含义
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <span style="color:#fe2c24">
      <strong>
       简单一句话： result = condition &amp;&amp; value; condition 为真取 value的值，condition为假就取condition的值，真取后假取前
      </strong>
     </span>
    </p>
    <pre><code class="hljs">// 示例 1: b 为真值
let b = 1;
let c = 2;
let a = b &amp;&amp; c;
console.log(a); // 输出: 2

// 示例 2: b 为假值
b = 0;
c = 2;
a = b &amp;&amp; c;
console.log(a); // 输出: 0</code></pre>
    <h4>
     应用场景
    </h4>
    <p>
     这种表达式常用于条件赋值，即只有当某个条件为真时才进行赋值操作。例如：
    </p>
    <pre><code class="hljs">let condition = true;
let value = 10;
let result = condition &amp;&amp; value;
console.log(result); // 输出: 10

condition = false;
result = condition &amp;&amp; value;
console.log(result); // 输出: false</code></pre>
    <p>
     在这个例子中，只有当
     <code>
      condition
     </code>
     为
     <code>
      true
     </code>
     时，
     <code>
      result
     </code>
     才会被赋值为
     <code>
      value
     </code>
     的值；否则，
     <code>
      result
     </code>
     会被赋值为
     <code>
      condition
     </code>
     的值（即
     <code>
      false
     </code>
     ）。
    </p>
