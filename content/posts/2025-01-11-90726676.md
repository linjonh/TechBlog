    <p>
     本篇博客介绍的是使用distutils +
     <a href="http://setup.py" rel="nofollow">
      setup.py
     </a>
     打包发布自定义的python包。
    </p>
    <p>
     假设python工程目录结构如下，
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/18d013d3279cb587b203e179322d2067.png">
      <br/>
      一个比较简单的需求：
      <br/>
      在打包时，我们希望
     </img>
    </p>
    <ul>
     <li>
      对于一些pkg，全部打包（所有py文件），对于其他pkg，部分打包。
     </li>
     <li>
      打包pkg里的配置文件
     </li>
     <li>
      打包不属于任何pkg的模块（单个py脚本）
     </li>
    </ul>
    <p>
     为了满足上述需求，
     <a href="http://setup.py" rel="nofollow">
      setup.py
     </a>
     脚本编写如下：
    </p>
    <pre><code>from distutils.core import setup
setup(
package_dir={'':'python'},
packages=['pkg1'],
package_data={'': ['config/1.txt']},
py_modules=['pkg2.p21','test1']
)
</code></pre>
    <p>
     为了简单起见，
     <a href="http://xn--setup-4h1hk050b.py" rel="nofollow">
      上述setup.py
     </a>
     省略了对待生成包名字/版本等描述。
    </p>
    <p>
     <strong>
      解释：
     </strong>
     <br/>
     1.package_dir描述了setup.py脚本与待打包pkg的路径关系： 是个字典类型，value表示以setup.py为起点的相对路径。
     <br/>
     2 packages 是在package_dir指定路径下的python pkg(必须有__init__.py), 可以写多个。
     <br/>
     3 package_data 填写准备打包的配置文件。
     <br/>
     4 py_modules 填写待打包的单个脚本，比如例子中的pkg2,有时侯并不像把pkg2里的所有脚本都打包，或者有时侯需要打包不属于任何pkg的脚本（
     <a href="http://xn--test1-gv5ij80i.py" rel="nofollow">
      比如test1.py
     </a>
     ),此时用py_modules就比较方便。
    </p>
    <p>
     <strong>
      打包：
     </strong>
    </p>
    <pre><code>python setup.py bdist (这里有多种参数可选择)
</code></pre>
    <p>
     打包结果 ：会生成一个build和dist目录。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7aeb4bfa9f84d79978dfa3639f7d86ab.png">
      <br/>
      <strong>
       本机安装测试
      </strong>
      <br/>
      可以用 sudo python
      <a href="http://setup.py" rel="nofollow">
       setup.py
      </a>
      install --record installed.txt 在本机安装验证，
      <br/>
      加 --record installed.txt 是为了记录安装路径，方便验证完删除。
      <br/>
      安装后，可以去python系统目录下验证是否安装成功。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7b7eb6674a6463e0365074f18266cb32.png">
       <br/>
       开启一个终端，测试安装成功。
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e5e2b7db4164cac145edddf98cef8db4.png"/>
      </img>
     </img>
    </p>

      https://docs.python.org/2/distutils/setupscript.html
     </a>
    </p>
