    <h3>
     Nanobrowser介绍
    </h3>
    <p>
     Nanobrowser 是一款基于 AI 驱动的 Web 自动化开源工具，以 Chrome 扩展程序的形式运行，让用户只需利用自己的 LLM API 密钥便能轻松启动多代理工作流，成为 OpenAI Operator 的免费替代方案。它不仅支持灵活接入您喜欢的 LLM 提供商，还确保所有操作在本地浏览器中执行，从而保障用户隐私与数据安全。通过透明的开源机制和直观的交互侧边栏，Nanobrowser 能够自动执行跨网站的重复任务，并实时反馈任务状态和对话历史，助您高效管理复杂的网络自动化流程。目前已支持 OpenAI 和 Anthropic，未来将不断扩展更多 LLM 集成，满足不断升级的自动化需求，真正实现高性价比与便捷操作的完美结合。
    </p>

     <img alt="" src="https://i-blog.csdnimg.cn/direct/c412f376da324978ad2ba8ab04d9d920.gif"/>

    <h3>
     Nanobrowser主要特点
    </h3>
    <ul>
     <li>
      <strong>
       多代理系统
      </strong>
      ：专门的人工智能代理协作完成复杂的网络工作流程
     </li>
     <li>
      <strong>
       交互式侧面板
      </strong>
      ：直观的聊天界面，可实时更新状态
     </li>
     <li>
      <strong>
       任务自动化
      </strong>
      ：无缝自动执行跨网站重复的 Web 自动化任务
     </li>
     <li>
      <strong>
       后续问题
      </strong>
      ：询问有关已完成任务的后续问题
     </li>
     <li>
      <strong>
       对话历史记录
      </strong>
      ：轻松访问和管理您的 AI 代理交互历史记录
     </li>
     <li>
      <strong>
       多个 LLM 支持
      </strong>
      ：连接您首选的 LLM 提供商，并将不同的模型分配给不同的代理
     </li>
    </ul>
    <p>
    </p>
    <p>
     Nanobrowser 是一款开源的 Chrome 扩展工具，专注于 AI 驱动的网页自动化。它基于多智能体系统实现复杂的网页任务，如信息提取、自动化操作等。用户可以使用自己的 LLM API 密钥，灵活选择不同的模型为不同智能体提供支持。
    </p>
    <p>
     Nanobrowser 完全免费，作为
     <a href="https://zhida.zhihu.com/search?content_id=254877282&amp;content_type=Article&amp;match_order=1&amp;q=OpenAI+Operator&amp;zhida_source=entity" rel="nofollow" title="OpenAI Operator">
      OpenAI Operator
     </a>
     的开源替代方案，支持在本地浏览器运行，注重隐私保护，不涉及云服务。它让 AI 在浏览器中变得更加智能高效，适用于信息收集、电商购物、内容创作等多种场景。
    </p>
    <h3>
     Nanobrowser 的主要功能
    </h3>
    <ul>
     <li>
      <strong>
       多智能体系统
      </strong>
      ：
      <ul>
       <li>
        <strong>
         Planner（规划器）
        </strong>
        ：负责制定和调整任务策略。
       </li>
       <li>
        <strong>
         Navigator（导航器）
        </strong>
        ：执行网页导航和操作。
       </li>
       <li>
        <strong>
         Validator（验证器）
        </strong>
        ：检查任务是否成功完成。
       </li>
      </ul>
     </li>
    </ul>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       交互式侧边栏
      </strong>
      ：提供直观的聊天界面，实时显示任务状态，用户用自然语言与智能体交互。
     </li>
     <li>
      <strong>
       任务自动化
      </strong>
      ：自动化重复性网页任务，如信息提取、数据整理等，节省时间和精力。
     </li>
     <li>
      <strong>
       多 LLM 支持
      </strong>
      ：支持连接多种
      <a href="https://zhida.zhihu.com/search?content_id=254877282&amp;content_type=Article&amp;match_order=1&amp;q=%E5%A4%A7%E5%9E%8B%E8%AF%AD%E8%A8%80%E6%A8%A1%E5%9E%8B&amp;zhida_source=entity" rel="nofollow" title="大型语言模型">
       大型语言模型
      </a>
      （LLM）提供商，用户根据需求为不同智能体选择不同的模型。
     </li>
    </ul>
    <h3>
     Nanobrowser 的技术原理
    </h3>
    <ul>
     <li>
      <strong>
       基于 LLM 的智能体架构
      </strong>
      ：Nanobrowser 的核心是多智能体系统，每个智能体由大型语言模型（LLM）驱动。智能体分工协作，高效完成复杂任务：
      <ul>
       <li>
        规划器 Planner ：制定和完善任务策略。
       </li>
       <li>
        导航器 Navigator ：浏览网站并执行操作。
       </li>
       <li>
        验证器 Validator ：确认任务结果，检查任务是否成功完成。
       </li>
      </ul>
     </li>
    </ul>
    <p>
    </p>
    <ul>
     <li>
      <strong>
       动态任务调整与自适应性
      </strong>
      ：当遇到障碍或任务失败时，Planner 智能体自动调整策略，重新规划任务路径，确保任务能够成功完成。
     </li>
     <li>
      <strong>
       集成多种 LLM 提供商
      </strong>
      ：支持连接 OpenAI、Anthropic 等主流 LLM 提供商，用户根据需求选择不同的模型，为不同智能体分配最适合的模型。
     </li>
    </ul>

    <h4>
     2. 安装
    </h4>
    <ul>
     <li>
      解压
      <code>
       nanobrowser.zip
      </code>
      。
     </li>
     <li>
      在 Chrome 中打开
      <code>
       chrome://extensions/
      </code>
      。 在页面右上角启用
      <code>
       开发者模式
      </code>
      。
     </li>
     <li>
      开启后，点击页面左上角的
      <code>
       加载已解压的扩展程序
      </code>
      按钮。
     </li>
     <li>
      选择解压后的
      <code>
       nanobrowser
      </code>
      文件夹。
     </li>
    </ul>
    <h4>
     3. 配置智能体模型
    </h4>
    <ul>
     <li>
      点击工具栏中的 Nanobrowser 图标，打开侧边栏。
     </li>
     <li>
      点击
      <code>
       设置
      </code>
      图标（右上角）。
     </li>
     <li>
      添加你的 LLM API 密钥。
     </li>
     <li>
      为不同智能体（Navigator、Planner、Validator）选择使用的模型。
     </li>
    </ul>
