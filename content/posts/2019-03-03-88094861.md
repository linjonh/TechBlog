    <h2>
     <a id="7__0">
     </a>
     第7天 和电脑玩猜数字游戏
    </h2>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e7e2222be31f5fee426312e714a8a475.png"/>
    </p>
    <p>
     猜数字（又称 Bulls and Cows ）是一种古老的的密码破译类益智类小游戏，起源于20世纪中期，一般由两个人或多人玩，也可以由一个人和电脑玩。今天我们就来用 Python 写一个猜数字游戏！
    </p>
    <h3>
     <a id="_8">
     </a>
     游戏规则
    </h3>
    <ul>
     <li>
      一个玩家从1~10（或者1~100）之间选取一个随机数；
     </li>
     <li>
      另一个玩家尝试猜这个数字，如果猜的数太大则提示 “小了”，如果猜的数太小则提示 “大了”，直到猜对为止；
     </li>
     <li>
      谁猜的次数最少，谁就获胜。
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/9e18e0bc2c50a8de75a66276e4218922.jpeg"/>
    </p>
    <p>
     为了实现这个游戏，我们需要解决两个问题：
    </p>
    <ol>
     <li>
      如何生成一个随机数？
     </li>
     <li>
      如何实现分支结构的控制流程？
     </li>
    </ol>
    <h3>
     <a id="_25">
     </a>
     生成随机数
    </h3>
    <p>
     Python 已经为我们准备好了 random 模块，使用它可以很方便地生成随机数。暂时先不用管模块是什么，我们先把它用起来！
    </p>
    <p>
     在 Python 交互式环境中逐行输入以下代码：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> random
random<span class="token punctuation">.</span>randint<span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">10</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     第一行代码使用 import 命令导入 random 模块；第二行代码使用 random 模块中的
     <code>
      randint()
     </code>
     函数（它是 random integer 的缩写，表示随机的整数），生成一个 1~10 之间的随机数。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c9a58046a822da47e1e4f1244c7dabdf.gif"/>
    </p>
    <p>
     重复运行多次
     <code>
      random.randint(1, 10)
     </code>
     看看是否出现不同的随机数。（
     <code>
      import random
     </code>
     在程序中只需要执行一次即可）
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5973a9b8dac4935257f0235b0cceaef5.png"/>
    </p>
    <h3>
     <a id="_46">
     </a>
     分支结构
    </h3>
    <p>
     现实生活有很多 “分支” 情况，比如家里的水管分支。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/3af00fbe94083081c2232528a7fd3726.jpeg"/>
    </p>
    <p>
     物种进化的分支结构促进了生物多样性。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0757c0fccf615093256ba409a40c0082.jpeg"/>
    </p>
    <p>
     同样，在程序的世界里，分支结构也无处不在，通过分支的组合，可以编写出很多有趣的程序。通常，分支结构通过条件判断来实现。
    </p>
    <p>
     根据判断的情况，又可分为单项分支、双项分支、多项分支。三种结构的伪代码如下：
    </p>
    <h4>
     <a id="_60">
     </a>
     单项分支
    </h4>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件<span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
</code></pre>
    <h4>
     <a id="_67">
     </a>
     双项分支
    </h4>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件<span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
</code></pre>
    <p>
     单项分支和双项分支就像开关一样，要么开，要么关。只不过单项分支只关心其中一种情况，双项分支两种情况都关心。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/07c357108383752505904ac21a8fa9a1.jpeg"/>
    </p>
    <h4>
     <a id="_82">
     </a>
     多项分支
    </h4>
    <pre><code class="prism language-python"><span class="token keyword">if</span> 条件<span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
<span class="token keyword">elif</span> 条件<span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
<span class="token keyword">else</span><span class="token punctuation">:</span>
    <span class="token comment">#执行的内容...</span>
</code></pre>
    <p>
     多项分支中可以有很多个
     <code>
      elif
     </code>
     分支。就像风扇开关那样，有不同的档位，每个档位都有不同的处理方式。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/885fd93fa7257a766a0833945ec5ed9a.jpeg"/>
    </p>
    <p>
     <strong>
      说明：
     </strong>
    </p>
    <ul>
     <li>
      分支结构的条件判断顺序是自上而下逐个分支进行判断的；
     </li>
     <li>
      在分支结构中，无论如何只会执行一个分支；
     </li>
     <li>
      当执行完一个分之后，分支结构就会结束，后面的分支是不会执行的。
     </li>
    </ul>
    <h3>
     <a id="_106">
     </a>
     猜数字游戏
    </h3>
    <p>
     有了前面的知识背景，我们就可以撸起袖子写代码了。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6b1fdffd8ca82cd6b1d2c529eeb08d1f.png"/>
    </p>
    <p>
     打开编辑器，输入下面代码，并保存为
     <a href="http://guess.py" rel="nofollow">
      guess.py
     </a>
     文件。
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> random

target <span class="token operator">=</span> random<span class="token punctuation">.</span>randint<span class="token punctuation">(</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">10</span><span class="token punctuation">)</span>
guess <span class="token operator">=</span> <span class="token builtin">int</span><span class="token punctuation">(</span><span class="token builtin">input</span><span class="token punctuation">(</span><span class="token string">"1~10之间猜一个数字："</span><span class="token punctuation">)</span><span class="token punctuation">)</span>

<span class="token keyword">while</span> guess <span class="token operator">!=</span> target<span class="token punctuation">:</span>
    <span class="token keyword">if</span> guess <span class="token operator">&gt;</span> target<span class="token punctuation">:</span>
        <span class="token keyword">print</span><span class="token punctuation">(</span>guess<span class="token punctuation">,</span> <span class="token string">"太大了"</span><span class="token punctuation">)</span>
    <span class="token keyword">elif</span> guess <span class="token operator">&lt;</span> target<span class="token punctuation">:</span>
        <span class="token keyword">print</span><span class="token punctuation">(</span>guess<span class="token punctuation">,</span> <span class="token string">"太小了"</span><span class="token punctuation">)</span>

    guess <span class="token operator">=</span> <span class="token builtin">int</span><span class="token punctuation">(</span><span class="token builtin">input</span><span class="token punctuation">(</span><span class="token string">"再猜一遍："</span><span class="token punctuation">)</span><span class="token punctuation">)</span>

<span class="token keyword">print</span><span class="token punctuation">(</span><span class="token string">"猜对了！{} 是正确的数字"</span><span class="token punctuation">.</span><span class="token builtin">format</span><span class="token punctuation">(</span>guess<span class="token punctuation">)</span><span class="token punctuation">)</span>
</code></pre>
    <p>
     执行
     <code>
      python3 guess.py
     </code>
     看看运行效果吧~
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/dbfaaabc1b8b55af86960d9240de7d57.png"/>
    </p>
    <h3>
     <a id="_137">
     </a>
     代码说明
    </h3>
    <ul>
     <li>
      第一行代码导入 random 模块，以便于待会可以使用 randint 函数生成一个随机整数。
     </li>
     <li>
      生成的随机整数存放在变量 target 中。
     </li>
     <li>
      玩家输入的数字存放在变量 guess 中。
     </li>
     <li>
      接下来，判断 guess 和 target 的大小。
     </li>
    </ul>
