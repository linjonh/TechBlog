     <p>
      建一个网站主要就3大块：服务器、域名、以及程序。
      <br/>
      1、
      <strong>
       服务器
      </strong>
      ：对于学生党来说，是各大云平台重点扶持的对象，所以可以享受优惠服务器选购活动，例如腾讯云的“云+校园”，阿里云的“云翼计划”等等，都推出了物美价廉的学生服务器，在平时这些服务器是同配置价格最低的服务器，特别适合学生党来应用；
      <br/>
     </p>
     <p>
      <strong>
       2、域名
      </strong>
      ：国内目前主要的域名注册商是阿里云和腾讯云，分别对应旗下的万网和dnspod，可以根据需求选择合适自己的域名，当然短域名或类似知名双拼域名现在都很难买到；对于题主只是想把作业显得高大上，域名买个好记的就行；com是国际域名，价格相对较高，基本要55元/年，cn是国内域名，价格一般在29元/年
      <br/>
      注册好的域名需要备案，如果备案不方便的话，需要购买境外服务那样是不需要备案的。
     </p>
     <p>
      下面罗列一下备案需要的材料：
      <br/>
      <strong>
       备案资料
      </strong>
      <br/>
      企业网站备案需要准备:
      <br/>
      1份(营业执照)副本彩色扫描件或复印件;
      <br/>
      1份网站负责人的身份证彩色扫描件或复印件;
      <br/>
      1张负责人的半身彩色照片(.jpg格式);(需有对应服务商提供的专门背景做幕布。)
      <br/>
      1公章;
      <br/>
      主办单位所在地的详细联系方式;
      <br/>
      填写《信息安全管理协议》;
      <br/>
      填写《真实性核验单》。
      <br/>
      ​
      <br/>
      <strong>
       3、程序（代码）
      </strong>
      ：
      <br/>
      <a href="https://ac.aliyun.com/application/webdesign/sumei?userCode=28kqeewo" rel="nofollow">
       阿里云建站“速美
      </a>
      ”，
      <br/>
      ​
     </p>
     <p>
      不会自己写的话，就可以用上边的模板建站，简单快捷，千套模板任意选择​
      <br/>
      题主自己学开发的，可以编写代码，想必程序对题主来说不说难事；对于大部分个人建站用户来说，常用的程序有wordpress、dedecms等等，一般我们通过搜索引擎直接搜索对应安装包并搜索相关模板或主题，可以轻易的得到安装包及安装方法，至于具体的安装细节，我们暂且不表。
      <br/>
      有了域名、服务器和程序，接下来的工作就是把程序上传到服务器，配置完成后将域名解析到服务器即可打开，那么如何配置呢？
      <br/>
      程序运行需要独立的运行环境，对于不是特别熟悉linux操作系统的同学来说，手动配置环境难度极大，并且易出错，那么我们推荐集成环境，例如我们常用的
      <br/>
      <a href="https://www.bt.cn/?invite_code=MV9zemR1b2M=" rel="nofollow">
       宝塔面板 - 简单好用的Linux/Windows服务器运维管理面板
      </a>
      <br/>
      ​
     </p>
     <p>
      ，就非常简单，只需简单的几条命令就可以完成安装，在此提醒下各位，宝塔面板需要使用端口8888，如果要使用phpmyadmin，则需要打开888，所以请务必在安全组打开这两个端口，避免无法连接。
      <br/>
      宝塔面板安装好之后，在左侧栏选择软件安装，根据提示安装对应的软件如web服务、数据库、php等等，安装完毕后即可上传网站，通过简单的配置即可通过域名来访问网站（此处需要先将域名解析到对应的IP地址）。
      <br/>
      自此，一个网站就搭建完毕，题主想要的高大上的站点就已经完成。如果对搭建网站有疑问，也可以留言咨询
     </p>
