    <p>
     cpuburn 是模拟CPU满载运行时，对CPU内核进行压力测试，考察CPU的稳定性。
    </p>
    <p>
     一、安装
     <br/>
     该软件是免安装软件，下载地址：http://patrickmylund.com/projects/cpuburn
     <br/>
     支持使用的系统：Linux (32-bit)、Linux (64-bit)、Windows
     <br/>
     二、使用
     <br/>
     说明：如果已经安装了go，则可以运行：go get github.com/pmylund/cpuburn（cpuburn二进制文件将添加到gopath/bin文件夹中）。也可不安装Go独立运行。
     <br/>
     使用步骤如下：
     <br/>
     <strong>
      方式一：.默认直接运行
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b1c1e7f6af6ce885f0de8d1028d1d25.png">
      <br/>
      打开任务管理器，可看到cpu使用率到100%
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6ed8d0ed55f74660d7b28b5ca4fc48bc.png">
       <br/>
       <strong>
        方式二：使用管理器运行
       </strong>
       <br/>
       1.进入命令管理器，切换至cpuburn所在的文件夹中：cd 路径
       <br/>
       2.输入cpuburn.exe -n 核数 -u 更新时间，（使用的命令可看第三部分），查看运行情况，如下如所示：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a629f3d263fa569ecad02bfa082c27cf.png"/>
      </img>
     </img>
    </p>
    <p>
     <strong>
      三、使用命令
     </strong>
     <br/>
     ./cpuburn 刻录所有可用的内核
     <br/>
     ./cpuburn-n 2（刻录两个内核）
     <br/>
     ./cpuburn-n 4-u 5（刻录四个内核，每5秒更新一次）
    </p>
