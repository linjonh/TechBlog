# securtiy_crt访问ubuntu报Key exchange failed问题

  1. securityCRT 连接linux报错

Key exchange failed. No compatible key exchange method. The server supports
these methods: [__curve25519-sha256,curve25519-sha256@libssh.org,ecdh-
sha2-nistp256,ecdh-sha2-nistp384,ecdh-sha2-nistp521,diffie-hellman-group-
exchange-sha256,diffie-hellman-
group16-__](mailto:curve25519-sha256,curve25519-sha256@libssh.org,ecdh-
sha2-nistp256,ecdh-sha2-nistp384,ecdh-sha2-nistp521,diffie-hellman-group-
exchange-sha256,diffie-hellman-group16-)

需要修改/etc/ssh/sshd_config

在最后一行增加（注意下面的内容是一整行）

KexAlgorithms curve25519-sha256@libssh.org,ecdh-sha2-nistp256,ecdh-
sha2-nistp384,ecdh-sha2-nistp521,diffie-hellman-group-exchange-sha256,diffie-
hellman-group14-sha1,diffie-hellman-group-exchange-sha1,diffie-hellman-
group1-sha1
