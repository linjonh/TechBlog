# 36 在Vue3中如何通过axios请求后端数据
## 概述
在Vue3中通过axios请求后端数据是非常常用的方法，这节课我们来简单学习下。
本节课需要依赖一个第三方库，我们先安装：
```bash
yarn add axios
```
## 基本用法
我们先创建public/data.json，用于模拟后端返回的接口数据：
```json
{
"msg": "成功",
"status": true,
"data": []
}
```
接着创建src/components/Demo36.vue，代码如下：
