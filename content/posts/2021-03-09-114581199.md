# 八 python 环境安装

1. 安装python3.7

```bash
apt-get install python3
```

2. 安装pip及检测版本

```bash
apt-get install python3-venv python3-pip

pip3 install --upgrade pip
```

3. 安装常用的库文件

```bash
pip3 install request 
pip3 install selenium 
pip3 install urllib3 ......
```
