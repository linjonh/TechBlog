    <h2>
     <a id="_7">
     </a>
     前言
    </h2>
    <p>
     本文将按照Python的各个版本进行介绍的是
     <code>
      Ubuntu16.04
     </code>
     下安装相应版本
     <code>
      Python
     </code>
     的过程，可选择所需的版本进行查看，各个版本的命令基本相同，其中
     <code>
      Python3.8的是图解版，其余的是命令版
     </code>
     。本文仅在Ubuntu16.04下测试过，对于其他系统版本未测试，一般情况下具有通用性，这里看个人情况而定。
     <br/>
     本文对于Python安装的定位是：
     <code>
      独立安装，不修改或覆盖系统原有python的文件，包括软链接，易卸载
     </code>
     。
    </p>

    <h3>
     <a id="1__16">
     </a>
     1. 安装所需依赖
    </h3>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> libffi-dev uuid-dev lzma-dev liblzma-dev libncurses5-dev libgdbm-dev sqlite3 libsqlite3-dev openssl tcl8.6-dev tk8.6-dev libreadline-dev zlib1g-dev build-essential <span class="token function">bzip2</span> libbz2-1.0 libbz2-dev libc6-dev libdb-dev libexpat1 libexpat1-dev libgdbm3 libncursesw5-dev libpcap-dev libreadline5 libreadline6 libreadline6-dev libsqlite0 libsqlite0-dev libsqlite3-0 libssl-dev libssl1.0.0 libxml2-dev libxslt1-dev sqlite tcl tk tk-dev xz-utils zlib1g zlib1g-dev <span class="token function">make</span>  
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b6f2653de9e51e0682844b911f054a83.png"/>

    <h3>
     <a id="2_Python_22">
     </a>
     2. 配置Python版本切换
    </h3>
    <p>
     Ubuntu系统自带有
     <code>
      python2.7
     </code>
     和
     <code>
      python3.5
     </code>
     ，这里先添加这两个的版本切换，即使用
     <code>
      python
     </code>
     和
     <code>
      python3
     </code>
     两个命令时使用哪个版本的python，由于Ubuntu系统对自带的两个python有一定程度的依赖，本文是不建议切换上述
     <code>
      python
     </code>
     和
     <code>
      python3
     </code>
     两个命令的Python版本的，否则会带来一些问题。
    </p>
    <blockquote>
     <p>
      <strong>
       例如：升级Python后无法打开终端，系统启动总是提示软件更新时发生错误，无法打开系统设置的软件和更新界面等
      </strong>
      <br/>
      <em>
       <strong>
        命令的版本对应
       </strong>
      </em>
      ：
      <code>
       python
      </code>
      =&gt;
      <code>
       python2.7
      </code>
      ，
      <code>
       python3
      </code>
      =&gt;
      <code>
       python3.5
      </code>
     </p>
    </blockquote>
    <p>
     <strong>
      ① 查看候选列表中已有的Python版本（没有配置过的话，是不存在的）：
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--list</span> python
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/42018cece7ccfdd45eaee45f6dd9af56.png">
      <br/>
      <strong>
       ② 添加 python &amp; python3 指向选择
      </strong>
      <br/>
      最后的序号表示优先级，数字越大，优先级越高，会自动选取优先级高的作为默认指向，可手动更改默认指向，这里将系统默认的指向的优先级设置为100
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python python /usr/bin/python2.7 <span class="token number">100</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python python /usr/bin/python3.5 <span class="token number">99</span>
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/bin/python3.5 <span class="token number">100</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cb3f995a0aeffc5670ed9688c95f8cf9.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/92b974644801628209947f054f3ec038.png">
       <br/>
       <strong>
        ③ 查看候选列表中已有的Python版本
       </strong>
      </img>
     </img>
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--list</span> python
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--list</span> python3
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/8db01ea92ae8516de2ea5af3cfd67b5f.png"/>
    </p>
    <blockquote>
     <p>
      此时，
      <code>
       python
      </code>
      指向
      <code>
       python2.7
      </code>
      ，
      <code>
       python3
      </code>
      指向
      <code>
       python3.5
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ebf10908a5b7766a46d24c08f6bb15e1.png"/>
     </p>
    </blockquote>
    <p>
     <strong>
      ④ 创建新版python的安装目录（便于卸载）
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /usr/local
<span class="token function">sudo</span> <span class="token function">mkdir</span> <span class="token parameter variable">-p</span> /usr/local/python
<span class="token function">sudo</span> <span class="token function">chmod</span> <span class="token parameter variable">-R</span> <span class="token number">777</span> ./python
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/22449d6ce1eadb468fe0be3eaee52092.png"/>
    </p>
    <p>
     下面将进行各个版本的安装，至于版本号的选择，可自行到下面的网址去找，后续操作中修改其中的版本号即可。
     <br/>
     官方网址：
     <code>
      https://www.python.org/ftp/python/
     </code>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f8de08078618cf84db743e174f6415d4.png"/>
    </p>

    <h2>

     二、Python3.8（3.8.11–详细图解）
    </h2>
    <h3>
     <a id="1_66">
     </a>
     1.下载源码安装包
    </h3>
    <p>
     这里选的是Python3.8的3.8.11版，如果下载较慢，可尝试连接手机热点进行下载。如果已准备了安装包，可以跳过该步骤。
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~
<span class="token function">wget</span> https://www.python.org/ftp/python/3.8.11/Python-3.8.11.tgz
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1f7539eb4469e9161d7522f6c28291f0.png"/>
    </p>
    <h3>
     <a id="2_74">
     </a>
     2.创建安装目录
    </h3>
    <p>
     在上述前期准备中，已经在
     <code>
      /usr/local
     </code>
     目录下创建了一个
     <code>
      python
     </code>
     目录，那么各个新安装的python都将放在这里（安装多个的情况下），那么需要对每个版本的进行独立分包。实际上安装时如果没有创建会自动创建。
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /usr/local/python
<span class="token function">mkdir</span> ./python3.8
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4de68e655865f35106b301e2f9c51a36.png"/>
    </p>
    <h3>
     <a id="3_81">
     </a>
     3.解压安装包
    </h3>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~
<span class="token function">tar</span> <span class="token parameter variable">-xf</span> ~/Python-3.8.11.tgz
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b24ec43169c1efc768670b84ba4de76e.png"/>

     4.配置将要安装的目录

    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~/Python-3.8.11
./configure <span class="token assign-left variable">prefix</span><span class="token operator">=</span>/usr/local/python/python3.8 --enable-optimizations

    <h3>
     <a id="6_106">
     </a>
     6.安装
    </h3>
    <p>
     一定要使用
     <code>
      altinstall

     <code>
      install
     </code>
     会覆盖安装，可能会导致Ubuntu崩溃的情况。
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> ~/Python-3.8.11
<span class="token function">make</span> altinstall  <span class="token operator">&gt;</span><span class="token file-descriptor important">&amp;1</span><span class="token operator">|</span><span class="token function">tee</span> make.log
</code></pre>

    <h3>
     <a id="8_pythonpython3_130">
     </a>
     8. 添加和选择python&amp;python3命令指向
    </h3>
    <p>
     <em>
      <strong>
       添加命令指向
      </strong>
     </em>
     【数值比前面设置的默认指向的优先级100小即可】
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python python /usr/local/python/python3.8/bin/python3.8 <span class="token number">38</span> 
<span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--install</span> /usr/bin/python3 python3 /usr/local/python/python3.8/bin/python3.8 <span class="token number">38</span>
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/096a004e9b4a4cdbf06b3b0e428572f6.png"/>
    </p>
    <p>
     <strong>
      选择
      <code>
       python
      </code>
      命令的默认版本
     </strong>
     【这里只是查看，按回车即可】
    </p>
    <blockquote>
     <p>
      可以2.7，3.5，3.8，至于选取的问题，随着使用情景不同而有所改变，但使用完最好更换为Ubuntu系统默认的指向（
      <code>
       python2.7
      </code>
      ），此时已经是默认的指向，会自动选取优先级高的。
     </p>
    </blockquote>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> python
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b4084ff48c1f2e81d18e3f5504794499.png"/>
    </p>
    <p>
     <strong>
      选择
      <code>
       python3
      </code>
      命令的默认版本
     </strong>
     【这里只是查看，按回车即可】
    </p>
    <blockquote>
     <p>
      python3一般选择与Ubuntu系统默认的指向3.5，否则无法在启动器栏打开新终端。
     </p>
    </blockquote>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> update-alternatives <span class="token parameter variable">--config</span> python3
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a8ccc53e4edf031e701b2f41c9e46bba.png"/>
