    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px">
      一、安装Java开发工具包(Java Development Kit, 缩写为JDK)
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312">
      <br/>
     </span>
    </p>
    <p>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px; color:#ff0000">
      1. 下载JDK (Java SE)网址
     </span>
    </p>
    <span style="font-family:KaiTi_GB2312; font-size:18px">
     <a href="http://www.oracle.com/technetwork/java/index.html" rel="nofollow noopener noreferrer" target="_blank">
      http://www.oracle.com/technetwork/java/index.html
     </a>
     <br/>
     双击即可安装(如果是压缩包则解压即可)
     <br/>
    </span>
    <p>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <img alt="" src="https://img-my.csdn.net/uploads/201301/25/1359078846_1167.jpg">
       <br/>
      </img>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <span style="color:#990000">
       安装Java开发工具包JDK 时，Java运行环境JRE（Java Runtime Environment ）也是安装
       <br/>
       的一部分。Java的虚拟计算机--JVM是JRE的一部分，实现Java程序的跨平台。
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px; color:#ff0000">
      2、配置环境变量
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      (1) JAVA_HOME（写JDK安装目录）
      <br/>
      (2) CLASSPATH（.;%JAVA_HOME%\lib\dt.jar; %JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\bin）
      <br/>
      (3) PATH（;%JAVA_HOME%\bin）
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px">
      Windows XP:
      <br/>
      <span style="color:#cc0000">
       我的电脑右键  －&gt;  系统属性 －&gt; 高级
       <br/>
       －&gt;环境变量
       <br/>
       Windows 7:
       <br/>
       计算机右键－&gt;  属性 －&gt;高级系统属性
       <br/>
       －&gt;环境变量
      </span>
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:18px; color:#ff0000">
      (1) . JAVA_HOME
     </span>

      新建系统变量

      例如：

      <span style="color:#cc0000">
       新建（或编辑）系统变量
       <br/>
       －&gt;变量名写 PATH
       <br/>
       －&gt;变量值写;%JAVA_HOME%\bin

       （或添加在原来值的后面）
