    <h4>
     <a id="_0">
     </a>
     问题描述
    </h4>
    <p>
     遇到个vlc录制的rtsp视频流，duration时长只有12分钟，但src duration有3个多小时（实际正确时长），而且用potplayer能播放3个小时的完整视频，但vlc只能播放12分钟。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/49ec8bf39fe64ddeb7287662ca51ab9b.png"/>
    </p>
    <h4>
     <a id="_3">
     </a>
     解决方法
    </h4>
    <p>
     下载ffmpeg，使用以下命令：
    </p>
    <pre><code class="prism language-bash">ffmpeg <span class="token parameter variable">-ignore_editlist</span> <span class="token number">1</span> <span class="token parameter variable">-i</span> input_1.mp4 <span class="token parameter variable">-codec</span> copy output.mp4
ffmpeg <span class="token parameter variable">-fflags</span> +genpts <span class="token parameter variable">-i</span> output.mp4 <span class="token parameter variable">-c</span> copy output_fixed.mp4  // 这句用于修复duration
</code></pre>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f616340976884c00a97e12282df1c606.png"/>
    </p>
    <p>
     注意！参数要作用在输入视频之前，否则可能像大模型给出的答案一样具有误导性
