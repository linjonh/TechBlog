       <p style="margin-left:0; margin-right:0">
        在
        <strong>
         R
        </strong>
        中，大小写是敏感的，也就是说 A 和 a 是不同的对象。
       </p>
       <ol style="margin-left:.375in">
        <li>
         <strong>
          对象命名规则
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           对象名不能以数字开头，但数字可以出现在对象名的中间或结尾。例如，a1 是合法的，但 1a 不是。
          </li>
          <li>
           在命名对象时，建议使用点（.）作为分隔符，这样可以提高可读性。例如：anova.result1。
          </li>
         </ul>
        </li>
        <li>
         <strong>
          示例：创建一个向量
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           你可以通过如下命令创建一个向量：
           <br/>
           a &lt;- c(10, 15, 21, 18)
           <br/>
           这个向量 a 包含了 4 个元素，数据类型为数值型（numeric），长度为 4。
          </li>
         </ul>
        </li>
        <li>
         <strong>
          命名注意事项
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           在命名对象时，要避免与 R 中的保留名称（关键字）冲突。以下是一些保留名称示例：
           <ul style="margin-left:.375in">
            <li>
             NA：缺失值
            </li>
            <li>
             NaN：非数值
            </li>
            <li>
             pi：圆周率常数
            </li>
            <li>
             LETTERS：包含所有大写字母的字符向量
            </li>
            <li>
             letters：包含所有小写字母的字符向量
            </li>
            <li>
             month.abb：月份的缩写
            </li>
            <li>
             month.name：月份的全名
            </li>
           </ul>
          </li>
         </ul>
        </li>
       </ol>
       <p style="margin-left:0; margin-right:0">
        避免将对象命名为这些保留名称，以免与 R 语言的内置功能冲突。
       </p>
       <p style="margin-left:0; margin-right:0">
        <span style="color:black">
        </span>
       </p>
       <p style="margin-left:0; margin-right:0">
        在
        <strong>
         R
        </strong>
        中，有几种基本的数据类型：
       </p>
       <ol style="margin-left:.375in">
        <li>
         <strong>
          数值型（Numeric）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           用于存储数字，包括整数和浮动点数。
          </li>
          <li>
           示例：100, 0, -4.335
          </li>
         </ul>
        </li>
        <li>
         <strong>
          字符型（Character）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           用于存储文本数据，通常用双引号或单引号括起来。
          </li>
          <li>
           示例："China"
          </li>
         </ul>
        </li>
        <li>
         <strong>
          逻辑型（Logical）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           用于存储逻辑值，即 TRUE 或 FALSE。
          </li>
          <li>
           示例：TRUE, FALSE
          </li>
         </ul>
        </li>
        <li>
         <strong>
          因子型（Factor）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           用于表示分类数据，也就是不同类别或水平的变量。因子型变量可以用来表示类别，如性别、地区等。
          </li>
          <li>
           示例：factor(c("Male", "Female", "Female", "Male"))
          </li>
          <li>
           可以用来输入患者的信息
          </li>
         </ul>
        </li>
        <li>
         <strong>
          复数型（Complex）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           用于存储复数数据，形式为 a + bi。
          </li>
          <li>
           示例：2 + 3i
          </li>
         </ul>
        </li>
       </ol>
       <ul style="margin-left:.375in">
        <li>
         <strong>
          向量（Vector）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           向量是同一类型元素的集合。向量是R中最基本的数据结构，可以包含数值、字符、逻辑值等。
          </li>
          <li>
           示例：
           <br/>
           v &lt;-
           <span style="color:red">
            c(1, 2, 3, 4)
           </span>
          </li>
         </ul>
        </li>
        <li>
         <strong>
          数组（Array）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           数组是多维的数据表，可以是
           <span style="color:red">
            任意维度
           </span>
           （维数 k）。数组的元素必须是相同的数据类型。
          </li>
          <li>
           示例：创建一个二维数组
           <br/>
           arr &lt;- array(1:6, dim = c(2, 3))
          </li>
         </ul>
        </li>
        <li>
         <strong>
          矩阵（Matrix）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           矩阵是数组的一个特例，具有
           <span style="color:red">
            2 个维度
           </span>
           （即行和列）。矩阵中的元素必须是同一数据类型。
          </li>
          <li>
           示例：
           <br/>
           mat &lt;- matrix(1:6, nrow = 2, ncol = 3)
          </li>
         </ul>
        </li>
        <li>
         <strong>
          列表（List）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           列表是一种可以包含不同类型元素的数据结构，可以包含
           <span style="color:red">
            向量、矩阵、数据框，甚至其他列表
           </span>
           。与向量不同，列表中的元素可以是不同类型。
          </li>
          <li>
           示例：
           <br/>
           lst &lt;- list(a = 1, b = "text", c = TRUE)
          </li>
         </ul>
        </li>
        <li>
         <strong>
          数据框（DataFrame）
         </strong>
         ：
         <ul style="margin-left:.375in">
          <li>
           数据框是由一个或多个向量或因子组成的二维结构，类似于数据库中的表格。数据框中的每列可以是不同的数据类型，但每列必须是
           <span style="color:red">
            等长的
           </span>
           。
          </li>
          <li>
           示例：
           <br/>
           df &lt;- data.frame(Name = c("Alice", "Bob"), Age = c(25, 30))
          </li>
         </ul>
        </li>
       </ul>

        外部数据读取：
       </p>
       <p style="margin-left:0; margin-right:0">
        <strong>
         1. 读取 .txt 和 .csv 文件
        </strong>
       </p>
       <p style="margin-left:0; margin-right:0">
        <strong>
         read.table() 函数：
        </strong>
       </p>
       <ul style="margin-left:.375in">
        <li>
         read.table() 是
         <strong>
          R
         </strong>
         中最常用的函数之一，用于读取由空格或制表符（Tab）分隔的文本文件。它的默认分隔符是
         <strong>
          空格
         </strong>
         或
         <strong>
          制表符
         </strong>
         （Tab），但可以通过参数 sep 来指定其他分隔符。
         <br/>
         示例：
         <br/>
         data &lt;- read.table("data.txt", header = TRUE, sep = "\t")
         <ul style="margin-left:.375in">
          <li>
           header = TRUE：表示文件的第一行是列名。
          </li>
          <li>
           sep = "\t"：表示数据之间是由制表符（Tab）分隔的。
           <br/>
           读取的结果会存储为数据框（data frame），data 变量就包含了从文件中读取的数据。
          </li>
         </ul>
        </li>
       </ul>
       <p style="margin-left:0; margin-right:0">
        <strong>
         read.csv() 函数：
        </strong>
       </p>
       <ul style="margin-left:.375in">
        <li>
         read.csv() 是
         <strong>
          read.table()
         </strong>
         的一种专门用于读取 CSV（逗号分隔值）文件的简化版本。它的默认分隔符是
         <strong>
          逗号
         </strong>
         （,）。
         <br/>
         示例：
         <br/>
         data &lt;- read.csv("data.csv", header = TRUE)
         <ul style="margin-left:.375in">
          <li>
           header = TRUE：表示文件的第一行是列名。
           <br/>
           读取的结果会同样存储为数据框。
          </li>
         </ul>
        </li>
       </ul>
       <p style="margin-left:0; margin-right:0">
        <strong>
         2. 其他文件格式的读取
        </strong>
       </p>
       <p style="margin-left:0; margin-right:0">
        R 也可以通过一些专门的程序包来读取其他格式的数据文件。例如：
       </p>
       <ul style="margin-left:.375in">
        <li>
         <strong>
          Excel 文件
         </strong>
         ：可以使用 readxl 或 openxlsx 包来读取 .xlsx 或 .xls 文件。
         <br/>
         library(readxl)
         <br/>
         data &lt;- read_excel("data.xlsx")
        </li>
        <li>
         <strong>
          SAS 文件
         </strong>
         ：可以使用 haven 包来读取 .sas7bdat 文件。
         <br/>
         library(haven)
         <br/>
         data &lt;- read_sas("data.sas7bdat")
        </li>
        <li>
         <strong>
          DBF 文件
         </strong>
         ：可以使用 foreign 包来读取 DBF 格式文件。
         <br/>
         library(foreign)
         <br/>
         data &lt;- read.dbf("data.dbf")
        </li>
        <li>
         <strong>
          Matlab 文件
         </strong>
         ：使用 R.matlab 包来读取 .mat 文件。
         <br/>
         library(R.matlab)
         <br/>
         data &lt;- readMat("data.mat")
        </li>
        <li>
         <strong>
          SPSS 文件
         </strong>
         ：使用 haven 包读取 .sav 文件。
         <br/>
         library(haven)
         <br/>
         data &lt;- read_spss("data.sav")
        </li>
        <li>
         <strong>
          Minitab 文件
         </strong>
         ：可以使用 foreign 包来读取 .mtw 文件。
         <br/>
         library(foreign)
         <br/>
         data &lt;- read.minitab("data.mtw")
        </li>
       </ul>
