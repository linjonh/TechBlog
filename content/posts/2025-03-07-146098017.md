 </div>

     <path d="M5,0 0,2.5 5,5z" id="raphael-marker-block" stroke-linecap="round" style="-webkit-tap-highlight-color: rgba(0, 0, 0, 0);">
     </path>
    </svg>
    <p>
     如果想让 Git
     <strong>
      忽略指定目录
     </strong>
     ，不进行更新或提交，可以使用
     <code>
      .gitignore
     </code>
     文件进行配置。
    </p>
    <hr/>
    <h3>
     <a id="__gitignore__5">
     </a>
     <strong>
      🛠 方法：使用
      <code>
       .gitignore
      </code>
      忽略目录
     </strong>
    </h3>
    <h4>
     <a id="1__gitignore__6">
     </a>
     <strong>
      1️⃣ 在仓库根目录创建
      <code>
       .gitignore
      </code>
      文件
     </strong>
    </h4>
    <p>
     如果你的项目目录下还没有
     <code>
      .gitignore
     </code>
     文件，可以新建一个：
    </p>
    <pre><code class="prism language-sh"><span class="token function">touch</span> .gitignore
</code></pre>
    <p>
     或者在 VS Code 中手动创建
     <code>
      .gitignore
     </code>
     文件。
    </p>
    <h4>
     <a id="2__gitignore__13">
     </a>
     <strong>
      2️⃣ 在
      <code>
       .gitignore
      </code>
      文件中添加要忽略的目录
     </strong>
    </h4>
    <p>
     例如，想要忽略
     <code>
      logs/
     </code>
     和
     <code>
      temp/
     </code>
     目录：
    </p>
    <pre><code class="prism language-plaintext">logs/
temp/
</code></pre>
    <p>
     <strong>
      ⚠️ 注意：
     </strong>
    </p>
    <ul>
     <li>
      斜杠
      <code>
       /
      </code>
      代表目录，例如
      <code>
       logs/
      </code>
      表示忽略
      <code>
       logs
      </code>
      目录及其所有文件。
     </li>
     <li>
      如果是单个文件，例如
      <code>
       config.json
      </code>
      ，则写：
      <pre><code class="prism language-plaintext">config.json
</code></pre>
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_3__gitignore_28">
     </a>
     <strong>
      🔄 3️⃣ 提交
      <code>
       .gitignore
      </code>
     </strong>
    </h3>
    <p>
     让 Git 识别忽略规则：
    </p>
    <pre><code class="prism language-sh"><span class="token function">git</span> <span class="token function">add</span> .gitignore
<span class="token function">git</span> commit <span class="token parameter variable">-m</span> <span class="token string">"Update .gitignore to exclude logs and temp directories"</span>
</code></pre>
    <hr/>
    <h3>
     <a id="_4__37">
     </a>
     <strong>
      🚨 4️⃣ 如果目录已经被跟踪，需先清除缓存
     </strong>
    </h3>
    <p>
     如果
     <code>
      logs/
     </code>
     或
     <code>
      temp/
     </code>
     目录之前已经被 Git 跟踪，即使你添加了
     <code>
      .gitignore
     </code>
     ，它们仍然会被 Git 识别。此时需要执行：
    </p>
    <pre><code class="prism language-sh"><span class="token function">git</span> <span class="token function">rm</span> <span class="token parameter variable">-r</span> <span class="token parameter variable">--cached</span> logs/ temp/
<span class="token function">git</span> commit <span class="token parameter variable">-m</span> <span class="token string">"Remove logs and temp from tracking"</span>
<span class="token function">git</span> push origin main  <span class="token comment"># 推送到远程</span>
</code></pre>
    <hr/>
    <h3>
     <a id="__gitignore__47">
     </a>
     <strong>
      📌 其他
      <code>
       .gitignore
      </code>
      示例
     </strong>
    </h3>
    <h4>
     <a id="_log__48">
     </a>
     <strong>
      忽略所有
      <code>
       .log
      </code>
      文件
     </strong>
    </h4>
    <pre><code class="prism language-plaintext">*.log
</code></pre>
    <h4>
     <a id="_node_modules_52">
     </a>
     <strong>
      忽略
      <code>
       node_modules
      </code>
      （前端项目）
     </strong>
    </h4>
    <pre><code class="prism language-plaintext">node_modules/
</code></pre>
    <h4>
     <a id="_build__dist__56">
     </a>
     <strong>
      忽略
      <code>
       build
      </code>
      和
      <code>
       dist
      </code>
      目录（编译产物）
     </strong>
    </h4>
    <pre><code class="prism language-plaintext">build/
dist/
</code></pre>
    <h4>
     <a id="_env__61">
     </a>
     <strong>
      忽略
      <code>
       .env
      </code>
      配置文件
     </strong>
    </h4>
    <pre><code class="prism language-plaintext">.env
</code></pre>
    <hr/>
    <h3>
     <a id="__68">
     </a>
     <strong>
      💡 结论
     </strong>
    </h3>
    <ol>
     <li>
      <strong>
       创建
       <code>
        .gitignore
       </code>
      </strong>
      并添加需要忽略的目录/文件。
     </li>
     <li>
      <strong>
       如果已经被 Git 跟踪，执行
       <code>
        git rm -r --cached
       </code>
       清除缓存
      </strong>
      。
     </li>
     <li>
      <strong>
       提交
       <code>
        .gitignore
       </code>
      </strong>
      并推送到远程仓库。
     </li>
    </ol>
