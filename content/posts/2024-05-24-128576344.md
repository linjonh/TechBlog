    <h3 id="blogTitle0">
     1. 前言
    </h3>
    <ul>
     <li>
      <p>
       笔者主要使用的编程语言是 Java。平时使用的 IDE 是 JetBrains 公司的 IntelliJ IDEA。有时候也会打开该公司旗下的 PyCharm、DataGrip、WebStorm 等工具。
      </p>
     </li>
     <li>
      <p>
       在 2021 年 11 月发布了 Fleet 的预览版。
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/f4acec0bb201cea4b25bbabd0e401cc8.png"/>
      </p>
     </li>
     <li>
      <p>
       可以通过
       <a href="https://www.jetbrains.com/zh-cn/fleet/" rel="nofollow" title="官网">
        官网
       </a>
       去申请。笔者也是第一时间就去申请了，不少朋友在一周内就收到了申请通过的回复。
      </p>
     </li>
     <li>
      <p>
       笔者在 2022 年六月中旬收到了申请通过的邮件，第一时间就去下载体验了。目前官网应该是关闭了申请通道了，大家可以关注他们的 JetBrains News 了解相关信息。
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/fe410e62a02ddf47b76a834fb340be69.png"/>
      </p>
     </li>
     <li>
      <p>
       可以通过 JetBraints ToolBox App 登录申请通过的账户下载 Fleet。
      </p>
     </li>
    </ul>
    <h3 id="blogTitle1">
     2. Fleet 的相关信息
    </h3>
    <p>
     官网介绍说是轻量级编辑器，笔者随便从 Github 上拉了一个【打飞机小游戏】项目下来，分别使用 IntelliJ IDEA 和 Fleet 打开该项目，其运行内存如下：
    </p>
    <p class="img-center">
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/d6696e5a75f393d5c280bfb3af6ca0ec.png"/>
    </p>
    <p>
     可以看出 IDEA 占用的内存明显高于 Fleet。其实这个也不难理解，毕竟 IDEA 里面自带了很多插件，就类似于一个载满货物的货车(IDEA)和空载的货车(Fleet)一样。据官网描述，正式版 Fleet 将支持 JetBraints 插件市场的插件。那么以后我们将可以按照自己个人需要定制属于自己的 Fleet。
    </p>
    <h3 id="blogTitle2">
     3. 基于 Intellij 的 Fleet
    </h3>
    <p>
     下面演示一下 Fleet 运行 Java 项目，和 IDEA 运行项目的差别：
    </p>
    <ul>
     <li>
      <p>
       IDEA
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/c7dd267e48c364d71ccada31220b5f60.png"/>
      </p>
      <p>
       查看一下其运行占用的内存，CPU占用率：
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/1cf2ef493268399cab991fe85a9f1af7.png"/>
      </p>
      <p>
       IDEA 启动【打飞机小游戏】占用的 CPU 是：0.9% + 9.9% = 10.8%
      </p>
      <p>
       占用的内存是：1.2GB + 183.2MB = 1228.8MB + 183.2MB = 1412MB
      </p>
     </li>
     <li>
      <p>
       Fleet
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0cbdd32407f558b0600b84447d413903.png"/>
      </p>
      <p>
       查看一下其运行占用的内存，CPU占用率：
      </p>
      <p class="img-center">
       <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/0679a3e29878a93ca3cb1b8766d0ca4f.png"/>
      </p>
      <p>
       Fleet 启动【打飞机小游戏】占用的 CPU 是：3.8% + 8.3% = 12.1%
      </p>
      <p>
       占用的内存是：823.8MB + 143.8MB = 967.6MB
      </p>
     </li>
    </ul>
    <p>
     这两者 CPU 的使用率基本是在同一个水平线上浮动，不过对于内存占用率上，IDEA 总体来说要比 Fleet 多 500MB 的内存。
    </p>
    <h3 id="blogTitle3">
     4. Fleet 多语言支持
    </h3>

    </p>
    <p>
     同一个 Fleet IDE 打开不同编程语言的项目，如 50 种编程语言的 hello world，可以不用更换 IDE。通过开启智能识别，辨别不同编程语言的项目

     5. Fleet 对于 Java 的支持

     Fleet 沿用了 IDEA 的很多编程快捷键，以及快速生成代码缩写等。

    <p class="img-center">

    <h3 id="blogTitle5">
     6. 总结
    </h3>
    <ul>
     <li>
      个人还是比较期待正式版发布，对于插件的支持。这样定制性强，开发者更能自定义适合自己的IDE。
     </li>
     <li>
      对于 Fleet，还能将后端放到远程机器上(Linux)，本机的 IDE 仅作为前端展示。对于大型项目、多人合作开发项目，也是一个很不错的解决方案。
     </li>
     <li>
      笔者在使用 JetBrains IDE 的几年里，对于其功能的强大，特别是IntelliJ IDEA，满分10分，我都要打到8.8分。同时该公司IDE也受到了广大开发者的好评。
     </li>
     <li>
      对于
      <a href="https://www.jetbrains.com/zh-cn/fleet/" rel="nofollow" title="Fleet">
       Fleet
      </a>
      这个站在巨人(IDE)肩膀上的宠儿，不知是否能与现在的
      <a href="https://code.visualstudio.com/" rel="nofollow" title="Visual Studio">
       Visual Studio
      </a>
      掰掰手腕。
     </li>
     <li>
      在网上很多博主也评过 Fleet，其对 git 的支持也是基本完善的，类似 IDEA，同时笔者也对此进行了测试，使用感受良好。
     </li>
    </ul>
