    <h2>
     <a id="1__2">
     </a>
     1. 问题重现
    </h2>
    <p>
     之前使用Pycharm 2024.1.1的时候，环境配置的Python 3.11.9，现在改成使用Pycharm 2020.2.2，结果Python解释器显示“unsupported python 3.1”，如下图：
    </p>
    <p>
     <img alt="unsupported Python 3.1" src="https://i-blog.csdnimg.cn/direct/e22adf2acdd44d1cac494768ada6cfd9.png"/>
    </p>
    <h2>
     <a id="2__8">
     </a>
     2. 原因分析
    </h2>
    <p>
     因为Pycharm 2020.2.2支持的Python最高版本就是
     <strong>
      Python 3.9
     </strong>
     。
    </p>
    <h2>
     <a id="3__10">
     </a>
     3. 解决方法
    </h2>
    <p>
     将环境的python版本改为3.9及以下就行。以conda install为例（pip install 也是一样的！）：
     <br/>
     <img alt="conda install" src="https://i-blog.csdnimg.cn/direct/fcd1eab1c0c94ce1b3a5783bb41f0fe8.png"/>
    </p>
    <p>
     Python 3.9 安装完毕后，Pycharm 2020.2.2里的Python 解释器就显示正常啦！
     <br/>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/04af6914ca2d4feebc1a4f0783b0ff82.png"/>
    </p>
