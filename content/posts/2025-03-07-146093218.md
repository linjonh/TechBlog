     </strong>

     <li>
      <p>
       <strong>
        ✅ 退出/进入全屏模式
       </strong>
       <br/>
       •
       <strong>
        浏览器等应用
       </strong>
       ：
       <code>
        ⌘ Command
       </code>
       +
       <code>
        Ctrl
       </code>
       +
       <code>
        F
       </code>
      </p>
     </li>
     <li>
      <p>
       <strong>

       +
       <code>
        W

       •
       <code>
        ⌘ Command
       </code>
       +
       <code>
        Shift
       </code>
       +
       <code>
        W
       </code>
       （关闭整个浏览器窗口）
      </p>

       </strong>
       <br/>
       •
       <code>
        ⌃ Control
       </code>
       +
       <code>
        ⌘ Command
       </code>
       +
       <code>
        Q
       </code>
      </p>

     <li>
      <p>
       <strong>
        🖥️ 打开终端（命令行）

       •
       <strong>
        直接启动
       </strong>
       ：
       <code>
        ⌘ Command
       </code>
       +
       <code>
        空格

        Terminal

       → 回车

       </strong>

     <strong>
      截图与录屏
     </strong>

    <ol start="6">
     <li>
      <strong>
       ✅ 截图快捷键
      </strong>
      <br/>
      •
      <strong>

      </strong>

       ⌘ Command

      +

       Shift
      </code>
      +
      <code>
       3
      </code>
      （保存到桌面）
      <br/>
      •
      <strong>
       区域截图
      </strong>
      ：
      <code>
       ⌘ Command
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       4
      </code>
      （拖动选择区域）
      <br/>
      •
      <strong>
       窗口截图
      </strong>
      ：
      <code>
       ⌘ Command
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       4
      </code>
      → 按空格键（点击目标窗口）

      • 💡
      <strong>

      <code>

       Shift

      <code>
       4

      ）

      •
      <strong>

      </strong>
      ：
      <code>
       ⌘ Command
      </code>
      +
      <code>
       Shift
      </code>
      +
      <code>
       5
      </code>
      → 选择录屏范围
     </li>

    <h3>
     <a id="_Terminal__50">
     </a>
     🖥️ Terminal 使用
    </h3>
    <hr/>
    <h5>
     <a id="_1__54">
     </a>
     🎨
     <strong>
      1. 终端界面调整
     </strong>
    </h5>
    <p>
     •
     <strong>
      ✅ 放大字体
     </strong>
     ：
     <code>
      Command (⌘) + +
     </code>
     <br/>
     •
     <strong>
      ✅ 缩小字体
     </strong>
     ：
     <code>
      Command (⌘) + -
     </code>
     <br/>
     •
     <strong>
      ✅ 恢复默认字体大小
     </strong>
     ：
     <code>
      Command (⌘) + 0
     </code>
     <br/>
     •
     <strong>
      🖥️ 全屏切换
     </strong>
     ：
     <code>
      Control (⌃) + Command (⌘) + F
     </code>
    </p>
    <hr/>
    <h5>
     <a id="_2__63">
     </a>
     ⚡️
     <strong>
      2. 进程控制
     </strong>
    </h5>
    <p>
     •
     <strong>
      ⛔️ 终止当前命令
     </strong>
     ：
     <code>
      Control (⌃) + C
     </code>
     <br/>
     💡
     <em>
      强制终止正在运行的前台进程
     </em>
     <br/>
     •
     <strong>
      ⏸️ 挂起当前命令
     </strong>
     ：
     <code>
      Control (⌃) + Z
     </code>
     <br/>
     💡
     <em>
      将进程暂停并放入后台，可通过
      <code>
       fg
      </code>
      恢复或
      <code>
       bg
      </code>
      后台运行
     </em>
     <br/>
     •
     <strong>
      🚪 退出当前 Shell
     </strong>
     ：
     <code>
      Control (⌃) + D
     </code>
     或输入
     <code>
      exit
     </code>
     <br/>
     💡
     <em>
      发送 EOF 信号，关闭终端标签/窗口
     </em>
    </p>
    <hr/>
    <h5>
     <a id="_3__74">
     </a>
     📺
     <strong>
      3. 屏幕操作
     </strong>
    </h5>
    <p>
     •
     <strong>
      🧹 清屏
     </strong>
     ：
     <code>
      Control (⌃) + L
     </code>
     或输入
     <code>
      clear
     </code>
     <br/>
     •
     <strong>
      🖱️ 滚动输出
     </strong>
     ：
     <br/>
     • 向上翻页：
     <code>
      Command (⌘) + ↑
     </code>
     <br/>
     • 向下翻页：
     <code>
      Command (⌘) + ↓
     </code>
     <br/>
     💡
     <em>
      部分工具如 iTerm2 支持
      <code>
       Shift + PageUp/PageDown
      </code>
     </em>
    </p>
    <hr/>
    <h5>
     <a id="_4__84">
     </a>
     📜
     <strong>
      4. 历史命令与编辑
     </strong>
    </h5>
    <p>
     •
     <strong>
      🔍 搜索历史命令
     </strong>
     ：
     <code>
      Control (⌃) + R
     </code>
     <br/>
     💡
     <em>
      输入关键词查找历史记录，按
      <code>
       Enter
      </code>
      执行
     </em>
     <br/>
     •
     <strong>
      ⚡️ 快速移动光标
     </strong>
     ：
     <br/>
     • 行首：
     <code>
      Control (⌃) + A
     </code>
     <br/>
     • 行尾：
     <code>
      Control (⌃) + E
     </code>
     <br/>
     •
     <strong>
      🗑️ 删除整行
     </strong>
     ：
     <code>
      Control (⌃) + U
     </code>
     <br/>
     •
     <strong>
      🧹 删除至行尾
     </strong>
     ：
     <code>
      Control (⌃) + K
     </code>
    </p>
    <hr/>
    <h5>
     <a id="_5__96">
     </a>
     📊
     <strong>
      5. 后台任务管理
     </strong>
    </h5>
    <p>
     •
     <strong>
      📋 查看后台任务
     </strong>
     ：输入
     <code>
      jobs
     </code>
     <br/>
     •
     <strong>
      ▶️ 恢复挂起的任务到前台
     </strong>
     ：
     <code>
      fg %n
     </code>
     （
     <code>
      n
     </code>
     为任务编号）
     <br/>
     •
     <strong>
      ⏯️ 后台继续运行挂起的任务
     </strong>
     ：
     <code>
      bg %n
     </code>
     <br/>
     •
     <strong>
      ⛔️ 终止后台任务
     </strong>
     ：
     <code>
      kill %n
     </code>
     或
     <code>
      kill &lt;PID&gt;
     </code>
    </p>
    <hr/>
    <h5>
     <a id="_6__105">
     </a>
     🔧
     <strong>
      6. 其他实用操作
     </strong>
    </h5>
    <p>
     •
     <strong>
      ⚠️ 强制终止进程
     </strong>
     ：
     <br/>
     • 前台进程：
     <code>
      Control (⌃) + \
     </code>
     （发送
     <code>
      SIGQUIT
     </code>
     ）
     <br/>
     • 通过 PID 终止：
     <code>
      kill -9 &lt;PID&gt;
     </code>
     <br/>
     •
     <strong>
      🔒 锁屏
     </strong>
     ：
     <code>
      Command (⌘) + Control (⌃) + Q
     </code>
     （需系统支持）
     <br/>
     •
     <strong>
      📑 多标签/窗口管理
     </strong>
     ：
     <br/>
     • 新建标签：
     <code>
      Command (⌘) + T
     </code>
     <br/>
     • 关闭标签：
     <code>
      Command (⌘) + W
     </code>
     <br/>
     • 切换标签：
     <code>
      Command (⌘) + 数字键
     </code>
     或
     <code>
      Control (⌃) + Tab
     </code>
    </p>
    <hr/>
    <h5>
     <a id="_7__118">
     </a>
     🌐
     <strong>
      7. 网络相关
     </strong>
    </h5>
    <p>
     •
     <strong>
      📥 快速下载文件
     </strong>
     ：
     <code>
      curl -O &lt;URL&gt;
     </code>
     <br/>
     •
     <strong>
      📶 测试网络连通性
     </strong>
     ：
     <code>
      ping &lt;IP或域名&gt;
     </code>
     <br/>
     •
     <strong>
      🔍 查看端口占用
     </strong>
     ：
     <code>
      lsof -i :&lt;端口号&gt;
     </code>
    </p>
    <hr/>
    <h5>
     <a id="__Linux__126">
     </a>
     ❗️
     <strong>
      与 Linux 的区别注意
     </strong>
    </h5>
    <ol>
     <li>
      📌 Mac 默认 Shell 为
      <code>
       zsh
      </code>
      （较新版本），配置文件为
      <code>
       ~/.zshrc
      </code>
      （Linux 常用
      <code>
       bash
      </code>
      和
      <code>
       ~/.bashrc
      </code>
      ）。
     </li>
     <li>
      📌 部分命令需通过
      <code>
       brew
      </code>
      安装（如
      <code>
       wget
      </code>
      、
      <code>
       tree
      </code>
      ）。
     </li>
     <li>
      📌 系统管理命令可能不同（如网络配置用
      <code>
       networksetup
      </code>
      ）。
     </li>
    </ol>

    <h3>
     <a id="_156">
     </a>
     <strong>
      🛠️
     </strong>
     关机和重启
    </h3>
    <p>
     🔧 终端命令方式
    </p>
    <ol>
     <li>
      立即关机
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">shutdown</span> <span class="token parameter variable">-h</span> now
<span class="token comment"># 或使用 AppleScript</span>
osascript <span class="token parameter variable">-e</span> <span class="token string">'tell app "System Events" to shut down'</span>
</code></pre>
    <ol start="2">
     <li>
      定时关机（示例：10分钟后）
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">shutdown</span> <span class="token parameter variable">-h</span> +10
</code></pre>
    <ol start="3">
     <li>
      立即重启
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">shutdown</span> <span class="token parameter variable">-r</span> now

<span class="token function">sudo</span> <span class="token function">reboot</span>
</code></pre>
    <ol start="4">
     <li>
      强制重启（系统无响应时）
     </li>
    </ol>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">reboot</span> <span class="token parameter variable">-q</span>
</code></pre>
