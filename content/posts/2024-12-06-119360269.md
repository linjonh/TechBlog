# Python 字母大小写转换

python把大写字母转小写字母使用lower 方法

例如

```
str_a = "PYTHON"
print(str_a.lower())
```

python把小写字母转大写字母使用lower 方法

```
str_b ="python"
print(str_b.upper())
```
