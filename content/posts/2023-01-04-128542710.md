    <h4>
     <a id="_14">
     </a>
     系统简介：
    </h4>
    <p>
     本系统使用RSA算法生成密钥对， RSA私钥用于用户身份认证；用户发送的数据将存储于IPFS， IPFS返回的CID（IPFS Hash）使用双方用户的RSA公钥加密后存储于区块链; 区块链部分使用Hyperledger Fabric,并用Hyperledger Explorer追踪交易
    </p>
    <h4>
     <a id="_17">
     </a>
     包含功能
    </h4>
    <ol>
     <li>
      基于Fabric v1.4.4 first-network，四个peer一个orderer节点，使用docker部署
     </li>
     <li>
      IPFS使用的是ipfs/kubo镜像，负责用户数据文件的存储，IPFS返回的CID存储于Fabric
     </li>
     <li>
      项目包含了Hyperledger Explorer（区块链浏览器），默认跟随脚本启动
     </li>
     <li>
      项目包含了tape对链码压测
     </li>
     <li>
      使用RSA公私钥鉴别用户身份（1024位）
     </li>
     <li>
      链码对传输记录进行存储，包含：发送者公钥、接收者公钥、文件在IPFS的加密CID（由发送者或接收者的公钥加密）、文件名、时间戳、Fabric交易id
     </li>
     <li>
      后端使用gin框架实现，前端使用Vue和Element ui实现
      <br/>
      使用go fabric sdk调用智能合约；使用go-ipfs-api上传与下载用户文件；使用uuid对用户的文件名（下载时）进行加密
     </li>
    </ol>
    <h4>
     <a id="_27">
     </a>
     安装步骤(默认是在本地虚拟机)
    </h4>
    <ol>
     <li>
      <p>
       安装ubuntu 20.04(或其他Linux发行版),docker,docker-compose,go1.19
       <br/>
       docker,docker-compose,go1.19安装方法请参考此文章：https://blog.csdn.net/qq_41575489/article/details/129129086
      </p>
     </li>
     <li>
      <p>
       向/etc/hosts 写入：
      </p>
      <pre><code class="prism language-bash"><span class="token number">127.0</span>.0.1 orderer.example.com
<span class="token number">127.0</span>.0.1 peer0.org1.example.com
<span class="token number">127.0</span>.0.1 peer1.org1.example.com
<span class="token number">127.0</span>.0.1 peer0.org2.example.com
<span class="token number">127.0</span>.0.1 peer1.org2.example.com
</code></pre>
     </li>
     <li>
      <p>
       项目在服务器上运行需要操作，如果是虚拟机则省略这步。
       <br/>
       修改以下两个文件中127.0.0.1 为服务器公网IP：
      </p>
      <pre><code class="prism language-bash">datashare/application/server/controller/controller.go
datashare/application/web/index.html
</code></pre>
     </li>
     <li>
      <p>
       启动区块链部分
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> blockchain
./start.sh
</code></pre>
     </li>
     <li>
      <p>
       启动前后端
      </p>
      <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> application/server
go run main.go
</code></pre>
     </li>
     <li>
      <p>
       如果是云服务器
       <br/>
       在防火墙放行9090和8080TCP端口
      </p>
     </li>
     <li>
      <p>
       打开网页
       <br/>
       ip：9090/web
      </p>
     </li>
     <li>
      <p>
       关闭项目或需要重启项目时清理现有项目：
      </p>
      <pre><code>cd blockchain
./stop.sh
在application/server运行后端时
使用键盘指令：ctrl+c
</code></pre>
     </li>
    </ol>
    <h4>
     <a id="tape_72">
     </a>
     tape测压命令：
    </h4>
    <p>
     在blockchain/tape中
    </p>
    <p>
     ./tape --config=config.yaml --number=100
    </p>

     <a id="_77">
     </a>

     </li>
     <li>
      提示密钥不对、服务器错误请检查是否修改好hosts（步骤2）
     </li>
