    <h4>
     <a id="3Calendar_113">
     </a>
     3.Calendar
    </h4>
    <h5>
     <a id="1_114">
     </a>
     1.概述
    </h5>
    <ul>
     <li>
      Calendar代表了系统当前时间的日历对象，可以单独修改、获取时间中的年，月，日
     </li>
     <li>
      细节：Calendar是一个抽象类，不能直接创建对象。
     </li>
     <li>
      获取对象的方法：public static Calendar getInstance() 获取当前时间的日历对象
     </li>
    </ul>
    <h5>
     <a id="2_118">
     </a>
     2.方法
    </h5>
    <p>
     <img alt="请添加图片描述" src="https://i-blog.csdnimg.cn/direct/6c9270360df744678cc622bc76630f03.jpeg">
      <br/>
      字段：属性，成员变量。比如：年，月，日
     </img>
    </p>
    <p>
     <strong>
      细节-----获取日历对象：
     </strong>
    </p>
    <ul>
     <li>
      底层原理：会根据系统的不同时区来获取不同的日历对象，默认表示当前的时间。
     </li>
     <li>
      把时间中的纪元，年，月，日，时，分，秒，星期，等等的都放到一个数组当中。
     </li>
     <li>
      月份：范围0~11，如果获取出来的是0，那么实际上是1月
     </li>
     <li>
      星期：在老外的眼里，星期日是一周中的第一天。
     </li>
    </ul>

      </strong>

     </li>
     <li>

     </li>
