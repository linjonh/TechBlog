    <h2>
     <a id="1Qt__3">
     </a>
     1.Qt 音频
    </h2>
    <ul>
     <li>
      在Qt中，⾳频主要是通过
      <code>
       QSound
      </code>
      类来实现
      <ul>
       <li>
        使⽤
        <code>
         QSound
        </code>
        类时，需要
        <strong>
         添加模块
        </strong>
        ：
        <code>
         multimedia
        </code>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       注意
      </strong>
      ：
      <code>
       QSound
      </code>
      类只⽀持播放
      <strong>
       wav
      </strong>
      格式的⾳频⽂件，所以
      <ul>
       <li>
        如果想要添加⾳频效果，那么⾸先需要将⾮wav格式的⾳频⽂件转换为wav格式
       </li>
      </ul>
     </li>
     <li>
      <strong>
       核心API
      </strong>
      ：
      <ul>
       <li>
        <code>
         play()
        </code>
        ：开始或继续播放当前源
       </li>
      </ul>
     </li>
     <li>
      <strong>
       示例
      </strong>
      ：
      <pre><code class="prism language-cpp">sound <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token function">QSound</span><span class="token punctuation">(</span><span class="token string">":/sound.wav"</span><span class="token punctuation">,</span> <span class="token keyword">this</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

2.Qt 视频
</h2>
<ul>
<li>
在 Qt 中，视频播放的功能主要是通过
<code>
QMediaPlayer
</code>
类和
<code>
QVideoWidget
</code>
类来实现
<ul>
<li>
在使⽤这两个类时要
<strong>
添加对应的模块
</strong>
<code>
multimedia
</code>
和
<code>
multimediawidgets
</code>
</li>
</ul>
</li>
<li>
<strong>
核心 API
</strong>
：
<ul>
<li>
<code>
setMedia()
</code>
：设置当前媒体源
</li>
<li>
<code>
setVideoOutput()
</code>
：将
<code>
QVideoWidget
</code>
视频输出附加到媒体播放器
<ul>
<li>
如果媒体播放器已经附加了视频输出，将更换⼀个新的
</li>
</ul>
</li>
</ul>
</li>
