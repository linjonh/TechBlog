     # 一、Rust的类型
    </h2>
    <h3>
     ## 1、mut
    </h3>
    <pre><code class="language-rust">let a = 123;
a = 2.55; // error
a = "abc"; // error
a= 456; // error</code></pre>
    <p>
     这是一件很奇葩的事情，我一个变量怎么就不能改变？看书里面是怎么写的：
    </p>
    <p>
     <img alt="" height="181" src="https://i-blog.csdnimg.cn/direct/e833a8b37d4d462285e0db8d187e6bb2.png" width="2141">
      <img alt="" height="318" src="https://i-blog.csdnimg.cn/direct/5878498449014bf39160209d1c054a98.png" width="2148"/>
     </img>
    </p>
    <p>
     解决方法：加个mut就好了：
    </p>
    <pre><code class="language-rust">fn main()
{
    let mut a = 123;
    a = 456; // right
}</code></pre>

       ## 4、静态变量
      </img>
     </img>
    </h3>
    <p>
     静态变量就是“全局变量”，这个RUST的静态变量既可以写到函数体外面，也可以写到函数体里面，其他的语言例如C++就只能写到函数体外面，因为在函数体里面是一个栈区了，要用到静态变量，那么就用static关键字了，rust中不允许直接修改静态变量的值。--》其实静态变量就是我们所熟知的随着进程的创建而生成的，随着进程的消亡而消亡的！
    </p>
