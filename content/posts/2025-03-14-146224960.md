    <h3>
     <a id="font_color0000FF_size4746font_3">
     </a>
     <font color="#0000FF" size="4">
      746.使用最小花费爬楼梯
     </font>
    </h3>
    <p>
     <strong>
      题目链接：
     </strong>
     <a href="https://leetcode.cn/problems/min-cost-climbing-stairs/" rel="nofollow">
      746.使用最小花费爬楼梯
     </a>
     <br/>
     <strong>
      题目叙述：
     </strong>
     给你一个整数数组
     <code>
      cost
     </code>
     ，其中
     <code>
      cost[i]
     </code>
     是从楼梯第
     <code>
      i
     </code>
     个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。
    </p>
    <p>
     你可以选择从下标为
     <code>
      0
     </code>
     或下标为
     <code>
      1
     </code>
     的台阶开始爬楼梯。
    </p>
    <p>
     请你计算并返回达到楼梯顶部的最低花费。
    </p>
    <p>
     <strong>
      示例 1：
     </strong>
    </p>
    <p>
     <strong>
      输入：
     </strong>
     <code>
      cost
     </code>
     = [10,15,20]
     <br/>
     <strong>
      输出：
     </strong>
     15
     <br/>
     <strong>
      解释：
     </strong>
     你将从下标为 1 的台阶开始。
    </p>
    <ul>
     <li>
      支付 15 ，向上爬两个台阶，到达楼梯顶部。
      <br/>
      总花费为 15 。
     </li>
    </ul>
    <p>
     <strong>
      示例 2：
     </strong>
    </p>
    <p>
     <strong>
      输入：
     </strong>
     <code>
      cost
     </code>
     = [1,100,1,1,1,100,1,1,100,1]
     <br/>
     <strong>
      输出：
     </strong>
     6
     <br/>
     <strong>
      解释：
     </strong>
     你将从下标为 0 的台阶开始。
    </p>
    <ul>
     <li>
      支付 1 ，向上爬两个台阶，到达下标为 2 的台阶。
     </li>
     <li>
      支付 1 ，向上爬两个台阶，到达下标为 4 的台阶。
     </li>
     <li>
      支付 1 ，向上爬两个台阶，到达下标为 6 的台阶。
     </li>
     <li>
      支付 1 ，向上爬一个台阶，到达下标为 7 的台阶。
     </li>
     <li>
      支付 1 ，向上爬两个台阶，到达下标为 9 的台阶。
     </li>
     <li>
      支付 1 ，向上爬一个台阶，到达楼梯顶部。
      <br/>
      总花费为 6 。
     </li>
    </ul>
    <p>
     <strong>
      提示：
     </strong>
    </p>
    <p>
     2 &lt;= cost.length &lt;= 1000
     <br/>
     0 &lt;= cost[i] &lt;= 999
    </p>
    <hr/>
    <p>
     <strong>
      解题思路：
     </strong>
     <br/>
     <strong>
      解法一：
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       状态表示
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a5973067df7a44c5b78f8afbb3c45e00.png">
       <br/>
       <code>
        dp[0]
       </code>
       表示爬到0位置的最小花费
       <br/>
       <code>
        dp[1]
       </code>
       表示爬到1位置的最小花费
       <br/>
       <code>
        dp[2]
       </code>
       表示爬到2位置的最小花费
       <br/>
       .
       <br/>
       .
       <br/>
       <code>
        dp[i]
       </code>
       表示爬到i位置的最小花费
      </img>
     </li>
     <li>
      <strong>
       状态转移方程
      </strong>
      <br/>
      <font color="#1BFE9B">
       用
       <code>
        i
       </code>
       之前或之后的位置的状态，推导出
       <code>
        dp[i]
       </code>
       的值
      </font>
      <br/>
      <code>
       dp[i]
      </code>
      表示到达
      <code>
       i
      </code>
      位置的最小花费
      <br/>
      要么到达
      <code>
       i-1
      </code>
      的位置一1步到达
      <code>
       i
      </code>
      位置，要么到达
      <code>
       i-2
      </code>
      的位置走两步到达
      <code>
       i
      </code>
      位置
      <br/>
      🍃
      <font color="#451BFE">
       根据最近的一步来划分问题
      </font>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/fd72ae86c3624c5c8201d11a77ac98bd.png">
       <br/>
       <font color="#FE1B81" size="5">
        <strong>
         dp[i] = min(dp[i - 1] + cost[i - 1],dp[i - 2] + cost[i - 2])
        </strong>
       </font>
      </img>
     </li>
     <li>
      <strong>
       初始化
      </strong>
      <br/>
      <font color="#1BFE9B">
       保证填表的时候不越界
      </font>
      <br/>
      <code>
       dp[0] = dp[1] = 0;
      </code>
     </li>
     <li>
      <strong>
       填表顺序
      </strong>
      <br/>
      从左往右
     </li>
     <li>
      <strong>
       返回值
      </strong>
      <br/>
      <code>
       dp[n]
      </code>
     </li>
    </ol>
    <p>
     <strong>
      代码实现：
     </strong>
    </p>

     <font color="#DC143C">
      <strong>
       时间复杂度：O(n)
      </strong>
     </font>
     <br/>
     <font color="#DC143C">
      <strong>
       空间复杂度：O(n)
      </strong>
     </font>
    </p>
    <p>
     <strong>
      解法二：
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       状态表示
      </strong>
      <br/>
      <code>
       dp[i]
      </code>
      表示：以
      <code>
       i
      </code>
      为起点到达楼顶时的最小花费
     </li>
     <li>
      <strong>
       状态转移方程
      </strong>
      <br/>
      分为两种情况：
      <br/>
      ➀支付
      <code>
       i
      </code>
      位置的费用后，从
      <code>
       i+1
      </code>
      的位置到终点
      <br/>
      ➁支付
      <code>
       i
      </code>
      位置的费用后，从
      <code>
       i+2
      </code>
      的位置到终点
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a2b830632d06490da396a0f9a2f08b15.png">
       <br/>
       <font color="#FE1B81" size="5">
        <strong>
         dp[i] = min(dp[i + 1] + cost[ i ],dp[i + 2] + cost[ i ])
        </strong>
       </font>
      </img>
     </li>
     <li>
      <strong>
       初始化
      </strong>
      <br/>
      若从
      <code>
       n-1
      </code>
      位置出发到达楼顶只需支付
      <code>
       n-1
      </code>
      位置的费用
      <br/>
      若从
      <code>
       n-2
      </code>
      位置出发到达楼顶只需支付
      <code>
       n-2
      </code>
      位置的费用
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a00d8eedc78d4362a531ceed189a6771.png">
       <br/>
       仅需初始化最后两个位置
       <br/>
       <code>
        dp[n-1] = cost[n-1],dp[n-2] = cost[n-2]
       </code>
      </img>
     </li>
     <li>
      <strong>
       填表顺序
      </strong>
      <br/>
      从右往左
     </li>
     <li>
      <strong>
       返回值
      </strong>
      <br/>
      <code>
       min(dp[0],dp[1])
      </code>
     </li>
    </ol>
