# linux ubuntu 玩游戏

```
1.软件中心安装wine，玩CS，魔兽什么的没问题。速度也不见得慢，至少我不觉有变慢。
2.软件中心也有不少游戏，nexiuz（FPS），SuperTuxKart(RACE)，Alien Arena(FPS)，Glest(Strategy)等都还是挺不错的。
3.google 一下play deb，linux上游戏其实没有想象里的少，play deb里面有“Click here to learn how to install games from PlayDeb”的链接，安装其中链接的一个“playdeb package”，然后以后安装游戏就可以直接在play deb点击“install this now”开始安装。

如果以上三个可以满足你的游戏欲望的话，我还真不建议安装 windows。我也没安。为了玩游戏重启系统，或者启动虚拟机更麻烦。
```

**下载安装 wine**

什么是 wine 呢？它是一款 Linux 系统平台下的模拟器软件，用来让 Windows 系统下的软件在 Linux 系统下稳定运行。wine 的更新速度很快，目前对 DirectX9 系列支持的还算不错，也就是说，在 Linux 下顺畅运行 dota 是没有问题了。wine 的安装方法有几种，这里针对新手介绍 Ubuntu 下的安装步骤。
