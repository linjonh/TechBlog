    <p>
     IBM Q Experience 是最早的量子计算开源云平台。
     <strong>
      只要注册账号即可使用。
     </strong>
     <br/>
     支持使用物理量子计算机和模拟两种模式。
    </p>
    <p>
     IBM Q官网:
     <a href="https://quantum-computing.ibm.com" rel="nofollow">
      https://quantum-computing.ibm.com
     </a>
     <br/>
     Qiskit的官网文档:
     <a href="https://qiskit.org/documentation/install.html" rel="nofollow">
      https://qiskit.org/documentation/install.html
     </a>
    </p>
    <ol>
     <li>
      <p>
       登陆
       <br/>
       可以自己注册账号，如果注册时有些规则无法通过，也可以使用第三方账号登陆。
       <br/>
       支持:IMB Id, Linkedin, Github, Google, Twitter 5中第三方账号，我在测试用使用Github第三方账号顺利登陆。
       <br/>
       <img alt="IBM Q 登陆" height="300" src="https://img-blog.csdn.net/20180309153629774?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbG9uZ2pp/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="IBM Q 登陆" width="400"/>
      </p>
     </li>
     <li>
      <p>
       算法设计
       <br/>
       登陆成功后，点击Composer导航，即进入设计页面。把右侧Gates的逻辑门组距拖放到左侧线路图上，即可完成自己的量子算法设计。完成设计后，点击页面的 Run 按钮，即可在物理机器上运行算法。

      </p>
     </li>
