    <h2>
     二.sqlite3函数
    </h2>
    <h4>
     【1】sqlite3_exec
    </h4>
    <p>
     int sqlite3_exec(
     <br/>
     sqlite3* db,               // 数据库连接句柄
     <br/>
     const char* sql,           // 要执行的 SQL 语句
     <br/>
     int (*callback)(void*, int, char**, char**), // 回调函数
     <br/>
     void* arg,                 // 回调函数的参数
     <br/>
     char** err_msg             // 错误信息
     <br/>
     );
    </p>
    <p>
    </p>
    <p>
     参数说明
    </p>
    <ol>
     <li>
      <p>
       <strong>
        <code>
         db
        </code>
       </strong>
       ：指向 SQLite 数据库连接的指针。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         sql
        </code>
       </strong>
       ：要执行的 SQL 语句。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         callback
        </code>
       </strong>
       ：回调函数的指针。如果为
       <code>
        NULL
       </code>
       ，则不会处理查询结果。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         arg
        </code>
       </strong>
       ：传递给回调函数的用户自定义参数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         err_msg
        </code>
       </strong>
       ：如果执行失败，会返回错误信息。
      </p>
     </li>
    </ol>
    <p>
     <img alt="" height="612" src="https://i-blog.csdnimg.cn/direct/75403cc2e7ac44f699bed540a73fa7ca.png" width="777"/>
    </p>
    <h4>
     【2】回调函数原型
    </h4>
    <p>
     int callback(void* arg, int argc, char** argv, char** azColName);
    </p>
    <ul>
     <li>
      <p>
       <strong>
        <code>
         arg
        </code>
       </strong>
       ：从
       <code>
        sqlite3_exec
       </code>
       传入的用户自定义参数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         argc
        </code>
       </strong>
       ：查询结果的列数。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         argv
        </code>
       </strong>
       ：查询结果的每一列的值。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         azColName
        </code>
       </strong>
       ：查询结果的每一列的列名。
      </p>
     </li>
    </ul>
    <p>
     回调函数的作用
    </p>
    <p>
     回调函数会在每次从查询结果中获取一行数据时被调用。如果回调函数返回非零值，
     <code>
      sqlite3_exec
     </code>
     会停止执行，并返回错误。
    </p>
    <p>
     <img alt="" height="238" src="https://i-blog.csdnimg.cn/direct/fba913e85f9a46499ecd7fe9e4f75c4c.png" width="712"/>

    <h3>

    </h3>
    <p>
     【 项目 】
    </p>
    <p>
     在线商城项目要求:
    </p>
    <p>
     1.用户可在Web浏览器中输入IP+PORT访问服务器主页
    </p>
    <p>
     2.用户可在主页完成用户注册,用户注销及登录功能
    </p>
    <p>
     3.用户登录后，可在登录后页面中输入搜索商品,并获得包含
    </p>
    <p>
     搜索关键字的所有商品信息(以列表显示) //排版布局
    </p>
    <p>
     4.用户可以通过点击商品信息查看商品详细信息(价格、折扣价格、
    </p>
    <p>
     剩余数量等信息)  //动态生成网页
    </p>
    <p>
     5.该界面共分为4级
    </p>
    <p>
     1.一级登录界面
    </p>
    <p>
     2.二级搜索界面/注册界面/注销界面
    </p>
    <p>
     3.三级商品列表界面
    </p>
    <p>
     4.四级商品详细信息界面
    </p>
