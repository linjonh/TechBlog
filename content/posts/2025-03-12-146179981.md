    <p>
     在前两篇文章中，我们已经使用 AI 生成了
     <strong>
      Java API
     </strong>
     ，并创建了一个简单的
     <strong>
      HTML + JavaScript 网页
     </strong>
     ，让用户可以与 AI 机器人聊天。但如果我们想要一个更美观、更专业的交互界面，该怎么办呢？🤔
    </p>
    <p>
     本篇文章，我们将利用
     <strong>
      Vue 3 + Element Plus
     </strong>
     ，快速搭建一个
     <strong>
      现代化 AI 聊天机器人网页
     </strong>
     ，让交互体验更加流畅！🚀
    </p>
    <hr/>
    <h3>
     <strong>
      一、准备 Vue 3 项目环境
     </strong>
    </h3>
    <p>
     首先，我们需要
     <strong>
      创建一个 Vue 3 项目
     </strong>
     ，并安装 Element Plus 组件库。
    </p>
    <h4>
     <strong>
      1. 创建 Vue 3 项目
     </strong>
    </h4>
    <p>
     在终端运行以下命令（确保已安装
     <code>
      node.js
     </code>
     ）：
    </p>
    <pre><code>npm create vite@latest ai-chat --template vue
cd ai-chat
npm install

</code></pre>
    <h4>
     <strong>
      2. 安装 Element Plus
     </strong>
    </h4>
    <pre><code>npm install element-plus
npm install axios</code></pre>
    <h4>
     <strong>
      3. 启动开发服务器
     </strong>
    </h4>
    <pre><code>npm run dev
</code></pre>
    <p>
     现在，我们已经创建好了 Vue 3 项目，并成功安装了 Element Plus。🎉
    </p>
    <hr/>
    <h3>
     <strong>
      二、让 DeepSeek 生成 Vue 代码
     </strong>
    </h3>
    <p>
     💬
     <strong>
      示例指令
     </strong>
     ：
    </p>
    <blockquote>
     <p>
      “请用 Vue 3 + Element Plus 实现一个 AI 聊天页面，支持用户输入、发送按钮、AI 回复，并调用 API（
      <a href="http://localhost:8080/api/chat%EF%BC%89%E8%8E%B7%E5%8F%96%E5%9B%9E%E7%AD%94%E3%80%82%E2%80%9D" rel="nofollow" title="http://localhost:8080/api/chat）获取回答。”">
       http://localhost:8080/api/chat）获取回答。”
      </a>
     </p>
    </blockquote>
    <p>
     DeepSeek 可能会返回如下代码：
    </p>
    <h4>
     <strong>
      1. 修改
     </strong>
     <code>
      <strong>
       App.vue
      </strong>
     </code>
    </h4>
    <p>
     打开
     <code>
      src/App.vue
     </code>
     ，替换为以下代码：
    </p>
    <pre><code>&lt;template&gt;
  &lt;div class="chat-container"&gt;
    &lt;h2&gt;AI 机器人聊天&lt;/h2&gt;
    &lt;el-card class="chat-box"&gt;
      &lt;div v-for="(message, index) in messages" :key="index" class="chat-message"&gt;
        &lt;p&gt;&lt;strong&gt;{<!-- -->{ message.sender }}：&lt;/strong&gt; {<!-- -->{ message.text }}&lt;/p&gt;
      &lt;/div&gt;
    &lt;/el-card&gt;
    
    &lt;el-input v-model="userInput" placeholder="请输入您的问题..." @keyup.enter="sendMessage" /&gt;
    &lt;el-button type="primary" @click="sendMessage"&gt;发送&lt;/el-button&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script setup&gt;
import { ref } from "vue";
import axios from "axios";

const userInput = ref("");
const messages = ref([]);

const sendMessage = async () =&gt; {
  if (!userInput.value.trim()) return;

  messages.value.push({ sender: "你", text: userInput.value });

  try {
    const response = await axios.post("http://localhost:8080/api/chat", {
      message: userInput.value,
    });

    messages.value.push({ sender: "AI", text: response.data });
  } catch (error) {
    messages.value.push({ sender: "AI", text: "出错了，请稍后再试！" });
  }

  userInput.value = "";
};
&lt;/script&gt;

&lt;style&gt;
.chat-container {
  max-width: 600px;
  margin: 50px auto;
  text-align: center;
}
.chat-box {
  max-height: 400px;
  overflow-y: auto;
  padding: 10px;
  margin-bottom: 10px;
}
.chat-message {
  text-align: left;
  margin: 5px 0;
}
&lt;/style&gt;
</code></pre>
    <p>
     🔹
     <strong>
      代码解析
     </strong>
     ：
     <br/>
     ✅
     <code>
      el-card
     </code>
     作为聊天窗口，存放 AI 对话
     <br/>
     ✅
     <code>
      el-input
     </code>
     让用户输入问题
     <br/>
     ✅
     <code>
      el-button
     </code>
     触发
     <code>
      sendMessage()
     </code>
     ，调用后端 API
     <br/>
     ✅
     <code>
      axios.post()
     </code>
     发送 HTTP 请求到后端
     <br/>
     ✅
     <code>
      messages
     </code>
     存储聊天记录，动态渲染对话
    </p>

     <strong>
      三、运行 AI 聊天网站
     </strong>
    </h3>
    <h4>
     <strong>

      Spring Boot

      http://localhost:8080/api/chat

    <h4>
     <strong>
      2. 启动 Vue 项目

    </h4>
    <p>
     在 Vue 目录下运行：
    </p>
    <pre><code>npm run dev
</code></pre>
    <p>
     然后在浏览器打开：
     <br/>
     👉
     <code>
      http://localhost:5173/
     </code>

     你就可以在网页上与 AI 聊天了！🎉🎉

    <h3>
     <strong>
      四、优化交互体验
     </strong>
    </h3>
    <p>
     目前，我们的 AI 机器人已经能聊天，但如果
     <strong>
      页面更炫酷、体验更流畅

    </p>
    <h4>
     <strong>
      1. 让 AI 生成更好的样式
     </strong>
    </h4>
    <p>
     💬
     <strong>
      示例指令
     </strong>
     ：
    </p>
    <blockquote>
     <p>
      “请优化 Vue 3 AI 机器人聊天页面的 UI，使其更现代化，并加上 loading 动画。”
     </p>
    </blockquote>
    <p>
     DeepSeek 可能会返回如下优化代码：
    </p>
    <pre><code>&lt;template&gt;
  &lt;div class="chat-container"&gt;
    &lt;h2&gt;🌟 AI 机器人聊天 🌟&lt;/h2&gt;
    
    &lt;el-card class="chat-box"&gt;
      &lt;div v-for="(message, index) in messages" :key="index" class="chat-message"&gt;
        &lt;p :class="{ 'user-message': message.sender === '你', 'ai-message': message.sender === 'AI' }"&gt;
          &lt;strong&gt;{<!-- -->{ message.sender }}：&lt;/strong&gt; {<!-- -->{ message.text }}
        &lt;/p&gt;
      &lt;/div&gt;
      &lt;p v-if="loading" class="loading"&gt;AI 正在思考...&lt;/p&gt;
    &lt;/el-card&gt;
    
    &lt;el-input v-model="userInput" placeholder="请输入您的问题..." @keyup.enter="sendMessage" /&gt;
    &lt;el-button type="primary" :disabled="loading" @click="sendMessage"&gt;
      {<!-- -->{ loading ? "处理中..." : "发送" }}
    &lt;/el-button&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script setup&gt;
import { ref } from "vue";
import axios from "axios";

const userInput = ref("");
const messages = ref([]);
const loading = ref(false);

const sendMessage = async () =&gt; {
  if (!userInput.value.trim() || loading.value) return;

  messages.value.push({ sender: "你", text: userInput.value });
  loading.value = true;

  try {
    const response = await axios.post("http://localhost:8080/api/chat", {
      message: userInput.value,
    });

    messages.value.push({ sender: "AI", text: response.data });
  } catch (error) {
    messages.value.push({ sender: "AI", text: "出错了，请稍后再试！" });
  } finally {
    loading.value = false;
  }

  userInput.value = "";
};
&lt;/script&gt;

&lt;style&gt;
.chat-container {
  max-width: 600px;
  margin: 50px auto;
  text-align: center;
}
.chat-box {
  max-height: 400px;
  overflow-y: auto;
  padding: 10px;
  margin-bottom: 10px;
  background: #f9f9f9;
}
.chat-message {
  text-align: left;
  margin: 10px 0;
}
.user-message {
  color: blue;
}
.ai-message {
  color: green;
}
.loading {
  color: gray;
  font-style: italic;
}
&lt;/style&gt;
</code></pre>
    <p>
     🔹
     <strong>
      优化点
     </strong>
     ：
     <br/>
     ✅
     <strong>
      添加 Loading 提示
     </strong>
     ，让用户知道 AI 在思考
     <br/>
     ✅
     <strong>
      区分用户和 AI 颜色
     </strong>
     ，让对话更清晰
     <br/>
     ✅
     <strong>
      美化聊天窗口
     </strong>
     ，更符合现代 UI 设计
