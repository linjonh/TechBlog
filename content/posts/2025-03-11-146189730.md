    <h4>
     概率质量函数（Probability Mass Function, PMF）详解
    </h4>
    <h5>
     <strong>
      1. 概述
     </strong>
    </h5>
    <p>
     概率质量函数（PMF, Probability Mass Function）用于描述
     <strong>
      离散随机变量
     </strong>
     的概率分布。简单来说，它给出了随机变量取某个具体值的概率。PMF 适用于那些
     <strong>
      只能取有限个或可数无穷多个离散值
     </strong>
     的变量，例如抛硬币的结果（正面或反面）、掷骰子的点数（1-6）等。
    </p>
    <h5>
     <strong>
      2. 概率质量函数的数学定义
     </strong>
    </h5>
    <p>
     设 XXX 是一个离散随机变量，取值范围为
     <img alt="x_1, x_2, \dots, x_n" class="mathcode" src="https://latex.csdn.net/eq?x_1%2C%20x_2%2C%20%5Cdots%2C%20x_n">
      ，则概率质量函数定义如下：
     </img>
    </p>
    <p style="text-align:center">
     <img alt="P(X = x) = f(x)" class="mathcode" src="https://latex.csdn.net/eq?P%28X%20%3D%20x%29%20%3D%20f%28x%29"/>
    </p>
    <p>
     其中：
    </p>
    <ul>
     <li>
      P(X = x) 表示随机变量 X 取值为 x 的概率。
     </li>
     <li>
      f(x) 代表该值的概率质量函数。
     </li>
    </ul>
    <p>
     PMF 必须满足以下两个条件：
    </p>
    <ol>
     <li>
      <strong>
       非负性
      </strong>
      ：对于所有可能的 x，
      <img alt="P(X = x) \geq 0" class="mathcode" src="https://latex.csdn.net/eq?P%28X%20%3D%20x%29%20%5Cgeq%200">
       。
      </img>
     </li>
     <li>
      <strong>
       归一化性质
      </strong>
      （所有概率之和为 1）：
      <br/>
      <br/>
      <img alt="\sum_{x \in S} P(X = x) = 1" class="mathcode" src="https://latex.csdn.net/eq?%5Csum_%7Bx%20%5Cin%20S%7D%20P%28X%20%3D%20x%29%20%3D%201">
       <br/>
       <br/>
       其中 S 是随机变量 X 的所有可能取值集合。
      </img>
     </li>
    </ol>
    <h5>
     <strong>
      3. 概率质量函数的直观解释
     </strong>
    </h5>
    <p>
     概率质量函数可以用
     <strong>
      柱状图
     </strong>
     或
     <strong>
      条形图
     </strong>
     来表示。例如，在图片中，我们看到一个离散随机变量的概率分布，横轴是
     <strong>
      可能取值
     </strong>
     ，纵轴是
     <strong>
      概率
     </strong>
     ，用不同高度的柱子来表示每个取值的概率大小。
    </p>
    <h5>
     <strong>
      4. 例子
     </strong>
    </h5>
    <h6>
     <strong>
      (1) 掷骰子的 PMF
     </strong>
    </h6>
    <p>
     假设掷一个公正的 6 面骰子，随机变量 X 表示掷出的点数。由于骰子是公正的，每个点数的概率相等，PMF 如下：
    </p>
    <p style="text-align:center">
     <img alt="" height="109" src="https://i-blog.csdnimg.cn/direct/5f4171bb29d84e0fbcc2bf64a85af92f.png" width="402"/>
    </p>
    <p>
     如果用柱状图表示，每个点数x（1 到 6）都会对应一个相等高度的柱子。
    </p>
    <h6>
     <strong>
      (2) 抛硬币的 PMF
     </strong>
    </h6>
    <p>
     设 X 为一次抛硬币时出现正面的次数，则：
    </p>
    <p class="img-center">
     <img alt="" height="104" src="https://i-blog.csdnimg.cn/direct/8dbc7dabbd3d43efb4859ef6267c3cce.png" width="383"/>
    </p>
    <p>
     ​
    </p>
    <p>
     这也是一个典型的离散概率分布。
    </p>
    <h5>
     <strong>
      5. PMF 与其他概率分布的关系
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       概率密度函数（PDF, Probability Density Function）
      </strong>
      ：PMF 适用于
      <strong>
       离散随机变量
      </strong>
      ，而 PDF 适用于
      <strong>
       连续随机变量
      </strong>
      。对于连续变量，例如人的身高或温度，我们不能用 PMF，而需要用概率密度函数。
     </li>
     <li>
      <strong>
       累积分布函数（CDF, Cumulative Distribution Function）
      </strong>
      ：PMF 只给出某个具体取值的概率，而 CDF 是
      <strong>
       从小到大累积概率
      </strong>
      ，定义为：
      <br/>
      <br/>
      <img alt="F(x) = P(X \leq x) = \sum_{t \leq x} P(X = t)" class="mathcode" src="https://latex.csdn.net/eq?F%28x%29%20%3D%20P%28X%20%5Cleq%20x%29%20%3D%20%5Csum_%7Bt%20%5Cleq%20x%7D%20P%28X%20%3D%20t%29">
       <br/>
       <br/>
       CDF 适用于 PMF 和 PDF，但 PMF 直接给出每个取值的独立概率，而 CDF 累加这些概率。
      </img>
     </li>
    </ul>
    <h5>
     <strong>
      6. PMF 的实际应用
     </strong>
    </h5>
    <p>
     概率质量函数在很多现实场景中都有重要作用：
    </p>
    <ol>
     <li>
      <strong>
       机器学习和数据科学
      </strong>
      ：离散数据建模，如分类问题、朴素贝叶斯分类器等。
     </li>
     <li>
      <strong>
       金融建模
      </strong>
      ：股票价格涨跌的离散状态概率。
     </li>
     <li>
      <strong>
       质量控制
      </strong>
      ：工厂产品的合格与不合格概率建模。
     </li>
     <li>
      <strong>
       自然语言处理（NLP）
      </strong>
      ：分析单词在文本中的出现概率。
     </li>
    </ol>
    <h5>
     <strong>
      7. 结论
     </strong>
    </h5>
    <p>
     概率质量函数是描述
     <strong>
      离散随机变量
     </strong>
     概率分布的重要工具，广泛应用于统计学、数据科学、金融等领域。理解 PMF 有助于更深入地研究概率论，并在实际问题中构建有效的概率模型。
    </p>
