    <h2>
     <a id="Vue__React__0">
     </a>
     Vue 常用的核心属性可以分为以下几类，并与 React 实现方式对比：
    </h2>
    <h3>
     <a id="1__1">
     </a>
     1. 核心属性
    </h3>
    <pre><code>// Vue 选项式 API 常用属性
export default {
  props: {},       // 类似 React 的 props
  data() { return {} }, // 类似 React 的 useState
  computed: {},    // 类似 React 的 useMemo
  methods: {},     // 类似 React 的函数声明
  watch: {},       // 类似 React 的 useEffect + 依赖项
  // 生命周期钩子
  created() {},    // 类似 React 的 useEffect(() =&gt; {}, [])
  mounted() {},    // 类似 React 的 useEffect(() =&gt; {}, [])
}
</code></pre>
    <h3>
     <a id="2__16">
     </a>
     2. 生命周期对比
    </h3>
    <pre><code>// Vue 生命周期          |  React 等效实现
//----------------------------------------------
beforeCreate() {}   =&gt; 无直接等效（可在自定义 Hook 中实现）
created() {}        =&gt; useEffect(() =&gt; {}, [])
beforeMount() {}    =&gt; useLayoutEffect(() =&gt; {}, [])
mounted() {}        =&gt; useEffect(() =&gt; {}, [])
beforeUpdate() {}   =&gt; useLayoutEffect(() =&gt; {})
updated() {}        =&gt; useEffect(() =&gt; {})
beforeUnmount() {}  =&gt; useEffect(() =&gt; () =&gt; {})
unmounted() {}      =&gt; useEffect(() =&gt; () =&gt; {})
</code></pre>
    <h3>
     <a id="3__30">
     </a>
     3. 模板指令对应关系
    </h3>
    <pre><code>&lt;!-- Vue 模板指令 --&gt;      | &lt;!-- React 实现 --&gt;
&lt;组件 v-model="data"/&gt;  =&gt; &lt;组件 value={data} onChange={handleChange}&gt;
&lt;div v-if="show"&gt;       =&gt; {show &amp;&amp; &lt;div&gt;}
&lt;div v-for="item in list"&gt; =&gt; {list.map(item =&gt; &lt;div key={id}&gt;)}
&lt;div :class="{active}"&gt; =&gt; &lt;div className={`${active ? 'active' : ''}`}&gt;
&lt;div @click="handler"&gt; =&gt; &lt;div onClick={handler}&gt;
</code></pre>
    <h3>
     <a id="4__API__41">
     </a>
     4. 组合式 API 对比
    </h3>
    <pre><code>// Vue 3 组合式 API        |  React Hooks
//----------------------------------------------
const num = ref(0)       =&gt; const [num, setNum] = useState(0)
const double = computed()=&gt; const double = useMemo()
watch(num, () =&gt; {})     =&gt; useEffect(() =&gt; {}, [num])
onMounted(() =&gt; {})      =&gt; useEffect(() =&gt; {}, [])
</code></pre>
