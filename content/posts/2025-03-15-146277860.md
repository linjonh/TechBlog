      宝塔面板

     </strong>
     搭建一个博客（如 WordPress）。

     <strong>
      主要步骤包括：
     </strong>
    </p>

      </strong>
     </li>
     <li>

      </strong>
     </li>
     <li>
      <strong>

     </li>
     <li>
      <strong>
       安装 WordPress 并优化
      </strong>
     </li>
    </ol>

    <h3>
     <a id="_1__11">
     </a>
     <strong>
      📌 1. 安装宝塔面板
     </strong>
    </h3>
    <p>
     <a href="https://www.bt.cn/new/index.html" rel="nofollow">
      宝塔官网
     </a>
    </p>
    <p>
     <strong>
      在 Ubuntu 服务器上执行以下命令：
     </strong>
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 更新系统</span>
<span class="token function">sudo</span> <span class="token function">apt</span> update <span class="token operator">&amp;&amp;</span> <span class="token function">sudo</span> <span class="token function">apt</span> upgrade <span class="token parameter variable">-y</span>

<span class="token comment"># 下载并安装宝塔面板（适用于 Ubuntu）</span>
<span class="token function">wget</span> <span class="token parameter variable">-O</span> install.sh http://www.bt.cn/install/install-ubuntu_6.0.sh <span class="token operator">&amp;&amp;</span> <span class="token function">sudo</span> <span class="token function">bash</span> install.sh
</code></pre>
    <p>
     安装完成后，终端会显示
     <strong>
      宝塔面板的访问地址、账号和密码
     </strong>
     ，请妥善记录。
    </p>

     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       在浏览器访问
      </strong>
      <code>
       http://服务器IP:8888

      <strong>

      </strong>

    </ul>
    <p>
     <strong>
      安装必备环境：
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       进入 “软件商店”
      </strong>
     </li>
     <li>
      <strong>

      </strong>

       <li>

        </strong>

        （数据库）

        <strong>

       <li>

        </strong>

      </ul>

      <strong>

      ：300 秒

     <li>
      <strong>
       启用
       <code>
        fileinfo

      </strong>

        <code>

        </code>
       </li>
       <li>

        </strong>
        ：服务器 IP
       </li>
      </ul>
     </li>
     <li>
      <strong>
       添加
       <code>
        www
       </code>
       解析
      </strong>
      <ul>
       <li>
        <strong>
         主机记录
        </strong>
        ：
        <code>
         www
        </code>
       </li>
       <li>
        <strong>
         记录值
        </strong>
        ：服务器 IP

    <h3>
     <a id="_4__66">
     </a>
     <strong>
      📌 4. 配置反向代理隐藏宝塔面板端口
     </strong>
    </h3>
    <p>
     配置反向代理可以参考
     <a href="https://blog.csdn.net/m0_58648890/article/details/143808051">
      Nginx 上安装 SSL 证书并启用 HTTPS 访问
     </a>
     。如果已经在宝塔上安装了
     <code>
      Nginx
     </code>
     就不要安装单独的
     <code>
      Nginx
     </code>
     ，不然两个冲突了确实挺麻烦的。
    </p>
    <ol>
     <li>
      <strong>
       进入宝塔面板
      </strong>
      →
      <strong>
       设置
      </strong>
      →
      <strong>
       绑定域名
      </strong>
     </li>
     <li>
      <strong>
       添加一个域名
      </strong>
      ，例如
      <code>
       panel.example.com
      </code>
     </li>
     <li>
      <strong>
       进入 Nginx 配置
      </strong>
      ，添加以下反向代理：
     </li>
    </ol>
    <pre><code class="prism language-nginx">server {
            listen 443 ssl http2;
            server_name 域名;

            # SSL 证书路径（Certbot 自动生成）
            ssl_certificate /etc/letsencrypt/live/域名/fullchain.pem;
            ssl_certificate_key /etc/letsencrypt/live/域名/privkey.pem;
            include /etc/letsencrypt/options-ssl-nginx.conf;
            ssl_dhparam /etc/letsencrypt/ssl-dhparams.pem;

            # HSTS（严格 HTTPS ）
            add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;

            # 反向代理到本地 21525 端口
            location / {
                proxy_pass https://127.0.0.1:21525; # 如果后端是 HTTPS，改为 https://
                proxy_set_header Host $host;
                proxy_set_header X-Real-IP $remote_addr;
                proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
                proxy_set_header X-Forwarded-Proto $scheme;
                proxy_set_header X-Forwarded-Host $host;
                proxy_set_header X-Forwarded-Server $host;                                                                                                      
                # 超时配置
                proxy_connect_timeout 60s;
                proxy_read_timeout 60s;
                proxy_send_timeout 60s;

                # 处理 WebSocket（如果后端有 WebSocket）
                proxy_http_version 1.1;
                proxy_set_header Upgrade $http_upgrade;
                proxy_set_header Connection "Upgrade";
        }
}

# 自动重定向 HTTP 到 HTTPS
server {
        listen 80;
        server_name 域名;

        location / {
                return 301 https://$host$request_uri;
        }
}
</code></pre>
    <ol start="4">
     <li>
      <strong>
       重启 Nginx
      </strong>
      ，然后你可以通过
      <code>
       https://域名
      </code>
      访问宝塔面板，而无需暴露端口
      <code>
       端口
      </code>
      。
     </li>
    </ol>

    <h3>
     <a id="_5__123">
     </a>
     <strong>
      📌 5. 在宝塔面板中添加网站
     </strong>
    </h3>
    <ol>
     <li>
      <strong>
       进入 “网站管理”
      </strong>
      →
      <strong>
       点击 “添加站点”
      </strong>
     </li>
     <li>
      <strong>
       填写信息：
      </strong>
      <ul>
       <li>
        <strong>
         域名
        </strong>
        ：
        <code>
         example.com
        </code>
       </li>
       <li>
        <strong>
         根目录
        </strong>
        ：
        <code>
         /www/wwwroot/example.com
        </code>
       </li>
       <li>
        <strong>
         数据库
        </strong>
        ：
        <code>
         MySQL
        </code>
       </li>
       <li>
        <strong>
         PHP 版本
        </strong>
        ：
        <code>
         PHP 8.0+
        </code>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       点击 “创建”
      </strong>
     </li>
    </ol>
    <p>
     <strong>
      修改 Nginx 伪静态（适用于 WordPress）：
     </strong>
    </p>
    <pre><code class="prism language-nginx">location / {
    try_files $uri $uri/ /index.php?$args;
}
</code></pre>

     🔹 下载并解压 WordPress （手动安装：可选）
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">cd</span> /www/wwwroot/example.com

<span class="token function">unzip</span> latest.zip
