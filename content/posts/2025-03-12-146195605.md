    <h2>
     <a id="Homebrew_0">
     </a>
     一、安装Homebrew
    </h2>
    <p>
     <code>
      提示：在安装 nvm 时，如果使用 brew 方式安装，就要先安装 Homebrew
     </code>
    </p>
    <p>
     1、打开终端，输入以下指令（官网可获取最新命令）：
    </p>
    <ul>
     <li>
      国外镜像
     </li>
    </ul>
    <pre><code class="prism language-bash">/bin/bash <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <ul>
     <li>
      国内镜像
     </li>
    </ul>
    <pre><code class="prism language-bash">/bin/zsh <span class="token parameter variable">-c</span> <span class="token string">"<span class="token variable"><span class="token variable">$(</span><span class="token function">curl</span> <span class="token parameter variable">-fsSL</span> https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh<span class="token variable">)</span></span>"</span>
</code></pre>
    <p>
     2、检查是否安装成功，输入以下指令
    </p>
    <pre><code class="prism language-bash">brew <span class="token parameter variable">-v</span>
</code></pre>
    <h2>
     <a id="HomebrewNVM_17">
     </a>
     二、使用Homebrew安装NVM
    </h2>
    <p>
     1、安装完Homebrew后，在终端中执行以下命令来安装NVM
    </p>
    <pre><code class="prism language-bash">brew <span class="token function">install</span> nvm
</code></pre>
    <p>
     2、检查是否安装成功，输入以下指令
    </p>
    <pre><code class="prism language-bash">nvm <span class="token parameter variable">-v</span>
</code></pre>
    <p>
     <code>
      提示：如果遇到 zsh: command not found: nvm，请配置环境变量
     </code>
    </p>
    <p>
     3、配置环境变量：
     <br/>
     根据默认
     <code>
      shell
     </code>
     来决定操作文件
    </p>
    <blockquote>
     <p>
      .bash_profile‌：主要用于Bash Shell。Bash是许多类UNIX系统的默认Shell，特别是在较旧的Mac系统或以Bash为主要Shell的环境中常用‌
      <br/>
      ‌.zshrc‌：与Zsh Shell相关。Zsh是一种功能强大的Shell，近年来在Mac系统中逐渐流行，特别是从macOS Catalina版本开始，Zsh成为默认Shell‌
     </p>
    </blockquote>
    <p>
     macOS Catalina之前版本，编辑或创建
     <code>
      ~/.bash_profile
     </code>
     文件
     <br/>
     macOS Catalina及以后版本，编辑或创建
     <code>
      ~/.zshrc
     </code>
     文件
     <br/>
     在文件中添加以下配置环境变量的指令：
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">export</span> <span class="token assign-left variable">NVM_DIR</span><span class="token operator">=</span><span class="token string">"<span class="token environment constant">$HOME</span>/.nvm"</span>
<span class="token punctuation">[</span> <span class="token parameter variable">-s</span> <span class="token string">"/opt/homebrew/opt/nvm/nvm.sh"</span> <span class="token punctuation">]</span> <span class="token operator">&amp;&amp;</span> <span class="token punctuation">\</span>. <span class="token string">"/opt/homebrew/opt/nvm/nvm.sh"</span>  <span class="token comment"># This loads nvm</span>
<span class="token punctuation">[</span> <span class="token parameter variable">-s</span> <span class="token string">"/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"</span> <span class="token punctuation">]</span> <span class="token operator">&amp;&amp;</span> <span class="token punctuation">\</span>. <span class="token string">"/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"</span>  <span class="token comment"># This loads nvm bash_completion</span>
</code></pre>
    <ul>
     <li>
      <p>
       方式一：
       <br/>
       1️⃣ 打开
       <code>
        访达
       </code>
       ，依次点击屏幕左上方
       <code>
        菜单栏&gt;前往&gt;前往文件夹
       </code>
       ，输入英文
       <code>
        ~
       </code>
       回车，找到
       <code>
        .zshrc
       </code>
       或
       <code>
        .bash_profile
       </code>
       文件（根据shell决定文件）并打开；
       <br/>
       2️⃣ 如果没有找到文件，
       <code>
        Command + Shift + .
       </code>
       （
       <code>
        .
       </code>
       为句号键）可查看隐藏文件
       <br/>
       3️⃣ 打开后复制上面配置环境变量指令，新起一行粘贴，
       <code>
        command + s
       </code>
       保存
       <br/>
       4️⃣ 最后输入
       <code>
        source ~/.zshrc
       </code>
       或
       <code>
        source ~/.bash_profile
       </code>
       （根据shell决定文件）使文件生效，再用
       <code>
        nvm -v
       </code>
       检查即可
      </p>
     </li>
     <li>
      <p>
       方式二：
       <br/>
       1️⃣ 使用
       <code>
        vim
       </code>
       编辑，打开终端，输入
       <code>
        vim ~/.zshrc
       </code>
       或
       <code>
        vim ~/.bash_profile
       </code>
       指令（根据shell决定文件），按照提示操作，在新打开的文件中粘贴上面配置环境变量指令，保存
       <br/>
       2️⃣ 如不熟悉 vim 操作，请先了解相关操作
       <br/>
       3️⃣ 最后输入
       <code>
        source ~/.zshrc
       </code>
       或
       <code>
        source ~/.bash_profile
       </code>
       （根据shell决定文件）使文件生效，再用
       <code>
        nvm -v
       </code>
       检查即可
      </p>
     </li>
    </ul>
    <h2>
     <a id="_nvm__node__51">
     </a>
     三、使用 nvm 安装，切换 node 版本
    </h2>
    <p>
     1、安装（数字代表哪个版本，需要哪个输入哪个）
    </p>
    <pre><code class="prism language-bash">nvm <span class="token function">install</span> <span class="token number">20</span>
</code></pre>
    <p>
     2、查看已安装 node 版本
    </p>
    <pre><code class="prism language-bash">nvm list
</code></pre>
    <p>
     3、切换 node 版本
    </p>
    <pre><code class="prism language-bash">nvm use <span class="token number">20</span>
</code></pre>
    <p>
     4、查看当前使用的 node 版本
    </p>
    <pre><code class="prism language-bash"><span class="token function">node</span> <span class="token parameter variable">-v</span>
</code></pre>
