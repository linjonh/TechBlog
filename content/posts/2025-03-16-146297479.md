    <blockquote>
     <h2 id="1.%20ECMAScript%E5%9F%BA%E6%9C%AC%E8%AE%A4%E8%AF%86" name="1.%20ECMAScript%E5%9F%BA%E6%9C%AC%E8%AE%A4%E8%AF%86" style="background-color:transparent">
      1. ECMAScript基本认识
     </h2>
    </blockquote>
    <h3 id="1.1%20%E4%BB%80%E4%B9%88%E6%98%AF%20ECMAScript%3F" name="1.1%20%E4%BB%80%E4%B9%88%E6%98%AF%20ECMAScript%3F">
     1.1 什么是 ECMAScript?
    </h3>
    <p>
     官方定义:
    </p>
    <p>
     ECMAScript 是
     <strong>
      定义脚本语言的规范
     </strong>
     , 而 JavaScript 是
     <span style="color:#fe2c24">
      <strong>
       遵循 ECMAScript 规范
      </strong>
     </span>
     实现的一种编程语言
    </p>
    <p>
     通俗说法: ECMAScript 好比是一本书的目录, 而 JavaScript 是这本书的具体内容
    </p>
    <hr/>
    <h3 id="1.2%20ES6%20%E5%92%8C%20JavaScript%20%E6%9C%89%E4%BB%80%E4%B9%88%E5%8C%BA%E5%88%AB" name="1.2%20ES6%20%E5%92%8C%20JavaScript%20%E6%9C%89%E4%BB%80%E4%B9%88%E5%8C%BA%E5%88%AB">
     1.2 ES6 和 JavaScript 有什么区别
    </h3>
    <p>
     ES6(ECMAScript 2015)
    </p>
    <p>
     是ECMAScript规范的第六个版本,而JavaScript是基于ECMAScript规范实现的编程语言。
    </p>
    <p>
     ES6可以被看作是 JavaScript 的一个重要的版本更新
    </p>
    <p>
     ES6引入了"
     <strong>
      块级作用域
     </strong>
     ",使用 let 和 const 关键字来声明变量和常量,使变量的作用域清晰可控
    </p>
    <p>
     引入了"
     <strong>
      箭头函数
     </strong>
     ",箭头函数比传统的函数语法更简洁, 具有更少的冗余代码
    </p>
    <p>
     引入了"
     <strong>
      类(Class)
     </strong>
     "的概念,这是一种基于原型的面向对象编程的语法糖, 使用类可以更方便地创建和管理对象
    </p>
    <p>
     引入了"
     <strong>
      模板字符串
     </strong>
     ", 使用
     <strong>
      反引号(`)
     </strong>
     创建字符串可以方便地嵌入变量和表达式
    </p>
    <p>
     引入了"
     <strong>
      解构赋值
     </strong>
     ", 这是一种新的赋值语法,可以将数组或对象的属性直接赋值给变量
    </p>
    <p>
     引入了"
     <strong>
      函数参数默认值
     </strong>
     "
    </p>
    <p>
     引入了"
     <strong>
      Promise对象
     </strong>
     ", 简化了异步编程, 使其更可读和可维护
    </p>
    <p>
     引入了
     <strong>
      Set、Map
     </strong>
     等
    </p>
    <p>
     引入了"
     <strong>
      模块化
     </strong>
     "(ES Module)
    </p>
    <hr/>
    <blockquote>
     <h2 id="2.%20%E5%8F%98%E9%87%8F%E5%92%8C%E5%B8%B8%E9%87%8F" name="2.%20%E5%8F%98%E9%87%8F%E5%92%8C%E5%B8%B8%E9%87%8F">
      2. 变量和常量
     </h2>
    </blockquote>
    <h3 id="2.1%20%E5%8F%98%E9%87%8F%C2%A0" name="2.1%20%E5%8F%98%E9%87%8F%C2%A0">
     2.1 变量
    </h3>
    <p>
     变量(Variable) 是用于
     <strong>
      存储数据
     </strong>
     的名称(标识符), 变量
     <span style="color:#fe2c24">
      <strong>
       可以是任何类型
      </strong>
     </span>
     , 如 "数值、字符串" 等
    </p>
    <p>
     用
     <strong>
      let
     </strong>
     来声明变量
    </p>
    <p>
     注意：
    </p>
    <p>
     ES6
     <span style="color:#fe2c24">
      <strong>
       不区分整型和浮点型
      </strong>
     </span>
     , 所有
     <span style="color:#fe2c24">
      <strong>
       数字都使用 number类型
      </strong>
     </span>
     来表示
    </p>
    <pre><code>    &lt;script&gt;
        // 变量
        let name = 'zzz';
        console.log(name);  // zzz
        console.log(typeof name);  // 检测类型  string

        let money = 100;
        console.log('金额',money);
        console.log(typeof money);  // Number
    &lt;/script&gt;</code></pre>
    <hr/>
    <h3 id="2.2%20%E5%B8%B8%E9%87%8F" name="2.2%20%E5%B8%B8%E9%87%8F">
     2.2 常量
    </h3>
    <p>
     常量(Constant) 是一个固定的值, 在程序运行中常量的值保持不变
    </p>
    <p>
     用 const 声明常量
    </p>
    <p>
     常量通常用来
     <strong>
      表示不会改变
     </strong>
     的值, 如数学中的 π(圆周率) PI≈3.14159265359
    </p>
    <pre><code>    &lt;script&gt;
        // 常量
        const PI = 3.14;
        console.log(PI);
        
        // PI = 3.1415;
        // console.log(PI);  // 报错
    &lt;/script&gt;</code></pre>
    <hr/>
    <blockquote>
     <h2 id="3.%20%E6%95%B0%E6%8D%AE%E7%B1%BB%E5%9E%8B" name="3.%20%E6%95%B0%E6%8D%AE%E7%B1%BB%E5%9E%8B">
      3. 数据类型
     </h2>
    </blockquote>
    <h3 id="3.1%20%E5%AD%97%E7%AC%A6%E4%B8%B2%E3%80%81%E6%95%B0%E5%80%BC%E3%80%81%E5%B8%83%E5%B0%94" name="3.1%20%E5%AD%97%E7%AC%A6%E4%B8%B2%E3%80%81%E6%95%B0%E5%80%BC%E3%80%81%E5%B8%83%E5%B0%94">
     3.1 字符串、数值、布尔
    </h3>
    <h4 id="%E5%AD%97%E7%AC%A6%E4%B8%B2%E7%B1%BB%E5%9E%8B%20string" name="%E5%AD%97%E7%AC%A6%E4%B8%B2%E7%B1%BB%E5%9E%8B%20string">
     字符串类型 string
    </h4>
    <p>
     定义 字符串类型用于存储字符序列  也可以用 ' ' 表示
    </p>
    <pre><code>let name = "zzz" //姓名为zzz</code></pre>
    <h4 id="%E6%95%B0%E5%80%BC%E7%B1%BB%E5%9E%8B%20number" name="%E6%95%B0%E5%80%BC%E7%B1%BB%E5%9E%8B%20number">
     数值类型 number
    </h4>
    <p>
     定义 用于存储数字, 可以表示整数、浮点数
    </p>
    <pre><code>let balance = -100 //账户余额为-100元
let weight = 60.5 //体重为60.5公斤</code></pre>
    <h4 id="%E5%B8%83%E5%B0%94%E7%B1%BB%E5%9E%8B%20boolean" name="%E5%B8%83%E5%B0%94%E7%B1%BB%E5%9E%8B%20boolean">
     布尔类型 boolean
    </h4>
    <p>
     定义 布尔类型只能取两个值,true(真) 和 false(假)
    </p>
    <pre><code>let tv = true //电视 - true(开)</code></pre>
    <hr/>
    <h3 id="3.2%20%E5%AF%B9%E8%B1%A1Object" name="3.2%20%E5%AF%B9%E8%B1%A1Object">
     3.2 对象Object
    </h3>
    <p>
     定义 对象是一种
     <strong>
      复合的数据类型
     </strong>
     , 可以通过
     <strong>
      键值对
     </strong>
     的方式
     <strong>
      存储多个值
     </strong>
    </p>
    <pre><code>    &lt;script&gt;
        const obj = {
            uname : 'zzz',
            age : 18,
            weight : 60
        }
        // 用 对象.属性名 表示
        console.log('姓名',obj.uname);
        console.log('年龄',obj.age);
        console.log('体重',obj.weight);
        console.log(obj);
    &lt;/script&gt;</code></pre>
    <hr/>
    <h3 id="3.3%20map" name="3.3%20map">
     3.3 map
    </h3>
    <p>
     定义 map 是一种特殊的数据结构,用于
     <strong>
      存储键值对的
      <span style="color:#fe2c24">
       有序
      </span>
      集合
     </strong>
    </p>
    <pre><code>        // map
        let girl = new Map([
            ['name','zzz'],
            ['age',18]
        ])
        console.log(girl);
        console.log(typeof girl);</code></pre>
    <p class="img-center">
     <img alt="" height="146" src="https://i-blog.csdnimg.cn/direct/7ff9df85c1554e3fb6e358f846d0e2d8.png" width="374"/>
    </p>
    <p>
     map 是 new 出来的一个
     <span style="color:#fe2c24">
      <strong>
       对象
      </strong>
     </span>
    </p>
    <p>
     数据类型比如
     <strong>
      字符串就需要加引号
     </strong>
    </p>
    <p>
     最后输出的是
     <strong>
      箭头
     </strong>
     的表示
    </p>
    <p>
     <span style="color:#fe2c24">
      <strong>
       注意
      </strong>
     </span>
     ：
    </p>
    <p>
     Map相对于对象提供了更灵活、有序、高效的
     <strong>
      键值对
     </strong>
     存储和操作方式,当需要在大量键值对中快速查找或删除特定键值对时, Map比对象更高效
    </p>
    <p>
     Map提供了一些便捷的方法来操作键值对, 如: get()、set()、delete()
    </p>
    <p>
     Map提供了一些便捷的迭代方法, 如: forEach()、keys()、values()、entries()
    </p>
    <p>
     这些方法和操作后续会介绍。
    </p>
    <hr/>
    <h3 id="3.4%20set" name="3.4%20set">
     3.4 set
    </h3>
    <p>
     定义 Set 是一种特殊的数据结构, 用于存储
     <span style="color:#fe2c24">
      <strong>
       无序且唯一
      </strong>
     </span>
     的值的集合
    </p>
    <p>
     使用 new 关键字
    </p>
    <pre><code>        // set
        let number = new Set([1,2,3,2,1,7]);   // 包含6个整数的set集合
        console.log(number);   // 1,2,3,7 无序且唯一
        console.log(typeof number); //Object</code></pre>
    <p class="img-center">
     <img alt="" height="198" src="https://i-blog.csdnimg.cn/direct/dd3791e3c3784f7ca7ad8e1994fb7343.png" width="258"/>
    </p>
    <p>
     如果将其中的类型修改
    </p>
    <pre><code>        let number = new Set([1,2,3,'2',1,7]);
        console.log(number);   // 无序且唯一
        console.log(typeof number); //Object</code></pre>
    <p class="img-center">
     <img alt="" height="227" src="https://i-blog.csdnimg.cn/direct/b015c9b27ebe4d138ee9cbc6d56ae479.png" width="287"/>
    </p>
    <p>
     类型不同的也会认为是不同的，所以会打印出来
    </p>
    <hr/>
    <h3 id="3.5%20%E6%95%B0%E7%BB%84" name="3.5%20%E6%95%B0%E7%BB%84">
     3.5 数组
    </h3>
    <p>
     定义 数组是一种
     <span style="color:#fe2c24">
      <strong>
       有序集合
      </strong>
     </span>
     , 可以
     <strong>
      <span style="color:#fe2c24">
       包含不同类型
      </span>
     </strong>
     的元素，并且数组的
     <span style="color:#fe2c24">
      <strong>
       长度是可变
      </strong>
     </span>
     的
    </p>
    <pre><code class="hljs">        // 数组
        let arr = ['a','b','c','b','c',1,2];
        console.log(arr);  // 不唯一、可以不同类型
        console.log(typeof arr);</code></pre>
    <p class="img-center">
     <img alt="" height="197" src="https://i-blog.csdnimg.cn/direct/9ea712dc62274dbcb26ea1feab254585.png" width="318"/>
    </p>
    <hr/>
    <h3 id="3.6%20%E5%87%BD%E6%95%B0" name="3.6%20%E5%87%BD%E6%95%B0">
     3.6 函数
    </h3>
    <p>
     定义 函数是一段可重复执行的代码块，可以接收输入参数并返回一个值或执行某些操作
    </p>
    <p>
     函数类似计算器，我只需要把数字给计算器即可，就能得到最后的值
    </p>
    <p>
     使用 function(){}
    </p>
    <pre><code class="hljs">        // 函数
        function add(x,y){   //add是函数的名字，x,y是传入的两个参数
            // 想让结果反馈 需要return
            return x+y;
        }
        // 首先需要调用函数，上述的内容才有效果
        console.log(add(1,2));  //3</code></pre>
    <hr/>
    <h3 id="3.7%20%E7%B1%BB" name="3.7%20%E7%B1%BB" style="background-color:transparent">
     3.7 类
    </h3>
    <p>
     定义 类是一种蓝图或
     <span style="color:#fe2c24">
      <strong>
       模板
      </strong>
     </span>
     ，用于创建具有相同属性和方法的对象
    </p>
    <p>
     比如人类：姓名年龄身高体重 等共有的属性 还有吃饭 等方法
    </p>
    <p>
     所有人类，随便一个人比如张三 他是一个个体，但是他属于人类这个类
    </p>
    <p>
     使用 class 关键字
    </p>
    <pre><code class="hljs">        // 类
        class Person {
            // name age
            // 定义构造函数 构造这个类的函数 constructor
            constructor(userName,userAge){
                this.name = userName;      //this指向他的这个类
                this.age = userAge;
            }

            // 定义一个类的方法来打印类的属性
            info(){
                console.log(this.name);
                console.log(this.age);
            }
        }
        let a = new Person('zzz',18);  // 实例化这个类
        // 调用类中的方法
        a.info();  // zzz 18

        // 不同人调用同一个类 实例化出不同的结果</code></pre>
