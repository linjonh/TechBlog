    <p>
     拓图

      要求：
     </img>
    </p>
    <ol>
     <li>
      为 office100 和 office200 分别配置地址池
     </li>
    </ol>
    <p>
     AR1接口配置
    </p>
    <pre><code class="prism language-bash">interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">192.168</span>.100.1 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface GigabitEthernet0/0/1
 <span class="token function">ip</span> address <span class="token number">192.168</span>.200.1 <span class="token number">255.255</span>.255.0 
</code></pre>
    <p>
     AR1路由器上创建office100地址池
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>dhcp <span class="token builtin class-name">enable</span> 		<span class="token comment"># 全局开启DHCP服务</span>
<span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>ip pool office100	<span class="token comment"># 创建office100地址池（名称随意）</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>network <span class="token number">192.168</span>.100.0 mask <span class="token number">24</span>	<span class="token comment"># 指定地址池所在网段</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>gateway-list <span class="token number">192.168</span>.100.1		<span class="token comment"># 指定该网段的网关</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>dns-list <span class="token number">8.8</span>.8.8 			<span class="token comment"># 指定DNS服务器</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>dns-list <span class="token number">114.114</span>.114.114		<span class="token comment"># 指定第二个DNS服务器</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>lease day <span class="token number">0</span> hour <span class="token number">8</span> minute <span class="token number">0</span> 	<span class="token comment"># 地址租约，允许客户端使用8小时</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>excluded-ip-address <span class="token number">192.168</span>.100.2 <span class="token number">192.168</span>.100.10		<span class="token comment"># 指定排除的地址范围（不能包括网关地址）</span>
<span class="token punctuation">[</span>AR1-ip-pool-office100<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     AR1路由器上，配置接口从地址池获取地址
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>interface GigabitEthernet <span class="token number">0</span>/0/0
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/0<span class="token punctuation">]</span>dhcp <span class="token keyword">select</span> global 
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/0<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     至此office100地址池配置完成，下面开始配置office200地址池
     <br/>
     AR1上创建office200地址池，并配置接口从地址池获取地址
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>ip pool office200	
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>network <span class="token number">192.168</span>.200.0 mask <span class="token number">24</span>
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>gateway-list <span class="token number">192.168</span>.200.1
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>dns-list <span class="token number">8.8</span>.8.8
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>dns-list <span class="token number">223.5</span>.5.5
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>lease day <span class="token number">0</span> hour <span class="token number">8</span> minute <span class="token number">0</span>
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>excluded-ip-address <span class="token number">192.168</span>.200.2 <span class="token number">192.168</span>.200.10
<span class="token punctuation">[</span>AR1-ip-pool-office200<span class="token punctuation">]</span>quit
<span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>interface GigabitEthernet <span class="token number">0</span>/0/1
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/1<span class="token punctuation">]</span>dhcp <span class="token keyword">select</span> global 
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/1<span class="token punctuation">]</span>quit
</code></pre>
