    <h2>
     <a id="IDEA__Spring_Boot__0">
     </a>
     <strong>
      IDEA 社区版如何创建 Spring Boot 项目（详细教程）
     </strong>
    </h2>
    <p>
     Spring Boot 以其简洁、高效的特性，成为 Java 开发的主流框架之一。虽然
     <strong>
      IntelliJ IDEA
     </strong>
     专业版提供了
     <strong>
      Spring Boot 项目向导
     </strong>
     ，但
     <strong>
      社区版（Community Edition）
     </strong>
     并不自带 Spring Boot 项目创建功能。
    </p>
    <p>
     那么，如何在
     <strong>
      IDEA 社区版
     </strong>
     中创建一个
     <strong>
      Spring Boot
     </strong>
     项目呢？本篇文章将手把手教你
     <strong>
      使用 IDEA 社区版 + Maven
     </strong>
     快速创建 Spring Boot 项目，并成功运行第一个 Spring Boot 应用！🚀
    </p>
    <hr/>
    <h3>
     <a id="1__8">
     </a>
     <strong>
      1. 前置准备
     </strong>
    </h3>
    <p>
     在创建 Spring Boot 项目之前，确保你的开发环境已经准备就绪。
    </p>
    <p>
     ✅
     <strong>
      安装 JDK（推荐 JDK 17 或以上）
     </strong>
    </p>
    <ul>
     <li>
      你可以在
      <a href="https://www.oracle.com/java/technologies/javase-downloads.html" rel="nofollow">
       Oracle JDK 官网
      </a>
      或
      <a href="https://openjdk.org/" rel="nofollow">
       OpenJDK
      </a>
      下载并安装。
     </li>
     <li>
      安装完成后，使用
      <code>
       java -version
      </code>
      检查：
      <pre><code class="prism language-sh"><span class="token function">java</span> <span class="token parameter variable">-version</span>
</code></pre>
      例如：
      <pre><code>openjdk version "17.0.2" 2023-09-20 LTS
</code></pre>
     </li>
    </ul>
    <p>
     ✅
     <strong>
      安装 IntelliJ IDEA 社区版
     </strong>
    </p>
    <ul>
     <li>
      下载地址：
      <a href="https://www.jetbrains.com/idea/download/" rel="nofollow">
       JetBrains 官网
      </a>
     </li>
     <li>
      选择
      <strong>
       Community（社区版）
      </strong>
      进行安装。
     </li>
    </ul>
    <p>
     ✅
     <strong>
      安装 Maven
     </strong>
     （可选，如果 IDEA 没有自带）
    </p>
    <ul>
     <li>
      下载：
      <a href="https://maven.apache.org/download.cgi" rel="nofollow">
       Maven 官网
      </a>
     </li>
     <li>
      配置环境变量，并运行以下命令检查安装是否成功：
      <pre><code class="prism language-sh">mvn <span class="token parameter variable">-version</span>
</code></pre>
      例如：
      <pre><code>Apache Maven 3.8.4
</code></pre>
     </li>
    </ul>
    <p>
     ✅
     <strong>
      确保网络畅通
     </strong>
     （国内建议配置阿里云 Maven 仓库）
    </p>
    <hr/>
    <h3>
     <a id="2__IDEA__Spring_Boot__42">
     </a>
     <strong>
      2. 在 IDEA 社区版中创建 Spring Boot 项目
     </strong>
    </h3>
    <p>
     虽然社区版没有直接的 Spring Boot 项目向导，但我们可以手动创建
     <strong>
      Maven 项目
     </strong>
     并引入 Spring Boot 依赖。
    </p>
    <h4>
     <a id="21__Maven__46">
     </a>
     <strong>
      2.1 创建 Maven 项目
     </strong>
    </h4>
    <p>
     1️⃣
     <strong>
      打开 IDEA 社区版
     </strong>
     ，点击
     <strong>
      “File” → “New” → “Project”
     </strong>
     <br/>
     2️⃣ 选择
     <strong>
      “Maven”
     </strong>
     ，然后点击
     <strong>
      Next
     </strong>
     <br/>
     3️⃣ 在
     <strong>
      “GroupId”
     </strong>
     和
     <strong>
      “ArtifactId”
     </strong>
     中填写项目信息，例如：
    </p>
    <ul>
     <li>
      <strong>
       GroupId
      </strong>
      ：
      <code>
       com.example
      </code>
     </li>
     <li>
      <strong>
       ArtifactId
      </strong>
      ：
      <code>
       springboot-demo
      </code>
     </li>
     <li>
      <strong>
       Version
      </strong>
      ：
      <code>
       1.0.0
      </code>
      （默认即可）
      <br/>
      4️⃣
      <strong>
       选择 JDK 版本
      </strong>
      （建议选择 17 或以上）
      <br/>
      5️⃣
      <strong>
       点击 Finish
      </strong>
      ，等待 IDEA 创建项目
     </li>
    </ul>
    <hr/>

     <a id="_1_Maven_168">
     </a>
     <strong>
      方法 1：使用 Maven
     </strong>
    </h4>
    <p>
     📌 在 IDEA
     <strong>
      终端（Terminal）
     </strong>
     中运行：
    </p>
    <pre><code class="prism language-sh">mvn spring-boot:run
</code></pre>
    <h4>
     <a id="_2_Java__174">
     </a>
     <strong>
      方法 2：使用 Java 运行
     </strong>
    </h4>
    <p>
     📌 在 IDEA
     <strong>
      左侧 “SpringbootDemoApplication.java” 右键 → “Run ‘SpringbootDemoApplication’”
     </strong>
     。
    </p>
    <p>
     ✅
     <strong>
      运行成功后，控制台会输出
     </strong>
     ：
    </p>
    <pre><code>Tomcat started on port 8080
Started SpringbootDemoApplication in 2.345 seconds
</code></pre>
    <p>
     📌
     <strong>
      在浏览器访问：
     </strong>
    </p>
    <pre><code>http://localhost:8080/api/hello
</code></pre>
    <p>
     ✅
     <strong>
      返回结果
     </strong>
     ：
    </p>
    <pre><code>Hello, Spring Boot!
</code></pre>

     <a id="5__194">

     <strong>
      5. 总结
     </strong>
    </h3>
    <p>
     🎯
     <strong>
      在 IDEA 社区版创建 Spring Boot 项目的完整步骤：
     </strong>
    </p>
    <ol>
     <li>
      <strong>
       创建 Maven 项目
      </strong>
      （手动选择
      <code>
       Maven
      </code>
      ）。
     </li>
     <li>
      <strong>
       添加 Spring Boot 依赖
      </strong>
      （修改
      <code>
       pom.xml
      </code>
      ）。
     </li>
     <li>
      <strong>
       编写 Spring Boot 启动类
      </strong>
      （
      <code>
       SpringbootDemoApplication.java
      </code>
      ）。
     </li>
     <li>
      <strong>
       创建 REST API 控制器
      </strong>
      （
      <code>
       HelloController.java
      </code>
      ）。
     </li>
     <li>
      <strong>
       运行 Spring Boot 项目
      </strong>
      （Maven 或 Java 直接运行）。
     </li>
    </ol>
    <p>
     ✨ **虽然 IDEA 社区版没有 Spring Boot 向导，但通过手动配置 Maven 依赖，我们依然可以顺利开发 Spring Boot 应用！**🚀
