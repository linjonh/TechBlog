    <h4>

     ### 一、说明
    </h4>
    <blockquote>
     <p>
      opencv-python仅依赖numpy，因此安装之前请确保numpy已经安装。
     </p>
    </blockquote>
    <h4>
     <a id="_2">
     </a>
     二、命令安装
    </h4>
    <p>
     通过以下命令安装python3-opencv
    </p>
    <pre><code class="prism language-bash">$ <span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> python3-opencv
</code></pre>
    <p>
     验证是否安装成功
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> cv2 <span class="token keyword">as</span> cv
<span class="token keyword">print</span><span class="token punctuation">(</span>cv<span class="token punctuation">.</span>__version__<span class="token punctuation">)</span>
</code></pre>
    <h4>
     <a id="_12">
     </a>
     三、源码安装
    </h4>
    <p>
     1、安装依赖
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> cmake
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> gcc g++
</code></pre>
    <pre><code class="prism language-bash"><span class="token comment"># for python2</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> python-dev python-numpy
</code></pre>
    <pre><code class="prism language-bash"><span class="token comment"># for python3</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> python3-dev python3-numpy
</code></pre>
    <pre><code class="prism language-bash"><span class="token comment"># 安装ffmpeg,gstreamer等环境</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> libavcodec-dev libavformat-dev libswscale-dev
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> libgstreamer-plugins-base1.0-dev libgstreamer1.0-dev
</code></pre>
    <pre><code class="prism language-bash"><span class="token comment"># support gtk2</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> libgtk2.0-dev
</code></pre>
    <pre><code class="prism language-bash"><span class="token comment"># support gtk3</span>
<span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> libgtk-3-dev
</code></pre>
    <p>
     2、下载opencv
    </p>
    <pre><code class="prism language-bash"><span class="token function">sudo</span> <span class="token function">apt-get</span> <span class="token function">install</span> <span class="token function">git</span>
<span class="token function">git</span> clone https://github.com/opencv/opencv.git
</code></pre>
    <p>
     3、编译安装
     <br/>
     进入opencv目录
    </p>
    <pre><code class="prism language-bash"><span class="token function">mkdir</span> build
<span class="token function">cd</span> build
cmake <span class="token punctuation">..</span>/
<span class="token function">make</span>
<span class="token function">sudo</span> <span class="token function">make</span> <span class="token function">install</span>
</code></pre>
    <p>
     4、验证
    </p>
    <pre><code class="prism language-bash"><span class="token function">import</span> cv2 as cv
print<span class="token punctuation">(</span>cv.__version__<span class="token punctuation">)</span>
</code></pre>
