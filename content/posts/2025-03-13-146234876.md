# 使用 Chrome Flags 设置（适用于 HTTP 站点开发）
### \*\*使用 Chrome Flags 设置（适用于 HTTP 站点开发）\*\*
1. 在 Chrome 地址栏输入：`chrome://flags/`
2. 在搜索框输入 \*\*“Insecure origins”\*\* 或 \*\*“Allow invalid certificates”\*\* 。
3. 找到 \*\*“Insecure origins treated as secure”\*\* 选项（或者 #allow-insecure-localhost）。
4. \*\*启用（Enable）\*\* 这个选项，并在文本框中输入允许的 HTTP 地址，
\* 例如：`http://localhost:3000`
5. 重新启动 Chrome。
