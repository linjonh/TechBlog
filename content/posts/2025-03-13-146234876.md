    <h3>
     <a id="_Chrome_Flags__HTTP__2">
     </a>
     <strong>
      使用 Chrome Flags 设置（适用于 HTTP 站点开发）
     </strong>
    </h3>
    <ol>
     <li>
      在 Chrome 地址栏输入：
      <code>
       chrome://flags/
      </code>
     </li>
     <li>
      在搜索框输入
      <strong>
       “Insecure origins”
      </strong>
      或
      <strong>
       “Allow invalid certificates”
      </strong>
      。
     </li>
     <li>
      找到
      <strong>
       “Insecure origins treated as secure”
      </strong>
      选项（或者 #allow-insecure-localhost）。
     </li>
     <li>
      <strong>
       启用（Enable）
      </strong>
      这个选项，并在文本框中输入允许的 HTTP 地址，
      <ul>
       <li>
        例如：
        <code>
         http://localhost:3000
        </code>
       </li>
      </ul>
     </li>
     <li>
      重新启动 Chrome。
     </li>
    </ol>
