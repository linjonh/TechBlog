    <p>
     由于第一次编写测试用例，若有不完整的地方，欢迎指正！！！
     <br/>
     首先将微信朋友圈分成以下模块：
    </p>
    <h4 id="1点赞功能">
     1、点赞功能
    </h4>
    <p>
     1、网速对点赞的影响
     <br/>
     2、点赞的人个数显示是否正确
     <br/>
     3、共同好友能否看到点赞状态
     <br/>
     4、能否显示点赞得人的头像和昵称，若能显示是否正确
     <br/>
     5、一行显示几个点赞的头像
     <br/>
     6、能否正常的点赞和取消点赞
     <br/>
     7、点赞显示能否按照时间的先后
     <br/>
     8、消息列表中是否显示点赞人的昵称，头像和点赞时间。
     <br/>
     9、不同手机操作系统显示界面
     <br/>
     10、能否及时刷新
     <br/>
     11、点赞后还能不能评论
     <br/>
     12、点赞是否有上限
    </p>
    <h4 id="2评论功能">
     2、评论功能
    </h4>
    <p>
     1、网速对评论的影响
     <br/>
     2、共同好友能否看得到评论，非共同好友能否看到评论状态
     <br/>
     3、评论能否按时间先后顺序显示
     <br/>
     4、评论能否显示评论人的昵称，若能显示是否正确
     <br/>
     5、能否回复评论
     <br/>
     6、是否可以既评论又点赞
     <br/>
     7、评论和点赞后是怎样现实的，分两次显示，还是一次显示
     <br/>
     8、评论是否有上限
     <br/>
     9、能否及时刷新
     <br/>
     10、未登录情况下能否看得到
     <br/>
     11、不同手机如何显示
     <br/>
     12、是否能将评论全部显示在朋友圈下面
     <br/>
     13、好友能否看到发圈人的评论及回复
    </p>
    <h4 id="3界面显示">
     3、界面显示
    </h4>
    <p>
     1、是否是显示发朋友圈的人的昵称、头像、以及具体内容
     <br/>
     2、是否按照发朋友圈的时间距离现在远近来排序
     <br/>
     3、图片显示是否正确
     <br/>
     4、是否显示自己的个人朋友圈背景
     <br/>
     5、下拉是否有更新
     <br/>
