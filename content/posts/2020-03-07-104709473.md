    <p>
     数据库管理员不是开发数据库的专门人员；
     <br/>
     数据库开发人员包括：
     <br/>
     1、DBA(数据库管理人员)
     <br/>
     参与数据库中的信息内容和结构
     <br/>
     参与数据库的存储结构和存取策略的设计
     <br/>
     参与确定数据安全性要求和完整性约束条件
     <br/>
     .监控数据库的使用和运行
     <br/>
     数据库的改进和重组
     <br/>
     2、系统分析员、
     <br/>
     系统分析员负责应用系统的需求分析和规范说明
     <br/>
     与用户及数据库管理员结合,确定系统的硬软件配置
     <br/>
     参与数据库系统的概要设计
     <br/>
     3、.数据库设计人员
     <br/>
     参加用户需求调查和系统分析
     <br/>
     确定数据库中的数据
     <br/>
     设计数据库各级模式
     <br/>
     4、应用程序员
     <br/>
     设计和编写应用系统的程序檬块
     <br/>
     进行调试和安装
    </p>
