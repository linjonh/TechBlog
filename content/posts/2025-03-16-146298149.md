    <h1 class="title-article" id="articleContentId">
     FX-std::queue
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <p>
     <code>
      std::queue
     </code>
     是 C++ 标准库中的一个容器适配器，它提供了先进先出（FIFO）的数据结构。
     <code>
      std::queue
     </code>
     是基于其他容器（如
     <code>
      std::deque
     </code>
     或
     <code>
      std::list
     </code>
     ）实现的，默认情况下使用
     <code>
      std::deque
     </code>
     作为底层容器。
    </p>
    <h4>
     基本用法
    </h4>
    <h5>
     1. 包含头文件
    </h5>
    <p>
     要使用
     <code>
      std::queue
     </code>
     ，首先需要包含
     <code>
      &lt;queue&gt;
     </code>
     头文件：
    </p>
    <pre><code class="language-cpp">#include &lt;queue&gt;</code></pre>
    <h5>
     2. 定义和初始化
    </h5>
    <p>
     <code>
      std::queue
     </code>
     的定义和初始化如下：
    </p>
    <pre><code class="language-cpp">std::queue&lt;int&gt; q;  // 定义一个存储 int 类型元素的队列</code></pre>
    <p>
     你也可以使用其他容器作为底层容器：
    </p>
    <pre><code class="language-cpp">std::queue&lt;int, std::list&lt;int&gt;&gt; q;  // 使用 std::list 作为底层容器</code></pre>
    <h5>
     3. 常用操作
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        <code>
         push()
        </code>
       </strong>
       : 在队列的末尾插入一个元素。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         pop()
        </code>
       </strong>
       : 移除队列的第一个元素。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         front()
        </code>
       </strong>
       : 访问队列的第一个元素。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         back()
        </code>
       </strong>
       : 访问队列的最后一个元素。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         empty()
        </code>
       </strong>
       : 检查队列是否为空。
      </p>
     </li>
     <li>
      <p>
       <strong>
        <code>
         size()
        </code>
       </strong>
       : 返回队列中元素的数量。
      </p>
     </li>
    </ul>
    <h5>
     示例代码
    </h5>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;queue&gt;

int main() {
    std::queue&lt;int&gt; q;

    // 向队列中添加元素
    q.push(10);
    q.push(20);
    q.push(30);

    // 访问队列的第一个元素
    std::cout &lt;&lt; "Front element: " &lt;&lt; q.front() &lt;&lt; std::endl;

    // 访问队列的最后一个元素
    std::cout &lt;&lt; "Back element: " &lt;&lt; q.back() &lt;&lt; std::endl;

    // 移除队列的第一个元素
    q.pop();

    // 再次访问队列的第一个元素
    std::cout &lt;&lt; "Front element after pop: " &lt;&lt; q.front() &lt;&lt; std::endl;

    // 检查队列是否为空
    if (!q.empty()) {
        std::cout &lt;&lt; "Queue is not empty, size: " &lt;&lt; q.size() &lt;&lt; std::endl;
    }

    return 0;
}</code></pre>
    <h5>
     输出结果
    </h5>
    <pre><code class="language-cpp">Front element: 10
Back element: 30
Front element after pop: 20
Queue is not empty, size: 2</code></pre>
    <h4>
     注意事项
    </h4>
    <ul>
     <li>
      <p>
       <code>
        std::queue
       </code>
       不支持随机访问，你只能访问队列的第一个和最后一个元素。
      </p>
     </li>
     <li>
      <p>
       <code>
        pop()
       </code>
       操作只会移除元素，不会返回它。如果你需要访问并移除元素，需要先使用
       <code>
        front()
       </code>
       访问元素，然后再调用
       <code>
        pop()
       </code>
       。
      </p>
     </li>
     <li>
      <p>
       <code>
        std::queue
       </code>
       的底层容器可以通过模板参数指定，但通常情况下使用默认的
       <code>
        std::deque
       </code>
       即可满足需求。
      </p>
     </li>
    </ul>
    <h4>
     总结
    </h4>
    <p>
     <code>
      std::queue
     </code>
     是一个简单易用的 FIFO 数据结构，适用于需要按顺序处理元素的场景。通过
     <code>
      push()
     </code>
     、
     <code>
      pop()
     </code>
     、
     <code>
      front()
     </code>
     等操作，可以方便地管理队列中的元素。
    </p>
