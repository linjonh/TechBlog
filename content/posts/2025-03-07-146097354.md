     三层交换机是指在OSI（开放系统互连）模型中的第三层网络层提供路由功能的交换机。它不仅具备二层交换机的交换功能，还能实现路由功能，提供更为灵活的网络配置和通信控制。
    </p>
    <hr/>
    <h2>
     一、概念
    </h2>
    <p>
     <strong>
      三层交换机的主要应用场景：
     </strong>
    </p>
    <ol>
     <li>
      大型企业网络：用于构建大规模的企业网络，实现内部网络的互联和通信。
     </li>
     <li>
      数据中心网络：作为核心交换机，用于连接不同子网或计算节点，实现数据中心内部的高效通信。
     </li>
     <li>
      云计算网络：用于构建云计算数据中心的网络架构，支持虚拟化技术和大规模的网络流量管理。
     </li>
    </ol>
    <p>
     <strong>
      三层交换机的主要作用：
     </strong>
    </p>
    <ol>
     <li>
      提供更灵活的网络拓扑结构设计，支持更复杂的网络架构。
     </li>
     <li>
      实现不同子网之间的通信和数据传输，实现跨网络的数据传输。
     </li>
     <li>
      支持 VLAN（虚拟局域网）和子网划分，增强网络安全性和隔离性。
     </li>
     <li>
      实现负载均衡和流量控制，提高网络性能和可靠性。
     </li>
    </ol>
    <p>
     <strong>
      三层交换机的配置一般需要通过命令行界面或Web界面进行，主要包括以下几个步骤：
     </strong>
    </p>
    <ol>
     <li>
      配置接口：为每个接口设置IP地址、子网掩码等参数。
     </li>
     <li>
      配置路由：设置静态路由或动态路由协议，实现不同网络之间的路由。
     </li>
     <li>
      配置VLAN：设置VLAN及VLAN间的路由，实现不同VLAN之间的通信。
     </li>
     <li>
      配置安全策略：设置访问控制列表（ACL）等安全策略，保护网络安全。
     </li>
     <li>
      监控和管理：配置SNMP、Syslog等，实现对交换机的监控和管理。
     </li>
    </ol>
    <h2>
     二、三层交换机配置示例
    </h2>
    <p>
     不了解交换机基础配置的兄弟，可看看上一篇博客。自学的毒鸡汤一搜一大把，这里就不再啰嗦了。
    </p>
    <p>
     <a class="link-info has-card" href="https://blog.csdn.net/2303_76492156/article/details/146021585" title="任务9：交换机基础及配置">
      <span class="link-card-box" contenteditable="false">
       <span class="link-title">
        任务9：交换机基础及配置
       </span>
       <span class="link-link">
        <img class="link-link-icon" src="https://csdnimg.cn/release/blog_editor_html/release2.3.8/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=P1C7">
         https://blog.csdn.net/2303_76492156/article/details/146021585
        </img>
       </span>
      </span>
     </a>
     上一篇博客了解了如何利用交换机实现两台不同网关的PC机连接，本篇博客示例，是想通过配置实现PC1和PC3的连接。对于这种情况，就需要配置三层交换机实现。
    </p>
    <p>
     <img alt="" height="0" src="https://img-home.csdnimg.cn/images/20230724024159.png?origin_url=https://img-home.csdnimg.cn/images/20230724024159.png?origin_url=undefined&amp;pos_id=twZjem81&amp;pos_id=twZjem81" width="0">
      <img alt="" height="633" src="https://i-blog.csdnimg.cn/direct/5329f93ac1b04fd6a5b2dabd01137582.png" width="930"/>
     </img>
    </p>
    <hr/>
    <p>
     <span style="color:#ed7976">
      <strong>
       对交换机LSW1的0/0/3接口进行配置
      </strong>
     </span>
    </p>
    <pre><code>sys
interface GigabitEthernet 0/0/3
port link-type trunk
port trunk allow-pass vlan all</code></pre>
    <p>
     <img alt="" height="675" src="https://i-blog.csdnimg.cn/direct/3b33501d2d5f4e45a137bbf8dbdfb164.png" width="992"/>
    </p>
    <p>
     <span style="color:#1a439c">
      <strong>
       同理，
      </strong>
     </span>
     <span style="color:#ed7976">
      <strong>
       对交换机LSW2的0/0/3接口进行相同的配置，
      </strong>
     </span>
     <span style="color:#1a439c">
      <strong>
       这里不再重复演示。
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      提醒：二层交换机（连接三层交换机）配置的接口需按照实际情况，示例中是0/0/3
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      最后，
     </span>
     <span style="color:#ed7976">
      <strong>
       对交换机LSW3进行配置。
      </strong>
     </span>
     <span style="color:#1a439c">
      <strong>
       将LSW1和LSW2的vlan都放到LSW3中，即可实现连接
      </strong>
     </span>
    </p>
    <pre><code>sys
vlan batch 10 20 30 40

//LSW3中的vlan需要与LSW1和LSW2中配置的ip地址对应
interface vlanif 10
ip address 10.10.10.1 24
q
interface vlanif 20
ip address 10.10.20.1 24
q
inerface vlanif 30
ip address 10.10.30.1 24
q
interface vlanif 40
ip address 10.10.40.1 24
q

//配置LSW3的接口
interface GigabitEthernet 0/0/1
port link-type trunk
port trunk allow-pass vlan all
display this
q
interface GigabitEthernet 0/0/2
port link-type trunk
port trunk allow-plass vlan all
display this
</code></pre>
    <p>
     <span style="color:#1a439c">
      <strong>
       配置完成后，在PC1中ping一下PC3的ip地址。
      </strong>
     </span>
     ping通，配置成功。
    </p>
    <p>
     <img alt="" height="800" src="https://i-blog.csdnimg.cn/direct/04ead3ff86c84de6a15314e804662e55.png" width="1172"/>
    </p>
    <p>
     通过以上步骤，可以实现二层交换机和三层交换机的连接，并确保不同VLAN之间的通信。二层交换机主要用于同一VLAN内的通信，而三层交换机则用于不同VLAN之间的路由。
    </p>
