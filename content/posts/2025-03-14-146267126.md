# GitHub Copilot 登录失败问题

最近在DIEA安装Copilot出现了，登录出现：Sign in failed. Reason: Could not log in with device flow on github.com: Request timed out from lsp server, request id: 6, error code: 1001报错。

主要问题就是访问不到Github，本且代理没有生效！

最后发现，只要在微软商店下载一个watt toolkit
  
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/aa8b2b67c65c4b6bae902612af786c52.png)

然后打开，选择GitHub加速即可。
