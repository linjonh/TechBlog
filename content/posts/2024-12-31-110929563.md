# 使用Vscode和微信开发者工具开发小程序为什么代码没有同步更新（Vue）

### 找到自己的vue文件夹后打开控制台输入

```
cnpm run dev:mp-weixin

```

### 让控制台处于打开状态，不要关闭控制台，此时点击微信小程序的编译按钮代码同步成功。
