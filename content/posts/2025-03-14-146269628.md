    <blockquote>
     <h4>
      【题目描述】
     </h4>
     <p>
      输入一个整数矩阵，计算位于矩阵边缘的元素之和。所谓矩阵边缘的元素，就是第一行和最后一行的元素以及第一列和最后一列的元素。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      第一行分别为矩阵的行数m和列数n（m&lt;100，n&lt;100），两者之间以一个空格分开。
     </p>
     <p>
      接下来输入的m行数据中，每行包含n个整数，整数之间以一个空格分开。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      输出对应矩阵的边缘元素和。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      3 3
      <br/>
      3 4 1
      <br/>
      3 7 1
      <br/>
      2 0 1
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      15
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e2 + 10;
int nums[N][N];

int main()
{
	int n, m; cin &gt;&gt; n &gt;&gt; m;
	int sum = 0;
	for (int i = 1; i &lt;= n; i++)
	{
		for (int j = 1; j &lt;= m; j++)
		{
			cin &gt;&gt; nums[i][j];
			if (i == 1 || j == 1 || i == n || j == m) sum += nums[i][j];
		}
	}
	cout &lt;&lt; sum;

	return 0;
}</code></pre>
