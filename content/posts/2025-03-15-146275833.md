    <h2>
     <a id="_1">
     </a>
     前置环境
    </h2>
    <ul>
     <li>
      安装anaconda环境，并保证环境变量已配置，检测命令
     </li>
    </ul>
    <pre><code class="prism language-bash">conda <span class="token parameter variable">--version</span>
</code></pre>
    <h2>
     <a id="_6">
     </a>
     配置镜像源
    </h2>
    <blockquote>
     <p>
      提醒：请务必使用管理员身份运行终端，防止后续操作因权限不够导致的错误。
     </p>
    </blockquote>
    <ol>
     <li>
      使用管理员身份运行终端，查看镜像源列表
      <pre><code class="prism language-bash">conda config <span class="token parameter variable">--show</span> channels
</code></pre>
     </li>
     <li>
      删除配置的非默认镜像源
      <pre><code class="prism language-bash">conda config --remove-key channels
</code></pre>
     </li>
     <li>
      设置全局国内镜像源
      <pre><code class="prism language-bash">pip config <span class="token builtin class-name">set</span> global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
</code></pre>
     </li>
    </ol>
    <h2>
     <a id="pyrotch_21">

     安装pyrotch

    <blockquote>
     <p>
      提醒：请务必使用管理员身份运行终端，防止后续操作因权限不够导致的错误。
     </p>
    </blockquote>
    <ol>
     <li>
      使用anaconda创建虚拟环境
      <pre><code class="prism language-bash">conda create <span class="token parameter variable">-n</span> pytorch_cpu <span class="token assign-left variable">python</span><span class="token operator">=</span><span class="token number">3.9</span>
</code></pre>
     </li>
     <li>
      执行conda初始化
      <pre><code class="prism language-bash">conda init
</code></pre>
     </li>
     <li>
      激活虚拟环境
      <pre><code class="prism language-bash">conda activate pytorch_cpu
</code></pre>
     </li>
     <li>
      安装cpu版的pytorch
      <pre><code class="prism language-bash">pip3 <span class="token function">install</span> torch torchvision torchaudio
</code></pre>
     </li>
