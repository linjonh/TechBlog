    <p>
    </p>
    <p>
     Java 实现单例模式的方式主要有以下几种，每种方式都有其
     <strong>
      优缺点
     </strong>
     和适用场景。
    </p>
    <hr/>
    <h3>
     <a id="1_Eager_Initialization_7">
     </a>
     <strong>
      1. 饿汉式（Eager Initialization）
     </strong>
    </h3>
    <h4>
     <a id="_8">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token keyword">final</span> <span class="token class-name">Singleton</span> <span class="token constant">INSTANCE</span> <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    
    <span class="token keyword">private</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span> <span class="token comment">// 私有构造方法，防止外部实例化</span>
    
    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">return</span> <span class="token constant">INSTANCE</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_20">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      实现简单，类加载时即创建实例，线程安全。
     </li>
     <li>
      访问速度快，调用
      <code>
       getInstance()
      </code>
      方法不会有同步开销。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       可能造成资源浪费
      </strong>
      ：即使实例可能长期未使用，仍然会在类加载时创建。
     </li>
    </ul>
    <h4>
     <a id="_28">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       对象创建成本低
      </strong>
      ，且
      <strong>
       类会被频繁使用
      </strong>
      的场景（如工具类）。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="2_Lazy_Initialization_33">
     </a>
     <strong>
      2. 懒汉式（Lazy Initialization，非线程安全）
     </strong>
    </h3>
    <h4>
     <a id="_34">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> instance<span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>

    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">if</span> <span class="token punctuation">(</span>instance <span class="token operator">==</span> <span class="token keyword">null</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
            instance <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
        <span class="token keyword">return</span> instance<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_49">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      只有
      <strong>
       在第一次调用时才创建实例
      </strong>
      ，节省资源。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       非线程安全
      </strong>
      ：多个线程同时调用
      <code>
       getInstance()
      </code>
      可能导致多个实例被创建。
     </li>
    </ul>
    <h4>
     <a id="_56">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       单线程环境
      </strong>
      ，或者
      <strong>
       不在多线程场景下使用
      </strong>
      。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="3__61">
     </a>
     <strong>
      3. 线程安全的懒汉式（同步方法）
     </strong>
    </h3>
    <h4>
     <a id="_62">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> instance<span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>

    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token keyword">synchronized</span> <span class="token class-name">Singleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">if</span> <span class="token punctuation">(</span>instance <span class="token operator">==</span> <span class="token keyword">null</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
            instance <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
        <span class="token punctuation">}</span>
        <span class="token keyword">return</span> instance<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_77">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       线程安全
      </strong>
      ，可确保只有一个实例被创建。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <code>
       synchronized
      </code>
      关键字会
      <strong>
       影响性能
      </strong>
      ，每次调用
      <code>
       getInstance()
      </code>
      都会加锁。
     </li>
    </ul>
    <h4>
     <a id="_84">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      需要
      <strong>
       保证线程安全
      </strong>
      ，但实例创建
      <strong>
       不会频繁调用
      </strong>
      的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="4_DoubleChecked_Locking_89">
     </a>
     <strong>
      4. 双重检查锁（Double-Checked Locking，推荐）
     </strong>
    </h3>
    <h4>
     <a id="_90">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token keyword">volatile</span> <span class="token class-name">Singleton</span> instance<span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>

    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">if</span> <span class="token punctuation">(</span>instance <span class="token operator">==</span> <span class="token keyword">null</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token comment">// 第一次检查</span>
            <span class="token keyword">synchronized</span> <span class="token punctuation">(</span><span class="token class-name">Singleton</span><span class="token punctuation">.</span><span class="token keyword">class</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
                <span class="token keyword">if</span> <span class="token punctuation">(</span>instance <span class="token operator">==</span> <span class="token keyword">null</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token comment">// 第二次检查</span>
                    instance <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
                <span class="token punctuation">}</span>
            <span class="token punctuation">}</span>
        <span class="token punctuation">}</span>
        <span class="token keyword">return</span> instance<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_109">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      线程安全，
      <strong>
       只有在第一次调用时才会加锁
      </strong>
      ，后续调用不会影响性能。
     </li>
     <li>
      <strong>
       结合了懒加载
      </strong>
      和
      <strong>
       高效访问
      </strong>
      的优点。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      需要使用
      <code>
       volatile
      </code>
      <strong>
       防止指令重排序
      </strong>
      ，Java 1.5 之前的版本不支持。
     </li>
    </ul>
    <h4>
     <a id="_117">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       多线程环境
      </strong>
      ，且
      <strong>
       性能要求较高
      </strong>
      的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="5__122">
     </a>
     <strong>
      5. 静态内部类（推荐）
     </strong>
    </h3>
    <h4>
     <a id="_123">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>

    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token keyword">class</span> <span class="token class-name">SingletonHolder</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token keyword">final</span> <span class="token class-name">Singleton</span> <span class="token constant">INSTANCE</span> <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">Singleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>

    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token class-name">Singleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">return</span> <span class="token class-name">SingletonHolder</span><span class="token punctuation">.</span><span class="token constant">INSTANCE</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_137">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       线程安全
      </strong>
      ，利用 JVM
      <strong>
       类加载机制
      </strong>
      保证只会初始化一次。
     </li>
     <li>
      只有
      <strong>
       在调用
       <code>
        getInstance()
       </code>
       时才会创建实例
      </strong>
      ，节省资源（懒加载）。
     </li>
     <li>
      <strong>
       无锁
      </strong>
      ，性能高。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       无法传递参数
      </strong>
      ，如果需要参数化实例，则无法使用该方法。
     </li>
    </ul>
    <h4>
     <a id="_146">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       高并发环境
      </strong>
      ，且
      <strong>
       希望使用懒加载
      </strong>
      的场景。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="6__151">
     </a>
     <strong>
      6. 枚举单例（最推荐）
     </strong>
    </h3>
    <h4>
     <a id="_152">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">enum</span> <span class="token class-name">Singleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token constant">INSTANCE</span><span class="token punctuation">;</span>

    <span class="token keyword">public</span> <span class="token keyword">void</span> <span class="token function">doSomething</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token class-name">System</span><span class="token punctuation">.</span>out<span class="token punctuation">.</span><span class="token function">println</span><span class="token punctuation">(</span><span class="token string">"Do something..."</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_162">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       线程安全
      </strong>
      ，由 JVM 保证。
     </li>
     <li>
      <strong>
       防止反序列化破坏单例
      </strong>
      ，因为枚举不会创建新的实例。
     </li>
     <li>
      <strong>
       代码最简洁
      </strong>
      ，推荐使用。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       无法懒加载
      </strong>
      ，类加载时即创建实例。
     </li>
     <li>
      <strong>
       不能继承其他类
      </strong>
      （但可以实现接口）。
     </li>
    </ul>
    <h4>
     <a id="_172">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       绝对保证单例性
      </strong>
      的场景，如
      <strong>
       数据库连接管理、日志管理器
      </strong>
      等。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="7__ThreadLocal__177">
     </a>
     <strong>
      7. 通过
      <code>
       ThreadLocal
      </code>
      实现（每个线程单例）
     </strong>
    </h3>
    <h4>
     <a id="_178">
     </a>
     <strong>
      实现方式
     </strong>
    </h4>
    <pre><code class="prism language-java"><span class="token keyword">public</span> <span class="token keyword">class</span> <span class="token class-name">ThreadLocalSingleton</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">private</span> <span class="token keyword">static</span> <span class="token keyword">final</span> <span class="token class-name">ThreadLocal</span><span class="token generics"><span class="token punctuation">&lt;</span><span class="token class-name">ThreadLocalSingleton</span><span class="token punctuation">&gt;</span></span> instance <span class="token operator">=</span>
        <span class="token class-name">ThreadLocal</span><span class="token punctuation">.</span><span class="token function">withInitial</span><span class="token punctuation">(</span><span class="token class-name">ThreadLocalSingleton</span><span class="token operator">::</span><span class="token keyword">new</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

    <span class="token keyword">private</span> <span class="token class-name">ThreadLocalSingleton</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>

    <span class="token keyword">public</span> <span class="token keyword">static</span> <span class="token class-name">ThreadLocalSingleton</span> <span class="token function">getInstance</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        <span class="token keyword">return</span> instance<span class="token punctuation">.</span><span class="token function">get</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_191">
     </a>
     <strong>
      优缺点
     </strong>
    </h4>
    <p>
     ✅
     <strong>
      优点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       线程隔离
      </strong>
      ，每个线程都有自己独立的实例。
     </li>
     <li>
      避免了全局同步，适用于
      <strong>
       多线程数据隔离场景
      </strong>
      。
     </li>
    </ul>
    <p>
     ❌
     <strong>
      缺点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <strong>
       无法做到全局唯一单例
      </strong>
      ，只在
      <strong>
       单个线程内保持单例
      </strong>
      。
     </li>
    </ul>
    <h4>
     <a id="_199">
     </a>
     <strong>
      适用场景
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       每个线程需要独立状态
      </strong>
      的场景，如
      <strong>
       用户会话、数据库连接
      </strong>
      。
     </li>
    </ul>
