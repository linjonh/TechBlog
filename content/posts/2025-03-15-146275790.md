    <div style="text-align:left">
     <span style="color:#000000">
      <span style="background-color:#f7f7f8">
       FFmpeg 是一个开源的跨平台
      </span>
     </span>
     <strong>
      <span style="color:#000000">
       <span style="background-color:#f7f7f8">
        多媒体处理工具
       </span>
      </span>
     </strong>
     <span style="color:#000000">
      <span style="background-color:#f7f7f8">
       ，可以用于
      </span>
     </span>
     <strong>
      <span style="color:#000000">
       <span style="background-color:#f7f7f8">
        录制、转换、编辑、流式传输
       </span>
      </span>
     </strong>
     <span style="color:#000000">
      <span style="background-color:#f7f7f8">
       音视频文件。它支持几乎所有常见的音视频格式，功能极其强大，是开发者、视频创作者常用的命令行工具。
      </span>
     </span>
    </div>
    <div style="text-align:left">
     <hr/>
    </div>
    <h4 id="TVRJ-1742006539847" style="text-align:left">
     <strong>
      一、FFmpeg 核心功能
     </strong>
    </h4>
    <div>
     <ol style="margin-left:0; margin-right:0">
      <li style="text-align:left">
       格式转换：将视频/音频从一种格式转换为另一种格式（如 MP4 → AVI、MP3 → WAV）。
      </li>
      <li style="text-align:left">
       剪辑/合并：截取视频片段、合并多个文件。
      </li>
      <li style="text-align:left">
       调整参数：修改分辨率、码率、帧率、音量等。
      </li>
      <li style="text-align:left">
       提取内容：从视频中提取音频，或从音频中提取片段。
      </li>
      <li style="text-align:left">
       流媒体处理：支持直播推流、拉流等网络传输操作。
      </li>
     </ol>
    </div>
    <div style="text-align:left">
     <hr/>
    </div>
    <h4 id="xCJe-1742006539860" style="text-align:left">
     <strong>
      二、基础安装
     </strong>
    </h4>
    <div>
     <ul style="margin-left:0; margin-right:0">
      <li style="text-align:left">
       Windows：官网下载编译好的二进制文件，解压后即可使用。
      </li>
      <li style="text-align:left">
       macOS：通过 Homebrew 安装：brew install ffmpeg
      </li>
      <li style="text-align:left">
       Linux：使用包管理器安装：
      </li>
     </ul>
    </div>
    <div>
     <pre><code class="language-bash">sudo apt-get install ffmpeg # Debian/Ubuntu 
sudo yum install ffmpeg # CentOS</code></pre>
    </div>
    <div style="text-align:left">
     <hr/>
    </div>
    <h4 id="e9Ca-1742006539874" style="text-align:left">
     <strong>
      三、基础使用命令
     </strong>
    </h4>
    <h5 id="H9bI-1742006539877" style="text-align:left">
     <strong>
      1. 通用命令结构
     </strong>
    </h5>
    <div>
     <pre><code class="language-bash">ffmpeg -i [输入文件] [参数] [输出文件]</code></pre>
    </div>
    <div style="text-align:left">
     <hr/>
    </div>
    <h5 id="AHfA-1742006539883" style="text-align:left">
     <strong>
      2. 常见场景示例
     </strong>
    </h5>
    <div style="text-align:left">
     <strong>
      (1) 格式转换（MP4 → AVI）
     </strong>
    </div>
    <div>
     <pre><code class="language-bash">ffmpeg -i input.mp4 output.avi</code></pre>
    </div>
    <div style="text-align:left">
     <strong>
      (2) 提取音频（从视频中提取 MP3）
     </strong>
    </div>
    <div>
     <pre><code class="language-bash">ffmpeg -i input.mp4 -vn -q:a 0 output.mp3
 # -vn: 禁用视频流
 # -q:a 0: 音频质量最佳（0-9，0为最高）</code></pre>
    </div>
    <div style="text-align:left">
     <strong>
      (3) 调整视频分辨率（缩放为 640x480）
     </strong>
    </div>
    <div>
     <pre><code class="language-bash">ffmpeg -i input.mp4 -vf "scale=640:480" output.mp4</code></pre>
    </div>
    <div style="text-align:left">
     <strong>
      (4) 截取视频片段（从第10秒开始截取15秒）
     </strong>
    </div>
    <div>
     <pre><code class="language-bash">ffmpeg -i input.mp4 -ss 00:00:10 -t 15 output.mp4
 # -ss: 开始时间
 # -t: 持续时间</code></pre>
    </div>
    <div style="text-align:left">
     <strong>
      (5) 合并多个视频文件
     </strong>
    </div>
    <div>
     <ol style="margin-left:0; margin-right:0">
      <li style="text-align:left">
       创建 list.txt 文件，写入待合并的文件路径：
      </li>
     </ol>
    </div>
    <div>
     <pre><code class="language-bash">file 'input1.mp4' file 'input2.mp4'</code></pre>
    </div>
    <div>
     <ol style="margin-left:0; margin-right:0">
      <li style="text-align:left">
       执行合并命令：
      </li>
     </ol>
    </div>
    <div>
     <pre><code class="language-bash">ffmpeg -f concat -i list.txt -c copy output.mp4</code></pre>
    </div>
