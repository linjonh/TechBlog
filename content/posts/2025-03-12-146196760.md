    <p>
     <br/>
     一、 项目介绍
     <br/>
     基于SSM的电影售票系统
     <br/>
     <br/>
     角色：管理员、用户
     <br/>
     <br/>
     管理员：管理员点击登录摁钮页面跳转到“爱眼”电影售票系统对系统首页、用户、影院信息、影厅信息、电影类型、正在上映、即将上映、电影资讯、系统管理、订单管理、我的信息等功能，进行相应操作，
     <br/>
     <br/>
     用户：当用户打开系统的网址后，首先看到的就是首页界面。在这里，用户能够看到“爱眼”电影售票系统的导航条显示首页、影院信息、影厅信息、正在上映、即将上映、电影资讯、在线客服、个人中心等。用户点击个人中心，在个人中心页面可以更新个人详细信息，还可以对修改密码、我的 订单、我的收藏 进行详细操作；
     <br/>
     <br/>
     二、运行环境
     <br/>
     开发语言：Java
     <br/>
     <br/>
     开发工具：IDEA，Eclipse,Myeclipse都可以。推荐IDEA
     <br/>
     <br/>
     JDK版本：1.8
     <br/>
     <br/>
     数据库: MySQL5.7
     <br/>
     <br/>
     Tomcat环境：Tomcat 7.x,8.x,9.x版本均可
     <br/>
     <br/>
     项目管理：Maven
     <br/>
     <br/>
     三、项目技术
     <br/>
     后端：SSM(Spring+SpringMVC+Mybatis)
     <br/>
     <br/>
     前端：Vue +ElementUI+Vue Router + Axios
