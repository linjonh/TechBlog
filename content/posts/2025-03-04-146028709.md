     <h3 style="text-align:left">
      <strong>
       <a name="_Toc15800">
       </a>
       1.1开发背景
      </strong>
     </h3>
     <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
      上个世纪末期，网络技术逐渐开始流行起来，互联网发展进入了起步阶段。到了新世纪初期，信息化技术开始逐渐发展，计算机应用技术逐渐体现出优势来。接着，大量的电脑进入了千家万户，企事业单位、高校、机关等部门。信息化时代逐渐来临，实践学分认定成为互联网时代下媒体管理的核心内容。实践学分认定是媒体中非常重要的一部分，然而由于其性质也致使它难以被实时衡量。长期来看，管理十分不便利，存在误差，查询难的现象，一旦数据过多，人工管理方式就很难维持。随着信息技术的应用拓展，逐渐引用了信息化技术管理，逐渐取代了人工管理模式，采用计算机系统来管理学校信息，带来的好处是查询便利，信息准确率高，节省了开资，提高了工作的效率。本次系统开发，主要以实践学分认定为对象，根据需求来完成功能设计[1][2]。
     </p>
     <h3 style="text-align:left">
      <strong>
       <a name="_Toc14542">
       </a>
       1.2开发意义
      </strong>
     </h3>
     <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
      随着现代实践学分认定的快速发展，可以说实践学分认定已经逐渐成为现代实践学分认定过程中最为重要的部分之一。但是一直以来我国传统的实践学分认定并没有建立一套完善的行之有效的实践学分认定系统，传统的实践学分认定已经无法适应高速发展，无论是从效率还是从效果来看都远远的和预计相差甚多，可以断言，传统的实践学分认定模式已经不再适应，甚至已经拖累了发展。因此，需要一个操作方便，高效的管理体系。
     </p>
     <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
      系统设计的主要意义在于，一方面，对于学院来讲，系统上线后可以带来很大的便利性，实践学分认定属于非常细致的管理模式，要求数据量大，计算机管理可以提高精确性，更为便利的就是信息的查询，大大的提高了工作效率，改善了管理质量；另一方面，通过系统的设计，自己对软件设计有了更加深入的了解，并且产生了浓厚的学习兴趣，在对问题分析方面有了独特的方式，是进入社会之前一次很好的锻炼机会[3][4]。
     </p>
     <h3 style="text-align:left">
      <strong>
       <a name="_Toc30286">
       </a>
       1.3研究内容
      </strong>
     </h3>
     <ol>
      <li>
       不同用户的登录问题：根据用户的类别（超级管理员、普通用户）实现用户操作权限的区分并显示不同的操作界面。
      </li>
      <li>
       数据库的连接问题：数据库连接为系统中的关键技术。
      </li>
      <li>
       数据的一致性和安全问题：本系统必须保证数据的一致性和安全，才能实现有效的管理。不能让没有权限的用户对数据进行操作并且定期对数据库进行备份。
      </li>

     第2章 主要技术和工具介绍
    </h2>
    <h3 style="text-align:left">
     <strong>
      <a name="_Toc8878">
      </a>
      <a name="_Toc1330">
      </a>
      2.1
      <span style="color:#000000">
       JSP
      </span>
      语言
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     本系统的开发主要用
     <span style="color:#000000">
      JSP
     </span>
     （Hypertext Preprocessor，超文本预处理器）这一门开发语言。作为开发语言，这是一种通用的开源脚本语言。这种语法主要是将C语言、Java和Perl等语言的相关特点进行总结吸收形成自身特点，使得更加方便学习，当前广泛在Web领域进行使用。此语法的独特之处在于将C、Java、Perl这三种语言进行了归纳，并创新出
     <span style="color:#000000">
      JSP
     </span>
     具有自己特点的语法。它在执行动态网页方面相对于CGI和Perl语言更加高效。用动态页面如果是使用
     <span style="color:#000000">
      JSP
     </span>
     语言，那么相对于其它的编程语言，
     <span style="color:#000000">
      JSP
     </span>
     的执行主要是体现在将程序嵌入到HTML文档中，相对于其它语言，其执行的效率一般都领先于其它的语言。此外，
     <span style="color:#000000">
      JSP
     </span>
     编译后代码方面也是做得比较好的，一般其编译能够使代码的运行速度比一般语言都要快[7]。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     其主要特性有以下几个方面：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     1）免费性
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     和其它技术相比，
     <span style="color:#000000">
      JSP
     </span>
     本身免费且是开源代码。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     2）快捷性
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     使用
     <span style="color:#000000">
      JSP
     </span>
     编程的程序一般开发比较快，而且系统运行也比较快，也易于学习[2]。嵌入于HTML的这种方式，使其相对于其它开发语言编辑起来相对简单，实用性也更加强，对初学者是比较适用的。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     3）跨平台性强
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     由于
     <span style="color:#000000">
      JSP
     </span>
     是运行在服务器的脚本，可以运行在UNIX、LINUX、WINDOWS、MacOS、Android等平台
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     4）效率高
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     占用相当少的系统资源。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     是面向对象且独立于架构的动态脚本语言，使用
     <span style="color:#000000">
      JSP
     </span>
     和HTML编写WEB页面，
     <span style="color:#000000">
      JSP
     </span>
     服务器先对页面的
     <span style="color:#000000">
      JSP
     </span>
     代码进行解析，然后把处理后的结果连同HTML内容一起传送到访问端的浏览器。
     <span style="color:#000000">
      JSP
     </span>
     是一种源代码开放程序，拥有很好的跨平台兼容性。
     <span style="color:#000000">
      JSP
     </span>
     代码可以在Window系统以及许多版本的linux系统上完美运行。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     语言是不需要任何特殊的开发环境的，所以用户可以直接在WEB页面中输入
     <span style="color:#000000">
      JSP
     </span>
     命令代码。在WEB页面中，所有的
     <span style="color:#000000">
      JSP
     </span>
     代码都被放置在“&lt;?
     <span style="color:#000000">
      JSP
     </span>
     ”和“?&gt;”中。
    </p>
    <h3 style="text-align:left">
     <strong>
      <a name="_Toc16851">
      </a>
      2.2 MySQL  数据库
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     目前在国际上比较流行的数据库管理系统当中，主要有ORACLE、MySQL 、SYBASE、
     <span style="color:#000000">
      MySQL
     </span>
     、DB2等。MySQL  2008版本具有MySQL  Server 7.0版本的所有优点，并在此基础上添加很多更先进的功能，从而更加方便使用、更好的伸缩性，集成度高。由于是微软的产品，因此MySQL  Server 2008在兼容性可以与微软的其他相关产品进行很好的适应和结合，可以跨越Microsoft Windows 2008到Microsoft Windows 98等多种平台。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     MySQL  的T-MySQL 支持ANSI MySQL －92标准，是结构化查询语言MySQL 的一种。MySQL  Server在身份验证上主要采用以下方式：登录验证、数据库用户帐号许可验证、二级安全验证等等。且在同一时间支持MySQL  Server验证和Windows NT验证两种模式。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     微软的MySQL  Server是较为完善的C/S系统。MySQL  Server需要Windows NT平台，而Windows NT可以支持Intel 386，Power PC，MIPS，Alpha PC和RISC等平台，它使MySQL  Server具备非常完美的功能。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     MySQL  Server 2008中具有失败转移集群和数据库镜像技术，失败转移集群和数据库镜像技术可以确保系统向用户提交的应用系统高度可靠和可用。微软的MySQL  Server 2008在设计的时候引入了一套集成管理工具和管理应用编程接口（APIs），对系统提供易用性、可管理性和对大型MySQL  Server配置的支持。由于采用了数据库加密、相比以前版本更加安全的默认设置、对密码政策和安全模型进行了加强以及实行了许可化控制的细化，使得系统在运用MySQL  Server 2008对数据进行管理时能提供安全级别非常高的功能。
    </p>
    <h3 style="text-align:left">
     <strong>
      <a name="_Toc20006">
      </a>
      2.3
      <span style="color:#000000">
       jsp
      </span>
      技术
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     web 的服务平台
     <span style="color:#000000">
      jsp
     </span>
     是由Microsoft创建的。这个平台最主要的功能就是可以把信息、人和设备用一种规范的，具有个性化的方法连接起来。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     对于微软来说，
     <span style="color:#000000">
      jsp
     </span>
     技术是它的一个终极梦想，可以用这个技术实现以前完全完成不了的功能，这是一次跃进。今后，无论什么时刻，任意地点我们都可以在任何设备上进行互联信息交互网。在以后会在世界形成一个巨大的全球互联网的终端设备和服务中心，将会随时成为每个人的智能型助手。到那时哪怕你没有闲暇的时间去处理信息分析访问，或者说不需要访问的信息分析，一旦它得到指令，就会自主的查找相关知识智能化集成处理，更高效的处理我们交代的“使命”[9][10]。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （1）JSP Framework的两个组件
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     JSP框架具有两个不同的元素：包以及框架的软件开发包（SDK）。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     当然若是要单纯的要运行
     <span style="color:#000000">
      jsp
     </span>
     程序，我们完全没必要开发
     <span style="color:#000000">
      jsp
     </span>
     应用程序。我们只要在安装发型包组件这里下功夫。因为
     <span style="color:#000000">
      jsp
     </span>
     Framework在发行包上没有进行过编译，所以最后它只能编译
     <span style="color:#000000">
      JSP
     </span>
     应用程序。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （2）框架的特点
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     框架的两个主要特点：跨平台和语言。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （a）跨平台
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     框架程序能够随意跨越操作系统平台和硬件环境，不管在哪台电脑上只会编写一遍。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （b）跨语言
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     .
     <span style="color:#000000">
      JSP
     </span>
     框架里面一种共同的语言系统（CommonLanguageSystem ，称为CLS）。规定了许多标准化的
     <span style="color:#000000">
      jsp
     </span>
     Framework数据种类。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Web应用程序可以用
     <span style="color:#000000">
      JSP
     </span>
     研发出来。现在已然变成了互联网全新一代Web应用程序开发工具之一，一般的编程代码的人员已经慢慢的开始接受、使用它了。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （3）
     <span style="color:#000000">
      JSP
     </span>
     概述
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     由微软于1996年11月推出的Web应用技术的发展。不单单可以看作是一个全新的框架，并且我们在工作当中能采纳任何产品的制备和应用程序代码，生成高效、动态、交互的vbscript、javascript应用镶嵌到HTML当中。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （4）
     <span style="color:#000000">
      JSP
     </span>
     的特点
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (a)代码逻辑和显示页面分离
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      JSP
     </span>
     包含程序和美工两个部分，文件从头到尾html标签和&lt;%%&gt;代码交错。通常，编程员按照以前所计划的开始编写代码，接着是美工按照设计页面和编程员讨论该用哪种布局页面最为合适,最后编程员把设计镶嵌进程序。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (b)编译运行
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     我们采用了一种虚构的编程框架，把VBScript和JavaScript作为编程语言，如果发现操作系统挪用了
     <span style="color:#000000">
      JSP
     </span>
     源代码一次，那么我们肯定正在访问某个页面。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (c)缓存
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     由于
     <span style="color:#000000">
      JSP
     </span>
     的缓存基本上没有，通常只能修改数据库来提高性能。并介绍了缓存的概念，一个高速缓存页的一页当中，对一些参数的控制和缓存的页缓存，缓存里的数据可以在最大水平上，帮助我们克服这个难题。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (d)代码隐藏
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     编码后生成的代码更易于编写，具有更直观的布局，大大削减了开发和维护系统的困难性以及制作成本 。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     （5）
     <span style="color:#000000">
      JSP
     </span>
     的关键技术
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (a)WebForm
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Form(表单)是一个包含不同类型和种类的容器控件，控件必需直接或间接地连接。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     (b)Web Service
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Web Service是Web服务调用其他Web服务方法。简而言之，是你的网站可以利用其他网站的资源[11][12]。
    </p>
    <h3 style="text-align:left">
     <strong>
      <a name="_Toc27006">
      </a>
      2.4ssm简介
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ssm是
     <span style="color:#000000">
      Spring、SpringMVC、
     </span>
     <span style="color:#000000">
      MyBati
     </span>
     组合框架，能更好的实现MVC模式，充分发挥了各自的优势。其中通过Struts在表示层中处理页面请求和转发工作；通过Spring的依赖注入（Inversion of Control，IoC）技术降低了程序之间的耦合性，也使程序员养成用接口编程的好习惯；通过Hibernate完成在持久层与数据库的交互工作。三层框架的整合形成一个结构良好、功能强大，层次清晰的框架体系。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     1.Spring
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Spring是一个独立的MVC模式框架，很好的实现了显示处理和业务处理逻辑之间的低耦合性。但Struts主要是针对表示层设计的，对业务逻辑方面的支持不是很强，比如事务管理、安全检查等本属于系统层面的，实现代码也穿插在整个业务逻辑中，降低了代码的可维护性，而Spring则可以很好地解决这些问题。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     2.SpringMVC
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     SpringMVC也是独立的MVC模式框架，通过IoC技术，降低各组件之间的依赖，而Spring提供的面向切面编程（Aspect Oriented Programming，AOP）技术，在事务管理上具有独特优势。Spring框架不仅能有效地服务中间层对象，而且易于同其他层框架进行无缝集成，使各层之间相互独立，实现了在架构上各框架间的低耦合性。
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     3.
     <span style="color:#000000">
      MyBatis
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      MyBatis 是集合多种操作型关系数据库的概念和方法，它是一个强大的数据访问工具和解决方法
     </span>
     <span style="color:#000000">
      [5]
     </span>
     <span style="color:#000000">
      。对比 Hibernate，MyBatis 是一个半自动框架，Hibernate 则是一个全自动框架，无法直接维护 MySQL  。MyBatis在书写 MySQL   的灵活性很好，Hibernate 就不行，会比较麻烦。本文使用 MyBatis 作为海田在线商城这种多变性的项目，Hibernate 比较适合稳定的项目。而且 MyBatis 速度相对于 Hibernate 的速度要快些
     </span>
     。
    </p>
