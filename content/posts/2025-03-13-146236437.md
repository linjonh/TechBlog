    <h2>
     <a id="docker_0">

     docker基础
    </h2>
    <h4>
     <a id="_2">
     </a>
     一、镜像命令
    </h4>
    <h5>
     <a id="1_4">
     </a>
     1、镜像检索
    </h5>
    <pre><code class="prism language-dockerfile">docker search nginx
</code></pre>
    <h5>
     <a id="2_10">
     </a>
     2、镜像下载
    </h5>
    <pre><code class="prism language-dockerfile">docker pull nginx
</code></pre>
    <h5>
     <a id="3_16">
     </a>
     3、镜像查询列表
    </h5>
    <pre><code class="prism language-dockerfile">docker images
</code></pre>
    <h5>
     <a id="4_22">
     </a>
     4、删除镜像
    </h5>
    <pre><code class="prism language-dockerfile">docker rmi
</code></pre>
    <h4>
     <a id="_28">
     </a>
     二、容器命令
    </h4>
    <h5>
     <a id="1_30">
     </a>
     1、运行容器
    </h5>
    <pre><code>docker run 应用名
docker run -d 应用名 # 后台运行
docker run --name 应用名 # 取别名
docker run -p 80:80 应用名 # 端口映射
</code></pre>
    <h5>
     <a id="2_39">
     </a>
     2、查看运行中的容器
    </h5>
    <pre><code>docker ps
docker ps -a # 查看所有容器（包括停止运行的容器）
</code></pre>
    <h5>
     <a id="3_46">
     </a>
     3、重新运行容器
    </h5>
    <pre><code>docker start 应用名
</code></pre>
    <h5>
     <a id="4_52">
     </a>
     4、停止运行容器
    </h5>
    <pre><code>docker stop 应用名
</code></pre>
    <h5>
     <a id="5_58">
     </a>
     5、重启容器
    </h5>
    <pre><code>docker restart 应用名
</code></pre>
    <h5>
     <a id="6_64">
     </a>
     6、查看状态
    </h5>
    <pre><code>docker stats 应用名
</code></pre>
    <h5>
     <a id="7_70">
     </a>
     7、查看日志
    </h5>
    <pre><code>docker logs 应用名
</code></pre>
    <h5>
     <a id="8_76">
     </a>
     8、删除容器
    </h5>
    <pre><code>docker rm 应用名
</code></pre>
    <h5>
     <a id="9_82">
     </a>
     9、进入容器
    </h5>
    <pre><code>docker exec
docker exec -it 应用名 bash # 进行容器交互
</code></pre>
    <h4>
     <a id="_89">
     </a>
     三、分享命令
    </h4>
    <h5>
     <a id="1_91">
     </a>
     1、提交镜像
    </h5>
    <pre><code>docker commit 应用名
docker commit -m 'update' 应用名 # 更新描述
</code></pre>
    <h5>
     <a id="2_98">
     </a>
     2、镜像保存
    </h5>
    <pre><code>docker save
</code></pre>
    <h5>
     <a id="3_104">
     </a>
     3、镜像保存
    </h5>
    <pre><code>docker load 应用名
</code></pre>
    <h5>
     <a id="4docker_110">
     </a>
     4、docker登录
    </h5>
    <pre><code>docker login -u username -p password
</code></pre>
    <h5>
     <a id="5_116">
     </a>
     5、镜像命名
    </h5>
    <pre><code>docker tag
</code></pre>
    <h5>
     <a id="6_122">
     </a>
     6、镜像推送
    </h5>
    <pre><code>docker push
</code></pre>
    <h3>
     <a id="_128">
     </a>
     四、高阶命令
    </h3>
    <h5>
     <a id="1_130">
     </a>
     1、上线
    </h5>
    <pre><code>docker compose up -d
</code></pre>
    <h5>
     <a id="2_136">
     </a>
     2、下线
    </h5>
    <pre><code>docker compose down
</code></pre>
    <h5>
     <a id="3_142">
     </a>
     3、启动
    </h5>
    <pre><code>docker compose start 应用名
</code></pre>
    <h5>
     <a id="4_148">
     </a>
     4、停止
    </h5>
    <pre><code>docker compose stop 应用名
</code></pre>

     <a id="5_154">
     </a>
     5、扩容
    </h5>
    <pre><code>docker compose scale 应用=3 # 让应用启动3
</code></pre>
