    <blockquote>
     <h4>
      【题目描述】
     </h4>
     <p>
      n×m矩阵大部分元素是0的矩阵称为稀疏矩阵，假设有k个非0元素，则可把稀疏矩阵用K×3的矩阵简记之，其中第一列是行号，第二列是列号，第三列是该行、该列下的非元素的值。如：
     </p>
     <p>
      0 0 0 5
      <br/>
      0 2 0 0
      <br/>
      0 1 0 0
     </p>
     <p>
      写简记成：
     </p>
     <p>
      1 4 5      //第1行第4列有个数是5
      <br/>
      2 2 2      //第2行第2列有个数是2
      <br/>
      3 2 1      //第3行第2列有个数是1
     </p>
     <p>
      试编程读入一稀疏矩阵，转换成简记形式，并输出。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      第一行为n和m,n为行数，m为列数；
     </p>
     <p>
      接下来是一个n×m的矩阵，矩阵中每个正整数小于1000。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      输出矩阵的简记形式。简记形式有多种形式，输出一种正确形式即可。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      3 5
      <br/>
      0 0 0 0 5
      <br/>
      0 0 4 0 0
      <br/>
      1 0 0 0 1
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      3 5 1
      <br/>
      1 5 5
      <br/>
      2 3 4
      <br/>
      3 1 1
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e3 + 10;
int nums[N][N];

int main()
{
	int n, m; cin &gt;&gt; n &gt;&gt; m;
	for (int i = 1; i &lt;= n; i++)
	{
		for (int j = 1; j &lt;= m; j++)
		{
			cin &gt;&gt; nums[i][j];
			if (nums[i][j] != 0) cout &lt;&lt; i &lt;&lt; ' ' &lt;&lt; j &lt;&lt; ' ' &lt;&lt; nums[i][j] &lt;&lt; endl;
		}
	}

	return 0;
}</code></pre>
