    <br/>
    C++20引入了简写函数模板（Abbreviated Function Template），这是一种更简洁的函数模板声明方式，允许使用
    <code>
     auto
    </code>
    或带有约束的
    <code>
     auto
    </code>
    来代替显式的模板参数声明。
    <p>
    </p>
    <h3>
     <a id="_5">
     </a>
     简写函数模板的语法
    </h3>
    <p>
     当在函数参数列表中使用
     <code>
      auto
     </code>
     或带有约束的
     <code>
      auto
     </code>
     时，编译器会自动为每个占位符生成一个虚构的模板参数。例如：
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">void</span> <span class="token function">f1</span><span class="token punctuation">(</span><span class="token keyword">auto</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 等价于 template &lt;class T&gt; void f1(T);</span>
<span class="token keyword">void</span> <span class="token function">f2</span><span class="token punctuation">(</span>C1 <span class="token keyword">auto</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 等价于 template &lt;C1 T&gt; void f2(T)，如果C1是一个概念（Concept）</span>
<span class="token keyword">void</span> <span class="token function">f3</span><span class="token punctuation">(</span>C2 <span class="token keyword">auto</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 等价于 template &lt;C2... Ts&gt; void f3(Ts...)</span>
<span class="token keyword">void</span> <span class="token function">f4</span><span class="token punctuation">(</span><span class="token keyword">const</span> C3 <span class="token keyword">auto</span><span class="token operator">*</span><span class="token punctuation">,</span> C4 <span class="token keyword">auto</span><span class="token operator">&amp;</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 等价于 template &lt;C3 T, C4 U&gt; void f4(const T*, U&amp;)</span>
</code></pre>
    <p>
     此外，简写函数模板可以像普通函数模板一样进行特化。
    </p>

     <a id="_15">
     </a>
     示例代码

        <span class="token keyword">return</span> a <span class="token operator">+</span> b<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    <p>
     在这个例子中，
     <code>
      get_sum
     </code>
     是一个简写函数模板，可以接受任意类型的参数并返回它们的和。
     <code>
      get_sum2
     </code>
     则使用了概念（Concept）来限制参数类型，确保它们是整数或浮点数。
    </p>
    <h3>
     <a id="_41">
     </a>
     优点
    </h3>
    <p>
     简写函数模板的主要优点是语法更加简洁，减少了模板声明的冗余，同时保持了类型安全和灵活性。
