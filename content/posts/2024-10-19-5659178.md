# Linux游戏也疯狂

<!-- @page { margin: 2cm } P { margin-bottom: 0.21cm } -->

一般人似乎都认为，



Linux



上游戏太少，网吧客户不喜欢。



No



，情况并非如此，实际上，



Linux



游戏玩得也疯狂。

进入网吧，玩游戏者大多并不关心所用的操作系统是什么，他们关注的是游戏本身是否刺激，是否好玩。于是，一些


Linux


游戏开发者想出一个办法，将


Linux


游戏放到


DVD


光盘上或自启动


U


盘里面，让客户根本看不见所用的是什么操作系统，拿来只管玩就是了，充分体现了


Live


模式的优势。

当前，


3D


游戏可以说是最时兴了，那么，


Linux


上有没有


3D


游戏呢？当然有，而且很“酷”。


6


月


8


日，一张


Linux 3D


游戏光盘（或自启动


U


盘）发布了，名字叫：“



linuX-gamers.net live 0.9.6”


（



4,477MB


），适合成人玩；也有袖珍版



CD


（



700MB


），游戏少一些，适合少年儿童玩。对于



XP


游戏迷，保证不曾见识过此类



Linux 3D


游戏。

Linux 3D


游戏的硬件要求并不算高，有



512MB


内存即可，但是，必须有



3D


加速显卡，而且，多数游戏可以接入局域网多人玩（比如：多人对打游戏）。网吧老板不必改造本地网络环境便可做到“即插即用”。（说明：实际上，这种



3D


游戏只能在



Arch Linux


发行版上才运行起来，当然，这完全不管硬盘的事儿，而且，是完全免费的！）

实际上，



XP


平台上的游戏也并非“汉化”得很彻底，游戏迷们也能上手玩，而且，玩得很痛快。我在想，将来要是这玩意儿流行起来了，微软告网吧就有点儿“缺理”了，网吧客户既是



XP


用户，也是



Linux


用户，你微软能够分得清楚吗？我在想，设法送一批自启动



U


盘给网吧老板试试，他们一准会喜欢，一分钱不用花，岂不是天上掉馅饼？他们也许把我拒之门外，把我当成是大骗子来了。

一般的



Linux


游戏（非



3D


类）多的去了，一抓一大把，有专门网站抓这件事情。



Linux


游戏也是一个产业。我在想，设法把《



Ubuntu


实验室》恢复起来，专门研究一下



Linux


游戏以及



Linux


技术的教育问题。但是，在哪里办呢？
