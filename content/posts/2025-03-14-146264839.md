### 摘要

随着汽车租赁市场的日益扩大，高效、便捷的汽车租赁管理系统成为行业发展的关键。本研究旨在开发一款基于Asp.net的汽车租赁管理系统，以提升租赁业务的管理效率和客户体验。系统涵盖了用户管理、车辆管理、租赁订单管理、支付结算等多个功能模块，实现了汽车租赁业务的全流程管理。通过采用Asp.net框架和MySQL数据库，系统具备高性能、可扩展性和安全性，能够满足汽车租赁公司的日常业务需求。同时，系统界面简洁友好，用户操作便捷，提供了优质的租赁服务体验。

### 研究意义

基于Asp.net的汽车租赁管理系统的研究具有深远的意义。首先，该系统能够显著提升汽车租赁公司的管理效率，降低运营成本。通过自动化、智能化的管理流程，减少人工干预，提高业务处理速度，降低出错率。其次，系统能够提升客户体验，满足客户的个性化需求。用户可以通过系统方便地浏览和选择合适的租赁车辆，实时查看订单状态，进行在线支付和结算，享受便捷、高效的租赁服务。此外，该系统还有助于推动汽车租赁行业的数字化转型和智能化发展，提升整个行业的竞争力和服务水平。通过数据分析和挖掘，为汽车租赁公司提供决策支持，助力企业实现可持续发展。

### 研究目的

本研究的主要目的是开发一款功能完善、性能稳定的汽车租赁管理系统，以满足汽车租赁公司的日常业务需求。系统需要具备用户管理、车辆管理、租赁订单管理、支付结算等多个功能模块，实现汽车租赁业务的全流程管理。同时，系统需要具备良好的可扩展性和安全性，能够适应未来业务的发展和变化。通过优化业务流程，提升管理效率，降低运营成本，为客户提供更加便捷、高效的租赁服务。此外，本研究还希望通过系统的开发，为汽车租赁行业的数字化转型和智能化发展提供有益的探索和实践经验，推动整个行业的升级和发展。

using System;
using System.Web.UI;
using System.Data.SqlClient;

public partial class Login : Page
{
    protected void Page_Load(object sender, EventArgs e)
    {
        if (!IsPostBack)
        {
            // 初始化页面控件
        }
    }

    protected void btnLogin_Click(object sender, EventArgs e)
    {
        string username = txtUsername.Text.Trim();
        string password = txtPassword.Text.Trim();

        if (string.IsNullOrEmpty(username) || string.IsNullOrEmpty(password))
        {
            // 提示用户输入用户名和密码
            Response.Write("<script>alert('请输入用户名和密码！');</script>");
            return;
        }

        // 连接数据库进行用户验证
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "SELECT * FROM Users WHERE Username = @Username AND Password = @Password";
            using (SqlCommand command = new SqlCommand(query, connection))
            {
                command.Parameters.AddWithValue("@Username", username);
                command.Parameters.AddWithValue("@Password", password); // 注意：实际应用中应使用哈希存储密码并进行验证

                connection.Open();
                SqlDataReader reader = command.ExecuteReader();

                if (reader.Read())
                {
                    // 用户验证成功，重定向到主页
                    Response.Redirect("HomePage.aspx");
                }
                else
                {
                    // 用户验证失败
                    Response.Write("<script>alert('用户名或密码错误！');</script>");
                }

                reader.Close();
            }
        }
    }
}
