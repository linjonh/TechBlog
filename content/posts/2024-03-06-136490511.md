    <h3>
     <a id="_9">
     </a>
     🌼什么是软件？软件的特点？
    </h3>
    <p>
     <strong>
      <font color="plum">
       定义：软件，即计算机系统的程序及其文档。
      </font>
     </strong>
     (计算机科学技术百科全书)
     <br/>
     程序，是计算机任务的处理对象和处理规模的描述；
     <br/>
     文档，是为了方便了解程序所需的阐明性资料。
     <br/>
     (这里需要注意的是，提到软件，程序很容易想到，但是还有文档！！)
     <br/>
     1983年，IEEE给出软件的定义：
     <br/>
     计算机程序、方法、规则和相关文档资料以及在计算机上运行时所必须的程序。
     <br/>
     简单来说，它的定义即是：
     <strong>
      <font color="plum">
       软件=程序+数据+文档
      </font>
     </strong>
    </p>
    <p>
     <strong>
      <font color="plum">
       特点：
      </font>
     </strong>
     (理解即可)
    </p>
    <ul>
     <li>
      软件是无形的，是不可见的逻辑实体。
     </li>
     <li>
      软件是设计开发的，而不是生产制作的。
     </li>
     <li>
      软件在使用过程中没有磨损、老化的问题。
     </li>
     <li>
      软件是定制开发的。
     </li>
     <li>
      软件是复杂的。
     </li>
     <li>
      软件的开发成本高。
     </li>
     <li>
      软件易于制作。
     </li>
     <li>
      软件的开发与运行都离不开相关的计算机系统环境。
     </li>
     <li>
      软件开发工作涉及到很多社会因素。
     </li>
    </ul>
    <h3>
     <a id="_31">

     🌼什么是工程？
    </h3>
    <p>
     提到工程，了解到我们日常生活中的工程，比如说建楼房。项目管理部负责人员管理、进度管理、质量管理等，然后设计师画出图纸，然后进行一系列的施工，定位放线、土方开挖、垫层、…、等等。
    </p>

    <h3>
     <a id="_59">
     </a>
     🌼软件生命周期
    </h3>
    <p>
     <strong>
      <font color="indigo">
       软件生命周期：软件产品从形成概念开始，经过开发、使用和维护，直到最后退役的全过程。
      </font>
     </strong>
     （记）
    </p>
    <p>
     <strong>
      <font color="plum">
       软件生命周期分为三个时期：软件定义、软件开发和软件维护（也称为运行维护）。
      </font>
     </strong>

     每个时期又进一步分成若干个阶段。
