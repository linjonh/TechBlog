# python3 manage.py makemigrations执行完无反应的解决办法

简单记一下。
  
跟着网上的教程学习django，结果到了

```
$ python3 manage.py migrate   
$ python3 manage.py makemigrations TestModel  
$ python3 manage.py migrate TestModel  

```

执行后完全没有响应。
  
试了半天之后才发现原来在我电脑上
`python`
命令就是
`python3`
。如果执行
`python3`
会跳转到Microsoft商店。
