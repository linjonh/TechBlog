        <code>

       </li>
       <li>
        <code>
         css

       </li>
       <li>
        <code>

        </code>

     </li>
    </ul>
    <ul>
     <li>
      页面全部存放在pages, 而且pages目录只能存放页面
      <ul>
       <li>
        页面包括4个文件，
        <strong>
         pages/页面名/页面名.js(wxss,json,wxml)
        </strong>
        <ul>
         <li>
          4个文件的文件名必须一致
         </li>
         <li>
          4个文件
          <ul>
           <li>
            <code>
             .js
            </code>
            页面逻辑
           </li>
           <li>
            <code>
             .json
            </code>
            页面配置
           </li>
           <li>
            <code>
             .wxml
            </code>
            页面结构
           </li>
           <li>
            <code>
             .wxss
            </code>
            页面样式
           </li>
          </ul>
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <code>
       app.js
      </code>
      是小程序入口文件，先忽略掉
     </li>
     <li>
      <code>
       app.json
      </code>
      <ul>
       <li>
        <code>
         pages
        </code>
        配置的数组，是有序的。第一个就是模拟器显示的页面
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="prism language-js"><span class="token punctuation">{<!-- --></span>
	<span class="token comment">// 所有的页面路径都应该配置在这个数组里面，如果没有配置，如果跳转</span>

      <code>
       app.wxss

    <h3>
     <a id="_43">
     </a>
     基本标签的使用
    </h3>
    <p>
     <a href="https://developers.weixin.qq.com/miniprogram/dev/component/" rel="nofollow">
      组件文档
     </a>
    </p>
    <blockquote>
     <p>
      组成程序的页面的并不是HTML标签，而是小程序的组件
     </p>
    </blockquote>
    <ul>
     <li>
      <p>
       <code>
        text
       </code>
       ：显示文本的
      </p>
      <ul>
       <li>
        相当于是
        <code>
         span
        </code>
        标签，是行内元素
       </li>
      </ul>
     </li>
     <li>
      <p>
       <code>
        view
       </code>
       包裹作用
      </p>
      <ul>
       <li>
        相当于是
        <code>
         div
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <code>
        image
       </code>
       就是显示图片
      </p>
     </li>
    </ul>
    <ul>
     <li>
      默认的高度和宽度
     </li>
     <li>
      开发时，需要设置高度和宽度
     </li>
    </ul>
    <ul>
     <li>
      <code>
       button
      </code>
      按钮
      <ul>
       <li>
        <code>
         size:mini
        </code>
        :表示小的按钮
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="prism language-js"><span class="token operator">&lt;</span>button<span class="token operator">&gt;</span>默认的按钮<span class="token operator">&lt;</span><span class="token operator">/</span>button<span class="token operator">&gt;</span>
<span class="token operator">&lt;</span>button size<span class="token operator">=</span><span class="token string">"mini"</span><span class="token operator">&gt;</span>小按钮<span class="token operator">&lt;</span><span class="token operator">/</span>button<span class="token operator">&gt;</span>

     语法——数据绑定
    </h3>
    <ul>
     <li>
      <code>
       {
       <!-- -->
       {}}
      </code>
      ：数据绑定使用 Mustache 语法（双大括号）将变量包起来
     </li>
    </ul>
    <ol>
     <li>
      <strong>
       文本渲染：
      </strong>
     </li>
    </ol>
    <pre><code class="prism language-js">	<span class="token operator">&lt;</span><span class="token operator">!</span><span class="token operator">--</span> 文本渲染，使用<span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span><span class="token punctuation">}</span> <span class="token operator">--</span><span class="token operator">&gt;</span>
	<span class="token operator">&lt;</span>text<span class="token operator">&gt;</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span>msg<span class="token punctuation">}</span><span class="token punctuation">}</span><span class="token operator">&lt;</span><span class="token operator">/</span>text<span class="token operator">&gt;</span>
</code></pre>
    <ol start="2">
     <li>
      <strong>
       属性绑定
      </strong>
     </li>
    </ol>
    <pre><code class="prism language-js">	<span class="token operator">&lt;</span><span class="token operator">!</span><span class="token operator">--</span> 属性绑定<span class="token punctuation">,</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span><span class="token punctuation">}</span> <span class="token operator">--</span><span class="token operator">&gt;</span>
	<span class="token operator">&lt;</span>image src<span class="token operator">=</span><span class="token string">"{<!-- -->{imgUrl}}"</span><span class="token operator">&gt;</span><span class="token operator">&lt;</span><span class="token operator">/</span>image<span class="token operator">&gt;</span>
</code></pre>
    <ol start="3">
     <li>
      <strong>
       条件渲染
      </strong>
      <ol>
       <li>
        <code>
         wx:if
        </code>
        和
        <code>
         wx:else
        </code>
        是固定写法，不能改
       </li>
       <li>
        条件满足就渲染，如果不满足不移除
        <code>
         dom
        </code>
       </li>
      </ol>
     </li>
    </ol>
    <pre><code class="prism language-js">	<span class="token operator">&lt;</span><span class="token operator">!</span><span class="token operator">--</span> 条件判断<span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span><span class="token punctuation">}</span>
	如果条件满足就展示对应的dom <span class="token operator">--</span><span class="token operator">&gt;</span>
	<span class="token operator">&lt;</span>view wx<span class="token punctuation">:</span><span class="token keyword">if</span><span class="token operator">=</span><span class="token string">"{<!-- -->{isHandsome}}"</span><span class="token operator">&gt;</span>很帅		<span class="token operator">&lt;</span><span class="token operator">/</span>view<span class="token operator">&gt;</span>
	<span class="token operator">&lt;</span>view wx<span class="token punctuation">:</span><span class="token keyword">else</span><span class="token operator">&gt;</span>丑<span class="token constant">B</span><span class="token operator">&lt;</span><span class="token operator">/</span>view<span class="token operator">&gt;</span>
</code></pre>
    <ol start="4">
     <li>
      <strong>
       列表渲染
      </strong>
      <ol>
       <li>
        <code>
         wx:for
        </code>
        是固定写法
       </li>
       <li>
        默认的元素别名
        <code>
         item
        </code>
        ，索引别名
        <code>
         index
        </code>
       </li>
       <li>
        <code>
         wx:for-item
        </code>
        可以指定元素别名
       </li>
       <li>
        <code>
         ？
        </code>
        的值以两种形式提供
        <ol>
         <li>
          字符串，代表在
          <code>
           for
          </code>
          循环的
          <code>
           array
          </code>
          中
          <code>
           item
          </code>
          的某个 property，该 property 的值需要是列表中唯一的字符串或数字，且不能动态改变。
         </li>
         <li>
          保留关键字
          <code>
           *this
          </code>
          代表在 for 循环中的
          <code>
           item
          </code>
          本身，这种表示需要
          <code>
           item
          </code>
          本身是一个唯一的字符串或者数字
         </li>
        </ol>
       </li>
      </ol>
     </li>
    </ol>
    <pre><code class="prism language-js">	<span class="token operator">&lt;</span><span class="token operator">!</span><span class="token operator">--</span> 遍历<span class="token punctuation">,</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span><span class="token punctuation">}</span>

	<span class="token operator">&lt;</span>view wx<span class="token punctuation">:</span><span class="token keyword">for</span><span class="token operator">=</span><span class="token string">"{<!-- -->{movieList}}"</span> wx<span class="token punctuation">:</span>key<span class="token operator">=</span><span class="token string">"*this"</span><span class="token operator">&gt;</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span>item<span class="token punctuation">}</span><span class="token punctuation">}</span><span class="token operator">--</span><span class="token punctuation">{<!-- --></span><span class="token punctuation">{<!-- --></span>index<span class="token punctuation">}</span><span class="token punctuation">}</span><span class="token operator">&lt;</span><span class="token operator">/</span>view<span class="token operator">&gt;</span>
</code></pre>
    <ul>
     <li>
      <strong>
       注意点
      </strong>

       data

    <h3>
     <a id="_144">
     </a>
     语法模块化
    </h3>
    <p>
     小程序中也支持模块化的概念:
     <a href="http://es6.ruanyifeng.com/#docs/module" rel="nofollow">
      es6模块化
     </a>
    </p>
    <ul>
     <li>
      作用：
      <ul>
       <li>
        把一些公共的代码抽离为作为一个单独
        <code>
         js
        </code>
       </li>
      </ul>
     </li>
     <li>
      使用：
      <ul>
       <li>
        直接使用ES6的模块化
       </li>
      </ul>
     </li>
    </ul>
    <ul>
     <li>
      传送门：
      <a href="https://es6.ruanyifeng.com/#docs/module" rel="nofollow">
       https://es6.ruanyifeng.com/#docs/module
      </a>
     </li>
     <li>
      使用方法
      <ul>
       <li>
        在模块js中暴露方法
        <code>
         属性
        </code>
        或者
        <code>
         对象
        </code>
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="prism language-js"><span class="token comment">// 对外暴露</span>
<span class="token keyword">export</span> <span class="token keyword">default</span> sayHello
</code></pre>
    <ul>
     <li>
      需要使用模块的js里面引入
     </li>
    </ul>
    <pre><code class="prism language-js"><span class="token comment">// 引入模块</span>
<span class="token keyword">import</span> sayHello <span class="token keyword">from</span> <span class="token string">'../../utils/hello'</span>
</code></pre>
    <p>
     <strong>
      注意点：
     </strong>
     <br/>
     <code>
      import
     </code>
     路径
     <strong>
      不支持绝对路径
     </strong>
     ，所以写相对路径即可
    </p>
    <h3>

     方法中的this
    </h3>
    <blockquote>
     <p>
      小程序中的
      <code>

      <code>
       页面实例
      </code>
      ，可以通过 this 取得对象上的属性与方法
     </p>
    </blockquote>
    <ul>
     <li>

      <code>
       data
      </code>
      属性
      <code>
       this.data.属性名
      </code>
     </li>
     <li>
      方法访问方法
      <code>
       this.方法名
      </code>
     </li>
     <li>
      改变
      <code>
       data
      </code>
      属性
      <code>
       this.setData({属性名:值})
      </code>
     </li>
    </ul>
