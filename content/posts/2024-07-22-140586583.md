# 开源智能助手平台Dify是什么？

### 1.背景

对于国内小公司，怎样通过Ai 将内部流程、产品重新做一次，从而提高人效、给客户带来价值，这是老板们在考虑的问题 ？ 当前市面上的你大模型例如：通义千问、文心一言、kimi、智谱清言、盘古 等，底层能力都非常强大（千亿级参数），但各个模型使用有非常大的差异，大家在用的时候，没法快速解决自己的问题。 能否有个平台能够聚合各个模型，而且针对专业领域，可以训练自己的机器人 ？ 答案是：有，目前最好用的是 Coze 和 Dify。本文重点介绍一下Dify。

### 2.Dify 基础知识

简介： Dify（Define Modify）是一个开源的 LLM 应用开发平台。其直观的界面结合了 AI 工作流、RAG 管道、Agent、模型管理、可观测性功能等，让你可以快速从原型到生产。

地址：https://dify.ai/zh

### 3.功能介绍

| 功能 | 应用场景 | 说明 |
| --- | --- | --- |
| 工作流 | 可以将各种ai组件、智能体串联起来 | 在画布上构建和测试功能强大的 AI 工作流程 |
| 全面的模型支持 | 可选择底层模型 | 与数百种专有/开源 LLMs 以及数十种推理提供商和自托管解决方案无缝集成，涵盖 GPT、Mistral、Llama3 以及任何与 OpenAI API 兼容的模型。 |
| Prompt IDE | 修改后马上可以调试结果 | 用于制作提示、比较模型性能以及向基于聊天的应用程序添加其他功能（如文本转语音）的直观界面 |
| RAG Pipeline | 支持常见文件扫描 | 广泛的 RAG 功能，涵盖从文档摄入到检索的所有内容，支持从 PDF、PPT 和其他常见文档格式中提取文本的开箱即用的支持 |
| Agent 智能体 | 快速集成Agent工具 | 可以基于 LLM 函数调用或 ReAct 定义 Agent，并为 Agent 添加预构建或自定义工具 |
| 后端即服务 | API调用 | 所有 Dify 的功能都带有相应的 API，因此你可以轻松地将 Dify 集成到自己的业务逻辑中。 |

### 4.功能比较（国内）

| 框架 | 优势 |
| --- | --- |
| Coze | 1）上手简单 2）完全免费 3）方便各类协同软件关联，可以实现用户在不同软件间快速切换 4）更适合C端用户使用 |
| Dify | 1）开源平台，用户可以选择自己需要的模型 2）提供不同模型，对比功能 3）更适合团队使用 |

### 5.总结

Dify 适合团队使用，可以解决团队内复杂的业务流程。 若个人使用，推荐Coze ，当前是免费的，后期用户量到一定水位后，大概率会收费，毕竟服务器、大模型都需要成本。
