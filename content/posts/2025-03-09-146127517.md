     scala传递匿名函数简化的原则
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      以一下这个标准调用为例说明问题。
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f((name: String) =&gt; {
      <!-- -->
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      println(name)
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      })
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <strong>
      <span style="color:#000000">
       <strong>
        i、
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        参数的类型可以省略，会根据形参进行自动的推导
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f((name) =&gt; {
      <!-- -->
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      println(name)
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      })
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <strong>
      <span style="color:#000000">
       <strong>
        ii、
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        类型省略之后，如果只有一个参数，则圆括号可以省略；其他情况：
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000">
       <strong>
        没有参数和参数超过 1 的永远不能省略圆括号
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        。
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f( name =&gt; {
      <!-- -->
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      println(name)
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      })
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <strong>
      <span style="color:#000000">
       <strong>
        iii、
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        匿名函数如果只有一行，则大括号也可以省略
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f( name =&gt; println(name) )
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <strong>
      <span style="color:#000000">
       <strong>
        iv、
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        如果
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#ff0000">
       <strong>
        参数只出现一次
       </strong>
      </span>
     </strong>
     <strong>
      <span style="color:#000000">
       <strong>
        ，则参数可以省略且后面的参数可以用_代替
       </strong>
      </span>
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f( println(_) )
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <span style="color:#000000">
      推导：对于多个参数，但是
     </span>
     <strong>
      <span style="color:#000000">
       <strong>
        参数只出现一次
       </strong>
      </span>
     </strong>
     <span style="color:#000000">
      ，那么每个参数可以使用
     </span>
     <span style="color:#000000">
      _代替，但是下划线的顺序和参数的顺序是一一对应的。
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      v
     </span>
     <span style="color:#000000">
      、
     </span>
     <span style="color:#000000">
      如果可以推断出当前传入的表达式是一个函数体，而不是调用语句，可以直接省略下划线
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="color:#000000">
      f( println )
     </span>
    </p>
