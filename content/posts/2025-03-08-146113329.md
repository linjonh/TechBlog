    <h3>
     <a id="2_69">

     2.解释
    </h3>
    <p>
     Vue 3 组合式 API 中
     <code>
      computed
     </code>
     返回数组的使用场景与原理
    </p>
    <h4>
     <a id="_72">
     </a>
     为什么使用数组？
    </h4>
    <p>
     在 Vue 3 的组合式 API 中，
     <code>
      computed
     </code>
     函数可以返回
     <strong>
      任意类型
     </strong>
     的值（基本类型、对象、数组等）。当需要表示
     <strong>
      一组相似的动作或配置项
     </strong>
     时，使用数组是自然且高效的选择，因为：
    </p>
    <ol>
     <li>
      <strong>
       动态迭代渲染
      </strong>
      <br/>
      数组可以通过
      <code>
       v-for
      </code>
      指令直接遍历，动态生成 UI 元素（如按钮、菜单项等）。
     </li>
     <li>
      <strong>
       结构化数据
      </strong>
      <br/>
      数组中的每个元素遵循相同的格式（例如
      <code>
       { text, disabled }
      </code>
      ），便于统一处理逻辑。
     </li>
     <li>
      <strong>
       响应式依赖
      </strong>
      <br/>
      当数组中的某个属性（如
      <code>
       disabled
      </code>
      ）依赖响应式数据时，
      <code>
       computed
      </code>
      会自动追踪变化并更新。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_85">
     </a>
     示例代码分析
    </h4>
    <pre><code class="prism language-javascript"><span class="token keyword">const</span> actions <span class="token operator">=</span> <span class="token function">computed</span><span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token operator">=&gt;</span> <span class="token punctuation">[</span>
  <span class="token punctuation">{<!-- --></span> <span class="token literal-property property">text</span><span class="token operator">:</span> <span class="token string">'查看处方'</span><span class="token punctuation">,</span> <span class="token literal-property property">disabled</span><span class="token operator">:</span> <span class="token operator">!</span>props<span class="token punctuation">.</span>item<span class="token punctuation">.</span>prescriptionId <span class="token punctuation">}</span><span class="token punctuation">,</span>
  <span class="token punctuation">{<!-- --></span> <span class="token literal-property property">text</span><span class="token operator">:</span> <span class="token string">'删除订单'</span> <span class="token punctuation">}</span>
<span class="token punctuation">]</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     <strong>
      解释：
     </strong>
     <br/>
     <strong>
      数组元素：每个对象描述一个动作的配置（如按钮的文本和禁用状态）。
      <br/>
      响应式逻辑：disabled 的值依赖 props.item.prescriptionId，当该值变化时，计算属性会重新计算。
      <br/>
      默认行为：未显式定义 disabled 的动作默认为 false（即不禁用）。
     </strong>
    </p>
