    <p>
     在 IntelliJ IDEA 中配置 Groovy 环境可以分为以下几个步骤
    </p>
    <h4>
     <a id="1__Groovy__3">
     </a>
     <strong>
      1. 安装 Groovy 插件
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       步骤
      </strong>
      ：
      <ol>
       <li>
        打开 IntelliJ IDEA，进入菜单栏：
        <strong>
         File → Settings
        </strong>
        （Windows/Linux）或
        <strong>
         IntelliJ IDEA → Preferences
        </strong>
        （Mac）。
       </li>
       <li>
        在设置窗口中，选择
        <strong>
         Plugins
        </strong>
        。
       </li>
       <li>
        在 Marketplace 中搜索
        <strong>
         Groovy
        </strong>
        ，找到官方插件后点击
        <strong>
         Install
        </strong>
        。
       </li>
       <li>
        安装完成后重启 IDEA。
       </li>
      </ol>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2__Groovy_SDK_12">
     </a>
     <strong>
      2. 配置 Groovy SDK
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       步骤
      </strong>
      ：
      <ol>
       <li>
        <strong>
         创建/打开项目
        </strong>
        ：新建项目或打开现有项目。
       </li>
       <li>
        进入项目配置：
        <ul>
         <li>
          <strong>
           Windows/Linux
          </strong>
          ：
          <strong>
           File → Project Structure
          </strong>
         </li>
         <li>
          <strong>
           Mac
          </strong>
          ：
          <strong>
           IntelliJ IDEA → Project Structure
          </strong>
         </li>
        </ul>
       </li>
       <li>
        在
        <strong>
         Project Settings → Modules
        </strong>
        中，选择当前模块。
       </li>
       <li>
        点击
        <strong>
         Dependencies
        </strong>
        标签页，点击
        <strong>
         + → Groovy SDK
        </strong>
        。
       </li>
       <li>
        选择已安装的 Groovy SDK 或通过
        <strong>
         Download
        </strong>
        按钮自动下载（需联网）。
        <ul>
         <li>
          如果本地已安装 Groovy，选择路径（如：
          <code>
           /usr/local/groovy
          </code>
          或自定义路径）。
         </li>
        </ul>
       </li>
       <li>
        确认 SDK 添加成功，点击
        <strong>
         Apply → OK
        </strong>
        。
       </li>
      </ol>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3__Groovy__26">
     </a>
     <strong>
      3. 创建 Groovy 文件
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       步骤
      </strong>
      ：
      <ol>
       <li>
        右键项目中的目标目录（如
        <code>
         src
        </code>
        ），选择
        <strong>
         New → Groovy Class/Groovy Script
        </strong>
        。
       </li>
       <li>
        输入文件名（例如
        <code>
         HelloWorld
        </code>
        ），创建
        <code>
         .groovy
        </code>
        文件。
       </li>
      </ol>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4__Groovy__33">
     </a>
     <strong>
      4. 编写并运行 Groovy 代码
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       示例代码
      </strong>
      ：
      <pre><code class="prism language-groovy"><span class="token comment">// 如果是 Groovy Script（无需类定义）</span>
println <span class="token interpolation-string"><span class="token string">"Hello, Groovy!"</span></span>

<span class="token comment">// 如果是 Groovy Class</span>
<span class="token keyword">class</span> <span class="token class-name">HelloWorld</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">static</span> <span class="token keyword">void</span> <span class="token function">main</span><span class="token punctuation">(</span>String<span class="token punctuation">[</span><span class="token punctuation">]</span> args<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        println <span class="token interpolation-string"><span class="token string">"Hello, Groovy!"</span></span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <strong>
       运行
      </strong>
      ：
      <ul>
       <li>
        直接右键代码编辑器，选择
        <strong>
         Run ‘文件名’
        </strong>
        。
       </li>
       <li>
        或点击代码左侧的绿色箭头运行。
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="5__52">
     </a>
     <strong>
      5. 验证配置
     </strong>
    </h4>
    <ul>
     <li>
      如果运行后控制台输出
      <code>
       Hello, Groovy!
      </code>
      ，说明环境配置成功。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_57">
     </a>
     <strong>
      常见问题排查
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       找不到 Groovy SDK
      </strong>
      ：确保在
      <strong>
       Project Structure
      </strong>
      中正确添加 SDK 路径。
     </li>
     <li>
      <strong>
       无法创建 Groovy 文件
      </strong>
      ：检查插件是否启用（
      <strong>
       Settings → Plugins → Installed
      </strong>
      确认 Groovy 插件已勾选）。
     </li>
     <li>
      <strong>
       依赖冲突
      </strong>
      ：如果是 Maven/Gradle 项目，可在
      <code>
       pom.xml
      </code>
      或
      <code>
       build.gradle
      </code>
      中添加 Groovy 依赖。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="MavenGradle_64">
     </a>
     <strong>
      附加：通过构建工具配置（Maven/Gradle）
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       Maven
      </strong>
      ：在
      <code>
       pom.xml
      </code>
      中添加：
      <pre><code class="prism language-xml"><span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>dependency</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>groupId</span><span class="token punctuation">&gt;</span></span>org.codehaus.groovy<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>groupId</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>artifactId</span><span class="token punctuation">&gt;</span></span>groovy-all<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>artifactId</span><span class="token punctuation">&gt;</span></span>
    <span class="token tag"><span class="token tag"><span class="token punctuation">&lt;</span>version</span><span class="token punctuation">&gt;</span></span>3.0.9<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>version</span><span class="token punctuation">&gt;</span></span> <span class="token comment">&lt;!-- 替换为最新版本 --&gt;</span>
<span class="token tag"><span class="token tag"><span class="token punctuation">&lt;/</span>dependency</span><span class="token punctuation">&gt;</span></span>
</code></pre>
     </li>
     <li>
      <strong>
       Gradle
      </strong>
      ：在
      <code>
       build.gradle
      </code>
      中添加：
      <pre><code class="prism language-groovy">dependencies <span class="token punctuation">{<!-- --></span>
    implementation <span class="token string">'org.codehaus.groovy:groovy-all:3.0.9'</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <p>
     按照以上步骤操作，即可在 IntelliJ IDEA 中轻松配置 Groovy 环境并开始开发！
    </p>
