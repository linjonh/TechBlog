    <h4 id="1%E3%80%81Python%E4%B8%8B%E8%BD%BD%EF%BC%88%E7%89%88%E6%9C%AC%E8%87%AA%E5%B7%B1%E9%80%89%E6%8B%A9%EF%BC%89">
     <strong>
      1、Python下载（版本自己选择）
     </strong>
    </h4>
    <p style="text-indent:33px;">
     <strong>
      <a href="http://www.python.org/download/" rel="nofollow">
       Python版本官网下载
      </a>
     </strong>
     <strong>
      <a href="http://www.python.org/" rel="nofollow">
       Python官网
      </a>
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      <a href="https://www.anaconda.com/download/" rel="nofollow">
       Anaconda官网下载
      </a>
     </strong>
     <span style="color:#393939;">
      <strong>
       <a href="http://python.jobbole.com/86236/" rel="nofollow">
        Anaconda python 使用博客
       </a>
      </strong>
     </span>
    </p>
    <h4 id="2%E3%80%81%E5%9C%A8Windows%E4%B8%8A%E5%AE%89%E8%A3%85Python">
     <strong>
      2、在Windows上安装Python
     </strong>
    </h4>
    <p style="text-indent:33px;">
     <strong>
      根据你的Windows版本（64位还是32位）从Python的官方网站下载Python 3.6对应的
     </strong>
     <a href="https://www.python.org/ftp/python/3.5.2/python-3.5.2-amd64.exe" rel="nofollow">
      <strong>
       64位安装程序
      </strong>
     </a>
     <strong>
      或
     </strong>
     <a href="https://www.python.org/ftp/python/3.5.2/python-3.5.2.exe" rel="nofollow">
      <strong>
       32位安装程序
      </strong>
     </a>
     <strong>
      ，然后，运行下载的EXE安装包；
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      特别要注意勾上
     </strong>
     <span style="color:#df402a;">
      <strong>
       Add Python 3.6 to PATH
      </strong>
     </span>
     <strong>
      ，然后点“Install Now”即可完成安装；
     </strong>
    </p>
    <h4 id="3%E3%80%81%E8%BF%90%E8%A1%8CPython">
     <strong>
      3、运行Python
     </strong>
    </h4>
    <p style="text-indent:33px;">
     <strong>
      安装成功后，打开命令提示符窗口(win+r)，敲入python后，看到下面的画面，就说明Python安装成功！
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/e9978f3772e8394c04f370fe25595b90.png"/>
    </p>
    <p style="text-indent:33px;">
     <strong>
      看到提示符
     </strong>
     <span style="color:#df402a;">
      <strong>
       &gt;&gt;&gt;
      </strong>
     </span>
     <strong>
      就表示已经在
     </strong>
     <span style="color:#df402a;">
      <strong>
       Python交互式环境中
      </strong>
     </span>
     <strong>
      了，可以输入任何Python代码，回车后会立刻得到执行结果。输入
     </strong>
     <span style="color:#df402a;">
      <strong>
       exit()
      </strong>
     </span>
     <strong>
      并回车，就可以退出Python交互式环境（直接关掉命令行窗口也可以）！
     </strong>
    </p>
    <p>
     <span style="color:#df402a;">
      <strong>
       注意：
      </strong>
     </span>
     <strong>
      得到一个错误：‘python’ 不是内部或外部命令，也不是可运行的程序或批处理文件。
     </strong>
    </p>
    <p style="text-align:center;">
     <img alt="" class="has" src="https://i-blog.csdnimg.cn/blog_migrate/6d70bd468d5993b8dca4a9c072affbd2.png"/>
    </p>
    <p style="text-indent:33px;">
     这是因为Windows会根据一个Path的环境变量设定的路径去查找python.exe，如果没找到，就会报错。如果在安装时
     <span style="color:#df402a;">
      漏掉
     </span>
     了勾选
     <span style="color:#df402a;">
      Add Python 3.6 to PATH
     </span>
     ，那就要手动把python.exe所在的路径添加到Path中 ，或者把Python安装程序重新运行一遍，务必记得勾上Add Python 3.5 to PATH
    </p>
    <h4 id="4%E3%80%81%E8%BF%90%E8%A1%8CPython%E7%9A%84%E6%96%B9%E5%BC%8F">
     <strong>
      4、运行Python的方式
     </strong>
    </h4>
    <p>
     <strong>
      4.1 Python解释器（CPython）
     </strong>
    </p>
    <p style="text-indent:33px;">
     从
     <a href="https://www.python.org/" rel="nofollow">
      Python官方网站
     </a>
     下载并安装好Python 3.6后，就直接获得了一个官方版本的解释器：
     <span style="color:#df402a;">
      <strong>
       CPython
      </strong>
     </span>
     。这个解释器是用C语言开发的，所以叫CPython。
     <strong>
      在命令行下运行python就是启动CPython解释器
     </strong>
    </p>
    <p>
     <strong>
      4.2 集成开发环境（IDE：Integrated Development Environment）: PyCharm
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      PyCharm 是由 JetBrains 打造的一款
     </strong>
     <span style="color:#df402a;">
      <strong>
       Python IDE
      </strong>
     </span>
     <strong>
      ，支持 macOS、 Windows、 Linux 系统。
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      PyCharm 功能 : 调试、语法高亮、Project管理、代码跳转、智能提示、自动完成、单元测试、版本控制……
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      <a href="https://www.jetbrains.com/pycharm/download/" rel="nofollow">
       PyCharm 下载
      </a>
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      PyCharm最新2017激活：server选项里边输入
     </strong>
     <span style="color:#003884;">
      <strong>
      </strong>
     </span>
     <span style="color:#003884;">
      <strong>
       <a href="http://idea.imsxm.com/" rel="nofollow">
        http://idea.imsxm.com/
       </a>
      </strong>
     </span>
     <strong>
      或者：
     </strong>
     <a href="http://elporfirio.com:1017/" rel="nofollow">
      <span style="color:#003884;">
       <strong>
        http://elporfirio.com:1017/
       </strong>
      </span>
     </a>
    </p>
    <p>
     <strong>
      4.3 Eclipse编译器中进行Python
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      在Eclipse中安装
     </strong>
     <span style="color:#df402a;">
      <strong>
       PyDev
      </strong>
     </span>
     <strong>
      插件，然后可以在Eclipse编译器中进行Python编程
     </strong>
     <span style="color:#003884;">
      <strong>
       <a href="http://pydev.org/updates/" rel="nofollow">
        Pydev
       </a>
      </strong>
     </span>
    </p>
    <h4 id="5%E3%80%81%E7%94%A8%E6%96%87%E6%9C%AC%E7%BC%96%E8%BE%91%E5%99%A8%E5%86%99Python%E7%A8%8B%E5%BA%8F">
     <strong>
      5、用文本编辑器写Python程序
     </strong>
    </h4>
    <p style="text-indent:33px;">
     <strong>
      用文本编辑器写Python程序，然后保存为后缀为
     </strong>
     <span style="color:#df402a;">
      <strong>
       .py
      </strong>
     </span>
     <strong>
      的文件，就可以用Python直接运行这个程序了
     </strong>
    </p>
    <p>
     <strong>
      5.1 安装文本编辑器
     </strong>
    </p>
    <p style="text-indent:33px;">
     一个是
     <a href="http://www.sublimetext.com/" rel="nofollow">
      <span style="color:#003884;">
       <strong>
        Sublime Text
       </strong>
      </span>
     </a>
     ，免费使用，但是不付费会弹出提示框
    </p>
    <p style="text-indent:33px;">
     一个是
     <a href="http://notepad-plus-plus.org/" rel="nofollow">
      <span style="color:#003884;">
       <strong>
        Notepad++
       </strong>
      </span>
     </a>
     ，免费使用，有中文界面
    </p>
    <p style="text-indent:33px;">
     <span style="color:#df402a;">
      <strong>
       注意
      </strong>
     </span>
     ：用哪个都行，但是绝对不能用Word和Windows自带的记事本。Word保存的不是纯文本文件，而记事本会自作聪明地在文件开始的地方加上几个特殊字符（UTF-8 BOM），结果会导致程序运行出现莫名其妙的错误
    </p>
    <p>
     <strong>
      5.2 安装好文本编辑器后写入代码
     </strong>
    </p>
    <p>
     <strong>
      例：print('hello, world')
     </strong>
    </p>
    <p style="text-indent:33px;">
     注意：print前面不要有任何空格。然后，选择一个目录，例如C:\work，把文件保存为hello.py，就可以打开命令行窗口，把当前目录切换到hello.py所在目录，就可以运行这个程序了（
     <strong>
      C:\Users\IEUser&gt;
     </strong>
     <span style="color:#df402a;">
      <strong>
       python
      </strong>
     </span>
     <strong>
     </strong>
     <span style="color:#df402a;">
      <strong>
       D:\hello.py
      </strong>
     </span>
     ）
    </p>
    <p style="text-indent:33px;">
     <span style="color:#df402a;">
      文件名只能是英文字母、数字和下划线的组合
     </span>
    </p>
    <p>
     <strong>
      5.3 Python的交互模式和直接运行.py文件的什么区别
     </strong>
    </p>
    <p style="text-indent:33px;">
     <strong>
      直接输入python进入交互模式
     </strong>
     ，相当于启动了Python解释器，但是等待你一行一行地输入源代码，每输入一行就执行一行。
    </p>
    <p style="text-indent:33px;">
     <strong>
      直接运行.py文件
     </strong>
     相当于启动了Python解释器，然后一次性把.py文件的源代码给执行了，你是没有机会以交互的方式输入源代码的。
    </p>
