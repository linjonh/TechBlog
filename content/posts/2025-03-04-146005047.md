     【网络】实现电脑与笔记本电脑之间的直接网络连接

  </div>

    <p>
     要实现电脑与笔记本电脑之间的直接网络连接，可以通过有线或无线两种方式。以下是详细的步骤指南：
    </p>
    <hr/>
    <h4>
     <a id="_4">
     </a>
     <strong>
      一、有线直连（通过网线）
     </strong>
    </h4>
    <h5>
     <a id="1__5">
     </a>
     <strong>
      1. 准备工具
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       网线
      </strong>
      ：使用交叉网线（适用于旧设备）或普通直连网线（现代设备一般支持自动翻转功能）。
     </li>
     <li>
      <strong>
       两台设备
      </strong>
      ：确保电脑和笔记本均配备以太网接口。
     </li>
    </ul>
    <h5>
     <a id="2__9">
     </a>
     <strong>
      2. 连接步骤
     </strong>
    </h5>
    <ol>
     <li>
      <strong>
       物理连接
      </strong>
      ：用网线将两台设备的网口相连。
     </li>
     <li>
      <strong>
       配置IP地址
      </strong>
      （手动设置）：
      <ul>
       <li>
        <strong>
         Windows系统
        </strong>
        ：
        <ul>
         <li>
          打开
          <code>
           控制面板 &gt; 网络和共享中心 &gt; 更改适配器设置
          </code>
          。
         </li>
         <li>
          右键点击以太网适配器 → 选择
          <strong>
           属性
          </strong>
          → 双击
          <strong>
           IPv4
          </strong>
          。
         </li>
         <li>
          设置静态IP（例如）：
          <ul>
           <li>
            电脑：
            <code>
             192.168.1.1
            </code>
            ，子网掩码
            <code>
             255.255.255.0
            </code>
           </li>
           <li>
            笔记本：
            <code>
             192.168.1.2
            </code>
            ，子网掩码同上。
           </li>
          </ul>
         </li>
        </ul>
       </li>
       <li>
        <strong>
         macOS
        </strong>
        ：
        <ul>
         <li>
          进入
          <code>
           系统偏好设置 &gt; 网络
          </code>
          → 选择以太网 → 配置IPv4为
          <code>
           手动
          </code>
          ，填写类似IP地址。
         </li>
        </ul>
       </li>
      </ul>
     </li>
     <li>
      <strong>
       测试连通性
      </strong>
      ：
      <ul>
       <li>
        打开命令提示符（CMD）或终端，输入
        <code>
         ping &lt;对方IP&gt;
        </code>
        （如
        <code>
         ping 192.168.1.2
        </code>
        ）。
       </li>
       <li>
        收到回复说明连接成功。
       </li>
      </ul>
     </li>
    </ol>
    <h5>
     <a id="3__24">
     </a>
     <strong>
      3. 共享文件或网络
     </strong>
    </h5>
    <ul>
     <li>
      <strong>
       共享文件夹
      </strong>
      ：
      <ul>
       <li>
        右键文件夹 → 属性 → 共享 → 设置共享权限。
       </li>
       <li>
        通过
        <code>
         \\&lt;对方IP&gt;
        </code>
        访问共享内容（如
        <code>
         \\192.168.1.2
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       共享网络
      </strong>
      （如笔记本需共享WiFi给电脑）：
      <ul>
       <li>
        在已连接WiFi的笔记本上，进入网络适配器设置 → 右键WiFi适配器 → 属性 → 共享 → 勾选允许其他设备通过此连接上网，选择以太网接口。
       </li>
      </ul>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_33">
     </a>
     <strong>
      二、无线直连（无需路由器）
     </strong>
    </h4>
    <h5>
     <a id="1_Windows_34">
     </a>
     <strong>
      1. 创建临时无线网络（Windows）
     </strong>
    </h5>
    <ol>
     <li>
      打开
      <strong>
       命令提示符（管理员）
      </strong>
      ，输入：
      <pre><code class="prism language-bash">netsh wlan <span class="token builtin class-name">set</span> hostednetwork <span class="token assign-left variable">mode</span><span class="token operator">=</span>allow <span class="token assign-left variable">ssid</span><span class="token operator">=</span>网络名称 <span class="token assign-left variable">key</span><span class="token operator">=</span>密码
netsh wlan start hostednetwork
</code></pre>
     </li>
     <li>
      另一台设备搜索该WiFi并连接。
     </li>
    </ol>
    <h5>
     <a id="2_WiFi_Direct_42">
     </a>
     <strong>
      2. 使用WiFi Direct（部分设备支持）
     </strong>
    </h5>
    <ul>
     <li>
      在支持WiFi Direct的笔记本或电脑上启用该功能，另一台设备搜索并配对连接。
     </li>
    </ul>
    <h5>
     <a id="3__45">
     </a>
     <strong>
      3. 设置共享与访问
     </strong>
    </h5>
    <ul>
     <li>
      同有线方式，设置IP地址和共享文件夹。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="_50">
     </a>
     <strong>
      三、注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       防火墙设置
      </strong>
      ：确保防火墙允许文件和打印机共享。
     </li>
     <li>
      <strong>
       网线类型
      </strong>
      ：若连接失败，尝试更换为交叉线。
     </li>
     <li>
      <strong>
       物理连接检查
      </strong>
      ：网口指示灯亮起表示连接正常。
     </li>
     <li>
      <strong>
       系统差异
      </strong>
      ：不同操作系统（Windows/macOS/Linux）需调整对应网络设置。
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="_58">
     </a>
     <strong>
      四、常见问题
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       无法Ping通
      </strong>
      ：检查IP是否在同一子网，关闭防火墙临时测试。
     </li>
     <li>
      <strong>
       共享文件夹无法访问
      </strong>
      ：确保共享权限设置为“Everyone”，并启用网络发现（网络和共享中心 → 启用网络发现和文件共享）。
     </li>
    </ul>
    <p>
     通过以上方法，可实现快速数据传输或网络共享，适用于无路由器环境的大文件传输或紧急网络共享需求。
    </p>
