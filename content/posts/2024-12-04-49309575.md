    <p>
     [摘要]本文介绍LUA中的数学库，包括abs、acos、asin、floor、log、max、min、random、sqrt等函数介绍。
     <br/>
     Lua5.1中数学库的所有函数如下表：
     <br/>
     math.pi 为圆周率常量 = 3.14159265358979323846
     <br/>
     %———————————————————————%
     <br/>
     abs 含义：取绝对值
     <br/>
     举例：math.abs(-15)
     <br/>
     结果：15
     <br/>
     %———————————————————————%
     <br/>
     sinh 含义：双曲线正弦函数
     <br/>
     举例：math.sinh(0.5)
     <br/>
     结果：0.5210953
     <br/>
     %———————————————————————%
     <br/>
     sin 含义：正弦函数
     <br/>
     举例：math.sin(math.rad(30))
     <br/>
     结果：0.5
     <br/>
     %———————————————————————%
     <br/>
     acos 含义：反余弦函数
     <br/>
     举例：math.acos(0.5)
     <br/>
     结果：1.04719755
     <br/>
     %———————————————————————%
     <br/>
     asin 含义：反正弦函数
     <br/>
     举例：math.asin(0.5)
     <br/>
     结果：0.52359877
     <br/>
     %———————————————————————%
     <br/>
     tanh 含义：双曲线正切函数
     <br/>
     举例：math.tanh(0.5)
     <br/>
     结果：0.46211715
     <br/>
     %———————————————————————%
     <br/>
     tan 含义：正切函数
     <br/>
     举例：math.tan(0.5)
     <br/>
     结果：0.5463024
     <br/>
     %———————————————————————%
     <br/>
     atan2 含义：x / y的反正切值
     <br/>
     举例：math.atan2(90.0, 45.0)
     <br/>
     结果：1.10714871
     <br/>
     %———————————————————————%
     <br/>
     atan 含义：反正切函数
     <br/>
     举例：math.atan(0.5)
     <br/>
     结果：0.463647609
     <br/>
     %———————————————————————%
     <br/>
     cosh 含义：双曲线余弦函数
     <br/>
     举例：math.cosh(0.5)
     <br/>
     结果：1.276259652
     <br/>
     %———————————————————————%
     <br/>
     cos 含义：余弦函数
     <br/>
     举例：math.cos(0.5)
     <br/>
     结果：0.87758256
     <br/>
     %———————————————————————%
     <br/>
     ceil 含义：不小于x的最大整数
     <br/>
     举例：math.ceil(5.8)
     <br/>
     结果：6
     <br/>
     %———————————————————————%
     <br/>
     floor 含义：不大于x的最大整数
     <br/>
     举例：math.floor(5.6)
     <br/>
     结果：5
     <br/>
     %———————————————————————%
     <br/>
     fmod （mod） 含义：取模运算
     <br/>
     举例：math.mod(14, 5)
     <br/>
     结果：4
     <br/>
     %———————————————————————%
     <br/>
     modf 含义：把数分为整数和小数
     <br/>
     举例：math.modf(15.98)
     <br/>
     结果：15 98
     <br/>
     %———————————————————————%
     <br/>
     deg 含义：弧度转角度
     <br/>
     举例：math.deg(math.pi)
     <br/>
     结果：180
     <br/>
     %———————————————————————%
     <br/>
     rad 含义：角度转弧度
     <br/>
     举例：math.rad(180)
     <br/>
     结果：3.14159265358
     <br/>
     %———————————————————————%
     <br/>
     exp 含义：计算以e为底x次方值
     <br/>
     举例：math.exp(2)
     <br/>
     结果：2.718281828
     <br/>
     %———————————————————————%
     <br/>
     pow 含义：得到x的y次方
     <br/>
     举例：math.pow(2, 5)
     <br/>
     结果：32
     <br/>
     %———————————————————————%
     <br/>
     sqrt 含义：开平方函数
     <br/>
     举例：math.sqrt(16)
     <br/>
     结果：4
     <br/>
     %———————————————————————%
     <br/>
     frexp 含义：把双精度数val分解为数字部分（尾数）和
     <br/>
     以2为底的指数n，即val=x*2n
     <br/>
     举例：math.frexp(10.0)
     <br/>
     结果：0.625 4
     <br/>
     %———————————————————————%
     <br/>
     ldexp 含义：计算value * 2的n次方
     <br/>
     举例：math.ldexp(10.0, 3)
     <br/>
     结果：80 = 10 * (2 ^3)
     <br/>
     %———————————————————————%
     <br/>
     log10 含义：计算以10为基数的对数
     <br/>
     举例：math.log10(100)
     <br/>
     结果：2
     <br/>
     %———————————————————————%
     <br/>
     log 含义：计算一个数字的自然对数
     <br/>
     举例：math.log(2.71)
     <br/>
     结果：0.9969
     <br/>
     %———————————————————————%
     <br/>
     max 含义：取得参数中最大值
     <br/>
     举例：math.max(2.71, 100, -98, 23)
     <br/>
     结果：100
     <br/>
     %———————————————————————%
     <br/>
     min 含义：取得参数中最小值
     <br/>
     举例：math.min(2.71, 100, -98, 23)
     <br/>
     结果：-98
     <br/>
     %———————————————————————%
     <br/>
     random 含义：获取随机数
     <br/>
     举例：math.random(1, 100)
     <br/>
     举例：math.random(100)
     <br/>
     结果：获取1-100的随机数
     <br/>
     %———————————————————————%
     <br/>
     randomseed 含义：设置随机数种子
     <br/>
     math.randomseed(os.time())
     <br/>
     在使用math.random函数之前必须使用此函数设置随机数种子
    </p>
