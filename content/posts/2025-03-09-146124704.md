    <h4>
     <a id="_123">
     </a>
     <strong>
      注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       时间格式
      </strong>
      ：插入时间时需使用
      <code>
       TIMESTAMPTZ
      </code>
      格式（带时区）。
     </li>
     <li>
      <strong>
       批量操作
      </strong>
      ：使用
      <code>
       PreparedStatement
      </code>
      和批处理提高写入性能。
     </li>
     <li>
      <strong>
       连接池
      </strong>
      ：生产环境中建议使用 HikariCP 或 PostgreSQL 的连接池。
     </li>
     <li>
      <strong>
       SQL 方言
      </strong>
      ：TimescaleDB 支持部分 PostgreSQL 扩展语法（如
      <code>
       CREATE HYPERTABLE
      </code>
      ）。
     </li>
    </ol>
