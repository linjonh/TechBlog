    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000">
      <strong style="background-color:rgb(255,255,255)">
       一、CacheCloud是什么？
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000; background-color:rgb(255,255,255)">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000; background-color:rgb(255,255,255)">
      CacheCloud提供一个Redis云管理平台：实现多种类型(Redis Standalone、Redis Sentinel、Redis Cluster)自动部署、解决Redis实例碎片化现象、提供完善统计、监控、运维功能、减少运维成本和误操作，提高机器的利用率，提供灵活的伸缩性，提供方便的接入客户端。
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000; background-color:rgb(255,255,255)">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000">
      <strong style="background-color:rgb(255,255,255)">
       二、CacheCloud提供哪些功能？
      </strong>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000; background-color:rgb(255,255,255)">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:KaiTi_GB2312; font-size:24px; color:#330000; background-color:rgb(255,255,255)">
     </span>
    </p>
    <ul style="padding-left:2em; margin-top:0px; margin-bottom:16px">
     <li style="">
      <span style="font-weight:600">
       监控统计：
      </span>
      提供了机器、应用、实例下各个维度数据的监控和统计界面。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       一键开启：
      </span>
      Redis Standalone、Redis Sentinel、Redis Cluster三种类型的应用，无需手动配置初始化。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       Failover：
      </span>
      支持哨兵,集群的高可用模式。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       伸缩：
      </span>
      提供完善的垂直和水平在线伸缩功能。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       完善运维：
      </span>
      提供自动运维和简化运维操作功能，避免纯手工运维出错。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       方便的客户端
      </span>
      方便快捷的客户端接入。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       元数据管理：
      </span>
      提供机器、应用、实例、用户信息管理。
     </li>
     <li style="margin-top:0.25em">
      <span style="font-weight:600">
       流程化：
      </span>
      提供申请，运维，伸缩，修改等完善的处理流程
     </li>
     <li style="margin-top:0.25em">
      一键导入：
      <a href="http://cachecloud.github.io/2016/04/17/%E5%B7%B2%E5%AD%98%E5%9C%A8Redis%E6%8E%A5%E5%85%A5CacheCloud/" rel="nofollow noopener noreferrer" style="text-decoration:none" target="_blank">
       一键导入已经存在Redis
      </a>
     </li>
    </ul>
    <strong>

    </p>
    <p style="margin:10px auto; padding-top:0px; padding-bottom:0px; line-height:1.5">
