    <h3>
     <a id="Kotlinwhen_0">
     </a>
     Kotlin关键字
     <code>
      when
     </code>
     的详细用法
    </h3>
    <p>
     在Kotlin中，
     <code>
      when
     </code>
     是一个强大的控制流语句，相当于其他语言中的
     <code>
      switch
     </code>
     语句，但更加强大且灵活。本文将详细讲解
     <code>
      when
     </code>
     的用法及其常见场景，并与Java的
     <code>
      switch
     </code>
     语句进行对比。
    </p>
    <h4>
     <a id="_4">
     </a>
     一、基本语法
    </h4>
    <p>
     基本的
     <code>
      when
     </code>
     语法如下：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> number <span class="token operator">=</span> <span class="token number">3</span>
<span class="token keyword">when</span> <span class="token punctuation">(</span>number<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token number">1</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"One"</span></span><span class="token punctuation">)</span>
    <span class="token number">2</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Two"</span></span><span class="token punctuation">)</span>
    <span class="token number">3</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Three"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Unknown"</span></span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_18">
     </a>
     二、匹配多个条件
    </h4>
    <p>
     可以使用逗号
     <code>
      ,
     </code>
     分隔多个匹配条件：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> day <span class="token operator">=</span> <span class="token string-literal singleline"><span class="token string">"Saturday"</span></span>
<span class="token keyword">when</span> <span class="token punctuation">(</span>day<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token string-literal singleline"><span class="token string">"Saturday"</span></span><span class="token punctuation">,</span> <span class="token string-literal singleline"><span class="token string">"Sunday"</span></span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Weekend"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Weekday"</span></span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_30">
     </a>
     三、范围检查
    </h4>
    <p>
     使用
     <code>
      in
     </code>
     关键字可以判断一个值是否在一个范围或集合内：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> score <span class="token operator">=</span> <span class="token number">85</span>
<span class="token keyword">when</span> <span class="token punctuation">(</span>score<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">in</span> <span class="token number">90</span><span class="token operator">..</span><span class="token number">100</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Excellent"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">in</span> <span class="token number">75</span><span class="token operator">..</span><span class="token number">89</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Good"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">in</span> <span class="token number">60</span><span class="token operator">..</span><span class="token number">74</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Pass"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Fail"</span></span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_44">
     </a>
     四、类型检查
    </h4>
    <p>
     <code>
      when
     </code>
     还可以用于类型检查，相当于Java中的
     <code>
      instanceof
     </code>
     ：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> obj<span class="token operator">:</span> Any <span class="token operator">=</span> <span class="token string-literal singleline"><span class="token string">"Hello Kotlin"</span></span>
<span class="token keyword">when</span> <span class="token punctuation">(</span>obj<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">is</span> String <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"It's a String"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">is</span> Int <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"It's an Integer"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Unknown type"</span></span><span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="when_57">
     </a>
     五、不带参数的when
    </h4>
    <p>
     <code>
      when
     </code>
     也可以不带参数，这时可以用作更灵活的条件语句：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> number <span class="token operator">=</span> <span class="token number">15</span>
<span class="token keyword">when</span> <span class="token punctuation">{<!-- --></span>
    number <span class="token operator">%</span> <span class="token number">3</span> <span class="token operator">==</span> <span class="token number">0</span> <span class="token operator">&amp;&amp;</span> number <span class="token operator">%</span> <span class="token number">5</span> <span class="token operator">==</span> <span class="token number">0</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"FizzBuzz"</span></span><span class="token punctuation">)</span>
    number <span class="token operator">%</span> <span class="token number">3</span> <span class="token operator">==</span> <span class="token number">0</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Fizz"</span></span><span class="token punctuation">)</span>
    number <span class="token operator">%</span> <span class="token number">5</span> <span class="token operator">==</span> <span class="token number">0</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span><span class="token string-literal singleline"><span class="token string">"Buzz"</span></span><span class="token punctuation">)</span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token function">println</span><span class="token punctuation">(</span>number<span class="token punctuation">)</span>
<span class="token punctuation">}</span>
</code></pre>
    <h4>
     <a id="_71">
     </a>
     六、作为表达式使用
    </h4>
    <p>
     Kotlin的
     <code>
      when
     </code>
     还可以直接作为表达式返回一个值：
    </p>
    <pre><code class="prism language-kotlin"><span class="token keyword">val</span> score <span class="token operator">=</span> <span class="token number">78</span>
<span class="token keyword">val</span> result <span class="token operator">=</span> <span class="token keyword">when</span> <span class="token punctuation">(</span>score<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">in</span> <span class="token number">90</span><span class="token operator">..</span><span class="token number">100</span> <span class="token operator">-&gt;</span> <span class="token string-literal singleline"><span class="token string">"A"</span></span>
    <span class="token keyword">in</span> <span class="token number">80</span><span class="token operator">..</span><span class="token number">89</span> <span class="token operator">-&gt;</span> <span class="token string-literal singleline"><span class="token string">"B"</span></span>
    <span class="token keyword">in</span> <span class="token number">70</span><span class="token operator">..</span><span class="token number">79</span> <span class="token operator">-&gt;</span> <span class="token string-literal singleline"><span class="token string">"C"</span></span>
    <span class="token keyword">else</span> <span class="token operator">-&gt;</span> <span class="token string-literal singleline"><span class="token string">"D"</span></span>

</code></pre>
    <h4>
     <a id="Javaswitch_86">
     </a>
     七、与Java的switch语句对比
    </h4>
    <p>
     与Java的
     <code>
      switch
     </code>
     相比，Kotlin的
     <code>
      when
     </code>
     更加灵活强大：
    </p>
    <ul>
     <li>
      Java的
      <code>
       switch
      </code>
      仅支持基本类型（如int、byte、short、char）、它们的包装类、枚举类型和String类型。
     </li>
     <li>
      Kotlin的
      <code>
       when
      </code>
      支持任意类型的表达式，包括范围检查、类型检查、集合检查等更高级的特性。
     </li>
     <li>
      Kotlin的
      <code>
       when
      </code>
      可以作为表达式直接返回值，而Java的
      <code>
       switch
      </code>
      语句在Java 14之前无法做到。
     </li>
    </ul>
    <h4>
     <a id="_94">
     </a>
     八、注意事项
    </h4>
    <ul>
     <li>
      <code>
       when
      </code>
      表达式必须是详尽的，或者必须包含
      <code>
       else
      </code>
      分支。
     </li>
     <li>
      在使用类型检查时，编译器会自动进行智能类型转换。
     </li>
    </ul>
    <h4>
     <a id="_99">
     </a>
     九、总结
    </h4>
    <p>
     Kotlin的
     <code>
      when
     </code>
     关键字通过更简洁的语法和强大的功能，极大提升了代码的可读性和可维护性。合理使用
     <code>
      when
     </code>
     可以让代码更优雅、更清晰。
    </p>
    <p>
     希望本文能够帮助你更好地理解并掌握
     <code>
      when
     </code>
     的用法，提升你的Kotlin编程技能。
    </p>
