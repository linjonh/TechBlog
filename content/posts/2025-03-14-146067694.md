    <h3 name="1.%20%E5%BC%95%E8%A8%80">
     <span style="color:#ed7976">
      1. 引言
     </span>
    </h3>
    <p>
     在 Flutter 中，
     <code>
      ElevatedButton
     </code>
     是一个常用的按钮组件，
     <span style="color:#fe2c24">
      它带有背景颜色和阴影效果，适用于强调操作
     </span>
     。
    </p>
    <p>
     <code>
      ElevatedButton
     </code>
     继承自
     <code>
      ButtonStyleButton
     </code>
     ，相比
     <code>
      TextButton
     </code>
     和
     <code>
      OutlinedButton
     </code>
     ，其默认样式更
     <span style="color:#fe2c24">
      具有视觉层次感，适用于主要操作按钮
     </span>
     。本文将详细介绍
     <code>
      ElevatedButton
     </code>
     的用法、主要属性及自定义方法。
    </p>
    <h3 id="2.%20ElevatedButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95" name="2.%20ElevatedButton%20%E7%9A%84%E5%9F%BA%E6%9C%AC%E7%94%A8%E6%B3%95">
     <span style="color:#ed7976">
      2. ElevatedButton 的基本用法
     </span>
    </h3>
    <ul>
     <li>
      <p>
       <strong>
        必填参数
       </strong>
       :
      </p>
      <ul>
       <li>
        <p>
         <code>
          onPressed
         </code>
         : 点击回调函数（设为
         <code>
          null
         </code>
         ，按钮会变为不可点击状，按钮禁用）。
        </p>
       </li>
       <li>
        <p>
         <code>
          child
         </code>
         : 子组件（通常为
         <code>
          Text
         </code>
         或
         <code>
          Icon
         </code>
         ）。
        </p>
       </li>
      </ul>
     </li>
    </ul>
    <pre><code class="language-Dart">ElevatedButton(
  onPressed: () {
    print('ElevatedButton 被点击');
  },
  child: Text('点击 ElevatedButton'),
)
</code></pre>
    <h3 id="3.%20%E4%B8%BB%E8%A6%81%E5%B1%9E%E6%80%A7" name="3.%20%E4%B8%BB%E8%A6%81%E5%B1%9E%E6%80%A7">
     <span style="color:#ed7976">
      3. 主要属性
     </span>
    </h3>
    <table>
     <thead>
      <tr>
       <th>
        属性
       </th>
       <th>
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         onPressed
        </code>
       </td>
       <td>
        按钮点击时的回调函数
       </td>
      </tr>
      <tr>
       <td>
        <code>
         onLongPress
        </code>
       </td>
       <td>
        长按时触发的回调
       </td>
      </tr>
      <tr>
       <td>
        <code>
         child
        </code>
       </td>
       <td>
        按钮的内容，如
        <code>
         Text
        </code>
        或
        <code>
         Icon
        </code>
       </td>
      </tr>
      <tr>
       <td>
        <code>
         style
        </code>
       </td>
       <td>
        自定义按钮样式
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     示例：
    </p>
    <pre><code>ElevatedButton(
  onPressed: () {},
  onLongPress: () =&gt; print('长按按钮'),
  child: Text('长按试试'),
)
</code></pre>
    <h3 id="4.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%8C%89%E9%92%AE%E6%A0%B7%E5%BC%8F" name="4.%20%E8%87%AA%E5%AE%9A%E4%B9%89%E6%8C%89%E9%92%AE%E6%A0%B7%E5%BC%8F">
     <span style="color:#ed7976">
      4. 自定义按钮样式
     </span>
    </h3>
    <h4 id="4.1%20%E4%BF%AE%E6%94%B9%E8%83%8C%E6%99%AF%E9%A2%9C%E8%89%B2%E5%92%8C%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2" name="4.1%20%E4%BF%AE%E6%94%B9%E8%83%8C%E6%99%AF%E9%A2%9C%E8%89%B2%E5%92%8C%E6%96%87%E6%9C%AC%E9%A2%9C%E8%89%B2">
     <span style="color:#956fe7">
      4.1 修改背景颜色和文本颜色
     </span>
    </h4>
    <pre><code class="language-Dart">ElevatedButton(
  style: ElevatedButton.styleFrom(
    foregroundColor: Colors.white, // 文字颜色
    backgroundColor: Colors.blue, // 背景色
  ),
  onPressed: () {},
  child: Text('点击 ElevatedButton'),
)
</code></pre>
    <h4 id="4.2%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%BD%A2%E7%8A%B6%E5%92%8C%E8%BE%B9%E6%A1%86" name="4.2%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%BD%A2%E7%8A%B6%E5%92%8C%E8%BE%B9%E6%A1%86">
     <span style="color:#956fe7">
      4.2 修改按钮形状和边框
     </span>
    </h4>
    <pre><code class="language-Dart">ElevatedButton(
  style: ElevatedButton.styleFrom(
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(10),
    ),
  ),
  。。。。。
)
</code></pre>
    <h4 id="4.3%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%A4%A7%E5%B0%8F" name="4.3%20%E4%BF%AE%E6%94%B9%E6%8C%89%E9%92%AE%E5%A4%A7%E5%B0%8F">
     <span style="color:#956fe7">
      4.3 修改按钮大小
     </span>
    </h4>
    <pre><code class="language-Dart">ElevatedButton(
  style: ElevatedButton.styleFrom(
    minimumSize: Size(200, 80),
  ),
  onPressed: () {},
  child: Text('大按钮'),
)
</code></pre>
    <h4 id="4.4%20%E9%98%B4%E5%BD%B1%E6%8E%A7%E5%88%B6" name="4.4%20%E9%98%B4%E5%BD%B1%E6%8E%A7%E5%88%B6">
     <span style="color:#956fe7">
      4.4 阴影控制
     </span>
    </h4>
    <pre><code class="language-Dart">ElevatedButton.styleFrom(
  elevation: 10, // 默认阴影
  shadowColor: Colors.blue, // 阴影颜色
)</code></pre>
    <h4 id="4.5%C2%A0%20%E6%B0%B4%E6%B3%A2%E7%BA%B9%E6%95%88%E6%9E%9C" name="4.5%C2%A0%20%E6%B0%B4%E6%B3%A2%E7%BA%B9%E6%95%88%E6%9E%9C">
     <span style="color:#956fe7">
      4.5  水波纹效果
     </span>
    </h4>
    <pre><code class="language-Dart">ElevatedButton.styleFrom(
  splashFactory: InkRipple.splashFactory, // 默认水波纹
  overlayColor: Colors.blue.withValues(green: 0.1), // 直接使用颜色
)</code></pre>

    <h3 id="5.%20%E7%BB%93%E8%AE%BA" name="5.%20%E7%BB%93%E8%AE%BA">
     <span style="color:#ed7976">
      5. 结论
     </span>
    </h3>
    <p>
     <code>
      ElevatedButton
     </code>
     是 Flutter 中常见的按钮组件，适用于强调操作。通过
     <code>
      style
     </code>
     属性可以灵活地修改背景色、形状、大小等。掌握
     <code>
      ElevatedButton
     </code>
     的使用可以帮助开发者创建更美观的交互界面。
    </p>
