    <blockquote>
     <h4>
      【题目描述】
     </h4>
     <p>
      已知一个n×n(2≤n≤20)的矩阵（方阵），把矩阵二条对角线上的元素值加上10，然后输出这个新矩阵。
     </p>
     <h4>
      【输入】
     </h4>
     <p>
      第一行为n;
     </p>
     <p>
      下面为一个n×n，矩阵中各正整数小于100。
     </p>
     <h4>
      【输出】
     </h4>
     <p>
      输出新的矩阵。共n行，每行n数，各数之间用一个空格隔开。
     </p>
     <h4>
      【输入样例】
     </h4>
     <p>
      5
      <br/>
      13 66 53 89 55
      <br/>
      11 72 61 62 46
      <br/>
      90 10 38 10 55
      <br/>
      95 20 43 27 19
      <br/>
      60 12 66 16 96
     </p>
     <h4>
      【输出样例】
     </h4>
     <p>
      23 66 53 89 65
      <br/>
      11 82 61 72 46
      <br/>
      90 10 48 10 55
      <br/>
      95 30 43 37 19
      <br/>
      70 12 66 16 106
     </p>
    </blockquote>
    <h4>
     【题解代码】
    </h4>
    <pre><code class="language-cpp">#include&lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e2 + 10;
int nums[N][N];

int main()
{
	int n; cin &gt;&gt; n;
	for (int i = 1; i &lt;= n; i++)
	{
		for (int j = 1; j &lt;= n; j++)
		{
			cin &gt;&gt; nums[i][j];
			if (i == j || i + j == n + 1)nums[i][j] += 10;
			cout &lt;&lt; nums[i][j] &lt;&lt; ' ';
		}
		cout &lt;&lt; endl;
	}

	return 0;
}</code></pre>
