    <h2>
     <a id="_0">
     </a>
     一、介绍
    </h2>
    <p>
     Bandicam（班迪录屏）是一款简单好用的电脑屏幕录制软件，也是一款可以高清流畅地录游戏，录视频的功能强大的录屏工具。
    </p>
    <h2>
     <a id="_2">
     </a>
     二、常用设置
    </h2>
    <h3>
     <a id="21__3">
     </a>
     2.1 视频保存位置
    </h3>
    <p>
     选择
     <strong>
      常规
     </strong>
     –在
     <strong>
      输出文件夹
     </strong>
     处可选择录制视频保存的位置。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/f9fcdb2a13c148878597ad2de8f41a1d.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="22__6">
     </a>
     2.2 快捷键设置
    </h3>
    <p>
     选择
     <strong>
      录像
     </strong>
     –在
     <strong>
      录制
     </strong>
     区域可以设置录制快捷键。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a5193abb3a7149a0b0a573eb7111d534.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="23__11">
     </a>
     2.3 鼠标点击效果设置
    </h3>
    <p>
     上方功能区选择鼠标图标，可添加鼠标点击效果。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6081889b1acf4c789cb9d036fd8e35fe.jpeg#pic_center">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/aa241540f1564578bea438c5469a5e40.jpeg#pic_center"/>
     </img>
    </p>
    <h2>
     <a id="_18">
     </a>
     三、视频录制
    </h2>
    <h3>
     <a id="31____19">
     </a>
     3.1 屏幕录制 - 全屏
    </h3>
    <p>
     全屏录制模式可以录制电脑整个屏幕，如果有多个显示器，需要选择要录制的显示器。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7479f58c281945a2a5cfbb9b53663fd0.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="32____22">
     </a>
     3.2 屏幕录制 - 矩形窗口
    </h3>
    <p>
     矩形窗口录制模式可以录制屏幕的局部区域，可根据需要调整录制区域大小。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/63a4c6d65983439095422f73ed761fdc.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="33____25">
     </a>
     3.3 屏幕录制 - 追随鼠标
    </h3>
    <p>
     追随鼠标模式是录制鼠标周围移动的区域所显示画面的录制方法。
     <br/>
     此方法在制作视频教程，或想要录制屏幕上的文本以使其清晰可见时使用会事半功倍。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bc9cf6dc6edb41c2b9333c550212ec02.jpeg#pic_center"/>
    </p>
    <h3>
     <a id="34__29">
     </a>
     3.4 游戏录制模式
    </h3>
    <p>
     游戏录制模式是录制在 DirectX（或 OpenGL）技术上运行的程序（例如在线游戏）时使用的方法。
     <br/>
     <strong>
      游戏录制模式
     </strong>
     可以录制比
     <strong>
      屏幕录制模式
     </strong>
     更高质量的视频，但只有使用【DirectX/OpenGL/UWP/Vulkan】技术运行的录制目标才能在
     <strong>
      游戏录制模式
     </strong>
     中录制。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/8a9fc9942357455689d2a1ad9ee93213.jpeg#pic_center"/>
    </p>
