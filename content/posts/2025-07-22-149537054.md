# PHICOMM(斐讯)N1盒子 - Armbian25.05(Debian 12)刷入U盘/EMMC



## PHICOMM(斐讯)N1盒子 - Armbian25.05(Debian 12)刷入U盘/EMMC

---

环境：

```
系统：Armbian 25.05
cpu: amlogic s905d (arm A53 4核4线程)
基于系统：Debian 12 bookworm
内核：6.1
glibc: 2.36
gcc: 12.2.0
unixbench(6.0.0)性能测试: 单进程334.5 4进程1149.6

```

所需文件：

```
Armbian_25.05.0_amlogic_s905d_bookworm_6.1.137_server_2025.05.05.img.gz (SHA: e882de1421306c9f16bea9a90863b995bf67144892d76a1c3cc0b9c532fe2524)

```

推荐选择基于Debian 12(bookworm)的版本(内核为6.1 LTS)，上述版本可以正常连接wifi。

### 前言

Armbian5.77(Debian 9)系统之前是N1比较稳定的系统，但是上游Debian 9的源在2023-04-23已经失效。

因此，需要升级一下Armbian系统。

目前，[ophub的Armbian](https://github.com/ophub/amlogic-s9xxx-armbian) 是比较稳定的支持晶晨、瑞芯微和全志等cpu的Armbian系统。

### 1. 确保固件版本为2.19

刷机前需要确认N1的固件版本为2.19，高于此版本需要先降级。  
 ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/c38eca8ad08044c9a77596533faae694.jpeg#pic_center)

### 2. 刷系统到U盘

balenaEtcher刷入Armbian_25.05.0_amlogic_s905d_bookworm_6.1.137_server_2025.05.05.img

### 3. 启动U盘系统

默认账号：root，默认密码：1234

进入系统后的步骤：

* 修改密码
* 新建账户(不新建账户不会初始化桌面系统)

### 4. U盘系统写入EMMC

```
# armbian-install

```

### 5. 关机拨U盘

```
# halt

```

拔掉U盘

### 6. 重新上电

Armbian系统正常启动。
