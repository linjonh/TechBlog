    <h2 id="%E5%8C%85%E8%A3%85%E7%B1%BB%E5%8F%98%E9%87%8F" name="%E5%8C%85%E8%A3%85%E7%B1%BB%E5%8F%98%E9%87%8F" style="text-align:center">
     <span style="color:#fe2c24">
      <strong>
       包装类变量
      </strong>
     </span>
    </h2>
    <p>
     <u>
      每个基本数据类型都有对应的基本类型的包装类变量
     </u>
     ，
     <strong>
      将
      <span style="color:#fe2c24">
       基本数据类型
      </span>
      通过
      <span style="color:#fe2c24">
       对应的包装类对象
      </span>
      载入着进入到
      <span style="color:null">
       类与对象
      </span>
      <span style="color:#fe2c24">
       面向对象体系
      </span>
     </strong>
    </p>
    <h3 id="%E4%B8%80%E3%80%81%E8%A3%85%E7%AE%B1" name="%E4%B8%80%E3%80%81%E8%A3%85%E7%AE%B1">
     一、装箱
    </h3>
    <blockquote>
     <p>
      <strong>
       Integer.
       <span style="background-color:#ffd900">
        valueOf(int)
       </span>
      </strong>
     </p>
     <p>
      <strong>
       —&gt; return
       <span style="background-color:#38d8f0">
        Integer
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="background-color:#a2e043">
        实例化创建这个整数对应的包装似复刻变量对象
       </span>
      </strong>
     </p>
    </blockquote>
    <h4 id="1.%E5%AE%9E%E4%BE%8B%E5%8C%96%E5%8C%85%E8%A3%85%E5%AF%B9%E8%B1%A1" name="1.%E5%AE%9E%E4%BE%8B%E5%8C%96%E5%8C%85%E8%A3%85%E5%AF%B9%E8%B1%A1">
     1.实例化包装对象
    </h4>
    <p>
     <em>
      实例化包装类对象时，可以通过
      <u>
       <strong>
        new Integer(int)
       </strong>
      </u>
      实例化创建一个包含这个整数的包装类的似复刻变量，但我们一般都使用
      <u>
       <strong>
        Integer类变量里面的
        <span style="color:#ed7976">
         静态
        </span>
        方法
        <span style="color:#fe2c24">
         valueOf(int)
        </span>
       </strong>
      </u>
      来实例化创建包上它这个整数的包装类对象
     </em>
    </p>
    <hr/>
    <h4 id="2.%E9%9D%99%E6%80%81%E7%BC%93%E5%AD%98%E6%B1%A0" name="2.%E9%9D%99%E6%80%81%E7%BC%93%E5%AD%98%E6%B1%A0">
     2.静态缓存池
    </h4>
    <p>
     每个包装类变量里面都有一个
     <strong>
      静态的数组引用变量成员
      <span style="color:#fe2c24">
       cache[]
      </span>
     </strong>
     ，
     <strong>
      <span style="background-color:#ffd900">
       它指向一个-128到127整数对应好
      </span>
      <span style="color:#fe2c24">
       <span style="background-color:#ffd900">
        已创建
       </span>
      </span>
      <span style="background-color:#ffd900">
       的包装类似复刻变量的似复刻变量引用数组
      </span>
     </strong>
     ，共装有256个已创好的包装类似复刻变量引用，
     <strong>
      <span style="background-color:#ffd900">
       直接已经创好可以直接供对应的-128~127这些整数使用的
      </span>
     </strong>
    </p>
    <p>
     <img alt="" height="199" src="https://i-blog.csdnimg.cn/direct/fbae10a8851a42dc8504cb6bdab856c9.png" width="1067"/>
    </p>
    <p>
     用Integer.valueOf(int) 调用创建-128~127对应的包装似复刻变量的时候，它是
     <strong>
      直接返回数组引用里指向的对应已创好的似复刻变量引用
     </strong>
     ，因此调用valueOf取包装类对象时-128~127整数的对应包装类对象是
     <span style="color:#fe2c24">
      <strong>
       不用去创建直接取到的效率会很快
      </strong>
     </span>
     ，如果调用valueOf创建对象的数字在-128~127以外的，就会去new创建新的包装类对象去实现包装
    </p>
    <hr/>
    <h4 id="3.%E5%86%99%E6%B3%95" name="3.%E5%86%99%E6%B3%95">
     3.写法
    </h4>
    <p>
     <strong>
      <span style="background-color:#ffd900">
       Integer iger = 10
      </span>
     </strong>
     ;其实是
     <strong>
      <span style="background-color:#38d8f0">
       省略了Integer静态方法valueOf的调用
      </span>
     </strong>
    </p>
    <p>
     <strong>
      —&gt; Integer.valueOf(10) return Integer
     </strong>
     ，即实现了
     <strong>
      自动装箱
     </strong>
    </p>
    <hr/>
    <h3 id="%E4%BA%8C%E3%80%81%E6%8B%86%E7%AE%B1" name="%E4%BA%8C%E3%80%81%E6%8B%86%E7%AE%B1">
     二、拆箱
    </h3>
    <blockquote>
     <p>
      <strong>
       iger.
       <span style="background-color:#ffd900">
        intValue()
       </span>
      </strong>
     </p>
     <p>
      <strong>
       —&gt; return
       <span style="background-color:#38d8f0">
        int
       </span>
      </strong>
     </p>
     <p>
      <strong>
       <span style="background-color:#a2e043">
        将此包装类对象里面装的基本类型数据返回
       </span>
      </strong>
     </p>
    </blockquote>
    <p>
     此方法是非静态方法，使用时也是省略隐式写的：
    </p>
    <p>
     <strong>
      <span style="background-color:#ffd900">
       double a = iger
      </span>
     </strong>
     ;
     <strong>
      <span style="background-color:#38d8f0">
       省略了它非静态方法的调用
      </span>
     </strong>
    </p>
    <p>
     <strong>
      —&gt;
     </strong>
     <strong>
      double a = iger.intValue() return int
     </strong>
    </p>
    <p>
     <strong>
      包装类里面
      <span style="color:#fe2c24">
       只有它对应
      </span>
      基本类型数据的转化方法
     </strong>
     ，即iger只有intValue整型包装类转对应整型数据的方法，它这里是先通过iger.intValue()返回了整数10，然后整数10进行了int向double的
     <span style="color:#fe2c24">
      <strong>
       隐式类型提升
      </strong>
     </span>
    </p>
