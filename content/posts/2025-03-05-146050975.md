# windows 利用nvm 管理node.js 2025最新版

1.首先在下载nvm
[下载链接](https://github.com/coreybutler/nvm-windows/releases "下载链接")

2. 下载最新版本的nvm

![](https://i-blog.csdnimg.cn/direct/f28410cd64ca409487a986d1aa53eb1e.png)

3. 同意协议

![](https://i-blog.csdnimg.cn/direct/ca66b5fe03504839b919765ee6c4d6d7.png)

注意：选择安装路径

![](https://i-blog.csdnimg.cn/direct/d323f50f12fc48f1932282b8d694dc09.png)

之后一直下一步即可  可以取消勾选 open with Powershell 勾选后它会自动打开Powershell

![](https://i-blog.csdnimg.cn/direct/3885ed53c78e44d49671fa9a64d2ab5d.png)

这里选用cmd 输入以下命令查看是否安装成功

```
nvm version
```

![](https://i-blog.csdnimg.cn/direct/9fc59add53fd448bb1b97558380ef286.png)

查看已经安装的版本  我之前自己下载了16.9.0 可以看到能够检测到

```
nvm list
```

![](https://i-blog.csdnimg.cn/direct/851d55e96bb74ebf99cc8a9d48d61827.png)

下载想用的版本 以v20.15.1为例

```
nvm install 20.15.1
```

![](https://i-blog.csdnimg.cn/direct/3c6b2d5fb9a94922b2e9cf91b40d27c3.png)

指定使用版本

```
nvm use 20.15.1
```

![](https://i-blog.csdnimg.cn/direct/db4131e5e8e54aa78ae6b5f57a1b9263.png)

查看系统node版本

![](https://i-blog.csdnimg.cn/direct/4c4af446807f44299463bee40ea4a32c.png)
