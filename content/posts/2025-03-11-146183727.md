    <h2>

     为TDengine安装的DBeaver教程
    </h2>
    <h3>
     <a id="_2311_DBeaver_1">
     </a>
     安装 23.1.1 版本以上的DBeaver
    </h3>
    <ul>
     <li>
      因为官方文档说
      <strong>
       这个版本之上的DBeaver才支持
      </strong>
      TDengine内嵌
     </li>
     <li>
      前往
      <a href="https://github.com/dbeaver/dbeaver/wiki/Installation">
       DBeaver 官方文档
      </a>
      进行版本下载
     </li>
     <li>
      滑到链接最下面点击进入
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3c704913cbe540f9bff7e012d925b604.png"/>
    </p>
    <ul>
     <li>
      点击download，进入选择下载版本
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/12e28a5549274d20a75bd643e2d906f8.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/de4c38e24bf84684b53adbbfe2747ad1.png"/>
    </p>
    <ul>
     <li>
      等待下载成功即可双击自行安装
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/dd9ba11db36d4e87b09b53a4da2bb15e.png"/>
    </p>
    <h2>
     <a id="TDengine_17">
     </a>
     打开数据库连接TDengine
    </h2>
    <ul>
     <li>
      数据库-&gt;驱动管理器
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/bcf2db2b84564642a5484cd903d8b31b.png"/>
    </p>
    <ul>
     <li>
      点开之后下滑，确认自己有TDengine
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/92e472faacbe40f980128233e97e7064.png">
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/32979876a2a043c9a59d5d9015bdfcd9.png"/>
     </img>
    </p>
    <ul>
     <li>
      如果没有，就自己点击新建，新建一个，自己新建的话就需要下载一个一个taos的jar包
      <code>
       taos-jdbcdriver-3.0.0-dist.jar
      </code>
      ，因为是我们自己新建驱动
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/863dcea16c6a4f1a9474849a29f7930c.png"/>
    </p>
    <ul>
     <li>
      照着自己项目连接的数据库去配置这些url、端口，配置好后在库内添加下载好的驱动
     </li>
    </ul>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3ce81a406f7349d39b52de62061af843.png"/>
     <br/>
     <strong>
      这样我们就配置好了一个TDengine数据库，接下来可以进行数据库的连接
     </strong>
    </p>
