    <p>
     <strong>
      摘要
     </strong>
     ：本文详细介绍了在Windows、macOS和Linux三大操作系统下安装MySQL数据库的完整流程，帮助开发者快速搭建本地MySQL环境。
    </p>
    <h3>
     一、MySQL安装前准备
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        官网下载
       </strong>
       <br/>
       访问
       <a href="https://www.mysql.com/" rel="nofollow" title="MySQL官网">
        MySQL官网
       </a>
       → 选择"Downloads" → 选择"MySQL Community (GPL) Downloads"
      </p>
     </li>
     <li>
      <p>
       <strong>
        版本选择建议
       </strong>
      </p>
      <ul>
       <li>
        <p>
         新手推荐：MySQL 8.0（最新稳定版）
        </p>
       </li>
       <li>
        <p>
         企业兼容：根据需求选择5.7版本
        </p>
       </li>
       <li>
        <p>
         系统对应：
        </p>
        <ul>
         <li>
          <p>
           Windows选择
           <code>
            .msi
           </code>
           安装包
          </p>
         </li>
         <li>
          <p>
           macOS选择
           <code>
            .dmg
           </code>
           文件
          </p>
         </li>
         <li>
          <p>
           Linux选择对应发行版的Repository
          </p>
         </li>
        </ul>
       </li>
      </ul>
     </li>
    </ol>
    <h3>
     二、Windows系统安装步骤
    </h3>
    <h4>
     1. 安装程序
    </h4>
    <ol>
     <li>
      <p>
       双击下载的
       <code>
        .msi
       </code>
       文件
      </p>
     </li>
     <li>
      <p>
       选择安装类型：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          Developer Default
         </strong>
         ：默认开发配置
        </p>
       </li>
       <li>
        <p>
         <strong>
          Server only
         </strong>
         ：仅安装服务器
        </p>
       </li>
       <li>
        <p>
         <strong>
          Custom
         </strong>
         ：自定义安装（推荐）
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       选择安装组件：
      </p>
      <ul>
       <li>
        <p>
         MySQL Server
        </p>
       </li>
       <li>
        <p>
         MySQL Workbench（可视化工具）
        </p>
       </li>
       <li>
        <p>
         MySQL Shell
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     2. 配置向导
    </h4>
    <ol>
     <li>
      <p>
       选择配置类型：
      </p>
      <ul>
       <li>
        <p>
         <strong>
          Development Computer
         </strong>
         （开发环境）
        </p>
       </li>
       <li>
        <p>
         <strong>
          Server Computer
         </strong>
         （生产环境）
        </p>
       </li>
       <li>
        <p>
         <strong>
          Dedicated Computer
         </strong>
         （专用MySQL服务器）
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       设置root密码：
      </p>
      <pre><code class="hljs">MySQL Root Password: ********
Repeat Password: ********</code></pre>
     </li>
    </ol>
    <ol>
     <li>
      <p>
       配置Windows服务：
      </p>
      <ul>
       <li>
        <p>
         服务名默认
         <code>
          MySQL80
         </code>
        </p>
       </li>
       <li>
        <p>
         建议勾选"Start the MySQL Server at System Startup"
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     3. 配置环境变量
    </h4>
    <ol>
     <li>
      <p>
       右键"此电脑" → 属性 → 高级系统设置 → 环境变量
      </p>
     </li>
     <li>
      <p>
       在Path中添加
      </p>
      <pre><code class="hljs">C:\Program Files\MySQL\MySQL Server 8.0\bin</code></pre>
      <p>
      </p>
     </li>
    </ol>
    <h3>
     三、macOS系统安装
    </h3>
    <h4>
     方法1：使用Homebrew
    </h4>
    <pre><code class="hljs">brew install mysql
brew services start mysql</code></pre>
    <h4>
     方法2：官方dmg安装
    </h4>
    <ol>
     <li>
      <p>
       双击
       <code>
        .dmg
       </code>
       文件
      </p>
     </li>
     <li>
      <p>
       拖动MySQL图标到Applications文件夹
      </p>
     </li>
     <li>
      <p>
       打开系统偏好设置 → MySQL → Initialize Database
      </p>
     </li>
    </ol>
    <p>
     按提示完成：
    </p>
    <ol>
     <li>
      <p>
       设置密码强度策略
      </p>
     </li>
     <li>
      <p>
       移除匿名用户
      </p>
     </li>
     <li>
      <p>
       禁止远程root登录
      </p>
     </li>
     <li>
      <p>
       移除测试数据库
      </p>
     </li>
    </ol>
    <h3>
     四、Linux（Ubuntu）安装
    </h3>
    <h4>
     1. APT安装
    </h4>
    <pre><code class="hljs">sudo apt update
sudo apt install mysql-server</code></pre>
    <p>
     2. 安全配置
    </p>
    <pre><code class="hljs">sudo mysql_secure_installation</code></pre>
    <p>
    </p>
    <p>
     按提示完成：
    </p>
    <ol>
     <li>
      <p>
       设置密码强度策略
      </p>
     </li>
     <li>
      <p>
       移除匿名用户
      </p>
     </li>
     <li>
      <p>
       禁止远程root登录
      </p>
     </li>
     <li>
      <p>
       移除测试数据库
      </p>
     </li>
    </ol>
    <h3>
     五、验证安装
    </h3>
    <pre><code class="hljs">mysql -u root -p</code></pre>
    <pre><code class="hljs">mysql&gt; SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+
4 rows in set (0.00 sec)</code></pre>
    <h3>
     六、常见问题解决
    </h3>
    <h4>
     1. 安装失败
    </h4>
    <ul>
     <li>
      <p>
       检查系统位数（32/64位）
      </p>
     </li>
     <li>
      <p>
       确认VC++运行库已安装
      </p>
     </li>
     <li>
      <p>
       查看错误日志：
       <code>
        C:\ProgramData\MySQL\MySQL Server 8.0\Data\*.err
       </code>
      </p>
     </li>
    </ul>
    <h4>
     2. 忘记root密码
    </h4>
    <ol>
     <li>
      <p>
       停止MySQL服务
      </p>
     </li>
     <li>
      <p>
       使用安全模式启动：
      </p>
      <pre><code class="hljs">mysqld --skip-grant-tables</code></pre>
     </li>
     <li>
      <p>
       新窗口登录修改密码
      </p>
     </li>
    </ol>
    <h4>
     3. 服务无法启动
    </h4>
    <ul>
     <li>
      <p>
       检查3306端口占用：
       <code>
        netstat -ano | findstr :3306
       </code>
      </p>
     </li>
     <li>
      <p>
       查看错误日志路径：
      </p>
      <pre><code class="hljs">mysql&gt; SHOW VARIABLES LIKE 'log_error';</code></pre>
     </li>
    </ul>
