对上式进行拉普拉斯变换：

```python
from sympy import symbols, laplace_transform, Function

t, s, R, C, L = symbols('t s R C L')
y = Function('y')(t)

Y = laplace_transform(y, t, s, noconds=True)
print(Y)
Y

```

```python
u = L * C * y.diff(t, 2) + R * C * y.diff(t) + y
U = laplace_transform(u, t, s, noconds=True)
print(U)
U

```
