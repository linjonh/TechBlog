# Anthropic 平台升级，解锁 Claude3.7Sonnet 与 prompt 创作新玩法

近日，人工智能领域的先锋企业 Anthropic 宣布对其开发者平台实施了重大升级。

![](https://i-blog.csdnimg.cn/img_convert/da829fda54693dcd6e09b37e4222536c.png)

此次升级重磅推出全新团队协作功能，并大幅拓展了 Claude3.7Sonnet 模型的推理能力，旨在为企业在落地人工智能解决方案时，有效攻克关键痛点。

如需了解 Claude3.7Sonnet 可查看往期文章：

[硬刚DeepSeek、o3，双思维模型Claude 3.7 Sonnet来了！！！](https://mp.weixin.qq.com/s/cxDCFOIT1Ws9fXBVZFRVjg)

### 革新团队协作，让 prompt 管理更高效

以往，团队成员间传递 prompt 文档，常常陷入版本管理混乱的困境，导致线上模型表现不佳。

如今，这些问题已随 Anthropic Console 的升级成为历史。升级后的 Console 带来
**可分享的 prompts 功能**
，如同为团队打造了一个共享的 prompt 宝库。在这里，开发人员、领域专家、产品经理以及 QA 工程师能够协同合作，共同打磨每一条关键指令。

![](https://i-blog.csdnimg.cn/img_convert/f478f793265bffe3ab6684c7723a5f40.png)

不再需要担心 Prompt 在各种聊天工具和文档中散落，集中的管理显著提升了协作效率。这就像团队合作的“一键同步”功能。

所有 prompt 实现集中化管理，彻底告别在各类聊天软件和文档中 “迷失” 的状况，极大提升了协作效率，堪称团队合作的 “一键同步” 利器。

### Claude3.7Sonnet 大脑升级，思考也能“可视化”

本次升级的另一大亮点，是对 Anthropic 最强模型 Claude3.7Sonnet 的全方位支持。该模型不仅具备近乎实时的响应速度，更突破性地实现了思考过程的可视化呈现。

![](https://i-blog.csdnimg.cn/img_convert/c9fbf2f377a2d11f4e8ab8bd9d2b7943.png)

就像为 Claude 安装了 “思考可视化” 插件，用户能够清晰洞察其得出答案的每一个步骤。同时，Console 还配备思考预算控制功能，用户可针对开启扩展思考的 prompt，灵活设置最大思考 tokens 数量，实现对模型性能的精细化优化，避免算力浪费。

### 打造 prompt 炼金术，助力高效创作

编写让模型精准理解的 prompt 曾困扰不少用户，如今 Anthropic Console 为大家打造了 prompt 炼金工坊。

mp4

**Workbench 功能**
提供交互式环境，方便用户构建 prompt、整合外部工具并测试 API 调用，如同搭积木般灵活组织指令与示例。

![](https://i-blog.csdnimg.cn/img_convert/4e1703a121c5fe382a6065248b176701.jpeg)

更强大的是，用户只需描述目标，Claude 便能运用多种 prompt 工程技巧（如链式思考 reasoning），自动生成高效、精准且可靠的 prompt，堪称 “懒人” 创作高质量 prompt 的福音。

![](https://i-blog.csdnimg.cn/img_convert/edf255191ba81acf2049cca5b3cbbd4b.png)

对于从其他模型迁移或自行编写的 prompt，Console 也可通过先进的 prompt 工程技术进行自动优化，提升其在 Claude 上的性能表现。
