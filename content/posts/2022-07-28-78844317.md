     QT 通用数据库数据导入导出方案
    </h1>
   </div>
  </div>
 </div>

  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <span style="font-size:12px">
     QT提供了通用的数据库接口可以很方便的处理各种数据库,在项目中经常会遇到数据库数据的导入导出的需求. 由于各个数据库的DDL差异比较大,无法做到导出表结构的通用方案,这个只能借助相应数据库特有的SQL语句或者工具来解决, 如postgresql的COPY命令和pg_dump工具等.
     <br/>
     Navicat工具的导入导出功能做的非常不错,能够导出包含表结构和二进制数据的sql文件.
     <br/>
     需解决问题:
     <br/>
     1. 取出数据,组装insert语句
     <br/>
     2. 对于二进制数据,无法做到像navicat那样可以直接执行sql语句执行. 只能转化为十六进制文本格式. 在导入时自动解析
     <br/>
    </span>
    <p>
     <span style="font-size:12px">
      3. 对于数据库建表语句, 目前无法提供通用的导出方式,只能借助工具
     </span>
    </p>
    <p>
     <span style="font-size:12px">
      <br/>
     </span>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       一. 数据导出
      </span>
     </span>
     <br/>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       /**
       <br/>
       *@brief 导出数据库数据到文件中
       <br/>
       *@param path 文件路径
       <br/>
       */
       <br/>
       void ExportDB(const QString &amp;path)
       <br/>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       QStringList vList;
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       // 导出数据库所有名称以auth开头的表
       <br/>
       <span style="white-space:pre">
       </span>
       QStringList tables=gAuthDB.tables();
       <br/>
       <span style="white-space:pre">
       </span>
       foreach(QString table,tables)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       if(!table.startsWith("auth"))
       <br/>
       <span style="white-space:pre">
       </span>
       continue;
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       QSqlQuery query(gAuthDB);
       <br/>
       <span style="white-space:pre">
       </span>
       QString sql=QString("select * from %1").arg(table);
       <br/>
       <span style="white-space:pre">
       </span>
       query.exec(sql);
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       QSqlRecord record=query.record();
       <br/>
       <span style="white-space:pre">
       </span>
       while(query.next())
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       QString prefix=QString("insert into %1(").arg(table); // 记录属性字段名
       <br/>
       <span style="white-space:pre">
       </span>
       QString suffix="values("; // 记录属性值
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       // 遍历属性字段
       <br/>
       <span style="white-space:pre">
       </span>
       for(int i=0;i&lt;record.count();i++)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       QSqlField field=record.field(i);
       <br/>
       <span style="white-space:pre">
       </span>
       QString fieldName=field.name();
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       switch(field.type())
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       case QVariant::String:
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=fieldName;
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=QString("'%1'").arg(query.value(i).toString());
       <br/>
       <span style="white-space:pre">
       </span>
       break;
       <br/>
       <span style="white-space:pre">
       </span>
       case QVariant::ByteArray:
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=fieldName;
       <br/>
       <span style="white-space:pre">
       </span>
       QByteArray data=query.value(i).toByteArray();
       <br/>
       <span style="white-space:pre">
       </span>
       if(data.isNull())
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+="null";
       <br/>
       <span style="white-space:pre">
       </span>
       }else
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=QString("E'%1'").arg(data.toHex().data()); // blob数据按16进制格式导出
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <span style="white-space:pre">
       </span>
       break;
       <br/>
       <span style="white-space:pre">
       </span>
       default:
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=fieldName;
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=query.value(i).toString();
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       if(record.count()==1)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=")";
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=")";
       <br/>
       <span style="white-space:pre">
       </span>
       }else if(i!=record.count()-1)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=",";
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=",";
       <br/>
       <span style="white-space:pre">
       </span>
       }else if(i==record.count()-1)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       prefix+=")";
       <br/>
       <span style="white-space:pre">
       </span>
       suffix+=")";
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       // 组装sql语句 insert into auth_test values(0,'hello',E'003f')
       <br/>
       <span style="white-space:pre">
       </span>
       QString iSql=QString("%1 %2;").arg(prefix).arg(suffix);
       <br/>
       <span style="white-space:pre">
       </span>
       vList.append(iSql);
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       QFile file(path);
       <br/>
       <span style="white-space:pre">
       </span>
       file.open(QIODevice::WriteOnly|QIODevice::Truncate);
       <br/>
       <br/>
       <br/>
       <span style="white-space:pre">
       </span>
       // 将sql语句写入文件
       <br/>
       <span style="white-space:pre">
       </span>
       QTextStream out(&amp;file);
       <br/>
       <span style="white-space:pre">
       </span>
       foreach(QString line,vList)
       <br/>
       <span style="white-space:pre">
       </span>
       {
       <!-- -->
       <br/>
       <span style="white-space:pre">
       </span>
       out&lt;&lt;line+"\n";
       <br/>
       <span style="white-space:pre">
       </span>
       }
       <br/>
       }
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
        二. 数据导入
       </span>
       <br/>
      </span>
     </span>
    </p>
    <p>
     <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
      <span style="font-size:12px">
       <span style="font-family:Helvetica,Tahoma,Arial,sans-serif">
        /**
        <br/>
        *@brief 读取sql文本内容,并写入至数据库
        <br/>
        *@param path sql文件路径
        <br/>
        */
        <br/>
        void ImportDB(const QString &amp;path)
        <br/>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        QSqlQuery query(gAuthDB);
        <br/>
        <span style="white-space:pre">
        </span>
        QFile file(path);
        <br/>
        <span style="white-space:pre">
        </span>
        file.open(QFile::ReadOnly);
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        QTextStream in(&amp;file);
        <br/>
        <span style="white-space:pre">
        </span>
        while(!in.atEnd())
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        QString sql=in.readLine();
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        // 通过分析values(E'),判断是否有二进制数据,如没有直接运行sql语句,如有则需要将16进制文本转换为blob数据
        <br/>
        <span style="white-space:pre">
        </span>
        QRegExp reg("E'([0-9a-f]{1,})'");
        <br/>
        <span style="white-space:pre">
        </span>
        if(!sql.contains(reg))
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        query.exec(sql);
        <br/>
        <span style="white-space:pre">
        </span>
        }else
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        int pos=0;
        <br/>
        <span style="white-space:pre">
        </span>
        QStringList bList;
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        // 探索所有的blob字段
        <br/>
        <span style="white-space:pre">
        </span>
        while((pos=reg.indexIn(sql,pos))!=-1)
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        bList.append(reg.cap(0));
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        QString blob=reg.cap(1);
        <br/>
        <span style="white-space:pre">
        </span>
        pos+=reg.matchedLength();
        <br/>
        <span style="white-space:pre">
        </span>
        }
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        // blob字段填充占位符
        <br/>
        <span style="white-space:pre">
        </span>
        foreach(QString key,bList)
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        sql.replace(key,"?");
        <br/>
        <span style="white-space:pre">
        </span>
        }
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        query.prepare(sql);
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        // 绑定占位符数据
        <br/>
        <span style="white-space:pre">
        </span>
        for(int i=0;i&lt;bList.size();i++)
        <br/>
        <span style="white-space:pre">
        </span>
        {
        <!-- -->
        <br/>
        <span style="white-space:pre">
        </span>
        // 去除E''
        <br/>
        <span style="white-space:pre">
        </span>
        QString hexBlob=bList[i].mid(2,bList[i].size()-1);
        <br/>
        <span style="white-space:pre">
        </span>
        // 还原16进制数据
        <br/>
        <span style="white-space:pre">
        </span>
        QByteArray ba=QByteArray::fromHex(hexBlob.toLocal8Bit());
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        query.bindValue(i,ba);
        <br/>
        <span style="white-space:pre">
        </span>
        }
        <br/>
        <br/>
        <br/>
        <span style="white-space:pre">
        </span>
        query.exec();
        <br/>
        <span style="white-space:pre">
        </span>
        }
        <br/>
        <span style="white-space:pre">
        </span>
        }
        <br/>
        }
        <br/>
       </span>
      </span>
