    <p>
     一、python pip的安装与使用
    </p>
    <p>
     1、pip 是 Python 包管理工具，该工具提供了对Python 包的查找、下载、安装、卸载的功能。
    </p>
    <p>
     目前如果你在
     <a href="https://www.python.org/" rel="nofollow">
      python.org
     </a>
     下载最新版本的安装包，则是已经自带了该工具。
    </p>
    <p>
     Python 2.7.9 + 或 Python 3.4+ 以上版本都自带 pip 工具。
    </p>
    <p>
     pip 官网：
     <a href="https://pypi.org/project/pip/" rel="nofollow">
      https://pypi.org/project/pip/
     </a>
    </p>
    <p>
     你可以通过以下命令来判断是否已安装：
    </p>
    <pre class="has"><code class="hljs language-python">pip --version</code></pre>
    <p>
     <img alt="" class="has" height="75" src="https://i-blog.csdnimg.cn/blog_migrate/e68e0151f8be2b33556259def567c7a7.png" width="407"/>
    </p>
    <p>
     2、如果你还未安装，则可以使用以下方法来安装：
    </p>
    <pre class="has"><code class="hljs language-python">$ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py   # 下载安装脚本
$ sudo python get-pip.py    # 运行安装脚本</code></pre>
    <blockquote>
     <p>
      <strong>
       注意：
      </strong>
      用哪个版本的 Python 运行安装脚本，pip 就被关联到哪个版本，如果是 Python3 则执行以下命令：
     </p>
     <pre>$ sudo python3 get-pip.py    # 运行安装脚本。</pre>
     <p>
      一般情况 pip 对应的是 Python 2.7，pip3 对应的是 Python 3.x。
     </p>
    </blockquote>
    <p>
     发现无法执行2.   查资料。我们先进行3、4
    </p>
    <p>
     <s>
      <strong>
       3、注意：要跳到python的安装目录E:\Python27\Scripts
      </strong>
      一路cd索引到easy_install.exe所在的地方，执行指令“easy_install.exe pip”
     </s>
    </p>
    <p>
     <img alt="" class="has" height="309" src="https://i-blog.csdnimg.cn/blog_migrate/b1ff3297314d15a48b2a3cb8cc36ff74.png" width="564"/>
    </p>
    <p>
     则Scripts文件夹下会出现一系列和pip有关的文件，其中有pip.exe。
    </p>
    <p>
     4、将pip.exe的路径也添加到环境变量PATH中。即E:\Python27\Scripts
    </p>
    <p>
     在cmd下输入“pip”，如果能识别"pip"指令，则说明pip安装成功了。
    </p>
    <p>
     <img alt="" class="has" height="442" src="https://i-blog.csdnimg.cn/blog_migrate/287db7c34251e4726bfb163fc556873a.png" width="677"/>
    </p>
    <p>
     <img alt="" class="has" height="94" src="https://i-blog.csdnimg.cn/blog_migrate/74d934fba04493c8301946fb2a466109.png" width="650"/>
    </p>
    <p>
     二、pandas模块
    </p>
    <p>
     pandas是一个开源的python库，使用其强大的数据结构提供高性能数据操作和分析工具
    </p>
    <p>
     下面我们利用pip安装pandas
    </p>
    <p>
     在cmd窗口输入pip install pandas
    </p>
    <p>
     <img alt="" class="has" height="442" src="https://i-blog.csdnimg.cn/blog_migrate/2b6a7815552b4a8519f89189e3faa856.png" width="677"/>
    </p>
    <p>
     哈哈哈，机智。安装成功
    </p>
    <p>
     项目中import pandas包不会报错啦
    </p>
