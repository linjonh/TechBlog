   <div class="htmledit_views" id="content_views">
    <p>
     本来想在原来的语音识别的基础上增加本地扩展本地词典，
    </p>
    <p>
     采用的语音识别是Vosk识别器，模型是 vosk-model-small-cn-0.22
    </p>
    <pre><code class="language-cs">// 初始化Vosk识别器
if (recognizer == null)
{
    using (Model model = new Model(modelPath))
    {
        string grammar = @"{""root"": [""你好"", ""退出"", ""电视剧"", ""我有一个朋友"", {""早上好"" : 0.5}]}";
        recognizer = new VoskRecognizer(model, 16000.0f, grammar);
        //recognizer = new VoskRecognizer(model, 16000.0f);

    }
}</code></pre>
    <p>
     注释掉的是原来能用的创建对象，但是增加了词典后直接报内存错误。
    </p>
    <p>
     经过搜索
    </p>
    <p>
     <strong>
      解决
      <code>
       System.AccessViolationException
      </code>
      内存错误的逐步指南
     </strong>
    </p>
    <hr/>
    <h4>
     🔍 原因分析
    </h4>
    <p>
     此错误通常由
     <strong>
      非托管内存操作越界
     </strong>
     引起，常见于 Vosk 等依赖本地库的调用场景。可能触发条件：
    </p>
    <ol>
     <li>
      1.
      <strong>
       模型文件损坏
      </strong>
      或路径错误
     </li>
     <li>
      2.
      <strong>
       音频数据格式
      </strong>
      不匹配（采样率/位深度）
     </li>
     <li>
      3.
      <strong>
       对象生命周期管理
      </strong>
      不当导致提前释放资源
     </li>
     <li>
      4.
      <strong>
       跨线程访问
      </strong>
      非托管代码未同步
     </li>
    </ol>
    <hr/>
    <h4>
     ✅ 解决方案步骤
    </h4>
    <ol>
     <li>
      1.
      <p>
       <strong>
        验证模型完整性
       </strong>
      </p>
      <ul>
       <li>
        重新下载模型（推荐使用
        <a href="https://alphacephei.com/vosk/models" rel="nofollow" title="Vosk 中文模型 0.22">
         Vosk 中文模型 0.22
        </a>
        ）
       </li>
       <li>
        确认解压后目录结构完整（应有
        <code>
         am
        </code>
        ,
        <code>
         conf
        </code>
        ,
        <code>
         graph
        </code>
        等子目录）
       </li>
       <li>
        代码中绝对路径访问：
       </li>
       <li>
        <pre><code>string modelPath = @"D:\models\vosk-model-cn-0.22"; // 避免中文和空格
Model model = new Model(modelPath);</code></pre>
       </li>
      </ul>
     </li>
    </ol>
    <p>
     结论：
    </p>
    <p>
     本来想弄小模型 vosk-model-small-cn-0.22   40m的语音识别，但这个小模型的功能局限性强，不能增加本地词典。
    </p>
    <p>
     还是得下载1.5G的vosk-model-cn-0.22 模型。
    </p>
