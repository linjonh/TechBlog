     2.讲解算法原理

     分情况讨论

    <p>
     5---》直接收下

     10---》找五元，收下

     20----》10+5△

     ----》5+5+5

    <p>
     由于5元更有用，则尽可能保留5元
    </p>
    <p>
    </p>
    <p>
     3.代码
    </p>
    <pre><code class="language-java">class Solution {
    public boolean lemonadeChange(int[] bills) {
        int five=0,ten=0;
        for(int x:bills){
            if(x==5){
                five++;
            }else if(x==10){
                if(five==0){
                    return false;
                }else{
                    five--;
                    ten++;
                }
            }else{
                if(ten!=0&amp;&amp;five!=0){
                    ten--;
                    five--;
                }else if(five&gt;=3){
                    five-=3;
                }else{
                    return false;
                }
            }
        }
        return true;
    }
}</code></pre>
