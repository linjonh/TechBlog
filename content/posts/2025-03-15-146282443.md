# 【Docker compose】neo4j容器安装apoc插件

### 一、下载apoc插件

查看镜像版本

    
    
    docker exec -it c52eaed57ab4 neo4j --version
    
    5.25.1
    

  * [apoc插件下载界面](https://github.com/neo4j/apoc/releases)，选择与neo4j兼容的apoc版本
  * [apoc与neo4j版本对应表](https://neo4j.com/docs/apoc/current/installation/)

### 二、Docker compose配置

卷映射

    
    
    - ./neo4j/config:/var/lib/neo4j/conf  
    - ./neo4j/plugins:/var/lib/neo4j/plugins
    
    

配置文件：  
有两个配置文件

neo4j.conf

    
    
    server.memory.pagecache.size=512M  
    server.default_listen_address=0.0.0.0  
      
    dbms.security.procedures.unrestricted=apoc.*  
    server.directories.logs=/logs
    

apoc.conf  
用于备份和恢复

    
    
    apoc.import.file.enabled=true  
    apoc.export.file.enabled=true  
    apoc.ttl.enabled=false  
    apoc.uuid.enabled=false
    

### 三、重启容器

    
    
    docker compose up
    
    访问：http://localhost:7474/browser/
    
    

测试函数：

    
    
    RETURN apoc.version()
    call apoc.help("apoc")
