    <h2>
     1.memcpy的使用和模拟实现
    </h2>
    <pre><code class="language-cpp">void * memcpy(void * destination, const void * source, size_t num);
</code></pre>
    <p>
     函数memcpy从source的位置考试向后复制num个字节的数据到destination指向的内存位置。
    </p>
    <p>
     这个函数在遇到' \0 '的时候并不会停下来。
    </p>
    <p>
     如果source和destination有任何的重叠，复制的结果都是未定义的。
    </p>
    <p>
     <span style="color:#fe2c24">
      注：只关注要拷贝的数据在哪里，要存放到哪里，拷贝几个字节，至于内存中存放什么数据，什么类型的数据，都不重要！！！
     </span>
    </p>
    <p>
     <img alt="" height="356" src="https://i-blog.csdnimg.cn/direct/b7b338c16ec446c391c94e1d59d4a78d.png" width="442"/>
    </p>
    <p>
     memcpy的模拟实现
    </p>
    <pre><code class="language-cpp">void* my_memcpy(void* dest, const void* scr, size_t num)
{
	assert(dest &amp;&amp; scr);
	void* ret = dest;
	while (num--)
	{
		*(char*)dest = *(char*)scr;
		dest = (char*)dest + 1;
		scr = (char*)scr + 1;
	}
	return ret;
}</code></pre>

    <h2>
     2.memmove的使用和模拟实现
    </h2>
    <pre><code class="language-cpp">void * memmove(void * destination, const void * source, size_t num);
</code></pre>
    <p>
     和memcpy的差别就是memmove函数处理的源内存块和目标内存块是可以重叠的。
    </p>
    <p>
     如果源空间和目标空间出现重叠，就得使用memmove函数处理。
    </p>
    <p>
     <img alt="" height="251" src="https://i-blog.csdnimg.cn/direct/897d7f1be4e4444fbcba34b3f88e125c.png" width="520"/>
    </p>
    <p>
     memmove的模拟实现
    </p>
    <pre><code class="language-cpp">void* my_memmove(void* dest, const void* scr, size_t num)
{
	assert(dest &amp;&amp; scr);
	void* ret = dest;
	if (dest &lt; scr)
	{
		while (num--)
		{
			*(char*)dest = *(char*)scr;
			dest = (char*)dest + 1;
			scr = (char*)scr + 1;
		}
	}
	else
	{
		while (num--)
		{
			*((char*)dest + num) = *((char*)scr + num);
		}
	}
	return ret;
}</code></pre>
    <p>
