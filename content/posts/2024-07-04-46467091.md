    <p>
     Java
     <span style="font-family:宋体;">
      零基础跑起第一个程序
     </span>
    </p>
    <h2>
     一 概述
    </h2>
    <h4>
     1  java
     <span style="font-family:宋体;">
      代码编译
     </span>
    </h4>
    <p>
     编译后才能在计算机中运行，编译就是把人能看懂的代码转换成机器能看懂的形式
    </p>
    <h4>
     2 java
     <span style="font-family:宋体;">
      的优点
     </span>
    </h4>
    <p>
     一次编译，到处运行。因为
     <span style="font-family:Times New Roman;">
      java
     </span>
     <span style="font-family:宋体;">
      代码是在虚拟机中运行，虚拟机消除了不同操作系统间的差异。
     </span>
     <span style="font-family:Times New Roman;">
      Java Virtual Machine---
     </span>
     <span style="font-family:宋体;">
      简称
     </span>
     <span style="font-family:Times New Roman;">
      JVM
     </span>
    </p>
    <p>
     <span style="font-family:Times New Roman;">
      <img alt="" src="https://img-blog.csdn.net/20150612073812756?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbml1X2dhbw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
       <br/>
      </img>
     </span>
    </p>
    <p>
    </p>
    <h4>
     3  java
     <span style="font-family:宋体;">
      的版本
     </span>
    </h4>
    <p>
     SUN
     <span style="font-family:宋体;">
      公司发布
     </span>
     <span style="font-family:Times New Roman;">
      Java1.2
     </span>
     <span style="font-family:宋体;">
      以后，叫把
     </span>
     <span style="font-family:Times New Roman;">
      Java
     </span>
     <span style="font-family:宋体;">
      叫做
     </span>
     <span style="font-family:Times New Roman;">
      java2
     </span>
     <span style="font-family:宋体;">
      了，所以现在都叫
     </span>
     <span style="font-family:Times New Roman;">
      J2XX
     </span>
     <span style="font-family:宋体;">
      。
     </span>
    </p>
    <p>
    </p>
    <p>
     功能上分为三个版本：
    </p>
    <p>
     J2SE
     <span style="font-family:宋体;">
      标准版
     </span>
     <span style="font-family:Times New Roman;">
      ----
     </span>
     <span style="font-family:宋体;">
      现在大部分开发都用它
     </span>
    </p>
    <p>
     J2EE
     <span style="font-family:宋体;">
      企业版
     </span>
     <span style="font-family:Times New Roman;">
      ----
     </span>
     <span style="font-family:宋体;">
      功能最多的，是
     </span>
     <span style="font-family:Times New Roman;">
      J2SE
     </span>
     <span style="font-family:宋体;">
      的扩展版
     </span>
    </p>
    <p>
     J2ME
     <span style="font-family:宋体;">
      微型板
     </span>
     <span style="font-family:Times New Roman;">
      ----
     </span>
     <span style="font-family:宋体;">
      只有部分
     </span>
     <span style="font-family:Times New Roman;">
      J2SE
     </span>
     <span style="font-family:宋体;">
      的功能，用于嵌入式系统。但不用于
     </span>
     <span style="font-family:Times New Roman;">
      Android
     </span>
     <span style="font-family:宋体;">
      系统
     </span>
    </p>
    <p>
     还有，
     <span style="font-family:Times New Roman;">
      android
     </span>
     <span style="font-family:宋体;">
      开发有自己的一个版本，不属于以上任何版本之一。
     </span>
    </p>
    <h4>
     4 java
     <span style="font-family:宋体;">
      开发环境搭建
     </span>
    </h4>
    <p>
     安装
     <span style="font-family:Times New Roman;">
      JDK
     </span>
     <span style="font-family:宋体;">
      和开发工具地址
     </span>
     :
    </p>
    <p>
     <a href="http://www.oracle.com/technetwork/java/javase/downloads/index.html" rel="nofollow noopener noreferrer" target="_blank">
      <span style="color:rgb(0, 0, 255);">
       http://www.oracle.com/technetwork/java/javase/downloads/index.html
      </span>
     </a>
    </p>
    <p>
     直接下载“Netbeans with JDK”，会把
     <span style="font-family:Times New Roman;">
      IDE
     </span>
     <span style="font-family:宋体;">
      和
     </span>
     <span style="font-family:Times New Roman;">
      JDK J2SE
     </span>
     <span style="font-family:宋体;">
      版一起装上。
     </span>
    </p>

     <span style="font-family:Times New Roman;">

    <p>
     8) 解释一下这段代码
    </p>
    <p>
     <span style="color:rgb(0,0,255);">
      package Hello
     </span>
     包名，源码文件必须放在一个包中。
    </p>
    <p>
     <span style="color:rgb(0,0,255);">
      public class Hello
     </span>
     声明一个类，类的内容是
     <span style="font-family:Times New Roman;">
      {}
     </span>
     <span style="font-family:宋体;">
      中的东西。
     </span>
    </p>
    <p>
     <span style="color:rgb(0,0,255);">
      public static void main(String[] args)
     </span>
     定义一个方法（或叫函数），方法名叫
     <span style="font-family:Times New Roman;">
      main
     </span>
     <span style="font-family:宋体;">
      。
     </span>
     <span style="font-family:Times New Roman;">
      main
     </span>
     <span style="font-family:宋体;">
      方法入有方法，
     </span>
     <span style="font-family:Times New Roman;">
      java
     </span>
     <span style="font-family:宋体;">
      程序从
     </span>
     <span style="font-family:Times New Roman;">
      main
     </span>
     <span style="font-family:宋体;">
      函数开始执行
     </span>
    </p>
    <p>
     <span style="color:rgb(0,0,255);">
      System.out.println("Hello");
     </span>
     调用一个方法，这个方法是
     <span style="font-family:Times New Roman;">
      Java SDK
     </span>
     <span style="font-family:宋体;">
      中带的，意思是在输出窗口中输出一个字符串“
     </span>
     <span style="font-family:Times New Roman;">
      Hello
     </span>
     <span style="font-family:宋体;">
      ” 。
