        SQL Server 2000定期自动备份详细图解

     </span>
    </p>
    <p style="text-align: left;">
     <span style="font-size: medium;">
      <strong>
      </strong>
     </span>
    </p>
    <p style="text-align: left;">
     企业管理器中的Tools，Database Maintenance Planner，可以设置数据库的定期自动备份计划。并通过启动Sql server Agent来自动运行备份计划。具体步骤如下:
     <br/>
     1、打开企业管理器，在控制台根目录中依次点开Microsoft SQL Server--&gt;SQL Server组--&gt;双击打开你的服务器;
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/1e7b45559218409a8572d227584f03ff.png">
      <br/>
      <br/>
      2、然后点上面菜单中的工具--&gt;选择数据库维护计划器;
      <br/>
      <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/7c4da2a5ff98427cb3a261a012b04dfd.png">
       <br/>
       <br/>
       <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/a69c1f6b899548b7899d13ec86ba2334.png"/>
      </img>
     </img>
    </p>
    <p style="text-align: left;">
     <br/>
     <br/>
     3、下一步选择要进行自动备份的数据
    </p>
    <p style="text-align: left;">
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/223387fe6aa24fcf961b300224654859.png"/>
    </p>
    <p style="text-align: left;">
    </p>
    <p style="text-align: left;">
     4、下一步更新数据优化信息，这里一般不用做选择
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/81e776680be443e3a97a91e6a15f7828.png"/>
    </p>
    <p style="text-align: left;">
     5、下一步检查数据库完整性，也一般不选择;
     <br/>
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/afb59112ece2430690687aec3a967177.png">
      <br/>
      <br/>
      6、下一步指定数据库备份计划，默认的是1周备份一次，点击更改选择每天备份后点确定;
      <br/>
      <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/e933f7c5be804da08259258c0b26fdb7.png"/>
     </img>
    </p>
    <p style="text-align: left;">
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/0f8af99ecdc1424295b68fbe54493481.png">
      <br/>
      <br/>
      7、下一步指定备份磁盘目录，选择指定目录，如您可以在D盘新建一个目录如：d:/databak，然后在这里选择使用此目录，如果您的数据库比较多最好选择为每个数据库建立子目录，然后选择删除早于多少天前的备份，一般设定4－7天，这看您的具体备份要求，备份文件扩展名默认的是BAK;
      <br/>
      <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/6e0f8f7146ee400087452afb2e64df82.png"/>
     </img>
    </p>
    <p style="text-align: left;">
     <br/>
     <br/>
     8、下一步指定事务日志备份计划，看您的需要做选择
    </p>
    <p style="text-align: left;">
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/c06c73267da64e48ba15cc5496322663.png"/>
    </p>
    <p style="text-align: left;">
     9、下一步要生成的报表，一般不做选择
    </p>
    <p style="text-align: left;">
    </p>
    <div style="text-align: left;">
     <img alt="" border="0" class="blogimg" src="https://i-blog.csdnimg.cn/blog_migrate/f8d956d10570eb654fad846a2c422734.jpeg"/>
    </div>
    <p style="text-align: left;">
     <a href="http://hiphotos.baidu.com/kxq0917/pic/item/bfe11c1696479345f2de32a8.jpg" rel="nofollow noopener noreferrer" target="_blank">
     </a>
    </p>
    <p style="text-align: left;">
     10、下一步维护计划历史记录，最好用默认的选项
     <br/>
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/29c50cfe09dc4fc78e457fa91088d020.png"/>
    </p>
    <p style="text-align: left;">
     11、下一步完成;
     <br/>
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/c39bcb515836427892d84e51b45e8101.png"/>
     <br/>
     <br/>
     12、完成后系统很可能会提示Sql Server Agent服务未启动，先点确定完成计划设定，然后找到桌面最右边状态栏中的SQL绿色图标，双击点开，在服务中选择Sql Server Agent，然后点击运行箭头，选上下方的当启动OS时自动启动服务;
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/236279c60e0141618ee5077ee9f33c69.png"/>
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/edb16612771d44649483e731e065dd39.png"/>
     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/967f0ec229e247d4952284bab4769ac5.png"/>
     <br/>
     <br/>
     13、可以设置自动启动sql server Agent:运行Services.msc，设置sqlserverAgent为自动启动。
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/d763b853244348f5bc35929af6a88b93.png"/>
     <br/>

     <br/>
     <br/>
     <img alt="" src="https://p-blog.csdn.net/images/p_blog_csdn_net/zhunj80/b5e2d879ed3a4b1d89eb4d44b3ceaec2.png"/>
     <br/>
     <br/>
     14、修改计划：
     <br/>
     打开企业管理器，在控制台根目录中依次点开Microsoft SQL Server--&gt;SQL Server组--&gt;双击打开你的服务器--&gt;管理--&gt;数据库维护计划       ;
