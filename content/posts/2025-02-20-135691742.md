    <h3>
     <a id="_7">
     </a>
     一.前言
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2eb857ee84c7e3aea10a91c14a19323e.png"/>
    </p>
    <p>
     疫情爆发以来，越来越多的用户借助于移动手机、电脑完成生活中的事务，许多的传统行业也更加重视与互联网的结合。本课题探讨利用不断发展和进步的网络技术，实现对个人信息修改、毕业生、老师、企业单位、企业用户、学院信息、专业信息、班级信息、资质申请、个人简历管理、招聘信息、宣教会、双选会、投递简历等主要功能的实现。
     <br/>
     本开题是基于微信开发者工具设计实现的毕业生就业管理微信小程序，扩展了普及毕业生就业管理信息的覆盖面。微信小程序免安装、使用便利、用户基数大的特性，为用户提供了更多更便捷的渠道。在疫情期间，这种线上的毕业生就业管理微信小程序，实现与用户只要连接网络就能查看招聘信息、宣教会、双选会等功能，更有效的减少了人流的聚集，减少感染几率，同时也大大的提高了工作效率，是本设计实现的意义所在。
    </p>
    <hr/>
    <h3>
     <a id="_17">
     </a>
     二.技术环境
    </h3>
    <p>
     jdk版本：1.8 及以上
     <br/>
     ide工具：Eclipse或者 IDEA，微信小程序开发工具
     <br/>
     数据库: mysql5.7 （必须5.7）
     <br/>
     编程语言: Java
     <br/>
     java框架：SpringBoot
     <br/>
     maven: 3.6.1
     <br/>
     详细技术：HTML+CSS+JAVA+SpringBoot+MYSQL+VUE+MAVEN+微信开发工具
    </p>
    <hr/>
    <h3>
     <a id="_28">
     </a>
     三.功能设计
    </h3>
    <p>
     本系统由毕业生、老师、企业单位、企业用户和管理员五大模块组成。用户界面显示在应用程序中，管理员界面显示在后台服务中，通过小程序端与服务端间进行数据交互与数据传输实现本系统的所有功能。
     <br/>
     小程序端页面的主要设计是：毕业生在注册登陆成功后，本系统实现底部导航栏页面设计，使用户在小程序端可以实现：首页、招聘信息、宣教会、双选会、我的（个人简历、投递简历、学生邀约、企业邀约、学生应约、企业应约、我的收藏管理）等功能。
     <br/>
     服务端有管理员直接管理：毕业生管理、老师管理、企业单位管理、企业用户管理、学院信息管理、专业信息管理、班级信息管理、资质申请管理、个人简历管理、招聘信息管理、宣教会管理、双选会管理、投递简历管理、学生邀约管理、企业邀约管理、学生应约管理、企业应约管理、已就业毕业生管理、企业用户申请管理、系统管理等信息。本网站模块设计的独立性强，用户体验良好、后期维护修改管理十分方便。
     <br/>
     老师服务端查看；系统首页、个人中心、毕业生管理、学生邀约管理、企业邀约管理、学生应约管理、企业应约管理等功能。
     <br/>
     企业单位服务端查看；系统首页、个人中心、资质申请管理、个人简历管理、招聘信息管理、宣教会管理、双选会管理、投递简历管理、学生邀约管理、企业邀约管理、 学生应约管理、企业应约管理等功能。
     <br/>
     企业用户服务端查看；系统首页、个人中心、企业用户申请管理等功能。
    </p>
