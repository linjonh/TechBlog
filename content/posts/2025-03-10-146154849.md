# python使用venv命令创建虚拟环境（ubuntu22）

## [专栏总录](https://blog.csdn.net/lstef/article/details/137601485 "专栏总录")

**使用ubuntu22为例**

命令如下：

## 一、创建虚拟环境

```
python3 -m venv 虚拟环境名称
```

## 二、激活虚拟环境

```
source 虚拟环境名称/bin/activate
```

## 三、退出虚拟环境命令

```
deactivate
```

## 四、删除虚拟环境

```
rm -rf 虚拟环境名称
```
