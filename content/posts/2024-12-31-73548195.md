    <p>
     树莓派的操作系统是基于Debian，所以不能正常显示中文，需要安装个中文字体库。
     <br/>
     树莓派1
     <br/>
     <strong>
      Raspbian系统中文化（中文支持、中文字体、中文输入法）
     </strong>
     <br/>
     <a href="http://www.guokr.com/post/520901/" rel="nofollow">
      http://www.guokr.com/post/520901/
     </a>
     <br/>
     树莓派3
     <br/>
     <a href="http://www.mz6.net/news/android/6864.html" rel="nofollow">
      http://www.mz6.net/news/android/6864.html
     </a>
    </p>
    <p>
     1.终端下 输入
    </p>
    <blockquote>
     <p>
      sudo raspi-config
     </p>
    </blockquote>
    <p>
     2.调出软件设计工具，选择5 语言 时区选择
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170621141327901?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
      <br/>
      3.然乎选择local
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170621141421665?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
       <br/>
       4.之后操作提示：按空格键在前面打勾或去掉勾（星号=勾），PageUp PageDown快速翻页，Tab键跳到OK按钮上
       <br/>
       <strong>
        去掉
       </strong>
       en_GB.UTF-8 UTF-8
       <br/>
       <strong>
        勾上
       </strong>
       “en_US.UTF-8 UTF-8”“zh_CN.UTF-8 UTF-8”“zh_CN.GBK GBK”
       <br/>
       <strong>
        下一屏
       </strong>
       幕默认语言选zh_CN.UTF-8
       <br/>
       <img alt="" src="https://img-blog.csdn.net/20170621141630718?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
        <br/>
        <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170621141641870?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
         <br/>
         5.之后 下载中文库终端输入：
        </img>
       </img>
      </img>
     </img>
    </p>
    <blockquote>
     <p>
      sudo apt-get install ttf-wqy-zenhei
     </p>
    </blockquote>
    <p>
     开始下载安装文泉驿中文字体库
     <br/>
     6.字体库安装后，浏览器可以正常显示中文，但系统依然是英文。可在主菜单中点击【Menu-&gt;Preferences-&gt;Raspberry Pi Configuraction-&gt;Localisaction-&gt;Set Locale】将系统语言设置成中文。
     <br/>
     <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170621141901005?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title="">
      <br/>
      7.OK。大功告成
      <br/>
      <img alt="这里写图片描述" src="https://img-blog.csdn.net/20170621141936902?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQveTUxMTM3NDg3NQ==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast" title=""/>
     </img>
    </p>
