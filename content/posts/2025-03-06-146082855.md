     Mac安装jdk教程
    </h1>
   </div>
  </div>

 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p id="">
     在Mac上安装JDK（Java Development Kit）的步骤如下：
    </p>
    <h4>
     一、下载JDK安装包
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        访问Oracle官网
       </strong>
       ：
      </p>
      <ul>
       <li>
        打开浏览器，访问
        <a href="https://www.oracle.com/java/technologies/javase-downloads.html" rel="nofollow" title="Oracle JDK下载页面">
         Oracle JDK下载页面
        </a>
        。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        选择JDK版本
       </strong>
       ：
      </p>
      <ul>
       <li>
        根据你的开发需求选择合适的JDK版本。例如，JDK 11、JDK 17或JDK 22等。
       </li>
       <li>
        注意选择与你的Mac芯片类型（Intel或Apple Silicon）相匹配的JDK版本。例如，如果你的Mac是M系列芯片，应选择Arm 64版本；如果是Intel芯片，选择x64版本。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        下载安装包
       </strong>
       ：
      </p>
      <ul>
       <li>
        点击下载链接，同意Oracle的许可协议（如果需要）。
       </li>
       <li>
        下载完成后，你将得到一个
        <code>
         .dmg
        </code>
        格式的安装包文件。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     二、安装JDK
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        打开安装包
       </strong>
       ：
      </p>
      <ul>
       <li>
        双击下载的
        <code>
         .dmg
        </code>
        文件，这将打开一个新的Finder窗口，显示JDK的安装程序。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        运行安装程序
       </strong>
       ：
      </p>
      <ul>
       <li>
        在Finder窗口中，找到并点击
        <code>
         .pkg
        </code>
        文件，这将启动JDK的安装向导。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        完成安装
       </strong>
       ：
      </p>
      <ul>
       <li>
        按照安装向导的提示，点击“继续”按钮。
       </li>
       <li>
        阅读并同意许可协议。
       </li>
       <li>
        选择安装目标位置（通常是默认位置）。
       </li>
       <li>
        点击“安装”按钮，等待安装完成。
       </li>
       <li>
        安装完成后，点击“关闭”按钮退出安装向导。
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     三、验证安装
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        打开终端
       </strong>
       ：
      </p>
      <ul>
       <li>
        你可以通过Spotlight搜索（按下Command + 空格键，然后输入“终端”并回车）快速找到并打开终端应用程序。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        检查JDK版本
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         在终端中输入以下命令，然后按下回车键：
        </p>
        <pre><code class="hljs">java -version</code></pre>
       </li>
       <li>
        <p id="">
         如果JDK安装成功，你将看到类似以下的输出，显示JDK的版本信息：
        </p>
        <pre><code class="hljs">openjdk version "11.0.xx" 202x-xx-xx
OpenJDK Runtime Environment AdoptOpenJDK (build 11.0.xx+xx)
OpenJDK 64-Bit Server VM AdoptOpenJDK (build 11.0.xx+xx, mixed mode)</code></pre>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     四、配置环境变量（可选）
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        查看JDK安装路径
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         如果你需要手动配置环境变量，首先需要知道JDK的安装路径。可以在终端中输入以下命令查看：
        </p>
        <pre><code class="hljs">/usr/libexec/java_home -V</code></pre>
       </li>
       <li>
        <p id="">
         这将列出所有已安装的JDK版本及其安装路径。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        编辑配置文件
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         根据你的Shell类型（通常是
         <code>
          bash
         </code>
         或
         <code>
          zsh
         </code>
         ），编辑相应的配置文件。
        </p>
        <ul>
         <li>
          对于
          <code>
           bash
          </code>
          ，编辑
          <code>
           ~/.bash_profile
          </code>
          文件。
         </li>
         <li>
          对于
          <code>
           zsh
          </code>
          ，编辑
          <code>
           ~/.zshrc
          </code>
          文件。
         </li>
        </ul>
       </li>
       <li>
        <p id="">
         打开终端，输入以下命令之一来编辑配置文件：
        </p>
        <pre><code class="hljs">vim ~/.bash_profile   # 如果使用bash
vim ~/.zshrc          # 如果使用zsh</code></pre>
       </li>
       <li>
        <p id="">
         或者，你也可以使用其他文本编辑器，如
         <code>
          nano
         </code>
         或
         <code>
          open -e
         </code>
         命令。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        添加环境变量
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         在配置文件的末尾添加以下行，将
         <code>
          [JDK_HOME]
         </code>
         替换为你的JDK安装路径：
        </p>
        <pre><code class="hljs">export JAVA_HOME=/Library/Java/JavaVirtualMachines/[JDK_HOME]/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH</code></pre>
       </li>
       <li>
        <p id="">
         例如，如果你的JDK安装路径是
         <code>
          /Library/Java/JavaVirtualMachines/jdk-11.0.2.jdk/Contents/Home
         </code>
         ，则添加：
        </p>
        <pre><code class="hljs">export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-11.0.2.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH</code></pre>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        保存并关闭文件
       </strong>
       ：
      </p>
      <ul>
       <li>
        在
        <code>
         vim
        </code>
        编辑器中，按
        <code>
         Esc
        </code>
        键退出编辑模式，然后输入
        <code>
         :wq
        </code>
        保存并退出。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        使配置生效
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         在终端中输入以下命令，使配置的环境变量立即生效：
        </p>
        <pre><code class="hljs">source ~/.bash_profile   # 如果使用bash
source ~/.zshrc          # 如果使用zsh</code></pre>
       </li>
      </ul>
     </li>
    </ol>
    <h4>
     五、注意事项
    </h4>
    <ol>
     <li>
      <p id="">
       <strong>
        权限问题
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p id="">
         如果在编辑配置文件时遇到权限问题，可以在命令前添加
         <code>
          sudo
         </code>
         来提升权限，例如：
        </p>
        <pre><code class="hljs">sudo vim ~/.bash_profile</code></pre>
       </li>
       <li>
        <p id="">
         输入密码后，即可进行编辑。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        多个JDK版本
       </strong>
       ：
      </p>
      <ul>
       <li>
        如果你需要在同一台Mac上管理多个JDK版本，可以考虑使用
        <code>
         sdkman
        </code>
        或
        <code>
         jabba
        </code>
        等工具来切换JDK版本。
       </li>
      </ul>
     </li>
     <li>
      <p id="">
       <strong>
        安装路径
       </strong>
       ：
      </p>
      <ul>
       <li>
        默认情况下，JDK会被安装在
        <code>
         /Library/Java/JavaVirtualMachines/
        </code>
        目录下。如果你选择了自定义安装路径，请确保在配置环境变量时使用正确的路径。
       </li>
      </ul>
     </li>
    </ol>
    <p id="">
     通过以上步骤，你应该能够在Mac上成功安装和配置JDK，为Java开发做好准备。
    </p>
