    <h2>
     2.讲解算法原理
    </h2>
    <p>
     使用当前数组中最大的数将它减半，，直到数组和减小到一半为止，从而快速达到目的
    </p>
    <p>
     重点是找到最大数，可以采用大根堆快速达到目的
    </p>
    <p>
    </p>
    <h2>
     3.代码
    </h2>
    <pre><code class="language-java">class Solution {
    public int halveArray(int[] nums) {
        PriorityQueue&lt;Double&gt; heap=new PriorityQueue&lt;&gt;((a,b)-&gt;b.compareTo(a));//创建大根堆
        double sum=0;
        for(int x:nums){
            heap.offer((double)x);
            sum+=x;
        }
        int count=0;
        sum/=2.0;
        while(sum&gt;0){
            double tmp=heap.poll()/2.0;
            sum-=tmp;
            count++;
            heap.offer(tmp);
        }
        return count;
    }
}</code></pre>
