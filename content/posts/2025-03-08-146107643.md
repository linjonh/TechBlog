    <h2>
     <a id="_0">
     </a>
     说在前面
    </h2>
    <blockquote>
     <ul>
      <li>
       操作系统：win11
      </li>
      <li>
       docker desktop版本：4.29.0
      </li>
      <li>
       docker engin版本：v26.0.0
      </li>
      <li>
       ragflow版本：nightly
      </li>
     </ul>

    <h2>
     <a id="docker_6">
     </a>
     安装docker
    </h2>
    <ul>
     <li>
      <a href="https://docs.docker.com/desktop/setup/install/windows-install/" rel="nofollow">
       官网
      </a>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/6df897a8e0604d2c8616c08b563b0a26.png"/>
     </li>
     <li>
      如果是win11，backend建议使用wsl2
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c0014d6ea0ed4bd68372e7d0925c3678.png"/>
     </li>
     <li>
      安装好后打开docker desktop，不然docker命令用不了
     </li>
    </ul>
    <h2>
     <a id="RAGFlow_12">
     </a>
     下载RAGFlow源码
    </h2>
    <ul>
     <li>
      直接git就行
      <pre><code class="prism language-shell"><span class="token function">git</span> clone https://github.com/infiniflow/ragflow.git

     </li>
     <li>
      修改相关配置，在build时候走国内源

     <li>
      修改Dockerfile
      <br/>
      将
      <code>
       ARG NEED_MIRROR=0
      </code>
      改为
      <code>
       ARG NEED_MIRROR=1
      </code>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/7bad498ef0504ce3abf64809db174df4.png"/>
     </li>
     <li>
      修改.env
      <br/>
      将以下红框内的取消注释
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/3c69251c7ef645c18b4120d0da5c4afe.png"/>
