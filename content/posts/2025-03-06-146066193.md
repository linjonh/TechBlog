    <h3>
     <a id="2_huggingface_9">
     </a>
     2. huggingface平台配置
    </h3>
    <p>
     <a href="https://huggingface.co/" rel="nofollow">
      huggingface官网
     </a>
    </p>
    <ul>
     <li>
      注册登录
     </li>
     <li>
      申请token（要有写权限）
     </li>
     <li>
      安装客户端
     </li>
    </ul>
    <pre><code class="prism language-bash"><span class="token comment"># 安装</span>
pip <span class="token function">install</span> <span class="token parameter variable">-U</span> huggingface_hub
<span class="token comment"># 登录并保存 Git 凭据</span>
huggingface-cli login <span class="token parameter variable">--token</span> <span class="token variable">$HUGGINGFACE_TOKEN</span> --add-to-git-credential
<span class="token comment"># 下载模型到指定目录</span>
huggingface-cli download DeepSeek-R1 --local-dir ./models
<span class="token comment">########## 常用指令如下</span>
<span class="token comment"># 登录</span>
huggingface-cli login
<span class="token comment"># 下载模型或数据集到本地</span>
huggingface-cli download
<span class="token comment"># 查看当前登录的账户信息，验证登录状态</span>
huggingface-cli <span class="token function">whoami</span>
<span class="token comment"># 输出当前环境配置</span>
huggingface-cli <span class="token function">env</span>
<span class="token comment"># 在 Hugging Face Hub 上创建新仓库</span>
huggingface-cli repo create
<span class="token comment"># 上传文件或目录到 Hugging Face 仓库</span>
huggingface-cli upload

通用参数
--help：查看指令帮助
<span class="token parameter variable">--endpoint</span> <span class="token operator">&lt;</span>URL<span class="token operator">&gt;</span>：指定 Hugging Face 镜像源
</code></pre>

    <h3>
     <a id="3__41">
     </a>
     3. 记录数据集
    </h3>
    <p>
     在笔记本电脑上运行以下命令以录制 2 集并将数据集上传到中心，可以通过添加 --control.resume=true 来恢复录制。
    </p>
    <pre><code class="prism language-bash"><span class="token comment"># 将 Hugging Face 库名存储在变量中</span>
<span class="token assign-left variable">HF_USER</span><span class="token operator">=</span><span class="token variable"><span class="token variable">$(</span>huggingface-cli <span class="token function">whoami</span> <span class="token operator">|</span> <span class="token function">head</span> <span class="token parameter variable">-n</span> <span class="token number">1</span><span class="token variable">)</span></span>
<span class="token builtin class-name">echo</span> <span class="token variable">$HF_USER</span>
<span class="token comment"># 录制并上传</span>
python lerobot/scripts/control_robot.py <span class="token punctuation">\</span>
  <span class="token parameter variable">--robot.type</span><span class="token operator">=</span>lekiwi <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.type</span><span class="token operator">=</span>record <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.fps</span><span class="token operator">=</span><span class="token number">30</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.single_task</span><span class="token operator">=</span><span class="token string">"拿起一块乐高积木，把它放进垃圾桶里"</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.repo_id</span><span class="token operator">=</span><span class="token variable">${HF_USER}</span>/lekiwi_test <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.tags</span><span class="token operator">=</span><span class="token string">'["tutorial"]'</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.warmup_time_s</span><span class="token operator">=</span><span class="token number">5</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.episode_time_s</span><span class="token operator">=</span><span class="token number">30</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.reset_time_s</span><span class="token operator">=</span><span class="token number">30</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.num_episodes</span><span class="token operator">=</span><span class="token number">2</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.push_to_hub</span><span class="token operator">=</span>true
</code></pre>
    <h3>
     <a id="4__62">
     </a>
     4. 数据集查看
    </h3>
    <p>
     如果录制时候使用–control.push_to_hub=true将数据集上传到云端了，则通过下面指令查到库ID后，在线上直接查看数据集。
    </p>
    <pre><code class="prism language-bash"><span class="token builtin class-name">echo</span> <span class="token variable">${HF_USER}</span>/lekiwi_test
</code></pre>
    <p>
     如果配置为-control.push_to_hub=falsehttp，未上传到hugging face云端，则通过下面指令后，在浏览器查看：http://127.0.0.1:9090
    </p>
    <pre><code class="prism language-bash">python lerobot/scripts/visualize_dataset_html.py <span class="token punctuation">\</span>
  --repo-id <span class="token variable">${HF_USER}</span>/lekiwi_test <span class="token punctuation">\</span>
  --local-files-only <span class="token number">1</span>
</code></pre>
    <h3>
     <a id="5__76">
     </a>
     5. 数据集回放
    </h3>
    <p>
     下面指令可以查看episode编号为0的数据集
    </p>
    <pre><code class="prism language-bash">python lerobot/scripts/control_robot.py <span class="token punctuation">\</span>
  <span class="token parameter variable">--robot.type</span><span class="token operator">=</span>lekiwi <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.type</span><span class="token operator">=</span>replay <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.fps</span><span class="token operator">=</span><span class="token number">30</span> <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.repo_id</span><span class="token operator">=</span><span class="token variable">${HF_USER}</span>/lekiwi_test <span class="token punctuation">\</span>
  <span class="token parameter variable">--control.episode</span><span class="token operator">=</span><span class="token number">0</span>
</code></pre>
