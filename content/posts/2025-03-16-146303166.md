    <h2>
     <a id="esxi65_16">
     </a>
     esxi6.5频繁退出登录
    </h2>
    <h3>
     <a id="_17">
     </a>
     表现
    </h3>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d822b39a8406486993a93f9d7dc614b4.png">
      <br/>
      传文件频繁退出登录
      <br/>
      操作过程非常卡
      <br/>
      所有的虚拟机出现断开连接
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/caf4d22a673940d691875adf0ea0dea1.png"/>
     </img>
    </p>
    <h3>
     <a id="_23">
     </a>
     解决方案
    </h3>
    <p>
     为vcenter和esxi分配许可证。
     <br/>
     6.5许可证：
     <br/>
     VMware vCenter Server 6 Standard ：NG6D0-DRH12-H89G0-7L2ZH-8KK24（vcenter系统许可）
     <br/>
     VMware vCloud Suite 6 Enterprise for vSphere (CPU)：4U4E2-2DJ5N-M84D0-LKAE0-3P8KD（esxi主机许可）
    </p>
