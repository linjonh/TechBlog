    <h3>
     <a id="_0">
     </a>
     <strong>
      前言
     </strong>
    </h3>
    <p>
     Android Studio 是官方推荐的 Android 应用开发环境，它集成了 Android SDK、模拟器和强大的代码编辑工具。本篇文章将详细介绍如何在 macOS 上安装 Android Studio，并完成基本配置。
    </p>
    <hr/>
    <h3>
     <a id="1__Android_Studio_5">
     </a>
     <strong>
      1. 下载与安装 Android Studio
     </strong>
    </h3>
    <h4>
     <a id="11__Android_Studio_7">
     </a>
     <strong>
      1.1 下载 Android Studio
     </strong>
    </h4>
    <ol>
     <li>
      访问
      <a href="https://developer.android.com/studio" rel="nofollow">
       Android Studio 官方网站
      </a>
      。
     </li>
     <li>
      点击
      <strong>
       “Download Android Studio”
      </strong>
      ，下载适用于 macOS 的
      <code>
       .dmg
      </code>
      安装包。
     </li>
     <li>
      同意许可协议，等待下载完成。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/c7f110ee1b314348962fae232fc766d7.png"/>
    </p>
    <h4>
     <a id="12__Android_Studio_14">
     </a>
     <strong>
      1.2 安装 Android Studio
     </strong>
    </h4>
    <ol>
     <li>
      双击下载的
      <code>
       .dmg
      </code>
      文件。
     </li>
     <li>
      将
      <strong>
       Android Studio
      </strong>
      图标拖动到
      <code>
       Applications
      </code>
      文件夹。
     </li>
     <li>
      进入
      <code>
       Applications
      </code>
      文件夹，双击
      <strong>
       Android Studio
      </strong>
      运行。
     </li>
     <li>
      如果 macOS 提示安全警告，点击
      <strong>
       “打开”
      </strong>
      。
     </li>
    </ol>
    <h4>
     <a id="13__Toolbox_App__20">
     </a>
     <strong>
      1.3 通过 Toolbox App 安装
     </strong>
    </h4>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/50f186055fb345c39d1c9fcdbda75b92.png"/>
    </p>
    <hr/>
    <h3>
     <a id="2__Android_Studio_26">
     </a>
     <strong>
      2. 初始配置 Android Studio
     </strong>
    </h3>
    <h4>
     <a id="21__28">
     </a>
     <strong>
      2.1 启动向导
     </strong>
    </h4>
    <p>
     首次启动 Android Studio 时，会进入
     <strong>
      Setup Wizard（安装向导）
     </strong>
     ，按以下步骤进行设置：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        导入设置
       </strong>
      </p>
      <ul>
       <li>
        选择
        <strong>
         “Do not import settings”
        </strong>
        （不导入旧配置），点击
        <strong>
         OK
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择安装类型
       </strong>
      </p>
      <ul>
       <li>
        选择
        <strong>
         “Custom”
        </strong>
        （自定义），点击
        <strong>
         Next
        </strong>
        。
       </li>
       <li>
        （如果选择 “Standard” 也可以，但自定义模式可以让你更自由地选择安装组件。）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择 UI 主题
       </strong>
      </p>
      <ul>
       <li>
        选择
        <strong>
         “Darcula”
        </strong>
        （推荐，深色护眼）或
        <strong>
         “Light”
        </strong>
        。
       </li>
       <li>
        点击
        <strong>
         Next
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        选择安装组件
       </strong>
      </p>
      <ul>
       <li>
        <strong>
         Android SDK
        </strong>
        （必选）
       </li>
       <li>
        <strong>
         Android SDK Platform
        </strong>
        （必选）
       </li>
       <li>
        <strong>
         Android Virtual Device（可选，模拟器）
        </strong>
       </li>
       <li>
        <strong>
         Intel HAXM 加速（推荐）
        </strong>
       </li>
       <li>
        点击
        <strong>
         Next
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        设置 SDK 路径
       </strong>
      </p>
      <ul>
       <li>
        默认路径为
        <code>
         ~/Library/Android/sdk
        </code>
        ，建议保持默认。
       </li>
       <li>
        点击
        <strong>
         Next
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        接受许可协议
       </strong>
      </p>
      <ul>
       <li>
        在 “License Agreement” 界面，勾选
        <strong>
         Accept
        </strong>
        ，点击
        <strong>
         Next
        </strong>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        下载组件
       </strong>
      </p>
      <ul>
       <li>
        Android Studio 会自动下载 SDK 及相关工具，耐心等待。
       </li>
       <li>
        下载完成后，点击
        <strong>
         Finish
        </strong>
        结束安装。
       </li>
      </ul>
     </li>
    </ol>

    <h3>
     <a id="3__SDK__67">
     </a>
     <strong>
      3. 配置代理（解决 SDK 下载问题）
     </strong>
    </h3>
    <p>
     如果 Android Studio 在下载 SDK 组件时遇到网络访问问题，可以手动配置代理。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4ba3749b35284c9ab1627c442495af86.png"/>
    </p>
    <h4>
     <a id="31__Android_Studio__72">
     </a>
     <strong>
      3.1 在 Android Studio 内设置代理
     </strong>
    </h4>
    <ol>
     <li>
      打开 Android Studio，进入：
      <ul>
       <li>
        <code>
         Preferences
        </code>
        →
        <code>
         Appearance &amp; Behavior
        </code>
        →
        <code>
         System Settings
        </code>
        →
        <code>
         HTTP Proxy
        </code>
       </li>
      </ul>
     </li>
     <li>
      选择
      <strong>
       “Manual proxy configuration”
      </strong>
      ，输入：
      <ul>
       <li>
        <strong>
         Host name
        </strong>
        :
        <code>
         127.0.0.1
        </code>
       </li>
       <li>
        <strong>
         Port number
        </strong>
        :
        <code>
         7897
        </code>
        （根据你的代理工具配置填写）
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       Check connection
      </strong>
      测试代理，成功后点击
      <strong>
       OK
      </strong>
      。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/4ba27da9fc374408ae53137f3f5b3261.png"/>
    </p>
    <hr/>
    <h3>
     <a id="4_SDK__84">
     </a>
     <strong>
      4. SDK 管理与更新
     </strong>
    </h3>
    <h4>
     <a id="41__SDK_Manager__SDK_86">
     </a>
     <strong>
      4.1 通过 SDK Manager 管理 SDK
     </strong>
    </h4>
    <p>
     SDK Manager 用于管理 Android SDK 组件，确保你的开发环境保持最新。
    </p>
    <ol>
     <li>
      在 Android Studio，打开
      <code>
       Preferences
      </code>
      →
      <code>
       Appearance &amp; Behavior
      </code>
      →
      <code>
       System Settings
      </code>
      →
      <code>
       Android SDK
      </code>
      。
     </li>
     <li>
      在
      <strong>
       SDK Platforms
      </strong>
      选项卡，勾选你需要的 Android 版本（建议安装最新稳定版）。
     </li>
     <li>
      在
      <strong>
       SDK Tools
      </strong>
      选项卡，确保以下工具已安装：
      <ul>
       <li>
        Android SDK Build-Tools
       </li>
       <li>
        Android Emulator（如果使用模拟器）
       </li>
       <li>
        Google Play Services（如果需要）
       </li>
       <li>
        Intel HAXM（如果使用 Intel 处理器的模拟器加速）
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       Apply
      </strong>
      进行更新。
     </li>
    </ol>
    <h4>
     <a id="42__SDK__98">
     </a>
     <strong>
      4.2 使用国内镜像加速 SDK 下载（可选）
     </strong>
    </h4>
    <p>
     如果 SDK 下载速度较慢，可以配置国内镜像源：
    </p>
    <ol>
     <li>
      在
      <code>
       Preferences
      </code>
      →
      <code>
       Appearance &amp; Behavior
      </code>
      →
      <code>
       System Settings
      </code>
      →
      <code>
       Android SDK
      </code>
      。
     </li>
     <li>
      进入
      <strong>
       SDK Update Sites
      </strong>
      ，手动添加：
      <ul>
       <li>
        <code>
         https://mirrors.tencent.com/android/repository/
        </code>
       </li>
       <li>
        <code>
         https://mirrors.aliyun.com/android/repository/
        </code>
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       Apply
      </strong>
      ，然后重新尝试下载 SDK。
     </li>
    </ol>
    <hr/>
    <h3>
     <a id="5__Android_AVD_108">
     </a>
     <strong>
      5. 配置 Android 模拟器（AVD）
     </strong>
    </h3>
    <p>
     如果你没有 Android 真机，可以使用
     <strong>
      Android Virtual Device (AVD)
     </strong>
     进行测试。
    </p>
    <h4>
     <a id="51__111">
     </a>
     <strong>
      5.1 创建虚拟设备
     </strong>
    </h4>
    <ol>
     <li>
      打开
      <code>
       AVD Manager
      </code>
      ：
      <ul>
       <li>
        在 Android Studio 右上角点击
        <strong>
         AVD Manager
        </strong>
        图标。
       </li>
       <li>
        或者
        <code>
         Tools
        </code>
        →
        <code>
         Device Manager
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       “Create Virtual Device”
      </strong>
      。
     </li>
     <li>
      选择设备型号（如 Pixel 6），点击
      <strong>
       Next
      </strong>
      。
     </li>
     <li>
      选择系统镜像（建议选择最新版本）。
     </li>
     <li>
      配置设备参数（如 RAM、存储大小）。
     </li>
     <li>
      点击
      <strong>
       Finish
      </strong>
      ，完成虚拟机创建。
     </li>
    </ol>
    <h4>
     <a id="52__121">
     </a>
     <strong>
      5.2 启动模拟器
     </strong>
    </h4>
    <ul>
     <li>
      在
      <code>
       AVD Manager
      </code>
      ，选择刚创建的设备，点击
      <strong>
       Run
      </strong>
      。
     </li>
     <li>
      如果模拟器启动缓慢，可以在
      <code>
       Preferences
      </code>
      →
      <code>
       Advanced
      </code>
      里调整
      <strong>
       HAXM 配置
      </strong>
      ，提高运行效率。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="6__Android__127">
     </a>
     <strong>
      6. 运行第一个 Android 应用
     </strong>
    </h3>
    <h4>
     <a id="61__129">
     </a>
     <strong>
      6.1 创建新项目
     </strong>
    </h4>
    <ol>
     <li>
      打开 Android Studio，点击
      <strong>
       “Start a new Android Studio project”
      </strong>
      。
     </li>
     <li>
      选择
      <strong>
       “Empty Activity”
      </strong>
      模板，点击
      <strong>
       Next
      </strong>
      。
     </li>
     <li>
      设置：
      <ul>
       <li>
        <strong>
         Project Name
        </strong>
        （项目名称）
       </li>
       <li>
        <strong>
         Package Name
        </strong>
        （包名）
       </li>
       <li>
        <strong>
         Language
        </strong>
        （选择 Kotlin 或 Java）
       </li>
       <li>
        <strong>
         Minimum API Level
        </strong>
        （建议 Android 7.0 及以上）
       </li>
      </ul>
     </li>
     <li>
      点击
      <strong>
       Finish
      </strong>
      。
     </li>
    </ol>
    <h4>
     <a id="62__139">
     </a>
     <strong>
      6.2 运行应用
     </strong>
    </h4>
    <ol>
     <li>
      连接真机（打开 USB 调试模式），或启动 AVD 模拟器。
     </li>
     <li>
      点击
      <strong>
       “Run”
      </strong>
      （绿色 ▶ 按钮）。
     </li>
     <li>
      如果一切正常，Android Studio 将编译并运行你的应用！
     </li>
    </ol>
