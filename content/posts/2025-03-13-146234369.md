    <ol>
     <li>
      PC3不允许访问其他PC和Server1
     </li>
     <li>
      PC2允许访问Server1服务器，不允许其他PC访问
     </li>
     <li>
      各设备IP配置如图所示，这里不做展示
     </li>
    </ol>
    <p>
     AR1接口vlan配置
    </p>
    <pre><code class="prism language-bash">vlan batch <span class="token number">10</span> <span class="token number">20</span> <span class="token number">30</span>

interface Vlanif20
 <span class="token function">ip</span> address <span class="token number">192.168</span>.2.254 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>
interface Vlanif30
 <span class="token function">ip</span> address <span class="token number">192.168</span>.3.254 <span class="token number">255.255</span>.255.0 
<span class="token comment">#</span>

<span class="token comment">#</span>

 port link-type access

<span class="token comment">#</span>

    <p>
     AR2接口配置
    </p>
    <pre><code class="prism language-bash">interface Vlanif1
 <span class="token function">ip</span> address <span class="token number">172.16</span>.1.254 <span class="token number">255.255</span>.255.0
<span class="token comment">#</span>
interface GigabitEthernet0/0/0
 <span class="token function">ip</span> address <span class="token number">10.0</span>.0.2 <span class="token number">255.255</span>.255.0
<span class="token comment">#</span>
<span class="token function">ip</span> route-static <span class="token number">0.0</span>.0.0 <span class="token number">0.0</span>.0.0 <span class="token number">10.0</span>.0.1
</code></pre>
    <p>
     下面开始配置基本ALC
     <br/>
     AR1配置ACL：PC3不允许访问其他PC和Server1
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>acl <span class="token number">2000</span>	<span class="token comment"># 配置ACL2000</span>
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>rule deny <span class="token builtin class-name">source</span> <span class="token number">172.16</span>.1.0 <span class="token number">0.0</span>.0.255	<span class="token comment"># 设置拒绝源IP地址流量</span>
<span class="token punctuation">[</span>AR1-acl-basic-2000<span class="token punctuation">]</span>quit
<span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>interface GigabitEthernet <span class="token number">0</span>/0/0	<span class="token comment"># 进入接口试图</span>
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/0<span class="token punctuation">]</span>traffic-filter inbound acl <span class="token number">2000</span>	<span class="token comment"># 把acl2000策略应用到该接口上，检测入站流量进行匹配安全策略</span>
<span class="token punctuation">[</span>AR1-GigabitEthernet0/0/0<span class="token punctuation">]</span>quit
</code></pre>
    <p>
     AR1配置ACL：PC2允许访问Server1服务器，不允许其他PC访问
    </p>
    <pre><code class="prism language-bash"><span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>acl <span class="token number">2010</span>
<span class="token punctuation">[</span>AR1-acl-basic-2010<span class="token punctuation">]</span>rule permit <span class="token builtin class-name">source</span> <span class="token number">192.168</span>.2.0 <span class="token number">0.0</span>.0.255	<span class="token comment"># 设置允许该网段的流量通过</span>
<span class="token punctuation">[</span>AR1-acl-basic-2010<span class="token punctuation">]</span>rule deny		<span class="token comment"># 拒绝所有流量（虽然是拒绝所有，但是会优先匹配上面一条规则）</span>
<span class="token punctuation">[</span>AR1-acl-basic-2010<span class="token punctuation">]</span>quit
<span class="token punctuation">[</span>AR1<span class="token punctuation">]</span>interface Vlanif <span class="token number">30</span>	<span class="token comment"># 进入vlan30视图</span>
<span class="token punctuation">[</span>AR1-Vlanif30<span class="token punctuation">]</span>traffic-filter outbound acl <span class="token number">2010</span>		<span class="token comment"># 将acl2010应用到vlan30接口上</span>
<span class="token punctuation">[</span>AR1-Vlanif30<span class="token punctuation">]</span>quit
</code></pre>
