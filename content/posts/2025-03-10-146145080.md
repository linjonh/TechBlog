    <h3>
     一、拓扑图
    </h3>
    <p>
     <img alt="" height="375" src="https://i-blog.csdnimg.cn/direct/f01db9114ed54bca8dfda6d55cfa1cb1.png" width="1158"/>

    <h3>
     二、实验思路
    </h3>
    <ol>
     <li>
      根据BGP的路由优先原则，首先要保证路由可达。
     </li>
     <li>
      但是IBGP间存在水平分割机制，因此实验可使用IBGP全互联，反射器或联盟来实现IBGP间路由可达，本实验使用全互联
     </li>
     <li>
      全互联缺点：将BGP路由引入到IGP（内部网关协议，如：OSPF IS-IS RIP），从而保证IGP与BGP的同步。但是，因为Internet上的BGP路由数量十分庞大，一旦引入到IGP，会给IGP路由器带来巨大的处理和存储负担，如果路由器负担过重，则可能瘫痪，因此不建议使用。
     </li>
    </ol>
    <h3>
     三、实验配置
    </h3>
    <h4>
     （1）先为每个路由器的接口配置IP地址
    </h4>
    <ul>
     <li>
      R1
      <br/>
      其中interface LoopBack 0可不配置，因为从ebgp到ibgp只有一跳（即R1到R2的生命周期只有1），但是实际应用场景中，R1在AS1内是ibgp，因此配置LoopBack 0 可以增强网络的可靠性。
      <pre><code>&lt;Huawei&gt;sys
[Huawei]sys R1
[R1]interface GigabitEthernet 0/0/0
[R1-GigabitEthernet0/0/0]ip add 12.0.0.1 24
[R1-GigabitEthernet0/0/0]qu	
[R1]interface LoopBack 0
[R1-LoopBack0]ip add 1.1.1.1 32
[R1-LoopBack0]qu</code></pre>
     </li>
     <li>
      R2
      <pre><code>&lt;Huawei&gt;sys
[Huawei]sys R2
[R2]int g0/0/1
[R2-GigabitEthernet0/0/1]ip add 12.0.0.2 24
[R2-GigabitEthernet0/0/1]int g0/0/0
[R2-GigabitEthernet0/0/0]ip add 23.0.0.1 24
[R2-GigabitEthernet0/0/1]qu		
[R2]int LoopBack 0
[R2-LoopBack0]ip add 2.2.2.2 32
[R2-LoopBack0]qu</code></pre>
     </li>
     <li>
      R3
      <pre><code>&lt;Huawei&gt;sys
[Huawei]sys R3
[R3]int g0/0/1
[R3-GigabitEthernet0/0/1]ip add 23.0.0.2 24
[R3-GigabitEthernet0/0/1]int g0/0/0
[R3-GigabitEthernet0/0/0]ip add 34.0.0.1 24
[R3-GigabitEthernet0/0/0]qu
[R3]int loop0
[R3-LoopBack0]ip add 3.3.3.3 32
[R3-LoopBack0]qu</code></pre>
     </li>
     <li>
      <p>
       R4
      </p>
      <pre><code>&lt;Huawei&gt;sys
[Huawei]sys R4
[R4]int g0/0/1
[R4-GigabitEthernet0/0/1]ip add 34.0.0.2 24
[R4-GigabitEthernet0/0/1]int g0/0/0
[R4-GigabitEthernet0/0/0]ip add 45.0.0.1 24
[R4-GigabitEthernet0/0/0]qu
[R4]int loop 0
[R4-LoopBack0]ip add 4.4.4.4 32
[R4-LoopBack0]qu</code></pre>
     </li>
     <li>
      <p>
       R5
       <br/>
       其中interface LoopBack 0可不配置，因为从ebgp到ibgp只有一跳（即R5到R4的生命周期只有1），但是实际应用场景中，R5在AS3内是ibgp，因此配置LoopBack 0 可以增强网络的可靠性。
      </p>
      <pre><code>&lt;Huawei&gt;sys
[Huawei]sys R5
[R5]int g0/0/0
[R5-GigabitEthernet0/0/0]ip add 45.0.0.2 24
[R5-GigabitEthernet0/0/0]qu
[R5]int loop0
[R5-LoopBack0]ip add 5.5.5.5 32
[R5-LoopBack0]qu</code></pre>
     </li>
    </ul>
    <h4>
     （2）在AS2内建立全互联
    </h4>
    <ul>
     <li>
      R2
      <pre><code>[R2]ospf 1 router-id 2.2.2.2
[R2-ospf-1]area 0
[R2-ospf-1-area-0.0.0.0]network 23.0.0.0 0.0.0.255
[R2-ospf-1-area-0.0.0.0]network 2.2.2.2 0.0.0.0</code></pre>
     </li>
     <li>
      R3
      <pre><code>[R3]ospf 1 router-id 3.3.3.3
[R3-ospf-1]area 0	
[R3-ospf-1-area-0.0.0.0]network 23.0.0.0 0.0.0.255
[R3-ospf-1-area-0.0.0.0]network 2.2.2.2 0.0.0.0
[R3-ospf-1-area-0.0.0.0]network 34.0.0.0 0.0.0.255</code></pre>
     </li>
     <li>
      R4
      <pre><code>[R4]ospf 1 router-id 4.4.4.4
[R4-ospf-1]area 0
[R4-ospf-1-area-0.0.0.0]network 4.4.4.4 0.0.0.0
[R4-ospf-1-area-0.0.0.0]network 34.0.0.0 0.0.0.255
</code></pre>

    <h4>
     （4）配置BGP邻居
    </h4>
    <ul>
     <li>
      R1
      <pre><code>[R1]bgp 1
[R1-bgp]router-id 1.1.1.1
[R1-bgp]peer 12.0.0.2 as-number 2</code></pre>
     </li>
     <li>
      R2
      <pre><code>[R2]bgp 2
[R2-bgp]router-id 2.2.2.2
[R2-bgp]peer 12.0.0.1 as-number 1
[R2-bgp]peer 3.3.3.3 as-number 2	
[R2-bgp]peer 3.3.3.3 connect-interface LoopBack 0
[R2-bgp]peer 4.4.4.4 as-number 2
[R2-bgp]peer 4.4.4.4 connect-interface LoopBack 0</code></pre>
     </li>
     <li>
      R3
      <pre><code>[R3]bgp 2
[R3-bgp]router-id 3.3.3.3
[R3-bgp]peer 2.2.2.2 as-number 2
[R3-bgp]peer 2.2.2.2 connect-interface LoopBack 0
[R3-bgp]peer 4.4.4.4 as-number 2
[R3-bgp]peer 4.4.4.4 connect-interface LoopBack 0
</code></pre>
     </li>

      R4

[R4]bgp 2

[R4-bgp]peer 3.3.3.3 as-number 2
[R4-bgp]peer 3.3.3.3 connect-interface LoopBack 0
[R4-bgp]peer 45.0.0.0 as-number 3</code></pre>

    <h4>
     （6）R1和R5发布所属网络路由，在R2和R4修改下一跳，实现AS1和AS2互通
    </h4>
    <ul>
     <li>
      R1
      <pre><code>[R1]bgp 1
