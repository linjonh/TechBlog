    <h3>
     Docker 和 Caddy 简介
    </h3>
    <p>
     Docker 是一个开源的平台，允许开发者打包应用及其依赖，形成一个轻量级的容器。这样，应用可以在任何环境中保持一致，减少“在我电脑上可以运行”的烦恼。Caddy 是一个现代的 HTTP/2 服务器，自动为你提供 HTTPS，配置简单，适合快速部署和开发。
    </p>
    <p>
     本次实验使用的是三丰云的免费服务器，配置为 1 核 CPU、1G 内存和 5M 带宽。这样的配置在本次实验中相对合适，整体体验尚可。
    </p>
    <h3>
     在 Ubuntu 上通过 Docker 部署 Caddy 服务器的详细步骤
    </h3>
    <h4>
     步骤 1: 安装 Docker
    </h4>
    <p>
     首先，确保你的 Ubuntu 系统上已经安装了 Docker。可以使用以下命令进行安装：
    </p>
    <pre><code>sudo apt update
sudo apt install -y docker.io
</code></pre>
    <h4>
     步骤 2: 启动 Docker 服务
    </h4>
    <p>
     安装完成后，启动 Docker 服务并设置为开机自启：
    </p>
    <pre><code>sudo systemctl start docker
sudo systemctl enable docker
</code></pre>
    <h4>
     步骤 3: 拉取 Caddy 镜像
    </h4>
    <p>
     接下来，拉取 Caddy 的 Docker 镜像：
    </p>
    <pre><code>sudo docker pull caddy:latest
</code></pre>
    <h4>
     步骤 4: 创建 Caddyfile 配置文件
    </h4>
    <p>
     创建一个名为
     <code>
      Caddyfile
     </code>
     的配置文件，内容如下（根据你的需求调整）：
    </p>
    <pre><code>:80 {
    respond "Hello, World!" 200
}
</code></pre>
    <h4>
     步骤 5: 运行 Caddy 容器
    </h4>
    <p>
     使用以下命令启动 Caddy 服务器，确保将
     <code>
      Caddyfile
     </code>
     的路径替换为你的实际路径：
    </p>
    <pre><code>sudo docker run -d -p 80:80 --name my_caddy -v /path/to/Caddyfile:/etc/caddy/Caddyfile caddy:latest
</code></pre>
    <h4>
     参数说明
    </h4>
    <ul>
     <li>
      <code>
       -d
      </code>
      ：后台运行容器。
     </li>
     <li>
      <code>
       -p 80:80
      </code>
      ：将主机的 80 端口映射到容器的 80 端口。
     </li>
     <li>
      <code>
       --name my_caddy
      </code>
      ：为容器指定一个名称。
     </li>
     <li>
      <code>
       -v /path/to/Caddyfile:/etc/caddy/Caddyfile
      </code>
      ：挂载本地的
      <code>
       Caddyfile
      </code>
      到容器内。
     </li>
    </ul>
    <h4>
     步骤 6: 验证 Caddy 服务器
    </h4>
    <p>
     打开浏览器，访问
     <code>
      http://你的服务器IP
     </code>
     ，如果看到 "Hello, World!"，说明 Caddy 服务器已成功运行。
    </p>
    <h3>
     总结
    </h3>
    <p>
     通过以上步骤，你已经在 Ubuntu 系统下成功通过 Docker 部署了 Caddy 服务器。这个过程简单高效，适合快速开发和测试。希望这个教程能帮助到你！
    </p>
