    <p>
     滑动窗口是什么？
    </p>
    <blockquote>
     <p>
      滑动窗口可以归为快慢双指针，一快一慢两个指针前后相随，中间的部分就是窗口。
      <strong>
       滑动窗口算法技巧主要用来解决子数组问题，比如让你寻找符合某个条件的最长/最短子数组
      </strong>
      。
     </p>
    </blockquote>
    <p>
     这道题目说人话版本：
    </p>
    <blockquote>
     <p>
      不含重复字符的子串：在这个连续的片段中，每个字符只出现一次。
      <br/>
      子串：是指一个字符串中的连续一段字符。
      <br/>
      在所有满足“不含重复字符”的子串中，找出那个字符个数最多的子串，并返回这个子串的长度。
     </p>
    </blockquote>

     <a id="_83">
     </a>
     问题与解答
    </h3>
    <blockquote>
     <p>
      [NOTE] 问题1
      <br/>
      getOrDefault()的用法？
      <br/>
      <code>
       getOrDefault
      </code>
      是 Java 中
      <code>
       Map
      </code>
      接口提供的一个方法，用于在获取指定 key 对应的值时，如果该 key 不存在于 Map 中，则返回一个默认值，而不是返回 null。这样可以简化代码，避免额外的 null 判断。
      <br/>
      语法：
     </p>
     <pre><code>getOrDefault(Object key, V defaultValue)
</code></pre>
     <ul>
      <li>
       <strong>
        key
       </strong>
       ：要查找的键。
      </li>
      <li>
       <strong>
        defaultValue
       </strong>
       ：当 key 不存在时返回的默认值。
       <br/>
       例如：
      </li>
     </ul>
     <pre><code>Map&lt;String, Integer&gt; map = new HashMap&lt;&gt;();
map.put("apple", 2);

// 获取键 "apple" 对应的值，如果不存在则返回默认值 0
int countApple = map.getOrDefault("apple", 0);  // 返回 2，因为 "apple" 存在且其值为 2

// 获取键 "banana" 对应的值，如果不存在则返回默认值 0
int countBanana = map.getOrDefault("banana", 0);  // 返回 0，因为 "banana" 不存在
</code></pre>
    </blockquote>
    <blockquote>
     <p>
      [NOTE] 问题2：
      <br/>
      charAt()的用法？
      <br/>
      <code>
       charAt()
      </code>
      是 Java 中
      <code>
       String
      </code>
      类提供的一个方法，用于返回字符串中指定索引位置处的字符。索引从 0 开始。
      <br/>
      例如：
     </p>
     <pre><code>String str = "Hello, World!";
char ch = str.charAt(7); // 获取索引为 7 的字符，即 'W'(前面有一个空格)
System.out.println(ch);  // 输出: W
</code></pre>
    </blockquote>
