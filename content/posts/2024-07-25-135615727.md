# HarmonyOS

**Error:**

App Launch:

To run and debug the Harmony device, configure the HarmonyOS runtime.

解决：build-profile.json5

```
{
  "apiType": 'stageMode',
  "buildOption": {
  },
  "targets": [
    {
      "name": "default",
      "runtimeOS": "HarmonyOS"
    }
  ]

```
