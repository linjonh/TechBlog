    <p>
     在 Java 中，
     <code>
      char
     </code>
     的取值范围是
     <strong>
      0 到 65,535
     </strong>
     （即
     <code>
      \u0000
     </code>
     到
     <code>
      \uffff
     </code>
     ），这两个描述是等价的，只是表示方式不同。下面deepseek详细解释：
    </p>
    <hr/>
    <h4>
     <a id="1_char__5">
     </a>
     <strong>
      1.
      <code>
       char
      </code>
      的基本特性
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       位数
      </strong>
      ：16 位（2 字节）。
     </li>
     <li>
      <strong>
       取值范围
      </strong>
      ：0 到 65,535。
     </li>
     <li>
      <strong>
       用途
      </strong>
      ：表示一个 Unicode 字符。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2__0__65535_12">
     </a>
     <strong>
      2. 为什么是 0 到 65,535？
     </strong>
    </h4>
    <ul>
     <li>
      <code>
       char
      </code>
      是无符号的 16 位整数类型。
     </li>
     <li>
      16 位二进制可以表示 (2^{16} = 65,536) 个不同的值（从 0 到 65,535）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3_Unicode__18">
     </a>
     <strong>
      3. Unicode 编码
     </strong>
    </h4>
    <ul>
     <li>
      Java 使用
      <strong>
       Unicode
      </strong>
      字符集来表示字符。
     </li>
     <li>
      Unicode 是一个全球统一的字符编码标准，支持多种语言的字符。
     </li>
     <li>
      <code>
       char
      </code>
      的取值范围
      <code>
       0
      </code>
      到
      <code>
       65,535
      </code>
      对应 Unicode 的
      <strong>
       基本多语言平面（BMP，Basic Multilingual Plane）
      </strong>
      ，覆盖了大多数常用字符（如拉丁字母、汉字、日文假名等）。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4_u0000__uffff__25">
     </a>
     <strong>
      4.
      <code>
       \u0000
      </code>
      到
      <code>
       \uffff
      </code>
      的含义
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        \u
       </code>
      </strong>
      是 Java 中表示 Unicode 字符的转义符号。
     </li>
     <li>
      <strong>
       <code>
        \u0000
       </code>
      </strong>
      ：
      <ul>
       <li>
        表示 Unicode 值为 0 的字符（空字符）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       <code>
        \uffff
       </code>
      </strong>
      ：
      <ul>
       <li>
        表示 Unicode 值为 65,535 的字符。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     因此，
     <code>
      \u0000
     </code>
     到
     <code>
      \uffff
     </code>
     就是
     <strong>
      0 到 65,535
     </strong>
     的另一种表示方式。
    </p>
    <hr/>

    <hr/>
    <h4>
     <a id="6__55">
     </a>
     <strong>
      6. 总结
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       <code>
        char
       </code>
       的取值范围是 0 到 65,535
      </strong>
      。
     </li>
     <li>
      <strong>
       <code>
        \u0000
       </code>
       到
       <code>
        \uffff
       </code>
      </strong>
      是 Unicode 值的表示方式，对应 0 到 65,535。
     </li>
     <li>
      这两种描述是等价的，只是表示方式不同：
      <ul>
       <li>
        数值范围：0 到 65,535。
       </li>
       <li>
        Unicode 表示：
        <code>
         \u0000
        </code>
        到
        <code>
         \uffff
        </code>
        。
       </li>
      </ul>
     </li>
    </ul>
