     如何选择服务器

    <p>
     现在的服务器市场竞争很激烈，服务器过于五花八门，担心不够稳定，担心被骗等因素，导致用户根本不知道从何下手。
     <br/>
     目前市面上常见的服务器大致分两种，云服务器和物理机服务器。
     <br/>
     一、云服务器：比如阿里云，腾讯云等，优点是成本低价格低，相对的配置带宽也低，但是云服务器基本上是没有防御的，可以说是一碰就完，云服务器加防御的成本太高，对于那种对配置带宽防御要求高的业务就不是那么合适了。
     <br/>
     二、物理机服务器：享有整台物理设施强大的性能、电力和独立网，可以定制系统环境，安装任意需要的应用程序，CPU硬件配置高，独享带宽，高防御，主机稳定，也一直是各大企业选择物理部署的重要原因之一。
     <br/>
     1.服务器配置
     <br/>
     对服务器来说配置越高价格也就越高。配置不同，其性能相差也很大，有些配置就不适合个别业务，各位用户要合理选择适合自己业务的配置。
     <br/>
     2.服务器带宽
     <br/>
     对于游戏业务来说，带宽非常重要，带宽过小可能会影响到游戏，导致操作卡顿。
     <br/>
     3.防御性能
     <br/>
     容易受到攻击的一般都是比较热门的业务，比如网络游戏，选择一家能够提供高防服务器的服务商尤为重要。
     <br/>
     4.售后服务
     <br/>
     使用服务器过程中难免会遇到一些问题，这时有7*24小时能够提供技术支持的售后就会让人安心很多，不用担心突发问题找不到人解决。
     <br/>
     以上都是选择一家好的服务商重要的点。
