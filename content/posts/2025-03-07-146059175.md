    <h1 class="title-article" id="articleContentId">
     小程序和页面生命周期详解
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="./../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

   <div class="htmledit_views" id="content_views">
    <p>
     UniApp 是一个基于 Vue.js 的跨平台开发框架，允许开发者使用一套代码构建多端应用（如小程序、H5、App 等）。UniApp 的生命周期结合了 Vue.js 的生命周期和各个平台（如微信小程序、App 等）的生命周期。以下是 UniApp 的主要生命周期钩子：
    </p>
    <p>
     <img alt="" height="1014" src="https://i-blog.csdnimg.cn/direct/d4edf3c8c1764ba09d4e58a31faf987a.png" width="662"/>
    </p>
    <h2>
     <strong>
      应用生命周期
     </strong>
    </h2>
    <p>
     应用生命周期是指整个应用从启动到销毁的过程，适用于 App 平台。
    </p>
    <p>
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <thead>
      <tr>
       <th>
        <em>
         生命周期
        </em>
       </th>
       <th>
        <em>
         说明
        </em>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        onLaunch
       </td>
       <td>
        当应用初始化时触发，全局只触发一次。
       </td>
      </tr>
      <tr>
       <td>
        onShow
       </td>
       <td>
        当应用启动，或从后台进入前台时触发。
       </td>
      </tr>
      <tr>
       <td>
        onHide
       </td>
       <td>
        当应用从前台进入后台时触发。
       </td>
      </tr>
      <tr>
       <td>
        onError
       </td>
       <td>
        当应用发生脚本错误或 API 调用失败时触发。
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     <strong>
      注意
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       应用生命周期仅可在
       <code>
        App.vue
       </code>
       中监听，在其它页面监听无效
      </strong>
      。
     </li>
    </ul>
    <h2 id="5-%E9%A1%B5%E9%9D%A2%E7%BA%A7%E5%88%AB%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">
     页面生命周期：
    </h2>
    <p>
     页面生命周期是指单个页面从创建到销毁的过程
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <thead>
      <tr>
       <th>
        <em>
         生命周期
        </em>
       </th>
       <th>
        <em>
         说明
        </em>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        onLoad
       </td>
       <td>
        页面加载时触发 (一个页面只会调用一次)
       </td>
      </tr>
      <tr>
       <td>
        onShow
       </td>
       <td>
        每次打开页面都会调用一次
       </td>
      </tr>
      <tr>
       <td>
        onReady
       </td>
       <td>
        页面初次渲染完成时触发(一个页面只会调用一次)
        <br/>
        代表页面已经准备妥当，可以和视图层进行交互
       </td>
      </tr>
      <tr>
       <td>
        onHide
       </td>
       <td>
        页面隐藏/切入后台时触发（当
        <code>
         navigateTo
        </code>
        或底部tab切换时调用。）
       </td>
      </tr>
      <tr>
       <td>
        onUpload
       </td>
       <td>
        页面卸载时触发（当
        <code>
         redirectTo
        </code>
        或
        <code>
         navigateBack
        </code>
        的时候调用。）
       </td>
      </tr>
     </tbody>
    </table>
    <p>
    </p>
    <h2>
     <strong>
      组件生命周期：
     </strong>
    </h2>
    <p>
     UniApp 的组件生命周期与 Vue.js 的生命周期一致。
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <thead>
      <tr>
       <th style="width: 133px;">
        <em>
         生命周期
        </em>
       </th>
       <th style="width: 365px;">
        <em>
         说明
        </em>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          beforeCreate
         </code>
        </strong>
       </td>
       <td style="width:365px">
        在实例初始化之后，数据观测 (data observer) 和 event/watcher 事件配置之前被调用。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          created
         </code>
        </strong>
       </td>
       <td style="width:365px">
        在实例创建完成后被立即调用，此时已完成数据观测，但尚未挂载 DOM
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          beforeMount
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件挂载到 DOM 之前。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          mounted
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件挂载到 DOM 之后。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          beforeUpdate
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件数据更新之前。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          updated
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件数据更新之后。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          beforeDestroy
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件实例销毁之前。
       </td>
      </tr>
      <tr>
       <td style="width:133px">
        <strong>
         <code>
          destroyed
         </code>
        </strong>
       </td>
       <td style="width:365px">
        组件实例销毁之后。
       </td>
      </tr>
     </tbody>
    </table>
    <h2 id="%E5%9B%9B%E3%80%81%E5%85%B6%E5%AE%83%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">
     其它生命周期
    </h2>
    <p>
    </p>
    <table border="1" cellpadding="1" cellspacing="1" style="width:500px">
     <thead>
      <tr>
       <th style="width: 165px;">
        <p id="%E5%9B%9B%E3%80%81%E5%85%B6%E5%AE%83%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">
         生命周期
        </p>
       </th>
       <th style="width: 333px;">
        说明
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td style="width:165px">
        onPullDownRefresh
       </td>
       <td style="width:333px">
        页面下拉刷新时触发。
       </td>
      </tr>
      <tr>
       <td style="width:165px">
        onReachBottom
       </td>
       <td style="width:333px">
        页面上拉触底时触发
       </td>
      </tr>
      <tr>
       <td style="width:165px">
        onShareAppMessage
       </td>
       <td style="width:333px">
        用户点击右上角分享时触发
       </td>
      </tr>
      <tr>
       <td style="width:165px">
        onPageScroll
       </td>
       <td style="width:333px">
        页面滚动时触发
       </td>
      </tr>
      <tr>
       <td style="width:165px">
        onResize
       </td>
       <td style="width:333px">
        页面尺寸改变时触发
       </td>
      </tr>
     </tbody>
    </table>
    <h2>
     <strong>
      页面与组件生命周期执行顺序
     </strong>
    </h2>
    <ul>
     <li>
      <p>
       页面加载时：
       <code>
        onLoad
       </code>
       -&gt;
       <code>
        onShow
       </code>
       -&gt;
       <code>
        onReady
       </code>
      </p>
     </li>
     <li>
      <p>
       页面隐藏时：
       <code>
        onHide
       </code>
      </p>
     </li>
     <li>
      <p>
       页面卸载时：
       <code>
        onUnload
       </code>
      </p>
     </li>
     <li>
      <p>
       组件加载时：
       <code>
        beforeCreate
       </code>
       -&gt;
       <code>
        created
       </code>
       -&gt;
       <code>
        beforeMount
       </code>
       -&gt;
       <code>
        mounted
       </code>
      </p>
     </li>
     <li>
      <p>
       组件更新时：
       <code>
        beforeUpdate
       </code>
       -&gt;
       <code>
        updated
       </code>
      </p>
     </li>
     <li>
      <p>
       组件销毁时：
       <code>
        beforeDestroy
       </code>
       -&gt;
       <code>
        destroyed
       </code>
      </p>
     </li>
    </ul>
    <h2>
     <strong>
      注意：
     </strong>
    </h2>
    <ul>
     <li>
      <p>
       <strong>
        应用生命周期
       </strong>
       仅在 App 平台有效，小程序和 H5 平台不支持。
      </p>
     </li>
     <li>
      <p>
       <strong>
        页面生命周期
       </strong>
       和
       <strong>
        组件生命周期
       </strong>
       在所有平台都支持，但不同平台可能会有一些差异。
      </p>
     </li>
     <li>
      <p>
       在小程序平台，部分生命周期（如
       <code>
        onPullDownRefresh
       </code>
       、
       <code>
        onReachBottom
       </code>
       等）是小程序特有的。
      </p>
     </li>
    </ul>
