    <h4>
     <a id="1_Pandas_Profiling_1">
     </a>
     <strong>
      1. Pandas Profiling
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        自动生成详细的统计报告，包含数据概览、单变量分析、相关性矩阵、缺失值分析等。
       </li>
       <li>
        支持交互式 HTML 报告，可导出为 PDF 或其他格式。
       </li>
       <li>
        适合快速生成数据集的全面摘要。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">import</span> pandas <span class="token keyword">as</span> pd
<span class="token keyword">from</span> pandas_profiling <span class="token keyword">import</span> ProfileReport

df <span class="token operator">=</span> pd<span class="token punctuation">.</span>read_csv<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
profile <span class="token operator">=</span> ProfileReport<span class="token punctuation">(</span>df<span class="token punctuation">,</span> explorative<span class="token operator">=</span><span class="token boolean">True</span><span class="token punctuation">)</span>
profile<span class="token punctuation">.</span>to_file<span class="token punctuation">(</span><span class="token string">"output.html"</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：
      <a href="https://github.com/ydataai/pandas-profiling">
       Pandas Profiling
      </a>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2_Sweetviz_19">
     </a>
     <strong>
      2. Sweetviz
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        生成交互式 HTML 报告，支持数据集比较（如训练集 vs 测试集）和目标变量分析。
       </li>
       <li>
        可视化直观，包含直方图、分布对比、相关性分析等。
       </li>
       <li>
        支持自动化分析（
        <code>
         sv.analyze()
        </code>
        ）和数据集对比（
        <code>
         sv.compare()
        </code>
        ）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">import</span> pandas <span class="token keyword">as</span> pd
<span class="token keyword">import</span> sweetviz <span class="token keyword">as</span> sv

df <span class="token operator">=</span> pd<span class="token punctuation">.</span>read_csv<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
report <span class="token operator">=</span> sv<span class="token punctuation">.</span>analyze<span class="token punctuation">(</span>df<span class="token punctuation">)</span>
report<span class="token punctuation">.</span>show_html<span class="token punctuation">(</span><span class="token string">"sweetviz_report.html"</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：
      <a href="https://github.com/fbdesign/sweetviz">
       Sweetviz
      </a>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="3_DTale_37">
     </a>
     <strong>
      3. DTale
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        提供交互式 Web 界面，支持数据筛选、聚合、图表生成（如散点图、直方图）。
       </li>
       <li>
        内置数据探索工具，如统计摘要、相关性热力图、缺失值矩阵。
       </li>
       <li>
        适合在 Jupyter Notebook 或终端中快速启动交互式分析。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">import</span> dtale
<span class="token keyword">import</span> pandas <span class="token keyword">as</span> pd

df <span class="token operator">=</span> pd<span class="token punctuation">.</span>read_csv<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
d <span class="token operator">=</span> dtale<span class="token punctuation">.</span>show<span class="token punctuation">(</span>df<span class="token punctuation">)</span>
d<span class="token punctuation">.</span>open_browser<span class="token punctuation">(</span><span class="token punctuation">)</span>  <span class="token comment"># 自动打开浏览器展示界面</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：
      <a href="https://dtale.palletsprojects.com/" rel="nofollow">
       DTale
      </a>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4_AutoViz_55">
     </a>
     <strong>
      4. AutoViz
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        通过一行代码自动分析数据集并生成可视化图表（如散点图、箱线图、时间序列图）。
       </li>
       <li>
        支持自动识别数据类型（分类、数值、日期等）并选择合适的图表。
       </li>
       <li>
        适合快速发现数据中的模式和异常。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">from</span> autoviz<span class="token punctuation">.</span>AutoViz_Class <span class="token keyword">import</span> AutoViz_Class

AV <span class="token operator">=</span> AutoViz_Class<span class="token punctuation">(</span><span class="token punctuation">)</span>
df <span class="token operator">=</span> AV<span class="token punctuation">.</span>AutoViz<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：
      <a href="https://github.com/AutoViML/AutoViz">
       AutoViz
      </a>
     </li>
    </ul>

    <h4>
     <a id="5_DataPrep_71">
     </a>
     <strong>
      5. DataPrep
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        结合自动化 EDA、数据清洗和特征工程，支持多种数据源（CSV、Excel、数据库）。
       </li>
       <li>
        提供交互式报告（如
        <code>
         profile()
        </code>
        函数生成的 HTML 报告）。
       </li>
       <li>
        兼容 Pandas，适合处理大型数据集。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">import</span> dataprep <span class="token keyword">as</span> dp

df <span class="token operator">=</span> dp<span class="token punctuation">.</span>read_csv<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
report <span class="token operator">=</span> dp<span class="token punctuation">.</span>profile<span class="token punctuation">(</span>df<span class="token punctuation">)</span>
report<span class="token punctuation">.</span>save<span class="token punctuation">(</span><span class="token string">"report.html"</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：
      <a href="https://dataprep.ai/" rel="nofollow">
       DataPrep
      </a>
     </li>
    </ul>

    <h4>
     <a id="6_Dabl_88">
     </a>
     <strong>
      6. Dabl
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       功能特点
      </strong>
      ：
      <ul>
       <li>
        专注于双变量分析，自动选择合适的可视化（如散点图、直方图）。
       </li>
       <li>
        需要指定目标变量，适合监督学习任务（如分类或回归）。
       </li>
       <li>
        自动清理数据（如处理缺失值、编码分类变量）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       使用示例
      </strong>
      ：
      <pre><code class="prism language-python"><span class="token keyword">import</span> dabl
<span class="token keyword">import</span> pandas <span class="token keyword">as</span> pd

df <span class="token operator">=</span> pd<span class="token punctuation">.</span>read_csv<span class="token punctuation">(</span><span class="token string">"data.csv"</span><span class="token punctuation">)</span>
dabl<span class="token punctuation">.</span>plot<span class="token punctuation">(</span>df<span class="token punctuation">,</span> target_col<span class="token operator">=</span><span class="token string">"target"</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <strong>
       官网
      </strong>
      ：

     <strong>
      补充说明
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        无代码/低代码需求
       </strong>
       ：
      </p>
      <ul>
       <li>
        如果希望接近
        <strong>
         Exploratory
        </strong>
        的无代码体验，
        <strong>
         DTale
        </strong>
        和
        <strong>
         DataPrep
        </strong>
        的交互式界面可能更合适，因为它们提供 Web 界面供用户直接操作。
       </li>
       <li>
        如果需要完全无代码，可考虑直接使用
        <strong>
         Exploratory
        </strong>
        本身，但 Python 库需要编写少量代码。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        自动化与灵活性
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         Sweetviz
        </strong>
        和
        <strong>
         Pandas Profiling
        </strong>
        是最常用的自动化 EDA 工具，适合快速生成报告。
       </li>
       <li>
        <strong>
         AutoViz
        </strong>
        更适合希望用极简代码快速发现数据模式的用户。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        数据清洗与高级分析
       </strong>
       ：
      </p>
      <ul>
       <li>
        <strong>
         DataPrep
        </strong>
        和
        <strong>
         Dabl
        </strong>
        提供了更多数据清洗和机器学习集成功能，适合复杂任务。
       </li>
      </ul>
     </li>
    </ol>
