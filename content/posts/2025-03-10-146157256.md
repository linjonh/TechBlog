     Pcdn的业务可以根据不同的分类标准来划分
     <br/>
     一、按线路类型划分
     <br/>
     汇聚模式：一个地方有多条线路，业务种类较多。通常使用X86或X99主板组装的服务器，或各品牌的准系统服务器。收益通常比单线模式更高。
     <br/>
     单线模式：一个地方只有一条线路，业务种类相对单一。常见的设备有工控机、N1盒子等。
     <br/>
     二、按业务场景划分
     <br/>
     短视频业务：如抖音、快手等短视频平台。对硬件需求较低，对网络带宽要求也相对较低，但跑量较大，收益稳定。
     <br/>
     长视频业务：如爱奇艺、腾讯视频等长视频平台。对视频质量和带宽要求较高，通常需要大带宽和充足的存储空间。
     <br/>
     直播业务：如斗鱼、虎牙等直播平台。实时性要求较高，需要低延迟和高并发的支持。
     <br/>
     三、按服务类型划分
     <br/>
     点播服务：用户可以按需选择观看内容，如电影、电视剧等。边缘计算可以提供快速、稳定的内容分发服务，确保用户流畅观看。
     <br/>
     直播服务：实时传输内容，如体育赛事、演唱会等。边缘计算可以提供低延迟、高并发的支持，确保用户能够实时观看直播内容。
