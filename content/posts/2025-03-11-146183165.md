    <h4>
     <a id="3__117">
     </a>
     3. 高级特性
    </h4>
    <ul>
     <li>
      <strong>
       容量管理
      </strong>
      ：
      <code>
       string
      </code>
      类会自动管理内存，但也提供了一些方法来控制容量，如
      <code>
       reserve
      </code>
      可以预先分配一定的内存空间，减少内存重新分配的次数。
     </li>
     <li>
      <strong>
       迭代器
      </strong>
      ：支持使用迭代器遍历字符串，如
      <code>
       begin()
      </code>
      和
      <code>
       end()
      </code>
      分别返回指向字符串首字符和尾后字符的迭代器。
     </li>
    </ul>

    <h4>
     <a id="3__402">
     </a>
     3. 注意事项
    </h4>
    <ul>
     <li>
      在进行文件操作时，要确保文件能够正常打开，否则会导致程序出错。可以使用
      <code>
       is_open()
      </code>
      方法检查文件是否打开成功。
     </li>
     <li>
      使用完文件后，要及时关闭文件，以释放系统资源。可以使用
      <code>
       close()
      </code>
      方法关闭文件。
     </li>
     <li>
      在文件读写过程中，要注意文件指针的位置，可以使用
      <code>
       seekg()
      </code>
      和
      <code>
       seekp()
      </code>
      方法移动文件指针。
     </li>
    </ul>
