    <ol>
     <li>
      <strong>
       问题1：沉积岩薄片识别模型设计
      </strong>
     </li>
     <li>
      <strong>
       问题分析
      </strong>
     </li>
     <li>
      <strong>
       核心任务
      </strong>
      ：基于“南京大学沉积岩教学薄片照片数据集”，构建多类别分类模型，区分火山碎屑岩、砂岩、泥页岩等9类沉积岩。
     </li>
     <li>
      <strong>
       特征提取需求
      </strong>
      ：
      <ul>
       <li>
        <strong>
         颜色特征
        </strong>
        ：矿物成分差异导致偏光下颜色分布不同（如灰岩的方解石高级白与砂岩石英的灰色）。
       </li>
       <li>
        <strong>
         纹理特征
        </strong>
        ：颗粒分选性（砂岩的均匀颗粒 vs 泥岩的无序黏土）与层理构造（页岩的薄层理）。
       </li>
       <li>
        <strong>
         亮度特征
        </strong>
        ：矿物透明度（如硅质岩高亮度 vs 泥页岩低亮度）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       数学建模关键点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         特征工程
        </strong>
        ：提取颜色直方图（HSV空间）、灰度共生矩阵（GLCM纹理）、局部二值模式（LBP）等传统特征，结合CNN自动提取的深层特征。
       </li>
       <li>
        <strong>
         多特征融合策略
        </strong>
        ：设计加权拼接或双流网络（如传统特征与ResNet输出融合），通过全连接层优化特征组合权重。
       </li>
       <li>
        <strong>
         类别不平衡处理
        </strong>
        ：对少数类（如蒸发岩）采用Focal Loss或过采样技术（SMOTE）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       模型选择
      </strong>
      ：以ResNet-50为主干网络，引入注意力机制（如SE模块）聚焦关键区域，输出层采用Softmax分类。
     </li>
     <li>
      <strong>
       挑战与解决
      </strong>
     </li>
    </ol>
    <ul>
     <li>
      <strong>
       特征冗余
      </strong>
      ：使用PCA或L1正则化筛选有效特征。
     </li>
     <li>
      <strong>
       显微图像噪声
      </strong>
      ：预处理加入非局部均值去噪与对比度受限直方图均衡化（CLAHE）。
     </li>
    </ul>
    <ol>
     <li>
      <strong>
       问题2：跨岩类模型泛化与改进
      </strong>
     </li>
     <li>
      <strong>
       问题分析
      </strong>
     </li>
    </ol>
    <ol>
     <li>
      <strong>
       核心任务
      </strong>
      ：验证问题1模型在变质岩、火成岩中的泛化性，并改进模型以提升三岩类分类精度。
     </li>
     <li>
      <strong>
       跨岩类差异
      </strong>
      ：
      <ul>
       <li>
        <strong>
         火成岩
        </strong>
        ：显晶质结构（如花岗岩）或玻璃质（如玄武岩）。
       </li>
       <li>
        <strong>
         变质岩
        </strong>
        ：片理构造（如片岩）或粒状变晶（如大理岩）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       改进策略
      </strong>
      ：
      <ul>
       <li>
        <strong>
         多尺度特征融合
        </strong>
        ：通过特征金字塔网络（FPN）捕捉不同岩类的宏观构造（如片理）与微观矿物特征。
       </li>
       <li>
        <strong>
         域适应技术
        </strong>
        ：使用对抗训练（Domain-Adversarial Neural Networks）减少沉积岩预训练模型向其他岩类的域偏移。
       </li>
       <li>
        <strong>
         混合特征增强
        </strong>
        ：将传统纹理特征（如Gabor滤波响应）与CNN特征在中间层融合。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       评估指标
      </strong>
      ：对比改进前后的混淆矩阵，计算跨岩类的F1-score提升率。
     </li>
    </ol>

    <ol>
     <li>
      <strong>
       问题3：附件1数据测试与结果汇总
      </strong>
     </li>
     <li>
      <strong>
       问题分析
      </strong>
     </li>
    </ol>
    <ol>
     <li>
      <strong>
       任务分解
      </strong>
      ：
      <ul>
       <li>
        <strong>
         步骤1
        </strong>
        ：用问题2改进模型筛选附件1中的沉积岩（二分类：沉积岩 vs 非沉积岩）。
       </li>
       <li>
        <strong>
         步骤2
        </strong>
        ：对筛选出的沉积岩，用问题1模型细分至砂岩、泥页岩等5类。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       数学建模要点
      </strong>
      ：
      <ul>
       <li>
        <strong>
         两阶段Pipeline
        </strong>
        ：级联模型减少计算量，第一阶段用轻量级MobileNetV3快速过滤非沉积岩。
       </li>
       <li>
        <strong>
         不确定性校准
        </strong>
        ：对沉积岩子类预测结果，采用蒙特卡洛Dropout估计置信度，过滤低置信样本（阈值&gt;0.8）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       结果处理
      </strong>
      ：
      <ul>
       <li>
        <strong>
         统计输出
        </strong>
        ：对每类沉积岩计数，生成频数分布直方图。
       </li>
       <li>
        <strong>
         结果验证
        </strong>
        ：随机抽样人工复核，计算误判率（如将白云岩误判为灰岩的比例）。
       </li>
      </ul>
     </li>
    </ol>
    <ol>
     <li>
      沉积岩的岩性描述是其分类和实际应用的核心，主要基于
      <strong>
       矿物组成、结构、构造、颜色及其他物理性质
      </strong>
      。以下是针对
      <strong>
       单独沉积岩
      </strong>
      的岩性特征详细分析：
      <br/>
      沉积岩的矿物成分反映其形成环境和物质来源：
      <br/>
      沉积岩的结构指颗粒的
      <strong>
       大小、形状、排列方式
      </strong>
      ，直接反映沉积环境能量：
      <br/>
      沉积岩的宏观构造是判断沉积环境的重要标志：
      <strong>
       断口特征
      </strong>
      ：页岩的平坦断口 vs 砂岩的颗粒状断口。
      <ul>
       <li>
        高孔隙砂岩（储油气） vs 致密页岩（隔水层）。
       </li>
       <li>
        <strong>
         硬度
        </strong>
        ：石英砂岩（高硬度，小刀划不动） vs 泥岩（低硬度，易碎）。
       </li>
       <li>
        <strong>
         密度
        </strong>
        ：煤（低密度） vs 燧石（高密度）。
       </li>
       <li>
        <strong>
         孔隙度与渗透性
        </strong>
        ：
       </li>
       <li>
        红色/褐色（氧化环境，含赤铁矿，如红层砂岩）。
       </li>
       <li>
        黑色/灰色（还原环境，含有机质或硫化铁，如黑色页岩）。
       </li>
       <li>
        <strong>
         原生色
        </strong>
        ：
       </li>
       <li>
        <strong>
         次生色
        </strong>
        ：风化或矿物浸染导致（如褐铁矿使岩石呈黄色）。
       </li>
      </ul>
     </li>
    </ol>

        </strong>

    <ol>
     <li>
      <strong>
       石英砂岩
      </strong>
      <ul>
       <li>
        矿物：石英＞90%，少量胶结物（硅质或铁质）。
       </li>
       <li>
        结构：中-粗砂，分选好，磨圆度高。
       </li>
       <li>
        构造：大型交错层理。
       </li>
       <li>
        颜色：白色或浅黄色（硅质胶结）或红色（铁质胶结）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       黑色页岩
      </strong>
      <ul>
       <li>
        矿物：黏土矿物为主，含有机质和黄铁矿。
       </li>
       <li>
        结构：极细颗粒（泥级），薄片状剥裂。
       </li>
       <li>
        构造：水平层理，可能含笔石化石。
       </li>
       <li>
        颜色：深灰至黑色，反映缺氧环境。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       鲕粒灰岩
      </strong>
      <ul>
       <li>
        矿物：方解石为主，鲕粒核心为石英或生物碎屑。
       </li>
       <li>
        结构：鲕粒直径0.2–2 mm，同心环带结构。
       </li>
       <li>
        构造：块状或层状，常见于浅海高能环境。
       </li>
       <li>
        颜色：灰白色或浅灰色。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       蒸发岩（岩盐）
      </strong>
      <ul>
       <li>
        矿物：石盐（NaCl），可能含石膏夹层。
       </li>
       <li>
        结构：等粒晶体，立方体解理。
       </li>
       <li>
        构造：厚层状，无生物遗迹。
       </li>
       <li>
        物理性质：易溶于水，味咸，硬度低（指甲可刻划）。
       </li>
