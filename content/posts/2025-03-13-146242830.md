    <h2>
     <a id="_1">
     </a>
     前言
    </h2>
    <p>
     在使用 Docker 部署应用时，通常需要从 Docker Hub 或其他镜像仓库拉取镜像。但有时候，我们可能需要在没有互联网的环境中部署镜像，或者希望直接分享某个镜像给同事、朋友，而不必让他们重新下载。
    </p>
    <p>
     因为很多人使用docker下载文件非常慢，因此我在这里分享一下我docker当前下载的anythingllm和open-webui压缩包。
    </p>
    <p>
     本文主要两件事：
    </p>
    <ol>
     <li>
      首先分享镜像获取方式，提供两种方式：一个是切换国内镜像，一个是使用我分享的百度云压缩包。
     </li>
     <li>
      如果使用压缩包，如何在本地docker中使用。
     </li>
    </ol>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/ed0156903aaa49759d00e1b08fabef8a.png"/>
    </p>
    <h3>
     <a id="_15">
     </a>
     第一部分：镜像获取
    </h3>
    <h4>
     <a id="__17">
     </a>
     🚀 方式一：切换国内下载镜像
    </h4>
    <p>
     备注：下面的国内镜像，应该下载的会快一些，我的教育网试了一下，用着没啥问题。
    </p>
    <h5>
     <a id="1_anythingllm_21">
     </a>
     ✅1. 下载anythingllm
    </h5>
    <p>
     访问地址：https://docker.aityp.com/i/search?search=anythingllm
    </p>
    <p>
     访问上述链接，可以直达anythingllm的下载镜像地址。如果使用docker拉取镜像，可以用镜像，使用如下命令：
    </p>
    <pre><code>docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/mintplexlabs/anythingllm:latest
</code></pre>
    <h5>
     <a id="_2_openwebui_33">
     </a>
     ✅ 2. 下载open-webui
    </h5>
    <p>
     open-webui:cuda:https://docker.aityp.com/image/ghcr.io/open-webui/open-webui%3Acuda
    </p>
    <p>
     访问上述链接，可以直达open-webui:cuda的下载镜像地址。如果使用docker拉取镜像，可以用镜像，使用如下命令
    </p>
    <pre><code class="prism language-cmd">docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/ghcr.io/open-webui/open-webui:cuda
</code></pre>
    <h5>
     <a id="_43">
     </a>

    <h4>
     <a id="_45">
     </a>

    </h4>
    <p>
     我只能分享anythingllm压缩包，因为open-webui超过8G，我百度云没法上传，都没法分享。。。。
    </p>

     <a id="_anythingllm_49">

    <h5>
     <a id="_openwebui_54">
     </a>
     ❎ open-webui压缩包
    </h5>
    <p>
     这个文件8G多，百度云还不能上传，没法分享了。
     <br/>
     感觉也不用谷歌云盘分享了，因为docker下载不了，大概率科学上网环境也没有。
    </p>
    <hr/>
    <p>
     下面我将介绍如何下载
     <code>
      mintplexlabs/anythingllm
     </code>
     Docker 镜像，并将其保存、分享和导入到另一台设备上。
    </p>
    <h3>
     <a id="_Docker__67">
     </a>
     第二部分：下载之后的压缩包导入 Docker 镜像
    </h3>
    <p>
     顺便说一下Docker如何把某个镜像保存为tar压缩文件。
    </p>
    <h4>
     <a id="1__Docker__tar__73">
     </a>
     1. 将 Docker 镜像保存为
     <code>
      .tar
     </code>
     文件
    </h4>
    <p>
     为了便于共享，我们可以将 Docker 镜像导出为
     <code>
      .tar
     </code>
     文件。
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> save <span class="token parameter variable">-o</span> anythingllm.tar mintplexlabs/anythingllm
</code></pre>
    <p>
     上述命令会在当前目录下生成一个
     <code>
      anythingllm.tar
     </code>
     文件，该文件包含
     <code>
      mintplexlabs/anythingllm
     </code>
     的完整 Docker 镜像。
    </p>
    <h4>
     <a id="_2__Docker__85">
     </a>
     🚀 2. 其他人如何导入 Docker 镜像
    </h4>
    <p>
     收到
     <code>
      anythingllm.tar
     </code>
     文件后，其他人可以使用以下命令导入 Docker 镜像：
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> load <span class="token parameter variable">-i</span> anythingllm.tar
</code></pre>
    <p>
     成功导入后，可以使用以下命令检查镜像是否已正确加载：
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> images
</code></pre>
    <p>
     如果看到
     <code>
      mintplexlabs/anythingllm
     </code>
     ，说明导入成功。
    </p>
    <hr/>
    <h4>
     <a id="3__anythingllm__104">
     </a>
     3. 如何运行
     <code>
      anythingllm
     </code>
     容器
    </h4>
    <p>
     导入成功后，其他人可以使用
     <code>
      docker run
     </code>
     命令运行该镜像，最后的镜像名字mintplexlabs/anythingllm记得改为docker images显示的anythingllm名字。
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> run <span class="token parameter variable">-d</span> <span class="token parameter variable">--name</span> anythingllm --add-host<span class="token operator">=</span>host.docker.internal:host-gateway <span class="token parameter variable">--env</span> <span class="token assign-left variable">STORAGE_DIR</span><span class="token operator">=</span>/app/server/storage --health-cmd <span class="token string">"/bin/bash /usr/local/bin/docker-healthcheck.sh || exit 1"</span> --health-interval 60s --health-start-period 60s --health-timeout 10s <span class="token parameter variable">-p</span> <span class="token number">3001</span>:3001/tcp <span class="token parameter variable">--restart</span><span class="token operator">=</span>always <span class="token parameter variable">--user</span> anythingllm <span class="token parameter variable">-v</span> %STORAGE_LOCATION%:/app/server/storage <span class="token parameter variable">-v</span> %STORAGE_LOCATION%<span class="token punctuation">\</span>.env:/app/server/.env <span class="token parameter variable">-w</span> /app mintplexlabs/anythingllm
</code></pre>
    <p>
     具体详细的内容可以直接查看博客：
    </p>
    <p>
     <a href="https://blog.csdn.net/qq_36693723/article/details/145588499?spm=1001.2014.3001.5501">
      Windows中使用Docker安装Anythingllm，基于deepseek构建自己的本地知识库问答大模型，可局域网内多用户访问、离线运行
     </a>
    </p>
    <p>
     <a href="https://blog.csdn.net/qq_36693723/article/details/145597221?spm=1001.2014.3001.5501">
      使用 Docker 安装 Open WebUI 并集成 Ollama 的 DeepSeek 模型
     </a>
    </p>
    <p>
     更多的详细信息可以查看官网链接：
     <a href="https://github.com/Mintplex-Labs/anything-llm/blob/master/docker/HOW_TO_USE_DOCKER.md">
      https://github.com/Mintplex-Labs/anything-llm/blob/master/docker/HOW_TO_USE_DOCKER.md

    <hr/>
    <h4>
     <a id="4__124">
     </a>
     4. 其他常见操作
    </h4>
    <h5>
     <a id="41__126">
     </a>
     4.1. 查看正在运行的容器
    </h5>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token function">ps</span>
</code></pre>
    <p>
     如果要查看所有容器（包括已停止的），可以运行：
    </p>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token function">ps</span> <span class="token parameter variable">-a</span>
</code></pre>
    <h5>
     <a id="42__138">
     </a>
     4.2. 停止容器
    </h5>
    <pre><code class="prism language-bash"><span class="token function">docker</span> stop <span class="token operator">&lt;</span>容器ID或名称<span class="token operator">&gt;</span>
</code></pre>
    <h5>
     <a id="43__144">
     </a>
     4.3. 删除容器
    </h5>
    <pre><code class="prism language-bash"><span class="token function">docker</span> <span class="token function">rm</span> <span class="token operator">&lt;</span>容器ID或名称<span class="token operator">&gt;</span>
</code></pre>
    <h5>
     <a id="44__150">
     </a>
     4.4. 删除镜像
    </h5>
    <pre><code class="prism language-bash"><span class="token function">docker</span> rmi mintplexlabs/anythingllm
</code></pre>

     结语

    <p>
     通过上述步骤，我们成功下载、保存并分享了
     <code>
      mintplexlabs/anythingllm
     </code>
     Docker 镜像。这样，即使在没有网络的情况下，其他人也可以轻松获得并运行该镜像，而无需重新拉取。
    </p>
