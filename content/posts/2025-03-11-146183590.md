    <p>
     要查看 Redis 缓存的时间，有下列两种方式：
    </p>
    <h2>
     <a id="_TTL__1">
     </a>
     一、使用 TTL 命令来获取缓存剩余的时间
    </h2>
    <p>
     Redis提供了多个命令来查看缓存数据的时间戳，其中最常用的命令是
     <code>
      ttl
     </code>
     和
     <code>
      pttl
     </code>
     。
    </p>
    <ul>
     <li>
      <p>
       <code>
        ttl
       </code>
       它返回的是以秒为单位的时间，表示 key 距离过期的时间还有多久。
       <br/>
       如果返回 -1，表示 key 没有设置过期时间；如果返回 -2，表示 key 不存在。
      </p>
     </li>
     <li>
      <p>
       <code>
        pttl
       </code>
       返回值的含义与“ttl”命令相同，只是单位变成了毫秒。
      </p>
     </li>
    </ul>
    <p>
     除了“ttl”和“pttl”命令，Redis还提供了其他命令来获取缓存数据的创建时间和最近一次修改时间。
     <br/>
     例如，
     <code>
      object idletime
     </code>
     命令可以获取缓存数据的空闲时间（即距离上一次使用的秒数），
     <br/>
     <code>
      object refcount
     </code>
     命令可以获取缓存数据的引用计数（即被多少个键所引用）。
    </p>
    <p>
     下面是使用 Redis 命令行客户端进行查看缓存时间的示例：
    </p>
    <ol>
     <li>
      打开终端，连接到 Redis 服务器：
     </li>
    </ol>
    <pre><code class="prism language-powershell">$ redis-<span class="token function">cli</span>
</code></pre>
    <ol start="2">
     <li>
      选择一个数据库（默认是 0）：
     </li>
    </ol>
    <pre><code class="prism language-powershell">&gt; <span class="token function">SELECT</span> &lt;db-number&gt;
</code></pre>
    <ol start="3">
     <li>
      使用 TTL 命令查看指定 key 的剩余过期时间：
     </li>
    </ol>
    <pre><code class="prism language-shell"><span class="token operator">&gt;</span> TTL <span class="token operator">&lt;</span>key<span class="token operator">&gt;</span>
</code></pre>
    <p>
     其中 是你想要查看的缓存的键名。
    </p>
    <ol start="4">
     <li>
      根据返回值进行判断：
     </li>
    </ol>
    <ul>
     <li>
      如果返回 -1，表示该 key 没有设置过期时间，即缓存永不过期；
     </li>
     <li>
      如果返回 -2，表示该 key 不存在，即缓存已经被移除；
     </li>
     <li>
      如果返回一个正整数，表示还有多少秒过期。
     </li>
    </ul>
    <h2>
     <a id="_Redis__TTL__38">
     </a>
     二、在编程语言中使用对应的 Redis 客户端库来执行 TTL 命令
    </h2>
    <p>
     例如，在 Python 中使用 redis-py 库：
    </p>
    <pre><code class="prism language-python"><span class="token keyword">import</span> redis

<span class="token comment"># 创建 Redis 客户端</span>
r <span class="token operator">=</span> redis<span class="token punctuation">.</span>Redis<span class="token punctuation">(</span>host<span class="token operator">=</span><span class="token string">'localhost'</span><span class="token punctuation">,</span> port<span class="token operator">=</span><span class="token number">6379</span><span class="token punctuation">,</span> db<span class="token operator">=</span><span class="token number">0</span><span class="token punctuation">)</span>

<span class="token comment"># 获取指定 key 的剩余过期时间</span>
ttl <span class="token operator">=</span> r<span class="token punctuation">.</span>ttl<span class="token punctuation">(</span><span class="token string">'your:key'</span><span class="token punctuation">)</span>

<span class="token keyword">print</span><span class="token punctuation">(</span>ttl<span class="token punctuation">)</span>
</code></pre>
    <p>
     注意，在使用编程语言访问Redis之前，你需要确保已经安装了相关的Redis客户端库。这些库通常可以通过包管理工具（如pip、Maven、NuGet等）来安装。
    </p>
