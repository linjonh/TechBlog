## 1. 整体描述

最近很多国企项目，都在推行国产化，国产化主要包括硬件上，服务器处理器国产化，操作系统国产化，即麒麟操作系统，然后就是数据库的国产化，我之前写过一篇文章，分析了目前主流的三个国产数据库：
[三个国产数据库调研(达梦，PolarDB，TDSQL)](https://blog.csdn.net/nhx900317/article/details/140513541)
，本文在简单总结下其他的国产数据库。
  
PS，这个名单基本上是目前一些国企里公认的可以使用的，满足信创的国产数据库名单。主要以集中式数据库为主。

## 2. 具体描述

### 2.1 达梦，PloarDB，TsSQL

这三个是名单的前三个国产数据库，也是使用比较多的，具体看我之前写的文章，对这三个数据库做了详细的描述。

### 2.2 瀚高安全版数据库

具体版本： 瀚高安全版数据库系统V4.5
  
官网地址：https://www.highgo.com/product_safe.html
  
瀚高安全版数据库系统V4.5是一款致力于为我国安全领域IT信息系统提供高安全，高可用，易使用，符合国家安全标准的企业级关系型数据库，为国家等级保护，商密算法落地，及涉及政企客户保密防护的信息系统提供基础支撑。
  
适配性：对MYSQL的兼容性较差，迁移的话需要消耗的成本较高，而且有一定的学习成本。

### 2.3 虚谷数据库

具体版本：虚谷数据库管理系统V11.0
  
官网地址：https://www.xugudb.com/
  
虚谷数据库是一款自主原创、安全可靠、功能强大、性能优越的关系型数据库管理系统，支持SQL标准语法，支持标准化、结构化数据、事务处理、分布式集群部署，提供国产化、面向多核的高性能、高并发、高可用、并行计算、弹性伸缩和数据安全服务。
  
适配性：对MYSQL部分适配

### 2.4 南大通用安全数据库

具体版本：南大通用安全数据库管理系统GBase 8s V8.8
  
官网地址：https://www.gbase.cn/
  
GBase 8s是天津南大通用数据技术股份有限公司自主研发的、成熟稳定的基于共享存储的数据库集群，拥有自主知识产权。产品达到安全数据库四级标准（国际B2），支持国密算法，支持SQL92/99、ODBC、JDBC、ADO.NET、GCI(OCI/OCCI)、Python接口等国际数据库规范和开发接口。支持集中式部署、共享存储高可用部署、两地三中心高可用部署，具备高容量、高并发、高性能等特性。
  
适配性：对MYSQL部分适配

### 2.5 海盒通用数据库

具体版本：海盒通用数据库管理系统（SeaboxSQL）V11.5
  
官网地址：https://www.seaboxdata.com/
  
SeaboxSQL是北京东方金信科技股份有限公司积累多年数据库开发经验，打造的—款拥有完全自主知识产权、面向事务型业务处理的企业级关系型数据库。SeaboxSQL提供丰富的企业级功能，在性能、安全性、Oracle兼容性、高可用性、易用性等方面均有较多技术特点和优势，支持HA、读写分离、共享存储集群部署架构。同时提供了针对数据分析场景的列存储引擎与向量执行引擎，极大提升在事务和分析混合业务领域的适用范围。
  
适配性：对MYSQL兼容性较差，迁移的话需要消耗的成本较高，而且有一定的学习成本。

### 2.6 金仓数据库

具体版本：金仓数据库管理系统KingbaseES V8
  
官网地址：https://www.kingbase.com.cn/
  
KingbaseES（简称KES）是面向全行业、全客户关键应用的企业级大型通用融合数据库产品，适用于事务处理类应用、数据分析类应用、海量时序数据采集检索类应用、要求苛刻的互联网应用等场景；可用作管理信息系统、业务及生产系统、决策支持系统、多维数据分析系统、运行日志管理系统、全文检索系统、地理信息系统、时序数据处理相关系统的承载数据库。 KES采用融合数据库架构，通过多语法体系一体化架构实现一套软件兼容Oracle、MySQL、SQL Server、PostgreSQL等多个异构数据库的语法； 采用多模数据一体化存储，支持对关系模型、文档模型、全文本、GIS数据、时序等数据的统一存储、混合访问、模型间转换； 采用集中分布一体化架构，满足不同级别的可用性，为客户提供不同级别的可用性、性能扩展、成本需求，确保业务连续，最大化投资价值。

### 2.7 海量数据库

具体版本：海量数据库G100管理系统V2.2
  
官网地址：https://www.vastdata.com.cn/
  
海量数据库Vastbase G100是海量数据基于开源openGauss内核开发的企业级关系型数据库。融合了多年对各行业应用场景的深入理解，Vastbase除了具有openGauss极致的性能以外，还在其原有功能基础上增加了大量国际主流数据库的兼容特性，提供了极佳的业务连续性保障，并遵循EAL4+标准进行了大幅度的安全增强，同时整合GIS空间组件等一系列专业应用领域的企业级功能，为用户数据库选型提供更高性价比。

### 2.8 万里安全数据库

具体版本：万里安全数据库软件V1.0
  
官网地址：https://www.greatdb.com/
  
万里安全数据库（集中式）GreatDB Enterprise 是一款自主研发的集中式数据库产品，基于数据冗余与副本管理确保数据库系统的稳定可靠无单点，同时提供完备的事务支持，适用于金融级在线事务处理和高并发业务场景。

### 2.9 优炫数据库

具体版本：优炫数据库管理系统V2.1
  
官网地址：https://www.uxsino.com/index.php/index/185.html
  
优炫数据库管理系统（简称：优炫数据库，英文名称：UXDB）是北京优炫软件股份有限公司研发的安全可信数据库，拥有自主知识产权，是自主可控国产数据库软件，符合ANSI SQL国际标准，提供完善的数据存储与数据管理功能，具有众多优异的产品特性，可满足各类信息化业务需求。 产品已完成与主流国产芯片、操作系统、中间件，以及应用软件的适配，满足我国政府、金融、能源、制造、医疗等各行业应用需求。
  
适配性：对MYSQL部分适配

## 3. 总结

以上是几个满足新创要求的数据库描述，在此做个表格总结一下。

| 数据库 | MYSQL兼容性 | 单机部署 | 官方地址 |
| --- | --- | --- | --- |
| 瀚高安全版数据库 | 否 | 是 | https://www.highgo.com/product_safe.html |
| 虚谷数据库 | 部分适配 | 是 | https://www.xugudb.com/ |
| 南大通用安全数据库 | 部分适配 | 是 | https://www.gbase.cn/ |
| 海盒通用数据库 | 否 | 是 | https://www.seaboxdata.com/ |
| 金仓数据库 | 基本适配 | 是 | https://www.kingbase.com.cn/ |
| 海量数据库 | 部分适配 | 是 | https://www.vastdata.com.cn/ |
| 万里安全数据库 | 是 | 是 | https://www.greatdb.com/ |
| 优炫数据库 | 部分适配 | 是 | https://www.uxsino.com/index.php/index/185.html |

从官网和使用数据库的厂商大概看下，其中虚谷数据库，南大通用安全数据库，金仓数据库，这三个使用的厂商多一些，相关资料也全一些，使用的时候需要对每个数据库进行仔细评估，决定用哪个。
