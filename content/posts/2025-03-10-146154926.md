    <p>
     简介：
    </p>
    <p>
     Scala 是一种多范式编程语言，结合了面向对象编程和函数式编程的特性。它运行在 Java 虚拟机上，并且可以与 Java 代码无缝互操作。我主要记录一些我之前看过这本书的记录等。详细说一下Scala 中的一些基本数据结构，包括数组、列表、元组、集合和映射。
    </p>
    <h3>
     1. 数组（Array）
    </h3>
    <p>
     数组是 Scala 中最基本的数据结构之一，用于存储相同类型的元素。Scala 的数组是可变的，意味着修改数组中的元素。
    </p>
    <h4>
     创建数组
    </h4>
    <p>
     通过多种方式创建数组：
    </p>
    <pre><code class="language-java">val greetStrings = new Array[String](3)
greetStrings.update(0, "hello")
greetStrings.update(1, ",")
greetStrings.update(2, "world!!\n")</code></pre>
    <p>
     这里我们创建了一个长度为 3 的字符串数组，并使用
     <code>
      update
     </code>
     方法为数组赋值。Scala 中的数组索引从 0 开始。
    </p>
    <h4>
     访问数组元素
    </h4>
    <p>
     <code>
      apply
     </code>
     方法来访问数组中的元素：
    </p>
    <pre><code class="language-java">for (i &lt;- 0 to 2) {
  print(greetStrings.apply(i))
}</code></pre>
    <p>
     简洁方式创建和初始化数组：
    </p>
    <pre><code class="language-java">val greetS1 = Array("hello", ",", "world!!")
for (i &lt;- 0.to(2)) {
  print(greetS1.apply(i))
}</code></pre>
    <h4>
     数组的不可变性
    </h4>
    <p>
     虽然数组本身是可变的，但数组的长度是不可变的。一旦创建了一个数组，就不能改变它的大小。
    </p>
    <h3>
     2. 列表（List）
    </h3>
    <p>
     列表是 Scala 中另一种常用的数据结构，与数组不同，列表是不可变的。这意味着一旦创建了一个列表，就不能修改它的内容。
    </p>
    <h4>
     创建列表
    </h4>
    <p>
     使用
     <code>
      List
     </code>
     对象创建一个列表：
    </p>
    <pre><code class="language-java">val s1 = List(1, 2)
val s2 = List(3, 4)</code></pre>
    <h4>
     列表的拼接
    </h4>
    <p>
     <code>
      :::
     </code>
     操作符来拼接两个列表：
    </p>
    <pre><code class="language-javascript">val s3 = s1 ::: s2
println(s3)  // 输出: List(1, 2, 3, 4)</code></pre>
    <h4>
     在列表前添加元素
    </h4>
    <p>
     使用
     <code>
      ::
     </code>
     操作符在列表的前面添加一个元素：
    </p>
    <pre><code class="language-java">val s4 = 1 :: s3
println(s4)  // 输出: List(1, 1, 2, 3, 4)</code></pre>
    <h3>
     3. 元组（Tuple）
    </h3>
    <p>
     元组是 Scala 中用于存储不同类型元素的数据结构。与列表不同，元组是不可变的，并且可以存储不同类型的元素。
    </p>
    <h4>
     创建元组
    </h4>
    <p>
     使用圆括号来创建一个元组：
    </p>
    <pre>val pari1 = (1, "22")</pre>
    <h4>
     访问元组元素
    </h4>
    <p>
     使用
     <code>
      ._1
     </code>
     、
     <code>
      ._2
     </code>
     等方法来访问元组中的元素：
    </p>
    <pre><code class="language-java">println(pari1._1)  // 输出: 1
println(pari1._2)  // 输出: 22</code></pre>
    <p>
    </p>
    <h3>
     4. 集合（Set）
    </h3>
    <p>
     集合是 Scala 中用于存储唯一元素的数据结构。Scala 提供了可变和不可变两种版本的集合。
    </p>
    <h4>
     创建可变集合
    </h4>
    <p>
     使用
     <code>
      mutable.Set
     </code>
     来创建一个可变集合：
    </p>
    <pre><code class="language-java">import scala.collection.mutable
val movieSet = mutable.Set("Li", "Wu")</code></pre>
    <h4>
     添加元素
    </h4>
    <p>
     使用
     <code>
      +=
     </code>
     操作符向可变集合中添加元素：
    </p>
    <pre><code class="language-Scala">movieSet += "love"
println(movieSet)  // 输出: Set(Li, Wu, love)</code></pre>
    <h4>
     不可变集合
    </h4>
    <p>
     Scala 默认的集合是不可变的，无需引入任何包即可使用：
    </p>
    <pre><code class="language-Scala">val immutableSet = Set("apple", "banana")</code></pre>
    <h3>
     5. 映射（Map）
    </h3>
    <p>
     映射是 Scala 中用于存储键值对的数据结构。Scala 提供了可变和不可变两种版本的映射。
    </p>
    <h4>
     创建可变映射
    </h4>
    <p>
     使用
     <code>
      mutable.Map
     </code>
     来创建一个可变映射：
    </p>
    <pre><code class="language-Scala">import scala.collection.mutable
val tmap = mutable.Map[Int, String]()
tmap += (1 -&gt; "guangzhou")
tmap += (2 -&gt; "beijing")
tmap += (3 -&gt; "shenzheng")
println(tmap(3))  // 输出: shenzheng</code></pre>
    <h4>
     创建不可变映射
    </h4>
    <p>
     Scala 默认的映射是不可变的，你无需引入任何包即可使用：
    </p>
    <pre><code class="language-Scala">val tiMap = Map(1 -&gt; "hebei", 2 -&gt; "chengde", 3 -&gt; "shijiazhuang", 4 -&gt; "beijing")
println(s"tiMap = ${tiMap}")  // 输出: tiMap = Map(1 -&gt; hebei, 2 -&gt; chengde, 3 -&gt; shijiazhuang, 4 -&gt; beijing)</code></pre>
    <p>
     以上就是我对这块知识点的总结，这是基础中的基础了。
    </p>

    <pre><code class="language-Scala">package ScalaBiancheng
