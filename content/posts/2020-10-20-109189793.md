     安装32位Pycharm出现的系列问题if you already have a 32-bit JDK installed详细解决方法
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     出现This installation contains JetBrains Runtime 11 which does not support Microsoft Windows 32-bit version.
     <img alt="" height="399" src="https://i-blog.csdnimg.cn/blog_migrate/32035ee3d8fe2d0716d6aec421e955c7.png" width="1200">
      这个导致的原因是电脑操作系统是32位的，不支持Pycharm64位的软件安装，因此需要我们去下载一个Pycharm32位的软件版本。可以在
      <a href="https://www.jetbrains.com/pycharm/download/other.html" rel="nofollow" title="Other Versions - PyCharm">
       Other Versions - PyCharm
      </a>
      这里进入，选择Pycharm低版本的，例如2018版本Pycharm2018.2版本，如果是老师学员，可以直接在群文件夹里面下载
      <img alt="" height="287" src="https://i-blog.csdnimg.cn/blog_migrate/996a32b9d2eb890d8fac337e5be46b7b.png" width="1063">
       下载之后进行常规安装，在如下图界面中一定要选择32-bit lanucher.
       <img alt="" height="693" src="https://i-blog.csdnimg.cn/blog_migrate/2276dfc206bf68c74517028f25f7d868.png" width="1200"/>
      </img>
     </img>
    </p>
    <p>
     运行之后如果出现Failed to load JVM DLL \bin\server\jvm.dll if you already have a 32-bit JDK installed, define a JAVA_HOME variable in Computer .....如下图所示：
     <img alt="" height="410" src="https://i-blog.csdnimg.cn/blog_migrate/082bea85cf71920164f195c8f63f4369.png" width="1200"/>
    </p>
    <p>
     这是由于操作系统缺少JDK文件导致的，所以安装JDK，并修改系统环境变量，JDK文件可以在
     <a href="https://www.jetbrains.com/pycharm/download/other.html" rel="nofollow" title="Other Versions - PyCharm">
      Other Versions - PyCharm
     </a>
     中下载
    </p>
    <p>
     <img alt="" height="550" src="https://i-blog.csdnimg.cn/blog_migrate/80bde966522a01f1ae5e17bb714f5fd2.png" width="1200"/>
    </p>
    <p>
     具体的安装和系统环境变量配置方法可以查看

      https://jingyan.baidu.com/article/f96699bb163475894e3c1be4.html
     </a>
    </p>
