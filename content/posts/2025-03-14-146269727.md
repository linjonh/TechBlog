     安装 g++-5v的版本：
    </p>
    <pre><code class="language-bash">sudo apt update
sudo apt install g++-5-mips64el-linux-gnuabi64</code></pre>
    <p>
     <a href="https://www.howtoinstall.me/ubuntu/18-04/g++-5-mips64el-linux-gnuabi64/" rel="nofollow" title="How to Install g++-5-mips64el-linux-gnuabi64 in Ubuntu 18.04">

     </a>
    </p>
    <p>
    </p>
    <p>
     安装 gcc/g++-7v的版本：
    </p>
    <pre><code class="language-bash">sudo apt-get install gcc-mips64el-linux-gnu* g++-mips64el-linux-gnu* -y</code></pre>
    <p>
    </p>
    <p>
     安装 gcc/g++-8v的版本：
    </p>
    <pre><code class="language-bash">sudo apt-get install g++-8-mips64el-linux-gnuabi64 -y
sudo apt-get install gcc-8-mips64el-linux-gnuabi64 -y</code></pre>
