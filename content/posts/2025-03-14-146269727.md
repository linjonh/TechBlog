# Ubuntu 18,04 LTS 通过APT安装mips64el的交叉编译器。

安装 g++-5v的版本：

```bash
sudo apt update
sudo apt install g++-5-mips64el-linux-gnuabi64
```

[How to Install g++-5-mips64el-linux-gnuabi64 in Ubuntu 18.04](https://www.howtoinstall.me/ubuntu/18-04/g++-5-mips64el-linux-gnuabi64/ "How to Install g++-5-mips64el-linux-gnuabi64 in Ubuntu 18.04")

安装 gcc/g++-7v的版本：

```bash
sudo apt-get install gcc-mips64el-linux-gnu* g++-mips64el-linux-gnu* -y
```

安装 gcc/g++-8v的版本：

```bash
sudo apt-get install g++-8-mips64el-linux-gnuabi64 -y
sudo apt-get install gcc-8-mips64el-linux-gnuabi64 -y
```
