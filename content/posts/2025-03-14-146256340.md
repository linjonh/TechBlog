    <h2>
     <a id="1_addWeighted_0">
     </a>
     1 addWeighted函数
    </h2>
    <p>
     在OpenCV 里，
     <code>
      addWeighted
     </code>
     函数的作用是对两个图像进行加权求和，常用于图像融合、图像过渡等场景。函数如下：
    </p>
    <pre><code class="prism language-python">cv2<span class="token punctuation">.</span>addWeighted<span class="token punctuation">(</span>src1<span class="token punctuation">,</span> alpha<span class="token punctuation">,</span> src2<span class="token punctuation">,</span> beta<span class="token punctuation">,</span> gamma<span class="token punctuation">[</span><span class="token punctuation">,</span> dst<span class="token punctuation">[</span><span class="token punctuation">,</span> dtype<span class="token punctuation">]</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
</code></pre>
    <h2>
     <a id="2__8">
     </a>
     2 参数解释
    </h2>
    <ul>
     <li>
      <code>
       src1
      </code>
      ：第一个输入图像。
     </li>
     <li>
      <code>
       alpha
      </code>
      ：第一个输入图像的权重，取值范围是 0 到 1。
     </li>
     <li>
      <code>
       src2
      </code>
      ：第二个输入图像，它的大小和通道数必须与
      <code>
       src1
      </code>
      相同。
     </li>
     <li>
      <code>
       beta
      </code>
      ：第二个输入图像的权重，取值范围是 0 到 1。
     </li>
     <li>
      <code>
       gamma
      </code>
      ：标量值，添加到加权和中（通常用于亮度调整）。
     </li>
     <li>
      <code>
       dst
      </code>
      ：输出图像（可选参数，直接通过返回值获取）。
     </li>
     <li>
      <code>
       dtype
      </code>
      ：输出图像的可选深度，若未指定，则使用输入图像的深度。
     </li>
    </ul>

    <h2>
     <a id="3__23">
     </a>
     3 注意
    </h2>
    <ol>
     <li>
      <strong>
       图像尺寸和通道数
      </strong>
      ：
      <code>
       src1
      </code>
      和
      <code>
       src2
      </code>
      必须具有相同的尺寸和通道数，否则会报错。
     </li>
     <li>
      <strong>
       权重和
      </strong>
      ：
      <code>
       alpha + beta
      </code>
      不需要等于 1，但若想实现透明度混合（如
      <code>
       alpha + beta = 1
      </code>
      ），需自行控制。
     </li>
     <li>
      <strong>
       数据类型
      </strong>
      ：若输入为
      <code>
       uint8
      </code>
      类型，结果会自动截断到 [0, 255] 范围（饱和操作）。
     </li>
     <li>
      <strong>
       gamma 的作用
      </strong>
      ：用于调整输出图像的亮度（例如，
      <code>
       gamma=10
      </code>
      会使整体亮度增加）。
      <ul>
       <li>
        若 alpha + beta &gt; 1，图像可能过曝（值被截断到 255）。
       </li>
       <li>
        若 gamma &gt; 0，整体亮度增加；若 gamma &lt; 0，亮度降低。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       与
       <code>
        cv2.add()
       </code>
       的区别
      </strong>
      ：
      <code>
       cv2.add()
      </code>
      是直接相加（无权重），而
      <code>
       addWeighted()
      </code>
      允许更灵活的线性组合。
     </li>
    </ol>

    <h2>
     <a id="5__64">
     </a>
     5 函数应用
    </h2>
    <ol>
     <li>
      <p>
       <strong>
        图像叠加
       </strong>
       ：例如，在制作幻灯片过渡效果时，就可以使用该函数实现图像的平滑过渡：通过调整
       <code>
        alpha
       </code>
       和
       <code>
        beta
       </code>
       实现淡入淡出效果。
      </p>
     </li>
     <li>
      <p>
       <strong>
        ROI 混合
       </strong>
       ：结合掩码（mask）对局部区域进行混合：
      </p>
     </li>
    </ol>
    <pre><code class="prism language-python">roi <span class="token operator">=</span> img1<span class="token punctuation">[</span>y<span class="token punctuation">:</span>y<span class="token operator">+</span>h<span class="token punctuation">,</span> x<span class="token punctuation">:</span>x<span class="token operator">+</span>w<span class="token punctuation">]</span>
