    <pre><code>git init

git pull origin master
git add .
git commit -m ‘init’
git push origin master
</code></pre>
    <p>

      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a9ee93f3e56b4b98820e7d94f8ab1885.png"/>
