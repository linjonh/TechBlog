    <h2>
     HTML 文本格式化
    </h2>
    <p>
     在构建网页的过程中，文本的格式化是一个至关重要的环节。HTML（HyperText Markup Language）提供了丰富的标签和属性来帮助我们实现各种文本格式化的需求。本文将详细介绍HTML中常见的文本格式化方法，包括字体、颜色、大小、对齐方式以及更多高级技巧。
    </p>
    <h3>
     字体格式化
    </h3>
    <h4>
     标题标签
    </h4>
    <p>
     HTML提供了
     <code>
      &lt;h1&gt;
     </code>
     至
     <code>
      &lt;h6&gt;
     </code>
     六个标题标签，用于定义不同级别的标题。
     <code>
      &lt;h1&gt;
     </code>
     为最高级别，通常用于网站的主标题；而
     <code>
      &lt;h6&gt;
     </code>
     为最低级别，适用于较小的副标题。
    </p>
    <pre><code class="language-html">&lt;h1&gt;HTML 文本格式化概述&lt;/h1&gt;
&lt;h2&gt;字体样式&lt;/h2&gt;
&lt;h3&gt;标题标签的应用&lt;/h3&gt;
</code></pre>
    <h4>
     字体样式
    </h4>
    <p>
     使用
     <code>
      &lt;font&gt;
     </code>
     标签，我们可以改变文本的字体、大小和颜色。然而，由于HTML5标准不再推荐使用
     <code>
      &lt;font&gt;
     </code>
     标签，建议使用CSS来控制字体样式。
    </p>
    <pre><code class="language-html">&lt;font face="Arial" size="5" color="#FF0000"&gt;Arial 字体，红色，字号5&lt;/font&gt;
</code></pre>
    <p>
     或者使用CSS：
    </p>
    <pre><code class="language-html">&lt;style&gt;
    .custom-font {
        font-family: Arial;
        font-size: 5em;
        color: red;
    }
&lt;/style&gt;

&lt;p class="custom-font"&gt;Arial 字体，红色，字号5&lt;/p&gt;
</code></pre>
    <h4>
     字体大小
    </h4>
    <p>
     HTML提供了
     <code>
      &lt;small&gt;
     </code>
     和
     <code>
      &lt;big&gt;
     </code>
     标签来控制文本大小。然而，这些标签在HTML5中已经被废弃，推荐使用CSS来设置字体大小。
    </p>
    <pre><code class="language-html">&lt;big&gt;这是大号文字&lt;/big&gt;
&lt;small&gt;这是小号文字&lt;/small&gt;
</code></pre>
    <p>
     或者使用CSS：
    </p>
    <pre><code class="language-css">.big-text {
    font-size: 1.5em;
}

.small-text {
    font-size: 0.8em;
}
</code></pre>
    <h3>
     颜色格式化
    </h3>
    <h4>
     颜色代码
    </h4>
    <p>
     在HTML中，可以使用颜色代码来定义文本颜色。颜色代码可以是十六进制、RGB或颜色名称。
    </p>
    <pre><code class="language-html">&lt;p style="color: #FF0000;"&gt;红色文本&lt;/p&gt;
&lt;p style="color: rgb(255, 0, 0);"&gt;红色文本&lt;/p&gt;
&lt;p style="color: red;"&gt;红色文本&lt;/p&gt;
</code></pre>
    <h4>
     颜色选择器
    </h4>
    <p>
     CSS提供了丰富的颜色选择器，如
     <code>
      rgb()
     </code>
     、
     <code>
      rgba()
     </code>
     、
     <code>
      hsl()
     </code>
     、
     <code>
      hsla()
     </code>
     等。
    </p>
    <pre><code class="language-css">p {
    color: rgb(255, 255, 255);
    background-color: rgba(0, 0, 255, 0.5);
}
</code></pre>
    <h3>
     文本对齐方式
    </h3>
    <h4>
     对齐方式
    </h4>
    <p>
     在HTML中，我们可以使用
     <code>
      &lt;div&gt;
     </code>
     、
     <code>
      &lt;p&gt;
     </code>
     等标签的
     <code>
      align
     </code>
     属性来控制文本的对齐方式。常见的对齐方式有左对齐、右对齐、居中对齐和两端对齐。
    </p>
    <pre><code class="language-html">&lt;p align="left"&gt;左对齐文本&lt;/p&gt;
&lt;p align="right"&gt;右对齐文本&lt;/p&gt;
&lt;p align="center"&gt;居中对齐文本&lt;/p&gt;
&lt;p align="justify"&gt;两端对齐文本&lt;/p&gt;
</code></pre>
    <h4>
     CSS对齐
    </h4>
    <p>
     使用CSS，我们可以通过
     <code>
      text-align
     </code>
     属性来控制文本对齐方式。
    </p>
    <pre><code class="language-css">p {
    text-align: justify;
}
</code></pre>
    <h3>
     高级格式化
    </h3>
    <h4>
     文本缩进
    </h4>
    <p>
     在HTML中，可以使用
     <code>
      &lt;pre&gt;
     </code>
     标签来设置文本缩进。
    </p>
    <pre><code class="language-html">&lt;pre&gt;这是一个缩进的段落。&lt;/pre&gt;
</code></pre>
    <h4>
     水平线
    </h4>
    <p>
     使用
     <code>
      &lt;hr&gt;
     </code>
     标签可以插入一条水平线，分隔不同内容。
    </p>
    <pre><code class="language-html">&lt;p&gt;这是一段文字。&lt;/p&gt;
&lt;hr&gt;
&lt;p&gt;这是另一段文字。&lt;/p&gt;
</code></pre>
    <h4>
     列表
    </h4>
    <p>
     HTML提供了有序列表和无序列表两种标签来展示列表。
    </p>
    <pre><code class="language-html">&lt;ul&gt;
    &lt;li&gt;列表项1&lt;/li&gt;
    &lt;li&gt;列表项2&lt;/li&gt;
&lt;/ul&gt;

&lt;ol&gt;
    &lt;li&gt;有序列表项1&lt;/li&gt;
    &lt;li&gt;有序列表项2&lt;/li&gt;
&lt;/ol&gt;
</code></pre>
    <h3>
     总结
    </h3>
    <p>
     本文详细介绍了HTML中常见的文本格式化方法，包括字体、颜色、大小、对齐方式以及更多高级技巧。掌握这些方法可以帮助我们更好地构建网页，提高用户体验。在编写HTML代码时，请尽量使用CSS来控制样式，以提高代码的可维护性和扩展性。
    </p>
