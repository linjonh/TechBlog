   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      一、安装python
     </strong>
    </p>
    <p>
     Ubuntu默认的是python 2.7版本，这里我们安装python 3.6版本。
    </p>
    <p>
     （1）添加源并更新
    </p>
    <p>
     <span style="color:#ff0000;">
      <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
       sudo add-apt-repository ppa:fkrull/deadsnakes
      </code>
      <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      </span>
      <br style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);"/>
     </span>
    </p>
    <p>
     <span style="color:#ff0000;">
      <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
       sudo apt-get update
      </code>
      <span style="background-color:rgb(249,242,244);font-family:Consolas, Inconsolata, Courier, monospace;font-size:14px;">
      </span>
     </span>
     <span style="background-color:rgb(249,242,244);color:rgb(199,37,78);font-family:Consolas, Inconsolata, Courier, monospace;font-size:14px;">
     </span>
    </p>
    <p>
     <span style="background-color:rgb(255,255,255);font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;font-size:20px;">
      （2）安装指定版本的python，这地方可以更改成你需要的版本(3.6 )：
     </span>
    </p>
    <p>
    </p>
    <p>
     <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
      <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
       sudo apt-get install python3.6 python3.6-dev
      </code>
      <span style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
      </span>
     </code>
    </p>
    <p>
     <span style="font-family:Consolas, Inconsolata, Courier, monospace;color:#000000;">
      <span style="font-size:14px;background-color:rgb(249,242,244);">
       安装32位库
      </span>
     </span>
    </p>
    <p>
     <span style="background-color:rgb(249,242,244);color:rgb(199,37,78);font-family:Consolas, Inconsolata, Courier, monospace;font-size:14px;">
      sudo apt-get install libncurses5-dev
     </span>
    </p>
    <p>
     <span style="background-color:rgb(249,242,244);font-family:Consolas, Inconsolata, Courier, monospace;font-size:14px;">
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
    </p>
    <p style="color:rgb(199,37,78);">
    </p>
    <h5 style="padding:0px;font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="font-weight:normal;">
      （3）切换python版本：
     </span>
    </h5>
    <p>
     <span style="color:#000000;">
      安装好后会在 /usr/bin/ 下出现 python3.6 这个可执行文件，现在需要切换版本(默认版本2.7)
     </span>
    </p>
    <span style="color:#000000;">
     如下就是设置python3.6的路径，并设置优先级为200，默认优先级最高的优先使用
    </span>
    <br/>
    <code style="color:rgb(199,37,78);font-size:14px;line-height:22px;padding:4px 2px 0px;">
     sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.6 200
    </code>
    <span style="color:#c7254e;">
    </span>
    <br/>
    <p>
     <span style="color:#000000;">
      <strong>
       <br/>
      </strong>
     </span>
    </p>
    <p>
     <span style="color:#000000;">
      <strong>
       注意：所以如果想要使用系统默认的2.7版本，需要键入
      </strong>
     </span>
    </p>
    <p>
     <span style="font-weight:700;">
      <span style="color:#cc0000;">
       <span style="color:rgb(199,37,78);font-family:Consolas, Inconsolata, Courier, monospace;font-size:14px;background-color:rgb(249,242,244);">
        sudo update-alternatives --install /usr/bin/python python /usr/bin/python2.7 250
       </span>
      </span>
     </span>
     <span style="color:#ff6666;">
     </span>
     <span style="color:#000000;">
      并设置优先级大于其他的版本
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="color:#000000;">
      使用
      <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
       sudo update-alternatives --config python
      </code>
      可以查看所有已添加的python程序路径和优先级，同时可以更改使用哪个python版本
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="color:#000000;">
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/e0b7fdb96f2bd5832ca12410f9d8f44d.png">
       <br/>
      </img>
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="color:#000000;">
      <strong>
       二、安装pip
      </strong>
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     <span style="color:#000000;">
      去官网复制get-pip.py脚本文件:
      <a href="https://bootstrap.pypa.io/get-pip.py" rel="nofollow">
       点击打开链接
      </a>
     </span>
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     复制好以后，终端键入
     <code style="font-size:14px;line-height:22px;padding:4px 2px 0px;">
      sudo python get-pip.py
     </code>
     自动下载pip，安装好pip后
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     终端键入
     <span style="color:#cc0000;">
     </span>
     <span style="font-weight:700;">
      <span style="color:#ff0000;">
       pip -V
      </span>
     </span>
     <span style="color:#ff0000;">
     </span>
     查看，是否正确指向python3.6包的路径并且结尾是否是python3.6 ，无误后就可以安装其他包什么的了
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
     当然，如果切换回python2.7也要运行安装pip的命令
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">
    </p>
    <p style="font-family:'-apple-system', 'SF UI Text', Arial, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'WenQuanYi Micro Hei', sans-serif, SimHei, SimSun;background-color:rgb(255,255,255);">

      <br/>
     </img>
