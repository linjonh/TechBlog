     CSS简介以及导入形式
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      CSS（Cascading Style Sheets，层叠样式表）
     </strong>
     是一种用于描述 HTML 或 XML 文档外观和格式的样式表语言。它用于控制网页的布局、颜色、字体、间距等视觉表现，使内容与表现分离，便于维护和更新。
    </p>
    <h5>
     主要功能：
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        布局控制
       </strong>
       ：定义元素的位置、大小、排列方式等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        样式设置
       </strong>
       ：设置字体、颜色、背景、边框等。
      </p>
     </li>
     <li>
      <p>
       <strong>
        响应式设计
       </strong>
       ：通过媒体查询实现不同设备的适配。
      </p>
     </li>
     <li>
      <p>
       <strong>
        动画与过渡
       </strong>
       ：创建动态效果，提升用户体验。
      </p>
     </li>
    </ol>
    <p>
     css语法：css通常由选择器，属性和属性值组成，多个规则可以组合在一起，以便于同时应用多个样式
    </p>
    <pre><code class="language-html">选择器{
    属性1： 属性值1；
    属性2： 属性值2;
}</code></pre>
    <h4>
     1.选择器的生明里面可以写无数条属性。
    </h4>
    <p>
     2.声明的每一行属性，都需要以英文分号结尾；
    </p>
    <p>
     3.声明中所有属性和值都是以键值对这种形式出现的；
    </p>
    <p>
     例如：
    </p>
    <pre><code class="language-html">/*这是一个p标签选择器*/
p{
    color: bule;
    font-size: 16px;
}</code></pre>
    <h4>
     CSS 的导入方式
    </h4>
    <p>
     CSS 可以通过多种方式引入到 HTML 文档中，常见的有以下三种：
    </p>
    <h5>
     1. 内联样式（Inline Styles）
    </h5>
    <p>
     直接在 HTML 元素的
     <span style="color:#fe2c24">
      <strong>
       <code>
        style
       </code>
       属性中
      </strong>
     </span>
     编写 CSS。
    </p>
    <p>
     以下的就会出现
     <span style="color:#0d0016">
      一个颜色为蓝色的一句“
     </span>
     <span style="color:#fe2c24">
      <strong>
       这是一个段落
      </strong>
     </span>
     ”。
    </p>
    <pre><code class="language-html">&lt;p style="color: blue; font-size: 14px;"&gt;这是一个段落。&lt;/p&gt;</code></pre>
    <p>
     <span style="color:#fe2c24">
      <strong>
       特点
      </strong>
     </span>
     ：
    </p>
    <ul>
     <li>
      <p>
       优先级最高。
      </p>
     </li>
     <li>
      <p>
       不利于维护和复用。
      </p>
     </li>
    </ul>
    <h5>
     2. 内部样式表（Internal Style Sheet）
    </h5>
    <p>
     在 HTML 文件的
     <span style="color:#fe2c24">
      <strong>
       <code>
        &lt;head&gt;
       </code>
      </strong>
     </span>
     部分使用
     <span style="color:#fe2c24">
      <strong>
       <code>
        &lt;style&gt;
       </code>
      </strong>
     </span>
     标签定义 CSS。
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="zh-CN"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;title&gt;内部样式表示例&lt;/title&gt;
    &lt;style&gt;
        p {
            color: blue;
            font-size: 14px;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;p&gt;这是一个段落。&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <strong>
      <span style="color:#fe2c24">
       特点
      </span>
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       适用于单个页面。
      </p>
     </li>
     <li>
      <p>
       优先级低于内联样式，但高于外部样式表。
      </p>
     </li>
    </ul>
    <h5>
     3. 外部样式表（External Style Sheet）
    </h5>
    <p>
     将 CSS 代码保存在独立的
     <span style="color:#fe2c24">
      <strong>
       <code>
        .css
       </code>
       文件
      </strong>
     </span>
     中，通过
     <span style="color:#fe2c24">
      <strong>
       <code>
        &lt;link&gt;
       </code>
      </strong>
     </span>
     标签引入。
    </p>
    <p>
     <strong>
      步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       创建 CSS 文件（如
       <code>
        styles.css
       </code>
       ）：
      </p>
     </li>
    </ol>
    <pre><code class="language-html">p {
    color: blue;
    font-size: 14px;
}</code></pre>
    <p>
     2，在 HTML 文件中引入：
    </p>
    <pre><code class="language-html">&lt;!DOCTYPE html&gt;
&lt;html lang="zh-CN"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;title&gt;外部样式表示例&lt;/title&gt;
    &lt;link rel="stylesheet" href="styles.css"&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;p&gt;这是一个段落。&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
    <p>
     <span style="color:#fe2c24">
      <strong>
       特点
      </strong>
     </span>
     ：
    </p>
    <ul>
     <li>
      <p>
       最适合大型项目，便于维护和复用。
      </p>
     </li>
     <li>
      <p>
       优先级低于内联和内部样式表。
      </p>
     </li>
    </ul>
    <h5>
     4. 使用
     <code>
      @import
     </code>
     导入
    </h5>
    <p>
     在 CSS 文件中通过
     <code>
      @import
     </code>
     引入其他 CSS 文件。
    </p>
    <p>
     css
    </p>
    <pre><code class="language-html">@import url('styles.css');</code></pre>
    <p>
     <strong>
      特点
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       适用于
       <span style="color:#fe2c24">
        模块化 CSS。
       </span>
      </p>
     </li>
     <li>
      <p>
       加载顺序可能影响性能。
      </p>
     </li>
    </ul>
    <h4>
     总结
    </h4>
    <table>
     <thead>
      <tr>
       <th>
        导入方式
       </th>
       <th>
        适用场景
       </th>
       <th>
        优先级
       </th>
       <th>
        维护性
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        内联样式
       </td>
       <td>
        单个元素样式
       </td>
       <td>
        最高
       </td>
       <td>
        差
       </td>
      </tr>
      <tr>
       <td>
        内部样式表
       </td>
       <td>
        单个页面样式
       </td>
       <td>
        中
       </td>
       <td>
        一般
       </td>
      </tr>
      <tr>
       <td>
        外部样式表
       </td>
       <td>
        多页面或大型项目
       </td>
       <td>
        低
       </td>
       <td>
        好
       </td>
      </tr>
      <tr>
       <td>
        <code>
         @import
        </code>
       </td>
       <td>
        模块化 CSS
       </td>
       <td>
        低
       </td>
       <td>
        一般
       </td>
      </tr>
     </tbody>
    </table>
    <p>
     根据项目需求选择合适的导入方式，通常推荐使用外部样式表以实现样式与结构的分离。
    </p>
