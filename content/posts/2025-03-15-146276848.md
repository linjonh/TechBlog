    <h2>
     <a id="java_1">
     </a>
     树型结构展示java集合框架常见接口和类
    </h2>
    <p>
     Java 集合框架中的接口和子类关系可以用树形结构来展示。以下是一个简化的树形结构，展示了主要的接口和一些重要的实现类：
    </p>
    <pre><code>java.util.Collection
├── java.util.List
│   ├── java.util.ArrayList
│   ├── java.util.LinkedList
│   ├── java.util.Vector
│   └── java.util.Stack
├── java.util.Set
│   ├── java.util.HashSet
│   ├── java.util.LinkedHashSet
│   └── java.util.TreeSet
└── java.util.Queue
    ├── java.util.PriorityQueue
    └── java.util.Deque
        ├── java.util.ArrayDeque
        └── java.util.LinkedList (also implements List)

java.util.Map
├── java.util.HashMap
├── java.util.LinkedHashMap
├── java.util.TreeMap
└── java.util.Hashtable
    └── java.util.Properties
</code></pre>
    <h4>
     <a id="_30">
     </a>
     主要接口说明：
    </h4>
    <ul>
     <li>
      <strong>
       Collection
      </strong>
      : 是所有集合类的根接口。
     </li>
     <li>
      <strong>
       List
      </strong>
      : 有序集合（也称为序列），允许重复元素。
     </li>
     <li>
      <strong>
       Set
      </strong>
      : 无序集合，不允许重复元素。
     </li>
     <li>
      <strong>
       Queue
      </strong>
      : 通常用于 FIFO（先进先出）访问元素的集合。
     </li>
     <li>
      <strong>
       Deque
      </strong>
      : 双端队列，支持在两端插入和删除元素。
     </li>
     <li>
      <strong>
       Map
      </strong>
      : 键值对（也称为关联数组或字典）的集合，不允许重复键。
     </li>
    </ul>
    <h4>
     <a id="_38">
     </a>
     主要实现类说明：
    </h4>
    <ul>
     <li>
      <strong>
       ArrayList
      </strong>
      : 基于数组实现的列表。
     </li>
     <li>
      <strong>
       LinkedList
      </strong>
      : 基于双向链表实现的列表。
     </li>
     <li>
      <strong>
       HashSet
      </strong>
      : 基于哈希表实现的集合。
     </li>
     <li>
      <strong>
       TreeSet
      </strong>
      : 基于红黑树实现的有序集合。
     </li>
     <li>
      <strong>
       HashMap
      </strong>
      : 基于哈希表实现的映射。
     </li>
     <li>
      <strong>
       TreeMap
      </strong>
      : 基于红黑树实现的有序映射。
     </li>
     <li>
      <strong>
       PriorityQueue
      </strong>
      : 基于堆实现的优先队列。
     </li>
     <li>
      <strong>
       ArrayDeque
      </strong>
      : 基于数组实现的双端队列。
     </li>
    </ul>
    <p>
     这个树形结构展示了 Java 集合框架的主要接口和实现类之间的关系，帮助你更好地理解它们的层次结构和功能。
    </p>
