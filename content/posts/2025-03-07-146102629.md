    <h4>
     <a id="1_ifconfig_1">
     </a>
     <strong>
      1.
      <code>
       ifconfig
      </code>
      ：查看和配置网络接口
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：查看或配置网络接口参数（如 IP 地址、子网掩码、MAC 地址等）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用选项
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         -a
        </code>
        ：显示所有接口（包括未激活的）。
       </li>
       <li>
        <code>
         up
        </code>
        /
        <code>
         down
        </code>
        ：启用或禁用接口。
       </li>
       <li>
        <code>
         inet [IP]
        </code>
        ：设置 IPv4 地址。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        示例
       </strong>
      </p>
      <pre><code class="prism language-bash"><span class="token comment">#临时修改 IP 地址或禁用网卡</span>
<span class="token function">ifconfig</span> en0            <span class="token comment"># 查看 en0 接口信息</span>
<span class="token function">sudo</span> <span class="token function">ifconfig</span> en0 down  <span class="token comment"># 禁用 en0 接口</span>
<span class="token function">sudo</span> <span class="token function">ifconfig</span> en0 <span class="token number">192.168</span>.1.100 netmask <span class="token number">255.255</span>.255.0  <span class="token comment"># 手动设置 IP 和子网掩码</span>
</code></pre>
     </li>
    </ul>

    <h4>
     <a id="2_networksetup_22">
     </a>
     <strong>
      2.
      <code>
       networksetup
      </code>
      ：管理系统网络配置
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        功能
       </strong>
       ：管理网络服务（Wi-Fi、以太网等）、代理、DNS、位置配置。
      </p>
     </li>
     <li>
      <p>
       <strong>
        常用子命令
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         -listallnetworkservices
        </code>
        ：列出所有网络服务。
       </li>
       <li>
        <code>
         -setairportpower &lt;service&gt; on/off
        </code>
        ：启用/禁用 Wi-Fi。
       </li>
       <li>
        <code>
         -setdnsservers &lt;service&gt; &lt;DNS1&gt; &lt;DNS2&gt;
        </code>
        ：设置 DNS。
       </li>
       <li>
        <code>
         -setwebproxy &lt;service&gt; &lt;host&gt; &lt;port&gt;
        </code>
        ：设置 HTTP 代理。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        示例
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token comment">#networksetup 命令在 macOS 中修改的是系统的永久配置</span>
networksetup <span class="token parameter variable">-listallnetworkservices</span>      <span class="token comment"># 列出所有网络服务</span>
<span class="token function">sudo</span> networksetup <span class="token parameter variable">-setdnsservers</span> Wi-Fi <span class="token number">8.8</span>.8.8  <span class="token comment"># 设置 Wi-Fi 的 DNS 为 8.8.8.8</span>
<span class="token function">sudo</span> networksetup <span class="token parameter variable">-setairportpower</span> en0 off  <span class="token comment"># 关闭 Wi-Fi</span>
</code></pre>
     </li>
    </ul>

         -an
        </code>
        ：显示所有连接（不解析域名）。
       </li>
       <li>
        <code>
         -r
        </code>
        ：显示路由表。
       </li>
       <li>
        <code>
         -p tcp
        </code>
        ：显示 TCP 连接。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        示例
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token function">netstat</span> <span class="token parameter variable">-an</span> <span class="token operator">|</span> <span class="token function">grep</span> <span class="token number">80</span>   <span class="token comment"># 查看 80 端口的连接</span>
<span class="token function">netstat</span> <span class="token parameter variable">-r</span>              <span class="token comment"># 显示路由表</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        场景
       </strong>
       ：
      </p>
      <ul>
       <li>
        检查端口占用或异常连接。
       </li>
       <li>
        查看路由表配置。
       </li>
      </ul>
     </li>
    </ul>
