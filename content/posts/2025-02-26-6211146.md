# DNN开发教程1-关于DNN

首先DNN 是我们所说的DotNetNuke 的简称。

**什么是DNN （DotNetNuke ）？**

DotNetNuke 是基于
Microsoft .NET技术的一个开源的网络应用框架. 它也是一个用来创建, 配置, 管理内部和外部网站的理想免费平台.

**DNN特色：**

**通用性----------**
几乎可以用来构建让任何Web应用：商务网站，企业内网／外网，在线内容发布。

****友好的用户体验--****
DotNetNuke 是一个易于管理员,文书编辑,开发者和设计人员全方位使用的系统, 方便不同的 角色便捷的管理自己的

资源.  操作精灵， 帮助文档， 和合理的用户界面提供了超级的用户体验.

**强大的机制**
–---- 只需要安装一个DotNetNuke实例，即可在上面运行多个网站. DotNetNuke 允许管理员管理无限多个网站

每个网站都 可以拥有自己独立的外观和内容. 另外分离的服务器端设置界面和 单独的网站管理界面另管理权限清晰分明.

**丰富的功能**
–---- DotNetNuke提供了丰富的内置功能，内建主机服务，网页设计，内容管理，安全性和会员功能这些全都可以通过简单

方便 的网页界面进行管理和定制.

**服务支持**
–------ DotNetNuke由DotNetNuke公司管理和维护, 同时由自愿开发者组成的核心团队成员和高度活跃的国际社区也对                                   DotNetNuke提供了良好的支持. 通过用户组，在线论坛，资源网站和精通DNN的专业咨询公司寻求帮助永远是一件

轻松的事.

**最先进的技术**
– DotNetNuke提供给开发者一个实现软件开发理念最佳实践的机会，通过应用最先进的技术例如 ASP.net,

Visual Studio, Sql Server, Windows Server 和 IIS.

**本地化**
―------- 到目前为止已经有27个语言包，能很轻松的把您的网站转换成几乎任何语言。语言包也是可扩展的，您可以很容易的

创建自己的特色语言包。

**可扩展性**
–---- DotNetNuke 拥有易插拔的构架，它可以在网站运行期间动态的增加新的功能和界面外观元素. 通过充分利用DNN

内容管理的基本功能，开发者和网站管理员都可以高效的专注于商务需求提供最佳的解决方案.

**被广泛认同**
–--  DotNetNuke是注册商标的名字简称是DNN, 这是一个在开源界被广泛认知的名字. 他拥有超过 60万注册用户

(只是官方网 站),DnnChina.net（中文）已经有超过2,600的注册用户和一群才华洋溢的开发者组成的团队，

DotNetNuke在通过吸取 业界的经验和用户反馈不断的发展壮大着.
