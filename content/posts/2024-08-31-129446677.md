# Windows 在美国采用率迅速下跌，只剩 57%，已至历史最低点！

流水的操作系统，铁打的 Windows。不过，自 2018 年微软宣布分拆 Windows 部门拥抱 AI 之后，作为一项服务的 Windows，其王者地位还能持续多久，也成为不少用户关注的话题。

近日，据 gizmochina 报道，由于 MacOS 和 ChromeOS 的兴起，微软的 Windows 操作系统在美国地区正在迅速失宠。

来自网络分析平台 Statcounter 统计的数据印证了这一趋势。截至 2023 年 2 月，Windows 在美国桌面操作系统市场份额已经降至 57.37%，是自 2009 年以来的数据最低点。有媒体预计，2023 年，Windows 的使用率还会下滑。

![](https://i-blog.csdnimg.cn/blog_migrate/92c5576bf9a0b21137da2a6306f8b459.png)

对于数据下滑的原因，通过其他操作系统使用率情况也不难猜测出来。截至 2023 年 2 月，macOS 在美国桌面操作系统市场的份额为 29.62%，Chrome OS 为 7.47%，Linux 占 2.55%，而其他杂项桌面操作系统占 2.97%。

![](https://i-blog.csdnimg.cn/blog_migrate/bfbe16164b873c7217c00078f2e7c986.png)

因此，macOS 和 ChromeOS 的不断普及无疑是造成 Windows 目前呈下滑趋势的原因之一。

如今，macOS 被广泛应用在苹果的 MacBook 和 Mac 系统产品中，如 Mac Mini、Mac Studio、Macbook Pro 和 Macbook Air。随着 2020 年苹果自研芯片 M1 的推出，越来越多的人选择性能更胜一筹的 MacBook 系列产品，一定程度上蚕食了 Windows 部分市场份额。

另外，Google 近几年对 ChromeOS 的投入也越来越大，搭载该系统的 Chromebook 设备也受到了不少美国用户的青睐。据 gizmochina 称，2010 年，美国的学前班、小学和中学教育机构在设备上通常会选用 Chromebook。这个年龄段的学生现如今长大，与 Windows 设备相比，他们或更熟悉 Chromebook。同时，在笔记本电脑市场上，Chromebook 通常比 Windows 笔记本电脑更便宜，导致许多学校以低廉的价格大量购买 Chromebook。

在此趋势下，据 IDC 数据显示，ChromeOS 在 2020 年的市场份额一度超过 macOS，成为全球第二大桌面操作系统。近期也有传言称，Google 欲将自家研发的 Bard AI 技术引入到 ChromeOS 中，如果能够成真，无疑也为该系统增加了一些竞争力。

当然除了市场竞争这一因素之外，云计算的崛起、开源操作系统的增多等都是导致 Windows 在美国市场份额衰减的原因。

不过，当前微软在深度拥抱 AI 时代之际，也欲将自家的产品全部升级。这不，日前，微软 Bing 官方博客宣布，随着基于 ChatGPT 开发的 Bing 预览版新增用户超过 100 万，Bing 搜索引擎的日活跃用户首次突破 1 亿，可以说，其用 ChatGPT 风格的大模型技术救活了沉寂多年的 Bing 搜索引擎。如今，​ 微软已经将基于 Bing AI 的聊天机器人引入了 Windows 11，能否让 Windows 的采用率更上一层楼，我们也将拭目以待。
