     <span style="color:#FF0000">
      Error:java: Compilation failed: internal java compiler error
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      原因：
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      编译的jdk版本不正确。
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      解决方式：
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      调整jdk到正确的版本。
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      步骤：
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      File --&gt; Setting --&gt; Build Execution, Deployment --&gt; Compiler --&gt; Java Compiler --&gt; Projecct bytecode version 设置版本号为1.7
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      同时，将 Target bytecode version 也设置成1.7.
     </span>
    </div>
    <div style="white-space:pre-wrap; line-height:1.875; font-size:14px">
     <span style="color:#FF0000">
      --&gt;Apply --&gt; OK. 搞定。
     </span>
    </div>
