# 2013微软精英大挑战之编程之美大赛

微软精英大挑战之编程之美大赛即将重燃战火！！！

在2012年成功举办以“大数据的可视化”为主题的全国编程大赛的基础上，微软亚洲研究院再接再厉，继续推出以“漫步云端体验编程之美”为主题的2013编程之美大赛。“漫步云端体验编程之美”的主题紧扣当下最流行的云计算话题，同学们在参加大赛，展示个人魅力与风采之外还能学习和体验微软最新推出的新一代Windows Azure云计算操作系统，通过大赛最新的教学视频与潮流技术前沿保持同步！

本届大赛设置了诸多奖项来回馈同学们的努力：

|  |
| --- |
| * 一等奖1名   访问美国西雅图微软总部，获得微软总部面试机      * 二等奖2名   奖金5000元      * 三等奖3名   奖金3000元     Ø 全国15强 免试录取为微软亚太研发集团实习生  Ø 全国60强 获得一年免费的IEEE学生会员资格；到微软中国总部切磋技艺，决战群雄  Ø 初赛前1000名 即可进入复赛，并获得2013微软编程之美大赛精美纪念T恤一件  Ø 复赛前20名2014年应届毕业生，可赴微软中国总部参加校园招聘直通车活动 |

大赛时间安排如下，供您参考：

|  |
| --- |
| 报名： 即日起至4月5日  登录官网进行在线注册。http://programming2013.cstnet.cn/  资格赛 ：4月6日 至 4月7日（周六周日）  资格赛允许参赛者在48小时内答算法题，题目共三道，答对任意一道即可获得进入初赛的资格。  初赛 ：4月13日 至 4月14日（周六周日）  初赛分为两场，每天下午2点至4点各一场，每场比赛时间为2个小时，共三道算法题，以最短时间内给出最多正确答案者胜出。每场决出约500人，共决出1000人进入复赛。每个ID只限参加其中一场比赛。  复赛 ：4月20日14：00至17：00  复赛比赛时长3小时，共4道算法题，以最短时间内给出最多正确答案者胜出。此次比赛决出约60人参加决赛。  决赛 ：5月  前60强在微软（中国）有限公司总部参加为期三天的全国总决赛，同微软“编程之美”团队华山论剑云技术。决赛将涉及微软最新的云平台windows azure，具体日期、比赛内容和形式将在5月上旬公布细则。  注： 更详细的比赛安排、技术文档请关注  大赛官网  。（如安排有变更，请以官网信息为准） |
