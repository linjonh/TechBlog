# 字符映射表 charmap

## 特殊字符（字符映射表）：常见问题

特殊字符是在键盘上找不到的字符。 “字符映射表”将列出选定字体中可用的字符。 你可以搜索特殊字符、复制单个字符或一组特殊字符，以及将字符粘贴到任何可显示它们的应用中。

[全部显示](#)

#### [如何将特殊字符插入文件？](#)

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 点击或单击“字体”

   列表，然后选择要使用的字体。
3. 点击或单击要插入到文件中的特殊字符。
4. 点击或单击“选择”

   ，然后点击或单击“复制”

   。
5. 打开文件，并选择文件中要插入特殊字符的位置。
6. 点击或单击

   “编辑”菜单，然后点击或单击

   “粘贴”。

![提示](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/alertset_warning.png "提示")

#### 提示

许多应用都允许你将特殊字符拖到文件中。 若要执行此操作，请点击或单击要复制的字符。 当字符放大显示时，将其拖到打开的文件中。

![注意](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/alertset_warning.png "注意")

#### 注意

如果在“复制字符”

框中没有删除以前复制的字符，则这些字符将随所有选择的新字符一起被复制。

#### 使用键盘

如果经常使用特殊字符，直接使用键盘输入可能更快一些。 你需要知道字符的击键值。

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 选择一个特殊字符，然后在窗口底部的状态栏中搜索击键值。

   ![“字符映射表”中特殊字符的击键值](https://i-blog.csdnimg.cn/blog_migrate/f40e786c46257b144a3944408511fedf.jpeg "“字符映射表”中特殊字符的击键值")

   特殊字符的击键值
3. 打开文件，将光标放置在希望特殊字符出现的位置。
4. 在 Num Lock 处于打开状态时，长按 Alt 键的同时按下数字键盘上代表要使用字符的键击值的键。
5. 在完成键入后，松开 Alt 键。

#### [如何创建自己的字符？](#)

如果在“字符映射表”中找不到要查找的字符，你可以使用“专用字符编辑程序”创建该字符。

#### [如何按名称搜索字符？](#)

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 在“字体”

   列表中，点击或单击要搜索的字体，然后选中“高级查看”

   复选框。
3. 在“字符集”

   列表中，点击或单击要搜索的字符集。

   如果所需的字符集不可用，请在“字体”

   列表中选择其他字体。
4. 在“搜索”

   框中，输入要搜索字符的完整或部分名称。 例如，若要搜索任何希腊字母，请输入“Greek”

   ，或者要查找各种类型的西里尔字符 Zhe，请输入“Zhe”

   。
5. 点击或单击

   “搜索”。

   “字符映射表”会显示与搜索相匹配的字符。

   ![“字符映射表”中的搜索结果](https://i-blog.csdnimg.cn/blog_migrate/66e8e28d0db52aecd6c94a6713b100b6.jpeg "“字符映射表”中的搜索结果")

   “字符映射表”中的搜索结果
6. 若要开始其他搜索，请点击或单击“重置”

   。

![提示](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/alertset_warning.png "提示")

#### 提示

若要查看特定字符的名称以及与其相关的十六进制代码，请长按该字符或指向该字符。 点击或单击该字符时，此信息也会显示在“字符映射表”窗口底部的状态栏上，同时，它也可以作为搜索词的一个例子用于其他搜索。

#### [如何搜索一组类型相似的字符？](#)

Unicode 类别用于将字体中的字符按相似类型进行分组。 例如，Currency Unicode 类别将所有适用于字体的货币符号进行分组。

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 在“字体”

   列表中，点击或单击要搜索的字体，然后选中“高级查看”

   复选框。
3. 在“分组依据”

   列表中，点击或单击“Unicode 子范围”

   。
4. 在“分组依据”

   对话框中，点击或单击要查看的字符类别。

   “字符映射表”会显示属于该类别的字符。

   ![“字符映射表”中的相似字符组](https://i-blog.csdnimg.cn/blog_migrate/3546fbdd8e5282004f824d2b24db07f2.jpeg "“字符映射表”中的相似字符组")

   “字符映射表”中的相似字符组
5. 找到要查找的字符后，点击或单击“选择”

   ，然后点击或单击“复制”

   。

#### [如何按 Unicode 值搜索字符？](#)

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 在“字体”

   列表中，点击或单击要搜索的字体，然后选中“高级查看”

   复选框。
3. 在“字符集”

   列表中，点击或单击“Unicode”

   。
4. 在“分组依据”

   列表中，点击或单击“全部”

   。
5. 在“转到 Unicode”

   框中，输入要查找的字符的四字符 Unicode 值。

   该字符将自动在字符网格中突出显示。

![提示](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/alertset_warning.png "提示")

#### 提示

在“字符映射表”中，某一特定字体的所有字符都按它们的 Unicode 值顺序显示。 如果你不知道要查找字符的 Unicode 值，你可以通过滚动浏览字符表来搜索该字符。

#### [如何使用字符发音来搜索并插入日文汉字、朝鲜文汉字、简体中文和繁体中文字符？](#)

知道要插入字符的发音后，就可以使用平假名来搜索日文汉字，使用朝鲜文来搜索朝鲜文汉字，使用拼音来搜索简体中文，或者使用注音来搜索繁体中文。

1. ![](http://res1.windows.microsoft.com/resources/4.1/helphub/shared/images/merged/hh_ppeic.png)
   点击或单击打开“字符映射表”。
2. 选中“高级查看”

   复选框，然后在“分组依据”

   列表中，点击或单击要搜索的字符分组。
3. 在“分组依据”

   对话框中，点击或单击与要搜索的发音相对应的字符。

   “字符映射表”会显示以特定发音开头的字符。
4. 选择要插入的字符。
