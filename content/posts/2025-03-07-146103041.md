# 第四章：go 不同进制的区分 for \_, r :=range 使用

#### **Go 语言里的字符串的内部实现使用 `UTF-8` 编码。 字符串的值为 `双引号(")` 中的内容。**

#### 我们可以借助fmt函数来将一个整数以不同进制形式展示。

**占位符%b表示二进制**

```Go
package main
 
import "fmt"
 
func main(){
	// 十进制
	var a int = 10
	fmt.Printf("%d \n", a)  // 10
	fmt.Printf("%b \n", a)  // 1010  占位符%b表示二进制
 
	// 八进制  以0开头
	var b int = 077
	fmt.Printf("%o \n", b)  // 77
 
	// 十六进制  以0x开头
	var c int = 0xff
	fmt.Printf("%x \n", c)  // ff
	fmt.Printf("%X \n", c)  // FF
}

```
