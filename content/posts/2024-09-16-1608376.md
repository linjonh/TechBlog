      </span>

    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      手机操作系统一般只应用在高端智能化手机上。目前，在智能手机市场上，中国市场仍以个人信息管理型手机为主，随着更多厂商的加入，整体市场的竞争已经开始呈现出分散化的态势。从市场容量、竞争状态和应用状况上来看，整个市场仍处于启动阶段。
     </span>
    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      目前应用在手机上的操作系统主要有
     </span>
     <u>
      <span style="FONT-SIZE: 10pt">
       PalmOS
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       、
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       Symbian
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       、
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       Windows CE
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       和
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       Linux
      </span>
     </u>
     <span style="FONT-SIZE: 10pt">
      四种。
     </span>
    </div>
    <div>
     <u>
      <span style="FONT-SIZE: 10pt">
       Palm OS
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       操作系统
      </span>
     </u>
    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      Palm OS
     </span>
     <span style="FONT-SIZE: 10pt">
      操作系统由
     </span>
     <span style="FONT-SIZE: 10pt">
      Palm
     </span>
     <span style="FONT-SIZE: 10pt">
      公司自行开发的，并授权给
     </span>
     <span style="FONT-SIZE: 10pt">
      Handspring
     </span>
     <span style="FONT-SIZE: 10pt">
      、索尼和高通等设备厂家，这种操作系统更倾向于
     </span>
     <span style="FONT-SIZE: 10pt">
      PDA
     </span>
     <span style="FONT-SIZE: 10pt">
      的操作系统。
     </span>
    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      Palm OS
     </span>
     <span style="FONT-SIZE: 10pt">
      在
     </span>
     <span style="FONT-SIZE: 10pt">
      PDA
     </span>
     <span style="FONT-SIZE: 10pt">
      市场占有主导地位。
     </span>
     <span style="FONT-SIZE: 10pt">
      Palm
     </span>
     <span style="FONT-SIZE: 10pt">
      的产品线本身就包括智能手机，又宣布与最早的智能手机开发者
     </span>
     <span style="FONT-SIZE: 10pt">
      Handspring
     </span>
     <span style="FONT-SIZE: 10pt">
      购并，同时将软件部门独立
     </span>
    </div>
    <div>
     <u>
      <span style="FONT-SIZE: 10pt">
       Symbian
      </span>
     </u>
     <u>
      <span style="FONT-SIZE: 10pt">
       系统
      </span>
     </u>
    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      Symbian
     </span>
     <span style="FONT-SIZE: 10pt">
      是一个实时性、多任务的纯
     </span>
     <span style="FONT-SIZE: 10pt">
      32
     </span>
     <span style="FONT-SIZE: 10pt">
      位操作系统，具有功耗低、内存占用少等特点，非常适合手机等移动设备使用，经过不断完善，可以支持
     </span>
     <span style="FONT-SIZE: 10pt">
      GPRS
     </span>
     <span style="FONT-SIZE: 10pt">
      、蓝芽、
     </span>
     <span style="FONT-SIZE: 10pt">
      SyncML
     </span>
     <span style="FONT-SIZE: 10pt">
      、以及
     </span>
     <span style="FONT-SIZE: 10pt">
      3G
     </span>
     <span style="FONT-SIZE: 10pt">
      技术。最重要的是它是一个标准化的开放式平台，任何人都可以为支持
     </span>
     <span style="FONT-SIZE: 10pt">
      Symbian
     </span>
     <span style="FONT-SIZE: 10pt">
      的设备开发软件。与微软产品不同的是，
     </span>
     <span style="FONT-SIZE: 10pt">
      Symbian
     </span>
     <span style="FONT-SIZE: 10pt">
      将移动设备的通用技术，也就是操作系统的内核，与图形用户界面技术分开，能很好的适应不同方式输入的平台，也可以使厂商可以为自己的产品制作更加友好的操作界面，符合个性化的潮流，这也是用户能见到不同样子的
     </span>
     <span style="FONT-SIZE: 10pt">
      symbian
     </span>
     <span style="FONT-SIZE: 10pt">
      系统的主要原因。现在为这个平台开发的
     </span>
     <span style="FONT-SIZE: 10pt">
      java
     </span>
     <span style="FONT-SIZE: 10pt">
      程序已经开始在互联网上盛行。用户可以通过安装这些软件，扩展手机功能。
     </span>
    </div>
    <div>
     <span style="FONT-SIZE: 10pt">
      在
     </span>
     <span style="FONT-SIZE: 10pt">
      Symbian
     </span>
     <span style="FONT-SIZE: 10pt">
      发展阶段，出现了三个分支：分别是
     </span>
     <span style="FONT-SIZE: 10pt">
      Crystal
     </span>
     <span style="FONT-SIZE: 10pt">
      、
     </span>
     <span style="FONT-SIZE: 10pt">
      Pearl
     </span>
     <span style="FONT-SIZE: 10pt">
      和
     </span>
     <span style="FONT-SIZE: 10pt">
      Quarz
     </span>
     <span style="FONT-SIZE: 10pt">
      。前两个主要针对通讯器市场，也是出现在手机上最多的，是今后智能手机操作系统的主力军。第一款基于
     </span>
     <span style="FONT-SIZE: 10pt">
      Symabian
     </span>
     <span style="FONT-SIZE: 10pt">
      系统的手机是
     </span>
     <span style="FONT-SIZE: 10pt">
      2000
     </span>
     <span style="FONT-SIZE: 10pt">
      年上市的某款爱立信手机。而真正较为成熟的同时引起人们注意的则是
     </span>
     <span style="FONT-SIZE: 10pt">
      2001
     </span>
     <span style="FONT-SIZE: 10pt">
      年上市的诺基亚
     </span>
     <span style="FONT-SIZE: 10pt">
      9210
     </span>
     <span style="FONT-SIZE: 10pt">
      ，它采用了
     </span>
     <span style="FONT-SIZE: 10pt">
      Crystal
     </span>
     <span style="FONT-SIZE: 10pt">
      分支的系统。而
     </span>
     <span style="FONT-SIZE: 10pt">
      2002
     </span>
     <span style="FONT-SIZE: 10pt">
      年推出的诺基亚
     </span>
