    <h4>
     <strong>
      1 检查是否已经存在mysql，若存在卸载，避免安装时产生错误
     </strong>
    </h4>
    <pre><code class="hljs">rpm -qa | grep -i mysql</code></pre>
    <p>
     如果存在，请参考这篇文章
    </p>
    <p>
     <a href="https://blog.csdn.net/u011911672/article/details/146181081?spm=1001.2014.3001.5501" title="Centos7 虚拟机删除mariadb-CSDN博客">
      Centos7 虚拟机删除mariadb-CSDN博客
     </a>
    </p>
    <h4>
     2. 删除mysql用户
    </h4>
    <pre><code class="hljs">userdel mysql</code></pre>
    <h4>
     3. 重新创建mysql用户和组
    </h4>
    <pre><code class="hljs">groupadd -g 1000 mysql

useradd mysql -g mysql -p mysql</code></pre>
    <h4>
     4. 上传mysql安装包到安装目录并解压
    </h4>
    <p>
     因为我用mobaxterm，直接在里面可以上传
    </p>
    <p>
     <img alt="" height="38" src="https://i-blog.csdnimg.cn/direct/1b49a47d25b744f496b2cbe53cf722ee.png" width="345"/>
    </p>
    <pre><code class="hljs">[root@localhost mysql]# tar -xvf mysql-8.4.0-1.el7.x86_64.rpm-bundle.tar
</code></pre>
    <p>
    </p>
    <h4>
     5. 安装
    </h4>
    <p>
     [root@localhost mysql]# rpm -ivh mysql-community-server-8.4.0-1.el7.x86_64.rpm
     <br/>
     执行上面的命令会报错，基本就是缺少依赖，那咱就挨个安装吧，反正下载的是bundle所有的以来都在里面，下面是安装顺序
    </p>
    <p>
     <img alt="" height="787" src="https://i-blog.csdnimg.cn/direct/5d58fe0855274707b201f66c9b762a27.png" width="1001"/>
    </p>
    <h4>
     6. 安装成功后查看MySQL服务状态
    </h4>
    <pre><code class="hljs">[root@localhost mysql]# systemctl status mysqld.service
● mysqld.service - MySQL Server
   Loaded: loaded (/usr/lib/systemd/system/mysqld.service; enabled; vendor preset: disabled)
   Active: inactive (dead)
     Docs: man:mysqld(8)
           http://dev.mysql.com/doc/refman/en/using-systemd.html
</code></pre>
    <h4>
     7. 启动mysql服务并再次查看服务状态
    </h4>
    <pre><code class="hljs">
[root@localhost mysql]# systemctl start mysqld.service
[root@localhost mysql]# systemctl status mysqld.service

   Loaded: loaded (/usr/lib/systemd/system/mysqld.service; enabled; vendor preset: disabled)

</code></pre>
    <h4>
     8. 获取临时密码
    </h4>
    <pre><code class="hljs">[root@localhost mysql]# grep "password" /var/log/mysqld.log
</code></pre>
    <h4>
     9. 连接数据库修改密码
    </h4>
    <p>
     mysql -uroot -p
    </p>
    <p>
     alter user 'root'@'localhost'  identified by 'your new password';
    </p>
    <p>
     如果没有生成默认密码或者生成的初始密码无法登录请参考
