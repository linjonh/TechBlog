    <p>
     <a id="_0">
     </a>
     <span style="color:#f3f3f4">
      <strong>
       <span style="background-color:#fe2c24">
        纯C语言实现，不涉及C++
       </span>
      </strong>
     </span>
    </p>
    <p>
     遍历是二叉树各种操作的基础，例如对于一棵给定二叉树求结点的双亲/求结点的孩子/求二叉树的高度/求叶结点个数/判断两棵二叉树是否相等……所有这些操作都是在二叉树遍历的过程中进行的。因此必须掌握二叉树的各种遍历过程，并能灵活用以解决各种问题。
    </p>
    <p>
     常见的遍历次序有：先序，中序，后序
    </p>
    <p>
     <span style="color:#4da8ee">
      -&gt;其中“序”是指根结点何时被访问。
     </span>
    </p>
    <p>
     先序：根结点-&gt;左子树-&gt;右子树
    </p>
    <p>
     中序: 左子树-&gt;根结点-&gt;右子树
    </p>
    <p>
     后序: 左子树-&gt;右子树-&gt;根结点
    </p>
    <p>
     对于递归遍历还是很好实现的，如
    </p>
    <h4>
     1. 递归先序遍历
    </h4>
    <pre><code class="language-cs">void preOrder(BiTNode* root) {
	if (root != NULL) {
		printf("%d ", root-&gt;data);// 访问当前节点（这里是打印节点的值）
		preOrder(root-&gt;lchild);
		preOrder(root-&gt;rchild);
	}
}</code></pre>
    <h4>
     <strong>
      2. 递归中序遍历
     </strong>
    </h4>
    <pre><code class="language-cs">void inOrder(BiTNode* root) {
	if (root != NULL)
	{
		inOrder(root-&gt;lchild);
		printf("%d ", root-&gt;data);
		inOrder(root-&gt;rchild);
	}
}</code></pre>
    <h4>
     3. 递归后序遍历
    </h4>
    <pre><code class="language-cs">void postOrder(BiTNode* root) {
	if (root != NULL)
	{
		postOrder(root-&gt;lchild);
		postOrder(root-&gt;rchild);
		printf("%d ", root-&gt;data);
	}
}</code></pre>
    <p>
     下面具体来说一下二叉树的非递归的先/中/后遍历：
    </p>
    <p>
     栈的
     <span style="color:#4da8ee">
      <strong>
       后进先出（LIFO）
      </strong>
     </span>
     特性使得它非常适合模拟递归调用过程中系统栈的行为。在二叉树的非递归遍历中，使用栈可以保存待访问的节点和控制节点的访问顺序，从而实现与递归遍历相同的效果。故：
     <span style="color:#4da8ee">
      使用栈来保存待访问的节点和控制节点的访问顺序。
     </span>
    </p>
    <p>
     下面结合代码模拟二叉树的非递归的先/中/后遍历的全过程：
    </p>
    <p>
     首先，需要先声明一下基本结构和栈的基本操作：
    </p>
    <p>
     typedef struct BiTNode {
     <span style="color:#f9eda6">
     </span>
     <span style="color:null">
      <span style="background-color:#f9eda6">
       //树结点
      </span>
     </span>
     <br/>
     ElemType data;
     <br/>
     struct BiTNode* lchild;
     <br/>
     struct BiTNode* rchild;
     <br/>
     }BiTNode;
    </p>
    <p>
     typedef struct stackNode {
     <span style="background-color:#f9eda6">
      //栈结点
     </span>
     <br/>
     struct BiTNode* pBiTNode;  // 指向二叉树节点的指针
     <br/>
     struct stackNode* next;  // 指向下一个栈节点的指针
     <br/>
     }stackNode;
    </p>
    <p>
     typedef struct stack {
     <span style="background-color:#f9eda6">
      //栈结构
     </span>
     <br/>
     struct stackNode* top;  //指向栈顶结点
     <br/>
     }stack;
    </p>
    <h3>
     1. 初始化栈
    </h3>
    <pre><code class="language-cs">void initStack(stack* s) {
    s-&gt;top = NULL;
}</code></pre>
    <h3>
     2. 判空操作
    </h3>
    <p>
     若栈顶指针为 NULL，则栈为空，返回 1；否则返回 0
    </p>
    <pre><code class="language-cs">int isEmpty(stack* s) {
    return s-&gt;top == NULL;
}</code></pre>
    <h3>
     3. 入栈
    </h3>
    <p>
     用于将一个二叉树节点压入栈中
    </p>
    <pre><code class="language-cs">void push(stack* s, BiTNode* node) {
	stackNode* newStackNode = (stackNode*)malloc(sizeof(stackNode));  
	if (newStackNode == NULL)
	{
		printf("内存分配失败!\n");
		return;
	}
	newStackNode-&gt;pBiTNode = node;  // 将新栈节点指向传入的二叉树节点
	newStackNode-&gt;next = s-&gt;top;  // 让新栈节点的 next 指针指向当前栈顶节点
	s-&gt;top = newStackNode;  // 更新栈顶指针为新栈节点
}</code></pre>
    <h3>
     4. 出栈
    </h3>
    <p>
     用于从栈中弹出一个二叉树节点,便于后面遍历操作中进行访问
    </p>
    <pre><code class="language-cs">BiTNode* pop(stack* s) {
	if (isEmpty(s))
	{
		printf("栈空，无元素可出栈!\n");
		return NULL;
	}
	stackNode* temp = s-&gt;top;   // 临时保存栈顶节点
	BiTNode* node = temp-&gt;pBiTNode; // 取出栈顶节点所指向的二叉树节点
	s-&gt;top = temp-&gt;next;  // 更新栈顶指针为原栈顶节点的下一个节点
	free(temp);   // 释放原栈顶节点的内存
	return node;
}
</code></pre>
    <h3>
     5.销栈
    </h3>
    <p>
     在遍历结束之后，栈中不再有未处理的结点，栈已经完成了它辅助遍历的过程，此时需要销毁栈结构以释放栈中所占用的内存
    </p>
    <pre><code class="language-cs">void destroyStack(stack* s) {
	while (!isEmpty(s)) {  // 当栈不为空时，不断弹出栈顶节点
		pop(s);
	}
}</code></pre>
    <p>
     好的，有了上面的基本辅助操作后，我们来开始正式对树进行操作
    </p>

}</code></pre>
    <h3>
     2. 销毁树结构
    </h3>
    <pre><code class="language-cs">void destroyTree(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	destroyTree(root-&gt;lchild);
	destroyTree(root-&gt;rchild);

    <h3>
     3. 非递归先序遍历
    </h3>
    <p>
     <span style="color:#f3f3f4">
      <strong>
       <span style="background-color:#fe2c24">
        具体思路
       </span>
      </strong>
     </span>
     ：
     <span style="color:#4da8ee">
      <strong>
       是先将根节点入栈，然后循环处理栈中的节点，每次弹出栈顶节点并访问，接着将其右子节点和左子节点依次入栈（注意顺序，因为栈是后进先出，所以先入右子结点）
      </strong>
     </span>
    </p>
    <pre><code class="language-cs">void preOrderTraversal(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	stack s;
	initStack(&amp;s);
	push(&amp;s, root);
	while (!isEmpty(&amp;s)) {
		BiTNode* p = pop(&amp;s);
		printf("%d ", p-&gt;data);
		if (p-&gt;rchild != NULL)
		{
			push(&amp;s, p-&gt;rchild);
		}
		if (p-&gt;lchild != NULL)
		{
			push(&amp;s, p-&gt;lchild);
		}
	}
	printf("\n");
	destroyStack(&amp;s);//遍历完记得销毁栈
}</code></pre>
    <h3>
     4. 非递归中序遍历
    </h3>
    <p>
     <span style="color:#f3f3f4">
      <strong>
       <span style="background-color:#fe2c24">
        具体思路
       </span>
      </strong>
     </span>
     ：
     <span style="color:#4da8ee">
      <strong>
       先将根节点及左子树的所有节点依次入栈，然后出栈访问节点，再让指针指向该节点的右子树，重复上述入栈、出栈访问、指向右子树的操作直至栈空且指针为空。
      </strong>
     </span>
    </p>
    <pre><code class="language-cs">void inOrderTraversal(BiTNode* root) {
	stack s;
	initStack(&amp;s);
	BiTNode* p = root;
	while (p != NULL || !isEmpty(&amp;s)) {
		while (p != NULL) {
			push(&amp;s, p);
			p = p-&gt;lchild;
		}
		p = pop(&amp;s);
		printf("%d ", p-&gt;data);
		p = p-&gt;rchild;
	}
	printf("\n");
	destroyStack(&amp;s);遍历完记得销毁栈
}</code></pre>
    <h3>
     5. 非递归后序遍历
    </h3>
    <p>
     相对复杂一些，需要使用两个栈来完成。
    </p>
    <p>
     <span style="color:#f3f3f4">
      <strong>
       <span style="background-color:#fe2c24">
        具体思路
       </span>
      </strong>
     </span>
     ：
     <span style="color:#4da8ee">
      <strong>
       先将根节点压入第一个栈，然后循环处理第一个栈，每次弹出栈顶节点并压入第二个栈，接着将其左子节点和右子节点依次压入第一个栈。最后，依次弹出第二个栈的节点并访问。
      </strong>
     </span>
    </p>
    <pre><code class="language-cs">void postOrderTraversal(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	stack s1, s2;
	initStack(&amp;s1);
	initStack(&amp;s2);

	push(&amp;s1, root);
	while (!isEmpty(&amp;s1)) {
		BiTNode* p = pop(&amp;s1);
		push(&amp;s2, p);
		if (p-&gt;lchild != NULL)
		{
			push(&amp;s1, p-&gt;lchild);
		}
		if (p-&gt;rchild != NULL)
		{
			push(&amp;s1, p-&gt;rchild);
		}
	}
	while (!isEmpty(&amp;s2)) {
		BiTNode* p = pop(&amp;s2);
		printf("%d ", p-&gt;data);
	}
	printf("\n");
	destroyStack(&amp;s1);//遍历完记得销毁栈
	destroyStack(&amp;s2);
}</code></pre>

    <h3>
     7. 完整代码
    </h3>
    <pre><code class="language-cs">#include&lt;stdio.h&gt;
#include&lt;stdlib.h&gt;
typedef int ElemType;

typedef struct BiTNode {  //树结点
	ElemType data;
	struct BiTNode* lchild;
	struct BiTNode* rchild;
}BiTNode;

typedef struct stackNode {  //栈结点
	struct BiTNode* pBiTNode;  // 指向二叉树节点的指针
	struct stackNode* next;  // 指向下一个栈节点的指针
}stackNode;

typedef struct stack {  //栈结构
	struct stackNode* top;  //指向栈顶结点
}stack;

//1.1 初始化栈
void initStack(stack* s) {
	s-&gt;top = NULL;
}

//2.1判空
// 该函数用于判断栈是否为空，若栈顶指针为 NULL，则栈为空，返回 1；
// 否则返回 0
int isEmpty(stack* s) {
	return s-&gt;top == NULL;
}

//3.1 入栈
 该函数用于将一个二叉树节点压入栈中
void push(stack* s, BiTNode* node) {
	stackNode* newStackNode = (stackNode*)malloc(sizeof(stackNode));  
	if (newStackNode == NULL)
	{
		printf("内存分配失败!\n");
		return;
	}
	newStackNode-&gt;pBiTNode = node;  // 将新栈节点指向传入的二叉树节点
	newStackNode-&gt;next = s-&gt;top;  // 让新栈节点的 next 指针指向当前栈顶节点
	s-&gt;top = newStackNode;  // 更新栈顶指针为新栈节点
}

//4.1 出栈，返回出栈的树结点
// 该函数用于从栈中弹出一个二叉树节点
BiTNode* pop(stack* s) {
	if (isEmpty(s))
	{
		printf("栈空，无元素可出栈!\n");
		return NULL;
	}
	stackNode* temp = s-&gt;top;   // 临时保存栈顶节点
	BiTNode* node = temp-&gt;pBiTNode; // 取出栈顶节点所指向的二叉树节点
	s-&gt;top = temp-&gt;next;  // 更新栈顶指针为原栈顶节点的下一个节点
	free(temp);   // 释放原栈顶节点的内存
	return node;
}

//5.1 销栈，在遍历结束之后，栈中不再有未处理的结点，栈已经完成了它辅助遍历的过程，此时需要销毁栈结构以释放栈中所占用的内存
void destroyStack(stack* s) {
	while (!isEmpty(s)) {  // 当栈不为空时，不断弹出栈顶节点
		pop(s);
	}
}

//1.1 创建新的树结点
BiTNode* createNewBiTNode(ElemType value) {
	BiTNode* newBiTNode = (BiTNode*)malloc(sizeof(BiTNode));
	if (newBiTNode == NULL)
	{
		printf("内存分配失败!\n");
		return NULL;
	}
	newBiTNode-&gt;data = value;
	newBiTNode-&gt;lchild = NULL;
	newBiTNode-&gt;rchild = NULL;
	return newBiTNode;
}

//2.1 销毁树结构
void destroyTree(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	destroyTree(root-&gt;lchild);
	destroyTree(root-&gt;rchild);
	free(root);
}

//3.1 递归先序遍历
void preOrder(BiTNode* root) {
	if (root != NULL) {
		printf("%d ", root-&gt;data);// 访问当前节点（这里是打印节点的值）
		preOrder(root-&gt;lchild);
		preOrder(root-&gt;rchild);
	}
}

//4.1 递归中序遍历
void inOrder(BiTNode* root) {
	if (root != NULL)
	{
		inOrder(root-&gt;lchild);
		printf("%d ", root-&gt;data);
		inOrder(root-&gt;rchild);
	}
}

//5.1 递归后序遍历
void postOrder(BiTNode* root) {
	if (root != NULL)
	{
		postOrder(root-&gt;lchild);
		postOrder(root-&gt;rchild);
		printf("%d ", root-&gt;data);
	}
}

//6.1 非递归先序遍历
//先序遍历的顺序是：根节点-&gt;左子树-&gt;右子树。
// 非递归实现可以借助栈来模拟递归调用的过程。
// 具体思路是先将根节点入栈，然后循环处理栈中的节点，每次弹出栈顶节点并访问，接着将其右子节点和左子节点依次入栈
// （注意顺序，因为栈是后进先出，所以先入右子节点）。
void preOrderTraversal(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	stack s;
	initStack(&amp;s);
	push(&amp;s, root);
	while (!isEmpty(&amp;s)) {
		BiTNode* p = pop(&amp;s);
		printf("%d ", p-&gt;data);
		if (p-&gt;rchild != NULL)
		{
			push(&amp;s, p-&gt;rchild);
		}
		if (p-&gt;lchild != NULL)
		{
			push(&amp;s, p-&gt;lchild);
		}
	}
	printf("\n");
	destroyStack(&amp;s);//遍历完记得销毁栈
}

//7.1 非递归中序遍历
//非递归中序遍历的核心思路是利用栈来模拟递归调用的过程，
// 先将根节点及左子树的所有节点依次入栈，然后出栈访问节点，再让指针指向该节点的右子树，重复上述入栈、出栈访问、指向右子树的操作直至栈空且指针为空。
void inOrderTraversal(BiTNode* root) {
	stack s;
	initStack(&amp;s);
	BiTNode* p = root;
	while (p != NULL || !isEmpty(&amp;s)) {
		while (p != NULL) {
			push(&amp;s, p);
			p = p-&gt;lchild;
		}
		p = pop(&amp;s);
		printf("%d ", p-&gt;data);
		p = p-&gt;rchild;
	}
	printf("\n");
	destroyStack(&amp;s);遍历完记得销毁栈
}

//8.1 非递归后序遍历
//后序遍历的顺序是：左子树 -&gt; 右子树 -&gt; 根节点。
// 非递归实现相对复杂一些，需要使用两个栈来完成。
// 具体思路是先将根节点压入第一个栈，然后循环处理第一个栈，每次弹出栈顶节点并压入第二个栈，接着将其左子节点和右子节点依次压入第一个栈。
// 最后，依次弹出第二个栈的节点并访问。
void postOrderTraversal(BiTNode* root) {
	if (root == NULL)
	{
		return;
	}
	stack s1, s2;
	initStack(&amp;s1);
	initStack(&amp;s2);

	push(&amp;s1, root);
	while (!isEmpty(&amp;s1)) {
		BiTNode* p = pop(&amp;s1);
		push(&amp;s2, p);
		if (p-&gt;lchild != NULL)
		{
			push(&amp;s1, p-&gt;lchild);
		}
		if (p-&gt;rchild != NULL)
		{
			push(&amp;s1, p-&gt;rchild);
		}
	}
	while (!isEmpty(&amp;s2)) {
		BiTNode* p = pop(&amp;s2);
		printf("%d ", p-&gt;data);
	}
	printf("\n");
	destroyStack(&amp;s1);//遍历完记得销毁栈
	destroyStack(&amp;s2);
}

	root-&gt;lchild = createNewBiTNode(2);
	root-&gt;rchild = createNewBiTNode(3);
	root-&gt;lchild-&gt;lchild = createNewBiTNode(4);
	root-&gt;lchild-&gt;rchild = createNewBiTNode(5);
	root-&gt;rchild-&gt;lchild = createNewBiTNode(6);

	printf("递归先序遍历的结果：");
	preOrder(root);
	printf("\n");

	printf("递归中序遍历的结果：");
	inOrder(root);
	printf("\n");

	printf("递归后序遍历的结果：");
	postOrder(root);
	printf("\n\n");

	printf("非递归先序遍历的结果：");
	preOrderTraversal(root);

	printf("非递归中序遍历的结果：");
	inOrderTraversal(root);

	printf("非递归后序遍历的结果：");
	postOrderTraversal(root);
