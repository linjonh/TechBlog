     验证
    </h3>
    <p>
     之前的版本有的需要自己手动更新环境变量，但这个版本已经不需要了。
     <br/>
     直接
     <code>
      win+R
     </code>
     打开命令行窗口输入
     <code>
      java
     </code>
     或者
     <code>
      java -version
     </code>
     检测，当出现如下结果时，已OK。
