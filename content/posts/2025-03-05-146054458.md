    <blockquote>
     <p>
      记录一种安装原版系统，并在用户使用前安装好所有驱动，以及配置好网络和工作环境，但不影响用户全新体验的方法。
     </p>
    </blockquote>
    <h2>
     <a id="_1">
     </a>
     前言
    </h2>
    <p>
     有的时候我们可能会遇到这样一种情况：
    </p>
    <blockquote>
     <p>
      需要给别人安装一个微软官网正版的全新系统，用户开机要有OOBE全新体验，但是要求可以正常使用WiFi，用户创建好账号进入桌面时，检查设备管理器，所有驱动都要是正常的，此外可能还要有所需的工作环境，比如python3，PS，CAD，立创EDA等。
     </p>
    </blockquote>
    <p>
     可是真实情况是：
    </p>
    <blockquote>
     <p>
      微软官网的正版系统，采用ISO+U盘的方式安装，安装后WiFi不一定可以驱动，此外设备管理器驱动也是不齐全的，更没有第三方软件。
     </p>
    </blockquote>
    <p>
     似乎能满足需求的只有打好驱动，安装好第三方软件的ghost系统，可这种方式并不通用，不同主板驱动不同，不能随意套用，此外这样不满足使用微软官网正版系统的前提条件，更不会有OOBE全新体验。
    </p>
    <h2>
     <a id="_9">
     </a>
     最佳方式
    </h2>
    <p>
     所以最佳的方式就是：
     <b>
      在使用U盘安装好微软官方ISO系统后，配置OOBE前，提前使用admin账号进入系统，配置好一切，再还原出OOBE界面
     </b>
     。能实现对方开箱即用。
    </p>
    <h2>
     <a id="_12">
     </a>
     关键词说明及快捷键介绍
    </h2>
    <p>
     更多可参考我在B站发布的专栏[1]：
     <a href="https://www.bilibili.com/opus/776549672398356488" rel="nofollow">
      完美解决华硕Windows无法完成安装，并找回预装系统
     </a>
    </p>
    <ul>
     <li>
      1）OOBE：新机一打开会弹出的设置向导
     </li>
     <li>
      2）OOBE界面跳转体验模式：也可以说是审核模式，快捷键是
      <b>
       ctrl+shift+f3
      </b>
      ，可预安装驱动，操作完重启，注意
      <b>
       保持默认不勾选
      </b>
      ；
     </li>
     <li>
      3）Windows登陆界面打开终端：
      <b>
       shift+f10
      </b>
      。
     </li>
     <li>
      4）Windows恢复环境WinRE：f12
     </li>
     <li>
      5）审核模式：超级管理员administrator专享模式，可以定制OEM系统，并可以反复进入。
      <b>
       一种进入途径是oobe界面按ctrl+shift+f3，然后电脑会重启自动进入审核模式（如看见音量条，那就先按fn+esc，再重复以上按键）
      </b>
      。此外，
      <b>
       退出的方法就是执行系统打包工具sysprep，千万不要勾选通用！然后会回到oobe
      </b>
      ；
      <b>
       另一种退出方式就是连续重启两次
      </b>
      ，第一次会看见defaultuser0和administrator，都是不可访问的，点右下角电源选重启，即可回到oobe。
     </li>
     <li>
      6）系统打包工具sysprep：一个古老的内置工具，从win7到win11一直都是3.14版本，用来在审核模式下对修改的OEM系统进行打包，并配置sysprep部署计划。注意，非必要
      <b>
       禁止勾选通用！
      </b>
     </li>
     <li>
      7）sysprep工具的“通用”选项：指的是运行sysprep会看见的一个框框，用来决定打包模式。打勾表示sysprep部署时扫描识别硬件并安装驱动，反之就是跳过识别硬件环节，直接安装驱动。
      <b>
       默认就是不打勾。所以非必要不可以勾选通用！
      </b>
      你不能包装驱动可以完美识别，一旦失败，千万不要着急重启，应先修复系统再重启！
     </li>
    </ul>
    <h2>
     <a id="_22">
     </a>
     具体操作流程
    </h2>
    <p>
     刚刚介绍完快捷键，现在回到主题，简单说需求就是
     <b>
      给用户安装开箱即用的正版系统（驱动环境齐全），用户要能够自行根据oobe创建账户，要有正版系统该有的体验
     </b>
     ，而不是像盗版系统那样，开机就会进入admin账户桌面。
    </p>
    <h3>
     <a id="1_Windows_24">
     </a>
     1. 安装正版Windows系统
    </h3>
    <h4>
     <a id="11_Windows_10_25">
     </a>
     1.1. 安装Windows 10
    </h4>
    <p>
     访问[2]
     <a href="https://www.microsoft.com/zh-cn/software-download/windows10" rel="nofollow">
      下载Windows 10
     </a>
     ，这是微软官网唯一获取最新版Windows 10系统镜像（ISO文件）的地方，是免费下载的。
     <br/>
     在
     <b>
      创建 Windows 10 安装媒体
     </b>
     那点
     <b>
      立刻下载
     </b>
     ，会下载一个介质创建工具到电脑上。之后将要安装系统的电脑插入此U盘，重启后按esc/f11进入快速启动菜单，选择此U盘，便可执行格式化磁盘，以及安装系统操作。
    </p>
    <ul>
     <li>
      方式1-使用 ISO U盘：插入一个容量不小于8GB的空U盘，然后运行这个介质创建工具，选择U盘，然后选和电脑匹配的版本（家庭版/专业版/工作站版，错了不能激活），然后便会自动下载镜像并刻录到U盘里。
     </li>
     <li>
      方式2-使用 PE U盘：和方式1相比，这种方式不需要空U盘，但需要准备一个 PE U盘。运行介质创建工具，然后选为别的电脑创建ISO文件，选好系统版本，便会自动下载ISO文件，完成后可将ISO文件保存到电脑中。之后便可使用PE工具（推荐微PE，为数不多可安装纯净系统的免费工具，详见：
      <a href="https://www.wepe.com.cn/" rel="nofollow">
       微PE工具箱
      </a>
      ），使用PE系统内置系统安装工具将此ISO镜像文件到目标电脑。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/b5f714e724524005ae26f0f5dd521133.png"/>
     </li>
    </ul>
    <h4>
     <a id="12_Windows_11_31">
     </a>
     1.2. 安装Windows 11
    </h4>
    <p>
     访问[3]
     <a href="https://www.microsoft.com/zh-cn/software-download/windows11" rel="nofollow">
      下载Windows 11
     </a>
     ，这是微软官网唯一获取最新版Windows 11系统镜像（ISO文件）的地方，是免费下载的。
     <br/>
     与Windows 10不同，Windows 11有两种方式获取ISO：
    </p>
    <ul>
     <li>
      一种是
      <b>
       创建 Windows 11 安装媒体
      </b>
      ，点
      <b>
       立刻下载
      </b>
      ，然后在本地运行介质创建工具创建ISO系统U盘或生成ISO系统文件，方式和前面Windows 10类似。
     </li>
     <li>
      另一种是点击
      <b>
       下载适用于 x64 设备的 Windows 11 磁盘映像
      </b>
      ，选好版本，下载来的便是ISO文件。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/e4ced532a0e943958f3caff007dadb3c.png">
       <br/>
       安装Windows 10 / 11的具体过程就不演示了。进入下一步，安装好系统后，进入审核模式，执行系统打包操作。
      </img>
     </li>
    </ul>
    <h3>
     <a id="2__38">
     </a>
     2. 进入审核模式
    </h3>
    <p>
     在上一步假设你安装好Windows 10 / 11系统了，他一定会自动重启，然后来到一个oobe欢迎页面，如下图所示，会让你联网创建/登录账户之类的。注意由于是给别的用户装系统，这里就和我们没关系了，是后续他自己的体验。
     <br/>
     在此
     <b>
      我们需要跳过这个oobe界面，提前进入系统，配置好所需的环境
     </b>
     。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/32a2ca160e764cc0a7089d70c5deae12.png">
      <br/>
      如前面快捷键介绍所说，来到oobe界面，按
      <b>
       ctrl+shift+f3
      </b>
      便可进入审核模式，审核模式会临时启用超级用户administrator，类似于Linux里面的root用户。注意
      <u>
       在审核模式里产生的的第三方文件、使用记录，在退出该模式后都会被自动抹除
      </u>
      。但是有几点例外，在审核模式里，
      <u>
       会保留安装的驱动文件、安装的任何软件，以及WiFi设置
      </u>
      。
      <br/>
      所以审核模式是我们配置用户电脑的绝佳环境，既不影响未来的用户体验，又能无痕配置好电脑。
     </img>
    </p>
    <h4>
     <a id="_314_44">
     </a>
     系统准备工具 3.14
    </h4>
    <p>
     进入审核模式后，会自动弹出系统准备工具（Sysprep），他会对电脑进行准备以便进行硬件独立和清理。
     <br/>
     说人话就是，
     <u>
      我们在审核模式修改系统，系统准备工具负责清理痕迹并打包系统，为新用户做准备
     </u>
     。
    </p>
    <ul>
     <li>
      系统清理操作默认是：进入系统全新体验(OOBE)。
     </li>
     <li>
      注意通用默认是不勾选的，也不建议勾选，否则系统可能会无法启动。
     </li>
     <li>
      关机选项默认是：重新启动，可选的是关机。
      <br/>
      <b>
       在审核模式下，我们先最小化这个系统准备工具（注意不要关闭他），然后执行安装驱动，配置网络，以及安装第三方软件等操作
      </b>
      。
      <br/>
      最小化系统准备工具的方式就是显示桌面，快捷键是
      <b>
       Windows + M
      </b>
      ，或者鼠标点屏幕最右下角的缝也会显示桌面。
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cc2dc1e81c8b4b209c29424f571d6738.png"/>
     </li>
    </ul>
    <h3>
     <a id="3__53">
     </a>
     3. 配置系统并退出审核模式
    </h3>
    <p>
     我们在审核模式下可以使用驱动人生、360驱动大师、驱动精灵等软件安装驱动，安装驱动过程中可能会提示需要重启，注意尽量不要使用驱动安装工具内置的重启方式，
     <b>
      要等到所有驱动安装完，再使用前面说到的系统准备工具来执行重启
     </b>
     。因为绕过系统准备工具强制重启，可能会损坏系统引起不开机。
     <br/>
     当安装好驱动，配置好工作环境，方可使用系统准备工具执行重启。系统准备工具保持默认参数即可（清理操作是：进入OOBE；不勾选通用；关机选项是重启）。
     <br/>
     重启后会来到OOBE界面，还是老方法用快捷键进入审核模式，检查驱动是否安装完整，确认无误，则将系统准备工具的关机选项改为
     <b>
      关机
     </b>
     ，执行后系统就会正常关机，下次用户开机就会直接进入OOBE全新体验界面，他创建好账户进入桌面，就是配置了需要工作环境的正版系统。
    </p>
    <h3>
     <a id="4__57">
     </a>
     4. 意外情况
    </h3>
    <p>
     假如进入审核模式不小心关闭了系统准备工具，切记不要直接重启，可能会导致系统不能启动。
     <br/>
     正确的操作是使用终端CMD调出系统准备工具，详细可参考：
     <a href="https://www.bilibili.com/opus/776549672398356488" rel="nofollow">
      完美解决华硕Windows无法完成安装，并找回预装系统
     </a>
     <br/>
     调出系统准备工具的具体操作是：
     <br/>
     CMD切换到System32/Sysprep目录，再输入sysprep.exe
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0b113a81de2946a28325a7fc28a6f94c.png"/>
    </p>
