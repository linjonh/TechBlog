     <p style="margin:10px auto">
      用过zsh的同学肯定对其自动补全功能印象深刻，通过简单的定制python交互式命令行也能实现类似功能，具体操作如下：
     </p>
     <ul style="margin-left:45px; padding-left:0px">
      <li style="list-style:inherit!important">
       在用户目录下新建".pythonstartup"文件，写入以下内容：
      </li>
     </ul>
     <div>
      <pre><code class="language-plain"># python startup file
import readline
import rlcompleter
import atexit
import os
#tab completion
readline.parse_and_bind('tab: complete')
#history file
historyfile = os.path.join(os.environ['HOME'],'.pythonstartup')
try:
    readline.read_history_file(historyfile)
except:
    pass
atexit.register(readline.write_history_file,historyfile)
del os,historyfile,readline,rlcompleter</code></pre>
      <br/>
      <br/>
     </div>
     <ul style="margin-left:45px; padding-left:0px">
      <li style="list-style:inherit!important">
       增加环境变量，编辑.bashrc或.zshrc文件（根据你的shell确定），加入以下内容：
      </li>
     </ul>
     <div class="codehilite">
      <div class="cnblogs_code" style="border:1px solid rgb(204,204,204); padding:5px; overflow:auto; margin:5px 0px">
       <pre style="margin-top:0px; margin-bottom:0px; word-wrap:break-word">export PYTHONSTARTUP="/home/ma6174/.pythonstartup"</pre>
