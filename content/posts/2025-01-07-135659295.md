# 推荐5款超实用的开源项目

**GitHub圈选**
项目推荐：

* **FreeControl（PC端远程手机控制工具）**
* **Cronicle（任务调度平台）**
* **PageSpy（远程WEB调试工具）**
* **Sqlmap（SQL注入工具）**
* **Upscayl（AI图像升级器）**

#### 1、FreeControl

`FreeControl`
是一款开源的手机远程控制工具，用户可以高效的进行远程的手机管理，支持用户在线进行分辨率、帧数等参数的修改和设置，支持有线和无线的两种连接方式，是用户远程控制手机不可缺少的一款软件。

该项目是基于
`scrcpy`
、采用 C# 编写的控制 Android 设备的 PC 桌面工具，提供了更加简洁的交互界面。

Star：
`898`

项目地址：https://github.com/pdone/FreeControl

![](https://i-blog.csdnimg.cn/blog_migrate/811e9a913a60677b21c413404523e4e9.png)

##### 特色功能：

* 实时显示Android设备显示屏
* Android设备的实时鼠标和键盘操纵
* 支持屏幕录像
* 支持截屏到png
* 支持无线连接
* 支持高达16个设备连接
* 支持全屏显示和在顶端显示
* 安装apk：将apk拖放到视频对话框中实现安装
* 传输文件：将文档拖到视频对话框以将文档发送至Android设备
* 后台录像：仅录影，无显示页面

![](https://i-blog.csdnimg.cn/blog_migrate/dfdb1bb85c34f9a420edaaefced29d4d.gif)

#### 2、Cronicle

`Cronicle`
是一款简单的任务调度和运行平台。该项目是用 Node.js 写的 cron 替代品，它开箱即用、自带 Web 界面、无需数据库，提供了执行 shell 命令、实时统计、自动故障转移、自动重试、多时区等功能。

Star：
`2.9k`

项目地址：https://github.com/jhuckaby/Cronicle

![](https://i-blog.csdnimg.cn/blog_migrate/182b1095896366f4c0d99b77b0bbe736.png)

#### 3、PageSpy

`PageSpy`
是一款用来调试远程 Web 项目的工具。

基于对原生 API 的封装，它将调用原生方法时的参数进行过滤、转化，整理成格式规范的消息供调试端消费；调试端收到消息数据，提供类控制台可交互式的功能界面将数据呈现出来。

Star：
`2.5k`

项目地址：https://github.com/HuolalaTech/page-spy-web

![](https://i-blog.csdnimg.cn/blog_migrate/b330bda87f317ae310839134952edbee.png)

#### 4、Sqlmap

`Sqlmap`
是一款强大的 SQL 注入工具。是一个 Python 写的渗透测试工具，可以自动检测和利用 SQL 注入漏洞，获得数据库服务器的权限。

它提供了强大的检测引擎和多种特性，包括识别数据库类型和版本、枚举用户、提权、获取数据等。

Star：
`29.3k`

项目地址：https://github.com/sqlmapproject/sqlmap

![](https://i-blog.csdnimg.cn/blog_migrate/620703ffaae1238fd514c14a44c87cb2.png)

#### 5、Upscayl

`Upscayl`
是一款免费开源的 AI 图像升级器，使用先进的人工智能算法来放大和增强低分辨率图像，而不损失质量。

该应用程序基于 Linux 优先理念，是一个跨平台应用程序，支持在所有主要桌面操作系统上使用。

Star：
`20.4k`

项目地址：https://github.com/upscayl/upscayl

![](https://i-blog.csdnimg.cn/blog_migrate/aebe0718bb94b4d759607d33cc73629d.png)
