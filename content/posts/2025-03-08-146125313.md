    <h2>
     <a id="MySQL_0">

     MySQL第一次作业
    </h2>
    <h5>
     <a id="_2">
     </a>
     要求：
    </h5>
    <p>
     新建产品库mydb6_product，新建4张表如下：
     <br/>
     employees表
     <br/>
     列1：id，整型，主健
     <br/>
     列2：name，字符串，最大长度50，不能为空
     <br/>
     列3:age， 整型
     <br/>
     列4：gender，字符串，最大长度10，不能为空，默认值“unknown"
     <br/>
     ：列5：salary, 浮点型
    </p>
    <p>
     orders表
     <br/>
     列1：id，整型，主键
     <br/>
     列2：name，字符串，最大长度100，不能为空
     <br/>
     列3:price，浮点型
     <br/>
     列4:quantity，整型
     <br/>
     列5:category, 字符串，最大长度50
    </p>
    <p>
     invoices表
     <br/>
     列1：number，整型，主键自增长
     <br/>
     列2：order_id，整型，外键关联到orders表的id列
     <br/>
     列3：in_date:日期型
     <br/>
     列4：total_amount：浮点型，要求数据大于0
    </p>
