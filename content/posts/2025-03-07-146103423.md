    <h3>
     <a id="C_std_1">
     </a>
     一、C++标准库——命名空间 std
    </h3>
    <p>
     C++标准库std是一个命名空间，全称为"standard"，其中包括标准模板库（STL），输入输出系统，文件系统库，智能指针与内存管理，字符串与文本处理等等，我们以输入输出流和标准模板库为例，简单地认识一下C++标准库里面有哪些内容：
    </p>
    <h4>
     <a id="1_3">
     </a>
     1.输入输出系统
    </h4>
    <h5>
     <a id="1Streams_4">
     </a>
     （1）流（Streams）
    </h5>
    <ul>
     <li>
      基础流：
      <code>
       iostream
      </code>
      （
      <code>
       cin
      </code>
      ,
      <code>
       cout
      </code>
      ,
      <code>
       cerr
      </code>
      ）
     </li>
     <li>
      文件流：
      <code>
       fstream
      </code>
      ,
      <code>
       ifstream
      </code>
      ,
      <code>
       ofstream
      </code>
     </li>
     <li>
      字符串流：
      <code>
       stringstream
      </code>
      ，用于内存数据格式化
     </li>
    </ul>
    <h5>
     <a id="2_9">
     </a>
     （2）格式化工具
    </h5>
    <ul>
     <li>
      <code>
       std::format
      </code>
      （C++20）：类型安全字符串格式化
     </li>
     <li>
      本地化支持：
      <code>
       locale
      </code>
      与
      <code>
       wcout
      </code>
      （宽字符输出）
     </li>
    </ul>
    <h4>
     <a id="2STL_12">

     2.标准模板库（STL）
    </h4>
    <h5>
     <a id="1_Containers_13">
     </a>
     （1） 容器（Containers）
    </h5>
    <ul>
     <li>
      序列容器：
      <code>
       vector
      </code>
      ,
      <code>
       list
      </code>
      ,
      <code>
       deque
      </code>
      ,
      <code>
       array
      </code>
      （C++11）
     </li>
     <li>
      关联容器：
      <code>
       map
      </code>
      ,
      <code>
       set
      </code>
      ,
      <code>
       unordered_map
      </code>
      （C++11）,
      <code>
       unordered_set
      </code>
      （C++11）
     </li>
     <li>
      适配器：
      <code>
       stack
      </code>
      ,
      <code>
       queue
      </code>
      ,
      <code>
       priority_queue
      </code>
     </li>
    </ul>
    <h5>
     <a id="2_Algorithms_17">
     </a>
     （2） 算法（Algorithms）
    </h5>
    <ul>
     <li>
      非修改序列操作：
      <code>
       find
      </code>
      ,
      <code>
       count
      </code>
      ,
      <code>
       for_each
      </code>
     </li>
     <li>
      修改序列操作：
      <code>
       copy
      </code>
      ,
      <code>
       transform
      </code>
      ,
      <code>
       replace
      </code>
     </li>
     <li>
      排序与搜索：
      <code>
       sort
      </code>
      ,
      <code>
       binary_search
      </code>
      ,
      <code>
       lower_bound
      </code>
     </li>
    </ul>
    <h5>
     <a id="3_Iterators_21">
     </a>
     （3） 迭代器（Iterators）
    </h5>
    <ul>
     <li>
      五类迭代器：输入/输出、前向/双向/随机访问
     </li>
     <li>
      范围遍历：C++20引入
      <code>
       ranges::
      </code>
      简化迭代操作
      <br/>
      在本期内容中，我们只需要认识一下输入输出流中iostream的cin,cout，完成一些简单的输入输出操作即可。
     </li>
    </ul>

    <h4>
     <a id="2coutcin_38">
     </a>
     2.cout与cin
    </h4>
    <h5>
     <a id="1cout_39">
     </a>
     （1）cout——标准输出流
    </h5>
    <ul>
     <li>
      基础输出语法：
      <pre><code class="prism language-cpp">cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Hello World"</span><span class="token punctuation">;</span>            <span class="token comment">// 输出字符串  </span>
cout <span class="token operator">&lt;&lt;</span> <span class="token number">10</span> <span class="token operator">&lt;&lt;</span> <span class="token string">" is the answer"</span><span class="token punctuation">;</span>   <span class="token comment">// 链式输出（拼接不同类型）  </span>
</code></pre>
     </li>
     <li>
      数据类型支持
      <br/>
      <code>
       cout
      </code>
      通过运算符重载支持所有内置类型：
     </li>
    </ul>
    <pre><code class="prism language-cpp"><span class="token keyword">int</span> num <span class="token operator">=</span> <span class="token number">2025</span><span class="token punctuation">;</span>  
<span class="token keyword">double</span> pi <span class="token operator">=</span> <span class="token number">3.1415926</span><span class="token punctuation">;</span>  
<span class="token keyword">bool</span> flag <span class="token operator">=</span> <span class="token boolean">true</span><span class="token punctuation">;</span>  
 
cout <span class="token operator">&lt;&lt;</span> num <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>    <span class="token comment">// 输出：2025  </span>
cout <span class="token operator">&lt;&lt;</span> pi <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>     <span class="token comment">// 输出：3.14159（默认精度6位）  </span>
cout <span class="token operator">&lt;&lt;</span> flag <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>   <span class="token comment">// 输出：1（true为1，false为0）  </span>
</code></pre>
    <p>
     在这里，我们就不需要像printf那样使用%d,%f,%lf等来输出对应类型的值了，因为cout可以自行判断。当然，如果需要输出小数点后指定多少位，或者一个数字占多少位，这里就不是那么简单了，在这种情况下，还是使用printf输出比较方便，因为C++是兼容C的。
    </p>
    <h5>
     <a id="2cin_58">
     </a>
     （2）cin——标准输入流
    </h5>
    <ul>
     <li>
      头文件：
      <code>
       #include &lt;iostream&gt;
      </code>
     </li>
     <li>
      本质：
      <code>
       cin
      </code>
      是
      <code>
       std::istream
      </code>
      类的全局实例，绑定标准输入设备（通常为键盘）
     </li>
     <li>
      缓冲机制：输入内容暂存于内存缓冲区，按需提取至变量
     </li>
     <li>
      基础数据读取
     </li>
    </ul>
    <pre><code class="prism language-cpp"><span class="token keyword">int</span> age<span class="token punctuation">;</span>  
<span class="token keyword">double</span> salary<span class="token punctuation">;</span>  
cin <span class="token operator">&gt;&gt;</span> age <span class="token operator">&gt;&gt;</span> salary<span class="token punctuation">;</span>  <span class="token comment">// 输入"30 85000.5"后，age=30, salary=85000.5  </span>
</code></pre>
    <ul>
     <li>
      自动类型转换：将输入字符序列转换为目标类型数据；
     </li>
     <li>
      空格分隔：默认跳过前导空白，以空格/换行为数据分隔符；
     </li>
     <li>
      链式操作：支持连续提取多个变量。
     </li>
    </ul>
    <h6>
     <a id="cinCscanf_71">
     </a>
     总而言之，cin就相当于C语言中的scanf。
    </h6>
    <h5>
     <a id="3endl_72">
     </a>
     （3）endl——流操纵器
    </h5>
    <ul>
     <li>
      <code>
       endl
      </code>
      是C++标准库中定义的流操纵器（Manipulator），其行为包含两个不可分割的原子操作：
     </li>
     <li>
      <ol>
       <li>
        换行输出：向输出流插入换行符
        <code>
         \n
        </code>
        ，使后续输出从新行开始；
       </li>
      </ol>
     </li>
     <li>
      <ol start="2">
       <li>
        强制刷新：调用
        <code>
         flush()
        </code>
        方法，立即将流缓冲区内容写入目标设备（如控制台、文件、网络套接字）。
       </li>
      </ol>
     </li>
    </ul>
    <h6>
     <a id="endln_76">
     </a>
     简单来讲，先将
     <code>
      endl
     </code>
     理解为
     <code>
      '\n'
     </code>
     即可。
    </h6>
    <h4>
     <a id="3_____77">
     </a>
     3.
     <mark>
      &lt;&lt;
     </mark>
     流插入与
     <mark>
      &gt;&gt;
     </mark>
     流提取
    </h4>
    <h5>
     <a id="_78">
     </a>
     流插入运算符
     <code>
      &lt;&lt;
     </code>
    </h5>
    <ul>
     <li>
      流插入运算符
      <code>
       &lt;&lt;
      </code>
      是C++中输出流操作的核心工具，其本质是
      <code>
       ostream
      </code>
      类中重载的运算符函数。通过该运算符，数据从内存对象流向输出设备（如控制台、文件、网络套接字）。
     </li>
     <li>
      什么是
      <code>
       控制台
      </code>
      ？黑框框就是：
      <br/>
      <img alt="示例5" src="https://i-blog.csdnimg.cn/direct/0261df876f6a4c93a327d2d6cb24f5b2.png#pic_center"/>
     </li>
    </ul>
    <h5>
     <a id="_82">
     </a>
     流提取运算符
     <code>
      &gt;&gt;
     </code>
    </h5>
    <ul>
     <li>
      流提取运算符
      <code>
       &gt;&gt;
      </code>
      是C++输入流（
      <code>
       istream
      </code>
      ）的核心操作符，用于从输入源（如键盘、文件、网络）提取数据到变量。其本质是
      <code>
       istream
      </code>
      类中重载的运算符函数，遵循链式调用规则。
      <br/>
      如图所示：
      <img alt="示例6" src="https://i-blog.csdnimg.cn/direct/769b370c67c542f1911a27ed0606b5e7.png#pic_center">
       <br/>
       这里
       <code>
        &gt;&gt;
       </code>
       就是将键盘上输入的10提取出来，赋给变量a。
      </img>
     </li>
    </ul>
    <h3>
     <a id="_86">
     </a>
     三、缺省参数
    </h3>
    <h4>
     <a id="1_87">
     </a>
     1.缺省参数的概念
    </h4>
    <p>
     缺省参数是声明或定义函数时为函数的参数指定一个缺省值。在调用该函数时，如果没有指定实参则采用该形参的缺省值，否则使用指定的实参，举个简单的例子：
     <br/>
     <img alt="示例7" src="https://i-blog.csdnimg.cn/direct/9971a34c40e6424fa6866ec9a9967f03.png#pic_center">
      <br/>
      如图所示，Fun函数中提供了一个缺省参数a=1，也就是说，当我们没有传参的时候，它就会默认调用a=1，输出结果为1；如果传参了，比如这里传了一个10，那么就传什么输出什么。
     </img>
    </p>
    <h4>
     <a id="2_91">
     </a>
     2.缺省参数的分类
    </h4>
    <ul>
     <li>
      全缺省参数
      <br/>
      如图所示：
      <br/>
      <img alt="示例8" src="https://i-blog.csdnimg.cn/direct/73b8e98aaaad49bb9f054aaeee57a733.png#pic_center">
       <br/>
       这里在函数定义中，三个参数都是缺省参数，还是这样的原则：没传参就使用缺省参数，如果传参那么就传什么输出什么，并且，这里是有顺序的，Fun(20)就是先把20传给a，b和c使用缺省参数；Fun(20,30)就是先传给a,b，c依然使用缺省参数。
      </img>
     </li>

      半缺省参数
