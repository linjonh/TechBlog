    <h4>
     1.10进制转换成x进制
    </h4>
    <p>
     <img alt="" height="426" src="https://i-blog.csdnimg.cn/direct/e6e1d5f4f1454170b4c50b34031f3711.png" width="526"/>
    </p>
    <p>
     这种题，我们一般都用递归做
    </p>
    <p>
     比如说对于5来说，它的二进制序列应该是101 我们可以把它拆成10和1，5%2应该是它的最后一位，所以我们就把5%2 然后再把5/2 也就是2拆开是10 合起来就是10+1了；
    </p>
    <p>
     递归伪代码
    </p>
    <pre><code class="hljs">n_to_x(int n,int x)
{
    if(n&gt;=x) dfs(n/x,x);
    cout&lt;&lt;n%x;
}
</code></pre>
    <p>
     我们可以把它当成一个模板记下来
    </p>
    <p>
     如果是10进制转化为2到9进制是可以这样的，但是一旦，比如说16进制，16进制的10 11 12 13 14 15是用ABCDEF来表示的，这时候我们可以用一个字符串字符0放在0下标，字符1放在1下标 字符A放在10下标，字符B放在11下标，然后我们根据下标打印字符就行了
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
int n,x;
string s = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
void dfs(int n,int x)
{
	if(n&gt;=x)
	dfs(n/x,x);
	
	cout &lt;&lt; s[n%x];
}
int main()
{
	cin &gt;&gt; n &gt;&gt; x;
	dfs(n,x);
}</code></pre>
    <p>
     如上述代码所示，这道题我们已经完成了
    </p>
    <h4>
     x进制转换为10进制
    </h4>
    <p>
     <img alt="" height="1093" src="https://i-blog.csdnimg.cn/direct/57745a2dd02b4b2dac465ff8d5ab8705.png" width="1228"/>
    </p>
    <p>
     关于如何把x进制转换为10进制，我们呢是有两种方法滴，
    </p>
    <p>
     法一：用循环从x进制最低位开始遍历，乘以权值
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;cmath&gt;
using namespace std;
int x;
string s;
int main()
{
	cin &gt;&gt; x &gt;&gt; s;
	
	int n = s.size();
	int j = 0;
	int ret = 0;
	for(int i = n-1;i&gt;=0;i--)
	{
		if(s[i]&lt;='9' &amp;&amp; s[i]&gt;='0')
		{
			ret+=(s[i] - '0')*pow(x,j);
		}
		else
		{
			ret+=(s[i]-'A'+10)*pow(x,j);
		}
		j++;
	}
	
	cout &lt;&lt; ret &lt;&lt; endl; 
	
	
	return 0;
}</code></pre>
    <p>
     法二，直接用我们的库函数
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
int x;string s;

int main()
{
	cin &gt;&gt; x &gt;&gt; s;
	int ret = stoi(s,NULL,x);
	cout &lt;&lt; ret &lt;&lt; endl;
	
	
	
	return 0;
}</code></pre>
    <h4>
     m进制转换成n进制
    </h4>
    <p>
     <img alt="" height="958" src="https://i-blog.csdnimg.cn/direct/501c15f29e22425780bb770ead4641a7.png" width="1305"/>
    </p>
    <p>
     这种题，我们先把m进制转换为10进制，再把10进制转换成n进制就行了
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
using namespace std;
string s1 = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
void n_to_m(int n,int x)
{
	if(n&gt;=x) n_to_m(n/x,x);
	cout &lt;&lt; s1[n%x];
}
int main()
{
	int n,m;
	string s;
	cin &gt;&gt; n &gt;&gt; s &gt;&gt; m;
	int theInt = stoi(s,NULL,n);
	n_to_m(theInt,m);
}</code></pre>
