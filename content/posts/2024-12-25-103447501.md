    <h2>
     <a id="Git_4">
     </a>
     共享文件夹配置Git服务器
    </h2>
    <p>
     本文说明在局域网中用一台电脑作为服务器，在服务器中创建一个文件夹，并设置文共享。然后该文件夹作为公开的版本库，局域网内其他电脑都可以访问。
    </p>
    <h3>
     <a id="_7">
     </a>
     配置公开版本库
    </h3>
    <h4>
     <a id="_9">
     </a>
     服务器端
    </h4>
    <p>
     首先在d盘创建一个文件夹，名字为GitTest.
    </p>
    <p>
     1.在GitTest目录下新建一个裸仓库名字为test,命令如下：
    </p>
    <pre><code class="prism language-cmd">git init --bare test
</code></pre>
    <p>
     test目录结构如下：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/517fa5759244b69777173f7888ff3aea.png"/>
    </p>
    <p>
     2.设置文件夹为共享文件夹：
    </p>
    <p>
     右键 --&gt; 属性，如下：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/04a5876c7f6c7c8b652285d3f02561ca.png"/>
    </p>
    <p>
     3.然后设置为所用用户，点击Everyone – &gt; 共享：
    </p>
    <p>
     <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/a31d19c1a01b4a2b4a3819ba5c3b0b1a.png"/>
    </p>
    <p>
     4.这个时候该文件夹属性上有个网络路径：
    </p>
    <img src="https://i-blog.csdnimg.cn/blog_migrate/8001602072418ac7e10ec52abb65ba98.png">
     <h4>
      <a id="_37">
      </a>
      客户端
     </h4>
     <p>
      1.在客户端右键我的电脑–&gt;映射网络驱动器，如下图所示：
     </p>
     <p>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/fc544f6447b1ae784d581ac8487aca01.png"/>
     </p>
     <p>
      这个时候会发现电脑下多了个Z盘。Z盘就是公开文件夹
     </p>
     <p>
      <img alt="" src="https://i-blog.csdnimg.cn/blog_migrate/da3949c63e2f40af9f48e860ca5d690b.png"/>
     </p>
     <p>
      test就是公共版本库。
     </p>
     <h3>
      <a id="_49">
      </a>
      测试
     </h3>
     <p>
      在客户端，创建本地版本库，加入该版本库下有一个项目。名字叫pc,
     </p>
     <p>
      先把项目push到版本库中，命令：
     </p>
     <pre><code class="prism language-cmd">git add .
git commit -m "pc"
git remote add origin z:\\test
git push origin master
</code></pre>
     <p>
      这个时候在另一个客户端需要修改该工程，只需在另一个客户端配置好公开版本库，把项目克隆下来：
     </p>
     <pre><code class="prism language-cmd">git clone z:\\test
</code></pre>
     <p>
      这样这个项目就在test目录下了。
     </p>
     <p>
      最后记得在不使用时要把共享文件夹取消共享。
     </p>
     <h2>
      <a id="GitStackGit_71">
      </a>
      GitStack搭建Git服务器
     </h2>
     <h3>
      <a id="_72">
      </a>
      安装部署
     </h3>
     <p>
      下载地址：
      <a href="http://gitstack.com/download" rel="nofollow">
       http://gitstack.com/download
      </a>
      <br/>
      下载后安装，安装完成后会自动跳出管理页面：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/719903545839eeea43ce18f8f631b74b.png">
       <br/>
       默认用户名和密码是admin/admin
       <br/>
       登录后首先找到右边的Repositories，创建一个版本库：
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/211dd5194f00b75be2cc1479a2071d30.png">
        <br/>
        user &amp; groups里面的User，创建一个用户：
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/599a323de0fd25a79eaa1a6b28ca5231.png">
         <br/>
         然后回到Repositories，把repo仓库授权给user
         <br/>
         <img alt="" src="https://imgconvert.csdnimg.cn/aHR0cHM6Ly9naXRodWIuY29tL1ZpSmFyLVdhbmcvRmlndXJlYmVkL2Jsb2IvbWFzdGVyL2ltZy8yMDE5MTIxMDA4NTIyNi5wbmc?x-oss-process=image/format,png">
          <br/>
          <img alt="" src="https://imgconvert.csdnimg.cn/aHR0cHM6Ly9naXRodWIuY29tL1ZpSmFyLVdhbmcvRmlndXJlYmVkL2Jsb2IvbWFzdGVyL2ltZy8yMDE5MTIxMDA4NTY1NS5wbmc?x-oss-process=image/format,png"/>
          <br/>
          到这里服务器就配置好了，只需要在客户端通过以下命令clone就好了：
         </img>
        </img>
       </img>
      </img>
     </p>
     <pre><code>git clone http://ip地址/repo.git
</code></pre>
     <p>
      运行后会让你输入用户名和密码，ip地址为服务器ip地址。
      <br/>
      但是你会发现一个问题，输入
      <code>
       git branch -a
      </code>
      ,发现远程和本地仓库都没有分支
      <br/>
      <img alt="" src="https://imgconvert.csdnimg.cn/aHR0cHM6Ly9naXRodWIuY29tL1ZpSmFyLVdhbmcvRmlndXJlYmVkL2Jsb2IvbWFzdGVyL2ltZy8yMDE5MTIxMDA5MTQwMS5wbmc?x-oss-process=image/format,png"/>
      <br/>
      我们需要创建一个本地分支并把该分支上传至远程库：
     </p>
     <pre><code class="prism language-cmd">git branch master
git push origin master:master
</code></pre>
     <p>
      然后使用
      <code>
       git branch -a
      </code>
      来查看所有分支，远程显示为红色。
      <br/>
      master分支用来存放初始文件，所以还需要在本地创建一个
      <code>
       ww
      </code>
      分支用来存放自己改过的文件。
      <br/>
      接下来就可以正常的开发操作了。
     </p>
     <h3>
      <a id="_100">
      </a>
      问题
     </h3>
     <ol>
      <li>
       安装后没有弹出页面问题
       <br/>
       （1）Gitstack默认端口为80，所以需要关闭80端口的程序。
       <br/>
       （2）gitstack以Apache为服务器，所以需要在服务中启动gitstack服务
      </li>
     </ol>
