##### 四、上传、更新、删除代码到github

###### 1、初次上传代码

> git init                                                          #会生成一个.git文件     ![](https://i-blog.csdnimg.cn/direct/6c982ab93a374cfab1d2547c8b87ec63.png)
>
> git branch -a                                               #检查当前分支状态
>
> git checkout -b main                                   #创建并切换到main分支
>
> git branch -D master                                  #删除本地master分支(master有点西方政治问题，
>
>                                                                      因此大多使用main)
>
> git add .                                                      #将文件添加到缓冲区
>
> git commit -m "初始提交"                           #提交添加到缓冲区
>
> git remote add origin xxx@xxx/test.git       #与github仓库建立连接
>
> git push origin main                                    #提交代码

![](https://i-blog.csdnimg.cn/direct/5ce403354b6d470bbb5c6fdabe612f5b.png)

提交后就可以看到代码了

![](https://i-blog.csdnimg.cn/direct/9cbe13cfead34f15bb807ea85f6b168c.png)

###### 2、更新新文件

![](https://i-blog.csdnimg.cn/direct/d3dd5bb3bf53414d848c9a88d6588545.png)

> git add 更新.txt  
>  git commit -m "更新"
>
> git push -u origin main

![](https://i-blog.csdnimg.cn/direct/2d60e319404f4f8b934662893002cf84.png)

更新完成

![](https://i-blog.csdnimg.cn/direct/129fff7e078b4ced83b547f6e740dfef.png)

###### 3、删除文件

> git rm -f '更新.txt'  
>  git commit -m 'del'  
>  git push -u origin main
