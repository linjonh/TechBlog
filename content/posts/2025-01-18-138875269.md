    <p>
     编写student实体类，及其controller和dao层，因为只是插入数据所以不需要加service层。

    <h2>
     方法一
    </h2>
    <p>
     最简单的方法就是利用循环，不断执行单条数据的插入指令。
    </p>
    <p>
     因此在dao中写一个insertStudent方法，并在xml文件中编写sql语句
    </p>
    <pre><code class="language-java">void insertStudent(Student student);</code></pre>
    <pre><code class="language-XML">    &lt;insert id="insertStudent" parameterType="com.example.bootdemo.entity.Student"&gt;
        insert into student (s_name,s_birth,s_sex) values (#{s_name},#{s_birth},#{s_sex})
    &lt;/insert&gt;</code></pre>
    <p>
     随后在controller中编写循环条用该方法，比如循环插入1000条数据，代码如下：
    </p>
    <pre><code class="language-java">    @ResponseBody
    @RequestMapping("/insertstudent")
    public Integer insertStudent(){
        System.out.println("开始插入");
        long start = System.currentTimeMillis();
        /**
         * 依靠循环插入
         */
        for (int i = 0; i &lt; 1000; i++) {
                Student student = new Student();
                student.setS_birth("20230206");
                student.setS_name("zjd");
                student.setS_sex("男");
                studentDao.insertStudent(student);
        }

        long end = System.currentTimeMillis();
        System.out.println("耗时："+(end-start));
        return 1;
    }</code></pre>
    <p>
     这种方式虽然可以实现，但是效率比较慢，因为每次执行插入都要执行一次sql，速度很慢。
    </p>
    <h2>
     <br/>
     方法二
    </h2>
    <p>
     在所有要插入的数据放在列表中，并在sql中利用foreach进行批量插入。这样执行一次sql就可以插入很多数据。
    </p>
    <p>
     xml编写中编写sql
    </p>
    <pre><code class="language-java">    &lt;insert id="batchInsertStudent" parameterType="java.util.List"&gt;
        insert into student (s_name,s_birth,s_sex) values
        &lt;foreach collection="students"  item="student" index="index" separator=","&gt;
            (
             #{student.s_name},
             #{student.s_birth},
             #{student.s_sex}
            )
        &lt;/foreach&gt;
    &lt;/insert&gt;</code></pre>
    <p>
     将数据方法List中执行sql语句。
    </p>
    <pre><code class="language-java">    @ResponseBody
    @RequestMapping("/insertstudent")
    public Integer insertStudent(){
        System.out.println("开始插入");
        long start = System.currentTimeMillis();
        /**
         * 批量插入，大量数据时不推荐使用
         */
        List&lt;Student&gt; students = new ArrayList&lt;&gt;(count);
        for(int i=0;i&lt;count;i++){
            Student student = new Student();
            student.setS_name("zjd"+i);
            student.setS_birth("20230206");
            student.setS_name("zjd");
            student.setS_sex("男");
            students.add(student);
        }
        studentDao.batchInsertStudent(students);

        long end = System.currentTimeMillis();
        System.out.println("耗时："+(end-start));
        return 1;
    }</code></pre>
    <p>
     这两种方法在数据量很大时都不推荐使用，第一种会很慢，第二种可能会因为数据过多，sql执行失败，直接报错。
    </p>
    <p>
    </p>
    <h2>
     方法三
    </h2>
    <p>
     既然第二种在插入大量数据时会报错，那么面对大量数据，我们可以将其分批插入，比如我可以每次直插入3000条数据，执行多次就可以实现大量数据的插入。
    </p>
    <p>
     代码如下：
    </p>
    <pre><code class="language-java">    @ResponseBody
    @RequestMapping("/insertstudent")
    public Integer insertStudent() throws InterruptedException {
        System.out.println("开始插入");
        long start = System.currentTimeMillis();
        CountDownLatch countDownLatch = new CountDownLatch(6);
        for(int i=0;i&lt;6;i++){
            List&lt;Student&gt; students = new ArrayList&lt;&gt;(count);
            int tempCount = 0;
            for(int n=0;n&lt;count;n++){
                if(tempCount==2999){
                    studentDao.batchInsertStudent(students);
                    tempCount=0;
                    students.clear();
                }
                Student student = new Student();
                student.setS_name("zjd"+i);
                student.setS_birth("20230206");
                student.setS_name("zjd");
                student.setS_sex("男");
                students.add(student);
                tempCount++;
            }
            studentDao.batchInsertStudent(students);
            long end = System.currentTimeMillis();
            System.out.println("耗时："+(end-start));
            countDownLatch.countDown();
        }
        countDownLatch.await();
        return 1;
    }</code></pre>
    <p>
     这样速度也会比单条循环插入要快很多。
    </p>
