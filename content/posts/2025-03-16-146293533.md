    <h1 class="title-article" id="articleContentId">
     CSS 盒模型
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     CSS盒模型（Box Model）是网页布局的核心概念，它将每个HTML元素视为一个矩形盒子，由内容、内边距、边框和外边距四部分组成。
    </p>
    <h4>
     <strong>
      组成部分：
     </strong>
    </h4>
    <ol>
     <li>
      ​
      <strong>
       内容区（Content）​
      </strong>
      ：存放文本、图片等元素的实际内容，由
      <code>
       width/height
      </code>
      控制大小
     </li>
     <li>
      ​
      <strong>
       内边距（Padding）​
      </strong>
      ：内容与边框的间距，通过
      <code>
       padding
      </code>
      调整，支持分方向设置
     </li>
     <li>
      ​
      <strong>
       边框（Border）​
      </strong>
      ：包裹内容与内边距的可见线条，可定义样式、颜色及宽度
     </li>
     <li>
      ​
      <strong>
       外边距（Margin）​
      </strong>
      ：与其他元素的透明间距，通过
      <code>
       margin
      </code>
      控制，垂直方向可能合并（取较大值）
     </li>
    </ol>
    <h4>
     两种盒模型
    </h4>
    <p id="a72919e313b8a05036311959ccf8fa98">
     CSS3中的盒模型有以下两种：标准盒子模型、IE盒子模型（也叫怪异盒模型）
    </p>
    <p>
     <img alt="" height="800" src="https://i-blog.csdnimg.cn/direct/eee42bc96fb346048c7a0ac7ce80c614.png" width="1309"/>
    </p>
    <p>
     <img alt="" height="753" src="https://i-blog.csdnimg.cn/direct/19fdd237de7c4c538f57415be6cd747b.png" width="1314"/>
    </p>
    <p>
     盒模型都是由四个部分组成的，分别是margin、border、padding和content。
    </p>
    <p>
     标准盒模型和IE盒模型的区别在于设置
     <strong>
      width
     </strong>
     和
     <strong>
      height
     </strong>
     时，所
     <strong>
      对应的范围不同
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       标准盒模型的width和height属性的范围只包含了content
      </p>
     </li>
     <li>
      <p>
       IE盒模型的width和height属性的范围包含了border、padding和content
      </p>
     </li>
    </ul>
    <p id="9e5972c95ce7fa2c85c1ba79b4d03a0d">
     可以通过修改元素的box-sizing属性来改变元素的盒模型：
    </p>
    <ul>
     <li id="aac7400b7bbdee3d8be9490e013a110f">
      <code>
       box-sizing: content-box
      </code>
      表示标准盒模型（默认值）
     </li>
     <li id="856b1a7ac79670f45c5b97f9b99d8b01">
      <code>
       box-sizing: border-box
      </code>
      表示IE盒模型（怪异盒模型）
     </li>
    </ul>
