    <p>
     在Windows、macOS和Linux系统下统计文件夹下的文件数量的几种方法。
    </p>
    <h4>
     <a id="Windows__2">
     </a>
     Windows 系统
    </h4>
    <p>
     <strong>
      使用命令提示符（Command Prompt）
     </strong>
    </p>
    <ol>
     <li>
      打开命令提示符。你可以通过在开始菜单搜索
      <code>
       cmd
      </code>
      来找到它。
     </li>
     <li>
      使用
      <code>
       cd
      </code>
      命令切换到你想要统计文件数量的文件夹。例如，如果你的文件夹路径是
      <code>
       C:\Users\YourName\Documents
      </code>
      ，你可以输入
      <code>
       cd C:\Users\YourName\Documents
      </code>
      。
     </li>
     <li>
      然后输入以下命令来统计文件夹下的文件数量（包括文件夹和文件）：
      <pre><code>dir /A /B | find /C /V ""
</code></pre>
      这个命令将列出文件夹下的所有内容（
      <code>
       dir /A /B
      </code>
      ），并统计这些内容的数量（
      <code>
       find /C /V ""
      </code>
      ）。
     </li>
    </ol>
    <p>
     <strong>
      仅统计文件数量（不包括子文件夹）
     </strong>
    </p>
    <p>
     如果你只想要统计文件的数量，可以使用：
    </p>
    <pre><code>dir /A-D /B | find /C /V ""
</code></pre>
    <p>
     这里的
     <code>
      /A-D
     </code>
     选项用于排除文件夹的统计。
    </p>
    <h4>
     <a id="macOS__Linux__22">
     </a>
     macOS 和 Linux 系统
    </h4>
    <p>
     <strong>
      使用终端（Terminal）
     </strong>
    </p>
    <ol>
     <li>
      打开终端。在macOS中，你可以通过Finder打开
      <code>
       Applications &gt; Utilities &gt; Terminal
      </code>
      。在Linux中，可以使用快捷键
      <code>
       Ctrl + Alt + T
      </code>
      或者在应用菜单中搜索
      <code>
       Terminal
      </code>
      。
     </li>
     <li>
      使用
      <code>
       cd
      </code>
      命令切换到目标文件夹。
     </li>
     <li>
      使用以下命令来统计文件夹下所有文件和文件夹的数量：
      <pre><code>ls -l | grep -c '^d'
</code></pre>
      这个命令首先使用
      <code>
       ls -l
      </code>
      列出所有内容的详细信息，然后
      <code>
       grep -c '^d'
      </code>
      统计以’d’开头的行数，这些行表示文件夹。如果你想要得到所有文件加文件夹的总数，只需移除
      <code>
       grep -c '^d'
      </code>
      的部分，用
      <code>
       ls -1 | wc -l
      </code>
      命令代替：
      <pre><code>ls -1 | wc -l
</code></pre>
      这里的
      <code>
       -1
      </code>
      选项使
      <code>
       ls
      </code>
      命令每行输出一个条目，
      <code>
       wc -l
      </code>
      则统计这些行的数量。
     </li>
    </ol>
    <p>
     <strong>
      仅统计文件数量
     </strong>
    </p>
    <p>
     如果你想统计文件夹下的文件数量，而不包括子文件夹，可以使用：
    </p>
    <pre><code>ls -l | grep -v '^d' | wc -l
</code></pre>
    <p>
     这里
     <code>
      grep -v '^d'
     </code>
     用于排除以’d’开头的行，即排除文件夹。
    </p>
