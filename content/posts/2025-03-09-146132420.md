## 基于长期主义的内容更新

可以说，Redis 已经在很长一段时间内被采纳为企业级的高速缓存解决方案了，2019 年我大一刚入学的时候，Redis 基本已经被视为后端开发必备的技术栈之一了。Redis 涵盖的内容非常的广泛，每一个点又有可供挖掘的深度，比如 Redis 的数据结构、使用 Redis 做分布式锁等。目前我个人的 Redis 实战能力还是不够强，要学习和进步的地方很多。抛开八股不谈的话，可以说 Redis 本身的应用我还没有吃透。

熟悉我的人都知道，我是一个长期主义的践行者。后续我将会继续在数据库这个专栏（当然，我也有可能将 Redis 从数据库这个专栏独立出去）总结与 Redis 相关的应用。欢迎持续关注。
