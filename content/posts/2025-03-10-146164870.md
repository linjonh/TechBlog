     linux 命令 cat

    <p>
     <code>
      cat
     </code>
     是 Linux 中用于查看、创建和合并文件的常用命令，全称
     <strong>
      concatenate
     </strong>
     （连接）。其核心功能是将文件内容输出到终端或重定向到其他文件/命令中。以下是详细用法及场景示例：
    </p>
    <h2 style="background-color:transparent">
     <strong>
      基本语法
     </strong>
    </h2>
    <blockquote>
     <p>
      cat [选项] [文件1] [文件2] ...
     </p>
    </blockquote>
    <p>
    </p>
    <table>
     <thead>
      <tr>
       <th>
        选项
       </th>
       <th>
        描述
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <code>
         -n
        </code>
        或
        <code>
         --number
        </code>
       </td>
       <td>
        显示所有行的行号（包括空行）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -b
        </code>
        或
        <code>
         --number-nonblank
        </code>
       </td>
       <td>
        显示非空行的行号（忽略空行）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -s
        </code>
        或
        <code>
         --squeeze-blank
        </code>
       </td>
       <td>
        压缩连续空行为一行。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -E
        </code>
        或
        <code>
         --show-ends
        </code>
       </td>
       <td>
        在每行末尾显示
        <code>
         $
        </code>
        符号（便于观察行尾空格）。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -T
        </code>
        或
        <code>
         --show-tabs
        </code>
       </td>
       <td>
        将制表符（Tab）显示为
        <code>
         ^I
        </code>
        。
       </td>
      </tr>
      <tr>
       <td>
        <code>
         -v
        </code>
        或
        <code>
         --show-nonprinting
        </code>
       </td>
       <td>
        显示非打印字符（如换行符、制表符等）。
       </td>
      </tr>
     </tbody>
    </table>
    <p>
    </p>
    <h2 style="background-color:transparent">
     <strong>
      核心功能与示例
     </strong>
    </h2>
    <h3>
     1.
     <strong>
      查看文件内容
     </strong>
    </h3>
    <p>
     <img alt="" height="111" src="https://i-blog.csdnimg.cn/direct/a9909455b4c14320adbef1025644213b.png" width="732">
      <img alt="" height="188" src="https://i-blog.csdnimg.cn/direct/9047b7462358469cb935fc697400f73d.png" width="848"/>
     </img>
    </p>
    <h3>
     2.
     <strong>
      创建新文件（或覆盖内容
     </strong>
    </h3>
    <p>
    </p>
    <pre> # 从键盘输入内容，按 Ctrl+D 保存并退出</pre>
    <p>
     <img alt="" height="230" src="https://i-blog.csdnimg.cn/direct/cab9b490a24d4fde8a1789bf4d49e31d.png" width="787"/>
    </p>
    <pre> # 追加内容到  a01file.txt（不覆盖原内容）
 <img alt="" height="154" src="https://i-blog.csdnimg.cn/direct/f08aef5dbd424e2c866f85e99c950c2c.png" width="345"/></pre>
    <h3>
     3.
     <strong>
      合并多个文件
     </strong>
    </h3>
    <blockquote>
     <p>
      cat file1.txt file2.txt &gt; combined.txt  # 合并 file1 和 file2 到 combined.txt
      <br/>
      cat *.log &gt; all_logs.txt                # 合并所有 .log 文件到 all_logs.txt
     </p>
    </blockquote>
    <h3>
     4.
     <strong>
      与管道符结合使用
     </strong>
    </h3>
    <blockquote>
     <p>
      cat file.txt | grep "error"    # 将文件内容传递给 grep 搜索关键词
      <br/>
      cat file.txt | less           # 分页查看大文件（替代直接 cat 大文件）
     </p>
    </blockquote>
    <h3>
     <strong>
      实际应用场景
     </strong>
    </h3>
    <h4>
     <strong>
      场景1：快速查看日志文件
     </strong>
    </h4>
    <p>
     cat /var/log/syslog | tail -n 20  # 查看日志最后20行（避免滚动刷屏）
    </p>
    <h4>
     <strong>
      场景2：合并多个配置文件
     </strong>
    </h4>
    <p>
     cat config-*.conf &gt; master.conf   # 合并所有以 config- 开头的配置文件
    </p>
    <h4>
     <strong>
      场景3：生成多行文本（Here Document）
     </strong>
    </h4>
    <blockquote>
     <p>
      cat &lt;&lt;EOF &gt; script.sh
      <br/>
      #!/bin/bash
      <br/>
      echo "Hello, World!"
      <br/>
      EOF
     </p>
    </blockquote>
    <p>
     <img alt="" height="211" src="https://i-blog.csdnimg.cn/direct/3c8ca6940ab64566ab62e13a02bae798.png" width="408"/>
    </p>
    <h4>
     <strong>
      场景4：检查文件隐藏字符
     </strong>
    </h4>
    <blockquote>
     <p>
      cat -v file.txt  # 发现不可见字符（如 Windows 换行符 `^M`）
     </p>
    </blockquote>
    <h2 style="background-color:transparent">
     <strong>
      总结
     </strong>
    </h2>
    <ul>
     <li>
      <p>
       <strong>
        简单场景
       </strong>
       ：
       <code>
        cat
       </code>
       适合快速查看、合并小文件或生成简单文本。
      </p>
     </li>
     <li>
      <p>
       <strong>
        复杂需求
       </strong>
       ：结合管道符 (
       <code>
        |
       </code>
       ) 与其他命令（如
       <code>
        grep
       </code>
       、
       <code>
        sort
       </code>
       ）协作，或使用
       <code>
        less
       </code>
       、
       <code>
        awk
       </code>
       等工具
      </p>
     </li>
    </ul>
