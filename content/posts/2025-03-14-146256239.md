     HTML5-基础知识
    </h1>

    <p>
     以下是关于 HTML（超文本标记语言）的详细知识点总结：
    </p>
    <h4>
     1. HTML 简介
    </h4>
    <ul>
     <li>
      <strong>
       定义
      </strong>
      ：HTML 是用于创建网页的标准标记语言，它使用标签来描述网页的结构和内容。
     </li>
     <li>
      <strong>
       作用
      </strong>
      ：通过各种 HTML 标签，我们可以在网页上展示文本、图片、链接、表格、表单等元素。
     </li>
    </ul>
    <h4>
     2. HTML 基本结构
    </h4>
    <p>
     一个基本的 HTML 文档结构如下：
    </p>
    <pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
  &lt;meta charset="UTF-8"&gt;
  &lt;meta name="viewport" content="width=device-width, initial-scale=1.0"&gt;
  &lt;title&gt;Document&lt;/title&gt;
&lt;/head&gt;

&lt;body&gt;
  &lt;!-- 这里是网页的可见内容 --&gt;
&lt;/body&gt;

&lt;/html&gt;
</code></pre>
    <ul>
     <li>
      <code>
       &lt;!DOCTYPE html&gt;
      </code>
      ：声明文档类型为 HTML5。
     </li>
     <li>
      <code>
       &lt;html&gt;
      </code>
      ：HTML 文档的根标签，所有 HTML 元素都包含在其中。
     </li>
     <li>
      <code>
       &lt;head&gt;
      </code>
      ：包含文档的元数据，如字符编码、页面标题等，这些内容不会直接显示在网页上。
     </li>
     <li>
      <code>
       &lt;meta charset="UTF-8"&gt;
      </code>
      ：设置文档的字符编码为 UTF - 8，确保支持各种语言字符。
     </li>
     <li>
      <code>
       &lt;title&gt;
      </code>
      ：定义网页的标题，显示在浏览器的标题栏或标签页上。
     </li>
     <li>
      <code>
       &lt;body&gt;
      </code>
      ：包含网页的可见内容，如文本、图片、链接等。
     </li>
    </ul>
    <h4>
     3. HTML 标签
    </h4>
    <h5>
     3.1 文本标签
    </h5>
    <ul>
     <li>
      <code>
       &lt;h1&gt; - &lt;h6&gt;
      </code>
      ：标题标签，
      <code>
       &lt;h1&gt;
      </code>
      是最大的标题，
      <code>
       &lt;h6&gt;
      </code>
      是最小的标题。
     </li>
    </ul>
    <pre><code>&lt;h1&gt;这是一级标题&lt;/h1&gt;
&lt;h2&gt;这是二级标题&lt;/h2&gt;
</code></pre>
    <ul>
     <li>
      <code>
       &lt;p&gt;
      </code>
      ：段落标签，用于定义段落。
     </li>
    </ul>
    <pre><code>&lt;p&gt;这是一个段落。&lt;/p&gt;
</code></pre>
    <ul>
     <li>
      <code>
       &lt;b&gt;
      </code>
      ：粗体标签，使文本加粗。
     </li>
    </ul>
    <pre><code>&lt;b&gt;这是粗体文本&lt;/b&gt;
</code></pre>
    <ul>
     <li>
      <code>
       &lt;i&gt;
      </code>
      ：斜体标签，使文本倾斜。
     </li>
    </ul>
    <pre><code>&lt;i&gt;这是斜体文本&lt;/i&gt;
</code></pre>
    <ul>
     <li>
      <code>
       &lt;u&gt;
      </code>
      ：下划线标签，给文本添加下划线。
     </li>
    </ul>
    <pre><code>&lt;u&gt;这是带下划线的文本&lt;/u&gt;
</code></pre>
    <h5>
     3.2 列表标签
    </h5>
    <ul>
     <li>
      <strong>
       无序列表
      </strong>
      ：使用
      <code>
       &lt;ul&gt;
      </code>
      和
      <code>
       &lt;li&gt;
      </code>
      标签，列表项前会显示圆点。
     </li>
    </ul>
    <pre><code>&lt;ul&gt;
  &lt;li&gt;列表项 1&lt;/li&gt;
  &lt;li&gt;列表项 2&lt;/li&gt;
&lt;/ul&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       有序列表
      </strong>
      ：使用
      <code>
       &lt;ol&gt;
      </code>
      和
      <code>
       &lt;li&gt;
      </code>
      标签，列表项前会显示数字。
     </li>
    </ul>
    <pre><code>&lt;ol&gt;
  &lt;li&gt;列表项 1&lt;/li&gt;
  &lt;li&gt;列表项 2&lt;/li&gt;
&lt;/ol&gt;
</code></pre>
    <h5>
     3.3 链接标签
    </h5>
    <ul>
     <li>
      <code>
       &lt;a&gt;
      </code>
      ：用于创建超链接，可以链接到其他网页、文件或同一页面的不同位置。
     </li>
    </ul>
    <pre><code>&lt;a href="https://www.example.com"&gt;这是一个外部链接&lt;/a&gt;
&lt;a href="#section2"&gt;跳转到页面内的 section2 部分&lt;/a&gt;
</code></pre>
    <h5>
     3.4 图片标签
    </h5>
    <ul>
     <li>
      <code>
       &lt;img&gt;
      </code>
      ：用于在网页中插入图片，
      <code>
       src
      </code>
      属性指定图片的源文件路径，
      <code>
       alt
      </code>
      属性用于在图片无法显示时提供替代文本。
     </li>
    </ul>
    <pre><code>&lt;img src="image.jpg" alt="这是一张图片"&gt;
</code></pre>
    <h5>
     3.5 表格标签
    </h5>
    <ul>
     <li>
      <code>
       &lt;table&gt;
      </code>
      ：定义表格。
     </li>
     <li>
      <code>
       &lt;tr&gt;
      </code>
      ：定义表格的行。
     </li>
     <li>
      <code>
       &lt;th&gt;
      </code>
      ：定义表格的表头单元格。
     </li>
     <li>
      <code>
       &lt;td&gt;
      </code>
      ：定义表格的数据单元格。
     </li>
    </ul>
    <pre><code>&lt;table&gt;
  &lt;tr&gt;
    &lt;th&gt;姓名&lt;/th&gt;
    &lt;th&gt;年龄&lt;/th&gt;
  &lt;/tr&gt;
  &lt;tr&gt;
    &lt;td&gt;张三&lt;/td&gt;
    &lt;td&gt;25&lt;/td&gt;
  &lt;/tr&gt;
&lt;/table&gt;
</code></pre>
    <h5>
     3.6 表单标签
    </h5>
    <ul>
     <li>
      <code>
       &lt;form&gt;
      </code>
      ：用于创建 HTML 表单，用于用户输入数据。
     </li>
     <li>
      <code>
       &lt;input&gt;
      </code>
      ：常用的表单输入元素，
      <code>
       type
      </code>
      属性可以设置为不同的值，如
      <code>
       text
      </code>
      、
      <code>
       password
      </code>
      、
      <code>
       radio
      </code>
      、
      <code>
       checkbox
      </code>
      等。
     </li>
     <li>
      <code>
       &lt;textarea&gt;
      </code>
      ：用于多行文本输入。
     </li>
     <li>
      <code>
       &lt;select&gt;
      </code>
      和
      <code>
       &lt;option&gt;
      </code>
      ：用于创建下拉列表。
     </li>
    </ul>
    <pre><code>&lt;form action="submit.php" method="post"&gt;
  &lt;label for="username"&gt;用户名:&lt;/label&gt;
  &lt;input type="text" id="username" name="username"&gt;&lt;br&gt;
  &lt;label for="password"&gt;密码:&lt;/label&gt;
  &lt;input type="password" id="password" name="password"&gt;&lt;br&gt;
  &lt;input type="submit" value="提交"&gt;
&lt;/form&gt;
</code></pre>
    <h4>
     4. HTML 属性
    </h4>
    <ul>
     <li>
      属性用于提供关于 HTML 元素的额外信息，通常以键值对的形式出现，写在标签的开始标签内。
     </li>
     <li>
      例如，
      <code>
       &lt;a&gt;
      </code>
      标签的
      <code>
       href
      </code>
      属性用于指定链接的目标地址，
      <code>
       &lt;img&gt;
      </code>
      标签的
      <code>
       src
      </code>
      属性用于指定图片的源文件路径。
     </li>
    </ul>
    <pre><code>&lt;a href="https://www.example.com" target="_blank"&gt;打开新窗口访问链接&lt;/a&gt;
</code></pre>
    <p>
     这里的
     <code>
      target="_blank"
     </code>
     是一个属性，它指定链接将在新窗口中打开。
    </p>
    <h4>
     5. HTML 注释
    </h4>
    <ul>
     <li>
      使用
      <code>
       &lt;!-- --&gt;
      </code>
      来添加注释，注释内容不会在浏览器中显示，仅用于开发者在代码中添加说明。
     </li>
    </ul>
    <pre><code>&lt;!-- 这是一个注释，用于说明下面的代码块 --&gt;
&lt;p&gt;这是一个段落&lt;/p&gt;
</code></pre>
    <h4>
     6. HTML 与其他技术的结合
    </h4>
    <ul>
     <li>
      <strong>
       CSS
      </strong>
      ：层叠样式表，用于美化 HTML 页面，控制元素的外观和布局。可以通过
      <code>
       &lt;style&gt;
      </code>
      标签内联样式、
      <code>
       &lt;link&gt;
      </code>
      标签引入外部 CSS 文件或使用
      <code>
       style
      </code>
      属性内联样式。
     </li>
    </ul>
    <pre><code>&lt;head&gt;
  &lt;link rel="stylesheet" href="styles.css"&gt;
  &lt;style&gt;
    p {
      color: blue;
    }
  &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
  &lt;p style="font-size: 16px;"&gt;这是一个段落&lt;/p&gt;
&lt;/body&gt;
</code></pre>
    <ul>
     <li>
      <strong>
       JavaScript
      </strong>
      ：用于为网页添加交互性和动态效果。可以通过
      <code>
       &lt;script&gt;
      </code>
      标签引入外部 JavaScript 文件或直接在 HTML 文件中编写 JavaScript 代码。
     </li>
    </ul>
    <pre><code>&lt;body&gt;
  &lt;button id="myButton"&gt;点击我&lt;/button&gt;
  &lt;script&gt;
    const button = document.getElementById('myButton');
    button.addEventListener('click', function() {
      alert('你点击了按钮');
    });
  &lt;/script&gt;
&lt;/body&gt;
</code></pre>
    <h4>
     总结
    </h4>
    <p>
     HTML 是构建网页的基础，它通过各种标签和属性来定义网页的结构和内容。掌握 HTML 的基本结构、常用标签、属性和注释的使用，是学习网页开发的第一步。同时，结合 CSS 和 JavaScript 可以创建出更加美观、交互性强的网页。
    </p>
