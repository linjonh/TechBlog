     <span style="color:#0d0016;">
      一，集合
     </span>
    </h4>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      <strong>
       1.概念
      </strong>
     </span>
    </h5>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      （1）集合  元素   有限集合   无限集合
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      （2）集合的表示方法
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      集合：A.   B.  C. ...     元素：a. b. c. ...
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      a ∈  A  (a属于集合A)  a ∉ A  （a不属与集合A）
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      表示方法：列举法、描述法
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      (3)集合之间的关系
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      A为B的子集 A⊂B  (A包含在B里)；      B为A的子集   B⊃A   (B包含在A里)；
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      两集合相等的充分必要条件是：A⊂B 且B⊃A
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      A⊂B 且A≠B  称A为B的一个真子集。
     </span>
    </p>

     <span style="color:#0d0016;">
      空集：

    <h5 style="margin-left:0px;">
     <span style="color:#0d0016;">
      <strong>
       3.区间与邻域
      </strong>
     </span>
    </h5>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      （1）区间
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       定义
      </strong>
      ：区间是实数轴上连续的一段数集，它表示了某一范围内的所有实数。
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       分类
      </strong>
      ：
     </span>
    </p>
    <ul>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        开区间
       </strong>
       ：表示不包括端点的区间，记作(a, b)，其中a和b是实数，且a &lt; b。图像上，开区间(a, b)对应于数轴上a和b之间但不包括a和b的所有点。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        闭区间
       </strong>
       ：表示包括端点的区间，记作[a, b]，其中a和b是实数，且a &lt; b。图像上，闭区间[a, b]对应于数轴上a和b之间包括a和b的所有点。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        半开半闭区间
       </strong>
       ：表示只包括一个端点的区间，有两种情况：[a, b)和(a, b]，其中a和b是实数，且a &lt; b。图像上，[a, b)对应于数轴上从a（包括a）到b（不包括b）的所有点；(a, b]则对应于从a（不包括a）到b（包括b）的所有点。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        无限区间
       </strong>
       ：表示区间的一端或两端可以延伸到无穷大，如(-∞, a)、[a, +∞)、(-∞, +∞)等。图像上，这些区间分别对应于数轴上小于a的所有点、大于或等于a的所有点、以及所有实数点。
      </span>
     </li>
    </ul>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       解释
      </strong>
      ：区间是实数集的一种子集，它用于描述实数轴上的连续范围。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      （2）邻域
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       定义
      </strong>
      ：邻域是一个特殊的区间，它以某个点为中心，包含该点周围的一定范围内的所有点。
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       分类
      </strong>
      ：
     </span>
    </p>
    <ul>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        δ邻域
       </strong>
       ：设a是实数，δ是正数，则开区间(a-δ, a+δ)称为点a的δ邻域，记作U(a; δ)或简单地写作U(a)。图像上，U(a; δ)对应于数轴上以a为中心、δ为半径的开区间内的所有点。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        去心邻域
       </strong>
       ：有时我们需要考虑不包括中心点的邻域，即去心邻域。点a的去心δ邻域记作U°(a; δ)，它等于(a-δ, a)∪(a, a+δ)。图像上，U°(a; δ)对应于数轴上以a为中心、δ为半径但不包括a点的开区间内的所有点。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        左邻域和右邻域
       </strong>
       ：还可以定义点a的左δ邻域U-(a; δ)为(a-δ, a]，右δ邻域U+(a; δ)为[a, a+δ)。这些邻域分别表示a点左侧或右侧的一定范围内的所有点。
      </span>
     </li>
    </ul>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      <strong>
       邻域公理
      </strong>
      ：邻域公理是现代数学拓扑结构的基础概念之一，它定义了空间上的整套邻域系。具体来说，对于拓扑空间X中的任意点x，存在一个包含x的邻域系U(x)，满足一系列公理（如U1至U4，详细见参考文章）。这些公理确保了邻域系具有良好的性质，从而可以通过邻域来定义拓扑空间的其他重要概念（如开集、闭集等）。
     </span>
    </p>
    <h4 style="margin-left:0px;">
     <span style="color:#0d0016;">
      <strong>
       二、映射
      </strong>
     </span>
    </h4>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      映射的概念: 映射（Mapping）是指两个元素的集之间元素相互“对应”的关系。具体来说，如果两个非空集合A与B间存在着对应关系f，使得对于集合A中的任意一个元素a，在集合B中都有唯一的元素b与之对应，那么这种对应就称为从集合A到集合B的映射，记作f:A→B。其中，b称为元素a在映射f下的像，记作b=f(a)；a称为b关于映射f的原像。
     </span>
    </p>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (1)成立条件
     </span>
    </h5>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      映射的成立条件可以简单表述为：
     </span>
    </p>
    <ol>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        定义域的遍历性
       </strong>
       ：集合A中的每个元素在映射的值域中都有对应的像。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        唯一性
       </strong>
       ：对于集合A中的任意一个元素a，集合B中都存在唯一的元素b与之对应。
      </span>
     </li>
    </ol>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (2)分类
     </span>
    </h5>
    <ol>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       <strong>
        单射
       </strong>
       ：如果集合A中任意两个不同元素a1和a2，它们的像f(a1)和f(a2)也不同，即f(a1)≠f(a2)，则称映射f为单射。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">

    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (3)应用
     </span>
    </h5>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      映射在数学及相关领域有着广泛的应用，如用于定义函数。函数是从非空数集到非空数集的映射，而且只能是一对一映射或多对一映射。此外，映射还用于描述各种数学结构之间的关系，如线性变换、同态等。
     </span>
    </p>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (4)注意事项
     </span>
    </h5>
    <ol>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       映射和函数的关系：函数是映射的一个特例，即函数是两个数集之间的映射，且只能是一对一或多对一映射。但映射的概念更广泛，它可以描述任意两个集合元素之间的对应关系。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       映射中的元素关系：在映射f:A→B中，集合A中的元素a称为原像，集合B中与a对应的元素b称为像。映射关系可以表示为b=f(a)。
      </span>
     </li>
     <li style="text-align:left;">
      <span style="color:#0d0016;">
       映射与集合的关系：映射是两个集合之间的一种特殊关系，它描述了集合A中的元素如何与集合B中的元素相对应。这种对应关系可以是满射、单射或一一映射。
      </span>
     </li>
    </ol>

    </p>
    <h4 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      <strong>
       三、函数
      </strong>
     </span>
    </h4>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (1)函数的定义
     </span>
    </h5>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      1. 传统定义
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      传统定义是从运动变化的观点出发的。设在某变化过程中有两个变量x、y，如果对于x在某一范围内的每一个确定的值，y都有唯一确定的值与它对应，那么就称y是x的函数，x叫做自变量。这种定义强调了函数是一种变量间的依赖关系，即一个变量（自变量）的变化会唯一确定另一个变量（因变量）的值。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      2. 近代定义
     </span>
    </p>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      近代定义则是从集合、映射的观点出发的。设A、B是两个非空的数的集合，f是从A到B的一个对应法则，那么从A到B的映射f：A→B就叫做函数，记作y=f(x)，其中x∈A，y∈B。原象集合A叫做函数f(x)的定义域，象集合C（C是B的子集）叫做函数f(x)的值域。这种定义将函数视为一种特殊的映射，即定义域和值域都是数集的映射。
     </span>
    </p>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (2)函数的要素
     </span>
    </h5>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      函数概念含有三个要素：定义域A、值域C和对应法则f。
     </span>
    </p>
    <ol>
     <li>
      <span style="color:#0d0016;">
       <strong>
        定义域A
       </strong>
       ：自变量x取值的集合，即函数中输入值的集合。它是构成函数的一个不可缺少的组成部分。
      </span>
     </li>
     <li>
      <span style="color:#0d0016;">
       <strong>
        值域C
       </strong>
       ：函数值的集合，即与定义域中的元素通过对应法则f得到的所有可能输出的集合。它是值域B的子集。
      </span>
     </li>
     <li>
      <span style="color:#0d0016;">
       <strong>
        对应法则f
       </strong>
       ：函数关系的核心，它描述了定义域中的元素如何映射到值域中的元素。对应法则可以是解析式、图像、表格或文字描述等形式。
      </span>
     </li>
    </ol>
    <h5 style="margin-left:0px;text-align:left;">
     <span style="color:#0d0016;">
      (3)函数的表示法
     </span>
    </h5>
    <p style="margin-left:0;">
     <span style="color:#0d0016;">
      函数有多种表示法，常见的包括列表法、图像法和解析法。
     </span>
    </p>
    <ol>
     <li>
      <span style="color:#0d0016;">
       <strong>
        列表法
       </strong>
       ：通过列出函数自变量与对应函数值的表格来表示函数。
      </span>
     </li>
     <li>
      <span style="color:#0d0016;">
       <strong>
        图像法
       </strong>
       ：通过绘制函数图像来表示函数关系，图像上的每一点都对应着函数的一个自变量与函数值的对。
      </span>
     </li>
     <li>
      <span style="color:#0d0016;">
       <strong>
        解析法
       </strong>
       ：用数学表达式（即函数解析式）来表示函数关系，如y=f(x)=ax+b等。
      </span>
     </li>
    </ol>
    <p>
    </p>

     <span style="color:#0d0016;">
      <strong>
       4.连续性（Continuity）
      </strong>
      ：
      <br/>
      如果函数在其定义域内的某一点处，当自变量趋近于该点时，函数值也趋近于该点处的函数值，则称函数在该点处连续。如果函数在其定义域内的每一点都连续，则称该函数在其定义域内连续。
     </span>
    </p>

     <span style="color:#0d0016;">
      <strong>

      </strong>
      ：
      <br/>
      如果函数在某一点处的极限存在，且等于该点处的左右导数，则称函数在该点处可导。与函数的极值、凹凸性等性质密切相关。
     </span>
    </p>

     </span>
    </p>
    <p style="text-align:left;">
     <span style="color:#0d0016;">
      <strong>
       7.有界性（Boundedness）
      </strong>
      ：有界性是函数的一个重要特性，它描述了函数在某个区间上的输出范围。具体来说，若存在两个常数m和M，使得对于函数y=f(x)在其定义域D内的所有x，都有m≤f(x)≤M成立，则称函数y=f(x)在D上有界，其中m称为下界，M称为上界。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      <strong>
       有界函数：
      </strong>
      设f(x)是区间E上的函数，若对于任意的x属于E，存在常数m、M，使得m≤f(x)≤M，则称f(x)是区间E上的有界函数。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      <strong>
       无界函数：
      </strong>
      与有界函数相反，若对于任何M（无论M多大），都存在x0∈D，使得|f(x0)|≥M，则称f(x)在D上无界。
     </span>
    </p>

       ①
      </strong>

     </span>
    </p>

     </span>

     <span style="color:#0d0016;">
      狄利克雷函数是周期函数，其周期可以是任意正有理数。然而，由于不存在最小正有理数和负有理数，狄利克雷函数没有最小正周期。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">

     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
      它在任何点都不可导。
     </span>
    </p>
    <p style="margin-left:0;text-align:left;">
     <span style="color:#0d0016;">
