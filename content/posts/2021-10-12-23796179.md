    paip.编程语言到底有没有优劣之分优秀之分
    <br/>
    <br/>
    人有没有优秀之分之分呢？？狗有没有优秀之分呢？？
    <br/>
    当然是有的，有好人坏人的说法，或者精英平民的区分。。狗也有好狗狗，坏狗，疯狗嘛。。
    <br/>
    所以，自然，编程语言与自然语言类似。。是有优劣之分优秀之分的。。网上争议很多，是因为只看到一个方面，不同的场合，会有不同的优秀语言。。
    <br/>
    <br/>
    <br/>
    作者 老哇的爪子 Attilax 艾龙，  EMAIL:1466519819@qq.com
    <br/>
    来源： http://blog.csdn.net/attilax
    <br/>
    <br/>
    从文化的角度来看,语言是没优劣之分的..宗教热情？？？政治正确的角度。。
    <br/>
    比如，你的父母比我的父母好？？没有说法，双方是难以比较出来的，但是无关的第三者就可以抛弃偏见，得出结论。
    <br/>
    <br/>
    但是,就学习一个新的语言,还是流行的语言更好...所以,从实用的角度来说,语言有优劣之分...越流行的语言越好..对于我们普通人来说，普通pc机，客户端、命令行就是最实用的环境了。。
    <br/>
    <br/>
    另外，从硬件场合来看，有优劣之分..也许只有一个语言适合..更好..
    <br/>
    <br/>
    从使用场合,可能有另一个语言...例如,学习的环境,也许c更适合,但是c在普通场合很不实用...
    <br/>
    <br/>
    从GUI界面精美的角度来看，c++就比java作的好多。。QT+CSS 很精美的界面，java来做只能说垃圾了。。
    <br/>
    <br/>
    从语言优雅上，有优秀之分
    <br/>
    <br/>
    大多数语言都只能注定是非主流，边缘化，和是否优秀无关。
    <br/>
    <br/>
    <br/>
    一下是我总结的最好的编程语言（普通pc机，客户端、命令行下）。。
    <br/>
    大部分场合下（80%），java还是最好用的，ide方便，网上代码多。类库多。。
    <br/>
    有GUI的情况下（10%），python+qt了。。
    <br/>
    保护源码的情况下，这种情况其实不多（10%左右），c++ qt 了。。
    <br/>
    以上场合都是PC桌面上用的，至于嵌入式，手机，web等硬件场合下，自然有不同的编程语言更加适合。。，
