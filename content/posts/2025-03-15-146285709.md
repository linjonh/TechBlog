    <h2>
     <a id="Dart__2">
     </a>
     Dart 数据类型
    </h2>
    <h3>
     <a id="_4">
     </a>
     常用数据类型
    </h3>
    <h4>
     <a id="Numbers_6">
     </a>
     Numbers（数值）
    </h4>
    <ul>
     <li>
      <strong>
       int
      </strong>
      ：表示整数。
     </li>
     <li>
      <strong>
       double
      </strong>
      ：表示浮点数。
     </li>
    </ul>
    <h4>
     <a id="Strings_11">
     </a>
     Strings（字符串）
    </h4>
    <ul>
     <li>
      <strong>
       String
      </strong>
      ：表示字符串。
     </li>
    </ul>
    <h4>
     <a id="Booleans_15">
     </a>
     Booleans（布尔）
    </h4>
    <ul>
     <li>
      <strong>
       bool
      </strong>
      ：表示布尔值，值为
      <code>
       true
      </code>
      或
      <code>
       false
      </code>
      。
     </li>
    </ul>
    <h4>
     <a id="List_19">
     </a>
     List（数组）
    </h4>
    <p>
     在 Dart 中，数组是列表对象，通常称为列表。
    </p>
    <h4>
     <a id="Maps_23">
     </a>
     Maps（字典）
    </h4>
    <p>
     Map 是一个键值对相关的对象，键和值可以是任何类型的对象。每个键只出现一次，而一个值可以出现多次。
    </p>
    <h3>
     <a id="_27">
