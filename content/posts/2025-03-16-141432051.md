      </li>

      </li>

       <font color="black">
        Git Bash的安装 （它允许在 Windows 上使用类似于 Linux/Unix 的命令行界面进行 Git 操作。）

    <blockquote>
     <h3>
      <a id="font_colorblack1_vscodeconda_12">
      </a>
      <font color="black">
       1 vscode下无法激活conda环境
      </font>
     </h3>
     <p>
      <font color="black">
       【
       <strong>
        原因
       </strong>
       】vscode终端默认使用的是PowerShell，该脚本环境下不支持激活conda环境。
       <br/>
       <font color="black">
        【
        <strong>
         解决方案
        </strong>
        】设置使用终端方式
        <br/>
        <font color="black">
         【
         <strong>
          具体操作
         </strong>
         】
        </font>
       </font>
      </font>
     </p>
     <ul>
      <li>
       <font color="black">
        1. 打开vscode，在vscode界面内打开搜索框 (快捷键
        <code>
         CTRL+SHIFT+P
        </code>
        )，在界面顶端会出现搜索框
        <br/>
        键入
        <code>
         terminal:select default profile
        </code>
        ，选择第一个即可。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/0f980ec8bdef4e3f9dc1aeac0df7da82.png#pic_center" width="600">
         <br/>
         <font color="black">
          2. 设置之后查看setting.json文件，会看到多了一行：
          <br/>
          <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a04d109f82ec4511a95284f507220e3a.png#pic_center" width="550">
           <br/>
           <br/>
           <font color="black">
            3. 重新打开终端，保证画圈的地方为【cmd】即可。
            <br/>
            <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/12273c8f94cb40d4b22a72183dee7e26.png#pic_center" width="550"/>
           </font>
          </img>
         </font>
        </img>
       </font>
      </li>
     </ul>
    </blockquote>
    <blockquote>
     <h3>
      <a id="2_font_colorblack_29">
      </a>
      2
      <font color="black">
       创建虚拟环境报错
      </font>
     </h3>
     <p>
      <font color="black">
       【
       <strong>
        现象
       </strong>
       】创建虚拟环境时会报错，如下图
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/446e8d05654f470c85f1218c68e7c6fd.png#pic_center" width="600">
        <br/>
        <font color="black">
         【
         <strong>
          解决具体步骤
         </strong>
         】
        </font>
       </img>
      </font>
     </p>
     <ol>
      <li>
       <font color="black">
        路径 C:\Users\own_name 下找到文件 .condarc。若该路径下没有该文件，则创建该文件：
       </font>
       <pre><code class="prism language-bash">echo. <span class="token operator">&gt;</span> .condarc
</code></pre>
      </li>
      <li>
       <font color="black">
        将里面的内容修改为
       </font>
       <pre><code class="prism language-bash">channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
show_channel_urls: <span class="token boolean">true</span>
</code></pre>
       <font color="black">
        若仍无法正确运行，则尝试文件中内容为
       </font>
       <pre><code class="prism language-bash">channels:
  - http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
  - http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
  - http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
  - http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/bioconda/
  - http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
show_channel_urls: <span class="token boolean">true</span>
</code></pre>
      </li>
      <li>
       <font color="black">
        重新打开终端，即可创建虚拟环境。
       </font>
      </li>
     </ol>
    </blockquote>
    <blockquote>
     <h3>
      <a id="font_colorblack3_pytorch_56">
      </a>
      <font color="black">
       3 安装pytorch
      </font>
     </h3>
     <p>
      <font color="black">
       激活虚拟环境，然后安装对应的库。
       <br/>
       <br/>
       <font color="blun">
        【
        <strong>
         安装pytorch-cpu
        </strong>
        】
       </font>
      </font>
     </p>
     <ul>
      <li>
       <font color="black">
        个人在windows下更多的是跑demo，所以一般安装的是cpu版本，安装依赖少速度也快。选用的命令如下图红框，然后使用的源替换成清华镜像。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/11a5e635398b41c98a8365ff125a4cef.png#pic_center" width="600">
         <br/>
         <font color="black">
          具体命令如下
         </font>
        </img>
       </font>
       <pre><code class="prism language-bash">pip <span class="token function">install</span> <span class="token assign-left variable">torch</span><span class="token operator">==</span><span class="token number">2.1</span>.1 <span class="token assign-left variable">torchvision</span><span class="token operator">==</span><span class="token number">0.16</span>.1 <span class="token assign-left variable">torchaudio</span><span class="token operator">==</span><span class="token number">2.1</span>.1  <span class="token parameter variable">-i</span> https://pypi.tuna.tsinghua.edu.cn/simple
</code></pre>
      </li>
     </ul>
     <p>
      <font color="blun">
       【
       <strong>
        安装pytorch-gpu
       </strong>
       】
      </font>
     </p>
     <ul>
      <li>
       <font color="black">
        若需要安装pytorch的gpu版本，首先要确认的是本地的cuda版本、显卡驱动版本，然后根据所需安装的pytorch版本看是否需要对其进行升级。
       </font>
       <ol>
        <li>
         <font color="black">
          打开命令提示符：按下Win + R键，输入cmd，然后回车打开命令提示符。
         </font>
        </li>
        <li>
         <font color="black">
          输入命令查看CUDA版本：
         </font>
         <ul>
          <li>
           <font color="black">
            可以输入nvcc --version或nvcc -V（两者作用相同，只是命令的全拼和缩写关系）来查看CUDA编译器的版本。如果系统提示找不到nvcc命令，可能是因为CUDA的bin目录没有添加到环境变量中，或者CUDA未正确安装。
           </font>
          </li>
          <li>
           <font color="black">
            另外，也可以使用nvidia-smi命令来查看NVIDIA驱动和CUDA支持的信息。
           </font>
          </li>
         </ul>
        </li>
        <li>
         <font color="black">
          根据想要安装的pytorch版本，来看是否需要升级显卡驱动和CUDA版本。然后在官网上找到对应的pytorch-gpu的安装命令安装即可。
         </font>
        </li>
       </ol>
      </li>
     </ul>
    </blockquote>
    <blockquote>
     <h3>
      <a id="font_colorblack4_Git_Bash_74">
      </a>
      <font color="black">
       4 Git Bash的安装
      </font>
     </h3>
     <p>
      <font color="black">
       Git Bash 是 Git 版本控制系统的一个组件，它允许在 Windows 上使用类似于 Linux/Unix 的命令行界面进行 Git 操作。以下是如何在 Windows 上安装 Git Bash 的简要步骤：
      </font>
     </p>
     <ol>
      <li>
       <font color="black">
        <strong>
         下载 Git Bash
        </strong>
        ：
        <br/>
        访问 Git 官方网站 (https://git-scm.com/downloads) 并下载适用于 Windows 的 Git Bash。
        <br/>
        一般来说下载速度很慢，可以去清华镜像进行下载安装包：
        <a href="https://mirrors.tuna.tsinghua.edu.cn/github-release/git-for-windows/git/Git%20for%20Windows%20v2.47.0.windows.1/" rel="nofollow">
         git-清华镜像下载
        </a>
       </font>
      </li>
      <li>
       <font color="black">
        运行安装程序：下载完成后，找到下载的文件（通常是一个名为 Git-2.xx.xx.exe 的文件，其中 xx.xx 是版本号），双击它以启动安装程序。
       </font>
      </li>
      <li>
       <font color="black">
        <strong>
         安装 Git Bash
        </strong>
        ：
        <br/>
        按照安装向导的指示进行操作。在安装过程中，你可以选择安装路径以及需要安装的组件。确保选中 “Git Bash Here”、“Git GUI”（图形界面）、“Git LFS”（大文件存储）的选项，如果需要的话也可以选择 “Associate Git Bash with .git extension”（将 Git Bash 关联到 .git 扩展名）。
       </font>
      </li>
      <li>
       <font color="black">
        <strong>
         打开Git Bash
        </strong>
        ：
        <br/>
        安装完成后，你可以通过双击桌面上的 “Git Bash” 快捷方式或在任何文件夹中右键点击选择 “Git Bash Here” 来启动 Git Bash 终端。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/d9f6257435224f9e8e885eae0a1fd96a.png#pic_center" width="300"/>
       </font>
      </li>
      <li>
       <font color="black">
        一个简单的命令行示例，展示如何在 Git Bash 中使用 Git 命令。
        <br/>
        示例命令中的【https://github.com/username/repository.git】替换为你要克隆的实际 Git 仓库地址，
        <br/>
        示例命令中的【 new-branch】替换为你想要创建的新分支名称。
       </font>
       <pre><code class="prism language-Bash"># 克隆一个仓库
git clone https://github.com/username/repository.git

# 进入仓库目录
cd repository

# 创建一个新的分支
git checkout -b new-branch

# 查看当前文件状态
git status

# 添加文件到暂存区
git add .

# 提交更改
git commit -m "Commit message"

# 推送到远程仓库
git push -u origin new-branch
</code></pre>
      </li>
     </ol>
    </blockquote>
