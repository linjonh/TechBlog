    <h2 align="center">
     keyboard scan code 表
    </h2>
    <hr/>
    <p>
     我发现搜到的许多其他键盘硬件扫描码的资料都与实际不符，而邓志先生整理的才是正确的。但邓志的MouseOS网站不知为何不能上去了，故转载google缓存的邓志的这篇文档以备查阅。
     <br/>
     转载来源
     <a href="http://www.mouseos.com/os/doc/scan_code.html" rel="nofollow noopener noreferrer" target="_blank">
      http://www.mouseos.com/os/doc/scan_code.html
     </a>
     <br/>
     作者邓志，对IA架构有深入研究，编有研究用的图形界面操作系统MouseOS
     <br/>
     <br/>
    </p>
    <p>
     当按下一键时，产生 mark 码，产生一次 IRQ1 中断。
     <br/>
     放开键时，产生 break 码，产生一次 IRQ1 中断。
     <br/>
     因此：
     <em>
      <strong>
       当按下 A 键放开，实际上产生了两次 IRQ1 中断
      </strong>
     </em>
     <br/>
     break 是 mark 码的 bit7 置 1 得来，也就是： break = mark + 0x80
     <br/>
    </p>
    <h4>
     1、scan code 集
    </h4>
    <p>
     这里只说一般日常应用中的键盘码集 set 2（有 set 1、set 2 以及 set 3）
    </p>
    <ul>
     <li>
      基本 scan code：绝大多数 scan code 是 1 byte 的。
     </li>
     <li>
      扩展 scan code：由
      <strong>
       e0
      </strong>
      、
      <strong>
       e1
      </strong>
      或
      <strong>
       e2
      </strong>
      引导
     </li>
     <li>
      特殊的 PrintScreen/SysRq 键：
      <strong>
       e0 2a e0 37
      </strong>
     </li>
     <li>
      物殊的 Pause/Break 键：
      <strong>
       e1 1d 45 e1 9d c5
      </strong>
     </li>
    </ul>
    <p>
    </p>

    <p>
     小键盘中的 &lt;home&gt;&lt;Up&gt;&lt;PgUp&gt;&lt;Left&gt;&lt;Right&gt;&lt;End&gt;&lt;Down&gt;&lt;PgDn&gt;&lt;Ins&gt;&lt;Del&gt; 由 NumLock 按下后再按得来。
     <br/>
     因此，会产生一个 &lt;NumLock&gt; mark 码后，再产生上述的 mark 码。 释放 &lt;NumLock&gt; 又产生一个 break 码。
