# 数据库操作常用的词汇

## ****数据库常用的词汇****

## ******数据表******

1. SHOW DATABASE;查看当前有哪些数据库
2. CREATE DATABASE ....;创建数据库
3. DROP DATABASE......;删除数据库
4. mysql -u root -p;登录mysql
5. USE ......;使用数据库
6. CREATE TABLE ...;创建表
7. PRIMARY KEY 主键约束
8. FOREIGN KEY ;外键约束
9. NOT NULL非空
10. UNIQUE 唯一性约束
11. DAFAULT 设置默认值
12. AUTO\_INCREMENT 自增
13. DESCRIBE/DESC 查看表的基本结构
14. SHOW CREATE TABLE...\G ;查看表的详细结构
15. ALTER TABLE .... RENAME TO ......更改表名
16. ALTER TABLE .... MODIFY ..... ...... 刚改字段名，数据类型
17. ALTER TABLE... CHANGE .......更改字段名
18. ALTER TABLE...... ADD......添加字段
19. ALTER TABLE...DROP字段名  ；删除表中的字段名
20. ALTER TABLE .... DEOP FORETGN KEY ... 删除外键
21. DROP TABLE 删除没有关联的表

## ******增删改查******

1. SELECT基本的查询
2. FROM 条件约束
3. BETWEEN AND 范围查询
4. LIKE 字符匹配
5. IS NULL 空值
6. ADN 多条件查询
7. OR 或 多条件查询
8. SELECT DISTINCT 查询结果不重复
9. ORDER BY 对查询结果排序
10. DESC 降序
11. ASC升序
12. INSERT INTO插入数据
13. UPDATE 更新数据
14. DELETE 删除数据
