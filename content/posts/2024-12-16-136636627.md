     一、简介
    </h2>
    <p>
     人工智能（AI）是一门研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统的新技术科学。它是计算机科学的一个分支，旨在生产出一种能以人类智能相似的方式做出反应的智能机器。人工智能的应用领域包括机器人、语言识别、图像识别、自然语言处理和专家系统等，它已逐渐成为许多领域的关键技术，如自动驾驶、智能推荐、医疗诊断等。人工智能可以对人的意识、思维的信息过程进行模拟，其研究范围涵盖了计算机科学、心理学、哲学等多个学科。人工智能可以分为弱人工智能和强人工智能，是一个包括十分广泛的科学，由不同的领域组成，如机器学习、计算机视觉等。总之，人工智能旨在使机器能够胜任一些通常需要人类智能才能完成的复杂工作，为计算机系统赋予类似于人类智能的能力，以实现更高效、智能化的解决方案。

    <h3>
     <a id="34AISD_31">
     </a>
     3.4、AI绘画SD
    </h3>
    <p>
     Stable Diffusion是一个文本到图像的潜在扩散模型，由CompVis、Stability AI和LAION的研究人员和工程师创建。它使用来自LAION-5B数据库子集的512x512图像进行训练，可以生成包括人脸在内的任何图像。
     <br/>
     这个模型的主要特点是它能够逐步对随机高斯噪声进行去噪，从而获得感兴趣的样本，例如生成图像。它采用潜在扩散模型（latent diffusion model；LDM）进行模型训练，通过在较低维度的潜空间上应用扩散过程而不是使用实际的像素空间来减少内存和计算成本。
     <br/>
     Stable Diffusion也可以用于其他任务，如内补绘制、外补绘制，以及在提示词（英语）指导下产生图生图的翻译。
     <br/>
     训练的神经网络通常为U-Net，在Stable Diffusion模型中使用的U-Net模型中的解码器与编码器之间增加了用于对文本嵌入的输出进行调节的交叉注意层。虽然Stable Diffusion具有生成高质量图像的能力，但是其去噪过程的时间和内存消耗非常大，主要原因是它在像素空间中运行，特别是在生成高分辨率图像时。
     <br/>
     效果图如下：
     <br/>
     <img alt="图片13" src="https://i-blog.csdnimg.cn/blog_migrate/4ffdb62704ce997c124a7494e4ff2a45.png#pic_center"/>
    </p>
    <ol>
     <li>
      提示词（描述词）：用户通过输入提示词，告诉AI绘画模型自己想要的画面效果，例如颜色、形状、风格等。AI绘画模型会根据这些提示词以及自己的学习数据，生成符合要求的艺术作品。提示词的编写对于AI绘画的效果至关重要。有时候，为了得到更精细、更符合预期的画面效果，用户需要精心编写提示词，甚至需要使用到反向提示词等技巧
     </li>
     <li>
      反向提示词（负标签）：是指为了获得某种特定的风格或效果，对AI绘画模型输入与预期效果相反的提示词。例如，如果想要一幅看起来很“艺术”的画，可能会给AI输入反向提示词“不自然、抽象、扭曲”，以获得独特和抽象的艺术效果。这种反向提示词的使用方法并非固定，主要取决于用户想要获得的效果，以及对AI绘画模型的理解。
     </li>
     <li>
      AI绘画迭代数：是指AI绘画过程中，对一张有瑕疵或不太满意的初稿进行多次渲染和优化的次数。每次迭代都是在上一次的基础上进行，通过不断优化画面细节，以达到最佳的视觉效果。AI绘画的迭代数越多，生成的图片质量通常会越高，但同时也会增加计算资源和时间成本。因此，在实际应用中需要根据需求和资源限制选择合适的迭代数。
     </li>
     <li>
      AI绘画采样器：是一种工具，利用人工智能技术生成艺术作品的样本。通过学习大量的艺术作品，分析其风格、色彩和构图等特征，AI绘画采样器可以生成新的艺术作品。这种工具可以帮助艺术家获得灵感，提供创作的参考，也可以用于艺术教育和研究。AI绘画采样器的出现使得艺术创作更加多样化和创新化，为艺术家提供了更多的可能性和创作空间。
     </li>
     <li>
      AI绘画随机种子：是一个数字序列，用于控制生成图像时使用的随机值。随机种子可以确保每次生成的图像都是随机的、独特的，也可以让生成的图像更加复杂。在某些情况下，设置特定的随机种子可以使得后续生成的图画基本保持一致，便于重复生成相同风格的图像。因此，随机种子在AI绘画中起到重要的作用。
     </li>
    </ol>

     3.5、文生图

     AI绘画文生图是一种基于人工智能技术的绘画生成方法。它通过分析大量艺术作品，学习其中的风格和特征，然后根据用户的文字描述或图像输入，生成符合要求的艺术作品。
     <br/>
     操作步骤：
    </p>
    <ol>
     <li>
      在提示词输入框处描述你想生成的画面。
     </li>
     <li>
      选择模型。
     </li>
     <li>
      设置参数（可不调，默认即可）
     </li>
     <li>
      点击生成等待出图。
     </li>
