# Wing IDE 如何设置 python版本

机器上同时装了Python3和Python2，使用
Wing IDE
, 因为Python2和3是有很大的区别的，所以时不时的需要更改IDE使用的Python版本，下面介绍方法：

1.打开
Edit
标签下的
Configure Python...
，如下图设置所用的，Python版本安装的目录即可

（Python Executable 和 Python Path 这个两个）

![](https://img-blog.csdn.net/20150821210509716?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast)

2.若使用默认的版本，再改回
Use default
模式即可
