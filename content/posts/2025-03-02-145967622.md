    <h4>
     <a id="1__31">
     </a>
     <strong>
      1. 网站功能设计
     </strong>
    </h4>
    <p>
     我们的网站将包含以下功能：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        首页
       </strong>
       ：展示欢迎信息和网站介绍（静态网页）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        留言板
       </strong>
       ：用户可以提交留言，留言会保存到数据库（动态网页 + 数据库）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        关于我们
       </strong>
       ：介绍网站的背景信息（静态网页）。
      </p>
     </li>
     <li>
      <p>
       <strong>
        JavaScript 交互
       </strong>
       ：表单验证、动态内容更新等。
      </p>
     </li>
    </ul>
