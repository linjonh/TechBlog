     PHP语法基础
    </h2>
    <h3>
     <a id="_2">
     </a>
     一，变量
    </h3>
    <p>
     在PHP中，
     <strong>
      变量
     </strong>
     是存储数据的容器，其灵活性和动态类型系统是PHP的核心特性之一。以下是PHP变量的详细解析，涵盖声明、作用域、类型转换及最佳实践：
    </p>
    <hr/>
    <h4>
     <a id="1__8">
     </a>
     <strong>
      1. 变量基础
     </strong>
    </h4>
    <h5>
     <a id="_10">
     </a>
     <strong>
      声明与命名规则
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        无需显式声明类型
       </strong>
       ：PHP是动态类型语言，变量类型由赋值决定。
      </p>
     </li>
     <li>
      <p>
       <strong>
        以
        <code>
         $
        </code>
        符号开头
       </strong>
       ：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$name</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Alice"</span><span class="token punctuation">;</span>  <span class="token comment">// 字符串</span>
<span class="token variable">$age</span> <span class="token operator">=</span> <span class="token number">25</span><span class="token punctuation">;</span>        <span class="token comment">// 整数</span>
<span class="token variable">$price</span> <span class="token operator">=</span> <span class="token number">19.99</span><span class="token punctuation">;</span>   <span class="token comment">// 浮点数</span>
<span class="token variable">$isActive</span> <span class="token operator">=</span> <span class="token constant boolean">true</span><span class="token punctuation">;</span> <span class="token comment">// 布尔值</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        命名规则
       </strong>
       ：
      </p>
      <ul>
       <li>
        以字母或下划线开头，后可接字母、数字、下划线。
       </li>
       <li>
        区分大小写（
        <code>
         $Var
        </code>
        与
        <code>
         $var
        </code>
        不同）。
       </li>
      </ul>
     </li>
    </ul>
    <h5>
     <a id="_28">
     </a>
     <strong>
      变量类型
     </strong>
    </h5>
    <p>
     PHP支持8种原始数据类型：
    </p>
    <table>
     <thead>
      <tr>
       <th align="left">
        <strong>
         类型
        </strong>
       </th>

         描述
        </strong>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td align="left">
        整型（int）
       </td>
       <td align="left">
        <code>
         $count = 100;
        </code>
       </td>
       <td align="left">
        无小数点的整数
       </td>
      </tr>
      <tr>
       <td align="left">
        浮点型（float）
       </td>
       <td align="left">
        <code>
         $price = 9.99;
        </code>
       </td>
       <td align="left">
        含小数点的数或科学计数法表示
       </td>
      </tr>
      <tr>
       <td align="left">
        字符串（string）
       </td>
       <td align="left">
        <code>
         $text = "Hello";
        </code>
       </td>
       <td align="left">
        文本（单引号或双引号包裹）
       </td>
      </tr>
      <tr>
       <td align="left">
        布尔型（bool）
       </td>
       <td align="left">
        <code>
         $isValid = true;
        </code>
       </td>
       <td align="left">
        <code>
         true
        </code>
        或
        <code>
         false
        </code>
       </td>
      </tr>
      <tr>
       <td align="left">
        数组（array）
       </td>
       <td align="left">
        <code>
         $colors = ["red", "blue"];
        </code>
       </td>
       <td align="left">
        有序键值对集合
       </td>
      </tr>
      <tr>
       <td align="left">
        对象（object）
       </td>
       <td align="left">
        <code>
         $user = new User();
        </code>
       </td>
       <td align="left">
        类的实例化
       </td>
      </tr>
      <tr>
       <td align="left">
        NULL
       </td>
       <td align="left">
        <code>
         $data = null;
        </code>
       </td>
       <td align="left">
        表示变量无值
       </td>
      </tr>
      <tr>
       <td align="left">
        资源（resource）
       </td>
       <td align="left">
        <code>
         $file = fopen("a.txt");
        </code>
       </td>
       <td align="left">
        外部资源句柄（如文件、数据库连接）
       </td>
      </tr>

    <h4>
     <a id="2__45">
     </a>
     <strong>
      2. 变量作用域
     </strong>
    </h4>
    <h5>
     <a id="_47">
     </a>
     <strong>
      作用域分类
     </strong>
    </h5>
    <ol>
     <li>
      <p>
       <strong>
        局部变量
       </strong>
       ：
       <br/>
       在函数内部声明，仅函数内有效。
      </p>
      <pre><code class="prism language-php"><span class="token keyword">function</span> <span class="token function-definition function">test</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token variable">$localVar</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Inside function"</span><span class="token punctuation">;</span> <span class="token comment">// 局部变量</span>
    <span class="token keyword">echo</span> <span class="token variable">$localVar</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token function">test</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "Inside function"</span>
<span class="token keyword">echo</span> <span class="token variable">$localVar</span><span class="token punctuation">;</span> <span class="token comment">// 报错：未定义变量</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        全局变量
       </strong>
       ：
       <br/>
       在函数外部声明，需通过
       <code>
        global
       </code>
       关键字或
       <code>
        $GLOBALS
       </code>
       数组访问。
      </p>
      <pre><code class="prism language-php"><span class="token variable">$globalVar</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Outside function"</span><span class="token punctuation">;</span>
<span class="token keyword">function</span> <span class="token function-definition function">readGlobal</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">global</span> <span class="token variable">$globalVar</span><span class="token punctuation">;</span>
    <span class="token keyword">echo</span> <span class="token variable">$globalVar</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "Outside function"</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        静态变量
       </strong>
       ：
       <br/>
       使用
       <code>
        static
       </code>
       关键字，函数多次调用时保留值。
      </p>
      <pre><code class="prism language-php"><span class="token keyword">function</span> <span class="token function-definition function">counter</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">static</span> <span class="token variable">$count</span> <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>
    <span class="token variable">$count</span><span class="token operator">++</span><span class="token punctuation">;</span>
    <span class="token keyword">echo</span> <span class="token variable">$count</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
<span class="token function">counter</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 1</span>
<span class="token function">counter</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 2</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        超全局变量
       </strong>
       ：
       <br/>
       预定义全局数组，跨作用域直接访问，如：
      </p>
      <ul>
       <li>
        <code>
         $_GET
        </code>
        ：获取URL参数。
       </li>
       <li>
        <code>
         $_POST
        </code>
        ：接收表单POST数据。
       </li>
       <li>
        <code>
         $_SESSION
        </code>
        ：会话变量。
       </li>
       <li>
        <code>
         $_SERVER
        </code>
        ：服务器信息（如
        <code>
         $_SERVER['REQUEST_URI']
        </code>
        ）。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="3__95">
     </a>
     <strong>
      3. 变量处理技巧
     </strong>
    </h4>
    <h5>
     <a id="Variable_Variables_97">
     </a>
     <strong>
      可变变量（Variable Variables）
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       通过变量名动态访问变量：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$varName</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"message"</span><span class="token punctuation">;</span>
<span class="token variable">$$varName</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"Hello!"</span><span class="token punctuation">;</span> <span class="token comment">// 等价于 $message = "Hello!";</span>
<span class="token keyword">echo</span> <span class="token variable">$message</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "Hello!"</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="By_Reference_107">
     </a>
     <strong>
      引用变量（By Reference）
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       使用
       <code>
        &amp;
       </code>
       创建变量别名，修改引用会同步原变量：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$a</span> <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token operator">&amp;</span><span class="token variable">$a</span><span class="token punctuation">;</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$a</span><span class="token punctuation">;</span> <span class="token comment">// 输出 20</span>
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="_118">
     </a>
     <strong>
      类型检测与转换
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        检测类型
       </strong>
       ：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$value</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"123"</span><span class="token punctuation">;</span>
<span class="token function">var_dump</span><span class="token punctuation">(</span><span class="token function">is_int</span><span class="token punctuation">(</span><span class="token variable">$value</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span>   <span class="token comment">// bool(false)</span>
<span class="token function">var_dump</span><span class="token punctuation">(</span><span class="token function">is_numeric</span><span class="token punctuation">(</span><span class="token variable">$value</span><span class="token punctuation">)</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// bool(true)</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        强制类型转换
       </strong>
       ：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$num</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"42.5"</span><span class="token punctuation">;</span>    <span class="token comment">// 转为整型（42）</span>
<span class="token variable">$str</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">string</span><span class="token punctuation">)</span><span class="token number">100</span><span class="token punctuation">;</span>    <span class="token comment">// 转为字符串（"100"）</span>
</code></pre>
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="4__137">
     </a>
     <strong>
      4. 安全性注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        未初始化变量
       </strong>
       ：
       <br/>
       直接使用未赋值变量会触发
       <code>
        E_WARNING
       </code>
       ，建议初始化：
      </p>
      <pre><code class="prism language-php"><span class="token comment">// 错误示例</span>
<span class="token keyword">echo</span> <span class="token variable">$undefinedVar</span><span class="token punctuation">;</span> <span class="token comment">// 警告：Undefined variable</span>
<span class="token comment">// 正确做法</span>
<span class="token variable">$undefinedVar</span> <span class="token operator">=</span> <span class="token keyword">isset</span><span class="token punctuation">(</span><span class="token variable">$undefinedVar</span><span class="token punctuation">)</span> <span class="token operator">?</span> <span class="token variable">$undefinedVar</span> <span class="token punctuation">:</span> <span class="token string double-quoted-string">"default"</span><span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        用户输入过滤
       </strong>
       ：
       <br/>
       对
       <code>
        $_GET
       </code>
       、
       <code>
        $_POST
       </code>
       等输入进行过滤，避免安全漏洞：
      </p>
      <pre><code class="prism language-php"><span class="token variable">$username</span> <span class="token operator">=</span> <span class="token function">filter_input</span><span class="token punctuation">(</span><span class="token constant">INPUT_POST</span><span class="token punctuation">,</span> <span class="token string single-quoted-string">'username'</span><span class="token punctuation">,</span> <span class="token constant">FILTER_SANITIZE_STRING</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        避免直接输出用户数据
       </strong>
       ：
       <br/>
       防止XSS攻击，使用
       <code>
        htmlspecialchars
       </code>
       转义：
      </p>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token function">htmlspecialchars</span><span class="token punctuation">(</span><span class="token variable">$_POST</span><span class="token punctuation">[</span><span class="token string single-quoted-string">'comment'</span><span class="token punctuation">]</span><span class="token punctuation">,</span> <span class="token constant">ENT_QUOTES</span><span class="token punctuation">,</span> <span class="token string single-quoted-string">'UTF-8'</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="5__165">
     </a>
     <strong>
      5. 最佳实践
     </strong>
    </h4>
    <ul>
     <li>
      <p>
       <strong>
        命名规范
       </strong>
       ：
       <br/>
       使用驼峰式（
       <code>
        $userName
       </code>
       ）或蛇形命名（
       <code>
        $user_name
       </code>
       ），保持一致性。
      </p>
     </li>
     <li>
      <p>
       <strong>
        类型声明（PHP 7+）
       </strong>
       ：
       <br/>
       通过严格类型提升代码可预测性：
      </p>
      <pre><code class="prism language-php"><span class="token keyword">declare</span><span class="token punctuation">(</span>strict_types<span class="token operator">=</span><span class="token number">1</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 启用严格模式</span>
<span class="token keyword">function</span> <span class="token function-definition function">add</span><span class="token punctuation">(</span><span class="token keyword type-hint">int</span> <span class="token variable">$a</span><span class="token punctuation">,</span> <span class="token keyword type-hint">int</span> <span class="token variable">$b</span><span class="token punctuation">)</span><span class="token punctuation">:</span> <span class="token keyword return-type">int</span> <span class="token punctuation">{<!-- --></span>
    <span class="token keyword">return</span> <span class="token variable">$a</span> <span class="token operator">+</span> <span class="token variable">$b</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        使用
        <code>
         ===
        </code>
        比较
       </strong>
       ：
       <br/>
       避免类型转换导致的意外结果：
      </p>
      <pre><code class="prism language-php"><span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token number">0</span> <span class="token operator">==</span> <span class="token string double-quoted-string">"0"</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token comment">// true（松散比较）</span>
<span class="token keyword">if</span> <span class="token punctuation">(</span><span class="token number">0</span> <span class="token operator">===</span> <span class="token string double-quoted-string">"0"</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token comment">// false（严格比较）</span>
</code></pre>
     </li>
    </ul>

    <h3>
     <a id="_202">
     </a>
     二，常量
    </h3>
    <p>
     在PHP中，
     <strong>
      常量
     </strong>
     是一种不可变的值，一旦定义后无法修改或重新声明。常量的全局性和持久性使其适合存储配置参数、固定值或全局标识符。以下是PHP常量的详细解析，涵盖定义方式、作用域、使用场景及最佳实践：
    </p>
    <hr/>
    <h4>
     <a id="1__208">
     </a>
     <strong>
      1. 常量的定义方式
     </strong>
    </h4>
    <h5>
     <a id="1__define__210">
     </a>
     <strong>
      (1) 使用
      <code>
       define()
      </code>
      函数
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        语法
       </strong>
       ：
      </p>
      <pre><code class="prism language-php"><span class="token function">define</span><span class="token punctuation">(</span><span class="token keyword type-hint">string</span> <span class="token variable">$name</span><span class="token punctuation">,</span> <span class="token keyword type-hint">mixed</span> <span class="token variable">$value</span> <span class="token punctuation">[</span><span class="token punctuation">,</span> <span class="token keyword type-hint">bool</span> <span class="token variable">$case_insensitive</span> <span class="token operator">=</span> <span class="token constant boolean">false</span><span class="token punctuation">]</span><span class="token punctuation">)</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        特点
       </strong>
       ：
      </p>
      <ul>
       <li>
        可在
        <strong>
         运行时动态定义
        </strong>
        （例如在条件语句或函数内）。
       </li>
       <li>
        支持定义
        <strong>
         大小写不敏感
        </strong>
        的常量（不推荐，PHP 7.3+已废弃此特性）。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-php"><span class="token function">define</span><span class="token punctuation">(</span><span class="token string double-quoted-string">"SITE_NAME"</span><span class="token punctuation">,</span> <span class="token string double-quoted-string">"MyWebsite"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token function">define</span><span class="token punctuation">(</span><span class="token string double-quoted-string">"MAX_USERS"</span><span class="token punctuation">,</span> <span class="token number">1000</span><span class="token punctuation">,</span> <span class="token constant boolean">true</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 不推荐使用大小写不敏感</span>
<span class="token keyword">echo</span> <span class="token constant">SITE_NAME</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "MyWebsite"</span>
</code></pre>
    <h5>
     <a id="2__const__231">
     </a>
     <strong>
      (2) 使用
      <code>
       const
      </code>
      关键字
     </strong>
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        语法
       </strong>
       ：
      </p>
      <pre><code>const CONSTANT_NAME = value;
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        特点
       </strong>
       ：
      </p>
      <ul>
       <li>
        必须在
        <strong>
         编译时定义
        </strong>
        （不能在函数、循环或条件语句中使用）。
       </li>
       <li>
        代码更简洁，推荐在类或全局作用域使用。
       </li>
      </ul>
     </li>
    </ul>
    <p>
     <strong>
      示例
     </strong>
     ：
    </p>
    <pre><code class="prism language-php"><span class="token keyword">const</span> <span class="token constant">API_KEY</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"abc123"</span><span class="token punctuation">;</span>
<span class="token keyword">const</span> <span class="token constant">PI</span> <span class="token operator">=</span> <span class="token number">3.14159</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token constant">PI</span><span class="token punctuation">;</span> <span class="token comment">// 输出 3.14159</span>
</code></pre>

      </strong>
      ：

        <code>

         MYAPP_LOGO_PATH
        </code>
        ）。
       </li>
      </ul>
     </li>

      </p>

    <h3>
     <a id="_468">
     </a>
     三，数据类型分类
    </h3>
    <p>
     PHP的数据类型可以分为标量类型、复合类型、特殊类型以及伪类型。以下是详细分类和说明：
    </p>
    <hr/>
    <h4>
     <a id="Scalar_Types_474">
     </a>
     <strong>
      一、标量类型（Scalar Types）
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        整型（Integer）
       </strong>
      </p>
      <ul>
       <li>
        表示整数，带符号（支持正负），取值范围依赖于平台（32位或64位）。
       </li>
       <li>
        可用进制：十进制（
        <code>
         123
        </code>
        ）、十六进制（
        <code>
         0x1A
        </code>
        ）、八进制（
        <code>
         0123
        </code>
        ）、二进制（
        <code>
         0b1110
        </code>
        ）。
       </li>
       <li>
        示例：
        <code>
         $a = 42;
        </code>
       </li>
       <li>
        类型检测：
        <code>
         is_int()
        </code>
        或
        <code>
         is_integer()
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        浮点型（Float/Double）
       </strong>
      </p>
      <ul>
       <li>
        表示小数或科学计数法数值（如
        <code>
         1.2e3
        </code>
        ）。
       </li>
       <li>
        精度问题：浮点数计算可能存在精度丢失（如
        <code>
         0.1 + 0.2 != 0.3
        </code>
        ）。
       </li>
       <li>
        示例：
        <code>
         $b = 3.14;
        </code>
       </li>
       <li>
        类型检测：
        <code>
         is_float()
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        字符串（String）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         表示文本，支持单引号（不解析变量）、双引号（解析变量和转义字符）、Heredoc和Nowdoc语法。
        </p>
       </li>
       <li>
        <p>
         示例：
        </p>
        <pre><code class="prism language-php"><span class="token variable">$str1</span> <span class="token operator">=</span> <span class="token string single-quoted-string">'Hello'</span><span class="token punctuation">;</span>  
<span class="token variable">$str2</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"World: <span class="token interpolation"><span class="token variable">$str1</span></span>"</span><span class="token punctuation">;</span>  
<span class="token variable">$str3</span> <span class="token operator">=</span> <span class="token operator">&lt;&lt;</span><span class="token operator">&lt;</span><span class="token constant">EOD</span>  
Multiline  
<span class="token constant">EOD</span><span class="token punctuation">;</span>  
</code></pre>
       </li>
       <li>
        <p>
         类型检测：
         <code>
          is_string()
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        布尔型（Boolean）
       </strong>
      </p>
      <ul>
       <li>
        仅两个值：
        <code>
         true
        </code>
        和
        <code>
         false
        </code>
        （不区分大小写，如
        <code>
         TRUE
        </code>
        也有效）。
       </li>
       <li>
        转换为
        <code>
         false
        </code>
        的情况：空字符串、
        <code>
         0
        </code>
        、
        <code>
         null
        </code>
        、空数组等。
       </li>
       <li>
        示例：
        <code>
         $flag = true;
        </code>
       </li>
       <li>
        类型检测：
        <code>
         is_bool()
        </code>
        。
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="Compound_Types_515">
     </a>
     <strong>
      二、复合类型（Compound Types）
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        数组（Array）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         有序键值对集合，键可以是整型或字符串，值可为任意类型。

       </li>
       <li>
        <p>
         示例：
        </p>
        <pre><code class="prism language-php"><span class="token variable">$arr</span> <span class="token operator">=</span> <span class="token punctuation">[</span><span class="token number">1</span> <span class="token operator">=&gt;</span> <span class="token string double-quoted-string">"a"</span><span class="token punctuation">,</span> <span class="token string double-quoted-string">"name"</span> <span class="token operator">=&gt;</span> <span class="token string double-quoted-string">"PHP"</span><span class="token punctuation">]</span><span class="token punctuation">;</span>
</code></pre>
       </li>
       <li>
        <p>
         类型检测：
         <code>
          is_array()
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        对象（Object）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         类的实例，包含属性和方法。
        </p>
       </li>
       <li>
        <p>
         示例：
        </p>
        <pre><code class="prism language-php"><span class="token keyword">class</span> <span class="token class-name-definition class-name">MyClass</span> <span class="token punctuation">{<!-- --></span><span class="token punctuation">}</span>  
<span class="token variable">$obj</span> <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token class-name">MyClass</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
       </li>
       <li>
        <p>
         类型检测：
         <code>
          is_object()
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        可调用（Callable）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         表示可调用的结构，如函数、方法、闭包或实现
         <code>
          __invoke()
         </code>
         的对象。
        </p>
       </li>
       <li>
        <p>
         示例：
        </p>
        <pre><code class="prism language-php"><span class="token variable">$callable</span> <span class="token operator">=</span> <span class="token keyword">function</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">echo</span> <span class="token string double-quoted-string">"Hello"</span><span class="token punctuation">;</span> <span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
       </li>
       <li>
        <p>
         类型检测：
         <code>
          is_callable()
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        可迭代（Iterable）
       </strong>
      </p>
      <ul>
       <li>
        <p>
         PHP 7.1+ 引入，可以是数组或实现
         <code>
          Traversable
         </code>
         接口的对象（如生成器）。
        </p>
       </li>
       <li>
        <p>
         示例：
        </p>
        <pre><code class="prism language-php"><span class="token keyword">function</span> <span class="token function-definition function">gen</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span> <span class="token keyword return-type">iterable</span> <span class="token punctuation">{<!-- --></span> <span class="token keyword">yield</span> <span class="token number">1</span><span class="token punctuation">;</span> <span class="token punctuation">}</span>
</code></pre>
       </li>
       <li>
        <p>
         类型检测：
         <code>
          is_iterable()
         </code>
         。
        </p>
       </li>
      </ul>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="Special_Types_568">
     </a>
     <strong>
      三、特殊类型（Special Types）
     </strong>
    </h4>
    <ol>
     <li>
      <strong>
       空值（Null）
      </strong>
      <ul>
       <li>
        唯一值
        <code>
         null
        </code>
        ，表示变量未赋值或已销毁（
        <code>
         unset()
        </code>
        ）。
       </li>
       <li>
        示例：
        <code>
         $var = null;
        </code>
       </li>
       <li>
        类型检测：
        <code>
         is_null()
        </code>
        。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       资源（Resource）
      </strong>
      <ul>
       <li>
        外部资源句柄（如文件、数据库连接），由扩展创建。
       </li>
       <li>
        示例：
        <code>
         $file = fopen("test.txt", "r");
        </code>
       </li>
       <li>
        类型检测：
        <code>
         is_resource()
        </code>
        。资源释放后变量可能转为
        <code>
         null
        </code>
        。
       </li>
      </ul>
     </li>
    </ol>

      </code>
      转整型为
      <code>
       123
      </code>
      ，
      <code>
       "abc123"
      </code>
      转整型为
      <code>
       0
      </code>
      。
     </li>
     <li>
      <strong>
       数组键类型
      </strong>
      <br/>
      字符串键
      <code>
       "8"
      </code>
      会转为整型
      <code>
       8
      </code>
      ，但
      <code>
       "08"
      </code>
      保留为字符串。
     </li>
     <li>
      <strong>
       资源管理
      </strong>
      <br/>
      资源需显式释放（如
      <code>
       fclose()
      </code>
      ），释放后变量可能变为
      <code>
       null
      </code>
      。
     </li>
     <li>
      <strong>
       PHP版本差异
      </strong>
      <br/>
      PHP 8 中部分资源改为对象（如
      <code>
       PDO
      </code>
      相关），需注意兼容性。
     </li>
    </ul>
    <hr/>
    <h3>
     <a id="_605">
     </a>
     四，数据类型转换
    </h3>
    <h4>
     <a id="1_607">
     </a>
     <strong>
      1.自动类型转换（隐式转换）
     </strong>
    </h4>
    <p>
     PHP 在表达式求值时会自动转换数据类型，例如：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$sum</span> <span class="token operator">=</span> <span class="token number">5</span> <span class="token operator">+</span> <span class="token string double-quoted-string">"10"</span><span class="token punctuation">;</span>  <span class="token comment">// 15（"10" 被转换为整数）</span>
<span class="token variable">$sum</span> <span class="token operator">=</span> <span class="token number">5</span> <span class="token operator">+</span> <span class="token string double-quoted-string">"10.5"</span><span class="token punctuation">;</span> <span class="token comment">// 15.5（"10.5" 被转换为浮点数）</span>
</code></pre>
    <p>
     <strong>
      转换规则：
     </strong>
    </p>
    <ol>
     <li>
      <p>
       <strong>
        字符串 → 数字
       </strong>
      </p>
      <ul>
       <li>
        只要字符串以数字开头，PHP 就会转换它。
       </li>
       <li>
        遇到非数字字符时，转换会停止。
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"123abc"</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 123</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"abc123"</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 0（无效转换）</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        布尔值转换
       </strong>
      </p>
      <ul>
       <li>
        <code>
         false
        </code>
        转换为
        <code>
         0
        </code>
        ，
        <code>
         true
        </code>
        转换为
        <code>
         1
        </code>
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token constant boolean">true</span> <span class="token operator">+</span> <span class="token number">1</span><span class="token punctuation">;</span>  <span class="token comment">// 输出 2</span>
<span class="token keyword">echo</span> <span class="token constant boolean">false</span> <span class="token operator">+</span> <span class="token number">1</span><span class="token punctuation">;</span> <span class="token comment">// 输出 1</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        空字符串、
        <code>
         NULL
        </code>
        转换
       </strong>
      </p>
      <ul>
       <li>
        空字符串
        <code>
         ""
        </code>
        和
        <code>
         NULL
        </code>
        转换为
        <code>
         0
        </code>
        。
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token constant">NULL</span><span class="token punctuation">;</span>    <span class="token comment">// 0</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">""</span><span class="token punctuation">;</span>      <span class="token comment">// 0</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        数组转换
       </strong>
      </p>
      <ul>
       <li>
        数组转换为整数时，始终返回
        <code>
         1
        </code>
        （如果是空数组，则返回
        <code>
         0
        </code>
        ）。
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token punctuation">[</span><span class="token number">1</span><span class="token punctuation">,</span> <span class="token number">2</span><span class="token punctuation">,</span> <span class="token number">3</span><span class="token punctuation">]</span><span class="token punctuation">;</span> <span class="token comment">// 1</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token punctuation">[</span><span class="token punctuation">]</span><span class="token punctuation">;</span>        <span class="token comment">// 0</span>
</code></pre>
     </li>
    </ol>
    <hr/>
    <h4>
     <a id="2__657">
     </a>
     <strong>
      2. 显式类型转换（强制转换）
     </strong>
    </h4>
    <p>
     显式转换使用
     <code>
      (type)
     </code>
     语法：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$var</span> <span class="token operator">=</span> <span class="token string double-quoted-string">"100"</span><span class="token punctuation">;</span>
<span class="token variable">$intVar</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token variable">$var</span><span class="token punctuation">;</span>      <span class="token comment">// 100</span>
<span class="token variable">$floatVar</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">float</span><span class="token punctuation">)</span><span class="token variable">$var</span><span class="token punctuation">;</span>  <span class="token comment">// 100.0</span>
<span class="token variable">$boolVar</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">bool</span><span class="token punctuation">)</span><span class="token variable">$var</span><span class="token punctuation">;</span>    <span class="token comment">// true</span>
</code></pre>
    <p>
     PHP 提供以下显式转换：
    </p>
    <ul>
     <li>
      <code>
       (int) 或 (integer)
      </code>
      - 转换为整数
     </li>
     <li>
      <code>
       (bool) 或 (boolean)
      </code>
      - 转换为布尔值
     </li>
     <li>
      <code>
       (float) 或 (double) 或 (real)
      </code>
      - 转换为浮点数
     </li>
     <li>
      <code>
       (string)
      </code>
      - 转换为字符串
     </li>
     <li>
      <code>
       (array)
      </code>
      - 转换为数组
     </li>
     <li>
      <code>
       (object)
      </code>
      - 转换为对象
     </li>
     <li>
      <code>
       (unset)
      </code>
      - 转换为
      <code>
       NULL
      </code>
     </li>
    </ul>
    <p>
     <strong>
      示例：
     </strong>
    </p>
    <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"123abc"</span><span class="token punctuation">;</span>  <span class="token comment">// 123</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">bool</span><span class="token punctuation">)</span><span class="token number">0</span><span class="token punctuation">;</span>        <span class="token comment">// false</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">float</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"3.14abc"</span><span class="token punctuation">;</span> <span class="token comment">// 3.14</span>
</code></pre>

    </p>

      <tr>
       <th>
        函数
       </th>
       <th>

<span class="token keyword">echo</span> <span class="token function">intval</span><span class="token punctuation">(</span><span class="token variable">$var</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 123</span>
<span class="token keyword">echo</span> <span class="token function">floatval</span><span class="token punctuation">(</span><span class="token string double-quoted-string">"3.14abc"</span><span class="token punctuation">)</span><span class="token punctuation">;</span> <span class="token comment">// 3.14</span>

<span class="token function">settype</span><span class="token punctuation">(</span><span class="token variable">$var</span><span class="token punctuation">,</span> <span class="token string double-quoted-string">"int"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

        </code>

<span class="token variable">$arr</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token keyword type-casting">array</span><span class="token punctuation">)</span><span class="token variable">$str</span><span class="token punctuation">;</span>

       </strong>
      </p>

<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">string</span><span class="token punctuation">)</span><span class="token variable">$arr</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "Array"</span>

    <h4>
     <a id="5__797">
     </a>
     <strong>
      5. 类型转换注意事项
     </strong>
    </h4>
    <ol>
     <li>
      <p>
       <strong>
        字符串转换为数字时，非数字部分被忽略
       </strong>
      </p>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"10abc"</span><span class="token punctuation">;</span> <span class="token comment">// 10</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        浮点数转换为整数时会截断
       </strong>
      </p>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">int</span><span class="token punctuation">)</span><span class="token number">3.99</span><span class="token punctuation">;</span> <span class="token comment">// 3</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        布尔值转换
       </strong>
      </p>
      <ul>
       <li>
        空数组、空字符串、
        <code>
         NULL
        </code>
        、
        <code>
         0
        </code>
        转换为
        <code>
         false
        </code>
        ，其余为
        <code>
         true
        </code>
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">bool</span><span class="token punctuation">)</span><span class="token string double-quoted-string">""</span><span class="token punctuation">;</span>  <span class="token comment">// false</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">bool</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"0"</span><span class="token punctuation">;</span> <span class="token comment">// false</span>
<span class="token keyword">echo</span> <span class="token punctuation">(</span><span class="token keyword type-casting">bool</span><span class="token punctuation">)</span><span class="token string double-quoted-string">"abc"</span><span class="token punctuation">;</span> <span class="token comment">// true</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        JSON 解析
       </strong>
      </p>
      <ul>
       <li>
        默认
        <code>
         json_decode()
        </code>
        返回对象，使用
        <code>
         true
        </code>
        作为第二个参数返回数组：
       </li>
      </ul>
      <pre><code class="prism language-php"><span class="token variable">$arr</span> <span class="token operator">=</span> <span class="token function">json_decode</span><span class="token punctuation">(</span><span class="token string single-quoted-string">'{"a":1,"b":2}'</span><span class="token punctuation">,</span> <span class="token constant boolean">true</span><span class="token punctuation">)</span><span class="token punctuation">;</span>

     <a id="_832">
     </a>
     <strong>
      结论
     </strong>
    </h4>
    <p>
     PHP 的数据类型转换相对宽松，允许隐式转换，但容易导致意外结果。最佳实践：
    </p>
    <ul>
     <li>
      <strong>
       使用显式转换
      </strong>
      (
      <code>
       (int)
      </code>
      ,
      <code>
       (string)
      </code>
      ,
      <code>
       (bool)
      </code>
      )
     </li>
     <li>
      <strong>
       使用转换函数
      </strong>
      (
      <code>
       intval()
      </code>
      ,
      <code>
       floatval()
      </code>
      ,
      <code>
       boolval()
      </code>
      )
     </li>
     <li>
      <strong>
       避免依赖隐式转换
      </strong>
      （如
      <code>
       if ($var)
      </code>
      可能带来意外行为）
     </li>
     <li>
      <strong>
       JSON 数据转换时注意数据结构
      </strong>
      （使用
      <code>
       true
      </code>
      作为参数转换为数组）
     </li>
    </ul>

      注意事项
     </strong>
    </h4>
    <ul>
     <li>
      <strong>
       避免作为数组键
      </strong>
      ：浮点数键会转换为整数，可能导致冲突。
     </li>
     <li>
      <strong>
       高精度需求
      </strong>
      ：使用
      <code>
       BC Math
      </code>
      或
      <code>
       GMP
      </code>
      扩展处理精确计算（如财务数据）。
     </li>
    </ul>
    <h4>
     <a id="_1037">
     </a>
     示例代码
    </h4>
    <pre><code class="prism language-php"><span class="token comment">// 浮点数定义与精度问题</span>

    <h4>
     <a id="_1057">
     </a>
     总结
    </h4>
    <ul>
     <li>
      浮点数适用于科学计算或无需高精度的场景。
     </li>
     <li>
      精确计算（如货币）建议使用整数（如分单位）或
      <code>
       DECIMAL
      </code>
      类型（数据库中）。
     </li>
     <li>
      始终警惕精度误差，合理选择比较方法和运算库。
     </li>
    </ul>

     <a id="_1065">
     </a>
     七，布尔型数据
    </h3>
    <h4>
     <a id="_1067">
     </a>
     一、布尔型的基本概念
    </h4>
    <p>
     布尔型（Boolean）是PHP中最简单的数据类型，只有两个可能的值：
    </p>
    <ul>
     <li>
      <strong>
       <code>
        true
       </code>
      </strong>
      （逻辑真）
     </li>
     <li>
      <strong>
       <code>
        false
       </code>
      </strong>
      （逻辑假）
     </li>
    </ul>
    <p>
     布尔型通常用于条件判断（如
     <code>
      if
     </code>
     、
     <code>
      while
     </code>
     语句）中控制程序流程。￥val
    </p>
    <hr/>
    <h4>
     <a id="_1078">
     </a>
     二、定义布尔值
    </h4>
    <h5>
     <a id="1__1080">
     </a>
     1. 直接赋值
    </h5>
    <pre><code class="prism language-php"><span class="token variable">$isTrue</span> <span class="token operator">=</span> <span class="token constant boolean">true</span><span class="token punctuation">;</span>
<span class="token variable">$isFalse</span> <span class="token operator">=</span> <span class="token constant boolean">false</span><span class="token punctuation">;</span>
</code></pre>
    <h5>
     <a id="2__1087">
     </a>
     2. 不区分大小写的常量
    </h5>
    <p>
     PHP中布尔常量不区分大小写，但推荐使用小写以保持代码规范：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$a</span> <span class="token operator">=</span> <span class="token constant boolean">TRUE</span><span class="token punctuation">;</span>   <span class="token comment">// 有效，但建议使用小写</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token constant boolean">False</span><span class="token punctuation">;</span>  <span class="token comment">// 有效</span>
</code></pre>

    <h4>
     <a id="_1098">
     </a>
     三、布尔转换规则
    </h4>
    <p>
     PHP是弱类型语言，其他类型值在需要时会
     <strong>
      自动转换为布尔值
     </strong>
     。以下是转换规则：
    </p>
    <h5>
     <a id="1__false__1102">
     </a>
     1.
     <strong>
      视为
      <code>
       false
      </code>
      的值
     </strong>
    </h5>
    <p>
     以下值在转换为布尔型时会被认为是
     <code>
      false
     </code>
     ：
    </p>
    <ul>
     <li>
      整型
      <code>
       0
      </code>
      、浮点型
      <code>
       0.0
      </code>
     </li>
     <li>
      空字符串
      <code>
       ""
      </code>
      和字符串
      <code>
       "0"
      </code>
     </li>
     <li>
      空数组
      <code>
       []
      </code>
      或
      <code>
       array()
      </code>
     </li>
     <li>
      <code>
       null
      </code>
     </li>
     <li>
      特殊对象（如空
      <code>
       SimpleXMLElement
      </code>
      对象）
     </li>
    </ul>
    <h5>
     <a id="2__true__1112">
     </a>
     2.
     <strong>
      视为
      <code>
       true
      </code>
      的值
     </strong>
    </h5>
    <p>
     其他所有值均被视为
     <code>
      true
     </code>
     ，包括：
    </p>
    <ul>
     <li>
      非零数字（如
      <code>
       1
      </code>
      ,
      <code>
       -1
      </code>
      ,
      <code>
       3.14
      </code>
      ）
     </li>
     <li>
      非空字符串（如
      <code>
       "false"
      </code>
      、
      <code>
       " "
      </code>
      ）
     </li>
     <li>
      非空数组（如
      <code>
       [0]
      </code>
      ）
     </li>
     <li>
      资源（Resource）类型
     </li>
     <li>
      所有对象（即使是空对象）
     </li>
    </ul>

    <h5>
     <a id="_1345">
     </a>
     <strong>
      计算机码的基本概念
     </strong>
    </h5>
    <p>
     计算机存储数据时通常采用
     <strong>
      定长二进制
     </strong>
     ，尤其是整数。负数的表示方式一般有三种主要编码方式：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        原码（Sign-Magnitude Representation）
       </strong>
      </p>
      <ul>
       <li>
        最高位（最左侧）作为符号位：
        <code>
         0
        </code>
        表示正数，
        <code>
         1
        </code>
        表示负数。
       </li>
       <li>
        剩余部分表示数值的绝对值。
       </li>
       <li>
        缺点：正负零的表示不同，且计算复杂。
       </li>
      </ul>
      <p>
       <strong>
        示例（8位存储）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         +5
        </code>
        →
        <code>
         00000101
        </code>
       </li>
       <li>
        <code>
         -5
        </code>
        →
        <code>
         10000101
        </code>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        反码（One’s Complement）
       </strong>
      </p>
      <ul>
       <li>
        正数的表示与原码相同。
       </li>
       <li>
        负数的表示是
        <strong>
         对原码的数值部分按位取反
        </strong>
        （0 变 1，1 变 0）。
       </li>
       <li>
        存在两个零 (
        <code>
         +0
        </code>
        和
        <code>
         -0
        </code>
        )，仍然不便于计算。
       </li>
      </ul>
      <p>
       <strong>
        示例（8位存储）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         +5
        </code>
        →
        <code>
         00000101
        </code>
       </li>
       <li>
        <code>
         -5
        </code>
        →
        <code>
         11111010
        </code>
        （取反）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        补码（Two’s Complement，计算机中最常用的表示方式）
       </strong>
      </p>
      <ul>
       <li>
        正数的表示与原码相同。
       </li>
       <li>
        负数的表示是
        <strong>
         反码 + 1
        </strong>
        （取反后加 1）。
       </li>
       <li>
        补码的优点：
        <ul>
         <li>
          计算更方便（加减法可直接使用二进制加法）。
         </li>
         <li>
          只有一种零表示方式 (
          <code>
           00000000
          </code>
          )。
         </li>
        </ul>
       </li>
      </ul>
      <p>
       <strong>
        示例（8位存储）
       </strong>
       ：
      </p>
      <ul>
       <li>
        <code>
         +5
        </code>
        →
        <code>
         00000101
        </code>
       </li>
       <li>
        <code>
         -5
        </code>
        →
        <code>
         11111011
        </code>
        （先取反
        <code>
         11111010
        </code>
        ，再加
        <code>
         1
        </code>
        ）
       </li>
      </ul>
     </li>
    </ol>

    <h5>
     <a id="1__1494">
     </a>
     1. 自增和自减运算符
    </h5>
    <p>
     <strong>
      基本概念
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       自增运算符（++）
      </strong>
      ：用于将变量的值增加 1。
     </li>
     <li>
      <strong>
       自减运算符（–）
      </strong>
      ：用于将变量的值减少 1。
     </li>
    </ul>
    <p>
     这两种运算符又分为两种形式：
    </p>
    <ul>
     <li>
      <p>
       <strong>
        前置形式
       </strong>
       <br/>
       写法：
       <code>
        ++$var
       </code>
       或
       <code>
        --$var
       </code>
       <br/>
       <strong>
        特点
       </strong>
       ：先对变量进行加减操作，然后再使用变量的新值。例如：
      </p>
      <pre><code class="prism language-php">php复制编辑<span class="token operator">&lt;</span><span class="token operator">?</span><span class="token class-name type-declaration">php</span>
<span class="token variable">$a</span> <span class="token operator">=</span> <span class="token number">5</span><span class="token punctuation">;</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token operator">++</span><span class="token variable">$a</span><span class="token punctuation">;</span>  <span class="token comment">// $a 先自增变为6，再将6赋给 $b</span>
<span class="token keyword">echo</span> <span class="token variable">$a</span><span class="token punctuation">;</span> <span class="token comment">// 输出 6</span>
<span class="token keyword">echo</span> <span class="token variable">$b</span><span class="token punctuation">;</span> <span class="token comment">// 输出 6</span>
<span class="token operator">?</span><span class="token operator">&gt;</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        后置形式
       </strong>
       <br/>
       写法：
       <code>
        $var++
       </code>
       或
       <code>
        $var--
       </code>
       <br/>
       <strong>
        特点
       </strong>
       ：先使用变量原来的值，然后再进行加减操作。例如：
      </p>
      <pre><code class="prism language-php"><span class="token delimiter important">&lt;?php</span>
<span class="token variable">$a</span> <span class="token operator">=</span> <span class="token number">5</span><span class="token punctuation">;</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token variable">$a</span><span class="token operator">++</span><span class="token punctuation">;</span>  <span class="token comment">// $b 先接收$a的值5，然后$a自增变为6</span>
<span class="token keyword">echo</span> <span class="token variable">$a</span><span class="token punctuation">;</span> <span class="token comment">// 输出 6</span>
<span class="token keyword">echo</span> <span class="token variable">$b</span><span class="token punctuation">;</span> <span class="token comment">// 输出 5</span>
<span class="token operator">?</span><span class="token operator">&gt;</span>
</code></pre>
     </li>
    </ul>
    <p>
     <strong>
      使用注意
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       表达式求值顺序
      </strong>
      <br/>
      在复杂表达式中混合使用前置和后置运算符时，需要特别注意求值顺序，以免造成难以预料的结果。例如，避免在同一表达式中对同一变量进行多次自增或自减操作。
     </li>
     <li>
      <strong>
       代码可读性
      </strong>
      <br/>
      尽量避免过度复杂的自操作运算符嵌套，以保证代码的可读性和易于维护。
     </li>
    </ul>

    <h4>
     <a id="_1572">
     </a>
     十二、
     <strong>
      三元运算符
     </strong>
    </h4>
    <h5>
     <a id="1_1574">
     </a>
     1.基本语法
    </h5>
    <pre><code class="prism language-php"><span class="token variable">$result</span> <span class="token operator">=</span> <span class="token punctuation">(</span>条件<span class="token punctuation">)</span> <span class="token operator">?</span> 表达式<span class="token number">1</span> <span class="token punctuation">:</span> 表达式<span class="token number">2</span><span class="token punctuation">;</span>
</code></pre>
    <ul>
     <li>
      <strong>
       条件
      </strong>
      ：一个布尔表达式。
     </li>
     <li>
      <strong>
       表达式1
      </strong>
      ：当条件为真（true）时执行或返回的值。
     </li>
     <li>
      <strong>
       表达式2
      </strong>
      ：当条件为假（false）时执行或返回的值。
     </li>
    </ul>
    <pre><code class="prism language-php"><span class="token delimiter important">&lt;?php</span>
<span class="token variable">$a</span> <span class="token operator">=</span> <span class="token number">10</span><span class="token punctuation">;</span>
<span class="token variable">$b</span> <span class="token operator">=</span> <span class="token number">20</span><span class="token punctuation">;</span>
<span class="token variable">$result</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token variable">$a</span> <span class="token operator">&gt;</span> <span class="token variable">$b</span><span class="token punctuation">)</span> <span class="token operator">?</span> <span class="token string double-quoted-string">"a大于b"</span> <span class="token punctuation">:</span> <span class="token string double-quoted-string">"a不大于b"</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$result</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "a不大于b"</span>
<span class="token operator">?</span><span class="token operator">&gt;</span>
</code></pre>
    <p>
     在上面的代码中，如果
     <code>
      $a &gt; $b
     </code>
     为真，则
     <code>
      $result
     </code>
     将被赋值为 “a大于b”；否则为 “a不大于b”。
    </p>
    <h5>
     <a id="2__1595">
     </a>
     2. 嵌套使用
    </h5>
    <p>
     当需要根据多个条件进行判断时，可以将三元运算符嵌套使用。但要注意，由于三元运算符是右结合（right-associative）的，建议使用括号明确表达顺序，避免产生歧义。例如：
    </p>
    <pre><code class="prism language-php"><span class="token delimiter important">&lt;?php</span>
<span class="token variable">$score</span> <span class="token operator">=</span> <span class="token number">75</span><span class="token punctuation">;</span>
<span class="token variable">$result</span> <span class="token operator">=</span> <span class="token punctuation">(</span><span class="token variable">$score</span> <span class="token operator">&gt;=</span> <span class="token number">90</span><span class="token punctuation">)</span> <span class="token operator">?</span> <span class="token string double-quoted-string">"优秀"</span> <span class="token punctuation">:</span> <span class="token punctuation">(</span><span class="token punctuation">(</span><span class="token variable">$score</span> <span class="token operator">&gt;=</span> <span class="token number">60</span><span class="token punctuation">)</span> <span class="token operator">?</span> <span class="token string double-quoted-string">"及格"</span> <span class="token punctuation">:</span> <span class="token string double-quoted-string">"不及格"</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$result</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "及格"</span>
<span class="token operator">?</span><span class="token operator">&gt;</span>
</code></pre>
    <p>
     这种写法先判断
     <code>
      $score &gt;= 90
     </code>
     ，如果不满足，再判断
     <code>
      $score &gt;= 60
     </code>
     ，依次返回对应的结果。
    </p>
    <hr/>
    <h5>
     <a id="3__1611">
     </a>
     3. 简化写法：短路求值
    </h5>
    <p>
     PHP 还支持一种简写形式，当你想要检查一个变量是否“真”并直接使用该变量时，可以使用“?:”运算符。其语法为：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$result</span> <span class="token operator">=</span> <span class="token variable">$var</span> <span class="token operator">?</span><span class="token punctuation">:</span> <span class="token variable">$default</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     这个表达式等同于：
    </p>
    <pre><code class="prism language-php"><span class="token variable">$result</span> <span class="token operator">=</span> <span class="token variable">$var</span> <span class="token operator">?</span> <span class="token variable">$var</span> <span class="token punctuation">:</span> <span class="token variable">$default</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     当
     <code>
      $var
     </code>
     的值被认为是真（非空、非零等）时，结果就是
     <code>
      $var
     </code>
     ；否则返回
     <code>
      $default
     </code>
     。例如：
    </p>
    <pre><code class="prism language-php"><span class="token delimiter important">&lt;?php</span>
<span class="token variable">$username</span> <span class="token operator">=</span> <span class="token string double-quoted-string">""</span><span class="token punctuation">;</span>
<span class="token variable">$displayName</span> <span class="token operator">=</span> <span class="token variable">$username</span> <span class="token operator">?</span><span class="token punctuation">:</span> <span class="token string double-quoted-string">"匿名用户"</span><span class="token punctuation">;</span>
<span class="token keyword">echo</span> <span class="token variable">$displayName</span><span class="token punctuation">;</span> <span class="token comment">// 输出 "匿名用户"</span>
<span class="token operator">?</span><span class="token operator">&gt;</span>
</code></pre>

       or
      </code>

     </li>
