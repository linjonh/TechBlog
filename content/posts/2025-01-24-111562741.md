    <h2>
     <a id="Pygame_1">
     </a>
     Pygame的介绍
    </h2>
    <p>
     Pygame是被设计用来写游戏的python模块集合，Pygame是在优秀的SDL库之上开发的功能性包。使用python可以导入pygame来开发具有全部特性的游戏和多媒体软件，Pygame是极度轻便的并且可以运行在几乎所有的平台和操作系统上。
    </p>
    <p>
     安装库：
     <code>
      pip install pygame
     </code>
     。如果安装速度慢，可以使用换源安装。
    </p>
    <p>
     我们可以执行第一个pygame库自带的小游戏
     <br/>

     </img>
    </p>
    <p>
     pygame理解：
    </p>
    <ul>
     <li>
      Python最经典的2D游戏开发第三方库，也支持3D游戏开发
     </li>
     <li>
      Pygame适合用于游戏逻辑验证、游戏入门及系统演示验证
     </li>
     <li>
      Pygame是一种游戏开发引擎，基本逻辑具有参考价值
     </li>
     <li>
      Pygame有些"过时"，但永远"不过时"
     </li>
     <li>
      使用Pygame可以开发出优秀的游戏！
     </li>
    </ul>

图像的基本使用
</h2>
<p>
pygame 默认是采用的笛卡尔坐标系。左上角的顶点是(0,0)
</p>
<pre><code class="prism language-python"><span class="token comment">#加载导入的图片</span>
sr <span class="token operator">=</span> pygame<span class="token punctuation">.</span>image<span class="token punctuation">.</span>load<span class="token punctuation">(</span><span class="token string">"xxxx.jpg"</span><span class="token punctuation">)</span> <span class="token comment">#放入图片的路径</span>
rectsr <span class="token operator">=</span> sr<span class="token punctuation">.</span>get_rect<span class="token punctuation">(</span><span class="token punctuation">)</span>
</code></pre>
<p>
路径下的图像载入游戏，支持 JPG、PNG、GIF(非动画)等 13 种常用图片格式。
</p>
<p>
Pygame 使用内部定义的 Surface 对象表示所有载入的图像，其中.get_rect()方法返回一个覆盖图像的矩形 Rect 对象
</p>
<p>
Rect 对象有一些重要属性，例如：top,bottom,left,right 表示上下左右 width,height 表示宽度、高度。
</p>
<pre><code class="prism language-python">
</code></pre>
<p>
<code>
ballrect.move(x,y)
</code>
矩形移动一个偏移量(x,y)，即在横轴方向移动 x 像素，纵轴方向移动 y 像素，xy 为整数。
</p>
<p>
<code>
screen.fill(color)
</code>
显示窗口背景填充为 color 颜色，采用 RGB 色彩体系。由于壁球不断运动，运动后原有位置将默认填充白色，因此需要不断刷新背景色
</p>
<p>
<code>
screen.blit(src, dest)
</code>
将一个图像绘制在另一个图像上，即将 src
<br/>
绘制到 dest 位置上。通过 Rect 对象引导对壁球的绘制。
</p>
<p>
<code>
pygame.time.Clock()
</code>
创建一个 Clock 对象，用于操作时间
</p>
<p>
<code>
clock.tick(framerate)
</code>
控制帧速度，即窗口刷新速度，例如：clock.tick(100)表示每秒钟 100 次帧刷新视频中每次展示的静态图像称为帧
</p>
<p>
<strong>
键盘使用
</strong>
<br/>
Pygame 采用事件来对应键盘操作
<br/>
获取事件将得到键盘输入
<br/>
不同按键编写操作函数即可
</p>
<p>
pygame.KEYDOWN：
<br/>
Pygame 对键盘敲击的事件定义，键盘每个键对应一个具体定义。
</p>
<pre><code>pygame.K_UP 上
pygame.K_LEFT 左
pygame.K_DOWN 下
pygame.K_RIGHT 右
</code></pre>
<p>
更多按键属性可以参考 pygame 官方文档：

<h2>
<a id="pygame_252">
</a>
pygame 色彩机制
</h2>
<p>
色彩
<code>
pygame.Color
</code>
</p>
<p>
Color 类用于表达色彩，使用 RGB 或 RGBA 色彩模式，A 可选代表透明度
<br/>
Color 类可以用色彩名字、RGBA 值、HTML 色彩格式等方式定义
</p>
<pre><code>Color(name) 例如：Color("grey")
Color(r,g,b,a) 例如：Color(190, 190, 190, 255) #取值范围 0-255 a 代表透明度
Color(rgbvalue) 例如：Color("#BEBEBEFF")
</code></pre>
<p>
RGB 色彩模式之外增加了第四维度：alpha 通道
<br/>
alpha 通道表示不透明度，取值 0-255，默认 255
<br/>
alpha 通道值越大，不透明度越高，255 表示不透明
</p>
<pre><code>pygame.Color.r 获得 Color 类的红色值 r
pygame.Color.g 获得 Color 类的绿色值 g
pygame.Color.b 获得 Color 类的蓝色值 b
pygame.Color.a 获得 Color 类的不透明度值 a
pygame.Color.normalize 将 RGBA 各通道值归一到 0-1 之间
</code></pre>

</h4>
<p>
向屏幕上绘制特定字体的文字
<code>
pygame.freetype
</code>
<br/>
pygame.freetype 是绘制文字的增强方法，建议使用必须额外增加 import 引用
<br/>
根据字体和字号生成一个 Font 对象
<br/>
用 Font 对象的方法绘制具体文字
<br/>
<strong>
Font 类
</strong>
<br/>
pygame.freetype.Font(file, size=0)
</p>
<pre><code>file 字体类型名称或路径
size 字体大小
</code></pre>
<p>
Font 类的绘制方法 Font.render_to(surf, dest, text, fgcolor=None, bgcolor=None, rotation=0, size=0) ——&gt; Rect
</p>
<pre><code>• surf 绘制字体的平面，Surface 对象
• dest 在平面中的具体位置，(x,y)
• text 绘制的文字内容
• fgcolor 文字颜色
• bgcolor 背景颜色
• rotation 逆时针的旋转角度，取值 0-359，部分字体可旋转
• size 文字大小，赋值该参数将覆盖 Font 中的设定值

Rect 返回一个 Rect 对象
</code></pre>
<p>
Font.render(text, fgcolor=None, bgcolor=None,rotation=0, size=0) —&gt; (Surface, Rect)
</p>
<pre><code>• text 绘制的文字内容
• fgcolor, bgcolor 字体颜色、背景颜色
• rotation 逆时针的旋转角度，取值 0-359，部分字体可旋转
• size 文字大小，赋值该参数将覆盖 Font 中的设定值

返回一个元组，包含 Surface 对象和 Rect 对象
</code></pre>
<h4>

pygame 的两个重要类型
</h4>
<p>
pygame.Surface 绘图层，或绘图平面，或图层
<br/>
• 用于表示图形、文字或图像的绘制效果
<br/>
• 与当前屏幕主图层可以并列存在
<br/>
• 如果不绘制在主图层上，则不会被显示
</p>
<p>
Pygame.Rect 矩形区域
<br/>
• 对应于当前主图层的某个具体区域
<br/>
• 相当于某个矩形区域的指针或标识信息
<br/>
• 可以指定图层绘制在某个矩形区域中
</p>
