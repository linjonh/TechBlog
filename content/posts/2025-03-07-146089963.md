    <h2>
     一、题目
    </h2>
    <p>
     <a href="https://leetcode.cn/problems/valid-palindrome/" rel="nofollow" title="125. 验证回文串">
      125. 验证回文串
     </a>
    </p>
    <p>
     <span style="color:#0d0016">
      如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个 回文串 。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      字母和数字都属于字母数字字符。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      给你一个字符串
      <code>
       s
      </code>
      ，如果它是 回文串 ，返回
      <code>
       true
      </code>
      <em>
      </em>
      ；否则，返回
      <em>
      </em>
      <code>
       false
      </code>
      <em>
      </em>
      。
     </span>
    </p>
    <h2>
     二、思路
    </h2>
    <p>
     <span style="color:#0d0016">
      1. 因为题目中说了正着读和反着读——想到双指针，一个从前往后，一个从后往前。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      2. 将大写字符转化为小写字符，非字母字符移除——不同的语言有不同的处理方法，但是不论如何处理，这一步必须在循环遍历开始之前（因为我们要用处理后的数组进行判断）。
     </span>
    </p>
    <p>
     <span style="color:#0d0016">
      3. 返回的是true和false——那个对应的情况更简单就用代码判断哪个。
     </span>
    </p>
    <h2>
     三、代码
    </h2>
    <p>
     ① JavaScript：
    </p>
    <pre><code class="language-javascript">function IsHuiwen(s){
    let cleanS = s.toLowerCase().replace(/[^a-z0-9]/g, '');
    let left = 0;
    let right = cleanS.length - 1;
    while (left &lt; right){
        if(cleanS[left] !== cleanS[right]){
            return false;
        }
        left ++;
        right --;
    }
    return true;
}</code></pre>
    <p>
     ② Python：
    </p>
    <pre><code class="language-python">import re
def isHW(s: str) -&gt; bool:
    cleanS = re.sub(r'[^a-z0-9]', '', s.lower())
    left = 0
    right = len(s) - 1
    while left &lt; right :
        if cleanS[left] != cleanS[right]:
            return false
        left += 1
        right -= 1
    return true
    </code></pre>
    <p>
     🧐补充一点：
    </p>
    <p>
     js代码中的  let cleanS = s.toLowerCase().replace(/[^a-z0-9]/g, '')   和python代码中的   cleanS = re.sub(r'[^a-z0-9]', '', s.lower())   完成的都是“
     <span style="color:#0d0016">
      将大写字符转化为小写字符，非字母字符移除
     </span>
     ”这一步。具体来说：
    </p>
    <blockquote>
     <ul>
      <li>
       <span style="color:#494949">
        <code>
         r'[^a-z0-9]'
        </code>
        是一个正则表达式模式，用于匹配
        <strong>
         所有非字母数字字符
        </strong>
        。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        <code>
         [^...]
        </code>
        ：表示
        <strong>
         否定字符集
        </strong>
        ，即匹配不在方括号内的字符。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        <code>
         a-z
        </code>
        ：匹配所有小写字母（a 到 z）。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        <code>
         0-9
        </code>
        ：匹配所有数字（0 到 9）。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        <code>
         [^a-z0-9]
        </code>
        ：匹配所有
        <strong>
         不是小写字母或数字
        </strong>
        的字符。
       </span>
      </li>
     </ul>
    </blockquote>
    <p>
     <span style="color:#494949">
      作用是：
     </span>
    </p>
    <blockquote>
     <ul>
      <li>
       <span style="color:#494949">
        将字符串
        <code>
         s
        </code>
        转换为小写。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        使用正则表达式
        <code>
         r'[^a-z0-9]'
        </code>
        匹配字符串中的所有非字母数字字符。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        将匹配到的非字母数字字符替换为空字符串
        <code>
         ''
        </code>
        ，即
        <strong>
         移除这些字符
        </strong>
        。
       </span>
      </li>
      <li>
       <span style="color:#494949">
        <code>
         g
        </code>
        ：全局匹配标志，表示匹配字符串中的所有符合条件的部分，而不仅仅是第一个匹配。
       </span>
      </li>
     </ul>
    </blockquote>
    <p>
     ③ C++：
    </p>
    <pre><code class="language-cpp">#include &lt;iostream&gt;
#include &lt;string&gt;
#include &lt;algorithm&gt;
#include &lt;cctype&gt;

bool isHW(const std::string&amp; s) {
    std::string cleanS;
    for (char c : s) {
        if (std::isalnum(c)) { // 如果是字母数字字符
            cleanS += std::tolower(c); // 转换为小写并添加到结果中
        }
    }

    int left = 0;
    int right = cleanS.size() - 1;
    while (left &lt; right) {
        if (cleanS[left] != cleanS[right]) {
            return false; 
        }
        left++;
        right--;
    }

    return true; 
}</code></pre>
