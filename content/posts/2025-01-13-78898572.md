      今天，为大家推荐几款优秀的 AR/VR 开源库，希望能对大家有所帮助~
     </p>
     <p>
      <br/>
      <img alt="" class="img-responsive center-block" src="http://igeekbar.com/igeekbar/networks/uploadimgthumb/d164fbf4-23c8-4c71-a11d-7b77948c613b.jpg"/>

     <div>
      <h3>
       <a href="https://www.oschina.net/p/ar-js" rel="nofollow noopener noreferrer" target="_blank">
        1、AR.js
       </a>
      </h3>
      <p>
       AR.js 是一款应用于 Web 的高效增强现实（AR）库，基于 three.js + jsartoolkit5，无需安装。它适用于任何带有 webgl 和 webrtc 的手机，且运行速度非常快，在手机上也能高效运行，包括 Android、IOS 和 Windows phone 。
      </p>
      <p>
       <img alt="" class="img-responsive center-block" src="https://dn-sdkcnssl.qbox.me/editor/0qDHRcBrmf6ZFX4kSkJX.jpg"/>
      </p>
      <h4>
      </h4>
      <h3>
       <a href="https://github.com/google/lullaby" rel="noopener noreferrer" target="_blank">
        2、Lullaby
       </a>
      </h3>
      <p>
       Lullaby 是 Google 为 AR/VR 开发而推出的高性能 C++ 开源库，支持完整的 3D VR 环境，包括几何世界、全景图像和空间音频。它可实现高效的 runtime 性能，是由数据驱动的开发工具，可进行快速迭代。基于 Java API ，可与现有的安卓应用集成，支持 Cardboard/Daydream 头显、DaydreamVR 控制器和 VR 键盘，适用于 Android、iOS、Linux 和 Windows 平台。
       <img alt="" class="img-responsive center-block" src="http://igeekbar.com/igeekbar/networks/uploadimg/185fc921-5fb2-404c-943c-443626e64158.gif"/>
      </p>
      <p>
      </p>
      <h4>
      </h4>
      <h3>
       <a href="https://www.oschina.net/p/a-frame" rel="nofollow noopener noreferrer" target="_blank">
        3、A-Frame
       </a>
      </h3>
      <p>
       A-Frame 是 Mozilla 开源的网页虚拟现实体验（ WebVR ）框架，旨在让创建 WebVR 体验变得更简单。它可在移动、桌面、Vive 和 Rift 在内的平台上运行，跨平台处理 3D 和 WebVR 模板。
      </p>
      <p>
       A-Frame 被设计成 Web 开发者很熟悉的模样，HTML 代码易于阅读和复制粘贴。
      </p>
      <p>
       <img alt="" class="img-responsive center-block" src="https://sdk.cn/news/images/image.png"/>
      </p>
      <p>
       <img alt="Image" class="img-responsive center-block" src="http://7xkvof.com1.z0.glb.clouddn.com/113744_HvWZ_2896879.gif"/>
      </p>
      <p>
      </p>
      <h3>
       4、
       <a href="https://github.com/exyte/ARTetris" rel="noopener noreferrer" target="_blank">
        ARTetris
       </a>
      </h3>
      <p>
       ARTetris 是一款用 ARKit/SceneKit 实现的 iOS 增强现实版俄罗斯方块游戏，和目前 iOS 11 的 App Store 里的 AR 游戏一样，要求具有 A9 及以上的芯片的设备。开始游戏之前，需找到水平面，如地板、桌面等。
      </p>
      <p>
       <img alt="未命名1508314413.png" class="img-responsive center-block" src="https://dn-sdkcnssl.qbox.me/editor/8G7RqtzbH1axo6oEKcXI.png"/>
      </p>
      <p>
      </p>
      <h3>
       <a href="https://www.oschina.net/p/react-vr" rel="nofollow noopener noreferrer" target="_blank">
        5、React VR
       </a>
      </h3>
      <p>
       React VR 是 Facebook 开源的一款用于构建在 Web 浏览器中运行的 VR 应用的框架，使用与 React 相同的设计，让您通过声明式的组件构建丰富的 VR 世界和 UI。它将现代 API（例如 WebGL 和 WebVR）与 React 的声明能力结合起来，通过各种设备提供适用于消费者的体验。
      </p>
      <p>
       <img alt="" class="img-responsive center-block" src="https://dn-sdkcnssl.qbox.me/editor/aCbo3aId4Dn-7TqNpGSv.png"/>
      </p>
      <h4>
      </h4>
      <h4>
       <a href="https://www.oschina.net/p/ideaspace" rel="nofollow noopener noreferrer" target="_blank">
        6、IdeaSpace
       </a>
      </h4>
      <p>
       IdeaSpace 是一款用于虚拟现实网页的 CMS 内容管理系统。可以像管理博客一样管理你的虚拟现实空间和资源。IdeaSpace 使用前面提到的 A-Frame 来实现主题和空间，提供强大的遵循 Web 标准的标识语言。可通过 Oculus Rift 或者 Google Cardboard 在浏览器上方便体验虚拟现实空间，无需安装插件和应用。
      </p>
      <p>
       <img alt="" class="img-responsive center-block" src="https://dn-sdkcnssl.qbox.me/editor/VBYc8je5Hrurv0MOmlP7.png"/>
      </p>
