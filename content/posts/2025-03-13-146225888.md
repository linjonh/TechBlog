# 使用elementplus的table表格遇到的问题
#### 使用表格时在el-table-column上加fixed时导致下边框消失的问题
###### 1、例如：

{{
scope.$index + 1
}}



{{ scope.row.name }}


###### 2、加上fixed后导致下边框消失
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/b298d8a487a44b688ff86a89bcf29055.png#pic\_center)
###### 3、解决方案
:deep(.el-table-fixed-column--left), :deep(.el-table-fixed-column--right) {
left: auto !important;
right: auto !important;
}
