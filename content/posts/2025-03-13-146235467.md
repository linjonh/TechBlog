    <h1 class="title-article" id="articleContentId">
     Vue3中 ref 与 reactive区别
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>
   <link href="../../assets/css/ck_htmledit_views-704d5b9767.css" rel="stylesheet"/>
   <div class="htmledit_views" id="content_views">
    <p>
     <strong>
      ref
     </strong>
     <br/>
     用途: ref 通常用于创建一个
     <strong>
      响应式的基本类型数据
     </strong>
     （如 string、number、boolean 等），但它也
     <strong>
      可以用于对象或数组
     </strong>
    </p>
    <p>
     返回值: ref 返回一个带有
     <strong>
      .value 属性的对象
     </strong>
     ，访问或修改数据需要通过 .value 进行
    </p>
    <p>
     使用场景: 适合
     <strong>
      处理单个值
     </strong>
     或需要明确访问 .value 的场景
    </p>
    <pre><code>import { ref } from 'vue';

const count = ref(0) // 创建一个响应式的数字
console.log(count.value) // 访问值
count.value++ // 修改值</code></pre>
    <p>
     <strong>
      reactive
     </strong>
     <br/>
     用途: reactive 用于创建一个
     <strong>
      响应式的对象或数组
     </strong>
    </p>
    <p>
     返回值: reactive 直接返回一个响应式代理对象，
     <strong>
      无需通过 .value 访问
     </strong>
    </p>
    <p>
     使用场景: 适合处理
     <strong>
      复杂对象或嵌套数据结构
     </strong>
    </p>
    <pre><code>import { reactive } from 'vue';

const state = reactive({
  count: 0,
  name: 'Vue'
})
console.log(state.count) // 直接访问属性
state.count++ // 直接修改属性</code></pre>
