     数据结构（队列）
    </h2>
    <h3>
     <a id="_2">
     </a>
     什么是队列？
    </h3>
    <blockquote>
     <ul>
      <li>
       队列和栈类似，也是一类特殊的线性表。特殊之处也是在于操作上。
      </li>
      <li>
       队列：只允许在一端进行插入数据操作（入队），在另一端进行删除数据操作（出队）的特殊的线性表。
      </li>
      <li>
       具有先进先出，后进后出的特点。
      </li>
      <li>
       进行插入操作（入队）的一端称为队尾。进行删除操作（出队）的一端称为队头。
      </li>
     </ul>
     <p>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/cb8924fc6e294762a6b5c357806ad343.png"/>
     </p>
    </blockquote>

       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/a087f91df018478dada38f18d6c0215d.png"/>

    <h3>
     <a id="_17">
     </a>
     队列的实现
    </h3>
    <blockquote>
     <ul>
      <li>
       <p>
        和栈类似，也有两种实现方式。一种是数组，也就是顺序表，一种是链表。
       </p>
      </li>
      <li>
       <p>
        两种方式都是可以的，不过相比之下，链表更优一些。
       </p>
      </li>
      <li>
       <p>
        因为队列是在队头出数据，也就是头部删除数据，那么顺序表要删除头部数据需要一个个的移动数据进行覆盖。所以我们优先选择链表实现。
       </p>
      </li>
      <li>
       <h4>
        <a id="_24">
        </a>
        链表类型的选择
       </h4>

       </p>
      </li>
     </ul>
    </blockquote>
