    <p>
     我们使用NBIOT的开发板，STM32+BC35-G进行温湿度采集，并发送到湖畔科技云平台。并在微信小程序查看。
     <br/>
     第一步，我们先在湖畔科技云平台（
     <a href="http://www.nnhpiot.com/%EF%BC%89%E6%B3%A8%E5%86%8C%E8%B4%A6%E5%8F%B7%EF%BC%88%E5%B7%B2%E7%BB%8F%E6%B3%A8%E5%86%8C%E4%BA%86%E7%9A%84%E5%8F%AF%E4%BB%A5%E8%B7%B3%E8%BF%87%E8%AF%A5%E6%AD%A5%E9%AA%A4%EF%BC%89%E3%80%82" rel="nofollow">
      http://www.nnhpiot.com/）注册账号（已经注册了的可以跳过该步骤）。
     </a>
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d399fe392d0f80275e68bc2a7e5aec1c.png">
      <br/>
      <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/627f99cf9fa42927628b67a6196d04c1.png">
       <br/>
       第二步登录湖畔科技
       <br/>
       <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/33bdd7062ebca25f01b668b9cad9e4cb.png"/>
      </img>
     </img>
    </p>
    <p>
     登录云平台后会出现下面这个图，管理中心，
     <br/>
     注意APIKEY很重要。
     <br/>
     <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e1d39fb1512dd050195940833c25514d.png">
      <br/>
      第三步：我们在管理中心添加一个新设备。
      <br/>
      1、在左侧菜单中，我的设备—&gt;添加新设备，如下图
      <br/>
      <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/510d9cabc96b61c870746fb5fbb527d9.png">
       <br/>
       2、选择设备类型（在这里我们以温湿度为例）—&gt;填写设备名称（用来标识你的设备用途）
       <br/>
       —&gt;生成ID及秘钥（这是唯一的，用于系统识别你的设备）
       <br/>
       —&gt;填写设备描述（可以描述设备的用途）—&gt;点击提交—&gt;创建设备完成。
       <br/>
       <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e306ba06c19535a865c268e538cd44bd.png">
        <br/>
        第四步、购买NBIOT的STM32+BC35-G开发板
        <br/>
        （有NB模块的也可以直接下载我们的例程如果没有的，可以在本链接购买）。
        <br/>
        <a href="https://item.taobao.com/item.htm?id=578653570291" rel="nofollow">
         https://item.taobao.com/item.htm?id=578653570291
        </a>
        <br/>
        <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b8f5d4cabc19d0045a998e48eb9f32f8.jpeg">
         <br/>
         第五步：在管理中心查看温湿度。曲线显示。
         <br/>
         管理中心—&gt;我的设备—&gt;我的设备。就可以查看到自己的所有设备列表。
         <br/>
         温湿度DHT11是我之前添加的设备
         <br/>
         <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/70dfb94921b3fc2c332258ff72dc56c2.png">
          <br/>
          点击右侧的查看设备。效果如下图
          <br/>
          <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d59ec386772d47f1db4f13f508868047.png">
           <br/>
           还可以设置温湿度的报警值
           <br/>
           <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7927f978e32c9e7944bd1d890cdff04d.png">
            <br/>
            第五步，在微信小程序查看温湿度情况。
            <br/>
            步骤：1、在左侧菜单中，打开有惊喜—&gt;微信控制
            <br/>
            —&gt;可以选择微信公众号或者小程序都可以连接到小程序
            <br/>
            <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ac9ca4c286883bf6dfcc50010dc2f3c5.png">
             <br/>
             如果是选择公众号的话，关注公众号可以跳转过去小程序
             <br/>
             <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/32ada1c5d8bcd7155b9e6100533ddb0f.png"/>
             <br/>
             小程序的效果图如下。
             <br/>
             <img alt="这里写图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/e8267ee959ae24ff716f9b013ed08538.png"/>
             <br/>
             展示结束。
             <br/>
             如需要源代码，可以关注“南宁湖畔科技”微信公众号。微信号：NNHPTCH
             <br/>
             或者扫描下面的二维码。发送：BC35DHT11即可获取到单片机源代码。
