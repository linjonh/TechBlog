    <p>
     好久没用sqlserver 了，连连接对象资源，服务器名称怎么填才能在本地创建数据库都忘了，想起来了，记录一下
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/0f36d461afc2db74f8b220e7886d8a4d.png">
      <br/>
      首先呢是本地的连接需要的服务器名称怎么填
      <br/>
      先来回忆一下，服务器名称需要的是本地的id 加上实例名
      <br/>
      格式：.\实例名 -或者-（local）\实例名
      <br/>
      问题来了
      <br/>
      实例名怎么看，每个人有不同的实例名
      <br/>
      1.首先打开找到我的电脑右键管理找到服务和应用程序
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a0bbecb32b236103ecb9454ecd4b1964.png">
       <br/>
       2.找到服务，找到里面的sqlserver
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/36522f61a9ff26d571fdfd2eeb3ffada.png">
        <br/>
        3.右击属性找到常规里面的可执行文件的路径，这里有个-s,这个-s后面就是实例名
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/85d6fd9e1a951f59e23935417114939f.png">
         <br/>
         本地的总结了
        </img>
       </img>
      </img>
     </img>
    </p>
    <p>
     如果以上办法没啥用，你也可以直接试试127.0.0.1 ，一般这个也能连就像你输入本机电脑名称一样
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f1fc241ad6888489919444da5332527e.png">
      <br/>
      还有连接远程的服务器的 你可以直接试试部署这个数据库的电脑的IP地址
     </img>
    </p>
