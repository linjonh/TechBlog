# docker+spring Boot 构建T1云平台-技术架构图

### 架构说明

#### 微服务

```
 一个标准的微服务 应该有自己的“存储功能”,“对外的API” 本系统中的微服务将由 mysql + spring boot 组合成 存储+rest API的微服务

```

#### 微组件

```
一个微组件是“微服务+微服务运行环境”打成的docker包

```

#### 组件包

```
由许多微组件打包而成

```

#### T1云

```
T1云平台是由微组件构造成的平台,可以对微组件进行   
 插拔操作, 权限分配,备份管理 安全管理等等一系列操作

```
