# Linux/Centos安装完python3还是提示python: command not found
因安装的Python为Python3版本，因此直接输入python是会报错，可以输入python3即可。
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog\_migrate/2c2a28bfba780b95be256e171f0fc9da.png)
现在教大家怎么修改python3变为python，即输入python不会报错：
1. 查看python3路径 which python3
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog\_migrate/bed657699bfcd88757c4a49753389576.png)
2. 更改默认版本
```python
alternatives --set python /usr/bin/python3
```
如果没有权限，可以加入sudo
3. 之后在输入python就不会报错
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog\_migrate/43a2523d6312dd85245aa01d1c2ea449.png)
