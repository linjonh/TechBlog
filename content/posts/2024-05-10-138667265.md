    <h3>
     <a id="_0">
     </a>
     一、安装
    </h3>
    <p>
     <strong>
      1.首先到官网下载Mac版的Typora,
      <a href="https://typoraio.cn/" rel="nofollow">
       下载地址
      </a>
      ：https://typoraio.cn/
     </strong>
    </p>
    <p>
     （1）打开默认中文站
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b1fc2140cdcf5616b76cf151ff7fb5df.png">
      <br/>
      （2）往下滑，下载Mac版
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/cf1e73d3dcf003564ec582f21c3786f4.png">
       <br/>
       <strong>
        2.下载完成后，会看到Typora.dmg文件，点击打开文件
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/213985c82dfff2d3058ba387e1e49983.png">
        <br/>
        <strong>
         3.打开Typora.dmg文件，鼠标落在左边app图标上，按住拖拽到右边图标，安装完成。
        </strong>
       </img>
      </img>
     </img>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/2d7fbccc774bf0e949cfb20e8965d171.png">
      <br/>
      <strong>
       4.打开启动台
      </strong>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7661c9b57057666fa1155ccb51c60865.png">
       <strong>
        ，可以看到Typora软件已安装成功。
       </strong>
       <br/>
       <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/450bf463d135647e9dc881ca044a4e6b.png"/>
      </img>
     </img>
    </p>
    <p>
     <strong>
      5.点击Typora进入app的欢迎界面，
      <strong>
       发现
      </strong>
      未激活，只能试用14天
     </strong>
     。下面进行激活的操作。
    </p>
    <h3>
     <a id="_18">
     </a>
     二、激活
    </h3>
    <p>
     <strong>
      1.打开访达
     </strong>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/7852d6993daffae0cb43ba524758b6ea.png">
      <strong>
       ，按快捷键Command+Shift+G 打开搜索，
      </strong>
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/4a69e88fea2408fe3266a52550bcc5fa.png"/>
     </img>
    </p>
    <p>
     <strong>
      2.在上图红框内输入
     </strong>
     <code>
      /Applications/Typora.app/Contents/Resources/TypeMark
     </code>
     <br/>
     <strong>
      点击回车，快速进入目录。
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f1649e8d85754e55c5b5ef94c0ec955d.png"/>
    </p>
    <p>
     此时会看到搜索结果有很多，选第一个即可;
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/ad32fb35b0d2dc179f388f7918337750.png"/>
     <br/>
     <strong>
      3.选中后点击回车，跳转到该文件夹下；
      <br/>
      打开文件：page-dist文件夹–&gt;static–&gt;js–&gt;LicenseIndex.开头的文件
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f0faac7114067ece10b4ffab7a63e5df.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/94876c96e5e69ebcfa3e194af7d8171d.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/1b19806c28b69ed347dec10d1862e6e7.png"/>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/5884d1f46bc4806b61d9876ddb63c5c1.png"/>
     <br/>
     <strong>
      4.选中该文件后右键（Mac的右键：control+鼠标单击）–&gt;打开方式–&gt;用「文本编辑」打开
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c9b58492da8572bc091c63aaa1ad7d70.png"/>
     <br/>
     <strong>
      5.打开后，点击顶部菜单栏「编辑」–&gt;「查找」–&gt;「查找」，进行搜索；
     </strong>
     <br/>
     搜索：
     <code>
      hasActivated="true"==e.hasActivated
     </code>
     ,
     <br/>
     并将这段替换为
     <code>
      hasActivated="true"=="true"
     </code>
     。
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d8ad41c5e4d4005c35453d9893b6fd28.png"/>
    </p>
    <p>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/b74dd0b10133031b204e64d1cdb20b21.png"/>
     <br/>
     <strong>
      6.替换后，参考黄色高亮部分，保存该文件，即可。
     </strong>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/a5a6275ba7317926badc004a296368f4.png"/>
     <br/>
     <strong>
      7.保存后，重新打开Typora软件，欢迎界面撒花🎉🎉🎉，显示已激活。okk结束！！
     </strong>
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/f316e4ea26ef7defdc9138159ba52d95.png"/>
    </p>
