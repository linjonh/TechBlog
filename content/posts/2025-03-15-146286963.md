    <p>
     在
     <strong>
      ESP32
     </strong>
     及其他
     <strong>
      Xtensa 架构
     </strong>
     MCU 中，内存被划分为不同的区域，以优化性能和存储管理。这些内存区域包括
     <strong>
      SRAM, IRAM, DRAM, IROM, DROM
     </strong>
     ，它们各有用途。
    </p>
    <hr/>
    <h2>
     <a id="1__4">
     </a>
     <strong>
      1. 内存区域总览
     </strong>
    </h2>
    <p>
     ESP32 的内存架构主要由：
    </p>
    <ul>
     <li>
      <strong>
       SRAM（Static RAM）
      </strong>
     </li>
     <li>
      <strong>
       IRAM（Instruction RAM）
      </strong>
     </li>
     <li>
      <strong>
       IROM（Instruction ROM）
      </strong>
     </li>
     <li>
      <strong>
       DRAM（Data RAM）
      </strong>
     </li>
     <li>
      <strong>
       DROM（Data ROM）
      </strong>
     </li>
    </ul>
    <p>
     组成，每个部分有特定的用途。
    </p>
    <table>
     <thead>
      <tr>
       <th>
        <strong>
         名称
        </strong>
       </th>
       <th>
        <strong>
         作用
        </strong>
       </th>
       <th>
        <strong>
         存储内容
        </strong>
       </th>
       <th>
        <strong>
         特性
        </strong>
       </th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td>
        <strong>
         SRAM（Static RAM）
        </strong>
       </td>
       <td>
        片上 RAM
       </td>
       <td>
        IRAM + DRAM
       </td>
       <td>
        速度快，供 CPU 使用
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         IRAM（Instruction RAM）
        </strong>
       </td>
       <td>
        指令 RAM
       </td>
       <td>
        存放可执行代码（运行时加载）
       </td>
       <td>
        运行时可修改
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         DRAM（Data RAM）
        </strong>
       </td>
       <td>
        数据 RAM
       </td>
       <td>
        存放
        <code>
         .bss
        </code>
        （未初始化数据）、
        <code>
         .data
        </code>
        （已初始化数据）
       </td>
       <td>
        运行时可修改
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         IROM（Instruction ROM）

       </td>
       <td>
        指令 ROM

       <td>
        存放 Flash 里的
        <code>
         .text
        </code>
        （程序代码）
       </td>
       <td>
        运行时不可修改
       </td>
      </tr>
      <tr>
       <td>
        <strong>
         DROM（Data ROM）
        </strong>
       </td>
       <td>
        数据 ROM
       </td>
       <td>
        存放 Flash 里的
        <code>
         .rodata
        </code>
        （只读数据）
       </td>
       <td>
        运行时不可修改
       </td>
      </tr>

    <h2>
     <a id="2__26">
     </a>
     <strong>
      2. 详细解析
     </strong>
    </h2>
    <h3>
     <a id="21_SRAMStatic_RAM_RAM_28">
     </a>
     <strong>
      2.1 SRAM（Static RAM，片上静态 RAM）
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       SRAM 是 MCU 内部的 RAM，供 CPU 使用
      </strong>
     </li>
     <li>
      由
      <strong>
       IRAM（存放指令）+ DRAM（存放数据）
      </strong>
      组成
     </li>
     <li>
      运行速度快（比外部 Flash 快很多）
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       SRAM 的划分
      </strong>
     </p>
     <p>
      <strong>
       SRAM0 + SRAM1（用于 IRAM 和 DRAM）
      </strong>
     </p>
     <ul>
      <li>
       <strong>
        IRAM：指令 RAM
       </strong>
       ，存放
       <strong>
        可执行代码
       </strong>
      </li>
      <li>
       <strong>
        DRAM：数据 RAM
       </strong>
       ，存放
       <strong>
        数据（变量、堆、栈）
       </strong>
      </li>
     </ul>
     <p>
      <strong>
       SRAM2（仅用于 DRAM）
      </strong>
     </p>
     <ul>
      <li>
       仅存放
       <strong>
        数据
       </strong>
      </li>
     </ul>
    </blockquote>
    <hr/>
    <h3>
     <a id="22_IRAMInstruction_RAM_RAM_47">
     </a>
     <strong>
      2.2 IRAM（Instruction RAM，指令 RAM）
     </strong>
    </h3>
    <ul>
     <li>
      存放
      <strong>
       可执行代码
      </strong>
     </li>
     <li>
      <strong>
       用于高实时性任务
      </strong>
      ，比如
      <strong>
       中断处理
      </strong>
     </li>
     <li>
      代码必须从
      <strong>
       Flash 加载到 IRAM
      </strong>
      才能运行
     </li>
     <li>
      IRAM 里的代码
      <strong>
       可以被修改
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      🔹 示例：
     </strong>
    </p>
    <p>
     在
     <strong>
      ESP-IDF
     </strong>
     里，想要让某个函数运行在
     <strong>
      IRAM
     </strong>
     ，需要使用
     <strong>
      <code>
       IRAM_ATTR
      </code>
     </strong>
     修饰符：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">void</span> IRAM_ATTR <span class="token function">gpio_isr_handler</span><span class="token punctuation">(</span><span class="token keyword">void</span> <span class="token operator">*</span>arg<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// GPIO 中断处理（放入 IRAM，提高执行速度）</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <h3>
     <a id="23_DRAMData_RAM_RAM_66">
     </a>
     <strong>
      2.3 DRAM（Data RAM，数据 RAM）
     </strong>
    </h3>
    <ul>
     <li>
      存放
      <strong>
       数据（变量、栈、堆）
      </strong>
     </li>
     <li>
      DRAM 里的数据
      <strong>
       可读可写
      </strong>
     </li>
     <li>
      运行时 CPU
      <strong>
       从 DRAM 读取变量数据
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       DRAM 里存放的内容：
      </strong>
     </p>
     <ul>
      <li>
       <code>
        .data
       </code>
       （已初始化的全局变量）
      </li>
      <li>
       <code>
        .bss
       </code>
       （未初始化的全局变量）
      </li>
      <li>
       堆（Heap）
      </li>
      <li>
       栈（Stack）
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      简单理解
     </strong>
     ：
     <strong>
      DRAM 里存放的是程序运行时需要用的数据，比如全局变量、栈、堆等
     </strong>
    </p>
    <hr/>
    <h3>
     <a id="24_IROMInstruction_ROM_ROM_83">
     </a>
     <strong>
      2.4 IROM（Instruction ROM，指令 ROM）
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       存放 Flash 里的程序代码（.text）
      </strong>
     </li>
     <li>
      代码不能被修改，
      <strong>
       只能执行
      </strong>
     </li>
     <li>
      运行时，CPU
      <strong>
       从 Flash 直接取指令执行
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       IROM 是存放在 Flash 里的代码，只有一部分需要提高执行速度的代码会被拷贝到 IRAM 执行
      </strong>
     </p>
    </blockquote>
    <hr/>
    <h3>
     <a id="25_DROMData_ROM_ROM_93">
     </a>
     <strong>
      2.5 DROM（Data ROM，数据 ROM）
     </strong>
    </h3>
    <ul>
     <li>
      <strong>
       存放 Flash 里的只读数据（.rodata）
      </strong>
     </li>
     <li>
      运行时
      <strong>
       CPU 直接从 Flash 读取
      </strong>
     </li>
     <li>
      <strong>
       不能修改
      </strong>
     </li>
    </ul>
    <blockquote>
     <p>
      <strong>
       DROM 里存放的内容：
      </strong>
     </p>
     <ul>
      <li>
       <code>
        .rodata
       </code>
       （只读数据，如
       <code>
        const char[]
       </code>
       字符串）
      </li>
      <li>
       其他只读配置数据
      </li>
     </ul>
    </blockquote>
    <p>
     <strong>
      简单理解
     </strong>
     ：
     <strong>
      DROM 是存放在 Flash 里的只读数据（比如
      <code>
       const char *string = "Hello"
      </code>
      ）
     </strong>
    </p>
    <hr/>
    <h2>
     <a id="3_ESP32__108">
     </a>
     <strong>
      3. ESP32 内存架构示意图
     </strong>
    </h2>
    <pre><code class="prism language-plaintext">+-------------------------+  0x40000000  (IROM)
|  Flash (代码)           |  存放 .text（程序代码）
+-------------------------+
|  Flash (只读数据)       |  存放 .rodata（只读数据）  &lt;--- DROM
+-------------------------+
|  IRAM (指令 RAM)       |  运行时加载的代码  &lt;--- IRAM
+-------------------------+  0x3F800000  (SRAM)
|  DRAM (数据 RAM)       |  存放变量、堆、栈  &lt;--- DRAM
+-------------------------+
</code></pre>
    <hr/>
    <h2>
     <a id="4__IRAM_124">
     </a>
     <strong>
      4. 什么时候用 IRAM？
     </strong>
    </h2>
    <p>
     <strong>
      需要高实时性，避免 Flash 访问延迟
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       中断处理（ISR）
      </strong>
     </li>
     <li>
      <strong>
       WiFi、蓝牙实时任务
      </strong>
     </li>
     <li>
      <strong>
       定时器回调
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      方法：使用
      <code>
       IRAM_ATTR
      </code>
     </strong>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">void</span> IRAM_ATTR <span class="token function">gpio_isr_handler</span><span class="token punctuation">(</span><span class="token keyword">void</span> <span class="token operator">*</span>arg<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// 高速执行的 GPIO 中断</span>
<span class="token punctuation">}</span>
</code></pre>
    <hr/>
    <h2>
     <a id="5__DRAM_142">
     </a>
     <strong>
      5. 什么时候用 DRAM？
     </strong>
    </h2>
    <p>
     <strong>
      存放变量
     </strong>
    </p>
    <ul>
     <li>
      <strong>
       全局变量
      </strong>
     </li>
     <li>
      <strong>
       堆（malloc 分配）
      </strong>
     </li>
     <li>
      <strong>
       栈（局部变量、函数参数）
      </strong>
     </li>
    </ul>
    <p>
     <strong>
      方法：直接使用全局变量
     </strong>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">int</span> counter <span class="token operator">=</span> <span class="token number">0</span><span class="token punctuation">;</span>  <span class="token comment">// 存放在 DRAM</span>
</code></pre>
    <hr/>
    <h2>
     <a id="6_158">
     </a>
     <strong>
      6.总结
     </strong>
    </h2>
    <p>
     ❓
     <strong>
      为什么有 IRAM 和 IROM？
     </strong>
     👉 IROM 在 Flash，CPU 取指令慢，所以部分代码（中断、WiFi 任务）需要拷贝到 IRAM 加速执行。
    </p>
    <p>
     ❓
     <strong>
      为什么有 DRAM 和 DROM？
     </strong>
     👉 DRAM 是可变数据，DROM 是 Flash 里的只读数据。
    </p>
    <p>
     ❓
     <strong>
      如何让代码运行在 IRAM？
     </strong>
     👉 使用
     <code>
      IRAM_ATTR
     </code>
     修饰符：
    </p>
    <pre><code class="prism language-c"><span class="token keyword">void</span> IRAM_ATTR <span class="token function">gpio_isr_handler</span><span class="token punctuation">(</span><span class="token keyword">void</span> <span class="token operator">*</span>arg<span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> <span class="token punctuation">.</span><span class="token punctuation">.</span><span class="token punctuation">.</span> <span class="token punctuation">}</span>
</code></pre>
    <p>
     ❓
     <strong>
      如何减少 RAM 占用？
     </strong>
     👉
     <strong>
      <code>
       const
      </code>
      变量存放在 Flash
     </strong>
    </p>
    <pre><code class="prism language-c"><span class="token keyword">const</span> <span class="token keyword">char</span> <span class="token operator">*</span>string <span class="token operator">=</span> <span class="token string">"Hello"</span><span class="token punctuation">;</span>  <span class="token comment">// 存入 DROM，节省 DRAM</span>
</code></pre>
