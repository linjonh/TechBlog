    <h3>
     1. Python基本环境安装：
    </h3>
    <p>
     <span style="white-space:pre">
     </span>
     在网站
     <a href="https://www.python.org/" rel="nofollow noopener noreferrer" target="_blank">
      Python官网
     </a>
     中下载Windows版本的Python安装包，双击下载下来的安装包，安装过程中要选中pip和
     <span style="font-family:tahoma,arial,sans-serif; line-height:18px">
      Add python.exe to Path两个选项，在安装按成之后Win+R进入命令行输入Python，得到如下结果说明安装完成
     </span>
    </p>
    <p style="text-align:center">
     <span style="font-family:tahoma,arial,sans-serif; line-height:18px">
      <img align="middle" alt="" src="https://img-blog.csdn.net/20160824181412035?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       <br/>
      </img>
     </span>
    </p>
    <p>
     <span style="white-space:pre">
     </span>
     要退出命令行下的输入模式按Ctrl+Z之后按回车Enter按键即可退出
    </p>
    <h3>
     2. Windows下Python基本依赖库的安装
    </h3>
    <div>
     <span style="white-space:pre">
     </span>
     因为在最初的Python安装过程中选择pip，便可以通过pip对python的附加依赖库进行管理，这里就将通过pip下载安装numpy和Matplotlib，这里以Matplotlib为例子作为讲解，其它也是按照这个模式进行安装，比较简单
    </div>
    <div>
     <span style="white-space:pre">
     </span>
     首先，更新一下Python Windows版本的pip，在Win+R 命令行中输入：
    </div>
    <div>
     <pre><code class="language-python">pip install pip –-upgrade</code></pre>
     <span style="white-space:pre">
     </span>
     然后输入下面的命令进行安装：
    </div>
    <div>
     <pre><code class="language-python">pip install Matplotlib</code></pre>
     <span style="white-space:pre">
     </span>
     <div style="text-align:left">
      <span style="white-space:pre">
      </span>
      下载安装包：
     </div>
     <div style="text-align:center">
      <img alt="" src="https://img-blog.csdn.net/20160824182323376?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      </img>
     </div>
     <div style="text-align:center">
     </div>
     <span style="white-space:pre">
     </span>
     安装完成：
    </div>
    <div>
     <div style="text-align:center">
      <img alt="" src="https://img-blog.csdn.net/20160824182449955?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
      </img>
     </div>
     <div style="text-align:left">
      <span style="white-space:pre">
      </span>
      Scipy的安装：
     </div>
     <div style="text-align:left">
      <span style="white-space:pre">
      </span>
      由于在命令行界面直接输入：
     </div>
     <div style="text-align:left">
      <pre><code class="language-python">pip install scipy</code></pre>
      <span style="white-space:pre">
      </span>
      无法正确安装报出下列错误：
     </div>
     <div style="text-align:left">
      <div style="text-align:center">
       <img alt="" src="https://img-blog.csdn.net/20160824191902435?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQv/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center">
       </img>

      <span style="white-space:pre">

       Win32版本
      </a>
      但是为Win32版本，在64bit机器上报错...后续解决

      4. 测试
     </h3>

     </div>

y = np.sin(x)
plt.figure("Sin")
plt.title("Sin curve")
plt.xlabel("X label")
plt.ylabel("Y label")
