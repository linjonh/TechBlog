    <h2>
     <a id="python_1">
     </a>
     一、python变量的命名规则
    </h2>
    <h4>
     <a id="1_2">
     </a>
     1、变量名大小写敏感
    </h4>
    <pre><code class="prism language-python">python变量名区分大小写，也就是Student和student在python语言中代表两个不同的名字。
</code></pre>
    <h4>
     <a id="2_8">
     </a>
     2、变量名只能包含数字、字母、下划线，但是不能以数字开头
    </h4>
    <pre><code class="prism language-python">student_id、student1、student_1、student、_student、_等都是合理的命名，但是1student就是非法的命名。
</code></pre>
    <pre><code class="prism language-python"><span class="token comment"># 错误的变量名</span>
my var <span class="token operator">=</span> <span class="token string">"hello world"</span>

<span class="token comment"># 正确的变量名</span>
my_var <span class="token operator">=</span> <span class="token string">"hello world"</span>
</code></pre>
    <pre><code class="prism language-python"><span class="token comment"># 错误的变量名</span>
1var <span class="token operator">=</span> <span class="token number">123</span>

<span class="token comment"># 正确的变量名</span>
var1 <span class="token operator">=</span> <span class="token number">123</span>
</code></pre>
    <h4>
     <a id="3_31">
     </a>
     3、变量名尽量使用小写字母
    </h4>
    <pre><code class="prism language-python"><span class="token comment"># 错误的变量名</span>
MyVar <span class="token operator">=</span> <span class="token string">"hello world"</span>

<span class="token comment"># 正确的变量名</span>
my_var <span class="token operator">=</span> <span class="token string">"hello world"</span>
</code></pre>
    <h2>
     <a id="_40">
     </a>
     二、不同风格命名的变量代表不同的类型
    </h2>
    <h3>
     <a id="_41">
     </a>
     命名法
    </h3>
    <p>
     python变量命名一般采用蛇形命名法（毕竟python的本意就是大蟒蛇），也就是如果变量名由两个单词组成，那么使用下划线来连接这两个单词，比如student_id。当然，我们也会在python中看到类似驼峰命名法的变量名，对于专业的程序猿来说，这些命名都不是随便写的，他们符合一定的标准，下面总结以下：
    </p>
    <h4>
     <a id="1module_47">
     </a>
     1、模块（module）的命名风格
    </h4>
    <pre><code class="prism language-python">模块尽量使用小写命名，首字母保持小写，尽量不要用下划线<span class="token punctuation">(</span>除非多个单词，且数量不多的情况<span class="token punctuation">)</span>
</code></pre>
    <h5>
     <a id="11__53">
     </a>
     1.1 正确的模块名
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">import</span> decoder <span class="token keyword">import</span> html_parser
</code></pre>
    <h4>
     <a id="2class_58">
     </a>
     2、类（class）的命令
    </h4>
    <h5>
     <a id="21CamelCase_59">
     </a>
     2.1、类名使用驼峰(CamelCase)命名风格，首字母大写，私有类可用一个下划线开头
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">class</span> <span class="token class-name">Farm</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
<span class="token keyword">pass</span>

<span class="token keyword">class</span> <span class="token class-name">AnimalFarm</span><span class="token punctuation">(</span>Farm<span class="token punctuation">)</span><span class="token punctuation">:</span>
<span class="token keyword">pass</span>

<span class="token keyword">class</span> <span class="token class-name">_PrivateFarm</span><span class="token punctuation">(</span>Farm<span class="token punctuation">)</span><span class="token punctuation">:</span>
<span class="token keyword">pass</span>
</code></pre>
    <h4>
     <a id="3function_72">
     </a>
     3、函数（function）的命名
    </h4>
    <pre><code class="prism language-python"> 函数名一律小写，如有多个单词，用下划线隔开。

<span class="token keyword">def</span> <span class="token function">run</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
<span class="token keyword">pass</span>

<span class="token keyword">def</span> <span class="token function">run_with_env</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">:</span>
<span class="token keyword">pass</span>
</code></pre>
    <h4>
     <a id="4variable_84">
     </a>
     4、变量（variable）的命名
    </h4>
    <h5>
     <a id="41__85">
     </a>
     4.1、变量名尽量小写, 如有多个单词，用下划线隔开
    </h5>
    <pre><code class="prism language-python"><span class="token keyword">if</span> name <span class="token operator">==</span> ‘main’<span class="token punctuation">:</span>
count <span class="token operator">=</span> <span class="token number">0</span>
school_name <span class="token operator">=</span> ‘’
</code></pre>
    <h4>
     <a id="5constant_93">
     </a>
     5、常量（constant）的命令
    </h4>
    <pre><code class="prism language-python">如果我们想用一个符号来代表常量（值是不变的量，比如光速、π等），采用全部大写，如有多个单词，使用下划线隔开。

MAX_CLIENT <span class="token operator">=</span> <span class="token number">100</span> MAX_CONNECTION <span class="token operator">=</span> <span class="token number">1000</span> CONNECTION_TIMEOUT <span class="token operator">=</span> <span class="token number">600</span>
</code></pre>
    <h4>
     <a id="6_100">
     </a>
     6、“下划线”开头的特殊命名
    </h4>
    <h5>
     <a id="61_101">
     </a>
     6.1、单下划线开头的命名
    </h5>
    <pre><code class="prism language-python">如果类变量以单下划线“_”开头，代表这个变量不能被直接访问，类似于C<span class="token operator">+</span><span class="token operator">+</span>中的protected型，这样的变量也不能被 <span class="token keyword">import</span>
module_name导入。 使用单下划线<span class="token punctuation">(</span>one
underline）开头的方法，则表示该方法不是AP的一部分<span class="token punctuation">,</span>不要直接访问<span class="token punctuation">(</span>虽然语法上访问也没有什么问题<span class="token punctuation">)</span>。
</code></pre>
    <h5>
     <a id="62_109">
     </a>
     6.2、双下划线开头的命名
    </h5>
    <pre><code class="prism language-python">以双下划线<span class="token punctuation">(</span>two underlines<span class="token punctuation">)</span>开头的类变量，表示为类的私有成员，不能被导入和其他类变量访问。
对于类中的方法，使用双下划线开头开头表示子类不能覆写该方法。除非你真的知道你在干什么<span class="token punctuation">,</span>否则不要使用这种方式。
</code></pre>
    <h5>
     <a id="63_116">
     </a>
     6.3、双下划线开头并且双下划线结尾的命名
    </h5>
    <pre><code class="prism language-python">魔法方法：以双下划线开头和双下划线结尾的变量是Python的专用标识，有其特殊的身份。我们一般称这样的方法为“魔法方法”（magic
method）。

魔法方法是python内置方法，不需要主动调用，存在的目的是为了给python的解释器进行调用，几乎每个魔法方法都有一个对应的内置函数，或者运算符，当我们对这个对象使用这些函数或者运算符时就会调用类中的对应魔法方法，可以理解为重写这些python的内置函数。
当你想让自己定义的对象也可以像Python内置的对象一样使用内置的一些函数或操作符<span class="token punctuation">(</span>比如<span class="token builtin">len</span>、add、<span class="token operator">+</span>、、<span class="token operator">==</span>等<span class="token punctuation">)</span>时<span class="token punctuation">,</span>你可以定义该类方法。
当然还有些属性只在末尾加了但下划线这仅仅是为了避免我们起的一些名字和Python保留关键字冲突<span class="token punctuation">,</span>没有特殊含义。

</code></pre>
