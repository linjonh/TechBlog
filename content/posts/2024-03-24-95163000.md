     <h3 id="前言-1">
      <span id="jump1">
       1.前言
      </span>
     </h3>
     <p>
      配置是win10+SQL Server 2012，使用的GUI管理工具是SQL Server 2012自带的SQL Server Management Studio（以下简称
      <strong>
       <em>
        SSMS
       </em>
      </strong>
      ）。本系列主要学习SQL Server基础，目的主要是为了使用Python连接、使用数据库。另外在这里贴上
      <a href="https://www.microsoft.com/zh-cn/download/details.aspx?id=29062" rel="nofollow">
       SQL Server 2012下载地址
      </a>
      。
     </p>
     <h3 id="建立数据库-1">
      <span id="jump2">
       2.建立数据库
      </span>
     </h3>
     <h5 id="通过ssms建立数据库">
      <span id="jump2.1">
       2.1 通过SSMS建立数据库
      </span>
     </h5>
     <p>
      2.1.1打开SSMS，连接上数据库。
      <br/>
      如图所示，找到数据库，右键单击数据库，选择新建数据库。
      <br/>
      <img alt="1250397-20171003162626302-1706834780.png" src="https://i-blog.csdnimg.cn/blog_migrate/7dfcf61153a1436da5254087f5a2c8b9.png">
       <br/>
       <img alt="1250397-20171003162541833-1032629359.jpg" src="https://i-blog.csdnimg.cn/blog_migrate/2b199f5b89f88cac6f828e93b455d396.jpeg">
        <br/>
        2.1.2在数据库名称处输入SSMSTest，选择确定。
        <br/>
        （主要数据文件：*.mdf，日志文件*.ldf，次要数据文件*.ndf）
        <br/>
        <img alt="1250397-20171003163201380-1706577992.png" src="https://i-blog.csdnimg.cn/blog_migrate/72fe2b039bca45c2c4dfb577c4b53709.png">
         <br/>
         2.1.3右键数据库点击刷新或者按F5，然后就能在数据库下面看到刚刚建立的SSMSTest了。
         <br/>
         <img alt="1250397-20171003163922224-1127499270.png" src="https://i-blog.csdnimg.cn/blog_migrate/9f63dc8f33d1bd206ea158a8cf44d660.png">
          <br/>
          （可以看到SSMSTest左边的图标是圆柱体，可以理解为每一个圆柱体都代表着一个数据库。）
          <br/>
          2.1.4至此，已经建立了名为SSMSTest的数据库。
         </img>
        </img>
       </img>
      </img>
     </p>
     <h5 id="通过sql语句建立数据库">
      <span id="jump2.2">
       2.2 通过SQL语句建立数据库
      </span>
     </h5>
     <p>
      2.2.1在最开始的界面点击新建查询。
      <br/>
      <img alt="1250397-20171003164317552-703786562.png" src="https://i-blog.csdnimg.cn/blog_migrate/6f38afcb3001988c028825cb8616ea74.png">
       <br/>
       2.2.2在点击新建查询出来的界面中输入以下代码：
      </img>
     </p>
     <pre><code class="language-sql"><code>create database SQLTest</code></code></pre>
     <p>
      <img alt="1250397-20171003165122958-1001601925.png" src="https://i-blog.csdnimg.cn/blog_migrate/4bb29dec7785eeeb93ae4a648c95981f.png">
       <br/>
       2.2.3点击上面的执行，或者按F5之后出来了如下画面。
       <br/>
       <img alt="1250397-20171003165407130-1897126779.png" src="https://i-blog.csdnimg.cn/blog_migrate/880e76b3abc2a6ee7eb7493f579f5815.png">
        <br/>
        2.2.4右键数据库点击刷新或者按F5，然后就能在数据库下面看到刚刚建立的SQLTest了。
        <br/>
        <img alt="1250397-20171003165512958-1512586061.png" src="https://i-blog.csdnimg.cn/blog_migrate/e5541a3b670b975748483437ba647f4b.png"/>
        <br/>
        2.2.5至此，已经建立了名为SQLTest的数据库。
       </img>
      </img>
     </p>
     <h3 id="建立表-1">
      <span id="jump3">
       3.建立表
      </span>
     </h3>
     <h5 id="通过ssms建立表">
      <span id="jump3.1">
       3.1 通过SSMS建立表
      </span>
     </h5>
     <p>
      3.1.1打开刚刚建立的SSMSTest数据库，找到表（数据库-&gt;SSMSTest-&gt;表）。
      <br/>
      右键单击表，选择新建表。
      <br/>
      <img alt="1250397-20171003170936458-1398540146.png" src="https://i-blog.csdnimg.cn/blog_migrate/a19c40fddc4c13bec6d9a79cf2f2940b.png"/>
      <br/>
      3.1.2在点击新建表出来的界面中按照下图输入：
      <br/>
      <img alt="1250397-20171003171245599-1733661447.png" src="https://i-blog.csdnimg.cn/blog_migrate/e2ebd6fed257be762cecbabcad02321d.png"/>
      <br/>
      3.1.3按Ctrl+S或者点击最上面菜单栏中的文件，选择保存。
      <br/>
      输入欲保存的表名称student，并点击确定。
      <br/>
      <img alt="1250397-20171003172434005-267914025.png" src="https://i-blog.csdnimg.cn/blog_migrate/7554a318ebaea8d0bc2f7ff4dbcb2bc1.png"/>
      <br/>
      3.1.4再次点击新建表，按照下图输入，并且保存为course。
      <br/>
      <img alt="1250397-20171003171832599-957990286.png" src="https://i-blog.csdnimg.cn/blog_migrate/8cdb51320ad6d28eff4bb6fd9f776294.png"/>
      <br/>
      <img alt="1250397-20171003171922411-1667799605.png" src="https://i-blog.csdnimg.cn/blog_migrate/9d3b1541ce78a6ab5e2c38b046a56646.png"/>
      <br/>
      3.1.5再次点击新建表，按照下图输入，并且保存为sc。
      <br/>
      <img alt="1250397-20171003172032068-318522633.png" src="https://i-blog.csdnimg.cn/blog_migrate/fb944b39fc32fa4100a42f6213a876e3.png"/>
      <br/>
      <img alt="1250397-20171003172543146-1136318406.png" src="https://i-blog.csdnimg.cn/blog_migrate/042976d1e35a19038d569cf045ef452f.png"/>
      <br/>
      3.1.6在 数据库-&gt;SSMSTest-&gt;表 那里刷新一下，就会发现刚刚建立的三张表了。
      <br/>
      <img alt="1250397-20171003172747583-432658398.png" src="https://i-blog.csdnimg.cn/blog_migrate/c54059194660649c1cc1dfb67ecf2b17.png"/>
      <br/>
      （选择 数据库-&gt;SSMSTest-&gt;表-&gt;dbo.student 右键单击选择设计，就可以查看刚刚建立的表的具体信息了。）
     </p>
     <h5 id="通过sql语句建立表">
      <span id="jump3.2">
       3.2 通过SQL语句建立表
      </span>
     </h5>
     <p>
      3.2.1选择 数据库-&gt;SQLTest，然后选择新建查询。注意左上角的那个框中要选中SQLTest，如果不是的话就手动选择一下。
      <br/>
      <img alt="1250397-20171003173131458-1827436388.png" src="https://i-blog.csdnimg.cn/blog_migrate/5e8650c5d0af4c0dd1d5f4040aef592b.png"/>
      <br/>
      3.2.2在新建查询出来的界面中输入以下代码：
     </p>
     <pre><code class="language-sql"><code>--这是SQL中的注释信息，使用两个减号来注释。
drop table student    --删除表student
create table student  --创建表student
(sno char(4),
sname char(8),
sage int,
ssex char(2),
sdept char(20)
)

drop table course --删除表 course
create table course --创建表 course
(cno char(4),
cname char(8),
cpno char(4),
ccredit int
)

drop table sc --删除表 sc
create table sc --创建表 sc
(sno char(4),
cno char(4),
grade int
)</code></code></pre>
<p>
3.2.3 点击执行，会出现下面的警告。这是因为我们还没有创建表，所以没东西删除。只需要再次点击执行即可。
<br/>
<img alt="1250397-20171003173652630-773120966.png" src="https://i-blog.csdnimg.cn/blog_migrate/d0738361b4b7fdd284309bd3e8f4afb3.png"/>
<br/>
<img alt="1250397-20171003173819771-2050753167.png" src="https://i-blog.csdnimg.cn/blog_migrate/62596da4ee8efcb639eb6193c536c054.png"/>
<br/>
3.2.4 在 数据库-&gt;SQLTest-&gt;表 那里刷新一下，就会发现刚刚建立的三张表了。
<br/>
<img alt="1250397-20171003174008224-1493302942.png" src="https://i-blog.csdnimg.cn/blog_migrate/95a45a3fb02eac2b10b33dc4b325e286.png"/>
<br/>
（选择 数据库-&gt;SQLTest-&gt;表-&gt;dbo.student 右键单击选择设计，就可以查看刚刚建立的表的具体信息了。）
</p>
