    <p>
     1.进入数据库，可能会出现恢复挂起，如下图所示，点击数据库，弹出无法访问数据库
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/fb7f537e4b49d3a08e85301e453215c3.jpeg#pic_center">
      <br/>
      2.点击新建查询，输入以下代码。(把test换成你的数据库的名称)
     </img>
    </p>
    <pre><code class="prism language-sql"><span class="token keyword">USE</span> master
GO
<span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> test <span class="token keyword">SET</span> SINGLE_USER
GO
<span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> test <span class="token keyword">SET</span> EMERGENCY
GO
<span class="token keyword">DBCC</span> CHECKDB<span class="token punctuation">(</span>test<span class="token punctuation">,</span>REPAIR_ALLOW_DATA_LOSS<span class="token punctuation">)</span>
go
<span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> test <span class="token keyword">SET</span> ONLINE
GO
<span class="token keyword">ALTER</span> <span class="token keyword">DATABASE</span> test <span class="token keyword">SET</span> MULTI_USER
GO
</code></pre>
    <p>
     3.点击执行，再刷新一下就OK了。如下图所示。
