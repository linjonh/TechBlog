    <h2 id="%E4%B8%80%E3%80%81%E5%8A%A0%E5%AF%86%E7%A8%8B%E5%BA%8F%E5%8A%9F%E8%83%BD%E4%BB%8B%E7%BB%8D" name="%E4%B8%80%E3%80%81%E5%8A%A0%E5%AF%86%E7%A8%8B%E5%BA%8F%E5%8A%9F%E8%83%BD%E4%BB%8B%E7%BB%8D">
     一、加密程序功能介绍
    </h2>
    <h3 id="1%E3%80%81%E5%8A%A0%E5%AF%86%E7%94%A8%E9%80%94" name="1%E3%80%81%E5%8A%A0%E5%AF%86%E7%94%A8%E9%80%94">
     1、加密用途
    </h3>
    <p>
     对软件权限加密可适用于设备催付款，限制使用日期。
    </p>
    <h3 id="2%E3%80%81%E5%8A%9F%E8%83%BD" name="2%E3%80%81%E5%8A%9F%E8%83%BD">
     2、功能
    </h3>
    <p>
     一台机器一个注册码，每个码都不一样。加密后客户尝试修改电脑时间或者更换电脑均无效。
    </p>
    <h3 id="3%E3%80%81%E7%A8%8B%E5%BA%8F%E8%AF%B4%E6%98%8E" name="3%E3%80%81%E7%A8%8B%E5%BA%8F%E8%AF%B4%E6%98%8E">
     3、程序说明
    </h3>
    <p>
     Demo程序示例分为两部分，用户程序部分加密程序、一个是后台权限注册解密程序。
    </p>
    <h3 id="4%E3%80%81%E5%8A%A0%E5%AF%86%E8%BF%87%E7%A8%8B" name="4%E3%80%81%E5%8A%A0%E5%AF%86%E8%BF%87%E7%A8%8B">
     4、加密过程
    </h3>
    <p>
     读取
     <strong>
      <span style="color:#fe2c24">
       电脑CPU
      </span>
     </strong>
     编码、读取
     <span style="color:#fe2c24">
      <strong>
       电脑硬盘
      </strong>
     </span>
     编码，设置限定日期，生成注册码。注册码经过MD5加密，写入注册表、写入硬盘本地。（删除或者修改注册表中或者硬盘中的任何一个文件都将导致用户软件无法使用）
    </p>
    <h3 id="5%E3%80%81%E6%8E%88%E6%9D%83%E7%9A%84%E6%B3%A8%E5%86%8C%E6%96%87%E4%BB%B6%E4%BF%9D%E5%AD%98%E6%96%B9%E5%BC%8F" name="5%E3%80%81%E6%8E%88%E6%9D%83%E7%9A%84%E6%B3%A8%E5%86%8C%E6%96%87%E4%BB%B6%E4%BF%9D%E5%AD%98%E6%96%B9%E5%BC%8F">
     5、授权的注册文件保存方式
    </h3>
    <p>
     授权的注册码经过MD5加密技术加密后分别存储在注册表和本地硬盘中，经过MD5加密后的数据无法进行逆向解密。
    </p>
    <p>
     在授权使用时间天数范围内，用户不能修改该注册码或者删除文件，否则导致授权损坏用户软件无法使用软件。
     <img alt="" height="897" src="https://i-blog.csdnimg.cn/direct/64c7707bf0a84134b915aa0d2584313b.png" width="1457"/>
    </p>
    <p>
    </p>
    <h2 id="%E4%BA%8C%E3%80%81%E5%8A%A0%E5%AF%86%E7%A8%8B%E5%BA%8F%E4%BD%BF%E7%94%A8%E6%AD%A5%E9%AA%A4" name="%E4%BA%8C%E3%80%81%E5%8A%A0%E5%AF%86%E7%A8%8B%E5%BA%8F%E4%BD%BF%E7%94%A8%E6%AD%A5%E9%AA%A4">
     二、加密程序使用步骤
    </h2>

     在用户程序中，将机器码复制拷贝到解密程序中，设置授权使用天数。

     (如果是永久授权使用，则授权天数输入36500即可)
    </p>
    <h3 id="%E2%80%8B%E7%BC%96%E8%BE%912%E3%80%81%E6%AD%A5%E9%AA%A4%E4%BA%8C" name="%E2%80%8B%E7%BC%96%E8%BE%912%E3%80%81%E6%AD%A5%E9%AA%A4%E4%BA%8C">

     3、步骤三

    <h2 id="%E4%B8%89%E3%80%81%E6%A0%B8%E5%BF%83%E4%BB%A3%E7%A0%81%E8%AF%B4%E6%98%8E" name="%E4%B8%89%E3%80%81%E6%A0%B8%E5%BF%83%E4%BB%A3%E7%A0%81%E8%AF%B4%E6%98%8E">
     三、核心代码说明
    </h2>
    <h3 id="1%E3%80%81%E8%8E%B7%E5%8F%96%E7%94%B5%E8%84%91CPU%20%E4%BF%A1%E6%81%AF" name="1%E3%80%81%E8%8E%B7%E5%8F%96%E7%94%B5%E8%84%91CPU%20%E4%BF%A1%E6%81%AF">
     1、获取电脑CPU 信息
    </h3>
    <p>
     每个CPU信息出厂后都是独立且唯一的
    </p>
    <pre><code class="hljs">        /// &lt;summary&gt;
        /// 获取电脑CPU信息的方法
        /// &lt;/summary&gt;
        /// &lt;returns&gt;&lt;/returns&gt;
        public static string GetCpuId()
        {
            ManagementClass mc = new ManagementClass("Win32_Processor");
            ManagementObjectCollection moc = mc.GetInstances();

            string strCpuID = null;
            foreach (ManagementObject mo in moc)
            {
                strCpuID = mo.Properties["ProcessorId"].Value.ToString();
                break;
            }
            return strCpuID;
        }</code></pre>
    <h3 id="2%E3%80%81%E8%8E%B7%E5%8F%96%E7%A1%AC%E7%9B%98%E5%8D%B7%E6%A0%87%E5%8F%B7" name="2%E3%80%81%E8%8E%B7%E5%8F%96%E7%A1%AC%E7%9B%98%E5%8D%B7%E6%A0%87%E5%8F%B7">
     2、获取硬盘卷标号
    </h3>
    <p>
     每个硬盘标号出厂后都是独立且唯一的
    </p>
    <pre><code class="hljs">         ///&lt;summary&gt;
        /// 获取硬盘卷标号
        ///&lt;/summary&gt;
        ///&lt;returns&gt;&lt;/returns&gt;
        public static string GetDiskVolumeSerialNumber()
        {
            ManagementClass mc = new ManagementClass("win32_NetworkAdapterConfiguration");
            ManagementObject disk = new ManagementObject("win32_logicaldisk.deviceid=\"c:\"");
            disk.Get();
            return disk.GetPropertyValue("VolumeSerialNumber").ToString();
        }</code></pre>
    <h3 id="3%E3%80%81%E6%9C%BA%E5%99%A8%E7%A0%81%E7%94%9F%E6%88%90" name="3%E3%80%81%E6%9C%BA%E5%99%A8%E7%A0%81%E7%94%9F%E6%88%90">
     3、机器码生成
    </h3>
    <p>
     根据CPU信息和硬盘标号，生成独立唯一的机器码。
    </p>
    <p>
     机器码使用MD5加密，
    </p>
    <pre><code class="hljs">        ///&lt;summary&gt;
        /// 生成机器码
        ///&lt;/summary&gt;
        ///&lt;returns&gt;&lt;/returns&gt;
        public static  string GetMNum()
        {
            string strNum = DES.DESEncrypt((DateTime.Now.ToString("MMdd") + Form1.DiskID+Form1.CpuID), "12345678", "87654321");
            string strMNum = strNum.Substring(0, 24).ToUpper();    //截取前24位作为机器码
            return strMNum;
        }</code></pre>
    <h3 id="3%E3%80%81%C2%A0%E7%94%9F%E6%88%90%E6%B3%A8%E5%86%8C%E7%A0%81" name="3%E3%80%81%C2%A0%E7%94%9F%E6%88%90%E6%B3%A8%E5%86%8C%E7%A0%81">
     3、 生成注册码
    </h3>
    <p>
     注册码保存使用MD5加密，
    </p>
    <pre><code class="hljs">        ///&lt;summary&gt;
        /// 生成注册码
        ///&lt;/summary&gt;
        ///&lt;returns&gt;&lt;/returns&gt;
        public static string GetRNum()
        {
            SetIntCode();
            string strMNum = GetMNum();
            for (int i = 1; i &lt; charCode.Length; i++)   //存储机器码
            {
                charCode[i] = Convert.ToChar(strMNum.Substring(i - 1, 1));
            }
            for (int j = 1; j &lt; intNumber.Length; j++)  //改变ASCII码值
            {
                intNumber[j] = Convert.ToInt32(charCode[j]) + intCode[Convert.ToInt32(charCode[j])];
            }
            string strAsciiName = "";   //注册码
            for (int k = 1; k &lt; intNumber.Length; k++)  //生成注册码
            {

                if ((intNumber[k] &gt;= 48 &amp;&amp; intNumber[k] &lt;= 57) || (intNumber[k] &gt;= 65 &amp;&amp; intNumber[k]
                    &lt;= 90) || (intNumber[k] &gt;= 97 &amp;&amp; intNumber[k] &lt;= 122))  //判断如果在0-9、A-Z、a-z之间
                {
                    strAsciiName += Convert.ToChar(intNumber[k]).ToString();
                }
                else if (intNumber[k] &gt; 122)  //判断如果大于z
                {
                    strAsciiName += Convert.ToChar(intNumber[k] - 10).ToString();
                }
                else
                {
                    strAsciiName += Convert.ToChar(intNumber[k] - 9).ToString();
                }
            }
            return strAsciiName;
        }</code></pre>
