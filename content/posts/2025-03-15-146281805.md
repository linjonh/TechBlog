     GitHub：添加ssh密钥

    <p>
     GitHub账户中没有公有SSH密钥，需要添加一个新的公钥，下面是操作过程：
    </p>
    <ol>
     <li>
      <p>
       <strong>
        生成SSH密钥对
       </strong>
       ：
       <br/>
       没有生成SSH密钥对，在终端或Git Bash中使用以下命令来生成：
      </p>
      <pre><code class="prism language-sh">ssh-keygen <span class="token parameter variable">-t</span> ed25519 <span class="token parameter variable">-C</span> <span class="token string">"your_email@example.com"</span>
</code></pre>
      <p>
       这里，
       <code>
        -t
       </code>
       指定密钥类型为
       <code>
        ed25519
       </code>
       （推荐使用），
       <code>
        -C
       </code>
       后面跟的是你的电子邮件地址，用作密钥的标签（不必使用
       <code>
        @openssh.com
       </code>
       ）。
      </p>
      <p>
       <img alt="外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传" src="https://i-blog.csdnimg.cn/direct/0de6757fa2174bef90cff5108e3771fb.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        保存SSH密钥
       </strong>
       ：
       <br/>
       系统提示输入文件名来保存新的密钥对，可以直接按回车键接受默认文件名（
       <code>
        ~/.ssh/id_ed25519
       </code>
       ）
      </p>
      <p>
       <img alt="外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传" src="https://i-blog.csdnimg.cn/direct/afbfe51e153a472b93409be5a4e17817.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        设置密钥权限
       </strong>
       ：
       <br/>
       安全起见，应该设置私钥文件的权限，确保只有你可以读取它：
      </p>
      <pre><code class="prism language-sh"><span class="token function">chmod</span> <span class="token number">600</span> ~/.ssh/id_ed25519
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        查看公钥
       </strong>
       ：
       <br/>
       使用
       <code>
        cat
       </code>
       命令查看公钥内容：
      </p>
      <pre><code class="prism language-sh"><span class="token function">cat</span> ~/.ssh/id_ed25519.pub
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        添加公钥到GitHub账户
       </strong>
       ：
      </p>
      <ul>
       <li>
        登录你的GitHub账户
       </li>
       <li>
        点击右上角头像，选择“Settings”（设置）
       </li>
       <li>
        在侧边栏中点击“SSH and GPG keys”（SSH和GPG密钥）
       </li>
       <li>
        点击“New SSH key”（新建SSH密钥）按钮
       </li>
       <li>
        在“Title”（标题）字段中，输入一个描述性的标题
       </li>
       <li>
        在“Key”字段中，粘贴你之前复制的公钥内容
       </li>
       <li>
        点击“Add SSH key”（添加SSH密钥）按钮
       </li>
      </ul>
      <p>
       <img alt="外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传" src="https://i-blog.csdnimg.cn/direct/fce0bdc5d51f480ca30dfcdaf851e7df.png"/>
      </p>
     </li>
     <li>
      <p>
       <strong>
        测试SSH连接
       </strong>
       ：
       <br/>
       添加SSH密钥后，使用以下命令来测试SSH连接是否设置正确：
      </p>
      <pre><code class="prism language-sh"><span class="token function">ssh</span> <span class="token parameter variable">-T</span> git@github.com
</code></pre>
      <p>
       成功显示：
      </p>
      <p>
       <img alt="外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传" src="https://i-blog.csdnimg.cn/direct/d2ef73f584684f81898ce29f156b49ef.png"/>
      </p>
     </li>
    </ol>
