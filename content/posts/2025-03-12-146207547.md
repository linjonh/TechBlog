     注册code命令
    </h2>
    <ol>
     <li>
      启动 VSCode 编辑器,按 shift + command + p输入 shell command，选择 Install ‘code’ command in PATH 选项， 安装code 命令
      <ul>
       <li>
        此操作会把 code 命令添加到系统的环境变量里。
        <br/>
        <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/direct/69d1ca3eab034bb99937bdcdc37d7e65.png"/>
       </li>
      </ul>
     </li>
     <li>
      打开 iTerm2 终端
      <ul>
       <li>
        在 iTerm2 中，cd 代码库根目录, 使用 code . 命令,就可以打开 VSCode 并加载当前目录了
       </li>
      </ul>
     </li>
    </ol>
