    <p>
     由于进入到移动应用开发领域，关注到了鸿蒙，接触了HarmonyOS、openHarmony。组织起了一帮子对此感兴趣的大二和大三学生，一起来学习。
     <br/>
     昨天做过了宣讲，同学们表现出了很大的热情。为帮助同学们能在起步时落实自主的实践性学习，整理一些我现在知道的资源，为同学们提供一点学习的建议。
     <br/>
     深圳大学HarmonyOS校园组织木棉花可以作为我们对标学习的对象，他们在Gitee页面整理的资源，可以作为我们学习的最近参考，见
     <a href="https://gitee.com/hiharmonica/awesome-harmony-os-kapok" rel="nofollow">
      [链接]
     </a>
     <br/>
     HarmonyOS开发有南向北向之分。上北下南，南向深入硬件层、系统层，北向关注应用层。下面就先讲北，再说南。
    </p>
    <h4>
     <a id="1__5">
     </a>
     1. 北向的自主学习
    </h4>
    <p>
     可以先从Harmony应用程序开发学起，而后去感知更多。
     <br/>
     Harmony应用程序开发所用语言现有三种：JS、eTS和Java。从哪种语言入手都可以。综合各方而的因素，推荐从Java入手。
    </p>
    <h5>
     <a id="1_9">
     </a>
     建议1：利用官方文档和资源学习
    </h5>
    <p>
     直接开展“正统”的HarmonyOS开发。
     <br/>
     在所有学习资源中，最推荐的是官方的文档。网址：
     <a href="https://developer.harmonyos.com/" rel="nofollow">
      https://developer.harmonyos.com/
     </a>
     <br/>
     应用开发的官方文档首页提供了集成开发环境DevEco的
     <a href="https://developer.harmonyos.com/cn/develop/deveco-studio#download_beta" rel="nofollow">
      [下载链接]
     </a>
     。安装过程可以参考
     <a href="https://developer.huawei.com/consumer/cn/forum/topic/0202347337194250475?fid=0101587866109860105" rel="nofollow">
      [链接]
     </a>
     。
     <br/>
     对于应用开发的学习者而言，可以选重点关注文档–&gt;指南：
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/6f45e53b974a1d46cb84c826aa5617e9.png">
      <br/>
      可以先在“指南”中找到快速入门，按指引完成第一个项目。在指南的各个部分，对应用程序的各种组件会给出详细的说明，也有一些很好的示例可以使用。
      <br/>
      在文档部分，API参考、示例代码等均后在日后进行开发时最重要的资料。HarmonyOS处于一个高速迭代的时期，版本更新快，使用官方文档，方可获得最新的一手资料。
      <br/>
      对于入门学习者而言，HarmonyOS是值得依赖的宝贵资源，里面包括了华为支持的不少免费课程，例如《HarmonyOS应用开发系列课（基础篇）》，报名即可学习（体会华为为推广Harmony下的功夫）：
      <br/>
      <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/c31629d71534072f9eee2f2f6c3364b4.png">
       <br/>
       在
       <a href="https://ilearningx.huawei.com/portal/" rel="nofollow">
        华为在线课程平台
       </a>
       中也有很多免费的课程。武汉大学赵小刚老师主讲的
       <a href="https://ilearningx.huawei.com/portal/courses/HuaweiX+EBGTD9042/about" rel="nofollow">
        《HarmonyOS移动应用开发》
       </a>
       适合应用开发初学者。（在平台中搜索“移动应用开发”，会出来很多同类课程。）
       <br/>
       泡社区（由观摩到深度参与）也是学习的一个好途径。推荐
       <a href="https://harmonyos.51cto.com/" rel="nofollow">
        HarmonyOS社区
       </a>
       和
       <a href="https://www.harmonyos.com/cn/community/" rel="nofollow">
        官方的开发者社区
       </a>
       ，不断去做适合自己水平往上一点点的事情。
       <br/>
       无论用什么资源，看书、看视频、动手实践结合起来是王道。实践路线有很多，再次推荐参考深圳大学木棉花组织中同学们的工作，见
       <a href="https://gitee.com/hiharmonica/awesome-harmony-os-kapok" rel="nofollow">
        [链接]
       </a>
       。
       <br/>
       无论学习还是工作，应用Git已经成为一个基础性的需要，就参考贺老师本人出品的
       <a href="https://blog.csdn.net/sxhelijian/article/details/122991450?spm=1001.2014.3001.5501">
        《Git入门实践教程》
       </a>
       。
      </img>
     </img>
    </p>
    <h5>
     <a id="2Android_24">
     </a>
     建议2：从Android开发入手学习
    </h5>
    <p>
     从应用程序开发角度，先学Android开发（用Java而非Kotlin），体会移动应用开发，再入Harmony开发，也不算绕路。
     <br/>
     推荐利用明日学院出版的《Android开发从入门到精通》（清华大学出版社），视频链接：
     <a href="https://www.bilibili.com/video/av22836860" rel="nofollow">
      https://www.bilibili.com/video/av22836860
     </a>
     。
     <br/>
     若最终希望回到HarmonyOS开发，实践环节可以不用Android Studio，而用DevEco Studio，后者在安装环节似乎更顺一些（每进一次坑都是上天对你的恩赐，能力是在出坑中练出来的）。
    </p>
    <h5>
     <a id="3OS_29">
     </a>
     建议3：希望从OS层面开展工作的同学
    </h5>
    <p>
     OpenHarmony开源项目见
     <a href="https://gitee.com/openharmony" rel="nofollow">
      https://gitee.com/openharmony
     </a>
     。
    </p>
    <h4>
     <a id="2__32">
     </a>
     2. 南向的自主学习
    </h4>
    <p>
     南向开发的官方文档见
     <a href="https://device.harmonyos.com/" rel="nofollow">
      ［链接］
     </a>
     。
     <br/>
     DevEco Device Tool支持Windows和Ubuntu系统，是HarmonyOS面向智能设备开发者提供的一站式集成开发环境，支持HarmonyOS的组件按需定制，支持代码编辑、编译、烧录和调试等功能，支持C/C++语言，以插件的形式部署在Visual Studio Code上。
     <br/>
     利用官方文档学起来依然是这个方向的有效策略之一。
     <br/>
     在51CTO中发现一个视频教程
     <a href="https://edu.51cto.com/course/25739.html" rel="nofollow">
      《HarmonyOS物联网应用开发——基于Hi3861》
     </a>
     。
     <br/>
     参加软通教育的师资培训班，获得“启航KP_IOT智能开发套件”，配着各种传感模块，可以跑通整个的开发流程，做各种开发实验。
     <br/>
     <img alt="在这里插入图片描述" src="https://i-blog.csdnimg.cn/blog_migrate/d799a4491625bdaa57ee1f5157ec9337.png">
      <br/>
      南向暂不能作为我的主要方向，没有更深度体验，也就推荐到这里。
     </img>
    </p>
