    <h4>
     join() 和 split() 函数
    </h4>
    <p>
     join() 函数可以将指定的字符添加到字符串中。
     <br/>
     a=['my', 'name', 'shi', 'wzngz']
     <br/>
     print('.'.join(a))
    </p>
    <p>
     输出结果：my.name.shi.wzngz
    </p>
    <p>
     split() 函数可以用指定的字符分割字符串
    </p>
    <p>
     a="my name shi wzngz "
     <br/>
     print(a.split())
    </p>
    <p>
     输出结果：['my', 'name', 'shi', 'wzngz']
    </p>
    <h4>
     1、解释//、％、* *运算符？
    </h4>
    <p>
     //(Floor Division)-这是一个除法运算符，它返回除法的整数部分。
    </p>
    <p>
     例如：5 // 2 = 2
    </p>
    <p>
     ％(模数)-返回除法的余数。
    </p>
    <p>
     例如：5 ％ 2 = 1
    </p>
    <p>
     **(幂)-它对运算符执行指数计算。a ** b表示a的b次方。
    </p>
    <p>
     例如：5 ** 2 = 25、5 ** 3 = 125
    </p>
    <h4>
     2、Python中append，insert和extend的区别?
    </h4>
    <p>
     append：在列表末尾添加新元素。
     <br/>
     insert：在列表的特定位置添加元素。
     <br/>
     extend：通过添加新列表来扩展列表。
    </p>
    <p>
     例子：
     <br/>
     numbers = [1,2,3,4,5]
     <br/>
     numbers.append(6)
     <br/>
     print(numbers)
     <br/>
     [1,2,3,4,5,6]
    </p>
    <p>
     insert(position,value)
     <br/>
     numbers.insert(2,7)
     <br/>
     print(numbers)
     <br/>
     [1,2,7,3,4,5,6]
    </p>
    <p>
     numbers.extend([7,8,9])
     <br/>
     print(numbers)
     <br/>
     [1,2,7,3,4,5,6,7,8,9]
     <br/>
     numbers.append([4,5])
     <br/>
     [1,2,7,3,4,5,6,7,8,9,[4,5]]
    </p>
    <h4>
     3、break、continue、pass是什么？
    </h4>
    <p>
     break：在满足条件时，它将导致程序退出循环。
    </p>
    <p>
     continue：将返回到循环的开头，它使程序在当前循环迭代中的跳过所有剩余语句。
    </p>
    <p>
     pass：使程序传递所有剩余语句而不执行。
    </p>
    <h4>
     4、区分Python中的remove，del和pop？
    </h4>
    <p>
     remove：将删除列表中的第一个匹配值，它以值作为参数。
     <br/>
     del：使用索引删除元素，它不返回任何值。
     <br/>
     pop：将删除列表中顶部的元素，并返回列表的顶部元素。
    </p>
    <ul>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       del——指定索引值删除
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       <strong>
        # del 列表[索引值]
       </strong>
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      del a_list[1]
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # 源列表：['Mecell', 18, 'Python', True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # del删除数据后的列表：['Mecell', 'Python', True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       remove——默认移除第一个出现的元素
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      #
      <strong>
       列表.remove[删除对象]
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      a_list
      <strong>
       <strong>
        .remove['Python']
       </strong>
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # 源列表：['Mecell', 18, 'Python', True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # remove删除数据后的列表：['Mecell', 18, True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      从结果可以看出，列表里面有两个'Python'，但是实际上只是删除了第一个，最后一个并没有删除，这就是remove的特点，需要大家注意！
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      pop——括号内不添加索引值，则默认删除列表中的最后一个元素；反之则默认根据索引值删除
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       # 列表.pop() --删除最后一个元素
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       <strong>
        a_list.pop()
       </strong>
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # 源列表：['Mecell', 18, 'Python', True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # pop删除数据后的列表：
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      ['Mecell', 18, 'Python', True, None, [1, 2, 3]]
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      <strong>
       # 列表.pop(索引值) --指定索引值删除
      </strong>
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      a_list.pop(3)
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # 源列表：['Mecell', 18, 'Python', True, None, [1, 2, 3], 'Python']
     </li>
     <li style="margin-left:0.0001pt; text-align:justify">
      # pop删除数据后的列表：['Mecell', 18, 'Python', None, [1, 2, 3], 'Python']
     </li>
    </ul>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     解释Python中map()函数？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     map()函数将给定函数应用于可迭代对象(列表、元组等)，然后返回结果(map对象)。
     <br/>
     我们还可以在map()函数中，同时传递多个可迭代对象。
     <br/>
     numbers = (1, 2, 3, 4)
     <br/>
     result = map(lambda x: x + x, numbers)
     <br/>
     print(list(result))
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     zip函数
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     zip函数获取可迭代对象，将它们聚合到一个元组中，然后返回结果。
     <br/>
     zip()函数的语法是zip(*iterables)
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      例子：
     </strong>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     keys = ["name", "age", "city"]
     <br/>
     values = ["Alice", 30, "New York"]
     <br/>
     print(zip(keys,values))
     <br/>
     r1 = list(zip(keys, values))
     <br/>
     r2= dict(zip(keys, values))
     <br/>
     print(r1)
     <br/>
     print(r2)
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     输出
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     &lt;zip object at 0x000001D449607800&gt;
     <br/>
     [('name', 'Alice'), ('age', 30), ('city', 'New York')]
     <br/>
     {'name': 'Alice', 'age': 30, 'city': 'New York'}
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     5、举例说明Python中的range函数?
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     range：range函数返回从起点到终点的一系列序列。
     <br/>
     range(start, end, step)，第三个参数是用于定义范围内的步数。
     <br/>
     <strong>
      number
     </strong>
     <br/>
     for i in range(5):
     <br/>
     print(i)
     <br/>
     0,1,2,3,4
     <br/>
     <strong>
      (start, end)
     </strong>
     <br/>
     for i in range(1, 5):
     <br/>
     print(i)
     <br/>
     1,2,3,4
     <br/>
     <strong>
      (start, end, step)
     </strong>
     <br/>
     for i in range(0, 5, 2):
     <br/>
     print(i)
     <br/>
     0,2,4
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     6、==和is的区别是？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ==比较两个对象或值的相等性。
     <br/>
     is运算符用于检查两个对象是否属于同一内存对象。
     <br/>
     lst1 = [1,2,3]
     <br/>
     lst2 = [1,2,3]
     <br/>
     lst1 == lst2
     <br/>
     True
     <br/>
     lst1 is lst2
     <br/>
     False
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     7、如何更改列表的数据类型？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     要将列表的数据类型进行更改，可以使用tuple()或者set()。
     <br/>
     lst = [1,2,3,4,2]
     <br/>
     更改为集合
     <br/>
     set(lst) ## {1,2,3,4}
     <br/>
     更改为元组
     <br/>
     tuple(lst) ## (1,2,3,4,2)
    </p>

     8、Python中注释代码的方法有哪些？

    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">

     <br/>

    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     9、什么是lambda函数？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     Lambda函数是不带名称的单行函数，可以具有n个参数，但只能有一个表达式。也称为匿名函数。
     <br/>
     a = lambda x, y：x + y
     <br/>
     print(a(5, 6))
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     10、iterables和iterators之间的区别？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     iterable：可迭代是一个对象，可以对其进行迭代。在可迭代的情况下，整个数据一次存储在内存中。
     <br/>
     iterators：迭代器是用来在对象上迭代的对象。它只在被调用时被初始化或存储在内存中。迭代器使用next从对象中取出元素。
     <br/>
     <strong>
      List is an iterable
     </strong>
     <br/>
     lst = [1,2,3,4,5]
     <br/>
     for i in lst:
     <br/>
     print(i)
     <br/>
     <strong>
      iterator
     </strong>
     <br/>
     lst1 = iter(lst)
     <br/>
     next(lst1)
     <br/>
     1
     <br/>
     next(lst1)
     <br/>
     2
     <br/>
     for i in lst1:
     <br/>
     print(i)
     <br/>
     3,4,5
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     11、Python中的生成器是什么？
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     生成器(generator)的定义与普通函数类似，生成器使用yield关键字生成值。
     <br/>
     如果一个函数包含yield关键字，那么该函数将自动成为一个生成器。
     <br/>
     A program to demonstrate the use of generator object with next() A generator function
     <br/>
     def Fun():
     <br/>
     yield 1
     <br/>
     yield 2
     <br/>
     yield 3
     <br/>
     x is a generator object
     <br/>
     x = Fun()
     <br/>
     print(next(x))
     <br/>
     1
     <br/>
     print(next(x))
     <br/>
     2
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     12、什么是pickling和unpickling？
    </h4>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     pickling是将Python对象(甚至是Python代码)，转换为字符串的过程。
     <br/>
     unpickling是将字符串，转换为原来对象的逆过程。
    </p>
    <h4 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     13、解释*args和**kwargs？
    </h4>
    <h6 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     *args，如果在参数名前加*，代表这个参数为可变参数
    </h6>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <strong>
      知识点如下：
     </strong>
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     传参时，接受任意的位置参数
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     会将其参数整体组装变成一个元组进行存储，整体赋值给变量名为*args
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     通过上面参数的科普，*args为位置参数
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     例子1：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     def func(*args):
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     print(type(args))# 调用函数，传入2个数值
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     func(1, 2)  # 输出的内容为 &lt;class 'tuple'&gt; ，通过元组的格式进行存储
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     例子2：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     def add（* num）：
     <br/>
     sum = 0
     <br/>
     for val in num：
     <br/>
     sum = val + sum
     <br/>
     print（sum）
     <br/>
     add（4,5）
     <br/>
     add（7,4,6）
     <br/>
     9
     <br/>
     17
    </p>
    <h6 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     **kwargs，是当我们想将字典作为参数传递给函数时使用的。
    </h6>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     知识点如下：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     传参数，接受任意的关键字参数
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     会将其参数整体组装变成一个字典进行存储，整体赋值给变量名为**kwargs
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     具体例子如下：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     # 自定义函数
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     def func(**kwargs):
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     print(type(kwargs))
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     # 调用函数，传入2个数值
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     func(a=1 , b=2)  # 输出的内容为  &lt;class 'dict'&gt; ，通过字典的格式进行存储
    </p>
    <h6 style="margin-left:0.0001pt; margin-right:0px; text-align:left">
     *args 和 **kwargs 结合
    </h6>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     上文已经知道关键字参数要在位置参数之后
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     具体例子如下：
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     # 自定义函数，包含*args和**kwargs
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     def func(argument, *args, **kwargs):
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     print(argument)  # 输出1
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     print(args) # 输出2
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     print(kwargs) # 输出3
    </p>
