    <p align="center">
     <strong>
      <span style="font-size:24px;color:#ff0000;">
       第2部分 数据库SQL语言
      </span>
     </strong>
    </p>
    <p align="center">
     <strong>
      <span style="font-size:18px;color:#ff0000;">
       数据库简介
      </span>
     </strong>
    </p>
    <p>
     <span style="font-size:18px;">
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      数据库是个通用化的综合性的数据集合，它可以供各种用户共享且具有最小的冗余度和较高的数据与程序的独立性。目前，国际上主导的大型数据库管理系统有
      <span style="font-family:Times New Roman;">
       ORACLE
      </span>
      、
      <span style="font-family:Times New Roman;">
       SQL SERVER
      </span>
      、
      <span style="font-family:Times New Roman;">
       SYBASE
      </span>
      、
      <span style="font-family:Times New Roman;">
       INFORMIX
      </span>
      和
      <span style="font-family:Times New Roman;">
       INGRES
      </span>
      等。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      数据库中常用的编程语言是
      <span style="font-family:Times New Roman;">
       SQL
      </span>
      语言，按其功能可分为四大部分：
     </span>
    </p>
    <p>
     <img alt="" src="https://img-blog.csdn.net/20140529184823265?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemhvdXpoYW94aW9uZzEyMjc=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast">
      <span style="font-size:18px;">
      </span>
     </img>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (1)
      </span>
      数据定义语言
      <span style="font-family:Times New Roman;">
       (Data Definition Language
      </span>
      ，
      <span style="font-family:Times New Roman;">
       <strong>
        DDL
       </strong>
       )
      </span>
      ，用于定义、撤销和修改数据模式。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (2)
      </span>
      数据查询语言
      <span style="font-family:Times New Roman;">
       (Data Query Language
      </span>
      ，
      <span style="font-family:Times New Roman;">
       <strong>
        DQL
       </strong>
       )
      </span>
      ，用于查询数据。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (3)
      </span>
      数据操纵语言
      <span style="font-family:Times New Roman;">
       (Data Manipulation Language
      </span>
      ，
      <span style="font-family:Times New Roman;">
       <strong>
        DML
       </strong>
       )
      </span>
      ，用于增、删、改数据。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (4)
      </span>
      数据控制语言
      <span style="font-family:Times New Roman;">
       (Data Control Language
      </span>
      ，
      <span style="font-family:Times New Roman;">
       <strong>
        DCL
       </strong>
       )
      </span>
      ，用于数据访问权限的控制。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      其中，数据定义语言包括
      <span style="font-family:Times New Roman;">
       CREATE
      </span>
      、
      <span style="font-family:Times New Roman;">
       DROP
      </span>
      和
      <span style="font-family:Times New Roman;">
       ALTER
      </span>
      语句，数据查询语言包括
      <span style="font-family:Times New Roman;">
       SELECT
      </span>
      语句，数据操纵语言包括
      <span style="font-family:Times New Roman;">
       UPDATE
      </span>
      、
      <span style="font-family:Times New Roman;">
       DELETE
      </span>
      和
      <span style="font-family:Times New Roman;">
       INSERT
      </span>
      语句，数据控制语言包括
      <span style="font-family:Times New Roman;">
       GRANT
      </span>
      语句。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      在数据库中，除了以上四种
      <span style="font-family:Times New Roman;">
       SQL
      </span>
      语言外，还涉及到表、索引、存储过程和事务等概念，它们在实际编程中也是不可或缺的。
     </span>
    </p>
    <p>
     <span style="font-family:Times New Roman;font-size:18px;">
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      在实际的软件开发项目中，与各类数据库打交道可谓是家常便饭。在作者的日常工作中，经常用到的数据库有两种：
      <span style="font-family:Times New Roman;">
       Sybase
      </span>
      数据库和
      <span style="font-family:Times New Roman;">
       Oracle
      </span>
      数据库。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (1) Sybase
      </span>
      数据库
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       Sybase
      </span>
      公司于
      <span style="font-family:Times New Roman;">
       1987
      </span>
      年推出了
      <span style="font-family:Times New Roman;">
       Sybase
      </span>
      数据库产品。这是一种基于客户
      <span style="font-family:Times New Roman;">
       /
      </span>
      服务器体系结构的数据库，由于其具备开放、高性能等特点，在通信类软件产品中应用得比较的广泛。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      在运行
      <span style="font-family:Times New Roman;">
       Sybase
      </span>
      数据库脚本的时候，我们会用到一款名叫“
      <span style="font-family:Times New Roman;">
       Sybase Central
      </span>
      ”的客户端软件，如图
      <span style="font-family:Times New Roman;">
       1
      </span>
      所示。
     </span>
    </p>
    <p align="center">
     <img alt="" src="https://img-blog.csdn.net/20140529184906687?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemhvdXpoYW94aW9uZzEyMjc=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast"/>
    </p>
    <p align="center">
     <span style="font-size:14px;">

      <span style="font-family:Times New Roman;">

      </span>
      “
      <span style="font-family:Times New Roman;">
       Sybase Central
      </span>
      ”客户端软件界面
     </span>
    </p>
    <p>
     <span style="font-family:Times New Roman;font-size:18px;">
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       (2) Oracle
      </span>
      数据库
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      <span style="font-family:Times New Roman;">
       Oracle
      </span>
      数据库是
      <span style="font-family:Times New Roman;">
       Oracle
      </span>
      公司的一款关系数据库管理系统，目前在数据库市场上的份额为第一。其主要的优点是具备分布式处理能力，能够同时处理大量的数据而保持其性能的稳定性。正是因为如此，在用户量巨大
      <span style="font-family:Times New Roman;">
       (
      </span>
      通常在百万级以上
      <span style="font-family:Times New Roman;">
       )
      </span>
      的软件产品中，
      <span style="font-family:Times New Roman;">
       Oracle
      </span>
      数据库有着极大的优势和广泛的应用。
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      在运行
      <span style="font-family:Times New Roman;">
       Oracle
      </span>
      数据库脚本的时候，我们会用到一款名叫“
      <span style="font-family:Times New Roman;">
       PLSQL Developer
      </span>
      ”的客户端软件，如图
      <span style="font-family:Times New Roman;">
       2
      </span>
      所示。
     </span>
    </p>
    <p align="center">
     <img alt="" src="https://img-blog.csdn.net/20140529184941046?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvemhvdXpoYW94aW9uZzEyMjc=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast"/>
    </p>
    <p align="center">
     <span style="font-size:14px;">
      图
      <span style="font-family:Times New Roman;">
       2
      </span>
      “
      <span style="font-family:Times New Roman;">
       PLSQL Developer
      </span>
      ”客户端软件界面
     </span>
    </p>
    <p>
     <span style="font-family:Times New Roman;font-size:18px;">
     </span>
    </p>
    <p>
     <span style="font-size:18px;">
      另外，作为一个开源的关系型数据库管理系统，
      <span style="font-family:Times New Roman;">
       MySQL
      </span>
      在中小型软件产品中也有一定的应用。
     </span>
    </p>
