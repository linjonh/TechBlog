    <h4>
     <a id="_16">
     </a>
     补充说明
    </h4>
    <pre><code>yum 安装的redis.conf 在/etc/redis/redis.conf
</code></pre>
    <h4>
     <a id="_21">
     </a>
     语法
    </h4>
    <pre><code>redis-cli (选项) (参数)
</code></pre>
    <h4>
     <a id="_26">
     </a>
     选项
    </h4>
    <pre><code>-a 输入密码 
-n 选择数据库 若无此参数默认选中0数据库
</code></pre>
    <h4>
     <a id="_32">
     </a>
     参数
    </h4>
    <pre><code>set 添加数据
keys 用于查询   此参数后 可输入正则查询 | keys "*"
del  删除数据
</code></pre>
    <h4>
     <a id="_39">
     </a>
     实例
    </h4>
    <h5>
     <a id="_41">
     </a>
     连接服务端
    </h5>
    <p>
     <code>
      无密码链接:
     </code>
    </p>
    <pre><code>redis-cli
</code></pre>
    <p>
     <code>
      有密码链接:
     </code>
    </p>
    <pre><code>redis-cli
127.0.0.1:6379&gt; auth abc@2021
OK
</code></pre>
    <h5>
     <a id="_55">
     </a>
     添加数据
    </h5>
    <pre><code>redis-cli set bb 0 					#添加到第0个数据库 键为 `bb` 值为 `0`
redis-cli -n 1 set bb 1				#添加到第1个数据库 键为 `bb` 值为 `1`
redis-cli -a pass set bb 2			#添加到第0个数据库 键为 `bb` 值为 `2` ,且输入了密码
</code></pre>
    <h5>
     <a id="_62">
     </a>
     查询数据
    </h5>
    <pre><code>redis-cli  keys "*" 			#查询第0个数据库,所有键
redis-cli -n 2 keys "b*"		#查询第2个数据库,以b开头的键名称
redis-cli -a pass keys "*"		#查询第0个数据库,以b开头的键名称,且输入了密码
</code></pre>
    <h5>
     <a id="_69">
     </a>
     删除数据
    </h5>
    <p>
     <code>
      无密码 删除
     </code>
     (没有选择数据库的情况下 默认第0个数据库 )
    </p>
    <pre><code>redis-cli keys "*XXX*" | xargs redis-cli  del
</code></pre>
    <p>
     <code>
      有密码 删除
     </code>
     (删除
     <code>
      bai-
     </code>
     开头的)
    </p>
    <pre><code>redis-cli -a pwd keys "bai-*" | xargs redis-cli -a pwd del
</code></pre>
    <p>
     <code>
      有密码 删除
     </code>
     (删除第二个数据库 所有数据)
    </p>
    <pre><code>redis-cli -a pwd -n 2 keys "*" | xargs redis-cli -a pwd -n 2 del
</code></pre>
