# Mac 上录制视频有几种常见方式

在 Mac 上录制视频有几种常见方式，OBS 是其中一个很专业的选择，但并不是唯一方案。这里给你一个分层介绍，方便你根据需求来选：

⸻

1. 系统自带工具（轻量场景）  
   • QuickTime Player  
   • 打开 QuickTime → 菜单栏选择 文件 > 新建屏幕录制。  
   • 可以录制整个屏幕或选定区域。  
   • 支持同时录制麦克风声音。  
   • 优点：简单、免安装，系统自带。  
   • 缺点：功能基础，不能做复杂场景切换或推流。  
   • 快捷键截图录屏  
   • macOS Mojave 及以上：按 Shift + Command + 5，可选择录屏或录制选区。  
   • 简单直接，适合临时录制。

⸻

2. 第三方专业工具（进阶录制）  
   • OBS Studio  
   • 免费开源，功能非常强大。  
   • 支持多场景切换、叠加摄像头、文字、图片等。  
   • 可以本地录制，也能直播推流（YouTube、B站、Twitch）。  
   • 适合录制教程、课程、直播。  
   • ScreenFlow（付费）  
   • 专业录屏 + 编辑一体化软件。  
   • 特别适合需要录完视频直接剪辑、加特效的人。  
   • 对课程视频、培训视频用户很友好。  
   • Camtasia（付费）  
   • 类似 ScreenFlow，偏向企业/教育场景。  
   • 内置大量模板，方便快速生成教学视频。

⸻

3. 选择建议  
   • 只需简单录屏 → 用 QuickTime 或快捷键 (Shift+Command+5)。  
   • 需要专业录制/直播 → 推荐 OBS Studio（免费且功能全面）。  
   • 录制 + 编辑一体化 → 用 ScreenFlow / Camtasia（付费）。
