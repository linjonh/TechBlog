    <h3>
     <a id="1__0">

     1. 创建统一资源管理类
    </h3>
    <ol>
     <li>

      <br/>
      manager.dart），将所有图片路径集中管理。这样在引用图片时，不需要每次都手动输入完整路径，只需通过常量引用即可。
     </li>
    </ol>

    <h3>
     <a id="2__flutter_gen__19">
     </a>
     2. 使用 flutter_gen 自动生成代码
    </h3>
    <p>
     可以使用 flutter_gen 这个插件，它会根据 pubspec.yaml 中声明的资源文件自动生成代码。
    </p>
    <ol>
     <li>
      添加依赖
      <br/>
      在 pubspec.yaml 中添加：
     </li>
    </ol>
    <pre><code class="prism language-csharp">dev_dependencies<span class="token punctuation">:</span>
  build_runner<span class="token punctuation">:</span> <span class="token class-name">any</span>
  flutter_gen_runner<span class="token punctuation">:</span> any
</code></pre>
    <ol start="2">
     <li>
      配置资源路径
      <br/>
      在 pubspec.yaml 中正确配置资源路径：
     </li>
    </ol>
    <pre><code class="prism language-csharp">flutter<span class="token punctuation">:</span>
  assets<span class="token punctuation">:</span>
    <span class="token operator">-</span> assets<span class="token operator">/</span>images<span class="token operator">/</span>
</code></pre>
    <ol start="3">
     <li>
      运行生成命令
     </li>
    </ol>
    <p>
     运行以下命令生成资源管理代码：
    </p>
    <pre><code class="prism language-csharp">flutter packages pub run build_runner build
</code></pre>
    <p>
     生成的代码通常会在 lib/gen/assets.gen.dart 文件中，可以直接通过常量引用图片：
    </p>
    <pre><code class="prism language-csharp">import 'package<span class="token punctuation">:</span>your_project<span class="token operator">/</span>gen<span class="token operator">/</span>assets<span class="token punctuation">.</span>gen<span class="token punctuation">.</span>dart'<span class="token punctuation">;</span>
Image<span class="token punctuation">.</span><span class="token function">asset</span><span class="token punctuation">(</span>Assets<span class="token punctuation">.</span>images<span class="token punctuation">.</span>logo<span class="token punctuation">.</span>path<span class="token punctuation">)</span><span class="token punctuation">;</span>
</code></pre>
    <p>
     flutter_gen 不仅支持图片，还可以管理字体、SVG 文件等其他资源。
     <br/>
     4. 自定义导出文件名称
     <br/>
     在pubspec.yaml新增以下内容，output就是要导出的文件名
    </p>
    <pre><code class="prism language-csharp">   flutter_gen<span class="token punctuation">:</span>
    output<span class="token punctuation">:</span> lib<span class="token operator">/</span>picture<span class="token operator">/</span>
