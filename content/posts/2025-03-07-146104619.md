     <p>
      课题摘要: 本文详细介绍了C++中继承和派生的概念、语法、类型、构造和析构过程、函数重写、多重继承和虚继承。继承允许派生类从基类继承属性和行为并进行扩展或修改。继承类型包括公有、保护和私有继承，影响基类成员在派生类中的访问权限。派生类构造和析构时，先调用基类相应函数。派生类可重写基类虚函数实现多态。C++支持多重继承，虚继承解决菱形继承问题。合理使用继承可构建层次清晰、结构合理的类体系。
     </p>
    </blockquote>
    <hr/>
    <p>
     在C++中，继承和派生是面向对象编程的核心概念，允许一个类（派生类）从另一个类（基类）继承属性和行为，并在此基础上进行扩展或修改。
    </p>
    <h2>
     <a id="_8">
     </a>
     一、基本概念
    </h2>
    <ul>
     <li>
      <strong>
       基类（Base Class）
      </strong>
      ：被继承的类，也称为父类。
     </li>
     <li>
      <strong>
       派生类（Derived Class）
      </strong>
      ：继承基类的类，也称为子类。
     </li>
    </ul>
    <p>
     派生类继承了基类的成员（数据成员和成员函数），并可以添加新的成员或重写基类的成员函数。
    </p>
    <h2>
     <a id="_15">
     </a>
     二、继承的语法
    </h2>
    <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">BaseClass</span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// 基类成员</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">class</span> <span class="token class-name">DerivedClass</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token class-name">access</span><span class="token operator">-</span><span class="token class-name">specifier</span> <span class="token class-name">BaseClass</span></span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// 派生类成员</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
    <ul>
     <li>
      <code>
       access-specifier
      </code>
      可以是
      <code>
       public
      </code>
      、
      <code>
       protected
      </code>
      或
      <code>
       private
      </code>
      ，决定了基类成员在派生类中的访问权限。
     </li>
    </ul>
    <h2>
     <a id="_29">
     </a>
     三、继承类型
    </h2>
    <ul>
     <li>
      <p>
       <strong>
        公有继承（public）
       </strong>
       ：
      </p>
      <ul>
       <li>
        基类的
        <code>
         public
        </code>
        成员在派生类中仍为
        <code>
         public
        </code>
        。
       </li>
       <li>
        基类的
        <code>
         protected
        </code>
        成员在派生类中仍为
        <code>
         protected
        </code>
        。
       </li>
       <li>
        基类的
        <code>
         private
        </code>
        成员在派生类中不可访问。
       </li>
      </ul>
      <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Base</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token keyword">int</span> publicVar<span class="token punctuation">;</span>
<span class="token keyword">protected</span><span class="token operator">:</span>
    <span class="token keyword">int</span> protectedVar<span class="token punctuation">;</span>
<span class="token keyword">private</span><span class="token operator">:</span>
    <span class="token keyword">int</span> privateVar<span class="token punctuation">;</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">class</span> <span class="token class-name">Derived</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token keyword">public</span> <span class="token class-name">Base</span></span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// publicVar 是 public</span>
    <span class="token comment">// protectedVar 是 protected</span>
    <span class="token comment">// privateVar 不可访问</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        保护继承（protected）
       </strong>
       ：
      </p>
      <ul>
       <li>
        基类的
        <code>
         public
        </code>
        和
        <code>
         protected
        </code>
        成员在派生类中都变为
        <code>
         protected
        </code>
        。
       </li>
       <li>
        基类的
        <code>
         private
        </code>
        成员在派生类中不可访问。
       </li>
      </ul>
      <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Derived</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token keyword">protected</span> <span class="token class-name">Base</span></span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// publicVar 是 protected</span>
    <span class="token comment">// protectedVar 是 protected</span>
    <span class="token comment">// privateVar 不可访问</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
     <li>
      <p>
       <strong>
        私有继承（private）
       </strong>
       ：
      </p>
      <ul>
       <li>
        基类的
        <code>
         public
        </code>
        和
        <code>
         protected
        </code>
        成员在派生类中都变为
        <code>
         private
        </code>
        。
       </li>
       <li>
        基类的
        <code>
         private
        </code>
        成员在派生类中不可访问。
       </li>
      </ul>
      <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Derived</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token keyword">private</span> <span class="token class-name">Base</span></span> <span class="token punctuation">{<!-- --></span>
    <span class="token comment">// publicVar 是 private</span>
    <span class="token comment">// protectedVar 是 private</span>
    <span class="token comment">// privateVar 不可访问</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>
</code></pre>
     </li>
    </ul>
    <h2>
     <a id="__77">
     </a>
     四、 派生类的构造和析构
    </h2>
    <ul>
     <li>
      <strong>
       构造函数
      </strong>
      ：派生类的构造函数会先调用基类的构造函数，再执行派生类的构造函数体。
     </li>
     <li>
      <strong>
       析构函数
      </strong>
      ：派生类的析构函数会先执行派生类的析构函数体，再调用基类的析构函数。
     </li>
    </ul>
    <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Base</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token function">Base</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Base Constructor"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span> <span class="token punctuation">}</span>
    <span class="token operator">~</span><span class="token function">Base</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Base Destructor"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span> <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">class</span> <span class="token class-name">Derived</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token keyword">public</span> <span class="token class-name">Base</span></span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token function">Derived</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Derived Constructor"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span> <span class="token punctuation">}</span>
    <span class="token operator">~</span><span class="token function">Derived</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span> cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Derived Destructor"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span> <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    Derived d<span class="token punctuation">;</span>
    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <p>
     输出：
    </p>
    <pre><code>Base Constructor
Derived Constructor
Derived Destructor
Base Destructor
</code></pre>
    <h2>
     <a id="Override_109">
     </a>
     五、函数重写（Override）
    </h2>
    <p>
     派生类可以重写基类的虚函数（
     <code>
      virtual
     </code>
     ），以实现多态。
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Base</span> <span class="token punctuation">{<!-- --></span>

        cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Base show"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">class</span> <span class="token class-name">Derived</span> <span class="token operator">:</span> <span class="token base-clause"><span class="token keyword">public</span> <span class="token class-name">Base</span></span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token keyword">void</span> <span class="token function">show</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token keyword">override</span> <span class="token punctuation">{<!-- --></span>
        cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Derived show"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

<span class="token keyword">int</span> <span class="token function">main</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
    Base<span class="token operator">*</span> b <span class="token operator">=</span> <span class="token keyword">new</span> <span class="token function">Derived</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>
    b<span class="token operator">-&gt;</span><span class="token function">show</span><span class="token punctuation">(</span><span class="token punctuation">)</span><span class="token punctuation">;</span>  <span class="token comment">// 输出: Derived show</span>

    <span class="token keyword">return</span> <span class="token number">0</span><span class="token punctuation">;</span>
<span class="token punctuation">}</span>
</code></pre>
    <h2>
     <a id="_136">
     </a>
     六、多重继承
    </h2>
    <p>
     C++支持多重继承，即一个派生类可以从多个基类继承。
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Base1</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token keyword">void</span> <span class="token function">show1</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>

    <span class="token punctuation">}</span>

<span class="token keyword">class</span> <span class="token class-name">Base2</span> <span class="token punctuation">{<!-- --></span>
<span class="token keyword">public</span><span class="token operator">:</span>
    <span class="token keyword">void</span> <span class="token function">show2</span><span class="token punctuation">(</span><span class="token punctuation">)</span> <span class="token punctuation">{<!-- --></span>
        cout <span class="token operator">&lt;&lt;</span> <span class="token string">"Base2 show"</span> <span class="token operator">&lt;&lt;</span> endl<span class="token punctuation">;</span>
    <span class="token punctuation">}</span>
<span class="token punctuation">}</span><span class="token punctuation">;</span>

    <h2>
     <a id="_166">
     </a>
     七、虚继承
    </h2>
    <p>
     虚继承用于解决多重继承中的菱形继承问题，确保基类只被继承一次。
    </p>
    <pre><code class="prism language-cpp"><span class="token keyword">class</span> <span class="token class-name">Base</span> <span class="token punctuation">{<!-- --></span>

    <h2>
     <a id="_192">
     </a>
     八、总结
    </h2>
    <ul>
     <li>
      继承允许派生类复用基类的代码。
     </li>
     <li>
      继承类型决定了基类成员在派生类中的访问权限。
     </li>
     <li>
      派生类可以重写基类的虚函数以实现多态。
     </li>
     <li>
      多重继承和虚继承提供了更灵活的继承机制，但也增加了复杂性。
     </li>
    </ul>
    <p>
     通过合理使用继承和派生，可以构建出层次清晰、结构合理的类体系。
    </p>
