     IEC61850标准下MMS 缓存报告控制块 ResvTms详细解析
    </h1>
   </div>
  </div>

   <div class="htmledit_views" id="content_views">
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     IEC61850标准是电力系统自动化领域唯一的全球通用标准。IEC61850通过标准的实现，使得智能变电站的工程实施变得规范、统一和透明，这大大提高了变电站自动化系统的技术水平和安全稳定运行水平。
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px">
     在 IEC61850 标准体系中，ResvTms（reservation time，保留时间）作为报告控制块（Report Control Block, RCB）的关键属性之一，主要用于缓存报告控制块（BRCB）,非缓存报告控制块用另外的参数实现类似功能。本文将对IEC61850标准下MMS 缓存报告控制块 ResvTms详细解析，分享给对IEC61850感兴趣的朋友~
    </p>
    <p style="margin-left:0.0001pt; margin-right:0px; text-align:center">
     <img alt="" src="https://i-blog.csdnimg.cn/direct/8bf79be474ca4c85b1cd947790f50a3f.jpeg"/>
    </p>
    <h3 style="text-align:justify">
     <strong>
      <strong>
       <strong>
        1. ResvTms 的定义
       </strong>
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ResvTms 是一个INT16型属性，用于表示报告控制块是否还在被占用状态。它通常用于缓存报告控制块（BRCB），而非缓存报告控制块（URCB）则没有该属性（使用另外的属性）。这是一个可选地属性参数，可以通过获取数据定义或者询问服务器开发工作者，确定是否存在相关属性以及是否支持修改等情况。
    </p>
    <h3 style="text-align:justify">
     <strong>
      <strong>
       <strong>
        2. ResvTms 的作用
       </strong>
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      ①　占用状态：
     </strong>
     <span style="color:#fe2c24">
      ResvTms
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#fe2c24">
       值为-1，
      </span>
      <span style="color:#060607">
       应表示BRCB当前根据配置被专门保留给一组特定的客户端。
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       此情况，需要服务端特定，一般不支持。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      ②　动态保留状态：
     </strong>
     <span style="color:#fe2c24">
      ResvTms
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#fe2c24">
       大于0，
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       表示BRCB被动态保留。该值表示在
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       断开关联
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       后，保留状态将
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       保持
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       的秒数。
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       到达
      </span>
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       保留时间后，ResvTms值应被本地重置为零（0）。
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     <strong>
      ③　未占用状态：
     </strong>
     <span style="color:#fe2c24">
      ResvTms 为
     </span>
     <span style="background-color:#ffffff">
      <span style="color:#fe2c24">
       值为零（0），
      </span>
      <span style="color:#060607">
       表示BRCB未被保留。
      </span>
     </span>
    </p>
    <h3 style="text-align:justify">
     <strong>
      <strong>
       <strong>
        3. ResvTms 的使用场景
       </strong>
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     在缓存报告控制块中，如果存在ResvTms属性，客户端在使能报告之前应该先校验 ResvTms 值的具体情况，具体可参考 ResvTms 的作用，以确保操作的正确性和有效性。
    </p>
    <h3 style="text-align:justify">
     <strong>
      <strong>
       <strong>
        4.  ResvTms 的保留时间机制
       </strong>
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     当客户端使能报告后，ResvTms 如果大于0，表示服务端会为该客户端保留报告控制块使用权（ResvTms等于-1未特殊情况，可参考对应的作用描述）。当客户端因网络中断等原因取消使能，服务端会保留该报告控制块一段时间（ResvTms 指定的时间），期间其他客户端无法使能该报告控制块，从而确保数据的一致性和可靠性。
    </p>
    <h3 style="text-align:justify">
     <strong>
      <strong>
       <strong>
        5.  ResvTms 的设置规则
       </strong>
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <strong>
       <span style="color:#060607">
        <strong>
         SetBRCBValues请求（用于设置ResvTms）应满足以下条件：
        </strong>
       </span>
      </strong>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       ①
      </span>
      <span style="color:#060607">
       如果BRCB的ResvTms值
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         非零
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ，并且SetBRCBValues请求是由
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         未保留该BRCB的其他客户端
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       发起的，则
      </span>
      <span style="color:#060607">
       回复
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         负响应
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       ②
      </span>
      <span style="color:#060607">
       如果BRCB的ResvTms值为–1，则
      </span>
      <span style="color:#060607">
       回复
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         负响应
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       ③
      </span>
      <span style="color:#060607">
       如果要设置的ResvTms值为
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         负数
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ，则
      </span>
      <span style="color:#060607">
       回复
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         负响应
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       ④
      </span>
      <span style="color:#060607">
       如果BRCB的ResvTms值为
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         零（0）
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ，并且要设置的值为
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         零（0）或正数
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ，则
      </span>
      <span style="color:#060607">
       回复
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         正响应
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       ⑤
      </span>
      <span style="color:#060607">
       如果SetBRCBValues请求满足以下条件，则
      </span>
      <span style="color:#060607">
       回复
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         正响应
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ：
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       a) 设置的ResvTms值为
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         大于等于0
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       ；
      </span>
     </span>
    </p>
    <p style="margin-left:.0001pt; margin-right:0; text-align:left">
     <span style="background-color:#ffffff">
      <span style="color:#060607">
       b) 该请求是由
      </span>
      <strong>
       <span style="color:#060607">
        <strong>
         通过正数值保留了BRCB的客户端
        </strong>
       </span>
      </strong>
      <span style="color:#060607">
       发起的。
      </span>
     </span>
    </p>
    <h3 style="margin-left:0.0001pt; margin-right:0px; text-align:justify">
     <strong>
      <strong>
       6.结语
      </strong>
     </strong>
    </h3>
    <p style="margin-left:.0001pt; margin-right:0; text-align:justify">
     ResvTms 是 IEC61850 标准中用于 MMS（制造报文规范）缓存报告控制块的一个关键属性，其主要作用是标识报告控制块的占用时间。在客户端订阅报告的过程中，ResvTms 起到了至关重要的作用。通过设置 ResvTms，可以确保报告控制块在同一时间仅被一个客户端所使用，从而有效避免了多个客户端同时访问导致的资源冲突问题。这种机制不仅保障了数据传输的准确性和可靠性，还使得客户端能够稳定地接收报告数据，对于基于 IEC61850 标准的电力系统通信等应用场景具有重要的实际意义。
    </p>
