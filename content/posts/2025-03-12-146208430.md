    <h2>
     1.具体要求如下：
    </h2>
    <ul>
     <li>
      基于rockylinux:8基础镜像；
     </li>
     <li>
      指定作者为Chinaskill；
     </li>
     <li>
      安装Redis服务，暴露6379端口；
     </li>
     <li>
      设置服务自启动。
     </li>
    </ul>
    <p>
     <strong>
      总结步骤：基于centos7基础镜像，安装Redis，配置启动命令，暴露端口，构建镜像，标记并推送。
     </strong>
    </p>
    <h2>
     2.创建Dockerfile
    </h2>
    <p>
     宿主机：
    </p>
    <p>
     创建redis目录并进入: mkdir redis  ;  cd redis
    </p>
    <p>
     vim Dockerfile
    </p>
    <pre><code>FROM rockylinux:8
MAINTAINER Chinaskill &lt;Chinaskill@123.com&gt;
RUN yum install -y redis
RUN sed -i.bak 's/bind 127.0.0.1/bind 0.0.0.0/' /etc/redis.conf
EXPOSE 6379
CMD ["/usr/bin/redis-server",, "/etc/redis.conf"]

</code></pre>
    <p>
    </p>
    <h2>
     3.制作镜像
    </h2>
    <p>
     docker build -t redis:v1  .
    </p>
    <h2>
     <img alt="" height="102" src="https://i-blog.csdnimg.cn/direct/19922a84472443f1b6b5fb0eb96b039a.png" width="499"/>
    </h2>
    <h2>
     4.查看镜像
    </h2>
    <h2>
     <img alt="" height="67" src="https://i-blog.csdnimg.cn/direct/9122a1ff33944e7ca638735f16131da6.png" width="937"/>
    </h2>
    <p>
    </p>
    <h2>
     5.
     <strong>
      标记镜像为私有仓库格式
     </strong>
    </h2>
    <p>
     docker tag redis:v1 192.168.234.15:5050/openlab/redis:v1
    </p>
    <p>
     <img alt="" height="65" src="https://i-blog.csdnimg.cn/direct/f696aa42a1094561bcdc0646f46e4a5f.png" width="906"/>
    </p>
    <h3>
     6.登录到私有仓库并推送镜像
    </h3>
    <p>
     docker login 192.168.234.15:5050
    </p>
    <p>
     <img alt="" height="145" src="https://i-blog.csdnimg.cn/direct/14b5aa49c4ff4c7ab21f4449b789acde.png" width="801"/>
    </p>
    <p>
     docker push 192.168.234.15:5050/redis:v1
    </p>
    <p>
     <img alt="" height="480" src="https://i-blog.csdnimg.cn/direct/0e91e0a2dca24e258b7ee402628e54c4.png" width="1261"/>
    </p>
    <p>
     推送成功！
    </p>
    <p>
    </p>
    <h2>
     7.redis测试（宿主机上）
    </h2>
    <p>
     docker run -d -p 6379:6379 redis:v1
    </p>
    <p>
     <br/>
     <img alt="" height="91" src="https://i-blog.csdnimg.cn/direct/1683c6ca2d2d4190a1c1f17e8898f3a9.png" width="1624"/>
    </p>
    <p>
     yum  install -y redis
    </p>
    <p>
     systemctl start redis
    </p>
