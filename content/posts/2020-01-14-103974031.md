![Alt](https://i-blog.csdnimg.cn/blog_migrate/e5967ffcc9a5f35d81e6452e8e994eed.png#pic_center)
  
大家看到这页内容，这些技术都很熟悉，我们现在处于数字经济时代中，数据是资产，AI是能力，安全是基础，物联网是连接，云是操作系统，通过区块链去中心化技术，实现了数据的共享。虽然区块链有非常重要的潜力、非常重要的地位，但区块链项目应用场景还比较单一。京东作为在全球范围内少数拥有区块链的丰富应用场景和落地场景的大型企业，拥有非常多的经验。接下来我会着重介绍下，京东云在区块链这块的技术探索和应用实践。

本次分享包括两大部分：
  
第一部分是区块链企业级应用，包含平台技术架构及行业应用案例
  
第二部分是区块链数据服务，也是分别从技术到应用介绍

先说第一部分

![Alt](https://i-blog.csdnimg.cn/blog_migrate/a81610346f53d51105d2542bd8f2da26.png#pic_center)
  
首先，我先从技术角度介绍下京东云企业级区块链BaaS平台，我们首先提供了区块链的服务，这个服务来讲简单分为几层，底下是IaaS层，这是我们整个区块链可以放在整个公有云，私有云、混合云体系里面。实现了30秒就可以构建一个全新的 BaaS 平台，BaaS 平台支持水平扩展，动态扩缩容。

第二，我们提供区块链底层平台，支持主流开源的联盟链Fabric、同样支持京东自研的 JD Chain，并且都做了优化。Fabric 优化到5000 TPS+，JD Chain能达到10000TPS+。目前JD Chain已经开源，这是我们的JD Chain官网链接：http://ledger.jd.com/。

![Alt](https://i-blog.csdnimg.cn/blog_migrate/6d842acbd3ac0f09218888dfa6441c2c.png#pic_center)
  
这是我们整体的一个产品架构图，接下来，我从京东在各个场景的真实案例介绍区块链技术对各行各业的普惠。

![Alt](https://i-blog.csdnimg.cn/blog_migrate/1686db3f59fd26744d5b4483f059681d.png#pic_center)
  
这个是零售场景的中的应用：防伪溯源

为了追求商品品质，大家可能会使用跨境电商上购物，但你可能会关注两个问题：比如我买了一罐奶粉，上面写着新西兰产地，到底是不是新西兰产的，如果它真的是，它在路上全链条从国际运输、保税仓、国内运输送到我手里的时候有没有发生什么样的变化。

而京东区块链防伪溯源平台已经接入了700多家顶级品牌商，有600万+查询访问量。在商品详情页用户可以看到商品生产，加工，运输，销售，售后等全链路的商品信息，从而实现了商品信息实时查询，实时追溯。

除了电商的场景外，药品溯源我们可以保证疫苗从出厂到中间的运输、冷链到使用，保证全链条的监管。

![Alt](https://i-blog.csdnimg.cn/blog_migrate/7a6ccdd9f494dbdc58b0106ff5e7e4f9.png#pic_center)
  
金融场景中的应用：数字存证

关于区块链数字存证平台，目前京东与广州互联网法院，北京互联网法院，以及一些第三方仲裁机构一起构建了一个网通法院平台。最大的价值是在这个平台上的数字存证信息是有效的，可信赖的；通过区块链技术大大缩短了仲裁的时间，从传统的 60 天缩短到只需要 7 天。

![Alt](https://i-blog.csdnimg.cn/blog_migrate/b00aab14325c5c8f93f97b40733e1711.png#pic_center)
  
第二个金融的案例是资产证券化案例。

比如京东的白条资产，白条是坏账率比较低，最大的问题是每一笔金额非常少，同时笔数非常多，而资产证券化整个的业务流程非常长，所有的认证方非常多。包括计划管理人、律师、会计、信用评级等等一系列。那么想要达成每一笔资产的确认，都是需要非常长的周期。而通过区块链把所有的信用联盟能够联系起来，就可以达到高效的共识。

京东区块链ABS案例荣获 “2019中国区块链优秀应用案例”，并且被成功收录于全球权威咨询与研究机构Forrester 2019年发布的区块链亚太区数字生态系统报告，亚太地区仅有四家厂商案例被收录，京东是中国唯一一家案例入选厂商。以上就是我今天的第一部分的分享内容。

**陈思：**
学习了，第一部分结束之后我其实有两个问题想要问一下朱老师，第一个是关于您刚刚讲到的应用场景，您能给我们讲解下目前区块链技术运用最广泛的应用场景吗？以及2020年区块链场景应用中最先会落地的是什么应用？有什么阻碍吗？

**朱琅：**
在实际业务场景中，目前我看到的是区块链技术在溯源、存证的场景中应用的最为广泛，也是大家提的最多的。然后关于2020年，其实从整体的基调上看，国家还是认可区块链技术的价值的，所以我觉得在民生方面的应用可能会最先落地。比如说企业办证中心，过去办理企业营业执照，你需要跑工商局，公安局，银行，税务局才能最终办下来，用了区块链技术，所有相关内容或者证明都上链之后，可以在一个地方就把所有事情都干了，大大提高办事效率。但这些场景因为涉及单位比较多，协作沟通上会遇到很大的挑战，因此，真正落地实施还有很长的路要走。

**陈思：**
感谢朱老师的解答，您刚刚也提到了区块链溯源的应用，第二个问题我想问，区块链溯源技术您觉得最具价值的又是什么应用？

**朱琅：**
我觉得在溯源上，关乎老百姓切身利益的应用是最有价值的，比如疫苗溯源，通过区块链技术来解决假疫苗问题，这个价值就很大。

**陈思：**
嗯嗯，能解决切实需求的的确意义重大。谢谢朱老师，关于第一部分的问题目前就这些，您可以接着给我们分享后面的内容。

**朱琅：**
ok，那我开始第二部分的分享，我会详解介绍区块链数据服务 BDS。先从技术角度：

![Alt](https://i-blog.csdnimg.cn/blog_migrate/37ae7e0e65323219a897916ab7254c01.png#pic_center)
  
区块链数据服务 BDS 是我们发布的第二款区块链产品，京东云想做一件什么事情呢，我们希望把孤岛的数据可以聚合起来，放到统一一个地方；希望把离散的，非结构的数据变成有关联的，结构化的数据。

目前我们接入了包括 BTC，ETH，XRP，LTC，BCH 等公有链，通过技术手段获取了交易所，论坛，资讯等一些第三方数据。以上的数据都会导入到我们的大数据分析平台，经过数据分析加工之后，得到了像交易金额，交易次数，交易金额分布，交易手续费，新增地址，活跃地址，地址余额分布，地址分类等数据指标。

最后，为了方便大家体验和使用，我们提供了区块链数据 BI 工具，支持 SQL 查询 和 条件查询，支持对查询结果包括但不限于表格，曲线，柱状图等多样的数据可视化方式；还有包括数据智能提供数据实时更新，实时监控等其他功能。

这个是我们的产品官网地址：https://bds.jdcloud.com/
  
这是BI工具地址：https://bds-console.jdcloud.com/

那在实际的场景中对于用户来说，有哪些应用呢，我给大家详细介绍下：

![Alt](https://i-blog.csdnimg.cn/blog_migrate/f4e3ed786bbeeeeebcee8fdb101aa16a.png#pic_center)
  
区块链数据服务应用场景——数字监管。区块链数据服务提供了链上用户日活、月活、留存率，交易频率等100+的自定义数据指标，以上数据真实、客观反映了区块链的运行状况，从而形成区块链的完整运营分析报表。

![Alt](https://i-blog.csdnimg.cn/blog_migrate/905cf5329f32a2307c686c51906841b9.png#pic_center)
  
这是我们 BDS 产品的系统架构图，系统架构的最左边是我们的数据源，分别包括了全节点，交易所，社交媒体，所有的数据都会同步到消息队列 queue 中；右边整体的一大块就属于 BDS 了，Splitter 会定于 Queue 里面的内容，一部分会交给我们的数据实时处理模块，一部分会直接存储到数据仓库中；实时处理模块会对原始数据做加工处理后再存储到数据集仓库中；这里我们还用到了机器学习，根据地址的行为，进行地址的标记；针对数据仓库的数据，我们提供了BI工具进行可视化，同时也提供了 API 模块，支持 Restful 接口调用，还有一个就是 Report Service，因为我们和做区块链资讯的企业有合作，这个模块就是形成区块链报告的服务。最右边我们还提供了事件服务，针对区块链上类似异常交易的行为可以进行订阅，告警通知；

![Alt](https://i-blog.csdnimg.cn/blog_migrate/6aee2e23b0712467d420d34b9ed4ca56.png#pic_center)
  
目前我们的这个项目已经在 Github 开源：这个是我们的开源项目地址 https://github.com/jdcloud-bds

目前我们的项目最多的赞已经达到1100+个，我们项目已经开源，至于开源目的主要有三块：

1. 标准化区块链的数据模型，制定区块链行业数据指标标准
     
   2.为区块链开源社区做贡献 - 首款提供区块链数据分析的工具
     
   3.以数据为驱动，吸引更多的人参与到区块链开源社区，参与区块链开发工作

我认为区块链进一步的发展，一定要对外创造价值，对社会有益。而区块链与云计算的结合，让用户在不同的场景中可以快速接入区块链技术，给生产生活带来价值，实现真正的普惠

![Alt](https://i-blog.csdnimg.cn/blog_migrate/eaa267bc20597af517fb423ab3918307.png#pic_center)

我还有一个跟咱们的业务结合比较紧密的问题：区块链与云计算结合是区块链落地实体经济的重要方向，也是区块链为实体经济赋能的重要方式；为此您如何看待？2020年区块链+云计算有什么机遇和挑战？

**朱琅：**
区块链的一大特性就是去中心化，运行区块链需要计算机等其他设备，而云能很好的和区块链进行相关结合，因为云售卖的就是计算，网络资源，同时云上也提供了IOT相关的解决方案。云的计算资源又是全球分布的，所以云和区块链的结合，可以认为是相辅相成。然后我觉得核心挑战还是说如何提升公有链的性能，来满足实际业务场景的需求。

**陈思：**
也算是在特性上很契合的结合了。

实际场景对系统的性能要求比较高，目前的公链是无法满足的。

**陈思：**
是的，未来更多应用场景的爆发，还是需要现有技术的突破来做前置条件。

**朱琅：**
这也就是我上面提到的 ETH 2.0 想解决的问题。

**陈思：**
