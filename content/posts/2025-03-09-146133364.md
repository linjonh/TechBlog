# 11. 盛最多水的容器（力扣）

11. 盛最多水的容器

给定一个长度为
`n`
的整数数组
`height`
。有
`n`
条垂线，第
`i`
条线的两个端点是
`(i, 0)`
和
`(i, height[i])`
。

找出其中的两条线，使得它们与
`x`
轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：**
你不能倾斜容器。

![](https://i-blog.csdnimg.cn/direct/ff73038304bc4a2da2c62dd5d52ec40e.png)

**提示：**

* `n == height.length`
* `2 <= n <= 105`
* `0 <= height[i] <= 104`

解答：

```
class Solution {
public:
    int maxArea(vector<int>& height) {
        int n=height.size();
        int m =0;
        int slow=0;
        int fast=n-1;
        while(fast>slow){
            m =max(m ,min(height[slow],height[fast])*(fast-slow));
            if(height[slow]<height[fast]){
                slow++;
            }
            else{
                fast--;
            }
        }
        return m ;
    }
};
```

**思路：**
双指针问题，左右指针分别从两端开始，如果左指针的值小于右指针，那么无论右指针如何向左移动，盛水量都不会增大，宽度变小，而高度不会增大，所以只有移动左指针，可能使盛水量增大。移动左右指针，直到他们碰面。
