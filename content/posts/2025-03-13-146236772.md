```cs

```

自己实现就是这种既视感:

![](https://i-blog.csdnimg.cn/direct/d325e3f077a142b7be33d70f12d73303.png)

```cs
bool RectIntersect(Rect rect1, Rect rect2)
{
    return rect1.xMax > rect2.xMin &&
           rect1.xMin < rect2.xMax &&
           rect1.yMax > rect2.yMin &&
           rect1.yMin < rect2.yMax;
}

```
