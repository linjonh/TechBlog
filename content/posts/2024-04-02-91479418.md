     微信小程序中关于腾讯视频的使用与踩坑
    </h1>
   </div>

 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">

    <div class="article-content">
     <p>
      在开发一个小程序项目时，需要在页面中观看视频，就需要使用到微信小程序的腾讯视频插件，这样就可以在页面中引用腾讯视频，而无需使用自己的服务器进行视频的播放，节约了很大的服务器开支。但是微信小程序插件的文档写的很不清楚，因此将我的解决方法写在这里。
     </p>
     <h4 class="heading">
      一、在小程序中引入插件。
     </h4>
     <p>
      进入微信公众平台，在设置中找到第三方服务，在插件管理中添加插件。搜索到腾讯视频，点击添加，同时打开腾讯视频的详情，在里面找到该插件的appid与版本号。
     </p>
     <p>
      接着是代码部分，在app.json中添加
     </p>
     <pre><code class="hljs bash copyable">  <span class="hljs-string">"plugins"</span>: {
    <span class="hljs-string">"tencentVideo"</span>: {
      <span class="hljs-string">"version"</span>: <span class="hljs-string">"1.1.1"</span>,
      <span class="hljs-string">"provider"</span>: <span class="hljs-string">"wxa75efa648b60994b"</span>
    }
<span class="copy-code-btn">复制代码</span></code></pre>
     <p>
      其中version是版本号，provider是appid。
     </p>
     <h4 class="heading">
      二、在需要使用的页面中引用该插件的组件
     </h4>
     <p>
      <strong>
       官方文档并没有关于该方法的具体说明，所以会导致出现无法找到对应playerid的txv-video的问题
      </strong>
     </p>
     <p>
      打开想要使用插件页面的json文件，在里面添加
     </p>
     <pre><code class="hljs bash copyable"> <span class="hljs-string">"usingComponents"</span>: {
    <span class="hljs-string">"txv-video"</span>: <span class="hljs-string">"plugin://tencentVideo/video"</span>
  }
<span class="copy-code-btn">复制代码</span></code></pre>
     <p>
      这就代表着可以在页面中使用txv-video组件啦
     </p>
     <h4 class="heading">
      三、在页面中使用组件
     </h4>
     <p>
      进入wxml文件，在里面添加
     </p>
     <pre><code class="hljs bash copyable"> &lt;txv-video vid=<span class="hljs-string">"vid"</span> playerid=<span class="hljs-string">"txv1"</span>&gt;&lt;/txv-video&gt; 
<span class="copy-code-btn">复制代码</span></code></pre>
     <p>
      这一步很简单，vid是腾讯视频的vid，playerid是该txv-video唯一识别码，这个组件不能通过css进行样式修改
     </p>
     <h4 class="heading">
      四、在js中进行引入
     </h4>
     <p>
      如果你不想在js中对该插件进行控制的话，可以不引入。
     </p>
     <pre><code class="hljs bash copyable">    const TxvContext = requirePlugin(<span class="hljs-string">"tencentVideo"</span>);
    <span class="hljs-built_in">let</span> txvContext = TxvContext.getTxvContext(<span class="hljs-string">'txv1'</span>);
    txvContext.play(); //播放
<span class="copy-code-btn">复制代码</span></code></pre>
     <hr/>
     <p>
      这就是我成功引用腾讯视频插件的全部过程了，希望能够帮助到一样使用这个插件遇到问题的人。
     </p>
