     <a id="1_Ubuntu__18">

     1. Ubuntu 简介
    </h4>
    <h5>
     <a id="11__Ubuntu_19">
     </a>
     1.1 什么是 Ubuntu？
    </h5>
    <p>
     Ubuntu 是基于 Debian 的开源 Linux 发行版，由 Canonical 公司维护，提供桌面版和服务器版，以
     <strong>
      易用性
     </strong>
     和
     <strong>
      社区支持
     </strong>
     著称。
    </p>
    <h5>
     <a id="12__22">
     </a>
     1.2 版本说明
    </h5>
    <ul>
     <li>
      <strong>
       LTS（Long-Term Support）
      </strong>
      ：长期支持版（如 22.04），提供 5 年安全更新。
     </li>
     <li>
      <strong>
       普通版
      </strong>
      ：每 6 个月发布一次，支持周期 9 个月。
     </li>
    </ul>
    <hr/>
    <h4>
     <a id="2__28">
     </a>
     2. 系统安装指南
    </h4>
    <h5>
     <a id="21__29">
     </a>
     2.1 安装前准备
    </h5>
    <ul>
     <li>
      <p>
       <strong>
        硬件要求
       </strong>
       ：
      </p>
      <ul>
       <li>
        64 位 CPU（x86_64 或 ARM）
       </li>
       <li>
        4 GB 内存 + 25 GB 磁盘空间（推荐 SSD）
       </li>
       <li>
        USB 启动盘（≥4GB）
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        制作启动盘
       </strong>
       ：
      </p>
      <pre><code class="prism language-bash"><span class="token comment"># 使用 Ventoy（跨平台工具）  </span>
<span class="token number">1</span>. 下载 Ventoy：https://www.ventoy.net  
<span class="token number">2</span>. 插入 U 盘 → 运行 Ventoy → 点击“Install”  
<span class="token number">3</span>. 将 Ubuntu ISO 文件复制到 U 盘  
</code></pre>
     </li>
    </ul>
    <h5>
     <a id="22__43">
     </a>
     2.2 安装步骤
    </h5>
    <ol>
     <li>
      <strong>
       从 U 盘启动
      </strong>
      ：开机按
      <code>
       F12
      </code>
      /
      <code>
       Esc
      </code>
      选择启动设备。
     </li>
     <li>
      <strong>
       选择语言与键盘布局
      </strong>
      。
     </li>
     <li>
      <strong>
       分区方案
      </strong>
      （推荐新手选择“自动安装”）。
     </li>
     <li>
      <strong>
       设置用户信息
      </strong>
      ：
      <ul>
       <li>
        用户名、密码（建议勾选“自动登录”）。
       </li>
      </ul>
     </li>
     <li>
      <strong>
       等待安装完成
      </strong>
      → 重启。
     </li>
    </ol>
