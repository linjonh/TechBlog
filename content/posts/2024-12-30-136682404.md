    <h3 id="h_editor_0">
     1. for 循环
    </h3>
    <p>
     <code>
      for
     </code>
     循环最常用于遍历（迭代）序列中的元素，遍历的对象可以是列表、元组、字符串或其他可迭代对象。语法如下：
    </p>
    <pre><code class="language-python">for &lt;variable&gt; in &lt;sequence&gt;:
    &lt;code block&gt;
</code></pre>
    <p>
     其中：
    </p>
    <ul>
     <li>
      <code>
       &lt;variable&gt;
      </code>
      ：循环变量，用于存储序列中的每个元素
     </li>
     <li>
      <code>
       &lt;sequence&gt;
      </code>
      ：要遍历的序列
     </li>
     <li>
      <code>
       &lt;code block&gt;
      </code>
      ：循环体，将在每个元素上执行
     </li>
    </ul>
    <p>
     <strong>
      下图展示了for循环的执行步骤：
     </strong>
    </p>
    <blockquote>
     （1）首先，初始化循环控制变量。
     <br/>
     （2）接着，对循环进行判断。如果判断为真，则进入循环结构；如果判断为假，则退出循环。
     <br/>
     （3）执行循环结构内容，并更新循环控制变量。
     <br/>
     （4）重复步骤（2）和（3），直到循环条件不再满足为止。
    </blockquote>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="449" src="https://i-blog.csdnimg.cn/blog_migrate/33215604ba07750eeca2f38dcd7f9a1c.png" width="750"/>
    </p>
    <p>
     示例代码：
    </p>
    <pre><code class="language-python"># 遍历列表
my_list = [1, 2, 3, 4, 5]

for i in my_list:
    print(i) 
</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-python">1
2
3
4
5
</code></pre>
    <p>

    <h3 id="h_editor_1">
     2. while循环
    </h3>
    <p>
     <code>
      while
     </code>
     循环在条件为真时重复执行代码块，直到条件变为假为止。语法如下：
    </p>
    <pre><code class="language-python">while &lt;condition&gt;:
    &lt;code block&gt;
</code></pre>
    <p>
     其中：
    </p>
    <ul>
     <li>
      <code>
       &lt;condition&gt;
      </code>
      ：循环条件，只要为真，循环就会继续执行
     </li>
     <li>
      <code>
       &lt;code block&gt;
      </code>
      ：循环体，将在条件为真的情况下执行
     </li>
    </ul>
    <p>
     <strong>
      下图展示了while循环的执行步骤：
     </strong>
    </p>
    <blockquote>
     （1）初始化循环控制变量。
     <br/>
     （2）判断循环条件，若为真，则执行循环体；否则，跳过循环体。
     <br/>
     （3）执行循环体内容。
     <br/>
     （4）重复步骤2和3，直到循环条件不再为真。
    </blockquote>
    <p>
    </p>
    <p class="img-center">
     <img alt="" height="474" src="https://i-blog.csdnimg.cn/blog_migrate/87aaa0406a391ad03b39be1358c52ee5.png" width="750"/>
    </p>
    <p>
     <strong>
      示例
     </strong>
    </p>
    <pre><code class="language-python"># 打印1到5
i = 1
while i &lt;= 5:
    print(i)
    i += 1
</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-python">1
2
3
4
5
</code></pre>

    <h3 id="h_editor_2">
     <br/>
     <strong>
      3. 简单语句组
     </strong>
    </h3>
    <p>
     在
     <code>
      for
     </code>
     、
     <code>
      while
     </code>
     循环中，简单语句组可以用缩进来代替循环语句，如果只有一条语句需要执行，可以将该语句放在循环头部，形成简单语句组。例如：
    </p>
    <pre><code class="language-python"># 等价于 for i in range(10): print(i)
for i in range(10):
    print(i)

# 等价于 while num &lt; 10: num += 1
num = 0
while num &lt; 10:
    num += 1
    print(num)
</code></pre>
    <h3 id="h_editor_3">
     <strong>
      4. for...else语句
     </strong>
    </h3>
    <p>
     <code>
      for...else
     </code>
     语句在
     <code>
      for
     </code>
     循环完成后执行
     <code>
      else
     </code>
     代码块，除非在循环中遇到
     <code>
      break
     </code>
     语句。
    </p>
    <p>
     语法格式如下：
    </p>
    <pre><code class="language-python">for item in iterable:
    # 循环主体
else:
    # 循环结束后执行的代码
</code></pre>
    <p>
     示例代码：
    </p>
    <pre><code class="language-python"># 查找列表中是否存在指定元素
fruits = ["apple", "banana", "orange"]
find_flag = False
for fruit in fruits:
    if fruit == "mango":
        find_flag = True
        break
else:
    print("未找到指定元素")

if find_flag:
    print("找到了指定元素")
</code></pre>
    <p>
    </p>
    <h3 id="h_editor_4">
     <strong>
      5. while ...else 语句
     </strong>
    </h3>
    <p>
     <code>
      while...else
     </code>
     语句在
     <code>
      while
     </code>
     循环条件为假时执行
     <code>
      else
     </code>
     代码块，除非在循环中遇到
     <code>
      break
     </code>
     语句。示例：
    </p>
    <p>
     <strong>
      语法格式如下：
     </strong>
    </p>
    <pre><code class="language-python">while &lt;condition&gt;:
    &lt;code block&gt;   # 循环主体
else:
    # 循环结束后执行的代码
</code></pre>
    <p>
     <strong>
      示例代码：
     </strong>
    </p>
    <pre><code class="language-python">count = 0
while count &lt; 5:
    print(count, " 小于 5")
    count += 1
else:
    print(count, " 不再小于 5")
</code></pre>
    <p>
     <strong>
      输出：
     </strong>
    </p>
    <pre><code class="language-python">0  小于 5
1  小于 5
2  小于 5
3  小于 5
4  小于 5
5  不再小于 5
</code></pre>
    <p>
     <code>
      while
     </code>
     循环会一直执行直到
     <code>
      count
     </code>
     的值达到 5。当
     <code>
      count
     </code>
     变成 5 时，循环条件变为假值，
     <code>
      else
     </code>
     语句块被执行。
    </p>
    <h3 id="h_editor_5">
     6. range() 函数
    </h3>
    <p>
     <code>
      range()
     </code>
     函数用于生成一个指定范围的数字序列，常用于
     <code>
      for
     </code>
     循环中。语法如下：
    </p>
    <pre><code class="language-python">range(start, stop, step) 
</code></pre>
    <ul>
     <li>
      start：序列的起始数字，默认为0
     </li>
     <li>
      stop：序列的结束数字，不包含在序列中
     </li>
     <li>
      step：序列的步长，默认为1
     </li>
    </ul>
    <p>
     示例代码：
    </p>
    <pre><code class="language-python"># 输出1-10之间的所有奇数
for i in range(1, 10, 2):
    print(i) 
</code></pre>
    <p>
     <strong>
      输出：
     </strong>
    </p>
    <pre><code class="language-python">1
3
5
7
9
</code></pre>
    <p>
    </p>
    <h3 id="h_editor_6">
     <strong>
      7. break &amp; continue语句
     </strong>
    </h3>
    <ul>
     <li>
      break语句用于提前终止循环，跳出循环体。
     </li>
     <li>
      continue语句用于跳过当前循环体，直接开始下一次循环。
     </li>
    </ul>
    <p>
     <strong>
      示例代码：
     </strong>
    </p>
    <pre><code class="language-python"># 循环输出1到10的数字，遇到5时跳出循环
for i in range(1, 11):
    print(i)
    if i == 5:
        break

# 循环输出1到10的数字，只输出偶数
for i in range(1, 11):
    if i % 2 == 1:
        continue
    print(i)
</code></pre>
    <p>
     <strong>
      输出结果：
     </strong>
    </p>
    <pre><code class="language-python">循环输出1到10的数字，遇到5时跳出循环:
1
2
3
4
5
循环输出1到10的数字，只输出偶数:
2
4
6
8
10
</code></pre>
    <p>
    </p>
    <h3 id="h_editor_7">
     <strong>
      8. 循环中的 else 子句
     </strong>
    </h3>
    <p>
     <code>
      else
     </code>
     子句还可以在循环中使用，不仅限于
     <code>
      while
     </code>
     循环，也适用于
     <code>
      for
     </code>
     循环。当循环正常执行完毕（即没有遇到
     <code>
      break
     </code>
     语句）时，
     <code>
      else
     </code>
     子句中的代码将被执行。
    </p>
    <p>
     <strong>
      在
      <code>
       while
      </code>
      循环中使用
      <code>
       else
      </code>
      ：
     </strong>
    </p>
    <pre><code class="language-python">count = 0
while count &lt; 5:
    print(count, "小于 5")
    count += 1
else:
    print(count, "不再小于 5")
</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-python">0 小于 5
1 小于 5
2 小于 5
3 小于 5
4 小于 5
5 不再小于 5
</code></pre>
    <p>
     <strong>
      在
      <code>
       for
      </code>
      循环中使用
      <code>
       else
      </code>
      :
     </strong>
    </p>
    <pre><code class="language-python">for i in range(5):
    print(i)
else:
    print("循环正常结束，没有遇到 break") 
</code></pre>
    <p>
     输出将会是：
    </p>
    <pre><code class="language-python">0
1
2
3
4
循环正常结束，没有遇到 break
</code></pre>
    <p>
     <strong>
      没有执行
      <code>
       else
      </code>
      子句的情况:
     </strong>
    </p>
    <p>
     如果在循环中使用了
     <code>
      break
     </code>
     语句，那么
     <code>
      else
     </code>
     子句中的代码就不会执行：
    </p>
    <pre><code class="language-python">for i in range(5):
    print(i)
    if i == 2:
        break
else:
    print("循环正常结束，没有遇到 break")
</code></pre>
    <p>
     输出：
    </p>
    <pre><code class="language-python">0
1
2 
</code></pre>

    <h3 id="h_editor_8">
     <strong>
      9. pass语句
     </strong>
    </h3>
    <p>
     <code>
      pass
     </code>
     是一个占位符语句，它不做任何事情。它的作用是在语法上需要语句的地方提供一个空语句，什么也不做。
     <code>
      pass
     </code>
     语句通常在代码块中需要语句，但是你暂时不想写任何代码的时候使用。
    </p>
    <pre><code class="language-python"># 这是一个无限循环，需要使用 Ctrl+C 才能终止
while True:
    pass

# 这是一个简单的 for 循环，但循环体暂时没有内容
for i in range(10):
    pass
</code></pre>

if flag:

while True:

    if guess == number:

        break
    elif guess &gt; number:
        print("你猜的数字大了。")
    else:
        print("你猜的数字小了。")

else:
    print("游戏结束！")
</code></pre>
