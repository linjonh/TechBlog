    <h1 class="title-article" id="articleContentId">
     如何安装旧版本的Pytorch
    </h1>
   </div>
  </div>
 </div>
 <article class="baidu_pl">
  <div class="article_content clearfix" id="article_content">
   <link href="../../assets/css/kdoc_html_views-1a98987dfd.css" rel="stylesheet"/>

    <p>
     不同的项目所使用的Pytorch版本可能不同，一般而言，高版本的Pytorch可以向下兼容的，但有时可能会需要旧版本的Pytorch。
    </p>
    <p>
     1、首先进入Pytorch官网（
     <a href="https://pytorch.org/" rel="nofollow" title="          PyTorch      ">
      PyTorch
     </a>
     ），下滑找到”
     <a href="https://pytorch.org/get-started/previous-versions" rel="nofollow" title="install previous versions of PyTorch">
      install previous versions of PyTorch
     </a>
     “，或点击”Get started“后找到”
     <a href="https://pytorch.org/get-started/previous-versions" rel="nofollow" title="install previous versions of PyTorch">
      install previous versions of PyTorch
     </a>
     “。
    </p>
    <p>
     2、点击”
     <a href="https://pytorch.org/get-started/previous-versions" rel="nofollow" title="install previous versions of PyTorch">
      install previous versions of PyTorch
     </a>
     ”，就能发现各种版本以及他们各自的安装方式。
    </p>
    <p>
     3、找到自己需要的版本，选择使用conda命令安装还是pip命令安装，接着根据自己的操作系统选择，然后根据自己是否有独立GPU还是只有CPU，以及硬件CUDA版本，复制对应的安装命令。
    </p>
    <p>
     4、到Anaconda Prompt中（适用于conda和pip命令），跳转到想要安装pytorch的虚拟环境下，使用3中复制的命令执行。
    </p>
    <p class="img-center">
     <img alt="" height="886" src="https://i-blog.csdnimg.cn/direct/dd041b3283fc4daca6019c614ea4f76b.png" width="1409"/>
    </p>
    <p class="img-center">
     <img alt="" height="854" src="https://i-blog.csdnimg.cn/direct/946288f924fc41f0a2fe72bde9cca917.png" width="1403"/>
    </p>
    <p>
    </p>
    <ul>
     <li>
      conda指令基本格式（Windows+GPU）：
     </li>
    </ul>
    <pre><code class="language-python"># CUDA 11.8
conda install pytorch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1  pytorch-cuda=11.8 -c pytorch -c nvidia</code></pre>
    <p>
     等号后面的均为版本号。
    </p>
    <p>
     如果官网通道没有该版本，或者下载速度太慢，可以将通道换为其他通道：
    </p>
    <pre><code class="language-python">conda install pytorch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1  pytorch-cuda=11.8 -c 通道地址</code></pre>

    <p>
     可以使用以下命令：
    </p>
    <pre><code class="language-python">conda search pytorch –c 镜像通道</code></pre>
    <p>
     来查找该通道中有哪些版本的Pytorch。
    </p>
    <p>
     使用以下命令：
    </p>
    <pre><code class="language-python">conda search pytorch=版本 –c 镜像通道</code></pre>
    <p>
     来验证镜像通道是否有这个版本的Pytorch。
    </p>
    <p>
    </p>
    <ul>
     <li>
      pip指令基本格式（windows+GPU）
     </li>
    </ul>
    <pre><code class="language-python"># CUDA 11.8
pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1 --index-url https://download.pytorch.org/whl/cu118</code></pre>
