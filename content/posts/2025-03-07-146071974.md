    <h3>
     一、Objective-C
    </h3>
    <h4>
     （1）标准的c风格
    </h4>
    <pre><code class="language-objectivec">for (int i = 0; i &lt; 5; i++) {
    NSLog(@"i = %d", i);
}</code></pre>
    <h4>
     （2）for in循环。
    </h4>
    <pre><code class="language-objectivec">NSArray *array = @[@"apple", @"banana", @"orange"];
for (NSString *fruit in array) {
    NSLog(@"%@", fruit);
}

//这个遍历输出的是值，而不是健</code></pre>
    <h4>
     （3）
     <strong>
      基于块的枚举（Block-Based Enumeration）
     </strong>
    </h4>
    <p>
     Objective-C 提供了基于块的枚举方法，例如
     <code>
      enumerateObjectsUsingBlock:
     </code>
     ，可以遍历集合类并执行块中的代码。
    </p>
    <pre><code class="language-objectivec">NSArray *array = @[@"apple", @"banana", @"orange"];
[array enumerateObjectsUsingBlock:^(NSString *fruit, NSUInteger idx, BOOL *stop) {
    NSLog(@"%@ at index %lu", fruit, (unsigned long)idx);
}];

apple at index 0
banana at index 1
orange at index 2</code></pre>
    <p>
     这个类似于python中的for index,item in enumerate(strkk):，可以得到索引值以及内容（值），而且可以通过设置stop值==yes来终止循环
    </p>
    <h4>
     （4）while循环
    </h4>
    <pre><code class="language-objectivec">int i = 0;
while(i&lt;5){
    NSlog(@"%d",i)
    i++;
}
//输出：0 1 2 3 4 </code></pre>
    <h4>
     （5）do...while循环
    </h4>
    <pre><code class="language-objectivec">int i= 0 
do{

   NSLog(@"%d",i)
   // 0 1 2 3 4 
   i++

}while(i&lt;5)</code></pre>

    <h3>
     二、Python
    </h3>
    <h4>
     （1）range()函数
    </h4>
    <p>
     for   in range(起始, 结束, 步长)
    </p>
    <pre><code class="language-python">string = "0123456789"
for i in range(0,len(string),2):
    print(f'输出的数据i==={i}')

#输出的数据i===0
#输出的数据i===2
#输出的数据i===4
#输出的数据i===6
#输出的数据i===8</code></pre>
    <p>
     注意range后面的参数，启始、结束、步长，写一个参数代表结束位置，不包括结束位置
    </p>
    <p>
     对比 Objective-C：
    </p>
    <ul>
     <li>
      <p>
       类似于 Objective-C 的标准
       <code>
        for
       </code>
       循环（
       <code>
        for (int i = 0; i &lt; 5; i++)
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       Python 的
       <code>
        range()
       </code>
       更简洁，不需要手动管理循环变量。
      </p>
     </li>
    </ul>
    <h4>
     （2）for in循环
    </h4>
    <pre><code class="language-python">fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)</code></pre>
    <p>
     对比 Objective-C：
    </p>
    <ul>
     <li>
      <p>
       Objective-C 的快速枚举（
       <code>
        for...in
       </code>
       ）与 Python 的
       <code>
        for
       </code>
       循环非常相似。
      </p>
     </li>
     <li>
      <p>
       Python 的
       <code>
        for
       </code>
       循环更简洁，不需要指定类型。
      </p>
     </li>
    </ul>
    <h4>
     （3）
     <strong>
      <code>
       enumerate()
      </code>
      函数，也是for in的一种，类似
     </strong>
    </h4>
    <p>
     <code>
      enumerate()
     </code>

Index: 1, Fruit: banana
Index: 2, Fruit: orange</code></pre>
    <p>

    </p>

     <li>

       <code>
        enumerateObjectsUsingBlock:
       </code>
       ）。
      </p>
     </li>
     <li>
      <p>
       Python 的
       <code>
        enumerate()
       </code>
       更简洁，不需要额外的语法。
      </p>
     </li>
    </ul>
    <h4>
     （4）while循环
    </h4>
    <pre><code class="language-python">i = 0
while i &lt; 5:
    print(f"i = {i}")
    i += 1</code></pre>
    <p>
     对比 Objective-C：
    </p>
    <ul>
     <li>
      <p>
       与 Objective-C 的
       <code>
        while
       </code>

     <li>
      <p>
       Python 不需要分号或大括号，使用缩进来定义代码块。
      </p>

    </ul>
    <p>
     Python 的设计哲学强调简洁性和可读性。
     <code>
      do...while
     </code>

     <code>
      while True
     </code>

     <code>
      break

     <code>
      do...while
     </code>
     语法。

    <h4>
     （5）
     <strong>
      列表推导式（就是for in循环中写表达式）
     </strong>
    </h4>
    <p>
     列表推导式是一种简洁的创建列表的方式，可以替代简单的
     <code>
      for
     </code>
     循环。
    </p>
    <p>
     [表达式 for 变量 in 可迭代对象]

    <pre><code class="language-python">squares = [x ** 2 for x in range(5)]
print(squares)

#[0, 1, 4, 9, 16]</code></pre>
    <ul>
     <li>
      <p>
       先看for in range函数，输出的x在加上前面的表达式，然后最终生成值输出
      </p>
     </li>
     <li>
      <p>
       Python 的列表推导式非常简洁，适合快速生成列表。
      </p>
     </li>
    </ul>
    <h4>
     （6）
     <strong>
      <code>
       zip()
      </code>
      函数
     </strong>
    </h4>
    <p>
     <code>
      zip()
     </code>
     函数用于同时遍历多个可迭代对象。
    </p>
    <p>
     for 变量1, 变量2 in zip(可迭代对象1, 可迭代对象2):
     <br/>
     # 循环体
    </p>
    <pre><code class="language-python">fruits = ["apple", "banana", "orange"]
prices = [1.0, 0.5, 0.8]
for fruit, price in zip(fruits, prices):
    print(f"Fruit: {fruit}, Price: {price}")

#Fruit: apple, Price: 1.0
#Fruit: banana, Price: 0.5
#Fruit: orange, Price: 0.8</code></pre>
    <p>
     这个其实没有特殊的，就是通过for in 同时遍历多个对象

     <li>
      <p>
       半开区间运算符（n…&lt;m），相当于数学的 [n, m)
      </p>
     </li>
    </ul>
    <h4>
     （1）for in 区间
    </h4>
    <pre><code class="language-Swift">//开区间，输出12到30包括12和30
 for index in 12...30{
    print("index==\(index)")
 }
//半开区间，输出12到29，不包括30
 for index in 12..&lt;30{
    print("index==\(index)")
 }</code></pre>
    <ul>
    </ul>

            print("输出index==\(index)")

         输出index==0
         输出index==3
         输出index==6
         输出index==9
         输出index==12

    <h3>
     五、Js
    </h3>
    <h4>
     （1）标准的c风格
    </h4>
    <pre><code class="language-Swift">for (let i = 0; i &lt; 5; i++) {
    console.log(`i = ${i}`);
}</code></pre>
    <h4>
     （2）
     <strong>
      <code>
       for...in
      </code>
      循环
     </strong>
    </h4>
    <p>
     <code>
      for...in
     </code>
     循环会遍历
     <strong>
      <span style="color:#fe2c24">
       对象的所有可枚举属性
      </span>
     </strong>
     （包括原型链上的属性），对于字符串，
     <code>
      for...in
     </code>
     会遍历字符串的索引（键），而不是直接遍历字符。
    </p>
    <p>
    </p>
    <h4>
     （3）
     <strong>
      <code>
       for...of
      </code>
      循环
     </strong>
    </h4>
    <pre><code class="language-javascript">        //这里遍历的是索引
        for (const index in string) {
            console.log(`输出的内容${index}`)
        }
        //for of遍历的是内容值
        for (const element of string) {
            console.log(`输出的内容${element}`)
        }       </code></pre>
    <p>
     js中for in循环出来的是索引，想要得到内容值需要for of，但是这里要注意，虽然
     <code>
      for...in
     </code>
     可以用于数组或字符串，但它会遍历所有可枚举属性，包括原型链上的属性，可能会导致意外行为。对于数组或字符串，更推荐使用
     <code>
      for
     </code>
     、
     <code>
      for...of
     </code>
     或
     <code>
      forEach
     </code>
     等方法，比如：
    </p>
    <pre><code class="language-javascript">// 给字符串的原型添加一个属性
String.prototype.customProp = "test";

const str = "hello";
for (const index in str) {
  console.log(`Index: ${index}, Character: ${str[index]}`);
}
