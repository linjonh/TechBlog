    <h3>
     <a id="QuickStart_4">
     </a>
     QuickStart
    </h3>
    <p>
     详情可见博客：https://www.cnblogs.com/nhdlb/p/18703804，使用docker拉取最近的一个版本进行创建
    </p>
    <pre><code class="prism language-shell"><span class="token function">docker</span> run <span class="token parameter variable">-it</span> <span class="token parameter variable">-d</span> <span class="token parameter variable">-p</span> <span class="token number">7474</span>:7474 <span class="token parameter variable">-p</span> <span class="token number">7687</span>:7687 <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> /disk5/neo4j_docker/data:/data <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> /disk5/neo4j_docker/logs:/logs <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> /disk5/neo4j_docker/conf:/var/lib/neo4j/conf <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> /disk5/neo4j_docker/import:/var/lib/neo4j/import <span class="token punctuation">\</span>
<span class="token parameter variable">-v</span> /disk5/neo4j_docker/plugins:/var/lib/neo4j/plugins <span class="token punctuation">\</span>
<span class="token parameter variable">-e</span> <span class="token assign-left variable">NEO4J_AUTH</span><span class="token operator">=</span>neo4j/password <span class="token punctuation">\</span>
<span class="token parameter variable">--name</span> neo4j neo4j:5.26.2
</code></pre>
    <p>
     docker参数含义解释：
    </p>
    <pre><code class="prism language-shell"><span class="token parameter variable">-p</span> <span class="token number">7474</span>:7474  宿主机端口映射，7474端口为Neo4j界面访问端口
<span class="token parameter variable">-p</span> <span class="token number">7687</span>:7687  宿主机端口映射，7687端口为后台API接口访问端口
<span class="token parameter variable">-v</span> /home/neo4j/data:/data  宿主机目录挂载，将Neo4j存储数据挂载到宿主机目录
<span class="token parameter variable">-v</span> /home/neo4j/logs:/logs  宿主机目录挂载，将Neo4j运行日志挂载到宿主机目录
<span class="token parameter variable">-v</span> /home/neo4j/conf:/var/lib/neo4j/conf  宿主机目录挂载，将Neo4j配置文件挂载到宿主机目录
<span class="token parameter variable">-v</span> /home/neo4j/import:/var/lib/neo4j/import  宿主机目录挂载，将Neo4j数据导入目录挂载到宿主机目录
<span class="token parameter variable">-v</span> /home/neo4j/plugins:/var/lib/neo4j/plugins  宿主机目录挂载，将Neo4j插件安装目录挂载到宿主机目录
<span class="token parameter variable">-e</span> <span class="token assign-left variable">NEO4J_AUTH</span><span class="token operator">=</span>neo4j/password  配置Neo4j的登录密码，用户名必须是neo4j，密码可以自定义
</code></pre>

    <h3>
     <a id="Demo_34">
     </a>
     Demo
    </h3>
    <p>
     详情可见b站：https://www.bilibili.com/video/BV13K4y187b4
    </p>
    <p>
     <img alt="image-20250312203448603" src="https://i-blog.csdnimg.cn/img_convert/c78bc4b2b71350d863ed56bc8398589e.png"/>
    </p>
    <p>
     信息补充：
    </p>
    <pre><code class="prism language-shell">华工科技 湖北 深交所
安琪酵母 湖北 上交所
国信证券 广东 深交所
中科创达 北京 深交所
招商银行 深圳 上交所
</code></pre>
    <ul>
     <li>
      <p>
       创建单个节点
      </p>
      <pre><code class="prism language-CQL">create(variable:lable{key1:value1,key2:value2}) return variable

create(n:Stock {name:'安琪酵母',code:'600298',lauchDate:date('2000-08-18')}) return n
</code></pre>
     </li>
     <li>
      <p>
       创建多个节点：股票、交易所、省市标签
      </p>
      <pre><code class="prism language-CQL">create(variable:lable{key1:value1,key2:value2}),(:lable{key1.1:value1.1,key2.1:value2.1}) return variable

// 创建股票节点
create(n:Stock {name:'安琪酵母',code:'600298',lauchDate:date('2000-08-18')}), (n:Stock {name:'安招商银行',code:'600036',lauchDate:date('2002-08-09')})return n
// 创建交易所节点
create(n:SecuritiesExchange{name:'上海证券交易所'}),(:SecuritiesExchange{name:'深圳证券交易所'})return n;
// 创建省市节点
create(n:Province{name:'湖北'}),(:Province{name:'北京'}),(:Province{name:'广东'})return n;
</code></pre>
     </li>
     <li>
      <p>
       建立关系：股票和省市、股票、交易所关系
      </p>
      <pre><code class="prism language-cql">match(a:lable1),(b:lable2)
where a.name ='value1' and b.name='value3'
create (a)-[r:relationship]-&gt;(b) return r

// 创建股票和交易所关系
match(a:Stock),(b:SecuritiesExchange)
where a.name = '安琪酵母' and b.name='上海证券交易所'
create (a)-[r:Exchange]-&gt;(b) return r
// 创建股票和省市名称关系
MATCH(a: Stock) ,(b: Province)
WHERE a.name='安琪酵母' AND b.name='湖北'
CREATE (a)-【r:Area】-&gt;(b) RETURN r
</code></pre>
     </li>
     <li>
      <p>
       查询节点和关系
      </p>
      <pre><code class="prism language-cql">match(a:variable)
where a.name = 'key1'
return a

// 查询某个股票的节点
match(a:Stock)
where a.name = '安琪酵母'
return a
</code></pre>
     </li>
     <li>
      <p>
       查询某个标签下所有节点
      </p>
      <pre><code class="prism language-cql">// 查询股票标签下所有节点
match(a:Stock)
return a
</code></pre>
     </li>
     <li>
      <p>
       查询两个节点之间的关系
      </p>
      <pre><code class="prism language-cql">// 查询两个具体节点的关系
match(:Stock{name:'安琪酵母'})-[r]-&gt;(:Province{name:'湖北'}) return r
</code></pre>
     </li>
     <li>
      <p>
       查询关系的类型
      </p>
      <pre><code class="prism language-cql">// type获取某个节点所有关系的类型
match(:Stock{name:'安琪酵母'})-[r]-&gt;()
return type(r)
</code></pre>
     </li>
     <li>
      <p>
       删除两个节点的关系
      </p>
      <pre><code class="prism language-cql">// 删除两个节点的关系
match (:Stock{name:'安琪酵母'})-[r]-&gt;(:Province{name:'湖北'}) delete r
</code></pre>
     </li>
     <li>
      <p>
       删除某个标签下所有的关系
      </p>
      <pre><code class="prism language-cql">// 删除股票标签下所有的关系
match(:Stock)-[r]-&gt;() delete r
</code></pre>
     </li>
     <li>
      <p>
       删除节点
      </p>
      <pre><code class="prism language-cql">match（a:Stock）
where a.name = '中科创达'
delete a

// 删除某个标签下的节点
match(n:Stock)
delete n

// 删除所有节点
match(n)
delete n
</code></pre>
     </li>
     <li>
      <p>
       增加节点属性
      </p>
      <pre><code class="prism language-CQL">match(a:Stock)
where a.name= '安琪酵母'
set a.abbr = 'AQJM' return a
</code></pre>
     </li>
     <li>
      <p>
       删除节点属性
      </p>
      <pre><code class="prism language-cql">match(a:Stock)
where a.name='中科创达'
remove a.abbr
</code></pre>
     </li>
    </ul>
