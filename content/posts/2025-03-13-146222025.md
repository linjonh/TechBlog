    <h3 id="%E4%B8%80%E3%80%81Tomcat%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F" name="%E4%B8%80%E3%80%81Tomcat%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F">
     一、Tomcat是什么？
    </h3>
    <p>
     Apache Tomcat是一个开源的
     <strong>
      Servlet容器
     </strong>
     ，实现了Java Servlet、JSP等规范。根据2023年JetBrains开发者调查报告，超过60%的Java开发者使用Tomcat作为Web服务器。
    </p>
    <p>
     <strong>
      核心功能
     </strong>
     ：
    </p>
    <ul>
     <li>
      <p>
       处理HTTP请求
      </p>
     </li>
     <li>
      <p>
       编译执行JSP页面
      </p>
     </li>
     <li>
      <p>
       管理Servlet生命周期
      </p>
     </li>
     <li>
      <p>
       提供Web应用运行环境
      </p>
     </li>
    </ul>
    <hr/>
    <h3 id="%E4%BA%8C%E3%80%81%E5%BF%AB%E9%80%9F%E5%AE%89%E8%A3%85%E6%8C%87%E5%8D%97" name="%E4%BA%8C%E3%80%81%E5%BF%AB%E9%80%9F%E5%AE%89%E8%A3%85%E6%8C%87%E5%8D%97">
     二、快速安装指南
    </h3>
    <h4 id="1.%20%E7%8E%AF%E5%A2%83%E5%87%86%E5%A4%87" name="1.%20%E7%8E%AF%E5%A2%83%E5%87%86%E5%A4%87">
     1. 环境准备
    </h4>
    <blockquote>
     <ul>
      <li>
       <p>
        JDK 1.8+（建议JDK11）
       </p>
      </li>
      <li>
       <p>
        操作系统：Windows/Linux/macOS
       </p>
      </li>
      <li>
       <p>
        内存：至少2GB可用
       </p>
      </li>
     </ul>
    </blockquote>
    <h4 id="2.%20%E4%B8%8B%E8%BD%BD%E5%AE%89%E8%A3%85" name="2.%20%E4%B8%8B%E8%BD%BD%E5%AE%89%E8%A3%85" style="background-color:transparent">
     2. 下载安装
    </h4>
    <blockquote>
     <ol>
      <li>
       <p>
        访问
        <a href="https://tomcat.apache.org/download-10.cgi" rel="nofollow" title="官网下载页面">
         官网下载页面
        </a>
       </p>
      </li>
      <li>
       <p>
        选择
        <strong>
         Core
        </strong>
        分类下的zip/tar.gz包（推荐10.0.x版本）
       </p>
      </li>
      <li>
       <p>
        解压到指定目录（无空格路径）
       </p>
      </li>
     </ol>
    </blockquote>
    <pre><code class="language-bash"># Linux示例
wget https://dlcdn.apache.org/tomcat/tomcat-10/v10.0.27/bin/apache-tomcat-10.0.27.tar.gz
tar -zxvf apache-tomcat-10.0.27.tar.gz</code></pre>
    <h4 id="3.%20%E9%AA%8C%E8%AF%81%E5%AE%89%E8%A3%85" name="3.%20%E9%AA%8C%E8%AF%81%E5%AE%89%E8%A3%85">
     3. 验证安装
    </h4>
    <pre><code class="language-bash"># 进入bin目录
cd apache-tomcat-10.0.27/bin

# 启动服务器（Windows执行startup.bat）
./startup.sh

# 访问 http://localhost:8080</code></pre>
    <p>
     看到
     <strong>
      Apache Tomcat
     </strong>
     欢迎页面即表示安装成功！
    </p>
    <p>
    </p>
    <hr/>
    <h3 id="%E4%B8%89%E3%80%81%E7%9B%AE%E5%BD%95%E7%BB%93%E6%9E%84%E8%A7%A3%E6%9E%90" name="%E4%B8%89%E3%80%81%E7%9B%AE%E5%BD%95%E7%BB%93%E6%9E%84%E8%A7%A3%E6%9E%90">
     三、目录结构解析
    </h3>
    <pre><code class="language-bash">├── bin     # 启动/关闭脚本
├── conf    # 配置文件（server.xml等）
├── lib     # 依赖库
├── logs    # 日志文件
├── temp    # 临时文件
├── webapps # 应用部署目录
└── work    # JSP编译文件</code></pre>
    <hr/>
    <h3 id="%E5%9B%9B%E3%80%81%E7%AC%AC%E4%B8%80%E4%B8%AAWeb%E5%BA%94%E7%94%A8%E9%83%A8%E7%BD%B2" name="%E5%9B%9B%E3%80%81%E7%AC%AC%E4%B8%80%E4%B8%AAWeb%E5%BA%94%E7%94%A8%E9%83%A8%E7%BD%B2">
     四、第一个Web应用部署
    </h3>
    <h4 id="1.%20%E5%88%9B%E5%BB%BA%E6%A0%87%E5%87%86%E7%9B%AE%E5%BD%95%E7%BB%93%E6%9E%84" name="1.%20%E5%88%9B%E5%BB%BA%E6%A0%87%E5%87%86%E7%9B%AE%E5%BD%95%E7%BB%93%E6%9E%84">
     1. 创建标准目录结构
    </h4>
    <pre><code class="language-bash">MyWebApp/
├── WEB-INF/
│   ├── classes/    # Java类文件
│   ├── lib/        # 依赖JAR包
│   └── web.xml     # 部署描述符
└── index.jsp       # 入口页面</code></pre>
    <h4 id="2.%20%E7%BC%96%E5%86%99%E6%B5%8B%E8%AF%95JSP" name="2.%20%E7%BC%96%E5%86%99%E6%B5%8B%E8%AF%95JSP">
     2. 编写测试JSP
    </h4>
    <pre><code class="language-java">&lt;!-- index.jsp --&gt;
&lt;%@ page contentType="text/html;charset=UTF-8" %&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;我的第一个Tomcat应用&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;&lt;%= new java.util.Date() %&gt;&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>

    <h3 id="%E4%BA%94%E3%80%81%E6%A0%B8%E5%BF%83%E9%85%8D%E7%BD%AE%E8%AF%A6%E8%A7%A3" name="%E4%BA%94%E3%80%81%E6%A0%B8%E5%BF%83%E9%85%8D%E7%BD%AE%E8%AF%A6%E8%A7%A3">
     五、核心配置详解
    </h3>
    <h4 id="1.%20%E4%BF%AE%E6%94%B9%E7%AB%AF%E5%8F%A3%E5%8F%B7" name="1.%20%E4%BF%AE%E6%94%B9%E7%AB%AF%E5%8F%A3%E5%8F%B7">
     1. 修改端口号
    </h4>
    <pre><code class="language-XML">&lt;!-- conf/server.xml --&gt;
&lt;Connector port="8080" protocol="HTTP/1.1"
           connectionTimeout="20000"
           redirectPort="8443" /&gt;</code></pre>
    <h4 id="2.%20%E9%85%8D%E7%BD%AE%E8%99%9A%E6%8B%9F%E4%B8%BB%E6%9C%BA" name="2.%20%E9%85%8D%E7%BD%AE%E8%99%9A%E6%8B%9F%E4%B8%BB%E6%9C%BA">
     2. 配置虚拟主机
    </h4>
    <pre><code class="language-XML">&lt;Host name="www.mysite.com" appBase="webapps"
      unpackWARs="true" autoDeploy="true"&gt;
    &lt;Context path="" docBase="MyWebApp" /&gt;
&lt;/Host&gt;</code></pre>
    <h4 id="3.%20JVM%E5%8F%82%E6%95%B0%E4%BC%98%E5%8C%96" name="3.%20JVM%E5%8F%82%E6%95%B0%E4%BC%98%E5%8C%96">
     3. JVM参数优化
    </h4>
    <pre><code class="language-bash"># bin/setenv.sh（新建）
export JAVA_OPTS="-Xms512m -Xmx1024m -XX:+UseG1GC"</code></pre>

    <h3 id="%E4%B8%83%E3%80%81%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98%E6%8E%92%E6%9F%A5" name="%E4%B8%83%E3%80%81%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98%E6%8E%92%E6%9F%A5">
     七、常见问题排查
    </h3>
    <h4 id="1.%20%E7%AB%AF%E5%8F%A3%E5%86%B2%E7%AA%81" name="1.%20%E7%AB%AF%E5%8F%A3%E5%86%B2%E7%AA%81">
     1. 端口冲突
    </h4>
    <pre><code class="language-bash"># Linux查看端口占用
netstat -tunlp | grep 8080

# Windows查看端口占用
netstat -ano | findstr 8080</code></pre>
    <h4 id="2.%20%E5%BA%94%E7%94%A8%E7%83%AD%E6%9B%B4%E6%96%B0%E5%A4%B1%E8%B4%A5" name="2.%20%E5%BA%94%E7%94%A8%E7%83%AD%E6%9B%B4%E6%96%B0%E5%A4%B1%E8%B4%A5">
     2. 应用热更新失败
    </h4>
    <p>
     <strong>
      解决方案
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       清除work目录缓存
      </p>
     </li>
     <li>
      <p>
       重启Tomcat
      </p>
     </li>
     <li>
      <p>
       检查web.xml配置
      </p>
     </li>
    </ol>
    <h4 id="3.%20%E5%86%85%E5%AD%98%E6%BA%A2%E5%87%BA" name="3.%20%E5%86%85%E5%AD%98%E6%BA%A2%E5%87%BA">
     3. 内存溢出
    </h4>
    <p>
     <strong>
      错误提示
     </strong>
     ：
    </p>
    <pre><code class="language-XML">java.lang.OutOfMemoryError: Java heap space</code></pre>
    <p>
     <strong>
      处理步骤
     </strong>
     ：
    </p>
    <ol>
     <li>
      <p>
       调整JVM参数：-Xmx增大堆内存
      </p>
     </li>
     <li>
      <p>
       使用jmap生成堆转储文件分析
      </p>
     </li>
     <li>
      <p>
       检查是否有内存泄漏
      </p>
     </li>
    </ol>
    <hr/>
    <h3 id="%E5%85%AB%E3%80%81%E6%9C%80%E4%BD%B3%E5%AE%9E%E8%B7%B5%E5%BB%BA%E8%AE%AE" name="%E5%85%AB%E3%80%81%E6%9C%80%E4%BD%B3%E5%AE%9E%E8%B7%B5%E5%BB%BA%E8%AE%AE">
     八、最佳实践建议
    </h3>
    <ol>
     <li>
      <p>
       <strong>
        生产环境必做
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         删除webapps下默认应用（docs/examples等）
        </p>
       </li>
       <li>
        <p>
         修改shutdown.sh默认端口（防止恶意关闭）
        </p>
       </li>
       <li>
        <p>
         开启访问日志（conf/server.xml配置AccessLogValve）
        </p>
       </li>
      </ul>
     </li>
     <li>
      <p>
       <strong>
        安全加固
       </strong>
       ：
      </p>
      <pre><code class="language-XML">&lt;!-- 禁用危险HTTP方法 --&gt;
&lt;security-constraint&gt;
    &lt;web-resource-collection&gt;
        &lt;url-pattern&gt;/*&lt;/url-pattern&gt;
        &lt;http-method&gt;PUT&lt;/http-method&gt;
        &lt;http-method&gt;DELETE&lt;/http-method&gt;
    &lt;/web-resource-collection&gt;
    &lt;auth-constraint/&gt;
&lt;/security-constraint&gt;</code></pre>
     </li>
     <li>
      <p>
       <strong>
        性能调优
       </strong>
       ：
      </p>
      <ul>
       <li>
        <p>
         启用NIO连接器
        </p>
       </li>
       <li>
        <p>
         合理配置线程池
        </p>
       </li>
       <li>
        <p>
         开启GZIP压缩
        </p>
       </li>
      </ul>
     </li>
    </ol>
